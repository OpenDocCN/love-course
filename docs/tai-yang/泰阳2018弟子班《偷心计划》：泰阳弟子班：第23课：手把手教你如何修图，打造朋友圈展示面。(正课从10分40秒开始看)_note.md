# 泰阳弟子班《偷心计划》第23课：手把手教你修图，打造朋友圈展示面 📸

在本节课中，我们将学习如何使用手机软件进行图片后期处理，以提升个人照片的质感与格调，从而打造出优秀的朋友圈展示面。课程将从修图的核心目的讲起，逐步演示具体的操作步骤与软件使用技巧。

## 课程概述

修图的根本目的是**展示一个更优秀的自己**。这包括三个层面：人物本身的帅气、照片的质感与清晰度、以及衣着与场景所传递的时尚感。优秀的照片能让他人对你产生良好的第一印象和幻想。

上一节我们明确了修图的目的，本节中我们来看看具体的操作流程。我们将主要使用两款手机软件：**美图秀秀**和**MIX**。

## 第一步：使用美图秀秀处理人物

美图秀秀的核心功能是处理人像。我们应遵循“先修人，再修图”的原则。

以下是处理人物的具体步骤：

1.  **去斑去痘**：首先使用“人像美容”功能下的“去斑去痘”工具。将画笔大小调整合适，点掉脸上和身上明显的痘痘、疤痕或痣。这是一个需要耐心的过程。
2.  **瘦脸瘦身**：接着使用“瘦脸瘦身”工具。这个工具不仅可以微调脸型（如下巴、鼻子、发际线），还可以修饰身体轮廓（如手臂、肚子）。通过推动像素点来实现调整，注意力度要自然。
3.  **磨皮**：最后使用“磨皮”功能来改善皮肤质感。可以选择“自动”模式，并调整磨皮强度滑块。注意不要过度磨皮，以免失去皮肤的真实质感。

操作过程中，可以使用右上角的对比按钮查看效果，或使用红色返回键撤销错误操作。完成人像处理后，务必点击保存。同时，记得在美图秀秀的设置中，将画质调整为“全高清”，以保证输出图片的清晰度。

## 第二步：使用MIX添加滤镜与裁剪

MIX的核心功能是为图片添加风格化滤镜并调整构图，这直接决定了照片的“格调”。

以下是使用MIX的步骤：

1.  **选择滤镜**：导入美图秀秀处理好的图片，在“滤镜”库中尝试不同风格。**电影色**和**极显胶片**系列是常用且不易出错的选择。选择滤镜时需兼顾人物肤色和整体氛围，避免让脸部过红或过暗。
2.  **调整滤镜强度**：选定滤镜后，若觉得效果过重，可以点击滤镜右下角的三个点，通过调整百分比来减弱滤镜强度，使其更自然。
3.  **裁剪构图**：点击左下角的“裁剪”工具。根据审美调整图片构图，例如裁掉不必要的背景、杂乱的元素，或改变长宽比例，使视觉焦点更突出。一个好的构图公式是：**主体突出 + 背景简洁**。
4.  **保存图片**：满意后，点击保存即可得到最终成片。

## 不同类型图片的修图思路

掌握了基本流程后，我们来看看如何应对不同类型的图片。

上一节我们学习了标准的人像修图流程，本节中我们来看看一些特殊情况的处理。

以下是几种常见情况的处理思路：

*   **不带人物的风景/物品照**：这类图片无需使用美图秀秀。直接导入MIX，核心任务是**通过滤镜提升质感与调整明暗**。例如，原图太暗，就选择能让画面变亮的滤镜。
*   **光线不足的人像**：如果原图人脸太暗，在美图秀秀中可尝试使用“磨皮”功能，它有一定提亮效果。随后在MIX中选择能整体提亮且不失格调的滤镜。
*   **背景杂乱的照片**：在MIX的“裁剪”环节，首要任务就是裁掉画面中无关的、杂乱的人物或物体，保持画面干净。
*   **无法修图的废片**：有些照片本身质量太差，例如**严重曝光过度（过曝）** 或**光线极度不足**，导致细节全无。这类图片应直接舍弃，因为后期难以补救。核心原则是：**前期拍摄的光线越好，后期修图越容易出效果**。

## 核心要点：审美是关键

修图技术本身并不复杂，其核心在于**个人的审美能力**。软件操作是固定的，但如何判断一张照片的好坏、如何选择恰当的滤镜、如何进行裁剪，都依赖于你对“美”的感知。

你需要培养的是发现照片缺陷（如痘痘、杂乱、过暗）的能力，以及知道如何通过工具将其转化为优势（干净、有格调、明亮）的眼光。你的修图成果，最终反映的是你的品味。

1.  使用**美图秀秀**进行人像精修（去痘、瘦脸、磨皮）。
2.  使用**MIX**添加滤镜、调整构图以提升格调。
3.  分析了不同类型图片的修图思路与取舍标准。
4.  强调了**审美能力**是决定修图效果的核心。

记住，修图的目的是为了展示一个更精致、更有品位的自己。多加练习，培养你的眼光，你也能轻松打造出令人印象深刻的朋友圈展示面。
