## 核心功能与使用方法

现在，我们进入实际操作环节，手把手学习如何使用世纪佳缘APP。

我已经打开了世纪佳缘APP的界面。主页会显示女生照片，这是主流的使用方式：左滑不喜欢，右滑喜欢。点击照片可以查看资料。

### 资料页详解

世纪佳缘的资料页信息非常详细，这符合其严肃交友的定位。

除了照片，你还可以看到：
*   个人独白
*   年龄、身高、体重、学历、婚姻状况、地区
*   收入、买房买车情况
*   属相、籍贯、户口、血型
*   择偶条件

资料设置明显区别于其他APP，带有严肃的交友目的。

资料页下方有四个按钮：
*   **喜欢**：点击表示喜欢。如果对方也喜欢你，即可匹配聊天。
*   **送礼**：**不建议使用**。初期送礼效果不大，第一印象取决于个人资料和展示。
*   **牵线**：由平台红娘介绍认识。**没有必要**，所有事情应靠自己。
*   **聊天**：主动向女生打招呼。即使对方没点击喜欢，你也可以先打招呼。**注意：此功能需要付费**，后面会统一讲解。

### 其他主要功能板块

看完资料页，我们来看看其他功能。

首先看右上角的**搜索功能**。点进去可以看到：
*   **附近的人**：显示地理位置附近的用户。
*   **擦肩而过**：如果双方使用APP时地理位置有过交集，系统会显示。但这只是系统设置，无需太在意。
*   **标签筛选**：可以根据标签筛选女生，但这并非核心。

**附近的人**和**擦肩而过**功能，点击头像可查看资料，操作同样是左滑/右滑。

第二个大板块是**相亲主题**。包含直播相亲等功能。但这种方式花费时间多、效率低，且可能涉及送礼、收费，**不建议使用**。

第三个板块是**消息板块**。所有打过招呼或匹配过的女生都在这里进行聊天互动。

这里涉及世纪佳缘的一个重要特色：**只要聊天，就需要花钱**。

如果未付费，你发送的消息对方无法直接查看。对方会收到提示，但需付费（如购买“邮票”）才能阅读。同样，对方发给你的消息，你也需要付费才能查看。后面会讲解如何通过开通会员解决此问题。

消息板块内还有“联系人”功能，分类显示聊友、新联系人等，属于通用功能，不做赘述。

前面的功能已介绍完毕，本节中我们来看看最高效的核心功能：**地图交友**。

在“发信”功能板块中，找到“地图交友”。界面类似百度地图，显示女生的分布。

2.  点击下方的 **“全部喜欢”** 按钮。
3.  系统提示将给一组人（例如10人）发送消息，点击确定。
4.  发送成功后，点击 **“换一组”**。
5.  新的女生出现，再次点击“全部喜欢”发送消息。

每日打招呼有上限（约100-200次）。通过几天时间，即可覆盖周边所有女生。

使用核心策略后，个人资料设置就成为重中之重。它决定了你的第一印象、吸引力和信任感。
