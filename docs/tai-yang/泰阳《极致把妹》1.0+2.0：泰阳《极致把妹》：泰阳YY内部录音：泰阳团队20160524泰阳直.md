## 核心概念解析 🔑

在深入对话分析前，我们首先需要理解几个关键概念。

**吸引力框架**：指在互动中，你通过语言、行为和态度所传递出的自信、价值和主导感。一个强大的框架能让你不被对方牵着鼻子走。
```
框架强度 = 你的确定性 - 对方的摇摆性
```

**废物测试**：指对方（通常是女生）为了测试你的自信、一致性或真实意图而提出的挑战性问题或行为。例如，“你身边是不是有很多女生？”

**关系升级**：指通过语言或肢体接触，将两人关系从普通朋友向更亲密方向推进的过程。关键在于自然和时机。

1.  **预设框架**：在见面前列出你的目标（如展示自信、快速升级）和可能遇到的测试。

5.  **主动推拉**：通过开玩笑、轻微否定（推）和表达欣赏、亲密（拉）来制造吸引力。
6.  **营造氛围**：选择合适地点，利用音乐、食物、共同活动来催化情感。
