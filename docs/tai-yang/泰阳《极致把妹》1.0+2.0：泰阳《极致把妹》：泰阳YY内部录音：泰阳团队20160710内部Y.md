# 课程一：作业批改与展示面拍摄要点 📝

在本节课中，我们将学习如何正确完成第一周的搭讪与展示面拍摄作业。课程内容基于对几位学员作业的批改，我们将分析常见问题，并明确合格作业的标准与改进方向。

## 作业提交规范与常见问题

上一节我们概述了课程目标，本节中我们来看看作业提交的具体要求和学员常犯的错误。

首先，作业提交需要规范格式。学员应在作业标题中写明自己的微信昵称和“第几周作业”，以便导师快速识别。

以下是本次批改中发现的几个关键问题：

1.  **搭讪记录不达标**：多位学员未完成“每天搭讪一次”的基本要求，搭讪次数不足。
2.  **记录方式错误**：搭讪记录不应只是自我总结（如“情绪紧张”）。合格的记录需要包含两部分：
    *   **客观事实**：记录搭讪时的场景、对话内容等实际情况。
    *   **主观心理**：记录自己当时的心理状态和感受。
3.  **任务理解偏差**：有学员认为应先完成形象改造再开始搭讪。实际上，**形象改造（如拍摄展示面）与日常搭讪练习是并行不悖的**，两者应同时进行。

## 学员作业案例分析：魏小虎

接下来，我们通过具体案例来深入理解。首先分析学员魏小虎的作业。

魏小虎的搭讪作业未达到次数要求，因此不予通过，需要下周重做第一周任务。但其展示面拍摄作业有可圈可点之处。

以下是对其四张展示面照片的具体分析：

*   **健身房照片**：
    *   **优点**：身着专业运动装备，体现了丰富多彩的生活爱好；取景和背景虚化处理得当；使用了Snapseed软件的“戏剧”滤镜，效果不错。
    *   **改进建议**：应尽量露出脸部，让女生更容易记住你。公式可表示为：**有效展示面 = 优质内容 (生活/颜值/价值) + 良好视觉处理 (构图/滤镜) + 个人辨识度 (露脸)**。
*   **海边照片**：
    *   **优点**：造型、服饰（白T、帆布裤、手链、项链）体现了精致感；姿势和滤镜运用得当。
    *   **改进建议**：受限于海水浑浊，背景质感稍差，但整体仍可用作展示面。
*   **沙发看杂志照片**：
    *   **缺点**：主题平淡（看杂志），无法体现高价值或品质生活；背景普通。此类照片不建议作为展示面。
    *   **改进技巧**：若想提升此类照片价值，可采用“低调展示”策略，即在角落不经意地露出贵重物品（如首饰、钱包），达到“无形装逼”的效果。
*   **酒店房间照片**：
    *   **优点**：环境有格调；采用看向别处的“抓拍”姿势，减少了刻意感，显得自然慵懒；发型（侧背头）和露出的纹身增添了细节和个性。
    *   **总结**：这张照片的拍摄技巧（姿势、场景选择）值得其他学员学习。

## 学员作业案例分析：刘步（静静等待）

上一节我们分析了相对优秀的案例，本节中我们来看看需要较多改进的案例。

学员刘步（静静等待）同样未提交合格的搭讪记录，因此作业不予通过。其展示面照片问题较为明显。

以下是其照片存在的主要问题及改进建议：

1.  **服饰质感差**：衣服视觉上显得廉价，类似“非主流”韩版风格，严重影响照片档次。**改进核心是升级服装质感**。
2.  **未使用滤镜**：照片直接输出，没有进行任何调色或滤镜处理，显得平淡且不专业。
3.  **拍摄场景重复**：多张照片在同一场景拍摄，内容重复。建议**一个场景精选一张最佳照片即可**。
4.  **修图手法粗糙**：使用低劣的“涂抹”功能来遮盖瑕疵，效果生硬。推荐使用专业的**仿制图章或修复工具**进行处理，使修改痕迹更自然。

关于修图软件，推荐以下两款：
*   **Snapseed**：功能强大的手机修图软件。
*   **Mix**：拥有丰富滤镜选择的编辑软件。

代码示例：虽然无法直接展示软件操作，但核心工作流可以概括为：`原始照片 -> 基础调整（亮度、对比度）-> 滤镜应用 -> 局部精修 -> 输出成品`。

## 搭讪技巧要点回顾

在分析了展示面之后，我们回到最基础的搭讪环节，总结一下关键技巧。

无论是直接开场还是间接开场，核心在于**自然过渡到收号环节，并给出一个简单的理由**。

*   **间接开场**：例如问路。流程可以是：“请问XX怎么走？” -> （对方回答后） -> “我怕找不到，方便加个微信，万一我迷路了再问你吗？” 这提供了一个合理的收号借口。
*   **直接开场（推荐）**：直接表达欣赏和意图，显得干脆利落。标准话术结构为：“你好，我刚刚看到你，觉得你挺可爱的/裙子很漂亮，想认识一下。但我朋友在那边等我，不然我们加个微信，下次再聊？”

关于收号后的行为：如果对方拒绝后继续离开，可以尝试提高音量再说一句话。若对方有停顿，可跟上继续交流；若毫无停顿，则应放弃跟随，强求到的号码后续联系成功率也极低。

## 课程总结与下周要求

本节课中，我们一起学习了第一周作业的批改要点。

我们明确了合格作业的标准：**搭讪需每日一次并详细记录客观事实与主观心理；展示面拍摄需注重主题价值、视觉美感，并熟练运用修图软件**。

最后，重申下周作业要求：
1.  严格完成每日一次搭讪，并按要求记录。
2.  在一周内选择一天，前往咖啡馆（或其他合适场所）拍摄两张展示面照片：一张人像，一张静物（景物）。
3.  对照片进行修图并添加滤镜。
4.  提交作业时，标题格式为：`[微信昵称]第X周作业`。
5.  请将YY语音平台上的昵称修改为微信群内的昵称，以便对应。
