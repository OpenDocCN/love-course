# 泰阳《极致聊天课》第四课：正确聊天应具备的几种心态 🧠

在本节课中，我们将要学习与女生聊天时应保持的几种核心心态。正确的聊天心态是有效沟通和建立吸引力的基础，它直接决定了你话语背后的能量和姿态。

上一节我们介绍了聊天中框架的重要性，本节中我们来看看如何通过调整内在心态，让每一句话都自然散发出吸引力。与女生聊天时，你说的每一句话都反映了你是一个什么样的人，以及你当下处于何种心态。

一般情况下，我们在聊天中需要保持和坚持以下四种心态。

以下是四种核心聊天心态的详细解析：

1.  **保持有趣与幽默的心态**

    这个回应假想了一个有趣的场景，让双方扮演特定角色，从而瞬间打开女生的情绪开关。

2.  **保持奖品性心态**

本节课中我们一起学习了聊天的四种核心心态：保持有趣幽默、坚持奖品性、建立自信不受影响，以及勇于破除焦虑。真正会聊天的人，大多能成为社交中的主导者。通过培养这些心态，不仅能提升你与异性沟通的魅力，更能将这种自信和领导力应用于生活的各个方面。
