# 为什么我支持货真价实的神职人员在两性领域劝善？

二、他们在“应然”上劝善的时候，是基于上帝存在的前提的。你要是当老实人正直人，不会吃亏的，这个不会吃亏不是在无神论前提下的。神职人员不会说没有神的话老实正直的人也是福报满满。
