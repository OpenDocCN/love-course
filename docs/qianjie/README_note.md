# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“Hello World”存储在其中：
```python
message = "Hello World"
```
这里，`message` 是变量名（盒子的标签），`"Hello World"` 是存储的值（盒子里的东西）。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的类型。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其说明：

*   **整数（int）**：代表没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数（float）**：代表包含小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**：代表文本信息，需要用单引号（`'`）或双引号（`"`）括起来，例如 `"Python"`, `'123'`。
*   **布尔值（bool）**：只有两个值，`True`（真）或 `False`（假），常用于逻辑判断。

## 如何查看与使用类型？ 🔍

知道了有哪些类型，我们如何查看一个变量到底是什么类型呢？Python提供了一个内置函数 `type()` 来帮助我们。

例如，执行以下代码：
```python
price = 9.99
print(type(price))
```
屏幕上将会显示 `<class 'float'>`，这表明变量 `price` 是一个浮点数。

在编程时，了解数据类型很重要，因为它决定了你能对这个数据做什么操作。例如，数字可以进行加减乘除，而字符串通常可以拼接或查找子串。

## 一个简单的综合例子 💻

现在，让我们把以上概念结合起来，看一个完整的例子。

以下是创建不同数据类型变量并打印其类型的过程：
```python
# 定义一个字符串变量
name = "倩姐"
# 定义一个整数变量
age = 18
# 定义一个浮点数变量
height = 1.65
# 定义一个布尔值变量
is_student = True

# 打印这些变量的类型
print(type(name))     # 输出：<class 'str'>
print(type(age))      # 输出：<class 'int'>
print(type(height))   # 输出：<class 'float'>
print(type(is_student)) # 输出：<class 'bool'>
```
