## 约会时间的选择 ⏰

上一节我们介绍了约会的总体重要性，本节中我们来看看如何选择最佳的约会时间。许多男生选择时间仅遵循两个原则：双方都有空，且最好是能相处一整天的周末。

实际上，约会时间不应贪多，效率更为重要。最佳时间段是**下午到晚上**。人在白天紧绷后精神会更放松，更愿意外出活动。同时，该时段给予女生充足的准备时间。

此外，傍晚光线较暗，能自动为人的外观增添一层滤镜效果。即使你的皮肤状态不佳，女生也难以看清。若你想进行牵手等亲密动作，女生也可能因夜晚的松弛状态而默许。
