在本节课中，我们将要学习如何通过与女生闺蜜的聊天方式获得启发，从而改善与女生的对话。许多男生发现与女生聊天困难，但她的闺蜜却能轻松聊得热火朝天。我们将整理出闺蜜聊天的三个核心话术，帮助你掌握更自然、有效的沟通技巧。

## 第一点：自说自话，自问自答

上一节我们概述了学习闺蜜聊天法的必要性，本节中我们来看看第一个核心技巧：自说自话与自问自答。

男生常做的一件事是等待对方回复消息。你发一条消息就等，对方回复后你秒回，然后又继续等待。请停止这种等待，尝试新的方法：既然对方不立即回答，你就自问自答。

女生并非24小时都与闺蜜无时差地秒回。与其期待她一条一条地回复，不如尝试自己把话说完。这样做可以大大减轻你给对方带来的压力，也能完整表达自己想说的话。

不必担心她不接你的话茬。例如，如果你想约她出去，就从打招呼直接说到好吃的餐厅；想讨论喜欢的电影，就从打招呼聊到自己喜欢的电影。至于表白，还是等约她出来再说，这不能自问自答。

**执行公式**：
```
如果 (对方未即时回复) {
    则 (继续表达，完成一个完整的话题段落)
}
```

以下是应用此方法时需要注意的事项：

*   此方法需根据两人的熟悉程度来执行。如果刚刚认识就如此发消息，肯定不合适。
*   对于刚接触的女生该如何聊天，欢迎点进主页深入学习。
*   如果你仍不清楚，也欢迎私信询问你的具体问题。

## 第二点：用情绪来说话

掌握了自说自话的技巧后，我们进入下一个关键点：用情绪来说话。我们常说男人理性，女人感性。感性与理性的碰撞，往往不会有好的结果。保持理性，是许多男生与女生聊不下去的根本原因。

心理学上有一个相似吸引法则，意思是人会被和自己相似的人所吸引。举个例子：女生说她贴了一张钢化膜花了100块钱，感觉被坑了。

*   **男生的典型反应**：理性分析。“这明显是坑，傻子才会这样。”或者建议“向相关部门举报”，直接提供解决方案。
*   **闺蜜的典型反应**：先输出情绪。“这老板太黑心了！你也太惨了！”然后才给出方法：“现在就得想法子把钱要回来，不能吃闷亏。”

可以看出，闺蜜是先给出**情绪**共鸣（共情安抚），再提供解决方案。而男生有时过于直白，只专注于解决问题。其实你只需多一步，进行简单的情绪输出，让她感觉你能感同身受，就能事半功倍。

**沟通模型**：
```
接收信息 -> **接受并反馈对方情绪** -> 最后输出解决方案或观点
```

所以，男生一定要学会用情绪说话。相信你也会因此变成她最亲密的人。

## 第三点：用关心结束对话

学会了在聊天中注入情绪后，我们来看看如何优雅地结束对话。每次和女生聊天，总不知道该如何结束，最后只能尴尬地挤出一句：“那今天就先到这里，你快去休息。”

无论之前聊得多好，气氛多到位，这样的结尾绝对是大扣分项。它像上班打卡，任务完成后通知下班。后面那句短促的关心，更显得敷衍，给人留下不好的印象。

那么闺蜜是如何结束话题的呢？答案就是用**关心**来自然结束对话。

以下是闺蜜式结束语的例子：

*   “你还没吃饭吧？快去吃点东西，别饿着了。”
*   “你追的综艺更新了，快去看吧！”

当然，要达到这个水平，需要你对对方有一定的了解，在合适的时间说出合适的话。而了解的途径，相信你在实践前面两点时已经有所收获。

## 总结

本节课中，我们一起学习了改善与女生聊天的三个核心话术，灵感来源于她闺蜜的沟通方式：

1.  **自说自话，自问自答**：减少等待压力，完整表达自我，根据熟悉度灵活运用。
2.  **用情绪来说话**：遵循“接受情绪 -> 反馈情绪 -> 输出观点”的模型，先共情再解决问题。
3.  **用关心结束对话**：以自然、体贴的关心作为话题的收尾，避免生硬和敷衍。

最后需要提醒大家，闺蜜聊天法只是一种拉近你与她关系的方法，一定要自己把握好度。过分亲近可能会变成“男闺蜜”。
