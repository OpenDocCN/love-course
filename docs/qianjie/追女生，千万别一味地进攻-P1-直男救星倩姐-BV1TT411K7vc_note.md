## 错误示范：过度纠缠的后果

上一节我们概述了课程主题，本节中我们来看看一个具体的反面例子。这个案例清晰地展示了不当行为如何迅速消耗女生的好感。

女生以时间太晚为由拒绝：“太晚了，下次吧。”

男生表达强烈想念：“可是我很想见到你。”

女生再次拒绝，并给出解释：“今天真的太累了，下次我一定出来。”

从女生的回复可以看出，她对男生有一定好感。否则她会直接拒绝，不会过多解释。

*   **正确选择：顺势晚安。** 这尊重了女生的意愿，保留了后续机会。
*   **错误选择：纠缠不放。** 案例中的男生正是如此，他的需求感过强，导致了后续一系列问题。

女生始终没有回复。男生在楼下等到凌晨两点，最终只能离开。

女生直到晚上才冷淡回复两个字：“干嘛。”

**成功建立吸引 ≠ 一味进攻**
