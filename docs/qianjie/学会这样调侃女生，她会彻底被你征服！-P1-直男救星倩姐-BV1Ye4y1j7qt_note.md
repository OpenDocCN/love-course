## 引入核心概念：边际效应递减

在经济学上有一个现象，叫做**边际效应递减**。一件事情，你第一次接触的时候，情感体验是最强烈的，第二次接触的时候就会淡一些，那么第三次会更淡，以此发展。我们接触这件事情的次数越多，我们的情感体验也就会越来越淡薄，一步一步的趋向于乏味。

单纯的去夸女生，是没有办法给女生带来情绪波动的，因为她不需要做任何的事情，就可以获得你的夸赞。你的夸赞对于她来说是不值钱的。

所以说不要再轻易的去夸女生了。甚至有的时候，我们还可以传递出来一种不认可，那么你偶尔再去认可她的时候，女生才会觉得你的认可更为珍贵。

就像我们上学的时候，比如说我们的老师，他非常严厉，很少夸赞我们。那么在这种情况下，如果老师偶尔对我们说了一两句夸赞，会显得更加的弥足珍贵。

老师偶尔的夸赞，可以更多的激发出你内心想要努力的动力，因为她对你的夸赞，是你付出努力之后而获得的。这样的夸赞，我们不会感到厌烦，也不会麻木。我们会为了追随夸赞，而不停的去努力。

那么我们该如何传递出这种不认可？这期视频想要讲的重点就是两个字：**调侃**。
