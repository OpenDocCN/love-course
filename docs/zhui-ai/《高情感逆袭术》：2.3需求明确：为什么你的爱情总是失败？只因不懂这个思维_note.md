# 《高情感逆袭术》：2.3需求明确：为什么你的爱情总是失败？只因不懂这个思维_note

在本节课中，我们将要学习亲密关系中的一个核心思维：**需求明确**。许多感情失败的根本原因，在于个人需求不清晰，这会导致关系中的一系列问题。我们将深入探讨如何明确自身需求，并建立稳固的内在系统，从而避免因贪婪、情绪化和人性弱点而破坏关系。

## 一、 感情失败的根源：需求不明确

上一节我们探讨了关系中的置换思维。本节中我们来看看，为什么人们常常难以实践“置换”，反而陷入“索取”的困境。

很多人感情的失败，源于没有想清楚自己到底想要什么。**需求明确**，首先是明确你自己的需求。这涉及到我们所说的置换与索取的问题。

大家都已经清楚，置换非常重要。所有东西都是换来的，而不是要来的。那为什么人们特别喜欢索取呢？除了自私，更主要的原因是大家不太清楚自己到底想要什么。

一个连自己想要什么都不知道的人，不可能体察到别人的需求，然后给别人想要的东西。他也不可能甘心舍弃自己想要的东西去跟别人置换。他一定是什么都想要。

所以，人们习惯于索取，就是因为自己内心想要的东西是不确定的。因此，这种人往往会失败。他太贪心了，而贪心源于需求不明确。

## 二、 实现需求明确的三个步骤

为了解决需求不明确的问题，你需要做到以下三件事。以下是具体步骤：

**第一件事：明确自身需求**
你必须问自己：你到底想要什么？你想在这段感情中获得什么？你想寻找一个什么样的人？

**第二件事：建立自我评价系统**
自我评价系统是自信的来源。你怎么评价自己，就会如何向外界证明自己。
*   如果你觉得自己很失败，就会努力向外界证明你是个失败者。
*   如果你觉得自己很成功，就会努力向外界证明你是个成功者。
*   你觉得自己没有存在感，就会拼命刷存在感。

自我评价系统必须牢不可破，你才有做任何事情的动力。如果你的自我评价系统不准确，你就会做一件事：希望在别人那里得到对你的评价。这会导致你极度缺失安全感，导致你在所有人那里都想表现得更好，或用各种手段获取他人评价。

就像小孩想要东西得不到时会哭闹，很多人把这个习惯带到了成年世界。虽然生理已成年，但内心没有，因为他们没有建立起属于自己的评判系统。这种人也是脆弱的（所谓的“玻璃心”），别人说两句就受不了，抗压能力差，就是因为他们没有一个坚实的自我评价。

**渣男摧毁你的核心，就是破掉你的自我评价系统**，让你觉得自己是失败者，应该为他服务，必须去挽留他，在感情中位置很低。所以，建立自己的评价系统至关重要。

**第三件事：避免因情绪改变需求**
没有稳固自我评价系统的人，特别容易陷入第三件事：因为情绪改变需求。

很多人看“毒鸡汤”文章后，会立刻拿着文章去质问伴侣：“你看，上面都说男人应该这么做，你怎么不对我这样？”可能这个女孩看文章之前的认知并非如此，但看完后立刻被洗脑，认为就应该这样。

而一个有自我评价系统的人，能清楚识别那是“毒鸡汤”，很扯淡。他能看到里面隐藏的逻辑线，知道这种文章是在洗脑、煽动情绪、鼓动自私，就不会做出这样的事。

如果你没有自我评价系统，你很容易因为情绪让需求变得不确定：一会儿想要这个，一会儿想要那个；一会儿因为这件事生气需求变了，一会儿又因为那件事需求变了。这会导致你永远找不到真正想要的、能陪伴下半生的人。同样，也没有人能受得了，会觉得你反复无常、情绪不稳定。这都是小孩会做出来的事。

**提高情感段位，本质是提高心智的段位**，让你拥有更稳定的内核。这时，你才能变成一个理性聪明的人。

## 三、 警惕环境与信息的影响

我们通过三个案例来看看环境的影响。我建议大家，不要跟感情失败者做朋友，或者不要轻信他们说的任何话。

有非常多感情失败的人，他们一定会甩锅：首先会说男人有多么不好，然后说自己有多么委屈，说对方多么自私，对自己有多么摧毁等等，最后评价“男人没一个好东西”。同样，感情失败的男人也会反过来说“女人都不是好东西”，诉说自己在感情里多么失败。

你长期接收这样的信息，就会变得和感情失败者一样充满怨念。虽然这么说可能不太好，但你身边那些离过婚的、感情很失败的闺蜜，少跟她们打交道。最后她们也会把你变成一个怨妇。这是实话，因为人接受的信息决定了他的认知，认知决定了他的行为。

你会因为这些信息，不断改变你对感情、对自我、对他人的评价系统，最后导致你的需求变得很混乱。所有心智不成熟的人都特别容易被别人干扰、洗脑。一段关系需要的是稳定，而你因为不确定的思维和认知，会把一段关系搅得乱七八糟。

## 四、 管理情绪，稳定需求

同样，有很多人一生气就改变自己的需求：生气前一个样，生气时一个样，生气后又是一个样。

我见过很多真正理性的女孩，比如我的一些前女友，她们在生气时会选择用一些方式化解掉心理的情绪，然后理性地解决问题。这其实是**男性思维**（偏向解决问题），而**女性思维**偏向解决情绪。但她不会让自己沉浸在那个情绪里。

举个例子，我们可以先做一个小约定：只要我们吵架了，就一定要去做一件事，并且吵架后一定要严格遵守这个基本流程。往往设置了这么一个东西之后，事情就不会变得很糟糕。

再比如，可以约定：可以大吵一架，但绝对不能冷暴力、冷战；吵完架后要互相道歉，或去把心里真实的想法沟通交流出来。如果真的理性，是可以做到这件事的。

如果你特别感性，每次吵架就翻旧账，或者一下冲动、失控，做出一些不好的事情，这是很可怕的。所有人里边最不能交往的，就是**不能做情绪管理的人**，即情绪失控的人。这种人最可怕：跟你分手时，半夜三点跟踪你、到你家敲门；生气后或要挽回你时，跪下磕头、扇自己嘴巴……类似的事其实挺常见。

这都是情绪不稳定的人。你遇到这种人，我建议你尽早远离。这都是因为他们自己的需求内核不稳定，不知道自己想要什么，所以一旦失控，什么事都做得出来。

## 五、 克制“人性三件套”

接下来讲一个关键点：**人性三件套**。其实所有人都有这三个需求：**懒惰**、**贪心**、**索取**。

各位可以好好想一想，这三条你占哪一条？
1.  **懒惰**：例如，“老公我饿了给我做饭吧”、“你去拖地吧”，自己坐在那指挥。你有没有这种情况或心理？
2.  **贪心**：什么都想要。例如，希望对方给钱，给了钱又贪心觉得“你怎么不给我陪伴”；陪伴和钱都有了，可能还要“我没有安全感”；有了安全感又会继续“你为什么猜不到我的心思”……类似的贪心越来越多，矛盾永远处理不完。因为你每一次的贪心，都是获得了一些你本不该获得的东西，对方也付出了他本无法付出的代价。一个脾气再好的人，也不会跟一个贪心无底洞的人打交道很久。
3.  **索取**：只想要，不想给。

说实话，每个人三条全占。但在生活里能克制自己这三个点的人，有多少？其实很少。

就拿一些成功人士的妻子举例，她们难道不会懒惰、贪心、索取吗？她们也会，心里也有这种想法。但是，她们能及时克制住，把它隐藏起来。

所以，各位想要在任何一段感情中保持稳定，并把感情运营得越来越好，这三点绝对是要克制的。这不光是对女性朋友讲的，男性朋友也一样。如果你找的伴侣这三点过于过分，你要学会引导他改过来，通过多次沟通和谈判。家庭关系、长期关系，一定要在吵架后来一次冷静的谈判，约定好底线。

当一个人屡屡突破你们设定的规则和底线时，这个人就不可救药了，不需要再跟他谈多久。这种人真的结了婚，就是一个祸害，因为他根本不尊重你们之间的感情。

对于“人性三件套”，可以适当满足，但不能过分压抑，也绝对不能过分放纵。一旦过分，感情绝对会失败。

## 六、 技巧：为需求加上“包装”

什么叫“外加包装”呢？比如你今天真的懒惰了或想贪心一下，你不要用命令、指挥的方式。要学会示弱，撒个娇，或者陈述一下今天想懒一下、想贪心一下的原因。示弱撒个娇，可能事情就搞定了。

不要直来直往，像吵架或指挥一样。因为任何人讨厌被改变、被指责、被限制，这是人最讨厌的事。所以不要往这个方向去做。

## 总结与行动指南

本节课中，我们一起学习了**明确自身需求**的重要性。感情失败往往源于贪婪和不确定，而这背后是自我认知的模糊和情绪的不稳定。

因此，我们要去明确自己的需求：到底想要什么？为了得到这个东西，你又需要去放弃哪些？

**课后行动**：听完这节课后，请列出你需求中最想要的那个人到底“长”什么样子（具备哪些特质）。

如果你发现这些需求里有很多对立的需求（比如我之前说的：有钱大概率没时间，有时间很可能没钱），你要把这些对立的需求摘出去。你能找到一个人满足你的三四点核心需求，而且是长期有价值的需求，那就真的很幸福了。

不过，你必须事先明确好你想要的。你不能在感情当中越来越贪心、越来越懒惰、越来越想索取，那不会有好下场。
