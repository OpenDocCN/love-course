# 《四横四纵2.0》：四横四纵2.0-1.宣言

欢迎来到四横思纵第一课宣言。本课程强调基本功，将深入讲解知识点，力求透彻理解重要且常用的知识点。

## 宣言基本定义

宣言的基本定义是告知对方，对方的行为对你产生了积极或消极的影响，导致对方拥有或失去你所需要的价值，从而触发你的社交需求。

### 宣言关键要点

1. **对方的行为**：具体描述对方的行为，而非笼统的个性描述。
2. **感受**：描述对方行为对你造成的感受。
3. **社交需求**：表达你因对方行为而产生的社交需求。

## 行为描述示例

- **正面行为**：在麦当劳手册中，鼓励员工时，具体描述员工的行为，如“你总是能够保持目光接触，并且面带微笑”。
- **正面行为表达**：夸赞街上气质佳的姑娘，“你今天打扮得非常有气质，特别是这条连衣裙有一种时尚模特的感觉，一下就吸引到了我”。

## 社交需求表达

- **正面行为表达**：表达对对方的喜欢和认可，如“让我忍不住想过来认识一下”。
- **负面行为表达**：表达对对方的不满和惩罚，如“你欺骗了我，让我非常的失望”。

## 讲程

讲程是指在对方行动或回应时，表达你的喜欢或不满。

### 讲程核心

- **表达喜欢或不满**：根据对方展现出的行为和品质，而非个人利益，来表达你的情感。

## 表达喜欢

- **温柔宽容宠溺的语气**：用正面感受和期望鼓励对方。
- **不求回报**：表达喜欢是单方面的，不期待对方的回报。

## 表达喜欢示例

- **正面行为**：对方生病时，表达关心，“啊你生病了，那还能亲亲吗，那还能慢慢打吗，我本来还想调戏调戏你的，看你生病了，那就放过你吧”。
- **负面行为**：对方欺骗你时，表达失望，“你欺骗了我，让我非常的失望，我本来对你是很有期待的，可现在我不得不来批评你的过错”。

## 表达惩罚

- **失格**：指出对方的负面特质，希望对方改正。
- **后撤**：在对方不改时，收回关注。

## 总结

本节课中，我们学习了宣言的基本定义、关键要点、行为描述、社交需求表达、讲程、表达喜欢和惩罚等知识点。希望这些内容能帮助你更好地理解和运用宣言技巧。
