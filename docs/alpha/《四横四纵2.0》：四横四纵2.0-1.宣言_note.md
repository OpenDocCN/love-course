# 《四横四纵2.0》课程1：宣言 🗣️

在本节课中，我们将要学习“宣言”这一核心概念。宣言是表达社交需求的基础工具，它用于告诉对方，他们的行为如何影响了你的感受，并因此促使你释放特定的社交信号。掌握宣言，是构建健康社交互动的重要第一步。

## 宣言的基本定义

宣言的基本定义是：告诉对方，对方的某些行为对你造成了积极或消极的影响和感受。这导致对方身上拥有或失去了你所需要的价值，让你不得不释放你的社交需求。

这段话高度概括了宣言的内涵，它包含三个关键要点。

## 宣言的三个关键要点

### 1. 描述具体行为，而非笼统个性

在宣言中表达社交需求时，无论是表达喜欢、认可，还是表达对方失去了某些资格或想要惩罚对方，都必须落脚到对方**具体做了哪些行为**上，而不是笼统地描述对方的个性。

以下是描述个性（应避免）和行为（应提倡）的对比：

*   **描述个性（模糊、应避免）**：
    *   “小王，你工作做得很好。”
    *   “小王，你今天表现得很棒。”
    *   “你人很nice。”
    *   “你怎么这么偷懒。”
    *   “你的态度非常恶劣。”

*   **描述行为（具体、应提倡）**：
    *   “我非常欣赏你招呼客人的方式，你总是能够保持目光接触，并且面带微笑，这一点做得非常棒。”（描述了“保持目光接触并微笑”的具体行为）
    *   “你今天打扮得非常有气质，特别是这条连衣裙有一种时尚模特的感觉，一下就吸引到了我。”（描述了“穿着特定连衣裙”的具体行为）
    *   “你欺骗了我，让我非常失望。”（描述了“欺骗”的具体行为）

上一节我们明确了宣言要基于具体行为，本节中我们来看看如何将行为与感受、需求结合起来。

### 2. 行为导致感受，感受触发需求

宣言的标准结构是：**对方的具体行为 → 对你产生的感受 → 你不得不释放的社交需求**。

这是一个完整的因果链条。例如：

*   **表达欣赏**：“你今天打扮得非常有气质……（行为）一下吸引到了我（感受），让我忍不住想过来认识一下（社交需求：希望建立联系）。”
*   **表达不满**：“你欺骗了我（行为），让我非常失望（感受），我不得不来指出你的过错（社交需求：希望对方改正）。”

“不得不”这种感觉，强调了是对方的行为强力驱动了你的反应，而非你无缘无故地主动索取。

### 3. 根据对方展现的品质进行“奖”或“惩”

当对方有所行动并给予反馈时，你需要根据对方**展现出的行为品质**来决定是表达喜欢（奖），还是表达失望、后撤（惩）。这被称为“奖惩”。

**奖惩的核心原则**：你表达喜欢还是失望，并不取决于对方是否满足了你的个人利益，而是取决于对方的行为是否展现出了使其成为“更好的人”的品质。

*   **奖（奖励）**：当对方展现出自信、真诚、上进等正面品质时，表达喜欢和认可。
*   **惩（惩罚）**：当对方展现出自私、欺骗、自负等负面品质时，表达失望并可能后撤。

人是在动态变化的，我们应根据具体事件中展现的品质来奖惩，而非给人贴上一个固定的标签。

## 如何表达“奖”（喜欢与认可）

用温柔、宽容、宠溺的语气和内容，表达你的喜欢、尊重、欣赏、渴望、仰慕、思念等一切正面感受和期望鼓励。表达之后，此事便告一段落，不求即时回报；但遇到新的机会，可以继续表达。

### “奖”与“讨好”的核心区别

这是一个关键误区，需要重点理解：

*   **奖（喜欢认可）**：核心是**不求回报**。你表达喜欢，是为了让对方享受被你喜欢的感觉，你因对方的优秀品质而快乐。公式是：**我喜欢你，是因为你做了X（具体行为/品质），这让我感到Y（正面感受）。**
*   **讨好**：核心是**期待回报**。你对对方好，是希望对方能回报你以好感、赞美或实际利益。这是一种隐性的索取。

**举例对比**：
当对方说“我生病了”：
*   **讨好（错误示范）**：“要不要我给你买药？我陪你去医院吧？你要照顾好自己，我好担心。”（背后逻辑：希望对方记住你的好并回报你。）
*   **奖（正确示范）**：“啊，你生病了？那快到我怀里来让我抱抱，你的病就好了~”（或女生说：“那你还能亲亲吗？看你生病了，先放过你吧。”）核心是**用宠溺的语气表达亲近和喜欢，而不聚焦于解决问题或索取感恩**。

### 表达“奖”的高级认知与注意事项

1.  **“表达之后就没有以后了”**：当你大方表达完喜欢后，这个话题就结束了，不要等待对方回应或立刻索取更多互动。这种“表达后抽离”会让对方产生“他到底是不是真的喜欢我？”的猜想，这种不确定性会激发对方更多的情感投入和幻想。
2.  **先言语，后行为检验**：首先从言语上大方、真诚地表达喜欢（例如：“我挺喜欢你这样的女孩子，希望我们能成为朋友”）。只有在对方不抗拒甚至接受这种言语上的喜欢后，再用行为（如邀约）去检验对方是否对你有好感。肢体接触是**检验关系程度**的工具，而非推进关系的主要手段。
3.  **避免“天天聊天”的行为化讨好**：不要每天无事找对方聊天。这会让你的“喜欢”变得廉价和烦人，同时也剥夺了对方陷入“思维怪圈”（猜测你是否真心）的机会。适度的抽离能增加你的吸引力和对方的情感投入。

## 如何表达“惩”（失格与后撤）

“失格”意为剥夺对方被你喜欢的资格。表达失格和后撤，并非为了侮辱对方，而是站在希望对方变得更好的高标准上，批评其展现出的自私、自负等负面（Beta/Omega）特质。

### “惩”的核心原则

1.  **目的是治病救人**：在对方犯错时，表达失望的同时应给予对方改正的机会。抱着“惩前毖后，治病救人”的态度。
2.  **后撤是最后手段**：如果对方在被告知问题后仍一意孤行，则需要通过“后撤”（如收回关注、暂时离开）来让对方意识到错误的严重性。因为没有失去的风险，人就不会学会珍惜。
3.  **针对品质，而非个人**：在Alpha的世界观里，爱的是一个人的品质，而非某个人本身。因此，失格和后撤也只针对对方不好的品质。只要对方愿意改正，随时可以重新接纳。

**表达“惩”的示例**：
“我跟你在一起，其实是非常想和你有未来的。但我感觉你每次都[重复的错误行为]，跟你说了好几次，你根本没有听进去。说实话，我真的很失望。本来我以为你是一个成熟有担当的男人，既然这样的话，我们都不要浪费彼此的时间了。”

**关键**：后撤不是在对方不理你时的赌气行为，而是在你做了所有努力（指出问题、给予机会）后，对方仍无改变时的最终警醒。离开，是为了用行动告诉对方：“你的问题需要被严肃对待。”

---

**本节课总结**：
我们一起学习了“宣言”这一核心社交工具。宣言用于基于对方的具体行为，表达我们的感受和社交需求。其关键在于：
1.  **描述具体行为**，而非笼统个性。
2.  **遵循“行为→感受→需求”** 的因果逻辑。
3.  根据对方展现的品质进行**奖惩**：“奖”是出于欣赏的不求回报的喜欢；“惩”是希望对方变好而进行的批评或后撤。
掌握宣言，能让你在社交中更清晰、更有力地表达自己，并引导关系向健康的方向发展。
