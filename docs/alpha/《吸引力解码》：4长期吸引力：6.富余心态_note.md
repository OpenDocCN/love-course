# 《吸引力解码》：4长期吸引力 - 6.富余心态

大家好！我是阿法，今天我们来探讨“富余心态”。首先，什么是富余？简单来说，富余就是一个人有很多选择。你可能经常听到我说男人要让自己有选择，不要匮乏之类的话。前面提到的自我提升的目的也是为了让自己有更多选择。

**富余心态**：

- **定义**：富余心态是指一个人在多个选择中感到满足和舒适的心态。
- **重要性**：富余是吸引力的源泉，高价值的行为背后都展现着富余。

**案例分析**：

想象一下，你班上有一个存在感很低的书呆子，还有一个受人欢迎的体育委员。书呆子试图与体育委员交朋友，场景会是怎样的呢？

- **书呆子**：存在感低，朋友匮乏，会非常看重每一个朋友，迫切希望有更多朋友。
- **体育委员**：受人欢迎，朋友众多，对朋友的增减并不敏感。

**互动分析**：

- **书呆子**：因为渴望与体育委员交朋友，会表现得更加主动，需求感强，得失心重，小心翼翼，非常在意对方的看法。
- **体育委员**：因为已有众多朋友，对书呆子的加入可能并不在意，互动随意，行为高冷，不在乎对方的看法。

**结论**：

当一个人达到富余心态，就会自然地表现出高价值行为，从而吸引他人。相反，匮乏心态会导致低价值行为，如秒回信息、吃醋、黏人等。

**课程总结**：

本节课中，我们学习了富余心态的重要性及其对吸引力的作用。希望大家都努力摆脱匮乏，进入富余心态，从而提升自己的吸引力。

---

**富余心态公式**：

\[ \text{富余心态} = \frac{\text{选择数量}}{\text{需求程度}} \]
