# 《四横四纵2.0》：四横四纵2.0-2.框架

在四横中的第二课，我们将学习框架。在Rolo的《The rational male》一书中，特别强调了框架的重要性，这是一个非常抽象但关键的概念。

框架的定义，引用朱少军老师在群里的一句话：“框架是价值交换的规则。”这里有两个核心点需要强调。

**核心概念**：
- **框架**：价值交换的规则，即人与人之间社交的规则。

### 框架的三个要点

1. **分而治之**：框架的规则会因人而异，根据对方的品质和与我们的关系亲疏而变化。
2. **清晰一致**：标准明确，不会因对方的不喜欢而轻易改变。
3. **律己恕人**：框架规则用于约束自己，而非约束他人。

### 三种框架

- **Beta框架**：无条件付出，然后道德绑架索取。
- **Omega框架**：让对方先付出，然后控制欺骗索取。
- **Alpha框架**：先付出，不求回报，关注对方心智成熟和人格发展。

### Alpha框架的七大特质

- **无私**：以对方的感受和利益为出发点。
- **智慧**：独立思考，不相信承诺。
- **潇洒**：敢爱敢恨，敢要肯给。

### 筛选

- **核心**：看到对方的部分价值，产生好奇心，深入了解。
- **误区**：过早确定关系，让对方失去争取空间。
- **流程**：问对方做过哪些选择，为什么做这个选择，以及具体例子。

### 默认前提

- **高级筛选方式**：在提问时默认对方有某个特质，然后验证。

本节课中，我们学习了框架的概念、三种框架的特点以及筛选的技巧。希望这些知识能帮助你在社交中更好地运用框架，建立良好的人际关系。
