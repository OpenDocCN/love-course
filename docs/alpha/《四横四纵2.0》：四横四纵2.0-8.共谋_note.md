# 《四横四纵2.0》：四横四纵2.0-8.共谋_note

在本节课中，我们将要学习“四纵”体系中的最后一纵——**共谋**。掌握共谋后，您将对“四横四纵”的所有技术点有一个完整的理解。共谋的核心在于将内卷转化为共赢，将价值榨取转变为价值创造。

上一节我们介绍了“平衡”，本节中我们来看看如何基于平衡更进一步，实现双方共同受益的合作关系。

## 概述：从内卷到共赢

共谋的核心要义是：**把内卷变成共赢，把榨取价值变成创造价值**。这建立在“平衡”的基础上，但更进一步。在“平衡”中，我们强调避免相互剥削，保持社交平衡。而“共谋”则要求双方**一起创造价值，一起去赚取外界的资源**，使双方都变得更好。

这与许多传统社交流派（如PoA）有本质区别。其他流派可能教导如何榨取异性资源、如何控制对方，这本质上是内卷，最终可能导致关系破裂和报复。我们的Alpha体系核心理念是合作与共赢。

## 共谋的核心概念

共谋意味着双方形成一个利益联合体，将矛盾统一向外转移。其核心公式可以概括为：

**内部受益（吹空调） + 外部受损（吹空调外机） = 共谋关系**

这个比喻很好理解：在房间里吹空调很舒服（受益），但空调外机的热风会让外面的人感到不适（受损）。在社交中，共谋就是让你和你的伙伴（“自己人”）一起“吹空调”，共同受益，而让关系之外的第三方或外部挑战来承担“空调外机”的角色。

世界上所有事情都有人受益、有人受损。绝对的自由或独立是不存在的，因为一方的自由往往需要另一方付出代价。宏观上，西方国家的福利建立在第三世界国家的劳动之上；微观上，一段健康的关系也需避免一方长期“吹空调外机”（即单方面付出或承受痛苦）。

## 如何建立共谋关系

理解了核心概念后，我们来看看如何在实际社交中建立共谋关系。以下是两个核心的方法论。

### 方法论一：寻找共同点

建立共谋的第一步是找到双方的共同点，这能快速拉近关系，形成“我们 vs. 外界”的阵营。

以下是具体可操作的两种方式：
*   **寻找共同讨厌的事物**：一起吐槽某个领导、某个现象或某个不喜欢的公众人物。这种共同的“敌人”能迅速增强你们之间的凝聚力。
*   **确立共同喜欢的目标**：一起完成某个任务或享受某个活动，例如：
    *   一起玩密室逃脱、打游戏（如王者荣耀）。
    *   一起爬山、旅游、探险。
    *   共同喜欢某个明星、作品或理念。

无论是共同讨厌还是共同喜欢，本质都是在划分“自己人”和“外人”，从而创造内部团结。这就是为什么有时外部阻力（如反对早恋的父母）反而会让情侣关系更紧密。

### 方法论二：合理转移矛盾

当两人之间出现矛盾时，聪明的做法不是相互指责，而是将责任或问题归因于第三方。这借鉴了“复格”中的思想。

具体做法是：
*   **将矛盾源头引向外部**：例如，感情出现问题，可以归因于“工作压力太大”、“原生家庭的影响”或“某个外界的诱惑”。
*   **核心话术**：强调“**我们是一体的**，问题来自外界”。这避免了内部对立，巩固了共谋关系。

例如，一方犯错后解释：“不是你的问题，也不是我的问题，是[某个外部因素]影响了我们。” 这样就将双方置于同一战线，共同面对外部“敌人”。

## 不同流派的共谋模式

共谋也存在不同的层次和模式，理解它们有助于您识别并实践更健康的Alpha共谋。

*   **Alpha共谋 🤝**：像**化学反应**，双方为了追求更棒的体验而合作。过程中互相鼓励、纠正、共同进步。在达成目标（如通关游戏）后一起庆祝、回忆并展望未来。这种共谋能自然催生好感与思念。
*   **Beta共谋 🫂**：在**世俗规则内抱团取暖**。例如，一方说“男人就该宠女人”，另一方附和“对，不宠就是渣男”。这种共谋基于社会规训，而非真正的共同创造。
*   **Omega共谋 👜**：一方将另一方视为**跟班或资源提供者**。例如，逛街时必须由男方拎包买单。这本质是一方榨取另一方，是内卷而非共赢。

我们的体系倡导的是Alpha共谋。

## 总结与体系回顾

本节课中我们一起学习了“共谋”，它是实现长期健康关系的关键。现在，让我们回顾一下完整的“四横四纵”体系。

我们的Alpha体系可以用四个词概括：**真实、无限、平衡、自然**。
*   **真实**：依据客观规律行事，而非主观臆断或世俗价值观。
*   **无限**：不设社交边界，通过持续共谋提升体验，而非只盯着某个特定目标（如发生关系）。
*   **平衡**：不剥削也不被驯化，以平等姿态追求共赢。
*   **自然**：让关系通过“四横四纵”的循环迭代自然推进，符合常理。

**“四横”是一套完整的社交动作循环**：
宣告（表达意图）→ 框架（筛选对方）→ 共情（理解对方）→ 反馈（给予回应），然后进入新的宣告。

**“四纵”是贯穿始终的思维内核**：
*   **复格**：为行为找到合理动机。
*   **市值**：被动展示价值。
*   **平衡**：保持付出与索取的均衡。
*   **共谋**：创造共同目标与美好回忆。

**“四纵”思维应贯穿于“四横”的每一个动作中**。在执行每一步时，都思考：我能否运用复格？是否展示了市值？是否保持了平衡？能否创造共谋？

通过“四横”的不断循环，并让“四纵”的思维贯穿社交始终，这套体系便能帮助你构建真实、共赢且自然推进的深度关系。
