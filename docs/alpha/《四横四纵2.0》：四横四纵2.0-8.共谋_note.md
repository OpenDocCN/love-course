# 《四横四纵2.0》：四横四纵2.0-8.共谋

## 概述
在本节课中，我们将学习四横四纵体系中的“共谋”概念，探讨如何将内卷转化为共赢，将价值榨取转变为价值创造。

## 共谋的核心
共谋的核心是将内卷转化为共赢，将价值榨取转变为价值创造。它基于平衡，更进一步。

### 共谋与内卷的区别
与其他流派不同，Alpha自然流体系不主张榨取异性的资源，而是与对方合作扩大市场，共同创造价值。

### 共谋的实践
共谋的实践包括：
1. **共同目标**：设定共同目标，共同创造价值。
2. **共同体验**：共同经历美好体验，如共同游戏、旅行等。
3. **责任分担**：在关系中合理分担责任，避免单方面剥削。

## 共谋的技巧
### 观察共同点
1. **共同喜好**：找到共同感兴趣的事物，如共同喜欢的明星、游戏等。
2. **共同目标**：设定共同目标，如共同完成某项任务。

### 合理甩锅
在出现矛盾时，将责任推给第三方，避免影响关系。

## 共谋的类型
### 阿尔法共谋
追求更棒的体验，共同克服挑战。

### 贝塔共谋
在世俗规则中互相支持。

### 奥米加共谋
一方将另一方视为跟班，进行榨取。

## 总结
本节课中，我们学习了共谋的概念、实践技巧和类型。通过共谋，我们可以将内卷转化为共赢，实现价值创造。
