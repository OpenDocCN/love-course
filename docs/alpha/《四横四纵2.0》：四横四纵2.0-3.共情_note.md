# 《四横四纵2.0》：四横四纵2.0-3.共情

## 概述
在本节课中，我们将学习四横四纵2.0模型中的第三横——共情。共情是理解他人行为动机、情绪感受、真实意图和内在品质的关键能力。

## 共情的核心概念
共情是指站在对方的立场，完全带入到对方的视角，以理解对方的行为动机、情绪感受、真实意图和内在品质。

### 共情的误区
1. **主观臆断**：站在自己的视角而非对方的视角去思考。
2. **共情等于同情**：错误地将共情视为满足对方的需求。
3. **共情等于满足对方**：错误地将共情视为直接满足对方的需求。

## 思考对方的行为动机
1. **过往经历**：人的行为和目的与其过往经验和经历密切相关。
2. **假设无辜**：在对方侵犯你利益或做出与你价值观相反的动作时，先假设对方是无辜的。

## 思考对方的情绪感受
1. **情绪表达**：人们用情绪化的方式表达自己，以引起他人的重视。
2. **情绪化表达**：在对方情绪化时，不要说教，而是理解对方的情绪。

## 思考对方的真实意图
1. **真实意图**：理解对方内心真正希望达成的目的。
2. **例子**：通过分析对方的行为和话语，理解其真实意图。

## 思考对方的内在品质
1. **内在品质**：评估对方的内在品质，以指导下一步的反馈。
2. **贝塔特质**：自卑、懦弱、虚荣。

## 需求戳穿
1. **需求戳穿**：当对方的言行不一致时，引导对方说出真实需求。
2. **例子**：在吵架时，引导对方说出真实原因。

## 总结
本节课中，我们学习了共情的重要性以及如何进行共情。共情是理解他人、建立良好关系的关键能力。
