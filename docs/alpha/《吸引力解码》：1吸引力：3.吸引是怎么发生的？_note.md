# 《吸引力解码》：1. 吸引是怎么发生的？

## 概述
在本节课中，我们将探讨吸引力的发生机制，从大脑决策过程到基因影响，以及黑暗三角人格等概念。

## 吸引力的发生机制
以下是吸引力的发生机制：

### 情绪决定
**公式**： 情绪 > 理性

人在大多数情况下，不是用理性来做出决定，而是先用情绪做决定，之后再用逻辑来合理化这个决定。

### 爬蟲腦与理性腦
**公式**： 爬蟲腦（本能）> 理性腦（逻辑）

动物的心理性，在择偶问题上，几乎总是我们的爬蟲腦先做出反应，比如男人看到一個性感的女人，會不自覺地產生生理反應。

### 黑暗三角人格
**公式**： 自戀 + 權謀 + 精神變態 = 短期吸引力

黑暗三角人格的三个点分别是自戀、權謀、精神變態，这些特質的人通常在性伴侶數量上比一般人多。

## 总结
吸引是感性的，是情绪的，是不自主的，而不是一种理性逻辑的行为。只要你展现出高价值的特质，女人就会被你吸引。而一个对你没感觉的女人，就算送她上下班，請她吃大餐，感人的告白都不会让你对她产生吸引，女人是吸引来的，不是追来的。
