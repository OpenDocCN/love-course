在本节课中，我们将要学习《四横四纵2.0》课程的绪论部分，了解这门课程的定位、核心价值以及学习方法。

---

对于恋爱社交行业，最常听到的诟病是：社交还需要学习吗？谈恋爱还要你教吗？我听过无数次这样的话。现在的我不会再像以前那样，苦苦劝解一个人了。

大部分人没有深刻思考过恋爱与社交，而是草率地为其下了定义，从不追问对错。当有人指出他走错了路，他会找出一万个理由证明自己是对的，你是错的。**爱是一场修行，一场极其痛苦的修行。**

就像一个人走错了路，回头却看到锋利的经济和利刃。他若想重回正轨，就必须穿越这些荆棘与利刃，这过程非常痛苦。因此，大多数人选择在错误的道路上继续走下去，即使知道这是错的。

他知道这是一条不归路，也不可能回头。他甚至会自我安慰说这是一条正确的道路。即使有人提醒，他也会声嘶力竭地怒吼“关你屁事”。他会想：回头多苦，面对痛苦多苦，面对失败多苦，面对身后那锋利的经济和利刃多苦。

想一想就觉得可怕，还是继续往前走吧。他仿佛英雄般孤傲地走上了没有光明的“正途”，前方是恶魔狰狞的笑声。但我相信，总有一天他会狼狈地、坚定地、义无反顾地回头，冲破所有阻碍。

他会走上正途。只是那时，他的青春可能已不在，岁月也已残破。但这都还好。最痛苦的事情莫过于：一个人在青春流逝、岁月雕琢、垂垂老矣时，还在感叹命运不公；在自己死后，墓碑上写满了“请你们爱我”，却从未有人去祭奠他。

《四横四纵2.0》这套课程，我将其定位为一次**让你打开新世界大门的钥匙**。此时此刻录制这套课程，我非常激动。

这套课程从去年此时我刚萌生“四横四纵”理念开始，到去年8月正式在线下落地实践，再到后来推出1.0版本，过程颇为曲折。中间需要不断摸索、调整，不断解构与重建这个体系。现在，它终于能以2.0版本与大家见面。

在绪论部分，我们需要回答三个问题。

---

### 第一个问题：四横四纵是什么？

**四横四纵**是一个比较完整、有体系、具备闭环逻辑的一套社交行为框架。可以这样理解：它不同于联盟的其他课程（例如朱老师的Alpha社交更侧重实战细节）。**四横四纵**课程更加专注于**社交的底层逻辑**和**更普适的应用场景**。

当然，将其应用于情感领域毫无问题，甚至可以说是“降维打击”。它拥有更广阔的应用前景。我相信你在学习过程中也会有同感：这套方法不只针对异性，对同性、领导、父母、孩子，都是**一模一样的、可操作的流程**。

---

上一节我们介绍了课程的核心定义，本节中我们来看看它与之前版本的区别。

### 第二个问题：它与1.0版本有何不同？

最直接的不同是它对知识体系进行了大幅度更新。1.0版本实际上是一套比较松散且稍显复杂的体系。坦白说，在实际执行中会遇到很多问题。

这些问题在我们1.0版本的基础上，通过社群内的交流，已经得到了较好的完善。同时，2.0版本也弥补了1.0版本的一个致命缺陷：1.0版本只注重**动作**，而忽略了动作背后关于**人性**与**爱**的更深层思考。

2.0版本引入了这些**高阶认知作为支撑**。有了这些信念和认知的支撑，再去理解那些具体动作时，你会有一种全然**豁然开朗**的感觉。

了解了课程的进化后，接下来是至关重要的部分：如何学习。

### 第三个问题：我该如何学习这套课程？

学习本课程，请牢记以下三个关键词：

以下是三个核心学习原则：

1.  **接纳**
    无论你之前是否学过1.0版本或其他课程，第一步都是**接纳**。课程内容可能与你旧有的三观和思维体系有冲突。第一步请你学会接纳，完全信任Andy，将旧有认知暂且放下，先去接纳这个崭新的体系。

2.  **自省**
    学习与应用过程中，你一定会产生许多疑问。因此，你必须学会**自省**：我到底哪里没理解透？哪里没执行好？如果自我反省比较困难，可以在我们的社群中交流。某种意义上，这种**社群内的反思比单纯听课更重要**。

3.  **紧跟**
    学习过程中，务必**完整学习**。请认真听完每一堂课、每一分钟的知识点，认真理解，不要跳步。相信完整学完本课程后，你至少能正式迈入我们**Alpha社交的门槛**之上。
