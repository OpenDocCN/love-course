# 《四横四纵2.0》：四横四纵2.0-6.示值_note

在本节课中，我们将要学习“四横四纵”体系中的“示值”概念。示值，即展示价值，是社交互动中一项重要的意识与技术。理解并掌握它，能让你在人际交往中更自然、更有效地传递自身价值。

经历了“四横”和“复格”的学习，相信各位对“四横四纵”体系有了更深入的理解。前面的五个技能点（四横与复格）是较为具体、可立刻实操的技术点。而最后三个节点——示值、过谋、平衡，则更多是一种社交意识。你需要将这些概念内化于心，在做某些动作时，保持展示价值、建立共谋、维护社交平衡的意识。

## 什么是示值？🤔

示值的核心就是展示价值。其中最重要的、能让你在操作中用起来的技术点，可以概括为两个字：**被动**。

我们需要区分“被动”与“主动”。所谓被动地展示价值，是指你因为一些不得已的原因，被迫去展示价值。这是一种非常不情愿、不得已而为之的状态，而非你主动想去“装逼”。例如，因为对方提问，你不得不回答，从而展示出你的知识或能力。关键在于营造出一种“你是被迫展示”而非“主动炫耀”的感觉。

那么，什么是主动装逼呢？主动装逼与被动示值的区别在于：主动装逼的目的是为了在对方心中刷取认同感，让对方来认同你。例如，直接向对方说“我有一台大G，你们女人不喜欢钱吗？”或展示你的豪车钥匙。这些行为都是在主动展示价值，目的是获得他人认同，这个过程就是主动装逼。

主动装逼是非常暴露需求感且不尊重他人的表现。而被动展示则不同，它是基于对方的需求。例如，你看到对方需要帮助，于是你说：“我正好懂这个，可以帮到你。”这就是一种自然的被动展示。因为对方有需求，让你“不得不”展示自己的价值，目的是提供帮助或解答疑问，而非寻求认同。

## 如何被动展示价值？💡

理解概念后，我们来看看具体如何操作。关键在于将你的价值“不经意”地嵌入到对话或情境中，而非直接陈述。

以下是两种展示方式的对比：

*   **错误示范（主动装逼）**：直接问对方：“你开过大G吗？”或“我有一辆大G。”从共情角度看，对方能立刻感觉到你在寻求认同和炫耀。
*   **正确示范（被动示值）**：在一个群里提问：“请教一下大家，大G的首保在哪里做？纯新手，刚提的车，一会儿有重要的接送任务，哪位懂车的帮帮忙？”这样，你的**重点在提问（首保）**，但**不经意间展示**了你拥有（且是新买的）大G，以及你可能拥有的财富与社会地位。价值是隐含在求助行为中的。

再举一个例子：假设你邀约一个新认识的对象，且预判对方很可能会拒绝。这时你可以说：“你周末有空吗？我带你去最好吃的一家日料店，里面的三文鱼刺身非常新鲜，全是日本空运的。”你知道对方可能拒绝，但借此机会**展示**了你知道高级餐厅、拥有一定品味和消费能力的信息。展示的核心依然是**被动**——以邀约为载体，而非直接炫耀。

## 什么是“价值”？💰

理解了“如何展示”，我们再来明确“展示什么”。价值是一个相对概念，它相对于需求而存在。对方有需求，你才有价值；对方没有需求，你再厉害对Ta而言也可能价值不大。

价值因人而异，但存在一些普世性的价值，即大多数人都会欣赏和喜欢的特质。一旦你有了“示值”的意识，在日常生活中就可以有意识地提升以下四个方面：

1.  **外在形象**：这不仅是五官长相，更包括气质、穿衣风格、打扮、谈吐等整体感觉。拥有独特、自信的风格，能让你迅速吸引他人注意。
2.  **社会地位**：这不单指财富，更指一种影响力和受尊重程度。例如，你带朋友去常去的餐厅，服务员和厨师长都认识并热情招呼你，这就在无形中展示了你的社会地位。
3.  **个性品质**：即我们之前课程中提到的Alpha七大品质：**高、质、勇、爱、前、绿、霄**（高度自我认同、有生活品质、勇敢、有爱心、有前瞻性、绿色社交、潇洒）。通过你的行为和故事来展示这些特质，能产生广泛的吸引力。
4.  **共情与赞美能力**：能够洞悉他人的努力，具有共情能力，并主动、真诚地表达喜欢与认可（即“宣告”）。这满足了人们渴望被深刻理解和认同的深层需求，是非常稀缺且有价值的能力。

实际上，当你在严格执行“四横”时，已经在进行“示值”。例如：
*   **框架**中展示你的筛选标准，表明你是有原则的人。
*   **宣告**中表达真诚赞美，展示了你的共情力和洞察力。
*   **共情**中理解对方真实想法，展示了你的理解能力。

人们不仅有表层的需求（如喜欢美貌），更有深层的需求（如被理解、被认同）。满足深层需求的价值更难被替代，因为大多数人并不具备这种能力。

## 示值的实用技巧 🛠️

上一节我们明确了价值的核心内涵，本节中我们来看看两个辅助你更好“示值”的实用技巧。

以下是两个有效的辅助手段：

1.  **通过故事传递价值**：与其直接说“我很勇敢”、“我很智慧”，不如将这些品质融入你的过往经历中。例如，想展示勇敢，可以讲述你曾如何保护他人；想展示领导力与慷慨，可以讲述你大学时如何组建团队并帮助队员。将多条价值线编织在一个好故事里，是更高级、更自然的展示方式。
2.  **利用“浅沟通”传递价值**：你的非语言信号也在持续传递价值。例如：
    *   **眼神接触**：能够坦然直视对方眼睛，传递出坦诚与坚定。闪烁的眼神则可能显得心虚。
    *   **肢体语言**：打开肢体（如打开手臂、舒展双腿），占据更大空间，能传递出轻松与自信。弯腰驼背、缩手缩脚则会显得拘谨或猥琐。

这些技巧只是辅助。只要吃透示值的核心要义——**被动**与**价值**，再结合“四横”中的具体技术，并在实战中不断理解和运用，你就能深刻掌握“示值”。

## 总结 📝

本节课中，我们一起学习了“示值”这一核心社交意识。

*   **示值**即展示价值，其核心在于**被动**展示，而非主动炫耀。
*   **价值**相对于需求而存在，包括外在形象、社会地位、个性品质、共情力等普世价值，也包含通过“四横”技术（框架、宣告、共情等）所展示的深层价值。
*   可以通过**讲故事**和**优化浅沟通**（如眼神、肢体语言）等技巧来辅助价值的自然传递。

掌握“示值”的概念后，你再回头执行“四横四纵”体系中的其他技术，会对整个体系有更深刻、更融会贯通的理解。记住，价值展示的最高境界，是“随风潜入夜，润物细无声”。
