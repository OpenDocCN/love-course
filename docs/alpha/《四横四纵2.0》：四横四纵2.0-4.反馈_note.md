# 《四横四纵2.0》：四横四纵2.0-4.反馈

## 概述
在本节课中，我们将学习四横四纵2.0体系中的反馈技巧，包括共情、回应、拒绝和积极消极处理。

## 反馈的核心定义
在共情出对方的行为动机和真实需求之后给予对方一个回应。

## 反馈的要点
1. **及时性**：及时给予反馈，避免让对方感到被忽视。
2. **积极奖励，消极惩罚**：根据对方的反应给予相应的奖励或惩罚。

## 质疑对方的需求
1. **质疑真诚性**：判断对方的需求是否真诚。
2. **质疑合理性**：判断对方的需求是否合理。

## 满足对方的需求
1. **表达喜欢和认可**：通过夸奖、赞美等方式表达对对方的喜欢和认可。
2. **给予安慰**：在对方遇到困难时给予安慰和支持。

## 拒绝对方的需求
1. **指出复格缺失**：指出对方没有达到你的标准。
2. **合理推诿**：给出合理的理由来拒绝对方。

## 积极消极处理
1. **自省**：反思自己的错误。
2. **允许**：允许对方按照自己的方式来做事情。
3. **表达歉意**：表达你的歉意，并尝试弥补对方。

## 担责致歉
1. **承担责任**：为你的错误承担责任。
2. **表达歉意**：真诚地表达你的歉意。
3. **弥补对方**：通过行动弥补对方的损失。

## 总结
本节课中，我们学习了四横四纵2.0体系中的反馈技巧，包括共情、回应、拒绝和积极消极处理。通过这些技巧，我们可以更好地与他人沟通，建立良好的人际关系。
