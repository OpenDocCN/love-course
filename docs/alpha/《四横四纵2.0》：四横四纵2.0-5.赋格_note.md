# 《四横四纵2.0》：四横四纵2.0-5.赋格

## 概述
在本节课中，我们将学习四纵中的第一纵——复格。通过学习四行，相信大家对Alpha的自然流已有全面了解。接下来，我们将学习四纵，这将极大地帮助大家理解四行。

## 复格的核心
复格的核心是合理化自己的一切行为。或者，可以借鉴共情中的方法，为你的欲望、目的、需求补充一个行为动机。这就是复格的核心要义，即说出你不得不这么做的原因。

## 复格的社交动作
复格本身没有明确的社交动作，它需要结合四行的每一步进行。因此，四纵的核心贯穿了四行的每一步，你应尽量做到所有四纵。

## 复格的两个基础点
1. **归因**：将事情的原因分类或确定。
2. **合理化**：对可能突破世俗价值观的行为进行合理化，让对方更容易接受。

### 归因
一个事情的发生必然有背后的原因，可能是因为你、我或他人。归因的核心是将事情背后的原因归咎于你我或他人。

#### 把原因归咎于他人
例如，父母常说的“我之所以XXXX都是为了你好，所以我都是为了你好”。这句话背后的含义是，不管客观上对你好不好，但出发点都是好的。

#### 把原因归咎于自己
例如，在社交开场时，你可以被动聊天。如果你主动，你会说“在干嘛？”但如果你学会了复格，你可以说“看你好久没有发朋友圈了，你最近没事吧”，这就是主动和被动的区别。

#### 把原因归咎于他人
例如，在表达失格时，你可以说“我之所以对你很失望，都是因为你的错，都是因为你展现出了什么品格，或者说你做了一个什么样的事情”。

### 把原因甩给第三方
例如，曹操在带兵打仗时，发现粮食不够，他找了一个外在的靶子，告诉士兵，饿肚子不是因为他，也不是因为你们，而是因为有第三方在操作。

### 把原因甩给第三方
例如，一个男人出轨，他会说“我出轨，你以为我想吗，都是他勾引我”。

### 逻辑合理化
在提需求时，你可以通过逻辑合理化来让对方更好地接受。

#### 例子
在搭讪时，你可以说“我正好要去买杯咖啡，就在前面那个便利店，你要不要跟我一块去，我请你喝一杯”。

### 总结
本节课中，我们学习了复格的核心、两个基础点以及如何应用。希望这些内容能帮助大家更好地理解复格。
