# 《四横四纵2.0》课程-第五讲：赋格 🧩

在本节课中，我们将要学习“四纵”体系中的第一纵——**赋格**。经过对“四横”的学习，相信各位对Alpha自然流已有较全面的了解。在此基础上学习“四纵”，将能极大地深化和巩固对“四横”的理解与应用。

## 概述：什么是赋格？

赋格的核心要义是：**合理化自己的一切行为**。具体而言，就是为你自身的欲望、目的或需求，补充一个外在的、易于被他人接受的行为动机。它并非一个独立的社交动作，而是需要贯穿于“四横”（宣言、共谋、市值、失格）的每一步之中。因此，“四纵”是渗透在整个互动流程中的技术。

理想情况下，在每一次表达中，都应尽可能结合赋格、共谋、市值与平衡等技巧。对于赋格，我们需掌握两个基础技术点：**归因**与**逻辑合理化**。

## 第一部分：归因——为行为寻找“责任人”

归因，即对事情发生的原因进行分类和确定。任何事情的发生都有其原因，可能在于“你”、“我”或“第三方”。赋格中的归因，就是巧妙地将行为背后的原因，归结到这三者之一。

### 归因于“你”（对方）

将行为动机归结为对方，意味着“我这么做，是因为你”。

**以下是几个典型场景：**

*   **经典话术**：如同父母常说“我这么做都是为了你好”。这句话的潜台词是：我的出发点是为了让你更好，而非满足私欲。
*   **提出需求**：例如，希望对方送你一件衣服。可以这样说：“你这件衬衫真好看，是XX牌的吧？我也想要一件。这样我们就能穿情侣款了，你带我出去也更有面子。”——将“我想要”转化为“为了我们共同的形象”。
*   **规范行为**：例如，希望对方盖上马桶盖。可以这样说：“听说不盖马桶盖，风水（财运）会流走。我提醒你是为了你好。”——将要求包装成对对方的关怀。
*   **主动发起互动**：避免直接问“在干嘛？”。可以改用：“看你很久没发朋友圈了，有点担心你，最近没事吧？”——将“我想找你聊天”归因为“你的行为引起了我的关切”。
*   **宣言开场**：如搭讪时可以说：“我本来没想打扰别人，但你刚才的样子太迷人了，我实在忍不住过来想认识你。”——将主动搭讪归因为“你的魅力吸引了我”。
*   **表达失格**：当因对方过错而失望时，可以说：“我这么难过/失望，完全是因为你做出了XX行为（或展现了XX品质），这违背了我对你的期待。”——明确将负面情绪归咎于对方的行为。

**过渡**：上一部分我们探讨了如何将原因归结于对方。接下来，我们看看另一种策略：将原因推向“第三方”。

### 归因于“第三方”或外部因素

将原因归结于你、我之外的他人或不可抗力，这常与“共谋”结合，制造出“我们vs外界”的阵营感。

**以下是几个应用实例：**

*   **历史典故**：曹操军中缺粮，他将士兵的不满归因于“管粮官贪污”，并将其斩首示众，从而平息了众怒。这是寻找“替罪羊”来转移矛盾的经典案例。
*   **为过错开脱**：例如，为出轨行为辩解：“都是TA一直勾引我/拼命追求我”、“我之所以会这样，是因为在TA身上找到了我们当初热恋的感觉，这都怪我们现在的关系变得平淡了”。——将责任甩给第三者或关系本身的问题。
*   **解释控制行为**：当被抱怨“太黏人”时，可以解释：“我也不想这样，是我之前的几任男友/女友因为我管得松而出轨，给我留下了心理阴影。我太害怕失去了，你能理解我吗？”——用过往创伤作为当前行为的理由。
*   **冷暴力与分手借口**：使用“我工作太忙了”、“我爸妈不同意我们在一起”等作为疏远或分手的理由。这实质上是将结束关系的责任推给外部因素，让自己避免扮演“坏人”角色。

**识别与应对**：作为Alpha，自身应慎用此技，但需能识别他人是否在“甩锅”。当对方使用诸如“我爸妈不同意”这类借口时，可以运用“四横”技巧直接戳穿：“你不用找借口，我知道你就是不喜欢我又不想当坏人。你连自己感情都做不了主，何必装深情？”

## 第二部分：逻辑合理化——让需求顺理成章

逻辑合理化通常用于提出可能让对方产生顾虑的需求时。其核心是**用逻辑框架去安抚和合理化对方的情绪**，让要求显得更自然、更易接受。

**以下是关键应用场景：**

*   **降低邀约压力**：搭讪后即时邀约可能显得唐突。可以说：“我正好要去前面便利店买杯咖啡，一起过去吧，我请你。”——用“顺路”、“很近”、“小事一桩”来合理化邀约行为。
*   **假性时间限制**：想开启聊天又怕对方觉得耗时，可以说：“我待会儿还得去找朋友，不过走之前，有件事想问问你……”——用“不会占用你太久”来打消对方顾虑。
*   **关系推进（核心应用）**：在两性关系推进中，尤其需要合理化亲密行为，以减轻对方（特别是女性）的道德包袱或焦虑感。核心原则是：**将亲密互动包装成“自然的意外”、“命运的安排”或“氛围使然”**，而非赤裸裸的性需求。
    *   **错误示范**：直接对有夫之妇表明“我想和你上床”，会将对方置于“明知故犯”的道德困境。
    *   **正确思路**：避免强调可能引发道德焦虑的身份（如“有伴侣”），以“朋友”名义相处。让一切发生在“朋友”的框架内，亲密行为则是“情难自禁的意外”。
    *   **邀约示例**：邀请对方到家中，可以说：“听说你也喜欢精酿啤酒？我家里收藏了不少，晚上可以一起尝尝，顺便做点吃的。”——将“来我家”合理化为基于共同爱好的朋友聚会。

**总结**：逻辑合理化的精髓在于，为你的社交动作设计一个合乎常理、易于被接受的“表面理由”，从而绕过对方心理上的直接防御或抵触。

## 课程总结 🎯

本节课我们一起学习了“四纵”中的第一纵——**赋格**。

我们首先明确了赋格的核心是**为自己的行为提供合理化动机**，它需要贯穿于所有社交互动中。接着，我们深入探讨了赋格的两大技术支柱：
1.  **归因**：学习如何将行为原因巧妙地归结于**对方**、**第三方**或**外部因素**，从而转移焦点、减轻自身责任或制造共谋。
2.  **逻辑合理化**：学习如何为可能引发顾虑的需求（尤其是关系推进），构建一个**合乎逻辑、易于接受的表面理由**，以安抚对方情绪，让互动顺利进行。

掌握赋格，能让你在社交中显得更自然、更有说服力，同时也能帮助你更敏锐地识别他人是否在使用这些技巧。请将这些原则融入到你对“四横”的实践中，你的社交直觉与掌控力将获得显著提升。
