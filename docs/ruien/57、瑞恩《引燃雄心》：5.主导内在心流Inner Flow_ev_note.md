# 课程57：主导内在心流 🧠

在本节课中，我们将学习如何提高社交，尤其是搭讪时的流畅度。核心在于理解并打通你的“内在社交心流”。我们将探讨什么是社交流畅度，分析阻碍心流的关键因素，并提供具体的方法来帮助你进入并保持流畅的状态。

## 什么是社交流畅度？ 🤔

上一节我们介绍了课程主题，本节中我们来看看什么是社交流畅度。

很多学员观看我的搭讪约会实战视频时，发现我与女性的每一次交流都非常自然流畅。他们常问我这一点是如何做到的。

我表面上展现出的交流流畅度，完全源于我内心的心流与流畅度。这种内心的流畅来源于我当下的状态。

这让许多人感到烦恼，因为他们都在拼命寻找这种状态，试图拉高自己。恰恰是这种“寻找状态”的努力，导致与女性对话时毫无流畅度，表现为吞吞吐吐、犹豫不决或脑子一片空白。总之，没有丝毫的社交流畅与舒适平和。

**心流的关键是你的状态**。如果你不在状态，进入状态的最好方法就是：在状态之外感觉良好。

要记住这句话：**一旦你在状态以外感觉良好，你就已经进入状态了**。

## 阻碍心流的根源：感觉不好 🔄

是什么让你的心流不畅？是你感觉不好。一直感觉不好，你就会陷入恶性循环，持续卡在感觉不好的坏状态里。

真实状态，是你的现实与你想要的感受完全相匹配。这个状态指的不是高能量或低能量，也不是好状态或坏状态，指的就是你当下真实的状态。因此，它可以是好的或坏的，也可以是高或低的。

当你能量低落、状态低下，但自身感觉依然良好时，这就变成了你新的现实。

简单来说，只要你当下的状态和你脑子里想要的感受是相同的，你就会拥有状态和流动。

例如，我现在出门，实际状态一般，不高也不低。我对此刻自己状态的感受还可以。我脑子里当下想要的状态就是这样。我完全接受，那么我现在就在状态里。我和任何人讲话都是流畅、顺畅的，不需要高能量，也不需要高点的状态。我们其实只需要找到当下的平衡。

在一整天里，你的能量都会上下波动。不管你的状态在哪儿，找到那个平衡。

## 如何进入心流：接受与匹配 ✅

可能很多人会问：如果我的状态就是不好，比如真实状态就是坏的，那我该怎么样进入心流？

答案还是之前说的：哪怕你当下的状态不是很好，但你应该接受你当下的状态。因为当你的脑子去想好与坏、批判自己的状态时，这个状态就已经不是你想要的了。所以你没有状态，也不可能进入状态。

你要记住一个重点：**当下自己感觉到的状态必须和你脑子里想要的完全相匹配，而不是去批判和抗拒**。那么你就立刻能够进入状态。

任何对自己当下状态不满意的想法，想要去拉高或摆脱这个坏状态，那么你也就瞬间失去了你该有的状态，也就不可能再进入所谓的状态。

## 社交心流核心二：放下抗拒与自我破坏 🧘

社交心流的第二个核心点是：你要放下所有的抗拒，放下所有的自我破坏，放下所有感觉不好的事情。

在本课程之前我提到过，一旦你产生选择性焦虑，全部注意力都在自己身上，你就没有状态。你做的一切、说的一切就都是在追逐所谓的高能量状态。这时，你的注意力是分散的，意图是分散的，投入不稳定，内心焦虑恐慌。你怎么可能有社交流畅？哪来的心流？

从一个好的视角去看，一个好的方法是：你要从内心彻底庆祝你自己没有状态。不要担心进入或者离开状态这件事。你越担心，你越没有状态。不要去想“我现在有没有状态”，最重要的是你能接受现在、接受当下。

你要明白，如果你能量低一点，你也不错。如果你能量高一点，也不代表你更好。你就是需要接受能量它此时此刻的样子。

当下，不要紧紧地抓住你的情绪，尤其是坏情绪。你必须了解，你的情绪是来来去去的，它们在身体里是流通的。情绪的停留非常短暂，不会永久留在内心，因为你的意识无法让某个情绪永久保持活力。

例如，你非常愤怒生气时，这股强大的情绪和力量会一直存留在体内吗？完全不可能。通常它的存活时间很短暂。过一会儿，你的愤怒就逐渐消失了。过一段日子，你可能都不记得哪天愤怒过。

所以，你必须接受情绪是来来去去的，不要去阻止内在情绪的流动。阻止它，反而会让它在身体里停留的时间延长。

大多数人搭讪社交时所经历的社交焦虑时间，比本应该经历的时间要长得多。就是因为他们会长时间专注并抓住当下这种恶劣的情绪，比如拼命试图让自己不焦虑。但实际上，他是在专注和抓住自己焦虑的情绪。

如果你不坚持抓住自己焦虑的情绪，这股情绪的流动自然会在你采取行动时逐渐消失。但现在它在体内停留的时间会更长，因为你在注意它、放大它，把自己的焦虑当成一件大事。所以焦虑会停留更长时间，并且一直在折磨你。你的情绪流动不顺畅，心流和社交流畅度自然也就没有。

情绪流动最大的体现，就是在我们做事情忙的时候、有行动的时候。例如，我可能在外面搭讪2到3个人，按照自然情况，你的焦虑其实会慢慢淡化，你就可以进入社交流畅的舒服自然状态里。

可是，你却无限放大自己的焦虑恐惧和内心投射，把注意力和精力全部放在压制自己的情绪上面。这时，你当然在强化这件事、这个情绪本身，所以你内在的这些情绪也就停止了流动。好比血管里的血液被你强行堵塞了。

这时，那些坏情绪会被无限放大、停留持久，直到吞食你自己行动的意志，彻底摧毁你。你怎么可能有社交流畅度？所以，你必须允许自己的情绪流动起来。

## 实践方法：接受、夸大与释放 🎭

如果你仔细学习了前面几节课，那么你知道人都会陷入自己的陷阱里开始自我破坏，尤其是在社交的时候。如果一开始就受挫，比如被女性拒绝，开始自我怀疑、自责，找各种借口，进入一个自闭的死循环。

最好的处理方式就是：首先接受并拥抱当下，不要做任何抗争。如果你强制性地去对抗，情绪只会越来越坏。你在内耗。所以第一步就是先接受并拥抱当下不好的情绪。也就是说，对不好的情绪产生好情绪。

例如，我现在很难过。好的，我接受。现在对这个难过，微笑，我接受它。

从这儿开始，夸大它。夸大你不好的感受，把它释放出来，表现出来。夸大到一个什么地步呢？就是连你自己都觉得这件事情很好笑，甚至很荒唐。

在这个过程中，你要持续地去行动，去释放这个不好的感受，同时夸大这个感受，直到自己慢慢把自己拉回一个好的情绪中。简单来说，就是从中找到乐趣。

夸大情绪这个操作可以让你进入心流的底层逻辑是：你的内在负面情绪被你及时通过对外表达、对外释放出来。释放后的感受等同于你释怀了。那么你就可以进入心流的这个状态中。

## 应对“头脑空白”：表达现状 🗣️

有些人可能会问：假如我搭讪的时候不焦虑，但我头脑就是一片空白，我也不知道说什么，这是属于什么样的状态？

如果你的脑子里是一片空白，你不知道说什么、做什么，这就是之前我提到过的完全没有微观意图。但是，无论状态是稳定的、不高不低，或是哪怕很低迷，你内心都不要去做斗争。必须先接受这个空白无话可说的状态。

同时，你需要释放这种状态。例如，和女性说：“我过来想跟你说话，但是我不知道具体要跟你说啥，我脑子里就是一片空白。” 你说这个就可以，这就是一个很好的开始。因为你在开始表达这种状态、这种现状。

同时，你接受了这个现状。这就能达到现实和脑子里的想法相匹配，从而成功开始让你进入到你想要的那个状态中。

## 生活实例：从抗拒到接受的转变 📚

我再给你举一个我人生的例子。例如，我小时候很讨厌写作业。我妈妈逼着我写作业时，我会有很大的负面情绪，内心非常抗拒眼前的课本。行为上，我自然就开始找各种理由不去写作业。

你要知道在一开始，这个负面情绪很严肃、很激烈。我一看见课本和铅笔，可能就开始皱眉头。这时，我会和妈妈沟通。比如我开始写两个字，然后我就会抱怨：“哎呀，我累了，我写不动了，我想睡觉。” 我妈说不行。好，又过一会儿我又写了一会，然后我又说：“我想上厕所，我憋不住了。” 我妈可能会说：“你快点。” 我厕所回来之后，我又没写两个字，我又说：“我肚子饿了，我不行了，我要晕过去了。”

这时，我妈都笑了。她会说：“你屁事真多，写个作业都要了你命了。” 我说：“就是要了命了啊。” 可能说着说着，连我自己都笑了，都感觉这个事情非常荒唐。

笑了之后，我还是会继续写作业。但我写着写着发现，其实也没有一开始那么严肃激烈的负面情绪了。这时，内心的抗拒也被成功化解了不少。有还是会有一点点存在（我不喜欢写作业这件事），但最起码内心剩余的负面情绪是我自己可以维持的，能够继续把作业写完。我发现这也不是一件那么不能去做的事情。

## 应用于社交：释放情绪，找到幽默 😄

基于刚才的例子，我们去社交接触女性也是一模一样的道理。

你一开始的抗拒心理和抵触情绪导致你没有状态，没有社交流畅度。同样的方法：你每一次去搭讪一个人，就真实地去释放你的负面情绪。

例如，你跟一个女性去打招呼，你就先跟第一个女性说：“我现在有点难过啊，如果你多看看的视频，可能你会舒服一点。” 这就是夸大负面的情绪。然后你搭讪下一个人，你再说：“你刚有点难过，但是你现在好了一点点了啊。” 然后再下一个，再下一个，直到你的负面情绪被接受以后转化为好的情绪。

其实你发现，幽默是化解一切负面情绪最好的良药。

当你接受当下，无论好还是坏，无论你是否在状态，你只要自己很舒服地做自己，你就可以畅所欲言。甚至你认为现在已经没有什么话是你说不出来的。比如之前告诉几个女人你难过的感受，然后慢慢好转，这就是从抗拒到接受的流畅过程。

你要相信并执行这个过程，你才有可能变得流畅。你在尝试十次里面，可能有两三次或者四五次，你真的会体验到当下就是放松的状态，在女性面前可以畅所欲言。但其他时候，你可能还是会继续抗拒。所以这是一个漫长的修复过程。

比喻就像医院里的医生在帮你治疗一个慢性疾病，一定会有一个疗程和周期，而不是一夜之间就让你彻底完完全全好了。他大概是这么一个逻辑。所以我们必须理解，要给自己一点耐心，给自己一点时间去完成这个慢慢转变的过程。

## 总结 📝

本节课中，我们一起学习了如何主导内在心流以提高社交流畅度。

我们首先定义了社交流畅度源于内心的状态与心流。接着，我们分析了阻碍心流的核心是“感觉不好”以及对状态的批判与抗拒。关键在于**接受当下真实的状态，并让你感受到的状态与你脑子里想要的感受相匹配**。

然后，我们探讨了第二个核心：放下抗拒与自我破坏，允许情绪自然流动，而不是抓住并放大负面情绪。

最后，我们学习了具体的实践方法：通过**接受、夸大并释放负面情绪**来化解内在抗拒，并通过**表达现状**（如“我脑子一片空白”）来启动心流。我们结合生活与社交实例，说明了从抗拒到接受、并从中找到幽默感的转变过程。

记住，这是一个需要耐心和持续实践的修复过程。接受当下的自己，是通往流畅社交的第一步。
