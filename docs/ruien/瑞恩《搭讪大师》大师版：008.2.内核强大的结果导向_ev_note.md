在本节课中，我们将要学习如何通过强大的内在心态，应对搭讪过程中的直接拒绝，并保持状态进行第二次尝试。我们将通过一个真实案例，分析被拒绝后如何调整沟通策略，以及核心的内核心态是如何支撑我们行动的。

上一节课我们介绍了第一次搭讪被迅速拒绝的场景。本节中我们来看看，面对同样的女生和同样的负面情绪，如何进行第二次接触。

## 理解拒绝的本质 🤔

在我的实战经验中，被女生快速拒绝的情况经常发生。这与大多数人的经历相同。区别在于，我拥有十几年的实战经验和持续的自身建设。

从我的视角来看，我完全能理解她为何会有那样的反应。我不会因此过度自责，或认为自己是被人否定、不喜欢或排斥。

这与我们自身的内在有很大关系。如果你发自内心地认为自己是一个优秀、有价值的男人，那么任何女生的拒绝都不会让你真正难过。

我内心100%坚信一件事：**只要任何一个女人给我机会认识和了解我，她一定会喜欢上我**。

这个想法可能被很多人视为自恋。但在我看来，这是一个非常健康和正常的想法。我并非认为所有女人都应该喜欢我。但我相信，只要大部分女人给我机会了解我，因为我知道自己在自身建设、思维逻辑、为人处事、内外在等方面投入了多少时间和精力。

基本上，只要我愿意，我都能给大部分女生带来非常美好的体验。但重点是，我们无法实际控制一个女生当天的情绪、心情，或她如何解读我上前搭讪这件事。

## 第二次搭讪的启动 🚀

在第一次搭讪被拒后，我看到她站在路边等车。我的执着让我决定进行第二次尝试。

以下是第二次上前时的对话核心：

```
我：你刚误会我。
她：咋了？
我：不是你刚才误会我。
```

这是我第二次搭讪。我没有再次表明意图，没有再说“我觉得你好看”之类的话。你们要明白，搭讪不是学习一个话术、模式或流程，然后反复套用在一个人身上。

归根结底，这是一个人与另一个人试图交流的过程。所以，我们讲求的是有效沟通。如果第二次上去，我依然重复之前的意图，而她之前已经拒绝了，这显然是无效的。

因此，我换了一个角度，以“你误会我了”作为开场。这样做的目的是让她听我说话。我需要一个铺垫，来获得她真正的注意力。

在第一次搭讪时，她完全没有给我丝毫注意力。第二次搭讪，我的首要目标是让她看着我，听我表达。

从男人的角度看，我上前时保持了一个中立的面部表情。我在试图与她进行逻辑沟通。当她听到“你误会我了”时，她可能会想“我怎么误会你了？”，这就吸引了她的注意力。

人与人之间的沟通是按照心理路径一步步进行的。没有任何一个女人会在你反复搭讪时，因为一句话就突然喜欢上你或被你吸引。她首先需要愿意听你说话。

我上去时的表达，丝毫没有受到之前被拒的影响。我保持了一个非常中立、正常的交流状态。这不是能装出来的。这源于你的自尊——发自内心地认为她不是拒绝你，她只是误会了，或她今天心情不好。

从她的视角看，她可能误以为我在骚扰她，或我是一个低价值的男性。而我想告诉她，她误判了。通过沟通，我想让她感觉到自己误判了，我其实是一个正常、优秀的男人。

## 第二次对话的展开与分析 💬

让我们回顾并分析第二次对话的完整过程。

```
我：你误会我了。
她：怎么了？
我：我没有恶意，我也不知道。我真的在等车，我知道。
她：没有说你们在。我刚吃了，等我朋友过来。
我：你是这边本地人还是北京人？我一开始以为你是游客过来玩的，你知道吗？
她：哦，我现在有点忙。
我：就你现在只要走？
她：对。
我：你知道我为什么来跟你讲话吗？你是知道还是不知道？
她：你不知道。
我：之前没有男生过来搭讪你？
她：没有男的过来搭讪。别说了，别说，我现在这心里就很烦。
我：那走之前可不可以加你个微信？
她：不加了不加了。
我：大不了做朋友也可以。
她：可以。又不加了。
```

以下是对话的关键点分析：

*   **建立安全与正常感**：我说“我没有恶意”，这是一个同理心陈述，旨在安抚她，让她不要认为我是坏人。当她强调“我真的在等车”时，我回应“我知道，我没说你不在等车”，以消除对抗感。
*   **尝试正常沟通**：我通过询问“你是本地人吗？”、“我一开始以为你是游客”来尝试建立正常的对话，交换一些基本信息，试图在此过程中提供一些价值。
*   **应对持续负面情绪**：尽管我尝试沟通，她仍在看手机，并表现出烦躁。我保持心平气和，并试图通过提问“你知道我为什么来跟你讲话吗？”来勾起她一丝好奇心。在搭讪中，第一步永远是学会勾起女人的好奇心。
*   **试探与最终被拒**：我接着问“之前没有男生搭讪过你吗？”，她以非常不好的语气否认。当她明确表示“别说了，我很烦”并最终拒绝加微信时，我选择了离开。

## 内核心态的深度解析 🛡️

到这里，99.9%的男人甚至100%的男人都不会再尝试了。你们会认为这个女人已经拒绝了你。

但我的解读不同。我认为她拒绝的是**当下这个情况**和**她自己的坏心情**，而不是我这个人。这很复杂，关键在于你如何看待自己、如何解读事件、如何不把事情个人化。

这个案例的经典之处在于，即使到了这个地步，最终仍可能有结果。她的负面情绪非常强烈。当我走开时，我的助理都以为我放弃了。

但我非常不甘心。我自己的感觉告诉我，这个女生是可以的，因为她很吸引我。即使她明确拒绝了，我内心知道还能再试一次。

这就是我想展示的：一个真正强大的内核是什么样的？**你的状态、心情、情绪完全不被任何外在反馈所影响。**

在这个案例中，她的反应足以击垮绝大多数男人，他们会解读为“我被这个女人拒绝了”。但在那时，当我走上台阶与助理对视时，我就知道我还能撩到她。我决定给自己，也给她第三次机会。

本节课中我们一起学习了如何通过强大的内核心态，冷静分析拒绝，并转换策略进行第二次沟通尝试。关键在于将对方的负面反应归因于情境而非个人，并始终保持稳定的自我价值感。下一节课，我们将揭晓第三次尝试的底层逻辑与最终结果。
