# AMG《纵横四海》：11.夜店流程解说 _ 带走关门

## 概述
在本节课中，我们将学习如何在夜店中成功带走目标女生。

## 夜店流程解说

以下是夜店带走流程的详细步骤：

### 1. 筛选冲口
- 打击完目标后，询问女生想要玩到几点。
- 如果女生喜欢你，她会给出一个具体的时间。

### 2. 提出休息建议
- 如果女生没有给出具体时间，你可以提出休息的建议。
- 你可以说：“我觉得差不多了，要不然我们走吧。”
- 如果她问要去哪里，你可以直接说：“我们找个地方休息就好。”

### 3. 处理其他问题
- 如果女生再问，你可以用轻松的语气说：“这么晚的，当然就找旅馆休息了。”
- 这种语气要显得很chill，很坦荡，理所当然。

### 4. 拉私密空间
- 如果遇到女生犹豫不决的情况，你可以尝试拉私密空间。
- 例如，你可以对她说：“我要去旅馆，可是我们今天不能干吗？”

### 5. 铺哏与转场
- 在提出休息建议之前，可以先铺哏，例如：“要不然三点我们去吃个宵夜，附近有家好吃的凉面。”
- 然后再转场，例如：“吃完宵夜再喝一杯，唱歌。”

### 6. 尝试不同方式
- 如果女生不愿意去旅馆，你可以尝试其他方式，例如：
  - 一起唱歌
  - 一起吃宵夜
  - 一起回家

### 7. 处理团队情况
- 如果女生是团队中的一员，你需要考虑以下情况：
  - 与她的朋友保持友好关系
  - 尝试将她的朋友支开
  - 将整个团队带走

### 8. 使用APP找休息地方
- 在夜店中，旅馆可能会客满，这时你可以使用APP找休息地方。
- 例如，你可以使用“方鬧”或“FUNNOW”等APP，搜索附近有空房的旅馆。

## 总结
本节课中我们一起学习了夜店带走流程，包括筛选冲口、提出休息建议、处理其他问题、铺哏与转场、尝试不同方式、处理团队情况以及使用APP找休息地方。希望这些技巧能帮助你成功带走目标女生。
