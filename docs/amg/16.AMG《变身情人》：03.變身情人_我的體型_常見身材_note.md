# 16.AMG《变身情人》：03.變身情人 _ 我的體型 _ 常見身材

## 概述
在本节课中，我们将学习如何根据不同的身材类型选择合适的穿搭。

## 常见身材类型
以下是几种常见的身材类型：

### 矮身材
- **特点**：身高较矮，肩膀和腰臀比例接近。
- **穿搭建议**：
  - 穿多层衣物，避免单调。
  - 避免穿紧身裤，以免显得腿短。
  - 选择低筒鞋，避免高筒鞋。
  - 选择短版外套，如牛仔外套或飞行外套。
  - 选择同色系穿搭，避免下半身过于分散注意力。
  - 避免穿大件衣物，以免显得更矮。
  - 避免穿Oversize长版外套，以免显得腿短。
  - 避免穿低腰裤，以免显得腿短。
  - 避免留厚重长发，以免显得头大。
  - 选择合适的发型，如短发。

### 瘦身材
- **特点**：身材较瘦，肩膀较窄。
- **穿搭建议**：
  - 穿多层衣物，增加分量感。
  - 选择厚磅材质的衣物，如毛衣、牛仔外套、卡其外套。
  - 选择Oversize T恤，增加肩部宽度。
  - 选择横条纹衣物，增加宽度感。
  - 选择亮色厚毛衣，增加暖男气质。
  - 选择硬挺材质的衣物，增加质感。
  - 选择厚实鞋底，增加分量感。
  - 选择合适的饰品，如项链、太阳眼镜。

### 胖身材
- **特点**：身材较胖，肩膀较宽。
- **穿搭建议**：
  - 穿多层衣物，遮盖身材。
  - 选择深色裤子，如黑色、军绿色、深蓝色。
  - 避免穿白色裤子，以免显胖。
  - 选择硬挺材质的衣物，增加质感。
  - 选择厚实鞋底，增加分量感。
  - 选择合适的饰品，如项链、太阳眼镜。
  - 避免穿紧身衣物，以免显胖。
  - 避免穿浅色衣物，以免显胖。
  - 避免穿厚真支毛衣，以免显胖。
  - 选择硬挺材质的衣物，增加质感。

### 胖矮身材
- **特点**：身材较胖，身高较矮。
- **穿搭建议**：
  - 选择合适的运动，如跑步、游泳等，以减轻体重。
  - 考虑进行缩胃手术，以减轻体重。
  - 选择合适的衣物，如硬挺材质的衣物、深色裤子等。
  - 选择合适的饰品，如项链、太阳眼镜。

## 总结
本节课中，我们介绍了不同身材类型的穿搭建议。希望这些信息能帮助你找到适合自己的穿搭风格。
