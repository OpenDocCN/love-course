# 课程16.AMG《变身情人》：01.变身情人 _ 引言：课程目标与大纲 📚

在本节课中，我们将要学习《变身情人》课程的整体目标与内容大纲。本课程旨在帮助你通过调整穿搭，提升外在形象，从而在约会等社交场合中更具吸引力。

## 课程目标

本课程只有一个目标：**让你变帅**。具体而言，是帮助“宅男变帅哥”。购买本课程，你一定能学到一些实用的技巧，让自己变得更帅。

我们最初制作这个课程，是因为许多学员来拍照。我们当然能把照片拍得帅，但摄影师凯西后来发现，这些学员其实可以更帅——前提是他们在穿搭上做出一些改变。我们在之前的《一级玩家》课程中介绍过一些基本穿搭，本课程则是在此基础上衍生，讲解更多单品并提供大量穿搭示范，让你了解一个帅哥真正会穿成什么样子。

**核心概念**：帅 ≠ 满身潮牌或名牌。穿名牌可能让人知道你很有钱，但不代表帅。

## 课程特点

以下是本课程的几个核心特点：

1.  **简单快速**：课程中讲解的主要穿搭单品均购自Zara。选择Zara的原因是它简单快速，无论是线上购买还是门店购买都很方便，退货流程也便捷。通过App下单，试穿后如不满意，可直接在App上操作退货，快递会上门取件。整个过程简单快速方便。需要声明，本课程并非Zara的商业推广。

2.  **选择Zara的原因**：我们认为Zara的质感优于H&M。Uniqlo的部分单品（如素T、裤子）不错，但整体若全部穿Uniqlo，效果可能不够帅。其外套等版型更偏向舒适与品质好，但不如Zara的版型干净利落。Zara风格多样、品项选择多，能快速让人穿上后有变帅、有个性的感觉。同时，Zara价格相对亲民，比起一些运动品牌或定价三四千台币的牌子更便宜，且常有折扣，符合省钱需求。

3.  **好模仿与出错率低**：课程中的示范照片均以四位教练和我们拍摄过的学员为主。他们都是台湾人，不同于你在网络或穿搭杂志上看到的、拥有模特身材和面孔的外国人。我们的示范涵盖了各种身材类型（高、矮、胖、瘦），你可以寻找与自己身材相似的对象进行模仿，从而大幅降低出错率。照着穿，你至少能达到“干净舒服”的效果。虽然不敢保证让你变成超级型男，但至少约会时，女生会觉得你干净、舒服且有品味。

## 为何需要改变

你来到这里的目的很明确：变帅。变帅最实用的好处在于，它能为你的约会和把妹过程加分。

必须说明，这与我们早期强调“内在游戏”的阶段已有很大不同。把妹其实是全方位的，需要**外表**、**经济能力**和**聊天技巧**共同作用。我们过去80%-90%的产品和影片都侧重于聊天技巧，但现在我们认为穿搭调整也非常必要，且往往只需稍作调整就能带来显著加分。

许多学员来拍照时，穿搭可能整体尚可，但一个小配件（如不合适的包包）就可能让整体感觉扣分。女生在约会时同样会注意到这些细节。摄影师凯西在为他们造型时，会认真想象“如果这是我男朋友，我会如何改造他，让他变得更帅？”这种细微的调整非常重要。

不要认为这是肤浅。所有人都一样，如果你在街上与一个外表邋遢的女生牵手，你也会感到不自在。因此，改变穿搭除了让自己看起来更舒服、更帅之外，也能在约会中为你加分。起码当你讲一些俏皮笑话时，不会显得猥琐，反而让人觉得幽默。反之，如果穿着很“宅”，讲同样的笑话效果可能就完全不同。

本产品的诉求很简单：**让你改变、变帅**。这样，无论是约会还是街头搭讪，你都不会感觉像身处“地狱模式”。第一印象非常重要，而外表仪容正是构成第一印象的关键部分。有些学员照片拍得不错，但实际约会时若穿回自己原本的衣服，就可能造成扣分。

## 什么是“好看”

那么，究竟什么才是“好看”呢？时尚潮流的标准因人而异，有人觉得宽松好看，有人觉得合身好看。

本课程由摄影师凯西主导，他认为“好看”的定义是：**配色和谐、整体看起来舒服、有质感**，而非一定要满身潮牌。

你可能会问，为什么要听他的？因为他的审美经过了实际验证。他负责为许多学员拍摄交友软件用的照片，在拍摄前，他会先帮学员打理穿搭。只有穿搭得体，拍出来的照片才会帅。经他打扮并拍摄后，学员在交友软件上的实测效果都很好。例如，有学员Joe的照片被女生称赞像“韩国欧巴”，还有许多学员反馈，约会时女生说“你跟照片一样帅”。因此，凯西的审美是从女生视角出发且合格的。

凯西是设计系毕业，虽然不是专业服装讲师，但他的审美符合交友软件上女性用户普遍喜欢的男性形象，可以说是大众眼光。这能给你信心，我们的标准是**女生会喜欢的标准**，照着穿去约会肯定没问题。

凯西喜欢的不是最花哨、最新的潮流，而是**简单、实用、有质感**的穿搭。如果你对某些示范 initially 没有感觉，我们仍建议你尝试跟着后面的范例做。

穿搭有很多面向，本课程主要聚焦于**约会与休闲**场景，让你穿得好看。很多人误以为穿搭必须非常花哨或突出（例如去夜店要涂黑指甲），传统把妹理论中称之为“半孔雀”，旨在像孔雀开屏一样吸引注意。但这通常只会吸引特定小众群体，且不一定被普遍认为是“好看”。我们的目标则是让你成为一个**有质感的正常人**，吸引大众族群的女生。我们的配色和单品选择都旨在让你穿起来不奇怪。

## 课程大纲

在进入正式课程前，我们先简单介绍一下课程结构。本课程共分为七大章节：

1.  **NG穿搭**：讲解你绝对要避免的穿搭错误。改掉这些，你的“宅味”大概就先少了一半。

2.  **我的体型与常见身材**：我们将针对四位教练的体型进行介绍，并讲解常见身材（如偏胖、偏矮）的穿搭注意事项，帮助你看上去比例更好。你可以在这里找到与自己身材最接近的参考对象，以便后续模仿。

3.  **百搭配色**：我们精选了十种（九加一）我们认为好搭、不易出错的配色。建议你的衣橱里主要保留这些颜色的衣物，其余一两成可以留给你个人真正喜欢的颜色。

4.  **购买顺序与穿搭照参考**：这是课程的核心部分。共有四个篇章告诉你应该按什么顺序购买哪些单品，并提供大量单品参考图，方便你模仿。例如，某件上衣该配什么颜色的裤子或鞋子。

5.  **必备服装、配件与发型**：这里将第四章“购买顺序”中的内容整理成清晰的表格，方便你按图索骥。同时，我们也会介绍配件和发型。凯西认为配件不是主要重点（不是戴什么表就会很帅），但发型非常重要。我们会讲解注意事项，例如头大的人如何通过发型调整整体比例。

6.  **衣橱分类**：这是一个附加章节。除了外在穿搭，你的房间和衣橱整洁度也很重要。这里将分享如何整理和分类衣橱。男生通常容易显得脏乱，保持整洁会让人更舒服。

7.  **推荐品牌**：这是AMG的加分环节。我和Alex将分享我们自己喜欢的、Zara以外的品牌。如果你平时在IG上喜欢我们的穿搭（直播时穿得比较随便），可以在这里找到灵感并进行模仿。例如，我偏好穿素色和运动机能品牌，而Alex则会购买很多潮牌。如果你觉得自己体型或风格与他更接近，可以参考他的推荐。
