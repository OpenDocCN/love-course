# 16.AMG《变身情人》：09.變身情人 _ 重點總整理 _ 配件

## 概述
在本节课中，我们将学习如何根据《变身情人》教程，整理并补充配件，以便更好地进行穿搭。

## 配件整理
以下是配件整理的详细内容：

### 单品分类与颜色
- **单品**：列出需要购买的衣物单品。
- **分类**：对单品进行分类，如上衣、下装、外套等。
- **颜色**：列出该单品适合的颜色。

### 体型与购买顺序
- **瘦子**：列出适合瘦子的单品，并标注优先购买顺序。
- **胖子**：列出适合胖子的单品，并标注优先购买顺序。
- **矮子**：列出适合矮子的外套和鞋子，并标注备注。

### 配件推荐
- **眼镜**：推荐购买粗框眼镜，以增加型格。
- **香水**：推荐购买淡香水，避免选择过于成熟的香味。
- **包包**：推荐购买黑色或深色系的包包，避免选择过大或过于花哨的款式。
- **太阳眼镜**：推荐购买雷朋（Ray-Ban）品牌的3025或2140型号。
- **帽子**：推荐购买老帽或New Era帽子。
- **饰品**：推荐购买Recovery品牌的饰品。
- **皮带**：推荐购买黑色或咖啡色的Mur字扣皮带。

### 髮型建议
- **头大的人**：建议选择利落轻盈的发型，并剃掉两侧头发。
- **肩膀问题**：通过穿着宽松的衣服、选择硬质衣服或锻炼肩膀来使肩膀看起来更宽。
- **尝试新发型**：尝试不同的发型，并听取他人的意见。

## 回家作业
- **香水**：购买一瓶香水，并每天洗澡保持干净。
- **眼镜**：考虑是否需要戴隐形眼镜或进行激光手术。
- **包包**：更换包包，选择黑色或深色系的反步包。
- **背包**：避免将水壶和雨伞放在背包外侧口袋。
- **发型**：尝试新的发型，并听取他人的意见。
- **头大问题**：考虑如何使头部看起来更小，如购买帽子、调整肩线等。
