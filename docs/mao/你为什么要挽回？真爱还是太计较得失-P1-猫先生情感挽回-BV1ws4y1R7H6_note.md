# 你为什么要挽回？真爱还是太计较得失-P1-猫先生情感挽回-BV1ws4y1R7H6_note

在本节课中，我们将要学习如何审视自己挽回一段感情的真正动机。许多人在考虑挽回时，其出发点可能并非纯粹的爱情，而是陷入了对自身付出的计较。理解这一点，是建立健康亲密关系的第一步。

很多咨询者找我挽回的时候，当我问起原因，他们的回答是：谈了很长时间了，不挽回那些时间不是浪费了吗。

我只想说，你想挽回的不是爱情，而是你在这段感情中的**沉没成本**。你所珍视的也不是对方，而是你自己对这段亲密关系的付出。

太过于计较个人得失的感情是不长久的。你每天都在想“我付出了多少，对方付出了多少”，然后把你们的付出放在天平的两端。发现你的那一边比较重，你就开始忧虑了：“他是不是不爱我了？”“我是不是要分手了？”“分手之后我付出的那些应该怎么办？”

太过计较这些只说明了一个点：在感情中你是缺乏信任的，是一个**爱情的乞讨者**。

这里所说的信任包括两个方面：对另一半的信任和对自己的信任。

如果你坚信自己是一个足够好、足够优秀、足够值得被爱的人，你自然不会去对比你们付出了多少，以及怀疑对方是不是爱你了。

你的自信能够感染对方，让对方不由自主地认同你，并且更加珍惜你。你的自信也给你带来了主动权，因为你相信自己足够好，你不会把自己放到一个低位，对对方百般讨好，却希望对方回馈给你同样的爱。

事实上，越卑微的一方，反而越不被珍惜。

你可能会觉得：“又要我重视对方的付出，又要我别太在意得失，这太难了。”

但其实这两者并不矛盾。我想告诉你，要有一个正确的爱情观：不吝啬给予，也不过分索取，更不过分计较自己和对方的付出是否绝对平衡。

---

本节课中我们一起学习了审视挽回动机的重要性。核心在于区分你是为了挽回“爱情”本身，还是为了挽回自己过去的“付出”（沉没成本）。健康的感情需要建立在自信与相互信任的基础上，而非对得失的精密计算。一个正确的爱情观是：**勇于付出，但不计较；信任对方，更信任自己**。
