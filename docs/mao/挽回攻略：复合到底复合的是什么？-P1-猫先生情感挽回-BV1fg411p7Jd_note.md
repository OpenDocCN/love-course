## 复合的本质与三个阶段 📊

人为复合的是情绪。从分手到重新建立爱情关系，这个过程会分为三个阶段：平息、和好、地境。

平息阶段是你们刚刚分手不久，需要处理自己的情绪，总结自己在这段感情中的得失，反思自己做错了什么，以及对方哪里错了。同时，也需要思考未来怎么改正这些问题。

和好阶段是你改善自己在对方心目中的负面形象，重新建立信任关系的过程。

地境阶段说的是你们建立信任关系之后，再走一遍由朋友到恋人的过程。

根据这三个时期做一个总结：**复合的实质是消除你在对方心目中的负面形象，重新建立信任关系的过程**。你的重心应当是对方的情绪和态度，而非单纯这个人。换句话说，如果分手后你在对方心目中不存在负面情绪，你也没有必要去复合。毕竟你不能复合一个不爱你的人。

上一节我们概述了复合的整体框架，接下来，我们将深入探讨分手后首先要做的事情。

## 一、分手后首要做的事情：平息阶段 🧘

平息阶段也就是分手后总结得失这个时间段。这一时期你的首要之事，不是向对方询问分手的理由，也不是否认分手的事实，更不是问“能不能不分手”这样幼稚的问题。

而是和对方达到和平分手的共识，调整自己的心态、总结得失、确定要不要复合、评估复合的可能性、制定复合方案。

### 如何调整自己的心态

调整自己的心态，说的就是发泄被分手的挫败感和痛苦感，不要把情绪积压在心里。有些人分手后不哭不闹，安安静静地做自己的事情，比如打一下午的游戏、看一下午的书。这些事情看似是在发泄情绪，实则是在把悲伤刻在心里。

一般来说，习惯这样发泄悲伤情绪的人，内心始终有一种不安全感，对外界秉承着一种怀疑的态度。对于这类特质的人群，下面这些思路不妨试一试。

以下是调整心态的三个步骤：

1.  **承认失败**：对你而言，最难的不是承认一段亲密关系的失败，而是不要将自己一段感情的失败，归咎于自己人生的失败。你要认识到，分手或许有你做不好的原因，但是也有对方做错的地方。将所有的原因归咎于自己是不合理的。承认自己某一方面能力不行，并不代表全盘否定自己。
2.  **转移方向**：承认自己经营亲密关系能力不行之后，要把这种挫败感转化为自己走向美好未来的动力，而非一辈子否定自己。应通过认识到自己的能力不足，来激发自己学习经营亲密关系的内生动力。
3.  **实际补偿**：这一步是第二步的现实反馈。激发自己变得更好的内在动力后，通过不断的努力获得实际的成就感，来确认自己真的变优秀了。

调整自己心态的同时，也要缓解对方的情绪。具体的做法就是：**切忌纠缠对方求复合**，给他一段自由的时间来发泄情绪，也就是要与他断联一段时间。

在断联结束后告诉对方，你尊重他的选择，体面地离开。这个时候，由于他对你的负面情绪基本上释放得差不多了，如果你的“演技”足够好，撤退得足够果断，对方可能还会怀疑自己才是被分手的那个，会怀疑分手决定是否正确。这就为你接下来的和好关系埋下了伏笔。

在调整好自身心态并初步处理了双方情绪后，我们需要更理性地看待这段关系是否值得以及可能挽回。

## 二、评估复合的可能性 🔍

心态调整过来之后，接下来就是综合内外部因素，评估一下复合的必要性和可行性。

评估需要考量以下因素：
*   **外部因素**：例如双方父母的意愿、现实利益、未来发展规划、家庭背景。
*   **内部因素**：例如分手后他对你的态度、双方核心矛盾能否解决。

这既是对自己手里可用筹码的总结梳理，也是质问自己是不是一定要跟他复合。

这里要详细介绍一点，那就是如何通过对方的行为反应判断其复合意愿。

### 如何判断对方的复合意愿

复合意愿从弱到强，可以依据两个节点划分为低、中、高三个层次。

以下是三个意愿层次的具体表现：

*   **低度意愿**：核心是态度厌烦，不愿同你交流。常见行为是拉黑删除、不接电话、不回信息或者回复很慢。
*   **中强度意愿**：核心是态度中立，愿意同你维持沟通交流，但是不能交流感情话题。
*   **高强度意愿**：核心是主动给你创造复合的机会。常见行为是想办法引起你的关注、主动联系、不拒绝你的肢体接触、营造暧昧的氛围。

可以依照自己的实际情况进行对比。当然，具体问题需要具体分析。
