# 情感挽回课程 01：如何应对“不拉黑不删除”的冷暴力局面？💔

在本节课中，我们将要学习一种特殊的情感挽回场景：当你的前任既不拉黑也不删除你，却对你采取完全的冷暴力，拒绝任何沟通时，应该如何寻找突破口。我们将深入分析对方的心理状态，并提供一个核心的应对策略。

## 冷暴力局面的本质分析

上一节我们介绍了课程的主题，本节中我们来看看这种“不拉黑不删除”的冷暴力局面背后，对方的心理状态是什么。

有一种情况是，无论你对前任说什么，对方都不予理睬。他既不删除你，也不拉黑你，完全是一种冷暴力的状态。遇到这种状态，应该如何去挽回呢？

要理解这一点，你需要从他的思维角度去思考一个问题：他为什么会对你有这样的态度？首先需要明确，能做这种事情的人，通常内心非常坚定。这个人非常有原则，而且有框架。在心理学上，我们可以将这种状态理解为 **“固着心态”**。

## 理解“固着心态”

上一节我们提到了“固着心态”这个概念，本节中我们来详细解释它的含义。

“固着心态”是什么意思呢？举例来说，如果我对一个事情的认知，认定它是方的，那就绝对不会变成圆的。无论你怎么跟我解释它是圆的，也改变不了我的看法。这种心态意味着内心极其固执，非常有个人想法，很少轻易受别人影响，内心非常坚定。

这是一个非常有韧性的人。如果加上你之前的一些做法，让他认为你就是这样的，你改变不了，那你就不适合他，你们没有他想要的未来，不可能再成为男女朋友关系了。这样的认知就固化在了这个人的心里。

他会不断用这个认知去深化对你的了解。所以，之前无论你说什么去试图改变他的想法，任何试图改变这个人的行为都会是徒劳的，他根本不会接受。

## 寻找唯一的突破口

理解了对方的“固着心态”后，我们面临的核心问题是：这种情况我们应该怎么去做，或者怎么去寻找一个突破口呢？

唯一的办法就是让他自己改变对你的看法，并且这种改变必须是潜移默化的。

以下是实现这一目标的具体思路：

*   **核心原则**：在不暴露需求感且合理化的相处过程中，你把自己的状态展示给他。
*   **关键条件**：这种状态的改变不是展示一天两天就可以了。所需时间根据难易程度决定。
*   **时间预期**：我之前见过坚持半年的案例，也见过做一两个月的案例，最后对方的确也回来了。
*   **策略评估**：这个办法如果运用得恰到好处，是有效的，但是需要的时间比较长。

所以，如果你确实遇到了一个具有“固着心态”的挽回对象，并且确确实实想去挽回，在你进行感情修复时，大概率别无他法，只有这一个途径。

## 课程总结

本节课中，我们一起学习了如何应对前任“不拉黑不删除”的冷暴力局面。我们分析了这种行为背后的 **“固着心态”**，并明确了唯一的突破口：通过长期、潜移默化地展示真实的自我改变，让对方从内部主动转变对你的看法。这是一个需要耐心和坚持的策略。
