# 猫先生的Python入门课 #1：变量与数据类型 🐱

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。它们是构建所有程序的基石，就像盖房子需要砖块和水泥一样。

## 什么是变量？ 📦

上一节我们提到了变量的重要性，本节中我们来看看它的具体定义。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等），而标签（变量名）则帮助我们记住盒子里装的是什么。

在Python中，我们使用等号 `=` 来给变量赋值。其基本语法是一个**公式**：

`变量名 = 值`

例如，我们创建一个名为 `age` 的变量，用来存放数字 `3`：

```python
age = 3
```

现在，名为 `age` 的“盒子”里就存放了数值 `3`。之后在程序中，我们使用 `age` 这个名称，就相当于在使用数字 `3`。

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有几个内置的基本数据类型，最常见的有以下三种。

以下是三种核心的数据类型介绍：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数、负数或零。
    *   例如：`10`, `-5`, `0`

2.  **浮点数**
    *   表示带有小数点的数字，用于更精确的数值计算。
    *   例如：`3.14`, `2.0`, `-0.5`

3.  **字符串**
    *   表示文本信息。在Python中，字符串需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   例如：`“Hello”`, `‘猫先生’`, `“123”`（注意：这里的123是文本，不是数字）

## 如何查看和使用变量？ 👀

理解了变量和类型后，我们来看看如何操作它们。在Python中，我们可以使用 `print()` 函数来查看变量的内容。

例如，我们定义几个变量，然后打印它们：

```python
name = “咪咪”  # 字符串
age = 2        # 整数
weight = 4.5   # 浮点数

print(name)
print(age)
print(weight)
```

运行这段代码，你会在屏幕上看到：

```
咪咪
2
4.5
```

你还可以在 `print` 中组合变量和文字：

```python
print(name, “今年”, age, “岁，体重是”, weight, “公斤。”)
```

输出结果为：
`咪咪 今年 2 岁，体重是 4.5 公斤。`

## 一个简单的实践：猫咪信息卡 🐾

现在，让我们把以上知识结合起来，创建一个简单的“猫咪信息卡”程序。

以下是创建信息卡的步骤：

1.  为你的猫咪想一个名字，存入变量 `cat_name`。
2.  设定它的年龄，存入变量 `cat_age`。
3.  设定它最喜欢的食物，存入变量 `favorite_food`。
4.  使用 `print` 函数，将这些信息漂亮地打印出来。

参考代码：

```python
cat_name = “雪球”
cat_age = 3
favorite_food = “小鱼干”

print(“=== 猫咪信息卡 ==”)
print(“名字：”, cat_name)
print(“年龄：”, cat_age, “岁”)
print(“最爱食物：”, favorite_food)
```

你可以尝试修改引号内的文字和数字，创建属于你自己的猫咪信息卡。
