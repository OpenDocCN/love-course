# 分手后如何正确挽回前男友：钓鱼式复合法加快你们复合 🎣 - 课程 P1

在本节课中，我们将要学习一种名为“钓鱼式复合”的情感挽回技巧。这种方法尤其适用于分手后前任态度冷淡、拒绝沟通甚至拉黑你的情况。通过学习并应用本教程的核心步骤，你将有机会扭转被动局面，增加复合的可能性。

## 核心原则与准备

上一节我们介绍了课程的目标，本节中我们来看看实施“钓鱼式复合”前必须做好的两项准备工作。这两点是后续所有操作的基础，目的是为了停止错误行为并调整自身心态。

以下是两项必须立刻执行的准备工作：

1.  **立刻停止所有猛烈的求和攻势**。在未掌握正确方法前，你的纠缠和哀求往往无法触及对方真正的需求，反而会暴露你的软弱。
2.  **马上收起玻璃心，停止自我感动的行为**。你的卑微和纠缠只会将对方推得更远，从任何角度看都得不偿失。

## 钓鱼式复合的核心策略

做到以上两点之后，核心策略可以概括为：**表面晾着对方，背地里刺激对方，同时顺势观察**。这是扭转局势的王牌技巧。

为什么要这样做？因为在分手初期，作为被分手且纠缠过的一方，你们的关系已经处于极度不平衡的状态。挽回的核心在于**拿回关系中的主动权**。如果关系不对等，有效沟通就无从谈起。

## 拿回主动权的具体步骤

上一节我们了解了策略的核心思想，本节中我们来看看如何通过具体步骤打破僵局，拿回主动权。其核心是**打破“窄化框架效应”**，即避免与对方直接对抗，合理运用以柔克刚、循序渐进的“柔韧性攻势”。

以下是分阶段实施的详细步骤列表：

1.  **顺从与冷却（约1周）**：既然分手是他提出的，就先顺从他的意思，给予双方空间。表面上完全“晾着”对方，不进行任何主动联系。
2.  **首次接触与刺激**：冷却约一周后，主动给他发一条信息。内容可以是请他帮个小忙，或咨询他擅长的领域。**无论他是否回复，两小时后都再发一条信息**，告知“问题已解决，不用麻烦了”。此举旨在进行轻微刺激。
3.  **情感触动与铺垫**：再间隔约一周，将他留在你处的物品打包寄回。附上一张便签，内容需包含：肯定他曾对你的爱、肯定你曾对他的爱、表达你的遗憾与无力感、最后附上简单的关心话语。字数控制在50-60字左右。
4.  **等待与回应**：寄出物品后，三天内他很可能会联系你。此时不要急于回复。你可以先随意发布一条与感情无关的朋友圈，等到晚上有空时再回复他。回复时需注意两点：**第一，不谈感情；第二，不恋战，见好就收**。

## 课程总结

本节课中，我们一起学习了“钓鱼式复合”挽回法的核心逻辑与具体操作步骤。我们从停止错误行为、调整心态开始，到通过“表面晾着、背地刺激”的策略扭转关系高低位，最后通过“顺从冷却—轻微刺激—情感触动—冷静回应”的四步法，一步步拿回关系主动权，为有效沟通和复合创造机会。记住，挽回的关键在于重建平衡，而非一味乞求。
