## 分手后决定回头的关键因素

上一节我们提到了课程主题，本节中我们来看看男生在真性分手后是否回头的决定性因素。

男生在真正分手后能决定他是否回头的因素并不在于他爱不爱你，或者说你有多爱他，而是这段感情值不值得他回头。

**核心公式：挽回可能性 ∝ 问题解决程度**

如果在分手之后的感情中，你们之间导致分手的重大缘由问题仍然没有解决，就算他能和你正常的聊天，偶尔还会和你暧昧、嘘寒问暖，但是你们两个人之间也很难走到最后。

说白了，你爱一个人，仅靠爱发电也没有用。起决定性作用的是你们能不能把导致分手的问题给解决掉。

---

## 男生分手后的常见心理与行为

理解了关键因素后，我们进一步分析男生分手后的典型心理状态。

很多时候男人是这样的。分手之后他会和你聊天，还会关心你，甚至你约他出来他也会欣然接受，但是呢，就是不答应与你复合。

为什么会这样？是因为在这个男人心里，你们两个人是没有未来的。你目前只是他过渡生活的替代品罢了。但是他心中认定你不是他的终身伴侣，所以不想给自己、给彼此过多的希望。

因此，对于真性分手的男生来说，需要一段时间的冷静期，基本上在三天到一周左右，去走出负面情绪，重新审视爱情。

---

## 冷静期内的正确行动指南

知道了需要冷静期，但这段时间并不是说你要坐以待毙，而是要做出切实的形象和性格的重塑。

以下是开始重塑前必须完成的准备工作：

1.  **明白彼此的分手动因**：即彼此之间到底存在着什么矛盾，是男生无法退让，只能选择分手的方式来解决问题。
2.  **审视自我**：是自己性格上面的缺陷，还是某些对方难以接受的习惯性行为，或是彼此思想观念上的冲突。

---

## 重塑自我形象与隐藏需求感

当你搞清楚分手动因后，正确的做法不是让你表现出自己已经认识到错误，并且急于求和。这样只会让对方再次拒绝甚至拉黑你。

你要做的是懂得重塑自我的形象，并隐藏自己的需求感。

**行动代码示例：**
```plaintext
IF 分手动因 == “过分自我，忽略对方感受”
THEN
    行动 = “尝试换位思考，分析矛盾”
    沟通身份 = “朋友”
    目标 = “通过交流展示改变”
END IF
```

比如说，你的前男友是因为你过分自我，从不照顾他们的感受而选择分手。那么你就要选择抓住这个导致分手的动因，积极地重塑自我。

具体方法是，尝试着站在他的角度重新分析分手之前的很多矛盾，反省自己曾经有哪些地方忽略了对方的感受。

同时可以试着以朋友的身份，暂时地和对方取得交流。让男生觉得从你的交流语气和态度中，能够感受到你的改变，从而潜移默化地改变对你的刻板看法。

---

## 展示改变与创造转机

这是你在实施确切挽回行为之前的前提。同样也可以通过朋友圈开始暗示，向男生表达自己的改变。

通过这样的方式，他会重新注意到你对他的爱，才会迎来转机。
