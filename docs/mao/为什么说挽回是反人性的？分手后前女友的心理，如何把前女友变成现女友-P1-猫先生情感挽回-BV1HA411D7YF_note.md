# 情感挽回课程 P1：挽回的本质与心态调整 🧠

在本节课中，我们将要学习为什么挽回过程常被称为“反人性”的行为，并分析分手后女性的心理状态。理解这些核心概念，是成功将前女友重新变为现女友的第一步。

## 挽回为何是“反人性”的？

很多人在挽回过程中最难克服的是害怕失去对方的冲动。这股冲动往往是导致挽回失败的根源。挽回之所以被称为“反人性”，正是因为这种本能的恐惧会驱使我们做出错误的行为。

在这种心理的驱使下，人的语言和行为会变得与平时不同。

以下是常见的错误语言表现：

*   我知道错了，都是我不好。
*   我没有考虑到你的感受。
*   你告诉我哪错了，我一定改。
*   求求你不要离开我，我只想和你在一起。
*   我这么爱你，我不能没有你，离开你我可怎么办。

以下是常见的错误行为表现：

*   进行信息骚扰和电话轰炸。
*   在朋友圈发布充满负能量的话语，希望引起对方注意。
*   一有机会就寻找对方、要求见面。
*   见面后不断哭泣和哀求。

这些表现都是内心害怕失去的真实写照，将人性中恐惧失去的本能暴露无遗。然而，这样的行为不但无效，反而会让前任更加排斥。

## 错误行为导致排斥的原因

上一节我们介绍了挽回中的常见错误，本节中我们来看看为什么这些行为会适得其反。这涉及到人性的几个层面。

**第一，相由心生。**
当你内心充满负面情绪时，整个人会由内而外散发出一种消极的气质。这种状态毫无吸引力可言。人与人之间的关系依靠相互吸引，你越是消沉，吸引力就越低。

**第二，女性的慕强心理。**
这是人类社会演化形成的本能。女性通常倾向于被各方面强于自己的男性所吸引。分手后你的卑微和可怜，只会让她更加反感和确信分手的决定。

**第三，情绪不稳定触发信任危机。**
我们都倾向于与情绪稳定的人相处，这样才能获得安全感。情绪不稳定的人会让人觉得幼稚、不可靠，从而想要远离。

**核心公式：错误挽回 ≈ 卑微讨好 + 死缠烂打 → 吸引力丧失**

现在你明白了吗？你越是卑微讨好、死缠烂打，挽回成功的几率就越低。一个严重的后果是对方可能将你彻底拉黑，使你失去所有联系渠道。

## 正确的挽回心态与基础步骤

男人最令人叹息之处在于：在不应联系时拼命联系，在该坚持时却选择了放弃。这归根结底是因为对亲密关系的认知和思维存在错误。

当你因害怕失去而情绪失控时，最好的方法是进行“断联”。这一点至关重要。

**记住：** 一定要找出你们分手的根本原因，而不是表面的浅层原因。如果找不到根本原因，前任只会认为你毫无改变，从而失去重建信任的宝贵机会。

同时，必须付出行动进行自我提升。在断联期间，你的改变和提升就应该开始，并且在后续的挽回过程中持续进行，绝不能是三分钟热度。
