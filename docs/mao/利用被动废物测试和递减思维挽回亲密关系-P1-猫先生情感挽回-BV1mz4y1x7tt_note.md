# 情感挽回课程 P1：利用被动废物测试与递减思维 🧠

在本节课中，我们将学习如何通过理解“被动废物测试”和运用“递减思维”来重建关系中的舒适感，从而为挽回亲密关系奠定基础。

上一节我们介绍了课程主题，本节中我们来看看舒适感被破坏后的直接后果。

做这个行为带来的直接后果就是把你们这段关系中的舒适感给破坏掉了。任何关系失去了舒适感都不可能维持长久。

所以，你现在要挽回，就要把这个舒适感重新建立起来。

## 重建舒适感不等于跪舔

但是，营造舒适感不等于跪舔。这不是让你顺从或者依附对方。

以下是挽回中常见的一个场景及其分析：

*   在挽回过程中，很多女人会反映对方说一些特别决绝的话。
*   例如：“你别再联系我了”、“咱俩没可能了”、“咱们已经分手了，我希望能认清这个事实”等等。

你觉得男人跟你说这些真的是在拒绝你吗？当然不是。男人都是特别怕麻烦的动物，他们要是真的想拒绝你的话，一开始就不会搭理你。

所以，他的这些狠话其实只不过是一种**废物测试**。其核心目的是：**测试你接近他的真实意图**。

他想要看看你接近他到底有什么目的：你是真的如你所说放下了，还是假装放下实际想挽回。

## 应对废物测试：递减思维打压法

这个时候，千万不要暴露自己的目的。正确的应对方法是使用**递减思维**打压回去。

递减思维的**核心公式**可以概括为：**承认事实 + 转移焦点 + 降低需求感**。

以下是具体回应的一个示例代码：

```text
我知道分手了呀，我早就放下了，真的没有想到你还这么耿耿于怀。
我只是单纯觉得跟你比较聊得来。那行吧我不说了，你自己先冷静冷静吧。
别什么事都往感情上面扯，过去的就让它过去吧。
```

当你这样说了之后，通常对方会觉得特别意外。他会怀疑自己是不是真的想多了，是不是真的误会你了。

## 测试后的关键操作：无视与重建

而你接下来要做的，就是假装这个废物测试没有发生。继续跟对方聊一些工作、生活上的事情。当然，这个聊天过程也可以与你社交平台的价值展示同步进行。

一般经过差不多35~50天的时间，这个舒适区基本上就已经建成了。

## 舒适区建成后的关系逆转

当舒适区建成以后，这个男人在潜意识里，对你的依赖感是很强的。这个时候，你再去用一些推拉的小技巧，去做一个高地位的转换，就很容易让对方主动来跟你提复合了。

我经常说，“作”其实一点都不可怕。可怕的是你有“作”的脾气，却没有“作”的能力。那这就不行了。
