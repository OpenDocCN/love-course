# 什么行为并不能挽回成功？-P1-猫先生情感挽回-BV1qP411D7bV_note

在本节课中，我们将要学习在情感挽回过程中，哪些常见行为不仅无助于挽回，反而会将关系推向更糟的境地。理解这些“无效行为”是成功挽回的第一步。

比起不知道如何挽回，更重要的是要知道什么行为并不能挽回成功，这才是重点。

## ❌ 常见的无效挽回行为

上一节我们明确了识别无效行为的重要性，本节中我们来看看具体有哪些行为属于此类。以下是几种典型的、无法挽回成功的行为模式：

*   **一哭二闹三上吊**：这种行为是在危险的边缘试探，这不是在挽回。
*   **死缠烂打带来的负能量**：这只会让你成为卑微的乞求者，这不是在挽回。
*   **沉浸于自我付出感**：在对方看来，你的行为更像是撒泼和无理取闹，这不是在挽回。
*   **不断承诺和表忠心**：这是典型的“空手套白狼”行为，这不是在挽回。
*   **不断恳求见面，见面后便开始各种质问**：这种行为同样不是在挽回。

## 🧠 行为背后的心理：失控的感性

明白了哪些行为无效后，我们不禁要问：为什么人们明知故犯？道理你都懂，可是你依然过不好这一生，主要的原因就在于，你控制不住你自己。

你之所以控制不住自己的主要原因，就是因为你的感性大于你的理智。失恋带来的痛苦让你无法承受，本能地寻求解脱，解决痛苦就成了你唯一的“救赎稻草”。然而，你所救赎的只不过是自己那痛苦的内心以及痛苦的生理反应，仅此而已。

## ⚖️ 挽回的核心前提：反人性

既然本能反应会驱使我们做出错误行为，那么正确的方向是什么？挽回的前提就是反人性。

关于“反人性”这三个字，已经强调了不下十遍，这在挽回中非常重要。这是挽回前必须打好的“第一层地基”。如果没有这一层做基础，你后面所做的一切都只是空中楼阁。

### 如何理解“反人性”？

“反人性”这三个字听起来可能有些抽象，用在通俗的话来说，就是要学会**忍**。具体公式可以概括为：

**成功的挽回 ≈ 克制本能冲动的能力**

即，忍住自己的情绪。小不忍则乱大谋，只有忍住了情绪，才有资格谈挽回。

## 💡 “反人性”在实际场景中的体现

为了让大家更明白具体是怎么一个情况，我们用一个常见的例子来说明“反人性”在挽回中的具体投射。

分手后，你给对方发消息，内容大意是：“我放下了，我们做朋友吧”，希望对方能因此接纳你。可是你说完之后就开始后悔，还没过两个小时就按捺不住情绪，转而去找对方祈求、哀悼、扮可怜等。

这种反复的行为，就是控制不住情绪。如果连最基本的一点都做不到，怎么去挽回？情绪是人的底层建筑，控制情绪、忍住情绪，首要目的就是**不让情况变得更糟**，**别让对方因为你的情绪而变得情绪很差**。这一点至关重要，是挽回过程中需要围绕的核心底层逻辑之一。

**本节课总结**：本节课中我们一起学习了情感挽回中的“红灯行为”。我们了解到，死缠烂打、情绪失控、不断施压等行为都无法挽回成功，其根源在于被痛苦驱使而失去了理性。真正的挽回始于“反人性”，即克制本能冲动，学会管理和忍耐自己的情绪，这是所有后续挽回策略的基石。记住，**稳住情绪，是挽回的第一步，也是最关键的一步**。
