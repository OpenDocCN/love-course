# 情感挽回课程 01：断联期间的核心纪律 🚫

在本节课中，我们将要学习情感挽回中“断联”阶段的一个关键原则：彻底避免视监对方。许多人在开始断联后，会陷入严重的情感戒断反应，其根本原因往往与无法停止关注对方动态有关。我们将详细分析这一行为的危害，并提供清晰的行动指南。

---

近期，许多人反馈在分手后选择或被迫开始断联，短短几天内就出现了严重的情感戒断反应。他们寝食难安，感觉断联的三天如同一年般漫长，每一分每一秒都是煎熬。

这些人每天会几十次地刷新所有能查看对方社交动态的APP、软件甚至游戏。他们对前任进行全方位的监视，猜测对方当天的行程，是工作还是娱乐，此刻正在做什么。同时，他们会回想曾经同时刻两人的幸福时光，并陷入“为何会变成这样”的痛苦中。

随后，他们控制不住再次刷手机，结果阅读了许多别有用心、渲染负面情绪的文案。越看情绪越崩溃，最终可能再占卜一次，然后彻底崩溃，询问能否去联系对方、发送信息或见面。

既然决定挽回，就需要决心。如果连这件事都无法忍耐并做好，如何成功挽回？仅仅断联几天，这算不上真正的断联。那些断联半年或一年的人是如何度过的？他们难道不吃饭、不工作、不过日子了吗？这种行为确实可能坏事。

然而，这种心态对于急于挽回的人来说是正常的。但现在这个时间点确实不合适。通常，真性分手的到来意味着事态已经严重化，尤其很多人已经折腾过，甚至有的人是保证了不再骚扰对方，才勉强未被全部拉黑。

这才断联几天就又想骚扰对方，这无疑是错误的行为。既然情况已很严重，你的纠缠骚扰已被视为不当，并且你已被迫开始断联以让双方情绪缓和，那么这个过程总需要一些时间。

试想，如果换作是你，有人让你感到非常厌烦，你能在几天内就忘记他的不好并原谅他吗？肯定需要等到负面情绪冷静下来，这个时间至少也需要七天左右，这甚至还算不上严格的断联。因此，小不忍则乱大谋。

最后，强调几点。断联期间，偶尔了解对方大致情况是可以的，但不要天天盯着对方看。失恋时人的判断力容易下降，不要整天猜想他在做什么，也不要回忆曾经此时二人世界的幸福。

既然已经分手，就不要再空想。可以这样想：反正之后是要复合的，那么现在浪费时间做这些事有什么用？不如去努力工作、学习。是钱已经赚够了吗？等对方回来时，你有结婚的房子吗？有谈判的资本吗？

搞这些无用的东西，只会浪费你的精力，让你的情绪更糟。这段冷静期，你需要做的是管好自己，调整状态，变得更优秀。这样，在即将到来的复联阶段，你才能以更好的状态和心态去应对。

---

本节课中，我们一起学习了断联期间必须遵守的核心纪律：**坚决避免视监对方**。我们分析了视监行为如何加剧痛苦、阻碍情绪平复，并强调了利用断联期进行自我提升的重要性。记住，成功的断联是为有效的复联做准备。
