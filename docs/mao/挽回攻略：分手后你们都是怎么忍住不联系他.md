# 情感挽回教程 第1课：分手后如何控制联系冲动？💔

在本节课中，我们将学习分手后为何会产生强烈的联系冲动，以及如何理性地控制这种冲动，为后续的挽回或自我成长打下基础。

分手后最痛苦的感觉之一是感到无能为力。每当静下来时，总会反复查看对方的微信、微博、抖音等社交软件。想联系对方却又不敢，害怕仅存的联系方式被删除或拉黑，更不知道该说些什么。

那些道歉、保证和解释的话已经说过无数遍，却没有任何回应。于是理智告诉自己暂时不要联系，但想联系的心却像被无数蚂蚁撕咬，又疼又痒。有时甚至会想，不如来个痛快，也算是一种解脱。

## 为何分手后难以忍受不联系？

恋爱时几天不联系都可以，为何分手后却无法忍受？在我看来，忍不住想联系对方无非出于两个核心原因。

### 1. 未知感在作祟

恐惧来源于未知。恋爱时，彼此拥有一种“所属权”关系，即 **我属于你，你也属于我**。因此即使发生矛盾也无所谓。但真正分手后，在不联系的时间里，我们无法得知对方的消息，从而产生一系列困扰。

他会开始新的恋情吗？这么久不联系，他会忘记或放下我吗？他心里还有我吗？这些低端问题会不断涌现。为了获取答案和安全感，自然会产生迫切联系对方的冲动。

这种情况在分手后很常见。我称之为“低端”问题，是因为它们通常只出现在分手后患得患失的状态中。分手后思绪混乱，各种想法涌现，这完全可以理解。

但理解之余，我们需要理性思考：你所担心的那些问题真的会发生吗？如果会发生，你的担心、忧虑和难受有用吗？既然没用，何必去想？

**关键公式：**
```
分手状态 = 双方成为自由人
他的行为 ∈ 他的自由范畴
你的担忧 ∉ 可影响他的因素
```

当你们分手的那一刻，双方就已经是自由人了。他做的任何事情都与你无关。如果你仍在纠结这些问题，说明你的内心尚未接受分手的事实。从基本思维层面，你们就无法达成共识。

思维基础存在冲突，怎么可能建立良好的沟通呢？**接受分手的事实是你必须迈出的第一步。**

### 2. 想满足自身情感需求

能够理解，分手后的人非常痛苦和难受。在这种痛苦中，似乎没有任何解药能缓解情绪，唯独那个分手的人——似乎只要他愿意和自己说说话，痛苦就能减轻。

这如同戒烟者对烟的渴望。但你们已经分手，对方没有义务继续为你的情绪负责。

因此，当我们忍不住想联系对方时，需要思考联系的目的：是为了继续挽回，还是因为自身情绪再次波动，不想继续痛苦下去才想联系他？

正如前文所述，分手后双方已是自由人。对方没有责任安抚你的情绪，你也没有权利再次向对方索取情绪价值。

## 如何回归成年思维并自我成长？

现在，让我们回归成年人的思维模式：有些情绪必须自己消化和调理。只有先成为一个能独立承担的人，才能形成属于自己的独立人格魅力。

在此基础上，才能复盘恋爱中自己的不足和缺点，进而改变自己。一段感情若没有好结果，至少应让自己在不完美的恋爱中完善情感能力。

这样，无论是挽回还是开始新感情，我们才不会重蹈覆辙。

外界常有声音说：“分都分了，还纠缠什么？”但在我看来，这些人并未理解透彻。他们只看到分手带来的痛苦和问题，却忽略了你们之间幸福的过程和回忆。

许多分手的原因，说穿了是可以改变的。只要不触及底线（如出轨或家暴），其实都有复合的可能，并且感情不会比之前更差。

犹豫的人往往害怕复合后出现同样的问题。其实不会。分手会改变两个人的观念：以前不成熟的人会变得成熟，以前只会作闹的人也会学会体谅。

既然彼此都还有感情，为何不再尝试一次？复合需要技巧和时间。
