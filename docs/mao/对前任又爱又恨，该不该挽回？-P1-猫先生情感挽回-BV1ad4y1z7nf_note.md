# 情感挽回课程 P1：🤔 对前任“又爱又恨”，我该挽回吗？

在本节课中，我们将要学习当对前任产生“又爱又恨”的复杂情感时，如何理性分析现状并找到正确的处理方向。我们将探讨这种矛盾情感的成因，并学习如何通过有效沟通引导关系走向积极发展。

## 矛盾情感的成因：主观与客观的“墙”

喜欢一个人时，有些人会受到主观意识的强烈影响，有些人则更容易被客观意识左右。

为什么这么说呢？从现实角度出发，你喜欢一个人的时候，可能世界里只有他。但如果这个时候旁边有很多的声音在和你说：“别喜欢他，他不值得你喜欢，他根本配不上你。”这些话很有可能会直接影响到你的判断。

## 外部声音如何影响你的判断

你会把身边人说的这些话落实到对方的身上。如果对方一开始还没有察觉到你在观察他，他是不会短时间内去改变自己的行为习惯的。而你就慢慢的开始怀疑自己的选择是否做错了。

对方好像真的像身边人说的一样，你的主观意识就开始发生了改变方向，对对方的感情也是忽冷忽热，甚至出现了又爱又恨的情况，根本不知道自己该怎么办。

## 核心原则：你的感受才是关键

如果出现这种情况，你就不要当下就定论或者决定对方是不是不爱你了，因为在和对方谈恋爱的人是你，不是旁人。别人对他的了解除了通过你，就是他们自己所猜想出来的。

如果对方确实做了一些不好的行为，那么你要做的就是引导他改正不好的行为，而不是一味的指责他，这样做不行，那样做不行。久而久之，你只会越来越厌恶这个人，但是因为你自身长期的投入和依赖性，你会发现你又离不开他。

## 从指责到引导：改变沟通模式

如果这个时候对方选择了放弃，那挽回的人就是你了。所以，在你发现这个问题的时候，你需要做的就是如何引导对方解决问题，而不是光想着问题变得越来越严重，越来越讨厌对方了。

上一节我们分析了外部影响和沟通误区，本节中我们来看看如何通过具体方法引导对方改变。

怎么引导对方改正自身问题其实并不是非常难的事情，关键在于你要有耐心并且懂得给予对方奖励。建立讲话机制是一个很不错的方法，做对的事情就奖励，做错的事情就惩罚，天下没有免费的午餐，道理都懂。

以下是实施引导策略时需要注意的几个关键点：

*   **把握分寸，循序渐进**：不能一下子就妄想着把对方全部不好的全改了，而且按照你的想法去改，这样对方心里也会有怨言的。
*   **聚焦行为，而非人身**：针对具体行为进行沟通，避免使用“你总是…”、“你这个人就是…”等全盘否定的表述。
*   **及时反馈，正向激励**：当对方做出你期望的积极改变时，及时给予肯定或奖励，强化正向行为。

## 管理情绪，回归关系正轨

又爱又恨这种感情确实是非常纠结的，但你不要因为自身的情绪而影响了你们彼此之间的感情。一旦你开始质疑对方，怀疑对方的时候，对方同样也会怀疑你的不足。

怀疑这段感情是否应该继续下去，你要做的就是让两人之间的感情回到正轨上，让两个人都越来越爱彼此。

本节课中我们一起学习了当陷入“又爱又恨”的情感矛盾时，应如何应对外界干扰、停止无效指责，并转向通过耐心引导和有效沟通来解决问题。记住，关系的改善始于你自身沟通模式的改变，核心在于将对抗转化为合作，共同让感情回归积极发展的轨道。
