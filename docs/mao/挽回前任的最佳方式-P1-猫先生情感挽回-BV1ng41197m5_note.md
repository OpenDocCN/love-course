## 情况一：对方当面提出分手 🤝

上一节我们介绍了“当面”的重要性，本节中我们来看看当对方当面提出分手时，你应该如何应对。

对方当面表达了希望和你结束这段关系的想法。无论你们分手的原因是什么，你首先要做的就是控制好自己的情绪，不要被负面情绪支配了你的行为。

很多无法复合的感情，并不是导致分手的原因，而是错误的纠缠使形象彻底恶化。没有人会想推开一个爱自己的人，除非他已经攒够了不满和委屈。

如果此刻你歇斯底里的死缠烂打，拒绝分手，在他感受到的并不是你的深情和不舍，而是你只顾自己的感受，却否定了他的委屈。这会越发觉得你无法理解到他，从而更加坚定了分手的决定。

### 核心行动指南

以下是面对当面分手时的核心行动步骤：

1.  **控制情绪，保持冷静**：不要带负面情绪，温柔冷静地与对方沟通。
2.  **表达理解与共情**：向对方表达你是可以理解到他的感受的。
3.  **使用共情技巧**：把对方的感受从你嘴里讲出来。例如，你可以说：“你一定很委屈很痛苦吧，怪我没能照顾好你，让你产生了这样的念头，我不是一个称职的男朋友/女朋友。”这样会使对方产生很强的认同感。
4.  **避免对立**：当一个人感受不到你的对立情绪时，他对你的抵触感和防范心理都不会太强。
5.  **注意话题边界**：你可以自我披露自己感情中的问题，甚至可以尝试聊风轻云淡的话题，但**千万不要主动提感情**。

## 关系升级的关键：肢体进挪 💞

上一节我们学习了如何稳定情绪和有效沟通，本节中我们来看看如何将关系推向和好。

当感受到对方的情绪平稳时，可以进行“肢体进挪”。关系是否升级的重要指标，就是自然状态下的肢体接触的程度。

**注意循序渐进**。例如：
*   轻轻拍背。
*   可以的话就搂着。
*   搂着可以的话就能摸到他的脸颊，帮他擦眼泪。
*   擦眼泪可以的话就能亲吻等等。

一切要自然。如果对方表现排斥就不要继续进挪。

**核心公式**：`关系升级 ≈ 自然的肢体接触程度`

到了这一步后，就算你们当时没和好，之后也断不了。
