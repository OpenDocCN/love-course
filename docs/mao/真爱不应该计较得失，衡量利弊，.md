# 真爱不应该计较得失，衡量利弊，

在本节课中，我们将要学习在感情中关于“主动”与“计较得失”的核心观念。我们将探讨主动是否意味着处于劣势，以及如何建立健康的感情观。

## 主动意味着“输”吗？🤔

在感情中，主动的一方就一定会输，这种观点并不正确。面对爱情应该主动。如果你想拥有一份爱情，反而还计较得失，在乎输赢，那么这份爱情还没开始就已经存在很大的问题了。

这时候你应该问问自己，你真正要的是爱情吗？如果是，那就不应该太在意得失。

## 明确感情的核心 💖

至少你应该相信那个人。如果你想要的是被重视、被呵护，想要的是对方为你付出很多，那这样的感情并不值得珍惜。因为你们两个人之间，感情本身不是主要的，而是“谁付出多”才重要。

上一节我们探讨了主动的意义，本节中我们来看看主动在关系中的真实位置。主动不代表着谈恋爱的时候就要处于弱势。

## 主动的本质是选择权 ⚖️

主动就是争取。当然，面对爱情，既然你可以主动争取，也可以主动放弃。如果对方不值得你争取，你随时可以主动放弃。

什么是输，什么才是赢？你应该想清楚自己需要的是什么，然后朝着你想要的结果主动去争取就可以了。

## 成功的关系与对等承诺 🤝

但话说回来，如果真的是一个懂得爱你、懂得珍惜你的人，你并不是失败者，你们都是成功者。所以，**选对人很重要**。

其实我们并不是追求爱情绝对平等，因为我们都明白，哪有完全平等的爱情呢？总有一个人更喜欢对方多一点。

我们希望的是有对等的承诺、对等的付出的感情，而不是让人觉得只有我一个人在意这段感情。那样也就不会在意承诺了。

## 总结 📝

本节课中我们一起学习了：
1.  主动去爱并非意味着失败或处于弱势。
2.  健康的感情应基于感情本身，而非过分计较付出与得失的权衡。
3.  主动的本质是拥有争取和放弃的选择权，关键在于明确自己的需求。
4.  成功的关系在于双方的对等承诺与珍惜，选对人是基础。
