# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号`=`来给变量赋值。

其基本语法是：
```python
变量名 = 值
```
例如，`name = “一只橙”` 这行代码创建了一个名为`name`的变量，并将文本“一只橙”存放进去。之后在程序中，我们就可以使用`name`来代表这个文本。

## 基本数据类型 🔢

理解了如何存放数据后，我们需要知道能存放哪些类型的数据。Python有几种内置的基本数据类型，它们决定了数据可以进行的操作。

以下是三种最常用的基本数据类型：

1.  **整数**
    整数是不带小数点的数字，例如 `-10`, `0`, `100`。

2.  **浮点数**
    浮点数是带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。

3.  **字符串**
    字符串是由零个或多个字符组成的文本，需要用单引号`‘ ’`或双引号`“ ”`括起来，例如 `‘Hello’`, `“Python”`。

## 如何查看和转换类型？ 🔄

在编程时，我们经常需要知道一个变量的类型，或者在不同类型之间进行转换。

*   **使用 `type()` 函数查看类型**
    我们可以使用 `type()` 函数来查看任何变量或值的数据类型。
    ```python
    print(type(100))      # 输出：<class 'int'>
    print(type(3.14))     # 输出：<class 'float'>
    print(type("文本"))   # 输出：<class 'str'>
    ```

*   **使用类型转换函数**
    有时我们需要将一种类型转换为另一种类型，这时可以使用以下函数：
    *   `int()`：将数据转换为整数。
    *   `float()`：将数据转换为浮点数。
    *   `str()`：将数据转换为字符串。

    例如：
    ```python
    number_str = "123"
    number_int = int(number_str)  # 将字符串"123"转换为整数123
    print(number_int + 7)         # 输出：130
    ```

## 动手试一试 ✍️

现在，让我们将以上概念结合起来，完成一个简单的练习。

以下是创建一个变量并操作它的步骤：

1.  创建一个变量`my_name`，存储你的名字。
2.  创建一个变量`my_age`，存储你的年龄。
3.  使用`type()`函数检查这两个变量的类型。
4.  将年龄（整数）转换为字符串，然后与名字拼接成一句话。

示例代码：
```python
my_name = "一只橙"
my_age = 25

print(type(my_name))  # 输出：<class 'str'>
print(type(my_age))   # 输出：<class 'int'>

# 将整数年龄转换为字符串后拼接
greeting = "大家好，我是" + my_name + "，我今年" + str(my_age) + "岁。"
print(greeting)
```
