# 一只橙框架课-399：第21课，如何培养老公对家庭的责任感？_note

在本节课中，我们将学习如何在婚姻中培养伴侣的责任感。责任感是维系家庭和谐、促进情感稳定的核心要素。我们将通过四种具体的方法，帮助你引导伴侣更主动地承担家庭责任，共同营造幸福的婚姻生活。

## 🤝 第一节：共情——建立情感连接的桥梁

上一节我们介绍了责任感的重要性，本节中我们来看看如何通过共情来培养它。共情是指在人际交往中，能够理解和体会他人的情绪与感受，对他人的处境有同理心。在婚姻中，缺乏共情会导致伴侣在家庭责任上的缺失。

许多女性会困惑，为何与丈夫无话可说，感觉聊天难以深入。这通常是因为双方未能建立有效的共情。缺乏共情，便难以敞开心扉交流，对话会变得枯燥，这可能是由于技巧不足或信任缺失造成的。

婚姻从浪漫过渡到现实后，热恋感会逐渐淡化，此时需要依靠信任搭建桥梁。在日常生活中，男性需要伴侣的关心，双方在做任何事时都应考虑彼此感受，养成相互体谅的习惯。这种习惯在心理学上被称为共情。

那么，如何与伴侣共情呢？以下是一个具体示例：

我曾有一位女同学做得非常出色。她怀孕初期，夫妻二人对新生儿充满期待。但她妊娠反应严重，孕吐持续了三个月，甚至喝水都会呕吐。她每天给丈夫发信息：“老公，今天好难受，一直吐，一口饭都吃不下。”丈夫看到她如此辛苦，非常心疼，深刻感受到了她的不易。

怀孕中后期，胎儿长大，她的脚肿得厉害，全身不适。她将这些感受委婉地告诉丈夫，有时脚还会抽筋。丈夫便开始上网搜索如何缓解孕妇不适、如何按摩，并立即付诸行动为她按摩。按摩时，她会表达感激：“老公你好贴心，嫁给你真的很幸福。”

之后，她和丈夫一起学习胎教知识，如何与宝宝沟通，以此培养父亲与孩子的亲密关系。关键在于理解对方的痛处，并委婉表达自己的感受，不要让对方觉得你是一个凡事都能自己扛的女强人。女性天生具有激发男性保护欲的优势，不应浪费这一点。这就是良好的共情。

## 🧩 第二节：依赖——在共同体验中升华情感

理解了共情的作用后，我们来看看如何通过依赖感来加深情感连接。情感是在依赖与被依赖的过程中升华的。一个人能做的事，也可以两个人一起做。在共同做事的过程中，双方会形成共同的体验，建立起牢固的依赖感。

这里需要强调，向伴侣表达依赖感，并非意味着你离不开他，尤其在情感稳定后，你可以适当表现出这种依赖。不要事事亲力亲为，可以邀请他一起晾衣服、洗碗，共同完成家务。情感正是在这种依赖与被依赖中得以升华的。

继续以我那位女同学为例。怀孕期间，她没有独自阅读育儿书籍，而是让丈夫一起看。之后，她让丈夫在网上挑选奶粉，并极力听从他的建议。她为何要这么做？她自己当然能做决定，但男性通常有征服欲和占有欲。丈夫选完奶粉后，她会夸奖他：“老公你好厉害，选的这个牌子对宝宝特别好。”她会流露出惊喜、赞美和欣赏的目光，投以信赖的眼神。作为一家之主，丈夫自然感到需要承担责任。

于是，丈夫每次从婴幼儿用品店回来，都大包小包地提着，即使她想帮忙拿，丈夫也不让，说：“你好好休息，这些粗活我来。”这位女同学还很聪明，经常给丈夫发送育儿文章，灌输“父亲参与培养对孩子很重要”的观念。整个孕期，她坚持敷面膜、注重衣着打扮，看起来不像孕妇，倒像一位肚子稍大的美人。我们常调侃她：“你哪像怀孕的人，活得这么滋润。”她正是在一步步培养丈夫的责任感。

## 🌟 第三节：鼓励——用正向反馈强化行为

学会了建立依赖感，接下来我们探讨如何通过鼓励来强化伴侣的积极行为。鼓励是指当对方做某事时，给予正向的反馈。我的这位女同学本是女强人，非常能干。有一次她工作时，丈夫发来短信：“一定要照顾好自己，路上小心。”按理说，成年人无需这种提醒，但她的回复方式值得学习。

她是这样回复的：“在你眼里，我永远是个不懂照顾自己、没有生活自理能力的小白。我现在已经是大人了，你还要这样盯着我吗？可是我好喜欢这种感觉，好喜欢被你关注的感觉。老公我爱你。”这是一种推拉技巧：先表示自己已是大人，无需过度担心，随即表达渴望被关注。她用温柔可爱的语言传达了被关心、被呵护的满足感。

她越是如此，丈夫就越激发起强烈的保护欲和责任感，对她的关爱也更加无微不至。两人的恩爱是有原因的。没有天生的渣男，男人有时可能比较固执，但通过恰当引导，可以变得更好。各位姐妹，请注意不要将对方推向冷漠。

聚会时，我们常问她：“为什么你老公对你这么好？有什么秘密？”其实要点就是：共情、依赖、鼓励。她经常鼓励丈夫与腹中宝宝说话：“你看，宝宝踢我肚子了。爸爸声音真好听，来跟他说两句。”这就是鼓励。鼓励不在于奉承或赞美，而在于强化对方的行为。一个人的行为之所以被强化，正是因为你的鼓励和奖励。

宝宝出生后四五个月，她开始鼓励丈夫带宝宝。她对丈夫说：“老公，宝宝四个月了，小男孩不应该总是妈妈洗澡，应该和爸爸一起洗。”于是丈夫带着孩子洗澡。她又鼓励丈夫喂饭：“爸爸喂饭，宝宝吃得更香。你看，宝宝更喜欢爸爸喂。”她持续鼓励，巧妙引导。现在丈夫已非常热衷于给孩子喂饭、做游戏、陪早教、洗澡。爸爸的心就这样被孩子牢牢吸引。

有一次，婆婆给孩子洗澡时孩子哭闹，爸爸在书房工作，一听到哭声立刻放下手头工作跑去洗澡。孩子一看到爸爸就不哭了。至此，父亲对孩子的责任感已培养起来。男人对婚姻的责任感随之加强，会更珍惜感情。试想，如果你将丈夫培养成这样的好爸爸，他的身心都放在孩子和你身上，外出工作时也会惦记家人，哪还有心思找小三？这是一个非常明智的方法。

## 🎉 第四节：炫耀——通过公开认可塑造形象

最后，我们来看看如何通过炫耀来巩固伴侣的责任感。炫耀是指女性要懂得适当曝光自己的生活，并公开称赞丈夫。我的这位女同学做得很好，她经常向朋友炫耀丈夫对她的好。

有一次，她在朋友圈这样写道：“今晚突然想吃鸡汤挂面，一整天没吃东西，就给加班的老公发了信息。他看到后马上回复，并去给我买了。真的很感动。”她接着写道：“老公你怎么对我那么好，长得帅，身材好，给我钱，还对我有求必应。女人常因小事感动。你给我买几百万的车我不一定心动，但我想吃鸡汤挂面时你给我买，我就觉得特别暖心。老公，我很感激，很爱你。”

这条朋友圈获得无数点赞，大家纷纷称赞她丈夫。连我们的男同学都觉得应该向她丈夫学习。后来，她丈夫在公司也被同事夸奖，他非常自豪，认为宠爱妻子是正确的事，于是对妻子更加关心。

这就是我们常说的，夸奖对方时，其实是在进行“认证”。当你给外界塑造了“他就是这样的人”的印象时，他就无形中被贴上了这样的标签。外界的认可和内心的道德感会促使他培养责任感。同时，他是被你激励、夸奖和炫耀的，不会感到压力或痛苦。这种由内而外的责任感、义务感和自豪感会自然产生。当他看到你生病时，就会觉得有义务、有责任陪伴你。这就是将丈夫培养成有责任感的男人的方法。
