# 一只橙框架课-399：第10课，占据爱情高位的身份框架五大招式（下）_note

在本节课中，我们将继续学习构建身份框架的五大招式，重点掌握后三招：巧用约法三章、建立奖惩机制以及引导投资框架。这些方法旨在帮助你建立清晰的情感边界，掌握关系主导权，让伴侣更加珍惜你。

上一节我们介绍了地位扭转和高阶洗脑，本节中我们来看看如何通过具体的互动规则来巩固框架。

## 第三招：巧用约法三章

“约法三章”听起来可能带有束缚感，但这里的爱情契约并非冰冷的条款。其核心作用是：**在感情浓度高时，共同建立一些默契的规则，让彼此在未来的相处中有章可循，减少摩擦，增加情感链接**。它不是计较与束缚，而是为了让爱更顺畅地流动。

以下是通过案例理解如何建立“甜蜜契约”：

我的一个闺蜜，她的家庭有一种独特的甜蜜契约。在她的家族群里，父母互称“小美”和“小帅”，而她与丈夫则互称“老头”和“老太太”。这种称呼源于恋爱时她的一句玩笑：“我要当你的老太太，你就是我老头儿。”这成了他们之间持续十多年的甜蜜默契。

这个例子说明，爱情契约可以融入日常，成为细水长流中的温暖习惯，而非强制义务。

许多夫妻矛盾源于育儿观念不合等问题。聪明的做法是在感情最好、浓度最高时，就未来可能的分歧提前达成共识。

你可以选择一个双方情绪都好的时机，用温和而坚定的态度提出你的想法。例如，可以这样沟通：

> “公司有位大姐，因为孩子教育和婆婆闹矛盾，夫妻关系都僵了，我看着很心疼。我在想，两代人观念肯定不同，为了避免我们以后也陷入这种琐事，有些事咱们是不是可以提前说好？比如，关于孩子的教育，我们俩必须共同学习，达成一致。如果老人用了不科学的老方法，我们有责任一起温和地沟通，帮助调整。咱们可以约定每周开一次家庭会议，重大决定一起商量。”

这种方式是在最佳时机，借他人案例引发共鸣，从而共同建立自家的“爱情契约”。

关于建立契约的细节，例如去对方父母家是否要做家务，也可以提前约定。

例如，约定“去我家吃饭，你负责帮忙；去你家吃饭，我多出力”。这样，双方父母看到的是你们为彼此着想的举动，反而更能感受到你们的幸福与和谐。

当关系出现矛盾，例如频繁冷战时，建立契约更是修复关系的关键。

我曾指导一位因频繁冷战而痛苦的姐妹。她过去的回应方式是抱怨和指责，结果冷战愈演愈烈。我帮助她转换思路，学习在对方道歉时，用建立未来契约的方式来回应。

以下是一个话术框架，核心是**表达感受、肯定对方、共同展望、提出具体约定**：

> “看着孩子们，我有时会发呆。我们明明相爱，为何却总在互相伤害？一个女人如果不爱你，不会接连为你生下三个孩子。生产时的剧痛，如果没有爱，是坚持不下来的。你的冷战，像一把钝刀在割我的心。所以，谢谢你今天的道歉和尊重。我相信这是我们最后一次冷战。我们都想给孩子更好的家，让老人少操心。我反思自己，可能太关注孩子忽略了你。以后，我们每周再忙也一起看场电影，每天至少交流15分钟，分享趣事或讲个笑话。我们约定，吵架不过夜，如果当时不想说，可以发消息留言，第二天谁早起，谁就给对方挤牙膏，作为和好的信号。”

这位姐妹使用后，关系迅速回暖。这些约定看似简单，但**持之以恒地执行**，才能让爱有序而稳固。

**总结**：爱情契约不是谈判，而是在爱的基础上，共同绘制一份让彼此都更舒适、关系更紧密的路线图。

掌握了如何建立共同规则后，我们来看看如何通过明确的反馈机制，来强化这些规则并掌控主导权。

## 第四招：建立奖惩机制

奖惩机制的目的是**让对方适应你的原则，而非你不断妥协去迎合对方**。通过及时、明确的反馈，你可以引导对方的行为，巩固你的框架。

许多姐妹在关系中不断降低底线，原谅对方的屡次迟到、忘记重要日子、甚至更严重的伤害。这会导致对方犯错成本变低，行为越发过分。

相反，对于对方好的付出（如体贴的行为），如果总是习惯性无视，没有任何正反馈，对方也会失去持续付出的动力。

**核心公式**：**对方行为 → 你的及时反馈（奖励/惩罚）→ 行为强化/修正**

以下是具体操作方法：

1.  **当他做对时，不要吝啬奖励**：
    *   语言肯定：“哇，老公你好厉害！”
    *   肢体亲密：猝不及防的吻。
    *   行动关怀：多做两道他爱吃的菜、帮他敷面膜、按摩等。

2.  **当他做错时，敢于立规矩并惩罚**：
    *   **惩罚与错误程度匹配**：例如对方约会无故迟到10分钟。
    *   **话术示例**：“看你还没到，我心里有点失望和落空。下次有突发情况，至少打个电话告诉我一声好吗？今晚你表现差评，原定的‘福利’要延迟发放了哦。”
    *   这种表达既表明了态度（惩罚），又留有撒娇和好的空间，气氛不僵。

你必须在一段关系中建立清晰的认知：

> **我是你的爱人、家人、良师、知己，但不是你的保姆、奴隶或妈妈。**

建立并严格执行奖惩机制，是贯穿情感各阶段、维护身份框架不可或缺的一环。

通过奖惩机制规范了日常行为后，我们将学习最后一招，从更宏观的层面让伴侣对你的投入越来越深。

## 第五招：建立投资框架

投资框架的核心是**引导对方为你投入时间、金钱、精力、情感，并且是心甘情愿、持续不断的**。投入越多，他就越难以离开这段关系。

那些让男人不断付出的女人，就像一款引人入胜的游戏或抓娃娃机，总是差一点得到，让人欲罢不能。她们具有“不可得性”和“渐进式奖励”的特质。

一个小故事：小男孩不自己抓糖，而是让手更大的老板抓，最终得到更多。这告诉我们：**在感情中，巧妙引导对方主动给予，远比你自己拼命索取来得更有效、更持久。**

以下是建立投资框架的完整逻辑：

1.  **持续自我提升**：这是吸引投资的基础。
2.  **大胆且有技巧地引导投资**：不要害怕接受付出。
3.  **追求投资的全面性**：引导对方在时间、金钱、情绪上都有投入。
4.  **关注“稀缺性投资”**：对方缺时间，那么他挤时间陪你比花钱更有价值；对方经济一般，那么精心准备的礼物比昂贵礼物更显用心。
5.  **给予正反馈**：对对方的投资行为给予肯定、鼓励和情绪价值，让他有持续投资的动力。
6.  **投资是相互的**：你也要提供对方所需要的独特价值，形成良性互动。

以下是一些引导投资的具体场景话术：

*   **场景一**：他为你点了下午茶。
    *   **行动**：拍照发朋友圈（仅对他和共同朋友可见）。
    *   **配文**：“今天这杯奶茶下肚，同事说我说话都糖分超标了！[偷笑]”
    *   **效果**：给予情绪价值，鼓励他重复该行为。
*   **场景二**：他下厨做饭。
    *   **行动**：开心地“偷吃”，认真摆盘拍照。
    *   **配文**：“有句话叫，当男人愿意为你拿起锅铲，就不必再说‘我爱你’了。”
    *   **效果**：高度捧高他的行为，满足其成就感。

爱需要空间，就像相声有逗哏和捧哏。不要填满所有投入的空间，有时需要“退一步”，让对方有主动付出的机会。做一个风情、灵动、知情识趣的伴侣，让爱鲜活而高级。

**本节课总结**：

在本节课中，我们一起学习了构建身份框架的后三大招式：
1.  **巧用约法三章**：在感情好时，用爱建立共同规则，为未来减少摩擦。
2.  **建立奖惩机制**：通过及时反馈（奖励与惩罚）规范行为，守住底线，掌握主导权。
3.  **建立投资框架**：引导对方持续、全面、心甘情愿地为你投入，让他越来越离不开你。

结合上节课的“地位扭转”与“高阶洗脑”，这五大招式能帮助你牢牢建立起属于自己的身份框架，在关系中占据高位，收获更健康、更甜蜜的爱情。
