## 概述

投资是一个有规定的流程，其核心在于引导对方从想法开始，进而产生行为，最终引发情绪。这个过程是建立吸引和连接的基础。如果顺序错误或缺失环节，则难以让对方真正对你产生兴趣或愿意提供帮助。

## 投资的流程规定

上一节我们概述了投资的重要性，本节中我们来看看其具体流程。投资必须遵循一个规定的顺序：**想法 → 行为 → 情绪**。

**流程公式：** `吸引 = 想法 -> 行为 -> 情绪`

为什么这是一个规定？因为你无法直接让对方心中接纳你。如果她不欣赏你或不想帮助你，你就无法让她投资于你的行为或认同你的观点。因此，你必须先让她在心中接纳你。

这是一个必须遵守的过程。首先影响她的想法，接着她会做出相应的行为来帮助你，然后她的情绪会逐渐投入。例如，一个女人意识到“我喜欢这个人”，并不是通过理论推导，而是通过感受。她会发现自己经常想念对方，当对方不在时会感到痛苦，从而确认自己的喜欢。

所以，正确的顺序是：思考先行，然后是行为，最后是情绪。

## 投资的四个方法

理解了流程规定后，我们来看看实现投资的具体方法。以下是四个关键方法：提示、挑战、做好、行动。我会逐一解释。

### 1. 提示
提示类似于开玩笑或给出一个小暗示。这是最初的情绪开放，目的是引发思考。

**提示示例代码（对话模拟）：**
```
你：“你太矮了吧？” （这其实接近挑战）
你：“你会有白人女孩的纹身吗？”
你：“你会有变性人邮票吗？”
```

提示是一种轻微的挑战，旨在开启情绪和思考。例如，问对方“你会是个甜心吗？”，或者说“我朋友说我们合不来，但我不确定，我们来看看”。

### 2. 挑战
挑战比提示更进一步，是一个小小的社交测试。

**挑战示例：**
```
你：“你读医学？他们都说读医的人都很……（让她证明并非如此）”
```

### 3. 做好
“做好”指的是展示价值或能力，但这部分在原文中未详细展开，通常指通过自身表现吸引对方。

### 4. 行动（请求帮助）
这是让对方向你投资行为的关键。提出小的请求，请对方帮助你。

**行动投资示例：**
```
你：“我背好痛，鞋带松了，能帮我一下吗？”（无论她同意与否，这都是一种投资测试）
你：“我肩膀很酸，一直在用电脑，你能给我按摩一下吗？”
```

在约会中，可以请求对方：
```
“我找到了这个关于学校/工作的计划，你能把相关资料发邮件给我吗？”
“能帮我拿一下这杯饮料吗？”
```

许多RSD学员在约会时，会让女生帮忙拿相机、调整摩托车后视镜、设置单反相机或手机焦点等。这些小事都是让女生进行投资。投资应在互动早期（比如五分钟内）开始引导，而不是最后。

## 投资与回报

上一节我们介绍了如何引导投资，本节中我们来看看投资与回报的关系。当你引导对方投资后，自然会考虑给予回报。

很多男生会问：“我应该请女生喝酒吗？”这取决于她是否先进行了投资。如果她愿意投资，比如愿意帮你一个小忙、打败朋友来见你、或在谷歌上帮你查资料，那么你可以请她喝酒。这是一种交换。

但核心是，首先要创造机会让对方投资。性也是一种投资。当女生在发生关系后，如果感觉自己从零投资跳跃到100%投资，可能会感到不适，认为投资增长过快。如果前期投资不足，她事后可能会感觉不好。

例如，如果你的前女友在第二次约会时就带酒下厨，那是一种投资。如果投资不足，女生可能不会珍惜你们共处的时间。

## 你的价值是投资的基础

要让别人愿意投资，你自身必须拥有可提供的价值。你的价值来源于你的经历。

**价值公式：** `你的价值 = 你的故事 + 你的观点 + 你的经历 + 你的知识`

即使你目前的人生尚未达到高峰，你依然拥有价值。你经历的一切、学到的教训、获得的知识，都是你能带给别人的价值。你值得别人投资。

例如，你的一位朋友在比特币投资中损失巨款，但他能提醒他人避免被骗，这个故事就具有价值。他榨取了经历中的“柠檬”，做出了“柠檬汁”。所以，你总能从经历中提炼出价值。

## 关于时间与情绪的投资

有朋友问，投资是关于时间的吗？与朋友的互动是怎样的？这很重要。

我可以公开谈论性，但如果没有情绪上的连接，那只是分享信息。当女生在不同的社交场合，如果她愿意离开自己的朋友群体与你相处，那就是一种重大的投资。例如，她带领团队，或者选择留在你身边而非朋友那里，这都是投资。

然而，并非所有深度交流都是投资。女生向你倾诉私人情绪问题（比如趴在你的肩膀上哭泣），可能只是一种情感宣泄，利用你作为倾诉对象，而不一定是投资。真正的投资体现在她接受你的挑战或为你付出行动。

女生请你吃晚餐，可能比性行为更具投资意义。因为共进晚餐需要一个明确的理由和承诺，而性行为可能被事后归结为“一时冲动”、“喝醉了”或“在派对氛围下”。前者更难以用借口开脱。
