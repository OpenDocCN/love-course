# 03.麦迪森BOSS【萌萌团队-皮卡内部提供】：核心内容：21BOSS搭讪-基于角色的搭讪_note

在本节课中，我们将学习一种名为“基于角色的搭讪”的核心方法。我们将探讨如何通过塑造一个清晰、有吸引力的个人角色来提升你的社交互动能力，并理解内在游戏与外在游戏之间的关系。

## 概述：什么是“基于角色的搭讪”？

我的整个游戏风格专注于一个核心概念：**人形基础选择**。这个概念指的是，你在社交互动（内容游戏）中的表现，会深刻影响你在社交圈之外（外游戏）的形象与结果。这就是整个理论的基础。

## 内在游戏与外在游戏：孰轻孰重？

上一节我们介绍了核心概念，本节中我们来看看一个关键争议：内容游戏（内在状态）和外在游戏（外在技巧）哪个更重要？

实际上，两者都重要，各占一半。这就像“先有鸡还是先有蛋”的问题，两者相辅相成。不像某些理论只强调其中一方，它们其实是互相协助的关系。

内容游戏的核心问题在于，你内心的真实状态可能无法有效表达。例如，我提到“灵体”这个概念。如果我处于理想的“灵体”状态，我知道我可以吸引女性，这是我的真实样子。但在现实中，这可能显得不切实际或不合理。然而，这恰恰是关于如何用你的言语和内在特质，在世界中与他人接触并引发反应。

## 如何构建你的核心角色？

我将专注于几个主要特质来构建角色：灵性、高贵、忠诚。你需要像弗兰克·辛纳屈歌曲里唱的那样——“我走我自己的路”。这就是我希望你们去思考的。

作为一个年轻人，无论是20、25岁，还是30岁出头，请思考：你奉行的人生信条是什么？你如何实践自己的信条？例如，如果你是田径运动员，你就践行田径的信条。人们看到你会觉得：“这很酷。虽然这不是我的路，但我很欣赏这个人，因为他知道自己是谁。”这个人不会愤怒地纠结于要在世界上做成什么。

当你获得这种内在的信息时，一切会变得毫不费力。你不需要在“游戏”中做任何刻意的努力。我指的不是技术部分，而是原理。这合理吗？这很合理。这就是参与者所处的环境。

## 学习者的三个阶段

当你第一次开始学习社交技巧时，你会经历不同的阶段。以下是每个阶段的特点：

**第一阶段：初学者**
你会专注于技术。一开始，你可能会想：“我讨厌女生，我讨厌这样。我不知道我能做什么。别人都做得很好，我做得不好。”你认为如果掌握了正确的技术或得到了正确的结果，你的感觉就会改变，你就会感觉良好，就会变得有吸引力。这是初学者的策略。

**第二阶段：进阶者**
你会开始获得一些成绩。例如，你会在社交场合或像Tinder这样的软件上获得一些互动。你开始有自信，但这种自信仍然来源于外在的结果。你会想：“我现在可以爱自己，欣赏自己，变得自信，因为我有了和女生互动的结果。”这就是进阶者的状态。

**第三阶段：精通者**
你会到达一个阶段，获得一种“原始变动”。你会觉得：“我是个玩家，我去过这个阶段，我疯了，我的表演不可思议。我的整个世界都在，我和女生互动都很认真，我当时觉得这太可怕了，但我也需要那段经历。”你需要那段时间来让你知道，你可以和女生很好地互动。

你必须经历这些阶段，不能跳过。你不能说“我只想成为进阶者，但不想练习进阶者的内容”。这是不正常的。你必须逐步前进，不能比你本应达到的阶段前进得更快。

## 理解“原始变动”

你会到达一个阶段，获得一种 **“原始变动”** 。原始变动是指，你所认知的一切都发生了根本性的改变。

我举个例子：几百年前，教会和科学家相信地球是宇宙的中心，而月亮是地球的中心。这就是他们当时的“原始变动”认知。后来伽利略等人带来了新的认知变革。

另一个例子是LHC（大型强子对撞机）。这是人类历史上最大的实验之一。科学家通过它进行实验，例如将两种物质合为一体再分开，以研究宇宙大爆炸。他们发现了“希格斯玻色子”（上帝粒子）的存在。这件事改变了整个游戏规则，改变了你所知道的一切，让所有事物从不同的光和角度呈现出来。

**BOSS** 就是你内心的一个变化。你以为你懂得游戏的一切，但其实你并不懂。如果你认为你懂，但你已经看了一两年RSD教学视频，却还没有达到你认为应有的水平，那说明你拥有很多信息，但并未完全理解。

当我和Tyler一起训练时，我已经练习了一年，但我们只训练了五六个最基本的东西：意见、精神、成果、生存、自由、合作、不刻意尝试、社会情绪基础。你们都知道这些东西，但你们不明白它们的深度。
