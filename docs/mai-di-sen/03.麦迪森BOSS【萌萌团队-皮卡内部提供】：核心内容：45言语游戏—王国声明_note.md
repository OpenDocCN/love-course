# 课程03：麦迪森BOSS【萌萌团队-皮卡内部提供】- 45言语游戏：王国声明 👑

在本节课中，我们将学习“王国声明”这一核心概念。这是一种通过清晰定义“我是谁”与“我不是谁”来建立强大个人信念与身份认同的方法。它能帮助你在人际互动中保持坚定，并获得他人的尊重。

## 概述：什么是“王国声明”？

“王国声明”是一种个人信念与价值观的清晰表达。它关乎你对自己身份的根本定义，即“国王之声”。这并非外在的成就或头衔，而是你内心深处的核心信念。它决定了你如何行动、如何回应世界，并引导你从“尚可”的结果走向“卓越”的结果。

## 核心概念：国王之声

上一节我们介绍了“王国声明”的整体概念，本节中我们来看看其核心——“国王之声”。

**国王之声**是你个人发展的基石。无论你来自北京、美国、庙堡、巴基斯坦、菲律宾、荷兰、法国还是加拿大，你此刻在这里，是你自己做出决定并为之工作的结果。这就是你的“国王之声”。

它是一份持续进步的工作。这份工作本身就已足够好，你无需追求“完成”或“到达”某个终点。进步的过程本身就是有价值的。

## 实践工具：“我这种男生不等于...”

理解了“国王之声”的重要性后，我们需要一个具体的工具来表达它。这个工具就是“我这种男生不等于...”的声明句式。

这个句式简单却极具力量。它通过否定你不认同的事物，来清晰界定和肯定你的立场与信念。

以下是使用“我这种男生不等于...”句式的一些例子：

*   我这种男生不等于想要支持别人。
*   我这种男生不等于想要做伤害自己的事情。
*   我这种男生不等于看不到人类的好处。
*   我这种男生不等于认为人类不会自我改变。
*   我这种男生不等于不享受自己的社交生活。
*   我这种男生不等于会使用损害自己的药物。
*   我这种男生不等于想要怀疑自己的爱。
*   我这种男生不等于会放弃对方的信仰。
*   我这种男生不等于不相信自己的灵性。
*   我这种男生不等于不理解对方的信仰。
*   我这种男生不等于想要钱本身。
*   我这种男生不等于想用钱来交易爱情、家人或朋友。
*   我这种男生不等于为了社交和文化而享受酒精。
*   我这种男生不等于会用酒精来损害自己、丧失自信。
*   我这种男生不等于不尊重合理的他人信仰。
*   我这种男生不等于不相信一段需要全心投入的爱情。
*   我这种男生不等于不重视健康，不维护心理安全与身体尊重。
*   我这种男生不等于认为健康是拥有美好生活的唯一途径。
*   我这种男生不等于来RSD工作只是为了对世界施加影响。
*   我这种男生不等于只追求肤浅的搭配或甜言蜜语。

这些是具有明确定义的声明。它们是你向世界表达“你是谁”的方式。

## 为何需要“王国声明”？

掌握了表达工具后，你可能会问，为什么我们需要如此清晰地声明自己？

你必须了解这些，因为在这个社交“乐园”中，在与他人（尤其是异性）互动时，人们可能会挑战你：“你以为你是谁？”“你不该在这里。”“这不重要。”“你不合适。”

他们会挑战你对世界的认知。因此，你必须预先知道自己是谁。你必须能够说：“我就是这样。”“我不是那样。”立场必须非常清晰。

所以，阐明你的信念、你的表达、你的立场以及你拒绝的立场，至关重要。定义你自己是谁、你将成为谁。你不再停留在过去，你做出了这些决定。

## 坚定声明的力量

当你做出并坚守这些声明时，你会获得更多力量。在这套人际“游戏”中，你不会失败。即使你身边的人不认同，你也没有失败。只要你基于自己认为最好的决定而行动，你永远不会失去他人的尊重。

你唯一可能失去的，是表达自我的机会。因此，阐明你的信念至关重要。

## 总结与行动

本节课中，我们一起学习了“王国声明”这一核心沟通与个人定位工具。

我们了解了“国王之声”是你内在的信念核心，并学习了通过“我这种男生不等于...”这样强有力的句式来对外清晰表达自己的身份与边界。这种声明帮助你在面对外界挑战时保持坚定，赢得尊重，并持续走在自我进步的道路上。

如果你还不清楚自己的“王国声明”是什么，可以尝试以下练习：列出10条以“我这种男生不等于...”开头的句子。独自在夜晚面对窗户时，思考一下，你会想到什么？把它们写下来。这就是你构建自己“王国”的开始。
