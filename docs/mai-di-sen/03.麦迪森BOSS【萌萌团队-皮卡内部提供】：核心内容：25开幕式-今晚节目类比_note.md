# 课程03：从“不想做”到“开始做” 🚀

在本节课中，我们将通过一个生动的比喻，学习如何克服行动前的情绪阻力，培养立即开始的习惯。我们将借鉴“制作《今晚的节目》”的过程，来理解启动任何任务的核心心态。

## 概述：一个关于开始的比喻

曾经，有人问我是否总想做《今晚的节目》。这个故事是这样的：坦白说，有时候我在家，根本不想做《今晚的节目》。我们去录影，我确实不想做。我走到我的车旁，他们叫我过去。我开车到工作室，我依然不想做。我坐在化妆椅上，他们给我穿上服装，我还是不想做。

## 核心过程：行动先于感觉

我走到舞台上，听到他们说“《今晚的节目》”，我和Jimmy Fallon一起。我仍然不想做。然后他们说：“大家好，我是Jimmy Fallon，欢迎大家，谢谢收看。”接着，他就做了《今晚的节目》。他做了，没有一秒犹豫。

这个过程揭示了一个关键方法：让你体验到那种“已经开始”的感觉。我仍然有那种“不想开始”的感觉，但我从未让那种感觉阻止我。因为一旦热度（激情或紧迫感）来临，比如想要十万个孩子或其他目标，那就是行动的唯一时机。那是一个过程，不是一个瞬间的决定。

## 核心原则：情绪不是行动指南

以下是基于比喻总结出的行动原则：

**不要让你的情绪带领你。** 不要让你的情绪说：“等我有了感觉再做。”她（指感觉）不是那个（行动的先决条件）。我不会等到有感觉才做。

**在状态中行动，而非等待状态。** 代码可以表示为：
```python
while not in_state: # 不要循环等待“有状态”
    start_acting() # 直接开始行动
    state = get_state() # 状态会在行动中自然产生
```

当你年纪增长，你的情绪会欺骗你，会误导你。不要相信自己的情绪。我想做什么，我都做了两遍。现在没有“游戏”（借口）了，不行。我处于“教练模式”，我没办法（找借口）。

## 心态转换：从评判到参与

她在哪里？她是谁？我没有喜欢她，她太丑了。我朋友来跟我说话，我太丑了。你是今晚的Teeqeelah Jacks吗？哦不，我不敢。哦，真的谢谢。我说，这几个人都……

你去Teeqeelah Jacks吗？我猜有很多Teeqeelah Jacks吗？有。好，这怎么办？好，好好。这是个美好的东西，这是个爱。她需要一个男人，她需要一个男人。我们需要一个男人，我们一起。对不起，兄弟。你是两个男女？好，当你就修好。

嗯，说中文。你从哪里来？做了什么。（此段比喻从自我评判和犹豫，转向接纳现状并寻找解决方案——“修好”，即开始行动并参与其中。）

## 总结

本节课中，我们一起学习了如何跨越“不想做”的情绪障碍。核心在于理解：**行动是感觉的催化剂，而非结果**。我们通过“制作《今晚的节目》”的比喻，明白了必须通过启动行动本身来打破惰性循环，而不是等待有状态、有感觉或完美的时机。记住公式：**开始做 > 感觉想做**。不要被情绪欺骗，直接开始，状态自会随之而来。
