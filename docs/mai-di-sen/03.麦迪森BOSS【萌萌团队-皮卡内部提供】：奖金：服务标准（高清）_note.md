# 课程03：建立个人服务标准 🧭

在本节课中，我们将要学习如何建立和维护个人在服务互动中的标准。这涉及到理解自我价值、设定界限，并通过实践来强化这些标准。

## 概述：为何需要个人标准？

上一节我们探讨了个人发展的基础。本节中我们来看看，在人际与服务互动中，明确的标准为何至关重要。

作为个人发展的起点，你最初可能没有明确的标准，或者只拥有自己模糊的内在标准。当你开始成长，你会逐渐发展出一些标准和价值观。这个过程让你成为一个更沉稳、平和的人。

这有几个原因。过去，当他人接近你时，你可能会选择迎合他们的世界，而非坚守自己的立场。你总是走向他们期望的方向，那条路始终被他人的利益所吸引。你必须做的是，希望走上这条路后，你能发展出自己的标准和价值，从而看到世界的另一面。那条始终被他人利益吸引的道路，会越来越接近你自己的核心立场。因此，你会获得更好的平衡，从可能80%给予他人、20%留给自己，转变为各占50%。所以，你必须发展自己，有时对情况说“好”，有时则说“不”。

## 如何练习说“不”？

那么，如何练习设定界限这件事呢？以下是关于练习拒绝的思考。

我认识一个人，上次我在他那里练习时，他想对全世界说“不”。他决定在森林里独处两个月，不与任何人交谈。这是他对世界的一个强烈表态。我不会建议你对全世界说“不”，而是建议你小范围地练习说“不”，例如每周几次。这不是一种对抗，而是因为这是一个新的尝试。你将在实践中受益，让人们学会尊重你的界限。

## 本周实践任务：拒绝为不合格服务付费

基于以上理念，本周的任务非常简单。以下是具体的任务步骤。

我想让你外出，去一个娱乐场所、餐厅或乘坐网约车。但关键点是，当服务不达标时——例如没有达到三星或两星的服务质量——不要支付小费或额外费用。以Uber为例，它们有五星评分体系，当你遇到两星的服务时，就不要完成给予小费这个动作。

我会解释为什么。你可能听过这个故事：我曾经在拉斯维加斯，和Max一起去过Cosmo酒店。那家酒店的服务费是13美元，我给了服务员15美元。那个人反应激烈，说：“什么？我不能用15美元来擦屁股！”他非常生气，因为我只给了2美元小费。这很荒谬，如果我得到好的服务，我会给更多小费，但他的态度太差了，我根本不想给他小费。他觉得自己很了不起。

这引申出一个核心概念：**在交易中，你需要为自己的金钱负责，设定服务的价值底线**。例如，在餐厅，如果服务员搞错了订单，服务就不合格，你必须决定如何应对。在酒吧，如果酒保给你倒了一杯酒，但这并不代表他们自动应得小费，这是基本的服务。如果他们迟到、忽略了你或服务不佳，那么你或许不应该给小费，或者应该少给一些。这条底线需要你来决定。

我认为有些交易，你不应该支付小费。文化上，这与世界其他地方不同。在欧洲，小费文化较弱；在美国，如果你不给小费，可能被视为不礼貌，服务员可能会不满。文化存在差异，这需要你来判断。

## 任务执行指南

你需要在本周去两三个地方，当服务不足时，拒绝支付小费。可能会遇到尴尬的时刻，对方可能会盯着你。你只需要说：“对不起，我认为服务没有达到我的预期。”或者说：“希望下次服务会更好。”然后与他们进行简短对话。在对话时，不要逃避，要坐下来沟通。不要反应过度或攻击对方。坐下来，感谢他们的工作，鼓励他们，并表达希望下次更好的愿望。

这就是本周的任务：去酒吧、餐厅或某个活动场所，如果服务不合格，就拒绝支付小费，并解释原因。
