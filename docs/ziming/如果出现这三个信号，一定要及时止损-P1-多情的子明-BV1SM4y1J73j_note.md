你需要清楚地知道女生的意思，不要在一个不喜欢你的人身上浪费青春。强扭的瓜不甜。看到对方的不喜欢，我们要释怀。可是很多兄弟总觉得，只要女生没有明确拒绝，就认为自己还有机会。

上一节我们介绍了识别信号的重要性，本节中我们来看看具体的三个关键行为信号。如果你正在追的女生有以下行为，你就不要再自欺欺人了。
