人与人之间的交流，超过一半的信息是通过非语言方式传递的。对于情感表达，尤其是初期的好感，微动作和身体语言是关键的信号。我们将逐一解析五个核心的微动作，帮助你读懂潜在的好感暗示。

## 一、视线追随你 👀

视线方向与停留时间是判断关注度的首要指标。行为心理学指出，人在初次见面时会有一个自然的审视过程。

**核心行为公式**：`目光轨迹 = 眼睛 -> 嘴巴 -> 喉咙 -> 眼睛`

如果她在完成这个审视轨迹时，与你的眼神有交汇且不躲避，这通常表示她对你有好感。

其次，在多人场合中，本能会驱使人们看向自己喜欢的人。

以下是判断视线好感的两个关键点：
*   **关注时长**：她投向你、关注你的时间，明显多于关注周围其他人。
*   **伴随表情**：如果她的视线经常伴随着笑容，这是一个非常强烈的积极信号。

当你能持续感受到来自特定方向的目光时，说明你已吸引了她的注意。理解了视线信号后，我们来看看距离所传达的信息。

当一个人喜欢另一个人时，常会无意识地去模仿对方的言行举止，这被称为“变色龙效应”。

这一点也可以反向运用：

共同点的增加会有效提升好感度。从行为模仿过渡到肢体接触的容忍度，是判断关系深度的关键一步。

对肢体接触的接受度，是测试好感度和安全感最直接的方法之一。

**测试技巧**：当她坐在你旁边时，你可以伸手去拿她附近的物品（如餐巾纸、饮料），观察你在俯身瞬间，她是否有下意识的身体躲避。

以下是基于接触反应的判断：
*   **不躲避**：说明她对你不设防，感到舒适和安全。
*   **愿意接触**：如果她愿意让你摸摸头，冷了愿意披你的外套，过马路愿意被你牵着，甚至主动创造轻微身体接触（如拉拉你的衣服），这都是极强的兴趣指标。

反之，如果她对任何肢体接触都本能抗拒，则说明关系尚未达到亲密度。最后，我们来看看她如何通过管理自身形象来传递信号。
