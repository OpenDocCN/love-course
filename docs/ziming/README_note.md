# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。每个变量都有一个名字（即标签），我们可以通过这个名字来使用或修改盒子里存放的数据。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如，`name = "子明"` 这行代码创建了一个名为 `name` 的变量，并将字符串“子明”存储在其中。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的类型。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型介绍：

1.  **整数**
    整数是不带小数点的数字，例如 `-10`, `0`, `250`。

2.  **浮点数**
    浮点数是带有小数点的数字，例如 `3.14`, `-0.001`, `2.0`。

3.  **字符串**
    字符串是由零个或多个字符组成的文本，需要用单引号 `'` 或双引号 `"` 包围起来，例如 `"Hello"`, `'Python'`。

4.  **布尔值**
    布尔值只有两个：`True`（真）和 `False`（假），常用于逻辑判断。

## 如何查看与使用类型？ 🔍

知道了有哪些类型，我们如何在代码中查看一个变量的类型呢？Python提供了一个内置函数 `type()` 来帮助我们。

例如，执行 `type(100)` 会返回 `<class 'int'>`，告诉我们100是一个整数类型。

在编程时，了解数据类型至关重要，因为不同类型的数据能进行的操作不同。例如，数字可以进行加减乘除，而字符串通常可以进行拼接和查找。

以下是使用不同类型数据运算的示例：
```python
# 数字运算
number = 10 + 5  # 结果是整数 15
price = 19.99 * 2  # 结果是浮点数 39.98

# 字符串拼接
greeting = "你好，" + "世界！"  # 结果是字符串 "你好，世界！"
```

## 一个简单的实践示例 ✨

现在，让我们把以上概念结合起来，写几行简单的代码。

假设我们要记录一位用户的姓名、年龄和是否为新用户。

我们可以这样写：
```python
user_name = "张三"
user_age = 28
is_new_user = True

print("用户姓名：", user_name)
print("用户年龄：", user_age)
print("是新用户吗？", is_new_user)
```
运行这段代码，会在屏幕上输出三条信息。通过这个例子，你可以看到我们如何声明不同类型的变量并打印它们。
