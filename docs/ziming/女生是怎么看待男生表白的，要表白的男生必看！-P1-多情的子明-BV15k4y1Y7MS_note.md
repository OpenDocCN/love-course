## 🤔 女生面对表白时的复杂心情

女生在面对不同的人表白时，心情是非常复杂的。

面对喜欢的人，她肯定是开心和激动的。但是，当面对不喜欢的人时，情况就不同了。虽然我们不会读心术，无法知道她的内心戏，但我们仍有办法判断在什么情况下是没戏了，以及在什么情况下说明还有希望。

## 🚫 明确拒绝：果断放手

上一节我们介绍了女生心情的复杂性，本节中我们来看看第一种明确无望的情况。

当你向她表白时，她果断拒绝了你。这表明她是真的对你没兴趣。我劝你别再想办法纠缠人家。

以下是此时应采取的行动步骤：
*   先从普通朋友做起。
*   以后的事情从长计议。

## 🔄 回避与兜圈子：缓和关系

另一种常见情况是，她开始跟你兜圈子、打马虎眼。

她的话题从家长里短绕到人生哲学，就是不正面回答你的表白。不仅如此，最后她还会刻意回避。女生的意思是：她不喜欢你。她只是不太想把话说得太明白，觉得太尴尬了。

以下是应对这种情况的建议：
*   没有必要追着问个所以然。
*   把缓和关系、恢复正常交往作为首要任务。
