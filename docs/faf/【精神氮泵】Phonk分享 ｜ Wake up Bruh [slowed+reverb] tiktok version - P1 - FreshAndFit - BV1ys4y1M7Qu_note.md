# 课程 P1：Phonk音乐制作入门 - 节奏与氛围构建 🎵

在本节课中，我们将学习如何制作类似《Wake up Bruh [slowed+reverb]》风格的Phonk音乐。我们将重点分析其标志性的节奏结构和氛围营造手法，并通过简单的步骤进行实践。

## 概述

Phonk是一种源自Memphis hip-hop的电子音乐流派，以其厚重的低音、扭曲的人声采样和缓慢的节奏著称。 slowed+reverb版本通过降速和混响效果，进一步强化了其梦幻、复古的氛围。本节课将拆解其核心元素。

## 节奏构建：重复的力量

上一节我们概述了Phonk的特点，本节中我们来看看其节奏是如何构建的。许多Phonk曲目，包括本节课的示例，都大量使用简单、重复的节奏型来驱动全曲，营造出催眠般的听感。

以下是构建基础节奏轨道的步骤：

1.  **选择鼓音色**：使用808鼓机音色，特别是低沉、有尾音的底鼓（Kick）和清脆的军鼓（Snare）或拍手（Clap）。
2.  **设置速度**：将工程速度（BPM）设置为较慢的范围，例如 **70-90 BPM**。slowed版本通常更慢。
3.  **编排节奏**：创建一个极其简单的节奏循环。例如，在四四拍中，每小节底鼓落在第一拍和第三拍，军鼓落在第二拍和第四拍。
    ```代码
    节拍： 1   2   3   4
    底鼓： X       X
    军鼓：     X       X
    ```
4.  **高频率重复**：将上述节奏型复制、循环，形成持续不断的节奏基底。这种密集的重复是营造驱动感和恍惚感的关键。

## 氛围营造：人声采样与效果处理

节奏轨道建立后，需要为其注入灵魂。人声采样及其处理方式是塑造Phonk独特氛围的核心。

以下是处理人声采样的关键步骤：

1.  **采样选择**：寻找简短、有张力或标志性的人声片段，例如“Yeah”、“Bruh”或“Wake up”。
2.  **降速处理**：对采样进行降速（Slowed），这会降低音高，让人声听起来更低沉、模糊。降速比例可通过公式计算：
    **目标速度 = 原始速度 × (目标音高 / 原始音高)**。实践中，直接使用音频软件的变速功能降低 **20%-50%** 即可。
3.  **添加混响**：为人声采样添加大量的混响（Reverb）。调整混响的衰减时间（Decay Time）使其足够长，让人声产生空间感和拖尾效果，与其他干声元素形成对比。
4.  **循环与穿插**：将处理后的短采样放置在节奏的空隙中，或使其以固定的间隔循环出现，作为音乐的标志性动机。

## 总结

本节课中我们一起学习了Phonk音乐制作的两个基础环节：通过**简单节奏型的高频重复**构建驱动感，以及通过对**人声采样进行降速和混响处理**来营造复古、梦幻的氛围。掌握这两个核心技巧，你就已经迈出了制作属于自己的slowed+reverb风格Phonk曲目的第一步。
