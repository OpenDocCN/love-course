# 课程01：理解“触发”与“社会正义战士”的互动模式 🧠

在本节课中，我们将通过一个具体案例，学习网络互动中“触发”现象的含义，并分析“社会正义战士”的典型反应模式。我们将拆解对话中的关键元素，理解其背后的逻辑与情绪动因。

## 概述

课程从一个假设场景开始：想象你因为一个简单的称呼而被某人强烈地“触发”或激怒。这个案例将帮助我们观察极端反应的言行特征。

## 场景描述

设想一个互动场景：一个人因为被称呼为“女士”而情绪失控。对话中反复出现“女士”、“区长先生”的称呼，最终以一句“去你的”结束。

## 反应模式分析

上一节我们描述了一个具体场景，本节中我们来分析其中展现出的反应模式。

以下是该模式中可观察到的两个核心言行特征：

1.  指责对方“容忍度极低”：当事人反复指责对方“你的容忍度太低了”。这是一种常见的反驳策略，旨在将自身情绪反应的责任转移到对话者身上。
2.  强调自身正常性：当事人以“对，这个人完全正常。完全不疯狂。”这样的陈述来宣告自身行为的合理性，这通常用于对抗外界对其反应过度的评价。

## 核心概念解读

在分析了具体行为后，我们需要理解其背后的核心概念。

*   **“触发”**：在互联网语境中，指因特定词语、观点或场景而引发的强烈情绪反应（通常是愤怒或不适）。其过程可以简化为：`特定刺激 -> 强烈的情绪反应`。
*   **“社会正义战士”**：通常指那些在社交场合，特别是网络上，以极端激进的方式推行其社会正义观点的人，其英文缩写为 **SJW**。

## 总结

本节课中，我们一起学习了一个关于“触发”反应的案例。我们描述了场景，分析了“因称呼而情绪失控”及“通过指责他人和宣称正常来辩护”的行为模式，并解释了“触发”和“社会正义战士”这两个关键概念。这有助于我们理解某些网络互动中情绪化对立的表层逻辑。
