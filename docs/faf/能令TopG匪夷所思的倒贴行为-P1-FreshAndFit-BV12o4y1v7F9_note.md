## 什么是“倒贴”？🤔
“倒贴”通常指在一段关系中，一方在情感、时间或物质上付出远多于另一方，且这种付出往往得不到对等的回报或认可。

上一节我们介绍了课程主题，本节中我们来看看“倒贴”的具体定义。

其核心公式可以描述为：
**关系投入失衡 = 一方付出 >> 另一方回报**

这种行为超出了传统社交互惠的预期。

---

## “倒贴”的主要表现形式 📋
理解了定义后，我们来看看这种行为在现实中是如何体现的。以下是“倒贴”常见的几种表现形式：

1.  **经济上的过度承担**：持续为对方支付共同消费的费用，或赠送贵重礼物，而对方鲜有回馈。
2.  **情感上的单方面供给**：总是主动提供情绪安慰和支持，但自己的情感需求被对方忽视或敷衍。
3.  **时间与精力的无限投入**：随时响应对方需求，优先处理对方事务，而自己的时间和边界不被尊重。

---

## 为何“倒贴”令人匪夷所思？🤯
对于崇尚对等价值交换的群体（如所谓的“Top G”）而言，倒贴行为违背了其核心社交原则。本节我们将探讨其背后的逻辑冲突。

上一节我们列举了“倒贴”的表现，本节中我们来分析其令人困惑的原因。

主要原因包括：
*   **违背互惠原则**：健康的关系建立在给予和获取的动态平衡上。单方面持续付出破坏了这一基础。
*   **暗示低自我价值**：过度付出而不求回报，可能被解读为个体认为自己提供的价值不足以换取对等回馈。
*   **投资回报率为负**：从理性角度看，这是一种长期来看“ROI”（投资回报率）为负的行为策略。

用简单的代码逻辑可以表示为：
```python
if 持续付出 and 回报 ≈ 0:
    关系状态 = “失衡的倒贴”
    理性评价 = “匪夷所思”
```
