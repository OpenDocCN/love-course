# 课程名称：为什么女性会向上择偶 ｜ 第1部分 🧠💑

在本节课中，我们将要学习一个跨文化的社会现象：女性向上择偶（Hypergamy）。我们将探讨其背后的生物学与社会学原理，并理解社会财政地位如何影响两性间的吸引力。

## 概述 📋

人类女性的繁殖价值非常高，这是一个真正的跨文化现象。在加拿大情况略有不同，存在些许淡化，但本质未变。社会中的主导权结构存在差异：女性通常拥有自下而上的影响力，而男性则多表现为自上而下的主导权。女性的社会财政状态，几乎不影响她对男性的吸引力。

## 社会财政状态与吸引力 💰

上一节我们提到了两性主导权的差异，本节中我们来看看社会财政状态如何具体影响吸引力。

男性的社会财政状态，是女性对其吸引力的主要决定因素。这并非指男性拥有的财产本身，因为这一点已被研究证实。关键在于男性**生产与分享资源的能力**。财产可能失去或消失，但持续生产和分享的能力是更重要的部分。

## 女性的择偶逻辑与需求 👶

理解了男性吸引力的核心后，我们现在从女性的视角来分析其择偶逻辑。

女性，尤其是因为需要生育和抚养孩子，会变得更加依赖。女性看待事物的逻辑，从发展与生存的角度来看，会寻求能提供足够支持的伴侣。因此，女性对男性会提出较高的要求。

这并不令人意外。

## 总结 ✨

本节课中我们一起学习了女性向上择偶现象。我们了解到：
*   女性的吸引力较少受自身社会财政状态影响。
*   男性的吸引力主要取决于其**生产与分享资源的能力**，而非静态的财产。
*   女性因生育和抚养后代会产生依赖需求，从而在择偶时逻辑性地寻求能提供强大支持的伴侣。

这种现象源于深层的生物与社会进化逻辑，是全球范围内一个普遍存在的趋势。
