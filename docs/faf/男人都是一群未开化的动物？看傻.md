# 男人都是一群未开化的动物？看傻

在本节课中，我们将要学习两性在择偶时关注的核心价值有何根本不同。我们将通过具体的行为分析和概念对比，来理解这些差异背后的逻辑。

上一节我们介绍了课程主题，本节中我们来看看具体的行为模式分析。

当你说：“这只是他的性格，这就是一个人在显示他的弱点，显示他的非人性，他不能超越他的动物性。我喜欢你如何为男人的想法开脱。那你呢？你离开你的动物性。”

以下是要求男性压抑本能的一种矛盾体现：
*   要求男性离开其动物性的需求。
*   去约会那些矮小、肥胖、不成功的男人。

这是一个错误的概念，因为女性不会改变她们的动物性需求。她们不会去约会矮小的男人，不会去约会肥胖的男人，也不会去约会失败者。她们想要约会的是一个能超越她们的男人。

那么，为什么女性的标准从不“欺骗”男人呢？因为双方关注的价值维度截然不同。

以下是两性关注的核心价值对比：
*   **男人看重繁衍价值**。他们更关注女性的美貌、青春、生育潜力、身材、是否肩窄、是否长发、大腿是否丰满等特征。
*   **女人看重生**存价值**。她们更关注男性的能力、地位、资源以及能否提供保护和稳定生活的能力。

本节课中我们一起学习了两性吸引力的基础模型。核心在于，男性倾向于关注与**繁衍**相关的特质（繁衍价值），而女性则更看重与**资源与保护**相关的特质（生存价值）。理解这一根本差异是分析后续更复杂情感动态的基础。
