# 课程一：现代男性孤独的根源与后果 🗿

在本节课中，我们将探讨为何全球范围内有如此高比例的男性感到孤独。我们将分析导致这一现象的两个核心社会原因，并审视其可能引发的深远个人与社会后果。

## 概述：杰弗里的困境

杰弗里做的一切都正确。他学业优秀，善于与女性交谈，品行良好。但女性们始终对他不感兴趣。年复一年，杰弗里因屡遭拒绝而感到崩溃，最终他放弃了尝试。

他在网络社区和不良网站中寻找慰藉，却越陷越深，愈发孤独。如果我们放大视角，会发现全球超过90%的男性都与杰弗里处境相似。

那么，另外10%的男性是怎样的？他们是“阿多尼斯”。但这并非指他们是坏人；事实上，阿多尼斯可能是最友善、最光荣、最支持伙伴的人。他希望身边的男性都能成长，去体验生活中美好的、高阶的一面。阿多尼斯并不孤独，但他可以选择孤独。

## 第一节：孤独的普遍性与个人体验

上一段我们介绍了杰弗里的普遍困境。现在，让我们看看一个具体的个人旅程。

当我开始这段专注于事业和自我提升的旅程时，我离开了居住的城市，回到父母家。当时正值疫情封城，我感到极度孤独。我完全独自一人。虽然有家人，但关系并不亲密，我也没和任何老朋友联系。

实际上我努力过。我尝试给他们发信息，但他们没有理会。我开始经营自己的生活，想创办公司，创建YouTube频道。我每天都会外出，希望能找到可以聊天的人。我仍在练习如何与男性接触。

我记得我拍了一段私人视频日志。在这12分钟的视频里，我不断诉说自己是多么孤独，在社会交往和性格方面有多么匮乏。我没有男性朋友，没有与女性约会，没有任何亲密关系。大概四五个月的时间里，我没有和任何人深入交谈。我没有朋友，我感到非常、非常孤独。

既然90%的男性都感到孤独，我们就需要想办法应对其后果。但首先，我们必须理解为何有如此多的男性陷入孤独。

## 第二节：男性孤独的两大核心原因

以下是导致现代男性普遍孤独的两个主要原因。

**第一，男性安全空间的缺失以及男性友谊的建立。**
**第二，约会市场的现状，以及我们如何糟糕地处理约会、建立爱情与关系、组建家庭。**

在当今时代，我们先探讨第二个原因。你应该了解一个现状：大多数女性都被同一小群高阶男性所吸引。我并非在批判这种现象，但这确实是正在发生的事实。

高阶男性，即那前10%的男性，他们富有、形象好、自信、具有偏女性欣赏的男性气质。这些高阶男性数量很少，但他们与大量女性发生关系。你或许觉得这很夸张，但有实际数据证明：前20%的男性吸引了80%的女性。

**公式：** 前20%的男性 → 吸引了80%的女性

这是一个基于个人水平的、回报差异巨大的数字。大多数男性完全无法获得女性的青睐。约会软件（如Tinder, Bumble）上的研究证实，大多数女性几乎对每个男性都向左滑（拒绝），而她们向右滑（喜欢）的，往往是同一小群男性。因此，一个男性可能与十位女性约会，而这些女性彼此都认识。

## 第三节：社会观念变迁与女性性行为

这一切的后果，在我们审视现代社会对性观念的改变时会变得更加明显，尤其是对女性而言。与100-200年前相比，女性在性方面拥有了更多的自主权和信心。原因如下：

这种情况历来如此。在伊斯兰教中，特定条件下允许男性娶四个妻子。在过去，强大、富有、有权势的男人往往拥有多个妻子，与许多女性发生关系。这是事实。有些男人至今仍与成千上万的女性发生关系。

第二个导致男性如今如此孤独的原因，是我们缺乏**男性的空间**。
