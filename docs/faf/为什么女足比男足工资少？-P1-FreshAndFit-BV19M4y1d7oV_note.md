*   女子世界杯创造了约 **6亿美元** 的收入。

---

基于以上分析，我们来探讨“同工同酬”这一概念。如果希望实现收入的绝对平等，在目前的经济模型下是一个复杂的问题。

核心逻辑在于，收入应与创造的价值相匹配。因此，追求的目标不应仅仅是“相同的收入”，而应是“与自身价值相符的收入”。

这可以用一个简单的原则来概括：
**Strive for getting what you're worth. And if you're worth more, then get more.**
（努力获取与你价值相符的报酬。如果你的价值更高，那么就应获得更多。）
