# 旭旭-全类别挽回攻略：4888男士自我提升攻略：6吸引力之心态 🧠

在本节课中，我们将要学习吸引力构建中一个至关重要的组成部分：心态。心态决定了你在与异性互动时的姿态和能量，直接影响关系的走向。前面我们介绍了硬价值、软价值、行为模式和思维模式，本节中我们来看看如何建立正确的心态。

## 核心概念：奖品性 🏆

在心态中有一个非常重要的概念，叫做**奖品性**。在传统追女生的过程中，男生往往喜欢把女生捧得过高，感觉对方高高在上，而自己非常卑微，是在追求和恳求对方。这时你们的关系是不平等的，处于高位的人就叫做“奖品”。他对你的所有好，都是一种奖励；而你对他的好，都是应得的、是一种恳求。

例如，他今天同意跟你约会，那是对你的一种奖赏，你得珍惜。而你请他吃饭，那是你应该的，因为你在“求”他。这种不平等的关系会无形中加大你的难度，因为你做任何事情都理所应当，他对你的好都需要你努力才能求得来。

所以，在追女孩的过程中，你最好要让自己变成奖品，而不要让对方变成奖品。当对方是奖品时，他对你的好都是一种奢求。例如，你请他吃饭是应该的，因为你在追求我（奖品）；而我对你好，那是我对你的奖励。

大部分男生在追女生时都会遇到这个问题：对方能跟你看一场电影，对你来说都变成了一种奢求。但实际上，你请他吃饭、看电影，同样在付出价值。关系不平等的原因在于，他是奖品，而你是追求者。

因此，我们在追女孩时，千万不要让对方变成奖品。一旦如此，你会发现做什么都特别难，连约对方看场电影都是一种奢求，而你对他的好他都会觉得理所应当。因为关系失衡了，他变成了奖品，你变成了恳求者。

最好的方式是把自己变成奖品。如果暂时做不到，至少保持平等关系。也就是说，你跟他说话时，不要以一副恳求的姿态。

以下是错误心态的典型表现：
*   “能不能陪我看场电影啊？”
*   “你什么时候有空啊？”
*   “在吗？人呢？怎么不回我消息了？”

这些行为都会让对方变成奖品，因为你降低了自己的地位。你应该正常地跟他说话。

以下是平等心态的邀约示例：
*   “这周末要不要一起出来玩？”
*   “明天我要自己做饭吃，你要不要过来尝尝？”

如果他不回消息，永远不要追问“人呢？怎么不回消息？”。他不回就不回了，这很正常。就像你和哥们聊天，对方不回你也不会追问。在追女生的过程中，一定要保持平等心态，而不是把他捧成女神的心态。

甚至有一些高手可以把自己变成奖品，明明是他在追对方，却能营造出对方追他的感觉。具体方法我们会在后续的聊天课程中详细讲解。

## 另一个关键心态：控制得失心 ⚖️

在追女生的过程中，还有一个非常重要的心态就是得失心。有很多人得失心非常重，觉得“没有这个人我就要死掉了”、“我一定要得到这个人”。例如，今天给他发消息他没回，就非常苦恼。这就是得失心重。

得失心重会让你去做很多傻事，失去理智。得失心来源于过高的期望。很多人觉得“我只要稍微努力一点，他就会成为我的女朋友”、“我只要怎么怎么样，我就能得到他”。但这往往是你自己一厢情愿的想法，是你给自己的希望太高了。期望越大，失望越大。

你应该抱着平常心去交流。他不回消息，那就不回了；他回了，自然更好。你越想得到他，反而越得不到。因为女人能感觉到你给她的压力太强。即使你内心特别想得到她，表面上跟他在一起时也要表现得平淡如水，千万不要给他太强的压迫感。

以下是得失心重的错误表现：
*   他不回消息，你就不断追问。
*   一次邀约失败，就疯狂地再次邀约。
*   对方不理你，就跑去找对方。

你越有这种表现，女人越会躲着你。你反而应该给他一些自由空间，他才会觉得跟你在一起很舒服。得失心会给他带来很大压力，让他感觉不舒服，所以一定要控制自己的得失心。

那么，如何控制得失心呢？很简单：你要降低他在你心中的比重。

你不能让你生命中90%都是他，那样你会每天朝思暮想，被他轻易左右情绪。他应该只占你生活的40%-50%，这已经算很多了。你更需要保持自己精彩的生活。

以下是构建平衡生活的关键部分：
*   你的事业
*   你的朋友
*   你的家人
*   你的社交圈子
*   你的梦想与追求

这些东西要占据你生命中的相当一部分。你的生命中不应该100%都是他。这不是说你不爱他，你爱他，但同时你也爱你的家人、朋友和事业。他们平均占据你的心态，这样你就不会被他左右，不会给他过大压力，也不会过度打扰对方。这种状态对双方来说都是最舒服的。
