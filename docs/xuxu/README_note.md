# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“Hello World”放入其中：
```python
message = "Hello World"
```
这里，`message` 是变量名（盒子的标签），`"Hello World"` 是存储的值（盒子里的东西）。

## 2. 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的类型。Python中几种常见的基本数据类型如下：

以下是几种核心的数据类型及其说明：

*   **整数 (int)**： 表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数 (float)**： 表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str)**： 表示文本信息，需要用单引号(`‘`)或双引号(`“`)括起来，例如 `‘你好’`, `“Python”`。
*   **布尔值 (bool)**： 表示逻辑判断的真假，只有两个值：`True`（真）和 `False`（假）。

## 3. 如何查看和转换类型？ 🔄

在编程时，有时我们需要知道一个变量的类型，或者在不同类型之间进行转换。

你可以使用 `type()` 函数来查看任何变量或值的数据类型。
```python
num = 100
print(type(num)) # 输出：<class ‘int’>
```

当需要转换类型时，可以使用以下函数：
*   `int()`： 将数据转换为整数。
*   `float()`： 将数据转换为浮点数。
*   `str()`： 将数据转换为字符串。

例如，将一个整数和字符串拼接起来，需要先将整数转换为字符串：
```python
age = 25
message = “我今年” + str(age) + “岁。”
print(message) # 输出：我今年25岁。
```

## 4. 变量的命名规则 📝

为了写出清晰易懂的代码，给变量起一个好名字至关重要。以下是需要遵守的基本规则：

为变量命名时，请遵循这些规范：

*   名称只能包含字母、数字和下划线（`_`），且不能以数字开头。
*   不能使用Python的保留关键字（如 `if`, `for`, `while`）作为变量名。
*   名称是区分大小写的（`myVar` 和 `myvar` 是两个不同的变量）。
*   建议使用描述性的名称，例如用 `user_name` 而不是 `n`。
