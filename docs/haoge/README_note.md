## 概述
在本节课中，我们将要学习Python编程语言的基础概念，包括变量、数据类型以及如何进行简单的数学运算。这些是编写任何Python程序的第一步。

---

## 1. 什么是变量？🧠
上一节我们介绍了课程概述，本节中我们来看看什么是变量。

变量是用于存储信息的容器。你可以把变量想象成一个贴有标签的盒子，标签是变量名，盒子里装的东西就是变量的值。

在Python中，给变量赋值非常简单，使用等号（`=`）即可。

**代码示例：**
```python
my_name = "昊哥"
my_age = 25
```

---

## 2. 基本数据类型🔢
理解了变量之后，我们需要知道变量可以存储哪些类型的数据。Python中有几种基本的数据类型。

以下是三种最常用的数据类型：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由字符组成的文本，需要用单引号（`'`）或双引号（`"`）括起来，例如 `"你好"`, `'Python'`。

**代码示例：**
```python
# 整数
score = 100

# 浮点数
price = 19.99

# 字符串
greeting = "欢迎学习Python"
```

---

## 3. 基础数学运算➕➖
我们已经知道如何存储数据，接下来学习如何操作这些数据。Python可以像计算器一样执行数学运算。

以下是几种基础的数学运算符：

*   **加法**：`+`
*   **减法**：`-`
*   **乘法**：`*`
*   **除法**：`/`
*   **取整除**：`//` （得到商的整数部分）
*   **取余数**：`%` （得到除法后的余数）
*   **幂运算**：`**` （计算乘方）

**代码示例：**
```python
a = 10
b = 3

print(a + b)   # 输出 13
print(a - b)   # 输出 7
print(a * b)   # 输出 30
print(a / b)   # 输出 3.333...
print(a // b)  # 输出 3
print(a % b)   # 输出 1
print(a ** b)  # 输出 1000
```

本节课中我们一起学习了Python编程的起点。我们了解了**变量**作为存储数据的容器，认识了**整数**、**浮点数**和**字符串**这三种基本数据类型，并掌握了使用 `+`, `-`, `*`, `/` 等运算符进行基础数学运算的方法。掌握这些概念是后续学习更复杂编程知识的重要基础。
