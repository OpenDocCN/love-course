# 昊哥《主导力训练营》：010.L09【塑造权威07】设立愿景 🎯

在本节课中，我们将要学习“提供奖励”维度的最后一个关键方法——设立愿景。我们将探讨愿景的本质、作用，以及如何有效地设立一个能吸引人、激励人的愿景。

上一节我们介绍了如何通过自我披露给予他人希望，本节中我们来看看如何通过描绘一个宏大的未来图景来提供更深层次的奖励。

## 什么是愿景？

说到愿景，很多人会联想到企业的使命和价值观。然而，我们今天讨论的愿景，其核心是描绘一个**美好、遥远且令人向往的未来图景**，以此作为一种强大的精神奖励。

为了理解这个概念，我们先来看几个例子。

以下是几个不同领域设立愿景的典型案例：

*   **佛教**：描绘了“六道轮回”与“天道”的极乐世界。它承诺，通过今生的善行、皈依与修行，死后可以进入一个没有痛苦、只有安宁与美妙的“天道”，甚至最终“跳出轮回”。
*   **基督教**：描述了“天堂”或“天国”的辉煌景象——那里由宝石与黄金筑成，没有疾病、痛苦，所有需求都能得到满足。进入天堂的“路径”是信仰、善行与奉献。
*   **伊斯兰教**：同样描绘了“天国”的美好：没有悲伤恐惧，居民永葆青春，没有阶级，享有华服、美食与宫殿。
*   **埃隆·马斯克**：提出了“多行星物种”的愿景，倡导人类成为跨星球文明，在火星建立基地，以避免局限于地球可能面临的灭绝风险。
*   **历史人物**：通过演讲描绘民族复兴、赢得尊严与“生存空间”的未来，激发追随者的行动。

观察这些例子，我们可以发现愿景的共同点：它们都指向一个**乌托邦式的完美世界**。更重要的是，每个愿景都配套了一条“实现之路”。

以下是各类愿景通常指明的实现路径：

*   **宗教路径**：遵守戒律（如不杀生、不妄语）、保持虔诚、进行传播（传教）以及付出物质贡献（供奉、捐钱）。
*   **企业家路径**：支持其事业（如购买产品、投资相关项目）。
*   **政治动员路径**：加入组织、贡献财力乃至人力。

**本质上，这条“实现之路”就是我们主导力模型中的“树立门槛”与“给予希望”的结合**：它强调目标极其艰难（门槛），但又承诺只要遵循特定方式付出，就有望达成（希望）。这形成了一个强大的激励闭环。

## 愿景的核心特点

那么，一个有效的愿景必须具备哪些特点呢？

1.  **兑现时间遥远**：愿景不是一个短期目标。它的实现周期很长，这避免了短期无法兑现而“爆雷”，也给予了设立者足够的缓冲空间。
2.  **描绘乌托邦**：它必须是一个极度美好、近乎完美的世界蓝图，这样才能强烈地激发人的向往和行动欲。
3.  **难以实现但有机会**：愿景必须被设定为困难重重，但同时又指明一条通过“付出特定代价”就有可能实现的路径。这正是“门槛”与“希望”的平衡艺术。

## 为何“虚拟”的愿景能成为“奖励”？

你可能会问，这种看似“画饼”的虚拟承诺，为何能成为有效的奖励？原因有以下四点：

1.  **愿景的本质是故事**：故事比理性说教更能激发情感共鸣，直接作用于大脑的边缘系统。人是情感驱动的生物，一个动人的故事（愿景）更容易被记住和信仰。
2.  **愿景反映设立者的价值**：一个宏大、合理且新颖的愿景，能反向证明设立者拥有卓越的阅历、知识体系和社会地位。例如，马斯克的技术背景支撑了他的太空愿景，这本身就成为他个人价值的一部分。
3.  **弥补意义感的缺失**：在现代社会，许多人感到生活缺乏意义。一个伟大的愿景能将个人与某种宏大的目标联系起来，提供稀缺的“意义感”和“价值感”，这是一种强大的精神效用。
4.  **提供“颅内高潮”**：即使未能实现，单纯在脑海中想象和憧憬那个美好未来，本身就能带来愉悦和满足感，就像阅读爽文一样，这是一种直接的精神奖励。

## 愿景与“画大饼”的区别

愿景和令人讨厌的“画大饼”在本质上是一样的，都是一套关于未来的美好叙述。它们的区别仅在于**听众是否选择相信**。

以下因素决定了人们是相信“愿景”还是鄙视“画大饼”：

*   **过往信用**：设立者是否兑现过过去的承诺？（例如，马斯克实现了许多看似不可能的承诺，建立了信用。）
*   **是否切中痛点**：愿景是否关联了听众最迫切的需求和欲望？
*   **是否看到行动**：设立者是否展现出为实现愿景而付出的切实努力和能力？

## 如何设立有效的愿景？

设立愿景是一项高级技巧，需要深思熟虑。以下是几个核心原则：

1.  **没有通吃的愿景**：任何愿景都只能吸引一部分具有特定利益、身份、世界观和知识体系的人。想清楚你想要吸引谁。
2.  **输出真实的想法**：最打动人的愿景往往源于你内心真实的价值观和欲望。不要阉割自己的想法去迎合他人，真实的表达才能提供真正的价值。
3.  **切中目标对象的痛点**：深刻理解你想要影响的人群，他们最缺失什么？是物质保障、发展机会，还是意义感？你的愿景必须回应这些核心痛点。

## 愿景的应用场景

愿景可以应用于多种场景：

*   **公司管理**：对外，可以像许多企业那样，宣称“让世界更美好”；对内，可以设立“让员工安居乐业”的具体目标。
*   **亲密关系**：在恋爱中，勇敢分享你对生活、对未来的真实看法和期望。例如，你可以表达：“我认为人生应该不断探索和变化，我希望我的伴侣是一个能共同拥抱不确定性、追求成长的人。”这本身就是一种吸引同频者的愿景。

一个完整的愿景体系应包括**长期的终极愿景**和**短期的阶段性目标**。长期愿景提供方向和意义，短期目标提供可实现的希望和路径。

## 愿景的力量

人本质上是“唯心”的动物，世界的发展方向往往由少数人坚信并推动的“愿景”所引领。一个强大的愿景能够：

*   **吸引追随者**：在亲密关系或团队中，人们会因为认同愿景而自发聚集，愿意付出更多。
*   **降低管理成本**：当大家为共同愿景奋斗时，会自发解决问题，减少了对微观管理和物质激励的依赖。
*   **提供深层激励**：它超越了柴米油盐和简单关怀，提供了精神层面的稀缺价值。

**正如《小王子》中的启示：如果你想造一艘船，不要急着雇人收集木材、分配工作，而是要激发他们对浩瀚海洋的渴望。** 设立愿景，就是描绘那片“海洋”和远方的“宝藏”。
