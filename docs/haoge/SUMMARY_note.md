# 昊哥《主导力训练营》课程大纲 📚

在本课程中，我们将系统学习如何塑造个人权威、给予他人希望、调整认知心态以及树立人际门槛，从而全面提升你的主导力与影响力。课程内容由浅入深，结合理论与实用技巧，旨在帮助初学者掌握有效沟通与领导的核心方法。

## 第一部分：塑造权威 👑

### 001. L01【塑造权威01】气场的本质——潜沟通
本节将探讨气场的核心，即“潜沟通”。真正的权威感不仅来自语言，更来自非语言信号传递的信息。

### 002. L02【塑造权威02】高/低位的语言模式 & 3种讲话习惯
上一节我们介绍了潜沟通，本节中我们来看看具体的语言模式。高位与低位的语言模式直接决定了你在对话中的主导权。
以下是三种关键的讲话习惯：
1.  陈述代替询问。
2.  减少不确定词汇。
3.  使用简洁有力的短句。

### 003. L03【塑造权威03】利用“紧张感”Tension
权威的建立有时需要适度的张力。本节将学习如何有意识地制造并运用对话中的“紧张感”（Tension），来掌控局面节奏。

### 004. L04【塑造权威04】制造压力感的四种技巧
延续对紧张感的讨论，本节将介绍四种具体技巧来向对方施加健康的压力感，从而巩固你的权威地位。
以下是四种制造压力感的技巧：
1.  沉默的力量。
2.  提出挑战性问题。
3.  设定时间框架。
4.  身体语言的压迫感。

### 005. 塑造权威04 课上视频素材
本节为上一课的理论提供实际视频案例，通过观察与分析，加深对制造压力感技巧的理解。

### 006. L05【给予希望01】自我披露
权威不仅需要压力，也需要亲和力。从本节开始，我们进入“给予希望”模块。恰当的自我披露是建立信任、给予他人希望的第一步。

### 007. L06【主导力三维模型】
在学习具体技巧前，我们需要建立一个整体框架。本节将介绍主导力的核心三维模型：**塑造权威**、**给予希望**、**树立门槛**。理解这个模型有助于你系统运用所学。

### 008. L07【塑造权威05】他人的刁难，压力测试
在建立权威的过程中，难免会遇到挑战。本节将探讨当他人对你进行“压力测试”或故意刁难时，应如何应对。

### 009. L08【塑造权威06】压力测试的6种应对技巧
承接上一课，本节提供一套具体的应对工具箱。
以下是应对压力测试的六种技巧：
1.  淡定接纳。
2.  反问澄清。
3.  幽默化解。
4.  拔高视角。
5.  直接破框。
6.  暂时搁置。

### 010. L09【塑造权威07】设立愿景
最高层次的权威来自于引领方向。本节学习如何为个人或团队设立一个清晰、动人、有凝聚力的未来愿景。

## 第二部分：给予希望 🌟

### 011. L10【给予希望02】树立共同敌人 & 共同目标
给予希望的有效策略是创造归属感。本节学习如何通过识别“共同敌人”或设定“共同目标”，快速将团队或个人团结在自己周围。

### 012. L11【给予希望03】高绩效示弱
真正的强者懂得适时示弱。本节介绍的“高绩效示弱”是指有策略地暴露无关紧要的弱点或过去的挫折，从而显得更真实、更可信，拉近与他人距离。

### 013. L12【给予希望04】看穿性格，冷读
要给予他人希望，需先了解他人。“冷读术”是一种通过观察和通用性描述，快速建立共鸣、让对方感觉被深刻理解的技术。

### 014. L13【给予希望05】看穿情绪，共情vs投射
除了性格，理解情绪更为关键。本节区分“共情”与“投射”，并学习如何真正洞察他人的情绪状态，从而做出恰当回应。

## 第三部分：认知心态 🧠

### 015. L14【认知心态01】停止纠结，人生Framework
内在认知决定外在表现。本节首先帮助你建立一个清晰的“人生框架”，用于在面临选择和纠结时，快速做出符合长期利益的决策。
公式：**决策 = 框架(选项, 价值观, 目标)**

### 016. L15【认知心态02】摆脱负面情绪——视角转换
负面情绪是主导力的常见障碍。本节学习通过有意识地转换看问题的“视角”，来化解愤怒、焦虑、沮丧等情绪。

### 017. L16【认知心态03】说服力-恐惧链vs欲望链
说服他人的底层逻辑有两种：推动的“恐惧链”和拉动的“欲望链”。
代码描述：
```python
# 恐惧链：如果不做X，就会发生不好的Y
persuasion_fear = “避免[负面后果]”
# 欲望链：如果做了X，就能得到美好的Y
persuasion_desire = “获得[积极收益]”
```
本节将学习如何灵活运用这两种链式逻辑进行说服。

### 018. L17【认知心态04】摸清任何人的需求-人类5大基本动力
要影响他人，必须了解其根本动力。本节介绍人类共通的五大基本心理需求，它们是所有行为的深层驱动力。
以下是五大基本动力：
1.  生存与安全。
2.  爱与归属。
3.  权力与认可。
4.  自由与自主。
5.  乐趣与成长。

### 022. L21【认知心态04】获得真实自信
自信是主导力的基石。本节探讨如何通过积累“微小成功”和调整自我对话，建立不依赖于外界的、真实的自信。

### 023. L22【认知心态05】10倍行动力——消除拖延
想法需要行动来实现。本节提供具体方法，破解拖延症的心理机制，极大提升行动效率。

## 第四部分：树立门槛 🚧

### 019. L18【树立门槛01】让人为你付出-内在一致性与认知失调
让他人愿意为你付出的关键心理原理是“认知失调”理论。当人们的行为与自我认知不一致时，会倾向于改变态度以认可自己的行为。

### 021. L20【树立门槛03】让一群人付出
将个人付出的原理扩展到群体。本节学习如何通过公开承诺、群体规范、渐进投入等机制，引导一个集体共同为某个目标付出。

## 第五部分：专题与内部会议 💼

### 039. L24长期亲密关系-依恋
将主导力模型应用于亲密关系领域。本节探讨依恋理论，理解不同的依恋风格如何影响关系中的权力动态与相处模式。

### 040. 补充课：聊天基础1
回归沟通基础。本节为初学者夯实聊天基本功，包括如何开启话题、延续对话以及避免冷场。
