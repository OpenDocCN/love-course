# 昊哥《主导力训练营》课程笔记：033. 内部会议013 - 如何摆脱他人评判，活在自己的世界里 🧠

在本节课中，我们将一起探讨一个核心社交心理障碍：过度在意他人评判。我们将通过昊哥在公开场合拍摄视频的真实经历，分析这种心态的成因，并学习如何将注意力从“别人的眼光”转移到“自己的表达”上，从而获得更自由、更有感染力的状态。

## 概述：被“他人眼光”束缚的表达

很多人不敢在社交场合中表达自己，无论是搭讪、公开演讲还是日常发言，其核心阻力往往并非能力不足，而是一种内在的恐惧——害怕被周围的人评判。这种心态会严重分散我们的注意力，让我们无法专注于自己想表达的内容本身。

上一节我们探讨了社交中的框架思维，本节中我们来看看如何破除“他人评判”这一具体的心魔。

## 核心困境：活在别人的脑子里，而非自己的

我在北京三里屯人流密集的户外录制视频时，经历了一次深刻的体验。尽管我对搭讪场景已很熟悉，但在公开讲述情感话题时，我依然感受到了巨大的压力。

*   **注意力被抽走**：我的眼睛看着镜头，但余光却在不断观察周围路人的反应。任何人的转头或驻足都会立刻分散我的注意力。
*   **“尴尬”时刻的卡顿**：当念到“渣男”、“泡妞”等可能引发争议的词汇时，尤其是附近有女性经过时，我会下意识地犹豫甚至说不出口。
*   **恐惧的根源**：这种感受的本质是，**我活在别人的脑子里**。我的关注点不是“我想表达什么”，而是“别人会怎么看待我”。

这种状态并非个例。以下是它在不同场景中的常见表现：

*   **在酒吧想认识心仪对象**：心里会想：“我过去搭话，我的朋友/周围的人会不会嘲笑我？那个女生会不会觉得我很奇怪、很饥渴？”
*   **在公司或学校想发言**：心里会想：“我主动发言，同事/同学会不会觉得我爱出风头、在舔领导/老师？”
*   **在KTV或晚会上想表演**：心里会想：“我唱得不好，上去会不会丢人？别人会不会背后议论我？”

**核心问题**：当你的注意力都放在“别人会怎么评判我”上时，你就无法**投入**到当前的事情中，你的表达会变得僵硬、缺乏感染力。你行为的驱动力从内在的“我想做”变成了外在的“别人会怎么看我”。

## 心态转变：从“寻求认可”到“自我表达”

要改变这种状态，首先需要颠覆一个错误的认知。

### 1. 真相：其实没人在意你

我们常常高估了自己在他人眼中的重要性。换位思考一下：

*   如果你在街上看到有人对着镜头说话，你通常只会瞥一眼，**根本不会在意他具体说了什么**，更不会去反驳他。
*   路人对你的注意力通常只有**2-3秒**，随后他们的思绪就会回到“我晚饭吃什么”这类自己的事情上。

**他人评判的恐惧，大多是我们自己内心戏的投射，而非事实。**

### 2. 反转：你的勇气换来的是羡慕

更进一步，当你敢于做那些“出格”的、大多数人不敢做的事时，旁观者的真实心理往往是**羡慕和佩服**。

*   **公司晚会的例子**：当有人自信地上台表演时，哪怕私下有人嚼舌根，但大多数人内心知道：“我做不到他那样，他真敢。”
*   **酒吧搭讪的例子**：当你不敢上前时，如果另一个男生去了并成功互动，你感到的不仅是遗憾，更是**“我为什么这么怂”**的懊恼。那个敢行动的人，在你心里是加分的。

在我们的社会文化中，“枪打出头鸟”、“不要做出格的事”等观念根深蒂固。但每个人的内心都渴望被关注、被欣赏。因此，**敢于打破常规的人，反而更容易获得他人的隐性尊重**。

## 实操方法：如何练习“活在自己的脑子里”

理解了心态，我们需要通过行动来巩固它。以下是两个有效的练习方法。

### 方法一：主动吸引目光，破除恐惧

既然害怕被别人注视，那就**主动去吸引注视**，让恐惧发生，从而发现它并无后果。

*   **具体操作**：在公开场合，刻意提高音量，加入更大的肢体动作。比如录制视频时，可以对着镜头挥手、喊出开场白。
*   **心理机制**：你恐惧的是“被关注”这件事。当你主动引发关注后，你会发现天并没有塌下来，路人看了你几秒后就走开了。这种“脱敏”练习能迅速降低你对他人眼光的敏感度。

### 方法二：区分“事实”与“评判”，建立自我标准

我们必须学会区分客观事实和别人的主观评判，并将自我价值的判断权牢牢握在自己手中。

*   **核心公式**：
    `他人的看法（Judgement） != 事实（Fact）`
*   **举例**：如果你认为“双方自愿的约会”是正常的，而有人说你是“渣男”。这只是他的**主观评判**，而非客观事实。你的行为没有违法，也没有伤害他人。
*   **关键心态**：不要将他人的评判与你的自我价值挂钩。你的好坏，应由你自己的价值观和行为准则来决定，而不是交由路人、同事甚至网络社群来裁决。
    *   很多消费品牌、社群文化都在试图用他们的价值观“殖民”你的思想，让你通过购买产品或认同他们来获得“好”的评价。**真正的心理自由始于“自我定义”**。

## 总结与回顾

本节课我们一起学习了如何摆脱他人评判的阴影：

1.  **识别困境**：我们常常因为害怕被评判而“活在别人的脑子里”，导致无法投入和自信地表达。
2.  **转变认知**：
    *   真相是：**根本没那么多人在意你**。
    *   反转是：你的勇气往往换来的是他人的**羡慕**。
3.  **掌握方法**：
    *   **主动脱敏**：通过刻意吸引目光来破除对关注的恐惧。
    *   **坚守自我**：严格区分“事实”与“评判”，建立并捍卫自己的价值判断标准。

摆脱对他人评判的过度关注，是一个将注意力从外部拉回内部的过程。当你开始**活在自己的脑子里**，专注于你想表达的内容、你想达成的目标时，你才会散发出那种笃定、自信的气场，而这本身才是最具吸引力的。

记住：**真正的自由，来自于你拥有评价自己的最终权力。**
