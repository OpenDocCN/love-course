# 染色体《它先生门徒核心课》：单人夜景篇（二）_ev 📸

在本节课中，我们将学习如何利用“指华修图”和“清单相机”两款软件，对一张夜景人像照片进行后期处理。我们将通过调整色彩、对比度、清晰度等参数，强化照片的氛围感与人物状态，最终得到一张适合分享的精修图片。

## 1. 导入图片与初步调整 🖼️

首先，我们需要打开“指华修图”软件，并导入需要修饰的夜景人像图片。

以下是初步调整的步骤列表：
*   **应用“继续”效果**：为图片增加基础氛围感。注意将效果强度控制在 **40%** 以内，避免画面过曝。
*   **调整饱和度**：适当提升饱和度，但不宜过高，以维持图片原有的色彩基调。
*   **使用“突出细节”**：增强图片结构，使画面更清晰。
*   **应用锐化**：适度锐化，使人物边缘与背景有更明确的区分。

完成上述步骤后，我们进入“调整图片”功能进行整体调节。

## 2. 整体画面调节 ⚙️

上一节我们完成了基础效果添加，本节中我们来看看如何对画面的亮度、对比度等进行整体调节。

在“调整图片”中，请按以下顺序操作：
*   **提升亮度**：将整体画面调亮一些。
*   **增加对比度**：适当增加对比度，让明暗关系更突出。
*   **检查饱和度**：根据画面效果决定是否调整饱和度。本例中，由于原图是在暖光环境下拍摄，饱和度可以保持不变。

这张照片拍摄于类似798的艺术区，其暖色调旨在传达一种空灵、孤独或孤傲的人物状态。构图时，让人物面朝的方向留有更多空间，并利用地面台阶等直线元素，可以增强画面的纵深感与立体感。

## 3. 深化细节与色调 🎨

现在，我们将对画面的局部细节和色彩倾向进行更精细的调整，以进一步突出人物与氛围。

以下是具体的深化调整步骤：
*   **减弱“氛围”**：稍微降低氛围值，可以让人物从背景中更突出。如果增强氛围，人物与背景的融合度会变高，区分则不明显。
*   **调整高光**：不建议将高光调得过高，以免画面刺眼。可以适当减弱高光。
*   **处理阴影**：阴影值向左调是加深，向右调是减淡。减淡阴影能显露更多背景细节，本例中保持原状即可。
*   **调整色调**：将色调向冷色调方向调整，可以强化男性清冷、独立的气质。

调整完成后，如果画面中有因打码产生的小瑕疵，可以使用软件的“修复”功能进行处理，让瑕疵融入暗部，不再显眼。

## 4. 使用清单相机添加滤镜 🧑‍🎨

在指华修图中完成调整并导出图片后，我们进入“清单相机”软件进行最终风格的塑造。

首先，在清单相机中找到已修好的图片（通常位于文件夹末尾）。打开软件后，主要操作为添加并调整滤镜。

以下是滤镜调整的核心步骤：
*   **选择滤镜**：在“人像”或“街头潮流”类目中尝试不同滤镜。本例选择了“不羁”滤镜，并将强度调至最高。
*   **基础调色**：
    *   **亮度**：不宜过高，避免画面过曝。
    *   **对比度**：向右适当增强。
    *   **清晰度**：轻微增强，注意值过高会导致地面出现锯齿。
    *   **锐度**：轻微增强以硬化边缘，同样需避免过高产生毛边。
*   **曲线调整**：轻微下拉曲线底部，使暗部更深沉。
*   **色彩倾向**：
    *   **色温**：向左（冷色调）调整。
    *   **色调**：向左（黄绿色方向）微调。
    *   **高光**：保持不动。
    *   **阴影**：向右微调，减淡阴影以透出些许细节。
*   **颜色分级**：
    *   **高光颜色**：选择偏暖的黄色，为亮部增添倾向性。
    *   **阴影颜色**：选择偏冷的紫色，与高光形成冷暖对比。
*   **其他选项**：本例中，“暗角”和“虚化”效果均不适用，无需调整。

## 5. 最终检查与导出 ✅

经过以上步骤，图片已基本修饰完成。最后检查整体效果：人物面部光线集中，深色裤子与地面光形成对比，人物形象鲜明。确认无误后，即可保存最终成品。

---

本节课中，我们一起学习了夜景人像的完整后期流程。我们从指华修图的基础调整开始，逐步深入到整体画面调节与细节优化，最后利用清单相机添加风格化滤镜并完成色彩分级。核心在于通过**控制对比度、清晰度**和**营造高光暖、阴影冷的色彩对比**，来强化照片的空间感、立体感以及人物想要传达的孤独、冷峻的状态。请同学们多加练习，尝试不同参数组合，找到最适合自己照片的风格。
