# 染色体《它先生门徒核心课》：旅拍单人篇_ev_note

在本节课中，我们将学习如何使用手机修图软件，对旅拍单人照片进行后期修饰，以提升照片的视觉冲击力和氛围感。我们将主要使用“型男相机”和“指划修图”两款软件，通过滤镜选择、色彩调整和细节增强等步骤，完成一张具有现代感的旅拍照片。

## 软件准备与基本思路

上一节我们介绍了课程目标，本节中我们来看看需要准备的软件和基本修图思路。

我们将使用“型男相机”和“指划修图”两款手机软件。Photoshop可以单独使用或与它们结合使用。通常，我会同时使用这两个软件，因为它们各有优势，可以互补功能。

以下是开始修图前的准备工作：
1.  打开“型男相机”软件。
2.  从相册文件夹中导入需要修饰的原始图片。

## 第一步：在“型男相机”中进行初步调整

现在，我们进入第一个软件的修图流程。首先从添加滤镜开始。

打开图片后，点击“修图”功能。第一步是添加滤镜。选择滤镜时，建议逐个尝试，没有绝对好坏的滤镜，关键在于找到符合个人喜好的视觉效果。

我个人偏好暗调、复古或未来现代的风格。因此，我会从这些风格的滤镜中寻找灵感。每个滤镜营造的氛围都不同，需要耐心尝试，这是决定照片效果的关键步骤。

经过尝试，我选择了“迷雾”滤镜。将滤镜强度调到最高，然后开始手动调色。

以下是具体的参数调整步骤：
1.  **亮度**：适当提高亮度，可以增强视觉冲击力。
2.  **对比度**：调高对比度会使亮部和暗部细节减少，画面更“硬朗”；调低则细节更丰富。为了整体效果，通常可以让细节稍少一些。
3.  **锐化**：增加锐化度，让画面看起来更清晰锐利。
4.  **曲线**：调整曲线，将暗部提亮一些，同时将亮部稍微压暗。
5.  **色彩**：
    *   **色温**：调暖一些（偏红黄），可以带来厚重感。由于原图背景是冷调的蓝绿色，用暖色调平衡可以增加稳重感。
    *   **色调**：可以调冷一点。
    *   **饱和度**：提高饱和度，让天空等部分的颜色更纯，与大面积的纯色和模特形成对比。
6.  **高光与阴影**：
    *   **高光**：控制受光部分的亮度。调低高光，亮部不会过曝；调高高光，对比更强。
    *   **阴影**：调低阴影值，暗部会加深；调高则暗部变淡。为了营造未来感，可以让阴影深一些。
7.  **色彩平衡**：
    *   **亮部颜色**：当画面有大面积纯色时，可以在亮部加入其互补的暖色调（如肉色），以平衡画面，避免刺眼。
    *   **阴影颜色**：让阴影部分微微偏绿，可以避免死黑，增加味道。
8.  **其他选项**：“淡化”会让颜色发灰，若需要明确对比则无需调整。“暗角”是为四边添加暗角，本例中不需要。
9.  **光影特效**：软件内的“光影”特效可以为照片烘托气氛。可以尝试添加光源（如补一个背景光），观察效果。但根据本图“清晨出发”的场景，我决定不加特效。

调整满意后，保存图片。

## 第二步：在“指划修图”中进行精细修饰

初步调整完成后，我们进入第二个软件进行精细处理，让照片质感更进一步。

打开“指划修图”软件，导入上一步保存好的图片。

我习惯首先使用“戏剧效果”滤镜。这个滤镜可以增加画面的对比层次。不需要强度太高，通过滑动滑块微调，有一点效果即可。同时，可以增加一点**饱和度**，让天空颜色更纯粹、画面更明快。

接下来，使用“突出细节”功能。
*   **结构**：增加结构值，可以看到云的颗粒感和路面的肌理变得更明显。
*   **锐化**：增加锐化值，使人物形象等轮廓更加清晰。

然后，使用“局部调整”功能对特定区域进行优化。
1.  将调整点放在模特脸部的暗区。
2.  调整该点的作用范围大小。
3.  对该区域进行微调：**提高亮度**、**略微增加对比度**、**增强饱和度**、**明确结构**。

调整完成后点击确定，此时图片的清晰度和细节已经比之前更出色。

最后，对图片进行整体调整。在工具菜单中：
1.  **亮度**：整体提高一点亮度。
2.  **对比度**：增加对比度，使画面更整体。
3.  **饱和度**：再适当增加饱和度，强化现代动漫感，但注意保留细节。
4.  **氛围**：调高“氛围”值，暗部细节会增多、变淡；调低则暗部加深、细节减少。根据需要稍加一点即可。
5.  **高光**：提高高光，让受光部分更亮。
6.  **阴影**：提高阴影值，让暗部稍微变亮一点。
7.  **色调**：色调偏冷（右）或偏暖（左）。因为是清晨，太阳光偏暖，但我们已经营造了亮调画面，可以稍向冷调调整一点。

修饰至此完成。长按图片可以对比修图前后的效果，差异非常明显。

## 拍摄构图技巧补充

在结束修图教程前，我们再来回顾一下拍摄时的构图要点，这对最终成片效果至关重要。

由于模特的视线看向前方，构图时应遵循“视线前方留白更多”的原则。同时，要避免人物和物体在画面中平均分布，整体构图应偏向视线方向（即画面左侧多留空间），以增强纵深感。

为了进一步加强这种纵深感，可以寻找类似桥梁围栏、具有透视结构的建筑作为背景。利用“近大远小”的透视原理，即使是一张静态图片，也能拍出动态感和带入感。

## 总结与导出

本节课中，我们一起学习了旅拍单人照片的完整修饰流程。

我们首先在“型男相机”中通过尝试滤镜和调整基础参数，奠定了照片的色彩基调与氛围。接着，在“指划修图”中利用戏剧效果、细节增强和局部调整等功能，让画面的清晰度、对比度和细节质感得到了全面提升。最后，我们还回顾了拍摄时关于视线引导和透视构图的技巧。

**最终导出时请注意**：在保存页面，请选择“**创建副本**”，这样不会覆盖原始图片。如果直接“保存”，则会替换原图。

掌握这项技能后，你不仅能创造出具有个人风格的优质照片，还能在分享过程中展示你的审美与能力。下节课，我们将探讨旅拍中双人照片的构图、拍摄与修饰技巧。
