# 《短期关系》第三课：准备工作 🛠️

在本节课中，我们将学习正式邀约前必须完成的准备工作。这些准备是建立短期关系的基础，直接影响到后续的成功率。我们将从展示面设置、社交软件选择以及邀约时间段三个方面进行详细讲解。

## 展示面设置 📸

上一节我们介绍了短期关系的核心理念，本节中我们来看看如何通过展示面进行有效吸引。展示面是你在社交平台（如微信朋友圈）上呈现的个人形象，是吸引异性的基础。无论追求何种关系，展示面至少需要达到及格水平（**60分**）。在短期关系中，展示面的设计有特定方向。

以下是四个关键的展示面设计方向：

1.  **正确使用预选**
    *   预选的定义是：**与两个或以上女性的合影**。这是强有力的吸引信号。
    *   错误示范是：与单个女性的单独合影。这容易让目标误以为你有固定伴侣或暧昧对象，在长期关系导向的互动中是重大忌讳。
    *   预选最适合用于短期关系场景，能有效传递你的吸引力和可得性。

2.  **植入夜场元素**
    *   最理想的预选拍摄地点是夜店、酒吧或KTV。
    *   可以发布在这些场所的自拍，或与女性朋友显得亲密、愉快的合影（需确保对方配合，表情自然，距离贴近）。
    *   夜场环境本身充满**性暗示**，此类展示面能明确传递你对于短期关系的开放态度。

3.  **展示身材优势**
    *   如果你有较好的身材（精瘦、有线条感为佳），应通过照片展示出来。
    *   示例拍摄方法：从较高角度拍摄侧脸，同时展现胸肌、腹肌等部位。
    *   如果身材普通，可通过修图、滤镜和拍摄角度优化。偏胖者可通过穿着紧身衣物塑造结实、强壮的感觉。

4.  **注重穿着与配饰**
    *   穿着打扮应时尚、有型。
    *   注重饰品点缀，如墨镜、手表、项链、手链等，能让你看起来更酷、更有魅力。
    *   有纹身也会增加对部分追求短期关系女性的吸引力。

关于展示面的具体拍摄和修图技巧，本课程不做深入展开。如需系统学习，推荐参考专业展示面设计师（如田正远）的相关课程。

## 社交软件选择 💬

讲完了展示面的拍摄方向，接下来我们看看通过哪些软件来接触潜在目标。我们需要选择合适的平台，并将资源有效集中。

以下是主要使用的社交软件及策略：

1.  **陌陌**
    *   这是历史较久的软件。目前每人每天只能发送**20个**打招呼信息。
    *   陌陌上的女性普遍因收到大量搭讪而显得“傲娇”，反应可能不佳（如回复问号或冷漠表情）。
    *   策略：**不能一上来就要微信**。需要在陌陌上先进行一些互动，建立基本联系后，再尝试将聊天转移到微信。

2.  **探探**
    *   这是当前主流软件。优势在于**双方互相喜欢后才能开始聊天**，匹配质量相对更高。
    *   策略一：匹配后简单沟通，然后直接索要微信（例如：“探探不常看，加个微信吧”）。
    *   策略二：匹配后直接打招呼并附上微信号，对方若有意自会添加。

**核心操作：将所有在陌陌、探探上接触到的资源，最终都引导至微信。** 可以将微信理解为你精心维护的**主鱼塘**，而其他软件只是**捕鱼的工具**。将所有“鱼”汇集到微信这个统一、展示面丰富的环境中，便于后续集中管理和互动。

## 邀约时间段 ⏰

最后，我们来探讨邀约的最佳时间段。选择正确的时间出击，能事半功倍。

邀约主要分为两个时间段，各有策略：

1.  **高风险高回报时段：凌晨0点至3点**
    *   在这个时间段能成功约出来的女性，**窗口（兴趣和可得性）非常大**。
    *   特点：约出后通常无需过多铺垫，可直接前往私密空间。但成功约出的概率相对较低。

2.  **常规操作时段：晚上9点至12点**
    *   这是更常用、更稳妥的时间段。
    *   策略：通常先进行一项公开社交活动，例如共进晚餐或在小酒吧喝一杯。目的是建立舒适感和联系，之后**再自然过渡到私密空间**。

关于如何在凌晨时段直接邀约，以及从公开场合转场至私密空间的具体操作技巧，我们将在下一节课程“入门级邀约实战”中详细讲解。
