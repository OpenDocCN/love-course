## 推荐学习路径

对于初学者，建议按照以下顺序进行学习，以确保基础扎实，循序渐进：

1.  **第一阶段：心态建设**
    *   从 **模块一《情感法则》** 开始，建立正确的核心心态和认知框架。
    *   学习 **模块七《前期知识课程》**，了解基本概念和场景差异。

2.  **第二阶段：原理理解**
    *   深入学习 **模块二《自然吸引原理》**，理解吸引力的底层逻辑。

3.  **第三阶段：基础实战**
    *   进入 **模块四《新手入门 Starter》**，进行最基础的步骤练习。
    *   同步学习 **模块八《常见错误与雷区》**，避免踩坑。

4.  **第四阶段：专项提升**
    *   根据你的目标选择：
        *   长期关系/全面技巧：学习 **模块六《恋爱训练营实战课》**。
        *   短期关系/速推技巧：学习 **模块五《短期关系方法》** 及 **模块九** 的相关视频。
        *   跨文化社交（英语）：学习 **模块三《PURE GAME 实战技巧》**。
        *   线上社交：学习 **模块十《网约专题》**。
