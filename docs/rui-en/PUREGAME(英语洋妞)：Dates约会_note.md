# PURE GAME 课程 02：约会 💑

在本节课中，我们将要学习关于约会的基本概念和核心原则。约会是一个双方建立联系、加深了解的过程，其核心在于真诚的互动与清晰的意图表达。

## 核心原则：意图的重复确认

上一节我们介绍了课程概述，本节中我们来看看约会的核心沟通原则。这一原则强调，在关键行动或承诺之后，需要通过重复陈述来强化和确认意图。

其核心行为模式可以总结为以下公式：

**行动 + 重复声明(意图) = 强化印象**

或者用一段伪代码来描述这个交互逻辑：

```python
while on_date:
    complete_an_action()  # 例如，吃完晚餐
    for i in range(3):    # 重复多次
        state_intention_clearly() # 清晰地陈述你的意图
```

这个模式的意义在于，通过语言对行动进行注解，可以避免误解，并让对方明确感知到你的后续计划和诚意。

## 原则的应用场景

理解了核心模式后，我们来看看它如何应用于实际约会环节。以下是该原则的一些具体体现：

*   在共同完成一项活动（例如用餐）后，立即重申接下来的计划或对你的感受。
*   在表达一个重要观点或承诺后，换用不同的措辞再次强调，确保信息被接收。
*   通过多次温和的确认，建立可靠和一致的沟通氛围。

## 总结

本节课中我们一起学习了约会中的一个基础沟通原则：**在行动后重复你的意图**。这种方法旨在通过清晰、一致的沟通，减少不确定性，从而建立起更真诚和有效的互动关系。记住，关键在于让语言和行动保持一致，并通过重复来加深理解和信任。
