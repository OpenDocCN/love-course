# 馄饨哥真男人教学课程（更新版）：第九集不花钱如何yue会_note

在本节课中，我们将要学习一个核心技巧：如何在不花费金钱的情况下成功与女生约会。我们将探讨心态建设、具体操作步骤以及如何识别并应对不合适的约会对象。

---

上一节我们介绍了约会的基本概念，本节中我们来看看如何实现“零成本”约会。

不花钱约会这件事，对所有男性都很有吸引力。但许多男性缺乏勇气去实践。实际上，即使对现在的我而言，不花钱约会也具有挑战性。因为通常约女生出来需要吃饭、看电影等一系列花费。在学校里约会也是如此。

学习了真男人把妹技巧后，我的做法是：在约出来之前，就让对方明确知道我是因为喜欢她才邀约。约会时选择散步，在散步过程中，根据内心感受进行一系列**肢体接触**，并观察对方的反应。这就是基本流程。

在学校或社会上把妹都很简单。直接约对方出来散步、溜达。这不需要花费金钱和时间。

以下是传统约会与真男人约会的对比：
*   **传统方式**：和女生见面，通常是一起看电影或吃饭。
*   **真男人方式**：晚上约出来散步、聊天，就是一种很好的约会。

但有些学员会认为，面对分数较高的妹子，必须通过请客吃饭看电影来展示财力，妹子才会喜欢。我以前也思考过这个问题：如果依靠这种方式吸引妹子，那岂不是在进行**性交易**？你必须达到妹子的某些要求，才能与她发展关系。我认为这是一件非常猥琐、低级且缺乏自信的人才会做的事。

大家都知道我的例子：曾用7块钱请妹子到家里喝粥，后来也是用7块钱。我们去夜店玩时，妹子也是来我家喝粥就解决了所有事情。之后又用18块钱请她们在家吃麻辣烫，然后发生了一些事情。这些妹子都是搭讪认识的。

妹子只会在聊天过程中问一些无关紧要的问题，比如“你经济收入怎么样”、“家里有几套房子”。对于这些问题，你只需如实回答。因为妹子这种“筛选男人”的思路其实是受他人影响的，她本身可能并不真正关心这些。妹子并不关心你有没有钱。如果妹子真的是因为钱才跟你在一起，这种妹子就不值得要。因为她图的不是感情，总有一天你没钱了，她也会离开。

你必须用真实的自己去吸引妹子。所以男人不要怂，直接尝试不花钱把妹。这能飞快锻炼你的约会技巧和肢体进攻能力。这才是真正的把妹。

那么，具体如何操作呢？不花钱把妹，可以花点小钱，比如请妹子喝瓶水。如果妹子坚持让你为她买礼物，那这个妹子就有问题，需要警惕。

例如在微信上约妹子，我说“请你吃饭”。即使说吃饭，我也很“鸡贼”。我直接把我家楼下的位置定位发给妹子让她过来。等妹子到了，我说“上楼坐坐吧，等位太饿了”。所有学员都知道我的这个小套路。点个外卖也就十几块钱，花不了什么钱。

但是，如果在微信聊天时，妹子斩钉截铁地一直问“我们去哪里吃”、“你先订好餐厅”，你就要开始警惕了。这就有问题，她可能不是冲着你这个人来的。我会觉得这样的妹子很**low**。

我们的观念是：我们不贪图妹子的钱，妹子也不应该贪图我们的钱，双方应该处在平等地位。当妹子在聊天中提到需要你金钱付出时，你心里就要对她有所判断了。这已经不是正常的妹子了。

当然，如果微信上妹子问订什么餐厅，你也可以敷衍几句。既然她这样，你也可以“使坏”。说“订好了”，让她直接过来。来了之后再找借口推脱，比如“刚才位置被人占了”，总之就是不花钱。一定要坚持这条路。

就是点外卖，爱吃不吃，不吃就走。我之前有个妹子，第一天约会我比较怂，去酒吧喝了两杯酒，花了100多块钱，我当时有点崩溃。第二天我请她吃饭，打算不花钱，就点外卖。妹子直接问：“你订的什么餐厅？”我说没有餐厅，吃外卖。她说不想吃外卖，就要去餐厅吃。我说：“那你就走吧。”懂吗？不要委屈自己去花钱。

当然，如果你心甘情愿为她花钱，那没人管得着。但我们最基本的原则是：你不贪图我的钱，我也不贪图你的钱，大家和平共处，地位平等。当她想要贪图你的东西时，就是有问题的，不是为了你这个人，而是为了你身外的物质。这是错误的，不符合我们把妹的理念。

现在许多世俗观念，包括很多妹子都认为：跟一个男人在一起，男人对她好、为她花钱是天经地义的。我认为这种思维非常错误。在我们真男人这里，男女绝对是平等的。凭什么我要给你花钱？男女平等。女的也不用为我花钱。难道你没有自己的经济实力吗？不会工作吗？没钱吗？就这么穷吗？

但是很多女生会说：“以后在一起，你不给我花点钱，那我为什么要和你在一起？”那就不要在一起，就这么简单。难道我一定要花钱才能跟你在一起吗？

有些妹子在约的时候会说：“你把打车费付了”或者“你先给我发个红包”。这就有问题。我的回应是：“我不喜欢跟穷志的人交朋友”，然后结束。就这么简单。我跟你这么**low**的人干嘛？在你向我索取价值之前，你并没有提供给我任何价值。除非你发些好看的照片，我高兴了，也许愿意花点钱跟你玩玩。整天只知道索取价值，我肯定不开心。就这么简单。

上一节我们探讨了不花钱约会的核心理念，本节中我们来总结关键心态和行动要点。

不花钱把妹这件事，首先你心里要认定这是非常顺理成章的，就应该这样。也就是说，**心态**一定要建设好一个真男人的心态。按照自己的游戏规则走。当妹子提出不符合你要求、必须让你花钱的时候，你肯定要不开心。那就把这种不开心展示给妹子看，让她知道这件事是错的。

不用去学其他人。你要学其他男人的方式，就去报其他导师的课，学他们花钱把妹。我这边就是**屌丝把妹**，不花钱把妹，做个真男人把妹。而且我们虽然是“屌丝”，经济上可能不富裕，但我们在思想等其他方面是富有的、充实的、健全的。反倒是那些有点钱、但把妹方式很**low**的人，才是真正的“屌丝”。听懂了吗？就这么简单。
