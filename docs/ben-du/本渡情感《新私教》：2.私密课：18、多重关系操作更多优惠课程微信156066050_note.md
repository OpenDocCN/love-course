# 本渡情感《新私教》：2.私密课：18、多重关系操作

## 概述
在本节课中，我们将学习如何根据与女生发生关系后的不同感受，进行关系定位和操作。

## 关系定位
### 定位关系
在宣传封面中，我们看到的关系定位包括情人、长期关系等。第一个字母是P。

### 学习目的
学习这一篇章的原因是，即使掌握了很多技巧，也不意味着每次都要与女生确定一段关系。有些女生可能更适合短期交往。

### 关系分类
在发生关系后，可能会产生以下四种感觉：
1. 无感：不想再发生关系。
2. 情人/毛友：享受在一起的感觉，但不想谈恋爱。
3. 想谈恋爱：想与她建立长期关系。
4. 长期关系：适合结婚。

## 关系操作
### 无感关系
- 不主动提及关系，假装没有发生过。
- 如果女生询问，诚实地告诉她只是睡一觉的关系。

### 情人/毛友关系
- 告诉女生你享受和她在一起的感觉，但不想谈恋爱。
- 解释不想谈恋爱的原因，如不想束缚对方。

### 想谈恋爱关系
- 直接向女生表白，表达你的感受和想法。
- 频繁约会，增加相处时间，让她对你产生投资。

### 长期关系
- 向女生提出要求和诉求，让她知道你需要她做什么。
- 展示你的规划和打算，让她看到你的潜力。

## 总结
本节课中，我们学习了如何根据与女生发生关系后的不同感受，进行关系定位和操作。关键在于敢于表达自己的想法，并付诸行动。希望这节课的内容能对你们有所帮助。
