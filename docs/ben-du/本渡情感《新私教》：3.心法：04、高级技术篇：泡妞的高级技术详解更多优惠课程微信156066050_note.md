# 本渡情感《新私教》：3.心法：04、高级技术篇：泡妞的高级技术详解

## 概述
在本节课中，我们将学习如何运用高级技术来吸引和留住女性的心。

## 1. 制造不确定性
**原理**：通过制造不确定性，让女性永远不知道你在想什么。

**公式**：不确定性 = 永远不知道你在想什么

## 2. 引导情绪投资
**技巧**：引导女性对你进行情绪方面的投入和投资。

**原理**：女性在你身上花费的时间、情感、经历、金钱和情绪都属于投资。

**公式**：情绪投资 = 时间 + 情感 + 经历 + 金钱 + 情绪

## 3. 朋友圈预选作弊技巧
**技巧**：让目标发现你身边还围绕着其他女性，并对你感兴趣。

**方法**：
1. 在朋友圈分享多人聚会的小视频。
2. 在朋友圈发布自己给自己买花的照片，并附上表白性话语。

## 4. 让目标对你越来越有兴趣的技巧
**技巧**：当女性不找你时，你也不找她；当她找你时，你对她特别热情。

**原理**：通过制造落差感，让女性为你投资，并产生不确定性。

## 5. 不同约会制造落差感
**技巧**：
1. 在约会中直接牵手或拥抱。
2. 约会结束后，不主动发消息，等待女性主动联系。
3. 在聊天中表达好感，但不要过于热情。

## 总结
本节课中，我们学习了如何运用高级技术来吸引和留住女性的心。通过制造不确定性、引导情绪投资、朋友圈预选作弊、让目标对你越来越有兴趣以及不同约会制造落差感等技巧，我们可以提高成功率，并建立稳定的关系。
