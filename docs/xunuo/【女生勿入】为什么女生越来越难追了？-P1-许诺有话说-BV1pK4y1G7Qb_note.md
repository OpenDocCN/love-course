## 案例背景与分析

我的朋友在一家互联网公司担任普通职员，22岁，外貌不错。

一天临近下班时，她的老板以工作为由，将她叫到办公室谈心。等到其他员工都下班后，老板突然向她表白。

老板告诉她，虽然她只入职一个月，但从她来上班的第一天起，自己就注意到她了。老板表示自己非常喜欢她，希望她能成为自己的女朋友。

我的朋友当时没有答应，也没有拒绝。她表示事情有些突然，自己还没想好，感觉进展太快。

她后来将此事告诉了我。我开玩笑说，她可能要嫁入豪门了。但我也提醒她，很多有钱的老板会利用自己的财富，追求刚步入社会、涉世未深的年轻女孩，她需要保持警惕。

第二天她告诉我，早上到公司时，发现老板为她准备了爱心早餐和便当，放在她的桌上。我问她是怎么想的，她的原话是：“我们老板年轻有为，也挺有钱的。如果对我是真心的，也不是不可以做他女朋友。”

两天后，老板将她的月薪从**4000元**直接调整到了**8000元**，并将她从忙碌的销售岗位调到了相对清闲的职能部门。她的休假也从单休变成了双休。随后，老板再次向她表白。

但我的朋友依然没有答应成为他的女朋友。

这时，她找到我，说感觉老板似乎是真心的。我告诉她，这需要她自己考虑，如果觉得老板人不错，可以尝试相处。我的朋友想了想，说：“也行，那就下周和他试试看。”

我本以为他们会就此在一起。但接下来发生的事让我大跌眼镜。

老板见她仍未答应，便开始赠送各种高档的包、香水和化妆品。他甚至表示，只要她愿意做女朋友，条件可以随便提。

老板开始向她展示自己的资产，说自己有三套全款购买的房产，存款有几百万，目前月收入是**30万元**，并且公司处于上升期，未来会更好。

自此之后，我的朋友再和我谈起此事时，她考虑的重点已经不再是是否要和老板在一起。

她开始觉得，即使不做他的女朋友，也能享受他带来的物质好处。而且她还很年轻，刚步入社会。她心想：“一个月收入30万、有几套房、有几百万存款、有事业的男人都这样追求我，我好像能找到更好的。”她打算一边接受对方的好意，但不用确定恋爱关系，同时再看看有没有其他更优秀的追求者。

这就是我朋友完整的心理变化过程：最初因为老板展示出的价值，在思考对方是否真心，并考虑是否接受这段关系；到后来，老板持续的过度付出，让她自信心急剧膨胀，觉得自己像“金凤凰”一样珍贵。

一个如此优秀、富有的男性不断“跪舔”，反而让女生逐渐觉得对方配不上自己了。

## 核心心理机制：价值感知与“降价”效应

上一节我们通过案例看到了过度付出的全过程，本节我们来剖析其背后的核心心理机制。

许多男性自身条件其实很好，完全配得上他们喜欢的女生。女生一开始也表现得很热情。但只要男性一旦开始放低姿态、过度讨好（即“跪舔”），女生立刻就会对他失去兴趣。

这可以用一个简单的市场交易比喻来理解：

> 你去古董市场，问老板一件商品的价格。
> 老板说：“5000块。”
> 你尝试讲价：“2000块行不行？”
> 老板回答：“2000块卖不了，最低4500。”
> 经过一番讨价还价，你最终以3500元成交。此时，你依然会觉得自己“赚到了”。

**同一个场景，另一种情况：**
> 老板说：“5000块。”
> 你问：“2000块行吗？”
> 老板想都没想就立刻答应：“行！”
> 此时，你会怎么想？你会怀疑：“如果我真2000块买了，是不是我亏了？老板答应得也太痛快了。”

所以，现在你明白为什么“舔狗”往往追不到女生了吗？

**核心公式：**
```
女生感知的伴侣价值 = 对方客观价值 - 对方表现出的讨好程度
```
过度讨好（跪舔）的行为，会在女生心中无形地降低你的客观价值，让她觉得“和你在一起，是我亏了”。

## 课程总结与建议

本节课我们一起学习了在亲密关系建立初期，“过度付出”和“跪舔”行为如何导致吸引力下降的心理原理。

关键在于保持自我价值的框架，进行平等、尊重的互动，而不是通过单方面的、无底线的付出来换取关系。健康的吸引建立在相互欣赏和价值的对等感知之上。
