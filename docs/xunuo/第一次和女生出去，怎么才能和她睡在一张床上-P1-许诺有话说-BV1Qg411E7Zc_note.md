## 开房策略：从提问到行动

1.  **预先沟通，给予选择**：在预订前说明安排与理由，体现尊重。

## 意图识别：行为背后的逻辑

**女生好感度 + 信任度 > 心理防备阈值 → 不拒绝同住一个房间**
