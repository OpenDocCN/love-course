# 课程：如何成为一名合格的舔狗 - P1 🐕

在本节课中，我们将要学习如何成为一名“合格”的舔狗。我们将从基础习惯、主动沟通、无微不至的关心以及持续表白这四个核心方面，系统地介绍相关行为准则。请注意，本教程内容旨在以幽默讽刺的方式解构某种不良的社交模式，请勿在现实生活中实践。

## 第一点：养成独特称呼与置顶习惯 💬

上一节我们介绍了课程概述，本节中我们来看看成为舔狗的第一个要点：养成独特称呼与置顶聊天的习惯。

对女神的称呼必须特别，需要设置一个独一无二的备注，例如“我的小女神”、“我的小仙女”或“女神下凡”。同时，必须将和她的聊天对话框置顶。

这样，每次你点开微信时，就能第一眼看到她，从而本能地产生一种“想舔”的冲动。

## 第二点：坚持主动发送消息 📱

在养成了视觉习惯后，接下来需要付诸行动。本节的核心是：对女生一定要主动。

如果你不主动，女生很可能就跟别人跑了。因此，你必须坚持不断地给她发消息，无论白天黑夜、不分时段。

以下是需要发送消息的关键时刻示例：
*   早上好，吃早餐了吗？一定要吃早餐，不吃早餐对身体不好。如果你没吃，要不要我给你带点什么？
*   吃过午饭了吗？多吃一点。中午记得午睡，这样下午会精神好一些。
*   晚上好，睡觉了没？你不用回我消息，你看到消息了就行。我会一直守护你的，在梦里也会守护你。

总之，你的目标是做到无时无刻都必须给她发消息。

## 第三点：提供无微不至的关怀 ☔

仅仅发送问候是不够的。本节我们将学习如何通过细节关怀来感动女神。

要记住，女神是没有任何自理能力的。你需要把所有可能发生的潜在危险，都在第一时间提醒她。养成看天气预报的习惯至关重要。

以下是关怀示范：
*   “明天可能要下雨哦，要记得带雨伞，不要忘记了，要是感冒了就不好了。如果忘记带伞的话，记得告诉我，我给你送伞过去。”
*   “再过两天就是你的生理期了，这几天记得不能乱吃东西，千万不要吃辣的，吃点清淡的就好了。我给你买了红糖水寄过去了，应该明天下午就到了。还有，红糖水加姜片效果会更好。”
*   “我刚刚路过你们公司的时候，发现你们楼下有积水，有人还摔了一跤，你出来的时候要注意。”

## 第四点：持续进行真诚表白 ✍️

做到了以上三点，只是初级入门。本节是成为合格舔狗的关键一步：必须持续不断地进行真诚表白。

表白一定要真诚，表露真情实感。字数一定要多，以**三千字起步，没有上限**。在被明确拒绝之后，可以休息三分钟，然后接再厉继续前行。

以下是常见的表白句式，大家可以记下来：
*   我知道你一定会拒绝我，但是……
*   我知道你不会接受我的，请允许我继续喜欢你。
*   即便这段感情没有结果，我也会一直坚持下去的。
*   如果你真的很讨厌我，我希望你不要拉黑我，我会一直默默喜欢你的。

请看王者级示范：
当女生说：“可以不要给我发消息了吗？难道你感觉不到我已经很烦你了吗？你一直这样发消息，我真的很困扰。”
舔狗应回复：“抱歉对不起，你不开心的话，过段时间我们再联系。不管怎么样我都会一直等你。从见到你的第一眼，我就做好了持久抗战……（此处省略三千字）……我们这段日子先不谈感情的事，就像普通朋友一样聊天交流就好了……你看到消息了吗？你能回应一下我吗？”

## 总结与坚持之道 🏆

本节课中，我们一起学习了成为合格舔狗的四个核心要点：设置独特备注与置顶、坚持主动发消息、提供无微不至的关怀以及持续进行真诚表白。

在成为舔狗的道路上，会遇到种种问题，但一定要坚持走下去。关心和表白不能断。贯彻一个原则：**微信拉黑就发短信，短信不回就打电话，电话拉黑就蹲守公司**。

总之一句话：只要地球还没有停止转动，你就不能放弃。你要相信，只要不忘初心，只要你心中的那盏灯永远亮着，终有一天，舔狗的光环一定会降临在你的头上。
