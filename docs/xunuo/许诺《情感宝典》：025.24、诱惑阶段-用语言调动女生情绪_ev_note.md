## 语言情绪的重要性

上一节我们介绍了诱惑阶段的基本概念，本节中我们来看看沟通中的核心要素——情绪。

心理学研究表明，人际沟通的影响由三部分组成：语言内容占7%，语气占38%，肢体语言占55%。由于我们讨论的是微信纯文字聊天，肢体语言的因素可以暂时忽略。

在微信聊天的语境下，文字内容本身的影响力约占20%，而文字所承载的“语气”或情绪则占据了高达80%的比重。因此，你在微信聊天中使用的“语气”至关重要。语气运用得当，情绪自然饱满。

我们给女生发送一条信息，本质上是进行情绪的沟通，或者说是一种潜在的感受交流，而不仅仅是传递字面上的信息。许多男生在与女生聊天时，话语中完全缺乏情绪。

## 缺乏情绪 vs 富有情绪的对话

以下是两种表达方式的对比，可以清晰地展示语言情绪的差异。

*   **缺乏情绪的对话**： 发送“我叫许诺，你呢？”。这句话仅仅传递了基本信息，显得平淡、正式，缺乏情感色彩。
*   **富有情绪的对话**： 发送“你好，我是羞涩的许诺~”。作为接收方的女生，除了知道你叫许诺并可能回复名字外，更能感受到一种俏皮、轻松和有趣的氛围。

女生本质上是情绪的动物。当语言情绪到位时，许多事情都会变得顺利。

## 如何掌握语言情绪

那么，如何才能学会运用语言情绪呢？或者说，女生喜欢怎样的语言情绪呢？这就是我们今天要解决的核心问题。

我总结了以下四种女生喜欢的语言情绪模式。它们之间的关系就像烹饪中的调味料。没有情绪，对话就寡淡无味；加入了语言情绪，对话立刻就变得鲜活起来。

### 1. 日常用语模式

日常用语的对立面是正式或书面用语。许多男生在与女生打招呼时，会使用“你好，很高兴认识你”这类非常正式的语句，这无形中拉开了双方的心理距离。

日常用语就是你平常生活中会自然使用的语言。它能有效传递情绪并拉近关系。

以下是日常用语的一些例子：

*   **表达情绪**： 使用“我靠”、“晕死”等词表达惊讶或无奈。
*   **称呼对方**： 使用“猪头”、“笨蛋”等亲昵的调侃称呼。
*   **使用方言**： 例如上海话中，称呼小女孩为“囡囡”，表达愤怒时说“册那”，说人笨是“港督”。

### 2. 网络用语模式

网络用语在聊天中妙用无穷，能瞬间让对话充满时代感和情绪。

以下是与女生聊天时可以使用的一些网络用语示例：

*   **称呼**： “老铁”、“集美”。
*   **活动**： 去酒吧喝酒可以说“尬酒”。
*   **评价**： 说别人厉害是“牛逼”、“社会”。
*   **大笑**： “哈哈哈”或“23333”。
*   **表达好感**： “我对你有点上头”。
*   **调侃**： 在女生极力辩解时，可以说“求生欲很强”。

需要注意的一点是，尽量避免使用过于过时（如“神马都是浮云”、“给力”、“666”）或含义低俗的网络用语。

### 3. 非逻辑语言模式

女性在情感交流中往往不遵循严格的逻辑。在调情时，如果你过于逻辑化，就会阻碍情绪的流动，无法打开她的情感开关。

什么是非逻辑的语言模式呢？它指的是话语之间没有直接的因果或理性关联，却能引发好奇和情绪互动。

以下是两个非逻辑对话的案例：

*   **案例一**：
    *   男：“今天洗澡的时候，在镜子里看到你不穿衣服的样子。”
    *   （解析：自己洗澡和女生不穿衣服并无逻辑关联，这会引发女生的好奇。）
    *   女：“？？？这跟我有什么关系？”
    *   男：“啧，发现我们胸围居然差不多了。”（顺势调侃，开启互动）
*   **案例二**：
    *   男：“那你被子盖好点，小心我从你被子里钻出来。”
    *   （解析：盖被子与从被子里钻出来没有逻辑关系，但能制造暧昧和有趣的想象空间。）

### 4. 表情包（动态表情）模式

如今，没有表情包的聊天似乎是不完整的。这里我们主要强调**动态表情（GIF/表情包）** 的作用。

动态表情已经发展到几乎可以不用文字就能进行交流的程度。它们本身自带强烈的情绪。

例如，有一类“贱贱的”表情。当你说了一句话，即使文字本身没有太多情绪，只要加上一个合适的贱贱表情，整句话的调性就会变得俏皮起来。

**案例演示：**
女：“我昨晚梦见自己被变态追着跑，好可怕。”
男：“让我来猜猜结局…是不是被自己的一个屁给崩醒了？” + [一个贱贱的坏笑表情]
如果没有后面的表情，这句话可能显得突兀甚至冒犯。但加上这个表情后，调侃、玩笑的情绪就准确传递了出去。

无论你想表达哪个方向的情绪，几乎都有对应的表情包与之呼应。例如，现在流行的“宋民国”等表情包风格多样，在前期互动中很好用。后期则可以适当使用更贱、甚至带点“污”的表情包，能快速拉近关系，聊一些彼此心照不宣的话题。
