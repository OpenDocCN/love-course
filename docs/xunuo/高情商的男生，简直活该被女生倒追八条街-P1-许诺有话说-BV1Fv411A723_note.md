## 什么是“给予价值”？🤔

具体什么是给予价值？举个最简单的例子。一个女生向你要钱，然后你也给了她钱。在这样一个互动中，你就是在给予价值。你的钱就是你的价值，而女生是索取价值的一方。

当然，价值有很多种形式，包括：
*   **知识**
*   **人脉**
*   **名气**
*   **地位**
*   **好的情绪**

任何让对方受益的东西都是价值。人会本能地被价值吸引，对索取价值者都会本能地排斥。

## 低情商 vs 高情商的对比 🔄

上一节我们介绍了“给予价值”的概念，本节中我们来看看它在实际场景中的应用。

以下是两个常见场景的对比：

**场景一：朋友买了新房**
假设一个朋友跟你分享，他刚花了150万买了套新房子。而你知道他买贵了，实际上那套房子没那么贵。
*   **低情商做法**：直接说“你买贵了，某某只花了100万就买到了。”
*   **高情商做法**：明知道对方买贵了，但是不会指出来。他会给予对方肯定和赞美，分享对方的喜悦。

有的人会说这是虚伪。但对方买了房是既定的事实。在无法改变既定事实的情况下，给予对方情绪上的肯定，分享对方的喜悦，是你能给予他最好的价值，而不是给他泼冷水。

**场景二：朋友让你猜新衣服价格**
如果你朋友身上穿了一件新买的衣服，然后让你猜价格。
*   **高情商的人**只会往高了猜，让对方觉得自己淘到宝了。
*   **低情商的人**则会往低了猜。

所谓高情商，就是能站在对方的角度思考，对方想要什么、需要什么，也就是共情的能力，然后**给予价值**。

## 聊天中的“给予价值”实践 💬

理解了核心概念后，我们来看看如何在日常聊天中避免成为“索取者”，转而成为“给予者”。

为什么女生讨厌男生聊天的时候“查户口”？因为查户口本身是一种**索取价值**的行为。你问了女生一连串问题，就算她回答了，结果就是你得到了信息，但她没有得到任何东西。这完全是一场单向的互动。

尤其一些男生会经常问“你在哪里”、“你在干嘛”等等。这些话有很强的索取意味。有情商的男生跟女生聊天，一定是**分享自我，多于提问对方**。

很多男生刚认识女生时，对方很热情，但聊着聊着就变冷淡了。大部分原因，就是你在聊天过程中无意识地**不断索取价值**。

## 核心聊天方法：从索取到给予 🗣️

很多人会说不知道要怎么跟女生聊天。那我分享一个最核心的聊天方法，同样也是**给予价值**的思路。

以下是具体的实践步骤：
1.  **平衡提问与分享**：如果你向女生提了问题，至少在她回答之后，对她的回复做出评价，表达你的观点。
2.  **分享相关故事**：分享自己与之相关的经历。因为你的观点、价值观和经历也是一种价值，能帮助她更了解你。
3.  **主动开启话题**：如果你不知道要聊什么，可以分享自己生活中好玩、好吃的东西。有时候一张照片胜过千言万语。

**核心代码逻辑可以概括为：**
```python
if 你提出了问题:
    在得到回答后，务必给予反馈或分享自我 # 这是给予价值
else:
    主动分享你的生活、观点或故事 # 这也是给予价值
```

如果你的生活很无聊，没有什么东西可以分享，这或许就是你做出一些改变的信号。
