# 许诺《情感宝典》：051.49、长期阶段-设定规则 📜

在本节课中，我们将要学习长期关系相处模型的第三点：设定规则。我们将探讨规则在情感关系中的重要性，并学习一个包含三个步骤的具体方法来有效地设定规则。

## 概述：什么是规则？🤔

上一节我们介绍了长期关系相处模型，本节中我们来看看“设定规则”。

设定规则是任何一段情感关系的开始。规则本质上是**你自己的框架、态度和标准**。很多人在感情中处于被动，正是因为缺乏明确的个人规则。

以下是规则的具体例子：
*   你不喜欢伴侣抽烟，这是一条标准。
*   你要求伴侣外出游玩时，不能超过晚上12点回家，这是一条标准。
*   你不希望伴侣去八卦和评价他人，这也是一条标准。

只有当你拥有足够多源自内心的标准时，这些规则才会让你们的感情向更好的方向发展。许多人不敢设定规则，误以为讨好对方就能获得爱。但事实上，**你越是设定并坚守规则，就越显得像一个高价值的个体；你越是设定规则，对方反而可能越尊重和喜欢你**。

## 如何设定规则？🔧

特别是在异地恋或长期关系维系中，如何设定规则是关键。我们将设定规则的过程分为三个步骤。

以下是设定规则的三个核心步骤：

1.  **制造意外**
    制造意外也可理解为“没事找事”，即需要有一件具体的事情来引发对规则的讨论。例如，你和伴侣吵架，往往是由一个“意外”事件引发的。

2.  **冷处理并升级意外**
    当意外发生后，绝不能轻易让事情过去。例如，对方多次不接你电话。如果不设定规则，常见的做法是选择算了，或者等对方道歉后就翻篇。但设定规则的做法是：必须对这件事进行“冷处理”，并升级这个意外。这意味着你需要让对方明确感受到你很重视这个问题，你非常在意、生气和不爽，从而意识到你的底线。

3.  **颠覆价值观**
    在对方意识到你的情绪和底线后，第三步至关重要：颠覆价值观。此时，你需要向对方解释你为什么如此在意这件事，给出一个合理且更高层面的价值观理由。例如，你可以说：“我生气不是因为你不接电话本身，而是我认为这反映了对感情的不尊重。我认为相互尊重是感情的前提，哪怕有一丝不尊重，这段感情就失去了意义。”

## 规则设定详解 🧩

### 第一步：制造意外

这个意外是设定规则的导火索。它必须是一个具体、可被双方感知的事件。例如：“你给伴侣打电话，对方没有接；你打第二个，依然没有接。” 这就是一个清晰的意外事件。

### 第二步：冷处理与升级意外

面对意外，不能妥协。妥协意味着你是一个没有规则和标准的人。你需要升级这个意外，让对方明确感知到你的情绪和底线。

以下是升级意外的具体方式：
*   **先感性**：不是完全不理对方，而是通过行为让对方感性上认识到你的不满。例如，消失几个小时或一整天；或者在简单指责几句后，不再回复消息。
*   **后理性**：在感性表达之后，重新开启对话，并理性地告诉对方：“你多次不接电话这种行为，我接受不了。” 注意，这是在冷处理之后、重新沟通时做的事情。

有些朋友可能会担心：“这样做，对方会不会生气去找别人？” 如果因此对方就选择离开，那或许说明你一开始就选错了对象。我们正是通过制造并升级意外，来一条条明确自己的规则和底线。

### 第三步：颠覆价值观与确立规则

当对方已经感受到你的不满并意识到你的底线时，就需要进行价值观层面的沟通。你必须解释行为背后的原因，并给出一个强大且正确的价值观。

**核心原则是：价值观只能被另一种更强大的价值观覆盖。只有你的价值观比对方的更强，你才能覆盖它，并获得对方的认可。**

以“不接电话”为例，你可以给出如下价值观陈述：
> “我之所以这么生气，并非因为你不接电话这件事本身，而是我认为这背后反映出的是你对感情的不尊重。我觉得如果双方对待感情有一丝一毫的不尊重，那么这段感情就没有继续下去的意义。我认为感情的第一位是相互尊重。虽然我们彼此喜欢和欣赏，但尊重是唯一的前提。”

这就是在输出你的价值观。接下来，便可以基于此设定规则：
> “如果以后还是发生类似情况，我觉得我们都需要重新考虑这段关系是否合适了。”

你一定要说出自己的底线，给予一定的压迫感。但关键在于，你提出的必须是一个**正确的、普世的价值观**（例如“相互尊重”）。没有人会认为相互尊重不重要，因此对方没有理由反驳，通常会认可。你将正确的价值观与具体事件（不接电话）联系在了一起，使得整个规则的设定变得合理。

相反，如果规则基于不正确的价值观（例如“你的微信好友里不能有任何异性”），则很难成功。

## 总结 📝

本节课中我们一起学习了长期关系中“设定规则”的方法。

1.  **设定规则是任何情感关系的开始**，它代表了个人的框架和标准。
2.  **合理的设定规则可以通过一个三步流程实现**：制造意外 -> 冷处理并升级意外 -> 颠覆价值观并明确规则。

掌握这个方法，有助于你在关系中建立健康的边界，赢得尊重，并引导关系向更积极的方向发展。
