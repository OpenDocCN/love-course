# 恋爱技巧课 P1：当女生说“别再来找我了”该怎么办？💔

在本节课中，我们将学习当女生说出“以后别找我聊天了”这类话时，应该如何正确应对。我们将通过一个真实案例，解析常见的错误做法，并学习如何运用平衡心态原则扭转局面，避免成为“舔狗”。

---

很多男生在追求女生时，一旦听到对方说“以后别找我聊天了好不好”，心里便凉了半截。剩下的半截不甘心，往往会跑去追问“为什么？我哪里做错了？我改”。这类男生追着追着，就容易变成人们常说的“舔狗”。而众所周知，女生大多数最后都不会喜欢上“舔狗”（当然，以索取钱财为目的的“捞女”除外）。

恋爱学中有一个核心原则：**平衡心态原则**。任何一方过度讨好（即“舔”），都会破坏这段关系的天平平衡。

那么问题来了：到底要怎么做，才能在追女生的时候，避免成为舔狗呢？

上一节我们介绍了问题的普遍性和核心原则，本节中我们来看看一个具体案例及其分析。

很早前有位兄弟找到我。他说自己以前创业不敢谈恋爱，现在事业小有成就，想找个女朋友。好不容易遇到一个喜欢的女生，聊了几天后，对方却说“不要找他聊天了”。他非常喜欢对方，不知该怎么办。

有的人可能会建议直接放弃，换一个目标。但既然这位兄弟找我帮忙，我决定帮他分析。我看了他发的聊天记录，发现他每天都进行各种嘘寒问暖：“吃了吗？没吃饱我给你点外卖”、“最近降温记得保暖”，以及日复一日的“早安”、“晚安”。而女生的回复都是“不用”、“好”、“谢谢”，直接终结了话题。

这位兄弟很不理解，他问我：“我看网上说追女生要多关心她，怎么到我这儿反而适得其反？” 很多兄弟都和他一样，陷入了不会聊天的困境。

以下是这位兄弟犯的一个非常致命的错误：

*   **他越级了**。他还不是女生的男朋友，却做了男朋友才能做的事情。越级的后果就是变成跪舔。

兄弟们一定要记住这个关键点：**不要做超出当前关系阶段该做的事情**。

看到这位兄弟如此执着，我决定帮他尝试挽回。女生说“以后别找我聊天了好不好”，简单思考一下，你会感觉怎么回复都可能把天聊死。

在了解情况后，我让他回复了一句：**“那意思是可以找你出来玩了？”**

女生回了一个省略号（……），表示对我们这句话很无语，可能觉得“哪里来的普信男”。她还说了句“感冒了不想说话”。这时，兄弟问我是否可以回“严重吗？我给你送药过来”。我立刻让他删了，这样回复不就又变回跪舔了吗？

我让他回复：**“你对着手机吹口气。”**

女生回了个“？”。这里，她对我们的下一步回答是好奇的。我们接着回：**“这样感冒就顺着手机传染给我了。”**

女生回了一个“无语”的表情包。兄弟们注意，“无语”的表情在这里需要解析：她虽然可能觉得你这个人不咋地，但对你刚才说的话还挺受用，有被暖到。“无语”在这里是一种傲娇的表现。

那么，这里我们可以接着回一句：**“这样你的感冒就好了。”** 这句话本身不重要，但我们可以用它来测试一下女生的反应。

女生回复：**“以前怎么没发现你这么会聊天？”**

兄弟们看到没？这就是聊天的魅力。短短两句话，女生就觉得我们是一个会聊天、有魅力的男生。也可以很明显地感觉到，女生的情绪已经从“不要找我聊天了好不好”，变成了“以前怎么没发现你这么会聊天”。这是一个窗口，意味着她现在愿意跟我们聊天了。

这时我们要做的就是让她保持高涨的情绪，所以我们回复：**“还有很多有趣的地方，你没发现。”**

女生果然好奇了：**“你说说看。”** 可以看到，女生的情绪已经完全处于愿意聊天的状态。

这个时候，我们要放出大招了。我让兄弟回复：**“有些东西网上可展示不出来，展示出来你也看不明白。”**

女生的胜负欲被激发了出来，她说：**“那有机会我倒是要看看。”**

这位兄弟问我，是否可以回“今天出来刚好我带你出去看看感冒”。兄弟们，给你们两秒考虑，这么回复合不合适？

**答案是：不合适。** 我们一定要记住：女生主动了，我们就要适当后退。这种时候千万不要心急。很多恋爱小白就是沉不住气，对方给点甜头，自己就翘尾巴了。这样会让女生觉得你这个男人特别容易得到，从而根本不会珍惜你的感情。

所以这里，我让这位兄弟回复：**“你好好养病，病好了有惊喜。”** 我们让女生保持对我们的好奇心，让她感觉和我们在一起的任何时刻都很有趣、很新奇，永远都看不透我们。慢慢地，这种好奇心就会变成好感，最后变成喜欢。

之后，这位兄弟在我的指导下，在第三天和女生进行了第一次约会。这其实是两年前的一个案例了，现在他们的朋友圈已经在晒宝宝的照片了。

---

本节课中我们一起学习了当女生发出“拒绝信号”时的应对策略。通过这个案例，我们想告诉兄弟们：在一段关系中，一定要摆正自己的位置。一味的死缠烂打只会适得其反。运用平衡心态，保持自身框架，通过巧妙的沟通调动对方情绪，才是建立吸引力的关键。
