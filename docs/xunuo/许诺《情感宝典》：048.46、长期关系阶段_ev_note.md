# 许诺《情感宝典》：048. 长期关系阶段

在本节课中，我们将要学习长期关系，特别是异地恋中的核心挑战与应对策略。我们将探讨为何会感到不安，并介绍一个维持健康长期关系的七步模型。

如果你已经掌握了前面的课程，找到心仪的女朋友应该已非难事。或许你已经找到了。那么从这节课开始，我将讲解长期关系的相处之道。接下来几节课程的内容，无论你是面对面恋爱还是异地恋，都完全适用。

首先谈谈异地恋。我本人并不支持异地恋。我认为感情需要双方在一起，通过争吵、摩擦和磨合才能日益深厚。而异地恋时，如果发生争吵，你只能在家干着急，或打个电话。一旦吵得激烈挂了电话，便无计可施。因此我不太支持异地恋。生命很精彩，应珍惜当下。虽然异地恋也能珍惜当下，但我认为终究缺少了一些东西。

然而，很多兄弟恰好遇到了一位深爱并决定共度一生的女生，又因种种原因，必须经历一到两年的异地恋，例如我和班嫂的经历。

在长期关系中，我们最担心的是因不善于处理关系、不了解女生的需求，导致两人厌倦、激情消退，最终分手。在这种情况下，如何让感情不减分反而升温？如何让你的女朋友能够抵挡住诱惑，不被身边其他男性，甚至一些优秀的男性趁虚而入？如何让你们持续保持激情？这才是最核心的关键。

许多处于异地恋的兄弟特别缺乏安全感，对女朋友不放心，这也是大多数人害怕异地恋的核心。换言之，这是对女朋友失去了掌控感。

异地恋因为距离拉开，会产生许多信息不对称。如果将恋爱比作放风筝，你会感觉像是在放一个断了线的风筝。你突然发现自己失去了对她的掌控感。

所以我们究竟在害怕什么？我们害怕的是失去这种掌控感。我们害怕风筝线断了，你无法掌控它。当你想再次收回时，却发现它已不在。无论是面对面恋爱还是异地恋，这都是你内心真正恐惧的地方。

因此，在长期恋爱关系中，如何掌控你的女朋友，如何重新获得这种掌控感，才是我们需要解决的核心中的核心。

那么，为什么我们会有这种失去掌控的感觉呢？主要是由以下几点造成。

以下是导致失控感的几个主要原因：
1.  你害怕她因为别的男人而移情别恋。
2.  如果是异地恋，你害怕她会因为距离而对你淡忘。
3.  你害怕她会因为各种原因给你减分。
4.  你不知道她能否抵挡住诱惑。
5.  你不知道她是否需要你。
6.  你不知道你们的感情基础是否足够牢固，能否突破空间带来的障碍。

也许听完这些你会感到有些痛苦，但这确实是每一个在长期关系中出现问题的人内心真实的想法。

针对如何掌握这种掌控感，我设计了一个长期关系的相处模型。只要按照以下七步去做，你的长期关系会相处得非常融洽。

以下是维持长期关系的七步模型：
1.  **保持精彩**
2.  **保持神秘感**
3.  **设定规则**
4.  **鼓励社交**
5.  **渗透生活**
6.  **引导投资**
7.  **共同成长**

可能你现在还不完全理解这些步骤的含义。没关系，在后续的课程中，我会对这七点进行深入讲解。

本节课中我们一起学习了长期关系，特别是异地恋中的核心挑战。我们了解到，问题的本质往往源于对关系失控的恐惧。为了解决这个问题，我们介绍了一个七步相处模型，为后续的详细讲解奠定了基础。

总结本节课程：
1.  若非迫不得已，尽量不要选择异地恋。
2.  长期关系出现问题后，你担忧的本质是害怕失去对另一半的掌控。
