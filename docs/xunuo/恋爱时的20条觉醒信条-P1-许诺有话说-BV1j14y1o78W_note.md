上一节概述了课程目标，本节我们将逐一解析这20条觉醒信条。以下是具体内容：

你死追的女生大概不会爱上你。如果你追一个女生花费了大量金钱和时间，她仍无动于衷，请趁早收手。她不是不想谈恋爱，只是不想跟你谈。

上一节我们介绍了前7条关于基础心态与行动的信条，本节中我们来看看关于具体互动技巧的原则。
