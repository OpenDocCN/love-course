# 许诺《情感宝典》：034.33、诱惑阶段-约会拥抱接吻方法_ev_note

在本节课中，我们将学习在约会阶段如何通过自然的拥抱和接吻来升级关系。这些方法旨在创造轻松有趣的互动体验，帮助你与女生建立更深的连接。

## 拥抱的方法 🤗

上一节我们介绍了关系升级的重要性，本节中我们来看看如何通过拥抱来自然地拉近距离。拥抱可以作为一个有趣的互动游戏，化解尴尬，并展示你的自信。

以下是两种发起拥抱的对话与行动示例：

1.  **利用体重话题顺势拥抱**
    *   在聊天中，如果聊到女生的体重，例如她说自己只有九十多斤。
    *   你可以回应：“测一下就知道。” 然后顺势给她一个公主抱，将她抱起来。
    *   抱着她走几步，然后说：“你果然没骗我。” 再将她放下。
    *   紧接着自然地说：“走，带你去吃点好吃的。” 并牵起她的手前往餐厅。

2.  **以“提高回头率”为名发起拥抱**
    *   你可以问女生：“你知道有什么办法能让你的回头率暴涨百分之几百吗？”
    *   她表示不知道后，你说：“我教你，能马上引起所有人的目光。”
    *   在说完这句话的同时，直接将她抱起来，并抱着走几步。
    *   放下后，她可能会对你“拳打脚踢”，此时你可以开玩笑说：“别打了，我没骗你吧？你看周围的人都看着我们，回头率是不是很高？”
    *   最后安抚道：“好了，别闹了，带你去那边吃好吃的。”

当你敢于自然地做出这些动作时，会给女生留下非常深刻的印象，让她感觉你与其他犹豫不决的男生不同。

## 接吻的方法 💏

在成功通过拥抱打破肢体接触的隔阂后，如果约会整体感觉良好，在后期可以尝试自然地升级到接吻。

以下是三种创造接吻机会的趣味方法：

1.  **“小魔术”方法**
    *   问女生：“你想不想看一个小魔术？”
    *   她同意后，盯着她的眼睛说：“好，那你认真看着我的眼睛，5秒钟，凑近一点。你不要动，我也不动。”
    *   两人靠近对视时，突然过去亲她的嘴唇一下。
    *   然后说：“魔术结束，这个魔术叫做‘把你的口红变到我的嘴上’。”

2.  **“心灵感应”方法**
    *   对女生说：“你相信我能感应出你心里在想什么吗？”
    *   她表示不相信后，让她闭上眼睛：“亲爱的，你闭上眼睛，心里默数5个数。数完时，我就能猜到你心里在想什么。”
    *   当她闭上眼睛数到“3”时，凑过去亲她一下，并马上撤回。
    *   她睁开眼后，你接着说：“别打了，跟你说认真的呢。我已经猜到你心里在想什么了——你肯定在想‘这个人怎么这么坏’，对不对？是不是被我猜中了？”

3.  **“对视游戏”方法**
    *   提议玩个游戏：“我们来玩个游戏吧，你坐过来一点。”
    *   让女生坐到你身边，注视着彼此的眼睛，规则是：“大家互相注视，都不能动。我输了，我请你吃饭；你输了，你什么都不要做。”
    *   对视几秒后，突然亲过去，然后说：“啊，我输了，我先动了，我请你吃饭吧。”

## 核心要点与心态调整 🧠

当一个女生愿意与你接吻，或在被你亲吻后没有真正生气或恼怒，这通常意味着她对你有好感。

成功应用这些方法的关键在于：

*   **自然与自信**：你的语言和肢体动作要流畅、笃定，不要让自己显得尴尬或犹豫。
*   **避免紧张**：不要害怕被拒绝，如果自己说话磕巴、行为畏缩，会给女生带来非常不好的体验。
*   **多加尝试**：掌握方法后，关键在于结合实际情况多多练习和尝试。

本节课中我们一起学习了在约会中通过创意和自信的方式进行拥抱和接吻，以自然升级关系。记住，真诚的互动和良好的氛围永远是基础。
