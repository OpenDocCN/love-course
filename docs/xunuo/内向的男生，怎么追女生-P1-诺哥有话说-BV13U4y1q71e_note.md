我告诉他，这与他小时候的家庭环境有很大关系。很多家长在小孩小时候，经常对孩子说：“我们家里穷，没钱，你要乖一点，别在外面给我惹事。” 这些孩子一般也很懂事，喜欢什么东西不会轻易说出来，因为他们知道父母赚钱不容易。所以，他们的很多想法和欲望是被压抑的。

长大后，生活改变了，但这种思想还一直停留着，很难突破。这导致了两个问题：第一，压制了自己的表达欲，不敢追求自己真正想要的东西；第二，与别人相处时不会说“不”，有不同的意见也不会说出来。别人让他说什么或做什么事情，尽管自己很不愿意，他还是会接受。我说完这些，他的眼眶有点湿润。

我说，你要突破自己的社交障碍，这对以后的工作和找对象都很重要。你现在不需要别人为你承担，更不需要父母照顾你。你不用有压力，大胆地去突破自己。你没有什么好失去的，不会因此掉一块肉。克服这个局面，就要大胆地去社交，大胆地去表达、锻炼自己。说白了，他就是锻炼得太少了。但这对他说来，仍然很难。在生活中迈出那一步是不容易的。

上一节我们分析了内向与社交障碍的成因，本节中我们来看看具体的改变方法。

对于这种情况的兄弟们，我们应该如何去改变？我给大家一点建议，这也是我昨天告诉他的。

以下是具体可行的行动步骤：

*   **寻找特定社交场景**：你可以找一下你们所在城市的桌游店，去玩“狼人杀”或“剧本杀”。因为这类游戏参与人数较多，经常需要拼桌。你可以打电话给老板询问，说自己是一个人，可不可以拼桌。
*   **利用环境优势**：在这个环境中都是陌生人，你不用担心自己做得不好会对你有什么影响。大不了下回不去这家店了。
*   **强制发言的练习**：这类游戏还有一个好处，每个人都必须发言，每个人都有表现的机会。你要做的就是表现自己，适应那种表达自己想法的、可能令你不自在的感觉。
*   **循序渐进**：一开始不适应时，你可以少说点，然后慢慢地一点一点锻炼。玩个七八次之后，你自然就好了。下次你跟女生见面时，自然也不会那么紧张了。

了解了如何通过游戏锻炼基础社交能力后，我们来看看如何将这种能力应用到实际约会中。

等你桌游玩熟了之后，你可以带女生去玩桌游。大家都在做游戏，自然没有必须不停聊天的压力了。不说话的时侯也不会无聊，该说的时候说两句就好。你们主要是在玩**游戏**，**聊天**是次要的。在游戏的过程中，你们也能够很好地了解彼此，同样能达到聊天的效果。

可以多参加一些社团活动，也有同样效果。这些社团也会组织各种各样的社交活动。
