## 心态是根源

上一节我们提到了课程的核心主题，本节中我们来看看一切行为差异的根源——心态。

人与人之间的沟通不仅仅是语言交流，还包括你传递给对方的感觉。这些感觉源自内心。自信与自卑的人在感情中的眼神都不同。

不同的心态下，说出来的话也是不同的。

## “撩”与“舔”的心态剖析

理解了心态的重要性后，我们来具体分析“撩”和“舔”分别对应着怎样的内心活动。

以下是“撩”的核心心态：

*   **欣赏而非必需**：心态是“我欣赏你，或许想得到你，但我并非非你不可”。
*   **目的在共享快乐**：撩你是为了让彼此都开心，享受互动的乐趣。
*   **提供情绪体验**：希望对方通过我的行为，感受到被喜欢的舒适感，体验我带来的积极情绪。

在这种心态下，你根本不计较得失，不在乎是否一定能得到对方。你只关注让两人互动的每一刻都开心。你会从女生的角度思考，不会要求对方必须接受你或秒回信息。女生没有压力，相处开心，自然愿意持续与你交往。当你比别的男生更会提供情绪价值时，女生就会觉得你与众不同。

以下是“舔”的核心心态：

*   **目的在换取回报**：心态是希望通过极端付出，让对方感恩，并以此得到这个女生。内心极其渴望回报。
*   **行为失去自我**：在这种心态下，会对女生言听计从，不顾自身感受去满足对方一切要求。
*   **引发对方低价值判断**：女生能感觉到这种急切和渴求，可能会认为“这个男生这么着急讨好我，是不是本身价值不高？”

很多采取“舔”模式的人意识不到或不愿承认自己本质是在索取。他们嘴上说着“我是真的想对她好”，但所有行为都透露出要求和索取。

## 核心区别总结

本节课中我们一起学习了“撩”与“舔”的本质区别。

简单来说：
*   **撩** (`撩 == 创造共享快乐`)：是给我们喜欢的人带来快乐，双方都享受这个过程——我享受撩的乐趣，对方享受被撩的快感。没有不甘心，不强求结果。
*   **舔** (`舔 == 付出以索取回报`)：是不断对对方好，并希望对方记住自己的好且给予回报。本质上是一种**索取行为**。

你可以审视自己：学习技巧的目的是为了“给女生提供情绪价值，让两个人都快乐”，还是“为了搞定某个女生”？后者不过是换了一种更隐蔽的“舔”的方式。

**内心如何，行为便如何。** 建立自信、平等、乐于分享的心态，才是正确“撩”的开始。
