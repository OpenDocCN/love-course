以下是常见的朋友建议，但这些建议可能导致你成为备胎。

*   朋友常说：“追女生得有点耐心。你看看张三，追求李四追了三年，最后不是在一起了吗？你喜欢她就得让她看到你的真心。”
*   如果你听信这些话，你很可能稳稳地成为备胎。

## 两个月反思期 ⏳

如果你喜欢一个女生，追求两个月仍未成功，就需要认真反思。

*   **公式：追求时长 ≥ 2个月 → 启动反思**
*   反思方向包括：方法是否正确，女生是否对你有意，是否应该考虑更换目标。
*   真正喜欢你的女生，不会让你等待太久。

## 单方面付出的陷阱 😟

很多男生会有这样的想法：“她不接受我，肯定是因为我没做好、不够努力。我要继续用真心打动她，她肯定是在考验我。”

*   持有这种想法，很容易成为备胎。
*   真正喜欢你的女生，不会让你长期单方面付出。
*   **核心原则：超过两个月仍未确定关系，必须认真反思这段感情，并学会放手。**

## 兴趣曲线与关系窗口 📉

你对一个人好奇心最强的时候，通常是刚认识的时候。那时新鲜感和好奇心都处于峰值。

*   随着彼此了解加深，如果关系未能推进，对方对你的兴趣通常会逐渐下降。
*   结果可能是关系趋于平淡，或者直接转变为“兄弟”关系。
*   两个月时间内，如果你在互动中已尽力做到最好，就不必再坚持。
