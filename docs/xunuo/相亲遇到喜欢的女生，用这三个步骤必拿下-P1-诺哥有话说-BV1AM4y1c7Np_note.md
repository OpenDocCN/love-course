## 第一步：选择相亲地点 🗺️

上一节我们介绍了课程概述，本节中我们来看看如何选择相亲地点。地点的选择至关重要，它影响着双方的安全感和初次印象。

## 第二步：开启聊天话题 💬

在确定了合适的地点后，下一步就是如何开启对话。相亲场合的聊天与其他时候有所区别，我们需要特别关注两个点：**共鸣**与**了解**。
