说通俗点，这就是“杠精”。如果还没明白，你只需记住一个聊天口诀：**一顺、二问、三夸、没立场**。

以下是这个口诀的具体解释：

*   **顺**：即顺着对方说，保留自己的意见，多说对方爱听的话。
*   **问**：即通过适当的提问，引导对方多说出自己的想法。
*   **夸**：即当对方说到自己引以为豪的事情时，你要及时给予夸赞。
*   **没立场**：意思就是你聊天不是为了来表达自己立场的。
