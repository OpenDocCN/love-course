# 许诺《情感宝典》：055.53、长期阶段-共同成长_ev_note

## 概述
在本节课中，我们将要学习长期关系相处模型中的一个核心概念：共同成长。我们将探讨共同成长的重要性、具体含义以及如何在实际关系中实践它。

## 什么是共同成长？
上一节我们介绍了长期关系的基本框架，本节中我们来看看“共同成长”这一具体要素。共同成长是指伴侣双方在关系存续期间，持续地进步、提升自我，并相互促进，最终达到“我中有你，你中有我”的融合状态。

如果一段关系持续了一年，对方却对你说：“我发现和你在一起的这一年，我一点变化都没有，反而越来越没有见识，越来越不自由。”这通常意味着关系即将结束。这是一件悲哀的事情。

因此，正确的做法是，在你和伴侣相处的时光里，你们双方都在不断进步。你的伴侣因你而变得更好，你也最终成为了更好的自己。这才是关系带来的最大价值。

**吸引力公式**可以概括为：
*   **喜欢** = 被对方无数的优点和魅力所吸引。
*   **爱** = 看到对方无数的缺点和遗憾，仍然愿意接受，并愿意**与对方一起成长**。

一段真正完美的长期关系，正是这样一个将“喜欢”升华为“爱”的过程。

## 如何引导共同成长？
理解了共同成长的意义后，我们来看看如何在实际关系中引导这一过程。作者经常对自己的伴侣说：“我每天都在学习，每天都在进步。每半年你都会发现我和之前有很大不同。如果你要跟上我的步伐，如果我们想相处融洽，我希望你也能提升自己。” 有时会以开玩笑的口吻补充：“万一哪天你配不上我了呢？”

这种方式能在对方心中建立一种积极的危机感。当她真正开始成长并取得进步时，她会深刻地感受到，和你在一起与和其他人在一起是截然不同的。

以下是引导伴侣成长的一些具体方式：

*   **替代低质量陪伴**：其他伴侣可能只会一起打游戏、看剧。而你可以选择一起旅行，探索世界。
*   **克服拖延，学习新技能**：鼓励并陪伴她学习那些一直想学却因拖延而未开始的技能，例如**烹饪**或**理财**。
*   **拓展优质社交圈**：带她结识你身边真正优秀、有才华、有能力的朋友。这能增长她的见识与阅历。

## 共同成长的三个步骤
明确了方向和方法后，我们可以将共同成长系统化为三个可操作的步骤。

以下是实现共同成长的三个具体步骤：

1.  **建立共同成长的目标**
    例如，你可以对伴侣说：“我们一起制定一个健身90天的计划，目标是你练出马甲线，我练出人鱼线。90天后，如果我们都完成了目标，就一起去国外旅行。” 这就建立了一个共同且富有吸引力的目标。

2.  **监督对方，成为更好的自己**
    双方需要互相监督。以上述健身计划为例，如果是异地恋，可以每天在健身房打卡，分享大汗淋漓的照片；如果在一起，则互相鼓励、督促。试想，当其他伴侣只是带着对方消磨时间时，你却在引领对方共同成长，帮助她成为更好的自己。这样的伴侣何其难得。

3.  **持续自我提升，保持吸引力**
    不要认为成长不重要。有些人觉得“成长”这个词有些虚幻。但实际上，每个人每天都需要进步。如果你想成为一个更优秀的男人，必须在多个维度上成长：**知识阅历**、**见识眼界**、**个人才能**、**表达能力**等。全面的自我提升公式是：
    `更强的自我 = 更丰富的知识 + 更广阔的见识 + 更出色的才能 + 更优秀的表达`
    只有这样，你才能持续吸引伴侣，并且有能力“战胜”那些围绕在她身边的其他潜在竞争者——无论是异地恋中的，还是现实生活中的。

## 总结
本节课中我们一起学习了长期关系中的“共同成长”。

首先，我们明确了共同成长的核心是达到“我中有你，你中有我”的深度联结，这是将“喜欢”转化为“爱”的关键。

其次，我们探讨了通过建立共同目标、互相监督鼓励以及持续进行多维度自我提升来实现共同成长的具体路径。

记住，共同成长是维系长期关系活力与深度的不二法门。
