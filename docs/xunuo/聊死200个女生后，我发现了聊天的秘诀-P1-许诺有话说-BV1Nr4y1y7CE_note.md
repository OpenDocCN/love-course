## 误区一：聊天频率过高 ⏰

上一节我们介绍了课程目标，本节中我们来看看第一个具体误区：聊天频率问题。

## 误区二：过度追求“神奇”话题 💬

聊完了频率问题，接下来我们探讨第二个关键误区：对聊天话题的误解。

许多人总是在寻找一种具有非凡吸引力的话题，奢望一句话就能让女生喜欢上自己。但根本不存在这样的话题。实际上，**聊什么话题并不重要，重要的是你能否将一个话题延续下去**。

在延续话题的过程中，关键在于能否借题发挥，对女生的话进行**曲解**，从而**暧昧升高**关系。即使是一个绝佳的话题，如果你的思维模式不变，依然可能一句话就让聊天陷入僵局。

此外，必须记住：线上聊天的内容应为线下邀约做铺垫，并在对话中适时植入一些**小暧昧**。

例如，你可以这样说：
```
见面时不要对我动手动脚哦。
```
或者
```
我第一次约会会比较紧张，一紧张就喜欢抓着对方的手不放。
```

聊这些内容非常重要。如果你只是漫无目的地闲聊，到最后连自己都觉得无聊时，女生又怎么会感到有趣？而且，如果你不进行此类铺垫，即使女生答应与你约会，你们的关系也可能停滞不前，除了为约会买单，对关系的实质进展毫无帮助。
