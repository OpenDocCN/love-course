# 许诺《情感宝典》：017.16、追求阶段的三个要素 📚

在本节课中，我们将要学习情感关系进入“追求阶段”后的核心任务与应对策略。这个阶段是关系升级的关键时期，也是女生进行“废物测试”最频繁的阶段。我们将重点解析如何通过三个核心要素，向女生证明你的“靠谱”，从而顺利推进关系。

上一节我们介绍了吸引阶段，当吸引建立后，关系便进入了追求阶段。这个阶段女生会通过一系列问题来测试你的诚意，这些问题看似刁难，实则都指向一个终极疑问：**你是不是真的喜欢我？**

这些问题，例如“我和你妈同时掉水里救谁”或“你为什么和前任分手”，其本质是女生在考察你的责任感和忠诚度，即她的第二个底层需求：**靠谱**。因此，我们在追求阶段的核心目标，就是通过各种行为证明自己是一个靠谱的、真心喜欢她的人。

为了让女生感受到你的真心，你需要把握以下三个要素。以下是具体阐述：

**第一点是筛选。** 筛选的目的是让女生感觉到你也在认真考察她。这传递出一个信号：你不是在随意玩闹，而是以认真恋爱为前提在接触。就像购物时精挑细选代表真想买，而一句“再看看”则透露出犹豫。具体做法是，在互动中自然流露出你对伴侣某些特质的在意，但不过分批判。例如，你可以问：“你谈过几次恋爱？” 她回答后，你可以说：“比我多一次呢（加一个委屈表情）。那你有过同居经历吗？” 无论她回答有或没有，你都可以根据情况表达“我有一点介意”或“也还好”。关键在于，你提出了问题（表明你在意），但没有因此否定她（表明你能接受）。这种“在意但接纳”的态度，会让女生觉得你是经过思考后依然选择她，这是真心的表现。

**第二点是暖男。** 暖男行为是指在追求阶段，通过对女生的好、关心和细节关注来表达喜欢。这与初识阶段的“跪舔”有本质区别，此时的关心是建立在已有吸引和好感之上的关系升温剂。关键在于捕捉并回应她的独特细节。例如，如果你注意到她吃饭时不挑葱花，下次单独约会点菜时，可以特意对服务员说：“所有菜都不要放葱花。” 当她惊讶地问起，你再告诉她你是如何观察到的。这种对她个人习惯的细致关怀，远比泛泛的“多喝热水”更能打动她，让她确信你的喜欢是专注且用心的。

**第三点是走心。** 走心关乎关系推进的节奏，特别是应对女生的主动。男生的角色是主动靠近，而女生的角色则是在男生靠近时给予配合。当女生偶尔主动挑逗或试探时，切记不要立刻顺杆爬、急于升高关系。正确的做法是**适当后退**。例如，女生开玩笑说“你是不是想追我呀？”，你若立刻承认或过于热情，会让她觉得你轻浮或自己的主动不妥。此时后退一步，幽默地回应“我还在考察期呢”或“你猜？”，反而能增加挑战性和趣味性。这种“推拉”会让女生觉得你更有框架，之后你再主动推进关系时，她会更愿意配合，关系基础也更稳固。

本节课中我们一起学习了追求阶段的三个核心要素。首先，我们明确了此阶段的目标是证明“靠谱”。其次，我们详细拆解了达成此目标的三个方法：**筛选**（表达在意并认真考察）、**暖男**（在正确时机给予细节关怀）和**走心**（掌握主动与后退的节奏，特别是在女生主动时）。记住，对女生好本身没有错，关键在于时机和方式。真诚与策略相结合，才能有效通过女生的测试，让关系健康地向前发展。
