## 错误一：在感情中自满

我在往期视频中讲过，男女的吸引力机制不同。一个男生对女生的吸引力主要取决于他行为上展现出来的社交支配性。

很多男生在追女生的时候，表现得上进、风趣、自信又有野心，很快就成功地吸引到了女生。跟对方谈了一段时间以后就自满了，开始把你们的关系当成理所当然。觉得女生已经是女朋友了，就没必要那么累地去维持关系了。

**核心概念**：吸引力源于持续的行为表现，而非固定的身份。公式可以理解为：
**初始吸引力 = 行为魅力（上进、自信等）**
**关系中的吸引力 = 持续的行为魅力**

这种行为上的转变会很快杀死你对女生的吸引。因为她当初是被你行为上的魅力所吸引才跟你谈恋爱的，而你在中途改变了你的行为，就相当于打破了对她的社交承诺。这就像你娶了个魔鬼身材、天使脸蛋的老婆，结婚后不久她就变成了200斤的胖子，是同样的道理。

## 错误二：舍不得放弃关系

很多男生因为自身（情感或选择上的）匮乏，即使自己所处的关系不是那么理想，也不愿意或不敢放弃这段关系。

在一段关系中，越不需要对方的人拥有越多的权利。相对地，另一方就得妥协更多来维持这种关系。

以下是女生在关系中可能进行的测试行为：
*   翻看你的手机。
*   未经允许动你的东西。

一旦她发现你为了维持关系会妥协你的边界和底线，不敢对她的行为做出（合理的）回应或“惩罚”，她就会得寸进尺，一次次做出更过分的行为。在你一次次妥协的过程中，你会失去你的尊严，她会失去对你的尊重和吸引力，结果可想而知。

**核心概念**：健康的长期关系需要明确的个人边界。代码逻辑可以描述为：
```python
if 对方行为 越过 我的底线:
    清楚表明态度，要求改变
    if 对方拒绝改变:
        舍得离开  # 这是维持自尊和吸引力的关键
    else:
        继续观察
else:
    关系正常进行
```

在一段长期关系中，你必须要有离开的意愿。如果她越过了你的边界，做了你认为过分的事情，你要清楚地表明态度。如果她拒绝做出改变，你要舍得离开她，结束这段关系。
