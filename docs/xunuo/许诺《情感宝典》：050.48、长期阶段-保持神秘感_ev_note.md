在本节课中，我们将要学习如何在长期恋爱关系中保持神秘感，以维持持续的吸引力。我们将探讨神秘感的重要性，并学习一些具体、实用的方法来营造和维系它。

黄淑惠有一首歌的歌词是这样的：我们要天天相恋，但不要天天相见，只要悱恻缠绵，绝不要柴米油盐。有共同的生活经验，但不要共同的房间。这首歌恰恰描述了当下都市男女的一种生活状态。

虽然不完全倡导以这种方式去恋爱，但爱情确实需要保持一份神秘感。人都有好奇心，对自己不了解的东西都怀有一种非得把它搞清楚不可的冲动。例如，你前面走着一个身材苗条、穿着得体、优雅大方的女人，你一定特别想知道她到底长什么样，于是马上就会产生一种走到她前面看个究竟的冲动。这就是神秘感带来的威力。

但在长期关系的相处中，使双方彼此失去吸引力的原因有很多，其中一点就是神秘感的缺失。恋爱是一个相互了解的过程，能彼此认识、了解是值得庆幸的。但是，如果了解得过于透彻，甚至一些不需要了解的也知道了，那么彼此之间的神秘感也就消失了。这对一段长期关系是不利的。

最常见的现象是，很多男生将自己的微信、QQ密码全部告诉伴侣。你身边有几个朋友、有几个女性朋友、你每天都会干什么，甚至你每天穿什么颜色的内裤，你的女朋友也都知道。最后就导致了女生感觉你的什么事情她都知道了，她一点想要继续探索你的欲望都没有了，这就会导致吸引力的减弱。

其实，每一对情侣都应该有自己一方神秘的、不被任何人所知的天地。所以，在长期关系相处中，你必须保持一份神秘感。而这种神秘感不是固定不变的，一方面，你要让你的女朋友一边不断地去探究，一边又不断地被新的内容所充实和替换。

因此，在恋爱过程中，除了加强自身的各方面修养外，还要注意不要过快、过于充分地将自己全部暴露，包括你的才能、收入、特长、经历以及肉体等等。要学会细水长流。

上一节我们介绍了神秘感在长期关系中的重要性，本节中我们来看看具体如何营造这种神秘感。以下是增强神秘感、保持你在长期关系中持续吸引力的六个方法：

**第一，创造生活情趣。**
改变单一的、日复一日的、没有变化的生活。例如，突然给对方来一个惊喜，突然去参加一个盛大的派对，或者改变自己的穿衣风格、变换发型、改变自己房间的布置等等，都会使你的女朋友感到新鲜和愉快。

**第二，保持礼貌，尊重彼此。**
尽管两个人经过热恋，彼此不分你我，但仍然要像初恋时那样保持礼节，不要失去原先的温柔和体贴。因为任何不尊重对方的言行，都会降低自己的吸引力。

**第三，学会偶尔做短暂的分离。**
俗话说，小别胜新婚。特别是在闹了一些矛盾之后，短暂的分离不但使双方都有时间去冷静地思考、反省，而且分离后相见时的神秘感也会成倍增加。恋爱中的约会，不要每次都有求必应，可以找一些合适的理由学会适当地拒绝。

**第四，将自己的长处、特点、优势做一些适当的保留。**
不要一上来就让你的女朋友知道你所有的优点。例如，这个星期她知道你会做饭；过了几个星期，她知道你还会弹钢琴；又过了一段时间，她发现原来你的收入不止她想象的那么多；进一步接触后发现，原来你的家庭条件还很优越，你不但有房有车，还投资了好几个门面。

**第五，避免频繁的、一成不变的亲密关系。**
恋爱中发生亲密关系是无可厚非的，也会美化和促进你们的爱情。但是，有些男生整天就想着和女朋友发生关系，一言不合就要进行，特别是每次就是一个姿势，每次几分钟就完事，只管自己，不管对方。这不但不会让女生感受到爱情的甜蜜，反而会让她感到恐惧，甚至以后惧怕亲密接触，这也是很多女生最后变成性冷淡的原因。

**第六，无论男女，多和不同的人去社交。**
让你的女朋友不知道你身边到底有多少厉害的人，到底又认识多少美女。这能营造一种你拥有丰富社交圈和未知潜力的印象。
