## 何时应该放弃追求？ 🚩

上一节我们介绍了追求期不宜过长的总体原则，本节中我们来看看几种具体的、应该直接放弃追求的情况。以下是几种关键的情况判断：

**一、不回复或回复敷衍**
很多男生追女生时会持续不断地给女生发消息。即便女生不回复，他也一直在发。或者是女生回复得很慢，基本上是几个小时或者第二天才回复你。即便回复也很敷衍，用“嗯”、“哦”或者表情之类的。这样的聊天根本进行不下去。又或者是出去玩、面对面聊天时，对方也是无视你的态度。你说你的话，他玩他的手机，偶尔抬起头来应付你几句。根本原因就是对方对你不感兴趣。碰到这种情况，还是直接放弃吧。

**二、不会为你付出**
如果你们认识了一段时间，也约会过、出去过几次，但每次吃饭、看电影、买零食等等都是你在花钱，对方则在旁边看着你付钱。我建议也不要追了。因为这样的关系从一开始就是不平衡的。即便你能追到手，后期关系也很难维持。谁都不想当“舔狗”。出现这种情况，往往是因为你不会聊天导致的。你可以看看我免费送大家的30套聊天案例解析，都是从陌生到聊成女朋友的完整聊天记录。很多看过的兄弟照着聊都顺利脱单了。想看的兄弟可以扣6，我会立马安排。什么都不需要，只要你的点赞、关注三连来换。

**三、与多名男生保持暧昧关系**
你在追求她的时候，她还在和其他男生保持很亲密的关系。但同时，她也不拒绝你。这样就趁早放弃吧。她可能只是在“养备胎”。

**四、公主病**
比如，你必须开车去她家楼下接她，否则她就不出门。这样的女生，不是说一定有公主病，也能够说明她故意给你刁难。
