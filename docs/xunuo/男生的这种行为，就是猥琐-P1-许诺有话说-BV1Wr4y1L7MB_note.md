# 恋爱避坑指南 01：识别并避免“猥琐”行为 🚫

在本节课中，我们将通过一个具体案例，分析在追求异性过程中哪些行为会被认为是“猥琐”的。我们将拆解这些行为背后的心态，并学习如何建立健康、舒适的互动关系。

---

今天，一位粉丝私信告诉我，他的朋友在与女生约会后，被对方认定为“猥琐男”，并且此事在学校传开。这位粉丝认为他的朋友为人不错，不理解为何会被如此评价。因此，我们今天就来探讨一下，什么样的行为会构成“猥琐”。

## 一、 得失心过重与企图心过强 💢

在追求他人的过程中，怀有得失心是正常的，因为每个人的时间、精力和金钱都有限。大家都希望付出能有回报，若不成，也希望能得到明确答复，避免浪费更多时间。

然而，我们必须承认一件事：**恋爱不是购物**，无法在短时间内迅速判断喜欢与否。部分男生看到女生外貌出众、身材性感，便不再考虑其他，一心只想确立关系。

因此，有人会提出“我们先在一起试试，不合适的地方以后再改”。但女生通常不这样想，她们需要判断你是真心喜欢她，还是另有所图。所以，前期以朋友身份相处是必要的。这个阶段你可能感觉良好，但这未必是男女之间的好感。需要度过这段了解期，再考虑后续发展。

有的男生则过于急躁，等不了这个过程。他们会质问：“你到底喜不喜欢我？你每次都回我消息，每晚都说晚安，你就是喜欢我，承认吧。”甚至有的男生会动手动脚，若遭拒绝便不开心，试图用强，并说出“装什么装”、“别扫兴”之类的话，错误地认为女生“嘴上说不要，身体很诚实”。

我建议有这些想法的朋友，可以看看我整理的**30套与女生的聊天案例解析**，了解女生哪些表现才代表真正的喜欢。许多朋友看完后照着聊，都成功脱单了。如果你还没看过，可以私信或评论“1”获取，只需用你的点赞、关注和三连来交换即可。

我们男生喜欢一个女生，想和她有亲密接触，这并不丢人。但有的人态度是“爱约约，不约滚”，或认为“你都答应出来了，就等于默认要发生点什么”。这种理所当然的想法非常令人讨厌。

他们的得失心特别强。一般人追求失败，最多沮丧一阵，不会记恨对方。但他们无法接受，会气急败坏，甚至恶语相向、威胁女生。更有甚者，会警告女生身边的所有异性，宣告“主权”，摆出一副“这个妹子我吃定了”的姿态。

人家父母将女儿养育成人，怎么就成了你的所有物？他们内心甚至理所当然地认为：“如果你拒绝我，之前为什么要和我聊天、接受我的邀请？你就是绿茶、拜金女，只想占我便宜。”这种不顾对方感受、强行推进关系，得不到就气急败坏的人，就是“猥琐”。

**一个男人首先要让女人和你相处感到舒服。** 即便是调情，也要让对方感到舒适。看过我聊天记录讲解的朋友，应该深刻体会过什么叫“调戏得让她舒服”。

## 二、 “嫖客”心态 💸

“嫖客”心态，顾名思义，是将所有事情都与金钱划等号，认为包括感情在内的一切都能用钱摆平。不要小看这种观念，你肯定不止一次见过有男人在喜欢的女生面前自吹自擂，生怕别人不理解他有多“厉害”。

“嫖客”心态主要有两大表现形式：

以下是其典型表现：
1.  **油腻地炫耀和吹嘘自己**：认为只要自己足够有钱有势，就能征服对方。
2.  **不顾对方意愿，单方面强行付出**：如果对方不接受，他们反而会觉得对方莫名其妙、不识抬举、有眼无珠。

这种人很多。就像我很多视频的评论区里，总有人说：“讲这么多废话干嘛，不就是钱吗？”或者“我单手开法拉利，会没有女人喜欢吗？只要有钱她就爱我了。”

对此我不反驳，但首先要搞清楚：**她爱的是你的钱，还是你的人？你们是在做交易，还是在谈感情？** 如果你们的关系是包养，你有钱包养一百个，那都是正常现象。但女生因为有钱和你在一起，她爱的究竟是钱，还是你这个人？她嫁给的是物质，还是爱情？

实际上，即使你真的很有钱，依然可能不被女生喜欢。找我咨询的粉丝里，有钱的也不少。有个人让我印象深刻，他说：“我给你八万，帮我搞定这个女生。”我了解情况后，表示搞不定。情况是：男生家里开公司，在上海有三套房和一个商铺；而女生的父母是上市公司股东，她出门逛街都开法拉利488。

可能有人会抬杠：搞不定还是因为钱不够多。但试想，一个小姑娘都开488了，更多的钱对她还有多大意义？她家里的钱已足够她花，何必为了钱去委屈自己，和一个不喜欢的男人在一起？

还有男生认为：“我都为她花这么多钱了，她还不应该感动吗？”如果遇到上述情况，她压根不觉得钱是多宝贵的东西，你花再多钱，可能还不如讲个笑话逗她开心来得实在。

有人又会说：“有钱的不好搞，爱钱不爱人是吧？我不在乎，我就找个好看但没钱的。我不要她的心，只要她的人不行吗？”结果接触后，如果小姑娘洁身自好，不吃这一套，他们又会说：“这小姑娘真傻，我有车有房有好工作她不选，难道去找个穷小子吃咸菜？活该穷一辈子，这女的眼睛瞎了。”完全是一副吃不到葡萄说葡萄酸的嘴脸。

最有节目效果的是，当小姑娘没有拒绝，真的考虑和他们在一起时，他们反而换了一副嘴脸：“这个女人是不是看上了我的房子？”“她是不是因为我的本地户口才和我在一起？”“她学历、工资都不如我，是不是想占我便宜？”没人喜欢时，就骂别人不识抬举；有人喜欢时，又嘀咕对方是来占便宜或套路自己的。

说他们是“嫖客”一点不委屈。因为**你用嫖客的观念和行为去追求，得到的是什么，自己心里应该清楚**。即便真有好的姑娘一时糊涂跟了你，也会被你各种猜疑和控制欲折磨到离开。

## 总结与过渡 🎯

说到“猥琐”这个词，大家脑海中浮现的，通常是那种鬼鬼祟祟、行为油腻、内心阴暗的男人形象。我认为，上述两种男生的行为，正是这种形象的真实写照。

无论是得失心和占有欲极强、恨不得“霸王硬上弓”的猴急，还是认为感情无非是生意、可以花钱购买的庸俗，**其本质都脱离不了油腻、不坦荡、不够阳光**。

他们通常是极度缺乏认可的人群，甚至自己都不喜欢和同类做朋友，看不起自己的所作所为。他们主观地认为所有人（尤其是女性）对自己充满恶意。由于他们本身就不真诚、不直率阳光，所以在他们眼里，谁都打算算计他们、玩弄他们的感情。任何拒绝他追求的女生，他都会试图在心理或行为上加以诋毁。

由于自身条件又未达到让女生倒追的地步，他们还必须主动追求。于是他们心浮气躁，会做出一些常人无法理解的极端行为，例如：看了一眼某女生就喜欢尾随，被发现就表白，对方不答应就试图强迫，对方躲避就继续尾随纠缠，动不动就暴躁焦虑、脾气坏没耐心。最终，成了一个从思维方式到言谈举止都十分“猥琐”的怪人。

本节课中，我们一起学习了两种典型的、容易被认定为“猥琐”的心态和行为模式：**得失心过重/企图心过强**与**“嫖客”心态**。关键在于认识到，健康的互动应建立在尊重、真诚和让对方感到舒适的基础上，而非强求、物化或交易。理解这些，是避免令人反感的行为、建立良好关系的第一步。
