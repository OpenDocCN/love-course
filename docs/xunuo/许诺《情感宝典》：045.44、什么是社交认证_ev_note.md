# 许诺《情感宝典》：045.44、什么是社交认证_ev_note

在本节课中，我们将要学习“社交认证”这一核心概念。社交认证是吸引范畴内的一种强大工具，它能间接地向他人展示你的高价值，从而建立吸引力。

## 什么是社交认证？

社交认证是吸引范畴内的一个概念。吸引所要做的核心是让女生知道我们是一个“猎人”。

我们通常通过**高价值展示**和营造**两种感觉**来实现吸引。社交认证与这两者不同，但它特别有效。

## 社交认证为何有效？

社交认证能让你拥有强大的吸引力。举个例子，当你带着一个女生去聚会时，如果聚会上每个人都尊重你、对你很客气，这就在间接地向这个女生表明你是一个受认可的“猎人”。

## 社交认证的主要方式

以下是社交认证的几种常见方式：

**第一点，身边朋友对你的认可。**  
与朋友一起出去玩时，要注意互相捧场。如果遇到总是贬低你的“损友”，则需要留心，因为这可能导致你的社交认证为**负值**。

**第二点，你交往的朋友都很优秀。**  
如果你认识交往的朋友都很优秀，那么侧面也可以证明你本身也很优秀。

**第三点，你过往或身边的女性伴侣很优秀。**  
这被称为“预选”。当其他女生已经对你做过正面判断时，新的女生往往会忽略自己的判断，更容易被你吸引。例如，如果你在路上看到一个其貌不扬的胖子身边有位美女相伴，你的感觉会完全不同，会觉得那个胖子“不简单”。这就是社交认证的力量。

## 社交认证的延伸应用

我们可以将社交认证的概念拓宽一些。当你与女生见面时，你的**穿着**、你出入的**场所**、你的**座驾**，这些也都是你的社交认证。

大家可能听说过这样一个比喻：一个男生花2000元给女生买手机，另一个男生花1500元租豪车带女生吃麻辣烫，结果后者搞定了女生。这虽然是个笑话，但说明钱要花在能产生**社交认证**的地方。同样是100元的预算，与女生在环境较差的餐厅吃饭，和与她喝一次高端的下午茶，后者的效果通常好很多。

## 一个现实案例

有一次我们带学员进行线下活动。在酒吧里，9个男生（6学员+3导师）拿了两张卡座。女生不是同时到达的。我们观察到一个特别的现象：最早到的是两位形象气质一般的女生，她们起初不太说话，只顾玩手机。学员打招呼时，她们也爱搭不理。

但是，当后面陆陆续续来了十几个女生，其中一些比她们更优秀时，这两位女生的“社交窗口”突然打开了。她们不再玩手机，变得愿意互动，积极聊天、喝酒、玩游戏。

从这里可以看出什么？最初，两个女生面对九个男生，场景是“男多女少”，她们可能觉得这些男生缺乏吸引力。当更多女生（尤其是更优秀的女生）加入，并且现场氛围友好、大家玩得开心时，她们自然就不再端着架子了。女生前后不同的反馈都源于潜意识，她们自己可能都未察觉这种变化。这就是**社交认证**在环境中的动态体现。

我们应当学会运用好这种“神秘武器”。

## 课程总结

本节课我们一起学习了社交认证的概念与应用。我们来总结一下重点：

**第一点，朋友对你的认可很重要。**

**第二点，身边的朋友价值很高能侧面证明你的价值。**

**第三点，过往交往的女生或身边的女性朋友很优秀，能形成强大的“预选”效应。**

掌握并善用社交认证，能有效提升你在社交互动中的吸引力。
