# 课程：如何成为一名合格的舔狗 - P1 🐕

在本节课中，我们将要学习成为一名“合格舔狗”的四个核心行为准则。课程内容基于网络流行概念，旨在以结构化方式解析其行为模式，仅供理解与讨论。

## 概述 📋

“舔狗”一词常用来形容在感情中单方面、过度付出而不求回报的一方。本节课程将分步拆解其典型行为模式，从基础习惯到进阶策略，提供一个完整的框架分析。

## 第一点：养成特殊习惯 💬

上一节我们介绍了课程概述，本节中我们来看看成为“舔狗”的第一步——建立特殊的互动习惯。关键在于通过细节营造独特性与优先感。

以下是具体操作指南：

*   **设置独特备注**：在社交软件上，为对方设置一个独一无二的昵称，例如“我的小女神”、“女神下凡”。
*   **聊天置顶**：将与该女神的对话窗口始终置于聊天列表的最顶端。

这样，每次打开软件时，你都能第一时间看到她，从而本能地产生联系的冲动。

## 第二点：保持绝对主动 📱

仅仅建立习惯还不够，主动是维持存在感的关键。如果你不主动，对方很可能与其他人建立联系。

因此，你需要坚持不间断地发送消息，无论时间与场合。

以下是消息发送的示范模板：

*   **早晨**：“早上好，吃早餐了吗？一定要吃，对身体好。没吃的话，我可以给你带。”
*   **中午**：“吃过午饭了吗？多吃点。中午记得午睡，下午精神更好。”
*   **晚上**：“晚上好，睡了吗？不用回我，看到消息就行。我会一直守护你，包括在梦里。”

## 第三点：提供无微不至的关怀 ☔️

在主动的基础上，需要通过持续的关怀来感动对方。要牢记一个核心观念：**女神被假定为缺乏自理能力**。你的职责是预判并提醒所有潜在风险。

养成查看天气预报的习惯，并在天气变化时第一时间发出提醒。

以下是关怀话术示范：

*   “明天可能要下雨，记得带伞，别感冒了。如果忘了，告诉我，我给你送过去。”
*   “过两天是你的生理期，这几天别乱吃东西，忌辛辣，饮食清淡些。我给你寄的红糖水应该明天下午到。**红糖水加姜片效果更佳**。”
*   “我刚路过你公司，发现楼下有积水，有人摔跤了，你出来时小心点。”

## 第四点：持续而真诚的表白 💌

掌握了前三点的你，可能认为自己已经入门。但这仅仅是初级水平。要成为一名“合格的舔狗”，必须在以上基础上，贯彻第四点：**持续不断地、真诚地表白**。

表白需真情实感，篇幅要长（建议三千字起），且在被拒绝后仅短暂休整（例如三分钟），便应继续努力。

以下是可参考的表白句式：

*   我知道你一定会拒绝我，但是……
*   我知道你不会接受我，但请允许我继续喜欢你。
*   即使没有结果，我也会坚持。如果你讨厌我，请不要拉黑，我会默默喜欢。
