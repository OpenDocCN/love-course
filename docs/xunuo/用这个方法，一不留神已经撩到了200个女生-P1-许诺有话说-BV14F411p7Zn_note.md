**聊天内容 = 状态描述 + 感受表达**

## 方法二：学会延续话题的技巧 🔄

仅仅开启话题是不够的，让对话持续下去同样重要。延续话题的核心在于抓住对方语句中的关键点进行回应和延伸。

以下是延续话题的一个实例步骤：

2.  对方回应：“这么惨的吗？哎，我正在吃小龙虾。”
3.  你延续话题：“我觉得你在搞事情。”（对对方的行为进行情绪化调侃）
