在女生看来，强者不会主动示好。所以即便女生对男人有好感，如果这个男人直接对女生说“喜欢她”，有些女生不会觉得开心，反而会觉得没意思。

*   **“我喜欢你”是示好**：因为我说了“我喜欢你”之后，我就得看对方的脸色。
