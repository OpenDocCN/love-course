# 许诺《情感宝典》：054.52、长期阶段-投资 💰

在本节课中，我们将要学习长期关系中的一个核心概念：**投资**。我们将探讨投资在感情中的具体形式，以及如何建立一种平衡、健康的双向投资关系。

上一节我们介绍了长期关系的基础，本节中我们来看看如何通过“投资”来维系关系的平衡与健康。

我们会发现一个普遍现象：往往在一段感情中付出最多的那一方，在分手后是最痛苦的。而这些付出就是我们今天要说的**投资**。这里的投资特指在一段感情中的投入。

## 什么是感情中的投资？ 📊

感情中的投资主要分为几个方面：**金钱投资**、**时间投资**、**情绪投资**和**肉体投资**。

在一段长期关系中，我们不仅要为伴侣付出这些投资，也需要引导伴侣为我们付出对等的投资。否则，你们的关系就会变成一边倒的状态。这种状态不正确，也不健康。

## 失衡投资的后果 ⚠️

很多男性在关系中会陷入一个误区：付出越来越多。他们一点麻烦都不想让她受，一点气都不敢让她生，一点钱都不想让她花。他们认为伴侣是应该捧在手心里的掌上明珠，一点委屈都不能受。这会导致对方觉得无需付出任何努力就能坐享其成。

最后的结果就是你越来越离不开她，可是一旦你们分手，你的痛苦一定会大于她。

## 如何引导双向投资？ 🤝

为了避免关系失衡，我们需要学会引导伴侣进行对等的投资。以下是具体的方法：

上一节我们了解了投资的类型，接下来我们看看如何在实际中应用。

**引导时间投资：**
以下是你可以尝试引导对方进行时间投资的方式：
*   要求她每天回家后，和你报个平安。
*   让她花时间为你做饭，或为你学习做饭。
*   请她为你织一件毛衣。
*   计划并邀请她陪你一起去旅行。

**引导情绪投资：**
不要让她事事都依着你。健康的情绪投资意味着共享所有情绪。以下是引导情绪投资的关键：
*   允许她因为你而生气或难过。
*   偶尔制造一些惊喜，让她因为你而感到开心。

**引导金钱投资：**
不要再拒绝她的一切花费。适当地接受投资，有助于建立平衡。以下是具体做法：
*   适当地收下她送给你的礼物。
*   出去吃饭时，也可以让她请你吃饭。
*   慢慢形成一种“你们俩出门，谁花钱都一样”的状态。

**引导肉体投资：**
性也是亲密关系中的重要投资。双方全身心投入，才能达到最完美的交流。以下是引导方法：
*   鼓励她为你学习一些性技巧。
*   引导她尝试一些之前没有尝试过的两性知识。
*   不要害羞，也不要担心她害羞或说你不正经。要引导她明白，和最亲密的人探索这些是很正常的事。

## 投资的“赌徒心理” 🎲

我今天说的这个投资，实际上就像赌徒去赌博一样。我们可以用一个简单的模型来理解：

**`已投入成本` 会驱动 `继续投入` 的行为。**

假如你有5万块钱，你去赌钱。前半夜你输了2万，你一定不会就此收手，因为你投资了2万进去，你想要赢回来。你不指望能赢钱，只要能保本就行。结果后半夜你把剩下的3万也输了，然后到后面甚至是借钱再去赌，然后越输越赌。

换到恋爱中也是一样，你投资得越多，沉没成本越高，退出就越难，最后可能输得更多。心里就会越来越不甘，然后会一直付出，直到输得体无完肤。

## 总结 📝

本节课中我们一起学习了长期关系中的“投资”概念。

一段健康的长期关系，是需要双方共同为之投资的，而不是一方单方面的付出。投资的方式可以多元化，包括**金钱投资**、**时间投资**、**情绪投资**和**肉体投资**。建立双向、平衡的投资模式，是维系长期关系稳定与幸福的关键。
