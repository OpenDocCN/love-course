# 许诺《情感宝典》：016.15、吸引阶段-朋友圈展示面的打造 📱

在本节课中，我们将要学习如何打造一个具有吸引力的朋友圈展示面。这是追女生最基础的一步，核心在于通过社交媒体扬长避短地展示自己，从而让女生愿意与你聊天和了解你。

## 展示面的核心概念与重要性 🎯

正确的展示自己属于追女生最基础的东西。简单来说，就是扬长避短地通过你的朋友圈告诉别人：**我是谁？我有什么？我的生活是什么样的？**

我们和女生聊天时，女生不回复或态度冷淡，有时并非因为不会聊天，而是女生根本不给这个机会。客观来讲，打造好展示面，才能让女生愿意跟你聊天，愿意去了解你。

## 展示面的五大组成部分 🧩

展示面主要由5个部分组成：**昵称、头像、背景墙、签名、常规朋友圈**。

上一节我们介绍了展示面的重要性，本节中我们来看看这五个部分的具体打造方法。

### 1. 昵称：品味的第一印象

昵称是他人对你品味的初步判断。

以下是昵称设置的原则：
*   **避免使用**：“追风少年”、“风度翩翩”、“哥是个传说”等非主流或暴露年龄的昵称。
*   **推荐使用**：简单明了的昵称。可以是一个英文名，或英文加汉字的组合。例如：`alen`。
*   **核心原则**：无论什么形式，**简单就是高级**。

### 2. 头像：你是谁的关键标识

头像告诉别人你是谁，证明你是一个真实的人。它非常重要，因为第一印象决定了对方是否愿意接近你。

在学习如何拍摄头像前，先了解哪些照片不适合做头像。

以下是绝对不能用作头像的照片类型：
*   风景照
*   动物照
*   证件照
*   卡通照
*   网图或画质很差的照片

接下来，我们来看看正确的头像应该怎么拍。

正确的头像拍摄技巧：
1.  **环境大于相貌**：选择高大上的背景，将视觉中心从人物相貌转移到环境上。即使相貌普通，整体感觉也会“逼格满满”。
2.  **借助道具营造氛围**：可以借助宠物或娃娃，拍出温暖的感觉。
3.  **制造神秘感**：利用饮料、手机、口罩、围巾、帽子等道具，不完全露出全脸，让女生产生好奇。

### 3. 朋友圈封面：你的黄金广告位

如果说头像是吸引女生浏览朋友圈的“引流器”，那么朋友圈封面就是你的“黄金位置广告牌”。使用不当会直接传达出“我很low”的信号。

以下是朋友圈封面一定不能用的类型：
*   网图
*   大头自拍（显得自恋）
*   过时的旅游照（特别是比剪刀手的）
*   普通的风景照（除非拍得非常艺术化）

那么，朋友圈封面应该选用什么样的照片呢？

封面一定要展现出自己的**价值**。所谓价值，要么是别人拍不出的照片，要么是能体现你生活水平、并非大多数人能经常接触的“高逼格”照片。

以下是合格封面图的例子：
*   潜水的照片
*   坐飞机头等舱的照片
*   冲浪的照片

**补充建议**：
*   如果生活单调，一张有设计感的纯文字图片也是不错的选择。
*   退一步，去网上找一张高逼格的图片也行。
*   **核心原则**：拍好的照片一定要修图。坚持“不修图，不发朋友圈”的原则，你的展示面就成功了一半。

总之，封面图需要告诉看你朋友圈的人：**我是个精致、生活丰富、具有高价值、审美在线的人**。

### 4. 个性签名：三观的简短陈述

签名告诉别人你的三观。它可以是一句意味深长的话，也可以是一句英文。核心目的是让女生觉得你是一个有品位的男人。

### 5. 常规朋友圈：你经营的核心产品

头像是“引流器”，封面是“广告牌”，那么日常朋友圈就是你为自己经营的“产品”。如果日常朋友圈质量低下，前两者做得再好也无济于事。

首先，我们需要明白什么样的朋友圈是**不能发**的。

以下是必须避免的朋友圈内容：
*   **转发的心灵鸡汤或无病呻吟的文章**：女生没兴趣看，会把你定义为“没有自己生活的屌丝”。
*   **纯文字**：提供的信息价值少，女生无法了解你。
*   **随意拍摄的、低质量的朋友聚会照**：“物以类聚，人以群分”，低质的环境和朋友会拉低你的价值。给朋友拍照也要认真，并选择好环境。
*   **工作广告推销信息**：请使用微信“分组可见”功能，将广告留给客户，精彩生活留给女生。
*   **太过专业的学术性内容**：除非你是摄影师、音乐人等适合展示的职业，否则大多数女生看不懂也不感兴趣。

了解禁忌后，我们来看看什么样的朋友圈才是**正确且吸引人**的。

以下是推荐发布的、能展示高价值的朋友圈内容：
1.  **美食**：发的是“美食”，不是日常三餐。目的是打开女生窗口，让她好奇地问“这是哪里？”。例如：`精致的西餐、日料、高颜值甜品`。
2.  **丰富的娱乐生活**：展示多样化活动，能开启不同类型女生的窗口。例如：`健身、骑马、旅行、游泳、射箭、参观画展或摄影展`。这传达出：你上进、懂得自我投资、有艺术修养。
3.  **适当的形象展示**：让女生知道你长什么样，培养安全感。但切忌频繁，最重要的是：**修图、修图、修图**。
4.  **宠物或小朋友的展示**：女生对可爱事物抵抗力低。这能极大打开女生窗口，同时展示你的爱心。
5.  **预选展示**：晒你和**多位**颜值不错的女性的合照（避免单独一人，以免误会）。这暗示你很优秀，吸引其他女生思考“为什么你身边女性多”。
6.  **工作学习展示**：在深夜或凌晨发布工作学习状态，简单配文如“又是天亮才出公司”，能塑造努力、有上进心的形象。
7.  **朋友圈配文原则**：**宁愿少，不要多**。目的是制造悬念，打开女生询问的窗口。如果讲得太清楚，女生就没了问你的欲望。若不知如何配文，可参考相关素材库。
8.  **音乐分享**：分享好听、歌词有深度的歌曲，能体现品味。如果缺乏音乐审美，可以关注品味好的歌单作为参考。
