# 只有心中的小男孩死去，你才会成为真正的男人-P1-许诺有话说-BV1YL4y1s7wX_note

在本节课中，我们将探讨一个普遍存在于男性成长中的心理障碍——过于“听话”和“乖巧”如何阻碍你展现男子气概，特别是在与异性相处时。我们将解析其根源，并提供清晰的转变思路。

大家好，我是许诺。最近我发现私信我的粉丝很多都有一个同样的问题：在普通社交中能言善辩，但一到和自己喜欢的女生相处时，就变得无话可说、容易紧张、心态崩溃，甚至脸红害羞。

这个问题本质上是一个大问题：意味着你还是一个“乖宝宝”，而非一个真正的男人。成为爷们的关键一点，就是不害怕与任何人起冲突。本篇内容我们不谈成熟，只聚焦于男子气概的培养。

我记得两年前看过一个国外男性魅力研究会的视频，其中一位心理学教授讲授心态课程。他讲了很多，但我只记住了三句话，至今印象深刻。

以下是那三句核心观点：

**第一，你心中的小男孩必须死去，男人才会产生。**

你一定注意到了，我们从小接受的教育，都在教导我们成为听话的孩子。父母和老师要求你努力学习，告诫你很多事情不能做。这就像动漫《进击的巨人》里描绘的，被告知墙外很危险，不要探索。

回顾历史，真正被公认有男子气概的人，如秦始皇、商鞅、刘邦、曹操、岳飞，哪个是听话的乖乖男？他们都是敢作敢为，为了内心野心、目标或正义，敢于与世界为敌的人。男子气概本质是一种狼性，是敢于按照自己的想法行动，而非唯唯诺诺。

然而，男孩在成长为男人的路上，至少会遇到两个让你持续扮演“乖乖男”的角色。

**第一个角色是你的家长和老师。**

我们大部分人并非富贵出身，缺乏与生俱来的勇敢。在无权无势的环境下，往往意味着要小心翼翼、夹起尾巴做人。我们害怕的不是见义勇为本身，而是其可能带来的后果。长此以往，这会成为你的性格，男子气概中的勇敢特质便慢慢消磨殆尽。

家长和老师出于保护，绝不希望你受到伤害，但这容易导致过度保护，培养出温室里的花朵。而男孩到男人的成长之路，必然铺满荆棘。你的长辈更可能鼓励你寻求安定稳定的生活，而非走上这条充满挑战的路。

**第二个阻止你成为男人的角色，是某些女性。**

如果你在青春期后遇到一位友善、温暖且支持你的姑娘，那非常幸运，她的爱与支持会让你越来越有爷们气概。

但很多人喜欢的，是被众多男人追捧、惯坏的所谓“女神”。这类人可能会在生活中教导你如何听话、如何为女人付出。例如，有粉丝曾私信我，他的女友要求将他父母买的两套房子中较小的一套给她弟弟，并询问这样是否合理，他不答应是否过分。

如果你不幸遇到一个自私自恋、懂得玩弄男性心理，且恰好有几分姿色让你动心的女人，她很可能会以“爱”的名义说服并要求你，让你根本无法成长为一个真正的男人。

这两个障碍非常巨大，因为它们往往以“爱”的名义出现：第一种告诉你“听话是因为爱你”，第二种告诉你“如果爱她就要听话”。因此，很多人心中那个乖乖听话的男孩从未死去，一直活了下来。

**第二，并非你温柔就是绅士，真正的绅士并不害怕冲突。**

以大家都熟悉的影视形象为例，美国队长总是冲锋在前、战斗不止，但他绝对是一位真正的绅士。

回想我们的成长过程，我们有过多少次选择漠视与沉默？又有多少次担惊受怕？我们都有过。这句话无需过度解读，因为它已说明一切：**并非你温柔就是绅士；而是你强大了，你的温柔才是绅士。** 如果你弱小、卑微、唯唯诺诺，那么你的温柔在别人眼中只是软弱的表现。
