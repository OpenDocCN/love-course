## 步骤解析

在分享用餐照片时，追求者会有意只拍摄同性朋友。这一行为旨在向对方暗示自己社交圈单纯、身边没有复杂的异性关系，从而塑造出“靠谱”、“单纯”的个人形象。

### 第四步：展现特殊关注
追求者会通过语言和行为强调对方的特殊性。例如，声称“工作时只回复你的消息”，或在忙碌后表达“不好意思让你等了那么久”。核心公式为：**特殊对待 = 表达在乎**。

```plaintext

    制造惊喜或小冲突;

    进行安抚与关怀;
```

追求者会密切关注对方社交动态，记录其喜好、生活习惯乃至生理周期等细节。并在适当时机（如对方身体不适时）提供精准关怀（如送上红糖水），以此营造“极度细心”和“真心”的印象。
