# 聊天基础课 01：为什么你的聊天总是失败？🤔

在本节课中，我们将要学习导致聊天失败的根本原因。我们将从“吸引力”这一核心概念入手，分析形象与聊天意愿的关系，并深入探讨如何通过调整聊天心态和方式，让对方感到愉快，从而建立顺畅的沟通。

## 形象是吸引力的基石 🧱

决定别人是否想和你聊天的首要因素，是你的形象是否具有足够的吸引力。这与男生喜欢和美女聊天，女生也喜欢和有魅力的帅哥聊天的道理相同。形象这一点不单指外貌。

你的形象取决于你的整体表现。例如，一个第一眼看起来是帅哥的人，如果他脏话连篇、举止粗鲁，女生也会生厌。反之，一个其貌不扬，但是幽默风趣且充满智慧的人，女生和他相处一段时间后，会因为他人格魅力而为他整体形象加分。

所以，不要因为自己先天长得不够好而产生自卑心理。只要你可能把自己稍微收拾一下，再养成一个好的性格，你的形象这一关就过了。

## 第一印象与聊天动力 💨

当女生第一印象觉得一个人“不靠谱”时，就无法幻想和这个人在一起生活。没有幻想，就没有聊天的动力。如果这个人还一直缠着自己，就只想远离他了。

说完了别人愿意和你聊天的前提，我们再来谈谈要聊下去我们该怎么做。

## 聊天的核心：提供愉悦感，而非辩论 🎯

你们一定要明白一个道理：不管是谁，人家和你聊天是为了找乐子的，不是来生气的。你和别人聊天最主要的是要让别人感觉跟你聊天愉快，而不是跟你聊完了心情不爽。

女生一开始愿意和你接触，结果过几天就不想理你了，态度越来越冷淡，这是为什么？因为跟你聊天让她心情不爽。为什么女生跟你聊完了心情不爽？因为你让她感觉话不投机。

## 导致聊天不快的根本原因 ⚠️

一切的话不投机，聊得让人不爽，和你聊的什么内容没有直接关系。主要是因为，你们总是带着自己的主观情绪和想法来跟对方聊天。你的观点和个人情绪越强烈，就越无法给别人提供情绪价值。

## 改变聊天心态：从“表达”到“了解” 🔄

男生的逻辑思维比较强，所以在聊到观点类话题的时候，就特别容易发表自己的看法，从而忘了我们聊天的目的。聊天的目的是为了给对方提供情绪价值，是为了了解对方。

你们一定要改掉这个坏习惯。聊天高手在聊天时的心态是这样的：你的观点或许不对，但我不会反驳，甚至可以顺着你说。反对意见我保留，我的目的不是和你争辩，而是了解你。

**高手聊天是没有自己立场的。** 你要立场干嘛呢？你又不是在辩论赛。一切以对方高兴为主就完了。你有自己的立场和主见是很好，但是你非要事事都表达出来，处处和别人观点不一样，这就是聊天的大忌。说通俗点就是“抬杠”。

## 实用聊天口诀：一顺二问三夸没立场 📝

如果还没明白，你就要记住一个聊天口诀：**一顺二问三夸没立场**。

以下是这个口诀的具体解释：

*   **顺**：就是顺着对方说，保留自己的意见，多说对方爱听的话。
*   **问**：就是通过适当的提问，引导对方多说出自己的想法。
*   **夸**：就是当对方说到自己引以为豪的事情时，你要及时地夸赞。
*   **没立场**：意思就是你聊天不是为了来表达立场的。

## 总结 📚

本节课中，我们一起学习了聊天失败的核心原因。首先，**吸引力（形象）** 是开启对话的门票，它由外貌、言行举止和人格魅力共同构成。其次，聊天的核心目标是**提供情绪价值**，让对方感到愉悦，而非争论对错。最后，我们掌握了一个实用的心态和口诀：放弃强烈的自我立场，采用 **“顺、问、夸”** 的策略，以了解和取悦对方为中心，这样才能让聊天持续进行下去。
