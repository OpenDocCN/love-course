## 付出与价值的错位

上一节我们介绍了课程目标，本节中我们来看看追求中一个根本性的认知误区。很多男生在追求过程中，会对喜欢的对象倾其所有地付出。

然而，我们需要先思考一个问题：你的付出在对方眼里究竟是什么？

我讲一个故事。有一个男生，他工资不高，自己赚的钱都舍不得花。他买最好的东西送给对方，买最便宜的泡面留给自己。他渴望通过这种方式表明真心，以此感动对方。

而事实结果是，他为了喜欢的人放弃了自己的尊严，对方要他干什么他就干什么。他最后换来的只能是敷衍或者瞧不上。因为他的**感知价值**比对方低，所以他只能试图通过“超额付出”来获得青睐。

这种情况很普遍，并由此产生了第一级错误。

## 核心误区：把感动当成爱

以下是第一个核心误区，即混淆了“感动”与“爱情”。

这个误区是指，男生对女生非常好，所做之事甚至把自己都感动了，从而越来越“爱”对方，并认为女生也一定懂得这份无私的付出。

但需要明确一个公式：
**感动 ≠ 爱情**

很多男生遇到喜欢的女生，就开始对她好、为她付出，同时总会犯一个相同的错误：直接表白。

```
错误示范：
“我喜欢你，你能不能做我女朋友？”
```

此时，大部分男生得到的并非一个肯定的答复，而往往是拒绝。许多人这时会产生如下想法：
*   我可能做得还不够好。
*   我可能没有感动到她。
*   她可能还没看清我的努力。

于是，经过反复思考，他们得出结论：要更努力地去“追”，去让她知道自己是认真的，去“感动”她。随之而来的行为是更疯狂的输出：送礼物、请吃饭、上下班接送，用各种方式讨好对方。

## 行为背后的压力与目的性

那么，当你做这些事情时，女生心里在想什么？她看到的往往不是真心，而是一种**压力**。

为什么？

因为如果你的表白已被拒绝，说明女生目前不喜欢你。她会认为你对她有所企图。你所做的一切，在她看来都带有强烈的目的性。

这好比有个人找你帮忙，你拒绝了。后来他开始经常请你吃饭，你一定会认为他“别有用心”，才会如此殷勤，对吗？然后你就会开始抵触，甚至看到他来电都感到厌烦。

所以，表白被拒后，你所有的好、再多的礼物、再露骨的真心，在她眼里都变成了“有目的性的行为”。女生因此感到压力，因为她还不喜欢你，却要面对你强烈的目的性，她便会开始躲着你。

## 从影视案例看吸引的本质

我们可以通过一个例子来加深理解。相信不少人都看过《爱情公寓5》。剧中角色赵海棠追求诸葛大力的方式就是死缠烂打。即便诸葛大力多次明确表示不喜欢他，赵海棠仍不放弃，结果大家也看到了，大力最终并未被其行为感动。

赵海棠是“高富帅”，却没能追到学霸诸葛大力。而为什么看似平庸的张伟能够吸引诸葛大力呢？关键在于张伟身上拥有吸引大力的特质：**纯真**和**善良**。

这个案例说明，在你追求一个女生时，不要认为自身条件优秀，或通过无条件付出和表白就能成功。核心应转向思考：**对方欣赏和喜欢具有什么样特质的男生？**

在电视剧里，赵海棠最后还能和大力做朋友。但在现实中，很多男生因逼得太紧，导致被女生厌恶甚至拉黑删除。
