## 有效延续话题的技巧 🔄

成功开启话题只是第一步，如何让对话继续下去同样关键。延续话题的核心在于抓住对方语句中的关键点进行延伸或互动。

以下是延续话题的一个例子：
1.  你发起话题：“好饿，加班加到现在还没搞完。”
2.  女生回应：“这么惨吗？哎，我正在吃小龙虾。”
3.  你可以延续：“我觉得你在搞事情。”（对“吃小龙虾”这一状态进行情绪互动）
4.  女生可能回复一个笑的表情。
5.  你可以进一步延续：“有人到现在还没有吃饭，你这话让他更饿了。”（将“饿”的感受与对方的“吃”联系起来）
