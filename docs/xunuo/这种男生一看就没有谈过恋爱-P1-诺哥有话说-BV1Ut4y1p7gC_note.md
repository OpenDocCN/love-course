# 这种男生一看就没有谈过恋爱-P1-诺哥有话说-BV1Ut4y1p7gC_note

在本节课中，我们将要学习，为什么有些男生会给人一种“没有恋爱经验”的感觉，并总结出几个关键特征。通过学习这些特征并加以避免，你可以更自信地展现自己，无论你是否有过恋爱经历。

上一节我们介绍了课程的主题，本节中我们来看看具体有哪些行为会暴露恋爱经验的缺乏。

以下是几个会让人感觉“没有谈过恋爱”的典型特征：

1.  **过分专一，不懂取舍**
    认为自己认定的就绝不悔改，即使暗恋也很难放弃。这显得有些一根筋。谈过恋爱的男性更懂得取舍，因为他们明白 `永远叫不醒一个装睡的人`。

2.  **几乎从不主动**
    他们很想主动，但不知道如何主动，并且害怕被拒绝后显得丢人。谈过恋爱的男性知道，被拒绝是正常的，开个玩笑也就过去了。因为他们明白，如果女生一开始就非常主动，`不是骗子就是图谋不轨`。主动，特别是主动暧昧，是建立关系的关键。

3.  **过于听从“禁止早恋”的教导**
    从小被长辈灌输“不要早恋，好好学习”的观念，并且真的听话照做。而很多有过恋爱经历的男生，往往都有过早恋的经历。

4.  **过于在意他人感受，但缺乏同理心**
    做事犹豫不决，生怕考虑不周，但往往猜不透别人的想法和感受。

5.  **条件不错但极度不自信**
    即使自身条件过得去，依然会怀疑、嫌弃自己，甚至感到自卑，完全不知道如何与女生交流。自信的秘诀有时在于 `“臭不要脸一点，面子就是个屁”`。

6.  **有心理洁癖或玻璃心**
    对他人的小毛病不够包容，一点小事就觉得无法接受，活在自己的世界里。

7.  **轻易喜欢上一个人**
    可以说是“见一个爱一个”，情感投入过快且缺乏深度。

8.  **有强烈的“处女情结”**
    认为结婚对象必须是处女，否则心里过不去这个坎。

9.  **总觉得自己有缺陷**
    例如认为自己的外形条件差、个子矮、皮肤差、不会说话等，这种自卑导致只敢默默喜欢，而不敢主动出击。

本节课中我们一起学习了哪些行为容易让人贴上“恋爱经验不足”的标签。核心在于，没有谈过恋爱并不是丢人的事，关键在于自信。如果你自己觉得这件事丢人并试图掩饰，即使别人相信了，你自己也会感到憋屈。建立自信，坦然面对，才是最重要的第一步。
