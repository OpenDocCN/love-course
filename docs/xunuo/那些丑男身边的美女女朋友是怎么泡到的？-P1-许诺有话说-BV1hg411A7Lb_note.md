# 恋爱教程 01：如何建立吸引力与赢得爱情 ❤️

在本节课中，我们将要学习如何理解吸引力在恋爱关系中的核心作用，以及如何通过提升自身价值来赢得心仪对象的青睐，无论你的起点如何。

---

相对于男生而言，女生择偶对于外貌的标准，并不像很多人想象的那么高。只要达到了一般水平，干净清爽就可以了。但前提是你不能丑。如果真的是能划分到丑这个等级的男生，大多数女生会直接选择放弃。甚至丑的男生，连当女生的备胎或者千斤顶的资格都没有。

你试想一下，一个丑男每天屁颠屁颠地给女生端茶倒水，忙里忙外地追求。让别人看到了，这个女生的身价和面子往哪里搁。大部分女生身边的朋友都会嗤之以鼻，背后更会有很多流言蜚语，难听的话不堪入耳。有句话不是说“你是什么样的人就会吸引什么样的人”吗？对于这种情况，女生甚至会想：“我到底做错了什么，你要喜欢我？我改还不行吗？”

那么，长得丑的男生，真的就不可能赢得爱情了吗？真有这种可能。不过对于一般没有受过感情刺激的女孩子而言，这不是首选。

上一节我们讨论了外貌的普遍门槛，本节中我们来看看一个特殊的成功案例。

上大学的时候，我们当时有一个专业的系花，就找了一个外表看起来挺丑的男生。这个男生没有别的，就是不断地对这个女生好，风雨无阻，而且被拒绝了也没有放弃。除了执着稳重一点以外，听说确实也没有什么过人的优点。可是几个月以后他成功了。

当所有人对他们在一起产生好奇的时候，这个女孩子接受他。他对她好只是一方面的，更重要的是女孩的心态。这个女孩由于一直挺受欢迎的，也谈过不少恋爱，只不过经常遇到渣男，每次都全身心投入，结果都是被伤得特别惨。于是，她对感情的需求和同龄的女孩不一样，她就想要一个真心实意对她好的人，丑点就丑点吧。而他们在一起也是因为这个男生踏实、靠谱，这也是女生欣赏到他身上的一些点。

当然，这是特殊情况。也不是每一个美女都有过这种悲惨的经历。如果你不懂得一些聊天技巧，你的成功率也是非常低的。

以下是提升成功率的资源介绍。

那么我给大家准备了30多套，我和各种类型女生的聊天案例解析。很多长相一般的男生看完之后照着去聊，也都成功地追到了自己的女神。还没看过也想看的兄弟，可以私信或者评论区扣个1来找我要，我会发给你。记得用你的一个点赞关注和三连来和我交换，不要白嫖我就行。

能够长期让彼此双方的感情继续下去，一定是吸引力的部分。有很多男生还是想不通，如果你身上没有任何可以吸引到对方的价值，对你好只能作为一个暂时的加分项，而且是暂时的。

以下是构建吸引力的核心公式。

如果你没有任何身上一些高价值的特质，比如：
*   **有能力 + 对他好**
*   **果敢 + 对他好**
*   **有担当 + 对他好**
*   **长得帅 + 对他好**
*   **情商高 + 对他好**

等等等等，那么：
**长得丑 + 对他好 ≈ 非常容易被女生嫌弃的舔狗**

女生会认为你本身没有任何价值。你要是长得丑，这本来就是个减分项了。如果你身上会有一个特别好的个人特质，比如勇敢、善良、双商高等，再综合对他好，女生也是会有可能喜欢你的。

其实大多数男生根本达不到丑的水平。而提升和挖掘自身身上的优秀特质才是关键。在我看来没有无可救药的丑男。单从外在上来说，只要能够积极地摆正态度提升自己，就算是一分的男生，也是很有可能提升到五六分的。更何况兄弟你真的没有一分那么差，不要放弃自己才是真的。

短期内提升自己的外貌，通过努力并没有那么难。而同时，优异的内在特质兼备，才是获得爱情的独二法宝。
