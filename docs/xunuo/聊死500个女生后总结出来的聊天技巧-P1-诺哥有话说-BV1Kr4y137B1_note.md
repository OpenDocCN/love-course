# 聊天技巧入门课 - P1：聊天的核心原则 🗣️

在本节课中，我们将要学习如何避免聊天中的常见错误，并理解让对话愉快进行下去的核心原则。我们将从“形象”的重要性开始，逐步深入到聊天时的具体心态和技巧。

## 形象是吸引力的基础 🎭

决定别人想不想和你聊天的，是你的形象够不够有吸引力。就像男生都爱和美女聊天一样，女生也喜欢跟有魅力的人聊天。形象这一点不一定光看外貌。

你的形象取决于你的整体表现。比如一个第一眼看起来是个帅哥的人，如果他脏话连篇，举止粗鲁，女生也会反感。再比如一个其貌不扬，但是风趣幽默还很智慧的人，女生和他相处一段时间，会因为他的人格魅力，而为他的整体形象加分。

所以，不要因为自己先天长得不够好而产生自卑的心理。只要你肯把自己稍微收拾一下，再养成一个好的性格，你的形象这一关就过了。

当女生第一印象觉得一个人不好的时候，就无法幻想和这个人在一起的生活。没有幻想就没有聊天的动力，再加上这个人还一直缠着自己，就只想远离了。

上一节我们介绍了吸引对方愿意聊天的前提——形象。本节中我们来看看，当对话开始后，我们该如何做才能让聊天持续下去。

## 聊天的核心：提供情绪价值 😊

你们一定要明白一个道理：不管是谁，人家和你聊天是为了找乐子的，不是来生气的。你和别人聊天，最主要的就是要让别人感觉跟你聊天愉快，而不是跟你聊完了心情不爽。

女生一开始愿意和你接触，结果过几天就不想理你了，越来越冷淡，这是为什么呢？因为跟你聊天心情不爽。

为什么女生跟你聊完了感觉心情不爽呢？因为你让她感觉话不投机。很多兄弟完全不会聊天。

一切的话不投机，聊得让人不爽，和你用不用话术没关系。主要是因为你们总是带着自己的主观情绪和想法跟对方聊天。你的个人观点情绪越强，就越无法给别人提供情绪价值。

## 高手聊天的黄金心态 🧠

好了，重点来了。男生的思维逻辑比较强，所以在聊到观点类话题的时候，就特别容易发表自己的看法，就忘了我们聊天的目的是为了给对方提供情绪价值，是为了了解对方。你们一定要改掉这个坏习惯。

聊天高手他们在聊天的时候心态是这样的：
> 你的观点或许不对，但我不会反驳，甚至可以顺着你说。反对意见我保留。我的目的不是和你争辩，而是了解你。

高手聊天是没有自己立场的。你要立场干嘛呢？你又不是在辩论赛。一切以对方高兴为主就完事了。

你有自己的立场、有主见是很好，但你非要事事都表达出来，你和别人观点不一样，就是聊天大忌。说通俗点就是杠精行为。

## 实用聊天口诀：一顺二问三夸没立场 📝

还没明白的话，你就记住一个聊天口诀：**一顺二问三夸没立场**。

以下是这个口诀的具体解释：

*   **顺**：就是顺着对方说，保留自己的意见，多说对方爱听的话。
*   **问**：就是通过适当的提问，引导对方多说出自己的想法。
*   **夸**：就是当对方说到自己定义为好的事时，你要及时夸赞。
*   **没立场**：就很好理解了，意思就是你聊天不是为了来表达立场的。

## 总结与资源 📚

本节课中我们一起学习了聊天的核心原则。首先，**形象**是开启对话的敲门砖，它由外貌、言行和人格魅力共同构成。其次，聊天的根本目的是为对方**提供情绪价值**，让对方感到愉快，而非争论对错。最后，我们掌握了高手的聊天心态和实用口诀 **“一顺二问三夸没立场”**，这能帮助我们避免成为“话题终结者”。

记住，聊天的艺术在于倾听和共情，而非说教和辩论。
