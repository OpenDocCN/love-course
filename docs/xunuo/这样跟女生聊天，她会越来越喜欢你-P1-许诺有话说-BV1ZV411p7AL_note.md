## 聊天的核心：提供情绪价值，而非争论立场 🎯

上一节我们介绍了形象的重要性，本节中我们来看看聊天的核心原则。你们一定要明白一个道理：不管是谁，人家和你聊天是为了找乐子的，不是来生气的。

你和别人聊天最主要的就是要让别人感觉跟你聊天愉快，而不是跟你聊完了心情不爽。女生一开始愿意和你接触，结果过几天就不想理你了，越来越冷淡，这是为什么呢？因为跟你聊天心情不爽。

*   **爱讲道理**：在轻松聊天中试图用逻辑说服对方。
*   **反驳观点**：当女生发表一个观点，你非要去发表相反的观点证明自己的睿智，结果女生觉得你很弱智。

男生的逻辑思维比较强，所以在聊到观点类话题的时候就特别容易发表自己的看法，就忘了我们聊天的目的是为了给对方提供情绪价值，是为了了解对方。

## 高手聊天的黄金法则：顺、问、夸、无立场 ✨

你们一定要改掉上述的坏习惯。而聊天高手他们在聊天的时候心态是这样的：**你的观点或许不对，但我不会反驳，甚至可以顺着你说。反对意见我保留，我的目的不是和你争辩，而是了解你，和你建立联系。**

明白了吗？高手聊天是没有自己的立场的。你要立场干嘛呢？你又不是在参加辩论赛。一切以对方高兴为主就完事了。你有自己的立场，有主见这很好，但你非要事事都表达出来你和别人观点不一样，这就是聊天的误区。

以下是让聊天顺畅进行的四个关键动作：

*   **顺**：顺着对方说，保留自己的意见，多说对方爱听的话。
*   **问**：通过适当的提问，引导对方多说自己的想法。
*   **夸**：当对方说到自己引以为豪的事，你要及时的夸赞。
*   **无立场**：意思就是你聊天不是为了来表达立场的，核心目标是建立融洽关系。
