# 聊天技巧课程 01：如何让夸赞自然有效？💬

在本节课中，我们将学习三种基于心理学的夸赞技巧。这些方法能帮助你更自然、更有效地夸赞女生，避免显得刻意或油腻，从而更好地拉近关系。

## 概述

夸赞是拉近关系的重要方式，但不当的夸赞可能被识别为讨好或奉承，反而引起反感。本节课将介绍三种心理学方法，帮助你进行自然、真诚且有效的夸赞。

## 一、提及共同团体 🤝

上一节我们概述了夸赞的重要性，本节中我们来看看如何通过提及共同团体来增强夸赞的效果。

国外某公司老板兼人际关系处理师曾指出，若想拉近关系，开始对话时应使用双方所处的共同团体作为话题，这能增强说服力并快速建立信任感。这对应了“老乡见老乡，两眼泪汪汪”的老话，其背后的心理学原理是“同温层效应”：人们更重视与自己信念相符的人和事物。

因此，想通过夸赞拉近关系，首先要找到与对方所处的共同组织。

以下是具体应用方法：
*   结合共同点进行夸赞。例如：“你也是台球俱乐部的吧？喜欢台球的姑娘，心思都很细腻啊。”

使用此方法需注意独特性。例如，在国内旅游遇到一位姑娘时说“你也是中国人，对吧？”，这不会让女生感到有特殊联系，且意图容易被识破，结果可能适得其反。

## 二、采用欲扬先抑的技巧 🎭

我们了解了通过共同点夸赞的方法，接下来看看如何运用说话技巧让夸赞更易被接受。

职场中，老板常说：“这话你可能不爱听，但是你太善良了，容易在工作中吃亏。”这实际在指出工作能力不足，但听者不会太难受，因为该技巧掩盖了原意，让对方更容易接受建议。

此技巧同样适用于夸赞女生。

以下是常用句式示例：
*   “我知道你可能不喜欢听这些，但是...”
*   “我这么说，你估计会讨厌我，但是...”

“但是”后面需接一个担忧和一个优点。例如：“我这么说，你可能会讨厌我，但是你太善良了，可能会在生活中吃亏的。”人都喜欢听到正面评价，这样不仅能延伸话题，还能增加联系感。

若不知如何延伸话题，可以参考提供的聊天案例解析。许多学员通过正确夸赞从陌生聊到确立关系。

## 三、在寻求建议时夸赞对方 ❓

掌握了直接夸赞的技巧后，本节我们学习一种更间接的方式：在寻求建议时融入夸赞。

社交中，许多人会在向对方寻求建议时示好。他们不直接说“我很佩服你的商业头脑”，而是在请教后说：“这个想法真不错，你怎么拥有这样的商业头脑呢？”这会让话语更有说服力，让对方觉得是诚心请教和感慨，而非讨好。

人们通常更喜欢欣赏或向自己请教的人。向对方请教并随后夸赞，会让对方感到被认同，从而觉得彼此是一类人。

使用此方法需注意两点：
1.  不能请教女生完全不熟悉的领域，或在她看来你本应擅长的领域。例如，向不运动的女生请教篮球问题，则无法达到夸赞目的。
2.  不能以过于琐碎的事情请教并夸赞，否则会显得能力很弱。例如：“你知道厕所怎么走吗？”这就过于夸张了。

## 总结

本节课我们一起学习了三种不露痕迹的夸赞技巧：提及共同团体、欲扬先抑、在寻求建议时夸赞。许多男生因急于拉近关系而不断夸赞，但一味夸赞可能让女生感到不真诚甚至反感。真诚需要双方共同感知。

一句完美的夸赞至关重要。我们忽略它的原因通常有两类：一是不懂如何不露痕迹地夸赞；二是不够坚持，因为夸赞的效果往往是潜移默化的。学会并坚持这些夸赞方法，可能会让你心仪的她最终被你的真诚所打动。
