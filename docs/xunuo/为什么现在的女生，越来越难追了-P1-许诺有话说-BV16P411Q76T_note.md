# 恋爱心理学入门 第1课：为什么“跪舔”行为会降低吸引力？🤔

在本节课中，我们将通过一个真实案例，分析在追求关系中，过度付出（俗称“跪舔”）为何往往会导致事与愿违，降低自身吸引力。我们将探讨其中的心理机制，并学习如何建立更健康、平等的互动关系。

## 案例背景：一位女性朋友的故事

一位22岁的女性朋友在一家互联网公司担任普通职员。某日下班后，她的老板以工作为由留下她谈心，并向她表白。老板表示，虽然她入职仅一个月，但从她来上班的第一天就注意到了她，并表达了强烈的喜爱之情。

这位朋友当时没有立即答应，也没有拒绝，只是表示感觉突然，需要时间考虑。

## 关系的发展与男方的持续付出

朋友将此事告知了我。我提醒她，许多事业有成的男性可能会倾向于追求刚步入社会、涉世未深的年轻女性，建议她谨慎考虑。

第二天，老板的行为升级了：他为她准备了爱心早餐放在桌上。朋友此时的想法是，老板年轻有为且经济条件优越，如果真心对待，并非不能考虑成为男女朋友。

随后几天，老板的行为进一步升级：
*   将她的月薪从**4000元**直接提升到**8000元**。
*   将她从繁忙的销售岗位调至较清闲的职能部门。
*   将她的休假从单休调整为双休。
*   再次表白。

尽管获得了这些实质性的好处，我的朋友依然没有答应成为他的女朋友。

## 心态的转变：从“考虑接受”到“感觉不配”

老板见她仍未答应，开始了更猛烈的物质攻势：
*   赠送各种高档包包、香水和化妆品。
*   承诺只要答应做他女朋友，条件可以任意提。
*   展示多本房产证，声称自己在上海拥有三套全款房产、数百万存款，目前月收入达**30万元**，且公司处于上升期。

然而，在查看他们的聊天记录后，我发现这位老板在情感交流上存在明显短板。他的聊天内容仅限于日常关心、点外卖和发红包，缺乏更深层次的互动。**不会聊天**是许多条件优越男性的一个常见痛点。

此后，我朋友的心态发生了根本性变化。她不再考虑是否要接受这段关系，而是产生了新的想法：**“不做他女朋友，也能享受他带来的物质好处。我还这么年轻，刚出社会就遇到条件如此优越的男性追求，或许我能找到更好的。”** 她打算一边享受对方的好，一边观望是否有其他更理想的追求者。

## 核心心理机制分析：“跪舔”如何降低自身价值？

许多男性自身条件优秀，完全配得上心仪的女生，初期也能保持吸引力。但一旦开始无底线地讨好、付出（即“跪舔”），女生往往会迅速失去兴趣。

这背后的心理可以用一个简单的类比来解释：

> 想象你在古玩市场看中一件商品。
> *   **情景A**：老板报价5000元。你讨价还价至2000元，老板表示最低4500。经过一番拉锯，最终以3500元成交。此时，你会觉得自己**赚了**。
> *   **情景B**：老板报价5000元。你试探性还价2000元，老板想都没想立刻答应。此时，你会怎么想？你会怀疑商品本身是否只值2000元，甚至觉得如果真以这个价格买下，自己可能**亏了**。

在恋爱关系中，“跪舔”行为就如同**情景B**中的老板。过快的、无条件的妥协和过度付出，不会让对方觉得“赚到了”，反而会引发对方的疑虑：“他为什么这么轻易就付出这么多？是不是他本身价值不够，才需要这样弥补？” 这无形中**降低了你自身的感知价值**，让对方觉得“和你在一起，是我亏了”。

## 本节课总结

本节课中，我们一起学习了过度付出（“跪舔”）在追求关系中的负面影响。通过案例分析，我们看到了女方面对猛烈物质攻势时，心态如何从考虑接受转变为怀疑和待价而沽。其核心机制在于，不平衡的、单方面的高强度付出会扭曲双方的价值感知，让付出方显得“廉价”，从而降低吸引力。

健康的亲密关系应建立在**相互尊重、价值对等和自然吸引**的基础上，而非单方面的讨好与物质堆砌。在下一节课中，我们将探讨如何通过有效的沟通和展示真实的自我来建立持久的吸引力。
