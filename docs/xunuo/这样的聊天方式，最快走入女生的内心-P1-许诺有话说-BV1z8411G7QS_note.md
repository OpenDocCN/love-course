有一个男生喜欢一位女生，毕业后鼓起勇气给她发信息：“你真的很好看。”他收到的回复只可能是“谢谢”、“？”或“莫名其妙”。原本女生对他印象不错，却因这句不恰当的夸赞感到无语。

这种不走心，就像做菜没放盐，考试没填答题卡。

### 2. 具体深入地夸赞

有句古话叫“爱美之心人皆有之”，任何人都抵挡不了真诚的夸赞。所以对女生的夸赞和赞美，一定要具体落到细节上，千万别说一些概括性的空话。

这类深入细节的夸赞，是不是比“小姐姐你好漂亮”、“小姐姐你身材好好”更有说服力？

### 3. 第三人称夸赞

简单来讲，就是如何借别人的口去夸别人。如果你觉得对方很优秀，又不善于表达，可以试试这个方法。只是这个方法需要两人有共同好友。

例如，小刘见到你心仪的女生后，对她说：“她曾经说你做事特别细心，人又大方，今天一看果然是这样。”这种间接方法会让人觉得特别真实，又有感染力。虽然这种方法并不直白，没有完全将自己的情感全部表达给对方，但效果确实很好。不信你可以试一试。

### 4. 让她分享有成就感的事

每个人一生中，无论多么平淡，肯定都会经历高光时刻。比如上幼儿园时拿小红花，去食堂吃饭时阿姨因为你长得漂亮多给你加菜等等。

但不少人应该有这种感觉：别人夸赞时，觉得没夸到点子上。其实就是对方还不够了解你，不知道你内心最骄傲的事是什么，所以也没夸到真正的点上。

男生可以利用这一点，在与她相处的过程中慢慢深入了解，知道她的高光时刻后，在此基础上再去夸赞她。这时女生就会变得非常高兴，并且愿意与你愉快相处。
