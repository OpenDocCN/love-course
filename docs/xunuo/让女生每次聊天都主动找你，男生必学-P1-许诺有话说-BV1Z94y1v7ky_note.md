## 第一原则：别让自己觉得你在追求她

*   **心态失衡**：你会不自觉地将自己置于较低的位置。
*   **患得患失**：你的控制欲会变强，情绪极易受对方影响。

具体表现为：你不敢反驳她的话，不敢表达自己的不开心，甚至会去做自己不愿意做的事。同时，你可能频繁表露真心、发送长篇消息（俗称“小作文”），试图通过讨好来维系关系。

```
错误心态：我在追求她 -> 我必须让她满意 -> 我压抑自我、过度付出
```

即使女生最初对你有好感，这种不平等的相处模式也会导致她“有恃无恐”，认为你已经完全被“拿下”，从而失去继续了解和推进关系的积极性。而你会变得异常敏感，过度解读她的一言一行（例如：“她为什么不回我消息？”、“她为什么不发我送的朋友圈？”），心态剧烈波动。

许多人在此时会做出最不明智的举动——正式表白。这会给女生带来巨大压力，迫使她在准备不足的情况下做出选择，结果往往是拒绝或退回朋友关系。

**核心转变**：你需要认识到，恋爱是**双向选择**。你只是在接触和观察她，看彼此是否合适。如果没有，你可以选择疏远。保持这种心态，你就能平稳、自信地互动。

理解了如何调整自我心态后，我们来看看如何影响对方的感知。

```
正确策略：创造相处机会 -> 自然展示自我价值 -> 互相吸引 -> 关系水到渠成
```

你需要把大部分精力放在**展示自身价值**上，而非追问结果。例如，在相处愉快时自然地进行一些轻微的身体接触（如过马路时轻扶），在共同经历中让感情自然升温（如看恐怖电影时的依靠）。

最理想的状态是：你们因机缘相识，彼此有好感，于是经常聊天。在这个过程中，她逐渐了解你的优点，你也在观察彼此的契合度。相处愉快时，礼尚往来（例如你请吃饭，她请看电影）。彼此心中有好感，但都不急于点破。直到某个合适的时机，感情自然流露，关系顺势确立。
