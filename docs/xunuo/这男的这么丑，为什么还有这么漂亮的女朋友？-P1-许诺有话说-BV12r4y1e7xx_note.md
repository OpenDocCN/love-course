## 女生择偶的外貌标准

上一节我们提出了核心问题，本节中我们来看看女性在择偶时对外貌的真实要求。

相对男生而言，女生择偶对于外貌的标准并不像很多人想象的那么高。只要达到一般水平的干净清爽就可以，前提是你不能丑。如果真的是能划分到丑这个等级的男生，大多数女生会直接选择拒绝。

甚至丑的男生连当女生的备胎或者千斤顶的资格都没有。你试想一下，一个丑男，每天屁颠屁颠的给女生端茶倒水，忙里忙外的追，让别人看到了这个女生的身价和面子往哪里搁。大部分的女生身边的朋友都会嗤之以鼻，背后更是会有很多的流言蜚语，难听的话不堪入耳。

那么长得丑的男生真的不可能赢得爱情了吗？有这种可能，不过对于一般没有受过感情刺激的女孩子而言，这不是首选。

以下是基于一个真实案例的分析：
上大学的时候，我们当时有一个专业的系花，就找了一个外表看起来挺丑的男生。这个男生没有什么特别的，就是不断的对这个女生好，风雨无阻，而且被拒绝了也没有放弃。除了执着稳重以外，听说确实没有什么过人的优点。可是几个月以后他成功了。

## 长期关系的核心：吸引力

能够长期让彼此双方的感情继续下去的，一定是吸引力的部分。有很多男生还是想不通。

**核心公式：长期关系 ≈ 基础吸引力 + 对你好**

如果你身上没有任何可以吸引到对方的价值，对你好只能作为一个暂时的加分项，而且是暂时的。如果没有任何你身上高价值的特质，比如：
*   有能力 + 对他好
*   果敢 + 对他好
*   有担当 + 对他好
*   长得帅 + 对他好
*   情商高 + 对他好

那么，**长得丑 + 对他好**，基本就等于非常容易被女生嫌弃的“舔狗”。女生会认为你本身没有任何价值。

## 自我提升是关键

其实大多数男生根本达不到丑的水平，要提升和挖掘自身身上的优秀特质才是关键。

在我看来没有无药可救的丑男。单从外在上来说，只要能够积极的摆正态度提升自己，就算是一分的男生，也是很有可能提升到五六分的。更何况，你真的没有一分那么差，不要放弃自己才是真的。

短期内提升自己的外貌，通过努力并没有那么难。而同时，优异的内在特质兼备才是获取爱心的不二法宝。
