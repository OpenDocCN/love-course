## 行为特征分析

上一节我们概述了本课程的目标，本节中我们来看看具体的八个行为特征。

以下是识别缺乏恋爱经验女生的八个关键行为模式：

1.  **生活模式以“宅”为主**
    *   公式：`日常活动半径 ≈ 宿舍或家`
    *   上课期间尚可出门，一旦无需上课，基本在宿舍或家以外的地方见不到其人影。

2.  **在异性面前容易紧张**
    *   代码模式：`if (出现异性) { 行为模式 = “拘谨/沉默” }`
    *   与女性朋友相处时活泼开朗，一旦有异性出现，立刻变得拘谨或沉默。这并非刻意伪装，而是源于不好意思。

3.  **回避暧昧信息**
    *   当接收到异性发出的暧昧消息时，会立刻用打哈哈、装傻等方式回避，让他人难以继续深入话题。

4.  **拒绝态度明确果断**
    *   面对不喜欢的人的告白，会立刻明确拒绝，通常不会给予“做朋友”的缓冲机会。她们难以接受将他人当作备胎的行为。

5.  **社交主动性呈反向关系**
    *   公式：`对异性的好感度 ∝ 1 / 主动交流的意愿`
    *   从不主动与异性交流。对越有好感的人，越是不敢说话；反而对没有好感的人，能够正常聊上几句。

6.  **对单身状态坦然处之**
    *   虽然单身，但并不焦虑。生活重心在于刷剧、看漫画等个人娱乐，不会主动为“脱单”付出努力，倾向于被动等待“缘分”。

7.  **兴趣爱好广泛且投入**
    *   除了“宅”，还可能“佛系”。拥有喜欢的偶像或极其广泛的兴趣爱好，以至于认为谈恋爱是在占用她从事这些爱好的时间。

8.  **长期单身状态的累积结果**
    *   如果符合以上多数特征，那么很可能直到大学毕业都未曾有过恋爱经历。
