# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要一个地方来存放它，这就是**变量**。同时，数据本身有不同的种类，比如数字和文字，这些种类被称为**数据类型**。掌握如何创建变量并了解不同的数据类型，是开启编程之旅的关键。

## 1. 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）用来存放数据（值），而标签（变量名）则帮助我们记住盒子里装的是什么。

在Python中，我们使用等号 `=` 来给变量赋值。其基本语法是：
`变量名 = 值`

上一节我们介绍了变量的比喻，本节中我们来看看如何具体创建一个变量。

以下是创建变量的一个简单例子：
```python
message = “你好，世界！”
```
在这行代码中，我们创建了一个名为 `message` 的变量，并将文本“你好，世界！”这个值存放了进去。

## 2. 基本数据类型 🔢

数据并非只有一种形式。Python有几种内置的基本数据类型，用于表示不同种类的值。了解它们有助于我们正确地使用和操作数据。

以下是三种最常用的基本数据类型：

*   **整数**：没有小数部分的数字，例如 `-5`, `0`, `100`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由一系列字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`, `“123abc”`。

## 3. 查看与理解数据类型 🔍

有时，我们不确定一个变量里存放的数据是什么类型。Python提供了一个非常方便的函数 `type()` 来帮助我们查看。

以下是使用 `type()` 函数的示例：
```python
age = 25
print(type(age))  # 输出：<class ‘int’>，表示整数类型

price = 19.99
print(type(price)) # 输出：<class ‘float’>，表示浮点数类型

name = “Alice”
print(type(name))  # 输出：<class ‘str’>，表示字符串类型
```
通过 `type()` 函数，我们可以确认变量 `age` 是整数，`price` 是浮点数，而 `name` 是字符串。

## 4. 变量的命名规则 🏷️

为了让我们和计算机都能清晰地理解代码，给变量起名字需要遵循一些规则。

以下是变量命名的主要规则和约定：

*   **可以包含**：字母（a-z, A-Z）、数字（0-9）和下划线（_）。
*   **不能以数字开头**：`1variable` 是无效的，但 `variable1` 是有效的。
*   **区分大小写**：`Age` 和 `age` 是两个不同的变量。
*   **不能使用Python关键字**：例如 `if`, `for`, `while` 等具有特殊含义的单词不能用作变量名。
*   **建议使用描述性名称**：使用 `user_name` 比使用 `a` 或 `x` 更能清晰地表达变量的用途。

## 总结

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。
我们知道了变量就像存放数据的带标签盒子，并通过 `变量名 = 值` 的语法来创建它。
我们认识了三种基本数据类型：**整数**、**浮点数**和**字符串**。
我们还学会了使用 `type()` 函数来查看数据的类型，并了解了给变量命名的基本规则。
掌握这些概念，你就已经为学习更复杂的Python操作打下了坚实的基础。
