# 聊天基础课 01：如何建立吸引力与提供情绪价值 💬

在本节课中，我们将要学习如何通过提升个人形象和改变聊天心态，来增加他人与你聊天的意愿，并让对话持续愉快地进行下去。

## 形象是吸引力的基石 🧱

决定他人是否愿意与你聊天的首要因素，是你的整体形象是否具有吸引力。这与单纯的外貌无关。

你的形象由你的综合表现决定。例如，一个外表出众但言行粗俗的人，会令人反感。反之，一个外貌普通但风趣幽默、富有智慧的人，其人格魅力会为他的整体形象加分。

因此，不必因先天条件而自卑。只要注重个人打理并培养良好性格，就能通过形象这一关。

当女生对一个人的第一印象不佳时，便无法想象与对方共同生活的场景。缺乏这种想象，就没有持续聊天的动力。如果此时对方还不断纠缠，只会让人想远离。

## 聊天的核心：提供情绪价值 🎯

上一节我们介绍了吸引力的前提，本节中我们来看看如何让聊天顺利进行。

必须明白一个道理：任何人聊天都是为了寻求愉悦，而非添堵。聊天的主要目的是让对方感到愉快，而非在结束后心情变差。

女生可能最初愿意接触，但几天后变得冷淡。这通常是因为与你聊天让她感到不愉快。

为什么不愉快？因为你让她感到“话不投机”。

## 避免“话不投机”的关键 🔑

“话不投机”、聊天令人不快的根本原因，与你是否使用话术技巧无关。核心问题在于，你总是带着强烈的主观情绪和个人观点进行对话。

你的个人观点和情绪越强烈，就越难为对方提供情绪价值。

男生的思维逻辑性较强，因此在讨论观点类话题时，特别容易急于发表自己的看法，从而忘记了聊天的根本目的：**为对方提供情绪价值**和**了解对方**。

必须改掉这个习惯。

## 高手的聊天心态 🧠

聊天高手的心态如下：
> 你的观点或许不对，但我不会反驳。我甚至可以顺着你说，保留自己的反对意见。我的目的不是与你争辩，而是了解你。

高手聊天没有固定立场。聊天不是辩论赛，一切以让对方感到高兴为主。

有主见是优点，但若事事都要表达，尤其当观点与他人相左时，便是聊天大忌。通俗地说，这就是“杠精”行为。

## 实用聊天口诀 📝

如果以上内容还不完全明白，请记住这个聊天口诀：**一顺、二问、三夸、没立场**。

以下是口诀的具体解释：
*   **顺**：顺着对方的话说，保留自己的意见，多说对方爱听的话。
*   **问**：通过适当提问，引导对方多表达自己的想法。
*   **夸**：当对方谈到自己引以为豪的事情时，及时给予夸赞。
*   **没立场**：聊天不是为了表达和捍卫自己的立场。

---

本节课中我们一起学习了：**吸引力源于整体形象**，而**愉快的聊天关键在于放下主观立场，以提供情绪价值和了解对方为核心**。掌握“顺、问、夸、没立场”的口诀，能帮助你有效改善聊天氛围。
