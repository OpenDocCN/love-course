# 许诺《情感宝典》：019.18、追求阶段-聊天时的心态_ev_note

在本节课中，我们将要学习在追求阶段与女生聊天时，应该建立和保持哪些核心心态。这些心态是顺畅沟通、建立吸引力的基础。

---

## 概述

聊天前的心理状态至关重要。许多男性知道需要自信和坦然，但实践起来却感到困难。本节课程将为你提供三种具体、可操作的重要心态，帮助你克服紧张，提升聊天效果。

---

## 一、 核心基础：去做心态 🛠️

上一节我们提到了心态的重要性，本节中我们首先来看看最根本的“去做心态”。传统的心态建设方法，如自我暗示“我很自信”，往往效果有限。因为世界的普遍价值观（例如女生可能偏好高富帅）不会因个人暗示而改变。

追求女生的核心不在于你如何看待世界，而在于**女生如何看待这个世界**。因此，你需要的心态是“去做心态”，即通过实际行动让自己真正变得优秀，从而获得由内而外的自信。

“去做心态”意味着：**停止空想，开始行动**。例如，改造个人形象、坚持健身、广泛阅读、增加社交实践。当你通过切实努力取得进步，并获得他人（包括朋友和异性）的正面反馈时，自信便会自然建立。

从“去做心态”这一基础出发，可以衍生出两个在聊天前至关重要的具体心态。

---

## 二、 衍生心态一：提供价值的心态 💎

在“去做心态”之下，我们首先要建立的是“提供价值的心态”。与女生聊天时，你需要明白一个核心公式：

**提供价值 = 产生感觉**

只有你提供的价值足够多，女生对你的感觉才会更好。相反，一味索取价值则会降低吸引力。

以下是提供价值的两种具体形式：

### 1. 情绪价值

情绪价值并非指一个人不停说话，而是指**你能影响对方的情绪，制造情绪波动**，例如让对方感到开心、紧张、期待或失落。

可以制造情绪波动的聊天技巧主要有三个：

*   **推拉**
    推拉是指先表达欣赏（拉近），再开玩笑式地贬低或否定（推开），以此制造心理落差和趣味。
    **代码示例（对话模拟）**：
    ```
    女：（发来与朋友的合影）
    你：一看就知道右边的是你。（拉近，暗示她好看）
    女：哈哈，因为我最美？
    你：不是，右边的胸大一点。（推开，开玩笑）
    女：打死你！
    你：但我喜欢胸小点的，穿衣服好看。（再次拉近）
    ```

*   **曲解**
    曲解是故意以幽默、自恋的方式误解对方的话，营造轻松有趣的氛围。
    **代码示例（对话模拟）**：
    ```
    女：昨天竟然梦到你了。
    你：占我便宜没？（曲解为对方对自己图谋不轨）
    女：（发表情）
    你：完了，肯定被你占了。
    ```

*   **打压**
    打压通常在女生态度突然变冷淡、摆高姿态，或聊天框架不平等时使用，目的是维持自己的框架和姿态。注意，打压不是人身攻击，而是态度上的强硬回应。
    **代码示例（对话模拟）**：
    ```
    （当女生回复敷衍时）
    你：你平常都这样和人聊天的吗？
    （当女生在暧昧后突然退缩时）
    女：可能我也不是你喜欢的长相吧。
    你：不靠谱你还聊？我反正没时间和我觉得不靠谱的人闲聊。
    ```

### 2. 实际价值

实际价值指**女生可以直接通过视觉感知到的你的生活状态、品味或能力**。可以通过分享照片或小视频来展示。

**代码示例（行动指南）**：
*   在精致的环境用餐时，可分享照片。
*   健身时，可分享健身状态。
*   开车、烹饪或参与有趣活动时，均可通过微信小视频功能分享。

**注意**：展示的内容必须本身具有价值。避免在杂乱差的环境下拍照分享，否则会产生反效果。

与提供价值相对的是**索取价值**，即不断向女生提问，如“在干嘛？”、“吃了吗？”，这会让对方感到被索取和压力。

## 三、 衍生心态二：游戏者心态 🎮

接下来，我们看看第二种重要的衍生心态：“游戏者心态”。这能有效帮助你降低需求感，保持从容。

游戏者心态包含两个方面：

1.  **一边玩一边撩**
    聊天时不应放下所有事情守着手机。正确的做法是像处理游戏通知一样：收到消息后，可以继续手头的工作或娱乐，过一段时间再回复。这能有效降低你的焦虑感和需求感。

2.  **有选择性**
    你不能只和一个女生聊天。如果资源单一，你会过度关注对方的一举一动，导致得失心过重。拥有多个聊天对象（即“资源”），就像游戏中有多条“命”，能让你心态更放松，表现更自然。有选择性既是一种心态，也是一种需要主动积累资源后才能拥有的状态。

## 四、 重要认知：接受不被所有人喜欢 ❤️‍🩹

最后，我们需要建立一个重要认知：**你无法让所有人都喜欢你**。

即使你已经做得很好了，仍可能遇到不回应你的女生。请记住，我们不是人民币，不需要也做不到让每个人都喜欢。

**行动准则**：只与那些对你反馈良好、态度友好的女生深入聊天。不要将时间和精力浪费在一开始就对你冷漠的人身上。在没有犯错的前提下，初始态度就不友好的女生，后期很难通过聊天扭转印象。

## 总结

本节课中我们一起学习了与女生聊天前的核心心态建设：

1.  **基础是“去做心态”**：通过实际行动提升自我，获得真实自信。
2.  **由此衍生出“提供价值的心态”**：牢记公式 **提供价值 = 产生感觉**，并通过提供**情绪价值**（运用推拉、曲解、打压等技巧）和**实际价值**（展示优质生活）来吸引对方。
3.  **同时具备“游戏者心态”**：保持“一边玩一边撩”的轻松感，并建立“有选择性”的资源池，以降低需求感。
4.  **最终要接受现实**：不必强求所有人喜欢，只专注于与对自己反馈积极的女生互动。

以上便是在微信聊天前，你应该具备和调整的心态与状态。掌握这些，你将能更从容、有效地开启和推进对话。
