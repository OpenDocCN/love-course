# 恋爱心理学课程 01：如何扭转单恋局面 ❤️🔥

在本节课中，我们将要学习如何应对“喜欢的人心有所属”这一常见困境。我们将分析单方面付出的误区，并探讨通过提供情绪价值来建立吸引力的核心方法。

## 案例背景与分析

一位粉丝咨询：他喜欢一位女生，但女生心有所属，正在等待另一位有女友的男生。粉丝不断付出，女生却劝他停止，认为不值得他这样做。

你的故事挺感人，可是你还是没有搞清楚爱情的本质。

上一节我们介绍了案例背景，本节中我们来看看问题的核心所在。

## 爱情的本质：不是同情

爱情不是同情。你以为对她说你用情专一，愿意为她付出一切，她就喜欢你吗？你辛辛苦苦的付出，得到的是她的可怜和同情，觉得你付出这么多也挺辛苦的。

但同情和爱情真的是离的十万八千里。

你和她其实都是同一类人，感觉为自己喜欢的人付出一切，就会打动对方。她现在心里还有那个男生，你心里还有她，你们其实都是同样的处境。

你看，她为了一个男人付出这么多，不也一样没有结果吗？你以为你用同样的方法，你就会有更好的结果？不会的。

**爱情的本质公式**：`爱情 ≠ 同情`。你苦苦的坚持是打动不了她的。

爱情是情不自禁的心跳加速，情不自禁的想起你，情不自禁的去猜你到底想什么。

## 核心策略：提供情绪价值

你现在要做的根本不是想尽办法对她好。她真的是个不错的姑娘，她一直在说服你停止对她好，而不是保持一直暧昧。如果她心安理得的接受你的付出，那你就一定会是一个可怜的备胎。

你现在要做的就是给她提供一些情绪价值。

想做到这一点，你必须提供这些感觉：
以下是需要提供的核心情绪类型：
*   开心
*   好奇
*   激动
*   嫉妒
*   生气
*   失望
*   伤心
*   牵挂

当她对你有各种七上八下的感觉，你已经开始左右她的情绪了。她的心她的心跳是为你而动心的。

## 实践方法：创造“吊桥效应”

如果我说了你还是不会，那么可以学习具体案例。接下来我举个例子，你可以带她做一次逃单游戏，她就会产生一次惊险的吊桥效应。

我在以前的案例中讲过逃单游戏。你约女孩吃饭先偷偷把钱付了，然后假装跟女孩说今天我们吃霸王餐吧，然后牵着她的手跑出去。

女孩会觉得很突然又有点刺激，自然会紧张心跳加速。等你们逃出来，你再告诉她你已经付帐了，她会打你一顿。她哭笑不得的反应也可以带给她很多的情绪。

**什么是吊桥效应？**

> 吊桥效应是指当一个人提心吊胆的过吊桥的时候，会不由自主的心跳加快。如果这个时候碰巧遇见一个异性，那么她会误以为眼前出现的这个异性，就是自己的生命中的另一半，从而对其产生感情。

这是因为情绪受到了行为影响。在吊桥上由于危险的情境，人们会不由自主的心跳加快，错把由这种情境引起的心跳加快，理解为对方使自己心动才产生的生理反应，故而对对方滋生出爱情的情愫。

为什么很多人旅行的时候容易有艳遇？因为旅行的时候经常穿梭各种陌生和未知的环境，有好奇心、有刺激和期待的感觉，也有各种惊喜的体验。要是有一个刚结识的异性相伴，你们就会共享这些体验，很快两个人就会产生爱情的情愫。

## 行动纲领：从“被动付出”到“主动吸引”

你想让女生喜欢你，就需要让她被你不断的影响，情绪随着你而改变，这样她的心智就会渐渐被你占领。当她脑子里经常浮现出来的人是你，那你自然就会置换掉那个她曾经念念不忘的人。

因为你做的太多跪舔的行为，已经很被动了。你现在应该感激这个男人，他现在心里只有她，你的对手只有一个。如果她没有喜欢的男人，你可能要面对更多的竞争对手，你根本没有时间去扭转被动的局面，她可能一早就和别的男人在一起了。

现在你知道为什么自己追得这么累了吧？因为你给她的感觉太确定了，太平淡了。她对你没有明显的情绪起伏，而你总是能够轻易的被她左右。她可以轻易的改变你的情绪，所以你对她有感觉，但她对你没感觉。

以下是扭转局面的具体行动步骤：
*   **停止讨好**：必须停止一切主动的讨好和献殷勤。
*   **调整关系定位**：从最舒服的朋友关系开始，然后把她当一个女生，而不是一个女神。
*   **改变互动模式**：不卑不亢的相处，敢于开玩笑，敢于挖苦她，敢于贬低她。
*   **创造体验**：敢于带她做一些刺激的事情。
*   **保持自我中心**：敢于偶尔忽略她，不再以她为生活的中心。

让她产生不确定的感觉，并让她因为你而产生各种情绪波动。这样你就不是一个被她同情的角色，因为你不再是一个可怜兮兮的男人，而是一个有主见有立场，而且能够吸引到她的男人。这样你才有机会占领她的心。

## 总结

本节课中我们一起学习了如何扭转单恋的被动局面。核心在于理解**爱情源于吸引而非同情**，并通过创造**情绪波动**和**吊桥效应**来建立吸引力。关键在于停止单方面付出，以平等、有趣且略带挑战性的方式互动，从而占据对方的心智。
