上一节我们介绍了课程的主题，本节中我们来看看，当表白被拒后，错误的行为会如何彻底破坏原有的好感。

一个女生分享了自己的经历：她原本对一位细心善良的男生朋友抱有好感。但在对方表白并被拒绝后，男生开始每天发微信、打电话，试图强迫她聊天。这种行为让她感到窒息和厌烦，最终连最初的好感也消失了。

以下是她的核心感受：
*   她认为，没有女生会喜欢上“卑微感”暴露得如此严重的人。
*   即便对方人品好、善良，但过度的纠缠只会引发“生理上的恶心”。
*   这种压迫感最终导致关系彻底断绝。

所以，给男生的核心建议是：**依靠自身的自信和魅力去吸引对方，而不是通过讨好和纠缠。**

上一节我们看到了错误示范的后果，本节中我们来看看其背后的心理机制。

关键在于**需求感的暴露**。如果你喜欢的女生对你并无好感，你过度表露需求感只会激发她的防御心理。

这个过程的生理基础是：当女生真正喜欢一个人时，身体会分泌多巴胺等激素，促使她产生亲近的意愿。反之，若她还未喜欢上你，得知你的想法会激发其**保护自己、害怕被伤害的生物本能**。这是进化过程中写入基因的**条件反射**。

上一节我们探讨了心理机制，本节中我们来分析男女在关系推进节奏上的根本差异。
