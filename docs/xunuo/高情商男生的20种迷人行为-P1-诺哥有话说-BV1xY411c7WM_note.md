# 课程一：高情商男生的20种迷人行为（上）🎯

在本节课中，我们将学习高情商行为的前半部分。这些行为核心在于**换位思考**与**精准表达**，能帮助你在社交、职场和亲密关系中建立良好印象，避免不必要的误会与冲突。

---

## 1. 办公室谈话的边界感 🚪

上一节我们介绍了课程概述，本节中我们来看看具体场景。在职场中，保持适当的边界感至关重要。

如果领导与下属性别不同，尤其是男领导与女下属进行一对一办公室谈话时，门最好不要完全关闭。可以开一条缝或保持半开状态。

> 我希望你永远不需要知道为何要这样做。

## 2. 维护朋友的隐私与形象 🤝

无论你与兄弟或闺蜜的关系多么亲密，即便你们私下相处非常随意，在其他异性面前也不要透露这些细节。

## 3. 在外维护伴侣形象 ❤️

同理，与另一半即使回家后争吵激烈，也最好不要在外人面前表现出来。可以轻微吐槽，但不要持续抱怨伴侣的缺点。

## 4. 识别并尊重对方状态 👀

与对方面谈时，如果发现对方身体未朝向您，或眼神游离，应及时中止对话。可以说：“今天先到这里，您先忙。”

## 5. 创造私人谈话空间 ☕️

在办公室对话时，若突然有人介入且对方谈话出现明显停顿，最好主动起身离开。例如：“我出去打杯水。” 这能为他们创造不便当面谈论某些事情的私人空间。

## 6. 迟复消息时给予解释 📱

微信聊天时，若未能及时回复，再次回复时最好简单说明原因。例如：“刚刚在开会。” 哪怕是一个借口，也能让对方感受更好。

## 7. 避免查户口式聊天 🗣️

直接连续追问对方年龄、工作等信息会让人不适。正确做法是先分享自己的信息进行引导。

> 如果你需要聊天指导，可以参阅相关聊天解析案例。

以下是有效聊天的关键点：
*   先分享自己的信息。
*   用引导代替追问。
*   避免连续提问私人问题。

## 8. 陈述事实而非结论 ⚖️

与对方交流时，多描述客观事实，而非直接下结论。

**公式：有效沟通 = 陈述事实 > 指责结论**

例如：
*   **事实**：“你今天迟到了半小时。”
*   **结论**：“你总是迟到，心里根本没有我。”

记住这一点，可以避免生活中大半的争吵。

## 9. 避免命令式口吻 🗨️

与任何人沟通都应避免使用命令语气。不要使用“你必须……”这类句式。建议以“麻烦”开头，以“谢谢”结尾。

**示例代码（错误 vs 正确）：**
```python
# 错误示范
command = "你必须按我说的做。"

# 正确示范
request = "麻烦你帮我点个赞，谢谢。"
```

## 10. 体察并代述对方不便 🤔

能够察觉对方的不便，并帮他说出不好意思开口的话。

以下是几种常见场景的应对：
*   “聚餐不用选这么贵的餐厅。”
*   “地点太远了，有没有近些的地方？”
*   “很晚了，改天再去唱K吧。”
