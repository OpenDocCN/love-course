# 恋爱教程 01：理解“感觉”的本质与创造情绪波动 💖

在本节课中，我们将要学习一个核心问题：当女生说“对你没感觉”时，这个“感觉”究竟是什么，以及如何通过创造情绪波动来让她对你产生感觉。

## 什么是“感觉”？🤔

面对男人的表白，女人最常使用的拒绝方式就是：“对不起，我对你没感觉，我们不合适。”问题来了，什么是感觉？这个问题困扰了很多男人。明明做了很多对她好的事，她却并不领情，最终只得到一句“对不起，我对你没感觉，我们不合适”。

这个问题确实令人困扰。记得几年前，我陪女朋友看了一部电视剧《咱们结婚吧》。剧中的女主角杨桃（高圆圆饰演）是一位大龄剩女，她母亲为她安排了无数次相亲，对方条件都很好，但全部被她拒绝了。每一次相亲后，杨桃的母亲都会问：“给你安排的对象条件这么好，你怎么就看不上？”杨桃的回答几乎都是：“没感觉。”后来她母亲也无奈地说：“别再跟我提‘感觉’两个字，听太多了，恶心。”

事实也是如此。很多女人如果对一个男人没感觉，即使这个男人条件不错，也看不上。我们看综艺节目《非诚勿扰》，经常会有一些条件很好的男嘉宾，最后却遭遇全部灭灯。他们有的薪水很高，外表也不错，甚至是上市公司的高管。

## “感觉”从何而来？💘

那么，为什么“没感觉”？到底什么是“感觉”？我现在反问一下你：你为什么会对一个女人有感觉？漂亮的女人到处都是，为什么偏偏对她感兴趣？

其实原因很简单：**因为情绪**。虽然逛街时到处都能看到美女，但只有那些给你带来过**情绪波动**的女人，才会让你真正动心。高圆圆是很多男人心目中的女神，但你不会因为她不喜欢你，就茶饭不思、愁眉苦脸。你可能还会祝福她有一段好姻缘。

如果一个女人长得不错，而且经常对你微笑，会跟你聊天，你是不是觉得你们有可能在一起？看到了和漂亮妹子在一起的希望，你自然会心情愉悦。后来，她突然对你不再热情，甚至经常和其他男人有说有笑，你是不是会紧张？心里会想：“发生了什么事情？我做错了什么吗？”她突然冷落你，甚至可能和其他男人关系亲密，你肯定开心不起来。

就是这样的一喜一忧，让你产生了情绪波动。你会不停地想她，为她焦虑不安、睡不着觉。慢慢地，你的脑子里都是她。这个时候，你已经无法自拔地喜欢上她了。这就是所谓的“感觉”——它是通过**情绪波动**产生的。

## 如何让她对你有感觉？🎯

好了，现在你知道自己是怎样喜欢上一个女生的。那么换过来，你也应该知道，为什么女生对你“没感觉”？**因为你没有给她带来情绪波动**。你只知道对她好，却不知道如何让她在喜怒哀乐中体验不一样的情绪。

女人对你产生感觉，是需要通过不同的情绪变化的：温馨、刺激、浪漫、紧张、苦恼、惊喜、担心等等。这样才会让她对你产生感觉。

以《咱们结婚吧》的男主角果然为例。和与杨桃相过亲的男人相比，他并没有什么优势，一开始还跟杨桃闹翻了，关系非常紧张，几乎是“仇人相见，分外眼红”。但后来他们居然在一起了。因为杨桃一开始非常痛恨他，后来发现果然有默默地帮她搞定一个拍摄项目的事情，突然觉得这个男人也不是太坏。这样一来，从**愤怒**到**好奇**，然后开始默默**关注**这个男人，再到有**一点好感**，这个过程都是有情绪变化的。其实整个就是一个比较长的“推拉”过程，一推一拉，最后他们变成了一对欢喜冤家，走到了一起。
