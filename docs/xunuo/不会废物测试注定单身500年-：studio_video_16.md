# 聊天基础课 01：为何死缠烂打注定失败？🤔

在本节课中，我们将要学习一个在追求异性时至关重要的核心观念：为什么“死缠烂打”的策略不仅无效，反而会严重损害你的吸引力。我们将通过简单的类比和逻辑分析，帮助你理解吸引的本质。

## 死缠烂打的负面效果

上一节我们提到了课程主题，本节中我们来看看“死缠烂打”具体会带来哪些问题。

## 吸引的本质：感觉与情绪

追求爱情是建立在吸引和感觉的基础上的。如果女生对你没有感觉，试图通过努力讨好和过分纠缠只会产生相反的效果。

以下是两个生活中的类比，帮助你理解这种感受：

*   **推销员**：购物时，推销员一直推荐你买一些你不想买的东西，拉着你不让你走，你会有什么感受？

和女生相处也是一样的道理。感觉和喜欢是一种情绪感受，它是在价值出现的时候，自然而然地触发的。通过纠缠讨好只能改变一个**逻辑**，而不是她的**感觉**。而人类做决定时，绝大多数情况都是跟随自己情绪的。

## 情绪如何驱动行为

为了说明情绪的主导力量，我们可以看两个自身的例子：

*   **学习目标**：当你下定决心要努力学习考清华北大时，结果凌晨被闹钟吵醒，你的身体和情绪都不想起床，最终放弃了逻辑目标。
*   **健身目标**：当你下定决心想要运动健身、吃素保持身体健康时，却被烧烤的香味所诱惑，难以自持，最终也放弃了目标逻辑。

如果你自己都无法控制自己的情绪，那么如何期望女生能够做到呢？毕竟女生本身就是情绪化的动物。因此，试图通过死缠烂打来控制女生的心，就像试图用手掌握住流沙：**越用力，它就流逝得越快**。

如果你想改变女生对你的感觉，你需要从价值和情绪方面入手。正确追求女生的核心之一是展示自己的价值。

价值是相对的。对于一个饥饿的人来说，一个馒头就是价值。对于一个富有的人来说，馒头并不是什么价值。这涉及到一个匹配的法则。

价值是多方面的。你的运动能力、烹饪技能是一种价值。你的幽默风趣、博学多才也是一种价值。
