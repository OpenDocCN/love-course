# 情商入门课 01：揭秘高情商的核心——给予价值 💡

在本节课中，我们将要学习高情商的本质，并掌握一个核心的社交方法论。我们将通过具体例子，理解如何在人际交往中运用这一原则，特别是在与异性互动时，避免常见错误，建立更顺畅、更有吸引力的沟通。

## 什么是高情商？🤔

一个懂社交、有情商的人往往受人欢迎，生活与感情都能顺利发展。一个低情商的人则容易在生活和感情中屡屡受挫。本节我们将探讨高情商的本质。

## 高情商的核心方法论：给予价值 🎁

上一节我们介绍了情商的重要性，本节中我们来看看高情商的核心行为模式。任何高情商的行为，本质上都是在**给予价值**。

所有高情商的人都懂得在与人相处中要给予价值。具体而言，**给予价值**是指任何让对方受益的行为。价值的形式多种多样，包括：
*   **知识**
*   **人脉**
*   **名气**
*   **地位**
*   **好的情绪**

人本能地会被价值吸引，并对索取价值者产生本能的排斥。

## 高低情商行为对比 ⚖️

理解了核心概念后，我们通过具体场景来看看高低情商的区别。

假设一个朋友分享他刚花150万买了新房的好消息，而你知道类似房子市场价约100万。

以下是不同情商水平的回应方式：
*   **低情商回应**：直接指出“你买贵了，某某只花了100万就买到了。”
*   **高情商回应**：不会指出价格问题，而是给予对方肯定和赞美，分享对方的喜悦。

对方买房已是既定事实。在无法改变事实的情况下，给予对方情绪上的肯定与喜悦分享，是你能提供的最好价值，而非泼冷水。

同样的逻辑也适用于其他场景。例如，朋友让你猜他新衣服的价格，高情商的做法是往高了猜，让对方感到自己“淘到宝了”，而不是往低了猜。

所谓高情商，就是能够站在对方角度，思考对方想要或需要什么（即共情能力），然后**给予价值**。

## 聊天中的常见错误：索取价值 ❌

上一节我们看到了正确给予价值的例子，本节我们来看看聊天中一种典型的错误模式。

为什么女生讨厌男生“查户口式”聊天？因为“查户口”本质是一种**索取价值**的行为。

你问女生一连串问题，即使她回答了，结果也只是你获得了想要的信息。从她的角度看，她没有获得任何新信息，也没有对你增加了解，这是一场单向的、索取的互动。

很多男生只会提问，是因为不会聊天。此外，在你自身价值不高的情况下，对方没有义务回答你的问题。连续提问只会让你看起来像一个不断索取价值的“乞丐”，令人厌烦。

尤其像“你在哪里？”“在干嘛？”这类问题，带有很强的索取意味。有情商的男生聊天，一定是**分享自我多于提问对方**。

## 如何实践“给予价值”式聊天 💬

认识到错误后，我们来看看如何将“给予价值”的思路应用到实际聊天中。

如果你向女生提了很多问题，至少可以在她回答后，对她的回复做出评价、表达你的观点，或分享自己相关的故事。因为你的**观点、价值观和经历也是一种价值**，能帮助她更了解你。

如果你不知道聊什么，可以分享自己生活中关于吃、喝、玩、乐的有趣内容。有时，一张照片胜过千言万语。

如果你的生活乏味、无可分享，那么这或许是你需要做出改变的信号。

## 总结 📝

本节课中，我们一起学习了高情商的核心理念。我们认识到，真正的高情商并非技巧堆砌，其本质在于**给予价值**。这要求我们具备共情能力，能站在对方角度思考其需求，并通过分享知识、人脉、积极情绪或自身经历等方式提供价值。同时，我们剖析了“查户口”式聊天令人反感的根源在于其**索取价值**的本质，并学习了如何通过分享自我、提供互动价值来建立更健康、更有吸引力的沟通模式。记住，吸引源于价值提供，而非信息索取。
