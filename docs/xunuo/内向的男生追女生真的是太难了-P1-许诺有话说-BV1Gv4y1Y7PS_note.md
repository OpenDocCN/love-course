# 恋爱指南 第1课：克服内向，迈出社交第一步 💪

在本节课中，我们将要学习内向性格在社交和恋爱中面临的挑战，并探讨如何通过具体的步骤和方法来克服社交障碍，从而更自信地与异性交往。

昨天，一位非常内向的粉丝与我交流。他今年27岁，今年见过好几位女生，但每次见面都很快结束。因为他会控制不住地紧张，不知道说什么，气氛特别尴尬。等到下次再见女生时，因为害怕自己表现不好，他变得更加紧张。他心里明知没什么，但就是会莫名其妙地紧张，无法克服。

最近，他父亲生病刚出院。而且，他家里今年养了100多头猪，结果高价买了猪苗，等猪出栏时，生猪价格跌到谷底。白折腾一年，活白干了，还赔了二十几万，把自己买房的首付款赔掉了。整个人都很沮丧。

我安慰他并聊了很多。我认为他是比较严重的内向，到了社交障碍的程度。我告诉他，这跟他小时候的家庭环境有很大关系。

许多家长在孩子小时候，经常对孩子说：“我们家里穷没钱，你要乖一点，别在外面给我惹事。”他们一般也很懂事，喜欢什么东西不会轻易说出来，因为知道父母赚钱不容易。所以，他们的很多想法和欲望是压抑的。长大后生活改变了，但这个思想还一直停留着，很难去突破。

**核心问题可以总结为两个公式：**
1.  **压抑的成长环境** + **未更新的思维模式** = **持续的社交障碍**
2.  **表达欲受压制** + **不懂拒绝** = **人际交往中的被动与压力**

第一个问题，压制了自己的表达欲，不敢追求自己真正想要的东西。第二个问题，跟别人相处不会说“不”，有不同意见不会说出来。别人让你说什么、做什么事情，尽管自己很不愿意，他还是会接受。

我说完这些，他的眼眶有点湿。我说，你要突破自己的社交，无论对以后的工作，还是找对象都很重要。你现在不需要别人为你承担，更不需要父母照顾你。你不用有压力，大胆地去突破自己。你没有什么好失去的，也不会因此掉一块肉。克服这个局面就要大胆地去社交，大胆地去表达锻炼自己。说白了，他就是锻炼太少了。

但对他来说，在生活中迈出那一步并不容易。如果在社交软件上不会聊天，我这里有30多套和女生从陌生聊成女朋友的完整聊天记录解析。想看的小伙伴可以私信，或在评论区扣1，我就会发给你。记得用你的点赞、关注和三连来和我交换。

对于这种情况，我们应该如何去改变？我给大家一点建议，昨天我也是这么跟他说的。

以下是具体的行动建议：

你可以找一下你们所在城市的桌游店，玩玩“狼人杀”或者是“剧本杀”。因为这样的游戏玩的人数比较多，经常需要拼桌。你可以打电话给老板问一下，说自己是一个人，可不可以拼桌。在这个环境里都是陌生人，你也不用担心自己做得不好会对你有什么影响，大不了下回不去这家店了。

这样的游戏还有一个好处，每个人都必须要发言，每个人都有表现的机会。你要的就是表现自己，适应那种表达自己想法的、不自在的感觉。一开始不适应的时候，你可以少说点，慢慢地一点一点地锻炼。玩个七八次之后，你自然就好了。下次你跟女生见面时，自然也没那么紧张了。

第一次跟女生约会，千万别去吃饭、喝茶、看电影这些没有互动性的事情。这种约会形式完全依靠聊天，压力很大。

等你桌游玩熟了之后，你可以带女生去玩桌游。大家都在做游戏，自然没有必须聊天的压力了。不说话的时候也不会无聊，该说的时候说两句就好。你们主要是在玩游戏，聊天是次要的。在游戏的过程中，你们也能够很好地了解彼此，同样达到聊天的效果。

可以多参加一些社团活动，也有同样效果。这些社团也会组织各种各样的社交活动。

---

所以，对于他来说，想要追女生，就有两个方面的事情要做。

以下是需要完成的两个核心任务：

第一，学习如何跟女生交往的一些知识和内容。
第二，突破自己的社交障碍。

这跟演讲一样。你已经把稿子背得滚瓜烂熟了，知道自己应该做什么，但是一上台一紧张，一样做不好。

我发现很多人在追女生这件事情上屡屡受挫，一般都跟他差不多，有两个方面的问题。

以下是导致恋爱受挫的两个常见原因：

一个问题是自身的“bug”。因为“bug”导致你屡屡失败，导致你没有什么恋爱经历，在爱情上是个“小白”，没有积累下来的经验和教训，所以总是会犯一些低级的错误。
当你学习了一些知识和道理后，明白过来之后，你还要发现自己身上的那些“bug”，一点一点地把自己的“bug”给解了。
