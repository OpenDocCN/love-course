## 🤝 第一点：初次加好友如何开场

上一节我们介绍了课程概述，本节中我们来看看具体如何与相亲对象开启第一次对话。

和相亲的女生刚加好友时，开场方式很重要。首先不要上来就套近乎。很多男生以为这样可以拉近关系。但在你们刚认识的阶段，这样做只会让女生觉得你很轻浮。

其实我们只需要简单的自我介绍就可以了。如果女生对你并不反感，她一定会回应你。接下来，你就可以围绕你们都认识的那个中间人开启话题。这一点就不展开说了。

我们重点说一下女生的回应比较冷淡的情况。如果她不回你，或者只回“嗯嗯”、“好的”，这个时候请你一定不要灰心。也许她并不是针对你，她只是抵触相亲这件事情。

这里分享给大家一个共情的思路。我们可以这样和女生说：“你也是被家人逼着来相亲的吗？看来咱们俩真的是同病相怜呀。”之后你们的聊天便会在一个轻松的氛围下展开。

以下是相亲聊天开场阶段的核心要点：
*   **避免轻浮**：不要过早套近乎。
*   **简洁开场**：从简单的自我介绍开始。
*   **利用纽带**：从共同认识的中介人切入话题。
*   **处理冷淡**：若对方回应冷淡，尝试使用共情话术化解抵触情绪。

相亲聊天最重要的就是不要直接把对方定位成女朋友，那样只会给女生带来压力。拥有一个良好的心态才能和女生有一个美好的开始。

## 🔑 第二点：如何建立轻松联系

在学会了如何开场之后，我们来看看如何进一步推进关系，化解相亲中的尴尬。

其实相亲对于女生来说多少都会有点抵触，因为她们会感觉到紧张、害怕、尴尬。想要巧妙地化解尴尬，大家可以记住一个万能的公式：

**理解对方 + 降低预期 + 表达可能**

举一个例子：“我很理解你，我也是被家里抓来相亲的。没关系，咱们慢慢聊，没准以后还能成为好朋友。我们可以加个好友，以后需要配合的时候我们互相打掩护。”

这样说不仅打消了女生对相亲的反感，还让气氛变得轻松了很多。如果进展顺利的话，本次见面后女生对你的印象会非常好，之后也会和你保持联系。

关于这一类技巧，核心就是说女生爱听的话，说女生没有压力的话，说女生能接得住的话。很多兄弟和女生聊不下去，就是因为聊天的内容让女生不知道回什么，要么就直接暴露需求，让女生感觉压力很大，或者让女生觉得很无聊，直接没兴趣和你聊天。

最后我们延伸说一下追女生最核心的点。其实就一句话：**追一个女生不是疯狂地告诉她你有多喜欢她，而是要告诉她“我的世界很有趣，你要不要进来坐坐”。**

你要学会把你的星辰和大海尽可能地展示给她，让她对你有所期待。那该怎么展示呢？

以下是两个展示个人世界的例子：
*   **在聊天中展示**：你们聊天的时候可以说：“先不聊了，我该练琴了，一会录个小样给你听。”

这样就引发了她的探索欲，她想要了解你的那种兴趣。另外，朋友圈也是一个很好的向对方展示生活的途径。你的厨艺、你的个人爱好，有利于对方更加地了解你。让他觉得你的世界很精彩的时候，他才有兴趣来探索你。
