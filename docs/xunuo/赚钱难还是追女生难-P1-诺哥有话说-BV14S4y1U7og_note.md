## 行为经济学验证：古玩市场讲价模型

为了更清晰地理解这个心理，我们可以引入一个简单的行为经济学模型。

想象你在古玩市场看中一件商品。
*   **场景A**：老板开价3000元。你砍价到300元，经过一番博弈，最终以600元成交。你会觉得自己**赚了**。
*   **场景B**：老板开价3000元。你试探问300元行吗，老板毫不犹豫地说“可以”。此时你会想：“他答应得太痛快了，这东西可能只值100元，我**赔了**。”

这个模型可以简化为一个决策公式：
**`买家感知价值 = 初始标价 - 卖家让步的轻易程度`**

当卖家（类比追求者）过快、过大地让步（即“舔”），买家（类比被追求者）感知到的商品（类比追求者本身）价值就会急剧下降。
