## 什么是高情商？🤔

一个懂社交、有情商的人往往受人欢迎。一个低情商的人则容易在生活和感情中屡屡受挫。如今，“情商”这个词已经非常普遍，书店里充斥着相关书籍，网上也遍布着教授情商的课程。

然而，大多数内容陈词滥调，实用性很低。本节中，我们将从社交动态的角度，揭开情商的本质，并分享一个实用的方法论。

## 核心方法论：给予价值 💎

任何高情商的行为，本质上都是在**给予价值**。所有高情商的人都懂得，在与人相处中要给予对方价值。

**什么是“价值”？**
价值的形式多种多样，包括：
*   **物质价值**：例如金钱、礼物。
*   **非物质价值**：例如知识、人脉、名气、地位。
*   **情绪价值**：例如肯定、赞美、快乐、共鸣。

任何能让对方受益的东西都是价值。人会本能地被**给予价值者**吸引，而对**索取价值者**本能地排斥。

## 高情商 vs 低情商：实例对比 ⚖️

上一节我们介绍了“给予价值”的核心概念，本节中我们来看看它在实际场景中的应用。

以下是两个常见的生活实例：

**实例一：朋友买了新房**
*   **低情商做法**：直接指出对方买贵了。“你买贵了，某某只花了100万。”
*   **高情商做法**：给予对方情绪上的肯定和赞美，分享对方的喜悦。因为既定事实无法改变，给予积极的情绪价值是更好的选择。

**实例二：朋友穿了新衣服让你猜价格**
*   **低情商做法**：往低了猜，让对方觉得自己的选择不值。
*   **高情商做法**：往高了猜，让对方感觉自己“淘到宝了”，获得愉悦感。

所谓高情商，就是能够站在对方的角度，思考对方想要什么、需要什么（即**共情能力**），然后**给予相应的价值**。

## 聊天中的常见错误：索取价值 🚫

理解了“给予价值”的原则，我们就能明白为什么有些聊天方式会让女生反感。

女生通常讨厌男生“查户口式”的聊天。因为这种行为本质上是**索取价值**：你提出一连串问题，获得了你想要的信息，但对方在这个过程中没有得到任何价值（新信息、新了解或好的感受）。这是一场单向的、索取的互动。

尤其在自身价值不高的情况下，连续的提问只会让你看起来像一个不断索取价值的“乞丐”，令人厌烦。例如频繁询问“你在哪里？”、“你在干嘛？”，这些话带有很强的索取意味。

## 高情商聊天法：分享与给予 ✨

那么，如何正确聊天呢？核心思路依然是**给予价值**。

以下是基于此思路的聊天方法要点：

1.  **多分享，少提问**：有情商的人聊天，一定是分享自我多于提问对方。分享你的生活、观点、经历和故事。
2.  **回应时提供附加值**：即使女生先提问，你在回答后，也可以对她的回复做出评价、表达你的观点，或分享与之相关的自身经历。你的观点和经历本身就是一种价值，能帮助对方更了解你。
3.  **用生活内容开启话题**：如果你不知道聊什么，可以分享生活中好玩、好吃的东西。一张有趣的照片有时胜过千言万语。
4.  **丰富自身生活**：如果你的生活很无聊，没有东西可以分享，那么首先应该做出改变，丰富自己的经历和价值。
