## 一位“老司机”的成长之路 🛣️

上一节我们提到了“不要脸”的核心概念，本节中我们来看看一位朋友是如何具体实践并取得成功的。他的成长主要沿着两条主线展开。

以下是他的两条核心行动线：

1.  **第一条线：混圈子交朋友**
    *   起初他性格内向，甚至有些社交恐惧。
    *   他强迫自己进行社交，参加各种活动。
    *   不到两个月，通过圈子介绍圈子，他结识了大量的异性与同性朋友。

2.  **第二条线：习惯性搭讪**
    *   关键在于“习惯性”。开始时他同样恐惧，担心被拒绝、被嘲笑、不知如何开口。
    *   他最怕的是不敢迈出第一步。但迈出之后，发现事情并没有想象中可怕。
    *   不到三天，他找到了搭讪的乐趣，变得在任何场合看到心仪对象都能直接上前认识。
    *   他还会进行强化训练，例如曾有一个月，他强制自己每晚下班后完成**20次**街边搭讪才回家。
    *   这样坚持不到一年，他便成为了经验丰富的人。

## 心态转变：从“要脸”到“不要脸”再到“要脸” 🔄

很多人天生内向、在意他人看法，容易在关键时刻退缩。但你必须明白一个事实：**你在多数人的人生中只是一个过客**，很多事情大家一笑而过，两个月后没人记得你做过什么。

例如搭讪时，你害怕尴尬、拒绝和丢脸。但丢脸又如何？一小时后谁还记得你？年轻时若做不到“不要脸”，年纪更大、更爱惜羽毛时，就更加难以做到了。

## 行动指南：脱单的具体步骤 📝

很多人将单身归咎于长相和财富，并因此拖延，最终陷入被动。找对象需要尽早培养。年轻时，感情因素占比更大；年纪增长后，现实考量会增多。

外形需要打理，金钱需要赚取，但这些都是长期工作。就脱单而言，你只需要做到：

1.  将外形打理到短期内你能达到的**最佳状态**。
2.  在养活自己的基础上，准备一些**约会的闲钱**。
3.  剩下的，就交给“不要脸”去执行：**去搭讪、参加活动、结交异性朋友**。被拒绝无需在意，一笑而过即可。

以下是心态随经验增长的三个阶段：

*   **接触超过30个姑娘**：面对异性时，紧张感开始消失。
*   **接触超过50个姑娘**：不再患得患失，言行举止更加收放自如。
*   **接触超过100个姑娘**：你将成为经验丰富者，能大致预判女生的常规反应。

当你通过大量实践成为经验丰富者后，任务并未结束。

此时，你需要**把丢出去的脸捡回来**。找回最初那份含蓄、容易脸红的样子，收起作为“老油条”的江湖气。将你的身份从主动出击的“猎人”，转变回富有吸引力的“猎物”。

到了这个阶段，你就真正毕业了。
