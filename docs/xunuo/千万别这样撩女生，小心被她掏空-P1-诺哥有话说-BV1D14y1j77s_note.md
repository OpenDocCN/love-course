上一节我们介绍了主动提供谈资的方法，本节中我们来看看其背后的核心原理——“反撇”。

**反撇**，指的是一个人的表现与外界对其的固有印象形成强烈反差，这种“出乎意料但又在情理之中”的感觉，能极大提升个人吸引力。

公式可以简单理解为：**吸引力 ∝ （实际表现 - 固有预期）**

以下是几个例子：

*   一位平日看起来不靠谱的同事，在某次会议上提出了极具建设性的意见。
*   一位聚会时从不唱歌的朋友，偶然被发现唱得非常好听。

这种反差就是“反撇”。它之所以有效，是因为我们很难成为女生遇到的第一个追求者。在她过去的经验里，多数男生会迎合她的兴趣聊天。

从你们相识开始，她思维上就有一种惯性预期，认为你也会像大多数男生一样。这是“情理之中”。

而当你没有一味迎合，反而自信地聊起自己精通且能展现价值的领域时，就创造了“意料之外”。这种反差会让她觉得你与众不同。

打个比方：吃完糖再吃水果，不会觉得水果有多甜；但喝完苦咖啡再吃糖，就会觉得糖格外甜。“反撇”就起到了类似“苦咖啡”的衬托作用。
