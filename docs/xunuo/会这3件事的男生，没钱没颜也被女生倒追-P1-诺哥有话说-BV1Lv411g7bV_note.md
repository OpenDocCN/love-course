## 1. 听不懂话里有话 🎭

上一节我们概述了课程内容，本节中我们来看看第一种低情商行为：听不懂对方的言外之意。

女生有时会委婉表达，以照顾对方感受。但若男生无法领会潜台词，坚持直来直往，往往会导致尴尬。

**核心问题**：未能解析对方话语中的**真实意图**。

以下是典型案例：

*   **女生说**：“我妈说她最近忙，咱俩呢才刚认识，过一阵你再去问候她们。”

**分析**：女生的潜台词可能是认为关系未到见家长的程度，或想婉拒邀请。男生的回复忽略了这层含义，只理解了字面意思。

## 2. 没有边界感 🚧

理解了言外之意的重要性后，我们来看看另一种更直接冒犯他人的行为：缺乏边界感。

没有边界感不仅体现在行为上过分亲近，也体现在言语上贸然闯入私人领域。

**核心问题**：在关系未达到相应程度时，过度探询**私人信息**或做出**越界行为**。

以下是典型案例：

*   **男生问**：“你平时工作忙不忙呀？”

*   **男生说**：“这么忙呀，气质也挺好的，一个月能挣个三五万不？”
*   **女生回复**：“...”

**分析**：询问收入属于敏感话题。这种问题会让对方陷入两难：回答具体数字可能显得炫耀或引发后续麻烦（如被借钱），不回答又显得尴尬。

## 3. 不询问就给出建议 💡

前面我们讨论了沟通中的理解和边界问题，最后一种行为则关乎如何有效提供支持：即在未了解全貌时就匆忙给出建议。

很多人会犯这种错误，本意是出于好心，但可能忽略了对方的真实情绪和需求。

**核心问题**：在未充分理解对方**处境**和**情绪**时，就提供**解决方案**。

以下是典型案例：

*   **女生说**：“我今天就辞职，老板太气人了，再也不伺候她了。”

**分析**：女生此刻可能更需要情绪共鸣和理解，而非实际建议。男生的直接否定和说教，可能让她感觉不被理解，仿佛被“推进火坑”。
