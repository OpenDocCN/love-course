# 聊天技巧入门课（一）：三种核心聊天思维 🗣️

在本节课中，我们将要学习三种核心的聊天思维方式。这些方法能帮助你理解对话背后的逻辑，从而更自然、更有效地与他人交流，特别是在与异性聊天时。

## 一、诱导式聊天法 🤫

上一节我们介绍了课程概述，本节中我们来看看第一种方法：诱导式聊天法。其核心是学会倾听，将对话的主动权让给对方，并根据对方的情绪牵引话题。

说一句很直白的话：想要学会如何好好聊天，你首先要学会闭嘴。很多人，尤其是男生，对聊天有一个深深的误解。他们认为自己无所不知，又能侃侃而谈，那么和女生聊几天一定不成问题。结果，一面对和女生聊天，就成为了让人厌烦的话痨。你扯东扯西聊了一通，碰到脾气好的女生，会附和一个表情包或者语气词。碰到脾气不好的，直接甩给你一个微笑，或者干脆不聊。

为什么会这样？因为你在聊天时的“爽”，全部建立在女生的不感兴趣之上。你根本不知道女生想聊什么，又抓不住女生的情绪点。在女生眼里，你就是在满嘴跑火车。对于这种思维跳跃的男生来说，你不断地向对方抛出让他们无感的话题，对方就要动脑子，费很多脑细胞去回答你。如果对方在根本对你没多大兴趣的前提下，这种“大聊”的方式，会在心理上引起对方极大的不舒适感。

因此，诱导式聊天法是让你在聊天当中当一个倾听者，将聊天的主动权让给女生，并且根据女生的情绪，适当地牵引话题。让女生在与你聊天时，既不尴尬，又能感觉到你有趣。

以下是几个诱导式聊天的例子：

*   **例子一**：
    *   男生说：嗨，掐指一算你是南方姑娘。
    *   女生回：你怎么知道呢？
    *   男生说：因为你的朋友圈很感性呀。南方姑娘们的心思，都是很柔软细腻的。

*   **例子二**：
    *   男生说：嗨，好久没有在食堂看到你了。
    *   女生回：哎，我最近没什么胃口。
    *   男生说：最近减肥啊，还是遇到啥事了？干饭这么有盼头的事情，都没兴趣了。

*   **例子三**：
    *   男生说：我最近养了只猫，有些事情想请教你一下。
    *   女生回：你怎么知道我也养猫呢？
    *   男生说：看你办公桌上都是小猫的挂饰，就感觉你一定是个职业铲屎官选手。

一定要记住，从女生的信息点中提取她有欲望聊下去的话题，同时可以合理的进行猜测，去诱导出对方更多的一些信息点。这时候，无论你猜的对与否都不重要了，因为你的开场白已经诱导出了对方的聊天欲望。你猜对了，她会顺着你聊下去；你说错了，她也会笑着和你解释原因。

聊天不能由你尝试带领节奏，反而可以通过一个其他轻松的话题，来侧面诱导出答案。这样不会暴露你强烈的聊天目的性，也能激起对方与你聊下去的欲望。

## 二、需求感捕捉法 🎯

了解了如何开启话题后，我们来看看如何让对话持续下去。本节介绍需求感捕捉法，其核心是识别并满足对方在对话中传递的情绪需求。

什么样的人能够让我们产生持久的聊天意愿？在聊天社交中，其实都是双向需求感的传递过程。我们之所以愿意和一个懂自己的人聊天，不仅仅是因为聊起来很舒适，而是这个人能够满足自身的情绪需求。

很多时候当我们在社交中说出自己的每一句话，尽管我们没有注意，但是我们每一句带有信息点的话，其实是在向社交对象传递自身的需求感。

比如当一个女生对你说：“你真讨厌，那个某某某同事今天又在打我小报告。”这个时候你要怎么回复？第一反应是去安慰她的小伙伴，在弹幕上打个起立。女生在向你抱怨的时候，其实她的需求来自于，想要找一个能与自己站在统一阵营的人。她需要自己的情绪得到安抚，同样也需要自己的需求能在你的身上得到满足。

所以你可以这样回复她：“这就有点过分了，等我晚上回去做个小人，明天咱们一起研究一下怎么扎她。”

再比如说一个女生对你说：“今天天太热了，我特意穿了一件白衣服。”她向你传递的真正需求并不是天热，而是在向你传递自己渴望得到来自于你的关注感。所以你可以这样回她：“你真是个小机灵鬼。你今天那件衣服我注意到了，我姐也有一件差不多的衣服，但是没你穿的好看。”

因此和女生聊天的时候，别总是习惯用自己的惯性思维去回复女生，而是多在心里问自己几个为什么：为什么她会在这个时候说这个话题？为什么她现在突然话风变了？多去站在女生的思维去思考这个说话的动机，很多的聊天话题也就迎刃而解。

## 三、逆向合理化法 🔄

最后，我们来学习如何通过聊天深入对方的内心世界。本节介绍逆向合理化法，其核心是利用人的双面性，从对方的外在表现预测其内在需求。

当女生感觉到“这个男生好懂我呀，和他聊天好走心呀”的时候，那么你的聊天就是能让女生特别心动的。但是毕竟不是所有人都会读心术，那怎么才能准确的预知到一个女孩子的心思呢？

人在潜意识中希望自己是一个完整的情感需求的个体。所以当一个人内心越缺少什么，他就越容易反向的外化出什么。因此，教你一个简单易懂、具有普遍性的逆向合理化预测方式：

以下是几种常见的逆向合理化预测：

*   如果她是一个外向热情的女生，那么你就“赌”她内心软弱、患得患失。
*   如果她是一个内向腼腆的女生，那么你就“赌”她渴望被关注、被赞扬、被发现。
*   如果她是一个人缘超好、会和人打交道的女生，那么你就“赌”她“每每照顾别人情绪很累吧”。
*   如果她是一个性格文静、社交狭小的女生，那么你就“赌”她其实也希望谈一次轰轰烈烈的恋爱。

人其实都是一种双面性很强的生物，自己的正面展示给外界，但是反面仍有一个不为人知的小世界。这样的逆向合理化预测，基本可以让你十拿九稳的读懂一个女生。即使你预测错了，对方也不会怪罪你，反而觉得你是一个心思细腻且有思想的人。

你为什么和异性聊天总是停留在表面，对方的态度也是不温不火？是因为你聊的话题也好，做出的反应也罢，仅仅是停留在普通朋友的社交层面。这个女生一直都对你保持着一种界限分明的态度，所以你才会感觉到女生对你还有隔阂感，彼此的关系很难推进下去。

因此，逆向合理化法其实是通过聊天的方式，打开女生心门的一把钥匙。其实每个人都在极力地掩饰那个真正的自己，同时呢，又在矛盾中渴望被一个人关注到内心的脆弱。

## 总结 📝

本节课中我们一起学习了三种核心的聊天思维：**诱导式聊天法**（学会倾听与牵引）、**需求感捕捉法**（识别并满足情绪需求）以及**逆向合理化法**（利用双面性预测内心）。当你把以上三条深刻的内化为己所用，那么你又何患聊天无词呢？
