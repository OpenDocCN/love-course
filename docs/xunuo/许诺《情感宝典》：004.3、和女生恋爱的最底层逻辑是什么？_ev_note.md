# 情感宝典 004.3：恋爱最底层逻辑揭秘 🔍

在本节课中，我们将探讨与女生恋爱的最底层逻辑。我们将从进化心理学的角度出发，分析现代恋爱观念背后的原始驱动力，并解释女性行为与择偶观的深层原因。

---

恋爱是一个复杂的观念，其结果受多种因素影响。有些人喜欢外貌出众的，有些人喜欢经济条件好的，有些人喜欢性格踏实的，有些人喜欢才华横溢的。

有些人虽然对女生很好，但女生并不喜欢他。有些人对女生很差，女生却深爱着他。女生常说的“没有感觉”，确实让许多男生感到困惑。如果问题是具体的，我们知道如何改进。但“感觉”这种抽象概念，我们该如何把握呢？本节课将解开这个谜团，厘清女性最底层的需求及其行为逻辑。

大约在2万年前，地球上大部分人还生活在部落社会。那时的人类与今天的人类基因差异只有千分之一。那时的婴儿若放在今天的社会环境中成长，与今天的人并无区别。现代社会过于复杂，使我们难以看清男女恋爱的本质。因此，我们需要回到部落时代，探究爱情最初的模样。

进化心理学指出，人类的身体适应的是远古环境。我们生活在21世纪，却拥有石器时代的大脑。近2万年来社会发展太快，人类的进化速度远远跟不上社会变化。因此，我们的许多思维模式和潜意识仍停留在部落时代。

以下是几个例子，帮助大家理解这一观点：

许多人喜欢吃甜食，尤其在欧美国家，这导致肥胖及相关疾病问题。据统计，2015年因肥胖导致的死亡人数达到400万。人们可能不知道自己为何喜欢甜食，但进化心理学家找到了原因：在部落时代，甜食能快速补充能量，不喜欢甜食的人容易被淘汰。因此，喜欢甜食成为一种保留的“感觉”。

我曾多次问学员一个问题：你为什么喜欢胸大的女生？最常见的答案是“感觉”，说不出具体原因。研究表明，胸部大小与产奶量并无直接关系。进化心理学给出的解释是：在部落时代，没有文胸，女性胸部大会随年龄增长下垂，而小而挺拔的胸部则代表更佳的繁殖年龄。因此，男性偏好成为了一种潜意识选择。

为什么多数男性喜欢长发的女性？人本身没有改变，但剪短头发后往往不受欢迎。答案同样是“感觉”。女性头发每年生长15到20厘米，拥有一头健康长发需要3到4年且无疾病。在部落时代，没有婚前检查，头发是判断健康的重要标准。

欧美人偏好金发碧眼，因为随着年龄增长，肤色和眼睛颜色会变深。金发碧眼的女性通常处于最佳繁殖年龄。由此可见，我们的“感觉”有最底层的原因，只是在现代社会中，我们难以理解。

那么，两性关系的底层逻辑是什么？女性又是如何考虑的呢？

首先思考一个问题：一个女生想生10个孩子需要多长时间、经历什么？一个男生想生10个孩子又需要多长时间、经历什么？

女性大约需要10年时间，且一直处于怀孕状态，生活不便，缺乏生存能力。抚养众多孩子也需要投入巨大精力与时间。男性理论上只需一个晚上。这种繁殖成本的巨大差异，决定了两性完全不同的策略。

男性倾向于寻找繁殖价值高的女性发生关系，以广泛传播基因。因此，男性的繁殖策略是**多播**。女性因生育成本高，其策略是**选种**。重点在于：**男性的繁殖方式是“多播”，女性的繁殖方式是“选种”**。这也是现代社会中男性显得花心好色的原因。

接下来，我们看看女性在这种繁殖策略下的择偶观。

在部落时代，优秀的男性是猎人——身材强壮、善于狩猎。这样的男性能确保女性在怀孕期间不至于饿死。其次，男性必须靠谱。如果男性只发生关系而不负责，女性及其孩子同样面临生存危机。没有这种择偶观的女性，在进化过程中逐渐被淘汰。因此，保留下来的女性基因本质上是现实和物质的。

本节课的重点是：女性最底层的需求，一是**猎人**（优秀的男性），二是**靠谱**。在现代社会，“猎人”的定义有所变化，可以简单理解为有能力的优秀男性。女性所说的“没感觉”，往往意味着她觉得你不够优秀。

我们来看看那些常被提及的女性喜欢的特质：有钱、乐观、幽默、善良、努力拼搏。这些不都是优秀男性的品质吗？包括常说的高和帅，这些我们将在讲解“吸引”时详细探讨。

本节课中，我们一起学习了恋爱的最底层逻辑。我们从进化心理学的角度，理解了男性“多播”与女性“选种”的繁殖策略差异，并明确了女性最核心的两大需求：优秀（猎人）和靠谱。掌握这些底层逻辑，有助于我们更好地理解两性关系中的各种现象。
