## 二、 核心要素一：强——展现领导力与主见

分析了问题所在后，我们来看看第一个解决方案：“强”。这里的“强”并非指财富或成就，而是在两性关系中展现的领导力。

在上述案例中，我们指导男生改为强势邀约后，女生的回应从“算了”变成了“好，等我半个小时”。对于有选择困难的女生，帮她做出优质选择，能极大提升好感。
