## 二、遇到喜欢的人不要等 ⏳

理解了“感觉”的重要性后，我们来看看阻碍许多男生行动的心理障碍。很多兄弟没有脱单，并非因为自身条件不好，也不是因为没有喜欢的人，而是因为顾虑太多，最终错过了机会。
