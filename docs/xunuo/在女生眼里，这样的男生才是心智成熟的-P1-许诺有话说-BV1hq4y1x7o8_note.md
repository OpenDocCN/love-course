## 一、不攀比，不对比 🚫

上一节我们介绍了课程概述，本节中我们来看看第一个核心特质：不攀比与不对比。许多女性认为，这是心智成熟者最基本的表现。

一个成熟的人应具备**自己独立的生活体系与内心秩序**。其行为逻辑如下：

1.  **不与他人攀比**：不会因不如他人而自卑，也不会因优于他人而自傲。
2.  **不拿身边人对比**：例如，不会总拿自己的伴侣与他人比较，挑剔对方不够温柔或能力不足。

心智成熟的人，其情绪与价值感不易受外界干扰，也不会持续挑剔他人或忽视他人的优点。他们通常是**宽容且上进**的。

不攀比是内在修养，而情绪管理则直接影响外在关系。**不情绪化**是当前女性极为看重的特质，她们视之为感情稳定与安全感的基础。

*   摆脸色

## 三、不失生活情趣 🎨

上一节我们探讨了情绪管理，本节我们来看看成熟与生活情趣的关系。请不要误解，心智成熟并不意味着放弃年轻人的娱乐与活力。

我见过一些人将“成熟”作为优越感的资本，嘲笑年轻情侣的互动为“幼稚”。实则不然。**保持童心、生活热情与好奇心，与心智成熟并不冲突**，这是任何年龄段都应具备的品质。

