## 应当放弃追求的几种情况 🚩

上一节我们介绍了追求期限的概念，本节中我们来看看哪些具体信号表明你应该放弃。以下是几种关键情况，如果遇到，建议直接停止追求。

**一、不回复或回复敷衍**

很多男生会持续不断地给女生发消息，即便对方不回复也一直发。或者女生回复很慢，基本要几个小时或第二天才回复。即便回复，内容也很敷衍，常用“嗯”、“哦”或者表情包。这样的聊天根本无法进行下去。又或者，面对面聊天时对方也持无视态度，你说你的话，他玩他的手机，偶尔抬头应付几句。根本原因是对方对你不感兴趣。碰到这种情况，建议直接放弃。

**二、不会为你付出**

如果你们认识了一段时间，也约会过几次，但每次吃饭、看电影、买零食等都是你在花钱，对方则袖手旁观。这样的关系从一开始就不平衡。即便你能追到手，后期关系也很难维系。谁都不想当舔狗。出现这种情况，往往是因为你不会聊天导致的。

**三、与多名男生保持暧昧关系**

你在追求她的时候，她还在和其他男生保持很亲密的关系，但同时也不拒绝你。这样就趁早放弃吧。她可能只是在养备胎。

**四、公主病**

例如，你必须开车去她家楼下接她，否则她就不出门。这样的行为不仅说明可能有公主病，也表明她在故意给你刁难。

**五、势利**

约她出去吃饭，她只愿意去高档餐厅或高级场所。如果约去普通场所，就用各种理由推脱。这样的女生最好也别追了。令她感兴趣的不是你本人，而是你所能提供的其他价值。
