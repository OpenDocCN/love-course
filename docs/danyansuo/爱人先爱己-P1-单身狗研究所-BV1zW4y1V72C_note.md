# 爱人先爱己 - 课程P1：理解自我关怀 🧘‍♀️

在本节课中，我们将要学习自我关怀的核心概念，并了解如何通过接纳自身的不完美与挑战，来保持前进的动力。自我关怀是个人成长和情感健康的基石。

## 核心概念：自我关怀与前进的动力

上一节我们介绍了课程主题，本节中我们来看看其核心思想。保持前进的关键，在于**相信未来会变好**，并**期待再次感受美好的时刻**。这种信念可以表述为一种心理动力公式：

**前进动力 = 对变好的信念 + 对美好未来的期待**

## 勇敢面对挑战的过程

理解了基本动力后，我们来看看实践中的挑战。在前进的道路上，勇敢是必不可少的。但勇敢并不意味着没有困难。

有时，我必须勇敢，而这并不总是容易的。有时我会动摇，会经历糟糕的日子。

## 关于“勇敢”的重要认知

以下是关于“勇敢”本质的一个关键点列表，帮助我们正确看待过程中的挣扎：

*   勇敢并不意味着轻松。
*   对我而言，勇敢是持续前进的关键方式。

## 总结与回顾

本节课中，我们一起学习了自我关怀的初步内涵。我们认识到，通过培养对未来的积极信念并接纳勇敢过程中的艰难与起伏，是支持我们持续向前的核心力量。记住，允许自己有“坏日子”并坚持下去，本身就是一种勇敢。
