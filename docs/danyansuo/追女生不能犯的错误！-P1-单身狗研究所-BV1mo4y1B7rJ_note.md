# 课程：烘焙入门 - 第1课：基础饼干面团制作 🍪

在本节课中，我们将要学习如何制作基础饼干面团。我们将从准备材料开始，逐步讲解搅拌、处理面团的技巧，并了解如何将面团塑形为适合烘烤的小块。掌握这些基础步骤是成功烘焙美味饼干的关键。

---

上一节我们介绍了课程的整体目标，本节中我们来看看制作饼干面团的具体步骤和核心操作。

制作饼干面团的第一步是处理黄油。你需要使用室温下软化好的黄油，这有助于后续的搅拌和混合。

以下是处理黄油和干性材料的步骤列表：
*   将软化黄油与糖放入搅拌碗中。
*   使用搅拌器将黄油和糖充分搅打至颜色变浅、质地蓬松。
*   将面粉、泡打粉等干性材料在另一个碗中混合均匀。

**核心搅拌公式**可以概括为：**软化黄油 + 糖 → 搅打至蓬松 + 混合的干性材料 → 翻拌均匀 = 基础面团**。

---

上一节我们完成了黄油和干性材料的初步混合，本节中我们来看看如何将混合物整合成面团，并处理可能遇到的问题。

当干性材料与黄油混合物初步结合后，会形成一个粗糙的面团。此时，你需要用手或工具将其聚拢成团。

以下是整合和处理面团的具体方法：
*   用手将碗内的所有材料按压、聚合成一个整体面团。
*   如果面团粘手，可以适量撒一些干面粉。
*   将大面团分割成几份，便于后续操作。

---

上一节我们得到了一个完整的面团，本节中我们来看看如何将面团处理成适合烘烤的形状和大小。

面团准备好后，需要将其分割并塑形。塑形的目的是确保饼干在烘烤时受热均匀，成品大小一致。

以下是面团分割与塑形的步骤列表：
*   将大面团在案板上搓成长条状。
*   用手或刀将长条状面团切成大小均匀的小块。
*   如果切出的面团块太大，可以用手指将其稍稍捏小、整理。
*   如果切出的面团块太小，可以用手指将其稍稍捏合、整理。
*   将处理好的小块面团有间隔地摆放在铺有烘焙纸的烤盘上。

---

本节课中我们一起学习了基础饼干面团的完整制作流程。我们从软化黄油和混合干料开始，经历了搅打、整合成团，最后将大面团分割塑形成均匀的小块。记住核心的搅拌公式和处理面团的技巧，你就能为烘烤出完美的饼干打下坚实的基础。
