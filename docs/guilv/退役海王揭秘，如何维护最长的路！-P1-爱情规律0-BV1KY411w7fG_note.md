# 课程 P1：爱情规律 0 - 退役海王揭秘如何维护最长的路 🛤️

在本节课中，我们将要学习如何与女性建立并维护长期稳定的关系。课程将从一个具体场景出发，逐步解析“认真”与“随便”这两种态度在情感互动中的核心作用，并揭示让关系自然深入的关键心态。

## 场景引入：一次搭讪与约会

首先，我们从一个具体场景开始。假设迎面走来一位十分有魅力的女性，我上前进行了搭讪。她起初有些紧张，问道：“这样好吗？” 我只是笑了笑回答：“随便，我觉得挺好。”

搭讪结束后，我给她发送了消息“嗨”，但她没有立刻回复。我的态度是：不回就算了，改天可以再认识其他人。后来她回复了，解释说刚才在洗澡。这是一个积极的信号，于是我们开始了聊天。

大概聊了两三天后，我们进行了第一次约会。约会时她问我待会儿做什么，我其实并没有具体计划，只是想着两个人待在一起就好，便提议去江边吹吹风。虽然天气很冷，但她同意了。吹完风后，我们又一起去吃了火锅。

在江边，听着轻盈的浪花声，我想牵她的手，但她把手放进了口袋。我并没有犹豫，直接把手伸进了她的口袋。她的表情呆滞了五秒，但最终还是把小手交给了我。那天我们就这样牵了一整天的手。

## “随便”是一种关键态度

上一节我们描述了一个具体的互动场景，本节中我们来看看其中蕴含的核心态度：“随便”。

**“随便”是一种态度。很多事情，一旦过于认真，你就输了。**

有些男性在遇到喜欢的姑娘时，对待和她说的每一句话都非常认真，反复思考。如果以这种方式聊天，基本上很难追求成功。因为她会感受到你的“认真”。认真本身没有错，但很多人并不理解什么是真正的认真。相反，“随便”这个词被许多人忽略了，而正是因为“不随便”，成为了大部分人追不到姑娘的原因。

为什么有时抽烟的男人或女人看起来很有魅力？正是因为那种“随便”的感觉。

因此，在一开始，你们必须先搞懂“认真”和“随便”之间的关系。

## 解析“认真”聊天的误区

在理解了态度的重要性后，我们来具体分析“认真”在聊天中可能带来的问题。

以聊天为例，“认真聊天”看上去似乎没有错，是一种认真的态度，感觉就像认真做作业一样。但问题在于，**聊天不是做作业**。你追求妹子，结果却让她感觉她只是你的一个作业本，你只是在解答思考题，那怎么可能成功？

我在朋友圈分享最新追求女生干货时讲过，这种“追女生”的状态本身就已经偏离了初心。因为从某个角度来说，追求女生并没有“追”与“不追”的说法。从相遇、认识到接吻，都应该是一个自然而然的过程。

你的初心应该是享受这个过程，与喜欢的人相处，就这么简单。

## 如何做到真正的“随便”？

认识到“认真”的误区后，本节我们来看看如何实践“随便”的态度，从而让关系自然发展。

真正地去做自己，你才能在自己觉得最舒适的状态里去了解喜欢的女生。你不会纠结，不会需求感爆棚，不会犹豫，一切都干净利落。

一个“随便”的男人在认识一个自己喜欢的女生后，他能做到“认识跟没认识差不多”。他本身的生活节奏依然保持不变。

有些学员应该亲身感受过，或者在我帮助他追求的过程中体会过。一个很“认真”的男人，发给女生的消息永远话很多，给女生的感觉怎么样都好像是他比较依赖女生。试想，如果你是女性，你需要这样的男人做什么？
