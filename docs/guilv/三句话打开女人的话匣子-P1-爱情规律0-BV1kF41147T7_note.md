## 理解沟通的同频原理

上一节我们概述了课程目标，本节中我们来看看沟通的核心基础。你和女孩聊天交流沟通的时候，你俩一定要在一个频道，在一个波段。一个频率上面。

**核心概念**：沟通同频。这可以理解为确保双方的交流信号在同一个“波段”上，避免误解。如果频率不一致，就会出现：女人要的是苹果，而你给她一车香蕉的情况。你们聊天的气氛，也会变得越来越差。

## 认识男女思维的差异

理解了同频的重要性后，我们需要探究频率错位的常见原因。这通常源于男女思维的天然差异。

男人常常这样认为：我来尽我所能地帮助你解决问题，而不是帮你消除不良情绪。这样子两人的矛盾就会激发。都感觉自己是对的，而且对方的思维，互相都不理解。

## 运用“先情绪，后道理”的解决步骤

现在都知道了问题所在，那解决的方法就来了。女人其实也是会讲道理的，关键在于时机和顺序。

以下是具体的操作步骤，其核心是 **“先情绪共鸣，后理性探讨”**。

首先，开始你按照发散思维，你先和她一起陷进去。你就必须让她觉得，你非常懂她的烦恼。

然后，等她情绪平静下来后，情绪稳定了，你再找她聊下讲道理，避免这样的事情发生。

为了让大家更好地理解上述步骤，我们来看一个具体的例子。

比如女孩埋怨职场中不爽的人。你按照发散思维，和她有一样的情绪。

你可以说：“**我最鄙视这种人了，他怎么这么无耻。在我们公司里也有一个，周围的同事也都不喜欢他。**”

现在你们是不是有点感觉了？对，就是这样。你要让她知道你是站在她这边的。她要的就只是你的感受，让她感觉你对她感同身受就行了。
