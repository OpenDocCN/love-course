## 核心观点阐述

上一段我们提到了本节课的主题，本节中我们来具体看看其核心论述。

谈恋爱时，真诚本身没有价值。真诚是一种可以被轻易做到的状态。在社会交往中，越倾向于完全真诚的人，往往越难以处理复杂的人际关系网络，并更容易被边缘化。

相反，那些更懂得运用策略、甚至带有些许“虚伪”色彩的人，因为其行为更符合普遍的人性预期与社会规则，反而社会适应性更强。这类人更容易取得成功，也拥有更灵活的进退空间。当你极度真诚时，你说的每一句话都过于实在和具体。

理解了基本观点后，我们来看看这一规律在追求异性时的具体体现。
