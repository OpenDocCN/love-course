上一节我们提到了肢体语言的重要性，本节中我们来看看它的具体定义。肢体语言是指不需要通过口头言语，仅凭身体动作和行为就能向他人传递信息的方式。

例如，当你与人交谈时，双手不知如何安放，时而摆弄手指，时而拉扯衣角。这些动作虽然无声，却明确地向对方传达了你紧张不安的情绪。即使你未发一言，你的身体已经“出卖”了你的内心。
