## 关系升级的常见误区与后果

2.  你和女生拥抱了吗？
3.  女生有主动送过你礼物吗？

如果以上答案均为“否”，那么无论当前好感度多高，贸然表白都很可能被拒。

这种表达方式即使被婉拒（例如“最近太忙了”），也不会让双方尴尬，你依然可以保持主动，不会因此变得被动。

通过询问女生是否介意你的某些行为，来试探她在潜意识中是否已将自己置于女友位置。

例如：
*   “我有些坏习惯，你介意吗？”

这个问题本身预设了“只有女朋友才会在意这些”的前提。只要女生正面回答了这个问题（无论是介意或不介意），都意味着在她的潜意识里，已经接受了你们之间超越普通朋友的关系框架。
