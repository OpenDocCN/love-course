## 遵循自己是吸引力的基础

一个不懂得遵循自己的人，是不够坚定的。不够坚定，也就会人云亦云，没有自我。女生会因此认为你是不靠谱的。在视频的最后有解决方案，请你一定要看完，而且要记得保存收藏。遵循你自己，也体现在追求姑娘的整个过程。

上一节我们介绍了遵循自己的重要性，本节中我们来看看一个具体的生活场景。

### 实例分析：当喜好冲突时

你喜欢吃辣，她不会吃辣，怎么办？你说：“走，吃火锅去。”姑娘说：“啊，我不会吃辣。”于是你接受，说：“这样，那就换吃披萨吧。”如果你这么做，你没姑娘喜欢再正常不过。因为你没有遵循自己。不懂遵循自己的人，是不靠谱的。

姑娘从来不为自己意识到这些，但姑娘天生的筛选机制很厉害。所以经常的，她就是对你没感觉，她也不知道为啥。那怎么做呢？遵循自己呀。所以换成我，我不会轻易满足她。我就是要带她吃一下辣，就是要“拉翻”她。

就是要看她被辣得鼻涕横流的样子。所以我经常在姑娘那里听到：“你怎么这样啊，就知道欺负人，哼。”因为在她说不吃辣的时候，我却拉起她的手说：“走，带你去感受一下，太辣可以捏我。”那一刻，不用言语。

所谓一个女人对你产生感觉，也就是在这么些小事情上开始的。

## “遵循自己”与“过度自我”的平衡

不过，遵循自己，可不是啥都跟着自己想法来。“做自己”和“太自我”之间，存在微妙的度。你得去琢磨，去把握。

以下是需要把握“度”的几个关键场景：
*   **时机**：什么时候上去认识她最好。
*   **节奏**：聊天时如何控制节奏感。
*   **行为**：约会的时候该怎么去牵手。
*   **分寸**：时间的度，语言的度，行为的度等等。

你找到了度，就有了质感。不断修炼，去无限接近那个最好的度。

## 懂自己是被人懂的前提

在粉丝群里我说，世间那么多人，懂你的太少。这种感觉男人女人都有。有时候我会想这是为什么，懂一个人很难吗？后来我才知道，这不是难易的问题，而是正常现象。

但是当一个男人，遇到一个女人的时候，很多男人的第一反应不是“叫喊”，而是内心想着“卧槽，这女生也太好看了”，而表情却是当做什么都没看见一样，淡定地与她擦肩而过，同时自己给自己戴上面具，名叫“不为所动”。装逼的人咋就那么多呢。

人们习惯了装逼，装得多了久了，自己都不知道自己是谁，自己都不懂自己了，谁还能懂你？所以要让别人懂你，你先要懂自己，才有资格让一样的人，愿意接近你。
