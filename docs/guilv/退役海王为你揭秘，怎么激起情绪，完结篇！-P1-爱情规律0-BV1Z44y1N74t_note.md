## 一个真实的故事：从“好男人”到“没感觉”

上一节我们提到了许多男性在感情中的困境。本节中，我们来看看一个具体的案例。

我最近搬家后，在电梯里结识了一位女邻居，并顺利获得了她的联系方式。我原本约她晚上外出，但临时改变了主意。

我拿着一瓶红酒和两个杯子去敲了她的门。我并非意图不轨。事实上，她的酒量很好。经过几句玩笑，她邀请我进屋。

她向我倾诉了许多过去的恋爱经历。由于我对“什么样的男人能吸引女人”这个问题特别敏感，我便有意询问她与前任分手的原因。

### 她的“好”前任

以下是关于她前任的描述：

*   她每天都会给他发消息，关心他过得如何。
*   天气变化时，会提醒他加衣服。
*   他生气时，她会发上百条消息请求原谅。
*   几乎逢节必过，并赠送礼物。

她列举了太多对他好的细节，但最终依然选择分手。分手后，前任甚至一度试图出家。如此爱她的男人，她为何要离开？

她的回答只有三个字：**没感觉**。

许多男性做了无数事情以为能感动女性。感动或许会有，但很短暂，这些行为**永远无法**让女性产生被吸引的感觉。

## 吸引的本质：感觉而非逻辑

上一节我们看到了“讨好”的失败案例。本节中我们来看看吸引的真正原理。

真正能让女性产生吸引的，往往需要反其道而行之。女性需要的是**张力**和**挑战**。

作为一个懂得吸引女性的男人，你必须记住：你是一个有挑战性的存在。你需要：
*   和她唱唱反调。
*   开开玩笑。
*   偶尔戏弄她一下。
*   让她觉得你飘忽不定，无法被完全掌控。

女性非常喜欢这种感觉。

## 另一个案例：“我不想再带孩子了”

我们接着聊到了她大学时期的一位男友。那位男友很有魅力。我问她为何分手，她的回答很精辟：**“因为我不想再带孩子了。”**

我立刻明白了。这类男性内心缺乏安全感，因此过度依赖女性：
*   他们会问女性“去哪吃饭”、“去哪玩”。
*   他们总是在意女性的意见和想法。
*   他们在生活和情感上都显得需要被照顾。

当我追问所有被她甩掉的男性有何共同点时，她的回答更加精辟：**“黏。”**

绝大多数不成熟的男性都曾如此：喜欢一个人，就把她置于生命的中心，像婴儿盯着母亲要奶喝。

而女性寻找的是一个**港湾**，一个成熟、自信、有安全感、清楚自身使命、**不以任何女人为中心**的依靠。这点女性或许不会口头承认，但这是事实。

## 如何学习吸引：从了解到实践

上一节我们分析了不吸引人的特质。本节中我们来看看如何学习正确的吸引方法。

那晚我们聊到深夜。我发现从女性视角看待感情非常有价值。我建议你，哪怕对方不是你喜欢的类型，也多与女性聊聊感情话题。你可以从她们的故事中明白，**什么样的男人不吸引人，什么样的男人吸引人**。

多年前，这些问题也困扰着我。我曾是一个和女生说话都结巴的男人。我过去只是被动等待，希望有人能懂我。

幸好我没有等下去，否则我永远不会明白这个改变我一生的道理：

### 核心概念

绝大多数男性试图通过**逻辑说服**和**讨好行为**让女性喜欢自己。他们送花、送礼、不断追求，但从不吸引。

如果你也这么做，女人不可能被你吸引。因为：

**吸引不是逻辑行为。**
女性无法通过逻辑思考来决定是否被一个男人吸引。吸引是一种**感觉**，一种**情绪**。

同理，如果女性对你产生了强烈的吸引情绪，她也无法用逻辑说服自己不喜欢你。

你的逻辑大脑无法直接让女人被你吸引，但你可以用它来**学习**产生吸引的方法，并**改变**自己。
