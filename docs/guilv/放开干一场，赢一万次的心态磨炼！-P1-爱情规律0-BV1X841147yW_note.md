## 第五步：放开情绪，而非控制情绪 😌

有人说能控制好情绪，才能牛逼。但我觉得这话只说对了一半。因为这样仍然是在控制。凡是让你意识到要控制的，都只是技能层面的东西。

以下是两种情绪处理方式的对比：

*   **控制情绪**：比如你被女生冷漠的反应搞得很烦躁，控制情绪就是不断暗示自己要冷静。
*   **放开情绪**：但还有种更牛逼的，就是你本来就很冷静，女生的冷漠反应根本无法触动你。这叫做放开自己的情绪，不要抓着情绪不放。

有时候你越是控制，就越是无法控制。越是暗示自己冷静，越是在提醒自己还没冷静。所以索性就放开它，接受它，忘记它。

**核心逻辑**：当你越是想完美，就越无法完美；越是想对抗，就越无法对抗；越是惧怕，就越是愈演愈烈。

放开手脚，就是一种服侍，扔掉那些束缚你的东西，在心态上给自己洗个澡。
