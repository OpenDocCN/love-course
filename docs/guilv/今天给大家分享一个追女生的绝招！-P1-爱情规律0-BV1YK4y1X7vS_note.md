## 实践方法：如何做到“上心”

要表现得细心一点，并非十分困难的事。关键在于养成一些简单的思维和行为习惯。

*   **遇事多思考**：遇到事情之后，多想一想“这是为什么”。理解事件背后的原因和对方的潜在需求。

*   **出言需谨慎**：自己要说的话，在出口之前先考量一遍，确认它是否恰当，是否可能伤害对方。
