## 无底线付出的悲剧

我见过无数男人为爱付出，无限制地付出，然而他们的结局很少有喜剧。这些男人为了讨好女友，花费无数金钱和时间，却并没有得到女友对他应有的待遇。这还不算悲哀。

可怕的是，他们认为女友对自己不够热情、不够关心，是因为自己还没有付出足够的努力。在这种情绪刺激下，这些男人会更加倾尽一切，甚至连尊严都愿意放弃。像他们这样无底线地作践自己，女生就会爱上他们吗？想太多了。你没有让女生看到你身上的魅力，反而让她认为，你可以轻易被索取。

## 单方面专一的误区

有人找到导师之后说的第一句就是：“我喜欢她，我对别的妹子都没有兴趣，我就要追到她，导师帮帮我。”专一钟情往往被认为是美德。

但成为美德的前提是对方对你有回应。因为爱情是两个人的事，单方面的专一毫无价值。所以每当学员这么说的时候，导师都告诉他：“我能帮到你的前提是先整理好心态。”

有的人就不乐意了：“我都没跟女生牵过手，谈什么心态啊？”

## 心态失衡的后果

没牵过手就不需要谈心态？那我们就好好聊聊，心态不好会怎么样。

以下是几种典型的心态失衡表现：

*   **极端执着型**：有些人正是前面那一类，认为自己这辈子非那个妹子莫属。当失去之后就感觉世界崩溃了，借酒消愁或是自暴自弃。好好一个大男人，把自己弄得活得不如一条狗。
*   **过度控制型**：还有一些人想把对方的一举一动，都想掌握在自己手中。认为只要自己把对方的一切，全方位考虑得周周到到，把他照顾得体体贴贴，这个女生就会喜欢自己。他们的心思是：“我**对你这么好**，就算你要离开我，你心里也舍不得。”
*   **卑微顺从型**：更有一些人怕女生离开自己，在女生面前尽情恭顺、放低姿态，甚至嘴都不敢跟女生顶一句，生怕自己说错了话、做错了事，对方就离开自己。这个心态的极端发展就是，整天疑神疑鬼，总觉得自己女朋友随时会离开自己，因此患得患失、惶惶不可终日。女生被他的疑神疑鬼搞得精神崩溃，到时候就真离开了。

所以，心态不重要吗？如果你心态如此失衡，牵了手又如何？
