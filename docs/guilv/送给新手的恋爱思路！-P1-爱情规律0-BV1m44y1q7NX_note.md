## 不值得追求的类型分析 🚫

以下是几种如果你遇到，建议放手的女生类型。

### 类型一：一直逃避你

面对你的主动，她们通常会有以下这些反应：

*   从不跟你说真心话。
*   聊天内容也不涉及生活。
*   对你从来都没有一点主动性，全靠你主动找她。
*   她总是搪塞你。

原因只有一个：**她不愿意接受你**。导师的建议也只有一个：**及时切断**。

上一节我们分析了逃避型女生的特征，接下来我们看看另一种类型。

### 类型二：简简单单就能被感动

在别人眼里一件再正常不过的小事，她们可能会被感动。然而也可能因为一件小事转眼不认人。

### 类型三：眼光非常高

她们对男生的要求非常非常严格，各种层次都有。单纯的一颗真心对她们更是毫无价值。

了解了哪些类型不值得投入后，你可能会产生新的疑问。
