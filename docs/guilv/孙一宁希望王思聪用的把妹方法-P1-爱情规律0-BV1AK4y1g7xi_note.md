## 什么是情感连接？💞

上一节我们介绍了课程的主题，本节中我们来看看什么是情感连接。

连接指的是连接异性之间的情感，让异性在情感上对你产生好感。

当你和你的女朋友在一起的时候，重要的不是吸引力，而是你们之间的精神连接，或者情感连接。

从把妹方法上说，和女生交往的时候，我们需要做的是连接她，尊重她，带领她，这是男人的责任。

## 行为平衡：二八法则 ⚖️

理解了情感连接后，我们来看看如何在实际互动中把握行为的平衡。

用二八定义来说，就是 **80%** 的情况下，我们和女生的交流是友好的，轻松的，正能量的。

掌握了行为比例，那么具体该如何操作呢？我们的切入点，是用什么样的态度去把握这个平衡。

这个态度就是理解和尊重。

*   **小女人**：渴望被照顾，认同，关怀。
