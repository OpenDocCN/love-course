# 爱情规律课程 0：揭秘情感潜规则 👑 - P1

在本节课中，我们将要学习如何理解情感关系中的潜在规则，特别是当沟通出现障碍或追求遇到挑战时，应该如何应对。我们将探讨“吸引”与“追求”的平衡，并提供具体的行动思路。

---

上一段我们提到了沟通受阻时的基本态度，本节中我们来看看关于“吸引”与“追求”的核心矛盾。

如果女生不听从你的解释，或不想理会你，最佳做法是暂缓行动。有错误就改正，有误会便解释。但需要认识到，缺乏安全感的女生常有一种表现：出现问题或犯错时，你首次解释她可能不听；你第二次耐心解释，她依然不听，显得不讲道理。

此时，你应该让她自己冷静。如果你持续讨好、贬低自己，女性反而会看不起你。

---

以下是关于情感建立方式的两种观点分析。

女性是吸引过来的，但更是追求来的。这看起来与“吸引”观点矛盾，实则不然。

“女性是吸引过来的，不是追求过来的”这个观点无疑正确。但我今天要反对这个观点。理由很简单：如果你不追求，到最后，好女孩都会被别人追走。这是血与泪的教训。

问题在于，许多人把自己看得太金贵，却缺乏支撑这份金贵的竞争条件。他们太把“吸引”当回事，从而忘记了符合爱情的过程，也是一个需要坚持、改善和付出的过程。

如果女孩子稍微考验你，或你制造了一点障碍，她就自我放弃，那她便放弃了。因此，女性是吸引过来的，但也是追求过来的。你不围着她转，如何吸引她？

关键在于度的把握。要很自然、经常地出现在她身边，这样就不会显得过于甜腻或是在刻意追求她。

---

以下是针对自身条件普通者的长期策略。

如果你自身条件不是特别优秀，那么在追求爱情的过程中，你打的必然是一场持久战。一见钟情是好事，但它往往是帅哥美女的专利。

---
