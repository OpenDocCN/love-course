# 课程名称：爱情规律0 - 第1课：刚加上女生开场白说什么好？ 💬

## 避免低效的开场方式

上一节我们介绍了课程目标，本节中我们来看看需要避免哪些常见的错误开场方式。

首先，不要使用网上那些自以为幽默的套路话术。

例如：
> 你有打火机吗？没有啊。那你是怎么点燃我的心的？

男生可能觉得这种开场白幽默有趣，但这完全是自娱自乐的行为。即使你条件优越，女生也可能会认为你情商很低。因此，不要使用这类从网上看来的、自以为有趣的开场白，这显得非常低级。

其次，避免一加上好友就“查户口”。

例如：
> 你好，你是哪里人？做什么工作的？

这样聊天根本无法吸引女生。通常聊不了几句，对方就不会再理你了。

## 掌握正确的开场方法：展现感受

那么，正确的方法是什么呢？核心在于展现友好，并表达你对她的**感受**。

你只需要用自己的感受去聊天，女生也会用她的感受来回应你。不必使用复杂的花招，方法其实很简单。

以下是一个正确的对话示例：
*   **男生**：我是在上海路活动时加你的那个。有专家说双马尾的女孩性格超好。
*   **女生**：我朋友都说我性格不好，哈哈哈。
*   **男生**：那个专家还说了什么，看见没？

女生通常对别人对自己的评价很感兴趣。这时，我们就能借“专家”之口，巧妙地夸奖女生。

例如，你可以接着说：
> 其实也没说什么，就说双马尾的女生肤白貌美大长腿。还说双马尾的女孩都很善良。

（注：专家并没有这样说过。）

为什么这么说呢？因为借用他人之口来夸奖女生，既能展现你的友好，又不会暴露你过强的需求感。

## 推进对话：提出有趣的问题

当女生回应后，例如她说“专家瞎说的，我没有这么好，你别期望太高，会失望的”，这表明她对你的兴趣在升高——她开始担心你会失望。

这时，我们可以通过问一个问题来继续聊下去。注意，不要问“你有什么爱好”、“最爱吃什么”这类容易终结话题的问题。应该问一个**有趣的问题**。

例如，你可以问：
> 你是一个坏女孩吗？

这个问题让女生去证明自己是不是一个“坏女孩”，创造了轻松有趣的互动氛围。在这样的气氛下，女生会对男生更有兴趣。

## 巧妙应对女生的反问

接着，女生可能会反问：“那你是好男人吗？”

许多男生会立刻进入“直男”模式，回答：“我是好男人，真的，我特会照顾女人。”

但聊天不是这样进行的。正确的回复可以是：
> 好男人不是我，我不是曾小贤。我没有当好男人的机会，我是一个直男。

这个回复非常巧妙。它暗示了女生：我因为单身（没女朋友），所以没有机会去实践如何当一个好男人。潜台词是，如果和你谈恋爱，我会努力成为一个好男人。
