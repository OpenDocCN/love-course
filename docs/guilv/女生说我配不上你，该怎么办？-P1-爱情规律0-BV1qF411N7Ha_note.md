## 概述：拒绝背后的真实含义

当女孩说“你会找到更好的”或“我不适合你”时，她的意思是：我对你还没有感觉，你还没有吸引到我。通俗地讲，这就是她在拒绝你的表白。

绝大多数男人都会遇到类似情况。他们身边可能有这样一个女孩，或许是同事、同学，起初聊得不错，感觉投缘，于是渐渐产生好感。也可能是以前就喜欢，但因故失联后又重逢。

无论原因、时间或地点，当你开始喜欢一个女孩，就会越陷越深。你会反复琢磨她说的每一句话，她的一个笑容能让你开心半天。直到你开始幻想：如果她能成为女朋友该多好。

## 第一节：常见的错误做法与后果

上一节我们了解了拒绝的含义，本节中我们来看看绝大多数男人会如何错误应对。

一旦男人产生“要让她成为女友”的念头，噩梦往往就开始了。他们会想方设法讨好女生，具体行为包括：

*   送女生礼物。
*   发关心她的短信。
*   向女生表白。

如果你还没有吸引到女生，那你的表白就好比扔硬币：
*   **正面**是拒绝。
*   **反面**是“我们还是做朋友吧”。
*   只有**侧面**（即把硬币扔得竖起来），她才会答应。

这概率非常低。如果你走到表白这一步，那么女生几乎百分百会拒绝你，并且会增加后续吸引她的难度。

*   晚上想玩游戏，她却不停打电话。

你会有何感觉？你会觉得烦，会更加厌恶她。你会不接电话、不回短信、删除好友，甚至怒斥她不要再骚扰。你绝对会这么做。

所以，女生也会。

在探讨正确做法前，我们必须认清一个事实：**表白对激起女生被你吸引的情绪没有任何帮助，甚至只会阻碍吸引的产生。**

接下去要讲的，可能是你最关心的：在表白失败后该怎么做。

**挽回最大的秘诀就是：不挽回。**
