## 第一招：镜像回应法

理解了测试的目的后，我们来看看第一种应对策略。其精髓在于：**你用什么招，我就还你什么招**。即用类似的态度或逻辑回应对方的要求，保持框架平等。

这种回应既接受了任务，又提出了对等的友好要求，将单向索取转化为双向互动。

## 第二招：价值交换原则
