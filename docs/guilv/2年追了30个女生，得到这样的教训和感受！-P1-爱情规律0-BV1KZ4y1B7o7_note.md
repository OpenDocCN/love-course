## 识别虚假考验与不平等关系 🚩

上一节我们介绍了课程概述，本节中我们来看看追求过程中可能遇到的第一个陷阱：以考验为名的不平等要求。

那些总是提出考验但迟迟不确立关系的异性，其动机值得怀疑。例如，对方提出“你减肥到130斤，我就和你在一起”、“你月薪达到2w+时，我就答应你”或“你能到我这个城市来陪我，我就当你女朋友”等条件。

这些话语只需听听，不必当真。即便你通过了某项考验，对方也未必会兑现承诺，反而可能设置下一个考验。即使最终确立关系，这种基于条件交换的关系也注定不平等。在交往中，一旦你未能持续满足其要求，对方便可能以分手相要挟。

**核心概念**：这种模式的核心并非喜欢，而是享受被喜欢的感觉。其关系公式可概括为：**“关系” = 持续满足对方提出的新条件**。
