## 概述

在本节课中，我们将要学习一种被称为“无畏”的核心心态。这种心态并非关于财富的多寡，而是一种敢于为自己所爱投入的勇气与洒脱。掌握它，能让你在情感关系中展现出独特的吸引力。

---

## 无畏心态的核心：敢于投入

为自己喜欢的东西花钱的男人，姑娘会非常非常崇拜。这不在于有没有钱，也不在于花了多少钱，只在于敢不敢花，舍不舍得花。

女生能在这样的男人内心里，感受到一种潇洒与坚定。只要喜欢就会努力去追求，而不被世俗束缚。拥有这种态度，那么放在喜欢的姑娘这里，也会一样竭尽全力。所以姑娘更愿意去喜欢拥有这种特质的男人，因为值得。

上一节我们介绍了无畏心态的核心是敢于投入，本节中我们来看看如何正确实践这种态度，避免走入误区。

---

## 实践的关键：态度高于物品

但这不意味着你堆砌潮牌、展示大牌Logo就能追到姑娘。因为大部分姑娘不会在意你的外在，她们只在意你驾驭外在的方式。

以下是错误示范与正确示范的对比：

*   **错误示范**：如果你好不容易买到了一件Supreme的卫衣，却老是有意无意炫耀，小心翼翼地穿着。这就没有了“无畏”，姑娘会瞬间无感。
*   **正确示范**：你可以展示自己的潮品，真实地表达自己。你可以和她说：“看，老子买到了Supreme，牛不？”你一定要大大方方地表达，随随便便地穿它、用它、造它。

你可以让你的Supreme卫衣上面，有火锅的辣椒油，也可以有姑娘们的唇印。但千万不要让它看着很干净。你所有喜欢的物品，都应该存有你使用过的痕迹。

所以，我的手机是不带壳的，每一道划痕都是我的专属印记。因为这就是我的手机，无人取代。这就是无畏。

理解了如何通过物品展现态度后，我们来看看这种心态如何直接应用于与女生的互动中。

## 应用于互动：直接与洒脱

当你理解了这些，你在路上遇见喜欢的姑娘，就不会想那么多了。直接上去认识她，就是无畏。你在表达自己的态度。

你的气质会自带Logo，就算穿着优衣库的素色衣服，也会自动成为Louis Vuitton。此时此刻，你就是奢侈品，你就是那位Luxurious Man。

所以，你不用怎么说话，就能和她认识。因为你的无畏之感，已经帮你说了。她感受得到，她也愿意接受你的洒脱与真诚。她想和你认识。

在与姑娘聊天的时候，你也不会有小心翼翼的共同心态。

以下是两种不同的聊天心态：

*   **谨慎心态**：比如姑娘回复了你一句，你非要存着放着，待洗个澡喝口茶，像玩策略游戏一样，斟酌字句，反复看N遍，才放心按回车。
*   **无畏心态**：再喜欢她，也随便一句就发出去了。因为你就是你，你说的任何一句话，都是在表达你自己，没什么好多想的。因为你是那位随意驾驭Supreme的男人，这就是你的节奏。

本节课中我们一起学习了“无畏”心态的完整应用。

你是你所消费的，你是你所表达的，你是你所创造的。无畏，是你内心的力量。它能让你在狂风骤雨里平静，能让你在苍茫黑夜里闪亮，能让你在孤独境地里微笑。
