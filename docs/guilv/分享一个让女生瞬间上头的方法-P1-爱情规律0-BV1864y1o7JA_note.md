## 场景选择与前提条件

上一节我们介绍了课程目标，本节中我们来看看具体实施的第一步：场景设置。

初次约会时，选择共进晚餐。在点餐时，**选择需要用手剥壳的小龙虾**。

**核心逻辑**：吃小龙虾时，双方通常都需要戴上一次性手套。这导致双手被占用，进行其他动作会非常不便。这个“不便”是后续所有操作得以成立的客观前提。

## 核心操作步骤

在创造了“双手不便”的客观条件后，以下是具体的操作流程，每一步都环环相扣。

1.  **提出“被拒绝”的请求**
    主动询问女生：“需要我帮你剥虾吗？”由于是初次约会，对方出于礼貌或矜持，**大概率会选择拒绝**。这个拒绝在我们的计划之内。

2.  **执行预备动作**
    无论对方是否接受，你开始自顾自地剥虾，并将剥好的虾肉集中放在一个干净的碗里。如果对方好奇你为什么剥了不吃，可以随意找个理由（例如：“先剥好放着，凉一下”），无需正面详细解释。

3.  **创造“不得不帮”的瞬间**

    *   如果你戴着项链，可以说：“项链老是碰到油，能帮我把项链拉到脖子后面吗？”

4.  **应用心理学原理**
    此时，心理学中的“**拒绝后让步**”效应开始起作用。对方已经拒绝了你一次（帮你剥虾的提议），出于内心的亏欠感，她第二次拒绝你一个小请求的可能性会**大大降低**。加之你确实戴着手套不便操作，她有充分的客观理由来帮助你。
