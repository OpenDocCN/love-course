## 概述：爱情的本质是概率

追求一个人时，我们其实能感觉到对方是否喜欢自己。只是初期很多人不愿承认，直到多次受伤后才醒悟。我们都明白，没有双向奔赴的爱情，真心便用错了地方，毫无价值。

我能谈多次恋爱，原因很简单。追求女生不是一个**养成游戏**，而是一个**概率游戏**。即使你是公认的高价值男性，能吸引99%的女性，也总有1%的女性对你无感。反之，即使你条件普通，99%的女性看不上你，也仍有1%可能因性格缺陷等因素对你产生好感。

如果你投入大量感情后，才发现你追求的是那1%对你无感的人，那么你所有的努力从一开始就是无用功。因此，想要成功，你必须认清并接受概率的存在。

## 概率可以人为提升

上一节我们介绍了爱情的概率本质，本节中我们来看看如何操作这个概率。与游戏不同，追求异性的成功率是可以人为提高的。

你的**颜值形象**、**生活质量**和**社交能力**越强，成功的概率就越高。公式可以简单理解为：
`成功率 = f(自身价值)`
游戏输了只能重开，但追求失败，你可以潇洒离开。最糟糕的情况是，你只有1%的成功率，却死磕一个目标，忽视了其他99条可能成功的路。

## “养成游戏”的风险

当然，万事不能一概而论。如果你真心喜欢某人，也可以选择“养成游戏”，即试图让一个最初对你无感的人喜欢上你。

理论上可行，但你必须做好准备：这是一个非常艰辛的过程，意味着你要在关系天平失衡的情况下**单方面付出**。这像一个无底洞，成功率低，且随时可能因一个更高价值男性的出现而功亏一篑。

即使最终她因感动而同意在一起，恋爱质量也难以保证：她是否尊重你？是否会无限推迟亲密关系？真正的挑战可能才刚刚开始，你的生活中仿佛埋下了一颗定时炸弹。

我曾听闻一个案例：一个男人在女友身上花费十多万，但两人约会时无话可说，肢体接触很少，且女方对外从不承认这段关系。
