分享者基于大量经验，总结出女生在交往初期的一些常见行为模式。

## 绝对原则：不主动即无兴趣 🚫

上一节我们列举了一些观察到的现象，本节中我们来看看分享者提出的一个核心且绝对的原则。

许多人会为对方的不主动寻找借口，例如性格慢热、工作繁忙或自身原因。但分享者指出，这些通常是自我安慰的托词。真正喜欢一个人时，很难做到如此被动和冷淡。

为了证明这一观点，分享者以自己的经历为例：

> 我在七年内接触了大量女生，尝试过各种策略，包括“慢慢发展”和“直接表明”。但事实是，无论采用哪种方式，对我没兴趣的女生最终都不会选择和我在一起。

因此，他得出了最终结论：

*   **关键公式**：`女生兴趣指数 ∝ 主动联系频率`

## 总结与延伸 📝

本节课中我们一起学习了基于大量两性交往经验总结出的“爱情规律0”。核心内容包括女生可能隐藏的行为模式，以及一个至关重要的判断原则：**缺乏主动联系是缺乏兴趣的明确信号**。

理解这些规律有助于避免单方面的过度投入和自我欺骗。两性关系的建立基于双向互动，持续的、单向的主动往往难以推动关系发展。
