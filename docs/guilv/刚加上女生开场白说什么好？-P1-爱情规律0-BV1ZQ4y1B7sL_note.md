## 错误开场方式分析

上一节我们介绍了课程目标，本节中我们来看看需要避免的错误开场方式。

许多男生会从网上寻找自以为幽默的开场白。

以下是两种典型的错误方式：

*   **使用低级的“幽默”话术**：例如“你有打火机吗？没有啊，那你是怎么点燃我的心的？”。这种话术只会让男生自我感动，在女生看来可能显得低级，甚至像“地情上的ATM”（意指只有经济价值、缺乏情感互动的人）。
*   **一上来就查户口**：例如“妹妹你好，你是哪里人？什么工作？”。这样聊天最多进行五六句，女生就不再回复了。

## 正确方法：展现友好与感受

了解了错误示范后，我们来看看正确的核心方法：**展现友好，展现感受，展现出你对她的感受**。

关键在于用自己的真实感受去开启对话，这样女生也会用感受来回应你，无需复杂的花招。

掌握了核心原则后，我们通过一个具体例子来看看如何应用。

## 如何延续对话

当对话气氛建立起来后，我们需要用有趣的问题来延续它，避免陷入僵局。

不要问“你有什么爱好？”或“最爱吃什么？”这类容易终结话题的问题。应该问一个有趣的问题，例如：“你是一个坏女孩吗？”

这个问题让女生去证明自己是不是“坏女孩”，能创造出友好、自然的聊天气氛，从而让女生对男生产生更多兴趣。

在友好的互动中，女生可能会反问：“那你是好男人吗？”

许多男生会直接回答：“我是好男人，真的我是好男人，我特会照顾女人，对女孩好。”但这不是最佳的聊天方式。

正确的回复方式是：

这个回复非常巧妙，它暗示了女生：**我因为没女朋友，所以当好男人的机会我没有，假如跟你谈恋爱了，我会是一个好男人。**
