## 原理概述：什么是“承诺与一致”？

上一节我们介绍了课程的主题，本节中我们来看看其核心定义。

“承诺与一致”是一个心理学原理，意指当一个人对某个观点、立场或标准做出公开的承诺（无论是口头还是行为上的）后，他/她未来的行为会不自觉地受到这个承诺的引导，以保持言行的一致性。

**核心公式**可以概括为：
**公开承诺 → 认知协调压力 → 未来行为与承诺保持一致**

## 实战应用：从“筛选”开始

理解了原理后，我们来看看如何在实际互动中运用它。一个有效的方法是“筛选”，即通过提问引导对方表达出符合你期望的标准或立场。

以下是“筛选”过程的一个典型步骤：

1.  **设定话题**：提出一个与你潜在关切相关，但对方大概率会给出“正面”回答的问题。例如，询问对方“是不是所有女孩都喜欢有钱的男人？”
2.  **引导承诺**：正如案例中那样，绝大多数人会否认自己只爱钱，并主动阐述自己看重的其他品质（如性格合得来、体贴等）。
3.  **获得框架**：当对方说出这些标准时，她实际上是在对你做出一个“承诺”。这个承诺会在后续关系中无形地约束她的行为。

让我们通过一个具体例子，看看“承诺与一致”是如何在关系中发挥作用的。

在课程开头的故事中，导师通过讨论“女孩是否都爱钱”这个话题，引导女孩做出了“我不是爱钱的女孩”的承诺。当她后来成为女友时，便会不自觉地回顾并遵守自己当初设定的这个标准，从而减少了过分的物质索取。她需要通过行动来证明自己言行一致，而非“光说不做”。
