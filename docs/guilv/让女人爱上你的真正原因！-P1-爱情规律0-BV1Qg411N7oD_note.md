## 核心：感觉即情绪

当女人爱上一个男人时，根本就不会在乎他帅不帅、高不高、有没有钱。关注的就不是这些。有太多的例子可以证明。

绝对不是。你是从很多男性口中听说的。这些男人接触的女性并不多，并且观点很片面。

那么，什么才是这种感觉呢？听起来很抽象。

其实，并不抽象。我换个角度给你说，你就可以理解了。为什么你喜欢这个女人？因为她漂亮、性格合适或者身材好。可是不管是什么，在你看到这女人后，她就能给你带来你想要的情绪。

你跟自己喜欢的女孩聊天，跟自己喜欢的女孩约会，见到喜欢的女孩，你整个人就会有一种说不出的感觉。这个说不出的感觉，其实就是一种情绪。她可以带给你这种情绪。

从这个角度来讲，让女人喜欢上我们，我们就仅仅需要给她们这种情绪，就能搞定她们了。

**公式：吸引力 ≈ 双方互动的“张力”**

让女人真正爱上一个男人的是情绪。女人为什么喜欢看韩剧？就是因为韩剧能带给女人情绪价值。你为什么一直追不到女人？也就是因为你追女人的一切手段，都是逻辑的、不感性的。
