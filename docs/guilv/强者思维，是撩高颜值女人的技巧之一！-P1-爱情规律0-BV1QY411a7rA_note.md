## 什么是强者思维？💡

上一节我们界定了强者思维的适用人群，本节中我们来看看它的核心定义。强者思维**并非对女孩进行人身攻击或取笑其身体缺陷**，而是通过某种含蓄的方式，表达出你对她并不特别感兴趣的态度。

我有个朋友的经历可以说明这种思维。她和前男友分手后，发现对方交了一个新女友，她因此产生了一种“被比下去”的不服气感。于是她主动联系前男友，关心起他的新恋情。前男友误以为她想复合，便与新女友分手了。然而，她的目的只是满足虚荣心，验证自己的魅力，并非真想复合，最终冷冰冰地拒绝了前男友的复合请求。这个故事提醒我们，任何时刻都要保持清醒，别太猴急。

## 为什么要使用强者思维？🤔
