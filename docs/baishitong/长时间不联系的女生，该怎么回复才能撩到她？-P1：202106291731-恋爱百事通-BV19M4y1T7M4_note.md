# 如何回复长时间不联系的女生以重新建立联系 - P1：202106291731

## 概述
在本节课中，我们将学习如何回复长时间不联系的女生，以便重新建立联系并可能引发进一步的互动。

以下是直男风格的回复示例：

- 怎么还没有回复的？

这种常规的聊天方式只会使对话陷入长时间不联系的尴尬局面，导致很多兄弟和女生在聊天中变得越来越冷淡。

- 聊天过于一板一眼，显得无聊。
- 她们可能觉得男人都是这样跟女生聊天的。

以下是高情商风格的回复示例：

这种回复方式能够打破常规，增加趣味性，从而可能引起女生的兴趣。

本节课中我们一起学习了如何通过不同的回复风格来重新建立与长时间不联系的女生的联系。通过使用高情商的回复方式，我们可以打破常规，增加互动的可能性。
