# 恋爱百事通 - P1：女人不拒绝你这三件事，暗示你进攻

以下是判断女性是否对你有好感的第一种方式：

- **现象描述**：女性不拒绝与你熬夜聊天。
- **行为分析**：女性愿意陪你熬夜，说明她在你心目中很重要，想要时刻与你在一起。
- **公式**：熬夜聊天 = 女性对你有好感。

## 列表2：不拒绝亲密行动
以下是判断女性是否对你有好感的第二种方式：

- **行为分析**：女性主动靠近你，说明她对你有安全感，觉得与你在一起不紧张，这是对你信任的体现。
- **公式**：主动靠近 = 女性对你有好感。

## 列表3：不拒绝独处
以下是判断女性是否对你有好感的第三种方式：

- **行为分析**：女性愿意与你单独出去，说明她对你有好感，并信任你。
- **公式**：独处 = 女性对你有好感。

## 总结
本节课中我们一起学习了如何通过观察女性在特定情境下的行为来判断她们是否对你有好感。希望这些技巧能帮助你更好地了解女性的心理，从而在恋爱中更加得心应手。
