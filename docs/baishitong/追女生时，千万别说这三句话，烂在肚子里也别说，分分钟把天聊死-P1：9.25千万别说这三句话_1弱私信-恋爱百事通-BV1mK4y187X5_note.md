# 恋爱避坑指南 第1课：追女生时千万别说这三句话 🚫💬

## 第一句：把你的照片发给我看看 📸

事实上，优秀的女生身边通常不乏追求者，你无需刻意去证实这一点。说出这句话反而可能导致女生将你拉黑。

## 第三句：我谈过三四个女朋友 💔

如果女生没有主动询问你的感情经历，你就不要主动提及。即使她问了，你也不一定要完全如实相告。因为女生通常比较在意细节，你永远不知道在谈论哪一任前任时，会无意中触怒她。她生气了，你可能还浑然不觉。这样一来，你们之间辛苦建立起来的联系，可能会瞬间瓦解。
