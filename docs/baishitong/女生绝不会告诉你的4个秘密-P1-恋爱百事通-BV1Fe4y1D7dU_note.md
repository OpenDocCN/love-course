# 恋爱课程 01：四个关键认知 🧠

在本节课中，我们将学习关于两性互动中四个常被忽视但至关重要的认知。理解这些要点，有助于建立更健康、更清晰的交往心态。

---

上一段我们提到了本节课的主题，接下来，我们将逐一详细解析这四个核心秘密。

## 1. 消息回复的真相 📱

女生不回复你的消息，其中约**10%**的情况是因为忙碌，而约**90%**的情况是**故意**的。这意味着，在绝大多数情况下，不回复是一种明确的态度信号。

## 2. 态度突变的根源 🔄

突然对你冷淡的女生，往往与另一个男生有关。她忙于冷落你，是因为她正将热情投入到另一个人身上。你不必追问原因，因为你能听到的只会是**借口**。

## 3. 在乎程度的反作用力 ⚖️

你越是通过言行表现出**在乎她**，她就越可能**得寸进尺**。这揭示了关系中需求感与权力动态之间的基本平衡原理。

## 4. 追求时间的现实框架 ⏳

追求一个女生的合理时长，短则几天，长则数月。那种追求数年最终靠“感动”赢得芳心的情节，几乎只存在于电影中。现实情况更可能是：**她并非被感动，而是没有其他更优选择**。

---

本节课中，我们一起学习了关于消息回复、态度变化、需求感影响以及追求时长四个关键认知。掌握这些现实层面的认知，是构建理性、平等恋爱关系的第一步。
