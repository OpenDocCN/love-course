快餐式恋情有一个最显著的特点就是时效短。两个人只要看对眼，立马就可以陷入到热恋中。但这番热情并不能维持过久。当费洛蒙失效，双方觉得索然无味了，就很容易分手。虽然并不是每一段快餐式的爱情都没有真爱。
