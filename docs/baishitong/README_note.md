# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，这个盒子用来存放数据。在Python中，我们通过一个名字（即变量名）来访问盒子里的数据。

创建一个变量非常简单，只需要使用等号`=`进行赋值即可。

```
my_name = “百事通”
my_age = 25
```

在这个例子中，我们创建了两个变量：`my_name`这个“盒子”里存放了字符串“百事通”，`my_age`这个“盒子”里存放了数字25。

## 核心数据类型 🔢

仅仅把数据放进“盒子”还不够，我们还需要知道“盒子”里装的是哪种类型的数据。Python有一些内置的核心数据类型，最常见的有以下几种：

以下是三种最基本的数据类型介绍：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数或负数。
    *   示例：`-10`, `0`, `100`

2.  **浮点数**
    *   表示带有小数点的数字。
    *   示例：`3.14`, `-0.001`, `2.0`

3.  **字符串**
    *   表示文本信息，需要用单引号`‘ ’`或双引号`“ ”`括起来。
    *   示例：`‘Hello’`, `“Python教程”`

## 如何查看与转换类型？ 🔄

在编程时，有时我们需要知道一个变量的数据类型，或者在不同类型之间进行转换。Python提供了方便的函数来完成这些操作。

使用`type()`函数可以查看任何变量的数据类型。

```
print(type(my_name))  # 输出：<class ‘str’>
print(type(my_age))   # 输出：<class ‘int’>
```

如果你需要将一种数据类型转换为另一种，可以使用特定的转换函数。

以下是常用的类型转换函数：

*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

例如，将一个字符串数字转换成整数进行计算：
```
num_str = “123”
num_int = int(num_str)  # 现在 num_int 是整数 123
result = num_int + 7    # 可以进行数学运算：130
```
