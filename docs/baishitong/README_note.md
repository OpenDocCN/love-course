# 🌟 百事通教程：Markdown格式基础

## 概述
在本节课中，我们将学习如何使用Markdown格式编写文档。Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成结构化的HTML格式。

## Markdown基础语法

### 标题

以下是Markdown标题的示例：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 段落

段落之间需要空一行。

### 强调

以下是Markdown强调的示例：

```markdown
*斜体*
**粗体**
```

### 列表

以下是Markdown列表的示例：

```markdown
- 无序列表
1. 有序列表
```

### 链接

以下是Markdown链接的示例：

```markdown
[链接文本](链接地址)
```

### 图片

以下是Markdown图片的示例：

```markdown
![图片描述](图片地址)
```

### 引用

以下是Markdown引用的示例：

```markdown
> 引用文本
```

### 代码

以下是Markdown代码的示例：

```markdown
`单行代码`
```

```python
# 多行代码
print("Hello, world!")
```

## 总结
本节课中我们一起学习了Markdown的基础语法，包括标题、段落、强调、列表、链接、图片、引用和代码。希望这些知识能够帮助你更好地编写文档。
