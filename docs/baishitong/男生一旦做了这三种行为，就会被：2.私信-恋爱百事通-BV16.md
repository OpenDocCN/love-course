# 男生一旦做了这三种行为，就会被：2.私信-恋爱百事通-BV16

在本节课中，我们将要学习男生在追求女生过程中，三种容易导致自身陷入被动、被“吃准”的行为。理解并避免这些行为，有助于你建立更健康、平等的互动关系。

很多男生在追求心仪对象时，常常感到困惑：自己明明付出了很多，为何却得不到积极的回应，甚至被敷衍和拒绝？这往往是因为踏入了一些常见的“雷区”。

接下来我们就来看看，男生追女生的雷区。一旦做了，就会被女生吃得死死的。

以下是三种需要警惕的行为：

**第一，当女生的免费钱包。**

很多兄弟肯定听过这么一句话：“不舍得为你花钱的男人，一定不爱你”。现在很多女生都有这种扭曲的恋爱观，所以就会经常试探对方是不是真的爱自己，就会开始用各种消费形式来确定男生的真情实意。

*   比如女生想买一份奢侈品包包，会要男生掏钱。
*   想买全套的奢侈护肤品，也会要男生掏钱。
*   有时候一日三餐的消费，都要男生来出。

如果有女生让你干这干那的，并且是免费的指使你，那你就要注意了。我们没有必要为了女生的虚荣去买单。

**第二，甘愿做备胎。**

上一节我们讨论了物质上的无条件付出，本节中我们来看看情感姿态上的问题。回想一下，很多兄弟在和女生聊天时，是不是把自己的姿态摆得很低？最后导致在女生面前失去尊严。

举个例子，就像张爱玲说的：“见了他，她变得很低很低，低到尘埃里。但她的心里是欢喜的，从尘埃里开出花来。”这句话真切地描述了卑微爱情的模样。让人觉得心痛的同时，也告诫了各位兄弟：**爱情是需要尊严的**。

对于很多兄弟来说，当他特别喜欢一个女生时，会为了这个女生自降身价，甘愿当备胎，以为这样自己就可以得到相应的机会。且不说男人当备胎这种行为会让女生心生鄙视，更主要的是这种卑微的行为换不来爱情。备胎是注定转正不了的，除了会把自己伤得“外焦里嫩”之外，得不到任何好处。

**第三，伤害无直线，原谅无底线。**

前面两点分别从物质和情感地位进行了分析，最后一点关乎个人原则。回想一下，你是不是总是不知不觉中，就因为女生丢掉了自己的底线和原则？

举个例子，我问大家一个问题：大多数兄弟在单身时被问及“将来你的女友出轨了，你会分手吗？”我相信答案显而易见，95%以上的男生都会选择分手。但是这样子的事情如果真实地发生在现实生活中时，我相信很多兄弟可能会一而再再而三地选择妥协、原谅。

然后呢？女生可能又会一遍一遍地伤害男生。就像陈奕迅歌词里的那一句：“得不到的永远在骚动，被偏爱的都有恃无恐。”男生在这段感情中付出得越多，陷得越深，女生就会看在眼里。但其实在一些女生的眼里，我们这样子的做法，啥也不是。

---

本节课中，我们一起学习了在追求关系中需要避免的三种行为：成为“免费钱包”、甘做“备胎”以及无底线地原谅伤害。以上的几种行为，兄弟们一定要清醒了，不能够再无条件的持续付出下去。最后伤害的人只会是自己而已。

记住健康关系的基础是**相互尊重**与**平等付出**。建立并坚守自己的原则和底线，是赢得真正尊重和吸引力的关键。
