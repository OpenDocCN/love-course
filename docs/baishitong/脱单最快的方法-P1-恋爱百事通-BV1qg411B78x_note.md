# 脱单教程 P1：高效筛选法 🎯

在本节课中，我们将学习一种高效的脱单策略。其核心在于通过快速、主动的社交互动，筛选出真正合适的潜在伴侣，避免在不合适的人身上浪费时间和情感。

## 核心理念：多电人，快筛选

上一节我们介绍了课程主题，本节中我们来深入理解其核心方法。该方法可概括为六个字：**多电人，快筛选**。这要求我们主动接触更多人，并迅速进行判断与选择。

以下是实现这一理念的三个关键筛选步骤：

1.  **筛选掉对你不感兴趣的人**：如果对方对你的主动接触（“电人”）反应冷淡或毫无兴趣，应果断停止投入，转向其他目标。
2.  **筛选掉相处毫无诚意的人**：在初步接触中，观察对方是否真诚、愿意投入时间交流。对于敷衍、玩弄态度的人，应及时排除。
3.  **筛选掉吊着你胃口的人**：警惕那些不明确表态、若即若离，消耗你情感和精力的人。这类关系应尽快止损。

## 核心行动：大胆电人，及时止损

理解了筛选标准后，我们来看看如何执行。关键在于行动要大胆，判断要果断。

“电人”意味着主动发起社交互动，表达兴趣。同时，必须结合“快筛选”的原则，一旦发现对方符合上述任一应被筛选掉的特征，便立刻停止深入，及时止损。

**执行逻辑**：`主动接触 -> 观察反馈 -> 快速判断 -> 符合标准则继续，否则止损 -> 寻找下一个目标`

不要在一个人身上纠结太久。你的策略应是在各种社交场合“多摸几次牌”，增加样本量。接触的人越多，遇到彼此合适的人（“好牌”）的概率就越大。

## 总结

本节课中我们一起学习了高效脱单的筛选法。我们明确了“多电人，快筛选”的核心，掌握了三条具体的筛选标准，并理解了“大胆行动、及时止损”的重要性。记住，扩大社交基数并快速识别不合适的人，是提高脱单效率的关键。
