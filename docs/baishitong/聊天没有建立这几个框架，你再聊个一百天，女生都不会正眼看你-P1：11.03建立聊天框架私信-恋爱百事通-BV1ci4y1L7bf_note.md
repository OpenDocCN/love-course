# 恋爱百事通 - P1：11.03 建立聊天框架 私信

## 概述
在本节课中，我们将学习如何建立聊天框架，以提升与女生的沟通效果。

## 建立聊天框架的重要性
**公式**：聊天框架 = 语言 + 行为 + 姿态 + 节奏

以下是建立聊天框架的重要性：

- **关系和阴魅**：与女生相处时，如果总有一种被碾压的感觉，那么无论聊什么，你都会处于下风。
- **女生态度**：女生会对你敷衍，即使你翻遍她的朋友圈，寻找共同语言，她也不会对你产生兴趣。
- **价值判断**：女生会认为你没有价值，即使你讨好她，她也会觉得你是没有价值的老实人。

## 建立聊天框架的方法
以下是建立聊天框架的方法：

1. **展示态度**：在聊天中展示你的态度，不妥协。
2. **引导女生**：将女生引导到你的聊天框架中，让她愿意和你聊天。
3. **扭转局面**：如果遇到拒绝，找准时机扭转局面。
