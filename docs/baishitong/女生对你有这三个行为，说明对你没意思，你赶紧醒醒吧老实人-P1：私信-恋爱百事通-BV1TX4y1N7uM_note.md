# 恋爱百事通 - P1：女生对你没意思的三个行为

## 概述
在本节课中，我们将学习如何识别女生对你没有兴趣的三个行为。

## 女生对你没意思的三个行为

### 行为一：无聊或负能量时才找你
**描述**：女生在你无聊或负能量的时候才会主动联系你。

**公式**：无聊/负能量 → 主动联系

**分析**：这种情况下，女生可能只是需要有人陪伴，而不是真的对你有兴趣。

### 行为二：只有在你有钱的时候才找你
**描述**：女生只在你有钱的时候才会主动联系你。

**公式**：有钱 → 主动联系

**分析**：这种情况下，女生可能只是看中了你的经济条件，而不是对你有感情。

### 行为三：忽冷忽热，对你忽好忽坏
**描述**：女生对你的态度忽冷忽热，时而热情，时而冷淡。

**公式**：忽冷忽热 → 偶尔热情，偶尔冷淡

**分析**：这种情况下，女生可能只是把你当作备胎，而不是真心对你。

## 总结
本节课中我们一起学习了女生对你没意思的三个行为。如果你发现自己处于这样的情况，那么可能需要重新审视你们的关系。
