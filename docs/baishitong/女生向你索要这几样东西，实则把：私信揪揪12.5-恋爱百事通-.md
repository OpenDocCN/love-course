# 女生向你索要这几样东西，实则把：私信揪揪12.5-恋爱百事通-

在本节课中，我们将通过一个真实案例，分析在追求关系中可能被当作“备胎”的迹象。课程旨在帮助你识别无效付出，将精力聚焦于建立相互吸引的健康关系。

---

前段时间，我遇到一件事。一天晚上，一位兄弟在公众号后台疯狂刷屏，刷了不知多少条信息。

私聊后得知，他刚和女朋友分手，心里堵得慌，想找个地方发泄。关键是他和这位女生在一起的时间总共不到一周，他们是在光棍节那天表白并在一起的。

情况大致如下。女生在双十一当天因购物与前男友吵架，随后被前男友提分手。女生悲痛欲绝，于是找到这位兄弟喝酒谈心，期间说了很多心里话。这位兄弟本来就喜欢该女生，一时激动便向她表白了。

谁知女生真的答应了。但这段关系来得快去得也快，才几天女生就提出分手，理由是忘不了前男友、不想耽误他。这位兄弟后来才恍然大悟，原来自己只是一场情感过渡中的“备胎”，即便女生分手了，他也没能成功“上位”。

确实，不少兄弟遇到喜欢的女生时会付出很多，认为只要做得足够多就能感动对方。但他们忘记了前提是**你必须先吸引女生的注意**，你的付出才有意义。否则，女生只会将你的好视为“备胎”的一厢情愿。

不明确接受你，却一直吊着你，可能是出于虚荣心，也可能是为了填补内心寂寞。无论何种原因，她们都只想享受你的好，而非真心喜欢你。如果你不想再被敷衍、被当作备胎，那么可以采取行动，学习如何扭转局面。

以下是几种常见的、被女生当作备胎的迹象，你可以对照检查自己是否符合。

**一、 总是叫你帮忙，但平时从不主动**

平时，她有事没事总会找你帮忙。例如，搬家时叫你帮忙，买东西让你去搬，快递叫你去取。你以为这是拉近关系的机会。

实际上，她可能只是将你视为免费的劳动力。虽然频繁的帮忙制造了你很重要的错觉，但她在平时从不主动找你聊天，态度非常冷淡，甚至不回信息，让你产生极大的心理落差。

**公式：** 她的主动频率 ≈ 0 （仅限求助时）
**代码表示：** `if (needHelp) { contactYou(); } else { ignoreYou(); }`

这就是女生不喜欢你的表现。面对真正喜欢的人，女生通常不会如此忽冷忽热。

**二、 照收你的礼物，但不明确态度**

每次你送礼物，比如包包，她都会收下，收下时甚至表现得很开心。你便认为她肯定对你有好感。

于是你更加卖力地讨好她。但在她眼里，这只是有人愿意当“冤大头”而让她开心，并非因为送礼的人是你。即使收了很多礼物，她也从不明确对你的态度，不界定你们之间的关系。

她总是躲闪、回避你的问题，目的就是拖延你。遇到这种情况，就说明你很可能被当成了备胎。兄弟，别再傻傻付出了。
