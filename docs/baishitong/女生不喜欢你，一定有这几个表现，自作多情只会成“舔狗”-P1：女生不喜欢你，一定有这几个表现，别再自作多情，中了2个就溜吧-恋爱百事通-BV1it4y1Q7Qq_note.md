# 恋爱百事通 - P1：女生不喜欢你的表现，别再自作多情，中了2个就溜吧

## 概述
在本节课中，我们将学习如何识别女生是否对你感兴趣，以及她们不感兴趣时的表现。

## 核心概念
**公式**：女生不喜欢你的表现 = 忙碌、情感距离、经济独立、不主动、拒绝送回家

## 女生不喜欢你的表现

### 以下是女生不喜欢的表现：

1. **忙碌**
   - 她总是以忙碌为借口，不回你的消息。
   - 她总是找理由拒绝你的约会。

2. **情感距离**
   - 她不接你的话，也不主动提起情感话题。
   - 她总是保持距离，不让你靠近。

3. **经济独立**
   - 她总是AA制，不欠你任何东西。
   - 她不看重你的物质条件。

4. **不主动**
   - 她从不主动联系你。
   - 你总是主动，她从不回应。

5. **拒绝送回家**
   - 她不想让你送她回家。
   - 她害怕被误会。

## 总结
本节课中我们一起学习了女生不喜欢你的表现。如果你发现她有以上几种表现，那么她可能对你没有兴趣。记住，不要自作多情，及时止损。
