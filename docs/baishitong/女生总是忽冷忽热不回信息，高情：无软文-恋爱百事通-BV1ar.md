## 问题背景：为何对方会忽冷忽热？

上一节我们介绍了课程目标，本节中我们来看看问题的根源。许多男性会遇到一种情况：与女生互动时，对方态度时而热情时而冷淡，信息回复也不积极。

这种情况可能源于无趣或带有压迫感的聊天方式。当对话充满机械式提问时，对方会感到厌烦，从而不愿回应。

## 核心技巧：什么是“妙问”？

面对不有趣的聊天导致对方不愿回复，我们该怎么办呢？下面教你们一招。

研究表明，有一种心理技巧叫做“巧妙的提问”，我们简称 **“妙问”** 。这是一种让人没有明显“被提问感”的委婉沟通技巧。

其核心公式可以概括为：
```
“妙问” = 提出一个具体的、包含你推测的陈述句，而非直接疑问句
```

## 技巧应用：从失败案例到成功案例

我们先把“妙问”运用在一个常见的失败聊天案例上，看一下效果。

**失败案例（直白提问）：**
- 男：大好周末你在做什么呀？
- 女：我在家里。
- 男：这样啊，你追的是什么剧啊？
- 女：最近在看《恋爱进行时》。
- 男：这剧讲什么内容啊？
- 女：（内心：哎呦怎么一直在问啊，烦死了。）

这种连续的直白提问显得非常呆板，容易让对方失去回复兴趣。

**成功案例（运用“妙问”）：**
- 男：周末放假，你去逛街了吗？（推测对方可能去逛街）
- 女：没有哦，最近一直在家里看剧呢。
- 男：这样啊，在看什么剧呢？
- 女：最近在看《恋爱进行时》。
- 男：《恋爱进行时》？那剧情一定很浪漫吧！（推测剧情性质）
- 女：对对对，男女主超甜的！

这样一来，聊天的氛围就会好很多。这个技巧用一句话概括就是：**把自己预想的内容，以陈述推测的形式告诉对方**。

## “妙问”的三大优势

结论上，直白的提问与“妙问”都能够达到获取信息的目的。但不同的是，“妙问”拥有三大显著优势。

以下是“妙问”技巧带来的主要好处：

1.  **让对话变得有趣，避免连续直白提问**
    直白提问会让语言单调。“妙问”通过先说出推测，无论对错，都能使对话氛围更加活跃有趣。
    *   **直白提问**：“你的兴趣爱好是什么？”
    *   **妙问**：“我觉得你最近身材更好了，你晚上睡前都会做运动吧？”

2.  **消除戒备心，挖出心里话**
    “妙问”不会给人一种被审问的感觉，能非常巧妙地获取更深层的信息。
    *   **妙问示例**：“你是在CBD附近住吗？我顺路的话可以送你。”（同时试探住址和是否单身）
    *   **对比直白提问**：“你住哪里？你有没有男朋友？”

3.  **增加信赖感，制造“你很懂我”的错觉**
    利用“妙问”会让对方产生“哇，你好懂我”的错觉。精准的推测会让对方觉得你理解她的喜好和性格，从而快速建立信赖感和亲密感。
