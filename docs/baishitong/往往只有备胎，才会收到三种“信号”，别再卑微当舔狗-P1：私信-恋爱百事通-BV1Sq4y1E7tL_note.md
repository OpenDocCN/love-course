### 信号一：忽冷忽热的态度
- **现象描述**：对方的态度让你感到不稳定，时而热情，时而冷淡。
- **公式**：忽冷忽热态度 = 不稳定情绪 x 不确定关系

### 信号二：网络与现实差距
- **现象描述**：在网络上，对方表现得非常热情，但线下见面后，态度大变。
- **公式**：网络与现实差距 = 网络热情 - 现实冷淡

### 信号三：推脱约会
- **现象描述**：当你提出约会时，对方总是找各种理由推脱。
- **公式**：推脱约会 = 避免面对面交流 x 不想承担恋爱责任

## 如何避免成为备胎
以下是一些避免成为备胎的建议：

### 1. 保持自信
- **现象描述**：相信自己的价值，不要因为对方的态度而怀疑自己。
- **公式**：自信 = 自我价值 + 自信态度

### 2. 明确界限
- **现象描述**：在恋爱关系中，明确自己的底线和期望。
- **公式**：明确界限 = 底线设定 + 期望表达

### 3. 私信询问
- **现象描述**：如果怀疑自己成为备胎，可以通过私信询问对方。
- **公式**：私信询问 = 直接沟通 x 了解真实想法
