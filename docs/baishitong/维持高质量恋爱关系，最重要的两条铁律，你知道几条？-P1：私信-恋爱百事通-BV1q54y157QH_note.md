* **T度**：指个体在恋爱中对关系的投入程度。
* **荷尔蒙自信**：指在恋爱中，个体因为荷尔蒙的影响而产生的自信。
* **冷淡**：指在恋爱中，个体保持一定的独立性和自我意识。

**公式**：

* T度 = 荷尔蒙自信 + 冷淡
