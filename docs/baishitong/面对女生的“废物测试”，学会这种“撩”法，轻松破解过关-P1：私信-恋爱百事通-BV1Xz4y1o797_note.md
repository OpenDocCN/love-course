## 女生的“废物测试”
以下是女生常见的“废物测试”问题：
1. 你喜欢我什么啊？
2. 你觉得我漂亮吗？
3. 你觉得我身材好吗？

## 男生常见的错误回答
以下是一些男生常见的错误回答：
1. 没有比你更好的选择了。
2. 你给我最特别的。
3. 你很普通，但我喜欢你。

## 正确的回答方式
以下是一些正确的回答方式：
1. 你很特别，我喜欢你的笑容。
2. 你很漂亮，我喜欢你的自信。
3. 你很可爱，我喜欢你的真诚。

## 代码示例
```python
# 正确的回答方式
def answer_question(question):
    if question == "你喜欢我什么啊？":
        return "你很特别，我喜欢你的笑容。"
    elif question == "你觉得我漂亮吗？":
        return "你很漂亮，我喜欢你的自信。"
    elif question == "你觉得我身材好吗？":
        return "你很可爱，我喜欢你的真诚。"
    else:
        return "我不知道怎么回答这个问题。"

# 测试代码
print(answer_question("你喜欢我什么啊？"))
```
