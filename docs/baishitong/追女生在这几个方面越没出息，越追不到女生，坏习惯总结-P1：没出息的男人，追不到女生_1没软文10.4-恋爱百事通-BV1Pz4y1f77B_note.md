# 恋爱避坑指南 01：阻碍你脱单的三大“没出息”习惯 🚫

在本节课中，我们将要学习在追求女生过程中，三种常见的、会严重阻碍关系发展的思维与行为习惯。许多男生在感情中投入很多，却总是事与愿违，往往是因为不自觉地陷入了这些误区。

许多男生都希望自己喜欢的女生恰好也喜欢自己，但现实往往事与愿违，很多时候只是一厢情愿。女生面对你时，总有无数的借口，不是在忙，就是永远有忙不完的工作和开不完的会。你甚至无法制造偶遇的机会。你可能会想不通，明明自己的外表与她相匹配，内在也不差，甚至努力满足她对男朋友的幻想，但女生对你依然若即若离，聊天也仅限于寒暄。很多时候，一些你未曾察觉的相处弊端，正是阻碍你们关系升级的巨大障碍。

以下是三种需要避免的核心习惯：

## 1. 得失心过重

这一点说得委婉些，是“想把事情做好、有野心”。这种特质在男人的事业或成就上能带来很大帮助，但放在与女生的相处中，却弊端很大。很多兄弟喜欢上一个女生后，就会一股脑地往前冲，拉都拉不住。对女生非常好，并且怀有强烈的执念，**目标函数**可以概括为：`必须让她成为我的女朋友`。

然而，这种近乎疯狂的执念会给女生带来巨大压力。一旦女生的反馈不如预期，你就容易心态失衡，钻牛角尖，觉得自己只差一点，于是放不下，继续用老方法追求。正确的做法是，从一开始就不要把她看得过于重要。如果她不回你信息，你不应该反复打开对话框等待，而应该**转移注意力**：`换一个聊天对象`。不要把眼光局限在一个人身上，格局要打开。

## 2. 试图用“感动”吸引女生

仍有很多兄弟在追求女生时，使用那些试图感动对方的小伎俩。这种方法或许对情窦初开的小姑娘有效，但对于有一定恋爱经历的女生，根本不起作用。女生普遍有一个观念：对一个人好，比如半夜送夜宵、嘘寒问暖、帮忙处理琐事，这些事情只要是喜欢她的人都能做到。在女生眼里，这并没有什么了不起。这种**低成本的付出**，其本质是：`可替代性极高的行为`，并不能真正吸引她。

## 3. 执念过深，不懂及时止损

有些兄弟会喜欢一位女生很多年，追求很多年，但你们的关系始终无法推进。自己内心却不肯认清现实，总觉得再坚持一下，就能修成正果。然而，追求一位女生真的不需要花费漫长的时间。如果需要你追上半年甚至很多年都无法在一起，那说明这位女生真的对你没有感觉，你们之间没有可能。即使你再耗下去，最终等来的可能是她的结婚请柬。所以，在追求时不要执念过重，**行动准则**应是：`不行就换下一个目标`。不给自己多一点机会，你怎么知道会不会遇到更好、更合适的对象。

本节课中，我们一起学习了阻碍感情发展的三大习惯：**得失心过重**、**试图用低成本感动代替吸引**以及**执念过深不懂止损**。认识到这些问题是改变的第一步，调整心态，扩大社交圈，提升自身吸引力，才是走向成功关系的关键。
