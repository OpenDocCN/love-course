# 明日恋爱会员课：1：先导课2（高考必须会解的四种不等式）_note

在本节课中，我们将要学习高考数学中必须掌握的四种不等式解法。掌握这些方法是后续学习函数、导数等知识的重要基础。

## 一、一元二次不等式

上一节我们介绍了课程的整体安排，本节中我们首先来看看最常见的一元二次不等式。

形如 **ax² + bx + c ≥ 0** 或 **ax² + bx + c ≤ 0** 的式子称为一元二次不等式，其中 **a ≠ 0**。为方便讨论，我们通常将二次项系数 a 化为正数。若 a < 0，则不等式两边同乘 -1，并记得改变不等号方向。

设方程 **ax² + bx + c = 0** 的两根为 **x₁** 和 **x₂**，且 **x₁ < x₂**。其图像是开口向上的抛物线。
*   当不等式为 **> 0** 时，解集为 **x < x₁** 或 **x > x₂**。
*   当不等式为 **< 0** 时，解集为 **x₁ < x < x₂**。

以下是几个具体的例子，帮助大家理解：

**例1：** 解不等式 **x² - 3x < 0**。
*   因式分解得 **x(x - 3) < 0**。
*   两根为 0 和 3，开口向上。
*   解集为 **0 < x < 3**。

**例2：** 解不等式 **-2x² + x + 6 ≥ 0**。
*   两边同乘 -1 得 **2x² - x - 6 ≤ 0**。
*   十字相乘因式分解：**(x - 2)(2x + 3) ≤ 0**。
*   两根为 2 和 -3/2，开口向上。
*   解集为 **-3/2 ≤ x ≤ 2**。

**例3（含参）：** 解关于 x 的不等式 **x² - 2x + 1 - a² < 0 (a > 0)**。
*   判别式 Δ = 4a² > 0，有两根。
*   十字相乘因式分解：**[x - (1 + a)][x - (1 - a)] < 0**。
*   由于 a > 0，故 **1 - a < 1 + a**。
*   解集为 **1 - a < x < 1 + a**。

**例4（双参数）：** 若不等式 **ax² + bx + 2 > 0** 的解集为 **{x | -1 < x < 1/3}**，求 a + b。
*   解集为两根之间，说明抛物线开口向下，即 **a < 0**。
*   -1 和 1/3 是方程 **ax² + bx + 2 = 0** 的两根。
*   根据韦达定理：-1 + 1/3 = -b/a， (-1) × (1/3) = 2/a。
*   解得 **a = -6, b = -4**，故 **a + b = -10**。

**例5（综合）：** 不等式 **ax - b > 0** 的解集是 **(1, +∞)**，解不等式 **(ax + b)(x - 3) > 0**。
*   由 **ax - b > 0** 解集为 **x > b/a = 1**，且 a > 0，得 **b = a**。
*   所求不等式变为 **a(x+1)(x-3) > 0**，因 a > 0，等价于 **(x+1)(x-3) > 0**。
*   解集为 **x < -1** 或 **x > 3**。

**例6（恒成立）：** 不等式 **(a-2)x² + 2(a-2)x - 4 < 0** 对任意 x ∈ R 恒成立，求 a 范围。
*   **情况一：** a - 2 = 0，即 a = 2，不等式为 -4 < 0，恒成立。
*   **情况二：** a - 2 ≠ 0，需满足：
    1.  开口向下：**a - 2 < 0**
    2.  与 x 轴无交点：判别式 Δ = **[2(a-2)]² - 4(a-2)(-4) < 0**
*   解得 **-2 < a < 2**。
*   综上，a 的取值范围是 **(-2, 2]**。

**例7（有解问题）：** 不等式 **x² + ax + 2 > 0** 在区间 **[1, 5]** 上有解，求 a 范围。
*   分离参数：**a > -x - 2/x**，x ∈ [1, 5]。
*   问题转化为 a 大于函数 **f(x) = -x - 2/x** 在 [1,5] 上的最小值。
*   f(x) 在 [1,5] 上单调递减，最小值在 x=5 处取得：**f(5) = -5 - 2/5 = -27/5**。
*   故 **a > -27/5**。

## 二、分式不等式

上一节我们学习了一元二次不等式，本节中我们来看看如何解分式不等式。

分式不等式的核心解法是 **“化分式为整式”**。形如 **f(x)/g(x) ≥ 0** 的不等式，等价于 **f(x)·g(x) ≥ 0** 且 **g(x) ≠ 0**。

**例8：** 解不等式 **x / (x+2) > 0**。
*   等价于 **x(x+2) > 0** 且 **x ≠ -2**。
*   解得 **x < -2** 或 **x > 0**。

## 三、绝对值不等式

理解了分式不等式后，我们进入第三种类型：绝对值不等式。

绝对值不等式的基本形式是 **|x| ≥ a** 或 **|x| ≤ a (a ≥ 0)**。
*   **|x| ≥ a** 的解集是 **x ≤ -a** 或 **x ≥ a**。
*   **|x| ≤ a** 的解集是 **-a ≤ x ≤ a**。

解题时，常将绝对值内的式子看作一个整体。

**例9：** 解不等式 **|x/(x+2)| > x/(x+2)**。
*   一个数的绝对值大于它本身，说明这个数是负数。
*   即 **x/(x+2) < 0**，解得 **-2 < x < 0**。

**例10：** 解不等式 **|1/(x-1)| < 2**。
*   等价于 **-2 < 1/(x-1) < 2**。
*   拆成两个不等式组求解，最终解集为 **x < 1/2** 或 **x > 3/2**。

**例11：** 解不等式 **|√(x-2) - 3| < 1**。
*   等价于 **-1 < √(x-2) - 3 < 1**，即 **2 < √(x-2) < 4**。
*   两边平方（因均为正）：**4 < x-2 < 16**。
*   解得 **6 < x < 18**。

**例12（几何意义）：** 若存在实数 x，使 **|x-4| + |x-5| < a** 成立，求 a 的范围。
*   **|x-4| + |x-5|** 的几何意义是数轴上点 x 到 4 和 5 的距离之和。
*   其最小值为 1（当 x 在 4 和 5 之间时取得）。
*   原式有解，只需 **a > 1**。

**例13（含参恒成立）：** 不等式 **|a - 2x| > x - 1** 对任意 x ∈ [0, 2] 恒成立，求 a 范围。
*   以 x=1 为界讨论右边 x-1 的正负。
*   **当 0 ≤ x < 1 时**，x-1 < 0，不等式恒成立，a 为任意实数。
*   **当 1 ≤ x ≤ 2 时**，x-1 ≥ 0，去绝对值得 **a-2x > x-1** 或 **a-2x < 1-x**。
*   分离参数并求最值，得 **a > 5** 或 **a < 2**。
*   综合两种情况，a 的取值范围是 **(-∞, 2) ∪ (5, +∞)**。

## 四、高次不等式与穿根法

最后，我们来学习解决高次不等式的利器——穿根法。

对于可因式分解为若干个一次因式乘积的高次不等式，如 **(x-x₁)(x-x₂)...(x-xₙ) > 0**，可使用穿根法。
1.  求出每个因式为 0 的根，从小到大标在数轴上。
2.  从数轴**最右端**上方开始画线。
3.  遵循 **“从右向左，从上到下”** 的规律，依次穿过每个根。
4.  **“奇穿偶不穿”**：若因式的指数为奇数，则线穿过该根；若为偶数，则线反弹，不穿过。
5.  根据不等式符号（>0 或 <0），选取数轴上方或下方的区间。

**例14：** 解不等式 **x(x+2)(x-3) / [(x+2)(x-3)] ≥ 0**。
*   化简（注意分母不为零）：**x ≥ 0** 且 **x ≠ -2, 3**。
*   用穿根法解 **x(x+2)(x-3) ≥ 0**，根为 -2, 0, 3。
*   从右上方开始，依次穿过 3, 0，在 -2 处因是奇次方穿过。
*   取 ≥0（数轴上方及零点）部分，结合分母限制，解集为 **[-2, 0] ∪ (3, +∞)**，其中 -2 处为空心。

**穿根法的图像应用（彩蛋环节）**

穿根法不仅能解不等式，还能快速画出某些复杂函数的**大致走势图**，前提是函数能写成**增函数的一次因式乘积**形式。

**例15：** 画函数 **f(x) = x(x-1)(e^x - 2)** 的示意图。
*   三个因式均为增函数，满足条件。
*   令各因式为0，得根：**x=0**, **x=1**, **x=ln2 (≈0.69)**。
*   在数轴上标出根：ln2, 0, 1（注意顺序）。
*   从右上方开始穿根，即可得到函数图像的增减趋势图。此图可用于快速分析函数零点等问题。

**高考对接**

**例16（高考真题变式）：** 函数 **f(x) = (x-a)(x-b)²**，其中 a < b，则其图像可能是？
*   根为 a, b，且 (x-b) 指数为 2（偶次）。
*   穿根法：从右上方开始，遇到根 b 时“偶不穿”，线反弹；继续向左遇到根 a 时“奇穿”。
*   根据此走势，可快速判断正确选项。
