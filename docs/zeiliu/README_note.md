# 课程一：贼六 🎲

在本节课中，我们将要学习一个名为“贼六”的骰子游戏。这个游戏规则简单，主要依赖运气，但了解其核心机制有助于更好地参与。

## 游戏概述 🎯

“贼六”是一个使用六面骰子的游戏。游戏的目标是通过投掷骰子，让骰子显示特定的点数组合来获胜。

游戏通常由两名或更多玩家进行。每位玩家依次投掷骰子。

## 基本规则 📜

以下是游戏的核心流程与胜负判定方式。

1.  玩家投掷一颗骰子。
2.  若投出点数为 **6**，则该玩家立即获胜。
3.  若投出点数为 **1**，则轮到下一位玩家投掷。
4.  若投出点数为 **2、3、4 或 5**，则玩家可以记下该点数作为自己的“暂存分数”，并选择是否继续投掷。
5.  如果选择继续投掷，则新投出的点数会累加到“暂存分数”中。但若在后续投掷中投出 **1**，则“暂存分数”清零并轮到下一位玩家；若投出 **6**，则用当前“暂存分数”直接获胜。
6.  玩家可以在任何一轮选择“停止”，将当前的“暂存分数”转为“永久分数”。首先达到或超过约定目标分数（如100分）的玩家获胜。

## 策略要点 💡

上一节我们介绍了基本规则，本节中我们来看看游戏中的一些简单决策思路。

虽然运气成分很大，但玩家可以决定何时停止投掷以锁定分数。以下是一个决策参考：

*   当“暂存分数”已经较高时，继续投掷的风险会大于收益，应考虑停止。
*   在游戏初期或分数落后时，可以尝试承担更多风险以获取高分。

## 代码模拟 🤖

我们可以用一段简单的伪代码来模拟单次投掷决策的逻辑。这有助于理解游戏流程。

```python
# 这是一个简化的回合示例
暂存分数 = 0
while 玩家选择继续:
    点数 = random.randint(1, 6) # 随机生成1-6的点数
    if 点数 == 1:
        暂存分数 = 0
        print("投出1，分数清零，回合结束")
        break
    elif 点数 == 6:
        print("投出6，用当前分数直接获胜！")
        break
    else:
        暂存分数 += 点数
        print(f"投出{点数}，暂存分数变为{暂存分数}")
```
