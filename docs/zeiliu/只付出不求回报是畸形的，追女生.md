# 只付出不求回报是畸形的，追女生

在本节课中，我们将学习一种名为“逆向合理化”的心理机制，并探讨如何在追求异性时，通过具体的行动来推动关系发展，避免关系停滞不前。

上一节我们介绍了课程主题，本节中我们来看看核心的心理机制。

有一种思维叫做**逆向合理化**。人的大脑无法接受没有逻辑的事情，会自动为已发生的行为寻找一个合理的解释。人会因为做了某些事情，而去合理化自己的行为。

例如，你和一个女生牵手了，她没有拒绝，你们就拉着手走了一路。女生回去后，便会开始逆向合理化自己的行为：“我为什么不拒绝呢？可能是因为我喜欢他吧。” 随后，她就会倾向于认为自己喜欢你。

理解了核心心理机制后，我们来看看它在两性关系中的具体体现。

女生会根据你们的关系来决定是否同意牵手、亲吻等亲密行为。但反过来，女生也会因为和一个人牵手了，而感觉你们的关系不一样了；更会因为和你亲吻了，而觉得自己是你的女朋友了。

以下是关系停滞的常见原因及后果：
*   很多人和女生约会很多次，但就是不敢尝试牵手。
*   导致双方关系一直停留在原地，无法向前推进。
*   最终感觉逐渐淡去，两人成为路人。

为了将理论应用于实践，我们需要采取行动。张爱玲曾说过一句意味深长的话，大意是：想要得到一个女人的心，就要先得到她的身体。这揭示了身体接触对心理认同的逆向影响。

“男不坏，女不爱”这句话，在某种程度上也包含了这个道理。主动的、略带冒险的举动（“坏”），有时能触发对方的逆向合理化，从而催化感情（“爱”）。

关于行动的重要性，有以下对比：
*   一个24岁却从未尝试过牵女生手的男生。
*   一个在高中时期牵手被拒绝过三次，但最终成功一次的人。
*   后者通常更有前途，因为他拥有行动的经验和勇气。

**失败并不可怕，最可怕的是从未行动过。**
