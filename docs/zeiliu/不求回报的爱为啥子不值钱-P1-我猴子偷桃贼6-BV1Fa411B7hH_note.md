# 课程：不求回报的爱为何不值钱 - P1 🧠

在本节课中，我们将探讨一种常见的爱情误区——不求回报的“宠爱”为何往往导致关系失衡甚至失败。我们将通过类比和逻辑分析，一层层揭开这种情感模式背后的真相。

---

上一节我们提到了“宠爱”这一概念，本节中我们来看看它在现实关系中的具体表现。

每个人生命中至少会经历一次“宠爱”。越是成长中缺爱的人，往往越渴望这种宠爱。但这通常是一个错误。若不反思，便容易在爱情中反复受伤，最终不再相信爱情。

这是一个复杂的命题。在构思内容时，我决定放下情感，将冷冰冰的事实呈现出来。我将带领大家，一步步拨开这层面纱，看清它的本质。

每个人的生命里，都可能出现一个刻骨铭心的人。我们曾在爱情里碰得头破血流、撕心裂肺，却再也回不了头。往往只有在彻底失去这个人之后，我们才开始慢慢明白爱情是什么，后悔当初的年少无知。只有当它永远消失于生命之中，我们才会真正醒悟。

起初爱一个人或被爱时，我们毫无经验。此时的“宠爱”没有任何技巧和方法，导致我们犯下各种无可挽回的错误。只有彻底失去后，才会在下一段关系中改变，明白自己当初错得多么彻底。但从此以后，我们很难再毫无保留、不计成败地去爱一个人。

以下是基于现实观察的一个例子：

*   一对父母非常宠爱孩子，有求必应。他们觉得自己当年吃了太多苦，希望满足孩子所有要求，给予其最好的童年。
*   然而，在这种家庭长大的孩子常常并不孝顺。他们的要求日益增长，最终超出父母的承受能力。
*   此时孩子想的不是体谅父母，而是质问：“为什么别人有，我不能有？是你们无能。” 父母的心也随之冰冷。

这与我们在爱情中的模式非常相似。

我们也是如此对待爱人，将最好的东西和内心最珍贵的位置都给了对方。对方却可能轻描淡写地说：“是你自己要做的，又不是我让你做的。” 这种感受与那对父母一样，感到自己的真心付出在别人眼中一文不值。

但我们也能看到另一种情况。许多山区留守儿童，父母常年在外打工，年仅几岁便要独自承担家务、照顾自己甚至整个家。当记者问他们最想要什么时，他们哭着说：“我什么都不想要，只想要爸妈在身边。”

为何往往在成长中获得更少的孩子，反而更能体会父母的爱，更懂得孝顺？

我们在“宠爱”对方时，像极了溺爱孩子的父母。这最终会导致你们的爱情或暗恋变得畸形而扭曲。尽管父母都疼爱孩子，但不能溺爱。爱情亦然。

很多人会说：“如果不是毫无保留，那就不是爱情了。我宁可笨拙地去爱，因为那样的爱情才真诚、发自内心。”

我小时候常看到许多妈妈边哭边打孩子。她不舍得，但必须这么做。因为太多事实告诉她，一味地“蠢爱”行不通。教育孩子需要方法，并非单纯靠发自内心的爱就够了。**爱情也需要方式方法。**

有人反驳：“不行，你这是渣男思维。爱情如此神圣，怎能掺杂别的东西？” 持有这种观念的人需要明白：爱一个人要有所保留。你要尊重别人，但更要**尊重你自己**。许多人在爱情里是卑微的，只尊重对方却不尊重自己。而健康的**两性关系需要平等**。

爱情里，那种扭曲的关系一旦建立，便难以回头。即使是暗恋，本质也一样。即便是最终在一起的情侣，如果建立的是这种“宠爱”关系，也基本无法回头。他们会陷入“争吵->分手->和好->回到扭曲关系->再矛盾->再分手”的循环。

总有一次，“宠爱”的一方会坚持不下去，他受够了，选择退出。等到“被爱”的一方彻底失去他，再也找不到一个如此毫无保留的人时，才终于明白。但一切已无法挽回。

当遇到下一个人时，他收敛了，懂得珍惜了，也学会付出了。但他们能回到之前的关系吗？不能。因为每次和好的前提，都是回到那份扭曲的关系。许多人可能听不懂，需要慢慢体会。很多人对爱情懂得太少。

但致命的是，他们**不求自己懂得太多**，认为那样就不纯粹了。他们幻想：“我会遇到一个人，他懂我所有，我们一拍即合，无师自通，顺其自然。”

**不是你当上父母，就知道怎么教育孩子；也不是你到了恋爱年纪，就懂得了爱情。你在什么地方花费时间和精力，才会在什么地方成长。** 在爱情上的“佛系”态度，终将导致爱情的悲剧。

我上一节课讲了“为什么现在的男生都不追女生了”，根源之一就是他们对爱情懂得太少，总是失败、灰心，却又**不求自己懂得太多**，认为那样就不纯粹。他们在等待一个“懂我所有、一拍即合、顺其自然”的女生出现。相信我，当那个人真的出现时，他们往往又会陷入上述痛苦的轮回。

到了我这个年纪，我明白很多人结婚并非因为爱情，而是因为年纪大了、身边人都结了、或者自己不想再坚持了。**如果你不懂爱情，这一天终究会到来。**

那么，该如何学习爱情呢？关注这个系列，我会将我所知的逐一分享。你的“三连”就是最大的支持。

有人说：“如果我帅，我就有爱情；如果我有钱，我就有爱情。” 难道没钱就不结婚了吗？不帅就不找伴侣了吗？帅的人是少数，有钱的人也是少数。我们大部分都是普通人，只需要拥有一份普通的爱情。不需要多大的地方，温馨就行；不需要多少钱，够花就行；最重要的是，**有你，有我**。

本节课中，我们一起学习了“不求回报的宠爱”在爱情中的危害。它源于善意，却因缺乏方法、不懂平衡与自我尊重，极易导致关系扭曲和最终破裂。真正的爱情需要学习、需要方法、需要平等的付出与尊重，而不仅仅是凭本能的一味付出。在接下来的课程中，我们将继续深入探讨如何建立健康的亲密关系。
