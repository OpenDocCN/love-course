废物测试几乎是所有男生在接触女生的过程中都会经历的事情。这也是很多人的困扰：本来和女生聊得很好，但在回答对方几个问题之后，女生突然就冷淡了，甚至不再理你。其实，你很有可能是遇上了“废物测试”。

## 第一种：劳务关系型 🛠️

有些女生会让喜欢的男生出力帮忙。千万不要觉得这个时候百依百顺是正确的做法。当她觉得你呼之即来、挥之即去的时候，你们就没有任何机会了。这个废物测试属于隐形的一类，不容易被发觉。

以下是常见的例子及应对原则：

*   **例子**：女生让你帮她写工作总结、点外卖、买东西。
*   **核心原则**：切记无谓的付出，尽量做出价值的互换，不要一味服从对方。
*   **应对方法**：当对方提出请求时，可以答应，但附带一个轻松的价值交换条件。
    *   **公式**：`同意请求 + 提出一个小的、轻松的回报要求`
    *   **代码示例（对话模拟）**：
        ```
        女：“能帮我写个工作总结吗？”
        男：“没问题。不过写这个要熬夜，可能得喝杯咖啡精神才好些（附上一个表情）。”
        ```

**注意**：所有提出的价值交换千万不要超越对方的要求，否则对方很容易认为你是一个很小气的人。例如，对方让你寄个快递，你让对方请你吃饭，这就有点过了。

当女生说“我觉得我最近腿粗了”、“我觉得我的脸好大”、“我觉得我好像胖了”等等，这就是自我否定型测试。这个情况需要分两种策略应对。

上一节我们处理了“劳务交换”，本节中我们来看看如何应对女生情绪上的试探。

*   **应对方法**：适当调侃，激发她的情绪互动。
    *   **代码示例（对话模拟）**：
        ```
        女：“哎，我最近好像胖了。”（其实身材很好）
        男：“你的小腿都还没我胳膊粗呢。”
        ```

**情况二：真实的不自信**
如果对方真的在否定自己，对自己的颜值或身材不自信。

*   **应对方法**：适当安慰，或与第三方进行比较。**千万不要去否定她的感受，更不要出谋划策**（如告诉她该怎么减肥），因为那就等于你认同了她“自己不好”的观念。
    *   **代码示例（对话模拟）**：
        ```
        女：“我感觉我最近胖了。”
        男1（安慰比较）：“你这样都说胖，那胖子听了都想打人了。”
        男2（轻松化解）：“一只手都能抱起来的就不算胖，我来试试看？”
        ```

## 第四种：备胎型 🎣

女生说“等我毕业了我们就在一起”、“如果我分手了，你就是我的首选”等等。相信我，如果你听到这些，那你很可能被定位为“备胎”。
