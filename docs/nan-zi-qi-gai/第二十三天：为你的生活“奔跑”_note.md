# 第二十三天：为你的生活“奔跑”_note

在本节课中，我们将学习如何通过一项极具挑战性的身体训练——奔跑，来突破舒适区，激发自身最大潜能，并以此为中心培养卓越的生活习惯。

上一节我们讨论了设立标准与捍卫自我信念的重要性。本节中，我们来看看如何将这种精神付诸于一项具体的身体挑战。

回顾第22天，那是艰难的一天。你不能允许不尊重的行为肆意蔓延。你设定了人们必须达到的表现水平。如果他们未能达到，昨天的行动表明你不惧怕在自己的领域审视他人。你是自己领域的国王，是一个真正的男人。你永远不会为了让他人舒适而放弃自己的信仰。你必须为自己或自己的信仰挺身而出。

舒适是一个无声的杀手。它会像癌症一样扩散，直到你发现自己超重、身体虚弱、精神崩溃，才会意识到自己正受此折磨。

因此，今天你要去做大多数人都讨厌的事情：奔跑。

你要像你的生活依赖于此一样去奔跑。不在乎外面的温度，也不在乎是否下雨。你会一直奔跑，直到你感觉心脏快要爆炸。无论你是出去跑十公里还是一公里，也不在乎你跑得多快。

唯一的要求是：你必须以 **100%的冲劲** 去跑。

在我们的舒适社会中，没有真正的死亡恐惧。正因如此，人们很少发挥自己的最大能力。今天改变了这一切。今天你体验了逃命的感觉。

只有当死亡将我们带离生活的世界时，我们才能充分欣赏生命。今天，你能够欣赏你的身体所能做的事情。今天，你不得不承认，你可能在其他领域并没有那么努力。

如果你能够像刚才所说的那样去冲刺，你还能从哪里得到更多的能量储备？去阅读、去爱、去领导……所有的一切。

**说“你付出了一切”很容易，实际上“付出你所有的一切”是很困难的。**

以下是本课的核心行动指南：
*   执行一次全力以赴的奔跑，感受生理极限。
*   在奔跑中，体验并欣赏身体的能力与生命的活力。
*   反思此次挑战，对比自己在其他领域是否同样竭尽全力。

致力于开发一种很少有人能够复制的性能水平。培养以最大限度发挥自身能力为中心的良好生活习惯。

你有更多的潜力可以给予。当你付出更多时，你也会得到更多。

本节课中，我们一起学习了如何通过极限奔跑来打破舒适区、激发潜能，并以此为契机，将全力以赴的态度贯穿到生活的各个领域，从而养成以卓越表现为中心的生活习惯。
