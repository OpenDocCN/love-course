# 课程19：家人唯一的希望就是你自己 👨‍👩‍👧‍👦

在本节课中，我们将探讨个人责任与领导力在家庭关系中的核心作用。你将理解到，改变始于自身，你的行动和标准直接塑造了家人的行为与家庭氛围。

上一节我们讨论了头衔与认可需通过努力获得。本节中我们来看看，这一原则如何更深刻地应用于家庭与亲密关系之中。

人们必须通过工作来获得你的时间和注意力的好处。与此同时，你必须表现出色才能获得与你所担任的职务相关的好处。你不能仅仅因为你是某人的男朋友或丈夫，就自动享有亲密关系中的所有益处。

你必须采取一些必要的行动。以下是几个关键的行动方向：

*   **采取主动**：例如，主动创造亲密和浪漫的时刻。
*   **保持吸引力**：不要做一个粘人的家伙，不要让你的女人对你的兴趣比一朵花更快干枯。
*   **享受生活**：不要忘记享受生活的乐趣，生命太短暂了。你不能不享受作为男人、兄弟和儿子的角色，拥抱生活即可。

你的家人和女人的表现符合你所接受的标准，而不是你所期望的。他们要以你为榜样，听从你的建议。表现的负担总是落在男人身上。

如果你在第19天还在坚持，那么很明显你已经决定采取必要的行动来改变你现在的处境。你的家人绝对需要你迈出第一步，因为你是一个容器，他们就像液体，成为你为他们设定的形状。

这就是为什么只和一个充满爱的男人在一起的女人，会表现得像个色情明星一样，得到她们应得的。如果你的家人和女人没有履行你的期望，那就不要把你的愤怒发泄在他们身上，然后来建立怨恨。

看看你的内心，问问自己是否允许这种情况发生。

结束第19天，你不能总是掌握你自己的船。你的生活、你的领导力要求你经常在这条船上走动，帮助那些需要帮助的人，并根据需要进行纠正。

如果你的船员不知道你要去哪里，他们怎么能支持你？以下是作为领导者需要做的几件事：

*   **展示方向**：给他们看一看地图。
*   **解释标准**：向他们解释你的期望和标准。
*   **培养自主性**：看看他们是如何在不被告知的情况下，做出你想让他们做的决定的。

这就是整个家庭共同成长的方式。他们的领导者设定了路线，他与他的领导的人分享他的使命。作为一个部落，他们朝着成功前进。

引导你的家庭，提高他们自己的标准。向他们展示地图，解释他们如何能够做到这一点。你可以在脑海中看到自己想要表达的境界，分享你的愿景。

**本节课总结**

本节课中我们一起学习了，在家庭中，真正的改变始于你自身。你通过设定高标准、采取主动行动并承担领导责任，来塑造家庭环境。家人的表现往往反映你内心所接受的标准。作为领导者，你需要清晰传达愿景、解释标准并帮助家人成长，从而带领整个家庭朝着共同的目标前进。核心在于：**你是容器，家人是液体；容器的形状决定了液体的形态。**
