# 第二十一天：建立你的人际网络_note

在本节课中，我们将学习如何有意识地构建和维护一个强大的人际网络。上一节我们探讨了审视自身知识技能缺陷的重要性，本节中我们来看看如何通过与他人连接来弥补不足、共同成长。

回顾昨天第二十天的内容，它迫使你认真审视自己在知识和技能方面的缺陷。这是你必须定期做的事情，以保持在任何潜在的不幸之前做好准备。

就我个人而言，我的车里就经常备有急救箱、刀、手电筒和其他的一些汽车工具。这些东西在我移动的时候会派上用场。你永远不知道什么时候一个孩子会在操场上受伤，而随手可得的药膏和创可贴甚至可能是挽救他生命的关键。这种心态应延续到生活的方方面面。

你永远不知道什么时候会有暴风雨来袭、洪水或者是暴雪等等。你必须努力成为拥有各种技能和装备的人中的“瑞士军刀”。请记住：**拥有而不需要，比需要而不拥有更好**。

## 站在巨人的肩膀上

那些“巨人”站在他们前面的人的肩膀上。如果没有别人的帮助，没有人能够独自完成任务。

大多数人都不愿意接触别人，因为社会已经给人与人之间的沟通带来了耻辱。沟通的艺术似乎正在消亡。

为了证明这一点，问问自己：你最后一次进行有意义的谈话是什么时候？这应该是与家人、你的朋友、你的伴侣等等的常态。但是随着人们固守自己的小想法，害怕触碰另一个小想法以免自己的想法破裂，这种情况变得越来越少见。

## 主动伸出你的手

把它带回来。伸出手，和其他人进行这样的对话。创建一个由积极的个人组成的专业和个人社交网络。

以下是构建人际网络的核心行动指南：

*   **与强者为伍**：曾经和比我强的人一起举重，和比我快的人一起跑步，和比我聪明的人一起学习。
*   **明确目标**：我不想成为最弱、最聪明的，或者是最强壮的。我想找到“师父”，和他们一起奔跑。

---

本节课中我们一起学习了建立人际网络的重要性。核心在于主动与那些在某些方面比你优秀的人连接，并从中学习和成长。记住，构建网络不是为了索取，而是为了创造一个能互相支持、共同提升的生态系统。
