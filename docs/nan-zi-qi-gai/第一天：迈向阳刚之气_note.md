# 课程一：迈向阳刚之气 🚀

在本节课中，我们将要学习如何通过纪律与承诺，开启一段回归男性本质的旅程。课程内容基于军事训练的原则，强调自我负责、行动一致以及目标设定，旨在帮助你培养被长期压抑的阳刚之气。

## 概述

第一天的课程旨在介绍整个挑战的基本框架与核心原则。我们将探讨阳刚之气的含义，以及如何通过严格的纪律和自我约束来实现个人转变。

## 军事训练的原则

上一节我们介绍了课程的整体目标，本节中我们来看看军事训练如何为我们的旅程提供借鉴。在新疆部队服役两年的经历表明，队内训练虽然严苛，但能培养出高度的纪律性和一致性。

你的每一个动作都要与他人保持一致，步调必须统一。动作的严谨和精确建立在团队贡献的基础上。即使过程痛苦，当你展示出训练成果时，你会感到自豪。

## 本月的核心目标

进入这个月，你只需要专注于一件事：改造自我，并致力于表达真实的男性自我。这种自我在多数情况下被压抑，或在许多人身上从未得到培养。

我的原则是**零容忍**。这意味着如果你违反了规定，你就无法对自己负责。

这些课程不同于你常见的心灵鸡汤。它们不会宣扬“每个人都是胜利者”这种观点。真正的转变需要纪律，而这正是当今社会所缺乏的。如果你认同这一理由，那么就没有任何借口。

## 你的责任与行动

这是我的计划、我发起的挑战和课程。你需要做的只是实践对自己负责，并遵守简单的规则。

正如我当兵时大队长常说的：**“闭嘴去做，总有一天你会知道为什么。”**

## 今日课程要点

今天的课程主要是介绍、解释，以及在你自我回归道路上的第一步。你已经在31天的自我回归挑战中迈出了第一步。

你需要理解以下几点：

以下是设定节奏与目标的要点：
*   你为这次挑战设定节奏。
*   你决定自己的目标是什么。
*   你判断这些目标是否合理。
*   你不是来评判或讨论该做什么的。

阳刚的男人独自走在自己的道路上。你是你自己的评判者。明天的课程会更加个性化。请记住，你与你过去所面对的任何事情都不同。

## 课程目的与展望

今天这节课的目的，只是带你参观一下“战场”。你现在已经知道对你的期望是什么，以及你将通过坚持获得怎样的自我。

**完全的承诺是必要的**。如果你能坚持到最后，那么一个最真实、最无需道歉的男性生活，将成为你的奖品。

你将不再生活在“供养者”的心态中，而是生活在一种阳刚、男子气概的心态里。这种心态能让你沉浸于当下，体验真正的快乐，因为你在做自己想做的事，而不是别人期待你做的事。

## 总结

本节课中，我们一起学习了如何以军事纪律为蓝本，开启培养阳刚之气的旅程。我们强调了自我负责、目标设定和完全承诺的重要性，并明确了你不是为了评判而来，而是为了行动和转变。记住，真正的改变始于坚定的决心和持续的行动。
