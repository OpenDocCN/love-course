# 课程22：奖励失败 🏆

在本节课中，我们将学习“奖励失败”这一核心概念。我们将回顾前一天的课程要点，并深入探讨如何通过评估与行动来实现自我提升。课程将引导你审视自身进展，并坚定地走向蜕变之路。

## 回顾与核心原则

上一节我们介绍了如何通过接触优秀的人来提升自己。本节中，我们来看看如何具体评估与行动。

第二十一天的设计是为了帮助你意识到，你无需在所有事情上都做到最好。你只需要和那些能让你在软弱之处变得强大的人在一起，接触那些已经证明了自己优点的人。

如果他们拥有你想要的东西，就和他们谈一谈。看看他们是否能帮你实现同样的目标。不要根据另一个男人的感情状况或年龄来做决定。一个离婚或者比你年轻的人，可能就是那个可以帮助你解决婚姻、财务等问题的人。

你需要确保你判断的是男人的价值，而不是他们的关系状态。你联系的那几个人，一定要确保他们的时间是值得的。如果他们回复了，就花点时间读读他们说的话，然后去行动。

没有什么比把时间和精力投入到阅读建议却无所作为的人身上更糟糕的了。

## 第二十二天：转折与挑战

第二十二天是第二个转折日。今天，我认为我们已经失去了最后一批无法通过这个项目的兄弟。他们要么自己手淫，不自律；要么不注意每一条规则；要么因为缺乏坚持几天以上的自律而放弃。

如果你们完成了今天的挑战，我很有信心你们会完成这段路程并且脱胎换骨。恭喜你们，继续前进，我们就要快到了。

## 自我评估清单

以下是你在过去三周需要审视的几个关键问题：

*   有多少人在这三周内没有达到你设定的标准？
*   你的女人遵守你的规则了吗？
*   你的女人听你的了吗？
*   你的朋友是在挑战你还是让你感觉更舒服？
*   你有你想要的性生活吗？
*   你的工资够花吗？

## 面对变化与抉择

在你加入这个课程的那一天，你就应该知道事情将会发生变化。它正在发生变化。你需要向那些仍然像第一天那样对待你的人表示感谢。

结束第二十二天时，请记住：当你不是一个真正的男人的时候，这个项目的目的是拯救你，而不是你的婚姻。如果你和你的女朋友或妻子是按照她和“蓝色药丸”世界统一的一系列规则进行的，那么现在你已经恢复了男子气概。

不能保证她不会抵触过去三个星期已经发展出来的你。这个星期，她应该对这些变化做出积极的回应。如果她不是，那就由你来决定成为一个真正的男人。

这些都是艰难的时刻，但是你必须面对它们。肩膀向后，眼睛向前。你永远不能回到那个遵守并生活在社会为他创造的模式中的人。

**核心公式：真正的男人 = 面对现实 + 坚持原则 + 持续行动**

你是你。你是活生生的，你热情洋溢，你感觉着每一种情感，去沉浸在其中。
