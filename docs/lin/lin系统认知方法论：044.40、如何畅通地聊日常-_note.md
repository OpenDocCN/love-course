# Lin系统认知方法论：044.40、如何畅通地聊日常 💬

在本节课中，我们将要学习如何有效地收集社交资源，以及如何进行自然、不尴尬的日常聊天。课程将分为两个主要部分：资源收集的核心策略和日常聊天的基本原则与技巧。

## 第一部分：资源收集 📱

上一节我们介绍了课程的整体目标，本节中我们来看看如何高效地收集社交资源。资源收集是建立联系的第一步，选择正确的平台并展示良好的个人形象至关重要。

### 1. 选择合适的社交软件

资源收集现在最好的渠道是软件。不同的软件对应着不同的社交属性。

以下是两种主要推荐软件及其属性：

*   **积木**：属性偏快节奏，活动多为喝酒、玩、参加剧本杀等活动。
*   **牵手**：属性偏长期恋爱。

其他相亲软件（如Solo、青藤等）不推荐，因为上面靠谱的资源较少。如果你偏年轻，没那么想稳定，就去积木。如果你想稳定一点，就去牵手。

### 2. 核心：优化个人照片

这两个软件最重要的因素是照片。照片相当于你在网上的个人展示，虽然制作成本高，但效果最好。

以下是关于照片的核心要点：

*   **路径清晰**：第一步是找对软件，第二步是做好照片。
*   **要求不高**：做好照片不需要很高的要求。如果你处于收集资源的迷茫阶段或收集不到资源，可以寻求帮助。
*   **基础即可**：解决穿搭问题，然后进行自拍，这个基础就可以了。因为大部分人的照片和生活质量并不高。
*   **避免被无视**：在互联网上，大部分男生处于劣势地位。女生会无视一部分人，我们要尽可能避免成为被无视的那一批。

**重点在于照片**。照片如果有问题，可以随时寻求建议。只要把这几步解决好，资源收集基本没有问题。软件很现实，你的照片能吸引什么样的女生，你就会匹配什么样的女生。

资源收集不要去想搭讪或其他方法。建议直接把精力放在照片上，通过照片结识更多人，这是最高效的方式。

## 第二部分：日常聊天 💭

讲完资源收集之后，就是日常聊天。日常聊天大家只需要记住一点：不要被技巧带偏。

### 1. 基本原则：说人话

最重要的还是尽可能说人话，不要套用技巧。因为技巧的使用场景很少。过于套路的聊天在2022年的市场上基本行不通，特别是面对比较优秀的女生。

优秀女生经历的男生比你多得多。她看到套路化的聊天，会直接判断你为段位不高的渣男。所以千万不要模仿套路聊天，日常一点即可。

什么叫日常一点？就是不要管聊天有不有趣，是不是直男或不幽默。只要保证你的态度和情绪到位就行。想象你跟这个女生在面对面沟通，不要说奇奇怪怪的话。

### 2. 聊天重点：交代有效信息

在这个过程中，重点是要把你的有效信息交代清楚。

以下是需要交代的有效信息示例：

*   你的工作是什么样的。
*   你的生活是什么样的。
*   你的职级。
*   你平时跟什么人打交道。
*   你平时喜欢做什么。

只有你在聊天中加入这些东西，你这个人才会更加立体化，给女生留下的印象才会更深。在这个基础上，再去优化和展示自己的优点特点。这样就把日常聊天中不知道怎么聊的部分聊过去了。

聊天中一定会出现空档期，即不知道该怎么聊了。这时就是去分享你生活里的工作，通过这些细节展示你是什么样的人。把这些东西加入聊天过程，虽然聊天会平淡一点，不那么幽默有趣，但这并不重要。

### 3. 判断女生意图与反馈

这时你要去判断一个点：女生是冲着什么来的。如果她是为了吃饭或随便找个男生陪，她大概率不会关心你的工作、职级或生活喜好。

你只需要做这个点的展示就好了。把这个点加入日常聊天，你就会明显看到女生对你的反馈和兴趣。比如她回得多回得少，态度是热情还是冷淡，会不会无视或接你的话题。这就是女生最真实的反馈。

### 4. 根据反馈采取行动

这时我们会发现反馈有好有坏。

**当反馈好时**，我们应该做以下事情：

*   持续展示自己。
*   了解对方。
*   表达好感。
*   多联系，做互动。
*   建立邀约。

**当反馈坏时**，很多兄弟会选择用套路或话术。但最大的问题是，本来反馈就不好，再用技巧性的东西，很容易拉崩关系，连朋友都做不了。

反馈坏时，我们主要要在自己身上找原因，并进行展示。

以下是反馈不好时的行动步骤：

1.  **优点展示**：主动展示自己的优点（如才华、身高、职业等）。如果对方反馈冷淡或觉得你在装逼，说明她对你没兴趣或你的优点不是她看重的。
2.  **价值展示**：展示你能为她做什么。如果你什么都提供不了，那这个女生你可能拿不下。
3.  **了解对方需求**：如果优点和价值展示都无效，可以摊牌做朋友，不暴露好感，去了解对方想要什么样的人。这也能为你自己的提升提供方向。
4.  **了解对方想法（关键）**：大部分人都不会做这一步。但去了解一个拒绝你的、可能比你优秀的女生的想法非常重要。这有助于你跟比你优秀的人打交道，抓住对方的需求和想法，提升自我价值。

### 5. 日常聊天举例

在日常聊天前期，你就去说：你是做什么工作，你是什么职级，你平时跟什么人打交道，你平时喜欢做什么。

例如，你可以给女生发信息：“我现在喝着咖啡听课呢（附照片），你在干嘛呢？” 如果女生回“我无聊呢/发呆了”，你可以继续开启话题，比如“其实我平时不太喜欢听课，觉得容易心情烦躁”。这时观察她是否接这个话题，就能判断反馈好坏。

如果你主动开启话题（分享状态+询问对方），对方回应后，你再开启一个话题去展示，但对方还是不搭理时，你就要知道这叫反馈坏了。这时你再去尝试优点展示、价值展示，了解对方的需求和想法即可。

## 第三部分：核心心态与学习顺序 🧠

在课程的最后，我们强调几个核心心态和正确的学习顺序。

### 1. 切忌急躁与饥渴

千万不能急，不能饥渴。在中国，急躁和饥渴对妹子就代表你是一个很low的男人，会被归为屌丝一类。这是一个恶性循环：越久不恋爱，越饥渴；越饥渴，越恋不了爱。

### 2. 坚定不移地优化照片

一定要搞好照片，千万不要摆烂。搞不好就寻求帮助。照片是一个人设，是你放在朋友圈里最重要的东西。再差，一个月也要出两张好的照片（例如每月15号和30号各发一张）。其他时间可以穿插日常分享。

### 3. 正确的学习顺序

学习的正确顺序是：

1.  **照片**：这是入门券。没有照片，就等于没有拿到考试的资格，却在研究怎么考试。
2.  **建立认知**：通过跟不同女生接触、认识、交朋友，达到认知的理解。
3.  **聊天**：在前两者的基础上，再进行聊天技巧的学习。

## 第四部分：答疑与补充 💡

本节中，我们针对一些常见问题进行解答和补充说明。

### 关于朋友圈展示

女生非常看重你是什么类型的人，她对你信息的了解程度重视度非常高。所以你必须在聊天中和朋友圈里，把自己是什么样的人清晰地介绍清楚。

女生不喜欢查户口，但必须了解你才会发展关系。你要扪心自问，是否在聊天和朋友圈里交代清楚了“自己是谁”。因为面对一个未知的东西不可能产生吸引，一定是一个明确的东西才会产生吸引。

展示自己，就是展示你的生活、工作、日常、吃喝玩乐。女生会通过这些生活细节来判断你是什么样的人。你需要不断提供关于你是个什么样的人的信息，供对方判断。

### 关于关系定位与“渣男”

*   **认识很久的女生还能发展成女朋友吗**：可以。但你要向对方表明你的想法，看她对你的表明有什么反馈。例如，你可以说“感觉你性格还挺合适的，咋还一直单身”，女生大概率懂得起你想发展关系，然后看她的反馈。
*   **老实人和渣男的区别**：没有具体的说法。老实人可能只是没有其他“牌”可打，所以只能打“老实”这张牌。渣男是有很多“牌”。只要不去骗感情，就不算渣男。有些女生自己是短期属性，在一起后却骂男生是渣男，这可能只是想甩锅或立牌坊。

### 如何判断女生为什么选择你

主要是看她关注什么内容、什么话题，她的生活是什么样的，她想要什么生活。如果她不愿意告诉你，你就去猜，在聊天中不断去验证。比如你觉得她看重钱，就去聊关于钱的话题和理念，然后不断判断验证。人都会隐藏，但不可能一直藏，你可以慢慢问慢慢聊。
