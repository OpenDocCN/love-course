# 02、《吸引力》：吸引力篇第八章.变帅_你蜕变的开始·身材塑性_note

在本节课中，我们将学习吸引力构建的最后一个关键板块——身材塑型。我们将明确塑型的目标，了解肥胖对外形的具体影响，并掌握一套结合运动与饮食的、可持续的健康管理方法。

---

上一节我们介绍了整体外型打理的两个板块。本节中，我们来看看构成外型吸引力的最后一项核心要素：身材塑型。

这里所说的身材塑型，并非要求你练出强壮的体格或八块腹肌。**其核心目标是保持一个体脂率较低、健康的正常体重。**

判定体重是否正常的一个关键标准，是你的脸型轮廓。具体表现为脸宽、颧骨、下颌、下巴的线条能否分明。当体脂过高导致肥胖时，最明显的标志就是脂肪会迅速堆积在脸部，形成圆脸或椭圆脸。无论男女，无论天生颜值多高，“一胖毁所有”都是成立的。

变胖前后的男女，会判若两人。具体表现包括脸部皮肤下垂、氧化发黑、鼻子视觉上变塌、双眼无神。女人过胖会失去稚嫩、性感和妩媚。男人过胖则会失去雄性吸引力，显得衰老且没有战斗力。

无论男女，变胖以后，纵使有神仙颜值也难以挽救。因此，培养运动锻炼和身材塑型的习惯，目的不是练成八块腹肌或健美运动员的身材，而是让你长年保持健康体重，拒绝肥胖。

好的身材能让穿衣更立体，走路坐姿等约会中的肢体语言，也能更好地呈现男人本身的阳刚和自信。你愿意为你喜欢的人，真正付诸行动去改变和提升自己吗？变帅是你蜕变的开始。

## 一、运动方案：双管齐下

控制体重、进行身材塑型，推荐运动加饮食双管齐下，可以事半功倍。

以下是运动方面的具体建议：

*   **项目选择**：有氧运动推荐**跑步**，无氧运动推荐**哑铃**。这两种是既省钱又能随时随地进行的个人锻炼项目，并且经过长期实践验证，减脂效果最佳。
*   **场地扩展**：如果你的居家面积较大，可以入手瑜伽垫，进行俯卧撑以及腹肌训练。
*   **计划制定**：你必须制定一个符合自身情况的一周锻炼计划表。例如：周一、三、五锻炼哑铃，周二、四、六跑步，周日休息。采用有氧和无氧交替进行的方式。
*   **时间投入**：正常情况下，锻炼哑铃每次20分钟，跑步每次40分钟。每天只投资不到一小时给自己的身体。
*   **效果与改变**：持续一个月，你会意外发现自己的身体和头脑恢复精力充沛的状态。持续三个月以上，随着身体养成运动锻炼的好习惯，你的心态也会转变，更加自律、外向、热情、自信。坚持锻炼的男人，性格往往更年轻、有激情、有战斗力。由于运动增强了新陈代谢，加速了血液循环，还能帮助你的脸部和身体皮肤抗衰老。

那么，具体如何锻炼呢？如果你的目标是为了练出十分健美的肌肉，建议找到专业教练指导。但如果你的训练目标是为了减脂和让穿衣更立体，可以参考我个人的训练方法和心得。

## 二、饮食策略：三分练，七分吃

在饮食方面，核心原则是：运动量和饮食量应该保持正比。

以下是具体的饮食执行策略：

*   **营养侧重**：减脂期应多摄取**维生素**，增肌期应多摄取**蛋白质**。
*   **动态调整**：当天运动量大，饮食摄取就多一些，以补充身体所需营养。当天运动量小，饮食摄取就少一些，以减轻身体代谢负担。具体而言：
    *   进行**无氧运动**当天，多补充蛋白质。
    *   进行**有氧运动**当天，多补充维生素。
    *   **不运动**当天，减少进食量。
*   **制定计划**：应根据你自行制定的锻炼计划表，来制定对应的饮食计划。

**不推荐节食、不吃晚饭等低效且痛苦的饮食方案。** 这类方法虽然可能让你在一个月内瘦下4公斤，但副作用巨大：你减掉的不仅是脂肪，还有肌肉。这会导致面黄肌瘦、营养不良、精神抑郁、头晕低血糖。最严重的是，一旦恢复正常饮食，体重会成倍反弹，因为长期节食削弱了身体正常的代谢功能，导致摄入的食物更容易转化为脂肪。

那么，最好的保持不胖的饮食策略是什么？答案是：**挑食**。这里的“挑食”并非指只吃喜欢的食物，而是用新的食谱替换你原本认知的早、中、晚餐。

无论你正在减脂或增肌，都需要一个高营养、饱腹、不易肥胖、不易反弹的饮食方案。

以下是三餐调整建议：

1.  **早餐与午餐**：这是白天工作和消耗能量的来源，不得不摄取碳水（碳水化合物，即米饭、面条、馒头等主食）以保证体力。但碳水含大量糖分，也最容易导致肥胖。如果你现阶段肥胖，务必少吃碳水，将减少的碳水份额换成维生素和蛋白质。这能保证饱腹感，同时加速燃脂和消化代谢。长期坚持，身体代谢加速，会逐渐变成“易瘦体质”。
    *   **早餐示例**：将面条、米粥、米粉等高糖分主食，换成鸡蛋、牛奶、豆腐、玉米、胡萝卜、黄瓜、鸡肉、牛肉等以蛋白质和维生素为主的食物。
    *   **午餐示例**：以控制并减少碳水摄取为主。有条件的，可将米饭、面条换成土豆、紫薯、玉米、糙米。选择非油炸的鱼肉、鸡肉、牛肉等脂肪较少的优质肉类。

2.  **晚餐**：夜晚身体代谢减慢，且不需要大量工作，因此应坚决抵制碳水摄入。
    *   **建议**：晚餐不吃任何碳水或主食，只吃以维生素和蛋白质为主的食物。
    *   **独家秘籍**：能让你晚餐进食且不发胖的秘诀是**自制生榨果蔬汁**。新鲜蔬菜搭配水果，未经高温烹煮，能加倍保留植物维生素，帮助肠胃刷出油脂并加速消化代谢。这是经过测试验证的既有效、又健康、低成本的晚餐瘦身食谱。
    *   **额外益处**：坚持一个月饮用自制新鲜果蔬汁，所补充的维生素还会改善脸部皮肤，恢复弹性，起到抗氧化和抗衰老的作用。此外，学会自制果汁，除了自身受益，邀请女生来家里品尝你亲手鲜榨的果汁，也是一个非常不错的社交理由。

关于果蔬汁的制作十分简单便捷：将各种蔬果**切成小块**，放入榨汁机即可。
