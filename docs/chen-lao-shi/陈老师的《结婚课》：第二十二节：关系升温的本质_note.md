**公式**：关系升温 = 情绪提升 + 感性体验

**误区**：通过微信聊天确定男女朋友关系

**注意事项**：在聊天阶段，不要试图通过暧昧话题或亲密称呼来拉近关系。

2. 在女生非常喜欢你，渴望见你时，偶尔使用亲密称呼可以提升她的情绪。
