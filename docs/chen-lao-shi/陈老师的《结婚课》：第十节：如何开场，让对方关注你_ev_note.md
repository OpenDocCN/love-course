# 陈老师的《结婚课》：第十节 如何开场，让对方关注你 🎉

## 概述

在本节课中，我们将学习如何在线上或线下与陌生人开场，让对方产生兴趣并愿意继续交流。

## 第一阶段：开场白的重要性

**核心概念**：开场白是建立第一印象的关键。

以下是开场白的一些要点：

- **简单大方**：开场白应简单直接，避免过于复杂或含糊。
- **刻头**：打招呼并询问对方如何称呼，这是基本的礼貌。
- **身份**：陈述自己的身份，让对方了解你是谁。

## 第二阶段：避免开场白中的禁忌

**核心概念**：以下行为应避免：

- **夸女生**：夸奖可能会让对方觉得你有意，从而产生压力。
- **刻意找话题**：避免明显的找话题，以免让对方感到不舒服。
- **聊感情问题**：避免一开始就聊感情问题，以免让对方感到不适。

## 第三阶段：开场白的模板

以下是一个开场白的模板：

1. 打招呼：“嗨，你好！”
2. 刻头：“我是XXX，很高兴认识你。”
3. 询问对方：“你叫什么名字？”
4. 陈述自己的身份：“我是XXX，在XXX工作。”

## 第四阶段：案例分析

以下是一个失败的案例：

- **错误的开场白**：“你好，名字真好听，兄弟们觉得呢？”
- **女生的反应**：女生可能会觉得你在找话题，从而产生反感。

## 第五阶段：正确的开场方式

**核心概念**：以下是一些正确的开场方式：

- **简单直接**：“嗨，我是XXX，很高兴认识你。”
- **询问对方**：“你叫什么名字？”
- **陈述自己的身份**：“我是XXX，在XXX工作。”

## 总结

本节课中，我们学习了如何在线上或线下与陌生人开场，让对方产生兴趣并愿意继续交流。正确的开场方式是简单大方，避免禁忌，并陈述自己的身份。希望这些技巧能帮助你更好地与人交流。
