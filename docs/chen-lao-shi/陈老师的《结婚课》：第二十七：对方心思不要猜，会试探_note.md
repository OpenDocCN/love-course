# 陈老师的《结婚课》：第二十七课 - 对方心思不要猜，会试探

## 概述
在本节课中，我们将学习如何通过测试来判断女生是否对你有好感，以及如何避免在追求过程中出现尴尬局面。

## 核心概念
**公式**：女生同意程度 = 测试结果 × 配合度

## 一、女生心思不可猜，要测试
女生的心思难以猜测，因此需要通过测试来了解她的真实想法。

## 二、测试的重要性
1. **避免捅破窗户纸**：男女关系中，不宜直接挑明，以免女生因矜持或害怕而拒绝。
2. **可进可退**：通过细小的动作试探，既可推进关系，又可避免尴尬。

## 三、测试方法
以下是几种常见的测试方法：

### 1. 走路试探
- **动作**：在散步时，故意靠近女生，轻微碰撞或触碰。
- **判断**：如果女生不排斥，甚至回应碰撞，说明她对你有好感。

### 2. 看电影试探
- **动作**：将手放在扶手上，将肘部伸向女生方向。
- **判断**：如果女生躲避或远离，说明她不喜欢你；如果她靠近，说明她对你有兴趣。

### 3. 喝饮料试探
- **动作**：买两杯味道不同的饮料，邀请女生品尝。
- **判断**：如果她愿意尝试，说明她对你有好感；如果她排斥，说明她不喜欢你。

### 4. 薯条测试
- **动作**：将薯条递给女生，看她的反应。
- **判断**：如果她接受，说明她对你有好感；如果她拒绝，说明她不喜欢你。

## 四、总结
本节课中，我们学习了如何通过测试来判断女生是否对你有好感，以及如何避免在追求过程中出现尴尬局面。记住，不要盲目猜测，要通过测试来了解女生的真实想法。
