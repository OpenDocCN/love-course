# 陈老师的《结婚课》：第十八节：7次沟通对话过程要学会变通 🎉

## 概述
在本节课中，我们将学习如何在7次沟通对话过程中灵活变通，以增强与女生的互动和吸引力。

## 7次沟通对话过程
我们知道，一般来说，沟通大约需要7次左右。每隔一天聊一次，大约7次。那么，这7次天该如何聊呢？

### 聊天量和节奏
聊天的量和节奏是关键。上一节我们已经讲过这一点。

### 具体聊天方式
以下是具体如何进行7次沟通对话的步骤：

1. **第一天**：打招呼，进行初次聊天。
2. **第二天**：发送表情，进行客套聊天。例如，发送一个打招呼的表情，然后聊天。
3. **第三天**：直接发送图片，例如美食图片，作为开场。
4. **第四天**：发送运动相关的图片，如篮球场、游泳馆等。
5. **第五天**：发送饮料相关的图片，如星巴克、喜茶店等。
6. **第六天**：发送“在干嘛”的图片，关心对方。
7. **第七天**：发送天气相关的图片，如天空照片，表达自己的活动。

### 核心概念
- **公式**：每次聊天都包含图片和简短文字。
- **代码**：`图片 + 7句话`

## 变通方法
在实际聊天中，我们需要学会变通。以下是一些变通方法：

1. **表情开场**：在初次聊天时，可以使用表情包进行开场。
2. **“在干嘛”开场**：在关系进展到一定程度后，可以使用“在干嘛”进行开场。
3. **新情天气开场**：根据天气情况，发送相应的图片和文字。
