# 陈老师的《结婚课》：第六节：第一阶段：相识

## 概述
在本节课中，我们将学习如何从相识阶段开始，建立良好的第一印象，并逐步推进关系。

## 第一阶段：相识

### 低印象到基础印象
从0到1，即从低印象到基础印象的阶段，是建立关系的关键。万事开头难，一个成功的开始就成功了一半。

### 理解第一印象
一个成功的开始意味着你的印象要在女生可接受的范围内。如果一开始印象不好，最好的办法是先抽离，过段时间再重新接触。

### 如何达到可接受范围
1. **外型打扮**：在社交场所或相亲会上，你的外型和打扮要过关。
2. **朋友圈**：在网上，你的朋友圈要吸引人。
3. **介绍人**：如果通过朋友或相亲介绍，介绍人要靠谱。

### 第一阶段原则
1. **不背不抗**：主动接触，不故作高冷。
2. **无欲**：不要让女生察觉出你对她的兴趣。

### 无欲的重要性
女生看重感觉，一上来就表现出追求的意图会减分。女生不想浪费时间在无关紧要的人身上。

### 接近女生的方法
1. **线下**：主动接触，不故作高冷。
2. **线上**：主动聊天，保持无欲状态。

### 总结
本节课中我们一起学习了如何从相识阶段开始，建立良好的第一印象，并逐步推进关系。希望对你有所帮助。
