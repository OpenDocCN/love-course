# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（值）。在Python中，我们使用等号 `=` 来给变量赋值。

其核心操作是赋值，公式如下：
**变量名 = 值**

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存储在其中。
```python
message = "你好，世界！"
```

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有几个内置的基本数据类型，最常见的有以下三种。

以下是三种最常用的基本数据类型：

1.  **整数**
    整数是不带小数点的数字，例如：`-10`, `0`, `100`。

2.  **浮点数**
    浮点数是带有小数点的数字，例如：`3.14`, `2.0`, `-0.5`。

3.  **字符串**
    字符串是由零个或多个字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如：`‘Python’`, `“123”`。

## 如何查看和使用变量？ 👀

知道了如何创建变量和数据类型后，我们来看看如何查看和使用它们。使用 `print()` 函数可以将变量的内容输出到屏幕上。你可以直接使用变量名进行数学运算或字符串拼接。

例如：
```python
name = “陈老师”
age = 30
print(name)          # 输出：陈老师
print(“年龄：”, age) # 输出：年龄： 30

# 进行运算
price = 9.9
quantity = 2
total = price * quantity
print(“总价：”, total) # 输出：总价： 19.8
```

## 动态类型特性 ⚡

Python是一门“动态类型”语言。这意味着你不需要提前声明变量的类型。同一个变量名可以在程序的不同阶段被重新赋值，指向不同类型的数据。

请看以下示例：
```python
my_var = 100      # my_var 此时是整数
print(my_var)     # 输出：100

my_var = “现在我是字符串” # my_var 被重新赋值为一个字符串
print(my_var)            # 输出：现在我是字符串
```

本节课中我们一起学习了Python编程的基石：变量与数据类型。
我们了解到变量是存储数据的容器，并通过 `=` 进行赋值。
