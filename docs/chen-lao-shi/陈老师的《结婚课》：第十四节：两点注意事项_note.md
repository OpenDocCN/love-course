# 陈老师的《结婚课》：第十四节：两点注意事项 📚

## 概述
在本节课中，我们将学习如何通过发送图片来开启聊天话题，并推进与女生的关系。我们将重点讨论发送图片的目的以及如何拍摄合适的照片。

## 发送图片的目的
发送图片的目的是为了分享自己的生活，让对方了解你的生活状态。**公式**：分享生活 → 了解生活状态 → 增加好感。

### 注意事项
1. **避免误区**：发送图片的目的不是描述图片内容，而是展示你的生活，让对方觉得你不错。**公式**：分享生活 → 展示生活 → 增加好感。
2. **照片拍摄**
   - **避免low的照片**：不要发送廉价、杂乱的照片，以免给人留下不好的印象。**公式**：避免low的照片 → 避免负面印象 → 增加好感。
   - **拍摄角度**：拍摄环境而非特写，以展现真实的生活状态。**公式**：拍摄环境 → 展现真实生活 → 增加好感。

## 总结
本节课中，我们一起学习了如何通过发送图片来开启聊天话题，并推进与女生的关系。通过分享真实的生活，并注意照片的拍摄，我们可以更好地展示自己，增加对方的好感。
