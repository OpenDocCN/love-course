# 陈老师的《结婚课》第十四节：两点注意事项 📝

在本节课中，我们将学习如何通过“发图片”这一技巧来开启聊天话题并推进关系。我们将重点探讨使用此方法时必须注意的两个核心事项，帮助你避免常见误区，更有效地展示自我。

上一节我们介绍了通过发图片开启话题的方法，本节中我们来看看使用这一技巧时有哪些关键的注意事项。

## 第一点：明确发图片的核心目的

发图片的核心目的是**分享你的真实生活**，以此向对方展示你是一个生活丰富、有品味的人。其关键在于传递“你正在经历某种优质体验”的状态，而非单纯讨论图片内容本身。

新手常有一个误区：误以为发图片是为了描述图片上的事物。这是一个需要纠正的观念。

以下是常见的错误示范：

*   **错误示例1**：发送一张旅游景点网图，然后问对方：“这个地方你去过吗？听说很不错。” 这与你本人的生活毫无关联。
*   **错误示例2**：发送一张网络上的帅哥照片，问对方：“你看这个男的是不是挺帅？” 这同样无法展示你的任何特质。

请记住公式：**发图片 = 分享生活状态 ≠ 讨论图片内容**。你的目标是让对方通过图片感知到你的生活，而不是围绕一张可能与你无关的图片展开对话。

## 第二点：掌握照片的拍摄要领

明确了目的后，我们来看看如何拍摄合适的照片。拍摄时需注意两个要点：避免低价值感和追求真实感。

### 1. 避免展示低价值内容

你分享的生活片段应当能为你加分，而不是暴露缺点。切忌发送那些显得你生活品质低下或邋遢的照片。

以下是需要避免的反面例子：

*   **示例A**：发送一份装在廉价塑料盒里的简陋快餐，并配文“我要美餐一顿了”。这只会让对方觉得你的生活很“low”。
*   **示例B**：发送一张自己混乱的宿舍照片，画面里包含脏袜子、杂乱物品等。即使你想展示真实，对方也只会注意到混乱的环境。

### 2. 追求真实而非过度精美

拍摄角度应侧重于营造“真实在场”的氛围，而不是拍得像商业广告或网图。过度修饰和特写反而会显得不真实。

正确的拍摄思路是：

*   **不要只拍物品特写**：例如，在餐厅不要只怼着食物拍大特写。
*   **要拍摄环境与上下文**：将食物与你所处的环境一同拍入镜头。例如，可以拍到高档餐厅的装修、餐桌对面的友人、或是用餐后的杯盘，这些都能自然地透露出你的生活场景和状态，显得真实可信。

本节课中我们一起学习了“发图片”技巧的两大注意事项：第一，始终牢记目的是分享生活状态，而非讨论图片内容；第二，拍摄照片时要避免低价值展示，并追求自然真实的场景感。掌握这两点，你就能更有效地利用图片开启话题，为关系推进做好铺垫。
