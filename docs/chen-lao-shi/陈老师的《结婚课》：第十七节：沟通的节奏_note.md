# 陈老师的《结婚课》：第十七节：沟通的节奏 💬

在本节课中，我们将要学习聊天中除了内容之外，另一个至关重要的方面：沟通的节奏。我们将详细探讨聊天的频率、聊天量、时间安排以及整体节奏，这些因素共同决定了对方与你对话时的整体感受。

上一节我们介绍了聊天的内容，例如聊天单元和话题寻找。本节中我们来看看如何把握聊天的节奏与频率。

## 沟通节奏的重要性

聊天的内容和聊天的节奏、频率、聊天量、时间共同组成了你与对方对话时，对方的整体感受。这个感受进一步决定了女生与你聊天的乐趣、对你的新鲜感以及好奇心。

这些感受最终会转化为她是否愿意了解你、是否愿意继续聊天、是否愿意出来见面的动力。内容方面之前已经讲解过，本节课重点补充节奏、频率、聊天量和时间方面的内容。

## 聊天量：讲究节制

首先讲解聊天量。做任何事情都要讲究节制，聊天量不能多，也不能少，需要合适。

以下是关于聊天量多与少的说明：

*   **聊天量过多**：许多人存在聊天过多的问题。例如，喜欢和女孩子彻夜长谈，或者从早上起床发“早安”开始，一直聊到中午、晚上，最后以“晚安”结束。这种从早聊到晚，或者连续聊一整夜的方式，只要经历两天或两晚，就一定会成为“舔狗”。无论聊天内容多么精彩、愉快，只要在聊天量上如此不节制，两天内就会让对方失去新鲜感和好奇心。这是一种特别“屌丝”的行为，因为这通常意味着你特别饥渴或没见过女人。一个不缺女人的成熟男人，不可能把所有精力都放在一个女人身上，他会有自己的工作。
*   **聊天量过少**：当然，聊天量也不能太少，否则无法建立联系。

那么，什么是合适的聊天量呢？可以参考之前课程中提到的“有效聊天单元”。

**一次合适的聊天量**是：发送一张图片，聊大约七八句话，最多十来句话。这就是一次有效的聊天量，一次聊天到此为止即可，不要再多聊。

## 聊天频率：避免高度频繁

频率和聊天量同样重要。频率是指你不能高度频繁地去聊天。高度频繁的聊天与不节制的聊天量会导致相同的结果：成为“舔狗”。

以下是关于频率的说明：

*   **高频率示例**：例如，每天都和女孩子聊天，甚至早上、中午、晚上像打卡一样聊天。有的“舔狗”会天天聊，坚持三十或四十天。
*   **正常频率**：在铺垫舒适阶段，正常的频率是平均每两天聊1到1.5次（以有效聊天单元计）。不要超过这个频率。

## 聊天节奏：打破规律性

节奏是指你不能在固定的时间点去找女生聊天，需要有所变化。这样女生在和你聊天时会对你有好奇感，而不会觉得“这个男的好像又按点来找我聊天了”。

节奏的关键是**不能有规律性**。例如，今天早上聊，第二天晚上聊，第三天不聊，第四天中午聊。总之，要让对方感觉没有固定模式。说白了，就是稍微随意一点，随心所欲一点，不要显得那么刻意。

## 聊天时间：避开敏感时段

最后说到时间。大家要记住：**早上10点之前和晚上9点之后，不要和女孩子聊天**。

以下是具体原因：

*   **早上10点之前**：如果一大早就和女孩子聊天，女生会觉得这个男人起床第一件事就是找我聊天，这会显得特别“舔狗”。她会认为自己是对方一天中最重要的事。而对男人来说，最重要的事应该是工作、赚钱和事业，不能是女人。当你把女人放在第一位时，你就是“屌丝”，就是“舔狗”。
*   **晚上9点之后**：尤其在铺垫舒适阶段，双方关系还不够亲密，尤其还没见过面时，晚上9点后聊天总有一种“不正经”的感觉，女生体验不好。而且此时女生可能聊两句就要说晚安、休息或洗漱，聊天容易仓促结束。

因此，合适的聊天时间是**早上10点之后到晚上9点之前**。

## 整体应用：铺垫舒适阶段示例

现在，我们回到“铺垫舒适阶段”（通常为认识后的10到20天），从整体上解读如何应用以上原则。

假设以中间值14天为例。从留下良好的第一印象后，进入铺垫舒适阶段。

在这14天里，总共需要进行**7次**聊天。每次聊天就是一个聊天单元，聊7、8句，10句话以内。

聊天节奏大致是**每隔一天聊一次**。

在这7次聊天中：
*   前1到2次聊天可以稍微多聊一点，因为刚开始不太熟悉，需要加深印象。尤其是第一次聊天，尽量多聊一点，可以完成**两个**有效的聊天单元。
*   后面的5到6次聊天，每次完成**一个**聊天单元即可。

在具体过程中，注意观察女生好感度的变化。当你感觉时机差不多时，就可以准备进入升级关系阶段了。

本节课中我们一起学习了沟通的节奏。总的来说，在铺垫舒适阶段，比较恰当的聊天安排是：合适的聊天量（每次一个单元）、正常的频率（约两天一次）、变化的节奏以及合适的时间（早10点后，晚9点前）。请大家务必记住这些要点。
