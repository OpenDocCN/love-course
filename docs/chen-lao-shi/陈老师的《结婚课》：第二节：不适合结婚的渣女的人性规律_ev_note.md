# 陈老师的《结婚课》第二节：不适合结婚的渣女的人性规律 👩💔

在本节课中，我们将要学习不适合结婚的“渣女”在恋爱中表现出的人性规律。上一节我们介绍了靠谱女生的恋爱规律，本节中我们来看看不靠谱女生的行为模式有何不同。

## 核心规律：重力不理性

靠谱的女生是**重情且理性**的，而不靠谱的女生则是**重力且不理性**的。她们的核心驱动力是利益，而非感情。

## 规律一：起点过高与进展过快

以下是渣女的一个典型行为曲线：

*   **起点过高**：她们在关系初期会表现出**过分的热情和主动**，起点远高于正常礼貌交往的水平。
*   **进展神速**：她们可能会在极短的时间内（例如一周内）就推动确立恋爱关系。

这种行为背后的逻辑是：为了快速从目标（尤其是有钱人）身上获取利益，她们会**投其所好**，制造出“深爱”的假象以让对方迅速上钩。这种“喜欢”是演出来的，并非真实情感。

## 规律二：情绪曲线不连续与忽冷忽热

靠谱女生的情感发展是**连续且合理**的，而渣女的情感曲线则可能**不连续**，表现为忽冷忽热。

以下是几种典型的不连续曲线：

1.  **热情后骤冷**：初期极度热情，一旦发现对方已经对自己产生兴趣或开始上头，便突然冷淡下来。这种极致的情绪波动旨在操控对方心理，让对方变成“舔狗”，然后再给予一点甜头进行控制。
2.  **冷淡后突热**：初期明确表示没兴趣、很冷淡，但在得知对方有利可图（如家境优越）后，态度发生180度大转弯，突然变得热情主动。
3.  **来者不拒的低温接受**：虽然对对方本人不感兴趣（兴趣值在零轴线以下），但依然接受对方的聊天、约会和礼物。这是一种较为低级的渣女表现，纯粹图利。

## 渣女图什么？

靠谱的女生图的是感情，而渣女图的是以下三者之一或组合：

*   **图钱**：看重你的经济条件、家庭背景。
*   **图帅**：看重你的外貌。
*   **图傻**：看重你容易控制、愿意付出的“傻”。

总之，核心都是**图利**。

## 核心特点总结

综上所述，识别渣女人性规律可重点关注两点：

1.  **不合理的过度热情**：关系初期的热情程度远超正常社交礼貌的范畴。
2.  **情绪曲线的忽冷忽热与不连续**：情感态度没有合理的发展过程，经常出现突兀的转折和变化。
