# 恋爱新学堂 P1：为什么女生聊着聊着就不理你了？ 🤔

在本节课中，我们将要学习情感课程中的一个核心内容：识别并应对女生在聊天中可能使用的几种“废测”。废测是对方通过言语测试你态度、框架和价值的一种方式。理解并正确处理这些废测，是避免聊天突然中断的关键。

上一节我们介绍了废测的基本概念，本节中我们来看看第四种常见的废测类型。

## 第四种废测：打压条件型

当女生对你说“你太胖了”、“你太矮了”、“你太不会聊天了”或“你太幼稚了”等话时，她可能在进行打压条件型废测。许多男生听到这类打压会急于解释自己，例如反驳说“我哪里幼稚了，我觉得自己挺成熟的”。

面对此类打压，正确的做法不是解释，而是**阐述自己的态度和原则**，并将问题交还给对方。

以下是两个应对示例：

*   **当女生说“你太幼稚了”时**，你可以回复：“有没有可能是我在你面前卸下了防备？”
*   **当女生说“你太矮了”时**，一个有自信的男生可以回复：“我们小区的电线杆就挺高的。不过，男人再高，不懂得和自己的女人低头，也和电线杆没两样。”

核心在于：**不解释，只表达**。通过回应展现你的自信和稳定框架，而不是陷入对方设定的评判标准中。

了解了如何应对外部的打压后，我们来看看一种更隐蔽的测试形式。

## 第五种废测：自我打压型

这种废测与打压条件型原理相似，但形式不同。女生不再打压你，转而开始打压自己。例如，她会说“我感觉我太胖了”、“我最近变丑了”。有时，女生明明不胖不丑，也会这样说。

许多男生听到这种话，会立刻讨好式地安慰：“你不胖不丑，你再胖我也喜欢你，你在我心里永远是最美的。”

我们先不讨论女生是否相信这些话，单是这种回应方式，就**显得你自身价值很低，处于讨好姿态**。

正确的做法是：**将话题引向你们之间的关系，并展现你的自信**。

当女生说自己太胖、太丑或太笨时，你只需要说：“其实在我面前，你可以自信一些。” 不要深入讨论她的“缺陷”，而是把焦点转移到“你和我”的互动上。

不管女生随后回复什么，你都可以接着说：“为什么在我面前你这么没有自信呢？” 这样既能体现你的自信，又能自然地**拉伸你们之间的关系**，这才是对话的目的。

应对了直接和间接的打压，我们最后来看一种最明显但也最危险的测试。

## 第六种废测：备胎型测试

这种废测最为明显，但也最让人防不胜防。女生可能会给出一些关于未来的、充满条件的承诺。

以下是几种典型话术：

*   “如果几年之后我还是单身，我就和你在一起。”
*   “等我累了想找个依靠的时候，我一定会来找你的。”
*   “如果我分手了，你就是我的首选。”

面对此类废测，无需多言。请记住一句话：**如果她真的想找你，现在就会行动，而不是给你一个虚无缥缈的未来承诺。**

不要为对方找借口，例如“她只是现在不想谈恋爱”或“她被前任伤害了才这么谨慎”。无论如何包装，这类话的潜台词都是：**你目前还配不上我，但我希望你先留着，作为我的一个备选。**

自欺欺人只会浪费你的时间和情感。认清本质，及时止损，才是对自己负责。

---

本节课中我们一起学习了三种常见的废测类型：**打压条件型**、**自我打压型**和**备胎型测试**。核心应对原则是：保持自信，坚守自我框架，不陷入对方的话题陷阱，不进行无效的解释或讨好。识别这些信号并正确应对，能有效避免在聊天中陷入被动，从而建立更平等、健康的互动关系。
