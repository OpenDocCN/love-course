# **不再感情的框架内谈感情 - P1 - 恋爱新学堂 - BV1zk4y1V7ap**

## 概述
在本节课中，我们将探讨感情破裂的核心原因，并学习如何在感情中保持独立和自我。

## 核心概念
**公式**：感情破裂 = 感情存在 + 感情淡薄

## 感情破裂的核心
感情破裂的核心在于感情的存在。如果感情本身就很淡薄，那么破裂的可能性就很小。因此，要避免被感情所困，首先要理解什么是感情。

## 框架之外寻找答案
一切的答案都在框架之外，而不是框架之内。在感情的框架内谈论感情是错误的。同样，在事业的框架内谈论事业，在婚姻的框架内谈论婚姻也都是错误的。

## 婚姻的框架
在婚姻的框架内谈论婚姻也是错误的。婚姻的答案不在婚姻之内。换句话说，结婚就像没有结婚一样。两个人不再以夫妻的身份相处，而是以合伙人、朋友或其他身份相处。当夫妻关系变得过于紧密时，就会忘记尊重对方，忘记给对方空间。

## 伴侣的真正意图
男生们，你以为伴侣在跟你谈爱情，其实伴侣在跟你谈事业、财富和社会地位。女生们，你以为伴侣在跟你谈爱情，其实伴侣在跟你谈长相、身材、身体健康和综合价值。

## 价值与关系
抛开这些价值，看看你的关系能坚持多久。一个男人越有钱，他对女人的要求就越高。一个女人无论多么暴躁或公主病，当她遇到一个能力强、有钱的男人时，她也会变成小女人。

## 真实与假象
不要再认为你的老婆或女朋友事多难伺候，这只是假象。如果你的收入是她的10倍、50倍或100倍，她随时可以变成你的女佣。

## 总结
本节课中，我们一起学习了感情破裂的核心原因，以及如何在感情中保持独立和自我。记住，一切的答案都在你看不到的东西之外。
