## 概述
在本节课中，我们将学习如何正确地搭讪，通过三个简单而有效的步骤，让你在社交场合更加自信。

## 步骤一：展现善意
**公式**：友好问候 + 自我介绍 + 表达意图

1. **友好问候**：例如，“哎你好啊。”
2. **自我介绍**：说明你为何想要认识对方。
3. **表达意图**：表明你希望得到对方的联系方式。

**代码**：
```markdown
你好，我叫[你的名字]。我刚刚在那边看到你，非常想认识一下。可以给我你的联系方式吗？
```

## 步骤二：减少压迫感
**公式**：提供选择 + 表达时间限制

1. **提供选择**：例如，“你要是方便的话，看是我扫你还是你扫我。”
2. **表达时间限制**：例如，“我朋友还在那边等我。”

**代码**：
```markdown
我朋友还在那边等我，你要是方便的话，看是我扫你还是你扫我？
```

## 步骤三：增加唯一感
**公式**：握手 + 自我介绍

1. **握手**：握手时只握对方的指尖，最多三秒。
2. **自我介绍**：例如，“你好我叫[你的名字]。”

**代码**：
```markdown
我们现在加上联系方式，算是朋友了吗？你好我叫[你的名字]，你现在知道我的名字了，我们已经跨过了陌生人的阶段。
```

## 总结
本节课中我们一起学习了如何通过三个步骤正确地搭讪。希望这些技巧能帮助你更好地融入社交场合，建立更多的人际关系。
