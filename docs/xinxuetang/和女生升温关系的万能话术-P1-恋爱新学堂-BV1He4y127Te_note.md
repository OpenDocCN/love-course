## 核心原理：双面评价法

上一节我们介绍了课程的目标，本节中我们来看看其核心原理。这套话术的核心在于使用“双面评价法”。这种方法不是简单的赞美或批评，而是通过一个看似矛盾、实则精准的描述，同时指出对方外在表现和内在感受的两个方面。这能让对方感觉你观察入微，理解深刻。

其核心公式可以概括为：**“你表面上看起来是A，但内心深处其实是B。”**
