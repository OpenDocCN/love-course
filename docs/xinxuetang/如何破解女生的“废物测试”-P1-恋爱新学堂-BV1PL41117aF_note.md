## 什么是“废物测试”？🤔

上一节我们介绍了课程主题，本节中我们来看看“废物测试”的具体定义。

“废物测试”通常指在交往初期，一方（通常是女性）通过提出带有挑战性、质疑性或看似否定意味的问题或陈述，来测试另一方的自信、框架稳定性和情绪价值。其目的并非真正的否定或攻击，而是一种筛选机制，用以判断对方是否足够强大、有趣和值得深入交往。

## 如何识别“废物测试”？🔍

理解了定义后，我们来看看如何在实际对话中识别出“废物测试”。

“废物测试”往往伪装成普通的对话或玩笑，但通常具有一些共同特征。以下是几种常见的“废物测试”类型及其示例：

*   **质疑型**：对你个人或能力的直接质疑。
    *   示例：“你该不会对每个女生都这么说吧？”
*   **打压型**：看似贬低你或你的价值。
    *   示例：“你看起来有点幼稚。”
*   **寻求认同型**：通过否定自己来观察你的反应。
    *   示例：“我最近是不是胖了？”
*   **框架挑战型**：挑战你的决定或主导权。
    *   示例：“为什么我要听你的？”

## 应对“废物测试”的核心策略 💡

识别出测试只是第一步，关键在于如何回应。本节中我们将学习应对“废物测试”的通用心法和具体技巧。

核心心法是：**保持框架稳定，不寻求认同，不辩解，不生气**。你的反应比回答的内容更重要。以下是三种有效的应对方法：

*   **曲解与幽默**
    这是最常用的方法。不正面回答测试问题，而是用幽默、夸张或曲解其字面意思的方式化解。
    *   **公式**：`曲解关键词 + 幽默表达`

*   **同意与夸大**
    完全同意对方的测试陈述，并把它夸张到一个荒谬的程度，从而反客为主，展示出你的自信和趣味性。
    *   **公式**：`完全同意 + 夸张延伸`
    *   示例：当被说“你看起来有点幼稚”，可以回答：“没错，我心理年龄大概三岁，所以你得请我吃糖。”

*   **无视与转换话题**
    对于一些低价值的测试，可以直接忽略问题本身，平静地转向一个新话题，这展示了你的筛选标准和不为所动的态度。
    *   **公式**：`简短回应（如“也许吧”） + 新话题`
