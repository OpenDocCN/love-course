## 讨好型人格的定义
**讨好型人格**是指个体在人际交往中过分关注他人的感受和期望，忽视自己的需求和感受，以取悦他人为主要行为动机。

**案例**：作者从小是留守儿童，拒绝他人对他来说非常困难。这种性格导致他在感情中任人摆布，被女生招之即来，挥之即去。他如何从一个桃花型人格成长为现在的情感博主？

作者指出，不要做完美主义者，要主动暴露自己的缺点，避免陷入人设绑架。
