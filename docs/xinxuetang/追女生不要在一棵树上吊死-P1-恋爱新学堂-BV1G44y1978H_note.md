# 恋爱新学堂 第1课：追女生不要在一棵树上吊死 🌳➡️👥

在本节课中，我们将要学习一个关键的恋爱策略：为什么在追求阶段不应只锁定一个目标，以及如何以更健康、更有效的方式进行社交互动。

## 引言：认清现代社交的现实

上一节我们介绍了课程主题，本节中我们来看看一个普遍存在的现象。22世纪了，不会现在还有兄弟追女生只有一个目标。我发现一个很有意思的现象，在很多男人看来，同时和多个女生接触是不道德的。事实上，当你还在犹豫不决时，女生的好友列表当中已经有一堆男孩子了。

## 追求阶段的策略差异

当你终于有个目标的时候，女生已经在她的好友列表当中完成了两轮筛选，和其中几个还不错的男生在加深了解了。当你下定决心，我要全心全意和一个女生接触，并且觉得我要是和第二个女生接触，我就是个渣男。这种想法是真的活该你单身。

本质上大家都是在社交。难道我现在好友列表当中，有很多女生我就是渣男的吗？我跟很多女生聊天我就是渣男的吗？凭啥呀，我只是在交朋友而已。难道现在交朋友都会被喷的吗？只不过我交的是异性朋友罢了。我单身我又没伤害谁。这个女生她不回我我就不理她咯，有啥嘞。

## 用求职比喻理解追求

你们根本没有搞清楚状况。我就问你们，你们当年投简历的时候是只投一家吗？如果这家不给你回应你就一直等吗？或者说你非要等到这家给你回应，你才会去投下一家吗？你为了确保成功率你肯定会多投几家呀。

换到这里他也是一样的。女生也会收到很多递来的简历的。他只不过是把他收到很多简历的事实，给隐藏起来了。他为了自己的名声，他很难去主动跟你说。你认为一个女生但凡有点姿色，他没有很多男人在追他吗？很多天真的哥们就真的认为，女生只和自己一个人接触了。但事实又是怎样的？你不想去筛选对方，你在这里犯傻。而你却不知道在对方的心里，已经把你筛过几次了才把你留下来的。

## 两性在关系初期的核心权利

以下是关于关系初期动态的核心观点，请记住这两句话：

*   **男生的福利**：一段关系初期，男生追求多个女生，形成互动深入了解，最终和其中最合适的女生步入恋爱关系，这本就是男生的福利。
*   **女生的福利**：相反，一段关系初期，女生被多个男生追求，并最终筛选男生，这本就是女生的福利。

而很多的男生他是怎么做的？在追求阶段就把这个女生当做他的唯一。你只有他一个买家，女生他有很多卖家。这就是现实。他是你的唯一，你只是他的二十分之一。你说你怎么可能不暴露需求感，你怎么可能不去舔他。

## 如何建立健康的互动心态

同样的，那如果现在是我，这个女生不回我，但是我的好友列表有一堆。你不回我就晾着你，但我不缺人理他。而你们是在干嘛？他不回我就等他回我消息。你的需求感怎么会不暴露，你告诉我。

作为一个男生，你多和异性接触是很正常的事，而且我觉得你也应该这么去做。如果你们非要说道德，那我觉得应该一视同仁。

以下是假设的、不现实的“道德”规则：
```
无论男女，三个月内你只能追一个人，或者只能被一个人追。三个月之后你才可以换人。
```
但你觉得现实吗？而且我觉得你这样去束缚别人，才是真正的不道德。**确定关系之后，你再来跟我谈专一。**

## 总结

本节课中我们一起学习了在恋爱追求初期的重要心态调整。关键在于理解社交的本质是相互选择和了解，而非单方面的认定与等待。在正式确立恋爱关系之前，与多位异性进行正常、健康的接触和交流，不仅是合理的，也是一种保护自己、避免过度投入导致心态失衡的有效策略。记住，专一适用于确定关系之后，而在那之前，给自己和对方留有选择和了解的空间，才是更明智的做法。
