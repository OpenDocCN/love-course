# 恋爱技巧课 P1：假定成交法 🎯

在本节课中，我们将要学习一种名为“假定成交法”的沟通技巧。这种方法源自销售领域，其核心在于通过“假定某事已成事实”的方式来引导对话，从而减少对方的防备心理，并自然地推动关系或事件向前发展。我们将探讨其基本原理，并通过具体案例展示如何将其巧妙应用于恋爱互动中。

## 什么是假定成交法？

上一节我们介绍了本节课的主题。本节中，我们来看看“假定成交法”的具体定义。

假定成交法，顾名思义，就是**假定一件事已经成为事实**，并基于这个假设进行对话和行动。它避免了直接询问对方“是否同意”，从而绕开了可能引发对方犹豫或防备的决策环节。

## 销售中的假定成交法

理解了基本概念后，我们通过一个熟悉的领域——销售——来观察这个方法的具体应用。

以下是销售场景中的一个典型对话示例：

*   销售员问：“张先生，您觉得这款车中意吗？”
*   客户回答：“嗯，它的性能看起来很不错。”
*   销售员回：“是的，而且这款车内部空间也非常宽敞。我们来确定一下颜色吧，您比较喜欢深色调的是吧？”

请注意，销售员并没有问“您到底买不买？”，而是直接进入了“确定颜色”的购买后续步骤，这就是在**假定客户已经决定购买**。

*   客户回答：“是的。”
*   销售员接着回：“您看要不要给爱车加上导航？”
*   客户回：“那倒不用。你们有赠送什么吗？”
*   销售员回：“有的张先生，这个合同已经注明了，我们一起来看看合同吧。”

在整个过程中，销售员通过连续假定“客户已经要买车”，引导对话自然走向成交，有效避免了客户产生“我要不要买”的防备心理。

## 恋爱中的假定成交法应用

看过了销售案例，现在我们将视角转向恋爱场景，看看如何运用同样的逻辑。

这就好比如果你直接问一个女生“可不可以当我的女朋友”，她会立刻进入谨慎的思考状态。而假定成交法则能创造不同的互动氛围。

以下是一个聊天示例：

*   女生说：“我酒量不好。”
*   男生直接回：“明知道我会灌你。”

这里，男生没有问“要不要一起喝酒”，而是直接**假定了“我们即将要一起喝酒”**这个场景。

*   女生回复：“我还没答应要跟你喝酒啊。”
*   男生直接回：“没答应的原因，有没有可能是没有挑日子？我们现在就把良辰吉日挑上。”

男生的回应继续忽略“是否答应”的问题，转而假定“需要挑日子”，从而挑逗了女生的情绪。

*   女生说：“哈哈，真喝起来我怕你喝不过。”
*   男生回复：“那为了我们的清白，下手轻点。”

这里，男生再次进行了**假定成交**，具体假定了“我们会一起喝酒”以及“喝酒时会有互动”的事实。

通过这个例子可以发现，我们始终没有问女生“可不可以跟我们喝酒”，但对话却围绕着“一起喝酒”的场景展开。如果我们直接问“周末一起喝酒吗？”，女生可能会产生一系列疑问（如“为什么？”“安全吗？”），需要你逐一解决。而假定成交法直接**引领话题，掌控了话题走向**，让女生没有直接拒绝这个提议的机会。

## 方法总结与拓展

本节课中我们一起学习了“假定成交法”的核心与应用。

这种方法不仅可用于邀约，在推进关系（如牵手、拥抱）时也同样有效，其精髓在于通过正向、自然的假定，创造积极的互动框架。总之，合理运用假定成交法，无论是在恋爱、工作还是生活中，都能让你更加**如鱼得水**。
