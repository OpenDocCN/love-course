## 概述

## 核心概念
自尊心（Ego）：个体对自己的评价和看法。

## 教程内容

### 1. 自尊心与自信
自尊心过强或过弱都会影响自信。**公式**：自信 = 自尊心 - 自卑感。

### 2. 自卑感的表现
- 过度关注他人感受
- 害怕被拒绝
- 过分拘束

### 3. 克服自卑感的方法
- 降低对他人期望
- 增强自我认知
- 勇于尝试

### 4. 保持自信的技巧
- 直接表达需求
- 勇于承担责任
- 保持积极心态
