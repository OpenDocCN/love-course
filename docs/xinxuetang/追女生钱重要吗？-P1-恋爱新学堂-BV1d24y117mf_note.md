我做的第一件事是提升外在形象。公式如下：
**外在提升 = 基础护理 + 发型设计 + 得体衣着**
我购买了洗面奶，换了新发型，并开始购置更得体的衣服。

### 第二步：培养兴趣爱好与技能

第二件事是培养兴趣爱好。我选择了学吉他和健身。

以下是当时的实践方法：
1.  **学吉他**：2013年街边有很多流浪歌手。我去给他们打下手，他们免费教我。他们唱累了，我就唱两句，并帮他们背吉他。（图中穿黑衣服、背吉他的就是我）
2.  **健身**：没钱办健身卡怎么办？我去健身房应聘业务员（发传单），因为健身房员工可以免费健身。

我之前学的这些兴趣爱好，无论是吉他还是后来学的钢琴，至今都很受用。现在我可以在女孩子面前无意中展示一下，这都是当年打下的基础，并且几乎没花钱。健身也让我养成了好习惯。

**核心逻辑**：你想做一件事，总能找到方法，而不是借口。

### 第三步：克服内向，锻炼自信

对于自己之前很内向、不自信的问题，我的方法是：**去做销售**。
我每天打电话，下大雨也去拜访客户，住在8人一间的上下铺宿舍。这些都是我过去的经历。关键在于，我真的去行动了。
