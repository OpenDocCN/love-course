# 恋爱新学堂 - P1 - “绿茶”的真正含义

在本节课中，我们将探讨“绿茶”一词在恋爱中的真正含义，以及它所代表的价值交换。

## 绿茶的定义
**绿茶**在恋爱中通常指的是那些懂得价值交换的女性。她们不仅能够给予情绪价值，还能在关系中保持平衡。


绿茶能够给予情绪价值，同时也能保持关系的平衡：

