# 在社交当中如何保持自信？-P1-恋爱新学堂-BV1mT411D7K5_note

在本节课中，我们将学习如何在社交互动中，特别是在与异性相处时，建立并保持自信。核心在于理解并调整我们对待“自尊心”的态度。

许多兄弟在社交中感到怯懦和自卑，根本原因在于过度关注自我，太在意他人的看法。这种心态会导致行为拘谨，影响社交体验。

上一节我们介绍了过度关注自我是自信的障碍，本节中我们来看看如何通过具体的心态转变来克服它。

## 核心心态：没有人在乎你的一言一行

你总感觉自己的言行被无数人注视着，担心说错话或惹人生气。但事实是，**没有人在乎你的一言一行**。就像销售人员在街上发传单，被拒绝是常事，但路人转眼就会忘记。

```
核心心态 = 他人的关注度 << 你自己的想象
```

即使做出一些看似“出丑”的行为，比如在街上大喊，别人也很快就会忘记。真正时刻在意你言行的人，只有你自己。

## 实践方法：从具体场景开始改变

以下是两个关键的生活实践，帮助你放下过度的自尊心。

1.  **进行“脱敏”练习**：主动将自己置于可能被拒绝或感到尴尬的境地。例如，尝试上街发放传单。这个过程能让你亲身体验到，他人的拒绝和看法其实对你没有实质影响。
2.  **模拟极端情况**：在内心问自己，即使发生最坏的情况（例如当众出丑），又能怎样？你会发现，这些事后的影响微乎其微，生活依旧继续。

## 约会中的应用：展现决断力

在与女生约会时，唯唯诺诺、不停询问对方意见，反而会破坏约会体验。自信的表现是温和的决断。

例如，点菜时可以直接询问对方忌口，然后自然地做出选择。这种姿态传递出稳定和可靠的感觉。

```
正确做法：询问忌口 -> 快速决策
错误做法：反复询问“你想吃什么？” -> 显得犹豫不决
```

## 根本之道：降低对他人评价的期待

你所有的紧张和情绪波动，大多源于对他人评价的过度期待。你害怕被批评，所以拼命证明自己。

成长的本质是**“杀死”过去的自己**。在你真正相信自己之前，无法赢得他人的信任。因此，请先拿掉那层脆弱、过度的自尊心。一个真男人不应扭捏作态。

在日常社交中，只要不直接影响他人利益，你的言行并不会被长久铭记。好的表现会被认为是高情商，差的表现也只会被当作一时的笑话。

**送给大家一句话：人的成长就是要把过去的自己给宰了。**

本节课中我们一起学习了建立社交自信的核心：拿掉过度的自尊心。我们明白了他人并没有想象中那么关注我们，并通过具体的方法在生活和约会中练习决断力。记住，自信始于自我接纳与勇敢实践。
