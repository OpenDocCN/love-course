# 恋爱情商课 P1：高情商沟通的两个核心技巧 🧠💬

在本节课中，我们将学习高情商沟通的两个核心技巧。这些技巧能帮助你在表达情绪时更有效，避免冲突，并增进人际关系。关键在于掌握表达不满和表达开心的不同逻辑。

## 技巧一：表达不开心时，对事不对人 😤

上一节我们概述了本课的核心内容，本节中我们来看看第一个具体技巧。当我们需要表达不满或批评时，应针对具体的行为或事件，而非针对个人本身。这样做能避免对方产生防御心理，使沟通更顺畅。

以下是“对事不对人”的具体应用方法：

*   **错误示范**：你这个人怎么这么马虎。
    *   **问题**：直接攻击对方的人格。
*   **正确示范**：这件事做得不够严谨。
    *   **解析**：将批评焦点从“人”转移到“事”。
*   **场景示例（朋友打架）**：
    *   不要说：你真是个冲动的人。
    *   可以说：刚才的那种行为很冲动。
*   **场景示例（女友生气）**：
    *   不要说：你真是一个爱发脾气的人。
    *   可以说：你这样（的行为）会让我觉得我们的感情，并不是那么的牢固。

**核心逻辑公式**：表达不满 = 针对 **行为/事件**，而非 **人**

## 技巧二：表达开心时，对人不对事 😄

学会了如何妥善表达不满后，我们来看看如何有效地表达赞美与开心。这个技巧与第一个技巧恰恰相反。当我们想夸奖或表达喜悦时，将赞美指向对方本人，而非仅仅针对某件事，效果会好得多。

以下是“对人不对事”的具体应用方法：

*   **错误示范**：K先生的视频真的很多干货。
    *   **问题**：赞美局限于单一事件（这条视频）。
*   **正确示范**：K先生真是一个思维逻辑严谨的人。
    *   **解析**：将赞美从“事”升华到“人”的品质。
*   **场景示例（朋友帮忙）**：
    *   不要说：你这件事做得真周全。
    *   可以说：你是个可靠的人。
*   **场景示例（女友做饭）**：
    *   不要说：这顿饭真好吃。
    *   可以说：你真是一个心灵手巧的人。

**核心逻辑公式**：表达赞美 = 针对 **人**，而非仅仅 **行为/事件**

---

本节课中我们一起学习了高情商沟通的两个核心技巧：**表达不满时对事不对人**，以及**表达赞美时对人不对事**。掌握这两个简单的逻辑转换，能显著提升你的沟通效果，让你在人际关系中更受欢迎。请务必在生活中实践这些技巧。
