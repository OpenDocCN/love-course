# 恋爱新学堂 第1课：假定成交法 💡

在本节课中，我们将要学习一种名为“假定成交法”的沟通技巧。这种方法源自销售领域，其核心在于通过“假定”某事已经成立或即将发生，来引导对方的思维和决策，从而自然地推进关系或达成目标。我们将探讨其原理，并通过具体案例学习如何将其巧妙应用于恋爱互动中。

## 什么是假定成交法？

上一节我们介绍了课程主题，本节中我们来看看“假定成交法”的具体定义。

假定成交法，顾名思义，就是**假定一件事已经成为事实**来进行对话和互动。它避免直接询问对方“是否同意”，而是以“事情已经或即将如此”的口吻推进，从而绕过对方的防备心理，引导局面向期望的方向发展。

## 销售中的案例

理解了基本概念后，我们通过一个熟悉的领域——销售——来观察这个方法的具体应用。

以下是销售对话中的一个典型流程：

*   销售员问：“张先生，这款车您觉得中意吗？”
*   客户回答：“嗯，它的越野性能很不错。”
*   销售员接着说：“是的，而且这款车内部空间也非常宽敞。我们来确定一下颜色吧，您比较喜欢深色调的是吧？”

请注意，此时销售员并未询问“您到底买不买”，而是直接进入了确认具体细节（颜色）的阶段，这正是在**假定客户已经决定购买**。

*   客户回答：“是的。”
*   销售员继续推进：“您看要不要给爱车加上导航？”
*   客户回应：“那倒不用。你们有赠送什么吗？”
*   销售员说：“有的张先生，这个合同已经注明了，我们一起来看看合同吧。”

在整个过程中，销售员没有直接询问客户是否购买，但通过一系列假定成交的提问，自然地引导客户完成了购买决策。如果直接问“您买不买”，反而容易激发客户的防备和犹豫。

## 在恋爱中的应用

看过了销售场景，现在我们将视角转向恋爱互动，看看如何运用这一方法。

这种方法在恋爱中同样有效。例如，当你想邀约女生时：

*   女生说：“我酒量不好。”
*   你可以直接假定成交，回复：“明知道我会灌你。”

这里，你假定了“一起喝酒”这件事即将发生，而不是在征求许可。

*   女生可能会娇嗔地回应：“我还没答应要跟你喝酒啊！”
*   你可以继续推进这个假定：“没答应的原因，有没有可能是没有挑日子？我们现在就把良辰吉日挑上。”
*   女生的情绪被调动后，或许会说：“哈哈，真喝起来我怕你喝不过。”
*   你可以接着假定一个更亲密的场景：“那为了我们的清白，你下手轻点。”

在整个对话中，你从未直接提问“可以一起喝酒吗”，但通过假定成交，不仅让女生接受了喝酒的提议，还营造了轻松暧昧的氛围。如果直接询问“周末一起喝酒吗”，女生可能会产生一系列疑问和防备，让邀约变得困难。

## 方法的核心与优势

通过以上案例，我们可以总结出假定成交法的核心作用。

这个方法的核心在于**引领话题，掌控话题走向**。它通过预设一个积极的、正在进行的互动框架，让对方在无形中进入这个框架思考，从而减少了直接拒绝的机会。这种方法可以灵活应用于推动关系升级的各个阶段。

## 总结与延伸

本节课中，我们一起学习了“假定成交法”。

我们从定义出发，了解了它**假定事实成立**的核心。接着，通过销售案例看到了它如何绕过防备、引导决策。最后，重点学习了在恋爱互动中，如何通过假定邀约或亲密场景来自然推进关系，避免直接询问带来的阻力。

合理运用假定成交法，不仅在恋爱中，在工作和生活的人际交往里，也能让你更加如鱼得水。
