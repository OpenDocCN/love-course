## 核心原则：避免询问“为什么”

## 正确的处理方法

那么，具体应该怎么做呢？关键在于聚焦于**事实**和**解决方案**，而非陷入对原因的追问。

**场景一：对方撒谎**
*   **事实**：撒谎就是一个事实。她没有“为什么”。
*   **错误做法**：问你为什么要撒谎？

**场景二：对方出轨**
*   **事实**：出轨是事实，是结果。
*   **错误做法**：控制不好情绪和她吵架，追问为什么。
*   **正确做法**：在确认事实后，转身离开即可。你只需要了解事实，并且让她为自己的行为付出代价。
