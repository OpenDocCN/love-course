# 恋爱新学堂 - P1 - 感情与人性的关系

## 概述
在本节课中，我们将探讨感情与人性的关系，通过分析人性中的社会价值排序和社交本质，来理解感情中的复杂现象。

## 感情的真相
**公式**：感情 = 虚假在一起，真实分开

感情的真相是因虚假而在一起，因真实而分开。今天，我们透过人性来看一下感情。

## 社会价值排序
**公式**：人性 = 社会价值排序

人性是社会价值的排序，我们根据一个人的社会、学历、背景、学识、原生家庭和经济状况来区分一个人的社会地位。

## 案例分析
以下是几个案例分析，帮助我们理解人性在感情中的作用。

### 案例一：拒绝搭讪
假设你在大街上搭讪两个女生，一个美女和一个丑女，她们都拒绝了你。你内心对美女的拒绝感到正常，但对丑女的拒绝感到不解。

**原因**：社会价值排序导致我们对不同的人有不同的期望。

### 案例二：环境对形象的影响
在一个脏乱差的环境中，你可能不会在意自己的形象；而在一个高逼格的地方，你会更加在意自己的形象。

**原因**：环境影响了我们的行为和态度。

### 案例三：借钱给谁
如果你的发小和老板同时向你借钱，你会选择借给谁？

**原因**：社会价值排序导致我们更倾向于帮助那些社会地位更高的人。

### 案例四：过年回老家
你坐在劳斯莱斯车上回老家，内心可能会感到道德压力，但同时也可能有一丝阴暗的快感。

**原因**：社会价值排序导致我们对自己的社会地位产生复杂的情感。

## 感情的本质
感情的本质是造假，社交的本质也是造假。感情中有一个原则：永远不要触发任何明面上的不公平。

## 恋爱的目的
恋爱的目的是向上社交，跨越阶层，而不是单纯的学习情情爱爱。

## 总结
本节课中，我们一起学习了感情与人性的关系。通过分析人性中的社会价值排序和社交本质，我们更好地理解了感情中的复杂现象。希望这些知识能帮助你更好地理解感情，实现向上社交的目标。
