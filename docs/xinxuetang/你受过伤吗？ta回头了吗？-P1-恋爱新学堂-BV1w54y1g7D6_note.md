# 恋爱新学堂 - P1 - 你受过伤吗？ta回头了吗？

## 概述
在本节课中，我们将探讨前任关系中的心理动态，以及如何正确处理与前任的关系。

## 前任心理分析
**前任永远不会因为良心发现而选择爱你。** 越是追求前任，他们越不可能回头。如果他们连体面的退场都不愿意给你，那么只有当他们发现自己失去了宝贵的东西时，才会回头求你。这就是人性。记住，真爱永远不属于弱者。

**你越卑微，越哀怨，前任就越远离你。** 你想为爱退让，而他只会觉得你软弱。如果你想听真话，按照这种方式，你其实是在玩一个必输的赌局。道理你都懂，但就是控制不住自己的心态。你是痛得还不够彻底吗？

**吃过的亏还不够多吗？** 你喜欢看他抱着心爱的人说：“看，这就是我的前任，还是那个酸样子的。”幸好我当初甩了他，爽吧。记住，挽回是能量守恒。如果你想挽回，就不要这么被动地去挽回，拿回你应有的主导权。

## 总结
本节课中，我们一起学习了前任心理分析，以及如何正确处理与前任的关系。希望这些内容能帮助你更好地理解自己和前任之间的关系。
