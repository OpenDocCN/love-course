# 恋爱新学堂 第1课：摆脱认知习气 🧠

在本节课中，我们将要学习“舔狗”行为背后的根本原因——认知习气。我们将剖析这种思维习惯是如何形成的，以及它如何影响你在情感关系中的判断与行为。理解并摆脱它，是建立健康平等关系的第一步。

## 舔狗行为的三个核心原因

男人之所以会成为“舔狗”，主要源于以下三个递进的原因：

1.  **明知不值得，仍坚持投入**：你清楚地知道对方不值得你付出如此多，却依然选择投入。
2.  **投入失败后，加大赌注**：当你的投入没有获得预期回报时，你不仅不停止，反而选择投入更多。
3.  **投入没有长期回报**：你的所有付出，都无法换来任何长期、稳定的情感回报，对方完全不懂得感恩。

## 反思社会观念的来源

上一节我们分析了具体行为，本节中我们来看看这些行为背后的观念从何而来。社会上存在大量“舔狗”现象，首先需要我们思考一个根本问题：为什么“男人追女人”听起来理所当然？

我们可以将这个问题推向极端以便理解：**为什么不是女人追男人？**

你认为，得到心仪女性的方式必须是“认真追求”。你需要思考：这个观念是谁、在什么时候灌输给你的？事实上，大多数男性都难以摆脱这种**认知上的习气**。

## 什么是认知习气？

为了帮助你更好地理解这个概念，我们来看一个简单的例子。

请看一张纸巾，它是什么颜色？你的第一反应是“白色”。当你看到这张白色纸巾时，你能在认知上“去掉”白色吗？不能。你的大脑瞬间识别出“白色”，这是一种无法轻易抹除的**认知习气**。

人在认知上无法摆脱的习气非常多。这种认知通常源于从小**耳濡目染**的环境。在情感领域也是如此：可能从小你的父母就告诉你，“要对女生好，才能找到好媳妇”。这导致你形成了“必须对女生好，才能收获爱情”的**固化思维**。

它已经成为一种**习惯性认知**，使你无法看到这个“真相”之外的其他可能性。

## 核心：衡量情感投入产出比

请注意，我们并非主张“不要对女生好”。关键在于，在付出之前，需要先进行理性衡量。

本视频希望传达的核心思想是：**在对女生好的前提下，应先衡量“投产比”**。这个概念可以用一个简单的公式来概括：

**情感健康度 ≈ (长期情感回报) / (情感与物质投入)**

遗憾的是，能真正理解并实践这一点的男性可能并不多。

## 课程总结

本节课中，我们一起学习了“舔狗”行为的三大表现，并深入其根源——**认知习气**。我们认识到，许多情感观念是社会与环境潜移默化塑造的结果，成为一种难以察觉的思维习惯。要建立平等健康的关系，关键在于**保持觉察，在付出前进行理性衡量**，而非被惯性思维驱使。
