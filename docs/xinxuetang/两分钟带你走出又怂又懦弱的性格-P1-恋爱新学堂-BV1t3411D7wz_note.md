# 两分钟带你走出又怂又懦弱的性格-P1-恋爱新学堂-BV1t3411D7wz_note

在本节课中，我们将要学习如何在社交和恋爱关系中摆脱讨好与自卑的心理，建立真正的自信。核心在于理解并实践“拿掉自尊心”这一关键心态转变。

许多人在社交中过于在意他人的看法，总觉得自己的一言一行都备受关注，担心会惹怒或冒犯他人。这种心态会导致行为上的唯唯诺诺和过度拘束。

上一节我们提到了过度关注自我的普遍现象，本节中我们来看看如何通过具体经历来理解“无人关注”这一事实。

我曾有半年时间在街上发传单。最初我不敢发，害怕被拒绝。当时我的领导说了一句话，让我印象深刻：**没有人在乎你的一言一行**。做过销售的人对此会有深刻体会。

为了进一步说明这个道理，我们可以做一个思想实验：
*   就像我现在去街上大喊“我是猪”，这又能怎样？
*   就像你们现在骂我一两句，这又能怎样？
*   就像我在屏幕前连说三遍“我是猪”，这又能怎样？

睡一觉起来，大家就都忘了。你们并不会时刻在意我。所以，请少把自己当回事。

这种心态在约会中尤为有害。有的兄弟和女生约会时唯唯诺诺，特别拘束，连点菜都不敢，不停询问女生意见。

以下是这种行为的后果：
*   你的唯唯诺诺会导致女生也变得拘谨。
*   整个约会的体验感会变得很差。

正确的做法应该是什么呢？如果是我，点菜时会直接拿过菜单，问对方：“**你有什么忌口吗？**” 事情就解决了，无需扭捏。

建立自信需要主动脱敏。如果你在社交中没有自信，可以尝试去大街上“捅个破碗要饭”来锻炼自己。如果你在感情中没有自信，可以试着目睹女生上了别人的豪车，这或许能激发你奋发图强。

你所有的负面情绪，其实都源于对他人评价的过度期待。你害怕别人戳到你的痛处，于是你抵抗、你证明，最终落入情绪陷阱。

我不想讲鸡汤，比如“要保持自信，要多运动多赚钱”。在那之前，请你先**拿掉你的自尊心**。真男人不该扭扭捏捏。

在日常社交中，只要没有切实影响到他人的利益，真的没人在乎你想什么、说什么、做什么。

以下是他人可能的反应：
*   你表现好，别人只会觉得你情商高。
*   你表现差，别人也只会把你当个笑话，一笑了之。

唯一时刻在乎自己是否出丑、会给别人留下什么印象的，只有你自己。

最后，送给兄弟们一句话：**人的成长，就是要把过去的自己给“宰”了。在你不相信自己之前，没有人会相信你。**

本节课中我们一起学习了如何通过“拿掉过度的自尊心”来克服社交中的懦弱与讨好心态。关键在于认识到他人并没有那么关注你，并勇敢地采取行动，从内心开始相信自己。
