上一节我们介绍了课程的主题，本节中我们来看看如何从一段情感文本中提取有用的对话开场思路。

以下是从文本中提取的关键情感元素和潜在开场白方向：

*   **表达时间与怀念**：例如，“离开你一百个星期，我回到了这里，寻找我们爱过的证据”。这可以转化为表达时间流逝和怀念过去的开场白。
*   **承认过去与自我反思**：例如，“过去我和你在一起，是我太怕你”。这提示了可以坦诚地承认过去关系中的问题或自己的不足。
*   **表达当前状态**：例如，“现在只剩我自己，偷偷的想你”。这说明了可以简单分享自己当前的生活或感受。
*   **借用共同记忆或象征**：例如，“玫瑰花它的过去”、“玫瑰花的葬礼”。这提示了可以提及双方共知的物品、地点或事件作为话题切入点。
*   **表达距离与思念**：例如，“Lord I‘m five hundred miles， Away from home”。这可以隐喻性地表达情感上的距离或物理上的距离，引发共鸣。

`问候 + 时间/事件参照 + 表达感受/状态 + 开放式问题`
