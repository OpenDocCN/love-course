## 概述

## 案例分析
以下是几个案例，用以说明需求感暴露在恋爱中的表现：

### 案例一
一个未见过面的女生要求你买奶茶，你如何回应？

**女生**：我想喝杯奶茶。
**你**：你连喝杯奶茶的钱都没有吗？
**女生**：你愿不愿意？

**分析**：女生用买奶茶来筛选你，表明她可能不尊重你。

### 案例二
女生要求你发红包，你如何回应？

**女生**：给我发个红包。
**你**：可以，我的红包大小是按照身材好坏来的，照片发来，要红包可以，你拿什么来换。

**分析**：女生试图让你为她买单，而你则提出了自己的原则。

### 案例三
女生要求你给她买包，你如何回应？

**女生**：你可以转给我，我自己去买。
**你**：从我这里拿走钱的只有员工和女朋友，你是想当哪一个？
**女生**：女朋友就算了。
**你**：员工还可以。
**女生**：能者多劳先灭世。

**分析**：女生试图让你为她买单，而你则坚持自己的原则。

## 核心概念
**需求感暴露**是指在恋爱中，一方对另一方的需求程度过高，导致对方产生压力或不适。

**公式**：需求感暴露 = 需求程度 / 满足程度
