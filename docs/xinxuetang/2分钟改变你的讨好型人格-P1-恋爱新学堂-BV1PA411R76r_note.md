## 教程内容

### 以下是改变讨好型人格的步骤

1. **理解自己的性格**
   - 从小是留守儿童，拒绝别人对我来说非常难。
   - 从小的性格导致我在感情中任人摆布。

2. **不被他人期望操控**
   - 曾经活在别人的期望中，现在不怕任何人对我失望。
   - 认识到别人的期望与你无关，不必为了满足他人而委屈自己。

3. **学会自私**
   - 承认自己的自私，减少怨气。
   - 更能理解人性的阴暗面，不逃避现实。

4. **树立真实形象**
   - 不要做完美主义者，主动暴露自己的缺点。
   - 不要把形象架得太高，以免人设崩塌。

5. **学会拒绝**
   - 当别人向你求助时，学会拒绝。
   - 不要因为担心关系尴尬而违背自己的原则。

6. **关注自己**
   - 不要总是为了让别人满意而忽略自己的感受。
   - 认识到自己的价值，不要为了取悦他人而委屈自己。

## 总结
本节课中我们一起学习了如何从讨好型人格转变为更加自信和果断的人。通过不被他人期望操控，学会自私，树立真实形象，学会拒绝和关注自己，我们可以摆脱讨好型人格的束缚，过上更加自由和快乐的生活。
