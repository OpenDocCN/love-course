# 恋爱新学堂 · 第一课：告别讨好型人格 🚫

在本节课中，我们将学习如何识别并改变讨好型人格。我们将探讨其核心成因，并通过具体的思维与行为转变策略，帮助你建立更健康、更自信的人际关系模式，尤其是在情感互动中掌握主动权。

## 从“讨好者”到“果断者”的转变

我曾是典型的讨好型人格。作为留守儿童，拒绝他人对我而言极其困难。这种性格使我在感情中处于被动，被他人随意对待。

那么，我是如何从一个讨好型人格，成长为如今能果断决策的情感博主的？这条视频内容我最具发言权。改变讨好型人格只有一个核心：**不被他人的期望所操控**。

过去，我活在别人的期望里。现在，我不再害怕任何人对我失望，这是我最大的转变。

## 核心心法：摆脱他人期望的操控

上一节我们提到了转变的核心，本节中我们来深入理解这个心法。

你的期望是你的事，与我无关。如果你因此失望，那是你自身的问题。我为何要费力去满足你的期望？

例如，我后台私信常有人请求我免费帮忙或教学。当被问及“我为何要免费教你”时，对方常回答因为他是我的长期粉丝。若是过去的我，我会道谢并免费解答。但现在，我只会回复两个字：**没空**。

原因在于：你关注我是因为我的能力，我分享的内容对你有价值。否则，你也不会关注一个毫无用处的博主。

## 关键策略一：放弃完美主义，主动暴露缺点

理解了核心心法后，我们需要具体的行动策略。首先，要放弃对完美形象的追求。

我们应主动暴露自己的缺点，否则极易陷入“人设绑架”。如果你在别人眼中是一个大方的人，你便很难在他面前表现出小气。

网络上许多人设崩塌案例，皆因形象树立得过高。例如，过去与女生相处时，对方借钱我总不好意思拒绝，即使拒绝了内心也深感不安。现在若遇到同样情况，我只会回复：**抱歉，我不喜欢穷人**。

我这样做的理由是：对方在提出借钱请求时，并未考虑如果我拒绝，双方关系该如何处理、是否会尴尬。她都不在意这个问题，我为何要在意？

请记住：**你树立的形象越完美，你获得的包容度就越小**。因为一个“好人”似乎就不该做任何“坏事”。

## 关键策略二：学会“健康的自私”

除了打破完美主义，另一个至关重要的点是学会优先考虑自己。

男人要想活得舒心，有一个比金钱更重要的点：**学会自私**。我发现敢于承认自己自私的人，通常怨气更少，因为他们更能理解人性的阴暗面。

不敢正视阴暗的人，也不可能看见真正的阳光。一个总是不想让任何人失望的人，最终辜负的往往是自己。

## 课程总结

本节课中，我们一起学习了如何改变讨好型人格。

我们首先明确了转变的核心在于**不被他人的期望所操控**。接着，我们探讨了两个关键策略：一是**放弃完美主义，主动暴露缺点**以避免人设绑架；二是学会**“健康的自私”**，优先关照自己的感受与边界。

改变始于思维，成于行动。从今天起，练习重视自己的需求，勇敢而坦然地说“不”。
