# 恋爱高手都具备的暗黑心法 - P1 🧠

在本节课中，我们将要学习一套被称为“暗黑心法”的思维模式。这套心法的核心在于通过“物化”思维来管理人际关系，旨在帮助你在情感互动中保持理性、提高效率并减少内耗。掌握它，你将在关系中更加从容。

## 核心概念：物化思维 🔧

暗黑心法，简单来说，就是**将人物化，包括自己**。成年人只能筛选，不去改变。这意味着把人当作工具看待。

**公式：人际关系 = 筛选工具**

合适就拿过来用，不合适就不要耽误彼此的时间。这种说法虽然缺乏人情味，但能最大程度提高效率，降低内耗。

## 物化思维的实践：从公司到个人 🏢

上一节我们介绍了物化思维的核心概念，本节中我们来看看它在现实中的实践。

公司招聘时永远只挑选现成的员工。你见过哪个公司会把不合格的员工招进来，培训完再上岗？公司只留下有价值的员工。一个公司想要做大做强，物化员工是第一步。

同理，一个人想要变强，物化身边的人，也是你的必经之路。你身上有我需要的特质，只要我需要，我能在这段关系中得到价值，那我就想办法来搞定你、接近你。

## 情感关系中的取舍原则 💔

理解了物化的普遍性后，我们将其应用到更具体的情感关系中。

一段感情如果不能让你成长，那么你就放弃它。谈了个女朋友，遇到了“吸血虫”怎么办？那么你就把这条虫狠狠地扯下来。

以下是需要秉持的两大原则：
*   **关我屁事原则**：区分哪些事真正与自己有关。
*   **关你屁事原则**：不干涉他人的私生活与选择。

很多时候你觉得失望，是因为你对那些事情给予了太高的期望。简单来说，你潜意识里把那些事看成是你自己的责任。

## 设立界限，减少情绪消耗 🚧

上一节我们提到了原则，本节中我们来看看如何通过设立界限来落实这些原则。

你想改变别人，但他们几乎不会被你改变。你要把真正和自己有关的事情，和无关的事情区分开，设立明显的界限。

就像我现在如果看到了一个烂人，我永远不会想去纠缠。我对他的私生活没有任何审判欲望，连评价他我都懒得评价。他需要自己救自己。这与我没有任何关系。

其实你没有必要和一些不必要的人，去说一些不必要的话，产生一些不必要的情绪。

## 长期主义：经营你的主战场 🗺️

谈完了对具体关系的处理，我们最后将视角放大，看看如何规划人生的整体战场。

一个男人最聪明的做法是选定一个城市，把这个城市当做你的主战场。用5年或者10年的时间，在这个城市生根发芽。

以下是实现这一目标的关键行动点：
*   把你的人脉资源延展到这个城市的各个角落。
*   切记避免一两年甚至半年就换一个城市。
*   避免人到中年仍无根基，连稳定的社会关系都没有。

## 总结与最后的导师 📚

本节课中我们一起学习了“暗黑心法”。其核心是通过**物化思维**进行高效的人际筛选与管理，在情感中秉持“关我屁事”和“关你屁事”原则以设立界限，并通过长期经营一个“主战场”来积累真正有价值的资源。

如果你还无法领悟这套心法，那么你人生最好的导师就是：父亲的低头，母亲的眼泪，朋友的背叛，空空的钱包。
