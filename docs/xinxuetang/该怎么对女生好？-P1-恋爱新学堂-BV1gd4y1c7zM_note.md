我回复：“给别人的呀。”
女生回复：“？”
（女生在责怪我没有说早安，并感到失落。）

同样的30块早饭，你点和我点区别很大。我说一句实话：其实女生吃不吃早饭，我并不是那么在意。因为在我和这个女生在一起前，我们没有共同经历和甜蜜回忆，我做不到那么心疼她。
