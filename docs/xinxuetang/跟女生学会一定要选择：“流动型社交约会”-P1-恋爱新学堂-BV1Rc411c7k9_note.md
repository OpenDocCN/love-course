## 传统约会的局限

目前排名前三的男女约会项目分别是：**吃饭**、**看电影**和**游戏**。

除了看电影之外，剩下的两个项目对一个人聊天的能力要求极高。例如，你约女生吃饭，两个人除了面前有一些食物以外，要靠彼此的沟通能力来增进了解、增加好感。

但如果两个人刚认识没多久，彼此之间没有太多的话题可聊，那么在这样的场景下吃饭交流时，你会觉得压力很大。你很容易因为表达能力不强、沟通技巧没有掌握好，然后就进入到尬聊的环节。

你必须依靠自己的话题储备，才可以支撑起整个约会。如果你无意间给对方留下无趣的印象，对方一旦把“无趣”的标签贴在你身上，那就是一件非常尴尬的事情。

因此，你一定要懂得“见机打铁”。而“见机”的要诀，就是所谓的“流动型社交约会”。

## 什么是流动型社交约会？

上一节我们介绍了传统静态约会的压力，本节中我们来看看能化解这种压力的“流动型社交约会”。

流动型社交约会的特征有以下四个：

以下是流动型社交约会的四个核心特征：
1.  **行进状态**：这种约会方式可以让两个人处于行进的状态。
2.  **明确任务线**：这种约会有一个任务线，并且完成这个任务非常容易。
3.  **外部互动**：这种约会在进行时，可能会跟很多人产生交集，有机会触发很多即兴的、随机性的事件。
4.  **共同记忆**：这种约会能够让你们两个人之间，建立起关于某个事件的共同记忆。

## 流动型社交约会的实例

理解了核心特征后，我们通过一个具体例子来看看它如何运作。

逛水族馆或者海底世界，可以算作流动型社交约会。

例如，在海底世界：
*   你走着走着，鞋带突然松了。
*   走着走着，对方突然被绊了一下。
*   走着走着，你们两个就迷路了。
*   可能你们在海底世界逛到某个地方，突然有个海豚朝你们招手。
*   可能你们走着走着，看到有个小朋友朝乌龟扔石头，你们两个人可以一起去阻止一下。

诸如此类的**即兴话题**，能够有效缓解你们的沟通压力。更何况在整个约会过程中，你们还可以建立起一些共同、独有的记忆。

不要觉得即兴的随机事件与自己无关。约会时要是出现这种事情，相当于上天给你们两个人送了一个礼物，等于你们两个马上有了一个即兴的话题。这个突然横亘在你们之间的话题，能够大量减少你们之间的交流压力。
