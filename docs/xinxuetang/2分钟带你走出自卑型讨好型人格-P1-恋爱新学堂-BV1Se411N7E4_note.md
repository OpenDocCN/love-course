# 恋爱新学堂 P1：🚀 两分钟摆脱自卑与讨好型人格

在本节课中，我们将学习如何克服在人际交往，特别是与异性相处时的自卑与讨好倾向。核心在于调整心态，减少不必要的自我关注，从而建立真正的自信。

许多人在社交中过于在意他人的看法，总觉得自己的一言一行都受到审视，害怕惹对方生气。这种心态往往源于过强的自尊心。

上一节我们提到了过度自我关注的问题，本节中我们来看看如何具体地“拿掉”不必要的自尊心。一个有效的方法是经历生活的锤炼。当你被生活或感情深刻伤害后，往往会发现自尊心不再那么重要。

我曾有半年在大街上发传单的经历。最初我不敢发，害怕被拒绝。我的上级当时说了一句令我印象深刻的话：**“没有人在乎你的一言一行。”** 做过销售的人对此应有共鸣。

为了进一步说明这个观点，可以做一个思想实验：
```
即便我现在当众大喊“我是猪”，或者你们在评论区骂我几句，又能怎样？睡一觉之后，大家都会忘记。你们并不会时刻在意我。
```
因此，在社交中，请减少不必要的自我重视。

这种心态会直接影响与异性的约会表现。如果你表现得唯唯诺诺、特别拘束，比如点菜时不停询问对方意见，女生往往也会感到拘谨，导致约会体验变差。

以下是正确的做法示例：
点菜时，我拿过菜单，直接问对方：“**你有什么忌口吗？**” 这样既干脆又显得有主见，无需扭捏。

如果缺乏自信，不妨进行一些“冲击性”练习。例如，感情上没自信，可以试想如果有女生当着你面上别人的豪车，你是否会因此奋发图强？

我们所有的负面情绪，常常源于过度在意他人评价，害怕被戳到痛处并进行抵抗与证明。与其空谈“保持自信”的鸡汤，不如先踏出第一步：**拿掉你过剩的自尊心**。

在日常社交中，只要没有切实影响他人利益，真的没人在乎你的具体言行。你表现好，别人顶多觉得你情商高；你表现差，别人也只会当作笑话，很快忘记。

唯一时刻在意自己是否出丑、会给别人留下什么印象的，其实只有你自己。

最后，送给各位一句话：**人的成长，就是要把过去的自己给“宰了”。在你不相信自己之前，没有人会相信你。**

本节课中我们一起学习了克服自卑与讨好心态的核心：通过降低对他人评价的期待、减少自我关注，并采取果断的行动来建立内在自信。记住，真男人，从不扭扭捏捏。
