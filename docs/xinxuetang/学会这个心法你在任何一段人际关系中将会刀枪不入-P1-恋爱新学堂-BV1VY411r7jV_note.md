# 人际关系暗黑心法教程 - 第一课：工具化思维 🛡️

在本节课中，我们将要学习一套被称为“暗黑心法”的人际关系处理原则。这套心法的核心在于通过工具化思维来审视人际关系，包括对待自己与他人，旨在帮助你在各种关系中保持高效并减少内耗。

## 核心心法：工具化思维

这套暗黑心法的核心，简单来说，就是“工具化”。这指的是将人（包括自己）视为工具。成年人只能筛选，不去改变。其本质就是把人当作工具来对待。

**公式：人际关系 ≈ 工具筛选模型**

合适就拿过来用，不合适就不要耽误彼此的时间。这种说法虽然缺乏人情味，但它能最大程度地提高效率并降低内耗。

## 商业世界的启示

上一节我们介绍了工具化思维的核心概念，本节中我们来看看现实世界是如何实践这一点的。公司招聘时永远只挑选现成的员工。你见过哪个公司会把不合格的员工招进来，培训完再让其上岗的吗？

公司只留下有价值的员工。一个公司想要做大做强，物化（筛选）员工是第一步。同样，一个人想要变强，筛选身边的人，也是你的必经之路。

## 人际关系的价值交换

基于工具化思维，人际关系可以看作一种价值交换。你身上有我需要的特质，只要我需要，我能在这段关系中得到价值，那我就想办法来搞定你、接近你。

以下是应用此原则的具体指导：
*   如果一段感情不能让你成长，那么你就放弃它。
*   谈了个女朋友，遇到了“吸血虫”怎么办？那么你就把这条虫狠狠地扯下来。
*   秉持“关我屁事”和“关你屁事”的原则。

很多时候你觉得失望，是因为你对那些事物给予了过高的期望。简单来说，你潜意识里把那些事看成了你自己的责任。

## 设立界限与减少内耗

你想改变这些人，但是他们几乎不会被你改变。因此，关键是要把真正和自己有关的事情，和无关的事情区分开，设立明显的界限。

就像我现在如果看到了一个烂人，我永远不会想去纠缠。我对他的私生活没有任何审判欲望，连评价他我都懒得评价。他需要自己救自己。度过去了他就成佛，度不过去就是他自己的能力问题，早晚他有这一劫，跟我没有任何关系。

其实你没有必要和一些不必要的人，去说一些不必要的话，产生一些不必要的情绪。

## 人生战略：扎根与积累

一个男人最聪明的做法是选定一个城市，把这个城市当做你的主战场。用5年或者10年的时间，在这个城市生根发芽。

以下是实施此战略的要点：
*   把你的人脉资源延展到这个城市的各个角落。
*   切记避免一两年甚至半年就换一个城市，导致没有任何根基。
*   否则混到了40岁你还在流浪，连个稳定的基本社会关系都没有。说得难听一点，连电话号码都经常换。

## 最后的领悟与总结

如果你还无法领悟这套暗黑心法，那么你人生最好的导师就是：父亲的低头，母亲的眼泪，朋友的背叛，以及空空的钱包。

本节课中我们一起学习了“暗黑心法”的基础——工具化思维。我们探讨了如何通过价值视角审视关系、设立个人界限以减少情绪内耗，并制定了长期扎根积累的人生战略。掌握这些原则，旨在帮助你在人际关系中更加清醒、高效，并专注于自身的成长与目标。
