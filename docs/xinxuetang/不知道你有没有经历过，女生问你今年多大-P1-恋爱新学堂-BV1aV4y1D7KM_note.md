# 恋爱新学堂 - P1 - 如何回答女生的年龄问题

## 概述
在本节课中，我们将学习如何回答女生询问年龄的问题。

## 步骤
以下是回答女生询问年龄的步骤：

1. **保持自然**：不要显得过于紧张或尴尬。
2. **直接回答**：直接给出你的年龄，无需过多解释。
3. **微笑**：微笑可以传达出你的友好和自信。

## 示例
以下是一个简单的对话示例：

**女生**：你今年多大呀？

**你**：我今年25岁。

**公式**：年龄 = 出生年份 - 当前年份
