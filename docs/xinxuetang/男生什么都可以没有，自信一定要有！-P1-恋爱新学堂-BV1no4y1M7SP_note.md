# 恋爱新学堂 第1课：自信的基石 🧱

在本节课中，我们将要学习自信的核心来源。我们将探讨如何通过“认识自我”来建立稳固的自信，并理解为何外界的评价有时会严重影响我们的情绪。

---

一个男性可以没有很多东西，但自信必须要有。如何让自己变得自信？方法很简单：找回你自己。很多人连自己是谁都不知道。你越清楚你是谁，你就越自信；你越不知道自己是谁，你就越不自信。

上面这段话至关重要，接下来请仔细理解。

为了说明这一点，让我们来看几个例子。

以下是两个具体场景：
*   **例子一：身高。** 假设我们知道自己身高有一米八。在这一点上，我们知道自己不矮。当外界说我们很矮时，我们会有情绪波动吗？不会，我们懒得理会。因为我们知道自己有一米八，我们知道自己是谁。
*   **例子二：专业能力。** 你知道自己的专业能力有80分。当外界说你很蠢、专业能力很差时，你会难过吗？不会。为什么？因为你知道自己的专业能力有80分。

从上面的例子我们可以看出，自信建立在对自我的清晰认知上。

上一节我们通过例子说明了自我认知的重要性，本节中我们来看看自信具体建立在哪些自我认知的维度上。

你的自信只存在于你对自己有多了解之上。你需要知道自己**值得什么**，**不值得什么**。你需要知道自己**可以做什么**，**不可以做什么**。你必须非常清晰、明确地知道自己的**价值观**是什么。

当你完全知道自己是谁的时候，外界对你的反馈所产生的影响会变得非常小。它们无法撼动你的自我价值感。

然而，如果我们不知道自己是谁，情况就完全不同了。

上一节我们讨论了清晰的自我认知带来的好处，本节中我们来看看自我认知模糊会导致什么问题。

当如果我们不知道我们自己是谁的时候，外界的反馈对我们的影响就非常大。外界说我们很好，我们就很开心、很高兴。外界说我们不好，我们就很难过。你把对自身的评价完全交给了外界。于是，外界一句话能让你哭，外界一句话也能让你笑。

当你不知道自己的时候，外界对你的影响就是有这么大。如果你不知道你是谁，外界就会帮你来定义你是谁。

---

本节课中我们一起学习了自信的根本在于“认识自我”。我们了解到，清晰的自我认知——包括了解自己的优势、能力和价值观——是抵御外界负面评价的盾牌。相反，若自我认知模糊，我们就会轻易被外界言论左右情绪和自我价值感。所以，建立自信的第一步，就是回答这个问题：**你是谁**。
