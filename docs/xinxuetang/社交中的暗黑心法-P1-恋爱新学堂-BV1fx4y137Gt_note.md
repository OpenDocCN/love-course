在本节课中，我们将要学习一套被称为“暗黑心法”的社交与关系处理策略。这套心法的核心在于通过理性“物化”他人与自己，来保护自身利益、提高社交效率并减少内耗。我们将逐步拆解其核心原则与应用方法。

## 概述：什么是暗黑心法？🤔

暗黑心法，说简单点，就是**把人“物化”**，包括自己。成年人只能筛选，不去改变。说白了，就是把**人当做工具**。

合适就拿过来用，不合适就不要耽误彼此的时间。虽然这么说没有任何人情味可言，但是它可以最大程度地提高效率，降低内耗。

---

## 第一节：从商业逻辑理解人际关系 💼

上一节我们介绍了暗黑心法的基本概念，本节中我们来看看如何从商业世界的逻辑来理解它。

公司招人的时候永远只挑现成的。你见过哪个公司把不合格的员工，招进来培训完再上岗的？公司只留下有价值的员工。

一个公司想要做大做强，物化员工是第一步。一个人想要变强，物化身边的人，也是你的必经之路。

**核心公式**可以概括为：
```
价值交换 = 我需要你的特质 + 我能从关系中获益
```

你身上有我需要的特质，只要我需要，我能在这段关系当中得到价值，那我就想办法来搞定你接近你。

## 第二节：在感情中应用筛选原则 ❤️➡️🚫

理解了商业化的筛选逻辑后，我们可以将其应用到更复杂的感情关系中。

一段感情如果不能让你成长，那么你就放弃它。谈了个女朋友，遇到了吸血虫怎么办？那么你就把这条虫狠狠地扯下来。

以下是处理此类问题的两个核心原则：

*   **关我屁事原则**：区分哪些事情真正与自己有关。
*   **关你屁事原则**：阻止他人不必要的干涉进入你的界限。

很多时候你觉得失望的原因，是因为你把那些东西，给予了太高的期望。简单点来说，你潜意识把那些事看成是你自己的责任。

你想改变这些人，但是他们几乎不会被你改变。你要把真正和自己有关的事情，和无关的事情给区分开，设立明显的一个界限。

## 第三节：如何对待“烂人”与无效社交 🚷

设立了界限之后，我们需要知道如何具体执行。本节中我们来看看如何对待那些消耗你的人。

就像我现在如果看到了一个烂人，我永远不会想去纠缠。我对他的私生活没有任何的审判欲望，连评价他我都懒得评价。

他需要自己救自己，度过去了他就成佛，度不过去就是他自己的能力问题，早晚他有这一劫，跟我没有任何关系。

其实你没有必要和一些不必要的人，去说一些不必要的话，产生一些不必要的情绪。

**行为准则代码描述**：
```python
if person == “烂人” or interaction == “无效社交”:
    response = “不纠缠、不评价、不负责”
else:
    response = “进行价值交换”
```

## 第四节：深耕你的“主战场” 🗺️📍

处理了消极关系后，让我们转向积极建设。一个想要变强的人最聪明的做法，是选定一个城市，把这个城市当做你的主战场。

用5年或者10年的时间，在这个城市生根发芽。把你的人脉资源延展到这个城市的各个角落。

以下是必须避免的误区：

*   切记一两年甚至半年就换一个城市。
*   避免没有任何一点根基。
*   防止混到了40岁还在流浪，连个稳定的基本的社会关系都没有。

说得难听一点，要避免连电话号码都经常换的漂泊状态。
