短折法是一种不以长期恋爱关系为目的的短期相处方式。它可以帮助你快速接触前任，同时避免直接表达复合的意图。

### 短折法的优势
- **快速接触**：通过短折关系，你可以名正言顺地与前任接触，无需任何理由。
- **提高成功率**：短折法可以提高复合成功率，因为它可以让你在轻松的氛围中逐渐建立联系。

以下是实施短折法的步骤：

### 1. 表达孤独感
描述自己分手后的孤独感，表达出有时候需要一个陪伴者。

### 2. 提出短折关系

### 4. 制造落差感
