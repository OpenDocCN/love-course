## 什么是细节的魅力？

兄弟们知道什么行为在女生眼中最有魅力吗？今天跟兄弟们聊点悄悄话，这个行为就是**注重细节**。

什么递水的时候把瓶盖拧开，走路的时候让女生走马路内侧等等，这些小细节都是最基本的，我就不讲了。

## 高阶细节控实战

上一节我们介绍了细节的基本概念，本节中我们来看看如何将细节做到高阶水平。我将直接上干货，分享具体可操作的方法。

**核心公式：魅力 = 关注 + 记忆 + 适时表达**

以下是两个高阶细节控的实战案例：

*   **案例一：朋友圈记忆法**
    比如我之前一个海王哥们和女生约会前，他都会提前翻女生的朋友圈翻到一年以前，女生发的心情类的朋友圈。
    然后在约会中无意当中说出这句话：“你一年以前发的朋友圈我很有感悟。”女生肯定会问哪一条，然后你再把这句话给说出来。女生就会觉得“我自己都忘了，你还记得”。这就是高阶版本的细节控。当然你是不是真的记住，我不知道。你可以提前把这句话记在备忘录当中。他就是靠这个细节，让很多女生喜欢上他。

*   **案例二：生理期关怀**
    再比如女生来大姨妈了对吧，重要的不是你去送红糖水啥的。送红糖水是个男生都会做。
    重要的是你能不能记住，她第一次来大姨妈的时间，然后下个月这个时间，你主动地去提醒她不要喝冰的。女生也会觉得“哇你怎么记得这么清楚”，从而觉得这个男人是真的很有细节。

你我之间的差距就是靠这些细节所产生的。

本节课中我们一起学习了如何通过**注重细节**来提升魅力。最低成本打动一个人的方式就是靠细节，不要觉得不可思议。
