# 13、社交定位（为你勾画动态社交的蓝图）：13.现场约会社交不知道说什么、没有话题，话题聊干了怎么办？！（上）

**概述**： 本节课将探讨现场约会社交时遇到的话题枯竭问题，并提供解决方案。

**以下是本节课的主要内容**：

### 1. 关注感受而非话题

- 当你感觉无话可说时，你的注意力可能集中在内部，担心自己的表现。
- 专注于感受，而不是寻找话题，可以避免冷场和尴尬。
- **公式**： 感受 > 话题

### 2. 传递轻松愉快的情绪

- 约会是为了放松，而不是进行逻辑演讲或谈判。
- 专注于传递感受，而非话题的合理性。
- **代码**： `氛围 = 感受`

### 3. 热情是关键

- 分享你对事物的热情，即使话题本身不新鲜。
- 热情可以吸引他人的注意力。
- **公式**： 热情 > 新鲜度

### 4. 状态幻体开关

- 打开有趣、放松和激情的开关，以吸引他人。
- **代码**： `状态 = 有趣 + 放松 + 激情`

### 5. 关注互动氛围

- 80%到90%的精力应关注互动氛围和感受。
- 10%到20%的精力用于过滤话题。
- **公式**： 氛围 = 80%感受 + 20%话题

**总结**： 本节课中，我们学习了如何应对现场约会社交时的话题枯竭问题。通过关注感受、传递轻松愉快的情绪、分享热情、打开状态幻体开关以及关注互动氛围，我们可以避免冷场和尴尬，让约会更加愉快。
