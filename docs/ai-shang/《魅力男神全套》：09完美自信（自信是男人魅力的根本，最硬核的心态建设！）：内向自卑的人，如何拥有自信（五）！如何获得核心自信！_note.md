# 《魅力男神全套》：09 完美自信（自信是男人魅力的根本，最硬核的心态建设！）：内向自卑的人，如何拥有自信（五）！如何获得核心自信！

## 概述
在本节课中，我们将探讨如何获得核心自信，这是建立内在魅力的关键。

## 自信与自我以及自私的区别
**公式**： 自私 = 先尊重自己，再尊重他人

自信与自我以及自私并无本质区别，关键在于如何表达和看待。

## 自负
自负并非真正的自信，而是不自信的表现。真正的自信者不会在意他人的看法。

## 场景自信与核心自信
**公式**： 场景自信 = 外在自信（外在因素影响）
**公式**： 核心自信 = 内在自信（内在因素影响）

核心自信是建立于内心的，不受外界因素影响。

## 如何获得核心自信
### 1. 不害怕失败
不要害怕失败，将失败视为成长的机会。

### 2. 百分之百认同自己
坚持自己的立场，做自己喜欢的事情。

### 3. 不在乎别人的评判
做自己，说自己想说的，不要被他人的观点束缚。

## 总结
本节课中，我们学习了如何获得核心自信，这是建立内在魅力的关键。通过不害怕失败、百分之百认同自己以及不在乎别人的评判，我们可以获得真正的自信。
