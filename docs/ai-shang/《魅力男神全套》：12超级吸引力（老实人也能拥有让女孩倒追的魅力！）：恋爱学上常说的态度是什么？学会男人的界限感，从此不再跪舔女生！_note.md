## 一、 拥有清晰的界限感 🚧

上一节我们概述了课程目标，本节中我们来看看“态度”的核心——界限感。所谓态度，就是拥有清晰的界限感、属于自己的标准和原则。

界限感可以具体分为以下四个方面。以下是这四个方面的详细阐述：

1.  **我自己做什么行为，我自己能够接受。**
    *   例如：作为男生，我可以接受自己化妆、打扮得很潮、修眉、吃西餐或请女生吃小龙虾。这些行为符合我个人的标准和原则。

2.  **我自己做什么样的行为，我自己不能接受。**
    *   例如：我绝对不能接受穿着脏衣服出门，也绝对不能接受女生要求我发红包。这些是我的底线。

3.  **别人对我做什么样的行为，我可以接受。**
    *   例如：长辈在饭桌上给我夹菜，或者朋友开一些不过火的玩笑，这些行为在我的接受范围之内。

4.  **别人对我做什么样的行为，我不能接受。**
    *   例如：别人开很过分的玩笑、女生要求我发红包或购买我经济能力无法承受的物品（如手机）、女生随意翻看我的手机。这些行为触及了我的界限。

如果你没有这四点清晰的认知，就意味着你缺乏自己的原则和标准。一个没有界限感的男人，就像一个可以随意揉捏的橡皮泥，别人怎么对待你都可以。这样的男人，就是恋爱学中所说的“没有态度”的男人。

**核心公式：**
```
清晰的界限感 = 明确自我接受的行为 + 明确自我拒绝的行为 + 明确他人可接受的行为 + 明确他人不可接受的行为
```

大部分中国男生的问题在于，别人对他做什么行为他都能接受，他自己做什么行为也都能接受，不存在“不能接受”的事情。这直接导致了你缺乏吸引力。请花点时间思考一下：你是一个有标准、有原则、有界限感的男人吗？
