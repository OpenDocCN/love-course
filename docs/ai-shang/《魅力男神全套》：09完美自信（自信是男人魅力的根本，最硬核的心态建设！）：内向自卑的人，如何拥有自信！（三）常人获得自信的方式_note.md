# 《魅力男神全套》：09 完美自信（自信是男人魅力的根本，最硬核的心态建设！）：内向自卑的人，如何拥有自信！（三）常人获得自信的方式

## 概述

本节课将探讨内向自卑的人如何通过四种方式获得自信，帮助大家建立真正的核心自信。

## 一、提升自己的世俗价值

**公式**：自信 = 世俗价值

**解释**：通过提升自己的世俗价值，如财富、地位、外貌等，可以短暂地获得自信。

**步骤**：

1. **形象改造**：改变发型、穿着，提升个人形象。
2. **提升财富**：努力工作，提高收入水平。
3. **提升地位**：争取更好的职位，获得认可。

## 二、通过小团体抱团取暖

**公式**：自信 = 小团体支持

**解释**：通过加入小团体，获得朋友的支持和陪伴，可以增强自信。

**步骤**：

1. **结识新朋友**：扩大社交圈，结识志同道合的朋友。
2. **参加活动**：积极参与各种活动，结识更多朋友。
3. **建立团队**：组建自己的团队，共同成长。

## 三、依赖某种特长或技能

**公式**：自信 = 特长或技能

**解释**：通过掌握某种特长或技能，如唱歌、跳舞、摄影等，可以增强自信。

**步骤**：

1. **发展爱好**：选择自己感兴趣的爱好，并努力提升。
2. **学习技能**：学习一项新技能，如摄影、编程等。
3. **展示才华**：在朋友和同事面前展示自己的才华。

## 四、依赖特定的场合中的特定角色

**公式**：自信 = 特定场合中的特定角色

**解释**：在特定的场合中扮演特定的角色，如领导、专家等，可以增强自信。

**步骤**：

1. **寻找适合自己的场合**：找到自己擅长的领域，如工作、兴趣爱好等。
2. **扮演特定角色**：在特定场合中扮演特定角色，如领导、专家等。
3. **获得认可**：在特定场合中获得认可，增强自信。

## 总结

本节课介绍了四种常人获得自信的方式，帮助大家建立真正的核心自信。希望大家通过学习，找到适合自己的方法，提升自信，成为更好的自己。
