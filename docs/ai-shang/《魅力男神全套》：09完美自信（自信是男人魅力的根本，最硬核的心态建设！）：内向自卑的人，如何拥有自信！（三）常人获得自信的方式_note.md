# 《魅力男神全套》：09 完美自信（三）常人获得自信的四种方式 💪

在本节课中，我们将要学习普通人通常通过哪四种方式来获取自信。理解这些方式，能帮助你清晰地认识自己当前的自信来源，并为后续建立更稳固的核心自信打下基础。

上一节我们区分了自信与自大，本节中我们来看看，绝大多数人是通过哪些具体途径来感受和建立自信的。

以下是常人获取自信的四种典型方式：

## 1. 提升世俗价值

第一种方式是**提升自己的世俗价值**。世俗价值指的是那些被社会普遍认可的外在价值，例如：财富、外貌、衣着、发型等。

**公式**：`自信感受 ≈ 世俗价值提升`

当你的世俗价值得到提升时，你往往会立刻感觉到自信增强。例如：
*   花钱时一掷千金，会感觉很有自信。
*   买了一件好看的衣服或换了一个新发型，感觉自己变帅了，从而获得自信。
*   月入一万会比月入四五千的人更有底气。

对于从未体验过自信的新手而言，这是最快、最直接的方法。进行一次形象改造，马上就能获得一种“场景自信”。这种自信并非坏事，它可以作为你体验自信、迈出第一步的跳板。

**过渡**：然而，依赖世俗价值的自信有一个显著特点：它依附于外物。接下来，我们看看第二种建立在人际关系上的自信方式。

## 2. 通过小团体抱团取暖

第二种方式是通过**小团体抱团取暖**来获得自信。这意味着当你身处一个熟悉的群体中时，你会感到更自在、更有底气。

以下是常见的表现：
*   上学时上厕所、吃饭总要拉上同学一起。
*   一个人时懒得出门，但和朋友一起就愿意参与活动。
*   在街上看到成群结队的年轻人显得很“拽”，但单独一人时却不敢如此。
*   搭讪或参加陌生活动时，必须有朋友陪同才敢行动。

这种自信源于群体带来的安全感和认同感。如果你是一个社交较少、缺乏朋友的小白，多交朋友、融入一些小团体，是获得初期自信的有效途径。

**过渡**：但如果你发现，一旦离开团体或需要独立行动时，自信就消失了，那么你可能需要警惕被这种“小团体自信”所限制。下面，我们探讨第三种更个人化的自信来源。

## 3. 依赖某种特长或技能

第三种方式是**依赖某种特长或技能**。当你拥有并展示一项优于他人的技能时，你会从中获得强烈的存在感和自信。

例如：
*   擅长弹吉他、跳舞、唱歌、摄影、滑板等。
*   游戏玩得很好，在虚拟世界里成为“老大”。
*   酒量很好，在饭局上感到自信。
*   学习恋爱技巧后，在和女生聊天时感觉更从容。

**代码**：`if (拥有特长 && 展示特长) { 自信状态 = true; }`

通过特长获得的自信也是“场景自信”的一种。对于新手，培养一个真正的爱好并从中获得成就感，是建立自信的绝佳起点。在恋爱领域，学习基础聊天技巧也能帮助你快速建立互动自信。

**过渡**：这种自信同样有局限性：一旦离开你擅长的领域，自信可能随之减弱。最后，我们来看一种与特定环境绑定的自信。

## 4. 依赖特定场合中的特定角色

第四种方式是**依赖特定场合中的特定角色**。你在某个特定环境或身份中会感到非常自信，但脱离那个环境或身份后，自信就大打折扣。

例如：
*   在公司是领导，说一不二，但离开公司去相亲时就变得自卑。
*   是学校学生会主席，在校园内很有威信，但处理私人感情时却缺乏自信。
*   只在熟悉的酒吧、茶馆或“据点”才感觉自在，像个主人。
*   有些人沉迷网络游戏或喜欢去夜总会，是因为在那里他们能轻易获得现实中没有的“存在感”和“尊贵感”。

如果你从未体验过自信，可以尝试寻找或营造一个让你感到舒适、有掌控感的“主场”。但如果你发现自己的自信完全被某个场景或角色“锁住”，一旦离开就无所适从，这说明你已受到场景自信的束缚。

**总结**：本节课我们一起学习了常人获得自信的四种方式：**提升世俗价值、小团体抱团取暖、依赖特长技能、依赖特定场合与角色**。它们都是有效的“场景自信”来源，能帮助新手快速体验自信的感觉。然而，它们的共同局限在于**自信依赖于外部条件**。如果你发现这些自信正在束缚你，让你无法在更广阔的场景中从容自处，那么就意味着你需要向下一阶段——建立不依赖外物的**核心自信**——迈进了。在未来的课程中，我们将深入探讨什么是核心自信以及如何获得它。
