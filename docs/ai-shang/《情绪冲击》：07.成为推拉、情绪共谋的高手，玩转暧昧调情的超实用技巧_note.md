# 《情绪冲击》：07.成为推拉、情绪共谋的高手，玩转暧昧调情的超实用技巧

## 概述
在本节课中，我们将继续探讨情绪冲击方法的后五点，帮助大家成为推拉、情绪共谋的高手，玩转暧昧调情。

## 1. 夸张曲解
夸张曲解是一种实用的手法，可以用来创造情绪体验。例如，如果你是第一次来听我们的直播，可以关注我们的微信公众号“爱上情感”或“爱上恋爱学”，与专业的情感分析师交流。

## 2. 情绪购买度
吸引是一种生物驱动力，是情绪感受。一个女孩子喜欢上你的过程，其实就是情绪上喜欢你的过程。情绪到位了，她就会合理化，即使你身上有一万个缺点，她也会找到一个优点来合理化。

## 3. 忽略思维的合理性
忽略逻辑的合理性，不要过于在意聊天内容的逻辑性。聊天是为了给女孩情绪体验，而不是解决实际问题。

## 4. 不要做蹊跷反射的男人
避免成为蹊跷反射式的男人，不要说女生问你一个问题，你的答案女生早就猜到了。要让女生无法预料到你的回复，她不知道你下一句会怎么回。

## 5. 模拟场景
模拟场景可以给女生创造情绪体验。例如，你可以跟女生说：“我做好吃的给你，你会从后面抱着我吗？”或者“我亲你一下，你会脸红吗？”

## 6. 情绪共谋
情绪共谋是你要跟女生站在同一个立场之上，例如她遇到了一些让她反感的事情，你要给她站队。

## 7. 反撇
反撇可以给女生制造一个情绪反转，例如先给她制造一个紧张感，一个悬疑感，然后再把它释放出来，再给她一个反转。

## 总结
本节课中我们一起学习了情绪冲击的十个要点，包括夸张曲解、情绪购买度、忽略思维的合理性、不要做蹊跷反射的男人、模拟场景、情绪共谋和反撇等。希望这些技巧能帮助大家在情感交往中更加得心应手。
