## 概述
在本节课中，我们将回顾之前课程中提到的聊天技巧，特别是聊天的金字塔模型，并探讨如何通过有效的聊天技巧来邀约心仪的对象。

## 聊天金字塔模型
金字塔模型分为五层，从底层到高层分别为：
1. **底层**：起歪号、赞美、表达兴趣。
2. **中层**：模糊邀约、潜台预期、暧昧话题。
3. **高层**：表达兴趣、拉升关系、邀约。

**公式**：聊天金字塔模型 = 底层（起歪号、赞美、表达兴趣）+ 中层（模糊邀约、潜台预期、暧昧话题）+ 高层（表达兴趣、拉升关系、邀约）

## 逆向关系推进思维
在进行任何行为之前，先考虑前面的步骤，例如：
- 想要牵手，先约会。
- 想要亲吻，先聊得很好，再见面。

**代码**：逆向关系推进思维 = 先约会 → 牵手 → 亲吻

## 聊天模板
1. **第一轮聊天**：
   - 发表情测试对方兴趣。
   - 表达兴趣，赞美对方。
   - 小拉升，展示自己。
   - 切断聊天。

2. **第二轮聊天**：
   - 称呼测试（小可爱、宝宝等）。
   - 小拉升（想我吗？）。
   - 进攻与平衡，展示自己。
   - 模糊邀约，释放预期。

3. **第三轮聊天**：
   - 聊更多第四层、第五层或第三层的内容。

## 无法进入金字塔高层的原因
1. **理性思维**：过于线性思维，无法将话题引向关系。
2. **词汇量不足**：缺乏词汇量，无法进行深入交流。
3. **缺乏感性思维**：无法运用发生性思维，无法将话题引向关系。

## 学习完课程后仍不会聊的原因
1. 课程内容未记住。
2. 聊天资源有问题（假人、固有印象、表白失败）。
3. 犯了课程中的错误。
4. 形成固有印象。
5. 思维问题（屌丝思维）。
