# 《魅力男神全套》：21 聊如指掌2.0：聊如指掌1.0：第四集

## 概述

在本节课中，我们将学习如何通过逆向关系推进思维来提升聊天技巧，并了解如何运用红绿灯模型和金字塔模型来引导聊天进程。

## 红绿灯模型回顾

### 红灯

- 对方敷衍，如表情冷漠、不回应等。
- 对方测试你，如提出难以回答的问题。

### 黄灯

- 对方测试你，如提出难以回答的问题。
- 对方说一些比较难回答的问题。

### 绿灯

- 主动问你问题。
- 表情和语气词积极。

## 逆向关系推进思维

### 核心概念

- 从最后一步开始，逆向思考每一步需要做什么。

### 应用示例

1. **性**：想和对方发生性关系，需要先舌吻，再搂抱，再调情，再约会，再喜欢对方。
2. **约会**：想和对方约会，需要先让对方喜欢你，再邀请对方出来，再认识对方。

## 金字塔模型

### 核心概念

- 聊天分为三个层次：底层（第一层）、中层（第二层）、高层（第三层、第四层、第五层）。

### 应用示例

1. **底层**：赞美、起外号等。
2. **中层**：邀约、暧昧话题等。
3. **高层**：表达兴趣、聊骚、约等。

## 进攻与平衡

### 核心概念

- 在聊天过程中，需要平衡进攻和吸引。

### 应用示例

- 进攻：通过提问、邀约等方式拉近关系。
- 吸引：展示自己、聊吸引话题等。

## 拉伸

### 核心概念

- 拉伸需要从小处开始，逐步推进。

### 应用示例

- 先从赞美开始，再逐步推进到邀约等。

## 总结

本节课我们学习了逆向关系推进思维、红绿灯模型、金字塔模型、进攻与平衡和拉伸等技巧，这些技巧可以帮助我们提升聊天技巧，更好地引导聊天进程。
