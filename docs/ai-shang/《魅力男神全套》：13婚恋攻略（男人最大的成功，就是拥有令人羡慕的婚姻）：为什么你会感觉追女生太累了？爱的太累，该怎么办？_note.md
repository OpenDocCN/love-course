上一节我们概述了问题的普遍性，本节中我们来看看这背后的根本原因。

### 一个关于“砍树”的比喻

为了更好地理解，我们先讲一个故事。有三个人去砍树：

*   第一个人用电锯，一分钟就能锯断一棵树。
*   第二个人用斧头，花了一个小时砍倒一棵树。
*   第三个人徒手劈树，砍了一天只伤到一点树皮，自己却双手血肉模糊，累得筋疲力尽。

这个故事形象地映射了许多兄弟在感情中的处境。你觉得累，正是因为你是那个“徒手砍树”的人。你缺乏有效的工具和方法，只是在做大量**无用功**。

**核心公式：**
```
疲惫感 ≈ 付出的努力 / 方法的效率
```
当方法效率极低时，即使付出巨大努力，收获甚微，疲惫感就会非常强烈。
