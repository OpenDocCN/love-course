# 《魅力男神全套》课程22：魅力内核（你的强力状态引擎）02.行动力和勇气 🚀💪

在本节课中，我们将要学习如何培养强大的行动力与勇气，这是构建个人魅力的核心引擎。我们将拆解阻碍行动的心理机制，并提供具体的方法来克服惰性与恐惧，最终帮助你以赢家心态面对社交与生活。

## 第一部分：培养行动力

上一节我们介绍了魅力内核的重要性，本节中我们来看看如何具体培养行动力。许多人无法成功蜕变的根本原因是缺乏行动力。缺乏行动力的原因无非是懒惰和恐惧。

恐惧情绪源于对负面反馈的推敲。懒惰情绪则是因为担心自己的行动无效，是白费力气。因此你会感到非常疲惫，提不起精神，不想行动。这其实是你的大脑在欺骗你。

大脑为了让你节省能量，会给你一种虚假的疲劳感。就像一个不经常运动的人，在开始运动后的短短几分钟，会立即感觉到非常疲惫。但此刻体力并没有真正消耗，大脑只是传递了一种虚假的疲劳信号。

大脑试图通过这种伎俩，让你重新躲回舒适区，以避免不必要的能量消耗。大多数人之所以躲在舒适区、好逸恶劳、追求稳定，原因就是我们的大脑不喜欢面临风险。因此人们在最大程度地追求舒适和安逸。

对于大脑来说，你活得好不好、是否精彩并不重要，重点仅仅是活下去。应对这种情况，我为大家提供几个方案。

以下是几个培养行动力的具体方法：

1.  **把过程拆解成更小的步骤**
    如果你一开始就想着要达到最终结果，你会很累、很焦虑，甚至一节课都看不下去。你需要明白学习是一个过程和习惯，没有人可以一瞬间提高自己。这是你必须经历的。当你每天能够进步1%时，那么100天之后，没有人可以阻止你的成功。每天坚持看完一节课。如果实在没有时间，就看半节课。给自己一个规划，每周或每两周去做一次形象建设，去探索一家网红店，去拍摄一组照片，慢慢地改变自己。不要因为慢或进步小就选择直接放弃，因为即使是非常小的步伐，也能够积累成伟大的旅程。

2.  **让自己进入心流状态**
    心流状态是指挑战和能力的平衡。如果挑战太大，能力不足，你就会非常焦虑。如果挑战太小，能力相对较强，你就会感到无聊。如果一件事你为自己设置的挑战和能力相对平衡，你就会沉浸其中，喜欢上这种节奏。换言之，就是挑选一块适合你胃口大小的骨头，既不会让你吃不饱，也不会让你吃到撑破肚子。例如，最开始你可以设置认识三个女生的目标。当你通过学习提升了搭讪和使用社交软件的能力后，可以设置认识十个女生的新挑战。进入心流状态最重要的就是爱上这件事，让社交成为你的爱好。当你发现自己很焦虑、很抗拒时，说明你对应此项挑战的能力不足。你可以降低挑战难度，或提升自己的能力来做出调整。

3.  **积累自己的参考经验库**
    当你出门社交，无论是搭讪、约会还是和女生聊天，不要想着达成太长远的目标。而是要把社交作为打开自己状态的方法。当你能够和异性正常交流时，你会收获愉快的体验，让自己变得更开朗、爱笑，更自然地表达和互动。你需要在这种状态中获得快乐，而不是只有在收获特定结果时才能获得快乐。如果你搞砸了，就把这次搞砸作为一次难得的一手经验。你可以反思暴露出的问题，可以建立不在乎结果的心态，可以让拒绝这件事慢慢变得不敏感。如果你成功了，你就收获了一次正向反馈，增加了信心，丰富了关于成功的参考经验，使你可以行动起来以获得更多成功。所以，只要你去行动，就可以收获参考经验并从中获益。

4.  **正视焦虑与投入**
    你可能会担心这个旅程很难，需要投入很多，还要面临拒绝，所以你不想行动。这其实属于挑战大于能力，让你感到焦虑。这种焦虑是必须的，因为只有在挑战中才能锻炼能力，以让你进入心流状态。你要意识到焦虑的来源是你的大脑不想让你做没有结果的事情。但是你要知道，在狼群中，作为头狼，它可以获得优先的交配权和更多的食物。它绝不是躺着享受这一切，而是要付出更多能量。它必须强壮勇敢，在狼群猎杀猎物时走在最前面。你要成为一个领导者，就必须花费时间精力，付出能量。一开始不要太执着于结果，你需要的是通过行动获得经验，然后在失败中成长。你必须彻底改变自己左蓝的生活。很多人重复这样的生活：做着自己完全不喜欢的工作，当完一天工具人，一脸无精打采地躲回自己的小屋，玩玩游戏，看一下小电影，自己来一发，然后就结束了这一天。也许这种生活很舒适安逸，但这种生活不会天上掉下一个女朋友。为此，你必须走出舒适区改变生活，才能够成为被女生所喜欢的那种男人。

## 第二部分：如何变得勇敢

上一节我们探讨了如何启动行动，本节中我们来看看支撑行动的勇气从何而来。我们首先要讲赢家心态和输家心态。

所谓赢家心态，就是“赢家”是我的身份，无论结果如何，我都是赢家。输家心态就是“输家”是我的身份，无论结果如何，我都是那个输家。

我们举一个例子。你要去参加一场比赛。如果你假定自己会输，那么当你和对方比赛时，就会有一种焦虑感、挫败感，甚至你会失去斗志，早早就在自己内心中投降了。那么最后多半是输掉比赛，而且是理所当然地输掉。就算你侥幸赢了，你可能也是在一种很消极的状态之下赢的，整个过程你会很累。如果比赛之前你假定自己会赢，而且以这种必胜的心态去打比赛，那么整个过程中，你会带着一种勇往直前的气势，你能够尽全力去发挥自己。最后如果你赢了，那么你是赢得天经地义、顺理成章。如果你输了，你也是尽了全力之后所得到的最好结果，你是无冕之王，虽败犹荣。所以外在的结果并不重要，重要的是你的心态。你是否完全打开自己，释放自己，带着必胜的信念去做这件事，并且接纳任何结果。

我们来讲一个百分百法则。所谓的百分百法则，就是一件事你要么做，要么不做。要做就百分之百地去做，不做就百分之百地放弃。最忌讳的就是拖泥带水、犹犹豫豫。“我百分之二十去做吧”，“我百分之二十去搭讪”，“我百分之六十去牵手吧”。**要么做，要么不做，要做就百分百地去做。** 在这个过程中，你一定会遇到负面反馈，一定会遇到失败。但是你想以成功者的姿态去承受失败，还是以垂头丧气的输家形态去等待命运给你成功的恩赐呢？

## 第三部分：如何摆脱痛苦和负面情绪的影响

上一节我们建立了赢家心态，本节中我们来看看如何与不可避免的负面情绪共处。如何面对和接纳痛苦以及黑暗面？

生活中总会有一部分人不喜欢你，你也不可能让所有人都喜欢你。你必须面对这一点。痛苦和黑暗面是永远存在的，就像硬币的两个面。生活就是如此，去面对它，接纳它，而不是抗拒。你所抗拒的仍然存在，不会因为你的抗拒而消失。而你的抗拒唯一带来的，就是你自己更加不舒服、不自在。

悲伤的过往不是好事，但是它帮你认识到那些应该避免的坑，让你以后不会再第二次掉进去。这些经历会成就你，让你变得更加强大、成熟、睿智和有深度。男人成熟不是麻木，不是圆滑世故，而是变得勇敢，能够与内心的痛苦和创伤和平共处。你能够察觉到负面情绪所在，而不会去抗拒和逃避。你会接纳它，拥抱它，因为这是你的一部分。
