# 《魅力男神全套》：09 完美自信（自信是男人魅力的根本，最硬核的心态建设！）：内向自卑的人，如何拥有自信！（四）常人获得自信的方式

## 概述

本节课将探讨内向自卑的人如何拥有自信，重点介绍常人获得自信的方式。

## 场景自信

**定义**： 在特定场景下，通过提供价值获得自信。

**特点**：

*   在熟悉场景中，深知自己能提供价值。
*   在熟悉场景中获得特殊地位，无法在其他场景获得。

**例子**：

*   大学教授在讲台上讲课。
*   酒吧酒保在酒吧里。
*   歌手在演唱会舞台上。

**危害**：

*   阻碍自我提升。
*   限制个人发展。

## 核心自信

**定义**： 在任何场景下，都能感到自信。

**建立方法**：

*   逐步摆脱场景自信。
*   提升自身价值。

## 案例分析

**案例**： 作者通过改变形象、提升自身价值，逐步摆脱场景自信，获得核心自信。

**步骤**：

1.  **改变形象**： 通过打扮、发型等方式提升形象。
2.  **提升价值**： 通过学习、提升认知等方式提升自身价值。
3.  **摆脱场景依赖**： 不再依赖特定场景获得自信。

## 总结

本节课介绍了场景自信和核心自信的概念，并分析了如何建立核心自信。通过逐步摆脱场景自信，提升自身价值，每个人都可以拥有在任何场景下都感到自信的核心自信。
