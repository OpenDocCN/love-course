# 《魅力男神全套》：04 再见，社交恐惧：02.自卑、内向、羞涩人格，走出恐惧的第一步（下集）

(音乐)

## 概述

在本节课中，我们将继续探讨如何克服自卑、内向和羞涩，勇敢地迈出社交的第一步。

## 一、自卑与恐惧

**以下是原文中关于自卑与恐惧的描述：**

*   **原文**：导致我变得更加内向，咱们说我在初中的时候吧，其实有很多次去接触我们班花的机会，就是我们班里边最好看的女孩子，有很多次机会，但是我每一次有机会的时候，我会无视她，我会想办法离她很远，比如说这个。
*   **翻译**：初中时，有很多次接近班花的机会，但我总是回避，尽量与她保持距离。

**核心概念**：

*   **公式**：自卑 → 恐惧 → 回避

## 二、失去机会

**以下是原文中关于失去机会的描述：**

*   **原文**：但是我心里面特别喜欢她，就是这样的一个过程，因为我害怕嘛，比如说心里明明想是吧，就是怎么都张不了口，怎么都没有办法去正式跟他，这个两个人的相处关系，有他在我就感觉我不是我自己了。
*   **翻译**：虽然心里喜欢她，但因为害怕，无法开口，无法与她建立关系，感觉失去了自我。

**核心概念**：

*   **公式**：喜欢 → 害怕 → 失去

## 三、勇敢尝试

**以下是原文中关于勇敢尝试的描述：**

*   **原文**：咱们说我们其实有大量的粉丝，就是因为你们吧，跟我跟当年的我可能有一样的一个所谓的心境，一样的感受，不敢尝试的去跟女生去接触是吧。
*   **翻译**：许多粉丝和我当年一样，不敢尝试与女生接触。

**核心概念**：

*   **公式**：恐惧 → 不尝试 → 失去机会

## 四、克服恐惧

**以下是原文中关于克服恐惧的描述：**

*   **原文**：咱们说最怕的是可能对方明明对你有好感，就是因为你选择胆怯是吧，人家本来对你的印象不错，结果你从来不去跟别人交流一次，你怎么知道他不喜欢你呢，是吧，我们说这种狗血的事情啊，在我上大学的时候经历过很多次。
*   **翻译**：最可怕的是，对方可能对你有好感，但因为你的胆怯，你从未尝试交流，你怎么知道他不喜欢你呢？

**核心概念**：

*   **公式**：恐惧 → 胆怯 → 失去机会

## 五、行动的重要性

**以下是原文中关于行动重要性的描述：**

*   **原文**：咱们说咱们幸运的是什么事啊，幸运的是我们其实很多学员已经选择了行动起来，他们不再去观望别人的人生了，他们试图在把人生的画卷把握在自己手里。
*   **翻译**：许多学员已经选择行动起来，不再观望，而是把握自己的人生。

**核心概念**：

*   **公式**：行动 → 把握人生

## 六、总结

本节课中，我们探讨了自卑、内向和羞涩对社交的影响，以及如何克服这些恐惧，勇敢地迈出社交的第一步。记住，行动是克服恐惧的关键，只有勇敢尝试，才能把握人生的机会。
