# 13、社交定位（为你勾画动态社交的蓝图）：现场约会、搭讪实战能力弱？！现场实战的3个核心沟通层面

## 概述
在本节课中，我们将探讨如何提升现场约会和搭讪实战能力，重点关注三个核心沟通层面：内容、表达和意图。

## 核心概念
- **身份感与内核**：真实的自我总会暴露出来，依靠伪装无法长久。
- **沟通三个层面**：内容、表达和意图。

### 身份感与内核
以下是关于身份感与内核的详细内容：

- **真实的自我总会暴露出来**：无论伪装多久，真实自我总会被发现。
- **内核的重要性**：没有坚实的内核，技巧和惯例只是表面功夫。
- **一致性**：真实自我与表达的一致性至关重要。

### 沟通的三个层面
以下是关于沟通三个层面的详细内容：

- **内容层面**：你说了什么话。
- **表达层面**：你是如何说的，包括语气、表情、肢体语言等。
- **意图层面**：你深层次的意图，包括提供价值和索取价值。
