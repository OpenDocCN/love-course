# 21.恋爱速成班：成为恋爱高手、万人迷，只需要解决的这3个问题（下）

**概述**： 本节课我们将继续探讨成为恋爱高手、万人迷需要解决的三个关键问题，分别是资源问题、聊天问题和所取价值问题。

## 资源问题

**核心概念**： 资源问题指的是稀缺心态，即人在资源稀缺的状态下，难以与异性正常交流。

**解决方案**： 
- 相亲渠道
- 社交软件
- 线下活动
- 酒吧夜店

**选择合适渠道**： 根据个人条件选择最适合自己的渠道。

## 聊天问题

**核心概念**： 聊天是建立关系的重要阶段，但存在以下问题：

- 没有聊天话题
- 为了聊天而聊天
- 过于逻辑

**解决方案**： 
- 学习聊天技巧和流程
- 顺应女生情绪需求
- 关注情绪而非逻辑

## 所取价值

**核心概念**： 价值是产生吸引的前提，包括：

- 生存和繁衍有力的东西
- 联好情绪

**价值与情绪**： 价值与情绪是一对映射关系，价值出现时，情绪会与之对齐。

**提供价值**： 
- 展示自己
- 分享生活
- 分享观点

**避免索取价值**： 
- 不要过度关心对方
- 不要围绕对方转
- 不要过度索取对方的关注

**总结**： 本节课我们学习了成为恋爱高手、万人迷需要解决的三个关键问题，分别是资源问题、聊天问题和所取价值问题。通过学习这些内容，我们可以更好地提升自己的恋爱技巧，成为更受欢迎的人。
