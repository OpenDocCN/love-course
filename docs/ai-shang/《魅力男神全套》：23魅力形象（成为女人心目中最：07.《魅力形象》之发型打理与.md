## 第一部分：发型打理演示

上一节课程介绍了发型的选择与理论知识，本节我们将进行实际操作演示。

打理发型的关键在于使用正确的产品和遵循步骤。以下是打理发型的核心流程。

### 发型打理步骤

1.  **吹干并塑形**
    洗完头后，先用吹风机将头发吹至八、九分干。吹风时需将头发吹得蓬松，并初步吹出你想要的发型轮廓。

2.  **使用发泥塑形**
    发泥是塑造发型的关键产品。取少量发泥于掌心，均匀揉搓开后，涂抹在头发上，并抓出你最终想要的发型造型。

3.  **使用发胶定型**
    仅用发泥造型不够持久，需要使用发胶进行最终定型。正确使用方法是：将发胶瓶口距离头发约20厘米处喷洒，让发胶均匀自然地落在头发上，以保持发型持久。

**核心产品公式**：理想发型 = **吹风机（基础塑形）** + **发泥（精细造型）** + **发胶（持久定型）**

## 第二部分：护肤步骤演示

在了解了护肤的基础知识后，现在我们来演示具体的护肤操作流程。

男性基础护肤主要包含四个步骤，目的是清洁、补水和锁水。

### 护肤步骤详解

以下是完整的护肤流程及对应产品作用。

1.  **洁面乳（清洁）**
    洗脸后，取适量洗面奶于掌心，揉搓起泡后在面部按摩清洗。好的洗面奶应性质温和，不刺激眼睛。

2.  **爽肤水（二次清洁与补水）**
    洁面后使用爽肤水。取少量于掌心，轻轻拍打面部直至吸收。它能深层清洁皮肤并为肌肤补水。

3.  **乳液（滋养与深度补水）**
    爽肤水吸收后使用乳液。同样取适量乳液，拍打面部促进吸收。乳液能为肌肤提供营养并深度补水。

4.  **面霜（锁水）**
    尤其在干燥季节，需要使用面霜锁住肌肤水分，防止水分蒸发。夏季油脂分泌多时可酌情省略此步。

**护肤流程代码表示**：
```plaintext
if (季节 == “秋冬” 或 皮肤干燥) {
  护肤步骤 = [洁面乳， 爽肤水， 乳液， 面霜];
} else {
  护肤步骤 = [洁面乳， 爽肤水， 乳液]; // 可省略面霜
}
```

### 附加护肤与形象管理

完成基础护肤后，还有一些提升细节形象的步骤。

-   **面膜护理**：面膜主要用于集中补水。每周使用1-2次，敷后清洗，再进行常规护肤步骤。
-   **眉毛修整**：使用修眉刀定期修理眉尾和眉峰的杂毛。初学者可先请专业人士修理一次作为模板。之后可使用眉笔或眉粉轻画眉尾，使眉毛更有型。
-   **备用物品建议**：为提升生活便利与社交体贴度，家中可常备卸妆水（油）、隐形眼镜盒及一次性牙刷等物品。
