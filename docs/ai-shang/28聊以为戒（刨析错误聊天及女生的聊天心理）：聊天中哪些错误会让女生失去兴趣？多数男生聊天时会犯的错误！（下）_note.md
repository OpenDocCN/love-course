# 28聊以为戒（剖析错误聊天及女生的聊天心理）：聊天中哪些错误会让女生失去兴趣？多数男生聊天时会犯的错误！（下）

## 概述
在本节课中，我们将继续探讨在聊天中哪些错误会让女生失去兴趣，以及大多数男生在聊天时会犯的错误。

## 聊天中的错误行为

### 1. 过度提问（夺命緬懷問）
- **错误行为**：不断向女生提问，如同查户口。
- **公式**：过度提问 = 失去兴趣。

### 2. 信息轰炸
- **错误行为**：频繁发送消息，过分关心。
- **公式**：信息轰炸 = 过分关心 = 跪舔。

### 3. 聊天平淡无趣
- **错误行为**：总是聊第三方话题，不知如何使话题有趣。
- **公式**：第三方话题 + 无趣 = 平淡无趣。

### 4. 不提供话题，被动接话
- **错误行为**：只知道接女生的话，不会主动提供话题。
- **公式**：被动接话 = 失去兴趣。

### 5. 过分夸赞
- **错误行为**：过分夸赞女生，将其捧得过高。
- **公式**：过分夸赞 = 社交地位不平等。

### 6. 内心戏太多，自作多情
- **错误行为**：对女生的话想得太多，自以为女生喜欢自己。
- **公式**：内心戏太多 = 自作多情。

### 7. 浪漫情节过多
- **错误行为**：过分聊浪漫情节，如同写剧本。
- **公式**：浪漫情节过多 = 过分幻想。

### 8. 逻辑研讨会
- **错误行为**：与女生进行学术性讨论，如减肥、电影制作等。
- **公式**：逻辑研讨会 = 女生反感。

### 9. 表白
- **错误行为**：过早表白，女生不会喜欢。
- **公式**：表白 = 拜拜。

### 10. 强撩强约
- **错误行为**：强行邀约女生，不会得到回应。
- **公式**：强撩强约 = 被拒绝。

### 11. 屌丝的开场白
- **错误行为**：使用屌丝式的开场白，如网上抄的段子、连续发消息等。
- **公式**：屌丝的开场白 = 女生不回覆。

### 12. 深陷友谊区
- **错误行为**：与女生陷入友谊区，无法跳出。
- **公式**：友谊区 + 调情 = 被打压。

## 总结
本节课中，我们学习了聊天中会让女生失去兴趣的12种错误行为。这些错误行为包括过度提问、信息轰炸、聊天平淡无趣、不提供话题、过分夸赞、内心戏太多、浪漫情节过多、逻辑研讨会、表白、强撩强约、屌丝的开场白和陷入友谊区。希望大家能够避免这些错误，学会正确的聊天方式，吸引女生。
