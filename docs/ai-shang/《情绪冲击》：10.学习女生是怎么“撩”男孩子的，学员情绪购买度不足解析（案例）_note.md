# 《情绪冲击》：10.学习女生是怎么“撩”男孩子的，学员情绪购买度不足解析（案例）

## 概述
在本节课中，我们将学习如何通过分析案例，了解女生如何通过情绪冲击来吸引男生，并探讨学员在情绪购买度不足时如何应对。

## 案例一：微信聊天记录分析

**以下是微信聊天记录分析：**

1. **女生主动发起聊天**：女生通过微信主动与男生聊天，并表达了对男生的思念之情。
2. **男生回应**：男生没有直接回应女生的思念，而是以轻松的方式回应，避免了正面回应，降低了需求感。
3. **女生提出加微信**：女生提出加微信，男生同意并提供了微信号。
4. **聊天内容**：聊天内容以轻松、幽默为主，女生表现出活泼、可爱的性格。

**分析**：

- 男生通过避免正面回应，降低了需求感，保持了社交地位。
- 女生通过轻松、幽默的聊天，传递出积极的情绪，增加了吸引力。

## 案例二：学员与女生聊天记录分析

**以下是学员与女生聊天记录分析：**

1. **学员主动发起聊天**：学员通过微信主动与女生聊天，并表达了对女生的好感。
2. **女生回应**：女生回应积极，但情绪购买度不足。
3. **学员邀约**：学员多次尝试邀约女生见面，但女生都以各种理由拒绝。
4. **聊天内容**：聊天内容以逻辑为主，缺乏情绪冲击。

**分析**：

- 学员没有关注女生的情绪，而是以逻辑思维为主，导致女生情绪购买度不足。
- 学员在邀约时过于急切，没有给女生足够的时间和空间。

## 总结
本节课中，我们学习了如何通过分析案例，了解女生如何通过情绪冲击来吸引男生，并探讨了学员在情绪购买度不足时如何应对。在今后的交往中，我们要关注对方的情绪，通过情绪冲击来提升吸引力，避免陷入逻辑思维，导致关系破裂。
