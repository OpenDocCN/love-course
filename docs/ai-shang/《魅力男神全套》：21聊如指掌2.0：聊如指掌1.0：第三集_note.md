# 《魅力男神全套》：21 聊如指掌2.0：聊如指掌1.0：第三集

## 概述
在本节课中，我们将学习如何运用聊如指掌1.0中的聊天金字塔模型来提升聊天技巧。

## 聊天金字塔模型
聊天金字塔模型由五层组成，分别代表不同的聊天阶段：

1. **陌生层**：初次接触，话题多为问候、天气等。
2. **熟悉层**：关系进一步，可以聊年龄、工作等。
3. **暧昧层**：关系更亲密，可以聊一些暧昧话题。
4. **情人层**：确定关系，可以聊私密话题。
5. **聊骚层**：接近确立关系，可以问很多私密问题。

## 金字塔模型应用案例
以下是一个聊天案例，展示如何运用金字塔模型：

1. **陌生层**：女孩发来一段英文，我直接指出是群发。
2. **熟悉层**：询问女孩是否在济南，进入熟悉层。
3. **暧昧层**：可以聊一些暧昧话题，如“你想我了没有”等。

## 总结
本节课中，我们学习了如何运用聊天金字塔模型来提升聊天技巧。通过了解不同聊天阶段的话题，我们可以更好地与对方交流，建立更亲密的关系。
