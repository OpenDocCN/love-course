# 《情绪冲击》：04.培养发散感性思维，把任意话题聊成“很撩”的感觉

**概述**： 本节课我们将学习如何培养发散感性思维，将任意话题聊成“很撩”的感觉，从而给女孩带来情绪冲击。

## 1. 情绪冲击的重要性

**核心概念**： 情绪冲击 = 情绪体验 + 情绪共鸣

**内容**： 情绪冲击对于吸引女孩至关重要。即使你的条件一般，只要你能给女孩带来丰富的情绪体验，她就会合理化你的优点，从而喜欢上你。

## 2. 男性思维与女性思维

**核心概念**： 男性思维：解决问题式；女性思维：分享感受式

**内容**： 男性思维容易陷入逻辑思维，导致话题变窄，不利于与女生互动。而女性思维则更注重分享感受和体验，话题更广泛。

## 3. 培养感性思维的方法

**核心概念**： 分享加感受句式、描述句式、拆掉思维墙

**内容**：

* **分享加感受句式**： 在分享内容时加入自己的感受，例如：“我今天吃了一碗面，还是原来的味道，哎呀，真是满满的青春回忆啊。”
* **描述句式**： 将疑问句转换为描述句，例如：“你今年多大？”改为“那你应该是95年的吧。”
* **拆掉思维墙**： 培养发散思维，将任意话题关联到情绪，例如：将“杯子”关联到“我好想做你的杯子，这样你就可以天天把我捧在手心里了。”

## 4. 刻意练习

**核心概念**： 刻意练习是提升感性思维的关键

**内容**： 通过刻意练习，我们可以打破原有的思维模式，培养从任意话题都能关联到情绪的能力。

## 总结

**内容**： 本节课我们学习了如何培养发散感性思维，将任意话题聊成“很撩”的感觉。通过刻意练习，我们可以提升自己的情绪冲击能力，从而更好地吸引女孩。
