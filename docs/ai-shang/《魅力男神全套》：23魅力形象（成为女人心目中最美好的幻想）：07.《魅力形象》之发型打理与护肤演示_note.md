## 发型打理

**以下是发型打理的步骤：**

1. **选择发型**：首先，去理发店选择一款你想要的发型。
2. **使用发泥**：洗完头后，用吹风机吹出你想要的造型，吹干后用发泥塑造发型。
3. **使用发胶**：为了保持发型，使用发胶进行定型。

**公式**：

发型打理 = 选择发型 + 使用发泥 + 使用发胶

## 护肤步骤

**以下是护肤的步骤：**

1. **洁面**：使用洁面乳清洁面部。
2. **爽肤水**：使用爽肤水进行深度清洁和补水。
3. **乳液**：使用乳液进行深度补水。
4. **面霜**：使用面霜锁住水分。

**公式**：

护肤 = 洁面 + 爽肤水 + 乳液 + 面霜
