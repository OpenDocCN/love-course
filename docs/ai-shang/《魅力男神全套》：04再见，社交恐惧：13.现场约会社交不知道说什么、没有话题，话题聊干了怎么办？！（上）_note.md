# 《魅力男神全套》：04 再见，社交恐惧：13.现场约会社交不知道说什么、没有话题，话题聊干了怎么办？！（上）

## 概述
在本节课中，我们将学习如何克服现场约会时的话题枯竭问题，以及如何通过调整心态和表达方式来提升约会体验。

## 1. 不要专注于“说什么”
**公式**：不要专注于“我应该说什么”，而要专注于“我感受到了什么”。

- 在约会中，不要过分担心话题是否合适，而是要关注自己的感受。
- 你的感受本身就是话题，无需刻意寻找。

## 2. 专注于氛围
**公式**：80%到90%的精力关注互动氛围，10%到20%关注话题内容。

- 不要将注意力过多放在话题本身，而是要关注双方的感受和互动氛围。
- 通过分享你的感受和热情，激发对方的兴趣。

## 3. 状态换体
**公式**：打开“有趣、放松、激情”三种情绪开关。

- 在约会前，先调整自己的情绪，让自己处于轻松愉快的状态。
- 将这种情绪传递给对方，让对方感受到你的积极能量。

## 4. 热情分享
**公式**：分享你热爱的事物，激发对方的兴趣。

- 不要担心话题是否新鲜有趣，只要你能表达出热情，就能吸引对方。
- 分享你对生活的热爱，让对方感受到你的活力。

## 5. 关注互动
**公式**：关注互动氛围，而非话题内容。

- 不要过分关注话题本身，而是要关注双方之间的互动。
- 通过关注互动氛围，让话题自然流畅地展开。

## 总结
本节课中，我们学习了如何克服现场约会时的话题枯竭问题。通过调整心态、关注氛围、状态换体、热情分享和关注互动，我们可以让约会变得更加愉快和成功。
