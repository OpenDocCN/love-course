# 聊如指掌2.0：6、聊天的作用与目的

2. **拉伸关系**：在吸引女生的基础上，进一步发展关系。

19. **情商**：展示你的情商。

### 吸引模型总结
将个人价值分为以下三点：

1. **当下的价值**：如旅行、职业、健身等。
2. **过去的价值**：如留学经历、创业经历等。
3. **未来的价值**：如上进心、目标感等。
