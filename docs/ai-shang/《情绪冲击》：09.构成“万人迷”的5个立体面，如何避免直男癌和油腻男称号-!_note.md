# 《情绪冲击》：09.构成“万人迷”的5个立体面，如何避免直男癌和油腻男称号

## 概述
在本节课中，我们将学习如何通过五个立体面来构建“万人迷”形象，避免成为直男癌和油腻男。

## 1. 有趣吸引
**公式**：有趣吸引 = 尝试其他男人不具备的、不敢尝试的东西

**描述**：想要在女生面前脱颖而出，你需要做一些与众不同的事情，尝试一些其他男人不具备的、不敢尝试的东西。

## 2. 浪漫贴心
**公式**：浪漫贴心 = 给女生一些浪漫贴心的东西，让她知道你对她的投入

**描述**：为了让女生感觉你对她有投入，你需要给她一些浪漫贴心的东西，让她知道你是对她一个人这样，而不是对所有的人都是这样。

## 3. 真实立体
**公式**：真实立体 = 敢于表达真实需求，展现真实的自我

**描述**：敢于表达真实需求的男人是强大的，是鹤立鸡群的。真实的男人是受女人欢迎的，让人感觉很特殊的。

## 4. 亦正亦邪
**公式**：亦正亦邪 = 一正一邪，张弛有度

**描述**：一正一邪是构成魅力男人最基本的两个面，可以立即让女生拥有情绪体验。任何东西都要张弛有度，适可而止。

## 总结
本节课中我们一起学习了如何通过五个立体面来构建“万人迷”形象，避免成为直男癌和油腻男。希望这些内容能帮助你提升自己的魅力，成为更好的自己。
