# 聊如指掌2.0：13.如何邀约——邀约的时机、理由

## 概述
在本节课中，我们将学习如何有效地邀约对方，包括邀约的时机、理由以及如何运用聊天金字塔模型来提升关系。

## 课程回顾
### 心态与模型
1. **心态**：保持积极、开放的心态，对聊天充满兴趣。
2. **聊天金字塔模型**：分为五层，从底层到高层，逐步提升关系。
3. **红绿绿模型**：识别聊天中的信号，判断对方的态度。
4. **细影模型**：通过细节了解对方。
5. **话题与关系**：选择合适的话题，建立良好关系。

### 金字塔模型详解
1. **底层（第一层、第二层、第三层）**：通过起外号、赞美、表达兴趣等方式提升关系。
2. **中层（第二层、第三层、第四层）**：进行模糊邀约，浅谈预期，聊暧昧话题。
3. **高层（第三层、第四层、第五层）**：表达兴趣，邀约，聊更多深入话题。

### 逆向关系推进思维
在进行任何行为之前，先考虑前面的步骤，例如先约会，再牵手。

### 聊天模板
1. **第一轮聊天**：发送表情，表达兴趣，展示自己，切断聊天。
2. **第二轮聊天**：称呼测试，小拉伤，展示自己，模糊邀约，释放预期。
3. **第三轮聊天**：聊更多深入话题。

### 无法进入高层的原因
1. **理性思维**：过于线性思维，无法运用发散性思维。
2. **词汇量不足**：缺乏词汇量，无法进行丰富聊天。
3. **资源问题**：聊天对象不合适，或已有负面印象。

### 改进方法
1. **培养感性思维**：学会发散性思维。
2. **积累词汇量**：多看书、看电影、电视剧等。
3. **重复学习课程**：加深理解，内化于心。

## 总结
本节课中，我们学习了如何邀约对方，包括邀约的时机、理由以及如何运用聊天金字塔模型来提升关系。希望大家能够将所学知识应用到实际聊天中，提升自己的聊天技巧。
