# 28 聊以为戒（剖析错误聊天及女生的聊天心理）：99%的男人都在犯的聊天错误：索取式聊天（缺乏灵魂案例）

## 概述
在本节课中，我们将分析一个典型的错误聊天案例，探讨男生在聊天中常见的错误，以及如何避免这些错误。

## 错误聊天案例分析

### 1. 缺乏回应
**案例**：男生收到女生“哈喽”的问候，只回复了一个笑的表情，没有提供任何话题。

**分析**：这种回应方式会让女生感到被忽视，话题中断，女生会等待男生继续聊天。

### 2. 缺乏关联
**案例**：男生在女生提到“加上之后小哥哥也不说话”时，没有做出任何回应。

**分析**：男生没有将话题与女生联系起来，导致女生感到被忽视，可能会撤回聊天。

### 3. 缺乏主动
**案例**：男生在女生主动发“新年快乐”时，只回复了“同乐”。

**分析**：男生没有主动延续话题，女生只能自己带领话题，这表明女生想要与男生聊天。

### 4. 缺乏自我
**案例**：男生在女生询问“你呢”时，没有回答自己的名字，而是询问女生的名字。

**分析**：男生没有主动提供信息，导致女生感到被忽视，可能会撤回聊天。

### 5. 缺乏情绪
**案例**：男生在女生提到“你这是终于舍得看手机了吗”时，没有做出任何回应。

**分析**：男生没有表达自己的情绪，导致女生感到被忽视，可能会撤回聊天。

### 6. 缺乏引导
**案例**：男生在女生提到“你也不给我说晚安，你也不给我说早安”时，没有做出任何回应。

**分析**：男生没有引导话题，导致女生感到被忽视，可能会撤回聊天。

### 7. 缺乏一致性
**案例**：男生在女生提到“你不在身边的时候，我吃的美食火锅可伤心了”时，没有做出任何回应。

**分析**：男生没有保持话题的一致性，导致女生感到被忽视，可能会撤回聊天。

## 总结
本节课中，我们分析了男生在聊天中常见的错误，包括缺乏回应、缺乏关联、缺乏主动、缺乏自我、缺乏情绪、缺乏引导和缺乏一致性。为了避免这些错误，男生需要学会主动回应、与女生建立联系、表达自己的情绪和想法，并保持话题的一致性。
