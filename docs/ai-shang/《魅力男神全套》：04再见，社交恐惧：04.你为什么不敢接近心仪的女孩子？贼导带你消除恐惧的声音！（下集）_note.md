# 《魅力男神全套》课程04：再见，社交恐惧 🧠
## 04. 你为什么不敢接近心仪的女孩子？贼导带你消除恐惧的声音！（下集）

在本节课中，我们将深入探讨阻碍你接近心仪女生的核心恐惧。我们将分析这种恐惧的先天与后天成因，并理解其运作机制。通过学习，你将明白为何单纯依靠“说服自己”无效，以及如何通过实际行动来削弱恐惧。

科学家曾进行过一个关于大雁的有趣实验。大雁在天上飞行时，有时排成一字，有时排成人字。实验中，一些大雁由人工孵化，而非由父母自然孵化。这些大雁从未见过成熟大雁的样子，也从未见过天敌（例如老鹰）的模样。

当向这些小雁展示成熟大雁飞行的画面时，它们会表现出向往和兴奋的情绪。然而，当向它们展示老鹰捕食的画面时，它们会立即表现出恐惧、挣扎和焦虑的情绪。这种反应就是本能。尽管它们从未见过大雁或老鹰，但在面对这两种事物时，身体本能触发了预警反应和条件反射。我们说，这就是刻在基因里的本能。

上一节我们介绍了恐惧的生物学本能基础，本节中我们来看看这种本能如何具体体现在社交中。

最常见的先天性社交障碍之一，就是接近女性时的焦虑，即“接近焦虑”。许多女性不理解这种焦虑，认为“不过是过来说句话，勇敢点就行了”。因为她们自身通常不存在这种接近焦虑，所以想当然地认为人不会有这种感受。

这种先天性社交障碍，在搭讪女性时尤为明显。如果你不确定自己是否有这种障碍，可以尝试一下：当一个心仪的女孩经过时，仅仅产生“我要去搭讪她”这个念头，预警系统就会开始干涉你，让你感受到紧张情绪。

这种先天性社交障碍的严重程度，与女性的吸引力正相关。对方条件越好，你越喜欢她，这种障碍就越严重，你的焦虑和紧张感就越强。这被称为“接近焦虑”，它是一种刻在男性骨子里的基因本能。

如果你也有相同问题，请相信，这并非完全由自卑或内向导致。这是一种先天性的社交障碍，绝大多数正常男性都会存在。这种障碍会一直存在于你的身体里，只要你是正常人，它就**无法被完全消除**，只能被**克制、削弱或放大**。

多数男人在遇到心仪的陌生女生时，都会有这种感觉：立即感到紧张焦虑、心跳加速、浑身不自在、无法直视对方。如果你还没有这种感觉，可以在听完课后明天去试试。当你产生接近她的想法时，这种恐惧感会加剧，心跳更快，身体进入一种面对威胁的戒备状态。

以下是当你想要行动时，脑海中可能出现的否定声音列表：
*   **自我怀疑**：我今天穿得不够帅。
*   **预设困难**：万一她有男朋友怎么办？
*   **害怕评价**：周围人看到我搭讪失败多丢脸。
*   **羞耻感**：我感觉自己不配占用她的时间。
*   **顾虑他人**：她好像正和朋友在一起，会打扰到她。

这些负面声音会从脑海中迸发出来，影响你的行动。

如果你没有被这些否定声音淹没，并开始接近她，随着距离缩短，紧张感会达到巅峰。尤其是在面对她、想说第一句话时，焦虑感最强。

然而，一旦你说出第一句开场白（例如：“嘿，你好，我想认识你”），紧张感会立刻大幅减弱甚至消失。即使对方反应不佳，没有给你联系方式，你也会因为紧张情绪得到释放而感到轻松。如果你能做到坚持跟女生打招呼，并且以后遇到女生依然敢这么做，那么你已经超过了绝大多数男人。

其实，这种焦虑感是一种假想敌，是大脑的预警系统。如果你去做了（搭讪），并发现女人没那么可怕，你就向大脑证明了“搭讪并不危险”。于是，大脑会解除或削弱这种预警系统，焦虑感就会降低。

反之，如果你每次产生搭讪念头却没有行动，这种行为会形成“创伤”，让恐惧感加剧，接近焦虑越来越高，最终导致你无法搭讪。

搭讪行为本身是一把**双刃剑**。它可以强化你，削弱焦虑感，为你赢得认识女生的机会（奖励）。但如果你没有完成行动，就会受到情绪上的惩罚（恐惧加深），这种行为模式在你心中就不再被支持。如果你成功了，它会成为被奖励的模式，你以后会更愿意去做。

如果你最终没有行动，没有得到紧张感的释放，你以后会更加不愿意搭讪，因为你受到了惩罚。搭讪会成为你内心深处的恐惧，你越不敢做，它就越根深蒂固。

如果你面对焦虑时行动起来，获得了成就感或机会，你就会受到强化，搭讪会成为你人生的奖赏。随着行动次数增加，恐惧感会被不断压缩。最终，你可能只是深呼吸一下，就能直接上前搭讪，没有任何问题。

我们人生中的很多事情，尤其是社交行为（如搭讪、演讲、在人群中引人注目），这些破坏“和谐感”的事情都会触发大脑的预警系统，产生负面声音和恐惧感。但如果你能直视它，它都能被消除或削弱到很低的程度。

关于消除恐惧，蝙蝠侠（诺兰三部曲）中有一句话：**消除恐惧最好的办法就是直面恐惧**。你害怕什么，就去接近它、研究它。时间久了，大脑会发现预警系统的必要性没那么强，恐惧感就会减弱。

就像普通人害怕蛇，这是一种先天性障碍。但专业的饲养员一开始也害怕，他们通过直面恐惧、了解蛇的习性，向大脑证明“蛇不会带来生命灾难”，从而让预警系统变弱。但这种天性不会被完全消除。

举个例子，我大学勤工俭学时，在一个加工鱼的厂子处理生鱼片。鱼死后取出内脏，但仍有神经反射，有时会突然跳动，让人害怕。但天天面对，我就不再害怕了。可几年后再次碰到类似情况，我依然会害怕。这说明恐惧感没有消除，只是在持续面对的那段时间被减弱了。

这种先天性焦虑是人类低级生存本能的一部分。我们的祖先可能因蛇受伤，于是“看到蛇就恐惧”被刻进了基因。即使生活在文明社会，这种本能也变化不大。对于先天性恐惧，你只能通过**重复行动、直面恐惧**来让它变弱，但无法完全消除。

消除这种先天性社交障碍的方法，就像饲养员一样：**直面你的恐惧**。很多时候，你在逻辑上认定一件事（例如：“我明天一定要去搭讪”），但真正执行时，你的情绪未必支持。情绪和本能会试图用负面情绪和障碍把你逼退。

我们之前讲过，你的逻辑目标（如：健身、学习、搭讪）可能不被情绪喜欢，而刷手机、吃垃圾食品这些行为却被情绪支持。这就产生了“合理化”。在搭讪时，你的“身份感”会发出否定声音，阻止你行动。

你的情绪有一个“身份感”，你的逻辑也有一个“身份感”。有时逻辑认定的身份感太高（“我应该是个很酷的人”），但情绪上的身份感很低（“我只是个普通人”）。执行时，情绪上的身份感会把逻辑拽下来，阻止你行动。觉得自己“不配占用她的时间”、“没资格认识女孩子”，都属于身份感问题。

出于本能原因，我们常常无法仅靠逻辑战胜情绪。当你遇到一个女孩，逻辑想搭讪，情绪却阻止你，双方对抗时，你往往会僵住。大脑认为这是高风险事件，并通过心跳加速、浑身不适等身体预警来极力劝退你。

这时，你不应该与情绪对抗，因为对抗无效。你应该做的是：**不要问情绪“要不要做”，先执行起来**。直接走过去打个招呼。完成后，大脑会发现“原来没有危险”，以后对你的束缚感就会减弱。

如果你与内心声音对抗，或者因不敢行动而自责（例如：朋友激将后仍不敢上，或自己挣扎后错过机会），你会更加内疚，认为自己“很怂、很差、很自卑”。这会加剧你的内向，更不敢迈出半步。这一点在我们的“魅力内核”课程中有详细讲解。

搭讪恐惧不纯粹是先天性社交障碍，也包含许多后天因素。例如，你可能会害怕周围人的看法：水果摊老板会不会觉得我是流氓？同事看到会不会嘲笑我？这些后天性的、不真实的恐惧也在影响你。

这些后天性的社交障碍，大多由“创伤”引起，即“社交疼痛”。我们在社交中感受到的不愉快、难受的情绪，就是社交疼痛。

社会通常知道身体受伤要送医，但情绪受伤却无人告诉你该怎么办。人们只会说“这个人很怂”，却从未真正关注内心的伤痛。神经学家研究发现，社交层面的负面反馈（如拒绝、否定、嘲笑）确实会带来伤痛，这种疼痛是身体层面的。大脑处理这种否定感的区域，与处理身体痛苦的区域基本一致，吃止痛药甚至能缓解。

遗憾的是，并非所有情绪痛苦都能靠吃药缓解。大部分情况下，人们只能“听天由命”，或依赖专业干预。没人会在意你内心的伤害，暴露它甚至可能被说“矫情”。这也是大多数人存在社交疼痛的原因：不愿暴露，也得不到治疗。

在文明社会，我们重视身体创伤，却对心理创伤视而不见，甚至嘲笑有社交障碍的人。这是因为无法感同身受。

有一个精心设计的实验：让被实验者和两个演员一起玩互相传皮球的游戏。玩到一半，两个演员开始只互相传，不再传给被实验者。被实验者立刻感到被孤立，产生痛苦、尴尬的情绪。

即使实验结束后，实验人员告知这只是设计好的游戏，演员并非针对个人，甚至谎称“球里是炸弹，不传给你是保护你”，被实验者的痛苦情绪依然无法立刻缓解。这说明，**逻辑说服无法缓解情绪层面的痛苦**。

你无法通过逻辑说服自己“这没什么大不了”来消除被拒绝的痛苦。你的情绪并不认为这是小事。我们无法用逻辑控制和改变情绪，只能是情绪影响逻辑（即“合理化”）。正如你无法逻辑说服一个女生喜欢你，你也无法逻辑说服自己不痛苦。

本节课我们一起学习了阻碍你接近心仪女生的恐惧声音。我们发现，恐惧很大一部分源于不敢直视它，让它成为了心中的幽灵。有些是先天性恐惧，只是大脑的预警系统，并非事情本身那么危险。如果你不去消除它，它会加深；如果你直面它，它会变弱。行为模式由此改变。我们还通过实验了解到，被拒绝的痛苦无法通过逻辑说服来缓解。
