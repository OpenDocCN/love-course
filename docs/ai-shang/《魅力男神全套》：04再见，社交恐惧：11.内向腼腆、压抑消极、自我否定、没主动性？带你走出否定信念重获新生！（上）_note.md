## 概述
在本节课中，我们将探讨内向腼腆、压抑消极、自我否定以及缺乏主动性等问题，帮助大家走出否定信念，重获新生。

## 核心概念
**RAS（Reptilian Attention System）**：爬行动物注意力系统，即人类大脑的信息过滤系统。

## 列表：否定信念的形成
以下是否定信念形成的过程：

1. **负反馈**：在成长过程中，遇到负面的反馈，如被嘲笑、被误解等。
2. **信念形成**：将这些负面反馈内化为自己的信念，如“我是个失败者”、“我是个白痴”等。
3. **RAS劫持**：信念形成后，RAS会过滤掉对自己不利的证据，只关注能证明自己信念的证据。
4. **循环强化**：通过不断寻找证据来证明自己的信念，使信念更加坚固。

## 列表：RAS劫持的表现
以下是RAS劫持的表现：

1. **选择性过滤**：只关注能证明自己信念的证据，忽略其他证据。
2. **过度解读**：对周围的事物进行过度解读，使其与自己的信念相关联。
3. **恐惧和焦虑**：害怕别人发现自己的信念，担心自己被否定。

## 列表：走出否定信念的方法
以下是走出否定信念的方法：

1. **反思**：回顾自己的经历，找出形成否定信念的原因。
2. **改变视角**：尝试从不同的角度看待问题，避免过度解读。
3. **积极行动**：通过行动来证明自己的价值，增强自信心。
4. **寻求支持**：与朋友、家人或专业人士交流，寻求帮助。
