# 21.恋爱速成班：《魅力内核》试看课：什么是魅力内核

## 概述
在本节课中，我们将探讨“魅力内核”的概念，了解其重要性以及如何通过提升内在来增强个人魅力。

## 核心概念
**内核（Inner Game）**：指的是内在的心态调整和自我提升的过程，旨在强化个性、社交能力和气场。

### 内核内容
1. **内核的作用**：
   - 调整心态
   - 提升状态和能量
   - 强化个性和社交能力
   - 增强气场

2. **内核与技巧的区别**：
   - 内核关注自我提升，而非对他人施加影响。
   - 内核帮助个人成为更好的自己，而非仅仅追求短期效果。

3. **内核修炼的重要性**：
   - 终身修炼
   - 帮助实现彻底的蜕变

### 恋爱、社交、自我提升的三个层次
1. **基础层次（Happiness层次）**：
   - 依赖物质条件（外貌、房子、车子、工作、社会地位）来吸引女人。
   - 容易陷入痛苦和自卑。

2. **技巧层次**：
   - 依赖社交技巧和惯例来吸引女人。
   - 长期依赖技巧会导致不自然和压力。

3. **本源层次（我是谁）**：
   - 关注内在的显空、认知、身份感、独特性和一致性。
   - 魅力自发，无需刻意追求。

## 总结
本节课中，我们学习了魅力内核的概念及其重要性，了解了恋爱、社交、自我提升的三个层次。通过提升内在，我们可以成为更好的自己，吸引到更多优秀的人。
