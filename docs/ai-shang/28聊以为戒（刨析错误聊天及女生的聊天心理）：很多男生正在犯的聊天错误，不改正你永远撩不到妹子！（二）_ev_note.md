# 28 聊以为戒（剖析错误聊天及女生的聊天心理）：很多男生正在犯的聊天错误，不改正你永远撩不到妹子！（二）_ev

## 概述
在本节课中，我们将分析一个男生与女生之间的聊天案例，探讨男生在追求女生过程中常见的错误，以及如何避免这些错误。

## 聊天案例分析
以下是男生与女生的聊天记录：

**男生**： 我知道懂得起。

**女生**： 不要在直播间里说加主播微信，我平时不怎么看私信。

**男生**： 昨天晚上一直都没有看到，我们不知道这是什么。

**女生**： 因为只是新鲜感。

**男生**： 时间是最好的证明。

**女生**： 过了这个劲儿，你就会发现接受不了。

**男生**： 接受不了什么？

**女生**： 前期没有粉丝的时候，新主播都是靠大哥尬聊互动来生存的。

**男生**： 这个我知道。

**女生**： 所以呢，我会去说一些带暧昧的话，但这都是必修课。

**男生**： 没有男人会接受自己喜欢的女孩子，跟别人谈情说爱，所以你还是不要太认真。

## 错误分析
1. **表白过早**：男生在女生对他没有好感时就表达了追求的意愿，这会形成追逐关系，不利于进一步发展。
2. **过度跪舔**：男生盲目夸赞女生，并试图通过关心和送礼物来赢得好感，这些行为反而会降低自己的吸引力。
3. **逻辑聊天**：男生只聊逻辑话题，不懂得如何与女生建立情感联系。
4. **过度关心**：男生过度关心女生的饮食和生活，这种关心反而会让人感到压力。

## 总结
本节课中，我们分析了男生在追求女生过程中常见的错误，并提出了相应的改进建议。希望同学们能够从中吸取教训，避免在追求女生时犯类似的错误。
