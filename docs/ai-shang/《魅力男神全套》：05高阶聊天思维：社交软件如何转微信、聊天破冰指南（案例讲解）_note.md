**社交软件定义与优势**

* 社交软件是收集资源的重要入口。
* 优势：无需面对面交流，可筛选潜在对象，效率高。

**为什么要转微信**

* 微信定位相对私密，社交软件为陌生人社交。
* 微信朋友圈更完善，用户活跃度高。
* 转微信可进行更深入的交流。

**照片墙的重要性**

* 照片墙是社交软件的核心。
* 通过照片了解个人形象、穿着、谈吐等。
* 照片墙对基础匹配至关重要。

**如何将社交软件转为微信**

1. **选择合适的社交软件**：选择需要双方匹配的社交软件，避免陌生人随机聊天。
2. **避免匿名玩法**：匿名玩法可能导致浪费时间，且对方价值不高。
3. **寻找有需求的人群**：选择有明确需求的社交群体，如相亲软件。
4. **关注照片墙建设**：提升照片墙质量，增加基础匹配机会。

**匹配难度加大**

* **男多女少**：社会环境导致女性身价上升。
* **女生被钓次数多**：女生对男性要求越来越高。
* **竞争对手升级**：年轻一代、中高产子弟对社交有更高要求。
* **二八法则**：80%的女性只选择20%的男性。
* **马太效应**：强者越强，弱者越弱。

**转微信难度加大**

* **生物驱动力不足**：对方可能只是出于无聊或虚荣心。
* **女生谨慎添加微信**：考虑对方是否真心。
* **聊天技巧不足**：照片好但不会聊天，容易被定义为聊友。
