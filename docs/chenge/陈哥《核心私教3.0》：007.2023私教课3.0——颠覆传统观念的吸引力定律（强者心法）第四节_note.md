## 吸引力定律与喜欢
**吸引力定律**是一种感觉，而非一种必须的需求。**公式**：吸引力 = 感觉 + 挑战性。

### 喜欢的短暂性
喜欢通常不会长久，它可能只是欲望的体现。**公式**：喜欢 ≠ 持久的爱。

### 习惯养成大于喜欢
习惯养成可以胜过喜欢一个人。**公式**：习惯养成 > 喜欢一个人。

### 降低期待值
降低对对方的期待值，保持真实自我，避免过度幻想和压抑自己的行为。

## 反向吸引
### 不要让对方过早摸透你
让对方永远摸不透你，保持神秘感，避免过早暴露自己的目的和需求。

### 不要过度主动
过度主动显得廉价，保持适当的距离，让对方有探索你的空间。

### 推拉技巧
通过适当的拒绝和热情，让对方产生情绪波动，增加吸引力。

### 拒绝的艺术
适当的拒绝可以增加你的价值，让对方更加珍惜你。

### 欲擒故纵
不要让对方轻易得到你的好，让对方付出代价，增加你的吸引力。
