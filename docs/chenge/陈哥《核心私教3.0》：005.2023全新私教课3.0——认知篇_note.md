# 陈哥《核心私教3.0》：005.2023全新私教课3.0——认知篇

## 概述
在本节课中，我们将学习如何提升个人软实力，包括认知、阅历、见识和故事讲述能力，以改善社交和两性关系。

## 认知提升
**核心概念**：认知差距是人与人之间最大的差距。

**步骤**：
1. **认识自我**：了解自己的优势和劣势，找到适合自己的赛道。
2. **持续学习**：不断学习新知识，提升自我认知。
3. **反思总结**：从失败中学习，总结经验教训。

## 阅历与见识
**核心概念**：阅历和见识是个人成长的重要途径。

**方法**：
1. **多接触**：扩大社交圈，接触不同的人和事。
2. **多经历**：积极参与各种活动，积累经验。
3. **多思考**：从经历中总结经验，提升见识。

## 故事讲述
**核心概念**：故事是最好的谈资，也是启发他人的方式。

**步骤**：
1. **学习故事结构**：时间、事件、人物、反转、结论。
2. **练习讲故事**：多听、多讲、多练习。
3. **分享个人经历**：将个人经历转化为故事，与他人分享。

## 总结
本节课中，我们学习了如何提升个人软实力，包括认知、阅历、见识和故事讲述能力。通过不断学习和实践，我们可以成为一个更加优秀的人，改善社交和两性关系。

## 课后作业
1. 分享个人经历，编辑成故事，发在群里。
2. 复盘聊天案例，总结经验教训。
