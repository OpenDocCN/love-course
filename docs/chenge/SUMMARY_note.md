## 一、吸引力开关

**公式**：吸引力 = 价值 x 信任

**代码**：`attraction = value * trust`

吸引力是建立关系的关键，而价值与信任是构成吸引力的两个核心要素。

## 二、个人形象建设

**公式**：个人形象 = 外在形象 x 内在形象

**代码**：`image = outer_image * inner_image`

个人形象是建立吸引力的重要基础，外在形象和内在形象共同构成了个人形象。

## 三、高价值自己的方式

**公式**：高价值 = 自我认知 x 自我提升

**代码**：`high_value = self_cognition * self_improvement`

高价值是建立信任的关键，自我认知和自我提升是构成高价值的两个核心要素。

## 四、展示面的作用

**公式**：展示面 = 展示内容 x 展示方式

**代码**：`presentation = content * method`

展示面是传递价值的重要途径，展示内容和展示方式共同构成了展示面。

## 五、聊天三大步骤

2. **深入交流**：挖掘共同点，建立信任。
3. **结束聊天**：留下印象，期待下次交流。

## 七、师娘精讲长期关系维护

**公式**：长期关系 = 沟通 x 理解 x 支持

**代码**：`long_term_relationship = communication * understanding * support`
