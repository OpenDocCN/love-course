# 陈哥 核心私教课2.0：私教课6：聊天三大步骤 🎤

## 概述

在本节课中，我们将学习如何进行有效的聊天，并掌握聊天的三大步骤。通过学习这些技巧，你将能够更好地与他人沟通，建立良好的人际关系。

## 聊天三大步骤

### 第一步：自来熟阶段

*   **心态**：将对方视为朋友，而非陌生人，营造自来熟的氛围。
*   **话题**：从日常话题开始，逐渐过渡到与对方相关的话题。
*   **展示自己**：通过分享经历、故事等方式展示自己的个性和价值。

### 第二步：展示自己

*   **表达自己**：直接表达自己的想法和感受。
*   **侧面展示**：通过行为、态度等方式间接展示自己的价值。

### 第三步：信息交换

*   **冷读**：通过猜测的方式获取对方信息。
*   **信息交换**：在轻松愉快的氛围中交换信息。

## 举例说明

以下是一个聊天案例，展示了如何应用以上步骤：

**A**：你还没睡呢？
**B**：刚吃完，咱们离得不远。
**A**：可惜咱们不是一块吃的。
**B**：哈哈，火锅啊，宵夜又吃火锅，我的烤鱼。
**A**：嗯呢，我不怎么吃烤鱼，怕心。
**A**：亲亲还能减肥，其实我就是从暗示了，其实我都是从一些敏感话题在暗示了。
**B**：哈哈，我不减肥了，不胖。
**A**：我减肥啊，是吧？
**B**：你又不胖对吧？你胖啊，你怎么不胖啊，多高多重啊，是吧？
**A**：目测多高啊，你1.84米，我离地面1.75米是吧？
**B**：哈哈，你1.75米吗？我说对呀，你呢是吧？你1.72米啊，我是巨婴，是不是？
**A**：好苗条，你说你1.72米，我说巨婴啊，我操是不是啊他哈哈是他很开心的，对吧？
**A**：影响亲亲抱抱举高吗？
**B**：看你力气喽，他没有拒绝，是不是？
**A**：我觉得不影响，对吧？
**B**：我112斤呢，你呢？
**A**：我吸G体，对吧？就是我直接把我的一些经历啊，我的些是每天生活的一些所我想要分享的一些事情直接去。把它去分享出去，那是不是话题呢？

## 总结

本节课中，我们学习了聊天的三大步骤：自来熟阶段、展示自己和信息交换。通过掌握这些技巧，你将能够更好地与他人沟通，建立良好的人际关系。
