# 陈哥《核心私教3.0》：008.2023私教课3.0——重建人设（第三节）_note

在本节课中，我们将要学习如何重建个人“人设”。人设即个人定位，它决定了你在社交中的价值感知。一个清晰、高价值的人设是吸引他人，特别是在两性关系中建立吸引力的基础。我们将探讨人设的重要性、核心构成要素以及如何具体塑造。

## 什么是人设及其重要性？

上一节我们介绍了课程框架，本节中我们来看看人设的核心概念。人设就是你对自己是什么样的人，以及想成为什么样的人的定位。

**人设 = 个人定位与价值呈现**

它之所以重要，是因为人与人之间的社交本质上是一种“销售”——你在向他人展示和销售你自己。如果你不清楚自己的产品（即你自己）定位、目标客户（即你希望吸引的人）需求以及核心竞争力，你就无法成功“销售”自己。

我们可以用一个比喻来理解：你就像一套房子。三四线城市的房价可能是每平方三四千元，而广州核心区的房价可能是每平方二十万元。这个价值差异一目了然。如果你把自己定位成“三四线城市的房子”，那么你的感知价值就很低。人们总是向往更高价值的事物。因此，一个低价值、很“low”的人设，会让对方更向往比你更好的人。

反观女性，她们通常有自己的人设。即使她们没有刻意描述或做什么，你也会对她产生“这个女孩感觉不错”的直观印象。这就是人设的力量。很多兄弟的问题在于，第一印象（人设）就树立错了，导致后续无论怎么聊天都难以挽回，因为基础定位已经错误，甚至会导致“人设崩塌”。

## 如何重塑人设？

理解了人设的重要性后，我们来看看如何具体重塑它。重塑人设的核心是**定位自己想成为什么样的人**。

高价值并不完全等同于物质经济条件，真正的无价之宝在于**思想和认知**。就像企业家稻盛和夫的观点：选择伴侣时，应看重对方的阅历、认知、习惯和眼界，这些需要时间和青春沉淀的特质才是真正有价值的。

在开始塑造前，你必须先想清楚一个关键问题：你的择偶标准是什么？一个重要的原则是：**愿意跟你在一起、喜欢你、想跟你结婚的人，就是你最佳的择偶标准**。很多人总在关注“我想要什么”，却忽略了“谁想给我”。这导致一次次错过了真正合适的人。

以下是重塑人设的三个核心步骤，每个步骤都旨在让你变得“特别”。

### 1. 培养独特的个性

首先，你需要有自己的个性。如果你对自己的人设感到迷茫，通常是因为缺乏独特的个性。

**个性 = 你区别于他人的独特性格与行为模式**

例如，陈哥本人的性格比较开朗。但曾经他也非常内向。他意识到这种性格不利于达成目标，所以主动寻求改变。性格是可以改变的，方法包括：改变圈子、更换身边的朋友，以及最重要的——**内心真正想要突破**。

举个例子，在集体活动中，很多人因为内向或社恐而不敢表达、不敢成为组织者。要突破这一点，你必须主动挑战自己。你可以从与陌生人进行简短交流开始锻炼。**突破性格障碍不是被迫的，而是你主动的选择**。当你多次突破后，这种行为就会成为习惯，进而成为你独特个性的一部分。一个敢于表达、能够主导场合的人，自然会成为受欢迎的焦点。

### 2. 践行“特殊化”原则

其次，人设的关键在于“特别”和“特殊化”。女性之所以会被某个男性吸引，往往是因为她觉得这个男人和她身边的所有其他男人都不一样。

**特殊化 = 在目标对象面前，刻意与她的其他追求者形成差异化**

所以，你的行为和想法要从现在开始独立，**做特别的自己**。具体方法是：**做你从不敢做的事，做别人不敢做的事**。一个有挑战性和冒险精神的男人会更受欢迎。

如何差异化？举个例子：她身边的男人可能总是秒回消息，那你就可以成为那个“不一定总是及时回复”的特别的人。陈哥经常不回消息，但这反而塑造了一种稀缺感和特殊性。关键在于，**你要敢于做那些你以前不敢做、但她身边其他“舔狗”从来不敢对她做的事**。

### 3. 树立“事业型男人”导向

第三，专注于成为“事业型男人”。数据显示，女性普遍喜欢有上进心、对事业或所做的事情专注执着的男性。

**事业型男人吸引力 = 生存保障 + 上进标本 + 价值体现**

这种专注不仅指事业成就，也包括专注于提升自己、保持学习习惯。即使你只是一个上班族，但如果你有持续学习和自我提升的良好习惯，这本身就是一种高价值体现。一个不断成长的男人，会让人感到可靠并充满潜力。

这种特质如何体现？例如，当被问及“你身边女孩是不是很多”时，可以回答：“**我的追求是我的事业，而不是女人。**” 这句话即使听起来有点“假”，但它传递出的专注和优先级，能立刻让你显得与众不同。关键是，你要发自内心地认同并朝这个方向努力。

**一个重要的心态是：目光看向远方。** 当你的眼里全是远方（事业、成长、更广阔的世界）时，女性的眼里才会看向你。反之，当你满眼都是她时，她的眼里却只会看向远方（寻找更好的人）。这是人性。

## 人设的日常维护与外在呈现

定位好人设后，你需要从内到外与之靠拢。以下是需要培养的具体习惯：

### 外在形象管理

第一印象至关重要。很多兄弟在网上聊得很好，一见面就“见光死”，原因在于外在形象与人设不符。

以下是需要注意的要点列表：
*   **眼神**：与人交流时眼神要坚定。
*   **姿态**：坐姿不要显得小气或紧张，站姿不要弓腰驼背，下颌可微抬。
*   **走路**：步伐稳健，不要外八字。
*   **卫生**：保持身体、口腔清洁，衣服整洁无褶皱异味。
*   **仪表**：修剪鼻毛、眉毛，保持发型清爽。

### 表达与肢体语言

你的表达方式也至关重要。有肢体语言的表达会更生动、更有感染力。

**表达感染力 = 语言内容 + 肢体语言 + 情绪状态**

说话时，配合自然的肢体动作来描述。同时，可以运用“点头”等心理技巧，你点头时，对方也会不自觉地认同。避免说话含糊、缺乏底气。

### 能量场塑造

始终保持向上的能量场。避免抱怨和负能量。即使遇到挫折，也要用高能量的方式表达。

例如：
*   **低能量表达**：“最近特别烦，业绩下滑。”
*   **高能量转化**：“最近业绩有挑战，正在加班努力突破。”

能量高的人永远更受欢迎，因为人们都渴望从你身上汲取积极能量。

## 总结与课后任务

本节课我们一起学习了如何重建高价值人设。核心要点是：**人设即价值定位，关键在于“差异化”和“特殊化”**。你可以选择成为“事业型男人”，或者至少确保你在心仪对象眼中，与她身边的所有其他追求者都不同。

你的课后任务是：
1.  **明确方向**：二选一，确定你是要塑造“事业型”人设，还是践行“差异化”原则。
2.  **收集反馈**：询问10个朋友或熟人，了解他们对你第一印象的描述，用这些“数据”来校准自我定位。
3.  **即刻行动**：从外在形象、表达方式和能量状态三个方面开始调整，让你的一言一行都向你向往的人设靠拢。

记住，学习是为了遇见更好的自己，而不是仅仅为了得到某个人。当你成为高价值的自己时，一切都会随之而来。
