# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：
```python
message = "你好，世界！"
```
这里，`message` 是变量名，`"你好，世界！"` 是存储的值。之后，我们可以通过变量名 `message` 来使用这个值。

## 基本数据类型 🔢

理解了如何创建变量后，我们需要知道变量中可以存放哪些不同类型的数据。Python有一些内置的基本数据类型，最常见的有以下几种：

以下是三种最常用的数据类型：

1.  **整数（int）**：代表没有小数部分的数字，如 `10`, `-5`, `0`。
2.  **浮点数（float）**：代表有小数部分的数字，如 `3.14`, `2.0`, `-0.5`。
3.  **字符串（str）**：代表文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `‘Python’`, `“学习编程”`。

## 如何查看和转换类型？ 🔄

我们如何知道一个变量里存放的是什么类型的数据呢？又如何在不同的类型之间转换呢？

Python提供了 `type()` 函数来查看变量的数据类型。同时，我们可以使用 `int()`, `float()`, `str()` 等函数进行类型转换。

请看以下示例：
```python
age = 25
print(type(age))  # 输出：<class 'int'>

price = 19.99
print(type(price)) # 输出：<class 'float'>

# 类型转换
number_str = "100"
number_int = int(number_str)  # 将字符串“100”转换为整数100
print(number_int + 5)  # 输出：105
```
请注意，并非所有转换都能成功，例如尝试将 `“hello”` 转换成整数就会导致错误。

## 变量的命名规则 📝

为了写出清晰易懂的代码，我们需要遵守变量的命名规则。

以下是命名变量时需要遵循的主要规则：

*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名是区分大小写的（`age` 和 `Age` 是两个不同的变量）。
*   应避免使用Python的保留关键字（如 `if`, `for`, `while`）作为变量名。

一个好的变量名应该具有描述性，例如使用 `user_name` 而不是 `a`。
