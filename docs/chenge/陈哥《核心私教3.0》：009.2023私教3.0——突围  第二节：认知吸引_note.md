# 陈哥《核心私教3.0》：009.2023私教3.0——突围  第二节：认知吸引

## 概述
在本节课中，我们将学习如何通过认知吸引来提升自己在追求异性时的成功率。

## 认知吸引的核心概念
认知吸引是指通过一系列的行为和沟通方式，让对方对你产生兴趣和好感。

### 错误的心理和行为
以下是几种常见的错误心理和行为：

1. **过早暴露目的性**：过早表白或透露自己的意图，让对方失去探索和了解的兴趣。
2. **主动示好讨好**：过度主动，失去自我，没有自信和主见。
3. **行为过于低级**：缺乏情商，不理解对方的感受，强迫对方做不愿意做的事情。
4. **缺乏主见，取悦对方**：为了取悦对方而放弃自己的观点和感受。
5. **毫无幽默感，过于正经**：缺乏幽默感，过于死板，限制了自己的表达。
6. **表现软弱**：在关系中表现出软弱，缺乏冒险精神。

### 正确的做法
1. **保持神秘感**：不要过早暴露自己的意图，让对方对你产生好奇。
2. **自信和主见**：相信自己的价值，保持自信和主见。
3. **幽默感**：学会幽默，让对方感到轻松愉快。
4. **冒险精神**：敢于挑战自己，敢于冒险。

## 作业
1. 挑战内心的恐惧，敢于去做自己不敢做的事情。
2. 培养冒险精神，敢于冒险。

## 总结
本节课中，我们学习了认知吸引的核心概念和正确做法。通过避免错误的心理和行为，培养自信、幽默感和冒险精神，我们可以提升自己在追求异性时的成功率。
