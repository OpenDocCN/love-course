# 恋爱心理学课程 P1：识别隐秘的“废物测试” 🕵️♂️

在本节课中，我们将要学习如何识别一种非常隐秘的“废物测试”。很多看似明显的兴趣信号，实际上可能是对方刻意为之的测试。我们将深入探讨真伪兴趣信号的区别，并教你如何通过下意识行为做出准确判断。

---

上一段我们提到了真伪兴趣信号的概念，本节中我们来看看什么是假性兴趣信号。

如果有一个女生做的一些行为，傻子都能感觉到这个女生很主动。比如说这个女生她主动的触碰你，比如说她主动的去跟你抛媚眼，主动的在微信里面跟你聊一些暧昧的信息，主动的配合你，这些往往是假性情绪。

女人但凡是刻意的行为，那么大部分都是假的，就是女人她刻意做出来的。不管女人还是男人，都是有演技存在的。你不要看她这个人表面上做了什么，说了什么，而是你需要从一些女人的下意识行为中，去判断这个女人是否对你有兴趣。

---

那么这些下意识行为就叫做隐性的兴趣信号。它们的目的是测试你。

真正的兴趣信号，它是能体现出这个女人对你的兴趣，而不是非常的直白，而不是傻子都能看出来这个女生对你有兴趣。

以下是常见的假性兴趣信号示例，它们也被称为“废物测试”：

*   夸你
*   主动的触碰你
*   聊天中假装很暧昧
*   假装暧昧的话题

这些大部分都是假性的兴趣信号。如果你傻傻的扑上去了，肯定你们两个就没有戏了。

---

本节课中我们一起学习了如何区分真伪兴趣信号。核心在于识别对方行为是**刻意表演**还是**下意识流露**。面对那些过于直白、傻子都能看出的“兴趣信号”，尤其是符合上述列表的行为时，需要保持警惕，这很可能是一次“废物测试”。正确的应对方式是保持淡定，不要立刻“扑上去”，从而通过测试，展现真正的吸引力。
