人们常说“只羡鸳鸯不羡仙”，爱情是生活中不可或缺的部分。每个人都渴望两情相悦的感情，但并非每段感情都能如愿。一段感情关系中最伤人的是“视而不见”：一方持续无谓地付出，另一方却视而不见。有这种表现的关系，通常被称为“备胎”关系。

上一节我们介绍了“备胎”关系的基本特征，本节中我们来看看具体有哪三种特质的男生容易成为备胎。我总结出来大致有三种。

以下是第一种特质：不主动、只想等待的男生。

接下来，我们探讨第二种特质：低自尊的男生。

低自尊男生非常容易成为备胎，因为他们不够尊重和爱惜自己。男生作为备胎的作用，往往是“提款机”、“垃圾桶”、“司机”、“搬运工”——承担一切女生需要的工作和帮助。女生想要什么东西，只要稍微表达一下，这类男生马上就会满足。相反，女生从来不听他的烦恼，不尊重他的感受。女生会直接忽略他的需求，甚至可以一个月不联系他。而当他忍不住不断找女生时，对方却像没事人一样，还照收礼物，但从不回赠。一个爱自己、尊重自己的男生肯定无法接受这种状况，一旦发生，一定会选择远离这个女生，不会容忍。然而，低自尊的男生总会欺骗自己，认为“精诚所至，金石为开”，自己的真心付出早晚会感动女生，让她看到自己更好的一面。即使人格被践踏，他们也可以接受。
