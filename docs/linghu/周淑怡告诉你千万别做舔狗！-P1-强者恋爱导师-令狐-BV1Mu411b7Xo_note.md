**核心概念：何为“舔狗”**

“舔狗”是一个网络流行语，用来形容在感情中**单方面、过度地讨好对方，即使得不到回应或尊重，也依然放弃尊严、失去自我地去迎合对方**的人。其行为逻辑可以简化为一个悲剧性的循环：

```plaintext
无条件付出 -> 期望获得好感 -> 被忽视或轻视 -> 更加努力付出 -> 自我价值感持续降低
```
