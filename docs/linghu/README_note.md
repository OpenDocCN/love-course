# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

变量可以理解为一个存储信息的“盒子”。在Python中，你可以给这个盒子起一个名字，然后把数据放进去。之后，通过这个名字，你就可以随时使用盒子里的数据了。

创建一个变量非常简单，只需要使用等号 `=` 进行赋值即可。等号左边是变量名，右边是你想存储的值。

```python
my_name = "令狐"
my_age = 25
```

上一节我们介绍了变量的比喻，本节中我们来看看如何具体地使用它。

以下是创建和使用变量的基本步骤：
1.  **命名变量**：选择一个描述性的名字，例如 `user_name`、`item_count`。
2.  **赋值**：使用 `=` 将数据与变量名关联起来。
3.  **使用变量**：在代码中直接使用变量名来代表其存储的值。

## 核心的数据类型 🔢

变量可以存放不同类型的数据。Python有一些内置的基本数据类型，最常用的有以下几种。

以下是几种核心的数据类型及其说明：

*   **整数 (int)**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
    *   **公式/代码**：`my_score = 100`
*   **浮点数 (float)**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
    *   **公式/代码**：`pi = 3.14159`
*   **字符串 (str)**：表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   **公式/代码**：`greeting = “你好，世界！”`
*   **布尔值 (bool)**：只有两个值，`True`（真）或 `False`（假），常用于逻辑判断。
    *   **公式/代码**：`is_learning = True`

## 查看与类型转换 🔄

有时你需要知道一个变量里存的是什么类型的数据，或者在不同类型之间进行转换。

你可以使用 `type()` 函数来查看任何变量的数据类型。

```python
print(type(my_name))  # 输出：<class ‘str‘>
print(type(my_age))   # 输出：<class ‘int‘>
```

数据类型之间可以相互转换。例如，将整数转换为字符串，或者将数字字符串转换为整数。

以下是常见的类型转换函数：
1.  `int()`：将数据转换为整数。
2.  `float()`：将数据转换为浮点数。
3.  `str()`：将数据转换为字符串。

```python
number_str = “123”
number_int = int(number_str) # 字符串“123”被转换为整数123

score = 95
score_str = “我的分数是：” + str(score) # 整数95被转换为字符串，才能与其他字符串连接
```

本节课中我们一起学习了Python编程的基石：变量与数据类型。

最后，我们还学会了如何使用 `type()` 函数查看类型，以及如何使用 `int()`, `float()`, `str()` 函数在类型之间进行转换。
