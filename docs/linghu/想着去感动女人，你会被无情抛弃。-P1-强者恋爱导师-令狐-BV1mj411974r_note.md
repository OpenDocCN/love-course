## 核心误区：试图“感动”或“绑架”对方

上一节我们介绍了本节课的主题，本节中我们来看看具体的行为模式。许多人在恋爱过程中存在一个根本性的误解。

他们认为，只要对女生好、为女生花钱，就能维系关系。这是一种试图用付出“感动”对方，甚至无形中“绑架”对方的行为。从行为本身来看，这是一种非常低效且愚蠢的策略。

**关系 = 资源供给**

让我们进行一个简单的换位思考。如果一个陌生人无条件地对你好，为你付出一切，久而久之，你很可能会觉得这一切都**理所应当**。

女性的心理机制也是如此。持续的单方面付出，只会让接受方习惯并麻木，而不会产生爱意或珍惜。

其核心公式应转变为：

**稳定关系 = 持续的自我价值提升**
