## 什么是尊尼？🤔

上一节我们介绍了本课程的主题，本节中我们来看看“尊尼”的具体定义。

“尊尼”代表一个抽象的计算单元。它的核心功能是对输入进行处理并产生输出。这个过程可以用一个基础的**公式**来描述：

**输出 = 函数(输入)**

这里的“函数”定义了“尊尼”内部的处理逻辑。

---

## 尊尼的核心特性 🔑

了解了定义后，我们进一步探讨“尊尼”所具有的几个关键特性。这些特性决定了它的行为模式。

以下是“尊尼”的主要特性列表：

*   **可配置性**：尊尼的内部参数可以调整，从而改变其处理输入的方式。
*   **状态性**：部分尊尼具有内部状态，其输出不仅取决于当前输入，也受历史状态影响。
*   **连接性**：多个尊尼可以相互连接，形成更复杂的网络结构以处理复杂任务。

## 如何操作一个尊尼？🛠️

理论需要结合实践。上一节我们介绍了尊尼的特性，本节中我们来看看如何实际操作一个基础的尊尼。

操作尊尼通常遵循一个清晰的流程。以下是基本的操作步骤：

1.  **初始化**：创建尊尼实例，并为其设置初始参数或状态。
2.  **提供输入**：将需要处理的数据传递给尊尼。
3.  **执行计算**：触发尊尼的内部处理逻辑。
4.  **获取输出**：从尊尼处接收处理后的结果。

这个过程可以用一段简单的伪**代码**来演示：

```
// 步骤1: 初始化
jonny = new Jonny(initial_parameters)

// 步骤2: 提供输入
input_data = [1, 2, 3]

// 步骤3 & 4: 执行计算并获取输出
output_result = jonny.process(input_data)

print(output_result)
```
