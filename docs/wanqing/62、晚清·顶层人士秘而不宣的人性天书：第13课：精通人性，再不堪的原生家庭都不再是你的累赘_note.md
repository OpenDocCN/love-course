在本节课中，我们将学习如何运用对人性的深刻理解，来扭转和改善原生家庭关系。无论你的原生家庭状况如何，掌握这些核心方法，都能帮助你将其从人生的负担转变为支持的力量。

## 课程概述

前面六课，我们列举了原生家庭的各种现象并探究了其成因。在最后一课，我们将给出终极的解决方法。学习的目的是解决问题，过上幸福的生活。本节课将详细分享如何运用人性智慧，彻底扭转原生家庭关系。

## 扭转过程：三个阶段

我对待原生家庭的心路历程分为三个阶段。第一阶段是未成年时的尝试，以失败告终。第二阶段是放弃与远离。第三阶段是运用人性智慧重新洗牌，并最终大获全胜。本节将重点分享第三阶段的具体实践。

## 核心策略与实践方法

以下是扭转原生家庭关系的核心策略，每一步都基于对人性的洞察。

### 1. 处理财务问题：建立规则而非施舍

首先需要处理的是父母的债务问题。我的先生曾建议我一次性还清所有债务，以维护“面子”。但我认为，每个成年人都应为自己负责，子女无需为父母的行为感到丢脸。更重要的是，轻易解决问题可能滋生更大的问题。

**我的做法是建立一套规则系统：**
```plaintext
IF 父母妥善规划每月零花钱并用于还债
THEN 增加节日红包作为奖励
ELSE (如果乱花钱)
THEN 下月零花钱暂停
```
大约两三年后，他们不仅还清了债务，还积累了储蓄。他们亲身经历了“有钱”和“没钱”的两种生活状态，从内心拒绝再回到“烂泥”中。

### 2. 提供尊重感：满足被看见、被认可的需求

一个过得再浑浑噩噩的人，骨子里都希望被尊重。当我被评为当地文化名人后，政府送去的感谢信和礼物，让我父母感受到了前所未有的尊重。那封信，父亲反复看了上百遍。

此外，我会偶尔带他们接触高端场合或成功人士，让他们直观感受到差距。涉及到我的面子时，他们会努力表现得体，因为他们希望我的成就与他们“有关”。**尊重，是驱动一个人向上向善的强大内在动力。**

### 3. 进行利益捆绑：构建家族共同体

原生家庭不仅是父母，还包括整个家族。人性决定了，每个人最关心的是自身的核心利益。因此，扭转关系的关键在于，**让你的成功与家族每个人的利益深度绑定**。

我在家族中清晰地传达：我的成功对每个人都有利——无论是子孙三代的兴旺，还是晚年的保障。当大家看到我确实在为下一代的成长做实事，并且我的成功能惠及整个家族时，嫉妒就变成了支持。整个家族凝聚成一个互助的整体。

## 通用方法论总结

无论你的原生家庭具体状况如何，以下三个核心方法都普遍适用。

### 1. 核心基础：强大自身能量

改变的前提是自身足够强大。**经济能力在任何时候都至关重要，它代表了你的话语权和分量。** 请永远记住四个字：**人微言轻**。如果你的能力和能量没有发生质变，想去改变原生家庭胜算不大。建议先专注于自身成长。

### 2. 关键心法：保持清醒，拎得清

改变原生家庭需要一个清醒的认知和长远的眼光。例如，对于赌博欠债的亲人，一边帮他还钱一边指望他浪子回头，这种做法只会让他更“混蛋”。**精通人性，意味着知道何时该插手，以何种方式插手，以及何时该彻底放手。**

### 3. 终极突破口：抓住父母的共同关切

几乎所有父母都有一个共同的软肋：**自身的养老与晚年生活质量**。从这个核心关切点入手，结合你的心理博弈能力，基本上就能掌握主动权。许多子女被父母拿捏，往往是怕背上“不孝”的骂名或被以死相挟。**如果你不能反制，就只会被一直拖向深渊。**

本节课中，我们一起学习了如何通过精通人性来扭转原生家庭关系。关键在于：**强大自我、建立规则、给予尊重、绑定利益**。只要你内心强大、意志坚定、精通人性并善于分析，你就能因势利导，不仅让原生家庭不再成为累赘，反而能使其成为你人生的助力。
