# 课程62：晚清·顶层人士秘而不宣的人性天书 🧠：第12课 - 原生家庭与终身成就的关系

在本节课中，我们将探讨原生家庭的好坏与个人终身成就之间的深层关系。我们将分析原生家庭影响的本质，并学习如何不被其束缚，反而将其转化为成长的动力。课程将提供具体的方法，帮助你掌握主动权，塑造自己的人生。

---

原生家庭是个人成长的起点，但它并非人生的终点。近年来，“原生家庭”成为热门话题。如果研究原生家庭是为了找到原因并积极改变，那么它具有积极意义。

但很多人提及原生家庭，并非为了解决问题，而是将其当作一个被广泛认可的借口。例如，将工作不顺、缺乏安全感归咎于父母关爱不足；将不自律、不愿学习归咎于父母未树立好榜样；将婚姻不幸归咎于父母婚姻不幸福的模板。总之，一切不如意都归因于原生家庭。

这样想或许能暂时获得心理安慰，原谅自己的不努力。但这对于改变现状毫无帮助，反而可能让人心安理得地停滞不前。

我完全承认原生家庭对个人的影响巨大。即便是我，一个在众人眼中已彻底摆脱原生家庭影响、实现逆袭的人，也必须坦白，原生家庭的影响依然无处不在，可能伴随一生。

然而，这是否意味着原生家庭的影响注定无解？当然不是。否则，我们的努力与学习将失去意义。如果把人生比作一次航海，原生家庭是我们的**起点**，但最终去向何方，并非由起点决定，而是由航行中的**方向**决定。这个方向由你自己的目标、追求和行动决定。

**公式：人生轨迹 ≠ 起点（原生家庭）。人生轨迹 = f(起点， 个人目标， 持续行动)**

起点最多影响你到达终点的速度，但无法决定终点。因此，原生家庭再差，也主要决定你未成年时期的生活。成年后的生活，原生家庭的影响可能只占**20%**左右，剩下的**80%**由你自己决定。这个比例源于：一个人从七八岁开始形成自我意识，十岁已初具主见，二十岁完全成年。此后的人生，主动权在你手中。

为什么说这节课也是治愈型的？因为世界不会完全限制一个人的成长。它关上一扇门，往往会打开一扇窗。

前段时间我参加了一个创业者聚会。当被问及“为何选择创业”时，大家的回答引人深思。

以下是几位创业者的故事：

*   一位女性说，因父母重男轻女，不断压榨她，甚至想用她换彩礼补贴儿子。她被迫离家，不知不觉走上了创业道路，虽然辛苦，但获得了重生。
*   一位男性说，家里太穷，从小渴望富裕，长大后毫不犹豫选择创业，因为不想重复父母的贫困生活。
*   另一位说，父母离异，无人管束，他早早进入社会，积累了丰富经验，最终选择创业，希望为自己的孩子创造更好的环境。
*   还有一位说，因原生家庭不好而受歧视，创业是为了干出一番事业，赢得尊重，没有退路反而闯出了一条路。

所有人中，只有一位朋友的原生家庭幸福正常。大家感慨之余，有人开玩笑说，或许要感谢糟糕的原生家庭，才激发了斗志，这算是老天的补偿。

我们当然不必感谢糟糕的原生家庭或伤害我们的人。但不得不承认，**磨难常常是一个人成长最好的养分**。我曾心平气和地梳理前半生。我先生常说，我有现在的一切，要感谢我的原生家庭，感谢我父母当年的“不靠谱”。

我不会感谢他们的不靠谱，但今天我想客观地分析。我曾设想，如果我生长在父母恩爱、家境殷实的家庭，我可能会在呵护下快乐长大，按部就班地结婚生子，平淡度过一生。我可能不会有强烈的成功欲和学习欲，因为无需改变什么。那样，我不会有坎坷的经历，同样也不会有绚烂的人生。

正因为原生家庭不如人意，而我又不想认命，所以每天都想着摆脱它，摆脱父母那样的人生。这个目标成了我的原动力，促使我努力学习，改变现状。也因为经历过贫穷困顿，更渴望成功。有这样的想法，怎么可能不努力？因此，我才能做到多年的自律。

从这一方面看，**确实是原生家庭促使我成功**。我也因此很早就与原生家庭和解。我明白，所有成功的人都不会一帆风顺，大多遭受过非凡的打击。

> “天将降大任于斯人也，必先苦其心志，劳其筋骨，饿其体肤，空乏其身，行拂乱其所为，所以动心忍性，曾益其所不能。” ——《孟子》

这句话曾是我的座右铭，在此分享给大家。

所以，从现在开始，我们要停止抱怨原生家庭，停止把一切推给它。因为抱怨无用。伴侣不好可以更换，但父母无法选择。因此，我们不要再做无用功。当你从抱怨中解脱出来，其实已经重生了。剩下的，就取决于你的努力程度。

接下来，我们分享具体方法，探讨如何在遇到不好的原生家庭时，不但不被拖累，还能借此成功。

以下是三条核心行动指南：

**第一条：心肠要硬**
如果你能做到这一点，无论原生家庭如何，都很难对你造成巨大影响。很多人一生被原生家庭拖累，正是因为心太软。父母一有经济困难或琐事相求，哪怕自己也不宽裕，仍会应承下来，最终导致大家一起陷入困境。

以我为例，刚毕业时，家里经济状况很差。虽然我工资不高，但有些积蓄。母亲常向我哭穷，暗示我给钱，但我从未给过。当时她说的话很难听，但我通通置之不理，选择住到公司，减少回家。我用工资买书、请老师学习，乐在其中。我想得很明白：以我当时的收入，要填满家里的经济窟窿，需要不吃不喝15年。届时我已青春不再。更重要的是，一旦我开始填窟窿，父母可能更加挥霍无度。结局将是我牺牲一生，而家庭问题依旧。这不是很多被拖累者的写照吗？

我当时想得很清楚：一人一命，每个人的现状都是自己造成的。不必心软，因为无人会心疼我们。当然，后来我创业成功，还是解决了家里的问题，但我并非盲目填补，而是运用了人性智慧（具体方法下节课分享）。几年过去，父母未再制造新问题，反而变得得体，家族也欣欣向荣。所以，精通人性，真的可以改变个人乃至家族的命运。

**第二条：终身学习**
原生家庭的印记，尤其是不良的三观，唯有通过学习才能摆脱。无论是曾受的伤害，还是被深深影响的世界观，学习都能带来改变。很多人认为我三观正，身上没有原生家庭的负面印记，反而走向相反方向。我的三观从何而来？正是多年学习所得。

学习让我懂得如何正确对待原生家庭，明辨是非，成为更好的自己。这一切，原生家庭给不了，但学习可以。观察周围，无论其原生家庭如何，只要一个人热爱学习，最终的生活通常不会太差。反之，不爱学习的人，大多只是在重复原生家庭的模式。

**代码比喻：`if (持续学习 == true) { 人生轨迹 = 优化(原生家庭印记); }`**

**第三条：坚定信念**
你要坚信，你所遭遇的一切困难，都是为了让你更成功。信念至关重要，你有什么样的信念，就会有什么样的人生。这些年做情感咨询，我遇到许多原生家庭和再生家庭都不幸的女性。她们有一个共同点：认为原生家庭不好就等于折断了翅膀，这辈子有人要就不错了。

我很想唤醒她们：谁说阳光总在风雨后？从人性上讲，命运也欺软怕硬。如果你早早认输，命运会不断欺负你，让你身边的人鄙视你、伤害你，夺走你仅有的一切。但如果你强悍不屈，命运也会畏惧你，并加倍补偿你——财富、亲情、爱情、友情、事业，都会以更好的形式回归。

所以，原生家庭与终身成就的关系，最终取决于你如何选择。

---

本节课中，我们一起学习了原生家庭与个人成就的复杂关系。我们认识到，原生家庭是起点而非终点，其影响虽在，但主动权始终在自己手中。关键在于停止抱怨，并采取行动：心肠要硬以设立边界；终身学习以重塑自我；坚定信念以转化逆境。记住，你的选择，最终决定了原生家庭是绊脚石还是垫脚石。
