## 概述

很多人在恋爱中，常采用两种策略：一是拼命展示自己的优点，希望对方更爱自己；二是努力对对方好，希望获得同等回报。然而，这些做法往往事与愿违，甚至导致受伤。根本原因在于不懂人心。

我们会看到两种截然不同的恋爱现象。有的姑娘条件很好，甚至是被男方追求许久才在一起，但不久便会从主导地位滑向从属地位，逐渐失去吸引力，最终被分手。

但有的姑娘，起初可能并无明显优势，却能很快让男人服服帖帖，心甘情愿为她付出。这些女性或许不漂亮、家世一般，但都是深谙人性的高手。人性高手在恋爱中所向披靡。

## 一个真实的故事

接下来，我将分享一个朋友的真实故事。通过这个故事，我们可以悟出许多恋爱中的人性道理。

这个故事发生在十几年前。当时我们有一个愉快的社交圈子，大家经常聚会聊天。其中一位朋友在一次聚会中认识了一位事业成功、外貌不错的男士。这位男士欣赏我朋友生机勃勃的状态，聚会后两人吃过几次饭，感觉不错。

有一次，男士送她回家时，随意提出：“我觉得我们挺谈得来的，要不交往试试？” 我朋友爽快地答应了。于是两人开始尝试交往。

十几天后，恰逢第一个情人节，他们计划外出吃饭看电影。出发前，男士接到一个电话，他略显尴尬地解释是合作伙伴有事。但我朋友感觉来电者并非合作伙伴，而是其他女性。她没说什么，只是让男士先去处理，然后便回来找我们了。

因此，当男士第二天联系她时，她告诉他：“你的世界太拥挤了，我喜欢宽敞一点。我们还是算了吧。” 男士什么都没说，便离开了。

大约十天后，他给我朋友打电话说：“我在你楼下，有话想跟你说，你能下来吗？” 朋友下去了。他正倚着车抽烟，对她说：“上次是我做得不好。其实我和对方早就分手了，但她一直不肯死心。这次她说如果我不出现，就要跳楼。毕竟是条人命，我不敢做得太绝，所以去看了看。现在已经和她说清楚了，以后不会这样了。我们还能继续交往吗？” 朋友不是矫情的人，便同意了。

那么，男士从此就专心致志地和我朋友谈恋爱了吗？当然不是。那种“过尽千帆皆不是，多年等待只为一人”的桥段，只会出现在言情小说里。现实世界要复杂得多。

我在现实生活中属于不会给人压力的人，所以朋友有什么事也喜欢跟我说。她知道他有其他女人，虽然对方声称都是“前任”，但她内心并不相信。然而，她既没吵也没闹。她是如何做的呢？

朋友知道，自己在这个男人心中渐渐变得不同了。但她并没有受宠若惊或做出改变，而是继续“没心没肺”地生活，没事就和我们一起聚会、讲故事。

这时，这位男士有点不爽了，说：“你和我交往，还和别的男生看电影，是不是不应该？”

朋友惊讶地说：“我以为我们两个都是自由的。你可以和其他妹子吃饭，我也可以和追求我的小男生看电影。大家都是高智商、高情商的人。”

男士当然明白她的意思。过了一会儿，他道歉说：“对不起，是我不好，我以后会改的。”

这一次，男士真的有了不小的改变，对这段感情专心多了。至少，我朋友没再发现什么。男士说：“我现在身边就你一个女的了，你是不是也和其他小男生保持距离呢？” 朋友确实保持了距离，没有再单独看过电影。但她经常组局，大家一起玩。

男士很郁闷，但也说不出什么，毕竟她没有单独相处，只是大家一起玩玩，众目睽睽之下也做不了什么。

男士哑口无言，自知理亏，但他并没有松口，反而变得暴躁易怒。有时候朋友相亲回来，他就故意找茬吵架。

后面的发展才是经典。朋友相了一段时间亲，觉得还是之前那个小男生好，知冷知热，条件也不错，关键对方很喜欢她，更愿意以最大的诚意娶她。朋友答应了，然后向这位男士提出了分手。

男士其实很郁闷，但他不想表现出来被看扁，就问朋友：“你真的想好了吗？你是真的喜欢他，还是故意刺激我？”

朋友云淡风轻地说：“我觉得他真的是一个不错的结婚对象。我迟早要结婚的，不如早点把握。” 然后，她就开始和小男生看婚纱、挑日子，把这男人抛到了脑后。

## 恋爱阶段分析与策略

上一节我们听完了整个故事，本节中我们来看看如何将故事中的智慧拆解并应用到恋爱的不同阶段。

### 第一阶段：恋爱初期 🎯

一般姑娘遇到男友疑似有其他异性时，会怎么做？答案是：争宠。她们会拼命讨好取悦这个男人，侵占他的时间，让其他女人无机可乘。

但这种做法有用吗？大概率男人会这样想：“原来只要我有很多女人，她们就会拼命讨好我、取悦我，为我争宠啊。难怪古代的男人都想做皇帝，后宫三千的滋味真不错，老子也想感受一把。” 所以，男人不但不会因为你争宠而放弃其他女人，反而会更享受游走在不同女人之间的感觉。

但我朋友完全反其道而行之。她的策略核心是：**“你有其他女人没关系，我不在乎。我可以好好安排我的人生。”** 所以她选择了追逐自己的梦想——学习服装设计。这一点非常重要。

**核心策略公式：**
```
你的价值提升 > 对他的时间侵占
```
*   此时如果你找男人要手机、包包，他大概率会看不起你。
*   但如果你用他的资源（如钱）去提升自己，他不但不会看轻你，反而会觉得你的上进与他有关，不会用世俗的眼光看你。

其次，她从未想过要侵占男人的时间，而是让自己的时间做主。你来约我，我有空就答应，我有其他安排就拒绝。而不是做一个永远有空、等着男人“垂青”的女人。

**关于“女追男”的看法：**
经常有姑娘问，女人能不能倒追男人？当然能，现在讲究男女平等。但我并不十分赞成女人倒追，除非你能做到收放自如，并且每天都能让男人觉得你比昨天更优秀。如果不是，建议轻易不要女追男。

男人和女人天生不同。**男人喜欢征服，女人喜欢被宠爱，这是天性。** 如果你倒过来追他，就剥夺了他的征服感和乐趣。所以，在恋爱初期，建议你时不时地拒绝一下。当然，不是故意吊胃口，而是不要让他觉得你已被他彻底征服。

更不用担心：“万一我拒绝，他撤退了怎么办？” 除非这个男人对你完全没兴趣，否则他轻易不会撤退。你适当的“逃跑”，只会让他觉得你是个有意思的姑娘，他反而会觉得更有挑战性，更幸福。这种幸福感来源于男人的受虐欲、征服欲和追逐欲。不要剥夺他们的这种幸福感，同时也请你享受女性被追求、被宠爱的幸福。

### 第二阶段：恋爱中期 ⚖️

上一节我们分析了恋爱初期的策略，本节中我们来看看当关系进入中期，对方开始为你改变时，应该如何应对。

在故事中，男人渐渐为我朋友改变了。换做一般女人，可能会觉得“老怀安慰，终于浪子回头金不换，守得云开见月明”，然后加倍对男人好。

这样做有什么后果？后果就是男人很容易反复。他是怕失去你才做了点改变，结果你马不停蹄地就接受了。他从你的反应里明白：“原来你这么在乎我啊！” 然后就可能再次肆无忌惮。

但我朋友并没有因为他的改变而欣喜若狂，反而出人意料地“没有领情”。这就进一步激起了男人的征服欲。这种“你来我往”的博弈，特别容易让一个男人陷进去。

**核心心法：**
```
精通人心的女人会让对方的“沉没成本”加大。
不懂人心的女人都是让自己的“沉没成本”加大。
```
最后，不懂人心的女人会抱着赌桌不愿离场，而精通人心的女人则始终掌握着离场的主动权。

### 第三阶段：关系后期与目标博弈 🎲

上一节我们探讨了如何在对方改变时稳住阵脚，本节我们来看看当双方目标出现根本分歧（如婚姻）时，该如何处理。

在故事中，朋友想结婚，但得知男人没有结婚打算。她没有吵，没有闹，就是干干脆脆地自己去相亲了，相到合适的就果断提出分手。

很多女人是怎么做的呢？刚知道时又哭又闹，然后试图改变男人。在这个过程中，男人什么都没改变，自己却越陷越深，最后可能放低底线，继续和这个男人相爱相杀，直到男人厌烦，自己伤痕累累为止。

最最关键的是我朋友的心态。有的女人也会用诸如“欲擒故纵”的手段，但为什么会失败？就是因为只想赢，连一点点失去的风险都不愿意接受。最后男人没“擒”住，自己倒是被“擒”得死死的。

**真正的“欲擒故纵”玩法：**
```
如果没“擒”住，就真的放手。而不是虚张声势。
```
你是真的洒脱，还是故作潇洒，男人不笨，他能感觉到。我朋友的心态是：“我做好了愿赌服输的准备。” 这种底气，才是所有策略生效的根基。

## 总结与核心心法

本节课我们一起学习了如何把握恋爱不同阶段的人性表现。

我们通过一个真实的故事，拆解了恋爱初期、中期和后期的核心策略：
1.  **初期**：聚焦自我提升，保持独立性，适当拒绝，激发对方的征服欲。
2.  **中期**：在对方为你改变时保持淡定，不要急于“领情”，加大对方的沉没成本。
3.  **后期**：当核心目标不一致时，尊重对方选择，更尊重自己的目标，敢于果断行动并承担后果。

最后，我想强调：每个男人都不一样。任何案例都只是供你参考，如果不动脑筋、照搬照抄，基本都会失败。你要根据你喜欢的男人的性格、弱点，用适合他的人性模式去“收服”他。

**学习的最终目的公式：**
```
学习人性 ≠ 获得固定答案
学习人性 = 掌握动态方法
```
这一点就是我们学习人性的意义：你不会直接得到答案，但是你会学到方法，从而在复杂的情感世界中游刃有余。
