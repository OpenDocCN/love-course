# 课程62：晚清·顶层人士秘而不宣的人性天书 🧠
## 第30课：如何精通人性，让自己远离嫉妒 😈

在本节课中，我们将要学习嫉妒这一人性弱点的深层逻辑，分析其产生的原因，并掌握一套实用的方法，帮助你在人际交往中识别并远离嫉妒，从而保护自己的能量与安全。

嫉妒是一种极具破坏力的情绪，它不仅能扭曲人心，甚至可能引发难以预料的伤害。理解嫉妒的运作机制，是精通人性、实现自我保护的关键一步。

上一节我们介绍了嫉妒的普遍性与危害性，本节中我们来看看一个生动的案例，它揭示了嫉妒如何让人丧失理智。

有一天，一位天使来到农庄一对老夫妇家中，对他们说，由于你们的纯真善良，上帝决定让你们许三个愿望。但有一个条件：不论你们许下什么愿望，你们的邻居都会同时得到双倍的赐福。

老夫妇听了很高兴，便说：“给我一座小山似的稻谷，这样今年我们就不用耕种了。”第二天一早，门前果然堆起一座小山似的稻谷。老先生兴奋极了，但想到谷仓不够放，便准备去镇上买木材扩建。

才走到一半，他就碰到了邻居。一问才知道，邻居也准备去买木材盖谷仓。邻居手舞足蹈地说：“哇！今天我们家门前突然多出了两座小山似的稻谷，我今年明年都不用耕作了！”顿时，老先生一阵妒意涌上心头。

一周后，天使又来了。老夫妻许下第二个愿望：“我们希望上帝能赐给我们一个可爱的宝宝。”十个月后，他们果然生下一个宝宝。正准备告诉亲朋好友，却看到邻居带着红蛋走进来，兴奋地说：“我太太生了，还是一对双胞胎呢！”

老夫妻听了很不是滋味。那天晚上，天使再度到访。老先生愤怒地说：“我要求上帝砍掉我的一条手臂！”天使吓了一跳。老先生接着狠狠地说：“我要让隔壁那个自得意满的家伙，双手尽失，一辈子不能做事！”

老先生跪在地上，却久久不见回应。一抬头，只看见天使泪流满面地说：“你这个要求上帝是不会答应的。愚蠢的人啊，你何必要伤害别人，把痛苦留给自己，同时又伤了神的心呢？”

这个案例表明，当一个人嫉妒心很重的时候，他宁愿伤害自己，也希望毁掉你。嫉妒可以使人面目全非，其破坏力常常超乎想象。

大多数人都会情不自禁去嫉妒别人，只是嫉妒的点或者程度不同。被人嫉妒说明你在某一方面比较优秀。假如你一无所有，便不会有人嫉妒你。

以下是嫉妒产生的三个人性规律：

第一个规律：**逆袭成功比天生富贵，更容易招人嫉妒**。人们可以接受那些原本就比自己高一大截的人取得更大的成功，但不能接受原本和自己差不多的人，却过上了自己难以企及的人生。这会引发强烈的嫉妒，因为这是一种对自身失败的证明。

第二个规律：**嫉妒基本上都发生在你周围或者同一阶层**。乞丐不会去嫉妒富豪，但会嫉妒每天比他多要100块钱的乞丐。这说明嫉妒具有“就近原则”，身边的人往往最危险。

第三个规律：**嫉妒往往发生在两个关系比较好的人身上**。嫉妒基本发生在姐妹、闺蜜、朋友、同事之间。关系越好，越容易产生嫉妒。对于仇人，你产生的是愤怒而非嫉妒。**“近”是嫉妒的必备条件**。

知道了嫉妒产生的条件，接下来我们分享如何远离嫉妒。虽然嫉妒者心胸狭窄，但如果自己注意方法，可以减少90%以上的潜在嫉妒。

以下是四个实用的方法：

第一，**低调为王**。默默享受成功，不要刻意炫耀。比如在社交圈中，只分享读书心得或工作进展，而非物质财富。在生活中保持朴素，可以避免许多不必要的麻烦。**核心原则是：避免“拉仇恨”**。

第二，**不同的人不同对待**。没有固定的做法，一切因人、因场合而异。参加高端聚会时可以衣着得体，但做人要不卑不亢。过于自傲会拉仇恨，过于放低自己则会让人想来踩一脚。需要灵活把握这个度。

第三，**努力让别人心理平衡**。多展现自己付出的努力和吃的苦，让别人看到成功背后的代价。与不如自己的人聊天时，多挖掘对方的优势，提及自己的劣势。例如，对宝妈说羡慕她能陪伴孩子，对自己创业的辛苦表示感慨。关键是**让对方感觉你是真诚的**，而非另一种炫耀。

第四，**构建利益共同体**。这是最有效的一招。将个人的成功与周围人的利益绑定。例如，向家族成员灌输“家族兴旺，人人受益”的观念，强调相互提携的重要性。当大家成为利益共同体时，便会盼望你更好，而非嫉妒你。**公式：共同利益 > 个人得失**。

本节课中我们一起学习了嫉妒的人性逻辑与应对之法。我们分析了嫉妒产生的三个规律，并掌握了低调处世、区别对待、平衡他人心理以及构建利益共同体这四种远离嫉妒的实用策略。精通人性，不仅在于洞察他人，更在于智慧地保护自己。希望你能将这些方法运用自如，在复杂的人际关系中从容前行。
