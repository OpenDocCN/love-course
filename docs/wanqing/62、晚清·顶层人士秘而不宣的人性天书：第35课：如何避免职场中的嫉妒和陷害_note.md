# 课程62：晚清·顶层人士秘而不宣的人性天书 🧠
## 第35课：如何避免职场中的嫉妒和陷害 ⚔️

在本节课中，我们将学习如何识别并有效应对职场环境中频繁出现的嫉妒与陷害行为。我们将通过一个真实的案例，剖析其背后的底层逻辑，并掌握一套可操作的策略。

职场中的嫉妒与陷害，与人际社交中的嫉妒有相似之处，但更为频繁和直接。社交中的嫉妒可能每月遇到一次，而职场中可能遇到十次。社交嫉妒多源于处事方式差异，很少造成实际损失。职场则存在大量直接的利益冲突和工作中的陷害机会，危险性更高。

下面，我将通过一个闺蜜的故事来具体说明。

我的闺蜜比我小两岁，事业心很强。她目标明确，努力与领导搞好关系，工作仅一年就连升三级，成为部门最年轻的主管。

然而，打击很快到来。升职后，她反而被团队孤立。同事聚餐不再叫她，工作中也对她布置的任务阳奉阴违，甚至故意陷害，声称未接到指令。她不敢激烈应对，只能自己承担大量工作，身心俱疲。

为了改善关系，她请同事吃饭、送水果，但收效甚微，反而显得软弱。她向我求助，不知如何让大家接受她。

我指出她的问题在于试图通过讨好来换取接纳。在同事眼中，这反而是软弱可欺的表现。我建议她主动向上司沟通现状，并同时提出解决方案。

她担心领导会觉得她无能或是在打小报告。我告诉她，主动沟通恰恰表明信任和依赖领导。领导提拔了她，她的处境也关乎领导的脸面，因此领导大概率会支持她。关键在于，沟通时要带着解决方案，而非单纯抱怨。

她采纳了建议，与领导沟通后获得了支持。有了底气，她不再讨好，而是严格执行工作分派。当年终考评时，她给闹得最凶的员工打了最差绩效，并依据规定取消了其年终奖和晋升资格。同时，她奖励了唯一态度尚可的员工。

面对质疑，她义正辞严地指出：**个人情绪不能影响工作和公司利益**。领导没有反对她的决定，这本身就是一种支持。此番操作后，团队表面工作态度明显好转。

事后她感叹，对人好反而被拿捏，树立威严后大家却听话了。我告诉她，仅靠强压不行，职场最终是利益为王。她随后将更多利益让渡给同事，积极为大家争取机会，逐步赢得了大多数人的心服。当然，仍有少数人看不惯她，但这已无法形成气候。

她的第一次升职危机就此解决，领导也对她的能力刮目相看。

通过这个案例，我们可以总结出应对职场嫉妒与陷害的四个核心策略。

以下是具体策略分析：

**第一，认清职场嫉妒的特殊性。**
社交中，低调和远离可有效避免嫉妒。但在职场，**升职加薪是显性的**，难以完全低调。做事必须高调，不能让自己成为边缘人。做人可以低调，但做事一定要让成果被看见。

**第二，做好孤独准备，别太在意他人态度。**
当你脱颖而出时，还期望所有人喜欢你是不现实的。要不断提醒自己，**职场的主要目标是实现个人价值与成长**，而非交际。一旦你表现出对他人态度的过度在意，这就成了别人攻击你的弱点。耐受孤独，反而会吸引真正认可你的人。

**第三，善用手中权力，必要时果断反击。**
面对嫉妒和陷害，低姿态讨好往往无效。有时，**直接、有力的回应**反而是最高效的解决方式。就像别人打你一拳，回击三拳让其忌惮，可能比单纯讲道理更有效。你需要展现领导的魄力。

**第四，采取分而治之的策略。**
此策略承接上一条。你不能与所有人为敌。关键在于：
1.  **事前与领导沟通报备**，获取支持，避免被视作擅自行动的危险人物。
2.  **精准打击**，只对少数带头者“杀鸡儆猴”，以达到震慑效果。
3.  **瓦解同盟**。职场同盟往往基于利益，并不牢固。通过**分享利益、给予好处**，很容易将对手转化为盟友。当你能提供更大利益时，对方的嫉妒很容易消散。

本节课中，我们一起学习了职场嫉妒与陷害的底层逻辑和应对策略。核心在于：认清职场利益本质，做好心理建设，善用权力与规则，并运用分化与利益共享的策略来化解危机。每个人的具体处境不同，但只要掌握这些底层逻辑，你就能靠自己分析并解决大部分问题。
