# 课程62：晚清·顶层人士秘而不宣的人性天书 🕵️‍♂️
## 第27课：人际交往中的大坑，99%的人都跌过

在本节课中，我们将探讨人际交往中两个普遍存在却极易被忽视的“大坑”。通过两个生动的案例，我们将分析“朋友变敌人”的潜在风险与“敌人变朋友”的智慧策略，帮助你理解复杂的人性，从而在社交中更好地保护自己并建立更稳固的关系。

上一节我们介绍了人性社交的复杂性，本节中我们来看看两个具体的人性陷阱。

### 案例一：高调炫耀引发的祸端

几年前，一位朋友开着新买的车来送请柬，邀请我们去他的新家吃暖屋酒。到达指定地点后，我和先生都感到惊讶，因为他的房子非常大。据说房子面积有3000平米，还不包括花园。花园的面积可能需要用亩来计算。我平时看楼盘较多，目测花园超过5亩。整个房产只能用“豪横”来形容。朋友几乎邀请了所有认识的人，门前停满了豪车。出于一种人性直觉，我认为这种做法存在很大风险。

我偷偷告诉先生，这样做太高调了。如果是我，我会选择默默搬进去，不会如此大张旗鼓。先生则认为，一个人奋斗多年终于成功，需要向他人展示成果。否则就像穿着华服在夜间行走，无人知晓。他觉得我有时过于谨慎，并提醒我朋友邀请我们是为了见证他的成功，我应该多夸奖他，以免被误认为是嫉妒。我表示自己明白这点情商。

那天，朋友站在别墅中央，显得神采飞扬。然而几个月后，在我先生生日宴上，这位朋友却没有出现。我们多次拨打他的电话，均无人接听。这很不寻常，因为他通常有事会提前通知。直到宴会结束，他依然没有出现，其他人也联系不上他。我们开始担心他是否出了意外。

第二天，他的电话仍然无法接通。我建议先生联系他曾托我们办事的姐姐。姐姐告知我们，朋友被抓起来了，人没事，只是被关押。这对我们而言难以置信，因为我们朋友圈中从未有人出过这种事。半年后，他母亲打电话给我们，才告知事情原委。

在朋友创业初期，他曾做过违法的事，可能是行贿或侵占了不属于自己的钱财。他母亲也说不清楚具体细节，因为成年子女通常不会事无巨细地告诉父母。这件事本已过去多年，无人知晓，但半年前被人匿名举报，最终导致他被判刑数年。他本人也认罪，认为这是自己行差踏错的因果报应，正在服刑争取早日出狱。

两年后的某一天，先生很晚回家，叫醒我并问我是否知道当初匿名举报朋友的人是谁。我瞬间清醒，说出了一个名字。先生很惊讶，以为我消息灵通。我解释道，从他惊讶的表情可以推断，举报者一定是与朋友很熟的人。我回想起朋友刚入狱时，他的一位闺蜜表现得最为关心，多次给我们打电话。虽然她表现得很惋惜，但我总有一种说不出的感觉，因此猜测是她。

先生感慨地说，真没想到是她举报的。要知道，朋友发达后帮了这位闺蜜很多，甚至借给她几百万元，对她比亲姐妹还好。没想到最终却被她举报。目前朋友还在服刑，我们不敢让他知道真相，但担心他出狱后得知此事，不知会作何反应。先生问我，她为什么要这样做？朋友犯法固然不对，但对她是掏心掏肺的好。我回答说，这源于女人之间的嫉妒。原本她和闺蜜条件差不多，但朋友创业成功后，拥有豪车别墅，一下子将她远远甩开。尤其是那次高调宴请，其实让很多人眼红。那天我听到一些不认识的宾客聊天，话里话外都带着酸意。有的人只是心里酸，但有的人心理阴暗，就会采取行动。

朋友因被闺蜜匿名举报而入狱，这让很多人感到意外。但意外之后，大家又觉得这在情理之中。今天，我将从人性的角度分析这件事。当然，前提是犯法行为肯定不对，受到法律制裁是应该的。

朋友对闺蜜毫无防备，连自己违法的秘密都告诉了她，从而埋下了隐患。从她的角度看，闺蜜绝不会出卖自己，所以告诉她没关系，这正说明两人关系亲密。但她没料到，自己可能看人不准，或者人性会变，最终导致最好的年华在牢狱中度过。

通过这个案例，我想告诉大家一个残酷的社交真相：**当朋友变成敌人时，比敌人更可怕**。

因此，你要么确保自身无懈可击，没有任何弱点和把柄；要么你真的拥有精准的识人眼光，从不走眼。否则，即使关系再好，也要保留最后一分警惕。因为你的朋友更容易对你产生嫉妒之心，背叛你的速度更快，也更清楚你的弱点在哪里。

世上能伤害你最深的人，往往不是陌生人或敌人，而是你最亲近的人，可能是亲人，也可能是朋友。而他们一旦做出对你不利的事，通常符合“快、狠、准”的特点。

以下是这三个特点的具体分析：

*   **快**：指当他们对你不满时，这种情绪会快速发酵。你可能还以为彼此感情深厚，对方却已在寻找机会伤害你。
*   **狠**：指你身边的人恨你时，做起事来会比其他人更狠。虽然现代社会很少发生极端事件，但性质是相同的。
*   **准**：指你身边的人非常了解你，清楚你的喜好和弱点。所以他们要么不伤害你，一旦行动，绝对是精准打击，让你难以躲避。

所以，朋友变敌人，绝对比敌人更可怕。敌人想打击你，未必知道你的弱点，其行动可能对你无关痛痒。他尝试十次，可能只有一次击中要害。但朋友不一样，他若想伤害你，十次里可能次次都不会落空。

在这里，同时也想和大家分享一些人性社交的常识。有一次直播时，有粉丝问我：“班金姐，你现在越来越成功了，你身边的人会嫉妒你、伤害你吗？”到目前为止，并没有。我也可以和大家分享一些我的做法，帮助大家避坑。

以下是我在人际交往中采取的几点做法：

1.  **保持警惕，主动远离**：在人性上，我是一个非常警惕的人。如果在逐渐成功的过程中，发现有人对我说酸话、嫉妒心很强，我不会直接冲突，但会默默远离这个人，直到我们不再有交集。我不喜欢挑战人性，更不喜欢拿自己冒险。
2.  **深知嫉妒危害，保持低调**：我深知嫉妒的危害性，因此在日常生活中非常注意。大家可以回忆一下，我平时在朋友圈或社交平台上，是否晒过容易引起他人（尤其是女性）嫉妒的东西，比如房子、车子、包包等。即使我们搬了新家，很多人要求看房子，我也没有晒过任何一张室内具体照片（小区环境除外）。我最多在直播间展示几件新衣服。毕竟创业多年，如果连几件新衣服都买不起，那就显得太装了，反而会让人不爽。
3.  **与更高层次的人交往**：我身边的朋友大多不嫉妒我，主要是因为他们大部分比我漂亮、比我有钱、过得比我轻松。他们甚至觉得我太辛苦，日子过得太累，同情我都来不及，没必要伤害我。
4.  **保持谦逊，真诚为他人高兴**：我是一个喜欢为他人鼓掌的观众。可能因为内心比较丰盈，我不会嫉妒别人，也愿意把自己放低一些。当别人有好事时，我能打心眼里替对方高兴。时间长了，大家知道我的性格——不喜欢和别人比较，只喜欢写写文章、玩玩翡翠，也就不会把我当成敌人。

在我们的日常行为中，要注意自己的言行是否会给自己树敌。我永远相信一句话：**低调可以避免很多麻烦**。

### 案例二：化敌为友的智慧

美国的RealNetworks公司曾向美国联邦法院提起诉讼，指控比尔·盖茨的微软公司违反反垄断法，并要求其赔偿1亿美元。但在官司尚未结束时，RealNetworks公司的首席执行官格拉塞却致电比尔·盖茨，希望得到微软的技术支持，以使自己的音乐文件能够在网络和便携设备上播放。所有人都认为比尔·盖茨一定会拒绝。但出人意料的是，比尔·盖茨对他的提议表示出欢迎。他通过微软的发言人表示，如果对方真的想要整合软件，他将很有兴趣合作。

众所周知，微软和苹果两大公司自80年代起就一直处于敌对状态。乔布斯和比尔·盖茨为争夺个人计算机市场的控制权展开了激烈竞争。到了90年代中期，微软公司明显占据领先优势，占领了约90%的市场份额，而苹果公司则举步维艰。然而，令所有人大跌眼镜的是，1997年，微软公司向苹果投资1.5亿美元，把苹果公司从倒闭的边缘拉了回来。2000年，微软为苹果推出了Office 2001。自此，微软与苹果真正实现双赢，他们的合作伙伴关系进入了一个新时代。

这就是典型的“把敌人变成朋友”的故事。社交中真正聪明的人，不是想着给敌人难堪、把敌人搞死，而是想办法把敌人变成朋友。

首先，当你决定打败敌人时，敌人也想着打败你。他既然能成为你的敌人，就一定跟你实力相当，不好对付。退一步讲，就算你历经艰险终于将他打败，谁能保证某天他就不会东山再起？到时候你又要提起十二分精神积极备战。所以，最好的办法不是打败他，而是像比尔·盖茨那样，友好地站到敌人的身边去，把他变成自己的朋友，实现双赢。

敌人变朋友，通常有两种原因：一是在敌对过程中渐渐欣赏这个对手；二是出现了双方共同的利益点。无论是哪一种原因，这个“前敌人”都不会轻易来伤害你。

以下是敌人变朋友后关系更可靠的几个原因：

1.  **弥补心理与诚意展示**：你们曾经是敌人，想必都做过一些伤害对方的事。如今成了朋友，为了弥补以前的过失，也为了向对方表示自己的诚意，自然会比普通朋友更用心。
2.  **实力相当，明白两败俱伤**：你们能够成为敌人，必定旗鼓相当、势均力敌。否则一方早就把另一方打败了。因此，双方都明白，敌对下去只会两败俱伤。当你想着打败对方时，对方也在想着如何打败你。但如果化敌为友，就能实现强强联合。彼此都有这个认知，便不会轻易撕毁新的朋友关系。
3.  **相互欣赏，胸襟磊落**：如果你们是在敌对过程中相互欣赏，最终化敌为友，说明你们都是有胸襟、有气度的人。有胸襟气度的人，又怎么会在背后捅你刀子呢？所以，虽然是曾经的敌人，但只是当初立场不同，本质上都是磊落的人。
4.  **利益共同体，关系牢固**：如果你们是因为有共同的利益点而从敌人变成了朋友，那么有句话说得好：“这世上没有永恒的朋友和敌人，只有永恒的利益。”既然是利益共同体，又怎么会继续敌对呢？毕竟这么做对谁都没有好处。只要共同利益点不被破坏，谁也不会率先破坏这种关系。有时候，共同的利益比任何关系都牢靠。
5.  **避免反复，珍视名誉**：已经和你做过一次敌人、后来又成为朋友的人，基本都不想和你第二次做敌人了。毕竟反复无常是小人行径。大家都是社会人，也不希望留下这样的名声。

所以，**敌人变朋友会比朋友更可靠**，就是因为这些原因。只可惜很多人在社交中恰恰相反：对朋友无条件信任，哪怕朋友已经表现出不对劲（比如在你成功时不是高兴而是酸溜溜），也装聋作哑，认为对方只是心情不好或碍于多年情分而不愿理智对待；而对“化敌为友”的可能性却始终不相信，总觉得“一日为仇，终生为仇”，有时宁愿信任前一种“朋友”，也不愿思考如何把敌人变成朋友，更不愿相信曾经做过敌人的人。这就是人性中的大坑。

我相信对很多小伙伴而言，“从敌人变成朋友”的经历未必人人都有，但是“被朋友伤害过”的人一定比比皆是。

本节课我们就分享到这里。只要在社交中避免踩入这种人性大坑，其他方面做得不好，了不起就是人缘差一些，不至于如此伤筋动骨。

**总结**：在本节课中，我们一起学习了人际交往中的两个关键陷阱。通过第一个案例，我们明白了“朋友变敌人”可能带来的巨大伤害，根源往往在于嫉妒与毫无防备，因此需要对亲近之人保持一份清醒的警惕。通过第二个案例，我们看到了“敌人变朋友”的智慧与益处，这源于相互欣赏、共同利益以及对“两败俱伤”的理性认知。核心在于，**对待朋友需保留一分警惕，而对待值得的敌人则可尝试化敌为友**。掌握这些人性规律，能帮助我们在复杂的社交世界中更好地自处与前行。
