在本节课中，我们将要学习一个核心原则：永远不要与人性为敌。我们将通过具体案例，深入剖析人们在日常生活中如何不知不觉地违背这一原则，并探讨如何通过顺应他人的人性、对抗自身的人性来获得成长与成功。

## 引言：我们真的没有与人性为敌吗？

上一节我们介绍了人性洞察的重要性，本节中我们来看看“与人性为敌”这一普遍现象。

很多人认为自己不会与人性为敌，但这往往是一种错觉。现实中，许多行为都在不知不觉中违背了人性规律。

以下是几种常见的“与人性为敌”的表现：

*   面对品行不佳的伴侣，幻想对方会改变，而非果断止损。
*   面对不公正对待自己的父母，加倍付出以期获得认可。
*   面对陷入债务的子女，无条件替其偿还，希望其能“幡然醒悟”。

这些行为看似是出于爱或责任，实则是在对抗“人性趋利避害、难以轻易改变”的本质。

## 核心原则：顺应他人，对抗自己

我之所以能准确预判许多事情的走向，并非因为我有超能力，而是因为我**从不与人性为敌**。真正精通人性的人，其行为模式可以概括为一个核心公式：

**成功策略 = 顺应他人的人性 + 对抗自己的人性**

这个公式意味着，在与人交往时，要理解和顺应对方的人性需求；在自我成长时，则要克服自身的惰性和弱点。

## 案例分析：当婚姻遭遇人性考验

为了更好地理解这一原则，我们来看一个真实的咨询案例。

一位女士与丈夫同时应聘一家龙头企业的职位，且只录取一人。在最终面试前，丈夫以“事业对男人更重要”为由，请求妻子放弃机会。

我的建议是：**“好好面试，各凭本事，不要让。”** 因为丈夫提出此要求，已暴露其自私、功利的人性。一个真正为家庭着想的人，不会轻易要求伴侣牺牲重大前途。

然而，这位女士因害怕婚姻破裂，选择了顺从丈夫，放弃了机会。此后，她又听从丈夫建议，辞职在家带孩子。最终，丈夫在事业上升期出轨并提出离婚，理由是妻子“与社会脱节，没有吸引力”，并迅速与一位能助其事业的高管亲属结合。

这个案例的结局，早在丈夫第一次提出不合理要求时就已注定。女士的两次顺从，都是在与真实的人性为敌：第一次是忽视丈夫自私的人性，第二次是忽视“价值对等”是亲密关系的基础这一人性规律。

## 实践指南：如何应用这一原则

上一节我们通过案例看到了违背人性的后果，本节中我们来看看具体应该如何做。

### 对待他人：顺应与洞察

当你发现一个人表现出某种稳定的负面特质（如自私、虚荣），正确的做法是：

1.  **承认现实**：接受这就是他真实的人性，不要幻想他会为你改变。公式表示为：**一次侵害行为发生概率 > 0 → 二次侵害行为发生概率 ≈ 1**
2.  **保持距离**：及时设立边界，避免给予对方第二次伤害你的机会。用代码逻辑表示就是：
    ```python
    if person.shows_harmful_trait == True:
        response = maintain_safe_distance()  # 保持安全距离
        # 而不是：hope_for_change() # 期待改变
    ```
3.  **顺应利用**：如果你想从互动中获益，就去满足对方的人性需求。例如，对方爱占便宜，就设计规则让他持续感到“占了便宜”，从而达成你的商业或社交目的。

### 对待自己：对抗与成长

对待自身，则需要反向操作，对抗与生俱来的惰性。

*   **延迟满足**：当你的本能倾向于轻松、享乐时，选择那条更艰难但能带来长期收益的路。
*   **逆本能而行**：在情感上舍不得离开一段有害关系时（自我人性），理智上就必须执行“断舍离”（对抗人性）。这可以表示为：**行动方向 = -1 × 情感本能方向**

成功本质上是一场“反人性”的修行。别人娱乐时，你需要学习；别人妥协时，你需要坚持。

## 境界自测：你在哪一重？

我们可以将对待人性的态度分为不同境界，用以自我审视。

以下是对待他人人性的三重境界：

*   **第一重：顺应人性**—— 能识别并接受他人的人性，不因此愤怒或对抗。
*   **第二重：利用人性**—— 在顺应基础上，能巧妙引导或利用人性特点达成目标。
*   **第三重：批判人性**—— 居高临下地批判人性弱点，往往导致冲突与失败。

以下是对待自身人性的三重境界：

*   **第一重：对抗人性**—— 能够克服惰性、恐惧等弱点，自律向上。
*   **第二重：摇摆不定**—— 在对抗与屈服之间反复挣扎，内耗严重。
*   **第三重：屈服惰性**—— 完全被本能欲望支配，放弃成长。

## 总结

本节课中我们一起学习了“永远不要与人性为敌”这一核心智慧。

关键要点在于：**在外部，要洞察并顺应他人的人性，不抱幻想；在内部，要持续对抗自身人性的弱点，追求成长。** 商业上的成功者（如拼多多）深谙此道，通过满足大众“占便宜”的人性而崛起。个人层面的成功，也同样始于对自己惰性的每一次克服。

记住，让别人伤害你一次，是对方的问题；允许同一个人伤害你两次，则是你需要反思的智慧缺口。掌握人性，不是为了操控他人，而是为了清醒地行走于世，保护自己，并最终驾驭自己的人生。
