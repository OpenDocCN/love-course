## 第09课：为什么你的父母把面子看得比你重？🎭

在本节课中，我们将深入探讨原生家庭中一个普遍存在的现象：为什么有些父母会将“面子”置于子女的感受和幸福之上。我们将从人性角度剖析其背后的原因，帮助你理解并解开可能的心结。

***

### 概述

本节课旨在分析父母重视“面子”胜过子女的深层原因。我们将通过具体案例和人性剖析，解释这一现象背后的心理与社会因素。理解这些原因，有助于我们更好地处理家庭关系，并做出符合自身幸福的选择。

***

### 个人经历与感悟

我写下这个标题时，许多往事涌上心头。虽然早已与自己和解，但回想起来仍觉心酸。我分享自己的经历，既是对这个问题的感悟，也是我最初对人性产生兴趣的原因。

我从小观察力敏锐，几岁时就能看清许多事，包括大人之间的事。我从未因别人觉得我家穷、看不起我们而愤怒。我认为，好吃懒做、家徒四壁的家庭被人看不起是合理的。

难道这样的家庭还应被高看一眼吗？那将置那些刻苦耐劳的人于何地？但你可以在心里看不起，上门挑衅、阴阳怪气就不应该了。我家偏偏有这样一个邻居。小时候，我一直认为他对我们家特别有敌意。

长大后才发现，他对很多人都有敌意。但那时我小，并不知道。他和我妈的关系是相爱相杀的。他们几乎天天在一起打麻将，不在一起时也常聊天串门。但我特别不喜欢她，小时候常赶她走。

他母亲是最早一代的女强人，很早就在上海做生意，经济条件很好。但她长得很丑，却喜欢上了我们那里最帅的男人。那男人不喜欢她，但架不住她家里条件好，最后还是娶了她。听说她还使了许多不光彩的手段。

类似于撬墙角之类。这些都是我妈和别人聊天时，我听到的。每次她到我家来，就会对我妈说：“哎，你们家当家的也是做生意的，怎么年年做年年赔啊？这日子怎么过得下去？” 我妈就会讪讪地笑。

她会继续说：“你女儿也不小了吧，可以让她干家务了。小丫头片子不好好学家务，以后到了婆家会被看不起的。” 我妈会顺着她的话说：“谁说不是呢？但她爷爷宠她呀，我哪敢叫她干活？不然又去找她爷爷告状去了。她爷爷最疼的就是她了。”

她会冷哼一声说：“不管怎么样，你都是她亲妈，还做不了她的主啦？她要是不听话，你就趁她爷爷不在的时候打她，就不信打不服她。” 她说这些话时，从来不避开我，甚至是故意带着满满恶意。

我是谁？我大学的专业是法律，就是因为我从小口才就相当了得。我妈是绝对不会在外人面前维护我的。她会觉得如果她帮我说话，别人会觉得她护短、不会教女儿。所以我只能靠自己。我绝不会让自己吃亏，立刻对她说：

“你好像也有女儿吧？你让你女儿干家务了吗？饭好像都端到她手上的吧？怎么你女儿不这样教，跑到人家家里来挑拨离间？你心里扭曲啊。” 她气个半死，说：“我女儿比你小两岁呢。” 我说：“那好，我看看你两年后会不会这样教你女儿，会不会让你女儿到婆家去当牛做马。如果你做不到，现在就给我闭嘴。”

她气得发抖，和我妈说：“你女儿才几岁呀？太厉害了吧！你就不管管她，让她这样和大人叫板？这样没家教的女儿以后怎么嫁得出去？” 我妈面子上挂不住了，我却还在怼她。我说：“你以为你有家教呀？你妈没教你别跑到人家家里当家教吗？你一把年纪都还没学会做人，好意思跑到我家里来教我做人？我都替你脸红。”

但最先爆发的不是她，而是我妈。她恶狠狠地用最难听的字眼骂我，叫我闭嘴。其实那时，我对那个人的讨厌远远不如对我妈的讨厌。明明是别人欺负她的女儿，而且是当面欺负。她一句话都不维护我，反而为了所谓的面子，用最难听的话骂我、诅咒我，甚至还想打我。

如果不是我爷爷奶奶姑姑们从小就宠我，我妈不敢过于动我，我小时候的日子可能更加苦不堪言。还有一次就更加过分了。那一次是那个女人的母亲到我家里来，估计也知道她女儿被我反抗的事。

所以看到我就来刺激我，说我小小年纪，牙尖嘴利，长大以后肯定没人要。我当时觉得这母女俩都是从一个模子里出来的，说来说去就是“长大后没人要”这几句话。于是我立刻回怼：

“这个就不用你操心了。你这种心理扭曲的老太太都有人要，你女儿又丑、心里又扭曲，都有人要。看到你俩，我一点都不担心了。再说了，你们的终身目标就是为了有个男人要你们吗？难怪你女儿会使那些下三滥的手段去撬人家墙角。你们不以为耻，反以为荣，还到处说。还有，你们一把年纪了，天天在一个10岁的女孩子面前说这么恶毒的话，以后是要下拔舌地狱的，造的业，实在太重啊。”

那次我妈是真的想打我了，幸亏我跑得快，立刻躲到我爷爷那里去了。那老太太还跑到我爷那里去告状。我爷爷对我的疼爱不是假的。他说：“我这孙女，我清楚。她不是一个主动会挑事的人，肯定是你先惹了她。你一把年纪了去欺负一个小孩子，你怎么好意思的呀？我觉得她做得很对。你都欺负到她头上了，还不准她反抗呀？兔子急了还咬人呢。她这么做都是我教的。我教她，我们不主动欺负别人，但别人欺负上门了，没必要忍着。我孙女在学校是班长，每次考试全是100分，谁不夸她聪明呀？怎么就你们母女看她不顺眼呢？说明这就是你们的问题呀。”

后来我小叔叔到我家来串门的时候，笑着对我和我爷爷说，那老太太差点被我气得脑溢血，还说都是被我爷爷惯得不成样子了。我爷爷说：“她活该。我孙女我都舍不得打、舍不得骂，她是哪棵葱呀？跑到我家里来教训我孙女了，当我死了吗？”

但我妈就不是这种反应了。她恶狠狠地骂了我好多次，说我丢了她的脸，搞得她很没面子。我那时候真的搞不懂。在我的观念里，她和我爸天天打牌，其实别人是打心底里看不起她的，和我有什么关系呢？你再去讨好取悦别人，别人都上门来羞辱了，你还含笑受着，人家只会更加看不起你，还不如直接反击？再说了，任何事我们要有是非对错。明明是别人的错，为什么不能反击？

但是她觉得我的反击说明她根本治不住我，连自己的女儿都搞不定，是特别没面子的事儿。那时候我就知道了，在我妈心里，我的感受、我的委屈，还没有她的面子重要。

所以从那以后，我再有什么委屈，我会和我爷爷姑姑们说，我从来不会去和我妈说，没必要自讨没趣。有一次我妈对我说：“你的心肠确实挺硬的。在我的印象里，你小时候哭过后，我已经多少年没有见过你的眼泪了。你真的不会哭吗？” 我哼了一声，我说：“我的眼泪很值钱的，一般人是看不到的。”

因为我自己有过这些经历，所以我做这些课的时候感受特别深刻。那时候我就在想，为什么在父母眼里面子比自己的孩子还重要呢？其实我很久之前就已经分析研究过了，也知道为什么了。今天就和大家好好地，从人性上来剖析一番吧。

### 核心概念界定

在分享之前，有一点要着重说明。我们这节课里讲的主要是指：**子女的正常行为和父母的预期不合，父母把面子看得比子女重**。而不是子女有杀人放火这种行为，父母觉得丢脸的这种正常反应。那只要是还有羞耻心的父母都会觉得丢脸，不觉得丢脸反而有问题。

***

### 原因剖析：为什么父母会把面子看得比孩子重？

上一节我们界定了讨论的范围，本节中我们来看看背后的具体原因。以下是三个核心原因：

#### 1. 父母内心不强大

第一个原因是父母内心不够强大，无法面对子女的选择超出普通人的常规做法。

**公式：父母的不安全感 ≈ 对“异常”选择的恐惧**

我举一个最常见的例子。比如，有的姐妹婚姻生活特别不幸福，老公家暴赌博，她的日子过得水深火热，唯有离婚才是她的解脱。但是她不敢离，因为父母不同意。如果她敢离婚的话，父母不但不会提供任何援助，还会让她永远不准回娘家。

理由很简单：如果家里出了离婚的女儿，传出去多丢人？以后怎么面对左邻右舍和亲戚朋友的眼光？所以不管怎么样，你都不准离婚。这些苦命的妹子也不是内心多强大的人，哪怕娘家提供一些援助，她们都未必敢真正离婚，何况娘家是这种态度？她们就更不敢离婚了。

我记得有一个学员和我讲过一个故事。她说她老家在山村里，她妈妈和她讲过一件事。她邻居姐姐嫁了一个混蛋老公，经常被打得遍体鳞伤。后来她逃回了娘家，但是娘家却把她送了回去。不到一个月，这个姐姐就死了。对外称是得疾病死的，但是收殓的人却看得出来，那是被活活打死的。但娘家也无人追究，说追究起来撕破了脸不好看。

我当时就听得很郁闷，所以更加坚定了我要引领很多女性一起成长，减少这种惨剧发生。

#### 2. 父母的认知水平比较低

第二个原因与父母的认知水平有关。大家有没有发现，越是没什么文化的父母，越是不懂尊重孩子的感受。而文化水平比较高的父母，相对会更尊重孩子的感受。

**代码示例：**
```plaintext
if (父母认知水平高) {
    反应 = “这是你的人生，你的选择。只要你觉得好就可以。”;
} else if (父母认知水平低) {
    反应 = “你这样让我很没面子！必须按我说的做！”;
}
```

比如有的孩子不想结婚或者不想生孩子。父母认知水平比较高的话，就会觉得这是你的人生，你的选择，只要你觉得好就可以。他们很少会考虑你这样选择会不会让他觉得没面子，他也不会觉得你的选择会让他没面子。

认知水平高的父母会认为，这世上有很多种活法，自己开心最重要。至于丢不丢面子，根本不在他们的考虑范围之内。他们从心里觉得每个人有选择自己生活的权利，和面子无关。如果你家的是这种父母，那我真的要恭喜你了。你遇上了这世界上最好的父母，你很幸福。

但大多数父母达不到这种水平。他们只能接受世俗的生活。所有和世俗不同的行为，在他们眼里都会让他们没面子。你和他们谈什么理想、自由，和对牛弹琴其实是差不多的。因为以他们的认知，他们理解不了。而且他们也不是纯粹觉得自己没面子，有一部分是他们以自己的认知在“为你好”，觉得你的选择是错的。所以是很多综合原因促成的。

#### 3. 父母自身的成就与期望投射

第三个原因，也是最扎心、最深刻的一点，就是：**越没本事的父母，越在乎面子**。把面子看得比自己孩子还重要的父母，往往自己这辈子没什么成就和出息。

**公式：父母自身成就感低 → 将期望与面子寄托于子女**

所以，孩子在他们眼里就成了最后是否能够“扳回一局”的重要存在。如果父母本身成就很高，已经通过自己的成就受到别人的尊敬和认可，那么他们在孩子身上找补面子的欲望就弱很多。

而自己没什么本事的父母，就会把所有希望都寄托在孩子身上：希望你考名校，希望你进世界500强，希望你找富二代等等。所以不是有个很扎心的笑话吗？这世界上有三种鸟：一种是先飞的，一种是嫌累不飞的。还有一种最讨厌，自己不飞，就在窝里下个蛋，让下一代使劲飞。

你不飞，或者你飞的路线不是他想要的，不能满足他的虚荣心，他就不会放过你。

### 核心原则与应对策略

其实父母爱面子也是人之常情，甚至也不是多大的错。但是我们自己心里要明白：**到底是自己的选择追求重要，还是父母的面子重要？**

一个人最可悲的就是为了父母的面子，让自己的一生过得扭曲又拧巴，这才是最最不值得的事。

-   你喜欢文学，父母希望你当律师、当医生，你觉得你会开心吗？
-   你喜欢自己找的男朋友，你父母希望你嫁给他们安排的，你屈服了，你觉得这是孝顺吗？
-   你喜欢自己创业，你父母希望你安安稳稳打工，你放弃了自己的理想，你觉得值得吗？

为了父母的面子，放弃自己的坚持，我觉得是最愚蠢的。因为你不开心的事，你自然做不好。所以你工作没激情，婚姻没指望，你能成为你父母的骄傲吗？你放弃了一切，最终你父母还是不满意你，因为你没给他们“长脸”。

**所以，你应该按照自己的想法，努力让自己成功。等你成功了，他们自然觉得有面子了。而不是本末倒置。**

### 总结

本节课中，我们一起学习了父母可能将“面子”置于子女幸福之上的三个核心原因：
1.  **父母内心不强大**，恐惧非主流选择。
2.  **父母认知水平有限**，难以理解并尊重个体差异。
3.  **父母自身成就感低**，将未实现的期望和虚荣心投射到子女身上。

理解这些原因，不是为了指责父母，而是为了让我们自己更清醒。人生的选择权最终在自己手中。真正的孝顺，不是盲目顺从以保全他们的“面子”，而是过好自己的人生，用真正的幸福和成就，成为他们晚年真正的骄傲。

希望这节课能让很多朋友理清这个因果，不再做出错误的选择。
