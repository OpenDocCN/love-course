# 课程62：晚清·顶层人士秘而不宣的人性天书 🧠  
## 00、宣讲课  

在本节课中，我们将要学习这套人性课程的核心理念与价值。课程将揭示那些成功者通常秘而不宣的人性法则，并解释掌握这些法则如何能彻底改变个人命运。

---

我是婉晴，是《做一个刚刚好的女子》系列百万畅销书的作者，也是公众号“倾我们所能去生活”的主理人，以及女性社群“晚晴成长会”的创始人。目前总粉丝数量超过300万。

为期一年的人性大课正式推出。课程尚未开始，已有众多学员预约。这套课程备受推崇，因为它是一套深刻、扎心且通常被成功者隐藏的宝典。

这套课程包含了所有成功者不愿公开的人性秘密。领悟这些秘密后，人生将变得顺利并实现蜕变。我本人来自小县城的贫困家庭，运营公众号六个月就积累了上百万粉丝，创业八个月从零做到一千万营收。

新书上市后成为黑马，销量超过百万册。这些成就都源于我对人性的精通。你是否想过，为何信任会被欺骗、善良会被利用、真心会被伤害、付出会被辜负？问题并非全在他人，而是源于对人性的无知。

因此，尊严被践踏、利益被侵占、情感被玩弄、合作被欺骗的情况屡屡发生。处处吃亏碰壁，至今一事无成。在底层横冲直撞却不知如何上升。向他人请教时，常听到“好好努力，老天自有安排”的安慰，但等待多年仍无改变。

大多数人一生都不会被告知人性的真相。从人性角度，人们通常不愿直白地谈论人性。我们从小听到最多的是“你要做一个善良的人”“你要对别人好”，总之是“一定要做个好人”。

但没有人告诉你，要做一个**善良而有锋芒的人**。因为这种教导通常只来自最亲近的人，而你身边的人未必懂得。于是，你逐渐成为一只毫无主见和判断力的羊，任人宰割。

例如，古代帝王教导太子时，传授的是**帝王心术与权谋手段**，但向天下宣扬的是忠君爱国、赤子丹心。因此，成功者通常有两套哲学：一套是只有自己和亲近者知晓的**对内哲学**，另一套是放之四海皆可谈论的**对外哲学**。

正因精通人性，他们能够俯瞰众生，成为金字塔顶端的人。上周末，我与先生参加了一场高端聚会，许多企业家在此为孩子铺路或安排终身大事。

一位父亲正在教育孩子。孩子提议增加员工幸福感，在每个员工生日时送蛋糕和鲜花。父亲说：“你的想法表面很好，但你了解人性吗？初期大家会高兴，但如果年年如此，大家会将其视为流程，感受减弱。而如果某年没做或遗漏某人，反而会成为过错。员工规模较小时可行，但当员工成百上千时，必须考虑做法的长远影响。”

我当时深有感触。当普通人思考该给孩子报多少学习班、学多少才艺时，一些成功人士教导孩子的是**人性与驾驭**。若干年后，这两类孩子的人生轨迹可以预见。

你的孩子即使报了再多学习班、考了再高的分数，很可能只是一个技能更全面的打工者；而成功人士的孩子早已精通人性。这如同幼儿园小朋友与成年人之间的比拼，谁领导谁一目了然。

但学习了这套人性大课，一切可能悄然改变。你将摆脱被人掌控的宿命，避开被人伤害的下场，看清各种陷阱与把戏，认清所有真心与假意，识别伪君子与真小人。从此，任何伪君子的手段在你面前都无所遁形。

你还可以见招拆招，不再让任何人伤害你，甚至能轻松收服他们。最关键的是，你将获得破解人性财富密码的钥匙，让别人心甘情愿地付钱给你，并对你感激涕零。上层圈子从不外传的秘法，即将被你掌握。
