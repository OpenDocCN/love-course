# 课程62：晚清·顶层人士秘而不宣的人性天书 🧠
## 第24课：人缘爆棚的人性社交密码 🔑

在本节课中，我们将要学习如何通过理解人性，掌握有效的社交方法，从而建立广泛且有益的人际关系。社交能力并非天生，而是可以通过学习和实践来提升的。我们将避开常见的误区，并掌握三条核心的社交密码。

很多人觉得自己不善于社交，没有朋友，想提升却不知从何做起。也有人认为不会社交也无所谓。但社交的重要性不容忽视。

假设你找到一份工作，但没有要好的同事，所有人都对你冷冰冰，你待在这样的环境里不会舒服。你的孩子进入学校，你与所有家长、老师都没有交往，消息闭塞，你也不会觉得舒服。再往后，我们会面临孩子升学、结婚、找工作，父母生病等问题，如果身边一个能说话的人都没有，滋味会很难受。等到孩子结婚后，你不想打扰他的生活，但你的生活里只有你一个人，孤独的滋味会更难受。

年纪越大，越需要社交，也越应该明白社交的重要性。我曾因家境原因不愿与人过多交往。后来自己创业，方方面面都要打理，社交能力就被慢慢逼出来了。我的感受是，身边多些朋友，人生会更温暖、更圆满。毕竟我们不是生活在原始森林里。退一步说，社交与个人财富紧密挂钩。没有社交，何来财富？假如你开了一家店，没有任何朋友来捧场，店能开起来吗？就算在网上做生意，没有一点社交能力，客户能和你成交吗？社交能力差没关系，但我们要重视这种能力，哪里薄弱就补哪里。相信今天这节课会直接把万能的社交钥匙交给你。如果你能掌握，我相信你会爱上社交。

在正式分享之前，我想提前规避两个社交误区。

以下是第一个误区：
*   **追求有效人缘，而非人人喜欢**：虽然标题有“人缘爆棚”四个字，但我们指的是有效人缘，而不是指人人都喜欢你。不必去做一个人人都喜欢的人。打个比方，你把一颗长得非常好的蔬菜放在狼的面前，狼不肯吃。狼不愿意吃，和你的蔬菜好坏一点关系都没有，因为狼喜欢吃的是肉。所以如果有些人不喜欢你，不必在自己身上找原因。你要做的是做好自己，然后吸引那些喜欢你的人。

以下是第二个误区：
*   **避免成为“老好人”**：不要为了人缘爆棚，就选择做老好人。我曾研究过很多老好人的人缘情况。大家看到他们会热情打招呼，因为老好人善良，没有攻击性，大多也曾受过他们的情。表面上看，老好人好像人缘挺好。但当这些老好人自己需要别人帮忙时，他们就会发现愿意帮忙的人很少。有些曾经被他们帮过很多次的人，也会选择装聋作哑。这种所谓的好人缘不但没有用，反而会消耗你大量的时间精力，最后被辜负了，还会形成内耗。

我们今天要分享的人缘爆棚的人性社交密码和这种“人缘”完全不同。

上一节我们明确了社交的目标与误区，本节中我们来看看第一条核心密码。

**第一条：你要有自己的特色**。

其实大家平时注意观察就会发现，那些社交能力很强的人都有自己的特色，要么是非常豪爽，要么是非常犀利，要么是非常豁达等等。而那些社交能力很弱的人，基本上都差不多，要么怯怯诺诺，要么说话都不敢大声，存在感非常低。

就像我们去参加一个聚会，有的人会让你很快记住，有的人可能见过10次，你都未必记得住他的名字，就是因为这个原因。当年我刚做公众平台时，研究了一些迅速做起来的平台，发现他们火起来的原因，就是一个个都是个性鲜明的人。而那些每天写感悟、写心得的人，哪怕文笔很好，点击量也寥寥无几。社交也是一样的道理，你要有自己的特色。

以我自己为例，我在社交上除非是那种非常正式的场合，我可能会端一下，大多场合我都是犀利有趣的风格。除非我故意收敛锋芒，不愿意去抢别人的风头，否则我大多时候存在感都很强。大家要想一想，那些成功人士有哪个是躲在一边的呢？你的社交能力基本决定了你的财富指数，你还想做个透明的社交小白吗？

从现在开始，想想你身上的特点。如果你生活中很大方，那么就让这种豪爽的印象深入到你身边每个人的心目当中。如果你是个很犀利的人，也大胆发表自己的意见。

当然，在这里有一点要提醒大家，我所说的特点必须是好的，哪怕不是优点，也必须是中性的。比如我的文章说话都很犀利，但我自己很清楚，我可以犀利，绝不能刻薄。大家愿意接受犀利，但绝对不会喜欢一个刻薄的人。所以，斤斤计较、自私透顶、尖酸刻薄这种特点，千万不能出现在社交场合，那会让所有人都避而远之。

在建立了个人特色之后，我们需要一种具体的方法来快速拉近与他人的距离。这就是第二条密码。

**第二条：记住你能记住的一切**。

我在社交上有一个很大的优势，就是我的记忆力非常好。其实在直播前我从来不会告诉别人我记忆力很好，只有身边很亲近的人才知道，因为这是我的社交法宝之一。我可以迅速记住别人的名字、什么地方的人、有什么样的履历。

打开人与人之间社交大门最好的那把钥匙，就是你能够准确地说出对方的一切。卡耐基说过，沟通中，我们一旦说出对方的姓名，就会给对方一种被关注、被用心记住的感觉。这种感觉就满足了我们被认可、被重视的需求。

所以，要和对方拉近距离，最好的方式就是要么你能够在第一次就准确地认出对方，要么就在第二次能够准确地认出对方。记住这个方法的要点是**准确认出**。你要是认错的话，那还不如不说，不然给人的感觉会很差。

我举几个例子。你第一次见某人，其实你已经在其他人那里得知他是谁，然后你就可以和他打招呼：“我以前在照片里见过你，所以冒昧来打声招呼，希望没有认错人哦。”这么说，既告诉对方你已经在关注他，也给自己留了点余地，万一搞错了，也不至于太尴尬。

再比如，你和对方第二次见面，但是你准确地叫出了他的名字，甚至记住了他的喜好。比如点菜的时候，知道他对什么过敏或者不吃什么东西，他对你会立刻好感倍增。这一点可以说是百试百灵的。

我们可以换位思考一下，当别人能够准确地认出我们时，我们的感受是怎么样的呢？是不是很好？对对方也会立刻友好起来。毕竟他在关注我们，大家对关注自己的人天生就会多一份好感，这就是人性。如果别人认错了我们，我们是什么样的感受呢？肯定觉得对方莫名其妙、毫不用心，第一时间就对他印象不好了。

我以前就经历过好几个尴尬的时刻。有一次我和一位企业家朋友一起去参加一个聚会。那位朋友事业做得很好。当时我们正在聊天，突然有一个人举着杯子过来敬他，对方热情地说：“秦总啊，很荣幸在这里见到你……”但我朋友的脸色已经不太好了，因为他根本不姓秦。这个“秦”只是他公司名字里的其中一个字。对方越说“久仰”，朋友的脸色就越疏离。要是真的久仰，怎么连对方姓什么都不知道呢？这就是典型的搞砸了。

对于这一点，我也是深有感触。记得有一次我建了一个群给大家答疑解惑。有一位粉丝说，关注我5、6年了，是我的老粉了。然后有什么问题想请教我。我看着他打的“晚晴”两个字，他把“晚晴”的“晚”打成了“婉约”的“婉”，“晴”打成了“钢琴”的“琴”。在他连续错了三次的时候，有位粉丝就受不了了，说：“你要是真的关注了晚晴姐5、6年，怎么连她的名字还全部打错呢？”其实我当时的感受和那位粉丝说的一样。所以我自己在这方面非常注意。如果不确定，我要么就笑脸相迎，要么就用尊称，比如“贵客”、“稀客”、“大忙人”，总之就是让别人舒服的称呼。

掌握了记忆与识别的技巧，我们将进入最高阶的社交心法，这需要更深入地洞察人性。

**第三条：如果你能让每个人都觉得自己是被特殊对待的，那你在社交上就所向披靡了**。

关于这一点，我就用故事来呈现吧，方便大家领悟其中的奥妙。很多小伙伴可能都看过我和我一位朋友每天互怼斗嘴的趣事，今天就和大家聊聊这位朋友。我这位朋友是我们当地的纳税大户，人脉资源非常广。我们认识到现在大概是4年时间，我都不知道这4年里，他帮了我多少忙了。

我现在还记得我第一次和他见面的时候，他身边围绕着一堆奉承他的人。虽然他一直被人奉承着，但我能够感受到他的不耐烦。我当时就在心里说，这些人不够聪明。身为一个企业家，他身边还会缺少奉承他的人吗？好话他早就听腻了。再说了，他难道不知道你们说好话的意图吗？反正我是不会这么干的。

我和他第一次交锋是关于男女两性相处的观点，他挺大男子主义的，我也不是一个忍气吞声的角色，加上我口才比他了得，把他怼得体无完肤。最后他说服不了我，我也说服不了他。但我知道，对于我这种根本不拿他当回事的人，他其实特别欣赏。从那以后只要见到我，他就喜欢和我争论，从男女的地位到重男轻女的话题，我们什么都怼，基本上他就没有赢过，但是屡战屡败，屡败屡战，斗志昂扬。

斗得久了，有一天他和我说：“我经常和你唱反调，你是不是很讨厌我啊？”我说：“我要是真的讨厌一个人，我的表现就是不理他。我经常和你怼，说明我内心其实很欣赏你，要不然理都不想理你。”他闻言笑得像个孩子一样说：“其实我也很欣赏你……”然后我们就变成了相互吹捧模式，但下一次见面基本还是互怼。因为在口才上比不过我，他就在人脉资源上下手，只要我稍微遇到点问题，他就立刻出手帮我搞定。现在差不多混成了我的管家兼秘书。

在我的影响下，他对老婆越来越好，对女性越来越尊重，甚至渐渐地认同我的那些思想了。他的跟班偷偷和我说：“M总脾气可大了，经常骂人，只有你能够把他怼得体无完肤。”

在这个案例中有几个要点呢？

以下是第一个要点：
*   **抓住对方人性中的弱点**：我在第一次和他聊天的时候，我就知道他欣赏的是那种桀骜不驯的人。越是不拿他当回事儿，他越觉得你有趣；你越是去讨好他，他会觉得又是一个不知道打着什么主意的人，根本不想理你。所以，以“怼”的方式再去夸奖对方，他反而会更受用，觉得你是真心的欣赏他才会夸奖他，和其他人不同。当然，事实上我也真的挺欣赏他的。

以下是第二个要点：
*   **每个人内心都住着一个孩子**：这是我研究人性后很感慨的一点。一个人不管年纪多大，事业多牛，都有幼稚的一面，只是大多时候不表现出来。如果你能让对方把他内心的小孩子放出来，那么我可以肯定，你一定是他极其信任的人。一个人能让对方放松警惕，卸下心房，用最舒服、最自在的方式社交，你就已经搞定这个人了。

以下是第三个要点：
*   **让对方觉得自己是独特的、唯一的**：我告诉M总：“其实我平时对别人都是礼貌而疏远的，但是你是个性情中人，大家没有必要装着端着，朋友之间嘛，就是打打闹闹才好玩。”他深以为然，要我好好保持，千万别把他当老总，那样就没有意思了。

其实每个人都有自己的特点和个性，只要你能够精准把握，真的没有搞不定的人。真正的人性高手从来不是以一种方式对待所有人，他会根据每个人的喜好去和他们相处。当然，这一点也是最难、要求最高的一点。你等于要为每个你看得上的人量身定制一套相处方法。但如果你能够做到的话，这套法宝可以助你登上难以企及的社交高度。

关于“人缘爆棚的人性社交密码”这一课，我们就分享到这里。接下来我们还会分享更多社交法宝。我相信等这个系列的课程学完，你会成为牛逼闪闪的“社牛”。

**本节课总结**：
在本节课中，我们一起学习了提升社交能力的核心密码。首先，我们明确了社交的重要性并规避了“追求人人喜欢”和“做老好人”两大误区。接着，我们掌握了三条关键法则：**建立鲜明的个人特色**、**用心记住他人的细节并准确识别**、以及最高阶的**通过洞察人性让对方感到被特殊对待**。社交是一门可以习得的艺术，关键在于理解人性并真诚实践。
