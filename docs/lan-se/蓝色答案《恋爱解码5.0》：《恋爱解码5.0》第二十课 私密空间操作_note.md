邀约时必须自信、自然。避免犹豫或不敢直视对方。用确定的口吻提出，例如：“我们找个地方休息一下”、“亲爱的，今晚我们住外面吧”或“今晚陪我好吗”。你的态度决定了女生的接受程度。
