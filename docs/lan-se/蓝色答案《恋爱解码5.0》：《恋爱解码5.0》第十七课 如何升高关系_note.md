2.  将手搭在她肩膀上，轻轻将她揽近。

4.  牵手。

关键原则是**循序渐进**。如果进行到某一步（如搂肩）对方表现出不适，就退回上一步（如继续抚摸头发），待对方放松后再尝试。整个过程中需保持聊天，让氛围轻松自然。

肢体接触不是为了满足单方面的需求感。长时间保持同一接触（如一直紧握手）可能让对方不适。
