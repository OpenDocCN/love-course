## 🔍 选择合适的伴侣

你一定要选择适合谈恋爱的女性。

*   **不要在意别人的眼光和议论**。

最终，你可能只是一个普通的老人。关键在于你整个生命过程中，是否自己掌握了命运，在这些选择中心里是否舒服、无怨无悔。我承担了责任，不埋怨任何人，我过得很开心。

所以，一定要注意：**自己去承担责任，要保持独立**。只有独立的人才配享受爱情。

所以，财务必须各自独立。男生应该负责日常共同开销。

*   **男生负责**：日常开销、房租、水电、两人外出吃饭、看电影等共同消费。

如果你觉得这样对男生不公平，这种想法不对。因为你是男人，需要多承担责任。你一定要学会承担责任，勇敢面对问题并想办法解决。

如果你没钱，应该与她沟通。你可以说钱不够了，提议吃东西不用太好，或减少娱乐活动。不要自己没钱又死要面子。要与她沟通面临的实际情况。

千万不要去刷信用卡等，这是非常糟糕的习惯。你在使用自己不具备的支付能力，最终会导致财务一团糟。有钱时多花点，没钱时少花点，与她商量着来，这是最合适的。

你自己的钱，除了共同开销部分，也由你自己处理。两个人财务分开，就不会彼此产生怨言（例如，你觉得她用你的钱太多，不公平）。

所以，财务安排应该是：**你负责两人共同开销，各自的开销各自负责**。这是一个让人舒服的财务状态，能为你们的关系打好基础。

## 🗣️ 长期关系中的五大要点（二）：成熟沟通

第二点是沟通方式。你与她的沟通方式应该是舒服的，而不是容易情绪波动。

不要很容易就生气、吃醋或不满意。兄弟们，你们一定要成熟起来。

**成熟意味着思考如何解决问题，而不是先发泄情绪。发泄情绪是不成熟的表现。**

回想我们的课程：所有技巧都需要你在与女性互动时运用，接收反馈，发现不足并改善，最终会有一个好结果。生活也一样，遇到问题，不是生气、痛苦、宣泄，这解决不了问题。

这与课程原理相同：找到问题的关键点，思考如何运用方法解决问题，这才是关键。我们要想办法，而不是出现问题后只想抱怨和发泄。这是严重不成熟的表现。

所以，一定不要有情绪。想一想问题出了，我怎么去解决。你与她有争执时，想想怎么解决问题。谁对谁错根本不重要。

有时她可能情绪不稳定，你要去包容。如果你心里有想法，要告诉她，而不是情绪化地发泄。不要很生气地指责：“你怎么这样！”

你可以说：“亲爱的，你这样说，我心里会有点不舒服。” **注意语气，是沟通的语气，而不是高声、不爽的状态**。否则她会觉得你根本不在意她的感受。

这些细节一定要做好。你与她在一起时，不要有那么多负面情绪，而是要去沟通、解决问题。当你把自己的想法告诉她，她也会在乎你，知道自己错了，你们的关系就修复了。

而不是你在那边很想发泄，她觉得你根本不关心她，两个人情绪对立。一定要记住：很多人说“我们经常吵架但还在一起很开心”，这完全错误。

**好的关系是不会吵架的**。好的关系是两个人一起去解决问题。吵架有什么用？她不会吵架。一定要注意。

第三点非常重要：我们一定要有各自独立的空间，并支持对方实现自我价值、完成梦想。

你们两人必须有自己的事情、事业和奋斗目标。两个人有各自的空间和要做的事情。然后，在休息时、周末或每天回家后，可以互相分享新的想法、构想，进行碰撞。每天都有新鲜感，这种状态才是积极、好的。

平时要多去表达。
