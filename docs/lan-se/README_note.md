# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。这个“标签”就是变量名，而“盒子”里装的东西就是数据。

在Python中，为一个变量赋值非常简单，使用等号（`=`）即可。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如，`name = "蓝色答案"` 这行代码创建了一个名为 `name` 的变量，并将字符串“蓝色答案”存储在其中。

## 基本数据类型 🔢

理解了如何存放数据的“盒子”后，我们来看看“盒子”里可以存放哪些不同类型的“物品”。Python中有几种内置的基本数据类型，它们决定了数据可以进行的操作。

以下是三种最常用的基本数据类型：

1.  **整数（int）**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
2.  **浮点数（float）**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
3.  **字符串（str）**：表示文本信息，需要用单引号（`'`）或双引号（`"`）括起来，如 `"Hello"`, `'Python'`。

## 如何查看和转换类型？ 🔄

有时我们需要知道一个变量里存储的数据是什么类型，或者在不同类型之间进行转换。Python提供了方便的函数来完成这些任务。

要查看一个变量的数据类型，可以使用 `type()` 函数。
```python
x = 42
print(type(x))  # 输出：<class 'int'>
```

要在不同类型之间进行转换，可以使用 `int()`, `float()`, `str()` 等函数。
```python
num_str = "123"
num_int = int(num_str)  # 将字符串"123"转换为整数123
print(num_int + 7)      # 输出：130
```

## 动手试一试 ✍️

理论需要结合实践。让我们通过一个简单的例子来运用刚才学到的知识。

假设我们要记录一个用户的姓名、年龄和余额，并打印一条欢迎信息。

以下是完整的代码示例：
```python
# 定义变量
user_name = "小明"
user_age = 25
user_balance = 1500.50

# 打印信息
print("欢迎用户：" + user_name)
print("年龄：" + str(user_age))  # 将整数年龄转换为字符串后才能拼接
print("账户余额：" + str(user_balance) + "元")
```

运行这段代码，你将在屏幕上看到对应的用户信息。
