## 1. 座位选择：避免尴尬对视

上一节我们介绍了课程概述，本节中我们来看看约会时的座位安排。与女生约会并需要坐下时，应避免坐在她的正对面。

## 2. 气味管理：营造良好第一印象

解决了视线尴尬问题后，另一个直接影响第一印象的因素是气味。清新宜人的气味能大大加分。

以下是快速改善气味的方法：
*   **使用香水**：将香水喷洒在耳后、颈后等脉搏处。
*   **处理体味**：如有狐臭，需加强使用止汗或香体产品。
*   **保持口气清新**：随身携带口香糖或漱口水，即使自我感觉良好也应准备，以备不时之需。
*   每次约会都应以一个愉快的收尾为目标，良好的气味是基础。

## 3. 个人卫生：展现基本尊重

在关注了环境气味后，更根本的是个人卫生。整洁的外表是对约会对象的基本尊重。

以下是男生需注意的个人卫生要点：
*   清洗头发，保持清爽，避免油腻和头皮屑。
*   刮净胡子，修剪鼻毛。
*   穿着干净整洁的衣物。
*   如果穿白色鞋子，出门前应擦拭干净。
