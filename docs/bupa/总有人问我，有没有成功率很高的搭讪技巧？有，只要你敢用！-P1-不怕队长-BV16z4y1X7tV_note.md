# **搭讪技巧教程 - P1：如何优雅地搭讪**

在本节课中，我们将学习如何使用高成功率的方法进行搭讪。

## **搭讪情境分析**

漂亮的女生一天会被十几个男生搭讪，正常她会有两种选择：

1. **直接给微信**：直接要求女生提供联系方式。
2. **下次再见时给**：在下次相遇时再提供联系方式。

你会选择哪种呢？

## **搭讪技巧选择**

不要使用过于直接或冒犯性的语言，例如：“不要撩我，对不起对不起打扰了，没事我等一下跟你说。”

## **应对策略**

当你遇到这种情况时，选择第二种方式可能更为合适：

1. **礼貌回应**：谢谢，这么巧啊又见面了。
2. **邀请互动**：走啊。

以下是搭讪技巧的**公式**：

```
礼貌回应 + 邀请互动
```

## **总结**

本节课中我们一起学习了如何优雅地搭讪，通过礼貌回应和邀请互动，提高搭讪的成功率。
