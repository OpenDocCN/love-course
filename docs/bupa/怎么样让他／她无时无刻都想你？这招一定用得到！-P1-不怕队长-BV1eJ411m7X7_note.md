# **如何让他/她时刻想念你？技巧篇 P1**

🌟 **概述**：本节课我们将学习如何通过一些简单的方法，让心仪的人时刻想起你。

## **一、选择一个共同的喜好**

**以下是选择共同喜好的步骤**：

1. **观察对方喜好**：注意对方喜欢什么，比如饮料、电影、音乐等。
2. **选择一个共同点**：比如，你发现对方也喜欢喝某种饮料。
3. **日常提及**：在日常生活中，多次提及这个共同点，比如：“这个饮料真不错，我每天都喝这个。”

## **二、创造共同回忆**

**以下是创造共同回忆的方法**：

1. **共同购买**：为对方购买他/她喜欢的物品，比如饮料。
2. **分享体验**：在对方面前享用这个物品，并分享你的感受。
3. **日常提及**：在日常生活中，提及这个共同体验，比如：“今天我又喝了这个饮料，想起了我们上次一起喝的场景。”

## **三、利用联想法子**

**以下是利用联想法子的步骤**：

1. **选择一个物品**：比如，对方喜欢的饮料。
2. **创造联想**：将这个物品与对方联系起来，比如：“每次喝这个饮料，我都会想起你。”
3. **日常提及**：在日常生活中，提及这个联想，比如：“今天我又喝了这个饮料，想起了你。”

## **四、总结**

**本节课中我们一起学习了如何通过选择共同喜好、创造共同回忆和利用联想法子，让对方时刻想念你**。希望这些方法能帮助你实现你的目标。
