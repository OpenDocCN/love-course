# **电影版《我不喜欢乞丐》情感分析教程 - P1**

## 概述
在本节课中，我们将分析电影《我不喜欢乞丐》中一段重要的情感对话，探讨人物情感变化和人际关系。

## 对话内容分析

### 顧婶的情感表达
顧婶表达了对主人公的深情，但同时也透露出她的偏见。

**顧婶**：
- 我爱你，你能和我在一起吗？我对你好是因为你是我的病人，我对所有的病人都这样，我不喜欢其他人。
- 我爱你，对不起，我不喜欢送外卖的，我不是送外卖的，我是工地常工，我不喜欢工地常工。

**分析**：
顧婶的这些话表明，她对主人公的感情是基于职业关系，而非个人喜好。她不喜欢送外卖的和工地常工，这反映了她的偏见。

### 主人公的情感变化
主人公在对话中经历了从自卑到自信，再到对金钱和外表的重视，最终意识到情感的真实价值。

**主人公**：
- 我喜欢有学识的人，台商月在天上，心上人在眼前。
- 我现在是一个有学识的人，我不喜欢有学识的人，我喜欢有钱人。
- 我现在中了有史以来最高的彩票，我现在是个有钱人了，我爱你，可是我现在不喜欢有钱人了。
- 我喜欢帅哥，顾婶，我呀，我拿所有的钱去整了你，你不是说你喜欢帅哥吗？其实你变成什么样子我都不会喜欢你的，谢谢。

**分析**：
主人公最初喜欢有学识的人，但随着时间的推移，他的价值观发生了变化。他开始追求金钱和外表，但最终意识到，无论外表如何变化，顧婶对他的感情都不会改变。

## 总结
本节课中，我们一起学习了电影《我不喜欢乞丐》中一段重要的情感对话。通过分析顧婶和主人公的情感表达，我们探讨了人物情感变化和人际关系。这段对话揭示了人们对于爱情和人际关系的不同看法，以及情感的真实价值。
