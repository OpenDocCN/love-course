# 恋爱观课程 01：成年人的恋爱观 🫂

在本节课中，我们将要学习成年人应具备的恋爱观。核心在于理解健康的关系建立在相互尊重与双向奔赴的基础上，而非单方面的执着与付出。

上一节我们介绍了课程主题，本节中我们来详细看看其核心原则。

## 核心原则：爱是双向奔赴

爱一个人不能姿态过低。打电话对方不接，就无需再反复拨打。发信息对方不回，也不必继续发送。

如果一个人心里有你，他不会让你长时间等待，更不会让你感到痛苦。

**爱不是一厢情愿，而是两情相悦**。这可以用一个简单的公式来理解：

**健康的关系 ≈ 主动 + 回应**

没有希望的关系，不必强行坚持。得不到回应，为何还要单方面主动？

---

以下是基于上述原则的具体行为指南：

*   **保持自尊**：不过度迁就，不丧失自我。
*   **识别信号**：关注对方的行动而非言语，缺乏回应本身就是一种回应。
*   **及时止损**：当感受不到对方的在意时，应懂得停止付出。

---

本节课中我们一起学习了成年人恋爱观的核心：**爱是相互的**。单方面的执着与卑微付出无法维系健康的关系。真正的爱建立在双方自愿、平等且彼此回应的基础之上。
