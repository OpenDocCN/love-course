## 冷读术简介
冷读术是一种心理技巧，通过观察和分析对方的言行举止，推测其内心想法，进而与之产生共鸣。

## 如何运用冷读术
以下是运用冷读术的步骤：

### 1. 观察对方
**公式**：观察 → 发现 → 分析

- 观察对方的言行举止，寻找线索。
- 发现对方的特点和习惯。
- 分析其背后的心理动机。

### 2. 推测内心想法
**公式**：观察 → 推测 → 共鸣

- 根据观察到的线索，推测对方的内心想法。
- 与对方产生共鸣，拉近彼此关系。

### 3. 应用冷读术
**代码**：
```python
def cold_reading(observation):
    # 分析观察到的线索
```
