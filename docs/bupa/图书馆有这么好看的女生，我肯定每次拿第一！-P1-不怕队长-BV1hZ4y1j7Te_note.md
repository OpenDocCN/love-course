# 课程名称：图书馆座位获取策略 🧠 - 第1课

在本节课中，我们将要学习一种在图书馆寻找座位的观察与行动策略。我们将通过分析一个具体场景，拆解其核心逻辑，并将其转化为可操作的步骤。

## 概述

课程从一个常见的图书馆场景开始：当图书馆没有空位时，如何通过观察和判断来创造或发现就坐的机会。核心思想在于主动识别潜在的可利用空间，而非被动等待。

## 场景分析

上一节我们介绍了课程的整体目标，本节中我们来看看具体的初始场景描述。

图书馆看书时如果没有位置，可以采取主动观察的策略。视频中展示的方法是：发现一位独自就坐的读者旁边有空闲的椅子空间。此时，可以主动上前询问或直接利用该空间就坐。

如果对方表示同意或没有明确反对，那么座位问题就得到了解决。如果对方表现出明显的抗拒或场景不合适，那么继续停留就没有意义，应该离开去寻找其他机会。

## 核心策略拆解

基于以上场景，我们可以将策略归纳为几个核心步骤。以下是具体的行动指南：

1.  **观察环境**：快速扫描图书馆，寻找那些看似独自一人且旁边有物理空间（如空椅子或桌面空间）的读者。
2.  **评估可行性**：初步判断上前使用该空间是否会造成对方的强烈不适或违反图书馆规定。
3.  **采取行动**：礼貌地询问或自然地使用该空间。关键行动代码可简化为：`if (hasSpace && !isOccupied) { takeSeat(); }`。
4.  **处理反馈**：根据对方的反应决定下一步。公式可表示为：**继续留下 = 获得许可 || 无负面反馈**。
5.  **灵活应变**：如果反馈为负面，则果断执行 `leave();` 指令，转而寻找其他目标。

## 策略总结

本节课中我们一起学习了在图书馆等公共空间获取座位的一种主动策略。其精髓在于从“寻找空位”转变为“发现空间”，并通过清晰的观察、评估和行动循环来解决问题。记住，核心是礼貌与观察力，并在行不通时果断放弃当前目标，保持灵活性。
