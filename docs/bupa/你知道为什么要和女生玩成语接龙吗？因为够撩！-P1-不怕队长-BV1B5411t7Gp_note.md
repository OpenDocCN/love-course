# 成语接龙游戏教程 - P1

成语接龙是一种流行的文字游戏，通过接龙的方式，让参与者运用成语，增加语言表达能力和乐趣。以下是一份简单的成语接龙游戏教程。

## 概述

在本节课中，我们将学习如何进行成语接龙游戏，了解游戏规则和基本技巧。

## 游戏规则

以下是成语接龙游戏的基本规则：

1. **选择成语**：游戏开始时，由一名玩家选择一个成语作为起始成语。
2. **接龙**：下一个玩家需要接上一个成语的最后一个字，并开始一个新的成语。
3. **限制**：每个成语必须至少包含四个字。
4. **胜负**：如果某个玩家无法在规定时间内接出成语，则该玩家输掉游戏。

以下是一个简单的成语接龙游戏示例：

1. **起始成语**：不怕队长
2. **接龙**：公子尔卿
