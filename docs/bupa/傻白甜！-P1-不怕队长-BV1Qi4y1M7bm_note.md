## 什么是变量？🤔

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。

在编程中，**变量**就像一个贴有标签的盒子。你可以把数据（比如数字、文字）放进这个盒子里，并且通过标签（变量名）来找到和使用盒子里的内容。它的核心作用是**存储信息**，以便程序后续可以方便地引用和修改这些信息。

我们可以用一个简单的公式来理解变量的赋值过程：
`变量名 = 值`

## 如何创建变量？✍️

理解了变量的概念后，接下来我们学习如何实际创建一个变量。在Python中，创建变量（也称为“声明变量”）非常简单直接。

以下是创建变量的基本步骤：
1.  为你的变量想一个名字（例如 `my_number`）。
2.  使用等号 `=`，这个符号被称为“赋值运算符”。
3.  在等号右边写上你想存储的值（例如 `10` 或 `"你好"`）。

让我们看一个具体的代码示例：

```python
# 创建一个名为‘age’的变量，并在其中存储数字18
age = 18

# 创建一个名为‘name’的变量，并在其中存储文字“不怕队长”
name = "不怕队长"
```

在这段代码中，我们创建了两个“盒子”。一个标签为 `age`，里面放了数字`18`；另一个标签为 `name`，里面放了文字`"不怕队长"`。现在，只要我们使用 `age` 或 `name`，Python就知道我们指的是哪个盒子里的数据。

## 变量的命名规则 📝

我们已经学会了创建变量，但并非任何名字都可以作为变量名。为了代码清晰且能被Python正确识别，变量命名需要遵循一些规则。

以下是变量命名的主要规则和约定：
*   **可以包含**：字母（a-z, A-Z）、数字（0-9）和下划线（_）。
*   **不能以数字开头**：例如 `1variable` 是无效的，但 `variable1` 是有效的。
*   **区分大小写**：`Age`、`AGE` 和 `age` 是三个不同的变量。
*   **不能使用Python关键字**：例如 `if`、`for`、`while` 等具有特殊功能的单词不能用作变量名。
*   **建议使用描述性的名字**：使用 `user_age` 比单纯使用 `a` 或 `x` 更能让人理解变量存储的是什么。

## 使用变量 🔄

创建变量之后，我们可以在程序中使用它们。使用变量意味着引用它存储的值。

你可以直接打印变量的值，也可以在计算中使用它们。例如：

```python
# 使用之前定义的变量
print(name)          # 这会输出：不怕队长
print("年龄是：", age) # 这会输出：年龄是： 18

# 在计算中使用变量
birth_year = 2024 - age
print("出生年份大约是：", birth_year)
```

当你使用 `print(name)` 时，Python会找到标签为 `name` 的盒子，取出里面的值 `"不怕队长"` 并显示出来。变量让代码变得灵活，如果需要修改年龄，只需改变 `age = 18` 这一行赋值语句，所有用到 `age` 的地方都会自动更新。
