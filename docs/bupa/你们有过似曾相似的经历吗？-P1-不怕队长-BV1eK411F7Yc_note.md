# 电梯幻想 - P1 - 不怕队长 - BV1eK411F7Yc

## 概述
在本节课中，我们将探讨一个常见的幻想场景：在电梯中遇到心仪的人。

## 电梯中的幻想
每次坐电梯，我都在幻想，电梯门打开的那一刻会出现一个美女。这种事情真的会发生在我身上吗？我要不要主动跟她说句话？她万一有男朋友怎么办？她要是喜欢上我，她男朋友肯定会打我。她是不是冲我撩头发了。

## 决定行动
不管这么多了，我就跟她说我们是不是在那见过，会不会太土？土就土吧，准备上电梯了。哎，这个电梯实在是太讨厌了，为什么这么快啊。
