# 课程名称：基础编程入门 - 第1课：变量与赋值 🎯

在本节课中，我们将要学习编程中最基础也是最重要的概念之一：变量与赋值。理解这个概念是编写任何程序的第一步。

## 概述

我们可以把计算机的内存想象成一个巨大的储物柜仓库。每个储物柜都有一个**唯一的地址**，用来存放数据。但是，直接通过地址（比如0x7FFF5A3B）来查找和操作数据非常困难且容易出错。因此，编程语言引入了**变量**的概念。

**变量**本质上是一个我们为某个内存位置起的易于理解和记忆的名字。通过这个名字，我们可以方便地存储、查找和修改其中的数据。

## 什么是变量？

上一节我们介绍了内存与地址的比喻，本节中我们来看看如何具体使用变量。

创建一个变量并为其赋予一个值的过程，称为**赋值**。在大多数编程语言中，这通过等号 `=` 来完成。但请注意，编程中的等号 `=` 通常表示“赋值”或“设置为”，而不是数学中的“等于”。

其核心操作可以用一个简单的公式表示：
**变量名 = 值**

这意味着将**等号右边的值**，存入**等号左边变量名所代表的内存空间**中。

以下是变量赋值的一个基本代码示例：

```python
# 将数字 10 赋值给名为 number 的变量
number = 10

# 将文本“你好，世界！”赋值给名为 greeting 的变量
greeting = "你好，世界！"
```

## 如何使用变量？

理解了赋值操作后，我们来看看变量在程序中的实际用途。

一旦变量被赋值，我们就可以在代码的其他地方使用这个变量名来代表它所存储的值。程序会自动找到对应的内存位置并取出值进行计算或操作。

以下是使用变量的几个关键点：

1.  **声明与初始化**：在许多语言中，第一次为变量赋值的行为，既声明了变量的存在，也初始化了它的值。
2.  **值可更改**：变量的值是可以改变的。你可以对一个变量进行多次赋值，新的值会覆盖旧的值。
3.  **参与运算**：变量可以像具体的数值一样，参与各种数学或逻辑运算。

让我们通过一段代码来演示这些特性：

```python
# 第一次赋值：声明变量 price 并初始化为 50
price = 50
print("商品价格是：", price) # 输出：商品价格是： 50

# 更改变量的值：将 price 的值改为 45
price = 45
print("打折后价格是：", price) # 输出：打折后价格是： 45

# 使用变量参与运算
quantity = 2
total_cost = price * quantity # 使用 price 的当前值 45 进行计算
print("总价是：", total_cost) # 输出：总价是： 90
```

## 变量命名规则

为了写出清晰、可维护的代码，为变量选择一个好名字至关重要。以下是通用的变量命名规则：

1.  **由字符、数字和下划线组成**：名字中可以包含字母（A-Z, a-z）、数字（0-9）和下划线（_）。
2.  **不能以数字开头**：例如 `1variable` 是无效的，而 `variable1` 是有效的。
3.  **区分大小写**：`myVar` 和 `myvar` 是两个不同的变量。
4.  **避免使用语言关键字**：不要使用像 `if`、`for`、`while` 这类编程语言本身有特殊含义的词作为变量名。
5.  **力求清晰有意义**：使用描述性的名字，如 `user_age`、`total_score`，而不是简单的 `a`、`x`。

## 总结

本节课中我们一起学习了编程的基石——变量与赋值。

我们首先将计算机内存类比为储物柜，解释了变量作为“储物柜标签”的作用。然后，我们学习了如何使用等号 `=` 进行赋值操作，其核心是 **`变量名 = 值`** 的公式。接着，我们探讨了变量的主要用途：存储数据、允许值被修改以及参与程序运算。最后，我们介绍了为变量命名的基本规则，这是写出好代码的重要习惯。
