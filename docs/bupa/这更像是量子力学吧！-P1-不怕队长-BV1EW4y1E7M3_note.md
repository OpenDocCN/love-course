# 量子力学入门教程 - P1

## 概述
在本节课中，我们将探讨量子力学中的基本概念，特别是作用力和反作用力的相互性。

## 作用力和反作用力
根据牛顿第三定律，力的作用力和反作用力是相互的，且大小相等。用公式表示为：
\[ F_{\text{作用}} = -F_{\text{反作用}} \]

### 以下是作用力和反作用力的几个关键点：
- **相互性**：当一个物体对另一个物体施加力时，另一个物体也会以相同大小、相反方向的力反作用于第一个物体。
- **大小相等**：作用力和反作用力的大小总是相等的。
- **方向相反**：作用力和反作用力的方向总是相反的。

## 力的相互作用
在量子力学中，力的相互作用同样遵循上述原则。以下是一些具体的例子：

### 例子 1：电子和质子之间的电磁力
电子和质子之间的电磁力遵循牛顿第三定律。当电子受到质子的吸引力时，质子也会受到电子的相同大小、相反方向的吸引力。

### 例子 2：光子和物质之间的相互作用
光子与物质相互作用时，也会遵循作用力和反作用力的原则。例如，当光子撞击物质表面时，物质会对光子产生反作用力，导致光子的反射或吸收。
