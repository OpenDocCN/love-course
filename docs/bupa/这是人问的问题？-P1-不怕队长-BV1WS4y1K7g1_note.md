# 🐶 如何识别“舔狗”行为 - P1

## 概述
在本节课中，我们将探讨如何识别“舔狗”行为，这是一种在人际关系中常见的现象。我们将通过分析具体案例来理解这一概念。

## 什么是“舔狗”？
**舔狗**通常指的是在人际关系中过度付出、过分讨好对方，却得不到相应回报的人。这种行为往往源于对对方的过度关注和依赖。

## 识别“舔狗”行为的特征
以下是识别“舔狗”行为的几个关键特征：

### 1. 过度付出
- **公式**：舔狗行为 = 过度付出 / 期望回报
- **描述**：在关系中，舔狗会无休止地付出，包括时间、精力、物质等，但往往期望回报很少或没有。

### 2. 过分讨好
- **描述**：舔狗会不断寻找机会讨好对方，哪怕对方的要求不合理或超出自己的能力范围。

### 3. 缺乏自我价值
- **描述**：舔狗往往忽视自己的需求和感受，过分关注对方。

### 4. 情感依赖
- **描述**：舔狗对对方有强烈的情感依赖，对方的一举一动都牵动着他们的心。

## 案例分析
以下是一个关于“舔狗”行为的案例：

**案例**：你做过舔狗吗？没有，你有什么舔狗经历吗？没有，你有心仪的女孩吗？啊有，发展到什么地步了，就还是微信好友啊，昨天晚上他给我发了一个晚安的表情包，就连做梦都是舔的，啊。

### 分析
- **过度付出**：对方仅发了一个晚安的表情包，舔狗就感到非常满足。
- **过分讨好**：舔狗没有要求对方回报，反而感到幸福。
- **缺乏自我价值**：舔狗没有意识到自己的价值，过度关注对方。

## 总结
本节课中，我们一起学习了如何识别“舔狗”行为。通过分析案例，我们了解了这一行为的特征和表现。希望这些知识能帮助你更好地理解人际关系中的这种现象。
