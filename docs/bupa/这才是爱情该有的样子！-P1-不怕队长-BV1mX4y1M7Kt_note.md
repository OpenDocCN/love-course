在本节课中，我们将要学习如何从心理学的角度，初步理解情感表达与外在行为之间的关系。我们会通过一个简单的例子，分析表面行为与潜在情感可能存在的差异。

上一节我们介绍了课程的主题，本节中我们来看看一个具体的生活场景描述。

“宝贝晚安，宝贝早点睡”是一种表达关怀与道别的言语行为。
“我还要去下一场派对”则陈述了说话者接下来的行动计划。

以下是基于这个场景，我们可以进行的几点分析：

*   **行为与言语可能分离**：叮嘱他人休息的言语，与自身前往派对的行动，指向了不同的目标。
*   **情感表达的复杂性**：表面的关怀话语下，可能隐藏着个人优先的社交需求或其他情感。
*   **情境的重要性**：理解一句话的含义，需要结合说话者的**后续行动**（`action = "参加派对"`）和具体情境来综合判断。

从以上分析我们可以过渡到，在人际沟通中，仅仅听取言语往往是不够的。
