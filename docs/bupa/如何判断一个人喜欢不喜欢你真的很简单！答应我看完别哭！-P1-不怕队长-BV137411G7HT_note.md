# 课程名称：情感判断基础 🧠 - 第1课

在本节课中，我们将学习如何通过观察对方的行为模式，理性地判断一个人是否真正喜欢你。我们将摒弃主观臆测，聚焦于几个关键的行为信号。

## 概述

感情判断常让人困惑。本节将提供清晰的视角，帮助你识别核心信号，避免陷入不必要的内耗。

## 核心行为信号分析

上一节我们概述了本课目标，本节中我们来看看几个具体的、可观察的行为模式。

以下是判断对方情感态度的三个关键信号：

1.  **态度反复无常**：如果一个人将你视为备选，其态度会呈现**忽冷忽热**的模式。
2.  **让你缺乏安全感**：如果对方的情感投入不足，你会持续体验到**患得患失**的感觉。
3.  **需要你单方面祈求**：在关系中，如果你总是卑微索取，这反映了地位的不对等。第一次犯错可能是疏忽，第二次是**傻**，第三次则是**贱**。

## 核心原则与自我审视

基于以上行为信号，我们可以总结出一个根本原则：**他若爱你，怎会舍得让你难受？他若不爱你，你连呼吸都可能被视作错误。**

当你开始反复质疑“他/她到底喜不喜欢我”时，这本身就是一个重要的警示信号。它意味着你在这段关系中的感受未被妥善关照。

## 总结

本节课我们一起学习了如何通过观察“态度是否稳定”、“是否让你感到安全”以及“关系是否对等”来判断对方的情感。请记住，你的感受是重要的。你不必在不确定的关系中反复挣扎。你值得拥有一份明确、坚定且让你安心的感情。
