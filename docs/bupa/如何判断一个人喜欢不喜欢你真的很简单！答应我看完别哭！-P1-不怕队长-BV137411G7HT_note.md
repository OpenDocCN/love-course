## 判断方法
### 以下是判断方法：

1. **忽冷忽热**：如果一个人把你当备胎，他一定对你忽冷忽热的。
2. **患得患失**：如果他没那么爱你，你会感觉到患得患失。
3. **感情中的道理**：在感情中，他若爱你，怎么可能让你难受。他若不爱你，你连说句话呼吸都是错的。

## 核心概念
- **忽冷忽热**：`如果一个人忽冷忽热，那么他可能并不真心喜欢对方。`
