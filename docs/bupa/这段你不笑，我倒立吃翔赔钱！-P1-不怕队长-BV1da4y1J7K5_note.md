在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：
```python
message = "你好，世界！"
```
现在，`message` 这个“盒子”里就装好了我们的数据。

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有几个内置的基本数据类型，它们决定了数据可以做什么操作。

以下是三种最常用的数据类型：

*   **整数（int）**：代表没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数（float）**：代表带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**：代表文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `‘Python’`, `“学习编程”`。

## 如何查看和修改变量？ 🔄

我们知道了如何创建变量，接下来看看如何查看它的内容以及修改它。

查看一个变量的值非常简单，只需使用 `print()` 函数：
```python
print(message)
```
运行这行代码，屏幕上就会显示出“你好，世界！”。

修改变量的值同样使用赋值操作：
```python
message = “新的消息内容”
print(message)
```
现在，`message` 变量里的内容就被更新了。

## 一个简单的实践示例 💻

让我们把以上知识结合起来，写几行简单的代码。

以下是完整的代码示例：
```python
# 创建变量并赋值
name = “不怕队长”
age = 100
score = 99.5

# 打印这些变量
print(“姓名：”, name)
print(“年龄：”, age)
print(“分数：”, score)

# 修改变量的值
age = age + 1
print(“明年年龄：”, age)
```
运行这段代码，你将看到变量的创建、打印和修改过程。
