## 模拟情侣应对骚扰教程

### 概述
在本节课中，我们将学习如何模拟情侣关系来应对骚扰。

### 模拟情侣应对骚扰步骤

以下是模拟情侣应对骚扰的步骤：

1. **请求帮助**
   - 小姐姐小姐姐帮帮我，那个女生一直在纠缠我，你假装一下我女朋友好不好？

2. **建立亲密关系**
   - 好，我们假装亲密，这个是我女朋友，所以请你以后不要再骚扰我。

3. **确认关系**
   - 你是她女朋友吗？对呀，我是她女朋友。

4. **询问关系时长**
   - 你们在一起多久了？我们在一起两个多月了。

5. **证明关系**
   - 怎么证明你们两个是男女朋友之间的关系？这样还不够证明，我们俩是男女朋友吗？

6. **结束骚扰**
   - 走，给我走。

### 总结
本节课中我们一起学习了如何通过模拟情侣关系来应对骚扰。
