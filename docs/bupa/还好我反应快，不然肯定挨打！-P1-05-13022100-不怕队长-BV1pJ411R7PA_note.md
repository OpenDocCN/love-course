# Python基础教程 - P1

## 概述
在本节课中我们将要学习如何使用Python编程语言进行基础编程。

## Python简介
Python是一种广泛使用的高级编程语言，以其简洁的语法和强大的库支持而闻名。

## 安装Python
首先，您需要安装Python。您可以从Python的官方网站下载并安装最新版本的Python。

```bash
# 在命令行中安装Python
pip install python
```

## 第一个Python程序
以下是一个简单的Python程序，它将打印出“Hello, World!”。

```python
# 第一个Python程序
print("Hello, World!")
```

## 变量和数据类型
在Python中，变量用于存储数据。以下是一些常见的数据类型：

- **整数**：使用`int`关键字定义，例如`x = 10`。
- **浮点数**：使用`float`关键字定义，例如`y = 3.14`。
- **字符串**：使用单引号或双引号定义，例如`name = "Alice"`。

## 控制流
Python使用`if`语句进行条件判断。

```python
# 控制流
if x > 10:
    print("x大于10")
else:
    print("x不大于10")
```

## 循环
Python支持`for`和`while`循环。

```python
# 循环
for i in range(5):
    print(i)
```

## 函数
函数是组织代码的一种方式。

```python
# 函数
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
```

## 总结
本节课中我们一起学习了Python的基础知识，包括安装Python、编写第一个程序、变量和数据类型、控制流、循环以及函数。
