## 词汇解析：什么是“黑风雷”？ ⛈️

“黑风雷”是广东地区（尤其粤语使用区域）对下雨天所打雷的一种特定称呼。它并非气象学标准术语，而是富有地方色彩的口语表达。

上一节我们介绍了课程主题，本节中我们来看看这个词汇的具体构成。

“黑风雷”由三个字组成：
*   **黑**：形容天气阴沉、乌云密布的状态。
*   **风**：指伴随雷雨的大风。
*   **雷**：即指雷电现象。

这个词汇生动地描绘了雷雨交加、天色昏暗的天气场景。

以下是包含“黑风雷”一词的一段日常对话，它展示了该词汇在现实交流中的应用。

```
A: 问你个问题。你知道广东下雨天打的雷叫什么雷吗？
B: 叫什么雷？

B: “黑风雷”……这样我们加个微信。你是用大号还是小号？

A: 是的。

```

在这个对话中，说话者A通过提问引出了“黑风雷”这个词，随后对话转向了添加微信好友的日常交流。这体现了方言词汇可以自然融入日常谈话。
