# 课程名称：Python基础入门 - 第1课：变量与赋值 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的概念之一：变量与赋值。理解这个概念是编写任何程序的第一步。

## 概述

我们可以把计算机的内存想象成一个巨大的储物柜。每个储物柜都有一个唯一的地址，但地址很难记忆。为了方便，我们可以给某个储物柜贴上一个标签，这个标签就是**变量名**。而把数据放进这个储物柜并贴上标签的过程，就是**赋值**。

上一节我们介绍了编程的基本比喻，本节中我们来看看如何在Python中具体实现它。

## 什么是变量？

变量是程序中存储数据的容器。每个变量都有一个名称（变量名）和一个值（存储的数据）。在Python中，创建一个变量并为其赋值非常简单，无需预先声明类型。

其核心操作是使用等号 `=`。`=` 被称为“赋值运算符”，它的作用是将等号右边的值，赋予等号左边的变量名。

**公式：**
`变量名 = 值`

## 变量赋值实战

下面我们通过几个具体的例子来理解赋值操作。

以下是几个基本的赋值示例：

```python
# 将数字 10 赋值给名为 age 的变量
age = 10

# 将字符串“不怕队长”赋值给名为 name 的变量
name = "不怕队长"

# 将布尔值 True 赋值给名为 has_girlfriend 的变量
has_girlfriend = True
```

执行以上代码后，计算机内存中就会创建三个“储物柜”，标签分别是 `age`、`name` 和 `has_girlfriend`，里面分别存放着 `10`、`"不怕队长"` 和 `True` 这三个数据。

## 变量的使用与重新赋值

变量一旦被赋值，我们就可以通过变量名来使用它里面存储的值。并且，变量的值是可以改变的。

以下是变量使用与修改的步骤：

1.  **使用变量**：在代码中直接使用变量名，就代表使用它当前存储的值。
    ```python
    print(name)  # 这行代码会输出：不怕队长
    ```

2.  **重新赋值**：给同一个变量名赋予一个新的值，它会覆盖旧的值。
    ```python
    age = 10
    print(age)  # 输出：10
    age = 25    # 将 age 变量的值改为 25
    print(age)  # 输出：25
    ```

3.  **基于原值赋值**：可以使用变量当前的值来计算新值，再赋给它自己。
    ```python
    count = 1
    count = count + 1  # 先计算 count+1（等于2），再将结果2赋值给count
    print(count)       # 输出：2
    ```

## 给初学者的重要提示

为了让你写的代码更清晰、更容易被自己和他人理解，请遵循以下变量命名规则：

*   **名称要有意义**：尽量使用英文单词或拼音，让人一看就知道这个变量存的是什么。比如用 `student_name` 而不是 `a`。
*   **只能包含特定字符**：变量名可以由字母、数字和下划线 `_` 组成，但不能以数字开头。
*   **区分大小写**：`age`、`Age` 和 `AGE` 是三个不同的变量。
*   **避免使用Python关键字**：不要用 `print`、`if`、`for` 等Python语言本身使用的词汇作为变量名。

## 总结

本节课中我们一起学习了Python的变量与赋值。我们了解到，变量就像带标签的储物柜，用于存储程序中的数据。`=` 符号用于赋值，可以将各种类型的数据（如数字、文本）存入变量。变量值可以随时被读取和修改。遵循良好的命名习惯是写出优秀代码的基础。

掌握变量是编程之旅的第一步，接下来你就可以用它们来存储信息，并进行更复杂的计算和操作了。
