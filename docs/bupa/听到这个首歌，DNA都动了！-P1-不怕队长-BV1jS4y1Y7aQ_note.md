大脑中有一个叫做“海马体”的结构，它是长期记忆形成的关键区域。音乐作为一种强烈的感觉刺激，能够直接激活海马体及与之相连的情感中枢（如杏仁核）。

**核心公式**：`音乐信号 → 听觉皮层 → 海马体/杏仁核 → 提取关联记忆与情感`

这个过程几乎是自动和瞬间完成的，这就是为什么某段音乐能让我们立刻“回到过去”。

以下是音乐触发记忆的几个核心要素：

1.  **旋律的重复性与熟悉度**：熟悉的、重复的旋律线条最容易被大脑识别和关联。
2.  **特定的和声进行**：某些和弦走向（如经典的“卡农进行”）在大量流行音乐中被使用，形成了文化上的共同记忆。
