# 🎯万能搭讪术教程 - P1

## 概述
在本节课中，我们将学习一种简单有效的搭讪技巧，帮助你在社交场合中更加自信地与人交流。

## 万能搭讪术
以下是一种简单而有效的搭讪方法：

**步骤**：

1. **开场白**：直接而友好地打招呼，例如：“小姐姐，你好。”
2. **引入话题**：提出一个简单的问题或话题，例如：“我在做社会调查，你能帮我填个问卷吗？”
3. **赞美**：对对方进行真诚的赞美，例如：“你这个有点厉害啊。”
4. **请求帮助**：礼貌地请求对方帮助，例如：“好吧。”

## 代码示例
```python
def chat_with_girl():
    print("小姐姐，你好。")
    print("我在做社会调查，你能帮我填个问卷吗？")
    print("你这个有点厉害啊。")
    print("好吧。")

chat_with_girl()
```
