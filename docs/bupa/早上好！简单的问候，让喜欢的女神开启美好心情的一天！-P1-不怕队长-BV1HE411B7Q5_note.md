# 🌅 早上好！如何用代码开启美好心情的一天 🌅 - P1：早上好

## 概述
在本节课中，我们将学习如何通过简单的代码和问候语来开启美好的一天。

## 代码示例
以下是使用Python编写的简单问候程序：

```python
print("早上好！")
print("准备去上班了吗？")
print("记得去吃早餐哦！")
```

## 代码解释
1. **print("早上好！")**：输出“早上好！”
2. **print("准备去上班了吗？")**：输出“准备去上班了吗？”
3. **print("记得去吃早餐哦！")**：输出“记得去吃早餐哦！”

## 列表：问候语的作用
以下是问候语的一些作用：

- **提升心情**：简单的问候可以让人感到温暖和愉快。
- **开启一天**：一个美好的开始可以让人充满活力。
- **相互关心**：通过问候，可以表达对对方的关心。

## 总结
本节课中我们一起学习了如何使用简单的代码来发送问候语，希望这些问候语能帮助你开启美好的一天。
