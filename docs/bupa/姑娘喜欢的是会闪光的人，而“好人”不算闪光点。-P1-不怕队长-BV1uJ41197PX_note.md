# 如何吸引心仪的人 🌟 - 001

## 概述
在本节课中，我们将探讨如何吸引心仪的人，特别是如何展现自己的闪光点，而不是仅仅做一个贴心的好人。

## 1. 了解吸引的本质
你喜欢上一个人，不是去追求她，而是去吸引她。**公式**：吸引 = 个性魅力 + 闪光点。

## 2. 行动与吸引
当你遇到一个你喜欢的人，你会疯狂地去追求她。以下是几种常见的追求方式：
- 给她发微信
- 找话题聊天
- 询问她的日常

## 3. 甜苦心人的故事
尤其是像我这种甜苦心人，我会在大早上4点钟起来给人家姑娘煮水，冬天零下十几度排队买早餐，送早餐。**代码**：感动 = 爱心 + 动作。

## 4. 误解与原因
我把自己都感动得不行了，但有人不理解，为什么我对你好，你却不喜欢我。原因很简单，因为那些事情是情侣之间才应该做的，而她还没有答应和你在一起。

## 5. 闪光点的重要性
暖男谁都可以当，但姑娘喜欢那些有闪光点的人。以下是一些闪光点：
- 长得帅
- 会写文章
- 会唱歌
- 会跳舞
- 会赚钱
- 个子高
- 会画画

## 6. 总结
重点是让她看到你的闪光点，而不是做一个贴心的好人。这不算闪光点，你懂了吗？
