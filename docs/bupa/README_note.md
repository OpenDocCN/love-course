# 🚀 不怕队长 - 编程入门教程

## 概述

在本节课中，我们将学习如何使用不怕队长进行编程。不怕队长是一个简单易用的编程工具，适合初学者入门。

## 安装不怕队长

首先，您需要下载并安装不怕队长。您可以从[不怕队长官网](https://www.bupaqz.com)下载最新版本。

```bash
# 下载不怕队长
wget https://www.bupaqz.com/download/bupaqz-latest.tar.gz

# 解压安装包
tar -xvf bupaqz-latest.tar.gz

# 进入安装目录
cd bupaqz

# 安装依赖
./install-dependencies.sh

# 启动不怕队长
./bupaqz
```

## 基本语法

不怕队长使用类似于Python的语法。以下是几个基本语法示例：

### 变量赋值

```python
# 定义变量
a = 10
b = "Hello, World!"
```

### 输出

```python
# 输出内容
print("Hello, World!")
```

### 条件语句

```python
# 条件语句
if a > 5:
    print("a大于5")
else:
    print("a不大于5")
```

### 循环语句

```python
# 循环语句
for i in range(5):
    print(i)
```

## 实例：计算两个数的和

以下是一个计算两个数和的实例：

```python
# 定义变量
a = 10
b = 20

# 计算和
sum = a + b

# 输出结果
print("两个数的和为：", sum)
```
