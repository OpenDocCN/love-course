# 🎙️ 高情商聊天技巧教程 - P1

## 概述
在本节课中，我们将学习如何运用高情商的聊天技巧，避免在交流中引起不必要的误会或冲突。

## 一、高情商聊天话术
以下是几种高情商的聊天话术：

### 1. 遇到不知道的问题
**公式**：当对方问到一个你不知道的问题时，可以说：“这个我不太清楚，但我可以帮你了解一下。”

**示例**：好啊，楼上住的小姐姐还蛮好看的，你知道吗？不知道啊，什么时候的事啊？可是书上说不能说不知道，我马上了解一下。

### 2. 对方抱怨没有男朋友
**公式**：当对方抱怨自己没有男朋友时，可以说：“这种情况，我可以帮你想办法。”

**示例**：她总是抱怨自己没有男朋友怎么办，我也没办法呀，可是书上说这种情况，我应该说我来想办法。

### 3. 对方询问是否会出轨
**公式**：当对方询问你是否会出轨时，可以说：“这个我知道，书上说不会的事情应该说我可以学。”

**示例**：你会出轨吗，这个我知道，书上说不会的事情应该说我可以学。
