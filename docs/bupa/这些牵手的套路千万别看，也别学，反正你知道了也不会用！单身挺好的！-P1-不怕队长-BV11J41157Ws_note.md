## 删除牵手的套路
以下是几种常见的牵手套路，我们一起来了解一下，并尽量避免学习它们。

### 套路一：假装无意
**描述**：在公共场合，假装无意地触碰对方的手，然后顺势牵起。

**公式**：无意触碰 → 顺势牵手

### 套路二：借机帮忙
**描述**：在对方需要帮忙时，趁机牵起对方的手。

**公式**：帮忙 → 牵手

### 套路三：制造惊喜
**描述**：在对方不知情的情况下，突然牵起对方的手。

**公式**：惊喜 → 牵手
