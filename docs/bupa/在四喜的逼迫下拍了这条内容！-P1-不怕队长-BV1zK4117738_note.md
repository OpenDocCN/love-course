在本节课中，我们将要学习Python编程中最基础、最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要一个地方来存放它，这就是**变量**。同时，数据有多种形式，比如数字、文字等，区分这些形式的就是**数据类型**。本节将详细解释这两个概念。

## 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）用来存放数据。标签（变量名）帮助我们识别盒子里装的是什么。

在Python中，我们使用等号 `=` 来给变量赋值。其基本语法是：`变量名 = 值`。

以下是创建一个变量的例子：
```python
name = "不怕队长"
```
这行代码创建了一个名为 `name` 的变量，并将文本“不怕队长”存放进去。

## 基本数据类型 🔢

数据有不同的类型，Python中常见的基本数据类型有以下几种。了解它们有助于我们正确地使用和操作数据。

1.  **整数**
    整数是不带小数点的数字，可以是正数或负数。
    例如：`age = 25`

2.  **浮点数**
    浮点数是带有小数点的数字。
    例如：`price = 19.99`

3.  **字符串**
    字符串是由字符组成的文本，需要用单引号 `'` 或双引号 `"` 括起来。
    例如：`message = "Hello, World!"`

4.  **布尔值**
    布尔值只有两种可能：`True`（真）或 `False`（假）。它常用于逻辑判断。
    例如：`is_uploaded = True`

## 如何查看和检查类型？ 🔍

上一节我们介绍了不同的数据类型，本节中我们来看看如何在代码中查看一个变量的值和类型。

我们可以使用 `print()` 函数来输出变量的值。使用 `type()` 函数可以查看变量的数据类型。

以下是使用这两个函数的示例：
```python
name = "不怕队长"
print(name)          # 输出：不怕队长
print(type(name))    # 输出：<class 'str'>
```

## 变量的命名规则 📝

为了代码清晰且可维护，给变量起一个好名字非常重要。以下是命名变量时需要遵守的几个核心规则。

*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如，`Name` 和 `name` 是两个不同的变量）。
*   避免使用Python的保留关键字（如 `print`, `if`, `for` 等）作为变量名。
