## 搭讪步骤
以下是搭讪空姐的步骤：

1. **按服务铃**：按下头上的服务铃，引起空姐的注意。
2. **提出交换信件**：当空姐过来时，对她说：“小姐姐，我想用我的一封信换你的另一封信。”
3. **交换信件内容**：无论空姐说什么，你都可以告诉她：“我想用我的表扬信换你的微信。”
4. **成功几率高**：这种方法通常能够成功。

## 表扬信模板
以下是表扬信的模板：

```
亲爱的空姐，

您好！在这次飞行中，您的服务让我印象深刻。您的微笑、热情和专业让我感到非常舒适。我想用我的表扬信换您的微信，希望能与您保持联系。

再次感谢您的优质服务！

祝好，
[您的名字]
```
