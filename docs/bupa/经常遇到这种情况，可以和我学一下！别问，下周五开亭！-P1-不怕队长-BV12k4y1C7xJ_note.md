# 📚 教程：如何借一根烟

## 概述
在本节课中我们将要学习如何礼貌地请求借一根烟，并强调健康的重要性。

## 请求借烟的礼貌方式
以下是请求借烟时可以使用的礼貌用语：

- 能借一根烟吗？
- 能借我抽一下吗？
- 我不会抽烟，只是想借一下。

## 健康的重要性
**公式：** 健康 = 无烟 + 健康的生活方式

**代码：**
```plaintext
function 健康生活方式(无烟, 运动, 饮食)
    if 无烟 == True and 运动 == True and 饮食 == True
        return "健康"
    else
        return "不健康"
    end
end
```

## 示例对话
以下是借烟时的一个示例对话：

- **A:** 能借一根烟吗？
- **B:** 能借你抽一下吗？其实我不会抽烟，只是想借一下。
- **A:** 哇。
