上一节我们介绍了课程主题，本节中我们来看看具体的行为准则。

在公共场所使用手机、电脑等设备播放音频时，声音外放可能会影响到周围的人。为了维护公共环境的安静与和谐，佩戴耳机是一个有效的解决方案。

以下是关于公共场合使用耳机的几个核心要点：

*   在图书馆、咖啡馆、公共交通等安静或共享的公共空间，应主动佩戴耳机收听音频内容。
*   选择耳机时，注意音量不宜过大，以免损伤个人听力或产生漏音。
*   当需要与他人短暂交流时，应及时暂停播放或摘下一边耳机，以示尊重。

通过以上做法，可以体现个人对他人的尊重和对公共秩序的维护。

本节课中我们一起学习了在公共场合使用耳机的基本礼仪。核心在于认识到个人行为对公共环境的影响，并主动采取佩戴耳机等措施，以避免对他人造成不必要的打扰。这不仅是礼貌的体现，也是现代社会中一项重要的社交技能。
