# **超能力教程：如何让人忘记你的身份** - P1

## 概述
在本节课中，我们将学习一种超能力，即如何让人忘记你男朋友的身份。这是一种有趣的心理技巧，可以帮助你在社交场合中更加自如。

## 超能力介绍
**小姐姐，我有一个超能力，什么超能力，就是我能让你忘记，我是你男朋友这件事情。**

### 核心概念
```python
def forget_relationship(partner_name):
    return "你忘记了我男朋友的身份，我是" + partner_name
```

## 实施步骤
1. **创造距离**：在与对方交流时，尽量保持一定的距离，避免过于亲密的肢体接触。
2. **独立活动**：鼓励对方参与独立活动，减少两人共同度过的时光。
3. **分享兴趣**：引导对方关注你的兴趣爱好，而不是将注意力集中在你们的关系上。
