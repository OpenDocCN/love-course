# 课程01：Python基础入门 - 变量与赋值 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的概念之一：变量与赋值。理解这个概念是编写任何程序的第一步。

## 概述

我们可以把计算机的内存想象成一个巨大的储物柜仓库。每个储物柜都有一个唯一的地址，但地址本身很难记忆。为了方便，我们可以给常用的储物柜贴上标签，这个标签就是**变量名**，而储物柜里存放的东西就是**变量的值**。Python中的赋值操作，就是完成贴标签和存放物品的过程。

上一节我们介绍了变量类似于带标签的储物柜，本节中我们来看看如何具体进行赋值操作。

## 变量的赋值

在Python中，我们使用等号 `=` 来进行赋值操作。等号左边是我们想好的变量名，右边是我们想要存储的值。

其核心公式为：
**`变量名 = 值`**

以下是一个简单的代码示例：
```python
name = “Brilliant”
```
在这行代码中，我们创建了一个名为 `name` 的变量，并将字符串 `“Brilliant”` 存储在其中。之后，在程序里使用 `name`，就相当于在使用 `“Brilliant”` 这个值。

## 变量命名的规则

为了正确有效地使用变量，我们需要遵守一些命名规则。以下是主要的命名规则：

1.  **只能包含特定字符**：变量名可以由字母（a-z, A-Z）、数字（0-9）和下划线（_）组成。
2.  **不能以数字开头**：例如 `1name` 是无效的，但 `name1` 是有效的。
3.  **区分大小写**：`Name`、`NAME` 和 `name` 是三个不同的变量。
4.  **不能使用保留字**：不能使用Python语言本身已经占用的单词，如 `if`、`for`、`while` 等作为变量名。

## 使用变量的好处

现在我们知道了如何创建变量，那么为什么要使用变量呢？以下是使用变量的几个主要好处：

1.  **提高可读性**：`student_name` 比直接使用一个复杂的值更容易理解其含义。
2.  **方便复用**：一旦将值存储在变量中，就可以在代码中多次使用该变量，而无需重复书写相同的值。
3.  **易于修改**：如果需要更改某个值，只需在变量赋值的地方修改一次，所有使用该变量的地方都会自动更新。

## 总结

本节课中我们一起学习了Python中变量与赋值的基础知识。我们了解到变量是存储数据的命名容器，通过 `=` 运算符进行赋值。同时，我们掌握了变量命名的基本规则，并理解了使用变量可以让代码更清晰、更易维护。这是你编程之旅坚实的第一步，接下来我们将学习不同类别的数据可以放入这些“储物柜”中。
