## 概述
在本节课中，我们将学习如何通过三个简单的问题来测试你对心仪之人的感情。

## 教程内容

### 问题一：你喜欢不喜欢我？
**公式**：喜欢 = 是

**解释**：直接询问对方是否喜欢你，如果对方回答“是”，那么你就可以继续下一步。

### 问题二：你愿不愿意做我女朋友？
**公式**：愿意 = 是

**解释**：如果对方在第一个问题中回答“是”，那么你可以询问对方是否愿意成为你的女朋友。如果对方再次回答“是”，那么你就可以进行下一步。

### 问题三：我能不能抱你一下？
**公式**：不可以 = 可以

**解释**：这是关键的一步。如果对方在第二个问题中回答“是”，但在第三个问题中回答“不可以”，那么根据这个游戏的规则，你就可以直接抱对方。
