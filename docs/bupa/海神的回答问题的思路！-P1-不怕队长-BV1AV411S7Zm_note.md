## 思维导图
以下是回答问题的思路：

### 1. 理解问题
- **步骤**：仔细阅读问题，确保完全理解问题的含义。
- **公式**：理解 = 阅读问题 + 思考问题

### 2. 分析问题
- **步骤**：分析问题的类型和可能的原因。
- **公式**：分析 = 类型识别 + 原因分析

### 3. 构思答案
- **步骤**：根据问题的类型和原因，构思一个合适的答案。
- **公式**：答案 = 类型 + 原因 + 创意

### 4. 表达答案
- **步骤**：清晰、简洁地表达答案。
- **公式**：表达 = 清晰 + 简洁 + 准确

## 实例分析
**问题**：如果我跟你妈掉进了水里，你会先救谁啊？

**分析**：这是一个道德和情感问题，需要考虑亲情和责任。

**答案**：这个问题没有标准答案，但可以表达出对双方的爱和尊重。

**表达**：我会尽力救两个人，如果只能救一个，我会根据当时的情况和对方的意愿来决定。
