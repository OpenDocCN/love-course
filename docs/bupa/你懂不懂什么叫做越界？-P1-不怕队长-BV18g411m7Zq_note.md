## 概述
在本节课中，我们将探讨“越界”这一概念，并通过具体例子来理解其含义。

## 什么是越界？
**定义**：越界是指个人行为或言语超出了正常社交界限，侵犯了他人的个人空间或隐私。

### 核心概念
**公式**：越界 = 行为/言语 + 超出社交界限

## 例子分析
以下是关于“越界”的例子分析：

### 例子1
**原文**：你為什麼不回我消息啊，怎麼回 回什麼 你想讓我回什麼，你想讓我每天都跟你早安晚安，還是我做點什麼事情都跟你報備，我都不跟我媽報備 我還得跟你報備，我為什麼一定要回你消息啊。

**分析**：
- 这段话中，说话者对对方不回消息的行为感到困惑和不满。
- 说话者提出了对方可能的行为模式，如每天早安晚安或报备所有事情。
- 这种行为模式超出了正常朋友间的界限，属于越界。

### 例子2
**原文**：可是你的消息我都是秒回 我一直在等你啊，我逼你秒回了嗎 還是我讓你等我消息了，不是你自願的嗎 你可以不回啊，反正我也不在乎，我真的會很難過，關我屁事啊 你難過去唄，再說了 我們只是朋友啊。

**分析**：
- 这段话中，说话者表达了对对方不回消息的失望和痛苦。
- 说话者试图理解对方的行为，并强调自己并不强迫对方回消息。
- 然而，说话者的情绪表达和对方的行为模式仍然属于越界。
