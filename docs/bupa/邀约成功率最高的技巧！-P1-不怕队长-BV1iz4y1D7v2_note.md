## 射心术
射心术的核心在于直接提供选择而非询问对方是否有空。这种方法可以避免让对方感到压力，从而提高接受邀请的可能性。

### 步骤
1. **提出选择**：直接给出具体的时间或活动选项，例如“今晚一起吃饭”或“明晚一起吃饭”。
2. **避免询问**：不要询问对方是否有空，因为这可能会让对方感到不舒服或被迫做出决定。

### 代码示例
```python
def invite_success_rate():
    # 提供选择
    options = ["今晚一起吃饭", "明晚一起吃饭"]
    print("有空一起吃饭吗？")
    print("以下是我们的建议：")
    for option in options:
        print(option)
    # 获取用户选择
    choice = input("请选择一个时间：")
    print(f"好的，我们选择了{choice}。")
```
