## 歌词分析
以下是歌词的分析：

- **音乐**：歌词开头有音乐，表示这是一段歌曲。
- **自恋**：歌词中提到“这么自恋”，表示说话者认为自己在想别人，但实际上可能并不是这样。
- **湿气重**：医生说说话者最近湿气重，可能是指身体不适。
- **清热解毒**：医生建议说话者需要清热解毒，可能是指需要治疗身体不适。
