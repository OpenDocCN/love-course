## 概述
在本节课中，我们将学习如何通过一个简单的对话场景，运用基本的社交技巧来吸引他人。

## 教程内容

### 1. 主动搭讪
**公式**：主动搭讪 = 问候 + 自我介绍 + 简单话题

**代码**：
```plaintext
你，小姐姐，可以給你加個微信嗎，我叫Bridian
```

### 2. 引导话题
**公式**：引导话题 = 共同兴趣 + 询问对方

**代码**：
```plaintext
加個微信可以，但是你為什麼扛單車啊
```

### 3. 提出帮助
**公式**：提出帮助 = 表达意愿 + 提供帮助

**代码**：
```plaintext
一會送你回家吧，反正我有車，你幫我吧
```

### 4. 结束对话
**公式**：结束对话 = 简单回应 + 表达期待

**代码**：
```plaintext
還行，為何你停留不走
```

本节课中我们一起学习了如何通过简单的对话场景，运用基本的社交技巧来吸引他人。希望这些技巧能够帮助你在日常生活中更加自信地与人交流。
