# 如何让喜欢的人一直想你，喜欢上你？ - P1 - 不怕队长 - BV1kz4y1o7Pb

## 概述
在本节课中，我们将学习如何通过心理学原理，让喜欢的人经常想起你，并逐渐产生好感。

## 核心概念
**公式**：重复出现的事物会强化记忆，从而增加被记住的可能性。

## 教程内容

### 1. 选择一个共同点
**以下是选择共同点**：
- 你喝不喝星巴克，我不喝，那我去买一杯美式。
- 你喝不喝咖啡啊，我不喝，那我去买杯美式，等我一下。

**解释**：选择一个共同点，比如都喜欢某种饮品，然后通过重复出现这个共同点，来强化对方的记忆。

### 2. 重复出现
**以下是重复出现的例子**：
- 我去星巴克买杯美式，我每次都要喝星巴克的。
- 你笑一下，笑一下就甜了。

**解释**：通过重复出现这个共同点，让对方在心理上形成对你的记忆。

### 3. 强化记忆
**以下是强化记忆的例子**：
- 布瑞琳超喜欢喝星巴克的，喂，布瑞琳，你在干嘛，不知道他有没有想我。

**解释**：通过询问对方是否记得你，来强化对方的记忆。

### 4. 应用范围
**以下是应用范围的例子**：
- 不只是星巴克，任何的东西都可以。

**解释**：这个方法不仅限于星巴克，任何事物都可以用来重复出现，从而强化对方的记忆。

## 总结
本节课中我们一起学习了如何通过重复出现共同点来强化对方的记忆，从而让喜欢的人经常想起你，并逐渐产生好感。希望这个方法能帮助你实现你的目标。
