## 投石问路的基本原理
**公式**：投石问路 = 提出开放式问题 + 观察对方反应

### 以下是投石问路的一些基本步骤：

1. **提出开放式问题**
   - **示例**：你平时喜欢做什么？
   - **代码**：`question = "你平时喜欢做什么？"`

2. **观察对方反应**
   - **示例**：对方回答“我喜欢看电影”，你可以继续询问：“那最近有什么好看的电影吗？”
   - **代码**：`response = "我喜欢看电影"`；`follow_up_question = "那最近有什么好看的电影吗？"`

3. **根据反应调整话题**
   - **示例**：如果对方对电影感兴趣，你可以继续讨论电影，或者询问对方喜欢的类型。
   - **代码**：`if response == "我喜欢看电影":`
     `    continue_discussion = "那最近有什么好看的电影吗？"`

4. **提出邀请**
   - **示例**：如果对方愿意继续对话，你可以提出邀请：“要不改天我加个微信吧。”
   - **代码**：`if response is positive:`
     `    invite = "要不改天我加个微信吧。"`
