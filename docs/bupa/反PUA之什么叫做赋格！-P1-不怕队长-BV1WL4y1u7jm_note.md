# 反PUA教程 第1课：什么是赋格？ 🎭

在本节课中，我们将要学习一个在人际交往，特别是情感互动中常见的重要概念——“赋格”。我们将了解它的定义、常见形式以及如何识别它。

## 什么是赋格？ 🤔

赋格，是指在追求他人或恋爱过程中，主动为对方“制造”或“赋予”一种特定的性格或人格特质的行为。

**核心定义**：`赋格 = 为他人人为设定性格标签`

## 赋格的常见例子 📝

以下是赋格在现实生活中的具体表现。

**1. 情感关系中的赋格**
最常见的例子是，一方通过陈述过去经历来为另一方设定标准。例如，女生对男生说：“我前男友对我特别好，我想要什么他第二天就买，过节都会给我转账发红包。” 这实际上是在赋予当前对象一种“应该像前男友一样付出”的性格预期。它的潜台词是：如果你连我前男友都不如，那我为什么要和你在一起？

**2. 直接的性格设定**
另一种更直接的方式是声明自己的偏好，从而引导对方的行为。例如，你对女生说：“我不喜欢花我钱的女生。” 这句话就是在赋予对方“不物质、独立”的性格特色，并期望她朝这个方向表现。

## 如何应对赋格？ 🛡️

上一节我们介绍了赋格是什么，本节中我们来看看如果遇到他人对你进行赋格，可以如何思考与回应。

当你察觉到对方可能在对你进行赋格时，可以保持清醒，并进行理性反问。

**关键反问思路**：`如果（你描述的）那么好，为什么（当时的）关系会结束？`

例如，针对“前男友对我特别好”这个赋格，一个有效的回应是反问对方：“既然前男友对你这么好，你们为什么还会分手呢？” 这个问题有助于将对话从单方面的标准施加，拉回到对关系本质的讨论上。

## 赋格的广泛运用 🌐

赋格这个概念听起来简单，但在我们的情感生活和职场人际中，其应用非常广泛。它本质上是一种隐性的期望管理和行为引导。

掌握识别赋格的能力，可以帮助我们更清晰地理解沟通中的潜台词，保护自己的边界，避免被他人不合理的预期所绑架。未来有机会，我们可以再深入探讨更多关于赋格的高级形态与应用。当然，我们探讨它是为了反PUA和自我保护，这与PUA操纵术有本质区别。
