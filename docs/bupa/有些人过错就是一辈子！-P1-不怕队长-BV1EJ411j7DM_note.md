# 课程名称：Python基础入门 - 第1课：理解变量与赋值 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的概念之一：变量与赋值。理解这个概念是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的主题，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，这个盒子用来存放数据。在Python中，我们通过一个名字（即变量名）来访问这个“盒子”里的内容。

变量的核心作用是存储信息，以便在程序后续的部分中使用。例如，我们可以用一个叫 `score` 的变量来保存玩家的游戏分数。

## 如何给变量赋值？ ➡️

理解了变量的概念后，接下来我们需要知道如何将数据放入这个“盒子”里，这个过程称为“赋值”。

在Python中，我们使用等号 `=` 进行赋值操作。等号左边是变量名，右边是要存储的值或表达式。

其基本公式为：
```
变量名 = 值
```

例如，下面的代码将数字 `100` 赋值给变量 `score`：
```python
score = 100
```

执行这行代码后，Python解释器会在内存中创建一个值为100的数据，并将变量名 `score` 与之关联。之后，当我们使用 `score` 时，Python就知道我们指的是那个存储着的数字 `100`。

## 变量的命名规则 📝

创建变量时，为其选择一个清晰的名字非常重要。以下是Python中变量命名需要遵循的主要规则：

*   **可以包含的字符**：变量名可以由字母（a-z, A-Z）、数字（0-9）和下划线（_）组成。
*   **开头字符限制**：变量名不能以数字开头。例如，`1variable` 是无效的，但 `variable1` 是有效的。
*   **区分大小写**：Python是区分大小写的语言。这意味着 `Score`、`score` 和 `SCORE` 是三个不同的变量。
*   **避免关键字**：不能使用Python的保留关键字（如 `if`、`for`、`while`、`def` 等）作为变量名。

一个好的变量名应该能够清晰地描述其存储的数据内容，例如 `user_name`、`total_price`、`is_active`。

## 变量的使用示例 💡

现在，让我们通过一个简单的例子将以上概念串联起来。假设我们要计算一个长方形的面积。

```python
# 1. 创建变量并赋值
length = 10
width = 5

# 2. 使用变量进行计算
area = length * width

# 3. 打印结果
print("长方形的面积是：", area)
```

在这段代码中：
1.  我们创建了两个变量 `length`（长）和 `width`（宽），并分别赋值为10和5。
2.  我们利用这两个变量计算面积，并将结果赋值给新变量 `area`。这里 `length * width` 是一个表达式。
3.  最后，我们使用 `print()` 函数将结果输出到屏幕。

运行这段代码，你将在屏幕上看到：“长方形的面积是： 50”。
