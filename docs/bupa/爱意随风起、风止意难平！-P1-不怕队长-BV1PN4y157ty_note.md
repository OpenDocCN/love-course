## 爱的定义
**公式：** 爱 = 无条件付出 + 牺牲 + 幸福追求

**代码：**
```python
class Love:
    def __init__(self, sacrifice, happiness):
        self.sacrifice = sacrifice
        self.happiness = happiness

    def expression(self):
        return f"爱是{self.sacrifice}，追求{self.happiness}"
```

**分析：**
爱不仅仅是付出和幸福，它也可能包含霸占、破坏和毁灭。为了得到对方，有时会不择手段，甚至不惜让对方伤心。

## 爱的情感和后果
**列表：**
- 霸占
- 破坏
- 毁灭
- 不择手段
- 伤心

**分析：**
爱可能带来复杂的情感和后果，包括上述列出的各种情况。
