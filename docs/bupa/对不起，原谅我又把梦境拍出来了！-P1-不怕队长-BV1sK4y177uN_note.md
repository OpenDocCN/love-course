## 什么是梦境拍摄？ 💭

上一节我们介绍了课程目标，本节中我们来看看核心概念。梦境拍摄并非指用摄像机记录睡眠中的幻象，而是一种创作手法的比喻。它指的是将那些具有超现实、情感化或潜意识特征的场景与叙事，通过影像手段具象化地表达出来。其核心在于捕捉一种**氛围 = 意象 × 情感**，而非遵循严格的逻辑叙事。

## 拍摄前的核心准备 🛠️

理解了基本概念后，我们需要为实际的拍摄行动做好准备。充分的准备是成功将脑海中的梦境转化为画面的关键。

以下是进行梦境拍摄前必须完成的三个步骤：

1.  **意念捕捉**：随时记录灵感。当任何类似梦境的画面或感觉在脑中闪现时，立即用文字或草图记录下来。可以使用手机备忘录或随身笔记本。
2.  **元素解构**：分析记录下的灵感。将模糊的感觉分解为具体的视觉元素，例如：**关键物体**（如悬浮的时钟）、**色调**（如蓝紫色调）、**情绪关键词**（如“宁静的疏离感”）。
3.  **简单规划**：制定最小可行拍摄方案。无需复杂分镜，但需明确**拍摄主体**、**主要场景**和**基础光线**（自然光或单一光源）。用简短的代码表示一个基础拍摄指令可以是：`shoot(subject=“玻璃杯”, location=“窗台”, light=“黄昏逆光”)`。

## 从概念到实践：你的第一次尝试 🚀

掌握了核心概念并做好准备工作后，便可以尝试将想法付诸实践。这个过程是从内部思维走向外部呈现的关键一步。

以下是启动首次拍摄的行动清单：

*   **设备从简**：使用智能手机即可。初期重点在于表达，而非设备优劣。
*   **寻找场景**：根据你的元素解构，在身边寻找匹配的场地。一个安静的角落、一道特殊的光影都可作为起点。
*   **专注感受**：拍摄时，专注于还原你记录下的“情绪关键词”，而不是追求技术完美。
*   **即时回顾**：拍完立即查看，判断画面是否传递出了你想要的初步感觉，并进行简单调整。
