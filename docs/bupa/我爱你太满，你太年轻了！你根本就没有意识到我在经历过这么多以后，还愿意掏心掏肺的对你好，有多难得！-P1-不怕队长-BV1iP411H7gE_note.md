# 情感分析入门教程 第1课：文本情感识别基础 🧠

在本节课中，我们将学习如何对一段中文文本进行基础的情感分析。我们将以一段情感表达强烈的文本为例，拆解其核心要素，并学习如何用简单的逻辑和代码来识别其中蕴含的情感倾向。

## 概述

情感分析是自然语言处理中的一个重要领域，旨在识别和提取文本中的主观情感信息。一段文本可能包含喜悦、悲伤、愤怒等多种情感。我们的目标是理解文本所表达的主要情感色彩。

上一节我们概述了情感分析的目标，本节中我们来看看如何具体分析一段文本。

## 文本预处理

在分析情感之前，我们需要对原始文本进行预处理。这包括去除无关的语气词、整理语句结构，以便更清晰地捕捉核心含义。

原始文本包含一些强烈的语气和重复表达。预处理后，我们得到以下核心陈述：
> 我们分手吧。我爱你太满了，你太年轻了。你根本就没有意识到，我在经历过这么多以后，还愿意掏心掏肺地对你好，有多难得。我以为忘了想念，而面对谁。

## 情感关键词提取

接下来，我们从整理后的文本中提取可能指向特定情感的关键词或短语。这些词汇是判断情感倾向的重要依据。

以下是提取出的关键情感词列表：
*   **分手**：表明关系结束的行动。
*   **爱得太满**：形容付出程度极高。
*   **太年轻**：指出对方的特质。
*   **没有意识到**：表达对方的认知状态。
*   **经历过这么多**：暗示说话者的过去。
*   **掏心掏肺**：形容付出的真诚与深度。
*   **多难得**：强调价值的稀有性。
*   **忘了想念**：描述一种试图遗忘的状态。

## 情感倾向判断逻辑

基于提取的关键词，我们可以建立简单的逻辑规则来判断整体情感倾向。这可以通过一个条件判断函数来实现。

我们可以用伪代码来描述这个逻辑判断过程：
```python
def analyze_sentiment(text):
    negative_indicators = ["分手", "难过", "忘了"]
    positive_indicators = ["爱", "掏心掏肺", "对你好"]
    regret_indicators = ["太满", "太年轻", "没有意识到", "多难得"]

    if any(word in text for word in negative_indicators):
        sentiment = "负面（悲伤、遗憾）"
    elif any(word in text for word in regret_indicators):
        sentiment = "复杂（混合遗憾与珍视）"
    else:
        sentiment = "其他"

    return sentiment

# 应用于我们的文本
result = analyze_sentiment(processed_text)
print(f"情感倾向: {result}")
```
这段代码检查文本中是否包含我们预先定义的情感指示词，并据此做出判断。

## 核心情感归纳

通过以上分析，我们可以对这段文本的核心情感进行归纳。它并非单一的愤怒或悲伤，而是一种复合情感。

文本主要表达了两种交织的情感：
1.  **遗憾与失落**：由“分手”、“太年轻”、“没有意识到”等词驱动。公式可概括为：**付出 > 理解**，导致关系失衡。
2.  **珍视与不舍**：由“掏心掏肺”、“多难得”等词体现。这强调了情感付出的价值，公式为：**真诚付出 = 高情感价值**。

## 总结

本节课中我们一起学习了情感分析的基础步骤。我们从一段具体的中文文本出发，经历了**文本预处理**、**关键词提取**、**逻辑判断**到最后**情感归纳**的完整流程。我们认识到，一段文本的情感往往是复杂的，分析时需要综合考虑多种词汇和语境。核心判断逻辑可以用 `if-elif-else` 这样的条件语句来初步实现，帮助我们理解**负面情感**与**复杂情感**的构成要素。
