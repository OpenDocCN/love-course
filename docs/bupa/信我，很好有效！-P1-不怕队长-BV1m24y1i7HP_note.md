## 概述

## 变量与赋值

在Python中，变量用于存储信息。你可以把变量想象成一个贴有标签的盒子，里面可以存放数据。

创建一个变量并使用它非常简单，其核心操作是赋值，公式如下：
**`变量名 = 值`**

以下是具体步骤：
1.  使用等号（`=`）进行赋值。
2.  等号左边是你为变量起的名字。
3.  等号右边是你想存储的值。

例如，我们可以这样使用：
```python
name = "不怕队长"
message = "我们下期再见吧！"
```

## 使用 Print 函数输出

将信息显示出来是程序与用户交互的基本方式。在Python中，我们使用`print()`函数来实现这个功能。

`print()`函数的基本用法是将你希望显示的内容放在括号内。以下是其基本语法：
**`print(要输出的内容)`**

你可以直接输出文字（字符串），也可以输出变量的值。
```python
print("谢谢。")
print(name)
print(message)
```

## 一个简单的程序示例

现在，让我们将以上概念组合起来，写一个简短的程序。这个程序会存储一些信息，然后将它们打印到屏幕上。

以下是完整的代码示例：
```python
# 定义变量
greeting = "我们下期再见吧！"
response = "可以啊。"
farewell = "谢谢。"

# 打印输出
print(greeting)
print(response)
print(farewell)
```
运行这段代码，你将在屏幕上看到三行分别输出的句子。
