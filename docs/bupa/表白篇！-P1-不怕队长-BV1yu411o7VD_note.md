# 表白篇！ - P1 - 不怕队长 - BV1yu411o7VD

## 概述
在本节课中，我们将学习如何用简单而真挚的语言表达爱意。

## 我喜欢草莓
我喜欢吃草莓，草莓汁不行，草莓果冻也不行，总之不是草莓就不行，就像我喜欢你一样，长得像你不行，性格像你也不行，总之不是你就不行。

**核心概念**：
- **喜欢草莓**：`喜欢 = 爱好`
- **非草莓不可**：`非...不可 = 必须是...`

## 世界末日
如果一定要把这个世界写成一首诗的话，我一定会把我喜欢你这句话，写到诗的最后一句，因为即使那样，到了世界末日的那一天，我都会记得，我爱你，或许要不客气。

**核心概念**：
- **世界末日**：`世界末日 = 世界的终结`
- **诗的最后一句**：`最后一句 = 最重要的部分`

## 总结
本节课中我们一起学习了如何用简单而真挚的语言表达爱意。希望这些内容能够帮助你更好地表达自己的情感。
