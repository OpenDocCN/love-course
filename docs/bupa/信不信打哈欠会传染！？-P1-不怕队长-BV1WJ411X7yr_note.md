## 设备与前期准备

上一节我们介绍了课程目标，本节中我们来看看拍摄前需要做的准备工作。成功的延时摄影始于充分的准备。

以下是拍摄所需的核心设备与前期检查清单：

*   **拍摄设备**：Google Pixel 4 智能手机。
*   **稳定工具**：一个稳固的三脚架或手机支架，这是保证画面稳定不抖动的关键。
*   **电力保障**：确保手机电量充足，或连接移动电源。延时摄影耗时较长，耗电量也大。
*   **存储空间**：清理出足够的手机存储空间，以保存大量的连续照片。
*   **拍摄构思**：明确拍摄主题（如人物打哈欠的过程）和大致时长。

## 拍摄设置与参数

完成准备工作后，我们就可以进入具体的相机设置了。正确的参数是获得理想效果的基础。

以下是需要在 Google Pixel 4 相机应用中调整的关键设置步骤：

1.  **打开相机应用**，滑动到“更多”选项。
2.  **选择“延时摄影”模式**。其本质是相机以固定间隔自动拍摄一系列照片，后期合成视频。公式可以简单理解为：**延时视频 = 间隔拍摄的照片序列 + 时间压缩**。
3.  **设置手机**：将手机牢固地安装在三脚架上，并构图取景。
4.  **调整间隔时间**：根据拍摄主体的变化速度设置拍摄间隔。对于人物动作，通常 2-5 秒的间隔较为合适。代码逻辑类似于：`setInterval(takePhoto, 3000)` // 每3000毫秒（3秒）拍摄一张。
5.  **对焦与曝光**：点击屏幕对主体进行对焦和测光，并长按锁定（AE/AF 锁定），避免拍摄过程中画面明暗闪烁。

## 拍摄执行与注意事项

参数设置妥当，接下来就是启动拍摄并耐心等待了。这个过程需要一些技巧和耐心。

在拍摄过程中，请务必注意以下几点：

*   **保持绝对稳定**：拍摄开始后，切勿触碰或移动三脚架。
*   **保证拍摄时长**：延时摄影需要拍摄大量照片，一段10秒的视频可能需要拍摄数十分钟。请耐心等待。
*   **观察环境光线**：如果是在户外长时间拍摄，需注意光线变化对曝光的影响。

## 后期处理与导出

拍摄完成后，我们得到了一个照片序列。现在，我们将这些照片合成为流畅的视频。

大多数手机在延时摄影模式下会自动合成视频。你也可以使用专业软件进行后期调整，流程通常如下：

> 导入照片序列 -> 调整播放帧率（例如 24fps 或 30fps）-> 进行色彩校正 -> 导出最终视频。
