## 🔬 核心原理：流速与压强的关系

上一节我们概述了课程主题，本节中我们来看看其背后的科学原理。这个现象主要与**伯努利原理**有关，该原理描述了流体在流动过程中，流速与压强的关系。

其核心公式可以表述为：

**P + 1/2ρv² + ρgh = 常数**

其中：
*   **P** 代表流体的压强。
*   **ρ** 代表流体的密度。
*   **v** 代表流体的流速。
*   **g** 代表重力加速度。
*   **h** 代表高度。

对于一个水平流动的简化情况，公式可以简化为：流速**v**越大，流体产生的压强**P**就越小。

## 🧪 现象解释：为什么一根吸管更“甜”？

理解了伯努利原理后，我们将其应用到喝可乐的场景中。当我们用吸管喝饮料时，吸管内的空气流速变化会导致压强变化。

以下是关键点分析：

用一根吸管时，你口腔形成的低压区集中，能有效地将可乐通过吸管“吸”上来，可乐流速稳定，口感集中。

使用两根吸管时，空气流通的横截面积增大。根据连续性方程，在相同吸力下，每根吸管内的气流速度会降低。

代码可以模拟这种关系：
```python
# 简化模型：假设总吸力恒定
total_suction = 1.0
num_straws = 2
suction_per_straw = total_suction / num_straws
print(f"每根吸管分到的‘吸力’：{suction_per_straw}")
# 输出：每根吸管分到的‘吸力’：0.5
```

流速降低导致吸管内的压强下降不够显著，因此驱动可乐上升的“力量”被分散了。这可能导致可乐上升速度变慢、不流畅，甚至需要更费力地吸，从而影响了饮用时味蕾对甜味的集中感知，感觉“不够甜”。
