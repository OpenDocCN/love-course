# Python基础教程 - 1.1 数据类型

## 概述
在本节课中，我们将学习Python中的基本数据类型，包括数字、字符串和布尔值。

## 数据类型
以下是Python中常见的数据类型：

### 数字
数字用于表示数值。Python支持整数（int）、浮点数（float）和复数（complex）。

**公式**：
- 整数：`int(a)`，例如 `int(5)`
- 浮点数：`float(a)`，例如 `float(3.14)`
- 复数：`complex(a, b)`，例如 `complex(2, 3)`

### 字符串
字符串用于表示文本。在Python中，字符串被引号包围，可以是单引号、双引号或三引号。

**代码**：
```python
name = "Alice"
print(name)
```

### 布尔值
布尔值表示真（True）或假（False）。

**代码**：
```python
is_valid = True
print(is_valid)
```
