2. **情感表达**：表达自己的真实感受，例如：“你笑什么，我问你，假如未来你有小孩的话，你会觉得你是一个什么样的妈妈？”

4. **情感深化**：通过对话深化情感，例如：“渣男，套路是我学的，但是撩你们，是真心的。”
