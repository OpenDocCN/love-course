## 应对“老渣女”的策略 🦊

上一节我们介绍了私密空间过渡的基本概念，本节中我们来看看一种特殊情况的应对方法：遇到看似开放、实则善于操控关系的“老渣女”。这类女生行为开放，但在关系升级的关键时刻会突然刹车。

6.  **达成无需负责的关系**：通过此方法建立的关系，往往能形成一种彼此心知肚明、无需负责的“地下情人”模式。这在某些社交圈中并不少见。

除了上述情况，女生常见的“废物测试”之一便是在关系升温时突然“拒绝”。例如，牵手、拥抱后，在接吻前突然推开你。

此时的核心心态是：
**`if (遭遇拒绝) { 心态 = “无所谓”; 行为 = “退回安全阶段，等待下次机会”; }`**

千万不要因此怀疑自己，更不要试图通过“对她更好”来弥补。一旦你产生弥补心态，就容易沦为“舔狗”。正确的做法是：不追问，不解释，不受影响，退回到让她舒适的距离，像没事发生一样继续互动。

2.  **不无条件答应，并顺势反撩**：切勿直接、无条件地满足要求。而是以此为契机，提出你的要求或进行调侃，将命令转化为互动。

核心原则是：**在互动中要求回报，设立交换条件，避免成为单方面付出的工具人。**
