# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如一个数字或一段文字）。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是赋值语句：
`变量名 = 值`

例如：
```python
name = "校长"
age = 25
```

这里，我们创建了两个“盒子”。一个名叫`name`的盒子里存放了文字“校长”，另一个名叫`age`的盒子里存放了数字`25`。

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有几个内置的基本数据类型，它们决定了数据可以做什么操作。

以下是三种最常用的基本数据类型：

1.  **整数（int）**
    代表没有小数部分的数字，可以是正数、负数或零。
    例如：`10`, `-5`, `0`

2.  **浮点数（float）**
    代表带有小数点的数字。
    例如：`3.14`, `2.0`, `-0.001`

3.  **字符串（str）**
    代表一段文本。在Python中，文本必须用单引号(`‘`)或双引号(`“`)括起来。
    例如：`“Hello”`, `‘Python’`, `“123”`（注意：这里的123是文本，不是数字）

## 如何查看和检查类型？ 🔍

在编程时，我们有时需要知道一个变量里存放的是什么类型的数据。Python提供了一个非常方便的函数`type()`来查看。

其使用格式为：
`type(变量名或值)`

例如，继续使用我们之前定义的变量：
```python
print(type(name))   # 输出：<class ‘str‘>
print(type(age))    # 输出：<class ‘int‘>
print(type(3.14))   # 输出：<class ‘float‘>
```
`print()`函数的作用是把结果打印到屏幕上，这样我们就能看到`type()`函数的检查结果了。

## 一个简单的实践示例 💻

现在，让我们把以上知识结合起来，写一个简单的程序。

```python
# 定义几个变量
student_name = “小明”
student_score = 95.5
years_of_study = 3

# 打印这些变量
print(“学生姓名：”, student_name)
print(“考试分数：”, student_score)
print(“学习年限：”, years_of_study)

# 检查它们的类型
print(“姓名的类型是：”, type(student_name))
print(“分数的类型是：”, type(student_score))
print(“年限的类型是：”, type(years_of_study))
```

运行这段代码，你会在屏幕上看到变量的值和它们对应的数据类型。
