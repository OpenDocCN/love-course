# 2023校长高情商VIP课程：补丁14 - 逆向思维是关键 🔑

在本节课中，我们将要学习人际交往中一个核心概念：逆向思维。我们将通过对比“顺向思维”与“逆向思维”的具体表现，帮助你理解并掌握跳出对方框架、坚持自我主张的沟通技巧。

## 理解顺向思维

要清楚逆向思维的概念，首先需要点清楚什么叫顺向思维。

顺向思维是指，在互动中，我们进入对方的框架去满足其要求，并为对方说的话做出解释。

以下是顺向思维的例子：

*   **场景一：女生说“你乘人之危吗？”**
    *   **顺向思维回应**：“不好意思，你想多了，我不是。” 或者 “你可能想多了，我的意思是……”，然后开始向女生解释。

*   **场景二：女生问“你是不是喜欢我？”**
    *   **顺向思维回应**：“可能是吧，但我不是你想象的那种，我没有明确要求，只是觉得你可爱，关心一下你，你别多想。”

上一节我们介绍了顺向思维的表现，本节中我们来看看什么是逆向思维，以及为何它是关键。

## 掌握逆向思维

逆向思维是我们整个互动过程中的核心内容。我们使用的诸如打压、高傲、调侃等方法，乃至最后的关系推进（TD）环节，都是围绕逆向思维展开的。

逆向思维是指，不顺从对方表面的言语框架，而是坚持自己的想法和节奏来操作。

以下是逆向思维在不同场景中的应用举例：

**1. 关系推进（TD）场景**
*   **女生说**：“你把我带回房间，难道就是这样吗？你以为我是这样的女人吗？”
*   **逆向思维回应**：“你看错人了，而且大错特错，我比你想象的厉害得多。” 然后继续行动。
*   **若女生说**：“我要叫了！”
*   **逆向思维回应**：“你叫吧，你越叫我越兴奋。”
*   **若事后女生问**：“我们到底是什么关系？”
*   **逆向思维回应**：“什么关系？进去磨合一下我再判断。爽了就是高潮的关系，没爽就下次继续。”

**2. 挽回场景**
*   **女生说**：“我们分手吧，不合适。”
*   **顺向思维**：“哪里不合适我改，我错了，再给我一次机会。”（这常导致假性复合）
*   **逆向思维**：看着对方，抱住对方，然后说：“其实我想了很久。我们矛盾很多，可我恶习难改。你付出比我多，我始终愧疚。夜里我常想，可能不只是性格不合，是我配不上你，我的很多行为让你失望了。” （通过表达脆弱和自我贬低，实则传递“我已尽力但仍无法满足你”的信号，让对方看到希望但掌握主动权）

**3. 邀约场景**
*   **前提**：吸引力已建立。
*   **你说**：“周末过来一起参加个活动。”
*   **女生问**：“为什么？” 或 “什么活动？”
*   **逆向思维回应**：“别管为什么。我一个外地朋友来，萨克斯吹得超棒，我给他吉他伴奏。你来了可别流鼻血哦。朋友多，加你一个名额，叫声爸爸我说不定答应你。”
*   **顺向思维回应**：“就是一个朋友的活动，我们想让你来，赏个脸吧。”

**4. 肢体进挪（如牵手）场景**
*   **场景**：你尝试牵女生的手，她躲开并质问：“你干嘛？我们还不熟，我不喜欢这样。”
*   **顺向思维**：道歉并把手收回。
*   **逆向思维**：跳出“牵手=不尊重”的框架，建立自己的逻辑。可以回应：“我一直在想，你到底是不是个好女孩。现在我心里有答案了。” 然后更坚定地牵上去。
    *   **核心逻辑**：我牵手是为了测试你是否是好女孩。你反抗，说明你可能是好女孩，那我更应该坚持我的想法（牵手）。这对女生是逆向的，对我们自己是顺向的。

我们所有的回答和做法，都应坚持我们本身的想法，同时避免陷入对方的理论圈套。例如，对方设下的圈套是“你牵手=你是流氓”。我们的逆向思维是建立新逻辑：“我牵手=测试你的品质”。

## 课程总结

本节课中，我们一起学习了人际沟通中的两种思维模式：
1.  **顺向思维**：进入对方框架，满足对方要求并做出解释，往往陷入被动。
2.  **逆向思维**：跳出对方设定的言语框架，坚持自己的主张和节奏，通过建立新的逻辑来引导互动，掌握主动权。

逆向思维是化解大多数互动难题的关键。只要掌握了这个方法，诸如对方询问你的身高、外貌、经济状况，或进行各种测试时，你都能从容应对。我们课程中提到的各种技巧，其内核大多源于逆向思维的灵活运用。
