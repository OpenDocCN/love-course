# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

变量可以理解为一个贴有标签的盒子，用于在计算机内存中存储信息。你给这个盒子起一个名字（变量名），然后把数据（值）放进去。之后，你就可以通过这个名字来使用盒子里的数据。

在Python中，给变量赋值非常简单，使用等号 `=` 即可。等号左边是变量名，右边是你想存储的值。

**核心公式：**
`变量名 = 值`

例如：
```python
name = "Sunnie"
age = 25
```
这里，我们创建了两个“盒子”：一个名叫 `name`，里面存放了字符串“Sunnie”；另一个名叫 `age`，里面存放了数字25。

上一节我们介绍了变量的基本概念，本节中我们来看看Python中常见的数据类型。

## 基本数据类型 🔢

数据类型决定了变量可以存储什么种类的数据，以及能对这些数据做什么操作。Python有几个内置的基本数据类型。

以下是几种最常用的数据类型：

*   **整数 (int)**： 表示没有小数部分的数字，可以是正数、负数或零。
    *   例如：`10`, `-5`, `0`

*   **浮点数 (float)**： 表示带有小数点的数字。
    *   例如：`3.14`, `2.0`, `-0.001`

*   **字符串 (str)**： 表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   例如：`"Hello"`, `‘Python’`, `“123”`（这里的123是文本，不是数字）

*   **布尔值 (bool)**： 表示逻辑判断的结果，只有两个值：`True`（真）和 `False`（假）。
    *   例如：`5 > 3` 的结果是 `True`，`1 == 2` 的结果是 `False`。

理解了数据类型后，你可能会问，我们怎么知道一个变量里存的是什么类型呢？接下来我们就来解决这个问题。

## 使用 `type()` 函数查看类型 🔍

Python提供了一个内置函数 `type()`，可以让你查看任何变量或值的数据类型。

**核心代码：**
```python
print(type(变量名或值))
```

让我们用之前的例子来实践一下：
```python
name = "Sunnie"
age = 25
height = 1.75
is_student = True

print(type(name))      # 输出：<class 'str'>
print(type(age))       # 输出：<class 'int'>
print(type(height))    # 输出：<class 'float'>
print(type(is_student))# 输出：<class 'bool'>
```
`print()` 函数的作用是将结果打印到屏幕上，这样我们就能看到 `type()` 函数的返回结果了。

## 变量的命名规则与规范 🏷️

为了写出清晰、易读的代码，给变量起一个好名字非常重要。这需要遵循一些规则和约定。

以下是变量命名的主要规则和规范：

*   **规则（必须遵守）**：
    *   变量名只能包含字母、数字和下划线 `_`。
    *   变量名不能以数字开头。
    *   变量名不能是Python的关键字（如 `if`, `for`, `while` 等）。

*   **规范（建议遵守）**：
    *   使用描述性的名字，让人一眼就能看出这个变量是做什么的（例如用 `user_name` 而不是 `a`）。
    *   通常使用小写字母，多个单词之间用下划线 `_` 连接（例如：`my_variable_name`）。这种风格称为“蛇形命名法”。

例如，`my_age` 是一个好名字，而 `2name` 或 `my-age` 则是无效的。

## 总结 📝

本节课中我们一起学习了Python编程的基石：
1.  **变量**是存储数据的“带名字的盒子”，通过 `变量名 = 值` 来赋值。
2.  **数据类型**定义了数据的种类，我们重点介绍了四种基本类型：整数 (`int`)、浮点数 (`float`)、字符串 (`str`) 和布尔值 (`bool`)。
3.  使用 **`type()` 函数**可以查看变量或值的数据类型。
4.  给变量命名需要遵守一定的**规则与规范**，这有助于写出高质量的代码。

掌握这些概念后，你已经可以开始用Python存储和识别不同类型的数据了。在下一课中，我们将学习如何让这些数据“动”起来，进行基本的运算和操作。
