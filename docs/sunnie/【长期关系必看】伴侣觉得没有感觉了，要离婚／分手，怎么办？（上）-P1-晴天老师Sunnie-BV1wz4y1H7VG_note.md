# 课程一：当伴侣感觉“没爱了”时，如何挽救关系？💔

在本节课中，我们将要学习当长期关系陷入“没有感觉”的困境时，如何理解和应对。许多伴侣分手或离婚，并非因为激烈的怨恨，而是因为感受不到爱意。我们将探讨这种现象背后的原因，并提供初步的解决思路。

---

一段关系变得“死寂”是常见的危机。上周一位女性朋友咨询，她的婚姻状态是双方早出晚归，共进晚餐时也沉默以对。

做完家务、辅导孩子作业后，一天就结束了。婚姻中充满了倦怠感，却感受不到爱。我问她当初为何选择对方。妻子说，两人在大学聚会相识，男方是活动发起人。

他安静地在一边规划行程，他的沉稳和领导力吸引了她。但结婚生子后，妻子不再是当初那个仰慕成熟的小女孩。她更需要有人能为无趣的生活注入活力。经过了解，我发现她的丈夫也有类似不满。

丈夫原本很喜欢妻子崇拜自己的样子。但现在的妻子只关心家庭和孩子。曾经吸引彼此的特质逐渐消退，双方的喜好和追求也发生了变化。关系缺乏培育的土壤。

爱情的花朵即使曾经再茂盛，也将凋谢。这样的离婚理由并不只发生在他们身上。我整理过婚姻咨询案例，发现只有四分之一是因外遇，四分之一是因各种分歧，包括金钱、相处和社交矛盾等。

最后剩下的一大类，都可以归结为“没有爱了，没有感情了”。也就是说，当问起为何不得不离婚时，多半会得到类似回答：“我们之间变得没有感觉了”。这个过程很像衰老，明知道目的地是死亡，人却无能为力。

我很能理解这种感受。很多人受不了，才会在缘分彻底耗尽前选择离婚。难道婚姻真的是爱情的坟墓吗？这里存在一个悖论：我们看不得激情消逝，这本身恰恰证明我们依然渴望热情和爱。

既然双方都希望回到彼此爱着的感受，都渴望付出爱和得到爱，为什么还会有分歧呢？今天这期视频就是告诉你，我们要怎样去挽救看似没有感觉的婚姻。

---

## 第一节：拓宽对“爱”的认知

第一个导致我们觉得“没有感觉”的原因，是我们对爱的认识太狭隘了。说一个最常见的矛盾：妻子对丈夫说“我觉得你不爱我”。丈夫可能会回答：“我当然爱你，你要缺什么我立马去买”。

妻子会反驳：“如果你爱我，怎么结婚后有空都不多陪陪我？”丈夫觉得自己一直在表达爱，对关系也尽心尽力。妻子却一点都没感受到，因为她在乎的是日常生活中，对方有多需要自己、能多陪伴自己。

两个人的爱都没有传递给对方，导致需求错位。其实，我们接受爱的层次是不同的。很多人都以为世界上只有一种爱，就是融合期的迷恋：和对方待在一起就感到心动，听他说话都觉得开心。

所以你想见对方，想跟他一直待在一起。大多数人觉得这才是真爱。但这只是第一层的爱：**迷恋**。这是短期碰撞出的火花，你们爱彼此的举动都由情绪驱使。

第二层的爱叫做**依恋**。你把对方叫起来吃早饭，两个人靠在一起看电视，觉得有他陪伴就很温暖。两个人愿意为彼此安排专门的时间，共同努力去做一些事情。这时候的爱建立在情感基础上。

更深层次的爱，即第三层的爱，叫做**关爱**。这意味着伴侣的需要已经变成了自己的责任。我们为对方做的任何事情，都是在为自己做。举个例子：当他添置家具时，会考虑到你洗衣服的辛苦，所以打算购买洗衣机，把这事完全当作自己的事来处理。

我们很多人以为，只有痴迷和狂恋才是真爱。然后会拿自己认为的爱去要求对方，却忘记了，其实很多都是爱，只是爱在不同阶段的不同形式而已。当我们只专注于其中一种形式时，我们对爱的感受就会变得狭隘。

如果习惯关爱的丈夫，遇到了需要依恋的妻子——一个在第三层，一个在第二层，就像前面举的例子一样——丈夫已经视妻子和自己为一体，他就会觉得妻子依恋的需求有些奇怪。他认为自己是爱妻子的，才会把她当成自己人。

同样的，妻子也会不满：“你这么理所当然地对待我，是不是已经没爱了？”但这是个误解。所以，如果我们感受不到爱，要先考虑是真的没有爱了，还是你们认识不到彼此的爱。在我们认知有限的时候，我们是盲目的。

这也会让爱白白流失掉。《爱的五种语言》一书中，把爱的形式当成沟通的语言，分成了五种：
*   肯定的言辞
*   精心的时刻（高质量的陪伴）
*   接受礼物
*   服务的行动
*   身体的接触

你看，爱有这么多种形式。所以第一点，我们先要认识爱、发现爱。

## 第二节：如何发现并沟通彼此的爱

上一节我们介绍了爱的不同形式，本节中我们来看看如何具体地发现并沟通彼此的爱。

我这里列了一个问答表，你可以自己对照一下。哪些话完全说中了你的内心需求，那你就是喜欢哪一类的“爱的语言”。我们先要认清自己感受爱的形式。

另外，我们发现爱最简单的方式，就是要和对方沟通。我建议你们可以找个时间坐下来，拿出纸和笔，玩一个问答游戏。

以下是具体步骤：

1.  你先写下这个题目：**“我喜欢你怎么对我？”**。双方各自写下这个问题的答案。
2.  完成之后，互相交换纸上的回答。
3.  一起阅读对方的回答。要尽量避免评判对方，不要说出“你和我不一样”之类的话，而是要试着理解对方的感受、需求以及他的喜欢之处。
4.  接下来，我们再写下第二个问题：**“我爱你，所以我做了……”**的回答。
5.  同样的，交换答案并阅读。

这时候你可能就会发现，你们写的很多条都和对方期待的不一样。因为我们是按照自己的理解去爱别人的，而他又并非你想象中那样不关注你，所以依然能写下很多自己爱你的证明。

当你了解对方是怎么爱你的，既可以解答“他不爱你”这种情绪上的误解，也能够发现他对你的爱。同时也是告诉彼此：你们表达的爱与各自想要的爱，是有一定差别的。两个人都要去学习对方喜欢的爱的形式。

让我们的爱的沟通变得更加顺利。这两张纸的回答，就是你们接下来生活中“爱的参考”了，也是对方亲自列出来的。所以可以贴到比较醒目的地方，比如冰箱上或门上面，以供参考。

其实，爱本身就是你们需要学习和训练的艺术。

## 本节总结

本节课中，我们一起学习了当伴侣关系陷入“无感”危机时的初步应对方法。我们首先探讨了“无感”的普遍性及其背后的情感需求悖论。然后，我们重点分析了第一个核心原因：**对“爱”的认知过于狭隘**。我们通过**迷恋、依恋、关爱**三层模型和**爱的五种语言**理论，理解了爱的多种形式。最后，我们通过一个具体的**问答沟通练习**，提供了发现并校准彼此“爱的语言”的实用方法，为改善关系迈出第一步。
