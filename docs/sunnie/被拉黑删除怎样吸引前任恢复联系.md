# 被拉黑删除怎样吸引前任恢复联系
## 课程编号：P1

在本节课中，我们将学习当前任拉黑或删除你后，如何通过一系列步骤重新建立联系，并引导关系走向复合。被拉黑删除是挽回中常见的挑战，但通过正确的方法和心态调整，你完全有可能扭转局面。

---

被拉黑删除的人通常认为，前任已决定划清界限，两人再无可能。前任在心理上主动回避，物理上切断联系，这看似是挽回的双重阻碍，容易让人产生放弃的想法。但你需要先了解一个事实。

分手后拉黑删除是很常见的情况。了解到这一点，你就不会过度悲观。事实上，分手后仍能保持正常联系的情侣才是少数。我处理最多的案例正是拉黑删除的情况，并由此形成了一套完整的应对流程。你可以按照以下步骤操作。

很快就能和前任重新恢复联系。

## 第一步：稳定心态，保留联系渠道 🔗

上一节我们提到了被拉黑删除后的普遍心态，本节中我们来看看被断联后首先要做什么。

当你发现自己被断联，不要一气之下也拉黑删除对方。这个行为很可能让你错过挽回时机。大部分时候，断联是前任的单方面行为。他不想留着你，或者说不想看到你。

前任独自做了这个决定。他给自己留了后路，不明说断联，也不解释其含义。如果他未来想加回你，依然可以悄悄操作，因为很多聊天软件的单方面删除和添加不会留下痕迹。我见过太多案例。

有人上午拉黑，下午气消后悔，又把对方加回来。但如果你也删除了他，当前任想加回你时，发现自己已不是你的好友，会产生两种想法：第一，觉得自己好不容易跨过心理障碍的努力白费了；第二，双方心知肚明。

彼此造成了伤害。前任会认为你是在报复他的拉黑删除，那他再联系你，就显得很不明智。因此，许多人的分手问题本不严重，此时你更要给前任一个台阶下。他闹情绪，你保持心态稳定，让感情实现“软着陆”。

当前任加回你时发现，你的态度依然良好，双方交流没有因互删而产生的尴尬，很多人在这一步耐心坚持就能够复合。如果前任在拉黑删除你之前，就告知要断联，甚至要求你也拉黑删除他，那你不要傻傻照做。

你要意识到前任这样断联的目的。他想让分手更彻底，但你并不想分手。你的首要目标是保留联系渠道。你无法影响他的判断，但总能控制对自己有利的行为。他拉黑删除是他的事。

但你要保留他的联系方式，未来才不会因没有联系方式而发愁。并且在挽回后期，你可以表明自己的心意一直未变，这个行为其实很加分。

## 第二步：尝试初步沟通，根据类型施策 💬

上一节我们讨论了保留联系渠道的重要性，本节中我们来看看如何尝试初步沟通，重新加回好友。

刚被断联时，能够尽快加回好友是最好的。前任对你未必抱有极大敌意，情绪往往来得快去得也快。如果你们只是通过常见聊天方式（如微信、QQ）被拉黑删除。

这种拉黑删除，往往可能是因为你在这些平台上说了过激的话、纠缠对方，或哀求挽回等行为，违背了前任意愿，让他感到不快。但你们之间可能并没有那么多深层矛盾。删除拉黑后，前任可能会觉得对你太过分，产生些许愧疚。

所以你可以先去联系一下，尽量大事化小，因为你们很可能是假性分手。不要害怕主动挽回。这里的沟通技巧很重要，可以根据前任的类型而定。

以下是两种沟通策略：

*   **针对偏理智型前任**：从理性角度沟通。你可以说：“我想跟你道个歉，之前对你死缠烂打，是我太冲动了。我尊重并接受你分手的决定。过去的事就让它过去吧，我会记得你的忠告，好好改正我的毛病。还是希望能和你加回好友。”
*   **针对偏感性型前任**：照顾对方感受。你可以说：“分手了你肯定很难受，我还纠缠你让你更难受了，我想跟你道个歉。我现在明白了，分手是既定事实，但我想即使我们做不成情侣，也不至于老死不相往来。可以加回好友吗？以后有什么事也好联系。”

内容不宜过长，否则容易踩雷。男性和女性分手后的心理状态变化是不同的。公式可以概括为：
*   **男性心理变化阶段**：`不满 -> 反思 -> 怀念`
*   **女性心理变化阶段**：`不适应/后悔 -> 怀念 -> 想开`

因此，**男挽女一定要趁早**，而**女挽男**的话，建议可以先给男生几天自由时间。他可能会一心扑到工作、学习或游戏上，这是转移注意力的方式。你最好在他疯狂打了几天游戏后，感到空虚的时候，再去发送上述信息。

当你把消息发给他后，就不要主动联系他了。如果对方被打动，会加回联系方式并主动找你。**事不过三**，如果前任无动于衷，没有拉回你的意愿，说明他尚未卸下心理防备，也不太相信你的话。此时你就不应再做此类尝试，因为这很可能意味着真心分手。

另外还有一种情况，就是对方全网拉黑了你，你可能只剩一个手机号，无法承担贸然联系的风险。那我建议你直接转到下一步。

## 第三步：着手铺垫，等待并创造时机 ⏳

上一节我们探讨了初步沟通的方法，但如果沟通无效或风险过高，本节中我们将学习如何为挽回做长期铺垫。

你要为挽回做准备工作，同时等待下一个适宜联系前任的时机。很多人觉得几周都挽回不了，心态就先崩溃了。但实际上，挽回经常面临这样的情况。我有一个学员，只剩抖音没被拉黑。

第一个月他发了一些积极生活的视频，前任没有任何动静。结果第二个月时，前任突然给他上个月一条弹钢琴的视频点了赞。我让学员赶紧再发一条类似的，结果对方又点赞了。所以，挽回所做的铺垫经常具有延迟性，效果可能突然出现。

为什么呢？从心理学角度来说，爱情并非一个持续线性变化的过程，而是一个阶段一个阶段的。刚分手时的阶段，我们称为 **“失恋期”** 。前任可能忙于伤感或愤怒。过了一个月，进入另一个阶段，称为 **“缅怀期”** ，可能持续半年之久。

这段时间他可能会时不时想起你，这对我们挽回非常有利。他想你，就会想知道你现在过得怎么样。他有好奇心，我们就更好办。虽然这种想念不至于让你们立刻复合，但他起了心思，我们在铺垫时就要提前做好准备。

多制造一些能让他把这份心思延续下去的机会。如果等到他想你时你再准备，可能就太迟了。所以，在你被迫断联的这几个月，一定要着手解决问题。前期不要把过多关注点放在对方身上，要先放在自己身上。

修复一段感情，首先你要尽人事。以下是你可以着手提升的方面：

*   **理解异性思维**
*   **学习情绪管理**
*   **提升沟通表达技巧**
*   **寻找平衡感情高低位的相处模式**
*   **进行修复信任感的学习**

这些都是你可以通过个人练习在短时间内提升的。还有一些挽回者可能要面对更长期的矛盾，例如：

*   处理与父母的关系
*   提升自己的学历或解决工作问题
*   解决双方异地问题

对于这些，你需要先找到一个可行的解决方案，按计划推进即可。没有人指望你当下必须立刻解决所有问题，但你要拿出愿意给出满分答案的态度。前任总会有渠道看到你的努力。

如果你已经做到50分，还没到100分，可能在前任那边差不多就可以过关了。为什么说可以先做到50分？因为解决长期问题需要过程，难免需要时间，例如调动工作或花一年时间准备考试。如果你等到那时再去挽回，前任可能早已没有感觉了。

所以后期，你既要诚心解决问题，又要关注前任动态，善于抓住挽回时机。这个阶段你还可以做的是：**找共同好友从侧面打破前任对你的成见**。

对方回避你，最难突破的就是他的心理障碍。他觉得你不适合在一起，这可能是一种成见。你要做的就是撕掉这个标签，但不能由你亲自来撕。因为对方主观上会防备你，会负面解读你的行为。

例如，你发练习做菜的视频，他可能会理解成你在假装热爱生活、作秀。他对你的评价体系已完全随着成见产生偏差。这时，你需要引入一个对方能够认可的新的评价体系。

所以我们可以通过共同好友作为联系的桥梁，最好是和对方同性的好友。让这位好友在和他聊天时，不经意间透露你的改变。例如，还是拿做菜举例，好友可以在聚会聊天时说：“上周我们去谁谁家玩，他竟然带了一袋自己做的牛轧糖出来，还挺好吃的。”

点到即可。如果说得太多或态度太好，反而容易引起对方的怀疑和反感。共同好友是一种很好的互联途径，但他不应过多插手你们之间的感情，所以实际操作时一定要注意尺度。

此外，我们还可以通过刷存在感来诱导对方好奇，稍微接触一下前任。方法不限，例如：

*   **制造偶遇**：如果你知道对方散步的路径，可以多在这条路上走走，装作不经意碰到，然后对他笑笑或打个招呼就走过去。
*   **网上购物填错地址**：但你买的东西一定要明显不是前任会用到的。当作无意寄错，快递被签收后第二天，你再联系他说：“没想到寄到你那边去了，方便的话寄给我吗？”然后付邮费，顺便聊几句无关情感的话题。如果对方愿意让你去拿或带给你，就更好了。

但是注意，千万不要用同一种方式反复刷存在感。一般这三招过后，前任甚至可能会主动加回你，再看看你的朋友圈。但如果他还是毫无波动，我们也不能干等对方给机会。

最长最长半年，一定要主动尝试联系对方。所以进入第四步。

## 第四步：直接联系，奠定复联基础 🎯

上一节我们学习了如何为挽回做长期铺垫和创造机会，本节中我们来看看如果铺垫到位，如何直接联系前任，重新加回好友。

第四步叫做“打直球”。如果你们有工作上的联系就最好了，可以直接加了说事情。如果没有，那你可以说有事想询问他，或者说从朋友口中听说他遇到了什么麻烦。

再没有理由的话，那你就直接问问对方过得好不好，表明你已经释怀以前的感情，双方还可以做朋友吗，麻烦他加回来。这个时候，之前挽回铺垫的作用就显现出来了。如果他没有新伴侣，一般不会拒绝一个优秀的潜在择偶对象。

不过，加回好友只是第一步，**加上不被删才是我们的重点**。所以不要着急跟对方倾诉你这几个月怎么过的、有多想他，一定要克制。我前面的视频也讲过“发什么消息给前任才能聊得好”，讲的就是如何聊天，让前任愿意看、愿意回、愿意把你留下来。

所以今天只简单提一个重点：人性的秘密是，**会对和自己相关的，以及他当下在乎的东西更上心**。所以你跟他聊的理由，一定要是和他相关的。具体可以查看那期视频，细节很多。

我们可以这样理解：作为挽回者，不管你多优秀，只要前任不想看，他就无法看见，也就无法正确评估你的价值。所以我们要做的，就是让前任重新意识到自己忽略了这一部分，让他认可你有留在他社交圈的价值。方法前面都说了，如果你想挽回，可以去试试。

而且，被拉黑删除作为挽回中最难解决的部分，如果你能顺利走过去，后面如何复联、如何重新培养感觉，也就相对容易了。
