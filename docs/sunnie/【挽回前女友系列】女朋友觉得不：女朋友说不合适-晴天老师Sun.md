# 【挽回前女友系列】女朋友觉得不：女朋友说不合适-晴天老师Sun

在本节课中，我们将要学习当女朋友以“感觉不合适”为由提出分手时，如何进行有效挽回。我们将深入分析分手原因，并掌握四个关键步骤来重建沟通与吸引力。

从常规来看，正常的热恋期大约持续三个月。若在相恋不久后对方就认为“不合适”并提出分手，通常有两个原因。第一种称为“冲动型相恋”，即在暧昧氛围下盲目建立了关系。第二种是“实际与预期不相符”。

例如，从多年朋友关系转变为恋人后，仍以旧的眼光看待新关系。面对这类分手，你的逼问越紧，对方可能抗拒得越厉害，导致你相当被动。因此，挽回的第一步是弄清分手的深层原因。

上一节我们介绍了“不合适”分手的常见原因，本节中我们来看看具体的解决方法。以下是建议采取的四个核心步骤。

## 1. 倾听内心的声音 👂

想让“小刺猬”放松，就需要营造安全的环境。从心理学角度说，人在面对失去时，最先出现的情绪是怀疑，因此你可能会急切地问对方为何分手。

这种“为什么觉得不合适”的问法，本质上是安全确认。但对前任而言，这可能被感知为批评和指责，容易激化矛盾、偏离事实。所以，若有沟通机会，务必表现出包容与理解。

不要试图讲道理。应先肯定他的想法，再用具体事件引导他表达内心真实想法。对话中多使用疑问句，而非辩驳“我没有，是你误会了”。例如，前任说：“我认为我们不合适。”你可以回答：“的确有一些地方需要磨合。不过我很好奇，你最近回复消息比较慢，是觉得没有安全感吗？”

肯定对方想法，不代表接受他的决定。通常，对方在心理上感到被承认后，才愿意多说。用事件引导的目的，也在于纠正事实的偏离。

情侣间的不满，常源于某件事引发的情绪反馈，而容易忽略背后的需求。例如，女生抱怨男友天天打游戏，但她不会直接说“我希望你多花时间关注我”，只会埋怨打游戏的行为。男生则会觉得女友不让自己打游戏。实际上，矛盾根源并非打游戏本身，而是男友缺乏关注。只有引导对方说出真实想法，才能进入下一环节。

如果对方抵触心理强烈，说明当下氛围不适合沟通。可以先找理由离开，给予对方空间。后续再找机会重复上述步骤。

**提醒**：不要轻易道歉。一旦道歉，你就坐实了问题。若希望后续有交流余地，应尽量展开话题聊开。

## 2. 疏导对方的情绪积压 🧘

矛盾出现后，对方会经过一段时间衡量。“不合适”不一定是情绪化反应，但必定累积了负面情绪。引导对方说出真实想法后，疏导其情绪压力是扭转局势的关键。

几乎无人能抵御一个温柔温暖的人。我们可以协助对方进行“情绪按摩”，针对他坦露的真实原因进行疏导。继续以上述例子为例，你可以说：“和你相处我一直很开心。这几天委屈你了，憋在心里一定很烦闷吧？为什么不早点跟我说呢？”

包容与理解能进一步打消他的戒备心理，引导对方多说也能检验所暴露问题的真实性。在此过程中，你需要察觉对方想表达的重点，同时小心不被其负面情绪带偏。

大部分人希望通过解释来证明清白、澄清误会。但解释是乏力的，可靠的行动表现才可能引导对方。我们可以根据对方的表达延伸其看法。例如，若他觉得你自私、没情趣，可以反问：“那相处这么久，你觉得我有哪些做得好的地方呢？”让他主动挖掘你的价值。

感情中的负面情绪常伴随压力，而压力会弱化感知力，如同悲伤时不觉得饿。疏导对方情绪的目的，是在关系末期展示你好的一面。一段关系能被记住的瞬间，要么是相遇初期，要么是分离末尾。给自己留下体面，对方在挽回过程中才更容易记起你的好，激发其“损失厌恶”心理。

## 3. 创造不可得性 🚪

面对“不合适”分手，最容易出现优柔寡断、又怀疑又痛苦的情况。特别是当对方指出你的不是后，你可能会揽下所有罪责，哀求对方别离开。但这往往事与愿违，因为你只是将控制对方的行为赋予了“爱”的名义。

如前所述，其实无需做出解释。你可以回复：“嗯，我知道了。”这种模棱两可的答案，等到“小刺猬”舒展开后，他会重新审视之前发生的一切。你要做的是在这段时间内，不过分关注他的动态，自己也不发社交动态。

过一段时间后，再投其所好，发布与他相关的朋友圈。若前两项完成到位，此时期保持弱联系即可，例如偶尔聊天（频率约一周一次），或邀请对方游戏。过程中绝不提感情。当你感到氛围不错时，便找借口离开。**核心目的**是学会创造不可得性——太容易得到，反而不易珍惜。

## 4. 提升与丰富自己 🌱

男生在感情末端表现得越优柔寡断，女生越容易厌烦。对方产生厌烦的原因，也在于你过多的托付心态和负能量影响。

如果你有过恋爱经历，会发现分手后或多或少都想知道对方近况。越不知道，就越想知道。任何人对待关系，都下意识先考虑自己会获得什么，而非想要被索取什么。当你发现只能通过纠缠去挽回时，说明你已缺乏新鲜感，翻来覆去还是那几句话。这也在提醒你，需要赶紧丰富自己的生活。

去提高眼界，从宏观把握感情流动的方向。例如多看书、培养一个兴趣爱好、做志愿者，或完成一些一直想做但没时间做的事。通过自省或与他人的互动，发现关系的多面性。对于恋爱和恋人的理解，也可能产生新的认知。

从理论上说，挽回过程更像一场博弈，**核心**在于：`谁更离不开谁，谁更害怕对方离开，谁更敢于承担分手风险`。当你总在思考该不该发消息、该不该邀约时，实际上你是在进行情感投资。投资越多，你就越痛苦。反之，如果他时常留心你的动态，贪恋你提供的情绪价值，那他也在进行投资。投入多了，自然也就离不开了。
