# 撒娇术课程 08：如何利用「撒娇术」化解矛盾？💖

在本节课中，我们将学习如何运用撒娇技巧，巧妙化解亲密关系中的小矛盾与冲突。正确处理矛盾是维持感情愉快、避免失望累积的关键。通过撒娇，我们不仅能缓和紧张气氛，还能让对方更愿意包容与理解。

---

## 矛盾处理的重要性

看过我们挽回系列视频的同学应该都知道，分手有一个最大的原因，就是矛盾累积导致的失望型分手。感情相处是否愉快，最需要具备的能力就是处理矛盾的能力。矛盾不会解决，会导致吵架或冷战。如果感情基础不深，甚至可能直接分手。

即使感情深厚，如果忽视矛盾、继续回避，这种处理方式也会累积矛盾。最终一方爆发时，会因为已经积累了许多负面情绪和负面印象，从而彻底否定你这个人。这也会加大我们挽回的难度。

上一节我们介绍了矛盾处理的重要性，本节中我们来看看如何用撒娇术具体化解生活中的小矛盾。

---

## 场景一：生活中的小过失

面对生活中的小过失，我们可以采用“妄自菲薄法”。例如某些物品的零件掉了而你装不上去，或者不小心弄脏了他的衣服、搞坏了什么东西等。

以下是具体步骤：

1.  先无辜地瞪大眼睛看向他，眉心向上抬，委屈地撅一下嘴巴。
2.  接着很愧疚地和他说：“怎么办，我好像把这个东西弄坏了。”
3.  这时男生会过来查看发生了什么，然后我们可以补上一句：“完了完了，要被讨厌了。”说这句话时最好配上仰天长啸的绝望表情。
4.  99%的男生都会很快反应过来，然后表示“怎么会讨厌你，这是件小事而已”。
5.  接下来他几乎一定会试图去修复你造成的小破坏。这时还没有结束，你要看着他忙碌的样子，由衷地感叹一句：“我今天太笨了，我哥/老公怎么办？”

男生的心思其实很单纯。这样先承认自己的小过失，然后妄自菲薄一下，之后给他一点小小的夸奖和撒娇，将男生捧一捧，就能巧妙避开我们造成的小麻烦，同时还会让他觉得自己是你离不开的英雄。

## 场景二：吵架时的处理

如果两个人真的吵架了，这时候首先要及时拉住“冷战的缰绳”。但如果问题比较严肃，或者你可以感受到对方已经明显很生气，我们就不要继续撒娇了。

以下是处理吵架的正确步骤：

1.  **首先，及时控制住自己的情绪**。不要急躁，将语气放慢、放软一点，不要一生气就口不择言。
2.  **举例**：假如你或对方因为吃醋而将气氛搞僵，这时千万不要轻易说出“反正你就是烦我的”、“你以前不是这样的”这一类任性的话。这已经完全脱离撒娇的范畴了。
3.  **正确做法**：如果对方在闹别扭，那我们就做那个率先服软的人。你可以用温柔的语气向他表示：“你在气头上，我们冷静一下再谈好吗？我先去给你做点吃的。”要表现出冷静之中带着些许委屈，但你仍然十分关心他的态度。

这样的处理方式，能够快速地让对方意识到自己在闹脾气。而你为他做的这件小事，恰好又可以给他一个台阶，让他乖乖走下来。

**重点归纳**：首先要避其锋芒，同时不要逼着对方说话，但也不要沉默地去应对他的沉默。你要表现出自己的冷静和小委屈，然后为他做一件小事来铺台阶，等他自己走下来。

## 场景三：冷战中的破冰

有的同学可能会问，如果我没有控制住，已经陷入冷战了，该怎么给自己一个台阶，让他接住我呢？不用担心，这种情况我们也有办法用撒娇来顺利解决。

首先明确一点：你是否确定自己已经真的不生气、真的想通了、真的想要快速结束冷战、重回对方的怀抱呢？如果是的话，那么你有两种撒娇的办法可以快速和对方回暖。

### 方法一：肢体撒娇结束冷战

这种不说话的撒娇方法，不要犹豫、不要害羞、也不要碍于所谓的面子。直接去抱他。

以下是具体动作（“背后环抱式”）：

*   双手从他腰后侧穿过去。
*   不要以自己握住自己手腕的方式去环抱他，而是要两只手臂上下分开，手心去贴住他的腰部，手指微微张开。
*   整个侧脸贴住他的后背，不要说话，自然而然地流露出你的妥协和对他的依恋。

一般男人都会顺势握住你的小手，转头问：“不生气了？”这时你就可以维持脸贴他后背的姿势，幅度稍微大一点地点头。点完头之后，再将手臂收紧一点抱住。

**点头、收紧手臂**，就这三个动作，小矛盾就迎刃而解了。

### 方法二：语言撒娇结束冷战

第二种方法，我们要正面突袭。

以下是具体步骤：

1.  走到他面前站定，先和他对视两秒，然后低头垂一下眼眸。
2.  两只手最好放在身前，手指绞在一起，可以表现出一种纠结又鼓足勇气的模样。
3.  说一句话：“我不生气了，你能抱抱我吗？”声音低一点，委屈一点。
4.  说完之后再慢慢地抬眼看他。如果眼眶含泪那就最好不过，如果实在挤不出来，就瘪瘪嘴、吸一下鼻子也是可以的。

一般女生都这样服软了，表示自己不生气，还主动给一个可以抱一抱的台阶，没有男人是不乖乖走下来的。这招真的非常好用，百试不爽。

## 核心原则与总结

聪明的女孩子，一定要能够看清问题的实质，乐于撒一个娇、偷一个小懒，从而减少一个小矛盾、小问题。

不过，我们虽然可以利用撒娇来化解之前无伤大雅的小过失、小矛盾，如果真的遇到严肃的问题，还是希望大家要有一个端正的态度去解决。凡是自己该承担的责任，也要毫不犹豫地承担起来。

**本节课中我们一起学习了**：
1.  处理矛盾在感情中的重要性。
2.  用“妄自菲薄法”处理生活小过失。
3.  吵架时如何控制情绪、率先服软并提供台阶。
4.  冷战时通过“肢体撒娇”和“语言撒娇”两种方法有效破冰。

记住，撒娇是一种润滑剂，而非逃避责任的工具。合理运用，能让你们的感情更加甜蜜顺畅。
