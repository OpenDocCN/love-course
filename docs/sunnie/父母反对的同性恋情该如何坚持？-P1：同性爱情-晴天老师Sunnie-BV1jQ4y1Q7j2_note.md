# 情感课程 01：如何应对父母对同性恋情的反对？ 🏳️🌈

在本课程中，我们将探讨当同性恋情遭遇父母反对时，个人应如何应对与坚持。课程将分析压力的来源，并提供具体的策略与心态调整建议，旨在帮助处于此情境中的个体更清晰、更坚定地面对挑战。

---

提到同性之间的感情想要一个未来，父母的阻力其实要比一般的感情中大很多。虽然异性恋、同性恋、双性恋、无性恋等性取向都已被学术界认可以及大多数年轻人所接受，但是试图改变中国父母的固有观念是一件特别艰难的事情。

有的父母完全无法接受，乃至以死相逼。我身边就有一个同性朋友的父母，知道他的性取向后甚至说出“你怎么不去死”、“我希望从来没有生过你这样的儿子”这样残忍的话。我们只能劝他不要往心里去。

可是在意的人说的话，怎么可能不往心里去呢？除非自己不在意这个人。伤害是必然已经造成了，但是你们不必否定自己。没有人强大到足以用固有的思维观念来使你消极，除非你自己愿意让他们影响你、控制你。

上一节我们分析了父母反对带来的情感冲击，本节中我们来看看这种反对观念的根源。父母特定的社会历史成长环境塑造了他们的观念，他们至今为止的一切都是凭借自己特有的思想经历建立起来的。有一些坚持对他们来说可以说是信仰一般的东西了。

他们对于内心固有观念的坚持，可以让他们生出很多反对的理由，比如“不合伦理”、“别人会嘲笑”、“无人养老”等等。可以理解这个情况，跟父母的接受度不够有关。他们在有生之年确实可能没有接触过，而且因为年纪大了，想要迎接新的生命，这给了他们生活的希望。

父母反对这个问题的根源，其实还在于主流社会的不认可。确实在国内，同性之间的关系还没有被官方真正地承认。在咨询行业从业这么久，也遇到过同性咨询者因为在体制内工作，而面临着工作和爱情的艰难选择。所以，就算有的父母同意或者默认，但是还是会被要求结婚，因为这样才能免于被人用异样的眼光看待。受大环境压力影响，困难也会增加。

我有一位在政府工作的朋友，他向父母坦白出柜后，父母就说：“不管怎么样你还是要结婚的，结婚对象你自己看着办吧。我们不在意，毕竟你还是要考虑一下你的单位环境。如果你不想结婚，那你的工作还是要好好衡量一下，究竟要选择哪一个。”最后，我的朋友就和女友协商找人形婚，对方也是男同，两个人结婚后各过各的，朋友和他的女友也幸福地生活在一起。

父母这一辈大部分都40岁甚至更大的年纪，已经接受传统思想的熏陶了。你忽然抛出一个自己性取向的问题，家里肯定会炸开的，除非你有一个理由可以让父母接受。

那么，父母这一关究竟该怎么过才是最好的选择呢？可以參考以下幾點。当然，具体情况还是要看自己家庭环境。

以下是几个关键的应对策略：

1.  **给父母喘息的机会。** 有时有些人会直接告诉父母，没有一点铺垫。你可以选择告诉父母一个善意的谎言，例如“曾经被一个异性伤害了，无法结婚，看到异性就会出现生理的反胃”。这种情况就可以给父母铺垫，等你真正摊牌的时候，也不至于给父母太大的冲击。万一父母身体不好，受刺激进了医院，也不太妥当。
2.  **确保自己的经济能力完全独立于家庭。** 甚至可能家里的所有开销都可以由你来承担。不是说现在不需要家里承担费用，而是你最少身上要有一定存款，能够保证遇到突发情况时，不用急着讨论是工作重要还是爱人重要。现实毕竟不是童话，工作也不是你想丢就丢的。也不是所有的父母都是开明的，家里都是富二代。没有那么好的环境给你去创业或干其他工作，一定要综合考量方方面面。争取另一半的同意也很重要，千万不要擅自做决定。
3.  **不要展示自己懦弱的一面。** 说这个不是要让你和家里人闹起来，而是不希望你在父母让你去相亲，或者认识异性的时候，心里抱着“我就是去认识认识，不会深入发展”这样天真的想法。你一步步退让就会导致真的最后一步步妥协，直到最后你不得不结婚，背负不得不承担的责任，然后痛苦地生活。

独立是根本，首先是经济独立，然后是精神独立。我们总说中国式父母如何如何，那你有没有想过自己可能也是中国式子女呢？在你想要独立的时候，你就拿西方父母的理解和开明说事，而在你需要支持和关爱的时候，你又习惯性地去依赖父母。

至于很多人分手的根本原因，其实不是因为父母反对，而是因为五个字：**看不到未来**。然后父母的反对外界的压力就让你们摇摆了。什么是未来呢？你们真的决定长久在一起需要具备哪些条件呢？必然是需要你们一方甚至双方能够承担起生活的责任，能够共同面对周围的流言蜚语，包括经济上的。不是说两个人穷就不能生活得很好，只是万一你们之间有什么意外，或者两个人有更好的发展机会，那时候该怎么办？

还有一点，你们要面临一种风险：父母万一就是完全没办法接受你们，以后你们可能就要老无所依，无依无靠就剩彼此了。出现这种情况其实就能理解一部分压力了。但是，我希望大家不能因此就完全的退缩，要堅持。如果你们真的选择了前行，就一定要好好地在一起，不要轻易地放弃以及动摇自己心中所爱。

父母之心可以理解，他们也是爱你。不爱你的父母也不会让你产生这么大的感情压力。其实最根本的还是你们自己能不能好好照顾好自己。因为大部分的女性相较男性来说，身体方面的力量没有男性那么强大，所以父母亲可能不放心。你没有必要为父母改变，也不需要他们为你改变自己的想法。只是你们还是要多交流，要让他们知道你的想法。成年人需要自己为自己的决定负责任，时间也可以慢慢证明。

前面我们已经说过了外界的种种压力的强大和痛苦。感情走向末路，其实根本原因还是内部出现动摇了，也就是你们自身的感情出现了问题。根据心理学上的 **罗密欧与朱丽叶效应**，当出现了干扰恋爱双方爱情关系的外在力量时，恋爱双方的情感反而会加强，恋爱关系也因此会更加牢固。

很多人问我同性挽回和异性挽回有没有什么不一样，其实没有很大的不一样。亲密关系都是相通的，你要解决的不只是外界的因素，更多的还是导致分手的重要原因。可能是你缺乏安全感，你的占有欲控制欲给对方太大压力了；或者你们缺乏互动，让对方独自承受了很多压力。

很多性少数者会出现一种偏执的情况，因为出现感情和负面情绪的时候，你的倾诉渠道是很少的。找同性朋友吧，可能怕引发矛盾；找异性朋友吧，又不能理解你们感情中的问题。所以你只能把压力施加在对方一个人身上，这样也可想而知对方需要承受多少感情的压力。

所以，一方在这段感情中，如果感觉**痛苦 > 快乐**的时候，可能他就想离开了，因为人都是趋利避害的。可能最让人心寒的是，有一方在积极地为两人创造有利条件，而另一方却在悄悄打退堂鼓。最后哪一方来挽回都是有可能的。

最后我想说，如果你的伴侣有一方坚持不下去了，不要去抱怨指责对方懦弱不坚定，你也要理解一下他的压力，不是吗？很多人遇到这些情况、这些问题都不知道要如何去解决，甚至想要逃避。但是逃避是解决不了任何问题的。如果你自己都不能真正地面对这些，你还怎么去带领对方一起去面对呢？

也许对方也可能真的有更多的选择。除了亲情、爱情、工作和爱情的抉择，还有友情和爱情的抉择。因为爱情的排他性，对方可能还为你放弃了部分的友情，必然就会心生怨怼，觉得为你做出了很大的牺牲。你们可能也总是认为自己和别人是不同的，总觉得自己付出了更多，受到了更大的委屈。但两个人在一起，图的是开心和快乐呀。很多时候你太过于在意这些付出，反而会让简单的东西不再简单了。

有些人可能本身是双性恋，并不单纯是同性恋，所以会面临很多其他诱惑。在遇到感情问题的时候，他在你这边受了伤，那他们可能就会移情到另一个性别中去。如果是这样，你应该思考有什么能让对方放弃更多选择的资本。其实这个时候还是需要给对方一些空间或者补偿的。自己呢，也要尝试去扩大你的交际圈，不要世界只有他一个人，从另一个角度可以解决问题，这样两个人也不需要这么累。

你要掌控好自己的幸福，不要活在他人的牢笼里。而如果你的爱人在牢笼里出不来了，你是否愿意去拉他一把呢？终究还是为了自己的幸福，因为被爱无价，自己爱上一个人也不容易。如果你去挽回、去坚持，对方还是选择听父母的想法和异性结婚，那最后失去幸福的就是他了。毕竟，一个人被另一个人坚定的选择，已经是他最深的福德了，可能这辈子再也遇不到了。

我说了这么多，还是希望你不要太担心，总会有办法解决的。你太在意很多东西，反而容易使得自己在这段感情中处于低位，反而会给对方压力。总是越怕失去的那一方，越容易失去呢。如果你太累了，觉得坚持不下去了，你也是可以哭的，你也是可以放弃的，你也是可以休息一小会的。

**先爱好自己，你才能去爱别人。只有自己好好生活，才是最重要的。**
