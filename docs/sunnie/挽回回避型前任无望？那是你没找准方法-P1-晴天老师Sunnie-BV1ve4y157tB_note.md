之前关于挽回回避型前任的视频，引发了广泛共鸣。我收到许多自愈者的私信。一个有趣的现象是，许多想挽回的人，都认为自己遇到了回避型的前任。这让他们感到无计可施，不知如何是好。

回避型依恋是成人依恋风格理论中的一个类型。它指的是在成人亲密关系中，对于情感和亲密感采取回避方式的人。概括来说，他们对待伴侣冷漠，因问题不爱沟通，基本不主动靠近，也很少表达爱意。

你可能被分手后也感到云里雾里，不知问题出在哪里。当你想弄清楚分手缘由时，发现对方油盐不进，说什么都得不到回应。你不仅要接受失恋的痛苦，还会被对方的回避态度反复折磨。

我们的挽回者基本上有一个通病：揣测对方的想法。无论是自己猜测、与朋友分享，还是在网上看视频对号入座，最终目的都是想知道前任到底在想什么。但我一直强调，挽回中最不应该做的，就是给前任乱贴标签。

有时分析完你们的感情，在我说话前，你可能就会下一个判断：“我觉得他是回避型，完全不回消息，也拒绝沟通。” 仅凭对方不回消息这一点就贴上“回避型”的标签，其实很武断。

所以，在你挽回之前，首先要弄清楚他是否真的是回避型。有时对方并非回避型，只是不想理你而已。分手后关系破裂，没有交集自然缺乏交流。加上前期你可能做了一些让他讨厌的事情，才导致他回避你。他不想与你有牵扯，也不愿看到满屏的文字。他理性分析过你们的未来，觉得不合适，也知道回复你可能给你希望，所以索性把你放在一边，让你自己感到无趣后退缩。

在现实生活中，真正的回避型人格并不多见。更常见的情况是，对方不愿意和你沟通。你们恋爱初期，对方并不回避交流，后期不愿沟通，主要问题出在沟通方式上。例如，你的沟通重点在情绪上，而对方的重点是要解决问题。两人根本不在一个频道，自然无法沟通。针对这种情况，你要做的就是学会非暴力的有效沟通，同时还要了解和理解对方的需求。

那么，真正的回避型依恋是什么样的呢？我曾接触过一位回避型咨询者。他硬价值很高，长相帅气，事业有成，也曾交往过几位价值匹配的女友。但每当他进入亲密关系后，就会觉得自己被束缚，没有自由。他并非不爱女友，但害怕进入长期关系，因为他觉得对方会伤害他。所以他拒绝过度亲密，会对女友冷淡，有时还会贬低对方。一旦感情有进展，他也会立刻疏远一段距离。长期以往，他的女友无法忍受他的性格，结束了这段感情。

总结一下回避型依恋的行为模式共性，通常有以下几点。感兴趣的话，你可以对照一下。

以下是回避型依恋的典型行为模式：

1.  **轻视依赖感**：回避型依恋者在亲密关系中，会有不同程度的逃避或拒绝亲密感。他们的感情需求低，极度重视独立，需要个人空间。于是会想方设法降低两人的亲密行为。他们多半也不愿与伴侣分享心事，所以你只能猜测他的心思。猜对了能拉近关系，猜错了他们也不会告诉你，但你心里肯定不是滋味。如果你的感情需求比较旺盛，在恋爱时也会让对方感到压力，觉得被束缚。
2.  **存在“鸵鸟心态”**：心理学家这样定义“鸵鸟心态”：人们面对问题时通常采取回避的态度。回避型的人普遍有这种逃避心理。他们很少积极解决两性矛盾，往往直接回避。即使你想解决或跟他沟通，他还是会选择逃避。
3.  **过度美化爱情**：大部分人的依恋风格其实是小时候形成的，长大后只是延续。回避型依恋者的父母长期对他们不闻不问，有时还会嫌弃他们（比如成绩不好）。这在他们心中植入了一个错误理念：**“我是不完美的，所以父母不喜欢我。如果我完美了，我才能够被爱。”** 这种观念根深蒂固。因此，他们认为一个人只有完美才能被爱。他们知道自己不完美，所以为了保持爱情，就会保持一段距离，不让对方发现缺点。这种心结还会投射到伴侣身上，他们认为靠近伴侣就能发现对方的不完美，从而对爱的幻想破灭。所以他们越爱越恐惧。

除了这些，还会有其他表现，比如忽视伴侣感受、贬低伴侣、自我封闭、不爱表达、不敢付出等。

看到这里，你是否觉得回避型的人太难相处了？是的，回避型的人确实一开始很难相处。你也不知道自己做错了什么，就被他一言不发地放弃了。

但是，为什么又有那么多人想去挽回回避型？通常，回避型的人比较独立，也比较优秀。他们有一个很大的优点：虽然你很难走进他们的心，但一旦你能懂他，理解他的行为，他也能够逐渐付出真心，并且坚定到底。

所以，如果你仍然想要挽回回避型，我们要怎么做呢？

对方现在不回消息，对你回避。即使你有很多想说的话，也请暂时不要联系他。你的首要目的是先让他不再回避你，要让他接收你的消息，形成信息互动。这样，后期的挽回策略才能跟上。

回避型的人有很强的界限感，他们独立且自重。分手后，不太可能主动联系你，但这不代表他们彻底放下了或不喜欢你了。他们只是本能地选择回避。既然他回避，那你也不要强制性地找他。如果前期纠缠过，那就隔半个月左右再去联系。联系时保持一颗平常心，频率不用太密集。你就像对待普通朋友一样对他，说话方式不要太强硬，尽量温柔一点。如果对方没有回应，也不要太焦虑，不要急着一定要让对方回复。

挽回回避型有一个重点：**要理解他，要懂他**。回避型的人对感情需求不高，他们更在意的是你是否能懂他。之前说过，他们小时候被养育者忽视，没有怎么感受过被爱的感觉，所以也不懂得如何去爱别人，天生对爱不信任，才会用冷漠的方式逃避爱。虽然你们已不是恋人，但你仍然要去关心他，让对方感觉到你是了解他的人，你一直站在他身后。当然，不能急功近利，千万不要做出超越现阶段关系的事情，这是回避型最讨厌的点。例如，你着急想进入他的世界，想了解他是否有感情动态，这与你们目前关系的基调相反，所以我们不要去问这件事。

能够正常交流之后，我们就需要注意依赖的问题。他们轻视依赖感，所以在此期间，你要避免过度依赖他，但也不能表现出冷漠。如果你比对方还冷漠，他就只会更加不信任亲密关系，那你为挽回所做的所有努力都白费了。

他会接受我吗？最好的方式是保持自己的独立生活，并且还要告诉对方你目前在忙的事情。另一方面，你也要多关注他的生活，在他需要的时候给予帮助，多鼓励对方。如果对方开始进行投资了（哪怕只是简单回复），你都要肯定他的行为，不要觉得理所当然。因为回避型的人不可能一下子转变成安全型，所以一定需要一个积极引导，逐渐给他建立安全感的过程。

不得不承认，挽回回避型的过程非常痛苦且难熬，比正常挽回还要小心，不能犯错。所以一定要有足够的耐心和毅力，这是挽回的基础。我一直强调，心态是第一位的。如果你是抱着焦虑的心态去挽回回避型，那简直是两场灾难。

当然，回避型的人也没有那么可怕。他并非永远保持回避。只要你找对方法，循循善诱，把他从回避的漩涡中慢慢拉出来，你们的亲密关系是能够更进一步的。
