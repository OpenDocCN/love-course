# 《高位挽回逻辑》

在本节课中，我们将继续学习“高位挽回”的核心逻辑，重点探讨如何通过增强对方的核心信念、增加其主动行为的回报，以及应对特殊人格类型，来引导对方主动提出复合。我们将使用简单直白的语言和具体步骤，帮助你理解并应用这些策略。

## 增强对方的核心信念 💡

上一节我们介绍了高位挽回的基本心态，本节中我们来看看如何具体操作。高位挽回的第二个逻辑是增强对方的核心信念。无论最初是谁主动追求，对方决定与你在一起时，必然持有坚持走下去的信念。这信念源于你当初给予对方的选择动力。

如果对方仍为你的朋友圈点赞，或未将你屏蔽，这意味对方在某种程度上接受你的存在。接受的真相是对方目前不排斥你。若你不信，可以进行试探。

以下是试探方法参考，基于一位学员的真实案例：

*   **第一步：展示积极生活。** 学员在晚上发布了海南自驾游的风景照，其中包含一张个人美照。
*   **第二步：创造互动契机。** 在预估前任可能活跃的时间段，学员又发布了一张自己开车的照片，配文为：“连夜开往三亚，祝我一路平安。”
*   **第三步：观察并回应。** 分开两个月后，其前任首次回复了“一路平安”。通常，此阶段对方已浏览过你的朋友圈。若你不回复，对方可能会点赞你后续的内容或自己发布朋友圈。你只需隔段时间为其点赞即可。

通过逐渐增强这类互动，可以强化对方的核心信念。你传递的信息是：如果对方主动联系，你不会排斥他。这本身也是一个相互试探的阶段。

## 增加对方主动行为的回报 🔄

高位挽回的第三个逻辑，是增加对方主动行为的回报。这并非要求你热情回应，而是记住一个比例：**如果复合需要走100步，你只走3步，让对方走97步。**

以下是具体三步操作：

1.  **第一步：礼貌回应，分割过去。** 通过你的回应，让对方看不到你过去的负面情绪。公式为：`回应 = 礼貌 + 中立`。
2.  **第二步：逐步幽默回应，二次强化初始印象。** 回归你当初吸引对方的优势（即你之前记在纸上的优点列表）。代码逻辑可表示为：
    ```python
    if 对方开启话题:
        回应方式 = 幽默 + 展现旧日优势
    ```
3.  **第三步：你来我往，增强沟通默契。** 利用你们自有的情感基础，逐步优化聊天节奏。人会不断美化和强化过去的记忆。

基本上完成以上三步，约75%的男性会主动提出复合试探。此时不建议立刻答应，适当“欲擒故纵”，让对方付出一些努力，他们才会更加珍惜。

## 应对特殊人格类型：低自尊与高自尊 🧠

最后，我们需要了解哪类人最难主动挽回。如果你遇到的是低自尊或高自尊的人，可能需要你更主动地出击。这两类人的内在心理都源于自卑，被动状态更能让他们获得安全感。

在挽回前期，你需要主动化解你们之间的刻板印象。同时，在化解过程中，要增强对方主动行为的“奖励”。

例如，分开后你有事请他帮忙，即使他只给出一个小建议，你也需要感谢他。并且，字里行间不能附带其他情绪。因为低自尊者习惯回避挑战和可能造成损失的情况，即使想复合也可能会逃避。

低自尊者通常有两种表现类型：

*   **类型一：被动回应型。** “我不主动联系你，但你联系我，我会回应。”
*   **类型二：冷淡回应型。** “和你聊天我很开心，但我担心说错话让你烦，所以我会回应得更冷淡。”

低自尊与高自尊者有一个共同点：**他们的负面信念占据主导**。例如，如果你强势提分手或在分手前数落他，他们会更容易相信自己是被你讨厌和排斥的，从而避免沟通。

## 关于被拉黑/删除及后续准备 📝

对于被拉黑或删除的情况，不必过于担心。超过半数的恋人在分手约一个月后，会回忆前任并试图窥探其现状。因此，你必须提前做好准备。

如果对方在窥探你时，发现你仍是老样子——伤春悲秋、不修边幅等——那么他回头可能性将大大降低。不必纠结对方是否重视过这段感情，只要人存在“损失厌恶”心理，且你未犯原则性错误，绝大多数人都无法彻底放弃一段感情。

关于担心对方已有新欢（空窗期）：每个人恋爱的对象虽不同，但你过去喜欢的人与对方现在喜欢的人，总有一些共通点。有时候，人确实在失去后才懂珍惜。**只有先失去，才能产生高浓度的、失而复得的喜悦感。**
