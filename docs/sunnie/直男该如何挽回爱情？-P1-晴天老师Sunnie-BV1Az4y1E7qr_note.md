# 直男挽回爱情指南 💔 - 课程01

在本节课中，我们将要学习“直男”在亲密关系中常见的思维误区，以及如何通过有效的步骤来挽回爱情。课程将帮助你理解女性心理，并指导你进行切实的改变。

但在女人看来，这可能是自我感动。你口口声声说爱她，却不从自身出发解决行为和态度的问题，也看不到她的需求和困扰。那么，再多好意也是多余的。

我通常将第二种“直”称为**小男孩式的直**。其表现像孩子以自我为中心。他并非没有同理心，但亲近人的方式是硬塞给对方自己喜欢的玩具，因为他自己玩得开心，便认为对方也会开心。若对方不陪他玩或忽视玩具，他就会生气，因为他觉得自己的“好意”被辜负，自己“亏了”。

事实上，许多直男在亲密关系中仍在扮演“小男孩”。

以下是几种典型的小男孩式直男表现，你可以对照自查：

当然，人无完人。有缺点意味着有改进空间。当你意识到自己是直男并犯了上述错误时，问题反而不那么严重了。

在这种心态下，你的挽回之路会变得更加明朗。剩下的就是一步步执行。

同样，被分手不意味着失败。你应将其视为了解异性、学习两性关系的宝贵机会。学习挽回，实质是在提升情商。你的情商远未达到极限，不要被暂时的困难打倒。

以下是三个针对性的挽回步骤：

**第一点：先反思，再行动**

许多直男有心直口快的毛病。女友提出分手后，自己急得像热锅上的蚂蚁，严重时还会幻想女友与他人约会的场景，痛苦不堪。

“先反思再行动”对你而言，是给了一段冷静的时间，用以思考过去相处中存在的问题，特别是自己可以做出哪些改变和提升。对对方而言，她也获得了淡化对你负面印象的时间和空间。

你突然的“抽离”会给她一种“中断的不完整感”。人对未完成的事情印象更深刻，这在心理学上称为**契可尼效应**。这种“未完成感”对你后续重新联系和挽回会有帮助。

无论你的女友属于哪种类型，都要学会揣摩她的心理，立足于她的真实需求去付出。这样才能避免陷入“你给的她不想要，她想要的你给不了”的窘境。

你现在学习这些技巧并不晚，毕竟你与前任有情感基础。学会分析心理只是加分项。

*   “多喝热水”比一言不发要好。

改变是循序渐进的。不可能今天还木讷说不出话，明天就能甜言蜜语哄得她团团转，那样反而会让前任怀疑你是否换了个人。

这是一场持久战。只有当前任看到你身上有切实改善的迹象时，她曾经绝情冷漠的态度才会自然缓和，对你的靠近也不会那么抵触。
