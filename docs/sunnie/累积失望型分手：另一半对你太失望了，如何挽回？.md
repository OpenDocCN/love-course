# 累积失望型分手：另一半对你太失望了，如何挽回？

在本节课中，我们将要学习一种常见的情感破裂模式——累积失望型分手。我们将分析其成因，并提供一个清晰、可操作的挽回框架。理解这种分手的本质，是成功修复关系的第一步。

前几天有一位咨询者找我，说他自己有次睡着了，没有及时的回女朋友消息，所以女朋友决定跟他分手。

大家乍一听是不是觉得很惊讶，甚至觉得女生在小题大做。

但是我在看过他们之前的聊天，相处细节的时候才知道。

这段感情的破裂，并不只是因为这个不起眼的一件小事。

这件所谓的小事，对他来说，实际上是压死骆驼的最后一根稻草。

在这件事之前，男方也早就做了无数件，让他对这份感情失去信心的事情。

只是没回消息这件事情，刚好引出了女生长久以来积累的不满而已。

这个就是典型的累积失望性分手。

## 核心概念：什么是累积失望？📉

上一节我们提到了一个具体案例，本节中我们来看看其背后的心理学定义。

毕业于多伦多约克大学的心理学博士，安东尼奥曾经说过，痛苦是在帮助我们辨认具体失去了什么。

伴侣关系中的失望是一种难以避免的现象。

尤其是在长时间的相处中，伴侣之间会逐渐认识到彼此的缺点和不足。

任何妥协，就会逐渐的累积成失望。

当失望累积到一定程度的时候，分手就可能成为不可避免的结局。

**公式：** 单次失望 × 累积次数 = 关系破裂阈值

分手后我们可能会继续尝试跟对方联络。

但是为什么你出的招对方不接，你所谓的求和示好对方无动于衷呢。

你要知道，在一个对你已经心灰意冷的人面前，你的每一次算计，侥幸和对问题的避重就轻，都会让对方感觉你是一个虚伪的利己主义者。

他们会觉得都到这时候了，还是看不到你的真心。

那他又怎么才能够放下芥蒂，跟你重归于好呢。

对你失望的人往往也不是因为不爱了，而是因为太爱了。

你可以好好回想一下，你们刚在一起的时候，他是不是对你抱有一些期待呢。

而你又一次次的亲手破坏掉了这些期待。

## 挽回心态建设：正视问题与优势 🌱

理解了失望的累积过程后，我们需要建立正确的挽回心态。这是所有后续行动的基础。

那么到底我们应该如何去挽回这一类，累积失望情的恋爱。

在挽回前我们首先要放平心态，要意识到对方的态度冷漠是正常的。

不要觉得对方说了很多绝情的话，就没有希望了。

实际上能够经历累积失望直到分手，反而说明你们在一起的时间够久，存在深厚的感情基础。

只是在相处过程中问题和矛盾逐渐显现了，才导致了两人分道扬镳。

所以累积失望性分手的优点就在于，你们的感情基础和对于彼此生活的了解，是他人短时间内无法替代的。

## 挽回三步走框架 🛠️

基于以上分析，针对这种失望性分手的挽回，我们也有一个基本的框架。以下是具体的三个执行步骤。

### 第一步：引导对方发泄情绪

当对方对你失望并急切想要分手时，沟通会变得异常困难。以下是处理此阶段的关键点。

对方对你失望，急切的想要跟你分手的时候，你肯定会控制不了自己，想要去跟对方沟通解释清楚。

但是对方此时的状态肯定是拒绝沟通的，他不会想要和你有任何的交流。

所以这时候你要学会去引导对方，让他将不满的情绪对你发泄出来。

当他发泄情绪的时候，不管他说什么，好听的也好，难听的也罢，不要去跟他辩驳。

**错误示范代码：**
```
if (partner.complains()) {
    defend_yourself(10); // 用十倍的话为自己辩护
}
```

拼命的去解释自己这样做的行为，那他不会觉得你是在跟他解释。

他只会觉得都到这时候了，你还在跟他唱反调。

这种行为对于我们挽回来说是没有任何好处的。

还有一个非常重要的点就在于，不要把所有的责任都往自己身上揽。

这个是大忌。

不顾一切的把错误都归结到自己头上，只会让对方觉得你就在敷衍我。

为什么在一起的时候，你从来没有承认过这些错误呢。

现在要分开了，你仿佛突然意识到你自己做的不对了。

这种行为在对方看来，堪比一种羞辱。

就好像明明你知道怎样做是对的，但是你却不愿意在关系存在的时候，为对方做这些事情。

这个时候我们其实只需要做一件事情，那就是做一个合格的倾听者。

做一个有同理心的合格的倾听者，听对方表达自己的感受。

### 第二步：重建舒适区

在情绪发泄之后，关系会进入一个敏感期。以下是平稳度过此阶段的要点。

第二步就要建立舒适度了。

在对方跟你发现完负面情绪之后，在当下他肯定是不愿意跟你复合的。

并且会伴随着挤出心理，不愿意再跟你深入的去沟通交流。

这个时候你们发现并没有达到自己想要的复合效果。

所以有些人就会控制不了的去纠缠对方，比如说发小作文，长篇大论的去讲一些没有意义的话。

这个又是一个大忌。

这个时候千万不要心急，不要急于求成，这样只会让关系比刚分手的时候还要糟糕。

这个时候我们应该以朋友的身份，去跟对方沟通交流。

去聊一些对方不那么抵触的话题，而且还要避免就两人此时的关系进行讨论。

而是要把心态转换成，我们两个人目前只是朋友的关系。

不要说一些过分亲密的话，找好自己的定位。

慢慢的从朋友的身份，再次进入对方的生活。

总而言之在这个阶段，我们要做到以下4点：

以下是本阶段必须遵守的四个原则：

1.  **不聊感情**：避免谈论过去的感情或未来的可能性。
2.  **不过度关心**：将关心控制在普通朋友的程度。
3.  **不聊二人的关系**：不讨论“我们现在算什么”。
4.  **不做越位的事情**：不做超出朋友界限的亲密举动。

### 第三步：展现新的自我

当前两步为关系打下基础后，最后一步是通过切实的改变来重建吸引力。

接下来第三步就是要展现自己新的一面了。

你可以复盘一下你跟他交往的过程中，那些让对方逐渐失望，并且对方希望你做出改变的地方。

如果确实是你做的不对，或者做的不够好，那你就立刻开始改变。

千万不要觉得改变这些地方就失去了自我，就是去迎合对方了。

我们要永远记住此时的目标是为了复合。

改变是为了让对方看见你有所成长了，也表示你是尊重，你是重视他说的话的。

这时候最重要的就是要学会换位思考。

或者说用对方喜欢的方式去喜欢他。

同时要向他展现你全新的状态。

要让他意识到，你已经不再是当初那个处理不好关系的那个你了。

这个才是我们现阶段的目标。

本节课中我们一起学习了累积失望型分手的成因与挽回策略。

总的来说累积失望型分手，挽回的可能性是非常大的。

只要我们按照步骤逐步的去推进，去承担好自己应该承担的责任和后果。

坦然的面对承认自己的错误。

最后用行动和新的状态，去迎接和既有人的新关系新可能。
