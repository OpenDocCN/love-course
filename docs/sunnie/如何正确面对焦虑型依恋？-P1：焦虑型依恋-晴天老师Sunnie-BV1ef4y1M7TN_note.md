# 课程一：认识焦虑型依恋 🧠

在本节课中，我们将要学习什么是焦虑型依恋。我们将通过一个经典的心理学实验来理解其起源，并探讨它在成人亲密关系中的表现。

20世纪70年代，心理学家玛丽·艾因斯沃斯进行了一项名为“陌生情境”的测试。实验对象是12~18个月大的孩子。在母亲的陪同下，孩子被放置在一个有玩具的房间。随后，一个陌生人进入房间并与孩子和母亲互动。接着，母亲离开房间，留下孩子与陌生人独处。几分钟后，母亲返回房间安抚孩子。

实验发现了三种主要的依恋类型：
*   **安全型**：孩子能自信地探索环境，将母亲视为安全基地。母亲离开时会哭，但母亲返回后能很快平静下来。
*   **回避型**：孩子表面上能自信应对环境，似乎不关心母亲是否在场。但母亲离开时，其生理指标（如心跳）会加速。
*   **焦虑型**：即使母亲在场，孩子也不太关注周围环境。母亲离开时会哭得非常厉害，并且需要很长时间才能平静下来。

因此，焦虑型依恋也被称为矛盾型或反抗型。这种模式会延续到成年。成年后的焦虑型依恋者，内心常伴随不安与焦虑。他们习惯依赖这种情绪来获取伴侣的关注和照顾，以安抚自身的不安。

以下是焦虑型依恋在亲密关系中可能表现出的行为：
*   要求伴侣随时报备行踪。
*   联系不上伴侣时，会不停打电话或发信息（“夺命连环扣”）。
*   撰写长篇大论的“小作文”表达情绪。
*   禁止伴侣与其他异性来往。
*   反复询问或要求对方证明爱意。
*   发生问题时，会与对方不断争吵，抓住一个问题不放。
*   不分场合、时间地反复与对方纠缠，歇斯底里地抗议。

**但是，即使你有过上述行为，也请不要轻易给自己贴上“焦虑型依恋”的标签。** 即便你确实是焦虑型，也并非有问题。你只是对待感情问题更敏锐、反应更快、更强烈。我们需要做的，是在感情问题出现时，不要立刻表达焦虑甚至挑起争执。因为你比别人快，所以要有意识地让自己慢下来。

## 课程二：焦虑的根源与影响 🔍

上一节我们介绍了焦虑型依恋的定义和表现，本节中我们来看看其核心特点及对关系的影响。

关于焦虑带来的过度依赖，与其说你们不独立，不如说你们对感情质量和亲密度的要求比别人更高。如果你恰好找了一个对亲密度要求低、极易满足的伴侣，就容易产生冲突。

这好比两个人对物质生活的追求不同：一个想拉着伴侣一起奋斗赚更多钱，另一个则安于现状，觉得当下生活已很满足，不必活得那么累。

在亲密关系中，不同依恋类型对亲密度的需求也不同：
*   **焦虑型**：`我要更多，我要更亲密，我要你更爱我。`
*   **安全型**：`合理程度就够了。`
*   **回避型**：`有一点就不错了，没有也行。`

焦虑型依恋者是情感中最纯粹的一批人。他们没有虚情假意，物质追求也不高，正是因为对爱有着极致和纯粹的追求，才导致了他们的焦虑。**追求爱没有错，这份心意也没有错。**

许多伴侣受不了的，并非这份纯粹的爱，而是这份极致爱意下的“副产品”：胡思乱想、作闹、测试考验、情绪不稳等。

任何人的精力都是有限的。在情感、事业、家庭、健康等方面，任何一方投入多，其他方面投入就会少。例如，把大量精力放在恋爱上的人，事业追求通常不会特别高；而事业心强的人，在恋爱上的投入相对就较少。

安全型也只是能达到相对平衡。既然精力有限，如果你的伴侣与你类型不同，他就会把你投入感情的时间，花在其他方面（如事业、爱好、生活本身）。你选择他，可能正是看中了他投入其他方面后所展现的魅力。你现在又强迫他变得和你一样，这种强迫和控制带来的压力，才是问题所在。

## 课程三：如何应对与改善焦虑 🛠️

前面我们探讨了焦虑型依恋的特点及其成因，本节我们将聚焦于具体的应对与改善方法。

面对焦虑型依恋，不要过于害怕，甚至因此加深自我怀疑和焦虑。**感情上有野心没有错。** 你可以将对情感需求的精力，转移到学习和提升亲密关系的能力上。

审视你自身的需求和能力是否匹配。想要高质量、高亲密度的关系，你需要问自己：
*   是否对感情规律有足够的认知？
*   是否具备爱的能力，以及处理复杂情感问题的能力？
*   当伴侣与你需求不一致时，你该如何调和以达到最佳效果？
*   假如矛盾短期内无法磨合，你又应做出何种选择？

一般来说，当你有了有效行为，自然就能减少无效焦虑。仔细分析，焦虑型人群的焦虑通常集中在以下三个方面：
1.  担心自己在情感里有问题。
2.  担心对方放弃自己。
3.  担心时间久了感情就淡了。

我见过的焦虑人群，基本上三样都占了。很多人习惯了情感焦虑，情感就真的会往令人焦头烂额的方向发展。想要从焦虑的泥潭中脱身，方向要对，要慢慢改变。如果只是关注并担忧自己“焦虑”的毛病，对解决问题并无大用。

许多焦虑型依恋者常出现情绪失控，这也是他们自己最害怕的，因为行为可能脱离自我设定的轨道，自认为正常的担忧在别人看来可能是麻烦。

以下是判断是否为焦虑型依恋的一些标准，如果你符合多条，可能需要关注：
1.  感情受挫时心烦意乱，可能做出电话/微信骚扰、愤怒指责等行为，内心希望得到安慰但通常没有勇气说出来。
2.  感到可能被伴侣拒绝时，会高度警觉。
3.  希望每天和伴侣黏在一起，但伴侣可能想要个人空间。
4.  希望对方爱你，害怕做错事让对方失去兴趣。有自信时会非常支持对方，但需要经常确认对方的爱意。若对方反应未达期待，就会担心。
5.  和伴侣分开时，会担心对方忘记你或另寻他人。
6.  感觉感情的走向完全取决于对方。如果失去对方，会觉得自己再也找不到需要你的人了。
7.  失恋后会对过去念念不忘、自责，可能需要很长时间才能恢复。

## 课程四：转变的路径与方法 🌈

在了解了焦虑的表现和影响后，本节我们将一起学习如何从焦虑型向安全型转变的具体路径。

首先需要知道，依恋模式的形成与原生家庭和父母的教养方式有关。也许你的父母做得不够好，但世界上没有完美的父母，也无需一味埋怨原生家庭。每一次不符合预期的背后都有深层原因。埋怨的同时，你就在逃避，可以心安理得地不去思考对方行为背后的动机，不用面对自己的不足和问题。

**但是，你已经是一个成年人了，你拥有解决问题的能力。请相信自己，焦虑型是可以转为安全型的。**

以下是四个关键的转变步骤：

**1. 寻找或营造安全的环境**
最好的方法是找一个安全型的伴侣。当两个焦虑型相爱时，好的时候可能非常紧密热烈，但出现矛盾时，战争可能越演越烈且不知原因，关系非常不稳定。当焦虑型遇到回避型（目前最常见的一种组合），一方会显得弱势、焦虑，另一方则显得独立、强势，双方会强化彼此的负面形象，陷入感情跌宕起伏的怪圈，这对焦虑方的自尊可能造成长期损害。因此，寻找安全型伴侣，或将关系保持在安全的一端，至关重要。

**2. 学会控制情绪，将关系保持在安全状态**
当你情绪激动时，对方只能看到你的情绪，而察觉不到你的需求。如果对方也是焦虑型，你的情绪会点燃他的情绪，变成情绪对抗而非解决问题。如果对方是回避型，他会为了躲避你的情绪而被你认为是逃避问题、忽视需求。因此，在情绪要爆发时，可以回想之前情绪失控带来的负面影响，给大脑一个信号：`这个行为没用，这个沟通方式没用，我要换一种有效的沟通方式。` 持续传递这种“有害无力”的信号，有助于改变认知系统。

**3. 学习有效的沟通方式**
每种情绪背后，都有某种需求未得到满足。但无论是情绪还是需求本身，都需要通过有效沟通才能被理解和满足。沟通方式在亲密关系中非常重要。你可以向身边擅长沟通的人学习，也可以通过书籍、网络课程等途径学习。如有必要，也可以向擅长亲密关系的情感心理咨询师寻求指导。

**4. 必要时寻求专业帮助**
如果你的焦虑情况没有改善，可能是认知出现了问题，可能需要进行专业的认知行为疗法。这是一种常见的焦虑治疗方法，能帮助你识别并纠正错误的认知和思维方式，带你走出泥潭。

## 课程五：拥抱真实，缓解焦虑 💖

本节课是我们系列的最后一节，我们将一起探讨如何以更健康的心态看待感情，从而从根本上缓解焦虑。

最后希望大家了解，完美的感情，我们可以通过努力不断去接近，但一定要记住，**我们永远无法到达完美**。因为真实就意味着有缺陷，真实的感情必然是不完美的。

我们无需一味追求完美。活在当下，珍惜拥有，缓解焦虑，客观看待事实，让彼此相处舒适。这样的爱情可能没有那么理想化，但它更可能带来一种踏实的幸福感。

3.  判断自身是否属于焦虑型依恋的参考标准。
4.  从焦虑型转向安全型的四个具体路径：寻找安全环境、控制情绪、学习有效沟通、必要时寻求专业帮助。
