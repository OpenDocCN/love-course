# 恋爱关系课程 01：如何化解“追逃模式” 🕊️

在本节课中，我们将要学习恋爱关系中一种常见的负面互动模式——“追逃模式”。我们将分析其成因，并学习具体的步骤来打破这种循环，建立更健康、积极的沟通与相处方式。

很多情侣都会有这样的感受：每次和伴侣吵架，似乎只有自己一个人想解决问题，对方总是回避冷处理。即便你表明自己真的只是想解决问题，对方都一脸厌烦，表示压力大。这就是典型的“一个追，一个逃”。

时间久了，这种模式往往导致关系走向破灭。所以今天我们单独分析一下“女追男逃”的情况，希望帮助大家尽快脱离关系中的负面循环。

我们女生可能会比较悲观地看待男人逃避这件事，觉得对方根本不在乎自己，或者就是喜欢逃避。其实，哪怕再爱伴侣的人，也都会有逃避的行为。一时的行为并不能判定他这个人如何。我们更应注意是什么导致了他不停地逃避，才能治本。

## 男人为何逃避？🔍

上一节我们提到了“追逃模式”的现象，本节中我们来看看导致男性逃避行为的几个核心原因。

以下是三个主要原因：

1.  **过多的负面情绪触发防御机制**
    每个人承受负面情绪的能力都有限，就像一个容量固定的瓶子。大部分能够进入亲密关系的男人，其实不缺照顾女友感受的责任心。但一直接纳伴侣的负面情绪，日积月累，内心的瓶子总会满溢。可能女朋友再多一句责怪或掉掉眼泪，都会超出他承受的极限。此时，男人的自我防御机制就会生效，指挥他赶紧远离这些压力事件，表现出来就是像逃跑一样。

2.  **双方缺乏正向的沟通**
    安妮宝贝说过，很多人对待他人的方式就像是在清点他的行李，只是确认他在不在，在可控的范围内占有他，并非真的认真对待和享受彼此的存在。太深切的执着于爱人或恨人，都是一种自私。女人的步步紧逼会让男人发现沟通无效，次数多了他就会选择逃避。而男人持续的消极应对，也会最终错过两个人互相理解的时机。等到沟通环境恶劣到不可挽回时，逃跑的人就真的只能逃跑了。

3.  **习得了逃避的处理方式**
    我们第一次体验到的亲密关系其实是父母的婚姻。父母言传身教给孩子生存的智慧。如果他们相处的方式就是追逃模式，就会让后代下意识地模仿这种对待伴侣的方式。在后代眼里，父母都这样过了大半辈子没分开，那自己也可以用逃跑去应对冲突。还有一些人可能已经意识到了问题想要改变，但早年习得的相处模式并不容易修正。在冲突发生时，还是会下意识选择逃跑，因为这样做让他觉得很熟悉、很安全。

我们可以看出，男人逃避的初心，也是为了感情想要回避冲突。人都是追求更舒适的情感体验的，他们内心其实也期盼能够堂堂正正地面对感情中的自己。之所以深陷追逃模式，并非他们所愿，而是在漫长的冲突中，他们无法用更健康的相处模式去应对。

## 如何打破“追逃模式”？🛠️

了解了原因后，我们来看看具体怎么做。我们女生能做的就是放下追逐之心，为双方摆脱追逃模式创造合适的条件，等待他迈出改变的那一步。

以下是三个具体步骤：

*   **第一步：主动改变追逐模式**
    停下一切的追逐行为，不要逼迫他回应。逃避的人最怕你追着他发泄。一方面他纠结于想尽伴侣的义务，另一方面他内心的痛苦也很真实，所以只能去否认或掩盖。当他流露出拒绝回应的意图时，你要敏锐地察觉他的痛苦情绪，不要再尝试对他表达或追问“你为什么这么对我”。这时时机不对。你可以换个时间，比如在你们和好之后，再问他：“你当时是怎么想的呀？”先询问他的感受以解开心结，不要急着表达自己被忽略的情绪。礼尚往来，安抚好他，他才能够接下你的情绪。

*   **第二步：建立正向沟通环境**
    我们可以坦诚沟通内心的感受和想法，并尽力去理解对方的感受。最常见的例子是情人节送花。女生内心的感受可能是：玫瑰花代表了她对我的爱和关心，她没买，那是不是我在她心目中就没那么重要了？由此分析，女方浅层的需求是玫瑰花，但深层的需求是**安全感**和**爱人的肯定**。而男生的想法通常是：在一起这么久了，我对你怎么样，还需要用一朵玫瑰花去证明吗？我是平时做得不够吗？就因为玫瑰花被人比下去了？所以，送花的行为在两人心中的含义完全不同。如果你能先有坦诚、接纳、理解的态度，再用温和的话语去暗示你并没有威胁她的意思，那沟通就会顺利很多。

*   **第三步：日常给予足够的接纳与爱**
    解决冲突时的合理沟通很重要，但相处模式的养成都在日常生活中。想让男人做什么事情时，放缓语气，可以用一些撒娇的办法。事后要夸他处理得很周到、工作能力很强，你很满意，让他有成就感。或者平时对他主动关心，尊重男生的爱好。就算他拉你去打游戏、看足球、夜跑等，即使你不想或无感，也要有耐心去陪伴他。自己先做到有仪式感，再谈节日的仪式感。你自己逢年过节也可以给他一些小惊喜。这样做的好处是，当你给到他足够的接纳和爱时，对方即使有情绪也能很快消解，因为他知道你能满足他内心对爱的需求，并不需要用回避来防备你。

我特别喜欢的一本书叫《沟通的无形语言》，它把人对爱的渴望比喻成一个“爱箱”。**公式：伴侣的负面行为 ∝ 爱箱的空缺程度**。当我们无法满足伴侣对爱的渴望时，这个爱箱就是空的，会引发对方退缩回避的行为、刺耳的言行以及吹毛求疵的心态。其实这都是一些提醒：伴侣是太缺少爱的滋养，所以心生匮乏。但如果反过来，你把这个爱箱填满了，这就会让伴侣变得很宽容、很富足，甚至有动力回馈我们的关怀。这样长期下来，爱意就会在双方之间流动。

## 总结 📝

本节课中，我们一起学习了恋爱关系中的“追逃模式”。我们分析了男性逃避的三个主要原因：负面情绪超载触发防御、缺乏正向沟通、以及从原生家庭习得的逃避模式。接着，我们探讨了打破这一循环的三个核心步骤：停止追逐、建立正向沟通环境、以及在日常生活中持续给予接纳与爱。记住，改变模式需要耐心和双方的共同努力，从理解开始，用行动创造安全感和爱的流动。
