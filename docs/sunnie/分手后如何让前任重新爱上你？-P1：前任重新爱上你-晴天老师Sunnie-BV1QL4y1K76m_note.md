# 分手后如何让前任重新爱上你？ - P1：前任重新爱上你 🎯

在本节课中，我们将要学习如何系统性地重建与前任的关系，并引导对方重新对你产生爱意。我们将把挽回过程量化为一个计分模型，并详细拆解获得额外40分好感度的四个关键步骤。

---

很多求助的粉丝都有这样的想法：只要自己回归到过去双方在一起时的最佳状态，前任就能重新爱上自己。这个逻辑似乎正确，但仅这样做还远远不够。

按照这种设想，最多只能达到复联、彼此不抗拒联系的状态。如果挽回是计分制，此时只能算**50分**。因为前任能和你联系，很大一部分原因并非出于好感，而是念及旧情。

念及旧情看似是你的优势，但也是你的劣势。这说明前任对你的抵触心理并不强烈，但这并不意味着他已经放下了对你的成见。虽然你可能已达到了在一起时的巅峰状态，但在这个过程中稍有懈怠，双方关系很可能急转直下。

**50分仅仅是前任对我们产生好感的及格线。只有达到90分的好感度，才可能重新赢得前任的心。**

那么，我们如何才能多拿40分呢？接下来，我将分成4个部分进行讲解，每个部分占比10分。你们可以对照参考，其中最后一个10分尤为重要。

---

## 第一个10分：培养主场思维 🧠

上一节我们介绍了挽回的计分模型，本节中我们来看看如何获得第一个10分：培养主场思维。

主场思维需要贯穿整个挽回过程。你知道为什么“海王/海后”在感情中总是游刃有余吗？因为他们的控场能力很强。在一段关系中，他们更多关注自己的感受和标准，很少因对方的回应而自乱阵脚。

面对当下情况，我建议你先接纳对方，减轻前任的心理压力。你可以主动发送一条简短消息，表明自己状态很好。例如：
```
现在这种做朋友的状态，我感觉还蛮轻松的。
```
无论前任如何想，你表达清楚这个意思即可。这样做的目的是让前任松口气，尽快打消他的戒备心。

这段时间，你无需在意对方是否喜欢或关注你。在生活中随心所欲地做自己。在朋友圈，除了不要发消极（emo）的内容外，可以无规律地展示自己积极乐观的一面，让前任产生一种“拿不住你”的心态，从而逐渐夺回控场权。

心理学上，人对于违背自己意愿的情景会产生焦虑感。对前任而言，他没有复合意愿，当你流露出攻势，就容易激起他的防御心理。

**记住主场思维的要点：抓大放小。** 不要抠细节，也别总是留意前任的一举一动。你要让前任觉得你的确放下了，不会再纠缠。既然话说出去了，就要言行合一。

分手后，只要你带着挽回的目的，沟通中就多少会下意识地想要表现自己。但挽回是一个做减法的过程。你需要提醒自己：关系越缓和，就越需要减缓行动节奏，千万不要着急推进关系。

很多求助者有这样的困扰：只要一和前任沟通，就无法安心做自己。核心原因很可能是你的内心仍在抗拒过去被拒绝的经历。

实际上，你只需把前任看作你的异性兄弟姐妹或红颜知己来对待即可。只要你表现得无所谓，在不暴露需求感的前提下大胆畅所欲言，前任可能反而更容易接受你的存在。

最直观的体现是：聊天时，前任不会那么惜字如金，也不会总是选择性回复。你们之间可以随意开玩笑，不再拘束。

达到这个状态，基本上就拿下了第一个10分。我们可以进行下一步了。

## 第二个10分：在聊天中营造舒适感 💬

上一节我们掌握了主场思维，本节中我们来看看如何获得第二个10分：在聊天过程中营造舒适感。

当彼此能够不计前嫌地做朋友后，接下来我们要进一步深化双方关系，让联系变得更加自然紧密。核心是在沟通中营造舒适感。

以下是提供舒适感的两个要点：

**1. 调动你的幽默机制**
很多朋友在跟前任沟通时，表现得太认真，这让聊天氛围枯燥乏味，容易产生审美疲劳。你可以在聊天中说一些大众梗，比如最近流行的“刘畊宏女孩”、“王心凌男孩”、“孤勇者小孩”等。这类熟知的网络梗很容易营造愉快氛围。

此外，你可以在氛围不错时，提一些过去交往时的有趣梗，或根据聊天内容创造你们自己能听懂的新梗。这也能提升你们之间的默契值。

可能有朋友会说：“我一直很严肃，就是没有幽默感怎么办？”但其实，幽默是可以学习和模仿的。你可以多观察、多体会，或在网上看一些幽默有趣的聊天记录。同时，你也可以利用联想法聊天，营造画面感。例如：
*   提到**高跟鞋**，可以延伸到**水晶鞋**。
*   聊到**青蛙**，就能想到**青蛙王子**。

不过，幽默不是要你玩低俗的梗，凡事都需要把握尺度。

**2. 避免观点冲突**
你们之前有过交往经历，相对比较了解对方的雷点。同样，前任也会因为“废测”（废物测试），时不时试探你是否真的放下了。

所以在这个阶段，聊天中我们需要着重避免观点冲突，防止前任再次产生排斥感。不要为了一时情绪跟对方抬杠。就算吵赢了对方，输了机会，也得不偿失。

总之，你要给他营造一种轻松自由的氛围以及“抱持感”。说得再直白一点，就是他在和你的相处中，能感觉到被尊重，并享受到充分的自由。

做到以上这些，你们大概率可以成为一种“友达以上，恋人未满”的关系了。

不过，这里提示两点，是这个阶段的大忌：
1.  不要对前任提供的一点甜头就立刻上头回应，暴露需求感。
2.  不要“事短”，总是期待前任立刻给你积极的情绪反馈。

## 第三个10分：持续增强自身吸引力 ✨

上一节我们学会了营造舒适感，本节中我们来看看如何获得第三个10分：持续增强你自身的吸引力。

上个阶段我们营造了舒适感，这个阶段我们要营造暧昧的氛围。想要拿捏住对方，你需要持续引导前任付出，并在对方开始主动联系你之后，用以退为进的方式让他对你产生依赖感。

**首先，把握契机，引导付出。**
当前任主动跟你联系、主动延伸话题或经常秒回时，一定要把握好契机。只要对方没有停下来的意愿，你就持续跟他沟通。你们聊得最好时，让前任第二天都感到困倦，就是最佳效果。因为这就是前任在付出。**前任付出得越多，他就越舍不得你。**

这里有一个注意点：不用纠结最后一句话是谁说的，我们的目的达到了就行。

**其次，学会留白，制造神秘感。**
当你们发现前任开始贪恋这种感觉时，我们就进行第二步：留白。亲密关系中有句经典的话：**得不到的永远在骚动。**

你要将自身的优势展现给对方70%，剩下的30%留给对方去挖掘。这样，前任才可能在你这边感受到神秘与诱惑。

举个例子：前两天前任可能跟你聊得非常火热，今天你可以突然借口有事不聊了，或者干脆不回消息。我们也可以在朋友圈提前发一个模棱两可的内容，比如：
```
今天也是认真生活的一天。
```
配图是一桌好饭好菜，但唯独多出了一份碗筷。

这时，前任找不到你，可能会好奇你去哪里了。看了你的朋友圈后，他可能略微心安，但又发现多出一副碗筷，这又会让他胡思乱想，引起更多疑问和在意。

这里，调动对方的情绪是层层递进、持续的过程。你要让前任觉得，虽然你跟他有联系，但不会事事顺着他，你是随时可能离开的。这样才会显得令人着迷。对方停留在你身上的注意力越久，他就越容易被你吸引。

**不过要注意：不要故意去引起对方的醋意。** 这很容易影响自身形象，显得你没有边界感。

之后，只要对方没有主动询问你为什么没回复，或问你前一天干什么去了，你都不要主动详细交代。可以轻描淡写地回复：
```
有朋友来家里吃饭了，聊得太晚就没回你。
```
如果前任进一步挖掘更多信息，说明他这时已经被你拿捏了。如果他没有，我们也不用着急。这样的“推拉”多进行几次，前任迟早会顺应你的节奏来。

记住，在感情中：
*   你仅仅让一个人高兴开心，他可能会**喜欢**你。
*   但你只有让他能够为你伤心纠结、患得患失，他才可能**爱**上你。

前者是让前任心情愉悦，而后者才会让他对你难以捉摸。

## 第四个10分：具备解决问题和矛盾的能力 ⚖️

上一节我们探讨了如何增强吸引力，本节中我们来看看获得最后一个，也是至关重要的10分：具备解决问题和矛盾的能力。

很多咨询者求助，都是因为在这个部分没有处理好，导致挽回搁浅。解决矛盾和沟通能力薄弱，还是因为“托付心理”太强，总想着前任能顺应我们的态度改变。但恋爱中没学会的，挽回中我们就要做到。

随着关系渐入佳境，我们可能会不自觉地将自己再次带入伴侣角色，需求感一不小心就会暴露。你会开始跟前任提要求、耍小脾气。

但是，还记得我开头说过的吗？**主场思维要贯穿整个挽回阶段。** 你要学会抓大放小。我们的目的是为了挽回，要有大局观。越到最后，越要藏好我们的目的。

出现矛盾时，不要急于去解决矛盾，而是要分清形势，尽可能降低矛盾点。最好的方法就是：**承认对方是对的，同意对方的观点。**

挽回到最后阶段，前任多少还会刻意激怒你，引起你的不适。这绝大多数其实是建立关系前的“废测”手段，也是对你的终极考验。你表示赞同，会让前任出乎意料，更容易稳固你前期铺垫的形象。你们没有引发新的争论，矛盾点消失了，自然也会让他对你彻底改观。

任何关系都会产生对立面，这很正常。但愤怒通常都不是智者的选择。如果你实在无法认同他，只需找一个合适的理由安静就好。

## 总结 📝

本节课中，我们一起学习了如何系统性地挽回前任，并引导他重新爱上你。我们将过程量化为一个计分模型，核心目标是获得额外的40分好感度。

我们详细拆解了四个关键步骤，每个步骤价值10分：
1.  **培养主场思维**：贯穿始终，抓大放小，控制节奏。
2.  **在聊天中营造舒适感**：运用幽默，避免冲突，提供轻松自由的氛围。
3.  **持续增强自身吸引力**：引导付出，学会留白，制造神秘感和依赖感。
4.  **具备解决问题和矛盾的能力**：降低矛盾点，通过认同化解终极考验，展现成熟格局。

当你成功拿到这40分，结合最初的50分基础，达到90分的好感度时，前任想不回头都难。记住，挽回是一场精心策划的自我提升与关系重建之旅，核心在于改变自己，从而影响对方。
