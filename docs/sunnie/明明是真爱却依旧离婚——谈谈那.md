# 明明是真爱却依旧离婚——谈谈那

在本节课中，我们将要学习一种在婚姻中常见的现象——“结婚不适症”。我们将探讨其表现、成因，并学习如何通过调整自身来改善婚姻关系，而不是试图强行改变伴侣。

---

虽然有一个词叫做“七年之痒”，但许多情侣怀着彼此是真爱的信念结婚，认为距离七年还很遥远，共同生活总会越来越好。结果发现，婚后两到三年间离婚的情况最为普遍。这不仅仅是因为投入较少，或离婚后仍有资本。

这些都是可以离婚的条件，但主要还是看想离婚的动机。你会发现，大多是因为发现对方不适合过日子，在相处过程中没有感受到爱，或者婚姻生活未达到自己的预期。

从想离婚一方的视角来看，当这些问题刚出现时，他们通常比较乐观。他们相信通过与伴侣沟通，能够一起改正或协商出好的解决办法。

但最大的问题往往就在这里。他们的伴侣可能要么暂时答应却屡教不改，要么觉得不是大事，听不进建议，甚至试图让对方不要在意这些问题。一两次可以容忍，时间久了，问题就会越发明显。

而从被离婚一方的视角来看，很多人经常诉苦，指责婚姻中的种种不好。这种指责对方、不愿静心思考问题所在的态度，正是导致被离婚的缘由。没有人想无缘无故离婚，其实都是迫不得已。只是你觉得还能再忍忍、再等等，还不至于到离婚的地步。

我们可以称之为“结婚不适症”，这是一种慢性病。一个人变成了两个人，你无法适应，却依然用一个人的思维去考虑两个人的事情。

人们无法很快调节自己以适应环境变化，包括但不限于：异地上学、意外事故、搬家、结婚等。其中，情感关系的变化最为磨人，因为你无法准确找出是什么原因导致你无法与周边环境协调。而出于人的生活惯性，改变本身也很难。

许多女人在恋爱时都强调，爱一个人就必须爱她的全部。可是到了婚姻里，却总是积极地想要去改变对方，并且声称是为了对方好。

但是男人通常不会感谢一个单方面“为他好”的女人。因为他想要得到的是赞赏、表扬和肯定，而不是约束、管教和批评。如果妻子极力想要改造丈夫，丈夫就会把自己伪装起来，不再表露本性，因为他不想发生冲突。

换位思考一下，如果你的伴侣总是逼迫你按照她的要求去生活，你同样会感到痛苦。伪装一时容易，伪装一辈子很难。戴着面具过日子是最累的，长此以往，随时都可能爆发婚姻危机。

所以，如果你不想伴侣成为伪装者，就应该在婚姻中给对方留出沟通的余地，不要过于强势。如果说恋爱的主题是爱上彼此的优点，那么婚姻的主题，其实就是包容彼此的缺点。

当你得了“结婚不适症”，想要发展亲密关系，却发现无论如何都无法与伴侣的观念达成一致时，这往往意味着你不擅长处理人际关系，不擅长改变。你会觉得自己很委屈、很累。

这时，你就需要专业人士帮助你进行心理疏导并提出建议。你需要重新体验一段与伴侣的新关系，而不是陷在回忆里痛苦迷茫，或用已经固化的思维方式去应对，这样很难做出有效改变，对挽回关系也无益。

例如，当你在婚姻中体会到深深挫败时，咨询师会允许和接纳这一部分。当你有机会去讨论这部分感受，你也会得到新的见解，才能想要走出自己的封闭思维，学着去适应，并找出与伴侣不同的部分达成和解。

当情绪有了出口，生活上也就有了改变的能量。你就能用切身的行为去影响伴侣，提供情绪价值，促使他看到你想要在一起的心，从而被感染并做出改变。

也许你们不应该总想着去改造对方，因为我们只有选择的权利，却没有改变别人的能力。好的婚姻应该是让双方都更加舒适的。要在舒适的环境里努力成为彼此更好的人，而不是努力装成对方想要的样子。

所以，当你无法改变另一半时，不妨想想有没有什么折衷的方法，可以既不逼迫对方，又能让自己舒服。

爱并不是我们唯一的情感需要。根据心理学家的观察，我们的基本需要是**安全感**、**自我的价值**和**意义感**。爱与这些需要相互影响。

如果你感觉到伴侣爱你，你就可以放松，因为你知道伴侣不会伤害你。在伴侣面前，你会觉得安全。因为在工作中你可能要面对许多变化无常的事，在人生的其他方面也可能有“敌人”，但与伴侣在一起时，你会觉得安全。这其实是你放开了个人界限，允许伴侣进入。

同样，你要试着走进她的界限，互相证明对方的价值，这才能让生活更加融洽。伴侣爱你这个事实，也满足了你对自我价值的肯定。毕竟，如果她爱你，你必然是值得被爱的。

关于你的价值，原生家庭也许给了你负面或混淆的信息，但你的伴侣给了你第二次机会，让你在成年后被重新认识并被爱。她的爱建立了你的自尊，肯定了你的价值。

至于感觉“有意义”这种需要，是我们很多行为背后的情感力量。对于什么是有意义的，每个人有自己的想法，并会为此努力以达到目标。感觉到被伴侣所爱，会增强这种有意义的感觉。你会相信，如果有人爱你，那你的存在必定是有意义的。

爱并不是一切问题的答案，但它制造了一种安全的气氛。在爱带来的这种气氛里，你们可以慢慢去寻求那些烦扰你们的事情的答案。

在爱带来的安全感里，一对夫妻可以讨论差异而没有责难，因为冲突可以被化解。你们会发现如何展示彼此最好的部分，这就是爱的奖赏性。

婚姻的核心也是爱，表现形式就是为爱所做出的行为。但如果在婚姻环境里，你们感觉不到爱，彼此间的差异就会扩大。你们会开始把彼此看作是对自己快乐的一种威胁，并会为了自我的价值和意义而战。这时，婚姻就变成了战场，所以才可能闹到离婚的地步。

主动走出自己的舒适区，去做那个先改变的人，而不是想要去改变对方的人。多给婚姻提供一些“爱的供养”，才是治疗“婚姻不适症”的最佳方法。

---

本节课中，我们一起学习了“结婚不适症”的概念、表现及其对婚姻的破坏性。我们认识到，试图改变伴侣往往适得其反，而调整自身、提供安全感与情绪价值、在爱中寻求和解，才是改善关系的有效途径。记住，婚姻的目标是共同舒适地成长，而非变成对方期望的样子。
