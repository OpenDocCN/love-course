# 不合适分手了还想复合，你需要同

在本节课中，我们将要学习如何通过“同频”技巧来挽回因“感觉不合适”而分手的前任。许多分手者将原因归结为模糊的“不合适”，这常常让挽回者感到困惑。我们将深入剖析“不合适”背后的核心——不同频，并提供四个维度的具体操作方法，帮助你重建吸引，让沟通重新顺畅。

## 概述：为何“同频”是挽回的关键

很多前任会把分手的原因归结为“感到不合适”。这时常让挽回者困惑，因为分手者往往只能说“感觉不对”，无法指出具体问题，或只能举例一些小事上的意见不合。

分析后发现，分手时所说的“不合适”，很多其实是“不同频”导致的。例如，一个人看了推荐想拉对象去旅游，对象却很不耐烦地说：“这风景跟家附近的有什么区别？”又或者一个人忙到天黑回家，累到不想说话，另一半却质疑是因为不爱了才沉默。

频道不同的话，沟通了也不能互相理解。最后你会发现，就算近在眼前，两人却是两个世界的人。1970年普渡大学的研究者刻意安排社会政治观点相似或不相似的男女进行盲约。结果是，观点相似的学生会比不相似的更加喜欢对方。这说明了**同频在亲密关系中举足轻重**。

在挽回中也是如此。同频道的沟通会更加顺畅，你和前任聊起来没有隔阂，他也会更愿意接纳一个懂他的人。另外，同频具有“镜像性”。如果你和前任有同样或相似的思考方式，几乎就像是另一个环境中的他，能够侧面给他认同感。就像前任会肯定自己一样，他也会慢慢肯定你。

**所以，和前任重新同频，才是再次吸引他的关键。**

## 实现“同频”的四个维度

想要跟对方同频，我们一般有四个维度。以下是具体介绍：

### 1. 言语契合：成为“懂他”的人

在言语契合方面，很多人并不知道同频的感受要怎么去营造。其实最简单的方式就是说话用词。

我小时候很讨厌兰州的一个亲戚，他每次来我家做客都会说我“傻”。当时他和我妈说“傻得很”，两个人一起笑，只有我很尴尬地站在一边。后来我才知道，“傻得很”的意思其实是“好得很”，这才对他有了印象上的改观。

在我们平常的沟通中，人和人交流也会有这种语言上的代沟，通常是生活环境、学识认知等差异造成的。同样的语言在不同的人那里，表达的意思以及你理解的意思都可能不一样。

**要和一个人同频，你就要知道他喜欢用什么词，这个词一般是什么含义。** 如果一个词有多种意思，你要知道对方会习惯用哪个引申义，也就是这个词会带来怎样的感觉。最后你说话时，也习惯性地把这些词讲出来。当你重复他的常用词、口头禅时，他在潜意识中就会把你当成懂他的人。

我们可以把这种挽回方式简化为：用“回聲”的方式去重复他们的话。也就是要**提取对方话里的高频词汇，组成句子再复述一遍**。

我之前接过一个婚姻咨询，丈夫来挽回妻子。妻子觉得他一直听不懂自己说话，所以两人老是吵架，最后约定离婚。他找我时两人已分居，但共同照顾孩子，即使那时也常为怎么抚养孩子吵架。所以我建议他从交流孩子上去改善沟通质量：先倾听妻子说话中的高频词语，比如“教育”、“压力大”。

挽回者反馈，他主动提了两三次“教育”后，妻子态度缓和了很多。他跟着感叹“带孩子压力确实大”，妻子也就愿意顺着话往下聊。

前面说的都是语言用词上的契合。如果你能暗示前任，连两人的身体语言都非常相似，效果会更好。想让一个人感到你和他相处舒适，就要按照他的行为方式去规范你的举止。

这里要求你观察对方习惯怎么做。成长于不同环境的人，举止投足的差异都会很大。我们也接过很多为身体语言争执过的情侣。一位咨询者回忆，前任列举的“不合适”中，就有两人听音乐方式的不同：前任喜欢戴耳机听，而咨询者喜欢手机外放，觉得这是两个人在共享音乐。直到后来感情不太好时，前任特别指出，经常被外放音乐打扰到，所以认为两人很多方式都不一样，相处很累。

这怎么解决？其实很简单。**发现两人有差异的地方，不妨用前任的习惯去试一试。** 多试几次，那种行为上的陌生感也就消失了。你也可以用前任习惯的方式去对待他。

### 2. 共同兴趣：营造“灵魂伴侣”感

第二个维度就是共同兴趣爱好。电影《花束般的恋爱》里，男女主就是凭借爱好一致，认定了彼此是灵魂伴侣。这样的同频其实很好营造：多花时间了解对方的兴趣爱好，并且表现出热情的回应。

很多人都做到了第一步，关键问题可能出在第二步上面：你看上去可能并没有去充分了解这些的热情，这可能会打击到你的对象。事实上，前任如果觉得你对他的兴趣爱好不感兴趣，两人很有可能就聊不动。

所以挽回时，你最好要正视对方的兴趣爱好，甚至可以暗示他你也渐渐喜欢上这项活动了。我对象就特别喜欢股票，每天早上一起坐车时，他都会跟我讲很多金融方面的消息。但大部分名词我其实都听不懂。他每说完一段会停下来看着我，期待我的认同。如果看到我在刷微博，他会非常不爽。所以相处很久后我也学聪明了：我会在他停下来之前找机会打断问他：“哇这么厉害，那你怎么想到的？”然后他就会更激动地讲下去。

人都是有倾诉欲的。如果你和前任聊天还算愉快，比如前女友在朋友圈发了穿新衣服的自拍，你可以在下面评论“哇这衣服真的很好看”，或者指出衣服有什么特别之处，前任会更容易接话。因为在他眼里，你是对他换新衣服感兴趣的人，他的想法是可以和你分享的。

之前我们也专门讲过怎么和前任聊天的视频，话题最好选对方感兴趣的，其实用的也是同频的技巧。比如他喜欢滑雪，那你最近解锁了相关新技能，就可以从这个点入手去和前任分享，引起对方的好奇心和倾诉欲。

### 3. 价值取向：展示观念上的相似性

第三点就是价值取向相投。比如你在看社会事件时是什么反应？是悲愤激动想维护正义，还是希望自己身边的人不要再受伤害？而前任又是什么反应？和你一样吗？

与社会上性别对立的风气相反，真正在亲密关系里，两人基本的价值观、人生信念、行为模式和世界观相似非常重要。至少你们不能为此产生矛盾。很多人刚开始会说“相爱就行”，最后却因为三观不合分手，暴露的其实就是对这种同频的需求。交往后期，另一半肯定会介意此类问题，因为你们可能就是要过一辈子的。

挽回中如何展示价值取向的相似性？我们可以聊一些共同圈子里朋友的八卦，或者说重大事件新闻之类的分享，很容易延伸到价值观类的话题，给我们展示改变的机会就来了。

分手后，我们不能直接去跟对方写改过自新保证书。挽回时，偷偷道歉也容易引发反效果。但我们完全可以**通过第三方的经历，去传递我们思想观念上的变化**。以讨论朋友们发生的故事为契机，和对方分享你的看法。在这一类交流中自然流露，比你强调自己已经改变会有说服力得多。

### 4. 关系期待：调整节奏，达成共识

第四点是对关系的期待一致。这其实是男女双方对爱情和婚姻的默认需求：我们的关系应该保持多少亲密、多少距离、多少独立、多少依赖、多少给予、多少奉献。其实这种同频也是可以磨合调整出来的。

有时候我们会接到一些工作和学生交往的恋爱咨询，主要矛盾是生活节奏不同导致的错位。大学生可能不能理解对象下班了还在喊累，觉得对方是在拿累当借口拒绝陪伴自己。工作党也会觉得学生一直在闹情绪，是想要人陪，自己精力可能根本跟不上。

另外也不只是学生和上班族的恋爱。总是出差见不到面的夫妻、白班夜班交错的情侣、一个人养家的丈夫和全职在家带娃的妈妈、时差巨大的异国恋等等，在生活节奏上多多少少都会对彼此有意见。因为这种分手，双方都会有点无奈，我们知道是环境差异太大造成的不理解。

所以首先，**你要配合对方的节奏去聊天，要跟上他的节奏**。抓各种时机先把前任聊舒服了，我们再专门腾出时间，全程去配合对方的生活节奏。要让对方先感受到，其实你是能够做到让他舒服的，只是两人现在的生活节奏不允许。在他能接受的情况下，我们再聊“生活节奏不同频的状态，我们可以怎么去调整、怎么做”。这样在对方态度软化的情况下，我们的挽回才会有机会。

另外我也一直说，挽回中其实是可以用“巧劲”偷懒的。同频并不是要把你改造成和前任一个模子，而是要在对方在意的波段上面和他很像。比如你对一个前任说“我和你信仰是一致的”，在我们国家可能没有意义。但我接过的案例里，留学生和英国人谈恋爱，挽回过程中我建议他“了解一下基督可能是有必要的”，这样才能更好地理解前任信念上的差异。当然这是一个有趣的特例。

我们一般人分手也能用这个方法。比如有些前任喜欢安逸富足的生活，那你挽回时，要更多地展现你有很强的工作能力，不要让对方觉得和你在一起以后的生活，他是可以过得毫无压力的，那对方愿意跟你在一起的可能性就非常大。另外，如果你暂时没有很强的赚钱能力，那你至少得表现出一个应有的上进心和学习能力，要让对方相信你们能够有一个安稳的未来，这样挽回的成功率也会大大提高。

再说一个有意思的：男人挽回女人的话，如果对象是20多岁，那你可能就要更加注意你的外表打扮，会吃会玩会关注他的感受，可以给他提供情绪价值。但如果对方是30岁，那你可能就要尊重他的价值观，你要表现出和他的想法逻辑差不多，那你挽回时，有特别地注意去改善相处的部分，包括对家务的理解、对家庭关系的调节，不说一致至少要非常相近。年龄不同确实会导致人对情感的需求不同，前任希望你同频的波段可能也不一样。

当然也不用担心是不是现在看到已经太迟了，因为任何时候，你开始和对象同频都能取得很好的效果。

## 总结

本节课中，我们一起学习了如何通过“同频”来挽回因“不合适”而分手的前任。我们首先剖析了“不合适”背后往往是“不同频”在作祟，并理解了同频在亲密关系中的重要性。

接着，我们详细探讨了实现同频的四个核心维度：
1.  **言语契合**：通过使用对方的常用词汇和模仿其身体语言，成为“懂他”的人。
2.  **共同兴趣**：深入了解并热情回应对方的爱好，营造灵魂伴侣的默契感。
3.  **价值取向**：通过讨论第三方话题，自然地展示你们在核心价值观上的相似性。
4.  **关系期待**：调整自身节奏，配合对方的生活步调和对关系的需求，达成共识。

记住，同频的关键不在于彻底改变自己，而在于在对方最在意的方面与他产生共鸣。从今天起，有意识地在这四个维度上向前任靠近，你们的沟通将重新变得顺畅，挽回之路也会更加清晰。
