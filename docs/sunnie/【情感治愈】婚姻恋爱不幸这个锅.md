# 情感治愈课 P1：原生家庭与亲密关系 🏠

在本节课中，我们将探讨一个常见但复杂的话题：原生家庭是否应该为个人在婚姻与恋爱中的不幸负责。我们将分析原生家庭的影响，并通过具体案例了解其在不同人格类型中的体现。最后，我们将学习如何减轻这些影响，主动经营健康的亲密关系。

---

关于原生家庭的话题一直备受热议。然而，将人生的不顺与婚姻的不幸全部归咎于原生家庭，这种做法是否正确？不可否认，原生家庭确实在一定程度上对我们造成了伤害，但这种伤害并非绝对不可逆转。

希望本课内容能帮助你找到答案。著名导演李安在不幸福的原生家庭中成长。从他早期的“家庭三部曲”电影中，他便探讨了两代人的关系。李安自己说过：“我一生的内心纠葛都是和父亲有关的。”

高晓松在《奇葩大会》上也谈及他的原生家庭。他因得不到父亲认可，花了大半辈子时间才逐渐走出原生家庭的阴影。举这两个例子是想说明，并非只有你一人处在不幸福的原生家庭环境中。

因此，无需过于畏惧原生家庭。原生家庭对李安的影响，集中体现在他电影中对父亲角色的刻画。而原生家庭对高晓松的影响，则集中体现为他讨厌他人干预、爱较劲的处事风格。那么，在两性亲密关系中，原生家庭的影响又体现在何处？

我们可以先为受原生家庭影响最深的人做一个分类。

以下是第一种常见类型：

*   **妈宝男或妈宝女**：这类人认为母亲的话永远正确。在心理学中，伍志宏老师曾提出一个观点：在中国，母亲与儿子或女儿常处于“共生关系”。这意味着，明明是你选择共度余生的伴侣，但你的母亲也参与了选择过程，最终你的选择很可能就是你母亲的选择。
*   这类人在中国家庭中很常见。常说婚姻是两个家庭的事，需尊重父母意见。一旦父母不同意，双方便可能分道扬镳。此类故事屡见不鲜，你身边可能也遇到过这类人。

接下来是第二种类型：

*   **无底线的低自尊者**：以朋友Z为例。他在父母的争吵与打骂中长大，父亲沉迷赌博，母亲借酒消愁。两人一吵架便迁怒于他。为不被责怪，Z拼命学习，成绩名列前茅，但情况并未好转。成年后，他遇到对象，以为命运会改变。Z考入国内一所211院校，外表出众；而其对象高中未读完便经商。在世俗眼光中，两人并不般配，但他们仍在Z毕业后结婚生子。周围朋友不解，但Z说“他对我真的很好”。婚后生活却并不幸福：怀孕不久便发现丈夫出轨，且此后几年屡次发生，Z每次都选择原谅。这令朋友们震惊，因为学术上的他据理力争，在婚姻中却毫无自尊与底线。一次聚会，Z酒后吐真言：他很痛苦，但害怕丈夫离开，觉得自己再也遇不到对他那么好的人。他会定期买礼物讨好丈夫，明知这可能源于丈夫的愧疚补偿，却仍紧紧抓住不放。**从小生活在缺爱环境中，旁人的一点点好都会被视若珍宝，因此不愿放手，只要对方不离开，什么都可以忍受。**

第三种类型最为常见：

*   **不会表达需求与爱意的人**：中国文化含蓄委婉，父母很少直接说“我爱你”，多通过行动表达关心，批评教育也常被冠以“为你好”之名。这导致许多孩子成年后，在亲密关系中也无法直白表露情感。尽管爱着对方，却总寻找对方的小毛病，美其名曰“因为在乎”。同时，从不直接告诉对方自己的需求，总让伴侣猜测，猜错便可能闹脾气。

第四种是我们常提的类型：

*   **焦虑型依恋者**：这类人极度需要安全感，希望伴侣时刻相伴，随时报备行踪。但这仍无法满足其需求。他们也不明白究竟怎样才有安全感，因此对感情表现出饥渴状态，通常格外粘人。这是因为他们在成长过程中，父母给予的反馈时好时坏、无法预测，导致他们强烈渴望爱，又怀疑爱的真实性。

那么，我们为何要不断谈论原生家庭？

近年来，关于原生家庭的讨论越来越多。但这并非让你将所有亲密关系问题归咎于原生家庭，追究责任此时已不重要。重要的是，要知道在与爱人相处时，如何避免重蹈覆辙。

东野圭吾在《时生》中写道：“谁都想生在好人家，可无法选择父母。发给你什么样的牌，你只能尽量打好它。”抱怨与推卸不能解决问题。对于深受原生家庭影响的我们，应如何减轻它给亲密关系带来的负面影响？

以下是四个具体方法。

第一个方法是摆脱被掌控的爱情。

为何我们总在原生家庭的圈子里打转？自己在不幸福的原生家庭长大，最终又成为自己孩子的原生父母。为何难以打破这个循环？因为我们常活在被掌控与掌控他人的生活中。“多年媳妇熬成婆”正是此理。至今，在一些封建思想根深蒂固的地区，仍存在“婆婆 > 儿子 > 媳妇”的食物链，婆婆掌控一切，拥有至高权力。等媳妇当了妈，循环再次开始。因此，我们必须彻底从这种掌控中抽离。爱情是你们自己的选择。虽然“婚姻是两个家庭的事”这话不假，但主体仍是你们自己。父母可为你找到他们眼中的理想伴侣，却不能替你与对方共度一生。更多时候，我们需要站在伴侣一边，因为这才是属于你们的小家。人生属于你自己，爱情也是。旁人无法替你感受亲密关系中的喜怒哀乐。

第二个方法是，不把伴侣当作救世主。

在原生家庭中受过苦的人，常渴望找到慰藉，将未曾拥有的爱寄托于另一半。但伴侣并非爱神，也不是你的治愈良药。你们因相爱而在一起，不要奢求只有单方面付出。你可以向伴侣寻求帮助，因为伴侣需要同甘共苦，但绝不能将其视为救世主，指望他永远无私奉献。

第三个方法是克服低自尊心理。

儿童时期，父母有意无意忽视孩子的情感需求，容易造成低自尊心理。孩子长大后，这会在亲密关系中体现：他们总觉得自己不被重视，不敢轻易提要求。遇到一个对他稍好的人，便会像抓住救命稻草般拼命抓住，哪怕低声下气。改变这种心理确实不易，毕竟是从小养成。但为了亲密关系的和谐，我们也要试着迈出第一步：不要总想“他是不是不爱我”，不要沉浸于负面情绪。试着寻找伴侣爱你的证据。这个改变无需操之过急，可以从生活点滴观察中，慢慢重建自信心。

第四个方法是直面原生家庭带来的伤痛。

你需要与伴侣坦诚沟通，告诉他你为什么在感情中缺乏安全感、为什么爱发脾气、为什么占有欲强。彼此分享童年经历，让他了解原生家庭给你带来了哪些负面影响。这样他会更理解你，知道你的行为有因可循，而非无理取闹。但不要因为他的包容，就理直气壮地不做任何改变。原生家庭不能成为你任性的借口。你需要反思：为何总吵架？为何安全感永不满足？而不是将责任推给另一半或原生家庭。你们需要共同面对问题，想办法克服原生家庭带来的伤痛。

人生总不完美，原生家庭的影响只是其中一部分。所以，不要害怕面对它。我们已是成年人，要为自己的生活与爱情负责，不要一出问题就责怪原生家庭。有句话我很喜欢：“我命由我不由天。”代入两性关系，即：美好的两性关系，需靠自己去争取和经营。不要让原生家庭阻碍你寻找爱的步伐。如果原生家庭没给你足够的爱，那就自己去创造爱，主动为自己铺就一条通往幸福的路。

---

本节课中，我们一起学习了原生家庭对亲密关系的潜在影响，识别了四种常见的受影响人格类型，并探讨了四个减轻负面影响、主动经营关系的方法。记住，认识影响是第一步，而如何行动则掌握在你自己手中。
