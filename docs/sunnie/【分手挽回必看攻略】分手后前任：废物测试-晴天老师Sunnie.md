# 情感修复课程 01：识别与应对“废物测试” 🧪

在本节课中，我们将要学习亲密关系（尤其是分手挽回阶段）中的一个核心概念——“废物测试”。理解并有效应对它，是重建关系的关键一步。

## 概述：什么是“废物测试”？

在恋爱中，女性经常会给男性出一些“送命题”。例如：“你妈妈和我掉进水里，你先救哪一个？”“游戏重要还是我重要？”“工作重要还是我重要？”“没小礼物行不行？”“没小礼物你养我？”这些都是**废物测试**的一种，考察的是男人的情商和求生欲。

**废物测试**是女性在进化中形成的，对于男性的择偶筛选系统。她会通过大量的废物测试，来判断眼前的男人是否有价值，是否值得在一起。而在挽回中我们发现，无论男女都喜欢用废物测试。当对方决定和你分手的那一刻，其实大部分人是没有通过这个测试，被判定为“废物”了。

大家不要觉得“废物”这个词不好听。实际上它通俗可以解释为：**这个人不满足我合格伴侣的要求**。所以，当对方还愿意理你，愿意重新测试你的时候，其实你挽回的机会已经出现了。

很多时候你都意识不到，对方给你的是测试。对方一句话或者一会儿没理你，就能让你崩溃。其实只要你答得稍微好点，重新通关这个废物测试，“补考”及格了，挽回也就通关了。在挽回中，废物测试既是个“坑”，也表明对方对你有点兴趣或者需求了。这也是广大挽回者的福音：你已经被对方从“废品”收回重测了。

所以，我们一定要学会的是：**如何破解对方的废物测试**。

## 分手初期的常见测试与应对

在刚分手阶段的时候，往往会有这么几个常见的废物测试。你要是能通过，就会为你挽回打下很好的基础。

以下是几个典型场景及应对策略：

1.  **测试话术**：“你不要再缠着我了。”
    *   **错误回复**：“可不可以不要不理我？”“不要离开我。”
    *   **参考答案**：“不好意思让你感到困扰了，我只是有那么忍不住的想和你分享。以后我会注意分寸的。”

2.  **测试话术**：“我不爱你了。”
    *   **错误回复**：“我不信，你以前不是这样说的。”“我哪里做得不好，你说出来我改还不行？”
    *   **参考答案**：“我明白自己有很多不足。爱不爱我是你的自由，我也不会强求你现在爱我的。”

## 复联初期的核心：避免过度暴露需求

上一节我们介绍了分手初期的直接测试，本节中我们来看看关系稍有缓和后的陷阱。

复联初期常见的废物测试是“是否过度”。也就是说，对方愿意回复你了，但你因此就摆不正自己的位置，稳不住，过度地暴露需求感。过度的热情、过多的废话、过度的情绪化，这些迹象都通通表明：你没有真正的改变，还是目的性太强，还是不顾他的感受。很容易就会再次被pass掉。

我们复联前期一定是追求 **`低频率，高质量`** 的沟通。这个时期必须稳住心态。

## 关系进阶：理解“选择性服从”

如果感情渐入佳境了，甚至进入暧昧状态了，测试就关乎两个人相处中的细节了。要的就是你要能够读懂对方心里想要什么。

最为普遍的就是**服从性的废物测试**。意思是说，对方提出一个要求，看你的服从度。当然，到了这个阶段，对方已经在给你机会了，挽回几率已经很大了。但服-从度并不是越高越好。

以下是关于“服从度”的平衡艺术：

*   **服从度太高**：就类似于“舔狗”了，对方会自动地将你归为废物一类，觉得你低价值。
*   **反服从行为**：对方也会讨厌完全拒绝的行为，因为人都不喜欢被拒绝。如果对方有任何的需求，你都不能主动及时地起到一定的作用，他就会觉得你无价值、不诚心。

前者使人厌倦，后者使人心冷。所以当对方已经能主动地找你分享他的事情，透露自己的需求，甚至直接对你提要求的时候，正确的方法应该是**选择性服从**。言下之意就是：你不能无条件服从，要在有条件的基础上服从，或者说涉及到原则底线的事情，可以偶尔拒绝的。但对方真正需要你的时候一定要出现。

`无条件服从 -> 容易处于低位，沦为“废物”`
`有来有往的选择性服从 -> 更容易感情升温`

很多人在挽回的时候，不自觉的就把自己的姿态放低了，不敢拒绝对方的任何要求，怕他生气。但实际上，人会莫名地被“拒绝”吸引。我们失去或者被拒绝的时候，大脑所受的影响是得到满足时的数倍。所以在你拒绝他的时候，他的大脑会更多地被你占据，会想“你为什么…”，或者“居然会拒绝他呢”。很多“逆袭”追到女生，也就是学会用这样的招数。因被拒绝而吸引，这是人的天性。

所以，即使自己需求感再强，挽回中也还是要保持自己的框架的。我们想要一件东西，大多数人都是不会主动要的，而是希望对方主动察觉到，并且主动给。从前是他觉得你不懂他，你自己也这么觉得。如果你能达到如此的细致入微，现在他就会认为，世界上再没有人比你更懂他了。

## 心态与提升：超越测试的本质

废物测试往往都能破解，考验的只是你基本的情商以及两性相处的能力。如果你真的改变提升了，废测可以证明你不再是“废物”了。当你对自己的能力没有底气，才会被测出来真的是“废物”。

大多数人其实还是心态问题。平时在生活中也没有那么“废”，只是面对前任要么太过情绪化，要么太过小心翼翼了。

还有，我说的成长提升，不是非要达到对方的“标准”的。比如他跟你说：“我不喜欢不成熟的男人。”一般男人会怎么回复呢？“我会努力的”“我会改的”“我会变得成熟的”。其实你甚至可以这样子调侃他：

> “男生太幼稚，确实会让女朋友很累。可我虽然想懂自己女朋友，但我也不喜欢当人家爸爸。”（这里就是暗示了他也很幼稚）

你的态度可以是：**没错，我是无法达到你的标准，但是我是我自己，我有我其他的优点，只是你还没有看到。** 挽回中不要过度地捧高对方，而贬低自己，这样会让对方觉得他的分手是对的，确实是因为你配不上他。

但是你的回答一定要因人而异。有的人废物测试的时候，倾向于服从性多一点，有的人倾向于反对服-从性多一点。这个要看你们曾经的矛盾是：你亏欠对方多，还是因为吸引力不够，还是伤害太多呢？

## 总结

本节课中，我们一起学习了“废物测试”这一核心概念。我们了解到：

1.  它是关系中的一种筛选机制，尤其在挽回阶段频繁出现。
2.  应对不同阶段（分手初期、复联期、暧昧期）的测试需要不同的策略，核心是**保持框架**与**平衡需求感**。
3.  关键在于**心态稳定**和**选择性服从**，而非一味讨好或完全抗拒。
4.  真正的提升是成为更好的自己，而不是机械地满足对方的所有标准。

废物测试其实在两性相处中太常见了，我也不能所有的回答都一一举例。之后有时间我会更新这个话术系列。现在如果不知道具体怎么回复前任才能通过废测，可以分析具体情境来寻找思路。
