# 情感修复教程 01：理解共情能力与分手挽回 🧠

在本节课中，我们将要学习共情能力的概念、水平划分，以及它如何影响亲密关系。特别是当伴侣因“共情能力低”而提出分手时，我们该如何理解问题并找到挽回的方向。

## 测试你的共情能力 📊

首先，我们可以通过一个简单的测试来评估你的共情能力。请先思考以下情境，并给出你的回答：

> **情境**：你的朋友失恋了，非常痛苦，向你倾诉。

以下是五种可能的回答，对应着五种共情水平：

1.  **“你太矫情太脆弱了，失恋有什么好难受的，谁没有失恋过。”**
2.  **“我都失恋好几次了，不过都是我甩别人。”**
3.  **“过一阵子会好的，不要难过了。”**
4.  **“你应该试着理解对方为什么这么做，为什么你们会分手。对方拉黑删除你是因为你没有承认你们已经分手的事实，还去纠缠他。”**
5.  **“你还爱他不想分手，所以他拒绝和你联系，因为这个会让你痛苦。你以为你们再也没有机会了，所以你觉得痛苦。但是你们以前感情很深，我也觉得你们这样分开太可惜了。你可以先给对方空间，这段时间先找到你们感情的问题，改变自身。之后联系方式还是可以加回来的，何况你们还有别的联系途径不是吗？挽回还是有机会的。”**

**第五类回答属于共情水平5**。作为心理咨询师，共情水平一般要求达到5，这样才能与咨询者逐步建立信任关系，有效解决问题，帮助对方走出痛苦。

## 共情能力的重要性与普遍问题 🔗

上一节我们介绍了共情水平的划分，本节中我们来看看共情能力在生活中的实际影响。

在生活中，很多人的共情水平较低，并且不明白为什么自己说的话总让别人感到不舒服，或者在平等关系中，仍然无法通过沟通说服对方接纳自己。

在与伴侣或朋友相处时，有些人习惯先否认他人，只表达自己。他人感觉不到被尊重和接纳，自然也就不愿倾听。例如，在情侣相处中，当女方哭泣或发脾气时，很多男性一开始可能会吼叫，但并未理解对方情绪产生的原因。虽然可能暂时安抚了情绪，但次数多了，男性会开始烦躁，说出“你烦不烦啊”或直接冷落对方，导致对方彻底失望。慢慢地，对方就不愿再在你面前展现情绪，你们的情感纽带也就被阻断了。如果不及时修复，分手是必然结果。

## 共情能力的个体差异与选择 🤔

共情能力低如果未对生活产生重大影响，不过度在意他人感受和看法，也能让人活得更轻松。并非只有高共情才是更好的，前提是保持善良。寻找伴侣时，你也可以找与你一样不会想太多、性格大大咧咧的人。

但如果不巧，你喜欢的人恰恰比较敏感、共情能力高，那么建议你努力提高自己的共情能力，否则你们的感情会很危险。

## 什么是共情能力？ 📖

人人都有共情能力，只是水平有高低之分。有些人的共情天赋尚未被激发。

共情能力的意义非常重大。它能够穿透人的痛苦和恐惧，找到我们生而为人的共通之处，使得人与人之间产生更深刻和紧密的连接。

用通俗的话说，**共情能力就是能够体验他人的内心世界，就好像是你自己的内心世界一样**。

由于遗传基因和社会要求不同，男性的共情能力普遍比女性低，因为男性更注重理性思维，而女性更注重情绪感受。

## 共情能力的成因与类型 🧬

有一类人天生共情能力低，这类人通常属于回避疏离型人格。他们极度理智，共情能力缺失，在反社会人格（犯罪人格）中也比较常见，因为这可能导致个体产生冷漠或攻击行为。自身很少有疼痛经历，或童年有过度的痛苦经历，都容易让人在长大后共情能力低。童年时期，养育者对自身情绪感受的忽视，也容易让孩子长大后共情能力低。但一般都属于正常的低共情范畴。

另一类人天生共情能力高，适合从事心理咨询师、疗愈者等工作。

## 高共情与自我边界 ⚖️

在人际关系中，我们需要较高的共情能力。但是高共情很多时候也伴随着高敏感，对他人痛苦感同身受，总是站在别人角度考虑问题，往往很容易忽略自身。

人都需要共情能力，但也需要自己的框架和底线，要保证自身不被打扰和损害。

## 共情能力低的具体表现 📝

以下是共情能力低的一些具体表现，你可以对照查看自己是否具备这些特质：

*   难以识别和理解他人的情绪信号。
*   在对话中经常打断别人，专注于表达自己的观点。
*   对别人的痛苦或快乐反应平淡，甚至无法理解。
*   在冲突中，习惯于指责对方而非尝试理解对方的立场。
*   经常说出让场面尴尬或伤害他人的话而不自知。
*   认为逻辑和道理比感受更重要，并以此要求他人。
