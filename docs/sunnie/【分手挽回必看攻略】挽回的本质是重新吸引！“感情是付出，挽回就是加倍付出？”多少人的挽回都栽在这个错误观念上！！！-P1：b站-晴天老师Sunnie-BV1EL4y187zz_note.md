# 挽回课程01：挽回的本质是重新吸引 💔➡️💖

在本节课中，我们将要学习情感挽回的核心本质。许多人误以为挽回是加倍付出，但我们将揭示，成功的挽回关键在于重建吸引力。我们将通过三个核心法则来理解并应用这一原理。

---

## 概述：为什么付出无法挽回？

对方铁心分手的原因，并非你付出不足。许多人认为感情是付出，因此挽回就是要不断找对方聊天、送礼物、送关心。

然而，感情的本质其实是**吸引**。这不只适用于恋人，朋友关系也是如此。除了父母，长期陪伴在你身边的朋友，并非因为他们的付出而成为朋友，而是因为他们身上总有某种可贵的价值或磁场吸引着你。

无论是友谊还是爱情，都以相互吸引和相互喜欢为前提。

---

## 吸引力法则一：成本 🧮

上一节我们介绍了吸引力的核心地位，本节中我们来看看影响吸引力的第一个关键因素：**成本**。

心理学上的吸引力法则指出，你对一个人有持续的吸引力，他才会愿意与你保持长期关系。成本，是指对方和你在一起是否“费力”。

**为什么距离越近越好挽回？**
因为距离能够降低感情成本。如果要去挽回，距离越近就越好挽回，并且常常不需要断联。例如同一屋檐下、同一办公室或同一个班级。

异地恋容易分手的原因，也在于恋爱成本太高。你需要付出更大努力去证明这段感情能带给彼此足够价值，才能维持。

我们很多时候都需要身边一直有这么一个人：想要聊天倾诉时，那个人就在身边；想要出去走走，也只需问一下就能得到答案。

因此，判断两个人感情基础深不深，除了看在一起多久，还要看真正待在一起的时间有多久。往往感情基础越深，矛盾也可能越深，伤害也越深，需要花费更长时间和更多精力去解决。但只要坚持下来，都能挽回成功。

比起相距遥远的人，和附近的人相处成本更低：更少的时间、更少的金钱，以及对未来的不确定性也更少。感情需要灵魂的靠近，而首先生活上的交集能够为你们的吸引拉近距离、降低成本。这是因为彼此触手可及，所以才能相处起来毫不费力。

**所以，挽回的另一个关键元素是：和你相处起来不费力、不心烦。**
这一条主要针对两人地理距离较近的情况。如果你的沟通让对方感到费力，甚至不想与你交流，那就需要好好反思自己。正如咨询需要彼此沟通良好、彼此信任，你的前任需要的伴侣也是如此。

## 吸引力法则二：简单暴露法则 📱

理解了成本对吸引力的影响后，我们来看看如何在分手后通过“曝光”来重建吸引力。这就要讲到构建吸引力的第二个法则：**简单暴露法则**。

当一个人在我们面前反复曝光时，我们会提高对这个人的喜欢程度。

一项美国心理学实验中，4名具有同样吸引力的女性研究员假扮成学生，在一学期内以不同上课次数出现在同一门课上。学期结束时，教室里的学生为这4名女生打分。结果显示，学生看到某个女生的次数越多，就认为自己喜欢她的程度越高。

因此，当你频繁地出现在别人面前时，你可能正在让别人更喜欢你。当然，反之，如果是不喜欢的人或事物反复出现，也可能提高你对她的厌恶程度。

**以下是应用此法则的注意事项：**

*   **避免信息轰炸**：我们往往建议你断联，或至少不要持续不断地出现在对方面前进行信息轰炸。刚分手时，对方处于对你的“情绪否定期”，死缠烂打只会加深厌恶，降低你的吸引力和价值，让对方觉得离开你是最正确的决定。
*   **管理朋友圈内容**：如果你分手后常在朋友圈发负能量内容或毫无意义的转发，很容易被屏蔽。现在连微商都知道发朋友圈要写段子或提供价值，且不能频繁刷屏引起反感。给别人带去困扰是不好的，将自身失恋的低落、愤懑等负面情绪带给别人也是一种困扰。就像心理咨询师当情绪垃圾桶需要付费一样，因为负能量本身也在消耗他人。
*   **提供情绪价值**：朋友愿意倾听你的烦恼，是因为你们之间有相互的感情关系。而分手后的恋人如果做不成朋友，就没有了这种相互关系。吸引力价值的关键是你能给对方带去**情绪价值**，让对方觉得你的存在是美好的，和你在一起比他一个人或跟别人在一起更开心、更快乐。这个价值比外形、事业、财富上的价值都重要得多。

情绪价值与一个人的情商挂钩，包括了情绪处理能力、沟通能力、共情能力。之所以叫价值，就是你要能够传递高情绪能量给对方。首先你自己一定要有足够的正能量，必须先从失恋的悲观消极情绪中走出来，学会用正面的态度和预期去处理解决问题。

总而言之，挽回前任时，在少数朝夕相处的情况外，你要增加在对方面前的曝光度，发朋友圈是必须的。至于怎么发、发什么内容，是一门很大的学问。

## 吸引力法则三：共振 🎯

学会了通过曝光增加好感后，我们还需要让吸引力持续和深化。这就引出了影响吸引力的第三个因素：**共振**。

很多人容易被和自己差异大的人吸引，因为对方身上有你自己没有的东西。但很多人分手也往往是因为三个字：**不合适**。

吸引既需要你自身的独特性，更需要两个人有相似性，才能达到更多的磁场共振。谈起挽回时，我们常说要“投其所好”。

在一项关于室友的研究中，具有相似个性的室友会对他们的关系更加满意，并且下一年继续住在一起的几率也更大。在人际关系中，人们倾向于喜欢那些与他相似的人，包括相似的价值观、态度、兴趣、背景等。每多一个相似的地方，都可能提高那个人对你的好感度。

很多人已经懂了建设吸引力的方法，要去引起对方注意，但聊天仍然找不到话题，导致感情无法进一步升温或维持长期吸引力。

**要解决这个问题，我们需要寻找彼此更多的相似性，创造更多的交集以及共同话题。**
想象自己成为他，你才能更懂他。理解其实是比吸引更加难得以及重要的，吸引力只是开关而已。

我们在选择伴侣时，不仅要考虑对方有没有魅力，想要长期相处的话，还要考虑最终能否跟这个人展开一段你会享受的关系。

## 总结与核心心态 🔑

本节课中，我们一起学习了挽回的本质是重新吸引，并深入探讨了影响吸引力的三个核心法则：**成本**、**简单暴露法则**和**共振**。

最后，我们需要树立正确的挽回心态。挽回圈流行两句话：“你不爱了，他就回来了”、“你不怕失去了，他就不会离开了”，但99%的人都做不到。

之前提到分手时，我反复提到过一个词叫“沉没成本”。我遇到的案例中，很多在一段感情中投资越多的人，在分手后也会比对方更加放不下。

所以我们挽回和谈恋爱一样，重点不是付出，而是吸引。“爱就付出”这个几乎形成本能的思维，应该和我们幼时从父母那里接触到的获得爱的方式有关。

喜欢一个人，我们大多数人都会愿意为对方付出。但我们挽回要的是什么？我们要的不是加倍去付出，而是要让对方重新喜欢你。所以，你是要**让对方去为你付出**。

很多人总想感情也要秉持礼尚往来的原则：“我对你付出越多，那你就要为我付出越多”。其实这是没有道理的逻辑。对方不想为你付出，所以不接受你的付出，这并没有错。一切都是你自己的选择和心甘情愿。

获得爱最有效的方式，是要做到自己值得被爱。和自己相处得好，你才能和伴侣相处得好。最好的结婚伴侣其实是你自己。你的挽回，其实更多的是在和你的情绪相处，和你自己相处，而不是别人。

在挽回中能够戒掉自己的情绪，戒掉自己的怯懦以及患得患失，并且坚持下来的人，很难不成功。要有吸引力，还需要一颗强大的心脏。挽回是要成长和提升的，这意味着你要走出你的舒适圈，而且不要害怕失去谁。

那么，我们今天关于挽回吸引力的内容就到这里结束了。
