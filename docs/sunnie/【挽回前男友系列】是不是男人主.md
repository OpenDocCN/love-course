# 【挽回前男友系列】是不是男人主

在本节课中，我们将要学习一个核心问题：当男生主动提出分手时，这段感情是否真的无法挽回了？我们将通过分析男性思维、常见错误以及正确的应对策略，帮助你理解情况并找到挽回的希望。

我们一直听说的都是，女朋友如果说分手的话，往往不是真的想分手，但是男朋友说分手的话，一定是真的想分了。从有一些来咨询女性挽回者的故事，我也发现了，很多女孩子说分手，其实只是可能被忽视了。

需求没有得到满足，并不是真的想分手。但是她们都提到了，前男友提分手的时候，浑身上下都透露着冷漠和疏离，似乎是真的想要你赶紧走。所以她们最恐惧的是，是不是男生如果主动提分手了，就真的没有挽回的可能性了。

其实也不是这样。男生其实没有你想象的那么强硬，他们大多都很耿直也很单纯。只要你能够找到撬开矛盾的那个点，那么挽回男生还是很大希望的。所以今天我就要教给大家，如何正确地去挽回你的前男友。

## 第一点：姿态要放低，但别太低

上一节我们介绍了男生提分手并非绝对无法挽回。本节中我们来看看挽回时的第一个关键：姿态管理。女生在挽回前男友的时候，很容易走两个极端。

以下是两种常见的错误极端：

1.  第一种是想着，反正每次都是他先低头的，这次他肯定也会低头的。于是就赌气说，好啦那你走了永远就别回来了。
2.  第二种是他刚提出分手，女生的大脑运算机制瞬间就崩盘了。在一堆乱码的驱使下，她可能不管不顾了，立刻就拉着前男友不放，不断地哭闹，求求你不要走，你说什么我都改。

但是这两种极端，都会导致无法挽回的结局。

一般来说，男性思维更偏理性线。面对矛盾和冲突，他们也更为隐忍。所以一旦男性他主动提分手，通常都是在深思熟虑之后做出的决定。他们可能是真的疲倦了，也是真的想要分开一段时间了。这个时候你还以为自己是小公主。

对着他大发脾气，对不起，这个只会让他更加厌恶你。而如果你想感情用事，想要靠疯狂地输出自己的爱意，来挽回对方的停留，也很抱歉，因为这样的停留片刻也不会有。所以正确的做法就是我这个标题说的。

**不要不敢放低姿态，但是也别太放低姿态。**

卓文君看到司马相如在外面花天酒地，失望不已，于是留下了一首《白头吟》和诀别书，就离开了司马相如。诀别书的最后几句是这样的：“白头吟，伤离别，努力加餐勿念妾。锦水汤汤，与君长诀！”就凭着这一诗一文，她又成功地挽回了司马相如。

我们可以分析一下这里，卓文君她的姿态是怎么样的？不卑不亢又温情犹存。嘴上说着我们永远分开别见面了，但是又补了一句要多吃饭别忘了我。这个手段若即若离，欲迎还拒，男人就最吃这一套了。

## 第二点：理解他的行为，而非对抗

理解了姿态管理的重要性后，我们来看看分手后男生的几种具体行为，以及如何正确解读。有些女生会遇到这样的情况，男生一分手，就立马拉黑了她的微信。这个时候她就更着急了，会觉得要是不立马挽回，就错过了最佳时机了。

其实这样的想法做法也是不对的。男生他拉黑删除是因为他真的心烦了。回想一下你们在一起的时候，你是不是隔几分钟就要问他一句在干嘛？如果他一两个小时没看手机，是不是就会接到你的夺命连环call呢？

还要先哄你一顿。如果你事事都体谅对方了，他肯定不会把你删除拉黑的。他拉黑删除你只有两种可能：

1.  第一种可能就是你纠缠打扰了他了。
2.  第二种可能就是因为他预判到了，如果不拉黑删除的话，可能就会出现让他为难的情况。

所以为了回避这个问题，他就觉得先拉黑冷静一下再说。那遇到这种情况我们该怎么办呢？这里我们先不着急解决这个问题，因为它跟我视频后面提到的第三种情况，还有一定的联系，所以我们往后放一放。

还有就是有一些男生，他会在分手之后邀你最后再吃个饭，或者看场电影，体体面面和平分手。这种情况你就大大方方去。前男友会这样做，通常是因为对你还有点不舍，或者说多少有几分歉意。

如果你能够优雅从容地出现在他面前，表现出你对待分手的理解和大度，那么可能都不用吃完这顿饭，你们就能够重回朋友区。这样的话后面挽回的事情也会顺利的多。

## 第三点：有效沟通，寻找需求平衡点

面对男生的行为，除了理解，沟通更是关键。但这里的沟通不是争吵。面对男生提分手女生还容易走两个极端。

有些姐妹可能会说的，我是想跟他讲道理，但是他说的那些东西我并不认同。你不认同，那是因为你没有了解对方想跟你讲道理背后，其实是想要跟你沟通。而决定你俩沟通走向的关键，是你们对彼此的需求了解吗？

*   **男方的需求**：不想自己比女方差；希望得到女方的陪伴。

## 第四点：恨意与遗忘的真相

在沟通中我们学会了寻找需求。但有时，对方表现出的可能是强烈的负面情绪。有一些女生在分手后还可能会遇到这种情况，男生对她说：“求求你了放过我吧，我要再喜欢你了我就是傻。”

女生听到这种话一般都会难以接受。那个曾经说要陪你到老的人，那个曾经深夜跑三条街给你买药的人，为什么如今会恨你到这个地步呢？

伦敦大学的心理学家塞米尔，曾经做过一个脑电波的实验。他的团队试图研究爱和恨的关系，结果令人震惊地发现，爱和恨他在脑区中发生的位置几乎是重叠的。这个说明了什么呢？

**一个人曾经越是爱你，那么分手过后他做的事情可能会越决绝。** 他会这么说恰恰说明他曾经爱过你，爱你爱到无力了，他才会不由自主地说这样的话。男生情绪化说明他还没有彻底放下你，你其实还是有机会的。

接着我们再来解决我们前面提过的那个，男生他删除拉黑你了，是不是意味着他要遗忘你？我很遗憾地告诉大家，男生如果这样做，他确实是有这个打算的。但是这也并不意味着就没有挽回的可能了。

## 第五点：制定挽回行动计划
