# 课程名称：亲密关系中的边界与沟通 🧩 - 课程编号：P1

在本节课中，我们将要学习为什么在亲密关系中“管得太严”反而可能导致关系破裂。我们将探讨其背后的心理动因、行为模式，并学习如何通过调整沟通方式和建立边界感来改善关系。

## 概述：管束背后的困境

最近有一个案例值得分享。一位妻子求助，她的丈夫嫌她管得太严，两人频繁争吵，婚姻已到破裂边缘。这位女士想不通，她提出期待和要求是为了让关系变好。

从她和丈夫的聊天记录看，丈夫确实很不好受。妻子对任何事情都要干涉，脾气暴躁。丈夫在外面喝酒或晚归，她会生气到反锁房门。男方稍加反对就会引发争吵。

两个人都很累。妻子可能很顾家，但这阻止不了男人想要离婚的念头。很多夫妻都陷入这样的困境，其实是掉入了一个陷阱：认为男人是“管”出来的。严重地说，我们的社会传统就有这样的风气。

结婚时“刁难新郎官”的流程，也暗示日后妻子要把丈夫“拿捏住”。这些都是老一辈传下来的方略：要抓话语权，要抓财政权。为的是以后如果真有什么事，能有个退路。

我们自己也可能认同要管住另一方的想法。毕竟本意不是伤害对方，而是努力让伴侣朝着更好的方向前进。这种想法的出发点是好的，都是为了让自己在婚姻中更幸福。但如果真的这样去做，结果却未必好。

## 管束行为的本质：利己而非利他

上一节我们介绍了管束行为常见的出发点，本节中我们来看看这种行为的本质。

这种“管”并非利他行为，而只是利己行为。当你开始管另一半时，你的做法可能包括：

以下是几种典型的管束行为模式：

*   **喋喋不休与吹毛求疵**：不断纠正另一半的行为。
*   **不断批评**：批评他让你恼火的习惯，并讲大道理。
*   **归咎责任**：认为婚姻出问题都是他的责任。
*   **双重标准**：很多事情自己做可以，自己有苦衷；但伴侣这样做，就大动肝火。
*   **道德绑架**：不断吐槽他让你多难受，潜台词是他本该避免这些行为，为什么还要给你施加痛苦。
*   **展现无力感**：在他面前祈求神灵或权威来代表你的正确性。例如转发“专家说幸福的婚姻需要对方怎么做”。
*   **强调付出**：提醒另一半你为他所做的一切，督促他该如何做才能回报你。

丈夫作为成年人，能够意识到你的要求是出于“自律”。他也会和周围人的婚姻质量做比较，从而发现你是在剥夺他正常的生活体验。

## 从“标准”到“诉求”：调整期望

当然，妻子希望跟丈夫约法三章，为家庭做财政规划，或让丈夫改掉不健康的生活习惯，这些诉求是合理且可以理解的。

但是我们要意识到，这是一个**诉求**，而不是一个**标准**。诉求是希望对方能够顺着自己，但如果对方没有做到，并不是对方的过错。

既然是诉求，就要抱着协商或征询的态度，寻找双赢的方案。例如：

以下是几个将“标准”转化为“协商”的例子：

*   **财政管理**：如果你让他每月上交工资，那你也需要为他的一些休闲娱乐（如烟酒）提供预算。
*   **生活习惯**：如果你希望家里没有烟味，那么对他偶尔在阳台抽烟的行为，可能需要视而不见。
*   **陪伴需求**：如果你渴望他的陪伴，那也要理解他拥有自己的娱乐，并尝试与他一起体验。

夫妻二人从来不是对立的。有些人会反对：对方爱玩、爱打牌，也不怎么管家事，难道我也不该管吗？或者抱怨：“我可不是为了这样的生活才结婚的。”

确实，大部分人结婚时抱有基本期待。即使认识到另一半不可能完全符合期待，我们也只是做出某些妥协。只要他好好履行责任，我们也能给予有限度的包容和爱。

说句实在话，如果你真的讨厌他，那就不是他想离婚，而是你会干净利落地去民政局了。你想要管他的前提是，你对他有需求，希望他为你留下，为你变好。

## 解决方法：超越“管教”的思维

那么，如果自己真的无法接受他的某些行为，除了“管教”，我们还有什么方法能解决呢？

**第一个要辨明自己的诉求是否合理。** 如果你把自己不能接受的伴侣行为，一律直接定义为“不好的、必须改变的”，这本质上是一种由不安引发的心理诉求，你渴望掌控一切。

这就涉及到一个心理学概念：**边界感**。

所谓边界感，是一种区分“自己课题”和“他人课题”的能力。亲密关系会给双方一种“共生”的错觉，模糊彼此的心理边界。例如，有的人觉得“我不开心，另一半也不能笑”；或者期待伴侣能有“读心术”，能看穿自己的小心思。

在这类人的理解中：“我不明白你做某些事情的逻辑，你却沉浸其中，这让我非常不安。于是，我需要你不断向我解释和证明。” 这种逾越个人心理边界的期望和要求，在亲密关系里就是不合理的诉求。

如果你的诉求确实是合理的，能否有方法让他改变呢？我的观点是：**你改变不了他，但是你可以给他一个易于改变的环境。**

人们很难被他人改变。他们可能会调整行为去适应环境，但不会从本质上改变自己的性格或自我。男女都是如此。

举个例子：妻子总是喋喋不休地唠叨，丈夫以离开作为威胁。妻子可能因此停止唠叨。丈夫觉得妻子改变了，问题解决了。但千万不要觉得对方照做你就成功了。他可能仅仅是伪装，并非发自内心、由内而外的改变。这些到最后都可能爆发。

我们大多数人期望的都是自发的改变，最终指向对自己有利，或对双方都有利的共赢。例如，另一半领悟到他的唠叨是一种防卫或操纵机制，会妨碍亲密关系，于是自发减少这种行为。这就是我们说的“变成熟了”。

关于这一点，我多年前看过一本日本小说，它形容人的成长像竹节一样。比如主角一直很淘气，但有一天突然有了要照顾妹妹的兄长意识。人的心理发展是分阶段的，他们也许不会因为别人的评判而改变，却可以在条件合适时“蜕变”。

我们要做的，就是给另一半利于改变的环境：第一，缓解目前的紧张氛围；第二，诱发他行为的改变。

## 实践方法：沟通从态度开始

具体怎么做呢？首先，你要意识到，你和另一半的沟通首先都是在表达**态度**，这个态度就是你的情绪。所以，不管你要说什么内容，对方首先感受到的是你的态度：是接纳的，还是排斥的。

了解到这一点之后，希望你们之后的沟通，就不要急着用排斥的话语去逼迫对方了。我在指导咨询者面对感情问题时也注意到，接纳的话术会营造一种顺畅或轻松的感觉。

以开头提到的那位妻子为例，她又一次遇到丈夫喝酒晚归的事情，但这次她做了改变：

以下是妻子调整沟通方式的具体步骤：

1.  **不质疑、不摆脸色**：不去质问对方“为什么这样做”，也不摆出难看的脸色。
2.  **主动询问经历**：主动询问丈夫喝酒时经历了什么。
3.  **询问体验感受**：询问丈夫喝酒的体验感如何。如果他懂酒，可以聊一些酒的相关话题。
4.  **表达反思**：在丈夫惊讶时，表达一下自己的反思以及之前对他的不理解。

当然，一开始并非很顺利。妻子说她这样做时会有种“失控”的感觉。但后来发现，丈夫并没有变本加厉，虽然暂时没有太大改变，但两个人的氛围缓和了很多，也算是一个新的开始。

## 总结

本节课中，我们一起学习了“管得太严”对亲密关系的危害。我们分析了其利己的本质，探讨了如何将“标准”转化为可协商的“诉求”，并引入了**边界感**的概念。最后，我们认识到，人们很难被直接改变，但可以为对方创造易于改变的环境。关键在于沟通时优先传递**接纳的态度**，而非排斥的内容。

如果你也因为管得太严而没有获得想要的效果而困扰，不如试试这种相反的方法：放轻松点。别人首先感受到的是你的态度，其次才是你说的内容。
