# 父母反对的感情如何应对？带你打：父母反对-晴天老师Sunnie

在本节课中，我们将学习当恋爱关系遭遇父母反对时，应如何正确理解和应对。我们将探讨核心原则、沟通策略以及如何通过实际行动逐步赢得父母的认可与支持。

---

当恋爱以结婚为目的时，见家长是不可避免的一关。双方父母都满意自然最好。如果遭遇父母反对，我们该怎么办？你肯定不想就这样放弃这段感情。从相识到相爱，一路走来经历了许多磨合，你们都坚持了下来。结果到了最后一步，还没来得及展示自己就先被否决，心里肯定不甘。

事实上，父母反对都有他们自己的理由。上一代和我们这一代，难免有很多观念上的冲突碰撞。想要继续走下去，就需要得到父母的支持。今天就来跟大家说说，如何正确化解父母的反对。

## 核心原则：绝对不要硬杠

首先记住一个大原则：**绝对不要和父母硬杠**。

假如你的对象听到父母不满意你，就冲上去维护你。也许你觉得他很爱你，但这样做的后果其实是把你们的感情推向深渊。中国传统文化里，百善孝为先。他的父母看到儿子竟然为了一个外人而忤逆自己，首先会给他扣上“不孝”的帽子。而且，就算他们觉得儿子做错了，更多还是会指责你，认为是你把他们的儿子带坏了。那他们对你的印象还能好吗？

所以，得知父母反对后，你的另一半千万不要和父母吵架、为你说好话，或者指责他们不理解你们等等。这样只会加剧问题的严重性。

当然，如果他什么都不做，那基本上也可以考虑分手了。如果他连你都不保护，让你独自面对父母的刁难，甚至为了顺从自己的父母而放弃你，那么即使最后勉强走到一起，婚后生活同样不得安宁。共同说服父母他都退缩，更别说婚后如果他的父母再与你产生矛盾，他会为你做什么了。

## 理解普遍性：你并不孤单

事实上，生活中大约有70%的情侣，父母一开始都是不大同意的。所以你遇到父母反对其实是正常情况，并不是你个人独有的问题。

而且，大部分父母反对子女的感情，都留有一条“宽敞的后路”，并不是全方位的回绝。这就是你们需要努力的方向。可以通过时间和一些方法，去争取对方父母的支持。

## 应对策略分步指南

上一节我们明确了核心原则和问题的普遍性。本节中，我们来看看具体可以采取哪些步骤来应对。

以下是四个关键的应对步骤。

### 1. 信任伴侣，让他主导沟通

你的伴侣事实上拥有两个身份：第一个是你的伴侣，第二个是他父母的儿子。所以他夹在你和父母之间会非常难受。你不要再给他压力，我们要理解和支持他。

有的父母反对你们，可能只是一个试探的过程，来看看你的脾气秉性如何，是不是真心爱他们的孩子。如果这个时候你不断施压，或者用危险的手段逼迫对方父母同意，不仅不能解决问题，还可能激化你们俩之间的矛盾。

所以你一定要站在他身边，给他力量，让他去做父母的思想工作。退一步说，他父母反对你们，说到底是他自己家里的事情。如果你还没进门就在那边指手画脚，肯定会让情况恶化。

**行动指南**：这时你要充分相信你的伴侣，让他和父母直接沟通，你不要插手。同时，在你们两人独处时，你要体谅他，和他共同解决问题。

### 2. 明确反对的具体原因

对方父母反对，肯定有他们自己的原因。有些原因你可能觉得无语，但没办法，你仍然需要知道他们反对的理由。这可以让他帮你去探口风。

但是他探口风的话术千万不要太生硬，要尽量客观，这样才比较方便套出他父母的真心话。

**示例话术**：
> “爸妈，我知道终身大事很重要，所以我肯定是尊重你们意见的。今天我们就交交心，我想知道你们是怎么想的，这样我心里也有点数。”

这段话的态度是中立的，可以避免和父母再产生冲突，也表明了他愿意听父母的心里话。

**原因分析与对策**：
如果你是女孩，他父母可能在意的条件多半是外在的，比如：
*   你的学历不高。
*   身材比较矮小。
*   不够好看。
*   不懂得做家务等等。

如果是一些你无法逆转和改变的缺点，那你就需要他去做父母的工作，让他们看到你的优势。比如虽然你个子矮，但是你工作能力强，那就要让他去着重强调你的工作能力，把身材因素的担心降到最低。

### 3. 通过伴侣的改变建立信任感

让他父母对你产生信任是至关重要的环节。他父母对你有不好的看法，本质上其实是觉得儿子和你在一起不能过得更好。

如果你让伴侣在他父母面前多夸夸你，说他很好之类，那他父母心里想的可能是：“你还要在我面前再提他？” 这无疑会继续加深对你的看法。而且你再怎么好，跟他们其实没有多大关系，他们关心的是自己的儿子，并不是你。

所以，你不需要跟他们传递你多优秀（他们本来就在否定你）。你取得他们信任的途径，在你的伴侣身上。你要让他的父母觉得，**他儿子跟你在一起过后，变得越来越优秀了**。

**具体方法**：
*   **案例1**：如果他之前爱抽烟，父母怎么讲也不听，但跟你在一起后，再也没在家里抽过烟。那么当妈妈问起时，他可以说：“我女朋友说抽烟有害身体，所以督促我把烟戒了。”
*   **案例2**：如果他之前做事没计划，经常出纰漏，然后他爸妈安排的一件事，他很有条理地按时完成了。那么当爸妈询问时，他可以说：“我的对象总是叫我要事先规划好，要有条理地做事，果然这样省时省力。”

只有当伴侣的改变明显时，他父母对你的看法才会开始缓和。但这还不够，你还要再添一把火。

**关键时机**：
前面我说过不要让他帮你说好话。但如果他父母态度已经缓和之后，这个时候他就可以帮你说好话了。
*   如果他父母觉得你娇生惯养，那你的伴侣可以说：“他其实可会做料理了，我加班忙的时候，都是他做好饭带到公司给我的。”
*   如果他们觉得你学历低，那他可以这样说：“其实他很爱看书，英语口语比我厉害，经常跟外人打交道。”

**核心公式**：
`说好话的时机 = 家长态度改变之后`
而不是一开始上来就帮你说。父母看到自己儿子变好，心里有底，对你印象改观之后，这时说好话才是有效的。

信任需要你努力去构建，也需要你伴侣的帮助。你走的每一步，做的每一件事，对方父母都看在眼里。等到信任感累积足够，问题自然就解决了。

### 4. 用诚意与时间化解偏见

很多事情其实都没有特定的标准。最能打动父母的其实是时间和诚意。如果他们一反对你，你就开始退缩或者埋怨伴侣，那就很难得到他们的认可。

**真实案例**：
我们曾遇到一个男生的咨询，也是因为父母反对。他本人性格温和、顾家。刚见女生父母时，他们很满意。但没想到有一天，他在厨房帮厨时，被女方妈妈看到了手臂上的纹身。对方妈妈就坚决反对他们在一起，认为有纹身说明这个人不正经、不靠谱。后来任他们怎么做工作，父母还是不同意。

**解决方案**：
我建议他先去把纹身洗掉，然后真心诚意地对待女朋友和她的父母。经过小半年的优秀表现，他的父母终于接受了他。

有时候，父母确实会凭借一个刻板印象来判定一个人。上一辈人的思想很难改变，你只有用真心和持续的优秀表现去打动他们，或者这也是他们考验你的方式之一。

**重要提醒**：
另外，如果你和他父母之间的矛盾还没解决，尽量不要让你的父母和他们见面，更不要把自己的委屈说给自己父母听。否则只会让情况变得更难以收拾。

对方父母反对你，你也不要发泄负面情绪，不要灰心。你要坚定决心，把这个反对看作是你们加深感情的一个过程，让对方父母看到你的诚意。

## 总结与核心信念

本节课中，我们一起学习了当感情遭遇父母反对时的应对方法。

我们首先明确了**不要硬杠**的核心原则，理解了这种情况的普遍性。接着，我们分四步走：**信任并支持伴侣进行沟通**、**明确反对的具体原因**、**通过伴侣的积极改变来建立对方父母的信任**、最后**用长期的诚意和实际行动化解偏见**。

爱情是你自己的。当你认定它是你的幸福时，你就努力去取得他父母的同意，让对方父母承认你。而不是委屈求全地顺应他们。应该是你们两个人共同经历父母的考验，这个过程也能够让你们更加珍惜这段感情，有充足的信心去迎接未来。
