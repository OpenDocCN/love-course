# 关于挽回，先接纳分手，再认清自

在本节课中，我们将学习如何通过系统性的步骤来修复一段情感关系。核心在于：**先接纳分手事实，再认清自身情感问题，同时寻求解决方法，最终改变长久以来的行为模式**。这样的质变对挽回才是最有效的。

---

多年前，我曾深受被分手的痛苦。我经常反问自己是哪里不够好，能够让他把我们过去这么多年都当作不存在，让他对我挽回的消息视而不见。当时甚至一度很迷信占卜术，我也知道我是病急乱投医了。

但是只有占卜在说我们还可以在一起，而周围的人都是在唱反调的。我花了几个星期去看星座走势，学习塔罗牌的算法。有时候似乎对了，可是好像下一秒又不对了。我总是感觉他的情绪是很反复的。其实现在回过神来，反复的才是我自己。

当时我已经失去了对感情的判断力，不知道自己该做什么，也不知道怎么做才能够达到在一起的目的。连续几个月都只有迷茫和痛苦。到后来连自己为什么要在一起都已经不清楚了，脑子里只有当时甜蜜的回忆。

但是，我跟他在一起真的还能够幸福吗？忍受不了这样的自己，我才决定改变。有希望并不是不对的，可怕的就是心存空想。没有人一开始就会对情感精通。其实那个时候我的前路也是在迷雾中。如果你也在这样的状态里，我希望你能够看看自己。

挽回是你现阶段的想法，是达成了让你幸福的方法。作为方法的话，它必须有头有尾可以操作。你需要去掌握它，而不是让你的生活被它掌握。本末倒置了，那你依然会痛苦。

挽回爱情是为了让自己幸福，挽回的过程必然不能违背自己的本心。而你的本心就是自己从一段感情中获得支持，是自身为了应对社会生存形成的习惯，是想要将来有一个圆满的小家庭。

那你就要改变跟他在两个人亲密关系里面不好的相处模式，也要保持自己的价值持续增长，跟上社会的步伐，保证自己舒适的生存。你现在为了挽回做的一切行为，会给你一种为了幸福在努力积极的感觉吗？如果是有的，那你就可以继续保持。你离成功可能只差一点时间和一点点技巧。如果是没有的话，那你就需要听一听我下面这些经验之谈了。

## 第一步：郑重告别，接纳事实 🚪

上一节我们探讨了挽回的初衷与心态。本节中，我们来看看挽回的第一步：如何与过去告别。

我之前被分手，为了不让自己闲下来（因为闲下来我就会忍不住拼命地去回忆），我讨厌想起自己被抛弃这件事情，只能拼命去求挽回，想要当后来发生的一切都没有发生过，结果直接就被拉黑了，根本也没有渠道去挽回。

联系方式其实还是其次的，打击最大的其实还是有的时候，你都看到他在线却不回你，但是只能不断地反思，是不是自己又做错了什么，又说错了什么。

然而，我们想要挽回的话，必须要先敢于承认事实。不敢相信分手，那你就失去了认清自己现在状态的可能性。那你所做的事情其实都是架空在“我和他还保持着一个良好关系”的幻想上面。

虽然我自己也曾经否认这一点，可是事实它不会随着我的设想改变。何况只有我在修复感情，他已经放手了。这种不对等的做法引发矛盾就是必然的。他在刚分手的时候，也试过劝我要理解一下他的想法，但是我装作没有看见没听见，很快他也不想再听我说重复的话了。

**核心概念：接纳公式**
```
挽回成功的前提 = 承认分手事实 + 停止幻想
```

告别失败的恋情并不是否认，更不是放弃。我们从来都可以否认一段感情的破裂，可是如果你想要挽回的话，你否认这段感情破裂，就不是什么好的应对方法。

“是他抛弃了你”、“你们的感情没问题”……看看这些理由其实都是来自于外部。而你在其中具体做了什么，扮演什么样的角色？哪怕你说你什么都没做，有没有想过这种“什么都没做”，其实可能才是导致你们感情出问题的原因。更不要谈那些可能被你忽视掉的、已经做了的行为。

有时候一种生活方式，它可以适合你身处社会角色，适合陪你走了很久的家人，但是偏偏可能就是其中的一部分，与你的另一半冲突，与情感的本性是冲突的。毕竟情感和生活在某些方面就是有分歧的。

比如你从小就享受父母的保护，你的生活中到处都是你原生家庭的影子，这是你最舒服的状态。如果有人愿意照顾你的情绪，他希望能够跟你走下去，他可能做出了一些妥协，例如搬来你的城市，或者放弃他原有的工作，或者为你负担部分的经济。可是如果有一天他的妥协，一不小心超过了他的底线，或者说他长期都没有可以支持下去的回馈，也许你注意到了，但是你还是想要再享受一点点，或者说你沉浸在你圆满的生活里，根本没有看见。

爱情的付出一定是双方的。我们需要能够使自己保持正轨的方式，来处理这段走偏的爱情。能够自如地说出“我分手了”，反而可能是卸下了一块大石头。这意味着你已经承认，并且告别了这段确实很糟糕的情感。

当然，你也不用铺天盖地在朋友圈去发自己的告别。你只是对某一段情感告一个段落。此时的感悟，你可以告诉他，你可以告诉你的家人，你也可以告诉你亲近的朋友，甚至告诉专业的人士，向他们寻求下一步的支持。最重要的是你要知道，这段感情，它还会有新的篇章。

## 第二步：真实自述，复盘问题 📝

在学会了如何告别之后，我们需要向内审视。本节中，我们将学习如何进行真实的情感复盘。

所以，我们挽回的第二步叫做真实的自述。就算是专业的情感咨询师，他也不会读心术。可是很多人来咨询的时候，要么出于羞愧等原因，可能会隐瞒一些自己的问题，只是说现在发生了什么状况，对过往绝口不提，而且很多看上去好像都是对方的问题。

并不是说你这样错得就非常离谱了。我曾经也有过这样的经历。怨恨痛苦比起解决情感问题，更多的可能需要的是一个宣泄口。最终你能够走出来，是因为你会发现只有情绪，不过是深陷自我的催眠，会离你解决实际的情感问题越来越远。

已经走偏走错的爱情，最终酿成绝情分手。要改变这个创伤，你就还得往自身看。看得越清楚，那你切除的坏死点就越多。那些让感情不稳定的毛病，一下子全都爆发出来。你再隐藏也只会腐烂。放弃这一段再等到下一段感情，它其实也不能够自行痊愈。

**核心操作：复盘方法**
1.  **不要只在脑子里想**，因为大脑它会蒙蔽你。
2.  **不要只去询问一个好友**，好友当然会顺着你的思路来说。
3.  **最好的就是拿笔记下**你渴望解决的情感问题，写下你目前遇到的挫折。

也许很长，也许很短，也许你感觉在记流水账没什么用，或者说你觉得这样的时间不值得。但是重点它并不在于你自述了多少，而是要冷静下来，静下来去思考。对情感的复盘，写下来的东西，你再回头看的时候，你才能够看出一些自己逻辑上的矛盾。

这个时候你能讲出来的情感经历，几乎就是脱离为自己辩解的真实情况了。你也会对自己有一些更清楚的了解。

## 第三步：跳出舒适圈，促成质变 🚀

完成了对问题的梳理，接下来就是行动的时刻。本节中，我们将探讨如何突破自我，实现真正的改变。

挽回第三步，我们称之为跳出情感的舒适圈。其实真不是我话说得太满，事实上很多的改变，它就是像龙卷风一样。但是我们大家挽回的时候总是觉得，挽回似乎没有效果，糟糕的情况还是一成不变。其实是因为你的准备是多过于你的行动的。

为什么不行动？还不是因为怕脱离思想的舒适圈吗？某种程度上，做好了前两步还想挽回的人，会很清楚他们的举止，哪些会将前任推远。可以推断的是，在挽回中遇到的困难，在他跟其他重要的家人朋友相处的过程中，也是无法避免的。只不过在情感中，这些问题的影响更大。

我们说的舒适圈，就是你认为无法改变的理由，一种既定的对你来说很有效的处事方式。在你原来所处的环境里，周围的人和你达成了一种平衡，会包容你那些也许不那么好的地方。

只有当情感冲突超过了你想要的舒适，通常只有当你的行为产生了很严重的后果的时候，你才会有足够的动力做出改变。

**触发改变的常见情境（列表）**
以下是几种可能促使你跳出舒适圈的情况：
*   当你总是在纠缠前任，惹得对方很讨厌，前任拒绝听你的解释。
*   当你的情感生活没有周围同龄人美满的时候。
*   当你喜欢的人告诉你他对你非常失望，根本不相信你的改变的时候。

这个时候也许才会让你意识到，舒适圈已经不能够满足你了。如果经过这样的事情，你觉得自己经受的痛苦已经变得不可忍耐了，才会有足够的动力去接受挑战，去改变长久以来的行为模式。那这样的质变对我们挽回才是最终有效的。

---

**本节课总结**

在本节课中，我们一起学习了情感修复的三个核心步骤：
1.  **郑重告别**：勇敢接纳分手事实，停止不切实际的幻想，这是看清问题的基础。
2.  **真实自述**：通过书面复盘，冷静客观地梳理自身在感情中的问题，看清逻辑矛盾。
3.  **跳出舒适圈**：当痛苦累积到无法忍受时，抓住动力，果断改变旧有行为模式，实现个人成长的质变。

挽回的本质是一场深刻的自我革新。只有当你先处理好与自己的关系，改变那些导致感情破裂的内在模式，重建的关系才有稳固的根基。
