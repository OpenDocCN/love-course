# 课程01：如何挽回“可做朋友，不可做恋人”的前任？🧩

在本节课中，我们将通过一个真实的咨询案例，学习如何分析前任的性格类型，理解分手的深层原因，并掌握针对特定性格（社群型）的挽回策略与相处建议。我们将借助“九型人格”体系中的本能类型理论，帮助你理清关系中的矛盾核心。

---

最近，一位女生向我咨询。她对前任付出很多，也忍让很多，但最终还是被分手了。无论她如何尝试挽回，前任都坚决不同意复合，坚持说他们“可以做朋友，但不能做恋人”。女生对此完全无法理解。

在咨询过程中，我发现这对情侣其实有很深的感情基础。分手的主要原因在于双方性格不同，彼此付出的东西并非对方真正所需。长此以往，他们的相处模式让两个人都感到疲惫。

例如，有一次女生过生日，她非常期待和男朋友单独度过一个浪漫的夜晚，并收到惊喜礼物。然而，男朋友却提议：“我有个朋友今天也过生日，要不我们一起过吧？”女生听后非常生气。

原本期待的浪漫约会变成了多人聚会，但她还是勉强参加了，只是全程板着脸。聚餐结束后，大家兴致很高，提议去唱歌。女生直接表示不想去。男生却说：“这么多朋友给你过生日，你是主角，怎么能先走呢？”

女生已经忍耐了很久，于是坚决地自己打车回家了，留下男生和朋友玩到很晚。女生回家后憋了一肚子气，等待男生来哄她。而男生也很生气，认为女友太任性，让他在朋友面前很尴尬。两人因此大吵一架。

女生认为是男生的错，等待对方道歉；男生则认为是女生不对。类似的矛盾在生活中越积越多，长期得不到解决，逐渐消耗了原本的感情。从心理学角度看，两人都没有绝对的对错，只是性格不同，关注的焦点不同。

根据“九型人格”体系中的本能类型理论，每个人都有三种本能：**自保本能**、**一対一本能**和**社群本能**。但每个人性格中，这三种本能的占比各不相同。

**公式：个人性格 = 自保本能占比 + 一対一本能占比 + 社群本能占比**

例如，一个人如果自保本能占50%，一対一本能可能占30%，社群本能占20%。正是这种占比差异，导致了人与人之间的性格差异。

在这个案例中，男生的性格里**社群本能**占比最高，其次是**一対一本能**，最后是**自保本能**。我们称他为**社群型**。而女生的性格里**一対一本能**占比最高，其次是**自保本能**，最后是**社群本能**。我们称她为**一対一型**。

上一节我们介绍了本能类型的基本概念，本节中我们来看看**社群型**性格的具体特点。

以下是社群型人格的三个主要特点：

1.  **朋友多，人脉广**：社群型的人将注意力焦点放在拓展人脉资源上，乐于交友，慷慨大方，因此往往人缘很好。当他们需要帮助时，通常会有很多朋友愿意帮忙。以案例中的男生为例，他下班后几乎天天有饭局，节假日还要陪领导，时间大多给了“外人”。但女生后来回忆，男生曾利用他的人脉，帮她奶奶联系到好医生做手术，也帮她弟弟解决了大学入学问题。
2.  **在公共场合一视同仁，先人后己**：社群型的人在公共场合会优先照顾“外人”，而把伴侣视为“自己人”放在后面。案例中的女生抱怨，男生单独对她很好，但有朋友在场时就对她不那么热情了。这其实是社群型性格的典型表现，并非“重友轻色”。
3.  **顾全大局，大方得体**：社群型的人关注群体，在社交中会照顾每个人的感受，不会突出自己或搞特殊。他们希望自己的伴侣在社交场合也能端庄得体，不任性、不小家子气。案例中女生多次拒绝参与或提前离场，让男生觉得很没面子。

了解了社群型的特点，我们再来看看案例中女生所属的**一対一型**性格。

以下是一対一型人格的三个主要特点：

1.  **希望被偏爱，被独宠**：一対一型的人渴望成为伴侣眼中的唯一，希望得到全部的关注和偏爱。案例中，男生出差回来送她一个娃娃，她本来很开心，但听说男生给表妹也买了一个后，立刻就不高兴了。她认为送给恋人的礼物必须是独一无二的。
2.  **随性，挑人**：一対一型的人对自己喜欢的人非常热情，对不喜欢的人则可能很高冷，甚至不愿敷衍。因此，女生对男生的朋友不感兴趣，经常拒绝聚会，即使去了也表现得不开心，这让男生在朋友面前很没面子。
3.  **喜欢深度交流，而非泛泛之交**：一対一型的人朋友圈较小，追求少数几个知心好友。所以当男生想带她进入自己的朋友圈时，她并不乐意，表现得不够热情，久而久之，男生就不再带她参加活动了。

在分析了双方的性格差异后，女生终于明白，许多矛盾源于彼此的行为不符合对方的性格期待。她觉得被辜负，实际上她也让男生很失望。

既然理解了矛盾根源，我们就可以针对性地调整相处模式。本节课主要探讨如何与**社群型恋人**相处。

以下是三条与社群型恋人的相处建议：

1.  **在外给足对方面子**：社群型的人非常重视在公众场合的形象。作为伴侣，在社交活动中不要过于黏人，要表现得体大方，最好能协助他照顾朋友。当他外出应酬时，不要频繁打电话催促，这会让他在朋友面前尴尬。
2.  **在家做好贤内助**：社群型的人将大量精力消耗在外，回家后非常需要一个温暖、安静的港湾。伴侣应尽量理解他在外的辛苦，不要用琐事烦扰他，更不要在他疲惫时吵闹或无故怀疑。
3.  **采用“非暴力沟通”**：良好的关系需要沟通。与社群型的人沟通时，需要理性、直接地表达感受和需求，而非情绪化指责。例如，不想参加聚会时，可以说“我今天身体不太舒服，状态不好”，而不是“我心情不好就是不想去”。只要理由合情合理，他们通常能够接受。

做到以上三点，与社群型恋人的相处会融洽很多。案例中的女生在理解彼此性格后，针对男生的性格特点调整了策略，最终成功挽回了前任。男生也发现她变得更懂自己，两人的关系更加和谐。

“性格不合”是一个常见的分手理由。但想要挽回，就需要搞清楚到底哪里“不合”。在我们看来，没有绝对的“性格不合”，只有“性格不同”。**先理解，再共情**，才是解决问题的关键。

当然，人的行为会受到环境和各种因素的影响。理论上，一个人的本能类型是天生的，但具体行为表现会因情境而变化。相同性格的人可能表现不同，不同性格的人也可能表现相似。想要全面分析一个人，必须深入了解其内心，透过现象看本质。
