## 课程概述

在本节课中，我们将探讨一个常见的情感难题：为什么在感情中做错事导致分手后，即使道歉了也常常没有效果。我们将深入分析无效道歉背后的逻辑错误，并学习如何做出真诚、有效的道歉，以修复受损的关系。

## 为什么道歉会失效？ 🤔

上一节我们概述了课程目标，本节中我们来看看道歉失效的核心原因。

举一个最形象的例子。如果有一个人他重重地踩了你一脚，然后说“不好意思我也想好好看路，但是我在玩手机”，你会不会一肚子气？他也道歉了，但是这个话听上去就是不舒服。其实我们很多人呢，也是这样道歉的。

我有一个学员，他是七夕分手的。他晚上和同事玩到了第二天凌晨两点，对着等他回家的女友说：“对不起，我本该陪你的，没有注意到我们七夕了。”本来就很生气的女友呢，直接就把他扫地出门了。

这个男生可能是真的想道歉，但是话说出来振振有词，让对方觉得这哪里是道歉呢，其实是来挑事的吧。出现了这种问题呢，就是因为你的道歉缺少了核心：**真诚**。

我也看过了不少道歉的聊天记录，通常都是这样的一个逻辑。

以下是典型的错误道歉逻辑：

1.  **“对不起，我不是故意的。”** – 上来先迫不及待地把责任推出去，减轻自己的压力。
2.  **“你能不能原谅我？”** – 问题还没有解决呢，就厚着脸皮想让对方取消惩罚。
3.  **“像以前一样好好的，别再生气了。”** – 好像对方提分手是很无理取闹一样。
4.  **“我们可以怎么样怎么样相处，我一定改。”** – 这样说的人呢，看似是在解决问题，其实根本没有想和对方达成共识，而默认了自己提出的设想就是未来，也希望对方能够这么接受。
5.  **“我一定改。”** – 这个真的是很苍白无力的道歉模板。

道歉呢，本来应该是一记安慰剂。意思是：我知道你现在很伤心很难过，所以我主动认错、担责，来让你不那么受伤，你再考虑要不要撤下对我的防备。

结果呢，我们大部分人都是按照上面那个错误的逻辑。一番道歉下来，目的是：闯祸了，怎么摘除我自己，怎么保护自己。对方的伤口还在隐隐作痛呢，哪能那么轻易地给出原谅呢。

## 道歉的价值与基础

理解了无效道歉的模式后，我们来看看道歉本身的价值和基础原则。

当然了，道歉肯定是比不道歉要高明一点。只要你能够顺利地道歉，对方至少能够意识到你还是关心他的，你们就有一个交流的可能。

无论你道歉得怎么样，至少不会背上“回避性依恋”、“没有心的”骂名。前任他最多说你不懂他、你不走心，但是不会心灰意冷直接放弃。

即使是不愿意原谅你的人，他也会期待你有个态度。否则会在心里抱怨：“虽然我再也不想看到他了，但是毕竟分手了，他连道个歉、挽留一下也不愿意，果然是真的不爱我。”

所以呢，你缺的只是**真诚**。

## 三种有效的道歉方法 🛠️

上一节我们明确了真诚是道歉的基础，本节中我们来学习三种具体、有效的道歉方法。

要想让你的道歉真的有效的话，有以下这三种办法你可以参考一下。

### 方法一：关注对方的感受

一方面，你要跟对方感同身受。这里并不是要你强行说“如果我是你，我也会难过”。你是你，他是他，你需要体会的是他受伤时的那个情绪。

**公式：** `有效共情 = 陈述事实 + 细化情绪`

比如说：“因为我的[具体过错]，给你造成了[哪些损失]。我知道你现在心里感觉肯定是觉得被背叛了/很难受啊等等。”一定要去细化这个情绪，让对方觉得你能够感受他受伤的细节。

这里呢，其实是一种心理按摩。你能够感受到他的情绪越细的话，他就会越觉得你懂他。

我们拿上面那个七夕没有陪女友过节的例子来讲。你就要直接点出，女友她没有等到你，非常的失落。

如果你不知道用什么词汇来描绘对方的感受的话，可以参考来自萨提亚家庭研究院的“500个描述情绪的词汇”，网上也能搜得到。比如愤怒、恼火、气愤、悲伤、痛苦、反感、憎恨等等。肯定有能够准确地对应到他当下情绪的那个词。

另一方面，千万不要过度道歉。如果对话的结果变成了受伤害一方反过来要关注你的感受的话，你就失去了做出真诚道歉的机会，因为对方他在对话里产生了非常糟糕的感受。

很多人做错的事情分手，找对方哭啊，表现得很绝望啊，说什么“我真的把事情搞砸了，我对不起你啊”，恨不得让对方马上反过来安慰自己。这其实就已经是一种情感绑架了。

虽然呢，你嘴上道歉，实际上你给人的感觉是，他对不起你。因此呢，受伤的一方反而要去理解一下你的处境有多悲惨，明白你的自责自悔多深刻。

### 方法二：要有补救行动

比如，你把对方的电脑一屁股坐碎了，那你道十次歉，也没给对方补偿或者说帮他换一个屏幕，那这个怎么能算是真心实意的道歉呢？

再比如，约会迟到了。那这个时候呢，你光道歉也不够，指望下次再让对方玩得开心、忘掉你迟到的事实也不现实。

所以啊，一定要主动地提出你经过深思熟虑的一个替代方案。针对当下，你多陪他，比如说他想喝茶，你多给他买一份。或者说针对以后如何才能不迟到，及时给出补救方案。

如果你们已经分手了，你也是需要去补救的，而且还不能拖。因为你越拖，对方他的怨恨感可能就会越大一分。

有些人觉得已经这样了，那我道歉的话也挽回不来吧。其实呢，这种想法只会增大你去道歉的压力。迟来的道歉并不轻贱，反而可以显示出是你深思熟虑之后，仍然顶着压力去跟对方表达歉意了。你从来没有忘记过他从你这里受到的伤害。

那这种道歉呢，其实还能够一笔消除两个人在这种伤害和被伤害的关系中累积的精神重负。

不过呀，这种补救行动不能大而全。比如说，你承诺“你下次一定不会再犯”，这其实就很假，很难让人相信。

比如说，如果你是一个一激动就容易大声吼对象的人。那这种无法控制自己不犯错的绝望啊，不仅会让对方心累，也会让你慢慢对解决这个矛盾失去信心。

所以我们最好要把道歉中的一些具体行动去落实。比如如何降低让对方受伤的可能，或者说告诉对方，你可能确实是有这样的毛病，你目前一时也改不掉，但是呢，你已经想好你要怎么做。

**代码示例（行为计划）：**
```
如果 情绪激动：
    执行 去隔壁房间冷静
    执行 暂时不说话
直到 情绪平复
```

这些具体的措施就能够体现出你对避免这件事情再次发生有多在意了。对方他就不会只盯着你喜欢吼人这个错误，而是去关注你怎么样克制自己的情绪。

### 方法三：直接开门见山，直奔主题

不要觉得自己道歉就很丢人，所以藏着掖着的。

在《有效道歉》中，作者约翰·卡多认为，当我们需要为了冒犯或委屈他人而承担责任的时候，我们就应当道歉，而且要以直接、明确和人性化的方式表达懊悔，同时做出弥补，并且保证以后绝不再犯。

换句话说，真诚有效的道歉不能有“但是”、“可是”这样的字眼。你错了就是错了，不需要在道歉里面强调你错的理由。

对方他受伤了，第一时间只关心你的态度。只有感受到你的真诚之后，他才会有空去理解你也有难处。

所以呢，你就直接说你问题在哪就好，不要急着去补充自己也很委屈的缘由。

另外呢，关于道歉，它并没有一个固定的形式。《道歉的五种语言》以及《爱的五种语言》中都指出了，你怎么样对待另一半需要看对方他喜欢什么样的形式。千万不要自我感动。

因为每个人他对于道歉表达的接受程度不同。可能你认为这样的表达已经足够真诚了，但是可能换一个人他并不这么想。

有些人可能听到一句“我错了”他就能够感到释怀。但是有些人他不一定这么想，他更希望听到你说“我保证了以后绝不再犯，如果再犯的话怎么样”。只有这样的道歉才能够让他不再耿耿于怀。

这里最好的方法就是你去回忆一下，过去你们的矛盾是如何解决的。比如说上一次吵架，是他先求和的话，那他是用什么样的方式？他是给你塞了礼物，然后写了很正式的信表达歉意？那就说明这是他觉得道歉最好的方式了。

我们大多数人做事都是按照自己习惯的行为逻辑去的。这里你可以模仿对方他处理矛盾的方式。那你同样的给对方寄些礼物或者说写封信，说明你知错了。那就算不能够直接复合，也能够让对方明白你这是在后悔。

## 关于道歉的时机 ⏰

上一节我们学习了三种方法，本节我们补充一个关键因素：时机。

道歉一定要趁早。最简单的道歉是那种你还没有（严重）犯错时的道歉，此时其实是对对方经历的麻烦表达一个感同身受。

比如说：“你经历这些实在是太不容易了。”或者说呢，为自己给对方造成不便表达歉意：“真抱歉啊迟到了，路上呢有一起事故造成了大塞车等等。”

这里的道歉表示了：我们理解对方，我们重视对方。早安抚就不会拖到对方他对你的道歉上纲上线的地步了。就像易燃气体的化工厂总是要燃明火一样，就是为了把泄露出来的这些气早一步烧掉。

如果已经做错事了，对方也已经很生气了，你就要好好地准备一下道歉。因为受伤的一方往往会有一种报复心理。他想问“为什么会这样”的时候，也是希望你能够体会到他所受的伤害的，这个也很正常。

这时候你的道歉就要踩在对方的需求上。因为**猫才不会炸毛嘛**。

有时候我们去道歉，对方可能就简单地回了一个“嗯”、“哦”、“反省了”，可能就在朋友圈发一些出门吃喝玩乐的图。我知道了这个可能就会很打击我们挽回者的信心。

但其实，前任的潜台词是：“哼，我现在过得多好，你当初那些伤害算什么？”其实本质上，他依然是受伤了，只是他把自己伪装起来了。

那你该道歉的话，建议还是按照我前面建议的这个流程。这是最能够打破对方冷漠的面具。其实这也是对方的一种“废物测试”了，因为他受伤太深了，他一定要你主动地去验证自己是爱他的。让他看到你实际的、真诚的道歉，他可能才会愿意放下防备。那这时候你的道歉也才是有效的。

## 课程总结

通过道歉去修补人际关系，其实非常有效，也非常有必要。让对方的注意点从“你犯了多大的错”转移到“允许你去挽回他”。

始终你要记得一句话：**道歉是为了两个人关系变得更好，而不是为了你自己过得更好。**
