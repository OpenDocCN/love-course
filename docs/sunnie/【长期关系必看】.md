# 【长期关系必看】

在本节课中，我们将继续探讨长期关系中“感觉消失”的另外两个核心原因，并提供具体、可操作的解决方案。我们将学习如何将关注点从伴侣身上移回自身，以及如何为关系重新注入爱与活力。

上一节我们讨论了“感受不到爱”的问题，本节中我们来看看另外两个导致关系疏远的关键因素。

## 1. 过度关注伴侣导致关系压力 ⚖️

第二个常见原因是：虽然爱客观存在，但个人将全部的关注度和幸福期待都集中在了伴侣身上，导致始终感到不满足。

当今社会存在一种趋势。以下是两种典型的片面期待：
*   女性期待的“幸福”常被定义为：伴侣下班后没有社交、工资卡全数上交、牢记每个节日的仪式感、并能随时陪伴自己。
*   男性期待的“幸福”则可能是：伴侣包揽全部家务、懂得撒娇依赖自己、给予完全自由的时间、并能无条件理解和支持自己的爱好。

当我们把全部的关注和期待都压在伴侣身上时，会给婚姻关系带来巨大压力。双方都会感到担惊受怕，觉得自己无法满足对方的需求。

而过度关注对方的一方，自己也会患得患失。会抓住对方的一个错处，就怀疑他是否还爱自己。这种单方面的依赖和过高期望，会造成相互间的压力。结果就是我们把婚姻的道路越走越窄，提起婚姻满是对方的过错与怨恨。

自己却又没有能力去改变现状。

## 2. 幸福的核心：转向内在满足 🌱

但心理学研究已经证明，一个人的幸福主要取决于自己。幸福是一种主观感受。例如，两个同样离婚的人：
*   甲可能认为：“我30岁离婚被抛弃太可怕了，没人要我了，我这辈子完了。”
*   乙则可能认为：“30多岁还可以去爱新的人，我很幸运能够经历不同的人生。”

这本质上体现了 **“幸福在于他人”** 与 **“幸福在于自己”** 的区别。后者意识到，幸福感是自己内在的状态。无论经历什么，都有能力稳定地拥有幸福。

这就需要我们从内心深处去寻找满足和快乐，并且学会依靠自己的资源与能力去创造幸福。我一直在强调，婚姻中我们也要自我成长。很多人误解为：只有成长了，才有能力让别人爱上自己。

但其实不是的。你的成长，是为了培养出一个**自己都能欣赏、自己都能爱上的自己**。之后，你才有多余的爱去分给另一半。

## 3. 解决方案：先自富足，再爱他人 💡

那么，对于因过度依赖导致的爱意流失问题，我们如何解决呢？核心原则是：**先自富足，再爱他人**。

具体可以这样做。以下是几个可操作的步骤：

*   **定期接触新事物**：让自己定期接触一件新事物。它可以是一个新地方、一个新项目、一个新兴趣，或者只是读一本新书、看一部新电影、听一个新乐队的歌。
*   **拥有专属空间**：即使是同处一个屋檐下，也要有完全属于自己的区域。比如你的专属书架、你打理的盆栽。当伴侣看到这个空间，他会产生一种“离开了一成不变的生活”的感受，从而意识到你还有他可以探索的神秘另一面。
*   **设立长期目标**：给自己设立一个长期的生活目标，比如“两年内晋升”，并为之持续努力。重点不在于目标是什么，而在于你正朝着这个方向努力。这种努力会让你每隔一段时间都有新的成长，而这种新的成长会给你们的关系释放积极的信号，带来新鲜感。相信我，对方也会有这种感觉，感觉你变年轻了、有活力了、更有内涵了。

## 4. 创造爱的流动：共同投入的时光 💞

最后要讲的是，如何把你因自我成长而充盈的爱，分享给你的伴侣。所以我们讲的第三点，就是要有一些**共同去投入的时间**，专门空出来，让爱在你们之间流动。

婚姻关系本身提供了这种共同投入的机会，比如养育孩子。我看到很多夫妻本来在争执，一提到孩子的教育，就统一了战线，矛盾也不存在了。有了外界的压力，往往能够做到互相信任、互相支持、共同应对。

这是一个挺有意思的现象，在心理学上我们称之为 **“罗密欧与朱丽叶效应”**：外界压力越大，两个人的关系反而越紧密。

但同样的问题在于，一旦外界的压力消失，关系可能立马成了一盘散沙。我国有一个现象叫“高考离婚潮”，意思是很多人在孩子上大学后会选择离婚。两人说：“我们之前是为了孩子在一起，孩子离开了，发现也没意思了。”

其实，这就是在养孩子过程中，渐渐把对方的付出视为理所当然，也不去考虑个人的感受和需求。情绪起伏都借孩子传递出来。孩子一远离，立刻不适应，也不知道怎么跟伴侣讲话了。

这就是我们需要跨越的难题：**怎么在长时间的相处中，让爱有流动的空间？** 关键是你们本身要主动与对方分享话题，而不是时间久了就怠慢了。

如果总是借助外务——借孩子教育、借房贷压力、借婆媳关系——才不得不去和对方交流，那么这种方式虽然是一起投入了，但流动的全是负面情绪。

我们需要的爱，是周末一起无所事事晒太阳的下午，是并肩散步一起感受的微风，是没有孩子吵闹、两个人和谐相处的夜晚。不要因为别的东西，就忘记了相爱的初心。**爱是夫妻共同生活维系的前提。**

## 总结与行动指南 🎯

本节课中，我们一起学习了长期关系感觉流失的另外两个深层原因及应对策略：

1.  **警惕过度关注伴侣**：将全部幸福期待寄托于对方，会给关系带来巨大压力，导致双方疲惫不堪。
2.  **践行“先自富足”**：幸福源于内在。通过接触新事物、拥有专属空间、设立个人成长目标，先成为一个能让自己欣赏和喜爱的人。
3.  **主动创造“爱的流动”**：专门预留共同投入的高质量时间，分享积极体验，而不是仅靠外部压力维系交流。

如果你们在婚姻关系上感到困扰，应该约定：无论多忙，**“让彼此感到爱”享有优先权**。这才是把亲密关系看得比其他事情更重要的表示。

无论你觉得当下的关系形势多么困难，都可以按照前面说的方法试一试。记住，修复关系是一场始于自我成长的旅程。
