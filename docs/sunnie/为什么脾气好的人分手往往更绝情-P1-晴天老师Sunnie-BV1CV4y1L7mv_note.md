在本节课中，我们将要学习一个常见的恋爱现象：为何平时脾气很好的人，一旦决定分手，态度会表现得异常决绝。我们将深入探讨其背后的心理机制，并理解情绪压抑与亲密关系之间的复杂联系。

## 脾气好的本质：压抑而非消失

上一节我们介绍了课程主题，本节中我们来看看“脾气好”的本质。我们要明白，脾气好的人并不是没有脾气，只是他们习惯于压抑愤怒情绪。脾气好的人往往给人宽容、大度、好相处的感觉。他们对外攻击性不强，却需要更强的内在防御机制，例如生闷气或冷暴力。一个向外没有攻击性的人，往往会转向向内进行自我攻击。

## 情绪的能量：压抑愤怒的后果

我们很多人也会排斥各种情绪，例如愤怒、焦虑、悲伤。但其实任何一种情绪都是一种能量，愤怒更是一种十分强烈的心理能量。过度压抑愤怒会导致能量过度堆积，也就是我们常说的“怨念”。因此在挽回过程中，我不排斥让大家通过适量的情绪发泄来激起对方的情绪。你可以委屈，可以难过，可以悲伤。但是我从不建议使用指责、抱怨和愤怒，因为这些是带有强烈攻击性的情绪，容易使得对方的自我防御机制加强。

话再说回来，很多人面对脾气好的人，就失去了将心比心的能力，以为对方不在意，自己就更觉得不用在意了。但是你不知道，脾气好的人内心是格外渴望被在意的。往往自己都不在意自己情绪的人，是因为从未被在意过。

我们总是指责有些人不愿意表达，不说出来，你也不知道你的行为是伤害，不知道对方也会不开心，也会生气，也会有感情。脾气好的人现在的绝情，只是因为曾经太过于深情，太过于照顾你的感受，而忽略了自己的感受。

## 脾气好的人为何形成？五大心理原因分析

脾气好的人是更能够体谅他人的，知道哪些情绪会让自己难受，所以为了不给他人造成痛苦，选择了压抑自己的不良情绪。当然，这样的压抑并不好，对于亲密关系的建立和处理也是不利的。我们经常说，不要把你最恶劣的情绪给你最亲密的人。但是世人都有情绪，有时候没有最亲密的人，我们的情绪也会变得无从发泄，只能压抑。我们知道的很多精神性疾病，其实都源于对情绪的压抑，例如抑郁就是对愤怒和悲伤的压抑。

当你有一个合适的机会，这些堆积的能量会一次性爆发出来，会比正常宣泄愤怒的人能量更大。也就是他们嘴上说的：“已经忍够了。”因为愤怒的情绪长期在体内堆积，他的潜意识知道，如果再不发泄出来，可能躯体就会造成生理病变了。潜意识的作用远比我们知道的要重要得多。我们需要让他们从认知模式上，改变对这些“不符合社会期待的情绪”的观念。脾气好的人是善良的人，也是因为长期忽略了自己的情绪和感受，很多回避型人格也是因此形成的。

所谓的“脾气好”，原因很可能有以下几点：

以下是五大心理原因的具体分析：

1.  **情绪表达能力缺失**
    对方脾气好不是因为他有多包容你，而是因为他根本不懂得发脾气，正确的运用情绪表达态度的能力，在他的意识中是缺失的。我也有很多朋友和咨询者，都是看上去脾气很好的人，但他们很多都有一个不太好的童年记忆。比如我有一个闺蜜，从小生活在需要看人脸色的环境中，所以比一般人更敏感、更懂事。父母给他的物质生活还不错，但是缺乏感情的关心。后来新家庭中又有了新的孩子，为了不被父母冷落，便从小养成了压抑内心情绪需求、去讨好人的习惯。所以从我们认识起，我就从未见过他发脾气。长期不被关注、不被爱的童年造成的影响，就是让他认为情绪无用，或者说从未掌握正确运用情绪的方法。但是他们一旦愤怒，触发点应该是相当高的。

2.  **缺乏安全感**
    第二个原因，可能是缺乏安全感。我们有一个女性咨询者，因为家里添了一个新的弟弟，工作忙碌的父母照顾不过来，所以从小就把他寄养在外祖母家。这其实是一个感觉到自己在父母眼中地位不明显，从而更加重视讨好技巧的案例。他说小时候最盼望的就是父母来看他的时候，但他也知道无论自己多想父母、多依赖父母，他们终究还是会离开，需要去照顾另外一个更加年幼的弟弟。这也让他在成年之后，形成了一种问题人格，表现出来的总是一副缺爱、不敢相信的样子。因为内心欠缺安全感，所以他们极少动怒，原因其实就是太害怕身边的人会因此离开。

3.  **家庭情感忽视**
    第三个原因，是家庭情感的忽视。如果幼年的家教太严，成长在专制的家庭，每次表达态度都会被父母以“父母都是为你好，你要听话”为由压回去。这类人的情绪不是没有，而是在长期与父母对抗无用后，习惯了自我压抑。但是这些被压抑的情绪，并非都被妥善化解了，而是越积越重。让他们动怒的出发点，可能看上去一点都不严重，但都是长久以来因为不善于表达情绪而累积下来的不满，有些甚至还是迁怒。这种类型在我看来是最危险的，因为他们心态悲观，明明自己不愿意忍，但还是在一忍再忍。而你看不到的那些情绪，都在内心反复作用。**`情绪积累量 = ∑(每次被压抑的不满)`**。打个比方，就好像是一台高压锅，当压力值达到一定范畴，又找不到正确排气出口的时候，就很容易爆炸。

4.  **冲突畏惧者**
    第四类人，是冲突的畏惧者。这类人视冲突为破坏关系的洪水猛兽，并不认为冲突是一种解决问题的正面办法。他们过于重视一种平衡的氛围，而逃避冲突。为了达成平衡，不惜以掩饰真实感受为代价。这种人最擅长的也是自我合理化，但是还不能算是完全的回避型。他们或许是因为曾经吃过冲突带来的大亏，在心里形成了根深蒂固的畏惧感；也或者是先天的和平爱好者，冲突的环境和感觉会让他们感到极度不适，想要逃离；又或者是因为自知能力有限，因此要尽量把局面控制到最低的危险度才好。这其实也是出于人趋利避害的本能。不过，这可能也只是个人价值观的取舍而已，他们选择或者习惯牺牲自己的部分利益，去换取自己的舒适区。

5.  **理想化回报认知模式**
    第五种原因，就是理想化回报认知的模式。“投桃报李”这种思维并没有什么不对。他们认为只要自己宽容、忍耐、讨好，就可以换得对方同等的对待。无论对方怎么骄纵、霸道，都不敢发脾气。但人是很复杂的生物，不是所有的关系都那么理想化，都能够以心换心。在感情中很忌讳的一点就是不对等的付出。由于一方价值远远大于另一方，或者亏欠和负罪心等等，这种容易在感情中形成高低位，造成感情失衡。**`关系健康度 ≠ (付出A + 付出B)， 而 ≈ (感受A × 感受B)`**。这其实也是很不健康的一种关系状态，也是很多感情分手的原因。一方过度付出，而另一方并不领情，长久以往就容易堆积怨恨和失望。

## 现象背后：情绪转移与内在伤害

人们总是容易忽略现象表面之后的东西。看起来付出与忍耐较多的一方，最容易被同情，也最容易让曾经受到他们“脾气好”好处的人有苦说不出。还有一些人根本不是脾气好，而是把情绪都给了自己真正信得过的人，偶尔对外流露的时候，就会让旁人大吃一惊。比如我对外大部分时候情绪都很稳定，但是却容易对非常在乎自己的恋人动怒。比如我有朋友看上去也非常友善大方，但是对家人却格外苛刻、容易生气。这其实都是一种情绪转移。曾经可能是因为对外表达没有安全感，在意长期以来维持的个人形象，内心偏完美主义，或者说受过较多的礼仪教育，认为对外表达情绪是丑陋的行为，却无法接受自己的另一面等等原因。他们堵住了自己情绪的出口，除了自己以外没有人可以抱怨，所以会选择向内伤害。

## 如何应对与挽回：关键在于引导与治愈

真正有一定情商而又懂得沟通交流的人，是会把情绪表达当做工具来用的。在必要的时候能够加强你的态度，也能够给自己的情绪找一个突破口。所以，如果你要挽回一个脾气好但是分手又很绝情的人，首先需要你自己的情商够高，爱意够足，才能够给予对方足够的爱意和引导。你要知道，他们最需要的其实是治愈。你要变得更加成熟和强大才行，这个才是我一直强调的挽回中自我提升的根本，并不是去健个身然后换几身衣服就可以了。做一个合格的伴侣是很难的。

另外，如果你是在关系里面脾气更好的那个人，看到这期视频我也希望你能够懂得，向恋爱的另一半适度地要求、索取与表达，这样的人际关系才算是真正的为人并为己着想。互不勉强但是也不纵容，有喜悦流露但是也不避讳去表达痛苦。总之，当他认为你足够可靠，他也就不再压抑，他才能够顺利的表达自己的情绪和你沟通。

## 总结

本节课中我们一起学习了脾气好的人在分手时表现绝情的深层心理原因。我们了解到，这并非他们本性冷漠，而是长期情绪压抑、忽视自我感受后的集中爆发。关键原因包括情绪表达能力缺失、缺乏安全感、家庭情感忽视、畏惧冲突以及理想化的回报认知。无论是想挽回对方，还是自我改善，核心都在于学会健康地表达情绪，建立平等、开放的沟通模式，并给予彼此真正的理解与治愈。
