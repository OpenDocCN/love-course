# 课程13：《型男日记》第07天：如何寻找短期关系的合适人群 🎯

在本节课中，我们将学习如何通过微信将潜在对象成功邀约出来，并掌握在不同时间段进行邀约的策略与话术。课程将涵盖三个核心时间段的特点、具体的邀约步骤，以及高效的“钓鱼”与“被钓鱼”技巧。

---

## 一、 选择正确的邀约时间段 ⏰

上一节我们介绍了如何筛选精准的短期关系人群。本节中，我们来看看如何根据不同的时间段，采取不同的邀约策略。

我们把邀约见面分为三个时间段，每个时间段对应不同的关系和操作方式。

以下是三个时间段及其核心特征：

1.  **下午 (例如：14:00 - 18:00)**
    *   **活动**：吃饭、喝咖啡、看电影。
    *   **对象特征**：高颜值、具备长期发展潜力、你想认真发展的对象。
    *   **核心公式**：`下午邀约对象 = 高颜值 + 长期属性`

2.  **晚上 (例如：21:00 - 23:00)**
    *   **活动**：主要邀约喝酒。
    *   **对象特征**：符合我们上节课筛选出的、具备短期关系属性的对象。
    *   **核心公式**：`晚上邀约活动 = 喝酒`

3.  **深夜 (例如：00:00 - 04:00)**
    *   **活动**：直接邀约。
    *   **对象特征**：明确有短期关系意向的对象。
    *   **核心公式**：`深夜邀约 = 直接邀约`

**规律总结**：从时间段1到3，达成关系的速度和成功率递增，但对象的平均颜值可能递减。你需要根据目标对象的属性，选择正确的时间段发起邀约。

## 二、 核心邀约话术与技巧 💬

了解了时间段后，我们来看看具体如何邀约。对于晚上（第二个时间段）的邀约，有一套成熟的“邀约三部曲”。

以下是“邀约三部曲”的具体步骤：

1.  **模糊邀约**
    *   提出一个不具体、客套的邀请。
    *   **示例话术**：“有空一起出来喝酒。”

2.  **判断对方时间**
    *   试探性地询问对方的空闲时间，了解其作息。
    *   **示例话术**：“你一般什么时候有空？”

3.  **敲定邀约**
    *   根据对方的回答，给出选择题而非判断题，最终确定具体时间。
    *   **示例回答**（对方说“下班后有空”）：**“明后天你哪天方便？”**（优于“明天方便吗？”）
    *   **示例回答**（对方说“周末有空”）：**“那这周六或周日？”**（若在周二邀约，则应先铺垫“那周末提前约”，等到周四或周五再具体敲定）。

**注意事项**：
*   **反悔处理**：如果对方反悔，需明确责任。可能是你的吸引力不足或邀约细节有误，也可能是对方确有急事。对于无礼的反悔（如“朋友突然叫我”），可以果断放弃。
*   **信息交换**：在晚上及之前的邀约中，建议互换**姓名和电话**，这是基本的诚意和安全感建立。

## 三、 深夜直接邀约与“钓鱼”玩法 🎣

对于深夜时间段，方法更为直接。同时，我们将介绍两种高效扩大战果的玩法。

### 1. 深夜直接邀约
当在深夜（00:00-04:00）聊天且窗口良好时，无需“三部曲”，可直接邀约。
*   **核心话术**：“出来吃个宵夜？”（在此语境下，“吃宵夜”等同于发出邀约信号）。
*   **给予台阶**：对方可能会委婉推辞（如“太晚了”、“我都卸妆了”）。此时需要“给台阶”，即解决她的顾虑。
    *   **示例**（对方说“太远了”）：**“打车很方便，或者我来接你。”**
    *   **示例**（对方说“卸妆了”）：**“我就喜欢素颜。”**

### 2. “钓鱼”玩法（主动）
此方法适用于微信好友数量较多（建议>400）且活跃资源丰富的学员。
*   **操作方法**：在深夜（如凌晨3点）发布一条暗示性的朋友圈。
*   **内容示例**：配一张腹肌/身材/侧脸照，文案为 **“深夜健身，还有谁？”** 。文案需具备互动性。
*   **后续操作**：无论对方评论或点赞，**一律转为私聊**，并直接切入“吃宵夜”主题。
*   **重要细节**：
    1.  发布前**屏蔽**家人、正经发展对象（下午档）等人。
    2.  无论成功与否，在**第二天天亮前删除此条朋友圈**，以免被其他好友察觉。

### 3. “被钓鱼”玩法（被动）
此方法能帮助你发现那些主动发出信号的潜在对象。
*   **操作方法**：在深夜（01:00-04:00）主动刷新朋友圈。
*   **目标**：寻找那些发布伤感、寂寞或精致自拍动态的女性。
*   **行动**：一旦发现，**第一时间私聊**（而非评论），表达关心并快速发出邀约。
*   **话术示例**：“怎么了？看你好像不太开心。我也还没睡，要不出来坐坐？”
