# 中级情感私教 第八课：情感关系的核心 💖

在本节课中，我们将要学习情感关系的核心本质。我们将探讨恋爱与长期关系的基础，理解心灵沟通的重要性，并学习如何建立健康、持久的亲密关系。

## 一、恋爱关系的核心：心灵沟通优先

上一节我们介绍了课程主题，本节中我们来看看恋爱关系的核心原则。

单身男女谈恋爱时，不应急于进行身体层面的沟通，而应将心灵沟通放在第一位，这样关系才能走得更长远。

当两个人互相吸引并产生愉悦感时，我们会感受到爱。此时，双方会彼此吸引，并愿意逐渐打开自己的保护壳，体验更深层次的连接。

许多人会忍不住进行身体上的亲密接触。然而，现代人的恋爱观念与过去相比发生了很大变化。过去的人更注重精神交流和爱意，而现代恋爱可能仅通过微信聊天、吃饭、看电影或喝酒就快速确立关系。

这样的恋爱关系往往后患无穷。两个人想要长久地走下去，需要的不仅仅是身体交流，更重要的是心灵上的交流，这包括思想层面和心理层面的沟通。

人与人之间发生身体亲密接触很容易，但在精神、思想和灵魂上进行沟通则非常珍贵。现代爱情观往往过于浮躁和急躁，人们很容易开始一段恋情，也很容易放弃或更换对象。

我们真正需要的是与伴侣在心灵、心理和灵魂层面的沟通与交流。如果你刚进入或打算进入一段恋情，请记住不要急于用身体去沟通，而要用你的思想、感受和灵魂去与对方交流。

世界上最幸福的爱情和婚姻，其中一个最重要因素就是三观一致。双方有共同话题，三观基本一致，能够真正懂得并理解对方的内心世界和感受。

完美的爱情和婚姻能够彼此疗愈伤痛。之所以不要急于进行身体层面的沟通，是因为人在刚认识时，都会展现自己最好的一面，即保护层或面具。

两个人真正想要在一起，需要向对方展露彼此的脆弱层。只有能够疗愈彼此的伤痛，才能建立心与心的连接。我们需要花时间进行深层交流，才能知道是否适合一起走下去。

如果不考虑这些，而是急于通过荷尔蒙或多巴胺的分泌进行身体沟通，很多人会不珍惜这段关系，关系的感觉和本质也会很快变质。

爱情非常美妙，每个人都体验过它的甜蜜和愉悦。然而，多巴胺和荷尔蒙的分泌终有一天会消失。真正能让两个人长远走下去的，是共同的目标、三观和意图，可以理解为同频共振。

如果你有另一半，可以带着他一起学习，或者确保他的思想层面与你相当，这样才能保证双方三观一致，在遇到问题时能够先反省自己，懂得沟通的艺术。

如果你和另一半情商都很高，即使发生摩擦冲突，双方也能快速冷静下来。你可能会想：“我刚刚情绪过激，攻击了我的另一半，我想去道歉。”而你的另一半可能也有同样的想法。

这样双方都想道歉，争吵就不会升级。这才是能够持久的婚姻，而不是互相斗气，非要对方先道歉。

两个人的三观、思想和共同目标一致非常重要。我与许多婚姻幸福美满的前辈和企业家交流过，他们给了我两个字：**同修**或**共修**。

同修共修意味着大家要朝着同样的目标努力成长。几乎所有能够同步进步成长的夫妻都能走得更长远，他们的婚姻和恋爱质量也更高，因为他们彼此能够真正理解对方。

选择另一半时，首先要问自己三个问题：
*   你真的信任他吗？
*   你真的愿意在对方面前坦诚吗？
*   对方能给你想要的安全感吗？

这三个问题至关重要。如果无法满足，那么对方很可能不是对的人。亲密关系意味着毫无恐惧的信任。如果你无法对伴侣全然敞开心扉、绝对信任、疗愈彼此伤痛并获得安全感，说明这段恋情可能并非你的最佳选择。

我之所以讲这节课，是因为很多年轻人面临感情困扰。希望大家学习这些核心后，能够收获幸福甜蜜的爱情。经营恋爱也需要智慧。

我们人都渴望爱、和平与开心。谈恋爱不就是为了开心吗？如果谈恋爱让自己很难受，那为什么要谈呢？有人会说：“我刚开始也没想这么多。”这正是因为开始没想清楚，问题就随之而来。

## 二、长期关系：理解关系的三层结构

上一节我们探讨了恋爱核心，本节中我们来看看长期关系的结构。

人与人之间都是关系的产物。我们每个人有三层壳。

第一层是**保护层**，即经过包装美化的第一印象。大多数人际关系都停留在保护层。

当两个人互相吸引，保护层就会打开，这时会看到对方的**脆弱层**。

脆弱层包含一些负面行为、过往经历留下的伤痛，或像小孩子的一面。例如，男生恋爱时会像小男孩，女生恋爱时会像小猫咪，还原到纯粹的孩子状态。

很多夫妻或情侣一开始觉得对方很好，但看到对方的脆弱层后，可能会觉得对方有很多恶习。特别是在深度相处，如同居或长途旅行后，会看到意想不到的一面。

你可能会发现心中的“女神”有不爱洗澡、抠脚等习惯。这就是所谓的“人设崩塌”。因为每个人内在都有一个孩子，那个孩子可能不讲道理、自私，但也渴望爱。每个人都希望得到理解、包容和共情。

恋爱后，如果双方都只想向对方索取爱，而无法给予爱，这段关系就很难维护。当我们向亲密的人打开脆弱层，我们渴望得到的是**共情**。

很多人际关系出现问题，正是因为打开脆弱层后得不到共情，于是又会重新关闭脆弱层，甚至加厚保护层。

你可以回想一些经历，比如向好朋友敞开心扉却被嘲笑，你会很受伤，然后马上关闭脆弱层，以后也不再对他打开。或者对父母也是如此。

很多人长大后性格有缺陷，原生家庭的影响特别严重。例如，孩子想要得到父母的理解和共情，却被说笨或做不好，这会打击他的自信心，导致自卑。这些影响在长大后就会呈现出来。

能向你打开脆弱层的人，是基于零恐惧的信任。所以，以后无论遇到谁，只要他向你敞开脆弱层，请好好共情，不要讲道理。他不需要解决方法，只需要你能疗愈他的伤痛。

例如，你是我的学员，经历了一段不愉快的爱情后向我倾诉。如果我直接说：“叫你好上课你不听，现在遇到问题了吧？”你听到后会崩溃，以后也不会再找我敞开心扉。

当别人向你敞开心扉时，请好好关爱对方。宇宙是二元对立的，你想要别人共情你，你就要去共情别人。你经历的许多痛苦，可能正是你之前给予别人的东西。

请管好自己的嘴巴。当你说别人自私时，其实也是在说你自己自私。

第三层是**本我层**，即最根源的灵魂深处。能够伤害你的人，往往就是你最信任的人。

现在很多人嘴上说的爱，并非真爱。我观察了许多案例后发现，大多数爱是基于**占有、执着、索取和控制**。

很多时候，你说“我爱你”的信念其实是：“我很喜欢跟你在一起的感觉，我想永远占有这种感觉。”这就是大多数人的信念系统。因为人的本质都是渴求爱的。

婚姻很多时候是一种需求，需要满足四个方面：
1.  **生理需求**：身体层面的需求，如抚摸和性。
2.  **物质需求**：经济基础，因为生活需要钱。
3.  **头脑需求**：思想交流、三观是否一致等认知层面。
4.  **情感需求**：内心需求，如“你爱不爱我”、“我是不是你的唯一”。

每个人内心深处都渴望无条件的爱，但我们往往给不出，因为自己杯中的水没有满。我们给出去的东西常常要求回报，而这往往没有好效果，反而让自己难受。

例如，年轻人经常请客。如果你和朋友出去总是你请客，对方却毫无表示，你也不是特别富有，你就会想：“这人怎么这样，太自私了。”这说明你给出去的东西是求回报的。

有意思的爱情是，什么都没发生，但和这个人在一起就会莫名开心傻笑。爱是一种喜悦感，看到他就开心。但如果你认为“我的开心需要对方负责”，痛苦就开始了。

当你希望对方为你的开心负责，就是把开心建立在自己身体以外的东西上。例如，如果你希望酒能带来开心，往往就不开心；如果你是因为开心而去喝酒，就会很尽兴。抽烟也是如此，你把放松寄托在烟上，没烟时就会难受。

所以，请马上改变“希望对方为你负责”的信念。没有人需要为另一个人负责，否则痛苦就会来临。

“情”可以理解为一种**感恩**。很多人爱情谈着谈着变成亲情，是因为感情不断累积加深。

如果你想拥有幸福的婚姻和恋爱，必须先从自身改变。你必须为自己的开心负起100%的责任。当你对对方没有任何需求时，对方反而会离不开你。你会从索取开心变成吸引开心。

这段话对于与异性相处非常重要。如果你想改善与所有人的关系，必须为自己的开心负起100%的责任。当你对他人没有需求时，对方就会离不开你，关系会从索取变为吸引。

我们最大的敌人是负面信念。有同学听了情绪管理课后，虽然有了意识，但经历某些事时还是会难受。知道问题所在，就是改变的开始。改变积累了二十多年的信念系统不可能一蹴而就，但这是根治的第一步。

后面需要你自己不断觉察。就像发烧看医生，医生开药后，还需要你自觉多喝水、调整饮食。宇宙的规律就是如此。

当你爱的人出现时，不要急于表白、发生关系，或急于满足彼此需求。因为有需求就可能有失望。所有的期待总是以失望告终，因为有需求就会有落差。

你应该做的是与对方确定**共同的愿景和目标**。这是我马来西亚的一位大哥告诉我的，他和妻子恩爱几十年，从底层打工做到拥有多家幼儿园。

他告诉我，谈恋爱要学会“谈判”，不是讲条件，而是确立共同愿景和目标。当双方认可共同目标时，就能永远同频，共同成长。这就像合伙开公司，必须双方都认同这个想法和目标，才能一起努力。

婚姻也是如此。谈恋爱千万不要想着去改变另一半，因为人无法改变另一半，只能影响和创造。

如何创造另一半？例如，你觉得另一半怎么看都不顺眼，越来越丑。这时你要改变信念，不断告诉自己：“我的另一半就是最漂亮的。”坚持这样说，他就会变漂亮。就像矫正牙齿需要时间一样。

刚刚说的是正念，那正语是什么呢？就是**只说对方的优点**。如果你希望对方是什么样的人，就用肯定的话语去说。

想象一下，如果你结婚后，每天早上出门工作，老婆都会抱一下你说：“老公今天要加油哦。”然后亲你一下。你回到家，老婆说：“老公你回来了，家里多亏有你，我们才能过得这么幸福。”即使你很累，听到这些温暖的话，也会瞬间充满能量。

中国人大多很内敛，觉得做比说重要。但并不是每个人觉察力都很强，默默付出的人可能会觉得委屈：“为什么我付出这么多你永远看不到，而我犯个小错误你就一直放大？”这样的关系肯定会出问题。

当你的另一半很贴心或做了一些事情时，一定要赞扬他。例如，他突然给你做早餐，你就要感恩他。爱除了要做，还要表达出来。这是最廉价、最不需要钱的，你只要说就可以了。

## 三、自爱与自私：本质的区别

上一节我们讨论了长期关系的构建，本节中我们来区分自爱与自私。

很多人都搞不清自爱与自私的区别。很多人把爱理解为只在乎自己的利益和感受，只顾自己。

其实自爱与自私的能量完全不同。就像自信是高正频高能量，自我是低正频低能量；自爱也是高正频，自私是低正频。

爱是发自内心纯粹的无条件的爱，而自私是出于恐惧、匮乏和不安全感。

当我们说爱自己时，爱的不是我们的头脑或自我，而是内在真实的自己，即我们每个人的真实身份——那个无条件的爱。每个人的内在都渴望喜悦、和平、爱，渴望无缘由的喜悦。

你有没有发现有时会莫名开心？这就是无缘由的喜悦。例如，有学员听了课后，感觉像发现新大陆，整个下午都非常兴奋。这种喜悦是很宁静、和平的。

真正的自爱是让我们回归到本能、真实的本我状态，就像小朋友一样。小朋友表达爱时，是本能纯粹的，没有夹杂很多意识或社会认知。他们不是想得到什么，只是想和爸爸妈妈黏在一起。

而我们带着很多意识和杂念，做事情往往有目的性，就不纯粹了。

如果你很难体会那种感觉，建议多跟小朋友玩耍或观察他们，你就能找到那种感觉。很多人所谓的自爱，出发点是自私，会陷入自我中心，是一种负能量，这不叫自爱。

真正爱自己的人，不会伤害任何人，也不会让任何人感到不舒服。因为真正的爱自己，是让自己活出光的状态，成为光，这样黑暗靠近你也会变成光。这就是爱的能力。

真正的爱自己，是专注在提升自己的内核能量上，而不是专注在头脑上。因为我们的身体和感受永远不会骗人，但我们的脑袋经常欺骗自己。

当我们顺应宇宙规律做事时，会感觉良好，因为与内心同在。当我们逆天行事时，灵性会远离我们，我们会感到种种负面感受。

例如，你打车发现后座有别人遗漏的50万现金。你内心可能有个声音说不能拿，但另一个声音说你需要钱。如果你拿了，就是在逆天行事。或者，你去陷害某人，即使别人游说你，你内心也会纠结不安。

做不好的事情会让你一辈子不安宁，因为你与内心断开了连接。我们的内心渴望爱、和平、喜悦，做让自己不舒服的事就是违背内心。

所以一定要分清楚，自爱与自私不一样。就像我不断学习后，能量越来越强，能够去爱身边所有人。我教学辅导时，不会指责学员，只会帮他照镜子、找到本质、激励鼓励关爱他。学员会感到温暖，因为我在用心讲课辅导。

这就是爱。做教育一定要有爱，而不是把学员当韭菜割。只要面对大众，想让生意越来越好，就一定要有爱，特别是做教育。

对于我们的内心来说，它没有分别心，不会把人分为高低贵贱。无论你对别人做什么，你的内心都会认为这是在为自己做，这就是二元对立。

佛教讲因果，就是你种什么因，得什么果。例如，你现在骗人钱或感情，终有一天会有另一个人来“收拾”你。天道有轮回，就是如此。

所以，你想要有爱，就要给出爱；你想要有贵人帮你，你就要成为别人的贵人。我找到了人生的意义：不断给别人提供价值。所以我活得特别自在，也有很多大咖认可我，愿意带我一起赚钱、介绍优质资源给我。

如何自爱呢？你每天至少要留1到2小时，做一些让自己开心、享受、有乐趣的事情。当然，打游戏可能只是满足头脑刺激。你要做一些真正让内心开心的事，比如学习、健身、种花、遛狗等。去给自己充电、滋养自己、补充能量，这才是真正的爱自己。

当你成为一个自爱的人，你会发现身边的人全部都会换掉。磁场一变，就会吸引完全不同的人。我这一年的转变非常大，就是因为转换了磁场。

## 四、终极建议：像单身一样生活

最后，我要讲最重要的一点：无论你是否谈恋爱或结婚，都请你像单身那样去生活。

很多人对生命中最重要的事情表现得很盲目，比如结婚、生孩子。很多人在进入婚姻或成为父母前，并没有想清楚如何经营婚姻或做好父母，导致这方面的教育和专业知识非常匮乏。

一旦进入婚姻或成为父母，生活就开始出现无穷尽的问题和烦恼。生活中大多数人的情况是：没结婚的想结婚，结了婚的想离婚，离了婚的不知道还要不要结婚。

想要了解真正的问题，我们需要去学习婚姻到底是什么，成为父母的意义又是什么。婚姻的本质我在前面已经讲了。

单身生活意味着你要能够为自己的人生负起100%的责任。你要完全有能力照顾好生活、财务，管理好情绪，调节痛苦，过上自己想要的生活。这才是单身生活的完美标准。

当你这么做之后，就会吸引到类似的人。但世界上很多人的单身生活并非如此，他们总是在等待另一个人来满足自己、解决所有问题和困难。

很多人幻想世界上有一个人可以完全理解、疼爱、无条件宠爱并满足自己一切需求。抱着这样的幻想进入婚姻或成为父母后，就会收获无尽的失望和无奈。

婚姻也讲究同频相惜。一个自爱的人，会吸引另一个自爱的人；一个匮乏的人，就会吸引匮乏的人。

有同学问我：“为什么我总是接触到那些要索取我钱的人？”如果他没上过我的课，我可能会直接说：“因为你也是这样的人。”宇宙就是同频相惜，你是什么样，就吸引什么样的人。

有人说：“我整天遇到渣女/渣男。”其实你自己检讨一下，你是什么体质？如果你是渣男体质，就会吸引渣女；你是渣女体质，就会吸引渣男。如果你能找到反例，请告诉我。

有人总是渴望：“我的另一半什么时候出现？我希望他/她是爱我的。”这句话真正想表达的是：“我希望有一个人来为我的生活、我的喜怒哀乐负责。”当我们这么做时，不可能拥有快乐和幸福。

所以，你对另一半的需求越少，你在这段婚姻中的幸福感就会越高。这适用于情感领域。我知道有些人还没定性，想体验更多恋情，这是你自己的选择。

无论如何，你对一个人（无论是异性还是同性）的需求越少，你的满足感就会越高，而且不会有失望，因为你没有期待。你可以自己过得很好、非常精彩。

这时，你会吸引很多人想要靠近你。这是我的亲身经验。我活出了自己，身边每个人都喜欢我，他们觉得我的眼神温暖、亲和力强，非常信任我。他们只要看到我、感受到我的能量场，就会产生信任。

这并不是用了什么套路技巧，而是掌握了道的本质。我知道我是什么样的磁场，就吸引什么样的人。当你能活出状态、自爱时，就会散发出无限魅力和吸引力，很多人会想要得到你、欣赏你、喜欢你。

真正的所谓“女生倒追”，其本质和道就是如此。我之前把赚来的钱都用来学习，增长智慧；出去旅游，开拓世界观；每天不断提升自己，同时又把学到的好东西分享出去。这就形成了一个正向循环，所以我越来越好。

很多人问我为什么成长那么快。原因就是我在不断付出。当我无条件付出时，自然就会得到不同的回馈。这真的很神奇。

本节课中我们一起学习了情感关系的核心。我们探讨了心灵沟通在恋爱中的优先性，理解了长期关系的三层结构（保护层、脆弱层、本我层），区分了自爱与自私的本质区别，并最终领悟到：无论关系状态如何，都要像单身一样为自己的人生负起全责，通过提升自身能量和减少对外需求，来吸引并建立健康、持久的亲密关系。
