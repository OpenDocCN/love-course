## 什么是正确的展示面？ ✅

上一节我们介绍了课程主题，本节中我们来看看什么是正确的展示面。展示面是个人形象在网络上的视觉呈现，关键在于通过照片传递积极、有吸引力的个人状态。

以下是正确与错误展示面的对比示例：

*   **错误展示面示例**：人物在著名景点（如印度尼西亚）拍摄，但照片中人物显得肥胖、姿态不佳，未能与环境协调，未能体现个人魅力。日常打扮油腻的照片也不适合作为展示面。
*   **正确展示面示例**：在澳门网红酒店等优质场景拍摄，照片清晰，人物姿态经过纠正，能凸显人物状态和氛围。例如在香港尖沙咀等地拍摄的照片，人物与环境融合得当。

通过这些例子，我们可以明确一个原则：展示面的拍摄和选择需要技巧，目标是呈现清晰、有状态、环境协调的个人形象。

---

## 第一步：学会模仿构图 🎯

理解了展示面的标准后，我们面临第一个问题：如何拍出构图优秀的照片？本节将介绍最直接有效的方法——模仿。

对于初学者，无需深究复杂的构图理论（如三分法、黄金分割点）。最高效的方法是使用“小红书”等平台，直接模仿网红博主的成功构图。

以下是具体操作步骤：

1.  **确定地点与关键词**：打开“小红书”APP。根据你所在或想要拍摄的城市（如澳门），搜索相关地点关键词（例如“澳门 巴黎人 酒店”）。
2.  **寻找参考样张**：浏览搜索结果，找到你认为构图、氛围优秀的照片（通常是网红小姐姐的打卡照）。将喜欢的照片截图保存。
3.  **分析并模仿构图**：仔细分析参考照片的构图。例如，一张埃菲尔铁塔与人物的照片，其黄金分割点大多落在铁塔上，人物仅置于角落，这样既突出了环境，人物又不显突兀。
4.  **实地拍摄**：前往相同或类似机位，严格按照参考照片的构图进行模仿拍摄。即使天气、光线条件不同，先保证构图准确。

这个方法的核心理念是：**拍照的第一步是模仿**。通过模仿优质作品的构图，你能快速提升照片的基础质量。

## 第二步：人像精修——基础面部调整 ✨

在获得一张构图合格的照片后，接下来需要对人物本身进行优化。本节我们将学习使用“天天P图”软件进行基础的面部修饰。

修图的正确顺序非常重要。首先应从面部调整开始，流程是：**修饰脸型与皮肤 -> 全局裁剪/旋转 -> 整体滤镜调整 -> 局部精细补偿**。

以下是使用“天天P图”进行面部精修的具体步骤：

1.  **打开软件并导入照片**：启动“天天P图”APP，点击“美容美妆”，导入需要修饰的人像照片。
2.  **使用“瘦脸瘦身”工具**：
    *   点击“瘦脸瘦身”功能。
    *   处理头发边缘杂乱的发丝，用手指或画笔向头发内侧推动，使其看起来更整洁。
    *   **公式**：`修饰后轮廓 = 推动(原始杂乱边缘， 朝向头发主体方向)`
3.  **使用“祛痘祛斑”工具**：
    *   点击“祛痘”功能。
    *   点按面部明显的斑点、痣或皱纹，软件会自动将其消除。
    *   **代码逻辑**：`for 斑点 in 面部斑点列表: 应用算法(斑点区域, 替换为周围皮肤纹理)`
4.  **使用“五官重塑”工具**：
    *   这是核心功能，需谨慎使用。
    *   **整体调整**：“整体”滑块控制脸宽，左滑变胖，右滑变瘦。切勿过度调整导致失真。建议先大范围调试，再微调至自然状态（例如，调整30%以内）。
    *   **局部调整**：可分别调整脸颊、下巴、额头的大小或位置。原则是弥补缺陷，追求自然协调。
    *   **眼部调整**：“眼睛”滑块可调节大小，适度放大（10%-20%）可使眼睛更有神，但切忌过度。
    *   **眼距调整**：“眼距”滑块需非常谨慎，轻微调整（如5%）即可，过度会导致斗鸡眼或五官分散。

**关键提示**：所有调整都应以“自然”为第一准则。了解自己的脸型特点，进行有针对性的微调，而不是套用模板。
