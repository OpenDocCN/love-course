# 课程17：《情敌攻略视频课》+《情敌手册书》：10.后记(1) 📖

在本节课中，我们将对《情敌攻略》的全部内容进行总结，并深入探讨学习这套策略的正确心态与行业现状。这不仅是课程的结束，更是你真正实践的开始。

以上我们讲解的就是关于情侣策略的所有内容。具体细节你需要去阅读《情敌手册》。我建议，如果你完全是一个新手，书中的很多内容，最好在你进行一些实战之后再来回顾。如果你已经具备一定的与女生相处的经验，我建议你将书中的内容多看几遍。

我在撰写这本书时，很多内容都是反复推敲，写了又删，删了又写。这本书基本上保证了一件事情：**全书没有一句废话**。因此，有时你一眼带过的某句话，可能就是我数百次实战经验的总结。希望大家能够认真研读。

接下来，我需要说明一点。这本书可能会让许多小白兄弟，或者之前对“把妹”存在错误理解的兄弟，在阅读后感觉似乎不是他们想要的，或不是他们想学的。

这时你就需要思考了。我写的是 **“1%的情侣攻略”**。如果这都不是你想学的，那只能说明一个问题：你压根就不想学习如何与异性建立关系。

我培养了这么多学员和导师，他们都成功地找到了伴侣。我衡量这套方法是否有效的标准在于：我把它教给别人，别人能否成功？那些成功的人看到这些内容是否有共鸣？令我欣慰的是，我已经将抢先版发给了许多实战经验丰富的朋友，他们的反馈非常好。

如果你觉得这个课程对你没有帮助，似乎不是你想要的，那有可能你对“把妹”的思考本身就是完全错误的。

举个例子：你想要健身，想要变瘦。你却想去买一种“灵丹妙药”。结果发现，它告诉你要去跑步、要做器械，同时指导你如何正确跑步、如何正确进行器械训练，包括如何制定整体的健身课程规划。

但你却躺在床上抱怨：“这不是灵丹妙药吗？我应该有一种药，吃了之后躺在床上就能立刻长出八块腹肌。” 这只能说明你对“灵丹妙药”这四个字的理解有问题。**这种东西是不存在的**。

如果市面上有人告诉你，吃了某种药三天就能瘦十斤，你就应该明白那是骗子。那么，你为什么会有这种“寻找灵丹妙药”的思想呢？

原因在于，现在全国所有从事情感教育的那帮人，手里没有真材实料。他们写一篇文案，十个人抄袭，然后发在抖音上。你看了觉得舒服又简单，就付钱买课。因为你觉得那是“灵丹妙药”——这个人吃了药，一下就瘦了20斤，效果很快。

但事实上，真正的改变过程是痛苦的。他们为了收割你的钱、为了赚钱，怎么会告诉你过程如此痛苦呢？只有我会告诉你：**这个过程很痛苦**。但只有这样，才能改变你在情感上没有进展的现状。

最近半年，我加入了一个同行群。群里面永远在讨论如何搞流量，如何发布内容能吸引你关注他，但从来没有讨论过如何解决那些在情感中遇到问题的男性的实际困难。

这帮人虽然是同行，但根本不进行业务交流，天天想的只是如何赚钱。这让我想起十年前我去参加一个行业峰会After Party的情景。当年，论坛上有一堆所谓的大神，天天在网上鼓吹自己把妹多厉害。

我去了那里参加峰会，晚上去夜店实践时，发现那帮傻逼在包厢里干嘛？他们并没有在实践把妹，而是在讨论如何赚钱。我一下子就明白了：这帮人就算自己会把妹，也绝不会教你真东西、真本事。

为什么？因为他们的目的是赚钱。不然，行业峰会应该讨论如何把妹，对不对？凭什么讨论如何赚钱呢？我来参加峰会，不是来讨论如何赚钱的，而是想看看你们这帮“大神”在把妹上到底有什么新东西可以让我学习一下。

因为学员，因为在座的各位，你来学的是把妹，你来学的不是如何赚钱。所以说，我们行业的这些峰会也好，内部群也好，应该讨论如何解决大家的情感问题，而不是讨论如何赚钱。

所以我觉得这个行业没救了。于是，我在2012年进入了这个行业，创立了 **“实战派”** ，把这帮人都赶走了，全部赶到了哪里？赶到了“女性情感”领域。因为很简单，在男性领域混不下去了，就开始去教女性了，毕竟女性领域没有“实战派”跟他们竞争。

到了2017、2018年，因为我过于注重实战，大家也都知道，我被针对了。然后这两年，这帮孙子们看到没人管他们，又冒出来了。一个个的在那里搞什么“键盘话术”，鼓吹“一句话扭转女人对你的心意”，这让人很崩溃。

因此，我思考了很久，想要写点真东西、教点真本事给大家。这就是我这次推出《情敌攻略》的原因。

**你去看，去做，去想。关注群里的反馈，坚持三个月。**
