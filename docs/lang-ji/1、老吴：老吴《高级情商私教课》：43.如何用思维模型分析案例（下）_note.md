# 1、老吴：老吴《高级情商私教课》：43.如何用思维模型分析案例（下）_note

在本节课中，我们将继续深入分析一个实战聊天案例，学习如何运用思维模型，特别是ABC模型（A-事件，B-信念，C-反应），来拆解对方的言行并做出高情商的回应。我们将重点探讨在面临强势框架时，如何避免无效的对抗与解释，转而使用更巧妙的沟通策略。

## 案例背景回顾

上一节我们分析了一个女生的发言，她的框架非常强，让很多同学感到难以招架。她的发言是：“男人都是下半身思考的动物。”

对于这样的言论，常规的回应方式往往效果不佳。上一节我们介绍了ABC模型，本节中我们来看看如何具体应用这个模型来分析和应对。

## 核心策略：避免“头对头”的对抗

当对方抛出强势框架时，人的本能反应有两种，但这两种通常都会导致沟通失败。

以下是两种常见的错误回应方式及其后果：

1.  **用认知对抗/教育对方**：例如回复“那是你认为的吧”或试图讲道理。这相当于用自己的大脑（认知）去攻击对方的大脑，结果往往是争吵和输赢之争。即使你“赢”了辩论，也可能“输”了关系。
2.  **低姿态解释**：例如急忙否认“我不是那样的男人”。这会被视为低姿态向高姿态的辩解，不仅显得框架弱，还会暴露你非常在意对方的看法，陷入被动。

既然这两种路径都行不通，我们就需要寻找第三条路。

## 高情商破局法：装傻与转移

在解释不行、对抗也不行的两难境地中，一个非常有效的策略是：**装傻卖萌**。

具体操作是，用一个简单的“啊？”来回应。这个字像“万金油”，能瞬间转移对方的注意力，打断其原有的强势框架。女生通常会因此感到好奇，因为一般的男人不会这样回复。

**核心代码/行为描述：**
```
当 收到强势框架言论（如“男人都是下半身思考的动物”）：
    如果 选择解释或对抗：
        结果 = 大概率沟通失败
    否则：
        回复 = “啊？”
        目标 = 转移注意力，跳出原有框架
```

## 深度分析：运用ABC模型理解对方

在做出“啊？”的回应后，我们内心需要运用ABC模型进行深度分析，而不是真的懵懂无知。

*   **对方的C（反应/言论）**：就是她说“男人都是下半身思考的动物”。
*   **对方的B（信念）**：为什么她会有这样的结论？可能源于她的经历（A）。例如：
    *   她之前遇到的男生可能都只想发生关系，之后便离开。
    *   她身边的闺蜜朋友可能经常灌输“男人很坏”的观念。
    *   她所处的社交环境（如爱玩的圈子）塑造了这种认知。
*   **对方的A（诱发事件）**：上述的负面经历或环境输入。

理解到这一点，我们就明白，她的言论**并非针对个人的攻击，而是其自身信念系统的投射**。作为一个高情商的人，我们要学会理解和包容这种认知差异。

## 进阶心法：升维思考，降维沟通

理解了对方的B和A之后，我们的心态会发生根本转变。我们不再视其为需要击败的对手，而是可以将其看作一个认知受限的“小朋友”。

**核心公式：**
```
高情商心态 = 理解（对方信念源于其经历） + 包容（不将其言论视为攻击） + 升维视角（将自己置于更高维度）
```

因此，当女生问“啊什么？”时，可以回复：“没有啊，觉得你说话很可爱囉。” 这句话的背后含义是：我看到了你的认知模式，但我并不计较，反而觉得这种直率的表达很有趣。这体现了包容和更高的维度。

## 实战对话拆解与技巧延伸

基于这种心态，后续的聊天就能如鱼得水。

1.  **巧妙衬托，不落下风**：当女生回应“这么会说话的”，可以回复：“**没有你配合，会说话也没有用啊。**” 这句话既肯定了对方，又保持了自身框架，不是跪舔，而是高超的共情与话术。
2.  **应对标签，重新定义**：当被女生贴上“特会撩妹”的标签时，避免直接承认或否认。
    *   **错误回应1**：否认解释（“我没有”），显得心虚。
    *   **错误回应2**：自豪承认，坐实“花心”标签。
    *   **高情商回应**：“**能让人开心不好嘛？**” 将“会撩”重新定义为“提供情绪价值”的能力，与“欺骗感情”的“渣”划清界限。
3.  **反客为主，引导话题**：当聊天深入，女生试图评价你“经常去夜店”时，可以抢先一步说：“**一句话暴露了你经常去夜店的行踪。**” 这实现了话题主导权的转换。
4.  **创造共鸣，深入连接**：聊到对方擅长或喜欢的话题（如夜店、旅行），鼓励对方多分享。心理学表明，当一个人向另一个人倾诉很多时，内心会不自觉地拉近关系。这适用于寻找真正的心灵契合。
5.  **友好互动，而非对抗**：通过适度的自黑（如说自己30岁老了），可以引发对方的安慰（“三十岁正是一朵花”），从而将聊天氛围从“对抗”转向“友好互助”，让双方都感到愉悦。

## 课后练习与核心要求

要掌握这种分析能力，仅靠听课不够，必须投入实践。

以下是给你的练习指南：

1.  **精细拆解**：拿到一段聊天记录，用Word等工具逐句分析。
2.  **列举选项**：对于每一句 incoming message，思考可能的回复方式（至少3-5种），并推演每种回复可能带来的后果。
3.  **做出选择**：基于分析，选择你认为最合适的回复，并说明理由。
4.  **验证与倒推**：根据对方的实际回复，验证你的预判是否正确。如果不正确，进行倒推分析：为什么对方会这样回？他的B（信念）和A（经历）是什么？
5.  **持续追问**：不断运用“为什么”和“凭什么”这两板斧，深入挖掘言语背后的本质。

**关键提醒**：成功取决于你的投入程度。课程提供了最核心的思维模型（如ABC）和逻辑思考方法，但如果你不愿意思考、不完成作业、不刻意练习，那么知识将无法内化。这就像有人为你剥好了龙虾，甚至喂到你嘴边，你仍需自己咀嚼吞咽。
