## 拍照的重要性

一个男孩子会拍照，现在成为恋爱中最重要的一步。随着科技发展，认识异性的渠道越来越多。想象一下20年前，村头和村尾各开一家小卖部，它们可以互不影响地做生意。但放大到现在，互联网如此发达，如果有一家店被公认为不错，其他人都会跑到这个不错的地方买东西，而做得不好的就无人问津了。

现在谈恋爱也是同样的道理。但凡一个女孩子的朋友圈里出现一两个条件很不错的男生，其他朴实无华的男生就无法竞争了。这里问一下，你有谈恋爱吗？没有。为什么呢？身边没有好的。身边为什么没有好的呢？因为她见过了鲜花，谁来理这种野草。

所以说，在女孩子身边现在流行着一种话：宁愿高质量的单身，也不愿意低质量的恋爱。原来，女孩子没有看到朋友圈或社交网络上有这么多好的男孩子，她可能随便找个朴实无华的男生将就。那个时候，对于很多朴实无华的男生，还是有脱单的机会。但现在不一样了，女生不愿意去将就了。所以说，如果你的照片没有做好的话，你得到的最终结果，就是可能没有恋爱可以谈。

## 快速提升照片质量的秘诀

很多男孩子说我不会拍照。今天教你一个秘诀：去找一个会拍照的异性好友，比你去学半天靠谱得多。

以下是具体操作步骤：

*   **第一步：线下约拍**。找一个周日或周末的下午，约这样的异性好友出来，下午茶喝一喝，让他给你拍几张照片，再帮你修修图。这非常靠谱。
*   **第二步：线上求助**。有了这样的异性好友，平时你的自拍照如果不知道怎么去修，直接发给他。5分钟后，一张修好的照片就出现了。
*   **第三步：多加好友**。当你不知道怎么去拍照的时候，可以多加几个会拍照的小姐姐在你的朋友圈里。每天翻翻朋友圈，一来让你心旷神怡，二来久而久之的潜移默化，你就会形成一种独特的拍照风格。

上一节我们介绍了快速提升照片质量的社交方法，本节中我们来看看，如果作为拍照新人，找到了会拍照的朋友，该如何起步。

很简单。在你的每个城市，都有这种拍照的网红打卡胜地。就算你的拍照技术次一点，就算你的穿着打扮次一点，就算你的形象外在次一点，但在这种网红的打卡地，它就自带非常漂亮的背景。无论怎么乱拍，首先照片看起来也是人模人样。

所以说以上就是如何学习拍照的要点。

## 拍出好照片的四个核心要素

不知道兄弟们学会没有。那么，男孩子想要拍出一张比较漂亮的照片，需要具备4个要素。

以下是四个核心要素的详细说明：

1.  **构图**。比如说我要拍一张照片，拍一下过后，如果是全身的话，你的相机要**由下往上拍**；如果是半身的话，要**由上往下拍**。这是一个构图的细节。
2.  **光线**。千万不要坐在背光的地方。如果你的脸上没有亮度，或者说很阴暗，这样照片90%都非常不好看。
3.  **环境**。一定要选择一个干净的环境。这种环境后面不能有消防栓、垃圾桶、指示牌等与主题无关的闲杂元素。
4.  **动作**。你的动作一定要自然，不能给别人一种很强烈的凹造型或摆拍痕迹。那么怎么拍出自然的照片？就是你必须用一个背景拍非常多的照片，然后从中选出最自然的那张。据我所了解，一个女孩子她的朋友圈里，如果你看到是张单纯站在路边的照片，有可能是她拍了100张过后才选出来的。

## 社交动态的发布与互动技巧

在社交软件上发布照片，也是需要运营技巧的。

上一节我们讨论了如何拍出好照片，本节中我们来看看如何有效地发布和运营这些照片。

以下是发布与互动的关键点：

*   **发布时间**。你一般什么时候会打开朋友圈翻来翻去？一般就是中午或者晚上回家休息的时候。那么我们给大家推荐的发布时间，就是上下班的通勤时间，以及社交软件使用的高峰时刻——晚上十点过后。在这三个阶段发布你的社交动态，可能会受到更多人的关注。
*   **配文技巧**。千万不要去网上搜索“如何给我的照片配文”这种土味情话或土味段子，90%的人其实是不喜欢的。类似于“我只能中午发照片，因为早晚你都会爱上我”之类的。
*   **互动方式**。当你发布了照片后，怎么去和女孩子互动？首先，如果是女生发的照片，你千万不要只给她点赞，因为她没有办法回复你。最好的方式就是给她评论。如果她再评论了你的话，就可以切到私信跟她聊天。对于跟你点赞或者评论的女生，你可以直接去私信跟她互动。
*   **内容分寸**。最后一点你要学会分寸，要照顾到你朋友圈里不同类型好友的感受。还有最最关键的点，我要提醒大家：我们每个人其实都有生活中的高光时刻，但不要去故意显摆或炫富，更不要去发布一些虚假的照片。现在网上有很多识图软件，一旦你被识别出来，就会非常尴尬。
