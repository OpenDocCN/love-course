## 第一小节：社交力学的原理 🔬

上一节我们介绍了课程概述，本节中我们来看看社交的基础原理。社交力学旨在帮助我们理解“好人缘”背后的真相。

什么是好人缘？好人缘等于一个人受人欢迎的程度。要理解它，我们可以先思考其反面：我们不欢迎的人有哪些特点？

以下是四个方面的总结：

1.  **语言层面**：嘴臭、爱说脏话、粗鲁、刻薄、爱钻牛角尖、爱抬杠、撒谎（表演艺术除外）。
2.  **行为层面**：爱占便宜、粗鲁野蛮。
3.  **外表层面**：邋遢、不修边幅，让人不想接近。
4.  **气味层面**：体味过重、口臭等个人卫生问题。

反之，好人缘的特点则是其反面：语言谦虚诚恳、行为大方稳重、外表整洁精致、气味清新芬芳。这些点本质上是在教我们如何“做人”。一个不会做人的人，在任何社交场合都难以建立良好的人缘。

这里引入一个重要的心理学原则：**七三八五五原则**。我们对一个人的印象，**7%** 来源于说话内容，**38%** 来源于语气语调，**55%** 来源于外形与肢体语言。肢体语言包括站姿、坐姿、行走姿态等。

了解了这些特质，我们如何提升自己呢？以下是针对不同方面的练习方法：

**关于语气语调的练习：**
*   模仿电影、电视或新媒体中富有魅力的男性角色（如梁朝伟、彭于晏、胡歌）。
*   通过大量阅读提升内涵，所谓“腹有诗书气自华”。
*   每天对着镜子练习说话。
*   录下自己的声音，回听并调整。

**关于行为与肢体语言的练习：**
*   模仿身边你认为有魅力的人（上级、同事、家人等）。
*   对着镜子练习肢体动作、表情和眼神。
*   录制视频，观察自己在镜头前的形象是唯唯诺诺还是落落大方。

**关于外表与形象的提升：**
*   模仿他人的穿搭。
*   多去服装店试穿不同风格的衣服，找到适合自己的款式。
*   找一个靠谱的发型师，设计一个适合自己的发型。
*   坚持每日护肤（清洁、保湿、敷面膜等）。

**关于气味的保持：**
*   建议每半年洗牙一次，并检查蛀牙。
*   坚持早晚刷牙，每日洗澡洗头，内衣最多穿两天。
*   选择一款清淡的香水或止汗走珠（尤其夏天）。

## 第二小节：行情决定论 📈

上一节我们探讨了如何塑造良好的第一印象，本节中我们来聊聊社交中的“行情”。行情决定了你在社交市场上的吸引力。

什么是行情？在投资领域，股市、房市、期货都有行情。行情指在特定时间段内，被市场所认定和赋予的价值。它不是一成不变的，具有时效性，是短期表现与长期潜力的综合评定。

例如，一个刚入学的北大学生，短期看可能没有财富，但长期看因其潜力而拥有“行情”。相反，一个中了彩票却挥霍无度的人，短期有“行情”，长期则没有。

以股票为例，贵州茅台在2001年上市时每股34.5元，到2019年每股约1157元，18年间上涨超过30倍。这就是“潜力股”的行情。

那么，社交中“渣男/渣女”有行情的根本原因是什么？在于他们当下拥有一定的**硬价值**或**软价值**。
*   **硬价值**：颜值、身材、车子、房子、地位、金钱。
*   **软价值**：情商、幽默感、趣味性、潜力、思考能力。

最终比拼的是一个人的**综合价值**。价值可以通过交换获得。作为一个普通人，价值的创造通常分为三个阶段：
1.  **投入时间，获取经验**。
2.  **积累经验，形成能力**。
3.  **运用能力，换取价值**。

例如，一个大学毕业生进入公司（投入时间），积累工作经验（形成能力），最后获得薪资报酬（换取价值）。

## 第三小节：社交障碍 🚧

上一节我们明白了行情可以靠努力获得，本节中我们直面阻碍许多人脚步的核心问题：社交障碍。

为什么我们会在社交中产生障碍？比如不敢搭讪、不敢在社交软件上主动聊天？其核心原因并非外在条件，而在于**心智没有打开**。

智商高代表学习能力强，情商高代表处理人际关系能力强，但**心智**关乎一生的成长与突破。打开心智的法门在于：**观察自己的思考过程**。

例如，想搭讪一个女生却最终没去，大脑可能会找理由：“她走得太快了。”但真实的思考过程是：首先你“不想”去搭讪，然后才为这个决定找理由。如果你决心“必须去”，那么无论她走得快慢，你都会行动。

我们的大脑在做决定时分为两部分：**理性思维**（客观分析）和**感性思维**（注重感受、体验）。我们找借口，往往不是理性分析的结果，而是感性部分（恐惧、懒惰、怕丢脸）在主导。

就像减肥，理性告诉你“早起跑步就能瘦”，但感性在闹钟响时说“被窝好暖和，多睡会儿”。我们找各种理由，本质上是自己先下了“不改变”的决心，再用理由去搪塞。

因此，克服社交障碍的关键是：**不要被自己找的那些理由所掌控，要学会接管并控制你的大脑**。

## 第四小节：社交必胜论 🏆

上一节我们拆解了内心的障碍，本节中我们将建立一个强大的信念：主动社交是一件必胜的事。

我们来做一道选择题：主动社交有哪些害处或损失？1. 时间成本；2. 丢面子；3. 自己不喜欢；4. 困难。

但仔细想想：
1.  **时间成本**：掌握任何技能都需投入时间，这不叫损失。
2.  **丢面子**：越没实力的人越爱面子。普通人无需过分在意“面子”。
3.  **自己不喜欢**：“喜不喜欢”往往是成功者的特权。在成长阶段，应更关注“需不需要”。
4.  **困难**：做什么事不困难？躺平最不困难，但对提升自己毫无帮助。

相反，主动社交的核心好处是：**把虚无缥缈的“缘分”把握在自己手里**，让自己拥有选择权。

为了坚持主动社交，我们需要给自己积极的**正向暗示**。以下是四个关键暗示：
1.  **必胜**：只要主动出击，目的就已达到。我们的目的是享受社交过程、锻炼自己，而非一定要到联系方式。
2.  **强与弱**：敢于冒风险的人才是强者。在繁华街头，大多数男生都想认识心仪的女生但不敢行动。只要你去做，就已超越了99%的人。
3.  **必须主动**：在当前的社交环境下，男性必须承担主动的角色。不要奢望女生主动。
4.  **重新定义拒绝**：不要把搭讪当成“面试”（对方决定你的去留），而应视为“国会投票”。你投出了你的一票（表示欣赏），对方也有权投出她的一票（接受或拒绝）。被拒绝是对方的权利，不代表你失败或做错了事。

## 第五小节：保持善念与正向信念 💡

上一节我们建立了主动必胜的心态，本节我们将探讨社交乃至为人处世的底层逻辑：保持善念。

为什么要保持善念？
1.  **协作的基础**：没有善念就没有协作，社会将寸步难行。从外卖到互联网，所有现代服务都依赖于大规模协作。
2.  **恐惧的世界**：如果人人充满敌意，生活将陷入无尽的恐惧和猜疑中。
3.  **避免极端剥削**：如果没有基本的善念，社会财富和资源将被极少数人垄断，绝大多数人会沦为“奴隶”。

一个有志向的人需要明白：**小富靠勤，中富靠趋势，大富靠德**。
*   **趋势**是预知未来的能力。
*   **德**是你的信念与价值观。
    *   **信念**：你坚信什么（如坚信努力成功 vs 坚信关系上位）。
    *   **价值观**：你认为什么有价值（如赚钱并帮助他人 vs 只要赚钱就行）。

看看大人物的信念与价值观：
*   **李嘉诚**：诚实、厚德，不让合作伙伴吃亏（“九一原则”），从而赢得更多机会。
*   **马云**：满足他人需求、成就他人。阿里巴巴通过解决社会需求而获得价值。

正确的社交信念与价值观可以归结为两点：
1.  对方需要什么？
2.  我能为对方提供什么？

秉持“利他共赢”的思维，你就能在社交中树立正确的信念。

## 课程总结 📝

本节课我们一起学习了2020年社交策略的核心内容：

1.  **第一印象决定社交的难易**：良好的个人形象和举止是社交的敲门砖。
2.  **行情可以通过努力获得**：通过提升硬价值和软价值来增强自身吸引力。
3.  **社交的前提是解开内心枷锁**：识别并克服自我设限的思维模式。
4.  **主动是最好的社交方法**：建立“主动即胜利”的信念，勇于行动。
5.  **正向的信念与价值观要贯彻始终**：保持善念，秉持利他共赢的思维。
