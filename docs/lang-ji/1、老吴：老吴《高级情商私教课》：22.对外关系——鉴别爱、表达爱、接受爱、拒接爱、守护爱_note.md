# 1、老吴：老吴《高级情商私教课》：22.对外关系——鉴别爱、表达爱、接受爱、拒接爱、守护爱_note

在本节课中，我们将要学习如何与他人建立更深层次的情感连接。我们将从五个核心方面展开：如何鉴别爱、如何表达爱、如何接受爱、如何拒绝爱以及如何守护爱。掌握这些能力，将帮助我们建立更健康、更滋养的人际关系。

## 第一部分：鉴别爱与表达爱 🔍

上一节我们介绍了与他人相处的基础，本节中我们来看看如何鉴别和表达爱。我们一生中最重要的学习之一，就是学会如何给予爱。这需要长时间的练习，并非一蹴而就。正如前面课程所讲，如果我们自己不够自爱，那么给出的爱往往带有索取性质，或者让自己感到难受和匮乏。

因此，我们的目标是达到一种境界：**我不在乎你是否爱我，我只在乎我是否爱你**。因为全部的喜悦和开心都蕴含在“爱”这个行为本身，与你是否爱我无关。**我愛你，與你無關**。

我们常听到“活出自我”。李嘉诚先生曾总结八个字：“活出自我，创造无我”。“活出自我”就是让自己变得开心、喜悦、充满爱。至于“创造无我”，那是更高的境界，我们后续会讲到。

我们从小到大所接触的爱，大多是有条件的爱。我们习惯了接受有条件的爱，也习惯性地给予有条件的爱。虽然生活在这种模式中，但我们内心深处依然渴望无条件的爱。

实际上，想要真正爱别人，我们必须先做到“水满自溢”。因为我们无法给出自己没有的东西。如果你自己不快乐，就无法让别人快乐。如果你没有金钱，也无法给予他人金钱。如果你的身体不健康，也无法让他人健康。所以，无论我们想给予什么，首先自己必须拥有它。无论我们多想爱别人，都必须先足够爱自己。

爱自己意味着与自己建立深度连接，活出内在力量，让自己的小宇宙爆发，达到“水满自溢”的状态。这种状态是心想事成，即使遇到痛苦，也能将其转化为开心的体验。

当你的内心充满喜悦、平和与爱，当你的物质世界丰盛富足时，你自然会有余力去爱别人。活出这种状态时，给予爱就成了自然而然的事，无需刻意去想为对方做什么。你自然就能给予对方所需，提升对方的能量，将溢出的爱给予身边所有人。

以我自己为例，因为我非常爱自己，所以也会爱我身边的人。我能将能量传递给你们，即使隔着屏幕，这种发自内心的东西也能传递出去。

你会发现，生活中许多人只会索取爱，不懂给予爱。提高情商需要学习和落地的第一步，就是好好爱自己，提升自我意识，打破固有认知，提高自身能量。当你的能量很高时，你会爱身边的每一个人，接纳身边的每一个人。因为那时你看待他人，会像看待三岁孩童一样。

那种状态就是：**我不在乎你是否愛我，我只在乎我是否愛你**。你会发现，给予比索取更开心、更幸福、更有力量。更重要的是“我到底爱不爱你”，因为我们的全部喜悦都在“爱”里面，与你爱不爱我无关。**我愛你，與你無關**。这就是我们要达到的目标。

达到这个目标时，我们每天都生活在给予爱的状态中，因为给予就是得到，这才是真正的开心。同学们可以在生活中尝试练习，将所有对爱的渴求转化为给予爱。一开始可能会犹豫或难受，这很正常。我们要学会在乎他人感受，满足对方需要，并力所能及地提升对方的能量和意识，让对方感受到爱。

以下是表达爱的几种方式：

*   **用语言**：直接、真诚地表达你的关心与欣赏。
*   **用行动**：通过具体的行为来体现你的爱意。
*   **用祝福**：发自内心地为对方祈愿美好。

你会发现，在给予爱时，你自然充满力量。这里所说的“爱”，并非仅限于男女之情或父母对子女之爱，也不是人们通常理解的有条件的爱。我所说的是一种超越人类普通情感的爱，是能达到“无我”、“天人合一”状态的爱。这意味着真正意识到：你就是我，我就是他，他就是你。我们之间并无分别，内心本质相同。

为什么这么说？我昨天给大家发了课程回访，想了解大家的收获。作业第一名的同学在凌晨一点多给我发了一大段文字。早上我看到后回复了他，并提醒他注意身体，别熬夜。他回复说：“我刚刚还在想老吴会不会关心我，结果就真的给了。”这是为什么？因为我们的内心有连接。

我看到他发来大段文字，第一眼注意到的不是内容，而是发送时间——凌晨一点四十四分。这意味着他为了反馈牺牲了睡眠时间。我自然能感受到他的付出，因此会肯定他，关注这个重点。于是，心与心之间产生了神奇连接。

当我们活出自我时，不会在乎对方是否爱我们，只在乎自己是否爱对方，是否能体验到给予爱的喜悦。这与他爱不爱我们无关。所以，当遇到你想爱的人时，可以在心里（或表达出来）说：**我愛你，與你無關，我只想好好愛你**。因为我很快乐，想把快乐给你；我很丰盛，想让你一样丰盛；我很健康，想让你一样健康。仅仅“我爱你”这件事，就让我感到非常开心。

希望大家听完这部分能有所启发。我对你们每个人都一样，希望你们能变得更好。这非常简单、纯粹。

## 第二部分：接受爱 🤲

上一节我们探讨了如何鉴别与表达爱，本节中我们来看看如何接受爱。实际上，没有人能拒绝一个真心对自己好的人，这非常神奇，你可以去体验。

在以前的课程中我讲过，无论学习多少社交或聊天技巧，最终都会回归一句本质的话：人与人关系的核心是**信任**。还有一句话请大家记住：**真心換來真情，虛情換來假意**。

这个世界上，所有动物中只有人类可以做到不真诚，只有人类可以虚情假意、可以伪装。所以在生活中，人与人交往最累的就是不真诚。生活中常有这样的人：背后讲你坏话，当面却说好话；明明讨厌你，却装作不讨厌。我也见过很多这样的人。我觉得人与人交往中最累的，就是因为不真诚。他们为什么要活得这么累？

我曾看过一位很厉害的人——钟萨仁波切的讲话，很有启发。他说：“当一个人可以做到绝对真诚的时候，他就已经无坚不摧了。”这表明，**真诚是一种强大的力量**，并非人人都能做到。

需要补充的是，真诚并不意味着讲话没有智慧、单刀直入伤害别人、不经过大脑思考。我们要在真诚与智慧之间拿捏好度，这是需要学习和提升的方式。

人与人的关系大多停留在虚情假意中，表面的客套、寒暄或礼貌，其实都是为了保护自己。但有一种力量非常难以拒绝，那就是**真心对你好的人**。你可以回想一下，在你的人生中，有哪些人是真心对你好的？在这个世界上，有一个人真心对你好，是非常幸福的事。

因为人都不傻，一个人对对方好不好，对方心知肚明；一个人为对方付出多少，对方心里也很清楚。我经常想起我的贵人、家人、朋友，包括你们这些学生对我的种种好。每当想起别人对我的好，我都会非常感动和感恩。大家对我的好非常难得，因为我们非亲非故，所以我会好好珍惜。

这个世界上，其实没有什么是难以解决的问题。所有难以解决的问题，根本原因都是“你对他不够好”。我昨天也讲了，所有问题的本质都是关系问题，而关系的本质就是信任，对不对？他会因此得不到爱。所以，如果我们能学会如何真心对别人好，所有问题都会迎刃而解。

我们要学习真心对一个人好，但并不是要对每一个人好。因为每个人的精力、能量和爱都是有限的。关于这部分，我们至少能学到的是**真诚**。因为让我去爱世界上所有人，我目前还做不到。我能把身边的人照顾好，已经非常厉害了。

所以我们至少要做到：待人真诚，不要虚伪。如果不喜欢一个人，就不要假装喜欢他。如果不想对他好，也不要假装对他好。至少我们可以活得坦诚一点，把自己有限的能量和关注力放在自己喜欢、自己想要、愿意对他好的人身上，放在生命中最重要、最关心、最在乎的人身上。如果能做到这一点，我们的关系就已经非常棒了，比如我们的父母、贵人。慢慢地，让你的爱去爱更多的人，这本身就是一个提升的阶段。

我们不可能所有人都做到“你打我的左脸，我把右脸也给你打；你羞辱我，我微笑对你说谢谢”。要达到这种境界非常不容易。但在通往“仁者无敌”的道路上，我们至少可以做到对人真诚，对自己真诚。所以，把自己的真心用在自己在乎和重要的人身上。

我下面一节课会详细讲解如何真心对别人好。这是一个非常重要的思考。真心对别人好就是：**发自内心，不求回報，沒有利益，沒有條件，純粹地想為對方好**。

所以我希望大家身边都能有一个真心对你好的人。如果你想了想，一个都没有，那么你就成为那个对别人好的人。当你真心对别人好，总会有另一个人出现真心对你好。宇宙永远是对等的。

所以，在生命中遇到那些你在乎、感恩、喜欢、最重要的人，就真心去对他好。以上就是我们第二部分，关于接受爱的内容。

## 第三部分：拒绝爱 🚫

上一节我们讨论了如何接受爱，本节中我们来看看同样重要的课题：如何拒绝爱。这部分内容相当于给大家打预防针。

为什么要讲这个？包括我自己在内，因为我经常无条件对身边的人好，所以也会有人很想对我好。但有一点要注意：**不要随便利用他人对你的好**。

你们有没有发现，很多女生因为条件好，很多男人就对她献殷勤，买这买那。甚至有些人，会把别人对他的好当作一种利用工具。但你会发现，这样的人十有八九下场不好。

包括你们之后认真学习，都会成为高情商的男人，个人魅力会越来越强。那时，经常会有很多人对你好，但我们不要随便利用。因为我们要负责任，男人的担当非常重要。**接受爱和拒绝爱，都是对他人的一种尊重**。

喜欢是一种感觉，不喜欢却是一种事实。如果你们遇到一个女生，她很爱你、很喜欢你，然而你却不喜欢她。这时请与她保持一定的距离，不要想着因为别人喜欢你，就去利用她做各种各样的事。当然，我相信你们不会这样。

当我们拒绝别人的爱时，要学会：
*   **感谢对方的欣赏**：尊重对方的感情。
*   **态度明确，表达清晰**：不能模棱两可，要言行一致，切记。
*   **换位思考**：别人发自内心地爱你、喜欢你，也是想找一个更好的、合适的伴侣。如果你模棱两可，把她吊在那里，想想对方的心情会怎样？我也讲过换位思考。你们都追求过女生，喜欢过女生。如果对方把你吊着，你是不是很难受？所以，当发现有人对你表达爱意而你不喜欢时，要明确表达。学会拒绝爱，非常重要。

## 第四部分：守护爱 🛡️

上一节我们学习了如何恰当地拒绝爱，本节中我们来看看如何守护已有的爱。你们会发现，人与人的关系是越亲密，就越容易亲近。当我们和一个人关系遥远时，距离产生美，我们很容易看到对方的优点和三观亮点。

而当我们和一个人距离很近时，就会习惯性地挑毛病，看到他身上有限的、不完美的地方，这就是人性。所以人们常开玩笑说：“老婆都是别人家的好，老公都是别人家的好，孩子也是别人家的好，父母也是别人家的好。”这是我们大脑的机制，对于亲近的人，总是习惯性地找问题，对吧？

即使是世界上最伟大、最成功、最有影响力、最有成就的人，如果他天天跟你在一起，你肯定会发现他有很多缺点和毛病，甚至无法忍受。这就是我们大脑的运作方式，我们总是喜欢看到别人的缺点和不足。

为什么会这样？因为我们的大脑喜欢寻找完美，渴望世界上有完美的存在，尤其是对人。我们总是希望对方是一个完美的人。但宇宙的法则是二元对立的，一个人有多少优点，就会有多少缺点；一个人有多少神性的光彩，就会有多少人性的阴暗。

所以很多人会进入一种执着和执念，总想把自己身上所有不好都消灭掉，只留下好的存在。但这样做很容易陷入无限循环的痛苦中。所以还是回归到之前的课程内容：在面对与自己很亲密的关系时（家人、另一半、很铁的朋友），我们都要学会抽离自己，跳出来，告诉自己一句话：**世界上沒有完美的人**。

我们就是在关系交往的过程中学习爱。而学习爱就是学习**接纳和允许**。有时我们会看不起身边最亲近的人，觉得他没有那么伟大，甚至会因爱生恨，对他比其他人有更多要求，更加挑剔和苛刻。这都是陷入了大脑的误区。

所以我在上一节课讲到，我们要学会放大别人的闪光点，享受他的优点，利用他的长处，发现他身上好的一面，然后把他不完美的一面放下、看轻、看淡。不要关系越亲近就越挑剔，因为你越亲近就会越亲密。很多亲近的关系经常彼此伤害和折磨。在亲密关系中就会出现这种事。所以你看到有些人相爱相杀，经常吵吵闹闹。有些人在吵的过程中感情越来越好，但有些人吵着吵着就没了。

所以我们在亲密关系、在跟人的关系中，是给予能量去滋养对方，让对方变得更好，而不是去折磨对方。切记！有同学听到这里会说：“我也不想折磨他，是他在折磨我。”如果你有这个反应，就要回去看原来的课程内容了。你总是在把责任推卸给他人。这时你要回到爱自己的状态。如果对方不能滋养你，那么请你自己滋养自己。如果对方不能提升你的能量，那么我们首先要做到自己提升自己的能量。

我在很久之前的课程讲过不可控因素与可控因素，对吧？有很多不可控的因素，我们不要盯着它，只需要盯着自己那些可控的因素。所以无论你们以后娶谁，无论跟谁在一起生活，首先你永远要记得：**你必須得跟自己相處得很好**。你必须要跟自己结婚，必须要能够自己爱自己、照顾好自己。

所以当你能活出这种状态、能够自爱的时候，无论你娶谁，无论你跟谁有亲密关系，你都可以滋养到这些人。而且当你到达这个程度时，你会发现你吸引来的另一半、女生，也会是那种自爱的人。真的，我发现我现在接触到的那些女生，条件好，还特别拼命，把自己照顾得非常好，我真的很欣赏。

所以当你真的能把爱给别人时，你会发现身边的人都特别喜欢你，特别想亲近你，特别需要你。他们跟你在一起时就会被滋养，会觉得开心和幸福。所以你们那天看到我那个长得很像Indra Baby的好朋友，他对身边的人都特别好，是无条件地给予。所以他特别吸引人，很多人都喜欢他，成为他的贵人。这是真实案例。

所以我们在面对亲密关系时，如果你还在不断挑剔、指责、找毛病，想让对方变得完美，千万不要这样。你要把注意力放在自己身上，把自己的能量调高，然后去跟对方共振，让对方也可以提升。千万记住，跟女生在一起不要讲道理，也不要想去改造对方，让对方变得更完美。

其实亲密关系就是不断给对方爱，不断让对方变得更好，然后大家一起制造更多的欢乐和开心。珍惜每一个对你好的人，我们要学会守护，因为关系越好的人就越容易挑剔，大家要记得这一点，特别是对家里人。我不知道你们对爸妈会不会特别挑剔。

## 第五部分：理解——关系的终极钥匙 🗝️

上一节我们探讨了守护爱的方法，本节中我们来看看关系中最稀缺的元素：理解。很多人认为理解就是善解人意，能够站在对方角度、设身处地为他人着想，但这还不是真正的理解。

那么，到底什么才是真正的理解？比如前段时间，我和两个朋友在一个高级餐厅吃饭。晚餐即将结束时，隔壁桌一位男士用非常粗鲁的语言和方式对待服务员。粗鲁到无法用语言形容。当时餐厅只剩我们两桌，非常安静，显得他的辱骂声很刺耳。他说的话非常肮脏，有羞辱性，咒骂服务员，说了很多难听的话，声音很大。一个四五十岁的男人，用如此下流的语言骂一个20出头的女服务员，原因只是服务员没有及时给他加热水，这让人很不解。

整个环境充满负能量，服务员低着头不敢回应，因为高级餐厅有标准，不能顶撞客人。我朋友感慨，没想到优雅环境会有这么没素质的客人，我们就匆忙离开了。

我就在思考这件事。我告诉过你们，凡事有果必有因。生命中没有什么无缘无故的事，背后一定有东西让你去学习和成长。我思考如何看待这样的事，它对我们有什么意义。我反思自己过往经历中有没有骂过别人或说过难听的话。想了想，好像有过，以前愤怒上头时会骂人或诅咒对方，责怪别人。

我们之前也讲过，每个人（无论认识与否）都是一面镜子。他发生的场景，就是给你提示，让你看看自己有没有犯过类似问题。如果有，请注意，优化自己。当你不断照镜子，把每个人都当成一面镜子，你就能越来越看透表面，发现问题本质。

比如骂服务员的男士，他不会无缘无故发火。服务员倒水慢只是导火索。真正让他发火的原因可能很多：可能刚投资失败，或被女人劈腿，或被亲近的人背叛，或长时间积压了很多压力痛苦，或行为背后有很多负面信念，导致他在那个当下进入疯狂的程序中。所以如果你有同样经历，你也会发狂。当你能看到这一点，你就能真正理解。

然后我突然想到，之前有段时间一个朋友发微信希望我帮他解决问题。有时我信息多，看了点开准备回，突然有事就把屏幕关掉了，所以这条信息变成了已读，我也忘了。后来那个人来找我，非常有情绪，说对我非常失望，根本不关心他，不在乎他，没有及时回答问题。我就说不好意思，昨天突然有事忘了。这个人就更生气，挂了电话。但他不知道我每天有很多事要做，很忙碌、很累、很疲倦。

所以有时我们会“上脑”，经常评判一个人，说这个人自私、强势、傲慢。但我们在批判别人时，永远无法理解他为什么会成为这样一个人。如果你看到这个人所表现出来的特质，是因为他经历过某些事情（包括原生家庭等），才导致他在你面前呈现这样的状况。所以我们要学会看每件事背后的原因。

所以说，世界上并不缺乏爱和仁慈，我们可以去爱某个人或对某人表现仁慈，但真正要做到理解，真的难上加难。所以你们很多人有时给我发信息，或不敢给我发信息，其实都是在理解我，觉得老吴平时比较忙，没什么重要事就不来问我。这就是一种理解。

我也可以理解你们。比如我们叫你们写作业，为什么要写？写作业是内化和输出，检验你内化和输出的成果。作业不是写给我看的，是写给你自己看的。所以我也能理解你们有时要陪另一半，有时要加班。我不会骂你们不写作业、不认真学习，因为我能理解。

那么，理解还有一个终极解释，大家注意听：**我能夠理解你對我的不理解**。因为我们无法要求别人都理解自己，但我们可以尝试学习去理解别人。理解到：任何事情，只要存在就有原因；只要存在，既是合理的。这也是在加强我们允许别人跟我们不一样，允许别人做他自己，按照他的生活方式生活。因为我们常犯的一个错误就是喜欢对别人评头论足、指指点点，这个后面我也会讲。
