## 第三部分：多功能修图软件详解

单一的滤镜可能无法满足所有需求，因此我们需要一款功能更全面的工具。现在，我将详细介绍我最常用的一款多功能修图软件。

这款软件不仅提供滤镜，还包含精细的图片调整工具。它是我处理风景或氛围图片的主要工具。

**核心功能**包括调整图片的`光线`、`亮度`、`对比度`、`高光`和`氛围`。

以下是使用该软件进行调色的常用方案：
*   **方案一：戏剧效果**。在调整基础光线后，加入“戏剧效果”，能让图片产生强烈的视觉冲击力和油画质感。
*   **方案二：HDR景观 + 戏剧效果**。先应用“HDR景观”功能增加颗粒感和素描效果，再叠加“戏剧效果”，能创造出富有艺术感的画面。
*   **方案三：复古色调**。在光线调整选项中，开启“复古”功能，可以营造出中间亮、边缘暗的晕影效果，带来一种怀旧、略带忧伤的氛围。

每种方案的效果强度都可以通过滑动条进行自定义调节，例如`饱和度`、`样式强度`和`阴影强度`，这取决于个人的审美偏好。
