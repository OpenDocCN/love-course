# 浪迹大尧《简爱》课程_序幕 👋

在本节课中，我们将要学习“社交直觉”这一核心概念，并了解它在恋爱关系中的重要性。我们将探讨为何有些人更擅长与异性相处，以及如何通过理解关键的时间节点来提升自己的恋爱能力。

很多单身男性在恋爱中感到不顺心，通常会将原因归结为两个问题。第一是认为自己不够帅，颜值不高，觉得女性都喜欢韩国欧巴那种类型。第二是认为自己没有钱，认为经济条件是决定因素。

同时，很多兄弟会注意到，恋爱学的导师通常外形不错。但相比之下，模特圈或演艺圈的从业人员普遍颜值更高，也更懂时尚。另一方面，很多兄弟也觉得约会学的导师经济收入似乎也不错。但相比之下，企业家协会这类群体的收入显然更高。

那么，约会学导师为何比其他群体更懂女性呢？我们认为，最本质的区别在于“社交直觉”。

为了说明这个概念，这里举一个常见的例子。我们和姑娘在一起时，经常会说“你猜”。听到这个词，有的人会笑出来，会联想到具体的场景。姑娘的反应各不相同：有的会逗趣地说“你猜我猜不猜”；有的会真的去猜测一个答案；也有的可能感到无语，回复一个“额”。

她们为何有不同的反应？很多时候，在恋爱或交往过程中，问题并非你做错了某件事，而是你把这件事做错了时间点上。究竟什么时间点该做什么事情？这便引申出我们对于“社交直觉”的定义。

那么，如何最系统地定义“社交直觉”呢？我认为有两个方面。

第一，叫做**你了解女人言行背后的意思**。她的每一句话、每一个表达，背后究竟在透露什么讯息？她的一个小行为、一个小动作，都在暗示她处于什么样的心境。

第二点，叫做**你对双方关系当前亲密程度的判断**。你们现在的关系进行到了哪一步？你们的话题能深入到什么程度、多么私密？你们的肢体接触可以多么亲密？你是否可以拉她的手、拥抱她，甚至亲吻她？

总体而言，言而简之，**在对的时间做对的事情，就叫做社交直觉**。

上一节我们介绍了社交直觉的核心定义，本节中我们来看看它在恋爱过程中的具体应用阶段。在恋爱学中，究竟有哪些非常关键的时间阶段呢？

在《简爱》课程中，我们为大家罗列了四个关键点。

以下是四个关键的恋爱阶段：

1.  **相见**：也就是传统中常说的资源收集。你如何认识更多女性，尤其是自己心仪的女性。
2.  **相知**：也就是所谓的建立吸引力。你如何构建一个优秀的自己，哪些特质会形成吸引女性的魅力，并在此基础上将其展现出来，从而让女性喜欢上你。
3.  **相恋**：当你们两人互有好感时，如何让感情更深更浓，从而确定恋爱关系。
4.  **相许**：当你们两人确定关系、并有意向长期发展后，该如何去经营好这段感情。

兄弟们，如果你对某一个环节有特别关注的点，可以直接跳转到相应阶段的课程进行学习。但我个人建议，重新梳理自己整个的恋爱思路，从头学起会更为系统。

本节课中，我们一起学习了“社交直觉”的概念及其两个核心方面：理解女性言行背后的含义，以及准确判断双方关系的亲密程度。我们还概述了恋爱过程中的四个关键阶段：相见、相知、相恋、相许。理解这些阶段并培养社交直觉，是提升恋爱能力的基础。
