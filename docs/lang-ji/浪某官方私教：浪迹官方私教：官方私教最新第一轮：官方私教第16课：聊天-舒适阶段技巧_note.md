## 课程概述

聊天可以分为四个阶段：破冰、了解、熟悉和舒适。破冰是两人刚认识。了解阶段是彼此分享基本信息。熟悉阶段是利用基本信息聊到更个人化或走心的话题，使关系从点头之交变为可以分享兴趣爱好的朋友。舒适阶段是两人在见面前通过聊天能达到的最后阶段，其特点是彼此非常了解，可以像好朋友一样舒适地相处。

上一节我们介绍了熟悉阶段，本节中我们来看看如何进入并经营舒适阶段。

## 舒适阶段的定义与目标

舒适阶段的定义是两个人已经彼此非常了解。在这种了解下，两人可以当做好朋友来相处。好朋友相处的特点在于想说话就说话，想聊天就聊天，想开玩笑就开玩笑，想聊什么就聊什么。在这种状态下，聊天必然是舒适的。

在舒适阶段，对方会有一些心灵的流露，会有一些走心的、敞开心扉的流露。我们抓住那些敞开心扉的机会，去读懂女生并分享自己的想法，两个人的心就可以慢慢彼此走近。这就是我们在舒适阶段要达到的目的。

## 舒适阶段标准流程

1.  **开场**（可跳过）
2.  **充分互动**
3.  **理性交流**（熟悉阶段的事）
4.  **感性交流**
5.  **精准邀约**
6.  **见面前的聊天**

我们一个个来说。

**首先，开场。** 在舒适阶段，当两人已经进入可以随便聊、随便开玩笑的阶段时，开场是可以跳过的，可以直接进入第二个部分：聊一个话题。

**其次，充分互动。** 例如，聊最近的话题（如某个明星事件），可以发一个截屏过去。充分互动后，两人可以从这个事件讨论到感情。

1.  如何“陪聊”与倾听。
2.  如何表达走心（上节课讲了如何表达观点，走心该如何表达）。
3.  如何使用言语升级。

感性交流完后，进行邀约，然后是见面前的聊天。
