# 1、老吴：老吴《高级情商私教课》：21.三、对外关系——角色扮演，信任_note

在本节课中，我们将要学习如何通过理解“角色扮演”和建立“信任”来优化对外关系。课程分为三个核心部分：首先，认识并扮演好生活中的多重角色；其次，理解信任是关系的基石；最后，学会发现他人的特点与优点。掌握这些内容，将帮助你更顺畅地处理各种人际关系。

## 第一部分：角色决定立场

上一节我们探讨了关系的重要性，本节中我们来看看如何通过“角色扮演”来管理关系。我们每个人都像演员一样，在生活中扮演着超过十个不同的角色。关系的本质分为两种：**亲密关系**和**交换关系**。许多关系问题的根源在于“角色错位”，即没有在正确的关系中扮演正确的角色。

为了理解不同角色的运作方式，以下以我本人为例，列出我所扮演的十个角色：

*   儿子
*   朋友
*   未来的爱人
*   父亲
*   老师
*   老板
*   学生/徒弟
*   慈善家
*   亲戚
*   体验者

当我们用“上帝视角”观察自己，会发现自己在不同角色中是天差地别的。处理好所有关系的关键，在于清晰意识到自己在每一段关系中所要扮演的角色。关系顺畅，通常意味着角色扮演正确；关系出问题，往往源于角色错位。

以下是角色错位的常见例子：

*   **老板角色带入家庭**：一位成功的老板将管理公司的强势作风带回家中，对家人也发号施令，导致家庭关系紧张。
*   **社会角色与亲情混淆**：一位医生被亲戚要求利用职务之便“开后门”，这混淆了“医生”的职业角色和“亲戚”的亲情角色，给双方带来困扰。
*   **朋友关系与商业合作混淆**：好朋友一起做生意，却因未能区分“朋友”和“商业伙伴”的角色，最终可能伤害感情。

因此，我们需要时常跳出来审视自己：你在面对不同的人时，扮演的角色是否正确？能否在不同角色间自由、清晰地切换？**角色决定了立场**。同一件事，不同角色的人会持有完全不同的看法。例如，员工犯错，老板会批评，母亲会宽容，朋友则会支持。没有绝对的对错，立场取决于你所处的角色。

在生活中，人们还容易陷入三种不健康的角色模式：

1.  **迫害者**：攻击、伤害他人，试图通过打压对方获胜。
2.  **受害者**：遇事推卸责任，认为都是别人的错，自己很无辜。
3.  **拯救者**：自以为是，总想改变别人，而非在对方求助时提供帮助。

然而，我们最根本、唯一的真实角色是 **创造者** 。你人生中发生的一切，都是你创造出来的。意识到自己是创造者，意味着你要为人生负起100%的责任，这是非常强大的觉醒。

**总结**：想要成为高情商的人，就要有觉知地扮演好每一个角色，享受其中，但不要“入戏太深”。要明白，你既是演员，也是自己人生游戏的**规则制定者**。清晰的角色定位，是理顺一切关系的基础。

## 第二部分：关系的核心是彼此信任

上一节我们介绍了如何通过角色扮演理顺关系，本节中我们来看看维系关系的核心——信任。我们的一生离不开各种关系，而所有关系问题的背后，核心都是关系本身的质量。关系的真正本质是一种**彼此之间的信任**。

建立真正的信任并不容易，需要时间和共同经历。然而，信任一旦被破坏，就像摔碎的盘子，即使粘合，裂痕也始终存在。因此，世界上最有影响力和成就的人，往往是值得高度信任的人。成为一个让他人信任的人，是极高的生命品质。

要想获得信任，最核心的奥秘在于 **信守承诺** 。生活中，我们常常轻易许下承诺却难以兑现，这会严重损害信任。无论是亲密关系还是商业合作，空头承诺都会让关系名存实亡。

生命中最重要的关系，是 **与自己的关系** 。你必须先学会信任自己。你的内在信念系统决定了你信任自己是什么样的人。如果总觉得自己不够好、不值得，你就很难与自己相处融洽，进而会陷入焦虑和恐惧。当你真正信任自己，相信“凡事必有美意”，你就会创造出更多积极的事物。

从商业角度看，所有销售的本质也是销售 **信任** 。客户购买产品，首先是对你个人或品牌的信任。转化率高的核心就是信任度高。许多品牌投入巨资做广告，目的就是为了建立大众信任。

所以，不要再简单地说“我认识谁”。你需要问自己：这世上有多少人完全信任你？你又能完全信任多少人？请珍惜并守护好与他人的每一份信任，努力成为一个值得信任、也愿意信任他人的人。当你的信任口碑越来越好，你自然会吸引更优秀的人进入你的生命。

## 第三部分：发现他人的优点与特点

在学习了角色扮演和信任之后，本节我们将探讨如何以更高的视角看待他人。世界上没有人是完美的，每个人都有自己的长处和短处。提升情商的一个重要方面，就是学会**发现他人的优点和特点**。

首先，需要**提升自己的能量和意识层面**。当你的能量较低时，容易看人不顺眼，想去批评、改造对方（即陷入“拯救者”角色）。当你的能量提高后，你会变得更包容，更能欣赏他人，从而吸引更高能量、更优秀的人来到身边。

其次，要明白，人**没有绝对的优缺点，只有特点**。一个特点在不同情境或不同人看来，性质完全不同。例如，一个人的“强势”和“自我”，在需要他决断的领导岗位上可能是“有魄力”的优点；在家庭生活中，可能就成为让家人难受的缺点。这本质上只是他的一个特点。

中国有句俗话：“人抬人上等人，人比人中等人，人踩人下等人。”能量高的人，善于发现并运用他人的特点，认为每个出现的人都能给自己带来贡献和启发。能量低的人，则倾向于对比、批判和伤害。

因此，请停止给人贴“缺点”的标签。你所认为的缺点，在别人或另一种情境下，可能就是优点。例如，“宅”对于喜欢探险的伴侣可能是缺点，但对于同样享受居家生活的伴侣则是优点。学会用“特点”的视角去看待人，你的世界会开阔很多。

**本节课总结**：我们一起学习了优化对外关系的三个关键。第一，要像演员一样有觉知地扮演好每个社会角色，避免角色错位，明白**角色决定立场**。第二，要认识到**信任是关系的基石**，并从信守承诺开始，成为一个值得信任的人。第三，要**提升自身能量**，学会用“特点”而非“优缺点”的视角去欣赏他人。掌握这些，你将能更从容、智慧地处理生活中的各种关系。
