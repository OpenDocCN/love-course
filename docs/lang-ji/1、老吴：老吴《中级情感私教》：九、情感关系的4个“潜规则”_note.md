# 1、老吴：老吴《中级情感私教》：九、情感关系的4个“潜规则”_note

在本节课中，我们将探讨影响情感关系的四个核心心理机制。这些机制常常在潜意识层面运作，理解它们有助于我们更好地处理与伴侣、家人乃至自己的关系。

## 第一部分：与母亲的关系——所有人际关系的基石

上一节我们探讨了情感关系的基础，本节中我们来看看原生家庭，特别是与母亲的关系，如何深远地塑造了我们所有的关系模式。

我们每个人与母亲的关系，决定了自身所有的人际关系，尤其是亲密关系。无论我们是否承认，生命中最亲密的初始关系就是与母亲的关系。在我们诞生之前，即尚在母亲子宫中时，每个人都通过脐带与母亲相连。生命最初的状态，让我们在内心深处意识到自己与母亲是合二为一的完整一体。这正是母亲对我们每个人影响如此深远的原因。

这也解释了为何现代孕妇非常重视胎教。当孩子在母亲子宫中时，母亲的思想、情绪及能量状态，会被孩子完全感知和接收。每个孩子都能完全体会母亲内在的真实状态，甚至将这些状态内化为自己长大后的一些核心信念或“种子”。

如果一个孩子生来就非常恐惧，这是因为他在母亲子宫中时，感受到了许多恐惧、压力与负面情绪，导致他与生俱来就带着恐惧胆小的情绪。有的孩子生来就暴躁、情绪化、不安分，同样是因为在母亲子宫中感受到了各种强烈的负面情绪。

母亲对孩子的影响不仅限于子宫时期，更在于同一家庭环境下抚养孩子的成长过程，即我们所说的“原生家庭”。在每个孩子内心深处，父母最初都是完美且如神明般强大的。他们不应有任何差错。然而，每个孩子在成长过程中都会对父母感到无比失望，发现父母也会犯很多错，拥有许多不堪的负面情绪、念头，甚至智慧不足、人品不够好或能力有限，无法创造更好的物质生活等。

无论喜欢与否，认可与否，孩子几乎都在百分百地模仿父母。如果我们内心深处认可和喜欢父母，我们就会有意识地去模仿他们。如果我们内心深处抵触、不认可、不接受父母，我们仍会在潜移默化中模仿他们。很多人长大后，发现自己成了曾经最讨厌的人。这在《易经》中称为“DNA的遗传”，在生物学中称为“基因的遗传”。无论如何，父母对我们的影响是终生的，它决定了我们的婚姻、财运、家庭、事业等，以及各种人际和亲密关系的相处模式。

父母的影响中，最深层次的是母亲的影响。如果在互动中，母亲让孩子感觉自己没有得到足够的爱、认可和安全感，孩子内心深处就会永远渴求爱、抓取爱，因为他缺乏安全感，不信任、不相信爱情与婚姻。仔细观察单亲家庭的孩子，某方面会特别明显。一个孩子要健康成长，绝对离不开父爱与母爱。

许多人在内心深处常会想：“如果我自己的妈妈都不爱我，这个世界上还会有谁爱我？”一个充分得不到母爱的人，内心深处会有强烈的不值得被爱的感觉，他会觉得自己不够好。因此，在未来现实生活中，无论去到哪里，发生什么事，他总会觉得自己不够被爱，不值得拥有更好的人生，不值得有人来爱自己。

当你拥有这样深层次的“种子”和信念时，在余生中就会创造许多让自己更加确认和感受到“不值得”的情景发生。例如，我们会和完全不爱你的人结婚，和错误的人一起工作、生活，创造各种各样的事情来被对方伤害，从而更加确认内心的种子信念：“我不够好，我不值得被爱，没有人会爱我。”

以上所述并非特例，几乎是普遍现象，是每个人内心最深处的现实。只不过有的人能够觉察到，有的人头脑防御强大，用厚厚的盔甲把自己脆弱的灵魂包裹在深处，不愿承认罢了。

如果父母是非常自私自利、自我中心的人，就会让孩子也感受到“只有这样才能保护自己”。孩子可能非常厌恶自己的母亲，但长大后却被评价为同样自私自利、自我中心的人。因为孩子从小没有得到足够的爱，我们又怎能要求他给出足够的爱？就像一个空杯子，没有水，它怎么倒水给别人？因此，真正的慈悲不是简单的理解，而是能够真正看到每一个受伤生命内心深处的苦。

当我后来不断提升自己的智慧和思想维度后，我看到很多人内心其实非常痛苦。这也是我讲述这些内容的原因，我希望通过这些内容、我的能量和话语，给大家带来疗愈。你们来听我的课，包括以后听其他老师的课，会发现同样的内容由不同的人来讲，差距非常大。因为老师本身就是一个输送能量的过程。

为什么我们最终都会成为自己最讨厌或想要抗拒和远离的人？因为几乎每位父母都会给孩子带来不同程度的伤害，这是无法解决的现实问题。大多数父母在做父母前根本没有受过任何教育，没有学习过如何成为合格的父母。父母尽了最大努力，以自己认为最好的方式爱孩子，却可能给了孩子各种各样的伤害。

我们在内心深处不认可自己的孩子，希望孩子完美，希望孩子出人头地让自己更有面子。当你有这样的信念时，就会对孩子施加许多控制、掌控和要求，却打着“爱”的名义认为这是为他好。我相信你们听到父母对你们控制、掌控、要求时，会非常不爽。而父母会觉得：“我是在爱你，为你好。”但很多所谓的“为你好”，只是站在他的角度而已。

父母对孩子的爱，某种程度上也是自私自利的爱。我们希望孩子成功，是因为这样自己能够更成功、更被认可。所以我们无法接纳孩子的不完美，给孩子带来很多冲突。就好像很多父母自己并不成功，却希望孩子无比优秀。例如，有的父母婚姻不好，却希望孩子拥有美满的婚姻；有的父母赚不到钱，却希望孩子将来事业成功、出人头地；有的父母有很多陋习、恶习，德不配位，却希望孩子当个好人。你们觉得这可能吗？

因此，很多人说最好的教育是“言传身教”，这四个字非常重要。当我后来懂得并深刻理解这些之后，我知道作为一名老师，我必须“讲我所做，做我所讲”。如果我表面讲一套，内里做另一套，肯定不行，你们会感受到我表里不一。

如果你与父母的关系有问题，那么你的情感关系中一定会创造出各种各样的问题。为什么？因为我们从父母那里获得的“种子”，会成为自己内心深处或DNA中最具威力的种子。我们会在后天生活中，不断创造重复的情景来证明自己的信念是对的。

那么，我们应该怎么办？许多人说“我理解父母，原谅父母”、“我已经能和父母和平相处甚至相处得很好”，但这些往往是头脑层面、表面的内容。我们要从内心深处真正清理掉这些负面种子，就需要在情绪管理层面做疗愈。不是去原谅父母，而是去疗愈你自己内心深处的伤痛。一个内心没有伤痛的人，又怎么会去作恶呢？当我们理解这一点，就能真正体会到所谓的“慈悲”。

我们看到那些说你坏话、搬弄是非、总是伤害你或不懂感恩的人，其实他们很可怜。如果能把怨恨转化为同情，我们就迈出了慈悲的第一步。任何人若想真正“断恶修善”，一定要懂得“离苦得乐”。所谓离苦得乐，就是管理好你的情绪，能够从痛苦的情绪中抽离自己。关于父母这一块，我们不是要去原谅父母，而是要自己去疗愈。

那些深藏在骨髓里很久、甚至你意识不到的痛苦，如果没有清除掉，你仍然会创造出许多你不想要的生命情境。因为人就像一个投影仪。我们的大脑像投影仪，你的“CPU”（核心处理器）想着什么，它通过眼睛投射出去的世界，就是你想的样子。你觉得这个世界的人都是坏人，你接触的就都是坏人。你觉得都是好人，你就会接触到很多好人。

**总结**：好好地去审视与父母关系中的问题，并解决它。这是我们建立健康人际关系的基础。

## 第二部分：依恋与伤害——为何离不开伤害你的人

理解了原生家庭的影响后，我们来看看它在亲密关系中的一种具体表现：为何有些人离不开伤害自己的伴侣。

这部分内容关于男女相处的一个难题：有些人似乎离不开伤害自己的人。但这并非因为他们喜欢受虐。因为人际关系是复杂的，爱情作为一种人际关系，同样存在复杂性。很多人在爱情中感到受伤，却奇怪地离不开伤害自己的爱人。为什么别人伤害了自己，却还不愿离开？很多人用“受虐狂”三个字来总结这种态度，但实际上，没有人喜欢莫名其妙地受虐。本节课我们就来研究，为何在爱情中受到伤害，却往往无法离开伤害我们的爱人。

首先，要搞清楚爱情对人类来说到底是什么。人类学家费舍尔和神经学家布朗曾共同进行一项实验。他们找了37位处于爱情不同阶段的测试者，其中有的人正在热恋，有的人刚刚失恋。两位学者记录了这些测试者的大脑活动，并与对事物上瘾者（如对药物、酒精或游戏上瘾）的大脑活动进行对比。研究发现，恋爱中的人的大脑活动状态与上瘾者非常相似。这表明爱情的确具备让人上瘾的特质。

有些不吸毒的人，看到吸毒者将针扎入血管，会觉得“好痛”。但对吸毒者来说，这点痛完全比不上毒品带来的快乐，这就是上瘾现象。当你感觉到的痛苦完全比不上接收到的快乐时，快乐就会盖过痛苦，你就不会在意痛苦。因此，在爱情中受到伤害的人，相比于爱情带来的快乐，那些伤害可能算不了什么。

以下是理解这种状态的几个角度：

**第一个角度**：不要称自己是受虐狂，因为你接受的是对方的爱，而不是对方的“虐”。就像你养宠物，爱的是它的陪伴，只不过它同时会拉屎拉尿需要你处理。有不少女生说，她们喜欢的就是霸道总裁、男子汉、大男人。这些形容词恰好说明，女生喜欢的男生可能具备这些特质：他们很自我中心，很有主见。这些特质也许让他们在事业上更容易成功，但也可能在爱情中给另一半带来伤害。这样的男生给予伴侣爱的方式，可能就是很霸道、很自我中心的。这就是我们所说的爱情中的“杂质”，好比黄金和泥沙混在一起，且黄金比例很低，泥沙比例非常高。

很多人在爱情里受到许多伤害，才能感觉到一点点爱。就像伴侣如果出轨三次，伤害非同小可。但当你劝受伤害的人离开这样的伴侣时，受伤害的人反而会维护伴侣，说起对方平常对自己的种种好处。如果你也是这样的人，就表示虽然你不是受虐狂，不喜欢受虐，但你在接收到“虐”和“爱”时，有能力把“爱”的部分放大。就像在一卡车的泥沙里，你能看到一小颗一小颗闪闪发光的黄金。虽然旁观者会觉得你的比重放得很奇怪，但你自己心里知道，这一点点的爱，就足以帮助你抵抗那个人带来的各种伤害，这成了你离不开对方的原因。

**第二个角度**：往往伤害我们的人，提供了一些我们很向往却不具备的特质。例如，在一部美剧《马男波杰克》中，主角波杰克身边有很多“花花草草”，他的女友凯洛琳因此受到很多伤害，因为波杰克永远与身边人纠缠不清，也永远不愿结婚。但这个女生仍然非常迷恋波杰克。追根究底，是因为这个女生的人生目标就是“被需要”，她非常喜欢和重视那种被需要的感觉，而整个世界上最强烈表达需要她的人，就是这个“渣男”波杰克。从旁观者角度看，你会觉得这个男生对你不忠又如此依赖你，你为什么还会喜欢他？但偏偏“不忠”是你受不了的事，而“依赖你”是你最喜欢的事。他越是依赖你，你就越是离不开他。这虽然称不上值得向往的特质，但偏偏证明了你在世界上独一无二的价值和最重要的存在感。

另外有一些女生，觉得对方能提供类似父亲的感情，给自己很有智慧的人生指引。这也是同样道理，在对方身上感受到一种非常向往的特质，而无法离开。

我们提供这些角度，是希望你不要认为对方给你的全部叫做“爱”。这份爱里同时包含了许多杂质，不要把爱的杂质与爱画上等号。你也可以想一想，你所接收到的爱里，是否有其他杂质？这些杂质有严重到伤害你吗？如果会，这些杂质构成的伤害，有没有严重到超过了你所收到的爱？如果你感受到爱的成分超过了杂质成分，那就构不成威胁，你仍可以享受这份爱情的美好。就像生活中的任何事情，都是优点和缺点并存的，你无法只要优点不要缺点，这就是宇宙的“二元对立”。

如果你觉得已经受够了，真心想要离开这个伤害你的伴侣，那么我提供一个心态：**主动离开**。很多人一听到“主动离开”，就理解为“主动求去”，其实不是这个意思。“主动求去”是指你已受到无数伤害，再也承担不了更多，只好离去。这其实是在绝望中被动的离去，不是主动的离开。就是我们常说的那句话：“当你的失望攒够了，你就会离开。”但这种离开是被动的，即使如此，也会在你心里留下巨大伤痕。

我们所建议的“主动离开”，是指在两性关系中，你觉察到自己最需要对方提供什么特质，然后主动在自己身上建立起这个特质，你就不再需要对方来供给。例如，你虽然辛苦工作，却从来不懂宠爱自己，喜欢的东西都不舍得买，却把钱花给伴侣。所以，这个伤害你的伴侣如果有一次买了你喜欢的东西给你，你就会非常感动，感觉到被宠爱。如果你想离开这样一个会宠爱你但也会不断伤害你的伴侣，你要做的就是：**一定要宠爱你自己**。一旦你宠爱自己，就不再需要从对方身上得到这份宠爱了。这也是我们之前说过的“自爱”。

还有一部美剧《了不起的麦瑟尔夫人》，故事中的女生原本很崇拜丈夫，帮助丈夫完成脱口秀梦想，直到丈夫出轨，她才意识到自己缺乏自我。于是她开始追求自己的梦想，磨练表演能力，自己站到了脱口秀舞台上，成为出色的脱口秀演员，从此不再需要服务她的丈夫。这就是我们所说的“主动离开”：在自己身上建立向往的能力，把自己的缺憾补足。

在爱情中，与在任何人际关系中一样，你不能指望改变对方，你只能改变自己。

**总结**：审视关系中你真正需要的是什么，并尝试通过自我成长来满足它，而非依赖一个可能伤害你的伴侣。

## 第三部分：“作”的行为——权力关系与安全感的索取

在探讨了依赖的复杂性后，我们进入第三个主题：亲密关系中常见的“作”的行为，其本质往往是权力博弈与安全感索取。

只要我们一不小心，每一种人际关系都可能变成一种权力关系，包括爱情。例如，亲子关系常常变成权力关系，爸爸妈妈会认为“孩子是我生的，我可以控制他的人生”。在爱情关系中，也常出现权力现象，比如谈了女朋友就觉得“她是我的，我必须控制她”。光是想一下常用形容词，我们会说“这个人臣服在对方脚下”、“拜倒在她的石榴裙下”。这些“臣服”或“拜倒”，都是权力关系中的用语，表示对方具有权威，而你必须要服从。

但当爱情关系一直保持这种权力关系状态，会是好事吗？这部分内容要研究，为什么谈恋爱常常会造成“公主病”，让一些本来心理平衡的女生突然发作。

权力是非常微妙的东西，只要你不使用它，它就仿佛不存在。“公主”这样的身份也一样，对方如果把你当成公主，你就有了公主的身份和权利；但如果对方根本没把你放在眼里，不管你是不是真的公主，你都发挥不了公主的权利。讲到这里你们应该明白了，有时候说女生“作”，并不是女生想作，而是你给了她“作”的权利。你会发现，很多问题的本质根本都出在自己身上。

如果你看过经典电影《罗马假日》，就会看到一个欧洲小国的公主，当她以普通女孩身份活动在罗马街头时，不认识她的人都把她当成一般女孩看待。只有回到皇宫，进入要求外交礼仪的场合，公主的身份才会显现，公主的权利才能够发挥。

首先，“公主”是一个非常没有道理的身份。虽然很多童话故事以公主为主角，但实际上公主除了身在皇家之外，不需要任何认证。她不需要证明自己，不像英雄需要经过一番打斗和作为才能得到大众认可。所以说穿了，童话里的白雪公主、豌豆公主，对于社会没有任何贡献，她只不过身在皇家，有了一个头衔罢了。如果她温柔可爱还行，如果她任性、喜欢虐待别人，那这个公主肯定一无是处。

从这个角度理解“公主”身份，我们就知道，任何向往变成公主的人，其实是在向往一个不劳而获的身份。因为她只想要身份的尊贵、锦衣玉食，而不想为争取这个身份做任何努力，也不想要随之而来的义务与责任。而本来活得好好的“公主”，突然“公主病”发作，最理想的场合就是在谈恋爱时。因此，直到谈恋爱的此刻，才终于有人会拜倒在自己跟前，说出“就算你要天上的星星，我也会爬到天上去把星星摘给你”。想想看，在一夕之间，你突然头戴皇冠，被对方当成公主或女王。如果不趁机使用一下女王和公主的权利，那这个权利还有什么意思？这个身份又有什么意义？这就说明了为什么一谈恋爱，很容易有些人就会“公主病”发作。

恋爱中的女生，常常想用艰难的任务考验另一半。例如，自己过生日时，想要另一半送一个需要排24小时队才能得到的礼物，女生会觉得特别有成就感。但如果她的另一半刚好有特殊关系可以轻松拿到，女生就会觉得这礼物一点也不值钱。因为女生要的不是这个东西，而是你为得到它所付出的心力。

当然，在爱情里不只是女生会折磨对方。有的男生看到女生为了跟自己交往而与家人相处不好时，就会对女方说：“你必须在你家人和我之间做出选择”，而不会帮女生想办法解决家庭问题。甚至很多人在热恋期会提出一些连自己都无法理解的荒谬要求。例如，“如果我传短信给你，你要在10分钟内回我，不然就表示你不重视我。”有的人如果10分钟内没收到回信，接下来就可能打100个电话“夺命连环Call”。

现在，我们在各种平台都能看到生活中的短视频，在这些视频中，交往中的恋人有时互相捉弄一下，嘻嘻哈哈，无伤大雅，我们会觉得他们在“撒狗粮”。但有的人，仿佛要把另一半弄到很羞辱的状态才会觉得过瘾。例如，要求对方穿奇怪的衣服到公众场合大声表示有多爱自己；或者不管对方正在上课或开会，都要第一时间回复、第一时间开视频，才能证明足够爱自己。这就是把得到对方的爱当作一种权利，这种权利短期内能带来过瘾和满足。但每一次过瘾和满足，也意味着在感情中消耗了对方的耐心和信任。这是我们要理解“不要把爱情当成权力关系”的另一个角度。
