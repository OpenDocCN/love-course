# 11、浪迹·研习社：第5课展示面之我看（老吴特别课）_note

在本节课中，我们将要学习如何打造一个真实、自然且具有吸引力的个人展示面。展示面是你在互联网时代，特别是社交圈中的形象名片，它对于建立吸引力和快速推进关系至关重要。本节课将分享我从2014年至今，在展示面打造上的核心心得与总结。

## 概述：展示面的核心价值

在互联网时代，朋友圈等社交展示面变得尤为重要。一个好的展示面能够帮助你快速吸引异性，并推进关系。展示面的核心并非制造虚假形象，而是**真实地记录并呈现你生活中美好、积极的一面**。

## 展示面的核心理念：真实与自然

上一节我们介绍了展示面的重要性，本节中我们来看看打造展示面的核心原则。

展示面不是让你去做虚假的东西。我们应该做一个**真实的男人**。这种真实，不是让你发一些平庸或低价值的内容，而是将日常生活中美好的人、事、物记录下来。

我2018年的朋友圈，内容并不多，主要围绕健身、工作以及旅行见闻。关键在于，展示面应该带有一种**随缘**的性质去拍摄，不要刻意。

例如，我近期去纽约，并没有疯狂拍照。只是在空闲时于街头随意走了走，在下小雪时拍了两张照片。其中一张就是我现在的头像。

## 如何拍出自然的展示面照片？

很多人拍照会显得尴尬和不自然。关键在于捕捉**动感**和**生活感**。

我那张街头照片是找路人帮忙拍摄的。我告诉他，我过马路时请他连续抓拍。这样拍出来的照片是动态的、自然的。我从大量抓拍照片中选出了最好的两张。

**核心技巧**：让拍摄对象（路人）不要在意构图，只管连续拍摄。你在过程中投入真实的情感和状态，最后从大量照片中挑选最自然的一张。

如果路人拍得不够好（例如把我拍矮了），我会使用修图软件进行后期调整。我常用**美图秀秀的拉高功能**来优化比例。

## 展示面的内容与角度：呈现多面性

一个好的展示面应该从多角度呈现自己，而不是只有千篇一律的正面照。

我发现最近我的朋友圈有一个特点：**没有正面的照片**。无论是街头照还是风景照，都是侧面或背影。生活是多面性的，我们应该通过正面、侧面、背影等多个角度来呈现自己，让观众对你有一个立体的认识。

例如，我在纽约世贸大厦拍了一张背影照。这张照片传递出一种从容、展望的感觉。照片**要有内容，有故事**，能让人细细品味，而不是单纯地炫耀。

## 如何培养“发现美的眼睛”？

有人会问，如何让展示面看起来赏心悦目？这需要你培养一双**发现美的眼睛**。

以下是训练方法：

你可以通过以下途径获取灵感：
1.  关注微博、公众号上的时尚博主。
2.  使用Instagram、Nice等国内外图片社交APP，观察热门用户的照片。

学习重点不是模仿他们去不了的地点，而是学习他们的**穿着搭配、修图风格和构图技巧**。你完全可以在自己的城市，用自然的状态和良好的光线，拍出同样感觉的照片。

## 展示面与个人成长：从展示到成为

做展示面更深层的意义在于**自我提升**。

展示面是你个人的直接表现。别人通过它来了解你。你可以通过展示面，有选择地呈现自己的优点，暂时隐藏不足。当你展示的优质内容足够多时，偶尔展示一些无伤大雅的“不足”或幽默自黑，反而会让你显得更真实、与众不同。

更重要的是，长期坚持打造展示面，会促使你不断去**发现生活的美**，并努力让自己成为展示面中那个优秀的人。最终，当你的真实人格与你的优质展示面高度一致时，展示面本身就不再是必需品了。

**核心公式**：`持续打造优质展示面 -> 倒逼自我提升 -> 个人真实价值接近展示面 -> 降低对展示面的依赖`

## 差异化：打造你的个人特质

在众多展示面中脱颖而出，需要**差异化**。

不要盲目模仿主流。你要发掘并发挥自己的**个人特质**。上帝关上一扇门，总会打开一扇窗。你要把自己的特点，甚至是别人眼中的“缺点”，变成独特的记忆点。

例如，我以前很在意自己的声音，但现在我认为这是我的特色。很多人一听声音就知道是老吴。再比如，如果你觉得自己矮，那就去发挥聪明、幽默或其他内在优势。

## 常见问题解答（Q&A）

以下是课程互动环节中，对部分典型问题的解答：

*   **问：大学生该如何做展示面？**
    *   答：呈现你真实的大学生活即可。多参加社团活动，记录校园内外的精彩瞬间。

*   **问：拍照和修图有什么技巧或软件推荐？**
    *   答：核心是**构图**和**光线**。修图软件推荐**美图秀秀、Instagram、Snapseed、VSCO**。关键是思路，而非工具。

*   **问：展示面配文怎么写？**
    *   答：男性配文应简洁，注重**互动性**。可以用幽默的方式化解可能存在的“装逼”感。例如：“不要因为我在认真吃东西，就说我装逼。”

*   **问：不同职业（如魔术师、厨师）如何拍展示面？**
    *   答：可以展示职业相关的内容（如魔术小视频、烹饪过程），但**频率不宜过高**，避免让人觉得你的生活只有工作。核心仍是呈现丰富、有品质的多元生活。

*   **问：如何一个人拍出好的展示面？**
    *   答：**大胆请求路人帮助**。这是最有效的方法。一次拍不好，就换一个人再试。

*   **问：展示面的高效作用是什么？**
    *   答：在社交软件上，优质的展示面能大幅提升你匹配到的**资源数量和质量**。这意味着在相同时间内，你的效率更高，能接触到更多潜在对象，从而节省时间成本。

## 总结

本节课中我们一起学习了打造个人展示面的核心心法。

我们明确了展示面的目标是**真实呈现一个积极、美好的自己**，而非虚构人设。我们探讨了如何通过**随性抓拍、多角度呈现、培养审美**来制作自然的展示面。更重要的是，我们认识到展示面是一个**自我驱动成长**的过程，最终目标是让真实的自己变得和展示面一样有吸引力。最后，**差异化**和**发挥个人特质**是让你从人群中脱颖而出的关键。

记住，听课后一定要去**实践**。从审视和优化你下一张朋友圈照片开始，逐步打造出属于你自己的、具有个人魅力的展示面。
