# 浪迹老佟《短期关系》课程：短期方法_聊天实战案例分享（二）_note

在本节课中，我们将通过分析一段完整的实战聊天记录，学习如何通过微信聊天推进短期关系。我们将重点关注如何接话、铺垫邀约、判断对方情况、应对测试以及控制约会时间等核心技巧。

## 实战聊天记录分析

上一节我们介绍了聊天开场与破冰，本节中我们来看看如何在实际对话中层层推进。

### 开场与接话

加了这个姑娘的微信之后，我有20分钟没有理她。在这20分钟之内，她就主动给我发了消息。她说：“我要把你签名盗走。”

这个开场白与常见的打招呼方式不同。这时你需要接住对方的话。我的回复是：“拿什么来换？”

这句话其实已经埋下了伏笔。无论她回复什么，我都可以顺势提出“见一面”的邀约。

### 初次邀约与时间预期

她的回应是：“明天吧。”随后我们互发了位置，判断彼此距离。

我再次强势地推进：“这么近就赶紧的。”但她仍然坚持明天。这时我明白了：这个姑娘可以约见面，但时间定在明天。

因此，我下面一句话是：“明天要开一天的会。”这句话背后有两个意思：
1.  表明明天可以约。
2.  为明天做时间预期铺垫，暗示只有晚上有空。

她的回应是：“那就明天晚上吧。”在简单闲聊几句后，她说：“那就压马路吧。”我说：“可以啊，就是不知道会不会和今天一样晚。”

我继续在做明天的预期，意思是明天可能会很晚。当她接受了晚上约会而不是白天，我就得寸进尺，试图把时间拖得更晚。越晚对你越有利。她说：“没关系。”这表示她通过了这个测试。

### 判断对方背景与排除障碍

然后，这个姑娘开始判断我的情况。当姑娘想进行一段短期关系时，她会担心你的背景，以获取安全感。

我说：“是呀，都没空出去玩。”紧接着我又说：“看你家住在北京。”

这句话非常关键，用于判断姑娘的几种情况：
1.  她是外地人还是本地人。
2.  从籍贯判断她的住宿情况，例如是否与父母同住，以排除这个最大障碍。

她的回应是：“我不，我是河南人。”由此可知，这个姑娘是从河南来到北京，可能独自居住，也可能与朋友合住。我说：“很近啊，回家很方便。”

当我判断出这个信息后，我立刻把话题扯开了。

### 应对废物测试与话题引导

再往后，这个姑娘问：“你经常这样约女孩子出去玩吗？”这是一个很常见的问题。

这是一个废物测试。我对她的回应是：“不。”我的回应有一点敷衍。

她追着不放，说：“我才是。”这时，我使用了一个常用话术：“我也是有标准的。”她问：“什么标准？”我说：“你这样的。”

通常，“有标准的”后面一句话，姑娘都会问“什么标准”。这时，你只需说她这样的标准即可。

这时姑娘说：“小嘴可真甜啊，看来我是达到标准了。”她一次性回复了两个话题。

在我们的聊天套路中，一直强调永远只回复一个话题、一句话。这两个话题你可以任意挑选一个来回答。我挑了第一个，回复：“还没尝就知道嘛。”

这叫做调情。她说：“真是讨厌没个正形。”她的情绪被挑逗起来了。我说：“不喜欢假正经。”

紧接着，我又做出了当晚的邀约：“哎不然就提前吧。”背后的意思是：我们聊得投机，不如现在就见一面。但她还是拒绝了。因为明天的约会已经敲定，拒绝就算了。

### 避免成为情感垃圾桶

我们再往后看。这个姑娘一直在判断我的情况，为自己增加安全感。我说我是八九年生，她说她是九二年。我说：“要喊哥是不是？”但她没有搭理这句话。

她的心思在另一件事上，她说：“你说一个男孩子十天半个月都不主动联系我，他是真的喜欢我吗？”

很多兄弟在这里会误入歧途，变成情感垃圾桶。面对这种问题，我的回应是：“不知道。”对于姑娘明显想把你当情感垃圾桶的时候，你必须拿出态度。

她说：“你是个男孩子，你知道。”我说：“难说。”你要给她一种态度：不要把我当傻子，我不是你身边那些屌丝。

她很知趣，问：“你是不是不喜欢这个话题？”我又把话题引回来，说：“我饿啊。”她说：“你快去吃东西吧。”我说：“就一起吧。”我还在坚持邀约。当时可能也有点心不在焉，因为我经常同时聊很多人，你不答应我没关系，还有其他人。

后来她没理我了。这个对话就到了当天晚上十一点多。

### 控制约会时间与情绪挑逗

紧接着，第二天（6月14号）下午7点17分，她说：“还约吗？不说话，我吃完零食睡觉了。”

这时，我的策略是拖时间。因为她七点多就想约，虽然也算晚上，但不利于快速推进。策略是把时间尽量拖到晚得不能再晚，但不要拖到两三点，要在对方可接受的安全范围内。

她说：“大概几点我懂你吗？”我说：“还要到昨天那会儿。”昨天那会儿可能是十点多。我还说了一句话：“你先吃饭呀。”我使用一个客观条件：我因为忙，所以没时间出来吃饭或喝东西。这给了合理的借口，降低了约会成本，没有花一分钱。

她说：“十点都忙完吗？”她给出了心理时间预期，十点钟她可以接受，但说不定还可以拖更晚。

再往后，我看到这个女生十点多都愿意出来见面，说明她非常想见。这时，我又得寸进尺，发了一个亲亲的表情。

她的回应是：“哎呀。”面对这两个字，我是这样调情的：“咬到舌头了吗？”她的情绪又被挑逗起来了。我说：“哈哈晚点见。”

大概到了十点半，我说：“还没结束你在哪？”快到十一点的时候，她还在坚持等我，说明我的拖时间策略正确，虽然很晚，但仍在安全范围内。

我又继续拖时间：“看着不像快结束的样子。”其实，我当时在跟道长拍展示面。

最后她说：“那我要等到什么时候？”这时时间不能再拖了，拖太久对方会失去耐心。我说：“半个小时吧。”给她一个明确的回复。

她说：“这个第一次约会，我等别人还这么久，半个小时不出现，我就回家拉黑有劲。”显然，她有点生闷气。

我的回应是把这个情绪立马反转：“刚好你趁这个时间打扮的漂漂亮亮的。”还发了一个害羞的表情去增加性张力。

她说：“话说那么黑，你又看不清。”这时，挑逗情绪的关键时刻又来了。我说：“那凑近一点。”然后加一个害羞的表情。这个姑娘已经被挑逗得花枝乱颤了。

这时我不再拖时间，说：“快没电了，结束了联系你。”手机确实没电了。回到宾馆，用充电宝充电后，打了她的电话。我们后来在工体门口见了面。

### 约会后关系维护

大家再看后面，时间点跳到了6月15号半夜3点13分。那时，我已经把她送回去了。她说：“我睡不着了，好有罪恶感。”跟我约完之后很有罪恶感。

我说：“还沉浸在你的气息中，没回过神。”这还是一种很暧昧的感觉。
