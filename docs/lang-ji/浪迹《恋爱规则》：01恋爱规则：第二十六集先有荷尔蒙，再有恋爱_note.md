# 浪迹《恋爱规则》：01恋爱规则：第二十六集先有荷尔蒙，再有恋爱_note

在本节课中，我们将要学习一个核心的恋爱观念：真正的恋爱始于双方自发的“荷尔蒙”吸引，而非单方面的追求或妥协。

## 什么是“荷尔蒙”？

上一节我们介绍了课程主题，本节中我们来看看什么是“荷尔蒙”。在恋爱语境中，“荷尔蒙”指的是一种本能的、冲动的感觉，具体表现为：**想看着她，恋着她**。这是一种心动和渴望与对方在一起的自然情感。

## 单方面冲动不是恋爱

当两个人产生恋爱关系时，必须是你有这样的冲动，对方也有。很多时候，只有一方有冲动，而另一方没有同样的想法。

在这种情况下，主动方可能会尝试使用各种方法去打动对方。

以下是常见的、但并非基于“荷尔蒙”的追求方式：
*   使用物质条件。
*   付出极大的耐心。
*   表现得非常细心。
*   依靠长期的坚持。

最终，对方可能因为被打动而选择在一起。但你需要记住一点：**你获得的不是恋爱，而是妥协**。对方可能是认命了，或是为了物质条件、为了一个“对自己好”的人，而放弃了寻找真正心动的感觉。

## 对恋爱的常见误解

很多人错误地认为，两个人只要在一起或发生关系，就是恋爱。这其实是对恋爱的误解。

以下是几种典型的误解情况：
*   **粉丝与主播**：粉丝给女主播刷礼物，女主播回应甜言蜜语，粉丝便认为这是恋爱。
*   **酒后冲动**：两个人喝酒后，在晕眩且没有更好选择的情况下住在了一起，便认为这是恋爱。

以上的这些呢，都是大家对恋爱的误解。

## 真正的恋爱是什么？

而真正的恋爱是：你看到一个女生，有**心动的感觉**，想要跟她在一起。**并且，女生同样也对你有感觉**。双方的感觉缺一不可。

所谓的那一些，一个男生一来就看上了一个女生，但是女生对男生不理不睬的情况。男生通过套路、技巧、话术，天天去纠缠、追求这个女生。

最后，女生可能被他的“诚意”打动，或是慢慢发现了他的“好”，而选择跟他在一起。

那么这样的关系呢，浪哥会认为，这不是恋爱，而仅仅是“两个人在一起”。当然，在这种情况下，如果你和一个女生分手后，她也会难过。但她并不是因为“失恋”，而是因为“不习惯”——不习惯没有之前那种被照顾的生活。

## 总结与期望

本节课中我们一起学习了“先有荷尔蒙，再有恋爱”的核心规则。在这个世界上，确实存在许多没有恋爱感觉的结合。

但是我希望大家呢，是真正的因为**爱**和**喜欢**在一起，而不是因为一些外在的物质，或是因为各种妥协的原因。
