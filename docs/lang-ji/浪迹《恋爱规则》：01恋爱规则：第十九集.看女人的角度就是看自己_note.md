# 浪迹《恋爱规则》：01恋爱规则：第十九集.看女人的角度就是看自己_note

在本节课中，我们将要学习一个核心观点：你如何看待女性，本质上反映了你如何看待自己。你的内在信念和性格特质，会直接投射到你对他人行为的解读上。

## 核心观点阐述

你看待女性的角度，就是看待自己的角度。善良的人倾向于认为他人也善良，而内心邪恶的人则常以恶意揣测他人。你如何看待这个世界，就会以同样的方式看待女性。对于同一件事，不同的人会得出截然不同的结论。

## 实例分析：不同视角下的同一邀约

以下是针对“今天一起吃饭吧”这句简单邀约，不同心态的人可能产生的想法：

*   **防备心重的人**认为：“对方是不是要来蹭我的饭？”
*   **自大的人**认为：“你同意跟我出来，肯定就是喜欢我。”
*   **猥琐的人**认为：“你愿意跟我吃饭，那我是不是可以亲她？吃完饭怎么带走？”
*   **普通的人**认为：“她愿意跟我吃饭，是想跟我进一步相处吧。”

## 相处的重要原则

上一节我们看到了视角的差异，本节中我们来看看基于此的相处原则。

在未能充分了解对方时，切忌随意给女性下定义。同时，必须尊重每个女性个体所表现的善良或戒备，因为这个世界本就存在各种各样的人。

既然你选择与一位女性相处，无论结局如何，都需要准备好承担相应的后果。虽然从宏观上看，善良的人最终更容易遇到善良的人，但这并不意味着你可以毫无策略。

## 浪哥的建议：思维与行动的分离

以下是浪哥提出的一个具体相处策略：在思维层面，你可以将一个女性以你能想象到的最坏、最可恶的方式去预判其可能性；但在行动层面，你必须用你认为最善意、最友善的方式去对待她。
