# 浪迹大尧《简爱》：简·爱课程_简爱模型--相见_note

在本节课中，我们将要学习简爱模型中四大环节的第一项——“相见”。这个环节更通俗的说法是“收集资源”，它旨在解决一个普遍存在的问题：社交圈狭窄，不认识足够多的异性。我们将系统地探讨为何需要收集资源、如何操作以及具体做什么。

## 为什么需要收集资源？ 🌐

上一节我们介绍了“相见”环节的核心目标。本节中，我们来看看为什么在当今时代，资源收集变得如此重要且与以往不同。

核心原因在于我们进入了**互联网时代**。过去，即使你认识很多女生，管理她们的联络信息（如电话号码）也是一大难题。你需要为每个人做大量备注来区分她们，效率极低。

如今，以**微信**为代表的社交软件改变了这一切。我们可以通过头像、朋友圈动态轻松了解和管理联系人，并能通过标签进行分类。这为我们大量收集和管理资源提供了先决条件。

**互联网的本质是信息交换的便利性**。正如淘宝让我们能接触无数商家，外卖软件让我们能获取众多餐厅信息一样，现代社交软件让我们能快速接触到大量潜在异性。这是时代的大趋势。

传统行业（如国美、苏宁）向线上转型（如苏宁易购）的例子也说明了这一点。我们的社交形态同样从以线下见面为主，转向**先在线上收集资源并进行初步筛选，再落实到线下**。如果你拒绝使用社交网络，就如同传统企业拒绝转型，在别人进步时，你实际上是在退步，最终可能被淘汰。

因此，我们必须跟上时代，采用符合当下环境的玩法。

## 如何收集资源？ 🛠️

了解了时代背景后，我们来看看具体如何操作。以下是三种传统的资源收集方式：

1.  **社交圈**：通过融入不同圈子来认识新朋友。这种方法难度较高，更适合社交能力较强或处于后期阶段的人。
2.  **搭讪**：在现实场景中主动认识陌生人。这种方法起点高，对个人综合能力要求多，且有一定运气成分。值得注意的是，在许多女生看来，搭讪与社交软件认识的区别已经不大，她们可能认为“你能搭讪我，也能搭讪别人”。
3.  **社交网络**：这是当前时代最主流、最推荐的方式。

接下来，我们重点介绍如何使用社交网络。目前最流行、我个人最推荐的软件是**探探**。

它的核心机制是**双向选择**：系统会依次推荐附近的异性，你可以选择“喜欢”（点心）或“跳过”（点X）。只有当双方彼此“喜欢”时，才能开始对话。这有效地帮你筛选掉了对你完全没有兴趣的人，提高了沟通效率。

除了探探，还有其他类型的社交软件：
*   **传统交友软件**：如陌陌，以及一些更小众的软件。
*   **相亲软件**：如世纪佳缘，更侧重于以长期稳定关系为目的的交友。使用这类软件时，务必认真、详细地填写个人资料，特别是关于经济状况、家庭情况和对伴侣的期望等信息。

关于每个软件的具体操作步骤和技巧，我们提供了详细的文本教学资料。你可以向群内的导师索取，并对照着一步步实践。

## 做什么：目标与量化 🎯

最后，我们来明确具体要做什么，并设定一个可量化的目标。这里引入一个**358**的数字参考体系：

*   **300**：对于身处一、二线大城市的学员，我们通常要求资源收集量从300个起步。
*   **500**：我个人及团队助教导师的要求通常是500个起步。
*   **800-1000**：晋升到导师级别，通常需要有过收集800到1000个女性资源的经历。

这个数字听起来可能夸张，但在当今时代完全可行。我分享一个真实数据：去年在深圳，我仅用一款社交软件，在24小时内收到了约1200个“喜欢”；三天总计约2000个。经过层层筛选（过滤不靠谱账号、匹配成功、筛选不感兴趣的类型），最终能有效接触的仍有数百人。

如果你身处三、四线城市，可以对此数字适当打折，但在几天内认识上百人仍然是可能实现的。如果你达不到，问题往往出在个人资料（展示面）的设置上。你可以将个人资料发给我检查，我们一起优化。

## 总结与比喻 📝

本节课中，我们一起学习了简爱模型的第一个环节——“相见”，即资源收集。

最后，用一个比喻来强调其重要性：**把自己想象成一家小店，恋爱就是推销自己的过程**。
*   **过去**：就像小店开在偏僻处，客流量有限。你必须把产品（自身建设）做得极其出色，依靠口碑传播，才能吸引远方顾客。
*   **现在**：互联网就像把你的小店开到了全国乃至全球最繁华的商业街（如淘宝）。即使你是一家普通店面，只要曝光在足够大的市场里，就能获得巨大的流量。

提升经济实力、颜值等内在条件固然重要，但往往很难。而这个时代给了我们一个更直接的策略：**优先将自己投入到更广阔的市场中去**。因此，请立刻开始努力地收集资源吧！
