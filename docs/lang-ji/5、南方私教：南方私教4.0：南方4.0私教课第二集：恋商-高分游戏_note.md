# 课程5：恋商高分游戏 🎮

在本节课中，我们将学习如何通过个人提升与心态优化，在情感互动中建立优势。课程将围绕2019年情感核心理念展开，并深入探讨自我投资与时间管理的重要性。

## 一、2019年情感核心理念

上一节我们介绍了课程大纲，本节中我们来看看2019年情感领域的核心总结。其核心理念主要包含两个方向：**个人提升**与**社交直觉**。

### 1. 社交直觉

社交直觉是指在人际交往中，能够敏锐地察觉并理解他人行为、表情和言语背后未明说的潜在含义的能力。它不只适用于两性交往，也贯穿于我们所有的社会互动中。

以下是社交直觉的几个关键点：

*   **定义多样**：有人认为它是下意识分辨有价值人事物的能力；有人认为它是基于过往经历的潜意识判断。
*   **与感性相关**：女性通常被认为直觉更准，部分原因在于其思维更偏感性，且一生中经历的追求场景更多，积累了丰富的“社交数据”。
*   **核心是读懂信号**：例如，在社交场合中，高社交直觉的人会主动递烟以快速拉近关系，而低社交直觉的人可能只是旁观。
*   **应用广泛**：它体现在线上聊天和线下互动等方方面面。

### 2. 个人提升三大要点

社交直觉是我们未来会深入探讨的话题，而个人提升是本节课的重点。2019年的个人提升主要围绕以下三大要点展开，它们是一个循序渐进的系统：

1.  **个人品牌包装**：指对个人形象、气质、展示面进行系统性的塑造与优化。
2.  **资源收集**：指拓展社交圈，认识更多潜在交往对象。
3.  **实战技术提升**：指在实际约会与互动中运用技巧的能力。

这三者关系紧密：**个人品牌包装**是基础，能帮助你更有效地**收集资源**；资源充足后，**实战技术**才能在实践中得到锻炼和提升。试图跳过前两步直接追求技巧提升，往往是无效的。

## 二、核心法则：73855法则

在探讨个人提升时，我们必须理解一个关键法则：**73855法则**。该法则指出，在人际沟通（尤其是情感沟通）中，影响力的构成比例如下：

*   **55%** 取决于 **“你是谁”** （视觉形象、气质、身份、状态）
*   **38%** 取决于 **“你怎么说”** （语气、语调、肢体语言）
*   **7%** 取决于 **“你说什么”** （具体的谈话内容）

这个法则揭示了多数人的认知误区：过度关注“说什么”（那7%的话术和套路），而忽略了更重要的“是谁在说”（55%的个人建设）和“怎么说”（38%的沟通方式）。

**类比**：这就像考试。**55%的“你是谁”** 好比考生长期积累的学习基础和实力；**38%的“怎么说”** 好比基于大量练习掌握的应试技巧；**7%的“说什么”** 好比具体的考题内容，每年都在变化。执着于猜题（话术），而忽视夯实基础（个人建设）和练习技巧（沟通方式），是本末倒置。

## 三、自我投资：金钱与时间

理解了“你是谁”的重要性后，我们来看看如何通过自我投资来强化这55%。自我投资主要分为金钱投资和时间管理两方面。

### 1. 金钱投资的方向

金钱投资的目标是实现内外兼修的蜕变。你的投资应聚焦于能提升个人价值、增加吸引力的领域，而非单纯消费或谋生。

以下是几个值得投资的高分关键方向：

*   **健身**：塑造良好体魄，直接提升性吸引力与自信。
*   **烹饪**：展现生活情趣与照顾他人的能力，是长期关系中的加分项。
*   **摄影/视频**：提升审美，能制作优质展示面，是一项实用的社交技能。
*   **舞蹈**：提升肢体协调性与表现力，增强个人魅力和性张力。
*   **专业知识**（如心理学、情感咨询、性健康知识）：构建深度认知框架，在交流中能提供独特价值。

投资的核心在于 **“学习-实践-产出”** 的闭环。例如，投资摄影课并坚持练习，最终能拍出好照片，获得他人认可，从而增强自信，形成正向循环。

### 2. 时间管理：要事第一

仅有金钱投资不够，还需要高效的时间管理来确保执行。我们可以使用“重要-紧急”四象限法则来规划事务。

以下是四象限的分类与处理原则：

*   **第一象限：重要且紧急**（立刻做）
    *   例如：老板交代的今日必须完成的工作；即将到来的重要考试复习。
*   **第二象限：重要但不紧急**（计划做，持续做）
    *   例如：健身、学习新技能、听课程自我提升、规划个人发展。**这是投资自我的核心区域，需分配固定时间坚持**。
*   **第三象限：不重要但紧急**（酌情做，或委托他人）
    *   例如：不重要的朋友聚会、突如其来的琐碎请求。需控制在此类事务上的时间。
*   **第四象限：不重要且不紧急**（尽量不做）
    *   例如：无目的地刷短视频、沉迷游戏。应尽量避免，防止时间浪费。

**应用工具**：可以使用如`滴答清单`、`极简待办`、`小日常`等时间管理APP来辅助规划任务和培养习惯。

**在情感中的应用**：
*   **重要且紧急**：按时参加私教课学习（立即提升认知）。
*   **重要但不紧急**：制定并执行每周健身、学习计划（长期价值投资）。
*   **不重要但紧急**：女生发来无关紧要的闲聊信息（可稍后礼貌回复）。
*   **不重要不紧急**：反复刷对方朋友圈、漫无目的网上冲浪（应避免）。

## 总结

本节课我们一起学习了恋商提升的核心路径。我们首先明确了**个人提升**远比单纯追求话术更重要，并引入了**73855法则**，强调“你是谁”占据了影响力的55%。接着，我们探讨了如何通过**金钱投资**于健身、摄影等技能来实现内外蜕变，并学习了利用**时间管理四象限法则**来优先处理“重要不紧急”的自我提升事务，确保行动力。

记住，情感游戏的高分秘诀，始于对自身的认真投资与高效管理。
