## 约会需有明确安排 🧭

上一节我们探讨了约会的基本心态，本节中我们来看看如何具体规划一次约会。一次成功的约会必须有清晰的安排。大的方向应由男生主导，例如决定约会的主要活动和流程。小的细节则可以配合女生的喜好进行调整。关键在于，你需要明确知道约会要做什么，而不是在与女生见面后才开始漫无目的地商量或闲聊。

## 准备两个备选方案 🔄

除了安排好主要约会项目，一个周全的计划还需要预备应对突发状况的方案。以下是两个必须考虑的备选方案：

**第一个备选方案：应对对方迟到或爽约。**
如果对方放鸽子，或者让你等待超过半小时，你需要有备用的行动计划。

**第二个备选方案：应对现场突发状况。**
约会过程中可能出现各种意外，你需要提前想好对策。例如：
*   你约女生去游泳，但她突然来生理期。
*   你预订了一家特色餐厅，但餐厅临时关门或需要排长队。
*   你们需要在约会地点之间转场，却一直打不到车。

在约会之前，就应做好相应的应急准备。
