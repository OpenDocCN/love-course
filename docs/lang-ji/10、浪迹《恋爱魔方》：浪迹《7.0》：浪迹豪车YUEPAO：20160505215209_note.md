## 核心概念：真诚表达与资源收集

上一节我们概述了课程主旨，本节中我们来看看具体的方法论。把妹过程可以简化为四个环节：**收集资源、建立吸引、深化关系、关系维护**。

其中，**收集资源**是第一步，方式多样，例如：
*   搭讪
*   社交软件（APP）
*   社交圈扩展

**搭讪**只是收集资源的一种手段。关键在于最终能否获得资源，而非拘泥于具体形式。搭讪时说什么并不最重要，**真诚地表达自己**才是核心。

**公式：把妹 ≈ 收集资源 + 建立吸引 + 深化关系 + 关系维护**
