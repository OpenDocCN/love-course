# 课程17：《情敌攻略视频课》+《情敌手册书》：4. 颠覆你对把妹的认知：人设 🎭

在本节课中，我们将要学习一个核心概念——**人设**。这个概念将帮助你理解，为何同样的技巧在不同人身上效果不同，并指导你根据自身情况，制定最有效的互动策略。

以上内容介绍的都是关于学习本网络课程的一些基础概念，并未涉及具体操作过程。从第四章开始，我们将正式讲解“解题”方法。请将书翻到第23页。

在这里，我将介绍一个在把妹中至关重要、但你可能未曾听过的概念：**人设**。那么，人设究竟是什么呢？根据我多年的经验，有时我所说的某些话，其他兄弟无法复制，或者复制后女孩子的反应并不理想。

大家需要明白，我传授的所有内容并非凭空想象，而是基于我近十年来与成千上万名女性相处，以及教授成千上万名学员后，根据他们的反馈总结得出的。

我传递思想的方法，之所以不讲固定的话术和惯例，并非为了标新立异，而是因为它们确实不适用。因此，我认为现在告诉你的内容，对你当前的把妹实践是适用的。那么，人设到底是什么呢？

首先，我们需要把“把妹”定义为一项**社交活动**。在本书的第一行就写道：社交活动的关键在于，女人不是一个死板的对象。例如，如果我做一套试卷，应用题描述得很清楚。

我解题最终只有一个答案，因为题目是固定的。所有人面对这道固定题目，答案总是一致的。很多人认为把妹也是如此，女人每说一句话，总有一个标准答案。这其实是错误的。

为什么？因为把妹是一种社交活动，这意味着它是**动态变化**的。相当于这道题本身、题干、内容以及其中的数据都在不断变化。你不可能知道每道题的答案，你只能掌握每道题的**解题思路**。明白我的意思吗？

说到解题思路，就必须归结到一个新问题：**谁在做这道题**？如果你只有小学三年级的学习能力，你只能做三年级的试卷。若以三年级的能力去做高中试卷，即使学会了策略，你的智商也无法支持。首先，你得明白自己处在“几年级”，然后再去解相应的题。

那么问题来了。我们所有的网络课程，特别是2022年最新这一期，只讨论一个特定范围：**年轻漂亮的女生**。如果你要追求30岁以上的女性、18岁以下的女学生、外貌不佳的女生、普通女性，或者网络上那些初级女权主义者，本课程策略并不适用。因为她们的想法，与我们目标女性的想法是不同的。

你对女孩子提出了要求（年轻漂亮），那么相应地，你自己是否也应该有所要求？你之所以找那些普通的妹子，是因为你自己也普通。因此，在本书中，我们开始的第一件事，就是一定要看清下面这张图。

请翻到第29页，查看这张**男性吸引图**。图的横轴代表你的**价值**，越往左价值越高，越往右价值越低。纵轴代表你的**颜值**，越往上颜值越高，越往下颜值越低。图中的“T”代表10分。

例如，左上角的“TT”代表颜值10分，价值10分。本书讨论的范围，**不涉及“七七”以上**的区域（即颜值7分且价值7分以上的组合），也不讨论图中的空白区域。如果你发现自己落在空白区域，那么你需要去提升自己。书的后半部分有关于如何提升的内容，请自行翻阅。

但这里要讲的是，如果你落在了书中的空白区域，并不意味着你把不到妹，而是你所处的情况不适用于本书所教的策略。这相当于我不教小学六年级以下，也不教大学一年级以上的内容，只教初中一年级到高中三年级这个范围。我们这本书就是针对这个范围。

接下来，我们来看图中关于“人设”的关键点。你会发现，图中“49”（颜值4分，价值9分）这个方块的顏色，与“57”（颜值5分，价值7分）、“65”（颜值6分，价值5分）的顏色是一样的。

这意味着，对于年轻漂亮的女孩来说，**你非常有钱但长相一般**、**你长相不错且有点小钱**、**你长得好看但没啥钱**，这三种情况对她的**吸引力可能是一样的**。

但是，为什么我还要提出“人设”这个关键点呢？因为即使吸引力相同，你与她们相处的**策略却应该不同**。女人因为你的条件好而跟你在一起，和因为你的长相好看而跟你在一起，你与她的相处方式其实是不一样的。

下面，我们将通过一个实际案例来说明这一点。

书中这里写了一个案例。女孩子问：“你对所有女生都是这样OK的吗？” 接下来，我会根据人设的不同（即你落在图的不同区域），来分析你应该如何回答。书中为此定下了一些原则，请大家仔细查看。

这时就会出现一个问题：如果你的颜值和价值有所不同，你对女生同一句话的回应也应该不同。我来告诉大家一个核心原则：**你对女孩子的吸引力越高，你的态度就应该越平和；如果你的吸引力没那么高，你就应该越主动**。

这是什么意思呢？就是说你要去**平衡你自己的人设**。如果你已经对她很有吸引力了，却还表现得很主动，女生就会觉得“事出反常必有妖”——你这么好的人为什么这样做？反之，如果你的吸引力不足，却还端着架子，女孩子会觉得你在装。

书中详细解释了为什么要这样做。在本节中，我要讲的是，你落在每一个区间（比如同样是回答“你是不是对所有女生都这样”），因为区间不同，你选择的方案也应不同。当然，你首先要判断自己在这个**特定女人眼中的区间**，而不是严格意义上的客观区间。这就是本书要解决的问题。

此外，如果你现在落在图中的空白格子里，其实不必过于担心。对于男生的颜值，我们的要求并没有那么高。只要一个正常的男生花点时间、花点心思努力健身和提升自己，你的颜值也能达到标准以上。因此，只要你按照我们的一些策略去实践和体验，这本书的策略对你依然是适用的。

以上是关于一个具体案例的讨论。本书会将你不同的人设、在不同区间的定位以及应采用的策略都详细写入。如果你看了不明白，可以在群里提问。

因为书中的内容相当于一个“框架”，你看完后首先需要对其有定义。至于具体如何使用，我们可以在群里交流，毕竟每个人的情况都不一样。你首先还是需要去阅读并理解它。
