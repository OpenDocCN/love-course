# 课程一：老吴承情速成课《女人倒追》第十八课 - 承情暧昧后的升高 🚀

在本节课中，我们将要学习如何在与女性建立初步联系和暧昧氛围后，有效地“升高”关系，推动关系向更亲密的方向发展。课程将分析几种典型的关系发展曲线，并重点讲解“逆袭”场景的核心要点，以及线上聊天时升高关系的具体方法。

## 关系发展曲线分析 📈

上一节我们介绍了关系发展的几种模式，本节中我们来看看其中最具挑战性的一种：“逆袭”。

“逆袭”曲线非常特别，它的前端特别低。这意味着在前期，我们和某个女生可能很长时间都没有开启话题，或者互动很少，之后才因为某些情况关系得以推进。

通常，我们加上女生微信后，会在24小时内找她聊天，进行基本信息互换。在这个过程中，你会感知到女生是否有“窗口”，即她对你的兴趣值高不高，是否愿意和你继续发展。

但是，有些女生在前期你的判断和测试过程中，没有流露出明显好感，甚至反应比较差。因此，你们的关系会停留在“普通朋友”甚至“微信陌生人”的状态，双方都不好意思主动联系。

然而，可能在未来的某一天，因为一个机缘巧合，你又和这个女生建立了联系，并发现她的窗口变大了，最终成功邀约并发生关系。这就叫做“逆袭”。

## “逆袭”的核心要点 🎯

以下是实现“逆袭”需要把握的几个核心要点。

### 1. 持续展示与存在感

首先，很多人朋友圈里都有一些很久没聊天的正妹。你从什么时候开始重新聊天呢？通常是在出现一个临时契机或窗口时。

你抓住了这个契机进行评论或互动，发现女生窗口较大，于是这段时间着重关心她，直至最终确立关系。这是一个“逆袭”的标准剧本。

什么样的男生能够逆袭？关键在于能一点一点抬高自身价值，引起女生注意。

第一点，如果发朋友圈，应多展示自己的积极社交状态。因为如果你不展示，女生就无法了解你的近况。

通过持续的状态刷新，即使女生当时有男朋友或其他对象，她也会注意到你这块“品质优良的肉”，并在脑海中形成对你的个人印象。当这个印象建立后，就方便进行下一步邀约了。

如果女生脑海中的印象是负面的（例如“挫男”），邀约就会困难。但如果你的展示面不错、有格调，女生会记住，即使当时不出来，在未来的某个契机点也可能答应邀约。

### 2. 寻找并抓住特定契机

“逆袭”需要找寻一个“契机”作为开场点。如果你不知道如何找寻，最简单的方式是多刷朋友圈，并偶尔关注她的朋友圈。

例如，她今天恰巧发生了某件事，而这件事是你比较在行或懂的。可能就因为这么一个机缘巧合，你们能建立一个短暂连接，并更深入地互相了解。

核心理论很简单：**你要频繁地、以非骚扰的方式出现在女人的视野中**。即使她对你的印象模糊，频繁的正面出现也会让她产生基本安全感和熟悉感。

通过一个契机互动后，你们可能会有一种“相见恨晚”的感觉。在曲线上，通常前半段是“没窗口”的，可能因为她有男朋友，或当时有更优质的暧昧对象。

### 3. 契机出现后快速升高关系

在“逆袭”过程中，除了抓住契机，另一个关键点是 **“快速升高”**。

因为你们是“相见恨晚”的感觉，是“缘分”。如果你不通过这个契机点马上扭转之前的相处模式，关系就会滑回以前那种平淡或爱搭不理的状态。

例如，一个正妹在我朋友圈躺了一年。今天我发了一张陈奕迅演唱会的票，她也很喜欢陈奕迅，于是主动找我聊。此时热络指数最高。

但如果过两天我们没有实质关系进展，关系可能又会跌回普通朋友状态。因此，“逆袭”的升高曲线是特别陡峭的：前期缓和没窗口，一旦契机出现，关系就笔直向上，甚至可能借着一次约会就直接确立亲密关系。

**“逆袭”最重要的手段和方法是：发现女生有窗口后，立即快速升高关系。**

有些妹子是朋友圈里平时撩不动、没窗口的。但凡她因某件小事对你打开窗口，你就应该意识到要快速升高。因为可能这件事让你在她眼里有了特殊价值，分数暂时变高。但如果时间一长，她清醒了或兴趣过去了，可能就没机会了。

所以，“逆袭”一定要快。一旦有妹子在你朋友圈点赞评论、窗口大开，你一定要第一时间把握住，否则两人终究会越走越远。

## 线上聊天的关系升高技巧 💬

上一节我们分析了“逆袭”曲线，本节中我们来看看在线上聊天（特别是微信）中，如何具体操作来升高关系。主要有三个要点：话题私密性、肢体语言亲密（线上模拟）、称呼亲密。

### 1. 话题的私密性

在微信上聊天，升高关系最重要的一个点是：**聊的话题不能太公开、太普通**。

例如，“今天吃什么”、“天气很好”这类话题，在任何阶段、对任何妹子都可以讲，没有太多私密价值，也就无法推进关系。

“私密”意味着让对方感觉“我跟你说了心底的小秘密”。如果你和一个女人说的“小秘密”越多，你们的关系就越深。

在聊天中，如果女生跟你聊起前男友、受伤的经历、展示脆弱面、刷心疼指数等，这说明她已经对你打开了很大的窗口，愿意和你交流私密话题。

私密话题包括走心的，也可能包含走肾的。有些女生会发一些色色的表情，如果你接得好，对话可能转向更私密的领域，甚至约到酒吧等地。话题私密性是帮助升高关系的第一点。

### 2. 肢体接触的亲密（与线上铺垫）

两个人有过肢体接触后，自然会产生情愫和暧昧感。在网上聊得再好，没有搂抱、牵手、接吻，感觉总是缺失的。

肢体接触越深入，相当于你对这个人的“标记”越重。这里引用一个观点：**“这个世界上没有爱情，有的只有爱的证明。”**

世界上绝大多数东西都可以用计量单位衡量，但爱情没有。你不能说爱A比爱B多三分之一。因此，我们通过“爱的证明”来判定爱情。

举一个极端的例子：前女友A只牵过手，B只接过吻，C上过床。那么大多数人会认为C的羁绊最深，付出最多。肢体接触的程度越高，证明亲密关系越深。

再进一步：A上过床，B还有过口交等更深入的行为，C甚至为你堕过胎。那么大多数人会认为C是这三个女人中最爱你的。

因此，女人对你付出的多少，可以通过身体接触的全面性和深入程度来证明。在撩妹过程中，有时光说没用，女生会认为那是甜言蜜语或编排好的台词。但如果你真正付诸行动（包括线上为线下接触做铺垫），她会觉得你是真心的。

所以，要善用“肢体接触的亲密”来升高关系，而不是只打嘴炮。

### 3. 称呼的亲密

称呼亲密其实很简单，就是经常和妹子起一些专属的小外号。

这让姑娘感觉到你和她是独属的一个团体，你们说的话、用的称呼，只有你们两个人能互相理解。这是恋爱中一种很好的状态。

我的建议是保持常态，多做“称呼亲密”。例如，每天叫女朋友“宝宝”、“小乖乖”、“小可爱”等。

前两天我和一个姑娘在一起时，她问：“你喜欢我吗？”我说：“这哪叫喜欢，这简直是爱。”她问：“你有多爱呢？”我说：“我对你的爱，就好比现在我对空调的爱。”姑娘说：“原来我还不如空调啊？”我说：“你不是不如空调，你就是空调，你是我的小空调。”她问为什么，我说：“因为你平时对我很高冷，我希望你对我热情一点。”之后我叫她“小空调”时，她反应特别好。

这说明，通过这种有典故的昵称，女生会觉得你们之间经历了特殊的事情，这个称呼有特殊意义。这就是一种关系的升高和升级。

这个惯例可以直接用：在夏天称姑娘为“小空调”，说“我对你的喜欢，跟我现在对空调的热爱是一样的”。

## 终极实战技巧：语音通话升温 📞

最后，分享一个我私教课程中的内容，也是我现在最常用的升高关系方法：**语音通话**。

通常，我和姑娘网上聊得很好，约见面后，如果当晚没事，我会考虑和她打个电话。

因为现在人与人初级交往多在微信上，是冷冰冰的文字和图片。如果你能和姑娘通电话，用声音刺激她，她对你的好感会有剧烈加成。

具体操作：如果你打算最近几天和这个姑娘确立关系，怎么做“中断诱惑”和升高关系呢？答案是：给她打电话。

例如，你们交往挺好，见过一两次面，姑娘对你也挺好。你想推进关系，可以问她：“亲爱的你忙不忙？今晚我们要不要语音一会儿？”聊20分钟到1小时，随便聊聊。

语音和见面不同。见面时，很多信息靠眼睛捕捉，一旦冷场，气氛不好把握。但语音通话很少冷场，不是你在讲就是她在讲，状态更轻松。

你可以找个地方舒服地躺着，举着电话轻松地聊。女生听到你的声音会产生联想。人与人颜值差距大，但声音差距没那么大。很多女生会因脸拒绝你，但很少因声音拒绝你。

打电话时，女生是拿出专门时间陪你做“升高关系”这件事，而不是一边看电视、聊天或做其他事一边回你信息。这让她觉得，你是她社交圈中一个特殊的存在。

因此，我最大的建议是：只要我和一个妹子聊得不错，并打算最近推进关系，我就会问：“方不方便打个语音？”稍微调节气氛，然后见面确立关系。

这是我现在最实战、最好用的升级套路：在关系中断期，通过语音通话加一把剧烈的油，让感情迅速升温，然后在见面时自然推进。

**打电话的技巧**：可以下载“唱吧”等有给陌生人打电话功能的软件练习。你可以选择“只给女人打电话”，虽然接通率不一定高，但可以练习聊骚，锻炼自己的状态。最重要的是“入戏”，自己先沉浸在那种状态和感觉里，才能带动对方。
