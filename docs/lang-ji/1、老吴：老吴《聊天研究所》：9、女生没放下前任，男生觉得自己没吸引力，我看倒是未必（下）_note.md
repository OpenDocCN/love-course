# 聊天研究所 第9课：当女生未放下前任时，如何有效沟通（下）🔍

在本节课中，我们将继续分析一个具体案例，学习当女生未完全放下前任时，男生应如何避免错误沟通，并通过共情与支持来建立信任与吸引力。

---

上一节我们讨论了错误沟通的危害，本节中我们来看看具体的聊天记录分析，并学习正确的应对方法。

我们首先来看男生的第一句话。  
他发送了多个省略号表示无语，七分钟后说：“你还是对他抱有希望，你这样会活在痛苦中。”  
这种行为属于在对方伤口上撒盐。

女生的回复是：“我不知道自己该怎么办。我不够狠心。”  
作为刚分手的当事人，她很难独自处理情绪，因为她尚未从上一段感情中脱离。这不是狠心与否的问题，真正爱过的人很难快速放下。

如果我是男生，看到这句话后会选择表示共情。人心是肉长的，没有必要再去揭开她的伤口。  
如果你真心喜欢这个女生，最佳处理方式是避免再提此事，做好自己该做的，帮助女生进入开心快乐的氛围，而非沉浸于悲伤。

然而，男生又回复：“他要是爱你，会这样对你吗？”  
这再次在女生伤口上撒盐，显得情商很低。

从男生角度看，他的出发点是认为女生应该怎么做，并想帮助她走出痛苦。但这种方式没有处理好，因为女生此时不想听到这样的话。  
作为正常人，选择伴侣时都会认为对方独特，分手则意味着自己当初看走了眼。谁都不愿面对这个现实，即责怪自己最初的选择。

男生不应在女生伤口上撒盐，而应安抚她的情绪。  
设想一下，女生失眠到凌晨两点，发了一大段文字，而你已经睡着没有回复。早上时，你应该关心她熬夜是否因痛苦所致，而不是一上来就戳她的痛处。她本就痛苦，能量很低。

我在情感课程中讲过，两人在一起是为了追求开心。女生不开心、情绪低落时，正是在向你展示内心的脆弱，向你求助。  
“放下”对大多数人来说非常困难。很多人嘴上说放下，心里却需要很久，甚至一辈子都放不下。

这个男生给人一种站着说话不腰疼的感觉。他应该做的是共情，因为女生善良心软，人都有感情。  
当对方展示脆弱时，你应站在同一立场，而非以讲道理、将事情简单化的角度回应。否则只会让女生关闭心扉、戴上面具，可能只愿与你做朋友，而你再也无法走入她的内心。

就像与猫狗玩耍时，它们若喜欢你，会翻身让你摸肚子。  
女生将痛苦展示给你看，你应去爱护并帮助她疗愈。当女生说“我连一个不爱我的人都放不下”时，你应表示共情：“爱过一个人，轻易说放下真的很难。如果那么容易放下，只能说明没有爱过。”

结果男生又说：“不要再跟他联系了，好吗？”  
这是一种强制命令的语气。说真的，女生又不是你的谁，为何要这样说话？每一句话都在伤害她，造成二次伤害。

接着女生说：“我没联系他，忍得很痛苦，假装不在乎。昨晚没忍住发给他，也是想叫他互删。发完之后也撤回了。”  
从这几句话可以看出，女生还没放下，想放下却放不下，这很合理。  
你只需按我刚刚的方式回复：“真爱过一个人，放下是需要时间的。”表示同理即可。

男生又说：“我不知道该说什么好。你在想别人的同时我也在想你。”  
看到了吗？他永远在考虑自己的感受。如果简单地删除好友就能放下，那说得太容易了。只能说大家消失在好友列表里，但心结一直存在，没有打开。  
我能理解男生确实很想拥有这个女生，但有些话不能这么说。

女生回复：“就不要想，想想怎么努力赚钱都好。”  
这就是我刚刚说的，女生向你打开了脆弱的一面，你却没有妥善处理，于是女生自己转移了话题。  
这个女生真的给了男生很多次机会，但男生一直没抓住重点。

女生转移话题后，男生还说：“遇见你不容易，我不想放弃。”  
这真的很糟糕。

后面的聊天记录开始变得冷淡。虽然女生还会关心男生，但要看男生能否很好地共情并疗愈女生受伤的内心。

所以，正在听课的你，如果是男生，遇到女生内心受伤时，请好好帮助对方疗愈。  
姑且不论疗愈后她是否会选择你，但如果你真能帮她疗伤，她对你的信任度十有八九会非常高。  
即使没有在一起也没关系，你也算做了一件好事，因为你帮助别人从痛苦中抽离。这也是我们提高情商的重要性所在。
