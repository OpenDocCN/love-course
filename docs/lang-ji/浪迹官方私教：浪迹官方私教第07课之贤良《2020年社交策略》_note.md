# 浪迹官方私教第07课：2020年社交策略 📚

在本节课中，我们将学习社交的核心原理与实用策略。课程将帮助你建立正确的社交认知，理解“行情”的本质，破除社交障碍，并掌握主动社交的必胜心态。我们将从基础概念到实践方法，系统地讲解如何提升你的社交能力。

## 第一小节：社交力学原理 🔍

上一节我们介绍了课程的整体框架，本节中我们来看看社交的基础原理。社交力学旨在揭示“好人缘”背后的逻辑。

好人缘等于受人欢迎的程度。要成为受欢迎的人，首先需要了解不受欢迎的人有哪些特点。

以下是四种不受欢迎的特质：

1.  **语言类**：说话刻薄、爱钻牛角尖、习惯撒谎。
2.  **行为类**：爱占便宜、举止粗鲁野蛮。
3.  **外表类**：形象邋遢、不修边幅。
4.  **味道类**：身体或口腔有异味，令人难以接近。

了解反面案例后，我们再来看看受欢迎的人具备哪些特质。

以下是好人缘的四个维度：

1.  **语言**：谦逊、诚恳、诚实，不随意评价他人。
2.  **行为**：处事大方、优雅稳重。
3.  **外表**：整洁、精致。
4.  **味道**：清新、干净。

在社交中，第一印象至关重要。这涉及到“七三八五原则”。

**七三八五原则公式**：
-   对一个人的印象，**7%** 来源于说话的内容。
-   **38%** 来源于说话的语气和语调。
-   **55%** 来源于外形与肢体语言。

这个原则说明，外形和肢体语言在社交中比说话内容更重要。

明白了原理，接下来我们看看具体的提升方法。核心在于四个字：**刻意练习**。

以下是针对不同维度的练习方法：

*   **语气语调练习**：
    1.  模仿电视或新媒体中有魅力的男士。
    2.  大量阅读并练习朗诵。
    3.  每天对着镜子练习十分钟。
    4.  录下自己的声音，反复听并改进。

*   **行为与肢体语言练习**：
    1.  模仿电影、电视中有魅力角色的举止。
    2.  每天对着镜子练习十分钟姿态和表情。
    3.  录制自己的视频，观察并调整不协调的肢体动作。

*   **外表形象提升**：
    1.  模仿穿搭品味好的人。
    2.  去服装店试穿至少100套不同风格的衣服，找到适合自己的类型。
    3.  寻找一位审美在线的靠谱发型师。
    4.  坚持每日护肤（清洁、保湿、每周敷两次面膜）。

*   **味道管理**：
    1.  每半年洗牙一次，早晚刷牙。
    2.  每日洗澡洗头，勤换内衣。
    3.  选择一款清香型的男士香水，少量喷洒在衣物上即可。

## 第二小节：行情决定论 📈

上一节我们探讨了社交的基础原理和提升方法，本节中我们来分析一个关键概念：“行情”。

行情具有时效性，它指的是在特定时间段内，被市场（社交圈）所认可的价值。行情会随时间变化，可能升值也可能贬值。

渣男渣女之所以有“行情”，根本原因在于他们当下拥有一定的硬价值和软价值。

**价值构成公式**：
-   **硬价值**：颜值、身材、车子、房子、职位、财富。
-   **软价值**：高情商、幽默、有趣、有潜力、善于思考。
-   **综合价值** = 硬价值 + 软价值

价值可以通过交换和努力获得。如果你目前不具备这些价值，可以通过以下三个阶段来创造：

**价值创造三阶段**：
1.  **投入时间，获得经验**。
2.  **积累经验，产生能力**。
3.  **利用能力，获取价值**。

不要抱怨自己“不会”，而应思考投入了多少时间，积累了哪些经验，培养了何种能力。

## 第三小节：社交障碍 🚧

上一节我们理解了“行情”源于价值，本节中我们直面阻碍很多人迈出第一步的核心问题：社交障碍。

社交障碍最核心的原因，并非技巧不足或环境限制，而在于**心智未开**。即对世界的认知不足，内心被自我设定的枷锁禁锢。

关键在于理解“目的论”与“原因论”的区别。

**案例分析**：
-   **原因论**：看到心动的女生，心想“她走路太快了”（找理由），所以不去搭讪。本质是先下了“不去改变”的决心，再为这个决定找借口。
-   **目的论**：我的目的就是去认识她。无论她走路快慢，都是一次独特的体验。关键在于迈出第一步。

我们的行为常受两种思维影响：
1.  **逻辑思维**：客观分析（如健身有益健康）。
2.  **感官思维**：注重即时体验（如健身很累）。

社交障碍往往源于我们听从了感官思维的“劝阻”，并为“不改变”寻找各种客观理由。要破除障碍，必须运用“目的论”，坚定“我要去做”的信念，而非纠结于“为什么我不能做”的原因。

## 第四小节：社交必胜论 🏆

上一节我们剖析了社交障碍的根源在于心智，本节中我们将建立一种关键心态：社交必胜论。主动社交是一件百利而无一害、必定胜利的事情。

首先，我们来破除关于主动社交的常见错误认知。

以下是关于主动社交“害处”的辨析：

1.  **时间成本**：掌握任何技能都需投入时间，这是必要过程，而非损失。
2.  **丢面子**：在获得实质成功前，过度关注“面子”是自我设限。
3.  **自己不喜** 欢：喜不喜欢是成功者的特权。在成功前，应以目标为导向。
4.  **困难**：做难事必有所得。正是克服困难的过程带来成长。

主动社交的核心好处在于：**将虚无缥缈的缘分，掌握在自己手中**。实力决定缘分。

为了坚定行动，我们需要给自己积极的“正向暗示”。

以下是四大正向暗示：

1.  **必胜**：只要我主动了，目的（社交本身）就达到了，我就已经胜利了。
2.  **强与弱**：敢于主动社交的人是强者，敢于冒险的人才是真正的强者。我超越了自己和99%的人。
3.  **必须主动**：社会文化期待男性主动，这是男性的责任，也是机会所在。
4.  **平等**：社交是一次双向选择（投票）。我有权认识对方，对方也有权拒绝。拒绝只代表对方行使了选择权，不意味着我这个人失败或没有价值。

## 第五小节：保持善念 🤝

上一节我们确立了主动社交的必胜心态，本节作为课程的升华，我们来探讨支撑长期社交与个人成长的基石：保持善念。

善念包括利他精神、乐观态度和坚韧的意志。一个信念不坚定的人，容易因他人看法或几次挫折而放弃。

为什么必须保持善念？

1.  **没有善念就没有协作**：社会运行依赖于基本的规则和善意，缺乏善念会导致混乱和恐惧。
2.  **无人愿生活在恐惧中**：一个充满恶意的世界是所有人的噩梦。
3.  **防止被奴役**：如果强者不持善念，普通人将面临被压迫的风险。历史上有过深刻的教训。

一个有志向的人必须明白的道理：**小富靠勤，中富靠趋势，大富靠德**。

-   **趋势**：预知未来的能力。
-   **德**：信念与价值观，即做人的道理。
    -   **信念**：你坚信什么（如坚信努力带来成功）。
    -   **价值观**：你认为什么有价值（如认为赚钱同时帮助他人有价值）。

正向的、被社会广泛认可的信念与价值观，是个人取得长远成就的根基。

**案例**：
-   李嘉诚：诚信与厚道，不让合作伙伴吃亏。
-   马云（阿里巴巴）：满足他人需求，成就他人（帮助买家安全购物、帮助卖家经营）。

正确的社交信念与价值观应是：
1.  **换位思考**：对方需要什么？（己所不欲，勿施于人）
2.  **他者贡献**：我能为对方提供什么？

## 课程总结 📝

本节课我们一起学习了2020年社交策略的核心内容：

1.  **第一印象决定社交反响**：重视七三八五原则，外形与肢体语言比说话内容更重要。
2.  **行情可通过努力获得**：价值由硬价值和软价值构成，可通过投入时间、积累经验、创造能力来提升。
3.  **社交的前提是解开内心枷锁**：社交障碍的核心是心智未开，要运用“目的论”而非“原因论”来推动自己行动。
4.  **主动是最好的社交方法**：主动社交有百利而无一害，能让你掌握缘分的主动权。
5.  **正向的信念与价值观贯穿始终**：保持善念，具备利他精神和坚韧意志，是长期社交与个人成长的基石。
