# 课程名称：搭讪实战指南 - 第25课：搭讪的具体套路 🚶♂️💬

在本节课中，我们将要学习搭讪的具体操作流程、常见问题应对方法以及核心步骤。我们将从开场白开始，逐步讲解过渡、沟通和收尾阶段，帮助你掌握一套完整的搭讪策略。

## 概述

上一节我们介绍了搭讪的组合类型和难易程度。本节中我们来看看搭讪中会遇到的具体问题以及如何应对。首先，我们需要准备一个开场白。

## 开场白类型

开场白分为直接和间接两种。间接开场白通常需要一个相对自然合适的机会，因此通常不建议使用。直接开场白在2013年教学时给出了一个模板，现在被广泛使用。

许多女生可能都听过类似的开场白。以下是直接开场白的一个示例：

```
你好，我刚刚在那边等朋友，看到你经过这里，我觉得你很漂亮，所以想过来打个招呼。我叫吴，怎么称呼？
```

根据女生的反应和窗口，继续进行下面的对话。

## 应对女生反应

如果女生问“你有什么事情”，说明她此时有很大的不安全感。我们应该立即露出微笑，身体靠后一两步，然后放慢语速，详细重复之前的开场白。甚至可以加上一个时间限制。

例如：

```
是不是吓到你了？是这样的，我和朋友约在那边见面，她还没到，我在等她。刚好你经过，我觉得你的气质很不一样，所以鼓起勇气和你说话。可能这种方式太唐突，吓到你了，实在不好意思。
```

如果通过观察发现女生的表情状态放松了一些，安全感提高了，就可以开个小玩笑，让气氛更轻松。例如：

```
你放心，我不会让你去买毛鞋做头发的。其实我现在紧张得要死，也不知道用什么更好的方式认识你，所以只能豁出去了。
```

如果女生没有离开，而是站在原地，说明她已经不再那么害怕。我们可以进一步建立社交联系。例如：

```
哦，对了，我叫吴过，是做咨询工作的（或做互联网的）。你怎么称呼？
```

女生通常只会告诉你姓氏，比如“我姓张”。这时我们可以追加询问并开个玩笑：

```
你叫张什么？不如开个小小的玩笑，张总你好，以后多多关照。
```

如果我们的肢体、表情和状态都没有问题，女生通常会开心地笑起来。如果女生完全放松下来，开场阶段基本完成。

## 过渡阶段话题

接下来，我们需要给出一个过渡阶段的话题。在初步的社交联系中，可以继续维持对话。通常可以把话题转移到女生的打扮或当下环境上。

例如：

```
我觉得你的裙子和鞋子颜色搭配起来非常好看，气质也很不一样。你是做媒体或时尚行业的吗？
```

这是一个小小的恭维和赞美，同时也能进一步了解女生的信息，比直接问“你是做什么行业的”更委婉。

女生可能会回答“不是，我是做互联网行业的”或“我是做教育的，我是老师”。我们可以继续开玩笑：

```
你是做互联网的，难道你是传说中的IT程序员吗？我印象中的女程序员不是长你这个样子，她们一般不是两眼无光、表情呆滞、戴着大大的眼镜。你这眼睛太不符合了吧。
```

或者：

```
你是老师？感觉这么熟，我觉得你和我的小学英语老师长得特别像。她特别亲和友善，而且人又特别漂亮，我们班当时很多小男生都暗恋她。
```

到这里，女生可能会笑起来。如果在搭讪的开场和过渡阶段，女生对我们逗笑了四次以上，就可以为这两步打90分。安全感和舒适感都有了，接下来可以更自然地建立联系感。

## 搭讪禁忌

在搭讪最开始的开场阶段和过渡阶段，最忌讳的是没有微笑、没有眼神接触、过度紧张、讲话时不思考或停顿太久。这些都会让女生本来就不高的安全感进一步降低，导致女生离开。

## 过渡阶段技巧

在过渡阶段，还可以借助女生说话时的口音或身高感觉来提问。例如：

```
我听听口音，感觉你不像西安人。你是在西安上学的吗？
```

如果女生说“我有工作了”，我们可以用前面提到的方法，对女生的穿着打扮进行恭维，然后询问她的职业。如果女生说是的，我们可以说：

```
原来是大一的小学妹。
```

女生可能会纠正你：

```
没有啦，我可能毕业了，我是大三。
```

在这里，不管你觉得女生的实际年龄如何，往小里说绝对没错。例如，你觉得她可能是25岁，你要说她21岁；你觉得她23岁，你要说她18岁。

接着可以说：

```
我也不是西安的，我家是陕北的，这个地方你可能没听过。我在西安工作已经有7年时间了。
```

这里算是一个测试窗口的测试点。如果你说完这句话，女生问你“你是在做什么工作的”或“你家是在陕北哪个地方”，就算正式从过渡阶段进入沟通阶段。真正的交流可能从这里才开始。

你依旧可以开个小玩笑：

```
我家在陕北的小县城，你可能没听过。我是做咨询工作的（或做互联网的）。白天在公司被老板各种各样折腾，晚上下了班之后在健身房被私教猛逼折腾。对了，我的老板也姓张，我现在满脑子都是张总。
```

女生可能会被这种笑话逗笑。到这个时候，你们应该有了5到8分钟的互动交流。

## 邀约与收尾

如果女生还没有主动提出离开，说明她可能没有太多别的安排。你可以尝试邀约她到旁边找个地方坐下来聊一聊，喝杯东西或咖啡，直接进入约会阶段。

如果你提出邀约后，女生委婉拒绝，比如“不用了，我还约了朋友，现在该走了”，此时我们应该掏出手机，提出留下对方的联系方式，比如微信号：

```
我们去加一下，后续有时间在微信上再交流。
```

女生通常会答应。之后在微信上聊天时，你会发现她的窗口或聊天上升程度很容易。因此，搭讪收号其实是收尾阶段要做的事情，而不是在搭讪开始时就要做。

## 搭讪目的误区

有的同学可能有一个误区，认为搭讪的目的就是为了收号。其实搭讪的目的是为了了解、沟通、建立吸引、二次邀约进行约会。很多同学也有体会，有时搭讪收了号，但在微信上根本无法进行有效聊天。原因在于搭讪时根本没有给对方留下太多印象，只是为了要号码而搭讪，而不是为了认识这个女生。这样的号码即使要过来了，也是废号。除非你的朋友圈建设得非常精彩、有吸引力，否则这些号码收回来没有任何意义。

## 搭讪四大阶段

搭讪分为四大阶段：开场阶段、过渡阶段、交流阶段、收尾阶段。那些10秒收号、30秒收号的，往往是在开场之后紧接着就开始收尾，相当于跑上去说“你好再见”。看似很厉害，其实没有任何难度。

## 搭讪评分标准

90分的搭讪是突破自我进行搭讪、和女生沟通、邀约女生并获取联系方式，完成这四个任务。

80分的搭讪是突破自我进行搭讪、和女生沟通、获取联系方式，完成这三个任务。

77分的搭讪是完成突破自我进行搭讪、获取联系方式这两个任务。

只完成突破自我进行搭讪这一个任务，只能算合格。值得肯定的是，毕竟战胜了自己的紧张、焦虑和恐惧，主动迈出了这一步。相信随着练习的增加，成绩也会越来越好。

## 常见错误

以下是搭讪中一些常见的错误：

*   在女生没有窗口时直接纠缠女生。
*   拦住女生的去路。
*   拍女生的肩膀。
*   靠女生太近。
*   不敢看女生。
*   不敢微笑。
*   说话声音太小。
*   以为女生不说话就没有窗口，过早放弃。

你一定要好好反思、总结这些错误。
