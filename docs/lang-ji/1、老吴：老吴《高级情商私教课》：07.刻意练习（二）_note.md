# 老吴《高级情商私教课》07：刻意练习（二）🚀

在本节课中，我们将要学习刻意练习的两个核心方法：**对标**与**举一反三**。我们将通过具体的思维模型案例，深入理解如何将这些抽象概念内化为直觉，并应用到生活和工作的不同场景中，从而提升我们的决策与思考能力。

---

## 刻意练习的本质回顾

上一节我们介绍了刻意练习的重要性，其核心目标是**将思维模型训练成直觉调用**。大脑的运作模式是先感性直觉，再理性直觉。由于我们装载的思维模型太少，感性直觉常常不准。因此，我们需要将思维模型装入大脑，并通过刻意练习将其转化为可靠的直觉。

那么，究竟如何在反思的场景下进行刻意练习呢？在实践中，我将其分为两个部分。

---

## 方法一：对标练习 🎯

第一个方法是**对标练习**。在日常生活中，我们学习了思维模型后，常常找不到使用的场景。例如，在《穷查理宝典》中，查理·芒格盛赞思维模型能解决很多问题。但初学者可能会觉得，书中那些高级的投资案例与自己的生活毫无关系。

这引出一个问题：那些优秀的投资者经常提到的“**复利**”模型，与我们普通人有什么关系？复利难道只适用于金融投资吗？

其实并非如此。问题不在于复利不起作用，而在于我们**看不到复利如何起作用**。解决这个问题的关键就是**对标**。

**对标**，即**把思维模型当作你思考问题的标准**。

我开始思考，复利如何在生活中运用？除了银行利息、高利贷的“利滚利”，还有什么？我意识到，复利的**核心是：做一件事A，其产生的结果B会反过来强化A本身**。这是一个**A→B→A**的强化循环。

符合这种循环的场景，就是复利场景。例如，我最近学习商业时，想到了亚马逊公司的“**增长飞轮**”模型。

亚马逊的增长飞轮是一个典型的复利模型：
*   **A** (极佳的用户体验) → **B** (用户满意并带来新用户) → **C** (更多用户和收入) → **强化A** (有更多资源提升体验)。

这个**A→B→C→A**的正向循环，让亚马逊越滚越大。亚马逊是2C（面向消费者）公司，一切以用户体验为核心。相比之下，淘宝/阿里巴巴是2B（面向商家）公司，其规则更倾向于保护商家利益。两者因核心用户不同，导致了不同的飞轮模式。

这个模型同样可以用于个人成长。以我自身为例，我的“增长飞轮”是：
*   **A** (用心服务学员，提供高价值内容) → **B** (学员满意度高，进行口碑推荐) → **C** (新学员加入，收入增加) → **强化A** (将新增收入投资于学习更前沿的知识，再反馈给学员)。

你看，**复利**和**增长飞轮**的思维模型，完全可以从商业领域迁移到个人发展领域。当我意识到这一点时，大脑仿佛被打通，产生了强烈的学习更多模型的动力，如反脆弱、边际效应等，并尝试将它们拿到生活中验证。

因此，刻意练习的第一个方法总结如下：
> **刻意地把思维模型当作思考问题的标准工具，主动寻找可以应用该模型的场景。**

这就是“对标”练习。但仅有对标还不够，我们需要通过反复练习将其内化为直觉。

## 方法二：举一反三练习 🔄

上一节我们介绍了对标练习，本节我们来看看如何通过反复练习来内化模型，这就是第二个方法：**举一反三**。

“举一反三”这个词很好地描述了这种方法的核心：**当你学习了一个思维模型后，要努力将其应用到至少三个完全不同的场景中**。如果你找不到三个场景，说明你还没有真正理解这个模型。

今天，我以心理学中的“**锚定效应**”模型为例，演示如何进行举一反三。

**锚定效应**是指，人们在决策时，容易受到最先获得的数字或信息（即“锚”）的影响，后续的判断会围绕这个“锚”进行调整。

以下是应用锚定效应的三个不同场景：

**场景一：商业定价**
*   **案例**：星巴克在店内售卖20多元一瓶的矿泉水。当咖啡价格在30元左右时，这瓶水就成为了一个“锚”，让顾客觉得“一瓶水都这么贵，不如买杯咖啡”，从而提升了咖啡的性价比感知。
*   **核心**：通过设置一个参照物（锚），影响你对主产品价值的判断。

**场景二：内容创作与变现**
*   **案例**：微信公众号赞赏功能默认提供几个固定金额选项（如5、20、50元）。大多数人会选择最低的5元。如果你将选项改为2、6、19、66、96、186元，那么6元就成了新的“最低锚”。假设有1000人打赏，平均打赏额从5元变为6元，总收入就提升了20%。
*   **核心**：通过调整选项设置，改变用户心中的价格“锚点”，引导其行为。

**场景三：求职谈判**
*   **案例1 (HR使用)**：HR对毕业生说：“公司标准薪资是5000元，但我特别欣赏你，可以为你申请到6000元。”这里先用5000元设定了一个较低的“锚”，让6000元的报价显得格外有诚意。
*   **案例2 (求职者使用)**：求职者说：“我专业排名前三，另外两位同学拿到的offer分别是15000元和12000元。我相信贵公司能给我一个公平的待遇。”这里用同学的高薪设置了“高锚”，从而提升HR对自己薪资的预期。
*   **核心**：在谈判中，主动设置一个有利于己方的“锚点”，影响对方的价值评估框架。

通过以上三个截然不同的场景（消费、产品设计、人际谈判），我们可以看到“锚定效应”这个思维模型的普适性。生活中类似的例子无处不在，比如真功夫快餐店常开在麦当劳、肯德基旁边，也是利用后者作为“锚”来凸显自己中式快餐的差异化价值。

掌握一个模型后，将其放在不同场景中不断应用、变换思考，才能逐步内化成直觉。这个过程需要投入时间，在生活和工作的具体场景中去思考、验证。

## 完整学习过程的视角

我们前面探讨了刻意练习的本质和方法。但必须认识到，一个**完整的学习过程是由输入、内化、输出三个环节**组成的。刻意练习主要属于**内化**环节。

然而，输入环节（如知识获取的质量）和输出环节（如实践与教授他人）会深刻影响内化的效率。如何将这三个环节高效匹配，从而提升整体学习效率，是我们下一部分将要探讨的内容。

## 本节课总结

在本节课中，我们一起学习了刻意练习的两个具体方法：
1.  **对标练习**：主动将思维模型作为思考标准，在万事万物中寻找其应用场景，打通知识与生活的链接。
2.  **举一反三练习**：深入学习一个模型后，强制自己将其应用于至少三个不同领域，通过跨场景的反复应用，将其内化为直觉反应。

思维模型的威力不在于知晓，而在于**高频、多场景的刻意应用**。当你发现一个模型能屡屡解释和优化你的生活时，那种“脑高潮”的体验将成为你持续学习的强大动力。
