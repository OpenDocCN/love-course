# 高级情商私教课 01：思维模型 🧠

在本节课中，我们将要学习思维模型的核心概念、价值以及如何发现、学习和应用它们。思维模型是提升决策质量和创新能力的底层工具，掌握它们将对我们后续的学习和生活产生深远影响。

## 第一部分：什么是思维模型？

上一节我们介绍了课程概述，本节中我们来看看思维模型到底是什么。

思维模型是经过广泛实践或科学验证的原理与规律。查理·芒格曾说，每个人都需要掌握至少100个思维模型，才能解决生活中大部分问题。思维模型主要有两个作用：
1.  **提高决策的质量和正确率**。
2.  **提升创新力**。

我们每天都在做决策，但很少思考决策的过程。心理学家发现，我们的决策并非简单的“刺激-反应”模式，而是 **ABC模式**：
*   **A** (Activating event)：诱发刺激（外部事件）。
*   **B** (Belief)：信念（我们对事件的看法和解读）。
*   **C** (Consequence)：行动结果（我们的行为和后果）。

公式表示为：**C = f(A, B)**

对于同一个刺激A，由于信念B不同，会产生完全不同的行动C。例如，被人打了一巴掌（A）：
*   如果信念B是“好汉不吃眼前亏”，行动C可能是忍耐。
*   如果信念B是“引刀成一快”，行动C可能是反击。
*   如果信念B是“把生活活成喜剧”，行动C可能是捂着脸开玩笑。

我们过去的经验和原生家庭塑造了我们的信念B，它是一把双刃剑。一方面，它让我们固守己见（“江山易改，本性难移”）；另一方面，当我们用更可靠的依据替换掉旧的、不靠谱的信念时，就实现了 **认知升级**。而认知升级的核心工具，就是思维模型。

**什么是“笨”？** 笨就是在类似情况下，总是采取相同行为，却期待不同结果。

思维模型是更可靠、更坚实的决策“套路”。例如，赞美他人时，一个有效的思维模型是 **FFC法则**：
*   **F** (Feeling)：感受。表达对方带给你的感受。
*   **F** (Fact)：事实。描述你观察到的具体事实。
*   **C** (Compare)：比较。加入一个恰当的对比。

```python
# FFC赞美法示例
feeling = “你的打扮让我眼前一亮。”
fact = “你的绿色衣服、项链和耳环搭配得特别和谐。”
compare = “你是今天在场最亮眼的一位。”
compliment = feeling + fact + compare
print(compliment)
```

使用FFC法则的赞美，远比简单说“你真漂亮”更真诚、更有效。这就是学习思维模型的价值：用更好的“套路”替代有限的经验。

## 第二部分：如何发现并学习思维模型？

上一节我们了解了思维模型的定义和价值，本节中我们来看看如何在生活中发现和学习它们。

思维模型是分层次的：
1.  **解决具体问题的模型**：如FFC、SWOT分析、AARRR模型。工具属性强，即学即用。
2.  **学科原理/规律**：如经济学中的边际效应、市场营销中的4P理论。解决某一类问题。
3.  **跨学科原理/规律**：如反脆弱、熵增定律、进化论。解释属性强，更为抽象。

很多人喜欢学习第一类即学即用的技巧，但**创新力往往来源于那些看似“无用”的抽象模型**。创新是“生产要素的重组”，关键在于**重构我们看世界的方式**。掌握更多理解世界的底层模型和视角，才能打破旧有关系，进行要素重组，产生创新。

那么，如何从生活中发现思维模型呢？这里介绍一个方法：**“问思模”法**。
*   **问** (Question)：遇到有启发的现象时，追问它解决了什么**问题**。
*   **思** (Thinking)：探究解决这个问题的**思路**和思考过程。
*   **模** (Model)：将这个思路联系到更底层的思维**模型**。

例如，观察到一位老师从不冷场。通过“问思模”法分析：
1.  **问题**：如何避免尬聊？
2.  **思路**：老师通过“请教对方擅长领域的问题”来打开话匣子。这转换了视角，从“我该说什么”变为“如何让双方和信息流动起来”。
3.  **模型**：这个思路底层关联着**系统思维**（沟通是我、对方、信息三要素的系统）和**心理学**（人人以自我为中心，请教满足了对方的自尊心和分享欲）。

要从现象中学到模型，需要三个转变：
1.  **心态转变**：保持好奇心，主动追问“为什么”。
2.  **重点转变**：学习重点从记忆答案，转向理解**思考过程**。
3.  **方法转变**：不停留在具体答案，而是建立**知识之间的联系**。

心理学家桑代克认为，**学习就是联结**。高手会将模型与模型之间建立联结。查理·芒格也从“栅格理论”（知识分格存放）发展到强调模型联系的“多元思维模型”。对于一个系统而言，**结构决定性质，而非要素本身**。例如，碳原子既可以排列成柔软的石墨，也可以排列成坚硬的钻石，区别在于原子间的连接结构。

除了自我挖掘，向高手学习是重要渠道。如何高效学习？推荐 **“五星笔记法”** 。每次学习，尝试回答五个问题：

以下是五星笔记法的五个核心问题：
1.  这节课的核心内容是什么？
2.  老师讲述的逻辑是什么？
3.  我有哪些疑问和启发？
4.  这些疑问和启发背后的思维模型是什么？
5.  有哪些我可以立刻行动的事情？

这种方法强迫你关注老师的思考路径，而非单纯的知识点。就像经济学家张五常，反复听同一位老师的课，不是为了记知识，而是为了学习其讲课思路。

## 第三部分：如何应用思维模型？

上一节我们探讨了如何发现和学习思维模型，本节中我们来看看如何将它们应用到实际生活中。

对多元思维模型的应用，关键在于 **多层次** 和 **多视角**。
*   **多层次**：用不同层次的模型分析同一个问题。
*   **多视角**：在同一层次中，用不同角度的模型分析问题。

**1. 多层次分析示例：中国足球为何进不了世界杯？**
*   **道德层次**：球员不敬业，生活作风问题。
*   **商业/组织层次**：联赛机制、管理体系有问题。
*   **数学/系统层次**：足球是11个人的点线面互动系统。最高效的进攻是“去中心化的三角网络”（如巴萨的Tiki-Taka打法），这涉及最优概率的数学问题。

世界是分层次的（如物理学->化学->生物学->心理学->社会学）。尽管高层知识可还原为底层原理，但许多特性（如水的流动性）只在高层涌现。因此，我们需要多层次理解。

**2. 多视角分析示例：拼多多为何崛起？**
产品经理梁宁曾从微、中、宏三个视角分析：
*   **微观视角（用户体感）**：精准拿捏人性“贪便宜”的心理，运营策略奏效。
*   **中观视角（产业/竞争）**：
    *   **供给侧视角**：淘宝打假，挤出大量低端供应商，为拼多多提供了商家资源。
    *   **用户端视角**：智能手机普及和移动支付、物流等基础设施成熟，降低了用户和平台成本。
    *   **竞争者视角**：淘宝、京东品牌升级，无暇顾及低端市场，留出生态位。
*   **宏观视角（时代/趋势）**：第二曲线、价值网络等（此处不展开）。

如果只用单一视角（如“就是靠微信砍价”），认知就会片面。世界是复杂的系统，每个层次都有不同的因果。**多元思维模型能帮助我们整合视角，最大限度地接近真相，从而做出更优决策。**

## 总结与学习技巧 🎯

本节课中我们一起学习了：
1.  **思维模型是什么**：可靠的决策套路和认知升级工具，能提升决策质量和创新力。
2.  **如何发现与学习**：通过“问思模”法从生活中提炼，用“五星笔记法”向高手学习，核心是关注思考过程并建立知识联结。
3.  **如何应用**：运用“多层次”和“多视角”分析问题，整合不同模型以更全面理解复杂世界。

最后，分享三个立即可用的学习技巧：

以下是三个高效学习技巧：
1.  **笔记重在整理自己的思路**：做笔记或思维导图时，重点不是复制老师讲的内容，而是整理你的疑问、启发，并将其与自身经验结合。
2.  **主动收集启发点与疑问点**：听课中遇到启发或疑问，立刻用简单方式（如记关键词、打标记）记录下来。课后回看时暂停、思考、整理，务必搞懂再继续。
3.  **采用“记忆生成”法**：思考知识将来怎么用，就怎么去记它。例如，在停车场停车后，拍照记车位号，并回头观察路径特征，这样“找车”这个应用场景就决定了你的记忆方式。
