# 老吴承情速成课《女人倒追》第二十四课：约会原则 🎯

在本节课中，我们将要学习约会的基本原则。很多人在约会时不清楚状况，不明白约会的主基调是什么，也不知道在约会过程中应该做什么。我们将首先梳理约会的大框架，因为如果连大框架都不明白，就失去了良好的约会基础，更无法谈论如何在现场表现和把事情做得更好。只有在方向正确的基础上，我们才能讨论其他细节。

## 一、时间原则 ⏰

上一节我们介绍了课程概述，本节中我们来看看第一个核心原则：时间原则。这个原则源于学员经常提出的一个问题：如何让约会对象接受短期关系并同意开房。

在咨询过程中，我会首先询问两件事情。

以下是需要了解的两个关键点：

1.  **线上聊天基础**：询问学员与姑娘在微信上的聊天情况。是简单聊了几句就约出来，还是聊了很久并有时间跨度？如果是后者，还要区分聊的是走心的内容还是走肾的内容。如果聊了走心的内容，姑娘出来约会的目的很可能是寻找男朋友。如果她不接受上床、气氛不对，那是理所当然的。你的期望必须与一开始构建的大方向一致，否则会很麻烦，需要在约会现场改变方向。
2.  **约会具体时间**：询问学员约会（如喝咖啡、看电影、吃饭）的具体时间点。学员通常会回答是下午或晚上。

基于此，我提出了一个明确的细分原则：以晚上**7点**为界。

*   **7点之前**：绝大多数城市在这个时间已经吃完晚饭。约在7点之前（例如下午3点喝咖啡）更偏向于普通朋友社交、谈事情、交流，或者是第一次见面、相亲、朋友聚会。
*   **7点之后**：更偏向于男女关系或短期关系。

因此，给学员的建议是：

*   如果约会的姑娘颜值、家境、性格等各方面都不错（属于A组），建议将时间定在**7点以前**。因为这类姑娘价值可能比你高，你需要采取长期策略，不暴露需求感，慢慢发展。这类似于一个低分对象追求你时，需要小心翼翼、提供价值（如有趣的聊天、技巧性邀约），你才可能考虑见面。对于高分妹，她希望获得男人提供更多价值（真心、套路、技巧），而不是直接提出短期要求。将邀约定在7点前，是向对方传递走长期关系的信号。
*   反之，如果目标是短期关系（炮友、一夜情、暧昧），则应将邀约时间定在**7点之后**。夜晚的氛围（昏暗、暧昧）更有利于关系升高。例如，7点见面，8点看电影，10-11点电影结束，此时公共交通可能已停运，为“不方便回家”提供了借口，更容易带对方离开。

**核心公式**：
`长期关系 / 高分对象 => 邀约时间 < 19:00`
`短期关系 / 低分对象 => 邀约时间 > 19:00`

## 二、地点原则 📍

刚才我们讨论了时间原则，接下来看看与之紧密相关的地点原则。约会地点同样向对方传递着强烈的信号。

以下是不同类型关系对应的典型地点：

*   **长期关系导向**：咖啡店、逛街、饭店。**电影院**是一个相对中性、承上启下的场所，介于朋友社交和暧昧之间，是容易升高关系的场合。
*   **短期关系导向**：酒吧、清吧、夜店、私人电影院、夜宵。这些地点本身就带有暧昧气氛。

**核心逻辑**：从你们认识方式或第一次见面的地点，很大程度上已经决定了关系的走向。例如，在酒吧认识的姑娘，就不要抱有过多的长期幻想，那是在搏一个很小的概率。我的意见是，尽量不要去搏那个很小的概率。

接下来，我们逐一分析几个主要约会场景的操作要点。

### 1. 咖啡店操作要点

咖啡店适合相亲、家人见面或抱着长期专一目的交往。

**操作流程**：
1.  **初期寒暄**：前15分钟，可以坐在对面，通过一两个话题热络气氛。
2.  **位置转换**：气氛打开后，尽量找借口坐到对方身边（例如，“我给你看我手机上一张有趣的照片”）。并排坐比面对面压力小，更容易产生亲密感和进行肢体接触。
3.  **控制时长**：建议在咖啡店不要超过45分钟，以免后期无话可聊。之后可以转场去逛街或吃饭。

### 2. 逛街操作要点

逛街非常适合新手，可以有效避免冷场尴尬。

**优势与技巧**：
*   **话题源泉**：沿途店铺（包子铺、餐厅、服装店、书店）可以提供源源不断的话题。
*   **测试兴趣**：可以故意在某些店铺（如书店）停留，观察对方是否会主动跟随或在你附近等待，以此测试对方对你的兴趣程度。
*   **观察肢体语言**：在空旷街道行走时，可以逐渐向外侧偏移，观察对方是否会调整步伐靠近你，以保持两人作为一个整体的距离。这能反映对方对你的亲近感。

### 3. 饭店操作要点

吃饭在约会中收益较低（花钱多、吵闹、不便进挪），有时是下策。

**关键技巧——点单**：
*   不要直接把菜单推给姑娘说“你来点吧，别客气”，这显得屌丝。
*   正确做法：把菜单递给对方，问“你想吃点什么？”。如果对方推辞，可以说“不用跟我客气，想吃啥就点啥”。如果对方再次推托，可以调侃“怎么，我选的餐厅你不满意？”，通过对方的反应判断其在意你的程度。
*   点单前询问有无忌口，也是一种关心和测试。

### 4. 电影院操作要点

电影院是升高关系的重要场所。

**操作流程**：
1.  **选片**：优先选择恐怖片（最能调动情绪）或爱情片。
2.  **入场**：切记自己走在前面，为女生带来安全感，并预先看好座位号。
3.  **座位技巧**：选择中间扶手能抬起的影厅。在找到座位、坐下前的瞬间，迅速将扶手抬起，营造“天生就该如此”的自然感。
4.  **道具使用**：爆米花是极佳的进挪道具。（具体爆米花进挪技巧可添加微信获取）
5.  **观影行为**：大部分时间应沉浸在电影中。可在情绪点（笑点、恐怖点）与对方简短交流一兩句，或用眼神互动。避免长时间交头接耳。

### 5. 短期关系场所要点

*   **清吧/夜宵**：能量场合中等，适合喝至微醺、升高情绪，但离最终TD还有距离。
*   **酒吧/夜店/KTV**：偏向短期属性。（更多内容在夜场课程中详述）
*   **私人电影院**：一站式TD场所，成功率极高。

**私人电影院必备与操作**：
*   **必备物品**：安全套、纸巾、两瓶矿泉水（用于解渴或必要清洁）。
*   **操作节奏**：进入后至少有半小时缓冲时间，不要立即动手。可以先共用枕头、依偎观看。随后逐步进行暧昧举动（如头靠头、闻耳朵/头发），进而舌吻。手部动作应从腹部滑向上方，连胸罩一起握住，而不是直接伸入。感觉到位后，即可进入正题。

## 三、预期原则 💬

在约会前，双方都会有预期。根据对象不同，预期策略也不同。

*   **面对高分/长期对象**：**少预期、轻预期或不预期**。不要直接提出过多要求（如穿着要求），否则对方可能觉得你分数低还要求多。
*   **面对价值低于你的对象**：可以给予预期，主要分为两种：
    *   **情绪预期**：描述见面后可能产生的感觉和状态。例如，“万一见了面觉得你太可爱，忍不住想亲你怎么办？”这更像是撩拨和情趣。
    *   **行为预期**：直接提出具体行为要求。例如，“希望你穿着丝袜来见我。”
    *   **建议比例**：即使与炮友聊骚，也应以**情绪预期为主**（约占3/4），**行为预期为辅**（约占1/4）。过多行为预期会显得需求感过强，像“挺着鸡巴的屌丝”。

## 四、进挪原则 🤝

进挪（肢体接触）原则建立在之前的原则基础上。

*   **长期关系定位**：最保守且不错的策略是 **“第一次不进挪，第二次必须进挪”**。如果女生愿意见第二次，说明你已基本过关，只要稳扎稳打即可。
*   **短期关系定位**：更偏向“见风使舵”。可以遵循 **“前场看窗口，中场撩情绪，后场看反应”** 的原则。
    *   **前场**：判断女生对你的兴趣窗口。
    *   **中场**：主动进攻，说暧昧情话，把情绪撩起来。
    *   **后场**：进行筛选测试（如“明早要不要早起？”“和谁一起住？”），判断能否带走。

（注：进挪是大学问，涉及大量肢体语言细节，更适合线下教学或图文展示，此处不过多赘述。）

## 五、约会设计 🗺️

最后，也是极其重要的一环：约会流程设计。这关系到整个约会的逻辑结构是否正确。

一个完整的约会流程应包含三个层次的地点设计：

1.  **常规约会场所**：如商业街区（咖啡店、逛街）。
2.  **升高关系场所**：如电影院、清吧、KTV。
3.  **私密空间**：如酒店、家中。

**设计关键**：你需要提前规划，并考量地点间的**距离**。
*   第一场（常规）与第二场（升高）的距离是否过远？过远会打断节奏。
*   第二场（升高）与第三场（私密）的距离是否过远？找不到酒店是极其糟糕的体验。

**建议**：在你常驻的城市，开发出一套固定的、可靠的约会路线。结合一些低成本但浪漫的地点（如海边），让约会显得多元化。女生很在意男生是否安排好整个约会流程，如果做得不好，她会觉得这个男人“不行”。
