在本节课中，我们将要学习如何通过手机修图软件，快速提升照片质感，打造出类似网红的展示面。课程将详细介绍老吴常用的九款软件及其核心功能，并通过实际案例，手把手教你从磨皮、调整脸型身材到添加滤镜的完整修图流程。

## 软件清单与功能介绍

上一节我们介绍了课程的整体目标，本节中我们来看看具体需要使用哪些工具。以下是老吴最常使用的九款修图软件及其主要用途：

1.  **Facetune**：主要用于处理皮肤（磨皮）、调整脸型和五官大小。
2.  **美图秀秀**：在Facetune处理的基础上进行微调，例如瘦脸瘦身、面部重塑、拉长腿部。
3.  **美颜相机**：拍摄人像时使用，拍出的照片基础较好，可减少后期工作量。
4.  **Snapseed**：主要用于精细调整各项图片参数，操作有一定难度。
5.  **Instagram**：核心软件，用于添加高质量滤镜，是打造网红感照片的关键。
6.  **美妆相机**：专门用于修饰脸部妆容的软件。
7.  **天行VPN / SuperVPN**：用于翻墙，以便注册和使用Instagram。
8.  **VUE**：专门用于拍摄和编辑短视频的软件。
9.  **Layout**：用于图片拼接（拼图）。

其中，**Instagram** 是最核心的软件，而 **Facetune** 和 **美图秀秀** 是本次课程重点讲解的基础修图工具。

## 核心修图流程详解

了解了工具后，我们进入实战环节。本节将通过一张街拍照片的案例，详细拆解从原图到成片的每一步操作。

### 第一步：使用Facetune进行基础处理

首先，我们需要对皮肤和脸型进行基础美化。以下是具体操作步骤：

*   **磨皮（平滑功能）**：使用“平滑”功能处理皮肤瑕疵，如斑点、痘印或粗大毛孔。如果问题较严重，可以使用“更加平滑”功能。
*   **调整脸型与头部大小**：在菜单栏找到“调整”功能。通常先使用“调整大小”整体缩小脸部，再使用“调整”功能对局部进行微调，例如将头发向中间拉扯，可以使头部视觉上变小。

**核心操作**：`平滑 -> 调整大小 -> 局部调整`

### 第二步：使用美图秀秀优化身材比例

基础皮肤和脸型处理好之后，接下来优化身材比例，让照片更具视觉吸引力。以下是美图秀秀中的关键功能：

*   **面部重塑**：对面部细节进行进一步调整。
*   **瘦脸瘦身**：用于微调身体轮廓。
*   **去痘去斑**：补充处理Facetune未完全消除的瑕疵。
*   **增高（拉长）**：这是改变比例的关键。将图片底部的定位点向下拉以拉长腿部，同时将上身适当缩短，可以营造出“腿长身短”的优越身材比例。

**核心技巧**：`拉长腿部 + 缩短上身 = 优化身材比例`

### 第三步：使用Instagram添加滤镜

完成形体调整后，最后一步是为照片注入灵魂——添加滤镜。以下是使用Instagram滤镜的核心要点：

1.  **翻墙与注册**：首次使用Instagram需要借助VPN（如天行VPN）翻墙注册账号。注册成功后，日常修图可断开VPN，避免图片自动上传。
2.  **选择滤镜**：导入图片后，下方会显示一系列滤镜。网红常用的有 **Hudson**、**Amaro**、**Sierra** 等。
3.  **调整滤镜强度**：**这是最关键的一步**。选择滤镜后，**务必再次点击该滤镜图标**，会出现一个强度滑块（0-100）。不要直接使用100%强度。
4.  **设置强度参数**：将滤镜强度调整到 **50% 到 70%** 之间（例如61%）。这样既能提升照片质感，又不会留下过重的修图痕迹，显得自然。

**核心公式**：`网红感照片 = 基础修图 + 滤镜强度(50%~70%)`

## 不同场景的修图实例

掌握了通用流程后，我们来看看如何将其应用到不同场景中。

### 案例一：风景与人像照（吉隆坡独立广场）

这张案例展示了如何拯救一张“游客照”式的原图。
*   **问题分析**：原图因光线强烈，脸部较暗，构图杂乱，姿势随意。
*   **修图步骤**：
    1.  使用Facetune和美图秀秀提亮脸部、缩小头型、拉长腿部。
    2.  使用美图秀秀的“编辑”功能，裁剪掉画面中多余杂乱的部分。
    3.  使用Instagram的 **Hudson** 滤镜，并调整强度至60%左右，让整体画面变得明亮、有质感。
*   **对比效果**：通过裁剪重构画面焦点，通过滤镜统一并提升色调，最终照片从普通的打卡照变成了有范儿的展示面。

### 案例二：静物与美食照（糖果/马卡龙）

静物照的修图逻辑与人像类似，但更注重色彩和氛围。
*   **修图步骤**：
    1.  直接使用Instagram添加滤镜。
    2.  尝试 **Clarendon** 或 **Amaro** 等滤镜。
    3.  同样将滤镜强度控制在70%左右。
*   **核心要点**：即使是原本摆放整齐的原图，通过一个合适强度的滤镜，也能大幅提升视觉美感和“食欲”，让照片看起来更精致、有生活品味。

## 网红的审美与标准

为了让大家建立正确的审美目标，我们分析了网红照片的共同特点。
*   **干净明亮**：照片整体色调倾向于明亮、干净，给人以舒适、高级的感觉。
*   **生活化与美感结合**：照片内容多是生活场景（咖啡、旅行、美食），但通过构图和修图呈现出超越日常的美感。
*   **自然不夸张**：修图的最高境界是“修了像没修”，所有调整都恰到好处，不虚假。
*   **设备建议**：为了保证颜色显示准确，避免安卓手机可能出现的色差问题，建议使用苹果手机进行修图和发布。二手苹果手机即可。

## 课程总结与作业

本节课中我们一起学习了从工具准备到成片输出的完整网红修图流程。
*   **核心流程回顾**：`Facetune基础处理（磨皮、调脸）-> 美图秀秀比例优化（拉腿、瘦身）-> Instagram滤镜加持（强度50%-70%）`。
*   **核心心法**：修图旨在“锦上添花”，而非“改头换面”。保持自然、干净是男生修图的第一要义。
