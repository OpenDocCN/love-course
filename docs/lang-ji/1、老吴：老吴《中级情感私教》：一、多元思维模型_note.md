# 1、老吴：老吴《中级情感私教》：一、多元思维模型_note

在本节课中，我们将要学习什么是思维模型，为什么它对我们的生活、决策和情感沟通至关重要，以及如何发现、学习和应用多元思维模型。

## 概述

思维模型是我们理解和应对世界复杂性的重要工具。它不仅能提升我们决策的质量与正确率，还能培养我们的创新能力。本节课将分为三个部分：首先，我们将探讨思维模型的定义与核心作用；其次，我们将学习如何在生活中发现和学习思维模型；最后，我们将了解如何应用多元思维模型来多视角、多层次地分析问题。

## 第一部分：什么是思维模型及其作用

很多人知道我擅长聊天。这本质上是因为我的大脑里装载了较多的思维模型，使我更懂得如何更好地回应他人。这背后有一套自己的聊天思维模型。

巴菲特的搭档查理·芒格曾说，每个人在一生中至少需要掌握一百来个思维模型，才能解决生活中百分之八九十的问题。因此，思维模型对我们非常重要。

那么，思维模型究竟是什么？查理·芒格认为，思维模型是重要学科的重要原理。在学习不同课程时，我们会接触到许多思维模型，例如商业课程中的反脆弱、低弱代偿、双S曲线、第一性原理等。

你可能会想，我是来学习情感的，为什么要讲这些？学了有什么用？在生活中如何使用？这正是我们今天第一节课要讨论的核心问题：为思维模型“去昧”，帮助我们理解所有这些思维模型究竟如何帮我们在生活中解决问题。

在我看来，思维模型有两个比较重要的作用。

### 作用一：提升决策质量和正确率

做决定是我们生活中每一刻都在做的事情。生活本身就是一个又一个决定的延续。如果我们在选择伴侣时能做出更好的决定，在工作时能选择更好的方向，这对我们的人生影响巨大。

然而，人很少仔细思考自己是如何做决策的。这个过程可以理解为：当你看到女生对你说一句话，你会如何回复她？这就是一个做决策的过程。

心理学家发现，在外部刺激和做出反应之间，并非简单的“刺激-行为”模式，而是一个 **ABC模式**。

*   **A (Activity/Events)**：诱发刺激。
*   **B (Belief)**：信念反应。
*   **C (Consequences)**：行动结果。

对于同一个刺激A，由于背后的信念系统B不同，我们就会做出不同的行动结果C。

例如，有人打了你一个耳光。结果，有的人会含泪不哭，因为他的信念B是“好汉不吃眼前亏，君子报仇十年不晚”。有的人会直接反击，因为他的信念B是“引刀成一快，不负少年头”。还有的人，左脸被打，却捂住右脸，因为他相信“生活再糟糕，我也要把它活成喜剧”。

因此，因为打耳光这个刺激行为A，由于每个人的信念系统B不同，会做出不同的行动结果C。

运用到情感领域，这意味着当我们和女生聊天时，女生回了一句话，这就是她的行动结果C。很多人想都不想，直接根据这个行动结果C就回复了，压根没有去思考女生的信念反应B是什么。更少人会去思考，为什么女生会有这种信念，从而去探寻本质源头A。

就像我在公开课里讲过的例子：一个女生两次问我是不是双鱼座。她的行动C是很想知道我是不是双鱼座。我的信念反应B猜想是：这个女生可能被双鱼座伤害过。那么我就在想，她可能受过一些刺激A。只要你能够找到这个A，问题就能够从根本上解决。

其实我们跟别人聊天，只需要找到他的A，就知道怎么处理了。就好像上楼梯，如果找不到电梯，就只能爬楼梯。如果连楼梯都找不到，就上不去了。

每个人在面对同样的事情时，会有不同的反应。差距就在于我们背后决策的信念。由于过往所有的经历、经验，甚至原生家庭的影响，我们每个人大脑中都会形成并使用一些决策依据。

但这把双刃剑。一方面，它让我们一旦形成一个信念后就很难改变，这就是所谓的“江山易改，本性难移”。我们常说别人“笨”。什么是笨？笨就是遇到类似的情况，每次采取同样的行为，却还期待有不同的结果。

另一方面，这把双刃剑也有好处：一旦你做决策的依据发生了彻底改变，那你真的就是洗心革面、重新做人了。这一点通常我们叫做 **认知升级**。就是面对同一个情况，你有了不同的认知方式，能做出完全不同的决策。

例如，在你的潜意识里，你觉得被打就一定要打回去。如果我告诉你，被人打千万不要打回去，应该怎样怎样，并把权衡利弊说清楚，你的认知就会升级。所以你面对“被人打”这个情况，除了原有的处理方式，又有了一个新的处理方式。

在聊天上也一样。平时你看到某句话，都用同样的方式回复。但通过学习之后，每一句话你都有三四个不同类型的回复方式。这时，做决策的依据就变多了。

我们成长进步的过程，就是用更好的做决策依据，来替换掉自己原有的那些有限的经验，以及过往学到的不太靠谱的决策依据，比如一些很套路的聊天、话术、生搬硬套的东西。从今天开始，我们就要开始替换它们，用好的东西覆盖掉不好的东西。这个替换的过程就叫做 **认知升级**。

而认知升级的工具，就是我们的 **思维模型**。思维模型不一样，它是经过广泛实践证明可靠的，或者是科学研究验证过的，又或者是不证自明的原理规律。用这些更可靠、更坚实的原理来做决策，我们做决策的质量自然大不相同。换句话说，我们做决策时用的“套路”质量不一样，做出的决策自然就不一样。

### 作用二：提升创新能力

创新是你想跟别人与众不同的重要目的。但如何通过学习思维模型来提升创新能力？感觉好像没什么关系。这是因为思维模型是分层次的。

例如，我刚刚讲的FFC模型，是解决具体问题（如何赞美他人）的思维模型。像我们听过的SWOT分析、冰山模型，以及做运营的同学都知道的AARRR模型，这些都是解决具体问题的思维模型。

还有一类思维模型，适用范围更广，是某一学科的原理或规律。例如，经济学里的边际效益、价值投资理念，市场营销学里的4P原理。这些是属于解决该学科领域问题的思维模型。

当然，还有更底层的思维模型，那就是更抽象的跨学科的原理和规律。例如，哲学的第一性原理、物理学的机械论、生物学的进化论、物理学的熵增定律等。这些最底层、很抽象的东西，非常有用。

我们可以看到，思维模型像一个金字塔。越往上，它的解释属性越强；越往下，它的工具属性越强，是解决某个具体场景的问题。

有用的思维模型可能很多人都喜欢去学习，因为拿来就能用。就好像外面很多讲情感的课程，会告诉你很多技巧，大家都喜欢这些，因为拿来马上就能用。但你会发现，随着学习深入，你要懂得它的原理和规律才有用。那些所谓的技巧，有些很快就会过时，有些人家已经免疫了。当大家都在用同一种“武功”时，女生会对你有什么感觉吗？没有。

所以我们更要学习的，是一些顶端的、抽象的、好像没什么直接用处的思维模型。这跟我们现阶段的国情有关。中国发展越来越快，很多行业已经超越了美国。我们本来只是一个跟随者，慢慢转型到一个引领者的角色。

过去，我们可以按照前人已经探索好的方向，市场直接跟着做就行了。比如几年前学的情感课程，直接拿来就能用，你能够享受到增长的红利。但现在，懂这些的人越来越多，已经没有红利了。这时怎么办？必须创新。

所以为什么我要在此时选择“复出”？就是我希望教给大家的，除了能解决具体问题的方法，还要让大家懂得这些背后的原理、规律、逻辑。大家懂得这个东西之后，你想怎么聊，随便你。

因此，我们相当于走在情感领域很前沿的位置。因为我纵观整个市场，就没人教这一块的东西。

我再给你们分享一个很厉害的人，他叫熊彼特。他说创新来自哪里？创新是 **生产要素的重组**。生产要素要怎么样才能重组？关键在于我们要 **重构** 我们看世界的方式。我们只有掌握了更多理解世界的方式、更多解释世界的方式，才能够打破我们过去经验世界里各要素的关系，用新的视角来看待各要素之间的关系，进而重组产生创新。

什么意思呢？举个例子。我们以前在看待两性关系时，是很直接的。但现在我要教你们能够多视角、多层次地看待同一个问题，你就会发现一些很不一样的切入点，你就能够去创新，就能够跟别的男生不一样。当你跟其他男人不一样的时候，自然就会有魅力。有魅力就有吸引力，有吸引力，别人就会来追求你。

这也就是为什么我会不断学习新内容，并在这套课程里跟你们讲这些。为的就是让你们能够学到一些更底层、更基础、能用不同理解世界的思维模型和视角，这是帮助我们真正培养和训练创新能力的基石。

## 第二部分：如何发现并学习思维模型

前面我们讲了学习思维模型非常重要。查理·芒格在他的《穷查理宝典》里说，一个人一辈子要学上百个思维模型。他自己更狠，90多岁时还在阅读和学习。有人会问，《穷查理宝典》中只写了十几个思维模型，其他的去哪里学？

其实我们生活中有很多场合都可以拿来学习和发现思维模型，只不过我们没有方法就不知道罢了。

### 方法一：问思模

如何在生活中发现思维模型？这里有一个方法叫做 **问思模**。它是三个词的缩写：**问题**、**思路**、**模型**。

*   **问题**：我们在生活中经常会遇到一些让你觉得特别有启发的事情。当我们知道一个启发的小技巧时，往往不去追问这个启发背后到底在解决什么问题。
*   **思路**：当我们追问了问题之后，就进入第二个环节“思”。我们要思考对方解决问题的思路是什么，他看待问题的视角与我们有何不同。
*   **模型**：我们要再追问这个思路，和更底层、更基础的思维模型有没有关联。把具体的技巧和思考过程，总结归纳为更通用的模型。

例如，我认识一位老师，和他在一起从不冷场。我发现他的技巧是：当没话题时，他会说：“我知道你在XX领域很擅长，刚好我有个问题想请教你。”对方打开话匣子，气氛就融洽了。

如果只学技巧，到这里就结束了。但要学习思维模型，这远远不够。我们要追问背后的思路：这位老师思考问题的视角，其实是一个转换角色、转换视角的过程。把我们从单一的“我”出发，变成了由“我”、“对方”和“交流信息”三方面要素构成的系统视角。

进一步，我们要追问这个思路底层的模型是什么。这可能与 **系统思维**（增加系统维度，解决问题的角度就增加）和 **心理学**（每个人以自我为中心，请教他人能满足对方的自尊心和分享心态）有关。

所以，看似简单的交流不冷场，背后其实有很多底层的思维模型。我们从生活现象找到答案，从答案归纳到产生答案的思维过程，最后找到背后的科学原理。

### 方法二：向更厉害的人学习（五星笔记法）

除了自己在生活中挖掘模型，还有一个现成且非常重要的渠道：向比你更厉害的人学习。

怎么能在学习中更高效呢？很多人上课会分点记下老师讲的知识点、重点和思维模型。这样有用吗？有点用，但重点不在这里。因为我们学这些思维模型、思路，是学习解决问题的过程。

记笔记的方法，我分享一个 **五星笔记法**。每次记笔记时画五颗星星，每颗星星后面都有一个问题，一共五个问题。上课就是回答这五个问题，回答完了，笔记也就做完了。

以下是五个问题：

1.  这堂课/这次分享，老师要讲的核心内容是什么？
2.  老师讲述的逻辑是什么？
3.  我受到的启发和产生的疑问点是什么？
4.  这些启发和疑问背后的思维模型是什么？
5.  有哪些事情可以让我回来就马上开始采取行动？

例如，第一个问题“核心内容是什么”，我根本不是听老师上课讲什么，而是讲完后我要自己总结。你可以提前预习，了解这节课大概要学什么，解决什么问题。然后去听老师上课，看看老师讲的跟你理解的有何差别。

重要的不是老师讲的那些知识点，知识点满大街都有。重要的是学习老师组织和推演这些知识之间关系的思考过程。在后面我们的案例分析中，你就可以看到我是如何一步一步去分析和拆解的。

华人著名经济学家张五常，上大学时很喜欢一位经济学老师，他把这位老师的同一门课听了六次。老师很奇怪，问他是不是记不住。张五常说，老师讲的知识点他早就知道了，他要学的是老师讲课的思路。张五常的学习方法恰恰是关注解决问题、思考路径的过程。

第三个点“疑问和启发”，在听课时要迅速列出来。第四个点，去思考背后是怎么回事。我在学习时，会做大量的笔记和整理，把老师的知识逻辑、观点、疑问全部整理出来，并思考如何与已有知识联系。

心理学家桑代克（Edward Thorndike）是联结主义的建立者。他在《教育心理学》中说：**学习就是联结**。人之所以善于学习，主要是因为他能够形成大量的联结。学习会使人成为异常复杂而精致的联结系统。

在刚才的例子中，学到的知识不仅是一个全新的知识，它还把我们过去学到的心理学、系统学、视角的知识，和其他所有熟悉的知识进行了联系。而这种联系，就是模型和模型之间的联结。这种联结使得真正的学习成为可能。

桑代克认为，学习之所以有助于另一个学习，是因为这两种学习具有相同因素。这就是桑代克的“相同要素说”。

真正的高手，会把模型和模型之间也建立联结。查理·芒格一直提的是 **格栅理论**（Latticework of Mental Models）。我们后来聊的都是 **多元思维模型**。多元思维模型更重要的就是强调模型和模型之间的联系。

对于一个系统而言，**结构远远大于要素本身**。系统的性质取决于要素的结构，而不是要素本身。什么意思？例如，碳原子既可以排列成钻石，也可以排列成石墨。这并非取决于碳元素本身，而是它们之间相互形成的系统结构。

同样，就算你把查理·芒格所有的思维模型都学会了，但如果你不知道在应用时它们的关系、结构、权重（先使用什么，后使用什么，哪些之间会有冲突），你也成不了很厉害的人。因为你只是知道了很多道理，但不懂如何组合、串联起来。相当于你只是装了很多东西，但没有用处。

## 第三部分：如何运用多元思维模型

前面我们讲到了思维模型很重要，我们有很多方式和方法来学习并掌握。但在生活中，我们怎么使用呢？

在我看来，多元思维模型应用的关键在于：**多层级、多视角**。

*   **多层级**：你可以把一个问题，用不同层级的思维模型来进行分析。
*   **多视角**：你在同一层级当中，用不同角度的思维模型来分析问题。

### 多层级分析

例如，分析“为什么中国足球队进不了世界杯”。

*   **道德/个人层面**：有人从人的道德层面分析，认为中国球员拿着高薪却泡夜店、不认真踢球，道德有问题。
*   **商业/体系层面**：有人从商业层面分析，认为问题出在联赛机制、体系上，是过分市场化或体系不健全。
*   **数学/效率层面**：有位老师提出，是因为中国足球队“数学没学好”。足球比赛本质上是11个人的点线互动，用最高的进攻效率把球踢进对方球门的一种数学运算。数学家研究发现，采用去中心化的三角形网络进攻方式效率最高，这正是欧洲足球（如巴塞罗那队）的打法。

你看，简单一件事，我们分为了道德、商业、数学三个不同层次来理解和分析问题。每个学科都有一个自己解释问题的层次。

美国哲学家丹尼尔·丹尼特（Daniel Dennett）曾说过，各个学科就是一系列解释的层次。在最低的层次上是描述亚原子粒子的物理学；物理学上面是化学；化学上面是生物学；生物学上面是心理学；心理学上面还有社会学、经济学等。这就是从硬学科到软学科的层次划分。

你可能会说，那么底层的学科不重要吗？我只学底层的知识不就好了？并非如此。尽管高层次的知识可以还原到低层次的学科上，但有些特征和性质只在高层次上 **涌现**。

例如，水由氢和氧构成。但水的“流动性”，氢元素和氧元素能预测吗？预测不了。它是在“水”这个层面突然涌现的性质。因此，为了更好接近水的真相，我们就需要多层级去解释和理解。

有一种分层认识世界的方法，就是分为 **微观、中观、宏观**。为什么讲这个？后面我们在分析女生时，会用到很多不同的角度，所以这是在打基础。

### 多视角分析

除了多层级，多元思维模型使用的另一个重要方法是 **多视角分析**。

以分析“拼多多为何崛起”为例。

*   **微观视角（用户体感）**：分析拼多多如何拿捏人性的恐惧和贪婪，进行精心运营。
*   **中观视角（产业与竞争）**：
    *   **供应商视角**：淘宝打假清退低端供应商，这些供应商需要新渠道，拼多多承接了他们。
    *   **用户与基础设施视角**：智能手机普及、移动支付（微信/支付宝）成熟、物流网络完善，使得拼多多能以低成本利用社会已有基础设施快速崛起。
    *   **竞争者视角**：淘宝、京东作为上市公司，对品牌形象要求高，无法顾及低端便宜产品市场，给拼多多留下了生存空间。
*   **宏观视角（趋势与网络）**：用“第二曲线”、“价值网络”等模型思考拼多多未来可能遇到的问题。

所以你看，简单一个公司的崛起，能分析出来的东西远超想象。我们生活的世界是一个复杂的系统。在这个复杂系统中，每个层次都有不同的关系和因果，这些又会在更高层次涌现出更多的关系和因果，各种因果纠缠。

对于复杂的真相，我们可能永远无法穷尽，但我们可以不断接近真相。如何接近？你要懂得用多元思维模型，多视角来分析和理解。

如果我们只是凭自己有限的经验、过去的习惯、单一的视角来理解世界，那永远只是用一个“切片”来看世界，这会大幅度失真。因此，我们需要配合不同的视角。如果我们还有分层的视角，拥有不同的思维方式来辅助我们共同理解一件事情，做出正确决策的概率就要远远大于单一视角。

我之所以那么强调多元思维模型，因为事实的真相往往就是复杂的、分层的。这个世界可能本来就是这个样子。而我们知识追求的最终目的，就是用来反映世界的真相。
