# 1、老吴：老吴《高级情商私教课》：学员分享（十）告别自我中心，一切自然顺利_note

在本节课中，我们将通过学员欧阳景天的真实分享，学习如何通过转变以自我为中心的思维模式，来改善人际关系、提升做事效率并增强个人自信。核心在于学会**换位思考**与**自我调整**。

## 概述：从直觉行事到理性思考

欧阳同学首先分享了他两个多月学习后的核心转变：从依赖直觉做事，转变为在行动前进行更多思考。

以前他做事单凭直觉，想到就做。现在，他会先问自己几个问题：**“我为什么要这样做？”**、**“做了之后别人会怎样想？”**以及**“别人为何会这样反馈？”**。这种预先在脑海中设定情景的思考方式，让他对事情的掌控力更强，时间安排也更合理。

## 核心转变一：学会换位思考

上一节我们介绍了思考模式的转变，本节中我们来看看这种转变如何具体应用于人际交往。关键在于**站在对方的角度考虑感受**。

欧阳同学意识到，过去自己比较自我，认为事情就该如自己所想。别人不听，他就会发脾气。现在他明白了，别人不听是别人的自由，源于其不同的思想和理念。不能将自己的思想强加于人，否则会导致双方都不愉快。

以下是他在实践中总结的具体改变：

*   **停止强求**：不强迫别人做不喜欢的事。
*   **以身作则**：自己先做出改变，别人看到后会更积极地回应。
*   **管理情绪**：不再轻易发脾气或争吵。
*   **预先考量**：说话或做事前，先考虑对方的感受。

这些改变让他的家庭关系，特别是与父母、兄弟姐妹的沟通，变得更加和谐。

## 核心转变二：优化个人习惯与工作方法

思维转变也带动了行为习惯的优化。欧阳同学开始主动调整作息，提前规划时间。家人看到他的积极变化，也给予了更和谐的反馈。

在工作上，他同样应用了换位思考的原则：

> 替别人着想，多做一点，不会吃亏。对方反而会觉得你大度、诚恳。

此外，他做事变得更有条理。他会将任务按**一、二、三、四、五**的顺序排列。公式可以表示为：

**有序任务列表 = 优先级1 + 优先级2 + 优先级3 + …**

这种方法让他和同事的节奏更协调，做事更顺畅，避免了以往同时处理多件事导致的混乱和遗忘。

## 核心转变三：深刻自省与自信调整

除了对外关系的改善，对内自省是更重要的成长。欧阳同学认识到，过去许多问题的根源在于自己的**自我中心**和**独断专行**。

这常导致他言语伤人且事后不补救，使双方心存芥蒂，无法敞开心扉合作，最终结果常与预期不符，造成时间和精力的浪费。

发现问题后，他积极改正，凡事多考虑别人的感受，并注意调整说话的语气和用词。这使他近期做事更顺利，也遇到了更多愿意帮助他的“贵人”。这印证了一个核心观点：**你如何对待别人，别人就会如何对待你**。

最后，他谈到了关于自信的调整。他意识到自己常在**不自信（恐惧逃避）**和**过度自信（狂妄浮夸）**两个极端间摇摆。

以下是他的调整方法：

*   **面对不自信时**：问自己“为什么会害怕？”，想清楚原因后，往往发现没什么大不了，从而更容易面对。
*   **面对过度自信时**：问自己“凭什么这么狂妄？是否考虑了后果？”

目前，他已能将“不恐惧”的心态应用于工作咨询等场景。当然，他坦言在面对陌生异性时仍会紧张、不自然，这是他正在通过学习攻克的下一个课题。他也反思了自己的一个盲点：**不敢提问**。他认识到，向懂的人提问是节省时间、避免内耗的最优路径，而非弱点。

## 导师点评：进步与优化方向

导师老吴对欧阳同学的分享做了点评。

首先，他肯定了多个优点：

1.  **思考更深入**：不再单凭直觉，对事情的掌控力增强。
2.  **具备换位思考能力**：能考虑他人感受，摆脱自我中心。
3.  **情绪管理能力增强**：不再强加思想于人，不乱发脾气。
4.  **行为习惯改变**：以身作则，起到了带领作用（因为很多人是“看见了才相信”）。
5.  **家庭关系改善**：沟通变得更好。
6.  **生活更规律**：调整作息，获得家人积极回应。
7.  **做事更有条理**：人生因此顺畅许多。
8.  **懂得自省**：从自身找问题，从而遇到了更多贵人。
9.  **勇于面对**：会考虑后果并做心理调整，更勇敢。

同时，导师也指出了可优化的方向：分享时部分内容重复次数较多，整体条理性可以更强。建议未来可以按板块结构化分享，例如：

*   情感关系心得
*   家庭关系心得
*   工作关系心得
*   人际关系心得
*   自我成长启发

这样能使听众的体验更加顺畅。

## 总结

本节课中，我们一起学习了学员欧阳景天的成长历程。他的经历生动地展示了，**告别以自我为中心的思维模式**是情商提升的关键一步。通过**换位思考**、**优化个人习惯**、**深刻自省**以及**调整自信状态**，我们能够显著改善人际关系，让工作和生活都变得更加顺畅自然。记住，改变始于自身，当你开始转变，周围的世界也会随之给出积极的回应。
