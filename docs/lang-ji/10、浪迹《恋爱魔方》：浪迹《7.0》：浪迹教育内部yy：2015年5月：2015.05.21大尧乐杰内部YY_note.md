## 实战技巧：搭讪与社交直觉

在资源收集中，搭讪是关键一步。本节我们聚焦搭讪中的核心技巧。

导师大尧分享了一次在四川传媒大学的搭讪经历。他提到，搭讪时“真诚”是首要原则。不要背诵刻板的开场白，要让对方感受到你是真心想认识她。

> “女生反馈说，搭讪时最重要的就是‘真诚’。你是否真的想认识她，对方是能感受到的。”

以下是处理常见搭讪问题的要点：
*   **应对多人组合**：如果目标女性有同伴（障碍），必须同时与两个人交流，并尊重障碍的意见。
    *   **公式**：`表达意图` -> `兼顾目标与障碍` -> `获得障碍默许` -> `收号`
*   **培养社交直觉**：这是聊天的关键。要理解女性话语背后的情绪和真实意图，而非表面文字。
    *   **例子**：女性问“在干嘛？”，潜在意思是“我想你了”。标准回应可以是“我也想你了”。
*   **聊天原则**：避免冗长、低效的“死聊”。关键在于营造轻松愉快的氛围，而非寻找特定话题。
