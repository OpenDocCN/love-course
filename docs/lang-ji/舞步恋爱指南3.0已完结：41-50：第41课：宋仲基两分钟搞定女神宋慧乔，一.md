# 舞步恋爱指南3.0已完结：41-50：第41课：宋仲基两分钟搞定女神宋慧乔，一

上一节我们介绍了从影视作品中学习恋爱技巧的总体思路。本节中，我们来看看《太阳的后裔》中宋仲基与宋慧乔初遇的经典场景，并拆解其中蕴含的互动智慧。

剧中，宋仲基饰演的特种兵柳时镇因枪伤就医，遇到了宋慧乔饰演的医生姜暮烟。双方通过一段简短的对话，迅速建立了暧昧联系。

在视频中我们可以看到，宋仲基对于自己身上的枪伤，做了一个轻描淡写的玩笑来描述，以此缓和气氛。这种举重若轻的感觉，就是和女生聊天时经常使用的一个技巧，它叫做**反撇**。

**反撇**的公式可以概括为：**用轻松、幽默的方式描述严肃或尴尬的事，以降低压力、展示自信**。

当宋慧乔主动参与到关于伤口的对话中时，宋仲基顺势开了个玩笑。此时，两人有了短暂的眼神交流。俗话说，眼睛是心灵的窗户。眼神能够传递的信息，远比语言要多。虽然只有一两秒钟，但这次眼神交流让双方心中萌生出悸动的感觉。

很明显，宋慧乔感受到了这种暧昧的悸动，随即把话题转向了伤口处理的具体事务上。这不但化解了暧昧可能带来的尴尬，也引出了我们今天课程的重点：**识别情感信号灯并采取正确应对**。

以下是对话中信号灯变化的详细解析：

当宋慧乔问“军队也有医院吧？”，宋仲基反问“可以到这里来吗？”。这里，他把刚才对方稍微躲避掉的暧昧气氛，重新做了试探性的推进。

宋慧乔回复“这里不远吗？”。这就像一个**黄灯**，表示略有顾虑但未明确拒绝。

请记住，特别对于我们男生，在和异性打交道时，只要没有看到明确的**红灯**（直接拒绝），就应该尝试继续推进关系。宋仲基没有停留，而是进一步推进框架，问道：“每天都可以来吗？”

宋慧乔的回复是：“每天来是不是有点多？……一周三次吧。” 这表明她对宋仲基的推进开了**绿灯**，给予了许可。

紧接着，宋仲基继续推进，去问：“是医生（你）来做吗？（伤口消毒）”。宋慧乔的回复再次属于**黄灯**：“医生的选择重要吗？”

宋仲基看到后，继续推进并加大了“诱惑”的力度，他说：“重要，特别是医生的美貌。” 这个话题已经属于明显释放暧昧的信号，将关系推进到了**调情阶段**。

如果在这种情况下，女生没有亮起**红灯**，那我们就可以知道，在她心里基本接受并喜欢对面的男生。后续的关系发展就会变得相对容易。

在接下来的剧情中，宋仲基继续在语言和肢体上，逐步拉近两人的关系，暗示双方成为男女朋友的可能性，并得到了女生的积极回应。

在男女恋爱中，关系推进的责任通常应由男人来承担。作为男人，千万不要害怕、懦弱或不敢有所作为。这种气质的男生很难赢得美女的青睐，常常被发“好人卡”。

**“好人卡”** 的潜台词往往是“懦弱感”。但另一方面，“不害怕”也不代表可以不顾女生感受胡来。正确的方法是：小心地试探女生的心意，根据对方打开的“窗口”，勇敢进攻，而把不好意思和害羞留给女生。

**操作方法总结**：**识别信号（绿灯/黄灯） -> 大胆试探 -> 逐步推进 -> 将害羞感留给对方**。
