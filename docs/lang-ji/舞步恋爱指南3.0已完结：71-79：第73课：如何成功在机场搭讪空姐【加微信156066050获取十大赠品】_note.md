# 课程73：如何成功在机场搭讪空姐 ✈️💬

在本节课中，我们将学习如何处理搭讪过程中女性可能给出的模糊暗示或“废物测试”，特别是通过一个留下不完整联系方式的案例，来理解其背后的筛选逻辑，并掌握正确的应对方法。

---

在之前的课程中，我们介绍了当有其他人来搭讪你身边的女性同伴时应如何处理。本节中，我们来看看另一种常见情况：女性在搭讪时给出的回应并非简单的给或不给联系方式，而可能包含一些含糊不清的暗示。

这些暗示通常是一种“废物测试”，旨在考察搭讪者的诚意或应变能力。有些测试是女性有意识发出的，有些则是下意识的反应。

以下是搭讪时可能遇到的几种典型“废物测试”话语：
*   “我有男朋友了。”
*   “我们又不熟，干嘛要给你电话？”

对于此类测试，一个幽默的回应往往是有效的。一旦对方笑了，测试便算通过。例如，对于“我们不熟”的测试，可以这样回复：
**“你看现在天气这么热，我们很快‘就熟了’。”**

当然，除了这种快速化解的方法，我们还需要更多的解决方案。让我们通过一个具体案例来深入分析。

以下是一个搭讪场景的视频片段描述：
> 男：“我不知道说什么了，其实我只是想跟你做个朋友，这个是我的名片。有笔吗？”
> 女：“有有有，谢谢谢谢。”
> （女性在名片上写下号码后）
> 男：“爱美丽…美丽，天使一样的名字…怎么只有九个号？”

在这个案例中，女性留下了9位数的电话号码（通常电话号码应为11位）。这是一种测试搭讪者诚意的常见方式。

如果运气不佳，尝试补齐最后两位数字需要拨打最多**100次**电话（从00到99）。这是有经验的女性用来筛选那些礼貌、友善且有涵养的男性的一种方式。

其筛选逻辑如下：
*   如果男性只是随意搭讪，通常不会愿意花费大量时间拨打这么多电话。这样的男性，不联系也不可惜。
*   或者，男性可能直接认定这是假号码而放弃。

无论哪种情况，该男性都会在女性的这第一轮筛选中被淘汰。同时，女性也避免了显得小气或直接拒绝对方。

因此，兄弟们，下次如果遇到女性只留下部分号码的情况，需要明白：这是她在测试你的诚意并进行筛选。如果你真的喜欢对方，必须想方设法联系上她。

在这个过程中，虽然会花费一定时间，但一旦通过测试，后续的交往往往会顺利很多。
