## 加载与前期调整 🔧

上一节我们介绍了Snapseed软件的基本界面，本节中我们来看看如何加载图片并进行前期调整。

以下是加载与前期调整的步骤：

1. 打开Snapseed软件，点击加号加载一张图片。
2. 加载图片后，首先进行旋转调整。点击工具，选择旋转功能，软件会自动校正图片角度。
3. 接下来进行裁剪调整。点击工具，选择裁剪功能，根据构图线调整图片大小和位置，确保主体位于黄金分割点。
4. 调整完成后，点击打勾确认。

## 调整图片功能详解 🛠️

在完成前期调整后，我们需要进一步优化图片的明暗和色彩。本节将详细介绍调整图片功能的使用方法。

以下是调整图片功能的操作步骤：

1. 点击工具，选择调整图片功能。
2. 屏幕顶部显示各调整选项的参数程度，默认值为0。通过左右滑动可以调整参数数值。
3. 屏幕底部依次为关闭按钮、调整选项菜单、自动调整按钮和完成按钮。
4. 在调整选项菜单中，可以通过上下滑动选择不同的调整选项，如亮度、对比度、高光、阴影、饱和度、暖色调和氛围。

## 明暗调整参数 📊

明暗调整是修图的基础，本节将详细讲解亮度、对比度、高光和阴影这四个参数的作用。

以下是明暗调整参数的具体说明：

- **亮度**：调整整个画面的明暗程度。公式为：`亮度 = 画面整体明暗值`。调整时需注意避免画面过亮或过暗，导致细节丢失。
- **对比度**：调整画面明暗对比的强烈程度。公式为：`对比度 = 明暗差异值`。增加对比度会使画面更鲜明，但过度增加会导致细节丢失；减少对比度则会使画面发灰。
- **高光**：调整画面中高亮部分的明暗。公式为：`高光 = 高亮区域明暗值`。增加高光会使高亮部分更亮，减少则能保留更多细节。
- **阴影**：调整画面中暗部的明暗。公式为：`阴影 = 暗部区域明暗值`。增加阴影会使暗部更亮，减少则会使暗部更暗。

---

## 色彩调整参数 🎨

除了明暗调整，色彩调整也是修图的重要环节。本节将介绍饱和度和暖色调这两个参数的使用方法。

以下是色彩调整参数的具体说明：

- **饱和度**：调整画面色彩的鲜艳程度。公式为：`饱和度 = 色彩鲜艳值`。向左滑动减少饱和度，画面会变灰甚至黑白；向右滑动增加饱和度，画面会更鲜艳。
- **暖色调**：调整画面的色温，影响画面的冷暖感觉。公式为：`暖色调 = 色温值`。向左滑动画面偏蓝（冷色调），向右滑动画面偏黄（暖色调）。

---

## 氛围调整功能 🌟

氛围调整是Snapseed独有的功能，能够综合调整画面的色彩和层次感。本节将详细介绍氛围调整的作用。

氛围调整的公式为：`氛围 = 综合调整值`。提高氛围值可以使画面色彩更鲜艳、细节更突出、层次感更强；降低氛围值则会增强明暗对比，但不会出现死黑或死白的情况。

---

## 画笔工具的使用 ✏️

画笔工具可以实现局部调整，适合对特定区域进行精细编辑。本节将介绍画笔工具的具体操作方法。

以下是画笔工具的使用步骤：

1. 点击工具，选择画笔功能。
2. 在画笔选项中，可以选择加光减光、曝光、色温或饱和度。
3. 以饱和度为例，选中饱和度后，在需要调整的区域进行涂抹。
4. 点击右下角的小眼睛可以查看涂抹范围，调整完成后关闭小眼睛查看效果。

---

## 局部工具的使用 🎯

局部工具可以针对特定区域进行明暗、对比度、饱和度和结构的调整。本节将介绍局部工具的具体操作方法。

以下是局部工具的使用步骤：

1. 点击工具，选择局部功能。
2. 点击加号，在需要调整的区域（如面部）添加控制点。
3. 用两指缩放控制点范围，确保只覆盖目标区域。
4. 上下滑动调整亮度、对比度、饱和度或结构参数，优化目标区域的效果。
