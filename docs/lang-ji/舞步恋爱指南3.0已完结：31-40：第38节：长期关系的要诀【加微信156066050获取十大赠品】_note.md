# 课程名称：长期亲密关系要诀 🏠

## 概述

在本节课中，我们将要学习长期亲密关系的核心要诀。课程将探讨三种影响我们一生的长期亲密关系类型，分析其内在联系，并提供清晰的行动思路。

## 长期亲密关系的意义与本质

绝大多数人一生都会经历三种类型的长期亲密关系。这三种关系会持续对我们的人生产生深远的影响和改变。

## 第一种关系：父母与我 👨‍👩‍👦

我们在经历的第一种长期亲密关系是与父母的关系。这段关系奠定了我们一生的性格以及与他人关系的基石。

中国有句俗语：“三岁看大，七岁看老”。西方心理学也证明，六岁前，特别是三岁以前的经历，对我们整个人生有着非常深远的影响。

这是因为我们在潜意识形成的关键阶段，最亲密的两个人就是父母。他们对待我们的态度，以及他们彼此对待对方的态度，会深深烙印在我们心灵最深处，并终身影响我们。

**核心概念**：童年与父母的关系模式，会成为我们成年后人际关系的“内在脚本”。

我记得微信上有一位女生曾问我：“我要怎么样才能和我的男朋友更加亲近起来？” 我问她是否喜欢对方，她回答“喜欢”。我问她是否想亲近，她回答“想”。接着我问她：“那你和你老爸的关系好吗？” 她说“一般般”。我问：“你会靠在你爸爸身边撒娇吗？” 她回复说“不会，觉得那样很别扭”。

我说：“对，因为你和自己生命中最亲近的男人——你的爸爸，关系都不亲密，那你如何有办法和其他男人关系亲密起来呢？” 这就是一种典型的心理映射。

我还记得有一位学员，今年32岁，外形条件尚可，但找不到女朋友。在三天的课程中，他几乎没主动说过话，自我介绍时声音也非常含糊。后来在一对一环节中，他告诉我，父母在他很小时就去南方打工，常年不在家，他是留守儿童，由爷爷奶奶带大。

他本来挺活泼，但小学时被其他孩子欺负殴打后，感觉父母不在身边，没有依靠，于是开始变得沉默寡言。这种状态持续了二十几年。因为不常与人沟通，他的普通话变得含糊，工作也一直没什么起色。我问他多久和父母联系一次，他说大概三四个月一次。我告诉他，这太少了，并且他心里对父母有怨恨，这种负能量已经不单单影响他的情感生活，更影响了他的工作和交流的方方面面。情感问题只是这种内在伤痛最直接、最赤裸的呈现。

包括在课程中，还有很多学员总是呈现出一副唯唯诺诺、不知所措的样子。跟他们深入交谈后才知道，他们的母亲都非常强势，不仅打压他们，也打压他们的父亲。他们从小生活在一个父亲萎靡不振、母亲只手遮天的环境中。他们渴望母亲的爱，又恐惧母亲的爱。因为父亲身上没有自信阳刚的特质，他们长大后也不知道如何做一个自信阳刚的男人。所以他们渴望女人的爱，又恐惧女人的爱，在女人面前永远是一副逆来顺受、低三下四的样子。没有女人会喜欢这样的男人。

因此，我要分享一句话：我们和母亲的关系，决定我们这辈子和女人之间的关系；我们和父亲的关系，决定我们这辈子和男人之间的关系。成年后与恋人的关系，深受童年与父母关系的影响。

了解这个内在逻辑之后，目的不是去抱怨父母、推卸责任，而是让我们认识到问题的核心根源，然后努力勇敢地去面对和解决。毕竟，父母也没有学习过如何当父母，他们只是给了他们认为最好的方式。

## 第二种关系：恋人与我 💑

上一节我们介绍了原生家庭的影响，本节中我们来看看人生中的第二段长期亲密关系：与恋人的关系。

有同学可能会问：我们没有时光机，过去的事情无法改变，但现在和女朋友总是相处不好，该怎么办？

我们无法坐时光机回到过去修复心灵的伤口，但可以借助现在和身边亲近的恋人来修复。在成长过程中，最亲近的人（如父母）给我们心灵留下了伤口。这些伤口在成年后，当其他最亲近的人（比如女朋友）对我们做类似的事情时，可能会被重新激活。

简单来说，我们的一些旧伤口会被重新揭开。这时我们可能会很疼、很难过、很不爽，但这正是修复自己内心缺口的好机会。如果能在此时意识到这一点，主动、冷静、负责任地和女朋友一起处理好这些昔日的伤口，自然是最好的。

如果做不到，或者被动地让生活推着走，也可以，只不过速度会相对慢一点。通常，我们不经历两三段恋爱，很难真正步入婚姻。即使步入婚姻，也可能面临离婚。因为在这几段恋爱过程中，我们经历了一个被动的修复过程。否则，一个满是伤口、裂痕、凹凸不平的内心，不可能承载起一段圆融、美好的关系。

有同学可能会问：那我怎么意识到什么时候该修复呢？这很简单：当你的情绪要爆发时，你能驾驭住自己的情绪，做正确的事情，而不是被情绪控制，做出让自己后悔的事情。这也是成熟的标识。女生都喜欢成熟的男人，而成熟就是要学会承担责任，驾驭自己的情绪。

## 第三种关系：我与孩子 👶

如果责任感迟迟得不到提升和成长，我们会发现，等到经历下一段长期亲密关系——即与孩子的关系时，可能才是真正噩梦的开始。这对孩子和我们自己，肯定都是不好的。

## 总结与核心要诀

本节课中我们一起学习了长期亲密关系的核心框架。

总结今天的课程重点：命运为我们安排了三种类型的亲密关系，分别是我们与父母的关系、我们与恋人的关系，以及我们与孩子的关系。这三段长期关系不断相互影响。

如果把一生所有的亲密关系比作一栋大楼，那我们就要经历 **被建造**、**被修补**、**被复制** 这三个过程。

**核心公式**：
`人生亲密关系质量 ≈ (原生家庭基础 + 自我修复努力) × 责任感`

这栋楼如果没建好，或者盖歪了，修补起来会很累。但为了下一栋楼（与孩子的关系）能更好地被复制，我们还是要努力、负责地去修缮，不要让同样的错误继续延续下去。

## 结束语
