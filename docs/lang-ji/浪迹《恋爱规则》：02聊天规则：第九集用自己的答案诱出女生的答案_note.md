# 浪迹《恋爱规则》课程02：聊天规则 🗣️  
## 第九集：用自己的答案诱出女生的答案 💡

在本节课中，我们将学习一种高效的聊天技巧：通过主动分享自己的信息来引导女生给出答案，从而避免直接提问可能带来的压力。

在上一节中，我们介绍了用陈述句代替疑问句的提问技巧。本节中，我们来看看如何用“分享答案”代替“索取答案”。

直接向女生提问有时会显得过于急切。更好的方法是先说出自己的情况，以此引导对方自然回应。例如，若你想让女生自我介绍，不必直接问她“你能介绍一下自己吗”，而是可以先介绍自己。

以下是具体操作示例：

*   你可以说：“我叫王欢宇，86年出生，白羊座，平时喜欢篮球、游泳、健身、阅读和拍节目。”
*   这样，你没有提问，但给出了一个清晰的自我描述模板。聪明的女生通常会领会，并跟着分享她的信息：“我叫张叉叉，00年出生，29岁，射手座，平时喜欢下午茶、逛街和跟闺蜜蹦迪。”

再看另一个常见场景。当你想知道女生“在干嘛”时，不必直接发问。

以下是另一个应用示例：

*   你可以直接发一张自己正在健身的照片。对方看到后，很可能自然地回复你她的状态，比如：“我还在吃饭。”

这种方法的核心优势在于：它能有效避免因连续提问而暴露过多需求感。同时，在询问他人之前先分享自己，会让女生觉得你更真诚、更乐于分享。这种氛围下，你也更容易获得想要的信息。

本节课中，我们一起学习了“用自身答案引导对方答案”的聊天技巧。关键在于主动、真诚地分享，以营造轻松的交流氛围，从而更自然地获得女生的信息。
