# 课程三：加藤非《心肾全能速成课》第十四课【肉体流，短期速推】💪

在本节课中，我们将要学习关于“短期速推”的核心策略与实战方法。我们将探讨如何识别目标人群、利用特定场景，以及通过勤奋的行动来达成短期关系。课程内容基于实践交流，旨在为初学者提供清晰、可操作的指导。

---

上一节我们介绍了课程的整体框架，本节中我们来看看“短期速推”的具体执行策略。

很多时候，兄弟们并不清楚“速推”需要哪些内容和条件。这部分内容是我与资深同行深入交流过的，今天分享给大家，希望能帮助大家对“速推”这件事建立正确的认识。

首先，第一点需要明确的是作息时间。如果你有晚上一点钟以前就睡觉的习惯，那么你进行“速推”的难度会极大增加。因为凌晨一点钟以后，很多在外饮酒、寻求释放的女生会更容易接触。

接下来，我们探讨适合“速推”的目标人群。以下是几类需要重点关注的人群：

1.  **外来人口**：在你所在的城市，尤其是一二线城市，外来人口是很好的目标。他们不像本地人那样有诸多社交顾虑，更容易放开。
2.  **独居女性**：判断女性的住宿情况非常关键。如果她是一个人居住，并且对你有一定的好感窗口（例如聊天主动、允许肢体接触），那么成功率会很高。独居意味着私密和方便。
3.  **旅游人群**：在当地旅游景点遇到的女性通常带有“艳遇”情节。因为旅行本身带有短暂、浪漫的属性，小说和影视作品也强化了这种印象。对于这类目标，行动一定要快，可以以“难得来一次，明天就要走了”为理由迅速邀约。
4.  **夜店常客**：经常出入酒吧附近的女性，本身就有较强的社交和娱乐需求。这是为什么我们强调搭讪要去这些地方。

在识别了目标人群后，我们需要谈谈核心的执行力。关于“速推”，其实没有太多复杂的技术，全凭两个字：**勤奋**。

这不是指颜值或话术，而是指你是否愿意在深夜依然保持活跃，是否愿意在天气不好时依然赴约，是否愿意付出大量的时间和精力去接触、筛选目标。很多所谓的“战绩”背后，是常人难以坚持的付出。

当然，对于新手而言，通过初期的“速推”可以快速提升自信和社交直觉。大约经历 **20段** 左右的短期关系后，你会获得显著的成长，对个人魅力建立信心。

但是，请记住，在这20段经历之后，“速推”更多就变成了一种个人生活方式的选择。你可以选择继续投入时间在这上面，也可以将这些时间用于健身、学习、提升事业等其他个人发展项目。这只是一种娱乐方式。

---

上一节我们强调了目标和勤奋，本节中我们来看看提升“速推”成功率的关键环境：夜店。

为什么夜店如此重要？根据沟通的“73855法则”，文字内容只能传递7%的信息，语调占38%，而肢体语言则占了55%。在夜店，昏暗的灯光、震动的音乐和近距离接触，让肢体语言和非语言交流的优势最大化。

对于很多只在网上聊天，一见真人就紧张、手抖的兄弟来说，夜店是绝佳的练习场。它能极大地提升你的“社交直觉”，让你学会在真实互动中运用眼神、触碰和氛围推进关系。

去夜店不一定很花钱，有以下几种低成本方式：

*   **空手流**：不消费，只进入场所，以社交和认识人为目的。
*   **一杯流**：点一杯酒，喝一整晚，重点在于互动和转场。

在夜店，目的不是灌醉对方，而是利用酒精和氛围作为“合理的放纵理由”。有时，适当的“装醉”也是演技的一部分，可以为肢体接触和转场提供借口。

---

上一节我们讨论了夜店这个重要场景，本节中我们来看看如果没有条件或不愿去夜店，还有什么其他关键。

如果没有夜店，那么创造 **私密空间** 就变得至关重要。很多兄弟无法“速推”，是因为约会总停留在吃饭、逛街、看电影这些公开场合，双方没有独处的机会。

男女之间那层窗户纸，往往只需要一个合适的私密空间就能捅破。只要约会氛围愉快，彼此有好感，能带到无人打扰的私密空间，就有很大概率发生关系。

私人影院（确保能反锁）、家中、甚至一些刺激的半公共场所（如安全的楼道顶层），都比目的性太强的酒店更有情趣和推进空间。

在互动中，调情的艺术在于营造“悬而未决”的期待感。最刺激的时刻，是双方都知道可能会发生什么，但不知道何时会发生。这种推拉和挑逗，有时比最终的结果更让人回味。

---

在本节课中，我们一起学习了“短期速推”的核心要点。我们明确了**勤奋**是核心驱动力，识别了**外来人口、独居、旅游人群、夜店常客**这四类目标，强调了**夜店**作为提升社交直觉和利用氛围的高效场景，并指出了**私密空间**对于转化关系的必要性。

记住，初期可以通过实践积累自信（约**20段**经历），但之后应将其视为一种可选择的娱乐方式，而非生活的全部。将更多精力投入长期的自我提升，才能获得更高质量的关系和人生体验。
