# 浪迹《恋爱规则》：02聊天规则：第三集不过分追求一鸣惊人的聊天效果_note

在本节课中，我们将要学习一个核心的聊天原则：避免过分追求标新立异和“一鸣惊人”的聊天效果。我们将探讨为何这种做法常常适得其反，并了解如何构建自然、持久的有效沟通。

许多人在聊天时，初衷是希望自己与众不同。例如，对女生说“你的眼睛很像梅花鹿”，或者说“你的脸上有一种东西，叫做气质”。他们认为说这些话会显得自己很特别。

确实，有些女生可能会觉得你很特别。但同样，也有些女生会觉得这种特别“特别不正常”，甚至认为你是神经病。所以，问题的根源往往在于思考逻辑出现了偏差。

他们错误地认为，正常人是谈不到恋爱的，只有这种“不正经”的人才能成功。其实这是错误的。世界上99.99%的恋爱，双方之间的互动都是正常的。因此，你不应该使用这些看似不正常或与众不同的“话术”来凸显自己。

过分追求特别，实际上可能会显得你很愚蠢。例如，有兄弟会问：“怎么聊天可以让女生眼前一亮？”然后就用各种情话去轰炸对方，比如“我眼里都是小星星，看到你以后”，或者“这是我的手臂，那你是我的什么？你是我的宝臂”。

结果往往是失败的。上次用这种方法的兄弟，成功交到女朋友了吗？对方很可能回复“我才不是丈母娘”。中国是一个讲究“中庸”的国家。所谓“中庸”，就是不要过于突出或极端。

“枪打出头鸟”、“人怕出名猪怕壮”这些古老的谚语，一定要记在心里。这揭示了社会交往中的一个普遍规律：过度偏离常态会带来风险。

我们可以用一个比喻来加深理解：动物和猩猩的基因有92%是相似的。正是剩下的8%不同，造成了一个是动物，一个是人类。

你和女生聊天也是同样的道理。在你们的对话中，最多可以有**5%** 的内容是你自由发挥、展现个性的空间。而**95%** 的交流，仍然需要在一个正常的、社会共识的框架内进行。

如果你的聊天内容有**10%** 或**20%** 以上都和别人不一样，你就会显得不正常，就像那个“猩猩”一样。这就是为什么很多人聊天像机器人，或者让女生觉得奇怪——因为他们把所有的对话都生搬硬套成“土味情话”。

那么，什么是“土味情话”呢？它可能是一种调剂。也许你聊100句话，其中有一句土味情话，女生会觉得有点意思。但它绝不能成为你聊天的主体。

如果你的聊天全是土味情话，女生会觉得你是一个“话术惯例机器”。所谓的话术和技巧，只能偶尔使用，作为聊天的调剂是可以的，但不能成为常态。

与女生相处，千万不要想着走捷径。你必须脚踏实地，一步一步地去沟通，才能最终通往幸福。那些你一两句话就能“搞定”的女生，要么特别傻，要么关系来得快去得也快。试想一下，你一两句话可以搞定她，别人同样也可以。

因此，我认为与女生聊天时，你的“特别之处”不应该在于你聊得多么与众不同。你的特别点应该在于：**你与她聊天的时间特别长**，**你们聊天的感觉特别好**，**你特别在意她**。你的“特别”应该放在这些实实在在的投入和关怀上。

本节课中，我们一起学习了避免在聊天中过分追求“一鸣惊人”的重要性。关键在于理解**95%正常框架 + 5%个性发挥**的平衡原则，放弃对浮夸“话术”的依赖，转而通过持久的、真诚的沟通来建立稳固的关系。记住，持久的关注和真诚的互动，远比一句惊艳的开场白更有力量。
