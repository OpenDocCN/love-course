# 聊天研究所 第39课：男人说话要有点立场 🧭

在本节课中，我们将通过分析一份学员的真实聊天记录，学习在沟通中保持立场和框架的重要性。我们将看到，随意的让步和缺乏原则的回应，会如何影响对话的走向和双方的地位。

---

## 聊天开场与初步互动

在通过好友验证后，女生主动发送了自己的名字。这是一种兴趣度的表现。男生回复：“听起来有点耳熟”，回应了对方的名字，并随后报上了自己的名字。

这些都属于正常的初步沟通。

接着，女生开启了第一个话题：“感觉我们好像加过微信”。此时，这位男学员的回复非常有趣，他说：“你干嘛删我，以后不许这样。”这相当于一上来就与女生进行了一次轻微的调侃。

这个操作没有问题。女生则回复了一个“捂脸”表情，并说：“我删了吗？你吧。”这里，女生有点像在“踢皮球”——你把问题抛给她，她又把问题踢了回来。

## 立场问题的首次暴露

好，那么这个时候，男生的回复就出现了问题。当对方把“皮球”踢回来时，你直接接住并说“都一样啦”，这相当于表明你是一个**没有原则和立场**的人，显得很随意。

接着，男生直接转换话题问女生：“现在在干嘛？”以了解女生现状。女生则回复：“不一样耶，被人删…然后呢在家。”

女生对男生的两句话都做了回应。我们可以看到，女生对“都一样啦”这句话比较在意。她说：“不喜欢被人删。”说真的，没有人喜欢被删除，女生这么说，说明她比较爱面子，有自己的**框架**。

## 框架的丧失与地位逆转

然后男生回复了一句：“我让着你嘛你说什么都行，哈哈。”你看，这和上面的“都一样啦”一样，显得自己很没有立场。一般情况下，如果你允许别人不断**侵略你的领土**，对方就会随着你的让步得寸进尺。

所以你可以看到，女生说话的态度和方式就变了。她说了一句：“那谢谢你。”这是一种明显的、站在**高位**调侃男生的语气。结果男生回复：“应该的。”这又显得没有立场。

从一开始女生主动说话，男生是处于**高位**的，包括男生说“所以你干嘛删我”这里，都是男生在高位，女生在低位。但从后面的“都一样啦”、“我让着你”、“应该的”这些回应开始，男生把自己又变到了**低位**。

## 不必要的回应与框架巩固

而且他还去回应女生的表情，说：“这么圆圆的小脸。”我觉得这句话没有必要。因为对方只是发了一个搭配“谢谢你”这句话的表情，这样回应会让人觉得有点懵。

当我们遇到一个人站在高位打压我们低位的时候，肯定不能一味挨打。因为你越挨打，地位就会越卑微。男生回“应该的”，就相当于直接进入了**女生的框架**。

然后女生就继续“吊打”这个男生，说：“还一点都不客气。”虽然女生是带着调侃的方式，但我们很明显能看出来，现在女生在**高位**，男生在**低位**。

接着男生继续让步，说：“谁都会让着的谁叫你是可爱。”这又是在挨打，并且**暴露了需求感**。如果对方是一个经验丰富的人，男生这样的处理方式很有问题。

## 兴趣度支撑与话题转折

从以上所有分析看得出来，这个女生对这个男生有兴趣，所以才会继续聊下去。但你也看到了，男生越让步，女生越得寸进尺。女生回复：“哈哈只有爸妈让我。”这说明她觉得在外面都没人让着她，现在突然有个男的让着她，她感到很开心。

然后男生回复：“现在多了一个（让着你的人）。”这又是在说自己，再次**暴露需求感**。结果，女生没有给出正面回应，而是说：“我在吃饭啊，等会说。”这就是**抽身离去**。当然，因为这个女生喜欢这个男生，所以她吃完饭后又跟男生说“吃完了”。

## 新话题的展开与错失机会

好，那么男生就接着这个话题说：“哦这么晚才吃完，是自己做的吗？”女生回应：“没有我爸做的。”男生回复：“原来是个幸福的小公举。”这些对话都没问题。

然后女生回复：“因为回家了嘛，平时独立够了，回家就要做儿童。”这也是很多年轻人的做法。这个时候，男生的回复就有问题了。他说：“平时都自己做饭一样？”这句话听上去会让人不太舒服。但因为女生有兴趣，所以还会去回应，解释说：“平时哦我妈心情好，我就做。”

如果是我，在看到女生说“平时独立够了，回家就要做儿童”时，其实这句话可以做一些话题的延伸。从“女生独立”这个点可以展开很多内容。因为现在社会发展，很多女生越来越独立，但从人性天性的角度，女生更需要被保护，更偏向于主内。事实上，很多女生需要关怀和爱，经常独立的女生背后其实挺不容易的。所以可以从这一块切入，去表达关怀。

## 本节课总结

本节课中，我们一起学习了在聊天中保持立场和框架的重要性。通过这份记录，我们看到了：
1.  随意的让步（如“都一样啦”）会让自己丧失立场。
2.  进入对方的框架（如“应该的”）会导致自己处于低位。
3.  不必要的回应和暴露需求感会削弱吸引力。
4.  应抓住对方话语中的关键点（如“独立”）进行深度延伸和关怀，而不是提出可能让人不适的质疑。

核心在于：**沟通是框架的博弈，保持温和而坚定的立场，是赢得尊重和吸引的关键。**
