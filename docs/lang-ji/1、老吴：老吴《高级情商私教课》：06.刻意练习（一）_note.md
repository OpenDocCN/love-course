# 1、老吴：老吴《高级情商私教课》：06.刻意练习（一）_note

在本节课中，我们将要学习“刻意练习”的本质。我们将探讨什么是真正的刻意练习，它与常见的“21天习惯养成”或“1万小时定律”有何不同，以及它如何从根本上改变我们的大脑和思维习惯。

## 什么是刻意练习？

上一节我们介绍了思维模型，它可以帮助我们提高决策效率和创新能力。那么，我们如何才能算真正掌握这些模型呢？

关键在于，你是否能在生活和工作中遇到问题的瞬间，就立刻调用合适的思维模型。这就像电光火石之间，你马上就能用起来。这比仅仅在大脑中储存大量知识更有用。

很多人事后才想起：“啊，刚刚我没有发挥好。”这就像“事后诸葛亮”，用处不大。刻意练习，正是解决这个问题的关键能力。

## 对刻意练习的常见误区

很多人对刻意练习存在误解。以下是两种最常见的误区：

1.  **21天养成习惯论**：这个说法有一定道理，但只适用于培养简单的身体记忆。例如，坚持刷牙21天，你就能养成习惯。这是因为刷牙是一个简单、重复的动作。
    *   **公式**：`简单习惯 ≈ 21天重复`
    然而，对于复杂场景，21天往往不够。例如，一个人追求异性时，可能在21天内表现得无微不至，但关系确立后，这些“习惯”就可能消失。这说明，21天难以养成涉及复杂情感和情境的深层习惯。

2.  **1万小时天才定律**：这个理论在某些领域成立，但它有前提条件。它适用于规则明确、评判标准固定的领域。
    *   **公式**：`专家级技能（规则固定领域）≈ 1万小时标准训练`
    例如，跳水、体操等项目，有明确的动作难度和评分标准（如水花大小）。按照标准训练1万小时，确实可能达到专家水平。

但是，我们学习的思维模型是灵活多变的。生活在一个复杂的世界里，我们遇到的人、事、场景都不同。同一句话，对A说合适，对B说可能就行不通。因此，我们需要的是**独立判断哪个思维模型更适用**的能力。这种能力的训练，远比遵循固定标准复杂。

生活没有永远的标准答案，所以我们不能简单地认为“不断练习”就能掌握一切。如果生活像机器人训练一样有固定程序，那么AI就可以替代我们了。我们不是机器，我们的决策需要因人、因事、因时而异。

## 刻意练习的核心：结合场景

那么，我们该如何有效地训练思维模型呢？核心方法如下：

**核心关键是结合场景。** 我们需要在不同的场景中不断训练、反思这些思维模型，思考哪些可以在生活中应用，并进行匹配。这是一个不断迭代认知、不断练习的过程。

在这个过程中，我们要做的是：用**更适用、更科学、更多元的新思维模型**，替换掉我们大脑中已有的、直觉性的旧思维模型。这本质上是替换我们神经中枢习惯的过程，只能通过大脑的刻意练习来实现。

## 思维模型内化的五个阶段

将新思想训练成直觉，需要经历五个阶段：

以下是思维模型内化的五个渐进阶段：

1.  **第一阶段：想不到**
    遇到可以使用模型的场景，但你完全想不起来，意识不到自己思维的局限。

2.  **第二阶段：事后诸葛亮**
    经过一些训练后，你会在事后想起来：“我好像学过某个思维模型可以解决这个问题。”

3.  **第三阶段：事中有启发**
    你做事做到一半时，突然想起来可以用某个学过的思维模型。

4.  **第四阶段：事前会想起**
    在你准备沿用旧习惯做事之前，大脑会提醒你：“我还有一个新模型可以用。”这时，你会在行动前主动调用新思维。

5.  **第五阶段：不思量自难忘**
    你根本无需思考，新模型已经内化成你的直觉。看到事情，你就知道怎么做，所有模型都在自然而然中被使用。

从这五个阶段可以看出，将思维模型训练成直觉是一个阶梯式的过程。很多人觉得“学完用不上”或“没感觉”，不是因为他们笨，而是因为大脑结构决定了我们无法立刻将新知识内化成直觉能力。

## 替换的是大脑的“路”

我们这里要替换的，不是像“刷牙”那样的简单身体习惯，而是**大脑的思维习惯**。

例如，在第一节课讲到的ABC模型（诱因A -> 信念B -> 结果C），通过刻意练习，我们要替换的就是中间那个“信念B”的部分。你可以理解为一种“自我升级”，用更新、更前沿的认知，去替换掉大脑里旧有的、可能不适用的思维模式。

这就像手机系统升级：用iOS 13覆盖掉iOS 12。社会进步太快，我们必须不断更新大脑里的“操作系统”，才能跟上节奏。

从生理学原理简单解释：我们的大脑依靠神经元连接（神经链）来传递信息和形成习惯。过去的经验习惯，因为被反复使用，就像大脑里一条被走出来的“旧路”。

刻意练习，就是要在“旧路”旁边，**修一条“新路”**（建立新的神经链）。我们不是要完全放弃旧路，而是通过不断训练、使用这条新路，让它变得越来越宽敞、好走。同时，旧路因为久不使用，会逐渐“老化”、变得脆弱。最终，你会自然而然地一直走新路。

很多人遇到问题解决不了，正是因为总在用“旧的思维方式”解决“新的问题”。这就像在2019年，还想用2009年的方法一样困难。你必须掌握最新的方式。

## 如何创造练习环境：助推与反思日记

既然内化思维模型需要大量练习，而我们生活忙碌，该如何进行呢？关键在于创造“助推”环境和工具。

诺贝尔经济学奖得主理查德·泰勒在《助推》一书中指出：**要持续产生一个行为，外部环境的影响特别重要。**

例如，餐厅把高热量食物放在显眼易取的位置，还是放在隐蔽难取的位置，会显著影响顾客摄入的卡路里。同理，要不断练习思维模型，我们也需要一个能“助推”我们反复思考和触发的环境。

一个非常有效的方法是：**写反思日记**。

古人说“吾日三省吾身”。我们通常只在“栽了跟头”、事情不顺利时才会反省。但很多人痛过就忘了，或者几个月、一年后才偶然想起。写反思日记，就是主动创造“反省场景”，把每天的触动、思考和可能的模型应用记录下来。

例如，我以前分享时容易滔滔不绝，但现在我会通过训练，留意听众状态，觉得够了就马上刹车。这就是通过反思和训练强化的结果。

反思日记作为一个场景工具，能帮助我们坚持训练。久而久之，你的思维模型就会内化成一种“自觉”——自动自觉地走新路，替代旧路。

本节课中，我们一起学习了刻意练习的本质。我们明白了刻意练习不是简单的重复，而是**结合多样场景，用新思维模型替换旧大脑习惯的过程**。它需要经历从“想不到”到“不思量自难忘”的五个阶段。为了实现这一过程，我们可以利用“助推”原理，并通过**写反思日记**来创造持续的训练环境，最终将思维模型内化为直觉反应。
