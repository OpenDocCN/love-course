## 核心功能详解 🛠️

软件主要用于对人像进行美化处理，使皮肤更光滑、五官更立体。

### 1. 平滑功能
此功能用于消除皮肤瑕疵，如痘痘、斑点。

**操作流程**：
1.  选择“平滑”工具。
2.  在皮肤有瑕疵的区域进行涂抹。
3.  可重复使用，使皮肤达到理想的光滑度。

### 2. 细节功能
在平滑处理后，面部可能缺乏立体感。此功能用于增强五官轮廓。

**操作流程**：
1.  选择“细节”工具。
2.  在需要突出的部位（如鼻梁、眼睛、嘴唇）轻轻涂抹。
3.  这能使五官轮廓更清晰、更具立体感。

### 3. 重塑功能
此功能用于调整脸型或身体轮廓。

**操作流程**：
1.  选择“重塑”或“调整”工具。
2.  用手指推动屏幕，向内或向外拉伸，以改变特定区域的形状。
3.  例如，可以使脸型更对称或更瘦。

### 4. 修补功能
此功能用于复制画面中完好的皮肤区域，覆盖到有瑕疵的部分。

**操作流程**：
1.  选择“修补”工具。
2.  首先点击一处完好的皮肤作为样本。
3.  然后点击需要覆盖的瑕疵区域，样本皮肤即会覆盖上去。

关于牙齿美白等更具体的功能，我们将在后续课程中进行补充讲解。
