# 老吴《高级情商私教课》：03. 逻辑思考（一）🧠

在本节课中，我们将要学习逻辑思考的基础概念。我们将探讨什么是逻辑、为什么需要学习逻辑，以及如何将逻辑思考应用于日常生活和工作中。

## 什么是逻辑？

很多人可能会问：什么是逻辑知识？为什么要学习逻辑？这部分内容究竟要学什么？

包括我之前展示的经商课提纲，你可能会发现，为什么我在课程前面要讲思维模型、逻辑思考和刻意练习。这或许让你感到奇怪。市场上大多数情商课程都在讲沟通技巧。

这里需要讲清楚一个概念差别：**什么是知识，什么是思考方法**。我们以前在学校学的是知识。例如，学习蜘蛛有八条腿，每条腿叫什么名字，有什么功能。这些答案就是知识。

而思考方法则不同，它不在知识层面，而是在知识背后的东西。我在第一节课讲了很多大家平时很少接触的内容，目的是让大家能更好地通过现象看到本质。例如，在沟通中，如果你不了解人性，不了解自己，就无法分析到位。

所以，你在群里看到我分析问题时，会讲得非常细致。为什么你分析不了？因为你对人性不够了解，对自己也不够了解，同时逻辑和自我模型建立得不够全面。这很正常，但随着学习，我们会逐步建立这些能力。

三个月课程结束后，你再回头看自己，会发现翻天覆地的变化。如果我们无法把问题分析到位，自然很难解决问题，而且会经常碰壁。例如，你天天与人沟通、聊天、训练，但因为看不到本质，等于一直在外围打转。

你没有进入最根本的层面，所以做多少都是在浪费时间，效率非常低。但很多人喜欢在外围浪费时间，不想突破自己，不想钻研深层的东西。当你懂得深层的东西后，就会发现这些都是融会贯通的。

例如，我很会分析聊天。但以前的我也是个直男，根本不知道对方在想什么，总是从自我出发思考。我之所以变成现在这样，是因为我大量学习、分析，通过聊天回顾、复盘，从中找出思维模型和思考过程。

你要学的就是如何找出这个视角，即我是如何找到这个视角的。这是第一个点：如何找到看问题的角度。你要学会站在我的维度视角来看，就会发现“哦，原来是这样”。

找到视角还不够。第二步是从这个视角中推论出许多不同的道理，从浅层到深层。你要学会我是如何思考这些道理的。这是第二步。

想到道理还是其次，更重要的是如何证明这个道理。这是第三点：如何证明道理。我们从找到视角，到推论道理，再到证明道理，这整个过程就是从找视角到推论道理，再到证明。

这三步就是达尔文生物进化论的思考方法。听起来可能很高深，但我们已经把精华浓缩了出来。那么，逻辑是什么？请记住：**逻辑就是从找到角度，进而论述道理，进而证明道理**。

这条总脉络就是逻辑。我们学习所有知识，最重要的就是掌握底层的思考方法。这与上节的思维模型是衔接在一起的。逻辑就是从找到角度进而论述道理，进而证明道理。我们学习所有知识，最重要的就是掌握底层的思考方法。这是重点。

了解这种思考方法后，我们就要琢磨这套方法能否帮助我们在生活、工作、私人生活、甚至做生意、公司运营等各个层面应用。这就是我们要学习这门课程最重要的一点。

## 为什么需要学习逻辑？

我们常常听到别人说“学逻辑”，你不觉得这句话听起来有点奇怪吗？我觉得很奇怪。为什么逻辑要去学呢？请思考一下这个问题。

其实，逻辑是我们每一个人先天就有的。请记住这句话：**逻辑是我们每个人先天就有的**。我们常常误以为不学逻辑就是没有逻辑，这是不对的。

举个例子，像以前的原始部落，古人本来就有逻辑。例如，有几个村民去一个森林。第一个村民摘了一个蘑菇吃，结果毒死了。第二个村民也摘了同样的蘑菇吃，也毒死了。第三个村民也吃了同样的蘑菇，结果也毒死了。

这个部落的人会怎么想？他们当然会说：“这种蘑菇有毒。”这种想法原始部落的人都有。你知道这是什么吗？这叫做**归纳法**。

同样一件事情，其因果链条在不同情况中高频率重复出现，我们自然而然会把它归纳起来，认为这是一个道理：此蘑菇有毒。这种归纳法在产生作用，不用外求，也不用学，他们都懂。因为大家可以想象，我们古时候的祖先根本没有文字。

再举一个例子。你有没有发现，几乎全球每个地方的部落都有所谓的祝祷仪式？你可以理解为求神拜佛的仪式。例如，向天祈求神灵帮我们解决问题，比如把病治好。到现在，我们还有很多人去求神拜佛，祈求佛祖保佑发财、身体健康。

为什么大家都会相信这个事情？其实其中也有逻辑。我分析一下：我之所以得病，大家相信是因为冒犯了神灵，或者不够忠诚、不够信仰、不够专注。宗教经常这么说，对不对？冒犯了神灵该怎么办？所以你就得去求神，希望上天保佑。

一连串的仪式、祝祷和上天沟通，然后送上鲜花、水果等供品，希望神灵原谅、保佑。拜了一次之后，就相信病会好。相信求神拜佛能把病治好这件事，其中就有逻辑观在起作用。

大前提是：有病是因为我冒犯了神灵。所以，我向神灵沟通，让他宽恕我，神灵就会帮我把病治好。这是什么语法的雏形？这叫**演绎法**。刚刚讲的是归纳法，现在是演绎法的雏形。

所以你看，人类自古以来，甚至是原始人本身，在我们还没有知识和文字的阶段，我们其实先天就有逻辑。说到这里，你会不会有个很大的疑问：既然逻辑是我们每个人先天就有的，那为什么我还要在这里教你学逻辑？

我不是在忽悠你，让你浪费时间听我吹牛。逻辑这个东西得花功夫去学。记住重点，有两个主要原因。

### 原因一：直观答案阻碍我们启动思考

我们人这个脑袋其实很懒。只要我们对世界的疑问在脑海中浮现一个答案，我们很自然就会让脑袋停止思考。听明白这句话了吗？我们人的脑袋很懒，然后只要疑问浮现一个答案，我们就会停止思考。

举个例子。有的家长把孩子送去学音乐。有些音乐课堂会打出宣传标语：“学音乐的孩子比较乖。”身边的家长把孩子送去学音乐，是因为这个原因吗？但你想，学音乐的孩子真的会比较乖吗？

会不会是因为他本来就很乖，定得下心去学音乐？我们再设想，还有没有另外一种可能性？那就是之所以他有条件去学音乐、学钢琴，是因为他家庭条件本身就比较好。家庭条件好，代表孩子的父母教育水平本来就比较高。

我们都知道一个孩子遗传了爸爸的一半和妈妈的一半。所以我们经常说基因很重要，为什么门当户对？就是因为真的会遗传到孩子身上。这些父母受教育水平高，所以他们带小朋友会比较得心应手。

因为他们受过高等教育，知道怎么教育孩子。就算不懂，他们也会去学习。所以他们带孩子就会得心应手，比较能够把孩子教得乖一点，没那么调皮。所以你说，是学音乐让孩子变乖吗？不一定。因为你要去思考因果是不是倒置的。

所以你会发现，当我们心中启动这一连串逻辑推敲时，就会发现更多潜藏在背后的真正答案。我们要让脑袋动起来。记住，我们要让脑袋动起来。

所以为什么你们会发现，我在情商训练营群里分析问题时，不会一下子给你们答案。我一定要让你们去思考为什么。你只有逼自己去想，才会发现脑海里的东西太少，有些东西根本找不到根本。这时你再去跟人家聊，也解决不了问题。

例如，为人民服务这位同学，你聊了几个月的女生，发现有什么进步吗？进步很少，但你花了很多时间。你会发现，投入那么多精力去学习，为什么进步那么浅？这不是坚持训练多久的问题。努力会让你进步，但方法不对，进步就会非常慢。

这个问题的根本在于你脑袋装的东西还不够多。所以无论怎么努力，都是在浪费时间。所以我们为什么要学逻辑？这是第二个原因。

### 原因二：世界变得越来越复杂

为什么我们先天有逻辑，还得去学逻辑？因为这个世界变得越来越复杂。以前原始人的社会环境很简单，大家没得上网，没得手机玩，每天就是种田、养鸡养牛养羊。晚上也没有娱乐节目，最多就是篝火晚会，这已经很开心了。

压根没有我们现在生活这么丰富。我们现在的世界很复杂，生活中所有的事情其实是交叉互相影响的。我相信大家都听过蝴蝶效应。

蝴蝶效应起源于南美洲，一只蝴蝶展翅可能在地球另一端造成不知多少级的龙卷风。再举一个例子，大家都应该经历过团购。团购是怎么来的？是美国一家叫Groupon的公司上市后，因为中国市场空白，大家发现这个商业模式很好。

这时几千家公司都在做团购，最后只剩下美团。这就是蝴蝶效应：一个地方有一个事件，影响到另一个区域。我们在商场、各个方面的一些场景都是这样。你永远不知道现在这个现象究竟是因为什么。

你甚至很难用直观逻辑发现，自己的某一个行为到底会对地球另一端或其他人造成什么样的影响。再举个例子，今天中国政府发布一条政策：不准抬高房地产价格。如果你是一个互联网公司的CEO，你觉得这个政策会对你造成什么影响？

一个房地产国家管控房地产，对互联网行业的公司有什么影响？还是没有影响？可能你的反应是：房价涨了跟互联网有什么关系？这就是我们的直观逻辑，我们觉得不知道。

一开始乍想时，没法发现有什么关系。因为你怎么发现所有因果关系是如何产生的呢？所以我们得用逻辑去反复推敲。我后来学了很多商业的东西，就发现商场是一个非常复杂的地方，包括人性也是一样。

以前我们教大家用流水线作业，只要掌握了这条流水线，大家也可以像我们一样。但后来我对人的研究越来越深刻，就发现人有很多面，不同类型的人，应对方法完全不同。所有商业抉择永远不会只有一个面向的效果。

这就是为什么我们在面对事情、面对当今社会时，得去锻炼自己的逻辑。它能让我们的视角很宽、很开阔，而不是局限于一个点。你们有没有发现，很多人在思考问题时，老是盯着那个点，没办法把自己抽离出来，把东西拉开来看。

其实，逻辑能让我们通过反复推敲，对每一个行为的前因后果、影响的广度有更开阔的认识。通过更加精密的判断，你就知道了为什么要学逻辑。

## 学逻辑究竟要干嘛？

接着我们就来想一件事情：我讲了那么多，那我们学逻辑究竟要干嘛？很多人直观的理解是：我们要去学什么？刚刚讲了演绎法、归纳法，那我们是不是要去学这些？

什么循环论证、A能推导出B、非A推导出非B？不对，我不是要你们去学演绎法跟归纳法。这些属于比较高级的课程。我觉得锻炼逻辑这件事，并不是要你去学逻辑的知识。

因为你会发现，生活中包括我身边，其实有很多人学了很多逻辑知识，但说实话，逻辑不咋地。他看东西还是很片面，只针对那个点一直绕，搞得好像自己逻辑性很缜密，但又是进入了自己的世界。

这是两回事。脑袋中装了一脑袋的逻辑知识，不代表你的逻辑能力特强。逻辑其实是一种能力。请记住这句话：**逻辑其实是一种能力，不是一个知识**。

我们去学逻辑这件事，更精准的说法是：**锻炼我们先天具有的逻辑能力，让他变得更犀利、更精准**。怎么来判断这个事情？我现在问你两个问题，你自己检测一下，就可以或多或少了了解。

注意听：
1.  **能否更快速的找到信息中的脉络？** 同样是面对各种各样不同繁杂的信息，你是不是能够比身边的人更快的将他们整理归纳，甚至弄出一条脉络出来？你懂不懂这一堆信息意味着什么，有什么道理潜藏其中？如果你能够更快速、更精准地把其中的道理梳理出来，那就代表你的逻辑能力比你身边一般的朋友更强一些。
2.  **能否更快的找到道理中的漏洞？** 同样是听一段道理，你能不能比一般人更快速、更精准地找到其中的漏洞？所有的道理一定有假设的前提。请记住：**所有的道理一定要有假设的前提**。所以有些时候人家问一些问题，我说这个问题本来就是一个伪命题。因为问问题的时候，你要先思考这个问题它有一些假设的前提。

例如，外面一些很low的老师，他们跟你讲要怎么做、不能怎么做，但他压根就没有一些假设的前提。比如你们以前学过一些情感课程，可能老师就跟你讲一定要做展示面。那我就反问他一句：不做展示面就不行吗？很多东西你反问一下，他就不知道怎么回答你了。

所以，所有道理的推论，特别是我们一般口中说出的道理，一定是有推论的空隙。如果你能够比一般人更快速、更精准地找到其中的空隙或漏洞，这也代表你的逻辑能力比别人强。

包括为人民服务同学在群里发了一个视频，说导师讲在客户面前要自信，要说自己擅长的内容。这个东西就缺乏了一个假设前提。这个假设前提就是：听你讲话的这个人，要对你讲的东西感兴趣。你自信地去说才有用。人家压根对你没兴趣，你在那里巴拉巴拉讲，人家越听越难受。他没有假设前提吗？

所以这些人讲东西是很有漏洞的。包括我在看抖音，很多人在分享内容，通通都是漏洞，真正讲得好的没多少。所以你听完我这节课，再去看抖音一些人分享内容，你会发现乱讲，瞎七八乱讲，误导人。

所以你听我讲到这里，应该可以意识到，其实逻辑更多的是一种**逻辑感**。请记住这三个字：它是一种感觉。那么我们学逻辑、锻炼逻辑是在干嘛呢？举个例子，它就像我们在锻炼跑步。

我们每个人都有两条腿，都能让自己跑起来，根本不用去学习。只要是个人，稍微长大了，四肢健全，通通都能跑。那么我们锻炼跑步、学的是什么呢？
*   第一，怎样的起跑能够让你跑得比别人更快？
*   第二，跑的过程中用什么样的动作、方法，才能让你不喘气？
*   第三，如何更高效地跑到目的地？如何在同一个距离里花最短的时间、用最确定的速度、最舒服的方式跑到？
*   甚至，我们还要学怎么样跑才能避开其他一起跑步的人？因为我们常常会不小心受伤，跑步会崴到脚等等。

这就是锻炼跑步的诀窍。锻炼跑步就是了解这些注意事项、诀窍后，能够反复运用在日常跑步锻炼中，想办法把它内化成一种属于你自己的跑步能力。

## 本节总结

本节课中我们一起学习了逻辑思考的基础。我们探讨了逻辑的本质是从找到角度、论述道理到证明道理的过程，并理解了逻辑是每个人先天具有的一种能力，而非单纯的知识。我们分析了需要专门学习逻辑的两个原因：直观答案会阻碍深度思考，以及复杂的世界需要更犀利的逻辑能力来应对。最后，我们明确了学习逻辑的目的是锻炼这种先天能力，使其更快速、更精准，并初步了解了评估逻辑能力的两个关键问题。
