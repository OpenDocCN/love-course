# 浪迹《升华聊天集训营》：14.路径选择2_note

在本节课中，我们将深入探讨聊天中的路径选择策略，核心在于如何通过对话引导关系发展，并避免常见的错误。我们将分析不同回复路径的优劣，学习如何在保持自身框架的同时，有效推进对话。

***

上一节我们介绍了路径选择的基本概念，本节中我们来看看一个具体案例，分析不同回复选项背后的逻辑与风险。

把选择权交给女生后，如果她对你有好感，可能会主动说“那我们下次聊点别的吧”。你会感到舒适，因为女生懂得互动节奏。但如果她不接话，最终仍需由你说出“咱们聊点别的吧”。这便浪费了一个回合。

你试图展现的姿态可能白费，因为女生不接招。因此，你不应给予女生这种“摆姿态”的机会。这是不推荐某个选项的原因。

另一个选项是进行调侃。聪明的男生节奏很快，而直男则可能在此处纠缠不休。这属于典型的把路径带偏。

这句话完全无助于拉近两人关系。你指望女生回复什么？“你聪明吗？”然后两人开始斗嘴，争论谁更聪明。这对拉近关系毫无帮助。

所以你们为什么要选择这个路径？还好没人选它。选择它就是偏离聊天的主线和核心，这是一个典型错误。我相信一些追求“直男式幽默”的人会选这个，因为这正是我某位学员的回复。

那位直男学员认为每一句话都要给出一个能“噎住”女生的答案，以显得自己聪明、机智。这确实是直男思维的典型体现。你们回想一下，在听我讲解如何推进聊天之前，你们是否也可能这样回复？我相信有可能，因为我曾经也会这样。

我曾经认为每一句话都要展示自己的幽默和才智，结果无人回应。

现在我们来看第五个选项。这个回复其实包含了第一个选项的部分内容。

它有一个优点，在于距离“带领”对话只差一点点，只是没有明确说出来。它是在放大情绪。

它放大了感受，说话方式文绉绉，有种走心的感觉。但这一步其实有风险。

说难听点，我这么回复就是为了“博”女生说出某句话。在我的知识体系里，只要是女生主动说出来的，就代表是她想做这件事，而不是我们求着她做。这就是第五个选项和第一个选项的区别。

但第五个选项有风险。之所以敢这么回复，某种程度上需要“装个逼”，需要咬定女生对你的情绪不差，认为她是客气、礼貌、对你有基本好感的，才敢这么一搏。

话又说回来，如果现在让我再看这段对话，说实话，我不够放心。对于一个2020年5月1日版本的“升华”（指主讲人自己）来说，控制欲极强，什么事都想掌握在自己手里，只偶尔允许女生自由活动。在这种情况下，我一定会像第一个选项那样去带领。

因为我害怕出问题。我宁愿多暴露一点需求感，也要把聊天路径死死掌握在自己手里。不知道大家能否接受这个想法，这是当代2020版“升华”会有的行为。

分析了三大类案例后，你们应该记住：所有聊天路径都不能偏离核心目标。即使要绕远路，也是不得不绕，目的都是为了在推进主干关系时更加自然、舒适。

还记得我上节课随口说的一个案例吗？我说“抓到一个小可爱”，她说“我是小可爱，你岂不是大可爱”。然后我回复“你亲我一下的话，我就是大可爱了”。女生说“你这么高，我怕是要亲不到”。这时我突然绕出去了，没有继续调情，而是问“你多高”。

为什么我要问“你多高”？因为我判断，在刚刚加上微信的初期，突然进行高强度的暧昧调情可能显得虚假和急切。所以我不着急，马上切换到“身高”这个支线话题绕一下，让女生感觉我没有那么急色。聊完身高，再找机会回到调情的主干上。

这样绕一圈再回来，但原则是：能不绕就不绕。这种感觉需要大家去练习。我当时的考虑是：刚加上微信，彼此不熟，如果连续几次把聊天快速推到暧昧阶段，对后续发展不利，因为你无法每次都维持这种高强度。除非对方是很容易进入状态的人。

基于这个考虑，我感觉不应急于推进关系，可以往边上绕一绕。但一旦绕开，另一个思维必须出现：在合适的时机马上要绕回来。大家要去练习这种感觉。以后看到“好果子”不要着急吃，但也要注意别因此错过很多“好果子”。需要自己事后复盘和判断。

以下是本节课的核心要点总结：

在路径选择前，首先要准确判断女生的**情绪**和**类型**。任何路径都有优劣，都可能遇到意外情况，因此需要随时调整聊天方向。

路径能否成功，**起手只占一半，后续应对占另一半**。你说出一句话后，女生会给出回应。如果你发现回应没有按原思路走，不要放弃对聊天的控制。此时应该思考两件事：
1.  我原来的思路，现在还能否把她带回来？是否费力或勉强？
2.  如果勉强，就先跟着她聊一段，再找机会。如果还能拉回来，就尝试用别的方式聊回原路径。

不要觉得一句话没接好，后面就全崩了，也不要觉得一句话不行了，就一直在那里尬聊，那都是不明智的。

坚持选择一种路径，可以增加**个人态度**的植入，但也容易导致关系崩断。就像刚才有同学说的，想再强势一点，保持一致性。但在两人关系还不熟时，过于强势很容易导致关系崩断。昨天就有一个案例，幸好遇到一个脾气比较软的女生，不然就容易出问题。

我（主讲人）给这个互动下了定义（“下锚”），一个话题似乎结束了，但其实完全可以延续。请问，接下来该怎么聊？

最后，我们来做一个现场课堂练习，分析另一个案例。

女生说：“前任都在今天过节是吧？”（注：当天是清明节）
男生回：“行行行，就陪你们练，陪你们练，我就是个陪练。前任都在今天过节是吧？我不是这么绝情的人，我信。不然我们也聊不来。我的微信里还有前任的微信，前两天天还有一个前任加我，我也同意了。”

现在，你是这个男生，你会怎么回复？并告诉我你后续的路径选择思路。

我们来看一些学员的答案：
*   A学员：“看来你还是没放下。” （理由：先别说自己，引导她说下去，看她反驳还是顺从。）
*   B学员：“暗示女生如果跟了自己就要忘记过去。”
*   C学员：“你打算凑个麻将桌吗？”
*   D学员：“后来呢？” （不表现自己态度，引导她说。）
*   E学员：“不应该是全網無前任嗎？你倒是挺坦率。”
*   F学员：“那你应该是个比较重感情的人。”
*   G学员：“我想戳痛他。你想再重蹈覆辙吗？”
*   H学员：“我的微信里也有前任…” （然后解释自己的看法）
*   I学员：“一个合格的前任不是应该像死了一样吗？”

我们需要判断，此时应该**聊对方**还是**聊自己**？这要根据前面的聊天来判断。

男生开头试图调侃（“前任都在今天过节”），但女生非常冷静地反驳，表明自己“不是那么绝情的人”，展示了包容的态度。男生试图带节奏失败后，立刻“变节”附和（“不然我们也聊不来”），这是一种前后态度不一致。

女生的倾诉欲，源于她想纠正男生的误判，展示自己的与众不同，并期待获得认同。所以，当前应该**先聊对方**，让她的倾诉欲得到释放，营造舒适感。

在“聊对方”的选项中，**反駁**比**順從**更好，因为至少和开场时（试图调侃）的态度保持一致，可以掩盖中间“变节”的软弱。但反駁的分寸感要好，不能攻击性过强。

例如，“看来你还没放下？”如果用装傻或显得很懂的语气，女生可能会继续热衷地解释自己如何宽厚。在她倾诉一阵，获得舒适感后，你可以再表达自己不同的观点，例如：“嗯，这么说也有道理。不过我们俩截然相反，我所有分手的前任都会先进入黑名单。”

需要注意的是，“前任”这个话题本身值得深入聊吗？对于想建立短期吸引或避免争论的情况，不值得过多纠缠，应尽快自然脱手。如果想发展长期关系，可以借此了解对方感情观，但也要注意分寸，不宜让对方单方面陈述过多，以免后期难以平衡观点。

**本节课总结：**
我们一起深入学习了聊天中的路径选择。核心在于**准确判断情绪、保持自身框架一致性、灵活调整后续应对**。起手选择重要，但后续的引导和调整更为关键。要避免在关系不熟时过于强势，也要防止因害怕冲突而丧失态度。聊天的目标是拉近关系，所有路径和绕弯都应为这个核心目标服务，并努力找到最高效、最舒适的那一条。
