你总是去切别人，说明你没有什么值得别人来切你。所以好好整理一下，让自己有什么东西是值得别人来切你的。别人来切你肯定比你去切别人容易，因为你去切别人相当于去求别人。你是主动进攻的一方，而主动进攻方往往是被动的。因为你不了解对方，就像打仗一样，主动进攻别人，你不知道人家的战斗力怎么样，不知道人家会怎么回应你。

如果是别人来切你，那么你就知道她大概什么情况，主动权就可以掌控在自己手里。我也跟国外的白富美讨论过这个问题，她们也说，如果很喜欢一个男生，会把这个男生能翻的东西都翻出来。所以我们要把有利的那些东西呈现出来。

我有些聊天记录，那些女生兴趣度特别高，高才生，比如哥伦比亚大学的都有，长得很好看，家里很有钱，我们就是在国外认识的。

有一天晚上，她来私聊我说你们走了没有，找不到我们。我就跟大家讲等一下，他们还有两三个小伙伴。我们在车上等了快一个小时，接上他们一起回酒店。然后我们约好第二天一起出发。第二天晚上我们准时集合走。
