# 12、冬瓜《校园把妹》：第三课校园撩妹和社会撩妹的差别_note

在本节课中，我们将要学习校园撩妹与社会撩妹的核心区别，并重点分析在校园环境中吸引女生的关键筹码。理解这些差异，能帮助你在校园里更有效地与心仪的女生建立联系。

古语有云：知彼知己，百战不殆。既然目标是校园里的女生，她们与社会上的女性存在显著差异。这好比行军打仗前需要对战役进行评估，了解伤亡率和成功率。撩妹也是同样的道理。

我们需要清晰地认知一个核心概念：**自我筹码**。你可以将其理解为**高价值展示**，或是**建立奖品性**。那么，筹码具体包含哪些呢？根据多年经验，我认为男人吸引女人的筹码主要分为三类。

以下是三类核心筹码的详细说明：

*   **第一类：身材外貌**
    身材外貌在很大程度上是天生的、不可逆的。我们基本无法改变身高、骨架、相貌或某些生理曲线。当然，得体的穿着打扮可以在一定程度上改进或掩饰这些先天不足。

*   **第二类：财富地位**
    财富与地位虽然可以通过后天努力获得，但这通常需要长时间的努力和机遇。例如：**财富**、**权势**、**社会地位**等。

*   **第三类：浅沟通（人格魅力）**
    浅沟通，你也可以理解为人格魅力。这类特质可以在相对较短的时间内有效地打造出来。例如：**自信**、**幽默**、**口才**、**身体语言**、**社交力**等。

以上三点是吸引异性的主要特质表现。如果你想在社会上“玩耍”一些比你等级高的姑娘，我认为三者缺一不可。请注意，这里指的是“玩耍”比你等级高的，而非向下碾压。

上一节我们介绍了三类通用筹码，本节中我们来看看校园环境的特殊性。我们身处校园，我相信很多兄弟并不只想找一个普普通通的姑娘谈恋爱，然后无疾而终。大家都希望找一个能“领得出去”、让自己有面子的对象。这类对象就是我们理解的校园里的“高分”女生，或者说，是比你等级要高的。

那么，校园撩妹与社会撩妹的核心差别是什么呢？关键在于**第二类筹码——财富地位，在校园里显得不是尤为重要**。

原因很简单：在大学期间，大多数同学的生活费水平是相近的。因此，我们只需要做好以下两点：
1.  打造一个看起来还不错的外形（这一点在前几节课已讲解过）。
2.  学会并运用与女生进行“浅沟通”的能力。

关于第一点外形打造，我们之前已经讲解过，这里不再展开。现在，我们重点探讨“浅沟通”如何学习和实践。

浅沟通主要通过两种方式习得：

*   **第一，源源不断地有意识培养这些特质**，从内到外彻底改变自己，实现脱胎换骨。这也是我们常说的**自我提升**。
*   **第二，通过使用惯例技巧**，让自己立刻伪装出那些特质。例如**三秒法则**：在搭讪女生时给自己设定目标，从看到她到采取行动必须在三秒内完成。又比如，向女生讲述你事先编好的故事，只要将故事带入你的节奏，情节跌宕起伏就能吸引她。

但我想强调的是，一旦这些技巧惯例用完，你的原本面目还是会浮出水面。因为你没有通过自我提升，将展示出来的特质真正内化到自己身上。甚至一些聪明的女生在你们见面之前，就会通过网上聊天来测试你，看你是否真的如你所说是个优质男人，是否**保持前后一致性**。

可能上面的知识点有些兄弟听得不太明白。没关系，你只需要记住两点：
1.  在女人面前**保持前后一致性**。
2.  对于惯例技巧这类“拿来就能用”的东西，千万不要一味地复制粘贴。我们首先要尝试理解这些技巧背后的含义，然后将其内化到自己的思维中。
