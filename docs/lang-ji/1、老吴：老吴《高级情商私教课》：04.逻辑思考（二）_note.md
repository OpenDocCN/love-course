# 老吴《高级情商私教课》04：逻辑思考（二）🚀

在本节课中，我们将要学习如何启动并运用逻辑思考。我们将探讨大脑的运作模式，理解感性直觉与理性思考的关系，并掌握两个核心工具来激活逻辑思维，从而更清晰地分析问题、接近事实真相。

## 大脑的两种模式：感性直觉与理性思考

上一节我们介绍了逻辑思考的基本概念，本节中我们来看看如何启动它。首先，我们需要理解大脑的组成架构。为了便于理解，我们可以将其粗略地分为两个部分。

一部分是**理性思考**，也就是我们所说的逻辑思维。例如，通过观察“三个村民连续吃同一种蘑菇后中毒死亡”而归纳出“这种蘑菇有毒”，这就是一种逻辑思维。

另一部分是**感性直觉**。它指的是我们遇到任何事或任何人时，脑海中第一时间产生的、最直观的感受或答案。例如，“我一看到某人就觉得他很讨厌”，这就是一种感性直觉。

## 为何需要主动启动逻辑思考？

我们每个人天生都具备逻辑能力，但为什么还需要学习如何启动它呢？关键在于人脑的运作顺序。

**人的大脑总是先启动感性直觉，然后再决定是否启动理性思考。**

例如，当你对某人产生“他很讨厌”的第一印象后，往往会直接屏蔽他，停止进一步思考。你不会去分析他为什么讨厌、他为什么那样穿着。感性直觉一旦做出负面判断，理性思考通常就不会被启动。

这会导致我们处于一种不开放的心态，无法认真聆听和分析信息，从而可能错失真相或做出错误判断。特别是在人际交往中，如果对方对你的第一印象（感性直觉）不佳，无论你后续沟通多漂亮，对方都可能拒绝接受。

因此，我们必须有意识地训练自己，在感性直觉之后，主动启动理性思考。

## 启动逻辑的核心工具：两板斧 🪓

为了时常保持理性思维的运转，我教大家一个简单有效的方法，称为“两板斧”。

**每当你接收到一个信息或面对一个判断时，都先问自己两个问题：**
1.  **为什么？**
2.  **凭什么？**

以下是具体运用方法：

假设有人对你说：“我们家的公司，资金老雄厚了。”
1.  你的感性直觉可能立刻反应：“他好有钱！”
2.  此时，启动逻辑，问第一个问题：**为什么**你说公司资金雄厚？
    *   对方可能回答：“因为我们有几千万的现金储备。”
3.  不要停在这里，继续问第二个问题：**凭什么**证明你有几千万现金储备？
    *   对方可能出示银行流水。
4.  继续深入，再问：**为什么**拥有几千万现金储备就算资金雄厚？
    *   对方可能解释：“这够公司不干活也能运营一年。”
5.  再次追问：**凭什么**说能养活公司一年就算很厉害？如果竞争对手有几个亿现金呢？

通过不断交替使用“为什么”和“凭什么”进行追问，你会层层深入，越来越接近事实的真相，而不是停留在表面的感性判断上。人类所有的知识体系，其实都是在这样不断的“为什么”和“凭什么”的追问中构建起来的。

## 逻辑思考的四个方向 🧭

拥有了“为什么”和“凭什么”这两把思考的“刀”之后，我们还需要注意“砍”的方向。如果方向不确定，即使不断挥刀，也可能只是在原地打转。

我们在运用逻辑思维时，大致有四个不同的思考方向：

1.  **事实判断**：关注“是什么”。事情是否真实？证据是否确凿？
2.  **功利判断**：关注“有什么用”。权衡利弊得失，计算成本与收益。
3.  **价值判断**：关注“该不该”。是否符合道德、伦理或个人的价值观。
4.  **审美判断**：关注“喜不喜欢”。是否符合个人情感上的偏好与感受。

**举例说明**：一个女孩接到刚认识的男性的午夜电影邀约。
*   **事实判断**：他身高真有1米8吗？真的有八块腹肌吗？
*   **功利判断**：我今晚要加班，如果去看电影，工作报告完不成怎么办？
*   **价值判断**：刚认识就答应看午夜场，会不会显得我不够矜持？
*   **审美判断**：我内心真的喜欢他吗？和他相处感觉愉悦吗？

思考时，我们需要**先设定一个方向，并在这个方向上用“为什么”和“凭什么”想通想透**，然后再切换至另一个方向进行同样深度的思考。避免在几个问题间跳跃却都没想清楚，那样只会陷入思维的混乱。

当多条路都想通之后，最后一步就是根据自己的情况进行**权衡**，做出最终抉择。

## 思考应该停在何处？⏹️

“为什么”和“凭什么”可以无限追问下去，最终可能进入哲学范畴（例如：“人为什么要活着？”）。但我们学习逻辑思考的主要目的，通常是为了解决实际问题。

那么，思考应该停在何处？这里有两个实用的判断标准：

1.  **答案已能指导行动**：当前层次的答案，已经足够让你做出明确的行为决策。
2.  **风险与收益可承担**：你对这个决定可能带来的最好与最坏结果都有了清晰的了解，并且能够承担相应的风险。

只要满足以上两点，你的思考深度就已经足够支撑你解决问题了。对于日常决策，不必无休止地追问；但对于想要锻炼思维深度的事情，则可以多花时间深入思考。

---

本节课中我们一起学习了如何启动逻辑思考。我们认识到大脑会优先启动**感性直觉**，因此需要有意识地调用**理性思考**。我们掌握了“**为什么**”和“**凭什么**”这两把核心工具来层层剖析问题。同时，我们了解到逻辑思考有**事实、功利、价值、审美**四个方向，思考时需要先定方向、再深挖，最后进行权衡。记住，思考的终点是能**做出决策**并**承担其后果**。不断练习这套方法，你的逻辑思维会越来越清晰、有力。
