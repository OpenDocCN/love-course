# 恋爱规则课程01：第十八集. 不要轻易定义女生 💎

在本节课中，我们将要学习一个核心原则：在追求女生时，切勿仅凭外在形象（如外貌或身材）去判断她的价值或追求难度，也不应草率定义她的三观与价值观。

## 外在并非唯一判断标准

上一节我们强调了避免以貌取人的重要性，本节中我们来看看为何这种判断方式常常出错。

长相并非评判一个人的唯一要素。许多男性见到女生容貌出众、气质优雅，便下意识认为她难以追求。然而，这类女生可能私生活混乱或经济拮据。人们常因看到他人优秀的一面，便**错误地推断**其所有方面都同样优秀。

## 原石与开窗的比喻

以下是理解此概念的一个生动比喻。

我们可以将了解一个人的过程，比作鉴别翡翠原石。翡翠蕴藏在粗糙的原石内部。有时，人们会在原石上切一刀，即“开一个窗”，并仅凭这一小窗口显露的质地来判断整块翡翠的价值。

这种做法并不可靠。因为这一刀可能恰好切在了原石**最精华的部分**，窗口之下实则空无一物。反之，这一刀可能切在了普通处，但继续深入挖掘，却发现内部是真正的宝藏。

## 应用于识人的核心要点

基于以上比喻，以下是识人时应遵循的要点。

1.  **外表具有欺骗性**：许多女生外表光鲜亮丽，内心却可能并不善良；许多女生外表朴实无华，内心却单纯善良。
2.  **需要全方位观察**：你必须**投入耐心**，从多角度、长时间去观察了解一个女生，而非仅凭外在印象就掏心掏肺、义无反顾地付出。
3.  **内在品质的持久性**：美好的外在，其“保值期”至多二三十年。到了四五十岁，再美丽的女性也会衰老。但一个人的品德、德性与三观，却会持续一生。因此，你的选择应当基于这些持久的内在特质。

## 核心行动建议

浪哥给出的最终建议，可以总结为一个简单的行动公式：

**追求过程 = 切开观察 (深入接触了解) ，而非 外部溜达 (仅凭表面判断)**

这就像切原石，你需要切开它，看到内部，而不是只在外面随意看看就下定论。
