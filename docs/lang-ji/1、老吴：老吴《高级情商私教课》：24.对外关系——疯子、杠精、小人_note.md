# 1、老吴：老吴《高级情商私教课》：24.对外关系——疯子、杠精、小人_note

在本节课中，我们将要学习如何处理三种具有挑战性的对外关系：疯子、杠精和小人。我们会探讨他们的行为特征、背后的心理原因，并学习具体、实用的应对策略，帮助你在生活和工作中更好地保护自己，维持内心的平和与能量。

## 第一部分：不与“疯子”互动 🚫🤪

上一节我们介绍了情商在自我关系中的应用，本节中我们来看看如何应对外部世界中的“疯子”。这里的“疯子”并非医学定义，而是指那些深陷痛苦、失去理智、具有攻击性和毁灭倾向的人。

一个典型的例子是重庆公交车坠江事件。一名女乘客因错过下车点与司机激烈争吵，导致司机情绪失控，最终车辆坠江，无人幸存。这类因个人情绪失控而引发极端后果的事件，在生活中并不罕见。

我们的世界是阴阳对立的，有人活在爱与光明中，也有人活在黑暗与痛苦中。人的状态没有绝对的好坏对错，只有能量高低之分。低能量状态下，人的言行往往带来负面影响。

根据之前的课程，人的痛苦状态可分为五种。其中一种称为“无明之人”，即对自己身处痛苦毫无察觉。当一个人无法意识到自己的痛苦时，就会产生攻击性和报复性，将自身痛苦归咎于他人，希望他人为自己的痛苦负责。

面对这样的人，我们应该怎么做呢？

以下是核心应对原则：

*   **人是无法被改变的**：我们最多只能影响他人，无法改变他人。切记这一点。
*   **我们有选择权**：我们可以选择让什么样的人进入自己的世界，也有权让“疯子”从自己的世界中消失。
*   **不要讲道理或纠缠**：当对方处于“上头”的失控状态时，任何道理和争辩都无效。

因此，当你身边出现“疯子”时，唯一要做的就是**转身离开**，或尽快让其从你的视线中消失。这并非冷漠，而是自我保护。

世界上没有无缘无故的事。如果有“疯子”出现在你身边，往往是你自身内在能量状态的一种外在显化。当你能量高时，会吸引高能量的人事物；当你能量低时，就容易吸引“疯子”。

例如，当你状态不佳时，可能会打到服务很差的出租车。此时，与其发脾气，不如静心觉察：**“是我现在的磁场状态不好，才吸引了这样的经历。”** 意识到这一点，就能平复情绪，避免陷入无谓的冲突。

## 第二部分：理解与应对“杠精” 🦹♂️💬

理解了“疯子”的不可理喻，本节我们来看看另一种常见角色——“杠精”。杠精是指那些喜欢抬杠、争论，常常陷入偏激立场的人。

在互联网时代，我们尤其容易遇到偏激的人，也容易让自己变得偏激。偏激源于立场、价值观不同所引发的激烈情绪。互联网基数庞大，无论你的喜好多么小众，都能找到“同温层”——一群与你观点一致的人。

长期沉浸在“同温层”中，会带来一个问题：**你会误以为自己的认知就是整个世界**。例如，一个女生因被少数男人伤害，就认定“天下男人没一个好东西”。如果她带着这个信念，就会不断吸引和注意到符合这个信念的“坏男人”，形成恶性循环。

互联网加速了我们找到小团体的过程，但也让我们更容易陷入“团体极化”。在群体中，为了获得认同，言论可能走向极端。例如，在视频弹幕中，骂得越狠、越恶毒，有时反而能获得更多喝彩。

那么，我们该如何在充满偏激言论的网络世界中保持平衡呢？

关键在于为自己建立一个“水平仪”。就像专业三脚架上有个小气泡来判断是否水平一样，在互联网上，你需要保持内心的那个“小气泡”。**即使外界地面是倾斜的，你也要确保自己的言行没有随之倾斜。**

当你遇到偏激的“杠精”时，可以将其视为一种提醒：

*   **提醒你世界的多样性**：这个世界充满了不同的价值观。
*   **提醒你需要沟通与妥协**：完成事情往往需要与不同观点的人沟通妥协。

面对杠精，切记：**不要跟他们争对错、论输赢**。这没有意义，只会消耗你的能量。以巴黎圣母院火灾事件为例，不同的人展现了至少九种不同的反应流派，从祈福到嘲讽，这正说明了价值观的多元。高情商的人，能够看清并包容这种不同，不会因此陷入激烈的情绪对抗。

## 第三部分：识别与应对“小人” 🐍🛡️

看过了情绪失控的“疯子”和观点偏激的“杠精”，最后我们来探讨最需警惕的一类人——“小人”。老祖宗说“可怜之人必有可恨之处”，这提醒我们不要扮演“受害者”角色。凡事有因有果，我们的境遇往往与自身的选择和能量相关。真正的创造者会为自己的人生负责。

在工作中，我们尤其需要学会识别小人。鬼谷子曾总结八种小人，结合现代语境，我们可以关注以下几种特征：

以下是常见的小人行为特征：

1.  **见人说人话，见鬼说鬼话**：两面派，态度因人而异。（需与高情商的灵活沟通区分，后者是知人而善言，前者是毫无原则的迎合。）
2.  **搬弄是非，造谣生事**：习惯用“听说”开头，通过贬低、丑化他人来衬托自己，或设计陷害竞争对手。
3.  **挑拨离间，暗箭伤人**：唯恐天下不乱，制造矛盾，事后又假装和事佬。
4.  **善於撇清责任，栽赃同事**：自己出错或胡说时，拉别人背黑锅。
5.  **落井下石，踏血晋升**：对同事的挫折幸灾乐祸，说风凉话，甚至趁机打压。
6.  **不懂感恩，翻脸无情**：你帮助他成功后，他非但不感恩，还可能反过来抹黑打压你。
7.  **阳奉阴违，见风使舵**：吃苦时不见人影，邀功时冲在最前；善于拍马屁、打小报告，套取秘密背后陷害。

了解了小人的特征，我们该如何应对呢？硬碰硬并非上策，需要策略。

以下是四个核心应对方法：

1.  **学会适当反击**：一味忍让会让小人得寸进尺。在掌握足够证据和底气时，可以果断反击，**“以其人之道还治其人之身”**。但切记，反击前要隐忍，收集证据，等待时机。
2.  **保持安全距离**：这是应对小人的基本原则。不主动亲近，不拒绝必要的表面接触，但绝不深交。避免让他们知道你的隐私和软肋。**“不主动，不拒绝，不深交”**。
3.  **搞好核心关系**：与其耗费心力与小人缠斗，不如花时间与正直的同事和领导建立良好、信任的关系。当你的人际根基稳固时，小人的谗言就很难动摇你。
4.  **相信因果与时间**：小人往往得意一时。多行不义，终将自食其果。有时不是你消灭他，就是环境或其他人会淘汰他。保持自己的正念和成长，是长远之计。

本节课中我们一起学习了如何应对三种棘手的人际关系类型。面对“疯子”，我们要果断远离，保护自身能量；面对“杠精”，我们要保持内心水平，不陷入无谓争论；面对“小人”，我们要学会识别、保持距离、稳固自身并适时反击。记住，**情商的高阶应用，不在于改变他人，而在于清醒地认知环境，智慧地选择应对方式，最终保护好自己内心的宁静与力量。** 希望这些方法能帮助你在复杂的人际交往中更加从容、坚定。
