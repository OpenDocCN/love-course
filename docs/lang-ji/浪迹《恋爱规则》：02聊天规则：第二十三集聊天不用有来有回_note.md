# 浪迹《恋爱规则》课程02：聊天规则 🗣️  
## 第二十三集：聊天无需有来有回 🔄

在本节课中，我们将学习一个基本聊天规则：聊天无需有来有回。通常90%的情况下，男性应作为最后回复的一方，但偶尔也有例外。不回复女性信息，除了可能让她觉得不够礼貌外，基本上没有其他坏处。

不仅如此，通过不回复信息，你还能在聊天中建立三点优势。

---

上一节我们介绍了聊天无需有来有回的基本概念，本节中我们来看看具体能带来的三点优势。

以下是三点优势的详细说明：

1.  **占据聊天主动**  
    当你不回复她的信息，之后再开场与她说话时，代表是你有空的时候。你永远主动开启话题，而不是被动等待她的回复再去回应。这让你在聊天中占据主动。

2.  **让她患得患失**  
    当你不回信息时，女性可能会思考自己是否说错话、你在做什么、你是否与其他女性在一起。她在思索这些问题的过程，就是在对你进行情感投资。即使你没有说话，你的形象却占据了她一整天，这相当于你与她进行了一整天的交流。

3.  **判断她对你的态度**  
    如果你不与她说话，她开始追问你的信息，说明她对你有兴趣。反之，每次你不说话，她也不说话，则说明你们的对话仅依赖于你的主动开场。

---

通过不回复女性的信息，我们可以在聊天中判断女性对我们的兴趣程度，掌握聊天主动权，甚至让她产生患得患失的情绪。这是一种有效的聊天策略。

本节课中我们一起学习了如何通过“无需有来有回”的聊天方式，来建立优势并判断对方兴趣。掌握这一规则，有助于你在互动中保持主动。
