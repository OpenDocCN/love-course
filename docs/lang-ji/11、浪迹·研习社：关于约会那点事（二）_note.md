# 11、浪迹·研习社：关于约会那点事（二）_note

在本节课中，我们将要学习约会中的具体操作方法，重点探讨约会时应该聊什么内容以及如何通过整体表现来吸引对方。

上一节我们介绍了约会的本质与核心思路，本节中我们来看看如何将这些思路转化为具体的、可执行的约会操作。

## 约会操作的两大核心

约会中的具体操作主要围绕两件事展开：第一是约会时聊什么话题，第二是选择什么地点进行约会。以下是具体可执行的方法。

## 聊天的真正作用与误区

80%的男性认为约会失败的原因在于不知道如何与女生聊天。其实并非如此。约会失败的主要原因往往不是不会聊天，而是没有合理地展现自己，或者没有向女生传递出能够吸引她的价值观。

例如，你是一个上进的人，你需要告诉女生，你们在一起后你能保护她、给她美好的未来。这些信息需要在约会时，通过语言或实际行动传递出去。聊天是传递这些信息的重要方式，但有一个关键区别。

在微信等网络聊天中，**聊天内容的权重高达90%**，因为对方只能看到文字。而在现实约会中，情况完全不同。

**现实约会吸引力公式**：
`约会整体吸引力 = 30% 聊天内容 + 70% 非语言信息`

这里的“非语言信息”包括你的肢体语言、眼神、行为、说话方式和语速。聊天内容本身，只要有一个大致的友好框架即可。

回想你过去的恋爱经历，你很可能记得第一次约会做了什么，但绝不可能记得当时说过的每一句话。你可能只记得一两句关键的话，比如“你真漂亮”或“你的手好软”。如果真的是靠你说的话吸引了对方，你应该会记得，但事实是我们都记不住。这说明，在真实的约会相处中，帮助你吸引女生的是你作为一个人的“整体”。

## “整体”才是关键

女生看待男性，不是割裂地看鼻子是否好看、五官是否帅气，而是看一个综合的“整体”。这个整体包括：
*   发型
*   五官
*   皮肤
*   身高
*   穿衣打扮
*   眼神气质
*   肢体语言

当你坐在女生面前，她接收到的是以上所有点综合起来给她的“感觉”。如果这个整体感觉不好，你说任何话都难以补救。因此，在约会中，你的言语内容只占30%的影响力。

## 聊天内容的核心原则

既然整体感觉占70%，那是否意味着可以不用说话？理论上，如果你的整体表现能做到100分，你确实可以不用主动说话，因为女生会主动开启话题。但通常我们的整体表现可能只有70分，那么就需要通过聊天来弥补分数，达到及格线以上。

那么，我们应该说什么内容呢？核心原则是：**支持、鼓励与赞美**。

与任何人相处，包括和女生约会，在见面聊天时，你都应该更多地支持、鼓励和赞美对方。特别是在约会开始的前十分钟，聊天内容应该以真诚的赞美为主。

## 如何进行真诚的赞美

真诚赞美的关键在于“言之有物”。你不能空洞地说“哇，你好漂亮”。你需要指出具体的细节。

以下是一个言之有物的赞美示例：
> “你今天左边耳朵戴的耳环挺好看的。你觉得你右边耳朵的耳环和左边的相比，哪一个你更喜欢？我个人更喜欢你右边戴的这款。”

通过关注并赞美一个具体的细节，你传递出的关注和欣赏才是真实可信的，这远比一句泛泛的夸奖有效得多。

## 本节课总结

本节课我们一起学习了约会中的具体操作方法。我们明确了约会操作的两大核心：聊天内容和约会地点。我们纠正了一个常见误区，即约会失败主因不是不会聊天，而是没有展现整体吸引力。我们通过一个公式强调了在现实约会中，非语言信息（肢体语言、气质等）比聊天内容更重要。最后，我们确立了聊天内容的核心原则是真诚的、言之有物的赞美，并给出了具体的操作示例。记住，约会是你整体形象的展示，而不仅仅是语言的交锋。
