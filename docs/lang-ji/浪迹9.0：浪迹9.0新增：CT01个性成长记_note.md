# 浪迹9.0课程09：CT01个性成长记 🧭

在本节课中，我们将要学习如何科学地分配个人时间，以实现有效的个性成长。课程将围绕工作、生活与社交的平衡展开，帮助你建立清晰的时间管理框架，从而提升生活品质与个人魅力。

## 理解时间构成 ⏳

上一节我们介绍了课程概述，本节中我们来看看个人时间的构成。每个人每天除了8小时工作，剩余时间主要包括社交聊天、用餐和购物等活动。许多人并不清楚如何分配这些时间。

## 核心时间分配法则 📊

以下是三个核心的个人时间分配板块，你需要为每个板块规划出专属时段。

1.  **社交聊天时间**：每天应规划4小时用于社交。这4小时用于与朋友或异性进行线上或线下的正常交流。合理分配社交时间是应用教学理论的基础。不会分配时间会导致生活混乱，缺乏目标。

2.  **购物与期待时间**：每天应安排时间接触新事物，例如拆快递包裹。拆包裹前的期待感和拆开时的喜悦能改善心情。这种对新鲜事物的期待是积极情绪的源泉。

3.  **用餐与品质时间**：无论是与朋友还是异性共进餐食，都应规划时间并懂得享受生活品质。每一餐都是改变自己的机会。用餐方式的选择，例如是粗糙应付还是像吃西餐般细致品味，直接反映了生活品位的差异。

## 假期与个人安排规划 🗓️

在规划好日常时间后，我们来看看如何安排假期与个人活动。首先重点是社交活动，无论是与朋友还是异性相处，都应在有个人安排的前提下进行。

以下是几种个人活动安排的建议：
*   第一，可以选择去击剑。
*   第二，可以选择去打保龄球。
*   第三，可以选择去打高尔夫。

每个人对个人时间的规划规范不同。人们常忽视时间的真正效用。每天仅有24小时，忽视任何一小部分，短期可能无感，但长期积累会导致许多想做的事未能完成。例如，你是否尝试过攀岩？是否体验过骑马？许多人只有想法，却未邀请朋友参与或将计划付诸实践，这实际上造成了时间的浪费。

## 个人建设与习惯调节 💪

接下来，我们探讨个人建设与不良嗜好的调节。首先看运动建设，很多人对自身建设投入的时间很少，认为健身太普遍或太累，难以坚持。另一些人则坚持每日健身。

对此，有一个个人见解：从家到健身房通常至少需要10分钟。如果你能利用这10分钟在家进行徒手健身，可能比去健身房更能让你坚持下去。如何开展徒手健身？你可以向我索取一份徒手健身资料。

第三点是调节个人不良嗜好，核心在于“突破”。你曾突破过什么？提供一个建议：你敢不敢穿很花哨的衣服？如果敢，这就是一个小小的突破。第二，你是否尝试过在街上用三脚架自拍？这也是生活中的一种小突破。如果连这些细节都不敢突破，就谈不上成长。

因为成长的内涵包括：若想成长，就要去做一些自己从未敢做的事。例如：
*   第一，去蹦极。
*   第二，在朋友们面前跳一段即兴的尬舞。比如在夜店时，你敢站在桌面上尽情尬舞，让朋友们对你刮目相看。这些事你以前可能没做过，但成长后，你可以想到就立刻去做。

## 付诸行动的真正改变 🚀

最后，我们讲第三点：改变。若想真正改变自己，必须付诸行动。

以下是两个关键行动点：
1.  **去搭讪**：主动开启与陌生人的交流。
2.  **在聊天中分享动态**：例如，你可以自然地发一条消息说“我今天才忙完工作回到公司”，并附上一段简单的自拍视频。过去你可能只发朋友圈，但现在你可以在聊天中直接分享给对方。因为在改变的过程中，你往往会收获一个不一样的自己。
