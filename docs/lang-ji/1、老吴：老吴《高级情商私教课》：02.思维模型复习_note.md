# 老吴《高级情商私教课》02：思维模型复习 🧠

在本节课中，我们将复习思维模型的核心概念、学习方法与应用技巧。思维模型是提升决策质量与创新能力的关键工具，掌握它们能帮助我们更清晰地认识世界并做出更优选择。

## 一、思维模型的作用

上一节我们介绍了思维模型的重要性，本节中我们来看看它的两个核心作用。

### 1. 提升决策质量与正确率

我擅长沟通的原因在于我掌握了大量思维模型。积累更多思维模型能提高我们决策的质量和正确率。

例如，当有人在路上打了你一巴掌，多数人的直接反应是反击。这种从“被打”到“打回去”的反应，可以归纳为 **ABC模式**：
*   **A (Antecedent)**：诱发刺激（原因）。
*   **B (Belief)**：信念反应（你的想法）。
*   **C (Consequence)**：行动结果。

公式：**A -> B -> C**

由于每个人的信念（B）不同，导致行动（C）也不同。有人选择隐忍，有人选择反击，还有人会害怕另一边脸也被打。信念由原生家庭、过往经历、DNA等多种因素构成。我们常说“江山易改，本性难移”，指的就是人固有的性格模式。

所谓的“笨”，是指**遇到类似情况，每次都采用相同行为，却期待产生不同结果**。了解ABC模式后，我们明白许多错误决定源于思维模型单一。我们需要用新的思维方式替换旧的，实现认知升级。

一个实用的原则是 **FFC赞美法则**：
*   **F (Feeling)**：表达感受。
*   **F (Fact)**：描述事实。
*   **C (Compare)**：进行比较。

例如，当女生问“你为什么喜欢我？”或“我比不上你前女友”时，她可能缺乏安全感，需要获得“自己是特别的”这种信心。运用FFC可以这样回应：
1.  **感受 (Feeling)**：“和你在一起我感觉最开心、最特别。”
2.  **事实 (Fact)**：“比如上次纪念日，你亲手做了……（具体事例），这让我印象深刻。”
3.  **比较 (Compare)**：“你比我之前遇到的人都更细心。”

这样比单纯说“你是最美的”更有说服力，因为它提供了具体的依据。

### 2. 提升创新能力

思维模型的第二个作用是提升创新能力。我不希望教出来的人和我一模一样，创新需要独特的思维组合。

思维模型是有层次的，像一个金字塔：
*   **底层**：即学即用的工具型模型（如FFC、SWOT、AARRR运营模型）。
*   **中层**：经过验证的方法原理（如4P营销理论）。
*   **高层**：解释性强的抽象理论（如进化论）。

越往下，工具属性越强；越往上，解释属性越强。学习中高层的思维模型能帮助我们理解事物发生的深层原因，拓展认知边界。

如今许多“套路”已失效，因为大家都在用相同的方法。学习的关键在于掌握**创新的思维模型**，通过积累和组合不同的模型，形成自己独一无二的思维网络，这才是真正的竞争力。

## 二、如何发现与学习思维模型

理解了作用，我们来看看如何发现和学习思维模型。核心方法是 **“问思模”**：从问题出发，经过思考，提炼出模型。

### 1. “问思模”三步法

以下是“问思模”的具体步骤：

*   **问 (Question)**：从生活场景中发现问题。例如，我以前不擅长沟通，与陌生人在一起会尴尬。但遇到一位情商很高的老师时却不尴尬。问题来了：为什么和他在一起不尴尬？不是因为他口才好，而是他“很会打开话匣子”。
*   **思 (Thinking)**：分析问题本质。不成熟的人常以自我为中心，只关注“我要什么”。成熟的沟通包含三个要素：**自己、对方、信息**。视角从单一的“我”转变为三者互动，思考维度就提升了。
*   **模 (Model)**：提炼底层思维模型。追问上述思路与哪些底层模型相关。例如，这里涉及到 **“转换视角”** 模型：人性以自我为中心，但我们可以反其道而行之，**始终以对方为中心**进行沟通。

举例：助教想让学员提交学习目标。
*   **以自我为中心**：“请把你的问题发给我。”（强调“我”需要）
*   **以对方为中心**：“为了更好记录你的成长，帮助你在课程中跟进目标、更轻松地达成结果，我们需要记录你的目标。”（强调“对你有利”）

后者满足了对方的自尊心和分享欲，效果更好。我们要学会满足他人的心理诉求。

### 2. 学习时的三个关键转变

除了“问思模”，在学习过程中还需要完成三个转变：

*   **心态转变**：保持**好奇心**。不要停留在“他很厉害，我不行”的表面，要主动探索“他为什么厉害”背后的原因。
*   **重点转变**：学习重点不是死记硬背答案，而是花时间在**思考过程**上。多问“为什么”，琢磨逻辑。
*   **方法转变**：不要停留在具体答案上，要注重**知识之间的联系**。不断将新答案与过去经验结合，探寻更深层的原因。

例如，约女生时说“约吗”，有人被拒绝，有人成功。为什么？
1.  第一层思考：说什么不重要，**是谁说**很重要。
2.  第二层思考：因为说话的人对女生有**吸引力**。
3.  第三层思考：吸引力来源于外表、聊天方式、经济实力等。

不断深入追问，才能找到根本原因。高手擅长将不同模型连接起来，形成**多元思维模型**。查理·芒格的“格栅理论”指出，要在不同学科的思维模型之间建立联系。就像碳原子因排列方式不同，可形成石墨或钻石。我们需要不断学习和应用，让思维模型产生联动。

### 3. 向高手学习：五星笔记法

如果自己难以发现模型，可以向更厉害的人学习。关键不是记录他说了什么，而是思考他背后的逻辑。推荐使用 **“五星笔记法”** ，在听课时问自己五个问题：

1.  老师讲的核心内容是什么？
2.  老师讲课的逻辑是什么？
3.  我受到了什么启发？有什么疑问？
4.  这些启发和疑问背后的思维模型是什么？
5.  哪些事情我可以立刻采取行动？

通过这个方法，你能更快吃透内容，关注高手解决问题的方法而非仅仅答案本身。

## 三、如何应用多元思维模型

学习之后，关键在于应用。本节我们将探讨如何让多元思维模型落地。核心是 **“多层次”** 与 **“多视角”** 分析。

### 1. 多层次分析

**多层次**是指用不同层次的思维模型分析同一个问题。

以“中国足球为什么进不了世界杯？”为例：
*   **道德层次**：球员不努力，生活不自律。
*   **商业/体制层次**：联赛市场化问题，体制不完善。
*   **数学/战术层次**：足球是11人的运动，最优战术是“去中心化的三角形网络”（如巴萨的踢法），而我们战术落后。

从一个问题出发，运用道德、商业、数学三个不同层次的模型进行分析，理解会更全面。

### 2. 多视角分析

**多视角**是指在同一个层次内，用不同角度的模型进行分析。

以分析“拼多多为何崛起”为例（借鉴产品经理梁宁的分析）：
*   **微观视角（用户体感）**：抓住人性贪便宜的心理，运营手段强悍。
*   **中观视角（行业生态）**：
    *   **供给侧**：淘宝清退低端供应商，这些供应商需要新平台，拼多多调性与之匹配。
    *   **用户侧**：智能手机普及，低价购物流行。
    *   **竞争侧**：淘宝、京东上市后注重品牌形象，放弃部分低端市场，留下空白。
*   **宏观视角（商业理论）**：运用“第二曲线”、“网络效应”等模型分析。

很多人只看到“拼多多卖便宜货+微信裂变”的表象。通过多层次、多视角分析，我们才能更接近真相。每个事物背后都有复杂的因果和关联，掌握多元模型能让我们看得更透彻。

## 四、线上学习技巧与总结

最后，分享三个有助于掌握思维模型的线上学习技巧：

1.  **整理思路，而非抄录笔记**：不要急于记录思维导图。核心是通过老师的知识点，**整理形成自己的思路**，这才是真正内化。
2.  **即时记录启发与疑问**：课程信息密度高，听到有启发的点（如“多层次”）或产生疑问时，立刻用关键词记下，课后统一梳理。
3.  **生成记忆线索**：学习像在停车场找车，需要创建线索。例如，听到一个新理论，可以联想一个具体场景或图像，或者思考“这个模型能解决我生活中的什么问题”，从而加深记忆。

**本节课总结**：我们一起复习了思维模型。首先，明确了它的两大作用：**提升决策质量**与**创新能力**。其次，学习了通过 **“问思模”** 方法发现模型，并需完成**心态、重点、方法**三个转变，运用 **“五星笔记法”** 向高手学习。最后，掌握了应用时的 **“多层次”** 与 **“多视角”** 分析法，以及三个实用的学习技巧。通过持续学习和刻意练习，我们可以构建自己的多元思维模型，像大师一样迅速做出精准决策。
