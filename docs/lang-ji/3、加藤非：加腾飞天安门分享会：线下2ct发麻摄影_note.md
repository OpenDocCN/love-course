# 摄影与个人展示入门教程 03：从零开始的蜕变与展示面构建 📸

在本节课中，我们将学习如何通过摄影和个人形象展示，实现从零开始的个人蜕变。课程将涵盖心态调整、形象管理、生活记录以及如何通过展示面有效吸引他人。

---

## 心态调整与目标设定

上一节我们介绍了基础概念，本节中我们来看看如何从心态上为改变做好准备。

我叫CD，来自嘉宾组，是飞鸽培养体系的模拟型迷你版成员。我的专长是发马摄影。发马摄影的原创研究始于2017年，当时我对此一无所知，只能自己慢慢摸索。那时，并没有人指导如何构建个人展示面。很多人讨论如何通过展示面吸引异性或展示生活，但多数人并不真正理解。

以下是我个人经历的一个小故事。

这是我2016年以前的样子。当时我的状态非常“左”。请保持图片不动。按上面，对，就这样。大家能看清楚吗？清楚就好。这就是2016年的我，当时比较“左”。

那时我只有渴望、幻想和羡慕。我羡慕什么呢？当时我还没来成都，每天只羡慕一件事：渴望走出去。我来自沿海地区，对很多事情都不懂。我的祖辈世代都是渔民，以打鱼为生。而我这一代刚好走了出来，没有继续海上生活。这是我父亲当渔民时，我为他拍的一张照片。当时的我只会养鱼或帮父亲打鱼，其他什么都不懂。

我整天都在幻想如何找到一个理想型的伴侣。但每次我和朋友、发小出去，他们总会诋毁我。你身边可能也有这样的朋友，或者读书时的好兄弟，他们总会说你。比如，他们会说“你很臭啊”。我说我洗澡了，没有鱼腥味了。但他们还是会说“还是很臭啊，看到你的样子都觉得臭”。每次我们出去约会，和女生在一起互相聊天时，我总是最惨的那个。他们永远会给我挂上一个标签：有一身鱼腥味。这就是我的好朋友、好发小在聊天时给我的“助攻”。

当时我还在看保密地图，学习应该如何改变自己。那时，我很羡慕那些拥有异性朋友圈的人。我觉得自己这辈子都不会和网红成为朋友。我真的觉得不会。当时的我认为自己很“错”，家庭一般，而网红很高端，认识的人也多，怎么会认识我们这些无名之辈呢？我觉得自己这辈子都不会有机会认识那些网红小姐姐，和她们成为朋友或伴侣。

于是，我做出了一个决定。

## 行动与蜕变之旅

我在2016年3月21号正式来到成都。我从机场出去的时候，也是砸锅卖铁才买了一张机票，身上只剩下300块钱。走出机场时，我就在想：这个城市能让我蜕变吗？这个城市能让我改变吗？这个城市将来会给我一个位置吗？

我给自己设定了三年的预期。在做任何事情时，我都会给自己一个预期。比如，我认识了100个漂亮的小姐姐，但只要能和一个小姐姐约会，我就很开心了。所以我的欲望很低，不会因为被别人拒绝，或约会失败就感到受挫。我经常遇到一些私教学员跑来说：“导师，我约会失败了，心态很差。”飞哥之前有一个私教更极端，每次约会失败都想跳楼。我当年也有类似的想法，曾因失恋打算结束自己的生命。

当时我没工作，没有太多好朋友，经济状况也差，觉得生活没有意义。我去了当地人民医院的11楼，准备跨出去跳楼。但当我准备跳的时候，我发现：太高了，改天再跳吧。跳下去好像有点接受不了。从那一刻起，我就有了恐高症，以前我一点都不怕高。最终我没有跳，而是选择了来成都，从零开始。我给自己设定了很足的三年预期。

我觉得没有必要想太多，不要求自己每件事都做得完美，或者一开始就要做得很好、第一次就要成功。很多人一开始并不会成功，只有经过时间的磨砺，自己才会成功。所以我选择从零开始，实现蜕变之旅。

蜕变之旅也包括外在形象。以前我不会穿衣服，觉得自己穿得很丑。包括我刚加入家庭组的时候，飞哥在我旁边说：“CK，我给你预支三千块，你去买点衣服改造一下好不好？我觉得你这样子很丑，你要改。飞哥支持你。”我说可以，我会做的。当天晚上我去选了很多衣服，找飞哥要了两千块钱，买了很多。但我不知道自己的定位在哪里，根本不会穿。别人说我适合什么类型，我也不懂。

就像很多学员问我：“CK，你觉得我这样子的类型，适合穿什么样的衣服？”我说，你什么类型都应该去试一下，不要只拍点照片。所以到现在为止，我也没有觉得自己穿衣服很差，因为我什么风格都要去驾驭、去尝试。不是说你适合哪一类，就像萝卜青菜，你有不喜欢吃的，也有喜欢吃的。所以你先去尝试，有没有结果不重要，最重要的是你赶快出去试。

或者，跟一个网红小姐姐出去约会，你可能觉得自己失败率很高，但你自己还愿意去尝试。就像徐振权，飞哥的私教，以前也觉得自己不敢跟小姐姐出去。他以前很胖，我刚认识他那会他很胖，现在瘦了。一个人慢慢去改变的时候，会发现自己的眼光和履历会增长。尝试跨出你的第一步，先去做一些你原本不愿意甚至不敢做的事情，你会发现收获会慢慢增长。

## 面对挫折与自我总结

当然，收获增长的同时也离不开挫败。挫败会有很多。我在成都，曾连续十场约会都失败了。如果一个普通人没有调整好心态，会很阴郁。想象一下，连续十场约会都失败。当时没人告诉我失败的原因在哪里，我就自己给自己找原因，每天回去总结。

可能是我哪句话让她不开心了。
可能是我哪个行为太过于急切了。
可能是我说的东西不是她喜欢听的，或者让她感到不安，让她觉得我是一个不正经的男人。

经过这么多失败以后，我总结了自己很多问题。我没有去找别人问。当你盲目地，比如报了一个私教，有任何问题，甚至今天拉了好几次肚子，都去问导师，那他马上会崩溃。你会觉得自己所有事情都没有主见。作为一个男人，没有主见是最可怕的，你什么事情都不能自己做主了。

所以我就给自己做主，去分析我自己的问题在哪里。因为我将来可能也会为别人去解决这个问题。在解决了这些失败的原因后，第十一场约会，我终于成功了。我终于觉得自己有成就感了。我觉得，这应该是九十九场约会失败当中的其中一场成功了吧。我就这样给自己一个安慰。

有时候我们自己要给自己一些奖励。比如你最近在健身，不能吃有益的东西，但今天你做了一件很开心的事情，是不是应该奖励一下自己，吃一顿好的？健身餐真的很难吃，我自己也不吃。所以我觉得适当的奖励自己也是一种鼓励。

我每天早上起床的时候，因为我超喜欢的偶像G-Dragon（权志龙）在一次采访中说过：“我每天起床，很喜欢对着镜子说，哎，你看帅啊，我好帅啊。”天天鼓励自己，不管什么样子，你都觉得自己很帅。哪怕你现在很“拙”，你对着镜子说：“今天真的蛮帅的，可以啊，哎，身体可以啊，蛮帅的呀。”天天给自己一点小小的鼓励，时间久而久之，你会觉得自己真的很美。

举个很简单的例子，你走出去以后，你看到一个女生，每个人都说她是美女，她觉得是理所当然的。当然你天天这样鼓励你自己，别人说你帅哥的时候，你觉得也是理所当然。有些人会觉得自己并不帅。为什么觉得自己不帅？我觉得很奇怪。你觉得自己不帅的原因在哪里呢？你也是吃饭长大的，他也是吃饭长大的。你也是有爸妈生的，他也是有爸妈生的。凭什么他帅你不帅？我觉得每个人天生我才必有用，没有谁生来就是挫败的。

我以前觉得我身高只有一米六，这辈子就注定了，只能仰望别人，甚至觉得这辈子也不可能去跟一米七五的小姐姐谈恋爱。牵手的时候，我的手在这个位置，她的腰在我这里，我就像个弟弟。但她说：“没事啊，但是你没有魅力啊？我就喜欢你这样子有魅力的男生。”我觉得我的魅力在哪里？待会就给你们讲到。

## 珍惜机会与持续学习

这是我们的家人组。我是从2016年12月，当时公司的选择部解脱了以后加入的。飞哥像在座各位一样筛选人群。最后飞哥被迫无奈，还要多选一名的时候，我在一个角落里盯着，有没有人选我？如果没人选我，我可能就要离职了。结果飞哥看到了我，把我捡了回来。

我很珍惜这次机会，因为我知道机会给到你身上，你自己要怎么去发挥。所以我很珍惜，飞哥只要在旁边直播，讲了一些东西，我都会记下来。我的手机笔记里，任何一个人在这里讲课，我可以复述出他80%的每句话。我以前打字都有问题，后来飞哥说你要懂得去做笔记，把人家的东西记下来，你自己再说一遍，你会吸收很多不一样的知识面。所以我慢慢让自己形成习惯，别人说话我就在旁边记，让自己成长得更快。这就是我的恩师，飞哥。南方哥也是我一个最好的大哥哥、前辈。实践的观点就不多介绍了。

## 自我展示的核心

接下来我们看看自我展示。经常有人觉得，展示面，那展示的优势有什么呢？这里我分为两点。

**第一点是生活的记录。** 你自己生活的记录，是你最好的新开始。如果你不懂得展示你的生活，你觉得你的生活很“拙”，那是因为你不会修图，你不敢展示自己“拙”的一面。我觉得没必要一定要发朋友圈才叫展示。飞哥很喜欢有一个小动作：我现在在办公准备直播了，来给你看一下，我在跟会场所有的兄弟们在这里。展示不是一定要发朋友圈，而是通过小视频、拍照，或者你自己通过一些言语去展示出来。不一定说是发朋友圈。

为什么很多人就以为，展示一定是要发朋友圈，让所有人知道呢？你让你身边，你觉得你今天就喜欢这个小姐姐，你就发给她就可以了。让她知道你的生活在干嘛。或者，那个小姐姐对你没什么窗口了，在这里相处一下，给她发一点你最近在干嘛的东西，我觉得也OK。这也是展示的一点。不一定说是发朋友圈。发朋友圈是展示你最优越的，或者说你修图修得最好的、你比较羡慕你自己的成就的东西，你才展示出来。

而我的朋友圈做得很生活化，我也没有什么好车之类的。但我唯一一点优势，我觉得我修图很牛逼。我敢这么说，因为我很有冥想的力量。我能够听一首歌，想很多的画面出来。我敢想就敢去做，我不会让自己犹豫。因为一个男人把你想一件事情超过三分钟都不去做的时候，基本上我采访过很多人，真的是事实，超过三分钟都不想去做。

比如想去健身，出门的时候下雨了，改天再去吧，明天去吧。然后明天来了，准备出门去健身的时候发现，好像十点钟还有一个约会，待会九点钟去健身，会不会有点赶呢？还是明天吧。然后第三天的时候，觉得前面两天工作很累了，适当奖励一下自己，那今天就奖励自己吧，也不去了。健身就慢慢荒废了。自己总是喜欢给自己找借口。作为男人，我说真的，你找借口只是骗你自己。

我自己很自律，我觉得我的生活全程是靠我自律而来的。所以这也是我生活上记录的东西，我不会让自己打嘴炮。每天起来给自己做一个规划，你今天能做一些什么样的事情。比如今天我九点钟要到公司打卡，五点钟我下班，我今天做了一些什么样的事情，你看自己收获了什么。而不是让自己觉得，今天很累，我不想说了。我觉得今天工作我做错了，被领导骂了。你被骂的时候也是你成长的时候。当你没有被骂的时候，你连这个价值观你都没有了。别人都不想骂你，骂你干嘛呢？你又不会好端端觉得你老总会骂一个路人。

**第二点是服装与魅力值。** 一个男人你觉得你自己有魅力的时候，那最新的开始是给人家的第一眼的感觉。女生也是从第一眼的感觉，看你这个人怎么样，而造成吸引的。而不是说你这个人多会讲话，你多会讲话，你有多少钱，别人不知道。从第一眼开始，那就是你自己本身的服装。这也是我一直以来很简单的一些服装搭配，非常简单，都是在朋友圈当中可以看得到的。我的朋友圈基本上是保持在五颜六色，别人只要一点进来都很有欲望去刷。

我很喜欢看影子，所以我觉得我被什么吸引住，我就会找这个原因，我会做给别人看。比如举个很简单的例子，你觉得你突然间很喜欢一个小姐姐，是因为她什么？因为她可能就是身材很好，或者说胸很大，屁股很翘。那女生喜欢男生什么？可能第一眼也是看外在品质，你的服装。第二，你有没有身材，你有没有腹肌。这也是你自己的优势。所以说你的魅力值也会吸引到别人，那就是你的身材到底有多优越。

## 状态与吸引力

**状态等于你的吸引力。** 那就是你工作状态，我刚刚也讲到了，那就是你的生活上的一些小事情。我自己工作状态，我基本上我会展示给你们看。就比如下面我在家里面用手工制作我的一些产品，包括有时候我的一些作品，我拍出来，我用手机拍出来的一些东西，会被别人去买我的版权。并不是说一张照片值多少钱，也能够值三百到五百。一张手机随手拍的照片，我当时在泰国拍了一张我的影子，一个倒影，太阳就从这边下来，我就拍了一个影子。一张照片我卖五百，人家是买我一个版权。但是要知道照片呢，还是我自己的，我可以使用，但不能转交给别人使用，因为我是美人。所以我的作品我觉得很精华。

特别是我这个，这个当时卖了三百万？三百还是五百？反正都是三百到五百之间。都是我用手机拍的随手拍的。我当时只是比如你在问我在干嘛，我就拿手机我在录课啊，啪拍了一张，我就发出去了。发出去以后，我当时就发在了微博，人家私聊：“哎你好，您这张照片版权可以让给我吗？我能够出三百或者五百的价格去购买它吗？”我说：“可以啊，没问题啊，你直接转账，我就把原图发给你就可以了。”钱马上到位。我觉得手机摄影还可以赚钱的，也不错，对不对？展示一下我自己的生活状态，也OK。

同时我自己展示在朋友圈里面，有很多异性会来留言：“哎你生活为什么这么有趣？”想来了解你，他是被吸引到了。色彩第一是色彩，第二是你的一些细节的一些状态，等于别人想来评论，想来点赞，想来了解你，想来深挖你。如果你自己本身你不喜欢展示你个人状态的话也OK，因为有一些人是不喜欢发朋友圈的，觉得自己啊，我又不是公众人物，干嘛要发这些东西吧，对不对？但是我可以这么说，每个人都有自己的一些魅力的地方，让别人所看到想去了解你的地方。

## 个人品牌与销售

**朋友圈等于你的广告，你的个人就等于你的产品，你自己本身说话就等于你在销售。** 你跟一个男女之间可能你会觉得，你要销售你自己。怎么去销售？你能不能说出去嘛？你能不能把自己卖出去？能不能卖给这个女的？“我就做你男朋友”，能不能做到？不能做到是不是？我用嘴巴说了你能做到，那就是销售的力量。而你自己产品的价值，你自己也很清楚啊，对不对？同时你要有你的广告啊。

这也是我自己本身的。有一些兄弟可能不认可这些方式，我也可以理解。但是我只是说我的一面，你可以就参考，这没有什么硬性的。不要说就是展示，你自己也展示面就很牛逼，知道吧？没有。你只是任何一个女生，举个更简单的例子，就像女生一样，她永远喜欢说，喜欢别人说她好听的。你也喜欢别人说你好听的，不喜欢别人动不动就说你：“我觉得你头发很丑啊，我觉得你戴眼镜很low啊，我觉得你今天穿的很丑啊。”你不喜欢听到这些话。你更喜欢听到的可能就是：“哎我觉得你怎么穿的那么帅，什么情况啊？”你会觉得你做什么事情都是对的。

当别人不认可我的时候，我最经常会听到一些这样子的话：“像你这种身高怎么把妹？我靠我这种身高根本保不了妹子，好不好？根本和异性谈不了恋爱，知道吧？”但是我经常呢，被别人当做你爱对象而已，这是我的自信点。而你自己的自信点呢，我觉得你要开拓早一点。什么叫脸皮厚？你脸皮有多厚啊？可能老师就可能有点了解我，我是脸皮很厚的这种人。我第一我家里又没钱，第二我长得又不高，第三我长得又不帅。哪怕去搭讪，你说别人拒绝我了，我会有点难过，有点看不起。有些人喜欢找借口知道吧？那我觉得我很丢脸啊？你有什么脸好丢的吗？第一你又不是很帅，第二你又不是很有钱，第三你又不是什么公众人物。你除了靠一个脸皮厚，我就是靠脸皮厚。别人怎么说我会生气的？我会生气？我就觉得放他们一马，谢谢。别人说我怎么不对都可以。
