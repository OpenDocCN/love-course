## 三、 核心心态建设：摆脱患得患失

**本节我们将探讨一个普遍的心态问题：患得患失。** 很多人在感情中特别在意一个女生的得失，心态容易波动。

**核心概念：预选与选择权**
心态强大的背后往往不是空洞的“自信”，而是拥有**更多的选择**。就像一个真妹同时有4到5个男生在追她，其中有一两个是有魅力的，她自然不会太在意其中某一个是否失去。

因此，改变心态的第一步，就是要去掉患得患失的心态。最重要的方向就是**要去多交往女生**。

以下是具体操作步骤：

1.  **扩大社交圈**：同时去发展至少两到四个追求目标，并且都发展到暧昧阶段。
2.  **降低需求感**：当你身边同时有几个暧昧对象时，你就不会因为其中一个女生的一句话、一个动作而患得患失。
3.  **心态类比**：就像你从来没吃过冰淇淋，突然有一个掉到你面前，你会吃得很小心，生怕它掉到地上。但如果你冰箱里有很多冰淇淋，你就不会在意其中一个是不是掉了、是不是少吃一口。

就这么简单。不用听什么心态改变的课，你就去这么做，你的心态自然就好了。当你发现你的“冰箱”里有很多“冰淇淋”时，你还会在意一个冰淇淋是不是掉了吗？
