很多同学询问，给女朋友买什么礼物比较好。这取决于平时是否用心相处。如果用心，你自然会知道她喜欢、想要或期待什么。偷偷准备好她期待的东西送给她，会让她非常开心。

礼物能表达你对她平时的关注、在意和爱意。这些是女生最需要的部分。

有的同学可能会说，送鞋是否有“分手”的寓意，因为听起来像“送走”或“分离”。类似的说法还有送梨（离）或送扇（散）。其实现在不必过于顾虑这些说法。
