## 爱的表达：需要具体事件作为依托 💖

上一节我们介绍了恋爱进程的阶段。本节中我们来看看“示好阶段”表达爱意的关键。

在恋爱进入第四阶段，即示好阶段时，必须在日常与女生接触中，用心记下那些令人心动的瞬间。这些瞬间可以在当时或事后，向女生表达爱意时，作为爱的证明。

爱的表达通常分为两种。一种是联系到之前相处点滴所产生的爱意。另一种是在具体事情发生后，心灵即刻涌现的爱意并立即表达。

总之，关键点在于爱意的表达必须与生活中的具体事件联系起来。否则，脱离具体事件，空泛地说“我爱你”或“我喜欢你”，不但不会让女生感动，反而可能激发她们的不安全感，觉得男生在哄骗她们。

例如，在一个视频片段中，开始时男生突然跑进去对女生说“我爱你”。这种表达突如其来、无根无据，女生不以为然。但后续男生给出两人相处中的具体细节时，这种爱的表达就有了依托。过往的点滴得到相互关联，这种表达激活了女生的感受和情绪，产生了共振。女生感受到男生的真实爱意，便自然地接受了他。

当然，必须再次强调，整个示好阶段出现在恋爱的第三阶段，即在吸引阶段之后。否则上述概念不成立。
