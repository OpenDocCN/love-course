# 修图入门教程 01：人像精修与滤镜应用 📸

在本节课中，我们将学习如何使用多个软件对一张街拍人像照片进行系统性的美化处理。课程将涵盖面部与身材重塑、皮肤平滑处理、身高调整以及滤镜应用等核心步骤，旨在让初学者掌握一套完整的人像修图流程。

## 软件准备与初步观察 🛠️

首先，我们需要准备本次修图将用到的软件。主要工具包括 **FaceTune**、**美图秀秀** 和 **Instagram**。请确保你已安装或可以获取这些应用程序。

现在，我们打开需要处理的原始图片。这是一张在室内眼镜店拍摄的街拍照片，光线偏暖，我们的目标是优化人像的轮廓、比例并调整整体色调。

## 第一步：使用 FaceTune 进行面部与身材重塑 👤

上一节我们介绍了本次修图的目标和工具，本节中我们来看看如何使用 FaceTune 进行基础的人像调整。我们首先将图片的脸部区域放大。

以下是使用 **面部重塑** 功能的具体操作步骤：
1.  在软件中找到并点击 **面部重塑** 功能（通常图标为一个带方格的头部轮廓）。
2.  通过拖拽功能点，向内微调脸部边缘，使脸型看起来更协调、更美观。
3.  同样使用该功能，适当调整衣领区域，使其与新的脸型匹配。
4.  根据个人情况，调整嘴巴的弧度，使其呈现更愉悦的表情。
5.  对于身材部分，使用相同功能调整肩宽，使其看起来更宽阔、有气势。
6.  针对胳膊、腿部等较粗的部位，向内收拢，使整体身材比例更匀称。如果身材偏瘦，则可向外拉宽。

**核心概念**：面部重塑功能通过控制面部和身体轮廓上的关键点来实现形变，其原理类似于网格变形算法。

完成轮廓调整后，我们需要让皮肤看起来更细腻。

以下是使用 **平滑** 功能的步骤：
1.  在调整好整体轮廓后，使用 **平滑** 功能。
2.  将画笔应用于脸部皮肤区域，抚平瑕疵，使皮肤质感更加光滑细腻。

为了让配饰更突出，我们使用 **局部** 功能。

以下是操作步骤：
1.  选择 **局部** 调整功能。
2.  将其应用于戒指、耳钉等金属饰品上，提高其亮度和对比度，使其在画面中更醒目。

至此，FaceTune 部分的基础调整就完成了。

## 第二步：使用美图秀秀调整身高 📏

在 FaceTune 中完成了轮廓修饰后，我们可能会发现身高比例仍有优化空间。本节我们将使用美图秀秀的“增高”功能来拉长腿部。

以下是操作步骤：
1.  在美图秀秀中打开上一步处理好的图片。
2.  找到并选择 **增高** 功能。
3.  软件会生成一个蓝色调整区域，将其覆盖在腿部。
4.  缓慢向下拖拽区域下边缘，拉长腿部。此过程需反复微调，并利用对比功能查看调整前后的效果，确保比例自然协调。

**核心概念**：`增高功能` 本质上是针对选定区域进行内容感知的拉伸变换，需谨慎使用以避免失真。

## 第三步：使用 Instagram 添加并微调滤镜 🎨

完成了所有的形体调整后，最后一步是为照片赋予理想的色调和质感。本节我们将使用 Instagram 的滤镜和编辑工具。

首先，我们需要为照片选择一个合适的滤镜。

以下是选择滤镜的步骤：
1.  在 Instagram 的编辑界面中浏览各种滤镜。
2.  逐一尝试，根据照片的具体环境、光线和个人喜好选择最合适的一款。注意滤镜强度不宜过高，以免显得不自然。通常将强度设置在 **20-50%** 之间为宜。

选定基础滤镜后，我们进入更精细的参数调整。

以下是手动调整核心参数的步骤：
1.  **亮度**：调整整体画面明暗。
2.  **对比度**：增加画面明暗对比，让照片更“通透”。可轻微调整，例如 **-10%** 以软化效果。
3.  **高光**：控制画面中最亮区域的亮度，可适当降低以恢复细节。
4.  **饱和度**：控制色彩鲜艳程度。轻微增加，如 **+5%**，可使色彩更生动。
5.  **锐度**：增强细节边缘的清晰度，让照片看起来更立体。可设置为 **+18%** 左右。
6.  **色温**：调整画面冷暖色调。如果原图偏暖（黄），可向冷色调（蓝）方向微调以中和。

**核心概念**：这些参数共同定义了图像的视觉风格，其调整可抽象为对图像像素值进行一系列变换函数。

## 成果对比与课程总结 ✅

现在，让我们对比一下修图前后的效果。原始照片存在脸型较圆、腿部较粗、光线偏暖等问题。经过我们上述三个步骤的系统处理之后，人像的面部轮廓更清晰帅气，身材比例更协调，整体色调也变得更加干净、有质感。

本节课中我们一起学习了从形体修饰到色调调整的完整人像修图流程。我们首先使用 **FaceTune** 重塑了面部与身材轮廓，并平滑了皮肤；接着利用 **美图秀秀** 的增高功能优化了身高比例；最后通过 **Instagram** 添加滤镜并精细调整了色彩参数。

修图是一项需要练习的技能，建议你下载提供的原图，跟随步骤反复尝试，逐步掌握每个工具的使用技巧和度。
