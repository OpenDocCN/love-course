## 如何测量身体尺码？📏

要进行精准的形象改造，首先必须了解自己的身体数据。许多人不清楚自己的具体尺码，这导致无法获得准确的穿搭建议。

**精细的尺码数据能让你买到合身的衣服**，避免衣服过大或过紧的尴尬效果。这里指的是常规剪裁的服装（如西装、正装衬衫），一些特定的宽松潮流版型（如Oversize）除外。

如今网购盛行，清楚自己的尺码尤为重要。网络尺码表通常包含胸围、身长、臂长等数据，你需要有自己的数据与之对照。

对于男性而言，衣橱里至少应有一套合身的西装。一套好的定制西装对尺寸要求极为严格，能极大提升气质。建议年满18岁的男士，都应备有一套定制西装以备正式场合之需，它能为你带来自信。

以下是具体的测量方法，你可以购买一把软尺自行测量：
*   **肩宽**：测量两个肩胛骨端点之间的距离。
*   **胸围**：软尺绕过乳头位置一周。
*   **腰围**：软尺绕过肚脐位置一周。
*   **臂长**：测量肩部到手腕处的长度。
*   **上身长**：测量颈部到臀部最高点的长度。
*   **臀围**：软尺绕过臀部最丰满处一周。
*   **大腿围**：测量大腿最粗部位一周。
*   **下身长**：测量肚脐到脚踝的长度。
*   **脚长**：测量大脚趾到脚后跟的长度。可将沾湿的脚踩在A4纸上测量印记。

请务必测量并记录这些数据，无需死记硬背方法，但要将自己的结果牢记于心。

## 认识你的脸型 🎭

了解尺码后，我们进入面部分析。根据“三庭五眼”、颧骨曲线和下巴形状，脸型大致可分为六种。

**“三庭五眼”是面部美学标准**：
*   **三庭**：发际线到眉骨、眉骨到鼻底、鼻底到下巴，这三段距离相等。
*   **五眼**：以一只眼睛的长度为单位，从左太阳穴到右太阳穴的宽度应能容纳五只眼睛。

大多数人并非完全符合此标准，这很正常。了解概念即可，目的是为后续选择发型和配饰提供依据。

以下是六种基本脸型，请对照判断自己属于哪一种：
1.  **圆形脸**：脸部轮廓较圆，下巴较短或不明显。例如演员发哥，显得有亲和力。
2.  **蛋形脸（鹅蛋脸）**：被认为是最理想的脸型，圆润中有棱角。这种脸型能驾驭大多数发型。
3.  **长形脸**：脸型较长，三庭比例较长。通常身材也较高大。
4.  **尖形脸**：下巴尖锐，两颊轮廓较直，呈倒三角形。上镜效果好，但需注意审美勿过度。
5.  **菱形脸**：两侧颧骨突出，是脸部最宽处。例如萧敬腾，显得有男人味。
6.  **方形脸**：面部线条硬朗，下巴较短。例如演员张涵予，需通过发型修饰。

**了解脸型的目的，是为了在后续课程中为你选择适合的发型、墨镜和衣服风格**。S1老师的课程将系统解决你关于形象的所有疑惑。

## 分析你的体型 🧍♂️

最后，我们来分析体型。体型对穿衣搭配至关重要，前面郭敬明等反例已说明，选择符合自身体型的衣服是关键。

每个人的体型都不同，**不能只追求自己喜欢的款式**。在形象改造中，老师推荐的风格可能与你以往不同，但这往往是当前流行、女性欢迎且适合你的。许多学员在改造后，社交成功率得到了显著提升。

你可以通过以下公式初步判断自己的体型：
`标准体重(kg) = (身高(cm) - 100) × 0.9`
*   实际体重 > 标准体重10%-20%：**壮实型**
*   实际体重 > 标准体重20%以上：**肥胖型**
*   实际体重 < 标准体重10%-20%：**偏瘦型**
*   实际体重 < 标准体重20%以上：**极瘦型**

结合身高（矮、中、高），男士体型可总结为以下六种：
1.  **正常体型**
2.  **完美体型**（肌肉匀称）
3.  **瘦高体型**
4.  **瘦矮体型**
5.  **胖高体型**
6.  **胖矮体型**

**只有完美的体型才是“行走的衣架”**。偏激的体型无法适应所有衣服。好身材不仅能穿出好衣服，更能带来自信。**“如果你连自己的嘴都管不住，如何掌控自己的人生？”** 严于律己，才能成为稀缺、高价值的“艺术品”男性。
