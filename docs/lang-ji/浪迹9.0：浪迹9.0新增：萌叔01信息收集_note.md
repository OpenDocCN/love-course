# 浪迹9.0：浪迹9.0新增：萌叔01信息收集_note

在本节课中，我们将要学习情感狙击与挽回过程中的核心第一步：情报收集。情报收集如同医生诊断病情，是制定有效策略的基础。只有充分了解对方和你们的关系状况，才能采取针对性的行动。

上一节我们介绍了情报收集的重要性，本节中我们来看看具体从哪些渠道进行收集。

情报收集主要分为线上和线下两个部分。

## 线上情报收集 🌐

线上渠道是现代人信息的重要载体，是了解对方过往和现状的宝贵资料库。以下是主要的线上情报来源：

*   **微信朋友圈**：微信朋友圈是个人名片。我们可以从中分析对方的经济水平、兴趣爱好与生活状态。其核心价值在于**构建人物画像**。
*   **微博**：很多人将微博视为私密空间，会抒发情感或吐槽生活。通过微博可以了解对方的情感经历，甚至可能发现其与前任的联系线索。
*   **QQ空间**：QQ空间记录了个人的成长历程。例如，从过去的照片发现对方曾有“逆袭”（如减肥、形象改变）经历，这有助于理解其性格特质。
*   **聊天记录**：双方的聊天记录能最直观地反映互动模式。通过分析聊天记录，可以判断你在关系中的地位（如 `地位 = 对话主导权`），以及你是一个乏味、跪舔还是幽默有趣的人。

刚刚我们探讨了线上情报的收集，接下来是同样关键的线下部分。

## 线下情报收集 👥

线下情报涉及现实互动与关系网络，对于评估关系质量和分手原因至关重要。以下是线下情报收集的核心考量维度：

*   **相识方式**：你们是通过社交软件、朋友介绍、相亲还是搭讪认识？相识方式 (`相识方式 ∈ {社交软件， 朋友介绍， 相亲， 搭讪}`) 会影响后续的策略制定。
*   **个人基础信息**：在线下接触中，可以了解到对方的客观条件，如身高、颜值、罩杯等。这些是评估其**婚姻市场价值**的参考因素。
*   **共同经历**：你们是否互相赠送过礼物、一起旅游过？这些共同经历 (`共同经历 = 礼物 + 旅游 + ...`) 反映了双方的情感投入程度。
*   **亲密关系**：是否发生过亲密关系是一个重要指标。在中国，许多分手源于亲密关系不和谐。
*   **社交关系网**：你们是否有共同的朋友、经济往来或共同认识的亲人？这些社交关系 (`社交关系 = 共同朋友 ∪ 经济往来 ∪ 共同亲人`) 会影响挽回策略的复杂性。

在掌握了充分的情报后，我们需要对分手原因进行归类。萌叔根据过往案例，将分手大致归为两类。

## 分手原因归类 🗂️

了解分手原因是“对症下药”的前提。以下是两种主要的分手类型：

1.  **情绪型分手**：
    *   **常见矛盾**：例如，女生生气是希望获得情绪照顾 (`女生诉求 == “寻求情绪照顾”`)，但男方未予理会反而打游戏，导致矛盾。
    *   **面子问题**：在朋友聚会中未给对方面子，使其难堪。
    *   **低情商处理**：当女生生气时，男方选择讲道理而非哄劝 (`应对方式 != “哄劝”`)，或进行无下限道歉和信息轰炸，导致关系恶化。

2.  **吸引力缺失型分手**：
    *   **个人价值下降**：在长期关系中，一方不注重自我提升，形象邋遢、体重飙升 (`吸引力 ∝ 1 / 体重` )。
    *   **三观与习惯不合**：相处后发现三观不一致，生活习惯差异大。
    *   **关系维护不足**：在长期关系中，没有持续为感情投入和制造新鲜感 (`感情浓度 ∝ 持续投入`)，导致感情变淡。

本节课中我们一起学习了情感狙击与挽回中情报收集的全过程。我们从**线上**（微信、微博、QQ、聊天记录）和**线下**（相识方式、个人条件、共同经历、亲密关系、社交网络）两个维度全面收集信息，并最终将分手原因归类为**情绪型**和**吸引力缺失型**。只有完成这些扎实的情报工作，才能为后续制定精准的挽回策略奠定坚实基础。
