**最重要的一点（作业）**：**聊天记录不准删**。即使女生把你删了，你也不能删聊天记录。因为你要去回顾、反思、总结问题出在哪里。很多兄弟心态脆弱，被删后愤怒地也删掉对方和记录，导致无法复盘，问题一错再错。不解决问题，就永远无法成长。
