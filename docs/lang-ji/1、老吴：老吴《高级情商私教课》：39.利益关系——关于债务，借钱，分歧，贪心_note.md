# 高级情商私教课 第39课：利益关系——关于债务、借钱、分歧与贪心 💰

在本节课中，我们将要学习如何处理人际关系中的利益问题，特别是债务、借贷、意见分歧以及贪心这四个核心议题。理解并妥善处理这些关系，对于个人成长和财富积累至关重要。

## 第一部分：关于债务与借钱

上一节我们介绍了利益关系的总体框架，本节中我们来看看具体的债务与借贷问题。

一个人借钱的能力，等同于他赚钱和消费的能力。许多商人或企业家在发展过程中，都需要进行资金周转或向银行贷款，这很常见。

借钱本身没有问题，但关键在于借钱者是否**明确拥有偿还能力**。公式可以表示为：
**偿还能力 ≥ 借款金额**

如果借钱时没有清晰的偿还能力，借钱就变成了一种赌博。这种出于恐惧或匮乏的起心动念，往往会导致失败。

借钱本质上是人与人之间的**信任链接**。对方愿意借钱给你，是基于信任。

以下是处理借钱问题的几个要点：

*   **关于写借条**：如果决定借钱，建议主动要求写借条。这是保护双方权益、明确借贷关系的凭证。
*   **关于借钱不还**：如果钱已借出且难以追回（无凭证情况下），应停止费力追讨。专注于提升自身能量和价值，财富会从其他途径到来。
*   **关于起诉**：如果持有借条等凭证，对方逾期不还，可以考虑法律途径解决。
*   **关于契约精神**：最高级的信任是无需借条的“契约精神”。这建立在双方极高的信任度和品德之上。

金钱的本质是**价值的交换**。易经中强调“**德行配位**”，即内在的品德、能力要与外在的拥有相匹配。

一个人最有价值的财富，是他人对你的**信任**。在借贷关系中，务必觉察自己和对方的起心动念是否出于贪婪、恐惧或匮乏。出于这些负面信念的行动，很难获得金钱上的成功。

借钱不还或恶意拖欠的人，其财富能量管道容易堵塞，因为他们内心种下了“内疚”和“持久匮乏”的种子。

如果你已负债且压力巨大，请静心思考：**你能为他人创造什么价值？** 把焦点从“我要赚钱”转移到“我能帮别人赚钱”上。

## 第二部分：如何处理工作中的意见分歧

了解了借贷中的信任问题后，我们来看看工作中遇到意见分歧时该如何处理。

工作中与老板或同事产生分歧时，“妥协”常让人感到委屈。但妥协并非只有“忍受”这一种结果。

实际上，面对“要不要妥协”这个问题，可以有三种思考方向：

1.  **不妥协**，坚持己见。
2.  **妥协，但清楚自己为何妥协以及能得到什么**。
3.  **妥协，但只盯着自己因此失去了什么**。这是最坏的选择，会让你持续感到委屈。

真正的委屈感，往往来自于你选择了妥协，却一直聚焦于自己的损失。

那么，如何做出选择呢？心理学家推荐的方法是 **“现实检验”**。

现实检验，就是**在现实中亲自尝试不同选择（妥协与不妥协），用真实体验作为判断标准**。例如，纠结于选择高薪不喜欢的工作还是低薪喜欢的工作时，最好的方法就是分别去尝试，亲身体验哪种情况更能承受或带来快乐。

将此方法应用于“老板提倡加班，你不想加班”的情景：

*   **实验一（妥协）**：尝试连续加班一周，感受妥协带来的具体影响。
*   **实验二（不妥协）**：尝试连续一周不加班，观察不妥协带来的后果。

通过为期两周的实验，你就能找到适合自己的工作与生活的平衡点。此方法同样适用于情感等其他生活领域。

## 第三部分：道理都懂，但做不到怎么办？

解决了行动选择的问题，我们再来探讨一个普遍困惑：明白了很多道理，为何难以付诸实践？

很多时候，做不到并非因为懒惰，而是因为我们依赖**外在动机**（如奖励、他人认可），而忽视了**内在动机**（如自我成长、兴趣爱好）。

研究表明，由内在动机驱动的行为更持久、更有效。例如，为了零食（外在动机）画画的孩子，其练习量远不如为了兴趣（内在动机）画画的孩子。

因此，改变的第一步是：**厘清并专注于你的内在动机**。问问自己，想做出改变（如学会感恩）是为了外在认可，还是为了内在成长？

第二个方法是 **“角色扮演”**。

将你想要实现的改变，转化为一种“角色”去扮演。例如，希望自己更善于社交，就可以在一段时间内扮演一个“社交达人”，思考并模仿这个角色会如何思考和行动。

角色扮演的关键原则是：**一段时间内，只专注于一个改变，扮演一种角色**。集中精力才能更好地贴近角色，内化其行为模式。

总结来说，实现改变需要：
1.  **聚焦内在动机**。
2.  **通过角色扮演，用体验贴近改变**。

## 第四部分：工作中什么都想要，是贪心吗？

最后，我们来探讨工作中常见的“贪心”心态：面对机会总想全都要，无法确定自己最重要的需求。

这未必是贪心，更可能是因为**缺乏清晰的价值体系**。就像没有购物清单就去逛街，很容易盲目消费。

要找到核心需求，首先需要建立工作的价值体系。心理学家指出，我们对工作的需求可归纳为六大核心：

*   **三种性质**：
    1.  **稳定性**：工作安稳、可控。
    2.  **未知性**：工作充满挑战和惊喜。
    3.  **成长性**：能获得知识和能力提升。
*   **三种感受**：
    4.  **意义感**：感觉工作有价值。
    5.  **连接感**：能与他人建立融洽关系。
    6.  **奉献感**：感觉自己被需要、被认可。

建立价值体系的第二步，是对这六大需求进行**排序**。可以采用“**加权计分法**”：

1.  准备六张纸，分别代表一种需求。
2.  在每张纸上，写下符合该需求类型、你内心希望发生的具体事件（例如“稳定性”下列出：薪水符合预期、有规范制度、固定工作时间方便家庭等）。
3.  每写下一件具体关注的事件，就计1分。
4.  计算每个需求板块的总分。

根据得分高低，你就能搭建起自己的“需求金字塔”，看清哪个需求对你最重要。将目标细化、写下来，问题就解决了一半。
