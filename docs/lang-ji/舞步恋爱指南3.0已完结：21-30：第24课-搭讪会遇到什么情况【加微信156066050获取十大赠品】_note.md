# 舞步恋爱指南 第24课：搭讪时会遇到的五种组合类型 🧑🤝🧑

在本节课中，我们将要学习在搭讪时会具体遇到的五种组合类型，以及针对每种类型的应对策略和核心要点。理解这些组合类型，能帮助你在实际搭讪时更有准备，提升成功率。

上一节我们介绍了搭讪的底层思维，本节中我们来看看搭讪时会遇到的具体情况。

在搭讪中，我们只会遇到五种组合类型。以下是这五种类型的逐一介绍：

*   **单女组**：只有一个女孩的组合。
*   **双女组**：两个女生的组合，包含一个目标和一个障碍。
*   **多女组**：一个目标，两个或两个以上障碍的组合。
*   **男女组**：一般是一男一女，或一男两女的组合。
*   **母女组**：妈妈带着女儿的组合。

接下来，我们将详细分析每种组合的特点与应对方法。

## 母女组 👩👧

从难度上讲，母女组的难度是最低的。因为女生和妈妈在一起时，她的安全感最高，最不害怕。只要我们的外形、表达、态度大方得体，妈妈很多时候反而会促成女儿多认识一些朋友。

搭讪母女组时，记得先和妈妈说话，表达来意。征得妈妈同意之后，再和女生说话。搭讪这种组合的重点，其实是和妈妈的互动与表达。因为当我们在和妈妈交流时，女生是在看着我们的，但同时又不会感觉有太大的压力。如果妈妈不反对，女生通常都会答应。

这种组合往往都是女生和闺蜜，两个人有说有笑，或者情绪很高昂地走着或坐着，成功的可能性非常大。

在双女和多女组合中，障碍（闺蜜）分为两种类型：

2.  **捣乱打断型**：这种闺蜜会阻止我们的搭讪，然后直接拉走目标。她可能是在妒忌，也可能是在保护朋友，或者她自己的心情不好。我们不必追究闺蜜阻止的原因，做好自己，礼貌真诚地表达。如果目标没有窗口，就不要做太多纠缠，直接放走。

针对多女组，我们要注意观察谁是**最高雌性领袖**。一般搭讪雌性领袖，表明来意，然后再对目标说话。多女组一般不太会遇到闺蜜阻止的情况，闺蜜们反而会起哄或走开，主要看目标对我们是否感冒。但是在这种组合中，因为女生人数比较多，能量比较强，对于我们的自信心和心理素质要求更高，通常用来训练胆量是一种很不错的选择。

## 男女组 👫

关于男女组，我想重点说明一下。通常我们会遇到一个男生带着两个女生的情况。一般这种情况，可能是情侣带着女生的闺蜜。

以下是处理这种组合的策略：

还有一种男女组是一男一女的组合。我不建议大家轻易搭讪这个组合，除非你非常非常喜欢这个女生。如果我们通常观察确定这不是男女朋友，就直接进行搭讪，那么可能会毁掉这个男孩很长一段时间的努力。
