## 四周课程大纲 📅

首先，我们以周为单位，整个月分为四个大环节。

**第一周：自我提升与自身建设**
物以类聚，你是什么样的人，就会吸引什么样的人。我们都喜欢长发飘飘的漂亮女生，而不是短发的假小子或比自己还邋遢的。第一周，我们将教大家自身建设，包括外貌、形象、穿衣搭配、细节处理。例如，我们看到老童弄了发型，衣服简单但有项链配饰、戴耳钉、修理胡子等。我们将教你如何根据个人风格树立自己的形象。

**第二周：短期关系（一夜情缘）**
这是由我们浪迹教育最厉害的短期关系导师负责。课程编排的第一个纬度是目的性：你想要什么结果？第一周是备战状态，第二周则是教大家如何高效地“吃到快餐”，至少让TD数从零变成一，甚至十个。身为男人，需要释放基础欲望，满足内心真实需求。如果还是处男，却空谈“情感自由”，那是不现实的。你需要先经历，尝试不同“口味”，才知道自己真正想要什么，这才是对自己负责。第二周将涵盖短期关系核心、大量案例解析，以及老童的课程内容。“一夜情缘”说得文艺，说白了就是约炮。具体内容包括：
1.  如何吸引想跟你约炮的女人，而不是每次都只想跟你谈恋爱。

几乎所有的女人都约过炮，这是必然的，只不过她们可能自欺欺人，称之为“恋爱”。选择比努力更重要，选对人，事半功倍。

我们的成功率在业界是翘楚，大概25%-30%，即约3-4个能成功一个。第二周课程旨在满足大家最原始、最真实的欲望。

**第三周：长期关系（灵魂伴侣）**
由道长负责。道长擅长处理多重长期关系。2015年“铁三角”时期，在新夏的“融成计划”中，老童创下“彩虹周”记录（七天七个），而道长则是“家里彩旗飘飘”，曾同时维护七个女朋友（“七仙女”），并懂得如何分配时间、满足不同女友的诉求。如果你只想稳定一个女生，这部分也非常关键。

由大姚负责。理论上我精通各种聊天风格，但个人风格比较强烈。我通常展示的是发散性思维，比如我写的段子集、身边小故事等。如果你真的想在聊天环节达到高深造诣，第四周会带来更多内容。在大多数人的第一认知里，技巧就是学聊天。

今天开学典礼的第一个小时，主要是介绍课程、说明细节。我会多讲一会儿，不只是简单介绍。首先，这是四个星期的安排。

## 情感问题核心框架 🔍

第三，我们正式开始课程内容。大家可能更想知道具体操作，比如修图、找资源、第一句话怎么说等。但本课程更倾向于教你“做什么”以及“如何做”。很多人不知道“做什么”（比如形象改造、拍展示面），知道了又不知道“如何做”得好。更重要的是，你需要明白“为什么这么做”，否则容易陷入瓶颈，思路不系统。

第一节课不会讲太多细节操作，而是告诉你应从哪些方面入手。在大姚的概念里，男性情感问题大致分为四个环节：
1.  **资源**
2.  **魅力**
3.  **互动**
4.  **长期**

无论新旧学员，都可以回顾一下，你的所有问题是否都在这四个环节内。

**1. 资源问题**
很多人追求形式，比如“大扇视频”、刷“探探”等。但我们的目的是收集资源，这些只是形式。时代在进步，形式会变（如“积木”软件已近乎报废，骗子多），但目的不变。
*   **问题**：身边女生资源太少。
*   **标准**：手机里至少维持50个相对鲜活的女性资源。这是及格线。根据研究，人一生平均能维持约150人的关系，除去亲人、朋友、同事，留给女性的位置大约50个。即使在小城市，几十万人口中认识50个适龄女性也并非难事。在一线城市，资源可以上千。我们的门徒曾在两个月内收集1000个资源。
*   **方法**：努力是关键。例如，每天刷满“探探”的500个推荐额度（2015年我们要求学员每天划1万次）。资源收集方式包括社交软件、朋友引荐、街头搭讪等，各有优劣，需要了解。

**2. 魅力问题（自身建设）**
即你是一个什么样的人。包括发型、着装、眉毛、隐形眼镜、耳钉、胡子、指甲、个人风格等。以及如何展示魅力：你很优秀，但需要让别人知道。
*   **原则**：在同级中做到最优。如果你的月收入是1200元，不要和月入120万的人比，但你可以比同样月入1200元的人更优秀——你的妹子更多、质量更好、关系推进更快、甚至妹子为你花钱。这是技巧能达到的。

**3. 互动问题**
这是大多数人认为的“技巧”核心：开始与女性聊天、约会、推进关系。但请注意，就像周杰伦上台表演五分钟，背后是大量的创作和练习。你与女生互动的一句妙语，背后可能有几十个小时的积累。不要轻视前两个环节。

**4. 长期关系问题**
如何选择靠谱的长期对象、维持关系、处理争吵、掌握关系主导权等。

课后我们会在群里分享大姚多年前的旧照，展示形象改造的历程。大家可以通过对比，理解个人风格打造的重要性。

以上是我们的教学思路，也是大家自我剖析的方向。思考自己为何多年未能实现情感自由，问题出在哪个环节。

今天算是开场白，梳理背景思路，并讲解初期形象改造的初步指南。明天开始正式课程。

## 从零到一的快速路径 🚀

一开始我们统计了三个数字：TD个数、年龄、了解我们的时长。这让我们对学员有了大致了解。

我想强调的是，如何快速从无到有、从零到一。当你没有任何情感经历时，我对你说的任何理论你可能都觉得有道理。现在告诉你最快速的蜕变方法。

传统观念中，大部分男人会竭尽全力只追一个女孩（学生时代常见），结果可能花费数月甚至数年时间，最终失败，痛苦不堪。这就像赌博，输了就倾家荡产。

我们教授的观念与此完全相反：**你必须广撒网**。同时拥有大量资源。如果你只追一两个，失败了就全盘皆输。如果你把精力分散在十个妹子身上，即使基础差，也存在成功概率。我们开篇就要求50个资源起步，这叫“磨刀不误砍柴工”。只要资源量足够，从零到一的速度会非常快。

当你从众多资源中成功一个，你就拥有了从陌生到亲密的全流程经验。哪怕过程狼狈，你也知道了沿途风景。接下来，再去接触其他姑娘，成功率会提升，可能十个中能成功八九个。

之后，你会不满足于现状，发现妹子质量有差异。你会开始追求更高质量的女生（比如从“4分”到“5分”）。这时，之前在低分妹身上有效的套路可能失效。你需要重复从零到一的过程：放弃低分资源，进军高分领域，并再次依靠资源基数（如50个高分资源）来确保结果。成功后，你就掌握了这个等级女生的方法，之后应对同等级女生会变得容易。

最终，你会追求顶级高分女生（如明星、网红级别）。这时，你必须进行**自我提升**：提升生活品位、个人修养、文化内涵等。这取决于真实的内在，而不仅仅是技巧。例如，为了接触顶级妹子，我本人也在学习英语。学历、证书是形式，真才实学才是关键。

最终回归起点：自身建设。

## 自身建设开篇：理念与初步指南 ✨

今天先讲自身建设的开篇。第一个概念：不要只做表面功夫，要理解背后的“道”。

我常说：“你那叫头发，不叫发型。”发型是精心打理的结果。秃头可以戴假发、帽子，或修剪出造型。关键在于你的关注点是“生存”还是“生活”。生存与生活之间，只差了一丝精致。头发只是为了保暖遮羞（生存），发型则为了美观（生活）。同样，穿衣是为了遮体保暖（生存），还是为了好看（生活）？

大部分男人的关注点在功能性。但我们要的是美观。举个例子：你想啪啪啪，是为了爽，还是为了生孩子？如果为了爽，请扣一波“B”。（此处为课堂互动隐喻）

你可以按照这个思路塑造自己。例如，我们几位导师都近视，但都不戴框架眼镜，而是戴隐形眼镜，因为更美观。我们打理发型、修理胡子（老童的胡子不同部位长度都不同），这些都可能不舒服，但为了形象我们愿意付出。

**第一个原则：我们的很多建议可能让你不舒服。** 比如垫内增高、戴隐形眼镜、穿修身但不透气的衣服等。但为了形象，需要做出取舍。

个人形象改造分为以下几个大环节，大家可以记录：
1.  **毛发管理**：包括头发、眉毛、胡子、体毛。我个人会剃阴毛，原因：a) 显大；b) 干净；c) 女性更愿意口交。方法：先用剪刀剪短，再用安全剃须刀涂剃须泡沫刮净。新长出的毛茬不会扎人，定期修剪即可。**不要使用电动剃须刀**，以防意外。
2.  **皮肤护理**：大部分直男不护肤。最基本的是控油、保持清洁、去除痘印。明天会系统推荐产品、讲解选购技巧和护肤原理（如油性皮肤不能单纯靠频繁洗脸）。
3.  **衣着搭配**：找到适合自己的风格。我和老童现在的穿着（可能指直播时的休闲装）太随意，不适合初次约会或拍展示面。会推荐网店和购物技巧。
4.  **身材管理**：保持健康体态。关于频率，每周约三次是正常的性生活频率，不会导致“虚”。我个人欲望较低，每周一两次即可。
