## 思维模型：ABC理论

在第一节课中，我们介绍了**ABC思维模型**。这个模型能帮助你清晰地理解，为何人与人面对相同事件时反应会截然不同。

**公式：A (诱发刺激) → B (信念系统) → C (行为结果)**

当时我举了一个“打巴掌”的例子：有人打了另一个人的右脸，却可能引发三种不同的反应。这说明了，每个人对同一件事或同一句话的反应都不同。造成这种差异的根本原因并非性格，而是每个人内在的**信念系统(B)**不同。而信念系统的差异，又源于他们所经历的**诱发刺激(A)**本质不同。

以下是优秀学员的应用案例：
*   **场景迁移**：一位同学将这个模型应用到古代名将韩信“忍受胯下之辱”的故事中。如果韩信当时无法控制情绪，选择了“士可杀不可辱”，就不会有后来的十面埋伏和击败项羽。这体现了模型的应用价值。
*   **深度理解**：另一位同学总结道：不同的人在相同的A刺激下会产生不同的C，是因为B不同；而同一个人在不同时间、场景下对相同的A也可能产生不同的C。
*   **结合自身**：该同学还将此模型结合自己对健身的感悟进行分析，做到了举一反三。

## 逻辑思维：归纳法与演绎法

上一节我们介绍了思维模型，本节中我们来看看逻辑思维。在第二节课，我们讲解了逻辑思维，并让大家练习用归纳法和演绎法去解释一个问题。

一位学员选择的问题是：“为什么男生大多会被美女拒绝？”

以下是他的分析过程：
*   **使用归纳法**：他观察到，好看的姑娘追求者多，作为稀缺资源，供需不平衡使得她们有资本选择优质男性。因此归纳出：美女倾向于选择价值更高、更能满足自身需求的男性。此外，女性如果仅靠外在，在25岁后价值可能走低，因此会在黄金时期最大化选择权。
*   **使用演绎法**：他假设“像高价值男人一样对美女进行高物质投入，即可追到美女”，并列举了硬价值。但他也意识到，现实中一味投入可能并无好结果，这说明了演绎法需要结合实际情况。

我们大部分日常思考都依赖于**归纳法**。例如，如果你身边炒股的人都亏钱，你就会归纳为“炒股大多亏钱”；如果身边人都赚钱，你就会觉得炒股容易。这就是归纳法的运作方式。

此外，我们还探讨了“为什么”和“凭什么”，这两者用于探寻事物本质；而事实判断、功利判断、价值判断、审美判断则用于做出决策，两者目的不同。

该学员还设计了自我改良方案：
*   每天锻炼逻辑思考能力，对身边事物进行全流程思考。
*   对周围人的问题，用假设演绎的形式进行推敲。
*   学习他人逻辑上的优点，弥补自身短板。

人的逻辑基于已有认知进行分析。许多人无法分析问题，一是因为懒惰，不愿突破思考的舒适区；二是因为缺乏必要的认知储备。

## 实践、复盘与风格养成

理论与实践结合至关重要。在聊天实践中，关键是要激发你“渴望解决沟通问题”的内心动力，而非带着“必须搞定对方”的欲望。后者会让你陷入各种聊天困境。

以下是有效的复盘步骤：
1.  **完整记录**：将一次从认识到结束的聊天完整截图。
2.  **逐句分析**：在每句话旁标注，分析自己为何这样说，当时如何想。
3.  **对比预期**：如果对方的回复出乎意料，思考为何对方没按你预期回应。
4.  **修正认知**：这能检验你认知的偏差，明确需要升级和完善的地方。

通过持续复盘，你会进步神速。成功不可复制，我的聊天风格不一定适合所有人，但沟通中的重要节点必须掌握。当你能与人良好沟通、走心交流，让对方感到有趣并喜欢上你，后续的一切便水到渠成。

课程的价值取决于你努力的程度。学习与成长是相辅相成的，如同经营公司，需要双方朝着共同目标努力。

以下是高效学习的具体方法：

当你能够将所学知识应用出来，才意味着真正完成了内化。分享也是极佳的提升方式，越分享，越能发现自己的不足，并获得新的心得体会。
