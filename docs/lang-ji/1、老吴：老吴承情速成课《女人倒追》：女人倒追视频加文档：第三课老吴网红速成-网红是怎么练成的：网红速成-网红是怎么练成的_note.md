## 核心修图流程详解 🛠️

了解了工具后，我们进入实战环节。本节将以一张街拍原图为例，分步演示完整的修图过程。

### 第一步：脸部精修（使用Facetune）

首先，我们需要对脸部进行精细化处理，这是提升照片质感的基础。

1.  **磨皮（平滑功能）**：针对皮肤上的斑点、痘印或不平整处进行平滑处理。如果问题较严重，可使用“更加平滑”功能。
    *   **核心概念**：`平滑功能 -> 磨皮`
2.  **调整头部与脸型（调整功能）**：如果觉得头部或脸部较大，可使用“调整大小”功能先缩小整体，再用“调整”功能进行局部微调，使脸部显得更小巧精致。
    *   **操作顺序**：`调整大小（缩小） -> 调整（局部微调）`

### 第二步：身材比例调整（使用美图秀秀）

脸部处理完毕后，我们来优化身材比例，让整体视觉更协调。

以下是美图秀秀中用于调整比例的关键功能：

*   **面部重塑**：对五官进行微调。
*   **瘦脸瘦身**：用于局部细微调整。
*   **去痘**：点状瑕疵修复。
*   **增高（拉长）**：**本课核心技巧**：将图片底部（脚部区域）拉长，同时将上身区域缩短。这样可以营造出“腿长身短”的优越身材比例，使人看起来更高。
    *   **核心概念**：`拉长腿部 + 缩短上身 = 优化身材比例`

### 第三步：滤镜应用与质感提升（使用Instagram）

调整好人物主体后，最后一步是为图片注入风格和质感，这是网红照片的点睛之笔。

1.  **选择滤镜**：导入图片后，在下方滤镜库中选择一个喜欢的风格（例如：HUDSON, AMARO, SIERRA）。
2.  **关键技巧：调整滤镜强度**：点击已选滤镜，会出现一个0-100的强度滑块。**切勿使用100%强度**，那样会显得假。将强度调整到**50-70**之间（例如61），能让图片在提升质感的同时保持自然。
    *   **核心公式**：`滤镜强度 = 50% ~ 70%`
3.  **对比与保存**：可以尝试对同一张图片应用不同滤镜并调整强度，对比后选择效果最佳的一张保存。

## 不同场景修图案例 🖼️

掌握了基本流程后，我们来看看如何将这些技巧应用到不同场景中。

### 案例一：旅行打卡照（吉隆坡独立广场）

1.  **原图问题**：原图因光线问题脸部较暗，构图包含多余杂物，姿势像普通游客照。
2.  **修图步骤**：
    *   使用Facetune和美图秀秀调整脸部和身材比例。
    *   使用美图秀秀的“编辑”功能裁剪掉图片中多余杂乱的部分。
    *   使用Instagram的HUDSON滤镜，并调整强度至明亮舒适的程度。
3.  **效果对比**：修图后照片明亮、干净、有质感，人物突出，摆脱游客照感觉。

### 案例二：静物与美食（糖果/马卡龙）

1.  **修图重点**：强调色彩和食欲感。
2.  **操作演示**：直接使用Instagram滤镜（如CLARENDON或AMARO），并将强度调整至70左右，能让食物颜色更鲜艳、画面更诱人。

### 案例三：网红风格参考（风景与生活）

通过展示一组网红风格的风景/生活照（使用Instagram修图），让学员建立对“优质展示面”的视觉感知。这类图片通常明亮、干净、有生活格调，能传递出良好的审美和生活层次。

## 常见问题与核心原则 ❓

在修图过程中，请牢记以下原则以避开常见误区：

*   **滤镜强度**：永远避免使用100%的滤镜强度，50-70%是安全且自然的效果区间。
*   **修图分寸（尤其男生）**：修图目的是“干净、自然”，而非过度改变容貌。应在原有基础上优化，避免修得“很假”或“很娘”。
*   **设备建议**：为保证图片颜色在不同手机上显示一致，建议使用**苹果手机**进行修图和发布。安卓手机可能存在色差问题。
*   **自拍建议**：多数女生不喜欢男生自拍，会显得做作。建议请路人帮忙拍摄，多拍几张选取最佳。
*   **Instagram使用**：只需在注册时使用VPN翻墙，登录成功后日常修图无需VPN。
