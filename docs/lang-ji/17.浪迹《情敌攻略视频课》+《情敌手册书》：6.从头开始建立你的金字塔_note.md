# 课程17：浪迹《情敌攻略视频课》+《情敌手册书》 🎯 6. 从头开始建立你的金字塔

在本节课中，我们将学习如何通过绘制个人频率偏移图，来诊断你在与异性交往不同阶段中可能存在的问题。核心在于理解并应用“30%递减”原则，通过量化分析而非纠结具体话术，来发现并解决自身问题。

## 概述：频率图的意义

上一节我们介绍了“频率”和“70%”的核心概念。本节中，我们来看看如何将这些概念具体化，即绘制属于你自己的频率金字塔图。这张图能直观地揭示你在各个交往阶段的表现与标准模型的偏差。

## 绘制标准频率金字塔

正确的频率金字塔每一层应按**30%** 递减构建。这是因为你的主动进攻频率只有**70%**。标准模型图示如下（此处为概念描述，书中附有具体图示）：

*   认识的女生的总数设为100%。
*   能进入聊天阶段的女生约为上一层的70%。
*   能成功邀约见面的女生约为聊天阶段的70%。
*   愿意再次约会并关系升级的女生约为见面阶段的70%。

以下是构建金字塔的核心递减公式：
`下一阶段人数 ≈ 上一阶段人数 * 70%`
（或表述为：每阶段过渡流失率约为30%）

## 诊断个人频率偏移

当你画出自己的频率图后，肯定会发现它与标准图存在差异。这种差异就是“偏移”。通过对比，你可以定位问题发生的具体阶段。

我将交往过程划分为以下几个关键阶段，每个阶段的正常过渡都应遵循约30%的递减率：
1.  你认识的女生
2.  你能够聊天的女生
3.  聊天后愿意与你见面的女生
4.  见面后愿意再次约会的女生
5.  能进一步关系升级的女生

以下是每个阶段频率偏移可能对应的核心问题（书中已详细列出）：
*   **阶段1到2偏移**：可能涉及初始吸引力或社交圈开拓问题。
*   **阶段2到3偏移**：可能反映聊天技巧或兴趣建立能力不足。
*   **阶段3到4偏移**：可能说明线下互动或安全感建立有待加强。
*   **阶段4到5偏移**：可能关乎关系深入或吸引力持续的问题。

## 核心行动步骤

现在你要做的是：
1.  根据书中指导，回溯你的真实经历，绘制出个人的频率偏移图。
2.  对照标准模型，找出频率在哪个阶段发生了显著偏移。
3.  根据书中对应章节，理解该偏移所揭示的自身问题（例如，可能是“把妹风格不一致”等概念性问题）。

请注意，我在这里完全不讨论“某一句话应该怎么说”。因为你面对的是一个由多种类型个体组成的女性群体，讨论具体话术效率低下。你只需关注**频率**。只要频率发生偏移，就一定能找到对应的自身问题。你的任务是解决自身问题，而非解决“女人”的问题，因为总有约30%的女生在任何阶段都不会对你产生兴趣。
