# 浪迹《恋爱规则》课程01：第二十五集 🧭 方向比努力程度更重要

在本节课中，我们将要学习一个核心的恋爱观念：在追求爱情的过程中，选择正确的方向远比单纯的努力程度更为关键。

上一节我们探讨了恋爱中的基础心态，本节中我们来看看如何将努力用在正确的方向上。

让瘸子去跑步，让瞎子去射击，让哑巴去唱歌。让笨蛋当科学家，这些选择都可能让他们终身一事无成。

但是，如果让瘸子去射击，他断了的腿或许正好能稳定地当作枪托。让瞎子拉二胡，他可能会成为下一个阿炳。让一个在数学上没有天赋、缺乏逻辑概念的人去唱歌，他可能会成为歌唱家。而让一个哑巴去做计算工作，他可能会成为下一个科学家。

因此，我们在学习时讲究因材施教，在恋爱时，同样也是这个道理。你应该选择适合你的女生去谈恋爱。

以下是几种不同恋爱目标所对应的方向选择：

*   想要感受年轻的刺激，你就去蹦迪。
*   想要一起奋斗的爱情，可以找个同行。
*   想要结婚或安稳生活，你可以找一个年龄大的、想结婚的女生。
*   你想要宝宝，就找一个喜欢宝宝的女孩。

很多时候，我们遇到的恋爱问题，根源在于一开始就选错了对象。他去电影节搭讪爱玩的女生，回来后一直念念不忘，想和对方谈恋爱。而真正想和他谈恋爱的女生，他又只想和别人愉快玩耍。这最大的原因，就是在一开始的时候，选错了对象。

那么，要怎么样才能明白你适合什么样的女生呢？个人认为，需要经过千百次的试错过后，你可能才会知道什么适合你。

所以，如果你还年轻，你应该放手去爱，不怕被伤害。用你的身体，用你脆弱的内心，去经历这些冲击，去经历这些暴风雨的洗礼。在伤痛中成长，最后你会明白，你到底要的是什么样的恋爱。

本节课中我们一起学习了：在恋爱中，**方向 > 努力**。正确的选择是成功的基础。通过明确自身需求、敢于尝试并从经历中学习，才能找到真正适合自己的恋爱关系与伴侣。
