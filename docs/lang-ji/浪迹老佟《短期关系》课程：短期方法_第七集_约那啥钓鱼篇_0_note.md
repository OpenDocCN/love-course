# 浪迹老佟《短期关系》课程：短期方法_第七集_约那啥钓鱼篇_0 🎣

在本节课中，我们将要学习一种被称为“钓鱼”的社交技巧。这是一种通过发布特定内容的朋友圈，来筛选并吸引潜在约会对象主动联系你的方法。我们将详细拆解其原理、操作步骤以及注意事项，帮助你高效运用此策略。

## 概述：什么是“钓鱼”？

我们经常会发现有的女生会发类似这样的朋友圈：“盗墓笔记上映了，谁陪我去看呀”、“什么什么电影即将上映了，有没人跟我一起去看”、“好想吃小龙虾，好想吃火锅等等”。或者说她就很直截了当地说：“有没人陪我去吃火锅”、“有没人带我去哪哪玩”。我们把姑娘这种行为叫“钓鱼”。

我们广大的男性同胞呢，看到这条朋友圈就激动得不行，纷纷地去点赞，纷纷地去评论，甚至有的迫不及待地就开始去切私聊了。

## 第一节：如何判断“钓鱼者”的资源量

那么我要明白一个事情，就是你去判断这个姑娘有多少资源，她是不是一个资源收集者。

怎么去判断呢？你会不会经常撞到一些妹子，比方说你看到有其他的男人，或者说其他姑娘在她的朋友圈下面去评论。那你就明白了，这个姑娘她有非常多的资源，她可能有数以千计，甚至是一两千、两三千，甚至微信都快满了。

这种情况下，你看到姑娘发了这个朋友圈，我肯定会选择回避去评论她、去私聊她这个事情。因为我明白她背后有太多的资源，你的竞争对手太多了。

那另外一种情况，你可能就直接给她评论、给她点赞了。她的资源如果说没那么多，就意思是你的竞争对手没那么多的话，那么你的成功率可能会高一点。但是你要记住了，对于这种钓鱼的姑娘，也不是省油的灯，你是不是需要充当一个买单的角色。

## 第二节：男性“钓鱼”的完整套路

根据姑娘的这种朋友圈的发法，我们总结出来一套男人如何去钓鱼的套路。

### 1. 选择正确的时间段

首先，时间段是在半夜零点到三四点之间，跟之前我们约宵夜的套路是一样的。

### 2. 发布精心设计的朋友圈内容

在这个时间段，我们去发一个什么样的朋友圈呢？

以下是朋友圈内容的核心要素：

*   **包含个人照片**：首先要有你自己的照片，可以是自拍，也可以是别人帮你拍的。如果说你有身材照会更好一点。你一定要把你最帅最酷的一张照片拿出来去发。
*   **确保时间吻合**：这个时间段也要吻合，就是大家避免掉白天，避免掉拿白天的朋友圈去充当。你的照片的时间是跟你发的时间要吻合的。
*   **使用暗示性配文**：你的配文千万不要学姑娘们这些目的性很明确的表达。我不会说“有没人出来吃宵夜”、“有没人出来玩”。我会用一种暗示性的描述。

我给大家举一个例子。之前我在大概晚上两点钟左右，也去发了一个钓鱼的朋友圈。

这个朋友圈我是怎么发的呢？我在大马路上和一个男生还有一个女生的，我们三个人的合影。那么他们俩是一对（其实他们俩是不是一对并不重要，你只需要发出这个朋友圈）。这个朋友圈是在大马路上面夜深人静时拍的，可能路上都看不到行人、看不到车辆了。

我的配文是如何描写的呢？我说的是：“二位走好。”

我用这个配文向我的朋友圈里的资源暗示一个事情：他们俩是一对，他们俩待会就要走了，有可能去开房了，也有可能去干嘛干嘛去了，总之最后剩下来是我一个人。这个就给姑娘极大的暗示：待会我一个人可能要在这个马路上回家、搭车回家了，或者说去其他地方了。

如果说你的资源库里面，平时那些对你有窗口、反应还不错的姑娘，这个时候她有可能就会给你点赞、给你评论。那么你顺势去约她就会简单很多。

## 第三节：为何不用群发而用朋友圈？

有的兄弟难免会问，我为什么不用群发的套路去约姑娘呢？

群发会有几个问题：

*   **第一个问题**：睡着的姑娘她看不到。那他第二天就会看到你这个人怎么大半夜给我发了一条消息，奇奇怪怪的。一般正常我们聊天都是发生在晚上或者说白天，深更半夜你找别人聊天，很显然背后姑娘可能会觉得你图谋不轨。
*   **第二个问题**：你群发出去这些没睡着的姑娘呢，她有的看到之后，她并不想理睬你。她可能对你一点窗口都没有，那么你这个群发就是无效的。

但是发朋友圈这个事情就规避了刚刚的两种情况。

*   **对于第一种情况**：睡着的人他看不见。有的兄弟说他第二天醒来不就看见了吗？那么你发的这个钓鱼的朋友圈，不管你这条鱼有没有钓成功，这条朋友圈最后都是要删掉的。所以说睡着的姑娘他看不到。
*   **对于第二种情况**：没睡着的姑娘呢，她有的看到你的朋友圈，她就扫了一眼，可能就翻过去了，也没想跟你评论，也没想搭理你。或者说你朋友圈发得不够好，或者说也是你平时吸引不够，她对你本来就没有窗口。那么对于这些人也就算了。

我们说愿者上钩。那些平时对你窗口大的，这个时候就给你切私聊、给你评论、给你点赞。

## 第四节：发布前的关键设置

我们还要注意一点，就是你发这个朋友圈的时候，一定要屏蔽那些无效资源。

什么是无效资源呢？就是除了你这些“鱼”之外的，比方说你的亲戚朋友，比方说男性同胞，全部屏蔽掉。因为他们给你点赞、给你评论没有任何意义。

那么还有一类人群呢，这个至关重要。有一个小细节，不知道大家有没有关注到，就是你要把这些你平时有在白天约会、你平时有在用心经营的这些姑娘给屏蔽掉。你可能想跟他走一段长期关系，你可能想让他当你的女朋友。那这些人呢，你千万不要让他看到你在半夜发了一条钓鱼的朋友圈。不然他之前可能对你印象还不错，结果发现你这个人半夜在这约炮，他可能就对你印象瞬间全无。

## 第五节：如何回复与推进

这条朋友圈发出去之后呢，对于这些点赞和评论，你如何去回复？

有的兄弟说，那我就在下面评论他的回复呗，然后我再切私聊。那其实呢，我们正常的套路是这样。但是对于钓鱼这个事情呢，就是时间短暂，而且是在深夜，我们的目的就很明确了。而且给你点赞、给你评论的，窗口都是很大的。

这个时候我统一切私聊，不管是点赞还是评论，全部切私聊。

那如何去发这个消息？你可以发一个表情。这个表情，跟我们平时发的又不一样了。我们平时可能发一个笑脸、发一个调皮的表情。那么我们在钓鱼的朋友圈当中呢，我们切私聊的时候，就直接发这个害羞的表情，或者直接发眨眼的表情，具有一些性张力的一些表情。因为很显然，这个点还给你点赞评论的，窗口已经很大了，就直接了当一点。

或者你直接一句话回复他，你就说：“还没睡？”、“也没睡呢？”。然后下面的套路就是什么呢？就是我们约宵夜的套路了。你也不用跟他建立太多的舒适感，你也不用跟他扯太多。你就直接从这么多的有效资源、你的精准目标里面，直接去筛选，约他出来宵夜。

这么多有效的资源，你只需要挑一个出来，就可以完成当晚的钓鱼。

## 第六节：如何识别女性的“钓鱼”信号

我们前面说，这个姑娘她钓鱼是为了别人约她请她吃饭、别人请她看电影等等其他的事情。当然我们也不排除一些女人，她真的会在晚上，跟我们男人做同样的事情，去真正的去钓鱼，真正的去约男人出来愉快玩耍。

那么你如何去寻找这些目标呢？兄弟们，可能它是一个相对辛苦的事情。

我们平时没事，所有的朋友圈都会看，会关注女人的动态，关注女人发朋友圈是一个什么样的状态。那么在深夜零点钟到三点钟这个时间段，你会发现发朋友圈的人就非常少。但是你也会偶尔发现，有一些女人在12点到3点之间这个阶段，去发一些真正的钓鱼的朋友圈。

说白了，这个时候只有眼疾手快的人，和社交直觉强的人，才能分辨出她这个朋友圈是在干嘛。

我的印象当中呢，我被钓鱼的，我有几个被钓鱼的经验，大多都是发生在这个12点过到3点之间。

*   **案例一：文字暗示**。有一次是有一个姑娘，发了一个朋友圈，叫做“喝多了”三个字，然后什么也没有了。我就明白了，这个姑娘现在一个状态，在酒吧喝酒而且喝多了。如果她跟朋友玩得很开心、很愉快，她是不会很单纯地发这样一段文字的。我就判断出来，她可能是有想法。我没有评论她，我就直接切私聊，我就说“还没睡呢？”，然后她说“喝多了”。然后我就直接说了一句：“需要借宿吗？”，然后就这样一来一往，就把她约到了我的家里面。
*   **案例二：表情暗示**。之前还有一个姑娘，她发了一个不开心的表情。有的兄弟不明白，这个姑娘为什么大晚上发这样一个表情，她是背后有什么含义吗？我跟你说，她在大晚上发这样的表情，就是想约。你会发现她其他的朋友圈，都是一些丰富多彩的，有自拍，有各种吃喝玩乐的一些。但是她今天晚上发的就有点特殊，她发了这样一个不开心的表情。然后这个时候我也直接切了她的私聊，然后就约了出来，我是去了她的家找的她。

## 第七节：实施“钓鱼”的前提条件

还有一个事情，就是我发现很多兄弟呢，他资源比较少，可能只有三四十、四五十人。那这种情况你是不适合钓鱼的，因为你的鱼池太少了。

可能你鱼塘很大，但里面的鱼太少。你说你可能跑到一个角落去钓鱼，但是你可能钓一天都钓不上来一条。所以说呢，它一定是建立在你有足够的资源的基础之上。就最起码一二百人的这个朋友圈，你才会钓到鱼。

当你的朋友圈的资源达到四五百，说真的，你每天晚上什么也不用干，你也不用跟姑娘出去约会，你直接每天就在家躺在床上钓鱼就OK了。

## 总结

本节课中，我们一起学习了“钓鱼”这一社交技巧。我们从识别女性的钓鱼行为开始，详细讲解了男性主动“钓鱼”的完整流程：**选择深夜时段 → 发布包含个人帅照和暗示性文案的朋友圈 → 精准屏蔽无效及长期关系对象 → 对点赞评论者统一切入私聊并直接邀约**。同时，我们也探讨了如何识别女性发出的“钓鱼”信号，并强调了足够大的资源基数（建议200人以上）是成功实施此策略的前提。记住，这是一种筛选机制，核心在于吸引对你已有窗口的目标主动上钩。
