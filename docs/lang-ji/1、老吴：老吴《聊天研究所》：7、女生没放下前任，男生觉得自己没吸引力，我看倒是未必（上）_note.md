# 1、老吴：老吴《聊天研究所》：7、女生没放下前任，男生觉得自己没吸引力，我看倒是未必（上）_note

在本节课中，我们将要学习一个复杂的聊天案例。我们将通过分析一段真实的对话，来理解在追求阶段，男生如何因错误的沟通方式错失良机，以及女生看似拒绝的背后可能隐藏的真实兴趣。这个案例将分为三节课讲解，本节课是第一部分。

## 案例背景与开场

首先，我们看到男生发起对话，他问女生：“在干嘛”。这是一种询问对方状态的开启方式。

接着，女生很快回复：“刚回来”。

## 核心概念：价值索取与情感账户

“在干嘛”这三个字，从专业角度被称为**索取价值**。即通过提问去了解对方信息，例如问“喜欢什么”、“多高”等，类似于查户口。

这种行为之所以发生，十有八九是因为对对方感兴趣。然而，这类问句（如“吃了吗”、“睡了吗”）不能随意使用。它就像从银行取款。

我们可以把男女双方的关系，想象成一个共同的情感账户。这个账户里存着双方的好感与默契。如果账户里本来就没有多少“存款”（即情感基础），而你一直在“取款”（索取价值、问问题），那么这个账户很快就会透支。

这就是为什么很多人不断提问时，会让对方感到厌烦。因为你们之间的共同账户没有“存钱”（建立舒适感和吸引力），你却一直想“取钱”。

在这个案例中，男生问“在干嘛”相当于试图取钱。女生回复“刚回来”相当于把钱给了他。

## 男生的铺垫与女生的好奇

这时，男生描述了自己的状态：“我过来攀鱼”。这句话说明女生也在攀鱼，但他没有说具体来做什么，从而引起了女生的好奇。

女生于是问：“你在攀鱼干嘛？” 这表明她对男生的行为产生了兴趣。

女方会这么问，说明她对男方有一定兴趣。此时，男方表达了真实意图：“刚送朋友过来想要见见你”。

因此，他前面的两句话，第一句是打开聊天，第二句则是为“想见面”这个想法做铺垫。

## 女生的回应与时间节点分析

接下来，我们来看女生的回复。女生先发了一个害羞的表情，然后说：“那么晚，我正准备去洗澡呢”。

在我的情商课中，讲到过人的信念系统。我们站在女生的角度思考：在晚上十点半，收到一个男生说想见面，女生会有什么想法？对于一个女生来说，之前没有任何招呼或确认，突然在大半夜提出见面，这是非常敏感的。

如果她答应出来，说明她对你的兴趣度极高。但女生通常的思维是：“一个男生这么晚约我，如果我答应，会显得我很随便。”

接着，女生描述了她当下的状态“正准备去洗澡呢”。这其实是在给男生一个理由：时间很晚，我还要洗澡。言下之意是：我们不要见面了。

当然，这不代表她不想见这个男生，只是这个时间点确实尴尬。

## 男生的错误应对与自我中心

作为一个正常的男生，此时可以采用共情的方式。例如可以说：“这么晚突然找你，其实让你也挺为难的吧？” 这样说出女生的心里话，然后观察她的回复，来判断邀约是否可能成功。

然而，这个男生回复的是：“是啊这么晚了就想见见你”。他再次强调了自己的意图，目的性非常强。

我们来看女生的回复。她说：“这么晚了见面还能做什么”，并发了个无脸表情。接着又说：“想洗澡洗头敷面膜”。

如何分析这两句话？
1.  女生问“这么晚还能干嘛”，相当于在要求一个**非常合理的理由**，她才有可能出来。因为按理说，如果女生对男生没兴趣，完全可以不回复，或直接说“下次早点约”。大家都知道男人这么晚约她想干嘛。
2.  女生同时再次强调她想洗澡洗头敷面膜。并且，女生最后这几句话都带有皱眉、无脸的表情，说明她其实也有点尴尬，不好意思直接拒绝。

男方此时又说：“见一面胜过千言万语说不出来什么”。我刚刚提到，女生需要一个合理化的理由。如果你突然这么晚约我，必须有一个能说服我的理由。如果你的理由无法说服我，我为什么要出来？

再看男生说的这些话，他整个聊天过程总是**站在自己的角度说话**，没有考虑女方的感受，一直处于一种很自我的状态。

## 关键线索：女生的真实态度

因为男方没有给出合理化的理由，只是单方面表达想法，所以女生再次强调：“可是我想洗澡呀”。

这时还有一个重要信息：女生回复这句话时是晚上10点37分。我们倒回第一张截图，男方邀约是在10点16分，中间隔了20多分钟。

一个正常人如果说要去洗澡，基本上马上就去洗了。她隔了20分钟都没洗，这说明了什么？我的分析是：这个女生还没洗澡，可能还没卸妆，其实简单打扮一下就可以出来见他。她只是在等男方给出一个合理化的理由。

所以，这个男生的处理又出了问题。他居然说：“洗完澡出来呗”。这完全不懂女生。

一个女生洗完澡、敷面膜等，需要很长时间，而且那时是素颜状态，可能已经换上睡衣准备休息。你让她洗完澡再出来，这非常不合理。

从这个回应又能看出，这个男生很自我。所以男同学们遇到类似情况，千万不能像他这样处理。

## 正确的应对策略与案例总结

这个案例中的男生已经追求这个女生一段时间了。女生还没放下前任，但我看了聊天记录后，很明显能发现女生对这个男生有兴趣，只是男生在一些处理上没做好。

如果是我的话，我会这样回复：“反正你现在还没洗澡，其实简单整理一下就可以下来了，对吧？” 这是一种**逆向合理化**的回复方式，让女生觉得：“对哦，反正现在出来也很方便。”

因为第一，男生也在攀鱼，女生出门就能见到。可以说：“反正刚好过来了，你现在下来，我们去吃个宵夜，有些事情在微信上也说不清楚。”

首先，女生一直没去洗澡，可能确实也想见这个男生。所以，男生应该让女生快速做出判断，并且不要让她觉得出来压力很大。

这时，女生说：“我洗完澡都多少点了”。因为女生洗澡确实需要时间，洗完都很晚了，还见什么？其实男生等到这时候才说“我就在附近”，这个处理相对晚了。应该用我刚刚说的方式，让女生觉得赶紧现在还能出来就出来。

如果实在约不到，也不要勉强别人。我们要尊重女生。本来你大半夜突然约人家，就是一种不太尊重的表现，毕竟你们还不是男女朋友关系。

我们看到女生的回复，隔了十几二十分钟才回：“回去吧，那么晚了，洗完澡都11点了，你个大猪头”。

我觉得这个女生，从10点多说要洗澡，讲到10点52分，讲了那么久一直没休息，正好印证了我刚才说的：她其实只需要一个合理化的理由就可以出来。而且女生叫他“大猪头”，这是一种比较亲昵的叫法，说明女生对他有好感。

男生一直没有做出合理化的回复，接着又说：“还不是因为想你，感觉每天都在想你，做什么事都不能专注”。

有没有发现，男生又沉浸在自己的世界里了？总是在“我我我我”，说“因为想你每天做事情都不能专注”。这相当于把责任推到别人身上。想一个人是合理的，但想到无法专注，这有点过了。

此时男生的回复已是晚上11点。从对话开始到现在差不多40分钟，本有足够时间做出更正确的选择，但他一直没有回复好，导致时间越拖越晚。时间越晚，女生出来的可能性就越低。

从这部分可以看出，女生其实一直很客气、很委婉，也给男方台阶下，但男方一直处于很自我的状态。
