# 1、老吴：老吴《中级情感私教》：七、情绪管理_note

在本节课中，我们将要学习情绪管理的核心概念与实践方法。情绪是我们所有思想、言语和行为的底层驱动力，理解并管理好情绪，是获得幸福与成功的关键。

## 什么是情绪化？

上一节我们介绍了自我认知，本节中我们来看看情绪如何塑造我们。人是100%的情绪化动物。情绪化指的是一个人的思想、言语、行为和决定等一切，包括人生观、价值观和世界观，都由自身的情绪所决定。

生命只与我们的感受有关。每一天、每一个当下的思想、言语和行为，都受到自己情绪的影响。然而，大多数人并未意识到这一点。许多人认为是外在的情况或原因导致了自己产生某种情绪感受，但事实正好相反：**是因为我们内在有什么样的情绪感受，才会导致我们看待世界的方式发生变化**。我们的外在世界，是内在世界的一个投射。

## 情绪如何影响我们的判断与关系？

情绪状态直接影响我们的能量水平、判断力和人际关系。当我们处于高能量、感觉好的状态时，我们能感受到舒服、自在、感恩和满足。此时，我们的思想会变得非常正向，会说正确的话，做正确的决定和事情。

相反，当我们处于低能量、负面情绪时，我们的思想、言语、行为和决定都会变得非常负能量。更重要的是，**同样一件事情，在能量高和能量低时，我们看待它的角度、做出的决定和产生的想法会完全不同**。

以下是情绪影响关系的一个具体例子：

假设一对男女去看一部悲伤的电影。女生看电影时不断流泪。男生的反应会因双方关系和女生当下情绪而异。一句简单的“**你还好吗？**”往往比“别难过了”或仅仅递纸巾更有力量。因为这四个字表达了对对方感受的关心和理解，而非试图解决问题或否定情绪。

## 痛苦的来源与“圆满”的真谛

我们的人生并非一帆风顺，总会遇到挫折、困难和挑战。宇宙是二元对立的，我们的内在世界也是如此。我们体验过多少开心、愉悦，就会体验多少痛苦、绝望。

很多人追求“完美”，但完美没有具体标准。我们应该追求的是“圆满”。真正的圆满不是只要正面情绪、不要负面情绪，而是能够**坦然接受黑白并存的世界**，体验人生的各种滋味，并始终保持初心，有意识地创造自己想要的生活。

## 如何识别与处理痛苦（负面情绪）？

所有问题的核心背后往往是“受苦”，即感受到各种负面、不舒服的情绪。大多数人从未思考这些感觉从何而来，总是向外寻求答案，试图通过解决外在问题来改善内在感受，但这往往陷入“解决一个问题，又出现新问题”的循环。

关键在于：**先解决情绪，再处理问题**。如果不处理情绪，我们就无法从痛苦中解脱，并可能在负面情绪中说错话、做错决定。

正向思考很重要，但其背后的原因是**正向的能量**。我们的能量状态决定意识状态。人是情绪化动物，所有理性思考也受当时情绪感受决定。每个决定背后都有动机，这个动机就是我们当下的感受。

生命只与感受有关。想要人生幸福美满，就需要对自己的感受拥有绝对的觉察力，把感受放在第一位。

## 提升情绪管理能力的核心练习

认识自己之后，还要认识自己的情绪，因为情绪是流动变化的。提高情绪管理能力，就能在遇到问题时更好地处理情绪。

有一个关键的心理转换练习：**使用第三人称描述自己的感受或行为**。

例如，将“我感到很生气”转换为“他感到很生气”。这样，你就从承受情绪的主体，巧妙地转换成了观察者，让自己能“看见”自己。

这背后的思维模型是：在遇到问题时，先把自己抽离出来，用上帝视角（第三人称）去看待自己和问题，而不是陷入自己的主观视角。

## 痛苦的四个层面与“臣服”的智慧

痛苦（感到不开心）分为四个层面：
1.  **物质层面**
2.  **身体层面**
3.  **头脑（大脑）层面**
4.  **精神层面**

痛苦本身是一种恩典，它让我们经历、觉醒。常见的负面情绪有后悔、愤怒、恐惧、悲伤、羞耻等。

提高情绪管理能力，是为了在面对痛苦时，能够快速缩短痛苦的时间。**人不可能没有痛苦**。生活有多光鲜，就可能经历多少苦难。痛苦是普遍的。

根据对痛苦的觉察和处理方式，可以把人为为四种：
1.  **无明者**：完全觉察不到自己正在受苦。
2.  **以苦为荣者**：知道自己在受苦，却以此为荣（如信奉“吃得苦中苦，方为人上人”但未获提升）。
3.  **习惯受苦者**：知道自己受苦，并以此为习惯（如某些固守痛苦模式的老人）。
4.  **寻求解脱者**：知道自己受苦，并想寻求解脱方法。如果你处于此阶段，恭喜你，你已走在正确的道路上。

## 当痛苦来临时的“锦囊妙计”

当处于负面情绪时，我们的智商可能是“负数”，会说错话、做错事，伤害自己和他人，例如通过熬夜、酗酒来逃避，但这只会让第二天更难受。

此时，请遵循以下步骤：
1.  **停止行动，停止说话，不要做任何决定**。
2.  **承认并接纳**：承认自己此刻正在受苦，接纳自己可以有负面情绪。不要与情绪对抗。
3.  **允许释放**：如果想哭，就哭出来。眼泪是灵魂的洗澡水，是一种释放。要像孩子一样，想哭就哭，想笑就笑。
4.  **进行自我对话与臣服**：可以对自己说：“我现在很不舒服，我也不知道该怎么办。我选择臣服。” 臣服不是放弃，而是承认当下能力可能无法解决，停止纠结和执着，调整好情绪。往往情绪调整好后，解决方案或贵人会自动出现。
5.  **寻求帮助**：如果问题确实无法独自解决，该学习就去学习，该找能帮忙的人就去找。冷静时，你才能思考谁可以帮你。

## 情绪防身术：如何应对外在的语言冲击？

我们的情绪很容易受到他人话语的影响。但需要明白：**并非那句话让你不舒服，而是你如何解读它**。很多人说话只是凭习惯，并无恶意。

关键在于：**不接招**。所有的情绪交流需要一来一往才能成立。如果对方丢来一个“羞辱的球”，你接住并丢回去，羞辱才成立。如果你不接这个球（不认同、不反击），对方的攻击就无法完成。

例如，当有人用言语攻击你时，尝试“同意”对方的说法。
*   攻击者：“你这个笨蛋！”
*   回应：“嗯，是啊，我有时确实会做些蠢事。”

当你同意对方，对方的攻击就失去了着力点。这在人际交往中，尤其是在和异性聊天时非常有用。不要轻易认为对方在攻击或测试你，很多时候那只是对方习惯性的表达。

## 课程总结

本节课中我们一起学习了情绪管理的核心框架：
1.  **情绪是底层驱动力**：它决定我们的思想、行为和世界观。
2.  **先处理情绪，再处理问题**：在负面情绪中不要做决定。
3.  **练习观察者视角**：用第三人称描述情绪，抽离出来看问题。
4.  **理解痛苦是普遍的**：追求“圆满”而非“完美”，接纳所有情绪。
5.  **掌握痛苦应对步骤**：停止、承认、释放、臣服、求助。
6.  **学会情绪防身术**：不接招，不把别人的话语轻易解读为攻击。

记住，情绪管理的目标不是消灭负面情绪，而是**提升对情绪的觉察力，缩短受负面情绪影响的时间，并保持内在的稳定**。当你能管理好自己的情绪，你就能更好地管理自己的人生。
