# 1、老吴：老吴《聊天研究所》：14、办公室认识的同事怎么聊（中）_note

在本节课中，我们将继续分析一个与办公室同事聊天的实际案例。我们将拆解对话中的关键节点，指出常见的聊天误区，并学习如何更自然、有效地延伸话题与进行邀约。

---

上一节我们介绍了聊天节奏的重要性，并分析了案例的前半部分。本节中，我们来看看后续对话中，男生如何开启新话题以及其中暴露的问题。

男生开启了一个新话题，说：“看样子你也是00后。” 这是从女生的年龄切入。女生只能回答是或不是。女生回复：“是的”，并强调“刚好00”。

我们来分析男生的下一句话。他说：“小龍女梅，最近龍王發火天天下雨，原來是他的孩子走丟了。” 这是从00年属龙进行的联想和话题延展。

这个玩笑尚可，女生的反应也不错（回复了“哈哈哈哈”）。但如果是我，从年龄切入时，可以选择另一个角度。例如，可以说：“看来已经成年了，是不是？” 或者聊成年人与未成年人的不同。这样能展开更多对话，而不是跳到“龙王的女儿不见了”这种难以延续的话题。

男生又犯了之前的问题：**一个话题只聊两句就更换**。这样聊天很容易陷入僵局，因为聊着聊着就没有内容了。所以女生直接回复：“去看我的朋友圈嘛。” 因为男生多次从朋友圈找话题，且聊得很浅。

女生能明显感觉到：第一，你不太会聊天；第二，你过于关注她的朋友圈。从更深层看，这暴露了你对她有兴趣。因此，像“看樣子”、“發現”、“突然覺得”这类词不要常用，因为它们会暴露你的需求感。情商高的女生都能察觉，这会使主动权转移到女方手中。

男生此时说：“说明大多数时候比较自主。” 如果我是一个女生，我会很困惑：喜欢暖色系为什么等于自主？这之间没有逻辑关联。我们之前课程讲过，**每次说话围绕一个话题即可**。男生又补充了一句：“明天休假。” 因此女生直接忽略了他上一句话。这是很多男生会犯的错误。

以下是女生回复的分析：
*   **“是的，但是也没有任何安排。”**：当男生询问女生是否休假时，这通常是邀约的前奏。女生表示有空，这便是一个契机。
*   **“你和培训老师也会来跟学生交流聊天，有点意想不到。”**：这说明女生对男生有关注和好奇，这些都是兴趣指标的表现。好奇本身就是一种兴趣。

结果男生回复：“不下雨爬爬山還不錯，我對深圳還不是很熟悉。” 我觉得男生不够直接。既然对方已表现出兴趣，想约爬山就应该直接提出，而不是如此委婉。而且，他挑选的活动可能不对女生胃口，所以女生回复：“还是待在房间打游戏吧。”

我详细讲解一下男生这句话的问题。“我對深圳還不是很熟悉”这句话完全没必要说。女生是外来务工人员，你需要带领她，如果你也不熟悉，如何带领？不如约对方去吃美食或进行其他活动。你需要先了解女生是否爱运动、生活健康，她才可能喜欢爬山。

如果是我的话，在此处会这样回应：“刚好大家都有空，我们出去玩呗。” 这是一个**模糊邀约**，用于确认女生是否愿意约会。如果愿意，再共同商量双方都喜欢的项目。这是很多人在邀约时会犯的问题。

男生接着回复：“遊戲是誰？你居然要在房間打他，有點暴力。” 这个梗源自“睡觉是谁？”，但用在“打游戏”上并不合适。关键在于，此时聊天氛围平淡，且他没有回应女生的好奇（即“意想不到”那句话）。如果是我，我会反问：“难道我看样子有那么严肃吗？” 这样可以更好地互动。

男生还加了一个脸红表情，自以为幽默。但如我所言，这个梗很烂，女生不知如何接话，因此没有回复。很多男生此时会想：“女生是不是对我没兴趣？”，却从不思考自己的聊天方式是否有问题。

---

时隔五天，男生重启话题，称呼对方：“小朋友”。这是一种较亲密的称呼，女生没有反感，回了一个挖鼻孔的表情，且回复速度很快。

男生接着说：“看樣子還躺在沙發上吃雞。” 这运用了一些过去教的小套路。但在我看来，这些套路已有些过时。因为这相当于赌博，猜中了还好，猜错了就难以接话。如何增加成功率呢？你可以换成：“这回应该还躺在沙发上玩手机吧。” 这样命中的几率肯定比“吃鸡”更高。

女生的回应是：“没有，不玩吃鸡的。” 男生直接愣住，不知怎么聊。这是典型的“想找话题却选错话题，导致聊天崩盘”的情况。所以男生又开始重启话题。我们发现他总是在犯同样的错误：找错话题，或没有很好地延伸话题。

女生的回应是：“哪里有做饭的地方？” 她在反问男生。在我看来，男生说话的逻辑有问题。站在女生角度，我会疑惑：“为什么在外面住就挺适合做饭，在家里住也可以做饭？这有什么区别？” 女生关心的是具体地点。

男生的回应是：“我厨房以后借你一个小时十块钱。” 这个回复有问题。第一，女生并未说要借你的厨房；第二，你还要收费。这是**双重的筛选**，很有问题。如果是我，我会说：“告诉了你之后你要做饭吗？”

这句话引起了女生的小反感，她直接发了两个“拜拜”的表情，并回复：“幸好我不需要，因为我不会做饭。” 此时，男生忘记了女生之前说过的话。女生曾自称是“潮汕的女孩子”，并说“温柔又贤惠”。现在却说不会做饭，这相当于打自己的脸。

如果是我，我会幽默地说：“之前不知道某人说过自己很温柔很贤惠的，现在饭不会做。” 用幽默的方式调侃，而非打压对方，切记这两者不同。
