# 浪迹9.0 第九课：阿坤07 - 最吸引女生的技巧组合 🎯

在本节课中，我们将要学习在建立吸引力之后，如何通过“赋予可得性”和“进行筛选”这两个关键步骤，最终促成与心仪女生的关系。这是将前期吸引力转化为实际成果的核心技巧。

通过前面几个小节的学习，你已经对如何增强自身吸引力有了比较全面的了解。假设你已经建立起足够的吸引力，即在价值、能量、展示三大方面都做得很好。

那么，我们就能得到喜欢的女生了吗？我有很多国内外的学员，他们自身并不缺乏价值，外在条件也都挺好，从小到大可能都是身边的女生追他。但是，当他遇到自己真的很喜欢的女生时，女生常常对他说：“你人很好，很优秀，但是不会和你在一起。”

学员跑过来问我这是为什么。这种情况，除了吸引力的构建，还差一点就是“赋予可得性”。

举个例子，我们玩游戏。如果游戏难度设置得太简单，你玩两把就不会有什么兴趣。但是，如果游戏难度太大，每次你都没有办法通关，你也会丧失兴趣。所以游戏难度要设置得恰到好处。那么就算你失败了，你也还是想一次又一次地尝试，觉得很有挑战性，有趣味性。因为你知道花时间练习就可能通关，这就是一种“可得性”。

前面我们讲到过，很多男生在对女生没有构建起吸引力之前，就对女生展开攻势、死缠烂打，这是“跪舔”，会让女生对你敬而远之。但如果是对女生有了吸引力之后，同样的表露需求感，这才是正确的解锁姿势。

我们称之为“赋予可得性”，这会促进你和女生关系的达成。

再举一个“可得性”的例子。一个颜值、身材、气质都挺不错的女生，25岁了想谈恋爱。有两个男生同时在追求她：一个是高富帅，要什么有什么；另外一个是外形、经济都很普通的一个男生。

你说这个女生会选择谁？大部分的女生都会选择高富帅。

可是，如果再加上一点限制条件：高富帅比较爱玩，喜欢蹦迪、旅行、赛车等等，居无定所。而普普通通的那个男生，有一份初见起色的稳定工作，前不久刚刚贷款买了一个小户型和一辆普普通通的代步车，每周都会约女生出去吃饭看电影，还会经常回家探望爸妈。

你说这个女生更倾向谁？想要认真谈恋爱的女生，肯定会选择后者。因为后者更具有“可得性”。而高富帅适合玩一玩，体验一下刺激，但不适合认认真真地谈恋爱，太没有安全感了。

所以，光有强有力的吸引也不行，必须给予女生适当的可得性。

也有学员问我这样的问题：“坤哥，虽然我跟自己喜欢的女生约过会、牵过手，甚至还亲吻过了，但没多久，这个女生又对我说不适合在一起，这是怎么回事呢？”还有女孩子问过这样的问题：“你喜欢我什么？”这种问题我都不知道怎么回答。

那么在这里，你需要对女生进行“筛选”。

“筛选”的作用是告诉她，你为什么喜欢她，她哪些特质吸引你。你想和她亲密接触，是仅仅因为她长得漂亮、身材好，甚至想和她发生性关系吗？如果仅仅是因为你想和她发生关系，那女生是不能接受的。如果时间一长，失去了兴趣，有更漂亮的女生出现，你就很容易离开她。

所以，我们日常和女生接触的时候，要多留意女生的一些细节，注意到她有意无意展现出来的优秀的内在品质。也就是我们之前提到过的“内在价值”，不过这里变成了女版。

而且你注意到之后，还要表达出来。

以下是进行有效“筛选”时表达的具体例子：

*   “原来你还上了厅堂，下了厨房，做饭色香味俱全。”
*   “没想到你这么喜欢小孩，挺会照顾人的。”

要注意，越具体的事情越好。这才能让女生感受到，你是真的喜欢她这个人，在意她这个人，而不是只出于最原始的性冲动。

好了，最后我们总结一下。

本节课中，我们一起学习了在构建吸引力之后，促成关系的完整技巧组合：**对女生建立吸引力 → 赋予可得性 → 进行筛选**。记住这个公式：`吸引力 + 可得性 + 筛选 = 关系达成`。这才是最能撩到女生的核心技巧组合。
