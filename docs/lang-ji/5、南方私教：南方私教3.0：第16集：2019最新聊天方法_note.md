## 历史回顾：聊天方法的演变

上一节我们介绍了课程概述，本节中我们来看看聊天方法的历史演变。理解过去，才能明白为何现在需要新的策略。

2008年到2010年是“炮学”的元年。这个阶段主要接触了《谜男方法》。该方法的核心是使用**罐装材料**和**惯例**与女性互动。大部分学习者只学会了一些固定的对话模式。这是国外的舶来品。

到2012年至2014年，浪迹教育开发了一套聊天技巧。在此之前，几乎没有系统性的聊天技巧教学。这套技巧包括最初的**聊天六步法**或**聊天四步法**。

2015年是中国移动互联网4G时代普及的开端。4G网络使得传输图片和视频变得非常快速。这个时代，**展示面**变得至关重要。在这个阶段，**展示面占80%，聊天只占20%**。聊天的核心目的非常明确：**为了邀约**。

## 当前环境：为何旧方法失效？

上一节我们回顾了历史方法，本节中我们来看看当前环境发生了哪些变化，导致旧方法逐渐失效。

2017年末，“PUA”一词被污名化。这是因为浪迹教育的私教课程被盗版，价格极低。许多购买盗版课程的人，只学习了聊天技巧，却**不做形象改造，也没有展示面**。

以下是这些人的典型问题：
*   **形象糟糕**：不注重发型、衣着，身材管理差。
*   **展示面空洞**：使用卡通或风景头像，朋友圈充斥着无关转发，没有个人生活展示。
*   **行为鲁莽**：搭讪时话语生硬，聊天时急于使用惯例和邀约。

当大量形象不佳的人使用同一套话术去搭讪、聊天时，女性会产生警觉。她们发现很多男人的套路一模一样，再结合2017年关于PUA的负面新闻，便会将这种行为归为“PUA套路”，从而产生反感。因此，**2019年，大家都在用的方法就没用了**。

## 核心理念：重塑聊天目的与个人建设

上一节我们分析了旧方法失效的原因，本节中我们将深入探讨新方法的核心理念。

*   **邓**：指像邓通一样富有。如果前两者不足，需要通过展示面营造高价值生活。
*   **小**：指心思细腻，能体贴、逗女生开心。
*   **闲**：指有时间陪伴。

大部分兄弟可能只占了一个“闲”。如果不具备前几项，又不愿投入建设，只想通过聊天约出妹子，这是行不通的。你的**个人形象价值百万**。

对于A类女生，采用**阶段性、提供价值的群发**方式进行互动。目的是让她们在不知不觉中了解你的生活，感受到你的不同。
