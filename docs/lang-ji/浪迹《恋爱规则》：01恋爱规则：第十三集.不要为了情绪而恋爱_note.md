# 浪迹《恋爱规则》：01恋爱规则：第十三集.不要为了情绪而恋爱_note

在本节课中，我们将要学习一条关键的恋爱规则：追求女生应基于真正的喜欢，而非为了证明自己或赌一口气。我们将探讨这两种动机的本质区别及其可能带来的不同结果。

上一节我们介绍了恋爱中的基本心态，本节中我们来看看一个具体的动机问题。

## 核心规则阐述

你可以不顾一切地追求一个女生，也可以全心全意地去追求一个女生。但前提是，你真正地喜欢这个女生，而不是为了“义气”或证明自己去追她。

## 为何“证明自己”在恋爱中行不通

在大多数事情上，我们懂得“知难而退”。明知不可为，便不会去做。但很多人会为了证明自己或赌一口气而行动。

以下是生活中其他领域“证明自己”可能成功的例子：

*   **目标恒定**：例如，篮筐高度是固定的，清华北大的分数线每年也相对固定。你可以通过设定明确目标并努力达成。
    *   `目标 = 灌篮` 或 `目标 = 达到录取分数线`
*   **路径清晰**：努力的方向和最终评判标准是明确的、可衡量的。

然而，在追求女生这件事上，这套逻辑并不通用。

## 恋爱目标的不可控性

追女生之所以不同，核心在于目标对象是**会变化的**。她今天这样，明天可能那样；今年的想法，明年可能改变。你不能为了一个不确定的结果去努力。

`女生(状态) = f(时间, 经历, 环境, ...)` // 女生的状态是一个随时间、经历、环境等因素变化的函数

你努力的方向可能基于她当前的状态，但她下一刻可能就改变了。因此，千万不要为了证明自己、为了达成某个“努力奋斗就应有结果”的执念而去追求女生。

## 为何要“为喜欢而追”

我们建议基于真正的喜欢去追求女生，原因在于过程体验。

如果你真的喜欢一个女孩子，那么你在追求和努力的过程中，**你自己是快乐的**。你在做一件让你本身感到愉悦的事情。

类比而言，我们玩游戏是因为**玩游戏的过程本身很快乐**，输赢反而次要。追女生也应如此：追求自己喜欢的女生，让你在追逐过程中感到快乐和开心。

那么，最终能否追到她，其实都无所谓了。这才是追求女生的真谛。

## 错误动机的潜在危害

反之，如果你是为了义气或证明自己，你的所有目的都聚焦于“结果”。

*   **过程痛苦**：在追求过程中，你可能并不开心，甚至会感到难过。
*   **结果失落**：当你达不到预期结果时，你会更加难过。
*   **本质是赌博**：当结果本身无法确定时，你的行为就变成了一场赌博。参与一场本身让你不开心、结果又无法把握的赌博，是最不明智的选择。

你想要的并非与这个女生在一起，而是“证明自己”。这反而可能伤害到对方。

## 总结与建议

本节课中我们一起学习了追求女生的核心动机差异。

浪哥认为，证明自己或意气用事的机会有很多，**不要用在追女生上**。

**核心建议公式**：
```
追求动机 = 真正的喜欢 // 推荐
追求动机 = 证明自己 || 赌一口气 // 不推荐
```

如果在追求一个女生的过程中，你没有感到快乐、幸福或开心，那么浪哥建议你不要继续。否则，你可能最终“竹篮打水一场空”，既没有享受过程，也得不到想要的结果。

**最终判断标准**：你的追求行为本身，是否让你感到持续的愉悦？
