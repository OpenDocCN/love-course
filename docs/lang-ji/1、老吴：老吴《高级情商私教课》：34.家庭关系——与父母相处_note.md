# 1、老吴：老吴《高级情商私教课》：34.家庭关系——与父母相处_note

在本节课中，我们将学习如何与父母建立和谐、滋养的家庭关系。课程将涵盖家庭的核心意义、沟通技巧、处理矛盾的方法以及如何提升父母的自我价值感。掌握这些核心原则，能帮助你构建更幸福的家庭生活。

## 家和万事兴：家庭的根基

上一节我们介绍了课程主题，本节中我们来看看家庭关系的核心基础。

中国有一句古训：家和万事兴。这句话看似简单，但非常有效。若想在人生各个领域获得成功，拥有一个和气、幸福、美满的家庭是基础、根源和最重要的前提。

家是什么？家是让我们感受到最轻松、最舒服、最自在、充满喜悦和爱的地方。因此，家不是讲道理的地方，而是讲爱的地方。

家人指的是我们的父母、伴侣、孩子，即生命中最重要的亲密关系。家人是可以在他面前完全真实的人。家人是尽管有争吵、有矛盾，也可以真正原谅彼此的人，是无论如何也分不开、一辈子都脱不了关系的人。

决定我们每个人喜怒哀乐和生活品质的，正是我们与家人的关系。许多人认为，越是亲近的人，越可以不在乎，越可以肆无忌惮地伤害彼此。在佛教中，这类关系被称为“冤亲债主”，即来到身边甩不掉、分不开的关系。

以下是四种主要的“冤亲债主”关系类型：

1.  **冤家**：例如夫妻一见面就互相看不顺眼，每天吵架，但又不愿分开。
2.  **亲人**：指两人一见如故，莫名就想对对方好，例如某些婆媳关系。
3.  **还债的**：例如某些父母为孩子掏心掏肺，付出一切。
4.  **讨债的**：例如某些子女成年后仍需父母供养。

无论哪种关系，我们都能通过觉察和落地实践，将亲密关系改善到最佳状态。最佳状态就是和气的状态，因为和**气生财**。和气就是开心、幸福的状态，这是我们每个人都想要的终极目标。

## 沟通的艺术：从语言开始滋养关系

理解了家庭的基础后，沟通是维系关系的桥梁。本节中我们来看看如何与家人进行有效沟通。

我们每天都要和家人说很多话。许多人认为既然是一家人，关系好，就不用说好听话，专挑缺点和毛病说，哪句难听说哪句。这是非常愚蠢和错误的沟通方式。

即便是家人，在语言沟通上也要注意方式方法。越是亲密的家人，对方就越在乎我们说的话。因此，我们要多说感恩、赞美、鼓励和肯定的话。要知道，亲人在一起是为了彼此相爱，而不是互相伤害。

### 描述性赞扬：让表扬更走心

与父母相处时，学会“描述性赞扬”非常重要，这与常见的“评价性赞扬”不同。

评价性赞扬是直接给出价值判断，例如：“妈，你好厉害啊，连这个都会玩！”这种表扬用在父母身上，可能让他们觉得敷衍，仿佛在说“没想到你这么老了还会用这个”。

描述性赞扬则更具体、更有诚意。例如，看到妈妈在用新APP，可以说：“老妈，你怎么想到装这个的？是自己一个人搞定的吗？真跟得上潮流，以后看到有趣的东西记得推荐给我哦。”

描述性赞扬包含以下三个层次：

1.  **表扬努力，而非结果**：赞扬对方的学习能力和与时俱进，例如“一个人搞定了这件事”。
2.  **说出自己的感受**：表达对方行为带给你的正面感受，例如“你这么做让我觉得我妈很潮，我很开心”。
3.  **描述行为的影响**：指出对方行为带来的积极价值，例如“以后记得推荐给我”，暗示妈妈能为家庭带来新价值。

这种方法不仅适用于与父母沟通，也可用于日常生活和工作。

**练习**：请思考，如果爸爸在家种菜，你该如何使用“描述性赞扬”来表扬他？（提示：避免简单夸“勤劳”）

## 处理分歧：化解与父母的“闷气”

沟通中难免有分歧。当与父母意见不合时，很多人会生闷气，觉得“你为什么不支持我”。本节我们来学习如何处理这种情绪。

生闷气在心理学上是一种典型的自我攻击行为，源于缺乏表达不满和愤怒的能力。因此，在进一步沟通前，需要先内部舒缓情绪。

除了之前课程讲的情绪管理方法，这里介绍一个“画小人”的技巧：

1.  拿出一张纸，画出你生气的对象（如父母）的样子。
2.  把你想说但说不出口的话全部写在小人周围。可以写“不讲道理”、“顽固”、“不懂我”等，任意发泄。
3.  最后，处理掉这张纸，可以撕碎、揉成团或烧掉。

这个方法听起来孩子气，但非常管用。它帮助你将怒气具象化并释放，为理性沟通创造条件。

## 正向表达：用语言塑造现实

许多家庭认为“一家人不说两家话”，专挑难听的说，还美其名曰“为你好”。这实际上会让对方感到受伤、不被认可，自信受损。

越是亲密的家人，越要多说“甜言蜜语”。感受到最重要的人的感恩、认可、信任和赞美，自己才会成为一个更自信的人。

家人之间尽量避免直接指责问题。如果你想改变或教育对方，不要说他哪里不好，而是用肯定的句式给出期望。

以下是问题转化示例：

*   **原句（指责）**：“你怎么总是生病？”
*   **转化后（肯定）**：“我觉得你的身体越来越好了，一天比一天健康。”
*   **原句（抱怨）**：“你总是没时间陪我和孩子。”
*   **转化后（感恩）**：“谢谢你百忙之中抽空陪伴我们，我真的很珍惜和你在一起的时光。”

你的言语在滋养对方的同时，也在滋养自己，并且具有强大的塑造现实的力量。**正信正念**，首先相信并说出对方的好，对方就会越来越好。

## 仪式感：为家庭生活注入活力

在相处中，还有一件重要的事叫“仪式感”。许多人认为家人都很熟悉了，不必搞复杂。但实际上，家庭仪式能为平淡的生活增添意义和乐趣。

以下是仪式感的例子：

*   共同庆祝生日、纪念日。
*   约定全家每年旅行一两次。
*   设定每周家庭日，一起吃饭并分享感恩、赞美的话。
*   每日出门前，伴侣间一个简单的拥抱和亲吻。

小小的仪式感对关系起着重要作用。它让家人感受到被重视、被爱，是维系感情热度的关键。越是亲近的人，越要讲究相处的模式和仪式感。

## 特殊情况：当父母自我价值感偏低

最后，我们来探讨一种特殊情况：如何与自我价值感偏低的父母相处。

有时我们给父母送礼物或表达关心，他们会拒绝或贬低礼物的价值。这背后可能不仅是节俭，还可能是父母觉得自己“不配”或“没有价值”了。

当子女的成就超过父母，或父母一贯习惯于牺牲奉献时，他们可能通过拒绝好意来维持心理平衡。他们可能害怕接受了物质关怀，就失去了子女的情感关注。

面对这样的父母，我们的目标是提升他们的自我价值感。

以下是两个具体建议：

1.  **珍视他们的精力与见解**：耐心倾听父母唠叨或发表意见，即使内容你已熟知。像他们当年倾听我们一样，表现出兴趣和重视。
2.  **明确他们的价值**：以巧妙的方式让父母感到他们的付出“有价值”。例如，父母帮你带孩子，你可以把省下的保姆费以“请帮我先存着”的名义给他们，而不是直接作为酬劳。

最重要的是，父母渴望的是**陪伴**，这超越了金钱。多花时间陪伴他们，是提升他们价值感和幸福感的最根本方式。

## 成为好父母：给予孩子最好的礼物

作为课程的延伸，我们简要探讨如何为将来成为父母做准备。能给予孩子最好的礼物有两样：一是让他感受到爱，二是正确的教育。

爱需要平衡，避免**溺爱**（无底线满足）和**冷漠**（不懂表达）。可以通过“声语意”来表达爱：多抚摸、拥抱孩子，营造充满欢笑的家庭氛围，而非讲道理、吵架的地方。

同时，要让孩子尽早明白，他要为自己的生命和情绪负起100%的责任。父母不是神，我们尽己所能去爱，但更重要的是让孩子早日“**开窍**”，意识到自己是命运的主人，学会独立和成长。

最重要的教育是**言传身教**。孩子是父母的镜子。你希望孩子成为什么样的人，自己首先要做到。自己婚姻幸福，才有资格期待孩子婚姻美满；自己情绪稳定，才能培养出情商高的孩子。

本节课中，我们一起学习了构建和谐家庭关系的核心智慧：

2.  掌握了“描述性赞扬”等高效沟通技巧，学会用语言滋养家人。
3.  学习了用“画小人”等方法处理与父母的分歧和闷气。
4.  认识到将指责转化为肯定表达的重要性，用语言塑造积极现实。
5.  明白了“仪式感”对维系家庭温度的关键作用。
6.  探讨了如何提升自我价值感偏低的父母的幸福感，核心在于陪伴与肯定。
7.  简要了解了成为好父母的原则：给予有界限的爱，并以身作则。

记住，**家是讲爱的地方，不是讲道理的地方**。爱就是在乎对方的感受，满足对方的需要，让对方开心。这需要我们持续学习和实践。从今天起，多对家人说感恩、赞美的话，多为家庭创造温馨的仪式感，你将收获更幸福的家庭生活。
