# 浪迹《恋爱规则》课程02：聊天规则 - 第五集：聊天的目的在于增进了解 🗣️💡

在本节课中，我们将学习现代社交聊天的一个核心理念：聊天的首要目的是增进彼此了解，并在此基础上寻找合适时机进行邀约。我们将剖析一个常见的误区，并介绍一个结构化的聊天模型。

上一节我们探讨了聊天的基础，本节中我们来看看聊天的核心目的。

很多学员听过早期的课程，其中提到“聊天的目的就是邀约”。因此，部分学员在聊天时，仅仅交流一两句话，就急于提出邀约，例如：“今天天气不错，出来溜达溜达”。

然而，时代已经发展到2020年以后。如果继续使用2012年的聊天方式，就如同在智能手机普及的时代仍在使用诺基亚功能机。虽然诺基亚也能打电话，但时代已经改变。

那么，现在聊天的目的是否是邀约？答案可以是，也可以不是。我们认为，现阶段聊天最主要的目的，是**了解对方**。

通过了解对方，掌握其兴趣爱好，再找准契机发出邀约，这才是我们现阶段的正确聊天目的。我们发现，许多学员因早期课程的影响，陷入了“为了邀约而邀约”的误区，这完全背离了课程的本意。

因此，我们在此重新构建一个清晰的聊天模型，其流程如下：

以下是结构化聊天模型的四个阶段：

1.  **开场**：建立初步联系。
2.  **相互了解**：交换基本信息与兴趣爱好。
3.  **互相有兴趣**：培养共同话题与好感。
4.  **找准契机邀约**：在共同兴趣的基础上，于恰当时机提出邀约。

所以，我们聊天首先要解决“相互了解”这个问题。如果双方缺乏了解，女生也会感到困惑：我对你一无所知，为什么要答应你的邀约？

正确的顺序是：先相互了解，然后培养**共同的兴趣**。当双方拥有共同兴趣和爱好后，再寻找合适时机。例如，当发现彼此都想做某件事，且时机地点恰好合适时，邀约便水到渠成。

如果没有经历相互了解和培养兴趣的阶段就直接邀约，就如同建造楼房时不建一、二、三楼，而试图直接建造第五层，这是不稳固且难以成功的。

本节课中，我们一起学习了现代社交聊天的核心目的——增进了解。我们纠正了“聊天即邀约”的过时观念，并介绍了一个从开场、相互了解、培养兴趣到最终邀约的四阶段模型。记住，扎实的了解是成功邀约的基石。
