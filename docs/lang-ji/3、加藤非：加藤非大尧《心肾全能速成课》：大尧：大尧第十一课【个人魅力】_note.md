## 自身建设的核心细节 🔍

上一节我们介绍了课程概览，本节中我们来看看自身建设的具体细节。首先，大家可以询问任何关于我自身建设的问题，无论是我吃穿用住的东西，还是其他小细节。我会告诉大家如何用便宜划算的方式，展示出具有一定格调的自我。

现在，我来分享我的朋友圈，从最早的一条开始。

**第一条朋友圈（2015年3月4日）**
这是我的第一条朋友圈，后来被无数学员模仿。这是一张对镜自拍。当时的环境比较杂乱，后面是一块黑板。我里面穿的是优衣库的白衬衫，外面是一件在Zara买的400块钱的夹克。

这张照片的拍摄方法是：我对着镜子，一位叫BB的摄影师朋友站在我脖子后方位置进行拍摄。核心要点是：因为是远景，头顶可以离照片边缘稍近一些；服装要经过打扮；表情要管理好。这些基本规范我一直沿用至今。

**第二条朋友圈（2015年3月17日）**
这是在成都一家很有格调的“方所”图书馆拍的。一共四张图：一张全景、一本魔术书（我的专业）、一本名为《西方艺术工作者》的书，以及我的一个创意。

我的配文是：“这世界上并不缺少丑，只是缺少神丑的眼睛。” 当时我和学员老童一起，互相拍了照。我里面穿黑色T恤，外面还是那件Zara夹克。这组展示面同样没有花费一分钱。

**第三条朋友圈（2015年4月10日）**
这是一张我父亲年轻时的照片，用来展示家庭。我父亲身高一米八七，在那个年代非常帅。这张照片也是零成本。

**第四条朋友圈（时间不详）**
这是一张用了当时美图秀秀热门“花棉袄”马赛克特效的照片。这是在KTV里一个妹子用美图手机拍的。我穿的是优衣库149元的白衬衫。这展示了“与时俱进”，能让人感觉你了解最新潮流。

**第五条朋友圈（工作相关）**
这是在成都太古里拍摄搭讪视频的工作照，使用了公司的摄像机，对我来说也是零成本。

**第六条朋友圈（2015年7月）**
这是在上海出差时打车的经历。我们叫了普通Uber，但平台派来了一辆奔驰。我拍下了方向盘，配文说司机很像某个选秀歌手。这暗示我习惯了乘坐高端车辆，但实际上只花了普通车的钱。

**第七条朋友圈（2015年7月30日）**
这是在成都丽思卡尔顿酒店吃自助餐。关键点是：吃自助餐要拍海鲜。当时我和一位年长的大哥一起。这属于“高品质吃喝玩乐”的展示。

**进阶与尝试**
在2015年6月，我发布了一张吃简餐的照片，没有使用任何滤镜。到了7月，我首次尝试给照片加滤镜，虽然现在看有些过时和做作，但那是重要的开始。

**其他低成本展示面示例**
*   **高尔夫球**：花了200元进入场地，租了衣服，拍了几张照片，显得很专业。
*   **酒店泳池**：和上述自助餐是同一天，在丽思卡尔顿酒店游泳并拍照，展示体重变化（半年瘦了22斤）。
*   **创意灯泡**：和学员老童吃烤鱼时，利用一杯造型像灯泡的西瓜汁拍照。这条朋友圈帮我吸引了四个女生。
*   **与家人用餐**：在成都IFS一家汉堡店，一顿两三百元的饭，看起来却很奢华。
*   **艺术角落**：在一家有趣的餐厅，给一个别致的小角落拍特写。
*   **二次对镜自拍**（2015年12月）：在服装店“太平鸟”的镜子前拍摄，标志着我形象的成熟。拍摄要领是：脸要贴近镜子，身体侧对镜头，摄影师从脖子后方拍摄。

## 展示面的核心思路与原则 🧠

上一节我们回顾了许多具体的展示面案例，本节中我们来总结其中的核心思路与原则。

以下是打造有效展示面的关键点：

1.  **行动力至上**：最基础的是行动力。我告诉你方法，你去执行。如果一开始就说“因为XX原因做不了”，那就真的做不成了。
2.  **性价比优先**：我的大多数展示面成本极低，甚至是零成本。关键在于创意和发现美的眼睛，而不是昂贵的消费。
3.  **展示成长与变化**：我的朋友圈记录了我从无到有、从稚嫩到成熟的整个过程。这种真实的成长轨迹本身就有吸引力。
4.  **构建社交圈**：早期我的朋友圈只有自己，这是大问题。后来我开始放入和朋友、同事的合影，展示丰富的社交生活。没有社交圈就主动去构建、去拍摄。
5.  **配文精炼**：朋友圈配文的原则是：不装逼、不做作、不写大段心灵鸡汤。要短小精悍，文字比图片次要。把照片拍好才是关键。
6.  **针对长期关系**：我的展示面风格更倾向于长期关系，显得踏实、有生活气息。如果是短期关系，会有更多夜场等飘忽的内容。
7.  **持续更新**：我没有长时间断更朋友圈。持续更新展示了生活的连贯性和活力。

## 关于个人成长的思考 💪

本节课中我们一起学习了如何通过展示面构建魅力。最后，我想分享一些关于个人成长的思考。

我来自一个普通家庭，在澳洲留学时也打过工，做过咖啡师、铺过地板，手被烫伤、拍肿。我不是拿着家里的钱在国外享受。一步步走到今天，明天是我加入浪迹教育整整两年，也将去提车。

如果你总是给自己找借口，说因为这个做不了、那个做不了，那么两年后你可能还是老样子。但如果你知道自己该做什么，并且去行动，两年后你可以看看自己变成了什么模样。

我们团队也在不断吸纳有不同经历和特长的人（如海归、演员、媒体人），因为每个人都能带来独特的视野和价值。把妹也是一样，你不一定比对方优秀，但你能提供她所没有的东西——可能是见识、情感体验或其他价值。

**精英课**的核心，就是教你如何成为一个更精致、更优秀的男人。这不仅仅是把妹，更是全方位的自我提升。
