# 浪迹老佟《短期关系》课程：第三课：准备工作 🛠️

在本节课中，我们将要学习正式邀约前必须完成的准备工作。这些准备是成功建立短期关系的基础，主要包括展示面的设置、聊天软件的选择以及最佳邀约时间段的规划。

## 展示面的设置 📸

上一节我们介绍了短期关系的核心理念，本节中我们来看看如何通过展示面进行有效展示。展示面是吸引异性的基础，无论长期或短期关系，展示面至少需要达到60分。在短期关系中，展示面的设计有特定方向。

以下是展示面设置的四个核心方向：

1.  **正确使用预选**：预选是最强有力的吸引，其定义是**与两个以上女性的合影**。错误的预选是**与单个女性的单独合影**，这会让其他女性误以为你有女朋友或暧昧对象。预选最适合用于短期关系，有利于邀约。

2.  **植入夜场元素**：最理想的预选拍摄地点是夜店。可以拍摄在夜店、酒吧或KTV的自拍，或与女性的合影。拍摄时需确保女性配合，动作亲密，距离贴近。夜场是充满性暗示的场所，植入此类展示面符合短期关系的语境。

3.  **展示身材优势**：如果你有不错的身材，应通过照片展示出来。例如，从高角度拍摄侧脸、胸肌、腹肌和人鱼线。如果身材偏瘦但有些线条，可通过修图、滤镜和拍摄角度优化。如果身材偏胖，则不建议脱衣拍摄，可穿紧身衣物显得结实强壮。多数女性更喜欢有线条的精瘦身材，而非过度健壮的大块头。

4.  **注重穿着与配饰**：强调通过饰品进行点缀。例如，夏天佩戴墨镜会显得更酷。首饰、手表、项链、手链等配饰，或有纹身，都会更吸引寻求短期关系的女性。这些点缀能让你看起来更酷，使对方更愿意赴约。

关于具体的拍摄和修图技巧，本课程未详细概述。如需深入学习，可参考田正远的装逼课。田正远是浪迹团队多位导师的展示面设计师。

## 聊天软件的选择 💬

讲完了展示面的拍摄方向，接下来我们看看通过哪些软件来接触目标女性。

以下是用于接触女性的三款主要软件及其策略：

*   **探探**：当前主流软件。优势在于**彼此喜欢后才能开始聊天**，这使得后续接触更简单。可以在软件上简单沟通，也可以更直接地打招呼后尝试添加微信。常用话术如：“探探不常看，微信加一个”或“我不怎么看这个软件，我加你微信吧”。目的是将资源转移到微信。

*   **默默**：上市已久的软件。现在每人每天只能打20个招呼。默默上的女性因常年被众多男性搭讪而可能显得比较“傲娇”，初始反应可能不佳。因此，**不能一上来就要微信**，需要在软件上多互动几句，建立初步联系后再尝试转到微信。

*   **世纪佳缘**：这款软件主打严肃交友，在短期关系中成功率极低，**应首先排除**。

**为何要将资源统一转移到微信？**
可以将探探和默默视为小鱼塘，微信则是**一个大的、生活环境更好的鱼塘**。将所有资源收集到微信后，便于集中管理和进一步互动，无需再分散精力维护多个平台。

## 最佳邀约时间段 ⏰

第三部分要讲的是邀约的时间规划。我将最佳时间段分为两类。

以下是两个关键的邀约时间段及其特点：

1.  **高风险高回报时段（凌晨0点至3点）**：在这个时间段成功邀约女性出来，对方的“窗口”（即接受短期关系的意愿）非常大。但相应的，成功邀约出来的概率相对较小。这个时间段邀约成功后，**无需再进行吃饭、喝酒等过场**，大环境已暗示了约会性质，可直接前往私密空间。

2.  **常规操作时段（晚上9点至12点）**：这是更常用的时间段。通常会先进行铺垫，例如**一起吃顿饭或去小酒吧喝一杯**。在建立一定舒适感和联系后，再尝试转场到私密空间。

关于如何在凌晨时段进行操作，以及具体的邀约技巧，将在下一节“入门级邀约”课程中详细讲解。

本节课中我们一起学习了短期关系邀约前的三大准备工作：**针对性设置展示面**（预选、夜场、身材、配饰）、**有效利用聊天软件**（探探、默默）并将资源导流至微信，以及**规划两个最佳邀约时间段**（凌晨高风险时段和晚间常规时段）。做好这些准备，将为后续的实际操作打下坚实基础。
