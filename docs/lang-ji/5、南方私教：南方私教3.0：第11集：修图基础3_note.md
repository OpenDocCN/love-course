# 5、南方私教：南方私教3.0：第11集：修图基础3_note

在本节课中，我们将深入学习Snapseed软件中两个强大的局部调整工具：**画笔**和**局部**。通过它们，你可以对照片的特定区域进行精细的色彩与明暗调整，这是让照片脱颖而出的关键技巧。

上一节我们介绍了Snapseed的基础调整功能，如亮度、对比度和饱和度。本节中，我们来看看如何对照片的局部进行更精准的控制。

---

## 一、课程准备与回顾

首先，请确保你已打开Snapseed软件，并准备好一张用于练习的照片。

上一节课我们讲解了“调整图片”工具中的各项功能，包括：
*   **明暗调整**：亮度、对比度、高光、阴影。
*   **色彩调整**：饱和度、氛围、暖色调。

这些是全局调整，会影响整张照片。今天，我们将学习如何针对照片的特定部分进行修改。

---

## 二、画笔工具详解 ✏️

画笔工具允许你像使用真实的画笔一样，手动涂抹照片的任意区域，并对涂抹区域应用特定效果。

以下是画笔工具的核心功能列表：

1.  **加光减光**：对涂抹区域的明暗程度进行增加或衰减。公式可理解为：`区域亮度 = 原始亮度 ± 调整值`。
2.  **曝光**：比加光减光效果更强烈，对涂抹区域进行无上限/下限的提亮或压暗。
3.  **色温**：调整涂抹区域的色彩冷暖倾向。
4.  **饱和度**：增加或降低涂抹区域色彩的鲜艳程度。

**如何使用画笔工具：**
1.  点击下方 **工具** > 选择 **画笔**。
2.  在底部选择想要的效果（如饱和度）。
3.  用手指在屏幕上涂抹需要调整的区域。涂抹时，可以点击右下角的“眼睛”图标（👁️）来显示/隐藏涂抹的红色蒙版区域，以便精确定位。
4.  如果涂抹过度，可以点击底部效果图标（如将“饱和度+10”向下滑动），切换到“橡皮擦”模式，擦除不需要的效果。
5.  调整完成后，点击右下角“√”确认。

> **核心技巧**：使用“眼睛”图标辅助查看涂抹范围；使用“橡皮擦”修正错误；调整效果强度（如将饱和度从+10改为+5）以获得更自然的效果。

## 三、局部工具详解 🎯

局部工具通过添加“控制点”来智能识别和调整照片中的某个区域，非常适合处理具有明确边界的物体（如人脸、衣服）。

**局部工具的核心操作：**

1.  **添加与控制点**：
    *   点击 **工具** > 选择 **局部**。
    *   点击屏幕下方的“+”号，然后在照片上需要调整的位置点击，即可添加一个控制点。
    *   上下滑动屏幕，可以调整该控制点的 **亮度、对比度、饱和度、结构** 四个参数。

2.  **控制范围（精准微调）**：
    *   用两根手指在屏幕上做捏合手势，可以放大或缩小该控制点的影响范围。红色高亮区域即受影响范围。

3.  **复制与删除参数**：
    *   点击一个已设置好的控制点，选择“复制”。
    *   添加一个新的控制点，移动到目标位置，点击该点并选择“粘贴”，即可将上一个控制点的所有参数应用过来。这非常适合统一调整多个相似区域（如双臂的肤色）。
    *   长按控制点可将其删除。

> **重要提示**：每张照片一次最多可添加8个局部控制点。如果不够用，可以先完成当前调整并保存，然后再次使用“局部”工具添加新的控制点。

## 四、实战应用与总结

本节课我们一起学习了Snapseed中两个至关重要的局部调整工具：**画笔**和**局部**。

*   **画笔** 适用于需要手动精细涂抹、边界不规则的区域调整。
*   **局部** 适用于对照片中具有明显色彩或明反差别的物体进行智能、快速的区域调整。

两者的结合使用，可以让你从“整体调色”进阶到“精修细节”，显著提升照片的专业感。记住，修图的关键在于**多练习、多尝试**。不要害怕调整参数，利用好对比功能（右上角对比图标）和撤销功能，大胆探索每个工具的效果。
