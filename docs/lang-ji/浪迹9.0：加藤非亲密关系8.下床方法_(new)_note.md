# 浪迹9.0课程：加藤非亲密关系（八）—— 下床方法 🛌➡️🚪

在本节课中，我们将要学习如何在发生亲密关系后，若发现对方并非理想伴侣，能够以相对平和、伤害最低的方式结束这段短期关系，即所谓的“下床方法”。

---

上一节我们探讨了亲密关系的建立，本节中我们来看看关系如何妥善地结束。有些女生虽然与我们发生了亲密关系，但在后续相处中，我们发现她并非理想的长期伴侣。

一个重要课题随之出现：当女生喜欢你，而你却不愿继续发展时，分手伤害在所难免。如何才能将伤害降到最低，实现“好心分手”？

在正片开始前，需要说明一点：如果在亲密关系中，因自身“气量”或时间问题导致体验不佳，这本身就是一个简单直接的“下床”契机。现在，我们回归正题。

## 一、前期短期属性的建立 🏷️

核心在于，在发生关系前就设定好预期。以下是具体做法：

我们的兄弟在与女生发生亲密关系之前，就应传递出自己是一个**放荡不羁爱自由**的人，不会为某个女人停留，同时身边也不乏其他追求者。

这样，当关系发生后，女生心中会有一个预期：你无法被长期持有。这使得后续的“下床”会简单很多。

## 二、事后关系定位 🗣️

发生亲密关系后的两小时是黄金沟通时间。以下是操作步骤：

我们的兄弟可以在此期间，与女生进行一次深入交流，互相交换彼此的感情观、价值观和生活观。

如果发现三观不合，兄弟可以向女生展示自己的**短期属性**，表明自己还不想为一段感情安定下来。同时，需要进行事后安抚，告诉对方：你只在乎当下在一起的感觉，而非着眼于未来。

在此之后，疏于联系。女生会逐渐明白，你们之间只是一个短期选择，从而减少纠缠。

## 三、以“工作忙”为理由 💼

这是最常用且不易被反驳的理由。以下是实施方法：

例如，我们的兄弟在朋友圈应始终塑造以工作为重的形象，经常展示出差动态。

当女生来电或发消息时，总是匆匆结束对话，避免更多见面接触。让时间冲淡一切，从而达到“下床”的目的。

以上三条是非常通用、常见且适合新手兄弟使用的下床方法。

---

2.  **向女生借钱**：以事业困难为由向女生借钱，例如：“亲爱的，我最近遇到点困难，你能借我五千块钱吗？你不借我，是不爱我吗？”通过制造压力促使对方离开。
