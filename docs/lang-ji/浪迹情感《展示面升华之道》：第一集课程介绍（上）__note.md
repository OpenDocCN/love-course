## 第一部分：学什么？🧐

首先，我们来看课程将学习哪些内容。请参考课程表，本课程共分为四个板块：
1.  展示面的理论认知
2.  展示面的拍照技巧
3.  修图
4.  个人IP经营

这是一个非常完整的流程，从理解展示面的核心理念，到如何拍摄、修图，再到如何将成果呈现在朋友圈或社交软件上，逻辑链条清晰。

接下来，我们详细看看每个板块包含的具体课程。

### 板块一：展示面理论认知

以下是该板块的核心内容：

首先，我们会介绍展示面的重要性，即为什么要拍摄展示面。

接着是“心理建设”课，讲解拍摄时应保持何种心态，以帮助你在拍摄时更加放松并拥有清晰的目标感。

之后，我们将用本板块的大部分课时，重点讲解展示面理论中最重要的 **“三要素认知”**。一张优质的展示面必须在以下三个方面表现出色：

1.  **环境要素**：照片中需要有出色的环境背景表现。
2.  **形象要素**：照片中需要呈现帅气的你。
3.  **价值要素**：分为审美价值和物质价值。

关于每个要素，我们都会用一整节课的时间，结合大量案例，帮助大家透彻理解其重要性及呈现方法。

这是理论认知板块的主要内容。最后会有一个板块复习，然后进入第二部分。

### 板块二：展示面拍照技巧

上一节我们介绍了展示面的理论基础，本节中我们来看看如何实际拍摄。

以下是该板块的核心内容：

首先，我们会讲解一些基础且核心的拍照技巧。

其次，也是本板块的重点，即 **“如何把人拍帅”**。这包括如何摆造型，甚至细致到手部动作，我都会一一教授。

这部分包含两大分支：**摆拍**与**接拍**（或称抓拍）。这是展示面拍摄的两大流派。

*   **摆拍**：常被称为“高端展示面”，旨在塑造如贵公子般的高价值形象。
*   **接拍（生活化拍摄）**：我认为这是更重要的分支，专注于如何在日常生活中将你拍得帅气。我会侧重讲解这部分内容。

此外，我们还会提供特定场景的拍摄指导，例如在酒店、自然景色或街道上如何拍摄。

最后，我们会教授如何拍出令人认可的自拍照。

### 板块三：修图

掌握了拍摄技巧后，我们进入第三个板块：修图。一张好的展示面经过恰当修饰，效果能更上一层楼。

以下是修图的主要步骤和内容：

我会讲解修图的正确顺序。

接着，重点教授 **“如何将人修得更帅”**，包括需要注意的修饰技巧。

然后是图片的颜色处理，主要包括**调色**和**滤镜选择**。

最后，还会介绍其他修图小技巧，例如如何**截图**能让照片效果更好，以及一些修图工具的非传统用法，以达到美化照片的效果。

最后一个板块是个人IP经营，这个名字听起来有些抽象，我们可以通俗地理解为：如何经营你的朋友圈以及在社交软件上发布照片和动态。

这些内容决定了你在网络上给陌生人留下的第一印象，因此非常重要。

在本板块中，我们将探讨展示面应如何建设，以及个人社交名片应如何打造。

从自己拍出一张好的展示面，到以最佳状态将其呈现在网络上，让异性及所有网友对你刮目相看，这就是本课程完整的逻辑链。
