# 1、老吴：老吴《高级情商私教课》：05.逻辑思考（三）_note

在本节课中，我们将要学习如何提高逻辑思维的效率。仅仅拥有逻辑形式并不足以确保我们总能做出正确判断。为了高效且准确地找到答案，我们需要掌握一些关键方法。

上一节我们介绍了如何启动和引导逻辑思维。本节中，我们来看看如何让这个过程更高效。

## 一、逻辑高效的前提：充分的信息

逻辑思维要得出准确答案，有一个至关重要的前提：我们脑海中的信息必须足够丰富。

例如，计算一个长方形的面积。如果只知道长为10公分，而不知道宽，我们无法求出面积。公式 **面积 = 长 × 宽** 在此刻无法应用。只有当同时知道长（10公分）和宽（20公分）时，才能计算出面积是200平方公分。

这个道理在更复杂的场景中同样适用。比如去看医生，如果只告诉医生“我头疼发烧”，医生无法做出准确诊断。发烧可能由感冒、炎症或其他疾病引起。医生需要更多信息：是否关节疼痛？验血结果如何？只有收集了充分的信息（前提），才能找到正确的治疗方法。

**核心要点**：在试图分析或解决问题时，必须先将所有相关信息充分列举出来。信息不足，逻辑推导就可能出错。

## 二、提升效率的关键：掌握已验证的真理

如何让逻辑思维更加高效？一个直接有效的方法是：将已经被反复验证过的真理或公式，直接装入脑海。

例如，已知直角三角形的两条直角边分别为2和3，求斜边长。我们无需测量，立刻可以运用勾股定理 **a² + b² = c²** 计算出斜边c约为3.6。因为这个定理是公认的真理，我们直接调用即可。

在生活中也是如此。许多经过验证的规律，例如“吸引比追求更重要”，一旦成为我们的认知基础，遇到相关问题时就能迅速反应，无需重新推导。

**核心要点**：通过学习，将各领域已验证的知识、规律和公式内化。当问题出现时，这些“思维快捷方式”能让我们瞬间找到可靠答案，极大提升决策效率。

## 三、突破惯性：寻找“第三条路”

人们常常受惯性思维束缚，习惯于用过去成功的方法解决所有类似问题。但世界是变化的，完全相同的场景不会出现第二次。为确保不被过往经验限制，我们需要有能力在每次遇到问题时，找出新的、更多的可能性——即“第三条路”。

以下是寻找更多元答案的一个诀窍：

**类比思维**：观察当前问题，看看在其他完全不同的领域，是否有性质类似的情况，并将其解决方案借鉴过来。

例如，在团队管理上，传统思维可能强调严格监督和绩效至上，让员工像机器一样运转。但通过类比思维，可以找到新方法：
*   **目标共识法**：与员工协商确定清晰的目标和回报。例如，与视频编导约定：月薪6000元，需完成粉丝增长和脚本数量等具体目标。之后只需定期检查进度，无需时刻紧盯。这激发了员工的自主性。
*   **“懒蚂蚁效应”**：生物学发现，蚁群中总有一些“懒蚂蚁”四处闲逛。但当蚁巢遭遇危机时，正是这些熟悉不同路径的“懒蚂蚁”能带领蚁群找到新的生存方向。类比到公司管理，意味着团队需要多样性。允许一部分人进行看似“闲散”的探索、思考和创新，他们可能在关键时刻为团队找到新的出路。不要追求所有人都是一个模子。

**核心要点**：避免陷入非此即彼的单一思维。运用类比思维，跨界寻找灵感，能为任何问题创造出超越常规的、更优的“第三条路”。
