## 如何建设私密空间：五感营造

上一节我们了解了建设私密空间的好处，本节中我们将具体学习如何通过营造“五感”来建设私密空间。

*   **灯光选择**：选择暖光灯或黄光灯，避免白炽灯。注意照明光源和装饰光源的比例。照明光源用于基本照明，装饰光源如落地灯、地灯、壁灯等用于增添浪漫感觉。总结来说，亮的地方越多，浪漫程度就越小。就像在私人电影院，黑暗环境能营造适合调情的氛围。你还可以关灯点蜡烛，当女生问起，可以开玩笑回答“为了省电，攒钱娶老婆”。

听觉建设很简单，只需要一个音质还不错的音响，并准备一些适合私密空间浪漫情调的音乐，尽量多选择英文歌曲。平时可以多积累合适的歌曲，收藏很有情调的音乐。如果没有，可以私信获取推荐。
