# 老吴《高级情商私教课》第31课：情感关系——责任、牺牲、异地恋与出轨 🧠💔

在本节课中，我们将深入探讨情感关系中的几个核心议题：如何理解责任与承诺、区分付出与牺牲、应对异地恋的挑战，以及面对伴侣出轨时的处理思路。我们将提供多角度的分析，帮助你建立更成熟、更清晰的认知。

## 第一部分：关于责任与承诺 🤝

上一节我们介绍了课程的整体框架，本节中我们来看看情感关系中的责任。首先需要矫正一个认知：我们应该是尽我们所能，而不是超出我们所能。

很多人对于爱情里的承诺，总是抱着最高的向往以及期待。觉得一旦得到了对方的承诺，彼此就要好好守住这个承诺，厮守一生。所以这也是很多“渣男”得逞的原因。或者是很多男生为了快点得到女生，就乱下承诺。然后女生以为对方承诺了，就在一起了。

如果你跟着我学习了一个多月的课程，我讲得最多的一个观点就是：关系是会变的。它是一个无常的。现在在说话的我跟上一秒的我，已经不是同一个人。今年跟去年的我也不一样。我们也不会期待三年后、十年后的我，跟现在的我一模一样。因为我们人生的目的就是体验和进化。我们不是被制造出来一成不变的机器人，我们是生活中的人。而所谓的生活，不管是成长或是活动，生活就意味着有变故、变动。因为我们一直在变化着，我们就像一条河流，不是一滩死水。

那么听到这里，你们就会知道，以后那种海誓山盟、做出终生承诺的“我”，是此刻的我。它只是一个当下的我。另外，课程也一直强调一个事情：我们要达到每一个境界，要去做到每一种训练，我们的建议都是一步一步来，而不是说突然之间、一夜之间就变成大神。也就是说，人们在感情当中做出了山盟海誓的承诺，也需要靠一小步一小步逐渐靠近，逐渐去达成目标。而不可能是说，做出承诺的那一瞬间，一切都完成了。

在童话故事里，当男主角克服了各种困难，终于跟女主角在一起的时候，作者通常会在结尾说一句话：“从此公主和王子就过上了幸福的一生。”为什么作者要用一句话，就把这个故事收尾了呢？你们有没有思考过这个问题？因为很多人以为，当幸福的生活开始，就永远都会这样。这就是很多人的认知误区。你想一下，在一起之后，柴米油盐酱醋茶，各种问题就来了。爱情是需要去经营的。

所以，如果我们已经认同了，作为一个成熟的人，都可以理解到：真实的生活是离不开变化的。逃不掉一步一步逐渐慢慢来。我们就不会天真地把感情孤立在人生之外来看待，不会相信感情里的承诺就不需要接受各种变化，或者各种耐心的考验。

当然，人对于安定是有需求的。所以希望能够靠着某些方式来达成对于承诺的信守。为什么要领结婚证呢？结婚证就相当于签合约，来守住我们的承诺。就是“你真的爱我吗？那真的爱我们就去结婚登记处。”有人会说，爱情这么浪漫，怎么可能跟签合约有关系，又不是做生意。可是实际上，你想一下，我们结婚时所签的婚约，不就是一种合约吗？它是一种约束双方的东西，类似一个证明盖章。

有的人除了婚约，还会签婚前协议书。通常要签婚前协议书的人，是因为自己拥有的财产过于巨大，他得提防对方。因为有时候，你跟我说你爱的是我的人，说得特别真诚。只要过了一段时间之后，变了，变得是爱我的钱了，那怎么办？所以如果跟自己结婚是为了钱而来，那么总有一天我们要防范对方靠着离婚这种手段，把自己的财产分掉一半。这种大家看新闻应该看多了，比如亚马逊的总裁贝索斯，他的老婆分了他一半财产，马上变成女首富。那么，就是签一个很冷酷的婚前协议书来讲好：一旦离婚，你也不可以分走我一半的财产。

一方面，在婚约里我们听到那些誓言还是说：不管贫穷或富裕，不管是生老或病死，只有死亡才能够将我们拆散。可是另一方面，又靠着白纸黑字的婚前协议书来限制对方可能分走的财产数字，这是不是很矛盾？

所以，我今天给大家讲的就是，怎么样去面对感情里的那些承诺和责任。首先我们要提供的一个角度就是：当我们听到对方对感情做出了承诺之后，我们心里可以有一种态度——**我要跟你一起努力，而不是我把这件事情就交给你**。这是两个完全不同的概念。

如果一听到对方做出了山盟海誓的承诺，立刻就反应到“我下半辈子的幸福就交在你的手中了”，那这个想法就非常天真，或者很懒。不管是感情或者感情以外的人生，把自己完全交到对方手中，那你接下来要干嘛？在家里躺着收钱吗？还是要躺着等死？

你也可以想一下，如果你现在要花钱去买股票，你之所以买这家公司的股票，是因为这家公司的老板承诺了，他会很努力地经营他的公司，你相信他提出的那种营运计划。虽然你没有办法付出心力跟着这老板一起去经营这个公司，可是你出了钱，代表什么？你在支持他实现他的计划。你所出的钱买的股票就代表对他们公司的支持，这就是你所谓的“跟他一起努力”。而这个公司的经营者到底有没有做到他所做的承诺，就会影响到你的股票到底是亏还是赚。

而当你的伴侣跟你说出承诺的时候，如果你想要跟他一起努力的话，你付出的不是钱，而是心力。成熟的态度就是“我要跟你一起努力”。而为什么你想到的是“我要跟你一起努力”，而不是童话故事里说的“公主和王子从此就幸福地在一起”？因为你知道承诺没有那么容易去达成，它是需要努力的，还不见得能够克服万难达到目标。所以我在一开始的时候就告诉你们要把自己的目标写下来，你会发现写了目标之后还是需要付出很多努力的，并不是写了目标就可以实现。

第二个就是我们想要提供的视角来理解感情中的承诺：**每一句承诺恐怕都只说了一半的真相，另外有一半没有说出来**。其实我以前就想到跟你们说不要欺骗女生，对，我们不能去欺骗别人，但是我教会你们的一个方法就是：我们说事实的一半。什么意思呢？就是说“我要永远让你幸福”，这当然是一句听上去很动心的承诺，可是这个承诺它只说了好听的一半，至于不好听的那一半没有说出来。

就像你买股票的时候，公司的经营者承诺会好好努力的，会把公司经营好的。他不太会提醒你听到不好听的那一半，比如遇到什么经营困难、经营失败，对吧，金融危机等等，你的股票会变成泡沫。你想一下如果这个公司的经营者跟你说：“如果我等会经营不好，你的钱都打水漂了。”你还会买股票吗？是不是你肯定会嗯，愣了一下。所以当你的另一半承诺说“我要让你永远幸福”的时候，他没有说出那一半不好听的，就是“如果我做不到，那我们就分开吧”。这就是潜台词。

今天大家都可以看到中国的离婚率非常高，很少有人会对离婚这件事大惊小怪了。你想一下所有离了婚的人，当初在结婚的时候没有山盟海誓过吗？好，那么所有的有商业经验、有契约精神、合约精神的人都知道，合约里面不会只讲好听的事情。合约里面会讲到“我要用一个亿来买你公司，可是你的公司要确保每一年能至少带来2000万的营业额，如果达不到营业额的话，你这个钱是要退给我的。”这就相当于什么？对赌。就好像我后面在招聘一些人的时候，因为我不知道他的真正实力有多少，他跟我说的是好听，然后我就会采用这种对赌的方式。比如他说他要一万的月薪，那么我就把一万的月薪拆解出来，达到了就给他，达不到就按多少，这样就非常合理。只要这个人有实力，他就会来接受这个对赌，因为对他是有保底的，并不是说他达不到一分钱都没有，然后甚至他超出了还可以得到更多。这就是合理地利用这些东西。

因为承诺，它只是一份口头浪漫的合约。我们当然不需要在热烈的时候很扫兴地去拆穿他，对吧？可是我们心里呢，始终要知道合约终究是合约，里面就永远隐藏着签约的真相，那就是：**如果我们无法达成，我们就只能终止合作，终止合约**。

再来一个我们希望提醒的角度就是：不管是怎么样跟自己亲近的人，当对方做出承诺的时候，我们都要有一个自我提醒，就是**不要把该归自己的责任推在、赖在对方身上**。我们可能曾经都很热血沸腾地跟爸妈讲：“我一定会努力用功的，我一定会好好读书，好好做人。”可是当我们如果考试考不好，我们并不会认为爸妈会因此而感到崩溃，我们当然也不认为爸爸妈妈会叫我们去死，对吧？我们人生的幸福呢，始终还是要靠自己扛起来。而对方人生的幸福也不至于完全依赖在你们身上。不管是在商业上、情感中、亲情里面，都逃不过这个本质。

所以，当我们的伴侣没办法实现他所做的承诺的时候，我们的人生也不用因此而崩溃。因为我们一开始就没有打算把归自己的人生完全压在对方身上。爱情中的承诺是双方的向往以及善意，并不意味着我们可以把自己的人生责任转让给对方。

所以当你听完这部分的内容之后，你应该对责任、承诺这个东西要有一个清晰明确的认知。包括以后你去向别人说承诺的时候，你也知道你这个承诺代表着什么，对吧？又或者对方向你承诺，你开心OK，但是你要保持平常心，因为谁都不知道未来会发生什么事情。我们只能做好当下的每一天。

## 第二部分：关于付出的误区 ⚖️

上一节我们探讨了责任与承诺的本质，本节中我们来看看经常被混淆的“付出”与“牺牲”。我们经常听到有人说“为爱牺牲”，这个其实也有一些认知的误区。当一个人为爱去牺牲，他就属于“上头”了。上头这个东西，运气好就好，运气不好就容易牺牲，对不对？

就好像拿我们去看那种体育比赛，比如那些球星，他有些时候上头确实很牛逼，对吧？就创下一些新历史，比如像麦迪、科比、詹姆斯很多，可是他们常常都能够这样子发挥吗？不是的吧？有些时候他因为他的上头，也会导致球队失败。那么，我们在相处的时候，应该先追求自己的平稳，才有可能追求两个人的平衡。

当我们讲到那些伟大的爱情故事的时候，比如说什么刻骨铭心、山盟海誓、海枯石烂，这些动人的字眼可能爱情故事会用到这些很严重的字眼，为什么呢？多半是因为这个爱情里面发生了有人为爱牺牲的情节，他也许是为自己的爱人自杀，或者是为了爱去杀别人，对吧？如果真的发生了这种事情，你大概可以说这些爱情故事刻骨铭心、可歌可泣，可是你不会说这些故事是幸福的。也就是说，爱情里面的牺牲跟爱情的幸福很少能够画上等号。

如果你现在回顾你曾经谈过的恋爱，而感觉到自己有所牺牲的话，那么我们必须来搞清楚一件事情：**牺牲跟付出是不一样的**。我们在两人世界里，我们有可能为了照顾对方付出了很多的精力，可是你不会说“我为了要照顾你而牺牲了我自己”，对吧？如果想要建立一个分辨的标准，我们的建议是：我们应该把付出当成是心甘情愿的付出，而牺牲是当成是不甘心、不情愿的付出。这个就有点类似我之前说的，比如，你想要去追求财富，你是真的发自内心的渴望，还是发自于头脑的欲望？牺牲，它就像头脑里面的执着、自我。内心的付出就相当于明白了，是渴望，是发自内心的，愿意去做的，心甘情愿的，没有人强迫你的。

我们来说一下一个有趣的国家，日本。在日本呢，它的结婚习俗是什么？女性会放弃自己的工作，在家庭当主妇。可是根据日本前段时间的劳动力调查结果显示，结了婚、变成家庭主妇的主妇当中，有超过60%以上的女性，她们依然强烈地想要工作，她们感觉到是因为结婚而牺牲了自己的事业。那么我们就可以推测，这60%以上的家庭主妇，她们虽然想要回去工作，可是她们工作的内容未必称得上什么大事业。可是呢，她们会认为这是牺牲，因为她们觉得不情愿。如果用这个标准的话，难道先生工作就是情愿的吗？做先生的是不是有可能就会觉得，为了要扛起家庭的收入，他牺牲了他生活的自由，他牺牲了本来对人生所拥抱的梦想。

两个人因为相爱而结婚，这当然应该算是在追求人生的幸福。可是如果追求人生幸福的两个人，双方各自都感觉到：留在家里的觉得，他牺牲了人生的梦想和自由；出去工作的那一方，他会觉得他牺牲了梦想和自由。那你说这个婚姻是不是结得很荒谬？

听到这里，我想所有感觉到自己曾经在两性关系里做出了重大付出而感觉到牺牲的人，可能要想一想自己，之所以界定自己的行为不是付出而是牺牲，这到底是因为什么呢？这可能意味着，做出牺牲的这一方，他们希望得到的是牺牲之后换来的回报，或者他们希望收到的是在自己做出牺牲之后，另一方也为自己做出重大的牺牲。这部分跟我们之前学习的“我们给另一半提供价值，也希望他来提供价值，从而保持平衡”就是一个道理。为什么？因为能给出的有限，或者是自己觉得“我已经把我所拥有的那些都付出给你了，你也要为我做些什么？”其实从这些思维方式就可以看得出来，这些人是很缺爱的，里面内心是非常匮乏，缺爱缺安全感等等的。这很正常没有什么好与不好，只是我分析到这个本质。

那我们都听过一句话叫“英雄自古难过美人关”，“不爱江山爱美人”。这句话经常被人拿出来讲，是关于曾经的英国国王爱德华八世。爱德华八世爱上了辛普森夫人，而这个夫人不仅具有贵族的身份，本人还离过两次婚。英国王室基于宗教的原因，对于这样的身份是有意见的，所以王室就逼爱德华八世说，你必须要做出一个选择：你要不然就选择所爱的辛普森夫人，要不然就放弃当国王的机会。结果这个爱德华八世他选择了辛普森夫人，放弃了王位，这是在当时是一个轰动一时的决定，从此爱德华八世就成为“不爱江山爱美人”的代名词。

以前有个流行天后叫麦当娜，她自编自导了一部电影叫《W.E.》，讲的就是爱德华八世和辛普森夫人的故事。如果你有看过这个电影，你就可以看到，她所采用的角度是：爱德华放弃了王位之后，在生活当中变得自暴自弃，不断要求辛普森夫人去承担她这种失落的情绪。爱德华他觉得“自己放弃了王位，就应该从此跟辛普森夫人长相厮守”。可是如果两个人在一起的时候，她的心里满怀怨恨，她就等于把自己永远关进了情绪的牢笼当中。这个牢笼叫做“伟大的爱情”。而在里面，即使是辛普森夫人也会感觉到窒息。

如果是我们处于辛普森夫人的位置，我们也会非常的为难。第一，我们已经没有能力做出比对方牺牲了王位更大的牺牲。你想一下，一个国家的王位他都不要了，这个牺牲的非常大。我们也无从为对方的牺牲做什么重大的回报。对方牺牲了当国王的机会，能够回报他什么呢？让他觉得是值得的呢？

所以很多伴侣平常在没有事情的时候，总是会觉得自己就是在心甘情愿的付出。可是一旦吵架，情绪崩溃，就会脱口而出说出两个字——“牺牲”。最常见的就是那些结婚的，男的就会说：“我为这个家庭牺牲了很多东西，每天在外面没日没夜的工作，为了养活这个家庭，我牺牲了非常多。”女方就会说：“我为了这个家庭，我也不去工作，也没有收入，把所有的心血都付出在这个家庭，变成了一个黄脸婆。”这些对白，无论是生活当中、电视剧经常看到。当你说出这样的话的时候，就表示了你的心态就是这样想的。你所经营的婚姻、你所照顾的家庭、你所照顾的孩子，不再是你心甘情愿付出得到的人生成就，而变成你做出重大牺牲之后所换来的代价。而一旦你在吵架的时候，你用到了“牺牲”的字眼，就很容易激怒对方，两个人就吵起来了。然后对方肯定会说：“你这样子哪叫什么牺牲啊？我付出的、我牺牲的才多呢，对不对？”一直吵啊吵啊吵，没完没了。

你可以想一下，如果你已经有小孩子了，然后你当着小孩子的面，两个人在吵来吵去，那小孩子他接收到的信息会是什么呢？你别以为小孩子不说话什么都好像不是很懂，但是呢，小孩子他从那个时候就会植入一个种子，就是什么？“哦，原来我的存在是靠著父母亲分别牺牲的自己才得到的。”这就是会在孩子的心目中种的一个种子。那么你想一下，这个孩子以后结婚的时候，他也会什么？因为他这个种子一直都在，他也会重复他爸妈的事件，而他这个种子、这个磁场，他又会吸引到跟他类似的。所以呢，几辈子都是没完没了在吵架，是不是很神奇？这些就是规律。

所以，我今天看到一个也是资深的那种育儿专家，他就说过，当你要跟一个人结婚，你就问他一个问题，问他爸妈相处的状态。如果他很开心地跟你分享说明什么？他是一个在有爱的家庭里成长的。而如果你问到他爸妈，他突然脸色就变了，然后支支吾吾也讲不出什么。你明白吗？你就可以觉察得到，对方的父母有一些什么事情，那么有些东西就会对他有所影响。

如果我们希望自己避免这种心态的话，我们可能要培养另外一个原则，就是：**我们在付出的时候，这个付出的本身就是一种回报**。人生并不是一次任务，也不是说为了完成任务而必须要牺牲掉特定的事物。在生活的过程当中，我们会遇到一些人，去了一些地方，做了一些事情，这个就是生活本身的过程，而不是说为了达到一个结果而采用的步骤。

很多人他会用一些流行的字眼，叫做“活在当下，享受当下”，可是实际上呢，我们没有别的选择。我们并不是故意要告诉自己要活在当下，才会活在当下。而是我们根本没有办法活在过去，或者活在未来。我们只能够活在当下的这一秒钟而已。这些是从佛法里面衍生出来的，我用了一些很通俗的解释跟你们分享。那心理学上面呢，很勉强地对于“当下”这个词做了一个长度的界定，大概就是三秒钟。也就是说，在三秒钟的时间宽度里面，我们会感觉这是现在，这是当下，可是我们过了三秒之后呢，就会觉得那是刚刚，那是过去。我们也许对未来有想像、有计划，可是我们依然没有办法活在未来。那既然没有办法活在过去，也没有办法活在未来，我们唯一能够做的就是活在此刻。如果一个人，他摆明了自己只想活在过去，或者想要活在未来，那表示他已经不再活在现在了，他已经走偏了人生的轨道了。

当我们在两人关系里面，就很容易感到自己做了重大牺牲的时候，我们可能要问自己：难道这场恋爱，每一个当下，我们都没有享受到人生吗？我们都没有充分地感受到自我的存在，都没有燃烧的感觉吗？如果有的话，这就是这场恋爱的价值，而不是把它当做一次任务，把它当成是我们做了重大的牺牲，来达到一个什么未来的目标。

爱德华八世，他如果放弃了王位选择的爱情，他就应该认识到，他觉得爱情比权利更重要。如果后面他后悔了，那只能够说明他选错了。如果他以为，他做了这样的选择，是为世界上示范了一种浪漫的伟大，而希望有什么人跑出来颁一个比国王更大的王位奖给他，那是虚妄的幻想。

所以，我们一旦在两人关系里面，感觉到自己做了重大牺牲的时候，我们第一个并不是要求别人，而是检讨自己：为什么会产生这样子的心情？那自己想要什么样的东西来做回报吗？如果你不能够及时修正这个心态，你也许可能能重新回去享受爱情的甜美，而不是徒劳无功的空等一份那种你永远都不会到来的回报。

以上就是第二部分的内容，关于付出的一些误区。
