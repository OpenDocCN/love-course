到了学习的中期，我们会要求学员在稳定自身情绪的同时，注意观察女生的反应。有条件的话，尽量在现场与女生进行更多交流，甚至直接邀请她喝杯咖啡。这才是我们所推崇的方式。

**核心公式：有效社交 = 搭讪 + 即时约会 + 关系升级**

也许很多同学会觉得这一切发生得太快，简直不可思议，认为电影太会忽悠观众。但在这里，我要负责任地告诉大家，类似爱情故事的开始，不仅仅发生在电影里。
