# 1、老吴：老吴《中级情感私教》：二十一、如何把自己变成高手_note

在本节课中，我们将学习如何通过借鉴不同领域高手的思维与训练方法，来系统地提升自己，实现个人成长与突破。我们将剖析几位成功人士的核心观点，并将其转化为可操作的步骤。

## 高手是如何训练出来的

上一节我们探讨了情感关系的底层逻辑，本节中我们来看看那些在各个领域取得卓越成就的人，他们的成长路径和训练方法有何共通之处。这些内容源于对《朋友圈的尖子生》一书的思考与总结。

### 1. 杨超：兴趣、擅长与价值的三元驱动

杨超认为，个人的所有行为基本都可以用兴趣来解释。对世界充满好奇，并喜欢用理论去解释世界，是驱动成长的关键。

以下是找到个人发展方向的三个核心切入点：

*   **找到喜欢的**：喜欢是指能让你进入“心流”状态的事情，即做起来不知疲倦、充满愉悦感。例如，帮助他人成长会带来巨大的满足感。
*   **找到擅长的**：擅长即个人天赋。每个人天生都有不同的特长，应朝着自己擅长的领域去发展。例如，有些人天生就适合传播智慧、帮助他人。
*   **找到有价值的**：价值意味着所做之事能利他，能为他人创造正面价值。这件事应当是你的使命，且必须是正面的、符合社会道德的。

结合这三点进行思考，就能找到真正适合自己的方向。好奇心至关重要，缺乏好奇心是件可怕的事情。

此外，对任何行业都应保持敬畏。当你在某个领域小有成就时，很可能会发现世界上还存在你完全不了解的玩法，从而意识到自己的局限。个人成长就是一个不断发现过去自己“很傻”的过程，更新换代的速度越快，说明成长越快。

### 2. 李倩：通过交流与深度研究发现机会

李倩有一个特点：在做一件事之前，会与所有认识且认为有价值的朋友聊天，从中发现事业机会。

以下是她的核心方法：

*   **与优秀者交流**：当不知道自己适合做什么时，应主动与比你优秀、有正能量、有想法的人接触和聊天，这有助于发现潜在机会。
*   **判断人的潜力**：评价一个人是否“牛逼”，应看其未来潜质，而非当下的学习成绩。成绩只是特定考核标准下的产物。
*   **保持好奇心并深度研究**：例如，想创业就去参观和面试多家创业公司，通过不断向面试官提问来了解行业全貌。为了深入了解一个领域，她会买几十本该领域的经典书籍，潜心研究、做笔记并整合信息。
*   **专注与抗干扰**：她受干扰少，抗打击能力强，做事情非常专注，有原则和底线。她习惯从原理出发揪出事情本质，先明确目的再想办法。因为商业环境、时代、公司模式都在变化，成功经验难以复制，但失败教训往往相似。我们能学习的是成功者的特质，并将其内化。
*   **建立知识框架**：看教科书能建立知识框架，但看经典书籍时需注意验证，因为有些内容可能已不适用于当下。真正有效的方法是结合专业理论与实际情况。

**一个启发性的案例**：如何在5分钟内找到中国政法大学校长的电话？常规思路是找人脉，但他的方法是：校长是公开职务，必然参加过公开会议。于是，他搜索该校长过往参加的学术会议信息，最终在网上找到一份留有他联系方式（当时是13X开头的手机号格式）的会议文档，通过试号找到了电话。

这个故事给我们的启发是：**不要依赖别人教或固有经验。当被逼到窘境时，只要思考事情的本质和目的，就能自己找到解决方案。**

他还强调，在任何企业工作，都必须懂得其核心业务。每个行业都有一把“钥匙”（即规律），找到规律后，结合资源、认知和努力去思考，就能成为行业头部的20%。很多人只是为了工作而工作，并非想创造或引领行业。

他追求的是“认知效率”，即追求对事物本质的思考。进入新行业，应先想清楚这个行业是怎么回事，再叠加“技术效率”。同时，要保持好奇心，了解行业最好的样子是什么。

关于个人品牌，他提出了差异化打造方法：成为“交叉领域专家”。例如，在营销圈里最懂投资，在投资圈里最懂营销。这种差异化身份能帮助获取流量。

对于毕业几年感到迷茫的年轻人，他给出三点建议：
1.  **找到好机会**：选择比努力更重要。
2.  **明确目的**：做任何事都不要为了做而做，要想清楚目的，高效寻找最短路径。
3.  **向顶尖者学习**：不要看大众的成功学，直接向该领域最顶尖的实践者学习。

很多高手解答题（执行）水平一流，但选择题（决策）能力很差。做好解答题需要把自己逼到窘境去全力解决；做好选择题则需要明确选择依据，判断这是否是达成目的的最短路径。

### 3. 马文亚：财富认知与目标管理

马文亚少年得志，他的成功源于家庭观念、教育、个人理念和努力的共同作用。

他的核心观点如下：

*   **财富认知**：在想如何把钱变多之前，先学会如何不把钱变少。很多事情急不得。
*   **信息套利案例**：他通过观察发现硬币两面图案的稀缺性不同（如“长城”版比“国徽”版少），就用常见的“国徽”版硬币向同学换稀缺的“长城”版，再到钱币市场用1个“长城”版换10个“国徽”版，利用信息不对称获利。这启示我们，未来社会比拼的是“硬核”实力，个人品牌将愈发重要。
*   **消费观念**：花钱时要问自己：是买到了真正想要的东西，还是只买到了别人觉得你应该买的东西（如名牌）？检验标准是“幸福感”：购买后若不告诉任何人，心里会不会憋得慌？如果会，说明是为了炫耀而非真正需要。
*   **研究方法**：要进攻一个领域，他会收集该领域过去的新闻杂志进行深入研究，对比历史与现状的差异，从而掌握事物规律。
*   **目标管理建议**：
    1.  **锁定目标**：从众多想做的事情中，筛选出1-2个这辈子一定要完成的核心目标。
    2.  **拆解目标**：将大目标拆解为小目标。例如，将“环游世界”拆解为“每年去几个国家”。
    3.  **每日行动**：每天为目标做点事，哪怕只有5分钟。例如，为旅行目的地做5分钟调研。
    4.  **微量开始**：对于重要的事（如读书），如果没时间，就从每天读1分钟开始。
    5.  **养成习惯**：每天进步一点点，定期复盘，不要等到年底才总结。

### 4. 马薇薇：管理期待与精简社交

马薇薇（奇葩说辩手）的观点侧重于心理调节与人际关系。

*   **管理社会期待**：过高的社会期待是危险的，它会让原本厉害的事情变得不厉害。**口碑 = 实际获得 - 期待值**。一旦期待过高，口碑就难做。因此，要学会降低他人对自己的期待，避免给自己过大的压力。
*   **精简社交**：减少不必要的社交活动，少参与纯商务应酬。多与有趣的人交往，对于不喜欢、不好玩的活动，果断拒绝，不要勉强自己。

### 5. 李教授：战略思维与知识体系构建

李教授（百度前副总裁）的方法论更具系统性，强调战略与思维模型。

*   **观察与刻意练习**：他观察力强，能从打架中发现玄机。大多数人靠勇猛，而他研究各流派技法，总结共通套路，并刻意练习。这本质上是**寻找思维模型、逻辑思考、刻意练习**的过程。
*   **策略性思考**：做任何事，如果不知道怎么做，就一定要想办法找到一个“招”（方法）。如果想不到，宁可暂时放弃。
*   **挣脱“价值网”**：个人和公司要取得突破性成长，常常需要挣脱原有的价值网络（即固有的生存结构与依赖关系）。
*   **获取影响力的两种方式**：
    1.  **通过关系**：优势是连接紧密，缺点是覆盖范围小，维护成本高。
    2.  **通过知识**：优势是影响范围广、速度快，缺点是连接较松散。
*   **创造差异化附加值**：商业成功的关键是持续创造并提供差异化的价值。**影响力来源于“附加值”**，即那些没有你就不会发生的事情。要想获得影响力，必须塑造差异化的竞争力。
*   **克服从众心理**：培养差异化是违反人类“从众”本能的，会面临巨大压力。勇敢不是无所畏惧，而是**有所畏惧，依然前行**。
*   **建立战略知识体系**：最大的成功学是“战略学”。**战略 = 达到目标的“招”**。要具备战略思维和设计战略的能力，用战略指导行为。要区分**目标**（如业绩增长50%）和**战略**（达到目标的方法），目标是战略的结果。
*   **构建系统化知识体系的方法**：关键在于**建立知识之间的联系**，即连接不同的思维模型。
    *   **具体做法**：每学到一个有用知识，停下来寻找联系，思考这个理论还能解释哪些其他现象。要求自己至少找到3-5个关联实例。
    *   **案例：视觉化思维**：理论指出“人的大脑（爬行脑）更喜欢视觉化信息，而非抽象信息”。如果只是听过就忘，知识便无法内化。
        以下是寻找联系的例子：
        *   **产品宣传**：乔布斯说“把1000首歌放进口袋”，比“小体积大容量MP3”更形象。
        *   **广告文案**：“德芙，纵享丝滑”。
        *   **理论传播**：雷军说“站在风口上，猪也能飞”，比“抓住机会，能力不高也可能成功”更易传播。
        *   **谚语**：“枪打出头鸟”、“早起的鸟儿有虫吃”，都具有强烈画面感。
        *   **谣言**：“WiFi会杀精”比“WiFi可能影响健康”更令人印象深刻。
        *   **演讲**：马丁·路德·金说“我梦想有一天…奴隶的儿子和奴隶主的儿子坐在一起…”，画面感极强。
        *   **情感表达**：“我希望我们七十岁时，还能手牵手走在巴厘岛的沙滩上”比“我会永远和你在一起”更动人。

通过这种刻意建立知识联系的方法，你的知识储备才能真正形成体系，并转化为实际能力。聊天中善用比喻、营造画面感，就是这一理论的应用。

## 总结

本节课中，我们一起学习了多位高手成长的核心心法：
1.  **向内探索**：结合兴趣、擅长与价值，找到人生方向。
2.  **向外连接**：通过与优秀者交流、深度研究行业本质来发现机会。
3.  **理性决策**：在行动（解答题）上全力以赴，在选择（选择题）上明确依据。
4.  **管理资源**：建立正确的财富观，并通过锁定、拆解、每日行动来管理目标。
5.  **调节心理**：管理社会期待，精简社交以保存精力。
6.  **构建体系**：用战略思维指导人生，并通过建立知识间的联系来构建系统化的知识体系，尤其要善用“视觉化”等思维模型。

最关键的是，要勇于创造差异化价值，克服从众本能，即使有所畏惧，也要坚定前行。将前几节课的底层逻辑思考方法与此处的高手训练法结合，便能解决绝大多数成长问题。
