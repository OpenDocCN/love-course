# 1、老吴：老吴《中级情感私教》：十五、情感矛盾的解法及吸引如何建立_note

在本节课中，我们将探讨情感关系中的几个核心矛盾与解法，并学习如何建立持久的吸引力。课程将分为三个主要部分：剖析分手后的不甘心情绪、探讨与前任做朋友的可能性，以及系统性地讲解如何提升个人魅力以吸引他人。

## 第一部分：为何在意前任的新欢？🤔

上一节我们结束了关于情感关系的探讨，本节中我们来看看分手后一种常见的微妙心理：为何我们会在意前任是否很快开始了新恋情。

分手意味着双方重获自由。既然自由了，对方是否有新欢，似乎与我们无关。但很多人心里却放不下，甚至会产生“无缝接轨就是出轨”的阴谋论想法。

我们来研究产生这种感受的原因。

**第一个原因是“损失确认”心理。** 这类似于投资股票。当股票上涨时，股民倾向于获利了结。但当股票下跌时，股民往往不愿卖出，因为一旦卖出，亏损就确定了，不再有反弹的希望。在恋爱中，分手后我们心里常存眷恋，认为彼此留下的痕迹是“刻骨铭心”的。然而，一旦对方找到新欢，我们就会恍然大悟：原来那些痕迹并非刻骨铭心，而像雪地脚印，一场新雪就能覆盖。这等于在分手后，再次体验到失恋的感觉。

**第二个原因是“主动感被戳破”。** 我们都希望对自己的生活有控制能力。一个心理学实验将受试者分为两组，都在有巨大噪音的房间答题。第二组身边多了一个可以关闭噪音的按钮，但无人使用。结果第二组答题效果远好于第一组。这说明，当人认为自己有控制权时，焦虑会下降。有些人分手后，仍幻想自己有一个按钮可以干涉对方生活或恢复关系。对方找到新恋人，就彻底戳破了这种对主动控制能力的幻想。

**第三个原因是我们喜欢在爱情中“比赛”。** 无论是恋爱中还是分手后，我们都倾向于比较谁爱得更多、谁的损失更大。一旦发现对方领先我们找到新对象，就意味着我们留恋这段爱的程度超过了对方。

要渡过这种微妙的情绪，心理学上有一个建议：培养 **“可获得感”** 。对于向往的事物（如一件衣服、一份工作或一段新恋情），如果我们怀有能够获得的信念，就不会一再沉溺于过去，追究是谁的责任。我们要提醒自己，一段恋情结束，意味着人生中被占据的空间又空出来了。我们应该思考如何填满这个新空间。前任是否有新交往对象，已与这个空间无关。

## 第二部分：分手后还能做朋友吗？🤝

在理解了分手后的复杂情绪后，我们来看看另一个常见问题：分手后还能做朋友吗？答案是可以的，但关键在于掌握好边界。

要弄清关系，先要弄清自己。不论是交朋友还是谈恋爱，都讲究“交心”。但一个人一辈子的心力有限。前任比起陌生人，更懂我们、更了解我们。如果不能再当恋人，将其变成朋友，是否比重新与陌生人建立友谊更合理？

以下是想要与前任维持朋友关系时需要注意的事项。

人与人交往要拿捏分寸，这在心理学上叫做 **“边界感”** 。从恋人转为朋友，两人之间的边界也随之转换。每个人对边界的认知不同。人际关系中的边界，就是要明确在日常行为中应透露哪些信息、遵循哪些规则。

之前网络上有一个国外知名博主的分手视频。视频结尾，女主角痛哭，男生本想亲吻以示关怀，但意识到已非情侣关系，最终选择了拍拍她的背。从认识到成为伴侣，是逐渐突破边界感的过程。要从情侣退回朋友，也需要将边界感从亲密关系一步步退回。对这种边界感的尊重，是分手后还能做朋友的重要前提。

当然，最简单的一个原则是：搞清楚自己为何想维持朋友关系。如果经过自我探索，发现心底隐约想要复合，那就做不成朋友了。因为你怀揣的不是做朋友的想法，不可能尊重新的边界感，总想找机会突破边界，重回情侣关系。这也是一些经验之谈认为“分手后不如不再联系”的原因，因为朋友间的边界感很难掌控。

顺着这个原则，另一个要点是：分手后若要做朋友，心里就不应再期望享受对方作为恋人时的那些美好。就像从金卡会员降为普通会员，就不应再奢望金卡特权。把握好这个原则，既是为对方着想，也是为自己着想。分手后，我们应鼓励自己，人生中爱情的空间又腾出来了，应该准备好迎接新恋情。若沉溺于过去，就无法容纳新的可能。

情商追求的是把自己和别人都当作独立完整的个体看待。如果把前任看作独立完整的个体，就会体会到对方身上有许多值得珍惜的成分，比如志趣相投、心灵相通、三观一致。当处在恋爱关系时，不在一起会互相怀念甚至痛苦；当只是朋友时，在一起会感到愉快自在。这并非两种不同的感情，只是程度不同。如果非常珍惜与对方的相处时光，觉得很多回忆独一无二，或觉得与他最聊得来，不让其变成朋友实在可惜。

这样想时，一定要把握的是：不会对对方造成困扰和痛苦。如果你只想做朋友，而对方一心想要复合，就会使他痛苦挣扎。这时最好的做法是顺其自然放手，等对方调整好，两人才有机会恢复朋友关系。

还有一种微妙状况：有些人当朋友时特别好相处，但当情侣时就变得爱吃醋、斤斤计较、疑心重、爱吵架。一旦解除紧张的情侣关系，这个人又恢复到朋友时代的轻松愉快。这表明他很适合在分手后继续当朋友，因为免除了恋爱时的压力，反而让他最美好的部分呈现出来。就像有些衣服适合休闲穿，但不适合出席重要活动。谈恋爱是一种非常重要的人际关系，在这种关系中，每个人最好和最坏的部分都会呈现。如果我们珍惜这个人当朋友时带来的安慰和力量，那他就是值得在分手后继续当朋友的人选。

刚分手时，痛苦感还很强烈，此时维持朋友关系可能困难。人生峰回路转，等待一段时间，或许两人各自有了新交往对象后，再考虑能否成为朋友，可能是更实际的做法。

## 第三部分：如何建立吸引力，让女生喜欢上你？✨

在探讨了情感矛盾的解法后，我们进入本章节的重头戏：如何系统性地建立吸引力。这部分内容基于大量的观察与研究。

社会发展越快，人群同质化越严重，很多人迷失了自己。在快速迭代的社会中，寻找自己的位置尤为重要。谁都想让身边人崇拜自己，但找不到方法。实际上，只需掌握三个核心板块，就能将自己变成一个受欢迎的人。

你可以把自己想象成一个社交达人。这三大板块是：**人**、**沟通**、**圈子**。它们贯穿了我们整个人。

**“人”** 分为内在与外在。很多人提升自己时，会选择从形象入手，或去包装自己、混圈子。这些是技巧，是“术”的层面。但更重要的部分是“内核”。外在改变最快，但会遇到瓶颈，需要内在东西做支撑。我们的课程体系，除了情商课，也会有形象穿搭课，教大家包装外在。在移动互联网时代，认识一个人常会先看其朋友圈。好的朋友圈有内容、会讲故事、能吸引人。

**“沟通”** 分为线上与线下。随着个人成长与自我完善，你的能量场会越来越高，吸引越来越多的人关注，时间也会越来越宝贵。提高情商后，你就能快速筛选身边的人，将优质人脉筛选出来，变成你的圈子。

在当今时代，要变成一个很有魅力的人，可以遵循一个 **“五部曲”**：
1.  完善自己的内在外在。
2.  做好自己的展示面（如朋友圈）。
3.  学会情商沟通。
4.  快速筛选人脉。
5.  形成自己的优质圈子。

但请注意，这五部曲少了一个关键东西：**资源（人脉）**。如果你想找优质女生，需要从海量人群中寻找。如果空有一身本领却没人认识你，等于零。现在已不是“酒香不怕巷子深”的年代。

因此，正确的流程应该是：
1.  完善自己的内在外在。
2.  打造好自己的展示面。
3.  **吸引你的受众人脉资源。**
4.  通过沟通进行筛选。
5.  形成自己的圈子。

整个过程就像一条工业流水线，按流程执行即可。很多人总想一步登天，不愿投资自己或执行，总想着“蹭”，把自己包装得高大上。但明眼人一看就知真假，因为很多照片呈现出的形象气质会说明一切。

所以，想成为受欢迎的人，得先做好最重要的一步：**你自己**。很多时候，问题的根源不在于你说了什么、做了什么，而在于“载体”是谁。**你是谁，很重要。**

以下是几个例子：
*   **公式：反应 = f(行为者, 行为)**
*   李嘉诚和一个不知名企业家说同样的话，你更信服李嘉诚。
*   吴亦凡和一个普通男生发“约吗”，得到的反应截然不同。女生对吴亦凡可能很开心，对普通男生可能觉得变态、屌丝。“约吗”这两个字本身无错，错在谁说。
*   女人都喜欢花，但喜欢的人送和不喜欢的送，区别很大。送花行为本身无错，错在送的人是谁。

这样的案例成千上万。所以，在成为有魅力的人这条路上，**你是一个什么样的人，显得尤为重要**。这也是为什么整天强调要去提升自己。

再换一个方式表达：大家都知道华为。它之所以瞩目，是因为掌握了芯片技术等核心科技。这相当于我们的 **“内核”** 。你看到很多手机产品都不错，但你会更推崇华为，因为它是民族之光，且有核心竞争力（芯片）。同时，它的推广做得很好，从海量用户中筛选出的业绩自然好。你可以把这理解为一个好产品的销售过程。

我之前强调做好朋友圈等社交媒介展示面，因为它就像你这个人的 **“广告页”** 。朋友圈相当于你公司的广告。我之前有个门徒做生意，朋友圈的车从五菱宏光换到本田，再换到路虎，现在又创业。他这么做就是在做广告、推广自己，这样去谈合作效果就不一样。好比你是公司老板，A来自BAT，B来自不知名公司，看简介时你心里会优先考虑A。所以，人的重要性、背书、朋友圈很重要。有时就因为某点没做好，第一步就被筛选掉了。

当你开始提升自己，会发现了解你的人越来越多，喜欢你和不喜欢的也越来越多。这时，你只需从你喜欢、同频的人里，找到核心资源和人脉进行筛选即可。

**第二个要点：清楚人是关系的产物。** 社交就是人与人的交际往来。在人生不同阶段会遇到不同困难，有的简单能自解，有的需要他人帮助。这时社交的重要性就凸显了，因为有的人一句话、一个电话就能帮你解决问题，让你少走弯路。你就不应把时间浪费在不必要的事情上。

很多人在面对社交时带有恐惧和抵触心理。我以前也很内向，不爱社交，不懂社交。上大学后才开始改变，接触不同圈子。我的初心不是为了达到某种目的而去接触（这样会很累），而是以正面、学习的心态去面对。多接触人事物，对自己没有害处。

关于心态调整，前面的课程讲了很多。为什么会有“看人下菜碟”的区别？就是因为 **吸引力** 的不同。吸引力来自于你自己的内心、你的能量场、你的频率。你发射的能量场高，吸引来的就是高价值的人。

再给大家讲一个概念：**“目击效应”**。这个术语源自老祖宗庄子“呆若木鸡”的故事。故事是这样的：
> 古代，训鸡高手纪渻子为君王训练斗鸡。十天后，君王问可否上场，纪渻子说不行，它杀气腾腾，一上场就横冲直撞。又过十天，还是不行，它一听见别的鸡叫就斗志昂扬。再过十天，仍不行，它一看见鸡影就来势汹汹。最后十天过去，纪渻子说大功告成。如今这只鸡在竞技场上，无论别的鸡如何挑衅，它都像木鸡一样无动于衷，内心充满德行。现在无论什么样的斗鸡遇见它，都会落荒而逃。

大家听懂这个故事的意义了吗？这其实就是一个训练 **需求感** 的过程。打个比喻：如果把这只斗鸡换成一个男人，其他斗鸡换成女人，会有什么感觉？训练官（比如我）说不行，这个男人不能上场，因为他杀气腾腾（需求感太强）。我要控制他的需求感。过了10天，他一听到女人声音又很兴奋，欲望上来了，所以还不行。再10天后，他听到声音没反应了，但看到女人又兴奋了，还不行。最后，他看到女人没感觉了，听到也没反应了。这时他就没有需求感了。无论什么样的女人看到他，都会觉得这个男人很特别；无论女人怎么勾引他，都没用。

那为什么会产生这种“目击效应”呢？主要原因如下：
1.  **目击型的人具有强大吸引力。** 古人说“大智若愚”。这种人话语不多，但说出的每句话都在理、富有人情味，有巨大的说服力和感召力。目击型的人一般欲望淡薄、内心沉静、坦率憨厚，给人一种无形的可接近感、合作感、融合感。跟这样的人在一起不需要戒备，有安全感和幸福感。
2.  **目击型的人外形像木鸡，实则是成熟的智慧。** 他们是久经锻炼后，才拥有不外露而内敛的人格魅力。他们不是不善于表现，而是认为轻易自我表现会有损形象和魅力。所以他们惜字如金，应验了“沉默是金”的道理。他们要么不说，一说就说到点子上并产生效果。
3.  **目击型的人因木讷，给人一种高度可测的距离感。** 这种距离感使人无法真正了解其内心世界，从而产生一种威严感和威慑力。他人会认为听从是最好的选择，不听从则可能对自己不利，因此不得不从态度和行动上跟从。
4.  **目击型的人给人一种神秘色彩。** 说他木讷，他怎能令人信服？说他精明，他又为何如此木讷？给人一种琢磨不透、神秘兮兮的感觉，带来好奇心和关注。人们又会觉得这种人是“大智若愚”者，从而产生敬畏感和崇拜感，自然就被吸引，愿意跟随。

这就是吸引力一个很好的解释。就像给你很多钱买车，你首先会考虑BBA（奔驰、宝马、奥迪），因为这是品牌的力量。包括“脑白金=过节送礼”的slogan，已深深植入我们脑海。

不同的人会吸引不同的人。你想吸引什么人，就得懂得分析这些人的需求，然后根据他们的喜好去建立相应的吸引点，定位好自己。关于定位，在个人品牌课程会讲到。好的图片会说话，帮你建立吸引。你们可以看到，我以前发那些朋友圈，一发就一堆人上来找我，反应特别好。当你的朋友圈做好，发个朋友圈就能吸引一堆人，那你离“变现”还会遥远吗？当然，这些朋友圈需建立在真实的基础上。

当你开始吸引到很多女生后，要懂一个技能：**筛选**。即在同类事物中，去掉不需要的，留下需要的。你认识了1000个女生，她们目的不同：有的喜欢你的人，有的喜欢你的钱，有的喜欢和你聊天，有的需要你帮助。并非所有女生都值得你花时间，所以你要懂得筛选。玩圈子时也要懂得这点，不要把时间浪费在不必要的人身上。

我们都听过“物以类聚，人以群分”。什么样的人，就跟什么样的人在一起。当你发现身边优秀的人越来越多，不好的人远离你，恭喜你，说明你在不断进步。当你发现身边的人没什么变化，说明你也没什么变化。所以有时身边的人流走，你要看一下这个人是牛人还是普通人。普通人流走很正常。我这十年起码认识五六千人，很多人把我删了，我也会删一些人。很多人只有一面之缘，后来有些玩得挺好，因为我成长了，他没成长，大家频道不一样了，就再见。所以我们要学会透过现象看本质。你看那些牛人，对自己的投资都非常多。我们也不要觉得一个人光鲜亮丽就很牛，也不要看一个人低调朴素就觉得他不行。

那么，如何提高自己的吸引力呢？上面的课程已经讲了。如果你还有需要，可以进一步学习个人品牌的内容。

很多人听完会有疑问：那我怎么样把自己变得更独特呢？只需要做一件事：**改变自己的生活方式**。抛开出生、家庭、环境背景，每个人都是公平的，每天都有24小时。你不能决定出生，但能决定以后的路怎么走。

我把生活方式分为四个板块：
1.  **衣食住行**
2.  **工作学业**
3.  **兴趣**
4.  **社交圈子**

这四个部分组成了我们的一生。你的圈子社交包括了家人、同学、朋友、爱人、工作伙伴等。衣食住行是生存基础。兴趣也要有，比如有人喜欢琴棋书画，有人喜欢运动、健身、阅读、旅游、蹦迪等。没有这些，生活会极其单调。这四个部分就组成了你，人与人的区别就是通过这些体现出来的。

没有对比就没有伤害。我们只需把精力集中在自己能控制的因素上，把这些价值发挥到最大化，而把那些不可控的因素放到一边，不要太在意，因为在意了也没有任何意义。
*   你会穿衣服有品味，就会让人赏心悦目。
*   你很会吃好吃的，人家就会找你问，跟你一起出去吃饭都会有幸福感。
*   关于住和行，住在好的小区环境，能量场会让你感觉特别舒服。你们去过不同小区，肯定有这种感觉。所以去舒服的小区会很安心，让你住城中村会受不了。这回归到我们课程的核心：人就是体验与进化。
*   同样是工作一小时，不同的人创造的价值不一样。牛人工作效率高，就有更多时间做更多事，慢慢就跟你拉开差距了。
*   关于兴趣，我接触了那么多，觉得最能提升自己的方式就是：**学习阅读、健身旅游、还有圈子**。“读万卷书不如行万里路，行万里路不如阅人无数，阅人无数不如明师指路”，说的就是以上几点。当然，其他如琴棋书画、体育运动、唱歌等兴趣，都是从不同角度和维度让你感受不一样的世界，丰富精神层面，让你的谈资更多。有的人经历多但不能表达，这也不行。所以我们要学会提高情商，让自己变得会沟通、会聊天。

大家学了这部分内容后，再看案例，就会比普通人看得更加透彻，因为这是一个根本的东西。

我们每一天就是要不断丰富自己的生活方式，让自己变得更好，也就是 **进化**。比如你很会穿衣服，就会有很多人想来咨询你的意见。你能够给别人一些穿衣搭配建议，这就是你的价值输出，是在给别人的“社交银行”存钱。你能存的银行多了，钱存得多了，说明你的“存款”就更多，说明你这个人越有价值。因为“术”的东西（套路技巧）谁都可以模仿，唯独你自己这个人是没办法模仿的。所以你只需要不断把自己变得更好就可以了，因为每个人都是独立的个体。

在当今社会，未来的趋势是 **玩圈子**。因为一个人的力量是有限的，你必须去融入高端的圈子，然后组建自己的圈子。这些是我们其他更高阶内容会讲到的。
