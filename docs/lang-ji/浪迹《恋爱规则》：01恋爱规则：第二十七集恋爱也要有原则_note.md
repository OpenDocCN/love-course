# 恋爱规则课程01：第二十七集 🛡️ 恋爱也要有原则

在本节课中，我们将要学习在恋爱关系中建立并坚守个人原则与底线的重要性。健康的感情需要双方互相尊重，而非单方面的妥协与牺牲。

上一节我们介绍了维护自我在关系中的价值，本节中我们来看看如何具体地设立并坚守恋爱中的原则。

很多人为了和女生在一起，会一而再再而三地退步。这样的恋爱最终即使得到了，也不会有好的结果。很多时候，男生为了让女生开心，会去做一些让自己难过的事情。这时自己感到难过，但女生可能会误会你是愿意去做的。其实只有你自己知道内心有多么不甘心。

但是，当你看到女生因此表现得快乐开心时，你又只能选择认了，把这种心酸藏在肚子里。然而，女生会认为你可以做这样的事情，那么下一次你也可以做。久而久之，你做的让自己不开心的事情会越来越多，而她却可能变本加厉，因为她一直不知道你的底线在哪里。

迟早有一天，你会无法再做那些你真正不愿意做的事情。这时，女生反而会责怪你，质问为何之前都做了而现在不做。

以下是关于如何建立原则的具体建议：

如果你有些事情不想做、不愿意做，在一开始的时候，就不要为了让对方开心而委屈自己。你一定要坚守住自己的这条底线。无论是任何人、无论是什么事，都不能为此而打破原则。

为什么要这样做呢？目的是让女生形成不去触碰你底线的习惯，不要让她一而再再而三地试探。也许，你在第一次拒绝女生的时候，她会不开心。

但是，请思考一下：你愿意和一个把她的开心建立在你的痛苦难过之上的女孩子相处吗？这样的女生真的适合当你的女朋友吗？一个女生如果真的喜欢你，她是绝对不会勉强你去做你不愿意做的事情的。就像你不会勉强你自己喜欢的女生，去做她不愿意做的事情一样。

**核心态度公式：坚守底线 + 明确沟通 = 相互尊重的健康关系**

做男人一定要有态度。坚守住自己的底线，做个真正的男人。学会告诉对方你的原则和底线，然后让两个人共同来经营感情，这才是真正的爱情。
