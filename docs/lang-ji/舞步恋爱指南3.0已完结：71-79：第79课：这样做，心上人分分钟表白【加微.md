## 什么是“预选”？

上一节我们提到了错误示范，本节中我们来看看“预选”的正确定义。

“预选”顾名思义，是指**已被预先选择**。这是一种社交信号，其本质是**社交认证**。具体而言，女生会通过观察其他女生对我的态度和好感度，来判断我自身的价值。

然而，“预选”经常被错误地理解为“花心”。试问，有哪个女生会喜欢一个花心的男生呢？

以下是“预选”的一个简单判断标准：

*   **真正的预选**：是**女人对我们的态度和好感**，它源自于**他人**（其他女性）的客观反馈。
*   **错误的表现（花心/炫耀）**：是**我们自己主动表达**有很多女人喜欢我们。这种行为本质上是装逼和炫耀，非但不是高价值展示，反而是一种低价值行为，会导致女生态度急转直下。

## 核心概念解析

两者的根本区别可以用一个公式来概括：

**有效的预选 = 他人（女性）展示 → 对你产生吸引力**
**无效的炫耀 = 自我展示（女性数量） → 对你产生反感**
