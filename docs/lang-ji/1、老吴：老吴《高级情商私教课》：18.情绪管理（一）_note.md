# 老吴《高级情商私教课》：18. 情绪管理（一）🧠

在本节课中，我们将要学习情绪管理的基础知识。我们将探讨情绪的本质、负面情绪的起源，并学习如何识别和处理痛苦情绪。课程将通过具体实例和直接的方法，帮助你理解并初步掌握管理自身情绪的技巧。

## 理解情绪的本质

上一节我们介绍了课程的整体框架，本节中我们来看看情绪到底是什么。

我们人类是100%情绪化的动物。情绪化指的是一个人的思想、言语、行为和决定等一切，包括我们的三观，都由自身的情绪决定。

这背后的原理是信念系统（B）的不同。情绪不佳时，就会产生负面的信念（B）。我们每一天、每一个当下的思想、言语、决定和行为，都会受自己的情绪影响。

然而，大多数人无法意识到这一点。很多人认为，是外部发生了某些情况或原因，才导致自己产生这样的情绪感受。实际上，情况正好相反：正是因为我们内在有这样的情绪感受，才导致我们看到外在世界的变化。

怎么理解呢？因为你的内在不开心，所以你反射出来看到的世界就是不开心的。

## 案例分析：如何应对他人的情绪变化

为了更具体地说明，我们来看一个例子。以下是处理他人情绪变化时需要注意的要点：

*   **不要主观臆断**：当别人情绪有变化时，千万不能把自己认为的（例如，哭就代表难过）强加给对方。哭有很多种：喜极而泣、难过或有感触。情感有不同的颗粒度，即细腻程度不同。
*   **使用“共情”方法**：最直接稳妥的方法是表达关心，询问“你还好吗？”。这会让对方觉得你在关心他内心的感受，而不是在错误判断他的情绪。
*   **避免破坏氛围**：千万不要犯傻去跟对方讨论具体情节或追问原因（例如，“是不是想起了什么？”），这会破坏当下的氛围和情绪。

**核心公式**：**先处理心情，再处理事情**。

## 认识与辨证自己的情绪

情绪就像性格一样，是波浪形起伏的，并非固定不变。你当下的情绪，就是你当下的自己。

我们每一个人都需要去辨证当下的自己正处于什么样的情绪，从而渐渐了解自己。这是一个过程。当你尽量了解自己之后，就会一步步具备安放情绪的能力，这是情商中自我管理非常重要的一部分。

情绪本身没有正面、负面之分，只有我们看待事情的态度有正负之分。当我们改变了看待事物的态度，自然而然就成为了情绪的主人。

一个心理学测试证明了这一点：观看引发愤怒电影的人，会冲动地批评政策；而观看引发悲伤电影的人，则会安静地逐条分析，给出理性评价。这表明，悲伤能引导人们寻找出路和答案。

当我们遇到挫折时，不要连续追问“为什么是我？”，这只是在发泄情绪。应该试着问自己“怎么办？”。当你问“怎么办”时，你的初心是在考虑如何行动，通过行动就能带来改变和希望。

## 理解痛苦的四个层面

既然讲到情绪管理，核心是教会大家如何从痛苦的情绪中解脱。要解脱，首先要理解痛苦的几个层面。

以下是痛苦的四个层面：

1.  **物质层面**：例如缺钱、羡慕他人收入高，产生落差和痛苦。
2.  **身体层面**：例如觉得自己长得没别人好看、没别人健康，体弱多病，从而痛苦。
3.  **头脑层面**：例如思考问题想得头痛，却找不到答案。
4.  **精神层面**：例如在家庭、恋爱等亲密关系中，感受到精神层面的痛苦。

什么是痛苦？**痛苦就是一切负面情绪的感受**。更深层次的理解是：苦是一种恩典。因为有这种苦，它能够让你经历、成长和觉醒。

## 痛苦的五种人与来源

痛苦的人分为五种：

1.  完全不知道自己在痛苦。
2.  知道自己在受苦，还以受苦为荣（例如，信奉“吃得苦中苦，方为人上人”）。
3.  知道自己在受苦，并以受苦为习惯。
4.  知道自己在受苦，想寻求解脱但找不到方法。
5.  找到解决痛苦的方法，过着舒服的生活。

那么，痛苦从哪里来？主要有四个来源：

1.  **自我中心，自私自利**：什么都只考虑自己。
2.  **因为恐惧和害怕**：要敢于直面内心，问自己“我到底在害怕什么？”。
3.  **不够爱自己，不自爱**：典型反应是觉得自己不够好、不值得、不配拥有。
4.  **很执着**：例如，非常喜欢一个人，总想得到或占有，对方反应不好时内心就波澜起伏。

## 痛苦的表现与解决方法

产生痛苦后，通常会有以下表现：

1.  没有智慧，说错话，做错事。
2.  持续伤害自己或他人，破坏关系。
3.  持续制造更多痛苦（例如，借酒消愁）。
4.  不会感恩，无法享受现已拥有的东西。
5.  封闭自己，与外界断联。极端情况下，可能毁灭一切。

如何解决痛苦？以下是具体步骤：

1.  **觉察与承认**：首先要知道并承认自己正处于痛苦的状态。
2.  **暂停与释放**：停止行动、停止说话、不要做任何决定。如果有悲伤或愤怒，要去释放它，不要压抑。哭相当于给灵魂洗澡，是一种健康的情绪释放。
3.  **自我对话与臣服**：可以与自己内心对话（例如，给内心起个名字）。当你去臣服、求助时，内心会给你启发。**核心是相信自己的内心**，而不是向外求助于算命等。
4.  **转变观念**：认识到生活本身没有痛苦，一切的痛苦都是自己想象和制造出来的。一念之间，天堂与地狱。

本节课中我们一起学习了情绪管理的基础。我们理解了情绪如何主导我们的信念和行为，学习了通过共情应对他人情绪，认识了痛苦的多个层面和来源，并掌握了解决痛苦的四步法：觉察、暂停、自我对话和观念转变。记住，管理情绪的第一步永远是**先处理心情，再处理事情**。
