上一节我们明确了问题可能出在内核而非技巧，本节中我们来分析第一个核心问题：为何身边没有姑娘。

大家有没有考虑过这个问题？你想一下自己身边为什么会没有姑娘。

或者说，你有姑娘却没有把握好。那是什么原因导致的呢？

造成你追不到姑娘的原因，其实是你承受了很多压力。

以下是这些压力的具体分类：

1.  **经济压力**：没钱。最现实的问题是没钱。很多学员一开始就跟我说：“老师，我没有钱，我怎么样去把妹？” 没有钱怎么谈恋爱？这样的压力给你带来了阻碍，让你可能觉得约会不如在家躺着，因为在家躺着不用花钱。
2.  **学习压力**：很多人是学生，面临考研或高考等问题。工作后也会面临学习压力，比如想要升职，需要考各种试、考各种级。这些都会给自己造成压力，导致没有时间去跟女生接触。
3.  **交往压力**：有些人天生性格内向，不敢或不愿意跟异性交往。这可能源于家庭环境等因素。跟别人聊天社交是一件非常有压力、很困难的事情，成为一种负担。
4.  **世俗观念的压力**：指的是那种传统的观念。例如，父母认为纹身、打耳钉的人是流氓、小混混。从小父母和老师告诉我们“男女授受不亲”，不要轻易搭讪，外面的坏人很多。这种观念会给你造成心理负担。身份也会带来压力，例如警察可能觉得搭讪是耍流氓，老师可能觉得师生恋有问题。过年时被催婚也是世俗观念的压力。

上一节我们分析了导致身边缺乏异性资源的四种压力，本节中我们来看看这些压力会引发哪些负面心理状态。

这些压力会导致你产生以下几种心理状态：

1.  **焦虑心理**：想要马上改变，立刻找到女朋友。这种焦虑心理会阻碍你冷静地分析和思考，让你无法沉下心来学习，总想立刻得到结果，反而增加了阻碍。
2.  **自卑心理**：在成长过程中，很少受到鼓励，甚至经常被父母或他人打压。这导致你害怕失败，没有信心去完成任何事情，包括追求女生。
3.  **闭锁心理**：因为外界压力太多，你宁愿自己待着，减少与人沟通，把自己封闭起来。这是一种逃避心理，因为你觉得沟通意味着承受更多压力和不理解。
4.  **抑郁心理**：不愿意社交，整天郁郁寡欢，觉得生活没有意义，看什么都灰暗。在当今社会压力下，很多人多多少少都有点抑郁倾向。
5.  **负疚心理**：例如，觉得自己年纪大了没娶到老婆，对不起父母的养育之恩。特别是在被逼婚或觉得自己挣钱不够多时，这种愧疚感非常沉重。

上一节我们了解了压力导致的五种负面心理，本节中我们来探讨这些心理问题的根源。

为什么会有这样的心理？除了压力之外，还有以下三种环境因素：

1.  **生长环境（原生家庭）**：你的家庭环境直接影响你现在的状态。例如，父母离异、保守的教育方式等。
2.  **学校/社会环境**：在学校或社会环境中经历的事情也会塑造你的心理。
3.  **一次惨痛的恋爱经历**：一次失败或受伤的恋爱经历，可能会让你长时间不相信爱情，害怕再次开始。

上一节我们探讨了心理问题的根源，本节中我们来看看如何打破没有姑娘的困境，并引入核心概念——自尊。

如何打破没有姑娘的困境？关键在于**自尊**。

今天这是非常重点的内容。我希望大家做好笔记。这是学习恋爱过程中一个很重要的核心，也是我把它作为第一节课来讲的原因。我需要大家认清自己现在缺的是什么。

你缺的是自尊心。

首先，大家来回答这几个问题：
*   你认为自己现在是一个有价值的人吗？
*   你觉得自己拥有很多好的品质吗？
*   你对自己的生活满意吗？
*   你对自己持肯定态度吗？

如果所有问题你都回答“是”，那么恭喜你，你是个**高自尊**的人。高自尊的人可以通过技巧方法获得进一步提升。

如果其中有一点没有满足，那么你可能是一个**低自尊**的人。低自尊的人在学习方法和技巧时，很难进步，因为会被自己的不自信打败。

**公式：自尊水平 = 对自我价值的整体感受**

这就好比游戏里，我有5套神装给你，但发现你自己的等级不足，穿不上去。这不是方法技巧的问题，而是你本身个人的问题，是你内在的问题。

所以，为什么有些兄弟学了一个月就找到女朋友，有些学了三个月、六个月还是没有？老师教的方法是一样的。这就是问题的本质和根本，并非方法技巧能解决你所有的疑惑和困扰。

如果你发现学了再多技巧，还是很困难、很吃力，那就是你人的问题本身：你是一个低自尊的人，不是一个高自尊的人。

上一节我们引入了自尊的概念并区分了高/低自尊，本节中我们来看看这两种状态的不同表现和循环模式。

高自尊的人对自己是肯定的。他们会处于一个良性的循环（类似飞轮效应）。由于在过程中相处良好，姑娘也会给他良性反馈，使得他不断提升自尊。优秀的人会变得越来越好，是因为他们走上了一个高自尊的轨道。

低自尊则表现为一个恶性循环。低自尊的表现有以下几点：

1.  **自暴自弃**：经常讨厌自己，觉得这也不行，那也不行。例如，总想着是不是要整容，要拍豪车，要请模特拍照。
2.  **自怨自艾**：经常唉声叹气，对生活充满失望，整天觉得自己不行。
3.  **自轻自贱**：总觉得自己不如别人，因为条件不好、没钱、长得不帅、身高等原因。这会进一步伤害自己，甚至让自己看不起自己。

我曾经有个学员，他是个公务员，工作优秀，有车有房，但30岁了从未谈过恋爱。他能量很低，说话小声，不敢抬头。他每天下班就回家看视频，周末睡觉，过年都不敢回家。他当时甚至有过自杀的念头。这就是低自尊的典型表现，最终会严重伤害自己。

上一节我们对比了高/低自尊的循环，本节中我们来深入理解“自尊”作为心理资本的重要性。

自尊心是**心理资本**。很多人认为把妹最重要的是有钱、有车、有房。我承认，物质条件确实能带来好处，但这只是物质资本。

你还需要**心理资本**。就像我，在成都也没车没房，还不是一样跟女生接触、聊天。我不会因为物质条件不足而自卑，反而觉得物质可以慢慢积攒。我坚信“厚积薄发”。我给自己心理资本是非常丰富的。

心理资本的优势，决定了我们的心理世界是富裕的还是贫穷的，是幸福的还是不幸的。心理资本不仅给你能量，还给你信心。

**核心观点：你的心理资本比任何事情都重要。** 心理资本才是成就你成功的重要东西。其他所谓的技巧、套路、聊天方法都会过时（看看15年到19年的变化就知道了）。时代在变，女生在变，技巧也在变，但不变的是我们的“心”。这个才是你要真正去懂得、运用和成长的东西，才是让你在这条路上越走越远的东西——心理资本。

上一节我们确立了心理资本的核心地位，本节中我们来看看如何具体提升自尊心。

如何提高自尊心？这里有一个关键公式：

**公式：自尊 = 成功 / 抱负（即：自尊水平与成功成正比，与抱负水平成反比）**

意思是，自尊心不仅取决于你获得了多少成功，也与你设定的目标（抱负）大小成反比。因此，在提升自尊心时，一定要**把目标降低**，你的自尊心才能提高。

很多人好高骛远，目标定得太高（例如“我要成为把妹达人”）。目标定得太大，在完成路上每一步都会很艰难，让你觉得离目标越来越远，怎么会有高自尊？

你应该把大目标拆分成小目标。例如，不要一上来就定“我一定要找到个女朋友”。应该先想：先把自己形象弄好，然后去认识更多女生，再把每个女生聊好，然后约会，最后才确立关系。把目标拆分成每一步。

小目标能促进你的自信心和自尊，让你在这条路上更有信心地走下去。

**举例1：搭讪**
很多人觉得上去要微信号很难。你可以分两步：第一步，先上去跟陌生人说句话（完成一个小目标）；第二步，再要微信号。心理负担会少很多。或者，先从你觉得难度较低的姑娘开始搭讪，而不是一上来就挑战最高分的。

**举例2：课程设计**
我们的课程都先从外在形象改变开始。因为外在吸引是个相对容易达成的小目标（换衣服、换发型），而内在吸引是长期积累的大目标。先完成小目标（形象改变），你的信心增加了，再去进行下一步（比如搭讪、聊天）就更容易。当你有30个、50个姑娘的联系方式时，你的自信心会提高，因为你不怕聊死一个就没得聊了。

这种小目标的完成，会慢慢积累你的自信心，就像积累经验值一样，最终实现升级。这个方法不仅适用于恋爱，也适用于工作和生活。

上一节我们学习了通过降低目标来提升自尊的具体方法，本节中我们来看看自尊的“自我保护机制”及其影响。

自尊心还存在一个**自我保护的机制**。它会阻碍你踏出舒适区，以保护你不受伤害。

**表现包括：**
1.  **搭讪时**：内心声音说“这个姑娘走太快了，要不下次吧”、“她长得太好，我驾驭不住”。这是自尊心在保护你，害怕被拒绝。
2.  **推进关系时**：明明已经牵手了，但不敢去升级关系。是因为自尊心在保护你，害怕被女生拒绝。
3.  **表达自己时**：面对喜欢的女生不敢表达，害怕失去她。
4.  **处在舒适区**：每天躺在床上不愿意动，觉得现在很舒服。自尊心告诉你，踏出去会很困难、痛苦。

处在舒适区，就是自尊心的一个自我保护机制。它在保护你不受到外界的讽刺、抨击和伤害。

自尊是一把双刃剑。你要合理利用它给你带来的强大，也要认清它现在在保护你什么。如果你明知道这件事可以再进一步，那就要**突破这个自我保护的机制**。突破了，你才能进入下一个成长阶段。
