## 第一节：态度基石——资源富足

上一节我们提到了情感富足的基础。本节中我们来看看如何实现它。许多男生情感经历寡淡，根本原因在于周围缺乏足够的潜在交往对象。

**什么是“足够多”的概念？**
一个基本的标准是：你的通讯录中至少应有**300个**符合条件的单身女性。你可能会说自己已经有一百多个联系人，但其中可能包含已恋爱的、外形你不满意的、或对你无兴趣的。经过层层筛选，真正可发展的可能寥寥无几。当这个基数扩大到300个时，你的心态会发生根本变化：你会感到自信、从容，态度自然建立。

如果你目前还没有足够多的资源，请回顾我们前几集课程，学习如何使用社交软件（如探探、陌陌）或街头搭讪来认识女性。如果不愿执行这一步，后续的态度心法将难以生效，因为**没有资源，就无从建立态度**。

## 第二节：2017年三大核心态度心法

回顾2014年，我们提出的态度包括真实、尊重、理解和提供价值等。在2017年，我们在此基础上新增了三大心法。

### 1. 舍得 🪓

“舍得”指的是，当一个女生对你反应冷淡时，你要学会战略性放弃。

**具体做法：**
当女生在对话中频繁回复“哦”、“嗯”等冷淡词语时，你的第一反应不应是“如何用话术吸引她”，而应是“她可能对我不感兴趣”。此时，你应该采取对等或更冷淡的回应，例如也回复一个“哦”、“嗯”，或者暂时不回复。

**核心公式：**
```
女生兴趣低 -> 你的投入应减少或停止
```
有舍才有得。只有舍弃那些消耗你时间、对你无意的女生，你才能腾出精力去接触真正属于你的缘分。

### 2. 接受失败 🛡️

“接受失败”是指，你要明白自己不是人民币，不可能让每个女生都喜欢。

**具体做法：**
当女生拒绝你或给出负面反馈时，坦然接受这个结果。这可能是你的问题，也可能是对方的问题。不要对失败耿耿于怀，也不要过分在意女生对你的负面评价。即使是我，也经常收到女生的不佳反馈。接受失败和不完美，是一个男人成熟的标志。将你的重心放在那些喜欢你的人身上。

### 3. 淡定 🧘

“淡定”意味着，你不因女生的应约而狂喜，也不因被拒绝而深陷失望。

**具体做法：**
学会控制自己的情绪，保持情绪稳定。这种淡定源于对局面的掌控感。当你明白女生是否出来约会，并非完全由你决定，或者你有足够多的成功经验时，你自然会变得淡定。

**如何培养淡定？**
以下是三个关键点：
1.  **拥有大量妹子资源**：每天有不同女生可聊，不会对单一对象过度关注。
2.  **拥有成功邀约经验**：约出过很多女生后，你会知道有人拒绝是正常现象，不会归咎于自己。
3.  **拥有丰富约会经验**：了解女生的思维和行为模式后，你会因熟悉而不再紧张，因不再“未知”而保持淡定。
