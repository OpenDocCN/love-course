# 老吴《聊天课堂》第2.2课：如何制作专属微信表情 😎

在本节课中，我们将要学习一个非常实用的技能：如何利用一款名为“逗图”的App，制作属于你自己的、个性化的微信表情。我们将从下载软件开始，一步步教你如何选择模板、编辑文字、保存图片，并最终将其添加到微信表情包中，用于日常聊天。

---

## 软件准备与表情选择

上一节我们介绍了聊天中表情的重要性，本节中我们来看看如何亲手制作它们。首先，你需要一个核心工具。

制作表情，我们需要使用一个名为 **“逗图”** 的App（也被称为微信逗图神器）。你可以在手机应用商店搜索并下载它。

打开这款软件后，你会发现一个海量的表情模板库。里面有无数现成的、流行的表情图片可供选择。

以下是选择模板的步骤：
1.  浏览软件中的各类表情，找到你喜欢或觉得有趣的一个。
2.  点击选中该表情模板，进入编辑页面。

## 编辑与自定义文字

选择了合适的模板后，接下来就是赋予它个人色彩的关键一步：编辑文字。

在编辑页面，你可以修改模板上原有的文字。例如，你可以将“哈喽”改成任何你想说的话，比如“你好，我是[你的名字]”。

编辑时请注意：
*   你可以修改文字内容。
*   软件通常提供调整字体颜色的选项，但**白底黑字**的经典搭配通常效果最清晰、最好。
*   编辑完成后，点击页面右上角的 **“发送”** 按钮。

## 保存图片到本地相册

编辑好文字并点击“发送”后，我们需要将成品保存到手机里。

点击“发送”后，软件会弹出一个对话框。在这个对话框中，务必选择 **“保存到本地”** 或类似选项。

操作成功后，这张你亲手制作的表情图片就已经保存到你的手机相册（或“相机胶卷”）中了。你现在可以随时在相册里找到它。

## 将表情添加到微信表情包

图片已经准备好，最后一步就是把它“安装”到微信里，方便随时使用。

以下是添加到微信表情包的步骤：
1.  打开微信，点击右下角的 **“我”**。
2.  进入 **“表情”** 选项。
3.  在表情商店页面，点击右上角的**齿轮（设置）图标**。
4.  进入 **“我收藏的表情”** 页面。
5.  点击页面中的 **“+”号**。
6.  在弹出的相册选择中，找到你刚刚保存的那张自制表情图片，选中并确定。

至此，这个专属表情就已经成功添加到你的微信个人表情包集合里了。你可以在任何聊天窗口的表情面板中找到并使用它。

## 表情的进阶玩法与运用思路

掌握了基本制作方法后，我们可以进一步探索它的创意用法，让聊天更有趣。

你可以在“逗图”App中探索不同的分类，比如“挑衅”、“可爱”、“熊猫头”等，找到更多风格各异的模板。结合不同的文字，能产生非常生动的效果。

以下是一些实用的运用思路：
*   **制作专属打招呼表情**：为特定的聊天对象制作一个带有她/他名字的可爱表情作为开场，会让人感觉非常特别和用心。例如：“Hello，[对方昵称]baby”。
*   **引导对方投资**：你可以为对方制作一个专属表情后，用开玩笑的方式“索要回报”，例如：“亲我一下，就给你做个更可爱的”。这能在互动中自然地加入调情元素。
*   **增强聊天表现力**：利用搞笑或传神的模板配上合适的文字，可以更准确地表达你的情绪和态度，让文字聊天不再枯燥。

本节课中，我们一起学习了从零开始制作微信专属表情的完整流程：**使用“逗图”App选择模板 -> 编辑自定义文字 -> 保存图片到本地 -> 添加到微信收藏表情**。更重要的是，我们探讨了如何将这种自制表情作为一种有趣的社交工具，用于打造独特的开场白、增强聊天互动以及营造轻松氛围。多加练习和创意发挥，你的表情包将成为你聊天中的一大亮点。
