# 恋爱军团男神养成计划：第五课 👔 服装挑选与衣柜整理

在本节课中，我们将学习如何通过整理衣柜和挑选服装，来提升个人形象，向“男神”目标迈进。核心在于理解“衣服在精不在多”，并学会果断舍弃不适合自己的衣物。

前几天课程，大家的作业有坚持打卡。我们和男神的距离已经越来越近，只有坚持才能看到一个焕然一新的自己。

老师想起很久之前看过一部黑色喜剧。其中有一个非常有意思的桥段。开着豪华跑车的男主，趾高气昂地跟大家指点江山说，如果不把衣柜里的旧衣服都丢掉，新衣服是进不来的。周围的人听到这句话，瞬间醍醐灌顶，马上回家收拾自己的衣柜，一股脑地把不要的衣服全部丢掉了。没料到不出几天，其中一个人惊讶地发现，男主身上的T恤和他前几天丢掉的那件竟然一模一样。

虽然这是一个充满讽刺意味的桥段，但是你是不是也有这种别人不刺激你，你就不会去处理旧衣服的习惯呢？所以，衣物整理时，不再心动的衣服就赶紧丢掉。

或许你的衣柜已经满到再也塞不进一件衣服。但是你有没有发现，其实这些衣服中经常穿的也就是那么屈指可数的几件。每天早晨，你是不是都会在衣柜前犹豫好久，为不知道穿什么衣服出门而愁眉苦脸，皱着眉头感慨自己没有衣服穿。令人恼火的起床气和时间紧迫的慌乱已经让你分分钟想爆粗口。为了一整天的心情，是时候做点什么了。

打开你的衣柜好好规划一下，仔细想想你平时购买衣服的那些习惯。然后慢慢地，你期待自己以后要以怎样一个帅气的形象在别人面前展示自己呢？其实，迈出这一步，你和男神之间的距离就没有那么遥远了。

你有没有发现呢？当你的衣柜整整齐齐，一目了然的时候，你就很容易灵光一现，搭配好出门的行头。所以，整理衣服你首先要学会这一条诀窍：**保留几件高品质的、你经常穿的、并且符合你风格的衣服**。至于那些既没有价值，又浪费空间的衣服，还是赶紧把它们处理掉。

是的，衣服再精不在多。不要纠结你要扔掉的衣服是不是今年新推出的明星爆款。也不要纠结你是不是当初为了买它，省吃俭用过很长一段时间的苦日子。不要纠结你过去是不是经常穿着这件衣服和心仪的人出去约会，很有纪念意义，所以舍不得丢掉。一件衣服只要你穿着不好看，不适合你，你就没有理由让它继续占着你的衣柜。

下面开始划重点。以下是需要扔掉的衣物类型：

**第一，不合身的、破旧的、不符合自己风格的都该通通扔掉。**

不合身的衣服大致可以分为以下几类。这些衣服可能是你瘦的时候或者胖的时候经常穿的衣服，也可能是别人送的买错尺码的衣服。还有一种情况就是衣服本身因为穿着洗涤次数过多，缩水或失去弹性而变得松松垮垮。太大的衣服穿在身上会显得很邋遢，太小的衣服又很滑稽。所以这类衣服都要尽快从你的衣柜里请出去。

那再来找一下破旧的衣服。不小心刮破蹭破的衣服，因为穿太久发黄洗不干净的衣服，油渍血渍清洗不掉的衣服，或者褶皱熨烫不平的衣服，穿太久褪色失去原有光泽的衣服，纽扣、拉链、装饰物丢失或者损坏无法修好的衣服。

最后是不符合自己风格的衣服。要判断衣服是否符合你的风格，你首先要对自己有一个清晰的定位。你要知道你是谁，你想要在什么场合对谁留下一个什么样的印象，达到什么样的效果。基于这些再来判断哪些衣服符合你的风格。接下来你还要根据自己的偏好，结合自身气质确定你的风格。不符合你风格的衣服，其实基本上你是很少把它们拿出来穿的，你自己其实很容易就能做出判断。

扔衣服的时候不要说舍不得。不要因为你对某件衣服怀有感情，就让它霸占着你的衣柜。更不要因为有些衣服你在买它的时候下了血本，就把它们留下来。也不要因为你觉得自己还有变瘦的可能，就把它们留下来。也不要因为一些衣服虽然不适合你，但你认为衣服本身很好看，就把它们留下来。请把它们立即处理掉。

如果你三年没碰过这些衣服，那么你就没有多大可能再穿它。你衣柜里的衣服应该都是你穿起来显得很精神，穿起来让你心情舒畅的衣服。把旧衣服当作居家服最后实行的可能性几乎为零。

如果你实在犹豫再三没有办法做出抉择的话，你就要换个方式。想想哪些衣服是你经常穿的，受过别人由衷赞美的，你想要珍惜重视的，必须要保留的。最简单的方法就是把它们先挑选出来，一件件试穿，看着镜子里的自己，问：这件衣服是不是好看，会不会提升你的幸福感。

某种程度上，整理衣服是一个你和自己敞开心扉，深入交流的过程。你开始认真定义自己的风格，深入了解自己的审美品位，并且对自己以后的购买会做一个规划。跟已经完成使命尽完责任的衣服，愉快地挥手告别吧。接下来，你的生活就要翻开崭新的篇章。

上一节我们介绍了如何筛选和舍弃衣物，本节中我们来看看购买新衣时需要注意的原则。老师这里也有买衣服的小贴士。

以下是三条核心的购衣原则：
1.  **剪裁 > 材质 > 款式**。合身的剪裁是第一位，好的材质能提升质感，最后才考虑款式是否流行。
2.  **有型是首要的，舒适是次要的**。这里“有型”指衣服能修饰你的身形，让你看起来挺拔精神。在保证有型的基础上追求舒适。
3.  **避免大的Logo**。过于显眼的品牌标识会显得浮夸，简约的设计更能体现品味。
