## 第150讲：感情的四个阶段 💑

在本节课中，我们将要学习心理学家提出的成熟恋情必经的四个阶段，并探讨如何在这些阶段中建立和维护健康的亲密关系。

上一节我们介绍了建立吸引力的基础，本节中我们来看看一段感情是如何发展和成熟的。

一位心理学家曾说，一个成熟的、称得上是真爱的恋情必须经过四个阶段。这四个阶段分别是：共存、反依赖、独立以及共生。

**第一阶段：共存（热恋期）**。这是热恋时期，双方不论何时何地都希望能与对方腻在一起。

**第二阶段：反依赖（稳定期）**。当感情比较稳定后，其中一方可能会想多花时间做自己想做的事，而另一方则可能感到被冷落。

**第三阶段：独立（自主期）**。双方都会要求更多独立自主的时间。

**第四阶段：共生（成熟期）**。双方共同达到一种新的相处之道，成为最亲密的人，在一起相互扶持、共同开创人生。此时双方不会觉得互相牵绊，而会感到能给予对方很大的帮助，促进共同成长。

但是事实上，很多人都是没有办法通过第二或第三阶段的，就选择了分手。其实很多事情只要好好沟通就会没事，但我们不可避免地会想太多，并且缺乏一点韧性。世界上遇到唯一的你（伴侣）是很不容易的，所以不要轻言放弃。

不知道你们有没有发现，其实两个人本来没有什么相同之处，外表不一样，性格也南辕北辙。但是因为在一起之后，日复一日，年复一年，你会慢慢觉得两个人变得更加相似了。不管是你们的眼神，还是你们的微笑，都会变得越发相似。

以下是关系深化后，双方可能发生的变化列表：
*   走路的步伐、说话的语气等会变得一样。
*   会养成一些共同的小习惯，比如爱喝同一杯饮料，有一样的口头禅。
*   甚至会猜到对方下面会说什么。

我们都会变成我们所爱的人。你在不知不觉中，也会为另外一个人改掉一些不好的习惯。这样的改变或许你们自己都不曾察觉，但在不知不觉中，你们就变了，由原来的初心变得非常细致体贴。如果你回忆很多年前的自己，可能都有点认不出现在的自己了。这就是我们在不知不觉中，成为了对方理想中的人。这种改变并不是刻意的，是两个人相处时间变久之后，气质自然而然地相近了。有一天你就会惊讶地发现，你们是非常默契的。或许再也不会遇到这样的感情了。

其实当你深爱一个人的时候，你是真的会一点一点去失去自己，但是你为什么还会觉得快乐呢？大概就是因为你在失去的同时也转了，因为你转到了他的气质，他的微笑。

那么，对于男生来说，恋爱这件事情应该是“工欲善其事，必先利其器”的。因此，对于男生来说，首先你肯定是要注重自己的形象，给女生留下一个美好的印象。

*   **补充恋爱养分**：其次是要多学习，要补充恋爱的养分。不光是要学习你的知识、技能、本领，也要学习爱的能力。知识不管在什么领域都是一种力量。所以要让自己变得有吸引力，你必须吸收更多有关男女互动的知识，对于不同的心仪女孩使出不同的招数。

*   想喝什么
*   常去什么地方玩
