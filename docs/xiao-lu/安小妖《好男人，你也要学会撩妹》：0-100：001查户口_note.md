# 安小妖《好男人，你也要学会撩妹》课程0-100：001 查户口式聊天 🚫

在本节课中，我们将要学习如何避免在聊天初期与女生进行“查户口”式的对话。我们将了解什么是查户口、它为何有害，并掌握几种更自然、更有效的沟通方法来获取信息并建立良好印象。

## 什么是“查户口”式聊天？

“查户口”这个词最早从QQ时代开始兴起。它指的是在聊天中不停地向女生提问的行为。

无论是在社交软件、电话还是约会时，都可能出现这种情况。

例如，男生问：“你是哪里人？”女生回答：“我是山西人。”男生接着问：“你是山西哪里的？”这就是典型的查户口。

又或者问：“你是做什么工作的？”女生说：“我是做会计的。”男生又问：“你在哪边做会计？”女生回答后，男生继续问：“你平时喜欢做什么？”这也是非常典型的查户口。

**查户口**就是不停地提问，尤其是在关系前期。这是一个非常糟糕的行为，极易降低女生对你的社交评价。说白了，这会让女生觉得你是一个很没有价值、很无聊的人。

## 为何前期聊天“容错率”低？

在前期跟女生聊天时，容错率非常低。

**容错率**就是你前期可以犯错的次数。与朋友聊天时，容错率很高，犯一两个错误无伤大雅，因为你们已经认识并有交集。

但刚认识一个女生时，前期聊天的容错率非常低。你犯一两个错，女生就可能不再理你，你们甚至可能再也见不到面。

查户口就是一个非常严重的错误。如果在前期使用这种方式聊天，那么你与这个女生再见面的几率将变得很低。因此，我们必须避免查户口式的聊天。

## 为何会产生查户口式聊天？

以下是两个主要原因：

**1. 不知道聊什么**
有些男生不知道与女生聊什么话题，但又害怕冷场尴尬。因此，他们会在聊天中不停地提问，认为只要不停提问就能避免冷场。

**2. 需求感过强**
**需求感**指的是这个男生非常想得到女生的注意力，想让女生喜欢上他。可能因为他身边女生少，好不容易遇到一个漂亮可爱的女生，生怕错过。于是，他前期就不停提问，想了解女生的个人信息、工作和兴趣爱好。

然而，在前期过多提问，反而会给女生留下不好的印象，非常扣分。

## 如何避免查户口并自然获取信息？

问问题是最弱的沟通方式，因为它需要女生单方面提供答案。我推荐大家使用**陈述**来获取女生的信息。

**陈述**是表达你自己的观点，而非直接提问。

例如，想知道女生是哪里人：
*   **提问**：“你是哪里人呀？”
*   **陈述**：“听你的口音好像有点像南方人。”

当你陈述一个观点（如“你听起来像南方人”），人们通常有纠错心理。如果她不是南方人，可能会说：“不是啊，我是北京人。”这样，她自己就把答案说出来了。如果她确实是南方人，可能会说：“你听得好准哦。”通过陈述，你就能合理、自然地获得信息。

再比如，想知道女生的职业：
*   **提问**：“你是做什么工作的呀？”
*   **陈述**：“我觉得看你的气质好像有一点点像做媒体工作的。”

这时，女生可能会说：“是吗？你怎么看出来的？”或者说：“不是啊，我不是做媒体的，我是做会计的。”当你给出一个错误答案时，女生反而会想纠正你。这就是陈述的作用：让对方主动纠正并说出个人信息。

## 前期可用陈述获取信息的核心话题

以下是几个前期可以通过陈述自然展开的核心话题：

**1. 籍贯（你是哪里人）**
使用陈述句猜测对方的家乡。

**2. 职业（你是做什么工作的）**
通过观察对方气质进行猜测。

**3. 年龄**
在前期直接询问年龄非常不礼貌，但用陈述句可以很自然地获得信息，且不失风度。

例如，你可以说：“我觉得你看起来像94年的。”如果女生像90年的，你应该往小了猜。因为往小了猜说明她外表显年轻，女生听到会开心，反而可能主动告诉你真实年龄：“哈哈不是啊，我都90年了。”

当然，也可以用开玩笑的方式。例如，看着女生说：“你看起来好小啊，你是18岁吗？”女生知道这是玩笑，但听到也会开心，能让聊天氛围更好。

对于工作，也可以用开玩笑的方式。例如：“我觉得你的气质啊，有一点像幼儿园老师。”当女生问为什么时，你可以说：“因为我觉得你的气质比较随和，而且跟小朋友的关系应该比较好。”这其实是在夸赞对方，能让对话更愉快。

## 可以提问，但切忌连续发问

前期并非完全不能提问，重点在于**不要连续发问**。一个问题接一个问题就变成了查户口，但只问一个问题则不是。

我推荐使用**基本信息交换**的聊法：先说自己，再问对方；或者先问对方，再说自己。有互动的聊天才是最好的。

例如：
*   **先说自己，再问对方**：“我是做互联网工作的，平时工作也比较忙。哎，你是做什么工作呢？”
*   **先问对方，再说自己**：“你是哪里人啊？”（女生回答后）“但我听你的口音好像没有什么山西口音。对了，我是北京人，但我北京口音比较弱一些……”

这种方式是自然的**信息交换**，而非单方面索取。一直提问是在索取，一直说自己也不是好的聊天方式。

## 总结与下节预告

本节课中，我们一起学习了如何避免“查户口”式聊天。

我们了解到，查户口是在前期不停提问的糟糕行为，会降低你的社交评价。其产生原因主要是不知道聊什么或需求感过强。

我们掌握了更有效的沟通方法：多用**陈述句**来表达观点，引导对方主动提供信息；可以提问，但应使用**信息交换**的方式，避免连续发问；还可以用轻松、开玩笑的方式营造愉快氛围。

在前期，陌生人之间更倾向于了解对方个人信息，因此聊个人信息是一个很好的话题起点。希望大家下次聊天时，不再查户口，而是多用陈述、信息交换或轻松愉快的方式，给女生留下良好印象，为后续发展打下基础。
