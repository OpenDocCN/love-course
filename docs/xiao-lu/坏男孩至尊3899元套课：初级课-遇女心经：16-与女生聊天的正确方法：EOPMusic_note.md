## 聊天注意事项  

### 1. 避免查户口式提问  
尽量少用疑问句，多用陈述句。例如，想问年龄时可以说：“你看上去不到20岁吧？”  

### 2. 一句话表达一个意思  
避免一句话包含多个信息点。例如，直接问“你在干嘛？”而不是“我吃过饭了，晚上去酒吧，你在干嘛？”  

### 3. 少用语气词  
语气词如“啊”“哦”会显得优柔寡断。例如，邀约时说“晚上出来”而不是“晚上出来吧”。  

### 4. 控制消息长度  
遵循等量原则，回复消息的字数尽量与女生相当。避免长篇大论，需求感过强会降低吸引力。  

**公式**：`回复字数 ≈ 女生消息字数`
