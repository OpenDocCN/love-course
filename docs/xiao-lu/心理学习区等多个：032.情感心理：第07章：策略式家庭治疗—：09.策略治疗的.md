# 心理学习区等多个：032.情感心理：第07章：策略式家庭治疗—：09.策略治疗的

在本节课中，我们将要学习策略式家庭治疗中的一项核心技术——家庭作业。我们将探讨其背后的哲学理念、布置原则以及几种常见且实用的家庭作业技术。

## 概述：家庭作业的哲学基础

上一节我们介绍了策略式家庭治疗的基本框架，本节中我们来看看其核心干预手段之一：家庭作业。在讲解具体技术前，首先需要理解家庭作业背后的哲学思考，这涉及到策略家庭治疗和系统治疗对于“改变”的基本看法。

传统观念可能认为，心理治疗的关键改变是发生在治疗室当中的。持有这一观点的最经典治疗是精神动力学治疗。精神动力学治疗的治疗频率会比较高，一般可能一周两到三次，高一点的话可能每天一次。为什么会采用这样的设置呢？因为精神动力学治疗认为，患者每一个细微的无意识感受和认知的改变，是需要在治疗当中通过和治疗师的每一个细节的对话，一点一点去积累的。

但是家庭治疗对于改变的观念不一样。家庭治疗里面有一句经典的话，那就是认为，关键的改变不是发生在治疗室里面，而是发生在生活当中。

为什么家庭治疗会这样认为呢？因为家庭治疗有一个核心信念：家庭是一个完全不可能被人为控制的、有自主运行能力的系统。治疗师能做的事情就是在治疗当中，去对家庭这个系统运作的规则进行一些探讨，然后实施一些必要的干预。就好像治疗师是拿着一根棍子，这个家庭是一个水池，治疗师拿着棍子在这个水池里搅一搅，把这个水池原本的规则改变和扰动一下。那之后这个家庭在生活当中会变成什么样，那就让家庭自己去寻找适合他们的新方法。

为什么要这样做呢？举个例子，比如一对夫妻来做治疗，他们带来的问题是婚姻生活很不幸福，觉得彼此根本不是一路人，觉得需要改变。谈到“改变”这个词的时候，可能有的治疗师就会认为，改变就是你们和好如初，那就把治疗的目标定为让你们和好如初。但是这位治疗师可能会发现，他跟这个家庭谈得越深，越发现这对夫妻不是一路人。最后他可能就会怀疑，治疗的目标真的是要让你们和好如初吗？也许你们好聚好散也是一个选择。

这就提示我们，我们原本认为的合适的规则，在不同的家庭身上不一定适用。也就是说，虽然我们的理论描述很好，觉得和睦的家庭就应该是夫妻彼此支持，孩子独立地发展自己，但是这样的一种改变，并不是在所有的家庭里面都能达成的。所以这就对我们咨询师的哲学思想提出了一个本质的挑战：我们能做的工作有多少？我们是不是就一定要求家庭在咨询室里面就发生改变？

因此，系统治疗和策略治疗提出：我们治疗的目的不是让家庭在治疗室里面就马上变成我们希望的样子。我们只需要去给他们布置作业，提供必要的干预，他们拿着这些干预带来的影响，自己回去寻找他们新的改变的出路就可以了。而且家庭治疗师也相信，每个家庭都有自己改变的资源。家庭治疗师的目的，只是扰动和开启改变。

我曾经见过一个家庭，那个家庭刚来的时候，女儿看上去病得很重，17岁的小姑娘抽烟喝酒不上学，堵在学校门口欺负同校的女同学。爸爸妈妈都是很有教养、知识教育水平很高的国企干部和政府官员。爸爸就很焦虑，想不通女儿怎么会这样，也很绝望。最开始来找我做治疗的时候，还拼命跟我要微信和手机联系方式，我就坚持没有给他。我就跟他说，我知道你很着急，但是这个过程需要慢慢来，你们就按照我们的约定定期来见我就好了。有时候治疗期间，女儿的情况出现一些反复的时候，爸爸也会想方设法通过介绍人想找到我，但是我的答复永远都是：情况我知道了，下次见面的时候我们再讨论。然后也一再叮嘱那个家庭，在家里继续做我布置的作业。

后来你就会发现，当你真正尊重、相信那个家庭有改变的资源的时候，那个家庭的资源自己就出来了。后来那个爸爸想了个什么方法呢？他就找了学校里他女儿比较信任的几个高年级的学长，也不知道他用了什么办法，反正就是和那几个学长建立了很好的关系，然后请那几个学长站在不远不近的距离，去帮助和关心他的女儿。所以你看，每个家庭都是有他们自己改变的资源的。那个爸爸最开始想直接控制那个女儿，但是发现没用。但是你真正不去命令他的时候，那个家庭自己的资源会出现。

所以，正是基于这样一些相信家庭存在资源，也认为治疗师的作用不是要去教家庭应该怎么改变的哲学理念指导下，也是认为我们没有办法去命令家庭的这样一个治疗性哲学观的前提下，才有了家庭作业这样一个治疗方法的产生。

实际上，家庭作业最开始的起源是来源于认知行为治疗。家庭治疗的发展过程当中，因为也受到认知行为治疗的影响，所以说家庭治疗实际上是一个整合的心理治疗方式，基本上是以探讨家庭关系为核心，在这个过程当中，哪些方法有用的，我们就拿过来用。

## 布置家庭作业的原则

在了解了家庭作业的核心理念后，我们来看看布置家庭作业时需要遵循的几个基本原则。以下是布置家庭作业时需要牢记的几点：

1.  **资源取向**：家庭作业一般是资源取向的作业，你要去关注这个家庭的资源，不是用批评、攻击、指责的方式来布置作业。比如我们经常会布置家庭一个“记红账”的作业，让他们去记彼此的优点和在治疗间歇期的改变。我们很少去让他们记彼此的缺点，因为他们一般在家里已经记够了彼此的缺点了。
2.  **形象具体有趣**：这个家庭作业要比较形象、具体和有趣。你不能布置那种文绉绉的、了无生趣的家庭作业。
3.  **针对元沟通层面**：作业一般要针对家庭的元沟通层面来开具。元沟通大家应该还记得我们讲过。
4.  **基于评估**：家庭作业要建立在对家庭关系和规则评估的基础来布置。你不能什么都不知道，就给别人盲目地乱布置作业。
5.  **基于良好关系**：家庭作业要建立在和家庭比较良好的治疗关系的基础上。

## 常见的家庭作业技术

明确了原则，接下来我们具体看看策略式家庭治疗中常用的几种家庭作业技术。这些技术旨在以有趣、非对抗的方式扰动家庭系统。

### 1. 单双日作业 📅

单双日是一种行为作业。一般这种作业我们会用在那样家庭身上：家庭表现出一种明显固化的生活方式。比如爸爸妈妈面对16、17岁的青春期的孩子的时候，特别想控制那个孩子，但是那个孩子又特别反抗，爸爸妈妈也坚持，孩子也坚持自己的生活方式的时候，我们可以去布置这个作业。或者有时候我们讨论出来，比如这个“厌学”有一个好处可以维持父母的婚姻，那我们也不去跟这个父母说，也不跟孩子说你要继续上学还是不上学，我们布置一个单双日的作业，让这个家庭体验一下，在不同的时间里面，用不同的方式来生活，他们的感受会有什么不一样。

这个作业一般我们会怎么布置呢？比如我们会这样给这个家庭布置：刚才我说的那个要控制青春期孩子的父母，我们可能会说你们回去以后，周一到周日，其中周一到周六，1、3、5或者2、4、6，你们选三天（1、3、5也好，2、4、6也好）。这三天当中，爸爸妈妈你们对待你们的儿子，要像对待四、五岁的小宝宝那样对待他。他从早上起来一睁眼，你们就要开始帮他穿衣服，要问他你今天要吃什么饭，他不敢去哪里玩，你们都要跟着他，都要告诉他说你们很担心他，因为要为他的所有事负责。那余下的三天，比如1、3、5把他当小宝宝，2、4、6了，你们就把他真正的当做一个17岁的大男孩来看，一个大小伙子来看。同时你要跟那个大小伙子说，就说1、3、5的时候，你就要配合你的父母做一个小宝宝；2、4、6的时候，他们真的把你当一个大人的时候，你也要配合他们，把自己变成一个真正的大人，你要为自己未来的计划、为自己的学习、为自己每天的生活负责，你要去想自己的卫生，自己的衣服要自己洗等等。你就让他们去体会1、3、5和2、4、6，在这种不同的生活方式下面，他们的感觉、他们的心情会有什么不同。然后剩下一天星期天，你可以跟他们说，星期天呢，你们自己决定。如果这个小伙子你觉得当个小宝宝好，你就在星期天想尽办法表现得像个小孩子；你们父母呢，如果星期天觉得把他当小宝宝看，你们也什么都不用说，你们就想尽办法通过你们的行动，把他变成一个小宝宝。你们都不用过多的解释，你们只要行动就好。当然了，如果星期天这个小伙子你觉得当一个小宝宝太难受了，父母也觉得把儿子当一个小宝宝太憋屈了，那你们就把他当成一个大人看。所以这就是所谓的单双日作业。

这个作业有一个作用：通过完成这个作业，可以促使家庭对两种截然相反的生活方式进行体会和比较，进而帮助他们对自己原本固化的、不合理的行为产生自己的领悟，并且自行选择进步的方向。可能做完这个作业一对比，那个父母就觉得说，那比起把他当小宝宝，还是把他当大人的生活更轻松一点；那个孩子也会体会到，原来我的一些行为，也在配合我的父母把我当小宝宝来看。

### 2. 记红账 📖

记红账就是治疗师会要求参加治疗会谈的家庭成员，在回去以后各准备一个小笔记本，让他们在笔记本上面记，从这次见面结束到下次见面开始之间，他们看到彼此身上好的变化。并且我们会跟他们说，你们回去准备那个小本本准备记的时候，你们可以刻意地去提醒将要被记录的那个成员，可以跟他说：“你记住哦，刘医生不是给我作业要记录你好的地方，你注意一点。”就是只准记录他们好的地方，不能记录彼此身上坏的地方。

然后这样的作业完成以后，我们可以在下一次治疗开始的时候，让家庭成员把他们的作业带过来，然后我们在治疗里面和他们一条一条地去回顾这个作业的完成情况。你和他们的成员去回顾这个作业完成情况的过程，就是帮助他们的成员为互相点赞，帮助他们认可彼此的过程。

我曾经见过这样一对夫妻，那个丈夫到了40多岁以后，就莫名其妙地出现身体上的各种不舒服，去很多医院看了也没有任何的问题。然后那个丈夫也没办法上班，就每天在家里面当个家庭主夫。那个妻子对那个丈夫是一肚子的怨言，她就觉得这个丈夫又生病，然后又不赚钱，但是也正是觉得这个丈夫生病，那个妻子的怨气又没办法直接地表达，所以就带来做治疗。那带来做了第一次治疗以后，我布置的作业就是记红账，我就让这个妻子和丈夫各自记对方身上好的变化。治疗间歇是两周，让他们各自记对方身上20条好的变化。结果第二次来治疗的时候，我就当着那个丈夫和妻子的面，把那个妻子写的丈夫的好处一一念出来。结果那个丈夫听完了以后，就还是非常地感慨，他就说：“以前我只觉得我老婆特别看不起我，特别鄙视我，现在看到她记的东西，我发现在她心目当中我还是蛮有价值的。”后来那个先生他就跟他老婆说，他说：“你知道吗，实际上我那个身体不舒服，很多时候就是当你否认我的时候，我就会特别不舒服。实际上我的那个不舒服是想跟你说，我希望你更多地认可我。”所以这就是记红账的作业。

### 3. 悖论处方 🤔

悖论处方的作业我们刚才已经讲过了，我们有两次课专门讲到。那悖论处方的作业有好几种。

第一种可能我们治疗师会故意鼓励家庭当中的某个成员保持或者夸大他们的症状行为和问题，以及当下有问题的互动模式。这种夸大可能会大到一种家庭自己都觉得荒谬的地步，然后进而促使家庭自发地对这些症状产生抵触和厌恶的情绪，从而达到让其改变的一个目的。就比如曾经我见过这样一个男孩，他就特别在意别人对他的看法，只要在学校里面有同学跟他说一些，他觉得有同学在背后议论他，他就会特别担心，他就会担心别人不喜欢他。后来我就给他布置了一个悖论作业，给他们家庭布置了一个悖论作业。那个作业就是让他回去以后，和他的父母去共同商量，制定一份调查问卷。那个调查问卷的目的，是要发给他的同学，那个问卷上面要问的内容，就是要去问他的同学，究竟有多少同学那么在乎他这个人本身到底是好还是不好。因为这个学生，他就觉得所有的人都很在乎我的表现，所以我没办法在他们面前表现得自如，我每做一个行为我都要小心翼翼。我当时就给这个学生说，你和你的父母去做这样一份调查问卷，然后做好以后你去印个200份，周一上学的时候，你就把这个问卷发给你所有认识的同学，让他们去填这个问卷然后反馈给你，你就去看看，到底有多少人真的那么在意你到底表现得好与不好。最后他就真的去做了这个调查，不过他没有对全校的同学做，他只是对他们的同班同学做。结果做下来以后他发现，也好像并不是那么多人关注我，大家实际上都很忙的。

那有时候呢，我们还会让家庭在每天选择一个固定的时间，在这个固定的时间，让他们的症状和问题集中地出现。比如我曾经面对一个强迫症的小姑娘，我布置了这样一个作业给她。她说我的强迫症我控制不住，我不想要它我想摆脱它，但是我没办法摆脱它。我就跟她说：“你知道吗，你不用急着要摆脱你的强迫症状的。你今天回去以后，你每天选一个固定的一小时，在那一个小时里面，你什么都别干。在那一个小时里面你就想方设法，把你强迫症的那些强迫思维和强迫的洗手行为，完全地表现出来。但是唯一的要求就是每天那一个小时你都要有，并且那一个小时要在固定的时间，比如下午3点到4点。即使你某一天3点到4点，你觉得你不想让你的强迫症状出来，不行你也得想方设法让它出来。就算你自己没办法让它出来，你就打电话给你的父母，让他们想帮你想办法，把那个症状引出来。”后来他就回去做了这个作业，做了两个星期以后他回来跟我说，觉得好像现在症状反而减轻了。我问他你为什么觉得症状减轻呢？他说：“当我真正想去让那个强迫症状出现的时候，我发现实际上我越想让它出来它越不出来，我越不想让它出来它反而就越出来。”所以这样一个悖论处方，本身也是把家庭放在一个不会输的位置上面。也就是说，他如果按照你的处方去做了这个作业，他的症状也得到了表达，并且他的症状也是在规定的时间内，按照他的意志在表达，那这就代表他对他的症状是有控制能力的。如果他觉得也有的家庭听了我这个作业觉得，你这个作业太愚蠢了我不要做，那他如果不做的话他的症状也就消失了。所以说悖论处方是把家庭放在一个不会输的位置上面。那当然你给别人布置了这样的作业以后，下一次再见他们的时候，你需要和他们去讨论他们做这个作业的过程、收获和感受。

当然了，有时候我也会布置第三种悖论作业，就是我会让家庭回去以后讨论一下，他们的症状有什么好处。比如刚才那个强迫症的小姑娘，我就给他们家庭布置了一个家庭会议的作业。我就说你们每两周当中每周选一个小时，这一个小时你们一家三口，你和你的爸爸妈妈就坐下来讨论一下，这个强迫症状出现以后，它有给你们的家庭带来什么好处。你们要讨论出十条好处，然后写下来带给我讨论。后来那个小朋友和他爸爸妈妈就真的是做了讨论，他们总结出十条好处。其中有五条好处，基本上是说这个症状出现以后，我的妈妈就不再唠叨了，我的爸爸就不再回避了，他们就开始有意无意地要去讨论我的病了，我们家反而因为我的病有了一个新的沟通的话题。另外五条就涉及到的是这个小姑娘，她觉得我有了这个病以后，我就不用天天去学校了，因为她不太喜欢那个学校。她就说我现在终于有了一个正当的理由，可以不去学校了。所以这样布置作业的过程本身，也是帮家庭去意识到这个症状的功能和意义。

### 4. 水枪射击与橡皮筋游戏 🔫

这个行为是可以直接地让我们来对家庭当中一些成员他们不想出现的行为，进行一个中断的处理。比如一个家里边，那个妈妈特别唠叨，爸爸又特别回避，那可能他们都不希望看到对方的这种行为。那这个时候我就会布置一个作业，比如我会跟那对夫妻说：“你们回去一个人买一把水枪，水枪里面装满水，你们就把那个水枪在家里的时候就别在腰杆上。如果丈夫你发现你的妻子回去以后唠叨又超过了五分钟，你就拿那个水枪往她脑门上射那么一枪。如果妻子你把那个水枪别在腰杆上，你如果发现你回去以后，你的话相同的话说了两遍你的丈夫还是像没听见一样，你也拿那个水枪往他的脑门上来那么一枪。”就这样的方式，它是一种有趣的、温和的、开玩笑的方式，用射水枪来向其他的成员传递你想表达的意图。那因为这对夫妻之前，那个妻子一看到丈夫回避，她就拼命地怒吼，她越怒吼那个丈夫就越回避，那他们原本是一追一逃。那现在我们用这种水枪射击的游戏，让他们用一种有趣的方式，代替原本的怒吼和逃避来表达他们的需要。

所以大家注意到，我们到目前为止布置的家庭作业都是比较好玩、比较有趣的。有时候天气冷了，没办法让他们去射水枪，你可以怎么用呢？你可以用橡皮筋。你可以跟他们说原理是一样的，如果那个妻子唠叨超过五分钟，你就用橡皮筋弹她的脑门一下，丈夫就弹妻子的脑门一下；如果那个丈夫又回避了，妻子也弹那个丈夫的脑门一下。那这样的一种方式呢，就是用有趣的方式，对家庭的某些不合理的行为进行扰动和挑战。

### 5. 书信与电话作业 ✉️

第五个作业也是我们常用的，不过现在因为信息的发展，这种方式有一些调整，就是书信和电话作业。这种作业对于居住在外地、不容易来规律地参加治疗的家庭比较有用。治疗师比如这个家庭来见你一次很不容易，你给他们布置的作业，或者说你见了他们几次以后，你就放他们回去自己改变了。然后这个时候你可以和家庭做一个约定，你可以说半年以后，比如女儿或者说父母，你们可以给我写一封信，来跟我说一下，在这封信里说一下你们最近家庭的变化，以及作业的完成的情况，或者打个电话给我，报告一下你们最近的变化。

那这个作业本身呢，可以让家庭觉得他们和治疗师还有连接，同时也可以把治疗干预、治疗师在咨询里面说过的话的影响，在无意识层面延伸到治疗师之外。那样的话，这个家庭心里面他就留了一个印象，他就知道，不管我们做什么改变，我们最后都需要和某某的治疗医生，比如刘医生，刘亮医生有一个总结。
