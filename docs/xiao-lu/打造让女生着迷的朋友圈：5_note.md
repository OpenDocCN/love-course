# 打造让女生着迷的朋友圈：5_note

在本节课中，我们将要学习如何打造一个具有吸引力的展示面，即优化你的朋友圈内容，使其在聊天前就能吸引异性，塑造积极的个人形象。

女人会通过我们的朋友圈来定位我们的形象，判断我们是什么样子的人。很多人喜欢把自己的朋友圈变成一个八卦的集合地，里面充斥着乱七八糟的链接和广告。当女人看到这些内容时，会觉得这个男人非常无聊，每天无所事事。这种低端的展示方式当然不会吸引女生。女人看重的是男人的品质、生活、价值和品味。她们都喜欢高品质的男人，但不喜欢那种装逼的男人。所以，本节课将教你如何正确地建立展示面。

## 第一步：打造吸引人的头像 📸

上一节我们介绍了展示面的重要性，本节中我们来看看打造吸引力的第一步：头像。

一个好的头像相当于门票。如果你想让女生回复你，头像的比重至少占百分之七八十。即使你的朋友圈内容再好，如果头像差，对方一样不会理睬你。所以，第一步就是要有一个帅气的头像。

有的同学可能会问，自己长得不够帅怎么办。没关系，颜值不够，发型和拍摄角度也可以让你的头像看起来很酷。你一定要找到属于自己的发型、衣服和拍照风格，同时注意拍摄角度。颜值不够就拍侧脸或者其他角度。总而言之，要扬长避短。拍照前花心思打扮一下，相信你会发现不一样的自己。

以下是拍摄头像时的几个核心要点：

*   **背景整洁**：背景要干净、整洁，避免宿舍、床铺等杂乱环境。可以选择纯色背景墙、有情调的咖啡厅或时尚的背景墙。
*   **突出面部**：头像应该放头像照或半身照，不要放全身照，因为全身照很难看清脸部，难以引起对方注意。尤其要避免背景是大山、人物很小的全身照。
*   **角度选择**：拍摄头像时，角度不要正面面对摄像头（如证件照），很难拍出美感。应选择侧面或45度角，这样会让头像更立体。具体操作是：脸不要正对相机，在水平面保持大概45度角；同时抬高手机约30度，采用仰视角度，这样看起来下巴曲线会更漂亮。

但是要记住，再好的头像，如果你的朋友圈质量跟不上，也会被对方认为是虚假的。所以，有了一个好的头像作为第一印象后，接下来我们就要开始学习如何建设我们的朋友圈了。

## 第二步：建设高品质的朋友圈内容 🌟

朋友圈的核心是**价值**。它主要通过你发布的照片，全方位展示自己，包括相貌、身高、经济水平、社会地位、个人品位。目的是让女生知道你与众不同，比其他男生更有特点、更有价值。我们要正确地展示出高生活品质和时尚感。

### 1. 时尚感的自拍

自拍不是单纯长得帅就好看，而是要让自己看上去是一个型男，很时尚、很会打扮。因此，服装搭配、场景选择以及后期修图都很讲究。你可以长得不帅，但照片拍出来一定要有时尚感，这样女生才会觉得你是一个有高生活品质的人。

以下是自拍的注意事项：

*   **利用装饰物**：可以找手链、帽子等作为前景或搭配，但不要放得太大、太突出，以免喧宾夺主。
*   **注意光线和角度**：良好的光线和角度能让照片更有美感。
*   **重视发型**：一个差的发型能直接毁掉整体的时尚感。建议找专业理发师，根据你的脸型和气质设计一个适合的发型。

### 2. 展示丰富多彩的生活

生活类照片的核心是：低调地炫出丰富多彩的业余生活，并且要表现得“小资”一些，因为物以稀为贵。

以下是几种值得展示的生活内容：

*   **美食**：偶尔发布卖相好的美食照片，对女生很有诱惑力，也能侧面说明你对美食有追求。拍摄时要保证食物看起来可口诱人，画面不要过于杂乱。除了食物，精致的餐具、特色饮品也能展示生活品质。注意，美食不是指兰州拉面、沙县小吃或烤串。发美食的目的是展示高生活品质，同时引发女生兴趣（如询问地点、味道）。展示厨艺（如卖相好的饭菜、正在做饭的样子）也能塑造居家好男人的形象，给人安全感。
*   **认真工作或做事**：专注的男人最有魅力，能无形中给人可靠的信任感。例如，在电脑前思考、修理家中物品、做饭等照片。拍摄这类照片时，尽量不要看镜头，否则会显得刻意。最好请朋友帮忙抓拍，眼神要自然集中在所做的事情上。
*   **与小动物或小朋友合照**：女生天生对可爱的小宠物没有抵抗力。发布遛狗、喂流浪猫或与小朋友的合照，能体现你的爱心。女人会认为，一个有爱心的男人更能照顾好她和家人。
*   **丰富的娱乐活动**：没有女生不喜欢生活丰富多彩的男生。不同的活动能最大程度引起女生的注意力和好奇心。例如，健身、跑步、游泳等照片，说明你热爱生活、身体健康。更高级的运动如滑雪、冲浪、攀岩能提升逼格。陶艺、弹琴、唱歌等照片则能展示艺术修养和审美水平。
*   **与其他女生的合照**：适当发布与美女的合照，可以让女生知道你有女人缘，身边有优秀女性，这会增加你的吸引力。但注意，合照不要显得很暧昧，以免让其他女生觉得你是花花公子。朋友圈里有一两张即可，配文要突出只是普通朋友或同事关系，例如：“万能的朋友圈，我的好朋友在找男朋友。”
*   **社交类照片**：这类照片可以体现你在人群中的地位和受关注程度。可以发布与朋友的合影，聚会场面越大越好（如会议讲话、餐桌聚会）。但要以你为中心，体现你是照片中的核心。这表达了你在同性中的地位和受尊重程度，反映出你是一个领袖人物。

上面介绍的这些只是正确展示面的一部分，也是最典型的。任何积极向上、能体现你价值的照片都可以发到朋友圈，重点是要自然表现出来，不要刻意突出。

## 第三步：优化与配文 ✨

好，我们知道该发什么朋友圈了，但是也不要忘了修图和配文。

拍摄出来的照片要用修图软件优化一下。人的脸上总会有一些瑕疵，皮肤也可能黯淡。我们要让自己看起来更有型。这里推荐使用**美图秀秀**进行美化，它有一键美颜功能，可以自动帮你修整照片。

有了高大上的照片，接下来还需要文字的搭配。告诉大家一个最基本的方法：**不要去直接描述你在做什么**。比如你在游艇上自拍，照片已经显示了高价值，如果你再配文“我在游艇上”，就显得很装。

我们在配文字的时候，要避开照片中想要直接表达的高价值。例如，有张照片你想表达自己在开保时捷，但文字不能配“开着保时捷出去兜风，好爽啊”。而是要借助其他话题，比如配文：“今天天气不错，兜风去。”并配上自己开车的照片。

## 需要避免的错误展示面 🚫

如果要发旅游的照片，要确保人和景完美融合，看着像有故事的。避免全是人头、到此一游式的照片。在环境一般的景区里，游客味十足的风景照，或者并不高档、没有格调的餐厅照片，也不适合发到朋友圈。

那种过度装的朋友圈，总是以适度为原则。过度装很容易引起女生的反感。就像一个朋友经常对你吹嘘自己很有钱，但请你吃饭只吃烤串，你会不会觉得这个人很虚伪？就是如此。

所以发朋友圈的时候，不要刻意展示某件事或物。一旦你控制不好度，很容易让女生觉得你是在装。当女生留下了这样的印象，你想改就难了。

第三类要避免的是各种充满了伪文艺青年气息的诗句和歌词。有的人喜欢走文艺青年路线，但做一个真正的文艺青年绝非易事，绝不是随便抄两句歌词、背两首诗就能做到的。很多同学一时兴起在朋友圈发诗句歌词，这并不能给妹子一种你很文艺的感觉，只会觉得你在装。

最后说一点，朋友圈不要天天发，几天一次就好了。要么不发，要发就发一些有意义的。朋友圈展示面的建设是一个长期的过程，所以你必须从现在就开始好好经营。
