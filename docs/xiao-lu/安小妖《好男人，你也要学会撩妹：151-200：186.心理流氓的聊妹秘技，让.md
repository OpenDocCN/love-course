# 安小妖《好男人，你也要学会撩妹：151-200：186.心理流氓的聊妹秘技，让

在本节课中，我们将学习如何运用心理咨询师的沟通技巧来提升与异性聊天的质量，避免冷场，并建立深度连接，从而成为一个会聊天、会撩妹的高手。

聊天中的尴尬冷场是许多人都会遇到的问题。好不容易将心仪的女生约出来，见面后却不知该说什么，导致气氛迅速冷却。仅仅打个招呼后便陷入沉默，即便事先准备了话题，也找不到合适的切入点。当女生谈论自己不熟悉的内容时，交流便难以继续，约会可能草草收场。

这样的聊天缺乏质量。这样的约会不仅无法展示自身价值，反而会让女生感到失望。约会的效果可能还不如不约。

那么，如何才能成为一个会说话、会聊天的人呢？下面将分享一些源自心理咨询师的说话训练技巧，并将其融合到恋爱场景中，帮助你成为撩妹高手。

## 一、 自身的输入 📚

我们这里说的“输入”，主要指内容的积累。就像与外国友人交流，你需要有足够的英语口语储备才能参与对话，而这需要你持续学习英语知识。

谈恋爱也是如此。与女生聊天，你需要有足够的生活阅历储备。如果女生想聊美食或旅行，而你因为生活仅限于两点一线（上班和回家），休息时只与泡面、游戏为伴，缺乏丰富体验，自然就接不上话题。

试想，当女生提到她感兴趣但又不甚了解的事物时，你若能自信地侃侃而谈，为她进行科普或发表见解，会带来什么效果？首先，这提升了聊天内容的数量与质量，展现了你的高价值。其次，这也增强了你的自信。

提升方法很简单：**读万卷书，行万里路**。努力成为一个有独立思维能力、生活丰富的人。不要再将自己局限于出租屋内的游戏和泡面。

## 二、 善于倾听 👂

倾听是心理咨询中非常有用的技巧。倾听不仅仅是听对方说了什么，更是感受对方话语背后所传递的情绪。同时，倾听也是对自身“输入”的一种补充。

更重要的是，善于倾听能够拉近你与女生的距离，赢得她的信任。很多时候，女生说话只是为了抒发情绪，你不一定需要解决她的问题，但一定要学会倾听。

许多人去做心理咨询，60分钟的咨询里，自己讲了55分钟。但神奇的是，经过几次这样的咨询，他们的苦恼就消失了。有人可能觉得这钱花得冤枉，咨询师似乎没说什么。

其实，咨询师在这个过程中扮演了优秀的倾听者角色，在关键时刻发表观点，与来访者形成良好互动，帮助对方梳理问题。对方倾诉完毕，自己也就轻松了。

## 三、 开启新话题的转移技术 🔀

当女生说完一段话，你想开启一个新话题时，可以运用咨询中的“转移技术”。在倾听过程中，记住对方话语中的一些**关键词**，然后进行**引申**、**过渡**和**联想**，从而将话题引导至自己擅长的领域。

简单来说，这是一个套路：**关键词 + 感受 + 新话题**。

## 四、 强制聊天系统的培养 💪

有人会说，我感觉开启不了话题，难以开口，比较自卑，控制不了场面。话到嘴边，就变成了“聊天终结三件套”：啊、哦、嗯。

这时，你需要强制培养自己的聊天习惯。方法很简单：与女生聊天时，**一定**要选取她刚才话中的一个关键词，并围绕它连续说上三句话。具体说什么不重要，重点是要在短时间内快速说出来。

久而久之，你就能获得自由聊天、自如开启话题的能力。这有点像心理咨询中的“满贯疗法”和“作业系统”。

## 五、 聊天中的目的性 🎯

很多人聊天其实没有明确目的，仅仅是因为无聊和孤独而找人说话。而且，女生也不会仅仅因为你说了什么就爱上你。

因此，聊天时，你必须营造一种让女生想要探索你的氛围。没有核心的闲聊，不仅无法将女生约出来，还会慢慢消耗你的吸引力。

所以，在适当的时候，切入关于**三观**、**生活**、**人生**的话题是十分必要的。这有助于带领女生了解你的内心世界。

## 六、 聊天中细节的展示 ✨

很多人之所以不会聊天，在于讲述时缺乏细节展示和内心独白，只是干巴巴地叙述，甚至不会举例。

举个例子：女生和你聊了一堆关于日本文化或东野圭吾的书。如果你只是回应“嗯”、“哦”或“我也喜欢”，然后就没有下文了，那么一路都会寂静无声。

但如果你也能谈谈书中的内容，表达对作者的看法，说明为什么喜欢，还喜欢他的其他哪些作品等等。这样的聊天，想让女生不喜欢你都难。

人生就像一场戏，重要的不是剧本，而是你能接得住对方的“台词”。如果你连对方的台词都接不住，那就要出局了。
