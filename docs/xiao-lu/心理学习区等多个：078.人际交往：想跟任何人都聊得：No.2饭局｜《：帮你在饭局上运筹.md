# 心理学习区等多个：078.人际交往：想跟任何人都聊得：No.2饭局｜《：帮你在饭局上运筹

在本节课中，我们将学习《中国饭局潜规则》一书的核心内容。饭局在中国文化中远不止吃饭那么简单，它是一个重要的社交场合，蕴含着许多学问和技巧。掌握这些规则，能帮助你在各种饭局中游刃有余，达成社交目的。

## 概述：饭局的重要性

饭局的重点不在“饭”，而在“局”。它不仅是聚餐，更是达成个人目的、建立关系、影响职业生涯的重要场合。一顿饭可能改变一生。

我们通过三个故事来理解饭局的影响。

**故事一：点菜不当丢工作**
赵小姐作为秘书，因点菜搭配不当造成浪费，最终失去工作。这说明了点菜是一门学问。

**故事二：奢华宴请吓跑合作**
一家东北企业为招待美国合作方，设下奢华宴席，反而让对方认为公司浪费、不靠谱，导致合作失败。

**故事三：结账方式影响姻缘**
李小姐因约会时点菜和结账的方式不当，多次错过姻缘。这体现了饭局中细节的重要性。

以上故事表明，无论是职场、商务还是私人交往，饭局都是必不可少的工具。其中涉及邀请、点菜、敬酒、结账等各个环节，都有其规则。

## 第一部分：请客前的准备

上一节我们了解了饭局的重要性，本节中我们来看看如何为一场成功的饭局做准备。充分的准备是饭局成功的基础。

请客前需要考虑以下事项：

1.  **考虑宾客的饮食禁忌**：特别是少数民族宾客。例如，回族人忌食猪肉，且不烟不酒；蒙古族喜食奶肉；维吾尔族饭前饭后必须洗手。
2.  **考虑地域饮食差异**：中国各地口味不同。例如，广东喜清淡，江西、湖南、四川喜辣，华北地区口味偏咸。
3.  **选择合适的餐厅**：一个基本规则是：**可以请熟悉的朋友去不熟悉的餐厅，但请不熟悉的客户应优先选择你熟悉的餐厅**。这样可以确保菜品和服务质量。
4.  **考虑特殊人群**：如有老人或小孩，需注意他们的喜好和忌口，例如是否有高血糖、高血压等。

**课后作业**：请课后搜集中国八大菜系及各地饮食风俗习惯，这对点菜很有帮助。

## 第二部分：如何发出邀请

准备好了宾客信息和餐厅，接下来就是发出邀请。如何邀请才能确保对方接受，避免被拒绝的尴尬呢？

以下是针对不同对象的邀请方法：

**邀请领导：以敬为先**
邀请领导通常有两个目的：工作晋升或有求于人。关键在于找到一个好理由。
*   **方法**：可以借请教工作之名，投其所好。例如：“经理，我正好有事想请教您这位专家，听说公司对面有家不错的江苏菜馆，要不我们边吃边聊？”

**邀请客户：以诚为先**
请客户吃饭是维护关系的重要环节。
*   **四种邀请方式**：
    1.  **电话邀请**：直接说明原因、时间、地点及参会人员，引起对方兴趣。
    2.  **发请柬**：正式且尊重的方式，适用于高端酒会或正式宴会，通常会注明着装要求。
    3.  **登门邀请**：适用于已有一定交情的客户，可附带小礼物以示重视。
    4.  **信件邀请**：方式简单，但措辞需体现对对方的重视。

**邀请同事：注意分寸**
邀请同事相对简单，但需注意两点：量入为出、注意身份。不要因请客去过于高级的餐厅而显得招摇。
*   **常用方法**：
    *   **开门见山**：直接提出邀请，适用于关系熟的同事。
    *   **喧宾夺主**：事先调查同事喜好，选择有特色的餐厅发出邀请。
    *   **借花献佛**：借自己发生的喜事（如中奖、加薪）为由邀请同事。
    *   **步步为营**：适用于第二次邀请同一人，可以上次愉快的用餐经历为引子。

**领导请下属：以情为先**
领导请下属吃饭能拉近关系。为避免下属猜疑，最好事先说明原因，营造轻松氛围。例如：“这段时间大家为了项目加班辛苦了，今天我请大家吃个庆功宴。”

## 第三部分：邀请的通用技巧

除了针对不同对象，作者还总结了一些通用的邀请技巧，共八种。以下是其中七种核心方法：

1.  **开门见山法**：直接提出邀请。
2.  **借花献佛法**：以自己喜庆之事为由发出邀请。
3.  **喧宾夺主法**：事先调查对方喜好，以此为由选择餐厅并邀请。
4.  **暗度陈仓法**：以其他事情（如交接文件、请教问题）为借口，实际目的是邀请吃饭。
5.  **声东击西法**：故意拖长交谈时间，待到饭点时顺势提出边吃边聊。
6.  **步步为营法**：第二次邀请同一人时，以第一次愉快的用餐体验为引子。
7.  **诱敌深入法**：先聊一些无关话题（如对方家乡、喜好），再自然过渡到邀请。

**练习案例**：
小于夫妇想邀请负责分房的经理吃饭，但经理早有防备。他们得知经理酷爱红酒，于是买了一瓶红酒，以“不懂酒，想请您帮忙鉴别一下”为由成功邀请。这种方法融合了“投其所好”和“暗度陈仓”。

## 第四部分：着装与打扮

成功邀请后，参加饭局时的着装也不容忽视。得体的着装是尊重他人的表现。

**男士着装要点**：
1.  **西装合身**，忌花哨。
2.  **遵循“三色原则”**：全身颜色不超过三种。
3.  **遵循“三一定律”**：鞋子、腰带、公文包颜色统一。
4.  **西装扣子**：单排扣可不扣；双排扣可全扣；三粒扣通常最下面一粒不扣。
5.  **忌黑皮鞋配白袜子**，应配黑色或灰色袜子。

**女士着装要点**：
*   一个公式：**三分姿色 + 一分化妆 + 两分服装 + 两分首饰 + 两分手袋**。
*   配饰不宜过多，不超过三种为宜（如手表、项链、耳环）。
*   **重要提示**：不要在他人面前补妆或整理衣物，如需整理请去洗手间。

## 第五部分：饭桌礼仪

来到饭局现场，从交换名片到入座就餐，每一个细节都体现着礼仪。

**交换名片**：
*   **接收时**：应看过并适当赞美，然后放入上衣口袋或名片夹。切忌塞入裤兜、在手中把玩或随意放桌上。
*   **递出时**：应起身，双手递上，让名字正对对方。
*   **注意**：在陌生人群中不要四处散发名片。

**寒暄与介绍**：
*   **打招呼顺序**：年轻者先向年长者问候，男性先向女性问候，身份低者先向身份高者问候。
*   **称呼**：以对方最高头衔称呼。
*   **自我介绍**：在饭局上，简洁介绍**姓名、供职单位、职务**即可，最多再加点兴趣爱好或特殊经历，时长控制在三到五句话。
*   **为他人介绍**：记住 **“尊者居后”** 原则。即先把身份地位低的一方介绍给高的一方。例如：“王总，给您介绍一下我的朋友李女士。” 然后再反过来介绍。

**入座礼仪**：
*   应从椅子**左侧**入座。
*   **座位尊卑**：正对门的位置为主座（上座），通常是主人或最尊贵的客人。主人右侧为主宾，左侧为次宾。
*   私人物品（如手机、钥匙）不要放在餐桌上。
*   坐姿端正，就餐时主人不动筷，客人不应先动（汉族习俗）。

**餐桌上的言行**：
*   **微笑是最好的名片**，尤其在陌生人多的场合。
*   **吃相即人品**：遇到太烫或变质的食物，应不动声色地处理（如用纸巾）。
*   尽量避免在餐桌上打哈欠、咳嗽、擤鼻涕。如需剔牙，请去洗手间。
*   **敬酒时**，酒杯应略低于领导或长辈的酒杯。
*   **倒茶**：**“酒满敬人，茶满欺人”**。茶倒七分满，酒倒九分满。

## 第六部分：点菜与饮酒

点菜是饭局的关键环节，点得好坏直接影响宾客感受。

**点菜技巧**：
1.  **考虑周全**：兼顾宾客饮食习惯、禁忌（如素食、回民、疾病忌口）。
2.  **他人请客时**：如被要求点菜，应点价格适中、大众化的菜品，切忌专挑贵的。
3.  **搭配原则**：注意荤素、冷热、鸡鸭鱼肉的搭配，追求色香味俱全。
4.  **招待外宾**：可点中国特色菜（如宫保鸡丁），但注意外宾通常不喜过辣。
5.  **菜品数量**：通常菜品总数约为用餐人数的1.2-1.5倍（如10人点12-15个菜），冷热搭配。

**饮酒注意事项**：
*   **与时令搭配**：冬天宜白酒，夏天宜啤酒。
*   **饮用顺序**：先低度酒，后高度酒；先普通酒，后名贵酒。
*   **特殊搭配**：吃螃蟹时，搭配性温的黄酒可以中和螃蟹的寒性。
*   **倒酒顺序**：主人第一次斟酒，应从主宾开始，按**逆时针**方向进行，最后给自己倒。

（注：更详细的敬酒、劝酒礼仪将在后续课程中专门讲解。）

## 第七部分：结账与西餐礼仪

饭局的结尾——结账，同样需要讲究技巧。

**结账注意事项**：
1.  **异性聚餐**：通常由男士买单显得有风度。
2.  **同事朋友AA**：不一定当场均摊，常见形式是“这次你请，下次我请”。
3.  **商务宴请**：如与领导同请客户，通常由下属现场办理结账手续（但实际由领导支付）。下属应有眼力见，领导通常也会事后补偿。
4.  **避免尴尬**：故事中李小姐的教训表明，在约会等场合，女方可以主动提出AA或事后发红包表示谢意，避免给对方造成“太会花钱”的印象。

**西餐礼仪简介**：
*   **座位**：女主人右侧为第一男主宾，左侧为第二男主宾；男主人右侧为第一女主宾，左侧为第二女主宾。
*   **餐具使用**：基本规则是**左手持叉，右手持刀**。
*   如需参加正式西餐宴请，建议提前查阅相关资料学习详细礼仪。
