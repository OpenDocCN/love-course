# 【魅力工程】2.0：男士自身建设：第2节如何提升自己的外形_note

在本节课中，我们将学习如何通过服装选择和搭配来有效提升个人外形。核心在于理解服装如何修饰体型，并根据自身特点与场合选择合适风格。

## 西装：修饰体型的利器

上一节我们介绍了外形建设的重要性，本节中我们来看看如何利用西装塑造理想形象。西装因其复杂的剪裁和内部结构，能有效掩盖多种体型缺陷。

其目的就是通过结构来修饰体型。例如，对于腰部不明显的体型，可以通过提高收腰位置来塑造腰线；通过胸部剪裁，可以营造出胸肌饱满的感觉。

画面上这个人其实挺胖的，但是你看，加宽的肩部和高位收腰将腰线衬托出来了。胸部多留点面料，胸肌膨起的感觉也出来了，看起来就有健壮感。

丹尼尔·克雷格的身形其实非常健壮，但是这件西装很好地消化了他过于健硕的胸肌等突出线条，让他看起来身材修长。所谓“脱衣有肉，穿衣显瘦”也要依赖好的衣服。

西装对于身材瘦弱的人同样可以起到修饰作用。如下图所示，穿T恤和穿西装的效果不一样。原本很窄的肩膀被西装修饰得比较宽、比较有力量，整体修长又不会觉得羸弱。

## 非西装服饰的选购与搭配原则

了解西装的作用后，我们来看看其他服饰的搭配。选购搭配非西装服饰一定要根据自己的体型特点，例如腿长臂短。

以下是几个关键方法：
*   **拔高腰线**：将衬衫塞进裤子里来拔高腰线，改变身材比例，这样可以显得腿长。
*   **裤鞋同色**：裤子和鞋同色也是拉长腿长的方法。如右图，选择和裤子同色系的鞋，视觉上起到一定拉长腿长的作用。

**把你的小肚子藏起来**。对于大部分身材偏胖的人来说，小肚子是最棘手的问题。穿衣服需要注意的是，不要一味求宽松，这样只会显得更胖。

最好选择有剪裁设计的、能帮你遮盖缺陷的衣服。最糟糕的就是扎一条腰带，反而勾勒出胖的线条了。

## 找到适合自己的着装风格

其实不管是什么衣服，都要找对适合自己的着装风格。这个应该很好理解。你的长相是什么类型、你在什么年龄阶段、希望将自己塑造成什么类型（如阳光、青春、个性、健壮、成熟、稳重等等），综合这些情况做出决定，才能根据目的来选择。

以下是几种常见的风格类型：

**休闲随意型**
喜欢休闲风格，那么你可以选择随意型的着装。搭配起来很简单，都是最基础、最平常的单品。你们一定都有T恤、衬衫、牛仔裤、休闲裤、套头毛衣、开衫、休闲外套等等，简单百搭。注意，带有花纹或者有图案的款式往往需要更高的搭配技巧，对搭配物的要求也高，所以慎选。

**运动型**
运动型也是比较休闲随意。通常搭配运动T恤、运动裤、运动鞋。同样，我们建议尽可能选择单色的、没有大块图案或拼接的款式，搭配起来更容易。

**文艺型**
文艺型的人走的是比较个性的风格。可以选择特殊质感和特殊款式的衣服。气质最好装得优异一点。

**硬汉型**
体型比较强壮、肌肉发达。受体格的影响，所以这类人穿T恤、衬衫等衣服，都能展现出硬汉的味道。但是如果你的身材不那么强壮，就要靠服装了。军装元素的衣服能让人很有硬朗的感觉。面料选择比较硬的丹宁布、帆布、皮质等等。颜色一般选择深色系，或者军用彩色，即军绿色和卡其色。

**绅士型**
绅士型男人在休闲时大多都穿西装夹克。不太喜欢太艳的可以选择较深的素色。想要更休闲也可以选择亮一点的颜色，如浅裸色、唇色、象牙白、浅卡其等等。夏季还有浅蓝色、浅粉色用处都很多。此外还有大格纹、小千鸟格。

其实还有更多的类型风格，这里就不分析得太细了。

## 有品位的选购要点

有品位的选购服装，必须知道以下几点。说到品位，显然不包括牛仔服、化纤运动服之类的。能提到台面上的，都是富含历史文化的服饰。这里面当仁不让的主流就是西装（西式服装系统）。

**合身为第一要务**
合乎身材的男士着装，以合身为第一要务。不管是穿衬衫还是西装，合身是一种优雅的态度。体现在正装上，合身的服装能精准地吻合体型，同时又能修饰体型。

**面料选择很重要**
面料等主要材料的选择很重要。许多人为此感到迷惑，追求着不切实际的地方，如面料支数。穿上200支纱的衬衫，却忽略了脚下的皮鞋是不是牛皮大底，就像木桶的短板。

就所谓的品味而言，纯天然材质是必须的。然而许多人并不知道什么才算真正的穿着天然材质。衬衫、西装、大衣还好说，当然是化纤含量为零。许多人不知道的是，皮鞋的鞋底必须是牛皮压制，而袖扣上不能含有奥地利水晶、波兰钻、施华洛世奇水晶这种实为玻璃制造的人造装饰物。

如果说服装的合身度决定了整体的视觉效果，那用料考究则拉近距离为你加分、提升品味。

**工艺与款式细节**
工艺水平则更多的是在接触和使用时才能感受得到的东西。当然，好的工艺其内在毋庸置疑地代表了对极致的追求。款式细节拼的是文化内涵。

品质好的西装衬衫不仅在用料工艺上讲究，其款式细节也是符合历史文化的，比如花眼、驳头、兜盖设计、葫芦腰剪裁等等很多。不甚高级的西装必定会疏漏在这些代表文化典故的细节上。

## 颜值不高者的着装雷点

首先，颜值不高、气场不强的人就不要穿一些设计夸张奇怪的衣服了。真的不适合，反而会给人一种“丑人多作怪”的感觉。比如画面中的这些衣服，就算是吴彦祖穿恐怕也是用“丑”字来形容吧。

其次，某方面气质不突出的人就不要走夸张的个性路线，比如嘻哈风、黑暗摇滚风、吸血鬼风等等。

**简洁最重要，简洁最重要，简洁最重要。** 重要的事情要说三遍。如画面中的这位，这种摇滚style真的非常不适合他，还配着一双红皮鞋，什么意思呢？

最后，颜值不高的人就是不要戴夸张的配饰、留夸张的发色、化夸张的妆容。来欣赏一下某小鲜肉自认为很美的发型和烟熏妆。我们再来看一下这位时尚界的灾难常客。

可以学学左边，把多余的帽子去掉，把磨白的水洗牛仔裤换成合身的黑色休闲西裤，相信会好很多。有颜值就不要“作”，下面这位不戴帽子就会好一点，清爽的短发更好看，有年轻的阳光男孩的感觉。

## 常见问题解答

学完这几节课，我相信你的脑子里会有很多问题，比如西装的挑选、鞋子的搭配等等。我们在这套时尚视频里有非常详细的选购指南和搭配方法，大家都可以去学习。最后解答几个常见的问题吧。

**问题一：背头的蓬松度怎么调整？**
其实用强力定型的发蜡就可以了。公式：`理想蓬松度 = 合适发蜡 + 正确吹风技巧`。

**问题二：天生卷发或发质软/直硬怎么做发型？**
欧美人头发两边都比较贴就很好弄，可是我们东亚人很多人两边头发是又直又硬又粗怎么搞？很多人的头发也很直硬，选对了定型产品就没有问题。比如某专业品牌的产品按照定型能力分成五档，再粗硬的头发都能征服。天生卷发也是一样，定型不能拉直但是能造出发型。参见之前的巴基斯坦人造型，之前的发质是卷的，造型之后呢？他额角的卷发是我们特意留的，定型产品可以消除这个小卷。

**问题三：是否都要穿西装？**
西装是最好的选择但不是唯一的选择。我个人认为，身材条件比较差的话就更要穿西装了。如果身材中等、略胖或者是略瘦，或者是身材虽差但不是很介意别人的看法，完全可以按照自己喜欢的路线，穿对应的非西装服饰，当然还是要注重造型的清新搭配。

**问题四：如果长相一般，是否发型很重要？请看患者脸部给个发型建议吧。**
选对服装穿好衣服会好看一些。发型跟服装一样能为人明显增色。但是别忘了面部造型，如修眉、胡须、眼镜等等这些附加部分。选择发型要根据相貌、个人风格和个人头发生长条件来决定，不是随便给你一个建议就好。从你的头像来看很像女星奥黛丽·赫本，男人长成你这样，我也不敢设计啊。
