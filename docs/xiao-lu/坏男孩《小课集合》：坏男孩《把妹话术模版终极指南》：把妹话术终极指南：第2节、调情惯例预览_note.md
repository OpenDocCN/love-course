# 坏男孩《小课集合》：把妹话术终极指南 🎯 第2节、调情惯例预览

在本节课中，我们将要学习两个核心的调情惯例。这些惯例旨在通过营造暧昧氛围和操控情绪波动，让关系自然升温。关键在于通过对话引导女生进入特定情境，使其情绪如坐过山车般起伏，从而拉近彼此距离。

## 惯例一：居家生活想象 🏠

上一节我们介绍了调情的重要性，本节中我们来看看第一个具体的话术模版。这个惯例围绕“做饭”展开，通过描绘共同生活的场景来营造亲密错觉。

**核心操作**：在对话中假设你们已经处于情侣关系，共同经营日常生活。

你会做饭吗？
（女生回答：会啊）
那挺好，那以后我来买菜洗碗，你来做饭。

你想的还挺美的。

关于做饭的话题非常适合与女生交流，因为它贴近生活且能引发情感共鸣。这个惯例非常实用，无论是在微信聊天还是实际约会中都可以灵活使用。

以下是这个惯例的要点解析：

*   **营造错觉**：男生的表述“我来买菜洗碗，你来做饭”是在给女生营造一种共同生活的错觉。
*   **代入关系**：买菜、做饭、洗碗这些行为，通常是情侣居家过日子才会共同完成的事情。在男生的引导下，女生会不知不觉地把自己代入到这种亲密关系中。
*   **巧妙调情**：这是一种不直接表白，却能有效传递兴趣和拉近距离的巧妙调情方式。

**核心操作**：先制造失落感（情绪低谷），再突然反转带来惊喜感（情绪高峰）。

（女生反应：不会吧）

（展示女生照片）你看就是这个，怎么样很漂亮吧。

这个惯例在微信聊天和实际约会中均可使用。在实际约会时，你可以直接打开手机中女生的照片递给她看。

以下是这个惯例的步骤与效果分析：

*   **制造悬念与低谷**：在已知双方单身的情况下，突然声称“自己有女朋友”，会让女生的心里“咯噔一下”，情绪瞬间跌入低谷。
