# 安小妖《好男人，你也要学会撩妹》：238. 五步详解二次吸引，一次性教会你挽回的全部套路 🧠💔➡️❤️

在本节课中，我们将要学习“二次吸引”的核心概念与五个具体步骤。分手后的挽回并非简单的纠缠，而是一场关于自我成长与关系重建的旅程。我们将系统性地拆解这个过程，帮助你理解如何通过提升自我价值，重新建立吸引力，并最终实现关系的修复。

被分手是痛苦的，尤其对于在原生家庭中经历过压抑与恐惧的冷漠型依恋人格者而言。成年后好不容易建立的亲密关系，带来久违的幸福，却常因自身的不成熟与孩子气而破裂。

这并不完全归咎于对方。反思自身，往往是因为不懂得什么是爱，也不懂得如何在亲密关系中相处。虽然社会长期存在男权气息，但从心理学角度看，女性通常比男性更早成熟。例如，女性的青春期始于11-12岁，而男性则在13-14岁，女性大约早熟一到两年。

如同电影《春娇与志明》中的张志明，男性往往需要经历挫折才能变得成熟。我也曾经历挽回，深知其中的艰难、心酸与崩溃。因此，今天分享关于挽回的知识与技巧。

首先介绍挽回中应用的核心概念：**二次吸引**。吸引大致分为两类：初次吸引与二次吸引。

**初次吸引**用于建立初始关系，即让对方成为你的女朋友。它更侧重于外形、着装、性格等显性条件，以及金钱、社会地位等硬实力。

**二次吸引**则旨在达成更深层次、更稳固的吸引，即人格层面的吸引。这包括你的人格魅力、生存能力的体现，以及对高品质生活的追求。

上一节我们介绍了二次吸引的基本概念，本节中我们来看看实现二次吸引的五个具体步骤。

## 第一步：抽离 🚶‍♂️

我们在前面的课程中讲过，被分手者通常会经历五个阶段：否认、愤怒、讨价还价、抑郁和接受。前两个阶段（否认与愤怒）中，人们常采取不理智的行为，如死缠烂打、在对方楼下苦等。

这些行为看似壮烈，实则仅感动了自己，在对方眼中是幼稚的。第三阶段（讨价还价，反复无常）不仅无效，反而会坚定对方离开的决心，因为无人愿意与反复无常的人相处。

对方因对你有诸多不满才选择分开。此时若你无法保持冷静，不给对方时间和空间，只会让他更加失望。因此，进行情感抽离是必要的。

抽离并非完全从对方世界消失，而是**降低你的情感需求**。你依然可以保持联系，但绝口不提感情，也不过度联系。目标是让对方感觉你并非为劝和而联系，仅仅是以朋友身份相处。先保持联系即可。

## 第二步：心态重建与情商提升 🧘‍♂️

在感情中，掌握技巧有必要，但若只照搬套路而不调整好心态，即使侥幸挽回，也可能因同样问题再次分手。正所谓“练武不练功，到老一场空”。

想要挽回，先要修心。空有套路而无深厚内力支持，再高深的招数也发挥不出威力。若有深厚内力，再搭配高明技巧，则将无往不利。

自身慌乱着急的心态会导致情绪起伏不定，因对方一举一动而忐忑不安。在实际咨询中，我们遇到许多案例：对方更换一个朋友圈签名，咨询者便开始胡思乱想。

若两人在一起，你可能根本不会注意此事。在网络互动中，也无需因此产生巨大情绪波动。你情绪波动越大，无形中暴露的需求感越强，你的地位就越低，价值也随之受损，自然无法达到二次吸引的效果。

良好的心态，结合分手后的低需求感与抽离技术，能改变对方对你的原有态度，使他开始产生疑惑：“我跟他分手是不是一个错误？” 这能让你在博弈中取得主动权，变被动为主动。

上一节我们探讨了调整心态的重要性，本节中我们来看看如何付诸行动，进行实质性的自我改造。

## 第三步：自我改造与提升 🛠️

在挽回期间，仅做到抽离不够。若想实现真正的二次吸引，必须趁此机会做出切实改变，而非做表面功夫。改变固然困难，但若不改变，现状也不会改变。

举例而言，我在公司人力资源部工作，负责员工入职、调岗、离职等事务。我们发现许多人辞职的理由看似合理，但所有辞职的潜台词都一样：**“我在你这里没有发展，不舒服。”**

同样，所有分手的潜台词也是一样的：**“我跟你在一起，看不到未来，不舒服。”** 明白这一点后，我们的自我改造便可从以下几个方面着手：

以下是自我提升的几个关键方向：

1.  **外型改造**：若你外型不突出，例如身高不占优势，就需在外型上下功夫。得体的发型、有品位的穿衣打扮、匀称的体型，都能为你的外型加分。
2.  **情商与情绪管理**：若你情商低、脾气不好，可以多学习心理学相关知识以提升情商。学会控制和管理情绪至关重要，许多人因一时冲动犯下难以弥补的过错，得不偿失。

## 第四步：信息传递与朋友圈建设 📱

朋友圈的展示面，是这个世界上**不收费的广告位**。我们可以通过朋友圈的展示，让对方知晓你当前的进步与改变，从而使二次吸引起到实际作用。

当他看到你的朋友圈与以往有很大不同，变得非常精彩，生活方式多元化且丰富时，他便会产生参与你人生的想法。这就是我们建设朋友圈的关键所在。

上一节我们学习了如何展示改变，本节中我们来看看挽回成功后，如何维系来之不易的关系。

## 第五步：学会维持长期关系 🤝

失而复得的感觉我体验过。说实话，它比最强烈、最刺激的感觉更持久，可以持续数天甚至十几天。但那之后，你需要花心思和时间去专业学习。

你需要学习如何经营长期关系，明白怎样的相处方式让双方最舒服、最开心，并以此设定明确的相处模式，建立对等且持续的感情。若挽回后长期关系处理不好，问题仍会发生。

本节课中我们一起学习了“二次吸引”的完整框架与五个核心步骤：**抽离、心态重建、自我改造、信息展示、长期关系维系**。挽回的本质是自我成长与价值重塑。希望这套系统的方法能帮助你理清思路，在修复关系的道路上稳步前行。今天的课程就先到这里，我们下期再见。
