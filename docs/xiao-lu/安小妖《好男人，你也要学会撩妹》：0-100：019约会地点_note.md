## 传统约会地点的局限性 🚫

## 转场的真正含义：转换情绪状态 🔄

我之前给大家介绍过“转场”这个概念，但只是介绍了一点皮毛。其实转场更深刻的意义在于，转换两个人的情绪状态。如果你是从餐厅转到咖啡厅，那这只是一个形式上的转场，并没有理解转场的真正含义。

真正的转场我给大家举个例子，比如从咖啡厅转到电玩城。这就是一个真正意义上的转场。为什么这么说呢？因为咖啡馆这个场所决定了你们的状态会比较拘束，两个人彬彬有礼，处于很紧绷的状态。

这时如果你转场到电玩城，就不一样了。因为周围的人都在玩得很嗨，你俩自然也就会进入一个比较轻松、比较好玩的状态。这个差别就好比听五月天的演唱会，和听爱乐乐团的新春音乐会的差别。

那个状态就是完全不一样的。在电玩城里，你跟女生玩个游戏，那就再自然不过了。而且她会非常认真地跟你玩。玩游戏的过程中，肢体接触就很容易了，关键是你俩都会觉得非常自然。

你俩在这个过程中非常开心，这才是正确的约会路线。经过这样一个过程，你再去牵手，就会比你生硬的牵手要好很多。

## 推荐约会地点一：电玩城 🎮

这就是我要说的第一个容易牵手的约会地点：电玩城。它是个特别有童趣的地方，来到这个地方，你们两个都会变得很天真、很活泼，像孩子一样。而且这里面游戏很丰富，有枪械类、运动类、跳舞机等很多种类，它能充分地调动你们的身体。

所以说，电玩城绝对是我重磅推荐的一个约会的去处。据我观察，现在电子游戏早就不再是宅男的专利了。优质的妹子其实都特别喜欢玩游戏，不光玩得好，而且对游戏周边的设备也特别懂。所以游戏其实是个约妹子特别好的切入点。

我还想起来一点，就是很多电玩城会捆绑一个“滑旱冰”的地方。这个地方也不大，就那么一小圈。其实这个地方简直就是为你牵手而设计的。那个地方没有人说我是去玩花样、修技术的。
