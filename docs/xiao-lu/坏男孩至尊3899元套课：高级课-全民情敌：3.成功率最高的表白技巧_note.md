闲散期是关系的初始阶段。此时吸引力尚弱，感情基础不牢，过于直接隆重的表白会给对方造成压力。

这些方式能让对方明白你的心意，又不会让她感到太大压力。

上一节我们介绍了闲散期的试探，本节中我们来看看如何经营暧昧期。暧昧是男女互动的重要部分，它处于非黑即白的灰色地带，且分量更重。暧昧能带来美好的感觉，因为不确定性会激发双方的想象和情感体验。

过了暧昧期，便进入明朗期。此时的表白可以非常直接，例如“我喜欢你，做我女朋友吧”。

5.  **制造预期**：暗示将要表白但迟迟不行动，在她心中建立期待和张力。这类似于“模糊邀约”。
