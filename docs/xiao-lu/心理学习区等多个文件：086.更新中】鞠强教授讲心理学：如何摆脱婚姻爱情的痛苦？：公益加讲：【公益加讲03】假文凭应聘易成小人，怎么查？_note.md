# 心理学与领导力课程 086：如何识别假文凭应聘者 🔍

在本节课中，我们将学习如何在招聘过程中有效识别使用假文凭的应聘者。假文凭现象在职场中并不少见，且持有假文凭者成为“小人”（即破坏规则、品行不端者）的风险较高。因此，掌握科学的鉴别方法对领导者至关重要。

## 假文凭现象的普遍性与危害

上一节我们介绍了课程主题，本节中我们来看看假文凭问题的严重性及其潜在风险。

在管理工作中，需特别提防小人。小人表现形式多样，其中一种重要形式是持假文凭应聘。目前，持假文凭者在应聘者中的比例高达百分之二十几。

许多管理者并未察觉此现象，因为非专业人员难以识别。大量假文凭在市场中流通。假文凭本身与个人能力无直接关系，但持假文凭者破坏规则的倾向较强。

这类人更容易出现贪污、背后捅刀子、虚报账目或向税务部门诬告等行为。管理者通常不会喜欢这类下属。新应聘者中假文凭的比例比想象中更严重，需设法识别这类人。

## 传统鉴别方法的局限性

有人认为识别假文凭者很重要，因为其成为小人的可能性大。传统招聘中鉴别假文凭的方法效率较低。

以下是几种常见但效果有限的方法：

*   **询问学校细节**：通过询问对方学校具体情况，观察其熟悉程度。这种方法效率非常低，判断过程模糊且主观。
*   **网上查询**：到互联网上查询文凭信息。此方法漏洞很大，因为许多文凭（特别是较早颁发的）网上没有记录，且耗时较长。
*   **发函至原学校核实**：向应聘者声称的毕业院校发函征询。这种方法时间成本和金钱成本都非常高。

管理需要不断创新，传统方法已不足以应对。

## 高效鉴别法：姓名测试法

那么，如何有效查验证书真伪呢？一般我会教授多种策略，但在此科普讲座中，介绍最简单的一种：**姓名测试法**。

假设一位应聘者声称自己是江西商学院2005年企业管理系毕业生。如何判定其文凭真假？网上查询或谈话都不是理想方法。

有效方法是使用一个应聘者不知道的姓名进行测试。例如，随机使用你隔壁办公室一位同事的姓名。假设我有一位同事叫**陆晓勇**。

测试对话可以这样设计：
> “啊，你是2005年江西商学院企业管理系毕业的？你们那时的系主任跟我很熟，关系很好，他姓陆，叫**陆晓勇**，对吗？”

如果对方回答“对”，那么他的文凭很可能是假的。如果对方说不对，那他是真文凭的可能性就比较大。

若想进一步确认，可以进行第二步测试。例如，我常使用我女儿的名字（换上我夫人的姓）进行测试。接着可以问：
> “哎呀，你‘领导心理学’这门课的老师我很熟，你成绩不错还得了个A。我知道你们学校教‘领导心理学’教得很好的老师姓肖，叫**肖子衡**，对吗？”

如果持假文凭者之前已表示不认识系主任陆晓勇，现在又不认识专业课老师，若是骗子便会心慌。通常骗子经不住两三次这样的追问，可能会慌乱地附和“对，子衡老师教得可好了”，从而彻底暴露。

这种方法非常创新、简单且成本低廉。

## 方法的灵活性与补充说明

有人可能会想：“现在知道鞠教授这样查假文凭，我以后去鞠教授的公司就知道怎么应对了。”这种想法是没用的。

姓名测试法只是众多鉴别方法中的一种。还有许多其他更巧妙、更意想不到、效果更好的查验证书真伪的方法。

因此，遇到鞠教授最好少耍花招。
