# 安小妖《好男人，你也要学会撩妹》：218.测试你是不是真男人的时候到了_note

在本节课中，我们将探讨一个在男性咨询中常见的问题——对性能力的担忧与自卑。我们将通过科学数据，分析问题的根源，并澄清关于男性生理特征的常见误解。

---

在以往的咨询过程中，我们遇到不少男性反映与伴侣的性生活不和谐，并因此感到自卑。其中，超过半数的人提到了阳痿或早泄的情况。

按照心理咨询的流程，当来访者报告此类问题时，我们首先会建议对方前往医院检查，以排除**器质性**问题。因为如果是器质性问题，则无法通过心理学方法解决，必须采取医学治疗手段。

然而，数据显示，绝大多数情况并非器质性问题。通过数据收集分析发现，真正的器质性阳痿可能只占阳痿总数的**10%到15%**，其余**85%到90%** 都是心因性的。

也就是说，大部分阳痿早泄是由心理问题引起的，生理上并无毛病，但就是“用不起来”。接下来，我们来分析导致这种心因性问题的三个主要原因。

以下是三个主要心理原因：

1.  **童年早期的负面性经历**。例如，中国末代皇帝溥仪，童年时被太监教唆与宫女嬉戏，成年后便患上了阳痿。他先后娶了五位妻子，却无法满足任何一位，这无疑给其男性尊严带来巨大打击。
2.  **有过不良的性体验**。尤其在青春期，性激素分泌旺盛，一些人可能通过非正常渠道获取被过分夸大或误导的性资讯。这可能导致在实际性行为中因过度紧张而表现不佳。例如，初次或早期性经历因紧张导致失败，若遭到伴侣的讽刺挖苦，或是在与性工作者发生关系后产生后悔、恐慌（如害怕染病），这些不良体验都会对性功能产生负面影响。
3.  **缺乏正确的性知识或自信心**。这一点尤为普遍和严重。许多男性担心自己的阴茎尺寸过小，无法满足伴侣。加之欧美色情影片中对特定身体部位的夸张渲染，让许多亚洲男性产生更深的自我怀疑和自卑。尽管阴茎是供女性使用的，但男性往往比女性更在意其尺寸。一个男性在其他方面（如身材、长相、事业）被贬低或许不会动怒，但若被嘲讽阴茎小，则很可能“拍案而起”。

上一节我们分析了导致性焦虑的心理原因，本节中我们来看看科学数据如何为我们解惑。我们将用实验数据说话，告诉你男性阴茎的真实平均尺寸，以及女性对此的真实看法。

首先，我们来看阴茎的平均尺寸。2015年，英国《国际泌尿学期刊》刊登了一篇论文，该论文分析了17份学术研究的数据，样本覆盖了全球各地的**15,521**名男性。这是一个超大样本，且数据采集遵循了标准测量程序，十分严谨。

研究得出的结果是：在勃起状态下，阴茎的**平均长度为13.12厘米**，**平均周长为11.66厘米**。

根据数据制成的长度百分位数图显示，阴茎的长度差距并不显著。**16厘米以上**的长度就已超过了95%的人，而**10厘米左右**的长度则处于小于5%的区间。

因此，大部分人的阴茎长度都在**10到16厘米**之间。上述数据包含相当比例的白种人。那么亚洲男性呢？

亚洲男性的阴茎，在松弛状态下长度约为**6到8厘米**，勃起长度约为**10到12厘米**，周长约为**7到10厘米**。

所以，不要轻信“18厘米”之类的夸张说法。正如我们常说的，**“打得准”比“打得远”更重要**。靶子在10米外，一炮打到100米外毫无用处。请以科学数据为准，避免不必要的自卑。

当然，作为亚洲人或中国人，如果低于平均值就一定不正常吗？也不一定。我国著名专家梅骅教授指出，阴茎勃起后短于**9.5厘米**只能算偏短，未必不正常。目前医学上认为，真正的阴茎短小是指长度小于正常人平均值**2.5厘米以上**。也就是说，勃起后如果短于**7.5厘米**，可能确实有些异常（也有观点认为是小于7厘米）。请注意，这里讨论的都是**勃起长度**。

此外，研究指出，目前并未发现阴茎尺寸与身高、手、鼻子、脚等其他身体特征之间存在明确关联。“鼻子大或手指长就代表阴茎大”等说法，均缺乏科学依据，不必采信。

然而，讨论阴茎尺寸本身意义有限。真正关键的是与之配套的**女性阴道**。

性学研究报告指出，女性的敏感点通常位于阴道口内**3-5厘米**处。刺激到最关键的部位并获得高潮，**6厘米的阴茎就已足够**。当然，女性阴道的长度也有差异。

大多数女性阴道在未兴奋时长度在**7到10厘米**之间。在充分的性兴奋状态下，阴道会延展，但其最常能舒适容纳的阴茎长度一般不超过**20厘米**。因此，过长的阴茎（如22厘米）并无实际用处。

而且，阴茎过大容易导致一个实际问题：**充血不足**。充血不足的直接后果就是**硬度不够**，这一点至关重要。

那么，从女性角度出发，她们喜欢多大的阴茎呢？我们也有调查数据。

一项针对110位年龄在18至34岁女性的调查显示（其中96%为异性恋），**89%** 的女性对伴侣的阴茎尺寸感到满意。当被问及伴侣尺寸时，**56%** 的人选择了“平均水平”，**33%** 的人认为伴侣的尺寸“偏大”。结合男性的实际尺寸数据，不难推断出，这89%感到满意的女性，其伴侣的阴茎长度绝大多数不会超过18厘米。

因此，女性并不真正需要特别大的阴茎，**10到16厘米**已经完全够用。说实话，与其浪费时间纠结尺寸大小，不如多练习技巧、学习不同姿势、丰富前戏内容，这些更能让女性获得愉悦的体验。
