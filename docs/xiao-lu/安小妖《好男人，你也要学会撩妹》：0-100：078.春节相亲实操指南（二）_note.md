## 相亲前的筛选与联系

上一节我们介绍了应对春节相亲的基本心态，本节中我们来看看具体如何操作。当长辈为你介绍相亲对象时，不要慌张，这相当于有人为你初步筛选了资源。你的任务是进行二次精准筛选。

以下是进行筛选的两个核心步骤：

1.  **初步筛选（合眼缘）**：首先从介绍人处了解姑娘的基本情况并查看照片。看照片的目的并非以貌取人，而是确保双方在初步印象上“合眼缘”。公式可表示为：`初步通过 = 基本信息 + 合眼缘度`。
2.  **深度筛选（基本信息匹配）**：如果初步印象通过，则进一步向介绍人了解对方的年龄、工作、性格及家庭情况等，从现实层面进行二次筛选。公式可表示为：`深度通过 = 初步通过 && (价值观契合度 + 生活基础匹配度)`。

如果以上两轮筛选均通过，即可向介绍人索要对方的联系方式（电话或微信）。加上微信后，应进行简短友好的自我介绍，并**快速邀约线下见面**。因为相亲目的明确，线上沟通宜简不宜繁，核心是推进到线下约会。

## 高效安排约会日程

春节假期短暂，而相亲对象可能不止一个。因此，必须对约会进行高效规划，避免时间冲突，实现“短时间内的多资源利用”。

你需要对有意向见面的姑娘进行排序，并根据兴趣度紧凑安排约会。一天内可安排2-3场约会，分别对应午饭、下午茶和晚饭时段。

以下是安排日程的具体策略示例：

假设有A、B、C三位姑娘，你的兴趣度排序为：B > C > A。
*   **兴趣度最高的B**：安排**晚饭**。因为晚餐后有机会进行延伸活动（如看电影、散步），能创造更多相处时间。代码逻辑可表示为：`if(兴趣度 == 最高) { 时间段 = “晚饭”; }`。
*   **兴趣度次高的C**：安排**午饭**。可以一起喝茶聊天，共进午餐，之后可礼貌结束，转场。
*   **兴趣度最低的A**：安排**下午茶**。用时较短，成本可控。代码逻辑可表示为：`else if(兴趣度 == 最低) { 时间段 = “下午茶”; }`。

**关键提醒**：若一天内安排多场约会，**必须提前制定详细时间表**，预留充足的转场时间（建议半小时至一小时），以防意外。最稳妥的方案是一天安排两场（午饭和晚饭），这样时间更充裕。

## 约会当天的执行要点

当你安排好日程后，需以积极心态赴约。相亲从见面起就是一场相互评估，细节决定成败。

以下是约会当天必须注意的几个核心环节：

*   **尊重对方**：约会时间、地点、餐厅的选择，应主动征求并考虑女生的意见。这能体现你的素养和尊重，是重要的加分项。
*   **形象管理**：务必重视个人形象。春节期间应打扮得精神抖擞、干净利落，良好的第一印象是成功的一半。
*   **保持风度**：切勿迟到。在交谈中应展现稳重、得体的一面。心态上保持平等，不卑不亢。公式为：`得体表现 = 尊重 + 真诚 + 自信`。
*   **自然交流**：放松心态，如同会见老朋友。自然展现真实自我，聊聊生活、工作和感情观即可，避免过度紧张或刻意表现。

## 约会后的妥善处理

约会结束后，应做到有始有终，根据感觉采取不同策略。

以下是约会后的两种处理方式：

1.  **感觉不佳**：若觉得对方不吸引你，出于礼貌，应向介绍人说明并感谢其好意，为此次相亲画上句号。
2.  **感觉良好**：若对姑娘有好感，应主动保持联系，争取进一步发展的机会。
