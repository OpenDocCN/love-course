# 安小妖《好男人，你也要学会撩妹》：230.3个提升情商的小技巧，不止教你撩妹，还能教你挣钱_note

在本节课中，我们将要学习三个实用的情商提升技巧。这些技巧不仅能帮助你在人际交往中更受欢迎，还能在追求事业成功和个人成长的道路上助你一臂之力。

---

情商常被拿来与智商比较。如果说一个人智商低，或许无伤大雅，甚至可能让人觉得单纯。但如果说一个人情商低，则往往意味着他在人际交往中存在严重问题，容易让人敬而远之。与成年后趋于稳定的智商不同，情商是可以通过学习和练习来提升的。高情商能让你像黄渤一样机智幽默，像李健一样充满魅力，或像蔡康永一样言语富有吸引力。更重要的是，情商的提升不仅能改善人际关系，还能为你的职业发展和财富积累带来积极影响。

上一段我们了解了情商的重要性，本节中我们来看看第一个核心技巧。

## 一、学会划定恰当的心理界限 🚧

在人际交往中，这是一项至关重要的技能。中国社会的关系往往界限模糊，掺杂着许多人情世故。有些人误以为界限不明是好事，能让人相处更随意。然而，这恰恰是许多麻烦和互相伤害的根源。长期界限不清的人，容易陷入病态焦虑，因为他们能感受到问题，却找不到问题的根源。

随着社会向更资本化、责任明确的方向发展，建立边界感变得尤为重要。因此，我们需要掌握一个原则：**让生活的回归生活，让工作的回归工作**。

以下是实践这一原则的具体方法：

*   **大方做人，仔细做事**：即使是关系亲密的朋友，涉及金钱往来时，也应明确细节。例如，借钱时应立下借据，写明金额、归还时间和方式。对方肯帮忙是“人情”，但具体事宜需要“分明”。
*   **作为被帮助者应主动明确**：如果你是接受帮助的一方，应主动提出明确条款，让对方安心，这体现了你的理解和担当。
*   **在各自领域努力**：明确界限后，双方都能在各自的角色（如朋友或同事）中更专注、更舒适地相处，关系反而会更简单、健康。

掌握了人际边界后，我们还需要管理好内在的情绪。接下来，我们看看如何在情绪激动时保持冷静。

## 二、找到适合自己的冷静方法 ❄️

高情商者能在压力下控制情绪，让血液流向大脑，进行理智思考。而低情商者则可能让血液流向四肢，陷入冲动、暴躁的状态，让事情变得更糟。科学研究证实，压力会使血液离开大脑皮层，导致行为失常，让动物本能主导行动。

那么，如何解决这个问题呢？关键在于在感觉快要失去理智时，采用有效方法让自己平静下来。

一个实用的指标是监测**心率**。当心跳超过**每分钟100次**时，意味着情绪可能已不稳定，需要调整。可以使用运动手环等设备进行监测。

如果需要调整，可以尝试以下方法：

*   **离开当前环境**：摆脱持续刺激你的人、事、物，是冷静的第一步。
*   **进行正确的深呼吸**：许多人觉得深呼吸无效，是因为方法不对。正确做法是：用**鼻子**深深吸气，让空气充满肺部，然后用鼻子缓慢呼气，**呼气时间应长于吸气时间**。保持节奏约一分钟，能有效放松身体。
*   **尝试水疗法**：对于持续性的焦虑（如工作压力），洗热水澡或定期泡温泉有助于舒缓情绪。
*   **尝试“艾登法”**：情绪糟糕时，将指尖放在眉毛上方和额头，大拇指按着太阳穴，同时进行深呼吸。据说几分钟后血液会回流大脑皮层，帮助冷静思考。

学会了管理情绪，我们还需要改变面对问题的习惯模式。最后，我们探讨如何用行动替代无效的抱怨。

## 三、用行动替代抱怨 🛠️

当遇到令人不满的情形时，许多人会陷入无休止的抱怨。然而，抱怨只会消耗时间和精力，无法改变任何结果，也不能让你真正好受。虽然向朋友倾诉可能获得共情与短暂安慰，但问题的本质依然存在。

压力虽然令人不适，但它也是促使你改变的力量。一旦停止抱怨，开始行动去解决问题，你的能力就会得到提升。

以下是面对问题时应有的思路：

1.  **停止抱怨，自我提问**：我是继续忍受现状，还是尝试改变？
2.  **接受压力的积极作用**：将压力视为推动你前进的动力。
3.  **下定决心行动**：如果情况确实需要改变，那么行动是唯一有效的途径。人生就是一个不断解决问题的过程，在这个过程中，你的抗压能力和问题解决能力都会不断增强。

---

本节课中我们一起学习了三个提升情商的核心技巧：**建立清晰的心理界限**、**掌握快速冷静的方法**以及**用积极行动取代无效抱怨**。这些技巧相辅相成，从界定外部关系到管理内部情绪，再到采取有效行动，为你提供了一套提升人际效能与个人能力的实用指南。坚持练习，你不仅能成为更受欢迎的人，也能在事业和生活中更加游刃有余。
