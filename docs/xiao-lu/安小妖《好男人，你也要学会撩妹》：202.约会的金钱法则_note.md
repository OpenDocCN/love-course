# 安小妖《好男人，你也要学会撩妹》：202.约会的金钱法则_note

在本节课中，我们将要学习约会中的四条核心金钱法则。这些规则旨在帮助你更高效地推进关系，避免因金钱问题造成社交尴尬，从而更快地赢得心仪女生的好感。

## 第一条：争取跳过晚餐环节 🍽️

上一节我们介绍了课程概述，本节中我们来看看第一条具体规则：约会应争取跳过晚餐环节。

一起吃晚餐是一个非常老套的技巧。对方很可能已有过数百次约会晚餐的经验。如果你在追求她时，却做和其他人完全一样的事情，这对你没有任何帮助。

有一条非常有趣的规律：如果你在追女生时，选择的晚餐越上档次、越高级，那么你等待她下次跟你约会的时间就会越长。你的目标是快速成为她的男朋友。然而，约女生吃饭是一种建立男朋友感觉相对较慢的方式。

约会晚餐时，你们很可能是面对面坐着。这种坐姿完全剥夺了进行身体触碰的机会。有一个简单的道理：你不能说服一个女人来跟你亲密。主动升高关系、进行亲密动作非常重要。但晚餐会极大地考验你的聊天技巧，同时剥夺肢体接触的机会。因此，这是一个非常不好的选择。

吃晚餐也没有其他娱乐来源，焦点完全在你身上。所以，通过吃晚饭来追女生确实很难。

## 第二条：安排有趣而非昂贵的活动 🎭

了解了跳过晚餐的重要性后，我们来看看如何安排约会内容。第二条规则是：你要安排那些有趣、不那么贵的活动，而不是安排一些非常昂贵但仅有一点小优惠的活动。

你完全有更多更好的选择。以下是部分活动建议：

*   参观博物馆、水族馆、动物园、美术馆。
*   观看现场音乐表演、话剧或喜剧。
*   进行徒步旅行或野炊。

这些都是很有意思的共同经历，更容易让人产生感觉。哪怕是分享一份你们自己做的小甜点，也是一个有意思的活动。有的活动甚至一项就能打发好几个小时的时光，效率很高。

有一条经验要告诉大家：将几个简单的活动安排在一起，效果要比安排一个单一的长时间活动好很多。

## 第三条：不要主动提出AA制 💳

安排好了活动，接下来不可避免会涉及消费。第三条规则是：不要试图主动提出AA制来付款。

如果女生在约会中坚持要把她那一份钱付给你，那你就可以顺着她。但你要确保她是真的这么想，而不是出于礼貌的谦让。

感情与金钱的界限到底在哪里？我们应该记住一点：不要让“谁来买单”这个问题造成社交尴尬。花钱通常不会直接帮助你得到什么。但花钱太多或太少，就会给你造成一些困难和伤害。这是一个需要去避免的问题，而不是一个制造吸引的工具。
