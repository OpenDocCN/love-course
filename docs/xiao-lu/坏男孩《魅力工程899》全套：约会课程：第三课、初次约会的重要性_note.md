## 第一印象的构成

上一节我们介绍了课程概述，本节中我们来看看第一印象的具体构成。研究表明，在初次约会时，你留给对方的第一印象，大约55%取决于你的外表和肢体语言动作。

**公式：第一印象 ≈ 55% 外表与肢体语言 + 38% 讲话技巧 + 7% 说话内容**

讲话技巧占38%，而说话内容仅占7%。由此可见，肢体语言在初次约会中占据至关重要的地位。

## 约会注意事项

以下是初次约会时需要注意的几个关键方面。

### 1. 肢体语言

肢体语言是传递自信和放松感的重要方式。合理的身体姿态能让你显得更加自然和吸引人。

**代码示例：理想坐姿**
```
坐姿：
- 身体稍微向后倾斜，保持放松
- 肩部自然下垂，不要耸肩
- 胸部微微向前，但不要过度挺胸
- 双腿自然分开，上身坐直
- 避免双手交叉在胸前
```

首先，注意你的身体姿态。假如你在咖啡厅与人谈话，坐在对方正对面时，身体可以稍微向后倾斜。这种放松的姿态能让你靠在椅背上，从而快速进入放松状态。

同时，你可以加入一些特殊的小动作，例如用手指轻轻划一划眉毛。这些动作不仅能让你显得更酷，还能帮助你与对方开启新的话题。通过这些动作，你也在向对方展示你的高价值。

你要时刻保持肩部放松并自然下垂，避免耸肩或过度紧张。另一个技巧是将肩往后靠，同时让胸部微微向前。注意不要将胸部挺得过高，以免肩膀无法自然靠后。

接下来，注意你的腿和脚。正常走路时，只要不过快，就能显示出自信。最大的问题在于坐下时的姿势。有些人一坐下来就翘起二郎腿，这种动作容易显得不够自信。此外，避免双腿不自然地颤抖或抖腿。

合理的坐姿是双腿自然分开一点，上身坐直。双手不要交叉在胸前，因为这种姿势会暗示你缺乏安全感，或是在刻意与他人保持距离。

### 2. 眼神接触

眼神接触可能是身体语言中最重要的一步。它能有效传递情感和亲近感。

**公式：眼神接触时间 ≈ 50% 倾听时间 + 90% 说话时间**

在你50%的倾听时间和90%的说话时间里，都应保持眼神接触。任何风情万种的人都清楚，眼神是一种非常有效的情感表达手段。

当两个不熟悉的人相对而坐时，注视对方的眼睛远比注视其他部位更容易让人感到亲近。这是因为相互注视时，大脑中与接收奖励信息有关的区域会受到刺激，从而使对方产生好感。

如果对方没有回应你的眼神或躲避你的眼神，那么从一开始你就占据了上风。

### 3. 共同经历恐惧

在第一次约会时，共同经历一次恐惧能快速拉近两人的距离。

**示例：共同活动**
- 一起观看悬疑或紧张的电影
- 一起去蹦极或坐过山车

没有什么比极端局面更容易使两个人走到一起。与观看平淡无奇电影的情侣相比，一起观看悬疑或紧张影片的情侣更容易产生爱慕之情。此外，一起去蹦极或坐过山车等活动，能因为恐惧而增加肾上腺素浓度，从而激发出爱意，让两人产生情绪共振。

### 4. 合适的背景音乐

背景音乐在约会中也扮演着重要角色。

**研究显示：**
- 轻柔的音乐更容易使女性对男性产生好感
- 爵士乐或完全寂静的环境可能产生相反的效果

美国的心理专家研究显示，轻柔的音乐更容易使女性对男性产生好感，而爵士乐或完全寂静的环境则可能产生相反的效果。

### 5. 一起大笑

爱情与欢乐总是相伴而行。共同体验欢乐会在生疏的人之间营造一种亲近感。

约会时尝试讲些笑话，爱情很可能就伴随笑声而生。

## 提升约会效率

刚才我们介绍了初次约会时需要注意的要点，接下来我们来看看如何提升约会效率。相信大家都遇到过这样的情况：在微信上与对方相谈甚欢，但约会后对方态度变得不冷不热，难以再次邀约。

在当今快餐时代，约会需要的是效率而非频率。以下是两种常见情况的分析。

### 情况一：微信互动良好，约会表现不佳

例如，男生将约会地点定在一家餐厅，但用餐时表现随意，吃相难看，边吃边说话，甚至喷出饭粒。这种形象容易引起对方反感，导致约会失败。

### 情况二：微信互动较少，约会表现良好

例如，男生在约会时风趣幽默，肢体语言和表达都有一定水准。用餐后，他邀请对方去唱歌，直至后半夜才回家。尽管约会体验看似成功，但这种操作可能暴露出过强的需求感。

对于与你没有一定熟悉或暧昧程度的人而言，初次约会就进行多次转场，只会暴露你的强烈需求感。除非对方目的也是一夜情，否则这种需求感会触发对方的自我防御机制，导致对方刻意与你保持距离。

因此，建议在第一次见面时，即使你充满热情，也要控制好自己的需求感。

## 后续操作的重要性

后续操作是决定对方对你态度和印象的下限的重要环节。如果处理得当，即使初次约会体验不佳，你也可能有第二次弥补的机会。若处理不好，对方对你的态度可能发生180度转变，让你丧失后续希望。

约会的关键在于尊重并了解对方的想法，换位思考，为对方营造舒适而愉悦的感觉。这才是让对方对你心动的根本前提。前提做好了，之后的事情也就水到渠成了。
