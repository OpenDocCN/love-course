# 坏男孩《魅力工程899》全套：型男计划：9衬衫的选择：3_note

在本节课中，我们将要学习如何正确地折叠衬衫。掌握这项技能不仅能保持衬衫的平整，还能在出差或旅行时高效利用行李箱空间。

上一节我们介绍了衬衫的材质与版型，本节中我们来看看如何通过正确的折叠方法来护理你的衬衫。

## 衬衫折叠步骤详解

首先，把衬衫放在一个水平面上铺平。一定要把褶皱都弄得平直。然后将衬衫的全部纽扣都扣上，最上面的一颗和最下面的一颗都需要扣上。

接着，将衬衫翻转一个面，使其背面朝上。把两只袖子平摊在衬衫身体的两侧。

以下是放置支撑物的步骤：
*   准备一张A4大小的纸或者一本杂志。
*   将其放在衬衫领口下方的位置。

以下是折叠袖子的步骤：
*   将一只袖子往里折，使其边缘与你放入的纸张或杂志的边缘对齐。
*   再将叠过去的袖子再次叠回来，使其与衬衫身体的侧边对齐。
*   另一侧的袖子重复同样的动作。

紧接着，将衬衫下摆一侧往上折叠到纸张或者杂志的下边缘。

最后，将整件衬衫翻转回正面。整理衬衫的领口，并小心地抽取夹在衬衫中间的纸张或者杂志。至此，衬衫的折叠就完成了。

## 出差旅行专用折叠法

如果你经常出差，可以采用另一种方法。将衬衫简单折叠后放在一个有硬度的纸盒里，衬衫会自然保持平整。

以下是具体步骤：
*   将衬衫正面朝上放置，抚平前襟与后片。
*   将两只衣袖沿肩线向后平整展开。
*   将衬衫下部分（约1/3处）向后折叠，叠在上部分下方。
*   将两侧衣袖交叉叠放在胸口位置。
*   直接将这样叠好的一打衬衫放入旅行箱即可。

本节课中我们一起学习了两种实用的衬衫折叠方法：一种是使用辅助物获得更平整的效果，另一种是适合出差旅行的快速折叠法。掌握这些技巧，能让你的衬衫时刻保持有型。
