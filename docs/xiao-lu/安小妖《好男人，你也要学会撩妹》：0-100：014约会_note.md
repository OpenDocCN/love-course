有些女生心思非常细腻且敏感。对她们而言，聊天成本很低，只要你不是特别差，她们都能和你愉快地聊天。

而且，这类女生一旦接受了你的邀约，在你们感情发展的中后期，你就会比较省力了。

剩下的，对你而言就是苦尽甘来。对待这样的女生，关键是要有足够的耐心，不断地聊，不断地约。

上一节我们了解了这类女生的心理，本节中我们来看看邀约前要做的第一个重要铺垫：以稍微严肃的态度聊情感话题。

你需要把双方的过往经历、现状以及对感情的态度都了解清楚。以此来判断你提出邀约的时机是否合适。

因此，在聊天中深入探讨两性情感话题是非常重要的。

通常，两人认识的第一阶段，聊天都围绕浅层次话题，例如吃喝玩乐、衣食住行、生活见闻等。

很多男生在这个阶段都聊得非常好。但后来会发现，话题聊不动了，缺乏后劲，再也找不到最初那种心花怒放的感觉。

原因就在于你一直在第一阶段停滞不前。如果你不去涉及情感层面，那么等待你的就只有关系的倒退。

有的男生认为，一直保持暧昧聊天却不涉及感情，才能显示自己会聊天、对女生不在乎，营造若即若离的感觉，显得自己很厉害。

如果你不能提供这些，她对你的兴趣和期待就会开始减少，直到消失。

所以，在和女生互动的过程中，一定要把情感话题说开。

在邀约女生之前，另一个要做的铺垫是：**提前透露你的约会计划**，让女生知道要去哪里、做什么。

你总要给女生基本的知情权和选择权。可能你想保留神秘感和惊喜感，所以不告诉女生约会内容。

其实，**惊喜绝对不适合在你们认识的初期使用**。惊喜应该是当你们感情进入平淡期后，用来哄女生开心的调味品。

认识前期，你的重点是让女生感到**安全**。在安全的基础上，才可以考虑制造惊喜。没有安全感时的“惊喜”，那不叫惊喜，叫搞事情、叫不靠谱。

如果你觉得自己特别有才，必须让女生觉得你是个有趣的人，我推荐你使用 **“转场”** 这个技巧。

这时，你的头上就会出现一种叫做 **“男朋友的光环效应”**。

但是，这种效果一般男生难以做到。第一，你要知道附近有哪些好玩、有趣的地方。第二，你要自己预先体验过。如果你自己都两眼一抹黑，那你一定无法给女生依赖感。

还有一个约不出女生的原因：**你的朋友圈暴露了低价值**。

有的男生自身条件不错，但容易以自我为中心，特别爱在朋友圈抱怨自己遇到的不顺心或讨厌的人。

你偶尔这样，别人可以理解，甚至可能来安慰你。但如果你老是抱怨，看到的人就会觉得，可能不是你的运气差或遇到的人不好，而是你自身就有问题。

所以，你要调整好心态。不要以为自己条件优秀，就可以表现出非常情绪化、任性、自私的一面，并以为女生都应该包容你。

其实，优质女孩的择偶标准很简单：**“我喜欢”**。她们根本不会迁就你的坏脾气和自私。好的条件不等于就是优质男生。你还需要具备与自身条件相匹配的良好修养和品性，才能成为最好的自己。

如果不是特别有吸引力的事情，她们一般不爱出门。而且据我所知，大部分女生出门前都要洗澡。如果遇到重要场合或约会，还要化复杂的妆。你从来没见过一个女生连续两天穿同样的衣服吧？

所以，加上选衣服、选鞋子的时间，出门约会对女生来说，时间成本和精力成本都非常高。如果你一个邀约就把她支到10公里开外，她不拒绝你拒绝谁呢？

同样的道理，除了确定位置，**提前确定女生的时间**也非常重要。

以下是几种常见情况：

*   如果她是大学生，时间会比较空闲，约在下午三四点都可以。

我上面所说的这些邀约前要进行的铺垫，既有比较宏观的方面，比如确定女生的情感状态、摸清女生的情感态度；也有一些比较细节的地方，比如约会地点不要离女生太远、要让她知道基本的约会安排等。

当你注意到这些点之后，女生接受你邀约的成功率就会变得更高。
