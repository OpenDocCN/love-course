# 【魅力工程】2.0：维持长期关系 🫂 第1节：长期关系必须要做的事

在本节课中，我们将要学习如何经营一段健康的长期关系。我们将探讨长期关系的核心需求、常见问题以及维持关系的关键方法。

一段不太融洽的长期关系，如果你在其中发现自己越来越爱对方，而主动权却离你越来越远，那么本节课的内容将帮助你找到问题的根源。

在我看来，一段关系确立三个月以上，才能算是长期关系。现在很多男性只擅长短期关系，认为学习技巧只是为了最终发生亲密关系。作为女性，我不评论他人的道德，只想提醒一句：不忘初心。

不要忘记你最初的目的。如果你已经学过很多技巧，有过很多段关系，不妨问问自己：你的初心是什么？现在还在吗？男女感情最终的核心，还是长期关系。这也是绝大多数来学习情感艺术技巧的朋友的初衷。

如果你正在恋爱，或曾有一段不太融洽的长期关系，并且发现你越来越爱对方，主动权却越来越远，慢慢地，对方变得不那么爱你、烦你、不想跟你待在一起，没有一开始那么甜蜜。

甚至频繁出现争吵，最终分手甚至讨厌你。那么在这节课上，你一定能找到问题所在。人生命中有两个基本的心理需求：一个是在所爱之人心中占据首位；另一个最基本的心理需求，是被人无条件接纳的需要。

可惜，这既是长期关系中最迷人之处，又正是日后关系最大困难的地方。我想到很喜欢的一句话：不要把自己当作神，也不要把对方当作神。我想唯有神才能彻底满足人心这两大需求。爱情却能带来一个幻觉。

让人以为爱人就是那位可以满足所有需要的神。但是，如果你把爱人当神，把对方当成你的唯一，那么你必将失望。接下来，我想聊聊恋爱中，情侣在长期关系上会出现的问题以及解决建议。

以下是维持长期关系必须做到的八件事：

**1. 控制火花的弥漫速度**
在爱情的火花期，需要避免一开始就如胶似漆，24小时都腻在一起，或一开始就倾诉所有。如果你每分每秒都跟他待在一起，并互相倾诉彼此的过去，这是很危险的事情。因为这样你在对方眼中会变得透明。

慢慢地，他就会对你不再感兴趣，你就失去了吸引力。太快交心会减少你“奖赏”对方的能力，从而降低你的吸引力。一个人能给别人最好的礼物，是你内心深处真正的想法与感受。但你不要把这珍贵的人生果实随便给人。

这些珍贵的心灵礼物，只能给那些通过你“安全性测试”、渐渐赢得你信任的人。所以请你务必记住一点：**心灵亲密的程度应该与交往的阶段成正比**。

**2. 如何“同生共死”地延续一段关系**
刚刚说过了，切莫一开始就付出一切。但是，你可以很有技巧地去维持一段关系，这个技巧就是“同生共死”。也许“同生共死”在前期吸引比后期维持更有效，但也不容忽视它在后期的维系作用。

在这里我只想简略提一下“同生共死”在后期的运作方法。具体的前期与后期操作方法我们有机会再聊。当你们在一起久了，就会不自觉地一起做一些事情：一起吃饭、逛街、整理房间、买东西、旅游。

“一起”就是只有你和他的独家记忆。当你与他的回忆足够美好、足够快乐，他是不会轻易离开你的。就算有一天分道扬镳，他也会感激遇上你并与你有独家记忆。这就是“同生共死”的实践基础。

**3. 必须学会相互投资**
这里的投资并不单指钱财上的投资，更多的是感情上的投资。所谓的互相投资，就是你与他一起的投入。当他对你的投资越多，那么他就越离不开你。但在长期关系中，你一定要有投资。

不要一直让女生投资。她投资而你不投资，就会让对方觉得没有安全感，认为你不在乎她。

**4. 独一得当**
在长期关系中，必须懂得“独”，也必须懂得“一”，并在这二者之间弹性调整。“独”的含义是两个人要给彼此预留空间，最好能够有独立的社交圈，能够彼此相互尊重。

“一”的含义是两个人要互相依靠，给予彼此安全感。比方说，她感冒了你要为她买药，她遇上事了你要为她出主意。

**5. 实质紧扣**
实质紧扣也叫“尾声的承诺”，就是你要确定你和她是要走同一条路，你们两人愿意一辈子生活在一起。并且两个人的期待必须明确：你们是仅仅想发展短期关系，还是想要一直走下去？

这可以有效避免对方认为你们只是“试一试”的关系，而你自己却认为她已经离不开你了。

**6. 两性亲密互动**
长期关系里必然要有两性的亲密互动，也就是亲密关系。在你们两人已经坦诚相见之后，你和她的缺点、性格上的缺陷，以及两人潜在的冲突，也会慢慢显现。这也是很多婚前同居者，临近结婚却闹掰的问题所在。

对于这个问题，我接下来将会尽量详细地表达我的观点与建议。作为女性，我不想过多讲述关于亲密关系的内容，只是想从我的角度说明：两性互动中，愉悦的亲密关系对于情感的互动有促进及加分的作用。

**7. 保持神秘感**
保持神秘感一定是恋爱“保险剂”中特别重要的组成部分。保持神秘感也需要区分初期吸引与长期关系。在初期吸引时，你要做到尽量避免太快把自己过去的软弱、隐私、心中的不安全部告诉对方。

在一段长期关系中，保留某种神秘在里面，是保持新鲜感特别重要的方法。长期关系就像两个圆圈，中间有很多交集，交集可大可小，随时间迁移而变化。当两人受到压力时，交集的地方比较多。

当压力比较小时，两个人可以向外去探索。每个人可以有若干百分比是单属于自己的秘密。但随着彼此信任感增加，彼此可以更加透明，只是不能勉强。所以最理想的方式是：建立良好的爱的关系。

而透明只是这个关系的结果。

**8. 如何应对长期关系中的吵架**
前些天听闺蜜说了这样一件事：一对交往了近两年的男女朋友，因为女方去酒吧玩没有提前告知男方，男方知道后与女方大吵大闹，最后以分手收场。我想说这个故事，是因为很多恩爱情侣，仅仅因为一时情绪冲突就会分道扬镳。

情绪分为三种：第一，情绪高涨；第二，情绪一般；第三，情绪差。长期关系中会出现各种各样的情绪。当对方出现愤怒、你们吵架时，你应该怎么做？如果你在火山口上对他发火，他也对你发火，比如他和你闹分手。

如果你比他更严重，然后，估计就没有然后了。在一段长期关系中，当我们情绪不好时，最好不要向伴侣完全宣泄情绪。要宣泄，可以有技巧地宣泄。我一直告诫身边的男性朋友，一定要在伴侣前控制好自己的情绪。

不要在情绪不好时，对他大吼大叫、数落他，或无脑地和他对骂。其实最好的解决方法是：你可以把自己的情绪放一放，心情平和一点去和他交流。只要把他的情绪状态转变过来，让他脱离难受、愤怒、生气等负面情绪。

然后你再抱一下他，和他开个玩笑就好了，他自己也就恢复了。当女生情绪好的时候，所有事情都会迎刃而解。之前有很多男人告诉我，他们虽然智商不高，但是情商很高。而我每每听到他们这样说，也只是笑笑。

我们经常所说的情商，无非就是五大要点：了解自身情绪、管理情绪、自我激励、识别他人情绪、处理人际关系。由此看来，所谓的情商，绝大部分都是在说能够有效地控制自己的情绪。

一时让他开心，一时让他失落，让他的情绪坐过山车，他才能更好地爱上你。而情商永远比智商重要，无论是在感情方面、人际关系方面还是事业方面，都能很好地处理问题。

**9. 爱情持久的燃料与氧气**
*   **一、爱情的燃料**：对自己保持有信心，让自己变成一个有意思、有吸引力的人。有空的时候，利用这些时间去提升自己、改变自己。
*   **二、爱情的氧气**：两个人需要空间，不要老黏在一起，也不需要什么都告诉对方，保持一点神秘感。爱情最怕的就是：开始的时候老黏在一起，但过了一段时间，一个人变成“空间追求者”，另一人则是“亲密关系追逐者”。

这也是很多长期关系会出现的一个问题：一开始主动权在你这里，后来慢慢地你越来越黏对方，越来越想天天和对方待在一起，慢慢就改变了游戏规则，主动权就到了对方手里。

本节课中，我们一起学习了长期关系的核心心理需求，并详细探讨了维持健康长期关系必须做到的九件事：控制节奏、创造共同经历、相互投资、平衡独立与依赖、明确承诺、重视亲密互动、保持神秘感、智慧处理冲突，以及为关系持续提供“燃料”和“氧气”。掌握这些原则，有助于你建立并维护一段稳固而美好的长期关系。
