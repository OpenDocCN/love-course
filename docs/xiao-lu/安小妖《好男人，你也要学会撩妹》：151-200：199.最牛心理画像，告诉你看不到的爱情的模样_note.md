许多人认为爱情是抽象的，只听说过却从未见过其具体形态。本节将使用心理画像的方法，为你描绘爱情的真实模样。我们将按照从低到高的顺序，解析爱情的十个阶层。

## 第一层：喜欢 😊

喜欢是爱的基础。它意味着对方拥有某些令你欣赏的特质。

以下是喜欢的两种主要来源：

*   **特质吸引**：例如对方的长相、风度、谈吐或学识吸引了你。
*   **行为吸引**：例如对方的热情、乐于助人、活跃气氛的能力，或对你的关心、体贴与善解人意。

**核心公式**：`喜欢 = 特质吸引 ∪ 行为吸引`

与对方在一起感到愉快，就是喜欢。但必须明确：**喜欢并不等同于爱**。许多人会混淆两者，错把好感当爱情，最终可能导致自己受到伤害。

## 第二层：依恋 🤗

上一节我们介绍了喜欢，当喜欢的程度加深，便会进入依恋阶段。依恋包含三个关键词：**思念**、**依赖**和**纠缠**。

喜欢一个人会渴望与其共处，分离时则会产生焦虑与不适，希望继续在一起，这便是依恋。

依恋主要表现为两种形式：

*   **思念**：不在一起时会想念对方，反复回忆共处的快乐时光。
*   **冲动**：总是想着去找对方，渴望立刻与其重逢。

当“想见他的冲动”强烈到一定程度，就会直接付诸行动，仿佛没有对方就无法生活，这就形成了**依赖**乃至**纠缠**。

## 第三层：占有 🛡️

当依恋达到一定程度，便会催生占有欲。这是因为如果对方与别人相处时间增多，与你共处的时间就会减少。为了满足自身的依恋与依赖，你便可能不允许对方将时间花费在他人身上。

占有阶段包含五个关键词：**嫉妒**、**控制**、**改造**、**索取**。

以下是占有的主要表现：

*   **嫉妒与吃醋**：若他人占用了“我的时间”，或对方与你在一起时心不在焉，便会引发嫉妒。
*   **控制行为**：为了完全占有对方，可能采取极端行为。这里的“控制”分为两种：
    *   **硬控制**：如跟踪、调查、惩罚、争吵、提要求、限制自由等主动强硬的措施。
    *   **软控制**：如讨好、利诱、使用苦肉计或要挟等被动软性的措施。
*   **改造对方**：尽管总体上喜欢对方，但对其某些方面不满意，于是提出要求、帮助甚至亲自动手改变对方，将其塑造成理想模样。这种改造本质也是一种占有与控制。

占有与控制都属于**索取**，即主动寻求被爱。人们常忙于寻求被爱，却忽略了自身是否具备爱的能力。弗洛姆曾说：“爱是一门艺术。”希望每位寻爱或正在爱的人，都能培养爱的能力，成为创造爱的艺术家，而非仅仅索取被爱。

## 第四层：交易 🤝

从占有中的“软控制”可以自然过渡到交易阶段。交易带有更明显的互换色彩，它有两个关键词：**迎合**和**顺从**。

交易是指为了让对方喜欢我并达到独占目的，我愿意主动做一些事让他开心，使他**自愿**而非被迫地与我在一起。

例如：
```python
# 交易行为举例
action = ["打扮帅气", "迎合喜好", "顺从心意", "关心帮助", "取悦对方"]
for a in action:
    print(f"通过{a}，使对方更愿意与我在一起。")
```

交易强调公平性，即付出应得到相应回报。如果付出多而回报少，则可能退回到“控制”阶段。在恋爱中，适度的交易合乎情理，但若使用极端方式，则需警惕。

诗人伊萨科夫斯基说过：“爱情不是一颗心去敲打另一颗心，而是两颗心共同撞击出火花。”这一点至关重要。

## 第五层：报答 🙏

有两种交易形式不那么赤裸，那便是报答。报答阶段有两个关键词：**感恩**和**情感投资**。

*   **感恩**：这是报答的一种。因为已经从对方那里得到好处，所以想要回报对方。即使对方不求回报，但我的回报能给他带来快乐，使其更愿意与我相处。同时，不报答会让我内心不安，报答则能了却心愿，获得心理平衡。与交易相比，感恩显得不那么功利。
*   **情感投资**：表面看是不求回报，实质是“放长线钓大鱼”。许多父母对孩子的爱便是如此。他们无私付出，一心希望孩子好，但待孩子成年后，可能会“慢慢算账”。当然，父母之爱的根本原因在于孩子是他们生命的延续，爱孩子某种意义上就是爱自己，因此可以不需要回报。父母之爱也可看作是对“孩子可爱”的一种回报。

控制是逼迫对方爱我，交易是让对方自愿爱我。在交易中，我的付出可能并非完全心甘情愿，但**在报答中，我的付出一定是心甘情愿的**。

每一段感情都需要投资，无论是时间、精力还是金钱。不仅在追求期需要投资，在相处后，为了维持长久的爱，更需要源源不断的投资。可惜许多人相处后便觉得爱已稳固，肆意发泄情绪，最终导致关系破裂。实际上，双方可能都是好人，也非故意，只是不懂得如何经营，因此需要学习。
