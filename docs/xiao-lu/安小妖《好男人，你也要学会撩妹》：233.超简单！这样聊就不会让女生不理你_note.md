## 原则二：减少使用问号

上一节我们学习了如何避免连环提问，本节中我们来看看如何减轻对方的回复压力。回答本身带有责任和压力。如果必须询问，句子末尾尽量不要加问号。

我们可以通过一个例子来分析：
*   “你在忙吗？”加上问号，意味着“你有没有时间？我想跟你说点事。”这是通常的表达习惯。
*   “你在忙吗”不加问号，感觉则不同。它像在陈述一件平常事，没有“我要跟你说事”的压力感。

这一点需要仔细体会。如果女生愿意告知其当时状态，我们就能根据情况互动，展开头脑风暴，从而开启一段丰富精彩的对话。
