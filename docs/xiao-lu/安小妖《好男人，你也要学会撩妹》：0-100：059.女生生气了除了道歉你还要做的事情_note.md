## 判断生气类型：真生气还是傲娇

上一节我们介绍了课程主题，本节中我们来看看如何判断女生生气的性质。首先需要区分她是“真生气”还是“傲娇”。

**傲娇**是指女生耍小脾气，例如嘟嘴、翻白眼、垮下脸或说“我生气了”。这通常发生在你开一些无关痛痒的玩笑时，比如“你最近好像圆润了”或“这件衣服不太适合你”。

以下是应对傲娇的方法：
*   立即找补，例如说“我就喜欢你这样，一点都不胖”或“这件衣服衬托不出你的气质”。
*   嘴甜一点，使劲夸她和哄她。
*   将这个过程视为交流感情的好机会，耐心哄她开心。

女生傲娇表明她在意你，这是你们感情更亲密的机会。

## 识别真生气的场景与表现

然而，如果女生是真的生气，情况则不同。女生真生气常由严重事件引发。

真生气的场景包括：
*   忘记她的生日。
*   约会迟到。
*   与其他女生暧昧。
*   欺骗她（例如为上网骗她说要回家）。

## 应对真生气的三个步骤

当你确定她是真生气后，需要遵循以下步骤来解决问题。

### 一、弄清她生气的原因

第一时间弄清楚她为什么生气。女生心思细腻，你可能需要主动察觉或询问。

以下是两种弄清原因的方法：
1.  **当面询问**：态度要好，适当示弱。例如说：“女王大人，我好像又惹你生气了，可我不知道错在哪，请指条明路。”
2.  **求助闺蜜**：如果她不愿说，可以联系她的闺蜜了解情况，因为女生常向闺蜜吐槽。

### 二、无论对错，先认错

弄清原因后，要端正态度认错。核心逻辑是：**你惹她生气了，就是你的错**。

正确的认错方式不是敷衍地说“我错了”，而是：“宝贝我错了，又让你不开心了。”然后引导她说出不满和期望，进行良好沟通。

不要争论对错，先在情绪上安抚她。

### 三、口头认错，行动跟上

认错和承诺必须付诸行动。

行动跟进示例：
*   **因迟到生气**：承诺后，未来务必守时。
*   **因前女友联系生气**：承诺后，彻底断联。
*   **因不关心她生理期生气**：承诺后，记住日子，准备红糖水、暖手袋。

在安抚情绪并沟通后，可以通过送花、小礼物或带她吃美食等方式，进一步抚平她的不快。
