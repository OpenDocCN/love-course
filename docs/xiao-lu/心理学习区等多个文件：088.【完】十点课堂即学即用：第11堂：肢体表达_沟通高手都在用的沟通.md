# 高情商沟通课 第11堂：肢体表达 🚀 沟通高手都在用的沟通助推器

在本节课中，我们将要学习沟通中一个极其重要却常被忽视的组成部分：**肢体语言**。我们将系统地了解肢体语言的核心概念、常见误区以及如何通过SEHART模式，像沟通高手一样运用肢体表达来增强沟通效果，建立信任与亲近感。

## 概述：被低估的沟通力量

沟通的重点不仅仅在于语言表达，**肢体表达更为重要**。美国人类学家阿尔伯特·梅拉宾教授在非语言沟通中指出，只有**7%**的信息是通过语言表达的，而其他**93%**的信息是通过声音、语调和肢体语言来表达的。

**什么是肢体语言？** 它是通过头、眼、颈、手、肘、臂、身、胯、足等人体部位的协调活动来传达思想、形象和情绪的一种沟通方式。例如，我们习惯性地认为：
*   鼓掌表示兴奋。
*   顿足表示生气。
*   搓手表示焦虑。
*   垂头代表沮丧。
*   摊手表示无奈。
*   捶胸代表痛苦。

## 一、快速建立亲近感的万能法则

如果问，如何在与人第一次沟通的5分钟之内，让对方感觉到亲近感？有一套万能的法则：**歪头、微笑、模仿动作**。

上一节我们概述了肢体语言的重要性，本节中我们来看看这套具体的实践方法。

**1. 歪头**
头部稍微向左或向右倾斜，可以使人感觉到温和、友善，也更有吸引力。女性做此动作会让人觉得可爱。而头部非常笔直则会给人一种强势、不可亲近的感觉。

**2. 微笑**
微笑会释放脑内啡，带给别人幸福感，是显著的友好信号。沟通中要展示真诚友好的微笑，特点是嘴角上翘，颧骨上移，眼角出现皱褶。

**3. 模仿动作**
想要快速给人亲近可信的印象，模仿对方的肢体语言是非常简单的方法。以下是具体步骤：
*   **观察**：仔细看对方的坐姿、手势、脚如何摆放。
*   **调整**：慢慢调整成和对方相似的姿势。
*   **同步**：尝试掌握对方的肢体韵律（如点头频率），然后换一个类似但不完全相同的动作。

以下是模仿动作的三个核心技巧：
*   **选择性模仿**：只模仿最自然、符合自身性别的动作。
*   **调整动作强弱**：如果对方手势夸张，你可先做小幅度的，再慢慢调整一致。
*   **延迟模仿**：对方做完动作后，过几秒再模仿，避免引起对方反感。

最重要的是，一旦开始模仿，就必须全心全意地倾听和观察对方，使肢体频率与对方一致，最终达成心理学上的“边缘共振”。大脑中的“振荡器”神经元具有模仿天性，当双方肢体语言趋同时，对方会不由自主地分享更多。

**歪头、微笑、模仿动作**，无论是第一次见客户、约会还是面试，都能轻易给对方留下亲近可信的印象。

## 二、常见的肢体语言误区

然而，如果肢体语言使用不当，反而会适得其反。人们会误解你的肢体语言。了解这些误区至关重要。

上一节我们学习了如何用肢体语言加分，本节中我们来看看哪些肢体语言会扣分。

### （一）沟通中常见的十种错误肢体语言

以下是十种在一般沟通中容易传递负面信息的肢体语言：
1.  **夸张的手势**：暗示你在夸大事实，让人感觉不适。
2.  **频频看时间**：暗示焦虑、自负和不尊重，意味着你有更重要的事。
3.  **语言和面部表情不一致**：让对方产生怀疑，认为你在欺骗。
4.  **坐立不安或摆弄头发**：表示焦虑、精力过剩、自负或注意力不集中。
5.  **无精打采**：是不尊重的表现，显得厌倦、不想待在那里。
6.  **身体与交谈者距离不当**：距离太远显得不感兴趣或不信任；太近（小于50厘米）则让对方没有安全感。
7.  **避开目光接触**：让人怀疑你隐藏事实，也显得对沟通不感兴趣或缺乏自信。
8.  **频繁夸张地点头**：透露出焦虑，对方可能认为你在掩饰不赞成或不理解。
9.  **不停转动眼珠**：是缺乏尊重的表现，可能是个坏习惯，但控制它会带来好效果。
10. **皱眉或其他不开心表情**：会让对方感觉你心烦意乱，把周围的人拒之门外。

### （二）面试中的六种错误肢体语言

工作面试时，得体的身体语言至关重要，否则语言再好，身体语言也可能出卖你。
1.  **摸脸**：触摸鼻子、嘴唇、前额，表示紧张或不诚实。
2.  **晃腿**：传达“我焦虑不安，想尽快逃离”的信息。
3.  **环胸抱肩**：传递封闭、防御或百无聊赖的信号。
4.  **手上的小动作**：摆弄物品显得紧张、笨拙、毫无准备。
5.  **不适度的眼神交流**：不要东张西望，也不要一直盯着对方眼睛或某个部位。应适度转移视线。
6.  **手势乱放**：手势应在“镇定区”（肚脐至锁骨之间），显得沉稳专注。放在其他区域（如摸脸）则不得体。

## 三、肢体语言沟通的核心与结果

肢体语言沟通的核心是**传递正确的沟通信息**，是对语言信息的补充。借助肢体语言，能让人们对沟通信息产生信任感。例如握手：力度过轻显得缺少权威和自信，过重则显得有攻击性。力度应视情况调整，但始终保持坚定。

上一节我们认识了需要避免的错误，本节我们来深入理解肢体语言的作用原理与目标。

肢体语言比口头语言能表达更多信息。理解他人的身体语言是理解其人的重要途径。从他人的目光、表情、动作、姿势以及彼此距离，我们都能感知对方的心理状态。例如，一位男士立正站立、衣着得体、肩膀自然下垂，说明他想展示挺拔姿态以引起注意；如果身体微微前倾倾听，则更能表明好感。

**肢体语言沟通需注意场合**。不同场合有不同表达要求。与不熟的人交流时，肢体语言必须明确，避免误会。在第一次约会或工作面试等场合尤其要重视。例如，女士整理头发在心仪男士面前可能传达好感，但在面试等正式场合则显得自恋或焦虑。

**肢体语言沟通的目标是达成亲切感与信任感**。语言能达成共识，肢体语言则主要消除陌生感。例如，紧握拳头在认可对方观点时可表示支持，但一开始就紧握拳头则表示你并未持开放态度，会让对方紧张。

身体就像一个无法关闭的传送器，时刻传递着我们的心情和状态。人们常说，在面对面交流中，语言只占所表达信息的十分之一，而肢体语言则表达了一半以上的信息。善用肢体语言，你就能提升人际交往质量，从而提高生活质量。

## 四、沟通高手的肢体语言系统：SEHART模式

那么，沟通高手如何巧妙地利用肢体语言呢？让我们系统地学习SEHART模式。

在开始使用SEHART之前，你需要建立三个基本认知：
1.  **理解自己的肢体语言要自然**：即使能控制得很好，也可能看起来很假。动作需要自然，不能像机器人。
2.  **识别自己的肢体语言模式**：有意识地注意自己在与不同人交往、处于不同情绪（生气、紧张、高兴）时，会使用什么不同的动作。
3.  **确保肢体语言与言语一致**：检查你的动作是否让你看起来自信。如果非语言表达和语言一致，沟通会既清楚又富有魅力。

**关注整体效果**，不必追求每个细节到位。接下来，我们从六个方面分解SEHART模式。

### **S - Smile Communication 微笑沟通**

恰到好处的微笑能使人愉悦、获得安慰；不合时宜的微笑则会打击对方、伤害感情。微笑传递的信息量大于语言。

**训练微笑的方法：**
1.  **放松唇部肌肉**：大声、清晰地逐个发出“Do、Re、Mi、Fa”的音节，每个音念三遍，尽量将唇形做到最满。
2.  **锻炼嘴角弧度**：用门牙轻轻咬住筷子，让嘴角对准筷子两边翘起，保持10秒，确保嘴角连线与筷子水平。
3.  **训练保持微笑**：对着镜子，保持最满意的微笑至少30秒。

**微笑的场合感：**
*   **应该微笑的场合**：初次见面、寒暄、问好、说开场白、告辞、邀请、请求、征求意见时。
*   **需要收敛笑容的场合**：谈及严肃话题、表示疑问、恳切表达意见时。时而微笑、时而严肃的表情，比始终微笑更有说服力。

### **E - Eye Communication 眼神沟通**

眼睛是最有力的沟通工具之一。眼神接触能促进情感交流，表达专注与互动。

**凝视范围**：让对方舒服的凝视范围是以对方鼻头到额头中央的区域（三角区）。不要过度直视眼珠超过10秒，会造成压力。

**注意要点**：
*   如果交谈时间长，眼神应有转移，可从三角区扩大到整个面部，避免直勾勾盯着。
*   男士与女士交流，目光宜停在女士胸部以上区域。
*   对群体讲话时，要看到在场每一个人的眼睛。
*   对个人讲话时，适当停顿并与对方眼神接触2秒以上，以示重视。

### **H - Hand Communication 手势沟通**

手是人类最灵活的部分，手势语言丰富且历史悠久。正确使用手势有助于提高沟通效果。

**手势应用规范：**
*   **介绍或指示方向时**：手心向上，四指并拢，大拇指张开，以肘关节为轴，前臂自然上伸。上体微前倾，视线随手势移动。
*   **打招呼、致敬时**：根据场景控制手势力度、大小、速度和时间。
*   **谈到自己时**：用手掌轻按左胸，显得端庄、可信。
*   **禁忌**：不可用拇指指自己鼻尖或用手直接指向他人（含教训意味，极不礼貌）。

**注意事项**：
*   注意手势的**地域文化差异**，同一种手势在不同国家含义可能相反。
*   避免不雅习惯动作，如抠鼻孔、剔牙、摆弄小物件、摸头、搔痒等。

### **A - Attitude Communication 姿态沟通**

身体的姿态能诚实反映心理状态。开放的姿态表示自信、坦诚；抱臂、弓腰等收敛姿态则表现出防备或自卑。

**关键信号**：人们常通过准备起身的姿势（如身体前倾，双手放膝盖或扶手上，移到椅子边缘）来示意希望结束对话。讲话时若发现对方有此姿态，应引起重视并尽快结束讲话。

### **R - Reaction Communication 反馈沟通**

即时的反馈（如点头、说“对”、“没错”）能让对方更积极地沟通下去。没有回应，沟通难以进行。

**注意**：过度附和会让别人觉得是敷衍或不重视。在面试或与领导沟通时，频繁附和可能被当作拍马屁。

### **T - Touch Communication 触碰沟通**

握手、拥抱等触碰能拉近关系，促进共识，向双赢方向发展。对于“感觉型”的人，偶尔触碰对方肩膀或手臂能让关系更亲近。

**重要提醒**：不合适的触碰会让人反感。男性之间身体触碰可能习以为常，但对女性则需格外注意。故意触碰可能引起反感。

## 五、如何恰当地使用肢体语言：行动指南

恰当地使用自己的肢体语言沟通，应该做到以下几点：
1.  **经常自省**：检验自己以往使用的肢体语言是否有效、自然，是否使人产生过误解。
2.  **有意识地运用**：在沟通中主动、有目的地使用合适的肢体语言。
3.  **注意使用情境**：根据不同场合、对象调整肢体语言。
4.  **注意角色相称**：确保你的肢体语言符合你的身份和角色。
5.  **注意言行一致**：确保肢体语言传达的信息与你说的话一致。
6.  **改掉不良习惯**：有意识地纠正那些传递负面信号的习惯性小动作。
