# 【魅力工程】1.0：关系推进 - 第3节：避免错误使用KINO 🚫

在本节课中，我们将要学习在关系推进过程中，如何避免错误地使用身体接触（KINO）。错误的方式会破坏吸引力，而正确的方式则能自然地拉近关系。

## 什么是错误的升级方法

上一节我们介绍了身体接触的重要性，本节中我们来看看哪些做法是错误的。错误的升级方法通常发生在约会过程中，其特点是突兀和缺乏铺垫。

以下是几个典型的错误例子：

*   你突然之间很唐突地，直接抓她的手。
*   女生就会躲开你。
*   又或者是你在KTV，你直接搂着这个女生这样。

## 错误方法的后果

这种唐突的接触会带来负面印象。它会让女生觉得你是那种毛手毛脚，甚至乎会有一点点猥琐的男人，怪叔叔。你肯定不希望被女生贴成这样的一个标签。

## 错误模式的核心

其实错误的升级关系的方式就是，很多时候你会发现它是这样的一个过程的：**前面一点的身体接触都没有，然后突然之间就从0到100，直接进行大幅度的身体接触**。

每一个关系的节点，比如说像第一次的牵手，第一次的亲嘴，包括等等，那这些都是错误的方法。

## 什么是正确的方法

那么正确的方法其实是应该，就像温水煮青蛙，可以让女生不知不觉当中接受你。这个也意味着你从一开始的时候，就要跟她有很多很细微的这种身体接触。

以下是铺垫细微身体接触的一些方式：

*   可能是一些不经意的，比如说你跟女孩子讲话的时候，你碰到了她。
*   又或者是你们走路的时候，互相的这个肩膀靠在一起。
*   但是呢，你们还在聊天，望着前面，在聊别的东西。她的注意力并不是在你跟她的这个接触上。
*   又或者是你轻轻的，如果她说了一些很调皮的话，或者做了一些很捣蛋的事情，你可以打一打她，很轻的打一打她。

所有这些东西都可以铺垫你接下来的这个升级的关系。

## 核心原则与测试

那么我们很强调的一点就是：**你不要在女生准备好接受你之前，就去很唐突地做任何大幅度的身体接触**。

因此，我们会有一系列的所谓的测试指标，要你在真正的有重大的这些身体接触之前，先做这些测试。然后看一下女生她到底喜不喜欢你，对你是否有足够的接受度。
