# 心理学习区必修课28：理解痛苦的关键——社会暗示 🧠

在本节课中，我们将要学习一个心理学中的重要概念：**社会暗示**。我们将探讨它是如何成为我们痛苦感受的关键来源之一，并通过具体例子来理解其运作机制。课程的核心目标是帮助你识别社会暗示的影响，并学习如何减少它对你情绪的负面影响。

## 概述：什么是社会暗示？

一个人形成痛苦，原因有多方面。例如，你的潜意识关注信息的焦点在负面，容易形成痛苦。例如，一个人的视角过小，小事变大时，容易形成痛苦。

痛苦有多个原因，但是有一个原因也非常重要，是排在前几位的原因之一，叫**社会暗示**。它的核心意思是，当社会暗示你应该痛苦，或者社会暗示你很不正常、你应该痛苦，你就会感到痛苦。

## 什么是暗示？与明示的对比

那么，什么叫暗示呢？暗示是与**明示**相对应的一个词。

以下是明示与暗示的对比说明：
*   **明示**：直接、公开地表达要求或命令。例如，你要叫孩子睡觉，明示会这样说：“儿子，赶紧上床睡觉。”
*   **暗示**：间接、含蓄地传递信息，影响他人的想法或行为。例如，跟儿子说：“儿子，现在爸爸来跟你讲个故事，《孙悟空三打白骨精》，打完白骨精以后，大家晚上都睡着，睡得可香可香了。” 讲完故事，孩子可能就睡着了。

心理学研究表明，我们心理意识分两层：**意识**和**潜意识**。潜意识指挥意识，但意识本身并不知道。意识有一个“检阅”作用。这个意识对**明示**的检阅作用非常大。把话说得通俗一点（但不完全准确）就是：明示特别容易引起反感。准确的学术语言是：明示在意识层面的检阅作用特别大。

话越通俗就越不准确。越要准确，越要用学术语言，但用学术语言就越难听懂。听得懂是有代价的，那就是不精准。所以科普与课堂学习差异很大。课堂学习是精准的、系统的、双向沟通的、并特意练习技能的。音频学习只能是所谓科普。

## 社会暗示如何制造痛苦？

我们都不喜欢明示，不喜欢接受命令。你叫儿子睡觉去，他嘴上不一定说，但心里会有一种反感。而暗示则绕过了意识的直接检阅，更容易影响潜意识和行为。

我们很大的一个痛苦来源就是社会暗示造成的。

上一节我们介绍了暗示的基本概念，本节中我们来看看社会暗示在具体生活场景中如何引发痛苦。

以下是几个常见的社会暗示引发痛苦的例子：

*   **未婚者的痛苦**：有的人没有结婚，他为什么很痛苦？如果放在原始社会，他可能就不痛苦，因为那时没有婚姻制度。主要是他看见电视里的结婚典礼，或者马路上情侣搂搂抱抱，他就受到一个暗示：你没结婚不正常。他就因此感到痛苦。
*   **同性恋者的抑郁**：有的同性恋者，抑郁症罹患率要比社会平均水平高得多。这并不是同性恋本身有心理问题（心理学早已认为同性恋是正常的性取向，它与左撇子一样具有基因基础）。这是因为他们看到社会环境中（如电视、广告、故事）大量描绘的都是一男一女的组合，这形成了一个强烈的暗示：两个同性在一起“不正常”。这种暗示导致他们产生自我不认同感，进而引发痛苦。研究也发现，男性同性恋的抑郁比例高于女性同性恋，部分原因在于社会对女性同性恋的接纳度相对更高，她们感受到的“不正常”暗示较弱。
*   **离婚者的痛苦**：离婚的人为什么很难受？尤其是女性，女性离婚者抑郁的比例比男性高很多。因为女性从小受到一个社会暗示：**女人的价值在于男人的认可**。大量的化妆品广告、影视情节、历史故事都在强化这个暗示——女人要打扮漂亮以取悦男性。离婚后，她们受到这个暗示的影响，容易感到自我价值丧失，从而痛苦。
*   **物质比较的痛苦**：例如，一个售价二十多万的汽车广告，画面中一位戴金边眼镜、三十岁左右、看似本科以上学历的白领，开着这辆车在沙漠边缘飞驰。这就是一个暗示。它暗示：如果你有本科学历、三十岁左右，却没有一辆二十多万的车，你是“不正常”的，没过上“飞驰的人生”。于是痛苦就产生了。过去（如教授三十岁时），同龄人没车不会痛苦；现在在一些大城市，三十岁没私家车的人就可能因这种社会暗示而产生痛苦。

## 如何应对社会暗示？

每个人都会受到社会暗示，完全不受到社会暗示的人是不存在的。但是，每个人受到社会暗示的强度不一样。有的人特别容易受社会暗示影响，有的人受影响的程度则比较低。

这取决于什么呢？取决于你的**潜意识**。因此，要减轻痛苦，调整潜意识很重要。

同时，你们要给自己一个**反暗示**，那就是：**我要审视社会暗示。我就是我**。这能起到一定的作用。主动识别出哪些想法是来自外部环境的暗示，并坚定地维护自我的独立判断和价值，可以帮助抵御不必要的痛苦。

## 总结

本节课中，我们一起学习了**社会暗示**这一核心概念。我们明白了：
1.  社会暗示是导致痛苦的重要来源之一，它通过间接的方式让我们感到“应该”痛苦或“不正常”。
2.  暗示与明示不同，它更易绕过意识的检阅，直接影响我们的潜意识。
3.  社会暗示广泛存在于婚姻状态、性取向认同、性别角色、物质消费等多个生活领域。
4.  应对社会暗示的关键在于**调整潜意识**和建立**自我反暗示**，学会识别外部暗示并坚守自我认知。

理解社会暗示的机制，是摆脱许多无谓痛苦的第一步。
