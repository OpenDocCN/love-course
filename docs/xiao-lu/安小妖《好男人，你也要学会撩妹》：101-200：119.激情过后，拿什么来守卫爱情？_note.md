# 安小妖《好男人，你也要学会撩妹》课程119：激情过后，如何守卫爱情？💘

在本节课中，我们将要学习在确立关系、激情阶段过后，爱情可能面临的新问题及其核心原因。我们将通过一个承诺模型，分析影响爱情稳定性的三个关键因素，并提供相应的解决思路。

上一节我们介绍了如何激发激情、开启一段关系。本节中我们来看看，当激情逐渐消退后，我们该如何守卫和维系这段感情。

许多人在关系稳定后会遇到新问题。例如，感觉对方不够爱自己，或自己似乎不够爱对方。也可能感觉双方缺乏信任，关系变得冷淡，不像在谈恋爱。这些问题会让我们对爱情产生质疑。

要解决这些问题，首先需要理清我们在爱情中的核心需求。

我们在爱情中的需求非常复杂。最基础的需求是受自然与文化环境影响，对男女之爱本身怀有强烈的渴望与期待。在此之上，我们还有被关爱、被理解、被照顾的需求，希望有人能让我们安心。此外，我们也希望关系轻松愉悦、没有压力。或者，希望有人能支持我们不被理解的选择。也有人希望通过爱情照亮自己，引导自己变得更好。甚至，爱情也能满足我们对“面子”或虚荣心的需求。

与需求相对应的，是我们在爱情中的付出。付出包括物质、时间、精力，为对方牺牲的机会，以及为取悦对方花费的心思等。然而，付出与需求并不总能匹配，有时对方的付出无法满足我们的需求。

那么，如何让彼此在爱情中都感受到充分的爱意呢？这涉及到我们内心对稳定感情生活的渴望。我们都倾向于建立一段平稳、安定的感情，甚至组建家庭。这种倾向的强弱，决定了我们愿意为这段关系付出乃至牺牲多少。双方的互动很难完全对等，总有高低之分。付出多的人可能感到成就感与快乐，也可能感到不快乐与隐忍。

双方能否长远走下去，本质上由彼此对这段关系的“承诺”共同决定。

我们对伴侣的“承诺”大小，主要受以下三个因素影响：

以下是影响承诺的三个核心因素：
1.  **满意度**：指对伴侣各方面的满意程度，包括外貌、性格、爱好、生活方式、家庭背景、经济能力、生活情趣等。
2.  **替代性**：指解除当前关系、选择其他异性或单身生活的可能性高低。
3.  **投资量**：指在这段关系中已付出的资源，包括有形资源（如金钱、礼物）和无形资源（如时间、精力、情感）。

双方对彼此的承诺大小，基本遵循以下公式：
**承诺 = 满意度 - 替代性 + 投资量**

例如，刚认识一个女孩，觉得她好看、可爱、聊得来，这满足了基本需求，**满意度很高**。同时，认为其他认识的女孩都不如她，也不愿过单身生活，因此当前关系的**替代性很低**。因为喜欢她，花费很多心思制造惊喜，**投资量很大**。最终，很高的满意度减去很低的替代性，再加上很大的投资量，就会产生很高的承诺，即非常愿意与她长久相处。

反之，如果爱情出现问题，往往出在这三个环节上。

首先，是**满意度**环节。感到不快乐的首要原因，是对方未能满足我们的需求（我们对对方不满意），或我们未能满足对方的需求（对方对我们不满意），导致心理预期落空。若在此环节出问题，需要更深入地了解彼此对爱情的需求，加强沟通，创造机会尝试满足彼此，以提升满意度。

其次，是**替代性**环节。这一点在明星或高社会地位人群的爱情中尤为明显。他们满足对方需求的能力很强，但爱情往往不圆满，原因在于他们社交圈内存在大量优秀的潜在伴侣，导致当前关系的**替代性非常高**。如果伴侣本身不具备很强的独特性，就很难牢固吸引对方。相比之下，平民阶层的爱情忠诚度往往更高，正是因为社交圈内**替代性较低**。因此，如果感觉伴侣不够关注你，可能是因为他生活中存在较多优质的潜在伴侣选项，抬高了你们关系的替代性。可以尝试净化他的社交环境，降低关系的可替代性。

最后，是**投资量**环节。要尽量让双方的投入相对均等，避免悬殊过大。长期付出多的一方可能滋生优越感，而长期付出少的一方则容易习惯被关爱。当你试图让后者付出更多时，他会感到不适应，容易造成裂痕。这里的启示是：如果你习惯事事操劳，应试着放松，将部分负担分给对方共同承担；如果你习惯只顾及自己，则应审视并顾及对方的内心需求，避免自己成为只吸收能量、没有付出的“爱情黑洞”。

本节课中我们一起学习了激情过后爱情面临的问题根源。我们通过“承诺=满意度-替代性+投资量”这一模型，分析了影响关系稳定的三个核心因素：满意度、替代性和投资量，并针对每个因素提供了改善关系的具体思路。理解并平衡好这三个因素，是守卫长久爱情的关键。
