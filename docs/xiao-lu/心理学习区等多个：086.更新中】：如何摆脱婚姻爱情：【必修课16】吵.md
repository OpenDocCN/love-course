# 心理学习区课程086：如何摆脱婚姻爱情的痛苦？【必修课16】吵架时如何控制爱人的脾气？（下）💑

在本节课中，我们将要学习在伴侣冲突中，如何通过一种简单有效的方法——“重复对方的话”，来平复对方的情绪，从而为理性沟通创造条件。上一节我们介绍了通过皮肤接触来安抚情绪的方法，本节中我们来看看另一种实用的沟通技巧。

## 核心原则：情绪优先于道理

两人一旦发生冲突，人处于情绪化状态时就不讲理。所以与爱人发生冲突时，首要任务不是讲道理，而是把对方的情绪平稳下来。情绪平稳后，才可能展开讨论。

## 方法二：重复对方的话

除了皮肤接触法，另一种简单的方法是重复对方的话。以下是具体操作步骤。

**操作流程：**
1.  当伴侣表达不满后，首先认真倾听，并微微点头。
2.  然后你可以说：“老公/老婆，你刚刚提的意见，我归纳了一下，一共有X个要点。我不知道我听明白了没有，理解准确了没有，我跟你重复确认一下，是不是这个意思。”
3.  接着，你逐一复述对方提出的要点：“第一，你认为我……没做好；第二，你认为我……没做好；……”
4.  最后确认：“不知道我这样的复述准确了吗？”

## 方法为何有效

设想对方听到你准确复述后的反应：他会觉得被认真倾听和理解。此方法不能保证100%有效，但至少60%-70%的情况下，对方的情绪会因此平复。

复述确认后，是否必须落实对方的所有意见？从大量统计数字显示，通常不需要。家庭本质上不是一个讲道理的课堂，而是一个**讲情绪体验的场所**。只要情绪体验好，一切就OK了。

## 使用时的注意事项

在使用此方法时，需要注意以下两个关键点。

**注意事项列表：**
*   **表达主语是“我”**：正确的说法是“**我**担心**我**没听明白，跟你确认一下**我**理解得是否正确”。错误的说法是“**你**讲明白了没有？**你**确认是不是这个意思”。重点在于“我是否听明白”，而非“你是否讲明白”。
*   **自身情绪需冷静**：此方法要求使用者自身情绪冷静。如果使用者脾气比对方还大，那么问题就变成了如何控制自己的脾气。

## 关于自我情绪控制

控制自身情绪是一个更大的课题。比较临时的应急办法是，一旦感到脾气上涌，可以使劲揪自己一把，疼痛感可能让人瞬间冷静，但这并非长期有效的方法。

真正有效的自我心理状态调整，需要从潜意识和自我催眠入手。自我催眠是一项**需要训练的技能**，如同游泳和开车一样，无法仅通过看书或听讲学会，必须通过练习掌握。
