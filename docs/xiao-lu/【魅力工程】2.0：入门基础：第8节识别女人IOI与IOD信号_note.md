## 核心概念：IOI与IOD

在魅力工程中，**IOI（兴趣指标）** 是一个核心概念，它指代女性对你表达兴趣的行为信号。相反，**IOD（无兴趣指标）** 则指代女性对你缺乏兴趣的行为信号。

懂得分辨IOI和IOD，可以让你了解目标当前对你的兴趣度。大部分未经训练的男性会忽略这些兴趣指标，但经过学习，任何人都可以开始察觉女性发出的那些微妙信号。这不仅能帮助你更好地与对方互动，对于选择何种互动策略也至关重要。

## 识别无兴趣指标（IOD）信号 🔴

相反，如果女生对你缺乏兴趣，她可能会表现出以下IOD信号。以下是常见的IOD行为示例：

*   **转移话题或忽视你的话题**：例如，在对话中突然说“还是看电影吧”。
