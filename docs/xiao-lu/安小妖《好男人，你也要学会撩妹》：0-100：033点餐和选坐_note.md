上一节我们介绍了约会的整体挑战，本节中我们来看看如何通过具体的行动来化解尴尬，展现风度。

其实不仅在点餐时，在男女关系中，这样的矛盾始终存在。女生既要求男生听话、给安全感，同时又希望男生偶尔不听话，创造一些惊喜。

说完了点餐，我们再来说说选座位这个事情。选的座位虽然是个不起眼的小细节，但如果你注意以下几个知识点，你和女生在餐厅里的关系进展就会更加顺利。

到了冬天，注意不要选在离门口近的地方。好不容易营造的温暖氛围，门一开风一进来，温情就可能迅速冷却。
