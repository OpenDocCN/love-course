# 课程083：高难度沟通：在冲突中如鱼得水 🐠

在本节课中，我们将学习如何在高难度沟通场景中避免得罪他人，特别是针对内向者，提供一系列实用的说话策略与倾听技巧。我们将识别沟通中的常见陷阱，并掌握将冲突转化为有效对话的方法。

## 识别沟通中的三个“坑”

想要避免说错话得罪人，首先需要知道哪些话不能说，哪些沟通陷阱需要避开。

以下是三个常见的沟通陷阱：

1.  **急于否定别人**
    对有的人来说，对错这件事特别重要。在他们的世界里，凡事都是二元对立的，而且他们要在二元对立中始终做正确的一方。言语间流露出了一种优越感，即“我比你聪明，我比你正确”。这样一来，对方就会很自然地感受到被否定。这样的对话如果经常发生在职场或两性关系中，是非常可怕的。

2.  **语言微暴力**
    语言微暴力当中充斥着责备、抱怨、批评和推脱，是把所有的问题和不满都归咎于别人的一种表达方式。例如：“你怎么回事啊？”、“你永远都听不懂我在说什么是吗？”、“你从来都不关心我”。

3.  **频繁地使用反问句**
    反问句是说话的人已经把自己的观点植入到句子里的提问方式。例如：“你难道不能顺便去接一下孩子吗？”这种提问方式往往带有说话者非常强烈的情绪。当对方听到时，倾听的焦点会集中在讽刺和攻击上，从而忽略话语背后的请求，并开始为自己辩护。

## 避免得罪人的说话策略

上一节我们介绍了沟通中需要避开的陷阱，本节中我们来看看如何通过积极的倾听和表达策略来改善沟通。

### 策略一：掌握积极的倾听术

当朋友带着强烈的情绪跟你说话，而你不知如何回应时，采用积极的倾听是最好的选择。错误的倾听是只想到自己，并急于给出评判。

以下是三种积极的倾听技巧：

*   **借力使力**：顺着对方的话来回应，而不是重新建立新话题。在简短的回应中，表达出和对方相近或相同的立场。公式可概括为：**重复/认同对方感受 + 简短共情**。
    *   **示例**：
        *   对方：“这工作简直烦死了，我一天都不想干了！”
        *   你：“你想辞职啊？”
        *   对方：“是啊，天天加班…”
        *   你：“那也真是太过分了。”

*   **提问**：通过你的提问推动沟通往下进行，有助于对方倾诉出内心的积怨。
    *   **示例**：
        *   对方：“这工作简直烦死了…”
        *   你：“发生什么了？”
        *   对方：“这半年天天加班…”
        *   你：“怎么会这样？你其他同事也都这样吗？”

*   **支持性回应**：给予对方情感上的认同和鼓励，而不是具体的建议。
    *   **示例**：
        *   对方：“这工作简直烦死了…”
        *   你：“看你这么不开心，可能确实需要考虑要不要换一份工作了。”
        *   对方：“这半年我天天加班…”
        *   你：“天哪，那确实太累了。你都是怎么熬过来的呀？”

### 策略二：把体现尊重和重视的句子提前

当你一张口说出来的首先是替对方着想的话，而不是只顾着自己时，很多难题可以迎刃而解。

**对比示例**：
*   **生硬拒绝**：“不行啊，我現在也沒時間。”
*   **较好拒绝**：“你這個報表什麼時候要啊？不著急的話，我下周給你行嗎？”
*   **高级表达**：“我特別想幫你這個忙（体现尊重），不過你過來看一下我這周的安排就更好了。”

在亲密关系中同样适用：
*   **普通说法**：“不行，上火了，吃點清淡的吧。”
*   **高级说法**：“好啊，和你吃什麼都行啊（体现重视）。可是這兩天，我嘴裡好像上火了，要不今天晚上吃點清淡的？”

### 策略三：学会真正的称赞

真正的称赞都是基于事实的称赞。我们可以使用一个简单的公式：**事实 + 评价 + 感受**。

**公式示例**：
“你的眼睛（**事实**）長得真好看（**评价**），特別是你笑的時候，感覺好像所有煩惱都沒了（**感受**）。”

对于不习惯当面夸赞的内向者，可以先用文字形式（如微信）练习，关键是记住这个公式。

### 策略四：杜绝语言微暴力

印度哲学家克里希那穆提说过：“不带评论的观察是人类智力的最高形式。” 当我们能够留意观察到的事情，并简单地描述出来，就能在很大程度上避免语言微暴力的发生。

**对比示例**：
*   **微暴力指责**：“你這說話什麼態度啊？你們就這樣服務客人嗎？”
*   **观察描述**：“我發現這個房間的兩個窗戶都關不緊，漏風，所以房間裡始終不暖和。可以給我們換個包間嗎？”

此外，减少使用容易引发对立的词汇，如“老是”、“永遠”、“從來都不”，并将指责性的“你”换成以“我”开头的表达。
*   **攻击性**：“你怎麼回事？你從來都不和團隊合作！”
*   **缓和性**：“我覺得，如果你能跟團隊多合作，項目推進會更順利。”

### 策略五：避免急于否定别人

在信息爆炸的时代，很多知识和观点都存在不确定性。一个成熟的沟通者要能够接纳不同甚至相反的观点。

我们可以学习一些缓和否定的句式：
1.  “嗯，有道理，我同時覺得呢是這樣的…”
2.  “哎是啊，有時候是這樣，哎我還發現呀…”
3.  “有很多人跟你的想法是一樣的，不過我也聽到有人這麼說…”

### 策略六：让反问句变成观点的分享和讨论

把包含指责的反问句，变成一种开放性的观点分享，进而邀请对方讨论。

**对比示例**：
*   **反问指责**：“你做事到底有沒有原則啊？”
*   **观点讨论**：“我想知道這和你昨天的說法是不是有些矛盾啊？”
*   **反问指责**：“那你又怎麼解釋今天早上辦公室有人丟東西這件事呢？”
*   **观点讨论**：“小張，有沒有這種可能，就是你昨天走的時候以為鎖上了門，但實際上並沒有鎖好？”

## 课程总结

本节课中，我们一起学习了如何在高难度沟通中避免得罪他人的六个核心策略：
1.  运用积极的倾听术（借力使力、提问、支持性回应）。
2.  沟通时，把体现尊重和重视的句子提前。
3.  基于 **事实 + 评价 + 感受** 的公式进行真诚称赞。
4.  通过“不带评论的观察”和转换人称来杜绝语言微暴力。
5.  使用缓和句式，避免急于否定别人。
6.  将反问句转化为开放性的观点分享与讨论。

对内向者而言，有技巧的倾听是改善沟通效果的基础。沟通的本质，是向对方表达自己的“爱愿”——即关爱与善意。无论对方是谁，如果在交流中能让对方感受到这份爱愿，沟通就是成功的，也无需担心会不小心得罪别人。
