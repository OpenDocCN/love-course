# 坏男孩《魅力工程899》全套：网聊大师：6微信聊天案例：2_note

在本节课中，我们将通过一个具体的微信聊天案例，学习如何将对话从初期的轻松调侃，自然过渡到建立个人联系的中期阶段。核心在于掌握对话节奏，并运用“推拉”等技巧调动对方兴趣。

---

在上一节，我们探讨了如何开启轻松对话。本节中，我们来看看当互动氛围打开后，应如何深化对话。

初期的对话可以包含许多调侃和玩笑，以营造轻松氛围。但进入中期阶段后，这种对话方式应适当减少。

中期阶段的对话模式不应仅限于开玩笑。

核心目标应转向：尽可能地将话题回归到“我们”两人之间的个人关系上。这包括讨论彼此的关系、生活细节，或初次见面的印象。

不能永远只聊第三方信息，例如食物口味、旅游地点或宠物可爱与否。话题需要更多地回归到双方的互动本身。

以下是本案例中的一个关键转折点。我向女生发送了这样一条信息：

> “上次见你的时候，觉得你完全不像一个会开玩笑的人。”

这样做的目的是：回放我们初次相识的情景，将那种感觉重新拉入当前对话，从而让话题回归到我们两人的个人联系上。

她回复道：“那你觉得我是什么样的一个人？”

我回答：“就那样啊，说真话，往往女生都不爱听。”

这里我运用了一个小的“推拉”技巧。其原理是：女生通常不喜欢像面试一样机械、逻辑性过强的一问一答。

与女生对话时，最重要的是懂得如何持续挑起她的好奇心，引发她对话题及对话者更强烈的兴趣。

一个有效的方法是不断运用“一推一拉”的对话技巧。先说一些可能让她稍有失落或疑惑的话（推），再给予肯定或吸引（拉），制造情绪波动。

她说：“说啊，我喜欢听真话。”

女生在大部分情况下不会承认自己虚伪，因此她必然会按照这个模式回应，试图证明自己是个真诚的人。

我继续使用推拉方法，说道：“你平常都是这样骗男生的吗？哈哈哈。” 并附上了一个小表情，继续开玩笑。

这样做，是以一种轻松的方式尝试提升她对后续话题的预期。

她继续调情式地回应：“啊，你真坏，说一些又不说一些。”

我接着问她：“那你是不是一个小气的女生，会不会很容易生气？”

这时，大部分女生不会给出“是”或“不是”这种决定性的回答。女生很多时候不会给男生非黑即白的答案，更多的是模棱两可。

她当时回复了一个模棱两可的表情。

既然她这样回复，我便继续推进互动：“哦，那你就当我没有说过吧，我怕你会杀人。”

这里引出一个重要对比：很多男生在与女生互动时，总试图做所有“政治正确”的事，害怕犯错。

从我们的角度，其实需要更自信地聊天。如果过于谨慎、总怕犯错，反而可能向女生透露：你是一个害怕风险、缺乏价值的男生。

因为有价值的男人，如同许多电影中（如007、钢铁侠）受女性喜爱的角色，他们充满自信，敢于说出其他男人不敢说的话。例如我刚才说的“我怕你会杀了我”。

这时，她回复：“快说快说。” 并配上一个很着急的表情。

很明显，她的兴趣已经被调动起来了。

接下来，我需要进一步将她套入一个“框架”：让她感觉是在追逐我的答案，并且非常想与我聊天。

因此，我需要引导她进入一个正向的反馈循环。于是我问道：“那你答应我，你不要杀了我。”

她当然说：“OK啊，好我答应你。”

我接着问：“那其实有没有人跟你说过，你给人家的第一印象，有点奇怪？”

我相信大部分男生一生中都不敢这样与女性聊天，尤其是对一位漂亮的女生说“你第一印象有点奇怪”。原因在于，很多人见到女生时会试图拍马屁，说许多赞美讨好之言。

但我没有这样做。我是在给她下一个“心锚”，进一步放大她的好奇感。

她可能会因此感到一点点不舒服，毕竟刚认识就被说“奇怪”。但她不得不像被我按了按钮一样，必须通过我来“解锁”答案。

因为如果我不回答这个问题，她会感到相当不适。

她当时回应：“没有。”

既然她说“没有”，我便继续用我的框架来引导她，并轻微地再“推”开一点。

我说：“好吧，那你就当我没说过就好了。”

本节课中，我们一起学习了如何将对话从泛泛而谈深化至个人联系。关键点包括：在适当时机转换对话模式，核心运用“推拉”技巧制造吸引与好奇，并通过自信、非常规的框架引导对话走向，最终建立更紧密的个人互动。
