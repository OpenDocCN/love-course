## 第一部分：应酬前的准备及注意事项 🎯

上一节我们概述了课程内容，本节中我们来看看进行应酬前需要做哪些准备和注意哪些事项。充分的准备是成功应酬的第一步。

### 1. 留意你的身份
在应酬前，首先要明确自己在不同场合、面对不同对象时应扮演的角色。你需要根据对方的身份调整自己的姿态和言行。

**核心公式：**
```
应酬姿态 = f(对方身份， 场合)
```
例如，在上司面前应保持下属的谦逊姿态，在朋友面前应平等相待，在伴侣面前则应放下工作身份，回归家庭角色。

> 维多利亚女王的故事很好地说明了这一点。当她深夜回家，以“女王”身份敲门时，丈夫并未开门；直到她以“妻子”身份回应，门才打开。这提醒我们，交际时必须根据环境和对象及时调整自己的心理状态和身份定位。

### 2. 注意你的着装
着装是你的第一张名片。在应酬场合，得体的着装能给人留下良好的第一印象，甚至影响对方对你的判断。

以下是着装的基本原则：
*   **见什么人，穿什么装。** 根据应酬对象和场合的正式程度选择合适的服装。
*   **避免因着装造成误会。** 着装不当可能导致对方认错你的身份，造成尴尬。

### 3. 理解“八小时以外”的应酬
工作之外的应酬往往关乎人脉、资源和机会。虽然并非所有应酬都必须参加，但那些与职业发展密切相关的应酬应当积极参与。

**注意事项：**
*   理解应酬的必要性：在竞争激烈的环境中，良好的关系可能成为达成合作的关键。
*   学会选择性参加：无关痛痒的应酬可以拒绝，将时间留给家人；但重要的应酬不应轻易错过。

## 第二部分：应酬高手的常用技巧 🎭

了解了应酬的基础准备后，本节我们来看看那些应酬高手们常用的核心技巧。掌握这些技巧，能让你在人际交往中更加游刃有余。

以下是六种常用的应酬技巧：

1.  **装糊涂**
    即便知道某事，在他人提起时也表现出感兴趣的样子。这是一种“人为的单纯”，能鼓励对方更多交流，避免给人“万事通”的傲慢感。
    ```python
    # 伪代码示例：回应策略
    if 对方提及某话题：
        response = “是吗？这个我还不太清楚，能详细说说吗？” # 激发对方表达欲
    else:
        response = 保持倾听姿态
    ```

2.  **善用自嘲**
    当因自己失误造成尴尬时，最聪明的办法是自嘲。这不仅能化解尴尬，还能体现良好的修养和自信。
    > 例如，在同学聚会上说错话后，可以调侃自己：“你看我这张嘴，十几年了还是没把门的。”

3.  **不要轻易指点别人**
    直接指出他人的错误会打击对方的自尊，引发反击。更聪明的方式是委婉地表达不同看法。
    > 英国政治家查士德·斐尔爵士告诫儿子：“你要比别人聪明，但是不要告诉人家你比他聪明。”

4.  **思路灵活，不较真**
    应酬高手懂得世界并非非黑即白，存在“灰色区域”。他们不会过于刻板，懂得适时“睁一只眼闭一只眼”，避免因斤斤计较而破坏关系。

5.  **保持低调**
    低调是一种可进可退的处世策略。这体现在姿态、心态、行为和言辞上。
    *   **姿态低调**：避免盛气凌人，保持亲和力。
    *   **心态低调**：不因小有成就而看不起他人。
    *   **言辞低调**：谦虚说话，例如在受到称赞时，将功劳归于客观因素或团队。

6.  **学会“留缺口”**
    不要把所有事情做满，留一些空间或选择给对方。这既是一种管理智慧，也能在人际交往中创造回旋余地。
    > 书中例子：张某借钱给朋友后借条丢失，他给朋友发传真要求归还2100元（实际借款1200元）。朋友气愤地回传真更正，这封回函便成了新的“借条”。

## 第三部分：应酬中说话的套路 💬

掌握了通用技巧，我们进入更具体的层面。本节将探讨应酬场合中一些行之有效的说话套路，帮助你更好地应对各种对话情境。

以下是九个实用的说话套路：

1.  **“红脸”与“黑脸”**
    根据情境需要，交替使用温和与严厉的态度。一味和气可能被欺负，适时“变脸”能树立威信，让他人摸不透你的想法，从而更加谨慎。

2.  **老实人不说“傻”实话**
    在某些场合，并非所有实话都适合直言。例如面试时被问及离职原因，应避免抱怨前公司或同事，而是聚焦于个人发展或客观因素。

3.  **如何说“打埋伏”的话**
    在提出真实请求前，先说一段铺垫的话进行试探，避免被直接拒绝的尴尬。
    > 例如，想借朋友的车，可以先说：“今天有急事，租车又来不及，真郁闷。” 看朋友如何回应，再决定是否提出借车请求。

4.  **拒绝的艺术**
    如何拒绝才能不得罪人？有以下几种方法：
    *   果断拒绝，不拖延。
    *   通过电话或信息拒绝，留有缓冲。
    *   使用“欲扬先抑”法：先肯定对方，再用“但是”转折。
    *   采用模糊说法，如“再等等看”。
    *   提供替代方案。

5.  **当着矬子不说矮话**
    切勿当众揭人短处。这样做只会导致关系恶化或招致对方反击。

6.  **不在背后议论第三者**
    “宁在人前骂人，不在人后说人。” 在背后说人坏话会损害你自己的形象，让听者怀疑你也会在别处议论他。

7.  **保护隐私，避免冒犯**
    不随意打听他人的年龄、收入、夫妻感情等隐私问题。这是基本的社交礼貌。

8.  **在失意者面前不谈得意事**
    在失业、失恋等处于低谷的人面前炫耀自己的成功，无异于在伤口上撒盐。

9.  **应对他人在你面前说坏话**
    最佳策略是“入耳封存”：只听不评，并尽快转移话题。如果自己说人坏话被当事人知道，应主动道歉寻求谅解。

## 第四部分：如何求人办事 🙏

人际交往中难免需要求助他人。本节我们来学习如何有效地求人办事，包括送礼的学问和提出请求的技巧，以提高成功率。

### 1. 克服“不好意思”的心理
人际关系的本质是互帮互助。不要因为脸皮薄而羞于开口，适当的求助能加深彼此的交情。

### 2. 以礼开道：送礼的学问
送礼是求人办事常见的铺垫，需要注意以下几点：
*   **送对人**：找到有决定权的关键人物。
*   **投其所好**：根据对方的喜好选择礼物。
*   **权衡利益**：根据所求之事的重要性决定礼物的轻重。
*   **因时制宜**：选择恰当的送礼时机（如作者建议的早晨）。
*   **尊重习俗**：了解并尊重对方的文化或地方禁忌。
*   **体现真情**：根据对方的具体情况（如病人、老人、孩子、外宾）选择贴心或具特色的礼物。

### 3. 吃透对方心理，让对方难以拒绝
*   **“门面”效应**：先提出一个很大的要求（很可能被拒），再提出较小的真实要求。
*   **激发同情心**：适当示弱，唤起对方的帮助欲。
*   **“戴高帽”**：先抬高对方，使其不好意思拒绝你的后续请求。

### 4. 提出请求的巧妙方法
为了避免直接请求被拒的尴尬，可以采用以下方式：
*   **自我否定法**：“有件事想请您帮忙，但看您这么忙，估计没时间……”
*   **投石问路法**：先提出一个相关的问题进行试探。
*   **顺便提出法**：以随意、轻描淡写的口吻提出请求。
*   **开玩笑法**：用玩笑的口气提出，即便被拒也可归结为玩笑。
*   **打电话法**：通过电话提出，减少当面拒绝的尴尬。

## 第五部分：如何打圆场，化解困局 🤝

在应酬中，难免会遇到冷场、争执等尴尬局面。本节我们将学习如何扮演“和事佬”的角色，巧妙地化解各种人际困局。

### 1. 打招呼对方不理睬
保持淡定，可以用更直接、声音稍大的方式再次打招呼。如果对方是故意为之，则需反思是否之前有所得罪。

### 2. 聊天冷场，话不投机
提前准备一些“安全”话题，如天气、热点新闻、体育赛事、电影、共同爱好等。先通过广泛话题试探，再深入对方感兴趣的领域。

### 3. 调解他人纠纷
*   **做和事佬，两头致歉**。
*   **不偏不倚，各打五十大板**，寻找平衡点。
*   **主动揽责**：将过错引到自己身上，缓解双方对立情绪。
*   **圆中有方，劝中带威**：在劝解中隐含提醒或警告。

### 4. 劝架
*   **已动手**：立即隔开双方，高声制止，并呼叫旁人帮忙。
*   **未动手，只争吵**：保持冷静，倾听双方诉求，抓住核心矛盾点进行理性劝解。

## 第六部分：交友之道 👫

最后，我们来探讨长远的交友之道。良好的人际关系需要用心经营和维护，本节将介绍如何充实你的“人情账户”。

### 1. 充实“人情账户”
人情如同存款，平时要多“储存”。与朋友保持定期联系（建议一周或半个月一次），节日问候、生日祝福都是有效的投入。人情投资重在长期和真诚，而非短期突击。

### 2. 应对“突然升温”的友情
对于久不联系突然变得热情的朋友，要保持警惕（“无事不登三宝殿”）。
*   对方慷慨时，慎收人情。
*   若受帮助，加倍答谢，避免亏欠。
*   对方热情过头时，保持冷静观察。
