## 回顾：搭讪开场三要素

首先，我们简要回顾上节课的核心内容。一个完整的街头搭讪开场包含三个部分：

1.  **开场白**：与女生说的第一句话。在中国，推荐使用**直接开场**，即直接表明来意。公式可概括为：`开场白 = 直接表明意图`。这是因为中国女生的安全感普遍较低，常受推销或问卷打扰，直接开场能快速建立信任。
2.  **认识理由**：解释为何想认识她。**切忌**仅因对方漂亮而搭讪。更好的理由是观察她的气质或穿着。例如：“你打扮得很精致”或“你看起来气质很特别”。
3.  **表达感受**：分享你当下的真实情绪。例如：“我现在有点紧张，但如果不来打招呼，我肯定会后悔。”

掌握了开场阶段后，下一步的关键在于如何接近女生并展开对话。

## 如何接近：三种目标的肢体语言

静止目标指站立不动（如等人）的女生。接近时，需避免正面径直走向对方，这会产生压迫感和侵略性。

*   **距离**：保持约**一臂长**的距离。太近令人不适，太远则听不清讲话。

### 2. 移动目标（正在行走）

移动目标指正在行走的女生。其难度较高，因对方通常有明确目的地。关键在于避免从后方呼叫或并排跟随，这易被误认为推销人员。

*   **姿态**：然后**侧身回头**与她交谈。这能降低女生的防备心理。

    *   “我一会儿还有事，只有一分钟时间，但不过来认识你我会很遗憾。”

### 3. 坐着目标（餐厅、公园等）

坐着目标指坐着的女生。核心原则是尽量让自己也坐下，以保持姿态平等，避免引起周围人过度关注。

以下是正确步骤：

*   **礼貌询问**：有空位时，应先问：“**请问这边有人吗？**”若无人则可坐下；若有人，则只能短暂站立交谈，且时间不宜过长。
*   **保持平等**：避免长时间站着与坐着的对方聊天，这会将自身置于不舒适且价值感较低的位置。

## 开场后聊什么：基本信息交换

当我们以正确的肢体语言成功开启对话后，接下来需要进入初步的信息交流阶段。陌生人之间倾向于了解彼此的基本情况，这被称为“基本信息交换”。

信息交换应是**双向**的，而非单方面提问或陈述。你需要先提供关于自己的信息，再自然地询问对方。

以下是一个对话示例：
> “我是北京人，不过口音不太明显，可能是工作原因吧。你是哪里人呢？”

在这个例子中，“我是北京人”是**提供自身信息**，“你是哪里人呢”是**引导对方分享**。这种模式自然流畅，能有效推进对话。
