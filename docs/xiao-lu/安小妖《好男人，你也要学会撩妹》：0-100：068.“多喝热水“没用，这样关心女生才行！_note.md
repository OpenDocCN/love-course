上一节我们介绍了关心在感情中的重要性，本节中我们来看看具体如何操作。

追求女生时，关心是至关重要的。这种关心必须是真心的，而非别有所图的虚假表演。一个能体贴入微、时刻关心照顾女生的男生，容易让女生感动。女生常将关心等同于爱，认为只有爱她才会如此关心她。这是女生感性的一面。但这一点也常被不良男生利用，因此女生需要警惕，仔细鉴别关心是真心还是虚假。

在追求异性方面，男生通常比女生更主动和勇敢。但爱情仅靠勇气是不够的。部分男生认为只要坚持不懈甚至死缠烂打，就一定能打动女生。这有一定道理，但绝非最佳途径。对于有主见的女生，这种方法会完全失效。男生应该主动，但不该像苍蝇一样整天缠着不放。特别是当女生已明确表示没有好感时，死缠烂打只能说明这个男生无能或别有图谋。

女生在感情中最看重男生的真心。即便如此，男生在追求心仪对象时，仍不能忽视细节和必要技巧。技巧替代不了真心，但光有真心而不讲究细节技巧，也可能错失良缘。男生爱一个女生，如果只放在心里，女生无法知晓。因此，爱她就应付诸实际行动，让她感受到。

对女生而言，如果说男生最需要的是赞美和崇拜，那么女生最需要的就是男生的关心和照顾。

以下是今天分享的几个关心女生的技巧。

**第一点：运用反差技巧**

这种方式可理解为“欲扬先抑”。在行动前，先降低对方对你的期望值，使其与之后的关心形成鲜明对比。这样，你的关心在她眼中会被无限放大。

例如，女生买了瓶水但打不开盖子。你可以缓缓走到她面前，嘴角扬起戏谑的笑，看着她着急的样子。这时，你可以说：“真笨，你是不是得了帕金森？要不要带你去治治？”通常女生会假装生气地看着你。此时你什么都别说，默默帮她打开盖子递给她即可。这种口头和行为形成的反差，比纯粹的关心效果更好。

女生特别重视细节。如果你从细节出发关心她，她很快就能被你征服。

例如，在和女生打电话或语音时，听到她声音颤抖，你可以说：“我觉得你好冷，听你声音都在颤抖。”这不是她告诉你的，而是你自己从细节中留意到的。这样的细节会让女生觉得你是一个贴心温暖的人。

再比如，曾有男生在聊天时突然说：“睡前喝点牛奶，别半夜又醒了。”这样的关心看似平淡，却特别有杀伤力。因为说话者完全忘记了曾向对方提过自己半夜会醒。当对方突然提起这个被遗忘的细节，会让人感觉他非常细心和懂得照顾人。

部分男生常因关心被女生无视而感到生气。这时应该思考，你是真的用心关心了女生，还是只用了一个让自己满意的形式而已。

**第三点：进行情绪层面的关心**

这是最深层次的关心。所有言语或物质上的关心，都抵不过这一层，且很少有人能真正做到。为什么众多话术都围绕情绪展开？因为在感情中，情绪永远优先于逻辑，且判断更迅速。逻辑标准（如身高、长相、家庭）太多，一时无法定论；而情绪无需标准，仅凭感觉就能判断对方是否合适。

那么，情绪上的关心具体指什么？试想一个场景：对方坐在你面前倾诉：“我最近工作忙死了，事特别多，真的很头疼。”你会如何安慰？

可能很多人会回答：“别太辛苦了，注意休息。”“你看我过得多轻松，何必这么辛苦？”或“想开点，能有什么事。”这些回答背后，隐藏着对倾诉者情绪的**否定**。倾诉者此刻很累，想倾诉情绪，但安慰者并不支持这种情绪，反而告诉对方“不要这么累”。这等于在说“没什么大不了的”，是对情绪的彻底否定，会让倾诉者更不想聊下去。

明明想寻求安慰，却因对方说话失误导致情绪更糟，因为情绪一直得不到支持与理解。一次愉快的谈话，氛围永远大于内容。你和女生聊天的目的，不是说出一堆大道理把对方辩驳得哑口无言。相反，如果女生哑口无言，你就没戏了。

因此，若真心关心对方，应试着体会对方的情绪。浮于表面的关心谁都会，所以你永远无法在她眼里显得特殊。

当对方倾诉“真的很忙、很累、很头疼”时，如果能得到这样的回答：“话说我也一直觉得你给自己的压力太大了，真的。”这两句话看似普通，每个人都会说，但真的每个人都会这么说吗？当对方倾诉时，明显需要的是情绪上的支持。恐怕只有很少的人能做到：从不否定，完完全全地支持对方的情绪，给予包容和肯定。加上语气词“真的”，更加强了支持的含义。

耐心地把自己投入对方的环境中，与对方同甘共苦，纯粹地体会对方的情绪。这种情绪上的关心，才是最能够直击女生灵魂的。
