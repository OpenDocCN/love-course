## 声音的四大要素

那么，我们如何训练自己的声音，让其变得有魅力呢？在此之前，我们需要先了解声音的四大要素：**音色**、**音量**、**音高**和**音长**。这四个要素相辅相成，也互相制约。

### 1. 音色：声音的个性

音色，也就是声音的特色和个性。这由你口腔的形状、发音时舌头的位置以及先天的声带发育特点决定。

什么样的音色叫做纯正？吐字清晰、明亮、圆润是音色纯正的第一标准。用通俗的话说，就是说话能让别人听得清楚。

事实上，有很多人说话时就像在囫囵吞枣。声音很大，但发音粘连，让人听不清楚到底在说什么。有些人说话很快，像机关枪一样，也容易让人抓不到有效的语音信息。

圆润的意思是，说话声音不能太尖。像有的女生嗓音很细，说话声音大一点就容易往头腔位置走，这样就会造成声音听着不舒服。还有的人天生声音很低、很沙哑，像是我们说的那种公鸭嗓。

通常情况下，音色明亮会给人活泼开朗的感觉，而音色低沉会给人感觉比较成熟。如果你的音色比较虚，并且喜欢用尾音上扬，会让人觉得你没有自信。如果你的音色像很多女生那样比较嗲，就会给人一种装腔作势、矫揉造作的感觉。

因此，声音确实能够给人带来很丰富的信息。对于我们来说，指导意义在于：当你在任何情况下沟通交流时，一定要确保自己的状态非常好、非常亢奋。不要让自己的声音显得很疲惫、很低沉、很虚弱，这会给人带来不好的感觉。你应该让你的声音时刻处在一个饱满、明亮的状态。

### 2. 音量：声音的大小

顾名思义，音量就是声音的大小。有的人在描述一件事情时听着就很有趣，那是因为他讲述的过程中，声音的音量轻重缓急、忽高忽低，很有节奏和韵律感。

但有的人，同样是描述一件事情，就像是白开水一样。再有趣的事，从他嘴里讲出来都觉得像在听天气预报一样，很没劲。这就是音量控制的问题。

还有的人，会刻意为了让别人听到自己的声音，讲话时音量很高。这其实也不是一个非常好的选择。无论在什么场合，音量过高都是一种不成熟的表现，也是一种低素质的表现。

你看那些擅长演讲、擅长讲话的人，他们在讲话时，其实都不会出现刻意把声音放大的感觉。保持一个平稳、合适的状态就好。声音太高，其实有一种喧宾夺主、哗众取宠的姿态，是一种没有安全感的表现，说明内心极度渴望被人关注。

生活中也是一样。比如你去吃饭，需要服务员为你服务时，也不必大声喊叫让每个人都知道。包括你跟朋友分享一件自己的趣事或买了什么东西，也没有必要声音非常高让所有人都知道。这是一种不好的习惯。

在音量方面，我们需要注意的是：当你在表述一件事情、描述事态发展的时候，要去找到它的重心和重点。例如，在类似转折（“但是”）、揭示内心想法（“其实他内心是这么想”）的时候，当你想表达一些很重要的信息时，要有意地去加重一些音量，来让大家注意到你下面要说的内容。这是一种我们在音量上应该注意的表达策略。

### 3. 音高：声音的声调

音高就是声音声调的高低，就是这个Key。它取决于你的声带在一定时间内振动的频率。

这里说的高与低，并不是一个非常客观的标准，而是指在你个人表达中出现的音调高低之分。音调高一些的时候，表示你处在一种比较兴奋活泼的状态，会传达出一种高兴愉悦的心情。当你的音调比较低沉时，就会向别人传达出你比较忧郁、阴沉或平静的心理状态信息。

例如，同样一句话：
*   **低沉状态**：`“啊……”`（发音位置下沉，声调低）
*   **高亢状态**：`“啊！”`（发音位置上提，声调高）

你会发现两者区别在于发音位置的高度。当你感觉发音位置非常下沉时，发出的音调就比较低。当你人为地把胸腔打开、呼吸道打开，整个人挺直，头微微上扬，这时发音位置就比较高，发出的音调也比较高。

因此，给我们的启示是：**你可以通过调整自己发音时的身体状态，来调整发音的高度。**

### 4. 音长：音节的长短

音长是指每个音节发音的时长，以及是否干脆。就像弹钢琴时，每个音之间是有一个明显的断联，还是有一种流畅的感觉。

例如，同样的话，在不同情境下：
*   **白天安慰朋友**（音长较短）：“我觉得你这样肯定是不对的，你这样是一种不负责任的行为。”
*   **深夜谈心安慰**（音长较长）：“我觉得呢，说实话，其实这件事情并不能全怪你，因为没有任何一个矛盾的出现，是由一个人造成的。”

人为地把声音拖得比较长，所表达的感情状态是非常不同的。

对于你来说，应该训练自己驾驭不同说话感觉的能力。用不同的状态去尝试发音和练习。这样，你就能够在非常不同的场景之下，通过声音表达出不同的感情状态。再配上相应的内容，就会让你的表达变得很有魅力。
