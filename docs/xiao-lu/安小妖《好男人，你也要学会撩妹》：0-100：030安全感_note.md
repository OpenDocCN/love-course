## 什么是安全感？

上一节我们介绍了课程主题，本节中我们来看看安全感的定义。安全感是一种渴望稳定与安全的心理需求。它并非抽象概念，而是体现在具体的生活场景中。

例如，当你来到一个陌生城市，没有朋友、不认识路、也没有地图时，心中产生的慌张感，就是缺乏安全感的表现。在情感关系中，女生所需的安全感，核心在于感受到被爱、被关注、被在乎和被关心。
