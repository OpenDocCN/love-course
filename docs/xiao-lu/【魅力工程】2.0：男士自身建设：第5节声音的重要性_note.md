## 声音的四大要素

以下是声音的四个核心构成要素，理解它们是进行有效训练的基础。

**第一，音色。**
音色是声音的特色与个性。音色由口腔的形状、发音时舌头的位置以及先天发育特点决定。那么，什么样的音色算纯正呢？吐字清晰、明亮、圆润，是音色纯正的第一标准。用通俗的话来说，就是说话能让人听得清。

音色发嗲给人装腔作势、矫揉造作的感觉；音色过尖给人功于心机的感觉。

**第三，音高。**
音高就是声音的高低。它取决于发音体在一定时间内震动的频率多少，俗称就是声带。这里所指的高与低，是在个人不同的音高基础上相比产生的。音高较高通常表示兴奋、活泼、高兴等愉悦的基调。

## 高级阶段的声音练习

上一节我们介绍了声音训练的基础要求，本节中我们来看看如何让声音更具表现力和感染力。以下是高级阶段的练习方法。

**一，饱满圆润。**
尽量让每个字或每个词语的发音饱满圆润，不要把字的音节吞掉。一定要让目标听得清楚。公式可以理解为：`清晰发音 = 完整音节 + 适度口腔开合`。

**二，语速和缓。**
例如句子：“我在过马路，你人在哪里？”分别用两秒的时间和五秒的时间读一次，效果是完全不一样的。多读几次，熟练之后。

用四秒的时间再去读三遍，你就会进步。核心是控制节奏：`有效语速 = 内容字数 / 适宜时间`。

**三，重音强调。**
例如句子：“春天也隐隐地撩开了他那妩媚动人的裙裾摇摆。”不用任何重音读一遍。“春天**隐隐地**撩开了他那**妩媚动人**的裙裾**摇摆**。”读五遍，录音对比效果。

细心的朋友可以看出来，在关键性的名词、动词、形容词上用重音，效果会非常好。注意一点：不要用太多重音，过犹不及。另外，一句话当中，通常只有一个最重要的重音。注意把握好，不要把所有的重音都读得一样重。

**四，合理停顿。**
例句：“四月的到来，北京的天空终于慢慢褪去了那风霜傲雪时太灰色的样子，湛蓝的天空没有任何杂质。”读五遍后加入感情，继续录音对比效果。停顿的代码化示意：`句子 = 片段1 + [停顿] + 片段2 + [停顿] + ...`。

**五，富有感染力。**
像个演员那样，让声音充满表现力和感染力。如果你不能通过声音影响目标的情绪，那么互动效果一定不会好。
