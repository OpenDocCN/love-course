# 心理学习区：032.情感心理咨询技能大全（完结）第07章：策略式家庭治疗—刘亮：05.沟通的不同层面（下）👂

在本节课中，我们将要学习沟通的四个不同层面：**内容**、**需求**、**状态**与**关系**。我们将通过具体例子，理解为何沟通有时显得复杂，并掌握如何运用“四只耳朵”去倾听，以实现更有效的交流。

## 沟通为何显得复杂？🤔

上一节我们介绍了沟通存在不同层面，本节中我们来看看这些层面如何在实际互动中交织，使沟通变得复杂。

有时我们会觉得人与人之间的沟通很复杂。听者难以确定说话者到底想表达什么。例如，一个女孩对男朋友说：“我没事的，真的。”有的听话者会认为对方真的没事，于是选择去做自己的事，比如打游戏。结果却发现女孩很生气，并感到困惑，因为对方明明说没事。实际上，女孩想传递的信息是：“我有事，我需要你的关注。”

另一个例子是，一个男生对女朋友说：“我再也不想见到你了。”女朋友可能理解为双方需要冷静一段时间。但过了一会儿，男生又发来几十条微信。当女朋友质问时，男生可能解释：“我说不想见你，实际意思是让你现在立刻马上过来。”因此，处于恋爱中的人常常觉得对方的心思难以猜测，沟通显得非常复杂。

## 沟通的四个层面：用“四只耳朵”倾听👂👂👂👂

面对复杂的沟通，我们可以通过一个框架来理解。当一个人进行沟通时，他传递的信息同时存在于四个层面。我们可以形象地称之为用“四只耳朵”去倾听：**事实内容**、**关系**、**讲话者状态**和**诉求**。

这意味着，每一次沟通都同时在四个层面传递信息。

### 1. 事实与内容层面
这个层面关注话语表面的、客观的信息。

例如，我说：“不知道现在这个房间里面的空调是多少度。”如果听者只从内容层面回应，他可能会回答：“现在空调是17度。”这是对表面问题的直接答复。

### 2. 讲话者状态层面
这个层面关注说话者当下的情绪或身体感受，这些往往没有直接言明。

继续上面的例子，当我说出关于空调的问题时，我可能实际想传递的状态是：“我有点冷。”但我没有直接说出来。如果听者捕捉到这个状态信息，他可能会回应：“我把空调调高一点，17度你是不是有点冷？”或者尝试为我找件衣服保暖。

### 3. 诉求层面
这个层面关注说话者希望对方采取的具体行动或满足的深层需要。

我的那句话可能包含多种诉求。浅层诉求可能是：“能不能把空调调高一点？”或“我们要不要换个房间？”更深层的诉求，如果对方是我的伴侣，则可能是：“我希望你能主动关注我的感受，而不是总要我明说。”在临床案例中，常见类似情况。例如，妻子说想吃水煮牛肉，丈夫却点了锅包肉。妻子生气并非因为菜本身，而是透过这个行为，她感受到丈夫不关心她的需要。她真正的诉求是：“我希望你在任何事情上都能主动关注我的感受。”

### 4. 关系层面
这个层面定义了沟通双方如何看待彼此的关系。

例如，如果我以命令的口吻对小王说：“小王，你说这个空调多少度？”那么，无论是小王本人还是周围的旁观者，都可能从中解读出我与小王是上下级关系。小王可能会因此感到愤怒，因为他接收到的关系定义是“我被你看作下属”。这种对关系的认知会影响双方的互动。

每一次沟通都同时在这四个层面传递信息。这提示我们，无论是作为治疗师还是与家人朋友互动，在必要时都需要打开这“四只耳朵”。

## 如何应用：给听者的操作性建议🛠️

如果你发现自己的回应总让对方不满意，可能需要检查自己正在用哪只“耳朵”倾听。对方是想讨论事情本身，还是想表达状态、诉求，或定义关系？以下是针对不同层面的倾听建议。

### 层面一：倾听事实与内容
如果你只打开内容层面的耳朵，你的沟通方式将围绕事情本身展开。

以下是应对方式：
*   以中立的方式讨论事情本身。例如，当伴侣问“今晚我们去哪里？”，如果这仅是内容层面的询问，就只需讨论目的地。
*   无需过度猜测言外之意。如果无法确定，可以直接询问对方：“你问这个问题，是真的只想和我讨论去哪里吗？有没有其他没说的话？”

### 层面二：倾听讲话者状态
有些人不习惯直接表达自己的状态，会通过间接方式传递。

以下是识别与回应的关键：
*   注意观察对方的表情、语气和身体姿势。如果对方心平气和，可按内容层面回应。如果对方带着抱怨的语气说“今晚我们究竟去哪里？”，则可能伴有情绪。
*   此时需要关注并澄清对方的情绪。你可以说：“我注意到你好像有点不高兴，你真的只是想知道去哪里吗？还是有其他话想跟我说？”
*   回应的重点是**耐心倾听和承认对方的情绪**。可以说：“我能感觉到你确实挺生气的。你觉得现在我可以做点什么来安慰你吗？”避免一上来就说“你到底想我怎么样？”，这像是挑衅。

### 层面三：倾听关系定义
有时，话语背后是对方对双方关系的期待或定义。

以下是处理方式：
*   例如，妻子问“今晚我们去哪里？”，其关系层面的诉求可能是：“我希望在我们的关系中，你能更主动地承担规划者的角色。”
*   如果丈夫回应“你定就行了”，妻子会感到生气，因为丈夫没理解她对关系模式的期待。
*   作为治疗师，如果观察到家庭中反复出现这种模式，需要将其反馈给家庭。你可以说：“我注意到，当你问‘去哪里’时，似乎是想调整你们的关系模式，希望丈夫更主动。你们有意识到吗？你们觉得可以如何调整沟通方式？”请注意，治疗师不替家庭做决定，而是促进讨论。

### 层面四：倾听深层诉求
当对方不直接表达诉求，而是通过抱怨等方式暗示时，需要识别并澄清。

以下是具体步骤：
*   当对方的话语内容与情绪不符时（如生气地说“今晚我们到底去哪里？”），要警惕其是否有未言明的诉求。
*   可以直接询问：“你这句话背后，到底想告诉我什么？”
*   作为治疗师，需要帮助来访者澄清其背后的需要，并鼓励他在咨询中直接表达。例如：“我注意到你说这话时眼神比较犀利，你能告诉我你真正想告诉丈夫的是什么吗？”然后鼓励来访者心平气和地直接表达诉求，并可将此练习布置为家庭作业。

## 核心要点总结📝

本节课我们一起学习了沟通的四个层面及其应用。基于此，有三点核心建议：

1.  **对讲话者而言**：在说话前，想清楚你真正想传递的信息属于哪个层面（是内容、状态、诉求还是关系定义），并尽可能直接地表达出来。
2.  **对日常听者而言**：需要保持一定的敏感性，尝试理解对方话语中可能存在的多层含义。虽然这需要练习，但每个人都有自己的沟通模式，细心观察便能更好地回应对方的需要。
3.  **对咨询师而言**：必须保持对这四个层面信息的敏感性，及时记录、澄清并与家庭讨论他们沟通中反复出现的模式。

掌握用“四只耳朵”倾听，能帮助我们穿透复杂的表面信息，实现更深层次的理解与沟通。
