# 30：1、内在工程：第三十七集：生命的半自动导航_note

在本节课中，我们将要学习“生命的半自动导航”这一概念。我们将探讨我们的心理状态如何形成习惯，潜意识如何主导我们的思维与行为，以及如何通过自我负责来打破消极循环，从而更好地掌控自己的人生。

## 理解心理状态与习惯形成

上一节我们介绍了课程的主题，本节中我们来看看心理状态如何影响我们。想要影响其他人，你首先必须能够引领自己。我们都会回应我们自己的状态。

*   **内在属性强大的人**对自己的头脑状态负责。
*   **内在属性弱小的人**把自己的感受归因于世界。

我们感知内在的方式将向外投射，并对我们身边的人产生影响。情绪是会传染的。如果你希望在你身边的人都有个好心情，那么你自己首先就得要有一个好心情。

因此，让我们看一看应该如何调整我们自己。作为自适应的生物，我们通过我们最重要的精神状态调整自己适应环境。我们越多的经常进入一个特定的状态，那么就越多的奠定我们精神的轨道，我们就越容易再次进入某个状态。

**公式：精神状态 → 习惯性轨道 → 更容易进入相同状态**

例如，你经常不开心，就很容易进入不开心的状态。你经常开心，在日常生活中想要开心也是一件很容易的事情。因为你的精神轨道已经被你习惯为自然了。

随着时间的推移，我们经常感受到的心理状态将成为习惯。他们可能会有好或不好，但他们都是相似的感觉。因此我们习以为常，并在一定程度上对我们最熟悉的一种心理状态上瘾。你的习惯会成为你的潜意识，而你的潜意识将决定你的思维。

## 消极思维的循环与逆向合理化

因此我们的很多思维模式是循环的。例如，我们感到沮丧时，想法可能就会开始循环：“我感觉很糟糕，为什么？因为上班迟到了，被领导骂了，状态不好，然后跟客户吵起来了……”

从逻辑上讲，他们知道他们无法控制外界会发生什么事情，但是感情上他们熟悉不开心的状态。他们的人生在不断的重复着、循环着。

以下是消极思维模式的特点：

*   消极的人总是会找出一些事情抱怨。
*   消极的人会进行逆向合理化。你给他一件正面的事情，他也能找到理由或者找到角度来抱怨。

他以为是外界的事情引发了他不好的情绪，然而事实却是相反。首先他们有一个回到熟悉状态的倾向。当他们想去抱怨的时候，他们会寻求合理化他们感觉的证据。他们可能会认为有一些外部的事情使他们有不好的感觉。事实上他们只是想找到以前熟悉或是已经习惯的心理状态。

说到这可能有人会说：“我就经常不开心，我也知道很不好。你的意思是说，我会去主动找不开心的感觉？这怎么可能，我又不是傻子。”

前面说过了，当你对某一项东西已经成为习惯时，这个习惯会下沉到你的潜意识，变成你潜意识的一部分。而你的潜意识是比你的意识要强大得多。你的潜意识决定了你生活的大部分。

**代码示例：潜意识 vs 意识**
```
if (潜意识习惯 == “不开心”) {
    日常状态 = “不开心”; // 潜意识主导行为
} else if (意识想要 == “开心”) {
    // 改变需要克服强大的潜意识惯性
}
```

所以你会发现，愁眉苦脸的人总是如此，不开心的人也总是不开心，负面的人也一直是负面的。这就是他们的潜意识在决定他们的人生。如果他们不能意识到这一点，那么他们就会一直不断地循环。

## 自我负责：摆脱受害者心态

消极的人总认为是外界让他有了今天的局面。他的不如意、他的性格、他所有的一切的不满，都是因为外界。我们之前说过了合理化，每个人都需要为自己的行为或某项特质找到理由，好让自己舒服一点。于是他就开始逆向合理化。

很多人会怪家庭：“都是因为你们如何如何，导致我今天这样的局面。”例如，一个犯错的富二代责怪父母：“就是因为你们从小不教我，惯着我，所以我才会这样。”这是何等的巨婴心态。他不想自我负责，所有的原因都是外界的。

还有各种各样的把自己的问题归因于家庭或社会。例如：“我自己的性格就是家庭造成的，是这个社会造成的。”

> **引用示例**：“我从小就单亲，但是我从来没有拿这个作为借口。我也不会把问题归于外界。也就是说你必须自我负责。”

你是一个成年人，你应该能直面自己所有的问题。你所有的问题、你所有的不如意、你今天的境遇、你的人际关系，你都应该站出来直视这一切，并且自己承担责任。不要总是为自己找理由找借口。

以下是不同人的责任归属模式：

*   **积极的人**：为自己负第一责任。
*   **消极的人**：把原因归咎于外部环境。
*   **依赖者**：总是幻想着别人来帮他，或者老天爷让他脱离困境。

要知道，你才是唯一能够控制自己生活的人。你所有选择的集合让你成为现在的这个样子，只有你自己才能够摆脱它。当然会有人来帮助你，来指引你的道路，但是他们不能帮你控制你自己的生活。他们只能帮你领到门口，你必须利用自己的力量走进去，把自己所学到的知识投入到实践中。

## 从受害者到掌控者：重设你的按钮

所谓人生不如意十有八九，我们难免会遇到不如意的事情。这时候你的反应是怎样的？你的身上有这样的按钮：一个写着“快乐”，一个写着“不快乐”。谁会触动这些按钮呢？如果总是别人触动这些按钮，那么我们的生活是由谁掌控？

大部分的人都有这些按钮，他们的生活经常失控。他们的心智模式是：**外界/别人来掌控我的生命，导致了我现在的状态**。

一旦你安装了这个模式，那么他们只能小心翼翼地保护那个痛苦的按钮，不要让其他人按到。但如果说实在被按到了，除了沮丧，他们还能有什么样的办法呢？那么我们其实把这种人称之为 **“受害者”**。

受害者通常习惯把痛苦和快乐放在别人手上。有的时候是家人，有的时候是上司，有的时候是朋友，还有的时候可能是过去的自己。他们面带怨气，永远是一副受害的心态，不想自我负责。

这里讲个小故事：

> 一位禅师很喜欢养兰花。有一次他外出云游，就把兰花交给徒弟照顾。徒弟知道这是师父非常喜欢的东西，于是小心照顾。就在禅师回来的前一天，他不小心把兰花摔坏了。禅师回来后知道了，并没有生气。他告诉徒弟：“我当初种兰花，不是为了今天生气而来的。”

我们很难做到像禅师那样的境界，但是我们是不是可以逃离受害者的身份，打破这样的心智模式？一般来讲，受害者就是不断地推卸责任，他们的意识里没有所谓的自我负责。

其实这个世界上还有一部分人，他们有一个奇妙的心智转化。这类人好像就没有“痛苦”的按钮，只有“快乐”的按钮，而且按钮只在自己身上。例如刚刚说的禅师，即使兰花摔坏了，但总有比大发雷霆更好的选择。

不管外界怎么样，我都有能力对自己的状态负责。这种人总是能够找到更好的方法，因为他明白不管外界怎么样，下一步的生活都是他们自己的。

*   **老板发火了**：我可以选择去沟通，也可以选择离开。
*   **女朋友生气了**：我可以选择去沟通，或者调整自己讲话的方式。
*   **堵车的时候**：我可以选择下次错开高峰期，也可以选择用这个时间听听音乐或新闻。

这种人其实就是 **“掌控者”**。对于掌控者来说，每个事情都是生命的一件礼物，但是你可以选择是否打开它。

## 重组人生：从半自动导航到主动掌控

大部分的人，他的人生是半自动导航的。也就是前面所说的，你的潜意识在控制着你的所有。你根据社会教你的一些东西，以及自己所相信的现实来判定，什么是对你有利的，什么是对你有害的。

然而，解决这个问题有两条途径：

1.  **极高的保持意识**：然而这种基本上只有那种开悟的人，也就是佛家所说的了悟的人，才能够做得到。
2.  **改掉你的潜意识，重组你的人生信条**：自我负责，脱离受害者身份，让你的半自动导航能够更准确，往更好的方向驶去。

这难吗？当然。这是日复一日，月复一月，年复一年的累积的结果。**你越坚信新的现实，那么你就会看到新的世界。**
