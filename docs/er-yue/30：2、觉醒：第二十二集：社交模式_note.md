# 30：2、觉醒：第二十二集：社交模式_note

在本节课中，我们将要学习三种核心的社交模式。理解并应用这些模式，能帮助你建立更健康、更有效的人际关系，提升社交自信与互动质量。

## 概述

我们将依次探讨三种社交模式：**合作与竞争**、**进攻与防守**、**分享与防御**。每种模式都对应着不同的心态和行为，深刻影响着我们的社交体验与自我认知。

## 1. 合作与竞争 🤝

上一节我们介绍了课程主题，本节中我们来看看第一种模式：合作与竞争。

在关于自我与自尊的课程中，我们已多次讨论“比较”这一行为。人类天生倾向于与外界比较，尤其是与周围的人，如同学、朋友、同事进行比较。

比较会带来两种结果：
*   当比较后优于外界时，会产生**优越感**。
*   当比较后不如外界时，会产生**自卑感**。

频繁地与他人比较，容易滋生消极情绪，如后悔、愧疚、不满。总体而言，生活不幸福感与经常进行社会比较密切相关。

利用比较来寻求安慰并非好方法。长期如此，你会陷入社会竞争的游戏中。不断比较会把生活变成一场争夺优势地位的战争，例如比较穿着、社会地位、财富或伴侣。

比较的本质是将你与外界**分割**甚至**对立**。这个过程就是竞争，目的是拔得头筹，证明自己更厉害。人际交往中的竞争会让你与社会产生隔阂，感到孤立。

然而，结果常与预期相反：一心求胜的人未必能赢。强烈的求胜心会产生巨大压力，反而影响发挥，并可能加剧冲突，因为每个人都试图超越他人。这在博弈论中被称为**零和博弈**（`你的收益 + 他人的损失 = 0`）。

持有这种观念的人不仅想自己赢，还会设法阻止他人赢。以这种竞争视角看待社会生活，会导致扭曲的态度与心理疲惫。

自尊也会因此变得脆弱。在这种自私、充满竞争的环境中，一些人会过度依赖外界评价，内心缺乏安全感，非常不稳定。被外界认可和拥有一定的控制感是自尊的源泉，但若试图通过竞争和控制来获取，则是走错了路。

以下是竞争心态带来的主要问题：
*   **损害自尊**：比较和竞争的态度对自尊有害。
*   **降低关系质量**：逐渐削弱社会关系的深度与满意度。
*   **导致不稳定**：依靠统治或控制获得的自尊非常不稳定，唯有被人接纳才能稳固自尊。

当你的自尊接近明智状态时，正确的比较对象应是**自己**：与几个月前、几年前、或处于不同境遇时的自己比较。这是自我发展的最佳方法之一，因为活着就是不断塑造自我。

许多人热衷于与他人竞争，试图打败别人以证明自己，甚至抱有敌对态度。当你这样做时，一种暗黑的能量会附着于身，不友善会从言行中流露。最终，即便赢了，也会筋疲力尽。

只要想着竞争，你就处于**被影响**的状态，因为你仍在试图证明自己，这本身暗示了低价值感。

因此，我们应该选择**合作**。世界不应是分裂的，你就是整体的一部分。在与异性互动时，若过度想着控制框架、主导走向，只会把对方推远，因为这背后的动机是“证明自己是对的”。当然，你可以表达想法，走出自己的道路，但不该带着较劲的情绪去完成。

只有抱着**合作**的态度与异性互动，共同创造欢快、愉悦的能量，你们的关系才会越来越近。

在集体活动中也是如此：
*   **竞争心态**：若遭遇多次挫败（如游戏中“被打枪打多了”），容易进入恶性循环，感到被孤立和遗弃。
*   **合作心态**：你会把自己视为集体的一部分，视周围人为朋友，从而进入良性循环。即使独自说话，也能感受到集体的力量，更有底气。合作心态还能帮你摆脱虚伪的自我形象，丢掉“别人怎么看你”的包袱，因为你感觉大家是朋友，是一起的。

所以，第一点的核心是：**丢掉比较或竞争的心态，以合作的心态去任何地方与人共处，视大家为朋友**。无需比较穿着、收入、工作、车子、房子，你的自我认知不应由这些东西支撑。永远选择合作，世界就都是你的朋友。

## 2. 进攻与防守 ⚔️

理解了合作优于竞争后，我们来看第二种社交模式：进攻与防守。

设想一个社交情景：你去到对象的家里，对方父母热情招待，端茶倒水，嘘寒问暖，并不断询问你的个人及家庭情况。此时，你的基本状态是不断点头和回答问题。

在这个情景中：
*   **对方的父母处于进攻（主导）状态**。
*   **你处于防守（被动）状态**。

“进攻”和“防守”是为便于记忆而做的比喻，其本质是**主导**与**被动**。在社交或与异性相处时，总有一方处于进攻（主导）状态，另一方处于防守（被动）状态。

然而，防守状态永远只能**被影响**。因此，你应努力成为主动型的人，去主导并影响他人。

在陌生场合，最应该做的是立即向不认识的人**自我介绍**，并开始谈话互动。这能提升你的整体状态，使你处于主动方，而对方则处于被动方，无论其是否接受。

不要总是等待他人先释放交流信号或迈出第一步。无需想太多，直接行动即可。过度思考会让焦虑侵蚀所有动力。

被动型的人其行为完全取决于外界如何变化，疲于应对。在社交场合中，许多被动者会盲目跟随群体：别人笑也跟着笑，群体聊什么话题就跟着聊，但从未主导过话题。

以下是处于被动/防守状态的弊端：
*   **无法流通更多价值**：难以有效展示和传递自身价值。
*   **失去主导权**：无法决定互动走向，只能跟随。
*   **精力消耗**：疲于应对外界变化。

所以，我们永远要争取**主导**。当然，分寸很重要。例如，进入他人的场合（如女生的朋友圈），由于社交惯性，初期可稍处被动（防守），但应逐渐转入进攻（主导）状态。

主导话题的示例：
*   对女生：“你这么能说会道，有没有男朋友？没有的话我帮你介绍一个。喜欢什么样的？你不会喜欢大叔那种类型吧？大叔可不适合你。”
*   对男生：“听你口音不像本地人，你来这儿多久了？”

请注意，在他人场合，**切勿争夺“雄性领袖”地位**（即喧宾夺主），这会招人讨厌。你要做的是更好地做自己，同时把握分寸地主导，除非该场合的领袖是你的好友。

一个小诀窍是：去到任何场合，脑子里想着“**进攻**”二字。这会调动你的主导性。就像准备进攻打一个人时，身体会自动调整状态（握紧拳头，眼神聚焦）。“进攻”的意念会让你的脑回路自动调整，使你置于主人翁位置，开始主导话题。想象他们是来你家的客人。

最后强调：**这里的“进攻”意指“主导”，而非具有攻击性**，切勿混淆。当你发现自己处于被动时，应立即调整，进入进攻（主导）状态。在99%的情境中，你都应处于进攻状态，除非你是在接受采访。

2.  **进攻与防守**：避免陷入被动，要主动主导社交互动，把握分寸，成为价值的流通者。
