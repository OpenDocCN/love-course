# 30：2、觉醒：第五十二集：本我和小我（2）_note

在本节课中，我们将继续探讨“本我”与“小我”的核心区别，并学习如何通过专注于“提供价值”和“活在当下”来摆脱小我的束缚，从而获得更真实、更持久的自信与快乐。

---

上一节我们介绍了“小我”是如何基于过去的痛苦经历形成的。本节中我们来看看小我的运作方式及其不可持续性。

小我总是需要越来越多的证据来证明自己很厉害。小我的自信是不持续的，因为它总是需要外在的反馈和证明来支撑自己。**小我的模式**可以概括为：`自信 = 需要持续的外部证据`。

真正的“本我”则是可持续的，因为它不依赖于外部证明。为什么小我不可持续？因为你总需要外在的证明来维持自信。小我总是在寻求反馈，如果反馈不好，自信就可能瞬间崩盘。

所以，小我就像一个黑洞，总是在吸收，总是想要证据来证明“我很厉害”。但如果我们想达到真正强大的状态，就必须去掉所有的“需求感”。

小我总是活在过去和未来。它从过去寻找证据，并幻想在未来找到支撑。然而，当下状态才是幸福的源泉。当你意识到快乐是人类默认的状态时，你会发现当下才是你生存的唯一瞬间。我们不应去未来寻找快乐，快乐就在现在。

以下是大多数人遵循的“默认人生计划”：
*   先去幼儿园，然后去小学。
*   接着去初中，再去高中。
*   然后去大学，毕业后工作。
*   努力工作，可能结婚生子，创立公司。
*   最后退休。

这些事情本身值得荣耀吗？我们总是被教导去测量自己，为自己打分，并持续想着“下一件事”。但当你察觉到未来总是未知的，时间观念就变得不那么重要了，你开始能享受当下。这就是“心流”的产生。

当有人说“当下是你所能拥有的所有东西”时，他们的意思是：他们对正在做的事情感觉很好，只是在做喜欢的事。这个状态就是本我的状态。

当你纯粹活在当下时，你的创造力开始爆发，你说的每句话都自然流畅。你根本不会去寻求别人的好评，而你那些很棒的特质，会从内心自然流露出来。这个状态就是最佳状态。

我们常常为了某个目标（比如买一部新手机）辛苦工作，但得到后却发现“也就那样”，兴奋感很快消失。过程反而比结果更有趣。学习吸引异性也是如此，得到后反而可能觉得没有挑战性，不如互相调情的过程好玩。

也就是说，结果带来的快乐，总是无法与做事过程中的快乐相比。因为当你意识到快乐是默认状态时，你会发现过程和结果带来的快乐是一样的，甚至过程更多。

说到这里，你可能会问：那我们做任何事情的原因是什么？

其实，**提供价值**就是你所能做的最值得开心的事。在人际互动中，如果是“索取价值”，即你的动力来自于小我，你总想着操控或影响别人，并期待对方给出好的回应，这会让对方感到不适。

试想，如果有个人和你说话时，总想操控你的回应，总想得到你的认可，你会很烦。每个人都喜欢和那种“知道自己是谁”、完全在表达个性、却不在乎你是否回应的人聊天。他们像是在自说自话，即使你不回应，他们也不会感到不被重视。好朋友之间即使不说话也很舒服，就是这种感觉。

如果你是“提供价值”的状态，即一个人很坦荡、真诚地沟通，带着友善，人们就会察觉到你不是在操纵他们。你不会刻意要求对方如何回应，只是自然地沟通，分享你的想法和价值观。

你给自己设定的成功标准，绝对不是对方如何回应，而是**你讲出了自己最真实的想法**。这就是所谓的“你可以用任何东西做开场白”，你只是在表达个性，不管外界如何回应，你都相信自己有能力应对。

如果你和别人交往时只是在提供价值，那么你对他就是一种吸引。如果你是在索取价值，对方就会想逃开、会讨厌你。

我们的世界存在一些普遍的价值观。有些人沟通只是为了操纵或控制你，给你讲大道理。他们这样做的目的，绝不是为了提供真实坦荡的信息。你的大脑会感觉到被操纵，然后就想逃开，因为你感觉到他在索取价值。

所以，不要有操纵的心态。你只需要专注提供价值。有些人只是很真诚、坦荡地沟通，他们在不断提供自己的思维观点。在很多细节上，比如微表情、状态、握手方式，这个人都是一致的。他没有想过用正在做的事情来吸引外界或获得好的反馈，也就是说他没有索取。

如果你能有这种状态，就算你只是在谈论树木或者很普通的话题，只要你的人是真实的，人们就能感受到价值，你就会吸引一部分人。

也就是说，如果我们是在提供价值，不管你说什么、做什么，你都是有价值的。这就是为什么你要抛弃那些刻板的“话术”和“惯例”，因为它们正是你的束缚，让你感觉“我不能这样，我不能那样”。

其实，当你活在当下，知道自己是谁，知道自己的价值时，不管你做什么，意义都会被你吸引。所以，记住：**提供价值**。

接下来，要学会“自嗨”。如果你想让别人感到快乐和受吸引，你得先让自己快乐，也就是学会自我娱乐。

一开始学习社交时，我们总会去研究对方可能喜欢的话题，比如香水。但问题是，如果你自己不喜欢这个话题，却要讲给对方听，你就会表现得很别扭，并且很期待他的回应，因为你考虑了这么久，“索取模式”就开启了。

而我想要你们去尝试的，就是“自嗨”：讲自己觉得好玩的东西。你将得到的结果是令人震惊的。

你要做的，就是和对方讲你感兴趣、觉得好玩的东西。记住，你讲任何话题，是因为你自己想说，并且你从不依赖这个话题来获得外界的好感或改变外界对你的印象。也就是说，你所有的表达都是没有目的性的，你只是想表达而已，没有向外界索取价值。

当你能够这样自由自在地表达，那么你的行为本身就是一种吸引。记住：要讲让自己开心的话。

如果一个人喜欢时尚，你不一定要和他说时尚的话题。他其实感兴趣的是“你对什么东西感兴趣”。所以，如果你一直玩得很开心，然后你把他带到酒吧里见你的朋友，其实这样做你就已经在提供价值了，而且提供了相当多的价值。

所以，一个关键的转变是：你要讲一些自己喜欢的话题，或者讲自己喜欢的笑话，让自己开心。你带着对方来到你的圈子里玩，这就是提供价值。

但是，如果你很生硬地想要打入对方的朋友圈，这其实就是在索取价值。能理解吗？如果你和朋友玩得正嗨，突然有个陌生人很生硬地想加进来，你会不会觉得这个陌生人挺让人讨厌的？因为他是在索取价值。

但是，如果你和朋友在酒吧里玩，那些玩得最嗨的人要来和你们一起玩，你是不是会觉得“欢迎你”？感觉根本不一样。为什么？因为你觉得后面这个人（玩得很嗨的那个）来和你们一起玩，他是来提供价值的。所以，每个人都只会喜欢提供价值的人。

这就是为什么坦荡真诚的沟通，永远要好过于用技巧去伪装自己。再次强调，永远不要去做“变色龙”。你只需要保持你的一致性，保持中心感，让对方围绕着你。你要提供一种稳定性，让对方围着你转。

有一个比喻：你是一棵橡树，对方是围绕着你转的松鼠。你知道你是谁，你不用根据别人的反应来决定自己是否应该自信。

如果说，你有一天到了这个水平——全方位地接受自己所有的一切，让自己开心——那么你就可以轻松地做任何事。你没有需求，你根本不用去证明自己。也就是说，你达到了一个非常流畅的状态，你只是感觉很好，这就够了。

其实，到了这个时候，你已经完全不需要“状态”了，你已经完全没有了“状态”这个概念。也就是说，你会真正强大到相信异性会爱上真实的自己。你再也不在乎自己“在不在状态”了，哪怕不在状态，也依然无所谓。

我们看看那些真正非常自信的人，他们并不是一直处于打鸡血的状态。有时候他们就很淡定，一副“不管我怎么样，异性都会喜欢我”的样子。所以，你根本没有必要纠结自己在不在状态。

当你的心态越来越稳，你就能保持一致性，然后用你的身份及真实，来和这个世界产生互动。如果你能够做到这样，那么这个世界都是你的。

---

本节课中我们一起学习了“小我”的不可持续性，以及如何通过“活在当下”和“提供价值”来回归“本我”状态。核心在于保持真诚、一致，专注于自我表达和创造快乐，而非向外索取认可。记住，当你成为那棵稳定的橡树，整个世界都会围绕你旋转。
