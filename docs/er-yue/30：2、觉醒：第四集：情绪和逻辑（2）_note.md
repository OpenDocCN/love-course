# 课程30：觉醒 - 第四集：情绪与逻辑（2）🧠💖

在本节课中，我们将继续探讨如何通过**情绪**而非**逻辑**来建立“有趣好玩”的互动体验。我们将学习如何在社交中调整情绪与逻辑的比例，并理解如何通过创造积极的氛围来吸引他人。

---

上一节我们介绍了“有趣好玩”根植于情绪层面。本节中我们来看看如何在实际互动中应用这一原则，并避免过度逻辑分析带来的问题。

“有趣好玩”涵盖广泛的想象力，但它必须建立在情绪基础上，而非逻辑基础上。建立有趣好玩的状态需要你将全部注意力放在**过程**上，而非**结果**。当你关注结果时，便回到了逻辑层面。

因此，你需要用情绪去感受外界，同时停止评判。一旦开始评判，你就会陷入逻辑。你只需感受事物本身的样子。

例如，遇到乞丐时，若感到同情，可以给他买些吃的。无需分析“他一定是因为某些原因才落此境地”。你的分析或许有道理，但会破坏当下的情绪连接。

这类似于聚会时大家玩得开心，你忽然说“可惜明天要上班”。这句话虽正确，却会破坏氛围，消散快乐的能量。这就是逻辑的侵入。

逻辑很有用，它能解决问题，适用于工作、学习和技能分享。但在感受生命、社交及人际交往时，应停止过度分析。

并非社交中要完全摒弃逻辑，否则可能语无伦次。这里指的是整体层面上的**平衡**：分析居多还是感受居多？当分析占比过高时，你难以获得更多快乐。

在社交或与异性相处时，建议逻辑仅占三成，用以保持基本正常；其余七成应用于感受情绪、感受快乐、感受外界。这样便能建立有趣的互动。

以下是“有趣好玩”的表达示例：

*   你可以对女友说：“我恨你。你太温柔，让我天天沉溺，现在都不想伤心了。我讨厌你让我这么舒服。”
*   你可以对他人说：“你很有活力，感觉能当我的保镖。”
*   你可以故意冷落某人，在其情绪低落时忽然拉回身边。

总之，要像孩子一样玩耍。你可以拥抱对方后忽然推开，说：“你身上太热，是不是荷尔蒙在窜动？你真‘好色’。”也可以在亲吻后推开对方，说：“别太贪心，担心伤到你。”

要学会活在当下，感受此刻有趣的事物，并培养自己有趣的一面。当你活在当下，逻辑便会下沉，有趣的特质自然散发。

你还可以通过角色扮演来表达有趣，例如严肃地说：“小王，晚上来我房间一趟，有些人体艺术问题要探讨。”或者故意表现激动、好色或滑稽。

核心是让**逻辑下沉**，让**好玩有趣上升**。若方法得当，任何涉及影射的事情都能变得有趣。

当然，这一切都需要**校准**。你必须考量当下氛围，你的情绪和感觉是最好的指南。例如，在严肃氛围中开玩笑就不合适。

校准意味着向对方展示：你无需求感、聪明、有趣、有智慧、有社交意识、是中心人物且可靠。你是一个性感、令人犹豫（吸引人）的人，是众人愿意围绕的对象。

最重要的是，互动是**双向筛选**过程，双方都感觉在向对方展示自己。若不理解，请多读几遍。

让我们想象两组人与异性互动的情景。

第一组人紧张地走出去，对周围异性感到不安，因为他们不断自我分析，试图通过思考改善状态。但状态永远存在于**行动**中，而非思考中。

他们进入场景后先检查场地，四处搜寻认识的人或更好的异性，显得需求感很强。其行为让他们感觉自己不属于那里，最终靠墙坐，保护性地拿着酒杯。

行为与思维模式是共生关系。改变一端，另一端也会变。他们以“搜寻”为出发点行动，行为告诉大脑：“这里很陌生，我很紧张。”大脑接收此信息，思维随之改变。

熟悉环境本身没错，但出发点应是“检视自己的领地”，而非“陌生环境里东张西望寻找目标”。出发点错误导致行为错误，最终改变思维。

这组人相互不交谈，凝视环境，希望从他人处获得价值与刺激。他们可能搭讪了几个人，但仍处于“想从他人处获取价值”的状态。

他们未意识到：若不能在自身组合内创造良好氛围，也无法为他人创造。因此他们的接近显得做作无聊，微妙表达出自身价值不足，仿佛在请求他人允许加入。总之，他们未提供价值，而是在索取价值。

另一组人舒适地走出去，两个人在相遇他人前就已玩得高兴。进入场景后，他们相互开玩笑，组合内充满有趣好玩的氛围，周围人能明显感受到。

这组人也会搭讪他人，甚至他人会主动转向他们，因为他人早已注意到他们很有趣。好的氛围吸引人。他们讲自己的笑话，他人虽不完全懂，也可能跟着笑。他人越笑，越对他们感兴趣，于是陷入他们的框架，想与他们结盟。

他们知道若想，可与场内任何人交谈。一切都很对。这就是价值——在情绪基础上发展出的氛围。当你总能创造有趣氛围时，你总能成为社交中心，总有人围绕你。

当然，若每次创造好氛围只为成为中心、希望他人围绕，此技巧便失效。你的优秀应是为了**成为那样的人**，而非向外界证明自己。

最终，你必须相信：外界喜欢你是喜欢你本身，而非其他。这意味着你可直接上前与他人交谈，活在当下，无需多想，想到什么说什么，不努力争取特定结果，享受过程即可。

其次，你的**自我实现预言**：当你接近异性时，要能感受到内在的富裕。你必须对此坚定不移地相信并完全内化。当你完全相信自己能与任何人交谈，能自信地走上前，你就会拥有良好的预期与反应。

你的所有信念都会非常积极，从而形成正向循环，状态越来越好。
