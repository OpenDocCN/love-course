# 30：2、觉醒：第四十一集：认识你的情绪（2）_note

在本节课中，我们将继续探讨情绪的本质。我们将学习如何与自己的情绪建立联系，理解逃避情绪的常见方式，并掌握一种通过“抽离观察”来管理情绪的有效方法。核心目标是帮助你从被情绪支配的状态，转变为能够觉察并引导情绪流动的主动状态。

上一节我们介绍了情绪是你灵魂的一部分。本节中，我们来看看当情绪（尤其是痛苦情绪）出现时，我们通常如何反应，以及如何建立更健康的应对方式。

你的情绪通过一种戏剧性和亲密的方式来到你的面前。痛苦的情绪是你自己内在需要治愈的部分。那些滋养你的情绪，例如快乐、开心、愉悦，则是你自己内在已经治愈的部分。

当你随时察觉你所感受到的一切，那么你就在与你的内在进行持续的交流。如果你没有感受到所有的一切，那就说明你没有和你的内在进行交流。

想要去跟你的内在交流可能非常难，有时甚至非常痛苦。例如，当你胃痛时，要去倾听内在的声音是很困难的。

大部分人在愤怒或不开心的时候，会向其他人咆哮、甩东西，或找各种方式来泄愤。当你与内在交流的痛苦越强烈，想要逃避交流的冲动就越强烈。

以下是两种基本逃避体验感受的方式：

第一种方式是逃避到你的思想当中，也就是逃避到你的思维当中。人们相互评判，是因为相对于去面对他们境况中的痛苦，去相信别人要对他们的境况负责更容易。例如，产生“都是因为你害的”这种想法。

这种逃到思想里的方法，就像在一个伤口上涂麻醉剂。伤口并不会消失，它还是需要处理。当麻醉剂失效时，伤口和痛苦依然存在。

当与你的灵魂交流的痛苦变得强烈时，你处在一个关键点上。你所面临的选择，或者是通过逃进思想和活动来减轻痛苦，或者是将注意力放在内在，从而发现你不舒服的根源并治疗它。

把注意力放在外在，是希望通过外界的力量来治愈自己，然而这是不可能的。把注意力放在内在，是希望通过自己的力量去解决问题。外在的力量不可能解决你内部情绪的问题，因为你无法控制外界。

每一个思维都伴随着情绪。但是，如果你不愿意去发现你的身体在当下的感觉，你就会将自己一次又一次带到更多的思维、想法、计划和评判里。现如今有大量的思维性活动被创造出来，就是为了转移人们的注意力。

所以，下一次当你感到愤怒、感觉低人一等或高人一等、或不开心时，尝试不要转移注意力，花两分钟体验它。

如果你没有意识到自己身体的感觉和你在想什么，你就没有意识到当下，你就不会有力量。你的注意力就被你面前的东西所占据。

当你没有意识到自己情绪的时候，你的注意力是在周围环境上的。你体验中的主要要素，也就是那些一直在流经你的情绪，被你遗漏了。

虽然你可能觉得自己非常清醒，但是只要你还没有注意到自己的内在体验，对你的生命来说，你就处于昏睡中。没有情绪觉察，你就无法意识到当下，因为当下包含了你的情绪。

不过，很多人对他们的情绪认同非常强烈，以至于他们被情绪淹没了。你会发现有些人毫无节制地哭或笑，他们被恐惧、愤怒这些情绪所包围。这种时候，他们意识到的内在体验很少，或者一点都没有。

当你察觉到你的内在风景却无法从中抽离时，你完全沉浸在你的情绪之中，你无法从中退后以看清它们。

所以要学习从你的感受中退后一步，这样你就不再被它束缚，或者不再无法察觉到它只是一种感受。退后一步，让它能通过你而不会渗透你，它就不会阻止你去采取行动，不会让你产生负面思想和封闭情绪。

所以，你要做自己情绪的旁观者，要去观察自己的情绪。

当你处在任何负面情绪时，都可以使用以下练习：

首先，想象你在一条情绪的河流之中。
然后，想象你走出了河流，走上了一座桥。
你向下看着这条河流，看着它在你下面潺潺流动。
河流中的水代表你的情绪。让这水在你下面流动，而你在上面观看。
同时，去感觉能量的河流经过你的身体。

抽离能够让你在生活中保持对自己感受的觉察。当你抽离的时候，你的情绪就像水通过一个水管一样流过你。你是那个水管，情绪来得快去得也快。

当水龙头打开后，水不停地流过你的身体。你的情绪龙头从来不会光滴，那些你所感受的负面情绪也不会停止。

当你以这种方式去看自己的情绪时，你就能够从中抽离，而不会被它们控制。当你能够做到这一点时，你的情绪就能够从你的身上流过，就像你在沐浴时水流过你的身体一样。情绪来也匆匆，去也匆匆。

当你没有从自己的情绪中抽离时，你就无法将它们与你自己分开，情绪将会支配着你。

当你对自己的情绪有所觉察时，你就处在一个主导的位置上，你就可以去改变流经自己能量系统的能量被处理的方式。你的情绪不会将你卷走。

你的情绪给你带来了信息和重要的数据，而你无法沉浸于情绪的同时接受到这个信息。例如，当你很愤怒时，你无法一边愤怒地吼叫，一边意识到你身体的感受。你会觉得那个情绪就是你，它在支配你。

所以你必须要做一个选择：要么站在水中，让你的情绪来决定你的言行；要么站在桥上，看着你的愤怒、痛苦流过自己。

当你站在水中的时候，你的愤怒就是你的主人。这就是所谓的“失去理智”。而当你站在桥上去体验自己的愤怒时，不管你身体里面的感觉有多么痛苦，你都是自己愤怒情绪的主人，你在控制你的行为。

每一次你这样做，你的愤怒就失去了对你的控制力，而你获得了对愤怒的控制力。于是，你的内在力量就被你创造出来了。

当你意识到你的情绪，又意识到周围在发生什么，于是你就进入了当下。

你越是抗拒痛苦，就会有更大的愤怒更频繁地出现来掩盖痛苦。一个持续愤怒的人，是一个处于持续痛苦中的人。

愤怒让你有双重的痛苦：愤怒的体验本身就是痛苦，而愤怒之下所掩盖的痛苦则更加令人痛苦。在你能够面对并体验你愤怒下面的痛苦之前，你会继续愤怒下去。

你的愤怒并不是对某一个情境的抗拒，它是对你所体验的痛苦的抗拒。它在抗拒世界没有变成你想要的样子。愤怒就是因为你没有将世界和别人安排成你想要的样子而产生的挫败。从本质上来说，就是因为感到无力。

当你开始向自己的痛苦体验敞开自我时，你就开始进入自己愤怒的表面之下，你开始转化自己的生活。你变得不像以前那么固执或自以为是，你变得更加敢于呈现脆弱，接受其他人和他们的痛苦。你开始融入宇宙的柔软之中。

不过在此之前，你必须突破你制造的、让自己远离自己情绪的防御。简单来说，就是你要去挑战你的愤怒，才能够开启一切。

当你挑战自己的愤怒时，你就启动了一个此刻你看不到的更大的进程。你开始将自己的愤怒的根源拔出。

最先出来的是你的愤怒之下被掩盖了很久的痛苦。愤怒，一是对自己的无能，第二是为了掩盖痛苦，这就是愤怒的本质。

然后是这痛苦之下的恐惧。这恐惧是因为你无法控制那些你认为对自己的安全和幸福很重要的东西而产生的。

愤怒的核心起因就是自我价值的缺失，它是无力感的体验。无力感就是将自己看得没有价值。

当你感到没有价值的时候，你很畏惧你的生活。而当你畏惧你的生活时，你会持续地活在痛苦当中，试图将你的生活改变成你认为它应该是的样子。当这痛苦很深刻时，你就会用愤怒来掩盖它。

当你感到生气的时候，停止你正在做的，停止你正在说的，停止你正在想的，关注在你的感受上面。这是很不容易做到的，但是它值得你去尝试。

当你被强大的情绪所左右的时候，停止说话，停止行动，开始感觉自己的内在，将那股能量全部转入你的意识之中。这就是内部觉醒的开始。

所以你会发现，很多不够成熟的人一直被情绪所支配。而足够沉着、自制、成熟的人，是能够支配自己情绪的。

大街上为什么那么多人打架？小孩子为什么不能控制自己？就是因为他们觉得情绪就是他自己，他和情绪产生了强烈的认同。

但是，情绪只是你身体对你思维的反应而已，它不是你。否则我们为什么要说“我的情绪”？“我”才是主体，“情绪”才是客体。所以，情绪不是你。

“我的手机”、“我的衣服”，衣服不是我，衣服是客体。所以，“我的情绪”，情绪也是客体，它不是你。

所以，千万不要把自己认同为是你的情绪，或者把情绪认同为你。

首先你要做的，就是打破这个认同：你的情绪不是你，它只是你身体对思维的一个反应。

当你能够认识到这一点，然后按照这节课的内容来，慢慢地，你就能够开始支配你的情绪，而不是被你的情绪所支配。

---

本节课中，我们一起学习了情绪管理的核心：**觉察与抽离**。我们认识到痛苦情绪指向需要治愈的内在部分，而逃避到思维或行动中只能暂时麻醉。通过练习像“站在桥上看情绪河流”这样的抽离观察法，我们可以不再与情绪强烈认同，从而获得主导权。关键在于理解：**情绪 ≠ 你**，情绪是流经你的能量和信息。持续练习将注意力从外在转向内在感受，是培养情绪觉察力、进入当下并创造内在力量的开始。
