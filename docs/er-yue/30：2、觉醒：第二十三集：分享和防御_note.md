# 30：2、觉醒：第二十三集：分享和防御_note

在本节课中，我们将要学习社交互动中的两个核心概念：**分享**与**防御**。我们将探讨它们如何影响你的能量状态、人际关系以及个人成长。理解并掌握这两者，能帮助你在社交场合中更加从容自信。

上一节我们介绍了“合作”与“进攻（主导）”，本节中我们来看看“分享”与“防御”如何塑造你的社交表现。

## 概述：分享与防御的本质

分享是指向外界提供价值，它代表一种开放的态度。防御则是一种心理上的自我保护机制，通常在感到被评判或攻击时触发。两者的状态截然不同，深刻影响着你的社交能量。

## 1. 分享：成为价值的源头 🎁

分享意味着你处于开放状态，对外界没有设防。你没有提前预设，内心充满安全感。

**分享**的本质是：`能量向外扩散`。当你的能量向外扩散时，代表你是价值的源头，你在为周围环境提供价值。

以下是分享的具体表现：

*   提供快乐、积极的情绪能量。
*   贡献有趣的话题或奇妙的观点。
*   乐于帮助他人，不计较得失。

分享让你吸引他人，变得受人欢迎。你想吸引更多人、变得更快乐积极，就必须成为价值的源头。

## 2. 防御：自我的束缚与能量内收 🛡️

防御与我们之前讲的“防守”不同。防守是被动的，而防御是主动的心理**自我防御机制**。大部分人在社会中持中立态度，但受到攻击时容易陷入防御状态。

社交中的防御特指**心理自我防御**。它与自尊心强弱直接相关。

*   **低自尊（玻璃心）**：容易将外界信息（甚至是夸奖）理解为攻击，随即产生情绪反应，进入防御状态。公式表现为：`外界刺激 -> 理解为攻击 -> 情绪滋生 -> 自我防御`。
*   **高自尊**：拥有足够自信和安全感，不易被外界评论影响情绪。即使面对嘲讽，也能轻松应对，而不会进入防御状态。

在社交场合中，进入防御状态会让你能量内收，并启动**自我监控**（不断审视自己的言行），这是一种低价值行为。

以下是陷入防御状态的标志：

*   感觉被攻击或被评判。
*   担心留下不好印象。
*   不断自我维护、辩解。
*   思维和能量向内收紧，变得不知该说什么。

防御模式其实是自我的一种束缚，它让你向外界索取认可（“都给我”），并阻止你前进，让你待在舒适区。

## 3. 如何应对：从防御到分享的转变

当你感觉被评判且无法置之度外时，就会进入防御状态，能量越来越紧绷。关键在于成为自己能量的旁观者。

**核心方法是**：`切断负面反馈 -> 保持内在状态 -> 继续积极分享`。

如果你收到负面评论，但对群体中的其他人依然热情快乐，这就切断了那个评论对你的影响。你的积极情绪完全不依靠外界，这就是完全的自我。

面对不喜欢你或试图让你不开心的人，远离即可。不需要给他们太多注意力，但你的状态不受他们影响。

真正的自信不需要刻意维护。你可以选择反击，但心态上并非防御性的“维护自己”。例如，面对无礼嘲讽，反击的点在于对方，而非辩解自己。

## 4. 一个关于分享的故事

前阵子有一个广为流传的戏剧性视频：一位穿着名贵西装的CEO在披萨店误以为乞丐分享的披萨是自己的，并对乞丐颐指气使。当最后店员将真正的披萨端给CEO时，他才震惊地发现，自己一直在吃的是乞丐的披萨，而乞丐甚至愿意将最后一块分他一半。

乞丐只是笑着说：“我只是乐于分享。”

这个故事虽然经过编排，但揭示了深刻的道理：那个乞丐拥有**非常坚定的现实**，他乐于分享，为周围人提供价值。这应该是我们学习的对象。

## 5. 积累能量，突破防御

在社交或与异性相处时，很多人因害怕被拒绝（“打枪”）而感到被评判。那可怜的自尊会阻止你进一步行动，这时，你自己成了最大的敌人。

突破这个障碍需要决心和行动。

**成长的关键在于连贯的能量积累**：`持续行动 -> 承受失败 -> 能量累积 -> 突破临界点`。

如果你能摒弃脑中假想出来的障碍，持续行动，让过度思考退居幕后，那么每一次“失败”都是一次能量积累。当能量积累到临界点，你会突然“不在乎了”，踹开所有阻挡你的东西，进入“状态”。从此，你将不再在意外界评价，行动进入螺旋式上升通道。

请注意，这种累积必须是连贯的、高密度的（例如一小时内多次尝试），才能带来显著成长。

## 总结

本节课中我们一起学习了社交中“分享”与“防御”的核心区别。

去到任何社交场合，请务必记住这三个词：**合作、进攻（主导）、分享**。将它们作为你的行动指南，而非竞争、防守与防御。

当你能做到合作、进攻、分享时，你一定会看到一个不一样的世界。你会成为价值的源头，吸引他人，并真正成为自己情绪和状态的主人。
