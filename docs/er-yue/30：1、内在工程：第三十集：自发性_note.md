# 30：1、内在工程：第三十集：自发性_note

在本节课中，我们将要学习“自发性”这一概念。自发性是指我们的反应和行为由内而外自然散发，而非经过逻辑过滤或刻意设计。掌握自发性，能让我们的行为更自然、真诚，并在社交互动中建立更真实的连接。

## 什么是自发性？

自发性是指你所有的反应都由内而外自主散发，而非经过逻辑过滤或刻意设计。当你能够这样做时，你所有的行为都会充满自然性，而非刻意性。

自发性是建立在一致性基础上的一项特质。它代表了你所有的行为都是自发的，而非刻意为之。

## 自发性与刻意行为的区别

以下是自发性和刻意行为的关键区别：

*   **自发性行为**：你的第一反应。例如，你对某个行为感到厌烦并立即表现出来，他人会明白这是你的标准和真实感受。
*   **刻意行为**：经过思考、权衡利弊后表现出的反应。这种行为经过“包装”，他人能看出其中的刻意与不真实，认为这是你的“伎俩”，而非真实的你。

之所以能被看穿，是因为刻意行为伴随迟疑和不自然的表达。

## 自发性的适用范围

这里需要做一个重要区分：自发性主要针对**社交互动**，而非处理工作或解决具体问题。

例如，制定计划需要逻辑步骤（一二三四五），这不属于自发性范畴。自发性是一种由内在过程驱动、没有外力干预的自发状态和品质。你的行为不是由外在刺激引发，而是发自内在。

## 理解我们的内在过程

要做到自发性反应，有一个重要前提：行动者能够快速完成一个内在过程。通常，我们处理事情的内在过程分为四步：

1.  **观察**：感官接收关于事件及其变化的信息。
2.  **解析**：在情感和认知上对事件进行解读、下定义、贴标签。
3.  **回应**：在身体内酝酿做出行动或回应的状态。
4.  **表达**：在脑中检验并修改用词用句，最后进行口头陈述。

我们每一刻的行为，都可以理解为与正在发生的事情进行互动，需要经历`观察 -> 解析 -> 回应 -> 表达`的过程。这个过程顺利时，仅需一到两秒。

## 为何会失去自发性？

然而实际情况是，我们常常会在某个环节“卡住”。对于东方人，特别是中国人，很容易卡在“表达”环节。

当发生特定事情时，你可能会身体僵住、不知所措，或总是做出单一反应。这属于潜意识里的偏好和行为模式。

无法自发表达，通常有以下几种呈现：

*   **有所迟疑**：在发表观点时担忧表达是否准确，错过最佳时机。
*   **没有反应**：面对褒奖、贬抑等外在刺激时，仅以尴尬微笑回应，不知如何应对。
*   **事后不满**：事发当时没有行动，事后后悔“我当时应该那么做”。
*   **内在挣扎**：对事情有内在反应，但因社会压力无法找到恰当的呈现方式。

以上任何一种方式，都对内在是一种消耗。我们会感到生命缺少了什么，因为在每一个当下，都没有完成该完成的事，内心充满缺憾感。这导致僵化或抱怨，而非建设性与创造性。

## 案例分析：如何倾听与回应

让我们通过一个例子，对比两种不同的内在过程。假设一个姑娘泪眼婆娑地向你诉说被甩的经历。

**如果按照四步流程，内在过程可能是：**

1.  **观察**：听她诉说。（这是必要的）
2.  **解析**：内心活动开始。“那个男生真是人渣”、“这女孩好蠢”。（一旦开始贴标签，注意力就分散了，与真正的互动脱节）
3.  **回应**：酝酿回应状态。“我应该显得更难过、更正义”。（这依然活在他人的评价里）
4.  **表达**：边说话边在脑中修改用词，想留下好印象。（这是不自信，对外界有所求）

这样做的结果是：陈述可能不够连贯，底气因顾虑增多而减少。

## 自发性的核心：两步法 🎯

自发性其实只需要两步：**观察**和**陈述第一感受**。

继续上面的例子：
1.  **观察**：静静倾听她的全部陈述，期间不做任何解析、回应或表达酝酿。将全部注意力放在她的陈述上。这就是真正的倾听。
2.  **陈述第一感受**：听完后，直接说出你的第一感受。
    *   代码示例（示意）：`print("我感觉你现在很脆弱。")` 或 `print("我感觉你现在很难过。")`

这是最好的答案吗？可能不是。但这就是你的自发性。因为你无所求，所以不会处心积虑。如果你忽然想到一句意味深长的话，也可以直接说，但不会有任何顾虑。**你的第一感受是什么，就说什么。** 这就是自发性，它让你显得更真诚、实在。

## 常见疑问与核心基础

对此，可能有人会产生疑问：

*   **问**：“如果我听完想笑，难道真的要说出来吗？”
    *   **答**：这通常不是自发性问题，而是个人素养问题。缺乏同情心和同理心，对世界没有善意。自发性应建立在**友善**的基础之上。
*   **问**：“这是不是说话不过脑子？”
    *   **答**：并不是。你之所以在脑中权衡利弊，是因为你对外界有所求。你应该解除这些限制，**以友善为基础**，在此之上自发地行为举止。这样你将获得更好的社交能力。

## 总结

本节课中，我们一起学习了自发性的核心要义。

自发性就是自然地让事情发生，这原本是一个自然的过程，但在内在千万个不同念头面前变得障碍重重。`解析`、`回应`、`表达`这些步骤都在不断消耗你的能量，让你脱离与外界真正的互动。它们不是必要的，反而阻碍你找到与外界相处的最佳方式，也阻碍你找到更好的自己。

因此，一定要学会自发性，即信任并表达你的**第一反应**。去掉那些不停的算计、处心积虑和权衡利弊，让自己变得更纯粹。学会自发性，你的世界将更加纯粹，你也会挣脱诸多束缚，变得更自由。
