# 课程30：认识自己的思维 🧠

在本节课中，我们将要学习如何认识自己的思维。思维是我们生活中难以分割的一部分，但大多数人的思维存在偏差和扭曲。生活的品质与思维的质量密切相关。低质量的思维会让我们在金钱和生活品质上付出高昂代价。因此，我们必须对良好的思维进行系统培训。

## 什么是批判性思维？

上一节我们介绍了思维的重要性，本节中我们来看看什么是批判性思维。批判性思维是一种自我引导、自我训练、自我监控以及自我矫正的思维。需要特别说明的是，这里的“自我监控”是指监控自己的思维过程，这与之前课程中提到的其他自我监控概念不同，请勿混淆。

## 心灵的三个功能

要理解思维，首先需要认识心灵的三个不同功能：**思维**、**情感**和**欲求**。

### 1. 思维的功能
思维的功能是产生意义。它让我们弄懂生活中各种事件的意义，并将事件按照已命名的类别进行整理。

### 2. 情感的功能
情感（即情绪）的功能是监控或评估思维所产生的意义。它评估各类事件对我们的生活有多积极或多消极，并给出我们赋予它们的意义。

### 3. 欲求的功能
欲求的功能是激活能量以发起行动。它不断地告诉我们某事物是否值得争取，从而决定我们是否要为之行动。

## 三者之间的关系

在思维、情感和欲求之间存在着密切的动态联系，其中任何一者都在不断地影响着另外两者。

当我们认为正在遭受威胁（**思维**），我们便会感到恐惧（**情感**），并且明显地想要逃离或攻击（**欲求**）。

当我们认为参加一个会议将会浪费时间（**思维**），那么我们就会想要避免参加这个会议（**欲求**）。

你的思维会对一件事下定义，而你所下的定义会滋生相对应的情绪，你的情绪则影响着你的行为（欲求）。

## 从无意识到有意识

每个人都生活在与自己心灵的特殊而亲密的关系当中，但大部分人没有意识到这层关系。你们接下来要做的，就是将这种**无意识**的关系转变成**有意识**的关系。

我们所有的活动都源自我们的内部观念，这些观念是关于自己是谁、是什么、当下体验到什么、将要到哪里去、从哪里来等等。

对每一个我们心灵所相信的积极思想，它都倾向于自然地产生一个积极的情绪与之匹配。反之，对每个消极的思想，我们的心灵都倾向于产生一个消极的情绪。

## 思维如何塑造体验

我们体验到愉悦、快乐、沮丧、痛苦、困惑、欲望、激情、冷淡等各种情绪，这是因为：
1.  我们赋予每个情境我们所体验到的意义。
2.  我们用一种特定的方式思考它。
3.  我们将当前情境与我们以往认为的类似情境中所产生的情绪联系起来。

我们形成的意义可能基于深刻的洞见或客观事实，也可能是自己的一种偏见。

例如，两个处于同一环境中的人可能会有完全不同的反应。一个人可能体验到沮丧、痛苦，而另一个人可能体验到好奇、兴奋。

## 实例分析

以下是关于思维如何影响情绪和行为的例子：

想象有两个员工都面临相同的任务：改进办公室流程以提升工作效率。

*   **员工A**感到愤怒和不满，因为他认为工作进行得很好，却被要求改变。他赋予了这个任务负面意义，认为它没有必要且浪费时间。在这种负面思维下，他在过程中体验到负面情绪。
*   **员工B**则欢迎提升机会的到来，将这一情境界定为发挥创造性的机会。他用发展的眼光看待这个过程，从而产生了积极的情绪。

每件事都有若干视角。一个人应付挑战是困难还是容易，是决定接受还是彻底回避，最终是成功还是失败，从根本上都取决于这个人的思维如何解释这个情境。

## 掌控你的思维

不同的思维和行动产生不同的情绪。所以，你的思维基本上决定了你的人生。

当我们了解思维、情感和欲求之间的相互影响关系，当我们看到体验到的每一种情感状态都有一个相应的思维出现，这一思维又驱动我们去采取某些行动，那么我们便能开始分析处于我们情绪和欲求影响下的思维。

例如，当你在开会时感到沮丧，你就能问自己：
*   你实际上是对什么感到沮丧？
*   是什么思维导致了你的沮丧？

思维、情绪和欲求三者相互影响。改变其中一端，另外两端也会跟着改变。
*   `思维 -> 情绪 -> 欲求`：思维的定义产生情绪，情绪决定接下来的行动（欲求）。
*   `情绪 -> 思维 -> 欲求`：改变你的情绪也能影响你的思维及欲求。例如，当你很开心，你看待事件的角度可能改变，最终欲求也随之改变。
*   `欲求 -> 思维 -> 情绪`：你的行为（欲求）发生变化，那么你的思维也会变化，接着你的情绪也会跟着变化。

所以，思维、情绪和欲求的顺序并非一成不变。改变其中任何一端，另外两端都会发生改变。

## 总结

本节课中，我们一起学习了如何认识自己的思维。我们探讨了批判性思维的定义，分析了心灵的三个功能（思维、情感、欲求）及其相互作用。我们了解到，思维产生意义，情感评估意义，欲求激发行动。三者动态关联，相互影响。通过实例，我们看到思维如何决定我们对情境的解释，进而塑造我们的情绪和行为。最终，我们认识到，**你的思维控制着你生活的每一个方面**。因此，学会审视和引导自己的思维，是提升生活品质的关键。
