# 课程一：Python基础入门 🐍

在本课程中，我们将学习Python编程语言的基础知识，包括变量、数据类型和基本操作。

## 概述

Python是一种易于学习且功能强大的编程语言。本节将介绍其核心概念，帮助你打下坚实的基础。

## 变量与数据类型

上一节我们概述了Python，本节中我们来看看如何存储和表示数据。

在Python中，变量用于存储数据。每个变量都有其特定的数据类型。

以下是常见的数据类型：

*   **整数**：表示没有小数部分的数字，例如 `10`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`。
*   **字符串**：表示文本，需要用单引号或双引号括起来，例如 `‘你好’`。
*   **布尔值**：表示真或假，即 `True` 或 `False`。

你可以使用等号 `=` 为变量赋值，例如 `name = “小明”`。

## 基本运算

了解了如何存储数据后，我们来看看如何操作这些数据。

Python支持多种基本运算。

以下是主要的运算符类型：

*   **算术运算符**：用于数学计算，如加 `+`、减 `-`、乘 `*`、除 `/`。
*   **比较运算符**：用于比较值，如等于 `==`、大于 `>`、小于 `<`。
*   **逻辑运算符**：用于组合条件，如与 `and`、或 `or`、非 `not`。

例如，公式 `3 + 5 * 2` 将先计算乘法，再计算加法。

## 控制流程：条件判断

程序并非总是直线执行。本节我们将学习如何让程序根据条件做出不同决策。

`if` 语句用于条件判断，控制代码的执行分支。

其基本结构用代码描述如下：
```python
if 条件:
    # 条件为真时执行的代码
else:
    # 条件为假时执行的代码
```

## 总结

本节课中我们一起学习了Python的基础知识。我们介绍了变量与数据类型、基本运算以及使用 `if` 语句进行条件判断。掌握这些概念是编写更复杂程序的第一步。
