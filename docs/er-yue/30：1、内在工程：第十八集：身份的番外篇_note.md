# 课程30：内在工程 第十八集：身份的番外篇 🧩

在本节课中，我们将探讨关于自我身份认同的几个延伸话题。这些内容有助于我们理解情绪波动和自我价值感的根源，并学习如何建立一个更稳定、更健康的自我认知。

## 关于身份的波动

上一节我们介绍了身份认同的基本概念，本节中我们来看看身份认同不稳定的表现及其影响。

很多人的情绪缺乏稳定性，像火山一样时而高涨时而低落。情绪高涨时，感觉自己无所不能，前途光明。情绪低落时，整个人像被关在黑屋子里，连很小的事情都难以完成。

你的情绪判定你的自身价值。在大部分人的认知里，仿佛只有“王者”和“失败者”两种模式。如果身份认同是“王者”，面对问题会有无穷信心，能想尽办法解决并最终成功。如果切换到“失败者”模式，就会丢掉“我能行”的信念，仿佛被绑住手脚，只能发挥一两成功力，最终以失败告终。

## 波动的原因：外部依赖的价值观

为什么自我认识会如此剧烈地波动？当我们对自身的认识基本建立在外界反馈上时，就很容易因为环境变化而波动。

中国传统教育带有价值条件化的倾向，即用别人的评价来定义自身的存在。做到他人眼中的成功、得到周围称赞，我们会觉得自己有价值。如果把事情办砸，让信任自己的人失望，仿佛就成了一个一文不值的失败者。

当这样的价值观渗透到心灵中，我们就把自我决定权交给了外界，在他人的眼睛里寻找自己的模样。工作顺利、收获认可时，就像照了美颜滤镜，看到一个“无所不能”的自己。接连搞砸项目、听到怀疑声音时，镜头就成了恶毒的哈哈镜，将缺点无限放大，让人陷入一无是处的自我怀疑中。

没有建立稳定的价值观，通过他人评价来认识自己，我们的状态就容易大起大落，随着外界反馈而波动，有时像国王，有时又像失败者。

## 建立边界：对事不对人

因为得失和自我评价绑在一起，所以难以做到“对事不对人”。能做到宠辱不惊的人，通常会把专业表现和个人价值分割得很清楚。

即便做出一笔成功的投资，他也不会膨胀地觉得自己很有商业天赋，认为做什么都能成功。同时，就算最近投资亏损，他仍然相信自己的眼光，有信心把后续工作漂亮地完成。

学会在个人成败和自我认识间划一条边界，把自己从事情的结果中摘出来。我们会发现，不管事情好坏，我们的能力就在那里，不会因为别人都说好而被放大，也不会因为别人都指责而缩小。这就是一个相对稳定的状态。

**公式表示：**
`稳定的自我价值 ≠ 单一事件的结果`

## 自卑与自大：一体两面

自卑和自大其实是一体的两面。往往越觉得失败的人，越会拼命地鼓吹自己，用外表的膨胀来对抗内心的无力感。

正因为潜意识里认为“我什么都不行”，所以才会紧紧抓住现实中证明自己的机会，通过成功来找到王者般的尊严与权力。但其精神世界的内核仍是深深的挫败感。因此，一旦碰到现实打击（如工作困难、人际冲突），那“国王”般的自信伪装就会被撕裂，重新陷入想摆脱却如影随形的自卑感中。

这种“无所不能”的感觉，可能是自卑者的保护色，用来掩盖说不出的痛苦与无助。

## 如何保持稳定的自我认同感

以下是建立稳定自我认同感的两个核心方法。

### 1. 建立稳定的内在价值体系

首先要找到一面独属自己的镜子，建立稳定的价值体系。

可以做一个小的练习：
1.  写下在你眼中，自己是怎样的人（五个关键词）。
2.  为每个关键词填充具体的生活实例。
3.  在以后的生活中不断刷新和校正这些认知，慢慢探清自己真实的状态。

同时，要学会区分“我是怎样的人”和“别人眼中的我”，将两者风格开来，保持清醒的自我认识。

### 2. 对事不对人，避免灾难化思维

为什么一个看似普通的挫折，会将我们从天堂打入地狱？那是因为我们给得失赋予了太沉重的含义。“得到”就意味着整个人被肯定，“失去”仿佛连呼吸都是错的，整个人没有存在价值，情绪因此在大起大落间切换。

更好的方法是告诉自己：**“项目搞砸了，只能说明很多方面没有考虑到，下次注意就行。我有能力把它做好。”** 不管结果好坏，将影响限制在这件事情里，不要牵连整体的自我评价。

**代码描述这种思维：**
```python
def evaluate_event(outcome):
    # 分析事件本身
    lesson_learned = analyze_mistakes(outcome)
    # 将事件与自我价值分离
    self_worth = get_stable_self_worth() # 这是一个独立、稳定的值
    return lesson_learned, self_worth # 两者互不影响
```

### 3. 直面并接纳自身的局限性

要保持稳定的自我认同感，也需要处理内心根深蒂固的自卑感。

当我们发现自己有很多缺点，觉得赶不上“别人家的孩子”时，自尊可能备受打击。承认自己的局限性是痛苦的，会让我们下意识地逃避，甚至使用“反向形成”的防御机制——即夸大自己，用膨胀感来掩盖内心的自卑，把真实状态用相反的方式表达出来。

下次感觉像国王一样时，不妨问自己一句：**“我真的是如此自信，还是在用相反的言行来处理自卑感？”** 慢慢打开自我保护的壳子，试着看见并接受那个不够好的自己，我们反而能减少心理落差，变得相对稳定。

---

本节课中我们一起学习了身份认同波动的现象与根源。我们了解到，依赖外部评价会导致自我价值感随境遇大起大落，而自卑与自大实为一体两面。要建立稳定的自我认同，关键在于建立内在价值体系、学会对事不对人，并勇敢地直面和接纳自身的局限性。通过这些练习，我们可以逐渐让内心变得更平和、更有韧性。
