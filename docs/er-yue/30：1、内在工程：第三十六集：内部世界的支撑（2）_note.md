# 内在工程 第三十六集：内部世界的支撑（2）🏗️

在本节课中，我们将继续探讨现实感的构建。我们将深入分析依赖外部反馈的弊端，并学习如何建立稳固的内部支柱，以实现真正的自我价值与情绪稳定。

---

上一节我们介绍了现实感的概念，本节中我们来看看，如果一个人的现实感完全建立在他人的回应之上，会产生怎样的后果。

如果一个人的现实感建立于他人如何回应他这根“蜘蛛丝”上，他将永远将自己与他人对比。其良好的感觉完全由他人的回应决定。如果有人喜欢他，他就感觉良好。如果有人称他为失败者，他会立刻在内心反驳“我不是失败者”。他的接纳感会被削弱，被推回自己的头脑中，情绪状态会受到打击。

他人的现实将成为他的现实。他会呈现出失败者的特征。

以下是失败者通常具备的特点：
*   呆滞并害羞。
*   无法自然地与他人交往。

他知道有人对他好或喜欢他时，可能会开始有一点自信。但他的自信必须由外界持续的示好来维持。如果外界对他不好，他将再次以自卑告终。这就是他最终的结果。他浪费的是自己的生活。他低价值的状态以及负面的态度，会引出他人最坏的一面。

因为当外界看到你自卑、没有自信、扭扭捏捏，便不愿意给你好的反馈。而不好的反馈则会进一步让他的状态下降，最终形成一个恶性循环。但他从未考虑过，他对其他人的消极看法，其实是他自我感觉的一种映射方式。

就像自信的人，通过积极的行为让他人产生良好反应，自然最终也会得到自我实现。自我实现的概念在第一节课已详细阐明。

依赖外部“蜘蛛丝”的麻烦在于其不稳定的性质。因为外界是无常的。依靠外界反馈来决定自我价值的人，就像暴露在风中的芦苇。你的现实支柱绝不能完全来自于外界。很多人把自己的现实建立在外界的成功上：有钱、获得好的反馈、或拥有一个漂亮的女朋友。他们需要用这些来定义自己是谁。

如果有一天你忽然失败了、破产了、或外界的反馈变差了，这可能会让他的整个精神世界直接崩塌。为什么？因为支撑他世界的那几根柱子都断了。

就像有的人分手后变得过分压抑。实际上他在这段关系之前很好，很上进，本应有很好的未来。但后来他的现实完全建立在那段恋爱关系上。他能想到的是，生活如果没有这个姑娘，将会充满空虚。其实他就是缺乏一个内部现实感。

另一方面，一个真正知道自己是谁的人，其内部现实支柱非常丰富，能让他在逆境中感到安全。他相信自己永远能通过一种方式，得到自己想要的生活。他也知道什么是真正的危险。无论他人是否喜欢他，他知道即使他人看不到他最好的品质，那些品质也依然存在。

他的授权标准从来不来自于社会。确定他权力标准的是他自己。他认为，作为一个人类，他的生命具有稳定不变的内在价值，不论他人承认与否。你作为一个人的价值就在那里。

他自己思考也能看到自己的不足，但他从来不会以此来否定自己。他尊重有价值的人并向他们学习。真正自信的人能够直视自己的缺点，而自大的人则对自己的缺点采取隐藏或视而不见的态度。

自信的人很早就明白了人的局限性，所以他们会不断进入强化自己的过程，但他们对于最终的结果是无所谓的。我们一直在强调注重过程，不在意结果。

除此之外，自信的人可能也会有一些相对稳定的外部支柱。例如，许多人已经喜欢他、他的技能、精致的外在、幽默风趣，这些都可以成为他的外部支柱。但他的自我感觉和价值感从来不依赖于那些外部支柱。也就是说，那些外部支柱是锦上添花的东西，有就更好。

就像一个建筑，可以有多根支柱。只要外部支柱不是稳定整个建筑的关键支柱，他就可以去享受它们。也就是说，只有当你有了内部支柱作为立足点，你才能真正体会到外部支柱带来的好处，并且没有失去的恐惧。

同时，如果一个人的现实意义完全由内部确定，也可能产生小小的问题。因为他总是感到安全，也可能感到没有动力和激情去实现什么，因为他觉得无论怎样都很好。

所以我们需要一个平衡的状态。关于你是谁、你的自我价值感、你的自我感觉，这些都必须由你的内部现实支柱来决定。而你的外部支柱，则为你的人生带来各种不同的刺激。这些刺激会不断内化，从而进一步加强你的内部现实支柱。而你的内部现实支柱的加强，会让你再去感受和创造其他不同的刺激，如此循环往复。

大多数人都有基于内部和外部现实的支柱。问题在于你是否已经找到了正确的平衡。

外部支柱其实没有太多需要解释的。除了你生命本身以外，所有其他的东西都属于外部支柱。

以下是外部支柱的一些例子：
*   你的技能。
*   你的财富。
*   你的有趣。
*   你的知识储量。
*   你的审美与品味。

内部支柱看似由几根支柱组成：你的自信、你的自我价值感、你的情绪稳定程度。其实这些最后都会融合成一根核心支柱，那就是 **“你是谁”**。

**“你是谁”** 这根支柱包含：
*   你对自己的了解。
*   你决定自己是谁。
*   你的自信。
*   你的自我价值感。
*   你的情绪稳定程度。

这所有的一切都是为了让你明白 **“你是谁”**。也就是说，你越明白 **“你是谁”**，你也就越自信。它们是一个相互依存的关系。

所以，你的内部支柱该如何建立呢？先建立你的自信，明白你的价值感，搞清楚什么是情绪。这些都是我们课程涵盖的内容。也就是说，我们这整个课程，是在帮助你发现 **“你是谁”**。最终，你要自己定义 **“你是谁”**。我不能决定你是谁，这个课程也不能。我只是在用这个课程为你们指引方向。

记住：**只有你自己才能决定你是谁。**

---

本节课中，我们一起学习了过度依赖外部反馈的危害，以及构建内部现实支柱的重要性。我们明确了外部支柱与内部支柱的区别与联系，并指出真正的平衡在于以稳固的 **“你是谁”** 这一内部支柱为核心，让外部支柱成为丰富人生的积极刺激。最终，自我定义的权利和力量，始终在你自己的手中。
