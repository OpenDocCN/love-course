# 课程30：内在工程 - 第二集：自我证实 🔮

在本节课中，我们将要学习“自我实现预言”这一核心概念。我们将探讨你如何通过自身的信念和假设来塑造你所感知的现实，以及这个过程如何形成一个循环，不断强化你最初的预期。理解这一点，是掌握内在力量、改善人际关系的关键。

## 概述

自我实现预言是指：你对世界的假设决定了你观察世界的视角。你认为世界是怎样的，你的大脑就会自动屏蔽与你假设不符的信息。

你的所有注意力都会集中在你选择相信的事情上，并且你会不断放大这些事。通常，我们都戴着不同的“滤镜”来过滤世界的大部分信息。我们还运用自己不同的经验和印记来解释这些信息。

我们以此构建起一个内心世界，并以为那就是世界的真实状态。一切“真实”的事物其实都是心智模式，而一切心智模式对你而言就是当下的世界。我们的心智模式决定了我们能看见什么样的世界。

更有趣的是，这个自我构建的“真实”世界，又会反过来印证最初的那个模式给你看。

## 信念如何塑造现实

你觉得世界是怎样的，它就会以细节组合的形式呈现出来。

以下是信念影响现实的几个方面：

*   **寻找支撑证据**：如果你觉得这个世界是友善的，你就会不断寻找证据来支撑这个“现实”。如果你觉得自己充满魅力、很受女性欢迎，只要这个信念足够强，你也会找到足够的证据来支撑自己。
*   **框架驱动行为**：同时，这会以你的内在框架来驱动，让他人向你的“现实”靠近。

很多男人虽然知道吸引女性靠的是吸引力，但在接近女性时，满脑子想的仍然是如何证明自己的价值、如何让女性开心等等。你的“现实感”会根据你的设定，流入到对方的感知中。

这就等于把自己的力量交给对方来击败自己，反而会让对方处于更高的心理优势地位。因为她是筛选者，而你仍在寻求她对你的认同。

## 行为确证：期望如何成为现实

我们对某人有什么样的期望，就可能会用有利于实现这种期望的方式来对待她，从而也证实了我们最初的预期。

以下是行为确证的发生过程：

*   **负面预期的循环**：如果你觉得一个女生对你不感兴趣，你的心态就会开始变化。你会自动屏蔽她所有的兴趣指标，同时对她做出一些防范性或缺乏兴趣的肢体语言及口头语言。而女生则会根据你不友好、不自信的行为，最终真的开始疏远你。于是，你就看到了你想要的结果。这其实是一个恶性循环。
*   **依赖固有观念**：人们都会依赖自己的知识和观念，并一直被这种东西所束缚，还把这些称之为“现实”。因而才有了那句话：你看到的现实，只不过是你所选择相信的。

## 内在投射：你的世界由你创造

这个世界，其实就是你内在的投射。

以下是两种不同的内在投射及其结果：

*   **正面投射**：如果你觉得这个世界充满爱心和善意，你就会不断寻找证据来支撑这个想法。你的注意力会更多放在正面的社会新闻上，比如慈善、美好、互相帮助。当证据不断累积，你的“现实感”也会越来越强。
*   **负面投射**：如果你觉得这个世界充满阴暗和负面，你也会不断寻找证据来支撑这个想法。你的注意力会更多放在负面新闻上，比如斗争、尔虞我诈。当证据不断累积，你负面的信念也会越来越强。

因此，一旦你形成错误的社会信念，就可能引发他人做出某些行为反应，以支撑这种信念。这种现象就叫做 **行为确证**。

## 自我实现预言的循环

你的自我实现预言决定了你对外界的态度。而你对外界的态度，会直接引发外界对你的态度。外界对你的态度又会进一步强化你的自我实现预言。

我们可以用以下公式来描述这个循环：

```
正面/负面自我预言 -> 相应的态度 -> 引发外界相应反馈 -> 强化最初的预言
```

*   如果你的自我实现预言是正面的，你就会进入一个**正面循环**。
*   如果你的自我实现预言是负面的，你就会掉入一个**恶性循环**。

我们其实是在不断地为自己构建一个世界，然后又反复地强化它，最终让我们相信这个世界就是我们构建的那样。从这个角度说，我们就是自己生命的巫师。我们给自己搭建了一个幻想的世界，然后在现实中，让这个幻想慢慢地、不断地、一点一点地去实现。

## 期望的形成与验证过程

我们在做出假设后，会用我们的态度评估来预测这个人未来的表现。如果我们知道他们的态度，就可以预想出他们的行为。如果他们太过偏离我们的预期，我们可能会修改对他的标签，但我们总能用“暂时失常”的理由将几个特例合理化。

以下是期望验证的具体步骤：

1.  **形成期望**：预测目标会怎样行动。目标的年龄、性别、长相、社会阶层等信息，都会在你意识不到的情形下影响你的判断。
2.  **采取行动**：你通常会以与自己期望保持一致的方式来行动。你觉得外界友善，你也会友善；你觉得这个人冷漠，你也会冷漠。
3.  **对方解读**：对方会注意到你的行为，他对你行为的解释会影响他自己的反应。
4.  **对方反应**：多数情况下，对方做出反应的方式与你对他的行为方式是类似的。对热情抱以兴趣，对敌意则怀疑或反击。
5.  **解释结果**：在最后一步，你解释目标的反应时，不太可能认识到你自己在引起目标反应中所起的作用。你会把目标的行为归因于他自己的性格或心情。毕竟，当你在目标身上发现了自己期待的行为，还有更好的证据来证明自己的预期是正确的吗？

这也是人们在判断他人时往往过度自信的另一个原因。当错误的期望变成现实，人们从来不会意识到自己犯过的错误。

## 应用：改善人际关系

我们常从别人身上验证我们的期望，而这些期望中的行为，如果没有我们的推动，其实就不会发生。但我们很少意识到，正是我们的期望造成了我们想看到的现实。

对他人的期望能引导我们对他人的行为，因此，**期望具有触动作用**。

一个有趣的实验是：研究者先误导被测试者的期望，比如告知陌生人要么喜欢他们，要么讨厌他们。然后让被测试者与陌生人交谈。预期自己被人喜欢的被试者，在和陌生人打招呼时更有魅力；而预期自己被人讨厌的被试者，则小心翼翼、戒心重重。结果，对方也确实更讨厌后者。错误的期望使他们的行为变成了现实。

积极的期望对人际交往有益，而消极的预期则相反。经常忧虑被他人拒绝的人，他们的行为方式更可能遭人拒绝。对拒绝高度敏感的人，经常会紧张地感受到别人的怠慢，即使别人并非有意冷落。他们的反应过度、心轻胆战，比一般人表现出更多敌意和戒心，从而让别人更加讨厌他们。

## 实践技巧：运用积极的自我实现预言

那么，你知道影响他人（例如调教伴侣）最好的方法之一是什么吗？除了奖励机制，最好的方法就是不断地夸赞她尚未完全具备的优点，即运用积极的**自我实现预言**。

例如，你希望你的女朋友更温柔。你可以不断利用细小的瞬间来夸她：“我觉得你刚刚那句话很温柔，很打动我。”这时，她的潜意识会接收到这个信息，她会下意识地向你这个标准靠拢，因为每个人都希望自己被他人喜欢。

反之，如果你总是说自己的女朋友蛮横，接下来你就会看到你所“相信”的事实：
1.  你会自动忽略她所有通情达理的地方。
2.  你会不断地强化她的“蛮横”。

## 总结

本节课中，我们一起学习了**自我实现预言**的核心机制。我们了解到，你的信念和期望会像滤镜一样塑造你的感知，并通过你的态度和行为影响他人，最终让外界反馈来印证你最初的设想，形成一个或良性或恶性的循环。

记住关键公式：**你的信念 -> 你的态度/行为 -> 外界的反馈 -> 强化你的信念**。认识到这一点，能让我们对自己的观念保持警觉，并以更开放、积极的态度对待他人，从而主动创造更理想的人际关系和现实体验。你，就是自己现实的创造者。
