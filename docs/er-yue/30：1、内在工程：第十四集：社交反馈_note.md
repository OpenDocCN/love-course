# 课程30：内在工程 - 第十四集：社交反馈 🪞

在本节课中，我们将要学习社交反馈如何深刻影响我们的自我价值感、情绪状态和行为模式。我们将探讨社交反馈的运作机制、它如何塑造我们的身份，以及如何通过理解并管理社交反馈来建立更高价值的内在状态。

## 概述

社交反馈是我们在身份这个大章节中探讨的核心概念之一。它如同一面镜子，通过外界的反应来定义我们的价值、情绪和行为。理解社交反馈的机制，是掌握内在工程、构建稳定高价值身份的关键一步。

## 社交反馈决定自我价值与情绪

我们的情绪状态是判定自身价值的重要标尺。情绪良好时，我们会认为自己充满价值。情绪糟糕时，我们则可能认为自己毫无价值。

在社交互动中，情绪的好坏主要源于外界反馈。被外界接受时，情绪自然导向积极。被外界拒绝时，情绪则导向消极。

## 我们对社交信息高度敏感

我们对社交信息极为敏感，因为我们的生存几率受到社交圈的影响。如果高价值的人希望与我们交往，我们会感受到被肯定。如果高价值的人拒绝我们，我们则会感觉到被否定。

正面的社交反馈使我们感觉良好，负面的社交反馈使我们感觉糟糕。例如，在社交场合收到所有人的欢迎和正面反馈，会提升自信和状态。反之，如果遭遇冷漠，则会降低自信和状态。

## 价值差异决定反应强度

价值差异决定了我们对社交反馈的反应程度。对自身价值影响越大的潜在因素，会引发更强烈的情绪反应。

例如，一位很有魅力的姑娘追求你，比一位相貌平平的姑娘追求你，更能证明你的价值。同样，被一位有魅力的人拒绝，会比被一位乞丐拒绝，带来更糟糕的感受。

**公式：情绪反应强度 ∝ 感知到的价值差异**

这是因为我们内在的程序会自动计算事件对生存价值的潜在影响，并激发出相应的情绪。这一点在之前关于“价值评价”的课程中已有阐述。

## 社会反馈是一面主观的镜子

无论你是否意识到，你对现实的解释都是主观的，并受到社会环境反馈的大规模影响。

我们可以将社会反馈概念化为一面镜子。通过这面镜子，我们相互参照和验证自己的行为是否正确、是否正常。

例如，一个三岁小孩在地上捡东西吃。如果你露出厌恶的表情，小孩通过你的反馈就知道自己做错了。因此，通过社交反馈，我们形成了关于“什么行为是正常”的观念。

## 社交反馈定义行为与地位

我们倾向于只做符合大众价值观的事情。同时，我们通过社交反馈认识到自己在群体中的地位。认识到自身地位后，我们会采取与之对应的行为方式。

在与世界的互动中，我们还会获得关于“他人是谁”的概念，即其他人在团体中扮演什么角色、拥有何种社会价值。

无论是否意识到，我们的大脑总能自动分辨出地位高的人和地位低的人，并做出相应的情绪反应。对于高价值的首领，我们会本能地产生靠近的情绪。

**核心概念：只要是社会性动物（如人类、狮群、狼群、蚂蚁），就必然存在阶级概念。而独居动物（如老虎、熊）则没有。**

## 社交反馈塑造身份认同

对大部分人而言，社交反馈决定了“他们是谁”。他们不是自我定义，而是依靠外界反馈来定义自己：外界反馈好，就觉得自己很牛；外界反馈不好，就觉得自己很差。

当社会反馈暗示你的地位提高或降低时，你的思维会感到社交压力，并做出反应，采取在当时情况下最符合你感知到的身份的行为。

作为社会性动物，我们都是自适应的。我们处于何种地位，取决于感到压力后大脑所选择的“合适”身份。然而，大部分人在面对社交压力时，会选择退回之前的低价值身份，向现实妥协。

## 构建高价值身份需要坚持

你高价值身份的构建，并非一蹴而就。它需要在一天又一天的坚持中得来，需要你不断对抗外界的社交压力。

过程中你可能会自我怀疑，但如果能坚持到底，你将获得新的高价值身份。但这并非终点，你需要不断将这一身份内化，这可能需花费数年甚至更长时间。

内化所需的时间，取决于你遭遇社交压力的频率和强度。例如，如果你频繁参与各种聚会并坚持自我，可能更快地稳固新的高价值身份。

## 屏蔽以他人为中心的念头

在社交或其他领域，我们常会下意识地注重他人的反对意见。若反馈与预期不符，就容易产生自我怀疑。这本质上仍是一种以他人为中心的交流方式。

我们必须学会屏蔽这种念头，自由自在地表达真实的自己。外界的反馈和看法，仅是基于他们当时的心态。不必活在外人的反馈中，也不必刻意迎合这个世界。

## 资格准则：允许高价值行为的钥匙

每个人对社交压力的反应不同，这取决于他们内心的“资格准则”。资格准则，是你认为自己需要满足何种条件，才能采取高价值的身份和有吸引力的行为方式。

你的资格标准由以下一个或几个因素组成：

以下是构成资格准则的五个要素：

1.  **肤浅的社会标准**：例如高颜值、会赚钱、成功、拥有奢侈品等。
2.  **联盟**：与地位高的人为友，或拥有令人羡慕的伴侣/社交圈。
3.  **能力**：拥有他人希望获得的价值，如专业知识、有趣的故事、实用技巧等。**核心：能够提供价值。**
4.  **角色扮演**：在特定情境下扮演高价值角色，如老师、专家、权威人士。
5.  **身份**：源于内在信念的权利感，即坚信自己本就有价值。

当你的资格准则得到满足（满足以上至少一点），你的思维就会“允许”你做出所有高价值行为。这时，你就会“进入状态”。

## 进入状态：高价值身份的体验

当你的精神与情绪状态联通，这种现象通常被称为“进入状态”。进入状态是一种强大的体验。

当你在状态时，世界仿佛是你的舞台。你思维冷静，言行有效，幽默感直达人心，故事引人入胜。人们会自然跟随你的引领。最在状态的人往往拥有磁铁般的吸引力。

很多人都有过这种经历：在某个时刻感觉特别好，自信爆棚，无所不能。在状态的人不靠逻辑思考话题，因为他们超然物外，不在乎外界看法，对自己的言行充满信心。

**代码描述状态下的交流：**
```python
if 在状态 == True:
    话题 = 所见即所聊  # 看到什么就聊什么
    行为 = 自然流淌    # 一切靠心态支撑
    无需依赖 = 预设惯例或话术
```

这种能力让我们能根据现场情况即时创造互动，而非依赖预先准备的套路。这并非玄妙的境界，正如佛陀所言，我们本自具足，只是多数人认为自己没有。

## 在状态时的感觉

当你在状态时，通常会有以下几种感觉：

以下是进入状态后的四种典型感受：

1.  一种完整、圆满的感觉。
2.  积极性、稳定感和优势感急剧上升。
3.  对每件事都豁达开朗，具备自然的品质。
4.  成为环境中良好情绪的源泉。

状态本质上是身份的体现，你的身份是社会价值的反映。因此，最在状态的人，必然是拥有最高社会价值的人。

## 在状态与不在状态的对比

当你在状态时，你会毫不犹豫地表达个性，不在乎外界眼光，不试图说服或迎合他人。你的价值自然流通，他人会更珍惜与你的真实沟通。你连接了最真实的自我，只是在分享能量。

**在状态意味着高社会价值。** 当你感受到自我价值，你是放松、开放的。反之，如果你感受不到自我价值，你会紧张、内敛，总在思考如何装得自信、如何讨人喜欢，这决定了你“不在状态”。

当你不在状态时，你无法享受环境，感到紧张不满，总想着如何证明价值、提升地位。你会显得不自然，有被迫感，需求感增强，并试图改变自己去迎合他人，对自己的社交行为感到不安。

## 如何找到并保持状态

关键在于能够自由地表达个性。这本身就是你受人欢迎的方法。当然，不可能所有人都喜欢你，总会有人因为嫉妒等原因讨厌你，这很正常。

但如果你能自由表达，不在乎外界反应，多数情况下你会更具吸引力。警惕你的大脑变得“逻辑化”和“刻意化”。一旦你开始思考“我要说点有趣或有价值的话”，你就可能失去状态。

**做法：** 当意识到自己在刻意为之，就让大脑回到当下，想说什么就说什么。不要担心能否吸引对方。当你真正放下包袱，停止试图给人留下印象的行为，活在当下自由表达时，你对异性的吸引力将无比强大。

**核心总结：** 只要有一丁点“我能否吸引TA”的疑虑，吸引力就会消失。真正的吸引力，诞生于毫无疑虑的自我表达之中。

本节课我们一起深入探讨了社交反馈的运作机制及其对我们身份构建的核心影响。我们了解到，社交反馈如同镜子，定义着我们的价值感和行为模式。通过理解“资格准则”，我们可以解锁高价值行为，进入“状态”。最终，吸引力的源泉不在于技巧，而在于能否摒弃对外界认可的依赖，活在当下，自由且毫无保留地表达真实的自我。记住，当你停止证明自己时，恰恰是你最有魅力的时候。
