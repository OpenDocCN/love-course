# 浪迹升华《聊天技巧速成班：9.聊天延续技巧-敲打_note

在本节课中，我们将要学习聊天延续技巧的第三个核心方法：敲打。敲打是一种通过表达轻微不满，来提醒对方注意其行为并调整互动方式的技巧。它涉及到较高的沟通分寸感，对于建立平等、尊重的对话关系至关重要。

上一节我们介绍了其他延续对话的方法，本节中我们来看看如何运用“敲打”来引导对话走向。

## 什么是敲打？🤔

敲打是指用委婉程度较轻的表达，来传递你对对方某一方面的不满。这种情况通常发生在对方与你聊天态度不积极、不配合时。

当对方表现出与你期望相反、让你感到不适或对双方关系发展不利的行为时，通过敲打可以及时提醒对方，阻止情况恶化。

## 为什么要使用敲打？🎯

使用敲打主要为了实现两层效果。

第一层效果是提醒对方意识到自身问题，并在之后的互动中注意和改善。例如，如果对方总是敷衍回复消息或回应不认真，及时的提醒可以避免无效沟通，提升交流效率。这可以概括为：**有则改之，无则加勉**。

第二层效果是让对方感受到你的态度和原则，从而在内心深处对你更加重视和尊重。这表明你是一个有底线的人，不会无条件地包容一切。在人际交往中，建立平等和相互尊重的关系非常重要。

## 敲打的两种类型 🔧

敲打主要分为直接敲打和间接敲打两种。

直接敲打是指直接指出对方的不足，表达方式不委婉。间接敲打则是通过间接或委婉的表达，来提醒对方注意相应方面。

以下是三种常见场景中，直接敲打与间接敲打的示范对比。

**场景一：对方聊天中途突然消失，很久未回复。**

*   **直接敲打**：`你人怎么不见了？话才说一半。`
*   **间接敲打**：`你这是手机突然不在服务区了吗？`

**场景二：某次聊天后，对方很久没有再联系你。**

*   **直接敲打**：`你还在吗？怎么突然没声了？`
*   **间接敲打**：`你是被外星人绑架到外太空了吗？突然就杳无音信了。`

**场景三：对方性格被动，沉默寡言。**

*   **直接敲打**：`你有点被动/内向吗？`
*   **间接敲打**：`你是不是不太擅长跟陌生人打交道，有点不好意思呀？`

通过以上例子可以看出，间接敲打有时类似于“冷读”，但带有一丝表达意见的意味。

## 如何使用敲打？💡

了解了敲打的类型后，我们来看看具体的使用方法和注意事项。

以下是使用敲打时需要遵循的几个要点。

1.  **直接敲打的使用**：直接反映客观事实即可，无需特殊技巧。关键在于措辞，应避免使用过分指责、不满或埋怨的语气。只描述事实，不衍生负面情绪。例如，说“你人呢？”，而不是“你怎么这么没素质！”。
2.  **间接敲打的使用**：需要为对方的行为找一个不伤体面的理由，或者描述一个夸张、荒诞的结果，以此来弱化消息中的负面意味。可以使用询问语气或明显不切实际的借口，进一步缓和气氛。核心是在表达不满的同时，不让对方难堪。
3.  **选择直接或间接**：根据对方的性格特点及当时的聊天气氛来决定。如果对方性格直爽、聊天气氛热烈，可以更直接；如果对方性格内向、气氛一般，则应更委婉。
4.  **控制敲打轻重**：根据对方性格、当前聊天气氛以及两人关系的亲密度，决定敲打的轻重程度。关系越紧密、气氛越好，敲打可以稍重；反之，则应轻微、含蓄。
5.  **结合思维发散**：在使用间接敲打时，为其寻找荒诞的理由或结果，往往需要运用思维发散的能力，这能让敲打显得更巧妙、不突兀。

## 敲打实例分析 📝

下面通过几个具体例子，来感受直接敲打与间接敲打在应用中的区别。

*   **直接敲打**：`你这样回得有点慢哦，我等了好久了。`（陈述事实+自身感受）
*   **间接敲打**：`你是不是被绑架了所以不能发消息？要是你遇到危险的话，冲我眨眨眼。`（荒诞理由）
*   **间接敲打**：`咱们每天聊这样一两句，还以为是在写信呢。`（夸张对比）
*   **混合型敲打**：`你这么被动害羞，可是要失去我的。`（前半句直接指出，后半句用荒诞结果缓和）
*   **间接敲打**：`你有点少言寡语，是有什么心事吗？`（为行为寻找体面理由）

这些例子展示了如何在实践中平衡“表达不满”与“维持和谐”的微妙关系。

## 补充说明与定位 📌

敲打在聊天技巧中属于“情绪下压”的范畴，能使聊天气氛略带严肃和压迫感，从而制造情绪波动。

> 情绪波动 = 情绪拉伸（上升） + 情绪下压（下降）

我们之前课程主要涉及情绪拉伸技巧。敲打是情绪下压的一种较轻形式。更重的下压技巧（如打压、否定）需要更高的聊天能力和分寸感，使用不当可能导致对话崩盘。鉴于本课程面向初学者，掌握敲打这一基本方法已经足够应对大多数情况。
