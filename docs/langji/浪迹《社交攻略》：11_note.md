# 浪迹《社交攻略》：11 💬 聊天后期：从朋友到暧昧

在本节课中，我们将要学习聊天后期的核心目标与操作方法。我们已经介绍了聊天的初期和中期，本节中我们来看看如何在与女生见面之后，将关系从朋友阶段推进到暧昧阶段。

## 聊天后期的定义与目标

聊天后期并非指你与女生的线上关系有多好。真正的聊天后期始于你们至少有过一次现实见面。见面之后，你们已经从网络上的陌生人变成了现实中的朋友。

此时，你之前聊天的三个主要目的——展示自己、了解对方、邀约见面——都已不复存在。那么，在聊天后期我们聊什么？我们的新目标是什么？

**核心目标**：创造暧昧氛围，将你与女生的关系从朋友升级为情人。

## 为何要在线上创造暧昧？

你可能会问，任何话在现实中说不是更好吗？为什么要在网上做这些？这并非多此一举。

**首要目的**是试探情况。这是一个“奖励激进者的游戏”。你越主动、越愿意承担关系升级的后果，就越能在关系中占据主动，并获得女生更多的正向反馈。

**另一个重要原因**是避免当面升级关系时的尴尬。这主要是为了避免女生的尴尬，当然，如果男生脸皮够厚，这或许不是问题。

## 如何正确加入暧昧话语？

很多兄弟一听说后期要聊暧昧，就在对话中塞满暧昧和拉近关系的话语，这是错误的。

**核心原则**：把暧昧话语当成**调味品**，而非主菜。就像做菜，重要的是菜本身，而不是盐。

以下是具体操作方法：

*   **正常聊天是基础**：进行日常分享、阶段性聊天，朋友圈互动等。
*   **稀有地加入暧昧**：在聊天的某个节点，突然插入一句暧昧话语。
    *   例如：“告诉你，昨天梦到你了。”
    *   例如：聊到某家餐厅时，可以说：“这家看着不错，啥时候我们去尝尝？”
    *   例如：聊到酒时，可以说：“这酒看着很上头，想跟你喝了酒玩亲亲。”
*   **关键比喻**：就像炒菜放盐，要撒得均匀。如果直接吃一坨盐，会齁得难受，菜就不好吃了。暧昧话语也应如此，偶尔点缀，切忌过量，否则会让人反感。

## 一个重要提醒：放弃筛选测试

在当下的聊天环境，特别是与00后相处时，聊天后期**不要**加入所谓的“筛选”或“测试”。这会使你显得机械和猥琐。

如果你担心无法筛选会导致在错的人身上浪费时间，以下是四点解决方案：

1.  **当做成本**：将时间投入视为必要的探索成本。
2.  **增加后手**：同时与多位女生接触，分散风险。
3.  **提高社交直觉**：通过实践提升识人辨人的能力。
4.  **安排社交局**：将你无法确定的女生先约到多人社交场合，而非单独见面。
