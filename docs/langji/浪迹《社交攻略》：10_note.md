# 浪迹《社交攻略》：10_note

在本节课中，我们将要学习聊天进入中期阶段后，你应该采取的核心策略和心态。我们将重点探讨如何在对话中加入情绪，并围绕几个关键原则来构建自然、有吸引力的互动。

---

在上一节我们介绍了聊天初期的目标与禁忌，本节中我们来看看聊天中期应如何推进。

聊天中期，你的核心任务是：**加入情绪**。所有对话都应服务于三个核心目的，同时融入情绪化的表达。

但是在服务核心目的的同时，必须加入带有情绪的对话。

接下来，我们将探讨聊天中期常见的一个问题。这个问题在我最近直播点评照片时也经常发现。

许多人的照片显得僵硬、不自然，例如戴着墨镜、仰头等姿势。这种僵硬感其实会延伸到你的聊天中。

你可以想象一下那些拍照呆板的样子，那很可能就是你与女生聊天时的状态。

在这里，我列举了几个聊天中常见的问题：刻意、僵硬、做作、不自然和过犹不及。很多兄弟与女生聊天时都存在这些问题。

那么，如何改变呢？我们来看核心的五个原则。

以下是聊天中期需要贯彻的五个核心态度：

1.  **友善**
2.  **保持积极的姿态**
3.  **不要过度思考**
4.  **顺其自然**
5.  **无所谓**

我接下来将逐一讲解如何将这些原则运用到与女生的对话中。

**第一个原则：友善。**

可以用这样的例子来说明：“周末空吗？出来溜溜。” 这种聊天方式很友善。例如回复“行呗，那到时候再说”，而不是去指责或辱骂对方。

很多兄弟受到一些错误资料影响，认为需要展现攻击性、辱骂女生或直接对抗。这是错误的。

**一个男人越友善，代表你的修养和礼貌，并非代表你是舔狗。**

那么，什么才是舔狗呢？当女人对你不友善、辱骂你时，你仍保持友善，这才叫舔狗。但在刚开始认识、面对对方轻微拒绝时，你应该友善、礼貌地回应和接受，这是一个人的涵养。

因此，开始时一定要友善。何时才需要展现态度？当女人开始甩态度、对你不友善的时候。例如，聊天初期女人不回消息（NR），很多兄弟会去骂对方，这只会让女人觉得你无能、挫败。真正的价值在于：你无视我，无所谓，我自有其他选择。拥有这种心态，你就不会去骂对方，因为没必要帮助一个犯傻的人。

当你以友善礼貌的态度对待女生时，很多女生会意识到你是个有价值的男人，并可能为自己之前的行为道歉，这时你就占据了主动。

**第二个原则：保持积极的姿态。**

很多兄弟容易消极应对。女人发来消息，他爱回不回，或者以“在忙”为借口搁置，等想起来再聊时，机会已逝，又转而开始舔狗式追逐。这是错误的。

你应始终保持积极的姿态。即使你正面临事业打击或生活苦楚，对女人仍需保持积极，因为这是两码事。

很多兄弟喜欢混为一谈：事业不行，就对女人、生活、朋友都不行。事业受挫后整个人就垮了，因为他消极对待周围一切。

但你看我们，当年事业被重创，反而让我在感情上更加投入和顺利。因为不用去公司，有更多时间在外快乐玩耍，甚至因此结婚生子。如果当年事业蓬勃发展，哪有心思找老婆生孩子？如果等到40岁一无所有甚至锒铛入狱时再被打击，那就真的什么都没有了。

所以，不要让某些行为影响你的全部。女人也是如此，不能因为这个女人拒绝或冷淡你，就对其他女生抱有敌意，感觉全世界都欠你。很多兄弟曾被伤害，现在转而伤害别人，这是不对的。

始终保持积极的姿态。你被伤害了，那是你自己的问题，我对待其他女生依然很好，这样你才会收获真正的幸福。

**第三个原则：不要过度思考。**

很多兄弟喜欢过度解读女生的话，纠结“她这句话到底是什么意思？”。

如果你确实不知道女生某句话的意思，可以直接去问她，因为我可能也不知道，或者我根本不在乎。女生脑子里想什么，她们自己有时也不清楚，你为何要去分析呢？

想要成为撩妹高手，你需要不断与女生相处，不断犯错，并不断提升你的纠错能力。因此，你主要学习的是：判断错误后，如何纠正。这种纠错能力很重要，因为你永远无法百分百判断正确，你甚至有时都不明白自己在想什么。

所以，不要害怕判断，也不要怕犯错。你要学会在犯错后立即纠正。但很多兄弟的问题是：不会纠正，错了就结束了。因此，你需要在犯错的同时，马上改正，学会这种改错能力。

**第四个原则：顺其自然。**

什么是顺其自然？我们在这里讲得很清楚。你看一下为什么你无法顺其自然？为什么你想学习解决女生的某个问题？看完这一部分你就明白了。

其实没有那么多复杂的东西。你想要的可能是其他东西，比如增加经验。

**第五个原则：无所谓。**

这是聊天的最后一个关键态度。不要过于在意某些人，这个人可能跟你毫无关系。情商的培养需要锻炼，需要掌控你自己的想法。

本节课中我们一起学习了聊天中期的核心策略：**在对话中融入情绪**，并围绕**友善、保持积极姿态、不过度思考、顺其自然、无所谓**这五大原则展开互动。掌握这些心态和技巧，能让你的聊天摆脱僵硬，变得更加自然流畅，从而有效吸引女生。
