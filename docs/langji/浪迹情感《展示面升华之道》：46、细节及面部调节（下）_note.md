## 一、脸型调整回顾与过渡

上一节我们介绍了脸型调整的基础概念。本节中，我们来看看如何具体操作脸型的几个关键选项。

整个脸部的调整都基于对自己脸型的了解程度。面部重塑中，脸型有以下功能选项：

以下是脸型调整的五个关键选项：
1.  **脸宽**：调整脸部整体的宽窄。
2.  **太阳穴**：调整太阳穴的饱满或凹陷程度。年龄增长通常会导致太阳穴凹陷。
3.  **颧骨**：调整颧骨的突出程度。颧骨越突出，面部线条可能越不流畅。
4.  **下巴**：调整下巴的长短。
5.  **下颚**：调整下颚线的紧致度。下颚线越紧致，脸显得越小。

修图时，通常目标是让脸型在符合整体轮廓的同时显小，以衬托身材比例。

以下是作者分享的个人调整经验：
*   首先调整**脸宽**，将脸部向中间收窄。
*   其次调整**下巴**，将其拉长。在所用工具中，向右调节是缩小，作者将下巴长度拉至极致。
*   最后微调**下颚**，将其向内侧收紧少许。

经过这三个步骤，脸型会得到优化。

## 二、眼睛的调整 👁️

脸型调整完成后，我们进入五官细节的修饰。首先来看眼睛部分的调整。

眼睛的调整选项包括：大小、上下、眼高、长度、眼距和倾斜。

以下是每个选项的具体说明：
1.  **眼睛大小**：整体放大或缩小眼睛。**一般不推荐使用**，因为容易导致眼睛与脸部比例失调，显得不自然。
2.  **眼睛上下**：调整眼睛在脸部纵向上的位置。**很少需要调节**。
3.  **眼高**：拉长眼睛的上下高度。这是**推荐**的放大眼睛的方法，微增眼高即可让眼睛显大且不失真。
4.  **长度**：调整眼睛的左右长度。结合“眼高”一起微调，可以有效、自然地放大眼睛，类似于“开眼角”的效果。
5.  **眼距**：调整两眼之间的距离。如果五官紧凑，可适当调宽眼距，让布局更舒展。
6.  **倾斜**：改变眼角的倾斜方向。**一般不调节**，因为上挑会显凶，下垂会显丧。

以下是眼睛调整的核心经验：
*   从不调小眼睛。
*   放大眼睛主要使用 **`眼高 + 长度`** 进行微调。
*   除非眼型特殊，否则只调节大小（通过眼高和长度实现）和眼距，其他选项保持不动。
*   如果存在大小眼或眼型不满意的情况，可以使用后续会讲到的“瘦脸瘦身”工具进行局部推拉调整。

在作者的示例照片（侧脸）中，仅因拍摄时眯眼而微调增大了眼睛，其他选项均未改动。

## 三、鼻子的调整 👃

眼睛调整好后，我们来看另一个核心五官——鼻子。许多亚洲人的鼻子可能存在塌、大等问题，需要精细调整。

鼻子的调整选项包括：大小、提升、鼻翼、山根、鼻梁、鼻尖。

以下是每个选项的具体说明：
1.  **大小**：调整鼻子的整体大小。与眼睛不同，**鼻子通常可以调小**，因为小巧精致的鼻子更符合审美。
2.  **提升**：将鼻子的位置整体上移或下移。亚洲人鼻子可能因偏大而显下垂，可适当向上提升，使鼻子更挺。
3.  **鼻翼**：收窄或放宽鼻孔两侧的宽度。收窄鼻翼能使鼻子更显挺拔。
4.  **山根**：调整鼻梁起始点的凹陷或高度。垫高山根能增强面部的立体感。
5.  **鼻梁**：调整鼻梁的挺拔程度。亚洲人鼻梁常偏平，可适当调高。但需根据自身鼻梁基础调整，避免过度。
6.  **鼻尖**：调整鼻尖的高低。适当下压鼻尖，可以使鼻孔内收，外观更秀气。

以下是鼻子调整的核心经验：
*   鼻子一般会修小、修窄、修挺。
*   特别注意收窄鼻翼。
*   如果对鼻孔形状不满意，同样可以依靠后续的“瘦脸瘦身”工具进行微调。

在作者的示例中，主要将鼻子调小、鼻翼收窄，并做了轻微提升，这些调整在侧脸照中变化微妙但有效。

## 四、其他面部细节与注意事项

我们最后来了解眉毛和嘴唇的调整，以及面部重塑的重要注意事项。

**眉毛调整**
眉毛主要调节高低、粗细和眉距。
*   如果眉毛离眼睛太近，可以调细并适当抬高，使眼部区域更舒展。
*   眉距可跟随眼距调整，如果眼距调宽，眉距也可相应微调。

**嘴唇调整**
嘴唇可调整大小、厚薄、上下位置及微笑弧度。
*   嘴唇厚常伴随嘴大，可先整体调小嘴巴，再修薄嘴唇。
*   人中短可以通过将嘴巴位置略微下移来视觉上延长人中。
*   **微笑**选项非常实用，可以为嘴角添加自然的微笑弧度，提升亲和力。除非追求冷酷风格，否则让嘴角平和或微微上扬通常有益。

### 面部重塑核心注意事项

以下是两项必须牢记的注意事项：
1.  **效果与耐心**：面部重塑步骤繁多，单看对比图变化可能不大，但颜值的细微提升对整体形象影响显著。这是修图中耗时较长的步骤，但非常值得投入。
2.  **功能限制与备用方案**：如果人物距离太远、侧脸角度过大或脸部被遮挡，软件可能无法自动识别人脸，导致“面部重塑”功能失效。此时，需要使用 **`瘦脸瘦身`** 工具进行手动调整。

在最终对比中，经过调整的脸部会显得更清秀、比例更协调。关键在于**微调**而非“换脸”。初学者可能会经历过度调整的阶段，需多次练习才能掌握恰到好处的分寸感。
