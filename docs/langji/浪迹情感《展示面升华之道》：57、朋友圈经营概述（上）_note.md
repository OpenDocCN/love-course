# 浪迹情感《展示面升华之道》：57、朋友圈经营概述（上）_note

在本节课中，我们将要学习微信朋友圈经营的基础知识。朋友圈是你在社交网络中的“大本营”，是与他人建立并维持长期联系的核心平台。因此，如何建设好朋友圈的展示面至关重要。本节课将首先介绍朋友圈的内容构成与各类内容的分布比例。

## 朋友圈的内容构成

上一节我们强调了朋友圈的重要性，本节中我们来看看朋友圈具体由哪些内容构成。理解这些构成部分，是有效经营朋友圈的第一步。

### 1. 展示面

展示面是指自己制作或拍摄的视频及照片。这是朋友圈中最关键的元素。

以下是关于展示面的几个核心要点：

*   **1.1 视频部分**：本课程不涉及视频拍摄教学，但后续会说明如何在朋友圈发布视频。
*   **1.2 照片部分**：包括**本人出镜**和**非本人出镜**（如风景照）的照片。在个人形象已得到良好展示的基础上，可以发布自己拍摄的非本人出镜照片。
*   **1.3 配套元素**：一张展示面照片发布时，通常伴随以下元素：
    *   **配文**：照片上方手动输入的文字。
    *   **定位**：在值得分享的地点，可使用微信定位功能。
    *   **评论**：发布后自己或他人的评论。自己发布照片后，可自行评论，与配文形成呼应，这也是一种技巧。

### 2. 分享

分享是指转发非自己创作，但能引起共鸣、展示个人价值观或想法的内容。

以下是常见的分享类型：

*   **纯文字**：摘抄自书籍、公众号等平台的文字段落。
*   **文章**：转发认可的微信公众号文章链接。
*   **音乐**：分享能展示音乐品味的歌曲。
*   **图片**：分享他人创作的、符合自身审美的画作或摄影作品。
*   **视频**：分享来自抖音等平台的短视频。微信自带视频时长较长，从外部平台转发的视频时长通常限制在**30秒**以内。

### 3. 其他内容

这部分主要指与工作相关的内容。

以下是关于其他内容的说明：

*   包括工作宣传、公司要求转发的文案等。
*   建议将工作微信与个人微信分开。若无法分开，应严格控制这类内容的发布频率和可见范围，并尽量在发布几天后隐藏。

### 4. 各类内容简评

在了解了朋友圈的构成后，我们来看看这些内容各自的作用。

以下是各类内容的简要评价：

*   **4.1 展示面**：对展示个人特质效果最直观、作用最直接。对于旨在突出个人价值的人来说，朋友圈应以展示面为主。
*   **4.2 分享**：效果积累较慢，需要日积月累，但过程中存在“突变”可能，因此不可或缺。长期、稳定的分享能逐步建立个人形象，而保留一些早期的普通内容，能让微信号看起来更真实、可信，降低他人戒备心。
*   **4.3 其他内容**：作用有限，主要向外界传递“这是一个有正常工作的真人”的信息。除非工作内容本身极具价值，否则对个人魅力加分不多。

## 各类内容的分布比例

上一节我们分析了朋友圈的内容构成，本节中我们来看看这些内容应该如何分布，以达到最佳展示效果。

### 1. 本人出镜的展示面：重要性最高

本人出镜的展示面必须存在，即使发布频率不高。

以下是关于发布频率的核心建议：

*   在**一屏朋友圈**（即不上下滑动时手机屏幕显示的全部内容）中，**至少应包含一张本人出镜的展示面**。
*   示例：在一屏内容里，可能包含多条分享（如电影截图、文字感悟、歌曲），但其中必须穿插至少一条本人出镜的朋友圈。这确保了个人形象的核心展示。
