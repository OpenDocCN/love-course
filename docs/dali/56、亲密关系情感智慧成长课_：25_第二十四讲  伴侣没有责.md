# 56、亲密关系情感智慧成长课_：25_第二十四讲  伴侣没有责

在本节课中，我们将要学习如何理解和应对伴侣关系中常见的“缺乏责任感与担当”的问题。我们将分析这种行为背后的心理成因，并提供具体、可操作的应对策略。

---

## 问题概述与普遍现象

我相信大家身边可能会遇到这样的人。下班回到家后，不打扫家务、不做饭、不带孩子，而是直接躺在沙发或床上刷抖音、玩手机、玩游戏、看视频。这种行为让人非常气愤。当你请求对方帮忙时，对方往往会推三阻四，例如说“我忙着呢”。即使多次催促，对方可能只动一下，随后又继续忙自己的事。这是我真实咨询中的案例。

有的情况更离谱，当妻子说丈夫两句时，婆婆会站出来维护：“你老公在外面工作多累，让他回家休息休息怎么了？”这种情况非常普遍。

## 行为背后的心理模型分析

接下来，我们来分析这样的人是如何产生的。我们先通过两个关系模型来详细理解。

### 模型一：单身时的关系状态

第一张图有两个独立的圆圈，这代表我们单身时与他人关系的状态。

*   **公式：** `行为自由度 = 个人喜好 - 对他人利益的侵犯`
*   在单身时，我们可以根据自己的喜好和厌恶去做想做的事。只要不侵犯他人利益，自由度相当高。但我们与他人的关系处于比较疏离的状态。

### 模型二：伴侣关系中的状态

第二张图展示了进入伴侣关系后的状态。两个圆圈中间有一个交叉的红色阴影部分，这部分有两层核心含义。

1.  **关系的交织：** 进入伴侣关系意味着两个人生活交织在一起。我们必须摒弃单身时“想干什么就干什么”的模式，在行动时不仅要考虑自己，还要考虑对方。
2.  **共同的责任：** 组建小家庭意味着**责任**。两个人需要为这个小家庭共同负责、共同承担，以便更好地经营家庭。

两个圆圈的白色部分仍然代表我们自己，我们可以做自己喜欢的事，这没有问题。但前提是，我们必须先经营好共同的小家庭。如果忽视家庭责任，只顾自己任意妄为，那就是任性。

## 缺乏责任感心理的根源

进入伴侣关系后仍然没有责任感和担当，意味着此人心理上仍停留在单身状态，只凭个人好恶行事。这种心理状态通常源于**原生家庭**。

这样的人在小时候，父母往往为其包办了一切。家里所有活计和事情都由父母承担，生怕孩子磕碰，甚至不让孩子参与做饭、洗衣等基本家务。这养成了孩子“衣来伸手，饭来张口”的习惯，对家庭事务毫无参与感。

这样的孩子长大后进入伴侣关系，仍然缺乏参与感。即便组建了小家庭，他也会认为家务是“女人该做的事”，自己只需吃喝玩乐，做喜欢的事即可。这便形成了养尊处优的坏毛病。

## 现实挑战与常见阻力

回归到我咨询的案例。许多妻子看不惯丈夫的表现，要求对方承担家务，却常遇到阻力。

1.  **长辈干预：** 公公婆婆可能会站出来说：“你老公在外工作挣钱多辛苦，让他休息休息怎么了？”
2.  **伴侣的合理化借口：** 如果因此产生矛盾，丈夫会找各种理由，例如：“我最近在考公务员”、“我在准备司法考试，分心就学不好”。此时，长辈可能再次帮腔：“考试得让他好好准备啊！”
3.  **结果：** 丈夫通过不断“合理化”自己的理由，逃避作为伴侣应承担的责任。

## 核心应对策略

针对这样的人，我们该如何应对？这确实是个挑战。我们的目标不是让对方立刻变成极具责任感和担当的人，而是先让他**参与到家务事中**。这便成功了一半。

以下是具体的行动步骤。

### 策略一：确立共同解决问题的原则

首先，必须明确一个原则：当关系中一方感到不适时，另一方有责任配合解决对方的心理困惑。如果对方不愿帮助解决你的困扰，你需要拿出坚决的态度。

**核心对话框架：**
```
“我们还要不要继续这段关系？如果要，就必须解决这个问题。因为我现在非常不爽，心情糟糕。问题不解决，我没有心情继续过日子。我们需要一起参与解决。”
```

*   **过渡：** 明确了解决问题的原则后，我们可以借助外部力量来推动。
*   **寻求专业协调：** 找一个中间人（如心理咨询师）来协调双方的问题，帮助梳理相处中的矛盾，这对彼此厘清问题非常有帮助。

### 策略二：坚决执行家务平分计划

第二个方面，必须拿出坚决的态度，明确要求**平分家务**。

*   **具体做法：** 明确划分各自承担的家务，你做什么，对方做什么。
*   **应对借口：** 如果对方再次以考试等理由狡辩，你可以坚定回应：
    ```
    “你考试就不过日子了吗？要过日子，咱们就把家务分了。你承担一部分，我承担一部分。你是男主人，我是女主人。你挣钱，我也挣钱（有时甚至挣得更多）。家务活平分，这不合理吗？有什么问题吗？”
    ```

**关键点：** 务必从平分家务做起，让对方先参与进来。由于对方从小可能没有参与家务的习惯，你必须态度坚决，不能任由对方用各种理由糊弄过去。

## 总结

本节课我们一起学习了伴侣缺乏责任感和担当的问题。

1.  我们首先**识别了问题的普遍表现**。
2.  接着，通过关系模型**分析了问题背后的心理机制**，即个体仍停留在单身思维模式，未完成向伴侣交织与共同责任状态的转变。
3.  然后，我们**追溯了这种心理的根源**，通常与原生家庭中父母包办一切、孩子缺乏参与感有关。
4.  最后，我们探讨了**应对策略**：首先要确立共同解决问题的关系原则，必要时寻求专业帮助；其次要坚决执行家务平分计划，让对方从实际参与中逐步培养责任感和担当。

记住，改变需要时间和耐心。从让对方**参与**开始，是迈向有责任感关系的第一步。
