# 课程56：亲密关系情感智慧成长课_第二十二讲：成为一个像太阳一样的人 ☀️

在本节课中，我们将通过一个成功的婚姻案例，学习如何通过独立与给予，在亲密关系中成长为有能量、能温暖他人的“太阳型”伴侣，从而建立稳固而融洽的关系。

## 案例分享：一段成功的婚姻

我们分享一位女士的情感案例。她的婚姻情感特别成功，两人结婚20多年，一直相处得非常融洽。

他们非常恩爱。在家时经常拥抱，外出散步时也总是手牵手，很少例外。两人在结婚20周年时，隆重举办了纪念仪式，邀请了亲朋好友。

身边的人都非常感动和感慨。结婚20多年能相处成这样，让所有人都很羡慕。因为身边结婚多年的人，不是吵架，就是分手离婚，或因为家庭琐事产生各种冲突矛盾。

能相处成这样，已是大家心目中的成功婚姻。我听了对方的成功婚姻经历，特别好奇，便询问他们是怎样经营感情的。

## 经营之道：相互支持与独立

对方说，其实也没有很特意地经营。两人相处遇到问题时，都是相互支持，相互讨论。双方也都是相对独立的人，各有各的事情做。

事实上，她在工作和事业上做得比对方好。很多时候，她还会给对方一些支持和帮助，对方也比较感谢她。

## 关键事件：照亮家庭的力量

另一个方面，她对他们的家庭也特别好。记得有一次，婆婆得了癌症，不断看病也没看好。最后医生通知手术需要大额费用。

家里人都犯了愁，尤其是公公，觉得病已特别严重，到了中晚期，想放弃治疗。这位女士坚决提出来，要想办法给婆婆看病，不用担心钱。如果钱不够，就把房子卖了，一定要给婆婆看病。

关于找医生和医院，她会托朋友去找。经过她的努力，婆婆的病得到很好治疗，又活了十几二十年。全家所有人，包括婆婆，都对这位女士特别感激和感动，觉得她是一个特别好的媳妇。

作为老公，对自己媳妇所做的事情都看在眼里。虽然有时没有直接表达感谢，但从心底对这个女人充满了感激之情。所以在两人相处过程中，一直都非常融洽。

## 核心启示：成为太阳，而非黑洞

我们大家听到这位女士救助婆婆的事情，不知有何感受。我当时听到这里时，特别感动。我觉得这位女士的出现和行为，给整个家庭带来了光亮和温暖。她是一个非常有能量的人。

这位女士还分享了伴侣相处的其他方法，例如多关心对方，有问题共同商量等。其实讲了很多，我印象最深的就是她救助婆婆这件事。这一件事足以让我感受到她的力量、温暖和能量。我相信这是他们经营成功的一个很重要的方面。

我觉得婚姻情感要相处得好，一个很重要的方面就是我们能够去照亮对方，温暖对方。就像太阳一样能够照亮对方，让对方感受到光亮和温暖。

那么他自然就离不开你，因为对他而言，你是极其重要的。他可以离开任何人，但不能离开你。如果我们像黑洞一样，不断吸取别人的能量和情感，那对方自然就会远离你。

因为他觉得你在不断消耗他，那么就会出现你追他跑的局面。像黑洞一样吸取别人能量的人，是一个依赖的人。我们要知道，依赖就是消耗，依赖就是索取。没有任何一个人可以承受你无止境的消耗和索取。

即便两人最开始在一起时，他何等承诺对你好，最后都会被吓跑。所以我们大家不要成为一个依赖别人的人。

## 从独立开始：依赖的对立面

如果我们自身不能成为一个像太阳一样能照亮、温暖别人的人，起码我们先让自己独立起来。依赖的对立面就是独立。我们先让自己独立起来，先能照顾好自己，照顾好自己的情绪和身体，能够养活自己。这是一个很重要的方面。

很多女孩子都讲到，不在意对方有没有钱，只要对我好就行。我们来分解一下“对我好”是什么意思，它包含三层意思。

以下是“对我好”的三个层面：
1.  **你要能够认同我的价值观念。**
2.  **你要重视我的感受，能够理解我。**
3.  **你要包容我的情绪。**

无非就这三个方面。大家可以仔细琢磨，我们所说的“对我好”是不是这个意思？如果我们是一个依赖的人，那么我们就会不断从对方那里索取这三个方面。

大家可以感受一下，依赖的人是不是在索取这三方面：你要认同我，你要重视我的感受、理解我，你要包容我的情绪。无非就是这三个方面。所以如果我们要成为一个依赖别人的人，很多时候就是在强调这三方面。

## 迈向独立：三个层面的自我建设

如果我们要成为一个独立的人，那么我们就需要从这三个方面入手，先自身能够处理这三个方面。

以下是迈向独立的三个具体方法：

1.  **关于价值认同**
    要让自己独立，就需要先看到：对方可以认同我，也可以不认同我。他也是一个独立的人。有时他也需要被认同。他认同我，我也会去认同对方。如果我们遇到具体问题，可以求同存异，而不是要求对方迁就我。这样对彼此都比较公平。

2.  **关于感受重视**
    “重视我的感受”无非是希望你能把我放第一位，凡事先考虑我，先照顾我的需求，我不说你就能懂当然最好。但这基本上还是在要求对方把注意力吸引到你身上，先考虑你。
    其实，你有什么需求或想法可以先表达出来。你想要什么，需要对方怎么配合，都可以说出来。需要对方协助时，可以请求对方帮助你。这样，我们就可以让自己更多关注自身的情绪感受，自身先消化处理，而不是给对方带来压力。

3.  **关于情绪包容**
    有情绪就是有需求。你无非是希望对方能够满足你的需求。在这一点上，要想独立，那就有什么需求就直接说，不要让对方去猜。需要对方协助，也可以告诉对方。生活中每个人都比较累，没有太多心思一直把注意力放在一个人身上。凡事不要消耗对方，有事就说出来，共同解决。

如果这三个方面我们都能够自我消化处理，而不是给对方施加压力，那么我们就能够变得更独立一些。

## 成长路径：从独立到发光

当然，如果你自我独立也做得很好了，那么就可以不断提升自己的能量，让自己变得更有力量感，更有能量一些。你也可以分散出一些精力，给到对方更多的认同、情感上的关注和理解，能更多体谅对方日常生活中的困难。这都是可以的。

**成长路径可以总结为：独立 -> 提升能量 -> 成为太阳。**

就是我们先要独立，然后不断提升自己，才能够变得像太阳一样有能量，能够去照亮别人，温暖别人。所以我们大家可以先从独立做起。我们慢慢有力量了，就可以给到对方更多支持。

你给到对方更多支持时，对方就会离不开你，会更亲近你。其实两个人的关系就会越来越好。所以说咱们大家都要往成为一个像太阳一样的人而努力，让自己变得有力量感一点。
