# 56、亲密关系情感智慧成长课_：27_第二十六讲  好的伴侣关

在本节课中，我们将要学习伴侣关系中的一个核心认知模式：“你行我也行”。我们将探讨这种模式如何促进关系和谐，分析与之对立的“我行你不行”模式为何会引发冲突，并提供具体的实践方法来调整我们的相处方式。

## 核心认知：从“竞争”到“共赢”

上一节我们探讨了伴侣关系中的矛盾起源。本节中我们来看看，矛盾的深化往往源于一种错误的认知定位。

好的伴侣关系是“你行我也行”的状态。只有这样，双方才能感受到彼此的支持、肯定和认同，从而体验到高质量的伴侣关系。

如果关系中存在“我行你不行”的想法，相处过程中就会积攒许多怨气和愤怒等负面情绪，这会严重阻碍两人关系的发展。

## “我行你不行”模式的陷阱

我们之前讲到，伴侣矛盾最初产生于对同一事情有不同看法且无法达成共识。而无法达成共识的关键，常常是双方都想证明“我行，你不行”。

**为什么证明“我行”一定要建立在“你不行”的基础上？** 这个认知是阻碍伴侣关系和谐的重要因素。

例如，在教育孩子的问题上：
*   **丈夫**认为应该“放养”，让孩子顺应天性发展。
*   **妻子**认为应从小培养良好习惯，比如先写作业再玩。

两人的看法本身都没有问题。但如果丈夫坚持“我的行，你的不行”，妻子也坚持“我的行，你的不行”，关系就会陷入争斗模式，最终两败俱伤。

当认知进入“我行，你不行”的模式，关系就变成了对支配权的争夺，必然产生更多摩擦。

## 迈向“你行我也行”的和谐之路

当认知转变为“我行，你也行”时，相处就会更和谐融洽。

依然以孩子教育为例：可以认为“你的看法有道理，我的看法也不错”。双方的初衷都是为了孩子好，目标是一致的。既然彼此的看法都“行”，就可以在此基础上寻找让结果更好的方法。

例如，可以共同寻求专业心理咨询师的帮助，一起讨论如何融合双方的想法，达成对孩子更有利的方案。

## 实践应用：处理两类问题

基于“你行我也行”的认知，我们可以将伴侣相处中的问题分为两大类来处理。

以下是针对重大问题的处理思路：
*   **问题类型**：长期、持续的重大问题，如子女教育、购房购车等。
*   **解决方法**：共同听取更专业人士的建议。这能避免陷入“我行你不行”的争执。双方在“你行我也行”的基础上，进行审慎思考，做出更科学、更优的决策。核心是**共同寻求专业帮助**。

以下是针对琐碎问题的处理思路：
*   **问题类型**：日常琐事，如家务分工、购物、走亲访友等。
*   **解决方法**：这类问题通常没有绝对对错，双方看法都“可行”。关键在于**次序安排**。例如，这次由你决定，下次由我决定；或按单双日、单双周来轮流决定。

## 认知根源与调整方向

讲到这里，大家会发现，将伴侣相处模式转向追求“你行我也行”，很多问题都能迎刃而解。过去的困境往往源于追求“我行你不行”的错误认知，这导致双方根本无法进行良好沟通。

因此，我希望所有伴侣在追求关系良性发展时，能将方向调整为“我行，你也行”。

为什么很多人会在关系中追求“我行你不行”的认知？这常与原生家庭有关。许多父母采用一种权威、专制的相处模式，总让孩子按自己的想法来。孩子习得了这种模式，并将其带入伴侣关系，继续追求“我行，你不行”，试图证明自己正确、优越。

## 课程总结

本节课中，我们一起学习了伴侣关系中的关键认知模式。

我们希望从今天起，大家能逐步调整这一认知：**想要证明自己是“好”的、“行”的，并不必须建立在对方“不行”或“错”的基础上。**

如果我们的相处模式能转变为“我行，你也行”，那么两人的关系将会更加和谐，更能促进伴侣关系的长远发展。
