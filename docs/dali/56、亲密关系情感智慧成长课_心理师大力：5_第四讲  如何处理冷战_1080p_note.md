# 课程56：亲密关系情感智慧成长课 - 第四讲：如何处理冷战 ❄️

在本节课中，我们将要学习如何处理伴侣相处中常见的“冷战”现象。我们将深入探讨冷战的本质、双方的心理动机，并提供清晰的视角来理解和应对这一挑战。

---

冷战是伴侣相处中非常常见的现象。很多时候，我们不知道如何处理冷战，导致两人关系恶化，最终可能导致关系破裂。今天，我们将逐步分享关于冷战的问题，让大家以新的视角看待冷战究竟是怎么回事，以及其中蕴含的意义。

我们知道，冷战其实是伴侣相处过程中，一个人对伴侣的某些行为感到内心不满意。但他不说出来，自己憋着消化，又消化不了。然后在两人相处的过程中，不断挑刺，给对方冷脸色，让对方感到不爽。这是冷战常有的表现。冷战还会表现出一种逃避和拒绝沟通。例如，当你找对方沟通，希望解决问题时，对方可能说“没事”、“我觉得很好”、“没什么可说的”。对方拒绝沟通，但在相处中又让你感到不爽。这就是发起冷战的人经常表现出的行为。

当然，发起冷战的人往往不善于表达内心的情绪、需求和想法。我们知道，有情绪就意味着有需求，有强烈的情绪一定意味着有强烈的需求未得到满足。发起冷战的人内心是有需求的，他可能更多是不知道如何通过语言更好地表达自己的需求。就像一个小婴儿，有需求时不说话，而是哭闹、发脾气，让父母来猜测。很多时候父母不了解他真正的需求是什么，于是根据自己的理解给予回应。如果没有给到孩子想要的回应，孩子就会继续哭闹。因此，发起冷战的人似乎更多是希望伴侣能够猜测他的想法，或者及时满足他的需要。

从冷战这个问题，我们其实能看出发起冷战的人有不成熟或幼稚的地方。我们知道，冷战不仅对自己有伤害，对亲密伴侣关系也有很大伤害。例如，你天天发脾气，憋出内伤，情绪和心情受到抑制，脸上总是不开心，生活没有热情。同样，在关系中，你天天传递负面情绪，别人看到也会不爽、不高兴，也会没有心情。两个人在相处过程中就无法友好沟通。本来最亲近、最亲密的人，变成了一种冷漠疏离的状态。这样的生活能过得好吗？心情能愉悦吗？因此，冷战无论对自身还是对伴侣关系都没有任何帮助。

但是，发起冷战的人为什么仍然不愿意解决问题，处理自己的情绪状态，解决困扰自己的问题呢？这样不是更好吗？然而，发起冷战的人就是不愿意。为什么呢？大家可以一起来想一想。

我能想到的一个非常重要的点是：如果这个问题解决了，反而会带给我更不好的体验。只有让这个问题不解决，我才能获得最大利益。大家可能会说，老师你这样说有什么逻辑或依据吗？我们就逐步来分析。

大家可以想象一下，如果伴侣相处中的问题解决了，这代表什么意思呢？代表我不可以再埋怨你了，你不亏欠我了。如果问题不解决，那么你就是亏欠我的，我就可以埋怨你。我能够让你一辈子都欠我，都离不开我，心里都想着我。大家可以琢磨琢磨是不是这样。其实发起冷战的人就是在说这样一句话：“你欠我的。”如果问题解决了，你就不欠我了。大家可以想象一下，在一段亲密伴侣关系中，一个人天天在你耳边说“你欠我的，你欠我的，你就是欠我的”，那是什么样的感觉？并且，有的人结婚后相处几年、十几年、几十年，一年365天，说成千上万遍，天天在你耳边说“你就是欠我的”，谁能受得了？但这个发起冷战的人，潜在的观念里就有这样的意思：“你就是欠我的。”所以他不愿意去解决问题，因为解决了问题你就不欠我了。你只有亏欠我，我才能更多地依赖你，你对我进行补偿，是不是这样？如果真的把伴侣相处中的问题解决了，他反而会不乐意。这意味着你就不欠我了，不欠我了怎么能行呢？你亏欠我一辈子，那样才行。

有的人可能没有看到这一点，在伴侣关系中开始自我怀疑：“哎呀，是不是我真的做得不够好？是不是我做得太糟糕了？我是不是得真正地补偿对方？”这种冷战问题一直存在，那么被冷战的伴侣也需要承担一半的责任。为什么呢？因为一个人一天说几十遍，一年365天说成千上万遍在你耳边说“你欠我的，你欠我的，你就是欠我的”，你竟然也接受了、默认了。这不是你惯出来的吗？当对方第一次说“你欠我的”时，你就应该讲出来：“我哪里欠你了？你可以说一说呀。这样子好让我知道问题在哪儿，我们把问题解决解决。”你也不说，那你这是什么意思呢？你不说，又天天说我欠你的，你这是什么意思？

这个事情让我想到了罗永浩跟记者的一个辩论。一个记者说：“罗永浩，你们的产品是有问题的，一些检验部门已经检查了你们这个产品是有问题的。”那罗永浩就问这个记者：“哪个机构检验的？你告诉我是哪个机构。”然后这个记者说：“我不能告诉你，这是保密的。”那罗永浩就生气了：“你这不是很无耻吗？你往我头上扣了一个屎盆子，说我们有问题，你又拿不出来证据。那你这不是在侮辱我的智商？你这不是很无耻吗？”同样，在伴侣关系中，发起冷战的一方如果天天对我们说“你就是欠我的，你就是欠我的”，但当你真的问他，他又说不出一二三，又拒绝沟通，给我们冷脸色。你问他：“我哪里欠你的？”对方又拒绝沟通，又不说，那这不是很无耻吗？并且常年累月，尤其是结婚几十年的，仍然是冷战的关系状态，天天说你欠我的，这不是我们被冷战的一方纵容的结果吗？

因此，要解决冷战这个问题，被冷战的一方也需要站出来把问题澄清澄清。两个人是不是要继续过？要继续过，就把冷战这个问题拿出来解决解决，把问题处理处理。这需要被冷战的一方也拿出一个态度。如果过，咱们就说解决问题；如果不过，那冷战当然就不用解决了。需要拿出一个态度。被冷战的一方需要拿出态度，冷战的一方也需要看到自己到底在干什么。很多发起冷战的人根本不知道自己在干什么，他不愿意解决问题的背后，就是想让别人去欠他。

所以我们今天的分享，就是希望大家能够对冷战这个问题有更深刻的认识。冷战表面性的东西是“你让我不爽了”，但实际上，发起冷战的人其实是他不想解决问题。不解决问题，你们才能永远欠我，你们心里才能天天想着我，才能对我进行一些补偿。有太多太多的伴侣都上了这个当，无休止、无止境地去补偿，仍然填补不了发起冷战一方的欲望。因此，我们必须把这个问题分析透彻，分析透彻，我们才能更好地去面对。

冷战这个问题不处理，我们不可能更成长。冷战这个问题处理好了，伴侣双方都会有更进一步的个人成长。所以有问题不可怕，我们能够共同去面对、去解决，把这些问题弄清楚。因此，我们需要换一些视角，从新的角度重新看待我们伴侣相处中的问题，我们往往会有意外的收获。

---

本节课中，我们一起学习了冷战的本质及其对关系的伤害。我们分析了发起冷战一方的潜在心理动机，即通过维持“亏欠感”来获取关系中的利益。同时，我们也认识到被冷战一方在其中的责任，以及双方共同面对和澄清问题的重要性。处理冷战的关键在于双方愿意沟通、澄清问题，并共同寻求解决之道，从而促进个人与关系的共同成长。
