# 56、亲密关系情感智慧成长课_心理师大力：10_第九讲  如何处理愤怒情绪_1080p_note

在本节课中，我们将要学习愤怒情绪的本质、它如何产生，以及如何从根本上处理愤怒，从而改善伴侣关系与个人状态。

没有任何一种情绪比愤怒对人际关系的伤害更大。了解愤怒情绪，能帮助我们更好地处理它，改善人际关系与伴侣关系。

## 愤怒是如何产生的？🔥

上一节我们提到了愤怒的破坏性，本节中我们来看看愤怒究竟是如何产生的。

愤怒的人往往不知道如何通过语言更好地表达自己内心的需要、想法和感受。因此，他们通过发怒——一种让别人害怕的方式——来引起关注和重视，从而震慑住别人。这是愤怒者常运用的一种方式。

为了更具体地理解，我们可以通过一个比喻来演示。

假设有一个树枝，上面长着4片树叶。这个树枝代表一个人，而树叶代表他所拥有的东西或内在能量。

1.  当我们折掉第一片树叶时，树枝剩下3片。此时，树枝（人）可能会开始感到些许不安或损失。
2.  当我们折掉第二片树叶时，树枝只剩下2片。此时，树枝（人）的情绪可能转变为明显的焦虑或不满。
3.  当我们折掉第三片树叶时，树枝仅剩1片。此时，树枝（人）很可能感到强烈的危机感和被剥夺感。
4.  最后，当我们折掉最后一片树叶时，树枝变得一无所有。此时，树枝（人）极有可能感到极度的愤怒，甚至产生攻击性。

**公式：愤怒 ≈ 感知到的威胁 ÷ 内在资源**

当一个人内在资源（树叶）很少时，即使很小的索取（拿走树叶）也会被感知为巨大的威胁，从而引发强烈的愤怒。因为他感受到了入侵、伤害和不公正的对待。

## 为何有人易怒，有人不易怒？🌲

理解了愤怒产生的模型后，我们就能明白生活中人与人之间的差异。

容易愤怒的人，就像那根只有几片树叶的树枝。他内在没有多少东西（能量、资源、安全感），因此会拼命保护自己不被索取。当别人拿走一些东西（可能是实物、时间、尊重），他就会感到巨大威胁，从而通过愤怒来反击。

而不易愤怒的人，则像一棵参天大树。他拥有成千上万的树叶（丰富的内在资源）。别人拿走一片树叶，对他而言无关紧要，因为他给予得起，甚至乐于分享。

这解释了为什么生活中，有的人被说一句就气急败坏，而有的人则云淡风轻。核心区别在于内在的丰盈程度。

## 自我觉察：你是小树枝还是参天大树？🤔

在伴侣关系中，你可以对比一下自己是否是一个特别容易愤怒的人。

如果你特别容易愤怒，可能反映了你当下的状态就像一个小树枝，内在能量不足，倾向于拼命自我保护，避免被过度入侵和伤害。伴侣的一句话、一个眼神，都可能被你敏感地感知为伤害。

以下是两种状态的对比：

*   **小树枝状态**：内在资源少，防御性强，易将小事感知为威胁，反应激烈。
*   **参天大树状态**：内在资源丰富，包容性强，不易感到被侵犯，反应平和。

显然，我们都希望成为后者——一棵内在稳定、不易被激怒的参天大树。

## 如何成长为“参天大树”？🌱

认识到自己是“小树枝”并不可怕，关键在于如何成长。我们需要为自己“浇水、施肥、撒播阳光雨露”，而不是纠缠于伴侣相处中的鸡毛蒜皮。

成长的方向是让自己内心变得强大，改变对事件的认知和解释方式。人的痛苦和愤怒，很多时候源于我们的认知。

以下是几种可行的成长途径：

*   **靠近高能量者**：多与内心平和、有智慧的人接触，向他们学习。
*   **学习心理学知识**：通过阅读、课程了解情绪和行为的底层逻辑。
*   **参与系统培训**：参加类似本课程的系统学习，持续提升自我。
*   **转变认知核心**：认识到愤怒常源于自身的脆弱，而非事件本身。将注意力从“对方伤害我”转向“我如何变得更强大”。

处理愤怒最重要的方向，不是控制或压抑它，而是**从根本上提升自己的内在能量和认知水平**。

## 课程总结 📝

本节课中，我们一起学习了：

1.  **愤怒的本质**：是一种在无法有效表达时，通过震慑他人来保护自己的方式。
2.  **愤怒的产生模型**：当**感知到的威胁**与**内在资源**失衡时（威胁 > 资源），愤怒极易产生。
3.  **易怒的根源**：内在像“小树枝”一样资源匮乏，防御性强。
4.  **成长的方向**：通过学习和自我提升，让自己从“小树枝”成长为“参天大树”，增加内在资源，改变认知。

对愤怒情绪的理解，是改善伴侣关系的关键。因为没有任何一种情绪比愤怒对关系的破坏性更大。希望今天的分享能帮助你调转方向，专注于真正的自我成长，这才是处理情绪、经营好关系的最佳方式。
