# 李越习惯性亲密关系：12.Part4第十一课：习惯性亲密关系的心态构建及原理 🧠

在本节课中，我们将要学习如何构建适用于习惯性亲密关系的心态，并理解其背后的运作原理。我们将探讨在与特定人群相处时，如何通过模拟父母角色的心态来建立和维持关系。

经过前面的学习，我们了解了为什么有些人需要习惯性的亲密关系。这样的人可能对自己的欲望和人格边界缺乏清晰认识，因此在亲密关系中难以主导，反而更倾向于被主导。

被主导这一需求，引出了一个关键问题：我们应该以何种形象出现在对方面前？当我们决定用习惯性亲密关系的方式与人相处时，首先需要构建正确的心态。

## 心态构建的三个阶段

这种心态主要可以分为三个核心点。我们的角色本质上是对方的“父母”。因此，心态构建的第一点就是：以父母的态度开始与对方相处。

那么，父母与孩子（或对方）的相处通常会经历哪三个阶段呢？

以下是这三个阶段的具体内容：

1.  **第一阶段：哺育期（无限包容）**
    在关系初期，就像父母照顾婴儿一样，我们需要给予对方无微不至的关心、照顾和体贴。这个阶段的核心是无限的包容与纵容，因为对方可能像婴儿一样，对自身需求和边界缺乏清晰认知。这种态度有助于让对方更容易接受这段关系。

2.  **第二阶段：规训期（提出要求与约束）**
    当关系进行一段时间后，就像孩子长到六七岁，父母开始提出要求、进行约束和管教一样，我们也需要对对方提出期望和要求。对于习惯被主导的人来说，他们内心渴望对方主动提出这些要求，因为这符合他们熟悉的、由父母设定规则的模式。此时的心态应着眼于**共同的未来**，所有行为都应传达出“我希望你变得更好，我们一起构建未来”的期望。其核心公式可以概括为：**行为 = 对共同未来的规划与期望**。

3.  **第三阶段：回馈期（表达需要与承诺）**
    在最后阶段，就像父母期望孩子成年后孝顺、回馈家庭一样，我们需要告诉对方我们在未来需要他，需要他做出特定的承诺或行为来保证关系的持续。这种方式模拟了完整的亲子生命周期，能给对方带来安全感。

## 两个至关重要的心态要点

上一节我们介绍了心态演变的三个阶段，本节中我们来看看维持这种心态必须坚守的两个核心要点。

以下是这两个要点的具体阐述：

*   **接受关系中的高低位与分离可能**
    在亲子关系中，父母天然处于高位，掌握着“生杀大权”（如将孩子关在门外）。在成人关系中，我们无法真正掌控对方，但必须建立起“我可以接受你离开”的心态。只有当我们做好了对方可能离开的准备，并且不因此恐惧时，对方才能在潜意识中将你视为“父母的替代品”。否则，对方会醒悟“我可以离开你，你凭什么驾驭我？”，从而破坏关系的安全感基础。

*   **坚信行为的正当性与爱意**
    我们必须发自内心地相信，我们所做的一切——无论是包容、约束还是要求——都是**真正在为对方好，是在用我们的方式爱对方**。就像父母坚定地认为某些事是孩子“应该做的”一样，我们必须有底气。如果我们的态度是商量或犹豫的，一旦对方反驳我们就退缩，那么对方会对我们的要求产生嫌弃和厌恶。核心逻辑是：**因为你觉得安全，所以我这样爱你**。
