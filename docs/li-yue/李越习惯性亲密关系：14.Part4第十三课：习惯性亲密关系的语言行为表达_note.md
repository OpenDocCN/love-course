# 李越习惯性亲密关系：14.Part4第十三课：习惯性亲密关系的语言行为表达 🗣️💬

在本节课中，我们将要学习习惯性亲密关系的具体操作方法。上两节课我们已经学习了应持有的心态，以及如何识别伴侣是否处于习惯性亲密关系中。本节我们将具体讲述一个分为四个步骤的操作方法。

## 第一步：模拟希西弗斯的故事，给予无限宠溺与包容

上一节我们介绍了识别方法，本节中我们来看看如何开始操作。这个方法的第一步，是继续模拟希西弗斯推石头的故事。我们想象，一个人与父母相处的痛苦循环，就如同希西弗斯不停推石头的无限循环。他渴望逃离这种关系。

因此，在关系初期，我们绝不能告诉对方“你要继续推石头”。相反，我们要模拟婴儿阶段的相处模式。在婴儿阶段，父母给予的是无条件的宠溺与包容，没有要求和期望。

以下是操作的核心逻辑：
*   **核心目标**：告诉对方，只要与我们在一起，就可以逃离“推石头”的循环，获得幸福未来。
*   **操作方法**：识别并无限满足对方的缺失。这本质上是**提供价值**，建立“我们=安全港湾”的预期。
*   **具体做法**：根据对方缺失的类型，给予对应的满足。例如：
    *   对方**缺乏关注**，就给予高频、及时的回应与陪伴。
    *   对方**缺乏物质**，就在能力范围内通过礼物、红包等方式表达诚意。
    *   对方**缺乏善待与理解**，就无条件地站在他一边，**共同讨厌**他讨厌的事物。共同讨厌比共同喜欢更能快速建立共鸣。

通过这一步，我们在对方潜意识中植入一个信念：与我们在一起，就不再需要经历过去的痛苦循环。

## 第二步：建立有条件满足，模拟成长初期的规则

当我们完成了初期的无限宠溺后，对方开始享受这种好，但内心会产生不信任，担心这种好无法持续。为了建立更深的信任，我们需要进入第二步。

上一节我们通过“提供价值”建立了连接，本节中我们来看看如何深化信任。这听起来矛盾：对方想逃离旧模式，为何要让他回去？实际上，这模拟了人3-4岁时的成长阶段。那时父母开始设立规矩，做到某事就能获得奖励。

以下是此阶段的操作逻辑：
*   **核心目标**：将无条件的“提供价值”，转变为**有条件的价值交换**。让对方形成“我做到A，就能获得B”的思维模式。
*   **操作方法**：在关系稳定后，开始有条件的满足对方。例如，当对方表现出你期望的特质（如更体贴、更上进）时，才给予之前那种程度的关注或奖励。
*   **对方心理**：对方可能不爽，但不会轻易离开。因为他已将你视为“救命稻草”，并且会自然地想：“我只要做到某事，就能换回他的好。”这符合他习惯的互动模式。

这一步的关键在于，我们主动创造了规则，开始管理对方的预期和行为。

## 第三步：进行“关乎未来”的指责，模拟管教阶段

当关系进入有条件满足阶段后，对方可能会“作妖”来试探规则的边界，例如“我不做A，你是否还会给B？”。此时，我们需要进行第三步：指责。

上一节我们建立了规则，本节中我们来看看当规则被挑战时该如何应对。这模拟了童年时期父母对我们的管教。当孩子闹脾气不守规矩时，父母会指责。这种指责反而让孩子感到被在乎，关系是安全的。

以下是此阶段指责的核心原则：
*   **指责的出发点**：**不要着眼于当下事件的对错**，而要**着眼于共同的未来**。指责的理由是：“你这样做，会破坏我们未来的幸福。”
*   **指责的时机**：通过询问对方对未来的构想来判断。**当对方的未来构想中包含你时**，即可视为进入可以指责的阶段。因为这表明他已在考虑长期关系。
*   **指责的话术**：将具体行为与长远后果挂钩。例如，不说“你为什么不回我消息？”，而说“你总是忽略我的感受，这让我很担心我们以后能否彼此信任、好好沟通。”

这种“上纲上线”的指责，因为关联了“美好的未来”，反而能给予对方安全感，让他感到被深切地在乎。这是习惯性亲密关系中的核心沟通技巧。

## 第四步：循环运用奖励与指责，完成行为塑造

第二、三步并非一次性完成，而需要混合循环使用，以完成对对方行为的塑造。

以下是循环操作的步骤：
1.  **设立规则（第二步）**：提出期望，建立“做A得B”的条件。
2.  **对方试探/犯错（触发点）**：对方可能不遵守规则。
3.  **进行“未来式”指责（第三步）**：就事论事，关联未来后果进行批评。
4.  **对方改正（关键节点）**：若对方被说服并改正行为。
5.  **给予奖励（回到第二步）**：必须及时给予夸赞、奖赏或满足，强化其正确行为。**公式为：改正行为 -> 获得奖赏**。
6.  **循环往复**：重复此过程，不断塑造对方行为。

**核心要点**：很多人只会指责，却忘了在对方做对时给予奖励。这会导致对方失去持续做“对的事”的动力。对于自尊水平低、自我意识模糊的伴侣，这种“训养”是引导其走向健康关系的重要过程。

我们所有的操作，都像是在帮希西弗斯换一块**能推到山顶的石头**。我们模拟他熟悉的循环模式给予安全感，但目的却是导向一个**有终点（山顶）的健康平等关系**。只要初衷不是伤害，方法本身就没有问题。
