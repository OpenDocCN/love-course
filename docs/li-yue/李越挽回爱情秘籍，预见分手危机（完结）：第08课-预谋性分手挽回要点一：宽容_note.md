# 李越挽回爱情秘籍，预见分手危机（完结）：第08课-预谋性分手挽回要点一：宽容_note

在本节课中，我们将要学习预谋性分手挽回的第一个核心要点：宽容。我们将探讨为何宽容至关重要，以及如何在实践中运用它来为挽回创造可能性。

上一节我们介绍了预谋型分手的原因和常见手法，本节中我们来看看挽回的第一个关键步骤。

预谋型分手的挽回难度很高，但并非毫无办法。这些方法中包含许多需要被了解和掌握的要点，而很多人可能缺乏相关的意识。

例如，我们要讲的第一个挽回要点就是宽容。现实中的分手剧情往往不像电视剧里那样和平。大多数分手过程都充满争执，例如吵架、辱骂甚至肢体冲突，最终愤然分开。尤其是当一方发现对方的分手早有预谋、自己一直被欺骗时，情绪会更加激烈，可能导致口不择言。

或许很多人对此感到不满，认为对方欺骗了自己，难道还不能生气或责骂吗？并非不能生气，但我们需要明白，这种情绪宣泄会对挽回造成何种影响。毕竟，很多人在生气后仍然试图挽回，这看似矛盾，却是常见情况。

人在生气时容易失去理智。一旦失去理智，就会说出许多绝情且不留余地的话。这些话很可能触及对方内心深处不愿面对的部分。而很多人之所以故意说这些话，正是为了让对方无法面对和接受。

因此，很多人在分手时会选择最扎心、最让对方难以忍受的话去攻击对方，直接揭露对方内心自认为最不堪的一面。这正是导致许多人无法挽回、关系无法复合的原因，而他们自己可能并未意识到。

可以这样理解：如果你骂了对方，即使他内心有过复合的想法，他又如何能回头呢？你不是不能骂，但一旦骂了，就要做好放弃复合的准备。骂完之后你爽快了，还想复合，这通常不现实。如果你已经骂过了，我们后续再讨论，因为世间之事有得必有失。你享受了发泄的快感，就可能要付出挽回困难的代价。

所以，预谋性分手挽回的第一个要点就是宽容。即使我们知道对方的分手是有预谋的，我们也需要宽容。

我们需要宽容对方的许多方面：宽容对方的预谋、宽容对方的不喜欢、宽容对方的欺骗、性欲、虚荣、寂寞等等。你可能会问：“我为什么要宽容他？我凭什么不戳穿他？”如果你不想宽容，那就不要挽回了。只要你不打算挽回，你可以不宽容。但如果你想挽回，就必须宽容。事情就是这样。你又不想宽容，又想挽回，世界上没有这样的事。如果你坚持如此，那可能只是为了出气，而不是因为你有多么喜欢对方。

很多人可能并不理解宽容到底是什么。我们来举一个例子。

一个男生追求你、撩你，目的是满足自己的某些需求，本质上他并不喜欢你。这种行为本身就带有欺骗性质，是可耻的。同时，他非常害怕你认为他是坏人。他预谋和你分手，采用挑剔、不宽容和冷暴力的方式，也是害怕你认为他是坏人。他害怕被指责、辱骂和否定，因为他无法接受这样的事情发生在自己身上。他甚至非常害怕这种激烈冲突。这是一个非常复杂的问题，与他的原生家庭有很大关系。

试想，如果你在分手时选择直戳对方内心、加以指责辱骂和羞辱，对方确实会感到非常难受。但难受之后，对方只会更加逃避我们。因为我们知晓了他的秘密，并且成为了真正会攻击他这个秘密的人。我们找到了他的耻辱所在，我们本身就成了他的耻辱。只要他一看到我们，就会想起自己卑鄙、无耻、下流的一面。

在这种情况下，根本没有办法挽回。即使他内心曾想过复合，但面子和内心的尴尬也不允许他回头。因为他会觉得，一旦回头，我们就会嘲笑他。他甚至会觉得，他想复合这件事会成为我们羞辱他的工具。所以，对方基本上不会回头。

这就是为什么预谋型分手在现实生活中挽回的几率特别低。因为很多人分手后只顾宣泄怒火、发泄不满，完全忘记了这种行为只会加深对方内心深处的耻辱感和愧疚感。最终，对方内心的保护机制会把这些所有对自我否定的负面能量投射给我们。于是，很多人分手骂完后就开始恨对方、厌恶对方，这时谈挽回完全是无稽之谈。

如果你已经骂过了，那没有办法。你需要给你们两人一点时间，让时间去淡化冲突，缓和矛盾。期间最好不要有任何交集和交锋。如果对方已经把你拉黑，你可以尝试重新添加。如果对方没有同意，可以再等等，过一段时间再尝试。

所以，很多人分手后明白了这个问题，听了这节课，马上就去联系对方。何必呢？先等等，缓和一段时间，缓一个月、两个月或三个月。这时你可能会问：“要是他在这段期间找到了新对象呢？”那你就不要挽回了，你也可以去换一个新的人。我们讲这节课不是在讲成功学，事实就是如此。如果你非要钻牛角尖，那没有人能帮得了你。

如果你没有说过激的话，或者已经说过但时间过去了很久，你们之间的矛盾已经淡化，那么我们必须履行“宽容”这个原则。

宽容的目的就是让对方知道我们不怪罪他，让他可以心安理得地与我们相处。我们不怪罪你不喜欢我，不怪罪你只想发生关系，不怪罪你因为空虚、孤独、寂寞等等。我们不怪罪你的冷暴力，不怪罪你的挑剔，不怪罪你的欲望，诸如此类。

例如，我们完全可以去和对方聊天。记住，最好是面对面、心平气和地进行。因为这番话不仅仅是单纯的话语，而是许多有智慧的女生都会选择的一种表达方式。她们通常会陈述一些事实。

以下是你可以尝试的一种沟通方式和内容要点：

我们可以和对方坐在一起，面对面地聊。比如可以说：“其实你知道吗？我知道你当初并不喜欢我。你和我在一起，可能只是因为你想要（某个原因）。” 这里的“某个原因”需要你去分析你们两人在一起的原因，可能是因为空虚、对方的性欲或寂寞等。

我们假设现在面对的“某个原因”是性欲。我们可以这样表达：“现实情况是这样的。其实你知道吗？我知道当初你并不喜欢我。我知道你当初跟我在一起，其实只是想跟我上床。” 男生通常可能会否认：“不是啊，没有啊，你怎么会这样想我？”

这时你记得不能生气。永远记住你不能生气，因为你不是在怪他，你只是在陈述某些问题。这时最好的一种表达方式是：笑一下，然后说：“傻瓜，我说这些不是在怪你。我只是说，有些事情其实它根本就不重要。就像我很喜欢你，对不对？其实我能和我喜欢的男人上床，本质上……”

看到了吗？在挽回的过程中，你只能把他约出来，心平气和地说这些话。如果你后续还能约他出来，基本上挽回的几率就很大。但这只是要点一，因为要点二我们会在后面课程讲到。但你需要谨记，在这个过程中，你不能和他发生关系。

如果他想借此机会和你发生关系，并且没有说要复合或恋爱，那不好意思，他可能又来骗你了。难道你需要在一个坑里摔两次吗？难道你就不考虑这个人的人品问题吗？如果你还是要执意挽回，那不好意思，那你就自作自受。
