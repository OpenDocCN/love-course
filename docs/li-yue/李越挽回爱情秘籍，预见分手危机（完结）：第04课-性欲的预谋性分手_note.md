## 🧠 核心概念与本质

这种关系的核心公式可以概括为：
**建立关系（手段） → 发生关系（目的） → 分手（结果）**

**中期阶段：话题转向与空间试探**

一旦发生关系，关系会迅速冷却并走向终结。

*   **情感抽离**：在关系后期，尤其是进行性暗示期间，几乎不会有“我喜欢你”、“想你”等情感表达。

许多受害者难以认清事实，常陷入以下误区：

2.  **混淆手段与目的**：将对方为达成目的而采取的“建立关系”这一手段，误解为是感情发展的自然结果。
3.  **被关系身份绑架**：在对方以“男朋友”身份提出性要求时，感到难以拒绝，误以为这是恋爱的必然部分。
4.  **分手后陷入执念**：认为自己付出了感情与身体，不甘心被分手，从而强烈想要挽回。

*   **关于挽回**：此类关系的基础是欺骗与利用，而非真诚的感情。即使成功“挽回”，对方回头很可能也只是因为性欲再次得不到满足，从而形成“复合-发生关系-再分手”的恶性循环。**通常不建议挽回**。
