# 情感操控与反操控录音：第四课培育：满足弱点与缺失_note

在本节课中，我们将要学习如何针对选定对象的弱点与人格缺失进行培育。我们将探讨如何以特定形象出现，并通过一系列策略让对方逐渐卸下防备，从而为后续操控创造条件。

上一节我们介绍了如何初步筛选目标对象，强调了不应挑战过于强大的目标。本节中我们来看看，在划定范围后，我们应以何种形象出现在对方面前。

这个形象的核心可以是“暖”。暖是指表面上对对方很好，但这与“舔狗”有本质区别。舔狗通过持续付出实际价值以换取青睐，而我们只是营造一种具有迷惑性的表象。因为我们本质是“妖怪”，妖怪的特质是冷血，而“暖”只是用来掩盖这一本质的工具。

说真的，世界上只有两种人会无条件对他人好：一种是收取费用以划清情感边界的专业人士（如心理工作者）；另一种则是亲人或伴侣。除此之外的“好”，大多来自“妖怪”——即不停寻找猎物、无需休息的操控者。因此，当我们踏入此门，就不能再将目标视为平等的人，而是未来的“玩物”。

我们的“玩物”们总会通过各种方式暴露出人格的缺失。例如，他们可能在朋友圈发布寂寞或需要关注的内容。作为猎手，我们需要敏锐地捕捉这些信号。

以下是针对不同缺失信号的行动指南：

*   **需要被关注时**：无论内容如何，第一时间点赞。
*   **需要被关心或善待时**：发送表达关怀的简单信息，比如“比心”。
*   **需要被指导或点拨时**：主动打电话提供倾听或建议。

核心原则是：只要对方流露出一丝“血腥气”（即情绪弱点），就必须第一时间出现。即使只是小事，也要用多种方式（哪怕是“多喝热水”式的关心）让对方从郁闷中笑出来。我们的做法看似像“舔”，但实则不然，因为我们无需付出实质代价，只是在“制造幻觉”，让对方感到安全。

更多时候，我们要做的是抓住一切机会进行赞美。虚荣是人类难以逃脱的原罪。

以下是赞美的关键要点：

*   **方法**：永远顺着对方的意思说话，但赞美需保持一定分寸，不过分谄媚。
*   **心态**：无论话语多么肉麻，都要一本正经地声称是实话实说。不必担心被识破，因为即便对方明白话有虚假，但由此获得的快乐是真实的。
*   **效果**：持续的赞美能让对方感到快乐，从而更愿意向你展示他们的缺失和伤口，这正是你的机会。

做一个好猎手，关键在于更有耐心。反过来想，当我们自己被夸赞时，即便表面谦虚或防备，内心也是开心的。这种正向情绪体验，正是我们提供给对方的“诱饵”。

听到这里，或许有人觉得这样做会很累。但请记住，我们是“妖怪”，捕猎如同呼吸，不会感到疲惫。我们无需顾忌对方感受，也无需考虑对方是否会变得更好，因此我们说的话可以轻松地“像放屁似的往外蹦”。

当然，出于效率考虑，不能只等待一个目标暴露伤口。如果将全部精力集中于一人，看似在培养奴隶，最终结果可能是你反而成了对方的奴隶。

因此，这件事的核心是 **“全面撒网，重点培养”** 。不要有“不控制你就不罢休”的执念，否则反而是你被对方无形控制。我们要时刻保持关注，例如留意朋友圈中那些发布矫情文字和自拍的人，机会往往就在其中。

点开对话框，发送一个不咸不淡的称赞，目的只是刷存在感。如果你的初期判断准确，那些矫情的、缺爱的、缺关注的对象自然会主动跑来求安慰。

试想，一个自我认知不足的人，每天会有多少烦心事？上班被骂、同事甩锅、家庭矛盾、客户刁难、情感纠纷，甚至在网络上与人争执……这些事可谓“血流成河”。当对方向你吐槽时，便是绝佳的培育时机。

以下是针对不同吐槽内容的回应策略：

*   **吐槽领导**：附和“领导就一傻逼”。
*   **吐槽同事**：评价“那就是一蠢货，鼠目寸光”。
*   **吐槽父母**：表示“上一代不理解我们，跟不上新时代”。
*   **吐槽客户**：赞同“有两个臭钱了不起呀，装什么大爷”。
*   **吐槽伴侣**：共情“他真的不懂你”。
*   **吐槽网友**：定性“网友就是键盘侠”。
*   **谈论女权**：支持“女权当然万岁，最看不惯那些屌癌了”。

通过这种方式，你很容易成为对方心中“最懂我的那一个”。记住，问题是别人**主动**说的。在对方还有防备心时，我们只需赞美；若对方不说，我们就装傻。不要主动刨根问底，那才是“舔狗”行为。

在操作中，还需要注意几个关键点：

1.  **频率与范围**：不需要天天出现。错过某个机会无需懊恼。每时每刻出现只会沦为情绪垃圾桶，且效率低下。牢记“星辰大海”，对多个目标进行培育。
2.  **关注反馈比例**：要注意对方不开心时，你安慰成功后对方的回应积极程度。
3.  **关注主动比例**：要注意对方展露不开心时，主动找你倾诉的比例。

第二点和第三点的比例越高，说明这个目标越值得重点培养。因此，请自信一点。如果对方对你的安慰不感冒、夸赞很淡定、对你的“嘴炮”完全不感兴趣，那么离开即可。这类目标并不值得培养。

你无法改变一个已经识破你的人，但可以在那些享受着你的“温暖”的人面前，继续营造“霸道总裁”般的幻觉。抓住这样的人，我们就可以进行下一步的“育成”了。其实，在筛选的过程中，我们的“育成”就已经开始了。

本节课中我们一起学习了如何以“暖”的形象作为伪装，通过捕捉弱点信号、及时回应、持续赞美等策略，在“全面撒网，重点培养”的原则下，逐步培育目标对象的依赖感，并筛选出最易操控的对象，为后续步骤打下基础。
