# 李越《15天提高情商》：6、情商课6_note

在本节课中，我们将要学习如何通过预测自己和他人的情绪与行为，来提升情绪的稳定性和自我觉察能力。上一课我们介绍了通过强化训练和反思来识别情绪产生的节点，本节中我们来看看这样做的重要意义。

## 预测自我情绪的意义

预测自我情绪的核心目的是：**对自己下一次的情绪爆发做出预测**。这意味着你能够提前预判自己会因为哪些事情、产生什么样的情绪。

这件事情很好理解。如果你了解自己身上有哪些“情绪开关”，以及触碰这些开关后会引发何种情绪，你就可以提前设想并提醒自己：“我要小心”。

所以大家可以看到，这件事情可以带动你出现另外一种思维：**预测他人可能产生的行为**。例如，如果你知道自己被忽视后会产生负面情绪，那么在与人相处时，你就可以预测对方可能会忽视你的情况。当他真的忽视你时，因为你已经提前预知，你所产生的情绪反应就会减弱很多。

**公式表示**：
```
情绪反应强度 = 实际刺激强度 - 预知缓冲值
```
因为你已经知道“被忽视这个行为本身就会触发我的情绪”，那么当这个行为发生时，它对你的冲击力就会降低。

## 觉知与觉察：高情商的体现

这种能力叫做**觉知和觉察**。比如，知道自己正在生气，或者知道自己为什么会生气，这都是一个人高素养、高情商的体现。

再比如，你知道自己“被误解”时会产生情绪。那么当对方误解你时，你心中可能会滋生一种想法：“看，你果然误解了我，如我所料”。这时，你就更可能保持心平气和。

情商高的人，其情绪稳定的原因就在于他能够**预测自己的情绪**，并找到自己情绪爆发的关键点。

以下是高情商者在沟通中的一个常见表现：

*   当被误解时，他们会微笑着说：“可能是我没有解释清楚，请让我再说一遍。”
*   而不是愤怒地反驳：“我不是那个意思！你听错了！”

大家可以听出这中间的区别。前者是在管理情绪并引导沟通，后者则是被情绪所控制。

## 案例分析：缺乏预测导致的失误

我们来看一个案例。一位健身教练遇到了两位客户时间重叠的情况。她对一位老客户说：“有个新客人，我要照顾一下。” 这位老客户感到不舒服。

为什么说这位教练情商不高呢？除了没有安抚客户情绪外，**最关键的是她自己的情绪在那一刻根本不稳定**。时间冲突让她感到慌乱，而另一位客户的临时爽约也让她心里不舒服。她急于解决问题、渴望表现自己，因此说出了那样的话。

**她根本没有对自己可能产生的情绪做任何预测或预判**，也不会去预测自己的行为可能引发客户什么样的情绪。

作为一个教练，你应该预判到客户可能会爽约、时间可能会冲突。这应该是你职业准备的一部分。特别是新手，更应该培养这种**提前预测和准备的能力**。经验正是从这些地方积累而来的：经历的事情多了，你就知道该说什么话，该以什么样的平稳状态去沟通。

## 预测他人行为：以最坏情况做准备

觉知自己的情绪、预测自己的情绪爆发，并**预测他人可能产生的行为**，是一件非常重要的事情。

这里有一个重点：预测他人行为时，**应该预测最坏的行为**。

我们经常说，不要把很多事情想得太美好，永远要以最坏的角度去思考对方的言谈举止。为什么要这么做？

因为当你设想了最坏的情况（对方会如何言行），再结合你对自己的了解（自己会出现什么样的情绪波动），你就在心中完成了一次“预演”。

**代码类比**：
```python
# 在心中预演情绪反应
def predict_reaction(worst_case_scenario, self_knowledge):
    emotional_response = simulate_emotion(worst_case_scenario, self_knowledge)
    if emotional_response.is_negative():
        adjust_mental_preparation() # 提前进行心理建设
        modify_communication_plan() # 调整沟通方案
    return prepared_state
```
这就相当于你在心里运行了一遍程序，预览了可能出现的“Bug”（情绪问题）。既然知道了可能出现的问题，你就可以提前修改“程序”（调整心态和策略），给自己做好心理建设。

当事情真的发生时，整个情况就可能得到改善，甚至避免负面情绪的出现。

## 心理建设与情绪稳定

一旦我们知道了自己在什么情况下会出现什么情绪，那么下次再遇到相同或相似问题时，当预判中的情况真的出现，我们就已经做好了准备。

我们再举个例子。如果你预判到“我这么做，你肯定会骂我”，那么当你真的被骂时，虽然可能仍会不舒服，但情绪不会像毫无准备时那样失控。因为你已经知道“你骂我”这个行为会发生。

很多人在社交媒体发内容时，会预判到可能招致批评。他们提前知道了这一点，所以当批评真的来临时，他们会觉得“这很正常”，而不会因此产生剧烈的情绪波动。

所以，各位以后在生活与人交流时，无论是与伴侣、同事还是家人相处，都要预判可能面对的情况：对方可能忽视你、误解你、打压你、嘲笑你、拒绝你、欺骗你等等。人际交往中这些行为很常见。

如果你能预测这些行为，并知道它们会引发你何种情绪，那么当事情真的发生时，你就可以泰然处之。

## 能力养成：经验积累与持续练习

听到这里，大家可能会觉得这个方法听起来简单。但问题来了：**这是一学就会的事情吗？**

听起来简单，但事实上不是。它是一个**经验积累的过程**。我们需要通过思考、实践和总结，在每一次情绪爆发后去**复盘**，才能逐渐掌握。

很多人一生都在为同样的事情反复生气、争吵，就是因为他们从未在情绪爆发后复盘、思考过自己人生的各种“情绪节点”。

情绪的稳定是需要训练的。这种自我觉察、保持情绪稳定、预测情绪发展、预测他人行为的能力，我们统称为**情商能力**。

既然是一种能力，大家就一定要明白：**能力是在日积月累的学习、实践、失败和总结中获得的**。很多人希望听一次课就变成天才，瞬间成为非常厉害的人，这是一种不切实际的功利想法。成长是一个过程。

## 课程设计与学习心态

这也是为什么我们把“情商”这样需要细致讲解的内容，单独做成线上课程。在线下大课中，时间有限，我们更侧重于搭建核心知识框架。而像情商、聊天等细节内容，则通过线上课作为补充，让大家可以更系统、深入地学习。

所以，我们刚才所讲的整个内容，需要你在人生经历中不断学习、思考，在每次情绪爆发后去反思。

回到这堂课的开始，常有人问：“我总是控制不住生气怎么办？” 那么，你有没有思考过：**你为什么会生气？是什么具体节点或原因导致了生气？**

很多人只会问“怎么办”，却不会问“为什么”。如果你不问自己“为什么”，别人也无法帮你，因为我们不知道你生气的根源。

如何觉知情绪？核心不是觉知“我在生气”这个表象，而是**预测他人的行为，以及预测自己可能产生的情绪和行为**。

这件事无法一蹴而就。它需要我们在听完课程后，在接下来的人生中，不断预设和预想各种情境下可能出现的问题，进行思维锻炼。只有这样，你才能很好地做到另一件事：**觉知他人的情绪**。

## 从知己到知人：情绪系统的共通性

如果你无法觉知自己的情绪，就不可能觉知别人的情绪。整个世界的人都共用一套相似的情绪系统。

如果你都无法觉察自己因何行为产生何种情绪，就很难觉察到别人的情绪变化。而觉察不了别人的情绪，之后就很难去影响他人。

**逻辑链**：
```
预测自我情绪 -> 理解情绪触发机制 -> 预测他人可能行为 -> 预测他人情绪反应 -> 提前干预或调整沟通 -> 影响他人，避免冲突
```
如果你能预判对方接下来会生气，那么你在沟通时就可以采取策略，哪怕他生气了，你说几句他也可能很快平复。这就是高情商的沟通。
