# 李越《15天提高情商》课程七：觉知他人情绪 👁️🗨️

在本节课中，我们将学习如何觉知他人的情绪。这是继上一节“觉知自我情绪”之后的重要进阶。理解他人的情绪状态，是提升社交情商、进行有效沟通的关键一步。

上一节我们介绍了如何觉知自己的情绪，本节中我们来看看如何将这种觉知能力扩展到他人身上。首先，我们需要理解人在社交互动中的几种基本状态。

## 人的三种社交状态

根据在社交互动中的表现倾向，我们可以将人分为三种状态。

以下是这三种状态的详细说明：

*   **表现者**：这类人热衷于表达自己的欲望和情绪，极易被自身情绪主导。他们渴望成为焦点，获取他人的夸奖、赞美、认可。其行为多依靠本能或习惯，**缺乏对自身情绪和需求的觉知能力**。在交流中，他们只顾自我表达，常常让对话方感到不适。
*   **观察者**：这类人相对沉默，善于观察。他们对自己的情绪、欲望和需求有较强的觉知力，明确自己的目的。他们心中装着别人，明白**只有让对方感到舒适，才能获得认可与价值交换**。他们通过观察他人的言行、表情、语气来分析对方的欲望与需求，从而决定自己该如何沟通。
*   **混沌者**：这类人既不热衷于表现，也不善于观察。他们在社交中缺乏明确目标和存在感，行为混沌，可能只是被动地跟随环境。他们需要培养观察意识。

理解这三种状态后，我们需要明确，要提升情商，应努力向“观察者”靠拢，避免成为只关注自我的“表现者”。

## 如何成为观察者：理解情绪产生的链条

要有效观察他人，我们需要理解他人情绪产生的根本原因。情绪并非凭空而来，它遵循一个清晰的链条。

这个链条的逻辑关系如下：

1.  **需求**：人内心深处的渴望（如被关心、被认可、追求快乐）。
2.  **欲望**：需求在具体情境下催生的、想要达成的具体目标。
3.  **行为**：为了满足欲望而采取的具体行动。
4.  **情绪**：当行为与外界环境（他人反应、事件结果）发生碰撞时，内心产生的感受。

**公式：需求 -> 欲望 -> 行为 -> 情绪**

例如，一个人有“被重视”的需求（需求），在会议中产生了“想要发言被认可”的念头（欲望），于是他积极发言（行为）。如果发言被忽视或打断（外界碰撞），他就会产生失落或愤怒的情绪（情绪）。

## 观察的具体方法：由己及人

人是共通的。我们自身情绪产生时的外在表现，在很大程度上也适用于他人。因此，觉知他人情绪的前提，是深刻了解自己的情绪反应模式。

我们可以通过反思自身在不同情绪下的表现来建立参照：

*   回忆自己在尴尬、紧张、愤怒或开心时，会有怎样的**面部表情**、**肢体动作**（如挠头、低头）和**说话语气**。
*   注意自己在被忽视、误解或拒绝时，内心涌起的是哪种情绪，以及它如何体现在外在。

建立了自我认知的参照系后，观察他人就变得有迹可循。当你看到他人出现类似的外在表现时，便可以推断其可能处于相应的情绪状态。

观察的核心是关注以下几点：

*   **表情与眼神**：这是情绪最直接的窗口。
*   **语气与音调**：声音的高低、轻重、节奏都承载着情绪信息。一个情商高的人，善于运用语气和重音来精准传达重点。
*   **肢体语言**：不经意的动作往往透露真实状态。

**关键点**：一个无法觉知自身情绪变化的人，也无法真正觉知他人的情绪变化。

## 实践练习与注意事项

理论需要结合实践。在日常生活中，我们可以有意识地进行观察训练。

以下是一些实用的练习建议：

*   **专注观察**：在与他人交流时，有意识地将注意力从“自己接下来要说什么”转移到“对方现在的状态是什么”上。认真看对方的眼睛和表情，倾听对方语气中的细微变化。
*   **直播学习**：观看人际互动类直播（如连麦环节）时，不要只听内容。同时观察双方的反应：主播在什么情况下会表现出认真或不耐烦？连麦者在不同回应下，情绪和状态有何变化？思考这背后的原因。
*   **反思对照**：遇到让你感觉沟通不畅或不舒服的互动时，事后复盘。对方当时有哪些情绪表现（如眼神飘忽、语气急促）？这些表现可能对应什么情绪（紧张、不耐烦）？如果当时你觉察到了，可以如何调整沟通方式？

同时，需要注意避免走入另一个极端：**过度敏感和过度分析**。观察是为了更好地理解和连接，而不是为了妄加揣测。避免仅凭一两个外在细节就草率给人下结论。对人性本质有更深的了解，能帮助你更准确、更轻松地进行分析。

**本节课总结**：我们一起学习了情商提升的第七课——觉知他人情绪。我们首先了解了人在社交中的三种状态（表现者、观察者、混沌者），并明确了应向观察者学习。接着，我们剖析了情绪产生的链条：**需求 -> 欲望 -> 行为 -> 情绪**。最后，我们掌握了由己及人的观察方法，即通过深刻认知自身情绪反应来理解他人，并通过专注观察表情、语气、肢体语言来实践。记住，有效观察是建立良好沟通、提升社交情商的重要基石。
