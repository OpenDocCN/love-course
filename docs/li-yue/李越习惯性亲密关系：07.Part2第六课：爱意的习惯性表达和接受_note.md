# 李越习惯性亲密关系：07.Part2第六课：爱意的习惯性表达和接受 💕

在本节课中，我们将要学习“爱意”的习惯性表达与接受方式。我们将探讨一个人从小如何被教导表达爱，以及这如何塑造他未来在亲密关系中理解和接收爱意的方式。

上一节我们介绍了安全感的习惯性表达和接受。本节中我们来看看爱意是如何以类似的方式被塑造和传递的。

## 爱意表达的来源：掌控与鼓励

在四种相处模式中，期望和打压能给人带来安全感。同样，掌控和鼓励也能带来安全感，但这里需要特别注意。

父母在实施掌控或给予鼓励时，常常会以“爱”的名义进行。他们最常说的话是“一切都是为你好”，特别是在为你安排事务、要求你服从时。这整体上都是他们表达爱意的一种方式。

以下是两种核心的爱意表达方式：

1.  **掌控即爱意**：父母掌控孩子，是出于害怕孩子独自面对残酷社会和可能遭遇的困难、挫折甚至危险。他们的目的是爱护和保护孩子，希望孩子能更轻松地过上美好生活。孩子骨子里明白这是爱，尽管有时会反抗。公式可以理解为：**父母的掌控 = 出于爱护的保护行为**。
2.  **鼓励即爱意**：这里指的鼓励，并非纯粹的正面鼓励，而是当孩子符合父母期望、服从掌控或接受打压时获得的鼓励。对孩子而言，他们不会区分鼓励的正反面，只要获得父母正面的、温柔的认可，就会感到开心，并认为这是爱的表现。即：**父母的（条件性）鼓励 → 孩子感知为爱**。

当一个人从小生活在这样的环境中，他所接受的爱意就是通过被人掌控和接收（往往是附条件的）鼓励来传递的。这可能导致他对“正确的”爱意表达方式产生曲解。

## 习惯的冲突：当“正确”遇上“习惯”

如果一个人用心理学层面上心智成熟、人格健全的方式去表达爱意，而对方习惯的却是上述那种“掌控-鼓励”模式，那么对方很可能无法接受。

因为对方不认为这是一种表达爱的方式。

我们可以通过一个案例来理解：

一位女性学习了许多关于建立健康亲密关系的知识，旨在追求双方心智成熟与人格健全。当她与缺乏安全感的男友出现矛盾时（例如因忙碌回复消息慢），她选择用成熟的方式沟通：分析对方缺乏安全感的原因，试图讲道理帮助对方成长。

结果：男友不仅没听进去，反而感到被指责，两人争吵起来。

后来给这位女性的建议是换一种方式：用带有“掌控”和“打骂”意味，但内核是亲密和在乎的话语去沟通。例如：“老子不爱你？我累的时候都在想你会不会难过！你有时间吵架不如来找我！” 或者当男友问“在干嘛”时，回复：“是不是又闲着想我了？大男人天天不务正业就知道聊天，快去干活！”

效果：男友听后反而感到开心和舒服。

为什么这种方式有效？因为对这位男友而言，“打骂”和“掌控”对他而言就是爱，就是安全感。他习惯并需要这种表达方式。

## 社会现状与我们的选择

这揭示了一个社会现状：许多人唯一熟悉的相处模式就是这种“习惯性亲密关系”模式。如果你跳出来，用一种完全不同的、“正确”的方式去表达，反而可能被视作异类。

但这里有一个至关重要的区别：**结果导向**。

以下是两种不同的导向：

*   **导向A（利他）**：使用对方熟悉的表达方式（如案例中的“打骂式关心”）进行沟通和教导，目的是为了让对方变得更好，让关系变得更好。本质仍是爱意的表达，只是路径不同。
*   **导向B（利己）**：使用这种模式是为了操纵对方，让自己在关系中获利，让自己更爽。这本质是情感操控和道德绑架。

**我们的主张是导向A**。我们走一条对方熟悉的“小路”，但目的地和走“大路”（健康独立的关系）一样，都是为了共同的成长与幸福。如果你的发心是为了控制与索取，那关系注定不会好。

## 自我觉察：警惕习惯带来的盲区

这种习惯性影响会造成我们对安全感和爱意的接受与表达方式变得单一。

你需要警惕：将来若遇到一个真正精神独立、人格成熟、尊重你所有选择的伴侣，你可能会感到极度缺乏安全感。因为对方不干涉、不掌控、不“打骂”，你会觉得他不需要你、不在意你，甚至不相信他的爱。

这就像有些情感丰富的女性，无法与那些情感表露较少但内心深爱、给予充分尊重和自由的事业型伴侣相处，反而需要一个人天天“管着”自己，因为后者才让她感觉是“爱”。

本节课中我们一起学习了爱意的习惯性表达与接受。我们了解到，一个人从小接受的“爱”的表达方式（常通过掌控和条件性鼓励实现），会深刻影响他未来如何理解爱、表达爱和接收爱。当两种不同的“爱语”相遇时，会产生冲突。关键在于，我们使用对方熟悉的“爱语”时，其出发点应是引导关系向更好、更健康的方向发展，而非利用它进行操控。最后，我们需要自我觉察，警惕这种习惯可能让我们错失真正健康的爱。
