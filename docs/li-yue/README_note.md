# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是**赋值**，公式如下：
`变量名 = 值`

例如，我们写 `name = “李越”`，就意味着创建了一个名为 `name` 的“盒子”，并在里面存放了“李越”这个数据。之后在程序中，我们使用 `name` 这个变量名，就等同于在使用“李越”这个值。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的种类，这就是数据类型。Python有几种内置的基本数据类型，它们决定了数据可以进行的操作。

以下是三种最常用的基本数据类型：

*   **整数（int）**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数（float）**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**：表示文本信息，需要用单引号（`‘’`）或双引号（`“”`）括起来，如 `‘Hello’`, `“Python”`, `‘123’`（注意：这里的123是文本，不能直接进行数学计算）。

## 如何查看与转换类型？ 🔄

在编程时，我们经常需要知道一个变量的类型，或者在不同类型间进行转换。

我们可以使用 `type()` 函数来查看任何变量或值的数据类型。代码如下：
```python
my_number = 100
print(type(my_number)) # 输出：<class ‘int’>
```

有时，我们需要将一种类型转换为另一种类型。以下是常见的类型转换函数：

*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

例如，将字符串“123”转换为整数进行计算：
```python
num_str = “123”
num_int = int(num_str) # 现在 num_int 是整数 123
result = num_int + 7 # 可以进行数学运算，结果为130
```

## 动手试一试 ✍️

理论学习之后，最好的巩固方式就是实践。请在您的Python环境中尝试运行以下代码，并观察结果。

```python
# 1. 创建变量
age = 25
price = 19.99
greeting = “大家好，我是李越”

# 2. 打印变量及其类型
print(“年龄：”, age, “类型：”, type(age))
print(“价格：”, price, “类型：”, type(price))
print(“问候语：”, greeting, “类型：”, type(greeting))

# 3. 尝试类型转换
new_age = str(age) # 将整数25转换为字符串“25”
print(“转换后的年龄：”, new_age, “类型：”, type(new_age))
# 尝试将字符串和数字拼接
message = “我今年” + new_age + “岁。”
print(message)
```

本节课中我们一起学习了Python的变量与数据类型。我们知道了变量是存储数据的容器，并通过赋值语句`变量名 = 值`来使用它。我们还认识了整数（int）、浮点数（float）、字符串（str）这三种基本数据类型，学会了用`type()`函数查看类型，以及用`int()`, `float()`, `str()`进行类型转换。掌握这些是开启Python编程之旅的坚实第一步。
