# 李越习惯性亲密关系：17.Part5第十六课：习惯性亲密关系的危害——习得性

在本节课中，我们将要学习习惯性亲密关系中的第二种危害方式：**习得性无助**。我们将通过一个经典的心理学实验来理解这个概念，并探讨它在亲密关系中的具体表现与严重后果。

## 概述：什么是习得性无助？

习得性无助是一个心理学概念，描述了个体在经历反复的失败或挫折后，形成的一种“无论怎样努力都无法改变结果”的信念，从而放弃尝试的心理状态。

## 塞利格曼的实验

为了理解习得性无助的起源，我们首先来看心理学家马丁·塞利格曼在1975年进行的一项著名实验。

塞利格曼找来一批大学生，并将他们分为三组。

以下是实验的具体设置：

*   **第一组（无助组）**：被安排在一个房间内，持续遭受无法停止的噪音干扰。无论他们做什么，噪音都不会停止。
*   **第二组（可控组）**：同样被安排听噪音，但他们可以通过完成一些小任务（如拼图、解迷宫）来让噪音停止。
*   **第三组（对照组）**：不接触任何噪音，也不进行任何特殊任务。

紧接着，所有参与者被带出房间，立即进行第二个实验——“手指穿梭箱”测试。

这个测试的规则是：参与者将手伸进一个箱子，一开始会触发噪音。只有当手摸索到箱子另一侧的按钮并按下时，噪音才会停止。

## 令人震惊的实验结果

实验结果显示了一个关键差异。

第二组和第三组的参与者都很快学会了将手伸到另一侧按按钮来停止噪音。**唯独第一组的参与者**，他们大多只是将手放在箱子外，忍受着噪音，不再尝试寻找停止噪音的方法。

这个结果说明，第一组参与者由于在之前的经历中，**尝试了各种方法都无法让噪音停止**，他们“习得”了一种信念：自己的行为是无效的。这种信念就是**习得性无助**。

其核心公式可以概括为：
**反复的失败经历 → “行为无效”的认知 → 放弃努力 = 习得性无助**

## 习得性无助的危害与表现

上一节我们通过实验了解了习得性无助的形成机制，本节中我们来看看它在现实生活中，尤其是在亲密关系中的具体危害。

美国国家阅读委员会的报告指出，拥有习得性无助心理状态的人常表现出以下特征：

*   **懒散与怠慢**：缺乏行动力，对事物提不起兴趣。
*   **轻易放弃**：面对困难任务时，很快选择放弃。例如，学生可能不完成作业。
*   **焦虑与抑郁**：在面对考试或生活压力时，容易变得焦虑。现代心理学已将习得性无助视为**抑郁症的重要先兆**。

当一个人认为“无论怎么做都无法改变现状，人生没有希望”时，他就陷入了习得性无助，这离抑郁症可能只有一步之遥。

## 习得性无助在亲密关系中的产生

为什么在习惯性亲密关系中容易让对方产生习得性无助？

因为这种关系模式很可能在无意识地重复对方早年与父母的不良互动模式。即使你的初衷是带领对方走向健康关系，但若方式不当，对方会感到：

“我怎么又在重复推那块石头（指西西弗斯的神话）？”
“我和你的相处，怎么和跟我父母的相处一模一样？”
“是不是所有人都会这样对我？我永远逃不出这个怪圈？”

当对方产生“**这就是我的命运**”的想法时，无论你如何努力想修复关系，对方都可能拒绝改变，认为一切尝试都是徒劳。这段关系实质上已经“死亡”，变得死气沉沉。

## 关系中的具体雷区与后果

在习惯性亲密关系的操作中，有些行为特别容易将伴侣推向习得性无助的深渊。

例如，在争吵后，如果只有指责和愤怒，而没有在对方做对时给予鼓励和奖赏，没有在事后通过拥抱等方式表达爱意，对方就会认为“争吵没有意义，但我又无法避免争吵”。这会导致对方进入一种**麻木**状态，但这不是简单的麻木。

以下是习得性无助与单纯麻木的关键区别：

*   **习得性无助**：更像抑郁状态。**不是不想要，而是害怕没有回报**，所以不敢尝试。
*   **单纯的麻木**：更像一种“僵尸化”的隔离状态（我们将在下一章节详细讨论）。

习得性无助的持续发展可能导致抑郁、焦虑等身心疾病。与一位处于病态的伴侣相处，会对关系造成巨大的伤害和内耗，最终反噬关系本身，让双方都痛苦不堪。

## 如何避免造成伴侣的习得性无助？

既然习得性无助危害巨大，那么在使用习惯性亲密关系模式时，我们必须格外小心。

关键在于：**赏罚分明，并为冲突画上句号**。

*   该指责时可以指责。
*   但指责之后，必须通过行动（尤其是表达爱意）来修复关系。
*   要让对方明白，所有的争吵、磨合都指向一个共同的目标：**爱，以及共同的未来**。

这样，对方“习得”的就不是无助，而是“**我们在为长期关系进行磨合**”的健康认知。这就像为你们的互动安装了一个安全阀，公式如下：
**冲突/指责 + 修复/表达爱意 = 健康的磨合认知**
反之，如果缺少后半部分，公式就会变成：
**冲突/指责 + 0 = 习得性无助**

## 总结

本节课中，我们一起学习了习惯性亲密关系的核心危害之一——**习得性无助**。

我们从塞利格曼的实验出发，理解了它是如何因“反复失败”而形成的“行为无效”信念。我们探讨了它在亲密关系中的表现：让对方感到绝望、放弃努力，并可能引发抑郁。最后，我们强调了避免这一危害的关键：在关系中做到赏罚分明，并为所有冲突赋予积极的意义和温暖的结尾，引导关系走向健康的磨合，而非绝望的无助。

理解这一点，对于维护一段有生命力、共同成长的亲密关系至关重要。
