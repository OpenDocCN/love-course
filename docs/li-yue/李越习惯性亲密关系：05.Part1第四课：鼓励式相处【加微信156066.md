# 李越习惯性亲密关系：05.Part1第四课：鼓励式相处【加微信156066

在本节课中，我们将要学习习惯性亲密关系中的第四种核心相处模式：鼓励式相处。我们将深入探讨这种模式如何通过正向反馈，深刻地塑造一个人的行为与自我认知。

在上一节我们介绍了期望式、打压式和掌控式相处，本节中我们来看看鼓励式相处。这种模式常被忽略，但它贯穿于整个关系系统，是我们后期需要重点学习的一种方式。

很多人认为自己从小并未受到太多鼓励，这其实是一种错误认知。你实际上受到了很多鼓励，只是这种鼓励与你认知中“积极阳光”的鼓励形式不同。本节课的内容可能会颠覆你对“鼓励”的固有认知。

同样，我们将鼓励式相处分为三个部分来解析，以结合前三课的知识。

以下是鼓励式相处的第一个核心部分：**鼓励符合期望的行为**。

*   **定义**：当你的行为符合他人（如父母）的期望时，你会获得正向的鼓励、夸赞或奖赏。
*   **过程**：例如，父母期望你主动做家务。某天你按要求做了（即使是勉强做的），他们便鼓励你、夸赞你，甚至给予物质奖励。你因此感到“做对了”，并获得了正向回馈。
*   **结果**：这种正向鼓励会加深你对“符合期望”的认知。为了持续获得认同，你会不由自主地、越来越努力地去完成他们的期望，并朝他们期望的形象靠拢。

**公式表示**：`符合期望的行为 + 正向鼓励 → 强化该行为 → 逐渐成为期望中的人`

上一部分我们看到了鼓励如何强化符合期望的行为，接下来我们看看鼓励式相处的第二个部分：**鼓励接受打压**。

*   **定义**：当你接受了他人的打压，并因此否认掉自己的行为、想法或情绪后，你获得了正向鼓励。
*   **过程**：例如，你想吃糖但总被父母打压（责骂）。后来你学“聪明”了，不再表达想吃糖的欲望和行为。当你表现出“接受打压”（如拒绝父母给的糖）时，反而得到了“真乖”之类的表扬。
*   **结果**：你由此认为，压抑自己的欲望、想法和情绪才是“正确”的，因为这样做能获得认同和力量。这会加剧你接受打压的习惯，形成“只要我听话、隐藏自我，就能得到正向反馈”的认知。

**核心逻辑**：`接受打压 + 隐藏自我 + 获得正向鼓励 → 强化“接受打压”的行为模式`

理解了鼓励如何作用于接受打压后，我们来看鼓励式相处的最后一个部分：**鼓励服从掌控**。

*   **定义**：当你服从他人对你生活、人生抉择的掌控，且毫无怨言时，你获得了正向鼓励。
*   **过程**：例如，父母为你安排学业、生活甚至对象。你开始有自己的想法，但表达出来会遭遇打压。当你选择完全服从他们的安排，并表现得欣然接受时，他们夸你“听话”、“懂事”。
*   **结果**：这种鼓励让你的服从性变得越来越强。最终，你可能形成一种综合模式：努力成为对方期望的人、隐藏自我的情绪与想法、对安排言听计从。

**行为闭环**：`服从掌控 → 获得认可 → 强化服从性 → 更彻底地放弃自主抉择`

需要再次强调，我们并非评判这些方式的对错，而是客观分析这种**习惯性相处模式**的形态与成因。这种模式的核心在于，通过将**鼓励（作为爱意与安全感的象征）** 与特定行为（符合期望、接受打压、服从掌控）绑定，从而训练个体形成固定的思维和行为习惯。

一个人从小习惯于此种模式后，便会将其视为亲密关系的“常态”或“正确模板”。了解这一点，有助于我们理解他人，并在未来关系中，能更清醒地识别模式、引导关系走向更健康的状态。

本节课中我们一起学习了鼓励式相处的三种具体形式：鼓励符合期望的行为、鼓励接受打压以及鼓励服从掌控。我们看到了鼓励如何作为一种强大的正向反馈工具，在无形中塑造个体的行为习惯和自我认知。这是习惯性亲密关系第一部分的最后一课，接下来我们将进入第二部分，探讨这种关系模式带来的深层影响。
