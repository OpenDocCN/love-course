# 李越《15天提高情商》：第11课 🧠 如何影响他人的情绪

在本节课中，我们将要学习如何影响他人的情绪。我们将从避免带来负面情绪和主动创造正面情绪两个核心方面展开，帮助你理解人际互动中的情绪本质，并掌握实用的方法。

## 概述：情绪影响的两面性

影响他人情绪可分为两个方向：避免带来负面情绪，以及主动创造正面情绪。上一节我们介绍了如何避免自身情绪影响他人，本节中我们来看看如何主动为他人带来积极的情绪体验。

## 第一部分：避免带来负面情绪

我们已经知道，避免给对方带来坏情绪的关键在于换位思考、理解对方，并从客观事实层面解决问题。这些方法能有效防止自身产生负面情绪，同时避免被对方的负面情绪影响。

当我们做到这些前提，就能确保对方不受到我们负面情绪的波及。因为我们的出发点和思考层面都不存在损害双方关系的问题。

以下是实现这一点的核心方法：

*   **高维度看待生活**：不纠结于眼前小事。
*   **进行换位思考**：站在对方立场理解其感受。
*   **认清客观事实**：识别并着手解决真实存在的问题。

这样做的人往往显得情商很高，因为他们能超越琐事，支持对方，从而避免对方产生负面情绪。

## 过渡：关于“底线”的思考

有人曾问，这样做是否会显得没有底线。提出这个问题时，答案其实已经隐含其中。

如果你的目标是“必须和某个人在一起”，那么“在一起”本身就是你的最大收益。为了实现这个目标，你所做的其他妥协就是代价。当你无法接受“不在一起”的结果时，就必然会承受其他后果，例如双方都不开心，并且你可能因此不断退让底线。

**核心公式**：`目标 = 个体` 时，`达成目标的代价 = 其他方面的牺牲`

我们的目标不应是“与某个具体的人绑定”。举个例子，程序员抱怨996，但不愿辞职，是因为程序员工资高。他们为高工资所承担的代价，就是所抱怨的内容。想清楚这一层，就不会因此生气，也不会给他人带来负面情绪。

## 第二部分：创造正向情绪 🎯

本节课的核心内容是：如何给对方带来正向情绪。

### 1. 确立共同目标

上节课提到，拥有共同目标是关键。共同目标能让双方受益，按照这个目标前进，双方之间就不易产生负面情绪，反而会获得正向反馈。

**核心要点**：所谓的共同目标，必须让双方都受益。即使是最简单的恋爱关系，也应让两个人都比独处时更舒服。一段不能让双方都变得更好的关系，就没有真正的共同目标。

然而，世界上几乎不存在让所有人受益且无人受损的事情。就像吹空调，有人享受凉爽，就有人要承受空调外机排出的热风。

**核心隐喻**：
*   **受益** = 在室内吹空调
*   **受损** = 在室外吹空调外机

任何共同目标都意味着有人受益，有人（可能是未知的第三方）受损。因此，建立长期关系就像两人结盟，共同应对外部“敌人”（即让别人去“吹空调外机”）。

### 2. 划分“自己人”与“外人”

问题的本质在于利益分配。人们根据利益分配来定义“自己人”和“外人”。自己人共享收益，外人承担损失。

**核心原则**：若想让某人产生正面情绪、喜欢你，就要将对方与你视为一个整体。你们独立于外界，所有事情为这个整体服务。只要整体不受损，让他人受损有时是必然的代价。

“烂好人”很惨，是因为他企图讨好所有人。当周围所有人都在“吹空调”时，只剩下他自己来“吹空调外机”。想在所有社交中都处理好，本质上就会很吃亏。

我们需要分清谁更重要。对重要的人，我们可以吃点亏，大家一起“吹空调”；对不重要的人，则可以将“空调外机”对准他。

### 3. 应用实例：彩礼问题

以彩礼问题为例。争吵往往源于立场划分：你和父母是“自己人”（一起吹空调），男友和他的父母成了“外人”（吹空调外机）；反之亦然。双方都想让对方“吹空调外机”，矛盾必然产生。

真正的解决方案是：如果决定结婚，就应把你和伴侣视为一个整体（一起吹空调），共同面对双方父母可能的不满（让他们吹空调外机），并在中间调和。理想情况下，是你们六人成为一个整体，让社会上的其他人去承担代价。

### 4. 快速拉近关系的方法：寻找共同“敌人”

回想令人开心的聚会，和谐时刻往往源于大家有共同话题或共同吐槽的对象，尤其是集体吐槽某个人或某件事。

**核心方法**：与他人相处时，可以观察并寻找双方共同讨厌的事物。利用共同吐槽，能快速拉近关系，形成阵营。这本质上是将“空调外机”对准了阵营外的靶子。

即使是感觉“三观特别合”，也常常是因为找到了共同讨厌的东西。共同吐槽比共同喜欢更能快速拉近关系。

网络上的各种立场对立（如男女对立、种族对立），本质也是通过树立共同敌人，来增强群体内部的凝聚力。早恋被管得越严，情侣关系往往越紧密，也是因为外部压力让他们成为了共同对抗“敌人”的盟友。

## 过渡：从启示到方法

以上讨论给了我们一个启示和两个方法。

**核心启示**：人与人关系的本质是利益分配。人们根据利益分配定义自己人与外人，自己人享有收益，外人承担损失。这属于系统内的 **帕累托改进**。

以下是两个具体方法：

### 方法一：利用共同吐槽拉近关系

正如上文所述，找到共同讨厌的点并一起吐槽，是快速建立阵营、拉近关系的有效手段。

### 方法二：合理“甩锅”，转移矛盾

当双方出现矛盾时，可以学会将责任推给外界，尽量不影响彼此关系。

**公式**：`矛盾原因 = 外界客观因素`，`而非我们任何一方的责任`

例如，“我的原生家庭有问题”、“我缺爱”、“我缺乏安全感”，甚至“女人都是不讲道理的”这类公认的“鸡汤”，都可以成为帮你转移矛盾的“锅”。

在沟通中，可以使用这样的话术结构：
1.  **陈述客观问题**：因为客观上存在XX问题（如原生家庭）...
2.  **说明对自己的影响**：导致我现在出现了XX情况（如缺乏安全感）...
3.  **主动承担责任**：这件事当然跟你没关系...
4.  **表达理解与期望**：我现在明白了...，只是偶尔还会希望...
5.  **提出微小请求**：如果你能...（一个小举动），我就会很开心。

**示例话术**：“因为我小时候父母很少陪我...所以我特别希望有人陪伴。当然这跟你无关，我现在也明白每个人都有自己的事。只是偶尔还会幻想，如果你能在我需要时陪我一会儿，哪怕改天也行，我一定会很开心。”

这种表达包含三层意思：
1.  矛盾由外界造成，与感情本身无关。
2.  我正在努力解决自己的问题。
3.  如果你能偶尔帮助，我会很开心。

这样既没有指责对方，也没有自我贬低，而是将“空调外机”对准了外界因素。

**重要提醒**：这些话术可以说给别人听，但自己不要深信不疑，更不要以此为由停止自我成长。关键还是要积极解决自身问题。

## 第三部分：通过管理预期调节情绪 📊

### 1. 情绪产生的原理

情绪源于欲望/需求驱动的行为，与外界互动后产生的结果与预期之间的落差。
*   `结果 < 预期` -> 负面情绪
*   `结果 >= 预期` -> 正面情绪

### 2. 如何让对方开心？

本质上，通过影响对方对结果的预期，或改变实际结果，就能调节其情绪。
*   **满足或超越预期**：带来开心。
*   **降低预期**：即使结果一般，也可能因为符合低预期而不产生负面情绪。

**例子**：为什么男朋友骗你你会生气，而陌生人骗你你不会？因为你对男朋友有预期，对陌生人没有。欺骗本身不伤人，**预期落空**才伤人。

### 3. 方法论：降低对方预期

当无法完全满足对方需求时，合理降低对方对你的预期是关键。

**核心操作**：`主观否定` + `客观肯定`
*   你客观上好看/有钱/地位高。
*   但你主观上表示“我觉得自己不好看/经济一般/很普通”。

这样，对方原本“她肯定很难追/要求高”的高预期就被降低了，觉得“有机会了”。

**软实力方面**：如果你自知有些缺点（如“作”），可以在关系早期通过分享“原生家庭”等原因（用上述话术）主动降低对方预期。对方接受后，对你之后偶尔的“作”就有了心理准备，不易产生负面情绪。如果你做得比预期好，反而会成为**附加价值**，让对方惊喜。

**生活案例**：
*   男朋友总哭穷，当他真的没买礼物时，你更容易接受。
*   男朋友工作确实忙，当他偶尔回复不及时，你会觉得正常。
*   反之，一个很闲的人陪你，你会觉得理所当然。

### 4. 重要前提：自身硬实力

降低预期策略有效的前提是：**你自身拥有足够的硬实力或综合价值**。

*   `你好看 + 你说自己不好看` -> 对方觉得你谦虚，有机会。
*   `你丑 + 你说自己丑` -> 对方预期是降低了，但对你也没兴趣了。
*   `你有钱 + 你说自己一般` -> 对方觉得你认知不到优点，有机会。
*   `你穷 + 你说自己穷` -> 对方觉得你就是穷，可能看低你。

因此，这是基于你综合评分不错时，让对方权衡利弊后，愿意忽略你声称的“缺点”的策略。

## 总结 🎓

本节课我们一起学习了如何影响他人情绪：
1.  **避免负面情绪**：通过换位思考、高维视角和解决客观问题，防止自身情绪波及他人。
2.  **创造正向情绪**：
    *   **核心理念**：将对方纳入“自己人”阵营，共同面对外部，实现利益共享。
    *   **核心方法**：
        *   **寻找共同“敌人”**：通过共同吐槽快速拉近关系。
        *   **合理“甩锅”**：将矛盾归因于外界客观因素，使用结构化话术进行沟通。
        *   **管理预期**：通过“主观否定客观肯定”等方式降低对方预期，从而调节其情绪体验，前提是自身具备足够硬实力。

理解人际关系的利益分配本质，并灵活运用这些方法，你就能更有效地影响他人的情绪，构建更和谐、积极的人际关系。
