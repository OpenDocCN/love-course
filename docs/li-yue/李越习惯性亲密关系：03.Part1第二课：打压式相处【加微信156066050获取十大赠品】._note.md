# 李越习惯性亲密关系课程03.Part1第二课：打压式相处 🧱

在本节课中，我们将学习亲密关系中一种常见的负面互动模式——打压式相处。我们将详细拆解其表现形式，并探讨它对个人成长和亲密关系造成的深远影响。

在上一节课程中，我们介绍了“期望式相处”。本节中，我们来看看另一种更为普遍且伤害性更强的模式：“打压式相处”。这是许多人在成长过程中最熟悉、最习惯，同时也最感痛苦的一种相处方式。

与期望式主要通过注入想法、激发责任或愧疚感来施加影响不同，打压式本质上是一种非常负面的状态。它直接否定个体的内在价值。

以下是现实生活中，尤其在亲子互动中，最常见的四种打压方式：

**1. 打压想法**
所谓打压想法，是指父母会否认或否决你的想法。当你对某件事物产生看法或提出要求时，他们会直接否定，甚至可能在精神层面歧视你的想法，让你觉得自己很蠢、很“二”。长此以往，你会不敢表达甚至不敢产生自己的想法。
*   **例如**：小时候你想去某个地方玩，说出来就被骂；你想买某个东西，一提出来就被否决。你的观点和看法在成长过程中因不断被打压而被迫隐藏起来，最终你会完全质疑自己产生想法的权利。

**2. 打压行为**
打压行为是指你的具体言谈举止遭到斥责。你可能只是下意识地说了一句话、以某种方式走路、写字或与朋友玩耍，这些行为都会招致父母的严厉批评，被告知“这样做不对”。
*   **例如**：主讲人分享自己小时候爱看电视，父母不允许。以至于成年后，只要在家看电视时听到门响（联想到父母回来），内心仍会习惯性地产生焦虑和“做错事”的感觉。这种行为打压会让人在做某些事时，下意识地突破理智，认为自己不对，并急切地想去迎合他人的期望，从而获得安全感。

**3. 打压情绪**
打压情绪的影响极为深远，但很多人并未意识到其重要性。它指父母不允许孩子表达情绪，如不准哭、不准难过、不准愤怒。
*   **例如**：一位学员常年表情平淡，说话语调单一，即使在讲述悲伤经历时也毫无情绪波澜。深究原因，是小时候一哭就会挨打，被要求不准有任何情绪表达。这导致他习惯于用“麻木”来应对内心感受，严重影响了情绪感知与表达能力。**情商（Emotional Quotient）中的“情”指的是情绪**，即情绪管理的智慧。长期的情绪打压会严重损伤情商，使人无法感知、表达、处理自己的情绪，也无法在沟通中感染他人。

**4. 打压欲望**
欲望与想法不同，欲望是内心深处对事物的强烈渴望，比如对金钱、异性、快乐的渴望。打压欲望即否定这些渴望的正当性。
*   **例如**：过年收红包时，孩子内心想要，但又被教育“喜欢钱不对”，于是必须表现出被迫接受的姿态。这种打压会导致人在面对内心真正渴望的事物时，处于回避和纠结状态，不敢直接承认。
*   **另一个例子**：有些女生想要男朋友花钱，但无法直接表达“我想要钱”，于是必须为这个欲望套上一个“政治正确”的理由，如“生日礼物是理所应当的”。这本质上是欲望被打压后，一种扭曲的表达方式。

综上所述，打压式相处模式会全面否定一个人的想法、行为、情绪和欲望。它会导致个体：
*   否认自己的感受和想法。
*   变得拧巴、纠结、痛苦。
*   不知道什么该做、什么不该做。
*   **最终，极度渴望他人来为自己做决定**。

正是这种“渴望他人做决定”的心理，将我们引向了下一课的内容。

既然谈到渴望他人做决定，那就要进入我们的下一堂课：**掌控式相处**。掌控式是期望式和打压式的结合与深化，它会给个体带来更深的痛苦，并进一步固化其对亲密关系的扭曲认知。理解打压式，将帮助我们更好地理解下一课更具控制性的“掌控式”。

**本节课总结**
本节课我们一起学习了“打压式相处”的四种具体形态：打压想法、打压行为、打压情绪和打压欲望。如果你在成长中经历过这种模式，你可能在两性关系中更习惯于被对方打压，并快速屈从。认识到这一点，是迈向成熟、建立健康关系的第一步。下一课，我们将探讨“掌控式相处模式”，敬请期待。
