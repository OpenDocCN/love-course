## 第三阶段：升级关系阶段 🔼

经过铺垫叙事阶段的积累，我们进入第三阶段。在铺垫阶段后期，你能逐渐感受到女生对你兴趣的增强。

2.  她会介意或生气你不回消息。
3.  即使你发送一些关于心情或天气等可回可不回的内容，她也会认真回复。
4.  如果她因忙碌未能及时回复，事后会向你解释。
