# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如一个数字或一段文字）。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是赋值，公式如下：
`变量名 = 值`

例如，我们创建一个名为 `my_number` 的变量，并把数字 `10` 放进去：
```python
my_number = 10
```
现在，`my_number` 这个“盒子”里就装着了数字 `10`。

## 基本数据类型 🔤

仅仅知道如何存放数据还不够，我们还需要了解数据的“种类”，这就是数据类型。Python中常见的基本数据类型有以下几种。

以下是几种最常用的数据类型：

*   **整数（int）**： 表示没有小数部分的数字，如 `-5`, `0`, `100`。
*   **浮点数（float）**： 表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**： 表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `‘你好’`, `“Python”`。
*   **布尔值（bool）**： 只有两个值，`True`（真）和 `False`（假），常用于条件判断。

## 如何查看和使用？ 👀

理解了变量和类型后，我们来看看如何实际操作它们。你可以使用 `print()` 函数来显示变量的内容，使用 `type()` 函数来查看变量的数据类型。

以下是具体的操作示例：

```python
# 定义不同类型的变量
age = 25                     # 整数
price = 19.99                # 浮点数
name = “情叔”                # 字符串
is_student = True            # 布尔值

# 打印变量的值
print(age)                   # 输出：25
print(“我的名字是：”, name)  # 输出：我的名字是： 情叔

# 查看变量的类型
print(type(price))           # 输出：<class ‘float’>
print(type(is_student))      # 输出：<class ‘bool’>
```

## 变量命名的规则 ⚙️

给变量起名字不是随意的，需要遵循一些简单的规则。良好的命名习惯能让你的代码更易读。

以下是命名时需要遵守的主要规则：

*   名称只能包含字母、数字和下划线（`_`），且不能以数字开头。
    *   **有效**：`my_var`, `var1`, `_data`
    *   **无效**：`1var`, `my-var`, `my var`
*   名称是区分大小写的。`Age` 和 `age` 是两个不同的变量。
*   不能使用Python的保留关键字（如 `if`, `for`, `while`）作为变量名。
*   建议使用描述性的、清晰的名字，例如用 `user_name` 而不是 `n`。
