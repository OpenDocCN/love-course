# 富饶《2020完美男友vip》：完美男友第九集——幼稚到成熟下_note

在本节课中，我们将深入探讨从幼稚走向成熟所需掌握的社会生存与人际交往核心法则。课程内容基于大量真实社会经验总结，旨在帮助你理解复杂的人性，并在社交、职场及个人成长中保护自己、把握主动。

---

上一节我们介绍了从幼稚到成熟的部分概念，本节中我们将具体学习一系列在社会中生存与发展的关键原则。

以下是本节课的核心要点列表：

**1. 分析每句话的动机**
你和别人说的任何一句话，都要分析对方每一句话的动机。你才能明白他真正的意图，了解他究竟是在调侃、讽刺、夸你，还是有事找你，或者想跟你套近乎拉关系。无论男人女人和你说的每一句话，你都要分析对方每一句话的动机。因为很多时候，冤枉你的人比你还知道你有多冤枉。这个世界上坏人太多，所以你面对这样的人，必须谨言慎行。自己说的每一句话、每一个字，只要一张嘴开口说话，就得过脑，就得想这句话会造成什么后果，有没有危险，会不会伤害到人。同样，别人在和你说话的时候，你都要分析每一句话的动机。动机就是他为什么说这句话。例如，一个许久不联系的人突然来夸赞你，将你捧上天，其最终目的可能是为了向你借钱。

**2. 对他人好要循序渐进**
对一个人，你一定不能在一开始或者全程太好，要循序渐进，适可而止。不然你稍微对他不好一点，反而他会怪你、恨你甚至伤害你，他已经把你对他的好当成理所应当了。你对一个人循序渐进的好，才是真正的好，才是真正能让你们俩的关系持续长久。这并不是自私或吝啬，而是一种两个人相处的智慧。例如，在特定环境中，如果你一开始就对下属（或“马仔”）过于慷慨（如分享稀缺资源），一旦因客观原因无法维持这种“好”，对方不仅不会感激你过去的付出，反而会因你“变差”而怨恨甚至报复你。在工作中亦是如此，无原则、无底线的“好”可能换来的是背叛与伤害。

**3. 防人之心不可无**
防人之心不可无。这一点你必须要记住。这个人代表的是除了你父母之外的任何人。你和任何人讲话、和任何人说话，都不要把自己的秘密，或者能够很影响你的事情说出来。一点点影响你可以说，很影响的千万不能说，致命的千万不能说，只能和你父母说。

**4. 人人皆自私**
每个人都是极度自私的，对他有利的事才会去做。这句话可以用在你的朋友、工作上，还有情感关系上。每个人其实脑袋里只会想着自己，只会想到自己怎么能活得更开心、更快乐，对自己怎么利益能够最大化。所以他只会做对他有利的事。例如，在约会时，对方可能不愿意配合你拍摄对你有利的视频，但却会非常认真地拍摄能让他自己发朋友圈炫耀的照片。在工作中，朋友愿意帮你办事，往往也是因为这件事对他同样有利。因此，想让别人为你做事，就要思考对方能从中获得什么利益。

**5. 有人的地方就有斗争**
有人的地方就有斗争。任何圈子，任何群体，没有意外。我们没有所谓的和平盛世，永远都不会有人和人和睦相处的情况。小到两三个人宿舍，大到国家，再到各种圈层，没有不斗争的。当你进入一个社交群体或公司，选边站队很重要。不要幻想通过不说话、装老实、本本分分就能混得好。你自己想和平，但别人不一定愿意。总有人喜欢搞事、争当老大、阿谀奉承。如果你想在这个领域有所提升，可以阅读一些职场类书籍或《厚黑学》。

**6. 人性欺软怕硬**
任何人都是欺软怕硬的。在没有绝对的能力的基础上，脾气好、善良、不计较、宽容、好说话，拥有这些品质的人，永远都会吃亏被欺负。只有有能力的人，他脾气好、善良不计较、宽容，大家才会称赞他，发自内心地尊重他。如果你和大部分人处于平等状态，你善良不计较宽容，你永远都会吃亏。在任何场合事情当中，都要誓死维护自己应得的权利。例如，即使在最恶劣的环境下，摆出一副“光脚不怕穿鞋”、坚决捍卫自身权益的姿态，也能赢得基本的尊重。女人也会非常欣赏你这种坚守原则的品质。

**7. 人际互动中的影响力**
两个人在一起时，一定有一个被影响和一个影响他的人。谁看谁的眼神多一点，谁就是被影响的人。例如，在约会或与老板交谈时，如果你一直全身贯注地盯着对方，而对方却不时看手机或环顾四周，那么你就是被影响、跟着对方节奏走的人。如果你想显得自己地位更高或更具影响力，可以有意识地减少注视对方的时间，让对方更多地关注你。

**8. 真正厉害的人往往低调**
每个牛逼的人无一例外都很低调。但是越厉害的人，越是不会说自己的事。但同时每个人也都有虚荣心。只要这个人他在跟你说他自己多牛逼、赚多少钱，他一定不是多厉害的。真正厉害的人在他擅长的领域会觉得没有炫耀的必要，但他的虚荣心可能会体现在其他方面，例如谈论自己的理想或社会贡献。就像一些成功人士，他们很少谈论自己多有钱，而是谈论“改变世界”、“创造价值”。

**9. 拒绝要直接干脆**
拒绝别人，一定要直接干脆。不要害怕当众批评别人，或开别人玩笑会让他没面子。反而他会因此知道你做事的风格，从而更不会在和你相处过程中出错，因为他知道出错会让他付出很难受的代价，你并不会包容他。例如，对于不合理的要求或下属的错误，直接了当地指出或拒绝，比委婉表达更能树立界限，让对方明白你的原则，以后不敢轻易越界。

**10. 自我定位决定他人对待**
你自己有多看重自己，别人就有多看重你。例如，在学校或职场，如果你自己受了伤或遇到不公却表现得无所谓，那么下次你受再严重的伤或遇到更过分的事，别人也不会在意。反之，如果你非常重视自己的权益和感受，并明确表达出来，别人就会因此更尊重你，不敢轻易冒犯。在受到冒犯时，即使内心觉得无所谓，有时也需要表现出“有所谓”的态度。

**11. 嫉妒是常态**
能力比你强、过得比平级好，他一定会嫉妒你、怨恨你，背后给你使绊子。例如，你的朋友或同事看到你交到漂亮女友或业绩突出，表面恭维，背后很可能酸楚并挑刺。对于下级，最忌讳的是没威严、没威信，要让他们无条件服从命令。如果上司发现你过得比他好（例如更轻松、更受老板赏识），他也会对你有想法。因此，表现要恰到好处：比上级稍差，与平级差不多，对下级保持威严。

**12. 向上社交借力**
到了任何一个圈子，一定想尽一切办法和这个圈子里的前几号人物接触，从他们身上借力。例如，进入一个新公司或社交场合，先观察并设法接触其中最核心、最有影响力的几个人。与他们建立良好关系所带来的收益，远大于和圈子内大多数普通人交往。关键是要“眼里有活”、勤奋、会说话。

**13. 掌握他人弱点（打蛇打七寸）**
对待任何一个需要管理的人（如下属），都要想想他的弱点。他的哪些方面、哪些利益掌握在你的手上？他最看重的是什么（如稳定、金钱、成长）？一旦他不无条件服从命令，立刻“打蛇打七寸”，用他最看重的东西去制约他。例如，对看重钱的员工，强调不服从会影响其收入；对看重成长的员工，强调不配合会失去学习机会。要精准打击要害。

**14. 大众观点的局限性**
90%的人都是人云亦云的。同样一件事，不同的人会有完全不同的想法。因此，别人夸你，你不要全信；别人骂你，你也别全信。要以实践结果作为衡量对错的最真实标准。保持心态端正，不要因为外界的褒贬而轻易否定或膨胀自己。要知道这个世界什么样的人都有，坚持用事实检验自己。

**15. 找准自身定位**
每个人都有自己的定位，做好自己的事情就行。在不同的关系和社会角色中（如对父母、对下属、对朋友、对伴侣），你的定位和核心竞争力不同。你需要清楚自己在当前情境中的角色和优势是什么，并据此行动。例如，在恋爱中，如果对方年龄、阅历远高于你，你的定位可能就不是提供物质或绝对安全感，而是提供情绪价值和新鲜感。不要在不擅长的领域硬撑，要扬长避短。

**16. 世界险恶，谨言慎行**
这个世界上的坏人真的是特别特别多。一定要谨言慎行。遇难自保，逢危当弃。做的每一件事情，说的每一个字，表达的每个观点都要经过大脑思考，想一想它会不会给你带来祸害。同时，心态要摆正，不要盲目羡慕他人一时的风光，也不要因自己一时的低谷而绝望。每个人的发展轨迹不同，有年少得志，也有大器晚成，有厚积薄发，也可能有大起大落。专注于自身成长，以实践为尺，保持警惕，稳步前行。
