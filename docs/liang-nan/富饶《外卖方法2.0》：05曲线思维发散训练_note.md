# 富饶《外卖方法2.0》：05曲线思维发散训练_note

在本节课中，我们将要学习一种名为“曲线思维发散训练”的技巧，它通常被通俗地理解为“幽默式推拉”。这是一种通过制造情绪波动来增加聊天趣味性的基本思维方法。

## 推拉的基本概念

上一节我们介绍了课程主题，本节中我们来看看“推拉”的具体含义。推拉，通俗来说，就是把女生推开再拉回来。

它的核心目的是在聊天过程中制造情绪波动。推拉在聊天中主要起到一个作用：**让对话变得幽默、风趣、好玩、有趣，让女生觉得你很搞笑**。

其实，推拉在聊天过程中的作用就是让女生觉得你很好玩。几年前，我特别喜欢使用推拉，因为它能让女生在和我聊天时全程保持笑容。但随着时间推移，我个人使用得少了，这可能是因为自身变得更加成熟，风格有所改变。然而，推拉的作用依然显著，特别是对于吸引年轻女生，或在快速邀约、快速建立吸引力的场景中，它能让女生对你迅速产生兴趣。

## 幽默背后的原理：反差感

理解了推拉的目的，我们接下来探讨其背后的原理。很少有人能彻底搞明白“搞笑”或“让人开心”的原理。在我看来，其核心原理只有一个：**反差感**。

例如，周星驰的电影《功夫》中有一个经典场景：星爷在大杂院与人决斗，他喊“那边那个瘦子给我出来”，结果出来的是一位肌肉壮汉；他接着喊“那边那个小矮子给我出来”，站起来的却是一个两米高的大汉。这种预期与现实的巨大差异，就制造了“反差萌”，让人感到意外和有趣。

在聊天中应用这个原理，同样能产生幽默效果。

## 聊天中的推拉话术示例

以下是几个在聊天中运用“反差感”进行推拉的具体话术示例：

*   **示例1**：女生问：“你平时爱干嘛？” 你回答：“我平时爱看书、游泳、打飞机。” 当女生表现出惊讶时，你接着说：“哎，其实跟你开玩笑的，我这样的人怎么可能看书呢？” 这里，先建立一种形象（爱看书），再用自嘲推翻它，形成反差。
*   **示例2**：你说：“我们不要再联系了。” 女生问：“为什么？” 你回答：“因为我喜欢上别人了。” 当女生追问时，你说：“我喜欢那个人将来会叫你妈妈，叫我爸爸。” 这是一次从“推开”（断绝联系）到“拉回”（表达长远亲密关系）的推拉。
*   **示例3**：你说：“不和你聊天了，我要去环游世界了。” 女生问：“你要怎么环游世界？” 你回答：“等我见到你之后，围着你绕一圈，就环游世界了，因为你就是我的全世界。” 这里将宏大的“环游世界”与围绕个人的浪漫表达结合，制造反差。
*   **示例4**：你说：“我不想和你聊天了，我再和你聊天，我手机就快……”（发省略号）。女生通常会好奇地问“快怎么样？”，可能以为“快爱上我了”。结果你回答：“我觉得再和你聊天，我手机流量就不够用了。” 这打破了对方的浪漫预期，产生了幽默效果。
*   **示例5**：约会时，你看着女生说：“我们下次不要再见面了。” 女生问为什么，你回答：“因为我怕我再和你约会的话，我会爱上你。” 这是一次典型的“推”（拒绝再见）后立即“拉”（表达潜在爱意）。

总之，推拉的基本模式是：**一句话把女生推开，紧接着另一句话把她拉回来**。核心在于制造预期与结果之间的**反差**，从而产生幽默感。只要有幽默效果，推拉在聊天中的作用就实现了。

## 如何培养曲线思维与幽默感

了解了原理，你可能会问如何获得这种制造反差和幽默的能力。以下是几种推荐的训练途径：

*   **途径一：使用特定APP**  
    建议下载“皮皮虾”这款软件。里面充满了有趣视频和“神评论”，经常浏览可以感受和学习网络上的幽默表达方式。
*   **途径二：观看特定节目**  
    推荐每周观看《天天向上》这档节目。观察主持人间如何对话、抛梗、制造幽默，可以学到很多互动技巧。
*   **途径三：聆听相声**  
    多听郭德纲等相声演员的作品，有助于感受和理解传统幽默背后的节奏和逻辑。

当然，作为课程学员，你们也会获得一些经典的推拉话术库供参考和使用。关键是要明白，**推拉的基本思维就是制造反差萌，通过一推一拉来创造幽默**。

本节课中，我们一起学习了“曲线思维发散训练”，即幽默式推拉。

我们首先明确了**推拉**是通过“推开再拉回”来制造情绪波动，核心目的是让聊天变得**幽默有趣**。接着，我们揭示了幽默背后的核心原理是**反差感**，并通过多个具体的话术示例展示了如何在聊天中运用这一原理。最后，我们介绍了通过使用“皮皮虾”APP、观看《天天向上》节目、聆听相声等途径来培养这种思维和幽默感。

希望你能按照这些方法进行训练和尝试，在聊天中变得更加幽默，从而解决对话枯燥平淡的问题。
