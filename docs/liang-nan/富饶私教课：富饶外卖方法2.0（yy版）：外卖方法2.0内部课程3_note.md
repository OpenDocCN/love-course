## 聊天第一轮：建立吸引的五步流程

上一节我们回顾了外卖方法的整体框架，本节中我们来看看具体执行聊天的第一轮操作。这轮聊天的唯一目标是：**建立吸引**。

以下是建立吸引的五个核心步骤：

1.  **打招呼测试**
    *   这是聊天的第一步，目的是破冰并测试女生对你的初始兴趣窗口。操作非常简单：发送一个表情，或说一句简单的开场白（例如：“今天的小惊喜，认识你很开心”）。通过她的回复速度和态度，可以初步判断她对你的兴趣程度。

2.  **右脑思维开场**
    *   在得到女生回复后，进入第二步。此时，不要进行逻辑思考，直接根据你看到的她的信息（如头像、朋友圈照片、签名）进行即兴评论，从而开启一个自然的话题。例如，看到她的旅行照，可以直接说：“这张在海边的照片，阳光看起来好舒服。” 这能让女生感觉你真实、自然，容易拉近距离。

3.  **抛钩子引起兴趣**
    *   当话题开启后，为了让她更愿意、更多地回复你，需要使用“抛钩子”技巧。核心是 **话说一半，制造悬念**，引导她追问。例如，女生说“在吃饭”，你可以回：“真羡慕，我还在苦逼地加班。” 她可能会问“为什么加班”，这就创造了继续聊天的机会。公式可以概括为：**陈述状态 + 留下悬念**。

4.  **聊天四项线做吸引**
    *   当女生回复意愿变强时，使用“聊天四项线”技巧来全方位展示自己，建立吸引。这四条线是：
        *   **正经线**：展示你的价值观、生活态度。
        *   **逗比线**：展示幽默感，让聊天有趣。
        *   **聊自己线**：展示你的生活、经历、价值。
        *   **聊她线**：表达对她的关注和理解。
    *   通过在这四条线上穿插聊天，可以立体地展示你的吸引力。

5.  **撤离（推拉与朋友圈狙击）**
    *   在聊天氛围达到一个小高潮时，主动结束对话（例如：突然不再回复）。这是一种“推”的行为，能制造不确定性，让她产生情绪波动。随后，可以通过发布仅她可见的朋友圈（“朋友圈狙击”）来再次引起她的注意，为下一次互动埋下伏笔。这属于心理战术，能强化她对你的印象和兴趣。

## 补充技巧：推拉的作用

在聊天中，“推拉”（先表达否定或推开，再表达肯定或拉近）是一个重要的润滑剂和强化剂。它主要有三个作用：

以下是推拉的三个主要作用：

1.  **强化吸引**：先“推”（轻微否定或拒绝）再“拉”（肯定或亲近），能让随后的“拉”效果倍增。就像先饿一会儿，饭会更香。例如：“你别对我这么好了（推），我怕我会习惯然后喜欢上你（拉）。”
2.  **制造幽默**：利用前后反差制造笑料。例如，女生说“你是个渣男吧？”，你可以回：“是啊（推），每次见你我都紧张得忘了刮胡子（拉）。”
3.  **引导情绪投资**：通过让女生的情绪像过山车一样起伏，她会无形中在你身上投入更多情感关注。经典的“土味情话”多是此原理，例如：“我一点都不想你（推），我一点半再想你（拉）。”

推拉像是菜肴的调味料，能让聊天更有味道，但它不是主食。聊天的主干依然是前述的五步流程。

以下是针对学习过程中常见困惑的解答：
