# 富饶私教课：聊天强化课（苏晨）：推拉_note

在本节课中，我们将要学习一种核心的聊天技巧——推拉。这是一种通过交替使用积极和消极的言语，来创造情绪波动、增加吸引力并深化关系的有效方法。

## 概述

推拉是一种话术，主要用来创造对方的情绪波动和引导感情投资。它包含两个基本动作：“推”是采用相对消极的语言或行为，暂时推开对方；“拉”是采用相对积极的语言或行为，将对方拉近。两者的结合使用，能让交流充满张力，避免平淡或跪舔。

## 推拉的含义与作用

上一节我们概述了推拉，本节中我们来看看它的具体含义和多种作用。

**推拉**是一种用于创造情绪波动和感情投资的话术。
*   **推**：指推开对方，采用相对消极的语言。例如：“我觉得你今天精神不太好。”
*   **拉**：指拉回对方，采用相对积极的言语。例如：“但你整体气质非常棒。”

推拉结合，就是在质疑或否定（推）之后，当对方给出正面回应时，再给予肯定（拉），这能有效增加关系的张力，使其迅速升温。

以下是推拉的十大核心作用：

1.  **制造真实感**：纯粹的赞美可能显得虚假。推拉结合会让对方感觉你的评价更全面、更真实。例如：“你这衣服挺有品味的（拉），就是今天好像没睡醒（推），要是精神足就更帅了（再拉）。”
2.  **提高聊天张力**：如果聊天始终平铺直叙，就像一根硬邦邦的棍子，缺乏趣味。推拉能让对话像皮筋一样有松有紧，充满张力。例如：“你在郑州工作啊（中立），郑州不错（拉），就是太堵了（推），上次两公里堵了我半小时（具体化推）。”
3.  **制造笑点**：通过铺垫建立预期，然后突然转向，产生类似相声“抖包袱”的效果。例如：“你知道吗，你的一举一动都牵动我的心（拉），特别像我小时候走丢的那条大黄狗（推）。”
4.  **拉伸关系**：直接表达好感可能效果一般。先轻微推开，再拉回，能让关系升级更自然。例如，女生关心你：“早点休息。”你可以回：“你别再这样关心我了（推）……因为时间久了，我会容易动情的（拉）。”
5.  **引导感情投资**：让对方情绪像坐过山车一样起伏，她会不自觉地对这段关系投入更多感情。经典的“土味情话”多是此原理。例如：“你眼睛不好看（推）……因为我的眼里全是你（拉）。”
6.  **掌控情绪与展现自信**：能自如运用推拉的人，显得能掌控社交节奏，充满自信和魅力，这与小心翼翼讨好他人形成鲜明对比。
7.  **破解“废测”**：当对方提出刁难问题时（如“我和你妈掉水里先救谁”），可以用推拉巧妙应对。例如：“当然先救你（拉）……这样我家穷的时候就有猪肉卖了（推）。” 或者：“先救我妈（推）……因为救了她，我们俩才能一起救你呀（拉）。”
8.  **延续话题**：当对话陷入僵局（如对方只回“哦”、“好吧”），可以用推拉重新打开话题。例如，你夸她衣服好看，她回“还好吧”。你可以说：“这衣服让我想起我老家的窗帘（推拉结合）。” 如果她拒绝你的亲近表示，你可以说：“不让亲算了（推）……我本来还想先嚼点大蒜呢（拉）。”
9.  **展示原则**：想指出对方缺点或表达原则时，先拉后推，对方更容易接受。例如：“兄弟你这人特靠谱，上次帮忙真给力（拉）。不过有件事，你有时说话不太分场合（推），当然我知道你也是好心（再拉）。”
10. **带入暧昧**：推拉能自然地营造暧昧氛围。例如：“我觉得你根本不应该出现在这里（推）……你应该待在我心里（拉）。”

## 推拉的用法与层级

理解了推拉的作用后，本节我们来学习如何具体使用它，并从简单到复杂掌握其应用层级。

### 初级用法：推拉 / 拉推
这是最基本的单元，即一次推和一次拉的组合。
*   **公式**：`推 + 拉` 或 `拉 + 推`
*   **示例**：“你嘴这么甜（拉），是偷吃蜜了吗（推）？” 或 “你今天穿得真土（推）……因为我五行缺土（拉）。”

### 中级用法：连环推拉
将多个推拉单元串联起来，形成更长的、充满起伏的对话流。
*   **核心**：依靠联想能力，将一个话题自然延伸。
*   **示例**：“你嘴这么甜，说得我都想亲你了（拉）。给亲吗？” “不给（对方推）。” “好吧，那我亲你前一定嚼好大蒜熏晕你（推）。……知道为什么熏晕你吗？这样才好给你做人工呼吸呀（拉）。”

### 高级用法：框架内推拉
在一个大的“拉”或“推”的框架内，进行多次小的推拉，效果非常强烈。
1.  **拉中推拉**：先建立一个积极的假设框架，在里面进行推拉。
    *   **示例**：“如果我们在一起（大拉），你惹我生气我会收拾你的（推）。……你这么霸道（推）？小心我把你按在墙上（拉）。”
2.  **推中拉推**：先建立一个消极的拒绝框架，在里面表达为难与关心。
    *   **示例**：“我想我们还是不要在一起了（大推）。……因为和你在一起固然开心（拉），但我要去外地了，怕你需要时我不在（推）。……你是个好女孩，我不想你受伤（拉）。” （**注意**：此招需在关系稳固后使用，否则易真推走对方。）

## 推拉的应用场景分类

根据针对的不同对象，推拉还可以细分为以下几类，方便你更有针对性地使用：

*   **关系推拉**：针对你们两人之间的关系进行。例如：“如果我们约会，你会不会……”
*   **话题/事物推拉**：针对聊天中的具体事物、故事或第三方。例如：“你这眼镜真好看，特别像鸟叔。”
*   **情绪推拉**：针对对方当下的或想象中的情绪状态。例如：“你生气的样子（推），还挺可爱的（拉）。”
*   **进挪推拉**：针对肢体接触的想象或描述。例如：“我在想，抱你的时候你会不会像小猫一样（拉）……但我可能会忍不住使坏（推）。”
*   **陈述式推拉**：在一句话内完成推拉。例如：“你笑起来的样子好傻（拉+推）。”
*   **互动式推拉**：需要对方回应的推拉。例如：“你是那个聪明绝顶的小仙女吗（拉+推）？”
*   **表达式推拉**：用于展示真实自我，先说自己缺点（推），再合理化或说优点（拉），显得真实。
*   **评价式推拉**：用于评价对方，让评价显得中肯。例如：“你外表看起来很高冷（推），但了解后发现你其实多愁善感（拉）。”
*   **联想式推拉**：用“我在想……”开头，提供话题。例如：“我在想，我捏你脸时你会不会生气（推拉结合）。”
*   **曲解式推拉**：故意曲解对方的话进行推拉。例如，对方说“滚”，你可以回：“我才不跟你滚床单（拉），你那么重（推）。”

## 如何应对对方的推拉

在互动中，对方也可能对你使用推拉，你需要妥善应对以保持框架平衡。

*   **当对方“拉”你时**（如夸你）：
    *   **可以推**：谦虚一下，对方可能更想拉你。例如：“还好吧，也就一般帅。”
    *   **可以拉**：接受并强化，但之后最好跟上推。例如：“想我了？我也想你，想你傻笑的样子（拉）。……不过肯定没我想你那么多（推）。”
*   **当对方“推”你时**（如贬你）：
    *   **通常要推回去**：以保持框架平等，避免跪舔。例如，对方说“你个小屁孩”，你可以回“你才欠收拾呢”（推）。
    *   **反常道而行之（高阶）**：持续地“拉”，最后突然“推”，制造强烈反差和笑点。例如，对方说“我不爱你了”，你可以一直深情挽留（拉），最后说：“那你走之前，把床头那两盒冰套还给我（推）。”

## 推拉在不同关系阶段的使用重点

*   **前期（吸引阶段）**：**以“推”结尾为主**。避免显得像其他追求者一样一味讨好，制造神秘感和挑战性。
*   **中期（暧昧阶段）**：**以“拉”结尾为主**。用于升温关系，巩固好感。
*   **后期（稳固阶段）**：**用“拉”来稳固关系，用“推”来让对方改进缺点**（如“展示原则”作用）。

## 使用推拉的注意事项（切记）

1.  **尺度过大**：关系未到，使用过于亲密或冒犯的推拉词语。
2.  **打压恶俗**：使用“傻逼”等低俗词汇进行“推”，令人反感。
3.  **推/拉得过猛**：持续推可能推走对方；持续拉则变成跪舔。
4.  **缺乏代入感**：推拉前没有做好铺垫和描述，对方无法进入情境。
5.  **缺乏互动性**：自说自话，不留给对方回应和参与的空间。
6.  **缺乏趣味性**：话题本身无聊，再如何推拉也效果有限。
