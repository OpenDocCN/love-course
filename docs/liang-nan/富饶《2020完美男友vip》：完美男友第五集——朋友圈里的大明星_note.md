# 完美男友课程第五集：朋友圈里的大明星 📸

在本节课中，我们将要学习如何在后互联网时代建设一个优质的朋友圈。我们将探讨朋友圈的趋势、拍摄内容的选择、修图技巧以及实际操作细节，帮助你从众多展示者中脱颖而出。

## 朋友圈趋势与现状 🌍

上一节我们介绍了课程的整体框架，本节中我们来看看当前朋友圈所处的时代背景。

2020年，我们已进入后互联网时代。前互联网时代（约2015-2018年）的特点是，人们刚开始学习使用朋友圈进行展示，例如炫富或展示性感特质。这种方法在2019年后逐渐失效，尤其是在一二线城市。

如今，社会现状是全民都在拍摄和修图。从网红店到旅游景点，几乎所有人都在精心构图和修饰照片。抖音、快手等平台更是推动了全民拍摄视频和照片的热潮。

在这种情况下，拥有优质的朋友圈展示面，已经从一种“武器”变成了像“胳膊和腿”一样的基础标配。它不再能让你显得突出，但没有它则会显得不正常。

## 朋友圈的核心展示维度 🎯

了解了时代背景后，我们需要明确朋友圈应该展示什么。以下是朋友圈最能体现个人特质的四个核心维度：

1.  **社会阶层**：你通过朋友圈向他人展示自己在社会中的位置，例如是富人、中产还是其他阶层。
2.  **生活方式**：展示你的日常生活，包括饮食、居住、出行和兴趣爱好。
3.  **思想深度**：同样的事物，不同的展示角度能体现不同的思想深度。这决定了你是“真材实料”还是“徒有其表”。
4.  **个人品位**：通过分享的音乐、电影、书籍、穿着乃至修图风格，都能反映出你的审美和品位。

## 两种朋友圈建设路线 🛣️

基于现状，建设朋友圈主要有两种操作方向。

### 路线一：高酷炫风格

选择这条路线，意味着你的装逼程度需要超过90%的男人，达到与网红媲美的水准。要避免让人觉得你在“装”，需满足以下三点：

*   **装得足够大**：展示的内容要超越普通人的想象范围（例如私人飞机、与顶级明星的合影），这样反而不会被认为是在装。
*   **避免形式大于内容**：不要对过于普通或廉价的内容（如一杯咖啡、一顿火锅）投入过度的修图和时间成本，这会显得刻意和浮夸。
*   **保持一致性**：你的个人气质、穿着打扮必须与照片中的场景、背景相匹配。如果人配不上景，就会显得不真实。

### 路线二：隐形优质男人风格

这是更推荐的一种路线，旨在展示一个真实、有内涵的优质男性形象。其与单纯“装逼”的区别主要体现在以下几点：

*   **每条朋友圈应有意义**：展示的不仅是物品或场景本身，更是其背后的故事、梦想、追求或感悟。例如，发车不是秀车，而是秀“奋斗多年的礼物”或“儿时梦想的实现”。
*   **展示重点应是内涵与文化**：中国的传统文化推崇低调、内涵。即使展示奢华内容，也应设法体现“低调奢华有内涵”的感觉，而非单纯的纸醉金迷。
*   **避免“隐形摄影师”**：发布的照片要让人感觉是自然抓拍或由合理在场的人拍摄的，而不是刻意摆拍。需考虑“这张照片是如何被拍出来的”这个问题，避免需要做出尴尬行为才能完成的拍摄。

## 拍摄内容分类与技巧 📷

明确了路线，我们来看看具体应该拍什么。朋友圈内容主要可分为三大类。

### 第一类：形象类（展示你很帅）

这是当前朋友圈的重中之重。在非常看脸的时代，直观展示你的外貌优势至关重要。

**核心技巧：学会凹造型**
以下是三个基础的拍照姿势：
1.  **对镜自拍姿势**：胯部向一侧顶，顶胯侧的腿伸直，另一条腿微曲。上半身保持不动。
    *   **姿势描述**：`提胯 -> 同侧腿伸直 -> 对侧腿微曲 -> 上半身挺直`
2.  **站立他拍姿势**：胯部自然向前顶，膝盖微曲，双脚分开，上半身挺直，下巴微收。
    *   **姿势描述**：`胯部前顶 -> 膝盖微曲 -> 双脚分开 -> 挺胸收颌`
3.  **基础站姿**：双脚分开，一腿伸直作为支撑，另一腿微曲，腹部收紧。
    *   **姿势描述**：`双脚分开与肩宽 -> 一腿伸直 -> 一腿微曲 -> 收紧核心`

**推荐拍摄场所**：潮人街区（街拍）、网红店、豪华的家中或民宿、停车场。避免在普通办公室拍摄。

### 第二类：社交类

这类照片的核心是展示你的社交圈层和人脉质量。

以下是适合出现在你社交类朋友圈中的人物或场景类型：
*   德高望重的老师或教授
*   形象好、有品位的帅哥朋友
*   合伙人、股东或投资人
*   可爱、打扮时髦的小朋友
*   足够漂亮的女生
*   发布会、高端培训班、拍卖会、车友会、企业年会等线下活动

### 第三类：生活类

生活类展示主要围绕三个维度：吃、旅行、兴趣爱好。

**关于“吃”**：
*   **国内**：只发有故事、有特色、明星开办或装修独特的老字号招牌店。
*   **国外**：可以发进口的特色小零食。
*   **菜品**：优先选择最上镜的日料或西餐。

**关于“旅行”**：
选择能体现深度和独特性的目的地，例如西藏、新疆、斯里兰卡。参与山区扶贫、支教等公益活动的照片也非常加分。

**关于“兴趣爱好”**：
*   **运动类**：最稳妥且通用的选择是**健身房**。
*   **才艺类**：可展示**唱歌**、**跳舞**或演奏**钢琴、大提琴、萨克斯**等乐器（避免弹吉他）。
*   **加分项**：展示**收藏品**（如艺术品、古董）能极大提升逼格。
*   **关于“看书”**：要发得真实，照片应包含**书签**和**阅读褶皱**，并配上**真实的读后感**。

## 修图与滤镜指南 🖌️

拍好内容后，后期修饰是关键。以下是使用手机软件修图的标准化流程。

**所需软件**：`美图秀秀`（修人）、`Instagram`（修图调色，需VPN）。

**修图流程**：
1.  **一键美颜**：在美图秀秀中使用“一键美颜”功能，强度选择`50%`，让皮肤看起来干净。
2.  **去斑去痘**：使用祛痘功能，清除脸部所有瑕疵，如痘印、斑点。
3.  **增高塑形**：使用“增高”功能，将身高拉长`30%-50%`，优化头身比。
4.  **瘦脸瘦身**：使用“瘦脸瘦身”手动功能，调整身体轮廓（如驼背、腿粗、腰粗）。
5.  **面部精修**：按顺序调整五官：
    *   **鼻子**（侧面可拉高鼻梁）
    *   **下巴**（适当拉长）
    *   **眼睛**（使用“亮眼”和“放大”功能）
    *   **眉毛**（使用“亮眼”涂抹，使其更浓）
    *   **脸型**（整体均匀推瘦，注意左右平衡）
6.  **图片剪裁**：裁掉照片中不雅观或杂乱的部分（如垃圾桶、乱入的袜子）。
7.  **滤镜选择**：将照片导入Instagram，根据照片风格选择合适的滤镜。
8.  **调色**：在Instagram中微调`亮度`、`饱和度`、`对比度`等参数，直至满意。

**懒人技巧**：如果觉得自己修图麻烦或效果不佳，可以求助淘宝专业修图师，价格约`30元/张`。你可以提供想要的风格参考图。

## 实际操作与发布规范 📝

最后，我们来看看朋友圈各个细节的实际操作和发布规范。

**基础资料设置**：
*   **头像**：选择能清晰展示你脸部的最帅照片。让人记住你的长相是关键。
*   **背景墙**：选择一张最能体现你个人整体风格（穿着、身材、品位、性格）的高质量照片。
*   **昵称**：要么用**洋气的真名**，要么用**包含自己名字中一个字的接地气外号**（如“刘大懒”）。
*   **个性签名**：要么表达**当下的状态或价值观**，要么表明**你的身份或职业**。

**发布规范**：
*   **频率与时间**：每周发布`2-3`次为佳。发布时间可选择下午`6-8点`（下班通勤时段）或晚上`10点后`（睡前时段）。
*   **排版**：
    *   单条朋友圈配图以`1张`为最佳，`2-4张`也可接受。
    *   避免使用`九宫格`，显得过于迫切。
    *   不要发纯文字朋友圈。
*   **定位**：在同一地点（如一次旅行）期间，定位不要重复超过`2-3`次。
*   **配文**：
    *   **表达当下状态**（如“这是我吃过最辣的火锅”）。
    *   **体现深度正向价值观**（如透过一件事发表积极感悟）。
*   **发布禁忌（切勿触碰）**：
    1.  各种**投票、助力、抢票**链接。
    2.  **集赞**送礼活动。
    3.  公开**帮人介绍对象**（“拉皮条”）。
    4.  发布**钓鱼式互动**（如“没睡的举手”）。
    5.  为他人进行**互推**。
