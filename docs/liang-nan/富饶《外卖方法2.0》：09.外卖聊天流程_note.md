# 富饶《外卖方法2.0》：09.外卖聊天流程 💬

在本节课中，我们将要学习与女生从添加微信到建立吸引的完整第一轮聊天流程，以及后续如何安全、有效地拉升关系。我们将拆解每一步的核心目标与操作方法。

## 第一轮聊天：从破冰到建立吸引

上一节我们介绍了课程的整体框架，本节中我们来看看具体的外卖聊天第一轮流程。其核心目标是测试窗口、开启话题、建立吸引，并在高潮时主动撤离。

### 第一步：打招呼测试

首先，我们添加女生后的第一步是打招呼测试。目的是测试女生当下对你的反应和兴趣窗口。

以下是常见的打招呼方式：
*   发送一个表情，如笑脸 😊 或害羞 😳。
*   发送一句简单的开场白。

无论使用何种方式，关键在于观察女生的回复意愿。如果她愿意回复，哪怕你的开场白很普通，也说明她对“你这个人”至少不反感，存在基本窗口。如果她不回复，则可能暂时没有窗口。

### 第二步：右脑思维开场

当我们判断女生有基本窗口后，便进入第二步：右脑思维开场。其核心是**基于女生的个人信息（如朋友圈照片、签名）进行直观描述**，从而开启话题。

点开她的朋友圈，将你看到的最直观感受发给她。例如：
*   “你的眼睛好大，很有神。”
*   “这条裙子的颜色很衬你的肤色。”
*   “看到你去滑雪的照片，感觉你好酷。”

这种方式能快速让女生感到被关注，并愿意就你开启的话题进行更多交流。

### 第三步：抛钩子引起兴趣

开启话题后，我们需要使用“钩子”来持续引起女生的兴趣，引导她主动提问或提供更多信息，使聊天进入平行、互动的状态。

抛钩子的方法是**在对话中自然植入能引发好奇的片段**。例如：
*   女生说：“在吃饭。”
*   你可以回：“真羡慕，突然好想念郑州的胡辣汤。”
*   这时女生可能会问：“你现在不在郑州吗？”（钩子生效）

通过女生的追问，你便有机会自然地展示价值（如旅行、工作经历），让对话更加深入。

### 第四步：聊天四项线做吸引

当聊天进入良好互动状态后，我们使用“聊天四项线”来深化吸引。这是一个循环的聊天模式：**正经提问 → 接收回答 → 展示价值 → 适度调侃**。

公式表示为：`吸引 = 提问 + 价值展示 + 调侃（对己/对人）`

例如：
*   你问：“你是做什么工作的？”（正经提问）
*   她回答：“我是设计师。”
*   你说：“厉害，我最近正想学点设计，感觉能做出好看东西的人都很有创造力。不过我手比较笨，上次用PS差点把电脑搞崩。”（展示价值+调侃自己）
*   然后可以说：“那你平时灵感枯竭了，会不会也像我一样靠狂吃零食来缓解？”（适度调侃对方）

通过这种方式，聊天既有信息交换，又充满情绪互动，能让女生感觉舒适且被吸引。

### 第五步：高潮撤离

在通过聊天四项线进行了约20-30句交流，气氛达到高潮时，主动结束对话。这就是第五步：高潮撤离。

操作方法：
*   在她兴致很高时，突然停止回复。
*   或者延迟很久再回复。

这样做的目的是制造意犹未尽的感觉，为后续互动埋下伏笔，同时结合朋友圈等其他方式强化印象。

**总结一下第一轮聊天流程**：首先通过打招呼测试窗口；然后用右脑思维开场开启话题；接着抛钩子引起兴趣并互动；再使用聊天四项线循环深化吸引；最后在高潮时主动撤离。

## 第二轮互动：拉升关系

上一节我们完成了吸引的建立，本节中我们来看看如何将关系从普通朋友拉升到更亲密的阶段。核心原则是：**确保女生会同意，再发出拉升关系的信号**，而非冒险表白。

### 正确拉升思维：测试先行

传统错误思维是：先表白拉升，再等待女生同意。这有50%的失败风险。

外卖方法的正确思维是：**先通过测试确认女生会同意，再将关系在文字上拉升**。拉升是胜利的号角，而非开战的信号。关键在于拉升前的“测试”。

### 关系拉升前的测试方法

在说出任何拉升关系的话之前，必须进行多轮测试来确认女生的接受度。

以下是必须进行的四轮小动作测试：

1.  **称呼测试**：改变对她的称呼，如从名字改为“小可爱”、“宝”等，观察其反应。
2.  **早晚安测试**：引导进入互道早安晚安的模式。例如：“以后我每晚跟你说晚安，你早上叫我起床好不好？”
3.  **暧昧话题测试**：聊一些易于代入二人世界的话题。主要有三类：
    *   **做饭**：“改天我们一起买菜做饭，我露一手。”
    *   **自驾游**：“真想有一天能开车带喜欢的人去洱海边发呆。”
    *   **旅行**：“我觉得和喜欢的人去泰国晒太阳，会是特别美好的回忆。”
4.  **大动作测试**：描绘具体的、见面后的实质性场景。例如：
    *   “你那么瘦，下次吃火锅我得盯着你多吃点肉。”
    *   “听说你怕水？没事，下次游泳我手把手教你。”

当以上测试都获得女生积极或默许的回应后，说明她已准备好，此时进行关系拉升成功率极高。

### 拉升关系的技巧：3+1静挪法

“3+1静挪法”指用三种技巧辅助拉升关系，将拒绝风险降到最低。“3”指三种辅助方式：

1.  **话题加拉升**：将拉升意图包裹在一个具体活动中。
    *   错误示范：“我想牵你手。”（过于直接）
    *   正确示范：“我们手拉手去看那部新电影吧。”（将牵手融入“看电影”这个话题中）

2.  **价值加拉升**：在提出邀约时附带展示价值。
    *   普通邀约：“周末一起吃个饭？”
    *   价值加拉升：“周末带你去我朋友新开的日料店尝尝，他专门从日本请的厨师。”（融入“人脉”和“稀缺体验”价值）

3.  **幽默加拉升**：用幽默感软化拉升意图，避免尴尬。
    *   示例：“我刚啃完猪蹄没洗手，要不要一起手拉手看电影？”（用自嘲制造幽默）
    *   示例：“我准备了电影、零食和套套。”女生惊讶时，回复：“吃鸭脖用的手套，你想啥呢！”（用曲解制造幽默）

“+1”则是在前三者基础上，最终明确表达拉升关系的意图。核心是**让拉升的请求像润滑油一样自然，而非生硬的发动机**。

本节课中我们一起学习了外卖聊天的核心两阶段。第一阶段是从打招呼测试到高潮撤离的五步聊天法，旨在建立稳固的吸引。第二阶段是关系拉升，我们学习了“测试先行”的正确思维，以及通过“小动作测试”和“3+1静挪法”来安全、有效地推进关系。记住，所有文字上的拉升，都应建立在实质关系已经发展到相应阶段的基础之上。
