# 富饶私教课：精品视频课：03.形象改造_note

在本节课中，我们将要学习如何系统性地进行个人形象改造，涵盖发型、五官、皮肤、身材、身高以及穿着搭配等多个方面。我们将提供具体的操作建议和产品推荐，帮助你提升外在形象。

## 发型设计 ✂️

发型是形象改造的第一步。一个好的发型能极大地提升个人气质。

上一节我们介绍了课程概述，本节中我们来看看如何打造合适的发型。

不建议你带着明星或网红的照片去找发型师。每个人的脸型、肤色和气质都不同。最好的方式是让发型师根据你的个人脸型来设计。在修剪过程中，不要过度限制发型师的发挥，因为他比你更专业。你只需要清晰地表达自己的想法，并与发型师充分沟通。

以下是推荐的理发店选择：
*   **东田造型** 和 **八曲造型**：在一二线城市均有分店。
*   **其他城市**：如果你在三线或更小的城市，可以在APP上选择价格最贵的理发店。到店后，可以选择店内价格最优惠的发型师为你修剪。

剪完头发后，如果需要自己打理，推荐购买 **湿滑扣** 品牌的 **法尼** 和 **法拉** 两款产品。

## 眉毛与五官修饰 👁️

发型过后，我们来说眉毛。眉毛的整洁度对面部轮廓影响很大。

如果你的眉毛颜色偏淡，建议可以选择去纹眉。如果你的眉毛比较厚重但杂毛多，可以定期去理发店或商场里的修眉店进行修剪。修剪周期建议与发型同步，大约三周一次。

接下来是眼睛。这里不建议佩戴框架眼镜，原因有两点：
1.  框架眼镜会给人一种不健康的感觉，类似于看到别人腿上缠着绷带。
2.  几乎没有偶像明星或潮人会佩戴框架眼镜。

推荐佩戴隐形眼镜。品牌推荐 **博世伦**，可以购买月抛型，这是比较经典且镜片较软的品牌，佩戴起来更舒适。

然后是鼻子。如果鼻子上有黑头或粉刺痘痘，可以在淘宝上购买 **猪鼻贴** 等专门去黑头的产品。

最后是胡子。建议每天刮胡子。如果你是青少年或学生，胡子不整齐，也需要每天修整。如果你想留胡子并塑造有型的感觉，可以在淘宝上购买那种可以自定义长度的刮胡刀，直接刮出想要的长度即可。

## 皮肤护理与面部塑造 🧴

男生的皮肤保养可以简化为两步：**洁面** 和 **保湿**。

洁面通常指使用洗面奶控油和清洁皮肤。清洁之后，需要进行保湿。

以下是产品推荐：
*   **洁面**：推荐去痘洗面奶 **苦乐**。
*   **保湿**：推荐保湿产品 **马油**。
*   **遮瑕**：如果皮肤有痘印或瑕疵，可以选择 **CC霜**，品牌推荐 **密斯佛陀**，在淘宝可以买到。

## 身材管理与身高调整 🏋️

关于身材，需要根据自身体型进行管理。

如果你的体型偏胖，需要健身并做有氧运动（如游泳、跑步），同时不吃主食和夜宵。如果你的身材中等，这是女生最喜欢的身材，只需保持，避免吃宵夜和油炸垃圾食品。如果你的体型偏瘦，建议每天早上吃两个鸡蛋清，并坚持下去，体型会慢慢变得健壮。

关于身高，这里有一个明确的建议公式：
*   **北方城市**（如东北、河北、山东、河南及北上广深）：身高不足 **175cm** 建议使用增高鞋垫。
*   **南方城市**（如云贵川、长沙、福建）：身高不足 **170cm** 建议使用增高鞋垫。

具体垫高多少，以达到北方175cm、南方170cm的“及格线”为目标。例如，在南方身高165cm，可以垫高5cm达到170cm。

## 穿着搭配指南 👔

穿着是形象塑造的重要环节。根据你的身份和经济状况，有不同的品牌选择。

以下是针对学生或刚步入实习阶段的推荐品牌：
*   **Triniano**
*   **Zara**
*   **H&M**
*   **优衣库**
*   **无印良品**

这些品牌比较大众化，性价比较高。其中，Triniano是我们线下学员形象改造的御用品牌。

如果你的经济条件较好，或年龄稍长，可以选择一些奢侈或潮牌，例如：
*   **华伦天奴**
*   **巴黎世家**
*   **Gucci**
*   **Enzo**
*   **纪梵希**
*   **HBA**
*   **TB**
*   **BOY**

潮牌的优势在于，它能较好地修饰身材（无论高矮胖瘦）。在前期你不懂穿搭时，选择潮牌是最不容易出错的方式。后期当你对审美有自己的理解后，可以再自由搭配。

**注意**：现阶段不推荐你穿西服。因为普通衣服穿错尚可，但西服穿错会显得非常不得体。
