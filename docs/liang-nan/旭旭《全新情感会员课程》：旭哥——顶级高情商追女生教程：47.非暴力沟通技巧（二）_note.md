# 旭旭《全新情感会员课程》第47课：非暴力沟通技巧（二）💬

在本节课中，我们将要学习非暴力沟通的核心技巧。我们将深入探讨什么是暴力沟通，并学习如何通过区分事实与观点、表达感受以及明确需求，来实现更有效、更和谐的沟通。这些技巧不仅适用于情感关系，也适用于日常生活中的所有社交场景。

## 什么是暴力沟通？🚫

上一节我们介绍了非暴力沟通的重要性，本节中我们来看看它的对立面——暴力沟通。理解暴力沟通是避免它的第一步。

暴力沟通是指那些会引发他人痛苦、防御或对抗的说话方式。在我们的日常生活中，其实大量使用了暴力沟通的方式。

以下是几种常见的暴力沟通类型：

*   **道德评判**：当一个人的行为不符合我们的价值观时，我们就认为他是不道德的。例如，开车时有人向外扔垃圾，我们会说“真没素质”。或者当伴侣心情不好显得冷漠时，我们内心会评判“又开始装逼了”、“你就是不爱我”。
*   **进行比较**：拿一个人与另一个人进行比较。例如，“你看看隔壁家的孩子，再看看你”，或者“我闺蜜的男朋友都怎样怎样，你怎么就不行”。这种比较会严重伤害对方的自尊心。
*   **回避责任**：淡化自己的个人责任，将问题归咎于他人。常用句式如“要不是因为你，我怎么会...”、“我是被逼的”。我们需要明白，**除了你自己，没有人能控制你**。你做的每一个选择，无论是辛苦工作还是轻松生活，都是你自己的决定，并需要承担相应的结果。
*   **强人所难（威胁）**：通过威胁来迫使对方服从。例如，“你要是再不听话，妈妈就不要你了”、“你要是不给我买，我们就分手吧”。这种沟通方式会带来巨大的心理压力。

## 非暴力沟通的核心技巧✨

了解了需要避免的暴力沟通后，现在我们来学习积极、有效的非暴力沟通方法。

### 技巧一：区分事实与观点

这是非暴力沟通中最重要、最基础的一步。我们最常犯的错误就是混淆事实与个人观点。

**事实**是客观的、具体的、可观察的。**观点**是主观的、模糊的、带有评判色彩的。

以下是几种常见的混淆情况：

*   **使用模糊的形容词**：例如，“你经常迟到”、“你总是这样”。这里的“经常”、“总是”是观点。事实是“你本周迟到了3次，每次大约30分钟”。
*   **对他人的思想或情感进行推测**：例如，“你不爱我了”、“你肯定干不成这事”、“你就是想偷懒”。这是在猜测对方的内心，而非陈述事实。
*   **缺乏依据地说话**：仅凭表面现象就下结论。例如，看到年轻人扶起老太太，就说“肯定是你撞的”。
*   **将形容词当作事实**：例如，“这个人真丑”、“他真差劲”。“丑”和“差劲”是主观观点，而非客观事实。

**如何陈述事实？**
像播报新闻或签订合同一样，陈述**时间、地点、人物、事件**等具体信息。
*   错误（观点）：你怎么天天晚上都这么晚回来？
*   正确（事实）：老婆，你这周有5天都是晚上10点以后才回家。
*   错误（观点）：你能不能给我好好做饭？
*   正确（事实）：亲爱的，这周你已经连续4天晚上只给我煮泡面了。

### 技巧二：体会并表达感受

在陈述事实的基础上，我们需要学会表达自己真实的感受，而不是表达猜测或评判。

**感受**是关于心情和情绪的词，如开心、兴奋、感动、害怕、焦虑、伤心、孤单、生气、紧张、内疚、失落、困惑等。

**猜测/评判**则是关于对方想法或行为的论断，如“我觉得你不爱我”、“我觉得我被你抛弃了”、“你就是敷衍我”。

**将事实与感受结合，形成非暴力沟通句式：**
公式：**陈述观察到的事实 + 表达你的感受**
*   暴力沟通：你他妈天天不回家，是不是不爱我了？
*   非暴力沟通：亲爱的，你已经一周没有回家了（事实），我感到很孤单，也很担心你（感受）。
*   暴力沟通：你看看你考这点分，别念了！
*   非暴力沟通：孩子，你上次考了65分，这次考了55分（事实），妈妈有点担心（感受）。

### 技巧三：找到并表达自己的需求

所有的情绪，尤其是愤怒，都源于内心某个需求没有得到满足。非暴力沟通要求我们洞察并清晰表达这个需求。

*   **洞察需求**：当你感到生气时，先问自己：“我到底需要什么？” 例如，伴侣回家晚让你生气，你的需求可能是“需要他多陪伴”或“需要安全感”；伴侣做饭敷衍让你生气，你的需求可能是“需要被关爱和照顾”。
*   **表达具体需求**：提出的需求要像“事实”一样具体、明确，而不是模糊的指令。
    *   模糊需求：我希望你早点回家。/ 我希望你好好做饭。/ 你给我好好学习。
    *   具体需求：我希望你工作日能在晚上9点前回家。/ 我希望每周能有3天，晚餐是两荤一素的家常菜。/ 妈妈希望你下次数学考试能考到70分以上。

**完整的非暴力沟通流程可以是：**
陈述事实 + 表达感受 + 提出具体请求
例如：“亲爱的，你这周有3天都是晚上11点后才回家（事实），我一个人在家感到有些孤单和担心（感受）。我希望下周工作日，你能尽量在10点前回来，如果实在要加班，可以提前告诉我一声好吗（具体请求）？”

## 总结与预告📚

本节课中我们一起学习了非暴力沟通的三个核心技巧：**区分事实与观点**、**体会并表达感受**、**找到并表达自己的需求**。关键在于将模糊的评判转化为具体的观察，将指责转化为感受的分享，将命令转化为清晰的请求。这需要我们在日常生活中持续练习和反思。

非暴力沟通的本质是**好好说话**，它不仅能改善亲密关系，也能让你在与家人、朋友、同事的相处中更受欢迎。
