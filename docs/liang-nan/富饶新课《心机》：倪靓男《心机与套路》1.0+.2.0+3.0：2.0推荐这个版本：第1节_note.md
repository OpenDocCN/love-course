## 高价值女性（绿茶）分类学 🔍

    *   **机会**：相对容易通过“装逼”和展示价值来吸引，因为她们判断力不如高分绿茶成熟。

*   **小绿茶/新进绿茶**：
    *   **特征**：刚步入社会，初尝现实滋味，开始学习打扮和利用性别优势，但还未深度世故。
    *   **心理**：有一定物质向往，但还保留部分单纯，容易对“未来憧憬”产生好感。
    *   **机会**：最容易攻克的一类，需要的基础价值不高，一些情感技巧和画饼就可能生效。

*   **真白富美 & 普通正妹**：
    *   **特征**：前者家境优渥，自带气质与阅历；后者颜值高但可能生活圈子普通。

**过渡**：分类帮助我们明确目标。接下来，我们将探讨无论针对哪类绿茶，都至关重要的吸引要素。
