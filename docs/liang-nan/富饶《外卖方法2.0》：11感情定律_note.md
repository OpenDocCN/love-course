## 什么是感情定律？

感情定律指出，女生喜欢你、想跟你在一起、爱你的这种感情，是由两种因素构成的：**吸引**和**安全感**。

上一节我们介绍了感情定律的基本概念，本节中我们来看看这两种因素的具体含义。

### 1. 吸引：价值的相互契合

吸引是指，你所匮乏的，恰巧是对方身上所需要的，那么你就会被对方深深吸引。

以下是关于吸引的几个关键点：

*   **男人的匮乏**：男人通常好面子，渴望找到一个漂亮的女朋友，或者在对性方面有匮乏，渴望身材好的伴侣。

**吸引的公式可以概括为：`吸引力 = 你提供的价值 恰好满足 对方的需求`**

### 2. 安全感：感性与专情的需求

安全感源于女生的感性和对感情的专情态度。

以下是关于安全感的几个关键点：

*   **女生的专情**：女生对待认真的感情会特别专一，喜欢一个人就会不搭理其他男人。

*   **安全感的必要性**：你必须让女生对你有安全感，才能成功邀约她。

## 吸引与安全感的组合关系

理解了两种因素后，我们来看看它们如何共同作用。

### 只有吸引，没有安全感

这种情况就像走在大街上，看到一个身材好、长得好看的女生，男人就想跟她有结果。但女人更注重过程感受。女生即使被男生的外貌吸引，也不会轻易跟他去宾馆，因为她不了解对方是好人还是坏人，担心对方不负责任或影响自己的风评。

**结论：只有吸引没有安全感，女生不可能跟男生在一起做某些事情。**

### 只有安全感，没有吸引

这种情况就像男生和女生是十年的同窗好友，彼此非常了解，很有安全感，但也不可能在一起。因为女生对男生没有感觉，缺少恋爱的亲密感和甜蜜感。这也解释了为何女生认为男女间有纯粹友谊，而男生认为没有。

**结论：只有安全感没有吸引，两人也无法成为恋人。**

## 实践应用：如何植入吸引与安全感

在与女生聊天时，必须同时植入这两个元素，并注意先后顺序：**先吸引，后补足安全感**。如果先建立安全感，很容易被发“好人卡”，就像很多兄弟犯的错误：夺命三连问，疯狂暴露需求感，导致关系停滞在朋友区。

### 第一步：如何制造吸引（展现价值）

制造吸引的核心是展现你的价值。价值分为三种：

以下是三种价值的具体内容：

1.  **生存价值**：带给女生良好的生存体验。例如：有钱、有权、有车、有房、有地位。
2.  **繁衍价值**：与基因和外表相关的吸引力。例如：身材好、个子高、长得帅、会打扮、有品位、气质好、说话好听。
3.  **内在价值（软价值）**：个人的内在品质。例如：孝顺、阳光开朗、幽默、风趣、成熟稳重、有上进心、懂女生。

**关键点**：你侧重展现什么价值，就会吸引到什么样的女生。展现生存价值可能吸引物质型女生；展现繁衍价值可能吸引看重外表的女生；展现内在价值可能吸引注重内在、适合过日子的女生。这对应了饶哥所说的“女人分类，对症下药”。
