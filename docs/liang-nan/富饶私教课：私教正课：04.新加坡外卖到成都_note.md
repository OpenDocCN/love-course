# 富饶私教课：私教正课：04.新加坡外卖到成都_note

在本节课中，我们将通过一个具体的案例，学习如何通过简洁高效的沟通，快速建立吸引并达成约会目标。整个过程将展示核心的聊天思路、框架保持以及临门一脚的操作。

## 概述

本节课我们将拆解一个从线上认识到线下邀约的完整案例。核心在于展示如何用极少的对话（约20句），引导女生主动跨越地理距离（从新加坡到成都）进行约会，并最终达成亲密关系。我们将重点关注聊天中的**态度传递**、**框架控制**和**预期铺垫**。

## 案例背景与开场

首先，我们看一下这个案例的留言。我在5月31号加到了这个女生。晚上10点22分，她给我发了一个表情。

上一节我们介绍了案例背景，本节中我们来看看如何开启对话并测试反应。

我回复了“麻了”。这里“麻了”是一个测试，目的是观察女生此时此刻对我的反应是什么样子。随后，在昨天晚上11点54分，我跟她进行了4分47秒的语音通话。

## 初步邀约与框架测试

在语音过程中，她告诉我她在夜场，在兰桂坊酒吧喝酒。我直接说：“喝完就来找我”。她说她要来找我。在晚上12点的时候，我跟她切了个语音，她再次表示要来找我。

以下是初步邀约的关键步骤：
1.  **直接提出要求**：我发了我家的位置，并问：“你知道这个地方吗？”
2.  **应对无回应**：她没有回复，我就在凌晨加了个问号，然后又跟她开了个语音（她在夜场听不到）。
3.  **施加压力**：我又打了个问号。她回复说还要再玩一会儿。
4.  **明确态度**：我问“多久”，并说“应该不过来了”。这时她说“应该不过来了”。

但是后面她又跟我语音了1分30秒。我说：“我不想让你走”。她说：“怎么可能”。我接着说：“你这样我就不过来了”。在语音电话里，我对她说：“你今天晚上不要回去了，跟我一起住。”

## 升级要求与服从测试

她说她今天晚上必须要回家，因为爸妈在等她。我继续升级要求：“我想搂着你”。她说：“你过来呀”。接着她说：“我想回家睡觉了，先来找我再回去”。

我给出了明确的指令和奖励：“我上车你打吧，来了给你报销快点”。然后她就打了车。我上车后，发消息问：“到哪了？快到了”。之后见面就操作了。

这就是一个完整的案例。没有聊任何多余的东西，这就是吸引，这就是把妹。这展示了我如何和女生沟通，大概只聊了20句话。

通常来讲，我不会跟女生聊很多东西。所以这就是课程的核心：20句话，没有聊很多，直接一个语音让她来我家找我。这让她放弃了和小哥哥玩的机会。

## 核心心态与态度

这个女生长得很普通。到家之后，直接带回卧室，没有任何多余操作。

但是，这不是重点，因为很简单，我把妹都是这样的，啥也不说。然而，昨天我约了一个我很喜欢的妞，但没有成功。这个妹子非常好看。

她跟我说要跟我喝酒。我们吃了面之后，她说：“你要不要去喝酒？” 我说：“不喝”。她说：“那你要不要去看电影？” 我说：“都看过了”。她说：“那你跟我回家？” 我说：“为什么不回？” 她说：“我朋友喊我去喝酒，下次我们再去喝，今天先跟我回家。” 我说：“不要，你滚吧。”

然后，我就在旁边一直不说什么，玩手机。她说：“你怎么不说话，你是不是生气了？” 我说：“我不想理你”。然后她说：“你到底怎么说？” 我说：“我不想跟你说话”。然后她就走了，删了。

这是什么态度？你跟我回家，你不跟我回去。我想让你今天晚上陪我，你不跟我在一起，那么就只能拜拜，只能再见。

我不管你长得有多好看，不管你是什么样的人，只要你不听我的，那么直接拜拜。因为很简单，因为我女人足够多，我选择足够多，我没有必要在你身上浪费时间。你跟我装逼，老子比你更会装逼。你有态度，我比你更会有态度。

## 具体操作技巧

很多兄弟问怎么操作。我告诉你怎么操作。当女生跟你装逼的时候，比如她坐在我旁边了，我想让她跟我回去。

我会说：“亲爱的，我想告诉你一件事情”。女生会问：“什么事情？” 我说：“我想跟你一起那个”。如果女生说：“不好吧”、“不行”。那么我会很严肃地看着她，让她感觉她做错了，问：“为什么？”

如果这个时候她拒绝，你就把你手机拿出来，拍着她说：“那你走吧”。这个时候拿出手机开始约其他女生。你说：“好吧，我尊重你。”

很多兄弟呢，他只学到壳，还学不到魂。你看到我讲这个，很多兄弟就没有在听了，就觉得：“哇，不是我想要的，我想要的是某个技巧，某个聊天。” 那只能说你确实功力还不到。你没有一个男人的态度。

## 资本的建立

昨天我又聊了哪些妹子？这就是资本。这就是为什么我不搭理这些人的资本。为什么我有太多资本？就是这样。从来不理傻逼。

像这种消息：“明天几点？” “三点。” “晚点吧。” 然后不理。“你还没有起来吗？” “早就起了。” “那你不回我？” “很拽吗小朋友，不说话是吧？” “在忙呢，宝贝。”

这才是聊天，这才是聊妹的核心。就是你对待女生永远要有一种很炫酷的、不拘一格的态度，而不是处心积虑地去想我要怎么搞定这个女生。

因为你们还是现在处于这个阶段。你要想办法把自己变成我这样。为什么我可以这么多呢？因为第一个，我选择多，就是女人多。很多时候，你们聊不到女生的原因是你选择不多。因为你失去这个女生，你又没有更好的选择了，所以你就只能思考这一个，你就觉得把她当作最喜欢的、命中注定的，你感觉你就要和她一直在一起。那么最后呢，就只能造成这个女生觉得你需求感太强了，她觉得你这个男的很舔。

但是呢，她当和我聊天，我不怎么甩她，她觉得：“哇，这个男生很有标准，他也不会轻易地和其他女人这样说话，所以这种男人才是我想要的男人。” 我就是这样对待女人的。

最近，给大家分享一下这个案例吧。这个女生呢，是一个28岁的隔壁邻居的小少妇，非常精致，非常舒服。

首先，全程聊天记录给大家看。我跟她第一次见面就直接已经去她家了。

大家可以看到我前面四句话：第一句“我叫傅瑶”，第二句“你叫什么”，第三句“你名字备注我在心里了”，第四句回应质疑“谁说的”。就这样聊天。
