那么接下来重点就来了。既然我们知道妹子是这么现实的，那么我们怎么在这么现实的环境下，去泡到正妹呢？我们怎么突破或者怎么避免“门票”的问题呢？这个才是我们真正的重点。

好，你先明白这个点的时候，你会发现，你在泡绿茶的时候，这个点就解决了很多你泡不到正妹的问题。比如，很多人技术很好，懂得把妹要展示价值，懂得要推拉，懂得搞这些乱七八糟的东西。但你会发现，他永远泡不到正妹，只能泡到一些普通的四五分妹子。原因是什么？

因为正妹和普通妹子有一个东西是反过来的：**吸引力和台阶是反过来的。** 对于正妹来说是台阶的东西，对普通妹就是吸引。对于普通妹是吸引的东西，对于正妹就是台阶。
