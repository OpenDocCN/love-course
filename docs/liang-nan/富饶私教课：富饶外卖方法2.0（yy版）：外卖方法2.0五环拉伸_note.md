## 一、 课程主题：拉伸关系基本功

上一节我们介绍了吸引女生的各种套路，本节中我们来看看如何将吸引转化为实际的关系推进，即“拉伸”。

外卖方法拉伸关系基本功，是本堂课的核心主题。

首先，我们需要澄清关于拉伸关系的常见误区。普通男人在努力与女生互动时，其根本目的是向女生展示：**我是一个好男人，我是安全的，所以你应该出来和我约会**。他们试图证明自己正常、正经、无害，以此吸引女生。

但我们的外卖方法完全不同。我们的目标是在聊天过程中，向女生展示：**我就不是一个正经人，我见面后不会只是看电影、吃饭就结束**。

## 二、 错误 vs. 正确的拉伸思维

要理解如何做到这一点，首先必须回顾并对比两种截然不同的思维模式。

### 1. 传统错误的拉伸思维

以下是正常男人典型的错误拉伸思维：

*   **核心模式**：**先拉伸，然后等待同意**。
*   **具体表现**：他们通常会先表白（如“我觉得我还蛮喜欢你的”），然后提出约会请求（如“我们能不能一起出来约个会呀”）。
*   **本质问题**：在这种模式下，男人在说出拉伸话语时，心里是没底的，完全不知道对方是否会同意。这就像一场**赌博**，成功率只有50%。女生可能因为恰好也喜欢你而同意，你就会觉得聊天顺利；也可能拒绝你，你就会归咎于自己技术不行或女生不喜欢你。**最大的错误在于，你在不知道女生能否接受的情况下，就盲目发起了冲锋**。

### 2. 正确的拉伸思维

正确的拉伸思维模式是：

*   **核心模式**：**先获得同意，再进行拉伸**。
*   **核心公式**：`同意 + 拉伸`
*   **本质**：拉伸关系的话语，**不是发起战争的冲锋号，而是宣告胜利的凯歌**。这意味着，在你进行微信上的“拉伸”动作之前，你和女生的关系实质上已经达到了那一步。你的话语只是对已建立关系的确认和形式化表达。

**举例说明**：
*   **错误方式（先拉伸，后等待）**：直接说“以后每天晚上我跟你说晚安，你跟我说早安，好不好？”。这是在等待女生说“好”或“不好”。
*   **正确方式（先同意，后拉伸）**：先铺垫说“我想每天跟你说晚安”。如果女生回应积极（如说“好”），这等于她**同意**了这种暧昧状态。然后你再说“那你负责说早安”，这就是在同意基础上的**拉伸**。又或者，你可以先测试：“如果我们见面，我忍不住傻笑，你会害羞吗？”。如果女生说“可能会吧”，这表示她**同意**进入暧昧设想，此时你再说“那我们一起手拉手看电影”，就是顺理成章的**拉伸**。

**关键理解**：你并非靠某一句“拉伸关系的话”搞定女生，而是在说这句话**之前**，就已经通过互动让她进入了愿意接受拉伸的状态。这句话只是临门一脚。

## 三、 拉伸前的关键：测试

理解了正确思维后，我们进入实操环节。在正式拉伸关系之前，必须进行测试，就像发射火箭前需要多次检查一样。

这部分称为：**外卖方法拉伸关系前测试**，分为小动作测试和大动作测试。

### 1. 小动作测试

小动作测试是风险最低、最基础的测试，用于初步试探和升温关系。它分为三轮，必须逐级通过。

**第一轮：称呼测试**
这是最稳妥、即使被拒绝也不会“死掉”的升温方式。通过改变称呼来试探。
以下是常见的称呼测试列表：
*   宝贝 / 小可爱
*   小姐姐 / 小仙女
*   丫头
*   小蠢猪
*   小胖子
*   宝宝
*   么么哒
*   小懒猫
*   小朋友

**使用方法**：将称呼自然融入对话。例如：“你吃饭了吗，么么哒”、“我刚刚健完身，小可爱你在干嘛呢？”。

**第二轮：互说早安晚安测试**
当称呼测试通过，关系更近一步后，可以进行此轮测试。它比称呼更深入，但仍在朋友可做范围内。
以下是具体话术示例：
*   “以后每天我给你说晚安，你负责早安，好不好？”
*   “我明天早上有个重要会议，怕起不来，你要不要明早打电话叫我起床？”
*   “我们互说一周早安晚安，谁忘了就请对方吃一周苹果。”

**第三轮：引导暧昧话题测试**
当前两轮测试均通过，即可引导更具画面感和私密性的话题，进一步拉近关系。
以下是三个最容易引发暧昧联想的话题：
1.  **做饭**：描绘共同做饭的场景，如“你买菜，我做饭”、“你从后面抱着我，我炒菜”。（**注意**：这只是聊天测试工具，并非真要做饭。如果不会，后续有解决方案。）
2.  **自驾游**：聊共同自驾去丽江、大理等地的幻想，描绘蓝天白云、自由浪漫的二人世界画面。
3.  **旅游**：谈论一起去巴厘岛、泰国等地的旅行幻想，描绘在落地窗前、海边等场景的浪漫画面。

**核心要点**：聊这些话题的目的，是测试女生愿意在想象中与你进入到多亲密的地步，为后续“大动作”铺路。

## 四、 核心模型：大动作拉伸五环

当小动作测试全部通过，你和女生已处于暧昧状态时，便进入最关键的大动作测试环节。

大动作测试的核心是：**将你想和女生做的所有事情，按照由浅入深的顺序排列出来，然后逐环测试、推进**。这套模型能系统性地将关系拉伸到最后一步。

**大动作拉伸关系五环模型**（由浅入深）：
1.  **建立好感与共鸣**：在微信上聊得开心，确认是一类人，互相有好感。
2.  **提议见面约会**：将线上好感转化为线下约会的提议。
3.  **描绘私密互动场景**：跳过普通吃饭看电影，直接描绘更私密、有趣的场景，例如：
    *   一起手拉手看电影。
    *   看完电影手拉手去买菜。
    *   回家一起做饭（你从后面抱着我）。
    *   饭后在家看电影（我肩膀给你靠）。
    *   在阳台拥抱、抚摸。
4.  **引入肢体接触与亲密暗示**：
    *   询问“如果看电影时忍不住亲你，你会打我吗？”
    *   玩笑式威胁“你要敢欺负我，我就脱你一件衣服”。
    *   提出“回家必须先洗澡才能上床”等带有同居暗示的话。
5.  **明确表达喜欢与最终意图**：
    *   直接说“其实说了这么多，是因为我还蛮喜欢你的”。
    *   询问偏好“你喜欢浪漫的法式舌吻，还是蜻蜓点水的亲吻？”
    *   最终暗示“如果开心，我们就为爱鼓掌”。

**如何使用该模型**：你需要用聊天技巧，从第一环自然过渡到第五环。每一环都是对下一环的测试，如果女生在当前环节回应积极，你就可以尝试推进到下一环。
