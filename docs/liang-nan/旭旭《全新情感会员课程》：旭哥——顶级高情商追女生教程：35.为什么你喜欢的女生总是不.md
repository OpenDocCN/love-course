## 第一步：面部精修（Facetune2）

修图的第一步是对面部进行自然、精细的调整。目标是提升颜值，同时避免“见光死”（照片与真人差异过大）。

### 核心修图原则：只修小，不修大

修图时，切忌对某个部位进行大幅度、不自然的推拉。正确的方法是，在多个小细节上进行微调，累积起来实现整体变帅的效果。例如，不要直接将圆脸推成瓜子脸，而是分别微调下巴、颧骨、腮帮等部位。

以下是面部需要重点修饰的几个板块：
1.  **脸型轮廓**：重点调整下巴的线条，使其更清晰。
2.  **颧骨与腮帮**：如果拍照显胖，可轻微向内收敛，使脸型更立体。
3.  **苹果肌区域**：适当修饰，使面部线条流畅。

**操作代码示例（理念）：**
```
# 错误示范：大幅度单一调整
face.push(chin, -50) # 下巴直接收缩50个单位，导致不自然

# 正确示范：多部位微调
face.push(chin, -5)
face.push(cheekbone_left, -3)
face.push(cheekbone_right, -3)
face.push(jaw_left, -4)
face.push(jaw_right, -4)
# 累积效果自然提升
```

### 五官调整技巧

调整完脸型后，接下来是五官的精细化处理。

1.  **眉毛**：调整眉形，避免过于上挑显得凶悍，可适当拉平或微调长度。
2.  **眼睛**：如果眼睛较小，可适度放大。同时使用软件的“细节”或“亮眼”功能，涂抹眼球，使眼睛更有神。
3.  **鼻子**：许多人的鼻子略有歪斜，可放大后仔细调整鼻梁，使其看起来更端正。鼻翼过大也可适当缩小。
4.  **嘴巴与额头**：嘴巴一般无需大动。额头如果过高或过低，可轻微推拉发际线进行调整。

**注意事项**：修图时注意背景线条（如墙壁、门窗）是否因推拉脸部而变形，如有变形需修正，以免暴露修图痕迹。

### 皮肤处理（磨皮）

切勿使用一键磨皮功能，这会让照片模糊、质感尽失。应该使用Facetune2中的“磨皮”工具，手动在皮肤区域轻轻涂抹，避开五官、毛发和衣物纹理。这样既能改善肤质，又能保留真实的细节和立体感。

## 第二步：滤镜与色调调整（Lightroom/VSCO）

面部修整完成后，接下来是为照片注入风格和质感的关键步骤——调色。

1.  **使用预设滤镜**：可以使用Lightroom、VSCO或Instagram内置的优质滤镜快速改变照片基调。不同的滤镜能营造出复古、电影感、清新等不同风格。
2.  **自定义调色（高阶）**：使用Lightroom可以更精细地控制色彩。你可以调整：
    *   **高光/阴影**：改变明暗部的色调。
    *   **HSL（色相/饱和度/明亮度）**：单独调整画面中特定颜色（如草地绿、天空蓝、衣服颜色）的属性和饱和度。
    *   **色调分离**：为高光和阴影分别加入不同的颜色，营造独特的色彩风格。

**效果对比公式**：
**原图（灰暗普通） + 滤镜（风格化） + 精细调色（质感提升） = 高质量成片**

## 第三步：环境优化（Quickshot）

如果拍摄时天气不好，背景天空灰蒙蒙，可以使用Quickshot这款软件一键更换天空。选择与场景匹配的天空素材（如晚霞、蓝天白云），并调整融合度和亮度，能让平淡的照片瞬间拥有大片感。
