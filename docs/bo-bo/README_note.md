# 波波情感教程 01：情感分析基础 🧠

在本节课中，我们将要学习情感分析的基本概念。情感分析是自然语言处理领域的一个重要分支，它旨在通过计算模型识别和提取文本中的主观情感信息，例如判断一段文字表达的是积极、消极还是中性的情绪。

## 什么是情感分析？🤔

上一节我们介绍了本课程的主题，本节中我们来看看情感分析的具体定义。情感分析，有时也称为观点挖掘，是指利用自然语言处理、文本分析、计算语言学等技术，系统地识别、提取、量化和研究文本中的情感状态和主观信息。

其核心目标是让计算机能够理解人类语言中蕴含的情感倾向。一个简单的例子是，对于评论“这部电影太精彩了！”，我们希望模型能判断其为**积极**情感。

## 核心概念与公式 📊

理解了定义后，我们需要掌握几个核心的分析维度。情感分析通常从以下几个层面进行：

以下是情感分析的三个主要层面：

1.  **情感极性**：这是最基本的情感分类，通常分为**积极**、**消极**和**中性**三类。可以用一个简单的函数表示：
    `sentiment(text) -> {POSITIVE, NEGATIVE, NEUTRAL}`

2.  **情感强度**：不仅判断极性，还衡量情感的强烈程度。例如，“好”和“非常好”都是积极评价，但强度不同。这可以通过情感得分来量化，例如在-1（极度消极）到+1（极度积极）的区间内取值。

3.  **情感对象**：识别情感所针对的具体实体或方面。例如，在评论“相机画质很棒，但电池续航太差”中，对“画质”的情感是积极的，而对“电池续航”的情感是消极的。

## 基础方法简介 ⚙️

从概念到实践，本节将介绍两种实现情感分析的经典方法。主要可以分为基于词典的方法和基于机器学习的方法。

以下是两种基础方法的简要说明：

*   **基于词典的方法**：
    这种方法依赖于一个预先构建好的“情感词典”，其中包含了大量单词及其对应的情感极性（如“快乐”=积极，“悲伤”=消极）和强度分数。分析时，通过查找文本中单词在词典中的得分，并按照一定规则（如求和、平均）汇总，得到整段文本的情感分数。
    其核心过程可以概括为：`文本 -> 分词 -> 查词典获取单词情感分 -> 聚合计算 -> 最终情感分`。

*   **基于机器学习的方法**：
    这种方法将情感分析视为一个分类或回归问题。首先需要收集已标注好情感标签的文本数据（训练集），然后从文本中提取特征（如词频、n-gram），最后使用分类算法（如朴素贝叶斯、支持向量机或深度学习模型）进行训练，从而构建一个预测模型。
    一个简单的流程公式是：`已标注文本 + 特征提取 + 机器学习算法 = 情感预测模型`。

## 总结 📝

本节课中我们一起学习了情感分析的基础知识。我们首先了解了情感分析的定义与目标，即让计算机理解文本中的情感。接着，我们探讨了情感极性、强度和对象这三个核心分析维度。最后，我们简要介绍了基于情感词典和基于机器学习这两种实现情感分析的基础路径。掌握这些概念是进一步学习复杂情感分析模型和应用的第一步。
