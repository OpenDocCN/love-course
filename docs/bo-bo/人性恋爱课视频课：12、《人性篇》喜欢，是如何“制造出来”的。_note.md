# 人性恋爱课 视频课：12、《人性篇》喜欢，是如何“制造出来”的

## 概述
在本节课中，我们将探讨人的喜欢是如何产生的，以及一系列心理活动和情绪变化是如何演变的。

## 喜欢的产生
**公式**：喜欢 = 在乎 + 情感 + 情绪

1. **在乎的启动**：喜欢一个人，首先是因为在乎。在乎的产生源于不习惯，即某个模式被打破，导致不习惯。
2. **习惯的形成**：通过连续的行为，如每天送食物，可以形成习惯。**代码**：连续行为 → 习惯形成
3. **习惯的打破**：当习惯被打破时，如停止送食物，会导致不习惯，进而产生在乎。

## 不习惯与在乎
1. **不习惯的产生**：不习惯源于习惯，即习惯被打破。
2. **在乎的启动**：不习惯会启动内心的在乎，通过胡思乱想和自我暗示。

## 人性的原理
1. **存在感的投射**：当一个人突然消失时，另一个人会投射出这个人的存在感。
2. **习惯的制造与消失**：通过制造习惯和打破习惯，可以制造心灵空虚，进而产生在乎感。

## 案例分析
1. **聊天习惯的建立**：每天晚上固定时间聊天，形成习惯。
2. **习惯的打破**：突然停止聊天，打破习惯，产生不习惯。
3. **在乎感的产生**：不习惯导致在乎感，进而产生喜欢。

## 总结
本节课中，我们一起学习了喜欢是如何产生的，以及习惯、不习惯和在乎在其中的作用。希望这些内容能帮助你更好地理解人性，以及如何通过人性来制造喜欢。
