# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要一个地方来存放它，这就是**变量**。同时，数据本身有不同的种类，比如数字和文字，这些种类被称为**数据类型**。掌握如何创建变量并了解不同的数据类型，是开启编程之旅的关键。

## 1. 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）用来存放数据，而标签（变量名）则帮助我们记住盒子里装的是什么。在Python中，创建一个变量并给它赋值非常简单。

上一节我们介绍了变量的比喻，本节中我们来看看如何在代码中具体操作。

**核心操作：变量赋值**
其基本公式是：
`变量名 = 值`

例如，我们创建一个名为 `message` 的变量，并在其中存放一段文字：
```python
message = "Hello, Python!"
```
这行代码的意思是：将字符串 `"Hello, Python!"` 赋值给变量 `message`。现在，`message` 这个“盒子”里就装好了这段问候语。

## 2. 基本数据类型 🔢

数据并非只有一种形式。Python有几种内置的基本数据类型，用于表示不同种类的数据。了解它们有助于我们正确地使用和操作数据。

以下是三种最常用的基本数据类型：

*   **整数 (int)**：表示没有小数部分的数字，如 `-5`, `0`, `100`。
*   **浮点数 (float)**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str)**：表示文本信息，需要用单引号 `' '` 或双引号 `" "` 括起来，如 `"你好"`, `'abc123'`。

我们可以使用 `type()` 函数来查看任何变量或值的数据类型。例如：
```python
age = 25
print(type(age))  # 输出：<class 'int'>

price = 19.99
print(type(price)) # 输出：<class 'float'>

name = "李越"
print(type(name))  # 输出：<class 'str'>
```

## 3. 变量的命名规则与规范 🏷️

为了写出清晰、易读的代码，给变量起一个好名字至关重要。这需要遵循一些规则和最佳实践。

上一节我们了解了数据的类型，本节中我们来看看如何给存放数据的“盒子”起一个合适的名字。

以下是命名变量时需要遵守的规则和建议：

*   **硬性规则（必须遵守）**：
    *   变量名只能包含字母、数字和下划线（`_`）。
    *   变量名不能以数字开头。
    *   变量名不能是Python的**关键字**（如 `if`, `for`, `while` 等）。
*   **软性规范（建议遵守）**：
    *   使用描述性的名称，让人一眼就能明白变量的用途（例如，用 `student_name` 而不是 `sn`）。
    *   对于多个单词组成的变量名，推荐使用**蛇形命名法**：用小写字母，单词之间用下划线连接（例如：`user_age`, `total_count`）。

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。
