# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如数字、文字）。你给盒子起的名字就是**变量名**，而盒子里装的东西就是**变量值**。

在Python中，你使用等号 `=` 来给变量赋值。其核心操作可以表示为以下公式：

**变量名 = 变量值**

例如，我们写一行代码：

```python
name = "宇晨"
age = 25
```

这里，`name` 和 `age` 就是变量名，`"宇晨"` 和 `25` 就是存储在变量中的值。

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道盒子里装的是哪种类型的东西。Python中几种最常见的数据类型如下：

以下是几种你必须掌握的基本数据类型：

1.  **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
2.  **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
3.  **字符串**：由字符组成的文本，需要用单引号 `' '` 或双引号 `" "` 括起来，例如 `"你好"`, `'Python'`。
4.  **布尔值**：只有两种值，`True`（真）或 `False`（假），常用于逻辑判断。

你可以使用 `type()` 函数来查看任何变量或值的数据类型。例如：

```python
print(type(10))      # 输出：<class 'int'>
print(type(3.14))    # 输出：<class 'float'>
print(type("宇晨"))  # 输出：<class 'str'>
print(type(True))    # 输出：<class 'bool'>
```

## 变量的命名与使用规则 📝

了解了数据类型后，我们还需要知道如何正确地给变量起名字。以下是给变量命名时需要遵守的一些重要规则：

1.  变量名只能包含字母、数字和下划线。
2.  变量名不能以数字开头。
3.  变量名是区分大小写的（例如 `age` 和 `Age` 是两个不同的变量）。
4.  不能使用Python的保留关键字（如 `if`, `for`, `while`）作为变量名。
5.  命名应尽量清晰、有意义，通常使用小写字母和下划线组合，如 `user_name`。

> **小提示**：使用有意义的变量名能让你的代码更容易被自己和他人理解。

## 一个简单的实践示例 💻

现在，让我们把以上知识结合起来，写一个简单的程序。

```python
# 定义几个变量
user_name = "宇晨"
user_age = 25
is_student = True

# 打印这些变量及其类型
print("姓名：", user_name)
print("年龄：", user_age)
print("是否是学生：", is_student)

print("数据类型检查：")
print("user_name 的类型是：", type(user_name))
print("user_age 的类型是：", type(user_age))
print("is_student 的类型是：", type(is_student))
```

运行这段代码，你将在屏幕上看到变量的值和它们对应的数据类型。
