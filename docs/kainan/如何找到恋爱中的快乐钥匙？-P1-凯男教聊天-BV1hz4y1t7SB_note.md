# 恋爱中的快乐钥匙 🔑 - 课程 P1

在本节课中，我们将探讨一个普遍现象：为何在物质丰富的今天，我们的幸福感却似乎在流失。我们将从心理学和行为学的角度，分析影响幸福感的几个关键因素，并为你提供四把实用的“快乐钥匙”。

不知道大家有没有发现一个很奇怪的现象。当我们看电视、逛街或看电影时，如果偶然听到一首喜欢的歌曲，我们会很耐心地听完。但如果这首歌是从自己手机里的App播放出来的，喜爱感就没那么强烈，甚至可能会切掉。我们总在App里收藏许多喜欢的歌曲，但每次只听固定的几首。即便如此，这种感觉也比不上在现场听歌手演唱，或在出租车里听到广播播放时所带来的开心和幸福感。每当这时，我就不禁感叹。

为什么现在我们吃饱穿暖，娱乐节目众多，物质条件远胜几十年前，但幸福感却在逐渐流失？为什么大多数人在物质丰裕的时代会感到不幸福？到底什么才是获得幸福和快乐的钥匙？我们要怎么做才能更快乐、更幸福？今天我将通过一个视频给大家讲明白。我们先从原因入手。

## 第一把钥匙：降低预期 🔑

我们为什么会对偶然听到的歌曲感到特别开心？其本质在于，**预期与体验的差值决定了我们的幸福感高低**。

用公式表示就是：
**幸福感 = 实际体验 - 事前预期**

逛街时，听到歌曲的预期是零，而体验是正值，因此差值带来幸福感。自己从手机播放音乐时，预期已是正值，即使体验很好，差值也可能为零，幸福感因此降低。

这一点可以从度假旅游中获得证明。如果你是冲着网红打卡地去的，到达后心里有落差，大概不会快乐。但如果你通过自驾游去探索和发现，而不是考核终点是否符合预期，收获的快乐可能会更多。

钓鱼也是如此。如果钓鱼前就打算要钓多大的鱼，整个过程你就是在用预期考核结果。尽管有快乐，但快乐也仅限于此。

所以，让我们幸福和快乐的第一把钥匙就是**降低预期**，尤其是降低对他人的预期。

## 第二把钥匙：掌控自己的幸福 🧭

降低对别人的预期，引出了我们幸福的第二把钥匙。很多人存在一个问题：**把自己的幸福建立在别人身上**。

例如：“我最大的愿望就是我的另一半可以一直对我好”、“我一生中最大的愿望就是我的孩子可以考上重点大学”、“如果老板今天给我涨工资就好了”。把所有幸福感都建立在别人身上，相当于主动放弃了掌控自己幸福的权利，而别人是不可控的。

哈佛大学2003年的一项调查报告指出，现代社会决定我们幸福的最大因素就是人际关系。你在人际关系里收获不到快乐的一个很大原因，就是对别人的期望过高。别人的起起伏伏会导致你的幸福感波动，这其实是一件很愚蠢的事。

你的父母对你糟糕、你的另一半喜欢上了别人、你的好友与你形同陌路，这些事确实令人难过。但你还是你，你的价值没有变。你可以挣钱养活自己，可以环游世界，可以吃遍天下美食，赏尽世间美景。你将来会找到新的朋友、更好的伴侣。这一切都跟别人无关。这就是第二把幸福的钥匙：**掌控自己的幸福**。

## 第三把钥匙：调整对比体系 ⚖️

然而，我又不禁思考，为什么我们当代人预期这么高？小时候一根棒棒糖就能开心一整天，现在即便吃喝不愁，却依然焦虑。于是，我思考出了第三把钥匙，这也是影响幸福感的最重要因素之一：**你的对比体系出了问题**。

我们是最喜欢横向对比而忽略纵向对比的生物。绝大多数人不管取得多大成绩，都不愿与几年前的自己对比，而喜欢与周围的人、同龄人、朋友、同事对比。所以很多人不幸福，是因为他们追求的不是幸福，而是**比别人更幸福**。

人的际遇和环境总在变化。比如我的一位兄弟，中学时觉得自己是“神”，到了大学发现周围都是学霸、省市状元，便觉得自己是“渣渣”。后来出国留学，周围又出现了天才和连续创业者，他再次感到挫败。但如今他意识到，自己不论能力、生活还是经济状况，都比过去的自己进步了许多。

所以，让我们幸福的第三把钥匙就是**调整对比体系**，从“比别人幸福”换成“让今天的自己比昨天更幸福”。

## 第四把钥匙：打破“金钱至上”的内卷 💸

调整了对比体系，但为什么还是很难开心起来？工作生活稳定，周围人都羡慕，薪水可观，工作体面，但依然感觉不幸福？这就引出了第四点：**金钱至上的物质主义**。

物质主义比较自私自利。一味追求物质，会导致我们只关注财富等外在目标，而忽略了自我成长和人际关系等内在目标，情绪很容易受外在事物影响。

我举自己的例子。大学毕业后有段时间我走火入魔，特别热衷于搞钱，用钱衡量一切。这导致我内心每天都很焦虑，觉得不挣够一定数额就是浪费时光。最可怕的是，我的人际关系变得非常差，因为我认为任何与挣钱无关的事都是在浪费时间。你觉得这种状态能幸福吗？

我们常说的“内卷”，说白了就是一种金钱至上的物质主义游戏。从小卷到大学，再卷到工作，为了多挣钱在游戏里无休止地竞争，你觉得会幸福吗？

所以，第四把钥匙就是**打破内卷，收获幸福**。这需要你摆脱大众长久以来“金钱至上”的价值观。注意，这里不是让你躺平，而是不以钱多钱少来衡量成功与失败，或至少不以它作为唯一度量标准。这样，你可能会幸福很多。

---

本节课中，我们一起学习了获得快乐与幸福的四把钥匙：

1.  **降低预期**：幸福感源于体验与预期的差值。
2.  **掌控自己的幸福**：不将幸福完全寄托于他人。
3.  **调整对比体系**：与过去的自己比进步，而非与他人比高低。
4.  **打破“金钱至上”的内卷**：关注内在成长，丰富人生体验。

快乐的钥匙是什么？就是放弃一定的预期，减少对他人的预期；把对比的聚光灯放在自己身上；不以成败论英雄，不以钱多钱少量喜悲；将关注点更多地交给探索与发现的过程，关注当下，少一些目的性，多一些探索心。

在生活中，少一些期待，相信下一首歌可能更好听，下一顿饭可能更好吃。与喜欢的人相处，少做对比，多去观察和体会他做了什么。在工作中，关注你收获了什么，而不仅仅是应得多少利益。

让更多惊喜进来，让自己的内心平稳。这就是快乐之上的钥匙。
