# 聊天大师课 01：掌握情绪价值，成为高情商男人 🧠💬

在本节课中，我们将要学习**情绪价值**的核心概念及其在人际交往，尤其是与异性互动中的巨大作用。我们将从定义出发，层层深入，最终掌握提升自身情绪价值的六大实用步骤。理解并运用情绪价值，能让你在沟通中充满自信与魅力，真正影响他人。

## 什么是情绪价值？

上一节我们概述了课程目标，本节中我们来看看情绪价值的定义。

情绪价值是一个人对他人情绪的影响能力。

例如：
*   你工作受委屈了，找朋友倾诉。朋友说：“你老板真不是人，给你这么点钱让你干这么多活。” 你听后心里舒服多了。这说明，在这次互动中，朋友提供了高情绪价值，你成功索取到了情绪价值。
*   同样的情况，朋友说：“你说这些没用，你为什么不把工作做好让你老板无话可说呢？” 你听后心里更堵了。这说明，朋友提供的情绪价值较低，你索取情绪价值失败了。

从以上案例可以体会到，一个情绪价值高的人，能够让别人产生情绪上的依赖和信任，因为这类人能让别人感觉舒服。在长期关系中，情绪价值是至关重要的能力。一段感情可能因颜值、收入等要素开始，但往往终结于“不舒服”。因此，提供情绪价值的能力至关重要。

很多人误以为情绪价值就是幽默、附和。其实这是错误的，这样做最多只提供了“情绪”，没有“价值”。提供情绪价值的前提，是自己首先要处在**有价值的情绪状态**中。当你自身情绪状态好，对方就会被你感染，从而改变情绪。这种被感染的情绪才是真正有价值的。

举例来说：
*   观看歌手现场演出，歌手饱满的情绪状态能感染并调动全场观众的情绪。
*   面对沉稳的父母或前辈，你焦躁的情绪会因为他们而稳定下来。

这就是真正的情绪价值。拥有强大的情绪价值，你甚至无需刻意做什么，对方就能感受到。反之，如果你状态不好，无论怎么解释或试图幽默，对方也只能感受到局促和尴尬，这绝非有价值的情绪。

## 提升情绪价值的六大步骤 🛠️

理解了情绪价值的本质后，我们来看看如何系统地提升它。以下是六个由浅入深的步骤。

### 第一步：感知他人情绪

很多人在互动中对对方的情绪变化反应迟钝，这常被误解为“钢铁直男”。以下是精准捕捉对方情绪的三个要点：

1.  **解读语言背后的潜台词**
    *   **转折词**：如“不过”、“但是”。关注点应放在这些词之后的内容，那才是对方真正想表达的重点。
    *   **语气词**：如“哎”、“啊”、“呀”，代表不同的喜怒哀乐。
    *   **表情与标点符号**：需结合具体情境综合判断。

2.  **观察肢体动作的变化**
    *   例如，一个平时温柔的人突然变得急躁，这种行为变化的背后必有原因。当不确定对方情绪时，可以直接、温和地询问：“你是有不开心的事吗？”或“感觉你情绪好像有点不太对？”

3.  **多用开放式提问**
    *   目的是让对方袒露自己。将自己的价值观暂时搁置，不仅能看见对方的真实情绪，还能通过回答识别其价值观，从而给予对方真正想要的反馈。

### 第二步：关注聊天中的回应

我们思考一个问题：“我要吃饭了”和“我要去吃饭了”最大的差别是什么？
差别在于“去”这个动作。前者通常表示想结束话题，后者则可能想开启关于吃饭的话题，或暗示“回来再聊”。

不关注对方回应，就会错过大量信息。聊天高手能快速解读信息，深挖其价值。这种能力需要你与对方共情，才能抓住关系升温的瞬间和机会转折点。

### 第三步：减少负面反馈

在学习提供价值前，先抑制制造负面情绪的习惯。请避免以下错误表达方式：

以下是常见的负面反馈模式，请尽快纠正：
*   **指责**：“都是你的错！”
*   **打压**：“你什么都不行。”
*   **负面预判**：“按我对你的了解，你去做一定没好结果。”
*   **恶意揣测**：“你是不是故意气我？”
*   **阴阳怪气**：“不会吧，不会真的有人这样吧？”

如果你的表达习惯中有上述几项，请尽快纠正。

### 第四步：增加积极反馈

说完反例，我们来看正例。如何反馈才能提供情绪价值？

以下是给予积极反馈的几种方式：
*   **积极关注**：“你这件衣服今天真好看！”、“我觉得你的报告写得挺好呀。”
*   **赞美细节**：“这衣服的款式和布料感觉很高档，真羡慕。”
*   **用正面词汇表达期望**：“我喜欢……如果能……我会更开心。”
*   **提供建设性意见**：“你再多考虑这一点，想法会更容易落实。”、“这个方案的不足是……需要调整……来规避风险。”

这些积极的反馈方式，能将对方的注意力引向积极事物。认知影响情绪，这本身就是在提供高质量的情绪互动。

### 第五步：给予认同感

现代人普遍缺乏存在感和认同感，即缺乏展示自己的舞台和对其高光时刻的聚焦。

例如，你完成一个复杂任务后：
*   老板只关心其他项目进度。
*   同事敷衍后请你帮忙。
*   伴侣只顾着看晚上吃什么。
*   父母听不懂只催你回家。

你会感到挫败和虚无。那么，如何给予认同感呢？
*   **老板**：关心项目细节和你的关键思考，并诚恳地说：“项目干得漂亮，辛苦你了。”
*   **同事**：询问细节后表示羡慕，并说：“你一定要让我请吃饭，我想请教你。”
*   **伴侣**：放下手头事，努力理解项目有多难，然后说：“虽然不太懂，但明白你果然是最棒的。”
*   **父母**：直接说：“这些我听不懂，不过我的孩子怎么可能差。”

从这些例子可以直观看出给予认同感的方法：

以下是给予认同感的两大要点：
1.  **询问更多细节**：“这是什么意思？”、“怎么办到的？”、“这里你为什么这样做？”
2.  **根据理解程度和关系亲疏，给予正面反馈**。

认同感是稀缺资源。拥有制造“舞台”的能力，很多人会为了“登台表演”而与你进行价值互换。

### 第六步：培养“允许”的心态

以上五步是“术”，而“道”的层面，最大的情绪价值是 **“允许”的能力**。

允许什么呢？允许外界的人、事、物不按你的预期发展。
*   允许约会迟到。
*   允许父母想干涉你的感情。
*   允许公司存在不公。

你可能会觉得这是软弱。但之所以要“允许”，是因为无论你是否允许，事情已经发生。你的负面情绪没有意义。你可以用理智去讨论、商议（如下次能否准时、对感情的看法、工资结算方法），但情绪越多反而坏事。

人生不如意十之八九，总有些事无法改变。若因无法改变之事整天怨气满满，吃亏的只是自己。

**允许的能力越弱，你的内在世界就越小**，他人一个举动就能在你世界掀起轩然大波，你便只能不断索取外界情绪价值。
**允许的能力越强，你的内在世界就越大**，他人的风暴于你而言只是浩瀚宇宙的一阵微风，你便自然拥有了几乎无穷的情绪价值去影响他人。
