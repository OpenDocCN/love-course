# 莱特《执行力能量系统》无尽的执行和自律能量：1.系统：自律可能不存在！点破假大空的自律神话_note

在本节课中，我们将要学习一个颠覆传统认知的观点：自律可能并不存在。我们将一起点破关于自律的“假大空”神话，并探讨如何建立一个能自我驱动、不依赖意志力的“能量系统”。

## 1. 系统：自律可能不存在！点破假大空的自律神话

自律这个概念被谈论得太多，也被滥用了太多。大部分人说到自律都会告诉你，你必须设定一个目标，并拥有强大的意志力。这些话听起来空洞无物。

这些都是一些无法落地的、假大空的概念。我们常常把大部分事情做不好的原因归结于不自律。很多时候，这只是为了方便解释。在我看来，自律并非一个全能的解决方案。

甚至，自律可能根本就不存在。有些事情我们做起来很容易，不需要强大的自我管理和自我意识介入就能轻松完成，这难道是因为我们不自律吗？有些事情我们非常着迷，做起来得心应手，很自然地就进入了心流状态。

甚至还没来得及总结成功经验，事情就已经做完了。这难道是因为我们不自律吗？还有一些事情，我们逼着自己去做，比如早起，每周花固定时间强迫自己工作，但付出与收获总不成正比。这难道也是因为不自律吗？

我们必须意识到，那些厉害的人，并非依靠自律和取之不竭的意志力去完成一个又一个重大任务。要想找到内心的驱动力，我们真正需要的是建立一个系统。

这个系统是一整套适合自己、有前后支撑、让人不费力运转、让工作自然展开的做事方法。例如，管理一个500人的企业，如果员工都没有斗志，经理大喊“你们怎么这么不自律，我一走开就不干活了”，这样的经理恐怕要先反省自己的无能。

因为好的领导能制定好系统，调动人的积极性，让系统中的每个部分都朝着应去的方向自然运转。而无能的领导，只能依靠制度强压员工。

自我管理也是同样的道理。人一旦有了一套系统，而不仅仅是依赖自律，就能引导出自己做事的兴趣、热情和坚持。你会越做越有劲，越做越有成就感。

## 2. 缺乏好系统的例子

以下是几个缺乏好系统的具体例子，它们展示了仅靠“自律”或高压要求是如何导致失败的。

*   **例子一：公众号更新**
    我一开始想建立一个每周更新的公众号（社交光谱公众号）。但我野心过大，标准过高，要求自己每篇文章必须写到5000字以上，图文精美，格式整齐，每周定时定点更新，容不得一点失误。在自我领导的高压之下，迅速出现了自我打压和自我苛责。我的成就感还没建立起来，就已经建立了更多挫败感。很快，创作的乐趣消失了，每周的更新也就不了了之。

*   **例子二：坚持阅读**
    如果你想建立坚持阅读的好习惯，却强迫自己每周必须阅读一小时以上，每天按时按点阅读，每次都读最难最复杂的部分。随之带来的后果是，你对阅读产生了厌恶情绪。不到两周，你就会把阅读抛到脑后，然后总结为自己不适合阅读、自制力有限。

诸如此类的例子还有很多。更有效的方法不是拿圣人的标准压迫自己，而是用好的系统引导自己。

## 3. 意志力的局限与系统的优势

上一节我们看到了坏系统的例子，本节中我们来看看为什么依赖“自律”和“意志力”本身就有问题。

自律需要意志力，而意志力是有限的资源。任何人的心力、耐力、忍耐力、强制自己的能量都是有限的。在一段时间内，你的意志力资源是用一点少一点，因为它有一个有限的“能量槽”。

**公式：意志力资源 ≈ 有限能量槽**

就像打游戏一样，魔法值用光了就会枯竭。而我们这套课程真正有价值的地方，就是教你建立一个不太需要意志力就可以维持运转的系统。你在这个系统里游走，不仅不会降低能量，反而会增加能量。

一个好的系统就像一个好房子，走进去就会让你身心愉悦，能量十足。你在里面待得越久，越喜欢自己和这个世界。而坏的系统就像一个蛮横的军阀，永远用强制和约束去规定人的行为。

管理学把这些领导分成X型领导和Y型领导。前者坚信人必须用鞭子赶着走，后者相信人都能自我驱动。没有人喜欢X型领导，可我们自己却总变成自己的X型领导。重建一个系统，然后让系统带着自己运转，这才是王道。

## 4. 如何建立系统：核心是“能量感”

那么，我们怎样去建立一套行之有效、具有干劲的系统呢？这里我要带出一个关键词：**能量感**。

为什么能量感能帮我们持续找到做事的干劲？想一想你最喜欢做的事情、欲罢不能的爱好、总放不下的人或事。这些东西在不知不觉中增加了你当时的能量感。它们不需要你去“自律”，在你拿出“自律”之前，它们早已跑进你自循环的系统里，推动你持续行动。

不知不觉中，你所上瘾和放不下的东西，都在你脑中形成了一个个产生多巴胺和愉快感的回路。例如，你玩游戏或看视频时，这一行为触发了多巴胺分泌，导致能量感上升，结果就是你下次还想要，戒不掉，大脑上瘾了。

**代码示例（类比）：**
```
行为（如玩游戏） -> 触发多巴胺分泌 -> 能量感上升 -> 产生“还想要”的念头 -> 驱动下一次行为
```

那些与工作和金钱相关的、能持续自行运转的系统，一定需要能增加你的能量感才能维持。例如，得到金钱或领导认可这件事导致了你的能量感上升，结果是你下次还想要。这个“想要”的念头会促使你再次付出行动。

所以，想要建立一个系统，而不仅仅是依靠自律，首先要找到一条较为容易就能增加一件事所能给你带来的能量感的路径。至少在开始建立该系统的时候，不能让做这件事使你的能量值降低。否则，你再自律，也很难维持下去，你的热情和兴趣会一点一点耗尽。

## 5. 寻找你的能量来源与消耗项

什么样的事情能给你带来能量感？什么样的事情又会快速消耗你的能量感？如果你稍加观察，很快就会得出属于自己的答案。

以下是帮助你自我观察的几个方向：

*   **社交互动**：你跟人接触之后，能量感是增加还是降低？你一个人独处之后，能量感是增加还是降低？能量感在与他人相处后的增减，其实可以用来判断一个人的性格是外向还是内向。这与MBTI性格测试中的内向人格和外向人格相关联。
    *   **判断标准**：在独处之后能量感增加的人，是内向者。在与他人相处之后能量感增加的人，是外向者。
    *   **系统影响**：内向者如果做需要跟很多人接触的工作，能量值会掉得很厉害，意志力资源也会迅速耗光，好几天都缓不过劲来，系统就难以长期运转。反过来，外向者如果总是做一个人独处的工作，他会缺少“燃料”，缺少意志力和能量感。

*   **信息输入**：报纸杂志上的哪些话题总能引起你的兴趣，让你看完之后能量感爆棚？又有哪些话题让你看完之后觉得能量耗尽？

*   **工作任务**：你日常工作中的哪些任务会增加你的能量感，让你做完之后很兴奋、有趣、有成就感？而哪些任务让你想要躲避，感到焦虑、无聊或畏手畏脚？

*   **人际关系**：你身边的人，哪些会增加你的能量感，在相处之后让你感觉更有热情和做事的乐趣？又有哪些人会降低你的能量感，给你带来无尽的悲伤、忧虑和无力感？

以上这些方面如果总是呈现稳定的趋势，那么就能大概判断自己或一个人能量的流动形式。当然，人的个性和能量流动形式都不是恒定不变的。每个人生阶段或有重大感悟时，这些都可能发生改变。

但找到这些趋势，了解自己的能量来源与消耗项，是在当下建立对自己有效系统的基础，这非常重要。我希望你们看完这集内容后，一定要回忆我所讲的这些特征，去寻找属于你自己的能量来源。
