# 莱特《执行力能量系统》无尽的执行和自律能量：3.寻找：只属于你的路，找到你执行力的共性_note

在本节课中，我们将学习如何有效获得并维持“能量感”，从而打破低迷状态，持续推动个人目标的实现。核心在于找到专属于你、能持续激发你热情与干劲的独特路径，而非盲目跟随他人。

上一节我们探讨了“能量感”是推动行动的关键。本节中，我们来看看如何具体地提升能量感。

## 从案例看问题：小葵的困境

小葵面临考研或出国的选择，但无论选哪个，都需要良好的学习状态。她试图建立每天去图书馆的规律，并制定学习计划，却几乎从未完成。

以下是她的心路历程：
*   制定计划时非常兴奋。
*   第二天在图书馆时，无法按计划开始，总是不自觉地刷手机。
*   在图书馆待上一上午后毫无成效，下午继续烦躁并刷手机。
*   晚上感到沮丧、懊恼和自我失望。

小葵的状态本质上是陷入了**低能量感的恶性循环**：
**低能量感 → 不想做事 → 完不成任务 → 自我苛责 → 更低的能量感**

由于未找到提升能量感的突破口，过高的预期和目标只会产生反作用：制造压力、挫败感，打击自信，导致能量感进一步降低，使人进入慢性低迷。

## 正向循环：高能量感系统

一个能正向维持的个人系统依赖以下路径：
**高能量感 → 付出行动 → 完成目标 → 获取更多能量感 → 进一步行动**

因此，改变状态的关键在于：**如何将个人能量感从低提升至高，并持续产生有效的能量感。**

## 核心方法：撬动能量感的杠杆

以下是具体能撬动我们能量感的方法。

### 1. 找到专属的能量感途径

这意味着找到最能让你持续产生能量感、让你感到兴奋热情且不觉得累的路径，从而最佳地激活你的能量感，而不是盲从别人的方法。

观察身边能量感高且稳定的人，你会发现，最持续的能量感一定源于**从内到外地做自己**。这需要你充分了解自己。

我们以“学英语”为例。假如你是一年后期望成为高手的英语小白，如何选择能量感路径？

以下是不同人可能适用的不同路径：
*   **爱好型兴奋**：将英语与爱好结合，如读英文小说、看外国电影、听英文歌。
*   **社交型兴奋**：通过学校的英语角等社交活动学习。
*   **恋爱型兴奋**：找英语母语者谈恋爱，在恋爱中学习。
*   **助人型兴奋**：从事导游等需要运用英语帮助他人的工作。
*   **挑战/成就型兴奋**：通过参加英语竞赛、考试或争取奖项来学习。
*   **表现型兴奋**：在公众面前进行英语演讲、表演话剧或唱歌。

路径不止一种。你应该选择**最能使自己产生能量感**的那种，而不是大多数人使用的那种。最适合你的路径，取决于你独特的能量感触发机制。

发现并尊重你个人的喜好、热爱、性格特点（如内向/外向、是否喜欢竞争等）至关重要。最重要的是意识到：**每个人的能量感触发点不同。**

只有找到专属于你的能量感路径，你才能轻松而有干劲地实现目标，并从中获得更多能量感。当你的“打开方式”契合你的路径时，你会感到事情容易、有干劲、精力充沛。反之，则会感到疲劳、无趣、烦躁，无法坚持。

### 2. 如何发现自己的能量感来源

你可以通过问自己以下几个问题来探索：

以下是值得反复思考的问题列表：
*   在你过去的所有人生经历中，能量感最高的三个瞬间是什么？是由哪些具体活动、情境、缘由引起的？
*   在过去的一个月中，你能量感最高的三个时刻是什么？是由哪些具体活动、情境、缘由引起的？
*   如果展望接下来的一年，让你最兴奋、激动、期待的3到5件事情是什么？
*   在你朋友眼中，有哪些事情你做得轻松不费力，但别人觉得很困难？

建议每隔一段时间做一次这样的练习，因为人的能量感路径也会变化。当几次练习后答案趋于稳定，你就能从中发现**获得能量感时的共性**。

## 课程总结

本节课我们一起学习了打破低能量恶性循环、建立高能量正向系统的核心方法：**寻找并践行专属于你自己的能量感路径**。通过自我提问和反思，发现能真正激发你热情与动力的独特方式，是持续获得执行力与自律能量的关键。
