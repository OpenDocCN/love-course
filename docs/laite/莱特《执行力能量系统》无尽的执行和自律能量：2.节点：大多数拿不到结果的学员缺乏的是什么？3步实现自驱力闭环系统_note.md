# 课程2：执行力能量系统 - 节点：实现自驱力闭环 🧩

## 概述

在本节课中，我们将探讨为何许多学员难以取得成果，并学习如何通过一个三步系统来构建自驱力闭环，从而获得持续的行动能量。

上节课我们介绍了“能量”这个概念。本节中，我们来看看如何增加这种能量感。

## 从消耗能量到增加能量

有一件任务，你早就应该动手，却迟迟没有行动。这是因为执行它会降低你的能量感。

有没有办法能将这件事转换为增加你能量感的途径？关键在于，找到做这件事时能**轻松增加能量感**的路径，而非复杂的路径。这是一个非常简单、容易的增加能量感的路径。

以写作为例。如果能出版一本书，一定能增加成就感和能量感。但成功出版一本书，需要经历从构思、写作、联系出版商到最终印刷的漫长过程。这中间要消耗大量能量，周期也很长。除了“成书”这个大目标，其他过程以及达到能量增加的条件都太漫长了。

我们可以换一个视角。写作的乐趣、每篇文章获得的外部反馈、自己所欣赏之人的赞许、与他人交流的乐趣，显然都是更即时的能量来源。

如果一个人把出版一本书作为增加能量感的唯一途径，那么这个能量回收周期太长、太慢。事情做着做着就会失去激情和燃料。此时，半途而废导致的能量消耗反而会出现。

如果能找到一个或多个**花小力气就能增加能量感**的路径，在实现大目标的路上，不断用小目标的实现来增加能量感，那么“小圈带动大圈”的系统就能更容易运转。

**所以，从小步开始，快速完成循环并收回能量感，是开始建立一套系统最有效的办法。**

## 核心：闭合任务回路

如果有一件应该做的事，却让你始终找不到任何能量来源，该怎么办？这成为我们接下来关注的重点。

获得能量感的路径有很多种。其中一个关键是 **闭合任务回路**。我们必须为所做之事构筑一个能**收回能量感**的回路和路径。否则，能量感会不断漏掉。兴趣和热情只能维持一段时间的动力，终究不能长期推动系统运转。

所谓“闭合任务回路”，其使命就是为了创造一种**完结感**。一种通过自己的某个行动，导致某个事物状态达成完结的闭合感。这种感觉是非常个人化的，它不同于个人成就、业绩、认可或嘉奖。它不需要是一个宏大的终结，可以是一个又一个微小的完成。

我们在学业和职业路径上有许多显而易见的“完结点”。这些完结点的闭合，帮助我们收回能量感，进而去完成下一阶段的任务。

*   **例如**：参加四六级、托福、GRE考试，你的完结点是**考试结束、通过考试**。
*   **例如**：找工作时，你的完结点是**完成面试、拿到offer**。
*   **例如**：以项目为周期的工作，你的完结点可能是一个**咨询项目、客户服务项目或一份客户合同的完结**。
*   **例如**：以学期或年度为周期的工作，你的完结点可能是一个**学期的课程结束、一批学生毕业、年终个人业绩总结的提交**。

为什么我们在从学校到职场的转换中，常有无力感和挫败感？因为学校里有大量**清晰的、具有群体感的完结点**。意思是一群人同时拥有整齐划一的完结点。我们在无形中将这些外部世界的完结点内化为自己的完结点，因此在完成时，很容易实现心理上的闭合任务回路，从而拿回能量感。

人离开学校后，这种来自统一的、边界清晰的完结点会越来越少、越来越淡化。你和同事可能只是表面共享某个项目的完结点，但你感到与他们有不同的人生追求、现实挑战、成长目标和个人兴趣。实际上，你个人成长和人生进展的完结点变得越发模糊、不确定、缺少外界确认。闭合任务回路也越来越需要依靠自己去实现。

你可能没有意识到人类对闭合任务回路的需求有多大。

*   **例如**：有一种惩罚犯人的方法是让犯人挖一个坑，填平，再重新挖，再填平，循环往复。以此让犯人在持续的“未完成感”中受罪。
*   **例如**：我们心理上有种寻求闭合的需求。你没听完的歌会在脑内单曲循环；没得到答案的问题就放不下；莫名其妙被分手的人，总是对感情更难以割舍。

这种对完结感的需求是写在我们人类基因里的。所以，建立个人项目的闭合任务回路非常重要。

## 实现闭合任务回路的三要点

实现闭合任务回路有三个非常重要的要点。

以下是实现闭合任务回路的三个核心步骤：

1.  **清晰的完结点**：这是一条明确的、阶段性的终点线，是自己给任务划分小目标的能力，是一段持续努力的界限。完结点不需要是惊天动地的业绩。相反，它应该是容易实现的小目标。
    *   **例如**：`跑步10公里`可以是一个完结点，`跑步2公里`也可以是。`每天运动半小时`可以是一个完结点，`每天运动5分钟`也可以是。

2.  **仪式感的触发**：所谓仪式感，就是当达到完结点时，能让心理上明确感受到、接收到“已实现一个闭环回路”的触发。
    *   **例如**：**毕业典礼**就是典型的完结感仪式触发。
    *   **例如**：**搭讪并成功收到联系方式**，我们无形中也有这种仪式感，让自己知道完结点已实现，任务回路可以闭合了。这时能量感就收了回来。

3.  **个人心理上的认可**：以上两点能收回能量感的关键，是你在心理上对某个完结点和仪式感的**认可**。这是建立闭合任务回路至关重要的环节。
    *   **例如**：用自己感兴趣的东西奖励小目标的实现（一杯咖啡、一件衣服、一场电影、一场旅行），这是对当天努力回路的闭合。
    *   **例如**：向家人朋友宣布自己一个项目、一个阶段、一个小目标的达成。
    *   **例如**：在任务结束后记录总结它，也属于回路闭合。
    *   **例如**：长期努力的项目获得投资、计划书被接受、创作被人认可。

个人任务向前推进时，最容易出现“完结点不清”的问题。一门心思向前跑，但没有通过闭合任务来获取持续的能量感。同样跑了10公里的两个人，在内心实现了闭合任务回路的那个人会收回更多能量感，从而有更足的力量去跑接下来的比赛。

## 管理能量：减少无效流失

任务闭环可以增加我们的现有能量。我们的能量系统就像一个储蓄卡：要能快速频繁地获得能量感，也要能有控制、有意识地去分配现有能量。这样你的能量储蓄卡中始终有能量留存，用任务闭环不断带动新任务的发生。

那么，如何减少系统内能量的无效流失呢？

我们每天都会面对能量流失的问题。你一定有这种体会：做某些事所消耗的能量感，比其他事情更快、更多、更难恢复和补偿。

做任何事情都需要支出能量，哪怕是打字、上下班、听歌、与同事沟通、收拾桌面等。但有些事情的能量消耗显然高得离奇，消耗得非常无用。还有些事情，别人做起来好像没怎么消耗能量，而你做起来能量值就骤降。在另一件事上，情况又完全相反：你做起来生龙活虎，别人做起来索然无味。

由于一份工作中通常包含很多任务内容，你会发现，即便在同一份工作中，你在做某些任务时能量增加，某些任务上能量守恒，而某些任务上能量是流失的。例如写报告、开会、见客户、做总结、写计划、管理下属、向上级汇报，这些任务对于不同的人来说，能量消耗都有很大区别。

你的能量流失在哪里？做什么事、在什么情境下，能量明显漏出，却没有帮助任务的完成？

以下是常见的能量流失情境：

*   内容单一、重复性的工作带来的无意义感。
*   缺乏控制感。
*   害怕失败，害怕让别人失望。
*   不想面对真实的自己，产生挫败感。
*   对未来的不确定，对目标的模糊性。
*   后悔心理。

很多人可能没有意识到，工作中的重复、无聊、无意义，都会制造巨大的能量漏洞，让你迅速感到疲劳、不想做事、自我效能下降。人是一种需要意义感的动物，我们无法把自己变成重复劳动的机器。如果你的工作让你感到无聊和憎恨，你的能量感就会加速流失，迅速感到疲劳。这种疲劳感会从一项任务本身波及到其他任务，甚至波及到生活的各个方面。此时，再谈论自己不够自律、没有意志力也无济于事。自律无法帮你增加能量感，它只是拿着鞭子的教官，供养一个教官可能消耗更多能量。

另一个常见的能量感漏洞是**自我苛责**。任务本身没那么难，我们做得也没那么差，但自我苛责的念头突然从大脑中冒出来。于是，仅有的能量感都消耗在与自我苛责、自我斗争和太完美主义上了。

所以，第一层目标应该是关注**如何让能量消耗尽量发生在任务本身，而不是任务之外**。

*   **例如**：写一篇论文，如何让能量感消耗在思考、阅读、书写的动作上，而不是消耗在担心他人评价、讨论任务无意义以及抱怨未来不确定上。
*   **例如**：要去见一个女生，如何让能量感消耗在见面前的准备、对约会路线的筹划、约会过程的全力以赴上，而不是消耗在事前的担忧、对见面的抗拒、对自己的苛责以及对结果的悔恨上。

我知道你刚接触这些概念，这对你来说不一定简单。但多观察自己的能量从哪里漏走，是否真的帮到了任务本身，这会帮助你避开和降低无意义的损耗。

以下是几个值得花时间思考的问题：

1.  做什么事情会让我快速消耗能量感？
2.  跟什么样的人接触会让我快速消耗能量感？
3.  在什么情境下会让我快速消耗能量感？
4.  目前工作中的哪一部分会让我快速消耗能量感？

仔细思考这些问题后，你可能会发现，许多你一直以为在增补能量的日常活动，其实是在偷走你的能量。你之所以一直在做这些活动，只不过因为身边的人都这么做。

## 能量消耗的三种类型

总结来看，一个人的能量感消耗路径大概分为三种不同类型：

1.  **收入结余型**：所消耗的能量感带来了更多新能量的产生。
    *   **公式**：`消耗的能量 < 获得的新能量`
    *   **例如**：你熬了一周的夜完成一个大项目。项目成功后，你获得了客户、老板和自己内心的认可，进而有动力去做一个更大的项目。

2.  **收支平衡型**：所消耗的能量和所增加的能量感大抵相当，没有过分消耗也没有过多结余。
    *   **公式**：`消耗的能量 ≈ 获得的新能量`

3.  **收不抵支型**：所消耗的能量大于从做任务中所获得的能量，导致持续向前的能量感逐渐耗尽。
    *   **公式**：`消耗的能量 > 获得的新能量`
    *   **例如**：为了赶期末大论文临时抱佛脚，几天没睡觉。交完论文后，你三个月都不想再碰论文了。

如果仔细回想，你会发现几乎所有没坚持做下去的项目，都是因为能量感的“收不抵支”。例如，最经典的大脑中的担忧、犹豫、思前想后、害怕犯错、惧怕别人的目光。这些不会对现实产生任何影响的“大脑空转”，会消耗你所有的个人能量，导致你做其他事情时失去执行力。

**所以说，能量感是在行动中获取，而在大脑的胡思乱想和空转中漏光的。** 一定的计划性和未雨绸缪是必要的，事后反思也是必要的。但大脑空转的时间应该远远低于我们行动的时间。我们大脑空转得越多，行动就会变得越困难，闭合能量回路也变得越加不可能。

我经常在其他课程里说这样一句话：**“一想全是问题，一做全是答案。”** 因为闭合能量回路永远要依靠看得见、摸得着、能外化的行动和产出，而不是大脑的空转。即便是哲学家——这世界上最有资格进行大脑空转的人——也需要靠外化自己的思考（通过书写、记录和演讲）来完成能量回路的闭合。

现实生活中的我们要不断、反复、一次又一次地去行动，依靠从小到大各种行动回路的闭合来收回我们的能量感，从而维持持续向上的动力。

## 应用实例：坚持跑步的真相

如果你仔细思考，会发现几乎所有没坚持做下去的事情都是因为能量感的收不抵支。就拿跑步来做例子。

长期坚持跑步的人，是因为他们的意志力惊人吗？不，只是因为在长跑中他们获得的能量感，大于做其他事情所获得的能量感。因此，越做越想做事，很自然地就坚持下去了。意志力根本还没派上用场。

如果你去问那些喜欢长跑的人为什么喜欢跑步，你可以通过本节课的内容，找到他们获取能量感的通道：

*   在生理上引起多巴胺上升，增加愉悦感。`能量感提升 -> 下次还想跑`。
*   运动过后头脑清楚，工作状态提升。`能量感提升 -> 下次还想跑`。
*   达成所设跑步目标，增加成就感和自信心。`能量感提升 -> 下次还想跑`。
*   跟小伙伴一起跑步，社交效应增加愉悦感。`能量感提升 -> 下次还想跑`。
*   长期有规律的跑步让身体发生良性变化。`能量感提升 -> 下次还想跑`。

越跑越想跑。当跑步成为你增加能量感的途径，你就能带着更高的能量感去面对生活里的挑战、更高效地工作、面对更多压力。你自然会想要下一次继续通过跑步来获取更多能量感。

## 总结与回顾

那么，我们可以回过头思考一下你看这门课程的原因了。

为什么你想建立一个新的习惯，却总是无法坚持？并不是因为你的意志力不够或自律不够，而是因为你**并没有成功建立获取能量感的有效路径**。你并没有从所做的这些事情中拿回更多能量感，导致这个新习惯总是在消耗能量感的余额。

当一件事情总是在消耗你的能量，你很快就会自然而然地去寻找其他能给你增加能量感的事情（比如打游戏、看电影、逛街）。于是，你想建立的新习惯目标就会半途而废。

**本节课中，我们一起学习了：**

1.  将消耗能量的任务转化为增加能量感任务的关键，在于找到**简单易行的小路径**。
2.  维持长期动力的核心是构建**闭合任务回路**，它包括：**清晰的完结点**、**仪式感的触发**和**个人心理上的认可**。
3.  需要管理能量，减少无效流失，尤其要避免能量消耗在**任务之外**（如自我苛责、无意义空想）。
4.  能量消耗分为**收入结余**、**收支平衡**和**收不抵支**三种类型。无法坚持的事情，问题往往出在“收不抵支”。
5.  真正的坚持源于行动中获得的**正向能量反馈**，而非单纯依靠意志力。
