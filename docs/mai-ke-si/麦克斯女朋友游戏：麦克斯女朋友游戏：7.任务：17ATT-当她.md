# 麦克斯女朋友游戏：麦克斯女朋友游戏：7.任务：17ATT-当她

在本节课中，我们将要学习一个关键的关系管理概念：当你的伴侣遇到问题时，如何恰当地应对。核心原则是，你无需扮演“心理治疗师”的角色去解决她的问题。

---

## 概述

上一节我们讨论了关系中的互动模式。本节中，我们来看看一个常见场景：当你的女朋友向你倾诉问题或寻求解决方案时，你应该如何行动。核心行动是 **“不要做任何事情”**。

## 核心原则与行动指南

核心原则可以用一个简单的代码来描述你的应对策略：
```python
def when_girlfriend_has_problem(problem_description):
    # 核心行动：不做任何事情去解决她的问题
    action = "do_nothing"
    return action
```

以下是具体的行动步骤：

1.  **识别信号**：当你的女朋友给出提示，或直接告诉你她需要解决某个问题时，识别出这个场景。
2.  **停止行动**：立即停止你试图为她分析问题、提供建议或解决方案的任何冲动。
3.  **保持倾听**：你可以表示你在听，但不要将对话引向问题解决。

## 课后练习与反馈

完成上述步骤后，你需要进行自我反思并与社群交流。

以下是反思与分享的步骤：

*   前往指定的Facebook群组。
*   写下你当时的感受。例如，你是否感到自己像个笨蛋，或者感到无力？
*   描述具体发生了什么事。
*   让我们知道你的真实想法和体验。

## 总结

本节课中我们一起学习了在伴侣遇到问题时的正确应对方式。记住，你的角色是伴侣而非治疗师，核心行动是 **“不要做任何事情”** 去直接解决她的问题。通过实践这一原则并进行社群分享，你可以更好地管理关系中的动态。
