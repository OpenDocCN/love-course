# 麦克斯女朋友游戏【关系管理】【萌萌团队-皮卡内部提供】课程7：任务4 - 你是谁的男朋友 👫

在本节课中，我们将学习如何运用“Hustle Max”和“Action Taker”工具包，来明确你在恋爱关系中的身份定位，并分享你的进展。

## 课程概述

上一节我们讨论了关系管理中的具体行动。本节中，我们将聚焦于一个核心任务：明确“你是谁”的男朋友。这个任务旨在帮助你清晰定义自己在当前关系中的角色和阶段。

## 核心步骤与工具

我们再次回到“Hustle Max”（积极进取者）和“Action Taker”（行动派）工具包。无论你处于“Noop”（无行动）阶段还是“Player”（玩家/行动者）阶段，这个练习都适用。

以下是执行此任务的具体方法：

1.  **定位你的阶段**：首先，明确你当前在关系中所处的阶段。
2.  **写下你的陈述**：用一句话写下你的想法，格式为：“当我在‘男朋友’阶段时，...”。
3.  **填充思维框架**：在这句话中，完整阐述以下四个方面的内容：
    *   **你的想法**：你在这个阶段会有什么样的思考？
    *   **女性的想法**：你认为女性在这个阶段会有什么想法？
    *   **你的成果**：你在这个阶段会取得什么成果？
    *   **女性的成果**：女性在这个阶段会获得什么成果？

## 任务执行与分享

完成上述思考并写下你的句子后，请将你的想法和成果分享到指定的Facebook群组中，与团队其他成员交流。

本节课中，我们一起学习了如何通过“Hustle Max”和“Action Taker”工具来定义自己在关系中的身份。关键在于用一句清晰的话概括你作为“男朋友”的想法、对方的想法以及双方的预期成果。请务必完成练习并参与社群分享。
