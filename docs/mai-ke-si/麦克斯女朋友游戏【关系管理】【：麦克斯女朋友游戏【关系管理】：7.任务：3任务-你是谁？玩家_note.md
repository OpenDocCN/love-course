# 麦克斯女朋友游戏【关系管理】课程7：任务3 - 你是谁？玩家 🎮

在本节课中，我们将学习如何通过“你是谁？玩家”这个任务，来明确自己在关系中的角色定位和心态。这对于理解游戏进程和提升关系管理能力至关重要。

---

上一节我们探讨了关系中的阶段性目标，本节中我们来看看如何通过自我反思来定义“玩家”角色。

无论你是新玩家、男朋友角色，还是处于其他阶段，在当前的游戏进程中，你应该已经积累了一些观察和体会。

以下是完成此任务的具体步骤：

1.  **记录你的心态**：写下当你处于“玩家”阶段时，你内心的真实想法和感受。
2.  **分析对方心态**：思考并记录下在此阶段，女生的可能想法和反应是怎样的。
3.  **评估行为结果**：总结在这种心态和角色下，通常会导致什么样的关系结果。
