# 乐福教程 01：基础概念与操作 🧩

在本节课中，我们将要学习乐福的基本概念和核心操作。乐福是一种用于数据处理的工具，理解其基础是掌握更高级功能的关键。

## 什么是乐福？🤔

乐福是一个数据处理框架。它允许用户通过定义规则和流程来处理和转换数据。其核心思想是将复杂的数据操作分解为一系列简单的步骤。

## 核心概念：节点与连接 🔗

上一节我们介绍了乐福的基本定义，本节中我们来看看其最核心的两个概念：节点和连接。

*   **节点**：节点是乐福中的基本处理单元。每个节点代表一个特定的操作或功能。例如，一个节点可以用于读取数据，另一个节点可以用于过滤数据。节点的行为通常由一个**函数**来定义：`节点输出 = 函数(节点输入)`。
*   **连接**：连接定义了数据在节点之间的流动路径。它将一个节点的输出作为另一个节点的输入。通过连接，多个节点可以组合成一个完整的数据处理流程。

以下是节点的主要类型：
*   **输入节点**：负责从外部源（如文件、数据库）获取数据。
*   **处理节点**：对输入的数据执行计算、转换或分析。
*   **输出节点**：将处理后的结果保存或发送到指定目的地。

## 基础操作流程 🛠️

理解了节点和连接后，我们来看看如何构建一个简单的乐福流程。一个典型的操作遵循“配置-连接-执行”的模式。

1.  **创建节点**：首先，根据你的需求创建所需的节点实例。
2.  **配置节点**：为每个节点设置必要的参数。例如，为输入节点设置文件路径。
3.  **连接节点**：使用连接线将节点的输出端口与下一个节点的输入端口相连。
4.  **执行流程**：触发流程开始运行。数据将从输入节点开始，依次流经各个处理节点，最终到达输出节点。

## 一个简单示例 💻

让我们通过一个伪代码示例来直观感受一下。假设我们要读取一个文件，将其中的数字加倍，然后保存结果。

```python
# 1. 创建节点
node_read = Node(“读取文件”)
node_calculate = Node(“数值计算”)
node_write = Node(“写入文件”)

# 2. 配置节点参数
node_read.set_parameter(“file_path”, “input.txt”)
node_calculate.set_parameter(“operation”, “multiply”, “value”, 2)
node_write.set_parameter(“file_path”, “output.txt”)

# 3. 连接节点
connect(node_read.output, node_calculate.input)
connect(node_calculate.output, node_write.input)

# 4. 执行流程
run_flow(start_node=node_read)
```
这个例子清晰地展示了数据如何从一个节点流向下一个节点，并在每个节点进行相应的处理。
