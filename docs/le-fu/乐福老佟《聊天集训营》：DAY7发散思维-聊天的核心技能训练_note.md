# 乐福老佟《聊天集训营》：DAY7发散思维-聊天的核心技能训练_note

在本节课中，我们将要学习聊天中的一项核心技能——发散思维。这是一种能够让你的对话更加生动、有趣且富有创造力的思维方式。我们将探讨它的定义、特点，并通过具体案例学习如何在聊天中实际运用它。

## 概述：什么是发散思维？

发散思维，又称辐射思维、放射思维、扩散思维或求异思维，是指大脑在思考时呈现的一种扩散状态。它与我们通常习惯的、局限于单一答案或解决方式的“收敛思维”不同。发散思维强调**一体多解、一事多行、一物多用**，是创造性思维最主要的特点。

不少心理学家认为，发散思维是测定创造力的主要标志之一。例如，提到“埃隆·马斯克”，发散思维会让你联想到：世界首富、特斯拉、SpaceX、比特币代言人等众多关联事物，而不仅仅是他的名字。

## 发散思维的特点

上一节我们介绍了发散思维的基本概念，本节中我们来看看它的四个核心特点。理解这些特点，是将其应用于聊天的第一步。

### 1. 流畅性

流畅性是指能够根据自己的观念自由发挥，在尽可能短的时间内生成并表达出尽可能多的思维观念。在聊天中，这体现为“想到什么说什么”的快速反应能力。

以下是流畅性在聊天中的体现：
*   **高情绪开场**：例如，刚加上女生时说“已备注小可爱”。这是一种不经过多思索、基于当下感受的自由发挥。
*   **快速接话**：当女生讨论电视剧《东宫》剧情，并提到“谈恋爱吗，灭你全族的那种”这句台词时，可以快速回复“哈哈，谈恋爱吗？”，巧妙地将话题从剧情讨论转向情感互动。

如何培养聊天的流畅性？核心是：**想到什么说什么**。但这需要建立在两个前提之上：
1.  **衡量关系**：所说内容需符合你们当前的关系阶段。
2.  **不伤害对方**：确保你的表达不会对对方造成情感上的伤害。

### 2. 变通性

变通性是指克服头脑中僵化的思维框架，按照新的方式去思索问题的过程。在聊天中，当女生的回复看似“不好接”或局面陷入僵持时，变通性能帮你巧妙破局。

以下是变通性的应用案例：
*   **化解“害怕”**：当女生说“我怕你”（意指关系发展太快）时，可以变通地回复：“那是该怕，老公坏坏的。” 将“害怕”的情绪转化为带有暧昧色彩的互动，而非让关系冷却。
*   **应对平淡回复**：当女生回复“嗯”或表情等无信息内容时，可以接续自己之前的话来延续氛围。例如，之前说“有我在保护你”，对方回表情后，可以接着说“给你戴个头盔，严密保护”。

变通性的本质是**横向类比、跨越转化、触类旁通**。就像开车遇到墙，你会选择绕路而非撞墙。聊天中遇到阻碍，也应灵活地切换路径，导向有利于关系升温的方向。

### 3. 独特性

独特性是指在发散思维中做出不同寻常、异于他人的新奇反应能力。它基于强烈的自我认知，并能结合对方情绪给出令人印象深刻的反馈。

以下是独特性在聊天中的体现：
*   **非常规解读**：凌晨3点，女生说“女生熬夜老得快”。可以独特地回复：“女生老得快，不是因为熬夜，而是因为没有男朋友。” 这将话题导向了男女关系，而非养生讨论。
*   **升级常见回复**：对于“多喝热水”这种常见的直男回复，可以加入独特性：“傻子吗？我不在旁边，你就不能自己多喝点热水？” 这既表达了关心，又带有亲昵的调侃，感情色彩更丰富。

独特性要求你坚信自己的认知和框架，并以一种新奇、有力度的方式表达出来，让对方感受到你的男子气概和独特魅力。

### 4. 多感官性

多感官性是指在流畅性的基础上，为信息赋予浓厚的感情色彩，从而提高发散思维的速度和效果。它让聊天不止于信息交换，更充满情感张力与画面感。

以下是多感官性的应用：
*   **加入感情色彩**：聊天中说到“养你”，女生回复“那你先把我养胖”。可以多感官地回应：“那我要在你耳边哈气。” 这立刻将简单的对话升级为带有亲密感和画面感的调情。
*   **使用关系升温措辞**：通过特定句式注入情感：
    *   **动词+人物**：我要抱抱你。
    *   **动词+器官**：摸摸头。
    *   **动词+人物+器官**：亲亲你的嘴。
*   **描绘画面感**：为了描述更生动，可以构建场景：“宝宝你躺在我怀里，我抚着你的头发，看着远方的海鸥在海面觅食。夕阳西下时，我牵着你的手一起回我们的小屋。” 这能极大增强情感的传递。

多感官性让聊天从“逻辑层面”跃升至“情感层面”，是关系升温的关键。

## 如何培养发散思维？🌱

发散思维就像一棵树，主干是你的核心认知，枝叶是你自由生发的各种想法。要培养它，请遵循以下原则：

1.  **尊重每一个想法**：不要轻易扼杀自己脑海中冒出的任何念头，即使它看起来有点“天马行空”。在聊天中，只要不伤害对方、不严重破坏关系，就大胆尝试表达。
2.  **通过大量练习积累**：技能的提升离不开数量的积累。主动与10-20个不同的女生聊天，在实践中不断试错、调整和总结，你的“社交直觉”和发散思维能力会得到质的飞跃。
3.  **保持积极心态**：社会常常给我们设限，告诉我们“应该”怎么做。培养发散思维需要你打破这些内在限制，相信自己的灵感和创造力是宝贵且可以成长的。

## 总结

本节课中，我们一起学习了聊天核心技能——**发散思维**。我们首先了解了它的定义，即一种扩散的、求异的创造性思维方式。接着，我们深入探讨了它的四个特点：**流畅性**（想到就说）、**变通性**（灵活破局）、**独特性**（新奇反应）和**多感官性**（注入情感）。最后，我们明确了培养发散思维的方法：尊重想法、大量练习、保持积极。

记住，聊天不是逻辑问答，而是情感与思维的舞蹈。掌握发散思维，就是掌握了让这场舞蹈变得迷人而难忘的秘诀。多去想，多去练，你会在聊天中收获意想不到的惊喜。
