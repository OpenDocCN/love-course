# 课程一：人像美容基础 🖼️

在本节课中，我们将要学习如何通过优化个人展示面，特别是头像和朋友圈照片，来提升对他人的吸引力。核心内容包括认识资源的重要性、展示面的构建方法，以及使用手机软件进行人像美容的具体操作步骤。

## 认识资源与展示面

上一节我们介绍了课程的整体目标，本节中我们来看看如何开始第一步：认识资源和构建展示面。

拥有一个良好的朋友圈是展示自己的基础。他人可以通过你的朋友圈了解你的日常生活、兴趣爱好和个人特质，从而对你产生兴趣并被吸引。

以下是构建展示面的几个核心方面：

1.  **头像**：必须使用本人的真实照片。使用漫画、风景或非本人照片会显得不专业。一张得体的个人头像能展现自信，留下良好的第一印象。
2.  **展示面内容**：发布的内容应能体现个人价值和生活方式。这包括但不限于与宠物的互动、旅行风景、工作状态（如加班）等。内容需要精心设计，以区别于他人。
3.  **拍照基础**：展示面的质量很大程度上取决于照片本身。接下来我们将详细讲解如何拍出一张合格的照片。

## 拍照的核心技巧

在了解了展示面的重要性后，我们来学习如何拍出用于展示面的高质量照片。正确的拍照姿势和光线运用至关重要。

拍照时，请遵循以下步骤：

1.  **姿势与构图**：手臂伸直，手机以45度角仰拍。人物应位于画面正中央，头顶与画面上边缘保持约1-1.5厘米的间隙，避免人物过大或过小。
2.  **光线运用**：确保光线充足，脸部应对着光源。如果手机像素不佳或环境光不足，可以借助另一部手机的闪光灯进行补光，以减少面部阴影，使照片更清晰。
3.  **背景与角度**：选择简洁、不杂乱的背景。可以尝试多个角度和姿势进行拍摄，最后筛选出最佳的一张。

## 修图软件准备与基础美化

照片拍好后，需要经过后期处理才能发布。本节我们将介绍常用的修图软件并进行基础美化。

常见的修图软件包括美图秀秀、美颜相机等。我们将以美图秀秀为例进行讲解。请确保已下载该软件。

打开美图秀秀后，请选择 **`人像美容`** 功能，而不是“美化图片”。导入你拍摄的最佳照片后，我们将进行以下基础调整：

1.  **磨皮**：进入“磨皮”功能。软件通常会默认应用一定程度的磨皮，建议将参数调整至 **`30%`** 左右，以避免过度美化导致皮肤失真。
2.  **美白**：进入“美白”功能。同样建议将参数调整至 **`30%`** 左右。需根据原图的光线情况微调，光线暗则调高，光线强则调低。
3.  **肤色**：根据个人肤色调整。肤色偏黄则向“白皙”方向微调（约 **`20%`**）；肤色偏白则可向“红润”方向轻微调整（约 **`10%`**），使气色更自然。

**注意**：每一步调整后，务必点击右下角的绿色对勾保存设置，否则调整会失效。

## 面部细节精修

完成基础肤色调整后，我们可以对面部细节进行精细处理，让照片更加完美。

以下是针对不同面部问题的修图技巧：

1.  **去斑去痘**：使用“去斑去痘”功能。放大图片，用合适大小的笔刷点除脸上的痘痘、痣或斑点。为了保持真实感，可以特意保留一个较明显的特征。
2.  **瘦脸瘦身**：使用“瘦脸瘦身”功能。选择中等大小的笔刷。放大图片后，可轻微向内推挤脸颊两侧（腮帮子处）以修饰脸型，也可轻微拉长下巴。此功能也可用于修饰不整齐的发际线或发型。
3.  **去皱**：使用“去皱”功能。针对额头纹、法令纹（鼻翼两侧延伸的纹路）或眼角细纹。放大图片后，用笔刷沿皱纹轻轻涂抹或点按。如果去除后仍有残留，可用“去斑去痘”功能进行二次处理。

## 增强面部立体感

最后，我们可以通过增强面部光影对比来提升五官的立体感，使照片更具质感。

“高光”功能用于提亮面部凸起部位，增强立体感。具体操作如下：

1.  选择“高光”功能，并选择合适的笔刷大小。
2.  放大图片，在鼻梁骨正中（从鼻尖到眉心的位置）轻轻涂抹一道，使其比周围皮肤稍亮。
3.  同样可以在脸颊最高点（颧骨上方）轻微提亮。
4.  提亮的标准是：提亮部位的亮度略高于周围皮肤，但不超过照片整体的正常曝光度。完成后缩小图片查看整体效果，五官会显得更立体。

**核心技巧**：高光应集中在鼻梁、脸颊高点、额头中央等骨骼突出部位，不要全脸涂抹。

## 总结与练习

本节课中我们一起学习了从拍照到精修完整的人像美容流程。我们首先明确了优质展示面的重要性，然后学习了正确的拍照姿势、光线运用和构图技巧。接着，我们详细讲解了如何使用美图秀秀进行磨皮美白、祛斑瘦脸、去皱和增强立体感等一系列操作。

请记住，修图的目的是提升形象，但务必保持自然，避免过度修饰导致失真。课后请多加练习，将所学技巧应用于自己的照片中，逐步提升个人展示面的质量。
