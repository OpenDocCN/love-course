# 梵尘《9.0》网络课程 第十一课：修图教程（展示面）📸

## 概述

在本节课中，我们将要学习如何通过修图软件优化个人展示照片。课程将涵盖从拍摄技巧到后期修饰的全过程，帮助你制作出更具吸引力的社交媒体头像和展示面照片。

## 一、 修图的重要性与准备

在社交媒体时代，个人展示照片的质量至关重要。当你认识陌生人时，对方首先会通过你的头像和朋友圈内容来了解你。因此，拥有高质量的照片是必要的。

许多公众人物在媒体上的照片都经过精心修饰。我们同样需要学习如何修饰自己的照片。

以下是开始修图前需要做的准备：

1.  **下载软件**：首先需要下载两个修图软件。
2.  **美咖相机**：在苹果App Store中搜索并下载“美咖相机”。
3.  **美图秀秀**：下载“美图秀秀”应用，并确保更新到最新版本。

## 二、 拍摄技巧：构图与角度

上一节我们介绍了修图的准备工作，本节中我们来看看如何拍摄一张适合修饰的原始照片。修图的前提是有一张照片。拍摄时，应尽量选择光线充足的环境，光线越强，照片质感越好。

打开手机自带相机进行自拍即可，无需外接设备。自拍的角度很有讲究。

以下是常见的错误与正确的拍摄方法：

*   **错误示范**：许多人自拍时，头顶距离画面上缘有较大空隙。这种构图比例不佳。
*   **正确构图**：正确的构图应使头发顶端贴近相机画面的上边缘。
*   **拍摄角度**：推荐采用约45度角俯拍。这个角度能使脸型看起来更尖、更小。
*   **拍摄范围**：可以适当拿远相机，拍摄包含上半身的照片，展示衣着。
*   **姿势与眼神**：可以正面或稍微侧身。眼睛应盯着屏幕上自己眼睛的位置，而不是摄像头。
*   **背景选择**：背景应简洁，避免出现水桶、杂乱柜子等物品。这是一个错误示范，请注意。

## 三、 基础美化：使用美咖相机

假设我们已经拍好了一张照片。现在，我们使用第一个软件“美咖相机”进行基础美化。

打开美咖相机，选择“魔法美颜”功能，然后导入照片。导入后，会看到“美肤”和“素颜”选项。

以下是使用“美肤”功能的核心步骤与参数调整：

1.  **美白**：适度调整，让肤色变白，但避免过度导致不自然。公式：`美白强度 ∈ [低, 中]`
2.  **磨皮**：平滑皮肤，减少噪点和阴影。调整到皮肤看起来光滑但保留纹理的程度。
3.  **去斑**：让软件自动开启此功能。
4.  **去眼袋**：酌情轻微调整。注意保留眼睛下方的“卧蚕”，过度去除会显得不自然。
5.  **瘦脸**：软件可能自动应用。可手动微调，让脸型更小巧。代码式描述：`face_width = original_width * 0.9`
6.  **高鼻梁**：轻微拖动，使鼻子看起来更挺拔。
7.  **大眼**：适度增大眼睛，需遵循比例，避免过度放大导致吓人。比例控制：`eye_size_ratio ≈ 1.1 ~ 1.2`
8.  **亮眼**：提亮眼白部分，让眼睛更有神。建议数值不超过20。
9.  **微笑**：通常不太使用。如需调整嘴型，可轻微尝试。
10. **白牙**：如果照片中露出牙齿，可以使用此功能提亮。

调整完毕后，点击确定并保存照片。美咖相机的修饰功能比美图秀秀更为细致。

## 四、 精细调整：依据“三庭五眼”

基础美化完成后，我们使用“美图秀秀”进行更精细的面部结构调整。这种调整不是随意的，而是有科学依据的，即“三庭五眼”的面部美学比例。

**三庭**：指脸的长度比例。将面部长度分为三等份：
*   上庭：发际线到眉心的距离。
*   中庭：眉心到鼻尖的距离。
*   下庭：鼻尖到下巴的距离。

理想状态下，三庭长度相等。此外，嘴部位于鼻尖到下巴的**二分之一**处。

**五眼**：指脸的宽度比例。以一只眼睛的长度为单位，面宽应刚好是五只眼睛的长度。

如果不理解，可以搜索“三庭五眼”查看标准图示。

现在回到美图秀秀，根据“三庭五眼”的比例来修饰照片。

打开美图秀秀，导入刚才保存的照片，使用“瘦脸瘦身”工具。这个工具不仅可以瘦脸，还能调整多个部位。

以下是基于比例的精细调整步骤：

1.  **调整下巴**：可轻微拉长或收尖下巴，使下庭比例协调。
2.  **收窄脸型**：从两侧向内轻推，缩小面部宽度，使其更接近“五眼”比例。
3.  **修饰眉毛**：如果眉毛形状不佳，可以调整眉峰位置。男性的眉峰可以稍高。
4.  **调整眼睛**：从眼角向外轻拉，可适当增大眼睛。
5.  **缩小鼻翼**：从鼻翼两侧向内轻推，使鼻翼宽度落在两眼内侧。
6.  **修饰嘴巴**：根据比例，对嘴唇形状进行微调。

完成所有调整后，对比原图，你会发现人物看起来更加精神。保存这张照片。

## 五、 进阶技巧：光影与侧面修饰

经过上述步骤，照片已经可以用于朋友圈了。但如果你追求更佳效果，可以学习以下进阶技巧。

美图秀秀的“一键美颜”可能显得假，因为它缺乏光影关系。我们可以使用“去黑眼圈”功能（或类似笔刷）来模拟化妆时的高光和阴影。

**T区提亮**：就像女生化妆打高光一样。将笔刷大小调至适中，在额头、鼻梁（T区）以及眉骨下方轻轻涂抹，增加立体感。公式：`T区亮度 > 面部其他区域亮度`

**苹果肌提亮**：在脸颊苹果肌位置轻微提亮，使面部更柔和。

注意要一点一点地涂抹，避免过度。这样处理后的五官会显得更加立体。

接下来，我们学习如何修饰侧面照片。侧面同样有美学标准。

拍摄一张侧面照片。在美图秀秀中打开，使用“瘦脸瘦身”工具。

以下是侧面修饰的几个关键角度和标准：

1.  **鼻尖-下巴连线**：从鼻尖到下巴画一条直线，理想的侧面应使嘴唇刚好触及或轻微位于这条线内侧。如果嘴巴前突，可以通过后缩嘴唇或前推下巴来调整。
2.  **鼻唇角**：鼻子底部与上唇之间的夹角，理想角度约为90度。可以通过调整上唇或鼻尖位置来接近这个角度。
3.  **鼻额角**：眉弓到山根（鼻梁起点）的连线与鼻梁之间的夹角，理想角度约为120-130度。如果鼻根垫得过高，这个角会变得很小，显得不自然。
4.  **鼻尖角**：鼻小柱与上唇之间的夹角，理想状态也接近45度左右。

调整时，可以将笔刷范围调大，方便操作。例如，将下巴轻微前推，眉弓轻微抬高以加深眼窝，使侧面轮廓更立体。

修饰完毕后，对比原图，效果显著。修图如同产品包装，能让你在朋友圈中更具吸引力。但切记不要过度修饰，以免与真人差距过大。

## 六、 场景选择与总结

修图技术掌握后，拍摄场景也很重要。

**头像拍摄场景建议**：
*   **高端场所**：有条件可选择光线好、背景简洁的高端场所。
*   **平价选择**：如麦当劳、必胜客的某个干净墙角（避免标志性背景入镜），点一杯二三十元的下午茶即可。
*   **约会时拍摄**：与女生在咖啡厅约会时，利用不错的光线自拍或合影，既能获得好照片，也能活跃气氛。
