# 私教必学白鹤《三步恋爱》：05--心态篇

大家好，欢迎来到白鹤《三步恋爱》的心态篇课程。本节课我们将探讨如何建立正确的恋爱心态，提升个人吸引力。

## 01. 硬价值与软价值

**硬价值**指的是我们的生存价值，包括工作、职位、收入水平和社会资源等。**软价值**则是指我们的情感价值，即人格魅力和内在价值。

### 硬价值展示

*   **工作与生活环境**：通过朋友圈展示工作场景、生活环境和生活品质。
*   **生活品质**：展示衣食住行、穿衣打扮、娱乐活动等。
*   **预选**：展示与优秀人士的合影，体现自己的社交圈子。

### 软价值建设

*   **互相尊重**：尊重自己和他人，保持平等心态。
*   **真实**：做真正的自己，不做作，不做虚假展示。
*   **内心强大**：自信、认可自己，不受他人影响。
*   **无欲无求**：保持无需求感，不追逐、不讨好他人。
*   **付出**：展示高价值付出，提供情绪价值，保持社交平衡。

## 02. 建立男神心态

### 互相尊重

*   **尊重自己**：在他人不尊重自己时，保持自尊，不屈服。
*   **尊重他人**：相信他人，给予鼓励和认可。

### 真实

*   **做真正的自己**：不做作，不做虚假展示。
*   **做别人眼中的自己**：展示自己最好的一面，但不要过度伪装。

### 内心强大

*   **自信**：认可自己，不显得弱小。
*   **不受他人影响**：有自己的主见，不寻求他人认同。

### 无欲无求

*   **无需求感**：不追逐、不讨好他人。
*   **无欲无求**：随时做好失去的准备。

### 付出

*   **高价值付出**：展示自己的价值，提供情绪价值。
*   **社交平衡**：互相提供价值，保持社交平衡。
