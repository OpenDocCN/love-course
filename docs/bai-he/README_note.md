# 课程一：白鹤 🦩

在本节课中，我们将要学习一种名为“白鹤”的优雅姿态。我们将从基本概念开始，逐步拆解其构成要素，并学习如何将其组合起来。

## 核心概念

白鹤是一种平衡与延伸的姿态。它的核心在于保持身体中轴线的稳定，同时将四肢向远方舒展。这可以用一个简单的公式来描述其精髓：**稳定 + 延伸 = 白鹤姿态**。

上一节我们介绍了白鹤的核心概念，本节中我们来看看构成这个姿态的几个关键身体部位。

## 身体部位要点

以下是完成白鹤姿态需要关注的身体部位及其要求。

*   **头部与颈部**：头部保持正直，目光平视前方。想象头顶有一根线向上轻轻牵引。
*   **脊柱**：脊柱应保持自然的生理曲线，既不过度弯曲也不后仰，像一根稳固的立柱。
*   **肩膀**：双肩自然下沉并向后微微展开，避免耸肩或向前扣肩。
*   **手臂**：双臂向身体两侧水平展开，掌心向下，指尖向远方无限延伸。
*   **核心区域**：腹部微微内收，以提供身体的稳定性和支撑。
*   **腿部**：单腿站立，支撑腿的膝盖微屈但不要锁死。另一条腿向后抬起，与身体形成一条直线。

了解了身体各部位的要求后，接下来我们将这些要点串联起来，看看完整的执行步骤。

## 执行步骤

以下是完成白鹤姿态的连贯步骤。

1.  自然站立，双脚并拢，双手垂于体侧。
2.  将身体重心缓慢移向右脚，感受右脚扎实地踩在地面上。
3.  吸气，同时将左腿缓缓向后抬起，上半身随之向前倾。
4.  在身体达到平衡的瞬间，向两侧展开双臂，直至与肩同高。
5.  保持呼吸均匀，目光凝视前方固定一点，以帮助维持平衡。
6.  维持姿势15至30秒后，缓慢收回手臂和左腿，回到起始站姿。换另一侧重复。

在练习过程中，初学者常会遇到一些挑战。下一节我们将探讨如何避免常见错误。

## 常见错误与纠正

练习白鹤姿态时，请注意避免以下常见问题。

*   **错误：耸肩或驼背**
    *   **纠正**：时刻提醒自己“沉肩坠肘”，感觉肩膀向地面方向下沉。
*   **错误：支撑腿膝盖锁死**
    *   **纠正**：保持支撑腿的膝盖始终有微小的、有弹性的弯曲。
*   **错误：抬起的腿弯曲或下落**
    *   **纠正**：想象抬起的脚后跟正轻轻抵住一面墙，帮助腿部保持伸直和抬起的高度。
*   **错误：憋气**
    *   **纠正**：保持自然、深长的呼吸，用呼吸的节奏来稳定姿态。

掌握了正确的练习方法和纠错要点后，我们来看看这个姿态能带来哪些益处。

## 练习益处

坚持练习白鹤姿态，可以为身心带来多重好处。

*   **提升平衡能力**：强化脚踝和腿部的小肌肉群，显著改善身体稳定性。
*   **增强核心力量**：为了维持姿态，腹部和背部深层肌肉需要持续工作。
*   **改善体态**：打开胸腔，矫正圆肩驼背等不良习惯。
*   **培养专注力**：在保持平衡的过程中，需要全神贯注，有助于平静心神。
