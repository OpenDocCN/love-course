# 专题-心理建设 🧠

在本节课中，我们将要学习“心理建设”，也就是圈内常说的“inner game”。我们将探讨其重要性、核心组成部分，以及如何通过调整自我认知和对外界的看法来提升内在力量。

## 概述

心理建设是技巧中非常重要的一块。简单来讲，可以把技巧分成两大块。一块是所谓的外加功夫，包括各种技巧、桥段、话术，还有进挪的技巧或者诱惑的技巧。这些都属于外加功夫。

另一块就是你看不到的，强大的内在力量。这种内在力量是其中一种，还包括你对整个游戏的理解、对自己的理解、对女人的理解、对你跟女孩关系的理解。这样一种理解力，还包括各种知识。

还有你自己的气质或者心理状态，比如自信、坚持、放松的心态，或者一种玩的心态。这些东西简单来分可以分成这样两大类：一种是外在的技巧，一种是内在的心理。
