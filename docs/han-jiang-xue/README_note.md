# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：
```python
message = "你好，世界！"
```
这里，`message` 是变量名（盒子的标签），`"你好，世界！"` 是存储的数据（盒子里的东西）。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的类型。Python中几种最常见的基本数据类型如下：

以下是几种核心的数据类型及其简单说明：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由一系列字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`， `“学习编程”`。
*   **布尔值**：只有两种取值，`True`（真）或 `False`（假），常用于逻辑判断。

## 如何查看和转换类型？ 🔄

在编程时，有时我们需要知道一个变量的类型，或者在不同类型之间进行转换。

我们可以使用 `type()` 函数来查看任何变量或值的数据类型。
```python
num = 100
print(type(num)) # 输出：<class 'int'>
```

同时，Python提供了简单的函数来实现类型转换：
*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

例如，将字符串`"123"`转换为整数进行计算：
```python
number_string = "123"
real_number = int(number_string) + 1
print(real_number) # 输出：124
```

## 变量命名规则与建议 📝

为了写出清晰、易读的代码，为变量起一个好名字至关重要。以下是需要遵守的规则和一些建议：

给变量命名时，请遵循以下基本规则：

1.  **只能包含**字母、数字和下划线（`_`）。
2.  **不能以数字开头**。
3.  **区分大小写**（`age` 和 `Age` 是两个不同的变量）。
4.  **不能使用Python的保留字**（如 `if`, `for`, `True` 等）作为变量名。

此外，建议采用：
*   **使用描述性的名称**：例如用 `user_age` 代替 `a`。
*   **常见风格**：多个单词间用下划线连接（如 `student_name`），这被称为“蛇形命名法”。
