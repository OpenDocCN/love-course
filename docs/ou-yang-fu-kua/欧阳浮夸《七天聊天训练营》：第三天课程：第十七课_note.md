# 欧阳浮夸《七天聊天训练营》：第三天课程：第十七课 💬

在本节课中，我们将要学习升级关系的第三步：如何恰当地使用暧昧情话。我们将探讨为何要避免使用“土味情话”，并学习如何通过关键词自然地将对话引导至暧昧氛围中。

## 避免使用“土味情话” 🚫

上一节我们介绍了升级关系的前两步，本节中我们来看看关键的第三步——暧昧情话。在抖音等平台上有许多被称为“土味情话”的套路，但我不建议你使用它们。因为有些土味情话过于油腻，甚至可能引起反感。如果你对女生说太多这类话，可能会产生反效果。

## 使用有效的暧昧情话 ✨

以下是你可以参考和使用的一些有效暧昧情话示例，它们都来自我常用的、反馈效果不错的对话。

*   **当女生问：“你吃饭没有？”**
    *   你可以回答：“没有你在身边，吃什么都没胃口，不吃了。”
*   **当女生说：“晚安。”**
    *   你可以回答：“晚什么安，我只想睡醒你就在身边。”
*   **当女生说：“早上好。”**
    *   你可以回答：“睡醒发现没有你在身边，我才发现昨天的场景原来是一场梦。”

## 情话的核心技巧：关键词与发散性思维 🧠

说情话最好的方式不是生硬地插入，而是需要运用**发散性思维**。其核心公式是：

**捕捉对话中的关键词 -> 自然地将话题频移到暧昧情话上**

你不能莫名其妙地突然对女生说情话，否则她会觉得你行为怪异。正确的方法是，抓住对方话语中的某个**关键词**，然后顺着这个关键词，将对话自然地引导到你想表达的情话上。

本节课中我们一起学习了如何通过捕捉关键词并运用发散性思维，来自然、不油腻地表达暧昧情话，从而有效地升级双方关系。记住，自然和时机是关键。
