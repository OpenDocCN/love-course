# 欧阳浮夸《七天聊天训练营》：第一天课程：第三节2 🚫

在本节课中，我们将要学习聊天中需要避免的另外三个禁忌。上一节我们讨论了聊天节奏和话题选择的问题，本节中我们来看看那些会直接导致聊天失败的行为模式。

## 禁忌三：聊天方式过于无聊 😴

第三个聊天的禁忌是聊天的方式太无聊。这也是很多男性常犯的一个错误。和女生聊天时会很紧张，生怕自己会说错话，回复时过于一本正经。

这样做反而无法吸引对方。因为女性是情绪导向的生物。如果聊天一直保持严肃正经，女生会觉得和你聊天很无聊，从而失去继续交流的兴趣。

**核心方法**：`聊天状态 = 与朋友聊天时的自然放松状态`

不要过于认真。回想你平时是如何与好哥们聊天、如何展现幽默感的。与女生聊天时，就应使用同样的方式。释放你幽默风趣的一面，让女生觉得你这个人很有趣、很好玩，她才会更愿意回复你。

## 禁忌四：传递负能量与攻击他人 ⚠️

第四个聊天的禁忌是传递负能量。千万不能在聊天中向女生抱怨。例如抱怨生活艰难、工作太累或人生不公平。

女生是来和你进行愉快交流的，不是来听你抱怨的。负能量具有感染力，任何人都会本能地避开散发负能量的人。

同时，也不要攻击或讽刺女生。在彼此不熟悉时，就去取笑别人的短处或攻击对方，这是非常错误的。

**换位思考公式**：`陌生人攻击你 -> 你感到不爽 -> 对话终结`

换位思考一下，如果一个陌生人第一句话就来怼你，你会怎么想、怎么做？你肯定会感到不爽。一开始就让人不爽，后续对话自然无法进行下去。

## 禁忌五：使用不良表情包 🙅♂️

第五个撩妹的禁忌是使用过多不良表情。所谓不良表情，例如暴走漫画类、形象丑陋的黑人或女性表情，以及类似“张学友吃屎吧你”这类经典但粗俗的表情。

这些表情可能你觉得好玩，但如果在和女生聊天时发送，会让女生觉得你的品位**很低端**、**很low**，这是非常减分的行为。

以下是需要避免的表情类型列表：

*   **低俗恶搞类**：如暴走漫画、带有侮辱性文字的表情。
*   **形象丑陋类**：刻意丑化特定人群或性别的表情。
*   **负能量类**：如无脸的、垂头丧气的表情，会让人感到不舒服。

本节课中我们一起学习了聊天中需要避免的三个关键禁忌：**避免无聊、摒弃负能量、慎用表情包**。记住，聊天的目标是营造轻松有趣的氛围，而非进行严肃的问答或发泄情绪。保持积极、尊重和趣味性，是让对话持续下去的基础。
