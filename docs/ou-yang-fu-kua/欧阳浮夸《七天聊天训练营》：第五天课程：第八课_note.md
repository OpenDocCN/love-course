# 欧阳浮夸《七天聊天训练营》：第五天课程：第八课_note

在本节课中，我们将学习一套名为“六六大顺”的系统性聊天方法。该方法包含六个明确的聊天阶段和六项核心聊天技术。掌握此框架，能帮助你在与心仪对象交流时，建立有效沟通并提升好感度。

## 核心框架概述

这套方法的核心在于“六个阶段”与“六项技术”的结合。你需要在六个不同的聊天阶段中，灵活运用对应的六项聊天技术。如此，你便能成功掌握聊天技巧。

当你遇到喜欢的女孩时，你也能通过聊天让她对你产生好感。

## 自我检查与针对性提升

对于这六个阶段和六项技术，你需要进行自我审视。请检查自己在实际聊天中，哪一个环节做得不够好。

以下是具体的行动步骤：

首先，学会回顾你之前与女生的聊天记录。检查你是否遵循了这六个阶段的聊天流程。

接着，分析在这六个流程中，具体哪个阶段是你的薄弱环节。

然后，持续强化该阶段所需运用的特定技术。如果你不知如何操作，可以咨询你的导师，他会为你提供指导。

这些技术均来源于实践，是我在与异性互动时经常使用且效果显著的方法。我已将这些技术传授给线下学员，他们学习速度快，并能轻松上手。正因如此，我才将这些技术系统整理，用于本次教学。

本节课的课后作业如下：

第一项作业是翻看之前的聊天记录，审视是否遵循了六个阶段的流程。

第二项作业是识别六个流程中自己做得不足的阶段。

不要试图一蹴而就，指望一口气吃成个大胖子，这是不切实际的想法。专注于针对性练习，逐步提升。

本节课我们一起学习了“六六大顺”聊天法的整体框架，它由**六个阶段**和**六项技术**构成。关键在于通过回顾聊天记录进行自我检查，找出薄弱阶段并加以强化练习。持之以恒地针对性改进，是掌握聊天技巧的有效路径。
