# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述 📋

程序的核心是处理数据。为了处理数据，我们需要将数据存储在计算机的内存中，并给这些存储位置起一个名字，这就是**变量**。同时，数据有多种形式，例如数字、文字等，这些不同的形式就是**数据类型**。本节将详细解释如何创建变量以及Python中常见的数据类型。

## 1. 什么是变量？

变量可以理解为一个贴有标签的盒子，用于在计算机内存中存储数据。你给这个盒子起一个名字（变量名），然后把数据放进去。之后，你就可以通过这个名字来使用或修改盒子里的数据。

创建一个变量非常简单，使用等号 `=` 即可。`=` 被称为“赋值运算符”，其作用是将右边的值赋予左边的变量名。

```python
name = "欧阳浮夸"
age = 25
```

在上面的代码中，我们创建了两个变量：`name` 和 `age`。变量 `name` 存储了字符串“欧阳浮夸”，变量 `age` 存储了数字 `25`。

## 2. 基本数据类型

数据有不同的类型，Python会自动根据你赋予的值来判断其类型。以下是几种最基础的数据类型。

上一节我们介绍了变量的创建，本节中我们来看看Python中常见的数据类型。

以下是三种核心的基本数据类型：

*   **整数 (int)**： 表示没有小数部分的数字，可以是正数、负数或零。
    *   示例： `10`, `-5`, `0`
*   **浮点数 (float)**： 表示带有小数点的数字。
    *   示例： `3.14`, `2.0`, `-9.81`
*   **字符串 (str)**： 表示文本数据，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   示例： `‘Hello’`, `“Python”`, `‘123’`（这里的123是文本，不是数字）

理解数据类型很重要，因为它决定了你可以对数据执行哪些操作。例如，数字可以进行加减乘除，而字符串通常可以进行连接、切片等操作。

## 3. 查看与类型转换

有时我们需要知道一个变量的数据类型，或者在不同类型之间进行转换。

以下是相关的方法：

*   **使用 `type()` 函数查看类型**： 你可以使用 `type()` 函数来查看任何变量或值的数据类型。
    ```python
    print(type(age))    # 输出：<class 'int'>
    print(type(name))   # 输出：<class 'str'>
    ```
*   **类型转换**： 你可以使用 `int()`, `float()`, `str()` 等函数在类型之间进行转换。
    ```python
    number_str = "100"
    number_int = int(number_str)  # 将字符串“100”转换为整数 100
    print(number_int + 5)         # 输出：105

    my_float = float("3.14")      # 将字符串转换为浮点数
    my_str = str(123)             # 将整数转换为字符串
    ```

## 总结 🎯

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。

我们了解到，**变量**是存储数据的命名容器，通过赋值运算符 `=` 来创建。**数据类型**定义了数据的本质，我们重点介绍了三种基本类型：用于整数的 `int`、用于小数的 `float` 和用于文本的 `str`。最后，我们还学会了如何使用 `type()` 函数检查类型，以及如何使用 `int()`, `str()`, `float()` 函数在类型之间进行转换。
