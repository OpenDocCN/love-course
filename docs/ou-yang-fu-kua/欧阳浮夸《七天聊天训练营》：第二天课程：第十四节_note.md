# 欧阳浮夸《七天聊天训练营》：第二天课程：第十四节_note

在本节课中，我们将学习如何通过建立个人素材库，在聊天中自然、真实地展示自身价值，从而创造源源不断的话题并增强吸引力。

上一节我们探讨了聊天的基础框架，本节中我们来看看一个非常实用的“作弊”技巧——建立并运用你的个人素材库。

你需要养成拍摄照片或小视频的习惯，记录下你生活中做过的一些有价值的事情。例如：烹饪美食的过程、在健身房锻炼的瞬间、阅读书籍的时刻、与宠物互动的场景，或是旅行、工作中有意义的片段。

这些素材将成为你聊天中的宝贵资产。每当你新认识一位女生，你会发现，基于这些真实的素材，你们之间能衍生出许多自然且丰富的话题。

因为这些照片和视频就是你个人经历的证据，它们让你的分享显得真实可信。关键在于如何运用这些素材。

以下是具体的使用方法举例：

假设你正在吃白切鸡，这时女生发消息问你“吃饭了吗？”。你可以回复她一个**你上个月自己做饭的小视频**，并告诉她：“我正在做饭呢。”

这种方法的核心在于**扬长避短，展示你真实经历过的、有吸引力的一面**。同一个优质素材，完全可以分享给不同的聊天对象，这能将素材的价值最大化。

我从不担心与女生无话可聊，因为我积累了大量的高质量照片和视频。在聊天过程中，我可以随时自然地发出一个相关素材。

发送素材后，紧接着便可以围绕这张照片或这段视频背后的故事展开讨论。关于这个技术的核心，就是如此。

接下来，我们说一下本节的课后作业。

以下是两项需要你重点练习的内容：

1.  **关键词触发展示**：你要练习在聊天中，一旦对方话语里出现某个关键词，就学会运用这个关键词来展示你想体现的自身价值。
2.  **下钩子技术**：你要练习“把话说一半留一半”的技巧，然后顺势展示你想要展示的价值。这个技术对制造神秘感和吸引女生非常重要。

完成上述练习后，请将你的聊天记录发给你的导师审阅。导师会帮你分析并改良，指导你如何更好地运用这些展示价值的方式。

本节课中我们一起学习了建立个人素材库的重要性及其应用方法。通过积累真实的生活片段，并在聊天中巧妙运用“关键词触发”和“下钩子”技术，你可以更自然、高效地展示自身价值，从而持续吸引对方的兴趣。
