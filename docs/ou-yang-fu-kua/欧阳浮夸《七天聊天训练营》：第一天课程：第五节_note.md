# 欧阳浮夸《七天聊天训练营》第一课：第五节 - 发散性思维 🧠

在本节课中，我们将要学习一个核心的聊天技术：**发散性思维**。这项技术是主动带领话题走向的基础，能帮助你从任何关键词出发，创造出源源不断的聊天内容。

## 什么是发散性思维？

上一节我们介绍了聊天中主动带领话题的重要性，本节中我们来看看实现这一目标的核心方法——发散性思维。后续的所有聊天技巧，都建立在这个思维模式之上。

那么，什么叫做发散性思维呢？其核心是 **“由点及面”**。即当你面对一个关键词时，你需要有能力向四面八方联想，找出所有与这个关键词相关的线索。本质上，这就是**拆解关键词**的过程。

## 如何实践发散性思维？

以下是实践发散性思维的具体方法。其核心操作可以概括为一个公式：

**发散性思维 = 关键词 -> 联想（时间、地点、人物、细节…） -> 组织成句**

例如，关键词是 **“吃饭”**。你需要立刻围绕它展开联想。你可以暂停思考一下，关于“吃饭”你能想到什么？

以下是你可以联想的方向列表：
*   **地点：** 在哪里吃饭？
*   **时间：** 什么时间吃？
*   **人物：** 和谁一起吃？
*   **细节：** 菜的特色、味道、价格、辣不辣？
*   **环境：** 吃饭的环境如何？
*   **方式：** 是自己做还是出去吃？
*   **感受：** 味道怎么样？好不好吃？
*   **原因：** 为什么要出去吃？
*   **时间线：** 以前吃过什么？正在吃什么？以后想去吃什么？

答案可以有很多。你需要培养这样一种思维框架，从多个维度去拆解关键词，主要包括：**时间、地点、人物、味道、颜色、形状、声音、感觉、细节、环境、特色、价格、主题、原因、过去、当下、未来** 等等。

## 实战演练：以“旅游”为例

让我们用另一个关键词 **“旅游”** 来实战演练，巩固刚才的思维框架。

以下是围绕“旅游”进行发散联想的问题列表：
*   **时间：** 什么时候去旅游？
*   **地点：** 旅游的地点在哪里？
*   **人物：** 都有哪些人去？
*   **感官：** 吃的东西是什么味道？能看到什么颜色？东西的形状如何？能听到什么声音？
*   **感受：** 带来了什么样的感觉？
*   **细节：** 发生过哪些事情的细节？
*   **环境：** 那里的环境是什么样的？
*   **特色：** 有什么别处没有的特色？
*   **价格：** 吃、住、行等各项花费多少？
*   **主题：** 这次旅游的主题是什么？
*   **原因：** 为什么选择这个地方？
*   **时间线：** 曾经去过哪里？最近去过哪里？未来想去哪里？

只要你学会将所有有利的线索整理起来，任何关键词都可以被这样拆解。通过这种拆解手法，你将拥有聊不完的话题。

## 需要达到的程度

你需要练习到这种程度：**只要出现任何一个关键词，你都能立刻联想到与之相关的任何信息，然后组织成句子，把你想到的东西自然地说出来。** 这就是拆解式的发散性思维。

本节课中我们一起学习了**发散性思维**这项核心聊天技术。我们明白了它如何通过拆解关键词（如吃饭、旅游），从时间、地点、人物等多个维度进行联想，从而创造出无限的话题。掌握这个思维模式，是成为聊天高手的必经之路。
