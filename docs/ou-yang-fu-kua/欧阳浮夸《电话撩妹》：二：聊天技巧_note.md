# 欧阳浮夸《电话撩妹》：二：聊天技巧_note

在本节课中，我们将学习如何运用发散性思维、讲故事、幽默互动以及引导聆听等核心技巧，来构建与女生之间流畅、有趣且能产生吸引力的对话。

## 发散性思维与故事引入

上一节我们介绍了“疑点扩面”的发散性思维。本节中我们来看看如何将这种思维应用到实际聊天中。

发散性思维是指在聊天时，捕捉到对方话语中的某个关键词，并由此联想到任何与之相关的事物，然后自然地将这些联想表达出来。掌握这种思维后，我们需要学习下一个技术：**讲故事**。

讲故事不能凭空开始。如果直接说“我给你讲个故事”，会显得突兀。但如果在聊天中恰好聊到某个关键词，你的发散性思维让你通过这个词联想到一个故事，再讲出来就会非常自然。例如，聊到“旅游”时，你可以想起自己某次旅行的经历，然后分享出来。

## 讲故事的公式 📖

以下是讲故事的标准化步骤，请务必记住这个公式：

1.  **关键词**：捕捉对话中的核心词汇。
2.  **铺垫**：自然地将话题引向你的故事。
3.  **植入内容**：开始讲述故事，但不要一次性说完。
4.  **互动**：在讲述中与对方交流，等待回应。
5.  **开放式结尾**：在故事讲完后，提出一个开放性问题，引导对方继续谈话。

### 公式应用示例

假设聊天话题是“旅游”。

*   **关键词**：旅游。
*   **铺垫**：“说到旅游，我就想起了菲律宾，简直是终身难忘。”
*   **植入与互动**：当对方问“菲律宾有什么好玩的？”时，你逐步展开：“特别多好玩的，可以去潜水、和鲨鱼拍照、在悬崖边跳水。最令我难忘的是那里的海鲜，现在想起来都会流口水。” 对方可能会回应“有那么好吃吗？”，你再继续：“那里的皮皮虾，比我的手臂还粗，清水煮都好吃得不行。”
*   **开放式结尾**：在对方表示羡慕后，你可以说：“那有机会我带你去。” 接着提问：“那你去过哪些印象深刻的旅游地吗？” 或者 “有没有特别想去的地方？”

回顾这个例子，关键在于**互动**。不要一口气讲完整个故事，而是说一点，留一点，等待对方回应后再继续。这样能让对方融入故事，形成一来一往的聊天节奏。对方回应后，其话语中又会产生新的关键词，你可以再次运用“捕捉关键词 -> 铺垫 -> 植入内容 -> 互动 -> 开放式结尾”的公式，让话题源源不断。

**讲故事的本质**是：在聊天中出现关键词时，联想到相关经历，通过铺垫自然说出，在讲述中与对方互动，最后以开放性问题结束。多加练习，你就能熟练掌握。

## 故事的核心：植入吸引元素 💘

然而，仅仅会讲故事是不够的。对方如果想听故事，可以去听电台。我们讲故事的核心目的是：**通过故事让女生对我们产生好感，进而喜欢我们**。

因此，故事的选材至关重要。我们的故事中需要包含能吸引女生的元素，主要分为两大类：**阳元素**与**阴元素**。

*   **阳元素**：指正经的、体现实力与价值的特质，如：会做饭、专一、积极向上、有责任心、事业有成等。
*   **阴元素**：指不正经的、能调动情绪价值的特质，与“感觉”密切相关。

一个吸引人的故事需要**同时包含阳元素和阴元素**。阳元素建立基础吸引，阴元素则创造“感觉”。

### 如何植入阳元素

如果你想展示自己积极向上的一面，可以围绕“加班”这个关键词讲故事：“说到加班，我就想起刚实习时，每天工作12小时，除了睡觉就是上班，只为快速学习、胜任工作。” 这个故事就呈现了努力、积极的形象。

如果你想展示会做饭，可以说：“我最讨厌吃外卖了，没营养又难吃。所以只要有空，我都自己去买菜做饭。” 这样自然植入了“会照顾生活”的阳元素。

## 创造感觉：幽默与逗逼技巧 😄

仅有阳元素（硬价值）不足以让女生产生“感觉”。感觉的本质是**情绪波动**。如果聊天始终平平淡淡，没有激起她的情绪，她就无法对你产生感觉。

为了让女生产生情绪波动，我们需要运用阴元素，即**幽默加逗逼**的技巧。这包括了说俏皮话、搞怪、耍赖、无厘头等，核心就是一个字：**“皮”**。聊天中偶尔“皮”一下，能有效调动女生情绪。

以下是三种核心的“皮”的思维：

### 1. 自恋思维
即用夸张的方式喜爱自己，引人发笑。
*   **示例代码/话术**：
    *   “我从来不相信一见钟情，直到我照镜子的时候会有一丝动摇。”
    *   “我每次出门都很头痛，因为走到哪都是焦点。长得太帅不是我的错。”
    *   “像我这种长得帅又有才华的人，你和我走在一起压力大吗？”

### 2. 夸张思维
将事情夸大化，产生幽默效果。
*   **示例代码/话术**：
    *   （对方吃饭时）“你要多吃点，台风来了，你这么轻被吹到别人家怎么办？”
    *   （反说对方胖）“最近台风多，我好羡慕你，再大的台风都吹不动你，好安全。”
    *   （对方加班时）“让一个娇滴滴的小姑娘加班到现在？还有没有王法了，我建议你报警。”
    *   （称赞对方）“天啊，你是仙女下凡了吗？怎么可以这么好看。”

### 3. 反差思维
前后表达不一致，但合情合理，通常是“先扬后抑”或“先抑后扬”。
*   **示例代码/话术**：
    *   “你的眼睛好好看，尤其是那个黑眼圈，太迷人了。”（先赞后踩）
    *   “哇，你的照片好好看哦！感谢美图秀秀！”（先赞后调侃）
    *   “我这个人不能吃辣，你千万别放太多辣椒，放两斤左右就好了。”（前后矛盾制造幽默）

**如何练习这些思维？**
1.  **模仿**：多研究聊天案例，分析对方发笑（如回复“哈哈哈”）前，对话中运用了何种技巧。
2.  **学习**：有目的地观看《吐槽大会》、《脱口秀大会》、《奇葩说》等语言类节目。重点分析嘉宾如何运用技巧让人发笑，而非单纯娱乐。

## 高阶技巧：引导与深度聆听 👂

最后一个关键技术是**引导女生说话加深度聆听**。这能带来两大好处：
1.  **知己知彼**：对方说得越多，透露的个人信息（线索）就越多，便于你更好地了解她。
2.  **建立依赖**：能让平时高冷的女生打开话匣子，对你不停诉说，这容易让她对你产生情感依赖。

### 如何引导：开放式提问 + 牵引话术
引导的核心是 **“开放式提问”** 结合 **“牵引话术”**。
*   **牵引话术**是在对方倾诉时，引导她继续深入的话语。常用句式包括：
    *   “然后呢？”
    *   “接下来怎么样了？”
    *   “所以你的意思是？”
    *   “为什么你会有这样的想法/这样做呢？”
    *   “你继续说，我在认真听。”

**应用示例**：
*   对方说工作上遇到事情，你可以问：“然后呢？这是怎么回事？”
*   对方聊到养宠物，你可以问：“你为什么想养宠物？对宠物有什么看法？”
*   对方发表一个观点，你可以问：“为什么你会这样想？”

### 深度聆听：认同与共情
仅仅引导对方说话还不够，关键在于**听完后站在她的立场，认同她的看法、想法和观点**。这让她觉得遇到了知音。

**具体操作**：
*   对方抱怨工作不公，你可以说：“我也是，最痛恨欺负员工的人了。”
*   对方讨厌出轨行为，你可以说：“我也觉得这样的人好恶心，好好谈恋爱不行吗？”
*   对方主张“活在当下”，你可以说：“没错，人生短短几十年，该享受时就要享受。”

**核心原则**：人类都渴望被肯定和认可。在聊天中多使用以下表达：
1.  “我也经历过，所以很能理解你的感受。”
2.  “你说得很对，我非常认同你的观点。”
3.  “听了你的话，我受益匪浅。”
4.  “我懂你，那种感觉确实不好受。”
5.  “你简直就是我的知音。”

（注意：对于明显错误或违背道德的观点，如不孝顺父母，则无需认同。）

## 课程总结 📚

本节课我们一起学习了三大核心聊天技巧：

1.  **讲故事的技术**：牢记公式 **关键词 -> 铺垫 -> 植入内容 -> 互动 -> 开放式结尾**。通过互动让故事生动，并以提问延续话题。
2.  **创造感觉的技术**：运用**幽默与逗逼**技巧，掌握**自恋、夸张、反差**三大“皮”的思维，并主动向优秀的语言节目学习，以激发女生的情绪波动。
3.  **引导与深度聆听的技术**：多用**开放式提问**和**牵引话术**引导对方表达。更重要的是，在聆听后**站在她的立场给予认同和共情**，让她感觉被理解和珍视，从而建立深层连接。
