# 欧阳浮夸《七天聊天训练营》：第一天课程：第四节_note

在本节课中，我们将要学习如何在聊天中有效“加分”，避免减分行为。我们将深入探讨四个关键的加分点，并揭示成功聊天的两个核心步骤与六项必备技术。

上一节我们介绍了会“减分”的聊天方式，本节中我们来看看如何通过聊天为自己“加分”。

在聊天中能够让你加分的行为主要基于四个核心点。以下是这四个关键点的详细说明：

*   **主动带领话题与关系**：你需要主动引导话题的走向，并主动推动双方关系的升级。
*   **提供核心价值**：你能够向对方展示你的养育价值（如照顾、保护能力）或繁衍价值（如健康、基因优势）。
*   **提供情绪价值**：你能够为对方提供正面的情绪体验，例如快乐、舒适感和被理解的感觉。

只要你能够在互动中贯彻以上四点，聊天就不会像想象中那么困难。整个聊天过程可以高度概括为两个核心步骤：**吸引女生**和**升级关系**。你只需要专注于并做好这两步。

要达到熟练运用这两个步骤的程度，你需要掌握六项具体的聊天技术。我们必须先学会这六项技术，然后再学习如何在实战聊天中灵活地组合与应用它们。

当你完成了以上学习，你就能够系统地掌握聊天这门艺术。

本节课中，我们一起学习了聊天的四个加分点、两个核心步骤以及六项技术的学习路径。掌握这些框架是开启有效沟通的第一步。
