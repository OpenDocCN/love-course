# 欧阳浮夸《七天聊天训练营》：第一天课程：第七节 🧠 发散性思维训练

在本节课中，我们将要学习聊天中的核心技巧——发散性思维。我们将重点掌握其两个关键组成部分：拆解与平移，并通过具体的方法进行练习，以提升你与女生聊天时的应变能力和回复质量。

上一节我们介绍了聊天的基础框架，本节中我们来看看如何运用发散性思维来丰富你的聊天内容。

发散性思维包含两个核心点。第一个是**拆解**，即把对话中的关键词分解开来，然后围绕这个关键词进行全方位的联想，思考与它相关的一切。第二个是**平移**，即把关键词的含义或场景进行转移，平移到与你的聊天目的相关联的语境中去。

如果想要学会聊天，第一步就是先锻炼好拆解和平移的能力。

以下是具体的锻炼方法。今后你和女生聊天时，要养成一种习惯：看到对方话语中的一个关键词，不要急于回复。你需要先对这个关键词进行拆解。套用前面课程中教过的拆解公式，看看你能从这个关键词中分解出多少个不同的点。接着，再去尝试平移这个关键词，看看你能通过平移衍生出几个不同的方向。

最后，从你通过拆解和平移所构思出的多个句子中，挑选出一个你认为最佳的答案，再去回复女生。

在前期锻炼发散性思维时，你可能想出的点或方向不够多。但必须养成这个习惯。养成习惯之后，你就能做到只要出现一个关键词，立刻能联想到许多不同的回复方式。这个思维模式非常重要。

如果仅靠自己与女生聊天来练习，效率可能较慢。因此，建议你使用我的教学案例进行专项锻炼。我的案例都制作成了PPT形式，你可以慢慢翻阅。看到女生回复我的内容时，先暂停，不要看我的答案。你需要先自行拆解和平移女生的这个关键词，思考如果是你，会如何回复。思考你会如何拆解，如何平移，最终会选用基于平移还是拆解的回复。多构思几个不同的答案，并从中挑选出最佳回复。

这时，你再翻看我是如何回复女生的。对比你的答案和我的答案，看看区别在哪里，思考应该如何改进才能更接近我的回复思路。至少用20个以上的案例进行这样的翻阅和锻炼，你就能大致掌握拆解和平移的方向了。此后，在实际聊天中也要不断锻炼发散性思维，才能真正掌握它。

关于发散性思维的课后作业是：翻看我的聊天案例，至少看完20个。不断按照本节课教授的方法，锻炼你的发散性思维。将你的练习感悟写成笔记，发给你的导师检查。这就是你今天需要完成的课后作业。

本节课中我们一起学习了发散性思维的两个核心操作：**拆解**与**平移**，并通过分析案例和自主练习的方法来掌握这一技巧。坚持训练，你将能显著提升聊天的灵活性与吸引力。
