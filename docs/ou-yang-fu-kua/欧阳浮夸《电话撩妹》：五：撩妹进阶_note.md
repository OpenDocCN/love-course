## 升级关系的两个阶段

在撩妹过程中，有两个主要阶段：**吸引阶段**和**升级阶段**。吸引阶段的核心是不断展示你的硬价值（如财富、地位）和软价值（如性格、情商）。而升级阶段，则需要我们学会判断女生的兴趣值，并采取相应的行动。

## 判断女生的兴趣值

女生的兴趣值决定了你能说什么、做什么。如果她对你的兴趣值很低，你绝不能讲暧昧的话，否则会引起反感。如果兴趣值中等，可以说一些轻微的暧昧话语，但不能深入。如果兴趣值很高，那么你的表达可以更加自由。

兴趣值来源于你在聊天中展示的软价值和硬价值。提高兴趣值的方法就是持续展示这些价值。

以下是判断兴趣值的八个指标：

1.  她和你说话的内容多吗？
2.  她和你聊天时是否专心，还是边聊边做其他事？
3.  她会主动挑起新话题吗？
4.  她会主动与你分享她的生活吗？
5.  聊天时，她是否有情绪波动（如开心、生气）？
6.  她会主动关心你吗？
7.  她会主动约你吗？
8.  她会主动对你说暧昧的话吗？

如果你收集到的指标少于三个，请暂时不要尝试升级关系。如果达到三个或以上，则可以尝试。当然，指标越多越好。

## 电话升级关系的四大技巧

明确了兴趣值后，我们来看看在电话上具体如何操作。以下是四个循序渐进的升级关系技巧。

### 技巧一：真正的关心

很多男生不懂得如何关心人，方式很“直男”。例如，女生说不舒服、感冒或来例假时，只会说“多喝热水”，这会让女生觉得你在敷衍她。

真正有效的关心需要遵循一条核心公式：**细节 + 行动**。

**公式：关心 = 细节 + 行动**

我们要关心到细节里去，避免说别人说过的话。例如，当女生感冒时：

**错误示范**：“多喝热水，记得买药吃。”

**正确示范**：“感冒如果比较轻，我建议先别吃药，免得产生依赖。可以先喝点热水，一定要是烧开的水，晾到你能承受的最高温度再喝，这样效果最好。记得每小时喝一杯。同时保持室内空气流通，出门多穿件衣服。如果晚点还是难受，我给你买我常用的感冒药，你给个地址，我用美团跑腿送过去。如果吃了药还不好，我明天陪你去医院看看。”

再比如，关心女生按时吃饭：
“一定要记得按时吃饭，不然长期对胃不好，以后就尝不到美食了。如果你老忘记，我建议你设个闹钟。或者，以后每到饭点，我准时提醒你吃饭？”

运用“细节+行动”的方式，女生才能感受到你是真心关心，而非敷衍。

### 技巧二：禁止效应

“禁止效应”是指人容易对与自己相似或有共同经历的人产生好感。例如，在外地遇到老乡会感到亲切。

在电话中，可以通过以下三个手法实现禁止效应：

1.  **模仿她**：模仿她说过的话、语气和语调。例如，她常说“这不科学”，你也可以用同样的语气说“这不科学”。
2.  **经历她的过去**：在聊到童年或过往经历时，分享你们相似的经历。例如，她说小时候吃泡泡糖，你可以说“我也吃过，还不小心粘到头发上”。她说职场被欺负，你可以说“我理解，实习时他们也总把麻烦工作推给我”。
3.  **认同她的观点**：人都渴望被肯定。当她表达某个观点时，直接表示认同。例如，她说“最不能容忍出轨”，你可以说“对，我也觉得，出轨的人很恶心”。她说“人生要活在当下”，你可以说“是的，人生短暂，一定要珍惜现在”。

这个技巧能让你更容易走进女生的内心世界。

### 技巧三：说浅层次情话（土味情话）

运用前两个技巧后，女生的回应会越来越好。此时，可以开始说一些浅层次的情话，但切忌一开始就过于深入。

这里需要运用之前学过的**发散性思维**。在升级关系阶段，发散性思维不再用于联想展示价值的话题，而是用于联想情话。

**核心：通过关键词，利用发散性思维联想到相关的情话。**

例如：
*   **关键词：睡觉** -> “昨晚我睡得不安稳。” “为什么？” “没有你在身边，我怎么可能睡得安稳。”
*   **关键词：吃饭** -> “不能和你一起吃饭，我觉得吃什么都没味道。”
*   **关键词：上班** -> “不知道为什么，上班时一有空闲，我就会情不自禁想起你。”
*   **关键词：晚安** -> “晚什么安，我只想睡醒你就在身边。”

这些就是“土味情话”，在抖音等平台有很多。**但切记**，一定要在吸引到位后才能使用，否则会引起反感。

### 技巧四：说深层次情话（含肢体动作）

深层次情话的特点是**包含肢体动作的想象**。这有两个好处：1. 测试女生的接受底线；2. 在女生脑海中构建画面感，为见面后的实际接触做铺垫。

例如：
*   **关键词：洗澡** -> “等等，我眼睛正往你家的浴室飞过去。”
*   **关键词：身高** -> “你这个身高，踮起脚尖就能亲到我了。”
*   **关键词：吃饭** -> “再吃下去，我可能就抱不动你了。”
*   **关键词：做饭** -> “你做饭的时候，我要从后面抱着你。”
*   **关键词：见面** -> “见面时，我要给你一个大大的拥抱。”
*   **关键词：睡觉** -> “我想抱着你睡，把我的手臂给你当枕头。”

如果女生在聊天中不抗拒“牵手”的话题，那么见面后你就可以尝试直接牵手。同理，不抗拒“拥抱”的话题，见面就可以尝试拥抱。

## 核心要点与总结

本节课中我们一起学习了在电话中与女生升级关系的四大技巧：

1.  **真正的关心**：运用 **`细节 + 行动`** 的公式，避免敷衍。
2.  **禁止效应**：通过模仿她、经历她的过去、认同她的观点，建立相似性和亲密感。
3.  **浅层次情话**：在吸引到位后，利用**发散性思维**说一些土味情话，升温氛围。
4.  **深层次情话**：使用包含**肢体动作**描述的情话，测试底线并构建见面预期。

**运用的核心**与吸引阶段一样，在于**不断循环和重复**。不是用一次即可，需要持续、熟练地运用这些技巧，效果才会越来越好。
