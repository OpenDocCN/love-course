# 欧阳浮夸《电话撩妹》：一：聊天基础_note

在本节课中，我们将要学习电话聊天的基础技术。掌握这些技术是玩转电话游戏、与女生顺畅交流的前提。课程将逐一讲解六个核心聊天技巧，帮助你建立自信、自然的对话能力。

## 一、观与问的技巧 👀❓

上一节我们介绍了课程的整体框架，本节中我们来看看第一个核心技巧：观与问。这是开启对话的基础，关键在于如何从观察和提问中自然引出话题。

### 1. 观的技巧：所见即话题

你的眼睛能够看到的任何东西都是话题。这包括女生的头像、签名、网名、背景墙以及朋友圈内容等各种线索。

记住一个核心公式：**你看到的 + 你想到的 = 话题**。

以下是利用“观”的技巧开启话题的几个例子：

*   **观察头像**：如果女生的头像侧脸像某个明星，你可以说：“有没有人跟你说过，你的侧脸有点像某某某？”
*   **观察签名**：如果女生的个性签名显得有思想，你可以说：“我非常认同你的签名，感觉你挺有思想的，和一般的女生不大一样。”
*   **观察朋友圈**：如果女生经常发美食，你可以说：“看来你是一个小吃货。” 如果她喜欢旅游，你可以说：“你也是个闲不住的人，看你到处跑。”

打电话时，我们不仅能“观”，还能“听”。听到的任何声音也可以成为话题。

*   **听到声音**：如果女生声音好听，你可以说：“哇，你的声音也太好听了吧，如果你去参加中国好声音的话，肯定能拿个冠军回来。”
*   **听到笑声**：如果女生笑声夸张，你可以说：“你小声一点，不要把邻居家的小朋友给吓哭了。”

总之，你看到或听到的任何信息，加上你的想法，就能构成一个话题。利用好这个技术，你就能和女生有聊不完的话题。

### 2. 问的技巧：优化提问方式

90%的男生和女生聊天时喜欢“查户口”，这种聊天方式会让女生非常反感。但提问又是人际交往中必不可少的部分。我们无法避开提问，但可以优化它。

以下是关于“问”的两个核心技巧：

**技巧一：不要问，直接猜**

把你想要问的问题，直接转换成猜测。这能避免查户口的感觉。

*   **例子**：想知道女生的职业，不要说“你是做什么工作的？”，而可以说：“我猜你还是一个学生，还没毕业的那种。”
*   **其他猜测示例**：
    *   “我猜你今年只有22岁。”
    *   “我猜你应该是来自湖南的，那么喜欢吃辣。”
    *   “我猜你一定是个单身。”
    *   “我猜你现在肯定是躺在床上面玩手机。”

记住六个字：**不要问，直接猜**。

**技巧二：提问后必须给予反馈**

查户口式聊天的典型模式是连续提问，不给反馈。正确的做法是：对方给出答案后，你必须对这个答案给予反馈。

记住一个公式：**提问 -> 对方回答 -> 给予答案反馈**。

*   **例子1：问名字**
    *   你：“你叫什么名字？”
    *   女生：“我叫小方。”
    *   **错误做法**：接着问“你是做什么工作的？”（这是查户口）。
    *   **正确做法（给予反馈）**：你可以说：“村里有个姑娘叫小方，长得好看又善良。”（联想到歌词并反馈）

*   **例子2：问年龄**
    *   你：“你今年多大了？”
    *   女生：“20了。”
    *   **反馈**：“20岁是个适合谈恋爱的年纪。” 如果女生25岁，则说：“25岁，那是适合结婚的年纪。”

*   **例子3：问职业**
    *   女生：“我是做销售的。”
    *   **反馈**：“你的口才一定很好。”
    *   女生：“我是做护士的。”
    *   **反馈**：“原来是我们可爱的白衣天使。”
    *   女生：“我是做老师的。”
    *   **反馈**：“真是羡慕你的职业，可以天天都去欺负小朋友。”

*   **其他常见问题反馈示例**：
    *   **问**：“吃饭了没有？”
        *   女生：“没有。” -> **反馈**：“那赶紧去吃，不然会瘦的。”
        *   女生：“吃了。” -> **反馈**：“你要多吃一点，不然台风来了，就会把你刮到我家来了。”
    *   **问**：“下班了吗？”
        *   女生：“还没。” -> **反馈**：“报警吧，你们家老板真是没良心，让一个娇滴滴的女生加班到现在。”
        *   女生：“下班了。” -> **反馈**：“好羡慕，我还在苦逼的加班中。”
    *   **问**：“你在干嘛？”
        *   女生：“在看剧。” -> **反馈**：“这么无聊的吗，应该出去玩。”
        *   女生：“在打游戏。” -> **反馈**：“求带上陪你一起玩。”
        *   女生：“在逛街。” -> **反馈**：“缺不缺一个拎包的，我很乐意去做这个事情。”
        *   女生：“在做饭。” -> **反馈**：“申请一个蹭饭的名额。”

养成对女生的回复给予反馈的习惯。如果你不知道如何反馈，可以多看看导师的聊天案例，学习并记下来。

## 二、开放式提问与回答 💬

上一节我们学习了如何通过观察和优化提问来开启对话，本节中我们来看看如何让对话持续深入，避免“一问一答”就结束的尴尬局面。这需要掌握开放式提问与回答的技巧。

### 1. 开放式提问技巧

如果你和女生聊天时，一个问题很快聊完，接着又提新问题，很快又聊完，最后无话可聊，原因就在于你的提问和回答都是“封闭式”的。

*   **封闭式提问**：答案只有“是”或“否”，或非常简短。
    *   **例子**：“你喜欢宠物吗？” -> 女生只能回答“喜欢”或“不喜欢”。
*   **开放式提问**：引导对方展开描述，释放更多信息。
    *   **例子**：“你对养宠物有什么看法？” -> 女生可能会说出一大堆想法。

**将封闭式问题转化为开放式问题的关键词**：为什么、什么样的、怎么样、都有什么呢。

*   “你平时会去健身吗？”（封闭） -> “你对健身这个事情怎么看？”（开放）
*   “你喜欢陈伟霆吗？”（封闭） -> “你为什么会喜欢陈伟霆？”（开放）
*   “你喜欢成熟的男生吗？”（封闭） -> “你觉得成熟的男生会是什么样的？” 或 “成熟的男生都有些什么东西，会让你喜欢的呢？”（开放）

经常运用开放式提问，把它变成习惯。这样对方给出的答案会包含更多信息和关键词，你既能了解更多信息，也能抓住关键词延伸话题。女生也特别喜欢愿意倾听她诉说的男人。

### 2. 开放式回答技巧

很多男生除了只会封闭式提问，还会犯“封闭式回答”的错误。女生通常也不太会聊天，她们问的问题也往往是封闭式的。

*   **女生常问的封闭式问题**：“吃饭了吗？”、“下班了吗？”、“你喜欢猫吗？”、“你在干嘛？”

如果你也只回答“吃了”、“还没”、“喜欢”、“看书”，话题立刻就结束了。所以，我们还要学会开放式回答。

记住一个公式：**回答问题 + 描述你的状态 + 反问对方**。

*   **例子1：女生问“你吃饭了吗？”**
    *   **封闭式回答**：“吃了。”（话题结束）
    *   **开放式回答**：“我刚刚吃过了（**回答问题**）。吃了重庆火锅，现在辣得我怀疑人生，额头还是满头大汗（**描述状态**）。你呢？（**反问对方**）”
*   **例子2：女生问“你在干嘛？”**
    *   **封闭式回答**：“看书。”（话题结束）
    *   **开放式回答**：“我在看书（**回答问题**），现在准备去洗个苹果吃，有点饿（**描述状态**）。你呢，你在干嘛？（**反问对方**）”

将开放式提问和回答的方式变成习惯，不断练习到条件反射。这样，别人和你聊天会觉得很轻松，也会越来越喜欢跟你聊天。

## 三、疑点扩面（发散性思维） 🧠

上一节我们学习了如何让对话内容更丰富，本节中我们来看看一个非常实用且常用的高级技巧：疑点扩面，也就是培养发散性思维。它能让你从一个关键词联想到整个话题面。

培养发散性思维，需要做到以多维度去思考一个关键词，在此基础上进行想象，产生多条思路，并向外扩散。具体可以通过三种思维来练习：

**1. 平面思维**：给你一个关键词，你要看到它的整个平面（相关性和相似性）。
**2. 立体思维**：给你一个关键词，你要联想到它的整个立体（相关故事、经历、感受）。
**3. 逆向思维**：给你一个关键词，你要从事情的反面或侧面去思考（打破常规，跳出框架）。

**举例：关键词“电影”**

*   **平面思维（看到平面）**：科幻片、动作片、悬疑片、爱情片、搞笑片、2D/3D/4D。
*   **立体思维（联想到立体）**：
    *   你印象最深刻的电影（如《变形金刚》）。
    *   曾和暗恋的女生一起看电影的经历。
    *   在电影院遇到的趣事或尴尬事。
    *   看《前任3》时哭得稀里哗啦。
    *   每次看漫威电影都会等彩蛋。
*   **逆向思维（跳出框架）**：
    *   女生：“我喜欢看电影。”
    *   常规回复（在框架内）：“我也喜欢，你喜欢看什么类型？”
    *   **逆向破框回复（跳出框架）**：“我也超喜欢看电影的，有空我们一起约个电影吧。”（不再拘泥于聊电影本身，而是跳到邀约框架上）

**再举例：关键词“宠物”**

*   **平面思维**：猫、狗、其他动物；大型、中型、小型；白色、黑色、黄色；可爱的、傻傻的；金毛、萨摩耶、二哈、泰迪、阿拉斯加；波斯猫、加菲猫、英短、布偶猫。
*   **立体思维**：
    *   小时候被邻居家的猫抓过，打了疫苗，从此怕猫。
    *   朋友家的二哈把鞋子、袜子、电线全咬破了。
    *   自己养的金毛特别黏人，回家总跟着。
    *   朋友养的加菲猫超胖，肚子鼓鼓的，像动画片里一样。
*   **逆向思维**：
    *   女生：“我喜欢狗。”
    *   **跳出框架回复**：“那以后我们一起养一只吧，下班之后你牵着狗，我牵着你去散步。”

**如何练习发散性思维？**

开始留意身边的任何东西，进行联想练习。

*   看到一本书 -> 思考与书相关的一切（作者、内容、你读它的经历、类似的书）。
*   看到一个杯子 -> 思考与杯子相关的一切（材质、你用它喝过什么、有没有特别的故事）。
*   看到一张凳子 -> 联想与凳子相关的一切。

说白了，就是看到什么，就要立刻联想到和它相关的东西。在电话聊天中，你听到女生说的任何关键词（如“吃饭”），都要马上联想到相关事物，并立刻做出反馈。电话聊天不容许长时间停顿，所以必须通过不断练习，将这种思维练成习惯和条件反射。

真正的聊天高手，聊天时都是“自然流”，想到什么聊什么，顺着关键词一直聊下去。根据以上方法坚持练习，你也能做到。

**本节课总结**：在本节课中，我们一起学习了电话聊天的六个基础技术中的前三个核心技巧：
