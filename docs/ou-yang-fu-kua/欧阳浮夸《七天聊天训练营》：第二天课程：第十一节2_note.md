# 欧阳浮夸《七天聊天训练营》：第二天课程：第十一节2 💬

在本节课中，我们将要学习如何应对女生提出的封闭式问题，并掌握一种有效的回答公式，让对话能够轻松、自然地延续下去。

上一节我们介绍了开放式提问的技巧，本节中我们来看看如何应对女生常提出的封闭式问题。

女生在聊天时，提出的问题大多是封闭式的。这些问题通常只需要用“是”或“否”来回答。例如，“你喜欢吃辣吗？”、“你喜欢看电影吗？”、“你下班了吗？”、“你在干嘛呀？”。这些问题的答案范围很窄，你只能回答“吃了”或“没吃”，“下班了”或“没下班”。

如果你直接给出封闭式答案，对话很容易陷入僵局。例如，女生问“你下班了吗？”，你回答“下班了”。此时，从女生的角度来看，她很难接话。除非她再提出一个新问题或开启一个新话题，否则对话就会中断。这会让她觉得和你聊天很累，需要不断动脑筋。而女性通常不喜欢在聊天中过度思考。

因此，不要用封闭式答案来回应女生的问题。你需要记住一条核心公式：**回复对方 + 描绘细节 + 反问对方**。

以下是该公式的应用解析：
*   **回复对方**：首先直接回答对方的问题。
*   **描绘细节**：然后围绕你的答案，增加一些具体的、生动的描述。
*   **反问对方**：最后，将话题抛回给对方，引导她继续发言。

例如，当女生问你“你吃饭了吗？”，你可以这样回答：
1.  **回复对方**：“我刚刚吃过了。”
2.  **描绘细节**：“吃了重庆火锅，辣死我了。”
3.  **反问对方**：“你呢？你吃了没有？”

你这样回复，就为女生提供了多个可以接话的方向。她可以就“重庆火锅”这个关键词展开，也可以直接回答你的反问。你不需要她费力思考，她会觉得和你聊天很轻松。

我们再来举个例子。当女生问你“你在干嘛呀？”，不要简单地回答“在吃饭”、“在打游戏”或“在睡觉”。

记住，一旦女生问“你在干嘛”，这往往是你展示个人价值的最佳时机。运用的公式是一样的。

例如，女生问“你在干嘛？”，你可以回答：
*   **回复对方**：“我正在看一本关于心理学的书籍。”
*   **描绘细节**：“想多了解一下这些复杂的人性。”
*   **反问对方**：“你呢？你在干嘛？”

或者另一个例子：
*   **回复对方**：“我正在陪我父母看电视。”
*   **描绘细节**：“（看的是）那种无脑的抗日剧，看得我一直打瞌睡。”
*   **反问对方**：“你呢？你怎么还不睡？”

关键在于，不要只是傻傻地回复答案，而是遵循“**答案 + 细节 + 反问**”的结构，这样你们就能很顺利地继续聊下去。

以上就是关于“回答技术”的核心内容。那么，关于这项技术的课后作业如下：

以下是需要着重练习的三点：
1.  学会把你以后想问女生的所有问题，从直接“问”变成直接“猜”。（例如，不说“你喜欢看电影吗？”，而说“我猜你应该喜欢看科幻片。”）
2.  学会对女生回复你的答案，给予积极的反馈和延伸。
3.  多去运用开放式提问的方式，以及多去运用本节课学习的开放式回答技术。

这三点你需要着重练习，因为这项技术在聊天中，尤其是在现场互动时，会经常用到。
