# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（值）。我们通过一个名字（变量名）来找到并使用这个盒子里的内容。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

**代码示例：**
```python
my_name = "成成"
age = 25
```

这里，我们创建了两个“盒子”：
*   名为 `my_name` 的盒子里存放了文本 `“成成”`。
*   名为 `age` 的盒子里存放了数字 `25`。

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有一些内置的基本数据类型，最常见的有以下几种：

以下是三种最基础的数据类型及其介绍：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数或负数。
    *   **代码示例：** `count = 10`， `temperature = -5`

2.  **浮点数**
    *   表示带有小数点的数字，用于更精确的数值计算。
    *   **代码示例：** `price = 19.99`， `pi = 3.14159`

3.  **字符串**
    *   表示文本信息。在Python中，字符串需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   **代码示例：** `greeting = “你好，世界！”`， `name = ‘Alice’`

## 如何查看和使用变量？ 👀

我们知道了如何存放数据，那么如何查看和使用这些数据呢？你可以直接使用变量的名字。

**代码示例：**
```python
my_name = “成成”
print(“我的名字是：”, my_name)

x = 5
y = 3
sum = x + y
print(“两数之和为：”, sum)
```

运行这段代码，`print` 函数会将变量 `my_name` 和 `sum` 中存放的实际值输出到屏幕上。

## 一个简单的实践练习 ✍️

让我们将以上知识结合起来，完成一个简单的任务：交换两个变量中存放的值。

以下是实现变量值交换的步骤：

1.  创建两个变量，并分别赋予初始值。
    ```python
    a = 10
    b = 20
    ```

2.  打印交换前的值，以便对比。
    ```python
    print(“交换前：a =”, a, “, b =”, b)
    ```

3.  进行交换。我们需要一个临时“盒子”来帮忙。
    ```python
    temp = a  # 把a的值暂时存到temp里
    a = b     # 把b的值赋给a
    b = temp  # 把temp里存的原来a的值赋给b
    ```

4.  打印交换后的结果。
    ```python
    print(“交换后：a =”, a, “, b =”, b)
    ```

运行完整的代码，你将看到变量 `a` 和 `b` 的值已经成功互换。
