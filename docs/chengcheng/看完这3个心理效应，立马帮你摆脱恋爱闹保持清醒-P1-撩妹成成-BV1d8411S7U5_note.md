## 心理效应一：欺缺心态
**公式**：欺缺心态 = 过度需求 + 选择性忽视

**描述**：
- 当你对某个人过度迷恋时，往往是因为你自身出现了问题，你太需要一段感情了。
- 你可能会对对方产生光环效应，忽略他的缺点，实际上你爱的只是自己的幻想。
- 这种心态类似于马斯洛需求层次理论中的基础需求，只有满足了基础需求，才会追求更高层次的需求。

## 心理效应二：逆向合理化
**公式**：逆向合理化 = 沉默成本 + 无法接受损失

**描述**：
- 当你在错误的人身上投入了时间、精力和感情，产生了所谓的沉默成本。
- 当付出与回报失衡时，感性上会出现无法接受损失，不愿承受失败带来的负面情绪。
- 这种情况下，大脑会触发逆向合理化，让你产生一种很爱对方非他不可的错觉。

## 心理效应三：情绪与逻辑
**公式**：好感 = 情感触发 + 本能感受

**描述**：
- 好感是不讲逻辑的，尤其是在对方对你感兴趣之前。
- 你不能仅凭真诚本身去推销一个别人不需要的东西。
- 在对方喜欢你之前，试图用逻辑说服她接受你，反而会让她疏远你。
- 在恋爱初期，要展示那些情感电路所认可的东西，如丰富多彩的生活、阳光干净的外表、良好的社交地位等。
