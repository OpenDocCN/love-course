## 🧭 对抗路的定位与目标

上一节我们介绍了课程的整体框架，本节中我们来看看对抗路的核心定位。

对抗路的核心目标是建立线上优势。这通常意味着你需要压制敌方英雄的发育，并伺机支援队友或单带推塔。成功的关键在于对兵线、经验和金币的控制。

---

## ⚔️ 核心对线技巧

理解了对抗路的目标后，接下来我们探讨实现这些目标的具体技巧。对线期的细节处理往往决定了整局游戏的走向。

以下是几个关键的对线技巧：

*   **抢线权**：优先清理第一波兵线到达二级，可以占据等级优势发起进攻或规避gank。
*   **换血时机**：在敌方技能处于冷却状态时进行消耗，用最小的代价换取对方的血量优势。
*   **视野控制**：在河道草丛布置视野，防止被敌方打野gank，这是保证生存的基础。
*   **资源计算**：精确计算自身与敌方的伤害量，判断何时可以完成击杀，避免盲目上前。

---

## 🔑 关键概念：伤害计算与斩杀线

上一节我们介绍了对线的通用技巧，本节中我们深入一个核心概念：伤害计算。

能否完成单杀，很大程度上取决于你能否准确计算“斩杀线”。斩杀线指的是敌方英雄当前的血量值，低于这个值，你的一套技能连招加上普攻就足以将其击败。

一个简单的估算公式是：
**自身技能总伤害 + 若干次普攻伤害 > 敌方当前血量**
在实际操作中，你需要将敌方的护甲、魔抗以及可能使用的治疗技能考虑在内。

例如，在代码逻辑上可以抽象为：
```python
if (my_total_damage(considering_enemy_defense) > enemy_current_health):
    execute_all_in_attack()  # 执行全力一击
else:
    continue_farming()       # 继续发育
```

## 🧠 打法思路与决策

掌握了基础操作和伤害计算后，我们需要将这些元素组合成连贯的打法思路。对抗路不仅是操作的对决，更是策略的博弈。

根据游戏的不同阶段，你的主要任务会发生变化：

*   **前期（对线期）**：核心是稳健发育，寻找单杀机会，同时注意地图信号，避免被抓。
*   **中期（小规模团战）**：在拥有一定装备后，可以选择支援中路团战，或继续带线施加压力。
*   **后期（大规模团战）**：根据英雄特性选择切入敌方后排或保护己方核心输出，带线英雄则需把握兵线时机进行分推。
