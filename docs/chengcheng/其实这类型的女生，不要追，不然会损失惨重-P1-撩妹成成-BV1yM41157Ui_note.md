# **P1 撩妹成成 - 如何识别难以追求的女生**

## 概述
在本节课中，我们将学习如何识别那些难以追求的女生，并了解如何避免不必要的损失。

## 核心概念
**公式**：难以追求的女生 = 高段位 + 清晰目标 + 不接受爱情束缚

## 识别难以追求的女生
以下是识别难以追求的女生的几个关键点：

### 1. 高手型女生
* **特点**：懂得男生的心理，知道如何吸引男生。
* **行为**：在朋友圈发布钓鱼信息，如“没人聊天了，我想要甜甜的恋爱”或发布擦边照片。
* **应对**：不要轻易追求，保持距离。

### 2. 清晰目标型女生
* **特点**：非常清楚自己想要什么，不会轻易开始一段关系。
* **行为**：对男生的硬性要求高，不会轻易动心。
* **应对**：评估自己是否满足她的要求，如果不满足，不要浪费精力。

### 3. 不接受爱情束缚型女生
* **特点**：不接受爱情的束缚，追求自由自在的生活方式。
* **行为**：不愿意被爱情束缚，对男生很挑剔。
* **应对**：评估自己的段位是否高于她，如果不高于，趁早收手。

## 总结
本节课中我们一起学习了如何识别难以追求的女生，以及如何避免不必要的损失。记住，选择合适的人比努力更重要。如果你选错了人，要及时停止，否则最后只能感动自己。
