大姨妈通常指的是女性月经周期中的出血现象。

\[ \text{月经周期} = \text{卵泡期} + \text{排卵期} + \text{黄体期} \]

- **生理影响**：月经期间，女性可能会感到腹痛、腰酸、乳房胀痛等症状。

## 大姨夫

\[ \text{大姨夫} = \text{生理反应} + \text{心理反应} \]
