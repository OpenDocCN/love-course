# 恋爱心理学入门：为什么女生不主动？ - 课程 P1 🧠

在本节课中，我们将要学习一个在异性交往中常见的现象：为什么有些女生虽然聊天回应积极，却很少主动发起对话。我们将剖析其背后的心理状态“观望”，并探讨如何通过有效互动打破僵局，建立吸引。

## 现象分析：聊天积极但不主动的女生

你有没有遇到过这种情况：一个女生跟你聊得很好，但是从来没有主动找过你，每次都是你找她。这个女生到底是什么情况？她是在养鱼吗？很多男生就输在这里，觉得女生慢热就该自己多主动一点，结果搞得自己很被动。

现实中，女生如果主动起来，也很疯狂的。很多女生和我反映过类似的情况，说跟别人会回消息，但是几乎不会主动联系。她们行为背后核心的一个词叫做 **“观望”**。

## 核心心态解读：“观望”状态

她会敷衍地回你的消息，说明她并不排斥跟你聊天，最起码不讨厌你。但这肯定也不等于她喜欢你。可能出于礼貌，可能出于无聊的消遣，也可能想留个备胎。

如果女生真的很喜欢你，是会自然地想跟你分享她生活中的喜怒哀乐的。而所谓的“观望”，就是处在一个既不拒绝接触，但也不愿意做过多的投入的阶段。

很多人可能会说，女生这样子太渣了。其实不是这样的。我觉得一个比较正常的人，选择感情的事从来不是非黑即白的。在没有强烈的喜欢的时候，不足以让女生主动。但是她也没必要把关系闹得太僵，所以就处在这种中间的状态。

也不是说她图你什么。而是换个角度来看，这种状态起码说明，对方跟你接触是有机会的，并没有完全否定你。她对你的态度，更多是取决于你接下来的表现，能否建立起好感。

## “备胎”的另一种视角

有时候我们常说的“备胎”，也不一定是一件坏事。除非你有让女生一见钟情的能力，不然女生愿意把你当备胎，就是让你追求她的第一步。倘若她连你都不想理你，那才叫真的绝望。

而这里面的关键点，是很多人输在了心态方面。刚和女生认识的时候就想着：“如果她愿意跟我聊天就好了。”但是这个女的真的跟他聊上的时候，他又想着：“这个女生到底喜不喜欢我？”

首先，你要先问问自己：**你具不具备仅靠聊天就能让女生喜欢你的能力？** 其次，大多数人是不太懂怎么跟女生聊天的。

## 破局关键：制造情绪波动

对于处在“观望”状态下的人，不管是女生还是男生都一样，要去创造更强的吸引。那最重要的就是，能够制造他的情绪波动。

前面我们讲过，“观望”状态下的人内心比较佛系。“你要找我聊也可以，你不找我聊也行。”所以你现在在他们心目中，是一个平平淡淡的存在。要打破这种局面，你必须让他产生情绪波动，这样才能换来他的重视。

有些男生聊天很平淡，每天跟打卡一样：跟他说早安晚安，下班了吗吃饭了吗。这种聊天其实对拉升关系是没有任何意义的。

想要制造出情绪波动，更多的是要突破他对你的固有印象。例如：
*   **印象反转**：比如说他原来觉得你是个直男，你却跟他聊起了综艺节目。
*   **言行不一**：比如说他原来觉得你比较正经，然后你不正经地跟他开了句暧昧的玩笑。
*   **推拉技巧**：他觉得你还蛮喜欢他的，结果你却几天没有找他。

一点一点的了解，加上情绪波动，才能真正的吸引女生。
