## 月经的生理基础 🧬

上一节我们介绍了课程主题，本节中我们来看看月经的生理基础。月经是女性生殖系统周期性变化的自然结果，其主要过程涉及子宫内膜的增厚与脱落。

核心的激素调控周期可以用一个简化的流程表示：

**公式：**
```
下丘脑 → (释放GnRH) → 垂体 → (分泌FSH/LH) → 卵巢 → (分泌雌激素/孕激素) → 子宫内膜变化
```

这个循环大约每28天重复一次，脱落的内膜和血液经阴道排出，即形成月经。

## 月经期间的常见表现 🤒

理解了月经如何产生后，本节中我们来看看它在日常生活中常见的表现。这些表现因人而异，但存在一些普遍规律。

以下是月经期间可能出现的部分症状列表：

*   **腹部痉挛性疼痛**：通常由前列腺素引起子宫收缩所致。
*   **情绪波动**：激素水平变化可能影响情绪，如易怒或低落。
*   **身体疲劳**：能量消耗增加和不适感可能导致疲倦。
*   **腰部酸胀**：盆腔充血可能引发腰骶部不适。
