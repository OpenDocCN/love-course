## 1. 什么是变量？ 🧱

上一节我们介绍了本课程的目标，本节中我们来看看编程中最基本的概念之一：变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存放数据。每个变量都有一个名字（即标签），我们可以通过这个名字来使用或修改盒子里的数据。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

**核心公式/代码：**
```python
variable_name = value
```

例如，我们创建一个名为 `message` 的变量，并在其中存放一段文字：
```python
message = "你好，世界！"
```

---

## 2. 基本数据类型 📊

理解了如何创建变量后，我们需要知道可以在变量中存放哪些类型的数据。Python有几种内置的基本数据类型，它们决定了数据可以进行的操作。

以下是几种最常用的数据类型：

*   **整数**：表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`， `“学习编程”`。
*   **布尔值**：只有两个值，`True`（真） 和 `False`（假），常用于逻辑判断。

---

## 3. 查看与使用变量 🔍

创建了变量并存放了数据之后，我们如何查看它或者使用它进行计算呢？

在Python中，要查看一个变量的值，可以直接使用 `print()` 函数。变量之间也可以进行基本的数学运算。

**核心公式/代码：**
```python
print(variable_name)
```

让我们通过一个例子来实践一下：
```python
# 定义变量
age = 25
price = 19.99
name = "成成"
is_student = True

# 打印变量
print(name)
print(age + 5) # 对变量进行计算
print("价格是：", price)
```

运行这段代码，你将在屏幕上看到变量的值以及计算结果。
