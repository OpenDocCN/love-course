“渣男”通常指在亲密关系中行为不当、不负责任的男性。其核心特征是利用情感达到个人目的，缺乏真诚与尊重。

以下是“渣男”常见的几个基础行为模式：

1.  **忽冷忽热，操控情绪**
    他们通过间歇性的热情与冷淡，让你产生焦虑与依赖，从而掌控关系的主导权。

2.  **言行不一，缺乏承诺**
    他们可能做出许多口头承诺，但很少付诸实际行动。其行为模式可以概括为：**承诺 (Words) ≠ 行动 (Actions)**。

3.  **以自我为中心**
    在相处中，他们总是优先考虑自己的需求与感受，忽视或贬低你的想法。

4.  **界限模糊，缺乏尊重**
    他们可能不尊重你的个人边界，例如过早提出亲密要求，或在未明确关系时与其他异性暧昧不清。
