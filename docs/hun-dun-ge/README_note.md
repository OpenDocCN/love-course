# 课程名称：馄饨哥 🥟

在本节课中，我们将学习如何制作一道经典的中式面点——馄饨。我们将从准备馅料开始，逐步讲解制作面皮、包制馄饨以及调制汤底的全过程。

## 1. 准备馅料 🥩

上一节我们介绍了课程目标，本节中我们来看看如何制作馄饨的馅料。馅料是馄饨味道的核心。

以下是制作猪肉馅所需的主要材料和步骤：

*   **主要材料**：猪绞肉、生姜、葱、鸡蛋。
*   **调味料**：食盐、生抽、香油、白胡椒粉。
*   **步骤一**：将生姜和葱切末。
*   **步骤二**：将猪绞肉放入碗中，加入姜末、葱末。
*   **步骤三**：打入一个鸡蛋，依次加入食盐、生抽、香油和白胡椒粉。
*   **步骤四**：沿同一方向用力搅拌，直到肉馅上劲、出现粘性。

**核心搅拌公式**：`肉馅 + 调料 + 同一方向搅拌 = 上劲的馅料`

## 2. 制作与包制馄饨 👐

准备好馅料后，接下来我们处理馄饨皮并进行包制。市面上可以买到现成的馄饨皮，这大大简化了制作流程。

以下是包制馄饨的两种常见手法：

*   **元宝形**：馅料放皮中央，对折成矩形，再将两角向中间沾水捏合。
*   **抄手形**：馅料放皮一角，卷起至中部，然后将左右两角向中间叠压捏紧。

**包制代码逻辑**：
```plaintext
如果 选择“元宝形”：
    执行 对折 -> 沾角 -> 捏合
否则如果 选择“抄手形”：
    执行 卷起 -> 叠角 -> 捏紧
```

## 3. 调制汤底与煮制 🍲

馄饨包好后，最后一步就是煮制和调制汤底。清亮的汤底能衬托出馄饨本身的鲜味。

汤底的调制非常简单。

*   **基础汤底材料**：碗中放入紫菜、虾皮、葱花、榨菜末。
*   **调味**：加入适量食盐、生抽和香油。
*   **煮制**：锅中烧开水，下入馄饨，用勺背轻推防止粘底。
*   **冲汤**：待馄饨全部浮起后，先舀几勺煮馄饨的热汤冲入碗中，化开底料。
*   **盛出**：将煮熟的馄饨捞入汤碗中即可。

> 煮制时间不宜过长，馄饨浮起后再煮1-2分钟即可，以保持皮的口感。

本节课中我们一起学习了制作馄饨的完整流程。我们掌握了馅料的调配**公式**，了解了两种主要的包制**方法**，并学会了如何调制一碗清淡鲜美的汤底。现在，你可以尝试为自己和家人制作一碗热气腾腾的馄饨了。
