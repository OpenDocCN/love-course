# 香草情感 教程 01：情感分析基础 🧠

在本教程中，我们将学习情感分析的基础概念。情感分析是自然语言处理领域的一项技术，旨在识别和提取文本中的主观情感信息，例如判断一段文字表达的是积极、消极还是中性的态度。

---

## 什么是情感分析？🤔

情感分析，有时也称为观点挖掘，是计算语言学的一个分支。它利用计算机自动分析文本数据中表达的情感、观点和态度。

上一节我们介绍了本教程的主题，本节中我们来看看情感分析的核心目标。其主要目标是判断给定文本的情感极性。情感极性通常被归类为**积极**、**消极**或**中性**。在某些更复杂的模型中，还可能包括更细致的情感类别，如喜悦、愤怒、悲伤等。

---

## 核心概念与表示 🎯

为了量化情感，我们需要将文本转化为计算机可以处理的形式。一个基础而有效的方法是使用**词袋模型**。

词袋模型将文本视为一个词语的集合，忽略其语法和词序，只关注每个词出现的频率。我们可以用一个向量来表示一段文本，向量的每个维度对应词汇表中的一个词，其值表示该词在文本中出现的次数。

以下是词袋模型的一个简单表示：

```python
# 示例：将句子转化为词频向量
from sklearn.feature_extraction.text import CountVectorizer

corpus = [
    ‘这个产品太棒了，我非常喜欢！‘, # 积极
    ‘糟糕的体验，质量很差。‘ # 消极
]

vectorizer = CountVectorizer()
X = vectorizer.fit_transform(corpus)

print(vectorizer.get_feature_names_out())
print(X.toarray())
```

在这个例子中，`X.toarray()`输出的矩阵就是文本的数值化表示，每一行对应一个句子，每一列对应一个词的出现次数。

## 基本方法与步骤 📝

理解了文本的表示方法后，我们来看看构建一个简单情感分析模型的基本步骤。以下是实现情感分析的典型流程：

1.  **数据收集**：获取带有情感标签的文本数据，例如带有“积极”或“消极”标签的影评。
2.  **文本预处理**：清洗文本数据，包括去除标点符号、统一转为小写、分词等操作。
3.  **特征提取**：将清洗后的文本转化为数值特征，例如使用上文提到的词袋模型。
4.  **模型训练**：使用带标签的数据训练一个分类模型，例如朴素贝叶斯或逻辑回归。
5.  **评估与预测**：用测试集评估模型性能，之后即可用其预测新文本的情感。

---

## 简单规则方法示例 ⚙️

在深入机器学习模型之前，有一种基于词典的简单方法值得了解。该方法的核心思想是：预先定义一个情感词典，其中包含带有情感倾向的词语及其分值。

一段文本的情感得分可以通过对其中的情感词分数进行加总来计算。例如：

`情感得分 = Σ(词语情感值)`

假设我们有一个简单词典：`{“好”: 1, “棒”: 2, “差”: -1, “糟糕”: -2}`。那么对于句子“这个产品好，不糟糕”，其得分为：`1 + (-2) = -1`，可能被判定为轻微消极。这种方法直观，但难以处理复杂的语言现象，如否定和转折。
