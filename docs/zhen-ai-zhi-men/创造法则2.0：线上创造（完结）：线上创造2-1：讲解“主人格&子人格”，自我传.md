# 创造法则2.0：线上创造（完结）第2-1课：讲解“主人格&子人格”与自我传讯冥想练习 🧘‍♂️💬

在本节课中，我们将要学习关于“主人格”与“子人格”的核心概念，并掌握一种名为“自我传讯”的冥想练习方法。通过理解我们内在的不同部分，并学会与它们沟通，我们可以更深入地认识自己，并进行有效的自我疗愈。

## 什么是意识、主人格与子人格？🧠

首先，我们来探讨意识是什么。意识是此刻接管着你、掌控你的想法、言行乃至身体变化的东西。我们通常将其理解为灵魂或人格。

**意识 = 此刻掌控你思维与行为的“接管者”**

那么，什么是主人格呢？主人格就是我们绝大部分时间里所表现出来的“常规状态”。我们认为这是“正常的我”。但主人格其实只是我们允许表现出来的一部分自己，它通常等同于“小我”。

**主人格 ≈ 小我（我们长期允许表现出来的部分自我）**

为什么它看起来是“正常”状态？因为我们只允许这部分自己活出来，而压抑了其他许多方面。这造成了一种错觉，仿佛只有这部分性格和思维才是“我”。实际上，并没有绝对的主人格和子人格之分。

## 觉察、高我与内在小孩 👁️✨

上一节我们介绍了主人格，本节中我们来看看，当我们处于主人格（小我）状态时，如果脑海中冒出一个想法，是什么在觉察这个想法呢？那个能够“看到”想法的观察者又是谁？

那个观察者就是“高我”，也被称为“真我”、“太傻”或“觉察”。它是超越时间、能够观察我们所有想法和状态的部分。

**高我 = 真我 = 觉察（观察想法的超越性意识）**

接下来，我们谈谈“内在小孩”。假设绿色圆圈代表我们的主人格（常规模式），那么内在小孩就是另一部分的你自己，是另一部分的意识接管了你。

**内在小孩 = 另一部分被压抑或未被允许表达的自我意识**

当我们被嫉妒、受伤、愤怒等强烈情绪淹没时，其实就是对应的“子人格”或“受伤的内在小孩”接管了我们。此刻，主人格并不在场。

## 如何连接与疗愈内在小孩？❤️🩹

当我们情绪按钮被触动，感觉焦虑、抗拒或受伤时，那个内在小孩已经接管了我们。我们不需要费力去寻找他，因为他已经在这里了。

以下是面对内在小孩时可以采取的步骤：

1.  **觉察**：首先，意识到“此刻是另一部分的自己（内在小孩）接管了我”。
2.  **感受**：无条件地去感受这份情绪，不抗拒、不评判。
3.  **连接**：通过自我传讯，与这部分自己对话。

## 自我传讯冥想练习详解 📞

自我传讯是一种与内在自我（高我或内在小孩）直接沟通的冥想方式。它非常简单，你可以在任何有情绪升起的时候进行。

以下是进行自我传讯的基本步骤：

1.  **放松与进入**：找一个安静的环境，放松身体，闭上眼睛。
2.  **回想场景**：回想触发你情绪的那个具体事件或感觉。
3.  **感受与提问**：感受当时那个自己的状态，然后向他提问。问题可以从简单开始，例如：“我现在有什么感觉？”、“你为什么会有这种感觉？”。
4.  **聆听答案**：保持开放，让答案自然地从你内心浮现，而不是用头脑思考。答案可能会以话语、图像或感觉的形式出现。
5.  **对话与安慰**：根据得到的答案，继续提问或给予这部分自己安慰、理解和爱。
6.  **整合与回归**：对话结束后，可以给自己一个拥抱的想象，然后慢慢将意识带回身体，感受呼吸，睁开眼睛。

这个过程的本质就是**通灵**——与自己的灵（内在真实自我）进行沟通。每个人都有这个能力。

## 学员实践示范与要点回顾 🎯

在课程中，我们通过两位学员（Terry和任平生）的实时练习，示范了自我传讯的过程。他们通过与内在小孩对话，发现了情绪背后的恐惧（如害怕被拒绝、觉得自己不配），并给予了自我接纳和鼓励。

从示范中，我们可以总结出几个要点：

*   **答案非设计**：来自高我或内在小孩的答案，往往超出头脑的预设，充满智慧且直指核心。
*   **无需复杂准备**：不需要特定的环境或仪式，有情绪时即可进行。
*   **疗愈的关键在于“看见”**：一旦你真切地“看到”并理解了那部分受伤的自己，抗拒就会自然消融。
*   **应用广泛**：此方法不仅可用于处理即时情绪，也可用于探索人生课题、寻找使命等更深层的问题。

## 总结与课程预告 📚

本节课中我们一起学习了：
1.  **主人格**只是我们习惯表现的“小我”，并非全部。
2.  **子人格/内在小孩**是未被接纳的自我部分，常在情绪爆发时接管我们。
3.  **高我/觉察**是观察一切的超越性意识。
4.  通过**自我传讯**冥想，我们可以直接与内在小孩或高我对话，实现深刻的自我觉察与疗愈。
