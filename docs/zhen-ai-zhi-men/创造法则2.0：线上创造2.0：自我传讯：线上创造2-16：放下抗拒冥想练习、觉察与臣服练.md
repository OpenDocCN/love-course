# 创造法则2.0：线上创造2.0：自我传讯：线上创造2-16：放下抗拒冥想练习、觉察与臣服练

## 概述
在本节课中，我们将要学习如何通过觉察与臣服练习，来面对和放下内在的抗拒。我们将通过具体的冥想引导和案例分析，帮助你理解什么是抗拒，以及如何通过接纳和臣服来疗愈自己。课程内容简单直白，适合初学者。

---

## 一、理解抗拒的本质
上一节我们介绍了创造法则的基本概念，本节中我们来看看什么是抗拒。抗拒是当你内在感到不安全或匮乏时，想要赶走或改变这部分自己的那股力量。它通常表现为焦虑、烦躁或想要从外界寻求答案。

**核心概念**：抗拒 = 想要赶走“匮乏的自己”的那个“自己”。

例如，当你感到不安并向外寻求帮助时，其实有一个更深层的自己在抗拒这种“不安”和“向外求”的状态。这个抗拒的自己，才是你需要首先看到和接纳的对象。

---

## 二、觉察练习：看到抗拒的两层结构
要解决内在困扰，第一步是进行清晰的觉察。以下是觉察的步骤：

1.  **觉察第一层**：首先，看到那个感到不安全、匮乏、想要向外寻求答案的自己。
2.  **觉察第二层（关键）**：然后，去觉察那个“认为第一层的自己不好、想赶走他/杀掉他”的自己。**这第二层就是抗拒**。

很多人只看到了第一层的不安，却忽略了更重要的第二层——对这份不安的抗拒和评判。正是这两层自己在“打架”，让你感到困惑和无力。

**简单来说**：问题不在于“感到匮乏的自己”，而在于“讨厌这个匮乏自己的自己”。

---

## 三、臣服练习：如何放下抗拒
当我们觉察到抗拒后，下一步不是去分析或改变它，而是**臣服**。臣服意味着放下改变它的手，只是允许它存在，感受它。

**臣服的核心操作**：
1.  **允许感受**：允许那个烦躁、焦虑或任何不舒服的感觉存在。
2.  **不改变**：不试图让它消失，也不评判它。
3.  **感受身体**：将注意力放到身体感受上（如头部的紧绷、腹部的堵塞感），只是去感受这些部位的感觉。

**双臣服法**：如果你发现自己“无法臣服”，那就去臣服这个“无法臣服”的状态。接纳自己当下的任何状态，本身就是臣服。

## 四、针对身体紧绷的臣服冥想（引导摘要）
以下是课程中带领的一次放下抗拒的冥想练习要点，你可以跟随练习：

1.  **找到紧绷点**：舒适坐好，放松身体，将意识从头到脚扫描一遍，找到无意识紧绷的部位（常见于头部、眉头、牙关、肩膀）。
2.  **感受而非对抗**：将注意力集中在一个紧绷点（例如头部的紧绷感）。不要试图放松它，而是**允许它存在**，甚至尝试去“享受”这种紧绷的感觉。
3.  **对比体验**：刻意收紧该部位的肌肉（抗拒），然后再放松。你会发现，**抗拒（刻意收紧）会增加一重难受**，而允许则会带来一丝舒展。
4.  **应用于呼吸**：将注意力放到鼻腔，感受呼吸。不要用力呼吸去对抗可能的堵塞感，只是**允许呼吸自然、缓慢地进行**，感受气息进出时鼻腔肌肉的细微变化。你会发现，允许之下，呼吸会自然变得更通畅。
5.  **理解身心关联**：思维散乱（想烦心事）会导致身体再次紧绷。这验证了**身体的紧绷是内在状态（因）的反映（果）**。长期疗愈需要转变内在的“因”，但当下可以通过臣服身体的“果”来获得缓解。

**关键收获**：通过臣服身体的紧绷感，你实际上是在练习放下心理层面的抗拒和紧抓。身体是通往内在的桥梁。

## 五、学员常见问题与解答
在疗愈过程中，大家常会遇到一些困惑，以下是部分解答：

*   **问：学灵修后反而更焦虑、没动力了？**
    *   **答**：这可能是好现象。过去你习惯压抑情绪，现在开始允许它们浮现。这就像从“用技巧伪装”过渡到“面对真实自己”的必经阶段。先接纳这个看似“更糟”的真实状态，才能开始真正的疗愈。

*   **问：冥想时感觉“空”，连接不上情绪怎么办？**
    *   **答**：这种“空”或“平静”本身可能就是一种情绪，是抗拒感受深层创伤的表现。你可以尝试与这个“空”对话，问它为何在此，往往会有更深层的情绪或画面浮现。

*   **问：总担心被打扰，无法深入冥想怎么办？**
    *   **答**：这个“担心被打扰的自己”就是当下浮现出来的内在部分——一个“不安全的小孩”。你已经“挖到”他了！不需要再去别处寻找。接下来，正是要去接纳和疗愈这个“不安全的自己”。

*   **问：臣服和冥想是什么关系？**
    *   **答**：**冥想即是臣服，臣服即是冥想**。许多人把冥想想象成一套复杂的仪式，但其本质就是**在当下感受和臣服**。无论是坐姿冥想，还是在日常生活中感受呼吸，核心都是臣服于当下的体验。

## 六、“植入滤镜”觉察游戏
课程最后，我们通过一个游戏来觉察外在信念如何影响我们。许多我们深信不疑的“真理”，可能只是童年时被家人或环境植入的“滤镜”。

**游戏主题**：分享一个童年时被植入的信念，它让你对某件事/某类人产生了未曾亲身验证过的评判。

**学员分享示例**：
*   “男人要喜怒不形于色” -> 导致情感表达麻木。
*   “漂亮的女人都是坏女人” -> 影响两性关系。
*   “公务员都是靠关系” -> 阻碍职业选择。
*   “挣钱非常难，社会很黑暗” -> 带来生存恐惧和不安全感。

**游戏目的**：意识到我们的很多观念并非来自真实体验，而是被植入的“滤镜”。觉察到这些滤镜，是将其放下、重获自由选择权的第一步。

## 总结
本节课中我们一起学习了：
1.  **抗拒的本质**：是抗拒“匮乏的自己”的那股力量，它本身是需要被首先接纳的对象。
2.  **觉察的步骤**：看到两层结构——匮乏的自己，和抗拒这个匮乏自己的自己。
3.  **臣服的方法**：通过允许身体感受、不改变、不评判来放下抗拒。
4.  **身心关联**：身体的紧绷是内在抗拒的果，可以通过臣服身体来练习放下心理抗拒。
5.  **识别植入滤镜**：我们很多限制性信念来自童年植入，觉察它们是解脱的开始。

疗愈是一个渐进的过程，如同成长。不必追求一步到位，只需在每个当下，练习多一份觉察，多一份对自己的允许和臣服。
