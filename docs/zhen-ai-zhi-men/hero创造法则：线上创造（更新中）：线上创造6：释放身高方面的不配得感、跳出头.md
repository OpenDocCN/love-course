## 释放身高方面的不配得感

上一节我们介绍了课程的整体框架，本节中我们来看看如何释放身高方面的不配得感。许多人在成长过程中，因外界评判而内化了关于身高的负面信念，这些信念会持续影响自我价值感和人际关系。

### 核心问题分析

身高问题往往不是外在的，而是内在的限制。例如，学员提到：

> 我的体验是太深了，就我自己不在意这个事情之后，我发现其实根本没有人在意。在北方，像在山东，女生身高173或175都很正常，但我的体验是很少有人会直接评论身高。这完全是自己内在的限制。

这种内在限制通常源于早期的经历。例如：

> 从初中开始，就有女生在背后评论和嘲笑我的身高。高中时，体育老师因身高劝我放弃体育道路。这些经历让我感到痛苦和无助，并内化为自我评判。

### 释放不配得感的步骤

以下是释放不配得感的具体步骤：

1. **觉察情绪**：首先，需要觉察与身高相关的情绪。每当感到不配得或恐惧时，注意身体的感觉（如胸闷、紧张）。
2. **回溯早期经历**：找到最早与身高相关的负面经历，回忆当时的情绪感受。
3. **释放情绪**：通过冥想（如十字架冥想或藤环法）反复释放这些情绪。例如，在冥想中想象自己站在十字架上，面对恐惧并释放它。
4. **接纳与爱自己**：在日常生活中，随时接纳和爱自己，无论身高如何。告诉自己：
   > 我不管别人怎么样，我就是完全地爱我自己。哪怕我只有1米5，我也爱我自己。我的价值是独立的，不建立在身体或比较之上。

### 练习示例

假设你感到身高不配得，可以这样练习：

- **步骤1**：闭上眼睛，感受胸腹之间的紧张感。
- **步骤2**：回想早期被嘲笑的场景，感受当时的情绪。
- **步骤3**：想象这些情绪像烟雾一样从身体中释放出去。
- **步骤4**：对自己说：“我完全接纳并爱我自己，我的价值与身高无关。”

## 跳出头脑练习

上一节我们探讨了如何释放身高不配得感，本节中我们来看看如何跳出头脑，活在当下。许多人在生活中过度依赖逻辑思维，导致与内在感受脱节，从而产生焦虑和卡顿感。

### 什么是跳出头脑

跳出头脑是指将注意力从逻辑思维转移到身体感受上，从而活在当下。例如，学员提到：

> 我经常困在头脑里，感觉一卡一卡的，不灵活。但当我感受身体时，说话变得流畅，内心也平静了。

### 跳出头脑的步骤

以下是跳出头脑的具体步骤：

1. **感受身体**：将注意力集中在身体的感觉上，如胸部、腹部或小腹的紧绷感。
2. **呼吸穿过感觉**：深呼吸，让呼吸穿过这些感觉，帮助它们放松。
3. **想象与安抚**：将身体的感觉想象成一个小猫或小婴儿，用内在的能量拥抱和安抚它。
4. **保持感受**：持续将注意力放在感觉上，避免回到逻辑思维中。

### 练习示例

假设你感到焦虑，可以这样练习：

- **步骤1**：闭上眼睛，感受腹部的紧绷感。
- **步骤2**：深呼吸，想象呼吸像温暖的光穿过紧绷感。
- **步骤3**：将紧绷感想象成一只紧张的小猫，用内在的爱拥抱它。
- **步骤4**：对自己说：“我感受到你，我接纳你，我爱你。”

### 代码描述核心概念

跳出头脑的核心是注意力转移，可以用以下伪代码表示：

```python
while 困在头脑中:
    注意力 = 获取当前注意力()
    if 注意力 == "逻辑思维":
        转移注意力到身体感受()
        感受身体感觉()
        深呼吸()
    else:
        保持感受()
```
