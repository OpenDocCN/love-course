# 真爱之门 🚪💖

在本教程中，我们将学习如何构建一个名为“真爱之门”的简单程序。这个程序的核心是判断两个人是否匹配。我们将通过一系列步骤，理解其背后的逻辑与实现方法。

## 概述

“真爱之门”是一个基于简单规则判断两人匹配度的趣味程序。它接收两个人的名字作为输入，通过一个特定的计算过程，最终输出一个“匹配度”百分比。我们将逐步拆解这个程序，学习其中的字符串处理、循环和条件判断等基础编程概念。

## 1. 输入与字符串处理

首先，程序需要获取两个人的名字。在Python中，我们可以使用 `input()` 函数来实现。

以下是获取用户输入的代码：
```python
name1 = input("请输入第一个人的名字：")
name2 = input("请输入第二个人的名字：")
```

获取名字后，我们需要将它们合并成一个字符串，以便进行后续处理。这可以通过字符串拼接来完成。

## 2. 核心计算：统计“真爱”字母

程序的核心逻辑是统计合并后的名字中，字母 **T**、**R**、**U**、**E**、**L**、**O**、**V**、**E** 出现的总次数。这些字母恰好能拼成“TRUE LOVE”这个词组。

我们通过一个循环来遍历合并后的字符串，并使用条件判断来检查每个字符。

以下是统计字母的代码示例：
```python
combined_names = name1 + name2
true_love_letters = "TRUE LOVE"
count = 0
for letter in combined_names.upper(): # 转换为大写以统一比较
    if letter in true_love_letters:
        count += 1
```

## 3. 计算匹配度百分比

统计出总次数后，我们需要将其转换为一个百分比分数。一个常见的简化算法是：将统计次数作为十位数和个位数，分别相加，不断重复直到得到一个1到100之间的数字。

例如，如果统计次数是 **42**，那么计算过程是：4 + 2 = **6**。所以匹配度是 **6%**。如果次数是 **123**，计算过程是：1+2+3=**6**，结果也是 **6%**。

我们可以用一个循环来实现这个过程：
```python
def reduce_to_percentage(num):
    while num >= 10:
        num = sum(int(digit) for digit in str(num))
    return num * 10 # 为了得到一个看起来像百分比的数字（如60%），这里乘以10。原逻辑常为 num * 10。
```

上一节我们介绍了核心字母的统计方法，本节中我们来看看如何将这个数字转化为最终的匹配度结果。

## 4. 输出结果

最后，程序将计算出的百分比输出给用户。为了使结果更清晰，可以附带一些简单的评语。

以下是输出结果的代码：
```python
score = reduce_to_percentage(count)
print(f"{name1} 和 {name2} 的匹配度是 {score}%！")
if score > 80:
    print("你们是天造地设的一对！")
else:
    print("或许可以再多了解彼此一些。")
```

## 总结

本节课中我们一起学习了“真爱之门”程序的构建。我们从获取用户输入开始，经历了字符串合并、特定字母计数、数字简化算法，最终输出了一个匹配度百分比。这个程序虽然简单，但涵盖了输入输出、循环、条件判断等编程基础要素。希望你能通过这个有趣的例子，感受到编程的逻辑与乐趣。
