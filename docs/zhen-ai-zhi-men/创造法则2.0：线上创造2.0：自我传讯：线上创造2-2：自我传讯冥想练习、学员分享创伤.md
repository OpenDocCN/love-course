# 创造法则2.0 课程笔记（210424）：线上创造2-2：自我传讯冥想与创伤模式疗愈 🧘‍♂️💡

在本节课中，我们将学习一种名为“自我传讯”的冥想方法，并通过学员的真实案例，深入探讨如何识别和疗愈内在的创伤模式。课程的核心在于理解：**我们的外在体验（如果）是由内在的感受和信念（因）所创造的**。通过连接并疗愈内在受伤的部分，我们可以改变当下的感受和现实。

---

## 概述：什么是自我传讯冥想？

自我传讯冥想是一种深入潜意识、与内在自我（或称内在小孩）进行对话的疗愈方法。其原理基于：我们当下的情绪困扰和重复性模式，往往根植于过去未被疗愈的创伤记忆。这些记忆以“能量碎片”或“部分自我”的形式存在于我们的内在。

**核心公式**：
`外在困境（果） = 内在创伤感受（因）`
`改变内在感受 = 改变外在体验`

上一节我们介绍了线上创造的基本概念，本节我们将通过具体的冥想练习，来看看如何实际操作。

---

## 第一部分：冥想练习示范——与“抗拒”的感受连接

课程开始，导师带领学员阿伟进行了一次针对“工作理解困难”背后抗拒情绪的冥想。

**以下是冥想的关键步骤：**

1.  **定位感受**：首先，闭上眼睛，深呼吸，将意识回归身体。问自己：“我现在是什么感觉？”将注意力集中在身体有反应的部位（如胸部、腹部）。
2.  **询问根源**：温柔地问内在感受：“你为什么感到……（例如：生气/难过/抗拒）？” **注意：要求自己说出脑海中浮现的“第一个答案”，不加以修饰或评判。**
3.  **连接场景**：让这个感受带你回到记忆中最早出现类似感觉的场景。看看那个场景中的自己多大，正在经历什么。
4.  **对话与陪伴**：以现在成年自己的形象，走进那个记忆场景，找到当时那个受伤的自己。对他/她说：“你好，我自己。”然后询问他/她的感受和需要，并给予陪伴、保护或带去一个安全的地方玩耍。
5.  **整合与回归**：当内在小孩感到安全并被爱后，与他/她拥抱，想象双方融合为一体，成为你光芒的一部分。最后，弹响指，通过一个“时空隧道”回到当下的意识中。

**过渡**：通过上述步骤，阿伟疗愈了因童年惧怕母亲权威而产生的“低自我价值”和“恐惧犯错”的模式。接下来，我们看看其他学员分享的常见创伤模式。

## 第二部分：学员分享的常见创伤模式

以下是几位学员分享的典型模式，这些模式都源自原生家庭，并在成年后不断重复。

### 1. “找问题”与自我否定模式
*   **表现**：遇到任何挫折，第一反应是寻找自己的问题（“是不是我太笨？”“是不是我不够努力？”）。在亲密关系中尤其明显，容易陷入“我是否不够好”的焦虑。
*   **根源**：通常来自父母（尤其是母亲）的教养方式——习惯性指出不足而非给予鼓励。孩子内化了这种“审视者”的声音，形成苛责的内在父母。
*   **核心信念**：**“我必须完美，才有价值；出现问题，一定是我的错。”**

### 2. “被孤立”与社交疏离模式
*   **表现**：在集体中总感觉被排斥、被孤立，人际关系容易莫名恶化。行为上可能无意识地表现出冷淡或防御姿态，将他人推开。
*   **根源**：可能源于童年情感需求未被满足，或家庭氛围冷漠。学员分享中提到，父亲在冲突中“一走了之”的模式，也被自己继承。
*   **核心信念**：**“我是不被喜欢的/我注定是一个人。”** 这其实是一种内在的“自我孤立”。

### 3. 低自尊与身体症状模式
*   **表现**：
    *   **身体层面**：如无法控制的肥胖、长期疲惫感、或在特定情绪下出现荨麻疹等皮肤症状。
    *   **行为层面**：在行动前充满抗拒，难以开始（如学习、健身），但一旦投入又能享受过程。
*   **根源**：深层的不配得感、低自我价值感。身体是内在状态的镜子，症状是内在情绪能量的外在显化。
*   **核心公式**：`低自尊感/不配得感（因） → 肥胖/乏力等症状（果）`
    `抗拒的感受（因） → 拖延/难以启动（果）`

### 4. 情感空虚与成瘾行为模式
*   **表现**：经常感到莫名的空虚、孤独，无法独处。必须通过不断找异性聊天、刷社交媒体等外部刺激来填补空虚感，形成恶性循环。
*   **根源**：童年时期的情感忽视，导致内在形成一个“缺爱的小孩”。成年后，会在亲密关系（或追求关系）中，不自觉地寻求一种“被母亲认可”的感觉。
*   **核心信念**：**“我必须从外界获得关注和连接，否则我就会消失。”**

**过渡**：识别模式是疗愈的第一步。那么，理解了这些模式后，我们该如何在日常生活中应用自我传讯的方法呢？

## 第三部分：关键心法与答疑

在课程的最后，导师针对学员的疑问，澄清了几个核心心法：

1.  **疗愈不是“解决问题”**：受伤的内在小孩不是需要被解决、赶走的“问题”。把他/她当作需要被关爱、被倾听的“部分自己”。**疗愈的本质是爱，而非对抗。**
2.  **“怎么还没好”正是抗拒的声音**：当产生“我都疗愈了，怎么感觉还在”的想法时，这正是那个“不耐烦、不接纳现状的自己”在说话。你需要去感受的，恰恰是这份“不耐烦”。
3.  **过去记忆是“幻觉”**：那些让我们痛苦的记忆场景，其本质是承载着情绪的“能量幻觉”。正因为它是幻觉，我们才能通过冥想改变与它的关系，从而改变它对我们当下的影响。**修改了过去的“因”，当下的“果”自然改变。**
4.  **天赋与模式一体两面**：某些创伤模式也隐藏着天赋。例如，习惯“找问题”的人，可能拥有强大的觉察和分析能力；擅长语言的人，其天赋通道在于顶轮，能在做热爱之事时获得能量。

## 总结与行动建议

本节课中，我们一起学习了：
1.  **自我传讯冥想**的完整步骤：感受定位 → 询问根源 → 连接场景 → 对话陪伴 → 整合回归。
2.  几种常见的 **创伤模式**（自我否定、被孤立、低自尊、情感空虚）及其家庭根源。
3.  疗愈的核心心法：**以爱接纳，而非以力对抗**；识别并穿越“二次抗拒”。

**给你的实践建议**：
*   当任何负面情绪或重复困境出现时，将其视为一次自我传讯的邀请。
*   立即找一个安静的空间，闭上眼睛，按照课程步骤，温柔地问自己：“此刻，我内在哪一部分需要被关注？”
*   坚持练习，你会发现自己对情绪的反应速度、以及生活的轻松感，将发生根本转变。

记住，**你是一切体验的创造者。通过疗愈内在，你便重写了人生的剧本。** 🌈
