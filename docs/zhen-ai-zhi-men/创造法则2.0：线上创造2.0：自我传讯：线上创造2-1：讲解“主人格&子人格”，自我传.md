# 创造法则2.0：线上创造2.0：自我传讯：线上创造2-1：讲解“主人格&子人格”，自我传

## 课程概述

在本节课中，我们将要学习“主人格”与“子人格”的概念，理解什么是内在小孩与觉察，并掌握一种名为“自我传讯”的冥想练习方法。我们将通过理论讲解和现场示范，帮助你学会如何与自己的内在部分进行沟通，从而实现自我疗愈。

---

## 第一部分：理解意识、主人格与子人格

首先，我们来探讨一些核心概念。意识是什么？意识是此刻接管着你、掌控你的想法、言行乃至身体变化的东西。我们通常将其理解为人格或灵魂。

**主人格**，常被我们认为是“正常的”、“常规的”状态。它之所以成为常规状态，是因为我们绝大部分时间只允许这部分自己表现出来。我们不允许自己活出其他许多方面，从而产生一种错觉：只有这部分性格或思维是“正常”的。实际上，**主人格就是我们允许表现出来的这部分思维、状态和信念**。它本质上就是“小我”。

那么，我们如何知道自己是谁呢？例如，当我处于小我状态时，脑海中冒出了一个想法。当我们觉察到这个想法时，是什么在觉察它呢？那个冒出想法的“我”自己，是看不到它自己的。能看到它的，是另一个“我”。这就是**高我**，或称**真我**、**太傻**。它是时间之外的我，与高我等同。我们也可以用一个更纯粹的名字来称呼它：**觉察**。是有一个“觉察”在看着我的想法。

---

## 第二部分：认识内在小孩与情绪接管

接下来，我们看看什么是内在小孩。假设绿色圆圈代表我们的主人格，即常规模式。那么，内在小孩就是另一部分的你自己，是另一部分的意识接管了你。

当我们被情绪按钮触动，感到嫉妒、受伤或愤怒时，发生了什么？实际上，是一个嫉妒的、受伤的或愤怒的“你自己”——这部分**子人格**——接管了你。此刻，你不是平常状态的你，而是被子人格接管了。你感觉到的“我很嫉妒”，其实就是一个“嫉妒的你自己”接管了你。

这一点对于疗愈至关重要。只要你在情绪中，或者说你的情绪创伤被触动，这一部分就是我们所讲的**受伤的内在小孩**。

我们如何寻找受伤的内在小孩？你根本不需要寻找。当你在日常生活中情绪被唤起，感到焦虑、抗拒时，这个内在小孩或子人格已经接管你了。他已经在这里了。

---

## 第三部分：自我传讯的核心方法

那么，你能做什么？你能做的就是去**觉察他**。无条件地去感受他，无条件地去臣服于他。这等同于觉察他。

以下是进行自我传讯冥想练习的核心步骤：

1.  **识别接管状态**：当你感觉到情绪波动或状态不对时，意识到是某一部分子人格接管了你。
2.  **放松与进入**：闭上眼睛，放松身体，回想触发情绪的场景，感受或“看到”当时的自己。
3.  **提问与倾听**：向那个部分的自己提问，例如：“我现在有什么感觉？”“为什么会有这种感觉？”然后倾听内心自然浮现的答案，不要用头脑思考。
4.  **沟通与安抚**：根据得到的答案，像安慰一个好朋友一样，在内心与他对话，给予理解、安慰和无条件的爱。可以问他：“你希望我以后怎么对待你？”
5.  **整合与回归**：陪伴他，直到感觉平静。然后慢慢将注意力带回呼吸和身体，睁开眼睛。

这个过程的本质，就是与你内在的真实自我（高我/内在小孩）进行沟通。你得到的答案往往充满智慧，且无法通过头脑预先设计。

## 第四部分：学员体验示范与解析

为了让理论更清晰，我们来看一位学员（Carrie）的现场体验。他分享了自己在尝试搭讪时感到的紧张和抗拒。

通过引导他进行自我传讯，我们发现了藏在这些行为背后的恐惧子人格。他觉察到，一个“害怕被拒绝”、“害怕被人当不正经”的自己接管了当时的他。通过与该部分自己对话、给予关爱，他的紧张感得到了缓解，并感受到了胸口的温暖。

这个示范表明：
*   **自我传讯是一种有效的内在沟通**：它让你直接看到情绪背后的真实需求。
*   **答案源于内在智慧**：得到的解答（如“不管在哪里，我们都在一起”）超越了逻辑思维，带来深层的安慰。
*   **适用于日常课题**：无论是社交恐惧、工作焦虑还是自我价值问题，都可以通过此方法探寻根源并获得指引。

另一位学员（任平生）则示范了如何连接“觉得自己不配”的子人格，并追溯到童年时期“感到孤独和被孤立”的感觉源头，通过安抚那个内在小孩来疗愈当下的卡点。

本节课中，我们一起学习了：

2.  **核心机制**：认识到当情绪爆发时，是特定的**子人格接管**了我们，而非我们的“常态”。

4.  **方法优势**：这种方法直接、简单，无需特殊环境，在日常生活中随时可以应用，能帮助我们看到真实需求，从而无法再抗拒或忽视它。
