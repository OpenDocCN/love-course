# 英雄创造法则 第2.1课：找到“因” 🔍

在本节课中，我们将要学习创造的核心原则之一：思维模式。我们将探讨什么是思维模式，它如何深刻地影响我们看待世界的方式，以及为什么所有外在的改变努力，如果不触及内在的“因”，都将是徒劳的。理解并转变你的思维模式，是开启真正内在转变的第一步。

## 什么是思维模式？🧠

上一节我们介绍了课程的主题，本节中我们来看看什么是思维模式。

思维模式是我们每个人大脑中的认知世界的地图，或者说对一切人事物的看法。我们通过思维模式来认识和理解这个世界。你可以把它想像成你戴了一副隐形眼镜看世界，这副隐形眼镜会影响你看待世界的视角。

它影响你对一切人事物的理解，影响你对一切人事物的翻译，影响你对所有信息的筛选。但最有意思的是，我们从来没有意识到它的存在，我们从来没有察觉到我们被思维模式所影响。这是我希望你明白的第一个关键点。

## 我们并非客观看待世界 👁️

上一节我们定义了思维模式，本节中我们来看看它对我们的核心影响。

我们都觉得我们看待外部世界是绝对客观的，我们看到的外部世界就是它真实的样子。然而真相并非如此。你所看到的并非世界真实的样子，而是被你的思维模式制约着看到的样子。

记住，你所看到的并非世界真实的样子，而是经过你的思维模式处理后的样子。从你出生的那一刻起，从你来到这个地球上的那一刻起，各种限制性的信息，就通过各种途径不断地灌输给你。

以下是这些信息的主要来源：
*   你的家庭
*   你的朋友
*   你的学校
*   你的工作
*   电视媒体、新闻、互联网、社交媒体
*   电影、电视剧、音乐

这些限制性的信息，慢慢地塑造了你看待世界的视角，慢慢地塑造了你筛选外界信息的思维模式。另外，我们有时也把它叫做预设的思维模式，或者叫做预设的信念。

因此我们觉得我们看待事物是绝对客观的，大错特错。从你在这个世界降生的那一刻起，就仿佛有个人给你戴上了一副有色眼镜。正因为你一直都是这样看待世界的，因此你甚至意识不到你其实戴了一副眼镜。

## 两种核心信念：丰富与匮乏 ⚖️

了解了思维模式的隐蔽性后，我们来看看它的具体内容。为了方便你理解，我把人类的思维模式简化为两种最主要的信念。

第一种信念的出发点是：**你是丰富的**。你的内在状态是丰富的，或者说你是**十分**。

第二种信念的出发点是：**你是匮乏的**。你的内在状态是匮乏的，你是不配的，或者说你是**三分**。

如果你愿意，你也可以把这一对信念定义为：
*   你是快乐的 和 你是不快乐的（这里的快乐指内在状态，不是情绪很嗨）。
*   你是完整的 和 你是不完整的。
*   你是足够好的 和 你是不够好的。

总之，这一对信念是我们最根源的信念。而你在生活中的所有态度，所有的表现，所有的行动，发生在你身上的一切，都源自于这一对信念。你可以理解为此基础之上，我们形成了之后所有的思维模式。

一个人，要么是丰富的思维模式，要么是匮乏的思维模式；要么带着丰富的隐形眼镜，要么带着匮乏的隐形眼镜。

## 如何识别你的思维模式？🔍

上一节我们区分了两种核心信念，本节中我们来看看如何判断自己属于哪一种。

如果你想知道自己带的是哪一副眼镜，你可以做我在第一个视频里面教你的那个小实验：把意识带入你的身体，感受你的身体内升起的各种不同感受。如果按1到10分打分，给你的感受打分，你会打几分？

或者还有另一个办法，你可以简单地扪心自问：如果你什么都不做，你会怎么样？如果你停下你当下的所有努力，如果你停下自我提升的学习，如果你停下自信的建立，如果你停下追逐成功的脚步，如果你停止赚钱、攒钱，如果你断掉从外界获得认同和爱的所有途径……你会怎么样？

你会马上跌落到匮乏的状态，还是会扬升到丰富的状态？或者你会感觉没什么变化？

很不幸，就像我前面讲的，由于我们从刚出生的那一刻起，就被制约了，这导致我们每个人的内在的出发点，是匮乏的。匮乏的眼镜在一代又一代人之中传递，每个人刚一出生就戴上了“我是匮乏的”眼镜。

## 匮乏思维模式的后果与陷阱 🌀

认识到我们普遍从匮乏出发后，本节我们来深入探讨这种模式的后果。

你很快就会明白，匮乏的思维模式，是我们人类所有痛苦的根源。由于匮乏的思维模式，我们产生了小我，产生了自我厌恶、自我逃避、自暴自弃、自我阻碍，产生了长期的焦虑，甚至身体健康问题等等。

而且正因为你的内在的出发点是匮乏的，你在匮乏的出发点上所做的一切努力、一切改变，都是不管用的，都是不可持续的。在匮乏的基础上获得的好状态和快乐，都是不可持续的。你想要维持好状态，你必须不断燃烧自己的能量，不断地做做做。

除非我们转变我们的视角，除非我们把这副眼镜摘掉，觉察到我们自己的思维模式，并且开始对我们深信不疑的思维模式产生质疑，你才能找到真正的“因”，并且改变这个“因”。否则你做的任何事情，都只是在外在的“果”上白费功夫。

这就是为什么绝大部分自我提升、自我改变的办法，都好像是在溃烂的大伤口上贴上一个小创可贴似的。因为那些方法都是在外部做工作，在表面做工作。这是一种无休止的补偿行为。

## 为什么外在努力无效？🚫

我们已经看到了匮乏思维的危害，现在我们来分析为什么基于它的努力注定失败。

拿我们绝大部分人的处境举例子：我们每一个人都想要快乐，想要变得丰富，想要成为十分。但**我们是在匮乏的出发点上试图变得丰富，我们是在匮乏的思维模式中试图寻找丰富**。这就是为什么我们从未真正变得丰富，这就是为什么我们做的所有努力，都压根不管用。

因为不论你做什么，你的出发点都是匮乏的。不论你做什么，只会创造匮乏的果。丰富和匮乏是两种完全不同的状态。你想要丰富的思维模式，唯一的办法是先放弃匮乏的思维模式。

想一想学习自我提升和自身建设这件事吧。我们大部分人的出发点是什么？是你本来是差劲的，是不够好的，因此你需要学习自我提升来让自己变好。你在这样的思维模式下，所做的任何事情，都只会不断地强化你的出发点：你是不自信的、你是不够好的。

再比如另外一个很普遍的例子：要勤奋要积极。但你的出发点是，你不够勤奋，你不够积极。你的预设信念是，你是一个懒惰的、消极的。于是你不断努力让自己勤奋起来，积极起来。但你所做的一切努力，都只是在不断强化你“本来是消极的、懒惰的”这样的预设信念。

**核心公式：匮乏的因 → 匮乏的果**

我们无法成功的唯一原因就是：我们一开始的出发点就错了。

## 思维模式的强化与认同 🔒

理解了努力无效的原因后，我们来看看思维模式是如何被不断强化的。

有个比喻特别符合这两种思维模式：就假设你现在正在北京，但是有人给了你一张广州的地图。不论你多么努力地研究这张地图，你猜怎么着？都不管用的。**你拿错地图了**。

这个点很关键。记住：**只要你在某种思维模式中行为处事，你就无法从这种思维模式中跳出来。你从这种思维模式出发做的任何事情，会简单地强化这种出发点，以及建立在这种出发点之上的一切。**

不仅如此，你在一个东西上投入的越多，它也会变得越发坚固和强大。设想一下，你花了30年的时间拿着这张错误的广州地图，不断试图在北京找到正确的路。这个时候，有人告诉你：“你拿错地图了。”你会轻易扔掉它吗？绝不会。你的沉没成本太高了，你太认同于这张地图了。

你所面临的情况就像流沙一样。你被困在了这套思维模式之中。你使用这套思维模式的时间越久，你以这套思维模式为出发点做的事情越多，它就会变得越强化、越牢固。你在它上面投入的精力越多，你越不愿意放下它。就这样，你对这套思维模式上瘾了。

## 自我预期的应验 🔄

思维模式的认同不仅让我们难以摆脱，还会主动塑造我们看到的现实。

对于思维模式的认同也会导致“自我预期的应验”。简单地说，你会**在你遇到的一切人事物中，无意识地寻找符合你思维模式、符合你的预设信念的一切证据**。同时，你会自动忽略那些不符合你思维模式的证据和事情。

这个过程通常是潜意识发生的。你会不遗余力地找到证据，来强化你的“我不配、我不够好、我是三分”的思维模式。

**代码示例（比喻）：**
```python
# 你的思维模式就像一个过滤器
def perceive_reality(raw_input, belief_filter):
    if matches_belief(raw_input, belief_filter): # 如果输入符合信念
        return accept_and_amplify(raw_input) # 接受并强化它
    else: # 如果不符合
        return ignore_or_distort(raw_input) # 忽略或扭曲它
```

联想一下你过去经历的人生。比如你某一次特别生气，朋友过来跟你说了一个好消息，但你那一刻甚至完全不想听这个好消息，你的注意力全都放在了寻找其他可以让自己宣泄怒气的目标上。

这就是我们的思维模式的运作方式。你戴著它生活得越久，你就越强化这副眼镜，它就变得越坚固。而且你时刻都在无意识地，从生活中寻找各种证据，来证明和强化“我是匮乏的”这样的信念。

## 解决方案：内在工作的道路 🛤️

分析了问题所在后，本节我们终于来到解决方案：如何转变思维模式。

那么该怎么办呢？就像我最开始说的，你要从现在开始，逐渐将你的意识，带入你的思维模式。
1.  **开始去觉察**你的思维模式每天都在干什么。
2.  **后退一步**，像一个观察者一样去观察自己的思维和信念。
3.  **慢慢转变**你看待事物的视角。
4.  **检验**你的那些建立在匮乏的基础之上的那些思维模式和信念。
5.  **最后，你要放下它们**。

你需要重新审视那些你过去习以为常的想法。因为你坚持这些信念和想法太久了，已经把它们当做理所当然的了，所以你很需要去重新审视它们，放下它们。

有一个被用烂的例子，但是它真的很贴切：你和你的思维模式的关系，就像NEO和母体的关系。此刻你就在母体的幻象里。你**必须彻底转变你的思维模式**。

## 由内而外的创造法则 🌱

最后，让我们总结并展望这条全新的道路。

意思就是，你要走的这段旅程，是一条完完全全的内在旅程。**想要改变外在的果，你需要找到内在的因**，也就是找到你内在最深处的那核心状态，找到你内在最根源的思维模式，然后转变它。

你在现实世界所经历的一切人事物，一切外在的果，都是由内在的因创造的。如果你不在内在的因上下功夫，不在你内在的信念和思维模式上下功夫，你在外部所做的一切努力、一切事情，都只是在试图扭曲外在的果，只是在外在的果上白费功夫。

**核心法则：内在的因 → 外在的果**

你内在对于做你自己的体验，直接影响你对外部世界的体验。不转变这个内在体验，你无法真正改变你自己，你无法真正改变你的生活和你遇到的一切人事物。这完全是一个由内而外的工作。

一个很消极的人，无论他身处什么样的环境中，他总是会无意识地寻找能让他抱怨的理由和证据。一个总是害怕的人，他内心的恐惧就会把他的外部世界染色。一个总是生气的人，他内在的生气和不满就会把他的外部世界染色。

你对于外部世界的体验好坏，完全取决于你内在世界的体验是什么样的。这就是为什么，一旦你的内在状态转变为丰富了，你会在生活中的一切人事物之中，看到丰富，创造和体验丰富的结果。

因此，别再想着从外部改变了。外部的一切都是内在的果。你在外部做的任何努力，都只会强化内在的因。你要走一条和过去完全不同的道路了，一条内在工作的道路。我们要从内在的因上着手，从内在的思维模式上着手。

**本节课总结**

在本节课中，我们一起学习了：
1.  **思维模式**是我们认知世界的地图，它像一副隐形眼镜，但我们通常意识不到它的存在。
2.  我们并非客观看待世界，我们看到的是被自身思维模式处理后的版本。
3.  存在两种核心信念：**丰富（10分）** 与 **匮乏（3分）**。我们绝大多数人自出生起就被制约，从匮乏的出发点看待一切。
4.  在**匮乏的因**上所做的任何努力，只会创造出**匮乏的果**，所有外在的自我提升方法往往无效，因为它们只是在处理“果”。
5.  思维模式会自我强化，我们还会无意识地寻找证据来应验自己的预设信念（自我预期的应验）。
6.  真正的改变始于**内在工作**：觉察、观察、检验并最终放下旧的匮乏思维模式。
7.  创造的法则是**由内而外**的：转变内在的因（思维模式），外在的果（现实体验）自然会随之改变。

记住，如果答案真的在外部，你早就解决问题了。是时候停止在外在的果上白费功夫，转向内在，找到并改变那个最根源的“因”了。
