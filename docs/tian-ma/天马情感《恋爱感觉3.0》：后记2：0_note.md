## 警惕误区：为何不提倡“狙击”

说实话，我不喜欢教“狙击”女生。可能有些人不理解“狙击”的含义。它指的是你们常说的：“老师，我正和一个女生聊天，我要挽回/追她，我太喜欢她了，请教我如何搞定她。”这就叫狙击。

为什么这件事困难且成功率低？并非我没有教给你方法，而是这些方法对你来说很难执行到位。我用“狙击”这个词，本身就说明了这是一件困难的事。狙击手很重要，但任务也特别艰巨，比如击杀敌方将领。你刚认识一个女生，就非她不可，这对很多新手来说极其困难。

这相当于在你还是“小菜鸡”的时候，我教你如何举起一把几十斤重的狙击枪，卧倒在冰天雪地里，一动不动、不吃不喝地埋伏上一周。这会导致你很痛苦，一点也不快乐。高手虽然也用方法，但他们在使用时是快乐且享受的。

我自己在不太会把妹的阶段（大约接触了三四十个女生时），也曾尝试过死追（狙击）某个女生，结果特别累，效果也不好。那时无人指导。因此，我真的不希望你们学完课程后变得不快乐，这不是我教学的初衷。

我从事这个行业，是想帮助男人们懂女人，变得更轻松，不必为了追一个女生而纠结、窝囊，像个娘们一样整天苦恼。我不希望你是这样的。男人应该有男人的样子。作为新手，我希望你能享受“拔枪射击”的快乐，而不是年纪轻轻就执着于“狙击”，要求自己沉稳、稳住气。我不希望这样。

## 视野与格局：你为何“非她不可”

当你多思考，或更成熟一些时，可能会发现当前的执着并不值得。当然，你也可以固执地坚持你的想法，这没问题。毕竟，你现在可能还想不通。但如果你坚持要“狙击”，我希望你是快乐的，而不是一边坚持，一边又感到烦恼和痛苦。你有获得快乐的方式，何必为难自己，沉醉在自我感动的氛围里呢？没必要，成年人应该更理智。

你们才二十几岁，三十岁也很年轻，往后有大把时光。如果你说“我赌，我不后悔”，那没问题，去做吧。但我希望你是正能量且开心的。

当你长大一些，可能会发现，你“非追到这个女生不可”的想法，往往源于**你的世界太小了**。因为你没见过山川湖海，没见识过外面的花花世界，所以你把小溪当成了江流，把小小的土丘当做了高峰。你的圈子太小，因此你会把偶然接触到的女生当作一辈子的唯一，从而失去了探索未来的可能性。

你可能错过了攀登更高峰的机会。你会觉得：“老师，我觉得这女生很高分、很好看，她是空姐、模特、主播。”当我泡到第一个模特时，我也特别兴奋。但后来当我接触到一些演员（非一线明星）、空姐时，感觉就淡了。因为我遇到了更高层次的妹子，一些白富美远比空姐、模特优秀。

真实的花花世界有大把姑娘。当你掌握了把妹的方法后，你可以将接触到的所有女生都变成潜在的对象。那个你曾觉得特别爱的女人，可能只是路边的一束野花。对你们说这些，你们可能会觉得：“老师，不一样的，我是很专一的人。”大家可能以为我是一个很浪荡风流的人，那你就错了。

## 把妹的定位：是技能，不是生活全部

每个人的价值观不同。在我的价值观里，花太多时间在“泡妞”上有点傻。说实话，我并不是一个特别爱“泡妞”的人，但我是一个特别爱“把妹”的人。你或许没见过什么叫特别爱把妹，我身边就有这样的人，每天泡妞很开心、很兴奋。
