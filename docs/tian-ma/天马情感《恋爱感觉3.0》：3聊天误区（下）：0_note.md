## 误区六：聊天缺乏明确目的，关注点错误 🎯

你和女生聊天时可能没有明确目的，并且关注点错误。记住“关注点”这个词。

你根本不知道自己是为什么去找这个女生聊天的。对于恋爱新手，几乎没人想过这个问题。你们就是喜欢她，然后想着跟她聊天去追她。

**记住这是一个错误。**

一旦你不知道为什么去找她聊天，就会导致你在聊天时最在乎的东西出错。你们最在乎的其实是：女生有没有回复你，或者她回得好不好。她只要回你，你就开心；她不回你，你就失落。

但这全错。错在你很关注她。你关注她回不回你，你关注她的个人生活。这导致你越聊天就越喜欢这个女生，因为你越聊就越觉得自己很懂她、很了解她。

以下是你在聊天中可能了解到的关于她的信息：

*   你知道她经常被领导欺负。

*   你知道她是家里最小的。
*   你知道她很渴望被人保护。
*   你知道她很内向。
*   你知道她没什么社会经验。
*   你知道她被前男友伤害过。

你现在能说出很多关于这个女生的信息，因为她的人物形象在你的脑海里很丰富、有血有肉。

如果她对你一无所知，那她怎么会喜欢上你呢？你得让她了解你的性格，让她了解你的生活，她才会产生想象空间，才会幻想未来跟你在一起。

你跟她聊完天之后，如果让她觉得你这人很帅、很会说话、情商很高，那么她就有可能喜欢上你。

你最不该关心的是她回不回你信息，也不是她过得好不好、辛不辛苦。她缺钱、她前男友对她不好、她很傻逼、她很没社会经验，都不关你事。

那我再教你怎么让她对你有印象。你在聊天中不要话题一直围绕着女生去转，你要多聊聊自己，不要害怕去表达自己。

“我喜欢什么”、“我喜欢什么样的女人”、“我喜欢吃什么”，这都是你需要去大胆表达的东西。你不敢去表达的结果就是她对你没有印象。那她心里的那个属于你的小人就永远是空白的、没有色彩的，她也不会喜欢你。

## 误区七：盲目表白，增加追求难度 🚫

基本上，从是否犯第六个错误，就可以衡量一个人懂不懂得追女生了。我特别能理解，因为我以前读书的时候就很爱干这个事情。我追了六七个女生（陆陆续续，不是同时），最后都去表白，也都成功了。但这依然不能说明表白是个好事。表白对于我那个时候，包括对于很多人，都是一个错误的行为。

为什么错误呢？我给你解释一下。首先你得明白表白这个行为是怎么一回事。

表白是什么？我们抛开一些花里胡哨的东西，表白就是你去告诉一个人：“我喜欢你，非常非常喜欢你。” 那么这个行为就是表白。

这个行为的意义是什么呢？你们不懂。反正你们小白去追女生，包括我读书那会儿去追女生，喜欢上一个人就想告诉他，就要表白。但是，一个真正懂情感、懂女生心理的高手会告诉你：**表白是为了给女生安全感**。你去告诉一个女生“我好喜欢你”，让她知道你的心意，这实际上是给一个不安全的人一个安全感。

这是为什么呢？就是在男女关系已经基本确定的时候，你补上一个最后的仪式。为什么需要有这个表白行为？因为你确定女生对你有好感、喜欢你，但是女生不确定你是否喜欢她。她会想：“我都已经跟他这样了，他是不是喜欢我？” 那么你就必须得告诉女生，让她有安全感：“哦，我是喜欢你的。”

也就是说，你表白的时候，实际上你们已经十有八九马上要在一起了。

但是很多人都是反着来的，包括我读书的时候都是这样：先上来，我喜欢一个人，我要追你，我上来先表白：“我喜欢你。” 然后就被女生拒绝，被她说“哎呀你诚意不够”、“我太快了吧”、“我们还是当朋友”，然后你就接着努力追。应该都有经历过。

如果你还不是很明白，我再举个例子。你不表白的情况，就相当于你平时对你朋友很好，你很乐意助人，经常请他吃饭、帮助他。他就觉得你这人不错。然后有一天，你去找他借一笔钱，他听完爽快地答应了。为什么？因为他觉得你人很好，他就借你钱了。

那你一上来就盲目表白等于什么？等于你一上来，跟这个朋友都不是很熟，你就跟他说：“哎，你能不能借我一笔钱啊？” 那他干嘛？他肯定拒绝你。“你是谁啊？我对你又没好感，对吧？我又对你不信任，我为什么会借你钱？不可能。”

那接下来干嘛？接下来你就经常请他吃饭，你去帮助他，这就像你追求女生一样。可是那个朋友他觉得你干嘛？他就觉得：“哇，你这些行为都是抱有目的的嘛，你不就是想让我借你钱吗？” 所以你的这些好好的行为，请他吃饭、帮助他的行为，并没有让他对你的感觉、对你的印象变得好。所以最后你去找他借，他依然不会借你。懂了吗？

所以呢，盲目的表白就相当于你把正常追女生的一个顺序颠倒了、调换了。仅仅是这个行为就直接导致了你难度翻了几倍，你给自己再设置障碍。

所以呢，没事你就别瞎表白啦。女生问你是不是喜欢他，你就装傻。你要实在不懂怎么回，你就来问我。不然有时候你都不知道自己为什么失败了，或者为什么成功，就竟然不知道自己错在哪就死掉了，女生就不理你了。
