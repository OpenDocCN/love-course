# 情感关系分析课 01：前任评价与现任关系 🔍

在本节课中，我们将要学习如何通过一个男性对其前任的评价，来分析他的性格特质以及预测他未来在亲密关系中的行为模式。理解这一点，有助于你在关系初期做出更明智的判断。

## 前任负面评价的预警信号 🚩

为什么你选择的男性会向其他女性详细列举你的种种不是？这种行为的特性，他一定在之前就有所体现。是的，有所体现。他们都在骂前任。当我在形容他们骂前女友时，我听听就过去了，觉得反正跟我没关系。

我告诉你们，他的很多性格特征不会轻易发生改变。他为什么会向后来交往的女朋友传递这么多关于你的负面信息？这一定不是一天两天形成的事。这种性格上的东西，在前面就一定有所体现。

上一节我们提到了通过评价观察性格，本节中我们来看看如何在实际交流中识别这些信号。

## 如何通过提问识别缺乏自省的人 ❓

我身边遇到很多单身男性。在跟我沟通交流的过程中，我就会非常在意这个点。我会先问他：“你跟你前任是怎么分手的？”或者说：“你跟你前面的这些恋爱关系都是怎么结束的？”

如果一个男生向我讲述了三段恋爱经历。

以下是可能出现的回答模式及其含义：

*   **模式一：归咎对方**
    他能够向我表达出，是那个女生做了一些不好的事，然后导致关系结束。他最后归因时，一直停留在这个点上。

*   **模式二：自我反思模糊**
    我就会问他：“那么也就是说，在你过去所有的情感关系结束时，你认为都没有你的原因吗？”他们肯定会先愣一下，然后会说：“当然也有我的原因。”但当他说自己的原因时，永远含糊不清。他表达不出，这其实说明他没有很强的自省能力。

在亲密关系的经营过程中，他没有自省能力。他缺乏自省能力。因此，他会不自觉地把所有问题都归结到对方身上。

所以，像这样的人，你就要意识到：如果你跟他开展一段关系，那么以后如果你跟他分手了，他就会把你作为他的一个“错题”去讲给他的下一任听。

## 关于前任评价的常见误区 💡

我正好顺嘴说一句。很多女生，在刚开始跟一个男生认识的初期，在沟通了解他过去的时候，就会陷入一个误区。

你们特别怕这个男的说自己前任好。你觉得他肯定忘不了前女友。他忘不了前女友，所以我没有办法跟他开展一段关系，我可能会替代不了那个人。你们是不是会有这种想法？

其实，他说前任好的这种男人，就好过他说前任不好的那种男人。

以下是两种评价方式的对比：

*   **健康模式：客观评价**
    他如果能够非常坦诚地讲出彼此双方有什么问题，然后还能够去肯定他的前任。

*   **不健康模式：一味贬低**
    相反，只会抱怨和贬低前任的男性，往往缺乏自省和共情能力。

反而，能够客观评价前任的男人，是更值得你们去托付的。

## 课程总结 📝

本节课中，我们一起学习了如何通过男性对前任的评价来分析其性格。关键点在于：**一个习惯性将所有关系问题归咎于前任、缺乏清晰自我反思的男性，在未来关系中很可能重复这一模式**。而能够客观、平和地评价过去关系（包括双方问题与对方优点）的男性，通常具备更好的自省能力和成熟度，是更值得信赖的伴侣。
