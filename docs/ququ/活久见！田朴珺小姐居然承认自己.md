# 活久见！田朴珺小姐居然承认自己

在本节课中，我们将要学习一种个人成长与心态转变的方法。我们将通过分析一个具体案例，探讨如何通过观察、模仿与自我和解，褪去因奋斗而产生的“用力过猛”气质，从而获得更从容的生活态度和思维方式。

---

一位公众人物近期承认了某些事实，这引发了讨论。她出生于1981年，现已年过四十。

在四十不惑的年纪，她或许终于能够放下过去许多“用力过猛”的表现。

那么，什么是“用力过猛”的气质呢？通常，那些依靠自身努力从无到有、奋力跨越阶层的人，脸上常会带有这种气质。他们非常担心别人认为自己是依靠他人、能力不足或利用了性别优势。

我完全理解这种“用力过猛”，因为我自己的道路也是从零开始，一步步努力走来的。

然而，生活中许多不熟悉我的人，常误以为我出身于优越的家庭环境。

这背后的原因是，我花费了多年时间，刻意褪去了自己身上的“用力过猛”气质。

我是如何做到的呢？方法是通过与大量的富一代和富二代深入接触。例如，一起用餐、社交、创业或共同攻读MBA等。

在接触过程中，我专注于模仿他们身上的一些共性。

以下是模仿的几个主要方面：
*   **外在形式**：穿着打扮、饮食偏好等习惯。
*   **行为模式**：待人接物的态度与方式。
*   **内在状态**：寻找那种与生俱来或长期拥有财富所带来的“从容感”。

讲到此处，或许有人会感到不屑或抗拒，认为这样做没有必要且令人疲惫。

我想分享一段曾写在朋友圈的话作为回应：当你不再忧虑他人如何看待自己，而是将目标聚焦于“我所羡慕的人生是何模样”，并为之持续努力、不断逼近时，你的心态便会越来越开阔。

因为我羡慕他们，并渴望成为他们，所以我选择模仿他们。这种模仿不仅限于表面的作风，更深入到内在的心态格局与思维方式。

这与我幼年学唱歌时老师教导的道理一致：若想成为某位大师，首要之事便是模仿那位大师，乃至每一个咬字和换气都要力求相似。

因此，要祝贺这位女士在四十不惑之年，学会了与自我和解。

你是否也有特别想成为或想去模仿的人呢？

「你就是我的天堂」。

---

本节课中，我们一起学习了个人成长中的一个关键转变：从“用力过猛”到“从容不迫”。核心在于将关注点从“他人眼光”转向“理想自我”，并通过系统的观察与模仿——从外在习惯到内在思维——来实现心态的蜕变与和解。记住，模仿是迈向理想自我的第一步。
