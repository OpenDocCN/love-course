# 💰 财富思维课 01：为何“恨钱”会让你远离财富？

在本节课中，我们将探讨一个核心观点：一个人对金钱的态度，深刻影响着他获取财富的能力。我们将解析内心欲望与社会标准之间的冲突，并理解为何“憎恨金钱”会成为财富道路上的根本障碍。

---

没有人会完全不知道自己想过怎样的生活。所有模糊不清的感觉，都源于你内心的欲望与社会通行标准之间的摇摆。你会觉得，自己内心真正渴望的东西，似乎难以公开言说，甚至“见不得光”。

我之前推荐过一本书，名为《有钱人和你想的不一样》。书中有一个观点令我非常认同：**人永远无法获得你讨厌的东西**。

书中指出，穷人之所以难以变得富有，是因为他们内心深处在憎恨金钱。他们每天都在咒骂有钱人，抨击有钱人。当社会舆论对富人不利时，他们则在欢呼呐喊。那么，穷人实际上在做什么？他们是在让自己的“气场”与金钱相互排斥，让自己的能量远离金钱。

他们宣称“金钱是万恶的”，认为“爱钱就是物质的人”。我从小到大从不仇富。我只会羡慕，并渴望成为那样的人。我看到他们能用金钱办成的事、用金钱获得的能量，我就会羡慕。我会思考：为什么他能做到那样？我要如何才能变成那样？这就是我思考的方式。这正是《有钱人和你想的不一样》这本书所传达的理念。

字幕君也离开了。

---

本节课中，我们一起学习了财富思维中的一个关键心态：对金钱的憎恶与排斥，会在能量层面阻碍你吸引财富。真正的改变始于正视并接纳自己对财富的渴望，将羡慕转化为学习的动力和清晰的目标。
