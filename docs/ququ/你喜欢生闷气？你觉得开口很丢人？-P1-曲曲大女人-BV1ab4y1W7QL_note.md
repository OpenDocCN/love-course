# 沟通的艺术：从生闷气到有效表达 🗣️

在本节课中，我们将探讨一种常见的沟通困境——生闷气，并学习如何通过调整心态和表达方式，将消极的“冷战”转化为积极有效的沟通。

## 观察与模仿：原生家庭的沟通模式 👨‍👩‍👧

上一节我们介绍了课程主题，本节中我们来看看这种沟通模式是如何形成的。

在父母那一代，妈妈们特别喜欢生闷气。她们不高兴了就不说，让别人去猜。提出自己的诉求仿佛是一件很羞耻的事情。在这样的教育环境下，看到妈妈是这个样子，孩子也会变成这个样子。

## 一个亲身经历的故事 📖

以下是笔者童年时的一个真实经历，它揭示了生闷气对家庭关系的直接影响。

小时候，有一次和爸妈一起吃宵夜。妈妈不知为何生气了，她不吃。笔者吃得很香。爸爸一直在旁边哄她说“吃嘛吃嘛”。妈妈不吃，爸爸也不敢吃。笔者对爸爸说：“你别管她，她自己知道吃。”学着大人模样。爸爸当时仿佛听见了惊天噩耗，脸色一惊。妈妈把筷子一甩，站起来就走。笔者当时不知道怎么了。饭只吃了一半，吓得不行，赶快追上去。一家三口就这样追上去。后来还是爸爸拼命哄她，才把她哄好。后来妈妈对笔者说：“我觉得你不是我亲生的。”

## 生闷气的本质与循环 🔄

上一节我们看到了一个具体案例，本节中我们来分析这种行为背后的逻辑。

听到这句台词后笔者才明白，女人就是会生闷气。她们用沉默去表达内心真实的诉求和想法。她们不愿意沟通，回避沟通，然后喜欢让别人去猜。如果别人猜不准，她们就要继续生气。

## 陷入同样的模式：笔者的过去 💔

以下是笔者自身一度陷入同样沟通陷阱的经历。

有几年时间里，笔者也在走这个路数，因为是从妈妈那里学来的。曾经有一个很爱笔者的男朋友。那个时候，笔者也是会生闷气，不说话。他怎么说话笔者都不理。他就问：“你到底为什么生气？”那个时候笔者觉得自己很牛，觉得说出的话颇具智慧，但现在觉得当时是个傻叉。笔者说：“如果你知道，那么我不用说什么你也懂。如果你不知道，那我跟你说了还有什么意义呢？”笔者认为对方要懂自己。对方懂，那么不说也懂。对方不懂，说了也白说。这些都是过去的毒鸡汤，大家都是这样过来的。

## 核心转变：沟通的必要性 💡

现在笔者明白了一个道理：**一定一定要去沟通**。

当对方做的事情远离了你的预期，或者你对这段关系有某种要求和诉求的时候，一定要去表达。

## 两种错误的表达者 🚫

以下是两种常见但无效的“表达”方式。

一种人是不会沟通不会表达，一直等着别人去猜、去琢磨。

还有一种人也很极端，他们表达的时候都带着怨气。因为他们首先有一个错误的底层思维摆在下面。这个底层思维叫做：**“你就是不懂我，还要我开口讲。”**

当你的底层心态、底层思维整个错误的时候，你表达出来的话、你的措辞、你的态度就一定很糟糕，很让人难以接受，很不中听。这就不叫沟通，这叫泄愤。

## 建立正确的沟通心态 🧠

所以，当你们试图要去表达自己的任何一种诉求的时候，首先要树立一个正确的底层心态。

正确的底层心态是：**不是对方不懂我，而是如果我不说，他永远都不会懂。**

每一个人都是一个独立的个体。连你的父母都没有办法完全懂你。连跟你有血缘关系、从小到大一起生活的人，你们之间都会有理解的屏障和壁垒。你怎么还能试图让一个跟你没有任何血缘关系的人，就应该懂你呢？

好的恋人也好，好的朋友也好，其实都是靠培养的。

## 总结与核心理念 ✨

本节课中我们一起学习了从生闷气到有效表达的转变。

在我们的理念里头，这个世界上没有不能开的**口**，只有错误的**态度**和**措辞**。
