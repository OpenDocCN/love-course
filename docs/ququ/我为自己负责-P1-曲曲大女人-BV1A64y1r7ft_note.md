上一节我们看到了不同生活方式的存在，本节中我们来总结核心的个人责任原则。

你认为一个人必须专一，与某人约会时就不能与其他人约会。你可以这样想，但别人也可以不这样想。别人按照自己的想法行事，并没有伤害谁，因为他/她的态度是公开的。

以下是这种态度的核心表达方式：
**公式/代码：** `我的规则 = 公开透明；你的选择 = 知情同意`

这意味着：你要跟我在一起，就需要接受我的方式。这很正常。每个人都有自己的情感观和生活态度。我尊重我的意愿，也尊重你的意愿。这就足够了。为何要用自己的想法去约束别人？

以下是为你的人生负责的关键步骤：
*   **坚持自我选择**：按照自己的真实意愿生活。
*   **屏蔽外界噪音**：不要活在别人的评价里。
*   **审视传统观念**：不要被那些让你不快乐的道德观所束缚。

你想怎么样就怎么样，为你自己的人生负责。
