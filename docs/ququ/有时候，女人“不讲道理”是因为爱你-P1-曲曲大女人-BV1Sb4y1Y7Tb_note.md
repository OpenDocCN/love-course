我们常常听到一种说法：“女人不就是应该不讲道理吗？” 说出这种话，通常是为了实现某个**目的**或满足某种**诉求**，而不是真心认为女性天生就该不讲道理或情绪化。

上一节我们分析了这种沟通的表层目的，本节中我们来看看，为什么不能将这种沟通策略内化为个人价值观。

因为在一段感情中，你们必然会遇到**大事**和**大是大非**的问题，会面临需要你做出关键决策的时刻。如果你的内心**价值观**（即你坚信的“道”）就是“女人应该不讲道理”，那么在需要理智决策的关头，你就无法保持清醒。

**错误价值观 + 关键决策时刻 = 非理智决策**
