# 沟通课 01：如何正向沟通？放下对抗情绪 🧠

在本节课中，我们将学习如何通过“放下对抗情绪”来实现正向沟通。我们将通过一个具体的生活实例，分析对抗情绪的产生原因，并探讨如何避免它，使沟通更加顺畅有效。

---

## 什么是“对抗姿态”？🤔

经常有粉丝在咨询时，我会建议他们在亲密关系或其他人际沟通中“放下对抗姿态”。很多人不理解什么是对抗姿态。

上一节我们介绍了课程主题，本节中我们来看看什么是对抗姿态。

对抗姿态是指在沟通中，当听到对方的话时，不自觉地产生防御或反驳的心理状态，导致对话陷入争执。

## 一个生活实例 🏠

今天中午，我与家里的做饭阿姨有一段对话，很好地说明了这个问题。阿姨每天中午1点来做饭，我们家2点吃饭。但今天中午12点左右，她发信息给我。

她说：“我今天不过来做饭了，因为我要去做核酸检测。”我回复她：“阿姨，下次请假请提前一天告诉我。”结果阿姨的对抗情绪立刻上来了。

她说：“请什么假呀，谁想请假呀！我要不是因为路过一个小区码变黄了，怎么会请假？我也不想去做核酸。”看到她的话，我没再争论，只说：“没关系，阿姨注意安全。”

## 对抗情绪的产生原因 🔍

她为什么会有这样的对抗情绪？我能理解。因为她告诉我她要去做核酸检测时，我没有第一时间问她原因，而是直接要求她提前请假。她觉得我误解了她，因为她本意并非想请假。

但从我的角度，为什么没问她原因？我认为她不只在我家工作，可能还有其他客户。有的客户可能需要核酸检测报告，这是她的私事，我不应过问。这是出于礼貌的表现。

但她没有理解到这一层，直接认为我误会她临时请假。日常生活中，很多沟通都是这样。

## 选择性理解与情绪升级 ⚠️

当听到对方一句话时，我们常会选择性理解成第一反应的意思，而忽略对方背后的初衷。这时，愤怒值上升，说出的话就不好听。

如果对方也不控制情绪，很容易争执起来。这件事其实有多种正向沟通方式。

以下是正向沟通的示例：

*   在我要求提前请假时，她可以说：“我也不想请假，但因为路过小区码变黄了，为了你们的安全，我得去做核酸检测。”
*   这样沟通，双方都会更愉悦。

这就是“放下对抗情绪”。

## 如何实践正向沟通？🛠️

上一节我们分析了对抗情绪的产生，本节中我们来看看如何实践正向沟通。

以下是几个关键步骤：

*   **暂停第一反应**：听到对方话后，先别立刻反驳，停顿几秒。
*   **询问澄清**：用提问代替假设，例如：“你能多告诉我一些原因吗？”
*   **表达理解**：先认可对方感受，如：“我明白你觉得被误会了。”
*   **陈述意图**：清晰说出自己的初衷，如：“我那么说是出于礼貌，不想过问私事。”
*   **共同解决**：聚焦解决问题，而非争论对错。

## 总结 📝

本节课中，我们一起学习了如何通过放下对抗情绪实现正向沟通。关键点是避免选择性理解，在沟通中暂停第一反应，通过询问和澄清来理解对方初衷，并清晰表达自己的意图。

记住公式：**正向沟通 = 暂停反应 + 询问澄清 + 表达理解 + 陈述意图**。练习这些步骤，能让你的沟通更顺畅，减少不必要的争执。
