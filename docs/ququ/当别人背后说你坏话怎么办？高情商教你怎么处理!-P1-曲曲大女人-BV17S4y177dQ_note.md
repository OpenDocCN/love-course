# 情商课 01：如何应对背后的议论？🤝

在本节课中，我们将学习一种高情商的沟通策略，用于处理他人背后议论你的情况。这种方法的核心在于通过正向反馈来扭转局面，建立良好的个人形象。

---

当有人向你转述他人对你的负面评价时，无论涉及哪个方面，你的应对策略应是：**想办法去说那个议论你者的好话**。你需要理解，那些传递是非的人，本身就是信息传播最快的渠道。

上一节我们提到了应对的基本原则，本节中我们来看看其背后的逻辑。你之所以要这样做，是基于一个重要的认知：**凡是你能从此人嘴里听到他如何议论别人，你就要相信，被议论者同样会从他嘴里听到你如何议论他**。因此，运用这套逻辑，久而久之，周围人对你的印象会逐渐改善，你的处境也会得到扭转。

但其实，你个人内心如何看待那个议论者并不重要。无论你是否真的做错了事，或者即便你毫无过错，别人只是捏造事实，这些都不影响此策略的应用。你私下如何对待他是另一回事。

以下是具体操作时的回应思路示例：

假设情境：你开车不慎撞到了A的母亲（并无大碍），A因此在背后指责你。当有人（B）向你转述A的抱怨时，你可以这样回应：

> “这次确实是我撞人在先。我遇到他妈妈了，虽然没什么大问题，但作为女儿，A肯定非常着急。而且A这个人本来就很孝顺，有这种情绪完全可以理解。所以她这么说我，我完全能够理解。”
> 
> “我肯定要把我该做好的事情都做到位。”

---

本节课中，我们一起学习了一种以退为进的高情商沟通技巧。其核心在于：**通过公开肯定议论者的优点与处境，来间接塑造自身通情达理、勇于负责的形象，并利用“传声筒”效应，让正向信息回流，从而改善他人对你的看法**。记住，**控制你能控制的言论，而不是纠结于你无法控制的他人私下的评价**。
