# 吸引力法则课程 P1：如何重新建立吸引力 ✨

在本节课中，我们将学习如何通过关注自我、提升个人价值来重新建立吸引力。核心在于停止追逐他人，转而专注于自身世界，从而激发他人的好奇与关注。

---

上一段我们提到了吸引力的核心是自我价值。本节中我们来看看具体如何操作。

你无法通过抓住他来让他关注你。不如关注你自己的世界，让他产生好奇。

例如，当我在演讲或唱歌时，我的状态会让别人觉得“很厉害”。因此，当我在某人眼中的价值出现下滑趋势时，我会更积极地去做这些能让我“闪闪发光”的事情，从而将我的价值重新拉高。

你目前在他眼中可能已经缺乏吸引力，甚至让他感到厌烦。如果你此时还强行要求他：“快点跟我说话”，这只会加剧他的反感。

不要试图强行改变他。

不要试图强行让他向你靠拢。你需要让他对你产生一种“看不懂”的感觉。

你要让他发现：为什么我如此不开心？为什么家里每个人都充满负能量，唯独你一个人天天养花种草、读书唱歌？你究竟在快乐些什么？

设想一个场景：如果街上行人彼此都不认识，但其中有一个人突然停下脚步，抬头望向天空，摆出一副若有所思的样子，非常认真、沉迷地盯着天空看。

其他人都正常行走，只有他一个人站在那里对着天空发呆。这个时候，你或多或少会产生好奇，也会想看看天上究竟有什么。

人性即是如此。既然无法通过抓住他来获得关注，不如就专注于你自己的世界，让他对你产生好奇。

与其活在他的世界里，不如活在你自己的世界里。我并非在开玩笑，也不是在表演小品。我讲述的是切实可行的道理。

---

本节课中我们一起学习了重建吸引力的核心方法：**停止追逐，专注自我**。通过提升自身价值、沉浸于自己的世界并创造“不可预测性”，从而自然激发他人的好奇与关注，让吸引力得以重建。
