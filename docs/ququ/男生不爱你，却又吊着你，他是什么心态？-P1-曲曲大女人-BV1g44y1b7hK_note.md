## 行为定义与心态分析

上一节我们概述了课程目标，本节中我们来看看具体的行为模式。

“推菜”是指一个男性主动接近你后，又将你推荐给其朋友的行为。这种行为本身定义了他的意图：他并非认真追求，而是在“玩”。

以下是这种行为背后的核心心态：
*   **缺乏尊重**：他将女性视为可以互相转让的“菜”，而非独立的个体。
*   **逃避责任**：他通过将你推给朋友，避免了自身可能需要承担的情感责任或风险。
*   **寻求刺激**：初始的“撩”可能仅出于外貌吸引或无聊，是一种简单的试探，而非基于喜欢的认真开始。

## 如何解读行为事实

了解了基本心态后，我们需要掌握客观分析行为的方法。关键在于**剥离感性猜测，聚焦客观事实**。

以下是分析此类情况的步骤：
1.  **确认发起方**：事实是**他先主动联系并邀约你**。
2.  **识别矛盾点**：事实是**他在约会后，代替他的兄弟向你传达好感**。
3.  **逻辑质问**：如果他的兄弟是认真的，正常逻辑应由其兄弟直接与你联系。他代为传达的行为本身就是一个需要质疑的信号。

**核心逻辑公式**：`他的行为（代为传达）` ≠ `正常追求逻辑（本人直接表达）` => `动机存疑`

在厘清事实后，我们需要采取正确的应对方式。首先，必须调整自身心态：不要默认对方在“追求”或“试探”你。
