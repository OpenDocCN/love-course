## 概述 🎯

在本节课中，我们将要学习一种引导他人情绪波动的小技巧。这种方法的核心在于通过设定并“违背”未来的承诺，来引发对方的关注与情绪反应，从而在互动中占据一定的心理优势。我们将详细拆解其操作步骤与核心逻辑。

## 核心概念与操作原理

上一节我们介绍了课程的主题，本节中我们来看看其核心操作原理。这种方法并非关于真诚的承诺，而是一种有意识的互动策略。

其核心公式可以概括为：**设定未来承诺 → 不主动履行 → 等待对方反应 → 给出随意解释**。

这个循环的目的是打破对方对你的行为预期，制造心理落差。

## 操作步骤详解

理解了基本原理后，以下是具体的操作步骤，我们将分点进行说明。

1.  **设定未来时间线的承诺**
    主动向对方提出一个将在未来某个时间点进行的互动。例如：“我明天下午给你打电话”，或者“我睡觉前给你打语音”。

2.  **主动违背承诺**
    到了约定的时间点，不要主动履行承诺，就像忘记了一样。这是引发对方情绪波动的关键。

3.  **等待对方主动询问**
    当对方因为你的“失信”而感到不爽并主动联系你时，你的第一步目标就达到了。

4.  **给出随意、轻松的解释**
    面对对方的询问，不要郑重道歉。你需要给出一个看似合理但非常随意的理由。例如：“那会儿太晚了，我以为你睡了呢”，或者“我临时被我妈拉去给伯伯拜年了”。

## 操作频率与注意事项

我们已经了解了完整的操作步骤，接下来看看执行的频率和一些要点。

关于操作的频率，建议不要过于频繁。隔一段时间操作一回即可，比如可以说“我明天去看展，到时候把看到的发给你”，然后忘记。

在操作中，需要保持姿态轻松。你的解释要显得随意、即兴，而不是精心编造的谎言。核心是传递出“我给了承诺，但我并不那么在意”的潜台词。
