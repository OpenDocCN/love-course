在本节课中，我们将要学习一种重要的人生态度：通过降低对他人和关系的预期，来获得内心的平衡与快乐，从而避免患得患失的情绪困扰。

## 核心原则：降低而非抬高预期

对任何人、任何关系，我们首先要做的第一件事是降低预期，而不是不断抬高预期。抬高预期总会让人感到慌张、害怕和患得患失。降低预期后，对方给予的任何一点好，都会让你感到开心和快乐。

自己内心的平衡至关重要，因此，我们的行动方针是去降低预期，而不是去提高预期。

## 剖析无意义的情绪：恐惧

上一节我们介绍了降低预期的核心原则，本节中我们来看看一种与之相关的无意义情绪。

恐惧可能是这个世界上最没有意义的一种情绪。它可能是一种客观存在，也可能是未来有可能会发生的，然而你对引发恐惧的这件事本身，没有任何控制能力。

你无法预判它，也无法改变它。那么，恐惧的意义是什么？

## 应对无法控制之事：专注自我

既然我们认识到对某些事无法控制，那么现在应该怎么做？答案是：什么都不要强求。

如果一件事或一个人要离开，你是拉不住的，也管不着。这就像当初对方来喜欢你、追求你的时候，也不是你期盼而来的，是对方自己来的。

**核心逻辑可以概括为：**
```
它要来，就会来；它要走，就会走。
```
你的任务是做好你自己。在对方喜欢你的时候，你是什么样的人，就永远做那样的人。这样就可以了。
