## 为什么你总在后悔？🤔

上一节我们介绍了课程主题，本节中我们来看看后悔心态的核心原因。许多人做完决定后反复后悔，这种心态波动的根源在于对自己缺乏要求。

**核心观点**：你之所以习惯后悔，是因为你对自己没有原则性要求。你不要求自己成为一个“一言九鼎”、有原则有底线的人。

因此，你在做决定时很随意，做完决定后又轻易反悔。这种模式会导致人生陷入混乱。所有最终把自己活成一团糟的人，往往都是从年轻时就开始对自己毫无要求。

**行为惯性公式**：
```
年轻时的无要求 → 长期的行为模式 → 未来的糟糕状态
```

如果一个人在20岁时对自己没有要求，那么到80岁时，其生活状态很可能一团糟。

## 如何调整心态？🔄

我们已经了解了后悔的根源，接下来看看如何调整这种心态。关键在于转变关注点：从“我放弃了什么”转向“我要成为什么”。

**核心心态**：我的核心要求是**我要变得更强大、更好**。至于我曾经放弃的选项后来如何发展，与我无关。

以下是调整心态的具体方法：

*   **停止比较**：无需调节对过往选项的执念。别人过得好与不好，都与你当下的选择和成长无关。
*   **聚焦成长**：你之所以总想回头，根本原因在于自身没有进步。无论是收入、认知还是信念感，如果一切停滞不前，你自然会觉得曾经看不上的东西也可能变得诱人。
*   **建立决策自信**：许多人害怕自己的决定不对，因此反复纠结。必须建立一种信念：**做了的决定，就是对的决定**。

## 唯一错误的决定：回头 ↩️

前面我们讨论了心态调整，本节将阐述最重要的决策原则：唯一错误的决定就是“回头”。

**核心原则**：你做的任何一个决定都是对的，只有一个决定是错的，那就是“回头”。

这意味着，一旦做出选择，就必须断绝“试试不行再回来”的退路。我们的行动模式不应留有这种弹性空间。要么不动手，一旦动手，就只有向前一条路。

无论是放弃一个人、一份事业，还是选择一段关系，都应是深思熟虑的结果。在做出决定的那一刻，就要对自己承诺：即使未来某天过得不如现在，也绝不回头。

## 培养负责到底的心态 💪

我们知道了不能回头，那么如何支撑自己走下去呢？这需要培养一种为自己的选择完全负责的心态。

**心理建设**：我们的心理暗示应该是“开弓没有回头箭”。我们需要接纳一个可能性：即未来的结果可能不如现在。

我接纳这种可能性，但我追求的是那个50%变得更好的机会。当我选择去博取那50%的成功概率时，我就必须同时接受另外50%可能不尽如人意的结果。

**负责心态公式**：
```
自主选择 + 承担全部后果 = 负责的人生
```

只有坚持“自己的选择自己买单”，你才能不断磨砺自己，成为一个真正为选择负责的人。这样，你就不会轻易做决定。你的每一次决策都会因此变得更理性，每一次权衡利弊也会变得更清晰。

因为你不再允许自己反复修改答案。

## 对自己提出要求 📝

最后，我们来总结如何实践以上原则。这一切都始于对自己提出明确的要求。

**核心要求**：你必须对自己有要求。这并不是完全禁止你放松或“摆烂”。

以下是两种清晰的路径：

*   **路径一：对自己有要求**。选择这条路径，就意味着践行“不回头”的原则，严肃对待每一个决定。
*   **路径二：彻底放松**。如果你选择“摆烂”，那就彻底放松，同时不要在看到别人过得更好时心生嫉妒。彻底放弃比较的念想，也是一种人生选择。

关键在于，你的行为模式必须一致且坚定，不要在两种状态间摇摆、纠结。
