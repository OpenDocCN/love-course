# 情感关系认知课 01：单偶男与多偶男的识别与区分 👨❤️👨

在本节课中，我们将要学习如何从行为模式和心理动机层面，区分“单偶男”与“多偶男”这两种不同的男性类型。理解这些核心差异，有助于我们更理性地看待情感关系。

上一节我们介绍了课程主题，本节中我们来看看这两种类型的根本定义。评判一个男性属于单偶还是多偶，标准并非只看他某一时刻是否只有一个伴侣。多偶男在生命某个阶段，也可能只拥有一个伴侣。

这是因为当他所掌握的生产资料（如财富、社会资源）不足以维系多个伴侣关系时，他可能暂时呈现出单一伴侣的状态。但这并不改变他内在的多偶特质，以及未来可能的发展方向。

随着他的社会身份提高，掌握的生产资料越来越多，他内在的多偶特质可能会逐渐显现并被实践。

然而，单偶男也不代表他一生只会有一个伴侣。单偶男的核心特征在于，他在某一特定时期，其情感重心只能集中在一个人身上。

他的内心对伴侣有非常明确的主次之分。而多偶男则可以做到没有明确的主次，或者将情感投入平均分配。

多偶男可能对谁都不会投入过深的情感，他认为每个伴侣只能满足他的一部分需求。相比之下，单偶男在每一段时间内，都会有非常清晰的情感倾斜对象。

以下是单偶男的一个关键行为特征：如果他的次要关系影响到了他的主要关系，或者让他感到可能产生影响，他会当机立断地结束次要关系。

此外，单偶男与多偶男最大的区别在于其内心的道德认同感。当他道德上认同多偶行为时，他便是多偶男；当他不认同时，他便是单偶男。

因此，当一个道德上不认同多偶行为的单偶男，去做出多偶行为时，他会产生紧张、慌张、害怕、恐慌等情绪。

他的内心会产生大量的纠结与内耗。这就是单偶男在矛盾状态下的心理特征。需要明确的是，单偶男并非意味着绝对的、无条件的忠诚。

我们的分析模型并非基于不切实际的幻想。在情感关系的观察与实践中，我们应对男性不抱有任何不切实际的妄想。

本节课中我们一起学习了单偶男与多偶男在行为模式、心理动机和道德认同层面的核心区别。理解这些概念是理性分析情感关系的第一步。
