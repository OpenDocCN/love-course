## 核心要点总结

上一节我们剖析了家庭关系中可能存在的“装傻”现象，本节中我们来总结一下关键认知和行为准备。

以下是本节课的核心要点列表：

*   **认清本质**：某些家庭成员可能通过“装傻”和利用“道德感”来维持对你的情感操控，将你视为“摇钱树”。
*   **打破幻想**：你的父母可能并不像你想象的那样脆弱或高尚，他们或许清楚你的付出与牺牲。
*   **预见最坏情况**：最需要做的心理准备，可能是家人被对方说服，反过来劝你维持有害的关系。
*   **核心原则**：人生只有一次，**首要任务是为自己而活**。公式可以表达为：**人生价值 = 自我实现 > 仅为他人牺牲**。
