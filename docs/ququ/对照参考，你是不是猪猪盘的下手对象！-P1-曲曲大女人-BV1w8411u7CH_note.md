## 目标人群四大特征

“杀猪盘”诈骗者主要针对具备以下四种特征的女性下手。

以下是具体的特征描述：

1.  **年龄偏大且急于结婚**：受害者通常年龄较大，并对婚姻感到焦虑。
2.  **树立独立女性人设**：她们对外展示的形象是经济独立、生活自主。
3.  **身边异性朋友较少**：她们的社交圈中异性数量有限。
4.  **情感与经济经验不足**：她们在情感关系和财务处理上缺乏足够经验。

基本上，同时符合上述多条特征的人，就属于“杀猪盘”会优先考虑下手的对象。

## 高风险心态：“我只要爱”

上一节我们介绍了目标人群的客观特征，本节中我们来看看一种高风险的主观心态。

当一位女性在心中描绘出“我自己想要什么生活，我都会努力去争取，我会努力让自己拥有这样的生活，你只给我爱就可以了”的图景时，这种心态极其危险。

你只要流露出“你只给我爱就可以了”的态度，基本上你就成为了“杀猪盘”下手的第一梯队目标。

## 对比：不易受骗的心态

理解了高风险心态后，我们再来看看哪种心态不易受骗。

那些坚持要求男性用行动和付出来证明爱的女孩子，很难被“杀猪盘”得手。她们的原则是：如果你说爱我，那就拿出行动；如果你只是给我“画饼”，那就直接再见。

你可能会问，那骗子怎么得手呢？这类女孩确实不容易受骗。但有些女性则相反，她们自认为很有能力、在寻找真爱、非常优秀，并且觉得自己什么都有了，所以对方“给我爱情就行”。

## 虚假“爱情”的本质

别人承诺给你的那种“爱情”，真的是爱情吗？那不是爱情，那是砒霜。

那是包裹着糖衣的砒霜。为什么呢？因为他只需要给你提供“情绪价值”就足够了。

## 剖析“情绪价值”

那么，情绪价值到底是什么东西？

我们在亲密关系课程中，经常教授如何提供情绪价值。情绪价值在于，在特定的时间点，用特定的方式表达，以达到最佳效果。

**情绪价值 ≈ 言语上的付出（上嘴皮碰下嘴皮）**

这就是情绪价值，它几乎没有成本。你以为它有什么成本？唯一的成本是学习和研究人性的成本。

它并非从口袋里掏出来的实实在在的成本。相比之下，我每天起早贪黑挣来的钱，花在你身上，这才是实实在在的成本。

我们学习如何聊天、学习如何提供情绪价值，这些东西能算作成本吗？显然不同。正如我说过的，在我们这里学习亲密关系，花费两千多，但学到的技能可以受用一辈子，一辈子的情绪价值都够用了，管饱。
