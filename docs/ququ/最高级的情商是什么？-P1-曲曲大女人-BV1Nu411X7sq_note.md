# 情商进阶课 01：最高级的情商的本质 🧠

在本节课中，我们将要学习一种被称为“最高级情商”的沟通思维模式。这种模式的核心并非技巧，而是一种看待他人和世界的根本态度。掌握它，不仅能让你的人际关系更加和谐，还能让你在复杂的社会互动中始终立于不败之地。

## 怜悯之心：高级情商的基石

最高级的情商一定是对世界万物的怜悯之心。

上一节我们介绍了高级情商的核心是一种态度，本节中我们来看看这种态度的具体表现。它要求我们在评价他人时，能够超越简单的道德评判，转而尝试理解行为背后的成因。

## 从评判到理解：沟通模式的转变

当你与他人讨论第三方时，话语的本质可能是在谈论他人的缺点。但由于你秉持怜悯的态度，便可以将对话转变为你们两人共同以慈悲为怀的立场去同情他人。

以下是这种转变的关键步骤：
*   你能够分析出这个人自私的原因。
*   你不再局限于评价他自私这个事实本身。

这种沟通方式既陈述了事实，也并非夸奖对方，而仅仅是展现了你的包容性。

## 建立联盟：高级立场的吸引力

当你从一个怜悯的出发点，展现出对所有事情分析的诚意时，与你对话的人会觉得你的出发点很高级。

他会因此想要加入你的这个出发点，因为他想显得自己同样高级，这是人性使然。他会觉得，在你们的对话中，你们两人是共同高于被讨论者的。

## 化解风险：不留把柄的智慧

在对话中，你并没有否认那个人可恨的事实，但你告诉对方，这种可恨可能是情有可原的。

以下是可能的原因分析：
*   可能来自于他的原生家庭。
*   可能来自于他的某段特殊经历。

于是，你在认可其行为可恨的同时，理解了他的可恨。当对方加入你的这种探讨范畴时，意味着你们两人共同达到了一个更高的理解层次。

通过这种方式，你把一起讨论的同伴拉到了另一个高度。这种情况下，你绝不会给自己留下话柄或后患。未来流传出来的关于你的评价都会是正面的。

## 赢得人心：甚至获得当事人的认同

即使是被你们讨论的那个人听到了这番话，他也会觉得：“你这个人确实懂我，你和他们不一样。别人只看到了我的缺点，而你却理解了我。”

这是一个完整、自洽的逻辑闭环。

本节课中我们一起学习了最高级情商的本质——以怜悯之心作为沟通的出发点。我们探讨了如何将对他人的评判转化为理解，如何通过建立高级立场吸引盟友，以及最终如何实现不留隐患、甚至赢得当事人认同的沟通效果。这套完整的思维模式，是构建深度、稳固人际关系的强大基石。
