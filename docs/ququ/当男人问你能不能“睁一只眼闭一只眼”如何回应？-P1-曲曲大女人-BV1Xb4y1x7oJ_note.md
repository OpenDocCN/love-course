在本节课中，我们将探讨一个在亲密关系中可能遇到的棘手问题：当伴侣明确提出希望你对他未来的某些行为“睁一只眼闭一只眼”时，应如何理解和回应。我们将从个人经历出发，分析其背后的逻辑，并提供一个成熟、基于现实的应对框架。

多年前，当我的伴侣提出“如果我们结婚，你能不能做到睁一只眼闭一只眼”时，我认为这绝对是一个不该被摆上台面讨论的问题。当时，我们两人在价值观、底层逻辑、为人处事和生活目标上都非常一致。

然而，他提出了这个问题。当时的我感到很可惜。我可惜的并非他是这样的人，而是觉得：如果你不把这件事明说出来，我们或许就成了，我也许可以默认这种情况。

但当你把它拿到台面上来谈时，这就不是要求我“睁一只眼闭一只眼”了，而是明确告知我你的意图。

现在，如果我再遇到一个各方面都与我非常契合的人，并问我同样的问题，我的回应方式会与当初不同。我会告诉他：“你要我睁一只眼闭一只眼？不需要，我会把两只眼睛都闭上。”

我现在明白了一个道理：一个如此优质、各方面都与你非常契合的人，本身就很难遇到。要求这样的人在当下承诺未来的专一与忠诚，是无效的。即使他当下承诺绝对忠诚和专一，未来依然可能出问题，依然可能做不到。

说白了，你争取到的是什么呢？你只是争取到了一张“自己是正牌伴侣”的门票。什么是爱？爱是**因为我爱你，而被迫放弃了其他选择**。这是爱带来的自我约束。

如果在相处过程中，你们在精神、身体、资源和社会人脉上都充分占有彼此，那么他可能就很难再有多余的精力、时间和空间去发展其他关系。他可能做不到100%的纯粹，但也许能做到90%。

那时我们不禁会想：你为什么非要挑明这件事呢？说实话，是因为我内心还想自欺欺人。当我明白了成年人之间的游戏规则，明白了优质阶层会面临的诱惑后，我就不应该再自欺欺人。

我为什么要自欺欺人？如果我有能力、有能量，能搞定你的情绪、资源、生活圈和人脉圈，我们两人就是并肩作战的战友。当我能看到并做到这一切时，你自然会衡量背叛我的成本。这是人性。

忠诚不来源于要求。要求对方忠诚，是世界上最愚蠢的事。人的忠诚来源于**利益的捆绑**。你给了我足够的利益，让我不得不忠诚，这才是人性的底层逻辑。而不是因为道德要求而忠诚，那是不切实际的。

我不应在这件事上抱有侥幸心理来自欺欺人。这样，我的生活能避免很多痛苦。我与自己和解，也与他人和解。

如果你做得到（提供足够的利益捆绑），他就会忠诚。如果你做不到，那你就要为结果买单。如果我在情绪价值、能量或资源上无法匹配你，当你在这些缺口上寻求其他补给时，我认账。这才是一个成熟的人。
