# 掌控欲望，方能掌控人生！ - 课程 P1 🧠

在本节课中，我们将学习情绪管理与自我控制的核心原理。我们将探讨如何通过对抗即时欲望来建立情绪稳定性和人生掌控感，并理解立即行动对于维护个人框架与核心利益的重要性。

## 情绪不稳定的根源

上一节我们介绍了课程主题，本节中我们来看看情绪管理能力不佳的根本原因。

情绪管理能力不佳，是因为从未与自身欲望进行对抗。即欲望向左时，行动应向右；欲望向前时，行动应向后。必须永远与自身欲望相抗衡。

例如分手后的断联。求和是欲望，但断联是行动。行动与欲望方向相反。在此过程中，模拟出的是自我控制与情绪稳定。这需要从每一件小事开始。

## 对抗欲望与建立掌控感

理解了根源后，我们来看看如何通过日常练习来培养这种对抗能力。

没有在小事上与自己对抗的人，在需要斩断关系时，内心只有痛感。一想到剥离关系，就只有驱使他再次寻找对方的冲动性痛感。

因此，有些人的人生越往后走，越觉得受控，情绪越稳定。而有些人则越觉失控、不稳定，行事越冲动。

原因在于后者不断追逐短暂解脱，只为避免痛苦。痛感产生时，他们只想消磨它。他们不断追逐的是消灭痛感这件事。

所以他们没有与自己的心性做对抗。这方面的“心顿感”永远无法建立。当别人内心升起两股力量（欲望与克制）时，你缺少一股力量。因此在这个过程中，永远找不到成就感。

## 成就感的积累与情绪自律

那么，这种对抗欲望的成就感从何而来呢？本节我们将探讨其积累过程。

成就感源于很小的事情。逐渐地，你会发现对很多事情的掌控力越来越强，尤其是对自己情绪的掌控。这就是情绪自律的本质。

## 维护核心利益与立即行动

建立了自律基础后，关键在于如何将其应用于维护个人核心利益。本节将讨论立即行动的重要性。

现在如果有一个男孩子，相处再愉快，但如果他触及我的核心利益，我就必须选择斩断关系。无论见面频率是每天、每周还是每月，都不重要。

我不会想“虽然明白不能继续，但反正见面少，就等下次他找我时再不理”。我会在当下做决定的瞬间，把事情全部做完，该说的话全部说清楚，不给自己留回头机会。

因为时间会磨平情绪。当下能冷静看清这是个错误，但过一两个月后，可能就忘了。等一两个月后对方再来找你时，你会觉得“也没多大个事，相处一下也没什么”。因为你已经忘记当初准确的评估。

## 脱离他人节奏，建立自我框架

拖延行动会导致怎样的后果？本节将分析如何避免被他人带入节奏，从而建立自己的框架。

你们是如何不断被一个人套进他的节奏里的？当自己做了冷静理智的分析后，没有立即行动，而是在等。为什么等？因为当下做决定让你难受，所以就想拖延，认为没有即时影响。

但不可以这样。很多女孩子和男人分手，拖几年分不明白，正是如此。因为说出口时痛苦、难受，所以就拖着，等对方做决断。但问题在于，涉及核心利益损失的是你，不是他，他为什么要做决断？他不会的。

也就是说，只要他不做决断，你就永远做不了决断。只要他一两个月后再找你，你就会忘记当初的决断。这导致你永远跟着对方的节奏，永远没有自己的框架和节奏。

## 实践与长期修炼

最后，我们需要思考如何将今天的理论付诸实践，并理解这是一个长期修炼的过程。

今天讲的内容，需要大家自己琢磨。因为这并非一两天能做到，而是长年累月对万事万物理解的结果。

希望大家能理解这件事，并在未来的生活中去琢磨、实践，看看自己能做到什么程度。
