# 男人喝多了能信吗？ - 课程P1 🍷

在本节课中，我们将要学习一个重要的情感认知课题：如何理性看待男性在醉酒状态下的言行。我们将深入剖析“酒后吐真言”这一常见说法的真实性，并理解其背后的行为逻辑。

## 核心观点：不存在“酒后吐真言” 🚫

酒后吐真言是一种被浪漫化的想象。男性在喝多后说的话，一句都不要相信。所有在酒精影响下说出的话，本质上都是不负责任的胡言乱语。

## 醉酒言行的本质与动机 🔍

上一节我们明确了核心观点，本节中我们来看看男性醉酒后行为的具体动机。

酒后言行全部是为了“喝了就忘”和“失忆”做铺垫，其根本目的是为自己后续的不负责任行为预留退路和借口。

首先，我们需要确立一个基本观念：男性的思维模式在醉酒后会发生变化。当他因酒精而情绪亢奋、想要发泄或联系他人时，他不敢去打扰自己内心真正最爱、最敬仰或最想保护的那个人。因为他害怕破坏自己在那位“女神”心目中的完美形象。

## 行为转移理论：FIDS模型的应用 🎯

基于上述心理，男性的行为会发生转移。以下是其典型的行动逻辑：

他会转换目标，去找那个更容易“拿捏”、平时对他表现出明显兴趣的“傻白甜”或“恋爱脑”。他不敢对女神说的肉麻话、做的冲动事，会全部转移到这个替代目标身上。

**核心逻辑公式**：`不敢打扰女神 -> 寻找低风险目标（易拿捏/感兴趣者）-> 转移情感宣泄`

请相信这个结论。也许存在万分之一“酒后吐真言”的极端个例，但你也无需将其当真。

## 接受醉酒表白的潜在风险 ⚠️

如果你允许一个男性在非清醒状态下向你表白或推进关系，那么他将失去在清醒时鼓起勇气向你正式表白的动力。

试想，如果他喝醉时说的情话、做的承诺你都能全盘接受并感到受用，他还有什么必要在理智、清醒、需要为自己言行负全责的状态下，再来重复这些事呢？

这实质上是在**降低你自身的感情门槛和标准**。你值得被认真、清醒地对待。

## 唯一例外：酒后转账 💰

那么，是否存在可以接受的醉酒行为呢？有，且仅有一项：**酒后转账**。

这个行为是具体、可验证且具有实际意义的，与其他空洞的言语承诺有本质区别。
