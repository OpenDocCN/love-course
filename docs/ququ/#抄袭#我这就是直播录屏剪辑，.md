# 课程名称：优势人脉链接与称谓策略 🧠  
## 课程编号：001  

在本节课中，我们将要学习如何通过恰当的称谓与优势人脉建立有效链接。我们将分析具体案例，理解称谓背后的社交策略，并掌握核心原则。  

---

上一节我们介绍了课程主题，本节中我们来看看案例中的具体表现。  

视频中的人物一上来就喊了一句“小天”。很多人认为这只是亲切的称呼。  

什么是与优势人脉链接的核心之一？  

当你认为与对方有七分关系时，要用好像只有五分关系的姿态说话。  

以下是案例中的具体分析：  

*   第一点：你与你老公都不配提出姓名。“我不是小天吗？”  
*   第二点：你在我这里不配拥有姓名。你是靠你老公获得关注。  
*   第三点：这些称谓并非随意，而是经过思考。花费15分钟的原因就是为强调“强东”这个称呼。  

整个场景中，每一个称谓都经过斟酌。  

他想传达的是：“我与强东代表平等关系。”  

这种关系并非平等。  

一个像奶茶妹妹一样年轻、家境优渥、高学历、高颜值的女生，都在不断进步。  

如果你有一天能像奶茶妹妹一样成功，那么……  

这是哪位博主？这次抄得很好，下次别抄了。  

---

本节课中我们一起学习了优势人脉链接中称谓使用的策略。核心在于通过刻意保持略低于实际关系的称谓姿态，来维护与优势人脉的链接，并理解每个称谓背后传递的社交信号。
