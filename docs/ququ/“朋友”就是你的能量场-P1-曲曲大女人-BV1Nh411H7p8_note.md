# 人际关系管理课 P1：建立社交门槛，守护你的能量场 🛡️

在本节课中，我们将要学习一个核心的人际关系管理概念：为所有社交关系设置明确的门槛。这能帮助你有效筛选交往对象，保护自身能量，避免被低质量关系消耗。

## 人际关系需要设置门槛

在我们所有的人际关系中，存在多种类型，例如普通朋友、亲密朋友、PY、男女朋友、情人等。对于任何一种关系，你都必须为其设立明确的准入门槛。

**核心概念**：`关系质量 = 交往对象质量`。如果你的社交圈缺乏筛选，结果就是与低质量的人为伍。

---

## 低质量关系的危害

上一节我们提到了设置门槛的必要性，本节中我们来看看，如果不这样做会带来哪些具体危害。

如果你不为你身边所有的关系都设置门槛，你就会与一些“垃圾”做朋友。你也会被迫与这些低质量的人相处。

这些人会破坏你的气场，损耗你的能量。你或许立志要提升自我，但围绕着一群“垃圾”，你如何能提升自己？你或许认为“我只是和他做朋友而已”，但问题在于，这个人本身就不值得你付出友谊。

---

## 如何定义“不及格”的朋友

那么，什么样的人可以被定义为“不及格”、达不到朋友门槛呢？以下是一些关键特征：

以下是“不及格”朋友的一些典型表现：

*   **缺乏责任心**：一个不负责任、没有担当的人。
*   **无法提供价值**：思考一下，与他交朋友你能获得什么？是情绪消耗多于价值回馈吗？

如果一个人连做朋友的基本资格都不具备，他就不应进入你的核心社交圈。

## 课程总结

本节课中，我们一起学习了人际关系管理的第一课：**为所有社交关系设立门槛**。关键在于主动筛选交往对象，远离那些会破坏你能量和气场的低质量关系，从而为自己创造一个积极、健康的成长环境。记住，你的社交圈质量，直接决定了你个人成长的速度与高度。
