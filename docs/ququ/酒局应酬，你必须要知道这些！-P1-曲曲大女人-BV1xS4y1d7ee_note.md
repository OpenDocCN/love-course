# 酒局应酬核心法则 🍷 - 课程01

在本节课中，我们将要学习在商业与社交应酬中识别环境、保护自己并理解人性复杂面的核心法则。这些知识对于初入社会的年轻人，尤其是需要在酒局等场合与人打交道的朋友至关重要。

## 理解人性的两面性 🌓

上一节我们概述了课程目标，本节中我们来看看人性的复杂面貌。

不论在生意场还是官场，但凡能做出成绩的男性，都是一路求人过来的。他们的内心都有很扭曲的一面。内心都有一面是很压抑、很扭曲的。但是，越是在外面，他们越要包装成正人君子。

越是在外面，他们越要包装自己的慈善人设或其他人设。他们有一些东西就无法去释放，一定会找机会释放。因此，与人交往要学会“看人下菜碟”。

以下是核心要点：
*   如果你认识的是他们“阳光下”的一面，就不要试图揭露他们“月光下”的面貌。
*   在不同的环境、氛围、圈层以及不同的引荐人介绍下，你认识到的同一个人的面貌可能完全不一样。

## 自我认知与成长路径 📈

理解了人性的多面性后，我们需要审视自身，明确成长路径。

我知道自己有时脸皮比较薄。但是，我想要成长。成长需要在正确的人的指引下进行。

以下是需要警惕的“错误指引”：
*   社会上有些人属于“社会人”，他们不按规则出牌（不讲武德）。
*   与这类人交往，你请他们办事后，他们可能还会对你动手动脚。
*   如果第一个动手动脚没搞成，可能还会有第二个，甚至直接对喝醉的你下手。

面对这样的圈子，你需要自问：这是什么稀巴烂的圈子？有什么好结交的？你喜欢跟垃圾打交道吗？你跟垃圾讲什么面子？

## 提升段位与建立框架 🛡️

如果你发现自己容易陷入上述困境，根本原因在于自身段位不足、框架不强。

你现在整个段位太低了。你现在出来做生意，就是被人吃干抹净，最后骨头都不剩。生意没做成，人还可能被欺负，最后一场空。

你现在应该多学习，好好从基层开始。你有点好高骛远。即使你计划30岁家庭稳定后再思考这些复杂关系，但问题在于：你现在没有处理这些关系的能力。

面对社会上的乱七八糟的规则、人和稀烂的单子时，你自身的“内核”太弱，是一个没有框架的人。

**核心公式：自身框架弱 = 被他人随意践踏原则**

因为他们会随意践踏你的框架和原则。你没有框架，就等于任人摆布、任人拿捏。你不应该让自己处于这种境地。你现在没有这个段位，操作不了这个事。

## 关键策略：借力与保护 👥

既然独自应对风险很高，那么正确的策略就是寻求借力与保护。

如果你必须跟这些人打交道，要带着真正厉害的人一起去。不要自己单枪匹马地出去，你没有这个能耐。再怎么样，都要有人带你。没有人带，没有人告诉你游戏规则，你到时候就是个傻逼，跟他们一起玩，什么都搞不到。

以下是寻求“正确指引”的要点：
*   **警惕错误带领**：不能让那些“稀巴烂”的人带你。
*   **寻找正确保护者**：你得找能对付这些人的人，比如家里信得过的长辈或前辈。
*   **行动准则**：但凡要出去跟这些人应酬，不要自己去。否则，对你动手动脚的人可能只是第一个“动筷子”的，后面还多得很。

很多东西叫“言传身教”。言传身教是要让你看、让你跟，不是光听家里人或我说两句就有用的。关键在于，**你需要有人保护**。

## 核心认知总结 💎

本节课中，我们一起学习了酒局应酬中的核心生存法则。

我认知中所有的男人都有两面：白天一个样，晚上一个样。阳光下面一个样，月光下面一个样。有很多生意是你们永远看不见的。如果你们自己不出来做生意，不跟这些人在生意上打交道，你们永远看不到这一面。

**最终总结**：
1.  认清人性在公开场合与私密场合的两面性。
2.  客观评估自身实力，避免进入远超自身段位的局。
3.  **建立并坚守自己的原则和框架**。
4.  在复杂环境中，**务必借助可信赖的、有经验的前辈力量**来保护自己和学习规则，切勿单打独斗。
