# 课程一：控制情绪（一）🚫 不与生理做搏斗

在本节课中，我们将要学习情绪管理的一个重要基础原则：避免在生理状态不稳定时做出重大决定或采取行动。我们将探讨生理因素如何影响我们的情绪与判断，并提供具体的实践方法。

## 生理状态如何影响情绪与决策

上一节我们介绍了课程主题，本节中我们来看看生理状态与情绪之间的具体联系。人的情绪和决策能力深受体内激素水平、疲劳程度等生理因素的影响。当生理状态失衡时，我们的认知和情绪也会随之波动。

例如，在深夜、酒后或女性生理期等时段，体内的激素水平与正常状态不同。这种生理变化会直接导致情绪敏感、思维悲观或冲动，使人容易做出清醒后后悔的决定。

## 常见的冲动行为场景

以下是几个典型的、因生理状态影响而容易产生冲动行为的场景：

*   **深夜时分**：在深夜，尤其是11点之后，人容易感到孤独、感伤，产生强烈的倾诉或联系他人的冲动。
*   **饮酒之后**：酒精会抑制大脑的高级认知功能，降低自制力，使人做出平时不会做的行为，如拨打冲动电话。
*   **生理周期期间**：激素的周期性变化可能带来情绪波动，使人更容易陷入自我贬低或伤感的情绪中。

## 核心原则：不与生理搏斗

理解了生理对情绪的影响后，我们来看本节课的核心原则。试图在生理状态低谷期用意志力强行控制情绪或做出“正确”决定，就像逆水行舟，异常困难且往往失败。

因此，**最有效的策略不是“对抗”，而是“规避”**。核心行动公式可以概括为：

**行动 = 识别生理低谷期 + 主动暂停 + 延迟决策**

这意味着，当你意识到自己正处于容易冲动的生理状态时，首要任务不是思考“我该不该发这条信息”，而是“我现在应该停止任何决定，去睡觉或休息”。

## 具体实践方法

以下是可供操作的实践步骤，帮助你在关键时刻平稳度过情绪波动期：

1.  **设定行为截止时间**：严格规定自己在晚上11点前关闭手机或电子设备，切断深夜冲动的渠道。
2.  **建立“延迟”机制**：当产生强烈的联系某人或发布动态的冲动时，告诉自己：“一切决定等到明天早上再说。”
3.  **观察并记录**：第二天清晨，回顾前一天晚上的情绪和想法，你会发现大多数强烈的感受已自然消退，这能强化你对本原则的信心。

## 课程总结

本节课中我们一起学习了情绪管理的第一课：不与生理做搏斗。我们认识到，深夜、酒后等生理状态不稳定时期，我们的判断力会下降，容易做出冲动的、事后后悔的行为。关键在于识别这些生理信号，并采取“主动暂停、延迟决策”的策略，而非强行用意志力对抗。记住，太阳升起后，许多深夜里的“刻骨铭心”都会变得云淡风轻。
