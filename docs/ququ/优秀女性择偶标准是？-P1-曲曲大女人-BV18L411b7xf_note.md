在本节课中，我们将要学习如何从经济学和现实供需的角度，理性分析择偶市场中的不同选择标准及其可行性。

上一节我们介绍了课程主题，本节中我们来看看高收入择偶标准的现实挑战。

年收入一两百万的人比年收入过亿的人还难找。原因在于，寻找年收入一两百万的男性时，通常附加了多项高标准。首先，他的外貌不能太差，你需要觉得顺眼。其次，他的年龄不能太老。因为如果年龄偏大，你对他的经济要求会相应提高。

因此，你所寻找的所谓年收入一两百万的对象，是年龄、长相、身高和收入都处于较高基准线上的单身男性。这样的单身男性在整个相亲市场中，属于“一放进去就没了”的稀缺资源。

明白了上述难点后，我们来看看可行的调整策略。

以下是几种可行的择偶策略调整方向：
*   **提高经济容忍度，放宽年龄限制**：寻找收入过亿但年龄更大的对象，相对更容易。
*   **降低经济要求**：寻找年收入几十万的“百里挑一”的经理，数量会多很多。
*   **转变核心评判标准**：不以收入为唯一核心标准。

关于第三种策略，即转变评判标准，我们需要进一步展开。

如果不以收入作为核心评判标准，可以关注对方的职业属性与发展潜力。例如，医生、律师、教授、程序员等职业，其工作性质良好。关键在于，他在某一专业领域的能力能让你仰视，或者能对你的成长形成补给作用。这样的人是值得考虑的。

针对调整标准可能带来的顾虑，这里有一些分析和建议。

你担心降低经济要求后，时间久了男方心里可能不服。实际情况并非如此。市场上依然存在许多年轻英俊的男性，他们希望利用年龄和外貌优势，寻找经济实力较强、也较年轻漂亮的女性。

你可能会顾虑：“他要是花着我的钱开着我的车还去找小姐姐，那我心里不好想。” 应对之道在于，你可以像萧亚轩一样拥有随时换人的底气与资本。同时，他们也需要衡量犯错（即背叛）所需付出的成本。

本节课中我们一起学习了如何从供需关系分析择偶标准的可行性，并探讨了三种主要的策略调整方向：**提高经济容忍度**、**降低经济要求**、**转变核心评判标准（如关注职业与专业能力）**。关键在于认清市场现实，明确自身核心需求，并保持选择的主动权。
