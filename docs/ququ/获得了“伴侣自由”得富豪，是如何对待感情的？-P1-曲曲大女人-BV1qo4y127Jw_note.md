## “伴侣自由”的核心逻辑 🔑

通过上述案例，我们可以提炼出“伴侣自由”并非指随意行事，而是建立在**明确规则与共识**之上。其核心公式可概括为：

**伴侣自由 = 知情同意 + 规则前置 + 边界清晰**

这意味着，在进入一段主要关系（如婚姻）之前，所有相关方都对其他关系的存在、性质和时间安排有清晰的了解并达成一致，而非事后隐瞒或造成伤害。
