# 课程：目的导向思维 🎯 - P1

在本节课中，我们将要学习“目的导向”这一核心思维方式。我们将探讨目的导向人群的特征、行为模式，以及他们如何做出决策。通过理解这一概念，你可以更清晰地认识自己与他人的思维差异，从而在沟通和决策中更加高效。

## 理解目的导向人群 🤔

上一节我们介绍了课程的主题，本节中我们来看看什么是目的导向人群。目的导向的人以目标、价值和利益为核心驱动力。他们的行为和决策始终围绕“能否带来实际价值”这一标准展开。

我之前参加奇葩说培训时，明白了一个道理：世界上存在许多不以目的为导向的人。他们并非不懂或不会，而是认为没有必要这样做。他们不追求利益，也不以目的为导向。

这类人纯粹追求精神价值。这种取向没有对错之分，只是与我不同。世界上有一部分人像我一样，以目的为导向，以价值为导向，以利益为导向。我就是这样一个人。

## 目的导向的沟通方式 💬

上一节我们了解了目的导向人群的特征，本节中我们来看看他们如何沟通。当你与我沟通时，我们会本能地朝着这个方向探讨。我们会自然地认为，如果一个人或一件事没有价值，不能带来成长或提升，那么这个人对我的生活来说就没有意义。

以下是目的导向人群在沟通中的典型思考方式：

*   评估对方或事件是否能带来实际价值
*   判断是否有助于个人成长或提升
*   快速过滤掉无意义或低价值的信息

## 一次性杯子的比喻 🥤

上一节我们探讨了目的导向的沟通方式，本节中我们通过一个比喻来加深理解。对于目的导向者而言，许多事情就像一个**一次性杯子**。

我家里可能有很多陶瓷杯、漂亮的杯子。但这个一次性杯子，本来就是可有可无的东西。以我的性格，我不会站在货架前比较：这个一次性杯子是绿的，那个是粉的，这个有花，那个有草，哪个更漂亮、更符合我家装修风格。我不会在一次性杯子上耽误时间，消耗我的**算力**（指心智资源和精力）。

## 目的导向的应用与局限 ⚖️

上一节我们通过比喻理解了目的导向者的决策过程，本节中我们来看看它的应用与局限。需要明确的是，并非每个人都是目的导向者，也并非每个博主都是。但我是这样的博主，因此我只能解决与我同类型导向的人和问题。

你的问题是否是价值导向、利益导向？我们就站在结果的角度去探讨。当你们拿着一个“一次性杯子”式的问题来找我时，我确实会感到困惑。因为我无法理解为什么会有人提出这样的问题，甚至你提问的出发点都让我费解。

我觉得这东西为什么要思考？随便选一个不就行了吗？如果你跑来跟我连麦，还花了钱，想听取我有智慧的建议，我自己都会觉得在讲废话。比如我跟你说“喜欢就在一起，不喜欢就不在一起”、“开心就在一起，不开心就不在一起”。这种话还需要我来讲吗？

---

本节课中我们一起学习了“目的导向思维”的核心内涵。我们认识到，目的导向者以**价值**和**结果**为决策核心，会本能地过滤低价值信息（如同忽略一次性杯子的花色）。这种思维模式追求高效，但在面对不同思维模式的人时可能产生理解鸿沟。关键在于认识到思维差异的存在，并在沟通中明确彼此的导向。
