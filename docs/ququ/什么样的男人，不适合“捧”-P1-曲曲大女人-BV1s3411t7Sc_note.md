# 情感关系课 01：什么样的男人不适合“捧”？🤔

在本节课中，我们将探讨一个情感关系中的常见误区：盲目地“捧”或崇拜伴侣。我们将分析不同类型的人在被“捧”时的不同心理反应，并强调保持稳定自我认知的重要性。

## 内核不稳定的男人

上一节我们介绍了课程主题，本节中我们来看看第一种不适合被“捧”的男人类型。这类男性通常阅历较浅，自我认知和内在核心不稳定。

如果女性伴侣持续地崇拜他、称赞他，他会很快产生一种错觉，认为对方已经配不上自己，自己理应匹配更优秀的伴侣。

特别是当他接触到更广阔的世界、见识到更多之后，这种“我值得更好”的想法会愈发强烈。

## 内核强大的男人

与内核不稳定的男性相反，我们来看看那些自身条件优秀、内在强大的男性。对于这类男性，伴侣的欣赏和崇拜是一种情绪价值的提供。

在一个稳定内核的基础上，他能够识别并感激这种价值，从而给予伴侣积极、正向的反馈。这与条件一般的男性反应截然不同。

## 女性的类似处境

以上我们分析了男性的情况，类似的动态也可能发生在女性身上。如果一个自身普通的女性，被一位强大且愿意支持她的男性长期“捧”着。

这位男性甚至愿意为她赋能，给予她长期的正向反馈，这位女性也可能逐渐“飘”起来。

她会开始认为自己配得上更好、更顶尖的伴侣或生活，从而想去追逐所谓“更好”的。但结果往往是得不偿失。

因为她没有意识到，别人的捧场源于**对方自身的价值与稳定内核**，而非她自己有多么非凡。她只是在别人这面“镜子”里，显得很厉害罢了。

## 核心原则：寻求真实的自我

基于以上分析，我们可以总结出一个核心原则。在任何时候，无论面对他人的吹捧还是打压，关键在于穿透语言表象，**寻求真实的自我**。

你需要客观评估自己的真实分量。既不能因他人打压而妄自菲薄，也不能因他人吹捧就忘乎所以。

保持稳定的情绪状态，进行深刻的自我剖析，是至关重要的。其核心逻辑可以概括为：

**真实自我价值 = 稳定内核 + 客观自我认知**

否则，你很容易在正确的亲密关系中，因自我膨胀而放弃它，转而去追求错误的关系；或者深陷于一段错误的关系中无法自拔。

---

本节课中我们一起学习了“捧”这个行为在不同人身上产生的不同效果。关键在于识别自己和伴侣的“内核”是否稳定，并始终穿透外界评价的迷雾，寻求并坚守真实的自我认知。
