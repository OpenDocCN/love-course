# 课程：情感策略分析 - 第1课：林月云的策略解析 🧠

在本节课中，我们将深入分析被称为“宇宙最强第三者”的林月云所使用的情感策略与方式方法。我们将拆解其核心行为模式，并探讨这些策略如何影响她的人际关系与生活轨迹。

大家知道在我眼中的宇宙最强第三者是谁吗？这个称号属于林月云。林月云是谁可能很多人不知道。如果说侯佩岑的妈妈，你们大概就心里有数。在视频最后我会细致拆解。

这位宇宙最强第三者到底使用什么样的策略与方式方法。你们知道侯佩岑还有一个外号叫做假笑女王。在整个娱乐圈她是公认的情商高。她这样常年把笑容挂在脸上的习惯。

以及她的高情商都源于她从小就受到别人的指指点点。因为妈妈是第三者，她必须知道谁才是自己的老板。一定要哄好自己的老板才能保证自己从小到大像贵族一样的生活。

这个老板也就是她后来叫做爸爸的人。你们看侯佩岑，她过往交往的所有男性不仅仅是质量很高，而且都对她念念不忘。这都少不了她妈妈的言传身教。这里面包括了天王周杰伦和富商连胜文。

说完侯佩岑以后，我们来说一说她的师傅，也就是她的妈妈林月云到底为什么称之为宇宙最强第三者。她当初怀孕是跟自己闺蜜的老公侯世宏。结果她又挺着大肚子搞定了她另外一个闺蜜的老公。

也就是后来侯佩岑叫爸爸的这个男人邱家雄。两个被她挖了墙角的闺蜜都被气得吐血，一辈子都守着自己的婚姻。最终都没有能够让侯佩岑的妈妈上位。但是邱家雄不仅仅愿意帮林月云养孩子，而且对侯佩岑也是视如己出。

跟他们娘俩一直生活在一起。直到自己最后要病逝了，都把所有的财产留给了他们娘俩。让自己原来的妻子过来照顾自己，直到去世然后帮她来办葬礼。大家一定会非常好奇。

林月云到底是怎么能够让两个男人都这么的为她疯狂。甚至怀着别人的孩子都没有办法阻拦这个男人对她的爱的欲望。这里就要提到林月云的一句名言了。
