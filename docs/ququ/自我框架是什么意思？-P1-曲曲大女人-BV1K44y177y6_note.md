在本节课中，我们将要学习“自我框架”这一核心概念。自我框架是你个人原则、底线和价值观的集合，它决定了你如何与他人互动，并保护你免受伤害。建立稳固的自我框架，是维持所有健康、长远人际关系的基础。

## 什么是自我框架？🤔

自我框架可以被理解为你的个人边界和原则体系。它就像你内心的指南针，指导你如何行动、如何回应他人，以及你允许别人如何对待你。

如果说你想要跟所有人的关系处得长远，一定要把自己的框架拧好。如果你不把自己的框架拧好，你会发现你越容忍越退让，到最后就越被欺压。

## 自我框架的双重核心：欣赏与不信任 👥

我的价值观是：**我欣赏每一个人，但我不信任任何人**。我也是真的就是这么做的。这个理念包含两个相辅相成的部分。

### 1. 学会欣赏他人

因为如果你不欣赏别人，那么你这个人就会活在自己的世界里。你作为一个社会人，你跟别人的关系处不好，你没有办法在这个社会里面融入进去，因为你不懂得欣赏别人。你要懂得欣赏别人。

以下是欣赏他人的重要性：
*   别人能愉悦。
*   别人也会来欣赏你。
*   这是双向的愉悦。

### 2. 保持理性的不信任

但是你不能信任任何人。因为如果你对一个人是无底线的信任的时候，你一定会被背叛。这个事情没有特例。

人是尊重什么你知道吗？所有的人他都是尊重有原则、有底线、有框架的人。当你无原则、无底线、无框架的时候，他一定会想办法利用你。这个就是人性，因为你好被利用。

只有你自己有了自己一个非常完整框架的时候，人家他在想要去背叛你之前，他在想去利用你之前，他才会多动一下脑子。他需要去衡量一个成本：

`背叛成本 = 可能失去的关系 + 可能遭受的报复 + 个人声誉损失`

他要去衡量他的成本，背叛的成本，所有的成本他需要在心里去衡量一点。那么这个时候他才会选择忠诚。

如果说你对这个人是百分之百的、百分之一万的这样的一种信任状态，这个人是一定会背叛你的，就看打不打得到那个利益点上。这个东西无关性别，但这个就是人性底层的一个东西。

## 建立框架的目的：为了关系的健康与长久 💝

所以为什么我老是在跟你们说，我们要不断地在关系当中去建立自己的框架。为什么我们要把自己的框架稳住？

不是因为我不爱，也不是因为我想要压制你。正是因为我希望我们的关系能够走得长久，能够走得健康，能够让彼此都进步。

所以说我在这个关系当中，我更要有自我框架。我不能激发你人性当中的那个恶。我希望我的框架，能够让你永远都待在善的那一面。这就是我要做的事情。

并不是说我不想跟你好好在一起。相反，我正是因为想跟你好好在一起，那么我一定要把这个框架竖好。

## 自我框架的普适性 🌐

这个原则适用于所有类型的人际关系。

我们跟朋友、跟闺蜜也是一模一样的，在单位里头是一模一样的。无论是在亲密关系、友情还是职场中，稳固的自我框架都是你赢得尊重、保护自己并维持健康互动的基石。

本节课中我们一起学习了“自我框架”的含义与重要性。我们了解到，自我框架是一套结合了“欣赏他人”与“保持理性不信任”的个人原则体系。建立并坚守自己的框架，并非出于冷漠或控制，而是为了抑制他人人性中的“恶”，引导关系向“善”发展，从而维护所有社会关系的健康与长久。记住，**人们尊重的是有原则、有底线的人**。
