# 课程一：目标选择策略分析 🎯

在本节课中，我们将分析一个具体案例，探讨个体在实现特定目标时，如何基于对目标对象的深度剖析来制定策略。我们将以韦雪选择秦奋这一事件为例，拆解其背后的决策逻辑。

## 案例背景与核心问题

大家都知道韦雪。她搞定了知名富二代秦奋，生了一个娃并获得了巨额财富。但韦雪为什么会选中秦公子？她如何断定生了秦公子的孩子，就能获得巨大利益？

许多女性都试图通过与富有男性建立关系来改变命运，但结局往往相差甚远。例如吴绮莉，成龙始终未承认其孩子，也未提供经济支持。但韦雪却能凭借一个孩子实现阶层跃升。

这完全是因为她选择的对象合适。接下来，我们将分析像秦公子这样的男性具备哪些特点。请注意，近期秦公子负面新闻较多，建议先下载保存本内容。

## 目标对象特征分析

上一节我们提出了核心问题，本节中我们来看看秦公子所具备的几个关键特征。以下是三个核心特征的分析：

**第一，财富的非自我创造属性。**
秦公子的财富并非自己创造。大家记得秦公子当初是靠什么一战成名的吗？是两次豪车撞车事故，一次保时捷，一次法拉利。车子几乎撞毁，他却坐在车里玩游戏等待处理，丝毫没有慌张或愤怒的情绪。
试想，价值几千万的豪车撞毁后能如此泰然自若，彰显的不仅仅是财力，更多是对金钱毫无概念。**公式：财富来源 ≠ 个人创造 → 对金钱的珍惜程度低**。说白了，反正钱不是自己挣的，花起来一点也不心疼。

**第二，性格狂妄，容易轻敌。**
我为何这么说？王思聪曾在某博上说：“我交朋友不管对方有没有钱，因为都没我有钱。”秦公子留言道：“那我呢？”从这段对话能看出，秦公子性格比较狂妄。
而且他多年来与各路网红、女明星绯闻不断，他对自我的认知，肯定认为自己是一位段位很高的花花公子。这导致他轻视了韦雪的野心。他明知韦雪是从广西农村到大城市混圈的女性，且有婚史和两个孩子，却毫无警觉性，最终被韦雪“咬了一大口”。

**第三，忌打不忌疼，重复犯错。**
在已经因轻敌让韦雪抓住机会怀孕后，他却给了韦雪一笔巨款让她自行处理。正常来讲，此时秦公子但凡多一个心眼，肯定会全程监督韦雪将此事处理干净。但是他没有，而是又一次给了韦雪钻空子的机会，让她带着钱和孩子去了美国，最终将孩子生下。
因此，韦雪最后能带着三岁的儿子回国，再次找到秦公子并使其付出巨额代价，就一点都不稀奇了。

## 课程总结

本节课中，我们一起学习了如何通过分析目标对象的特征来制定策略。我们拆解了秦公子的三个核心特点：**财富的非自我创造性**、**狂妄轻敌的性格**以及**重复犯错的行为模式**。这些特征共同构成了韦雪策略成功的基础。理解这些，有助于我们在分析类似社会现象时，抓住关键逻辑。
