## 高攀的本质：自我欺骗与心理安慰 🤔

上一节我们介绍了课程主题，本节中我们来看看高攀关系的核心本质。

你处于高攀状态时，会期望对方为你改变。你内心明白自己不具备足够的力量促使他改变。你心里默念“他会为我改变”，这种行为并非基于事实，而是在寻求心理安慰，类似于求生拜佛。

在李靓蕾与王力宏的关系中，她最初在一起时便接纳了现实。但在关系存续期间，她持续进行选择性的自我欺骗。这好比现在让你与吴彦祖结婚，是同样的道理。

以下是高攀者的典型心理活动：
*   你选择性地闭上眼睛。
*   只要对方不主动放手，你便不愿脱离这段关系。
*   你认为自己高攀得厉害，因此渴望维持关系，即便对方行为不端。

## 关系破裂的爆发点 💥

上一节我们探讨了高攀者的心理，本节中我们来看看这种关系通常在何时爆发危机。

最终的爆发点在于：我已经忍耐到这种程度，你却仍然不要我。我可以与你共度十年，只要你不抛弃我，我都可以闭上眼睛。为什么我能结婚十年并留存所有证据？因为在这十年里，我一直在选择性地自我欺骗和自我麻痹。

只要你不离开我，凭借你的光环，我与你在一起便能获得众人的羡慕。这提供了巨大的情绪价值，使我能够不断说服自己，降低情感需求，减少相处频率，并对外界装作不知情。但拥有“太太”这个身份，能让所有人羡慕我找了个好老公。

这个女生与王力宏在一起时，可能也会不断追问：“你跟我在一起以后就不会这样了吧？”但这种追问得到的并非真相，只是为自我欺骗和麻痹寻找借口，让自己心里好受一些。

## 唯一的出路：自我提升 🚀

上一节我们分析了关系破裂的导火索，本节中我们来看看高攀者可能的出路。

最终促使你放弃这段关系的唯一原因，是他先放弃了你。只有在那个节骨眼上，你才会翻出旧账，列举对方多年来的种种不是。你之所以说出来，是因为你忍耐了这么多年，为了在一起而承受了这一切，并非你真的相信他是一个不会出轨的、专一的人。

认识到最终结果其实没什么，就当是自己与偶像结婚了。但选择这样的人，存在一条活路可走。

这条活路就是：**你成长起来了**。也许你最初配不上他，但在一起之后，你逐渐能够匹配甚至超越他。这是你唯一可行的出路，其他路径都是死路。

以下是核心出路公式：
**最终关系稳定性 ≈ 你的个人成长速度 / 你们初始的价值差距**

如果你只想找个条件好的，从此轻松度日，那是不可能的。你最终的结局，便会与李靓蕾一样。
