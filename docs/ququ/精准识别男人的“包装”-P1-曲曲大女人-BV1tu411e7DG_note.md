在本节课中，我们将要学习如何通过关注对话细节，来有效识别他人是否在“包装”或夸大自己的经历与成就。这对于在人际交往，特别是网络交友中保护自己至关重要。

最近有一部很火的纪录片，专门讲述“杀猪盘”诈骗，叫做《Tinder诈骗王》。片中讲述了一个男人将自己包装成超级高富帅，骗了许多女性。他为了包装自己，甚至动用了私人飞机这类道具。

相比在短视频平台上看到的那些自称成功、多次东山再起的人，这种直接展示私人飞机的手段要厉害得多。在这种情况下，人们很难不被欺骗。

其实，要判断一个人所讲述的内容是真实经历还是精心包装，是有方法的。这个方法的核心就是三个字：**问细节**。

之前我有一位1对1咨询的客户，她遇到一位看起来是高富帅的男性，但感觉对方各方面有些奇怪。后来我就问她，那个男人是怎么起家的。

客户告诉我，那个男人自称家境不好、白手起家，曾与一位合伙人开工厂并因此赚到第一桶金。我建议她追问一些细节问题。

以下是你可以追问的具体方向：

*   他是与什么样的人合伙的？
*   他们具体生产什么产品？
*   他们最初投入了多少钱？
*   最后赚钱了为什么会散伙？
*   散伙的具体理由是什么？

我给了客户很多这类问题让她去询问。如果你与人沟通对话的经验足够多，就会发现一个规律：真正有过成功经历的人，通常非常乐于别人询问他的发家史。

每个人都有炫耀自己的欲望，也有好为人师的需求。因此，如果你问到点子上，他一定会详细谈论自己是如何成功的，或者曾经经历过怎样的失败过程。

反之，如果他跟你讲述的只有结果，例如“我当时通过某个项目赚了一大笔钱”，这就像在说自己是买彩票中奖一样，缺乏可信度。

所以，反过来推论，如果在与一个男性接触时，你问了很多细节性问题，但他却支支吾吾、眼神飘忽，总是试图用无关紧要的内容来掩盖事件过程，那么他讲述的内容大概率是有水分的。

那么，我那位1对1客户的最终情况如何呢？聊完之后，我发现那个男人十有八九是靠女人发的家。他所谓的合伙人，极有可能就是他的前女友，而且年龄应该比他大不少。说白了，他很可能是一个“软饭男”转型上岸。

现在疫情期间，很多人只能通过网络寻找对象。因此，大家一定要学会如何通过“聊细节”来辨别真伪。

如果你在聊天中发现，对方总是只有结果没有过程，或者每次讲述的细节版本都不一致，那就需要提高警惕了。

这时，我建议采用一个方法：带上你身边真正有阅历或有成就的人，一起去验证这个人所说的到底是真是假。
