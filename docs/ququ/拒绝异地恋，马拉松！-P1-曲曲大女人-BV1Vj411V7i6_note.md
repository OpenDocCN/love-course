# 情感策略课 P1：拒绝消耗型关系 🚫

在本节课中，我们将探讨两种对女性而言极具消耗性的情感模式——异地恋与马拉松式恋爱。我们将分析其潜在风险，并介绍一种更稳健的情感策略：“步步为营”法。

---

青春被消耗，你的最大价值就被削减。有两件事我非常不认同：异地恋和马拉松式恋爱。这两种模式对女生来说，是一种极致的消耗。女孩子的青春是最宝贵的，你不能拿着你的青春去跟一个男人赌未来。

他说不结婚就不结了，他说感情不好就不好了。而你，成了一个被单方面宣判的对象。除非是你自己不想结婚，那另当别论。在你想结婚的前提下，异地恋和马拉松式恋爱是坚决不能接受的。

艺人类似的案例也存在。这里不评判对错，只想阐明一个观点：青春被消耗，你的最大价值就被削减。有几个人能在年龄增长后，价值还能持续走高、持续上升？如果你的价值能随年龄持续上升，对方也不会不跟你结婚。

这个道理很简单。你最终的结果很可能是年龄上去了，但价值没有相应提升。那么你的价值就是**价值 = 年龄增长 - 价值提升不足**，导致整体价值未增反减。之后再寻找伴侣，就很难找到理想的对象了。

这就是现实。世界上固然有赌赢的案例，但我们这里主张毫无赌性的人生。我的人生就是要步步为营，每一个台阶都要走扎实。这是我人生的主旋律，我不会去做赌性过大的事情。

我能接受的风险波动非常小。两年时间我可以接受，但十年呢？一生中有几个十年？最精华的就是那十年。拿一个十年去赌能否和一个人在一起，这是不可能的。除非我说，这十年我拼了命地提升自己。

那就无所谓了。我不怕你不跟我好，我总能找到更好的，并且成就更好的自我。那是另一回事。所以，我常说的“步步为营的道路”是什么呢？

“步步为营”的道路意味着：如果我的目标是婚姻，那么我就在两年内搞定。该掌握财政大权就去掌握，该谈婚论嫁就去谈，两年内把事情全部完成。如果我退一万步，选择走长线、谈马拉松，那必然是因为什么？

必然是因为我对你的价值还有些摇摆，而对我自己的成长空间有非常大的期待。所以我可以把你暂时“放著”。你对我来说只是一个调剂，一个用来稳定情绪、填补情感空白的工具。我还有更大的空间。

我要去做自己想做的事情，成就更强大的自我。这也叫步步为营。因为不管最后是否和你在一起，我都变得更优秀了。请你们想一想，现阶段的情感状态，到底是不是一种步步为营的打法？

你们需要自己反思一下所处的环境。否则，马拉松结束后，价值没有提升，你除了当“泼妇”没有别的办法，只能撒泼打滚、耍狠。除了这些，别人对你没有任何可谈的空间和必要。

这样你就把自己搞得非常被动。将自己的人生推入被动境地的，绝不是对方一个人造成的。这必然是双方共同造成的结果。你需要为这个结果承担一半的责任。

---

本节课我们一起学习了消耗型关系（异地恋、马拉松式恋爱）的风险，并介绍了“步步为营”的情感策略。核心在于，女性应避免用宝贵的青春进行高风险的情感投资，而应专注于在有限时间内明确关系目标或致力于自我价值的持续提升，从而掌控人生的主动权。
