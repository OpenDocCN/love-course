# 课程一：配得感的核心——坦然接受赞美 🌟

在本节课中，我们将要学习“配得感”的一个重要基础：如何坦然接受他人的赞美。许多人在被夸奖时，会下意识地否认或谦虚，这其实反映了内心配得感的不足。我们将通过一个具体的故事，来理解自信回应的力量。

---

我相信绝大部分女孩子都有这样的经历。当别人夸你漂亮时，你的第一反应是否是“哎呀，没有没有”？

我小时候正是如此。每次被人夸奖漂亮，我都会连忙否认。

这件事后来被改变了。改变我的人是一位大家都认识的名人——演员白冰。

在她出名之前，我们曾一起参加过一个比赛。那时我就觉得她很漂亮，因为她长得很像金喜善。

有一件小事让我印象极其深刻。当时我见到她，很本能地夸赞道：“你长得好漂亮啊。”

她当时只是微笑，然后说：“谢谢。”

这段对话非常简短，但却一下子颠覆了我的认知。我心想，原来还可以这样接受别人的夸奖吗？

我立刻反思了自己。在过去的那么多年里，别人夸我漂亮时，我是什么反应？我突然意识到，当别人夸奖你时，最好的、最能彰显自信、最站得住脚的方式，就是简单地回复：“谢谢。”

这样的回应会让你显得自我认知感非常强，框架感也非常强。

这其实就是原生家庭赋予的“配得感”。绝大多数普通家庭的教育倾向于让我们谦虚、收敛。但那些家境优渥的女孩子，她们天生就更为自信。

---

本节课中，我们一起学习了建立配得感的第一步：**坦然接受赞美**。关键在于改变下意识的否认习惯，用一句简单的“**谢谢**”来接纳认可，这能极大地强化你的自我认知与自信框架。下一节，我们将探讨如何将这种接纳的心态，延伸到其他方面。
