# 知道什么样的人最容易上当受骗吗

在本节课中，我们将探讨一个常见的心理现象：为什么人们在刚刚经历一次欺骗或挫折后，反而更容易陷入下一次的陷阱。我们将从心理机制的角度分析其原因，并提供实用的建议来帮助你避免重蹈覆辙。

## 概述：一个值得警惕的现象

你们知道一个人什么时候最容易上当受骗吗？就是他刚刚被骗过一次的时候。例如，你刚被一个渣男伤害了一次，那么你大概率很有可能又会遇到一个渣男。或者你刚刚被一个项目骗了一大笔钱，那么极有可能你马上就会再做出第二次的错误投资决策。

你们回想一下你们自己的人生有没有这样的经历。你们知道为什么会这样吗？我认为是三个原因造成的。

## 原因一：求生欲导致的决策失衡

上一节我们提到了这个普遍现象，本节中我们来看看第一个深层原因：求生欲。

当自己遭受到重大挫折以后，你的自信心会严重受损，而且想要逆风翻盘的心情就会占据到制高点。这个时候你就有可能去抓住身边任何一个看起来是机会的东西，把它当做救命稻草。但往往机会跟陷阱就只有一线之隔。

**核心心态公式**：`重大挫折 → 自信心受损 + 强烈翻盘欲 → 降低判断标准 → 将风险误判为机会`

## 原因二：不理智状态下的视觉盲区

了解了求生欲的影响后，我们再来看看第二个原因：视觉盲区。

一个人在没有足够的理智和冷静的时候，那么你大概率就会对很多的人和事产生判断失误。很多的风险跟问题你压根就看不到了，你就只有那个想要赢的心理。因为人往往只会去选择相信你想相信的那一部分。

以下是视觉盲区的具体表现：

*   例如，有的女生觉得这个男生突然之间就为自己停止付出了，那么她就会认为是这个男生不愿意为自己付出。
*   但是她却看不到现在这个男生的视野面临着极大的问题，所以导致他的自信恋出现了状况。
*   这就是她在自己不理智的情况下面出现了视觉盲区。

## 原因三：虚弱状态吸引捕食者

前面我们讨论了内部心理如何影响判断，接下来我们分析外部环境如何趁虚而入。第三个原因就是因为你散发出了虚弱的味道，引来了想要趁虚而入的猎人。

当一个人经历了一次极其失败的感情，或者是经历了一次错误的投资的时候，就像是刚刚打完了一场败仗，既损失了粮草又消磨了信心。那些渣男和骗子就会闻到你此时此刻非常的虚弱，他们就会把下一个圈套打包成是粮草和弹药的样子送到你面前，让你在急于翻盘的错误心态之下做出第二次错误决策。

## 核心建议：低谷期的正确行动指南

基于以上三个原因，我们可以总结出关键的应对策略。

所以说大家千万不要在刚刚经历完一次重大的人生挫折之后，就马上投身于一段新的感情或者是进行一次新的投资。越是低谷的时候你越是要去学习，用认知上面的进步来弥补自己当下对于自信心的匮乏，缓解自己对于失败的焦虑。

**核心行动准则**：`心态不稳，一切皆输。`
