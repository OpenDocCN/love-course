# 能量驯化基础课 - P1：识别与阻断负面情绪 🧠

在本节课中，我们将学习如何识别自身升起的负面情绪，并通过即刻行动与思维转换来阻断其蔓延。这是驯化自身能量与磁场的第一步。

---

上一段我们提到了及时睡眠对缓解情绪的作用，本节中我们来看看更主动的干预方法。

你能感知到自己的负面情绪。当负面情绪在脑海或心中升起时，你必然能察觉到。察觉的瞬间，通常关联着某件具体的事。

此时，你需要立刻停下。

以下是行动步骤：

1.  **暂停与觉察**：意识到负面情绪产生时，立刻在心中叫停。
2.  **追溯事由**：明确引发当前情绪的具体事件是什么。
3.  **思维转换**：问自己：**“我能否用正面的方式看待或解释这件事？”**

你需要刻意训练自己，养成从正反两面考虑问题的习惯。所有事情本身都具有两面性，但我们往往优先考虑负面，因为人对痛苦的感知力通常强于对快乐的感知力。

因此，当糟糕的念头产生时，要对自己说“停”。然后有意识地练习：**“这件事如果从正面考虑，该怎么想？”** 这就是驯化自己、将任何事情导向正面思考的能力。

---

本节课中我们一起学习了情绪管理的第一步：及时觉察负面情绪的升起，并通过主动暂停和正面思维转换来驯化自己的思考模式。核心在于**感知 -> 叫停 -> 重释**的刻意练习。
