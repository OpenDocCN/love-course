每个人生来不同。对他人而言的幸福，对自己可能是一种枷锁。他人感到满意的事，对自己可能是一种未尝试的遗憾。因此，理解他人即可。经历增多后，会更认同“道法自然”的理念。

所有事情发展到一定阶段都可能发生变化。接纳变化、拥抱变化，尊重自己当下的内心感受。不必用条条框框或最初的执念约束自己必须如何。

没有必要强行约束。现在与一个人相处舒服，就继续相处；不舒服就分开。例如，我与初恋分手时计算了时间。

按其他女生的标准，初恋很珍贵，她们希望关系持续下去。这是绝大部分女生的想法。

但我的想法不同。与初恋在一起时，我心想：我们已谈了一年，再不分就来不及了。我一直认为自己可能在24岁结婚。

结婚前，我应该多谈几次恋爱。否则，没有对比，我无法知道自己真正想要什么。只谈一次，我只知道这一种模式，无法知晓世界上是否有更好的选择。

因此，与初恋在一起时，我计算着时间。心想：到一年了，必须赶紧分手。如果不分，就无法实现在24岁前多谈几次的目标。

经历一些事情后，我才真正明白自己想要什么。
