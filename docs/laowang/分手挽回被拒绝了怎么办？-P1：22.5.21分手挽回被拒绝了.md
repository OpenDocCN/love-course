# 分手挽回被拒绝了怎么办？-P1：22.5.21分手挽回被拒绝了

在本节课中，我们将要学习当挽回请求被拒绝时，如何正确理解这一结果并调整自己的心态。这是挽回过程中的关键第一步。

被拒绝是一件非常正常的事。昨天，一位女士一上来就哭了，她不断质疑对方是否从未爱过自己，是否根本不爱自己，以及自己是否从一开始就做错了。安抚她的情绪花费了一个多小时。

像这位女士的情况，在复合过程中经常遇到。这其实是内心不合理信念在作祟。许多人潜意识里认为：“只要她还爱我，我想要复合，她就应该答应复合。”这是一种不合理信念。

为什么你想要复合，对方就必须答应你呢？为什么对方拒绝，就一定是对方的错呢？这个逻辑本身听起来就不合理。这与爱或不爱没有直接关系，而是对方基于自身需求与意愿的表达。

你要明白，感情之事也需要权衡利弊。例如，你在商场看到一块价值数百万元的手表，你会凭借每月五千元的工资去购买吗？你不会。因为理智告诉你，现实与理想之间存在差距。你会接受现实。

在复合这件事上，道理是相同的。你要知道，一开始你是主动方，你是有需求的一方。面对你的需求，对方会进行她自己的利弊权衡。因此，她可以拒绝。你也需要给予对方拒绝的权利。

只有这样，对方才会感到安全。同时，对方也才会愿意继续与你接触。

---

本节课中我们一起学习了：被拒绝是挽回过程中的常见情况，它源于我们“要求对方必须同意”的不合理信念。拒绝的本质是对方基于自身需求和现实的权衡，与爱无关。接受对方拥有拒绝的权利，是建立安全感、促使对方愿意继续接触的基础。
