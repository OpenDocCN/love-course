在本节课中，我们将要学习分手后如何通过提供“情绪价值”来逆转关系，引导对方主动寻求复合。核心在于管理自我情绪并精准满足对方的情感需求，而非纠缠与哀求。

上一节我们介绍了课程的核心目标，本节中我们来看看为何情绪价值如此关键。只要学会提供情绪价值，就没有复合不了的感情。无论是失望型、爆发型还是情感消耗型分手，此方法均适用。

## 第一部分：管理自我情绪

让对方感受到舒适的相处体验，是复合的第一步。管理不好自身情绪，一切技巧都无从谈起。

我有个粉丝正是因为情绪问题导致情感破裂。她的情绪难以自制，男友犯点小错就会引发她的情绪爆发，进而歇斯底里地争吵。

人的情绪会传染。没有人愿意与一个充满负能量、喜欢抱怨的人长期相处。你的负面情绪会让对方感到害怕和压力。

结局往往是拉黑删除。因为在对方眼中，你只会变本加厉地做出更极端的事。人一旦情绪失控，结果就难以预知。对方恐惧的正是这种无法预判的失控感。

我认为在感情中，最高的价值就是情绪价值。想要恋爱关系长久甜蜜，就绝对不要无理取闹。

你应保持乐观活泼，不恃宠而骄，不把负面情绪带给爱的人。如果你总认为对方不够爱你，并变着法去证明，甚至逼迫对方表态，结果就是双方耐心耗尽。最终，对方会主动离开。

## 第二部分：提供对方所需的情绪价值

在自我情绪稳定的前提下，给对方提供他所需要的情绪价值。提供情绪价值可以起到杠杆作用，弥补你自身其他价值的不足。

那么，如何培养情绪价值呢？以下是六个核心方面，你可以自测在感情中曾经或能够提供多少。

以下是情绪价值的六个维度：

1.  **分享**
    很多人喜欢与闺蜜分享美食、旅游打卡、八卦话题，这就是分享价值。对于爱人，你也需要分享，但前提是分享他喜欢的内容，而非仅仅是你喜欢的。

2.  **陪伴**
    男生同样需要陪伴，但与女生需求不同。男生需要你陪伴他享受喜悦，而非分担忧愁。切记，不要在对方心烦时强行关心和陪伴，那样只会起到反作用。

3.  **新鲜感**
    所谓“七年之痒”，源于长期相处中新鲜感的缺失。在恋爱或分手复合期间，你能带来新鲜感，展现不一样的自己，他就会对你重新产生兴趣。

4.  **治愈**
    学会共情是最难的一点，也是两人相处的最佳技巧。如果你善于共情，在对方难过时，你能做到换位思考，把话说到对方心坎里去。平复对方的伤痛，就像在黑暗中为他打了一束光。

5.  **鼓励**
    我有两个粉丝案例：一个从小被父母指责，即使考满分也得不到表扬；另一个则在家庭中受到各种鼓励。虽然父母出发点相同，但结果是被指责者缺乏自信，复合过程艰难漫长；而被鼓励者不到一个月就成功复合。请思考，在你的感情中，是指责更多，还是鼓励更多？

6.  **认同**
    我们每个人之所以是自己，是因为我们与众不同。在交往中，你是否具备足够突出的重要性让对方认可？简单来说，你的存在能让对方感到有面子、有自豪感。每个人都或多或少有些虚荣心，你带给他的认同感就能产生巨大的情绪价值。

本节课中我们一起学习了情绪价值在分手复合中的核心作用。我们了解到，管理好自身情绪是基础，而通过分享、陪伴、新鲜感、治愈、鼓励和认同这六个维度向对方提供情绪价值，才能有效吸引对方，让他主动求复合。记住，**提供情绪价值，是能让前任主动求复合的关键**。
