# 复合过程中如何扭转低姿态 - 课程P1 🎯

在本节课中，我们将要学习在感情复合过程中，如何识别并扭转自身“低姿态”的问题。我们将通过一个真实案例，分析低姿态的成因、表现及其对关系的负面影响，并学习具体的扭转策略。

## 低姿态的成因与表现

上一节我们介绍了课程主题，本节中我们来看看导致感情中处于“低姿态”的核心原因。

你之所以会被低位分手，是因为你太善良。面对感情的态度往往是从一而终，所以你没有太多的情感经历，不懂得如何讨女孩欢心。偏偏你们在感情当中又比较单纯，所以很容易暴露需求感，或者说变得被动。

## 案例分析：无效沟通与过度付出

了解了低姿态的普遍成因后，我们通过一个具体案例来深入分析其典型表现。

以下是指导一位粉丝的经历，你可以对照看看自己是否有这种情况。他当时找我时，情况已经相当被动。因为暧昧挺长时间了，但是一直没确认关系，他就特别没安全感。而对方呢，就因为他暴露太多的需求感，觉得压力很大，一直就试图跟他拉开距离。

仔细聊了很多之后才知道，他之所以会这样，真不怪对方。他真就是思维太单纯，他理解不了女人在沟通时的话外之音。经常性地无视对方的需求，就只顾表达自己的想法，让对-方误以为自己不被重视。

再加上跟对方聊天的时候，一直在进行无效交流，白白浪费太多交流机会，并且消耗对方太多兴趣了。

## 扭转策略：停止无效付出与重塑框架

分析了问题所在，接下来我们看看如何采取行动来扭转局面。

了解这些情况后，我给这位兄弟一个建议：停止再用无效的交流与关心消耗对方的兴趣。我反问他：对方是救过你的命吗？你这么不求回报地对他好？你这样无条件的付出，你只会让他觉得：既然我啥都不做，就能得到这些付出，我还有什么必要对你好？

感情里面你要想拿回主动权，核心是让对方发自内心的喜欢，而不是你自我感动式的付出。这个关系可以抽象为一个简单的**公式**：

**健康的吸引力 = 自我价值 + 适度神秘感 - 过度需求感**

当这位粉丝在我指导下，收回了那种卑微的姿态后，对方的态-度也发生了改变，开始试探他的状态，问他什么时候回来。

## 实战应用：以退为进与激发“损失厌恶”

理论需要实践来验证，本节我们看看上述策略在具体情境中如何应用。

这位粉丝当时为了朋友的事去到外地，打算玩两天再回来。收到对方的消-息后，他恨不得下一秒就飞到她身边。我就告诉他先稳住，直接告诉对方“晚点再回去”。

果然对方坐不住了，就追问他跟谁一起玩。这个时候我让这位粉丝回复：“想自己一个人散散心”。

在接下来几天，我让这位粉丝忍住自己的分享欲。对方感觉到他的变化后，主动找他聊天，试探他是不是有了新目标，甚至主动向他提出邀约。

之后这位粉丝在我指导下，改变了相处模式。对方也察觉到了他不像之前那种“求着、逼着、害怕失去”的恐慌，这让他逐渐失去了感情的主导权。所以说，借助对方的“损失厌恶”心理，他顺利地逆转了低姿态的地位。

## 总结与互动

本节课中，我们一起学习了感情中“低姿态”问题的根源在于过度暴露需求感与无效付出。扭转的关键在于停止自我感动式的付出，通过重塑自身框架和适度后撤，来激发对方的关注和“损失厌恶”心理，从而重新掌握关系的主动权。
