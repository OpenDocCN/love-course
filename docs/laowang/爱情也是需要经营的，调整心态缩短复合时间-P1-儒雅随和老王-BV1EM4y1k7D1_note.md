# 爱情经营课 01：调整心态，高效复合 💔➡️❤️

在本节课中，我们将学习如何通过调整自身心态，有效缩短与前任复合所需的时间。核心在于将关注点从对方身上移回自身，通过积极的自我建设来重建吸引力。

---

任何能从恋爱走到婚姻的感情，最初至少有过十次想要分手的念头，五十次想掐死对方的冲动，一百次的争吵，和一千次对当初选择的质疑。

世界上没有所谓的天作之合。所有美满幸福、白头偕老的爱情，都经历过无数次的相互磨合和漫长的复合过程。如果觉得不合适，就应该去磨合；磨合不了，也应该去包容。即使爱得很累、很难受，也不想放手。

因为没有什么比失去对方更难受。正如张国荣所说，在一起很容易，缺点可以改，性格可以磨合，但是错过了，就真的没机会了。

如果你决定要复合，请认真记下接下来的内容。这可以帮助你在短时间内达到复合的目的。

## 第一步：远离负面情绪与事件 🚫

在分手初期，很多人习惯性地到处找朋友诉苦，或者反复回忆那些令人难过的事。结果，当你越把心思放在负面情绪上时，负能量就会被无限放大。

这就像学生时代，考试前被老师或家长批评了。如果一直对此耿耿于怀，考试必定会受影响。成绩出来后，又会继续纠结分数，最后产生一系列连锁反应，可能导致连续几周心情糟糕，学习成绩直线下降。

同理，分手之后，最潇洒的表现就是尽快收拾好坏心情，让正能量的事情逐渐融入生活。

以下是你可以做的事情：
*   与朋友聚会、去旅游。
*   健身、运动。
*   阅读书籍。
*   专注于任何能让你开心、积极阳光的事。

只有这样，你才有可能在短时间内反转自己的形象，并且让自己变得更好。这是你重新吸引前任的重要筹码。

## 第二步：停止窥探前任 🔍

上一节我们介绍了如何管理自身情绪，本节中我们来看看如何正确处理与前任的“距离”。有些人在分手初期，不断去联系对方，或者每天查看对方的朋友圈、抖音等动态，试图从中找到蛛丝马迹，揣测对方的想法和状态。

这是典型的“恋爱脑”思维。分手后，你越看重得失，越看对方动态，就会越焦虑；越焦虑，就越想联系对方。你会把全部精力投入其中，不断思考：她这几天在干嘛？她有没有想我？我该怎么和她沟通？她还会和我在一起吗？

请记住：杯子烫手，就该放下。等温度降下来，再拿起来，它不就暖手了吗？

## 第三步：聚焦自我，提升价值 ✨

很多人在恋爱中，把所有心思都放在前任身上，却忽略了照顾好自己、让自己变得更好，结果反而导致对方越来越讨厌自己。

如果你把能量和焦点放回自己身上，持续地自我提升，让自己感觉更舒适，这样才能真正吸引前任回心转意。

这样做不只是让你更有魅力，更是让男方明白必须尊重你。如果你自己都不尊重自己，别人又怎么会尊重你呢？

你需要从里到外好好打理自己。当你越精致，他对你就越尊重。

## 第四步：坚定复合信念 💪

增加信心，相信自己一定能复合成功，这个信念至关重要。这与“瓦伦达效应”很相似，目的是为了避免患得患失的心情，防止琐事干扰复合进程，降低由心理因素导致的失败概率。

---

本节课中，我们一起学习了高效复合的四个核心心态调整步骤：**远离负面情绪**、**停止窥探前任**、**聚焦自我提升**以及**坚定复合信念**。透彻理解并实践以上几点，你离成功复合就很近了。别嫌麻烦，记住，爱情也是需要经营的。
