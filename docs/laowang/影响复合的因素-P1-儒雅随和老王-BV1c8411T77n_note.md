# 复合影响因素详解 - 课程P1 🧩

在本节课中，我们将要学习影响复合成功与否的两个核心因素。理解并妥善处理这两个因素，是推动对方主动回头的关键。

上一段我们提到了复合的可能性，本节中我们来具体看看是哪两个因素在起作用。

影响复合进程的主要有两个因素。

以下是第一个因素：对方的需求。

**需求**并非单指物质层面，更涵盖了情感需求。人的所有需求都基于**安全感**。长期未被满足的需求会导致关系难以维系。你需要反思，在过往相处中，你是否忽略了他的需求，而自己的需求却得到了满足。这种不平衡往往是分手后你想复合的根源。

理解了需求的重要性后，我们接下来看看如何通过互动影响对方的心理。

以下是第二个因素：双方的接触频率。

**接触频率**并非指每日死缠烂打，而是指以自然方式建立的联系频率。例如，分手后对方若能经常看到你的状态，或你们之间因合理事由保持偶尔联系，他想念你的频率就会增加。这为复合创造了有利条件。

回顾本节课内容，我们一起学习了影响复合的两个核心因素：**对方的需求**和**自然的接触频率**。只要你之前的方法得当，现在只需有针对性地解决这两个问题，就能有效提高对方主动复合的可能性。
