上一节我们提到了异地恋复合的独特优势，本节中我们来看看如何迈出第一步。

## 第二步：逐步恢复沟通 💬

如果对方没有表现出抗拒的情绪，就可以继续沟通下去。当对方开始习惯并且乐意跟你交流的时候，你就可以找机会跟他语音或者视频通话。

以下是此阶段的核心目标：
*   加深你在他心中的印象。

等到时机合适，你就可以去他的城市，跟他面对面沟通了。这是感情升温的杀手锏，它能唤起对方曾经对你的感情。

有时候感情比我们想象的要深，虽然看似被时间冲淡了，但仍然保留在我们的潜意识当中。

最后，需要给对方一个感情目标，也就是你规划结束异地的时间。让人放弃的有时不是困难，而是望不到头的那种绝望。

**核心公式：安全感 = 现实的未来规划**

只有一个现实的规划，才能给予对方踏实的安全感，使其愿意跟你一起分担异地恋当中的困难。
