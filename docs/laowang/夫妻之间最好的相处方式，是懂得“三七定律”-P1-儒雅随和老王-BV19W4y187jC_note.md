# 婚姻经营课 01：掌握“三七定律”，让夫妻关系更和谐 👫

在本节课中，我们将要学习一种被称为“三七定律”的夫妻相处智慧。这个定律的核心在于平衡付出与自我关爱，旨在帮助夫妻双方建立更健康、更持久的关系。我们将详细拆解这一定律的两个关键部分，并通过具体的方法来实践它。

上一节我们介绍了“三七定律”的基本概念，本节中我们来看看它的第一个核心要点：三分付出，七分爱自己。

许多人认为，婚姻中只要有爱就足够了。然而，想要夫妻生活越过越好，必须讲究方法。当你将自己全身心的精力都放在伴侣身上时，往往不会得到珍惜，反而会被视为理所当然。

因此，爱一个人，切勿爱得太深。爱情中最理性的状态是：**三分爱别人，七分爱自己**。这并非自私，而是在任何情况下，爱自己都是首要大事。

以下是实践“七分爱自己”的两个具体方向：

1.  **爱护身体健康**：努力保持一个健康明朗的好身体。唯有如此，才能安然享受生命中一切美好的事物，包括爱情、亲情和友情。
2.  **保持积极心态**：维持积极乐观的心态。当你的精神世界明亮时，任何时候都会对人生充满希望。

永远记得，好好爱自己才是建立一切良好关系的基础。

---

本节课中我们一起学习了夫妻相处的“三七定律”。我们明白了，在婚姻中，全身心的付出未必能换来珍视，平衡才是关键。通过践行“三分付出，七分爱自己”的原则，关注自身健康与心态，我们才能以更饱满、更独立的状态去经营爱情，从而让夫妻关系更加稳固和幸福。记住，爱自己，是终身浪漫的开始。
