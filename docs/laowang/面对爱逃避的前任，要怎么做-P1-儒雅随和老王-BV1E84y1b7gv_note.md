# 课程一：如何应对逃避型前任 🧠

在本节课中，我们将要学习如何应对在关系中习惯逃避、不愿解决问题的前任。理解其行为背后的成因，并掌握针对性的复合策略。

## 逃避行为的根源

上一节我们介绍了课程主题，本节中我们来看看逃避行为产生的原因。

如果前任一直逃避你，不愿意解决问题，这与他的父母教育方式有很大关系。我之前有个粉丝，他的对象是一个极其拧巴的男人，稍微有一点情绪就会马上消失。

## 核心复合策略：带领与适度逼迫

与这样的人复合，需要你去带领和适当地逼迫。因为他们害怕解决问题，更害怕面对情绪。他们宁愿让矛盾憋着，也不愿发生正面冲突。

如果你要与这样的人复合，你就需要一定的时间，让对方慢慢释放心里的不满和压抑。

## 应对不同类型逃避者

以下是几种常见的逃避者类型及其应对方法。

*   **框架性强、固执型**：有些人框架性很强，又犟又倔。一旦他认定你俩不合适，就会对你采取冷暴力。他要么删除你，要么把你拉黑，或者不接受你的任何信息以及任何试探。
    *   **应对示例**：我的一位粉丝情况特别深刻，他所有的联系方式基本上都被对方拉黑了。最后是通过游戏好友的途径重新取得联系，才慢慢吸引对方，使其重新产生认知性的改变。

*   **性格软弱、缺乏主见型**：不过，有的人性格也比较软弱。即使主动结束了一段长期关系，也很难与你划清界限。
    *   **应对公式**：对于这种人的复合，你一定得趁早。因为时间一旦拖久，就容易出现两种情况：一个是你成为他的备胎，第二个就是他有了新欢。

## 总结

本节课中我们一起学习了逃避型前任的行为模式与心理根源。关键点在于理解其逃避源于对冲突和情绪的恐惧，并需要采取“带领”与“适时推动”的策略。针对不同性格的逃避者（如固执型或软弱型），需采取不同的联系与复合方法，并注意把握时机。
