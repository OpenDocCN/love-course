上一段我们提到了课程的主题，本节中我们来看看男性在分手后的即时反应。

现在，我们对比一下大部分女性在此时的常见想法。

女性在分手后，常常对男性的逃避行为进行错误的归因。以下是两种典型但不正确的想法：

这种心态还会引发一个严重问题：过度付出与道德绑架。
