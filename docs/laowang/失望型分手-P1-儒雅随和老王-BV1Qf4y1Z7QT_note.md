# 失望型分手挽回指南 - 课程01 🧩

在本节课中，我们将要学习失望型分手的基本概念、核心原因以及挽回初期的一个关键策略——如何通过第三方可信地传递你的改变信息。

---

众所周知，失望型分手是所有分手类型中最难挽回的一种。但同时，它也是一旦挽回成功，两人感情最稳定的一种。失望型分手通常由两种类型的原因导致。

第一种类型，是因为你做得不够好，从而导致对方失望。

第二种类型，是因为对方对这段感情的期望值过高，从而导致其失望。因此，失望型分手往往并非因为感情消失，恰恰是因为感情过于浓烈，从而伴随着非常强烈的失望。

所以，如果你想要复合，就应该重点思考如何让对方重新意识到最初吸引他的那一面。此时你需要注意，不要试图通过语言说服对方回心转意。失望型分手根本不吃这一套，因为对方已经给过你无数次机会。

但是你都辜负了对方的信任。所以现在仅凭几句话，你注定无法扭转当前局面。因此，这个阶段你可以借助第三方来传递你的改变，让这种改变显得更加可靠。

一个好的第三方至少需要具备以下三个特征。

以下是第三方应具备的三个核心特征：

1.  **共同好友**：这个人是你们共同的朋友。
2.  **对你持正面态度**：这个人对你没有负面看法，甚至对你有些欣赏。
3.  **关系适度**：这个人跟你关系不那么亲密。否则会引起对方怀疑，认为你们是串通好的。

找到合适的第三方后，你可以让其帮你传递一些信息。例如，“我今天刷朋友圈，发现他变化挺大”，或者“我最近正好遇到他，发现他最近怎么怎么样了”。相比由你本人去说，由第三方介入讲述这些信息，可信度更高。

但是，当对方因为第三方的介入而产生一些动摇时，就需要你重新出现在对方眼前。你需要让对方亲眼看到你的变化。

---

本节课中，我们一起学习了失望型分手的核心是“高期望伴随强烈失望”，以及挽回初期应避免直接说服，转而通过**符合三个特征（共同好友、正面态度、关系适度）**的第三方来可信地传递改变信息，为后续的亲自展示奠定基础。
