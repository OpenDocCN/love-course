# 分手代表着吸引力消失吗？-P1-儒雅随和老王-BV1Rs4y1q7wm_note

在本节课中，我们将要学习分手后双方的心理变化过程，并探讨吸引力是否真的会随着关系结束而立刻消失。我们将借助心理学的研究发现，来理解情感联结的延续性。

很多人认为分手一定是因为吸引力消失了。但事实果真如此吗？分手不能代表感情戛然而止。一段真心付出的感情不会说断就断。分手之后那种痛苦的感觉，就是这段关系的延续。

心理学上有一个著名的21天效应。这个效应指的是什么呢？它指的是人为让旧的心理形象消失、新的心理形象建立，需要花费大约21天的时间。

上一节我们提到了21天效应，本节中我们来看看更长期的研究。伦敦的健康心理学研究院在2009年进行了一项更科学的心理学研究。该研究调查了在12周内，96个人新习惯的养成情况。

以下是该研究的关键发现：
*   研究结果发表在欧洲的《社会心理学杂志》上。
*   对一个人平均而言，忘掉旧习惯、创建新习惯的过程需要3个月以上。
*   确切地说，这个过程平均需要 **66天**。

换句话说，如果他想要忘掉你并开始全新的生活，那这可能要花费他3个月到8个月的时间。这看起来果断的分手，其实往后还是难舍难分的。

当伴侣长期在一起的时候，特别是已经习惯了彼此存在的那种关系，这两个人互相给予的依赖感会在分手之后缺失。他会感觉丢失了自己所关心的人，也失去了习以为常的陪伴。

这种分手的痛苦，就是源于上述的孤独感。但是，孤独感其实是吸引力最好的容器。我们在情绪波动的时候，更容易受外部因素影响。

你的前任忘不了你。反而根据记忆的规律，在分开之后半个月之内，他就会不由自主地陷入到回忆的漩涡。过往那些记忆，在这个时候会大量地涌现出来。

这一段时期，就是我们展现自己吸引力的最佳时机。它也是修复感情的最佳时机。

本节课中，我们一起学习了分手后心理适应的“21天效应”和“66天习惯养成”研究。我们明白了分手初期的痛苦是情感联结的延续，而随之产生的孤独感与回忆涌现期，恰恰是重新展现吸引力和考虑关系修复的关键窗口。吸引力并不会随着分手声明而瞬间消失，它会在心理层面持续一段时间。
