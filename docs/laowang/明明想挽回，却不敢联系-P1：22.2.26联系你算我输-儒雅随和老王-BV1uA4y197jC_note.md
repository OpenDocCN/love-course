# 挽回中的心理障碍与应对策略 🧠 - 课程 P1

在本节课中，我们将要学习在试图挽回感情时，人们为何会感到害怕并拒绝联系对方。我们将深入分析背后的心理原因，并探讨两种不同的归因模式如何影响我们的行为和情绪。

---

要说你拒绝或者说不敢联系对方，那无非就这三个原因。

**第一**，你害怕会遭到对方那种无情的拒绝，或者说听到一些让你十分伤心的话，你怕瞬间崩溃。

**第二**，你害怕失望。就你想问清楚对方一些事情，你觉得至少他应该说点什么。但是对方他就不回应你，或者说十分混蛋。

**第三**，你们可那会儿暗中较劲。在你这个认知里边，谁先联系对方，谁就输过了。但你不能输，你这个自尊心他就不允许你输。

所以说，害怕受伤、失望、无力，想要维护自己尊严，这都是正常心理。没有人会想着去面对这些玩意儿。

但关键的是啥？很多人回避痛苦的防御机制，它变成了自我否认。你的自我否认我能理解，但你这样做的结果就是啥？你就会陷入一种对抗的模式。

上一节我们介绍了不敢联系的三个核心原因，本节中我们来看看这种“对抗模式”具体是如何表现的。

跟自己对抗，那就是内耗。跟对方对抗，那就是向外归因。

以下是跟自己对抗的具体表现：

*   讨厌自己那么没用，就想着联系他。
*   埋怨自己没法忘记他。
*   你要花很多时间，很多精力来跟自己对抗，时常批判自己，对自己各种不满。这个就是内耗。

以下是跟对方对抗的具体表现：

*   事件之后，你先咒骂一下对方，臭渣男/女。
*   然后你就不停的跟周围人倾诉对方对自己的伤害。
*   就把所有的问题，都归结到对方身上了。

心理学有这两种基本的归因模式，就是**内归因**和**外归因**。

就是说啥？当一件事发生之后，我们大脑会把这件事情找到原因。因为啥？因为找到这个原因，就解决了我们如何处理这件事情。

比如说分手，外归因就是啥？就把分手归结于外部，也就是说啥，你就归结到对方身上。公式可以表示为：
**分手原因 = 对方的过错**

就觉得自己做的没啥不对，总是对方不理解自己，总是对方做的太过分。这样的思维模式，它就不是个成熟的思维模式。

我们没有办法说，看到分手的一个真正原因，会把自己伪装成一个啥？**受害者**。在进入一段感情的时候，也很容易因为同样的问题再分手。

---

本节课中我们一起学习了在挽回时不敢联系对方的三大心理原因（害怕拒绝、害怕失望、维护自尊），并深入分析了由此引发的两种对抗模式：**内耗**（自我对抗）与**外归因**（将问题全归咎于对方）。理解这些心理机制，是迈出健康应对第一步的关键。
