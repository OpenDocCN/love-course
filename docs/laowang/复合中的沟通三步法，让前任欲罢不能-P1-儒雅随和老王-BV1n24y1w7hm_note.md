# 复合沟通三步法：让前任欲罢不能 - 课程01 🧠

在本节课中，我们将要学习心理学中的“弃猫效应”，并理解如何将其应用于情感沟通中，以建立更健康、更具吸引力的互动模式。核心在于调整自身心态与行为，而非一味讨好或纠缠。

---

爱情中最常见的情况是：有心栽花花不开，无心插柳柳成荫。当你想要好好对待对方时，对方不珍惜；但当你想要放弃时，对方却突然回过头来找你。当你努力想要经营好一段感情时，对方会觉得你的付出是理所当然的。

你所有的忍耐与付出，并非被他个人的魅力所征服。当你忍无可忍，坚决想要放弃这段感情时，对方才意识到，原来你的付出背后早已标好了价格。此时，他反而开始回头珍惜你。

心理学上将这种现象称为 **“弃猫效应”**。它讲述了一只从小养尊处优、性格很差的小猫，直到主人忍无可忍将其抛弃后，它才明白，以往衣食无忧的生活并非理所当然。

流浪、饥饿、甚至死亡，才是流浪猫的常态。当它被下一任主人收留时，就变成了一只格外乖巧的小猫。这个效应说明了一个道理：**幸福是对比出来的**。没有经历过不幸，就无法懂得当下的幸福，也就不会珍惜。

因此，当他不理你时，你也没必要去找他。之前有粉丝问我这个问题，我建议他找点自己的事做，不要搭理对方。于是这位粉丝去参加了自己圈子里的活动。

他与活动主办方混熟了，还帮忙一起举办活动。不知不觉中，他认识的人越来越多，参与的活动和项目也越来越多。他还把这些经历发在朋友圈。他的前任看到后有些坐不住，开始点赞、评论，甚至询问他是否已和别人在一起。

所以，你需要让他体会到“失去你”的可能，他才会去追寻你曾给予的幸福。只有当你真正想放弃他时，他才能意识到即将失去你。此时，若你仍努力挽回，只会让他觉得你离不开他。

对待感情不能硬来。接受现实，反而可能柳暗花明。如果这段感情已经在消耗你，而你仍紧抓不放，关系不会有本质改变。即使对方回头，这段关系依然会持续消耗你，因为他总会觉得自己处于高位，回头也只是对你的施舍。

那样的关系，你无法保证他会一直爱你。只有当你拿出坦然接受的态度，接受“聚散不由你我”的心态，这种轻松、随意又自强自立的态度，反而会让对方对你着迷。
