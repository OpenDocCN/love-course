# 复合必修课01：停止内归因 🧠

在本节课中，我们将要学习一个在复合过程中至关重要的心态调整：停止内归因。许多想要复合的朋友，常常陷入过度自责的思维陷阱，这不仅会加深痛苦，还可能错失复合良机。我们将深入剖析这种思维模式，并学习如何走出困境。

## 什么是内归因思维？

上一节我们介绍了课程主题，本节中我们来看看什么是“内归因思维”。

明明是他提的分手，到头来反而你感到有些愧疚。这是因为你太喜欢自我反思，才会把所有问题都归到自己头上，总觉得自己当时如果怎样做，结果就不会这样。这在心理学中被称为“内归因思维”。

## 内归因思维者的特征

理解了概念后，我们来看看具有这种思维的人通常有哪些表现。

以下是内归因思维者的几个典型特征：
*   通常是想要去复合的一方。
*   总是觉得问题在自己身上，认为只要自己做好就能复合。
*   对过去的感情心怀愧疚，并因此想要去弥补。

你可能没有发现，一直陷入在这种自责的情绪当中，才是分手给你带来的最大伤害。对方只是选择结束了这段关系，但之后一直在贬低、否认你的人，是你自己。

## 内归因的危害

认识到特征后，我们必须正视这种思维模式带来的危害。

你得明白，一段关系的结束，并不一定代表你做错了什么。很有可能是你们两人不太合适，甚至问题出在对方身上。你过度地反思自己并陷入自责，不一定能让你变得更好，反而可能让你更加敏感脆弱，情绪也更不稳定。

在这样持续的精神内耗中，也会导致你错过复合的最佳时机。即使你跟对方重新联系上了，这种愧疚感也会让你不自觉地在对方面前矮一截。

## 如何行动：从自责到成长

既然内归因危害巨大，那么我们应该如何正确应对呢？

你与其自责，不如尝试专注自我提升，把握当下。你之所以会想“如果当时怎么样，他就不会分手了”，其实还是因为你不知道如何处理亲密关系，很多道理你都不懂。就算之后你再遇到类似情况，结果可能还是一样。

所以你要做的，不是陷入内疚情绪或不断批判自己，而是应该采取实际行动。以下是你可以选择的方向：
*   **提升自我**：专注于个人成长，成为更好的自己。
*   **努力复合**：在心态平稳后，采取理性、尊重的行动去争取。

无论选择哪条路，都比站在原地怨天尤人更好。**行动 > 自责**

## 关系的本质与释怀

最后，我们需要从更宏观的视角看待这段关系的结束。

感情归根结底是两个人的事，每个人都会有做得不够好的地方。但两个人在一起的意义，就是要一起变得更好。而当他狠心抛弃你的时候，他也背弃了曾经的承诺。因此，你也不再亏欠他什么了。

---

本节课中我们一起学习了“内归因思维”的表现与危害。我们明白了关系的结束不全是自己的错，过度自责只会带来伤害并阻碍成长。关键在于停止内耗，将精力转向自我提升或理性行动，并最终理解感情是双方的责任，从而真正释怀，轻装前行。
