# 情感修复课 01：断联后如何应对联系前任的冲动 💔

在本节课中，我们将学习“断联”这一情感修复策略的核心要点。当分手后忍不住想联系前任时，如何通过正确的步骤控制冲动、调整心态，并最终实现关系的逆转或自我提升。

---

断联之后，如果忍不住想要去找前任，应该怎么办？请听清楚。在复合过程中只有两个选项：要么高位复合，让他主动对你好；要么这个男人就不要了。不存在第三个选项。

一般控制不住想要去找前任的人，心里或许还抱有幻想，认为再找他一次，或许还能勉强维持关系甚至复合。但这种情况在现实世界里并不存在。如果非要第三个选项，那只能是卑微地求他，然后又被对方无情抛弃，颜面尽失。

不要担心在使用断联之后，对方会彻底离开。只要正确使用断联，就能让这个男人日思夜想求复合。

## 第一点：断联前必须做好铺垫

我之前有个粉丝，从网上学了一套断联复合法，但最后不但没效果，还把对方越推越远。他找到我后，我了解到他们断联的时机和方式都不对，从而导致误会增加，把复合的难度和成本提升了五个档次。

后来我告诉他，给前任发一条信息。在这条信息里，表明自己的态度和对过去的看法，并且向对方表示自己已经接受分手的事实，感谢对方一路的陪伴。然后直接开始断联。

**核心操作公式：**
```
发送信息 = 表明态度 + 接受事实 + 表达感谢
随后立即进入断联状态
```

这就是在后撤之前做好的铺垫。它既可以缓和对你的负面情绪，也能让对方在想起你时，念着你的好。如果希望断联之后对方会主动联系你，这一步必不可少。

要知道，为什么那些“绿茶”能让男人日思夜想放不下？就是因为她们能克制住情绪，在推开男人之前说点好听的话，做好这个铺垫。所以说，女人还是得有点“心机”。

## 第二点：断联期间重建心态

很多情人把断联当成吊对方胃口的手段，这简直是胡说八道。不要把断联看作是无脑地跟对方比狠。

断联是破茧重生的一次机会，是给双方消化负面情绪的时间，同时也是提升自己、打磨自己、再次吸引对方的阶段。因此，断联期间的重心应该放在自己身上。

很多人在和前任断联后总是脑补：他会不会有新欢？他有没有关注我的动态？他是真放不下我，还是不喜欢我？或者，他是不是从来没想过和我长期发展？

今天就来告诉你断联期间男人的心理，防止你再胡思乱想。

以下是断联期间对方的典型心理活动：

*   **首先**，即使是他提出的分手，也不代表他一转身就能彻底忘记你。多年的感情，谁都不可能立刻做到。他的决绝只是比你多做了一些心理准备而已。
*   **其次**，人都会高估自己的存在感。分手之后，男人会觉得你会难过、会伤心、会哭着来求他复合。这种期待，正中对方虚荣和自恋的心理。
*   **如果**这个时候，你哭着闹着求复合，对方会把你当成“地位低”的一方，自然对你爱答不理，态度不好。
*   **如果**你不按套路出牌，让他摸不着头脑，让他对你的态度百思不得其解，他才会重新审视你。

所以说，断联是对付男人最好的招式。我的粉丝在断联期间，大多数都是男人主动来联系的。

切记，只有在断联期间不断磨练自己，才能破茧成蝶，获得全新的爱情，让这个男人对你着迷。

---

本节课中，我们一起学习了断联策略的两个核心阶段。最后想强调的是，正确的断联，并不是断开联系这么简单。复合也不是简单重复过去的过程。

**分手的黄金定律是：通过断联，蜕变成全新的自己。** 只有这样，才能殊途同归，让这个男人更加倍地爱你。
