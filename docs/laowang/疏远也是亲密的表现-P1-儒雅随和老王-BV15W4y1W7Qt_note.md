# 课程 P1：理解回避型依恋模式 👥

在本节课中，我们将要学习一种常见的人际关系模式——回避型依恋。我们将探讨其核心特征、形成原因，并理解其行为背后的逻辑。

## 概述

有一种现象是，明明对方很喜欢你，但当你表现出你也喜欢他时，他反而会越来越疏远你。这是典型的回避型行为。这种行为模式的根本原因，是当事人从小缺乏安全感。

## 回避型依恋的核心特征

心理学上，回避型依恋模式通常表现为以下几个特征。

以下是其七个主要特征：

1.  **缺乏安全感**：认为感情不堪一击，也不相信任何关系能够长久稳定。
2.  **逃避沟通**：或者说缺乏沟通能力，经常将沟通转变为争吵。
3.  **表达能力弱**：习惯用说反话、做反事来表达自己的感情。
4.  **刻意远离亲密**：即使对方很喜欢自己，也会刻意疏远亲密关系。
5.  **鄙视亲密**：认为“恋爱脑”是对一个人最大的侮辱，也害怕自己被感情束缚。
6.  **感情中缺乏自信**：看似处于高位，实则内心低位。对自己要求苛刻，不允许自己犯错或出丑，认为必须“完美”才配被爱。
7.  **将克制与痛苦等同于安全**：并非不想被爱，而是认为必须结合“逃避”，这份爱才安全。公式可表示为：**安全感 = 爱 + 逃避**。

## 行为背后的矛盾心理

上一节我们介绍了回避型依恋的具体特征，本节中我们来看看其内在的心理矛盾。

回避型依恋者对亲密关系的整体态度，并非真正的厌恶，而是一种矛盾的“想要接近却不敢”。他们身披铠甲，看似熟练地疏离他人，但背后隐藏着与常人相同的需求：他们也需要被爱，也需要去爱，甚至需要别人的支持和温暖。

关键在于，正常人面对亲密关系的态度是“喜欢，所以我接近”。而回避型依恋者的态度是“喜欢，但喜欢没用，喜欢会让我受到伤害，所以我得逃避”。其行为逻辑可以概括为：
```text
if (感受到亲密):
    触发不安全感
    执行逃避行为
```

## 总结与启示

本节课中，我们一起学习了回避型依恋模式的特征与心理机制。核心在于理解，遇到回避型的人时，你不能用常规的、直白的逻辑去判断和应对他的行为。他们的疏远并非厌恶，而是其内心矛盾与自我保护机制的外在表现。
