## 第一步：封存关系，扭转局面 🛑

上一节我们概述了案例背景，本节中我们来看看第一步具体如何操作。

“封存关系”并非让你彻底断联，而是将这段关系暂时“封存”起来，只保留最低限度的交流。目的是降低对方的抵触情绪，同时消除男生对女生的负面印象。

把这段话发出去，直接亮明我们的态度：我们是爱他的，只是方法没用对。同时，这段话也共享了两人过去美好的回忆。用这种心平气和的语气，能让男生放下戒备。

**核心操作公式：**
```
封存关系 = 降低联系频率 + 消除负面情绪 + 传递平和态度
```

我们发出这段话后，男生过了三个小时回复：“已经对你没感觉了。”女生看到这样的回答，当然绷不住了，想去找男生，但被我及时制止了。我能理解她，当初是男生主动追的她，在一起时她也从未做过对不起对方的事，现在一句“没感觉了”就离开，任谁都难以接受。我相信屏幕前不少朋友遇到这种情况也忍不住。
