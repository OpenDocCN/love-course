# 复合是一场双向奔赴的旅途 - 课程 P1 🧭

在本节课中，我们将要学习复合过程中的核心心态与沟通策略。我们将探讨为何急于求成往往适得其反，并理解复合的本质是一场需要双方共同努力的旅程。课程将重点分析常见的错误提问方式，并介绍更有效的沟通技巧。

---

## 急于求成是复合的大忌 🚫

上一节我们介绍了课程概述，本节中我们来看看复合初期最常见的错误心态。

有些人误以为保持联系就能确保复合，因此按捺不住兴奋，不断试探对方的态度，暗示复合的念头。他们有一句经典的口头禅：“你觉得我们还有机会吗？”此话一出，局面必然陷入尴尬，提问者也必定后悔。

你以为对方和你聊了几次天、泡了几次茶、说了几句真心话、流了几次泪，两人之间所有的不满就能全部化解吗？一切就能回到从前，对方也一定会答应你吗？别做梦了。这就像喝多了酒产生的幻觉，是不切实际的。

## 复合是一场双向奔赴的旅程 🤝

上一节我们指出了急于求成的危害，本节中我们来深入理解复合的本质。

复合是一场双向奔赴的旅程。所有的误会，不会因为你们几次交心就瞬间消失。对方或许在某一刻被感动了，看到你的改变，觉得两人可以继续下去。

但也会在下一秒，又想起你往日高高在上的状态，从而再次变得退缩，改变之前的态度。这种亲近、靠近后又疏离的模式，会在你们复合的过程中反反复复地上演。

## 复合过程中的正常波动 📉

理解了复合的反复性后，我们来看看过程中会出现的正常现象。

他会突然不回复你的信息。你会看到他朋友圈突然出现了新的异性。这些都是正常的，你不必觉得慌乱。这只是对方在“试错”，他在找寻到底哪个人才是真正值得他托付和信任的。你需要给他时间，你需要学会沉默，学会循序渐进的渗入。

## 提问的艺术：从封闭到开放 💬

面对复合中的波动，沟通方式至关重要。以下是两种关键的提问方式及其区别。

**核心错误提问公式：**
`你觉得我们还有机会吗？`

**核心正确提问公式：**
`你现在是怎么看我的？`

你别问“你觉得我们还有机会吗？”。你应该换个问法：“你现在是怎么看我的？”这两个问题的区别在于：前者是**封闭式提问**，后者是**开放式提问**。

封闭式提问的回答通常是“有”或者“没有”，对话可能就此结束。而开放式提问则不同，它问到了对方灵魂深处，促使他好好探究一下自己对你的看法。

你们一字一句地聊着复合，但你也在慢慢地引导他进入你的思维世界。这才是高手的做法。
