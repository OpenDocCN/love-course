# 挽回心理学课程 P1：多看效应 🧠💡

在本节课中，我们将要学习一个在情感挽回中非常实用的心理学概念——“多看效应”。我们将探讨如何利用这一效应，在不引起对方抗拒的前提下，逐步恢复联系并提升好感度，为复合创造有利条件。

## 决定复合后的常见困境

决定复合之后，通常会遇到两种情况。

第一种情况是犹豫不决，始终不敢迈出第一步，直到发现自己已经错过了最佳机会，最终无奈放弃。

第二种情况是想做就做，不顾及对方感受，立刻采取行动。

很明显，无论是哪种情况，你都很难达到自己期望的结果。但是第二种情况相对而言还要好一些，因为这说明你有足够的行动力和执行力去做这件事。对于第一种情况，我只能告诉你，自信心是自己给自己的。如果你自己都不相信自己可以成功，总是在想那些非常糟糕的情况，结果可能比你想象的还要糟。

## 复合初期的核心策略

综合这两种情况，我建议大家在决定复合之后，一定要切记：不要突然联系对方，更不要表现出自己想要复合的心态。否则，对方可能一开始就识破你想复合的动机，并且对你产生抗拒。

上一节我们介绍了复合初期的错误做法，本节中我们来看看正确的策略。

你要学会多利用“多看效应”，开始慢慢地恢复你在对方视线能及的空间内的存在感，以此增加对方对你的好感度。至少这么做，也不至于因为不知道该怎么跟对方接触，而迟迟没有办法迈出复合的第一步。

心理学家已经有实验验证过了：**你在对方眼前出现的频率和次数，与对方对你的好感度成正比**。你切记。

## 如何应用“多看效应”

以下是应用“多看效应”的具体方法。

你可以通过朋友圈、对方可能看到的社交平台，或者侧面通过你们共同的朋友，透露出你的信息和近况，来让“你”充满对方的生活。

不过你得注意，因为你是主动跟对方提出分手的，所以千万不要表现自己在没有他的日子里，过得逍遥自在，甚至好玩不止100倍。不要盲目相信“二次吸引”和自我提升，要根据自己分手的实际情况来发布内容。

以下是发布内容的具体建议：
*   可以发一些自己分手之后对情感的感悟。
*   可以发一些认识到自己不足之处的反思，显示出自己有一个很谦虚、想改变的态度。
*   可以适当表达对之前那段感情、对他曾经对你的好的怀念，但不要太直白，不要太“舔”。

频率大概控制在 **3~5天更新一条** 就行了。

## 效应的生效信号与停止时机

发到什么程度为止？发到对方主动给你评论，或者说给你发消息为止。

点赞这个事不能说明问题。点赞只能说明他看到了，并且有一定的情绪和影响产生。但是，只有对方主动地给你评论或者发信息，这才能够说明“多看效应”生效了，对方已经不排斥跟你进行沟通。
