# 分手后如何把握复合时机 🕒 - 课程 P1

在本节课中，我们将学习如何在分手后识别并把握最佳的复合时机。核心在于：在最佳时机到来前，做好充分准备。如果不做准备，你将毫无机会；如果做了准备，即便结果不尽如人意，你也至少拥有了参与的资格，做到无怨无悔。

上一节我们概述了准备的重要性，本节中我们来看看如何具体识别复联的时机。

## 识别复联时机

并非所有时刻都适合重新联系对方。当对方表现出强烈抵触情绪时，强行复联通常效果不佳。

以下是判断复联时机是否成熟的关键迹象：

*   **对方开始关注你的社交动态**：例如，对方会不时查看你的社交平台，或点进你的朋友圈。
*   **对方愿意进行简单交流**：对方能够接受并与你进行一些基础的、非冲突性的对话。

当以上迹象出现时，往往意味着一个较好的复联时机已经来临。因此，只有随时做好准备，才能在这些机会窗口出现时，有效地把握住它们。

本节课中我们一起学习了识别复合时机的核心原则：**准备先行，伺机而动**。关键在于通过观察对方的行为变化（如关注你的社交动态、接受简单交流）来判断时机是否成熟，并在时机到来前做好自身准备。
