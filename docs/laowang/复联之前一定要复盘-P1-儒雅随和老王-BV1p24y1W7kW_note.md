# 复联之前一定要复盘 - 课程P1 🧠

在本节课中，我们将学习如何为成功复合前任做准备。核心在于，复合不能仅凭感觉行事，而必须通过系统性地复盘过去的矛盾，识别并改变错误的相处与沟通模式。我们将分步骤解析具体方法，帮助你避免重蹈覆辙，建立更健康的关系基础。

## 避免凭感觉行事 ❌

复合前任时，切勿仅凭感觉行事。不要因为觉得对方对你有感情，就认为自己可以随心所欲。如果仅凭感觉行动，只会导致你重复过去的错误。

凭感觉行事，意味着你仍在沿用过去那种错误的模式与对方相处，其结果自然不可能好转。

## 复盘矛盾以改变模式 🔄

上一节我们指出了凭感觉行事的危害，本节中我们来看看如何通过复盘来改变模式。在尝试复合之前，你必须通过复盘过去的矛盾，来改变你们之间错误的相处模式。

以下是复盘与改变的具体步骤。

### 1. 识别引发负面情绪的行为

首先，你需要找出那些曾给你带来负面情绪、但你为了维持关系而未表达不满的行为。这些不满虽然未被言明，但其影响依然存在。

这些不满实际上标记出了你们之间不合适的相处模式。你现在要做的，就是从这些让你不满的行为中，反向推导出线索，从而找到对你而言最合适的相处模式。

**核心操作**：`列出事件 -> 记录你的情绪 -> 分析对方行为模式`

### 2. 发现无意识的错误相处模式

接下来，需要找出在你们关系中无意识形成的错误相处模式。

以下是几种典型错误模式的例子：

*   **回避型应对**：通过逃避来应对问题，但无法真正解决问题。
*   **无理取闹**：通过吵闹来换取对方的妥协。

只有找到这些问题，才有可能在未来的相处中，有意识地避开它们。

### 3. 修正错误的沟通模式

最后，需要审视沟通模式上的问题。你们的矛盾之所以得不到解决，很大程度上是因为沟通方式带有攻击性。

以下是几种常见的错误沟通方式：

*   **指责与卖惨**：将问题推到对方头上。
*   **讲道理**：用所谓的“常理”掩盖自己的真实诉求，以居高临下的方式绑架对方以满足自身需求，容易让人感到窒息。
*   **讨好**：在感情中，并非满足对方需求就能万事大吉。你在关系中的委屈，即使不说出来，也会被对方感知。对方感知到你的不满，你却用讨好来掩饰诉求，这会导致双方心理距离逐渐疏远。

**核心概念**：健康的沟通应遵循 **“表达感受 + 提出具体请求”** 的公式，而非指责或讨好。

## 总结 📝

本节课中，我们一起学习了为成功复合所做的关键准备。我们明确了不能凭感觉行事，并系统性地探讨了如何通过三个步骤来复盘与改变：识别负面行为、发现错误相处模式、以及修正攻击性或讨好式的沟通方式。这些结论源于大量经验总结，请务必反复理解并应用，避免再次踏入别人走过的误区。
