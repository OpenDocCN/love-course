# 复合教程 01：降低需求感，避免低位求复合 🧠

在本节课中，我们将要学习一个关于复合的核心误区：为什么不能通过放低姿态去乞求对方，以及如何通过降低“需求感”来建立平等、健康的复合基础。

---

关于复合，我发现很多人会陷入一个误区。他们认为既然要复合，就应该主动放低姿态，必须去联系对方，必须去哀求对方。至少要让对方感觉到分开之后，自己有多不舍、多痛苦、多不习惯。

在这里我要说，一定不要有这种念头。我一直强调，感情里边最怕的就是不对等。复合绝不意味着你放弃自己原则跟身段，去乞求对方，去讨好对方。你这样是断崴的感情，红肠走的不会太长远。因为你会很怀疑，如果有一天你不再顺从，这就会变成你把爱自身受的理由。

而导致高低位的直接因素，就是“需求感”。这个高低位，说通俗点儿，就俩字儿：**迁就**。处于低位的那个人，总是在迁就处于高位的那个人。

而需求感太重，最常见的原因就是“托付心态”太重了。

那么，什么是托付心态呢？就是我跟你谈恋爱，你理所当然的就得对我好。你是我男朋友，你就得无条件爱我，你就得关心我，就得照顾我，就得包容我，满足我的需要。任何时候，你都得在我身边。我要是过得不好，反正就是你没有好好对我。

朋友们，当你心中有很多很多“理所当然”的时候，那你就是有这个托付心态。更严重点儿就是：你是我男朋友，无论我做错啥事，你都得爱我，你都是爱我的。无论我怎么去闹，你都会包容我，你都会忍让我，都会对我好。或者说反正你是爱我的，反正你是不会离开我的，所以说呢，怎么做我都无所谓。再就是你是我男朋友，没了我，我啥都不行了，你不在我身边，我就活不起。

这个托付心态太重，男人就会觉得很累，就会想要逃离。

如果说你有严重的托付心态，而且你已经意识到了，那现在还不如去加以改善。那么你相信我，你以后谈的每一次恋爱，都会以十来高。

所以我要再一次地提醒朋友们，把你的这个需求感收一收。提升一下自己的价值和吸引力，让自己变得更好，才是复合的王道。

---

本节课中我们一起学习了复合的核心误区——通过放低姿态乞求复合。我们明确了“需求感”过强是导致关系地位不平等的关键，而“托付心态”则是需求感过重的根源。记住，健康的复合始于平等的姿态和自我价值的提升。
