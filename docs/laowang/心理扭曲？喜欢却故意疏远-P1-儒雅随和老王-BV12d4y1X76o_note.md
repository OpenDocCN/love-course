在本节课中，我们将要学习一种常见的人际关系模式——回避型依恋。我们将探讨其核心特征、形成原因以及行为背后的心理逻辑，帮助你理解为何有些人会表现出“喜欢却故意疏远”的矛盾行为。

## 回避型依恋的成因

上一节我们介绍了课程主题，本节中我们来看看这种行为模式的根源。回避型依恋并非心理扭曲或变态，而是一种典型的回避性行为模式。

其根本原因在于个体**从小缺乏安全感**。

## 回避型依恋的核心特征

了解了成因后，我们进一步剖析其具体表现。以下是回避型依恋在心理学上显现出的几个典型特征：

1.  **刻意远离亲密关系**：即使感知到对方的好感，也会主动疏远。
2.  **逃避或缺乏沟通能力**：常常将本应平和的沟通升级为争吵或直接沉默（“闭嗓子”）。
3.  **表达能力弱，喜用反话/反行为**：通过说反话、做与内心相反的事来表达情感，核心目的是**希望引起对方更多关注**。
4.  **缺乏安全感，对感情悲观**：认为感情脆弱不堪，不相信亲密关系能长期稳定。
5.  **鄙视或恐惧深情**：视“恋爱脑”为侮辱，害怕被感情束缚。
6.  **感情中缺乏自信，自我要求苛刻**：其逻辑是**只有强者才配被爱**，因此不容许自己犯错或出丑。
7.  **将克制与痛苦等同于安全**：并非不渴望被爱，而是认为**必须通过逃避这份爱，才能获得安全感**。其心理公式可概括为：
    `接纳爱 = 可能受伤害` => `逃避爱 = 安全`

## 行为背后的矛盾心理

综合以上特征不难发现，回避型依恋者对亲密关系的整体态度并非厌恶，而是一种**矛盾的接近**。

他们身披疏离的外壳，看似在“树敌”，但这层防御背后，是与常人相同的需求：他们也需要被爱、去爱，甚至更渴望他人的支持与温暖。

两者的根本区别在于应对方式：
*   **常人逻辑**：`喜欢` -> `接近`
*   **回避型依恋逻辑**：`喜欢` -> `但喜欢可能导致伤害` -> `因此必须逃避`

本节课中我们一起学习了回避型依恋的成因、七大核心特征及其背后的矛盾心理。理解这种“喜欢却疏远”的行为，关键在于看到其逃避背后对安全感的强烈渴求与对受伤的深刻恐惧。
