## 断联的作用与前提 🔄

当你选择断联时，对方可能会认为你将从此忘记他。人们常担心双方是否会真的成为陌生人。

如果分手后，对方对你的态度仍然处于极度排斥的状态，可以采用断联这种方式，来缓冲对方的负面状态和情绪。

## 断联期间的三个阶段 📊

断联的时间通常总会超过一个月。在此期间，对方的心理变化一般分为三个阶段。

### 第一阶段：压力释放与轻松感 😌

上一节我们提到了断联的缓冲作用，本节中我们来看看第一阶段的具体表现。

分手后的负面情绪，加上你直接的纠缠，对方一定会更加排斥。这个时候对方是真心希望你可以离开。

如果这个时候开始断联，对方所承受的负面心理压力会快速降低。他会感觉到轻松，因为你不再烦他了。

### 第二阶段：优越感消散与好奇心产生 🤔

在压力释放之后，对方的心理会进入一个新的状态。

如果说分手之后你曾卑微地挽回，那么就会让对方产生一种居高临下的优越感。而在断联之后，随着时间推移，这种优越感会逐渐落下。

他会突然察觉到，你为什么不再去联系他了。这种心理落差，可能会让他对你产生一定的好奇。

### 第三阶段：关注、回顾与判断 🔍

好奇心一旦产生，便会引发更深层次的思考。

只要他对你产生好奇，他就会开始关注你，并回顾你们的过往。

他就想要判断你是否开始了新的生活，有了新的选择，是不是真的完全放下他了。

## 重要说明与总结 📝

当然，前面所描述的这些心理变化并非绝对。毕竟人的心理是很复杂的。

因此，我们需要具体情况具体分析。

本节课中，我们一起学习了断联后前任心理变化的三个阶段：**压力释放期**、**好奇产生期**和**判断评估期**。理解这些阶段有助于管理自身预期，并为可能的后续互动做好准备。记住，心理反应因人而异，灵活应对才是关键。
