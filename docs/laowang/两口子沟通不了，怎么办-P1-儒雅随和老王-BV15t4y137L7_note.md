# 婚姻沟通课 01：如何有效沟通，避免争吵 🗣️

在本节课中，我们将学习婚姻中最常见的沟通难题——说两句就吵架，该怎么办。课程将聚焦于两个核心技巧，帮助夫妻，特别是女方，掌握有效沟通的方法，让对话变得顺畅。

---

很多夫妻感到沟通困难，对话几句就容易争吵。这是婚姻中最常见、被询问最多的问题。解决这个问题，主要依靠女方。沟通本是女性的强项，男性的短板。

但许多女性搞错了方向，导致两人总吵架。其实，女性只需要学会两招，沟通就会变得特别容易。

上一节我们指出了沟通问题普遍存在且可解决，本节中我们来看看第一个核心技巧：如何提出请求。

**第一，直接请求，不要命令。** 你可以要求丈夫做事，但不要以强势的命令口吻。

例如，你想让他帮你拿快递。错误的命令方式是：“你去给我拿快递！” 而正确的请求方式是：

```text
“老公，我给孩洗个澡，你帮我拿个快递行不？”
```

这个方式包含了**说明原因、提出请求、征求意见**，他就能听懂。他可能会不乐意或抱怨，比如“你干嘛不早说，我刚上楼又得下去”。

这时，你千万别气呼呼地跟他争执。听他抱怨几句就好。只要他没有明确拒绝，让他发泄一下情绪。发泄完，他往往就乖乖去了。等他走到楼下，早就忘记了不满，甚至可能打电话问你：“我给你买点西瓜、买雪糕，行不？” 这样大家都幸福。

大家都是人，都会有疲惫和不满。并非你一提出要求，他必须立刻执行，才能证明他在乎你。男人只是不喜欢听命令而已。

学会了如何提出请求，接下来我们看看第二个关键点：如何明确表达你的需求。

**第二，你要教他怎么做。** 有些妻子喜欢当裁判，从丈夫的各种行为判断他是否“及格”。例如，节日没送礼物，减一分；送的口红色号不对，再减一分。丈夫会莫名其妙地变成“不及格”。

换位思考一下，这有点难为人。你丈夫不是你肚里的蛔虫，哪知道你想要什么？

你想要他做什么，就直接教他。例如，你生理期肚子疼，丈夫在旁边打游戏，你感到不高兴和委屈。你可以直接跟他说：

```text
“老公，你给我泡个红糖水，我肚子疼。”
```

他听了就会去行动。下一回再遇到类似情况，他就知道该怎么做了。很多时候，他不是不想做，而是真不知道该怎么做。

都说男人来自火星，女人来自金星。恋爱时，他从他的星球来到你的星球。现在结婚后，他又回到了自己的星球，在用他星球的方式爱你。这并不代表他对你的爱减少了。

---

本节课中我们一起学习了两个改善夫妻沟通的核心技巧：**1. 使用“直接请求”而非“命令”**；**2. 明确“教导”而非“裁判”**。掌握这两点，能有效减少争吵，让沟通回归顺畅与理解。
