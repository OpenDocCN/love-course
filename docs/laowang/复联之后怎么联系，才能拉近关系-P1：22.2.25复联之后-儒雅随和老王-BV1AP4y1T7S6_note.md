# 复联关系推进指南 - 第一课：复联初期的联系策略 🫂

在本节课中，我们将要学习在感情复合初期（即“复联”后），如何根据对方的反馈来调整联系策略，以拉近彼此关系，避免给对方造成压力。核心在于识别不同的关系状态，并采取相应的、克制的沟通方式。

上一段我们提到了复联后首次联系的重要性。本节中我们来看看，当首次联系获得对方反馈后，应该如何进行后续的联系。

## 1. 联系的基本原则

首次联系后，无论对方反馈如何，你的初步目的已经达到。后续联系策略需根据双方当前的关系状态和对方的态度来决定。

## 2. 三种状态与对应策略

以下是三种常见的复联后关系状态及相应的联系策略。

### 状态一：联系畅通，对方不排斥

这种情况下，双方各种联系方式都在，对方不排斥甚至愿意与你保持联系。

**核心策略：避免过于主动，运用“延迟满足”。**

*   **具体做法：** 例如，如果你有14次想联系对方的冲动，只主动发起第一次联系。对于第二次冲动，应克制自己，转身去做别的事情。
*   **公式表达：** `主动联系次数 = 冲动次数 / N (N > 1)`
*   **目的：** 避免让自己显得情绪不稳定，同时不给对方造成压力和厌烦感。

### 状态二：态度冷淡，明确拒绝

对方可能表示“对你没感觉了，别再联系”，大部分联系方式已被删除，仅存一两个，对方态度不乐观。

**核心策略：大幅降低预期和存在感，给予情绪缓冲期。**

*   **错误回应示例：** “我就知道你别起了，咱才刚分手，你太无情了！”
*   **正确回应示例：** “我知道咱俩之间有一些问题，你这样想，我也能理解。没关系，我不强求。”
*   **联系频率控制：** 例如，十次想联系的冲动中，只保留最多三次实际联系。目的是尊重对方的边界，为化解矛盾创造空间。

### 状态三：强烈抗拒，极度反感

对方表现非常抗拒，说出“别纠缠我了，真烦人”等话，联系方式几乎全无。

**核心策略：冷静应对，进行观察和试探，理解抗拒背后的需求。**

*   **错误回应示例：** “对不起宝贝，我错了，我真的很爱你，你再给我一次机会好吗？”
*   **正确回应示例：** “抱歉，没想到会让你这么不舒服。我只是单纯想分享（某件事），如果你不愿意，那就算了吧。”
*   **关键行动：** 必须将联系频率降至极低水平。同时要学会“冷静的观察和试探”。因为分手后，对方越是表现出强烈的抗拒情绪，往往越说明其内心对你仍有需求和未化解的情绪，这种抗拒本身可能是一种测试。

---

本节课中我们一起学习了复联初期根据对方三种不同态度（不排斥、冷淡、抗拒）所应采取的联系策略。核心在于**克制需求感**、**尊重对方情绪**并**灵活调整联系频率**。记住，复联是一个循序渐进的过程，保持耐心和冷静至关重要。
