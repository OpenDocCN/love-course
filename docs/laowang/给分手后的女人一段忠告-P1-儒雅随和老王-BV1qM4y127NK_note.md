## 识别自私型伴侣的核心特征

上一节我们概述了本课主题，本节中我们来看看自私型伴侣的具体行为表现。以下是其核心特征的详细分析：

**1. 情感投入具有条件性**
自私的男人不会为感情过多投入。其行为完全取决于自身心情：心情好时一切尚可，一旦发生矛盾或心情不佳，便会立刻翻脸。

**2. 缺乏自省与归咎他人**
他永远不认为自己有错。即使口头道歉，内心也将过错归咎于你。其逻辑可概括为：**所有问题 = 你的问题**。

**3. 情绪不稳定且具有攻击性**
翻脸速度极快，不讲道理，暴躁任性。缺乏包容与理解，总是在向你索取。一旦产生情绪，便会故意打击、刺激和贬低你。

**4. 用双重标准进行情感绑架**
他会用一些“毒鸡汤”式的爱情标准来绑架你，要求你单方面付出，却根本体会不到你的爱与牺牲，只顾自己痛快和发泄情绪。

**5. 漠视你的感受与付出**
他完全不在乎你的感受。每次发生争执，当你试图缓和气氛、息事宁人，甚至忍着委屈主动低头哄他时，他仍会恶语相向，挖苦你。

**6. 扭曲你的善意与爱意**
他不会接受你的好。他会将你的爱与主动妥协，扭曲为你犯错后的愧疚表现，从而更加坚信一切都是你的错。其认知模式如下：

```python
你的妥协与付出 == 你理亏的证据 # 这是其扭曲的逻辑
```

## 核心结论与行动指南

综上所述，与极度自私的男人复合，通常意味着你将持续陷入一种被索取、被贬低且不被看见的消耗性关系中。你很难拥有改造他的能力，即使勉强在一起，也难以获得幸福。

本节课中我们一起学习了如何识别自私型前任的六大行为特征。关键在于认识到：你的认真与尊重值得被对等地对待。放过那个自私的人，同时也是放过自己，为真正健康、相互尊重的关系腾出空间。
