# 分手后到底应该怎么提升自己？-P1-儒雅随和老王-BV1Lk4y1t7Yx_note

在本节课中，我们将学习分手后自我提升的真正方向。许多人误将提升重点放在外在，却忽略了导致关系破裂的核心问题。我们将探讨什么才是有效的自我提升，以及如何通过提升内在能力来增强个人吸引力。

所有人都在告诉你，分手之后，你得努力改变自己，努力提升自己，就让他看着你这改变。但是说你好不容易，挑了一个出太阳的天，美美的把自己打扮起来，约过一秘厨去，还喝了个下午茶，拍个照，批了好几个点照片。

又纠结了一个多小时的文案，终于说发出了朋友圈。结果呢，最后没等着一个前任反应，反而有不少的好友给你点赞瓶子，还找你聊天。你不欣喜，你又试好几次，没等着前任的回应。这个就是你的方向有问题，你并没找到说，让你们分开的真正原因，没有给他一个说服自己，回头找你的理由。

上一节我们指出了方向错误的问题，本节中我们来看看为何外在提升效果有限。其实很多男人他并不在意伴侣外貌上的提升。他们愿意跟你在一块，那说明外貌条件各方面基本上是达标的，那就说明外貌条件各方面是过了关的。再往这个方向努力，这个意义它就不是特别大了。

所以说你得明白，分手之后，你这个自我提升，你提升的其实是你情绪价值，你情商，你共情能力。还有就是啥，你经营感情思维，跟你的认知。就这些东西，才是你核心竞争力。在这个时代，就美丽的皮囊，但是我千篇一律，但有趣的灵魂，那绝对是万里挑一。

他之所以说愿意跟你在一块，愿意跟你经营这段关系，更重要的指标，还是说你内在感情当中，一起解决问题的能力，抓住问题本质核心的能力，以及说一个良好的沟通模式等等。这些都是让他选择你的一个刚性指标。

为了帮助你自我评估，以下是几个关键的自省问题，用于检验你是否具备维系关系的核心能力：

*   当你看到对方对你宣泄负面情绪的时候，你能不能通过他满身是刺的外表，看出他受伤的内心？
*   你能不能从他的冷暴力、占有欲、违逼行为当中，看出他这么做的原因？比如说原生家庭的问题，或者说想起上一次有什么行为，让他感觉到没有安全感。
*   面对问题的时候，你有没有承担问题的责任感，以及说解决问题的方案？而不是你搁那块，一味的把问题都推给对方去解决。

重要的是**透过现象看到本质的能力**，而不被自己的情绪控制。就觉得他这个冷暴力就是不爱了，对我发脾气，那就是不尊重我。当然一味的忍让熬也没啥用。这两种极度，都是没有真正让你触碰到问题的核心。

真正好的感情，不是说完全的不争吵，而是说你们能在争吵当中找着问题，解决问题。所以说别人的争吵，反而能让感情升温。

分手之后到底应该提升的是啥？就是你的**情绪价值**。控制好你的情绪，用理性去找着真正的问题。如果说你还不明白怎么提升，白无主义看我以往的一个视频，咱说好好学习一下吧。

本节课中我们一起学习了，分手后的自我提升不应局限于外在改变，而应聚焦于提升情绪价值、共情能力、问题解决能力和理性思维。这些内在的核心竞争力，才是修复关系或开启新关系的真正基石。
