# 分手后，怎么才能让前任相信我-：22.3.7我不会再相信你了-

在本节课中，我们将要学习分手后，当对方说出“我不会再相信你了”时，问题的本质是什么。我们将探讨信任崩塌的感性原因与常理范畴的客观原因，并理解为何空洞的承诺无法奏效。

## 信任危机的普遍性

前任不相信你，无论你说什么，对方都不再相信。这是一个常见的问题，尤其在分手时，不信任几乎是标配。无论你道歉还是承诺改变，对方可能都不相信，甚至直接告诉你：“我不会再相信你了”。

## 信任崩塌的双重原因

上一节我们提到了信任危机的普遍性，本节中我们来看看导致这种不信任的具体原因。这里既有感性的、偏情绪的原因，这很容易理解。但同时，它也存在着客观原因，属于常理范畴。这部分客观原因往往容易被大家忽略。

无论你是想复合，还是想单纯地与前任维持友谊，先处理好常理范畴的事，才是最具性价比的调整方向。

## 常理范畴的客观原因

以下是常理范畴内，导致不信任的几个核心客观原因。

**1. 谎话的比例问题**

谎话说多了，自然就失去了信任。就像“狼来了”故事里的放羊娃，之所以不被信任，是因为他之前没有一句实话。

这是一个比例问题。每个人都说过谎，有善意的、恶意的或无意的。如果一个人**10句话里有8句假话**，那他就不被信任；如果**10句话里只有2句假话**，他可能依然被认为是正直的。这就是常理。对方凭什么相信你？这是一个需要你用行动回答的命题。

**2. 避免即时损失的本能**

一个人要相信另一个人，还有一个非常重要的点：**我不会因为相信而马上蒙受损失**。请注意“马上”这个词。至于以后会不会损失，可以另说。但如果我相信你，我立刻就能看到自己吃亏，那么无论你说的是对是错、是真是假，我本能上都很难接受。

这就是常理。例如，一个男生有女朋友，你整天粘着他，告诉他“你女朋友是个绿茶”，并列举各种证据。我们暂且不讨论这种行为是否是一种好的介入方式。单说这样的说法，对方是否能舒服地接受？很难。因为我相信你，就意味着**我马上就要损失利益**。

---

本节课中，我们一起学习了分手后信任危机的本质。我们了解到，不信任不仅源于情绪，更有其客观的常理基础，主要体现为**过往谎言的累积比例**和**对即时利益损失的恐惧**。空洞的语言承诺无法解决这些问题，因为信任需要实在的行动和时间的验证来重建。在下一课中，我们将探讨具体的应对策略。
