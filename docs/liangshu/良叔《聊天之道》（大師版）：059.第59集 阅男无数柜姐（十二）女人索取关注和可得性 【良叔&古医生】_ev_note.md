在本节课中，我们将学习如何识别并应对女性在互动中“索取可得性”的行为，以及如何通过“临时邀约”和“定框架、拉低预期”来维持健康的互动节奏。

上一节我们讨论了女生主动关心和线下良好表现后，我们提出了“做饭”的要求。本节中，我们来看看她如何拒绝，以及我们如何通过“临时邀约”和后续互动来应对她“索取可得性”的行为。

女生拒绝了“下厨”的要求后，我发了一张刚下班的照片。她问：“才下班吗？”我回复说今天休息，一会去市区。她接着问我的公司是否在市区。

以下是关于模糊邀约的关键点：
*   我直接询问：“你家住哪呀？”她发来了定位。
*   我进一步说：“你发个位置，我看看距离。”在模糊邀约时，应避免目的性过强的废话，首要关注实际距离。如果距离过远，则无需强求。

她主动问我几点结束。我回复“不多，结束了”，并告知出门需要半小时。这时，她提出了邀约：“要一起吃晚饭不？”我同意了。

以下是女生发出邀约时值得注意的细节：
*   她说：“你选地方吧，不吃辣的，我收拾收拾。”这表明她记住了“我不吃辣”的信息。
*   这个细节判断出女生对你上心，并且是以你的感受为中心，意味着吸引力很强。这与许多学员以女生为中心、显得不够果断的状态相反。

我回复对附近不熟，让她选地方。她发来定位后，我出发并告知“快到了”。她回复：“你要等我，我还在弄。”

我直接表达：“不是去选美，扭着屁股来就行了。”这传递了“不想等太久”的意图。她回复十分钟后见。见面后，我发消息说“好乖，我要继续忙了”。她回复“加油搬砖，按时吃饭饭，饿了也可以吃宵夜”，这表明她了解我的习惯。

我再次提出：“想吃你做的皮蛋瘦肉粥。”她开始推开了。

以下是女生推开时的表述和核心原则：
*   她说：“你要接受我有缺点，不会做饭，也不愿意进厨房。我们太甜了，不好，容易迷失自己。”
*   **核心原则**：此时她可能因线下了解到我有家庭等情况，产生羞耻感，在维持自己的框架。**关键点是必须给她时间和空间，绝对不要做以下几件事**：
    1.  **不要**用逻辑去说服她。
    2.  **不要**长篇大论或写小作文。
    3.  **不要**乱给承诺或可得性。
    4.  **不要**逼她做选择。

我只回复了一个“OK”。当她感觉到我没有解释或逼迫时，她对我的打开程度反而可能增大。因为在她认知里，这种状态不对，但关系已进展到一定程度，突然放手双方可能都不适应。

于是，她又发来一段话，这也是本节课的重点：“我不和你这样发展下去了，我需要平凡的生活，找一个爱我到骨子里面的人，简简单单。以后不约了吧。但是还是可以找我买东西的 😏。”

**本节课的核心重点：定框架与拉低预期。**
我之所以不回应、不解释，是为了“拉低预期”。即拉低这个女人对以下各方面的预期：
*   对“我”这个人的预期。
*   对这段关系发展的预期。
*   对我所有要求、语言行为、聊天方式的预期。
*   对吃饭、见面、约会乃至性等方面的预期。

她这段话的本质是在“索取可得性”，即要求你给出明确的承诺（如表白、说爱她）。此时，依然不能给这样的承诺。

我再次只回复了一个“OK”，然后不再回复。这依然是给予时间和空间。她说要停，我就让她停。为什么要给时间？因为必须由她自己想清楚是否要在一起，这是她的选择。同时，由于我没有给过任何承诺，也就不会构成欺骗或伤害。

许多学员喜欢给女人乱承诺，最终往往导致关系破裂，一无所获，还会让女人痛恨你，你自己也会陷入自责。言而无信、答应的事做不到，对双方关系有弊无利。本节课就是教大家如何正确处理这类情况。
