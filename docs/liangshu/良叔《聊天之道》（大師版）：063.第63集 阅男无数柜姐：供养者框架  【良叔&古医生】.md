# 良叔《聊天之道》大师版：第63课 👩💼 阅男无数柜姐（十六）—— 女人的阴暗面：供养者框架

在本节课中，我们将学习在聊天互动中，如何应对女性情绪反复、主动邀约后被推开，甚至最终被删除的情况。我们将通过一个具体案例，解析“供养者框架”下的心理博弈，并学习如何保持自身框架的一致性。

---

上一节我们讨论了如何推开对方以测试关系。本节中，我们来看看在推开之后，对方主动发起联系时，应如何应对其情绪化的行为，并处理其反复无常的态度。

以下是本回合聊天的核心过程与分析：

对方以关心“嗓子好了吗”为话题重新开启对话，并提供了大量生活建议，表现出过度操心。

此时，我的回复保持精简，并未表现出高涨的热情。这样做是为了“收着能量”，因为当对方主动时，若我方反应平淡，会促使对方投入更多精力来获取关注和认可。

对方继续投入，询问饮食、分享自己带饭，并试图通过夸奖进行互动。我则保持一致性，适当给予注意力，但不过度热情。

对方主动提出邀约：“回来我带你吃东西”。我通过提问“养好之后要干嘛呀”进行轻度测试，将对话引向不正经、无厘头的方向，这是为了营造轻松氛围，避免陷入严肃的逻辑谈判。

当对方抱怨工作辛苦、工资低时，我没有落入“比惨”或追问细节的陷阱，而是用“我们俩是在比穷吗”将话题轻松带过，避免陷入对方的负面情绪框架。

对方持续分享生活细节（加班、吃烧烤），并再次主动邀约。我表达了个人偏好（吃不了太油），同时展示了内在特质（不浪费）。对方则持续尝试升级关系（如发送“抱抱勤劳的小蜜蜂”）。

我以“下次抱就不知道是什么时候了”轻微推开，并表达对城市“少了点什么”的感受。对方接话，暴露情感需求：“我习惯有人陪着…不然感觉很孤单，没有安全感”。

我实施关键推开：**“找多几个备胎难度不大…我给不了你你可以去找别人。”** 直接点明其“供养者框架”思维——即她可能在寻找能提供安全感的伴侣。

对方表现出矛盾，声称“备胎…浪费彼此时间”。我进一步戳破其前后不一：**“你前不是说不联系了吗？”** 旨在打破其摇摆不定的框架，要求明确。

对方承认“打脸了”，但态度依然迷茫。我建立清晰框架：**“我不要模棱两可…你希望我怎么配合你？”** 要求对方停止消耗彼此情绪的内耗行为。

最终，对方在情绪反复中表示“不找就不找”，在我再次强调“不喜欢内耗”后，她将我删除。这个结果显示了对方无法处理内心矛盾（感性依赖与理性放弃），情绪最终爆发。
