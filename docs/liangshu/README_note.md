# 良叔的Python入门课 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据。标签就是变量的名字，我们通过这个名字来找到并使用盒子里的数据。

在Python中，给变量赋值非常简单，只需要使用等号 `=`。

```python
my_name = "良叔"
my_age = 30
```

这里，我们创建了两个“盒子”：一个名叫 `my_name`，里面存放了文字“良叔”；另一个名叫 `my_age`，里面存放了数字30。

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有一些内置的基本数据类型，最常见的有以下几种：

以下是三种最基础的数据类型及其示例：

1.  **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
2.  **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
3.  **字符串**：由字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘你好’`, `“Python”`。

## 如何查看和修改变量？ 🔄

我们知道了如何存放数据，接下来看看如何取出和使用它们。

你可以直接打印变量的内容，也可以对其进行运算和修改。

```python
# 打印变量的值
print(my_name)  # 输出：良叔
print(my_age)   # 输出：30

# 进行数学运算
new_age = my_age + 1
print(new_age)  # 输出：31

# 修改字符串变量
greeting = “你好”
greeting = greeting + “，世界！”
print(greeting) # 输出：你好，世界！
```

## 动态类型特性 ⚡

Python有一个重要特性叫“动态类型”。这意味着你不需要提前声明变量会存放什么类型的数据，并且同一个变量可以在程序运行过程中被重新赋值为不同类型的数据。

```python
# 变量x先存放一个整数
x = 10
print(type(x))  # 输出：<class ‘int’>

# 之后，同一个变量x可以改为存放一个字符串
x = “现在我是文字了”
print(type(x))  # 输出：<class ‘str’>
```

`type()` 函数可以帮助我们查看变量当前的数据类型。

## 变量命名规则 📝

为了写出清晰易懂的代码，给变量起一个好名字至关重要。

以下是给变量命名时需要遵守的核心规则和良好习惯：

*   **可以包含**：字母（大小写敏感）、数字、下划线 `_`。
*   **不能以数字开头**：`1variable` 是无效的，`variable1` 是有效的。
*   **不能使用Python关键字**：例如 `if`, `for`, `while` 等具有特殊功能的词不能用作变量名。
*   **建议使用描述性的名字**：使用 `user_age` 比 `ua` 或 `a` 要好得多，能让代码一目了然。
*   **常见命名风格**：
    *   蛇形命名法：多个单词用下划线连接，如 `my_variable_name`（Python推荐）。
    *   驼峰命名法：除第一个单词外，后续单词首字母大写，如 `myVariableName`。
