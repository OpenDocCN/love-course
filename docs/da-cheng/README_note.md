# 大橙 🍊

在本教程中，我们将学习如何制作一杯名为“大橙”的饮品。我们将从准备材料开始，逐步讲解制作步骤，并最终完成这杯清爽的饮料。

## 1. 准备材料 🛒

上一节我们介绍了本教程的目标，本节中我们来看看制作“大橙”需要哪些材料。

以下是制作一杯“大橙”所需的全部材料清单：

*   一个新鲜的大橙子
*   200毫升纯净水
*   10克白砂糖（可根据口味调整）
*   5-6块冰块
*   一片薄荷叶（用于装饰，可选）

## 2. 处理橙子 🔪

准备好材料后，下一步是处理核心水果原料——橙子。

以下是处理橙子的具体步骤：

1.  将橙子彻底洗净。
2.  用刀将橙子切成两半。
3.  使用榨汁器或手动挤压，将橙子榨出果汁。
4.  用滤网过滤果汁，去除果肉和籽，得到约150毫升的纯净橙汁。

## 3. 混合与调味 🥤

现在我们已经得到了纯净的橙汁，接下来进行混合与调味。

此步骤的核心是**将橙汁、水和糖充分混合**。其混合比例公式可表示为：

**最终饮品 = 橙汁 + 水 + 糖**

具体操作如下：

1.  取一个干净的杯子。
2.  将过滤好的橙汁倒入杯中。
3.  加入200毫升纯净水。
4.  加入10克白砂糖。
5.  用搅拌棒或长勺搅拌，直至砂糖完全溶解。

## 4. 最终装饰与享用 🌿

饮品的主体已经完成，最后一步是为其增添风味与视觉效果。

以下是装饰与享用的建议：

*   向杯中加入5-6块冰块。
*   在杯口插上一片洗净的薄荷叶进行装饰。
*   插入吸管，即可立即享用这杯自制的“大橙”饮品。

---

本节课中我们一起学习了如何制作“大橙”饮品。我们了解了所需的材料，掌握了处理橙子、混合调味以及最终装饰的完整步骤。现在，你可以亲手制作一杯清爽可口的大橙饮料了。
