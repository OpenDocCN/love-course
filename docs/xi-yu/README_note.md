# 课程名称：西班牙语基础入门（第1课）📚

在本节课中，我们将要学习西班牙语的基础知识，包括字母发音、问候语以及简单的自我介绍。课程内容设计简单直白，旨在帮助初学者建立最初的语言认知。

## 字母与发音 🔤

西班牙语共有27个字母，其中包含一个特有的字母 **ñ**。其发音体系相对规则，大部分字母的发音是固定的。

以下是西班牙语字母表及其近似发音示例：

*   a - 发音类似中文的“啊”
*   b - 发音类似中文的“贝”，但双唇闭合不爆破
*   c - 在a, o, u前发“嘎”音，在e, i前发“塞”音
*   d - 发音类似中文的“得”，舌尖轻触上齿背
*   e - 发音类似中文的“诶”
*   f - 发音类似中文的“艾菲”
*   g - 在a, o, u前发“嘎”音，在e, i前发“赫”音
*   h - 不发音
*   i - 发音类似中文的“伊”
*   j - 发音类似中文的“赫”，喉音较重
*   k - 发音类似中文的“嘎”
*   l - 发音类似中文的“艾莱”
*   ll - 发音类似中文的“耶”或“杰”
*   m - 发音类似中文的“艾梅”
*   n - 发音类似中文的“艾内”
*   **ñ** - 发音类似中文的“捏”
*   o - 发音类似中文的“哦”
*   p - 发音类似中文的“贝”，双唇轻碰
*   q - 总是与u连用，qu发“嘎”音
*   r - 发音需要舌尖颤动，单颤音
*   rr - 发音需要舌尖强烈颤动，多颤音
*   s - 发音类似中文的“艾塞”
*   t - 发音类似中文的“得”，舌尖抵上齿
*   u - 发音类似中文的“乌”
*   v - 发音与b相似
*   w - 仅用于外来词
*   x - 发音类似中文的“艾吉斯”
*   y - 发音类似中文的“伊格里埃加”，可作元音
*   z - 在大部分地区发音类似中文的“塞”

上一节我们介绍了字母的基本发音，本节中我们来看看如何运用这些知识进行简单的问候与自我介绍。

## 基本问候与自我介绍 👋

掌握问候语是开始任何对话的第一步。西班牙语的日常问候直接且富有节奏感。

以下是常用的问候与自我介绍句式：

*   **¡Hola!** - 你好！（通用问候）
*   **Buenos días.** - 早上好。
*   **Buenas tardes.** - 下午好。
*   **Buenas noches.** - 晚上好/晚安。
*   **¿Cómo estás?** - 你好吗？（非正式）
*   **¿Cómo está usted?** - 您好吗？（正式）
*   **Muy bien, gracias. ¿Y tú?** - 很好，谢谢。你呢？（非正式）
*   **¿Y usted?** - 您呢？（正式）

进行自我介绍时，最核心的句型是“我叫...”。在西班牙语中，这通过动词 **llamarse** 的变位来实现。

**Llamarse** 是一个反身动词，意为“名叫...”。其现在时第一人称单数变位为 **me llamo**。

因此，“我叫玛丽亚”的完整句子是：
**Me llamo María.**

你也可以用更简单的句式：
**Soy María.** （我是玛丽亚。）

询问对方名字可以说：
**¿Cómo te llamas?** （你叫什么名字？非正式）
**¿Cómo se llama usted?** （您叫什么名字？正式）

本节课中我们一起学习了西班牙语的字母发音规则、日常问候用语以及如何用 **me llamo** 这个核心句型进行自我介绍。这些是构建西语对话最基础的砖石。反复练习发音和这些基本句式，将为后续的学习打下坚实的根基。
