# 爱情很简单，复杂的是人 - 课程P1：追求阶段的核心原则 🎯

在本节课中，我们将学习在追求心仪对象时，为何应避免过度付出，特别是经济上的付出。我们将深入分析其背后的五个关键原因，并明确在追求阶段真正应该专注的两件事。

---

追求阶段的核心策略是：**只做好“聊天”和“约会”两件事**。其他任何额外的、尤其是物质上的付出，都可能适得其反，为关系埋下隐患。

以下是过度付出可能带来的五个主要问题：

**第一，破坏关系的平衡性。**
爱情的本质是双向奔赴。如果你单方面“玩命”对他好，他就失去了表现和付出的空间。这种压抑感可能促使他转向他人去释放情感。

**第二，容易惯坏对方。**
你可以关心体贴，但不能将他捧上神坛。过度付出会模糊你的定位——你是在寻找平等的伴侣，而非需要供奉的“祖宗”。

**第三，设定不可持续的高预期。**
人的精力有限，不可能永远保持巅峰的付出状态。一旦你因疲惫而稍有怠慢，对方很容易认为“你变了”或“之前的热情都是伪装”，从而心生怨恨。

**第四，可能吸引错误的人。**
如果你主要通过花钱来维系关系，那么吸引来的可能是贪图物质而非真心喜欢你的人。这会导致“身在曹营心在汉”的局面，你无法获得真正的情感回报。

**第五，丧失个人魅力与尊严。**
人性倾向于幕强。过度卑微的付出会让对方在内心轻视你。当他不再将你视为一个独立、有魅力的个体，而是看作一个依附者时，爱情便难以产生。他喜欢的应是完整的你，而非失去自我的“附属品”。

---

上一节我们剖析了过度付出的风险，本节我们来总结正确的行动纲领。

爱情本身可以很简单，复杂的是人的行为和心态。如果你在追求过程中总是感到疲惫不堪，问题通常不出在爱情本身，而在于：**方法错了，或者人错了**。

**核心行动公式：**
```
追求阶段的有效投入 = 高质量的聊天 + 有吸引力的约会
```

**关键代码原则：**
```python
# 错误的做法：过度付出
def pursue_wrong(person):
    spend_money_excessively(person)
    sacrifice_self_respect(person)
    # 结果：关系失衡，魅力丧失

# 正确的做法：专注核心
def pursue_correct(person):
    engage_meaningful_chat(person)  # 专注聊天
    plan_engaging_dates(person)     # 专注约会
    # 结果：平等互动，自然吸引
```

本节课中，我们一起学习了追求阶段应避免过度付出的五大原因，并明确了应专注于“聊天”与“约会”这两个核心环节。记住，建立吸引的关键在于展示你作为一个独立个体的魅力，而非通过单方面的牺牲来换取好感。
