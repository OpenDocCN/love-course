## 应避开的三种活动

以下是三种在初次约会中应尽量避免的活动类型及其原因。

### 1. 大排档撸串/小龙虾

首先，应避开大排档撸串或吃小龙虾这类活动。享用这类食物时，个人的吃相容易变得狼狈。稍有不慎就容易出丑。同时，女生可能碍于自身形象，不敢放开吃。这会导致双方的约会体验感变差。

### 2. 逛街

上一节我们介绍了饮食类活动的弊端，本节中我们来看看消费类活动。第一次约会就选择逛街，不仅容易造成尴尬局面，还很容易伤害你的钱包。原因在于：当女生购物时，你在旁边很难处理。不买单会显得你很小气，买单则很容易让花费打水漂。如果遇到品行不端的女生，你的投入更可能如同“肉包子打狗”——有去无回。

### 3. 电影院（传统影院）

接下来，我们分析一项常见的约会选择——看电影。你必须牢记第一次约会的目的：是互相了解对方，并为对方留下一个好印象。在传统电影院看电影，会浪费掉宝贵的两个小时交流时间，阻碍你们达成约会核心目标。不过，这里存在一个例外情况。

**例外情况：私人影院**

**私密空间 + 放松环境 ≈ 更开放的交流**
