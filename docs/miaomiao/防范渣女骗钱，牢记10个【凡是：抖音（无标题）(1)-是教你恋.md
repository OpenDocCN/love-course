## 核心概念与行为公式

在了解了具体现象后，我们可以尝试提炼一个简单的判断逻辑。其核心在于观察对方**行为动机**与**关系进度**是否失衡。

一个需要警惕的关系模式可以用以下逻辑表述：

```
如果 (对方的行为 过度聚焦于 索取经济利益) 且 (关系阶段 明显早于 正常互信程度) 则
    触发警示信号
```

这并非精确的数学公式，而是一种决策思路：当对方在情感基础薄弱时，过早、过多地将互动引向经济索取，你就需要提高警惕。
