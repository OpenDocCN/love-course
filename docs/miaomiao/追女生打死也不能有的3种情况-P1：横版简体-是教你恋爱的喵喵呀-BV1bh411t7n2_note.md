## 情况一：目的性过强与“恋爱脑”发作 🧠

上一节我们概述了本课的主要内容，本节中我们来看看第一种需要避免的情况：目的性过强。

*   发送质问式消息，例如：“你是不是不喜欢我？”

*   指责：“你要是对我没意思，为什么总跟我说话？”

这些行为如同“被僵尸吃掉脑袋”，缺乏理性，只会将对方推得更远。
