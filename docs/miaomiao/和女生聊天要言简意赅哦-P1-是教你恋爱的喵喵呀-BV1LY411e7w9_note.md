## 善用短句，删除语气词

**核心操作**：`对话内容 - 冗余语气词 = 简洁表达`

上一节我们介绍了使用短句的重要性，本节中我们来看看需要避免的具体情况。尤其是在“写小作文”这种行为上，即女生只说了一句话，你却回复一大段文字；或是女生提出一个问题，你进行过度冗长的解释。

因此，必须精简对话，尽量减少字数，以维持双方能量的平衡。如果仅靠多说几句话就能让女生喜欢上你，那大家就都去写小说了。现实并非如此。
