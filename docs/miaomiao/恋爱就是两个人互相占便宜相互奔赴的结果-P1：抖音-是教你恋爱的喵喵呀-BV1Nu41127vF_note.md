# 恋爱教程 第1课：主动出击与相互奔赴 🎯

在本节课中，我们将要学习一种关于恋爱关系的观点：恋爱是双方“相互占便宜”并共同“奔赴”的结果。我们将探讨如何通过主动、果断的行为来推进关系，避免陷入无效付出的困境。

## 核心观点：主动“占便宜”以推进关系

上一节我们介绍了课程的主题，本节中我们来看看其核心主张。想成功开始一段恋情，需要采取主动姿态，抓住机会拉近物理与情感距离。其行为逻辑是：在尊重对方的前提下，尽可能快速地升级亲密接触，以此测试对方的态度并定义关系。

以下是具体的行为层级建议：
*   **能牵手就别只并肩行走。**
*   **能拥抱就别只停留在牵手。**
*   **能搂腰就别只满足于拥抱。**

**自私**（此处指优先考虑自身恋爱需求与效率）的人往往能更轻松地追求到心仪对象。不应表现得过度怯懦或被动，否则可能导致投入大量时间与金钱后，关系却毫无进展。

## 行动的意义：测试反馈与筛选对象

我们已经了解了主动行为的层级，接下来探讨这些行动背后的重要意义。肢体接触的升级是一种关键的测试手段。不尝试接触，就无法知晓对方的真实感受；不推动对方做出选择，关系就可能永远停滞不前。

以下是主动行为带来的两种核心结果：
*   **遇到真诚的女孩**：若你尝试牵手即被拒绝，这能帮助你及时止损，避免后续更多的情感与物质投入。
*   **遇到不良的伴侣**：若对方只愿接受付出（如吃饭、礼物）却始终拒绝关系升级，这能促使你认清现实，尽快抽身离开。

## 避免陷阱：付出与占便宜的平衡

上一部分我们讨论了行动如何用于测试，本节我们来看看不行动的后果。如果面对合适的对象却一味拖延，只知单向付出而不敢“占便宜”（即推进关系），可能会产生反效果。对方可能会因此冷静下来，进行过于理性的分析，最终可能以“你是个好人”为由拒绝你，转而接受另一个敢于主动升级关系的追求者。

好的爱情关系，本质上是**双向的“占便宜”与“相互奔赴”**。单方面付出而不求关系进展，就像是 **`自己挖坑自己跳`**，是一种自我损耗的行为。

---

**本节课总结**：我们一起学习了在恋爱初期通过主动、逐步升级的肢体接触来测试对方心意、筛选合适对象并快速定义关系的重要性。核心在于把握“相互占便宜”与“共同奔赴”的平衡，避免陷入单方面付出却无法推进关系的陷阱。记住，果断的行动比被动的讨好更能有效地开启一段恋情。
