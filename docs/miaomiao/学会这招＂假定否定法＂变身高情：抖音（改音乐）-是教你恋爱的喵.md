# 学会这招＂假定否定法＂变身高情：抖音（改音乐）-是教你恋爱的喵

在本节课中，我们将学习一种名为“假定否定法”的高效聊天技巧。这种方法能帮助你避免封闭式提问，让对话更自然、更有趣，从而提升与女生聊天时的互动质量。

## 什么是封闭式提问？ 🤔

上一节我们介绍了课程目标，本节中我们来看看需要避免的聊天方式。如果你和女生的聊天风格是不断直接提问，那么对话容易陷入僵局。

封闭式提问是指那些只能得到简单肯定或否定回答的问题。例如：

**代码示例：**
```
你吃饭了吗？
```

对方只能回答“吃了”或“没有”。接着你可能会问“吃了什么？”，对方回答“蛋炒饭”。这种一问一答的模式就是封闭式话题，它会让对话变得枯燥，难以深入。

## 高手如何提问：假定否定法 💡

了解了封闭式提问的局限性后，本节我们来学习高手的聊天方法。当你想要了解对方情况时，应使用“假定否定法”去提问。

这种方法的核心是：**用陈述句的形式，表达一个带有否定意味的猜测**。目的是开启话题，而非索取信息。

**公式：**
`我猜你（现在/肯定）没有在XXX` 或 `我猜你（现在/肯定）不是XXX`

例如，高手会这样说：
```
我猜你现在肯定没有窝在沙发上撸猫刷剧。
```

这时，女生的回复可能是“没有呢，刚吃完饭”。这样，我们不仅得到了她的状态（刚吃完饭），还获得了一个可以继续聊天的切入点（吃饭、休闲活动）。

## 如何延续对话：分享与共情 🌊

我们已经学会了用“假定否定法”打开话题，接下来看看如何让对话流畅进行。关键在于利用得到的切入点，分享自己的生活并共情对方的感受，以达到同频共振的效果。

例如，当得知女生刚吃完饭，你可以这样回应：
```
有点羡慕你。我现在刚加完班，感觉要走不动路了。
```

以下是这种回应的优点列表：
*   **提供谈资**：将话题从“她”延伸到“你”和“你们”，创造了更多聊天素材。
*   **输出价值**：通过分享自己的状态（加班、累），展示了你的生活和工作态度。
*   **调动情绪**：“羡慕你”表达了情感，让对话不再枯燥。
*   **引发共情**：对方可能会对你的状态产生回应，从而加深连接。

这样，聊天话题就被自然地打开了，避免了那种不断索取信息的无聊模式。

## 为什么不要连续提问？ 🚫

掌握了开启和延续对话的方法后，我们还需要理解为什么旧模式行不通。请你们想一想，如果女生一直面对连环提问，她们脑子里想的是：

以下是女生可能的心理活动列表：
*   我跟你熟吗？
*   我为什么要花时间回答这么多无聊的问题？

当她们产生这种想法时，就可能突然不想回复，然后结束对话。因此，聊天的目的不是追问，而是：
1.  在对话中寻找可延展的话题作为谈资。
2.  借此输出自身的价值感。
3.  调动对方的情绪。
4.  最终实现共情。

而不是无脑地追问对方问题。
