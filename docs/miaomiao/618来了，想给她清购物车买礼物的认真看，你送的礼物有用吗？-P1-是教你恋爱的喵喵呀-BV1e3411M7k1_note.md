# 恋爱教程 01：如何送出“有用”的礼物？🎁

在本节课中，我们将要学习一个关于送礼的核心观点：礼物是否有用，并不取决于其价格或形式，而取决于送礼者的姿态与目的。我们将分析三种常见的送礼心态，并指出其中唯一有效的一种。

---

礼物本身并不重要，而真正重要的是送礼物时你的姿态。你送的礼物有没有用取决于你的目的。

男生给女生送礼物有三种目的，而能成功的只有最后一种。

上一节我们介绍了送礼的核心在于姿态与目的，本节中我们来看看第一种常见但无效的送礼心态。

**第一种：装卑势。**
有些人会砸钱去弥补他自身魅力、软价值的不足，从而对女人达成一种交易的关系。这种人是最爱去物化女性的，但是其实他们内心很自卑。他不认为自己的内核能够吸引到女人，所以他就要直接花钱来购买对方的时间。实际上两个人最后就达成了一种默契，谁也不挑明这种交易关系。

了解了以交易为目的的送礼后，我们再来看看第二种试图以情感打动对方的心态。

**第二种：借助送礼物去感动对方。**
他们想的很简单，只要我够真诚就能打动你。但实际上女人会觉得，你除了这种小恩小惠以外，没有办法提供真正的价值。所以你送得越多，她们就越看不起你。

以上两种心态都难以成功。那么，什么是正确的心态呢？接下来我们看看第三种，也是唯一能够成功的送礼心态。

**第三种：也是唯一能够成功的，妇族心态（提供价值心态）。**
他只是去撩女生，而不是去追女生。他送的礼物是他硬价值、品味、自信融在一起的一个 **`sample`（样品）**。他愿意给女生一个机会去品尝一下他的味道。而他本身是不会在乎，更不会去恳求，说你收了我的礼物你就要怎么怎么样。

你们需要明白的就是这一点：很多时候，礼物本身并不重要，而真正重要的是 **送礼物时你的姿态**。

---

本节课中我们一起学习了送礼的三种心态。关键在于理解，礼物只是一个载体，其效用完全由你送礼时的姿态和目的决定。摒弃交易或讨好心态，以自信、提供价值的姿态去表达自己，才是让礼物变得“有用”的唯一方法。
