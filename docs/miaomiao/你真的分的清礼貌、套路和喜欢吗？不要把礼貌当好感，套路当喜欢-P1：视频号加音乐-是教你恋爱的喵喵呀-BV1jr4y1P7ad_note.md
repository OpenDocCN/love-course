# 恋爱基础课 01：分清礼貌、套路与喜欢 💖

在本节课中，我们将要学习如何区分人际交往中的三种不同“好感”：礼貌、套路和真正的喜欢。许多人之所以在恋爱中屡屡受挫，正是因为混淆了这三者的本质区别。掌握这些概念，能帮助你更准确地判断对方的心意，避免不必要的误解和尴尬。

## 礼貌：成年人的基本社交礼仪 👐

上一节我们介绍了课程的核心目标，本节中我们来看看第一种类型：礼貌。礼貌是成年人之间一种基本的社交理解行为。

以下是礼貌行为的具体表现：

*   你发消息，对方会回复，但回复字数很少，或者间隔很久才回复。
*   你感觉聊天进行不下去，话题难以深入。
*   如果对方连回复都不愿意，那说明连基本的礼貌都不想维持。

此时，如果你误将对方的礼貌理解为好感并发起“进攻”，结果几乎百分之百会被拒绝。例如，许多人仅仅因为对方回复了一条消息或给朋友圈点了个赞，就产生过多幻想，这是需要避免的误区。

## 套路：暧昧与不确定性交织的迷雾 🎭

了解了基础的礼貌行为后，我们进一步探讨一种更复杂的情况：套路。套路通常伴随着暧昧和容易让人多想的行为，许多“海王”或“绿茶”擅长此道。

以下是套路行为的典型特征：

*   对方会主动接近你、找你聊天、甚至暗示约会。
*   当你逐渐上钩并对ta产生兴趣后，对方开始有意无意地对你忽冷忽热。
*   这种若即若离让你“上头”，误以为ta喜欢你，从而表白。
*   结果往往换来一句：“我只是把你当朋友。”

**公式：套路 ≈ 20%的暧昧小行为 + 80%的不可预测与不确定性**

套路的核心在于，除了零星的、让你心动的小行为外，缺乏连续的、能让你明确感受到被喜欢的行为。对方的目的可能只是为了证明自己的魅力，或者另有所图（如物质利益）。一旦你表白，就等于落入了对方的套路，ta的目的达成后，便会抽身离开。

## 喜欢：藏不住的情感与连续的行动 ❤️

那么，什么才是真正的喜欢呢？记住一个核心观点：喜欢一个人是藏不住的，它会通过持续而明确的行为表现出来。

以下是真正喜欢一个人的行为标志：

```text
if (对方喜欢你) {
    回复消息热情、迅速，甚至“秒回”；
    主动与你分享生活点滴；
    即使你聊天很“尬”，ta也会努力开启新话题继续聊；
    你发出邀请时，ta总是有空，或主动提出其他可行时间；
}
```

当你在接触中能持续感受到以上这些信号时，才说明对方可能对你有真正的兴趣。在这种情况下，你的主动推进才更有可能获得成功。

---

本节课中，我们一起学习了如何区分礼貌、套路和真正的喜欢。关键在于观察对方行为的**连续性**和**明确性**：礼貌止于基本回应，套路充满暧昧与不确定性，而真正的喜欢则会通过热情、主动且持续的行动流露出来。希望你能运用这些知识，更清醒、更自信地面对人际与情感关系。
