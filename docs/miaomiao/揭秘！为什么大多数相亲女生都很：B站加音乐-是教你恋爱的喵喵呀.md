## 社会期待与主动责任链 🔗

上一段我们提出了问题，本节中我们来看看第一个核心原因：社会对男性角色的固化期待。在相亲市场中，存在一条被广泛默认的“主动责任链”。

*   **发起接触**：通常由男性主动添加女性微信，并由男性先发起对话。
*   **维持互动**：聊天话题的开启与延续由男性主导。
*   **推进关系**：线下邀约的提议、约会地点、行程安排需由男性规划。

在这条链条中，男性任何环节的“不够主动”行为，都可能被女方、其亲友、甚至男方自己的家人解读为“缺乏诚意”。这种社会共识构成了女性可以保持相对被动姿态的外部环境。

## 不对等的评价地位 ⚖️

理解了外部期待后，我们进一步分析互动中的内部权力结构。由于上述责任链的存在，女性在相亲互动中往往处于“评价者”的地位，而男性则处于“被考察者”的位置。

*   他只聊天不邀约 → **Pass**

## 核心动机：成本与收益的计算 🧮

最后，我们从最根本的行为动机来看。根据经济学中的**理性选择理论**，个体的行为取决于其对行动收益与付出成本的权衡。公式可简化为：

**行动动机 = 感知收益 - 感知成本**

对于许多现代女性而言，在相亲中“主动热情”或“积极回应”所带来的潜在收益（如开启一段普通关系），常常被认为不足以抵消其需要付出的情感、时间和机会成本。因此，她们的选择策略是：
1.  **保持被动**：除非遇到在价值上明显优于自己的对象，否则不轻易主动投入。
2.  **使用拖延话术**：对于感觉一般的追求者，常以“我比较慢热”、“还需要时间考虑”等理由应对。
3.  **区别对待**：只有当遇到真正心动、价值匹配的对象时，才会迅速改变态度，变得积极热情。其行为模式的转变速度，可以用代码来形象描述：

```python
if 遇到对象.综合价值 >= 自身.综合价值:
    态度 = “热情积极”
else:
    态度 = “冷淡观察”
```
