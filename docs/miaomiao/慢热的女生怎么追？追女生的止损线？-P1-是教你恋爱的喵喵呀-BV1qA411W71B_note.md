# 恋爱教程 01：如何追求慢热型女生及设定止损线 🐢💔

在本节课中，我们将学习如何区分女生是“慢热”还是“无感”，并掌握追求慢热型女生的核心策略与关键的时间止损线。

## 区分“慢热”与“无感” 🔍

追求慢热型女生的第一步，是准确判断对方的真实态度。你需要主动与对方持续交流约半个月。

以下是判断对方是否“无感”的关键行为列表：
*   态度始终爱搭不理。
*   多次尝试均无法成功邀约。
*   聊天时经常性失踪或断联。
*   从未主动发起过对话。
*   每次回复仅限于“嗯”、“啊”、“哦”等单字。

如果出现以上多数情况，则表明她可能对你并无兴趣。即便是慢热的女生，也会在互动中给予一定的**反馈**和**进度条**。当你排除了方法问题后，若情况依旧，那么原因很可能就是“人不对”。

## 慢热型女生的两种成因 🧩

上一节我们介绍了如何区分态度，本节中我们来看看真正的慢热型女生主要分为哪两类。

1.  **原生家庭影响型**
    这类女生因成长环境而缺乏对人的信任和安全感。她们建立任何关系（包括普通友谊）都需要经历一个漫长的过程，其核心特征是 **信任建立周期长**。

2.  **情感经历匮乏型**
    这类女生由于恋爱经验较少，在感情中表现得较为木讷和被动。她们更需要对方明确地**主动带领**。

如果你本身性格被动、不善于主导关系发展，那么面对这类女生时，你需要做出选择：要么主动改变自己，去一步步引导她；要么趁早放弃。

## 追求策略与核心心态 🧭

理解了慢热女生的类型后，追求过程中的策略和心态就显得尤为重要。

这个过程需要极大的耐心，你必须**放缓节奏**。除了保持积极态度，在开始追求前，你必须建立一个清晰的**心理预期**和**进度条**，这是决定成败的关键。

通常，这个时间框架可以设定为 **两到三个月**。如果在此期间关系仍无任何实质性进展，那么最明智的做法是果断放弃，及时止损。
