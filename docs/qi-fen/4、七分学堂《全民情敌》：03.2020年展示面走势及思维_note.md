# 4、七分学堂《全民情敌》：03.2020年展示面走势及思维_note

在本节课中，我们将学习如何重新定义和打造你的朋友圈，使其成为吸引异性的有效工具。课程的核心是引入“着陆页思维”，将朋友圈视为个人品牌的广告页面，而非简单的日记本。我们将详细探讨什么内容可以发，什么内容不能发，以及如何通过朋友圈展示高价值。

## 课程概述

本节课分为上下两部分，本节主讲上集，聚焦于朋友圈的打造。我们将颠覆你对朋友圈的传统认知，学习如何将其转化为一个展示个人价值、吸引目标异性的“着陆页”。

## 重新认识朋友圈：三种常见用法

在深入探讨新思维之前，我们先来看看人们通常如何使用朋友圈。以下是三种常见的用法：

1.  **只看不发**：自己从不发朋友圈，但会通过朋友圈观察他人的生活水平、状态和兴趣爱好。
2.  **生活日记**：将朋友圈当作分享个人日常生活的平台，记录吃喝玩乐、心情感受。
3.  **仅三天可见**：设置朋友圈权限，让他人只能看到最近三天的内容。

大多数人的用法属于前两种。然而，无论是“只看不发”带来的零印象，还是“生活日记”中可能充斥的低价值内容，都不是在社交时代有效展示自己的方式。

## 核心思维：朋友圈即“着陆页” 🚀

上一节我们认识了朋友圈的常见用法，本节中我们来看看颠覆性的新思维——**着陆页思维**。

什么是“着陆页”？当你在搜索引擎输入关键词并点击一个链接后，首先看到的那个旨在吸引你继续浏览的页面，就是着陆页。

同理，**你的朋友圈就是你的个人“着陆页”**。当女孩打开你的微信，浏览你的朋友圈时，你的目标是让她产生兴趣，愿意继续往下翻看，甚至主动了解你。微商的朋友圈是展示产品的着陆页，而你的朋友圈，就是展示你个人这个“产品”的着陆页。

思维需要彻底转变：
*   **旧思维**：朋友圈 = 私人日记 / 生活分享
*   **新思维**：朋友圈 = 个人广告页 / 自我推销平台

在这个时代，你完全可以通过精心经营的朋友圈，成为自己社交圈子里的“红人”或“大哥”。关键在于，你是否能将自己当作一个优质产品来包装和展示。

## 朋友圈内容指南：发与不发的艺术

理解了着陆页思维，我们就能清晰地判断朋友圈内容的取舍。核心原则是：**一切围绕“价值”展开**。

### 必须避免的低价值内容 ❌

以下是会严重减分、必须避免发布的内容类型：

*   **游戏内容**：频繁发布游戏战绩（如“五杀”），会给女生留下“玩物丧志”、“不务正业”、“无法照顾伴侣”的印象。
*   **低质文章**：转发标题夸张、内容低劣的文章（如“不转不是中国人”），会暴露你的低品味和低层次，让女生觉得与你不在同一档次。
*   **负面情绪与抱怨**：倾诉工作辛苦、生活劳累、失业等悲观内容。在女生看来，这代表你对生活缺乏掌控力，连自己的生活都处理不好，更无法照顾好她。
*   **心灵鸡汤与无病呻吟**：虽然可能获得亲友点赞，但这只是“铁杆粉丝”基于关系的支持，并不代表内容本身对目标异性（女生）有吸引力或价值。

**核心公式**：`低价值内容 = 对微信好友（特别是目标异性）未提供任何价值`

### 应该展示的高价值内容 ✅

能发的内容都具备两个字：**价值**。价值分为外在价值和内在价值，且具有相对性。

**1. 外在价值（易于直接展示）**
*   **生存与繁殖价值**：
    *   **外貌**：通过发型、衣着提升形象。若颜值自信，可多发正脸照。
    *   **身材**：展示肌肉、健康体魄。
    *   **身高**：身高有优势则展示全身照；劣势则巧妙用半身照隐藏。
*   **硬实力（需巧妙展示）**：
    *   拥有房、车、财富等硬实力，不建议直接炫耀（易吸引“绿茶”）。
    *   **建议方式**：通过展示高消费生活方式来间接体现，如海外旅行、高端场所消费，实现“无形装逼”。
    *   **价值相对性公式**：`你的展示价值 > 目标女生日常接触男性的普遍价值`
        *   例如：想吸引“村姑”，二三十万的车是价值；想吸引“小网红”，则需要展示远超这个层次的价值。

**2. 内在价值（需通过内容渗透）**
*   **预选**：展示周围有优质异性朋友（漂亮小姐姐），这能提升你的吸引力。
*   **内在特质**：如高情商、有趣、有涵养、懂生活。这些很难直接拍摄，但可以通过有深度的配文、独特的拍照视角和精选的生活内容来渗透。

## 落地执行：打造你的高价值“着陆页”

上一节我们明确了内容的取舍标准，本节我们来学习具体的执行步骤。

首先，进行自查与清理。打开你的朋友圈，检查最近发布的10条内容（不限于10条，此为参考量），确保其中没有我们之前提到的任何低价值成分。如果有，请立即删除。

**行动清单：打造基础着陆页**
1.  **删除所有低价值内容**：包括游戏截图、低质文章转发、负面情绪宣泄等。
2.  **确保首页内容纯净**：保证陌生人或新好友打开你朋友圈时，前几条内容都是高价值的。
3.  **开始积累高价值内容**：根据你的实际情况和目标，有意识地开始拍摄和积累可以展示外在、内在价值的内容。

## 思维进阶：从展示价值到塑造个性

当你严格按照着陆页思维，初步打造出一个具有高价值展示的朋友圈后，你就可以进入下一个阶段：**适度展示个性，增加真实感与亲和力**。

这是一个关键的进阶思维：
*   **初级阶段**：价值为零时，发低价值内容 = 屌丝。
*   **进阶阶段**：价值建立后，发低价值内容 = 个性标签。

例如：
*   周杰伦打游戏，那是偶像的爱好。
*   一个已经展示出事业有成、生活丰富的男人，偶尔发一句“工作累了”，这展示的是真实感和亲和力，而非无能。
*   在展示了丰富生活后，发一张宠物的照片，可能成为女生开启话题的切入点。

**核心要点**：**价值先行**。必须先通过朋友圈建立你的高价值形象，之后偶尔穿插的个性化甚至“低价值”内容，才会被解读为“这是一个活生生的、有血有肉的成功/有趣男人”，而不是一个 loser。

## 风格定位与反差营造

最后，你需要定义自己朋友圈的整体风格，并巧妙营造“反差感”，让人印象更深刻。

*   **风格定位**：你的朋友圈主线是什么？是“爱玩的生活家”，还是“专注的创业者”？确立一个主要基调。
*   **营造反差**：在主线风格中，加入与之形成对比的内容，能极大增强吸引力。
    *   *例如*：主线是“爱玩会享受”，偶尔穿插“深夜加班”、“认真工作”的内容，会塑造出一个“既懂生活又很努力”的立体形象。
    *   *例如*：如果你是学生，大家都在展示学习，那么你展示“带领团队创业”、“组织大型活动”等内容，就能脱颖而出。

**重要提醒**：
1.  **切忌盗图**：盗用他人的生活照片会产生严重的“不一致性”，极易被识破，彻底摧毁信任。
2.  **保持一致性**：照片的风格、滤镜、拍摄水准应尽量统一，避免给人拼凑、不真实的感觉。

## 课程总结

本节课中，我们一起学习了如何用“着陆页思维”重塑你的朋友圈：
1.  **转变认知**：将朋友圈从“日记本”重新定义为“个人品牌广告页”。
2.  **明确规则**：学会了区分必须避免的**低价值内容**和应该展示的**高价值内容**。
3.  **理解核心**：掌握了价值的**相对性**原则，即针对你的目标群体展示相应价值。
4.  **执行步骤**：从清理旧内容开始，逐步打造一个高价值的朋友圈首页。
5.  **思维进阶**：理解了在建立价值后，如何通过适度展示个性增加真实感。
6.  **整体规划**：学会定位个人风格并营造反差，打造立体、吸引人的个人形象。

课后作业：立即检查并清理你的朋友圈，删除所有减分项。从今天起，有意识地开始积累和发布能够展示你高价值的内容。
