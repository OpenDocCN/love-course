# 七分学堂《脱单训练营》：第九节修图第二章：第九节修图第二章_note

在本节课中，我们将学习修图的核心流程。上节课我们介绍了构图技巧，本节中我们来看看如何通过后期处理让照片更具吸引力。修图主要分为三个步骤：**修饰脸部细节**、**调整光影亮度**以及**添加合适滤镜**。我们将重点讲解基础且实用的方法，确保初学者也能轻松掌握。

***

## 一、 修饰脸部细节 👤

修图的第一步是优化人物面部。我会使用**美图秀秀**或**FaceTune 2**来处理脸型、五官和皮肤瑕疵。以下是具体操作步骤。

首先，导入照片后，使用“面部重塑”功能进行基础调整。例如，如果脸型较宽，可以适当缩小脸部轮廓。该功能包含眼睛、鼻子、眉毛等细分选项，可根据个人情况微调。

```示例
操作路径：美图秀秀 -> 人像美容 -> 面部重塑
```

其次，使用“瘦脸”功能进行更精细的轮廓修饰。注意调整幅度不宜过大，以免失真。

然后，处理皮肤瑕疵。使用“除皱”功能淡化眼纹、法令纹等，使用“祛痘”功能去除斑点或痘印。

最后，使用“美白”功能提亮肤色，使皮肤看起来更干净。完成上述步骤后，脸部细节的初步修饰就完成了。

***

## 二、 调整光影与亮度 💡

上一节我们处理了面部细节，本节中我们来看看如何优化照片的光影。有时照片局部可能过暗，需要提亮。我通常会使用**Snapseed**（叶子）软件进行局部调整。

打开Snapseed，选择“局部”调整工具。点击画面中过暗的区域（如脸部阴影），提高其亮度。此步骤并非必需，约80%的照片可能不需要此操作。对于初学者，如果整体曝光尚可，建议跳过此步，以免参数过于复杂。

```示例
操作路径：Snapseed -> 工具 -> 局部 -> 点击暗部 -> 提高亮度参数
```

***

## 三、 添加与调整滤镜 🎨

完成光影调整后，最后一步是为照片添加滤镜，统一色调并提升质感。我常用**Instagram**或**VSCO**这两个软件。

以下是选择和应用滤镜的要点。

首先，根据照片内容和想要的风格选择滤镜。例如，人像照片可选用清新或复古风格的滤镜。

其次，**滤镜强度不宜设置为100%**。通常将滤镜强度调整到70%左右，再微调其他参数。

最后，需要手动调整基础参数。主要调整以下四项：
*   **亮度**：控制画面明暗。
*   **饱和度**：控制色彩鲜艳程度。
*   **对比度**：控制明暗反差。
*   **锐化**：让细节更清晰。

此外，还可以调整色温。偏好冷色调可降低“暖色调”值，偏好复古感可适当提高。

**核心原则是自然**。最终效果应让照片看起来美观舒服，而非留下明显的修图痕迹。多参考优质博主的照片，有助于提升审美。

***

本节课中我们一起学习了修图的完整流程：从**修饰脸部细节**，到**酌情调整光影**，最后**添加并微调滤镜**。遵循这些基础步骤，你的照片质量将得到显著提升。修图的关键在于多练习并建立自然、协调的审美观。
