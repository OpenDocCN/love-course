# 七分学堂恋爱核武器2.0：十八、毕业典礼 🎓

在本节课中，我们将超越恋爱技巧本身，探讨更广阔的人生规划与价值观。我们将一起思考事业、金钱、学习与人生的意义，希望能为你带来启发，助你更好地规划未来。

---

来到直播间的同学，请先签到。现在开始我们今天的课程。

我原本预计今天听课的人数不会特别多，因为今天是七夕。但考虑到这是恋爱核武器2.0的最后一节课，我决定按时播出。本节课不会过多讲解恋爱知识，而是分享一些超越恋爱范畴的、关于人生的见解。即使你今天去约会了没能来听课，关系也不大，因为本节课旨在给大家带来启发和开悟，帮助大家在应对人生时更从容。

著名心理学家阿德勒提出，人生主要面对三大课题：**社交**、**爱情**和**事业（工作）**。我专注于爱情领域，这让我节省了大量时间，无需在个人感情和人际交往上耗费过多精力。我认为，学习谈恋爱是一件超级简单的事。这里的“超级简单”并非指你一个月后就能成为专家，而是指我们的知识和服务模式，**一定会让你得到提升和改变**。这就足够了。

每个人天赋和投入不同，学会恋爱乃至脱单，只是时间问题。师父领进门，修行在个人。你们会根据自己投入训练的时间、接受知识的天赋，或快或慢地进步。最重要的是，你是否被“领进门”，是否“开悟”了。这正是我们这套课程的价值所在：它能否帮你树立良好的恋爱规划和正确的恋爱三观。

我的一些终身私教学员，他们并非一直跟我学谈恋爱。他们大多是已经被我“点醒”和“开悟”的人。他们继续学习，是为了在未来的恋爱道路上少走弯路，或者他们已经不再需要向我咨询具体恋爱问题。当他们有了自己的审视标准后，就能自行判断和在恋爱中历练，不断进步。这是我们的课程能教给你的。

为什么这些人还报名我的终身私教？部分原因是为了认识我，觉得我学习能力强，或者看好七分学堂在恋爱领域的发展。例如，曾有学员花费20万报名我的终身私教。我大部分的私教学员，其实都是为了与我结识，希望在今后的人生中有更多交流和交集。

今天说这些，是为了鼓励大家。不要因为现在单身或尚未脱单而焦虑。**重要的是你正在提升自己**。至于何时能脱单，时间会给你答案。好，签到完毕，我们进入正题。

我的人生愿望并非在此挣多少钱。钱对我而言只是一种工具，是实现某些目的或赚取更多钱的工具，而非我的终极追求。我从事情感教育，源于我的兴趣。在学习钻研的过程中，我吸收了许多先贤圣人的思想，这些思想让我活得快乐、轻松。我很感激他们，因此我力所能及的回馈方式，就是尽可能地点拨更多人。这也是我回馈古人的一种方式。

我主要吸收西方哲学，国学方面则学习王阳明和老子的思想较多。孔子思想也有所涉猎，但相对较少，部分原因是古文理解有门槛，而西方古希腊思想更现代化，易于接受。这就是我在此教导大家的原因。同时，在传播思想、教授课程时，我也感到快乐。我希望通过今天的课程，大家也能诞生一些自己的人生理想。

接下来，我将分享我自己的人生梦想和规划，或许对你也有所收获。

首先，我希望你能挣钱。我并非纯粹的好人，我这样做是有目的的。我未来可能需要资金、资源或人脉。与其等到有机会时临时寻找，不如现在就开始培养坚实的用户基础。所以，我先教会大家如何活明白。你们在未来，或许会有很多与我合作的机会。**只有活明白了，才能好好生活；只有好好生活了，才会有出息；只有有出息了，我们未来才有无限可能**。

两年前，我们公司已有成熟团队独立出去成立事业部，接受市场考验。两年后，他们越发完善，已独立形成一家公司。我们与他们的关系是合作，我持有一部分股份。我们公司也在向正规MCN机构转化，孵化各类网红。无论是你们认识的主播、老师，还是公司内部员工，我并不奢望他们为我打一辈子工。我深知，有想法的人无法被驾驭，终有一天会为自己的事业奋斗。

因此，我们开始了内部合作与创业。只要你是优秀、有能力的人，就可以单独成立公司。我会占较小份额的股份，并提供资金、品牌等支持，但不会参与任何实际经营。如果你对此感兴趣，我推荐你去了解**阿里巴巴的内部创业模式**，以及**2010年前美国资本输入第三世界国家**的模式（2010年后中国提出了“一带一路”）。我在此不做过多讲解，大家可自行领悟。

此外，我也不会要求永久分红。我可能与你签订协议，在四五年内，只要你让我挣到我应得的回报，这部分股份我可以套现或返还，不再继续分红。因为你的公司我并未费心经营，我只是挣我该挣的钱，并不指望它为我源源不断生财。这一点，大家自行体会。

这与我对待员工的态度一致：我不指望你在我的公司干一辈子，但我需要你在职期间达到我要求的业绩。我们开诚布公：你想在这里干多久，想达到什么样子，想学到什么知识。这是以往公司很少做的事，因为“干几年后另立山头”的话题比较敏感。但在我看来这很合理，因为越是优秀的人，越无法被驾驭，你只能想办法与之合作或交朋友。

接下来，我分享我一生的规划。我今年再过一个月就25岁了。平时直播中我说自己28、29岁，这是一种包装，目的是让观众对我有更多信任和安全感，觉得老师有阅历、有经验。这才是真实情况。你会发现，世界很多东西都是包装过的，背后都有一定目的。我的目的是让大家不上当受骗，不买那些不专业导师的课，浪费金钱和时间。我是为了让大家学到最专业、最好的课程。

我创立七分学堂已有三年。我全力以赴，从0到1。我计划总共花**12年**时间，不惜代价地全力投入情感教育事业。我用这12年挣到花不完的钱，让未来能更自由地做想做的事。

**34岁之后**，我计划用大约**6年**时间，去美国耶鲁大学等顶级学府学习。我的学历是大专，之前在学习上没下功夫。所以我觉得深造是我的一个追求。同时，这个年纪也要开始抚养、教育孩子。我不认同那些自己不行却拼命“鸡娃”的家长。我认为应该“鸡自己”。所以，我选择在30多岁时去读书，同时用心陪伴孩子成长。

根据心理学，孩子**6岁前**的经历、记忆和原生家庭环境，基本决定了他的潜意识，将影响他一生。我们现在的许多心理软弱或问题，大多源于童年的阴影或不幸福的环境。因此，我认为这是最应该花时间陪伴孩子的阶段。最好的方式就是暂时不工作，一边读书，一边养育孩子。用6年时间培养自己，也用6年时间培养孩子，帮他养成积极的思维定式。我不指望他有多大成就，但希望他能开心、自由地长大。

**40岁之后**，我并非不工作，而是只做开心的事。人不能没有工作。心理学家划分的人生三大课题中，并没有“抚养孩子”。孩子有他自己的生活，我们应把他当作独立的人来尊重。最重要的三件事始终是：**社交**、**爱情**和**事业**。没有工作，我们会失去很多乐趣和精神追求，找不到自我实现的价值感。所以，我们仍需要工作。

到**50岁**，我会一直投入一个方向：**金融**。我认为，引领创新的是金融，绝对创新的是教育。金融和教育我都有所涉及，金融是我未来的方向。所以，现在大家努力挣钱吧。你们在各行各业发光发热，既为了自己，也为了未来。到时候，我或许可以用你们的钱、用别人的钱来挣钱。我摊牌了，我的目的很简单：未来，当我们的某个学员在事业、人生上达到一定巅峰时，我们或许会有合作，在顶点相遇。你可以把钱交给我，我去投资更多项目。我可能会做私募基金，成为投资人。这需要我不断进步，也需要在座的各位不断成长，需要你们过好当下，努力挣到每一桶金。

**50岁到55岁**，我想做一件事：**开一场演唱会**。有没有人听没关系，我可以花钱请你来听。最好能在“鸟巢”开一场十万人的演唱会。希望50年后，中国最好的演唱会场地仍然是鸟巢。如果是，我会花5年时间去做这件事。不会写歌就学，不会rap就学，不会trap就请人教。这时，只要用钱就能办成很多事。

**55岁到60岁**，我想去**拍电影**，做一名演员。把我的人生经历和经验传递出去，也为了过瘾和开心。我希望看过我电影的人能明白一件事：**万般皆下品，唯有学习高**。这个世界上你什么都可以没有，但不能没有学习。学习可以改变命运。有的人说没有安全感，需要买房、需要更多存款。但记住，再多的物质也无法带来永久的安全感。唯有内心充满智慧，了解自己的智慧，你才能强大，获得最真实的安全感。

**60岁以后**，我可能会花几年时间，**买一支电竞俱乐部**。我很喜欢电竞游戏，喜欢这种靠脑力取胜的过程。虽然我智商、逻辑思维和记忆力并非顶尖，但我喜欢并愿意钻研。目前郑州也没有电竞俱乐部，以后有能力的话，一定要在郑州组建一个。这都是我想做的事，是我的使命。虽然做电竞几乎百分之百赔钱，但没关系。当你能用钱买到生活中绝大多数东西时，钱就已失去价值。赔不赔钱没关系，更重要的是不白活一趟，活得真实。

至于**60岁再往后**，我没想太多。如果我能活到**80岁**，我希望身体相当于50岁，然后去**爬一下珠穆朗玛峰**。很多人50岁就躺床上了，很多年轻人20岁就躺平了。我建议你，如果躺平，不如早点“走”，躺着没意思，是受罪。你现在熬夜、饮食不规律、养成啤酒肚和烂习惯，都是在用健康换钱，很不值得。想要拥有未来，需要健康的身体，需要保持超级自律，好好生活。好不容易当一回人，不小心病了、走了，多可惜。

当我80岁回想一生时，希望觉得没白活，大部分想做的事都做到了，这一生不枉来一遭。别让你的人生太白活。

**81岁以后**干什么？还没想好，未完待续。生命的意义在于努力奋斗和自由选择。你会发现，我所有的选择都是为了自由。我想要开的演唱会、想买的俱乐部、想拍的电影，我都为之奋斗。这个追求梦想的过程，就是我生命的意义。而你，可能每天在放纵自己，过着对付的生活，喜欢的得不到，还要为了现实委曲求全地工作，浪费很多精力却无法一举两得。而我，好像这也有了，那也有了。这就是我们之间的区别：既有生命的意义，也有生活的意义。这样，我的生活和生命就不再单调。

我分享自己的人生规划，是因为现在许多年轻人都没有明确的梦想和目标。就像有的兄弟说的，到现在还不知道自己想要什么，没有方向感和目标感。对此，我不是很理解为什么有的人有目标，有的人没目标。我只能借助先贤的思想。王阳明说过：**志不立，天下无可成之事**。

在我人生的前18年，我和大多数学员一样：逃课、打游戏、混日子，读书半读不读，也没想过以后怎么办，没想过以后去哪里打工、从事什么工作。但那时，我就有一个坚定的想法：**房子一定会有的，事业一定会有的**。这股自信不知从何而来，我从未担心过买不起房，总觉得男人总会有自己的事业。这可能就是迷之自信吧。

后来，在追求梦想的过程中，我遇到很多挫折，但我从未放弃目标或认输。我看到的不是我这辈子买不起房、没未来，而是我有哪些不足。只要解决了这些不足，曾经的自信都会变成事实。

关于**创业**，我不建议大多数人尝试。首先，我见过的创业者，几乎都是其领域的顶尖人才或精英。反观你自己：你是不是你们厂的劳模？是不是部门的领导？是不是花一两个月就成了销售冠军？如果是，也不代表你就能创业成功，因为创业和打工是完全不同的概念。

创业需要你**上到直播讲课，下到通马桶、通网络**。在业务上，你既要熟悉售前，也要知晓售后；既要懂互联网思维，也要什么都懂、什么都知道。你必须自己就能完善地运作所有事情时，创业才比较靠谱。很多时候，我们容易把**平台带给我们的成就，看作自己的牛逼**。平台给了我们背后的支持，我们却误以为是自己行了。结果自己出去创业，碰得头破血流。这样的例子比比皆是。

我们要对自己有充足的认知，不要太盲目。特别是，**不要把平台带给你的成就，看作你个人的能力**。人很容易飘，没有自知之明的人，容易因为短暂的成功而过度信任自己。一个合格的老板，一定要对你所有的事务**略有知晓**，不一定精通，但都得了解。这是我对大家创业的一点建议。不一定每个人都适合创业，而且这也是一个九死一生的过程。

接下来聊**副业**。大家有没有听过“斜杠青年”？斜杠青年指的是拥有多重职业和身份的人。例如，我的一个终身私教学员，他既是作家，又经营实体酒吧和剧本杀店。这就是斜杠。

曾有人问我，生意失败后要不要去摆地摊、送外卖、跑滴滴、做兼职。我的建议是：**不要做**。这样的事情是在**消耗你的时间来换钱**。你现在赚到的每一分钱，都是你过去所有经验和知识变现的结果。这是一个长远的过程。如果你现在不积累任何经验、知识和机会，那么10年、20年后，你可能还是老样子。

**因运气得来的钱，一定会因实力而败光**。许多彩票中奖者下场很惨，就是因为没有驾驭金钱的能力。所以，如果不是走投无路，不要去做这些兼职。它们浪费了你提升自我价值的时间，这是最昂贵的代价。

但是，有一类副业我建议大家做：**与你当前工作有一丝联系的副业**。例如，刚才提到的作家学员，他写的是职场人际交往、销售技巧类的书，后来去开剧本杀店。他的优势是：不用费心找剧本，可以自己写或利用人脉找人写。这和他的主业多少有点联系。

为什么要找和自己主业沾边的事做副业？我举个例子：如果你想成为金牌厨师长，可能需要20年；想成为顶尖挖掘机驾驶员，也需要20年。但如果你另辟蹊径，花5年从厨师学校毕业，再花5年从技校学会开挖掘机，然后用挖掘机炒菜，你可能就开创了一个新领域，成为这个领域的绝对头部。这是一种思维方式，让我们辩证地理解社会，找到更适合自己的方法。

未来，特别是互联网行业，会呈现**幂律化分布**。简单说，就是头部10%的人会占据90%的资源，剩下90%的人瓜分10%的资源。互联网加剧了竞争，你想挤到头部会越来越困难。这时，选择“用挖掘机炒菜”这种独辟蹊径的方式，你才能独一份，挤到头部。我们不在原来的规则里硬挤，而是开启新的竞争通道。

那么，我的副业做什么？目前我有四五个事业：
1.  **情感教育（知识付费）**：这是你们熟悉的。
2.  **网红主播培养（MCN孵化机构）**：这也很显而易见。
3.  **电商**：虽然入场较晚，但我已做了两三年，不指望做到头部，但能不花我太多精力。
4.  **医药行业**：我对此行业比较懂，也有人脉和资源，投入了一些份额。
5.  **成人培训（国学培训）**：同样是培训，但我不花费任何精力。

这些事业，我大部分都没去过公司，因为我不需要花精力去管。我的精力有限，要留在七分学堂，用接下来9年时间努力挣钱。其他的，都算是投资。

说到**投资**，比如股票、基金，我不精通，也不建议大家玩，因为散户玩不过庄家。人只能赚到自己认知范围内的钱，那是我认知范围外的，我不打算深入。

但关于投资思维，我想起一本书叫《反脆弱》。书中有一个概念：拿出你资产的**10%**，去做高风险、高杠杆的事情（如炒股、投资新兴项目）。这部分钱即使全部亏光，也不会影响你90%的生活和事业。如果赚了，是意外之喜；如果亏了，也无关系要。

对于**学习上的投资**，我很舍得花钱。今年才到8月，我在学习上已投入十几万。公司会集体培训，我也会向高人请教。我觉得能用钱解决的事都是小事，特别是很多事情无法用钱解决。越是免费的东西，可能越浪费你的带宽。

你为什么贫穷？想想你是不是月光族，为了省钱住得离公司很远，每天通勤浪费一两个小时，没时间充电提升自己，工作状态不饱满，升职机会也轮不到你。因为你把精力投入在了看似省钱、实则昂贵的过程中。这就是你不会花钱、不会用钱走捷径的表现。

我这辈子追求的不是房子，钱对我只是走捷径的工具。我从不省钱，因为**省钱的思维会影响挣钱的思维**。你花大量心思想怎么省，就没时间去想怎么挣、怎么提升自我价值了。我们过去所有的金钱，都是自己经验知识的变现。如果你现在不积累任何经验知识，10年、20年后你还是老样子。

这10%仅指高风险投资，不包括学习费用。在我看来，学习所花的钱不算钱，因为总有一天会挣回来，且不止这点。有时我花几十块钱买本书，通篇都是废话，但只要学会一句有用的，就很值。这就是思维的差距。**思维决定行为，行为决定结果，结果连贯成线就形成你的一生**。

在生活中，很多方面我不在乎花钱，比如**慈善**。最近一次是郑州洪水，我捐了一万块钱。我妻子发了朋友圈，但我自己没发。因为我不需要别人知道我捐了钱，不需要别人认同我或认为我善良。我早期发朋友圈是为了吸引异性、打造人设。但越发成熟后，我深刻明白：我是什么样，关他人屁事。所以，我的私人账号除了发和妻子的恩爱照（其实是发给她看的），几乎不再发朋友圈。但我用于撩妹的账号，还是会发一些吸引女生的内容。

说到慈善，它的对立面是不是自私？其实我并不反感自私。我一点都不反对自私。慈善类似于“顾忌他人”。有本书叫《原则》，作者是桥水基金的前CEO。书中提到几位伟大的慈善家，如穆罕默德·尤努斯（小额信贷先驱）、杰弗里（照顾残障儿童）、比尔·盖茨。他们都将自己“顾忌他人”这一项的评分打得很低。为什么？因为“顾忌他人”的反面是“顾忌自己”。这些人在面临选择时，都选择了实现自己的目标。

更直白地说，在自私和照顾他人感受之间，这些人选择的是前者。比尔·盖茨不先成为世界首富，也成不了慈善家。这不代表他们的做法一定对，但让我们更深刻地了解了这个世界。
