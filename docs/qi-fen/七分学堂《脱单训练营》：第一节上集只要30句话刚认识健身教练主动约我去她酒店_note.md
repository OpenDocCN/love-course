## 案例全流程拆解

当时，她在朋友圈发了一句伤感的话，我进行了评论。随后，她主动私聊问我：“在干嘛”。

**核心要点**：当女生主动问“在干嘛”时，回复要**简短**，不要长篇大论地汇报状态。

我的回复是：
```

```
她说：“好吧”。我反问：“你呢？”。她回答：“家里”。我说：“又是自己”。她回：“嗯嗯”。

**情报分析**：通过这段简短对话，我们收集到关键情报：她一个人在家，感到无聊和伤感。在和女生聊天时，学会收集情报至关重要，这能帮助你判断后续该做什么。

### 第二步：描述状态与情感共鸣

接着，我描述了她的状态并表达共鸣。

我的回复是：
```
不喜欢一个人，感觉太孤寂了
```
**操作解析**：把她的状态描述出来，让她感觉被理解。她说：“我觉得我就是内心太空虚了”。我回复：“缺少陪伴”。

**关键技巧**：此时不要说“我来陪你”，目的性太强。她说：“也许吧，我也不知道”。

### 第三步：服从性测试与模糊邀约

我接着说道：
```
对了，宝宝还欠我一个么么哒
```
**策略解析**：这是将朋友圈的互动延续到私聊的**服从性测试**，用以观察女生的反应。她通过了测试，回复：“你真的太乖了，么么哒”。

服从性测试通过后，必须立即尝试“变现”，即提出**模糊邀约**。否则女生可能去和其他人聊天。

我说：
```

```
她表示：“你下班都是半夜”。我回复：“最近实在太忙了，我尽量早点12点”。她拒绝：“太晚了，我今天晚上早点睡”。

### 第四步：框架博弈与施加压力

我继续推进，问：
```

```
**逻辑解析**：现代人很少早睡，这是一种“冷读”和框架施压，表明我不是一个对她言听计从的男人。她说：“我最近天天喝酒很累”。我回复：“不喜欢拖”。她表现出抗拒：“不巧”。

**应对策略**：当女生“框架”很强（即态度抗拒）时，我们也要拉高自己的框架，进行博弈。

我发了一个“拜拜”的表情。她问：“你要删我了？”。我回复：
```
不喜欢打嘴炮
```
**效果分析**：我的强势回复让她有些语无伦次了。在框架博弈中，气势不能弱。如果你表现得与众不同，就已经超过了99%的男人。她说：“昨晚想找你的，那你完了联系我”。

### 第五步：敲定细节与引导服从

我敲定时间：“好大概11点半”。她提供了电话和地址：“我在家里，你完了联系我吧”。我要求：“好位置给我”，她发了位置过来。

临近约定时间，我发信息：“差不多结束了，宝宝好”。她开始犹豫：“我都开始陪我妈吃宵夜了”。我坚定引导：“那现在过来”。她再次推脱：“我都开始吃第二顿宵夜了”。

**核心心法**：无论女生如何反应，都要坚定自己的节奏和引导。你要告诉她该做什么。

我说：
```
不是说好的吗
```
她回：“嗯”。我继续引导：“现在还得待会儿”。此时她的回复已经显得犹豫和服从。我直接提出：“那我们去看电影吧，到现在就对你开车”。她同意：“可以”。
