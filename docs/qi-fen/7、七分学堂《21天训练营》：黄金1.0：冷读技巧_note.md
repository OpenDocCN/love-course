# 课程7：七分学堂《21天训练营》：黄金1.0：冷读技巧 🧠💬

在本节课中，我们将要学习互动聊天中的一个重要技巧——冷读。我们将探讨其核心原理、常见误区，并提供一个简单实用的公式，帮助你更自然、更有效地与女生进行情绪层面的互动，从而提升聊天质量。

互动技巧是聊天过程中难度较大、挑战性较高的环节。在这个环节中，你需要引导女生产生情感共鸣，才能顺利升级关系。这比单纯邀约要困难得多。

例如，在与女生交换了姓名、身高、体重、年龄、职业等基本信息后，对话常常难以深入，很难将情绪推向高点。这通常是因为我们在互动环节做得还不够。

上一节我们介绍了互动环节的挑战性，本节中我们来看看如何通过感性的聊天方式来引导对方。

或者说，如何运用互动技巧，让女生愿意并能够与你畅聊。这种方法基本能解决以下情况：女生对你有一定好感，你们也能找到话题断断续续地聊，但情绪始终不高。

为什么情绪始终不高呢？因为你们讨论的所有话题都停留在逻辑层面，有一说一，有二说二，缺乏情绪的互动，自然聊不起来。

你必须引导女生也逐渐变得感性，才能忽然间让对话内容丰富起来。

才能让她一次性回复五六行文字，你才能获得更好的机会去升级关系。而冷读技巧，可能是大家直观看来最厉害的套路之一。

但是大家实际运用时是什么样子呢？可能说完这句话，女生就不回复你了。

大家记住，你认为的厉害不一定真的厉害。你自己觉得有用的一些套路和话术，不一定适合当前的环境。

因此，大家切记两个要点。首先，如果认识不深，就不要大篇幅地运用冷读，不要组织大量话术直接抛过去，因为那样会显得很套路。

其次，如果你真的无法感受到女生是什么性格、什么类型，就不要胡乱运用各种话术，不要生搬硬套。

那么，冷读究竟是什么意思呢？它其实就是类似星座解析，或者说是一些“两面话”，是一些算命常用的话术。

是一些任何人听了都会觉得“这说的跟我很像”的话术。大家看星座是不是这样？比如我是摩羯座，我看摩羯座的解析，会觉得“对对对，我就是这样的人”。

“两面话”就是指那些适用于几乎所有人的描述。例如一句简单的冷读：“你认真起来是那种格外投入、心无旁骛的人。虽然平时可能大大咧咧，但一旦认真，就会格外专注。”

这不就是废话吗？谁认真起来还能分心呢？人的大脑就是如此，同一时间只能专注思考一件事。当你认真做一件事时，没有多余心思处理其他事，一心二用是非常不靠谱的。

那么，把这句话说给一百个人听，是不是一百个人都会有认同感？都会觉得“对对对，说的是我，你很懂我”。

但是，冷读最大的缺陷在于，说出来的话可能让人感觉“不舒适”。例如，今天在群里看到有位兄弟对女生说：“我感觉你为人真诚，但平时肯定没少遭到误会吧？”

你说这样一句话，它其实不像正常人说的话。什么叫“不像人说的话”？就是很突兀。你自己会不会感觉说不出口？你能感觉到吧？这叫“语感”。女生也能感觉到：为什么这个人忽然对我说了一句很奇怪的话？为什么这句话转折如此生硬？

虽然它可能引起我的共鸣，但女生更多的感受是不舒适。像这样“不像人说的话”，希望大家以后不要犯。照搬来的话术，我希望你能运用自己的词汇和说话习惯，将其重新加工，内化成你自己的东西。

而不是生搬硬套一句完整的话术丢过去，特别是越长越像话术，越长越官方，就好像你在不带感情地客观陈述一件事。

举个例子，比如长一点的冷读：“我觉得你呢，是那种可以不假思索就接受别人观点的人，但有时候，你又挺有主见的，甚至还有点小固执。”

大家觉不觉得我这两句冷读，就更像人说的话了呢？它几乎没有套路和话术的痕迹了。这是大家需要努力的目标。

如果你要运用这个套路，你的目标就是尽量用简单的话，真实地用心感受对方是什么样的人，然后简单地表达出来。这需要一定的社交直觉和情商。

你才能真正体会到那些素未谋面的陌生人，大致是什么感觉。因此，这设定为大家的一个目标。后面我会教给大家一些简单的、不留痕迹的话术套路。

冷读最大的敌人是，女生会觉得：“你都不认识我，凭什么这么了解我？”那么，我给大家的这个公式，就是解决“你不认识我”和“凭什么了解我”这两个点的。

这个公式可以理解为两句话，是我们可以对女生说的。

首先第一句话，是我们对她给出的初步负面印象（IOD），即说出对她的一些不那么好的、错误的理解。

例如：“你看起来很高冷啊”，或者“你好像很难沟通”。

然后第二句话，是我们在认识她之后，在聊天一段时间（比如半小时、一两小时，甚至认识几天）后，发现她与最初的印象不同，并给出新的正面印象（IOI）。

这个后面的印象一定是夸赞她的。我们等于先推开一下，再拉近一下，这只是把“推拉”技巧反过来用，就构成了我们的冷读。

大家发现没有，这都是一些简单的固定句式。而且这样的一句话，你表达给女孩子，她很难产生负面反应（IOD）。

我们是在她身上用了心，我们说的话不像生硬的话术，而是用自己的语气，表达自己的观念和看法而已。

像这样的形容词有很多，特别是那些每个人都可能有的情绪或特质，例如：自卑、自信、安全感、虚荣心、自恋等等。

大家如果想丰富自己的词汇，大可以去百度看看那些星座解析，里面全是这些东西，全是在说性格如何、做事如何。

用“窗口判断”来理解这样一句冷读的话，你会发现，虽然它先推开再拉近关系，但落脚点还是在正面印象（IOI）上。

因此，我们去跟女生冷读时，这句话本身是带有“窗口”（即好感信号）的。这句话具备着积极的、释放好感的能量。

所以我们不能在女生对我们表现出明显反感（IOD）时随便去用，不能在女生回复很少、窗口很小时胡乱去用。

必须是在这个女生很愿意跟我们聊天，你至少能感觉到她愿意聊，拥有这样一个“小窗口”的时候，再去回应这样的话。

这依然符合我们的能量守恒原则：女生必须对我们具备小的窗口，我们才起码可以运用这样的技巧。如果女生都不愿意回你了，或者回复全是负面信号（IOD），你想着“我今天学了冷读，我去冷读一下她的心理”，或者“明天学了表演型人格，我去植入一下画面感”，这都是没用的。

好，这是一个相对简单、可以套用公式，也不会让女生觉得你套路的方法。

我们前期运用冷读时可能会用得不准确。这并不是说套路，因为我们这句话本身已经避免了套路感，它很舒适，很像人话。

唯一可能产生不好效果的地方，就是你形容得不恰当。比如你说“我感觉你像……样的人”，女生可能不认同。这是我们目前可能会遇到的问题。

那么大家记住，如果你冷读形容了对方的性格后，对方不认同，先别慌。你可以用一句话轻松化解这个负面反应（IOD）。

你可以说：“这只是我的感觉而已”，或者“我随便说说，别在意”。

冷读成功的，我们回头好好复盘一下，我究竟是如何抓住她的一些心理活动的。冷读失败的兄弟也不用气馁，慢慢来。

我们前期一定是一个从无到有的过程，这个过程必然比较艰难。我们无法很好地理解女生内心，源于几个问题：

首先，我们对“窗口”判断得不是特别准确。

其次，我们与太少的女生沟通过，根本不知道女孩子心里想的是什么。

只要“窗口识别”这个技能点越来越强，只要我认识的女生数量逐渐增加，我慢慢也能理解女生的性格，慢慢也能识别清楚她背后究竟是怎么想的。

好，明天将给大家带来一篇冷读的实际案例，供大家跟着案例一步步学习。最后提一句，我当年是怎么学的呢？我是看了很多案例的。

我不是只看老师那句厉害的话是怎么说出来的，我会想：如果是我在当前和这个女生聊天，我会怎么说？想完之后，我再去看案例是怎么说的，我们俩的思维究竟差在哪里？
