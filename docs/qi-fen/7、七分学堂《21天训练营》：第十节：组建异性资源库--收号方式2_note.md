# 七分学堂《21天训练营》课程十：组建异性资源库——收号方式2 💬

在本节课中，我们将学习在社交软件上与异性互动时，如何高效地获取联系方式（收号）。我们将重点分析一种聊天战线较长的情况，并讲解如何把握节奏、化解尴尬、最终成功收号。

---

上一节我们介绍了收号的基本逻辑，本节中我们来看看当聊天战线被拉长时，我们应该采取何种策略。

对于这种战线时常拉得比较长的互动，我们能加对方联系方式就直接加，无需进行过多了解。我们也没有时间进行过多了解。可能我发个消息，对方隔两天回复；对方发我一个消息，我又隔两天回复。那么，我们要聊到收号，可能得耗费一两个月时间。

在一两个月时间里，很有可能被别的男生挤掉我在她心中的排名，或者她被别的女生挤掉在我列表中的位置。这都是有可能的。在社交软件上，我们尽量快、尽量准、尽量狠地加到女生联系方式即可。

大家可以看到我与这个女生的互动模式：她发一个消息，我要保证能量守恒，可能隔两小时回复。之后她可能隔一天回复，我又要保证能量守恒，不能秒回，可能又隔半天或一天回复。

之后她又隔半天或一天回复。这时我就知道，如果我不加她，我们真的会错过。对于这种战线拉得比较长的，大家抓住机会就直接要微信号。

接下来，给大家看一段较长的聊天记录。我们聊着聊着就偏离了预设的三个话题，转到其他方向去了。我们怎么聊？也希望大家能通过这篇聊天记录举一反三。因为你们在聊天时绝对会聊岔开，绝对会聊到其他东西。

你只要能够把握主线，只要能够做到不吃亏，同时完成展示并收到这个女生的号就可以了。

以下是聊天中的几个关键信息点分析：

*   **女生主动发消息**：主动发消息的女生绝对不多。这说明这个女生对我拥有比较大的窗口，有较大的兴趣。
*   **开场用语“你好”**：“你好”是社交中绝对不错的两个字，是非常礼貌的文明用语，是陌生人打招呼客气的两个字。但它在社交软件聊天中不是特别合适，因为它太过于客气，太有距离感。这种聊天氛围不适合开玩笑，太生硬，太有距离感。

所以，我们需要去化解这种生硬，化解女生设定的聊天框架。于是我说了：“这么正式我都不习惯了。”我的言外之意是，我们在一起聊天，我觉得轻松一点更好。因为轻松了，我才可以跟这个女孩开玩笑；轻松了，我们才可以相互打闹。

理解了这一点，我们继续看。然后这个女生开始提供话题，她说她婚纱的拉链坏掉了。我觉得这个话题不错，可以聊下去。我就跟随女生把这个话题聊了下去，没有继续走原来的老套路。我结合着婚纱聊了下去，我说：“画面一定很美，只不过可惜了，我没有看到这个婚纱掉下来之后的完美画面。”

由于我们认识程度非常浅，特别是在前期，我们不要聊太过于越界、太过于亲密或色色的东西。这很容易让女生觉得你是流氓、下流、低俗。果不其然，在我说出这样的话之后，这个女孩骂我了，她说：“你是流氓。”

“流氓”是非常严重的**IOD**（兴趣指标缺乏），是非常不好的一个负反馈。她基本上可能已经对我形成了一种负面印象。这样的负反馈我们需要化解掉。

如何化解问题？其实并非遇到问题才想如何化解，而是在说出一些非常危险的话之前，我就要提前想好怎么化解。我在说出“想看到你的婚纱滑落”之前，就已经想到了化解方式：我就这么一说，我又不可能真的看得见，我就过过嘴瘾，说一下蛮羡慕、蛮想看的，但是我也不会真的看见，你也不会真的吃亏。这就是我那句化解问题的由来。

我们在跟女生聊天时绝对不是盲目的，绝对不是无头苍蝇一样，撞出一个问题包，才去解决这个包。不，可能你如果这样做，这个包会大到你想不出解决办法。所以我们一定要像下棋一样深思熟虑，想到我这一颗子落下去，对方会怎么封杀我；对方如果封杀我，我要怎么样反败为胜，解决掉对方的封杀。等于说我们的思维要快人一步。我在说出这句非常流氓的话之前，就一定要想好，我怎么样去化解掉它。

第一次化解之后，大家可以看到，由于我聊得太“色”，这个女生回复非常差。她回复：“……”（无语，IOD），然后说：“然后呢？”（静静地看着我IOD），又说：“额……”（没有任何谈资）。看明白了吧？这就是反面教材，展示了我们如何在不正确的阶段，说了一些让自己爽，但是给自己增加难度的话。

然后我说了第二句化解她IOD的话：“这是在夸你有魅力呀。”这一句话大家做一个重点记号。这会广泛运用在你之后升级关系失败时，去解决问题上。例如：
*   **代码/公式示例**：`当 升级关系动作（如牵手）遭遇负反馈（如被甩开并质问）时，标准回应 = “还不是因为你太有魅力了。”`

我第二次的化解，终于把这个女生拉了回来，让我们的聊天回到了我的领域，回到了我自己的节奏。回到了她夸我，然后因为我的手滑，我错点了一个东西（可能是一个“真心话大冒险”的游戏）。这个女孩说了一句话：“我是你爸爸。”这个真心话大冒险的问题是：“说一句让你印象深刻的话。”于是这个女孩就说了“我是你爸爸”。

我去衔接她，我说：“你成功的引起了我的注意。”并且我告诉她，真心话大冒险是我随便点的，是我顺手带出来的。我们在跟女孩聊天时，可能会点错表情包，可能会发错话。我们做错了这些事情之后，一定不要撤回。撤回会显得我们内心非常焦虑。我们点错了就点错了，发出来就发出来，发出来之后只要跟女孩说一句，我们只是点错了，我只是顺带不小心点到了就可以了。

这个女孩可能不理解什么叫“点错了”，她就问：“啥？什么意思？”我们不需要解释。标化的人生一定不需要解释。你会发现，我跟女孩在一起聊天，错了就是错了，我可能会选择跳过它，可能会选择绕过去，也不会在这样无意义的话题上浪费时间和精力。我要花大篇幅去跟这个女孩讲述我是怎么样点错这个图片的吗？没必要。所以我就直接转移了她的注意力，我说：“你个二货。”

这时女生说：“我还是最爱你的父亲。”聊到了最后，我们还是落实到我们的实际点上，落实到我要怎么办，落实到我要什么——我要收这个号，我要加上她的微信。

这就是一套可能聊得比较多、聊得比较差、聊到别的东西的聊天，我们应该怎么做的完整分析。
