## 一、 修图的必要性与效果对比

上一节我们介绍了展示面的重要性，本节中我们来看看如何通过修图优化展示面。

修图并非简单的美化，其核心在于弥补瑕疵，让照片中的人物在保持真实感的同时更加赏心悦目。关键在于调整后，本人与照片的差距不应过大。

以下是修图前后的效果对比示例：

*   **案例一：眉毛的修饰**
    *   **修图前**：五官立体，但眉毛稀疏，整体神态略显无神。
    *   **修图后**：仅补充了眉毛，人物眼神立刻变得有神，整体精致度提升一个档次。核心操作是使用**修补**和**细节**工具。

*   **案例二：面部微调**
    *   **修图前**：五官端正，但下巴线条和皮肤质感有优化空间。
    *   **修图后**：收紧了下巴，平滑了皮肤，五官未做大改动，但面部更干净、立体。核心操作是使用**重调形状（调整）**和**平滑**工具。

修图的目的是在有限范围内，让人物变得更完美，同时确保线下见面时能被认出。

## 二、 核心修图软件与工具详解

了解了修图的效果，接下来我们学习实现这些效果的工具。本节将详细介绍修图软件 `FaceTune` 的核心功能。

以下是 `FaceTune` 中必须掌握的几个核心工具及其作用：

1.  **白画**
    *   **作用**：局部增白。例如，涂抹发黄的牙齿或衣物，使其变白。
    *   **代码/操作示意**：`选择“白画”工具 -> 涂抹目标区域`

2.  **平滑** 与 **更加平滑**
    *   **作用**：磨皮，消除皮肤上的痘痘、皱纹或斑点。`更加平滑`的效果比`平滑`更强。
    *   **代码/操作示意**：`选择“平滑/更加平滑”工具 -> 涂抹皮肤瑕疵区域`

3.  **细节**
    *   **作用**：突出局部细节，增强纹理。常用于让模糊的眉毛、发丝变得更清晰。
    *   **代码/操作示意**：`选择“细节”工具 -> 涂抹需要突出的区域（如眉毛）`

4.  **重调形状** 与 **调整**
    *   **作用**：调整形状。`重调形状`改动幅度大；`调整`用于微调，如收窄鼻翼、修饰脸型。
    *   **代码/操作示意**：`选择“调整”工具 -> 轻微推动需要修改的轮廓（如下颌线、鼻翼）`

5.  **修补**
    *   **作用**：复制画面的一部分来修补另一部分。常用于修补残缺的眉毛。
    *   **代码/操作示意**：`选择“修补”工具 -> 用一个圆圈选取源区域 -> 用另一个圆圈覆盖目标区域`

## 三、 实战修图：五官调整与禁忌

掌握了工具，现在进入实战环节。本节将以一张照片为例，演示如何综合运用上述工具进行面部修图。

**操作流程如下：**

1.  **基础磨皮**：使用`更加平滑`工具，涂抹皮肤，消除痘痘、皱纹等瑕疵。**注意**：避免涂抹眼睛、鼻子、嘴唇等五官，否则会失去立体感。
2.  **五官微调**：
    *   **鼻子**：若鼻翼较宽，使用`调整`工具轻微向内推动两侧鼻翼。
    *   **脸型与下颌线**：使用`调整`工具，从耳下到下巴勾勒出清晰、内收的**下颌线**。这是让脸型变立体的关键。
    *   **眉毛**：若眉毛稀疏，先用`修补`工具复制毛流进行修补，再用`细节`工具加深颜色。眉形应自然，眉头与鼻梁形成弧线，眉尾略长于眼角。
3.  **整体协调**：检查调整后背景线条是否错位，确保修图痕迹不明显。

**核心禁忌：**
*   **禁忌一**：不要用`平滑`工具涂抹眼睛、鼻子、嘴巴。
*   **禁忌二**：眉毛不要画得太粗太黑，避免不自然。
*   **禁忌三**：调整脸型时，注意不要扭曲背景中的直线物体。

## 四、 滤镜的选择与运用原则

修完五官后，我们通过滤镜来提升照片的整体氛围和质感。本节讲解如何选择正确的滤镜。

滤镜的核心原则是**增强原色，而非覆盖蒙版**。好的滤镜应该提升原有色彩的层次和质感，而不是给整个画面统一铺上一层颜色。

以下是选择滤镜的具体方法：

1.  **使用软件**：推荐使用 `VSCO` 或类似软件进行滤镜处理。
2.  **判断标准**：
    *   **避免**：选择会让肤色和背景颜色都发生巨大改变、像蒙上一层有色玻璃的滤镜（尤其是非主流风格的黄、绿色调）。
    *   **选择**：选择能**增强色彩对比度和明暗层次**，但基本不改变肤色和主色调的滤镜。例如，让暗部更沉稳、亮部更通透的滤镜。
3.  **调整强度**：滤镜强度不宜过高，通常调整到 **30%-40%** 即可，以达到自然增强的效果。

**公式：优秀滤镜 ≈ 原图色彩 + 增强的层次感与质感**

## 五、 自拍技巧：角度、表情与构图

优质的修图源于一张合格的底片。本节我们学习如何拍出一张好的原始自拍照。

自拍的目标是传递个人风格和情绪，避免呆板。关键在于找到适合自己的角度并运用肢体语言。

以下是提升自拍质量的几个要点：

1.  **寻找最佳角度**：
    *   **避免死亡角度**：避免正面直怼镜头、过度俯拍或仰拍。
    *   **常用角度**：将摄像头**略高于眼睛**，微微侧脸，找到自己最上镜的一面。
2.  **运用肢体语言与表情**：
    *   **不要面无表情**：可以尝试微笑、坏笑、沉思等表情。
    *   **加入手势**：用手轻托下巴、遮挡部分脸颊、整理头发等，能增加画面动感和自然度。
    *   **眼神接触**：自拍时请看着镜头，传递情绪。
3.  **利用辅助工具**：可以下载如`型男相机`等APP，利用其提供的**构图线**（如三分线、黄金分割线）来辅助构图，将眼睛或面部重心放在交叉点上。
4.  **背景与穿搭**：注意背景干净简洁，穿搭符合你想塑造的形象。

本节课中我们一起学习了从拍摄到后期的完整照片优化流程。

**我们来总结一下今天的核心知识点：**

1.  **修图软件掌握**：熟练使用 `FaceTune` 的**平滑、细节、调整、修补**等核心工具进行面部优化。
2.  **审美标准建立**：了解男性**清晰的下颌线、自然的眉形、立体的鼻型**等基本审美标准，并以此为指导进行修图。
3.  **滤镜运用原则**：使用 `VSCO` 等软件，选择能**增强原图层次**而非**覆盖颜色**的滤镜，强度控制在30%-40%。
4.  **自拍能力提升**：找到自己的最佳拍摄角度，运用**表情**和**肢体语言**，拍出自然、有情绪的原片。
