在本节课中，我们将要学习一种常见的不良心态——思维极端化。我们将探讨它的定义、表现、成因，并学习如何克服它，从而建立更健康的心态来处理生活中的挑战，尤其是在面对拒绝和挫折时。

## 什么是思维极端化？

上一节我们介绍了心态对个人改变的重要性，本节中我们来看看什么是思维极端化。

所谓思维极端化，是指一个人的世界里非黑即白。他只能看到事物的一面，多数情况下是坏的那一面，而看不到事情好的一面。这导致他不能合理地解决问题，心态出现失衡。这种心态是一种性格上的盲人摸象。

**公式：** 思维极端化 ≈ 非黑即白 ≈ 只看到负面

假如生活是一头大象，思维极端化的人如果碰到的是大象的鼻子，就会把他整个人生等同于大象的鼻子。如果他摸到的是大象腿的部分，他就会说我的人生是一个粗粗的柱子。他总是基于片面的经验，对整体下绝对的结论。

## 思维极端化的表现

以下是思维极端化的几种典型表现：

1.  **自我定性，看低自己**：经历几次失败后，就给自己下“我一无是处”、“我绝对不会成功”的结论。
2.  **心态失衡，难以快乐**：处于这种状态时，人特别容易沮丧，很难感到真正的快乐。
3.  **以偏概全，否定全局**：因为个人或他人的个别负面经历（如分手、看到明星离婚），就得出结论“我再也不相信爱情了”。
4.  **自我价值感低**：总是不信任自己可以做好一件事，或者自己能得到一样东西。例如，年轻但没赚到太多钱，就认定“我这辈子一事无成”。

## 思维极端化的成因

我们之所以会形成这种思维模式和自我低价值感，往往与成长经历有关。

以下是两个关键成因：

1.  **被他人贴上消极标签**：在成长过程中，父母或老师可能因为我们某个单一行为（如染发、两次没写作业），就给我们贴上“不三不四”、“不爱学习”等言过其实的标签。
2.  **内化标签，形成习惯**：当身边总有人为我们贴上消极标签时，我们很容易给自己心理暗示——“我就是他说的那种人”。人是思想的产物，当你把别人对你的看法移植到自己身上，就可能真的变成那样的人，或者至少变得自卑。

当你长大成年后，可能不再有人给你贴标签，但你自己已经养成了“随便给自己贴标签”的习惯。

**代码示例（心理暗示循环）：**
```
外部负面标签 -> 自我认同(“我就是这样的人”) -> 行为符合标签 -> 强化负面自我认知
```

## 如何走出思维极端化？

我们怎样才能从这种思维的怪圈里面走出来呢？正确的办法是：学会从正反两面去理解一件事情。再坏的事情发生了，都可以从另外一个角度去挖掘到它的价值。

当你遇到失败，开始认为自己没有价值时，请遵循以下步骤：

1.  **察觉情绪**：意识到自己产生了“悲伤”、“哀伤”、“郁闷”的情绪。
2.  **切断情绪，提问自己**：
    *   第一，是什么事情引起了你现在的难过？
    *   第二，你对这个事情的理解是不是还不全面？

例如，有人破产了，觉得一生奋斗化为乌有，于是自杀。这是典型的思维极端化。他只看到了失去的物质资产，却没有看到自己收获的经验、成长、思维模式、朋友关系以及家人这些更珍贵的、别人拿不走的无形资产。

> 人生除了生死，都是小事。你如果能把一件事从正反两面去理解，就不至于有那么极端化的想法。

## 案例与实践

摩卡自己就是一个活生生的例子。当年看到前女友跟别人结婚时，觉得天崩地裂，认为自己浪费了十年时光，太失败。但现在看来，如果没有那次失败和痛苦，就不会从事现在的行业，也不会有现在丰富的人生体验和自信。

**核心心法：** 当你的人生有了阴影，它才会变得立体丰满。

现在，当摩卡遇到不喜欢的人和事，产生消极情绪时，他会这样做：
1.  捕捉到自己的消极心态。
2.  对自己说：“**她是我的礼物。她的到来就是为了让我变得更好，我会通过她的存在去修炼自己。**”

这能有效停止抱怨自己或他人。

我们可以把人生看作一场游戏。所有让你不舒服、难以应付的人和事，都是游戏里的大BOSS。每当打败一个，我们就会收获经验值。经验值达到一定数值，我们就会升级。虽然面对BOSS时打得很辛苦，但打得越辛苦，收获的经验值就越高，成长越快，成就感和幸福感也越强。

**最终结论：** 只要你坦然接受了那些不好的，才会收获更好的。当你接受了自己的失败，你就会更有底气相信自己能成功。
