# 魔卡私教《框架推拉约会技巧》：框架推拉：小视频3：10_note

在本节课中，我们将通过一个职场沟通的实战案例，来学习如何运用框架推拉技巧。这个案例模拟了一位管理者向下属下达指令的场景，其中包含了建立权威框架、施加压力（推）以及赋予使命感和价值（拉）的经典元素。

## 概述：指令中的框架与推拉

上一节我们介绍了框架推拉的基本概念，本节中我们来看看如何在实际对话中构建并运用它。以下案例中的每一句话，都旨在巩固管理者的主导框架，并通过推拉结合的方式驱动团队行动。

## 案例指令分解

管理者下达了两项明确指令，并进行了动机动员。我们将逐条分析其中的框架推拉技巧。

### 第一项指令：明确任务与时限

**指令内容：**
第一，这个季度我们大客户部的销售奖金，到了核发的时候。你要共华东大区的Michael，华中大区的Charles，华北的Yvonne，西北的Amy，西南的Thomas，分别核对，收集其选项给HR部门审核。你负责跟他们协调，两天之内发给我签字。

**技巧分析：**
此部分通过清晰、具体、有时限的指令，建立了管理者“决策与监督者”的框架。使用“你要”、“负责”、“两天之内”等词语，是典型的“推”，即施加明确压力和责任感，确保指令被严格执行。

### 第二项指令：强调紧迫与优先级

**指令内容：**
第二，订一张马上到上海的机票，马上，立刻。最近这段时间，我们整个团队，把所有的精力投入通达微业的项目投标里。这关乎我们DB未来在中国市场的发展。

**技巧分析：**
“马上，立刻”的重复使用是强烈的“推”，制造紧迫感。随即，通过“这关乎我们未来在中国市场的发展”这句话进行“拉”，将个人任务提升到公司战略高度，赋予执行者使命感。

### 核心动机动员：价值拉升与框架巩固

在明确指令后，管理者通过一段动员来强化团队认同，这是“拉”的集中体现。

**动员内容：**
现在有太多的投资者，都在等着我们投标。所以我们要做的，就是要把投标投到，我们自己的公司。现在有太多人在我的耳边说，这是不可能的。但是在我自己里，没有不可能这句话。你们看，这是一个五十六年历史的品牌。当我们走到外面，双手将名片递给客户，我们每一个人都代表着它。我们不能让这个品牌，砸在我们手里，更不能让它在市场上红修。现在都去全新揍Proposal。

**技巧分析：**
1.  **树立共同目标（拉）**：“投资者在等我们”、“投给我们自己的公司”，描绘成功愿景。
2.  **破除限制信念（推拉结合）**：承认外界质疑（“太多人说不可能”），但立刻用“在我的字典里，没有不可能”强势推翻，巩固自身“必胜”框架，这是一种对内质疑的“推”和对团队信心的“拉”。
3.  **赋予身份与价值（拉）**：将个人工作与“五十六年历史品牌”的荣誉绑定，强调“每个人都代表品牌”。这是最高级别的“拉”，激发责任感和荣誉感。
4.  **施加集体压力（推）**：“不能砸在我们手里”、“不能让它在市场上红修”，将失败后果与团队每个人挂钩。
5.  **最终行动指令（推）**：以“现在都去全新揍Proposal”收尾，回归行动指令，完成从动员到执行的闭环。

## 总结：框架推拉的实战循环

本节课中，我们一起学习了如何在一个简短的指令场景中综合运用框架推拉技巧。

其核心流程可以总结为：**建立权威框架 → 下达具体指令（推）→ 阐明价值与意义（拉）→ 破除障碍、巩固信念（推拉结合）→ 强化集体身份与使命（拉）→ 最终指向行动（推）**。

这个案例展示了，有效的领导或沟通不仅是下达命令，更是通过推拉结合的方式，管理对方的情绪、认知和动力，最终让其在你的框架内积极主动地完成任务。在约会或人际互动中，原理相通：通过语言和行为，构建吸引力框架，并巧妙运用推拉来调节关系张力和进展节奏。
