# 魔卡恋爱方法2.0：042、高富帅会套路-等于流氓会武术-谁都挡不住(上)_note

在本节课中，我们将要学习一个常见的恋爱现象：为什么看似完美的“高富帅”在追求心仪对象时，有时也会遭遇挫折。我们将通过分析电影片段，理解“吸引力”与“可得性”这两个核心概念，并探讨它们如何影响一段关系的开始。

---

上一节我们介绍了高富帅在恋爱中并非总是无往不利。本节中我们来看看一个具体案例，分析问题究竟出在哪里。

电影片段中，由宋承宪饰演的林总是一位条件优越的集团总裁。他对由刘亦菲饰演的女律师周雨展开追求。从吸引力的角度看，林总几乎无可挑剔。

但正是这种近乎完美的形象，当真正出现在一个普通女孩的生活中时，却常常让女生望而却步。这就像成语“叶公好龙”所描述的情景：女生们虽然嘴上说着想找“霸道总裁”，但当真人出现时，第一反应往往是懵圈，然后是撤退。

周雨的反应就是典型的例子。为什么会这样呢？在恋爱心理学中，这被称为 **“可得性低”**。

许多高富帅都会面临这种情况：**吸引力指数很高，但可得性指数很低**。

为了帮助理解，我们可以用一个简单的比喻来说明：

假设有两笔钱，一笔是10万，另一笔是100万。如果让你任选一笔拿走，所有人都会选择100万，因为它的**价值**更高，**吸引力**更大。

现在，我们增加一个条件：选择100万，需要30年后才能拿到；选择10万，今晚就能到账。这时，大多数人的选择可能会改变。

你的决定之所以改变，是因为虽然100万的**价值（吸引力）** 很高，但它的**可得性**太低了。这个道理在恋爱中同样存在：价值再高，如果让人觉得无法获得，那对当事人来说就缺乏实际意义。

对于那些吸引力很高的高富帅，他们的情感困扰往往就出现在“可得性低”的范畴里。女生会想：“你是高富帅很好，但你未必是真心对我，或者你未必只对我一个人真心。”

---

从视频中我们可以看出，林总和周雨的第二次会面，林总在营造约会氛围、谈话内容、以及肢体接触升级等方面都没有问题，过程非常流畅。

从周雨在第一次酒吧被表白后逃离，却又再度应邀出来，以及她对林总肢体接触的自然反应来看，她心理上是喜欢林总的。

而造成周雨再次逃离的原因，并非是因为被触碰或亲吻而感到被冒犯。更多是因为她觉得两人条件相差太悬殊，或是对感情本身存在某种恐惧，甚至可以说是对自己有过婚史感到自卑。

总之，她担心林总不会真心、长久地爱她。即便林总已经明确表达了“以结婚为前提”交往，并且不在意她的婚史。

---

那么，这种因“高吸引力、低可得性”造成的困境，要如何解决呢？我们会在下期节目中为大家详细讲述。

本节课中，我们一起学习了恋爱中的两个关键指标：**吸引力** 与 **可得性**。我们通过电影案例看到，即使条件优越的追求者，也可能因为让对方感到“可得性低”而遭遇挫折。理解这一点，是破解许多恋爱难题的第一步。
