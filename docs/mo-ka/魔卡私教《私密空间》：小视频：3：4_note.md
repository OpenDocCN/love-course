# 魔卡私教《私密空间》课程：小视频制作（3：4）📐

在本节课中，我们将学习如何制作一个3：4比例的小视频。我们将从理解视频比例开始，逐步介绍内容规划、素材准备到最终成型的完整流程。

## 概述

视频的宽高比是决定其视觉呈现的基础。3：4是一种常见的竖屏视频比例，非常适合在手机等移动设备上观看。掌握这种比例的视频制作，能帮助你更好地适应社交媒体平台的需求。

## 视频比例基础

上一节我们概述了课程目标，本节中我们来看看视频比例的核心概念。视频比例指的是视频画面宽度与高度的比值。对于3：4的比例，其公式表示为：

**宽 : 高 = 3 : 4**

这意味着，如果视频宽度是3个单位，那么高度就是4个单位。在制作时，你需要确保所有素材和画布都符合这个比例，以避免画面被拉伸或裁剪。

## 内容规划与脚本

理解了技术参数后，内容的规划同样重要。一个清晰的脚本或内容大纲是视频的骨架。

以下是规划视频内容时可以遵循的步骤列表：
*   **确定主题**：明确视频要表达的核心信息。
*   **列出要点**：将核心信息分解为几个关键点。
*   **组织顺序**：合理安排要点的讲述顺序，确保逻辑流畅。
*   **撰写口播稿或对白**：将要点转化为具体的语言。

## 素材准备与拍摄

有了脚本，接下来就需要准备相应的视觉素材。无论是拍摄新片段还是使用现有素材，都需要遵循3：4的比例。

在拍摄或选择素材时，请注意以下几点：
*   **构图**：在竖屏构图中，注意主体位置，利用好画面的纵向空间。
*   **分辨率**：确保素材分辨率足够高，例如1080x1440像素（3：4比例的常见高清尺寸）。
*   **一致性**：保持所有素材的色调、风格大致统一。

## 剪辑与导出

所有素材准备就绪后，即可进入剪辑阶段。你需要使用视频编辑软件，并新建一个3：4比例的项目序列。

以下是关键的剪辑与导出步骤：
*   **创建项目**：在软件中设置画布比例为3：4。
*   **剪辑拼接**：将素材按脚本顺序排列在时间线上。
*   **添加元素**：根据需要加入文字、音乐或转场效果。
*   **检查比例**：最终输出前，务必预览确认画面比例正确，内容无重要部分被裁剪。
*   **导出设置**：选择正确的分辨率（如1080x1440）和格式进行导出。

## 总结

本节课中我们一起学习了制作3：4比例小视频的完整流程。我们从理解**宽高比=3:4**这一基础公式开始，逐步完成了内容规划、素材准备与拍摄、以及最终的剪辑导出。记住，保持比例一致是贯穿始终的关键。掌握这些步骤，你就能独立制作出适合竖屏观看的短视频内容了。
