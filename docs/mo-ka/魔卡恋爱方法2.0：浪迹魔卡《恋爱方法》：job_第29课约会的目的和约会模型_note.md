## 邀约窗口的判断练习

在介绍模型前，我们先来检验一下大家对邀约暗示和窗口判断的准确性。例如，女生发微信问“在干嘛”，这是一个邀约暗示吗？

可能有的同学会认为是，因为自己问“在干嘛”时通常是想约对方。但女生发“在干嘛”通常不是邀约信号，她可能此刻想你或感到无聊想聊天。总之，“在干嘛”基本等同于“我想你”，但“我想你”并不一定代表“想见你”，这仅仅是一个聊天信号。

此时，我们可以描述一个画面来开启聊天，例如：“我刚刚打扫完院子，为了我家小狗，这会儿正带他遛弯呢。”女生就可能基于你给出的信息（工作、宠物、生活）进行下一步聊天。

再比如，女生发微信问：“哎，你在哪？” 大家觉得这是不是邀约窗口？

很明显，相比“在干嘛”，这是一个比较明显的邀约窗口。这可能建立在之前有一定聊天了解的基础上，女生知道你的公司、家庭或常活动区域，正好她也在附近，有想见面的意思。如果方便和适宜，就可以见面。此时我们可以说：“刚和朋友谈完事情，你在哪？” 女生回复：“我在你公司附近。” 那我们就可以说：“哟，正好，那走一起吃晚饭去。” 这样就自然地完成了一次邀约。

再考大家一个例子：我们约女生出去唱歌，女生给出以下几种回复，大家看看哪一种窗口最大？
1.  我晚上约了朋友。
2.  你们去哪唱歌？
3.  几点啊？我晚上11点要回家。
4.  都是你的朋友，我去不好吧？

最好的回复是第四种。为什么？第一种是明显的拒绝。第二种有地点考虑因素（地方好玩才去）。第三种有时间考虑因素（需早离开）。前三种回复，女生都在考虑她自己的问题。只有第四种回复，她在考虑你的问题，即她去是否会不方便或影响到你和你的朋友。

第一次约会的目的是**建立好感不减分**。常见的减分项包括：

*   吃饭吧唧嘴。
*   对女生和他人的态度不一致。
*   对约会地点不熟悉或选择不当。
*   不会处理突发状况。

这里摩卡分享一句话：**和熟悉的人去陌生的地方，和陌生的人去熟悉的地方**。与熟悉的人已有了解和联系感，需要更多新鲜刺激，所以不要总在老地方转悠。而与刚开始约会的女生，最好去一些自己熟悉、最好有社交认证的地方。对场所熟悉能带来很好的带领感，突发状况少，自身舒适感高，不易紧张，能更好地与女生交流互动，俗称“主场作战”。

在第二次约会中，我们可以借助一些升级惯例，并变换约会场所和形式来拉近关系。**牵手**是需要攻克的第一个关卡。当然，并非第二次约会就必须牵手，没牵手也不意味着失败。大家按照“约会四步”走时，要根据现场情况灵活应对。只要你感觉女生这次约会情绪更好、交流更多、窗口更大，也是可以的。

第三次约会的目的是**继续拉伸吸引，创造机会亲吻女生**。

### 第四步：进入私密空间
