# 魔卡私教《窗口识别》课程04：窗口识别小视频部分 🎬

在本节课中，我们将学习窗口识别技术在处理特定视频格式（如4:3比例）时的应用。课程将通过一个具体的日期与电影名称的识别案例，来演示如何从文本信息中提取和确认关键数据。

---

上一节我们介绍了窗口识别的基本概念，本节中我们来看看如何从一段包含特定日期和事件的文本中，识别并结构化信息。

以下是从给定文本中提取核心信息的步骤：

1.  **识别日期与时间**：定位文本中的具体日期、星期和时间描述。
    *   例如：`1998年10月7號 禮拜三`

2.  **识别关联事件**：提取与上述时间点关联的具体事件或内容。
    *   例如：`看了一場電影，印象中是一部歌舞片`

3.  **识别后续时间与作品**：识别文本中提到的另一个时间点及其对应的明确作品名称。
    *   例如：`1999年10月 電影《少年之路》推出`

4.  **建立时间线**：将识别出的信息按时间顺序组织，形成清晰的事件脉络。

通过以上步骤，我们可以将一段叙述性文本转化为结构化的时间-事件数据对。

---

本节课中我们一起学习了如何从一段包含个人回忆的文本中，运用窗口识别的思路提取出结构化的时间与事件信息。关键点在于**顺序定位时间点**并**关联其对应的事件内容**。这种方法可以应用于日志分析、历史资料整理等多种场景。
