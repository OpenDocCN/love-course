# 魔卡私教《窗口识别》课程 7：3 🎬

在本节课中，我们将学习“窗口识别”小视频部分的核心语法点，即如何使用“是……的”结构来强调动作发生的时间、地点或方式。我们将通过具体的例句来掌握这一结构的用法。

上一节我们介绍了窗口识别的基本概念，本节中我们来看看如何运用特定语法结构进行强调表达。

“是……的”结构在中文里常用于强调句子中除主语以外的某个部分，例如时间、地点或方式。其基本模式为：**主语 + 是 + 被强调部分 + 动词 + 的**。当强调部分为动作本身时，动词需使用原形。

以下是使用“是……的”结构进行强调的几个关键点：

*   **强调时间**：用于说明动作是在何时发生的。
*   **强调地点**：用于说明动作是在何地发生的。
*   **强调方式**：用于说明动作是以何种方式进行的。
*   **强调对象**：用于说明动作的对象是谁。

为了让你更清楚地理解，我们来分析一个例句：“我认为你应该看着我”。

在这个句子中，如果我们想强调“你应该做‘看着我’这个动作”，我们可以使用“是……的”结构来重组。核心在于将原句中的动词部分“看着我”作为被强调的内容。

因此，转换后的强调句为：**我（主语）是（强调标志）认为你应该看着我（被强调的动词原形内容）的（结构助词）**。

本节课中我们一起学习了“是……的”这一强调结构的用法。通过例句“我认为你应该看着我”的拆解，我们了解到该结构可以将一个完整的动词短语作为焦点进行强调，从而让表达的重点更加突出。掌握这个结构有助于你在窗口识别和交流中更准确地传递信息。
