# 魔卡私教《私密空间》课程三：小视频比例3:2 📐

在本节课中，我们将学习视频制作中一个重要的基础概念：画面比例。具体来说，我们将深入探讨3:2这一特定比例的含义、应用场景以及如何设置它。

## 概述

视频或图像的宽高比，是指其宽度与高度之间的比例关系。3:2是一种经典的画面比例，意味着画面的宽度是高度的1.5倍。理解并正确使用这一比例，对于创作特定风格或适配某些平台的作品至关重要。

## 什么是3:2比例？

上一节我们介绍了画面比例的基本概念，本节中我们来看看3:2这一具体比例。在数学上，3:2比例可以用一个简单的公式来描述：

**宽 : 高 = 3 : 2**

这意味着，如果视频的宽度是3个单位，那么其高度就是2个单位。例如，一个分辨率为 **1920像素 x 1280像素** 的视频，其宽高比就是3:2（因为1920/1280 = 1.5，即3/2）。

## 3:2比例的应用场景

了解比例的定义后，我们来看看它的实际用途。3:2比例常见于一些静态摄影和特定的视频创作中。

以下是3:2比例常见的应用领域：
*   **经典摄影**：许多数码单反相机的原生照片比例就是3:2。
*   **电影感构图**：部分电影和艺术视频会采用3:2比例，以营造独特的视觉感受。
*   **社交媒体**：某些平台或内容类型可能更适合或要求使用3:2比例的封面或视频。

## 如何设置3:2比例

在视频编辑软件中设置3:2比例通常很简单。关键在于创建项目时或导出视频时选择正确的参数。

以下是在编辑软件中设置3:2比例的一般步骤：
1.  打开视频编辑软件，创建一个新项目。
2.  在项目设置或序列设置中，找到分辨率或帧大小选项。
3.  手动输入符合3:2比例的分辨率，例如：**宽度：1920，高度：1280**。
4.  确认设置，即可开始在此比例下进行编辑。
