# 魔卡恋爱方法2.0：浪迹魔卡《恋爱方法》：第06课 恋爱能量是什么东西 ⚡️

在本节课中，我们将要学习恋爱吸引力的另一个核心要素——能量。我们将探讨能量的本质、它如何影响你的吸引力，以及如何通过提升能量来改变你在吸引力象限中的位置。

---

在第二课“吸引力模型”中，我们介绍了构建心动指数所需的两大核心要素：**价值**与**能量**。上一节我们详细探讨了价值的重要性，本节中我们来看看另一个关键因素：能量。

能量是一种状态。从生物学角度讲，能量如同你的血液。就像在游戏中，英雄人物拥有生命值，外出战斗会消耗生命值，生命值低了就需要补充。在人生这场游戏中，作为我们自己选定的“英雄”，我们在生活中处理各种问题、应对挑战时，也必须时刻关注自己的“血值”，也就是我们所说的**能量**高低。

你可以将其理解为状态的好坏。例如，在运动场上奔跑的能量，与生病时躺在病房里的能量，是完全不同的。不仅人有能量，场合也有能量。咖啡厅的能量与夜店“动词打次”的能量截然不同，夜店的能量要高得多。很多同学第一次去夜店，可能会感到手足无措、紧张无助。

再举个例子，当我们私下里可以眉飞色舞地讲述一件事时，如果让我们上台面对几百人演讲同样的事情，我们可能会紧张得结结巴巴。这是为什么呢？这就是**能量不足**的原因。因为面对几百人场合的高能量，你自身的能量不足以驾驭。

所以，在构建好自身价值之后，我们还需要调整自己的能量去进行展示。能量与价值的组合，会形成四个基本的吸引力象限。

以下是四个吸引力象限的划分，你可以对照自身目前情况，看看自己正处于哪一个区间。

我们以Y轴为能量轴，X轴为价值轴：
*   **价值高，能量高**：这是“高富帅”象限。没有女生会不喜欢这个区间里的男人。
*   **价值高，能量低**：我们称之为“闷骚男”象限。
*   **价值低，能量高**：我们称之为“逗逼”象限。女孩子可能会和你玩得很开心，但她们绝对不可能喜欢上你。
*   **价值低，能量低**：这就是“屌丝”象限。

这个象限里的很多男生，可能终身只能在他们喜欢的女神的世界里，扮演路人甲或空气的角色。坦率地说，这个象限里的男人连空气都不如，空气至少是被人需要的，而他们却不被需要。在社交场合中，他们犹如透明人，没有任何人注意到他们的存在。很多时候，在一个三小时的朋友聚会里，直到最后，可能有的女生都不知道还有这样一个人坐在角落。

如果你认为自己正处于这个象限，并且觉得没什么问题、挺开心，那么外人包括摩卡，都没有资格对你的生活指指点点。但是，如果你正好处于这个象限，并且待得特别不舒服，已经受够了暗恋、收好人卡、被人无视和欺负、看着喜欢的女孩被其他男人追走，那么兄弟，从今天开始努力改变吧。建设自己的价值，提升自己的能量，努力让自己从“双低”象限向“双高”象限过渡。

如果你不想让你的基因无情的从人类的基因库里消失，如果你已经受够了年年节假日形单影只，如果你不想再对着电脑屏幕里的A片撸管，那么行动起来，跟着课程设立目标努力实践，尽快完成从屌丝到男神的蜕变。

可能有同学很感兴趣：那我到底如何提升我的能量呢？这里摩卡给大家三个小建议和一句座右铭。如果能够坚持，一段时间后你就会明显感觉到自己的能量和状态发生了根本性的改变。

以下是三个提升能量的具体建议：
1.  **设立一个目标**：目标不宜设定得过长，可以设立一个为期三个月左右的目标，计划完成一件具体的事情。**目标是调动能量的发动机**。
2.  **早起并规划一天**：俗话说“一日之计在于晨”，每天的能量从早上开始酝酿。所以，早起并规划一天的时间安排，能让你清晰、有数地知道有限的能量具体要分配到哪些任务上。这就像提前知道“我今天要打哪些怪，我的血够不够”，心中有数，做好准备，事到临头能量就不会散乱，能够有序、有效地输出。
3.  **活在当下**：不管之前的事情做得如何，既来之则安之，集中能量做好当下的事情。千万不要在约会的时候想着公司的事，在公司的时候想着约会的事。

最后，分享给大家一句座右铭：**冒险创造价值**。很多时候，我们因为对未知事物的恐惧和对前景的不了解而止步不前、犹豫不决，白白浪费了能量。其实，只要确定了什么是我们想要的，那就大胆调动你的能量去追求吧。青春只有一次，不要让自己留下任何遗憾。

---

本节课中，我们一起学习了恋爱能量的概念。我们明白了能量如同生命值，影响着我们在不同场合的表现和吸引力。通过价值与能量的组合模型，我们分析了四种吸引力象限，并认识到提升能量与建设价值同等重要。最后，我们掌握了三个提升能量的具体方法：设立短期目标、早起规划和活在当下，并记住了“冒险创造价值”的行动格言。
