## 🍹 点酒技巧：控制消费与展现魅力

在酒吧点酒，既是消费控制，也是个人展示。

**控制消费的方法**：
*   事先约定：例如“我请你吃饭，你请我喝酒”。
*   明确轮次：“第一轮我请”，暗示下一轮对方请。
*   熟悉酒单，避免红酒、洋酒等整瓶消费。选择特色鸡尾酒即可。

**为女生点酒**：
女生通常不懂酒，可以推荐 **大都会（Cosmopolitan）**、**僵尸（Zombie）** 等杯子漂亮、口感偏甜、酒精浓度不低的鸡尾酒。避免点过于俗套的酒款。

**为自己点酒**：
果断告诉酒保你要的酒，如一杯纯饮或短饮。一杯 **马天尼（Martini）** 或一瓶啤酒都是不错的选择。这种细节会为你加分。

## 🚕 成功转场：回家或下一站

当双方情欲投入时，即可提议转场。

**转场要素公式**：
`转场成功 = 合理化理由 + 制造困难 + 解决问题 + 状态描述 + 假性时间限定`

以下是具体解析：

1.  **合理化理由**：如“一起回家喝红酒”、“回家给你做点吃的”、“回家放个包”。
2.  **制造困难**：如“但我家没有开瓶器”、“家里没鸡蛋了”。
3.  **解决问题**：“看看你这里有没有？”或“我们去楼下买”。
4.  **状态描述**：描述接下来会做什么，如“我们可以听听音乐、聊聊天”，避免直接说敏感目的。
5.  **假性时间限定**：如“我们就坐半小时，然后出来吃宵夜”。
6.  **提前叫车**：千万不要让女生在路边长时间等车，这会极大降低转场成功率。
