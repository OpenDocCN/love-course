搭讪是指在没有任何朋友介绍的情况下，主动寻找机会认识陌生的异性。常见的搭讪场景包括街头搭讪和特定场合搭讪。

许多同学认为搭讪等同于耍流氓。这种看法并不正确。搭讪对于男生而言，是一件非常需要勇气的事情。有过搭讪经验的同学对此都能感同身受。没有搭讪经验的同学，也建议去尝试体会一下。

我们总以为搭讪可以认识女生。但实际上，通过搭讪，我们可以更清晰地认识自己。一次成功的搭讪，需要男生在外形、态度和技巧这三个方面都有不错的基础。

如果认为仅靠耍流氓就能在街上随便认识女孩子，那未免把女孩子想得太傻太简单了。搭讪虽然看似由男生主动，但选择权其实在女生手中。如果女生完全不配合，男生是没有任何办法的。

上一节我们分析了案例中开场白的应用，本节中我们来探讨如何更流畅地进行话题过渡。
