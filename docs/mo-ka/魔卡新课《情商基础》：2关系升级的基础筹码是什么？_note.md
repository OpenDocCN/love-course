# 魔卡新课《情商基础》：2关系升级的基础筹码是什么？_note

在本节课中，我们将要学习关系三要素（逻辑思考、情绪好感窗口、关系升级）的不同搭配组合，会如何将关系导向不同的方向。理解这些组合是避免陷入错误关系模式的关键。

上一节我们介绍了关系三要素及其重要性，本节中我们来看看它们不同的搭配组合，会将关系分别带入什么不同的方向。

## 第一种组合：友谊区模型

这种组合的特点是：**有逻辑思考，有情绪好感窗口，但没有关系升级**。

你觉得这样的搭配组合，能够把你们的关系带去什么方向呢？你以前有没有经历过这种情况？

没错，这就是友谊区的关系模型。例如，你们是大学同学，互相关系很好，也一起经历了不少事情。你们在一起的时候，多半也是开心愉快的。但是，你一直不敢抓住机会进行关系升级，勇敢地捅破那一层窗户纸，把友情升华为爱情，导致情绪好感窗口一直大量地被浪费。最后，你们的关系被固化在友谊区里面，不再有质的变化。结果要么是你把这一切深埋心底，独自品尝；要么是你终于鼓起勇气表白，然后大概率被拒绝，最后连朋友的关系也显得有些尴尬。

## 第二种组合：好人卡模型

这种关系类型的特点是：**可能有逻辑思考，也可能没有；没有情绪好感窗口；但尝试进行关系升级**。

例如，你喜欢隔壁班的一个女孩子。你们没有说过什么话，也没有单独相处过，可能你还不了解她。然后你通过朋友要到了女生的联系方式，在线上进行一通表白。女生感觉莫名其妙，她多半会回复你“我现在不想谈恋爱，我要集中精力学习”等推辞。

又或者，你们是同班同学，每天都见面，但单独接触比较少，了解停留在很表面的程度上。但是你就是暗恋她，终于有一天你忍不住找了个机会表白，但是被发了“好人卡”：“你很好，相信你会遇到比我更好的，我觉得我们还是做同学比较好。”这些都是女生常见的推辞。

## 第三种组合：骚扰/猥琐模型

这种组合的特点是：**没有逻辑思考，没有情绪好感窗口，但强行升级关系**。

如果是线下的肢体升级，这就是我们俗称的“流氓”，或者在日本叫做“痴汉”，也就是我们说的非礼。

那如果是线上的语言升级呢？比如你在社交软件上加了一个女孩，聊一两句就开色情玩笑，发羞羞的表情调戏人家，甚至说“哎，你不如来我家吃点宵夜啊”。这样就直接被女生贴上了“猥琐男”或者“渣男”的标签，然后被拉黑。

## 第四种组合：短期关系模型

这种组合的特点是：**没有太多的逻辑思考，但是有情绪好感窗口，有关系升级**。

这种往往就是一夜情或者短期关系的模型。比如你在酒吧夜店认识了一个女孩，然后两人喝酒、跳舞，玩得非常开心。情绪、气氛、场合一切就自然而然地发生了。但是往往第二天或者几天后，两人又迅速地由亲密转为陌生，关系就像流星划过夜空一样，转瞬即逝。

当然，也有的恋人是由一夜情发展而来的。那么他们一定是在发生关系之后，充分地补足了逻辑思考的部分，甚至在女生出现事后反悔之前，男生就已经做了很好的铺垫和处理。

现在你明白了，为什么那些夜店、酒吧、狂欢节聚会之后，总是很容易发生一些露水情缘。没错，就是因为那些场合和活动，让男女的情绪很容易“嗨”起来。只要男生抓住机会，适时地进行升级，有些事情就会自然而然地发生。

## 理想的长期关系模型

一段优质的长期关系模型，肯定是**逻辑思考、情绪好感窗口和关系升级**这三种全部满足的。

比如你看看那些要分手的情侣，或者婚姻质量很低的夫妇，无一例外都只剩下了逻辑思考，没有情绪好感，自然也不可能有关系升级。也就是“我知道你是我的女朋友，但是我的情绪不好，甚至对你感到厌恶或者嫌弃，我的身体抗拒或者排斥你，我不可能接受和你的关系升级”。

到这里，相信你已经意识到了，三要素当中好像比较重要的就是**情绪好感窗口**，因为它是关系升级的筹码和土壤。有它就可能进行关系升级，没有它绝对不可能进行关系升级。那如果长期没有它，就算你们曾经很亲密，也可能要面临分手。

你也可以把情绪好感窗口理解为关系这台车的**汽油**，那逻辑思考呢理解为**机油**，而关系升级呢，就像这台车的**油门**。

再重复一遍：**情绪好感窗口，是关系升级的基础土壤**。

快乐、高兴、积极、放松、刺激、好奇、激动、暧昧、期待，这些都是常见的良好的情绪窗口。

那么常见的关系升级有哪些呢？它分为线上的语言调情和线下的肢体升级。

*   **发生关系**：可以算关系达到了 `80` 分。
