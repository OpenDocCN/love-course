# 魔卡私教《吸引力原理》：吸引力原理小视频部分：6：3_note

在本节课中，我们将通过一段情景对话，学习在商业营销中如何处理因未经授权使用他人形象而引发的冲突，并理解基本的营销伦理与沟通策略。

---

上一节我们探讨了营销策略的制定，本节中我们来看看当策略执行出现伦理问题时，应如何应对。以下是对情景对话的解析与要点归纳。

**核心冲突**：未经许可使用个人肖像进行商业宣传。
**公式表达**：**不当营销 = 未经授权使用肖像 + 引发对象反感**

以下是基于对话梳理出的关键步骤与应对原则：

1.  **承认问题与保持冷静**
    当利益受损方（佳雅）情绪激动地前来质问时，当事人（库尔迪普）的第一反应是试图让对方冷静（“先拿杯冷饮”、“先冷静”）。这为后续沟通创造了可能性。

2.  **解释动机与呈现结果**
    当事人随后解释了行为背后的商业动机（“营销策略”），并提供了客观数据支持（“销售额就涨了百分之八”），试图将个人冲突转化为商业效益的讨论。

3.  **遭遇坚决拒绝与威胁**
    利益受损方完全否定了对方的商业逻辑，明确划清个人权益与商业目标的界限（“我可不是来给你提高销量的”），并提出了明确的整改要求与最后期限（“明天之前…撤下来”），否则将采取行动。

4.  **尝试缓和与后续铺垫**
    在冲突暂告段落时，当事人通过提醒对方遗忘物品（手机）这一细节行为，试图缓和气氛，并为可能的后续联系（存号码）埋下伏笔。

---

本节课中我们一起学习了如何处理因营销越界引发的直接冲突。关键要点在于：**未经授权的肖像使用是高风险行为**，一旦遭遇反抗，应立即寻求补救。有效的应对流程包括**快速安抚情绪、清晰解释缘由、尊重对方要求**，并为未来关系修复留有余地。商业成功不能以侵犯个人权益为代价，合乎伦理是可持续营销的基石。
