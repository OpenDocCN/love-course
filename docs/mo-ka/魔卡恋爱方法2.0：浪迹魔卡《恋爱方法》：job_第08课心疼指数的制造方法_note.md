# 魔卡恋爱方法2.0：浪迹魔卡《恋爱方法》：第08课 心疼指数的制造方法 💔

在本节课中，我们将要学习如何通过“暴露敏感面”这一核心技巧，来制造“心疼指数”，从而构建更扎实、更立体的吸引力。

上一节我们介绍了吸引力模型中的心动指数，本节中我们来看看如何制造与之相辅相成的心疼指数。制造心疼指数，关键在于在平时建立的积极形象之外，偶尔向对方展示自己内心脆弱、无助或疲惫的一面。

## 核心概念：暴露敏感面

简单来说，如果我们平时有90%的时间在与女生相处时都在“笑”，那么我们需要寻找或制造那10%的时间，让对方看到我们“哭”。这样，我们的性格才能立体起来，吸引力才能更扎实。

我们不单要让女生有女儿般崇拜、仰望、依赖我们的感觉，还要偶尔让女生有妈妈般疼爱、照顾、被我们需要的感觉。

## 如何寻找与表达敏感面

内心的敏感面无需编造。任何人在成长过程中都曾受过委屈，或在工作、生活中正遭遇瓶颈。我们可以从真实人生经历中寻找一两件事，进行适当的表述加工。

**重点在于**：表述时必须带出真情实感，才能达到效果。

以下是暴露敏感面时需要注意的几个要点：

*   **素材来源**：从自身真实经历中挖掘，尤其是与童年或父母情感相关的经历。
*   **表达核心**：重点是传递当时的感受与情绪，而非单纯叙述事件。
*   **情感共通**：人与父母的情感联结是内心最深处、最共通的情绪来源。

## 更深层的意义与收获

在摩卡的线下课程中，常有学员在心态训练或一对一交谈时落泪，这就是暴露敏感面。这往往能极大地加深彼此的情感连接。

如果不知如何寻找敏感面，可以尝试回溯童年经历，或思考当前行为与早年父母情感的关联。这通常能找到强有力的素材。

在利用这些素材的同时，如果我们回想起与父母未解的纠葛、矛盾或怨恨，摩卡建议：

**运用自信、成熟、担当、包容、孝顺等内在价值特质，主动去处理和修复与父母的关系。** 父母并非完人，他们也未曾学习如何做父母。

因此，通过暴露敏感面获得安慰是小的收获；**真正修复内心的敏感地带，改善与父母的关系，使自己变得真正强大、柔软和成熟，才是最大的收获。**

请记住：
**我们与母亲关系的远近，将决定我们一生与女人关系的远近。**
**我们与父亲关系的远近，将决定我们一生与男人关系的远近。**
