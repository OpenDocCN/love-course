## 第三节：破冰与对话脉络——从开场到深入交流

一上来的话题称之为破冰。如果目的性显得很强，会让双方都不舒服。可以发挥一下自己的想象力，从谈谈对女孩子第一眼的感受开始。

**核心技巧**：用对她正面的感受来打开话题。
```plaintext
举例：“感觉你挺会穿衣服的。”、“你的指甲油颜色很好看。”
```

### 经典的聊天脉络：引导与分享

经典的聊天脉络之前介绍过很多次。以下是关键步骤：
1.  **破冰**：用感受或第三方话题开场。
2.  **引导对方提问**：在对话中自然展示自己。
3.  **主动介绍自己**：内容要精心准备。

这个时候就可以主动的介绍自己了。自我介绍是建立好感的第一步，一定要认真对待。

**自我介绍公式**：
```
自我介绍 = 我是谁 + 我有什么优势 + 我怎么会来到这里
```
从这三部分的内容，让对方对你信任、有安全感，并认识到你的价值。

先介绍自己，再慢慢的引导对方去多分享她的故事，这是在相亲的中期通常会做的事情。
