# 魔卡私教《框架推拉约会技巧》课程2：型男形象改造（上）🧑💼

在本节课中，我们将要学习如何通过打理发型来提升个人形象。发型是塑造第一印象的关键环节，一个合适的发型能迅速提升你的气质与自信。我们将从选择产品开始，逐步讲解具体的使用方法。

## 选择合适的产品🛒

上一节我们介绍了发型对于形象的重要性，本节中我们来看看需要准备哪些工具。一款好的定型产品是打造理想发型的基础。

以下是推荐的产品及其购买渠道：

*   本课程中使用的发蜡品牌是“威娜”。
*   你可以在京东或淘宝等电商平台购买到此产品。
*   我本人是在京东购买的。

## 产品用量与使用💡

了解了在哪里购买产品后，接下来我们看看如何使用它。正确的用量是避免头发油腻、保持造型清爽的前提。

使用这款发蜡时，我们不需要取用太多。

*   大概一个手指指节的量就足够了。
*   因为过量的产品会让头发显得厚重、不自然。
