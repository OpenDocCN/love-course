在本节课中，我们将要学习人际交往中的一个核心概念——**框架**，并掌握一种名为**推拉**的技巧。我们将通过分析一个经典电影片段，来理解如何在两性互动中建立、维护自己的框架，从而保持吸引力，让关系健康持久地发展。

## 什么是框架？

框架是一个人为人处事的原则。每个人都应该有自己的框架。当我们和女生相处的时候，难免会碰到框架碰撞的情况。碰撞的结果无非就两个：这个事情最后是按你的意愿做，还是按我的意愿做。

如果我们在和女生相处的过程当中，永远服从女生的框架和意愿，按照女生的意愿去做，那么我们的框架就会变成“按照女生的要求来”。渐渐的，你的框架就会越来越弱，你的硬框架会越来越少，软框架会越来越多。

你的可得性会越给越高，你的吸引力会越来越小。如果你对她百依百顺、亲近所有，也许在她的理智里面，认为你确实是对她很好，她应该同样的去爱你。可是在她的荷尔蒙系统里面，她会发现自己渐渐的已经不爱你了。

因为你的吸引力在不经意之间就流失掉了。你以为这是爱她的一种方式，其实是以爱之名来掩盖自己的窝囊和懦弱。因为你的脑子里面没有框架的意识，所以你会流失掉吸引。她不喜欢你是必然的一个结果。

男人必须要有自己的框架，而且在发生框架碰撞的时候，我们也要能守住自己的框架。

## 如何守住框架？——引入“推拉”技巧

守住自己框架的方法就是使用**推拉**。为了更直观地理解，我们来看一个电影中的例子。

上一节我们介绍了框架的概念及其重要性，本节中我们来看看如何运用“推拉”技巧来维护它。

电影《查特莱夫人的情人》中有一个精彩的桥段。女主角是地位崇高的伯爵夫人，男主角是伯爵农场里的一个护林员（仆人）。电影讲述了仆人如何追求到伯爵夫人的故事。

有一天，伯爵夫人和仆人来到了平民区。道路狭窄，伯爵夫人不小心被路人撞了一下，对方没有道歉就走开了。伯爵夫人很不高兴地说：“真没礼貌，我们应该好好地教训他一下。” 然后这个男生说：“那你去教训啊。”

伯爵夫人用一种高高在上的眼神看着他，不满地说：“你好像很喜欢看我出洋相嘛。” 此时，伯爵夫人的硬框架显示出来了，这是一种审视仆人的、高高在上的感觉。她的潜意识里，觉得自己是上等人，对下等人的冒犯感到不满。

但这个男生本身就是下等人。他是这样回应的：

> **“要么呢，你是一个女人，要么呢，你是查特莱夫人，但你不能两个都是。”**

这句话的意思是：你现在和我在约会，要么你就把自己当做一个与我平等的普通女人；要么你就当你的伯爵夫人，作为我的上司而存在。这两种身份不能重叠，我对待你的态度也不同。你不能既想和我在一起，还想摆出主人的姿态，我不接受这种不平等的相处方式。

说得再直白一点：要么你乖乖做我的女人，适应我的生活环境；要么你就做回伯爵夫人，我也跟你保持距离。

在现实生活中，每个人都可能有多重身份，所以在面对另一半时，时常会不自然地摆错身份、用错态度。但绝大多数男人在与女生相处时，根本意识不到这一点。他对自己的框架不够敏感，没有意识到这可能触犯到了自己的框架，需要及时调整或打断。时间长了以后，他自己的框架就越来越模糊，他在女生心里也会越来越不受重视，最后就可能导致莫名其妙地被女生嫌弃甚至分手。

## 推拉技巧的实战分解

在电影里，当男生对女生说完那句话之后，女生瞬间感觉到了这个男生的自信和男人般的感觉。她感觉到了强大的一个框架。她不但没有生气，眼神当中反而流露出了一些崇拜和欣赏。

**为什么呢？因为硬框架增加了这个男生的吸引力。这就是一次“推”。**

“推”是指对女生的框架表示出不服从、不认同的态度。推的结果就是你的可得性会暂时减少。

这个时候，男主人公又做了另外一件事情：他轻轻地把女生搂到了自己的怀里面。

**这个是“拉”。**

也就是说，当女生服从了男生的这个框架，当她的眼神当中流露出了欣赏和崇拜之后，男生再把女生的头靠到自己肩膀上。这个动作表明：我希望你是我的女人。结果，女生就很顺从地靠在了他的怀里。这也是回应他：我才不要做什么上流社会的伯爵夫人，我要做你的女人。

这说明女生已经服从了男生的框架，而男生也就保持住了自己的框架。

然后这个男生又做了一件事情：他去替女生教训了刚才冒犯她的那个路人（用锤子砸了一下他的脚），然后拉着女生一起跑掉了。结果女生的情绪一下子就变得非常好，她大笑着问：“这边的生活都是这样有趣的吗？”

这个时候，女生已经完全进入到了男生的框架里面：我是你的女朋友，并且我欣赏你，我觉得和你在一起非常有趣。

## 先推后拉的重要性

大家可以想一想，如果当这个女生说“你快点帮我教训他”的时候，这个男生立马就过去教训她了，那么女生心里会有什么样的反应？

她会觉得：这本来就是你应该做的。你做得好是正常的，做得不好你会减分。

但是在电影里面，男生是在女生服从了自己的框架之后，再去替女生教训了那个路人。那么相当于，他做的这件事情就变成了一个奖品：**因为你服从了我的框架，所以我奖励你一个帮你出气的机会。**

这样一来给女生的感觉就完全变了。女生会从一种“理所当然”的心态，变为“哇，好开心好感动，原来我在你心里是这么的重要”。

所以说，框架推拉一定要**先推后拉**。你总是舍不得推、不敢推，总是往里拉的话，女生就会觉得没什么意思了。慢慢的，你和她的互动模式就会变成了她高高在上，然后你跪在地上跪舔。

## 长期关系中的框架维护

在谈恋爱的时候，特别是很多长期关系当中，有很多男生被女朋友劈腿了，都会跑过来对我说：“你看我做了这么多，我对她这么好，为什么她要这样对我？难道她一点都不记我的好吗？难道她一点都不懂得感恩吗？她为什么这样的坏呢？”

其实不是女生坏，而是你虽然做了那么多，但是没有让她感觉到你做了那么多。这个就是方法方式经营的艺术。也就是说，你做了100分的事情，但是对方只收到了10分或者1分。因为你一直拉一直拉，你的框架永远都是软的，她就麻木了，觉得你对她好是理所当然的。所以她不会对你有所感激，也不会觉得这样有什么不公平。当你的吸引力终于流失得差不多的时候，她就会毫不犹豫地离开你。

那么同样的事情，有的人只做了50分，但却可以让对方收到100分的感觉，就比如那个电影的男主角一样。这就像是两个成语说的一样：一个叫做**事半功倍**，一个叫做**事倍功半**。

## 现实场景应用：当女生提出强硬要求时

比如你的女朋友经常很强硬地跟你说：“你必须给我买这个，必须给我买那个。” 这个时候你应该怎么做？

首先你要想到自己的框架是什么：
*   你喜欢被别人命令吗？
*   你喜欢被别人强迫吗？
*   钱是你的对不对？你想怎么花需要别人来指指点点吗？

特别是在生活当中，有一些女孩子她会说：
*   “你不给我买我们就分手。”
*   “你不给我买就是你不爱我。”
*   “你不给我买你就别碰我。”
*   “你不给我买的话我就找别人去买。”

如果她每次威胁你的时候都可以得逞，那你的行为就在教给她一件事情：**你以后只需要变本加厉地威胁我就可以了，只要你威胁我就可以达成你要的目的，不然的话你就达不成你的目的。**

一旦你用行为教给女生这样的认知，那么你们的关系在后面就非常危险了。你的女朋友可能会形成一种这样的思维惯性：跟你说话不能太温柔，只要打骂才会达到我的目的。

## 推拉结合：维护框架与补足可得性

有人可能又会担心了：我这样维护自己的框架，那我的可得性就会流失了，万一女朋友生气了埋怨我，或者真的跟我分手的话怎么办？

很简单，你再找一个机会去补足这个可得性就可以了。

比如说过了三四天，你们又出来见面了，然后你突然把她想要的那个包包拿给她。她看到之后肯定会非常惊喜，问你说：“哎，你怎么又给我买了？你上次不是说不给我买吗？”

那这个时候你就可以说：“上次大爷心情不好哈，今天看你穿的这么漂亮，我特意给你变了个戏法魔术，把这个包给你变出来了。你以后要乖乖的哦。”

那么这个其实就是“拉”，和之前的拒绝（推）就形成了一个**推拉**。那么女生的情绪就会有一个波动。最后她也得到了她想要的东西，而且你也没有被她逼迫。这个就是女生在恋爱当中需要的一种浪漫的感觉，也是你们的关系能够平稳、长久、健康的一个大的基础。

## 核心要点总结

本节课中我们一起学习了框架推拉的内容。

框架推拉对于我们和女生产生框架碰撞的时候，会起到一个保持自身框架、保持吸引、引起女生情绪波动的作用。

当女生表现出让我们不舒服的强框架的时候，我们不能以爱的名义去妥协，去不情愿地完成她的要求。而是应该清晰的表现出自己的框架是什么（比如电影里的框架就是：我和你是平等的，我不是你的仆人，我是你的男朋友）。

所以这个时候你应该拿捏好分寸，根据具体情况去选择“推”。当她表示了服从的框架之后，你再找机会去补回自己之前流失掉的可得性，让她知道你是爱她的（比如给她小礼物、小惊喜），再把她“拉”回来。

这样有了推拉的过程以后，你在女生面前就能够保持住自己的框架，你的吸引力不会流失，同时又不会让她觉得你太自我，因此你的可得性也会很高。

**一个关键细节：** 当你“推”完她之后，在选择开启“拉”的这个时间点，不能够是在女朋友还在生气的时候。因为她生气的时候，她对你没有做到的这个事情还是保有高度的预期，并且她对你对待她的这个态度十分不满。所以应该等到她的气生完了之后，她对这个事情已经不抱预期的时候，在这个时候你再把她之前想要的东西给到她，那么就会有额外的惊喜。其实浪漫就是源于这种意外的惊喜。
