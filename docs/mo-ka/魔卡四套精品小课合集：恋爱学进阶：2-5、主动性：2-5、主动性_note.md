# 魔卡恋爱学进阶课程 2-5：主动性 🎯

在本节课中，我们将要学习男性在恋爱关系中的核心特质——主动性。我们将探讨为何主动性至关重要，以及如何正确运用它来建立和推进关系，同时避免常见的错误。

## 课程概述

上一节我们讨论了吸引力的基础，本节中我们来看看如何将吸引力转化为实际行动。男人的主动性，就如同精子向卵子的游动，是关系发展的根本驱动力。一个具备主动性的男人，需要融合**胆大**、**心细**、**脸皮厚**三种特质。

简言之，一个男人要能够成为一个吸引女人的高手，就必须具备**胆大**、**心细**、**脸皮厚**三种特质。

有的学员在咨询摩卡的时候问摩卡一个这样的问题：为什么我天天晚上给她打电话，每天晚上都和她聊一两个小时，但她却从来不会主动地找我呢？她是不是不喜欢我呢？

那我就想反问这个学员：你打电话给她，你们可以聊一两个小时，说明你的吸引力已经非常大了呀。女生愿意和你这样聊天，花这么多的时间，她已经对你相当主动了。那你还要她表现得多么明显呢？

一定要记住：**男人的主动是进攻，女人的主动是在男人进攻的时候给予配合**。女人的主动和男人的主动是不一样的，它是两种不同表现形式的主动。

很多人在恋爱的时候，他就是像小明。人都很好，心地也很善良，但是在爱情当中却每每碰壁。

这种主动，它是一种不按照规则行事的主动，没有心细，只有胆大和脸皮厚的部分。这是一种非常常见的普通男人的一个误区，就是什么？太猴急了。

其实不是不能和女人说性，而是不能在**没有相互了解、没有吸引、没有筛选**的情况下，直接和女人聊性。

那么今天摩卡讲的是男人在恋爱当中的主动性。男人在爱情中应该扮演主动进攻的角色，不要怕失败，也不要怕被拒绝。相比于处处瞻前顾后的好男人，女人更喜欢胆大心细、脸皮厚一点的坏男人。
