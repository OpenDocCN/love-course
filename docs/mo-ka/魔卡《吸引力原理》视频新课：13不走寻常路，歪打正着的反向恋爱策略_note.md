## 📚 课程概述

上一节我们介绍了恋爱发展的六阶段模型。本节中，我们来看看另一种常见但特殊的模式：反向吸引力模型。这种模型描述了两个人从最初的冲突和对立，逐渐转变为相互吸引和爱慕的过程。

## 🎬 案例场景分析

我们将通过一个包含多个回合互动的故事案例，来拆解反向吸引力模型的运作机制。

### 第一回合：冲突开局

案例中，男生（陆远）和女生（许莉）在第一次见面时爆发了明显冲突。两人针锋相对，极大地刺激了对方的情绪。男生给女生留下了**下流、无礼、傲慢**等负面印象。

**核心机制**：这种开局完全避开了过早“暴露需求感”的雷区。与六阶段模型逐步营造正面情绪不同，这种模型始于强烈的负面情绪刺激，让女生印象深刻。

我们可以用一个简单的坐标轴来理解情绪变化：
```
初始情绪值: -10 (强烈负面)
```
这个阶段，女生的情绪是向下走的。

### 第二回合：印象扭转

基于一个自然合理的原因（帮助女生的家人），两人进行了第二次互动。这次，男生表现出**善良、孝顺、有担当**等良好品质。

**效果**：女生对男生的印象回到了社交初始的零点附近，抵消了部分首次见面的极端负面印象。
```
当前情绪值: ~0 (中性，回到起点)
```

### 第三回合：特质展现与保护

在第三个互动场景中（餐厅偶遇前夫），男生“胆大心细脸皮厚”的特质再次展现。但这次，这些特质表现为**保护性**的（为女生出头解围），而非第一次的进攻性。

**效果**：这次互动为女生建立了非常好的舒适感和安全感，心动指数首次出现正面提升。
```
当前情绪值: +5 (正面提升)
```

### 第四回合：深度共鸣与心疼

在第四次互动中，男生向作为心理医生的女生袒露了一个深藏的秘密（因事故失去味觉，并为此坐过牢），狠狠地在女生心里刷了一波“心疼指数”。

**效果**：到此，男生在女生心里留下了**喜、怒、哀、乐**等丰富的情绪体验，且始终没有暴露对女生的需求感。
```
当前情绪值: +10 (强烈正面，混合心疼、理解等复杂情绪)
```

## 💡 核心原理总结

本节课我们一起学习了反向吸引力模型的运作机制。

**核心公式**：
```
情感投入 ≈ 情绪波动幅度 × 波动频率
```
女人是情绪化的动物，对情绪的感知能直接影响荷尔蒙系统。**谈恋爱就是在谈情绪**。如果你能让一个女人因为你而产生强烈且多样的情绪波动（无论是正是负），她就离爱上你不远了。

**关键点**：
1.  **由负转正的能量**：初始的负面印象（恨/讨厌）越强烈，后期扭转后产生的正面吸引力（爱）也可能越深刻。这类似于“放下屠刀，立地成佛”与“一辈子吃斋念佛未必成佛”的对比，改变固有印象（尤其是坏印象）需要巨大的能量，一旦成功，效果显著。
2.  **避免需求感**：在整个过程中，男方没有表现出对女方的需求，这使得情绪推动力显得更“纯粹”，并非刻意讨好。
3.  **情绪完整性**：模型提供了完整的情绪体验闭环（厌->惊->安->怜->爱），比单纯提供正面情绪更具冲击力和记忆点。
