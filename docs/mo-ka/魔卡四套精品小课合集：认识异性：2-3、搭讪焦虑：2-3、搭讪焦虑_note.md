## 概述：什么是搭讪焦虑？

你或许有过这样的经历：在街上或地铁里，看到心仪类型的女生经过，内心很想认识她，却不敢上前说话。你担心在众目睽睽之下这样做太冒险。你可能会想：如果对方不理我，我会当众出丑。即使成功要到联系方式，对方也可能不愿意聊天，所以不搭讪也无所谓。这些看似充分的理由，其核心目的只有一个：**说服自己不要搭讪，不要冒险，留在舒适区内**。

我们之所以寻找各种理由说服自己，根本原因在于对搭讪抱有深深的焦虑感。

## 两种主要的搭讪焦虑

上一节我们介绍了搭讪焦虑的普遍性，本节中我们来看看焦虑的具体类型。一般来说，搭讪时的焦虑主要分为两种。

### 1. 接近焦虑

**接近焦虑**是指男性首次接触陌生女性、准备上前搭讪时所感到的焦虑。这种焦虑源于我们对即将发生的事情产生了**负面的判断和想象**。

在多年的培训与教学中，我发现很多男生在与女生聊天、发信息或准备约会时，都有类似的焦虑。后来我将这种情况统称为 **“叶公好龙”**。自己明明很喜欢、很期待，但当机会真正来临时，却又选择退缩和放弃。这无异于“叶公好龙”，并且当事人往往还会为自己找一大堆理由。

我认为，如果这种“叶公好龙”的思维方向不加以改正，那么出问题的可能不仅仅是感情，它还有可能在我们人生的方方面面阻碍我们取得成功。因此，我们可以通过搭讪这种社交练习，来有意识地训练自己。有句话叫做 **“冒险创造价值”**，谁知道你这一次小小的搭讪冒险，会为你带来怎样的人生价值呢？

### 2. 情景焦虑

**情景焦虑**是指我们在面对不熟悉的环境时，对潜在的不确定性产生的焦虑。其本质依然是对即将发生的事情抱有负面判断。

以下是情景焦虑的具体表现：

*   当你进入一个陌生的环境，会感到非常不自在，甚至不知所措。
*   你不想与他人交流，整个身体处于对外界防备的状态。
*   例如，进入一家内衣店、陌生酒吧或他人主办的派对时，你无法自然地让眼神落在某个角落，会东张西望。
*   你的面部肌肉不自然地紧张，导致表情拘谨，如同戴上了面具。
*   走路时刻意控制脚下的速度和步伐。
*   坐下后身体无法完全放松，后背僵直，手脚感觉无处安放。

此时，你的理智会被屏蔽，进入一种**自我审视的状态**。你会无限放大自己的行为，认为所有人都在关注你的一举一动，等着挑你的毛病，让你丢脸。你希望躲进角落，不让任何人注意到自己。而你越这么想，你的状态和表现就会显得越紧张、越拘谨。这种焦虑就是由**环境变化引起的情绪变化**。

## 如何应对搭讪焦虑？

接近焦虑和情景焦虑在我们打算搭讪时都可能存在，它们会迫使我们找一大堆理由不去行动。有接近焦虑其实很正常，但在焦虑出现的同时，我们仍需清楚自己到底想要什么。这本身也是一种人生态度，可以通过搭讪练习来养成。

我们要去搭讪，就不能让焦虑影响行动。即使焦虑存在于体内，我们依然要过去搭讪，并学会控制它。

### 接纳并表达你的紧张

假如你过去搭讪后发现自己很紧张，那么就在第一时间**捅破这种紧张**即可。

例如，如果你发现自己声音颤抖、说话结巴（“我这个…那个…”）或手心出汗，你可以直接对女生说出来：

```plaintext
“我有点紧张，不好意思。”
```

你还可以进一步说：

```plaintext
“我平时没怎么做过这样的事情，但我确实觉得你气质很好，挺喜欢你这种类型的女孩子。现在跟你说话真的比较紧张，你给我一点时间，要不我们重来一次？我深呼吸一下，重新来一次好吗？OK，我叫小明，是这样子的...”
```

有时，如果你这么说，女孩子甚至会被你的行为逗笑。她会觉得你很有意思，并且很真实。

紧张时不要绷着，要去**接纳自己这种紧张的状态**。因为就算你不说出来，别人也知道你在紧张。女生看到你额头冒汗、眼神躲闪、表情不知所措、手脚发抖，你说自己不紧张，她也不会相信。**把紧张说出来，反倒是一种缓解社交尴尬的方式，也表现出了你的坦诚和勇敢。**

### 一个类比：公开演讲

这就像我们第一次做公开演讲。第一次上台，台下黑压压几百人看着你，你也会紧张。因为你在用一个人的气场对抗几百人的气场，或者你缺乏相关经验。这种情况下，你肯定会紧张，生怕说错话或出岔子被人嘲笑。

那么该怎么做呢？你可以直截了当地说：

```plaintext
“我第一次在这么多人面前说话，其实现在特别紧张。大家能不能鼓励鼓励我，给我点掌声？”
```

然后大家一鼓掌，或者底下有人喊“加油，别紧张”，你可能就会把这种紧张感释放掉，从而进入正常的交流或演讲状态。

**总之，千万不要一感到焦虑，就认为自己做的事情是错的，认为自己不该去做这件事。**

## 总结

本节课我们一起学习了搭讪焦虑。我们在搭讪女生时会感到焦虑，这与性格内向或外向无关，也与是否害羞无关。**搭讪焦虑是每个男性在搭讪时都会有的一种正常情绪。**

搭讪焦虑主要分为两种：
1.  **接近焦虑**：接触陌生异性时感到的紧张与激动。因为潜意识认为搭讪异性会遭到攻击，产生负面判断。这是埋藏在我们基因里的一种**自我保护的本能反应**。
2.  **情景焦虑**：面对陌生环境时，对潜在不确定性产生的焦虑。你会因此紧张并放大自己的行为，生怕被别人挑出毛病。

**焦虑是本能，我也有焦虑。但我们要学会用理性来驾驭本能。** 我们需要在焦虑的同时，清楚自己的目标，不让焦虑影响目标的执行。

你可以直接向女生坦白你的紧张，因为你之前没做过这样的事。女生不会像你想象的那样嘲笑你、责备你，甚至给你一巴掌。**只要我们有礼貌、不带攻击性，女性普遍不会对搭讪自己的男性感到太厌恶。** 毕竟女性没有必要讨厌一个欣赏自己的人，哪怕她并不喜欢你。

当然，说一千道一万，还是要注意自己的举止得体，言语有礼貌，尊重对方，也尊重自己。
