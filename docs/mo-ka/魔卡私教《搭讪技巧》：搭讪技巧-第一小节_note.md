## 搭讪的本质：社交负重练习

前两天有同学私聊时谈到，觉得搭讪是一种较新的认识异性的方式，似乎不被社会主流广泛认知或接受，甚至有点像“耍流氓”。我需要再次强调：街头搭讪是一种**社交负重练习**。

**公式：街头搭讪 = 社交负重练习**

一旦你通过练习掌握了这种能力，意味着你可以在一个陌生人面前，在街头，面对一位你感兴趣的女性，能够做到自如、礼貌、得体的社交表达，并与对方产生有质量的互动，赢得良好的第一印象。掌握这种能力后，你会发现，在公司年会、展览等任何有特定场合属性的社交环境中，与人结识都会变得更容易。因为这些场合本身提供了共同话题或圈层背景。

街头搭讪则不同。你与搭讪对象可能不属于同一社会阶层，没有共同兴趣爱好，风格迥异。你可能会遇到各种类型的人。因此，大家一定要明确：街搭是一种练习，是打开和锻炼你社交能力的**负重训练环境**。

就像去少林寺学艺，前两年可能是在劈柴挑水，练习基本功。跑步时腿上绑沙袋，也是负重练习。我再次强调，**街头搭讪是我们练习社交技能成本最低、频率最高的一种方式**。

当然，如果你能通过街搭本身认识一些人，甚至像我的某些学员一样找到伴侣，那当然更好。但我们要清楚，街搭本身通常不会直接产生太多结果。你不应因此进行自我打压或否定，更不应将其作为“自己不行”的借口。一定要搞清楚，当你在街搭上能做到七八十分的水平，在特定场合（非街头）的表现会更好。这种练习是零成本的，只要你克服了搭讪焦虑。

我们之前鼓励大家“日行一善”，即每天尝试搭讪一次，不论成功失败，旨在养成习惯：在任何场合，主动、得体、礼貌地向自己有好感的异性表达交流或认识的意愿。这绝不是耍流氓。**耍流氓是因为这个人本身就是流氓，而不是因为他做了某件事。** 事情本身不带正负属性，不同的人做同一件事，结果不同。坏人会把事情做成坏事，好人会把事情做成好事。这与事情本身无关。

## 社会观念的变迁

现在大家看抖音、快手等平台，90后、00后的大众已经广泛接受了“撩”或“搭讪”的概念。最常见的情景就是：“前面有个小姐姐，我们去撩一下。” 这类内容在平台上大量出现。

这在几年前是不可想象的。大约在2012到2014年我们密集进行搭讪实践时，无法想象在2018年后，这种“互撩”、“搭讪”的文化会变得如此普及。这从侧面证明，我们当初所做的属于新兴文化萌芽阶段，现在已被大众广泛接受。

昨天我与一位投资人聊天也谈到此问题。他提到，随着PUA文化的曝光和短视频中大量搭讪内容的出现，将来搭讪这种行为可能会逐渐消失。我觉得很奇怪，既然它已从小众变成大众，将来怎么可能消失？只可能变成我们生活中司空见惯的一件事。女孩子可能经常在街上被搭讪，男孩子也经常在街上搭讪。大家会将其视为一种平常、正常的社交或认识异性的手段。因为它天天发生，每时每刻都在发生，所以大家会慢慢习惯。它绝不会消失，只会愈演愈烈。

## 认识异性的五大渠道

前面闲谈了许多，现在我们正式开始今天的课程内容。

首先要说明：陌生搭讪，与大家看到的图片中的四种渠道，并称为认识异性的**五条渠道**。你找不出第六条。所有认识异性的渠道，按性质分类汇总，只有这五种：

1.  **天生缘分**
2.  **熟人介绍**
3.  **陌生搭讪**
4.  **社交软件**
5.  **相亲媒介**

以下是每种渠道的详细说明：

*   **天生缘分**：指一切自然而然的认识，例如同班同学、公司同事、拜访客户时认识的前台等。女生对这种“缘分安排”最有浪漫预期和幸福感。当然，它的效率最低，无法人为设计，只能“看天”。
*   **熟人介绍**：很简单，通过你认识的人（社交桥梁）介绍认识另一个人。可以是兄弟的朋友的妹妹、同学的表妹等，在任何聚会、派对上经由朋友、亲戚、同事介绍认识。

*   **社交软件**：大家很熟悉，从早期的陌陌、微信（附近的人、摇一摇），到探探、MarryU、美丽约、你好陌生人等同城交友软件，以及早期的QQ，都属于社交软件。

社交软件和相亲媒介有共同的好处和坏处。共同好处是**效率高**，你可以批量认识很多人。但反面带来的问题是认识**质量较低**，且存在大量骗子。对方可能是“抠脚大汉”冒充，或通过修图、美颜呈现虚假形象。现在甚至有骗子公司规模化运营，购买成套的女性社交资料（照片、视频）来伪装身份，在各大平台“撩凯子”骗钱。同时，现在也有女性学习“撩汉”技巧。竞争环境日益复杂和白热化。

希望大家通过学习摩卡的课程，提升自己的鉴别能力、沟通能力、管理能力和社交能力，在这片红海中脱颖而出，收获自己的果实。

## 搭讪的独特优势

相对于上述四条渠道，搭讪有几个明显优势：

1.  **自主可控**：你可以完全主动地安排，无需等待“缘分”或他人介绍。
2.  **效率高**：如果你坚持“日行一善”（每天搭讪一次），一年可出手365次。
3.  **真实可靠**：不会出现“货不对板”或照片诈骗的情况。你在街头搭讪到骗子的概率极低。
4.  **即时筛选**：你可以第一时间看到对方的真实样貌、气质、穿搭层次，判断是否与你匹配，避免线上长期聊天后见面失望的情况。

我们可以做一个简单的数学推演：

**代码/公式示例：**
假设每日搭讪1次，年搭讪次数 = 1 * 365 = 365次。
假设收号成功率30%，则年收号数 ≈ 365 * 0.3 ≈ 110个。
假设其中30%能进行有质量的微信互动，则优质联系人 ≈ 110 * 0.3 ≈ 33个。
假设其中30%能进行线下约会，则可约会人数 ≈ 33 * 0.3 ≈ 10个。
假设其中30%能发展深度关系，则潜在发展对象 ≈ 10 * 0.3 ≈ 3个。

这意味着，对于一个掌握搭讪方法的男生，一年内可能找到3-4个可以纵深发展高质量关系的目标。对于一个普通男生而言，这样的感情生活频率和状态可以达到70分左右。如果在二十多岁的几年间持续实践，你会积累丰富的感情经验，变得更加成熟，从而在未来做出更明智的婚姻选择，提高婚姻质量。这是摩卡多年来自然形成的认知逻辑，供大家参考。

## 克服搭讪焦虑

有同学问：“不敢搭讪，是不是对自己不够自信，比如对颜值、装扮等自身形象不自信？” 这有一方面原因。如果你状态很好，自信度会提升。但这并非根本原因。

不敢搭讪，最主要是因为你有**焦虑**。昨天在“窗口识别”最后一课我们提到，男性一般有三种焦虑：出现在认识阶段的**搭讪焦虑**、出现在诱惑阶段的**关系升级焦虑**、出现在交往阶段的**关系确定焦虑**。

焦虑的根源，都来自于对**未知的、即将发生的事物**有一种不可掌控或不可预知感，并且伴有相对负面的担心。如果你在搭讪前就知道对方会给你良好反应，甚至会成为你未来的伴侣，你就不会有任何焦虑。

具体到搭讪，你脑中可能浮现各种恐怖画面：对方男朋友冲出来打你、对方父亲指着你骂、对方扇你巴掌并叫警察……这些大多来自影视剧的夸张渲染。在实际街头搭讪中，**我遇到过最差的反应就是对方无视你、不理你、或翻个白眼离开**。只要你没有说出冒犯的话或做出猥亵行为，仅仅礼貌地表达认识意愿，对方采取激烈对抗行为的可能性为零。

所以，一旦脑中开始编造“恐怖片”，你要提醒自己：**“我现在开始编剧了，而且编的是神剧。”**

## 关于社交评价的担忧

有同学担心频繁搭讪会有“社交后遗症”或被打上“渣男”标签。在人口流动性小的小乡镇或学校，频繁搭讪同一批人，确实可能产生负面评价。但在大城市，这种风险极低。

试想，如果家里给你安排二十场相亲，没人会因此给你贴“渣男”标签。同样，你在街头礼貌、得体地向有好感的异性表达认识意愿，为什么就会被贴上标签呢？即使上《非诚勿扰》，也是同时与二十四位女生互动相亲。

搭讪只是一种**认识渠道**。你们通过搭讪认识，与通过相亲、熟人介绍认识，在性质上是一样的，都处于**认识阶段**。认识之后，关系仍需经历了解、吸引、追求、暧昧、交往等阶段。**千万不要把“认识阶段”的行为等同于“追求阶段”或“诱惑阶段”的行为**。认为“我搭讪你就是在勾引或追求你”，这种等式既不必要，也不合理，更不符合实际逻辑。

我反复强调**恋爱六阶段**：
1.  认识阶段
2.  了解阶段
3.  吸引阶段
4.  追求阶段
5.  暧昧阶段
6.  交往阶段

搭讪与其他一切认识异性的行为一样，只发生在**第一阶段：认识阶段**。不要将一个阶段的行为等同于其他阶段的行为。随着大家对情感认知的理性化，以及摩卡课程的普及，这种误解会逐渐消除。感情看似感性混乱，实则有其内在逻辑结构，就像化学元素的不同组合。

## 搭讪的挑战与唯一道路

虽然搭讪有诸多好处——效率高、防欺骗、自主可控——但它也有一个明显的挑战：**你需要战胜自己的心态，突破内在的焦虑**。它对内在心态、外在举止、话术技巧都有综合要求，其中最大的要求就是心态。你需要自信、自然、勇于自我突破的主动心态。如果没有这个基础，所有技巧、话术都无效。而且大家学习摩卡课程这么久，应该明白：你心里怎么想，别人是能感知到的。

这或许可以算作搭讪的一个挑战，但严格来说不能算坏处。因为一旦你能对抗并战胜这种自我压力、自我设限和不舒适感，对你整个人的精神面貌、个性及吸引力的提升都只有好处。

当然，这一关对很多人来说确实难过，尤其是那些自身能量低、自信度低、焦虑感重、内向胆小，或曾受过情感伤害的人。但无论突破难度大小，方向是确定的：你最终只有走上**不断提升自我、突破自我、主动社交，以实现情感自由**这条路。拖延、犹豫、逃避，只会让你离理想的情感生活越来越远。所以，既然面前只有一条路，不如早点开始。
