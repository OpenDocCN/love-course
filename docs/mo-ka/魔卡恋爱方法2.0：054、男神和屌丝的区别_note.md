上一节我们介绍了课程主题，本节中我们来看看“男神”与“屌丝”的根本差异。许多人会从外貌、财富、社交能力或专一程度等方面来区分两者。然而，所有这些表象都可以归结为两个核心参数：**价值**与**能量**。

男神的**价值**和**能量**都很高，而屌丝的**价值**和**能量**都很低。
