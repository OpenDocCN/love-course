# (老景599)2.0拥有强大自信理论篇：第7节.第六课：改写潜意识_note

在本节课中，我们将要学习自信构建中最核心、最关键的一步：如何改写我们的潜意识。我们将深入探讨为什么仅仅“知道”道理远远不够，以及如何跨越从“知道”到“做到”之间的巨大鸿沟。

## 从“知道”到“做到”的鸿沟

上一节我们介绍了许多颠覆性的思维模型。本节中我们来看看，为什么掌握了这些知识后，我们的生活可能依然没有改变。

到目前为止，我们已经学习了整整五课的内容。这五课里讲了很多颠覆性的知识，例如世界的底层逻辑是“反”的，核心自信与情境自信的概念，以及“比达小语”原则、弄不动、倒推法等。这些内容无疑颠覆了我们以前的全部认知。从这个角度去看我们的人生和世界，会发现前所未有的星辰。我们甚至会产生一种念头：懂得了这些东西以后，我们应该可以过好人生了。

但真的是这样吗？进入第六课，一开始我就要给大家泼一大盆冷水。大家现在知道了这些东西，有用吗？答案是：没用。你只是知道了。我从2003年开始踏上寻找真相的道路，6年后的2009年，所有这些知识我全都知道了，但是，没有用。从我知道这些东西，到我最后做到，又经过了整整9年的时间。摸到这些东西、懂得这些道理用了6年，懂得以后到做到用了9年，那是一个非常漫长的过程。

明白这些道理以后，尤其是比达小原则、弄不动、倒推法等，我知道以后我以为我的人生要颠覆性地改变了，要心想事成了。我当时就想，这些方法这么好，那我就按照这样的方法去做事吧。虽然我知道我还是一个没有底气的人，可能不那么自信，但是方法很好，我用这种方法去做事情就好了。一句话：装嘛。那我就装一辈子好了。虽然我知道我这个人不是那样的人，但是我知道怎么样去做成一个那样的人，方法我都知道，这属于技巧。我用这个技巧包装自己，把真正的自己隐藏起来，去做这些表面的功夫，让别人感觉我很厉害。要装，装全套，装一辈子。我就是这么想的，也是这么做的。

但是整整九年的时间没有任何结果，没有拿到任何结果。小小的成功有那么一点点，大成功一个都没有。无论是和别人搞好关系，还是我的爱情等等，没有任何突破。九年之后，到了2018年，我还在单身。我在成都碰到了一个以前认识的朋友，一个女孩子。认识的时候她还年龄很小，2018年碰到的时候，她年龄挺大了，30多岁了，我40多岁。两人见面以后觉得还蛮有缘分的，这么多年又碰到了，都觉得彼此挺好，咱们试一试吧。然后我就发现，我喜欢上人家了。

然后心态就崩了。所有的技巧我都知道，所有的招数我都知道，但是我全都使不出来了。没有那个心态作为支撑，这些表面的东西我做不到。最后就越来越紧张，生生地把这个事情给搞黄了。一个人灰溜溜地回到北京，心情特别低落。我认为我寻找了整整15年的时间，怎么是这样一个结果。那时候我对自己失望了，我觉得可能这个东西是做不到的吧。知道很容易，但是做到可能真的做不到。

那段时间我的整个世界都是灰暗的。记得有一天，我去中关村谈一个剧本（我是做编剧的）。回来的路上一直在想这个事情，心情特别低落。那天北京也是昏沉沉的，三月份的春天，我开着车从西边往东面走，过了北太平庄桥，去马甸桥的路上。就快到马甸桥的时候，我脑子里突然闪出一句话来。之前我一直在想这个问题，在思考到底怎么办，是不是没希望了。那一刻我突然脑子里蹦出一句话来：“老景啊老景，说你这些年都在做什么呀？你一直在拼命地、努力地、竭尽全力地去做一些事情，想把这些事情做好，来让别人喜欢上你、尊重你。试问，如果你要是厉害的话，为什么不是别人努力地去做一些事情、做好，来让你喜欢上他呢？”

那一刹那我愣住了，然后车子就冲下马甸桥了。桥下是黑的，但一瞬间我就明白了。这时候车子冲上来，那时候的感觉就是眼前一片白光。我事后明白，那个就是开窍的一刹那。那是我这辈子最重要的一个时间点。开窍的一刹那，之前明白了这些道理、钻研出了这些理论，都没那么重要。因为从知道到做到之间，还有一段漫长的路要走。

从那以后，我明白了所有的来龙去脉，开始去研究这套课程，研究训练方法。五个月之后，我把自己训练出来了，破冰了。破冰之后，人生发生了多大的变化呢？用四个字来形容，叫做翻天覆地，或者说沧海桑田。我曾经梦想中的世界，梦想中能够完成的东西，它就一个一个地实现了。无论是我和别人的关系、我获得别人的尊重、我在别人心里面开始慢慢地有位置、我自己觉得自己有底气了、开始很多人喜欢我，包括我的爱情关系等等，甚至后来包括我的事业也越走越顺。梦想变成了现实。那个过程，你不经历，你是不知道有多么神奇的。这个经历我自己，和所有之前训练出来的学员，我们都经历过。你曾经连想都不敢想的东西，最后有一天居然会拥有。这件事情太神奇了。

这就是我的经历，这也将是你们所有人训练完毕的经历。那是什么感觉呢？比如说咱们拿赚钱来说。我以前赚钱，每年年入几万块、十几万块，我也就想一想，要是能年入几百万、几千万、几个亿多好啊，想一想而已，甚至都不敢想，不觉得它是我能达到的、能属于我的。但是突然发现有一天，我真的做到了。我跨越了年入几十万，从几万跨越了几十万，直接奔上百万、上千万。我突然发现，赚钱这么容易啊？那时候你会觉得惊奇的。这就是我经历过的事情，很多学员经历过的事情。

那么好，你想经历吗？但是还差一步。到现在为止，五课结束，我们已经确实地知道了很多更高维度的道理和知识。我们觉得我们的大脑被撬开了，思维发生了翻天覆地的变化。我站得更高了，看得更远了，看得更深了，很多东西我明白了。但是咱们过往的人生，这种情况也有过。我们明白了无数的道理，学会了很多东西，但是你真实起来，好像不那么起作用，结果好像也不那么尽人意。为什么没有发生刚才我说的颠覆性的变化？从几万跨越几十万，直接奔几百上千万，没有经历过这些东西。差在哪里了？为什么？

因为一个人知道一些东西，到有一天能做到，有一段路要走。这对我来讲，是跨越那道门槛，九年的时间跨越那道门槛。这个时间我已经给你省下来了，但是剩下的，你要做的是从知道到做到，你还差一年的训练，朋友们。再重复一遍：从我们知道这些东西，到有一天我们能做到这些东西，你还差一年的训练。所以这个课程不是光看就行了，它是一套训练课程。

## 获取潜意识的支持

刚刚我们说到，从知道到做到有一段漫长的路要走。这段路到底在做什么事情呢？先说答案：**获取潜意识的支持**。

为什么我们很多时候，自己觉得我很想要这个结果，我也挺努力的，但是拿不到结果？什么原因呢？我们的理智认为我想做这件事情，我想要这个结果。但对不起，我们的潜意识是不想要的，它不支持你。

以下是判断理智与潜意识意愿的方法：

你看那些体重很重的一些朋友，他口口声声“我要控制体重，我要减下来”。你问他：“哥们你想减下来吗？”“当然想了，我为这个做了多少努力，我买了多少书、买了多少产品，每天都钻研，每天都跟这些做斗争。”但是你跟他说：“哥们，这样今天晚上咱们就别吃饭了，一起到外边溜达溜达，逛街逛四五个小时，你看怎么样？”“今天因为什么原因没怎么吃，这晚饭还是得吃。”“今天比较累，还是明天再去逛街吧。”各种理由，各种原因。

怎么判断什么是理智想要，怎么判断什么是潜意识想要？理智想要的，拿嘴说的。潜意识想要的是它的行动。嘴里想要，行动不做，就是理智上想要，潜意识不想要，理智不支持。

给大家举几个例子，我们一定要擅长去观察我们的潜意识，这样才知道我们内心深处真正想要什么，不想要什么。先说一个答案：**你内心深处潜意识真正想要的东西，是一定会实现的**。很棒吧？但问题是你搞不清楚你潜意识真正想要的是什么。

以下是分析潜意识真实意愿的案例：

比如说，我们一起去开公司想赚大钱。这个时候，我们投资了一百万，自己把所有的积蓄都拿出来了。一百万投进去，一评估，75%左右的成功率。如果现在再追加一百万能够达到85%，投不投？可能你还能咬牙，再借一百万再投。这时候再评估，如果再增加一百万，能到95%的胜算，投不投？都卖房子了，你讲：“哎呀，还是再看看吧，先这么做嘛，投入这么多了，再投入万一赔了怎么办？”可能有的人在这个时候还能投。但是这时候突然一评估，如果再投一百万能到99%的胜算，这时候绝大多数人都不投。

所以朋友们，如果是马云、王健林那些人，他最后这一百万，到99%这一百万，他但凡能找到方法把这钱搞来，他一定会投进去。他不会考虑其他乱七八糟的东西。所以那些人，他潜意识真正想要的是成功、赚大钱。而我们不是。因为你看，他的所有动作只要增加一分的成功可能性，就不遗余力地往里投。只要我还能有办法，我就投进去，眼睛只盯的是成功成功成功，其他都不考虑。这才是真正的：我理智上想要成功，我潜意识要的也是成功。

但我们呢？我们是还能投一部分，貌似想要成功，但一旦再多了，我会想办法，但是我不投了，我怕赔钱。所以我们真正内心深处和潜意识深处，真正真正想要的东西，不是成功和赚大钱，而是想要三个字，叫做“少赔钱”。朋友们，震撼。你貌似以为自己是要成功，但其实你的内心深处的潜意识，要的是少赔钱。

所以我们以这样的态度去做事业的时候，我们潜意识真正想实现的东西是一定会实现的。他一定会做到少赔钱。你肯定这辈子比马云赔的钱少得多，这件事情你做成了。你比马云还厉害，你赔的少得多，你做到了少赔钱。但是成功？那就看运气了。而人家的成功几乎是必然的。这一次全身心投入，倾家荡产了，投进去以后赔了，没关系不在意。我盯的不是赔钱，赔钱怎样？赔100万？等我成功一次，前面所有的损失全回来，不在乎。好，这时候我没钱了，回去打工，去摆地摊，就这样下去。你想想，人家每次是99%的成功率，你的成功率只是75%或85%。你知道吗？真正做成一件事情，就差最后那一点点力量。所以他多增加了10%的成功率的话，人可能三五回就成功了。你可能一辈子努力过100回，创业过100回还是成功不了。我们都知道，最后的成功，就看那个最后一下的坚持，你少这一步都完蛋。人家就是要成功，他只要有精力，一下一下到最后一下，最后的那一下人家那劲都使上。当然如果天时地利人和不允许，这次可能会失败，但没关系下次还这样。所以他每一次都是努力到最后一步。“谋事在人”这点已经完全做到了，“成事在天”那点只要运气别太差，几回他就成功。这个才是真正的潜意识要的是成功。

朋友们，这个案例希望大家警醒，去知道、检查自己身上的行为。我们的行为会暴露我们到底潜意识真的想要什么。

再举一个例子给大家。比如说我们恋爱。年轻的时候，一个女孩子，十来岁、二十多岁上大学了，咱们恋爱了对吧？那这时候我们怎么恋爱？全身心投入和付出，我不考虑那么多。我喜欢她我就有五块钱，然后我再借五块钱，我给她买一碗酸辣粉我自己饿一顿，就这样投入。但最后呢？有一天伤害了。别人会劝你：“哎呀你看你多傻呀，你做这么多得到什么回报？”于是下一次，不会这么投入。上次一百，现在变八十，然后又遇到问题变六十、四十、二十、零。到了三十多岁以后，会什么样的状态？哎，认识了一个人，感觉还不错。但是对不起，我是这样一种态度：自我保护的态度。我一点都不会投入，也不付出。我看你投入一步，我不会付出。你再投入一步，不付出。三五步以后我能付出一点点。

用刚才咱们的方法分析一下吧。这个人潜意识深处想要的是什么？你问他：“你想要幸福吗？”“想要，要不然我干嘛那么多次去相亲呢？”但是对不起，你的潜意识是想要的不是幸福，想要的是四个字，叫“不受伤害”。你所有的行为全是在追求“不受伤害”。而开始那个小女孩，她想要的就是幸福幸福幸福。我全身心过去了，没成功，没关系，我下次还这样还这样。最后遇到一个真正好的人，我们就在一起了。虽然伤害了一百次，一百零一次我成了。这个每次都不受伤害，但是基本上你的幸福只能渐行渐远了。理智上想要的是幸福，潜意识想要的是不受伤害。多可怕啊。

所以我们这辈子，学到这一刻，我们讲潜意识，你一定要知道我们理智上想要的是什么，我们知道，问你你一回答马上知道你理智想要的是什么。但是你潜意识想要的是什么？你去看看你怎么做，就分析出来。一定要分析出来我们潜意识要的到底是什么。

明白了这个道理以后，你就会知道了：当你去创业、当你去做事情的时候，你的理智“我想要这个东西”，我也会去做的。你就是一个人拉着这辆车，艰难地往前走着的时候，只是对不起，潜意识那一万人的军队，全在往后拉这个车。你累死这车也不会往前走，也很难成功。明白为什么我们之前做事情是那个样子了吧？

但是如果我们改写潜意识成功了，你想想有多么的棒。我一个人拉着车往前走，后边一万个人在帮我推。你想想那个多棒啊。我们每一个学员经过这个训练之后，最后都是潜意识支持我们做事情。我只要想做这个事情，潜意识就拼命的支持。你想想咱们所有的心态科学，我让你扒笔记之后，你现在停得住吗？你根本停不住。你今天就算理智上犯懒，潜意识会把你的屁股挪到椅子上，还是去扒笔记。你一个人再往前走，后边一万个人在帮你推车，你不成功谁成功？

所以咱们这道课程，讲的是整个我们人生做事的底层逻辑，好好地学，一定要认真地去训练。所以我们必须去感谢我们的潜意识。有一种说法是这么说的：人类这种动物，理智部分的力量，和潜意识部分的力量相比，潜意识力量比我们理智大一百万倍。咱们就把这个一百万倍的军队调出来吧，让它帮着我们一起来做咱们的事情，就成功了。

## 如何改写潜意识：心理学链条

那么下一个问题：怎么去改写我们的潜意识？怎么获得我们潜意识对我们的支持？

刚刚我们讲了要改写我们的潜意识。但是我们的潜意识怎么去体现它是怎么想的呢？我们理智是怎么想的，潜意识不支持它是那么想的，我们要怎么体现出来？怎么观察出来我们的行为、身上怎么做事情？

其实我们每个人是有这种心理概念的。我会发现，由于我身上有些毛病，导致我拿不到好结果。我们想改这个毛病。几十年以来，一次又一次，我们遇到了一些不好的结果，我们想改毛病。我们向别人道歉说“对不起对不起我错了，我下次一定改，我一定痛改前非”。就这种认错，从幼儿园到小学，一路到今天，跟我们的爱人认错、跟领导认错，认了一辈子了。改了吗？很难改吧？改不了吧？为什么？

我告诉你个答案。你了解潜意识、了解心理学的底层逻辑之后，你就会明白：直接改毛病，那是改不了的，那是逆天的行为。为什么呢？因为啊，比如说这棵树，树根出问题烂了，它在树干上会出其他的问题，树叶上出其他的问题。你要改这个树叶那改不了。那根上有问题咱们找到根，才能改。

比如说我这个人，内分泌失调了，然后导致我脸上起很多包，眼睛也发红，嘴里面也口臭发炎，胃还疼，身上起很多包，脚癣等等，一堆毛病都出来了对吗？但是我改脚上的毛病能改吗？你稍微改一点过两天又犯。其他所有的东西咱们都改不了。这时候很简单，你找到病根，内分泌失调了调一调，调好了这些毛病全没。这个也是我们心态课一个很神奇的现象：不用改毛病，你练到一定程度，你发现身上的毛病自动没了，很神奇。突然发现这个毛病怎么没了？这个毛病没了，我以前怎么会这么做呢？好奇怪，现在再也不会这么做了，自动改。这个牛不牛？

但是为什么？这里面有一个心理学的底层的知识，我们大家需要知道的。你看啊，我们身处的环境每个人是不一样的。比如说我的环境孤单寂寞冷，你的环境很温暖很多朋友。你的世界是温暖的，我的世界冷的。咱们世界环境不一样。这个环境怎么导致的？是因为我们每个人所作所为导致的。我做事特别自私，这种人讨厌，没人理我，我的环境肯定是孤单寂寞冷。你特别好，对大家都付出，暖暖的，于是大家都喜欢，你的环境就是温暖。所以我们的环境是由于我们的所作所为（行为）导致的。

而我们的行为呢？你知道了，要改变行为，改变了我环境就改变了。所以我们要改毛病吗？但刚才咱们说了，行为改不了。那怎么办呢？往根上再找吧。什么东西导致了行为呢？我身体里什么东西导致我这样去做事儿？而你的心里有什么不一样的地方导致你那样做事？咱们找这个病根儿，咱们改了不就完了吗？

价值观、三观、信条、底线、我们的规条信念系统等等。这些东西很复杂，一大堆心理学的词汇不用管，咱们就简单理解为“三观”就可以了。那就改三观呗。三观只要一改，我们这下面的行为必然改了。但是三观好改吗？不好改。很多人发现，我以前性格这个样子，后来去了大学认识很多新朋友，经过四年的熏陶，我很多毛病改了。发生过，那就是因为我的三观改了，被别人影响，三观改了。三观改起来很难。你想想如果我们改三观发财了，那很多夫妻两个人为什么会分开？不就因为三观不合吗？那你马上去民政局门口开个公司，“我能改三观”，那对中国的家庭幸福造成多好的影响啊。所以三观不好改。

那就再往上这一步吧。再往上，什么导致了我们每个人的三观不一样？也就是我们每个人心里潜意识最深处，刻的一个东西叫“身份认证”。里面写了很多话，内容是什么呢？“我是谁”，“我是一个什么样的人”。这个“我是谁”（身份认证）不是我们社会上的、理智层面的身份认证。理智层面社会上面是什么呢？我的名字叫景旭峰，身份证号码是123456789，电话号码多少，家住哪里，我的职业什么，我今年多大等等。这些东西是物理意义上的、我们理智层面的这个社会标签、社会上的身份。

但是潜意识的身份是什么内容呢？类似于：“我是一个有魅力的人”，“我是一个没有魅力的人”，“我是个放得开的人”，“我是个放不开的人”，“我是个勇敢的人”，“我是个不勇敢的人”等等这些东西。
