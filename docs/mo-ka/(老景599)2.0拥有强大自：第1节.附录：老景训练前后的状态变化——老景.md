# 🧠 老景599 2.0 拥有强大自信理论篇：第1节.附录：老景训练前后的状态变化——老景2017年视频

在本附录中，我们将通过观看一段2017年的视频，直观地对比我在心态训练前后的状态差异。这段视频是我亲自剪辑的，它记录了我训练前的真实样貌。我们将一起分析视频中的行为模式，并理解“自我觉察”与“内核改变”为何比单纯学习技巧更为根本。

## 📹 视频背景与自我觉察的缺失

这段视频拍摄于2017年5月，当时我在一家公司讲授写作技巧。从2018年8月完成心态训练后，我再未看过此视频。重新审视时，我深感当时自己**完全缺乏自我觉察能力**。

许多人在对比好坏案例时看不出区别，这正源于缺乏敏感的自我觉察。若无法感知对比，自然也难以察觉自身缺乏魅力的行为。视频中的我，当时自我感觉良好，却困惑于“为何拿不到结果”。其根本原因在于状态差、行为“小”，无法激发他人的“魅力开关”。

因此，心态训练的第一要务，就是培养**自我觉察能力**，即自省。

## 🔍 视频内容分析：理论与实践的割裂

拍摄此视频时，我早已掌握诸多写作与沟通的理论与技巧，例如“笔打小原则”、“弄不动倒推法”。但理论与行动之间存在巨大鸿沟。

**核心问题公式：**
`我知道 ≠ 我能做到`

以下，我们将心态课的核心概念作为标尺，逐一审视视频中的表现。

### 语速与音调分析

视频中，我的语速常常不由自主地变得**非常快**，音调会突然变得**很尖、上扬**，并且夹杂不少脏话和过多的肢体语言。

**代码式描述问题状态：**
```python
# 训练前的不稳定状态
speech_speed = “非常快”
voice_pitch = “尖锐、上扬”
body_language = “过多、杂乱”
self_awareness = False
```

整体给人“轻飘飘”、没有分量感的印象。用心态课的术语说，就是行为“比大小”全是**小的行为**。

### 技巧的局限性

当然，视频中偶尔也能看到我应用了一些已知的技巧，例如刻意放慢语速、压低音调、加入停顿。但这些全是**理智层面刻意控制的结果**。

一旦讲得兴奋，理智稍有松懈，所有技巧便被忘光，瞬间“打回原形”，表现如同“小丑”。这证明仅靠技巧是徒劳的，因为**潜意识并未改变**。

**核心结论：**
`技巧是表层的、易忘的伪装；潜意识是深层的、自动化的内核。`

## 💡 从“小我”到“大我”：心态训练的本质

上一节我们分析了技巧的不可靠性，本节我们来看看真正的解决方案是什么。

如果一个人内核是“小”的，却要伪装“大”的行为，这就像指挥一支万人军队，你不可能有精力时刻纠正每一个士兵的动作。唯一的出路是：**彻底改变内核，让自己从“小的人”成长为“大的人”**。

**成长路径公式：**
`小人内核 + 技巧伪装 = 易露馅、疲惫`
`大人内核 + 自然流露 = 稳定、有魅力`

训练完成后，你的举手投足会自然流露出“大”的行为。届时，让你再去刻意模仿“小”的、缺乏魅力的行为，反而会做不到了。

## 🎯 总结与行动指南

本节课我们一起观看了训练前的视频，并深入分析了“自我觉察缺失”与“技巧局限性”的问题。

**核心收获如下：**
1.  **自我觉察是起点**：缺乏它，你将无法识别自身问题。
2.  **技巧治标不治本**：它依赖于脆弱的理智控制，极易遗忘和露馅。
3.  **内核改变是王道**：心态训练的目标是重塑潜意识，让你从内而外成为一个“沉甸甸的”、有分量的人。

站在今天回顾，我通过训练已彻底改变了潜意识，再也无法做出视频中那种“轻飘飘”的表现。这证明改变是切实可行的。

**最终行动倡议：**
请忘掉所有取巧的技巧，踏实地从心态训练开始。投入约一年时间，完成从“小我”到“大我”的蜕变。我能做到，许多学员也能做到，你也一定可以。

让我们就此开启这段心态训练的旅程，迎接一个全新的、更有魅力的自己。
