# 魔卡私教《吸引力原理》：吸引力原理小视频部分：10：1_note

在本节课中，我们将通过一段历史叙事，来探讨早期经历如何深刻影响一个人的内心安全感与行为模式，并理解建立稳定情感连接的重要性。

上一节我们探讨了外部环境对心态的影响，本节中我们来看看个人历史创伤的具体表现。

故事描述了三位拥有极高权力（成为皇帝）的个体，却都经历了相似的悲惨命运。他们的核心经历可以概括为一个**公式**：**权力巅峰 + 暴力剥夺 = 深层恐惧**。具体遭遇如下：

以下是三位个体的主要经历：
*   一人被毒杀。
*   一人被逼迫自缢。
*   一人曾三次被强行掳走，即使在登基为帝后仍遭此劫难，最终在睡梦中被毒害。

这些并非他人的故事，而是讲述者“朕”的亲身家族历史，构成了其童年背景。

这种充满暴力和不确定性的成长环境，对讲述者的心理造成了直接且持久的影响。他每晚都生活在恐惧之中，核心**代码**可以描述为：`if (state == “sleeping”) { fear(“being taken away”); }`。这种恐惧深入骨髓，导致其长期被噩梦困扰，甚至在成年并掌握最高权力后，这种不安全感依然存在。

讲述者对此感到羞耻，不愿被人知晓其脆弱的一面。这引出了从创伤到治愈的关键过渡。

基于自身的痛苦经历，讲述者产生了强烈的改变意愿。他决心为下一代创造一个完全不同的成长环境。他的核心承诺是：**绝不让自己的孩子（或重要的情感对象）在恐惧中长大**。他承诺提供充分的照顾与安全感，旨在切断恐惧的代际传递，确保下一代不再被同样的噩梦纠缠。

本节课中我们一起学习了早期创伤性经历如何塑造一个人的核心不安全感，即使在外界看来已功成名就，内心的恐惧可能依然存在。同时，我们也看到了通过有意识的关爱与承诺，建立安全的情感连接，是打破恶性循环、培养健康心理基础的关键。故事在“三王”的悬念中暂告一段落，下集将继续展开。
