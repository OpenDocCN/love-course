## 识别教学“窗口”

从以上对话中，我们需要识别出可用于“情景回放”或“即兴表演”教学的核心部分。以下是关键步骤：

首先，定位教学意图的明确信号。视频中人物提到“来我们的场景还原一下是不是这样”，这标志着表演练习的开始。

以下是“情景回放”表演的核心对话内容窗口：

```
人物A（模仿）: 干嘛，你干嘛，女士女士我特别喜欢你。我觉得你特别美丽，别别别别别，太多人了，接受我的求婚吧，嫁给我吧，这么多人，你这么害羞，坐下坐下坐下。
人物B（反应）: 尴尬了。
```

其次，定位第二次情景转换的信号。对话后期出现“然后刚才是情景回放，要不这次来一个真的”，这标志着从“回放”练习切换到“真实”情境的表演。

以下是切换到“真实”表演的核心对话内容窗口：

```
人物A: 做我女朋友吧。
人物B: 真的吗？
人物A: 真的，当然是真的了。就这么多人见证的。
人物B: 别闹了，暫停會議。
```

## 核心技巧总结

本节课中我们一起学习了从生活化、闲聊式的视频素材中提取教学“窗口”的方法。关键技巧在于：

1.  **倾听信号词**：关注如“场景还原”、“情景回放”、“我们来演示一下”等明确指示表演或教学环节开始的语句。
2.  **划定边界**：以信号词为起点，以表演内容结束或下一个话题转折为终点，精确截取片段。
3.  **忽略干扰**：果断剔除与核心教学表演无关的个人故事、调侃、玩笑等冗余信息。

通过以上步骤，我们可以将一段冗长杂乱的对话，精简为清晰、聚焦的教学示范片段，这就是“窗口识别”在实际剪辑中的应用。
