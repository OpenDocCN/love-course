# 魔卡恋爱方法2.0：浪迹魔卡《恋爱方法》：job_第12课泡学解毒－高价值展示与装逼_note

在本节课中，我们将要学习吸引力模型中的一个核心环节：高价值展示。我们将重点区分真正的高价值展示与令人反感的“装逼”行为，并提供清晰的判断标准，帮助你避免常见的社交错误。

---

在第二颗吸引力模型中，摩卡告诉大家，当我们进入任何社交场合，面对任何社交对象时，首先要进行无需求的切入，紧接着要做的事情就是进行高价值展示。

但是，很多了解PUA文化或学习过相关课程的同学容易产生一个困惑：为什么做了高价值展示，不但没有建立吸引，反而引起了别人的反感？这是一个典型的泡学中毒现象，因为他们把装逼和高价值展示混淆了起来。

相信没有人会喜欢装逼的人，包括我们自己。既然我们自己都讨厌装逼的人，为什么我们自己还要去装逼呢？这里摩卡先给大家四个区间来帮助理解。

以下是四个价值展示区间：

1.  **有价值但不张扬（低调区间）**：从长久的时间范围来讲最受欢迎。真正的高价值是不用说出来的，而是靠被人发现的。
2.  **有价值但很张扬（高调区间）**：在短时间内极易对普通的女孩子造成快速的吸引。但相对来说显得轻浮不够沉稳，对于很多见多识广、有思想、有阅历、有教养、有追求的优质女性来说不大奏效。
3.  **没有价值但实事求是不伪装（实在区间）**：虽然不一定能吸引到优质女性成为恋人，但人品踏实、做人坦诚，不妨碍女生对他们有好感并成为朋友。
4.  **没有价值但伪装自己（装逼区间）**：用虚假的东西去骗取吸引。即使在短时间建立了吸引，也经不起更深入的了解和沟通。一致性极其容易断裂，一旦一致性断裂则前功尽弃，不但自己活得累，别人也讨厌。

现在大家可以对照一下自己的实际情况，看看自己处在哪个区间，想想为什么自己很努力地做了高价值展示，却越来越让人讨厌，吸引力反倒越来越低。

---

上一节我们介绍了四个价值区间，帮助我们自我定位。如果我们还不明白，那么摩卡给大家分享一个标识，用于衡量自己是在装逼还是在做高价值展示。

**核心衡量标准**：
真正的高价值是在处理实际问题时**被动**展示的，这个高价值是被问题给“带”出来的。焦点在于**处理问题**。

以下是高价值展示的示例：

*   银行里有一个不会英语的日本人，因语言问题无法办理业务。你作为一名日语专业的学生，帮助他完成了业务，成功处理了当时别人都无法处理的问题。与你同去的女生因此流露出对你的好感窗口。这就是高价值展示。
*   你公司里有一个脾气古怪的客户，其他人都搞不定。你通过努力沟通最后签单成功，获得了老板的嘉许和同事的认可。这也是高价值展示。

但装逼则不是这样。装逼的焦点**不在于处理问题，而在于彰显自己**。他不但不处理问题，也许还要制造问题。最终的结果不是被鄙视，就是被嘲笑。

为了更直观地理解，我们来看一个反面案例视频中的对话片段。视频中的男士试图通过炫耀来建立吸引，但其焦点明显是彰显自我而非解决问题或提供价值。

> “我是专家，我从事质量管理体系工作有十多年了……国际语言专家还是很少……很多我的朋友来自美国、英国，他们都说我的英语很好……I have special talents，我能歌善舞……如果满分是100分，女孩都给我打了90分以上……”

大家看到视频中的这位大叔，很明显他不是在高价值展示，而是在装逼，最终成为了主持人、其他女嘉宾以及现场观众的笑柄。所以我们大家一定要引以为戒，千万不要在我们自己社交时重蹈这样的覆辙。

本节课中，我们一起学习了高价值展示的核心要义，明确了它与“装逼”的本质区别。关键在于：**真正的高价值是解决问题时自然流露的，焦点在事；而装逼的焦点在自我彰显，往往脱离或制造问题**。请务必在社交实践中细心体会这一区别。
