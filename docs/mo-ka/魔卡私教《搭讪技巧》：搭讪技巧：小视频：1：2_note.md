# 魔卡私教《搭讪技巧》：搭讪技巧：小视频：1：2_note

在本节课中，我们将要学习如何把握日常生活中擦身而过的机会，将潜在的缘分转化为现实中的相识。很多人错失机会，仅仅是因为缺少一次简单的主动开口。

## 核心理念：机会在于主动

很多人认为，真正适合自己的人遥不可及。但事实是，这个人可能就在你身边，每天与你擦身而过。你只是没有把握机会去认识对方。

如果你放弃了认识对方的机会，他/她就只会成为你脑海中的一个模糊印象。你偶尔会想起，并可能因此感到后悔，后悔自己当初为什么没有更主动一点。

## 核心行动：从一句问候开始

上一节我们介绍了主动的重要性，本节中我们来看看如何将理念付诸实践。改变局面的方法非常简单，关键在于迈出第一步。

一个简单的行动就能改变一切。主动跟他/她打一声招呼，或者说一句“你好吗”。

这个简单的举动具有强大的力量。从此，他/她将不再仅仅是你回忆或想象中的一部分。

以下是这个行动带来的直接改变：
*   他/她会从一个抽象的“回忆”，变成一个具体的人。
*   这个人可能会成为陪你上班、与你聊天、分享生活的真实存在。

---

本节课中我们一起学习了缘分的主动性。真正的机会往往隐藏在每日的寻常相遇中。避免未来后悔的唯一方法，就是当下勇敢地主动开口，将一次擦身而过变成一段关系的开始。
