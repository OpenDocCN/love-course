# (老景599)2.0拥有强大自信理论篇：第1节.前言一如何使用这套课程_note

在本节课中，我们将要学习如何正确使用这套心态训练课程。这套课程包含高维度的个人成长心态知识与训练方法。正确使用方法是掌握其精髓的关键。

## 课程性质与使用原则

这套课程是一套训练课程，其内容需要反复学习。你每看一遍，收获都会不同。初学时可能只有初步理解，但一年、三年甚至五年后回看，随着个人知识维度的成长，你会获得全新的体会，看到更多不一样的东西。

因此，这套课程需要观看很多遍。

## 课程使用的两个核心工作

那么，具体该如何使用这套课程呢？进入课程后，我们需要完成两项核心工作。

### 1. 观看课程与记录笔记

第一项工作是观看课程。这类似于传统学习方式。

以下是具体步骤：
*   打开视频，准备笔记本或电脑。
*   一边观看，一边记录笔记。
*   观看过程中可以暂停，记录知识点和心得感悟。
*   通常，一小时的课程可能需要两小时来观看和记录。
*   完成全部课程后，你可能会积累数百甚至数千字的笔记。

通过这个过程，你可以记录下课程的知识点，写下心得，并了解整个课程的全流程框架。观看次数最少一遍，多则数遍，根据个人情况而定。

然而，仅靠传统观看学习是不够的。这套课程与我们以往学过的所有课程都不同，传统方法无法练出效果。课程中有一个最重要、最硬性的规定。

### 2. 执行“八笔记”训练

除了观看课程，另一项必须完成的工作是“八笔记”。这是本课程训练的核心。

“八笔记”有两种方法。

**第一种方法是集中八笔记。**
*   从进入课程的第一天起，养成习惯：每天早起半小时（为此需要早睡）。
*   早起后，有一个注意事项：**不能先去刷牙洗脸**（只能上卫生间）。
*   然后，在有些迷糊的状态下回到桌前，开始八笔记。
*   操作方式：听课程中的一句话，暂停，然后将这句话**一字不差**地记录下来。如此反复，听一句，八一句。

以下是关于八笔记工具的注意事项：
*   **最佳方法**是使用两台电脑。一台电脑播放课程视频（可提前将音频或视频文件准备好），另一台电脑用于打字记录。
*   这样便于精确控制播放进度（暂停、倒退、前进），避免因用手机回退时操作不便（如不小心倒退过多）而产生焦虑。
*   如果没有条件，使用手机播放并配合笔记本手写记录也可以。

集中八笔记的做法是：在进入课程后，集中一段时间（如半个月到一个月），不眠不休地将全部课程的笔记八完，获得完整的文字版，便于日后查找学习。

完成第一遍集中八笔记后，进入**第二遍八笔记**，这才是真正的训练。每天早起必须八半小时笔记，晚上最多再加半小时。这样每天大约能完成课程中八到十分钟的内容。按此进度，完整八完一遍笔记需要较长时间。

**第二种方法是渐进八笔记。**
*   白天或晚上静心观看课程，并记录笔记，以此了解全流程，可以观看多遍。
*   “八笔记”的训练则单独进行：每天早起八半小时（晚上可加练半小时），每天只八四到十分钟的课程内容。
*   按此速度，完成一课大约需要一周，全部18课可能需要18周（约六个月）。

两种方法可任选其一。

我们需要非常清楚，“八笔记”环节的目的**不全是为了学习知识**。其中只有**30%到40%** 是为了记录和巩固知识。另外**60%到70%**，实际上是心态的内功训练。所有坚持八笔记到一定时间的朋友会发现，这件看似枯燥的工作会让你上瘾，成为每天最开心的时刻。这种感觉与你以往的所有学习经历都不同。

## 训练流程的启动

了解了“看课程”和“八笔记”两项工作后，接下来该如何开始训练呢？

1.  首先，你可以先将**第18课**的笔记八完。这一课讲解了训练的全流程。
2.  然后，结合第18课的训练方法、日常的“八笔记”训练以及课程观看，正式开始我们的心态训练。

## 使用总结

本节课中我们一起学习了本课程的使用方法，现在来总结一下完整流程：

1.  进入课程，观看前言。
2.  打开第一课，开始执行“八笔记”（可选择集中完成第一遍，或每天渐进进行）。
3.  同时，安排时间观看课程，将知识刻入大脑。
4.  提前或同步完成**第18课**的笔记八写，以掌握训练全流程。
5.  之后，便进入正式训练阶段：**训练 = 八笔记 + 结合课程知识的实践**。
