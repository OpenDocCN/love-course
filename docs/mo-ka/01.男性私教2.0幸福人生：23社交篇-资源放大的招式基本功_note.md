# 01.男性私教2.0幸福人生：23社交篇-资源放大的招式基本功_note

在本节课中，我们将要学习社交能力的核心概念、评估维度，并通过一个真实的商业社交案例，深入理解如何通过正确的思维和行动来建立、维护并放大你的人脉资源。社交能力不仅是获取亲密关系的基础，更是个人成长和获取资源的关键。

## 概述：为什么社交能力至关重要

社交关系是我们生活中最重要的网络之一。它像一个漏斗，最外层是广泛的社交关系，向内是普通关系，核心则是亲密关系。许多亲密关系正是从普通关系发展而来。

抛开感情和婚姻不谈，作为男性，若想获取资源、提升社会地位、赚钱、结交朋友或开阔眼界，这些都离不开强大的社交能力。因此，提升社交能力是每个人的必修课。

## 评估社交能力的三个维度

上一节我们介绍了社交的广泛重要性，本节中我们来看看如何具体评估自己的社交能力。不要简单地用朋友数量或性格内外向来判断。

以下是衡量社交能力的三个核心维度：

1.  **广度**：你认识的人的数量。这是社交网络覆盖面的基础指标。
2.  **深度/质量**：你与他人关系的紧密程度和对方的价值。这指的是关系的质量，而不仅仅是认识。
3.  **可调用性**：当你需要帮助时，有多少人愿意且能够为你提供支持。这体现了关系的实际效用。

综合这三个维度，才能全面评估一个人的社交能力。正如智立方总裁杨石头所说，你经常交往的十个人的平均层次，大致就决定了你的层次。

## 摒弃“屌丝思维”：从索取到提供

在深入提升社交技巧前，我们必须先修正底层思维。阻碍许多人社交升级的核心是一种“屌丝思维”。

**屌丝思维 = 索取思维**，其特点是：总想占便宜、追求免费或少花钱多办事、希望别人多付出而自己少付出。

如果带着这种思维进入社交，你会发现寸步难行。因为大家都不傻，没人愿意一直被索取。真正的社交高手，其思维模式是**提供价值**。你需要尽早摒弃索取思维，才有可能脱离原有圈层。

但请注意，提供价值不等于简单地“助人为乐”或当“老好人”。这需要智慧和策略，而非一味付出。

## 核心案例：一次失败的商业合作与后续操作

下面，我们通过一个真实的案例，来具体看看如何在复杂情境中运用社交智慧。这个故事涉及我（摩卡）帮助一位艺术家朋友在深圳高端商场开店，但最终在品牌比选中失利。

**背景**：艺术家想开店，需要投资人和黄金铺位。我帮忙找到了投资人，并开始与目标商场的招商经理（一位女士）接洽。经过多轮谈判，最终商场选择了另一个品牌，我们落选了。

**关键节点**：招商经理在微信上正式通知我们项目未能中标。

此时，如果是你，你会怎么做？以下是几种常见思路：

*   放弃，另寻他处。
*   继续争取，尝试提高租金或条件。
*   送礼表示感谢，希望保持未来合作机会。

这些思路都有道理，但未必是最优解。我当时的做法是：**接受结果，但邀请这位招商经理单独吃了顿饭**。席间没有谈论项目翻盘，而是像朋友一样聊天，聊艺术、宠物、心理学等共同兴趣，让对方感到轻松愉快。

饭后，我借口想逛商场买件商品，她热心带我前往。看中商品后，我询问是否有折扣，她通过同事帮忙，最终为我争取到了**六折**的优惠，省下了大约2000元。

## 社交的核心原则：为所有价值“计价”与回馈

故事的重点从这里开始。绝大多数人在得到好处（比如省了2000元）后，容易沉浸在“占便宜”的喜悦中，顶多说声“谢谢”。但这是不够的。

本节课希望大家习得的一个核心原则是：**为你所获得的任何帮助“计价”，并给予恰当的回馈**。这不是功利，而是尊重和远见。

在我的案例中，我做了以下几件事：

1.  **对直接帮助者（招商经理）**：我以“感谢你帮我省了钱”为由，赠送了一份精心挑选的礼物（艺术家的作品）。这让她感到被重视，且回馈自然合理。
2.  **对间接帮助者（她的同事）**：我请她代为转达谢意，并说“下次有机会请他吃饭”。这为我链接下一位潜在资源埋下了种子。
3.  **对执行层（门店店长和店员）**：我向招商经理夸赞了他们的服务专业和热情，请她代为向品牌方转达好评。这使她的同事在与品牌方沟通时多了一个正面话题。

这一系列操作，将所有参与者都照顾到了，形成了一个**正向的价值回流**。每个人都在这个过程中获得了良好的感受或潜在的社交资产，而我也巩固并可能拓展了这条人脉链。

## 播种思维：关系的培养需要耐心

通过以上案例，我们可以总结出更高阶的社交思维：**播种思维**。

社交不是即时交易。你需要：
*   **播种**：主动提供价值或建立初步链接，不求即时回报。
*   **浇水施肥**：定期维护，在适当时机给予关注或帮助。
*   **等待生长**：给关系足够的时间自然发展和深化，切忌拔苗助长。

就像追求一段感情，关系有它自然的节奏。急于求成（比如想一次约会就确立关系）往往会导致失败。有耐心地播种和培育，最终结出的果实才会又大又甜。

## 总结

本节课我们一起学习了：
1.  社交能力对人生全方位的重大意义。
2.  从**广度、深度、可调用性**三个维度评估社交能力。
3.  必须摒弃**索取型的“屌丝思维”**，建立提供价值的思维。
4.  通过真实案例，学习了在社交中为所获帮助**“计价”并全方位回馈**的核心操作方法。
5.  建立了长期经营人脉的**播种思维**，明白关系需要耐心培育。

记住，**人即是资源**。你管理社交关系的能力，很大程度上决定了你能调动的资源总量，进而影响你人生的高度。从今天起，有意识地在每一次社交互动中练习这些原则。
