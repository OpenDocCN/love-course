## 什么是好感窗口？🤝

好感窗口是指女性在语言、身体、兴趣爱好、过往经历、未来规划、思想及行动上，表现出配合、靠近或愿意建立联系的一切行为。简单来说，就是一切主动或被动靠近的信号。

以下是好感窗口的具体表现示例：

*   **主动行为**：女生与我们四目对视、冲我们微笑、主动靠近、向我们寻求帮助、主动提问。
*   **兴趣表达**：女生对我们感兴趣的事物表示出浓厚兴趣、在沉默时主动开启话题、在对话中积极配合。
*   **线上互动**：微信回复速度快、回复字数多、经常使用开心的表情。
*   **行动同频**：例如，下课后我们留在教室，她也留着；我们离开，她也离开。在KTV，她去洗手间后仍坐回我们身边或附近。
*   **专注投入**：和我们在一起时不怎么玩手机、不接别人电话。交流时能量投入，膝盖、胸膛和脚尖等身体部位指向我们。
*   **试探与回避**：在不确定我们是否有女友时，主动提起相关话题进行试探（如“你女朋友一定很漂亮吧？”）。或者，在她有男友的情况下，与我们互动时从不谈论她的男友。

## 什么是被动好感窗口？🎭

基于多数女性相对内向、保守、矜持和害羞的特点，比起上述主动信号，更为常见的是被动好感窗口。

以下是其典型表现：

*   我们找女生聊天，她会回应，但从不主动找我们。
*   我们约女生，她会答应出来，但从不主动约我们。

其核心特征是 **“不主动、不拒绝”** 。许多女生不会主动靠近，但当我们靠近时，她们不拒绝、不走开，并予以配合。

## 如何训练与提升？🏋️♂️

识别窗口的能力需要持续练习。摩卡希望大家能借助日常生活中的各种社交活动，不断锻炼自己察言观色的本领，即识别好感窗口与反感窗口的能力。

目标是能够早日建立起一套属于自己的 **“窗口捕捉系统”** 。这套系统可以简单理解为：
```plaintext
观察行为 -> 归类窗口（好感/反感）-> 调整自身社交行为
```
在这个系统的基础上，我们才能进行进退有度、分寸得当的社交互动。
