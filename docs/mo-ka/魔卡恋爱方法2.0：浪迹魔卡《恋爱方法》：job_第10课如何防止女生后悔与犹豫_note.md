## 可得性与需求感：时机决定一切

上一节我们介绍了恋爱六阶段模型，在吸引阶段完成后，便进入追求阶段。本节中我们来看看这个阶段可以做哪些事。

追求阶段可以进行许多常被视为“禁忌”的行为，例如车接车送、微信频繁关心等。这些行为本身并非不能做，关键在于做的时机。如果在建立吸引之前过早进行，会给女生带来压力，导致失败。这种行为被称为 **“暴露需求感”**。

**暴露需求感**会降低自身的高价值感，从而削弱吸引力。可以将其类比为一款游戏：
```python
# 比喻：吸引力如同游戏难度
if 吸引力 == 低 and 行为 == 过早示好:
    结果 = “暴露需求感” # 降低吸引力，减分
elif 吸引力 == 高 and 行为 == 示好:
    结果 = “给予可得性” # 增加关系亲密度，加分
```

吸引是一道分水岭。有吸引之前示好，是暴露需求感，会减分；有吸引之后示好，是给予可得性，会加分。

那么，为什么建立吸引后需要给予可得性呢？我们可以用一个比喻来理解：一个装有不同金额现金的箱子，价值代表吸引力，能否立即拿走代表可得性。人们往往会在高价值但低可得性，与相对低价值但高可得性之间做出权衡。恋爱中也一样，只有当**吸引力**与**可得性**达到平衡时，关系才会真正产生。

在实际生活中，了解、吸引、追求等阶段可能存在重合，并非严格按顺序进行。但理解此模型有助于我们在实践中灵活运用。

## 筛选：让她知道“为什么喜欢你”

在追求阶段，另一个关键任务是“筛选”。筛选的目的是让女生清楚知道你为何喜欢她，是她的哪些内在特质吸引了你。

当女生问“你喜欢我什么”时，这通常是一个信号，表明她内心存在不安全感。她需要确认你的喜欢不是肤浅的，而是基于对她内在价值的认可。

以下是进行有效筛选的要点：

*   **抓住时机**：当女生表现出良好的内在价值特质时，及时给予认可。这些特质包括顾家、贤惠、善良、有爱心等。
*   **具体化**：筛选必须关联到具体的事情或行为上，避免空泛的夸奖。
*   **使用话术**：可以“想不到…”或“看不出来…”为开头，然后联系具体事例。

例如，发现女生经常喂养流浪猫，你可以说：“想不到你这么有爱心，还每天坚持喂这些小猫。” 发现女生家里整洁有品位，可以说：“看不出来你不仅人好看，还把家里收拾得这么雅致。”

通过筛选，你合理化了对其的亲近和追求，为后续关系阶段打下扎实基础。

## 课程总结 🎯

本节课中我们一起学习了防止女生后悔与犹豫的两个核心策略。

1.  **平衡吸引力与可得性**：“可得性”与“需求感”本质是同一类行为，但因出现的时机不同（吸引前或吸引后），会产生截然相反的效果。关键在于先建立吸引，再适时给予可得性。
2.  **主动进行筛选**：在追求阶段，通过具体事例，用语言明确表达对女生内在特质的欣赏。这能有效消除她的不安全感，让她明白你喜欢的理由，从而让关系进展更顺利。

正确运用可得性与筛选，能帮助你更稳健地推进恋爱关系，赢得属于你的爱情。
