在识别出窗口后，我们需要了解不采取行动的后果。如果女生已经给出绿灯信号，而你因为焦虑或犹豫没有行动，可能会产生负面影响。

*   **升级焦虑**：出现在暧昧阶段，不敢进行肢体接触或关系升级。
*   **确定关系焦虑**：出现在交往阶段，不敢承诺或推进关系。

需要特别注意的是，在感情中遵守“交通规则”至关重要：

*   **代码规则**：

        升级关系()  # 结果：招致反感，关系倒退

**核心概念**：`违规 = (无窗口时强行升级) 或 (有窗口时拒不行动)`。两者都会破坏吸引，导致不良结果。
