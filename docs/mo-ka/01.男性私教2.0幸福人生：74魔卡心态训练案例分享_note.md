# 课程01：男性私教2.0幸福人生 🧠 74魔卡心态训练案例分享

在本节课中，我们将通过一个具体的案例，深入探讨心态训练的核心原理与实践方法。我们将看到，即使经验丰富的人，在面对陌生高价值场景时，同样会面临心态挑战。关键在于建立自我觉察系统，识别并调整内心的细微变化。

上一节我们介绍了心态训练的重要性，本节中我们来看看一个发生在宾利4S店的真实案例，它生动地展示了心态波动如何影响我们的行为。

在我心中，我一直都想念你。这句话重复出现，象征着一种执念或固定的思维模式。其实，我觉得这个案例非常有价值。大家看或听魔卡讲约会聊天，可能觉得大神很厉害，和自己不是一个世界，不知何时才能变成那样。但是你会发现，在专门、专业的心态训练这条路上，其实我和大家站在同样的起跑线。

原来我和女生接触时，不管聊天、约会、搭讪、升级关系还是长期关系，确实会因为经验丰富、想得比较多，自然带出一些心态的东西。但今年魔卡请来了老井老师。虽然我原来在1.0或其他课程中都会穿插带出心态的内容，但毕竟不是一个系统的体系。

今年我们把心态训练变成了一套专门的视频教材，它是**系统化**的。系统化输出、系统化学习、系统化消化，这与原来穿插在课程中，大家去捡碎片理解思考，其价值和效果是不同的。

在专门、系统地去训练心态这件事上，我和大家站在同一起跑线。我以前可能有一些自己的感悟和经验，借助其他事情。相信大家也一样，因为你们都有各自擅长的地方。在那些事情上你之所以擅长并取得好结果，说明你做对了事情。只不过，你没有把那套做对的东西**意识化、逻辑化、条理化**，让自己知道是怎么做对的。

当你意识化、系统化、逻辑化之后，就可以把这些普世原理放到其他方面和领域，比如和女生打交道这件事上。你现在可能不得法，但你有做得好的地方，那些地方的方法和感悟，当你系统化、逻辑化、条理化之后，是可以被借鉴到别的地方的。所以大家不要着急。

今天我要分享一个我的心态故事或案例。阿勒克斯猜我会讲“路怒症”这件事，很抱歉你猜错了。但有关联，我要说的是一个跟车有关的事情，但不是在路上。我要说的事情，跟车有关。

卖个关子，大家猜一下。阿勒克斯说了跟路怒症有关，那我们跟上课堂节奏，大家都猜一下今天魔卡分享的心态案例。既然又跟车有关，大家想一下是什么场景？在什么场景上魔卡会去训练或觉察自己的心态，觉得自己是强是弱？大家觉得会是什么事情？

民医说：“你是买宾利还是买小鹏P7这件事？”Mist上说：“在车展上搭讪车模。”我觉得民医说到的已经非常接近了，就是“我是买宾利还是买小鹏P7”这件事。小鹏P7我已经买了，现在肯定不会再去思考到底是买宾利还是买小鹏P7了，因为小鹏P7已经买了一年，跑了三万公里。

老大说：“是在买车和4S店砍价。”我觉得，只要你们精力集中聚焦，其实是可以把事情做好的。你们都猜得八九不离十，而且是在猜另一个人还没讲的事情。你们通过能量或自己的猜测，根据碎片化信息，可以猜个八九不离十。那么为什么你自己在当下场景的事情上，就不能做到八九十分甚至一百分呢？大家是可以做到的，只是你的能量跑偏了，关注到外面去了。

当你的能量集中时，你根据信息去分析，就像我教大家的窗口识别一样，根据女生给出的回复和信息，判断并得出她心里的真实意图和想法。老大说到的“买车”，民医说到的“买宾利”，阿勒克斯说到的“路怒症”，这三个东西加起来，我们今天这个场景就出来了。

其实，魔卡前几天有空，正好在附近，所以去了宾利的4S店。这件事对我来说是一个全新的体验，我相信直播间的很多同学应该百分之百，如果去过可以打个1让我知道。我相信所有听直播的同学和魔卡一样，应该都没有去过宾利或劳斯莱斯的4S店。

我们有些同学可能去过Prada、Dior、路易威登、Gucci在商场里的奢侈品专柜，可能有进去或购买、逛一逛的体验。原来魔卡在好多年以前，包括现在其实还有，因为没有进行针对性训练，我到现在都还有一些感觉。我可以清晰记得在好多年以前，可能12、13、14、15年都有，15年和17年这两年相对高频和集中一些，因为这两年我逛奢侈品店会多一些。

那时手上有一些现金，人一有钱就想提高生活品质，就会想去使用或拥有一下奢侈品，或者五星级酒店、米其林餐厅。这些东西凭什么别人配拥有而我不配？我也要尝试一下。所以在15年、17年的时候，这两年我去的算比较多了，一整年下来去过五六次。平时或其他时候，我也没有刻意不进去或刻意进去，就是觉得这种店跟我没太大关系，因为就算进去也不会买，觉得没必要。

一条皮带四五千很正常，一个围巾四五千也很正常，一双鞋子便宜的四五千也很正常。其他衣服、鞋子、包包，价格往上就没有上限了，两万左右可能都算基本款。围巾、皮带、领带、钱包会相对便宜，钥匙扣这些，但没有几百块钱的东西，都是千元起跳。随便进去买一个最便宜的东西，可能都要一两千。贵一点的，十几万、几十万都没谱，还得预订，不能当时就买，订完还有到货时间。马上能拿走的往往不是很厉害的款式，规则就是这样。

那几年是这样，跟今天要讲的东西有什么关系呢？我觉得它其实可以是一条训练途径。当然，大家不用有事没事往人家的奢侈品店或4S店跑，今天去一趟明天去一趟。但我觉得大家应该有这种体验，以及为什么要有这种体验，我在今天这节课和后面跟大家说。

先说一下整个过程。魔卡去了一个宾利4S店，我是开着小鹏P7去的，停在马路边。我有点尿急，但也没那么急。其实我在车里下车的时候就想好了，我是有想买的打算，但现在还没有那么多钱。我觉得它是一个心愿或念头，可以有，有了之后买是你的事情，有了钱再说。但不妨碍你去看一看，再加上我已经开始和大家一样做心态训练了，所谓“扛得住”。

那么在女生面前发生任何事情，对我来说都没有什么扛不扛的问题，或者说用老井在心态课中说到的东西，就是“我都能扛住”。只要对面是一个女人，99%的情况下我都能扛住，或者干脆就没有“扛住”这件事，我觉得很正常，没有任何东西需要我来扛。但这是面对女生。

面对一些比较成功的男生、老板，或者说一些需要去沟通接洽的资源，如果价值相差太悬殊，其实魔卡还是会有一些东西需要我去扛。但我现在首先有这个意识了，原来连这个意识都没有。

到了宾利4S店，推门进去，有一个男生坐在办公桌那边。总共店里有三个人，我是后来才知道的：有一个人在办公室里面，有一个人在大厅的办公桌（或者说洽谈的小圆桌）那边，他有个笔记本电脑放在那敲电脑；另外还有一个类似店长之类的女生，当时不在店里，在洗手间。

我进去以后第一句话是：“你们这洗手间在哪？借我用一下。”其实后来我整个想下来，我这个也有点故意，就是好像要让自己觉得不是那种很怂很弱，或者说很不好。我就让自己显得好像很放松很随意，在你们这个地方借个洗手间，就跟去其他普通饭店借个洗手间一样，没有任何压力。但我后来察觉，其实我心里面还是有一点点、一丢丢感觉不是那么舒服的，还是会感觉到这种价值形成的无形的压力。

大家没事下次可以找个机会去试一下，去劳斯莱斯或宾利的4S店试一下。我当时进去就跟他说：“你们这洗手间在哪？借我用一下。”然后那个男生就笑了，他说：“我们这没有洗手间。”我就问：“那洗手间在哪？”他说：“这边出门之后往那边走，左拐就是洗手间。”

这个男生的回答以及他的态度有没有问题呢？没有任何问题。我后来复盘整个过程，这个男生是礼貌的、友善的，他的笑也不是那种蔑视或嘲笑，他就是觉得这个事情可能很可笑，或者觉得不太正常，可能没有人跑到宾利的4S店来借洗手间。当时我穿得也很随意普通。我就说“OK，谢谢”，我就走了。

我走的时候，正好他们那个女店长（应该是一个上级）推门进来，她应该是去完洗手间进来，所以我跟她打个照面就走过去了。我当时下车的时候想的就是：我先得去尿个尿让自己舒服了，完了以后，我再去问一些关于宾利、关于天悦、欧陆或者飞驰（有一款叫飞驰）的价格、配置以及购买流程什么的。

然后我就去尿了，尿完了之后回来。回来之后，一男一女他们就坐在圆桌洽谈座那里。我进来以后，又说了一句话。这句话和刚刚进门跟男生说“你们这有没有洗手间借我用一下”其实性质上是一样的，但就是更加尴尬了。

为什么尴尬呢？我待会告诉大家。大家猜一下魔卡说了一句什么话？我这都属于自曝自己的短处，但我觉得没关系，就跟大家在心态这条路上，我们是同一起跑线的，我们都是老井的同学学员。所以你会看到在这个地方，其实大家跟魔卡没有差别。我又说了一句很傻的话，大家猜一下魔卡说了句什么话？

阿勒克斯说：“男厕怎么走？”我已经上完厕所回来了，我还问他男厕怎么走干嘛？就是我上完厕所，然后迫不及待要回来开始我的表演了。我进门推开门说了一句话，第一句话，大家猜一下魔卡说了啥？

起风了说：“自助餐在哪？”阿勒克斯说：“男厕怎么走？”其他同学呢？老大说：“问车展厅在哪里？”我进的就是他们的车展厅，里面停了两台车，一台天悦的SUV，一台欧陆GT。我进去的就是展厅，他们的4S店里面就停了两台车。

斗斗说：“你们这是卖宾利的吗？”没有傻到这种程度。明明说：“你认识魔卡吗？”这个就更傻了。我觉得就算我在心态课上跟大家的起跑线一样，我也不会说出这样的话。“你们认识魔卡吗？”这个也太傻了，不会的。

好，我告诉大家我说了一句什么话，非常可笑的一句话。Marshall说：“最新款的是？”这个还好，其实有很多装逼的人（也不一定是装逼），有很多顾客他会这么说。但这句话当中，其实可以听到一些些、一丢丢、或多或少一丢丢这种装逼的成分。“你们这最新款、最贵的是哪款呢？”配合上眼神神态，会有一点爆发户装逼的感觉，它不够平，能量还是扬起来的。

OK，我说了一句什么呢？对啊，直一说：“有什么优惠吗？”明一说：“反正是一句有点装逼的话。”我当时的心理活动，我先告诉大家，就是我想表现出或呈现出：我不紧张，在这个地方，我这个人的价值和你们这个店以及你们产品的价值是匹配的。我想表现出来的是这个东西，就是我们是匹配的。

但实际上，当我经历过了之后，我会发现其实是不匹配的。你装不出来，或者说你要好好地、老老实实地去真的刻苦训练之后，才可能匹配上。你仅仅知道“我得扛住，我要练习”，但这个功夫没上身的时候，你只是拿着一本拳谱，拿着一本《九阳神功》这种拳谱，你还没有开始练，还没有练到位，还没有成为高手，那么你光有这本拳谱是远远不够的。

当时我进去的时候，说的第一句话是：“你们这里洗手间在哪？我要用一下洗手间。”这是第一句。后来我去完洗手间再回来的时候，说的第一句话跟这个话其实差不多，心里的起心动念和动力其实就是要表现出我很随意、很放松，一点都没有被你们的高价值压住，或者说觉得自己不配。

只不过魔卡今天给出的场景是在一个宾利4S店，而大家碰到的场景往往是面对一个很漂亮的高价值女生，不管是在搭讪、聊天还是约会。场景可能有区别，但里面的东西是一样的。就当你面对高价值的人与事的时候，你会发现，一旦自我觉察力建立起来，你会发现内心非常细微的变化。而那些非常细微的变化，会让你的行为走样，会让你的行为变low。

OK，所以呢，我说了一句什么话呢？我说：“你们这现在有什么车可以试驾一下吗？”大家觉得有没有问题？这是我说的，我第二次推开门、上完厕所之后，推开门之后，我进门说的第一句话。

推开门之后走进去，因为前面你可能觉得他认为我是一个来借厕所的。但实际上我是一个可能的种子客户，可能会买的。然后前面的你以为我是个来上厕所、借厕所的，结果我上完厕所迫不及待地回来，好像现在真实身份露出来了，我其实是你们的一个买家，你们对我的态度得好一点。

所以我进门的第一句话就说：“你们这现在有哪一台车是可以试驾的吗？”然后又是那个男生，旁边那个女生一直在回消息、在按手机。又是那个男生抬起头看我，又笑了一下，他说：“这个我们这里没有试驾的，不好意思，没有试驾的。”

我当时才意识到，这种高端豪华车和我们普通买的代步车，它的整个销售体系或者说销售系统都是不一样的。你会发现我们买几万、十几万甚至几十万的车，就算是BBA（奔驰、宝马、奥迪），试驾都很正常。但是你会发现一些高端车型，或者说到了宾利、劳斯莱斯、迈凯伦、法拉利这个等级的车，就可能没有试驾。

当然我不知道，我没有去过法拉利、布加迪、迈凯伦、劳斯莱斯的店，我去的第一家店就是现在跟大家分享的这个宾利4S店。那个男生又笑了一下，你说有没有不礼貌呢？其实后来我自己在回想复盘的时候，人家这个笑容没有太多不礼貌的成分，他只是觉得按照他平时工作和客户接待的场景，他觉得有点奇怪或有点好笑而已。

我只是说了一句话，或者说我说了两句话。第一句是“你们这里洗手间在哪？我借一下厕所”。第二句，我想一个王者归来的感觉。第一句你可能觉得我是一个借厕所的，借厕所的路人甲，跑到宾利的4S店来借厕所的路人甲，你不太搭理我，我觉得情有可原，很正常，大家时间都很宝贵。

但是我第二句上完厕所回来以后，我本来是打算或者感觉自己有一种王者归来的感觉，“我要试驾”，那其实我就是有可能是你的客户，对不对？然后我应该是这个位置起来了，这个价值起来了。没想到第二句我更low了、更傻了。

OK，所以我说完那句话之后，我才发现一个我原来不知道的东西：就是高端豪车它的整个销售、接洽或者所有东西，跟我们十几万、几十万的这种4S店的销售，它有可能完全不一样。这是我去了之后才知道的事情。

通过这个，我想告诉大家一个什么事情呢？就是我们原来有很多的男生，想比如说我攒一笔钱，比如一千块或两千块，带一个我很喜欢的女孩去吃一顿好的。比如在一个米其林餐厅，米其林一星、黑珍珠一钻，两个人吃一顿像样的饭，差不多一千块左右（如果不叫酒的话），有些地方有15%的服务费。有前菜、素菜、荤菜、汤、饮料，差不多齐全下来一千块左右，便宜点可能六七百，贵点可能一千一二，一千块上下。

那么，我碰到过，我们也有很多男生做过这样的事情：因为我要追求这个女生，她很漂亮，我感觉她价值很高，那么我要带她去高档次的好的地方，来彰显或来表达（其实就是我也不差），我带你吃好的，来提升我在你心里的好感。

但你发现，如果这个东西本来不在你生活当中，你是第一次或第二次接触，但你又要表现出“我很随意，我经常来，这个餐厅如何”，你是基本上装不住的，根本装不住。就一句话两句话，或者很多细节会让你露馅。除非说你真的经常去，可能每周去一两次，或者每个月去个五六七八次，它真的就是属于你生活当中的一个东西，你的生活品质或消费层次真的到了这里的时候，你花这个钱它才会有用。

对于我们很多男生，你想要通过花一笔大钱，带这个女生去吃一顿好的，以此来博得她对我的好感，或者说我要包装一个富二代的人设。那么为什么原来魔卡一直跟大家说不要去拍虚假展示面呢？因为拍虚假展示面的坏处实在是太多了。但我们今天在这个地方，就不展开讲虚假展示面的问题，我们就聚焦心态这个问题。
