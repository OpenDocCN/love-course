## 故事一：项目合作中的定位失误

上一节我们介绍了社交思维的重要性，本节中我们来看看一个具体的案例。

有一个人，他的朋友是一位实力更强的老板。这位大老板将一个项目介绍给他做。项目开始时，甲方支付了首付款。他收到款项后，给作为介绍人的大老板包了一个“茶水费”红包。项目完成后，他赚到了钱，并再次专程去感谢这位大老板，希望以后还能有合作机会。然而，此后很长时间，他再也没有从这位大老板那里获得任何新项目。

大家认为，在这个过程中，他哪里出了问题？

以下是同学们的一些看法：
*   Charles认为：利润也应该分一些。
*   小可爱认为：主要是利益关系。
*   Sol认为：给少了。
*   马月认为：前期要散财。

其实，这道题考察的是一个人的“江湖成熟度”。关键在于区分两种情况：

**第一种情况：资源和项目都是你自己的。**
你在执行过程中遇到某个环节卡住了（例如，某个审批不通过），需要借助他人的关系来疏通。此时，事后给予感谢、红包或茶水费是人之常情。

**第二种情况：资源和项目都是别人给你的。**
就像今天故事中的情况，项目是大老板的，他只是执行者。此时，他的思维误区在于，认为“事情是我全程做的，所以利润大头应该归我”。然而在社会上，**掌握核心资源（如地皮、关键人脉）往往比具体执行更为稀缺和重要**。他应该认识到，自己只是一个“执行单元”，正确的做法是主动将利润进行**分红**，甚至将大头给到资源提供者，这样才能被视为“懂事”、“可靠”，从而获得持续的合作机会。

许多人在取得一定成绩后容易“飘”，错误地将平台能量或他人协作的成果完全归功于自己。这种定位不清，会导致在社交中错失良机。

## 故事二：一瓶茅台酒的启示

理解了定位的重要性后，我们通过另一个故事，来看看如何将这种思维转化为实际行动。

我曾去拜访一位已成为行业大佬的高中同学，向他请教学习。他花了两个小时与我交流，核心意思是做人要“懂得感恩”（我理解为要机灵、懂事、情商高）。临别时，他要去参加一个饭局，并到一家熟悉的烟酒店取一瓶五斤装的飞天茅台。结账时，他对店主说“没带现金，回头再结”，然后便离开了。

我走在路上，反复琢磨他“没带现金”这句话。我意识到这可能是一种暗示。于是，我返回店里，询问价格（约一万七八千元），并用自己当时近一半的积蓄买下了那瓶酒，替他付了款。

这件事当时没有立即产生回响。但大约十天或半个月后，这位同学主动联系我，并介绍了一单价值近四十万的生意给我。

### 核心规律与总结

本节课我们一起学习了两个关于社交思维的故事，我们可以从中总结出一些核心规律：

1.  **位置决定行为**：在任何社交场合，首先要明确自己的位置和分量。是资源拥有者还是执行者？是主角还是配角？行为必须与位置匹配。
2.  **价值交换是本质**：高层次的人际交往基于价值交换。当你想要从他人那里获取资源或机会时，首先要思考自己能提供什么价值。
3.  **“吃亏”是福，但需智慧**：真正混得好的人，通常不愿亏欠他人。你的真诚付出和“懂事”，很可能会以意想不到的方式获得超额回报。这并非鼓励盲目当“冤大头”，而是需要结合情商的综合判断。
4.  **知道与做到之间存在鸿沟**：理解这些道理只是第一步。在真实复杂的社交场景中保持觉察，并做出恰当反应，需要持续的练习和反思。

社交能力的提升，是一条需要不断实践和思考的道路。它不仅能帮你获取更多事业资源，也能让你在包括与异性交往在内的各种人际关系中，更成熟、更具吸引力。
