# 心态训练课程：第16课 - 行动主义与热爱主义 🧠

在本节课中，我们将要学习如何通过调整心态来克服阻碍成功的核心障碍。我们将深入探讨“区别”与“差别”的智慧，并对比“行动主义”与“热爱主义”两种做事模式，最终掌握一种能让我们保持最佳状态、发挥全部潜能的心法。

## 核心问题：我们最大的敌人是什么？

经过前面课程的分析，我们现在非常清楚，阻止我们成功（这里的成功是广义的，包括爱情、人际关系、事业等）的最大敌人，是我们大脑里胡思乱想带来的各种情绪。

这些情绪包括焦虑、恐惧、自我怀疑等。它们导致我们在面对喜欢的异性时会紧张、表现失常；在与重要的人相处时姿态过低、讨好对方；在做事业时无法火力全开。

还记得小孩子学走路的案例。小孩脑子里没有任何杂念，于是做事情效率就会很高。杂念是我们最大的敌人，它导致我们根本无法深入一件事情，也就无法产生那种全神贯注、一片空明的“热爱”感觉。

这一切的根源，都是“小我人格”带来的负面影响。因此，从前面的课程我们已经知道答案：我们整个心态训练的最终目标，是要逐渐恢复“真实自我”的人格，并让“小我人格”衰退。

## 小我的“食物”：区别

要让小我人格衰退，一项重要任务是不能再“喂养”我们的小我。小我作为一种精神上产生的人格，需要精神食粮来滋养。你不断滋养它，它就越来越壮大，我们的状态就越来越差。

小我的食物有三样：**时间**、**区别**和**选择**。上一课我们已经讲过关于“时间”的误用。本节课，我们重点来讲“区别”。

“区别”这个概念有点难理解。它指的是我们因为世间万物的“差别”而产生的情绪波动。

例如：你长得好看，我长得不好看，和你在一起时我就自卑、状态差。或者，在学校里我的成绩差，这让我没底气。世界上所有事情千差万别，这是客观存在的“差别”。

但我们做一个词汇定义：这种客观上的不同，叫做“差别”。而“区别”，指的是我们不要因为世界上千差万别的事情不一样，而导致情绪被波动。

以下是“差别”与“区别”的核心定义：

*   **差别**：事物之间客观存在的不同。例如，他一天挣200元，我一天挣20元。
*   **区别**：因这种客观差别而在内心产生的情绪反应（如嫉妒、自卑、焦虑），并将其标记为“好”与“坏”。

所以，什么叫“没有区别”？就是我们在面对事情时的“不反应”。之前我们知道了面对人时要“不反应”（对方说我没魅力，我不因此焦虑或辩解）。面对事情时，我们同样要做到“不反应”。

比如，我满心欢喜以为努力后能得到大西瓜，结果只得到小芝麻。这个“差别”是存在的。但如果我因此情绪波动、焦虑痛苦、状态变差，那我就无法改变自己，以后可能永远只得到芝麻。

这一点在训练中很难。我们要意识到所有事情的“差别”（这座山比那座山高，这个手机比那个好），但情绪上不引起任何波动。这就叫做“没有区别”。

### 为何“无区别”如此重要？

我们面对一个人时，如果因为对方的样貌、地位等“差别”而产生“区别”心（觉得他好/我不好），就会紧张焦虑，魅力尽失。

同样，面对一件事时，如果因为事情的成败、得失等“差别”而产生“区别”心（觉得成功好/失败坏），我们的状态就会下降。状态一下降，就无法火力全开，自然难以拿到好结果。

**核心公式：**
```
成功状态 = 专注力 / 情绪波动
```
情绪波动（区别心）越大，专注力越低，成功状态就越差。

想抹平客观差别的唯一方法是：**只看到差别，不看到区别**。没有任何区别，觉得“都挺好”，保持心情愉快，沉进去、钻进去，你才能把事情做好。

所以总结一下：**这世间万物，有差别，没有区别**。这就是我们面对事情时的“不反应”。不因高低差别导致情绪波动，不因此产生“好与坏”的评判，我们才不会被外物牵动。一旦被牵动，情绪出来，我们就无法专注。

## 行动主义 vs. 热爱主义 🚀

理解了“无区别”的智慧后，我们进入第二小节，来看两种截然不同的做事模式：行动主义与热爱主义。我们生活的世界普遍推崇“行动主义”，但这恰恰是与成功规律相悖的。

“行动主义”即“努力主义”，认为通过咬牙奋斗、拼命行动就能成功。但这非常低效，因为你下的功夫和“热爱主义”比起来，肯定下不过它。这是底层逻辑，这个世界是反的。

有一个理论叫做 **Be-Do-Have**。

*   **Be**：成为（是什么样的人）
*   **Do**：做（去行动）
*   **Have**：拥有（得到结果）

普通人（行动主义）的做事顺序是：**Do -> Have -> Be**。
以下是两种模式的代码式对比：

```python
# 行动主义 (Do-Have-Be) 模式
def actionism():
    current_state = "能力不足者"
    while not have_desired_skill():  # 先做，直到拥有
        do_hard_work()  # 痛苦地努力
        feel_anxious()  # 充满焦虑
    finally_become_expert()  # 最终成为

# 热爱主义 (Be-Do-Have) 模式
def passionism():
    identity = "未来的高手"  # 先是：认同自己终将成为
    while not have_desired_skill():
        do_with_passion()  # 带着热爱和自信去做
        stay_present()     # 保持临在，享受过程
    naturally_acquire_skill()  # 自然拥有
```

以学英语为例：
*   **行动主义 (Do-Have-Be)**：我认为我英语很差(Be差) -> 我努力学(Do) -> 希望拥有好水平(Have) -> 最终成为高手(Be好)。在这个过程中，因为认为自己“不行”，遇到机会（如帮外国人指路）也不敢上，心想“等我练好再说”，从而错失实战机会，学习过程痛苦且低效。
*   **热爱主义 (Be-Do-Have)**：我坚信我**终将**成为英语高手(Be)，这只是时间问题 -> 我带着这种自信去学习和实践(Do) -> 我自然而然地拥有了英语能力(Have)。遇到外国人问路，我会想“我虽然是正在成长的高手，但不妨一试”，从而抓住一切机会练习，越练越有劲，过程愉快，结果高效。

**Be-Do-Have是一种没有杂念的真实自我的做事方法。** 它不是盲目自大，而是基于一种深刻的自信：我相信只要我沉下心来，排除干扰，我必然能达到我天赋的天花板。用这种“我终将是高手”的心态去做事，遇到困难不焦虑，遇到机会就抓住。

### 如何识别并切换到热爱主义？

只要你感觉到做事情时需要咬牙、觉得特别困难、不爱做、感到痛苦，那就一定出问题了。问题就在于你没沉下来，没有进入“热爱”的状态。

这时该怎么办？用我们的心态训练方法调整到“当下”。

**核心练习：**
1.  **意识**：意识到自己正在“努力咬牙”。
2.  **调整**：像“八笔记”（一种逐句记录的练习）一样，把注意力拉回当前手头最简单的一步。**八笔记之所以让人沉浸，就是因为它强迫你“一句一句”来，这就是“临在”**。
3.  **心态**：告诉自己，事情只有“差别”，没有“区别”。不做“好与坏”的评判，不让它影响情绪。
4.  **行动**：就在这种调整后的、平静而专注的状态下，继续一点点做。

一定要通过一个愉快的过程，到达愉快的终点。痛苦的过程根本不可能通向任何愉快的终点。我们训练的目标，就是让你每天都在一个愉快的过程中，最终到达每一个愉快的终点。

## 小我食物的第三点：选择的误区

小我的第三个食物是“选择的误区”。这同样难以理解。我们经常在做选择时患得患失，趋利避害。

想一想，当你觉得一个选择“有利”，一个选择“有害”并为此纠结时，你已经站在弱者的地位了。你觉得自己的魅力或能力不足以应对“有害”的选项，所以必须精挑细选“正确”的路。这种“趋利避害”的心态本身就是一种恐惧。

这就像之前“一万人军队”的例子。当你自信时（Be的状态），军队默认向有魅力的方向前进。当你惶恐地思考“信息该怎么发才对”时，就相当于你试图说服军队中的一个人（你的某个行为）向前，但你没意识到的其他所有部分（你的整体状态、微表情、能量场）都在向后拉。你要说服5001个人才能让车缓慢前行，这太辛苦了！

**所以，我们不应该“趋利避害”，而应该“研究规律”。**

两者的出发点完全不同：
*   **趋利避害**：出发点是我很弱，我必须选对，不然就完了。心态是惶恐的。
*   **研究规律**：出发点是我本身是OK的，我研究一下哪个方法效率更高、更好玩。心态是平和、探索的。

制定人生目标也是如此。不能站在趋利避害、自己是弱者的角度去选择。真正的选择是：这世界上这么多行业，我选哪个都能成功，因为我这个人没问题。那么我精力有限，就选一个我当下最喜欢的、最感兴趣的去做吧。

**目标只是方向和路标，是“翻牌子”**。就像面对一桌佳肴，你不是焦虑“吃哪个才不得癌症”，而是开心地想“今天我想翻哪个菜的牌子？”无论翻哪个，你都能享受到美味。

一旦做出选择，就要忘记目标（比如忘记“挣钱”这个结果），沉入当下，好好去做。你坚信，天经地义，你一定能做成。现在没做成，只是暂时没做成。

## 总结与核心心法 🎯

本节课中，我们一起学习了心态训练中最核心、也最高深的部分。

1.  **有差别，无区别**：认清世间万物客观存在差别，但在内心不因此产生情绪波动和“好与坏”的评判。这是面对一切人、事、物时保持“不反应”的智慧，是保持最佳状态的基础。
2.  **Be-Do-Have**：摒弃痛苦咬牙的“行动主义”，拥抱以终为始的“热爱主义”。先在内心中成为“终将成功的那个人”，然后带着这份自信和愉悦去行动，结果自然会来。
3.  **研究规律，而非趋利避害**：停止以弱者心态进行患得患失的选择。以强者的姿态，抱着研究和探索的心态去优化方法。选择目标就像翻牌子，选哪个都能成，选了之后就沉浸于过程。

**最终的心法，依然是那句最强现实框架的体现：**
> **我对于自己的观点（我终将成功）持有毫不动摇的坚信，没有一丝一毫怀疑；对于过程中的不同意见（暂时的失败、挫折）则报以最小的情绪反应（云淡风轻）。**

将这套心法用于人与事，你必然能解除封印，保持爆棚状态，走向你想要的成功。记住，世界是反的，真正的强大始于内心的平静与确信。
