# 《渣男研究院》课程12_3：拒绝被欺负：男朋友老发脾气 😠

在本节课中，我们将要学习如何分析和应对男朋友频繁发脾气的行为。我们将探讨这种行为背后的心理原因、不同类型女生的应对方式，以及评估这段关系是否值得继续。

## 概述：发脾气行为的根源

男朋友经常因为小事就生气发火。有时，他认为你已经属于他，因此觉得可以驾驭、指使、使唤和批评你。这种行为源于他以自我为中心的生活方式。

上一节我们介绍了发脾气行为背后的控制心理，本节中我们来看看这种性格的具体表现。

## 大男子主义与性格吸引

这种男生通常具有大男子主义，并且喜欢挑刺，内心比较自恋。值得注意的是，有些女生会被这种性格吸引。她们表面抗拒，但内心可能喜欢这种强势，尤其是本身气场较弱、性格柔顺的女生。

对于这类柔性女生而言，男生的**大男子主义不一定是坏事**。她们的应对方式往往是温和的。

以下是常见的温和应对模式：
*   “你不要发火了。”
*   “我知道错了。”
*   “好了你别说了，好不好？”
*   “我们别吵了，我们看个电影吧。”

这种方式有助于与这类男性和平相处。

## 冲突升级与“八字不合”

然而，如果你的脾气也比较冲，他冲你也冲，那么你们就属于“八字不合”。这种关系模式会导致天天吵架。

上一节我们讨论了性格适配的温和应对法，本节中我们来看看导致冲突的另一个关键因素。

## “宅”与情绪发泄

第二个重要的吵架原因是“宅”。在家时间太久，人容易动不动就想发泄情绪。例如，以前在国外留学时，我们经常不出门。

缺乏社交活动，不去市区，在家里就容易和同住的人吵架，说几句难听话来发泄脾气。因此，“宅”是一个重要因素。

“宅”的原因往往与**收入低**有关。经济拮据的人没赚到钱，憋在家里，更容易发脾气。

## 关系评估与建议

综上所述，你会发现，和这类状态不佳的男生在一起，可能天天受罪，生活品质不高，人生还要受苦。

本质上，我并不建议你继续与这种男生交往，尤其是如果你脾气也比较冲的话。长期处于这种压抑和冲突的环境中，对双方都无益处。

## 总结

本节课中我们一起学习了男朋友老发脾气背后的几种原因：**以自我为中心的控制欲**、**大男子主义与性格吸引**、以及因“宅”和**经济压力**导致的情绪发泄。我们分析了不同性格女生的应对策略，并最终指出，若长期陷入冲突且生活品质低下，需要慎重考虑这段关系的可持续性。核心在于识别问题本质，并做出对自己身心健康负责的选择。
