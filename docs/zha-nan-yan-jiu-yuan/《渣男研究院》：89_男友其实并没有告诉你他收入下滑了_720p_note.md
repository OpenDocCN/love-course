# 《渣男研究院》：89_男友其实并没有告诉你他收入下滑了_720p_note

在本节课中，我们将通过一个真实案例，学习如何识别并理解伴侣（尤其是男性伴侣）可能未主动言明的经济状况下滑。这种情况常导致关系中的误解与矛盾，理解其背后的逻辑有助于维护关系的稳定。

## 案例背景：收入波动与关系压力

我曾有一年创业亏损。第一年赚了500万，第二年却亏损且没有收入。当时我与一位女士交往，后来我们短暂分开后又见了一面。那时我的收入已严重下滑，甚至没有进账。

我每月实际上是在负债经营，需要倒贴成本。但她并不清楚这件事。

## 关系中的期望落差与内在压力

她依然按照我过去月入颇丰时的模式来期待我，例如让我购买礼物、为消费买单，并在朋友面前夸赞我。我只能咬牙坚持，但内心非常不舍。

每次买单时我都感到不舍。而她则会觉得：“你以前对我那么好，现在怎么变了？是不是不爱我了？” 你们可能会发现，曾经的男友每月给你零花钱，但某个月突然给少了、给一半，甚至不给了。

**核心点**：其实他内心承受着痛苦，只是没有告诉你。男人做生意常有起起伏伏。

## 经济周期的普遍性与沉默的原因

很多时候，尤其在市场行情不佳时（例如某些节日期间市场大跌），很多男性亏损严重，但他们选择不告知伴侣。疫情期间也是如此，许多人的公司未能盈利，只是你不知道。

男人有时不想说，是认为没必要让伴侣分担自己的忧愁。作为男女朋友，他想在你面前展现强大的一面，不愿让你看到自己落魄的样子。但事实是：

**公式**：`他的实际收入（大幅下滑或为0）` < `你的消费期望（基于其历史收入）`

## 压力下的应对策略与沟通尝试

比如我上个月一整月没赚到钱，而你还开口要继续礼物，还在不断提出消费要求。这时我可能会提议：“我们缓一缓，先这样保持点距离两三个月，我休整一下。”

这两三个月我生意亏损，接触你时心情也不好，感觉对你有亏欠。因为你需要我的补偿，而我目前又无法对你好，那我干脆选择不接触。毕竟我曾单身度过很长时间，甚至在国外男子学校读了三年，都能熬过来，不怕这两三个月。

所以，当我们的经济实力下滑时，我们可能会觉得：“对不起，我真的伺候不了你了。”

## 伴侣的智慧回应与错误示范

这时，如果女生聪明一点，可能会说：“这个月就不用了，我们一起度过，我陪你。” 这样关系才能维系。反之，如果你此时对我进行道德绑架，指责“你对我不好”，并因此争吵……

甚至到处找情感博主连线咨询：“Aidy，我男朋友前几个月对我好，突然不好了，是不是不爱我了？要不要分手？” 最后可能被不断洗脑，真的决定分手。

## 经济复苏与关系结局

结果却发现，**代码逻辑**：
```python
if 男人头脑聪明且有能力:
    搞钱速度可能很快
    # 上月亏损，下月可能迅速回血腾飞
```
然后你离开了。那么对不起，以后我们也没关系了，我也不会追回你。

## 缺乏耐心导致的后果

有时，你们就是没忍住、没撑住这一两个月，缺乏两三个月的忍耐力，太着急了。当你发现男人这个月对你不好，就说“那我走了”，并试图用离开来“将军”他，可能会发现这招没用。他说：“那你走吧。”

你发现这招无效，回家难受两天又想复合，但已经没机会了。感情不是你说走就走、说来就来的。你以为我这么好欺负吗？

## 男方的资源与选择权

一个年收入能达到几百甚至几千个W的男性，不会被这样对待。说实话，我们有一定的社会地位和资源对接能力，也能认识新的人。我只是怀旧，并且主要精力投入在赚钱和事业上，所以不想轻易换伴侣。

但如果你真的要换，那就换。**类比**：就像一个很牛的供应商说：“你的零部件全是我生产的，只能跟我一家合作。” 那我明天就换掉你试试。我们不想被威胁。

**核心概念**：没有任何人是不可替代的，尤其是在世界动荡、人生低谷时。

## 低谷期的考验与心态

男人会经历一种“墙倒众人推”的处境：员工走了，同事离职跳槽了，因为你发不起工资、分不了红。他们都想跑去别的地方或单干。这我们已经历过。

人生起起伏伏，你们要走就全走，OK。连女朋友你也要走？那你走吧。我们的心态是：大不了重头再来，大不了接下来半年辛苦一点，熬着熬着就又起飞了。

## 总结：短视行为与长期关系

所以说，为什么有些女生和男人只能是短期关系？因为你们自己存在短视行为。**结论**：`缺乏共情与耐心的短视行为` → `可能导致错过潜力伴侣` → `男人可能不会真正考虑与你结婚`

你一直处在不停“换大哥”的过程中，没有人会真正想娶你。
