# 《渣男研究院》课程114：为什么富二代不愿意付出 💰

在本节课中，我们将通过一个富二代的真实自述，剖析其在与不同经济条件的女性交往时，不愿轻易付出的心理动机与行为逻辑。课程将揭示其核心的价值交换观念与筛选标准。

## 个人经历与交往对象背景

上一节我们介绍了课程主题，本节中我们来看看讲述者自身的感情经历与经济背景。

我从高中到大学，以及毕业后交往的女朋友，经济条件都较为优越。高中和大学时期的女友，其家庭在上海拥有十几套房产。后来她父亲出事，房产几乎变卖殆尽，仅剩一套价值一千多万的住宅，在澳洲另有一套房产。

她家的资产从数亿元缩水至数千万元。我国内后，交往过一位年收入三百万的女友，另一位女友年收入高达五千万，且年龄比我小。还有一位女友是拥有三百万粉丝的博主。

她当时的年变现能力大约在两百万左右。在几乎快要追到这些女性的阶段，我都不需要“画饼”（即做出不切实际的承诺）。关系确定后可能会“画”一点，但在确立关系之前，绝对不“画”。

## 核心择偶观与价值评判标准

了解了讲述者的经历后，我们进一步分析其内在的择偶逻辑和价值评判体系。

我凭什么要对一个条件一般的女生付出？例如，如果你开口向我要礼物。

这肯定是因为你家庭条件不好。那么，我凭什么要送你礼物？你仅仅会拍照和修图而已。你只是身材好，但缺乏有趣的灵魂。你讲话甚至口齿不清，语言不标准。你读过硕士吗？你有留学背景吗？

你自己实现过年入百万吗？你什么都没有。你主要是想依靠男人，从他那里获取些什么。既然如此，我为什么要和你在一起？因此，我们内心本质上，对并非通过主流成长路径成功的女性，是有些抵触的。

## 付出逻辑与成本计算

基于上述的自我认知和市场定位，其付出行为遵循着一套严格的成本效益分析。

如果一开始就对你好，给你送礼物，那不就变相承认我油腻、没有魅力了吗？而且现在大家都会计算成本和回报率。其实男人都会算。
