# 《渣男研究院》课程95：男朋友不让看手机的原因是什么 📱

在本节课中，我们将探讨男朋友不让看手机背后的常见原因与心理动机。我们会分析隐私、信任与潜在隐瞒行为之间的关系，并学习如何通过一些日常细节观察对方的可信度。

---

男朋友不让看手机通常涉及个人隐私问题。没有规定女朋友必须被允许查看手机。这种行为被视为对个人隐私的侵犯。即使手机里没有特别内容，被要求查看也会被视为一种不信任的表现。对方可能会想：“你要看我的手机，但我可能确实有一点不想让你知道的事情。”

上一节我们提到了隐私与信任的冲突，本节中我们来看看隐瞒行为的心理动机。

这种隐瞒可能尚在萌芽阶段，当事人不希望被发现。他认为可以暗中进行，对方永远无法察觉。这样他既能与他人聊天，又能维持与你的关系。拒绝让你看手机，对他来说好处很大。

这类似于许多人从不承认自己做错事。承认错误意味着接受惩罚与制裁，而不承认则可能逍遥法外。例如，一个男生即使做了某些事也不会承认，更不会给你看手机，因为那等于自投罗网。
