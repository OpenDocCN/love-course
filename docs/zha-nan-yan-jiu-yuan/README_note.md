# 渣男研究院 第1课：识别基础模式 🕵️♂️

在本节课中，我们将要学习如何识别“渣男”在互动中常见的基础行为模式。理解这些模式是建立有效防御的第一步。

上一节我们介绍了课程的目标，本节中我们来看看几种核心的行为特征。

## 核心行为特征

识别“渣男”通常可以从以下几个反复出现的行为模式入手。以下是几种关键模式：

1.  **言行不一**
    其承诺与实际行动之间存在巨大差距。例如，口头表达关心，但从不付出时间或精力解决实际问题。

2.  **界限模糊**
    其会刻意模糊朋友与恋人之间的界限，或在不同异性间制造竞争感与不确定性。公式可表示为：`关系状态 = 模糊 + 不稳定`。

3.  **责任外推**
    当出现矛盾或问题时，其习惯将责任归咎于外部因素或对方，常用句式是“要不是你…，我也不会…”。

4.  **情感操控**
    其通过贬低、忽冷忽热、引发愧疚感等方式影响对方的情绪与自我认知。代码逻辑类似于：`if (对方产生自我怀疑) { 则操控成功 }`。

## 实践观察要点

了解了理论模式后，我们需要在实践中进行观察。以下是在日常互动中值得关注的几个要点：

*   **观察其如何对待他人**：特别是对待服务人员、前任以及朋友的态度。
*   **留意关系推进速度**：过快地表达深刻情感或要求确立关系可能是一个警示信号。
*   **检验问题解决方式**：面对分歧或困难时，其是积极沟通解决，还是逃避、指责或冷战。
