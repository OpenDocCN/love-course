# 《渣男研究院》课程16_3：拒绝被欺负：为什么分手后还老是说我 🛡️

在本节课中，我们将要学习如何理解并应对一种特定行为：分手后，前任仍时不时出言贬低或“敲打”你。我们将剖析这种行为背后的心理动机，并为你提供清晰的心态调整策略。

## 行为背后的心理动机

上一节我们介绍了分手后可能遇到的情感困扰，本节中我们来看看一种具体的贬低行为意味着什么。

如果你与某个男生分手三个月或六个月后，他仍时不时地说你两句，这种行为意味着此人存在某种病态心理。这类似于生活中有些人会动不动打两下狗，或把猫抓起来，其内心通常存在童年阴影。

这种行为反映出个体内心存在暴力倾向、焦虑症或多动症等问题。他通过时不时地“敲打”别人，来展现一种姿态比你高的人设。其核心目的是为了保持“他比你厉害”的优越感。无论实际情况如何，在他构建的叙事里，他永远比你强，而你在他面前永远是一个很普通、甚至低下的人。

## 这种行为的影响与本质

以下是这种行为对你产生的影响及其本质：

*   **你会持续感到被压制**：你会感觉自己永远抬不起头，永远处于被打压的状态。
*   **你会陷入自证陷阱**：你可能会不断想证明自己，例如告诉自己“我最近赚了一百万”、“我买了车买了房”。
*   **他的贬低不会停止**：即便如此，他仍然会觉得你“也就混成这样”。他的评价体系并不客观，目的并非评判，而是打压。

所以我们说，这类男生**通过打压他人的方式来证明自己的强大与厉害**。其逻辑可以概括为：**贬低他人 = 抬高自我**。

## 关键认知：问题在于他，而非你

这种态度并非针对你个人。他不论是对待朋友、父母、生意伙伴、女朋友还是前任，都会采用同样的打压态度。

因此，这与你的价值或你们过去的关系**没有本质联系**。核心问题是**这个人自身的人格与心理模式**。理解这一点至关重要。

## 心态调整策略

既然问题根源在于对方，那么你可以采取以下心态来应对：

*   **将其行为归因于其自身问题**：分手后他再来“敲打”你，你就在心里将其理解为“他有病”。
*   **此认知能有效缓解情绪**：只要你这么想，心里就会舒服很多。这能将你的注意力从他的贬低言论，转移到他自身的问题上，从而保护你的自我价值感。

## 总结

本节课中我们一起学习了如何解读分手后前任的贬低行为。我们认识到，这种行为源于对方病态的心理需求，旨在通过打压你来维持其虚假的优越感。关键在于理解**问题在于他自身，而非你的价值**。最后，我们掌握了最直接有效的心态调整方法：将他的行为视作其自身问题的外显，并由此释怀。

记住，你的价值不建立在任何人的片面评价之上。
