# 课程名称：Python基础入门 - 第1课：变量与数据类型 🦆

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。这个标签就是变量名，盒子里的东西就是变量保存的值。

在Python中，使用等号 `=` 来为变量赋值。其基本语法是一个简单的公式：

**`变量名 = 值`**

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存储在其中。

```python
message = "你好，世界！"
```

## 基本数据类型 🔤

我们已经知道如何用变量来“装东西”，接下来需要了解我们能存放哪些“类型”的东西。Python中一些最基本的数据类型包括整数、浮点数、字符串和布尔值。

以下是几种核心数据类型的定义和示例：

*   **整数**：不带小数点的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由引号包围的文本，例如 `"Python"`, `'123'`。单引号和双引号都可以。
*   **布尔值**：只有两种值，`True`（真）或 `False`（假），用于逻辑判断。

## 如何查看和使用变量？ 👀

知道了变量的创建和类型，我们来看看如何查看和使用它们。你可以直接使用 `print()` 函数来输出变量的值，也可以在表达式中使用变量进行计算。

以下是一个简单的操作示例：

```python
# 创建变量
name = "鸭哥"
age = 3
pi = 3.14159
is_learning = True

# 打印变量
print(name)
print(age)

# 在表达式中使用变量
future_age = age + 5
print("5年后，年龄是：", future_age)
```

运行这段代码，你将在屏幕上看到变量的值以及计算的结果。

## 动态类型与类型转换 🔄

Python是一门“动态类型”语言，这意味着你不需要提前声明变量的类型，并且同一个变量可以被重新赋值为不同类型的值。但有时，我们需要在不同类型之间进行转换。

以下是两个关键概念：

1.  **动态类型**：变量类型由当前赋值决定，可以改变。
    ```python
    x = 10    # x现在是整数
    x = "十"  # x现在变成了字符串
    ```

2.  **类型转换**：使用 `int()`, `float()`, `str()` 等函数进行转换。
    ```python
    num_str = "100"
    num_int = int(num_str)  # 将字符串"100"转换为整数100
    result = num_int + 50   # 现在可以进行数学计算
    ```
