# 课程一：朋友圈的发展史 📱

在本节课中，我们将要学习社交媒体平台“朋友圈”的演变过程，理解其功能与社交意义的变化。

朋友圈最初作为简单的社交分享功能出现。它允许用户发布文字、图片和视频。随着时间推移，朋友圈的功能逐渐丰富。它增加了点赞、评论和转发等互动方式。这些功能增强了用户之间的连接。朋友圈的内容形式也变得多样化。从最初的生活记录，发展到涵盖工作、兴趣和商业信息。朋友圈的算法推荐机制也经历了多次调整。这些调整影响了内容的可见度和传播范围。如今，朋友圈已成为许多人日常生活中不可或缺的一部分。它不仅是社交工具，也是个人形象展示的窗口。

上一节我们介绍了朋友圈的宏观发展脉络，本节中我们来看看其功能的具体迭代。

以下是朋友圈发展过程中的几个关键节点：

*   初期阶段：以纯文字和基础图片分享为主，互动形式简单。
*   功能扩展：引入小视频、位置打卡、@好友等功能，内容更立体。
*   算法介入：平台通过算法对内容进行排序和推荐，影响信息流。
*   商业化：出现广告投放、小程序导流、直播等商业形态。
*   隐私细化：提供更精细的权限设置，如“三天可见”、“分组可见”。

本节课中我们一起学习了朋友圈从诞生到成熟的发展历程，了解了其功能演进与社会角色的变迁。理解这段历史，有助于我们更好地认识和使用这个平台。

# 课程三：如何用手机将美食风景修得高逼格 📸

在本节课中，我们将要学习使用手机应用程序对美食和风景照片进行后期处理，以提升其视觉质感。

手机修图的核心在于调整几个关键参数。原图拍摄时需注意构图和光线。后期修图不是挽救废片，而是优化好照片。常用的调整工具包括曝光、对比度、饱和度等。针对不同题材，调整侧重点有所不同。美食照片需突出色泽与食欲。风景照片需强化层次与氛围。使用预设滤镜可以快速统一风格。但需根据具体照片微调参数，避免生硬。最终目标是让照片看起来自然且高级。

上一节我们建立了手机修图的基本认知，本节中我们来看看针对美食和风景的具体调色思路。

以下是针对两类题材的修图步骤建议：

**美食修图思路：**
1.  基础调整：适当增加**曝光**和**对比度**，让食物更明亮立体。
2.  色彩强化：微调**饱和度**，并单独提升**橙色**和**红色**的饱和度，使食物更诱人。
3.  质感突出：增加一点**清晰度**或**锐化**，展现食物细节。
4.  氛围营造：适当增加**暗角**（Vignette），引导视线聚焦于食物。

**风景修图思路：**
1.  恢复细节：调整**高光**和**阴影**，找回天空和暗部的细节。
2.  定调：使用滤镜或调整**色温**（偏蓝或偏暖）确定照片基调。
3.  层次强化：增加**对比度**和**清晰度**，让山峦、云层更有层次。
4.  色彩优化：调整**HSL**工具，单独加强天空的蓝色或植物的绿色。

本节课中我们一起学习了提升美食与风景照片质感的手机修图方法。关键在于理解**曝光、对比度、饱和度、HSL**等核心工具的作用，并针对不同主题进行有侧重的调整。

一张环境美食图不仅是记录，更是个人生活的切片。它间接传递了拍摄者的消费场所、审美品味和生活格调。环境氛围能烘托食物的品质。构图体现了拍摄者的观察力和构图能力。整体的色调和质感直接关联到“高级感”。因此，修图的目的在于强化这些价值信号。诱人的色调能激发观看者的共鸣与向往。本节将结合实例，解析价值点并演示调色。

上一节我们理解了环境美食图的价值，本节中我们来看看如何通过调色将其最大化。

以下是一个典型的诱人美食调色参数示例，以VSCO滤镜为例：

1.  **滤镜选择**：常用 **A6、C1、G3** 等滤镜作为基底，增添复古或清新质感。
    ```代码
    滤镜：A6/Analog
    强度：+8
    ```

    ```代码
    曝光：+1.5
    对比度：+2
    锐化：+3
    ```

    ```代码
    饱和度：+1

    阴影补偿：+2 (提亮暗部环境)
    ```
4.  **风格化微调**：
    ```代码
    色温：-1 (让画面稍微偏冷，更干净)
    色调：+1 (给阴影加一点绿色，增加复古感)
    褪色：+1 (可选，增加灰度质感)
    ```

本节课中我们一起学习了环境美食照片的价值传递原理，并掌握了一套实用的**VSCO调色参数**。记住，调色的目标是让照片看起来“色香味俱全”，并契合你想要传达的氛围与格调。

# 课程五：VSCO初级入门示范美食风景调色方法以及线下同学的爆炸蜕变 🎨

在本节课中，我们将要面向初学者，演示如何使用VSCO进行美食和风景调色，并通过案例展示学习前后的显著变化。

VSCO是一款功能强大且易用的手机调色软件。初学者可以从使用预设滤镜开始。理解每个参数的作用是进阶的关键。本课将通过对比图，直观展示调色带来的改变。案例来自线下学员的实际操作。我们将分解他们的调色步骤。你会发现，简单的调整就能让照片焕然一新。建立信心是学习修图的第一步。

上一节我们树立了通过VSCO改变照片的信心，本节中我们通过一个具体案例来分步操作。

以下是针对一张普通风景照的VSCO初级调色示范：

**原图问题**：画面灰暗，色彩沉闷，缺乏层次。
**调色目标**：让天空更蓝，草地更绿，整体通透。

**操作步骤：**
1.  选择滤镜 **C1/Chromatic**，强度降低到 **+6**，奠定色彩基调。
2.  进入“编辑”面板，进行以下调整：
    ```代码
    曝光：+1.0
    对比度：+3
    锐化：+4
    饱和度：+2
    ```
3.  使用 **HSL** 工具进行针对性色彩调整：
    ```代码
    蓝色：饱和度 +4，亮度 -1
    绿色：饱和度 +3，亮度 +1
    黄色：饱和度 +2 (让阳光感更强)
    ```
4.  最后调整 **色调分离**：
    ```代码
    高光色调：蓝色 +2
    阴影色调：绿色 +1
    ```

通过以上步骤，照片从“普通记录”变为“明信片风格”，实现了“爆炸蜕变”。本节课中我们一起学习了VSCO的初级调色流程，见证了参数调整对照片质感的决定性影响。多练习是掌握调色的唯一途径。

# 课程七：如何拍一张帅气的头像 🤵

在本节课中，我们将要学习拍摄一张适用于社交媒体的帅气头像的基本方法和要点。

一张好的头像是个人的第一张名片。它需要清晰、正面、且富有感染力。自拍或他拍均可，但需保证画质。光线是头像拍摄中最关键的因素。自然侧光或窗光能塑造良好的面部轮廓。背景应简洁，避免杂乱。表情和眼神需要自然放松。可以尝试微笑或略带思考的表情。构图以胸部以上特写为主。后期修图应适度，保留个人特征。

上一节我们确立了帅气头像的标准，本节中我们来看看实现它的具体操作清单。

以下是拍摄一张帅气头像的步骤指南：

*   **光线准备**：寻找柔和的自然光，避免顶光和直射强光。
*   **背景选择**：选择一面干净的墙或开阔的户外背景。
*   **角度与构图**：手机镜头与眼睛平齐，构图时头部上方留少许空间。
*   **表情管理**：深呼吸放松，想象愉快的事情，让眼神柔和有光。
*   **细节检查**：整理发型、衣领，确保眼镜无反光。
*   **连续拍摄**：使用连拍功能，捕捉最自然的一瞬间。
*   **后期微调**：仅做亮度、对比度、肤色的轻微调整，切忌过度磨皮。

本节课中我们一起学习了拍摄帅气头像的全流程。记住公式：**优质头像 = 良好光线 + 简洁背景 + 自然神态 + 适度后期**。多练习才能找到自己最上镜的状态。

# 课程十：男生ins风头像修图技巧（一） 🧑‍🎨

在本节课中，我们将要开始学习为男生头像打造流行的“Ins风”质感的修图技巧，这是系列课程的第一部分。

Ins风头像通常带有统一的色调和质感。它不等于过度美颜，而是强调风格化。基础调整是修图的根本。肤色处理是关键，要健康自然。画面干净是首要原则。本课重点在于使用Snapseed或VSCO进行基础定调和肤色校正。我们将从一张原图开始，一步步达到理想效果。

上一节我们明确了Ins风修图的方向，本节中我们使用Snapseed进行第一步——基础校正与肤色优化。

**操作步骤：**

1.  **整体调整**：打开Snapseed的“工具”->“调整图片”。
    ```代码
    亮度：+20
    对比度：+10
    饱和度：+5
    氛围：+15 (智能调整明暗部)
    高光：-10
    阴影：+5
    暖色调：-5 (让肤色更干净，偏冷一点)
    ```
2.  **局部肤色修正**：使用“工具”->“局部”。
    *   在面部皮肤区域添加控制点。
    *   双指缩放选择影响范围。
    *   向上滑动选择“亮度”，适当**+10**。
    *   再次向上滑动选择“对比度”，适当**-5**，让皮肤更柔和。
3.  **细节强化**：使用“工具”->“突出细节”。
    ```代码
    结构：-5 (轻微磨皮，使皮肤平滑)
    锐化：+20 (增强眼睛、眉毛、头发细节)
    ```

# 课程十一：男生ins风头像修图技巧（二） 🎭

在本节课中，我们将要承接上一课的基础，深入学习为男生头像添加风格化色调和氛围的进阶技巧。

基础调整后的照片干净但可能平淡。风格化调色能赋予照片情绪和辨识度。常用的Ins风色调包括青橙色调、黑金色调、低饱和冷淡风等。本课重点介绍使用VSCO打造一种通用的高级灰调。我们将通过调整色彩平衡和分离色调来实现。

上一节我们完成了照片的基础美化，本节中我们使用VSCO为其注入风格灵魂。

以下是在VSCO中打造高级灰调Ins风的参数：

1.  **滤镜选择**：使用 **HB2/Hypebeast** 或 **A4/Analog** 滤镜，强度降低至 **+4到+6**，作为色调基底。
2.  **精细色彩调整**：
    ```代码
    曝光：+0.5 (根据上一步结果微调)
    对比度：-1 (降低对比，营造柔和感)
    锐化：+3
    饱和度：-1 (整体降低饱和度)
    色温：-2 (让画面偏冷蓝)
    色调：+1 (给阴影加一点洋红，平衡冷调)
    ```
3.  **色调分离（关键步骤）**：
    ```代码
    高光色调：蓝色 +3
    阴影色调：橙色 +2
    ```
    *这个操作让高光泛蓝，阴影泛橙，形成微妙的色彩对比，是许多Ins大片的秘密。*
4.  **颗粒感**：适当添加 **颗粒 +2**，增加胶片质感。
