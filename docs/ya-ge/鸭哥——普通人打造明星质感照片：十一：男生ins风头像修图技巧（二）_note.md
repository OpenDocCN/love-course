## 核心修图逻辑：先修图，后调色

在开始实操前，我们必须明确一个核心流程：**先修图，后调色**。

很多同学会疑惑顺序问题。让我们来探讨一下：修图相当于烹饪时处理食材，比如切肉、洗菜，目的是让食材本身变得规整、好看。而调色则相当于添加油盐酱醋等调料，是为菜品赋予最终的风味和氛围。

因此，正确的步骤是：**先有型，再入味**。先把脸修得干净、轮廓清晰，再去调整照片的整体颜色和氛围。如果顺序颠倒，先调了一个很有感觉的色调，但人脸可能显得很黑或不好看，这时再回头修图就会非常麻烦，效果也容易不自然。

## 入门巩固：用轻颜相机快速提升颜值

在深入学习专业修图前，我们先通过轻颜相机巩固一个快速出片的流程。这个方法适合所有人，尤其当你没有专业设备或他人帮助时。

以下是使用轻颜相机美化自拍头像的快速步骤：
1.  **拍摄**：找一个光线好的地方，用轻颜相机的前置摄像头自拍。注意发型、配饰（如项链、眼镜）和衣领以上的细节。
2.  **基础美化**：导入照片后，使用“瘦脸”功能微调脸型和下巴。使用“美颜”下的“磨皮”功能改善皮肤质感，但注意不要拉满，保留一些皮肤细节。接着使用“美白”功能提亮肤色。
3.  **风格滤镜**：最后，选择一个适合的滤镜（如“质感”或“胶片”系列），为照片增加风格。

这个流程能让你在几分钟内获得一张远超原图的帅照。关键在于多尝试每个功能，找到最适合自己的调整幅度。

## 专业进阶：FaceTune精细修图全流程

现在，我们进入本节课的核心干货部分：使用FaceTune进行精细修图。我们将通过一个案例，展示如何将一张有痘痘、肤色不均的原图，修成一张干净帅气的头像。

### 第一步：修复瑕疵（祛痘、去印）

我们首先处理皮肤上的痘痘、痘印等明显瑕疵。

打开FaceTune，导入照片。使用 **“修复”** 工具（图标类似创可贴）。这个工具的原理是取样周围的好皮肤，覆盖在瑕疵上。

**操作要点**：放大图片，仔细地将每一个痘痘和明显的痘印点掉。这个过程需要耐心，目标是让皮肤看起来平整干净。美图秀秀的“祛痘祛斑”功能同理。

### 第二步：平滑肌肤（磨皮）

瑕疵修复后，皮肤可能仍有些许不平整或粗糙，这时需要进行磨皮。

使用 **“磨皮”** 工具。**切记不要在全脸随意涂抹**，否则会丢失所有皮肤细节，显得很假。

**操作要点**：避开眉毛、眼睛、嘴唇、鼻子轮廓等五官部位。只在脸颊、额头、下巴等皮肤区域进行轻柔的涂抹。可以调整磨皮强度，以达到“皮肤变好但仍有质感”的效果。

### 第三步：调整轮廓与五官（精修）

这是提升颜值的关键一步，相当于“数字微整形”。

使用 **“重塑”** 或 **“扭曲”** 工具。这个工具可以让你推拉脸部的特定区域。

**常用调整思路**：
*   **瘦脸**：将脸颊边缘、下巴向内侧轻微推动，让脸型更精致。
*   **增高鼻梁**：在鼻梁侧面向上轻微推动，让鼻子更挺拔。
*   **调整眼型**：可轻微放大眼睛或调整眼尾角度。
*   **调整发型**：如果头发塌陷，可以将其向上拉，增加颅顶高度。

**核心原则**：所有调整必须**轻微、自然**。修图的目标是优化，而不是变成另一个人。修完后缩小图片看整体，确保没有畸形感。

## 最终点睛：使用VSCO进行调色

完成所有修图步骤后，我们得到了一张干净、帅气的“素颜”照片。最后一步是为它注入氛围感，这就是调色的工作。

我们将照片导入VSCO。调色没有固定参数，但思路是相通的。

**基础调色思路**：
1.  **调整曝光**：如果照片整体偏暗，先适当增加曝光，让画面更通透。
2.  **调整对比度与饱和度**：适当增加对比度能让画面更清晰；微调饱和度让色彩更鲜活或更沉稳。
3.  **使用滤镜**：VSCO有大量优质滤镜。选择一个你喜欢的滤镜（如A系列、G系列），然后**大幅降低滤镜强度**（例如降到+3到+6）。这样既能获得滤镜的色调风格，又不会过于夸张。
4.  **微调色调**：在滤镜基础上，可以进入“色调”选项，微调“高光”（让过亮部分柔和）和“阴影”（提亮暗部细节）来平衡光影。

**公式**：**干净的人像 + 低强度的风格化滤镜 + 细微的光影调整 = 具有高级感的头像**
