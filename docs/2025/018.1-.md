# 恋爱必修课：018：挽回的核心流程与方法

在本节课中，我们将要学习情感挽回的基本流程与核心方法。课程将剖析分手的主要原因，并详细讲解挽回过程中必须遵循的步骤与必须避免的误区。

## 课程概述

从2016年下半年起，我接触了大量寻求一对一挽回服务的客户。客户构成广泛，从学生到老板，从职业球员到政府官员，从男性到女性，涵盖律师、医生、教师、警察等各种职业。在深入介入他们的情感案例后，我发现，无论当事人属于何种职业，导致他们在感情中失败的根本原因，都源于自身在情感上的不成熟。

## 情感消失的常见原因

以下是导致情感关系破裂的几种典型原因：

1.  **男生过于木讷**：性格太闷、内向、缺乏主见，完全不懂调情，也不够“坏”。一段时间后，女生会觉得与你在一起非常无趣，从而慢慢放弃。
2.  **男生过于轻浮**：太会调情、太“坏”，总是想迅速推进关系，过早流露出过多的性需求。被女生贴上此类标签后，对方便会敬而远之。
3.  **男生过于情绪化**：缺乏自信与安全感，占有欲强、多疑。女生不堪其扰，最终绝望离开。
4.  **男生能量消极**：工作生活状态向下，消极抱怨，一事无成。女生因为看不到未来的希望而最终离开。

无论由何种原因导致分手，挽回的第一步都是相同的。

## 第一步：冷静断联

分手初期，我们的情绪最为起伏、崩溃，也最难以承受。但请记住，这个阶段最错误的事情就是联系对方。

在大量案例中，我目睹了当事人进行短信、微信、电话轰炸，甚至跑到对方公司、学校或住处蹲守，以及寻找对方的朋友、闺蜜、同事、领导进行劝说等行为。这些行为除了让本已脆弱的关系变得更加不堪一击，并极大增加挽回难度外，没有任何实际意义。

上一节我们分析了分手的原因，本节中我们来看看为什么急于联系是错误的行为。

当对方不回你短信、不接你电话时，其行为本身就在传递一个明确信息：他/她目前不想与你接触。此时，若仍执着于“我就是想见他/沟通”，这便忽略了沟通的基本前提。

**有效的沟通**，需在双方都有意愿的情况下，信息才能有效传达，从而消除误会、增进感情。若对方已无意愿，再三的“沟通”实质上已变成**骚扰**。试问，谁会喜欢一个经常骚扰自己的人呢？

因此，在挽回的第一阶段，必须避免这种南辕北辙的行为。我们的目标是让对方重新喜欢我们，就千万不能做那些让对方讨厌我们的事。要做到这一点，首先需要**冷静下来**。

## 第二步：自我审视与重建

冷静之后，我们进入第二步：找一面镜子，审视镜中的自己。

问自己两个问题：我是否喜欢现在镜子里的这个人？我能否控制住镜子里的这个人？

如果我们连自己都无法控制，却妄想去控制对方，甚至控制对方身边的其他人，这便是一种**徒劳的妄想**。请记住，**控制别人的前提，是学会控制自己；自律方能自强**。同样，**别人喜欢我们的前提，是我们自己喜欢自己**。如果我们自己都不喜欢自己，凭什么要求别人喜欢你呢？

有的同学可能会说：“可他/她以前很喜欢我呀，现在怎么就不喜欢了？” 那么，此时就请看看镜中的自己。

*   以前的你是什么样子？现在你和以前的你有多大区别？
*   那时的你自信、阳光、神采飞扬，在他/她面前淡定、潇洒、不羁。
*   现在的你是否自卑、抑郁、无精打采，在他/她面前拘谨、无奈、不知所措？

回想一下，自己是怎样一步步变成今天这个样子的。**爱别人爱到失去自我，就会丢掉“心上人”。这里的“心上人”指的就是我们的自我。自我丢掉了，身边人自然会离开。**

道理很简单：如何挽回？**找回自我**。回想当初吸引到对方时的自己，那时你的生活是怎样的？现在，你需要把生活恢复到曾经的样子，甚至要**变本加厉地进行自身建设**。

在你的朋友圈、社交圈中分享积极的生活，让身边的人感受到你的变化，重新看到你的内在价值：自信、阳光、勇敢、孝顺、包容、担当等。只有这样，才能**重新建立吸引**，再续前缘。

## 关于结果与责任

当然，即便我们做好了这一切，挽回仍有失败的可能。地球并非围绕我们转动，努力做好自己，结果也不一定尽如人意，正所谓“尽人事，听天命”。

况且，弥补和挽回毕竟是事后行为，关系中曾经存在的阴影和痕迹难以完全抹去。如果我们曾给对方带来巨大伤害，令其绝望，那么最终的责任仍需由我们自己承担。

大家要记住，**挽回与搭讪**，一个发生在关系开始的初识阶段，一个发生在关系结束的最后阶段，但它们有一个共同点：**行动看似针对对方，但最终作用在于自身**。

我们以为通过搭讪认识了对方，其实是通过搭讪认识了自己；我们以为通过挽回是挽回对方，其实是通过挽回找回自己。

此外，必须告诫大家：**一切外因都通过内因起作用**。如果在感情中出现第三者，务必保持冷静，不要做出损人不利己、甚至亲者痛仇者快的事情。千万不可因情绪失控，对自己或对方造成任何程度的肢体伤害。这些行为都是幼稚、不成熟的表现，只会让对方及身边的人更加瞧不起我们。

既然我们是自己情感生活的源头，那么情感出现问题，我们就是**第一责任人**。多从自己身上找原因，才能让情感更加成熟。抱怨和指责，永远不会带来我们想要的关系。

## 课程总结

本节课中，我们一起学习了情感挽回的核心流程。

1.  **首要原则是冷静断联**，避免在情绪崩溃时进行无效甚至反效的纠缠。
2.  **核心方法是自我审视与重建**，挽回的本质是找回那个曾经美好、有吸引力的自我。
3.  **需要认清责任与结果的不可控性**，尽己所能，同时接受任何可能的结果。

请记住：**我们的“心上人”其实就是我们的自我。自我在身边，身边的人就在。挽回的不是别人，是曾经美好的自己。** 请大家努力认真学习这套恋爱必修课，将功夫下在平时，多“保养”，少“看病”。
