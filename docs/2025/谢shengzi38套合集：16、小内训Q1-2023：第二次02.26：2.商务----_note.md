## 课程内容

### 第一部分：人生语境下的商务 🌱
上一节我们介绍了课程的整体框架，本节中我们来看看如何做好自己人生的商务。每个人都可以将自己视为一家无限责任公司的CEO，人生的商务就是为自己挖掘和运作潜在的资源与人脉。

商务的本质可以概括为一个三角模型：**结识人脉 -> 运作人脉 -> 促成落单**。一个优秀的商务人员需要在这三个方面都有所建树。

在人生维度，有效社交的前提是明确你的人生要“促成”什么“单”。如果你对自己的人生使命或事业目标不够清晰，所谓的“有效人脉”往往流于表面。你需要想清楚自己未来几年要达成什么目标，才能进行精准的社交。

人生轨迹通常遵循“加法 -> 减法 -> 加法”的阶段：
*   **第一阶段（加法 - 试错）**：目标不明确，处于探索期。此阶段的商务重点是**结识人脉**，目标是获取广泛信息，帮助自己尽快进入专注阶段。此时应保持低门槛，友善、开放，无需过度展示自己。
*   **第二阶段（减法 - 专注）**：已确定专注领域。此阶段的商务重点是**运作人脉**，目标是促成落单。你需要围绕核心领域，绘制资源图谱（如官方、市场、流量、商业等方向），并进行精准、向上的社交运作。
*   **第三阶段（加法 - 跨界）**：在专注领域取得显著成就，资源和能力外溢。此阶段的重点是宣传实绩，提升整体咖位和影响力，进行跨界链接。

以下是运作人脉（特别是向上链接）的一些具体方法：
1.  **大方付出**：不占人便宜。如果是你发起的邀约，应主动买单。购买对方的产品/服务、在其直播时赠送礼物等“前置付出”是有效的破冰方式。
2.  **首次一对一沟通**：展示你对其长期的关注与了解，并表达自己的上进心。沟通时不卑不亢，多倾听。
3.  **跟进与维护**：根据重要性分“重跟”与“轻跟”。重跟需保证在一定周期内增加线下见面频率。沟通后当晚发送感谢信息。判断是否真正“勾搭上”的标准包括：进行过一对一交流、互相邀请参加小规模活动、通过彼此认识了新的人脉。
4.  **促成落单**：方向有二：以你的需求为导向，直接向对方提出明确的请求；或以对方的资源为导向，思考对方能轻松提供何种帮助，并提出相应请求。

### 第二部分：职场语境下的商务 💼
在了解了个人层面的商务后，我们进入职场语境，看看如何成为一名公司的商务拓展（BD）人员。

BD岗位的优势是离资源和行业信息近，劣势是若能力不扎实，积累的资源可能无法随个人迁移。其工作同样遵循“结识、运作、落单”三角模型，你需要根据公司或项目阶段，明确当前的工作重点。

**1. 结识人脉（开拓入口）**
对于创业公司或新项目，核心通常是打开知名度，结识人脉。
*   **线下论坛/活动**：提前做好背调，明确目标结识对象（如发言人）。准备充分的公司及个人资料（电子与纸质）。抓住机会与关键人物建立初步联系。
*   **线上勾搭**：运营好个人社交账号。沟通时清晰介绍自己与目的，表达欣赏，并设置一个让对方易于回复的“轻入口”（如“加个微信”），而非一上来就提出沉重请求。注意信息分段，并由你结束发起的对话。

**2. 运作人脉（激活资源）**
激活已建立联系的人脉。
*   常见方式是举办线上（如直播连麦）或线下（精品沙龙、定向邀请活动）活动，创造互动机会。

**3. 促成落单（实质推进）**
在商业语境中，促成落单需要锲而不舍的精神。
*   对方推脱通常是因为“交换条件没到位”或“需求表述太模糊”。
*   如果首次沟通被拒，不要放弃。可以间隔一段时间后再次联系，将合作价值描述得更清晰或略有提升，通常能获得二次回应。

### 第三部分：自媒体/个人IP的商务案例 📱
最后，我们以自媒体或个人IP接广告为例，将商务逻辑具体化，这也是对职场商务的案例深化。

广告投放通常有两个目的：**品牌宣传（品宣）** 或 **直接卖货**。你需要明确每次合作的主要目标。

整个投放链条是：**品牌方 -> Agency（代理机构）-> KOL（关键意见领袖）**。品牌方会制定分平台、分量级、分类别的投放策略。

一个好的自媒体商务需要做到以下几点：

**1. 主动链接新品牌方**
*   研究目标品牌在同类博主处的投放情况，确认其投放策略包含你所在的平台。
*   通过品牌官方渠道，撰写专业、清晰的合作邀约信息，争取加到对方商务的微信。
*   将你的成绩案例主动发送给对方，并长期维护商务账号的内容。

**2. 深度维系已有品牌方，争取复投**
*   理解与你对接的品牌方员工的KPI（关键绩效指标），帮助他超额完成任务。
*   合作前深入沟通Brief（工作简报）：了解产品卖点、投放期望（数据、调性等），索要过往成功案例并分析原因。
*   交付时追求“一稿过”，展现专业与高效。
*   合作后主动复盘，无论效果好坏，都表达共同优化、期待下次合作的态度。

**3. 促成落单与长期合作**
*   通过专业服务、深度沟通和积极复盘，将单次合作转化为长期伙伴关系。

## 问答环节精选
在课程的最后，我们针对一些常见问题进行解答。

**问：如果帮了对方，对方是“白眼狼”怎么办？**
*   选择链接对象时，应优先选择那些在其领域已取得成就、大概率“会做事会做人”的人。
*   付出时心态要平和，不急于即时回报。商务运作，尤其是人生维度的，不是快销逼单。
*   相信真诚用心的付出，长远看总会有回报。

**问：广泛结交和精准运作可以同步进行吗？**
*   可以。关键在于精力分配。可以先广泛开拓，再从中筛选重点进行精准运作。

**问：向上链接和自己努力深耕的比率如何把握？**
*   不应将两者对立。向外链接也是获取信息、学习提升、助力业务深耕的重要途径。
*   根据自身情况调整：若感觉实力被低估，应加强链接；若感觉链接后难以承接机会，则应回头深耕。

**问：如何恰当地“麻烦”别人？**
*   要么以你的明确需求为导向，要么以对方举手之劳便能提供的帮助为导向。提出请求时，要让人感觉你充分考虑了对方的便利性。

**问：如何把客户资源内化成个人资源？**
*   资源建立在公司平台之上，不完全属于个人。
*   通过极致的专业能力、到位的为人处世，让对方认可你“这个人”。即使更换平台，对方仍可能基于对你的信任而合作。当然，个人离开平台后的自身实力是关键。

**问：针对难题，如何有效请教？**
*   在自行充分调研和思考后，向你认可的专业人士付费咨询。这是表达诚意、尊重对方时间价值的有效方式，也更容易获得高质量的回答。
