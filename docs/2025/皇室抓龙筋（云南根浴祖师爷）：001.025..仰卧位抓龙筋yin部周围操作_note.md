# 皇室抓龙筋（云南根浴祖师爷）：001.025：仰卧位抓龙筋阴部周围操作

在本节课中，我们将学习仰卧位下，针对阴部周围区域进行抓龙筋操作的具体手法与流程。本节内容侧重于润滑与初步的放松推拿。

## 🧴 操作准备与润滑

开始操作前，操作者需在手上涂抹适量按摩油。随后，双手再次蘸取按摩油，为客人的操作区域进行润滑。

以下是具体的润滑步骤：
*   将客人的**阴阜**部位展开并涂抹按摩油。
*   在客人的**大腿内侧**区域涂抹按摩油。

润滑完成后，我们便可以开始进行持骨外层的推拿操作。

## 🤲 持骨外层推拿

上一节我们完成了区域润滑，本节中我们来看看持骨外层的具体推拿手法。操作时需兼顾身体两侧。

操作从腹股沟位置开始，手法向身体中线方向聚合。主要包含分推与合推两种手法。

以下是该环节的核心步骤：
1.  **分推**：双手从腹股沟向两侧分推。
2.  **合揉**：双手从两侧向腹股沟中线合拢揉按。
3.  **分油**：通过分推手法将按摩油均匀推开。
4.  **沿大腿内侧下推**：双手沿大腿内侧向下推按。
5.  **沿腹股沟上推**：双手沿腹股沟位置竖向向上推按，以放松该区域。
6.  **沿腹股沟弧形上推**：双手沿腹股沟走向做弧形向上推按。
7.  **弧形内推**：双手沿腹股沟向内侧做弧形推按，然后向下。

完成一侧操作后，以同样手法操作另一侧。

## 🥚 阴囊部位操作

完成了外层持骨的放松推拿后，我们接下来针对阴囊部位进行专门操作。

首先进行阴囊部位的竖向推拿。推拿路径沿阴囊正中间、左侧及右侧进行。

以下是该环节的操作列表：
*   **正中间竖推**：在阴囊正中间进行竖向推按。
*   **左侧竖推**：在阴囊左侧进行竖向推按。
*   **右侧竖推**：在阴囊右侧进行竖向推按。

竖向推拿完成后，转为环形手法。用双手环形兜住阴囊，做轻柔的提升动作。随后，双手以弧形路径向内、向下放松。

## 🔁 重复与结束

每一个上述动作都可以根据情况重复进行。通常，每个动作重复 **5到8遍** 即可达到预期效果。

本节课中我们一起学习了仰卧位下阴部周围抓龙筋的操作流程。我们从区域润滑开始，逐步进行了持骨外层推拿、阴囊部位竖推与环形提升等手法。掌握这些基础手法是进行后续深度操作的前提。
