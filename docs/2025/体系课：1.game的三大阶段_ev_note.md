# 真实流Game体系课：1：三大阶段概述

在本节课中，我们将要学习真实流Game体系的完整框架，了解从认识女生到建立关系的三大核心阶段及其底层逻辑。

任何成功的交往案例，都必然满足了特定条件并经历了特定阶段。市面上很少有体系能将其完整总结。

你可能听说过，在交往过程中要保持自然和松弛感。面对一些比较友好的女生，这种方法可能有效。但你也会发现，这种方法很多时候并不管用。当然，你可能也听说过交往需要“架子”。有了架子，你就可以为所欲为。

但你会发现，别说高分女生，甚至有些价值比你低的女生，明明你很有价值，但你拿她没办法。也就是说，无论是自然大方做自己，还是单纯拉高价值，都无法涵盖所有情况。这时可能有人会说，交往本就没有规律和套路，每个女生都不一样，需要随机应变。

听起来很有道理。但我们要知道，每个女人虽然各有特点，但本质上都是人，都是女人。就像我们男人一样。你喜欢的女生，大概率其他男人也喜欢。你喜欢女生对你做的事，大概率其他男人也喜欢。或许存在个体差异，但本质上是因为这个女人能满足某些需求和条件，所以她能获得大部分男人的喜欢。这是一种普遍情况。

同理，我们男人也一样。能让大部分女人喜欢的一些操作和过程是有迹可循的。你能成功，也是因为你满足了某些条件，经过了她的层层筛选，经历了某些阶段，最终她才选择和你在一起。

现在你应该明白，交往其实有一把“尺子”，可以衡量你和女生已经到达了哪个阶段。那么这把“尺子”是什么呢？就是接下来要介绍的真实流Game体系。

## 真实流体系的三大阶段

真实流体系共包含三个阶段，九个步骤。

1.  **第一阶段：进攻 - 构建价值感**。
2.  **第二阶段：后撤 - 建立联系感**。
3.  **第三阶段：定位 - 确定延续性**。

在详细介绍这三个阶段的步骤之前，需要先解释为什么交往会有这三大阶段。

## 第一阶段：为什么需要构建价值感？

想象一下，你走在路上看到一个乞丐。你会想和他产生关联吗？不会。因为他对你没有任何价值，你甚至不愿意给他注意力。第一眼你就判断了他没有价值。

但如果是一个穿着华丽的贵公子，或者一个身材妖娆的美女呢？你会怎么样？首先你会给他大量注意力，你会被他吸引，甚至可能回头看他。你可能会想，如果和这个贵公子成为好兄弟，能获得多少好处。你也可能会想，如果和这个美女在一起，该有多愉快。

**只有一个有价值、有魅力的对象，你才会给予注意力，才会产生拥有的想法。**

所以，第一大阶段就是构建价值感的阶段。**没有价值，就没有所有后续**。这也解释了为什么你一味的自然大方、为女生提供情绪价值，并不一定能吸引到她。如果女生判断你的价值很高、很有吸引力，那么你这么做是有用的。但绝大部分情况是，女生根本不了解你的价值时你就这么做，最后你反而成了“舔狗”或“小丑”。

听到这里，很多人会说：这不就是吸引力吗？我要不断拉高吸引、保持框架，对吧？我只能说你只知其一，不知其二。**单纯的价值并不完全等于吸引力**。你的实践也会告诉你，这样做并不一定能成功。

## 第二阶段：为什么需要建立联系感？

这就引出了我们的第二个阶段：联系感阶段。为什么要建立联系感？

沿用上面的例子，你见到一个贵公子或一个身材火辣的美女。你觉得他/她很有魅力、很有价值，对吧？但你会对他/她有什么投资或行动吗？通常不会。为什么？因为你觉得他/她跟你没什么关系，缺乏**联系感**。

所以，哪怕对方价值很高，对你的实际吸引力也可能不大。你不会为了他们去做有实质性付出的行动。这就是为什么很多高价值的男人，搞不定价值比他低的女人。

有人可能说，他会去搭讪那个辣妹。但事实上，大多数人只会搭讪那种“上不上下不下”、对自己有最大吸引力的女生。那种特别漂亮的，估计根本不敢去搭讪。就算搭讪了，发几条信息没反馈，马上就会放弃，根本不会投入太多。

真正让你投入多的是哪种女生？是那种长相身材尚可，然后给了你一些机会和窗口的女生。这种才会让你非常上头，可能寝食难安，天天想着为她付出。这种才能把你“玩得跟狗一样”，而不仅仅是越漂亮的女生越能拿捏你。

现在知道为什么**价值不完全等于吸引力**了吗？所谓的“吸引流”，其实是“价值流”。它的那些行为，只是帮你拉高价值感，比如不说话、装高冷等稀奇古怪的操作。这可以拉高你的价值感，但**不一定能拉高吸引力，甚至可能降低吸引力**。现在懂为什么学习所谓的“吸引流”却拿不到结果了吗？

## 第三阶段：为什么需要确定延续性？

听到这里，可能已经让你明白了以前犯的很多错误。但接下来要讲的内容将彻底颠覆你的认知，完全打通你的任督二脉。因为我要告诉你，**哪怕你做好了前面两个阶段，距离最终成功仍有很长一段距离**。这就是第三个阶段：延续性。

如果这个阶段你能做好，那么这个女生可以说百分百能拿下，并且她可能会真的爱上你、把自己托付给你。

讲一下底层逻辑。为什么有价值、有联系感，还是有可能拿不下女生？

还是用上面的例子。现在那个贵公子或性感美女，你不仅觉得她很有价值，而且你们也有联系感，聊得很好。正当你准备上头的时候，你突然得到小道消息：这个人三天后就要走了，而且永远不回来。你会有什么感受？

你跟这个高价值的人认识了，还聊得挺好，现在你偷偷知道对方马上就会永远离开。思考一下，你会做出哪些行为？你肯定会“下头”。本来脑子里全是她，头脑非常火热，突然就瞬间降温，理性重新占领高地。

当你听到这个消息，首先会去确认真假。但如果你发现真是这样，她真的会马上离开，那么后面你就不会有任何投资了，你会马上“弃牌”。

所以，现在知道为什么有的女生明明跟你热火朝天，突然有一天断崖式降温，消息不回、电话不接，就跟失踪了一样吗？你一定遇到过或将来会遇到这种情况。

你可能会觉得是女生认为你没价值了，或者你对她不够好。但你又会觉得，明明你的价值没变，相处也挺正常，很莫名其妙。现在明白了吗？因为女生觉得跟你相处**没有延续性**。她可能觉得你的发展不好，未来价值可能会下降；或者她觉得你就是想短期关系，以后根本不会跟她有联系。所以她就会弃牌，你们的关系也就破裂了。

**延续性就是收尾的临门一脚，是最后阶段最重要的一环。**

## 总结与体系的重要性

以上就是真实流Game的三大阶段。如果你能把这三个阶段做好，那么你几乎可以做到百分百收尾。因为这就是人性，是每个人在关系中最看重的三个方面，女人也不例外。

需要注意的是，三个阶段有先后顺序：
*   你不能在建立价值感之前做太多联系感，那样会让你变成舔狗。
*   你也不能在有了价值感之后不去做联系感，那样会让你变成装逼男。
*   你更不能在做好前两个阶段后，不去做最后一个阶段（延续性），因为这样很可能让女生在最后一刻拒绝你。

所以，一定要严格按照体系来操作，这能让你稳定地拿到结果。
