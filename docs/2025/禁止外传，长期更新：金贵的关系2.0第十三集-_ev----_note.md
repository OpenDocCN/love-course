# 金贵的关系2.0：第十三集：如何面对抄袭 🛡️

在本节课中，我们将要学习一个在商业和自媒体领域几乎无法避免的挑战：如何面对抄袭。我们将探讨为何无需惧怕抄袭，以及如何通过构建自己的壁垒和护城河来保持领先优势。

## 概述：抄袭是成功的必然伴随物

无论你是在做生意还是做自媒体，只要做出一点成绩，就一定会被人模仿甚至抄袭。这几乎是百分之百会发生的事情。面对抄袭，关键在于有效地打造自己的壁垒和护城河，这是在竞争中保持优势的重要环节。

## 第一部分：心态上不要怕抄袭的四个理由

上一节我们概述了抄袭的普遍性，本节中我们来看看为何在心态上无需惧怕抄袭。以下是四个核心理由。

### 理由一：在无法改变的事情上，要放弃焦虑

这是我们非常典型的一个价值观。在精进过程中，最重要的事情是做好精力管理。不要把精力浪费在无法改变的事情上，比如抄袭。相反，应关注那些投入产出比高、有高度正反馈的事情，以提高效率和发展自我。

抄袭基本上是一件完全无法改变的事情。只要你做得够好，特别是在电商和自媒体这些数据公开的领域，就一定会被模仿。你今天的营业额领跑行业，明天就可能有人拆解你的产品、话术和人设。

因此，我们不应指望别人不抄你，而应思考如何加强迭代意识和创新意识，让自己的环节更丰富，让别人越来越难抄袭。这要求我们每时每刻对自己的业务进行深度思考，以保持在行业中的领先状态和绝对竞争力。

### 理由二：几乎没有人能够做到一比一的复制

市场的绝大多数人非常懒。很多人说是在抄，但连抄都不会好好抄。例如，许多IP孵化机构拆解我，但他们没有深入摸清我的整个流程和链条，既没当过我的客户，也没去过我的线下课，甚至没和我们的客服深度交流。他们拆解出的东西是基于自己构建的想象，漏洞百出。

一个案例是，我们闺蜜联盟的一位小姐姐上过我的线下变现课后，又去了一家IP孵化机构的课程。该机构看似专业地拆解了我的业务模式，声称医美板块最赚钱。但这位小姐姐深知这不符合我们的实际情况。她听后感慨：“这个世界果然就是一个草台班子。”

我的整个业务体系没有保密机制，所有东西基本公开。但很多人连拆都拆不明白。当他们连你的基本框架都无法看明白时，又何谈一比一模仿你？整个动作会变形到“失之毫厘，差之千里”的程度。

再比如，很多情感博主看我做了《金贵的关系》课程后，也想做自己的精品课程，甚至找到了跟我合作的做客老师。但到了约访谈时，他们根本不愿意认真输出内容，只想拿到一个粗浅的结果，不愿意付出时间和精力。

更进一步说，即使有人认真抄你，也很难做到完全复制。因为别人能模仿的，往往只是你暴露在公众面前的内容，如营销手段和产品体系，这些是商业链条中最浮于表面的部分。

如果把商业链路比作一座冰山，他们抄到的只是水面上的冰山一角。水面下的大量工作，如创始人的价值观、管理体系、销售思路等，是隐藏的、难以模仿的。这就是为什么很多人一比一地抄我的课程，但变现量级却无法与我相提并论。

以我的管理体系为例，公司大部分员工是从学员转化而来的，他们与我的价值观高度一致，发心相同，方向一致。这一点，别的公司或IP想要模仿会非常难。首先，有多少人能有如此闭环和自洽的价值观？其次，即使有，也需要花很长时间打造影响力和吸引同频的人。每一步实操都非常耗时。等到别人真的一比一复制时，我可能早已迭代了好几个版本。他们可能抄得到“型”，但抄不到“魂”。

### 理由三：目标客群不一样

传统行业的抄袭也非常严重。例如，一个品牌出了爆款鞋子、包或家具，就会出现仿版满天飞的情况。以家具为例，一些海外设计师的高端品牌，一个沙发卖几十万。但在国内，常听到“5万块钱全屋定制”的广告，从图片看似乎一模一样。

然而，这些海外大品牌并不会因此感到无比焦虑。虽然他们讨厌这种行为，但本质上明白，盗版商家不会对他们形成非常大的冲击。一个重要因素是，品牌商和盗版商的目标客群往往不同。

一个愿意花5万配全屋家具的客户，不太可能去买几十万的沙发。反之，愿意花几十万买沙发的人，也不会考虑5万的全屋定制方案。以我最近买房为例，前房主用的所有家具都是正版，每个单品几十万起。一方面他有财力，另一方面他本职工作是创意行业，骨子里尊重原创，害怕自己不尊重别人的原创，自己的原创也不会被尊重。因此，他非常不愿意摆盗版。

再比如，很多情感博主模仿我，声称做高端用户。但真正比对时，发现目标客群落地时非常不同。我曾有一个客户纠结是否购买闺蜜联盟（十几万），并提出质疑：另一个情感博主的私董会才3万，你们的有什么不一样？

我让销售建议她：“不妨先买那个3万的私董会，买完后再考虑是否买我的闺蜜联盟。”我非常有信心，她买了3万的私董会后，进我闺蜜联盟的概率反而会大增。因为这里有很多小姐姐走了这个路径：当初觉得我的贵，换了别的类似产品，结果发现能量和磁场不是想要的，后悔浪费了3万，觉得应该直接来找我。

所以，被抄袭、被模仿不用太担心。有时目标客群不一样，这不完全是消费层级的问题，最大问题是底层价值观。服务好自己的客户，稳定好自己的价值观，才是最重要的。

### 理由四：学会转念，在抄袭环境中创新更容易

在一个全是抄袭的环境里，当所有人都在抄袭时，我创新的难度反而变低。我稍稍调整动作或理念，就可能领跑整个行业。如果行业里每个人都在拼命创新，充分角逐，这时我要做出遥遥领先的东西反而特别难。

高维打低维比较容易赚钱。在大家都在彼此抄袭的市场里，只要你的想法比别人先进一点点，就能形成降维打击。

例如，以前情感行业的人互相模仿抄袭，导致整个行业都是一种模式：倾向于做下沉市场，定价体系是“低转高”（先用低客单价吸引，后端销售游说升到高客单价），重头现金流来源放在“挽回”这个品上，并给客户很重的承诺。这几乎是我入行前所有情感机构的运行模式。

但看了他们的模式后，我不认为能达到理想效果，所以用自己的方式来做：不主动做下沉市场，不做低客单转高客单，铆定高端用户群体，直接售卖高客单价产品和服务，输出女性自我发展、向上攀爬的内容。这些动作的迭代在我看来并不是非常牛的措施，只是一点小小的意识创新。但由于之前市场太“草台班子”，只要我做了一点微小调整，就能直接吸引大批高端用户付费，而不用和同行一起卷下沉市场。

再比如，像我这样做IP相对成功的人，想拓展业务和扩大影响力，很多人会选择做IP孵化，或矩阵分发直播切片（我们也在做）。这是市场上目前的普遍做法。

但我没有孵化其他不相干的IP，也没有永远只做直播切片，而是做了“小曲曲”（曲曲分曲或曲曲实习生）。市面上有很多小伙伴真人出镜，输出我价值观体系下的内容。他们相对独立，又隶属于曲曲品牌体系下，类似影子主播。

相较于IP孵化和矩阵切片，做“小曲曲”项目反而更轻松，更容易撬动大结果。首先，矩阵分发直播切片容易产生同质化内容，在各大平台会受到查重影响，流量变低。我想让更多人接触我们的价值观，首先要避免查重。光做矩阵不够，而“小曲曲”模式很好地规避了这个问题：他们讲我体系下的内容，帮助我扩大影响力；同时他们是真人，会加入自己的思考和延伸，也能最大程度避免平台查重。

再看IP孵化，很多人觉得特别简单，以为是流水线生产。但实际上，孵化完全不同行业的IP是非常复杂的工作：帮他做内容、在台前直播、搭建后端产品体系等。每个环节，如果负责任，都需要投入大量精力和时间。所以，市面上做IP孵化的公司员工都特别多，因为交付太重，需要很多人同时服务一个IP。即使孵化成功，这些机构又面临极大风险：IP可能单飞。

而我在做“小曲曲”体系时完全不一样：他们讲的内容本质上是我的内容，只需要学习和模仿我；后端的变现方式也完全挂钩我的产品体系。所以我不怕他们单飞或离开，最大程度上降低了我的风险。

复盘“小曲曲”项目，它不能算特别牛的点子，只是IP孵化和矩阵打法的一个小小迭代和优化。可能在我之前，并没有很多人仔细思考除了IP孵化和矩阵之外，还有没有别的路子可走。绝大部分人看别人怎么走，他就怎么走。其实只要稍稍动点脑子，就能看到全新的风景。我相信这个课程推出后，市场上又会有很多博主跟上脚步，开始想做这种模式。但这没关系，因为我们还会持续保持进步和迭代。

以上是不要害怕被抄袭的四个核心理由。

## 第二部分：如果别人抄袭后做得更好怎么办？

讲到这里，可能还有人担心：万一别人抄袭后做得比我更好怎么办？面对这个问题，如果一个人抄了你，做得比你还更好，说明他在其他方面的综合优势比你更强。

他可能在管理维度上做了优化，在销售维度上做了迭代，或者资金链非常雄厚。总之，在你看不见的地方（冰山下面），很多点可能都做得比你好。他只是抄袭你的一个点，但最终在综合维度上对你进行反超。

举个例子，很多大型互联网公司的产品都是抄出来的。比如微信，一开始抄的是雷军做的米聊。我是米聊和微信都非常早期的用户。现在大家都用微信，米聊已被淘汰。难道微信对米聊是完全一比一复制吗？肯定不是。因为微信在复制米聊时，其财力、团队实力、腾讯体系下的用户基数等方方面面都碾压了米聊，形成了降维打击。

腾讯因QQ拥有强大的用户基数，有天然的流量优势；作为老牌互联网大厂，具备成熟的运营思路和拉新策略；还有大量的资金储备。这些都是米聊当初没有的。所以微信自然吃掉了米聊。

如果对方抄了你并超越你，你应该做的不是批判他，而是赶紧想一想，我到底有什么地方比不过他，导致这么明显的差距。

比如，很多人说雷军做小米SU7为什么能卖得这么好？我们发现当初被别人伤害的人，现在选择去伤害别人——因为小米SU7的整个外壳几乎抄了保时捷的外壳。但我们不要这么看问题，一定要带入保时捷的角度去想：小米SU7卖得好，难道仅仅是因为抄了我的外壳吗？其实不是，而是因为它在营销、宣发、价格上想得更透彻，更有综合竞争优势。那么，如果我想让保时捷的产品重新反超，就应该在这些点位上继续补强，避免未来再被别人超越。

## 第三部分：如何防止被抄袭和超越

上一节我们分析了别人抄袭后可能反超的原因，本节中我们来看看如何主动防止被抄袭和超越。以下是三个方面的思路。

### 思路一：不断迭代，领跑行业

不要惧怕被抄袭和超越，最核心的点是不要停下来，要持续迭代和创新。我们链条中每一个环节的打法都要不断优化。

*   **产品维度**：一直聆听客户的想法和意见，不断根据反馈改进产品。例如，我们的“曲曲新平台”和“AI曲曲”等新产品，都是不断创新后的结果。
*   **销售渠道**：持续拓宽更优质、更多元的销售渠道。
*   **内部机制**：从薪酬机制和管理架构上想办法，更好地激发团队战斗力。

这些做法的本质是“卷自己”，让我们领跑于行业最前端。当你跑得足够快、足够远时，根本不用害怕被抄袭。因为别人抄你时，可能跑到50米，而你因不断前进，已到了200米。抄不抄就无所谓了。核心还是要聚焦到自我提升和精进思考上。

### 思路二：搭建体系，变得复杂

所谓的护城河、壁垒、核心竞争力，一定不是单一维度的优势和长板。任何单一维度的优势都非常容易被模仿和抄袭。例如，如果你做直播带货，只有台前的表现力，那么有一个表现力比你更强的人，你就很容易被反超。

本质上，要搭建越来越完善的上下游体系，并且有多个维度的长板共同交织，才能形成护城河。

以直播带货为例，如果想构建绝对竞争力，需要哪些元素？既要有丰富的台前表现力，又能拿到相对较低的供应链价格，到后期还要发展几个品牌的独家带货权限（只有我能卖），甚至做自己的品牌。一定要多维度交织，不停想着怎么把护城河做得越来越高、越来越深。这样别人再想模仿时，需要打穿的维度太多，你就不太容易被取代。

例如，我们的“哼哈斯密”项目也是如此。如果我把手上的客户和流量直接倒给医院就不管了，那很多自带流量的博主都能干，很容易被模仿复制。然而，为什么目前我们在这个类目里打得非常响，很多人关注我们是如何做的？因为我们有好几层竞争力在交织，以至于不那么容易被模仿：

1.  **流量入口**：我这个IP本身是超级大的流量入口，能向后端项目源源不断提供客户。
2.  **专属团队**：我用自己团队运作项目，负责销售和后端服务。团队价值观与我一致，把项目研究得非常透彻。别的IP可能根本不会费这个劲。
3.  **定制课程**：哼哈课程不是搬市面上的现成东西，而是根据我的客户（价值观、收入水平、情感状况等）重新设计的。客户反馈我们的课程更高级。
4.  **过硬合作方**：后端的合作医院技术非常过硬。

这几个维度共同交织，让我在做这个类目时形成较强的竞争力。更进一步，这个项目被并入我的整个产品体系，实际上在为女性情感成长做终身服务时，也给整个项目带来更核心的竞争力，成为我不可取代的部分。

别人也做情感咨询，我也做情感咨询。但由于我们在“哼哈”维度上深入研究，有大量客户案例，意味着我们在前端做“脑子”咨询时也能更加通透。给客户建议时，不再停留于相处、情绪价值、聊天或个人价值层面，还可以有更多与身体有关的分析和分享。精神加身体双重打通，能给客户更落地的综合化建议。

再举个例子，我们闺蜜联盟有个小姐姐特别擅长发朋友圈，能让大家通过朋友圈感受到她的积极向上和有档次。多年来，所有人通过她的朋友圈觉得她成功、值得信赖。所以每当她做什么项目，都有很多人跟着做。后来她干脆开了教别人发朋友圈的课程，卖到一两万，很多人买，短短几个月赚了小几百万。

听起来不可思议。在闺蜜联盟饭局切片视频上我们聊过这件事。当时她跟我聊了她的商业体系，我觉得她厉害在于有勇气、有胆量，想到就赶紧去做，所以短时间内产生较好的变现结果。但在我看来，她的整个商业链路没有壁垒。别人拿到她的课，自己就可以去卖，还要她干什么？

所以我给她一个建议：如果想让你这件事的壁垒变得更高，应该去做别人就算抄了你的课程，依然很不愿意花精力去做的事——打造后端的供应链。需要找到一些供应链商家，无论是打造朋友圈过程中出境的服装、首饰，还是家居物品。学了你的课程、会发朋友圈的人，也可以变成你的项目合伙人。你除了教他们课程，还可以把朋友圈里提到的产品打包给他们。他们学了课程后，可以在自己朋友圈通过卖货赚钱。你不仅可以教他们课程，还可以作为供应链提供产品。

说白了，这算是微商模式的一种变种：一边搞培训，一边做代理招商，还能一边卖货。但我的目的是让她的项目打通上下游后，变成一个更复杂的体系。只有变成复杂体系，别人再想去模仿或抄袭就不会那么容易了。总之，当你变成一个庞然大物时，就不是那么容易被复制了。

### 思路三：完成自洽，渗透动作

这世界上最难以被抄袭的就是创始人的价值观。就算其他人跟你喊一样的口号、做一样的定位，但只要他的发心和底层价值观跟你不一样，就会失之毫厘，差之千里。你会发现，在动作层面上，很多细节处理都会不一样。

例如，很多知识付费博主也来抄我的课程和内容。但我骨子里的核心价值观，存在我脑子里、心里、流淌在血液里的东西，他们很难抄袭。我是一个超级长期价值主义者，不着急在任何一个当下就要赚到所有的钱。而绝大部分知识付费博主的价值观，可能是想趁着风口，三年赚完一辈子的钱。

所以，我们的动作层面就会体现得非常不一样。他们会在直播间不断拉高大家预期、过度承诺或以逼单方式让大家冲动消费。而我几乎不在直播间进行强势销售，而是选择把大家带到私域，不断在朋友圈展现我们的价值观、给客户带来的收获、我们的做事风格，让大家不断理解我、喜欢我，最终再来为我付费。

当你的价值观渗透到每一个动作，每一个动作都能反映你的价值观时，别人就非常难模仿你。

再比如，同样是做情感咨询，我们和很多情感机构的价值观也不一样。在我做这个赛道之前做过调研，刷到过其他情感博主说情感赛道就是要激发别人的焦虑。他们靠打压客户、制造恐慌，让客户在混沌关系中不要清醒过来。因为他们认为一个女人如果脑子清醒了，就不需要你帮她解决问题了，就不会再有续费了。说白了，他们根本不想真的帮客户解决问题，而只是希望客户如何能不断续费。

但我的价值观不是这样。我们公司有一句非常大的slogan：**为客户做最大限度的考量**。我在培训所有辅导师时都会讲：如果你一个月能解决完的事情，不要拖到两个月，坚决不要为了让客户续费而不去快速解决客户的问题。

讲到这里，相信很多人已经明白：看似我也在做情感咨询，别人也在做，产品似乎非常同质化。但实际上，只要创始人的价值观不一样、发心不一样，做出来的每一个动作都是完全不一样的。

## 第四部分：如何将价值观植入用户脑海并渗透企业动作

依然有很多人可能会问：我要怎么才能把我的价值观植入到用户的脑海当中，同时又能渗透到企业的每一个动作里头呢？我的回答是：创始人要始终保持在一线。

企业的创始人和高管代表的就是企业的价值观和风向标。我们需要保持在一线，就像把手永远放在水面上，不断感受水的温度，什么时候高了、低了，都能第一时间感受到。我不停去接触我的产品、接触我的客户，就能迅速从客户反馈中感受到我们哪一个环节是否出现了变形。这也意味着我能够迅速对企业的政策和动作进行调整。

所以到目前为止，大家还是会看到我每周定时定点在台前直播。我可以这样说，就算以后我的企业做得非常大，已经不需要我再在台前拉流量了，我依然会选择用其他方式跟我的客户保持高度链接。比如不断跟闺蜜联盟的客户沟通聊天，或进行一些全球巡讲、开办线下讲座等，想尽一切办法一线接触到我的客户。

虽然我前面说过创始人要想办法抽离出来，但实际上一个负责任的老板，不能想着完完全全去做一个甩手掌柜。所以，如果你想不被抄袭、不去模仿，保有自己的核心竞争力，就要持续在一线，持续彰显你的价值观，持续影响企业员工，以及持续了解客户反馈和自我精进。

最近非常吻合这个逻辑的，就好像我的学长雷军雷总，他是一个非常好的表率。虽然在企业内部，我相信他们也有职业经理人和高管把控每个环节，但他依然每时每刻都在一线关注客户动态，从而改进他们自己的产品和动作。在这一点上，我非常佩服，同时也希望大家都可以朝着这个方向去努力。
