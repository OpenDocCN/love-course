# 恋爱军团课程：03：视觉效果穿搭技能整体+5分 🧥✨

在本节课中，我们将学习如何通过整理衣柜和挑选服装，有效提升个人视觉形象，从而在恋爱中更具吸引力。

前几天课程的作业是否坚持打卡了？我们与理想对象的距离正逐渐拉近，只有坚持才能看到一个焕然一新的自己。这让我想起一部黑色喜剧中的有趣桥段。

开着豪华跑车的男主趾高气扬地指点江山说，如果不把衣柜里的旧衣服都清理掉，新衣服是进不来的。周围的人听后醍醐灌顶，立刻回家收拾衣柜，将不要的衣服全部丢掉。不料几天后，其中一人惊讶地发现，男主身上穿的T恤竟和他前几天丢掉的那件一模一样。

虽然这是一个充满讽刺意味的桥段，但你是否也有这种不被刺激就不去处理旧衣服的习惯呢？

因此，请将不再让你心动的衣物整理并丢弃。或许你的衣柜已经满得塞不下一件新衣，但你是否发现，经常穿的衣服其实屈指可数。

每天早上你是否都会在衣柜前犹豫许久？为不知该穿什么出门而愁眉苦脸，感慨自己没有衣服穿。令人恼火的起床气和时间紧迫的慌乱，可能让你瞬间烦躁。为了一整天的好心情，是时候做出改变了。

打开你的衣柜，好好规划一下。仔细回想你平时的购衣习惯，再思考你期待未来以怎样帅气的形象在他人面前展现自己。迈出这一步，你与理想之间的距离就不再遥远。

你是否发现，当衣柜整整齐齐、一目了然时，你很容易灵光一现，搭配好出门的行头。所以整理衣服，你首先要学会以下几条诀窍。

以下是整理衣柜的核心原则：

*   **保留高品质、常穿且符合个人风格的衣服。** 至于那些既无价值又占空间的衣服，应尽快丢弃。
*   **衣服贵精不贵多。** 不要纠结于要丢弃的衣服是否是今年的明星爆款，不要纠结于当初是否为它省吃俭用过，也不要纠结于它是否承载着特殊的纪念意义。
*   **一件衣服，只要你穿着不好看、不适合，就没有理由让它继续占据你的衣柜。**

下面开始划重点，明确什么样的衣服应该扔掉。

**不合身的、破旧的、不符合自己风格的衣服都应丢弃。**

不合身的衣服大致可分为以下几类：

*   体型变化（变瘦或变胖）后不再合身的旧衣。
*   别人赠送或自己买错尺码的衣服。
*   因穿着洗涤次数过多，导致缩水或失去弹性、变得松松垮垮的衣服。

太大的衣服穿在身上显得邋遢，太小的衣服则显得滑稽，这类衣服应尽快清出衣柜。

再来找出破旧的衣服：

*   不小心刮破或蹭破的衣服。
*   因穿太久发黄、洗不干净的衣服。
*   带有无法清洗的油渍、血渍的衣服。
*   褶皱严重且无法熨烫平整的衣服。
*   穿着太久褪色、失去原有光泽的衣服。
*   纽扣、拉链或装饰物丢失损坏且无法修好的衣服。

最后是不符合自己风格的衣服。要判断衣服是否符合你的风格，你首先需要对自己有清晰的定位。你需要知道你是谁，想在什么场合、对谁留下何种印象、达到什么效果。基于这些，再来判断哪些衣服符合你的风格。

接下来，你还要根据个人偏好和外形气质来确定风格。不符合你风格的衣服，你其实很少会穿，自己很容易就能做出判断。

扔衣服时不要舍不得。不要因为对某件衣服怀有感情就让它霸占衣柜，不要因为某些衣服买时价格昂贵就留下来，不要因为觉得自己还有变瘦的可能就留下来，也不要因为衣服本身好看但不适合你而保留。

把它们都处理掉。如果你三年都没碰过这些衣服，那么你很可能不会再穿它了。你衣柜里的衣服，都应该是穿起来精神、让你心情舒畅的。

你或许会说可以把旧衣服留作居家服，但把旧衣服做成家居服最后可行的可能性几乎为零。如果你实在犹豫不决，可以换个思路：想想哪些衣服是你经常穿、并受到过他人由衷赞美的？哪些是你珍惜重视、必须保留的？

最简单的做法就是将它们先挑选出来，一件件试穿，观察镜中的自己穿着是否好看、是否会提升幸福感。某种程度上，整理衣服是一个与自己敞开心扉、深入交流的过程。你开始认真定义自己的风格，深入了解自己的审美偏好，并对未来的购买行为做出规划。

与已完成使命、尽完责任的衣服愉快地挥手作别，接下来你的生活将翻开崭新的篇章。老师这里也有一些购买衣服的小贴士。

以下是购衣核心原则：

1.  **剪裁 > 材质 > 款式**
2.  **有形是首要的，舒适是次要的**
3.  **避免大Logo**

上一节我们介绍了整理与购衣原则，本节中我们来看看男士常见的穿搭风格。

老师先给大家介绍几种男士穿搭风格：

*   **商务风格**：搭配讲究和谐、简单、大方、得体，能搭配出成熟优雅的穿着效果，给人儒雅有修养的感觉。商务风格还分**商务正装**（通常是西装、衬衫和皮鞋的组合）和**商务休闲装**（兼具商务装功能与休闲装随意性）。
*   **休闲风格**：这是一个难以准确定义但最空泛的着装风格。这种风格的搭配比较普遍，搭配起来也更加简单随意。
*   **运动风格**：将运动服装的自由舒适、专业功能性与时装的曲线修身、潮流时尚巧妙融合，某种意义上属于休闲男装的一种，但更讲究功能性。典型代表有卫衣、运动套装、运动鞋等。它不仅是户外运动着装，也是日常休闲舒适的选择，深受青年男性青睐。

那么，男装具体要怎么挑选呢？

首先需要确定适合自己的个人风格。男人着装既要显示魅力，也要表达个性。挑选适合个人性格的男装，能展现个人品味与风采。因此，挑选时可从个人性格出发。

如果尚未找到明确风格，可以选择一些必备百搭单品。单品之间随意搭配也要讲究得体。

以下是男士必备百搭单品示例：
```text
西服、牛仔裤、白衬衫、素色T恤、休闲西装外套
```

每个人的身材都不同，有高矮胖瘦之分。穿衣时要尽量注意，避免暴露自身缺点。

记得试穿。除非刻意要买大尺寸，否则**修身**是必须考量的点。加上不同品牌的剪裁和材质差异很大，会影响穿着舒适度，所以务必试穿。

以下是针对不同身材的穿衣建议：

*   **梨形（普遍身形）**：穿衣服无明显缺点。上身可选短款西装或风衣；下身选深色直筒裤，避免紧身裤或太松的阔腿裤，裤子不要提到肚脐眼以上。
*   **番茄型（身体圆润）**：适合中长款外套，能很好遮掩腹部；下身选择深色裤装和黑皮鞋；尽量避免穿格子衬衫。
*   **菜豆型（纤细苗条、个子矮小）**：上身可穿有印花或重复图案的上衣；下身保持简洁，并搭配有增高效果的运动鞋，以最大限度显高；还可以在配饰上花些心思。
*   **黄瓜型（傲人长腿和纤细长身）**：这是天生的衣服架子。不要浪费两条大长腿，谨记**少而精**的原则。可尝试难度系数高的格纹窄腿裤，搭配同色系单品和衬衫，最大限度突出腿长的优点。
