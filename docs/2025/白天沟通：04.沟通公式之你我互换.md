# 白天沟通：04.沟通公式之你我互换

沟通公式之，你我互换。第一步啊，把你换成我把你能不能早点回来，变成我希望你今天可以早一点回来，把你出去能不能跟我说一声，变成我都不知道你出门了，有点担心，把你应该多花点时间在家里变成我需要你多陪陪我们。

说你应该就好像你得这么做，你没得选，你不这么做，就是错。哎，说我需要代表你有选择，你可以满足，也可以不满足权利在你手上。当我们沟通当中，用你来做开头的时候，往往带有一种命令和指责的感觉。哎。

你给我怎么怎么做啊。哎，你应该怎么怎么做啊。哎，你犯了什么什么么做，就会给对方一种压力，一种被攻击的感觉哦，换成了我对方觉得唉没有那么强烈的被数落，被埋怨的感，也不会对你产生防御机制。

并且他也知道自己该怎么做了哦，我得早点回哦，我有事得提前跟家人说，哎，我得多陪陪老婆孩子。那么这种一板一样的沟通方式，就非常符合男性的思维模式。第二步，把我换成我们比如啊早点回来。

我们就可以一起去看电影。啊，我们两个一起带宝宝就没有那么辛苦啦。这一步呢是潜移默化的把你和我的对立局面变成了我们共同去创造的协同关系。在这样的语境之下呢，冲突会慢慢的消失，两人的关系会越来越紧密。
