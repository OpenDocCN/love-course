## 方式一：俏皮话开场

常见的无聊开场通常是“在吗？”、“在干嘛？”、“吃了吗？”这类没有意义的问句。这样开场虽不会让女生反感，但很无趣，无法激起她的聊天欲望。可能聊两句话题还没打开，女生就开始敷衍了。

如果想让女生一开始就被点燃聊天欲望，不妨用一些幽默风趣的俏皮话作为开场。所谓俏皮话，简单理解就是能让女生心情愉悦甚至笑出来的话。这些话能快速调动女生的情绪，让她期待与你继续聊下去。

举个例子，你开场的第一句话是：
```
“我，骗子，打钱。”
```
这句话就能让女生眼前一亮。她可能会思考如何回复，也可能会会心一笑，期待你的下一句话。总之，她投入了情绪，聊天就很容易进行下去。

但使用这个方法要注意把握好**尺度**，不能显得太不正经，或是直接调侃女生。这些做法在关系发展前期并不利。恰到好处的幽默，才能让你和女生的聊天保持在一个愉悦的氛围中。

## 方式二：表情包开场

表情包之所以流行，是因为在面对面聊天时，我们能借助肢体语言和面部表情来辅助沟通。线上聊天如果只通过文字，会显得特别生硬。而表情包的存在，正是为了方便线上聊天来表达更丰富的情感。

善于利用表情包作为聊天开场，也能让聊天迅速进入愉快氛围。而且，表情包开场还有个好处：能有效**削弱目的性**，让女生不会立刻竖起防备。

比如，刚认识一个女生，聊天开场就可以发“打招呼”或“可爱”的表情包。如果你们已经聊过几次，那下次开场就可以发一些带搞笑属性的表情包，例如蘑菇头或者熊猫头系列。女生如果比较有幽默感，大概率也会回复表情包。你也可以从中分析出女生的基本属性，比如性格、审美等。

这里要提个醒：表情包的使用尺度要得当，不能是猥琐、恶心或内涵的表情包。聊天过程中也不用一直用表情包回复，沉迷跟女生斗图，很快就会让聊天变得乏味。

## 方式三：悬念式开场

人都有好奇心，对未知事物充满探索欲。所以，开场设置悬念就能有效引起女生的好奇，让她想跟你继续聊下去。

这个开场方式需要你跟女生已经处于朋友阶段，且对女生有一定了解才便于使用。所谓第三方话题，就是既可以和你们两个人有联系，又可以独立于你们之外的内容，比如工作、个人兴趣爱好、星座测试、影视动漫等等。
