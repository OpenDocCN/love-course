# 22破局最终章   道术合一_ev

好，兄弟们啊，大家应该发现的一个点啊，就是我在课程里面用到了大量倒数结合的手法，哎，几乎每讲一节课，都喜欢从倒数的两个层面去讲解。这是为什么呢？我估计有很多兄弟啊还是不太了解这个理念。

但是这个东西呢确实又非常的重要。所以我今天专门拿出一节课来啊，专门讲一下这个问题。正所谓道术结合哎，相得益彰，倒术相理，各见其害，道是指自然存在发展的规律啊，它是客观存在的，不以人的意志为转移。

顺则生逆则亡，而术啊就是具体的方法技巧，也就是我经常说的啊，万事万物都有它运行的规律和原理。哎，如果你想做成一件事，必须要遵循它背后的规律。如果没有遵循背后的规律，哎，不管你怎么努力啊，都是枉费心机。

因为方向错了。而好的技巧可以帮助你更快的做成那件事，可以更大程度的节省你的时间，经历，同时增加成功率，这里有一个很经典的故事啊，叫做庖丁解牛。大致的意思就是说。😊，战国时期啊有一个叫庖丁的人，哎。

这个厨师呢宰牛很厉害，哎，手起刀落之间，这个牛就被肢解了。在一旁观看了梁惠王啊就非常惊讶。哎，就问这个庖丁说你怎么这么厉害呀？庖丁就说其实我一开始的时候，也觉得非常的困难。根本不知道从哪里下手。

但是在通过不断的练习之后啊，刀工越来越好，对牛的身体结构啊更是了如指掌。啊，现在我就算闭着眼睛，只需要用手摸一下这个牛的身体就可以判断出来骨头和肉研睛的地方在哪里。所以我能够每一刀都做到很精准。

那么从这个故事中啊，就完美的体现了倒数合一的思想。哎，首先从道德层面来说啊，庖丁遵循的规律，遵循了牛的身体结构，所以他知道从哪里下手，怎么下手最快。那么我们学这个恋爱学其实也是一样的道理啊。

你的脑子里必须要得整个流程和光架非常的熟悉。哎，因为只有这样，你才能时刻清楚自己走到了哪一步，哪里出错了，可以及时改正。这也就是为什么我让你们发聊天截图的时候，需要发三张截图以上。

我是在判断你们到了哪一步。😊，所以大家记住了啊，第一步一定不是学什么话术，学怎么聊，而是学思维，学背后的原理，学整个的流程框架。以道欲术术善可求。有术无道指如术。这就是我经常说的，不要去盲目的背话术。

话术是根据人的思维创造出来的，只要懂了原理，有思维，你是完全有能力创造出属于自己的话术。第二点，庖林这个厨师啊解牛能够这么厉害，是因为他经过不断的练习，哎，技术已经非常熟练了。

并且掌握了非常牛的解流技巧，所以道术结合，缺一不可。那么我们在学习这个课程的时候也是一样的啊，原理告诉你了，具体的方法和技巧也给你了。那么接下来不断的去练习去实践就可以了。

这个和那些武侠小说里面演的是同样的道理。哎，顶级高手的武功啊，都是分为两个层次。不光有具体的招式，还有强大的内功心法。比如莲花楼里面的李莲花啊，不光有象夷太监，还有扬州漫这样的顶级内功心法。

但反过来啊拿我们的。恋爱学来说也是同样的道理。如果你想成为一个高手，你不光要知道其背后的人性规律和系统的流程框架，同时也需要掌握高超的技巧和强大的心态。你技术再牛，心态不好还是等于零。

随便被哪个女生啊骗一下就破房了，整天以泪洗面，患得患失，那结局可想而知啊，对吧？好，最后希望大家啊能够好好理解我今天说的话，以到处合一的这种思想，我们下节课见。😡。
