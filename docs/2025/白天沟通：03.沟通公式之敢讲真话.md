# 白天沟通：03.沟通公式之敢讲真话

沟通公事之敢讲真话，有很多人啊觉得婚姻过着过着就变成了男的呢觉得老婆一点小事就无理取闹。女的呢觉得老公根本不在乎她，为什么？因为你们存在着争宠式的沟通，举个例子啊。

比如你们经常因为老公看黄片这个事情大吵大闹。那么老公看黄片，就已经变成了你们的一个心结，你一点就炸，这个背后的底层逻辑到底是什么？来，我们具体分析一下，来再举个例子啊，你老公打游戏不理你，你会不会生气？

会如果生气的原因跟看黄片一样，那么说明你是因为害怕被冷落，害怕被忽略。来再来你俩吵架了，老公说咱俩别过了，你生不生气，如果生气且跟看黄片一样，那么你是因为害怕被老公抛弃。你不想让她在房间抽烟。

她偏偏还要抽，你生不生气，跟看小黄。片一样吗？是的话，那么你生气的点在于他不听你的话。所以刚才这些情绪，这些被冷落，被忽略，被抛弃，他不听话。那么跟看小黄片有关吗？没有关系，对吧？

你只是像一个小孩一样的抓住某一个事情，通过作和闹的方式去表达你的需求，但是他不是你肚子里面的蛔虫，他并不知道你的生层需求，他只知道啊，你看这个娘们这点小事又要闹了，闹了多少次了，对吧？你看闹的次数多了。

他真的会看到你都害怕。所以姐妹们啊一定要避免争宠式的沟通，大胆的直接的表达自己的需求，敢于说出口，敢于争取自己的利益，敢于争取自己利益的人，大家才不会怠慢他。畏畏缩缩的人只会让别人觉得哎，你好欺负。

好糊弄。所以啊想让老公多给家用，你就说哎想穿更多漂亮的衣服给你看，老公帮我买单，想让老公多陪陪你，你就说啊你在家里我就好安心，多陪陪我好不好？想让老公多做前戏，你就说哎呀，好喜欢你弄这里，轻轻的揉她。
