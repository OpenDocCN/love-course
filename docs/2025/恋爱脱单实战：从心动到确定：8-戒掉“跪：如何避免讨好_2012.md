# 恋爱脱单实战课：8：戒掉“跪舔”：如何避免讨好，赢得对方的尊重和喜欢 🚫

在本节课中，我们将要学习如何识别并戒除“跪舔”行为。我们将探讨“跪舔”的本质，理解其背后不平等的心理预设，并学习通过提升自尊水平来建立平等、健康的互动关系，从而真正赢得对方的尊重和喜欢。

## 什么是“跪舔”？

喜欢一个人时，我们本能地会付出时间、金钱或提供帮助。这个行为的底层动机是：通过付出向对方证明“我值得你的喜欢”。例如，我帮你做PPT是为了展示我的技能，让你看到我能让你的生活更轻松。

**问题在于，并非所有付出都能换来对方的认可和喜欢。** 有时，对方甚至可以在不喜欢我们的情况下，心安理得地接受我们的付出。面对这种情况，有些人能及时止损，而停不下来的人，心理上会发生扭曲。

上一节我们介绍了付出的本能动机，本节中我们来看看当付出失衡时会发生什么。

以下是“跪舔”循环中的典型心理变化：
1.  **忍受不对等**：总是单方面付出，对方没有回报或真心感谢，导致自我价值感被忽视。
2.  **迎合与自我压抑**：为了取悦对方，硬生生压住内心的不愿意，并自我合理化这种行为。
3.  **陷入无解纠结**：明知关系不对等，对方在忽视自己的付出，却选择忍耐并寄希望于未来，于是不断加码付出。

这个 **“付出 -> 回应不足 -> 自我合理化 -> 追加付出”** 的循环过程，就是“跪舔”。它源于内心默认了一个前提：追求是一种不平等的行为，追求方理应姿态更低。

## 为何“跪舔”无法赢得尊重？

当我们默认追求关系不平等时，会不自觉放低自尊，甚至将“死缠烂打”、“自我伤害”等低自尊行为误认为是追求方法的一部分，试图用“我为你连自尊都能抛开”来打动对方。

然而，这种行为传递的潜台词是：**“我自己也不尊重我自己的付出，我的底线可以一退再退。”** 换位思考，如果你是对方，也会被引导认为：这个人不需要我给予太多关注和尊重。

**人与人之间的关系模式，往往是我们自己教会对方的。** 如果我们不尊重自己的付出，就等于亲手教会了对方如何不尊重我们。

理解了“跪舔”源于低自尊和不平等预设后，我们自然会问：那该如何纠正？一种常见的错误反应是走向另一个极端——变得高高在上，通过踩低对方来自我防御。但这依然是一种不平等，会将互动变成充满攻击与防御的“战争”，消耗双方心力。

**因此，停止“跪舔”的关键不是踩低对方，而是提升自己的自尊水平，以达到平等的权利关系。**

## 如何有效提升自尊水平？

一个合格的自尊水平，其核心参考标准是：**权利关系平等**。所有行为应基于“我愿意”的平等原则，而非“委曲求全”。

例如，吃饭是否AA制不是关键。关键在于：我请你吃饭，是出于我自愿且不会感到在取悦对方，那就不算“跪舔”；反之，如果请客是为了榨取自己以换取对方施舍，而AA能让我感到自洽，那我就应该选择AA。

**判断标准是：行为背后是否透露出一种“上下之分”的不平等感。** 这种不平等预设不仅存在于情感关系，也存在于职场等任何关系。平等的本质是双方都拥有或都没有“选择权”。

在恋爱中，**选择权等于潜在的恋爱机会**，即你拥有结识异性的场景和可能性。这要求我们具备两点：自身的吸引力，以及向外扩展的社交场景。吸引力是本课程的核心，这里我们重点探讨社交场景。

### 向外求：构建选择权

我发自内心地建议，大家应该花精力维持一个能认识新朋友的社交场景。这能让你持续接触外界，获得“潜在恋爱机会”的信号。即使只是一个异性多看你一眼的瞬间，这种信号也能提升你的自尊水平，让你感到“我有选择权”。

然而，很多人会陷入盲目寻找社交场景的误区。关键在于筛选出适合自己的场景。

以下是两个核心筛选条件：
1.  **成本足够低**：你在该场景中感到舒适，无需勉强性格，也不会造成时间、精力或金钱上的沉重负担。不可持续的高成本场景应被放弃。
2.  **高活跃度**：社交场景有生命周期。死守一个没有新面孔、活跃度低的“死水”圈子，是在浪费脱单的宝贵时间和精力。应迭代掉此类场景，寻找能带来社交激情和稳定新机会的高活跃度环境。

向外寻求潜在机会，是通过外界反馈来确认自尊，这与流行的“内求”论不同，但效果显著。当然，更好的方式是内外兼修。

### 向内求：自我赋予信念感

所谓“向内求”，即**自我赋予信念感**，也就是“相信相信的力量”。你相信某件事会发生，就会不自觉地朝那个方向努力，从而增加其成真的可能性，这在心理学上称为 **“自证预言”**。

应用到“戒掉跪舔”上，我们应相信：**即便失去眼前这个人，我也能过得很好。** 当你深信这一点，你的行为就不会被对方牵制，从而获得强烈的自我掌控感。这种变化很具体，比如发消息后不再紧盯手机等回复，而是可以安心做自己的事。

但问题来了：如何发自内心地相信这一点？这就是为什么我们要先讲“选择权”。没有外界信号确认，信念容易变成空洞的鸡汤。而且，信念容易因敏感解读细节而动摇，例如：“消息回慢了，是我不够好”、“对方朋友圈是不是在暗示不想理我”。一旦产生“我是不是要改”的念头，就意味着信念不稳，开始滑向不平等。

一个稳固的信念是非常强势的。我们可以通过三件简单有效的小事来自我赋予信念感。

以下是自我赋予信念感的三步法：
1.  **给自己明确的暗示**：像上台前给自己打气一样，经常用动作和语气进行积极自我暗示（如“我能行”），注意暗示要合理，不要夸张。
2.  **习惯正向解读**：不要总盯着否定自己的线索，而是主动寻找并习惯正向解读。例如，对方婉拒约会时，解读为“TA详细说明了原因，是在照顾我的感受”，而非“TA讨厌我”。
3.  **重复与记录**：重复强化信念。更重要的是**用笔记录**下每天让自己满意的小事，或过去十件证明自己“很厉害”的事。书写能极大地巩固思维的土壤。

此外，这里补充一个**应急方案**。当信念快要崩塌时，可以问自己两个问题：
*   “过去我也失去过重要的东西，我现在不是也没事了吗？”（从**过去**维度确认自己的应对能力）
*   “如果我连这关都过不去，以后还怎么面对大风大浪？”（从**未来**维度将其视为训练机会）

这两个问题同时从过去和未来提供确定感和掌控感，能有效扶住摇摇欲坠的信念。

可能有同学会问：过度关注自身感受，会不会忽略对方的真实想法？这正是我们前几节课（判断对方兴趣、如何反应）的价值所在。**方法是保证行为不出错的底线**。在正确的方法框架下，配合建立平等心态，才更容易做到不绑定他人、尊重自己。

## 总结

本节课中我们一起学习了如何戒掉“跪舔”，建立平等健康的互动关系。

首先，我们明确了“跪舔”的定义。它源于喜欢一个人时的本能付出，但当付出与回报长期不对等且无法停止时，就会陷入 **“付出 -> 得不到回应 -> 自我安慰 -> 追加付出”** 的恶性循环。这背后的核心是内心默认了“追求即不平等”的错误预设，导致主动降低自尊。

接着，我们探讨了如何有效提升自尊水平。关键在于建立平等关系，而非走向高傲的另一个极端。我们通过向内、向外两个途径来实现：
1.  **向外求，构建选择权**：选择权源于潜在的恋爱机会，需要靠合适的社交场景支持。筛选社交场景需遵循 **“低成本”** 与 **“高活跃度”** 两大原则，以确保可持续性和有效性。
2.  **向内求，自我赋予信念感**：核心信念是 **“即使失去眼前这个人，我也能过得很好”** 。通过 **“积极自我暗示”、“习惯正向解读”** 以及 **“重复与书面记录”** 三件小事来稳固这一信念，并在危机时用两个预设问题作为应急方案。

总而言之，戒掉“跪舔”意味着建立平等的关系。很多人“跪舔”正是因为压低自尊导致了关系不平等。因此，关键是在追求过程中，通过内外兼修的方式，**长久而稳定地提升自己的自尊水平**。一个懂得尊重自己的人，才更容易赢得他人的尊重和喜欢。
