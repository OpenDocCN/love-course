# 白天沟通：02.沟通公式之“不”和“要”

沟通公式之，你为什么不和你为什么要来举个例子啊，我们模拟一段对话。我问你，你为什么不练习我们的意纸私密课程呢？你是不是会反驳，为什么要练呢？那我说练了，你可以变得更紧致，更水润，让你的味道能心会动。

爱爱体验感会更好哦，嗯，你会觉得还好吧，这个不是很重要。那你难道不觉得爱爱体验和更好很重要吗？对吧还好吧，我觉得练起来很麻烦，所以你是怕麻烦，你懒得去做，对吗？你会继续解释说，没有吧。

我只觉得没有必要而已。看到没有？当我问你为什么不的时候，你会拼命的解释，你为什么不，因为你不想被我改变，这就类似于你跟你老公说，你为什么不戒烟，你为什么不跟他断掉，他也不想被你改变。再举个例子啊，来。

我问你，你有没有尝试过去想要过去做微道训练，有没有想过，如果1到10分想去做训练的念头有多少？那你会说嗯大概五六分吧，那我会说哎比我想象中更高哦，为什么会这么想去呢？对吧？

那你会说嗯我想解决我的漏尿问题啊。哎，那我说那那个还好吧，也不是会经常露呀，嗯，感觉有点松了哦，我想变得紧一点嘛。哎，那这个其实也没什么大不了的呀，不是很正常吗？哎，你不懂紧一点爱爱才有感觉啊？

看到没有？当我问你为什么要的时候，当我在质疑你的时候，你会积极的去捍卫你的权益，拼命的强调要的好处，再回过头来，当你说啊微道训练的念头是5到10分。当你说你想要做微道训练的念头是5到6分。

那我问哎5到6分，那为什么不是8到9分呢？微道健康不是很重要吗？那你会怎么说呀？你会说哎呀，没有时间去做呀，得花钱呀，哎呀，去做很麻烦呀，紧直也没有那么重要啊，没那么迫切呀。你看是不是会绕回去了。

所以啊还是那句话没有人想要被你改变，当你的观点被攻击的时候，你会积极的去。捍卫去保护自己的想法，自己的观点，对不对？你看刚才其实你都并没有想去做微道训练，可是当我去问你做的理由是什么。

你就会去捍卫那个理由去增强那个理由？所以啊这就是选择权？你攻击他，你为什么不的时候，他没有选择权？你为什么不戒烟？你为什么不戒酒，你为什么不跟2加一断掉，他没有选择权，那么有选择权是什么样子？

有选择权是哎，好奇怪，你既然还想要有戒烟的念头。哎，原来你也是想跟他断掉的那为什么会这样想呢？是不是有了这样良好的引导式开头，那你们的沟通就会朝着你想要的结果去发展？
