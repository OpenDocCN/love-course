# 肢体接触教程：第11天：握手

在本节课中，我们将学习如何通过“握手”这一简单的肢体接触方式，开启与他人的互动。我们将从基础概念开始，逐步讲解筛选意愿、发起互动以及推进关系的完整流程。

## 🎯 核心概念：意愿筛选

上一节我们介绍了课程目标，本节中我们来看看执行任务前的核心准备：**意愿筛选**。其核心是判断对方是否有继续互动的意愿，公式可以表达为：

**互动意愿 = 对方积极回应 - 对方消极回避**

如果对方没有表现出继续互动的意愿，那么最合适的做法是礼貌结束，让对方离开。我们需要主动进行这种筛选。

以下是执行“握手”任务时，意愿筛选的几个关键观察点：

*   **积极回应**：对方停下脚步、微笑、进行语言回应。
*   **消极回避**：对方快速走开、回避眼神接触、给出简短否定回答。
*   **行动决策**：若判断为“无意愿”，则主动结束互动。

## 🤝 任务执行：发起握手互动

了解了筛选意愿的重要性后，我们进入实际操作环节。本节将演示如何从打招呼开始，自然过渡到握手，并尝试推进关系。

以下是“握手”任务的一个完整对话示例：

1.  **开场与破冰**
    *   发起方：Hi，你们俩是情侣吗？我挺喜欢你的。
    *   发起方：你好，过来一下。你好可爱呀。

2.  **基本信息交换与兴趣表达**
    *   发起方：你多大？等一下你大学生吗？
    *   发起方：加个微信，下次一起约。你在上海上学吗？我们加个微信，下期一起喝东西，好吗？

3.  **关系试探与确认**
    *   发起方：这个是你女同学吗？
    *   发起方：你是南方人吧？你那那你应该是喜欢男生吧，不喜欢女生。
    *   发起方：那就好，如果我是男生的话…你是男生的话，你会喜欢男生。对的。

4.  **表达欣赏与结束**
    *   发起方：刚刚在你那边看到你，你怎么这么可爱呀？拜拜。

在整个过程中，通过语言交流判断对方意愿，并逐步提出加微信、约下次见面等请求，尝试将简单的握手接触升级为更深入的社交连接。

## 📝 总结与回顾

本节课中我们一起学习了以“握手”为起点的互动方法。我们首先强调了**意愿筛选**是互动的前提，明确了如何判断并处理对方的无意愿情况。接着，我们通过一个完整的对话示例，演示了如何从打招呼、赞美、询问信息，到尝试建立联系方式并推进关系的全过程。记住，保持自然、尊重对方反馈是成功的关键。
