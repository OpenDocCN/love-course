## 身体状态与能量水平的关系
最近一段时间，我感染了病毒，身体状态不佳。这直接影响了我的能量水平和生活状态。

病毒非常顽固，需要长时间的调理、服药和充分休息才能逐渐排出体外。症状包括肺部不适、嗓子不舒服，甚至影响睡眠质量，经常在半夜自动醒来。

这种身体的不适直接导致了运动无法进行，健身和训练计划也被迫中断，持续了将近一个月。能量水平因此不高。

**核心概念**：当身体运行不顺畅，存在炎症或病毒时，个人的整体状态会下滑。自信也很难达到巅峰状态。当下的心态虽然不算特别好，但维持在一个基准线上，会有波动，而不会极度低落。但当身体崩溃时，状态会急剧下降。

身体不适的一个明显影响是反应变慢，头脑转不快，说话的逻辑和语言组织能力也与精力充沛时不同。

## 健康、能量与“百病消”
上一节我们讨论了身体状态对整体能量的影响，本节中我们来看看高能量状态对健康的反哺作用。

有学员提到被过敏性鼻炎折磨。过敏性鼻炎往往与阳气不足、身体湿气太重有关。以广西为例，当地湿度高，很多人体内湿气重，患有过敏性鼻炎。我以前也有，持续了很多年。

但现在没有了。这不是通过专门治疗治好的，而是身体变好、气血充足后自然消失的。这印证了一个观点：当你的能量达到高水平，气血充足时，许多病症会自然消退。

**核心公式**：
```
高能量状态 + 气血充足 → 百病消
```

所以，提升自身的能量水平和身体素质是根本。

## 大学时期的第一个女朋友
以下是关于我大学时期第一个女朋友的故事背景和经过。

**背景介绍**：
1.  我就读于天津外国语大学日语专业。
2.  我当时对这个学校和专业非常不满意，认为自己的实力远超于此。
3.  当时对日本文化几乎没有了解，仅有来自抗日影视剧和《灌篮高手》等动漫的模糊印象，对专业前景和就业也没有认真考虑。

**相识与相处**：
1.  班上有一个女孩，我最初并不觉得她特别好看。
2.  我们很聊得来，但我并没有太把她当回事。
3.  当时已经有其他男生在追求她，方式甚至有些“舔狗”的意味（例如，因为她手机欠费联系不上，就主动给她充话费），我还会拿这事调侃她。

**关系的转变**：
1.  大概在大一下学期或大二上学期，我们经常一起出去玩。
2.  一天晚上，我们在天津五大道散步，聊着聊着，不知怎么就牵上了手，然后就在一起了。
3.  当时的关系很单纯，就是谈恋爱。

**关系的裂痕**：
1.  在一起后没多久，我发现她对我的反应开始变得冷淡。
2.  我们日语专业有“语伴”制度，即中国学生和日本留学生结对练习语言。
3.  后来我发现，她的语伴是一个日本男生。对方身高不错，形象OK，也会穿搭（当时日本经济好，留学生普遍条件较好）。
4.  她回我信息越来越慢，我后来从她室友那里听说，她向那个日本男生表白了。

## 事件背后的分析与反思
上一节我们回顾了这段关系的经过，本节中我们来看看其背后的深层原因和我的反思。

这件事当时对我打击巨大，让我崩溃。在当时的认知里，中国人跟日本人在一起是难以接受的，感觉是极大的耻辱。

但现在回过头客观分析，原因其实很清晰：

1.  **经济因素**：当时（21世纪初）日本的经济远超中国，在很多中国人眼里，日本代表着更好的生活和发展机会。天津有很多日企（如天津丰田），日语专业的设立一定程度上就是为了向这些企业输送人才。很多女同学的目标就是去日本发展或定居。
2.  **地域与社会认知**：我是广西人，在部分天津人眼中，属于“偏远地区”。她父亲接电话时那句“还有广西同学”的嘀咕，也透露出一丝地域性的认知差异。她是天津郊县人，对日本的经济状况有更直接的了解。
3.  **核心规律**：**女人是经济动物**。这在两性关系中是一个底层逻辑。在当时中日经济差距悬殊的背景下，一个形象、经济条件更好的日本男生出现，对她产生吸引力，从她的价值判断来看，是符合其经济动物本性的。

**我的反应与后续**：
1.  我无法接受伴侣的怠慢，提出了分手。我坚信很多兄弟也是“宁死不舔”的。
2.  分手后大概两个月，她试图挽回。一次晚自习后，她主动挽我的手。
3.  我的反应是必然甩开。因为我们不是渣男，我们有契约精神，既然关系已经因背叛结束，就不会回头。

这件事让我低落了很久，是一种“雄竞失败”后的挫败感。

**给兄弟们的启示**：
1.  遭遇类似事情时必须保持理智。要明白这只是一次暂时的、基于当时条件的价值判断偏差，不代表你个人失败。
2.  后来那个日本男生并没有接受她，她又回头找我，也印证了这一点。

## 从经历中提炼的普适规律
通过我的这段经历，我们可以总结出一些在两性关系中普遍适用的规律。

**规律一：物质基础的重要性**
网上有一个案例：一个男人起早贪黑十年，创下几千万身家，以为这样就能获得爱情。这引出一个问题：有钱就有用吗？

答案是：**如果你没有正确的两性认知，采取错误的策略，那么再有钱也可能没用**。但是，**当你有了正确认知并采取正确策略后，钱是有用的，而且非常重要**。绝对不能否定物质在两性关系中的作用。

**核心结论**：
```
正确两性认知 + 正确策略 + 物质基础 → 有效
错误认知/策略 + 物质基础 → 可能无效
缺乏物质基础 → 处境艰难
```

因此，我希望所有人都要去拼事业。男人要把事业放在首位，女人除了生育价值，也要努力创造其他社会价值。

**规律二：择偶红利期**
当时我还有一些关系很好、无话不谈的异性朋友（非恋人）。但一毕业，她们就几乎全部消失了，联系不上。后来我才明白，她们是结婚去了。

在以前相对单纯的环境里，很多女性在决定结婚时，会主动切断与其他异性的密切联系，专注于经营婚姻和家庭。这是一种比较聪明的做法：在择偶红利期（最佳择偶年龄段）做出选择，而不是“骑驴找马”。

**定义**：女性的**择偶红利期**大致在**18岁到28岁**。在这段时间里，她们的选择权最多。

然而，现在的社会氛围被各种思潮洗脑，人性中的贪婪被放大。“骑驴找马”成为很多人的策略，看似聪明，但从长远看，后果往往不好。90后这一批人中，很多因此陷入困境的例子已经逐渐显现。

**规律三：建立选择权是核心**
目前婚恋情感市场中，很多方法（如打扮形象、修炼情商、学习话术）都不是最核心的。

**最核心的是：顺应自然规律，建立自己的选择权。**

*   **对于男人**：男人是为生存设计的，智商和体力续航能力普遍更强。男人的选择权核心是**提升生存能力**，即**事业和健康**。事业是男人的天。你的外形、身高、家庭是“初始设定”，但事业和成就是可以后天努力获得的“装备”。
*   **对于女人**：女人除了生育价值，还有一个重要功能是**照顾家庭**。在高层次男性的择偶中，女性作为家庭后盾的能力极为被重视。女人的选择权，首先是让生育价值得到更好展现（身材、外形），但这并非核心。更重要的是**情商、情绪稳定能力，以及对男人的真诚和忠诚**。当前一些思潮鼓吹“骑驴找马”、忽视忠诚，这是错误的。长远来看，社会会正本清源。

**关于我那位前女友的后续**：她后来嫁了一个在日本的中国人，但从她避谈老公的情况看，过得并不算太好。这或许也是一种印证。

## 当前环境下的策略：快速筛选
基于以上规律和当前的社会现状，我为大家提炼出一个核心策略。

当前社会，许多女性（和部分男性）被各种错误思潮严重洗脑，能保持清醒认知、建立正确潜意识的人不多。

因此，对于男性而言，在当前中国婚恋市场下的最佳策略是：**快速筛选**。

**为什么是快速筛选？**
1.  你不是救世主，不需要去“拯救”被洗脑的人。
2.  你的底气来自于你的事业和未来前景（即你的选择权）。
3.  当你确信自己未来会越来越好时，你要选择的是一位能与你同行、不拖你后腿的伴侣，而不是来者不拒。

**快速筛选的心态**：
你需要建立这样的认知：**我是筛选者**。我不是渣男/渣女，如果和我在一起，我会让对方幸福。如果你不答应，那是你没这个福分。
这种笃定源于对自我价值的极度自信。你不需要学习复杂的话术或技巧，核心在于**对自己价值的笃信**，而这又建立在**高能量生活和状态**之上。

一个每天熬夜、习惯不良、无所事事的人，没有资格进行筛选。而一个正经做事、努力上进、有良好生活习惯的人，自然具备筛选的底气。

**对女性的建议**：
最聪明的女性策略，是**帮助自己的男人成就事业**，或者说，**让男人成为自己独一无二的英雄**。当男人后方稳定、没有后顾之忧时，他会有极强的动力去实现自我，冲击事业。有智慧的女人不会“骑驴找马”，也不会强行要求男女在所有家务上平均分工（尽管男人可以主动做），而是理解双方分工不同。

**关于“搭讪”**：

**不要抱怨，从自身做起**：
不要仇视或抱怨现状。整个社会都在被洗脑，不仅女人，很多男人也被洗得很怂。男人靠不住，女人才被迫“独立”。我们要做的是从自己做起，反抗错误洗脑，坚持正确的规律行事。
