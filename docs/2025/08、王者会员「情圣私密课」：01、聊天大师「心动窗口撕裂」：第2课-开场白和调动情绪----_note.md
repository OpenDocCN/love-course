# 王者会员「情圣私密课」：01、聊天大师「心动窗口撕裂」：第2课 开场白和调动情绪

在本节课中，我们将要学习聊天流程中的两个关键步骤：如何设计有效的开场白，以及如何初步调动对方的情绪。掌握这两点，是开启一段愉快对话的基础。

## 什么是正确的开场白

上一节我们介绍了课程的整体框架，本节中我们来看看聊天的第一步：开场白。开场白是你说的第一句话，或是前两三句话，其作用是开启一个聊天话题。

开场白可以是一个表情，也可以是基于对方朋友圈或照片墙内容的一句话，甚至简单的“你好”也属于开场白。开场白的目的仅仅是表明你希望开始互动，并测试对方是否愿意与你聊天。如果对方积极回应你的话题，就证明她有聊天的欲望。

## 常用的开场白类型

以下是几种比较常用且有效的开场白方式：

1.  **表情开场**：发送两个不同的、有趣的表情。例如：😊😉。只发一个表情可能显得过于普通。
2.  **固定话术开场**：使用一句预设的、带有趣味性的话。例如：“活捉一只小可爱。”无论对方是何种类型的女生，这句话通常都能引发回应。如果她否认，例如说“我不是小可爱”，这恰恰为你提供了继续对话的切入点。
3.  **兴趣点开场**：根据对方朋友圈或照片墙展示的兴趣爱好开启话题。**公式**为：`“你也喜欢[对方的兴趣点]吗？”` 或 `“我觉得你也挺喜欢[对方的兴趣点]的。”`
    *   **示例代码（基于朋友圈内容）**：
        ```python
        if 朋友圈有“吃小龙虾”照片：
            开场白 = “你也喜欢小龙虾吗？”
        elif 朋友圈有“旅行”照片：
            开场白 = “我觉得你也挺喜欢旅行的。”
        ```

## 如何调动情绪

开场白完成后，就进入了聊天的第二步：调动对方的情绪。关于调动情绪的具体方法，我们将在后续课程（第6、7课）中详细讲解。这里我们先理解其核心概念。

通俗地讲，调动情绪就像“情感过山车”，目的是让对方的情绪产生起伏和波动，从而对对话产生兴趣和投入感，无法轻易结束聊天。

## 调动情绪示例

以下是一个结合了上述“固定话术开场”来调动情绪的连贯示例：

*   **男生**：活捉一只小可爱。
*   **女生**：哈哈。😊
*   **男生**：交出爽歪歪就放你走。（*制造一个小挑战或玩笑*）
*   **女生**：就不交。
*   **男生**：不交的话，抓回去给我当压寨丫鬟。（*将玩笑升级，引入角色扮演*）

如果女生继续接话，例如说“不当丫鬟，要当女主人”，则说明她对你有好感，并且情绪已经被调动起来。此时，可以顺势延续角色扮演，例如讨论“当压寨夫人需要什么能力”等虚构的趣味挑战，让对话更加生动有趣。
