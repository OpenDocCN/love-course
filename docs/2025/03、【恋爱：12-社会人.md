# 高情商恋爱学堂：03：社会人情味：一开口就征服人心

在本节课中，我们将要学习如何通过提升社交技巧和调整心态，在与人交往时，尤其是与不熟悉的人交流时，能够从容不迫、落落大方，从而建立良好印象并征服人心。

前面的课程我们围绕自我修养展开。接下来的课程将聚焦于大家特别关心的问题：情商。你是否遇到过以下情况？在路上或单位遇到上级或不熟悉的同事时，会纠结是否要打招呼、如何打招呼以及说些什么。如果对方拉着你不停说话，你也不知如何接话，甚至想如果对方没发现你，是否要绕道走以避免尴尬。

其实，你和熟悉的朋友相处愉快，但与不熟悉的人交往时常发挥失常。你可能认为这是社交恐惧。在生活中，你忽然不知如何开口，不知如何与陌生人交谈。这就像你不爱吃姜，但不能说你不爱吃所有菜。不擅长与特定人群打交道，不代表你情商低。

然而，女生通常欣赏落落大方的男生，希望自己的男友在人群中能侃侃而谈。在社交圈和职场上，这样的人也更容易获得认可。如果你遇到不熟悉的人又不想聊天，如何展现落落大方呢？

以下是具体方法。

首先，别怕对视。看着对方的同时，面带微笑并轻微点头。这个动作的含义是：“我看到你了，很高兴见到你，但暂时不想深聊。”点头微笑的打招呼方式全球通用，适用于任何人。

人与人的沟通包括语言沟通和非语言沟通。眼神、姿态、动作和表情都能传达信息，且往往更亲切。

不过，需要补充的是，那些让你感到不舒服的交谈对象，可能会引发你的认知偏差。认知与情绪存在关联。你对一个人的看法决定了你的情绪。例如，你认为一个女孩话多、总喜欢拉着人说个不停，这是你的认知。这种印象会让你自然对她产生抵触和反感情绪。

如果你尝试调整认知，会有什么变化？你可以这样想：她总喜欢拉着我说个不停，可能是因为她信任我，或至少认为我是个不错的聆听者，是个受欢迎的人。被人欣赏和信任是好事。她能欣赏我，说明我们有相似之处。这样一来，她似乎就没那么讨厌了。

老师举这个例子，主要是告诉大家，转换看问题的角度，此人在你心中的形象就会改变。这是一种自我修炼和心态构建。

很多学员问我：“我不会聊天，想学聊天；我不会约会，想学设计约会。为什么一开始要让我学心态？”无论原因如何，作为男人，情商代表一切外在表现，而心态是提高情商的基础。心态构建好，情绪稳定后，你甚至会产生一种期待。就像刚才例子中的女孩，说不定你会期待她找你聊天。

许多学员遇到的问题具有普遍性。长期单身的学员中，80%认为自己没有机会认识女孩，因为学校或单位女生有限，且没有交集。如果这个逻辑成立，部队里的兵哥哥是否都无法结婚？他们几年都在部队，更没机会认识女生和谈恋爱。那么多军嫂是从哪里来的？

如果你经常自我封闭，暗示自己不行，认为自己内向、不会说话，长此以往，难道等着女孩主动认识你、找你搭讪吗？

所以，今天老师要告诉大家的是如何让自己一开口就征服人心。

恋爱军团有一位学员，平时性格很好，为人热心，乐于助人。唯一不足的是表达能力可能过于激进，经常讲笑话，有时让人感觉滑头、油腻、不正经、不踏实。他很困惑，就连追女孩时，女孩也觉得他不靠谱。这样如何与对方建立亲密关系？

现在很多人都将自己定义为内向。但我们都有一个共同特质：都不喜欢冷场。我们都愿意与同事、新朋友有进一步认识，都不愿在办公室里当一个默默无闻的“盆栽”。你想融入环境，需要多长时间？

打个比方，如果班上有19个内向的同学和1个外向的同学，老师自然会认为那个外向的同学有问题。所以在群体生活中，内向通常不是问题。办公室也是一种群体生活，人们聚在一起是为了完成某项任务或工作，付出自己的专业和努力。在这种大环境中，哪怕你不大会说话，也并非问题。你可以将其视为一种中性状态，而非特别归为负面。它不一定会扣分。

试想，在一个合唱团里，一个声音很小甚至只张嘴不发声的人，犯错的概率就会很小。所以在群体生活中，做到不讨人厌就已经很好了。外向的人冒的风险是什么？是太活跃、爱讲话，容易说错话、得罪人。相比之下，内向是不是很好呢？

那么，和人聊天时讲什么呢？不妨尝试随意聊聊。

以下是可聊话题示例。

*   公司附近有什么好吃的。
*   我常在公司附近看到一个穿着打扮很奇怪的人。
*   公司附近一个老板娘打扮得好漂亮。

这些都是可以聊的事情。不用讲笑话，就讲你的心情和感觉，也就是讲自己的故事。这些都是让对方感受到你意愿和诚意的方式。你想与对方进一步交流，想让气氛活跃起来。如果是一群人的聚会，你抛出这类话题，大家都能接上话。但如果你说笑话，别人除了笑之外，可能接不上话。相比之下，是不是感觉“公司附近有什么好吃的”这类话更能带动气氛呢？

因此，不如在口袋里储备一些随时随地都能抛出来的话题，让人有参与的欲望。

最后想要提醒大家的是，遇到尴尬场景时，你肯定觉得自己很为难。但这种尴尬并非你一人独有，在场的所有人都会感到尴尬。所以不要把所有压力都放在自己身上，放轻松就行。就像天气冷，每个人都能感觉到。

如果你是一个亲切的人，愿意抛出一个让大家都能聊的话题，这才叫做破冰。这样两个人才能有更好的交流，关系更加密切。

实际上，对于我们来说，语言过分强调话术高超、言语精雕细琢和缜密，却缺少了一种人性的表达方式来表现内心的真实情感。我的理解是，你可能没有遇到真正了解你的人，或没有真正懂你的朋友，所以你一直存在双面性，也不愿表现真实的自己，自我封闭起来。

虽然我们不需要完全活在别人的评价中，但对于日常表达能力还是要把握的。如果你单单想通过正确的开场白话术解决问题，我们这里没有。我们只能帮你解决根本问题。你这么多年的说话问题，难道靠一套开场白模板就能解决吗？你心里清楚，很重要的一点是：想要一开口就得人心，那么你首当其冲的标签就是**真诚**。做到真诚，对方对你的兴趣度也会更大。如果你有类似感觉，请告诉自己：**走心的沟通一定比满满的话术更招人喜欢**。一开口能让对方有参与感，由浅入深地了解，你们的关系会进步更快。

把这些道理用在恋爱中，也可以总结出四点恋爱心法。

以下是四点恋爱心法。

1.  **九一法则**：刚开始和女孩接触时，无论是现场还是网络，你必须不断带领话题，女生只负责回应。公式表示为：`你的话题提供量 > 90%`。初期你必须提供90%以上的话题，女生只提供10%。绝大部分时间由你说话和带领话题，后期才能逐步平衡对话量。一开始若不遵守此法则，女生可能会失去兴趣。
2.  **浅尝辄止**：任何话题都不要聊得太深入，因为你们刚刚认识。聊得太深入会让女生产生别扭感，因为只有很熟的朋友或恋人才会深入交谈。这种别扭感来自陌生人，一旦女生察觉，就会给你贴上“陌生人”标签。你再想深入，只会引起反感。
3.  **提供价值**：当你们开始对话时，你需要主动为话题提供谈话价值，而不是不断问问题、索取价值。你需要承担带领角色，而不是被目标带领。代码逻辑可类比为：`if (首次对话): 提供价值，避免长篇大论和多余修饰词`。言简意赅能透露出你的态度，同时让目标觉得你真诚且有价值。
4.  **审视与调整**：当目标回应不好时，你要做的是审视聊天是否触犯以上错误。如果有，相应调整；如果没有，只需暂时搁置。因为女生此时可能没有交流窗口，我们应静待窗口出现，而非想着如何套路她，否则只会适得其反。

本节课中，我们一起学习了如何通过调整心态、运用非语言沟通、储备轻松话题以及秉持真诚原则，在社交中做到落落大方、一开口就征服人心。同时，我们将这些原则提炼为恋爱中的四个核心心法。关键在于实践与自我调整，从每一次交流中积累经验。
