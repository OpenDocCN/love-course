# 谢胜子38套合集：37、如何找到自己的属性与赛道2.0：第二讲

## 概述
在本节课中，我们将深入探讨如何识别并运用个人核心属性来规划职业发展。课程将围绕几个关键维度展开，帮助你定位自己是产品人、媒体人还是运营人，并理解如何在不同阶段运用合适的杠杆来突破成长瓶颈。

## 个人属性的发现与成长时机

每个人发现自身核心特质的时间点各不相同。有的人很早就意识到自己的强项，而有的人则发现得较晚。

如果你较早地发现了自己的某个特质，这可能意味着你的另一面能力相对薄弱。任何人从低阶向高阶发展，往往需要补足另一面的能力。

如果你较晚才发现自己的核心特质，这可能意味着你的另一面能力也很强，这反而会妨碍你找到自己最独特的一面。因此，当你从低阶向高阶发展时，补足弱点的速度可能会比那些特质非常鲜明的人更快。

这个观点本身具有一定的哲学思辨性，可能需要反复理解。不过，这个点并非本节课的核心，我们可以先将其放下。

## 第二个维度：定位产品人、媒体人与运营人

接下来，我们进入第二个维度，帮助大家进行自我定位。判断依据是：当你得知一段信息或一个故事时，你的本能反应是什么。

以下是三种典型的反应模式：

1.  **产品人**：本能反应是质疑信息的真实性，并希望深入调研、查证，获取更多上下游信息，打开新世界的大门。他们的特点是追求信息量和深度。
2.  **媒体人**：本能反应是思考如何将信息传播出去。他们不太关心信息的绝对真伪，而是擅长整合信息、添油加醋，用讲故事的方式向家人、朋友或社交媒体分享。
3.  **运营人**：本能反应是思考这个信息对谁有价值，谁能从中受益或感到焦虑。他们会考虑如何链接相关的人，整合资源，甚至思考由谁来讲述这个信息效果会更好。

通过以上讲解，你应该能初步理解这三种属性的区别。

举例来说，了解MBTI人格理论时：
*   **产品人**会深入研究16型人格，阅读大量相关文献和不同观点的帖子，力求全面掌握这个定位体系。
*   **媒体人**如果自己是ENTP，会更关注ENTP博主们的共性，并乐于分享“我们ENTP就是这样”之类的观点。

另一个辅助定位的方法是回顾你的“学霸模式”：
*   **纯看书式学习**：擅长通过阅读文字、论文进行深度学习。要走产品人深耕路线，对文字的吸收能力需要达到较高水平。
*   **复述式学习**：通过教别人来巩固和深化自己的理解。“教是最好的学”。这种方式更偏向媒体人属性。
*   **组织建群式**：热衷于快速获取信息并建立连接，喜欢在不同场合组建社群。这体现了运营人的特质。

你可以追溯自己过往的学习和行为模式，来辅助判断。

## 第三个维度：内在倾向与职场审美

前面两个属性应该比较清晰了。现在我们来看第三个属性，这关乎你更深层的价值观和审美。

以下是几组需要你品味和判断的倾向：

**第一组：战略风格倾向**
*   **倾向一**：更偏向内心深处、平衡、圆融、含蓄（甚至可称为更“阴性”）的战略。像腾讯早期的风格，不高调宣扬，而是像舰艇一样稳步前行，等待时机。
*   **倾向二**：向往断崖式领先、高调、拥有全民影响力的明星式打法。像马云早期的风格，在最高点时影响力辐射极广。

你内心更欣赏和崇拜哪种风格，反映了你的深层倾向。这种倾向可能会随着年龄和阅历增长而波动，例如年轻时可能更向往第二种，而年长后可能更偏向第一种。

**第二组：做事风格与职场审美**
*   **把一件事做绝**：不想提前铺垫关系，不想搭建草台班子，希望能力自然溢出后再扩展业务。这源于强烈的“明心欲”和“艺术欲”，而非“权力欲”或“政治欲”。
*   **把三件事做好**：可以适当提前“吹牛”，搞关系和处理业务都觉得有趣，能逐步积累政治智慧。这更偏向“权力欲”和“政治欲”，倾向于在幕后运筹帷幄。

这本质上是两种不同的“职场审美”。有人欣赏靠政治智慧安全着陆、左右逢源的人；有人则只崇拜拥有极强业务能力、靠专业说服他人的人。你内心更向往哪种，决定了你的发展路径。

如果你还比较年轻，对第三种属性的理解可以留待未来（例如26岁以后）再深入思考。举例来说，演员周迅更偏向“把一件事做绝”的明星属性，而赵薇则更偏向“把三件事做好”的操盘手属性。

**第三组：IP领域内的角色倾向**
*   **创始人/操盘手**：更理性、缜密，表达是为了商业盘服务，越表达逻辑和商业，魅力越强。他们更像高级别管理者，靠扎实的实力和让他人赚钱来建立领导力。
*   **IP本人/实商**：不能太理性，需要依靠情绪、自我真实魅力来吸引人。他们更靠个人魅力挣钱，通过建立个人崇拜来形成领导派系。

需要明确的是，没有哪种属性绝对适合或不适合当领导，关键在于你如何运用和研究自己的属性。

## 杠杆类型与阶段运作

每个杠杆都可以细分为不同维度。如何从学生时代的行为判断你的杠杆属性呢？虽然受环境影响，但更多取决于你自己。

以下是几种学生时代的典型模式及其对应的杠杆属性：

*   **对钱敏感**：从小在班里倒买倒卖（资料、二手、特产），收废品挣钱，不在乎面子，只想赚钱。
*   **流量选手**：从小是学校的风云人物（学生会主席、十佳歌手），谈恋爱的本质是为了让所有人知道。
*   **产品人（深耕型）**：高级学霸，阅人无数，看不上爱出风头的人，他们的杠杆通常只能靠产品深耕。
*   **运营人（平级路线）**：学习一般，但人员很好，会讲段子，受同学欢迎，走的是群众路线。
*   **运营人（上层路线）**：学习一般，但长得机灵，特别会讨老师开心。
*   **对钱有数**：对班里活动花多少钱、每人要交多少钱特别敏感。
*   **流量选手（表达型）**：喜欢发言、写演讲稿，但考试成绩一般。

最重要的是，定位好自身属性后，如何分阶段运作自己。

**关于A属性（创业/自由）与B属性（职场/平台）的运作：**
*   如果你的**A属性非常强，B属性为零**，很可能大学就开始创业、搞副业。
*   如果你的**B属性非常强**，很可能进入体制或大平台，工作7-10年内从未做过任何副业，也不认识行业外的人。
*   如果你是**AB属性兼具**，通常第一阶段（Face One）是先巩固B属性，做出成绩和知名度；然后找一个过渡状态（如跟随创业、跑通小闭环、自由职业）尝试AB结合；最后，如果A属性爆发，可以尝试向一号位（创始人）靠拢。
*   一般来说，B属性可以慢慢向A属性转化，A属性可以成为B属性的终局，但反之则非常困难。

**关于产品人、媒体人、运营人的分阶段运作：**

*   **产品人**：
    1.  **第一阶段**：先做出扎实的业务成绩和支点（能说能做）。
    2.  **第二阶段**：上**垂直类流量杠杆**（如公众号、播客），吸引业内关注，运作行业奖项，建立垂类知名度。
    3.  **第三阶段**：根据意愿，决定是否上团队杠杆、接受投资，向资源整合和公司化运作突破。如果始终热爱一线业务，那么拥有铁杆粉丝和产品，以“A+B”模式生活也不错。

*   **媒体人**：
    1.  **第一阶段**：先做出流量和知名度，这是你的热爱所在。
    2.  **第二阶段**：上**人的杠杆**。向下：组建团队，成立工作室/公司；向上：社交比自己咖位更大的人，为自己“抬咖”。
    3.  **第三阶段**：接触前科资本。媒体人也不擅长一开始就拿资本。

*   **运营人**：
    *   **高阶运营**：可以直接接触前科资本。
    *   **中阶运营**：先将运营能力产品化，然后打社交牌，再上知名度杠杆和融资杠杆。
    *   **初阶运营**：先做好运营本职工作。

**关于“实商”与“财观”属性的阶段突破：**
每个人的第一阶段都是先充分发挥自己的核心特质（实商或财观）。第二阶段则需要突破自己不擅长的点：一个靠“实商”吃饭的人，到一定程度需要融入一些“财观”的理性与规划，才能成为高阶实商；反之亦然。这体现了“突破弱指的是不够强，突破强指的是不够弱”的辩证思想。

## 案例解析

为了帮助理解，我们分析两个行业的案例。

**案例一：IP行业（产品人属性）**
*   **描述**：科技型博主，非常喜欢做内容（产品人属性），情绪稳定（非靠实商赚钱），靠实力和逻辑内容获得忠实粉丝和行业影响力，擅长输出系统方法论，不喜欢浮夸的无关社交。
*   **分析**：他拥有垂直类杠杆。要突破成长瓶颈，有两种路径：1) 突破内容洁癖，向泛C端突破，获取更大流量杠杆；2) 突破管理或向上社交的短板，向资源整合和公司化运作突破。

**案例二：IP行业（媒体人/实商属性）**
*   **描述**：极度擅长搞流量和引起情绪波动（实商重），是媒体人属性，已有自己的小公司（A属性强），朋友众多（人的杠杆强），产品定价不扎实，搞钱为王。
*   **分析**：此人实商属性明显，也有一些B属性。他擅长人和流量的杠杆。IP行业本身属性就更偏向实商、A属性和流量。因此，如果个人属性与行业属性高度契合，就会更耀眼和爆发。但这并不绝对。

**案例三：职场人（B属性/产品人倾向）**
*   **描述**：在大厂深耕5年，个人能力强，演讲输出强，能用业务能力搞定客户关系，但特别不喜欢高层政治斗争。
*   **分析**：此人偏B属性和产品人属性。他可以用“事”（专业）搞定关系，而非用“人”的感觉。政治智慧不足，有创业想法但未行动，自媒体想法也未实施。这很可能说明其B属性占主导（80%以上），终身走B路线（职场深耕）是完全可行的。未行动不是因为懒，而是属性使然。

**案例四：职场人（行业错配）**
*   **描述**：入职不到两年，完全不喜欢财务工作，想跳去娱乐文化行业，但无经验和人脉积累，喜欢文字但短视频能力一般，不知道能否做自媒体。
*   **分析**：此人是**行业错了，但属性没有错**。他依然是B属性。建议不是贸然去做自媒体（属性不符），而是应该**去娱乐文化公司做财务**，实现行业转换，再从内部寻求发展机会。他目前的状态还无法有效运用外部杠杆，打工路线是正确的，只是需要换到更喜欢的行业。

## 总结
本节课我们一起学习了如何通过多个维度（产品人/媒体人/运营人、A/B属性、战略倾向、职场审美、实商/财观）来深度剖析自己的核心属性。我们明白了属性没有高下之分，关键在于识别它，并学会分阶段、有策略地运作自己，在合适的时机运用合适的杠杆（垂直流量、人的连接、资本等）来突破成长天花板。同时，我们也通过案例区分了“属性错误”和“行业错误”的不同调整策略。有些观点可能需要时间消化，未来的课程还会从其他切面继续深化这些概念。希望本节课能帮助你更清晰地认识自己，为职业规划提供新的思路。
