# 社交心理急救指南：3：如何从容应对别人的调侃？

在本节课中，我们将要学习如何从容应对他人的调侃。课程分为两个主要部分：首先，我们将探讨为什么我们容易被调侃；其次，我们将学习具体有效的应对方法。

## 第一部分：为什么我们会被调侃？

上一节我们介绍了课程的整体结构，本节中我们来看看人们容易被调侃的第一个核心原因。

### 1.1 攻击性的压抑

根本原因在于我们的攻击性被压抑了。攻击性是每个人的本能，它是人能够生存和发展的基础。一个人如果没有攻击性，就容易被持续欺负，因为攻击者无需付出任何代价。

**攻击性的本质**：攻击性是人最原始的本能之一。从原始时代的捕猎生存，到历史上的朝代更替（如秦朝、汉朝等），攻击性（表现为战争或反抗）都起到了推动变革和保护自身的作用。

然而，许多人在成长过程中学会了压抑自己的攻击性。这通常是为了**换取**某些生存资源或安全感，例如童年时期为了获得父母的关注和爱，而选择表现得“乖顺”。这种压抑会形成习惯，导致成年后在面对冲突或调侃时，依然不敢释放攻击性进行反击。

这种自我压抑的人，往往在工作、学习和人际交往中不敢争取，发展受限。

### 1.2 他人的习惯

除了自我压抑，另一个原因是**他人的习惯**。有些人习惯通过调侃或攻击他人来获取存在感和关注。

以下是这类人的特点：
*   他们的本质内心是虚弱的，需要通过攻击行为来证明自己。
*   他们和攻击性压抑的人其实是同一类人，区别仅在于所处的环境和面对的对象是否让他们感到“安全”。
*   一旦他们面对更强的人，自己也会变成攻击性压抑的一方。

所以，喜欢调侃别人的人并非真正的强者，他们只是用这种方式来掩饰内心的弱小。

## 第二部分：应该如何应对别人的调侃？

上一节我们分析了被调侃的内在和外在原因，本节中我们将学习四个具体的应对策略。

在现实生活中，被调侃的感觉总是不舒服的。我们首先要做的是心理建设。

### 2.1 不要因被调侃而自责

很多人被调侃后，会陷入自责和自我攻击（例如：“我怎么这么没用，不敢反击”）。这种自责会消耗内心能量，让你变得更加虚弱和无力，反而更无法有效应对未来的调侃。

**正确做法**：接纳当下的自己，不要沉浸在糟糕的情绪中反复回忆。告诉自己“被调侃已经够难受了，不要再攻击自己”，然后去做其他事情，跳出情绪漩涡。

### 2.2 不带敌意地表达感受（针对亲近的人）

当调侃来自朋友或亲密关系时，一个有效的方法是平静地表达自己的感受。这能引发对方的内疚感，从而停止攻击。

**表达公式**：“我知道[对方提及的事实]，但你这么说话，让我感到很难过/伤心。”

**举例**：当朋友调侃你“天天减肥还像头猪”时，你可以说：“我知道我老是减肥减不下去，我也很努力。不过你这么说，让我觉得非常难过。”

### 2.3 不接招：不一定要接受对方的攻击

不要掉入对方的情绪陷阱。对方的调侃只是他个人观点的表达，你不必认同，也不必因此对自己产生负面评价。

**核心**：不评价自己。听到调侃后，只需“听到”，不要深入联想和自我否定。

**举例**：如果有人说“你的眼神怪怪的，是不是看不起我？”，如果你本身对自己的眼神不自信，就容易“接招”并陷入自我怀疑。正确的做法是：不对此进行自我评价，心想“这只是他的看法”，情绪就不会被牵动。

### 2.4 承认并夸奖对方（以柔克刚）

你可以承认对方话中的部分事实，甚至反过来夸奖对方。这会让对方的攻击像“一拳打在棉花上”，无处着力，从而不知所措。

**应对公式**：“对啊，我就是[对方调侃的点]。你真厉害/[你的优点]，我真羡慕你。”

**举例1**：别人说：“你怎么这么胖？” 你可以回：“对啊，我是挺胖的，一直减不下来。还是你最瘦了，真羡慕你。”
**举例2**：别人说：“你工作这么多年才挣这么点钱？” 你可以回：“是啊，我这个人努力但没找对方法。要是像你这么聪明会赚钱就好了。”

## 总结

本节课中我们一起学习了从容应对调侃的方法。我们来回顾一下四个核心策略：
1.  **勿自责**：停止自我攻击，接纳现状，跳出情绪。
2.  **表达感受**：对亲近的人，可以不带敌意地说出你的难过。
3.  **不接招**：不认同对方的攻击，不因此进行负面自我评价。
4.  **以柔克刚**：承认对方的话并夸奖对方，卸掉对方的攻击力道。

希望这些方法能帮助你在面对调侃时，能够更加从容和有力量。
