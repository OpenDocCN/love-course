# 聊天大师课程：01：心动窗口撕裂：第3课 联系感、安全感与邀约

在本节课中，我们将学习聊天流程中的第三步和第四步：如何与对方建立联系感与安全感，以及如何成功地进行邀约。掌握这些步骤，能让你们的互动从线上聊天自然过渡到线下见面。

## 建立联系感与安全感

上一节我们介绍了如何调动情绪，本节中我们来看看如何深化关系。你不会与一个毫无交集的人成为朋友。如果对方觉得与你根本不熟悉，便难以擦出火花。因此，你需要与对方建立一些共同点，形成一种联系感，作为你们之间的桥梁。

以下是建立联系感的一个实用技巧：**“也”字的运用**。其核心公式是：**“你也……”**。例如，“你也去过三亚？”或“你也喜欢吃小龙虾？”。这种表达能有效拉近双方距离，暗示你们拥有共同经历。

你可以根据对方的特点或其朋友圈展示的内容，寻找共同话题来建立联系。这是一种非常有效的聊天方式。

在建立联系感的过程中，还有一个关键要素：**安全感**。试想一下，当你与对方聊天非常舒适，通过开场白调动了她的情绪，并且你们已建立了一定的联系感，彼此有了一些了解。下一步自然就是考虑邀约。

但往往你一提出邀约，对方就可能变得犹豫不决或一再推脱。这通常说明你在安全感方面做得还不够。女性天生更为缺乏安全感，在她不清楚你的姓名、职业等基本信息时，会担心与你见面的安全性。

因此，我们需要进行一个常见步骤：**基本信息交换**。这指的是与对方互相分享一些如年龄、工作和星座等基本信息。

以下是进行基本信息交换的一个小技巧：**先分享自己的信息，再询问对方**。例如，先说“我是做设计的，你呢？”。这种方式能更自然地建立信任和安全感。

## 如何进行邀约

如果前面的步骤都完成得很好，邀约便是水到渠成的事情。在邀约环节，我们遵循三个核心原则。

以下是邀约的三个原则：

1.  **原则一：邀约要确定**
    真正的邀约必须有**确定的时间、地点和事件**。例如：“周六下午6点，我们在太古里那家甜品店见面。” 这就构成了一个明确的邀约。

2.  **原则二：先进行模糊邀约**
    在提出确定邀约之前，应先进行**模糊邀约**以测试对方的意愿。模糊邀约是指**只提及事件和地点，但不约定具体时间**。例如：“听说太古里有家甜品不错，改天我们可以一起去尝尝？” 如果对方回应积极，你再进一步询问：“那你什么时候方便？”，从而将模糊邀约转化为确定邀约。这能有效降低你的目的性，避免给对方压力。

3.  **原则三：文字要精简**
    提出邀约时，**对话应围绕邀约本身**，不要夹杂其他冗长话题。否则，对方可能被其他话题带偏，导致邀约事宜无法确定下来。

掌握以上原则，你的邀约成功率将大大提升。

## 课程总结

本节课中，我们一起学习了如何通过“也”字技巧建立**联系感**，以及如何通过**基本信息交换**来提供**安全感**。在此基础上，我们探讨了成功邀约的三个原则：**确定邀约**、**模糊邀约测试**和**文字精简**。将这些步骤融会贯通，你就能更顺畅地将线上互动推进到线下约会。
