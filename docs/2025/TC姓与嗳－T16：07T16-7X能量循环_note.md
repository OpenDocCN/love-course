在本节课中，我们将要学习性能量向上扬升的多种路径。除了上节课介绍的中脉，我们还将探索人体内部自成循环的能量系统，并学习如何运用它们来管理性能量，实现全身性的能量提升与转化。

## 玻璃瓶花园的启示

上一节我们介绍了中脉，本节中我们来看看性能量循环的其他路径。

1960年，一位名叫大卫·拉蒂莫的园艺爱好者进行了一项实验。他将一颗紫露草的种子种入一个巨大的玻璃瓶中，并加入少量水。随后，他将瓶子置于楼梯下方，那里可以接受阳光照射。

1972年，他曾打开瓶子浇过一次水，此后便再未管理。2013年，即50年后，这颗仅浇过一次水的植物依然在瓶内茂盛生长。这个玻璃瓶花园在50年间未补充新鲜氧气和水，却依然生机盎然。

这充分说明，植物内部形成了一个自给自足的循环系统。植物叶片通过光合作用产生氧气，自身再通过循环将排出的二氧化碳重新转化利用。

## 人体内部的自循环系统：小周天

一颗植物可以在内部形成循环。人体内也存在类似的、无需与外界交换即可让自体生机蓬勃的装置。

这个装置在人体内就是我们的**任督周天**，也称为**小周天**。这是一条非常著名的脉络。

*   **任脉**：位于身体前侧。起于会阴，沿肚脐、两乳中间、下巴，上行至下唇。因其在妊娠期间颜色会变深，故称“任脉”。因其在身体前侧，也称为前脉。
*   **督脉**：同样起于会阴，沿脊柱上行至头顶，再向下至上颚结束。

在上一节课讲“搭鹊桥”时我们简单提到，当牙齿轻轻抵住上颚，就等于连接了身体的前脉（任脉）与后脉（督脉）。此时，整个身体会形成如同玻璃瓶花园一样的内部自我循环系统。

## 小周天的性能量管理应用

在进行性能量管理时，对于男性而言，除了通过中脉将性能量向上扬升，也可以通过小周天（任督轮）来实现。

当性能量升起时，可以观想能量从会阴进入身体，沿尾骨、脊柱一直向上，来到上颚。能量在此处接续，转化为液体，再顺着身体前侧向下，回到会阴。这样就形成了一个源源不断的循环体系。

练习时可配合呼吸：
*   **吸气**时，可轻微收缩挤压会阴（PC肌），观想能量顺脊柱上行至**上颚**。
*   **吐气**时放松，观想能量顺身体前侧下行，回到**会阴**。

在道家练习中，通常以九的倍数进行循环，例如转36圈。九被认为是一个特别的数字，可以为神经系统进行编程。

## 综合技法：“巨无霸”练习

还有一种更高级的玩法，将浪潮呼吸、PC肌收缩和小周天结合。这一招对于调集能量、提升专注、控制射精、实现全身性多重高潮而言，是一个常见的“巨无霸”技法。

2.  **吐气**发声的同时，放松会阴，让能量顺身体前侧（任脉）下行，再次回到**会阴**。

```
吸气 + 收缩PC肌 -> 能量沿督脉上行
吐气 + 发声放松 -> 能量沿任脉下行
```
在练习中，当感觉能量充足或比较累时，可以停止收缩PC肌，但呼吸与结合小周天的观想练习不要停止。

## 小周天运行的三道关卡

根据前人修炼经验，小周天运行有三个比较难过的关卡。道家的“内景图”描绘了这个能量循环的线路。

图中右下角有海浪，可理解为能量。能量首先来到尾闾关，此处被描绘为有牛车在拉，意味着通过需要较大力气。过关后继续上行至两肩胛骨中间的夹脊关，图上有火象征。第三个关卡在后脑勺的玉枕关（也称泥丸宫）。这三关需要通过一些功夫才能打通。

## 详解三道关卡

小周天运转中较难通过的三关：
1.  **尾闾关**：对应尾骨区域，被认为是“精”的中心。它位于下丹田位置，是人体吸收食物水谷精华的存储中心，也是性能量的存储中心。无论男性的前列腺还是女性的子宫、卵巢，几乎都对应第一关。此关不通，能量便无法继续上行，会影响小周天气机与三个丹田的流通。尾闾关系到周身能量的发动与起动，运通全身阳气。现代解剖学上，尾骨与骶骨间的八髎穴有八个孔洞，神经血管由此穿过脊柱连接腹腔脏器。尾闾关一旦松动，忍带松解，会使脊髓液得到震动并上传至脑髓，形成直接而强有力的能量流动线路。尾闾一定要保持中正。日常生活中的不良姿势会导致其不正。尾闾如同身体能量发射装置的扳机，一提会阴就如同扣动扳机，尾闾前兜，性能量便走脊髓自动上扬。能量越多上扬，残留在性器官区域的就越少，将不再有射精或能量崩塌式高潮的需求，转而实现全身喜悦、酥麻畅快的能量扬升感。此时，性能量下走的通道才能真正被关闭。

3.  **玉枕关**：位于后脑勺，与眉心相对，类似于现代解剖学中的松果体位置，是“神”的中心。

这三个中心构成了“精、气、神”。道家前辈一致认为尾闾关最难通过。此关不通，能量无法继续上行。

## 卯酉周天（身体侧面循环）

小周天介绍完毕。接下来了解另一个可以帮助我们像玻璃花园一样锁住自身能量的循环：**卯酉周天**。

它位于身体侧面，在十二时辰中对应“卯”和“酉”，故称卯酉轮或卯酉周天。同样起于会阴，沿骨盆穿过大腿根，来到腰侧面，再顺腋下、脖子绕一圈，穿过腋下沿腰侧，再次穿过腿回到骨盆。这是在身体侧面的一个转法。

一开始转动时可能较慢，贴近身体轨迹。熟练后，轨迹可能变得更圆，不完全贴近肉身，但路线是在身体侧面无疑。

课件上的“周天璇玑图”同时展示了身体前后的小周天、左右的卯酉周天，以及接下来要说的身体横切面的带脉图。文字揭示了至高奥秘：身体前后的乾坤交媾（小周天，上乾下坤，一升一降）可帮助“收外药”，采集天地日月宇宙能量，称为“外交媾”。卯酉周天（左旋右转，一起一伏）则称为“内交媾”，收的是“内药”。当两者同时转起来时，自身就已形成一个生机勃勃的动力系统。

另一幅极具玄机的图片来自周敦颐的太极图。他提出的无极、太极、阴阳、五行等概念对后世影响深远。此图揭示的道理如“团气至柔”、“天道左旋，地道右旋”等，也影响了中医经络理论。“团气至柔”讲的是如何把身体里的精微元素通过旋转的方式凝结、运化。此图也将成为后续学习“丹道深骨高潮”的理论依据。图上有空、阴阳、动静，动极生阳，阳极生阴，万事万物在不断互化中，乾道成男，坤道成女。

通常卯酉周天也转36圈，可按九的倍数练习，对神经进行编程。
*   **从左往上转**：观想能量从会阴升起，来到左侧腰，穿过左腋下，沿左侧脖子环绕头部一圈，再从身体右侧回到会阴。熟练后轨迹可能更接近圆形。
*   **从右往上转**：可练24圈。

卯酉周天转起来后，并非我们实际扬升性能量的主要路径（主要路径是中脉和小周天）。但为什么还要讲它和接下来的带脉？因为当卯酉周天转起来时，可以巩固和提升我们内部的机能。好比扎灯笼，只有中间的中脉和前面的任督，结构不够稳定。在侧面（卯酉）和横向（带脉）再扎上“竹签”并捆绑，装置就会变得特别稳固。

注意，中脉、小周天（任督）、卯酉周天都有一个共同点：都起于会阴穴。只要我们持续练习会阴，这三条大脉络无论走哪条，都能起到活化和互相激发的作用。

## 带脉（身体横向循环）

在我们身体的横向，还有一个线路：**带脉**。顾名思义，它像一条腰带缠绕在腰中间。

它是一个横向循环，通常做顺时针旋转：从肚脐开始，到右侧腰，转到后背脊柱，再转回肚脐，如此源源不断一圈。

带脉与之前学的冲脉、任督、卯酉等纵向脉络的关系，是起到整体调节的作用。好比买了一把韭菜，需要用一根绳子从中间捆住。带脉就起到这种整体的加强和强化作用。它不是实际扬升性能量要用的，但当我们平时把这些“渠坝”建设工作完善好时，实际要用的脉络才会变得更加稳定。

**转动带脉的小窍门**：
转动带脉时，通常顺时针即可（也有人逆时针，无特别区别）。关键窍门是：当你转动带脉时，要**收窄一点**，绑紧一点。让这个能量像削苹果皮一样，越转圈越小，收到体内去。一开始可能转大圈，像缠在肚皮外面。慢慢越转越小，转到体内，比实际身体轮廓收小一点。这样就能起到把中脉、小周天、卯酉轮的能量收束起来的绝佳作用。

## 练习窍门：机在目

在转动各种“轮子”（小周天、卯酉、带脉）时，有一个小窍门：**机在目**。意思是机关、关键在我们的眼睛上。眼睛可以起到提纲挈领的作用。

*   **转小周天时**：闭上眼睛，但让眼珠跟随能量转动。能量从会阴后侧起，眼珠就向上转；能量从前侧下，眼珠就向下转。这样会超级有感觉，且容易转。
*   **转卯酉周天时**：同样闭眼，眼珠跟随转动。眼珠先朝左看（同时观想卯酉轮在转），朝上看，朝右看，再朝下看，完成一圈。

专注于眼睛动作，同时用眼睛引领脉络走向，好处是：第一，不容易走神；第二，非常能够带领能量转化。眼睛看对方向看够了，能量就会被撬动。眼睛还可以引领转动速度。眼睛转得慢，脉络转动也会慢些，可用来调整速度。

## 从“有为法”到“无为法”

刚开始转动任脉、卯酉、带脉时，都有一个共同点：一开始都是用“有为法”，即用意念（和眼睛）去带动。就像学骑自行车，刚开始要用力蹬几下。但蹬起来之后，自行车就会自己转，不需要一直用力踩。

我们转这些轮子也是先用意念、眼睛带起来。一旦带起来之后，它就会一天24小时都在转。你只需要偶尔（睁眼闭眼均可）内观一下它是否还在转。如果发现没转了，就用眼神或意念引导几下，它就会重新转起来。而且它们转动的速度会趋于均匀，自动与天体宇宙的轨迹同步。这时就进入了“无为法”的最佳状态：24小时三轮自转（任督、卯酉、带脉同时转）。此时，你基本上就是一个“金刚不坏身”了，外在不良信息难以干扰你，内在能量又能源源不断运化循环。

这时就是“法来练你”。如同我们说的“真息”，一开始我们练呼吸让自己沉下来。当真正进入真息，轮子可以自己转时，你会体验到毫不费力的轻松。

当所有气脉、微细脉管都打开时，我们会得到身体自发产生的大喜乐。这种喜乐比男女性交高潮的快感要强很多，而且绿色环保，无需借助外在的伴侣，自己就能享受这种内在的玄妙，享受飘飘然、喜悦的状态。这种状态结束后，不像一般射精高潮后有分离感、失落感、断崖式空虚感。通过这种功法循环起来的能量，让人精神奕奕、神采飞扬，事后也会觉得超级满足、超级有活力，可以用这能量去做各种各样的事情。

## 重要的意识调整

需要特别说明且很重要的一点是：**意识层面的调整**。

当我们练习各种脉络、能量通道时，意识里要有一个概念：并不是说“我的脉络很卡”或“尾闾关是死的，我动不了”。小周天、大周天，每个人都是开着的。只要是活着的人，经络都是开着的，否则生命就不存在了。所以我们只是没有感觉到它的存在。

你生而为神，生而具有这些，只是自己没有认出这种神性本质。我们透过课程，就是把意识之光带到你的意识里。让你发现：“哦，我本身小周天就是转得很快的，只是我之前不知道而已。”好比你家有百万存款，你有一个亿万富翁的爷爷，只是你爸没告诉你。现在告诉你，你就瞬间拥有了。现在告诉你，你的小周天、卯酉、带脉本来就是宽敞的。你只是之前不知道，现在知道了，就可以取用它。只是每个人继承的“财产”（根基）不同，需要做的功课有点不一样。

从“我的脉络没开，我要练开”（从无到有）的思维，转变为“本来就是开着的，我本来就是继承了亿万财产的，只是我自己不知道”。经过今天的课程，每个人都变成了“亿万富翁”。所谓“千年暗室，一灯即明”。这些脉络，今天就已经把它照进你的意识之中。接下来你只需要静静做一点点努力，去经营好这份“亿万资产”，让它变得更扩大。
