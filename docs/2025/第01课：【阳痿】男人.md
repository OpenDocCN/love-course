# 男性健康课程：01：勃起功能与勃起功能障碍（ED）全面解析

在本节课中，我们将系统性地学习男性勃起的生理机制、勃起功能障碍（ED，俗称“阳痿”）的科学定义、诊断标准以及主要病因分类。我们将摒弃常见的误解，用医学视角理解这一常见问题。

## 什么是勃起？

上一节我们提到了课程主题，本节中我们来看看勃起的科学定义。勃起并非仅在性刺激下发生。

医学上，男性的勃起主要分为三种类型：

*   **心理性勃起**：由与性相关的内容直接诱发，例如视觉（看到异性或色情内容）、听觉、嗅觉、性幻想或性思维。公式可简单理解为：**性刺激（视觉/听觉/思维）→ 大脑中枢兴奋 → 勃起**。
*   **夜间勃起（含晨勃）**：在睡眠中不自主动出现的勃起-疲软循环生理过程。这是男性的正常生理表现。健康男性每晚会发生数次夜间勃起，青春期男孩可达约6次，青壮年总时长可达约90分钟，随年龄增长次数减少。
*   **反射性勃起**：由直接的身体接触或摩擦生殖器所诱发。这通常是我们性生活时产生的勃起。

## 如何诊断勃起功能障碍（ED）？

了解了勃起的类型后，一个关键问题是：是否三种勃起都正常才算健康？答案是否定的。

诊断勃起功能障碍的核心标准是 **“同房时的勃起情况”** ，而非晨勃或手淫时的勃起情况。主要原因在于，正常性生活与手淫或观看色情内容时的心理状态、环境氛围和性刺激强度完全不同，二者没有可比性。

勃起功能障碍（ED）是指：男性在性生活时，虽有正常性欲并受到有效刺激，但阴茎**不能勃起**、或**勃起硬度不足**、或**勃起不能维持足够时间**以完成满意性交的疾病。

判断是否患有ED，需从以下三个层面综合评估：

以下是判断ED的三个核心维度：

1.  **持续性**：问题是否持续出现。偶尔、短时间的功能不佳不能诊断为ED。
2.  **勃起硬度**：能否达到并维持足够的插入硬度。临床上常用“勃起硬度分级”来评估：
    *   **1级**：阴茎增大但不硬，无法插入，类似豆腐硬度。
    *   **2级**：阴茎勃起但硬度不足，无法插入，类似剥皮香蕉硬度。
    *   **3级**：阴茎硬度足以插入，但未完全坚挺，类似带皮香蕉硬度。
    *   **4级**：阴茎完全勃起并坚硬，类似黄瓜硬度。**4级硬度为理想状态**。
3.  **完成满意度**：能否完成并获得双方满意的性生活。国际上常用 **IIEF-5评分表** 进行自测：
    *   **22-25分**：正常。
    *   **12-21分**：轻度ED。
    *   **8-11分**：中度ED。
    *   **5-7分**：重度ED。

## 勃起功能障碍（ED）的病因有哪些？

导致ED的原因复杂多样，从病因学上可分为两大类：功能性（心理性）病因和器质性（身体疾病性）病因。

### 一、功能性（心理性）病因

指身体器官无病变，主要由心理或精神因素导致。主要包括以下五点：

以下是五种常见的功能性病因：

1.  **家庭与成长环境影响**：如保守的宗教家庭背景、父母对性的消极态度、幼儿期性教育不当、早期性接触受到惩罚、首次性交失败带来心理创伤等。
2.  **自身情绪异常**：如自卑、对阴茎尺寸焦虑、害怕性交失败、害怕怀孕或染病、抑郁情绪等。
3.  **夫妻关系不和**：如夫妻间缺乏信任、感情淡漠、相互吸引力下降等。
4.  **性知识缺乏或行为不当**：如对性生活缺乏正确认识、长期手淫、性生活过度频繁等。
5.  **神经衰弱**：由于久病、过度疲劳、生活压力大、长期焦虑或抑郁等导致。

指身体本身存在疾病或问题，直接或间接导致ED。主要包括以下六点：

以下是六种常见的器质性病因：

1.  **内分泌疾病**：如糖尿病、甲状腺功能异常（甲亢/甲减）等。例如，长期糖尿病可引起末梢神经病变，影响性神经功能。
2.  **神经系统疾病**：如脑部肿瘤、脊髓损伤（外伤截瘫）、腰椎间盘突出压迫神经、骨盆外伤导致性神经断裂等。
3.  **血管性疾病**：这是常见病因，包括阴茎动脉供血不足、静脉闭合功能障碍（静脉漏）等，导致海绵体充血不足。
4.  **生殖系统病变**：如先天性或继发性生殖器畸形。
5.  **药物影响**：长期服用某些药物可能导致ED，如抗抑郁药、镇静药、降压药、激素类药物等。部分患者在停药后可自行恢复。
6.  **年龄因素**：随着年龄增长，勃起功能可能出现生理性衰退。

## 课程总结

本节课中我们一起学习了以下核心内容：

1.  **晨勃不等于性功能**：晨勃消失不直接等同于勃起功能障碍（ED）。
2.  **ED的诊断核心**：应以**同房时的勃起表现**为主要判断依据，包括硬度和维持时间，而非晨勃或手淫时的状态。
3.  **ED的定义**：指在有性欲和有效刺激下，阴茎无法达到或维持足够硬度与时间的勃起以完成满意性生活。
4.  **病因复杂多元**：ED病因主要分为**功能性（心理性）**和**器质性（身体疾病性）**两大类，且常混合存在。日常生活中可通过调整生活习惯、改善心理状态和伴侣鼓励来改善，必要时需寻求专业医疗帮助。
