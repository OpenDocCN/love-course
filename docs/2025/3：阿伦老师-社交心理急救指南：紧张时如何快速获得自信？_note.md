# 社交心理急救指南：3：紧张时如何快速获得自信？

在本节课中，我们将要学习在紧张时如何快速获得自信。课程内容分为两个主要部分：首先，我们会分析紧张时人们容易出现的错误行为；其次，我们将探讨如何快速获得自信的具体方法。

## 紧张时我们容易出现的错误行为

上一节我们介绍了课程概述，本节中我们来看看紧张时我们容易出现的错误行为。这些行为可能我们并未察觉，或者知道但不知如何改变。

以下是两个最常见的错误行为：

1.  **强迫自己不紧张**：这是最常见的问题。
2.  **出现逃避行为**：这是我们说最常见的两个点。

### 1. 强迫自己不紧张

在探讨这一点时，我们需要了解情绪的核心特点。情绪是虽然抓不到、摸不着，但可以真实感知到的东西。

以下是情绪的三个核心特点：

*   **时效性**：情绪持续的时间是有限的。无论是开心还是不开心，只要不去持续干预，其强烈程度会随时间推移而减弱。例如，获得心爱物品的喜悦感通常在几个月后就会淡化。
*   **不可控制性**：情绪，特别是通过主观意志去控制，是不可行的。主观意志通常只能干扰情绪的强度，且往往是放大负面情绪，而非减弱它。试图强行控制紧张，往往会适得其反。
*   **刺激反应**：可以通过外界刺激来引发不同的情绪反应。这是改变情绪的有效途径。例如，感到紧张或悲伤时，可以通过看喜剧电影、跑步锻炼等方式，用良性的外界刺激来产生放松或开心的反应。

只要掌握情绪这三个核心特点，就不会陷入“强迫自己不紧张”的误区。你知道紧张感会随时间减轻（时效性），知道它不可强行控制（不可控制性），并懂得用积极的外界刺激来调整状态（刺激反应）。

### 2. 出现逃避行为

出现逃避行为，是因为我们感觉自己无法处理当下产生的强烈情绪或突发状况。

逃避行为会产生以下问题：

1.  **逃避影响别人对自己的评价**：别人通常不会因为你的紧张而否定你，但可能会因为你的逃避行为而对你产生不好的印象。勇敢不是不紧张，而是面对问题不逃避。
2.  **逃避解决不了现实问题**：逃避行为无法让你完成任务或达成目标，反而会让你错失成长和展示的机会。
3.  **逃避会产生负面情绪**：根据“具身认知”理论，行为会影响大脑的认知。逃避的行为会向大脑传递“我很没用”、“我无法应对”的信号，从而产生沮丧、自我否定等负面情绪。这种情绪循环会让人陷入“逃避-自责-要求更高-再次逃避”的恶性循环。

## 紧张时如何快速的获得自信？

上一节我们分析了紧张时的错误行为，本节中我们来看看如何快速获得自信。

以下是四个具体的方法：

1.  **积极的社交态度**：在感到紧张时，内心态度往往是消极的。要快速改变，需要立即将消极、被动的态度转变为积极、主动的态度。可以对自己进行积极暗示，如“我能做到”、“我可以应对”。
2.  **调整注意力的方向**：紧张时，注意力常集中在“不要紧张”上，这反而会强化紧张感。正确的做法是将注意力转移到当下应该做的事情上，比如对方的谈话内容、你要表达的信息。随着注意力转移和时间的推移（时效性），紧张感自然会减轻。
3.  **让自己获得瞬间自信**：根据“具身认知”理论，身体动作可以影响心理状态。在紧张时，可以通过一些身体动作来快速提升自信感，例如：握紧拳头、轻微跳动一下、深呼吸，同时告诉自己“我能行”。这种瞬间获得的能量足以支撑你开启一次短暂的社交互动，而成功的互动体验又能进一步增加自信。
4.  **看轻紧张的情绪反应**：在社交中紧张是不可避免的。你是否觉得紧张很重要，决定了它对你的影响大小。如果你过分看重紧张，就会过度关注他人反应，容易产生误解。如果你能看轻它，不把它当作关注的焦点，那么不仅你自己受影响更小，对方通常也不会在意你的紧张。把注意力放在谈话本身，紧张感就会被“看轻”。

## 总结

本节课中我们一起学习了在紧张时快速获得自信的方法。

首先，我们认识到紧张时容易出现的两个错误行为：**强迫自己不紧张**和**出现逃避行为**。我们了解到情绪具有**时效性**、**不可控制性**和**刺激反应**的特点，理解这些能帮助我们避免强迫行为。同时，逃避会带来他人负面评价、解决不了问题并产生自我否定的情绪。

其次，我们学习了四个快速获得自信的技巧：建立**积极的社交态度**、将**注意力调整**到当下任务、通过身体动作获取**瞬间自信**以及**看轻紧张情绪**本身的重要性。

掌握这些要点，能帮助你在感到紧张时更好地调整状态，更从容地应对社交场合。
