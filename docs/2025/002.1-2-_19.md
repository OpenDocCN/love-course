# 吸引力模型课程：002.1-2：吸引力模型

## 概述
在本节课中，我们将要学习吸引力的核心构成要素，并掌握一个实用的社交模型。我们将了解如何通过构建“心动指数”与“心疼指数”来建立高质量的吸引力，并学习在社交互动中如何从无需求感开始，逐步推进关系。

## 吸引力的构成要素
我们都想构建良好的吸引力。吸引力由两个核心指数构成：**心动指数**与**心疼指数**。

### 心动指数
心动指数是指异性接触我们后，产生怦然心动、喜欢或好感的感受。心动指数由两个重要因素构成：**价值**与**能量**。这两个因素将在后续课程中详细讲解。

根据统计，现代女性普遍认可的亲密关系发展路径是：**好感 → 喜欢 → 亲密关系 → 真爱 → 结婚**。这意味着，良好的心动指数能让女性喜欢上我们，从而为关系发展奠定基础。

### 心疼指数
高质量的吸引力构建不仅需要心动指数，还需要心疼指数。这意味着，我们不仅要让女性觉得我们价值高、能量强、有魅力，还要让她们在某些时刻觉得我们可能疲惫、脆弱或需要被照顾。

构建心动指数，是让女性产生如女儿崇拜父亲般的感觉；构建心疼指数，则是让女性产生如母亲照顾孩子般的感觉。两者结合并根据实际情况调整比例，效果最佳。构建心疼指数的方法将在后续课程中教授。

## 吸引力社交模型
接下来，我们学习一个实用的社交模型。当我们进入社交环境面对任何个体时，应遵循以下步骤。

### 1. 呈现无需求状态
面对美女时，不表现出任何性需求；面对有地位的人时，不表现出对资源的需求。为什么不能表现出需求感？因为在未建立吸引前就流露需求感，结果往往不理想。

### 2. 进行高价值展示
在无需求的开局后，紧接着需要寻找合适机会进行高价值展示。高价值展示后，部分人会被吸引并给出好感窗口，部分人则无反应。

对于无反应的人，若想继续推进关系，则需在保持无需求感的基础上继续互动，并寻找机会进行其他形式的高价值展示，直到吸引形成、窗口打开。

例如，当你谈论手机应用而对方毫无兴趣时，下次在朋友圈发布自己的摄影美图，对方可能就会兴致勃勃地与你讨论。

### 3. 识别真假好感窗口
当你发现高价值展示带来吸引并看到好感窗口后，不要只顾高兴。因为此时尚不确定窗口的真实性与大小。有些女性可能出于其他目的表现出好感。

例如，相亲网站中有些女性表现主动，但目的是让你发红包或充话费，这并不靠谱。

### 4. 进行服从性测试
在看到好感窗口后，首先应及时进行更多沟通以建立联系感。在沟通过程中，可以适时进行服从性测试，例如请对方发照片、提供电话号码或直接邀约。

服从性测试的难度高低，应根据吸引形成的程度和联系的深度来决定。一旦多个服从性测试通过，即可确定吸引形成，应该向前推进关系。

以下是该社交模型的步骤总结：
1.  **无需求感入场**：不流露任何需求。
2.  **高价值展示**：寻找机会展示自身价值。
3.  **识别窗口**：判断对方好感的真伪与程度。
4.  **服从性测试**：通过测试确认吸引并推进关系。

## 总结
本节课中，我们一起学习了吸引力的核心构成。吸引力由**心动指数**（包含价值与能量）和**心疼指数**共同构建。此外，我们还掌握了一个社交模型：从**无需求感**入场，到**高价值展示**，再到识别**真假好感窗口**，最后通过**服从性测试**来确认并推进关系。
