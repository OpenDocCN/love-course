# 恋爱军团：02：女生的自我说服：让她不知不觉爱上你

在本节课中，我们将要学习一个关键的恋爱心理学概念——**女生的自我说服**。掌握这一点，能帮助你理解女生看似“善变”的行为，并引导她自然而然地对你产生好感，从而顺利推进关系。

## 课程概述

女人对男人来说，有时像一个谜团。男生可能不懂女生，也可能因为男女思维有别而产生困惑。有些兄弟会遇到这种情况：前几天女生对你还好好的，今天却突然变得冷漠。

很多同学可能都遇到过类似情况：你和女生交往初期比较顺利，彼此有好感，甚至快确定关系了。但不知为何，第二天你再联系她时，她像换了一个人，忽然表现得很冷漠，并告诉你她最近不想谈恋爱，建议先做朋友。

## 核心问题分析

这样的情况经常发生。例如，一个女生和男生约会，两人十分开心，她对男生感觉也不错。在浪漫氛围下，他们有了亲密互动。男生可能觉得关系差不多了，但在后续交往中想继续亲密接触时，却遭到了女生的拒绝。

这感觉很奇妙。原因恰恰在于之后的约会缺乏之前的浪漫氛围和感觉。失去了这些客观环境影响，加上男生可能操作不当，就很容易出现这种结果。

大部分男生都遇到过这种情况。若经验不足，很容易做出错误操作，例如质问女生“你到底什么意思”，甚至对女生发火，追问原因。这类行为往往会葬送掉最后一丝希望。

下面，我们来分析为什么会出现这种尴尬局面。其实，运用女性心理中的**自我说服**概念可以解决这个问题。有时你觉得女生不在乎你，正是因为你没有为她找到“在乎你”的充分理由。

## 引导女生自我说服的关键点

以下是几个引导女生进行自我说服、从而让她不知不觉爱上你的关键方面。

这一点很容易被忽略，但也非常重要：**你们尚未发生实质的亲密关系**。

很多男生觉得奇怪：我想珍惜她，不勉强她，希望关系更稳固后再进一步发展，以免她没有安全感，所以一直迟迟没有推进关系。但你可以回想一下身边的故事：发生关系前，总是你主动找她聊天；发生关系后，你会发现她开始经常粘着你。发生关系前，总是你对她嘘寒问暖；发生关系后，她会主动给你送早餐。

**核心公式**：`女生投入度（时间、情感、身体）` 与 `她进行自我说服的意愿` 成正比。

### 2. 生活交集与社交成本

你们之间的生活交集不够多，她觉得你不是她生活中不可或缺的人。

没有所谓的绝对忠诚，只是背叛的代价太大。如果你没有融入女生的生活，你们的交集就很少。试想，一个在你生活中可有可无的人，怎么可能占据你生命中非常重要的位置？

所以，兄弟们需要赶紧改善交集问题，融入她的生活圈。

长期暧昧，会让你身上只留下“玩耍”的标签。

### 4. 观察行动而非仅听言语

最后一点也很重要：不要过于相信女生嘴上说了什么，而要观察她的实际行动是怎样的。

如果你在这次机会中没有采取行动推进关系，那么女生会认为下次遇到类似情况，结果还是一样。慢慢地，女生对你敞开的“窗口”就会逐渐关闭，你也就失去了她。
