# 搭讪技巧：025.1-25：搭讪的具体套路

在本节课中，我们将学习搭讪的完整流程，包括开场、过渡、沟通和收尾四个阶段。我们将详细拆解每个步骤的具体做法和常见问题的应对策略，帮助你从理论到实践掌握搭讪的核心技巧。

## 开场阶段

上一节我们介绍了搭讪的基本概念，本节中我们来看看如何开始一次搭讪。开场白分为直接和间接两种。间接开场白通常需要一个相对自然的机会。因此，通常建议使用直接开场白。

以下是子明在2016年提出的一个开场白模板，该模板已被广泛使用。

```
你好，我刚刚在那边等朋友。看到你经过这里，觉得你很漂亮，所以想上来跟你打个招呼。我叫子明，你怎么称呼？
```

根据女生的窗口和反应，再进行后续对话。如果女生问“那你有什么事情”，说明她此刻有较大的不安全感。此时应该立即露出微笑，身体往后靠一两步，然后把之前的开场白放慢语速、更详细地重复一遍。

甚至可以加上一个假定的时间限制。例如：

```
我是不是吓到你了？是这样，我和朋友约了在这边见面，她还没有到。我在这边等他，刚好你经过了。我觉得你的气质真的很不一样，所以我鼓起勇气和你说话。可能这种方式太唐突了，吓到你了，实在不好意思。
```

如果此时通过观察发现女生的表情状态放松了一些，安全感有所提高，就可以开个小玩笑，让气氛变得更加轻松。例如：

```
你放心，我不会让你买……其实我现在紧张得要死，也不知道用什么更好的方式去认识你，所以只能豁出去了。
```

如果女生此时笑着，或者依旧没有走开，站在原地，说明她已经不再那么害怕。因此，我们可以进一步与她建立联系。例如：

```
对了，我叫子明，我是做咨询工作的（或者说我是做互联网的）。你怎么称呼？
```

一般女生在这个时候，往往只会告诉你姓氏。例如女生说“我姓张”。我们可以追加一句：“你叫张什么？”，然后不如开个小小的玩笑，比如：“张总，你好，以后多多关照。”

如果我们的肢体语言、表情和状态都没有太大问题，女生这个时候肯定会开心地笑起来。如果女生可能完全放松下来，那么开场白才算基本完成。

## 过渡阶段

开场完成后，我们需要一个过渡话题，以便将初步的社交链接维持下去。通常可以把话题转移到女生的打扮或当下环境上。

例如：

```

```

这是一个小小的恭维和赞美，同时也能进一步了解女生的信息，比直接问“你是做什么行业的”更委婉。

女生可能会回答：“不是，我是做互联网的”，或者“我是做教育的，我是老师”。我们可以继续开玩笑地说：

```

```

或者：

```
你是老师？我说怎么感觉这么熟悉呢。我觉得你和我的小学英语老师长得特别像，她特别亲和友善，而且人又特别漂亮。我们班当时有很多小男生都暗恋她。
```

到这里，女生可能会笑起来。如果在搭讪的开场和过渡阶段，女生有4次以上被我们逗笑，就可以为自己的表现打上90分，因为安全感和舒适感都已经建立。

## 沟通阶段

接下来可以相对更自然地进行更多关于联系感的沟通。在搭讪最开始的开场和过渡阶段，最忌讳的是没有微笑、没有眼神接触、过度紧张、讲话时不思考，或者停留太久。这些都会让女生本来不高的安全感进一步降低，导致她离开。

在过渡阶段，可以借助之前女生说话的口音或身高感觉去问她。例如：

```
我听你的口音，感觉你不像重庆人。你是在重庆上学吗？
```

女生可能说：“不，我已经工作了。” 这时可以利用前面提到的对女生穿着打扮的恭维，进而询问她的职业。如果女生说是学生，就可以说：“原来是大一的小学妹。” 女生可能会纠正你：“没有了，我快毕业了，我是大三。”

在这里要记住，不管你觉得女生的实际年龄如何，往小说绝对没有错。一般你觉得她可能25岁，你要说她21岁；你觉得她23岁，你要说她18岁。

接着可以说：

```
我也不是重庆的，我家是安徽的，这个地方你可能没听过。我在重庆这边工作已经有三年的时间了。
```

这里算作一个测试窗口的测试点。你说完这句话，如果女生问你：“你是做什么工作的？” 或者 “你家是安徽哪个地方的？”，就可以从过渡阶段进入到沟通阶段。真正的交流可能从这里才开始。

你依旧可以开小玩笑，例如：

```
我家是在安徽的一个小县城，你可能没听说过。
```

或者：

```
我是做咨询工作的（或者说互联网的）。白天在公司里被我们老板各种各样地折腾，晚上下班之后在健身房被我的私教猛力地折腾。对了，我们的老板也姓张，我现在满脑子都是张总。
```

女生可能又会被这种笑话逗笑。这时应该有5到8分钟的互动交流。如果女生还没有主动提出离开，说明她可能没有太多别的安排。你就可以尝试邀约她到旁边找个地方坐下来聊一聊，喝杯东西，直接进入约会阶段。

## 收尾阶段

如果你提出邀约被女生委婉拒绝，她说：“不用了，我还约了朋友，现在该走了。” 此时就应该掏出手机，提出留下对方的联络方式，比如微信号。

```
那我们加一下微信吧，后续有时间可以在微信上交流。
```

女生一般会答应。之后在微信上聊天时，你会发现相对而言，她聊天的窗口或上手程度是很容易的。因此，在搭讪中收号其实是收尾阶段该做的事情，而不是应该在搭讪开始就做的事情。

有的同学可能有一个误区，认为搭讪的目的就是为了收号。其实搭讪的目的是为了了解、沟通、建立吸引，并进行二次邀约和约会。相信很多同学也有体会，有时搭讪收了号，但在微信上根本无法进行有效聊天。原因就在于搭讪时根本没有给对方留下太多印象。所以，只是为了要号码而去搭讪，而不是为了认识这个女生而去搭讪，那么即使要过来号码，也是废号与死号。除非你的朋友圈建设得非常精彩、有吸引力，否则这些号码收回来是没有任何意义的。

## 搭讪的评分标准

搭讪分为四大阶段：开场阶段、过渡阶段、交流阶段和收号阶段。那些“10秒收号”、“30秒收号”的，往往都是开场之后紧接着就开始收尾了，相当于跑上去说“你好，再见”，看似很厉害，其实没有任何用处。

以下是搭讪的评分标准：
*   **90分的搭讪**：突破自我进行搭讪，和女生沟通，邀约女生并获得女生的联系方式。完成这四个任务。
*   **80分的搭讪**：突破自我进行搭讪，和女生沟通，获取女生的联系方式。完成这三个任务。
*   **70分的搭讪**：突破自我进行搭讪，获得女生的联系方式。完成这两个任务。
*   **及格**：只要完成“突破自我进行搭讪”这一个任务。

值得肯定的是，毕竟战胜了自己的紧张、焦虑和恐惧，主动迈出了这一步。相信随着练习的增加，成绩也会越来越好。大家可以参考子明给出的搭讪模板，根据实际情况灵活应用，不断调整和总结，逐渐做到可以跟陌生人自然、自在、自信地交流。
