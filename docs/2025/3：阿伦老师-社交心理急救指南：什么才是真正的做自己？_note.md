# 社交心理急救指南：3：什么才是真正的做自己？💡

在本节课中，我们将要学习“做自己”这一核心概念。许多人感到不快乐，正是因为未能真正地做自己。我们将探讨无法做自己的原因，并详细解析“真正的做自己”所包含的三个关键层面：不屈服于恐惧、拥有选择的自由，以及对自我负责。理解并实践这些要点，是获得内心快乐与自由的基础。

## 为什么你无法做自己？🤔

上一节我们概述了课程目标，本节中我们来看看人们无法做自己的根本原因。每个人在生命初期都曾“做过自己”。人在出生时如同一张白纸，其成长深受环境（如父母、亲友）的影响，逐渐形成独特的风格、性格与行为模式。

在幼年时期，人们会毫无保留地表达自己的需求、想法和感受。然而，并非所有的表达都能得到认同或采纳。当表达被认同时，个体的自恋（即认为自己还不错的感觉）得到满足，从而相信做自己是好的，并敢于继续做自己。

反之，若在成长过程中，个体发现做自己非常困难，甚至其家庭与环境阻碍自我表达，他就会逐渐变得不敢或难以做自己。因此，无法做自己的核心原因在于：**环境不允许**。我们最初拥有做自己的愿望和能力，但在成长过程中，这个愿望被隐藏了起来。

生活中，无法做自己的人通常有两种极端表现：过度自我压抑或过分释放自我。

以下是这两种表现的具体后果：

*   **过度自我压抑**：导致真实想法无法表达，形成巨大的内心冲突。这种冲突可能在不恰当的场合爆发，从而破坏人际关系。
*   **过分释放自我**：例如脾气急躁暴躁的人，容易伤害身边的人，同样会破坏关系。

这两种状态都不是真正的做自己。无法做自己的人在生活中常感到无奈和不快乐，无法体验到属于自己的自由。相反，有些人能够恰到好处地展现自己，这种状态被称为“中庸”——一种平衡且适宜的状态。正确的自我展现能带来快乐，错误的展现则让人难以感受幸福。因此，我们需要学习如何真正地做自己。

## 真正的做自己：三大核心要素 🔑

在理解了阻碍之后，本节我们将深入探讨“真正的做自己”所包含的三个具体要素。

### 1. 不屈服于内心的恐惧感 😨

每个人内心都存在恐惧，例如害怕公开演讲、害怕冲突、害怕挑战或害怕面对父母。一旦被恐惧支配，我们的行为就会“变形”。

**行为变形**指的是：该说的话不敢说，不该说的话却强迫自己说；该做的事不敢做，不该做的事却强迫自己做。

以下是行为变形的常见例子：

*   **该做不敢做**：拥有晋升机会却不敢毛遂自荐。
*   **不该做却强迫做**：经济拮据却每次聚会都强迫自己买单；不想借钱却不敢拒绝。
*   **因恐惧而扭曲行为**：在社交场合因害怕被看出紧张而完全沉默缩在角落；或因害怕被视作不合群而强迫自己进行不自然的夸夸其谈。

这些行为都源于我们害怕破坏关系、害怕不被认可。那么，如何做到不屈服于恐惧呢？关键在于**察觉并区分“该做”与“不该做”**。

*   **察觉恐惧**：当你感到特别焦虑、无助或被动时，就意味着恐惧出现了。
*   **该做的，再怕也要做**：例如，路上觉得应该打招呼，尽管害怕也要去做。
*   **不该做的，不必强迫做**：例如，在尴尬环境中，如果没有责任去打破僵局，就不必强迫自己说话，安静待着即可。强迫自己做不愿做的事，反而显得更不合群。

一个典型的“行为变形”例子是：追求异性时，因对方未及时回复信息，就因担心而不断发送质问信息。这会让对方感到奇怪、不舒服甚至有被侵犯感。**真正的做自己，是把恐惧感放到一边，该做的事即使再难也去做。**

### 2. 做自己是一种自由 🕊️

做自己意味着拥有**自由**，即一种有选择的生活状态。你可以自己决定想做什么、不想做什么，以及自己想成为谁，前提是不伤害他人。

**自由 = 自主选择权**

例如，你因故无法参加聚会，可以选择推迟或拒绝。不必因为害怕别人认为你“反复无常”而勉强自己。生活是动态的，变化是正常的。真正的做自己，不应被这些担心所限制。

自由的核心是**懂得说“不”**，并对不必要的事情说“不”。同时，也需要接受生活中的不确定性。

当然，自由不意味着固执己见。在坚持自我的同时，可以寻求折中的办法，前提是让自己内心舒服。

以下是寻求折中的例子：

*   已约好谈事却临时有要事，可以提议：“能否改到今晚或明天？”
*   或者询问：“是否可以通过电话讨论？”
*   也可以商量：“是否方便到你这边来？”

关键在于，你没有攻击或折磨他人。像拒绝、改变主意这类行使个人权利的行为，并不构成伤害。**真正的做自己，是拥有自由度、能进行选择性表达的自我。**

### 3. 做自己是对自己的负责 ✅

许多人误以为做自己就是放任自我，例如睡到中午、随心所欲地玩乐。但这并非真正的做自己。

**真正的做自己，是对自己的一种负责。** 它意味着更好地为自己的人生负责，从而赢得更多选择的机会和更大的自由。这需要懂得爱自己，让自己变得更好。

这与心理学中的“森田疗法”核心理念“顺其自然”有相通之处。许多人错误地将其理解为“任其自然”——对社交恐惧或强迫症等症状放任不管，不追求更好。但这其实是“放任自由”。

**真正的“顺其自然”是：** 接受我们当下的状态（包括问题），将其视为自然的一部分，然后**带着这些状态，去努力做好该做的事**，让自己变得更好。这是对自己的心态和未来负责。

因此，做自己并非想象中的浅层洒脱。它建立在一定的自我克制与负责的基础上，从而让我们体会到更深层次的洒脱与自由：不屈服于恐惧、自主决定人生、努力变得更好。表面上，我们失去了浅层的放纵自由，但实际上，我们收获了**更有价值、更有意义的深层自由**。

## 总结 📝

本节课中，我们一起学习了“真正的做自己”这一主题。

首先，我们探讨了**无法做自己的原因**：成长环境中对自我表达的阻碍，使得我们隐藏了做自己的愿望。

接着，我们详细解析了真正的做自己所包含的三大核心要素：
1.  **不屈服于内心的恐惧感**：察觉恐惧，区分并勇敢去做“该做”的事，同时不必强迫自己做“不该做”的事。
2.  **做自己是一种自由**：拥有自主选择的权利，懂得在必要时说“不”，并能在坚持自我时灵活寻求折中方案。
3.  **做自己是对自己的负责**：并非放任自我，而是接受当下，并主动承担责任，努力让自己变得更好，从而获得更深层次的自由。

实践这些原则，将帮助你更真实、更快乐地生活。我们下节课再见。
