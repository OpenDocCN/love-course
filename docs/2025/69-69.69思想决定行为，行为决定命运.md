# 69-69.69思想决定行为，行为决定命运

大家好，欢迎来到动车网邪淫系统自救指南视频课程，我是清风。本节课的主题是思想决定行为，行为决定命运，一个人的命运，和他的思想有密切的因果关系，就像植物都是从一粒种子开始的一样。

我们每个行为也必定是从一粒思想的种子开始萌发的。如果说我们的思想是一颗种子，那么从种子上萌发的精就是行为，而行为所带来的命运也就是这颗种子接触的果实。一个人思想上自私自利，行为上就把自己看得很重。

不顾他人感受，他的命运就必然容易遭受众人的反感，不被人喜欢。一个人思想上懒惰散漫，行为上就什么事都不想干，他的命运就必然一事无成，收入低下。一个人思想上喜欢抱怨，行为上就喜欢整天抱怨诉苦，他的命运。

就会越来越倒霉，因为他内心都是怨气，是满满的负能量，所以会更容易吸引不好的事情进入到他的生命中。这个原理前面讲过的，所以你看思想是种子，行为是种子上萌发的主干，命运就是果实，这时他们三者的关系。

那么请问如果一个人过得不好，他活得很痛苦，根源是什么？根源就是他的心里有不好的种子，这个人痛苦的命运，就是他心里的那些不好的种子所结出来的果实，现在再来看，我们前面讲了很多阴影，自私、贪婪功利狭隘。

抱怨，我直傲慢、虚荣，懒惰，一共是9个。这些都是不好的种子，对吧？这些种子都会直接间接的导致邪引，他们会造成痛苦的命运。人的心灵就像一块耕地。就看你是怎么去管理，怎么去耕耘自己的内心。

让内心播种正确的种子，最后接触好的果实。而前面我还讲了无私清近、淡薄豁达感激、无我超越自我、谦虚、踏实、劳动，这相反的9种价值观。我们要在这块地清除那些不好的错误的思想，培育正确的好的思想。

就像一个农夫拔除杂草，培育庄稼，从自私贪婪、功理、狭隘、抱怨，我直傲慢、虚荣、懒惰，转变为无私清近、淡泊豁达感激、无我谦虚、踏实、劳动，去实现这种转变。这就是在重塑自己的底层价值观，重塑自己的人格。

实现深刻的转变。看看这几张图自。丝之草贪婪之草公里之草可以看到，这些草的草根盘根错节，它的根系非常的繁盛，扎根到了心灵的深处，而拔草就是要把这些根系呀连根拔除。为什么说反省自己一定要足够深刻呢？

因为你反省不深刻，下面的野草根就除不尽。它就总是会继续发芽，长成大树，结出不好的果实。拔出自私之草，贪婪之草，宫里之草，狭隘之草，抱怨之草，我植枝草傲慢之草，虚荣之草，懒惰之草。

一定要做系统深入透彻的反省，才能够把这些草给拔出来。把这些草拔出来后，还要在内心播撒正确的种子。这些种子就是无私清净，淡薄豁达感激，无我谦虚踏实劳动，这些在前面都讲过。无我就是超越狭隘的自我中心思维。

超越小我在自己内心培育这些正确的思想，用这些好的思想来指引自己的行为。前面我们讨论过思想决定行为，行为决定命运，由此可见，一个人的处境是由自己内心的思想所决定的。所以当他的思想发生转变的时候。

他的处境也会相应的发生变化。种瓜得瓜、重豆得豆。好的思想会接出好的果实，而坏的思想会接出坏的果实。像前面讲的自私贪婪、功利、狭隘抱怨，我直傲慢、虚荣、懒惰，这9个都是属于低能量级的思想。

而无私清近、淡薄豁达感激，无我谦虚，踏实劳动是高能量级的思想。低能量级的思想特点时。自私阴暗邪恶欲望，而高能量级的思想特点是无私、光明、善良、清净。一个谐淫的人要从低能量级向着高能量级去跃迁去转变。

不能总是呆在自私，阴暗，邪恶的低级能量场当中。如果这个人他慢慢开始转变了，从低能量级慢慢变为高能量级，那么他的谐淫问题一定会减轻，乃至消失的。最后用一句话结束本节课的主题。

自己播下的种子必定会在自己身上开花结果。这个种子就是思想。戒色是有系统方法的，我相信人一定可以通过戒色改变命运。本节课就到这里，非常感谢大家的支持，我们下节课再见，拜拜。
