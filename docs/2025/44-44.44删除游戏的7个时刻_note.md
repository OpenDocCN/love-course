# 动车网谐淫系统自救指南：44：删除游戏的7个时刻 🎮

在本节课中，我们将学习识别玩游戏的欲望降至最低点的七个特定时刻。这些时刻是放弃游戏、果断删除的最佳时机。

## 概述

游戏虽然有趣，但也存在令人厌倦的一面。当游戏体验从娱乐转变为负担时，便是考虑退出的信号。本节将详细阐述七个这样的关键时刻，帮助你理性决策，远离游戏。

## 七个删除游戏的时刻

上一节我们概述了课程目标，本节中我们来看看具体是哪七个时刻。

### 1. 任务冗长时刻

绝大部分游戏都存在任务流程冗长的设计，要么是剧情拖沓，要么是需要不断重复刷取资源。例如《暗黑破坏神》这类“刷子游戏”，需要不断刷怪才能获得好装备。这个过程本身就是一种煎熬。

记得以前玩《魔域》，从110级升到125级需要持续刷怪近七八个小时，过程痛苦且毫无意义，感觉自己像一台刷怪机器。当游戏玩到这种程度，它就变成了一种折磨，作业感太强，如同辛苦劳作。在这种情况下，应果断删除游戏。

### 2. 需要大量重复努力的时刻

想要精通一个游戏，通常需要付出大量精力和时间进行重复练习。例如在《绝地求生》中练习枪法和身法。许多玩家甚至将昵称改为“苦练三指”、“苦练五指”，这使游戏变得与写作业没有本质区别，过程折磨且痛苦。

当游戏变成一种工作时，它就不再是娱乐，而是痛苦的来源。此时可以果断删除游戏。

### 3. 不好玩时刻

毁掉一首歌最好的方法就是单曲循环，游戏也是如此。玩一个游戏越久，越容易感到厌倦。例如《暗黑破坏神》玩多了会觉得没意思，角色技能固定，只剩下无尽的刷怪。《绝地求生》同样如此，一张地图玩了几百遍后，一切都会变得乏味。在游戏变得不好玩的时刻，便可以将其删除。

### 4. 总是输的时刻

如果玩游戏总是输，无法战胜对手，会让人感到烦躁。在这种情况下，游戏就失去了乐趣。例如玩《绝地求生》，开局不久就被淘汰，反复“落地成盒”或在途中被伏击，整天都在“送人头”，毫无体验可言。

这时应该明白，继续玩下去就是自虐。何必用自己的时间和精力去满足他人的胜利欲望，主动找罪受呢？这种情况下可以删除游戏。

### 5. 总是赢的时刻

总是轻松获胜也会让游戏变得索然无味。例如以前玩《DOTA》新手模式，一局能击杀十几二十个对手，因为对方太弱，每局都赢毫无悬念。再比如用修改器玩《暗黑破坏神》，将角色改为99级，出门就能秒杀所有怪物。

如果你想迅速降低对某个游戏的欲望，可以尝试使用修改器，将金钱、等级、属性等全部调到最高。通常不出一个小时，你就会对这个游戏失去兴趣。

### 6. 玩腻了的时刻

再好玩的游戏，玩到最后也难免会腻。很多内容都是重复的：重复的剧情、重复的人物、重复的地图。重复体验多了，自然会感觉没意思。当游戏进入这种“玩腻了”的状态时，你就可以考虑删除了。

### 7. 反省时刻

这是删除游戏最直接的时刻。当你连续玩了3到5个小时的游戏，抬头发现已是凌晨一点，内心涌起一阵后悔，质问自己“我干了什么？”，这就是反省时刻。你意识到自己浪费了时间，没有按时休息。

另一种情况是前一天熬夜玩游戏到凌晨两点，第二天起床后精神萎靡、脸色暗淡，然后懊悔“昨晚真不该打游戏”。在这种反省的时刻，应当立即删除游戏。

## 总结与理性认知

本节课我们一起学习了玩游戏的欲望降至最低的七个时刻。我们的游戏欲望并非一成不变，在这些时刻，欲望最弱，继续游戏无异于折磨。因此，这七个时刻是放弃游戏的最佳时机，该卸载就卸载，该删除就删除，该注销账号就注销账号。

有的人可能会说，删除后还会重新安装。这的确是常见情况，但至少从删除到重新安装之间有一段空白期，在这段时间里你不会玩游戏。

删除游戏后，请建立以下理性认知：

*   **我意识到，我不需要玩游戏。** 因为游戏并不能给我带来任何成长和价值。
*   **我意识到，游戏其实也不好玩。** 玩多了就会发现内容重复，无聊且没意思。
*   **我意识到，游戏从未让我变强，而是让我变弱。**
*   **我意识到，在游戏中投入大量时间、金钱和精力，最终不过是游戏厂商收割的“韭菜”。**
*   **我意识到，在游戏中的“辛苦劳作”毫无意义。** 不如在现实中好好工作和学习，结果都是看得见的。
*   **我意识到，我不需要通过游戏来填补内心空虚。** 没这个必要。
*   **我意识到，成功不是在游戏里杀了多少人，而是在现实中帮助了多少人。** 我的时间应该用来做有价值、有意义的事，而不是沉迷游戏，让游戏主宰我的生活。

建立这样的理性认知，将有助于你远离游戏。戒除不良习惯是有系统方法的，我相信人一定可以通过自律改变命运。
