# 白天沟通：06.拥有被讨厌的勇气

你的问题是别人的解决方案，很多时候啊我们的问题都是我们的期待。我们要真正面对的不是问题，而是自己的期待。当你认为很严重的问题，有的时候是别人的解决方案，举个例子啊，老公抽烟，你觉得是个大问题。

但是你老公觉得他是一个解决方案，他在工作上释放压力的方案。孩子逃课，你觉得是个大问题，但是你孩子觉得他是解决方案，解决他在学校里面被欺负的方案，所以啊你眼中的大问题，都是别人想了半天的解决方案。

包括他嫖娼，他找小三，是他解决性需求，情感需求的方案。当你老公跟你说工作压力很大，你习以为常，一抽烟了，你非常不高兴。哎，当你老公说性需求不满，你不觉得有啥，但他一出轨呢，你五雷轰顶。对不对？

所以啊谁痛苦谁改变他乱丢袜子，你很痛苦，你看到了你不开心，你改变他看到了你的痛苦，改变现实，符合期待，调整期待，符合现实。当然，他看到你比较痛苦。他可以为了你而改变。但这不是对错的问题。

不是做错了而调整，而是看到你难受而改变。如果说做错了，他要道歉。但是如果他没错，他改变了，你要感谢，拥有被讨厌的勇气。其实啊朱莉老师这他沟通公式里面，所有的方法都是树，而道德层面，最好的情绪价值。

那就是允许的能力，接纳的能力，允许别人的离开，接纳他没有那么爱你，接纳父母亲的重男轻女，接纳子女的平庸，这可不是让你摆烂啊，而是不把自己的情绪寄托在他人身上。他离开了你，你依然可以过得很好，很幸福他。

不爱你，你依然值得被爱，父母重男情女。没关系，他们的观念而已，我知道我不比男人差，子女平庸学习不好，也不代表我们的教育失败，更不代表子女没用，他们是独一无二的不可替代的存在。我们最终过的啊，不是说哎呀。

我比谁更能言善辩，更能说会道，哎，更能做更能干，而是我拥有把这一生过得幸福，过得快乐的能力，对吧？这种能力得长在我们自己身上。A，我拥有我跟谁过日子，我都能开心，都能幸福，谁跟我来过日子。

他也都能够感受到开心和幸福，对吧？这样子我们才不把希望拖在别人身上，感觉，哎，我跟A过日子，我就开心，跟B过就不开心，那么我得永远害怕失去A，对不对？但是如果说A我是开心的原体，我是幸福的源泉。
