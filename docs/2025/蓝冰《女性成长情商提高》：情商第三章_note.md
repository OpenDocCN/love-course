在本章中，我们将学习如何通过自我对话和具体的行为策略，来克服内心的脆弱、焦虑等低能量状态，实现自我激励与成长。核心在于练习内心对话，并针对六种常见的情绪困境，提供可操作的解决方案。

## 内心对话：与自我、本我、超我交流 🪞

上一节我们介绍了情商低的外在表现是内心脆弱。本节中我们来看看如何通过内心对话变得强大。

想让自己变得强大，必须练习内心对话的技能。你要学会用超我、自我和本我去对话。开始的时候，你可以站在镜子面前，把镜子里的你当做**本我**，即原始的我。而真实的自己，就是**自我**。同时，你也可以扮演成**超我**的角色，即理想中理智、完美的自己。

超我的作用是管控行为、调节情绪、反思想法。

## 识别并克服阻碍强大的因素 📝

接下来我们来看，在你内心脆弱和无助的时候，哪些行为思想阻碍了你变得强大。我们从这里下手，请准备好笔记本记录。

以下是六种常见情绪困境及应对方法。

### 1. 处理自怜、焦虑与抑郁

当你遇到自爱自怜、焦虑、忧郁或抑郁时，这种情绪很常见。它会浪费你的时间，产生负面情绪，甚至引导噩梦成真。长期陷入这种情绪，会让你忽视生活中的美好，并影响到人际关系。

处理这种情绪，给你提供两个方向。

**方向一：采取积极行动**

以下是两个具体行动建议：
*   **帮助他人**：想办法去帮助别人，哪怕是微小的行为。例如，在路上帮别人推一下车、搭把手。在做自己的工作或家务时，积极地去做，挑战一次自己做事的效率。告诉自己：`只要比平时做事快一倍，状态就会好一倍`。
*   **写感恩笔记**：回顾这么多年来你心存感激的人和事。你会发现，在这个过程中你能够找到幸福、感恩和克服焦虑的方向。

以上行为需要反复去做，才能改变焦虑抑郁状态。

**方向二：替换消极想法**

找到那些引起你焦虑的事情。从多个角度考虑你现在的处境是否真的那么糟糕。然后，想象一下你崇拜或欣赏的人面对这种问题时会怎么办、怎么想、会给你什么建议。接着，用那些切合实际的想法去替代引起你焦虑的想法，积极解决问题，并保持乐观状态。即使你不喜欢，也要这样做。

**举例**：引起你焦虑的想法是：“我这样继续下去，他会不会不爱我？”或“我这样继续下去，明年是不是没有收益了？”。比你优秀的人可能会想：“只要我努力付出，一定会有好结果”或“我今天的一份付出和积累，明天一定会有成果”。他们的想法是正向的。你焦虑的原因，往往是此刻没有全力以赴去做可能产生好结果的事情。当你完成这些步骤，就能更理智客观地看待处境，减少焦虑。

### 2. 摆脱他人对心情的影响

在生活中，你经常会被别人的一句话、一个眼神、评价或拒绝影响心情，让你失去愿景，甚至破坏别人在你心中的形象。这都是自我低价值的表现。

如何摆脱这种现象？以下是几个解决方案。

**解决方案列表**：
*   **使用有选择权的语句**：学会使用有选择权的语句与他人及自己沟通。例如，把“他这样看我，我该怎么办？”的想法，改成：“**我是一个什么样的人？我将展示出什么状态？我这样做是因为XXX。他看不懂我，所以我选择放弃/遗忘/不跟他计较。**” 多练习几遍，心态就会改变。
*   **建立健康的交往底线**：明确什么样的人坚决不交往，而不是不懂拒绝、来者不拒。要学会对自己的情绪、精力和时间负责。
*   **积极回应并学会原谅**：学会用积极理智的决定回应他人，并学会原谅他人。客观分析别人的反馈和批评，不要着急下结论。不管别人是否道歉，先学会原谅。可以用自我对话告诉自己：“我是一个大度、有格局的人。”这种精神胜利法在这里特别有效。

### 3. 克服对改变的恐惧

当你非常害怕改变，总会担心很多，往坏的方面预测，稍作尝试就退回。这些改变包括尝试性、情绪、习惯、行动、认知上的改变等。不想改变的后果是：你经常会困于泥潭不能自拔，无法学习新知识，生活无法变得更好，无法养成更健康的习惯，经常被人甩在身后。时间越长，你越难以改变。

打破这一切的方法很简单，只需要5个步骤。

**改变的五个阶段**：
1.  **无意识的小改变**：例如，把口头禅“哎呀”换成“OK”。
2.  **深度思考**：思考“我为什么要改变”。
3.  **制定策略**：准备改变的计划和方法。
4.  **开始行动**：行动起来。
5.  **保持习惯**：保持改变的状态和习惯。

具体操作时，首先确认你改变的意愿有多强（1-10分打分）。如果超过6分，就进入第二阶段：建立可行性计划，把注意力放在小的改变上，例如“明天早起20分钟，坚持7天”。在这个阶段，每月设立一个目标，考虑清楚想要取得的进展，制定基于能力的计划，确保每天都有进步。同时，在本子上预测可能遇到的困难及解决方法。建立责任意识（为什么要做），最后记录每天的进程，坚持写7天改变日志。你肯定会遇到问题，但要用理智综合情绪，决定是否需要继续改变，并评估改变失败的风险。可以做一个表格，评估维持现状和改变以后的性价比。最后，给自己定一个远期愿景计划，想象你想成为哪类人以及其样子，在心理上建立一个心理镜像，然后去执行。

### 4. 停止关注无法控制的事情

经常把注意力放在无法控制的事情上，尤其是女性，这通常是内心自我控制失衡的表现。人的内心有内控和外控两种状态。**内控**认为可以掌握人生的一切，自己的行为决定一切。**外控**（宿命论）认为很多东西被命运、运气主宰，努力无济于事。当你进入内控状态却又发现很多事情无法控制时，就会产生焦虑，甚至产生更强烈的罪恶感，浪费时间和精力，摧毁人际关系，对别人刻薄，提出无理的高要求。例如，很多女人试图控制男人的行为或想法，这通常是徒劳的。

解决方法有以下几点。

**解决方案列表**：
1.  **分配任务**：把手头的任务和职责分配给别人。
2.  **请求帮助**：需要帮忙时，学会请求别人帮助，不要自己逞强。
3.  **聚焦可控**：把注意力放在可以控制的事情上。
4.  **影响而非强迫**：学会影响别人，而非强迫别人。
5.  **找到平衡点**：找到内外控制的平衡点。有些东西确实无法改变（如性别），有些东西需要自我掌控（如努力带来好结果）。
6.  **合理归因**：不要把结果全部归罪于或归功于自己，你承担不了那么大的责任。

如果你能做到以上内容，就不会为无法控制的事情而焦虑。

### 5. 不再纠结于过去

喜欢纠结于过去，是所有女性在情绪和情感中容易把自己拖入泥潭的一种状态。总是在想“为什么以前对我这么好”，这会使你错过现实的美好，陷入情绪无法规划未来，降低决策力，产生抑郁情绪。美化过去对健康有害。要知道，每一个现在就是将来的美好过去。

陷入这种情绪后，按照以下方法去做，就能摆脱焦虑状态。

**行动步骤**：
1.  **回顾与提炼**：拿出本子和笔，适当回顾过去，从中提炼经验和教训。写下哪些是你看错的、轻信的、怀疑的。然后告诉自己：“我要往前走。”即便过程很痛苦，因为你回不到过去。
2.  **积极斗争**：如果还在纠结，用积极的态度与这种情绪做斗争，这样才能聚焦现实，筹划未来。想着明天、后天、未来你是否会后悔。如果不想为今天后悔，现在就应该承担昨天做错事带来的苦果。这叫理智调整。
3.  **调整思路**：对于过去发生的事情，写下事情的本质，不要在乎自己的痛苦、悲伤或后悔情绪。
4.  **与过去和解**：不断重复告诉自己：“今天的你是 **1/3的过去 + 1/3的现在 + 1/3的未来规划** 构成的。你愿意为过去的1/3而毁掉未来的2/3吗？”

反复进行以上的文字记录和潜意识自我对话，不出7天，你就能摆脱这种状态。

### 6. 避免重复犯同一个错误

你经常会犯同一个错误，在同一块石头上反复跌倒。遇到这种情况，你通常无法达成目标，没办法解决问题，并且会怀疑、质疑自己，不再努力，让周围监督你的朋友灰心丧气。这时你可能会建立非理性价值观，为犯错找借口，从此颓废。这也是很多女性在情感道路上经常遇到的问题。

运用下面的方法可以解决。

**纠正步骤**：
1.  **明确责任**：每当你犯了一个错误后，明确自己在这个错误中应负的责任。
2.  **制定防重蹈计划**：写下一个能防止你重蹈覆辙的计划。
3.  **识别原因与征兆**：找到犯错的原因，并能辨别出重蹈覆辙的征兆，这点非常重要。
4.  **练习改变与自律**：从第三步开始，练习改变习惯，让自己变得自律。首先练习忍受不适应的感觉，然后练习每天铭记目标，练习为自己设立限制（哪些能做/不能做，能想/不能想）。最后，列出你不想重蹈覆辙的所有原因，并不断激励自己（潜意识对话）。

每次犯错后，重复以上操作3到5次，你就不会再重蹈覆辙。这些都是经过大量实践得出的结论。
