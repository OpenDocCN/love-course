# 核心群系统资料：03：腾讯课堂聊天实战讲解（直接播放版）：朋友圈建设：第二课：常用APP修图实战案例

## 📖 概述

在本节课中，我们将学习手机修图的完整流程和核心软件的使用方法。课程将详细介绍几款必备的修图APP，包括它们各自的核心功能、使用顺序以及如何协同工作，最终将一张原始照片处理成适合发布在社交媒体上的高质量图片。

## 📱 第一节：核心修图APP介绍

上一节我们了解了修图的重要性，本节中我们来看看具体需要使用哪些软件。以下是几款在手机修图领域评价很高且使用频繁的APP。

1.  **Adobe Lightroom (LR)**
    *   这是后期修图调色的主力软件，在网络媒体上非常流行，无论是调整网红色调还是自然色调都非常出色。它是Adobe公司（Photoshop的开发商）出品的手机端软件，在手机修图领域占有统治地位。

2.  **VSCO**
    *   这款APP主要用于添加胶片风格的滤镜。在一张图片经过LR进行基础调整后，可以导入VSCO直接添加滤镜或微调参数，使图片快速成型。对于新手，常将LR和VSCO结合使用。

3.  **Facetune**
    *   这款软件的核心功能是**磨皮**，即对皮肤进行处理使其光滑。它在安卓和苹果系统上都能提供非常细腻、专业的磨皮效果，主要用于后期的人物皮肤修饰。

4.  **美图秀秀**
    *   这是一款入门级软件，使用者众多。它的核心优势在于**脸部塑形和身材拉伸**，其瘦脸、瘦身、调整五官细节的功能比前几款软件更强大、易用。

5.  **MIX**
    *   这款APP主要用于**添加天空、特效、海报文字和字体效果**。当需要对图片进行深度美化，如添加天空、制作海报或添加艺术字时，会用到它。

6.  **Retouch**
    *   这款软件的核心功能是**修复污点和移除穿帮物体**。例如去除脸上的痘痘，或移除照片背景中多余的胳膊、扫帚等杂物，它的处理效果非常出色。

7.  **泼辣修图**
    *   这款软件功能全面，包含图片拉伸、局部调整等功能。但其流畅度可能不如前述软件，使用频率相对较低，可以根据需要选择安装。

## 🛠️ 第二节：Lightroom (LR) 核心功能详解

上一节我们介绍了各APP的用途，本节中我们重点看看最核心的软件Lightroom的具体使用方法。LR是专业级的调色软件，其调整效果非常精准。

### 软件导入与界面

安装并打开LR后，点击右下角的“**+ 照片**”按钮即可导入图片。也可以通过手机图库选择照片，使用“**分享**”功能，选择“**添加到Lightroom**”来导入。

导入照片后，进入编辑界面。下方有一排工具，其中“**可选择**”工具（局部调整工具）和“**亮度**”、“**颜色**”等整体调整工具最为常用。

### 局部调整工具详解

“可选择”工具是LR的强大功能，允许你对图片的特定区域进行精细调整。它包含三种模式：

1.  **画笔工具**：手动涂抹出需要调整的区域。
2.  **径向渐变工具（椭圆）**：创建一个椭圆形调整区域，可移动和缩放。
3.  **线性渐变工具**：创建一条线性的调整区域，适合调整天空、地面等。

**操作逻辑**：用红色蒙版（门板）标出的区域，就是调整会生效的区域。未被红色覆盖的区域则不受影响。

*   **调整画笔大小**：选中画笔工具后，拖动滑块可以改变画笔的尺寸。
*   **擦除错误区域**：如果涂抹到了不想调整的地方，可以使用“**橡皮擦**”工具擦除多余的红色蒙版。
*   **调整参数**：创建选区后，下方会出现调整参数，如曝光、对比度、高光、阴影、色温、饱和度等。拖动滑块即可看到选区内的变化。
*   **删除调整点**：长按画面中蓝色的调整点，选择“**删除**”即可移除该局部调整效果。
*   **对比效果**：长按图片可以查看调整前（按下）和调整后（松开）的对比。

**公式/核心概念**：
*   **高光**：图片中最亮的区域。
*   **阴影**：图片中最暗的区域。
*   **白色色阶**：影响画面中较亮的大范围区域。
*   **黑色色阶**：影响画面中较暗的大范围区域。

### 整体调整工具详解

退出局部调整模式后，我们使用下方的“**亮度**”、“**颜色**”等工具对整张图片进行调整。这是修图的基础步骤，目的是恢复图片的正常影调和色彩。

1.  **亮度调整**：
    *   **曝光**：控制整体明暗。
    *   **对比度**：增加明暗反差。
    *   **高光/阴影**：分别压暗过亮的高光区域，提亮过暗的阴影区域，以恢复细节。
    *   **白色色阶/黑色色阶**：进一步微调亮部和暗部。

2.  **颜色调整**：
    *   **色温**：向左（蓝色）是冷色调，向右（黄色）是暖色调。
    *   **色调**：向左（绿色）偏移，向右（洋红色）偏移。
    *   **自然饱和度**：智能地增加色彩鲜艳度，效果更自然，**推荐优先使用**。
    *   **饱和度**：整体地增加或减少所有颜色的鲜艳度，容易导致失真。

3.  **颜色混合（HSL）**：
    *   这是LR的精华功能。它将图像颜色分解为**红、橙、黄、绿、浅绿、蓝、紫、洋红**8个通道。
    *   你可以单独调整每个颜色的**色相**（改变颜色本身）、**饱和度**（颜色鲜艳度）和**明亮度**。
    *   **应用**：例如，想让天空更蓝，就调整“蓝色”的饱和度和明亮度；想让人物肤色更白皙，可以调整“橙色”的明亮度（提亮）和饱和度（降低）。

### 裁剪与导出

*   **裁剪工具**：可以自由裁剪、旋转图片，或使用“拉直”功能校正倾斜的水平线。
*   **导出图片**：点击右上角的分享箭头，选择“**导出**”。在设置中，**格式选择JPEG，尺寸选“最大可用尺寸”，品质选100%**，以保证最佳输出效果。

## 🎨 第三节：VSCO 滤镜应用

上一节我们使用LR恢复了图片的基础色调，本节中我们使用VSCO为图片添加风格化滤镜，提升照片的质感。

1.  将LR导出的图片，通过“分享”功能导入VSCO。
2.  双击图片进入编辑界面，下方有丰富的滤镜库（如A系列、C系列、G系列、V系列等）。
3.  左右滑动尝试不同滤镜，点击喜欢的滤镜即可应用。
4.  点击滤镜缩略图中间的圆圈，可以调整该滤镜的**强度**（0%-100%）。
5.  点击下方菜单栏，可以进行更精细的**微调**，如对比度、饱和度、锐化等。
6.  **色调分离**是VSCO的特色功能：
    *   **阴影色调**：为图片的暗部添加一种颜色（如冷调的蓝色）。
    *   **高光色调**：为图片的亮部添加一种颜色（如暖调的黄色）。
    *   合理使用色调分离（如高光加暖、阴影加冷）可以增加图片的色彩层次和高级感。
7.  调整完毕后，点击保存，图片会自动存入手机相册。

## ✨ 第四节：Facetune 专业磨皮

经过调色和加滤镜后，本节我们进行人像精修的关键一步——磨皮。Facetune在这方面效果出众。

1.  将图片导入Facetune。
2.  选择工具栏中的“**平滑**”功能（而非“更加平滑”，后者效果过重）。
3.  将图片放大，用手指在需要磨皮的皮肤区域（如脸颊、额头、脖子）轻轻涂抹。
4.  **注意**：避开有清晰轮廓的部位，如眼睛、眉毛、嘴唇、鼻梁。如果不小心涂抹到，可以使用“**修复**”工具擦除恢复。
5.  磨皮程度通过涂抹次数控制，涂抹越多效果越明显，需适度。
6.  完成后点击右上角对勾，然后点击保存。

## 💃 第五节：美图秀秀 塑形与液化

本节是修图的最后一步，主要针对人像进行面部和身材的精细塑形。

1.  将图片导入美图秀秀，进入“**人像美容**”模块。
2.  使用“**面部重塑**”功能，软件会自动识别五官，你可以滑动调整脸宽、下巴长度、眼睛大小、鼻翼大小等参数，这是智能调整。
3.  更精细的调整使用“**瘦脸瘦身**”功能。
    *   选择合适大小的画笔（推荐中号）。
    *   在需要调整的部位（如推窄脸颊、收细胳膊、拉长腿部）轻轻向内推动。
    *   **注意**：调整要符合人体比例，避免失真。
4.  如有痘痘等小瑕疵，可使用“**祛痘祛斑**”功能点掉。
5.  调整满意后，软件会自动保存高清图片到相册（需在设置中开启“高清画质”）。

---

## 🧹 第六节：Retouch 移除杂物

在修图过程中，如果发现照片中有多余的物体（如路人、电线杆、瑕疵），可以使用Retouch进行移除。

1.  将图片导入Retouch。
2.  选择“**快速修复**”工具。
3.  用手指涂抹想要移除的物体。
4.  松开手指，软件会自动计算并填充周围背景，实现无缝移除。
5.  可以使用“**前后对比**”按钮查看效果。
6.  完成后，选择“**保存为副本**”。

## 🔄 第七节：完整修图流程实战案例

现在，我们将以上所有步骤串联起来，完成一个完整的修图案例。

**案例目标**：将一张原始户外人像照片，修整为色调高级、皮肤光滑、构图精致的成片。

**操作流程**：

1.  **基础定调 (LR)**：
    *   导入原图到LR。
    *   分析原图问题（如曝光不足、阴影太黑、色彩平淡）。
    *   使用“亮度”工具：提阴影、压高光、适当加曝光和对比度。
    *   使用“颜色”工具：调整色温色调，增加自然饱和度。
    *   使用“颜色混合(HSL)”工具：单独调整天空（蓝色）、植物（绿色）、肤色（橙色）等。
    *   使用“局部调整”工具：提亮面部等特定区域。
    *   导出图片。

2.  **风格化滤镜 (VSCO)**：
    *   将LR导出的图片导入VSCO。
    *   选择一个合适的滤镜（如G8、A8、V系列），并调整强度。
    *   微调参数：可能降低一点对比度或饱和度。
    *   使用“色调分离”：高光加一点暖黄，阴影加一点冷蓝。
    *   保存图片。

3.  **皮肤处理 (Facetune)**：
    *   导入VSCO处理后的图片到Facetune。
    *   使用“平滑”工具，对人物皮肤进行适度磨皮。
    *   保存图片。

4.  **形体塑形 (美图秀秀)**：
    *   导入磨皮后的图片到美图秀秀。
    *   使用“面部重塑”微调五官。
    *   使用“瘦脸瘦身”工具精细调整脸型和身材线条。
    *   保存最终成片。

**（可选）添加天空 (MIX)**：如果原图天空不好看，可在LR调色后、VSCO加滤镜前，先用MIX的“魔法天空”功能更换天空。
**（可选）移除物体 (Retouch)**：在流程的任何阶段，发现照片中有多余杂物，都可使用Retouch进行移除。

## 📝 总结

本节课中，我们一起学习了手机修图的完整知识体系：

1.  **认识了核心的7款修图APP**及其专长：LR（调色）、VSCO（滤镜）、Facetune（磨皮）、美图秀秀（塑形）、MIX（加天空/特效）、Retouch（去杂物）、泼辣修图（备用）。
2.  **深入掌握了Lightroom**的局部与整体调整工具，理解了曝光、对比度、HSL等核心调色概念。
3.  **明确了标准的修图流程**：**基础调色 (LR) → 风格滤镜 (VSCO) → 皮肤磨皮 (Facetune) → 形体塑形 (美图秀秀)**。MIX和Retouch作为解决特定问题的工具穿插使用。
4.  修图的本质是**还原与美化**：先恢复照片应有的正常影调和色彩，再进行风格化和精修，顺序不能颠倒。

请记住，软件只是工具，核心在于多加练习，培养自己的审美。通过反复实践，你就能熟练运用这些APP，打造出属于自己的高质量图片。
