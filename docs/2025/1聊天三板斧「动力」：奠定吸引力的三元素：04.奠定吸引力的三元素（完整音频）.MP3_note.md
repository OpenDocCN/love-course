# 聊天三板斧「动力」：1：奠定吸引力的三元素

## 概述
在本节课中，我们将要学习聊天的基础框架——“聊天三元素”。这是开启有效沟通、建立吸引力的核心蓝图。我们将详细拆解构成成功聊天的三个不可或缺的要素：动力、方向和操作。

## 课程内容

在接下来的三节课里，我会讲关于如何跟女生聊天，叫做“聊天三板斧”系列。第一板斧是今天讲的内容：聊天的三元素。第二板斧叫聊天的模型和技巧。第三板斧叫拉伸关系和暧昧。这三板斧我们会分三次课程讲完，今天是第一讲：聊天的三元素。

聊天的三元素好比一张缩略的小地图。更恰当的表达是，有了这三元素，才是组成一个聊天的核心。就像我们学习英语时，语法、单词和口语是学习英语的三元素。聊天也有一个三元素，一个都不能少。

聊天的三元素是什么呢？第一个是**动力**，第二个是**方向**，第三个是**操作**。你先记住这个就好了。动力、方向、操作，这是很抽象的说法。可能很多兄弟还不理解，没关系，我们今天就是把这部分内容详细讲清楚。

具象一点说，比如你把聊天想象成是开车。开车不仅需要动力（燃料、汽油），还要有正确的方向，最后我们还需要开车的操作技巧，才能到达正确的目的地。

在这里有一个误区，就是很多兄弟会认为仅仅只需要开车技巧就可以到达目的地。这其实是一个常识，谁都知道光有开车技巧没有油，是不可能去到目的地的。但是很多人聊天却会这么认为，他觉得只需要有聊天技巧就可以聊到女生，于是拼命想学聊天技巧，问我问题也是说“我该怎么跟他聊？”，并且认为自己吸引不到女生就是聊天技巧的问题，就是因为自己不会聊天。这样想其实也很正常，毕竟大多数人看问题都是看表面。

当你学了今天的聊天三元素之后，你就会发现这个想法是多么错误。我再重复一下聊天的三元素，你们先记住这三个词：**动力、方向、操作**，一个都不能少。

为什么说学了三元素之后，你就会觉得只有聊天技巧是错误的？我问你一句话：假如金城武是个脑残，根本不会聊天，女生也可能会非常期待金城武给他回信息，对不对？因为金城武太帅了，吸引力足够强大，他根本都不需要技巧。这就引出了我们所说的三元素之一：**动力**。

动力指的是什么？指的就是**吸引力**。这是我们前几节课一直在强调的。车要跑起来，肯定离不开动力；人要聊天，也离不开吸引力。否则的话，你没有任何吸引力，说得再好听的话术，对方看着也是没有感觉的，只能成为美女面前的跳梁小丑。有句话叫做“人微言轻”，就是你没有力量，没有能量，你说的话是没有人在乎的。泡妞是一个道理，你也需要先把自己打造成为一个有吸引力的人，有光环的人，你的聊天才是有力量的。

所以说呢，接下来总结来了：跟女生聊天的方向就是**关系**。那么正确的聊天方向就是**关系的递进**。记住这个词语，关系的递进，或者叫关系的升级。你聊天的方向就是关系的递进，聊天的方向必须是按照关系的递进去设计的。你们关系的逐步深入，才是正确的方向，正确的路。

很多人跟妹子聊天就是没有方向，也就是没有递进的关系，主要体现在三个问题上。第一个是**原地打转**。聊天聊了三个月，其实还是在朋友阶段，意味着其实你就是在原地打转。我记得我很早的时候，很早期的时候，我认识了一个女生，我晚上给她打电话。我记得我当时是在练习用电话跟妹子沟通，然后我准备了很多页的A4纸，把话题都打在上面。我跟她真的打了一个多小时的电话。理论上说，一个刚认识的女生跟你打一个多小时的电话，那是肯定是很靠谱的。但是后面我们连面都没有见过。因为这一个多小时，我就是在那里傻嗨，看起来在运动，实际上是在原地踏步，我们的关系没有任何的递进。也有兄弟跟我说，跟喜欢了十年的妹子还是老朋友，怎么都没有办法去确定这个关系。十年老朋友了，我想说不好意思，十年了，你跟她可能还只是萍水相逢的关系。有没有听过一句话：“有些人认识一辈子，也没有走进心里；而有些人只需要一天，就像是经历了一辈子。” 因为有些人的关系，他就在原地打转。

那么第二点就是**关系走偏**。就像之前我们说的，本来是想聊成女朋友，结果聊着聊着聊成男闺蜜了，或者说有些人干脆就被发好人卡了，女生就说“你人很好”把你打发了。造成这种原因的，要么是因为自己根本就不够坚定方向，被女生领导着，女生在带领这个方向，就变成了朋友。

那第三点呢，叫做**跨越式的递进关系**。啥意思呢？就是递进得太快了，想一步登天，刚认识就问人家做女朋友，就表达喜欢，就说要在一起。女生是不能够去接受太快的关系的。因为你想那么快，还是因为男人是目的性的思维，怎么说呢？赶紧把问题解决了。而女生还是那个原则：太快了，就显得自己很随便。但是注意，这个快并不是我们说的时间上的快。像很多这种1到2天就跟妹子确定关系的情况，其实看起来时间很短，但是他们的关系是一个逐步递进的关系，他们走完了所有的关系，那么最终关系就确定了。

关系的递进就像是给你一张地图，你按着这个地图去走，那么你一定能够到达正确的目的地。从朋友一步步到亲密的关系，这就是正确的方向。在聊天三板斧的第三板斧，我们会详细地讲解这个关系的递进图。你不仅要知道方向在哪里，我还会教你具体该怎么去递进关系。递进关系是有风险的，该如何去把这个风险规避到最小，以及在递进过程中如果遇到阻碍了该怎么办，也就是解决刚才我们说到的那三个问题：原地打转、关系走偏和跨越式的递进。这里就不多说了，等第三板斧的时候详细讲解。这就是我们的聊天三元素之二。

那么聊天的三元素之三，第三个是什么呢？第三个是**操作**。对，就是那个“哎这是什么操作”、“那个是什么操作”、“这是什么骚操作”对吧？就是个操作。那也可以说这个是很多兄弟提到的聊天技巧。很多兄弟经常会说：“哇，这句话回得好牛逼啊”、“哇，这个话术好好”，这就是我们所说的具体操作。操作很重要，这个谁都知道，但一定要在前两个的基础之上操作才有作用。也就是用开车来说事儿，就是开车要有汽油，还要有方向，操作才有意义。这叫做**全局的思考思维**，思考问题的方式。你想如果你开车，你只是有技巧，你又没有油又没有方向，这都不用说了，太蠢了，肯定不会这么觉得。但是很多人觉得聊天他就会这么觉得，只需要操作就可以，缺乏了全局的思考思维。

所谓的操作就是什么呢？我该怎么去跟妹子开场？我聊天的这个流程该是什么样子的？有兄弟经常问有流程吗？这些我都把它称之为是操作。要想掌握操作，首先你要明白三件事情，很重要。

第一，**操作并不能让妹子爱上你**。只有操作是不能让女生喜欢你的。这个我们刚刚说过。

第二点叫做**技巧是0，吸引是1**。操作是零，吸引是一，没有一，再多的零都没有用。操作的作用只是辅助你的价值，它不能凭空产生出吸引力。但我们可能听过小米公司经常说的那句话：“营销是零，产品是一，没有产品再好的营销也没有用。” 这句话用在这里同样是有效的。

那么第三点，叫做**操作可以让妹子更喜欢跟你聊天，让你们的聊天更顺畅**。它只是有这个功能。就有些人，比如说他明明动力也很足，但是因为他操作得很差，会让女生跟他聊天其实很痛苦，很不舒服，很没有感觉，慢慢地可能就不喜欢了。那么有些人他可能动力一般般，但是因为他操作得很好，技巧用得很好，然后让妹子很畅快地跟他聊天，这样有可能会辅助增加你的吸引力。

你看我一直在强调整体性，三元素中动力、方向、操作都很重要。我强调了很多遍了，甚至我从第一节课开始就在强调这个。这就是有些人为什么天天看一些免费的内容，但是永远学不会聊天的核心原因。因为聊天它是一个系统的事情，它不是单一的。很多人他只是，比如说网上去搜一些免费或者看我们那些免费的，他只看到聊天技巧，他并不知道这三件事情是组合在一起的，它是整个的一个系统。只有有了这个系统，你才能真正的学会聊天。我之所以强调这么多遍，就是为了把这个事情钉在你的脑海里面。

那我们继续看聊天的操作。所谓的操作包含我们常见的聊天技巧、话术细节，以及时机，比如说什么时候说这个话效果会比较好，比如说调情的话该怎么说，邀约该怎么邀约等等，这些都是属于操作。是不是听起来就很复杂的样子？是的。如果光说说话的技巧，你到淘宝上面或者说当当网上面去搜，你就会发现有几十本是写聊天和说话技巧的书。但是我想说什么呢？不是所有的技巧都是有用的。男人跟男人之间聊天，跟男人跟女人之间聊天是不一样的。你跟你上司之间的聊天和你跟妹子聊天也是不一样的。所以你只需要知道男女之间该怎么聊天就可以了。我呢，删删减减，一共留下了4个最有用的男女之间的聊天技巧。说实话，这四个聊天技巧涵盖了大部分人都容易掉进去的坑。这四个聊天技巧，你要是都能够掌握，那么你将不会犯错。人说“不会犯错也不牛逼啊”，你要知道，当你成为一个聊天不犯错的男人，你已经比98%的男人要强大了。而且呢，为了让大家更容易接受，我用了4个常见的具体的东西，用这个食物，就是实际的物体，有吃的，有玩的，来代替和解释这四个聊天的技巧，非常好理解。就是因为很多人讲聊天总是讲得很复杂，讲得很让人不懂很费解。那我就用四个东西去代替。

那么这四个聊天技巧是什么呢？我们将在聊天的第二板斧为大家系统地讲解，要卖个关子。你要知道，真正的高手肯定是让你看了又想看，追着想要听的。对女生也是一样，女生一旦接触了你，那不好意思，她要追着你跑，这就是高手。那大家也就追着这个第二板斧来跑。

这就是我们今天讲的聊天的三元素：一个是**动力**，一个是**方向**，一个是**操作**，一样都不能少。看起来我们好像只讲了一个大概的全貌，但是非常重要，它概括了聊天的所有环节。我一直觉得把你的脑袋清理一下非常重要。当你的思考方式变了，你的竞争对手就完全看不懂你了。那这三元素其实就是改变你的思考方式。别人可能会说，你的竞争对手可能会说：“哎，这个人好像也没有什么特别能说会道啊，就追到妹子了。” 其实是因为你功夫下在了朋友圈，也就是动力上面。那别人也可能会说：“哎这个人也不帅呀，他怎么就追到妹子了？” 结果他不知道的是你在关系递进上做到了循序渐进，严丝合缝。
