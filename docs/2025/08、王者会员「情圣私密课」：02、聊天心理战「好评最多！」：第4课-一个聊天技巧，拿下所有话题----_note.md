# 王者会员「情圣私密课」：02、聊天心理战「好评最多！」：第4课 一个聊天技巧，拿下所有话题 🧠💬

在本节课中，我们将要学习一个核心的聊天技巧。这个技巧的核心在于，如何基于已获取的信息，自然地引导对话，从而与聊天对象建立更深的联系。掌握这一技巧，可以帮助你在各种话题中游刃有余。

## 从信息入手，开启话题

上一节我们介绍了聊天的基础，本节中我们来看看如何具体展开对话。一切关于聊天的内容，都应从你获取到的对方信息开始。你需要围绕这些信息，挖掘她可能感兴趣的话题和事情。

这样，下次开启话题时，你便可以围绕这些兴趣点进行交流。

## 核心技巧：朝着对方的思维去聊

认真听清楚接下来的核心原则：**朝着他想聊的思维去聊**。

这里需要明确一点：这**不是**让你去迎合对方，更**不是**要表现出你在刻意迎合。正确的做法是，在对话中自然地穿插相关话题。

因为这项技巧的最终目的很简单：**仅仅是让女生知道你们有共同的语言而已**。让女生感觉和你能聊得来，仅此而已。

以下是实现这一点的关键思路：

*   **建立连接**：有了共同的话题，对话便能顺畅进行。
*   **深化关系**：随着话题的深入，会让女生觉得她和你在一起相处是有未来的，是可以很融洽地相处、聊天，甚至一起完成一些事情。
*   **明确本质**：朝着他的思维去聊，**只是一项心理技巧而已**。不要将其误解为跪舔或讨好。

再强调一下核心：**朝着他想聊的思维去聊**。请始终记住这仅是一种建立舒适感和共同语言的沟通策略。
