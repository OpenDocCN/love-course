# 自媒体与个人IP：03：小红书直播的人气秘诀 🚀

在本节课中，我们将要学习如何提升小红书直播间的观看人数与同时在线人数，即直播的“人气”。我们将从刺激老用户、吸引新用户以及提升用户留存三个核心方面，系统地拆解具体操作方法。

## 概述

直播人气的核心在于观看人数与同时在线人数。提升人气主要依赖三个环节：有效激活已关注你的老用户；通过平台推流机制吸引新用户进入直播间；以及运用内容与技巧留住进入直播间的新用户。

## 刺激老用户：做好直播预告与私域联动

上一节我们概述了人气的核心，本节中我们来看看如何最大程度地激活老用户。关键在于直播前的预告宣发要做到位。

如果你非常看重某场直播，可以多次发布预告。同时，必须与你的私域流量池联动。无论运营小红书、抖音还是微博，最终都应沉淀用户至私域，课程知识需融会贯通。

以下是具体操作方法：
*   **私域全面触达**：通过社群、朋友圈等私域渠道全面发布直播预告。
*   **公域预告覆盖**：在公域平台（如小红书笔记）也同步发布预告。
*   **宣发拉满**：目标是让所有已认识、关注或看过你直播的用户都能收到开播信息。

## 吸引新用户：理解并利用平台推流机制

成功激活老用户后，直播间便有了初始人气。接下来，我们要设法从平台的直播广场吸引新用户。直播间获得推荐需要满足一定条件。

起始观看人数（老用户）越多，获得平台推流的可能性就越大。这是一个马太效应：老用户多 -> 易获推流 -> 新用户进入 -> 新用户转化为老用户 -> 起始用户更多 -> 更易获推流。

如果你的初始老用户不足，可以通过提升直播间互动数据来增加被推荐的机会。平台特别看重以下三类互动：

以下是关键的互动指标：
*   **评论/弹幕量**：用户发言的活跃度。
*   **点赞量**：直播界面左上角的点赞数量。
*   **送礼**：用户赠送付费礼物的行为。因为礼物收入由博主与平台分成，所以平台尤为青睐此项互动。

例如，在直播中引导用户赠送“C位出道”等礼物，主要目的并非赚取礼物收入，而是向平台算法发送强烈信号，从而最大化获得直播广场推荐、吸引新粉丝的机会。

## 提升用户留存：用核心内容留住观众

新用户被吸引进入直播间后，最关键的一步是让他们留下来，即提升**用户留存时间**。这需要你准备一套专门用于拉留存的口播脚本。

你可以通过自我介绍、讲述金句、连麦互动或才艺展示等方式留住用户。你需要明确自己最具吸引力、最能展现直播张力的核心亮点是什么。

留存的核心最终依赖于你的实时反应能力和持续的内容输出能力。做自媒体与个人IP，归根结底需要用内容撬动一切。

以下是提升内容能力的长期方法：
*   **日常积累**：在生活中随时准备更多的“梗”和观点。
*   **建立素材库**：耐心、系统地整理自己的素材库。
*   **持续循环**：坚持“输入 -> 输出 -> 表达”的循环，不断精进。

高用户留存时间（即观众平均观看时长）是直播间数据健康的关键指标，这直接体现了你直播内容的吸引力。

## 总结

本节课中我们一起学习了提升小红书直播人气的完整秘诀。

首先，**做好直播宣发**，通过公私域联动最大程度触达老用户，为直播间积累初始人气。其次，**炒热直播间互动**，通过引导评论、点赞和送礼提升互动数据，从而增加获得平台推流、吸引新用户的机会。最后，**用优质内容提升留存**，准备拉留存的口播，并依靠持续的输入与输出来夯实直播内容，留住进入直播间的新用户。

记住，直播人气的提升是一个系统工程，始于宣发，成于互动，终于内容。
