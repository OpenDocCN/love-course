# 谭崔年课：06：举山式 🏔️

在本节课中，我们将要学习“举山式”。这个练习旨在引导能量在体内循环，并培养深层的自我觉察与内在观察者的意识。我们将通过特定的身体姿势、呼吸与意念的配合来完成。

## 姿势准备

首先，我们来建立基础姿势。

双腿分开，宽度与骨盆相同。脚跟向外转动，使脚掌外侧与垫子外侧平行。双手自然贴靠于身体两侧。如需精确定位，可将中指贴于裤缝中间。

以下是姿势调整的关键点列表：
*   膝盖略微放松。
*   骨盆保持放松。
*   肩膀保持放松。

## 能量引导

上一节我们建立了稳定的身体姿势，本节中我们来看看如何引导能量。

双手向身体两侧平举。掌心开始时朝下，当手臂抬至与肩膀和手腕同高的位置时，翻转掌心朝上。让大臂靠近耳朵。保持双眼闭合。

吸气时，感受大地能量经由脚心，沿着双腿内侧进入躯干，从会阴进入身体。随后，能量沿着脊柱向上移动。

呼气时，感受能量沿着身体前侧向下，再次回到会阴，并回归大地。

## 呼吸与能量循环

现在，我们将呼吸与能量循环结合起来。

吸气时，收缩挤压PC肌（骨盆底肌群）。感觉能量像被水泵抽取一样向上，抵达头顶，再顺着手臂发射向天空。公式可表示为：**吸气 + 收缩PC肌 = 能量上行**。

呼气时，放松PC肌。感觉能量从双手和头部回流，再次回到会阴。公式可表示为：**呼气 + 放松PC肌 = 能量回流**。

## 培养内在观察者

在建立了能量循环后，我们将注意力转向内在意识的探索。

将注意力完全导向意识的源头，即那个内在的观察者。在内心深处提出这个问题：“是谁在看？是谁在观察？”然后，从内在深处去聆听这个问题的答案。

我们再次进行能量循环：
1.  吸气，收缩挤压PC肌，感觉能量向上到达头顶，到达中指尖端。
2.  呼气，放松PC肌，感受能量回流。

## 深入静默与觉察

接下来，我们尝试进入更深的静默状态。

尝试停止脑海中的任何念头。去感受那个观察者，去感受那份“空”与“无”。在内在深处再次提出这个问题：“是谁在看？是谁在觉知着这一切？”只需提出问题，无需用头脑回答。

然后，深深地进入内在，去聆听这个问题的答案：“是谁在看？是谁觉知着这一切？”去听那个回答的声音。它从哪里发出？在哪里回响？同时问自己：“是谁听到了这个声音？是谁听到了这个问题的答案？”

## 收回与身体扫描

最后，我们将结束练习并整合感受。

准备收回姿势。双手仍然从身体两侧，翻转掌心朝下，然后让双手自然回到身体旁侧。

以下是身体扫描的检查列表：
*   扫描你的脚底。
*   扫描你的膝盖。
*   扫描你的臀部与骨盆。
*   扫描你的腰部与上背部。
*   扫描你的肩膀与手臂。
*   扫描你的脖子与脸颊。
*   觉察它们当前的状态。

## 总结与整合

本节课中，我们一起学习了“举山式”的完整练习。

我们透过简单的身体感受觉察，让意识与能量形成了一组二元互动，也让观察者、观察行为与被观察对象形成了二元关系。传说中，湿婆（希瓦）居于喜马拉雅山，并向外投射出整个世界。现在，当你从“举山式”回到“山式”时，你便拥有了如湿婆般清明的觉知。
