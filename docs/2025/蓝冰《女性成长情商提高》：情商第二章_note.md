# 女性成长情商提高：第二章：成为情绪的主人

在本章中，我们将深入探讨情商的核心——情绪。我们将了解情绪的本质、学习如何分类情绪，并重点掌握控制愤怒情绪的具体方法。最终，目标是让你能够识别、管理并超越情绪，成为情绪真正的主人。

## 什么是情绪？

上一节我们介绍了情商的五种能力。本节中，我们来看看情商的核心问题：情绪。

每个人，尤其是女性，都拥有丰富的情绪。那么，情绪究竟是什么？它从何而来？情绪是一种化学物质，具体而言，它是一种神经肽，属于神经递质的一种。情绪分子与激素类似，它们在我们的大脑和身体中流动，所携带的信息能够影响我们多方面的身心机能。

大脑中的**丘脑**负责管理这些化学物质和信息的流动。听上去可能有些神奇，但这正是情绪产生的生理基础。关于理论部分我们不再深入。情绪在大脑中的产生和流动过程，类似于肾上腺素的作用机制，它是一种信息流，是化学合成的物质。

了解情绪的本质后，我们知道：在自然状态下，情绪是一种不受我们自身控制的应激反应，是自我保护的本能。但经过有意的训练和锻炼后，我们可以有意识地控制情绪的流动与释放。

## 如何分类情绪？

了解了情绪的本质后，为了更清晰地认识自己的情绪，我们需要对其进行分类。以下是情绪分类的一种直观方法。

我们使用一个由横轴和纵轴构成的坐标图。横轴代表情绪的**感受**，分为舒适与不舒适。纵轴代表情绪的**强烈程度**，分为强烈与温和。由此划分出四个象限，帮助我们定位当前的情绪属于哪一种。

当然，对情绪的分类不仅限于此模型，还存在其他模型，如普拉切克模型、能量模型等。有兴趣可以自行搜索了解。

## 如何控制愤怒？

在所有情绪中，**愤怒**的破坏力最大，在人际交往中冲击力也最强。大部分女性感情关系的破裂、以及情商低的表现，常与无法控制愤怒有关。若想成为情绪的主人，第一步就是学习控制愤怒。

首先，我们需要认识愤怒。愤怒通常是对羞耻、拒绝、抛弃、负罪感、害怕、背叛、焦虑、受伤、挫败、缺陷等感受的最直接反应和摆脱方式。如果无法控制愤怒，你可能会被视为“暴脾气”，并出现以下状态：
*   怒气上升极快，几秒钟达到顶点。
*   一天内多次产生或轻或重的愤怒感。
*   容易变得有攻击性，对人事怀有敌意。
*   在人际交往中易动怒，且难以平息。
*   总让人觉得难以相处。

愤怒的表现形式多样，例如：
1.  **攻击型愤怒**：意图伤害或责骂他人。
2.  **消极对待型愤怒**：意图激怒或打击他人。
3.  **沉默型愤怒**。
4.  **否认型愤怒**。

这些表现无一例外都会害人害己，且通常无法帮你达成想要的结果。

那么，如何控制愤怒？这里提供三个工具，如果你能静心练习27天，将能显著改善对愤怒情绪的控制。

### 工具一：熄怒三步法

这是最有效、最快捷的工具。其核心是及时阻断愤怒的情绪链。

以下是具体步骤：

1.  **察觉愤怒**：运用自我意识进行观察。当你出现想骂人、呼吸急促、心跳加快、肌肉紧张、想攻击对方等征兆时，就表明愤怒即将产生。此时，你已察觉到愤怒。
2.  **喊“停”与深呼吸**：立即在脑中对自己喊“停”，让一切慢下来。停止思考所有引发愤怒的因素，包括停止争吵对话。转身，然后**立刻进行三次深呼吸**：深吸气，憋住10秒，再缓慢吐出。重复三次。这能给大脑充氧，帮助冷静。
3.  **分析与观察**：冷静后，在内心组织反击对方的话术，然后像旁观者一样分析这些话术的杀伤力，连续分析三遍。此时，你会发现自己不再想骂人，愤怒也消散了。

这个方法的核心在于通过深呼吸为大脑充氧，并在关键时刻切断情绪反应链。练习25遍以上，效果会非常显著。

### 工具二：填写愤怒日志

这个工具适用于事后回顾愤怒事件时使用。

填写日志时，先让内心平静，然后按以下步骤记录：
*   回忆事件细节，描述诱发事件。
*   记录当时的心情，并为自己的愤怒程度打分（例如0-10分）。
*   识别当时的身体反应。
*   评估并识别你当时的期望和动机。

多练习几遍，多写几篇愤怒日志，你就能更好地理解和控制自己的愤怒情绪。建议随身携带本子，随时记录。

### 工具三：运用正念、冥想与同理心

这个工具旨在从根本上改变你对情绪的感知和操控能力。核心是学会用健康的方式表达愤怒，做到不伤害他人、不为难自己，并理解愤怒是需要被解读的信息。

具体操作涉及正念练习、冥想引导以及培养同理心，由于内容较多，已放在课程附件中，可供详细阅读。如有疑问，可在解答群中讨论。

当你能够灵活运用以上工具（哪怕只是一个），你就能逐步驾驭自己的情绪：
1.  **识别情绪**：能清楚知道自己处于愤怒、悲伤、兴奋等何种状态。
2.  **控制情绪**：能控制情绪的走向和强烈程度。
3.  **改善与流动情绪**：能改善情绪，并让情绪自然流动。最直观的表现是，你将拥有自我选择的能力，将注意力从“别人怎么看我”转向“什么适合我”。

## 如何超越情绪？

在能够识别、管理自身情绪的基础上，我们还需修炼**超越情绪**的能力。其核心是拥有**同理心**，即能识别、感知他人的情绪，并与之共鸣。

同理心的应用可分为五个步骤：

1.  **识别肢体语言**：从对方的言谈举止、状态解读其想法（如不屑、敌视、怀疑等）。
2.  **用心倾听**：保持冷静，专注倾听对方的心声。
3.  **察觉他人痛苦**：体察语言和动作背后可能隐藏的情感创伤。
4.  **口头回应**：用语言肯定对方的经历，认同其情绪和状态。
5.  **行动回应**：通过安静坐着、认真倾听、身体前倾等动作表示接纳。

反复训练这些步骤20遍以上，你将能更好地与他人深入交流，观察其内心，感同身受。
