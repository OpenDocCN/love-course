# 恋爱成长课：25：囚徒困境：如何走出恋爱中的博弈，避免互相损耗

在本节课中，我们将要学习恋爱关系中的博弈与合作。我们将探讨为何博弈是必要的，它与权力平衡的关系，以及如何通过健康的博弈策略来维护长期关系的稳定与和谐。

上一节我们讨论了需求感，本节中我们来看看博弈与合作如何共同作用于恋爱关系。

## 第一部分：恋爱中的博弈是什么

恋爱关系中的博弈，本质上是两个人的**权力竞争**，是一种暗地里的较量。

那么，两个人谈恋爱是否一定要博弈？答案是肯定的。博弈是必须的。需要强调的是，**算计不等于博弈**。算计只是博弈中的一种极端表现。例如，双方在经济、学历等方面的差异构成了博弈，但这并非算计。我们可以不去算计，但很难完全避开博弈。

## 第二部分：博弈的最终目的——权力平衡

既然博弈是一种权力竞争，那么其最终目的并非单纯地“赢”或压制对方，而是为了达到**权力的平衡**，更准确地说，是**动态权力平衡**。

当权力严重向一方倾斜时，强势方会获得更多主动权、话语权和决定权，而另一方则会相对弱势，这就是权力失衡。例如，普通人与明星结合，或明星与更有权势者结合，都可能出现权力失衡。

动态平衡意味着，权力并非固定不变，它会随着情境、个人成长和关系互动而转移。例如，热烈追求后主动分开的一方，其权力就可能发生转移。

权力失衡时，真正的**合作**不易出现。合作的比重会降低，取而代之的可能是需求感和压榨。例如，无底线的单方面付出，实际上是在让渡自己的权力，导致对方“被偏爱的有恃无恐”。

反之，一个人自身的实力和优秀程度，通常决定了其在关系中的权力大小。拥有权力不代表必须行使它，正如拥有骄傲的资本不代表一定会骄傲。

## 第三部分：权力平衡是合作的基础

权力平衡容易催生合作。这从侧面解释了为何“门当户对”常是更好的选择，因为它意味着双方在权力上大致平衡。

当双方实力相当时，博弈的必要性降低，合作、互相帮助与扶持成为可能，这能让**团体利益最大化**。这种逻辑不仅适用于恋爱，也适用于商业和社交关系。例如，员工与公司的合作基于价值创造与报酬的平衡，一旦失衡，合作就可能破裂。

因此，长期合作的基础是**长期的权力平衡**。那么，门当户对的婚姻是否就一劳永逸？这取决于这种平衡能维持多久。

## 第四部分：权力如何转移

权力转移是核心问题。造成转移的因素有很多：
*   **主观因素**：例如，一方依赖感变强、付出更多、开始无条件妥协。所谓的恋爱技巧通常只能在这个层面发挥作用。
*   **客观因素**：例如，一方成为全职太太/先生，与社会脱节，而另一方在外不断成长累积资源。两人的权力差距便会拉大。

当一个人进入情感舒适区，失去危机意识甚至产生依赖时，就容易开始失去权力。权力严重不平衡时，对方会失去合作的积极性。因此，我们需要通过博弈来重新平衡权力。

现在我们可以发现，**博弈与合作并不对立**。因为合作需要权力平衡，而当权力不平衡时，就需要通过博弈来恢复平衡。它们的最终目标是一致的：以关系为出发点，帮助**团体利益最大化**。

## 第五部分：健康关系中的温和博弈

在一段相对健康、有安全感的关系中，我们应保持合作心态，同时关注博弈的部分。这通常意味着关注以下三件事：

以下是健康关系中需要关注的三个博弈要点：

1.  **保持自我提升**：这是最重要的。博弈是现实的较量，保持内在与外在的进步，能为我们争取更多博弈筹码。记住：**我可以骄傲，但我不一定要去骄傲**。许多人在恋爱中容易“把自己谈丢”，放弃个人成长、职业规划或外在管理，这会导致权力流失。
2.  **信息透明**：失去权力，往往从失去信息开始。就像打牌，如果对方清楚你的底牌而你不了解他，你就容易被拿捏。信息透明相当于“打明牌”，让双方都能更灵活、公平地调整相处方式。（具体方法可回顾第15集课程。）
3.  **在细节上控制需求感或依赖**：这与上节课理念一致。过多的需求感会在博弈层面让我们显得弱势，并非因为我们真的差，而是强烈的需求感让我们“自降身价”，可能导致对方不珍惜。（具体方法请回顾上一节课。）

以上是一种比较温和的博弈，旨在长期维持关系的健康与平衡。

## 第六部分：不安全关系与囚徒困境

当我们面对一段缺乏安全感的关系，或一个可能伤害我们的人时，情况就不同了。这时可以引用**囚徒困境**这个博弈论模型来理解。

**囚徒困境模型简述**：
你和同伙被捕，隔离审讯。你们面临四种选择：
*   **策略一（合作）**：都不揭发对方 -> 各判1年。
*   **策略二（你背叛）**：你揭发，同伙不揭发 -> 你释放，同伙判10年。
*   **策略三（你合作）**：你不揭发，同伙揭发 -> 你判10年，同伙释放。
*   **策略四（互相背叛）**：互相揭发 -> 各判8年。

困境在于，你不知道对方会如何选择。

在恋爱中，这对应四种情况：
*   **策略一**：你在乎他，他也在乎你 -> 权力平衡，相互合作。
*   **策略二**：你不在乎他，他在乎你 -> 你背叛，对方合作，对方损失权力。
*   **策略三**：你在乎他，他不在乎你 -> 你合作，对方背叛，你损失权力。
*   **策略四**：互相不在乎 -> 双双背叛，关系崩溃。

从团体（关系）利益角度看，**策略一**是最好的（合作共赢），**策略四**是平衡但糟糕的（双输）。**策略三**是最糟糕的选择（你合作却被背叛）。许多人因为害怕陷入策略三，而抢先选择策略二（先背叛，如养备胎），但这会导致权力失衡，破坏合作可能。

## 第七部分：如何跳出囚徒困境——TFT策略与长期博弈

如何引导双方走向理想的**策略一（相互合作）**？关键在于建立**权力平衡**。当双方权力平衡时，任何单独的背叛都无好处，合作成为自然选择。

以下是两种核心策略：

**1. 奖惩机制（TFT策略）**
TFT策略的核心是“以牙还牙，以眼还眼”，但有一个重要前提。
*   **第一条准则：不要主动背叛，优先给予信任**。信任是所有合作的基础。我们先摆出合作的姿态，才能为策略一创造可能。
*   **第二条准则：面对善意，回报善意；面对背叛，给予惩罚**。如果对方选择合作，我们也合作；如果对方背叛（如冷暴力、不尊重），我们就要给予明确惩罚（如表达不满、暂时疏远）。许多人在这里犯错：对方越不好，自己反而讨好得越厉害，这等于亲手制定了“你越坏我越好”的错误规则。

**2. 长期博弈与长远利益**
*   **选择多次博弈的对象**：就像景区饭店（一锤子买卖）倾向于“宰客”，而小区饭店（熟人生意）必须物美价廉一样。恋爱中，应尽量选择那些有意愿进行长期、多次博弈的人，双方自然更倾向于合作。
*   **让渡眼前利益，交换长期利益**：在长期关系中，有时可以适当妥协眼前利益，以换取更重要的长期关系利益。例如，今晚把电视让给伴侣打游戏，但约定周末一起看电影或让对方帮忙按摩。这就是用眼前的小让步，交换未来的共同体验和对方的其他形式付出。
*   **互惠原则**：文明的基础之一是互惠。大多数人基因里自带“拿人手短，想着回馈”的倾向。在长期博弈中，适当的让步和信任，通常能换来对方的珍惜和回报，从而在不同时间节点达到权力的动态平衡。

如果对方在得到让步后不履行承诺怎么办？回到**TFT策略**，给予相应惩罚。总体而言，一个正常人是遵循互惠原则的。

## 总结

本节课我们一起学习了恋爱中的博弈与合作。

1.  **博弈的本质**：是两人之间的权力竞争，旨在达到**动态权力平衡**。
2.  **博弈与合作的关系**：**合作需要权力平衡**，当失衡时，需要通过**博弈来恢复平衡**。二者最终目标一致——**团体利益最大化**。
3.  **健康关系中的博弈**：关注**自我提升**、**信息透明**和**细节上控制需求感**这三件温和的事。
4.  **不安全关系与囚徒困境**：面对可能伤害我们的人，关系可能陷入类似囚徒困境的四种策略中，我们最需要的是**策略一（相互合作）**。
5.  **跳出困境的方法**：
    *   采用**TFT奖惩机制**：先给予信任，然后根据对方行为回报善意或施加惩罚。
    *   着眼**长期博弈与长远利益**：选择多次博弈的对象，并愿意适当让渡眼前利益来交换长期关系利益，依靠普遍的**互惠原则**促进权力平衡。
