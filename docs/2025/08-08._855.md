# 戒色自救指南：08：远离诱惑，切断污染源 🚫

在本节课中，我们将要学习如何通过主动远离诱惑和切断污染源，来为戒色创造一个干净的环境。核心在于认识到垃圾娱乐信息是导致破戒的重要诱因，并采取具体行动隔绝它们。

上一节我们讨论了识别和应对欲望的方法，本节中我们来看看如何从源头上减少欲望的触发点。

现实生活中有很多导致邪淫的诱因。比如手机里的美女图片和短视频、路人的穿着、游戏里的性感女角色等。要远离邪淫，首先就要远离各种导致邪淫的诱因。

一个人闲着没事干就刷手机，这种行为相当于在臭水沟里找东西吃，是给自己的精神投毒。垃圾娱乐信息会源源不断输入你的精神世界，最终导致邪淫。当你深刻认识到垃圾娱乐会导致手淫，垃圾娱乐就是病根病因时，就应该远离它。

就像查理·芒格说过：“如果我知道自己会死在哪里，那我永远都不会去那个地方。”你现在知道了自己会“死”在这些诱因上，那就别去碰。要用一把剪刀，把所有垃圾信息输入的通道全部剪断，阻止它们继续熏染你。

这种主动远离诱惑、切断污染源的方法，最起码能保证脏水不要流到自己心里来。

以下是十条具体建议，帮助大家隔绝污染源，远离垃圾娱乐。

**第一条建议：删除垃圾App。**
常用的娱乐App，如社交媒体、短视频、浏览器等，设计得非常巧妙，让你快速上瘾并花费大量时间。如果你发现自己无法自控地在这些App上花费大量时间，那么就果断删除它们。这能减少手机娱乐时间，从而将更多时间投入到有意义的事情中去。

**第二条建议：实行手机隔离。**
习惯性玩手机是造成垃圾娱乐成瘾的主因之一。要改变这种情况，需要逐步改变使用习惯，在特定场景下设置手机隔离。
*   上厕所不要带手机。
*   床上不要放手机。
*   坐公交地铁不要玩手机。
*   吃饭时不要玩手机。
*   睡觉前不玩手机。
*   起床后不玩手机。
这种做法一开始可能不习惯，但时间长了会形成新的、更健康的习惯。

**第三条建议：手机不要放桌上。**
当你在工作或学习时，将手机放在桌上会让你很容易分心，不断想去查看。建议把手机放在远离桌子的地方。

**第四条建议：把手机放在视线范围之外。**
因为一看到就想玩。把手机放进抽屉或柜子里，不要让它出现在视线范围内或触手可及的地方。当手机不在视线内时，我们的大脑更容易忽略它，从而减少玩手机的欲望。

**第五条建议：关闭非必要通知。**
很多社交媒体和娱乐软件会推送各种信息，干扰我们的注意力。我们可以在手机设置的通知中心里，选择关闭这些应用程序的通知，有效减少打开手机的频率。

**第六条建议：屏蔽App内的垃圾信息。**
我们可以在App内部做一些设置来净化信息流。例如：
*   微信公众号：取消无关关注。
*   微信群：设置消息免打扰、折叠或退群。
*   利用App提供的“青少年模式”、“老年模式”等，过滤杂乱信息。

**第七条建议：在“清醒时刻”立即卸载。**
什么是“清醒时刻”？例如，你刷手机半小时后，突然意识到自己浪费了很多时间，此刻就是“清醒时刻”。在这个时刻，要毫不犹豫地反手卸载掉让你沉迷的App，断掉不健康的习惯。

**第八条建议：在“反省时刻”彻底清理。**
这与上一条类似。当你刷手机几小时后，感到空虚、内疚和后悔时，这就是“反省时刻”。在这个时刻，要立即进行一波删除操作，对手机里的垃圾娱乐内容来一次大清理。有位知乎用户分享，他在刷抖音到凌晨后幡然醒悟并卸载了它。要认识到，很多时候是“手机在玩你”。在这种反省时刻就要立即行动。

**第九条建议：使用手机要有明确目的性。**
很多人用手机是漫无目的地打开，这种状态是一种昏沉麻木的状态，很容易陷入垃圾娱乐。所以，在打开手机或某个App之前，先问自己：“我打开它是要干嘛？”如果没有明确答案，那就不要打开。这样可以避免漫无目的地浪费时间。为此，可以设置写有“我打开手机是要干嘛？”的壁纸来提醒自己。

**第十条建议：不要习惯性把手机放口袋。**
把手机放口袋里会使其触手可及，垃圾娱乐就变得很容易。只要不出门，就把手机放在包里或抽屉里，离自己远一点，让眼睛的余光都看不到它，这也能有效减少垃圾娱乐。

我们为什么要如此坚决地远离诱惑、切断污染源？原因如下：
1.  **臭水沟理论**：接近诱惑（臭水沟）容易引起欲望爆发。
2.  **为戒色而戒网**：网络上的擦边球和软色情信息防不胜防，因此要戒除无目的的网络闲逛。
3.  **减少干扰**：减少负面信息的干扰。
4.  **眼不见为净**：物理上远离诱惑源。
5.  **珍惜注意力**：如果不珍惜自己的注意力，就很容易被外界各种“钩子”夺走，失去注意力的主权。

告诉自己这些理由很重要，因为我们的行为需要充足的理由和价值观来支撑，这能让我们走得更远。**戒网不是完全不上网，而是不要无目的地在网上闲逛**。如果有上网需要，那就带着明确目的，做完就离开，不要流连忘返。减少对网络的使用，就是减少被“臭水沟”污染的机会。把时间用来读书、学习经典、培养情操，做一些充实有意义的事情。

本节课中，我们一起学习了如何通过清理手机和改变使用习惯来为自己建立一座“隔离墙”，创造一个干净无污染的空间。网络诱惑太多，一不小心就会掉进去。当我们建立起清醒的认知，意识到垃圾娱乐的巨大副作用后，就应该坚决远离污染源。

认真执行本节课的十条建议，将它们落到实处，才能有效减少垃圾娱乐。当垃圾娱乐行为减少，由它导致的邪淫可能性也一定会降低。这是在病根上下功夫，**改变原因，让结果不再发生**。
