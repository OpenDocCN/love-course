# 攻炮高手「社交APP聊天版」：01、聊天大师「心动窗口撕裂」：第3课 联系感、安全感与邀约 🎯

在本节课中，我们将要学习聊天流程中的第三步和第四步：如何与聊天对象建立联系感与安全感，以及如何成功地进行邀约。掌握这些核心步骤，能让线上互动更自然地过渡到线下见面。

## 建立联系感与安全感

上一节我们介绍了调动情绪的开场白，本节中我们来看看如何深化关系。你不会与一个毫无交集的人成为朋友。如果对方觉得与你根本不熟悉，便难以擦出火花。因此，你需要与对方建立一些共同点，形成一座联系的桥梁。

关于建立联系感，我们有一个非常实用的小技巧：**“也”字的运用**。

以下是运用该技巧的示例：

*   **你也去过三亚。**
*   **你也喜欢吃小龙虾。**

这样的话术能有效拉近双方距离，暗示你们拥有共同经历。根据对方个人特点或其社交动态（如朋友圈）中的信息，都可以用来建立联系感。这是一种高效的聊天方式。

当然，在联系感中，还有一个至关重要的组成部分：**安全感**。试想一下，在你与对方聊天氛围舒适，并通过开场白调动了她的情绪后，你们又建立了联系感，彼此有了一定了解。下一步自然就是邀约。

但往往你一提出邀约，对方就会犹豫不决或推脱。这说明你在安全感建设上尚有不足。因为女性天生对安全感更为敏感，在不清楚你的姓名、职业等基本信息时，她会担心与你见面的安全性。

因此，我们需要进行一个常见的步骤：**基本信息交换**。基本信息交换包括年龄、工作、星座等基本情况的相互透露。

以下是进行基本信息交换的一个小技巧：

*   一般采用 **“先说自己，再问对方”** 的模式。例如：“我是做IT的，你呢？” 这种方式能更顺畅地建立联系感和安全感。

## 成功邀约的三个原则

成功建立联系感和安全感后，我们便来到了聊天流程的第四步：邀约。如果前面步骤都完成得很好，邀约便是水到渠成的事情。

在邀约方面，我们强调三个核心原则。

**第一个原则是：确定。** 一个真正的邀约，必须具备确定的时间、地点和具体事项。

例如，这是一个确定的邀约：
`周六下午6点，我们在太古里那家甜品店见面吧。`

**第二个原则是：先模糊后确定。** 在提出确定邀约之前，应先进行模糊邀约测试。模糊邀约是为了测试对方是否愿意与你出来，其特点是**省略具体时间**。

例如，这是一个模糊邀约：
`改天有空的话，我们可以一起去太古里尝尝那家甜品。`

如果对方回应积极（如说“好”），你便可以顺势提出：“那你一般什么时候比较方便？”，从而将模糊邀约转化为确定邀约。这种方式能有效降低你的目的性，避免给对方造成压力。

**第三个原则是：文字精简。** 提出邀约时，话题应聚焦于邀约本身。

*   **正确做法**：直接、清晰地提出邀约。
*   **错误做法**：在邀约信息中夹杂其他话题，导致聊天主线偏离，最终可能忘了敲定邀约细节。

## 课程总结

本节课中，我们一起学习了深化聊天关系的两个关键阶段。首先，我们探讨了如何运用**“也”字技巧**建立**联系感**，并通过**基本信息交换**（先说自己，再问对方）来构建**安全感**。接着，我们介绍了成功邀约的三个原则：提出**确定**的邀约（时间、地点、事件明确），在此之前使用**模糊邀约**进行测试，并且在邀约时保持**文字精简**、话题聚焦。掌握这些步骤与原则，能帮助你更顺利地将线上互动推进到线下见面。
