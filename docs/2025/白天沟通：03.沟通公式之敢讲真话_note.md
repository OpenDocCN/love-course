# 沟通的艺术：03：沟通公式之敢于讲真话 💬

在本节课中，我们将要学习一个核心的沟通公式：**敢于讲真话**。许多人在婚姻或长期关系中，会陷入一种“争宠式”的沟通模式，导致双方产生误解和隔阂。本节将分析这种模式的底层逻辑，并提供直接的解决方案。

上一节我们探讨了沟通的基础，本节中我们来看看如何通过坦诚表达来化解矛盾。

## 认识“争宠式”沟通

许多人觉得婚姻过着过着就变了味：男方觉得妻子常为小事无理取闹，女方则觉得丈夫根本不在乎自己。这通常是因为双方陷入了“争宠式”的沟通。

以下是这种沟通模式的一个典型表现：

*   你们经常因为某个特定事件（例如丈夫看色情内容）反复争吵。
*   这件事已成为一个心结，一触即发。

## 分析情绪背后的真实需求

争吵的表面原因往往不是问题的核心。我们需要分析情绪背后的底层逻辑。

例如，丈夫打游戏不理你，你会生气吗？如果生气的原因与看色情内容时一样，那么说明你的真实情绪是**害怕被冷落和忽略**。

再比如，吵架时丈夫说“咱俩别过了”，你会生气吗？如果生气且原因同上，那么你的真实情绪是**害怕被抛弃**。

又如，你不想让他在房间抽烟，他偏要抽，你会生气吗？如果原因依然相同，那么你生气的点在于**他不听从你的话**。

所以，这些情绪——被冷落、被忽略、被抛弃、对方不听话——与看色情内容本身有直接关系吗？**没有关系**。

## 核心问题与解决方案

你只是像一个孩子一样，抓住某一件具体的事情，通过“作”和“闹”的方式来表达内心更深层的需求。但对方不是你肚子里的蛔虫，他无法洞悉这些深层需求。他只会觉得：“她又为这点小事闹了，都闹了多少次了。”

闹的次数多了，他真的会看到你都感到害怕和厌烦。

因此，必须避免“争宠式”的沟通。解决方案的核心公式是：

**大胆、直接地表达自己的需求 = 敢于说出口 + 敢于争取自己的利益**

敢于争取自己利益的人，别人才不会怠慢他。畏畏缩缩、拐弯抹角，只会让别人觉得你好欺负、好糊弄。

## 如何直接表达需求

以下是直接表达需求的几个例子，请注意其与“争宠式”抱怨的区别：

*   **想让丈夫多给家用**：你可以说：“我想穿更多漂亮衣服给你看，老公帮我买单好吗？”
*   **想让丈夫多陪伴你**：你可以说：“你在家里我就觉得好安心，能多陪陪我吗？”
*   **想让夫妻生活更和谐**：你可以说：“我好喜欢你抚摸这里，可以多停留一会儿吗？”

本节课中我们一起学习了“敢于讲真话”的沟通公式。关键在于识别自己情绪背后的真实需求，并学会用直接、清晰的语言表达出来，而不是通过指责和吵闹来发泄情绪。坦诚直接的沟通，才能有效解决问题，让关系更加亲密和稳固。
