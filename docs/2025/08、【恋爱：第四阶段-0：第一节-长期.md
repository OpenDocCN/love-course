# 恋爱军团6888：第四阶段：01 维持长期关系：第一节 长期关系必须要做的事

在本节课中，我们将学习如何经营一段健康的长期关系。我们将探讨长期关系的核心要素、常见问题以及实用的维护策略，帮助你建立并保持一段稳定、甜蜜的伴侣关系。

上一段我们提到了长期关系的重要性，本节中我们来看看维持长期关系必须做哪些具体的事情。

一段关系确立3个月以上才能算作长期关系。许多男性只擅长短期关系，认为学习技巧只是为了发生亲密关系。作为女性，我不评论这些人的道德，只想提醒一句：不忘初心。请思考你最初的目的是什么。如果你学过很多技巧，不妨问问自己，你的初心是什么，现在还在吗？男女感情最终的核心是长期关系，这也是绝大多数人学习情感技巧的初衷。

如果你正在恋爱，或曾有一段不太融洽的长期关系。如果你发现你越来越爱对方，而主动权却离你越来越远。你慢慢发现对方变得不那么爱你，不愿和你待在一起，关系没有一开始甜蜜。甚至频繁争吵，最终分手或彼此生厌。那么在这节课上，你一定能找到问题的所在。

人有两个基本的心理需求：一是在所爱的人心中居首位；二是被人无条件接纳的需要。可惜，这既是长期关系中最迷人之处，也正是日后产生最大困难的地方。不要把自己当做神，也不要把对方当做神。唯有神才能彻底满足人心这两大需求。爱情却能带来一种幻觉，让人以为爱人就是那位可以满足所有需要的神。但是，如果你把爱人当神，把对方当成你的唯一，那么你必将失望。

接下来，我们聊聊恋爱中情侣在长期关系上会出现的问题以及解决建议。

以下是维持长期关系必须做到的九件事：

**1. 控制火花的弥漫速度**
在爱情的火花期，需要避免一开始就如胶似漆，24小时都腻在一起并倾诉所有。如果你每分每秒都跟他待在一起，互相倾诉彼此的过去，这很危险。因为这样你在对方眼中会变得透明，慢慢会失去吸引力。太快交心会减少你欣赏对方的能力，降低吸引力。一个人能给别人最好的礼物，是你内心深处真正的想法与感受。但你不要把这珍贵的“人参果”随便丢出去。这些珍贵的心灵礼物，只能给那些通过你安全性测试、渐渐赢得你信任的人。请务必记住：**心灵亲密的程度应该与交往的阶段成正比**。

**2. 如何用“同生共死”延续关系**
刚刚说过，切莫一开始就付出一切。但是，你可以很有技巧地去维持一段关系，这个技巧就是“同生共死”。也许“同生共死”在前期吸引比后期维持更有效，但它在后期的维系作用也不容忽视。这里简略提一下它在后期的运作方法。当你跟他在一起久了，就会不自觉一起做一些事情：一起吃饭、逛街、整理房间、买东西、旅游。这些就是只有你和他的独家记忆。当你与他的回忆足够美好快乐，他是不会轻易离开你的。就算有一天分道扬镳，他也会感激遇上你并与你有独家记忆。这就是“同生共死”的实践基础。

**3. 必须学会相互投资**
这里的投资不单指钱财，更多的是感情投资。所谓互相投资，就是你与他一起的投资。当他对你的投资越多，他就越离不开你。但在长期关系中，你一定要有投资，不要一直让女生投资。如果他投资而你不投资，就会让对方觉得没有安全感，认为你不在乎他。

**4. 懂得“独”与“依”**
在长期关系中必须懂得“独”，也必须懂得“依”，二者之间可以有弹性地调整。“独”的含义是两个人要给彼此预留空间，最好能有独立的社交圈，彼此相互尊重。“依”的含义是两个人要互相依靠，给予彼此安全感。例如，他感冒了，你要为他买药；他遇上事了，你要为他出主意。

**5. 十指紧扣（尾声的承诺）**
“十指紧扣”也叫尾声的承诺，即你要确定你和他是要走同一条路。你们两个愿意一辈子生活在一起，并且两个人的期待必须明确：你和他是仅仅想发展短期关系，还是想要一直走下去？这可以有效地避免对方认为你们只是试一试的关系，而你自己却认为他已经离不开你了。

**6. 两性亲密互动**
长期关系里必然要有良性的亲密互动，也就是亲密关系。在你们两人已经坦诚相见之后，你和他的缺点、性格缺陷以及潜在的冲突，也会慢慢显现。这也是很多婚前同居者临近结婚却闹掰的问题所在。对于这个问题，我接下来会尽量详细表达观点与建议。作为女人，我不想过多讲述关于亲密关系的东西，只想以我的角度说明：两性互动中愉悦的亲密关系，对于情感互动有促进及加分的作用。

**7. 保持神秘感**
保持神秘感是恋爱保鲜中特别重要的组成部分，且需要区分初期吸引与长期关系。在初期吸引时，你要做到尽量避免太快把自己过去的软弱、隐私、心中的不安全部告诉对方。在一段长期关系中，保持某种神秘感，是保持新鲜感特别重要的方法。长期关系就像两个圆圈，中间有很多交集，交集可大可小，随时间迁移而变化。当两人受到压力时，交集的地方比较多；当压力较小时，两人可以向外探索，每个人可以有若干百分比是单属于自己的秘密。随着彼此信任感增加，彼此可以更加透明，但不能勉强。所以，最理想的方式是建立良好的爱的关系，而透明只是这个关系的结果。

**8. 如何应对长期关系中的吵架**
前些天听闺蜜说了一件事：一对交往近两年的男女朋友，因女方去酒吧玩没有提前告知男方，男方知道后与女方大吵大闹，最终分手。我想说这个故事，是因为很多恩爱情侣仅因一时情绪冲撞就分道扬镳。如果诸位有兴趣，推荐看一本情感纪实《我们为什么分手》。情绪分为三种：情绪高涨、情绪一般、情绪差。长期关系中会出现各种情绪。当对方出现愤怒、你们吵架时，你应该怎么做？如果你在火山口上，对他发火，他也对你发火（比如闹分手），如果你比他更严重，估计就没有然后了。在一段长期关系中，当我们情绪不好时，最好不要向伴侣完全宣泄情绪。要宣泄，可以有技巧地宣泄。我一直告诫身边的男性朋友，一定要在伴侣前控制好情绪，不要在情绪不好时对他大吼大叫、数落他以及无脑地对骂。其实最好的解决方法是：你可以把自己的情绪放一放，心情平和一点去和他交流。只要把他的情绪状态转变过来，让他脱离难受、愤怒、生气等负面情绪。然后你抱一下他，和他开个玩笑就好了，他自己也就恢复了。当女生情绪好时，所有事情都会迎刃而解。之前有很多男人告诉我，他们虽然智商不高，但是情商很高。我每每听到只是笑笑。我们常说的情商无非五大要点：了解自身情绪、管理情绪、自我激励、识别他人情绪、处理人际关系。由此看来，所谓情商，绝大部分是说能够有效控制自己的情绪。一时让他开心，一时让他失落，让他的情绪坐过山车，他才能更好地爱上你。情商永远比智商重要，无论是在感情、人际关系还是事业方面，都能很好地处理问题。

**9. 爱情持久的燃料与氧气**
*   **爱情的燃料**：对自己保持有信心，让自己变成一个有意思、有吸引力的人。有空时，利用这些时间去提升自己，改变自己。
*   **爱情的氧气**：两个人需要空间，不要老黏在一起，也不需要什么都告诉对方，保持一点神秘感。爱情最怕的就是：开始的时候老黏在一起，但过了一段时间，一个人变成空间追求者，另一个人则是亲密关系追逐者。这也是很多长期关系会出现的问题。一开始主动权在你这里，后来慢慢你越来越黏对方，越来越想和对方待在一起，慢慢就改变了游戏规则，主动权就到了对方那里。
