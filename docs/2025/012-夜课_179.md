# 冥想课程：012-05：慈爱散播冥想

在本节课中，我们将要学习一个名为“慈爱散播冥想”的练习。这个练习旨在引导你进入深度放松状态，并激发内心的爱与慈悲，通过观想将这份爱意逐步扩散至整个世界乃至宇宙。

## 概述

慈爱散播冥想是一种结合了身体放松、呼吸关注与积极意愿的冥想方法。其核心是通过一系列步骤，从关注自身开始，逐步将慈爱的感受扩展到更广阔的范畴。本节课将详细拆解这一过程。

## 准备与放松

首先，我们需要为冥想创造一个合适的环境和身心状态。

请找一个舒适的位置，坐好或者躺下。轻轻地闭上双眼，把自己的身心慢慢地调整到一个放松的姿态。

### 关注呼吸

呼吸是连接意识与身体的桥梁。通过观察呼吸，我们可以快速进入平静状态。

以下是进入放松状态的步骤：
1.  请把注意力放在呼吸上。
2.  进行自然而又均匀的呼吸。
3.  观察一呼一吸的过程。
4.  感受每次吸气带来平静与放松。
5.  感受每次呼气带走焦虑与烦恼。

随着呼吸，开始放松，不断地放松，进入到更深的放松状态。

现在，我们可以尝试进行几次深沉的呼吸。深深地吸气，感受气息充满整个肺部。每一次深沉的吸气，都会带来更多的平静与放松。每一次深沉的呼气，都会带走更多的烦恼与焦虑。

让我们随着深沉的呼吸，继续放松，直到进入更深的放松状态。

### 身体扫描

上一节我们通过呼吸放松了身心，本节中我们来看看如何系统地放松身体的每一个部位。

现在，让我们回到均匀自然的呼吸，把注意力放到我们的身体上。当我说到你身体的某个部位时，那个部位就会随着我们的觉察越来越放松。

以下是身体各部位的放松顺序：
1.  放松我们的头顶。
2.  放松我们的左肩。
3.  放松我们的右肩。
4.  放松我们的胸部。
5.  放松我们的背部。
6.  放松我们的肚脐。
7.  放松我们的小腹。
8.  放松我们的腰部。
9.  放松我们的髋骨。
10. 放松我们的尾骨。
11. 放松我们的左膝。
12. 放松我们的右膝。

如果你此时坐着，那么你的身体会呈现一个自然的支撑状态。如果你现在躺着，你就如同一片云朵，轻松而又自在。让我们去感受从我们的头一直到脚，全然地开始放松。

## 慈爱的生发与散播

在身体完全放松后，我们开始进入本次冥想的核心环节：生发内心的慈爱并将其散播出去。

### 对自己与亲近之人的祝愿

我们每个人都需要快乐，每个人都需要幸福。愿我的身体没有痛苦，愿我的精神没有痛苦，愿我解脱一切的苦难，愿我生活快乐。

同样地，将这份祝愿扩展出去。愿我们的父母、导师、亲友以及所有的共修者，愿他们的身体没有痛苦，愿他们的精神没有痛苦，愿他们解脱一切的苦难，愿他们生活快乐。

### 观想爱的能量中心

现在，感受我们的胸口，感受胸口的温暖，感受一直在那里的爱。你胸口的爱，就像一颗璀璨夺目的宝石，一颗**绿色的宝石**。这颗宝石随着你的祝愿，充满了越来越多的能量，充满着爱的能量，充满着宇宙中最强大的能量。

### 逐步散播慈爱

以我们胸口的宝石为中心，向四周开始散播爱的能量。

**第一步：散播至道场**
绿色的光芒散播到了整个道场。愿道场里所有的男众女众、所有的共修者、所有的圣者、所有的护法神、所有的发心者，愿他们没有身体的痛苦，愿他们没有精神的痛苦，愿他们解脱一切的苦难，愿他们生活快乐。

**第二步：散播至城市**
以我们胸口的宝石为中心，开始向外散布爱的能量。绿色的光芒散播到了整个城市。愿这个城市里所有的男人、所有的女人、所有的共修者、所有的圣者、所有的护法神、所有的发心者、所有的有情众生、所有天上地下的众生，愿他们没有身体的痛苦，愿他们没有精神的痛苦，愿他们解脱一切的苦难，愿他们生活快乐。

**第三步：散播至国家**
以我们胸口的宝石为中心，向外开始散播绿色充满爱的能量。直到绿色的光芒散播到了整个国家。愿这个国家里所有的男人、所有的女人、所有的共修者、所有的圣者、所有的护法神、所有的发心者、所有的有情众生、所有天上地下困苦的众生，愿他们没有身体的痛苦，愿他们没有精神的痛苦，愿他们解脱一切的苦难，愿他们生活快乐。

**第四步：散播至宇宙**
开始向外散播绿色充满爱的能量。直到绿色的光芒散播到了整个宇宙。愿整个宇宙所有的男人、所有的女人、所有的圣者、所有的护法神、所有的天人、所有空中的有情众生、所有水中的有情众生、所有困苦的有情众生，愿他们没有身体的痛苦，愿他们没有精神的痛苦，愿他们解脱一切的苦难，愿他们生活快乐。

## 总结

本节课中，我们一起学习了“慈爱散播冥想”。我们从调整姿势、关注呼吸、放松身体开始，逐步生发出内心的爱与慈悲，并以胸口观想的绿色宝石为能量中心，将这份慈爱由近及远、由小到大，依次散播给道场、城市、国家乃至整个宇宙的所有众生。这个练习有助于培养宽广的慈悲心，并带来深层的内心平静。
