# 个人影响力构建教程：13：如何让全世界看到你？ 🌍

在本节课中，我们将探讨构建个人影响力、让世界看到你的三个核心方法。这些方法基于持续输出、独特创作和识人用人。

## 概述

要让世界看到你，关键在于掌握并极致化三种核心能力：公开演讲、原创写作与团队管理。这三种能力分别对应着对外表达、思想传播与资源整合，是构建广泛影响力的基石。

## 核心方法一：极致化公开演讲 🎤

上一节我们概述了三种核心能力，本节中我们来看看第一种：公开演讲。

成为演讲高手需要极致的投入。你必须将演讲视为每日必修课，投入大量时间进行练习。例如，可以安排早晨、中午、下午各进行4小时演讲，全天总计12小时。晚上则用于复盘总结，并再加练2小时。这种投入程度意味着你需要将极大的热情乃至生命能量倾注其中。

进行演讲时，有一个至关重要的原则：**必须真人出镜**。如果无法做到真人出镜，那么就不要进行所谓的“演讲”。真人出镜是建立真实连接、传递真诚与信任的基础。

## 核心方法二：掌握“用嘴写作” ✍️

掌握了极致的演讲能力后，另一种强大的影响力工具是写作。但这里所说的写作方式与众不同。

你的写作能力必须非常出色。关键在于，不要抄袭或改编他人的文章，那是缺乏原创能力者的行为。真正有能力的人，始终在创作自己的内容。

写作的核心方法是：**用嘴巴写**。记住，用嘴巴写，你才能达到出神入化的境界；用手写，则难以企及高峰。许多思想大家之所以成为“神”，正是因为他们擅长“用嘴写作”，即先进行口述，再整理成文。

以下是实现“用嘴写作”的心法：
*   **摒弃借口**：不要以“读书少、写不出来”为借口。想象在极端压力下（只是一种思维实验），为了守护最重要的东西，你是否能激发出写作的潜能？
*   **激发潜能**：所有能力都是被激发出来的，而非简单刺激。人若不会激发自身潜能，永远无法成就大业。
*   **消化痛苦**：真正强大的人，能在受到伤害后消化痛苦，而非被其击垮或陷入永远的怨恨。一个不懂报仇的人，往往也不懂报恩。

## 核心方法三：研究人性，善用人才 👥

如果你觉得自己在演讲和写作方面都不擅长，那么第三条路是研究人性并善于管理。

第三个要点是：了解人的劣根性是什么。明白了这一点，你就可以经营好一百人、一千人，让他们为你的事业奋斗。

具体方法是根据人的特性进行分工：
*   适合演讲的人，让他负责对外宣传。
*   适合写作的人，让他负责统一团队思想。
*   你则需要研究清楚每个人适合干什么。

为什么这可行？因为你在研究人性，所以能让所有人都为你所用。这时，你需要将自己定位为资源的整合者与领导者，如同历史上的刘邦、朱元璋或毛泽东那样的人物。他们兼具演讲、写作和指挥千军万马的能力，是罕见的全才。

## 总结与三种人生路径

本节课我们一起学习了构建全球影响力的三种核心路径：极致演讲、原创写作和人性管理。

在我看来，世界上主要通过三种方式改变世界：
1.  **靠演讲改变世界**。
2.  **靠写作改变世界**。
3.  **靠管理（用人）改变世界**。

你需要明确自己是哪一种人，并全力以赴锤炼对应的“绝活”。如果这三种都不是，那么你需要深刻反思自己的定位。

**核心公式概括为：全球影响力 = 极致演讲(真人出镜) + 原创写作(用嘴写作) + 人性管理(知人善任)**。
