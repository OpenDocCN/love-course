# 沟通技术课程：2.20：技术讲解：借势 📚

在本节课中，我们将要学习一项名为“借势”的核心沟通技术。这项技术能帮助你巧妙地利用对话中或环境里的有利信息，以达到自己的沟通目的，尤其在增强互动、引导话题和提升技术成功率方面效果显著。

## 什么是借势？ 🤔

借势，是指在沟通中借用对方话语或当前情境中有利于实现自身目标的信息的一种技术。它不仅在辩论中常见，在日常交流，尤其是与异性的互动中，也极为实用。

**核心公式**：`借势 = 借用（对方话语/情境信息） + 达成（自身沟通目标）`

## 借势的主要用途 🎯

上一节我们介绍了借势的基本概念，本节中我们来看看它的具体用途。借势是一项通用性很强的辅助技术。

以下是借势技术的一些常见应用场景：
*   **辅助其他技术**：与其他沟通技术（如延迟、询问）叠加使用，能产生“1+1>2”的效果。
*   **增加回复概率**：通过引用对方说过的话，让对方更愿意回应。
*   **增强技术效果**：让单一技术的表达更有力、更合理。
*   **重启聊天**：利用过往聊天记录中的信息，自然而不突兀地重新开启对话。
*   **引导积极表达**：借用对方展示过的正面态度，鼓励对方继续积极互动。

## 实操案例解析 💡

了解了用途后，我们通过几个真实案例，来看看借势技术是如何在具体对话中运用的。每个案例都将展示借势的不同应用场景和搭配思路。

### 案例一：借势辅助拉近关系

这个案例来自线下学员的社交后续聊天，展示了如何用借势辅助“延迟”技术来拉近关系。

**技术点**：使用“@（艾特）”对方的方式进行借势。
*   **“今天叫我哥今天就做你哥哥”**：借用了对方叫“哥”这个行为，通过@对方来强调“因为你叫我哥，所以我才会这么说”，让话语显得合情合理。
*   **“宝宝跟哥哥两个都有的艾”**：同样用@的方式，借用了对方说“宝宝”的事实，来平衡双方框架（你叫我宝宝，那我也可以叫你宝宝）。

**思路概括**：在对方做出“哼”的撒娇行为时，为高效拉近关系，采用“借势+延迟”的技术组合。先抓住“歌”这个关键词做延迟回复，并用@借势来增强效果；当对方调侃时，再次用@借势将框架拉平。这个案例清晰地展示了借势作为辅助技术的好用之处。

### 案例二：借势用于正式场合

这个案例来自内部导师在社交软件上获取微信号的过程，展示了在正式场合下如何使用借势。

**技术点**：使用“你都说”这三个字进行借势，并结合了@功能。
*   **“你都说可以了，不是”**：借用了女方之前回复“哈哈可以”的表态，来应对女方当前的“访问”（试探），表达“既然你之前都答应了，现在又这样问”的潜台词，带有一点合理的强势感。

**思路概括**：在正式索要微信号时遇到女方访问。判断对方懂沟通技术且有些小傲娇后，采用借势技术，引用对方先前“可以多多沟通”的承诺。当对方依旧拉扯时，感知到其愿意但需要台阶，便结合“夸赞”技术抬对方一手，最终成功加上微信。这说明了对于懂技术的对象，正式场合配合借势效果更佳。

### 案例三：借势结束拉扯并取得主导权

这个案例展示了如何在陷入拉扯时，使用借势来破局并掌握聊天主动权。

**技术点**：使用“@”来解释借势的内容。
*   **“你这么问过那要一起想”**：借用了女方之前问过“你是想结婚吗？”这个问题，来回应女方当前的提问“你想到办法了吗？”，巧妙地将解决问题的责任共担，从而结束拉扯。

**思路概括**：女方因学员“不粘人”而表达失望。学员先用“访问”技术确认对方的重视感，再将“怎么办”的问题丢回给对方。对方反问后，陷入拉扯。此时采用借势技术，借用对方曾经的提问来回应，成功结束拉扯并取得了聊天的主导权。

### 案例四：借势用于关系内的幽默调侃

这个案例展示了在恋爱关系中，如何用借势进行不冒犯的幽默调侃。

**技术点**：使用“@”来解释借势的内容。
*   **“理性的爱人”**：借用了女朋友曾自称“理性”这一点，来调侃她此刻因情绪化想吃海底捞的行为，并顺势开玩笑让她请客。

**思路概括**：女朋友情绪外放想吃海底捞。学员不想轻易答应以免显得价值低，于是采用借势技术，借用对方“理性”的自我描述进行友好调侃，并开玩笑提条件。结果良好，且澄清了这并非不愿付出，而是维护关系中的平衡。这种调侃是幽默而非打压。

### 案例五：借势重启长期未聊的对话

这个案例展示了如何利用借势，优雅地重启一段沉寂已久的聊天。

**技术点**：使用“我记得”进行借势，并与“询问”技术叠加。
*   **“我记得某人说过是积极主动找话题的类型”**：借用了女生过去自称“主动积极”的信息，作为重启聊天的引子。
*   **“怎么不见踪影了？”**：这是叠加的询问技术，潜台词是“你怎么言行不一致？”，推动对方解释。

**思路概括**：学员与女生很久没聊，需要不掉价值地重启对话。采用“借势+询问”的组合，借用对方过去自我描述的性格特征来发起对话，并询问其“消失”原因，成功获得了对方的积极回复和后续互动。

### 案例六：借势处理约定变更

这个案例展示了当既定约定被对方取消时，如何用借势表达立场而不破坏关系。

**技术点**：使用“你之前”这三个字进行借势。
*   **“你之前说的这周结果没提早说”**：借用了对方原先答应“这周”见面的事实，来指出对方临时变卦的不妥之处。

**思路概括**：敲定好的约会被女方以“约满了”为由取消。围绕“不破坏关系但表明规矩”的思路，采用借势技术，引用对方先前的承诺来指出其行为不合适。对方道歉退让后，给予台阶但附加小条件（请喝咖啡），并立即敲定新的邀约时间，最终成功推进关系。

## 技术注意点 ⚠️

看完了丰富的实操案例，我们最后来总结一下使用借势技术时需要牢记的几个关键点。

以下是使用借势技术时的核心注意事项：
1.  **借利避害**：所借用的信息必须对把控聊天走向有利。借用不利信息会让聊天更难进行。
2.  **方式多样**：借势方式不限于@，还有“你之前”、“我记得你说过”、“你都说”等多种形式。避免方式单一，显得机械。
3.  **目的非争吵**：借势是为了更好地把控聊天方向与节奏，切勿沉迷于用它来争吵或指责对方，否则可能“赢了争吵，输了关系”。
4.  **搭配使用**：借势常与其他技术（如询问、延迟、夸赞）搭配使用以增强效果。因此，需要对其他主要技术有熟练的掌握。
5.  **思路重于术**：在实操中，不仅要注重技术本身，更要理解其背后的沟通思路。技术与思路（术与道）相辅相成，才能灵活运用。
