# 聊天大师「心动窗口撕裂」：第3课：联系感、安全感与邀约 🎯

在本节课中，我们将学习聊天流程中的第三步与第四步：如何与对方建立联系感与安全感，并最终成功进行邀约。掌握这些步骤，能让你的聊天自然推进，为线下见面打下坚实基础。

## 建立联系感与安全感 🤝

上一节我们探讨了如何通过开场白调动情绪。本节中，我们来看看如何让对话更进一步，建立彼此之间的连接。

你不会与一个毫无交集的人成为朋友。如果对方觉得与你根本不熟悉，便难以擦出火花。因此，你需要与对方建立一些共同点，形成联系感，作为你们之间的桥梁。

在建立联系感时，有一个非常实用的小技巧：**“也”字的运用**。例如，你可以说：“**你也去过三亚？**”或者“**你也喜欢吃小龙虾？**”。这类表达能有效拉近双方距离，暗示你们拥有共同经历。

以下是建立联系感的具体方法：
*   根据对方身上的特点或朋友圈展示的内容，寻找共同话题。
*   使用“也”字句来强调共同点，例如：“看来我们**也**都喜欢旅行。”

联系感中还有一个重要组成部分：**安全感**。试想，在与对方舒适聊天、情绪被调动、且已建立一定联系感后，下一步自然是邀约。但对方若在邀约时含糊其词或推脱，往往说明你在安全感建立上尚有不足。

女性通常是缺乏安全感的。在不清楚你的姓名、职业等基本信息时，她们会担心与你见面的安全性。因此，我们需要进行**基本信息交换**。

基本信息交换是指，与对方互相分享年龄、工作、星座等基本信息，让对方对你有一个初步了解，从而感到安全。

以下是进行基本信息交换的一个小技巧：
*   一般采用**先说自己，再问对方**的方式。例如：“我是做设计的，平时喜欢摄影。你呢？”

这种方式能比较自然地建立联系感和安全感。

## 成功邀约的三个原则 📅

在成功建立联系感与安全感后，聊天流程便自然进入第四步：邀约。如果前面步骤处理得当，邀约便是水到渠成的事情。

在邀约方面，我们遵循三个核心原则。

第一个原则是**确定**。一个真正的邀约，必须具备**确定的时间、确定的地点和确定的活动**。例如：“**周六下午6点，我们在太古里那家甜品店见面吧。**”这就是一个清晰的确定邀约。

然而，在提出确定邀约之前，我们必须先做一件事：进行**模糊邀约**。这是邀约的第二个原则。

模糊邀约旨在测试对方是否愿意与你出来，其与确定邀约的关键区别在于**不指定具体时间**。例如：“**改天有空的话，我们可以一起去太古里尝尝那家甜品。**”

如果对方回应积极（如说“好”），你便可以顺势询问：“**那你一般什么时候比较方便？**”从而将模糊邀约转化为确定邀约。这种方式能有效避免暴露过强的目的性。

第三个原则是**文字精简**。提出邀约时，请专注于邀约这件事本身，不要夹杂其他话题。否则，对方可能被其他话题带偏，导致邀约本身未被确认，聊天偏离主线。

掌握这三个原则，你的邀约成功率将大大提升。

## 总结 📝

本节课中，我们一起学习了聊天流程的关键环节。首先，我们探讨了如何通过寻找共同点和使用“也”字技巧来**建立联系感**。接着，我们说明了通过**基本信息交换**（先己后人）来构建**安全感**的重要性。最后，我们详细介绍了成功邀约的三个原则：先进行**模糊邀约**测试意向，再提出**确定**的时间地点，并且整个过程保持**文字精简**。遵循这些步骤，能让你的社交互动更加顺畅自然。
