# 约会艺术：02：如何更巧妙地聊天

在本节课中，我们将通过一个具体案例，学习如何在约会聊天中创造深度、保持吸引力，并巧妙地推进关系。我们将分析对话中的关键节点，并提供多种回应策略。

上一节我们分析了男生在回复工作话题时的状态。本节中，我们来看看聊天话题如何深入展开，以及如何通过回应展示价值、创造暧昧氛围。

---

**蓝叔**：那个工作应该不用休息的时候，一直盯着手机看吧。
**男生**：不有啊。除非是被强控了。
**男生**：Yeah。我工作需要。就是有重要的消息需要及时回的哦。
**女生**：是客户吗？还是什么？
**男生**：对对。😊，最难受最难受的时候。就是有时候生着病还得回消息那种感。
**男生**：那好在。在我晚上超过9点之后，基本上不会有人再打扰。就我可以安静的喝酒。
**女生**：喜欢喝酒啊，我就是每天睡觉，那你酒量是不是很好？

这里提供了一个男性视角的回应思路。如果女生问“那你酒量是不是很好？”，可以反问：“那你为什么不问我是不是喝酒为了麻痹自己、逃避生活？为什么要问我酒量好不好？”

从女性视角看，这样回复会让人觉得你有深度和故事感，引发好奇。这能避免聊天流于表面，为对话注入“营养”。很多人约会时聊天一眼看到头，对方会失去深入接触的兴趣。营养的成分在于你能否在话题中提供独特的见解或情绪价值。

对比案例中男生的回复：“看你怎么判断。比你好算好。” 这带有一些挑战意味。另一种直接的说法是：“我在不了解的情况下，我直接说你喝不过我。” 这也是可行的。年轻人有年轻人的套路，但也可以尝试不同的方式。

同时，男生在回复时扛住了压力，没有因为女生可能觉得他看手机不尊重人而慌张。他保持了内外一致的状态。部分学员会过度敏感，比如女生问能不能喝，就担心对方觉得自己爱玩。针对“酒量好不好”这个问题，可以提供以下方案：

以下是两种回应“酒量是不是很好”的策略：
*   **策略一**：你会不会给我贴标签？我回答“是”，你会不会觉得我在外面玩？我回答“不是”，那下次是不是还要约这种无聊的饭局？
*   **策略二**：以后你就知道了。

第一种回应给人两种感觉：一是让人觉得你洞察人心；二是为下一次邀约做铺垫，并暗示当前饭局可能无聊。酒本身是为了精神放松而存在的。你也可以不直接回答酒量，而是说：“我也不太知道，你说要不我们晚上约一下试试？” 重点不是答案，而是铺垫。
第二种回应“以后你就知道了”很好解释：因为想跟你有以后，所以开个玩笑。

**女生**：😊，我们继续，那你们放开水回去。我以为是个菜鸡，又菜又爱玩，就跟打麻将一样，又菜又爱玩。
**男生**：😊，那下次约你回。😀

男生顺势抓住了女生说自己“又菜又爱玩”的点，提出“下次约你喝酒”。潜台词是：你喝醉了，我才有机会。他没有像部分“正人君子”那样，因为女生喝不了就不约酒了，或者说“那我们喝奶茶吧”。情绪的推进就从这里开始。

如果是我个人，当女生说“我很菜”时，我会问：“是一杯就倒，还是一碰就倒？”

这句话的含金量很大。如果真这么回应，女生很可能会被逗笑。她会觉得这个男人蛮好玩的，而暧昧的氛围就在这些好玩的对答中产生。每个约会，不同的人有不同的玩法。

**女生**：我一般出去玩，我都是那种戏氛的。然后这朋友都会觉得我很能喝然但是其实。😊，可能两瓶就是我的量了。
**男生**：😊，两瓶啤酒吗啊。です。啊。
**女生**：就就是因为我会很太吸贵呀，然后然后朋友就会觉得哇它看起来一定很能喝。😊，然后我之前跟朋友我之前跟朋友们出去玩，就是蹦迪都还没开场之前就已经喝吐了，已经吐过一轮了。我现在不爱合诉，我就微信就可以了。就喝一瓶啤酒，你就先走吧。反正就是喝喝不完就真的就找朋友帮喝呗反正我朋友喝。😊，前面喜欢自己一个人在家。后来喜欢和他去兰桂坊做那种挑战，就是从头呃从头上的第一家酒吧开始，每一家喝一排是要从头喝到尾。
**男生**：这么牛，还有这种玩法，那我肯定不行，给我我肯定不行玩喽就了，中间再过一点的时候，就基本上都有点哦，那你酒量。
**女生**：我觉得酒天好的男生很加分。Yeah。

女生说“会喝酒的男生很加分”。真的是会喝酒加分吗？根本原因是眼前这个人做什么都加分。

以下是针对女生说自己是“氛围组”和“喝酒加分”的回应思路：
*   **针对“氛围组”**：可以调侃说：“氛围组是那种打扮得很漂亮，然后你懂的……” 或者更直接地调侃：“晚上夜班以这个也非常开……” 这只是一个轻松的调侃。
*   **引导预期**：可以说：“以后我们见面，我是不是应该被动一点？” 引导女生主动，给她拉高预期。
*   **针对“喝酒加分”**：可以回应：“喝酒是加分，但**喝完酒的我**会更加分。” 制造好奇，让她感受到每句话背后都有可以拆解的空间。

聊到后面，男生似乎没有捕捉到这些信息窗口。女生情绪很好，窗口很大，但有些被错过了。例如，当女生提到“氛围组”时，可以问：“氛围组岂不是每天晚上都有大哥？” 女生肯定会接这个话题。

男生当时呈现的状态有点“无欲无求”，女生不管说什么，他都只是“啊”、“嗯”，比较平淡。但中间有一个点他做得不错：当女生说自己酒量不好时，他没有迎合说“女生要少喝点”，而是反应“两瓶？”，这是一种不刻意迎合的状态，侧面展示了他是一个有主见、有自我意识的人。

当女生说“我以前都是喝吐了，现在我喜欢微醺”时，这是一个重要窗口。
以下是利用此窗口的两种思路：
*   **思路一**：可以说：“那我们差不多微醺的时候见？至少你知道自己在做什么。”
*   **思路二**：可以展示爱玩属性，并打预期。例如问：“你去酒吧是看哪个小妹妹？我不为喜欢去酒吧，你说得不到可以看……” 将话题引向关系推进。

约酒的话题可以聊出很多内容。例如，你可以通过这个话题去了解对方：“一周去几次？”、“一般在哪里玩？”，从而评估她是否符合你的标准。如果女生说经常在某个地方玩，可以回应：“那不去，去签一个男人的在那里亲亲我我多难受。如果有个女人，那我就没那么难受。” 这就在开始植入暧昧感了。

还可以问她：“你以前喝醉了，是清醒的还是不清醒的？” 如果她说不清醒，可以接：“没事，我也有过那个阶段。” 这里信息量很大。所以，喝酒是一个很好的话题点，但案例中错过了很多。
