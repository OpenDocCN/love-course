# 情感课程：022：诱惑阶段之在聊天中做预期

在本节课中，我们将要学习“做预期”这一核心聊天技巧。我们将探讨它在熟悉、暧昧和走心三个阶段中的不同作用与应用方法，帮助你与女生建立更自然、更深入的联系。

## 概述

聊天不同于恋爱，但可以划分为三个主要阶段：熟悉阶段、暧昧阶段以及走心阶段。这三个阶段相互联系，相辅相成。如果跳过或未完成某个阶段，直接进入下一阶段，通常会导致失败。因此，遵循循序渐进的感觉至关重要。

上一节我们介绍了聊天的三个阶段，本节中我们来看看什么是“做预期”。

## 什么是做预期？

做预期就是你去和女生描绘一个未来的画面。这个画面必须与你和她有关联。这样做的目的是，当你日后真的去做这些描述过的事情时，她不会觉得尴尬。同时，这也能增强你们日后互动的流畅性，降低女生的抵触心理。

做预期贯穿于每一个聊天阶段。它不仅仅是为了方便在约会中与女生牵手、接吻，而是在任何一个阶段都必须进行的铺垫。

## 各阶段如何做预期

接下来，我将和大家详细讲解每个阶段如何具体进行预期铺垫。

### 第一阶段：熟悉阶段

在熟悉阶段，聊天的首要目的是互相了解。此阶段的预期应让对方认为你们的聊天会延续下去，并且你是一个有标准、有选择的人。

以下是熟悉阶段常见的预期类型：

1.  **暗示会见面**：当女生说“我超级喜欢吃火锅的，而且很能吃”时，你可以回复：“别吹牛，以后见面的时候，我可是会点一大桌菜，到时候可别自己打自己脸嘛。” 这预期了你们未来会见面。
2.  **暗示以后会多聊天**：当女生说“其实呢我比较慢热”时，你可以回复：“看来以后你要多找我聊聊天了。我平易近人的小秘密，就只和你一个人说过。” 这为持续聊天埋下了伏笔。

### 第二阶段：暧昧阶段

在暧昧阶段，目标是升高关系。这个阶段需要给女生营造你们可能向情侣方向发展的预期，并委婉表达你的兴趣。

以下是暧昧阶段常见的预期类型：

1.  **委婉表达兴趣**：当女生问“你喜欢什么样的女生”时，你可以回复：“你再努力一下，我就快喜欢上你了。” 这是一种曲解，预期了关系可能的发展方向，并测试了女生的态度。
2.  **植入未来生活画面**：当女生说“我昨天不小心把手给划破了”时，你可以说：“拍个照片给我看看严重不严重。” 待她发来照片后，接着说：“怎么这么不小心，手伤了以后，怎么还帮我洗袜子和内裤呢？” 这植入了亲密的生活场景。
3.  **植入见面后的肢体接触预期**：在约好见面后，你可以说：“明天要见面了。如果你觉得我不是坏人呢，你就主动牵我的手。如果我觉得你是个好姑娘，那我就找一个没人的地方偷偷亲你一下，这个就当做是我们之间的暗号了。” 这直接为约会中的亲密行为做了铺垫。
4.  **植入性话题预期**：这是最容易遇到女生废物测试的领域。例如，当女生问“你有没有约过炮啊”，你可以回复：“我从不约炮，我只是制造爱情。” 或者在聊到喝酒时，可以说：“如果我喝醉了，记得带我回家。如果你喝醉了，我会送你回家。如果我喝得不省人事，请直接睡我。” 这为后续更深入的交流做了预期。

**核心法则**：只要女生没有明确拒绝，那就是默认。在约会时，对于牵手、接吻等行为，可以大胆尝试。

**常用技巧**：**曲解**与**推拉**在这个阶段经常使用，能有效带动女生的情绪。情绪被反复刺激后，她对你的感觉会越来越深。

### 第三阶段：走心阶段

在走心阶段，目标是建立真诚的情感连接。此阶段的预期方向是描绘美好、平淡而真实的生活画面，让女生感受到这样的生活容易实现。

以下是走心阶段常见的预期类型：

1.  **表述自己与未来另一半的生活**：当女生问“你对你未来的女朋友会很好吗”时，不要直接回答“会”或“不会”。可以说：“我要的爱情很简单，一起手拉手逛街，一起吃饭，一起看电影，一起在家里面窝着看剧，你看电视，我做饭，夸我做的好吃，然后呢，做完了之后呢，我们俩一起去洗碗，分享彼此的喜怒哀乐，偶尔呢我们也会吵架，我呢会过来哄你，你也很好哄。” 这将她规划进了你未来的美好生活中。
2.  **描绘自己与另一半的孩子**：当女生问“你喜欢男孩还是女孩”时，你可以说：“我想要一个男孩，一个女孩。男孩是哥哥，女孩是妹妹，我们一家四口穿着家庭套装一起出门，哥哥照顾妹妹，我做饭，你洗碗，我们两个小宝贝帮我们一起来做家务，就这样过着温暖的日子。我们看着他们慢慢长大，他们守着我们俩慢慢变老。这就是我想要的幸福。”

这种美好、平淡、真实的画面，是此阶段最主要的表达方向。要让女生感受到这样的生活触手可及，而非虚无缥缈。无论女生价值多高，内心都向往这种柔软、真实的美好。

## 总结

本节课中，我们一起学习了“做预期”这一核心技巧。我们明确了它在聊天三个不同阶段（熟悉、暧昧、走心）中的独特作用和具体操作方法。从暗示未来互动，到植入亲密画面，再到描绘长远的生活蓝图，做预期是让关系自然升温、降低女生抵触心理的关键。记住，循序渐进和画面真实感是成功运用这一技巧的要点。
