# 14_8-8早泄的分类

早泄肾气充足的早泄又可以分为肽敏感早泄和基础兴奋度过高早泄。肽敏感早泄又可以分为心理敏感早泄和生理敏感早泄。我先说一下肾气不足的早泄，肾气不足的早泄会身体会有4个迹象。第一，很少沉勃，第二，硬度不应。

第三，做爱容易中途疲软，第四，就射精无力。如果有上述的四种迹象，那么基本上可以断定为肾气不足的早泄，肾气不足的早泄主要集中的人群是30岁以上。他们随着年纪的增长，肾气也在不断的消耗而减少。

容易出现肾气不足。接下来我们讲一下肾气充足的早泄，肾气充足早泄可以分为太敏感早泄和基础兴奋度过高早泄。很多人是这两种情况都存在。既敏感又兴奋度过高，人群主要集中在十几岁二十来岁。

他们主要是因为性爱经验不足或者长期手淫而造成的敏感，或者没有对。以及分居造成的基础兴奋度过高，生理敏感指的是我们阴茎耐受力强度不够。而心理敏感指的是对于女生所发出的一些性刺激信号，触觉、嗅觉比较敏感。

通过本节的学习，我们要大概明白早泄的分类，同时可以自我评估身体情况，知道自己属于哪一类。接下来讲一下如何判断自己是否肾气充足。第一种方式，久微智能精准测评，通过填写身体的情况，系统给出测评结果。

第二种方式，如果很少沉勃，硬度不硬，射精无力，甚至是流出来，事后感觉身体特别累，做爱过程中经常出现中途疲软。如果有上上面种种现象，基本上可以判断为肾气不足。好的，今天的课程就分享到这里了。

重点就是要明白早泄的分类，可以判定出自己属于哪种情况的早泄。
