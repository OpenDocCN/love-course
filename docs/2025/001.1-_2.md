# 情感修复课程：001.1-1：分手后的自我审视

在本节课中，我们将要学习分手后首先需要想明白的几个核心问题。认清自己的真实动机和感受，是决定是否挽回以及如何挽回的第一步。盲目行动往往只会带来更多伤害。

很多人经常问爱情是什么。爱情是一种很悬的东西，如影随行，无声无息，触没在我们的心底，让我们牵肠挂肚，痛苦快乐并存。如果你目前还存在于真命的羁绊当中，就像大话西游里面桥段一样，曾经有一份真正的爱情摆在我面前，我们要好好珍惜。当我失去的时候，才后悔莫及。那么这个时候我们建议大家先冷静的思考一下。

你们的时机是否成熟，你是否真正的喜欢他、爱他，而不是因为不甘被甩而去想要挽回他。只有真正的处理好彼此的关系，那么结局的完美，不然很好。就算结局惨淡，也不要泄气。生活还得继续，明天阳光依然精彩。也许这并非是你的良人，只是生命当中一个过客而已。我们总会在那个对的时间，遇到对的人。

如果你真的想明白了，想要挽回真命。希望你在做出决定之前，先认真的思考一下几个问题。

以下是五个需要你认真思考的核心问题：

1.  **你为什么要挽回他？**
    盲目的挽回，最终都是无疾而终的。只有你真正的想明白了你喜欢他什么。有时候我们根本不明白我们为什么要去喜欢一个人，但真正当你意识到的时候，你会清晰的知道你的恋爱之路该如何去走。这里跟我们相处长期关系是类似的，想明白你为什么喜欢他、爱他。建议你真正的好好思考一下他身上的优点、缺点，以及你是否能够接受、容忍、包容，接纳他现实里的一切，愿意为他付出，因为两个人的情感修复去努力，再去做这个决定。只有这样才会有一个很好的结果。

2.  **你是否真的爱他，还是仅仅因为不甘心？**
    很多人一时用气，想要去冲动的挽回。这样的行为，其实只是在伤人伤己而已。因为你觉得自己的面子很重要，放不下，觉得被甩或者说失恋是一件很丢人的事情，想要去找回颜面，以此证明自己是对的、是没有错的。那么你做出这些行为以后，也许在未来会让你后悔莫及。所以当我想要去挽回的时候，一定要考虑清楚我们这个人真实的挽回原因。也许当你要回了这个面子，但是后面却给两个人造成了更深的痛苦，不仅是你痛苦，这个女孩也跟你一样痛苦。两个人就像在电视中看到桥段一样相爱相杀。男人其实就应该拿得起放得下面子，和我们幸福比起来，你觉得哪个更重要呢？

3.  **你是真心悔改，还是害怕找不到更好的？**
    很多人之所以想要挽回，是因为在现有的女生当中，他觉得自己前女友是最好的那一个。那么这只能说明你的社交范围相对来说比较狭窄，你没有更多的选择。如果你本身交际圈足够大，你也不缺乏一些优秀女孩子，那么你还是想要挽回她，你是真的喜欢她爱她，真的想要悔改。那么恭喜你，这个女人将是你最终的目标，是你的真命天女，是你未来能够一起走下去那个人。但如果仅仅是因为怀疑自己找不到更好的，先想办法占有她，其实只是满足自己的一个占有欲。那么这个时候，我希望你能够将自己的精力放在自我提升上去，想清楚自己未来到底想要什么。最后一天，你可以变得非常非常优秀，那么你也许会遇到一个跟你一样优秀的女孩子。

4.  **你是否能承受挽回路上的漫长与痛苦？**
    挽回不是说一天两天，或者说你给女生说两句话，就能够轻易挽回的。因为挽回是需要付出一定代价的。它比重新认识一个姑娘开始一段新的爱情要难得多，是一个纠错的过程。你必须为此付出一定努力，甚至你要去从自己身上改变原有的一些不足去不断精进，能够达到女生的要求点。这不是一个短期的行为，它是需要一定过程的，甚至需要一定时间沉淀，是有一个努力改变的过程。所以你是否已经下定了决心，真心地愿意为这个女生付出这么多。如果你真的做好这样的准备了，那么你的挽回才会有结果。

5.  **你们是否真的合适？**
    其实最关键的还是因为你们性格是否合适所决定的。如果你们重新在一起，你是否能够包容她离开你和离开你所做的一切。因为有可能他离开你之后，又会发生很多的事情，你是否能够接受？你是否有信心在你们今后的生活当中，能够更加幸福美满？如果说你们两个人没有在一起，也许你过得比现在更好。如果说女生她也拥有一个更好的生活，那对两个人来说都是一个美好的结果。所以说我们所有这一切，性格是否真的合适，能够合拍，完全能够生活在一起？你是否还是像以前那样天天跟他吵架打冷战？那么双方都不快乐。如果还是这样的话，那么为什么还要继续呢？而在一起本身就是一个错误，为什么要让这个错误继续下去呢？彼此折磨对方，不如将这个美好的错误当做一个回忆，让它成为我们的过去，彼此寻找一段新的美好的开始。

如果这些你都想明白了，想清楚了。那么这些问题在你的心目当中都不是问题。内心也真实认识到自己所犯的错误，能够包容他，并且不把责任都推到他的身上。那么恭喜你，他也许就是你生命当中的真命天女，他就是你最终那个目标，其实可以义无反顾的去做，努力的去追回他。这一辈子，也许是你最珍贵的财富了。可能你真的爱过这一次之后，你就不会再爱的这么深、这么彻底了。

因为挽回也是有一定的几率，理论上是能够达到百分百成功。但是现实当中有那么多的如果，必然有各种各样的因素会出现。而未来所发生的事情，都不是我们所能够决定的。但如果你仅仅是一个多情的种子，对你所喜欢的女生都投入很多的感情的话，无法释怀时的离开，我建议大家还是谨慎的操作。对于一段没有结果的感情，一开始就应该想清楚，我们可以这样做，但最终还是要审视自己的价值观。那么自己这么做，到底值不值得？你是否应该投入这么多？那到底是不是你最终想要去携手终身的那个人？如果说对方仅仅是一个过客的话，那么为什么我们要去强求呢？俗话说强求的瓜不甜，该过去的就让他过去。也许在未来我们某一天回忆起来，内心当中会有更加丰富的回忆，会觉得两个人彼此在一起的时候，都有一点美好的东西。

只有我们真正在感情道路上能够做到认清自己，那么我们才能够走得更长远一点。

本节课中我们一起学习了分手后决定是否挽回前，必须进行的五个自我审视。核心在于区分真爱与不甘、评估挽回代价，以及判断双方是否真正合适。认清自己，是情感修复道路上坚实的第一步。
