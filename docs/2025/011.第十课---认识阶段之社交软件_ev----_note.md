# 情感课程：011：认识阶段之社交软件

在本节课中，我们将要学习如何通过社交软件高效地积累女生资源。我们将重点讲解主流社交软件的使用方法、个人展示面的打造技巧，以及如何将线上资源有效转移到微信进行后续互动。

上一节我们介绍了改变自身形象和展示面的重要性，本节中我们来看看如何利用社交软件解决资源获取的问题。

## 探探的核心玩法与界面

探探的界面简洁明了，其基本操作规则是：**左滑表示不喜欢，右滑表示喜欢**。当你右滑一个女生，且对方也右滑了你，则代表匹配成功，双方可以开始聊天。

玩探探的核心目标是提高匹配率。匹配越多，能收集到的女生资源就越多。而匹配成功率，直接取决于你个人主页照片墙上的6张照片。这6张照片构成了你在探探上的展示面。

## 打造高吸引力照片墙

以下是关于6张照片的搭配与排列建议，目的是塑造一个形象良好、生活精彩的个人形象。

**照片搭配模式推荐**：4张个人照片 + 2张生活场景照。

**照片排列顺序建议**：
1.  **第一张**：放置一张清晰展示面部的大头照或半身照。这是决定对方是否愿意进一步了解你的关键。
2.  **第二张**：放置另一角度、同样清晰的大头照或半身照，提供更多面部信息。
3.  **第三张**：放置一张全身照，展示整体形象和衣品。
4.  **第四张**：放置一张能体现个人特点的照片（例如：秀身材、与小动物合照、开车时的照片等）。
5.  **第五张**：放置一张美食照片。
6.  **第六张**：放置一张任意高端场所的场景照片。

这样排列的逻辑在于：通过前四张照片建立基本吸引，证明你是真实且外形不错的；后两张生活照则进一步展示你精彩、有品质的生活，从而在女生脑海中形成“形象好、生活丰富”的积极印象。

## 个人信息的填写技巧

在照片墙之外，个人信息栏的填写也需注意。以下是三个重点区域的填写建议：

1.  **公司信息**：不建议填写具体公司名称，可能会显得直白且缺乏吸引力。建议留空。
2.  **个人签名**：建议采用**文艺**或**搞笑**的风格，避免平庸。
3.  **我的标签**：可以采用调侃式语句。如果身高有优势（如1.8米以上），可以明确标出，这对女生具有吸引力。

## 从匹配到微信：资源转移策略

探探等社交软件应主要作为**资源收集平台**，而非主要聊天平台。我们的目标是将匹配到的女生引导至微信进行深入交流。

**转移微信的时机**：不要一匹配成功就索要微信。应先简单交流五六句，建立初步熟悉感和印象后，再提出加微信的请求。

**转移话术示例**：
> “这个软件我不太上，要不我们加微信聊吧？”

成功添加微信后，建议在探探上解除与该女生的匹配，避免对方看到你频繁在线而产生负面印象。如果被问起，可以解释为“加到微信方便了，这个软件就不怎么用了”。

## 注意事项与风险防范

在使用社交软件时，必须注意以下事项：

*   **严禁低俗行为**：绝对不要发送骚扰信息、低俗言语或暴露照片，否则极易被举报封号。
*   **识别虚假账号**：社交软件上存在骗子、假人和酒托。酒托通常有几个特征：一上来就异常热情、绝不提供电话号码、聊天内容机械重复。请保持警惕。
*   **多平台并行**：如果觉得单一软件资源不够，可以同时使用多款社交软件（如积木、牵手）来收集资源，提高效率。

除了探探，还有其他软件可供选择：

*   **积木与牵手**：玩法与探探类似。目前积木的用户质量相对较高，牵手的匹配率较高。
*   **青藤之恋**：该软件需要进行学历认证（专科及以上），因此用户整体教育程度较高，但可能“子女”（指代不明，可能指特定用户群体）也较多。展示面原理相同，但个人介绍建议写得**正经、平常**一些，以符合平台相对严肃的氛围。

此外，线上社交软件可以配合**街头搭讪**来获取资源，关于搭讪的具体技巧我们将在下节课详细讲解。
