# 自媒体与个人IP：03：朋友圈内容策略：按工作性质而非行业划分

在本节课中，我们将学习如何根据不同的工作性质，而非简单的行业分类，来规划你的朋友圈内容。理解并应用这一策略，能让你更有效地通过朋友圈展示个人价值，放大你的专业影响力。

上一节我们探讨了朋友圈的基础定位，本节中我们来看看如何根据你的核心工作属性来定制内容。

## 🎯 理解三种核心工作性质

与其按行业划分，不如按工作性质本身来划分。当前及未来的工作性质主要可分为三种：

1.  **做传播的媒体人**
2.  **做产品的手艺人**
3.  **做资源整合的运营人**

接下来，我们逐一理解这三种工作的本质。

### 1. 媒体人：信息的传播者

媒体人工作的核心是**接收信息后进行传播**。例如老师、讲师、销售、公关等都属于此类。他们通常表达能力与获取信任的能力很强，人际交往（people skill）能力突出。

### 2. 产品人/手艺人：价值的创造者

产品人，或称手艺人、业务人，负责**生产、创造、研发与执行业务本身**。例如专业的律师、税务师、会计师、算法工程师等都属于这一类。

### 3. 运营人/资源整合者：关系的连接者

运营人负责**牵线搭桥与资源整合**。他们擅长将资金、用户、关系等要素盘活，创造更大价值。

这三种工作性质并无高下之分，任何一类做到极致，都可能成为领域的顶尖人物。

## 📝 三类人的朋友圈内容策略

理解了工作性质分类后，我们来看看每种属性的人应如何把握发朋友圈的调性。

### 第一类：媒体人（传播者）的朋友圈策略

如果你是传播类的媒体人，核心目标是**让他人养成阅读和订阅你朋友圈的习惯**，成为你朋友圈的读者。

以下是构建内容的两层策略：

*   **基本段位：成为优质信息源**
    持续传播你所在行业或领域内的有价值信息。例如，如果你是护肤品销售，不应只推销产品，而应分享护肤干货、新趋势、技术与品牌动态。公式为：**朋友圈内容 = 行业优质信息**。

*   **更高段位：信息源 + 个人观点**
    在传播信息的同时，结合并表达你个人的看法与解读。让他人对你的观点产生兴趣、信任甚至依赖。公式进阶为：**朋友圈内容 = 行业优质信息 + 个人独特观点**。

**总结**：媒体人的朋友圈主调性应是**优质信息源与个人观点的结合**。

### 第二类：产品人/手艺人（创造者）的朋友圈策略

如果你是产品人或手艺人，例如文案、剪辑师、摄影师、律师等，你的朋友圈应聚焦于展现两件事：

以下是需要展示的两个核心维度：

1.  **展示热爱**：不断向他人展示你对自己所从事事业的真诚热爱。因为足够热爱才会足够钻研，足够钻研才会足够专业。
2.  **展示擅长**：展示你在此事上的专业能力，例如做得快、做得好、服务过哪些客户、取得过什么成果。公式可概括为：**朋友圈内容 = 主观热爱 + 客观成果**。

**总结**：产品人需同时展示**主观的热爱**与**客观的擅长**。

### 第三类：运营人/资源整合者（连接者）的朋友圈策略

如果你是运营人或资源整合者，例如商务拓展角色，你的朋友圈应塑造特定的个人形象。

以下是构建形象的两个层次：

*   **基本段位：秀资源**
    展示你是一个有资源的人，认识的人多、去过的场合多、熟悉的品牌多。
*   **更高段位：秀魅力**
    在资源基础上，展示你的个人魅力，例如你是一个很友善、重情义、懂感恩的人，让大家都愿意与你共事。

**总结**：运营人的朋友圈应从**展示资源**向**展示个人魅力**升华。

## ⏱️ 内容发布频率建议

不同工作性质的朋友圈更新频率也应有所差异：

*   **媒体人（传播者）**：频率最高，建议**日更**（每天一次）。
*   **运营人（连接者）**：频率适中，建议**隔日一更**。
*   **产品人（创造者）**：频率无需过高，建议**每周2-3更**。

本节课内容值得课后制作思维导图以加深理解和消化。

## 🎓 总结

在本节课中，我们一起学习了如何根据三种核心工作性质——**媒体人（传播者）、产品人（创造者）、运营人（连接者）**——来规划朋友圈内容策略。关键在于理解自身属性，并采用对应的内容调性与发布频率，从而让朋友圈成为你个人能力的有效放大器。
