# 蓝图解码：3：无论如何，先干起来 🚀

在本节课中，我们将要学习如何克服行动障碍，掌握“先干起来”的核心心法。我们将探讨完美主义、时间成本、最小可用原则等关键概念，帮助你迈出行动的第一步，并建立起持续做事的正向循环。

## 课程概述

上一节我们介绍了如何补充和守护你的能量。本节中我们来看看如何将这些能量转化为实际的行动与成果。我们将深入探讨阻碍我们行动的各种思维陷阱，并提供一套简单直接的方法论，让你能够立刻开始做事，并在过程中不断调整和进步。

## 为何要将“能量”置于首位？

在之前的课程中，我们首先讨论了能量问题：第一，堵住能量泄露的漏洞；第二，以高能量的方式生活，补充能量。我之所以将身体、心理和心灵能量放在最前面讲解，甚至分享了一些通常被视为“秘密”的内容，是为了一个核心目的：**减少你们的胡思乱想和困惑**。

当你的大脑——这个人体最耗能的器官——在没有目的地空转时，它会消耗巨大的身体热量和精神能量，而你却什么也做不了。这是最大的能量浪费。明白了这一点，不再陷入无谓的思虑，你就能堵上这个最大的能量漏洞。拥有充沛的精力，你的身体才能听你使唤，真正开始做事。

**做事情，是我们建立良好心态的唯一坚实基础**。心态绝非空中楼阁，不能靠空想或伪装来维持。我们是要靠自己的双手去真正获取生活中想要的东西，无论是财富还是情感。因此，你必须先通过锻炼身体、早睡早起、保证营养、断除消耗来积蓄能量，才能真正开始做事。

## 干起来的甜头与踩过的坑

在深入方法论之前，我想分享最近的感受：当我们团队全身心投入做事，并自然地在朋友圈分享这种积极状态时，我与异性的互动变得异常轻松，对方原有的那种“高姿态”也荡然无存。这印证了一个道理：**只要你好好做事，并展现出积极向上的精神面貌，异性对你的反应自然会变好**。

然而，我也曾错过重要的机会。我几乎错过了短视频和直播的“大红利期”。所谓“大红利期”，就是每个平台或行业初期，只要你是敢于第一批尝试的人，做什么都容易成功。

*   **公众号红利期**：我敏锐地发现了公众号的潜力，并向当时的主管建议。但市场部负责人却认为扫描二维码关注太麻烦，断言“这东西不行”。我没有被他们的看法阻碍，自己做了公众号，但后来因为其他原因（包括在两性关系上踩的坑）未能持续，错过了成为早期自媒体大咖的机会。
*   **短视频红利期**：2018年抖音兴起时，我再次注意到了。但我被“容貌焦虑”、“对专业资历的不自信”以及“不知道具体该怎么干”这三个问题困住了。我花了大量时间去健身、减肥、学习，却**因为缺少一套有效的“成事方法论”而错过了风口**。

我分享这些“坑”是为了说明：思虑过多、追求完美、害怕开始，是阻碍我们抓住机会的最大敌人。

## 核心心法：无论如何，先干起来

那么，如何克服这些障碍？核心心法就是：**无论如何，先干起来**。

### 1. 先造一辆能跑的车，别想着造火箭

我们往往想得太多，阻碍了行动。第一点就是：**先造一辆简陋但能跑的车**。

这就像吉利汽车的李书福所说：“汽车不就是摩托车加两个轮吗？”然后他就干起来了。做事情也是如此，先让一个最基础、可用的版本运行起来。

*   **我们的实践**：你看我们现在的课程页面，最初就是一个极其简单的店铺：一张我的个人形象照、一段课程介绍、一个直播功能。主页甚至只有一行字。但这足够了，它能让人了解课程、完成购买、观看直播。**这就叫“跑起来了”**。
*   **给你的启发**：这个过程本身对你们就是一种激励。你们看到的是一个正在成长、迭代的过程，而不是一个已经完美无缺的成品。这更有感染力。

**记住：在所有成本中，时间成本是最高的，因为时间一去不复返。**

### 2. 拥抱“失败”，追求阶段性反馈

很多人因为“完美主义”而无法开始。今天我们就解决这个问题。

看看那些真正成事的人怎么做：**马斯克的SpaceX通过不断“炸火箭”来进步**。他们甚至用不锈钢造火箭，降低每次“失败”的成本。他们不认为爆炸是纯粹的失败，而是收集数据、不断精进的必经过程。相对于花十年造一个“完美”火箭，他们通过快速迭代，可能两三年就达到了目标。

**所以，忘掉“完美主义”。我们要追求的是“最小可用”和“快速反馈”。**

以下是行动的关键思维转变：

*   **从“追求完美”到“禁止完美”**：完美是虚无缥缈的。接受一开始的粗糙，追求“最小可用”。只要能用，就先跑起来。
*   **从“不求回报”到“我就是要回报”**：这里的“回报”不是向他人索取金钱或认可，而是**追求做事过程中的反馈**。无论是好是坏，反馈能让你知道是否走在正确的路上，以便调整。SpaceX炸火箭得到的各种数据，就是最宝贵的“回报”。
*   **核心公式：Think Big, Start Small**
    *   `目标要大，起步要小`。
    *   你想上火星？第一步可以是买一张太阳系地图。
    *   你想说流利英语？第一步可以是学会查单词，然后立刻开始阅读一句英文。

### 3. 最小可用原则

这是“先干起来”的核心操作指南。

**最小可用原则：只要这个东西的核心功能能用，就可以开始。**

*   **学英语**：最小可用不是背完2万单词，而是**会查字典**。拿起一段英文，查单词弄懂意思，就可以开始阅读了。
*   **拍短视频**：最小可用不是买齐专业灯光、相机、背景。而是**用你的手机，找一面干净的墙，在能看清脸的光线下，对着镜头分享你的知识**。比如，群里那位擅长西餐的朋友，完全可以从分享一道简单菜品的制作开始。
*   **我们的课程**：最小可用就是有一个能展示课程、能直播、能收款的页面。它不需要精美的设计，能转起来就行。

**当你做到“可用”的七八成时，实际上你已经击败了90%的竞争对手。** 因为绝大多数人都被“完美主义”束缚了手脚。硅谷有句名言：“如果你的产品发布时看起来不是很糟糕，那么就意味着你发布晚了。”

### 4. 无视噪音，保护梦想

开始行动时，你会听到各种“你不行”的声音。

*   **承认自己一开始会很糟糕**：这反而能让你轻松上路。不要害怕被指摘，拉黑那些打击你、用旧认知局限你的人（所谓的“村里人”）。
*   **无视一切否定**：无论是出于“为你好”的劝阻，还是单纯的打击，都要无视。电影《当幸福来敲门》中说：“You gotta protect your dream.（你必须保护你的梦想。）”马斯克造可回收火箭时，所有人都说不可能，他干成了。我当年去新东方面试教英语，我的专业是日语，简历根本不对口。我直接堵住校长递简历，第一次试讲被拒后，我调整方向再去试讲，最终成功了。别人说“你不行”时，对我毫无打击。

**你的重要性在于，你发现自己有的问题，很多人也有。你突破了，很多人却突破不了。所以，只要你敢跨出第一步，就已经超越了90%的人。**

### 5. 活在执行中，而非焦虑中

市面上常讲“要活在当下”，但这概念容易造成新的焦虑——总在检查自己“是否在当下”。

**忘掉“活在当下”这个概念。** 它是一个**必然的结果状态**，而非需要追求的目标。只要你以正确的方式投入做事，自然会进入这种状态。你这几周是否感觉焦虑和胡思乱想少了？这就是因为你开始在行动了。

**人活着就是“劳碌命”（张朝阳的调侃）。不活在目标的执行里，就会活在焦虑里。** 新东方最年轻的副总裁庄重，在公司上市财富自由后，挥霍一个月反而抑郁了，因为人生失去了方向和执行的目标。

**无论男女，都要找到你的事业去执行。** 男人开疆拓土，女人也可以经营自己的生活、事业、外形。当你专注于执行，魅力自然产生，情感问题也会随之化解。

### 6. 关于目标与创新

*   **不设死目标，但要知方向**：不要用有限的认知设定一个框死自己的KPI（比如今年必须赚XX万）。这会造成焦虑并限制你的潜力。**但要清楚行业顶级的标杆在哪里**，并去研究、模仿他们达成目标的路径。这让你既有信心，又有方法。像亚马逊这样的公司，内部甚至不讲预算和KPI，只讲如何服务好客户。
*   **你本身就是创新**：不要被“我要创造一个全新东西”的想法束缚。**太阳底下无新事，但你是独一无二的。** 只要你在做事，你独特的经历、知识和表达方式，就会让你的产出成为“非标品”。总会有属于你的受众为你买单。创新是在你行动的过程中自然发生的。
*   **关于“抄、超、钞”**：先模仿（抄），获得支持和能量；然后用资源放大能量，实现超越（超）；最终获得财富（钞），而财富是能量的放大器。这是一个合理的成长路径。

### 7. 利他是结果，而非出发点

最后，不要被“必须利他”的道德感压得无法行动。**利他是自然结果，而非强制出发点。**

强行要求自己“无我”、“利他”是反人性的。我们做事的出发点，可以是为了**体验过程中的乐趣和成就感**。当我们觉得做某件事有趣、投入其中时（比如我觉得在群里和大家互动、设计课程很有趣），这个过程自然而然就会对他人产生价值（利他的结果）。

所以，**以正确的方式，做你觉得正确且有趣的事**。你会感到快乐，同时也会为社会创造价值，在他人心中变得更有吸引力。情感上的成功，只是这个过程的副产品之一。

## 总结与行动号召

本节课我们一起学习了“无论如何，先干起来”的核心心法。我们探讨了如何用“最小可用原则”克服完美主义，如何通过追求阶段性反馈而非最终结果来保持动力，以及如何无视噪音、在行动中自然成长。

**记住：**
1.  **Think Big, Start Small.**（目标要大，起步要小。）
2.  **先造一辆能跑的车，别想着造火箭。**
3.  **做到七八成，击败90%人。**
4.  **你本身就是独一无二的创新。**
5.  **活在执行中，而非焦虑中。**

现在，是时候干起来了！无论你想在职场上突破，还是开展一项副业，或者就是开始健身、学习一项新技能。**在咱们的社群里大胆分享你的想法，讨论你的计划。** 思想的碰撞可能就会擦出火花，一个伟大的事业或许就此萌芽。

我们做“蓝图解码”课程的终极目的，不仅仅是情感，更是**成长**。当你们真正成长起来，创造了价值，我们相信一切美好的事物，包括财富和情感，都会随之而来。

**现在就行动起来吧！**
