# TC Yoga 1：01.拜日式

大家好，我们来演示半运式。首先，山是站在垫子的前侧，双脚分开，跟骨盆同宽，脚掌外侧和垫子外侧平行。好，自然呼吸来到祈祷，双手合十胸前，大拇指的指尖触碰锁骨的下人。自然呼吸调整，全身放松。吸气后弯。

骨盆往前推，性器官往前推，手掌心冲前，胸腔打开后弯。头部往后仰。好，吐气前弯。双手放在双脚。是。好，准备去骑马，以大步腿，右腿往后。抬头胸腔保持延展。后侧的右腿伸展。

我们现在做的起码和之前做的有一些不同，我们不需要把这个膝盖放到垫子上去了，双手掌心压实垫子一大步腿左腿往后。来到下蜷压缩脚踝，把臀部拎包顶吸。膝盖。胸腔下巴着地区肿，8体投地。吐气。然后吸气。眼镜蛇。

丙烯。下去。右手。指尖点地，右脚一大步往前迈。回到骑码，抬头往后看胸腔延展。双手指尖点地一大步，收左脚来到右脚的旁侧，双手放在双腿的。侧前湾。吸气。好啊。吐气。回到祈祷。好，我们再来一次。吸气。后湾。

掌心冲前。吐气。前弯。双手放在双脚的旁侧。在下一组的时候，我们先撤左腿往后。双手指尖点地一大步。推左腿往后，起码是胸腔延展抬头，双手掌心压实垫子一大步，推右腿往后。下去压缩脚踝，臀部拎高，并息。

好后吐气。下巴吸收着地。班体。吸气。蛇氏。丙烯。下去。吸气。左脚前迈。吐气右脚前迈前屈。吸气。好啊。吐气。回到祈祷。这样就完成了一组的练习，你学会了吗？
