# 自媒体与个人IP：03：表达的媒介1

## 概述

在本节课中，我们将要学习三种核心的表达媒介：文字、图片和短视频。我们将逐一分析它们的优势、劣势以及在实际应用中的策略，帮助你构建一个清晰、有效的自媒体内容框架。

---

## 文字表达

上一节我们介绍了个人IP的定位模式，本节中我们来看看具体的表达媒介。首先从文字开始。

从生产的角度来说，文字的产出是所有表达媒介中最简单、效率最高的。你只需要一台手机，就可以进行创作。例如，撰写短微博、金句或段子，一天产出十几条是可行的。相比之下，录制短视频一天产出3到5条已属高效。

从消费端来看，文字表达是最经典、最持久的媒介。一篇优秀的文章或长微博，其生命力可以延续数年。而一条短视频或一场直播的回放，其内容通常与特定时期的热点话题和镜头语言紧密相关，难以在多年后仍具回味价值。

在文字、图片、图文、短视频、中长视频、播客和直播所有这些媒介中，能够承载深度思考的主要有三种：**文字、长视频和播客**。而在这三者中，**文字**是最能承载深度内容的媒介。

然而，文字表达的劣势也很明显。当前是短视频与直播的时代，文字在互动感、冲击力和实时性方面相对较弱。因此，单纯依靠文字进行账号冷启动变得较为困难。

以下是关于文字表达的具体策略建议：

*   **核心平台**：最能发挥文字功底和文案优势的平台是**公众号、微博和私域朋友圈**。
*   **战略组合**：如果你的核心能力是文字，可以主打文字内容，但建议辅以直播或短视频进行公域引流。战略是通过视频内容吸引关注，再通过深度文字内容建立粘性。

总结来说，关于文字表达的观点很清晰：
1.  如果文字是你的核心能力，不要放弃。文字表达能够穿越周期，永远拥有受众。
2.  根据当前社交媒体现状，单纯依靠文字（如只做公众号或微博）启动账号较难，需要辅以视频或直播。

## 图片表达

接下来，我们探讨图片这种表达媒介。

在早期网红时代，纯靠图片成名的案例很多。如今，虽然纯图片网红难以复制，但图片的拍摄、展示和表现能力对所有自媒体人而言都是刚需。**图片搭配文字、作为短视频封面、直播封面等，能极大地增强个人品牌感和IP感**。

如何全方位提升图片能力？以下是整体的提升框架：

**静态摄影 + 自拍 + 他拍 + 修图**

1.  **静态摄影**：刻意提升静态摄影水平，包括拍摄食物、花卉、场景等。这能展现你的审美和品味，并通过拍摄工作、家居场景来强化人设。提升方法在于多拍、多思考、多观察。
2.  **自拍**：自拍私人感强，最能拉近与粉丝的距离，增强粘性。自拍技巧关键在于互动感、眼神和整体氛围，与自信和镜头感的关系大于长相。
3.  **他拍**：指被他人拍摄时的自如程度和镜头感。提升方法是多进行拍摄实践，可以从每次拍摄中学到一个摆拍技巧，积少成多。
4.  **修图**：修图能展示个人审美。至少应能熟练操作手机美图软件，学习PS则更有帮助。修图的轻重痕迹会带来完全不同的效果。

图片审美的提升是一个长期过程。可以多关注审美优秀的博主、电影导演的运镜与配色，或阅读像《Vogue》这类时尚杂志，这对优化小红书等平台的封面视觉协调性非常有用。

## 短视频表达

最后，我们来分析当前风头正劲的短视频。

虽然短视频热潮已持续近两年，但其红利依然存在。可以认为，短视频的上半场（靠多发、跟热点就能轻松起号）已过，下半场（需要差异化、精细化运营）已经开始。

如果你的能力允许，建议将短视频作为“一加X”模式中的主打媒介；如果不想主打，也最好将其纳入辅打范围。

最推荐发力短视频的平台包括：**抖音、快手、视频号、小红书**。值得注意的是，小红书目前对爆款视频的流量扶持远高于爆款图文。

关于短视频，有两个核心观点：
1.  短视频是当前所有表达媒介中最大的风口，能主做就主做，不能主做也要辅做。
2.  短视频应**优先采用真人出镜**，并尽量使用原声。真实感和亲近感比“上镜是否好看”更重要。好看可以通过打光、美颜等技术手段处理，可以专业钻研。
