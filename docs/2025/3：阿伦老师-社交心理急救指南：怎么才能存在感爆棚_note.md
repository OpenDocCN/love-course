# 社交心理急救指南：3：如何获得强烈存在感

在本节课中，我们将学习如何获得强烈的存在感。课程将分为两部分：首先分析为何缺乏存在感，然后提供具体方法来建立并增强存在感。

## 第一部分：为何缺乏存在感

上一节我们介绍了课程主题，本节中我们来看看导致缺乏存在感的核心原因。

### 自我存在感的缺失

自我存在感是指自我认同或感知自我存在的感觉。其核心公式可以描述为：
**自我存在感 = 自我感觉良好 + 自我接纳 + 自我理解**

当一个人拥有自我存在感时，他会觉得自己是存在的、是好的、是棒的。他理解并接纳自己。缺乏自我存在感，就很难发展出外界存在感。因为只有自我存在感稳固，个人才能释放魅力、行动力和对世界的掌控力，从而产生外界存在感。

例如，当你自我感觉良好时，你更乐于向外界表达自己，或是有精力去做感兴趣的事情。外界因此接收到你的信息，感知到你的存在。你通过说话、表达观点、释放个性来展现自己。这是一个由内而外的过程：**自我存在感 → 行动/表达 → 外界感知**。

拥有自我存在感的人精力充沛，内心没有冲突，会主动去做想做的事，甚至希望为世界做贡献、实现自我。一旦行动，外界就会关注。例如，一个叫阿伦的人讲授帮助社恐的课程，外界因此感知到他的存在。这一切都源于其自我存在感的提升。

向外界释放存在感主要有两种方式：
1.  **表达自我**：输出观点、认知和知识。
2.  **创造价值**：通过输出帮助他人。

外界存在感建立在自我存在感的基础之上。它是个人向外界释放能量、认知和世界观后，被外界感知、吸收并认同的结果。

一个人没有存在感，往往是因为其自我存在感薄弱。如果自我认同不足，即使想建立外界存在感，也会显得无力或难以持续。内心不够强大的人，在试图影响世界时，容易因困难而放弃，他们的第一反应常常是质疑事情的难易程度，而非思考如何行动。**自我存在感与外界存在感可以同步发展，但无法跳过前者直接获得后者。**

## 第二部分：如何建立强烈存在感

上一节我们探讨了缺乏存在感的根源，本节中我们来看看如何具体建立存在感。

### 建立自我存在感

短时间内建立自我存在感，需要做好以下三点：

以下是建立自我存在感的三个关键行动：

1.  **不要因为害怕而去表达**（在社交场合中）。这不是鼓励沉默，而是指不应为了缓解尴尬或不适而强迫自己表达。这种表达是逃避而非突破，反而会削弱存在感。真正的存在感有时源于适时的安静与淡定。
2.  **不要因为害怕而不表达**。当需要你发声、展现立场或输出观点时，不应因恐惧而沉默。例如，家人受欺负时应据理力争，职场中有好想法时应勇敢提出。该表达时不表达，也会丧失存在感。
3.  **挑战自己不敢做的事情**。这是指主动练习和突破舒适区。可能包括挑战自己在“不该表达时保持安静”，也可能包括挑战自己在“该表达时勇敢开口”。这两种都是对自我的锻炼。

做到这三点，就能在某种程度上建立自我存在感。因为你做到了“该做则做，不该做则止”，获得了自我掌控感，从而感觉良好，自我认同随之产生。

### 建立外界存在感

在建立了初步的自我存在感后，我们需要向外拓展，建立外界存在感。核心方法是：**必须主动表达自己**，向外界输出你的价值观、想法和对事物的态度。

无论通过文字、语音、视频还是当面交流，都要传递你的观点和价值。例如，讲师阿伦通过多种形式展现其价值观。建立存在感的最终目的，是服务于内心的自我认同和自信，让自己变得更好。

在这个大前提下，可以通过以下四点来增强外界存在感：

以下是增强外界存在感的四个方法：

1.  **提升自己的核心价值**。你必须有价值，说出的话才有人听。无论是学生学好习，还是职场人士做好工作、提高收入，都是在提升核心价值。价值是存在感的基础。
2.  **培养属于自己的兴趣爱好**。拥有独特的、真正热爱的爱好，能使你与众不同，让你在独处时也能感到快乐和充实。这种独立性本身就能增强你的存在感。
3.  **发自内心地帮助别人（提供价值）**。不求回报地、真诚地帮助他人。当别人感受到你纯粹的价值输出时，会认为你存在感强、有能量，更愿意尊重和与你相处。一旦帮助附带索取，存在感就会减弱。
4.  **做自己的事业**。广义上，把本职工作做到极致也是事业。更高层次是创业或从事能帮助更多人的事业，创造更大的社会价值。这种影响力能带来爆棚的存在感。即使像提供优质知识产品这样的事业，也能深刻影响他人，获得强烈的存在感。

## 总结

本节课中，我们一起学习了存在感的构建。课程分为两部分：

1.  **自我存在感**：觉得自己还不错的感觉。它是外界存在感的基础。
2.  **外界存在感**：以自我存在感为基础，通过影响外界，让他人觉得你不错。

建立方法如下：
*   **建立自我存在感**：
    1.  不要因为害怕而去表达。
    2.  不要因为害怕而不表达。
    3.  挑战自己不敢做的事情。
*   **建立外界存在感**：
    1.  提升自己的核心价值。
    2.  培养属于自己的兴趣爱好。
    3.  发自内心地帮助别人。
    4.  做自己的事业。

通过内外兼修，我们可以逐步建立起强烈的存在感，最终增强自信和自我认同。
