# 两性关系宝典：1：课程概述与背景

在本节课中，我们将要学习《两性关系宝典》课程的开设背景与核心目标。了解当前亲密关系面临的普遍挑战，并认识本课程将如何从医学与科学的角度提供解决方案。

## 课程背景

2015年，全国依法办理离婚手续的夫妻共有384.1万对，比上年增长5.6%。2016年，这一数字增长至415.8万对，比上年增长8.3%。离婚数据呈现持续上升趋势。

这一数据尚未包含正在办理离婚的人群。有人统计了离婚的六大主要原因。

以下是六大主要原因列表：
*   第六，购房需求。
*   第五，不良嗜好。
*   第四，婆媳关系不和睦。
*   第三，性格不合。
*   第二，家庭暴力。
*   第一，一方出轨。

出轨因素是多方面的，根本原因常源于某一方的需求未得到满足。这些需求包括生理需求与心理需求。受性文化教育欠缺的影响，人们常羞于启齿讨论两性需求。

即使面对最亲近的伴侣，许多想法与不满也常藏在心里，不愿沟通。久而久之，问题积累到一定程度，要么爆发，要么促使人们另寻他法。这些都会给婚姻生活或亲密关系带来严重威胁与破坏。

## 课程目标

王斌博士的《两性关系宝典》从医学角度出发，结合科学方法与十几年临床经验，指导人们如何在两性关系中实现自我满足。

课程同时指导人们如何关注对方需求，并避免常见误区。宝典第一章“性爱三部曲”涵盖性爱前准备、性爱中享受与注意事项、以及性爱后正确收尾。内容简单实用，易于操作。
