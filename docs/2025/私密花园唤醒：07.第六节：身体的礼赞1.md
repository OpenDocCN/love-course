# 私密花园唤醒课堂-加密：07：第六节：身体的礼赞1——发现

在本节课中，我们将学习如何通过一个具体的练习，来探索和审视自己对于身体各个部位的态度，从而开启接纳与关爱自我的旅程。

上一节我们了解了认识身体的重要性，本节中我们来看看如何具体地检索自己对于身体的态度。

## 态度检索练习

以下是进行自我身体态度审视的步骤。

1.  准备纸和笔，在纸上划分两个区域，一边标注“喜欢”，另一边标注“不喜欢”。
2.  将你身体的每一个部位，例如头发、眉毛、鼻子、嘴巴、眼睛、腿、脚等，逐一列出。
3.  分辨并记录下你喜欢和不喜欢的具体部位。例如，可能不喜欢自己的手、脚踝、脚、脚趾或乳房。
4.  需要着重强调的是，请特别留意你对于“优灵”（即阴道和外阴部）的态度，记录你是否喜欢它。
5.  对于喜欢和不喜欢的部位，分别写下你的理由。
6.  将写下的内容读出来，说出你为什么喜欢或不喜欢它们。最好能对着镜子进行。
7.  自己聆听并观察这些话语。
8.  可以尝试与你的爱人一起进行这个练习。

通过这个练习，你将看到自己对于身体的接纳程度，以及热爱和喜爱的程度有多少。这个过程可能会带来一些新的发现，并促使你开始学习接纳自己身体的全部。

## 接纳与自我批判

爱自己的核心是**接纳自己的所有**，停止对自己过多的挑剔和评判。自我批判有时像一把双刃剑，过度的自我怀疑和批判实际上是在打压和削弱自己的力量。当你内在对某个部分不自信，认为它不好看时，就会总想把它藏起来或刻意掩盖，这非常消耗能量，甚至会影响你走路时的自然姿态。

一个良好的形体与形态，并非仅指物理层面的“前凸后翘”。更重要的是，当你的内在没有完全接纳自己、缺乏充分的自信和自爱时，你如何能自然自如地呈现良好的身体状态？真正的形态，是你内在的光透过身体所呈现出来的真实状态。未来我们将有一些练习，专门转化你对于“优灵”等部位的不接纳与不欣赏。

关于追求“变得更完美”，无论是整形、整容还是丰胸，其背后的动机至关重要。如果动机是出于嫌弃和讨厌，那么底层的能量是负面的。如果是基于**接纳现状，并希望在此基础上更好地呈现自我**，那么能量则是完全不同的。这是两种截然不同的能量呈现。

## 身体感知深化练习

在完成上述喜欢与不喜欢的记录和陈述后，可以进行以下练习，以感知不同态度下身体的真实感受。

首先，挑选一个你喜欢的身体部位，例如胸部。

1.  将你的注意力完全放在这个部位。
2.  将呼吸带到那里，把意识和专注力全部集中于此。
3.  感受吸气时能否到达这个地方，并仔细体会此处的感觉。需要保持安静和专注。
4.  去感受那个地方是紧绷的还是扩张的，是热的还是冷的，待在那里是舒服放松的还是不舒服不自在的。请花些时间静静感受。

完成上述步骤后，接着挑选一个你不喜欢的身体部位，例如阴部。

1.  现在将你的注意力放在那个部位，把呼吸也带到那里。这类似于中国武术中的“意到气到”，将意念集中于某处，气息便会自然到达。
2.  闭上眼睛，用你内在的“心眼”去观想你的“优灵”，仿佛内在有一双眼睛看着它。
3.  意念在此处感知它，呼吸也汇聚于此。
4.  慢慢地吸气、吐气，去感受那个地方。你可能会感觉到紧张或颤抖。
5.  去感受那个空间。你甚至可能感知不到它，或者感知到它是黑的、白的、有光的、无光的、扩张的，还是收缩紧绷的。
6.  去感受那个地方是欢喜的，还是沉重的、哀伤的。

最后，对比喜欢的部位和不喜欢的部位，体会两者在能量场和身体感知上有何不同。通过这个练习，你将能够辨别出，无论你是否说出来，你的身体都能感知到喜欢与不喜欢所带来的不同影响。这个练习旨在帮助你在未来的练习中，更有动力放下对身体的批判、评判和不接纳，更愿意带着爱与欣赏去好好爱自己，迎接回自己。

本节课中，我们一起学习了通过书写和陈述来审视对身体的态度，并透过专注呼吸与意念的感知练习，体会了不同态度下身体的真实感受。核心在于理解**爱始于全然的接纳**，减少自我批判，才能让内在的光芒透过身体自然呈现。
