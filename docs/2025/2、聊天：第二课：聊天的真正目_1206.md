# 聊天艺术：第二课：聊天的真正目的

在本节课中，我们将探讨聊天的核心目的，并分析在开始聊天前需要明确的关键心态与常见误区。理解这些基础，是进行有效沟通的第一步。

## 认识方式与第一印象

在开始聊天之前，需要明确一件事。无论你使用微信、Line、Instagram还是短信进行聊天，认识女生的方式决定了第一印象的来源。

以下是不同认识方式下，女生形成印象的主要依据：

*   **社交软件认识**：女生通过你的展示面（照片）来判断和了解你。她不知道你是谁。
*   **搭讪认识**：女生更看重你当时的状态。
*   **真实生活社交局认识**：女生比较看重你的社交认证，即朋友对你的评价。

这就像我们看待女生一样，对于不好看的女生，我们通常也不会想和她聊天。因此，你必须先获得“入场券”，才能有后续的聊天。不要听信一些导师的说法，认为仅凭强大的内在就能取胜。试想，你走在大街上看到一个丑女，即使被告知她内核强大，你愿意去了解并亲近她吗？

## 聊天的核心目的

无论通过何种方式获得联系方式，在开始聊天前，你必须明确聊天的目的。我遇到许多咨询我的学员和导师，存在一种普遍情况。

以下是聊天缺乏目的的典型表现：

*   我看到他们的聊天记录，和女生聊了两个月却从不邀约。
*   每天聊天但没有目的性，女生也不知道他们想做什么。

这种行为很容易导致关系停滞。请务必明确：**聊天的目的是为了邀约**。语言是以结果为导向的工具，聊天一定是为了邀约见面。除非你暂时不想见面。就像我，很多女生每天都想来见我，我见不过来时，会用聊天将关系控制在我“想见时可以随时见，她们也不会闹情绪”的阶段。但这项技能比较深奥，我们只对有需要的私教学员专门教授。

## 聊天前的心态准备

在约会中，你需要保持自我并调整好心态，不要表现出过强的需求感。聊天也是一样的道理。在聊天开始前，很多人的心态就已经开始变质。

你们需要知道，和女生聊天时，你的线上与线下形象应该是同一个人。这意味着，如果你在线上疯狂想升级关系、需求感很强，那么到了线下你也必然是同样的状态。

普通人通常在聊天前会陷入以下三种错误心态：

1.  **不敢聊**：害怕说错话，或者害怕收到女生的负面反馈。
2.  **聊天很冲**：滥用土味情话，胡乱开启性话题。我必须负责任地告诉你，这些方法没有用。通过升级性话题“泡到”的女生，本身可能就是只想约炮的女人。如果你想和我一样吸引到高价值的女生，这样做根本不可能。如果你学过那些“很冲”的聊天方法，请你先忘掉。
3.  **有很强的不安全感**：女生一旦不回消息，就疯狂思考自己哪里说错了，哪里有问题。会疯狂给女生发消息，或者陷入严重的内耗，形成“不敢聊”的恶性循环。

这三种情况都很致命，很消耗心力。如果你之前总是约不出女生，请先想想自己是否犯了这些错误。

本节课中，我们一起学习了聊天的真正目的——邀约，并分析了不同认识方式对第一印象的影响。同时，我们重点剖析了在聊天开始前必须避免的三种错误心态：不敢聊、聊天很冲以及过度不安全感。明确目的、摆正心态，是开启有效聊天的基石。
