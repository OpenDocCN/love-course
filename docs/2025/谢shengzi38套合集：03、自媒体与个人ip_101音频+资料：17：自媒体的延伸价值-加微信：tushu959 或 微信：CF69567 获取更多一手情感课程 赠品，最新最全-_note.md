# 自媒体与个人IP：03：自媒体的延伸价值

在本节课中，我们将探讨自媒体活动超越商业回报的深层价值，学习如何以“正面”心态进行创作，从而实现个人成长与自我提升。

## 课程概述

在前面的章节中，我们深入探讨了自媒体可能带来的商业价值、社交价值以及流量影响力。本节作为本篇章的最后一节，我们将跳出事情本身，向更远的地方发散思考。

## 自媒体即修行

如果你真真正正是用心的在做自媒体，那么做自媒体本身就是一场修行。你在不断表达、不断与互联网受众互动的过程中，会更了解自我，会更加敏感于内心的感受和身边的生活素材。你面对不同的评价，会变得越来越淡定。如果你真的找到了一种很好的感觉和角度，那么你是可以通过自媒体获得快速的自我成长，从而变成一个更好的人的。

## 何为值得做的工作与事情

前几天我在和我的朋友聊，也在和我的团队聊，到底什么是一份好的工作。我觉得有一个定义，就是通过这份工作，你本身可以变成一个更好的人。这种好你可以理解为更美好，也可以理解为更强大。那什么是一件值得做的事呢？就是无论这件事情最终的世俗结果怎么样，这段经历本身就能够让你的能力，甚至是品格得到很大的提升。

## 正面做自媒体的心态

所以在这一节，我很想跟大家分享一些正面做自媒体的心态。什么是正面呢？例如正面饮食，就是我不会去想吃这个东西是会胖还是会瘦。我在吃甜点的时候，即使是一些比较垃圾的甜点垃圾食品，我也在用心感受它的甜和这一刻带给我的开心。我在吃一些比较素的东西的时候，就用心感受它的清爽和食物本身对话。像正面健身，就是把精力放在自己的动作和体能，精神很专注。

如果你在做自媒体的过程中，一方面你是有业务视角的、有商业视角的，同时你还有一个很正面的、很自我提升的视角，就会非常好。

## 业务视角与自我提升视角的结合

例如我写了一条微博，我不断的去看点赞、评论、转发，我不断的去感受用户喜欢什么样的内容，这个东西是要看的，这是你做自媒体的业务的视角。但是其实你还有一个角度，就是我为了日更微博（我之后会讲到微博篇，我说微博的秘诀在于持续的日更，让你的用户保持这种订阅你的内容的习惯），你就是保持了持续的记录，它本身就是一种自律的自我表达。

那么你为了写更多的微博呢，你必须更敏感于你身边所发生的一切。你可以不断的品味你自己写的微博，上一个月写的微博，去年写的微博，然后体会你的进步和你在不同的阶段所表达的角度的不一样。

## 超越世俗结果的价值

那么最后的最后，即使写微博这件事情并没有让你赚到一毛钱，你也已经得到了很多与你的人生而言。我觉得做很多事情呢，既要有一个世俗的角度，也要有一个不世俗的角度，这样会让你整体的心态更好。同时，不世俗的角度有可能是让你的世俗结果更容易达成的。正是你的情绪没有那么的起伏，没有那么的波澜，你的目的性显得没有那么的强，有可能你本身这个IP和你的内容本身是更加有感染力的，从而更加有了一些商业的价值。

## 课程总结与寄语

本节课中，我们一起学习了自媒体作为个人修行与成长工具的价值。这套课，除了“自媒体是最好的个人能力放大器”之外，其实还有一句slogan叫做“始于自媒体，高于自媒体”。这一个篇章我们讲商业模式，而在这最后一节，我们讲了一点点跟商业无关，但是跟心灵、跟你的能量、跟你的自我成长相关的东西。希望大家有非常丰富的心灵维度去面对自媒体和面对个人IP。
