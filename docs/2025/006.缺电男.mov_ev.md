# 006.缺电男.mov_ev

第一种积肋难啊，我管它叫缺电难。啥叫缺电难呢？就是它就像一块永远充不满电的电池一样，它给你的爱永远都是60分的。咱们也可以把他叫成60分男，他给你的情绪价值呀，也是60分，给你的物质上的付出呀。

也是60分。包括后期你看他的家庭参与度，他对你的关心永远也是60分。接触时间长了，你会发现他就是一个处不热，捂不熟的人，你跟他生活在一起之后会发现哎呀，吵也吵不起来。爱也爱不起来，喜欢也喜欢不起来。

讨厌也讨厌不起来。哎呀，生活当中呀能折磨死你关键这种类型的男生来相亲赏呀，还挺受家长喜欢的，为啥呢？你看一个月挣个万把块的，也有个基本的房车，工作不能说有多好吧，但是也还行，然后家长你看呀。

这这男孩子是吧？话少然后这人也老实以后也不会出轨，看着也挺踏实的，但凡你跟长辈们说，你不想跟他在处房下处了，长辈们都要说你的问题呢，说你又作又闹爱找事，是不是气住人家这个男生了，但只有跟他朝夕。

相处过的你知道这种男生呀，真的是骨子里呀，又自私又冷漠，永远只活在他自己的那个小世界里。姐妹们就这类型的男生，你只有跟他结了婚之后。😊，才能发现就是我之前讲的，他为什么要来结婚呀。

就是他给你物质上的付出，家庭上的付出，情绪上的付出，甚至劳动上的付出啊，就帮你跑个腿啥的，永远都是60分。我说的这四样，你看着吧，多要一点，他扭头就走，他就会逃避到自己也不是扭头就走啊。

就是逃避到自己的那个小世界当中去，总而言之，就总怎么总结这种人呢，就是捂不熟处不热，供不暖，骨子里自私又冷漠，永远就在自己的那个小世界里很自我跟他相处呢，大毛病没有，小毛病不断，你也懒得跟他吵架。

为啥呢？你跟这种人吵架吧，你也吵不起来，第一，他也不跟你吵，第二他也不跟你离嘴。第三，他也不跟你提建议，但是他也不跟你好脸色。那这种男生在婚姻当中唯一的优点就是大概率不会出轨吧，情绪还算稳定。

但真的是能折磨死你。我跟你讲，就是咱之前讲的一后烧不开的热水嘛，你会发现他在工作当中也是就是每天人家也准点上班，准点下班，也不迟早早退，但是呀每天去。😊，也就是那浑浑噩噩的去过日子，也没啥上进心。

也不会去向上搞关系。街上一条啊，千万不敢跟这种类型的男生稀里糊涂结婚啊。刚开始结婚，你就会很快发现他身上一个大毛病，什么大毛病大问题呢？能要了你的命，就是哎呀这日子不闲也不淡，不冷也不热，不温也不火。

处不熟，喂不饱供不暖捂不热，时间长了，你每天都得被他的那种带引号的情绪价值呀，给逼疯。我跟你讲。😊。
