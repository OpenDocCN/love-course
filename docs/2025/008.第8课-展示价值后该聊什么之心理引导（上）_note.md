引导性测试的做法分为五个步骤。

2.  **第二步**：我们不再正面回答，而是通过“轻轻推开”对方的方式回应。

4.  **第四步**：如果女生通过了测试，我们要给出奖励。
5.  **第五步**：满足她最初的要求。

这里先科普一下“服从性测试”：即提出一个要求，观察对方是否愿意满足。如果愿意，通常代表吸引到位。但直接提出服从性测试会显得突兀，让对方不舒服，不易通过。而在提出测试前，先利用女生的黄金IOI并加上一个“推”的动作，则更容易成功。
