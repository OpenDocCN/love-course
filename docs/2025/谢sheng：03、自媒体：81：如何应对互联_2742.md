# 自媒体与个人IP：03：如何应对互联网的各种评价

## 概述

在本节课中，我们将要学习如何应对互联网上的各种评价。对于每一位内容创作者而言，处理评论是必经之路。我们将探讨如何调整心态，并掌握具体的应对策略，从而不被评论左右，专注于个人成长与内容创作。

## 心态调整：回归评价的正常值

上一节我们介绍了互联网评论的普遍性，本节中我们来看看如何调整心态。互联网上的评价通常比线下更直接、更夸张。无论是正面赞美还是负面批评，其表达强度往往被放大。

因此，你需要建立一个“过滤系统”，将网上的评价回归到一个正常值。这意味着，无论是正向还是负向的评价，你都需要在心理上减轻其影响程度。

另一个重要的心态是保持“功利心”。你的核心目标是获取数据和热度。每一个评论，无论好坏，都是在为你的内容增加互动和曝光。你应该为此感到高兴，并将注意力集中在数据目标上，而非纠结于具体的言辞。这种功利心是对敏感的内容创作者的一种有效保护。

## 具体应对策略

调整好心态后，我们来看看具体的应对手法。以下是几种常见评论类型的处理方式。

### 应对低级杠精

遇到水平较低的杠精评论时，处理原则非常简单。

*   **不要回复**：无论你试图理性解释、用更高级的语言讽刺，还是直接回怼，只要你回复，某种程度上你就输了。博主的姿态需要高于杠精。
*   **可以不删除**：保留一些不同的声音，能展示评论区的人类多样性，增加可阅读性，有时还能引发支持者与反对者的讨论，从而提升数据。
*   **如无法忍受，则删除**：如果某些评论实在让你难以承受，直接删除即可，无需与之纠缠。

### 回应客观评论

对于比较客观的评论，你可以选择性地回复。但在回复时，心里必须有一根明确的准绳。

*   **你不是回给一个人看的**：你的每一次公开回复，都是展示给所有观众看的。因此，必须特别注意保持风度、注意措辞和选择角度。

### 设置评论边界

如果你发现自己情绪波动较大，尤其当发布了可能引发争议的内容后，暂时无法妥善处理评论时，最直接的方法是设置边界。

*   **关闭评论和私信**：这是一个有效的保护措施。有人可能会担心无法获取市场反馈，但一个专业的内容创作者，凭借对内容的敏感度，在发布时通常就能预判大致的反响和数据表现。培养对内容的“感觉”比依赖即时评论更重要。

## 总结

本节课中我们一起学习了应对互联网评价的方法。首先，我们需要调整心态，将网络评价“降噪”并聚焦于数据目标。其次，我们掌握了具体策略：对低级杠精不予回复，对客观评论保持风度地公开回应，在必要时果断关闭评论渠道。记住，成熟的博主体现在创作系统和心态的双重稳定上。下一期，我们将探讨一个更深入的话题：如何面对你的黑粉。
