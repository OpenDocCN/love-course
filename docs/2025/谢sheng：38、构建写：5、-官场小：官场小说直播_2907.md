在本节课中，我们将学习如何通过阅读官场小说来构建对人情世故、权力结构和事业规划的认知体系。课程内容基于一次闭门直播分享，将分为两大主题：一是近期参与的政府闭门培训的见闻与思考，二是官场小说的阅读方法与核心习得。

## 第一部分：政府闭门培训分享

上一节我们介绍了课程的整体结构，本节中我们来看看第一部分，即关于近期政府闭门培训的见闻与核心认知。

### 培训背景与目的

我近期参加了一个由上海市青联（青年联合会）推荐的培训。青联常被称为“小政协”。此外，培训还涉及“新联会”，即新的社会阶层人士联合会。从事网络、新媒体、流量相关工作的人士，加入所在地的新联会是有益的。

我之所以有意识地参加此类政府培训，是为了获得全方位的视角，避免局限于同行间的自嗨。信息来源和社交圈应包含以下几个层面：
*   同行视角（IP、博主）
*   用户视角
*   资本与商业视角
*   官方与政府视角

### 核心趋势认知

以下是培训中多位教授和领导分享的，对于宏观趋势的一些判断：

1.  **中美长期竞争是规律性事件**
    *   **核心公式**：`老大不允许老二崛起`。
    *   未来几十年，中美之间的对峙将无法缓和，这是不以短期外交政策调整为转移的必然事件。

2.  **中美争第一是时代主旋律**
    *   台湾问题等并非主要矛盾，核心是中美之间争夺第一的问题。

3.  **关于“中华民族伟大复兴”的表述**
    *   中国目前阶段不宜公开将“成为第一”挂在口头上。外部（如美国）可能会曲解此类表述，因此需要非常注意外交辞令。

4.  **线上与线下将实现真正融合**
    *   未来将不再有纯粹的实体企业与数字企业之分。“网络大V”的概念会过时，真正有影响力的人需要在线上线下都具备实力。对任何行业而言，掌握线上流量玩法将成为**必须项**。

5.  **90后的历史使命：托举下一代完全找到自我**
    *   我们这一代处于“半自我、半螺丝钉”的状态。我们的使命是部分实现自我，同时为下一代兜底，鼓励他们完全做自己。因为**人只有在完全做自己且有家庭兜底时，才拥有最强的创新能力**，这有助于提升整体国力。

6.  **2024-2035年的经济周期**
    *   2023年至2035年是关键周期，目标是到2035年成为中等发达国家。但培训指出，**接下来至少3年（约2024-2027年）经济可能仍将维持与当前相似的节奏，不会出现爆发式上行**。因此，个人在投资和创业上宜采取谨慎策略。

### 关于自媒体行业的官方视角

以下是网信办领导分享的，对自媒体内容方向的指导：

1.  **炫富类内容将受到极大打压**
    *   无论是直接炫富还是“轻炫富”（如“如何一年赚100万”这类标题），都已不合时宜。证明自身价值应从细节体现，而非直接宣扬。

2.  **对LGBT等议题的态度**
    *   官方目前的核心关切之一是人口问题（鼓励生育）。对于LGBT内容，官方的态度是：个人可选择生活方式，但不宜在网络上公开宣扬并形成影响力。

3.  **反对“躺平”情绪，鼓励奋斗**
    *   “躺平”这类消极情绪蔓延是官方不支持的。积极奋斗的公众形象更受青睐。

4.  **平台是公共空间，非完全自留地**
    *   抖音、小红书等平台并非完全的商业“自留地”，而是公共平台。官方媒体内容平台无权随意删除，且政府部门有责任保护正能量网红。

5.  **任何流量都需付出代价**
    *   追逐流量意味着在措辞和吸引的用户群体上都需要承担相应风险。例如，部分财经媒体已开始“去流量化”，不再将阅读量作为核心考核指标。

6.  **科技自媒体是受鼓励的蓝海**
    *   官方比较鼓励**科技自媒体**的发展。与其他类别（如生活、知识类）相比，科技自媒体门槛更高，但前景更受认可。

### 统战工作带来的启发

培训也涉及统一战线工作，其中一些思维对个人事业和团队管理很有启发：

1.  **“把敌人搞得少少的，把朋友搞得多多的”**
    *   这是统战的核心理念。应用于个人，即不要以最大恶意揣测他人（如黑粉），认为其完全不可争取。应先尝试沟通与争取。

2.  **反复认知带来认同**
    *   **核心概念**：`Recognize（认知） -> Re-cognize（反复认知） -> 认同`。
    *   应用于团队管理，需要经常同步思想，反复讲述公司愿景、价值观，才能让团队拧成一股绳。

3.  **利他方能利己**
    *   当你无视他人利益时，自己也难以获得利益。野心最大的人，往往也是最能做到利他的人。

4.  **新媒体从业者的社会阶层定位**
    *   我们属于“新的社会阶层人士”。可以主动了解并尝试加入所在地的“新联会”，这有助于更敏感地把握政策风向，并从官方视角理解自身行业。

### 个人提升方向

经过此次培训，我个人认为需要在以下维度加强：

1.  **提升对政府文本的解读能力**
    *   要预判市场周期和规划长远事业，需要能够读懂政府的指导性文件。这需要积累底层知识结构，方法可以是“硬读”——即使不懂也坚持朗读，以培养语感和意识。

2.  **构建系统化思维体系**
    *   网络信息是离散的，真正提升思维需要深度阅读和思考。学习的目的是更全面地把控时代，从而调整自身行为。

3.  **保持居安思危，打破行业自嗨**
    *   需意识到，个人或行业的发展在更大程度上得益于时代和国家的允许。应全方位获取信息，避免局限于行业内部的狭隘视角。

4.  **更加关注科技板块**
    *   无论身处哪个行业，都应思考如何与科技进行一定程度的融合。关注人工智能等前沿领域是趋势。

## 第二部分：官场小说的阅读方法与习得

上一部分我们探讨了宏观趋势与官方视角，本节中我们来看看如何通过阅读官场小说，来具体提升对中国式人情世故和权力结构的理解。

### 官场小说的核心价值

我在25-29岁期间阅读了大量官场小说。它们提供了在公开课程中难以获得的、赤裸而现实的洞察。核心习得包括：

*   对中国式人情世故的细节提取。
*   对级别、职权、实权/虚权部门的高度敏感。
*   对慈善、家族信托、美女经济等现象的现实看法。
*   对正常关系、权力和利益的深刻理解。
*   生态思维以及个人事业的动线规划。

### 具体习得与案例

以下是阅读官场小说后，在一些具体情境中的认知提升：

**1. 人情世故的细节：称呼与饭局**
*   **称呼领导**：需称呼其最高级别职务，且通常省略“副”字（如“王副局长”应称“王局”）。不可称呼“某秘书”。若无法判断，应私下请教可靠的前辈。
*   **饭局礼仪**：与高级别领导吃饭，若领导未到，其他人通常**站在椅子后等候**，而非先坐下。这是与普通商务饭局的显著区别。
*   **饭局发言**：
    *   当被夸“以后是你们年轻人的时代了”，不应只说“没有没有”，可回答：“会好好努力的，但还是需要前辈的指导和开路。”
    *   当被介绍时，可以说：“感谢X总/学长带我出来见世面。” **谦逊（“卑”）的表现是否娴熟，恰恰体现了你是否见过世面**。
    *   被称“才女”时，可回答：“不敢当，我还是学生，向大家学习。”

**2. 对权力结构的敏感**
*   **工商联**：小说中揭示，工商联是“民营企业家的娘家”，主要提供情绪支撑，但真正解决问题仍需依靠实权部门。
*   **级别与背景**：通过一个人在何地、何部门、何年龄达到何种级别，可以迅速判断其是否有家庭背景支撑。
*   **权、钱、名的关系**：小说赤裸地指出，在真正的权力和财富面前，“名”往往是被动选择或工具。**权 > 钱 > 名**是普遍的认知排序。

**3. 对某些职业和现象的冷峻看法**
*   **网红/主持人**：小说中可能将没有实业根基的网红描述为“没有社会地位和资源的自由职业者”。电视台美女主持人若没有事业底盘，在与成功人士对话中可能缺乏真正的话语权，更多是作为“装点”。
*   **影响力本质**：这种“影响力”可能非常脆弱，它建立在被允许的舞台之上。这促使思考什么是更坚实、更有能量的事业追求。

### 推荐书单与阅读建议

如果你想开始阅读，建议按作者入手，他们往往有相关经历，写得更真实：

*   **首推作者**：王跃文、周梅森。
*   **入门推荐**：
    1.  《沧浪之水》
    2.  《二号首长》
*   **进阶阅读**：
    3.  《国画》
    4.  《青瓷》
    5.  《人民的名义》（原著）
*   **延伸阅读**：钱穆《中国历代政治得失》（有助于理解制度背景）。

阅读时，可刻意提取关于人情世故、权力运作的具体细节，并思考其背后的逻辑。看完几本后，会对社会运行规则有质的理解提升。

## 总结与行动建议

本节课中，我们一起学习了两个部分的内容：一是从政府闭门培训中获得的宏观趋势认知与官方视角；二是如何通过阅读官场小说来微观地掌握人情世故与权力逻辑。

**核心总结如下：**

1.  **把握周期**：认清中美长期竞争与国内经济周期，未来几年宜“苟住”，管理好身体与现金流，避免大规模冒险投资。
2.  **切换视角**：摆脱行业自嗨，建立包含同行、用户、资本和官方的全方位视角，特别是理解官方对行业（如打压炫富、鼓励科技自媒体）的定位与红线。
3.  **学习统战思维**：在事业和人际中，争取大多数，反复同步价值观以实现认同，深刻理解利他即利己。
4.  **从小说中学习现实**：官场小说是洞察中国社会权力关系、人情世故的宝贵教材。通过刻意阅读，可以快速提升在这些方面的“悟性”和“敏锐度”。
5.  **明确追求**：思考“权、钱、名”的本质与排序，规划更坚实、符合长期趋势的个人事业动线。

**行动建议：**
*   **对于趋势**：保持关注，谨慎决策，在下一个上行周期（约2028年后）到来前夯实基础。
*   **对于认知**：开始阅读推荐的官场小说，并尝试在阅读后立即写下10-20条最触动你的要点或心得，这将极大强化学习效果。
*   **对于自身**：管理好健康（这将成为未来的重要优势），理清身边核心关系，在“苟住”中寻求线性的、稳健的进步。

通过将宏观视野与微观洞察相结合，我们能更好地理解所处时代，并据此调整自身心态与行动轨迹，从而更从容地面对未来。
