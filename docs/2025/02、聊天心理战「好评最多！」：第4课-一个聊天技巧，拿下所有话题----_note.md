# 聊天心理战：02：一个聊天技巧，拿下所有话题 🗣️💡

在本节课中，我们将学习一个核心的聊天技巧：如何围绕对方提供的信息，挖掘并主导共同话题，从而建立融洽的沟通氛围。

上一节我们介绍了聊天的基础，本节中我们来看看如何具体展开话题。

## 核心技巧：围绕信息展开话题

聊天的内容，应从你获取到的对方信息开始。围绕这些信息，挖掘对方感兴趣的话题和事情。

下次开启话题时，你就可以围绕对方感兴趣的点进行交流，并朝着对方想聊的思维方向去聊。

## 关键原则：引导而非迎合

朝着对方想聊的思维去聊。这里的原则是引导，而非迎合，更不是表现出你在刻意迎合。正确做法是适当穿插话题。

因为这项技巧的最终目的，仅仅是让对方知道你们有共同语言。公式可以表示为：**聊天方向 ≈ 对方兴趣点 + 自然引导**。

让女生感觉和你能聊得来。

## 技巧的深层作用

有了共同的话题，沟通便能深入。甚至随着话题的深入，会让对方觉得她和你在一起相处是有未来的，是可以很融洽地相处、聊天，甚至一起完成一些事情。

朝着对方的思维去聊，本质上是一项建立连接的心理技巧。

以下是运用此技巧时的注意事项：
*   核心是建立平等沟通，而非放低姿态。
*   重点在于发现共同点，而非一味认同。
*   目标是通过话题创造舒适感与未来感。

## 重要提醒

最后再次强调，朝着对方想聊的思维去聊，不要误解为讨好或跪舔。

本节课中我们一起学习了如何通过挖掘对方信息来引导共同话题，这是一项建立有效沟通和舒适关系的重要心理技巧。
