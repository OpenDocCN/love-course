# 一门给年轻人的恋爱成长课：3：亲近感是如何产生的？相互吸引的先决条件

在本节课中，我们将要学习亲密感的核心构成，并探讨如何通过物理距离和性吸引力来建立和加深与异性之间的亲近感。

## 亲密感的定义

上一节我们介绍了恋爱的基本概念，本节中我们来看看亲密感究竟是什么。心理学家埃里克森提出过一个定义：感觉到他人对自己的关注和爱护，也感觉到自己是被人需要的、有价值的、生活是幸福的。这也是一种对抗孤独的力量。

而狭义上的亲密感，特别是男女之间的亲密感，归根结底，就是**物理距离**和**性吸引力**。

## 物理距离与心理距离

当我们搞清楚亲密感的两个核心元素后，我们就来看看这两个元素是如何创造亲密感的。先说物理距离。

人类学家霍尔博士总结出了人和人之间由于关系不同，所适应的物理距离。普通朋友之间的物理距离，大体上是45厘米到120厘米之间。45厘米之内，就叫做亲密距离。陌生人适应的物理距离是360厘米到740厘米之间。礼貌性的社交距离，例如初次见面的朋友、同事、客户，最好保持在120厘米到360厘米之间。

我们能够接受一个人和我们的物理距离越近，就意味着我们和他的亲密感越强。反过来，心理上有个概念叫“逆向合理化”。心理距离决定了你能接受的物理距离，而物理距离又反作用于你预设的心理距离。说人话就是：我们可以通过缩短物理距离来制造人与人的亲密感。

以下是两种缩短物理距离的策略：

**1. 多次接近（安全策略）**
面对不熟的人，多次接近是个不错且安全的策略。这基于心理学上的“多看效应”：在排除其他变量后，你见到一个人的次数越多，越容易对他产生一种熟悉的安全感。有条件的话，就多出现在他的视野里，但不要刻意盯着别人。

**2. 真诚表达（直接策略）**
如果无法多次接近，可以真诚地向对方表达，你希望处在一个和他更近的位置，并希望得到允许。真诚能够降低侵略性，缩短心理距离，从而为缩短物理距离创造条件。

## 性吸引力：对亲密接触的渴望

如果我们和一个人已经认识，那么接下来就要重点讲性吸引力了。说完整一点叫做“性别之间的吸引力”，它会产生一种对于和一个人亲密接触上的渴望。

这种渴望包括了牵手、拥抱、想听到对方的声音、近距离注视等等，当然也包括了性关系，但绝不仅仅是性关系。在亲密关系里，这种渴望非常重要。缺乏它，两个人非常容易进入“友谊区”，或者恋爱久了激情消退，由爱情变成亲情。

那么，如何激发对方对于亲密接触的渴望呢？方法可以分为被动和主动两种。

### 被动激发：提升自身条件

最直观的是你长得好看，体态神情有魅力，或者穿着打扮得体。一个值得单独拎出来说的点是气味。科学研究发现，合适的气味本身就是一种性吸引力。个人小建议是，香水值得尝试，并且可以找异性朋友帮忙参考。

### 主动激发：语言与行为试探

主动激发是指我们通过语言和肢体的试探，主动向特定的人去表达你的魅力，以及你对和他亲密的渴望。具体可分为语言层面和行为层面。

**语言层面有以下三种方法：**

1.  **展示自身性别魅力**：通过在语言上展示我们的性别魅力，激发对方对于我们的想象。
    *   **示例**：“我夏天穿吊带比较多。” 这会引导对方想象你穿吊带的样子。

2.  **称赞对方的性别魅力**：夸奖并表达你对对方性别魅力的兴趣。
    *   **公式**：`夸奖 + 表达兴趣`
    *   **示例**：“你的手指很好看，我特别喜欢这样有美感的手指。” 或 “我看你经常去健身房，你是不是有腹肌呀？”

3.  **释放可得性**：传递出一种信息，即在未来的某些时候，对方是可以“得到”你的。
    *   **示例**：当对方说“那我还挺想看看的（你穿吊带）”，你可以回答“有机会的”。或者暗示：“我买了一件很性感的泳衣哦。”

**行为层面主要指肢体接触，我们将其分为三级：**

以下是肢体接触的三个等级，需要循序渐进：

1.  **第一级：礼节性肢体接触**
    例如握手、拍拍肩膀、击掌，或帮对方拍掉衣服上的脏东西。其特点是不刻意、不冒犯，无需区分性别。

2.  **第二级：理由性肢体接触**
    你可以做一些更亲密的肢体接触，但需要有一个合理的理由。
    *   **示例**：“我来帮你看个手相吧，我昨天晚上刚学会的。” 或者，女生可以涂护手霜后说“挤多了，要不也给你涂一点？”

3.  **第三级：主动性肢体接触**
    此时接触不再需要理由，因为双方已经成为恋人。拉手、拥抱、接吻等都算在内。

**重要原则**：如果遭遇对方抗拒，说明你越级了。此时应退回到上一级，重新拉近亲密感，升级关系后，再尝试进阶。

## 高阶方法：激发性张力

接下来给大家介绍一种高阶的方法去激发对方对亲密的渴望，就是去激发性张力。

**性张力** = **性吸引力** + **阻碍**

这里的“阻碍”不等于“拒绝”。拒绝是“不行，不可能”，阻碍是“现在还不行”。性吸引力在尚未被满足的时候最强烈。你想要得到这个人，但你们之间隔着一层暂时的禁忌，这种迫不及待想要打破禁忌的感觉，就是最强烈的渴望，性张力也达到最大。

**设置阻碍的原则**：
1.  这个阻碍必须是**暂时的**，不是永久的。
2.  这个阻碍是**双方可以克服的**，而不是客观上无法逾越的。

**示例**：当对方邀约时，你可以说：“今天我要加班，实在去不了（设置阻碍）。但是周末刚好有一个很好看的电影，我还挺想看的。如果有空，我们可以一起（引入新的锚点）。”

在亲密互动中，适时地将对方推开（无论是言语还是行为），也能制造性张力。作为被推开的一方，心里会产生更强烈的、想要突破这个阻碍的渴望。而当渴望再次被满足时，感觉会更强烈。

## 总结复盘

本节课中我们一起学习了男女之间亲密感的产生。男女之间的亲密感，归根结底是**物理距离**以及**性吸引力**。

*   我们可以通过**缩短物理距离**（如多次接近、真诚表达）来拉近心理距离。
*   **性吸引力**本质上是对于亲密接触的渴望。我们可以通过**被动提升自身条件**（如外表、气味）和**主动进行语言及行为试探**来激发它。
*   肢体接触需要**循序渐进**，分为礼节性、理由性和主动性三级，并在遇到拒绝时退回上一级。
*   更进阶的方法是制造**性张力**，其公式是 **性吸引力 + 阻碍**。设置阻碍能给予对方突破的动力，但需遵循“暂时”和“可克服”两条原则。
