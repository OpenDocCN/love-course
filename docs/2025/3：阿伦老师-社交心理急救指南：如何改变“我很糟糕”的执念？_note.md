# 社交心理急救指南：3：如何改变“我很糟糕”的执念？🧠

在本节课中，我们将要学习“我很糟糕”这一执念是如何形成的，以及如何通过调整核心观念来改变它。我们将从原因分析和解决方法两个板块入手，帮助你建立更客观、更友善的自我认知。

## 为什么你会觉得自己很糟糕？

上一节我们介绍了课程主题，本节中我们来看看“我很糟糕”这种感受的根源。它主要源于两个关键点。

### 1. 从小不被接纳

我们的自我观念并非天生，而是在成长过程中被塑造的。一个核心起源是“从小不被接纳”。

不接纳自己的人，很容易变成挑剔别人的人。这一点在亲密关系中尤为明显。例如，父母如果内心不接纳自己的紧张或不自信，就可能会挑剔孩子“跟别人玩时太紧张”、“不够活泼”。这种挑剔的本质，是父母将自己不接纳的部分，投射到了关系亲近、边界模糊的孩子身上。

关系越亲近，彼此间的边界越模糊，越容易将对方视为自己的延伸。父母希望通过挑剔让孩子变好，但不断接收“你不够好”信息的孩子，会逐渐内化这种评价，开始认为自己确实很糟糕。一个原本只是有点害羞的孩子，可能因此将“我是一个不自然、不好的人”当作事实。

### 2. 失败体验过多

第二个原因是积累了过多的“失败体验”。这通常源于我们用过高的标准要求自己。

**公式：高要求 → 易失败 → 更高要求 → 更多失败**

小时候，大人常以成年人的能力标准来要求我们。例如，要求一个孩子稳稳拿住碗，或轻松解出数学题。当孩子达不到这些高标准时，就会被批评“没用”、“笨”。久而久之，孩子也会内化这种高要求。

用远高于自身能力水平的标准来要求自己，结果自然是容易失败。而每一次失败，又会强化“我不好”的念头，进而对自己提出更高要求，陷入“要求越高，失败越多，自我感觉越糟糕”的恶性循环。许多人觉得自己事事做不好、没有成功体验，正是源于此。

## 如何改变“我很糟糕”的执念？

上一节我们分析了执念产生的原因，本节中我们来看看如何破解它。“执念”是一种执着且不易改变的观念，比如坚信自己就是糟糕的，即使有相反证据也无法动摇。以下是四个核心的认知调整方法。

### 1. 做错事不等于你糟糕

首先要区分“事”与“人”。做错事仅仅代表那件事情没有做好。

**核心认知：做错事 ≠ 你很糟糕**

我们容易将两者等同，往往是因为从小被灌输了一种逻辑：“你做错事，说明你这个人不好；你不好，就得听我的才能变好。”这实质上是一种控制手段。对方（可能是父母、领导或伴侣）通过将“事”的层面上升到“人”的层面，来制造你的恐惧感和服从。记住，事情做错了，改正即可，这与你作为一个人的价值无关。

### 2. 别人否定你不代表你很糟糕

别人的否定，更多反映的是他们自身的视角和状态。

**核心认知：别人的否定 ≠ 客观事实**

当别人否定你时，需要冷静分辨：是事情本身需要改进，还是对方在借题发挥？很多时候，对方否定你，是因为他们内心不接纳类似的特性（如紧张、粗心），从而将其投射到你身上。例如，一个无法接纳自己偶尔马虎的人，会对你忘关灯的行为大发雷霆。他的激烈反应，根源在于他对自己的不接纳，而非你罪不可赦。

### 3. 无需用否定自己的方式来成长

成长不需要伴随自我鞭挞。

**核心认知：自我否定 → 内心冲突 → 消耗精力 → 无力改变**

很多人误以为严厉否定自己才能进步，甚至用极端方式惩罚错误。但这只会制造巨大的内心冲突，消耗你的心理能量，让你更没有力量去真正改变。健康的方式是“对事不对人”：事情没做好，就专注于如何改进方法、提升技能，而不是攻击自己的人格。法律尚且给予人改过自新的机会，我们为何要对自己更严苛？

### 4. 你不需要那么好

放下对“完美自我”的执念。

**核心认知：追求完美 ≈ 掩饰对不完美的恐惧**

强烈的完美主义，常常只是为了掩饰内心深处对自身不完美的排斥和恐惧。它并不能让你变得完美，反而会让你活得很累，且更容易因小瑕疵而全面否定自己。人无完人，有时正是那些小缺点让我们显得真实、可爱。你不需要成为一个道德完人，也不需要让所有人喜欢。能够照顾好自己，承担基本责任，有一些无伤大雅的缺点，这完全正常。

---

本节课中我们一起学习了“我很糟糕”这一执念的成因与破解方法。我们了解到，这种感受源于从小被挑剔内化和高要求导致的失败循环。要改变它，关键在于学会区分“事”与“人”，明白做错事或遭人否定并不定义你的价值，停止用自我否定的方式驱动成长，并接纳自己“不需要那么好”的事实。记住，改变始于观念的松动，当你开始客观地看待事情与自己，执念便会慢慢消散。
