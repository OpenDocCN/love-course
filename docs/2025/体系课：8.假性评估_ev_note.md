# 体系课：8：假性评估 📊

在本节课中，我们将要学习“假性评估”技术。这是继“打开前提”之后的第三个步骤，也是“价值感阶段”的最后一个步骤。假性评估是整个互动环节中使用最多的技术之一，它关系到能否有效拉升吸引力，并推进关系向变现发展。

## 概述

假性评估是一种符合事实与逻辑的、假装进行挑选的评估类技能。它不仅能让你与对方顺畅地展开有情绪的对话，还能有效拉升你的吸引力和价值感，从而打破“不知聊什么”或“随便聊导致吸引力下降”的困境。

## 假性评估的底层逻辑 🧠

上一节我们介绍了假性评估的基本概念，本节中我们来看看其背后的原理。评估类行为之所以能产生吸引力，是因为它通常是由**高位者**或**有选择权的人**做出的。这类人会表现出挑剔和高标准，例如在挑选物品或人时。这种行为暗示了其自身的优越性和丰富选择。

相反，低价值者往往缺乏选择，容易迎合他人，甚至对虚假的兴趣窗口也会积极回应。假性评估正是通过模仿高位者的筛选行为，在互动中建立价值感。

## 假性评估如何起作用：一个例子 🍊

为了帮助你更好地理解，我们用一个例子来说明假性评估的运作机制。

假设你去水果店买橘子，你对店长说：“哎，感觉你这个橘子皮有点老啊。”这就是一个假性评估。

*   **符合逻辑**：指出橘子皮“看起来有点老”是基于可观察的特征，是合理的。
*   **隐含筛选**：你的话在浅层沟通中，隐含了将其与其他橘子或店铺进行比较的意思，但表达方式比较隐蔽。

这种符合逻辑且带有隐含筛选的评估，更容易让对方“吃进去”并产生后续互动。

## 核心概念：复格 ✅

在假性评估中，“复格”是一个关键概念。它指的是对方面对你的假性评估时，所进行的**自我解释**、**自我证明**或**迎合讨好**的行为。

复格之所以能证明假评生效，是因为只有**框架比你低**的人，才会倾向于向高位者解释和证明自己。如果对方没有吸引力，他更可能无视或拒绝你的评估。

以下是复格与无复格反应的对比：

*   **复格反应**：店长回答说：“不老不老，这个是今天早上刚进的货，可新鲜了。”（进行自我解释）
*   **无复格反应**：店长回答说：“要买吗？不买就走，不要妨碍我做生意。”（拒绝互动）

## 如何提高假性评估的成功率 🎯

既然你知道了什么是假评和复格，那么现在最关心的问题可能是：如何让对方更容易接受假评并产生复格？以及如果对方没有复格，我们该怎么办？

以下是提高成功率的两个核心思路：

1.  **增强逻辑性**：你的评估必须基于可观察的事实或合理的推断。毫无逻辑的假评（例如：“你这个橘子怎么有股草莓味？”）只会让对方觉得你莫名其妙，降低复格概率。
2.  **加大力度与精准度**：评估需要触及对方可能在意或能引发情绪的点。力度太弱（例如：“感觉你这个橘子有点黄。”）无法有效“击中”对方，也难以获得复格。

## 对方没有复格怎么办？🔄

如果已经说出假评但对方没有复格，处理思路依然是围绕上述两点。

*   **思路一：增加逻辑性**。例如，你可以拿起橘子指着说：“哎，你自己看，这里都皱了。”（前提是确实存在这个现象）这为你的评估提供了更具体的依据，可能促使对方改口解释。
*   **思路二：加大力度**。例如，你可以将评价范围扩大：“啊？什么店啊，这样做生意的。”从评价一个橘子上升到评价他的店铺，力度增大，也可能引发对方的复格反应。

## 获得复格后如何维持吸引力？🏆

对方复格只说明你在此刻处于高位，但这个位置是短暂且不稳固的。关键在于如何维持住这种吸引力和高位框架。

答案是一句话：**随机性地给到奖励**。

*   **为什么要奖励？** 因为正反馈（奖励）会鼓励对方重复那些能获得奖励的行为。在女生复格后给与奖励，她的潜意识会认为“迎合或跟随这个男人能获得好的反馈”，从而更愿意持续复格，稳固你的吸引力。
*   **为什么要随机？** 随机性机制（如抽奖、游戏）能极大增强吸引力和让人“上瘾”的程度。随机地给予奖励，而非每次固定回应，会让女人对你更着迷，更投入。

## 假性评估的闭环与进阶 🚀

至此，假性评估形成了一个完整的闭环：进行符合逻辑且有力的假评 -> 引导对方复格 -> 随机给予奖励以维持吸引。

那么，何时代表假性评估环节做得差不多了，可以进入下一阶段呢？

对于一个新手，通常在进行**4到5个成功的假评并获得复格**之后，就可以开始着手进入下一个“联系感阶段”。如果你已达到中阶或高阶水平，可以根据自己的感觉和判断，在**2到3个复格**后便推进。

当你的假性评估做得足够多，收集到的复格足够多时，你会发现对方真的开始喜欢上你。然而，仅有喜欢还不够，对方可能仍缺乏安全感和联系感。在她眼中，你像一颗闪闪发光的宝石，但她还不确定自己是否有资格拥有，也不知道自己有何处值得你喜欢，因此仍会有顾虑，不敢大量投入情绪、时间等资源。

所以，接下来我们需要进入“联系感阶段”。该阶段主要分为三大步骤：**征性评估**、**自我叙事**和**共同叙事**。下一节我们将从“征性评估”开始讲起。
