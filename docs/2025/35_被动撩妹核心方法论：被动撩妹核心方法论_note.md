# 被动撩妹核心方法论：1：核心方法论概述

在本节课中，我们将学习“被动撩妹”的核心概念与完整流程。这是一种通过系统化方法，扩大女性资源池并建立个人吸引力，从而引导女性主动与你互动的策略。

## 第一步：建立鱼塘

## 第二步：经营大鱼塘

在建立了资源渠道后，接下来我们来看看如何经营“大鱼塘”。所谓“大鱼塘”，指的是上述自媒体及社交软件等公开平台。在这些地方，你的目标不是搞定每一个女性，而是追求**高曝光率和将平台粉丝转化为私域好友**。
