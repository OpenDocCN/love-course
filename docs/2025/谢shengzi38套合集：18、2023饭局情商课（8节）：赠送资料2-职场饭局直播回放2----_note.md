# 职场情商课：18：饭局中的自我定位与实用技巧

在本节课中，我们将探讨职场饭局中的核心心态与实用技巧。我们将学习如何找到适合自己的社交“场子”，理解饭局中不同角色的定位，并掌握一些基础但有效的应对策略，帮助你在必要的社交场合中更加从容自信。

## 找到你的“场子”

上一节我们介绍了课程主题，本节中我们来看看如何定位自己。人在初出茅庐的头5到7年里，一个重要的事情是找到属于你的“场子”。你需要非常用力地去感觉什么属于你的场子，什么不属于你的场子。

以下是关于“场子”的两个关键点：
*   对于饭局，有些人天生感到兴奋并擅长钻研，饭局会成为他们的“场子”。
*   有些人则从一开始就反感或不理解饭局逻辑，饭局就不会是他们的“场子”。

没有任何一件事情是绝对必要的，但拥有属于你的1到2个“场子”是必要的。你不能在人情世故的各个方面都毫无建树。你需要慢慢去感受和发现自己的优势领域。

例如，有些人擅长**一对一**的深入沟通，能精准地搞定一个人；而有些人则擅长**一对多**的场合，比如作为饭局的“主持人”（Host），擅长把握全场节奏，让对话流畅进行。还有人可能擅长在正式场合进行**框架性的演讲**。

在你职业生涯的头5年，需要不断尝试和感受，以找到那个你相对能“hold住”的场子。这可能是做业务演讲时，也可能是以主人或客人身份参与饭局时。敏锐地发现和感受这一点至关重要。

## 闲聊是一种稀缺能力

上一节我们讨论了寻找个人定位，本节中我们来看看饭局中的核心技能之一。闲聊是一种能力，它比框架性地输出干货更“活”。这种能力与你的阅历、气质、成长环境以及是否见过形形色色的人有关。

以下是关于闲聊能力的核心观点：
*   **框架式输出**相对容易训练，只要具备一定的语言和逻辑能力。
*   **与人闲聊**的“活”的能力更稀缺，更难以标准化训练，因此在职场中更值钱。
*   能与任何人妥帖地聊两句，意味着这个人能够融合三教九流，心中没有成见。

许多年轻人心中带有成见，例如对“拍马屁”或“搞关系”的人抱有看法，并将这种成见误认为是“傲气”或“不同流合污”，这某种程度上是一种幼稚。成熟老练的人心中成见更少，因此也更稀缺。

## 饭局参与者的心态与角色

上一节我们分析了闲聊的价值，本节中我们来剖析饭局参与者的不同心态与角色。必须承认，有些人面对饭局天生感到兴奋。他们乐于观察形形色色的人在不同阶段的状态变化，并将饭局视为一个有趣的“剧场”。

对于天生反感或无感饭局的人，需要判断这个场合对你是否必要。如果必要，你可以通过刻意训练，掌握一些惯用的技巧作为“保护层”。以下是饭局中常见的几种角色分类：

*   **主持人（Host）**：负责组局、迎来送往、把控饭局节奏和氛围。这种人构建社交生态的能力很强。
*   **捧哏者**：幽默、会讲段子、知晓各种八卦，能活跃气氛。即使Host不一定特别喜欢他，他的存在也常是必要的。
*   **价值呈现者**：自身拥有核心Title、足够优秀或外形出众。他们的存在本身就能为饭局“撑场子”，不需要说太多话。
*   **配角**：身份和情商尚可，主要是在场参与。配角角色不固定，可能会轮换。

清楚自己在每一场饭局中的定位很重要。长期来看，你会逐渐形成一个自己越来越游刃有余的角色。

## 饭局中的实用基础技巧

上一节我们认识了饭局中的各类角色，本节中我们为希望提升饭局表现的朋友提供几个实用方向。如果你觉得自己在饭局中比较木讷尴尬，但又不反感，并希望有所提升，可以从以下几个方面入手。

以下是几个可以发力的方向：
1.  **会聊菜与酒**：这是一个常见的破冰和转话题的切入点。但门槛低，精通难。需要真正对美食、食材、不同菜系或酒文化有研究和积累，否则容易显得生硬。
2.  **准备灵活的自我介绍**：在需要介绍自己的场合，关键是**融入**而非突出。介绍的长度要适中，并最好能**呼应（echo）** 场上其他人提到过的信息（如家乡、行业、经历），自然地引出自己的相关点，这比自说自话更高级。
3.  **注重气场与状态**：饭局中，你的气场、眼神、坐姿和倾听神态，往往比具体说了什么更重要。少说话可以避免言多必失。通过得体的非语言交流，可以给人留下更好的印象。
4.  **把握进食的尺度**：在饭局上，吃得过多或过于投入显得不够优雅和自律。象征性地品尝每个菜，将更多精力放在倾听和观察上。尤其当不被重视的人讲话时，你能放下筷子专注倾听，会极大提升对方对你的好感。

社交和饭局在某种程度上是一场“表演”，需要一定的“演技”。你可以根据场合需要，调整自己的外在表现。

## 关于喝酒与拒绝的艺术

上一节我们介绍了一些行为技巧，本节中我们直面一个具体难题。关于喝酒，核心原则是：如果你不擅长或不喜欢，它不会成为你的长板，那不如明确将其作为短板。对于初入职场、业务不在自己手上的年轻人，不必强求。

如果遇到劝酒，可以这样处理：
*   **态度坚决，姿态柔和**：微笑着但明确地说“今天真的不行”。如果对方再劝，可以重复拒绝或提议“以茶代酒”。
*   **敢于停顿与对视**：在拒绝后，一个短暂的停顿和坚定的眼神，可以形成气场上的对峙。通常劝三句以上就是对方失态了。
*   **事后找补**：如果拒绝让对方有点没面子，可以在饭局后续过程中，通过帮他服务、捧他一下等方式，把面子找补回来。

需要理解的是，真正有分量的大领导或大佬，很少会直白地逼迫下属喝酒。他们更倾向于把选择权交给你，通过观察你的表现来做筛选。他们说话往往只说半句，需要你自己去“悟”。

## 高阶思维：领悟与筛选

上一节我们讨论了具体问题的应对，本节中我们提升视角，看看高层互动的本质。在中国商业和职场文化中，能量强的人说话方式具有共同特点：**从来不把话说透，从来不把话说破**。

他们通过提问、暗示或提及某个名字，让下属自己去领悟意图。例如，想提拔一个人时，可能只是不经意地提起“那个部门姓谢的小姑娘上次发言不错”，自然会有心领神会的人去推动。

因此，许多想象中的“领导逼你喝酒”的场景可能并不存在。真正的挑战在于，你是否能听懂那些没有明说的“半句话”，并做出恰当的回应。这需要阅历和对公司动态的敏锐洞察。

本节课中我们一起学习了在职场饭局中如何自我定位。核心是找到让你感到兴奋或至少能驾驭的“场子”，并理解闲聊等软技能的价值。我们分析了饭局中的不同角色，并提供了从聊菜、自我介绍到控制气场、得体拒绝等一系列实用技巧。最后，我们探讨了高层沟通中“领悟”高于“命令”的本质。记住，没有什么是绝对必要的，但拥有属于自己的“一招先”并在必要时懂得如何保护自己，是在复杂职场环境中前行的重要智慧。
