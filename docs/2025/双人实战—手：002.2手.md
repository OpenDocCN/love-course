# 手暖御龙：002.2：手爱18式之玉女戏珠

在本节课中，我们将要学习手爱技巧“玉女戏珠”中的第一个核心手法——“抓蛋”。这个技巧旨在通过特定的手法，对阴囊进行温和而富有节奏的刺激。

上一节我们介绍了“玉女戏珠”的基本概念，本节中我们来看看其第一个具体操作方法。

## 核心技巧：抓蛋

“抓蛋”是指使用五指指腹，以富有节奏且动作轻柔的方式抓握和刺激阴囊。

以下是具体的操作步骤描述：

*   用五指指腹接触阴囊。
*   动作需富有节奏且保持轻柔。
*   抓握过程类似于此动态描述：一下，这样下来，一下。

通过这种轻柔、有节奏的抓握，可以带来有效的刺激。

本节课中我们一起学习了“玉女戏珠”式的起手技巧“抓蛋”，其关键在于使用指腹进行节奏轻柔的抓握。掌握好力度与节奏是运用此技巧的核心。
