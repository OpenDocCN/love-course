# 白天沟通：09..真诚永远是必杀技

🎼学习完了第一章，我相信很多姐妹们开始跃跃欲试了，对不对？但是啊由于你们俩长期缺少一个高质量的双人游戏，所以导致你不知道怎么跟你老公开这个口，对不对？这个时候啊学会适当的勾引老公，非常的重要。

那么如何去勾引她呢？来，接下来教大家如何跟老公进行一个亲易的沟通，我跟你讲这一章非常重要了，你学会了跟老公如何的进行一个亲易沟通。那么接下来不管是双人游戏当中还是你们日常生活当中。

我相信你都是会如鱼得水的，亲易沟通的第一点就是要学会去表达自己内心的真实想法，你千万不要说反话，也不要说细话，更不要去说讽刺人商人的话，来啊，有多少姐妹平时是这样子的。比如说啊你老公在家干活。哎。

你就说。🎼这家是我一个人的吗？啊，就我一个人干活，这孩子是我一个人的，你做过一次家务吗？你带过一次孩子吗？你在家除了拉屎睡觉还会干什么？你是聋的吗？孩子哭了，你听不见你是瞎了吗？这地板这么脏，你看不见。

好不容易啊，你老公做了一点事情，哎，你又会说，哎呦，今天太阳从西变出来了，你也知道做家务了呀，你也知道带孩子呀。你也知道带娃很辛苦，做家务很累呀，天哪，你看你这个语气。

是不是比平时你那个不对付的同事还阴阳怪气，这就是我们很多姐妹们都会犯的错误，越是亲密的人，我们说话越难听怎么了？你是觉得他是赶不走的，对吧？我跟你讲，谁都喜欢听好听的话，尤其是男人，我们得夸。

男人是越夸越好的。😡，🎼哪怕她十几年都不做家务，只要他做了那么一次，你就给我狠狠的往死里夸。来啊，接下来就教姐妹们如何的去夸老公。
