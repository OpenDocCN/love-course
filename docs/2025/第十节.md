# 第十节

蓝兵出品必属精品。磁径中的三个框架。🎼这章我们来分享一下框架三要设置容错底线。你会发现每一个人呢都不是神，你不可能把每一次竞争都做到完全的掌握在手中，并且百战百胜，对吧？总会有失败的时候。

那些没有思考过失败的人，就会在失败中一败涂地，甚至是一蹶不振。所以在任何竞争开始之前，一定要先设置好失败时的应变条件，最重要的是错了以后怎么办。我们来想一想，在任何竞争中呢。

都要很清醒理智的知道自己的底线在哪里，词境也不例外，容错底线很简单的理解，就是一旦这件事情判断错误，或者说是事情朝着错误的方向发展，什么时候停止，什么时候止损，什么时候收手，什么样的情况算是错误的。

什么样的状态，算是失败了，这都需要你提前去规划去确定把确定以后的结果作为原则，坚定不移的在实践中去实施。你就会很轻松的去面对任何情景。举个例子吧。🎼举两个情感中的例子，你就明白了。比如你和一个男人交往。

你付出了情绪，付出了生理价值。你想获得这个男人的托举或者财富。那么如何设置容错底线呢？比如你现在年龄都是30多岁了，那你一定不能像20岁小女生一样去设置他的容错底线。

比如这个小女生可以设置两年为投资目标，容错底线就是一年，在这两年你没有得到任何收益，他就会离开这段感情，顶多再给这个男人一年的时间去验证。这一年就是容错底线，而你就不能这么设置，因为你不是小女生。

你要把容错的时间设置为3个月，比如说你和这个男人在一起超过了两年还没有得到你想要的结果，那么接下来的三个月内，无论如何你都应该给自己一个结论，就应该立即抽生或者重新设置获取目标。

而不能像那些从来没有设置过这种容错底线的女人一样，死磕到底死缠烂打，纠结悔恨内耗，耗费了半年一年甚至是一辈子为一件事情而痛苦。最后还是没有结果，这就是愚蠢。🎼再比如你爱上一个事业有成的男人，想结婚。

结果他却一直拖延，不愿意给你一个明确的承诺和答案。那么在这种情况下，容错底线的设置就显得尤为重要了。你必须要明确自己能够接受最长等待时间是多少。比如6个月或者一年，在这段时间内。

如果他依然无法给出一个明确的未来规划，那么你就应该立即重新评估这段关系的价值。甚至是在这个关系的同时，你要开始寻找下家了。也许你会觉得呀事业有成的男人是值得等待的。但事实上。

他正是抓住了你的这一个心理弱点，肆无忌惮的挥霍你的时间，时间对于每个人来说都是宝贵的。如果你已经给他足够的时间和机会，而他仍然无法确定你们的关系。那么继续等待，只会让你失去更多宝贵的机会。

🎼这个时候你的融错体线就是你的自我价值体现，也是你对自己。🎼人生幸福的负责任的一个表现。所以啊我们要知道，不但要在感情中设置容错的底线，在生活的任何竞争过程中都要去设置容错底线。

包括投资、事业、交友等等容错底线的设置。🎼能够让你不至于在一件事情上投入太多的精力，一错再错，让你有再次翻盘的机会，这一点非常重要。好，讲了那么多，我们来总结一下，容错底线的设置很简单。
