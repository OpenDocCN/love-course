# 女士八爪鱼训练技术：014.14：大雁飞 🦢

在本节课中，我们将要学习“大雁飞”这一动作。这个动作的核心在于掌握匀速、有控制的发力方式，它并非要求你记住复杂的动作，而是理解其背后的原理。我们将通过分解练习，帮助你体会控制力的运用，并将其与之前学习的呼吸技巧相结合。

## 动作原理与核心概念

上一节我们介绍了控制力的重要性，本节中我们来看看“大雁飞”如何具体体现这一点。

“大雁飞”的发力方式是匀速地扇动翅膀。这是一种**控制力**。老师不要求大家死记硬背动作，而是理解1到10这十个训练场景。理解了这些，以后无论看到天上飞的、地上跑的，还是水里游的，你都能联想到相应的练习方法。

匀速地抬起，如果没有力量控制而匀速地落下，结果就是掉下来。因此，这个动作的精髓就在于**控制力**。

## 分解动作练习

以下是“大雁飞”动作的分解步骤。我们将跟随呼吸的节奏来完成。

1.  **找到起始位置**：首先，找到动作的底位。
2.  **收缩阶段**：吐气，将力量收回到一半的位置；再次吐气，完全收缩。
3.  **释放阶段**：吸气，将力量释放掉一半；再次吸气，完全放松。

在这个过程中，明显运用到了我们之前学习的“吹蜡烛”式呼吸法。现在你应该明白，为什么老师让大家练习前面的内容，因为它们是相互关联的。
