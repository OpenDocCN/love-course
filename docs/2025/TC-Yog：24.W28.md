# 脉轮瑜伽：24.W28：眉心轮1－金刚坐+犁式

在本节课中，我们将要学习眉心轮的基础知识，并练习两个能够有效激活和平衡眉心轮的瑜伽体式：金刚坐与犁式。眉心轮是通往更高意识的门户，掌握它对于提升专注力、稳定情绪和扩展觉知至关重要。

## 眉心轮概述

上一节我们结束了喉咙轮的探索，本节中我们来看看位于更高层次的眉心轮。眉心轮，常被称为“第三眼”，是我们内在视觉和直觉的中心。它位于前额中央，而非两眉之间。这个能量中心对应着我们的心智、专注力以及超越二元对立的觉知能力。

眉心轮对应的感官是**内在视觉**。例如，当你闭上眼睛想象家中的场景时，脑海中浮现的画面就是内在视觉在运作。对应的行动器官是我们的**头脑（心智）**，其主要功能是思考与做出选择。

在能量系统中，眉心轮如同一个**控制中心**，管理者下方五个脉轮。你可以将其比作公司的老板，负责决策和指引方向。员工（下方脉轮）的执行力，很大程度上取决于老板（眉心轮）的指令是否清晰、稳定。

## 眉心轮的象征与功能

眉心轮的积极象征是**第三眼**。我们的肉眼是水平生长，用于观察物质世界，而第三眼是垂直的，能让我们“看见”能量、感受精微体验并洞察事物的本质。它帮助我们超越非此即彼的二元对立思维，进入更中正、全息的视角。

眉心轮对应的生理腺体是**松果体**。松果体对光线敏感，在黑暗中分泌褪黑素，帮助身体修复。在哲学层面，它常被视为意识与灵魂的居所。较高层次的腺体（如松果体和脑垂体）能够调节身体其他腺体的功能，这从生理层面解释了眉心轮为何能统御其他脉轮。

## 眉心轮在心灵层面的表现

我们的情绪、念头主要活跃在心灵层（或称情绪体）。眉心轮在这个层面如同电影院的**控制室**，它调控着整个“观影体验”（即我们的人生境遇）。

当眉心轮被唤醒，我们便获得了跳出情绪剧情、回到“控制室”观察的能力。例如，当因他人爽约而愤怒时，唤醒的眉心轮能让你以全息视角看到：对方的行为可能源于其自身的议题，而你的愤怒或许与过往未被疗愈的创伤模式有关。这种即时的、临在的觉知，能让我们在念头升起的瞬间就进行转化，而非被情绪裹挟，从而改变行为模式，获得内在的宁静。

因此，定期激活眉心轮，对于管理情绪、缓解焦虑抑郁、提升整体心理健康有显著益处。

## 激活眉心轮的核心工具：视觉观想

在谭崔瑜伽中，激活眉心轮的一个核心工具是**视觉观想**。传说中，世界正是由希瓦神的观想所显化。这揭示了一个创造法则：持续在意识中保持一个画面，就能将其显化到现实中。

以下是进行显化观想的步骤：
1.  **明确目标**：确定你想要显化的事物，例如理想的伴侣或事业成功。
2.  **构建生动细节**：尽可能细致地观想该场景的所有感官细节——视觉形象、声音、气味、触感。
3.  **沉浸式体验**：以“我已经拥有”的状态去感受，让自己沉浸在那个现实的频率中。
4.  **持续练习**：反复进行观想，让这个内在画面变得鲜活、稳定。

这个练习的关键在于，不是“渴望”成为，而是“已经”成为。让你的内在状态首先匹配你想要的外在现实。

## 体式练习一：金刚坐

现在，我们将通过体式来实际激活眉心轮。首先学习的是金刚坐，这是一个能同时平衡心轮与眉心轮的静态坐姿。

**练习步骤：**
1.  跪坐，双膝分开约一拳宽，大脚趾相靠近但不触碰，脚背贴地。
2.  将臀部缓慢向后，坐在脚后跟上。若感困难，可在臀下垫瑜伽砖或抱枕。
3.  保持脊柱直立，胸腔打开，双肩下沉。微收下巴，让头颈与脊柱在一条延长线上。
4.  双手自然置于大腿根部或下腹部。
5.  放松全身，特别是骨盆和双腿，仿佛坐在椅子上一样无需用力。
6.  将注意力平衡地放在心轮（胸腔中央）和眉心轮（前额中央）。感受能量如金字塔般向上汇聚，在眉心轮凝结如钻石。
7.  保持5-30分钟。退出时，缓慢伸直双腿向前，放松。

**益处与原理：**
*   **生理层面**：促进腹部血液循环，助消化，缓解便秘或腹胀。饭后练习尤为适宜。
*   **能量层面**：直立的脊柱与放松的下半身形成能量通道，引导能量向上聚集于眉心轮。
*   **心理层面**：平衡心与脑，让情感与理智协同工作，使人进入中正、宁静的冥想状态。

## 体式练习二：犁式

接下来我们练习犁式，这是一个能深度净化头脑、激活眉心轮的倒立体式。

**练习步骤：**
1.  仰卧，双腿屈膝，脚掌踩地。双臂放于体侧，掌心向下。
2.  吸气，双脚蹬地，将臀部与背部抬离地面，双腿越过头部，尝试将脚趾点向头后方的地面。若脚无法触地，可放在椅子或砖块上。
3.  一旦进入体式，将双手掌心转向上，置于地面。感受双腿吸收大地的能量，双手吸收宇宙的能量，两股能量交汇于眉心轮。
4.  保持身体稳定，颈部放松。意识保持中立，观察升起的念头但不与之认同，任其流走。
5.  保持1-5分钟。退出时，呼气，掌心压地，脊柱一节节缓慢回落，确保头部始终不离开地面。

**禁忌与注意事项：**
*   严重疝气、椎间盘突出急性期、高血压不稳定者避免练习。
*   女性月经期前两日及量多时避免练习。

**益处与原理：**
*   **象征层面**：如农具犁松土，犁式能“犁松”我们固化、僵硬的思维模式，为播种新意识准备“土壤”。
*   **能量层面**：整合大地（扎根）与宇宙（升华）的能量，在眉心轮汇合，极大激活此中心。
*   **意识层面**：强大的能量灌注与身体的静止，能极快稳定头脑，超越琐碎念头，体验意识扩展的宁静与极乐。

## 体式练习的要点：眼睛的引导

在练习犁式、肩倒立等体式时，眼睛的移动至关重要。眼睛是身体的**方向盘**，引导着动作的方向与幅度。

例如，在进入犁式时，让眼睛随动作自然看向胸骨方向，能协调颈部与背部肌肉，让体式更轻松、安全地完成。有意识地训练眼睛，还能帮助矫正因日常习惯导致的身体不平衡。

## 总结

本节课中我们一起学习了眉心轮的核心概念与激活方法。眉心轮作为意识的控制中心，通过视觉观想和特定的瑜伽体式（如金刚坐和犁式）可以得到有效唤醒与平衡。这能帮助我们提升专注力、稳定情绪、获得全息视角，并开启内在的直觉与智慧。坚持练习，你将能更好地掌管自己的“人生控制室”，创造更和谐的内在状态与外在现实。
