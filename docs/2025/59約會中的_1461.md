# 约会艺术：59：约会中的消极状态

在本节课中，我们将通过一个具体案例，分析约会中一方陷入消极情绪状态时的表现、成因及其影响。我们将学习如何识别这种状态，并探讨如何调整自身心态，避免陷入负面循环。

## 案例回顾与状态分析

上一节我们介绍了案例的背景，本节中我们来看看这位男生的具体表现。

在约会互动中，这位男生表现出明显的情绪化。情绪化通常意味着个体心理上的“退行”，即行为举止退回到更幼稚、不成熟的阶段。这位男生年龄接近30岁，但其表现却像一个小男孩，容易让人联想到其性格或脾气可能存在问题。

## 消极状态的具体表现

以下是陷入消极状态时可能出现的几种典型行为：

*   **倔强与沉默**：表现为头部倾斜、沉默不语、内心不服或不认同。这类似于童年被父母责打时咬牙倔强的状态。
*   **封闭自我**：完全沉浸在自己的负面情绪中，将自己包裹起来，无法再打开心扉与他人互动。
*   **应对失当**：当被女生拒绝或询问时，回应方式缺乏情绪，甚至可能制造新的误解。例如，当被问及“是否为其他女生做过同样的事”时，生硬地回答“有的联系有的不联系”，这会让对方产生更多疑虑。

## 心态调整与核心建议

当在约会中遭遇拒绝时，情绪受到影响是正常的。关键在于如何应对。

核心建议是：学会自我关怀，而不是将全部希望寄托于对方的安慰或认可。公式可以表示为：

**健康的自尊 = 稳定的自我认同 - 对他人认可的过度依赖**

如果将期待完全寄托于对方，而对方未能满足，个体就容易陷入“受害者”心态，认为对方伤害了自己。

## 理解女性视角与作业

从案例中女生的提问可以看出，女性在约会中非常在意行为的“唯一性”，这是其情绪需求的重要体现。她们会无意识地通过一系列问题来试探和确认。

面对女生的问题，关键在于理解其背后的情绪需求，而非仅仅进行事实层面的刻板回应。

本节课的最后，我们认识到，问题本身往往不是关键，**我们面对问题的态度才是决定因素**。案例中的男生将女生的拒绝归因于“自己不够优秀”或“职业问题”，这很可能源于其内心深处的**限制性信念**，例如“我不配得到爱情”。

因此，我们布置一个小作业：请深入反思，在过去的几十年里，你内心可能存在着哪些关于自我或关系的限制性信念？识别它们是改变的第一步。
