# 晚上舞蹈：13.乳爱

姐妹们这节课我们讲乳爱。我们唉乳爱它不同于乳胶，可能很多姐妹听到乳爱说，哎呀，老师我的胸好小啊，怎么乳爱呀，对不对？是不是只有胸大才能乳爱？你可能会把乳爱跟乳胶相关联了啊。可能你如果说你需要乳胶。

对不对？比如啊你需要用你的胸夹着它的阴茎上下的这样的搓动啊，乳胶的话可能就会要你的胸部要大一点。但是我们乳爱不仅仅是乳胶这一个动作哦，乳爱有很多玩法的，更多玩法其实根本不需要你胸大。

有胸就可以有乳头就可以玩，好不好？姐妹们啊来首先讲常规的啊，常规的也就是你们之前所认为的传统的乳胶的方式，对不对？就是去蹭它。😊，当对方躺好之后，你就可以用你的胸部去蹭他。胸大的宝贝。

你整个胸部都可以啊，无的是整个乳房乳头都可以。如果是胸小的姐妹，那就一定要练好你的乳头，对吧？用你的乳头去蹭它一样的啊，从弱敏感区到强敏感区，从阴囊往上走，对吧？

如果说你是没有抹油的那么你的力度一定要轻柔缓慢。你平时偶尔玩一下，可以不用抹油啊，当做情趣。如果你想正儿八经的玩一场很正式的如爱，那么还是要建议抹点油，有了油的润滑可以玩的更久，对方呢也会更舒服。

可以上下左右蹭，也可以打圈。从阴囊往上走。到整个阴茎体，再到它的龟头的位置，龟头这里很好玩啊，可以用我们的乳头去蹭它的冠状沟和龟头部位。就是用你的敏感去蹭对方的敏感去是非常舒服的。胸大的姐妹啊。

好吧。这个时候如果对方的钉钉够长，包括你的舌头也够长的话，可以用舌头去舔它的龟头，这样的话也能够让他的龟头也产生快感。这种形式的乳胶，无论是对方的视觉上还是感官上都是非常刺激的。有的姐妹就是条件还不错。

但就是胸部有点下垂怎么办呢？那这个时候你可以穿上内衣，然后把胸部从内衣里面掏出来。内衣不用脱啊，内衣就在下面做支撑，这样子胸部在上面，胸型就会更紧凑，更饱满。除了传统的用胸部夹着阴茎玩之外。

我们还有很多别的乳胶玩法。比如说啊你的整个胸部可以在他整个身上游走，对吧？在他的阴茎附近都蹭完了之后，可以继续往上。哎，从阴茎到他的肚子，然后再到小腹，用乳头啊、乳头、乳房都可以来到他的明明区。
