# 自媒体与个人IP：03：自媒体的延伸价值

在本节课中，我们将探讨自媒体活动超越商业回报的深层价值，学习如何以“正面”心态进行创作，从而实现个人成长与自我提升。

## 课程概述

在前面的章节中，我们深入探讨了自媒体可能带来的商业价值、社交价值以及流量影响力。本节作为本篇章的最后一节，我们将跳出事情本身，向更远的地方发散，聚焦于自媒体活动对个人内在成长的延伸价值。

## 自媒体作为一场修行

如果你真真正正是用心的在做自媒体，那么做自媒体本身就是一场修行。

你在不断表达、不断与互联网受众互动的过程中，会更了解自我，会更加敏感于内心的感受和身边的生活素材。你面对不同的评价，会变得越来越淡定。如果你真的找到了一种很好的感觉和角度，那么你是可以通过自媒体获得快速的自我成长，从而变成一个更好的人的。

## 定义“好工作”与“值得做的事”

前几天我在和我的朋友聊，也在和我的团队聊，到底什么是一份好的工作。我觉得有一个定义，就是通过这份工作，你本身可以变成一个更好的人。这种“好”你可以理解为更美好，也可以理解为更强大。

那什么是一件值得做的事呢？就是无论这件事情最终的世俗结果怎么样，这段经历本身就能够让你的能力，甚至是品格得到很大的提升。

## 以“正面”心态做自媒体

所以在这一节，我很想跟大家分享一些正面做自媒体的心态。什么是“正面”呢？我们可以类比“正面饮食”和“正面健身”。

*   **正面饮食**：我不会去想吃这个东西是会胖还是会瘦。我在吃甜点的时候，即使是一些比较垃圾的食品，我也在用心感受它的甜和这一刻带给我的开心。我在吃一些比较素的东西的时候，就用心感受它的清爽和食物本身对话。
*   **正面健身**：就是把精力放在自己的动作和体能上，精神很专注。

如果你在做自媒体的过程中，一方面保有业务视角、商业视角，同时还有一个很正面的、自我提升的视角，就会非常好。

以下是结合两种视角的实践方法：

*   **业务视角**：比如我写了一条微博，我不断去看点赞、评论、转发，不断去感受用户喜欢什么样的内容。这是你做自媒体的业务视角。
*   **自我提升视角**：我为了日更微博，保持了持续的记录，它本身就是一种自律的自我表达。为了写更多的微博，我必须更敏感于身边所发生的一切。我可以不断品味自己写的微博，体会自己的进步和在不同阶段所表达角度的不一样。

那么最后的最后，即使写微博这件事情并没有让你赚到一毛钱，你也已经得到了很多与你的人生而言宝贵的东西。

## 平衡世俗与心灵维度

我觉得做很多事情，既要有一个世俗的角度，也要有一个不世俗的角度，这样会让你整体的心态更好。同时，不世俗的角度，有可能是让你的世俗结果更容易达成的。正是你的情绪没有那么起伏，目的性显得没有那么强，有可能你本身这个IP和你的内容本身是更加有感染力的，从而更加有了一些商业的价值。

这套课，除了“自媒体是最好的个人能力放大器”之外，其实还有一句slogan叫做“始于自媒体，高于自媒体”。那么这一个篇章，我们讲商业模式。而在这最后一节，我们讲了一点点跟商业无关，但是跟心灵、你的能量、你的自我成长相关的东西。希望大家有非常丰富的心灵维度去面对自媒体和面对个人IP。

## 课程总结

本节课中，我们一起学习了自媒体作为个人修行和成长工具的价值。我们探讨了如何以“正面”心态平衡商业目标与自我提升，并理解了丰富的心灵维度对于打造有感染力IP的重要性。记住，一段有价值的经历本身就能带来能力和品格的提升。
