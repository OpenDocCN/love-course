# 搭讪教程：第26天：通过积极频率建立连接

在本节课中，我们将学习一个简单的互动游戏，其核心是通过保持积极的状态（特别是笑容）来建立连接，从而增加获得亲密互动的可能性。

上一节我们介绍了如何开启对话，本节中我们来看看如何通过一个具体的游戏来推进互动。

## 🎮 核心游戏：石头剪刀布

我们将使用一个简单的“石头剪刀布”游戏作为互动的载体。以下是具体的话术和步骤。

**核心话术**：
```
“嗨，你好，我感觉得你挺有气质的。想和你玩个游戏，石头剪刀布。如果我赢了的话，我可以得到一个快速的吻。”
```

1.  开启对话：看着对方的眼睛，说出上述话术。
2.  执行游戏：与对方进行石头剪刀布游戏。
3.  处理结果：
    *   如果获胜，可以尝试获得一个快速的吻。
    *   如果对方提出疑问或设定条件（例如“如果你输了呢？”），可以灵活回应，例如“那我奖励你一个拥抱”。

## 😊 关键心态：保持笑容

在整个互动过程中，保持笑容是至关重要的。这不仅仅是表情，更是一种心态。

*   **强迫自己笑**：即使没有值得高兴的事情，也要强迫自己微笑。
*   **笑到最后**：即使互动结束对方离开，也要笑着道别，例如说“拜拜，玩的开心”。
*   **笑的原理**：笑容能发出积极的频率。这种积极的频率能够吸引好事，让他人放下戒心。

简单来说，**笑 → 发出积极频率 → 吸引好事/降低防备**。不要把事情想得过于复杂。

## 💡 互动实例与总结

以下是一个简化的互动实例，展示了上述原则的应用：
1.  打招呼并自我介绍：“你好，我叫马克。”
2.  使用核心话术提出游戏邀请。
3.  在游戏和后续简短聊天中（如询问家乡），始终保持笑容和轻松的状态。
4.  无论结果如何（成功或对方婉拒），都愉快地结束对话。

本节课中我们一起学习了如何通过“石头剪刀布”这个简单的游戏框架进行互动，并深刻理解了**保持笑容和积极状态**是吸引积极回馈、降低他人防备的核心。记住，简单的事情重复做，关键在于你传递出的频率。
