2.  待其略微软化后，再次开始刺激，接近临界点时再次停下。

4.  在第6次时一次性释放。

采用此法比第一次就射精的感受更好。但需要注意，必须准确判断临界点。若在未达临界点时停止，却又忍不住释放，会导致快感缺失并感到疲惫。平时多加练习，更容易找准这个点。
