# 恋爱军团相亲宝典：04：沟通：相亲中女生必问的刁钻问题

在本节课中，我们将要学习在相亲初期沟通时，如何巧妙应对女生可能提出的几个刁钻问题。这些问题看似平常，实则暗含考察，理解其背后的意图并掌握恰当的回应方式，对于建立良好的第一印象至关重要。

无论是通过相亲活动还是亲友介绍，双方初次见面的了解主要依靠交谈。女生往往会在交谈中设置一些“陷阱”来考察男生。之前女生相亲群里流传过一个故事：一位女生想知道男生的经济状况，但觉得直接问不妥，于是她问：“你们小区的停车费一年交多少钱？”这个问题同时考察了房产、车辆和小区的档次。

不要觉得惊讶，女生在恋爱中的观察和推理能力很强。在你不知情甚至洋洋得意时，可能已经被对方否决。下面老师罗列了几个在相亲初期认识阶段常见的刁钻问题。请注意，如果双方已经决定深入了解或彼此有好感，则不必采用下述的应对方式。

以下是几个常见问题及其应对策略：

**问题一：你之前交往过几个女朋友？**
这个问题几乎是必问的。女生想了解未来伴侣的情史很正常。但回答多有被认作对感情儿戏的风险，回答少有被怀疑缺乏魅力或不懂照顾人。
**标准答案**：身边确实有过一些来来往往的人，但并没有人能够真正走进我的内心。

**问题二：你和你之前的女朋友是怎么分手的？**
女生会不自觉地将自己代入你讲述的故事中，并从你的描述里寻找你的问题。说得越多，被挑剔的方向就越多。
**标准答案**：当时的情况比较复杂，主要是性格不合/感情淡了/家里反对/异地。后来双方都没有心思继续，就和平分手了。分手后，我们再也没有联系过。

**问题三：你的工资/收入是多少？**
收入能直观证明男人的能力和预示婚后的生活质量。直接回答具体数字可能带来尴尬。
**标准答案**：工资不多也不少，反正不用为吃穿发愁。

**问题四：你为什么来相亲？**
直接回答“单身久了想找对象”会削弱吸引力，暗示自身质量有问题。要让女生觉得你是有选择的人。
**标准答案**：以前好奇相亲是不是像电视里那样有很多奇葩场景。最近见过几次，发现很多女孩其实非常不错，让我很惊讶。我很享受目前单身的状态，但如果遇见很投缘的女生，谁知道后来会怎样发展呢？

**问题五：你经常来相亲吗？**
这个问题具有测试性。回答“经常”可能显得挑剔或不受欢迎；回答“不经常”又可能显得对结婚意愿不强烈。
**应对方法**：不正面回答，可以转移话题。例如：“我阿姨从小对我很好，她好像对你印象很不错，让我一定要认识你。正好我这几天回老家有时间，我也想多认识一个优秀的朋友。”

**问题六：你的工作具体是做什么的？事业如何？**
女生想了解你的收入、生活压力、工作时间和潜力。干巴巴地回答职业名称很普通。
**应对方法**：描述职业属性和个人成就，并关联到能为未来伴侣带来的价值。例如，如果你是会计，可以说：“我从小对数字敏感，当会计后感觉比老板还了解公司。虽然工作不累，但需要心思很细腻才能胜任。”

关于事业成就的展示需要注意分寸。如果女生自身事业有成，她可能会认为自己的成就是努力得来的，未必欣赏你单纯炫耀收入。无论何时，必须让对方感受到你的工作能力能为“你们”的未来带来具体作用，否则可能引起反感。
**价值关联示例**：“我是旅行社的，我们每年都有员工家庭出行福利，一般去欧美国家。如果我以后有女朋友，每年都可以带她去旅游。”这就是将工作价值与另一半关联起来的说法。
