# 恋爱军团课程：03：植入观点 攻破她的内心防线（上）

## 概述
在本节课中，我们将要学习“植入观点”这一核心技巧，并探讨如何通过给予安全感来攻破对方的心理防线。课程将分析安全感在亲密关系中的根本作用，并介绍不同的依恋风格类型。

---

大家好，我是小鹿情感恋爱军团的金牌导师金子。今天由我为大家带来关于“植入观点”的课程。

今天的课程主题是攻破对方的心理防线。部分学员可能对“植入何种观点才能攻破女生心理防线”感到困惑。本次课程将讨论一个关键话题：给予女孩安全感，她才能感到安心。

在她心中会形成一个观点：与这个男人在一起能获得安心的感觉，那么选择他就是正确的。

在此之前，请允许我提出一个问题。两个人恋爱后，是否有权利查看对方手机？这包括知晓对方的开屏密码、浏览对方社交网络的动态和私信，甚至查看微信等即时通讯软件的信息。

部分人认为当然有此权利，另一部分人则认为这是侵犯隐私。对方是你的伴侣，并非你的所有物，为何要如此严格审查，不能保留一点隐私吗？

本节课不讨论此问题的标准答案。我希望能通过“是否查看手机”这个问题，揭示其背后的根本原因在起作用。

其实就是安全感在起作用。

安全感危机很容易被点燃。潜在情敌的出现、对方态度的转变、异地恋的风险，甚至“我最近要考研，我们的关系先放一放”这类表述，都可能引爆危机。许多安全感危机源于自我困扰。

例如，“我给他买了那件衬衫，他最近没穿。”实际上可能只是因为天气热了，不适合穿长袖。又如，“他最近回微信很慢。”但真实原因可能是考试季到了，他需要时间复习。再如，“他最近和一位同事走得很近。”实际上他们可能同属一个项目组，因领导催促而需要一起加班。

长期以来，人们普遍认为安全感的核心在于给予的量是否足够。如果一个人未获得足够安全感，就会采用各种手段来验证自己在他人心中的分量。

女孩因为缺乏安全感，才会对男人的经济和隐私严加看管。竞争关系因此产生，若处理不当，感情便可能破裂。

但你首先需要纠正一个原有观点，其中存在一个误区：即使提供了足够的安全感，也无法解决每一个因安全感缺失而引发的问题。

我曾看到一位学员的留言，是关于亲密关系的求助。在我看来，他已经提供了足够的安全感：女朋友要管理银行卡，他上交了；女朋友要天天查看短信和通话记录，他同意了；女朋友要求提供微信密码和登录验证码以便不时抽查，他也照做了。

他还需要说明每日花费去向，偶尔记不清便会引发争吵。这已不是他一个人的问题。

每个人对安全感的需求不同。有的人更自信、更放松，对安全感的要求就不那么具体和严苛。而有的人需要了解更多，必须对伴侣的信息了如指掌。

即使伴侣将所有人生规划告知后者，也未必能满足其要求。因此，安全感不仅仅是给予量多少的问题，更不是通过逼迫对方回答隐私问题来索取。

在讨论亲密关系中的安全感前，老师需要先说明一个不利因素。

首先告知大家一个坏消息：你在亲密关系中的安全感风格并非轻易可以改变。你的嫉妒、纠结，甚至对长期关系的恐惧，很大程度上源于童年经历。安全感也是童年经历的直接反映。

这与你的依恋风格密切相关，你对安全感的需求直接关联到你的依恋风格类型。

一项研究指出，一批出生于70年代末的婴儿，在孩童时期接受了安斯沃斯的“陌生情境测试”，用以评估他们的依恋风格。心理学家根据婴儿的表现进行了分类。

二十年后，研究者再次找到这些已成为青年的受试者，针对他们在爱情中的依恋类型进行了重新评估和定义。

实验表明，对安全感的需求和态度完全未变的人，占到了参与者总数的**72%**。

研究还发现，影响爱情中安全感的因素主要有三个：
1.  你的**基因**。
2.  最初为你提供安全感的人（通常是父母）对你的回应方式。
3.  环境因素，例如成长过程中是否经历重大变故，如家庭暴力或父母离异。

然而这些影响已经发生，无法改变。但你可以进行自我评估，了解自己属于哪种安全感类型。之后，老师也会讲解提升安全感的小技巧。

除了上述影响因素，依恋风格主要分为三种类型。

当一个人对外界评价积极，但自我评价消极，且经常焦虑时，就属于常见的**取悦型恋人**。这类恋人可能显得自卑、唯唯诺诺，做事小心翼翼。他们通常是“粘人型”恋人。

他们总是希望时刻与另一半在一起，时常缺乏安全感，但只要稍加关注，内心的不安便能快速缓解。对他们而言，拒绝是最大的伤害。

例如“妈宝女”类型。她们可能会做出一些看似荒唐的行为，比如故意刺激伴侣说“某个男生对我很感兴趣”。这么做的目的是刺激对方。

她们希望看到对方为此着急，通过引发对方的这种情绪反应，来获得安全感以及对自身的积极评价。这就是取悦型恋人的特点。

对于这类恋人，老师的建议很简单：**不要试图在爱情中寻找自信。首先要在生活中找到自我价值。如果你带着自我否定投身爱情，那么关系不出问题是很困难的。**
