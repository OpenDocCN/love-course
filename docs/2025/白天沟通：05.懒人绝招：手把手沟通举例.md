# 白天沟通：05.懒人绝招：手把手沟通举例

懒人终极沟通法。如果看到这节课，你依旧觉得沟通好难啊。那些好听的话，我就是说不出口。我的嘴巴里面就是蹦不出那样暖心的话，我不是那样的性格，对吧？我不是那样的人，我张不开嘴。好，朱老师就教你一个终极绝招。

你不用去改变任何你说话的方式和模式，你依旧还是想说什么就说什么，但是在说话的时候做到一个点，那就保持好语气加笑容，举个例子啊，把你怎么又在客厅抽烟了，你去阳台范成，你怎么又在客厅抽烟了，去阳台嘛。爸。

今天能不能带孩子做作业？换成今天能不能陪孩子写作业？怎么样？这个终极方法简不简单，就是把狠话软说对吧？硬话软说狠话温柔的说，可千万别再说啊，做不到啦。即便不是为了大猪蹄子，你为了自己的心理健康。

身体健康也要做到好不好？生气，对自己一点好处都没有。所以温柔一点，心态和平一点，人生嘛，只不过一张3万多天的体验券，自己开心才是最重要的。有效沟通话术举例。当他压力大的时候，可以这么说。

因为我知道为了这个家，你最近压力很大，一直以来你都很有上进心，希望能够做到最好，不让我和孩子受委屈，对不对？只不过呢任何事情啊都有一个过程都会遇到挫折。你看古往今来的伟人，哪有一下子成功的呢。

你已经做的很棒了，接下来只要调整好心态和方向，我相信你一定会度过难关。当他送错礼物，哎，你可以这么说。哇，你送的这个礼物一定准备了不少功课吧，颜色好少女心啊，而且是我喜欢的款式，我好喜欢，真的好开心。

比我以前我自己买的还要开心。可惜我已经有同款了，不知道什么时候才能用上你送的，要不这样吧，你去换一个颜色，这样呢我就立马可以用新的了，然后跟你去约会好不好？当它没有边界感，是一个暖男中央空调。哎。

你可以这么说。你是一个很温柔很善良的人，还记得以前我生病的时候，你特地请假来照顾我，我真的好感动啊。但是我希望你的好是我的专属，就像我会避免和异性单独接触，怕你误会。所以亲爱的我知道你是一个热于助人。
