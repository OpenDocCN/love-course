# 谈催年课：16：脐轮3+骆驼式

在本节课中，我们将继续探索第三脉轮——脐轮的特质，并学习一个与之相关的经典瑜伽体式：骆驼式。我们将深入了解意志力、力量、大笑与自信这四大脐轮主题，并掌握如何通过具体练习来增强这些品质。

## 脐轮与意志力

上一节我们介绍了脐轮的基本概念，本节中我们来看看脐轮的第一个核心特质：意志力。

意志力，是指我们有意识地将能量导向特定目标以实现目的所动用的力量。当能量有了特定方向，就会形成力。公式可以表示为：**意志力 = 能量 × 有意识的方向**。

欲望机制则倾向于让我们选择最愉悦、最少挑战的事情，如下班后刷手机、吃零食。这源于大脑中古老的无意识程序。意志力是比欲望机制更强大的机制，它让我们能基于主观选择和目标法则来行动。

伟大的灵魂具有意志力，软弱的灵魂只有欲望。著名的棉花糖实验表明，能延迟满足的孩子未来更易成功。因此，强大的意志力导向成就与快乐的人生，而薄弱的意志力被欲望驱使，则易导致痛苦与失败。

我们都有缺乏意志力的时刻。关键在于接纳当下，而非陷入自责。罪恶感只会降低能量，让我们更麻木。正确的做法是：接纳自己曾被欲望驱使，然后负起责任，在下一次用意志力做出更有益的选择。

小我会将欲望的满足误认为自由。真正的自由并非满足无数欲望，而是“不想做什么时，就可以不做什么”。这需要透过意志力，遵循更高价值，在极度自律中获得。

## 无法建立意志力的错误方法

在探讨如何正确建立意志力之前，我们先来看看哪些做法是无效的。

以下是几种无法真正建立意志力的方法：
1.  **依赖罪恶感与自责**：这只会降低能量，削弱行动力。
2.  **仅为外在奖励（如金钱）行动**：这只能创造一点点行动力，并非由内在火元素驱动。
3.  **依赖胡萝卜加大棒的奖罚机制**：无论是训练动物还是教育孩子，这种方法只会将行为与外在奖罚挂钩，无法培养出真正的内在意志力。

## 增强意志力的五大方法

那么，如何用正确的方法建立起我们的意志力呢？以下是五个简单有效的方法。

1.  **立刻行动，坚持微小习惯**：想到就去做。从极其简单的行动开始，如每天早晨掰断一根火柴或打开窗户，坚持执行。行动会自然增强意志力。当你做真正热爱之事时，甚至无需“坚持”。
2.  **设定可达成的目标**：避免设立根本做不到的目标，失败会削弱意志力。从每天10分钟专注练习开始，坚持二三十天后，意志力增强，再逐步增加难度。
3.  **激活脐轮**：这是增强意志力最直接的方法。只要练习脐轮（无论是体式还是冥想），意志力就会增长。将已建立的意志力再投资于脐轮练习，会形成滚雪球般的指数增长。
4.  **牺牲低层次满足，追求高层次愿景**：公式是：**牺牲低层次欲望 → 满足高层次愿景 = 增强意志力**。例如，为健康牺牲甜食，为关心他人牺牲刷手机的时间，为谦卑牺牲自我吹嘘。反之则会削弱意志力。
5.  **与宇宙意志力共振**：意志力是一种宇宙性能量。通过静心，将自己调频到充满意志力的状态，或与太阳、宇宙的意志共鸣，即可获得这份力量。

意志力是实现成长与转化的必备条件。将所学知识与意志力结合，才能真正改变人生。

## 日常提升意志力的12个练习

理解了核心方法后，以下是一些可以在日常生活中具体实践的练习，帮助你检验和提升意志力。

1.  **战胜拖延**：想到某事（如打扫、打电话），立刻行动，不等待。
2.  **克服惰性**：当感到“废了”时，立刻起身做5分钟“山式站立”或“站立前屈”，消除惰性，恢复活力。
3.  **锻炼不犹豫**：决定运动（散步、跑步）时，立刻执行，不犹豫。
4.  **戒断挑战**：针对巧克力、烟、酒等，尝试一周完全不接触，以建立控制感。
5.  **慎言**：想说闲话或废话时，保持静默。只说真正来自内心、有价值的话。
6.  **替代娱乐**：将刷八卦、购物视频的时间，替换为看有启发的纪录片、演讲等。
7.  **健康饮食替代**：想吃垃圾食品时，用新鲜水果、黄瓜替代。
8.  **转念**：消极念头出现时，立刻用积极肯定语替代，如“我是宇宙中最闪闪发光的”。
9.  **不延迟**：一日之事，想到就做，克服懒惰。
10. **性能量控制**：运用课程方法控制性能量，并保持觉知。
11. **暂停社交**：尝试一段时间（几小时到一天）完全停止社交活动，不看手机，体会内在力量的提升。
12. **规律作息**：只在固定时间睡觉，其他时间不打盹，保持清醒。

你可以选择其中几条开始践行。

## 脐轮与力量

拥有了意志力这项工具后，我们自然就获得了力量。当我们认识到自己可以成为人生的导演时，力量感便油然而生。

力量来源于意识与能量的结合。**力量 = 意识 + 能量**。当我们将带有意识的能量持续导向某个特定目标（如每天6点起床），能量就会在该处累积，形成习惯，也就是力量。

缺乏力量时，我们易被他人力量控制，进入第二脉轮寻找领导者。脐轮唤醒者通常自律、有能力，能坚持日常安排，如巴尔扎克般持续创作。

拥有更大力量时，需以智慧、道德和觉知力引导，确保力量用于正向。

## 脐轮与大笑

现在，我们来看脐轮的第三个特质：大笑。这与我们即将练习的骆驼式也有奇妙的关联。

大笑是一种特殊的呼吸模式，涉及全身超过600块肌肉，尤其是横膈膜和腹肌的剧烈震动。它能释放情绪与能量。

大笑与幽默的本质，是看待问题视角的突然转变，旧有视角附着的能量被释放。例如，看到严肃总统穿着滑稽服装，或惊吓后发现是恶作剧，紧张感瞬间释放为大笑。

这种“紧绷-释放”的模式是生命的张力，类似性高潮和分娩的收缩与扩张。肠道消化也依靠这种律动。很少笑的人可能较为刻板固执，缺乏生命的自然张力。

## 脐轮与自信

最后，我们来探讨脐轮的第四个特质：自信。自信就是相信自己的能力或特质。

自信有四个发展阶段：
1.  **零自信**：完全缺乏自信，回避社交，人生灰暗。
2.  **虚假自信/吹嘘**：表现得很自信，但内在虚弱，需要外力维持。
3.  **基于能力的真实自信**：因实际具备某些能力（如驾驶技术、所学知识）而产生的健康自信。
4.  **基于谦卑与自我认知的自信**：完全看清自己的优点与缺点，不偏不倚地接纳自己。如圣雄甘地，认识并转化自身的暴力。此阶段不受外界评价影响。
5.  **存在的自信**：超越所有物质属性（身体、念头、社会地位），单纯“存在”的状态。在深度冥想中可能体验到，是一种永恒、不被动的自信，带来巨大自由。

我们应朝向第三、四阶段的自信努力。

## 体式练习：骆驼式

了解了脐轮的四大特质后，我们将通过骆驼式这个强烈的体式，来整合和强化这些能量。

**做法（简易版与进阶版）**：
*   跪立于垫子，双膝分开与骨盆同宽。
*   **简易版**：脚掌回勾，踩地（垫高脚跟，降低难度）。
*   **进阶版**：脚背平铺压地（难度更高）。
*   双手扶髋，骨盆向前推，打开腹股沟，胸腔上提，脊柱后弯。**注意**：后弯主要来自胸椎伸展和骨盆前推，而非腰部。
*   头部后仰（甲状腺疾病者，如甲亢，请保持下巴找锁骨，勿抬头）。
*   双手自然向后，尝试抓握脚踝。若抓不到，可借助椅子或瑜伽砖。
*   保持十几秒到三分钟。若无法一次完成，可分多次进行。
*   **退出**：胸腔带领身体慢慢回正，以跪立姿势做**觉知**（可臀部坐于脚跟上）。这是少数采用跪姿做觉知的体式，便于观察下背部能量变化。

**能量流动**：
大地能量沿双腿上行至脐轮，再经双手回流至双腿，形成一个能量环路。

**功效**：
1.  强力打开胸腔，强化背部肌群，为长时间冥想奠定身体基础。
2.  平衡身体温度，感冒时练习可助康复。
3.  唤醒意志力、勇气与毅力。
4.  打开手臂能量管道，增强手臂的疗愈力量（适合疗愈师或需手臂工作者练习）。

骆驼式是强化背部、准备冥想的最佳体式之一。

## 总结

本节课中，我们一起深入学习了脐轮的四大核心特质：**意志力**、**力量**、**大笑**与**自信**。我们明白了意志力是引导能量实现目标的关键，并掌握了增强意志力的具体方法。同时，我们也认识到真正力量源于意识与能量的结合，大笑是能量的释放与视角的转换，而自信的最高阶段是超越外在的自我认知与存在。

最后，我们学习了**骆驼式**这一强大体式，它不仅能强化身体，更能整合与激活脐轮的火元素能量，帮助我们将这些理论落实到实践中。请记住，将知识与持续的练习、意志力相结合，才能真正乘上脐轮的火箭，实现生命的蜕变与成长。
