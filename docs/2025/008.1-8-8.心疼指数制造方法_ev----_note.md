# 情感课程：008.1-8：心疼指数制造方法

在本节课中，我们将学习如何通过“暴露敏感面”来制造心疼指数，从而构建更扎实、更立体的吸引力。

上一节我们介绍了吸引力模型中的心动指数，本节中我们来看看与之相辅相成的另一个重要概念——心疼指数。制造心疼指数，核心在于寻找合适的机会，在平时展现自信、阳光、勇敢等正面印象之外，偶尔向对方袒露我们内心脆弱、无助或伤感的一面。

简单来说，如果我们与女生相处时，90%的时间都在谈笑风生，那么就需要寻找或制造那10%的时间，让对方看到我们“在哭”。这样，我们的性格才会变得立体，吸引力才会更加扎实。这不仅能让她有女儿般崇拜和依赖我们的感觉，也能偶尔让她产生妈妈般疼爱和照顾我们的感觉，让她感到被我们需要。

当然，这种内心的敏感面无需编造。每个人在成长或工作中，都曾经历过委屈或瓶颈。我们可以从自己真实的人生经历中，选取一两件事进行适当的表达加工。关键在于，在重点表述时一定要带有真实的情感，才能达到预期的效果。

以下是暴露敏感面的一个核心行动指南：

1.  **寻找真实素材**：从过往的委屈、当前的工作瓶颈或与父母的深层情感联结中寻找素材。
2.  **进行适度加工**：对真实经历进行梳理和表达上的润色，但核心情感必须真实。
3.  **选择合适时机**：在双方有一定信任基础后，自然地将话题引向深层感受。
4.  **流露真实情感**：讲述时，允许自己流露出相应的脆弱或伤感情绪。
5.  **保持积极内核**：在暴露脆弱后，可以展现出现已走出或正在积极面对的态度。

我们来看一个影视片段中的例子：

> “我父母很早就过世了，所以我知道，靠自己最实在。”

这段对话在展现角色坚强（“靠自己”）的同时，也暴露了他内心的孤独（“父母早逝”），容易引发听者的心疼感。

在实践教学中，常有学员在心态训练或一对一交流时落泪，这本身就是一种有效的敏感面暴露。当一个人彻底敞开心扉后，更容易在对方心中留下深刻印象，并激发对方想要帮助和保护他的感觉。

许多深刻的敏感面素材，往往与童年经历或和父母的情感有关，这是我们情绪的重要来源。如果你不知如何入手，不妨从这些方面进行思考。

同时，子明也建议，当我们为了人际吸引而回顾这些尘封往事或与父母的矛盾时，也应借此机会，以成熟、担当、包容的态度，主动去处理和修复与父母的关系。无论过去如何，父母也并非故意，因为他们同样没有学习过如何做父母。

因此，**暴露敏感面**在人际吸引中是一个小的收获。而真正修复内心敏感面、改善与父母的关系，使自己变得真正强大和成熟，才是最大的收获。请记住：我们与母亲的关系，将影响一生与所有女性的关系远近；我们与父亲的关系，则将影响一生与所有男性的关系远近。

本节课中我们一起学习了如何通过暴露真实敏感面来制造心疼指数。关键在于从真实经历中提取素材，在合适时机融入真情实感进行表达，这能使你的吸引力层次更丰富、根基更扎实。
