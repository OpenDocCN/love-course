# 恋爱成长课：13：如何避免在恋爱里单方面付出？

在本节课中，我们将要学习如何识别并避免在长期恋爱关系中陷入单方面付出的困境。我们将探讨单方面付出的心理机制、其与关系中权力平衡的关联，并学习通过有效沟通和谈判来引导关系走向更健康的双向付出模式。

## 为什么会单方面付出？

上一节我们介绍了课程的主题，本节中我们来看看单方面付出的心理根源。付出是一个主观概念，通常体现在时间、精力、金钱和感情四个方面。当两个人的付出水平产生肉眼可见的失衡时，就构成了单方面付出。

那么，为什么我们会持续做一件明显“不合算”的事情呢？口语上的原因可能是“太喜欢了”、“乐于助人”或“傻”。但深究起来，单方面付出的人在潜意识里认为这么做“有效”。他们认为，只要持续付出（无论是时间、精力、感情还是金钱），就能让人看到关系朝着正向发展的可能性。换句话说，对一个人好，比起不对他好，看起来更可能“成功”。

然而，这种“有效”往往只是暂时的，是对方给予的某种短暂反馈。从宏观和长期来看，付出本身并不能直接导向一个好的结果。最终是否有用，还取决于一个至关重要的因素：两个人在关系中的权力是否对等。

## 权力失衡与付出的恶性循环

上一节我们提到，付出的有效性取决于权力对等。本节中我们来深入探讨权力与付出如何相互影响。付出本质上是一种寻求合作的行为。当双方权力不对等时，就变成了单方面寻求合作，这种“合作”很容易演变成一种压榨。

问题在于：我们不断付出，对方就不断勉强地、暂时地“合作”一下。这种勉强会进一步加剧权力的不对等。一开始的权力不对等会造成付出的不对等（总有一方付出更多）。反过来，付出的不对等也会加剧权力的不对等，因为很现实的一点是：对方接受我们的付出久了，他的期望值会改变，我们付出能带来的效果是**边际递减**的。

例如，如果单靠花钱去追求一个人，结果必然是需要花越来越多的钱，才能让对方的好感维持在早期较好的状态。同样，很多人在追求过程中过度体贴、妥协，结果对方却得寸进尺。这就是人性：我们越是无底线地付出和妥协，越容易让对方不自觉践踏我们所剩无几的底线。

这种长期的妥协和付出，会让付出者的心态彻底崩盘。一方面，付出似乎能带来微薄的希望，于是我们不断付出以维持它；如果效果变差，就只能加倍付出。这就形成了一个**死循环**。付出者会变得无比挣扎，因为我们的付出并非完全不求回报，我们至少希望对方能念着这份好，并对我们也好一点。然而，事与愿违，最终往往产生怨气甚至恨意。

过度的单方面付出就像一个一直被吹打的气球，终会爆炸。这解释了为什么“越懂事越不被珍惜”或“对他越好，他越作”。不是“懂事”导致了“不珍惜”，而是当一个人仅通过懂事、体贴、忍让等行为作为付出方式，并试图以此换取对方好感、让自己留在关系里时，才会导致不被珍惜。

## 属性与行为：关系的两大支柱

当我们跳出“死局”思维后，会有更好的思路。我们常听到一种说法：“好的感情不是追来的，而是吸引来的。”这句话很有道理，因为它映射了两个核心概念：**行为**和**属性**。

*   **行为** 是 **`doing`**，是我们“做什么”，例如嘘寒问暖、花钱、提供服务等。
*   **属性** 是 **`being`**，是我们“是谁”，例如长相、学历、赚钱能力、幽默感等。

你会发现，所谓的单方面付出几乎全部落在 **`行为`** 上。这就是问题所在。在一段健康的关系中，属性和行为都重要。我们自身的好（属性）和我们对对方的好（行为）都对关系有益，且应互相平衡。

*   仅有行为上的付出，没有属性的加持，是不堪一击的。
*   仅有属性上的展示，没有行为上的付出（只索取不回报），对方也会因单方面付出而心态失衡，关系同样会破裂。

因此，将精力过度倾斜在属性或行为的任何一方，都对长期关系不利。

说到这里，关于“属性”的概念，可以回答另一个关键问题：为什么人们总说“门当户对”？门当户对说的正是**属性**上的大抵相等。这相当于两个人在关系中的权力大抵对等。我们之前说过，权力不对等会导致付出不对等。所以反过来，权力如果对等，付出就更容易对等，也就不容易产生单方面付出了。从原理上看，门当户对是有道理的。

## 如何引导对方付出：谈判的艺术

那么，在已经产生付出不平衡，且短期内无法快速提升属性以实现权力完全平衡时，我们能否通过沟通来获得付出与回报的平衡呢？首先，排除“切断自我付出”这条思路，虽然它能实现平衡，但很可能导致关系迅速冷淡和疏远。

剩下的思路就是：**引导对方付出**。这本质上是一种**谈判**。也许有人会觉得在恋爱中“谈判”很别扭，认为开口索要不太好。这种纠结源于我们把**立场**和**利益**混为一谈。

*   **利益** 是我们具体的**目标或需求**（我到底想要什么？）。
*   **立场** 是我们对一件事的**态度或看法**（我该怎么提？会不会显得自私？）。

例如，我们希望对方多付出时间陪伴（**利益**），但可能因为怕显得“粘人”或“不体谅”（**立场**）而不敢开口。当利益和立场冲突时，原则上我们应该**关注利益，而非立场**。关注利益能让双方更真实、清晰地了解彼此的需求，这反而可能成为沟通的捷径。谈判的目的不是输赢，而是**拿到双方都想要的东西**。陷入立场就会陷入输赢思维；关注利益则能减少内耗。这是思维方式上的核心转变。

转变思维后，实际操作就是开启谈判：**把我们的需求和对方的利益统一起来，画上等号**。简单说，让他知道，他为我们付出，不是他亏了，反而对他自己有好处。

以下是具体操作时可能遇到的两个问题及应对思路：

1.  **问题一：没有表达清楚对方的利益。**
    *   **应对**：需要更加具体地描述对方的利益。例如，不说“你会有面子”，而说“我打扮得漂亮，你朋友都会羡慕你，你吹牛都更自信”。

2.  **问题二：没有切准对方的利益。**
    *   **应对**：我们的需求可能和对方的核心利益矛盾。这时不应在错误的方向做无用功，而应去探索对方真正的核心利益，并尝试将我们的需求与之关联。这要求我们足够了解对方，并在沟通中随机应变。

**核心思路始终是：尝试将“我们的需求”与“对方的利益”画上等号。**

## 谈判策略：设定目标与应对失败

如果我们已经站在对方利益的角度去谈，但对方仍固执己见怎么办？我们应该学习谈判专家，在谈判前做好最坏的打算，并设定清晰的谈判目标。

目标应有主次之分：
*   **主要目标**：最基本的、一定要争取到的东西（例如，获得穿衣自由、被尊重的权利）。
*   **次要目标**：主要目标之上的锦上添花（例如，对方为此买单或提供建议）。

谈判时，要保证主要目标，并尽量实现次要目标。如果谈不拢，在必要时可以选择**通过放弃次要目标来捍卫主要目标**。这是一种策略性让步，也符合心理学上的**互惠原则**——人们接受恩惠或让步后，会本能地想要回馈。例如，先提一个较高的要求（次要目标），被拒绝后再提出真正的主要目标，对方可能更容易答应。

如果连主要目标都无法达成，我们必须接受谈判失败。而最坏的情况是：不仅目标未达成，对方还在试图击穿我们的**底线**。底线是无论如何都不能妥协的部分。如果对方越过了底线，我们应当有离开谈判桌（即关系）的能力和勇气。因为在一段长期关系里，失去底线必然导致单方面付出和关系崩塌的死循环。

## 总结

本节课中我们一起学习了如何避免在恋爱中单方面付出：

1.  **单方面付出的根源**：付出者潜意识认为付出“有效”，能推动关系发展，但其长期有效性取决于关系中权力的对等。
2.  **恶性循环**：权力不对等导致付出不对等，付出不对等又加剧权力不对等，形成死循环，最终导致付出者心态崩溃。
3.  **属性与行为的平衡**：健康关系需要属性（`being`）和行为（`doing`）的平衡。单方面付出往往过度侧重于行为而缺乏属性支撑。
4.  **引导付出的方法**：通过谈判引导对方付出，核心是**谈利益而非立场**，将我们的需求与对方的利益统一起来。
5.  **谈判策略**：设定主要目标和次要目标，必要时放弃次要目标以捍卫主要目标。坚守底线，在对方击穿底线时勇于离开。
6.  **健康关系的愿景**：好的关系是正向循环——我们付出越多，对方也付出越多，彼此善待，关系越来越好。

坚守底线、寻求平衡、要求双方共同付出，这不是自私或计较，而是让关系健康长久走下去的基石。单方面付出就像一边翘起的跷跷板，没人能一直玩下去。
