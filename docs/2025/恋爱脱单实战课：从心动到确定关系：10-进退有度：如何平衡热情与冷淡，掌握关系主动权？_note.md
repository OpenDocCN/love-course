# 恋爱脱单实战课：10：进退有度：掌握关系主动权

在本节课中，我们将要学习如何在恋爱关系的“拉扯期”中，通过平衡“进”与“退”的行为，来掌握关系的主动权，实现权力平衡，从而推动关系健康、稳定地发展。

## 为什么爱情中会有博弈？🤔

上一节我们介绍了关系发展的不同阶段，本节中我们来看看“拉扯期”的核心特征。你可能会对以下情况感到熟悉：

以下是几种常见的情感状态：
*   你总是主动发信息，而对方却不主动找你。
*   送出礼物后，因对方没有回馈而感到失落。
*   对对方行为不满，却不明说，而是故意冷淡。
*   无意间将对方与潜在对象做对比。
*   吵架后想和好，却要等对方先开口。
*   在对方主动时，心里评估其诚意再决定是否投入。

这些思绪很日常，上升不到好坏评判，但却是两人间最真实的、暗地里的互动。将“恋爱”和“博弈”放在一起似乎不搭，因为博弈常让人联想到谈判或竞赛，显得冷酷。我们总希望爱情更纯粹美好。

那么，为什么情感中也会出现博弈？因为尽管披着浪漫滤镜，但只要是两人关系，就会存在**权力**因素。双方的情感投入和需求不可能始终完全同步，加上个人选择权大小、性格（如慢热或快速投入、独立或依赖）的差异，形成了**情感投入的不对称**。

这种不对称，总会产生权力较高的一方和权力较低的一方。如果两个人都想做权力高的人，博弈就出现了。

## 权力流动与连续博弈 🔄

我们明确了爱情中存在博弈，但这不解决关键问题。关键在于，当我们喜欢一个人并作为主动方时，通常处于权力较低的位置。但好在，**权力关系是可以变化的**。

随着互动、了解和投入增多，权力会在两人之间游走或流动。这个概念极其重要，它能解释很多现象。例如，有些人（尤其是女性）在关系初期明明是被追求方，拥有主动权，但随着情感发展，自己却变得越来越被动、在意对方。这背后的本质就是**权力转移**。

因为恋爱是一种**连续博弈**，而非一锤定音的单次博弈，这才有了许多柔性空间来帮助我们获得权力平衡。

**单次博弈**就像在路边摊买西瓜砍价：“5块不卖我就走。”你一走，摊主妥协。这种一锤子买卖是谁狠谁获利。
**连续博弈**则像长期在一个摊位买西瓜，双方形成默契，这次我让你五毛，下次你给我挑个熟瓜。大家着眼于长期关系。

在爱情中也是如此。将行为拉长到一段时间看，每一次对话、每一次赴约，都成为连续博弈的一部分，并直接或间接地影响后续关系中的权力分配。

买西瓜的例子恰好证明，**合作与博弈并不冲突**。长期买瓜是合作，但合作之下依然存在博弈（如议价）。感情也一样，两人间可以有博弈成分，但不意味着要把对方当敌人去争输赢。**我们的目的是实现权力平衡**，也只有势均力敌的关系，才可能出现稳固的合作。因此，**权力平衡是合作的基础**。

我们称当前阶段为“拉扯期”，正是因为双方都在努力争取主动权。这个时期可长可短，有人甚至能跳过直接进入确定期。如果进入了拉扯期，我们可以通过两件事来促成权力平衡：
1.  **掌握进退的尺度**（外在行为策略）。
2.  **管理自身需求感**（内在心态控制）。

这两件事目的相同，但着眼点不同。进退方式是通过“做什么”来激发对方情绪波动；需求感管理是通过“不做什么”来控制自身情绪和心态。两者相辅相成。本节课我们重点讲解第一点：进退的方式与尺度。

## 如何平衡“进”与“退”？⚖️

首先，快速定义“进”与“退”。
*   **进**：主动发消息、邀约、表白、送礼物、进行肢体接触等，是我们主动向前。
*   **退**：延迟回消息、拒绝邀约、刻意保持距离、回避肢体接触、对表白表示需要考虑等，是我们主动向后。

权力平衡的关键，在于**将进与退结合**。

如果**只进不退**会怎样？公式如下：
```
不断单方面主动 = 快速、剧烈地让渡权力给对方
```
例如，每天发上百条消息对方不回还继续发，没有理由地不断邀约表白。这会导致自己最终无权力可让，关系悬空。

如果**只退不进**会怎样？公式如下：
```
持续冷淡与拒绝 = 引发对方厌恶并可能彻底放弃
```
例如，总不回消息、冷落对方、各种拒绝挑剔。换位思考，对方也会产生负面情绪。当你退得太远，对方可能彻底放弃，关系便无法维持。

那么，怎样才算有效的结合？我们通过一个故事，用四个场景来说明。

### 场景一：邀约的进退

假设小明和小红互有好感，但未挑明。小明约小红隔天晚餐（这是一个“进”）。小红答应了，但下午突然发消息说临时有事来不了（这是一个“退”）。无论原因如何，这个“退”会让小明产生期待落空的失落感。

紧接着，小红又发消息：“不好意思，怪我。明天给你带早餐作为补偿吧。”（这是一个“进”）。这会让小明心情瞬间由阴转晴，觉得“我还有戏”。“有戏”这个词背后，意味着小明正在被拉扯，经历情感的投入和起伏。

抛开动机，小红完成了一次 **“先退再进”** 。如果小明对小红有意思，这种“失而复得”的体验，会让他对小红投入更多注意力和更高期待。

同理，在小明主动邀约（进）后，他也可以主动制造“退”的效果。例如，他可以说：“明天晚上我可能还有些工作，吃完饭得早点走。”这句话透露的信息是：他并非完全围着小红转，他有比约会更重要的事。这同样是一个 **“先进再退”** 的组合。

### 场景二：微信聊天的张力

为什么回消息快慢这件小事让人纠结？因为它背后是注意力投入和权力高低的体现。权力高的人不一定回得慢，但他**可以选择**回得慢。回得慢这个行为本身，就印证了他的权力较高。

接上例，小明在约会后一整晚都不主动发消息（这是一个“退”），会让小红产生期待、焦虑甚至一点小生气。注意，这种“小生气”意味着小红在投入更多情感和注意力。但此时“退”的弦不能崩断，继续断联会恶化情绪。

所以，退完之后，小明必须“进”。他可以在第二天用亲密的口气发语音说：“我昨天忙到凌晨3点，一看时间太晚了，就没和你发消息。但是我梦到你了。”小红看到后，情绪会经历又一次起伏，两人距离再次拉近。

### 场景三：约会中的意外之喜

第二次约会结束，小明送小红回家。在分别时，小红主动亲了小明一下，然后转身跑开（这是一个强烈的“进”后立刻“退”）。

在小明的视角，他会先错愕，随即被惊喜和兴奋取代，并开始浮想联翩。他获得了一种**不确定的奖励**，这促使他对小红投入更多情感和注意力。如果没有这个“进-退”组合制造张力，约会结束便平淡无奇。

### 场景四：模糊表达推进关系

小红的行为让小明确认了两点：1. 小红对他有好感；2. 小红可能在催促他表白。如果小明不想贸然直接表白，他可以利用暧昧技巧。

小明回去后给小红发消息：“你刚才这样太犯规了，搞得我回去一路上都在想你。”这是一种**把喜欢藏在模糊表达里**的方式，越让人猜不透，对方就越容易“上头”。只要后续不出错，关系便能水到渠成。

通过以上四个场景的几轮拉扯，我们说明了进退结合的方法。这些情节基于真实案例，仅为理解而安排得更为紧凑。这种拉扯在不同关系中存在时间不同，可能长达一两个月（如异地恋），也可能像案例中一样快速推进。

**关键在于，进和退一定要结合**。没有固定顺序，先退再进或先进再退都可以。其原理在于制造 **“不确定的奖励”** ，这是诱发人类持续投入的重要心理机制。公式化表达即：
```
不确定的奖励 -> 引发渴望 -> 促使投入更多精力确认自身位置
```
在拉扯中，两人的关系实际上是在拉近的。当双方都互相交付心意时，拉扯期便宣告结束。

## 课程总结 📝

本节课我们一起学习了如何在恋爱拉扯期中通过平衡“进”与“退”来掌握主动权。

1.  **博弈的根源**：爱情是一种权力关系，因情感投入不对称，会产生权力高低之分。若双方都想占据高位，便会出现博弈。
2.  **动态平衡的可能**：权力关系并非一成不变，而是会流动的。恋爱是一种**连续博弈**，博弈与合作并不冲突，**权力平衡是健康合作的基础**。
3.  **达成平衡的方法（一）**：掌握**进退的尺度**。核心在于将“进”（主动、热情）与“退”（矜持、冷淡）有机结合，避免单一极端。通过制造“不确定的奖励”，引发对方情绪投入，从而在动态中调整权力天平。

我们通过小明和小红的故事，在邀约、聊天、约会、表白四个场景中具体演示了如何实践进退结合的策略。

记住，我们这么做的最终目的，不是为了“赢”过对方，而是为了促成双方权力的对等与平衡，为关系的长远健康发展打下坚实基础。下一节课，我们将学习达成权力平衡的另一个关键：内在的**需求感管理**。掌握了这一内一外两点，你就掌握了拉扯期平衡权力的全部要义。
