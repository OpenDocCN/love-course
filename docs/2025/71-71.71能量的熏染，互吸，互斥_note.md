# 动车网谐淫系统自救指南：71：能量场的熏染、互吸与互斥

在本节课中，我们将要学习能量思维层次的核心概念。我们将探讨什么是能量场，以及它如何通过熏染、互吸和互斥的原理，深刻地影响我们的思想、行为乃至命运。理解这些原理，是改变自身能量状态、摆脱负面习惯的关键。

## 什么是能量场

上一节我们介绍了意识、无意识和阴影三个层次，本节中我们来看看更高维的能量层次。

能量，即一个人身上的磁场。有的人让人感觉如沐春风，相处舒适；有的人则让人不愿打交道。这源于每个人身上气场不同，能量形态各异。一个人沉迷于谐淫，正是因为他的身心携带了一个谐淫的能量场。这种能量场是阴暗、负面的，形态如同一个充满污秽的臭水沟。

## 能量场的五个主要特点

以下是能量场的五个核心特点，理解它们有助于我们认识并改变自身的能量状态。

### 1. 能量场的无形性

能量场无形无相，看不见摸不着，却真实存在。它很难被自身意识到，如同在水中却感觉不到水的存在。这种能量场会悄无声息地通过无意识平台影响人的意识和行为。

当能量场影响一个人时，此人往往未知未觉。例如：
*   一个自私的人，其内心是阴暗狭隘的能量形态。他受此能量场影响做出自私行为，却感受不到该能量场的存在，也意识不到自己的自私。
*   一个人喜欢刷手机看垃圾信息和短视频，是因为他的身心能量场如同臭水沟。他无形中受此影响，故而喜欢待在对应的信息环境中。当他进行垃圾娱乐时，往往意识不到这是内心能量场惯性（也称习气或业力）驱动的结果。

### 2. 能量场的沉淀

一个人身上的能量场通过重复的思想和行为沉淀形成。一种念头或行为不断重复，日积月累，便形成了一种底蕴。这种底蕴就是一个人的“魂”，即其能量场。

例如，谐淫搞多了，整个人会显现出猥琐气息，这就是能量场沉淀的结果。沉淀过程如同走路会在地上留下脚印。我们日常的思想和行为也会在心灵上留下痕迹。痕迹越积越多，便形成底蕴，即自身的能量场。

**身病源于心病，心病源于神病，神病源于灵病，灵病源于生命在很长一段时间里，扭曲阴邪的思想和行为不断重复、沉淀积累，直到形成厚厚的一层底蕴——人的能量场。**

环境是内心能量场的投影，内心能量场是终极投影源。改变终极投影源，外在世界自然改变。如果内在能量场阴暗污秽，那么投影到现实就必然是阴暗污秽的环境与境遇。所谓有其内必有其外。

**能量场操控无意识，无意识操控意识，意识操控行为，行为决定命运。** 这是从无形到有形、一环扣一环的决定关系。反过来，行为和意识不断沉淀形成无意识，无意识继续沉淀形成能量场。这是从浅到深的沉淀关系。这揭示了能量场与一个人无意识、意识、行为及命运的深层联系。

### 3. 能量场的互吸（同频相吸）

相同频率的能量场互相靠近时会互相吸引，如同磁铁相吸。

*   一个人走在路上看到网吧或电竞酒店就想进去，是因为其内心能量场与网吧游戏的能量场同频，互相吸引。这叫同气相求。
*   一个人老是喜欢听伤感情歌，是因为其内心能量场与伤感情歌同频，互相吸引。
*   一个人总是喜欢谐淫，是因为其内心能量场与谐淫的能量场高度匹配，所以互相吸引。

### 4. 能量场的互斥（异频相斥）

如果能量场不在同一个频道上，就不会互相吸引，而是会互相排斥。

一个喜欢在网吧打游戏的人，对路上的书店毫无兴趣，因为两者能量场根本不在同一频道。所谓道不同不相为谋。

**这代表着一个重要结论：如果我们能改变自己内心的能量场，使其不再是谐淫的能量场，久而久之，就不会与谐淫的能量场发生共振，也就对谐淫不感兴趣。**

为什么神仙对邪淫不感兴趣？因为神仙的能量场是清净光明的，与邪淫的欲望阴暗能量场完全不同。所以神仙根本不需要戒色。

### 5. 能量场的熏染

腌菜在盐水里泡久了会变咸，这就是腌入味了。同样，一个人在某种能量场里熏染时间太长，也会“腌入味”。

*   一个人经常看黄片，就是一直沉浸在色情的能量场中。久而久之，他身上就会带有猥琐气息，这是被邪淫能量场“腌入味”了。
*   垃圾娱乐、玩手机等行为，是把身心放进一个垃圾污秽的信息能量场中不断熏染。
*   反之，如果一个人天天接触清净、阳光、光明的能量场，熏染积极正气的能量，久而久之，他身上的正气阳气也会越来越充足。所谓近朱者赤，近墨者黑。

## 如何改变能量场：从“臭水沟”到“清水”

一个谐淫者的内心能量场如同臭水沟。如何将其转变为清水？有以下两个方法：

1.  **悔过**：清除内心的负面能量与垃圾。这相当于把臭水沟里的垃圾杂物捞出扔掉，臭水沟会慢慢变得清澈。
2.  **熏染**：将身心置于清净光明的能量场中，天天不断熏染，接触正能量、阳光清净的气息。这相当于给臭水沟引入清水，用清水不断冲刷，使其慢慢变清澈。

谐淫者需用这两个方法改变能量场。但能量场改变不易，如同行驶的火车有强大惯性。这个惯性就是自身的业力，是过去生命积累的强烈习气，需要下大功夫去改变，不可能一蹴而就。不要着急，日积月累，每天读书、反省过错、反省无意识的阴影和心结，用儒释道的经典书籍冲刷内心垃圾，久而久之便能改变能量场。

**当能量场改变，就不会再执着于邪淫。有一句话叫“近道无魔”。当你离道越来越近，心中的魔性（贪婪、自私、狭隘、阴狠、怨恨、自我中心等负面能量）就会减少。如果一个人的能量场提升，内心如道一般博大无私、清净、虚无，就会对邪淫完全不感兴趣。**

我们再来看这个原理：当谐淫者将其能量场提升到“道”的层次，他就会对“臭水沟”完全不感兴趣。这就是能量不相容，即“近道无魔”。

## 总结

本节课中，我们一起学习了能量场的核心概念与五大特点：无形性、沉淀性、互吸性、互斥性和熏染性。我们明白了内心能量场是外在境遇的终极投影源，并通过“悔过”与“熏染”两个方法，探讨了如何将负面能量场转变为正面能量场。记住，改变能量场需要持之以恒的努力，当你的能量场提升，自然会对低频、负面的诱惑失去兴趣，走向更清净、光明的生命状态。
