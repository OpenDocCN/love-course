# 王者心态课程：1：战胜一切恐惧的王者心态（上）

在本节课中，我们将要学习在两性关系中至关重要的核心——王者心态。我们将探讨为何心态比聊天技巧更重要，并理解如何通过建立自信和主导权来构建健康的关系基础。

## 心态是核心，而非技巧

上一节我们介绍了课程主题，本节中我们来看看问题的根源。我发现，许多人在两性相处中最大的问题，并非聊天话题或技巧的缺乏。根本问题在于聊天的心法。

我们回想一下，无论是在线上聊天还是实际约会中，面对女性时内心感到不安和慌张。这种情绪会导致语无伦次。当表达语无伦次时，再说出一些不恰当的话，其实你已经失去了自我。

因此，我能教给大家的，绝对不仅仅是某句话该如何回复，某个公式该如何套用，或是某个架构该如何使用。因为这些行为的根源都由一个东西支配，那就是你的心法。没有心法的支配，就像我是一个剑客，但没有内功，无法真正施展剑术。

大部分兄弟的问题源于心态，而非技术。技术是最简单的，因为它可以快速掌握、模仿和复制。但是，如果没有正确的心法，你敢说某些话吗？你敢做某些动作吗？你不敢。

## 约会中的心理博弈

在约会时，你以什么样的心理状态面对对方？我举一个简单的例子。无论是公开直播还是与女生第一次见面，我见到她的第一刻，我会思考一个问题：是她更慌，还是我更慌？两个未曾谋面的人，通常是女性会更慌张。

如果此时你比她更慌，那么在这段约会或见面过程中，你已经处于下风。我的做法通常是：第一，我不会关注她会怎么看我。第二，我不会过分在意当下环境如何。我只专注于保持自信：**下颚微抬，眼神坚定，每个肢体动作都肯定自我**。

这个过程，女性通过感官和直觉能感受到的就是**自信**。这种自信会吸引她，使她进入你的世界。当她进入你的世界，就意味着你主导了这段关系。

## 关系中的高位与低位

上一节我们讨论了自信的表现，本节中我们来看看关系的本质。两性关系其实不存在绝对的平等。关系双方总有一方处于**高位**，另一方处于**低位**。

例如，在一个家庭中，掌握经济大权的一方，说话的分量和主导权是不同的。很多男性在婚后没有话语权，核心原因在于妻子可能更优秀，掌握了经济大权。以我为例，我永远掌握经济大权，或者说，我掌握的是**赚钱的能力**。这赋予了我足够的话语权和能力。

恋爱关系中也一样。双方的经济收入水平如果差距过大，很难处于同一个世界。例如，你月收入五六千，对方月收入两万，你们几乎不可能同频。人性如此，强者通常不会向下兼容。她只会同情或怜悯，但不会平等视之。

因此，这个世界就是：**谁强，谁就拥有话语权和主导权**。

## 关系发展的三个阶段

以下是关系发展的三个关键阶段，理解它们有助于摆正心态：

1.  **相识与聊天阶段**：这是关系的起点。此时双方更关注是否有共同话题和感觉，通常不会直接涉及收入、家庭背景等现实问题。
2.  **恋爱与约会阶段**：这是培养感情的阶段。从线上聊天到线下约会，再到确定恋爱关系。但这只是**阶段性的胜利**，感情基础尚不牢固，双方都有选择权，容易放弃。
3.  **婚姻与现实阶段**：这是关系的考验。当关系发展到谈婚论嫁，感情退居其次，**现实问题成为主导**。双方家庭介入，经济保障、异地问题、彩礼等成为必须面对的挑战。只有解决了这些现实问题，才算关系真正稳定。

所以，喜欢在婚姻面前作用有限。婚姻需要的是**经济保障**。如果你的收入无法支撑家庭正常运作，关系就可能破裂。

## 心态决定一切

只要你们的关系还未走到谈婚论嫁、面临现实重击的那一步，那么之前所有聊天、约会的挫折都**不值一提**。许多兄弟的误区在于，为了一些尚未开始或微不足道的困难（如对方不回信息、约会失败）而情绪崩溃，这非常幼稚。

真正能影响我的，是像商定结婚日期时，未能顾及对方家庭感受这类**现实问题**。除此之外，其他事情都不值得耗费心神。

## 恐惧的来源与克服

我知道兄弟们恐惧什么：恐惧聊天、恐惧见面、不敢直视对方眼睛、不敢自信做自己。我们常常过度顾及对方感受，却忽略了自己是否开心，这就叫**迷失自我**。

我经常培训助教们：第一，**下颚必须微抬**。第二，说话必须**表露自信**。你的眼神要坚定，不要飘忽不定。第一步就是练习与他人坚定地对视。

例如，面对女性时，正确的姿态是：
```plaintext
眼神坚定地看着对方说：“我等你好久啦。”
```
女人永远尊重比她强的人，尊重关系中主导的一方。如果你不断围绕对方转，你就丧失了主导权。

## 强者心法

狼群中的头狼，必须挑战并战胜其他狼才能上位。现实也是如此，强者才能赢得尊重。我们为什么要讲强者心法？因为**哪怕我失去她，也不能失去我自己、我的原则和我的世界**。

有多少人宁愿失去自我，跪下来讨好对方？如果你以这种姿态去恋爱，我建议你不要开始。你不适合谈恋爱，只适合旁观。

本节课中我们一起学习了王者心态的重要性，明白了心态是两性关系的核心，技术只是皮毛。我们探讨了自信的表现、关系中的高位法则、关系发展的三个阶段，以及如何克服恐惧、坚持自我。记住，强者不会向下兼容，建立自信和主导权是赢得尊重和健康关系的基础。
