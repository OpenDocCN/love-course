# 白天沟通：13.夫妻间日常的性暗示

🎼除了撒娇，你跟你老公日常沟通也得有情绪，不要两个人一聊天，就像开会一样正经。两个人私下里嘛，可以适当的开开车，要有私密话题，不要每天都是柴米油盐酱醋茶。如果一开始你觉得害羞，不好意思开口。

你们俩可以先从那种瑟瑟的聊天表情包开始。姐妹们，如果说没有这方面的渠道，可以联系我，然后慢慢的过渡到文字聊天。比如你老公跟你说，想你啦，你就回想往哪里啦，上面还是下面。🎼有你老公问你吃饭了没有你就回。

🎼不想吃饭，只想吃你，再就是面对面的沟通。就比如你老公说嘛，哎，今天买的这是什么东西啊，哎，挺不错的，挺大的。🎼然后你就回。🎼再大也没有你的大，或者说有你的大吗？

就凡是你们交流当中涉及到一些大小呀、形状粗细等等话题，你都可以往那方面引导去调侃它。再比如说你们聊到某些地方，某些事情的时候，你可以引导他，一起去幻想某些场景。打个比方，你老公说，哎，待会儿吃完饭。

我们一起去公园散散步吧，你就可以回怎么你想跟我一起去钻小树林吗？🎼或者他说啊，今天啊在哪里堵车，堵了一个多小时。🎼可以回他哇，那如果我在你身边的话，那车子是不是就站起来了等等等等啊。

最后呢就是一些小动作。比如说有事没事的时候，你可以摸一摸他的胸肌，拍拍他的屁股，或者蹭一蹭他的三角区。总的来说，就是在生活的方方面面去营造只属于你和他的私密话题。🎼动作。
