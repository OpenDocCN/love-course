# 恋爱军团3888：01：第三节 基础知识：目标的僚机

在本节课中，我们将要学习约会互动中的三种核心角色，并重点解析“僚机”的作用与运作方式。

在上一节我们介绍了约会的基本框架，本节中我们来看看构成这个框架的具体角色。在整个约会过程中，你会主要遇到三种人：你的目标、你的障碍以及你的僚机。

**目标**指的是你喜欢的这个女生，你要追逐的这个女生。

**障碍**指的是她身边的一些对你构成障碍的人，不管是男生还是女生。这些可能是跟你一样想追逐这个女生的一些男性，也可能是她的闺蜜，也可能是她身边的一些朋友。我们把这些人称之为障碍。这并没有任何敌对的意识，只不过是一个通称。

**僚机**就是帮助你去更好地获取这个目标的认可的人群。这些人可能是你的兄弟，也可能是帮助你去说好话、衬托你的一些人。

以下是关于僚机的一个具体例子：

*   当你在约会中途去洗手间时，留下的僚机（例如，一位认识你多年的大学同学）可以在你的目标及障碍面前，谈论你们在大学时期的经历。
*   他可以提及你在大学时是一个很酷的人，或者很受女生欢迎。
*   这样做的目的是增加女生对你的认可度。

僚机其实就在于起了一个助攻的效果。

本节课中我们一起学习了约会中的三种关键角色：目标、障碍和僚机。理解并善用僚机，能有效提升你在社交互动中的吸引力与成功率。
