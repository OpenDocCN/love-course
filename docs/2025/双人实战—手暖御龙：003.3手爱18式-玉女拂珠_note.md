# 双人实战—手暖御龙：003.3：手爱18式-玉女拂珠

在本节课中，我们将学习“玉女拂珠”这一手法的第二个核心方法：揉弹。此方法专注于通过特定的手部动作来提供温和而有效的刺激。

上一节我们介绍了基础手法，本节中我们来看看如何通过揉弹来深化体验。

揉弹是指用手掌心握住阴囊，并进行轻柔的按压与揉动。其核心在于节奏稳定、力度轻柔。

以下是执行揉弹手法的具体步骤描述：

*   用手掌心完全握住阴囊。
*   以非常轻柔的力度开始按揉。
*   保持“一下、两下”的稳定节奏进行。

通过掌握揉弹技巧，可以丰富手法层次。关键在于掌心贴合与节奏控制。
