# 自媒体与个人IP：03：小红书接广告须知

在本节课中，我们将学习在小红书平台接洽商业广告合作时必须了解的核心概念与行业术语。掌握这些知识能帮助你更专业地与品牌方沟通，提升合作成功率。

上一节我们介绍了小红书作为广告变现平台的潜力，本节中我们来看看具体接广告时需要知道的关键事项。

---

## 核心概念扫盲

以下是接广告前必须理解的几组核心术语。

### 1. 报备与非报备

品牌方或代理机构一定会询问合作形式是“报备”还是“非报备”。

*   **报备**：指广告合作向小红书平台官方报备。你需要在笔记中明确标注为“品牌合作”，平台会从中抽取 **10% 至 15%** 的手续费。因此，报备合作的报价通常更高。
*   **非报备**：指与品牌方私下合作，笔记内容通常更偏向软性广告。报价相对较低，但存在被平台识别为广告而导致限流的风险，这可能对你和品牌方的数据效果都造成损失。

关于选择哪种形式，没有绝对的好坏。对于粉丝量级较轻、内容偏软广的博主，首次尝试非报备合作通常问题不大。

### 2. Brief（合作简报）

如果你未来从事公关、品牌或商务工作，会频繁接触到 **Brief** 这个词。

达成合作意向后，品牌方或代理机构会向博主发送一份 **Brief**。其中包含品牌介绍、产品信息、内容方向、撰写指南及笔记具体要求。收到 **Brief** 后，务必及时查阅。如有任何疑问，应尽快与对方沟通确认，然后再开始创作。拖延或忽视 **Brief** 会被视为不专业的行为。

### 3. Seeding（产品植入）

有时你会收到品牌方私信，表示欣赏你的内容，并询问是否接受 **Seeding**。

这通常意味着品牌方希望免费提供产品，请你在内容中进行软性植入。**Seeding** 合作本身没有报酬，但如果你非常喜爱该品牌，或希望与品牌创始人建立联系，在账号发展初期尝试一下也未尝不可。

### 4. 蒲公英平台与报价策略

当你的小红书粉丝数超过 **5000** 时，务必前往后台开通“蒲公英”平台。

在蒲公英上，你可以设置官方公开的图文和视频报价（视频报价通常高于图文），并接收平台定向推送的品牌合作机会。开通即意味着你的个人商务合作正式启动。

关于广告报价策略，不同博主方式各异：

*   有些博主采用固定报价，无论品类品牌，图文/视频统一价格。
*   我的策略会根据品牌和产品进行调整：
    *   对于可接可不接的品牌，我会提高报价，对方接受则合作。
    *   对于我非常看重、希望长期合作且产品优质的品牌，我会适当降低报价，以诚意争取合作机会。

重要的是，你需要明确自己从每次合作中追求的是 **名利、金钱还是品牌关系**，并形成自己的思路。

结合我们之前讲过的B端变现内容，请注意长期建立自己专业的商务对接风格与个人形象。接广告是所有博主的必经之路，也是衡量其专业度的重要部分。
