# 自媒体与个人IP：03：如何应对黑粉 🛡️

在本节课中，我们将要学习如何以正确的心态面对网络上的负面评论与“黑粉”。这是成为有影响力的自媒体人必须掌握的核心素质。

---

上一节我们讨论了博主的格局，本节中我们来看看具体应对黑粉的几种心态。选择最适合你的一种去理解和吸收即可。

**心态一：接受被片面评价的权利**

很多人会说：“你又不完全了解我，凭什么评价我？”这种观点并不成熟。**没有人能认识全部的你**，评价本身就是主观行为。网友有权对你的内容**断章取义**，甚至无需看完整个视频，就可以对某一点表达不满。作为博主，吸引流量就意味着要面对公众的各种声音。想通这一点，有助于释怀。

**心态二：理解喜欢与诋毁的平衡**

你在网上可能会收到**超出客观的喜欢**，那么相应地，收到一些**超出客观的诋毁**也是正常的。正负相抵，这是网络生态的一部分。

**心态三：认清攻击的实质**

一个人持续攻击你，往往是因为你的某个特点或价值观**刺痛了他**。他攻击的是那个刺痛他的点，或是他心中构建的幻象，而你只是这个情绪的载体。例如，鲜明的ENTP阿尔法型人格可能会吸引极度喜爱，也可能招致极度反感。作为博主，你的气度必须高于攻击者。

**心态四：正视同行竞争**

只要你做得更好，影响力变大，无论你多么谨慎谦逊，**出现黑你的同行是必然的**。受众的多样性决定了这一点。如果你的网络环境一片叫好，没有任何负面声音，那可能说明你还处于非常初级的阶段。

以下是另一种常见情况：

**心态五：因爱生恨的“黑粉”**

有些黑粉的出现，是因为互联网上有人**太爱你、太捧你**。这会让另一些人感到不以为然，进而通过攻击你来反驳你的狂热支持者。这是一种典型的“因爱生恨”现象。

---

能否顶住一定程度的“黑红”，是能否成为更有影响力的台前人物的**根本素质**。许多成功的公众人物都经历过这个阶段。能顶住**超大程度的全网黑**，则意味着你“命硬”，适合吃这碗饭。

此外，还有一种更复杂的情况：**“粉转黑”**。这有时与你的行为无关，可能只是对方的心态或你输出的内容发生了变化。这也很正常。

最后，我们总结两条核心建议：

1.  **找到化解心态**：你必须找到一句对你有用的话或一种心态，来化解和释怀面对黑粉的感受。
2.  **选择不予理会**：知道它的存在，但清楚它无法真正伤害你。**站台前的人要有这种素质**，格局和姿态一定要比黑粉高。
