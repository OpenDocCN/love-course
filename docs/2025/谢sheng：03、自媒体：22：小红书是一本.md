# 自媒体与个人IP：03：小红书是一本电子杂志

在本节课中，我们将要学习小红书平台的独特调性。理解这一点，是做好小红书内容的基础。

上一节我们介绍了内容能力与平台理解的重要性，本节中我们来看看小红书这个平台的核心特点。

## 平台调性：视觉化与视觉溢价

只有**内容能力**加**平台理解**，你才能够跑出来。自媒体人必须花时间钻研平台的游戏规则。内容能力是学习本身，而平台理解就像是刷题。只有学懂了并刷了足够多的题，你才有可能考出高分。

首先，小红书是全网最视觉化和最有视觉溢价的平台。因为小红书整体女性用户多，精致的内容多，购物分享多，图片好看，这已经是这个平台的基础调性。

在这样的一种整体氛围下，会自然而然地对每一种垂类博主的视觉审美要求都更高。一个亲身体验是：同样的视频传在抖音，大家可能更多在讨论观点。小红书对于观点的包容度要高一些，但对于你妆容的要求就很高。如果在小红书，用户前面看了好几条精致的内容，再刷到妆容稍有瑕疵的视频，就会产生落差感。所以小红书的视觉氛围确实比其他平台要强很多。

那么视觉具体是指什么呢？以下是提升视觉的五个维度框架：

1.  **IP本身的长相、穿搭和化妆**。
2.  **拍摄场景**本身是否敞亮、高级、有品位。
3.  **拍摄硬件**是否够好，画质是否高清。
4.  **封面页的视觉配色和设计**是否专业。
5.  **视频的剪辑、配乐、转场、字幕**是否高级。

这五个维度是：**IP主体形象 + 场景 + 画质 + 封面设计 + 剪辑审美**。如果你这五个维度都能做到很高分数，当然特别好。但事实上，五个当中有三个比较强就已经很好了。这是一个你提升的框架。如果你其中有一两个点实在做不到，那么你在其他几个点当中，审美的力度要加强一点。

前半句讲的是“全网最视觉化”，后半句讲的是“最有视觉溢价”。什么是视觉溢价呢？就是小红书完全可能由于你的镜头美、场景美而给你很多耐心，哪怕这个内容本身没有那么多的观点和含金量。所以，如果现在有人想做vlog，靠非常细致的精致的生活场景和剪辑的感觉，首选小红书，底盘一定会比抖音和微博要好。

## 个人页面：打造杂志感

上一部分我们了解了小红书的视觉特性，接下来我们深入探讨其页面设计的本质。

小红书的个人页面非常具有杂志感。所以这节课的标题是：小红书的本质就是一本电子杂志。

现在可以想一想，怎样的杂志高级，什么样的杂志会让你有想购买的欲望。一般来说，一本杂志，它图片的视觉冲击力强，标题的冲击力强，图文交错得很有设计感。同时，你乍一翻，能在所有看似凌乱的图文当中，把握到一种整体的设计和色调。这样的杂志，它往往高级。

所以，如果你真的想把小红书做好，配色功底是一定要有的。你的页面、你的封面大概主打什么色系，什么样的感觉，是要经过设计的。其实我们之前做公众号，在起号的时候都会有视觉设计的过程。这套视觉设计放在小红书应该是更加重要和刚需。

我建议所有真心想要做小红书的人，至少应该长期订阅两种杂志，最好是看纸质的，放在你办公和学习的地方，有事没事翻一翻。因为审美和配色这种东西，它真的是一种感觉。说实话，如果你为了做小红书，把这种配色、视觉、封面的感觉研究多了，那么你再去设计一些PDF或者PPT，对你的个人简历进行配色，应该是一种降维打击。

以下是几本有助于提升视觉审美的杂志推荐：

1.  **《VOGUE》的封面合集**：你可以仔细去看每一张封面上“VOGUE”五个字母的颜色，换成任何其他一种配色，应该都比不上封面上的那种效果来得好，这就是一种实打实的专业的配色功力。
2.  **《嘉人》**：这是一本非常关注女性文化和女性人物的杂志，有助于提升你整体的氛围性审美。
3.  **《时尚芭莎》**：照顾一下男用户，《芭莎》有专门的男士版，叫做《芭莎男士》也非常不错。《时尚芭莎》比起视觉和照片上的优秀，更加优秀的是它的标题。这些标题非常能够直接运用和迁移在小红书的标题里。

看优秀的杂志，从配色到标题到图文的编排，我们都可以去学习。内化了这些东西之后，都有助于你把小红书的视觉外壳往上提好几个档次。

当然，在刚开始做的时候，做得有些粗糙，审美也不是那么好，大家也不要着急。重要的是这个成长和进步的过程。我经常关注一些视觉非常优质的博主，然后去看他们最开始的内容，你会发现任何博主都有一个非常明显的成长过程。他们最开始的那几条也是非常有粗糙感的。所以这个成长的过程本身就非常美妙。
