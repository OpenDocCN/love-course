# 社交能量训练：第1天：挥手与吸收能量

在本节课中，我们将学习一个简单但强大的社交入门练习：通过挥手打招呼来吸收能量，并减少内心的焦虑。

## 核心概念：行动带来能量

大多数人感到焦虑，是因为思考过多而行动过少。当你开始行动，就能将自己从大脑的过度思考中带入现实世界，从而减少焦虑并获得能量。

**核心公式：减少焦虑 = 增加有效行动**

## 第一天的具体任务

上一节我们介绍了行动的重要性，本节中我们来看看具体的实践方法。

你的任务是向**20个陌生人**挥手并说“Hello”或“嗨”。

以下是任务的具体说明和注意事项：

*   **动作要求**：向对方挥手，并看向对方。
*   **语言要求**：简单地说“Hello”、“嗨”即可。
*   **距离要求**：无论对方距离是10米、20米甚至50米，都可以进行。
*   **目标要求**：目标是完成“打招呼”这个动作本身，而不是进行深入聊天。
*   **反应预期**：对方可能回应，也可能不理会。这两种结果都完全没问题。

## 为何这个任务有效

这个简单的任务之所以有效，是因为它完成了一次低风险的社会互动。你只是打了个招呼，没有复杂的对话压力。

即使对方没有回应，你也没有任何损失。相反，你通过“行动”本身，已经从自己的思维内耗中走了出来，这个过程就会为你带来能量。你会发现自己完成后的焦虑感显著降低。

## 任务总结

本节课中我们一起学习了社交训练的第一步：挥手打招呼。

总结来说，第一天的任务非常简单：找到20个人，向他们挥手说“Hello”。重点在于执行“挥手”和“注视”这个动作，通过最微小的行动将自己带入现实，吸收外部能量，从而有效减少内在焦虑。

现在，你可以开始去实践这个任务了。
