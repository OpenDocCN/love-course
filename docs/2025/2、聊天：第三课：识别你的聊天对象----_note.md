# 情感聊天技巧：第三课：识别你的聊天对象类型 🧐

在本节课中，我们将学习如何识别聊天对象的类型。了解对方的类型，有助于你调整聊天策略，但核心原则不变。我们将介绍几种常见的女性类型，并分析其特点，以便你在互动中做出更合适的判断。

上一节我们介绍了聊天前的准备，本节中我们来看看如何识别你的聊天对象。

## 识别女生类型

在开始聊天之前，你需要识别女生的类型。这个类型包括女生本身属于什么类型。我们划分为正常人、土骚女、捞女，还有TT Video（高级名媛型）。

你可以通过社交软件的展示面、搭讪时的状态，以及她的朋友圈来判断她是什么类型。以下是几种主要类型的特点：

*   **正常的女生**：她会有自己的生活，可能经常会发一些吃的、玩的、工作的内容。旅游没那么频繁。
*   **土骚女**：经常发布网红打卡内容，穿着比较性感、稍显廉价，经常去酒吧。
*   **捞女**：朋友圈经营得很精致，刻意感很重，内容倾向于吸引男性。她们是市面上某些情感导师的主攻对象。
*   **TT Video**：照片的质感很好，出国的频率比较高。比起捞女，她们更有生活气息。

我经常接触和匹配TT Video类型的女生。但你们懂了本质的东西后，其实都可以应对。只是需要根据女生不同的类型稍作调整。

## 识别类型的目的与影响

注意，这里说的要识别类型，不是让你去根据她的类型准备不同的话题或者是话术。记住，这些都不影响本质。说这个是为了让你知道，类型会影响你的“意愿度”表达策略。

举个例子，普通女生她的追求者没那么多，对外界没那么开放，也就是会更容易出现自闭、单纯、安全感不足这种特质。

这需要你在聊天中给更多的“意愿度”。也就是说，你想跟她接触的意愿要表现得更强烈一些。注意，这里不是让你们去“冲”的意思，而是让她感受到你想了解她，而不是只想发生关系，这和“冲”是不同的。

    *   **假兴趣**就是聊天中看起来对你很有兴趣，但其实她对谁都这样。
