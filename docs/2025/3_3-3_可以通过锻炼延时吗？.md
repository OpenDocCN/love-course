# 3_3-3_可以通过锻炼延时吗？

。时间可以通过锻炼延长吗？在具体回答时长是否可以通过锻炼延长这个问题时，我们首先要分清楚自己具体是属于哪种草险。早泄实际上分为两种。一种是器质性的早泄。器质性的早泄就是器官发生了一些病变。

这个属于医学范畴的问题。例如，生殖器官先天的畸形，包茎、龟头或者是包皮的炎症、尿道炎、钉丁炎。脊髓肿瘤、慢性前列炎等这些可反射性的影响脊髓中枢的一些疾病，使男性的射精中枢兴奋度啊降低。

也就是更容易发生射精，从而导致早泄。还有大约70%的男性是由动脉硬化和高血压所引起的这个阳痿早泄。高血压会引起血管内的一个受损。导致海绵体充血障碍，引起血管性的阳痿。同时。

治疗高血压的药物以及这个副作用就是会引起这个阳痿。糖尿病患者当中有一半左右不同程度的阳痿早泄。由于糖尿病会导致这个微血管病变，让钉钉海绵体充血功能变差。同时，糖尿病呢也会引起周围神经的一些病变。

会引起神经性的一个阳痿，在慢性酒精中毒疾病中，10%的患者可引起这个阳痿早泄。因为长期的饮酒会造成这个慢性中毒，对中枢神经系统产生难以逆转的一些损伤。同时。

严重的骨盆骨折、睾丸疾病、前列腺手术以及慢性前列腺炎症、慢性肾病、甲亢等也可以造成这个性功能的一些障碍，出现器质性早泄和阳痿，对于这种早泄啊需要尽快的去医院接受正规的一个治疗。先治病再解决早泄才是更加。

合理正确的一个顺序。第二种早泄呢是非器质性早泄而泄，大部分男性呢都是这种类型的一个早泄。这类早泄呢大多数不注意这个节制，经常在自慰的时候呢，强橹猛射，时间久了，习惯性的一个早泄，或者说你每天自慰很多次。

造成体质虚弱，腰膝酸软，干活喘气等等。还有部分男性呢出门有车，唉，吃饱就坐，不爱运动，会造成肌肉的控制力不好，还有一些外在的压力。比如说对自己尺寸的一个不满意，曾经表现不好，留下了一些心理的阴影。

过分担心伴侣的反应等等的一些心理因素，种种啊，这些都会导致你的一个早泄。而这类的早泄呢都是可以通过这些锻炼得到一个明显改善的，锻炼自己肌肉的一个控制力，锻炼自己自慰的一些方式，锻炼自己的这个耐受力。

建立健康的这个心理状态，一步步的达到延长时间的一个效果，心能力严格的来说呢，它就是一项身体技能和其他运动的技能呢是一样的。比如说这个举重打篮球、开车啊等等。这些呢都是可以通过后天的学习和锻炼。
