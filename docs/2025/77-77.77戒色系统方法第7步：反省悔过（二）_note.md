# 戒色系统方法：第7步：反省悔过（二）🔍

在本节课中，我们将继续深入学习戒色系统方法的第七步——反省悔过。我们将通过一个具体的例子，详细拆解反省悔过的五个步骤，帮助你理解如何通过自我剖析来化解内心的负面情绪，从而为戒除邪淫打下坚实的心理基础。

上一节我们介绍了反省悔过的基本框架，本节中我们来看看如何将这个框架应用到一个具体的情绪——怨气上。

## 反省悔过的五个步骤（以“怨气”为例）

以下是运用五步法分析“对老板心怀怨气”这一具体情境的完整过程。

**第一步：识别具体行为**
*   行为是：在公司上班时，对老板心怀怨气。

**第二步：剖析行为背后的念头**
*   念头是：抱怨自己干的事情太多，而工资太少。
*   （注：此处我们假设老板的薪酬安排是合理的，以便聚焦于自我反省。）

**第三步：追溯念头背后的心态**
*   这种抱怨念头背后，其实是**自我重心**与**自私自利**的心态。如果发现自己爱抱怨、喜欢责怪他人，就需要在“抱怨”这个点上深入反省。
*   我意识到，抱怨别人，根源在于自己太过自私，因为我只关心自己的感受。
*   当你怨恨别人对你不好时，可以问自己三个字：**凭什么**？凭什么别人要对我好？我又有何德何能？
*   世间万事万物都遵循能量平衡。我没有对别人好，别人凭什么要对我好？需要问问自己，究竟做了多少实实在在的事情。
*   想要得到某种待遇，应先问问自己是否具备对应的“德”。抱怨起不到任何作用，本质上是**自虐**和**自我折磨**。一旦想通这一点，就不会继续抱怨，而是转向“反求诸己”，从而化解抱怨的心结。

这里需要强调一下“德”与“得”的关系：
*   **德**：指道德的“德”，是**无形的价值能量**。
*   **得**：指获得的“得”，是**有形的物质回报**。
*   一个人所赚的有形钱财，是他向外界所贡献的无形价值能量的变现。如果嫌自己赚得少，就要反求诸己，问问自己对社会、对他人做了多少事，贡献了多少无形的价值能量。
*   用公式表示这个关系就是：**有形的物质回报 ∝ 贡献的无形价值能量**。
*   只要你潜心耕耘，不断贡献价值，该有的物质回报迟早会到来。**不怕没有“得”，就怕没有“德”**。
*   此外，天天抱怨别人，导致自己心情不好，这本身就是一种精神上的自虐。

**第四步：认清错误思想的危害**
*   认清抱怨是自私自利的表现，且本质上是自虐，毫无用处。

**第五步：确立正确的思想与行为**
*   正确的思想是：认识到怨气源于自私自利，抱怨即是自虐。
*   正确的行为是：如果想要获得更多物质回报，就应反求诸己，多问问自己为别人做了多少有用的事。
*   做事要用心。你有多少“心”，就能做多少事。例如：**真心、内心、恒心、决心、耐心、关心、诚心、专心**。问问自己拥有多少颗这样的心。
*   只有踏踏实实做好实事，才能获得好的回报。

## 总结与过渡

本节课中，我们一起学习了如何通过“五步法”对“怨气”进行深入的反省悔过。通过识别行为、剖析念头、追溯自私心态、认清危害并确立正确行为，我们可以有效减少内心的怨恨情绪，从而减少负能量。

这个过程就像是**拔除内心的“怨恨之草”，种下“踏实劳动”的种子**。持之以恒，内心的阴气、邪气会越来越少，阳气、正气会越来越多，自然就能远离邪淫的诱惑。
