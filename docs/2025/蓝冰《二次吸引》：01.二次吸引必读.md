# 蓝冰《二次吸引》：01.二次吸引必读

二次吸引这个名词呢，也许你已经耳熟能详了。也许你是第一次听到过。那么什么是二次吸引呢？在一段两性关系中，无论你是恋爱还是婚姻，那么都存在着一个现象，那就是两个人相处时间久了之后。

相互之间都已经慢慢的淡漠下去了，感情就会产生矛盾和分歧，甚至这段感情无疾而终。这个时候你会发现一个问题，强留对方的那一方使用了各种方法，都没有办法留住另一方这个时候的问题就产生了。

那么我们究竟用什么样的方法才能让这段感情继续下去呢？在这里呢从心理学的角度上面，我们就引入了二次吸引这个概念。其实二次吸引呢，无论是在婚姻关系中，在恋爱关系中，还是在任何一段两性关系中。

它都有着一个不可撼动的重要地位。为什么呢？因为我们都知道感情人的感情尤其是两性关系的感情都是起起伏伏，有高潮有低谷，不可能说是从一而终，一蹴而就的但在这里当一段。🎼感情消磨殆尽的时候。

当一段感情已经没有了新鲜感的时候，我们就要进行二次吸引，来让这段感情呢继续下去。二次吸引在两性关系中呢使用适用的场景是非常多的，在这里我就不一一列举了。那么我们现在就来探讨一个问题。
