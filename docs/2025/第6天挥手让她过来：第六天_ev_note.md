# 无声沟通入门：第六天：挥手示意

在本节课中，我们将要学习一种非语言的沟通方式——挥手示意。这是一种简单但有力的社交行为，能有效吸引他人注意并建立初步联系。

上一节我们介绍了非语言沟通的重要性，本节中我们来看看如何具体运用“挥手”这个动作。

## 核心任务与目标

今天的核心任务是：通过挥手动作，无声地吸引他人过来。你不需要说话，只需做出挥手示意“你好”的动作即可。这个动作传递出一种自信和友好的姿态。

**任务目标**：成功完成 **4到5次** 挥手示意。

## 动作分解与要点

以下是执行挥手示意动作的几个关键点：

*   **动作简洁**：自然地抬起手，轻轻挥动。
*   **保持微笑**：配合友好的面部表情，如微笑😊。
*   **无需言语**：整个过程不需要说话，依靠动作本身传递信息。
*   **感受力量**：体会这种无声沟通带来的影响力和自信感。

## 实践场景示例

你可以在多种场合练习这个动作。例如，向远处的朋友、同事或熟人挥手，示意他们过来。享受这个简单动作所带来的有趣互动体验。

Hello. Hey. OK.
