# 长期关系课程：026：长期关系阶段概述与常见问题

在本节课中，我们将要学习长期关系阶段的核心概念，并了解在这一阶段中普遍会遇到的几类典型问题。理解这些问题是建立健康、持久关系的第一步。

当我们和一个喜欢的女生真正用心谈恋爱，成为男女朋友之后，那并不代表我们已经成功了。相反，那只是说明你们两个真正开始进入彼此生活了。很多兄弟以为把一个女生追到手就万事大吉，后面就放松警惕，不再用心去经营这段关系。相处一段时间就会出现各种问题，最后也是以失败而告终。

其实长期关系本身就是需要用心并且适当的方法去经营的。因为长期关系本身就会遇见各种各样的问题，例如价值观的冲突、生活习惯的冲突、彼此标准的碰撞，还有情绪的积累，以及时间久了之后的平淡。这些都是必然会发生的，不以我们的主观意志为转移。而很多兄弟又没有经历过什么感情，所以对于这些问题也不知道怎么应对。最多听了一些朋友的意见或者网上的情感大师的胡扯，根本不能解决你的问题，反而搞得越来越糟。

长期关系基本上遇见的问题都是差不多的。以下是我们在长期关系中经常会遇见的几种典型问题：

1.  **过度迁就与自我价值丧失**：和女生在一起之后，过于迁就女生，不断降低自己的身份。最后只有你自己一味的付出。女生只知道坐享其成，把你所有的付出当做是理所应当的。
2.  **忽视对方感受与自我中心**：当你和女生在一起之后，就高枕无忧，不再去考虑和顾及女生的感受。什么事情都以自己的想法为中心，不断去消耗女生对你的热情和爱，并且从来不知道悔改。
3.  **逃避问题与无效沟通**：不会处理你们相处过程中遇见的问题，也不懂得如何有效的沟通，只知道逃避，期待着你们之间的问题会自然而然的消失。但是其实只是掩盖了问题，随着时间越积越多，最后积累到一定程度之后爆发。
4.  **新鲜感消失与关系倦怠**：相处一定的时间之后，和女生之间的新鲜感消失了，彼此之间没有了开始的时候的激情。女生有些迷茫，你也不知道怎么处理。然后，女生就会觉得和你相处没意思，腻了和你提分手。
5.  **控制欲过强与安全感缺失**：你自己非常担心女生会移情别恋，爱上别的男人，所以控制欲就会特别强。例如，禁止她和别的异性来往，每天要查她的手机，或者疑神疑鬼，什么都要去控制女生，为的就是填满自己的不安全感。
6.  **异地恋的特殊挑战**：异地恋除了会面对上面的这些问题之外，还会有更多的问题。例如见不到面，摸不着，双方对于这段感情都信心不足。然后，也不会懂得沟通。这个时候，你们就会误解彼此的意思，最后导致无疾而终。

上面这几种问题，就是我们在长期关系中经常会遇见的。绝大多数男生都不太懂得怎么样去应对，最后结果就只能是分手。然后谈了好几段，还是没有任何的结果，不断的去恶性循环，浪费了时间，也错过了很多好女孩。

不过这也是非常正常的，因为这些问题确实是需要方法、需要用心经营才会解决。如果说你没有认真学习过，确实是比较难以应对的。

本节课中我们一起学习了长期关系阶段并非终点而是新起点，并梳理了关系中常见的六大类问题：价值丧失、自我中心、沟通逃避、新鲜感消退、控制欲过强以及异地恋挑战。认识到这些问题是解决问题的第一步。

在接下来的课程中，我们将具体探讨如何去避免这些问题，或者说出现了这些问题，我们应该怎么样去解决。只要你们理解并且实践去运用这些方法，相信你们的长期关系一定会十分健康和长久。
