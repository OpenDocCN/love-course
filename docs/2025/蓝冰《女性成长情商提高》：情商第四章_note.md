# 蓝冰《女性成长情商提高》：第四章：高情商沟通指南 🗣️

在本章中，我们将学习如何进行高情商的沟通。沟通是必不可少的生理需求。医学研究人员发现，热爱沟通、拥有活跃人际关系的人，比性格孤僻、不爱社交的人更长寿、更健康。喜欢并善于沟通的人比普通人记忆力更好，智力更高。但爱沟通和会沟通是两个概念。会沟通是一种高情商的表现。

那种一沟通就对别人进行道德批判，用自己的标准评判他人的行为，如果对方不符合自己的价值观就认为其不对、不道德，这其实是**暴力沟通**。

## 识别暴力沟通 🚫

上一节我们提到了道德批判，本节中我们来看看暴力沟通的其他几种表现形式。

以下是暴力沟通的三种常见表现：

1.  **强行比较**：在沟通中把对方与他人进行比较。严格来说，这也是一种批判形式。例如，经常拿自己的伴侣与他人对比。
2.  **强人所难**：在沟通中使用批判、指责或命令的方式，将自己的意愿强加给对方，甚至以惩罚相威胁。
3.  **回避责任**：在沟通中为自己找借口，逃避对自己思想、情感和行动应负的责任。例如，用“你爱我就应该给我花钱”来掩盖自己赚钱能力的不足。

这些行为会让人感觉你情商很低。

## 实践非暴力沟通 ✨

那么，高情商的沟通该怎么做呢？真正的高情商沟通其实就是**非暴力沟通**。下面我们从四个方面循序渐进地了解。

### 1. 观察 👀

在沟通中观察自己与他人，不以自我为中心，从多角度观察事物和对方的行为，并客观如实地描述出来。切记要**描述**，而非**评论**。

> **描述**：“你这周有三天加班到晚上十点。”
> **评论**：“你总是不顾家。”

### 2. 感受 💖

在沟通中，清晰表述自己对事情的感觉，让对方更好地理解。注意，感觉不是想法。你需要建立自己的感受词汇表来准确表达内心。

以下是两类感受词汇示例：

*   **需求满足时的感受**：兴奋、高兴、喜悦、开心。
*   **需求未满足时的感受**：害怕、担忧、愤怒、尴尬。

### 3. 需要 📋

明确自己的需要。你可以借助马斯洛需求理论来分析自己在沟通中究竟需要什么。然后，清晰地说出自己的需要。当你明白自己的需要后，清晰地表达感受并提出请求，同时在沟通中理解别人的感受。

### 4. 请求 🙏

用合理的语言说出自己的请求。你需要确认谈话目的，提出具体请求，并通过对方反馈来确认其是否理解了你的想法。整个过程中，注意使用请求的口吻，而非命令，避免让沟通变成强人所难。

> **命令**：“你必须马上来帮我！”
> **请求**：“我现在需要你的帮助，如果你方便的话，可以过来一下吗？”

掌握以上思路，你便能与任何人进行舒适友好的非暴力沟通。你需要不断揣摩、记录并与自己的行为对比，才能发现问题所在。

## 核心沟通技巧 🛠️

在沟通的应用过程中，请注意以下三个核心技巧。

### 技巧一：倾听与确认

沟通时最重要的是倾听对方的心声，明白其内心的感受和需求，这是沟通的基础。你需要做到客观观察，放弃主观想法和判断，专注聆听对方的需求和请求。你可以通过提问来确认对方的感受和需求是否如你所理解。

### 技巧二：表达感激

在沟通中，如何让感激显得更走心很重要。以下是三步反馈技巧，能让对方感受到你的感激：

1.  说明对方做了什么事情使你的生活得到了改善。（“谢谢你帮我整理了报告。”）
2.  说明对方的行动满足了你什么样的需求。（“这让我节省了大量时间，能更专注于核心任务。”）
3.  表达自己的心情和感受。（“我感到非常轻松和被支持。”）

使用此技巧时，务必明确对方的行为，并清晰表达自己内心的情感。

### 技巧三：自我对话

在沟通中遇到事情时，首先要学会客观描述自己所处的境况和内心感受，然后明确自己需要什么。最后，结合自己的需要，对自己提出新的、契合实际的想法和要求。

总之，真正高情商的非暴力沟通是一种生活态度。它让尊重、理解、欣赏、慈悲和友情主导生活，善待自己，对人友好，用爱的语言为生活增添温暖和幸福。

生活中许多沟通高手都遵循一些规律：他们懂得根据对象选择适当的沟通方式（如眉目传情、直白语言或感激方式）；他们因势利导，拥有同理心，能感受对方的处境和心情；他们善于自我监控，不断问自己“我做得怎么样？状态情绪如何？需要调整吗？”；他们在沟通中会先思考再发问。

如果你也能使用同样的思维逻辑并多加练习，就能拥有高情商的沟通能力。
