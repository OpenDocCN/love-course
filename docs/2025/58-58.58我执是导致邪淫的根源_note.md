# 戒色自救指南：58：我执是导致邪淫的根源 🧠

在本节课中，我们将要学习“我执”这一核心概念，并深入探讨它如何成为导致邪淫行为的根源。我们将分析我执的具体表现及其带来的负面影响，帮助你理解并识别自身可能存在的问题。

## 概述：什么是“我执”？

我执，即自我中心倾向非常严重，是对自我的过分执着。思考问题习惯以自我中心出发。

邪淫患者往往就是我执很重的人。

## 我执与邪淫的关系

上一节我们介绍了我执的定义，本节中我们来看看它与邪淫有何关联。从表面上来看，好像没什么关系，但是其实是有很密切关系的。邪淫的人往往我执很重。

我执是自我中心的态度，会促使一个人更容易陷入邪淫行为。因为他只关注自己的欲望满足。

自我中心思维是指一个人过度关注自己的需求、欲望和利益，将自己放在首要位置，而忽视他人的感受。这种思维方式常常表现为自私自利和缺乏同情心。当一个人过度自我中心时，他会以满足自己的欲望为重，而不顾别人的感受和道德准则。

而邪淫行为则是满足自己欲望的一种方式。所以，一个人自我中心思维太重，必然会促使他更容易陷入邪淫行为。

## 我执背后的原因

我执背后的原因是什么？原因就是把自己看得太重。所有的思维都是围绕着自己来思考，认为自己是整个宇宙的中心。

这种过度关注自己的态度，容易对他人的需求和利益漠不关心。导致自我中心、自私、贪婪，喜欢满足自己的欲望，沉迷于各种感官刺激中。

## 我执的具体表现

了解了我执的成因，接下来我们具体看看我执有哪些表现。以下是十种常见的我执表现：

1.  **自我中心**
    我执重的人常常将自己放在中心，以自我为重。一切事情都围绕自我中心来思考，关注自身的需求、欲望和利益，而忽视他人的感受和痛苦。

2.  **虚荣**
    虚荣就是喜欢不停地显摆。这种态度是对自我形象和外在评价的过度关注，追求名誉、地位和外貌。虚荣反映了对自我认同的强烈追求，以外在的成就和认可为衡量标准，却忽视了内在的价值和意义。

3.  **傲慢**
    傲慢是对自己持有过高评价，认为自己很优越、高人一等。这种傲慢的态度源自对自我的高度重视和自我中心的观念，忽视了别人的价值和平等。这种心态在佛家就叫我执我慢。

4.  **固执**
    固执就是固执己见，听不进别人的意见和想法。表现为顽固、一根筋、死板，原因也是我执很重。固执的背后是对自我的强烈执着。

5.  **任性**
    任性就是任由自己的性子去做事，不考虑别人的感受。其实这也是我执很重的一种表现。任性表现为对自己的欲望和需求的过度重视，不考虑他人的感受和需要。任性的行为常常出于对个人意愿的追求，不愿意妥协或考虑整体的利益。

6.  **攀比**
    喜欢和别人做比较，特别是在外在成就、物质财富或社会地位方面。他们希望通过超越别人来证明自己的价值和重要性。

7.  **冷漠**
    一个冷漠的人，往往缺少对别人的关心和同情。他们可能忽视他人的感受和需求，只关注自己的利益和欲望被满足。

8.  **自我强化**
    一个我执很重的人，总是需要通过强化自我来增强自我感受和认同感。他们可能追求权力、地位、名声等外在的标志，以加强自己的自尊和自信。例如，有的人喜欢别人称呼他为“某总”才能找到自我存在感和优越感。

9.  **不接受批评**
    我执重的人通常难以接受别人的批评，或者被别人指出错误。他们可能会做出强烈的反应，如愤怒或抵触，因为批评触动了他们的自尊和自我认同。

10. **爱打游戏**
    我执重的人更爱打游戏，因为打游戏是在游戏里面逞强、击杀对手、释放占有欲和控制欲的场所。游戏里处处强调“我”的等级、装备、战斗力、击杀人数，这一切都与我执高度相关。游戏里的顶尖玩家，也往往是我执很重的人。

## 我执导致的结果

认识了我执的各种表现后，我们来看看这些表现会造成什么实际的结果。以下是主要问题：

*   **自我中心**：导致不关心他人，对别人缺少理解和同情。人际关系会变得很差，别人对自己的评价也会变差。还容易衍生出虚荣、自负、傲慢、任性、固执等问题。
*   **虚荣**：导致别人对自己失去信任。频繁吹嘘会让人感觉不够谦虚，从而失去认可。也容易招人反感，引发他人不满。
*   **傲慢**：导致对别人不尊重，破坏人际关系。同时影响个人形象，让别人认为你不够谦虚、不尊重人。
*   **固执**：导致自我封闭和孤立，听不进别人的意见。给人固执、较真的印象，影响自身形象。
*   **任性**：指只根据自己内在的欲望行事，不顾外在影响与不良后果。会导致人际关系疏远，行为冲动不理性，造成不好的后果。
*   **爱打游戏**：
    1.  作息不规律，经常熬夜。
    2.  时间被大量浪费。
    3.  影响学习和工作。
    4.  影响视力。
    5.  提高邪淫的可能性。
