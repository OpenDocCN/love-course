# 猛男必学之性福肾功：013. 强肾食疗法 🥗

在本节课中，我们将要学习如何通过日常饮食来增强肾功能。食疗是养生保健中非常重要的一环，通过摄入特定的食物，可以有针对性地滋养我们的五脏六腑。

## 食物颜色与五脏归经 🎨

上一节我们介绍了食疗的重要性，本节中我们来看看食物如何对应我们的身体。中医理论认为，不同颜色的食材归属于不同的脏腑经络。

以下是食物颜色与五脏的对应关系：
*   **黑色食材**：归**肾脏**。
*   **黄色食材**：归**脾脏**。
*   **红色食材**：归**心脏**。
*   **绿色食材**：归**肝脏**。

因此，想要强健肾脏，我们应多选择黑色的食物。

## 核心黑色食材推荐 ⚫

了解了基本原则后，我们来看看具体有哪些黑色食物对肾脏有益。黑色食物因其色黑，被认为能直接入肾经，起到滋补作用。

以下是几种常见的强肾黑色食材及其用法：
*   **黑豆**：可制作**醋泡黑豆**。将黑豆洗净，在锅中炒至八九成熟，放凉后放入容器，倒入三年陈醋浸泡约一周。每日食用约20粒。
*   **黑芝麻**：可直接食用或磨粉。
*   **桑葚**：可直接食用或泡水。

## 其他有益食材 🍖

除了黑色食物，还有一些其他类型的食材也对肾脏有很好的补益效果。这些食材通常具有温补或固精的功效。

以下是一些值得推荐的强肾食材：
*   **动物类**：会打鸣的**公鸡**、牛鞭、鹿鞭等。
*   **植物类**：**韭菜**及**韭菜籽**。用韭菜籽熬水饮用，对遗精、阳痿有一定的调理作用。
*   **海产及其他**：牡蛎、板栗、泥鳅、鸡蛋等。
