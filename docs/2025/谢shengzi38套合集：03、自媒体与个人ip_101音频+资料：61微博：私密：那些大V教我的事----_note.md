# 自媒体与个人IP：03：微博篇延伸：平台现状与大V视角 📊

在本节课中，我们将学习如何通过微博的财报数据来理解其平台现状，并分享一些资深微博大V对平台生态的真实看法。这有助于我们更理性地看待微博，并调整个人IP的运营策略。

上一节我们介绍了微博的基本运营方法，本节中我们来看看如何从更宏观的商业角度理解这个平台。

## 从财报看微博现状 📈

了解一个平台，分析其公司财报是非常有利的一环。这套课程确实融入了很多商业理解的内容。

根据2022年第一季度微博的财报，可以总结出以下信息：
*   **MAU（月活跃用户）**和**DAU（日活跃用户）**仍在上升。
*   但**用户的参与度**在降低。

时至今日，微博的用户体量上升，主要受春节假日、冬奥会及娱乐圈爆点等大事件影响。然而，用户使用微博的深度其实在降低。

因此，可以理解为微博正逐渐演变为一个**热点事件围观**的新闻平台和搜索平台。微博热搜的地位的确稳固。而用户参与度和使用深度的降低，必然导致其社交媒体IP属性的减弱，并引发广告收入缩水，品牌方在微博博主身上的投入也会减少。

从营收结构来看，微博虽然在直播、短视频、电商领域都有所尝试，但依然**非常依赖于广告业务**。微博的营收结构中，几乎九成来自广告。

稍有财报常识的同学都应知道，**收入模式单一**意味着平台的商业抗风险能力较差。从资本层面看，微博的估值走向不容乐观。

## 博主如何应用财报信息 🔍

那么，从博主角度应如何吸收这些财报信息，并应用于自己的IP运作中呢？以下是两点总结：

1.  **抓住热点事件引流**：在任何大型社会事件发生时，微博上的讨论一定会很热烈，微博热搜的地位依然稳固。因此，抓住时机蹭这些热点型事件引流，效果应该非常好。
2.  **分散商务风险**：不要把微博商务作为自己IP商务的唯一发力点。客观上，我并不看好绝大多数近两年起盘的IP在微博上的长期商务发展趋势。至少不能把所有鸡蛋放在这一个篮子里。

## 那些大V教我的事 👥

私下里，我认识较多微博大V，并与他们有很多交流。我的感觉是，只要是微博大V，他们一般不会随大流去唱衰微博。他们当然希望微博声量越来越大、发展越来越好，这作为既得利益者可以理解。当然，他们提供的很多信息也很客观。以下是几点供大家参考：

1.  **个人门面与影响力平台**：时至今日，微博依然是一个非常好的做个人门面和影响力的平台。像“微博红人节”、“微博之夜”这类活动，在业内影响力非常大。
2.  **圈子与平台关系**：微博是一个相对比较混圈子和混平台关系的平台。如果能链接到一些区域直接的运营负责人，那么账号的起势会比较明显。这一点比较敏感，我们点到为止。
3.  **老牌大V的商务现状**：老牌微博大V的商务影响，在这一两年并不是很大。因为很多品牌在微博的复投率较高，他们会去复投那些熟悉的博主。相比之下，像小红书，很多品牌和博主不管效果好坏，可能只合作一轮。因此，在微博一旦达成信任关系，持续的合作会比较轻松。很多新兴消费品牌可能着力点在抖音、小红书，但很多传统大品牌每年仍有相当多预算投放在微博，且预算较为充足。总的来说，微博大V的商务并未因近两年对微博广告的唱衰而受到太大影响。以上都是我得到的一些信息，供大家参考。
