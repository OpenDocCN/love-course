# 社交恐惧必修课：第五节：社交时出现恐惧，快速的恢复好状态的方法

在本节课中，我们将要学习当社交场合中恐惧感突然袭来时，如何快速调整并恢复状态。课程将分为两个主要部分：首先，深入理解什么是“社交时的恐惧”；其次，掌握一个清晰、可操作的公式来应对它。

## 第一部分：社交时的恐惧到底是什么？

上一节我们探讨了社交前的预期焦虑，本节中我们来看看在社交进行时发生的恐惧。

所谓社交时的恐惧，指的是社交恐惧症患者在身处社交场合时，所产生的强烈紧张感或恐惧感。此时，个体很容易出现脑袋空白、语无伦次、不知所措的表现。我们会因为这些表现而产生屈辱感、自责和自我否定，并因此产生强烈的回避行为。

为了更好地理解，我们可以拆解其关键词：
*   **场景**：出现在**社交场合**时。
*   **身体表现**：脑袋空白、语无伦次、不知所措。
*   **情绪表现**：产生屈辱感、自责、自我否定。
*   **行为表现**：产生回避行为。

**举例说明**：
上学第一天，在教室集合，老师要求做自我介绍。看着同学们一个个上台，我预期焦虑非常强烈。轮到我上台后，我表现出不知所措、表情僵硬、不敢对视、脸红等症状。由于症状出现，我无法正常表现，只是匆匆说了句“大家好，我叫某某某”就下台了。这次经历让我在全班同学面前展示了一个紧张、不自然的形象，感到非常痛苦。这就是典型的社交时的恐惧。

接下来，我们总结一下社交时的恐惧具备哪些特点。

以下是社交时的恐惧的四个核心特点：

1.  **负面情绪极其强烈**：当恐惧出现时，大量负面情绪会迅速席卷全身，不仅包括心理上的紧张恐惧，还可能伴随脸红、身体僵硬等躯体反应。这种情绪的强烈程度通常超过了社交前的预期焦虑和社交后的抑郁。
2.  **大脑出现逻辑性缺失**：当症状出现时，我们的所有注意力都会被症状本身占据，无法分配到当下该做的事情上，导致大脑缺乏逻辑性。又因为社交中必须表达点什么，从而出现语无伦次的问题。
3.  **出现强烈的对抗行为**：恐惧带来安全感的缺失，我们会下意识地强烈对抗出现的症状（如紧张、脸红），试图快速摆脱其束缚，但这种对抗往往只会加剧紧张。
4.  **出现回避行为**：与对抗行为同时出现的，是逃避或回避行为。例如，在症状出现时，草草结束对话或匆匆离开现场。

## 第二部分：解决社交时的恐惧的公式是什么？

理解了社交时的恐惧后，我们来看如何应对。我们将通过一个公式化的方法来提供清晰的解决路径。

### 为什么社交时的恐惧“最难解决”但也“可以不解决”？

在介绍公式前，需要明确一个关键认知：社交时的恐惧是社交恐惧三个场景（前、中、后）中最难解决的，但核心的解决重点却可以不在它身上。

**1. 为何“最难解决”？**
通过与社交前焦虑、社交后抑郁的对比，可以从四个维度看出：

*   **负面情绪影响程度**：社交时的恐惧带来的负面情绪最强烈。
*   **对抗症状的程度**：在恐惧发生时，我们想要消灭症状的愿望（对抗程度）最强烈。
*   **自我察觉度**：在强烈的恐惧中，我们很容易一味逃避和对抗，难以保持“看到”自己陷入症状的清醒（自我察觉度低）。
*   **可被解决的难易程度**：由于以上三点（症状影响大、对抗程度高、自我察觉度低），社交时的恐惧最难直接解决。

**2. 为何“可以不解决”？**
这基于两个重要原因：
*   **人人都有**：普通人面对压力场合（如上台）也会出现紧张恐惧，但他们不会因此形成严重的社交恐惧，关键在于他们**不会在社交前过度预期，也不会在社交后反复纠结**。
*   **解决核心在于“前后”而非“中间”**：社交恐惧症状的形成和强化，主要源于**社交前的过度预期（焦虑）**和**社交后的反复回忆（抑郁）**。因此，解决的重点应放在这两端。如果能处理好“前后”，社交时的恐惧感自然会减轻。当然，若能直接缓解当时的恐惧也很好，但这并非必需，也较难强求。

### 解决社交时的恐惧公式

尽管重点在“前后”，但若我们希望在恐惧来袭时能有所应对，以下公式可以提供帮助。**前提是，恐惧没有强烈到完全失控（如完全大脑空白）的地步**。如果已完全失控，接纳现状并优先处理“前后”的焦虑和抑郁即可。

公式分为三步：**1. 意识到自己陷入恐惧 -> 2. 理解恐惧的合理性 -> 3. 带着恐惧投入当下**。

#### 第一步：社交时出现恐惧，别人怎么看？（改变认知）

我们之所以恐惧，往往源于担心他人对我们的负面看法。改变这一认知是关键。

**核心概念：投射思维**
这是一种心理防御机制，指我们会**把自己的想法投射到别人身上，认为别人跟自己想的一样**。
*   **代码描述**：`我的想法 == 别人的想法`
*   例如：我对自己紧张表现是排斥和否定的，因此我也认为别人发现了我的紧张，并且同样讨厌我的紧张。但这只是主观的投射。

**别人真实的看法是什么？** 可以从四个维度理解：
1.  **情绪的发出人是谁**：紧张情绪的发出者是你自己，你对它最敏感。旁人并非发出者，对其感知非常低。
2.  **别人是否把充分注意力放在你身上**：根据“焦点效应”，人们大部分注意力都在自己身上。除非你极其特别，否则旁人不会持续高度关注你，因此也难以细致察觉你的紧张。
3.  **对方对你表现的态度**：紧张、不自然是普通人都会有的情绪表现，属于正常范畴。因此，旁人的态度通常是**可以理解、接纳**的，除非你表现出与环境严重不符的怪异行为。
4.  **态度的例证**：当你问一个普通人“如何看待说话紧张的人”，常见回答是：“这很正常啊，每个人都会紧张。” 这印证了紧张是一种被普遍接纳的情绪。

#### 第二步：意识到自己陷入恐惧，并理解其合理性

**1. 意识到自己陷入恐惧（自我察觉）**
这不是评价，而是“看到”自己的状态。判断标准很简单：
*   反复对抗自己的恐惧感，想摆脱它。
*   出现强烈的恐惧等负面情绪。
*   有强烈的想要回避当前社交场合的倾向。
意识到“我此刻陷入恐惧了”，就能带来一丝清醒，为下一步行动创造条件。

**2. 理解恐惧和紧张的合理性**
告诉自己：“我之所以恐惧/紧张，是因为我有社交恐惧的问题/因为这个场合让我有压力，这是正常的反应。” 理解其合理性，可以**减少主观对抗，提高自我接纳**。虽然恐惧感可能仍在，但我们不再与之激烈斗争。

#### 第三步：带着恐惧去投入当下

在意识到并理解恐惧后，最关键的步骤是：**带着这份恐惧感，继续做你该做的事情**。
*   **目标不是消除恐惧**，也不是追求完美的社交表现。
*   **目标是尽可能将注意力放在当下的对话或任务上**，而不是关注症状是否消失。
*   即使效率很低，表现不佳，也没关系。**允许自己表现不完美**，紧张时视线可以移开，缓一缓再继续。
*   **核心原则**：不要求自己100%投入，只是“尽可能”去做。越是“不要求”，反而越容易投入。

**案例演示**：
一位有对视恐惧的学员在社交时症状出现。他：
1.  **意识到**：发现自己陷入恐惧，不敢对视，并告诉自己不应强迫自己敢对视。
2.  **理解**：对自己说：“我有对视恐惧，现在不敢对视是正常的。” 接纳了这个事实。
3.  **投入**：带着不自然的状态继续聊天，注意力放在聊天内容上。紧张得受不了时就暂时移开视线，稍微放松后再尝试对视。过程中没有关注症状是否消失。
结果，他后续的聊天反而变得愉快，恐惧感也随之减轻。

**重要提醒**：
此方法适用于恐惧程度较轻或中等时。如果恐惧极其严重，无法坚持，**允许自己离开或回避，并且不要责怪自己**。解决症状的长期重点，依然在于管理好社交前的焦虑和社交后的抑郁。

## 本节课总结

本节课中我们一起学习了：
1.  **社交时的恐惧**的定义与特点：它是社交场合中爆发的强烈紧张感，伴随逻辑缺失、对抗与回避行为。
2.  **应对策略的核心认知**：社交时的恐惧虽难直接解决，但社交恐惧的解决核心在于管理好**社交前的预期焦虑**和**社交后的反复抑郁**。
3.  **快速恢复状态的公式**：当恐惧来袭，可以遵循 **“察觉 -> 理解 -> 投入”** 三步法。关键是改变对他人看法的错误认知（投射思维），接纳恐惧的合理性，并带着恐惧将注意力转向当下。

**课后作业**：
请将学到的解决社交时恐惧的公式，运用到实际生活中。一旦在社交场合感到恐惧，就尝试按步骤使用它。反复练习，但不必强求每次都能完美做到或100%投入。坚持不下去时，允许自己暂停，切勿自责。

通过持续的练习，你会逐渐发现，社交时的恐惧并没有想象中那么可怕，你对自己状态的控制力也会不断增强。
