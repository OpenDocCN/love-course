# 一门给年轻人的恋爱成长课：33：戒掉取悦症：拒绝无底线的对一个人好 🚫

在本节课中，我们将要学习“讨好型人格”或“取悦症”的核心表现、成因以及如何有效戒除它。这种人格模式会让人在关系中感到疲惫且难以获得真正的亲密感。

## 概述

讨好型人格的同学总是非常努力地做一件事，就是拼命对别人好，认为这样能让别人喜欢自己，能让自己受欢迎。结果却是从始至终，没有一个人真正地爱他在意他。这是一个很简单的真理：当一件事情越努力越起到反效果的时候，往往就在提醒我们，我们用错了方法，走错了方向。

上节课我们讲恋爱脑时谈到了一个悖论。恋爱脑为恋爱付出了很多努力，进行了几乎全部的资源倾斜，但结果却事与愿违，越努力越不顺利，越不被爱。讨好型人格也明显有这个表现。

## 讨好型人格的特点

以下是讨好型人格的四个典型特点。

### 1. 过度高估他人的重要性

讨好型人格一直在讨好别人，因为他想要被喜欢。被别人喜欢，甚至带着卑微的好，从逻辑上很容易说通，因为我们都认可被他人喜欢欣赏是重要的。人和人的关系具有相互性，对方是个活生生的人，所以这么做有用。我们不会去在乎一只蚂蚁喜不喜欢我们，那是因为蚂蚁没有这种相互性。

因此，我们得出他人是重要的，只是有多重要在这里有区别。一个路人喜不喜欢你，或者你的父母、伴侣喜不喜欢你，哪个更重要？显然是父母和伴侣。我们会觉得路人“关我屁事”。而对于一个讨好型人格的人来说，这就“关我的事”了。他们对于别人以及别人的喜欢这件事的重要程度的判断，远远超过大部分的普通人。他们会认为别人比自己更重要。所以，满足别人的需求比满足自己的需求更重要。

当这种需求有冲突时，他们会毫不犹豫地委屈自己成全别人，也就是讨好了。例如，你想在家里追剧，然后你的另一半想让你陪他出门，这个时候冲突了。讨好型人格的人一定会压抑自己的需求来满足陪对方出门这个需求，因为他觉得对方的需求更重要。再比如，你谈恋爱时，男朋友提出要同居怎么办？很多人会说“其实我不想，但是我害怕我拒绝对方会生气，会觉得是我不够爱他”。这同样是成全别人，委屈自己。

### 2. 只会用“讨好”这一种行为模式

人为什么会喜欢上另外一个人？想一下你为什么喜欢某个明星，是因为他有某些特质吸引到了你，可以是他很酷、他人品很好，或者他演技好等等。那些特质在他们身上是一种硬邦邦的存在。我们平常喜欢一个人，其实也是这样，还是因为你有某些特质吸引到了我。

而对于讨好型人格来说，会认为“我要获得你的喜欢，我就要把自己打磨成一个我认为你会喜欢的样子”。注意，是“我认为”你会喜欢的样子。这看起来也特别深情。但是在这种行为之后的结果，就是对方可能并不喜欢这个样子，或者说不喜欢你要变成这个样子的过程。讨好型人格只会用讨好的方式，就是“我要做什么让他喜欢我”，而不是“我要做什么，让我成为被喜欢的人”。

这两句话看起来是一个意思，但其中有微妙的差别。作为被讨好的一方来说，如果一个人身上本身就具备某种我喜欢的特质，我想要得到他，这是我主动“求证你”。而一个人本身不具备我喜欢的特质，为了让我喜欢，于是他去改变去获得这种特质，但这却是你主动“求证我”。这种微妙的区别是真实存在的。

我相信你可以感受到，这就是为什么上面我说这个过程他不一定让人喜欢。再退一步说，这种改变通常也不可能做到尽善尽美。例如，我说我喜欢一个温柔的人，于是对方为了我去变得温柔。他其实很难达到一个本身就温柔的人的那种温柔。你说你喜欢有少年感的人，于是我去穿白衬衫，可能最后少年感不一定出得来，但看起来还很像一个房产中介。所以这个时候，一方面是你求着我，另一方面我又没有办法真的去达到你想要的样子的时候，从被讨好方的视角来看，就比较沮丧了。

讨好人格就像是一块铁，他被磁铁吸住，跟着磁铁在后面跑，和自己本身就是那块磁铁，所以它能够吸引到所有的铁的差别一样。另外还有一个扎心的真相是，讨好其实是佐证了对方的相对优秀，是因为我们觉得对方足够的好，我们才去讨好。而这一点对方心里是可以隐约感觉到的。这个有点像我们之前讲的权力博弈。所以才会出现那很常见的现象：我们对一个人越好，反而有可能越不被珍惜。其实根本问题不出在“对人好”这件事情上，而是出在“好”这个字上。所以真相是什么？真相就是：**一个人对你好，不是因为他好，是因为你好。**

### 3. 不敢直面冲突，回避深度关系

这和上面不一样。他倒不是说为了要获得别人的喜欢才去讨好，是因为他害怕冲突。“我怕别人讨厌我，我怕别人生我气”。我们暂且先不说别人生气或者讨厌会造成多么严重的后果了。姑且就按这个逻辑来说，有冲突、有生气、有坏情绪，就一定破坏关系吗？

不一定。我们讲吵架那一期讲过，能够合理地表达负面情绪，是能够拉近关系的，意味着两个人的关系更真实跟亲近。想象一下，我们是能够在老板面前发火，还是能够在闺蜜兄弟面前发火？我们在老板面前只能戴着面具做人。正因为这个面具，所以不存在什么深度交流。大家不是朋友。要深度交流，就需要更直接地去表达情绪，尤其是像愤怒、悲伤这种负面情绪。

讲三元素的时候，我们也讲过，很多的走心对话，他未必是愉悦的。相反，那些不太愉悦的话题或者情绪，更容易走向深度交流。而这种适当的、恰当的低能量的沟通，不仅不会破坏人际关系，反而会增进关系。每一次的真实的表达，也意味着能够让对方更多了解你一些。

那么反过来说，怕冲突，怕暴露负面情绪，才不得不去讨好别人，也就意味着他没有给别人留机会来和他们增进关系。他们看起来是在讨好别人，但其实也是拿一个保护罩把自己罩起来。别人虽然没有办法去和他们互相伤害，但也没有办法走得更近了。这就是为什么我们总是发现，那些讨好型人格的人活的不仅累，而且也很不开心，看上去是别人眼中的好人，但实际上其实没有什么人和他们真正地亲近。

### 4. 隐形攻击

这部分更像是前面一切导向的一个结果。讨好型人格也不是圣人，他们无底线地对别人好，归根结底，也是希望别人能反过来对他们好。但是问题来了，当他们发现“我对别人好，但别人反过来对我不好的时候”，我会怎么样？打心底里，我会感觉到失望、委屈、愤怒。但同时因为那个保护罩，他会形成人格，我又没办法表达这种愤怒，甚至我都不太愿意承认我因为这件事而愤怒了。于是我会强行把这种怨气压下去，压制住，不让他表达出来。

可问题是能完全压下去吗？总会露出一点来，而那一点就会变成让人非常别扭的“隐形攻击”或者叫“被动攻击”了。被动攻击其实是心理上的一种应对机制。当我们害怕自己的坏情绪会导致不好的人际结果的时候，这种应对机制就启动了，然后就会表现出一种阳奉阴违的意思，也显得这个人很挣扎。

这里可能会有人觉得，那隐形攻击和冷暴力是不是一回事？澄清一下，是不一样的。我对你冷暴力的时候，我是知道我在用这种方式故意气你、伤害你。但是隐形攻击不是的。发起隐形攻击的人，他其实不知道他实际上是在释放攻击性。他从打心底里不高兴，但是因为他不想面对冲突，他又不能明确地说出来。所以这种保护机制，这种抗议就产生了。

## 讨好型人格的成因

说完了表现，那为什么有些人他就是会成为一个讨好型人格？根据上面的这一连串的分析，我们也能明白，讨好型人格的根源，在于一个根深蒂固的认知：**他们相信想要获得别人的喜欢，有且只有一种方式，就是对别人好。我对别人好，别人就应该对我好。如果别人对我不好，那就是我有问题，我做的还不够，我应该更努力地去讨好别人。**

你会发现这个认知的背后，是他们对自己的要求非常高，同时他们对别人的期待也非常高。再加上他们没有表达负面情绪的能力，于是这部分负面情绪会转而的绕个弯来攻击自己，最后变成一句“都怪我不好”。然后陷入一种恶性循环：“所以我要对别人更好，以此换得别人的喜欢。”

我们听到这里就能够意识到，这个认知根本上他有很大的弊端。就是被喜欢，他不一定要通过讨好这件事来获得。就是你对一些人好，那再好吧，他们可能也未必对我们好。退一步说，即使别人愿意对我们好，对方表达的方式也不一定是我们想要的。就像我在恋爱中，我经常会说，其实不打扰我也是一种对我好。很多人不明白，他觉得一直就要得给我点外卖、送药送汤之类的，才叫对我好，对吧？这就是表达方式上面的不同。

所以说到底，讨好型人格其实是一种认知上的误区。

## 如何戒掉讨好型人格

既然是认知误区，我们也应该从认知层面来解决。所以如何戒掉讨好？

### 1. 摆脱“被认可”的执念

人都需要被认可，被认可很重要。但是什么叫执念？就是“不被认可就不行”。那到底是不是这样呢？不一定。我们要接受我们不是神，世界上不可能每个人都会喜欢我们。对于一个不是讨好型人格的人来说，这句话就是废话。但是对于一个讨好型人格来说，这句话其实还挺有挑战性的，不然也不会被叫成执念了。

有没有可以实行的方法来帮我们去做到？这里我觉得有一件事可能会非常有意义。我刚开始拍视频的时候，大家如果看过的话，知道我拍过一些很傻的小剧场，就对着镜头装疯卖傻。要知道我平时其实还是一个比较正常的人。而那个时候我就反复告诉自己一句话，就是“没有人会因为我的视频而记得我”。真的，就这句话。言下之意，就是我怎么疯，说到底，不过就是个做视频的而已，不要把自己看太重。于是我发现，管用。我确实比较自如地做出了那些小剧场，运气也不错，大家还挺喜欢这些东西。无形中我就这么做的。

直到我后来看到了一个方法，他把我这个过程说的更详细也更具有实操性了，我才明白为什么我这么做会管用。这个方法就是：**我先假设某一个人他就是不喜欢我的，不认可我的。你在脑海中去想这个事情，想一下你身边的某个人，比如某个同事或者某个中意的对象吧，他不喜欢你。在脑海中去强化这件事，然后去感受这是一种什么样的心情。如果可以，你不妨可以把这些感受记录下来写出来，然后聚焦的去看那几行字，甚至大声读出来。**

这个过程中你会感觉到沮丧、难过、不开心。然后当你感觉过这一系列的负面感受后，你会发现，哦，不就是这样吗？最差最差，也就是我现在最坏的感觉。然后我们就会自动地在心里问出“所以呢？”以及给出答案：“其实没所谓了。”然后我们就会突然地发现，我并不需要世界上的每一个人都认可我。对方如果不认可我，并不代表我就是一个糟糕的人。

注意，这个过程是全自动的，很神奇。通过去模拟那种至暗时刻的感受，你就会感觉到当一个人不认可你的时候，也没啥大不了，你就会释然，会身心舒畅，会特别自由。我们可以尝试着在不同的人身上去做这个练习，从无关紧要的同事朋友开始，然后甚至到亲人爱人等等。我认为这个方法是属于那种只要你去做了，就会产生效果的好方法。当然我们在执行的过程里面，我们假设的那个人的不同，或给我们带来的情绪体验不一样。有些人无关痛痒，有些人带来的冲击力更大，没关系，这个练习的意义就在于“让暴风雨来吧”。

### 2. 打破“非要做好人”的执念

这里的“好人”我们就打上引号。我们不是说道德上的好坏了，就“做好人”是讨好型人格的一种标准执念。但是我们没有办法时时刻刻都在每一个人那里当一个“好人”。同样，这对于不是讨好型人格的人来说是很容易接受的。但讨好型人格接受不了：“怎么能不当个好人呢？我想象不了。”他们有可能不知道自己除了当个好人以外，还有什么其他价值。

那或许我们就可以来找一找，除了做个好人，我们还有什么好的地方。这也是《取悦症》这本书里面提到的一个方法：**就是试着列出十条能够描述自己的正面词汇，漂亮、聪明、坚强、乐观、努力都可以，但是不要出现“好”或者“懂事”这样的词。** 如果你实在想不出来，也可以拉着朋友帮你一起想。

这个时候我们就会发现，哎，除了当好人以外，还有挺多词可以拿来形容我的啊，我的优点也挺多，我的价值不仅仅是做个好人。就算偶尔没当成好人，我也可以拥有其他的价值和魅力。

而进阶的做法就是：既然我们现在已经拥有这么多优点了，那以后我可以在每一次拿出一段时间，挑出一个关键词，然后只去扮演这个关键词。比如我今天这两个小时我就是要去扮演“努力”和“勤奋”。那么如果有朋友因为聚会要叫我出去，怎么样？为了扮演努力勤奋，我就必须把工作学习忙完了才会去。这个时候我们没有做到朋友心中的“好人”，但是我们做了那个“努力勤奋”的人，我依然很棒，我依然很有价值。

这种尝试多做几次下来，你就会发现，其实我们身上还有很多其他的魅力点，还有其他可以让人喜欢的地方，而不仅仅是去讨好才能获得别人的喜欢。

### 3. 消除“应该”讨好别人的自我要求

前面也提到了，讨好型人格的内在逻辑就这样：“我应该对别人好，同时别人应该对我好”。最关键的一个关键词在于“应该”。“应该”是什么？应该是一种自我要求，不做就是不对的，就是对我们这个人的否定。

那我们现在去掉这个词会变成什么？就是没有应该，没有要求，没有对错。我们要把“应该对别人好”这概念替换成“如果我愿意的话，我可以对身边的人好”。同样，“如果他们愿意的话，他们也可以来对我好”。怎么样？发现区别了吗？他不再是一件理所应当的事情，不再是我们对自己的硬性要求。反而我们开始注重自己的意愿，把自己的意愿摆在前面。

“如果我愿意，我可以对别人好。”这句话背后的潜台词是什么？是“如果我不愿意，我也可以不这么做”。同样，别人不愿意，别人也可以不这么做。哇，这是不轻松很多了？就是抱着这种认知，我们就不会对自己和他人抱有过高的期望。别人愿意的话，自然也会对我们好。但毕竟也有不方便的时候吧，也有get不到我们需求的时候。所以这叫什么？这叫包容自己，也包容别人，也是一种人际关系里面的松弛感。

### 4. 爱自己

OK，这个词很土，爱自己。如果你觉得比较虚，那也可以理解成就是“对自己好一点”。我能够想象，很多习惯于讨好别人的人，几乎已经快要忘记怎么对自己好了，忘了做什么事情才叫做善待自己。

那怎么做才叫做爱自己？其实就是在尊重的前提下，多想想自己的感受，而不是对方的感受。不要总是以对方的感受为中心，别人说什么都对的，就去附和，所以就讨好嘛。要多问问自己是什么感受，以及自己想要什么样的感受。

我特别记忆犹新的一个例子。我朋友顺路载我的另外一个朋友一定要回去，但是被载的那个人的家里更远。也就是说我这个朋友要送他回去，就得绕远路。这两个人就普通朋友，没有什么这种异性的感情夹杂在里面。这个时候如果他是一个讨好型人格，他一定会选择绕路，对吧？一定会送他回去。

但是我这个朋友当时说了一段话，他说：“对不起，我可能没办法送你回去了。倒不是因为别的，就是因为绕这段路是很麻烦的。如果我这么做了，我回来的这段路上，我自己肯定会不开心。我不想我不开心，所以我希望你能理解。”然后怎么样？这个朋友一下就理解了。相反，他觉得这个人很真实，说话的分寸拿捏的也不错，完全没有影响到两个人的关系。这就叫要尊重自己的感受，对自己好一点，或者说更好一点。

我真的觉得这位朋友做了一个完美的案例。因为后来我发现这个剧情可以套用在任何一件事情上。这个剧情就是：**“抱歉，我可能没办法怎么怎么样。因为什么什么，如果我这么做了，我肯定会啊有某种不好的感受。我不想这样，希望你能理解。”** 这个看起来有点像完形填空，这个中间的空，大家可以随便填。我相信这个简单的剧情是可以帮大家在日后遇到这种问题的时候，非常舒服地过渡过去的。

而且这个剧情倒不一定要用在互动里面。就很多时候我们做一件事情特别难受的时候，我们可以用这句话来自我梳理，对吧？我能不能做一件事情，他会给我一种什么样的感觉，我喜不喜欢这种感觉。如果不喜欢，我是否应该这么做？如果喜欢，我是否应该这么做？就像比如说购物，这不是人际关系，我们经常说“千金难买心头好”，贵一点，可能在外人看来，你这被割韭菜啊，你花冤枉钱啊，但是值与不值，我们自己心里清楚。

所以，1234，我就不信，这四点做到还救不回一个讨好型人格。

## 总结

本节课中我们一起学习了如何识别并戒除讨好型人格。

首先，我们讲了讨好型人格的四个特点：
1.  过度高看他人的重要性。
2.  只会用“讨好”这一种行为模式。
3.  不敢直接面对冲突，回避深度关系。
4.  隐形攻击。

其次，我们分析了为什么有的人会成为讨好型人格。根源在于一个根深蒂固的认知：他们相信想要获得别人的喜欢，有且只有一种办法，就是对别人好。他们对自己的要求非常高，同时也对别人的期待非常高，于是总是做出这种事与愿违的讨好。

最后，我们详细探讨了如何戒掉讨好：
1.  **摆脱被认可的执念**：通过模拟“至暗时刻”的感受，体会不被认可也没什么大不了，从而做到释然。
2.  **打破“做好人”的执念**：尝试列出十条描述自己的正面词汇（不含“好”或“懂事”），并尝试在特定时段扮演某个正面特质，发现不做“好人”也没什么大不了。
3.  **消除“应该”的自我要求**：用“如果我愿意，我可以…”替换“我应该…”，重塑自我认知，建立人际关系的松弛感。
4.  **爱自己**：在尊重的前提下，多关注自己的感受。我们提供了一个实用的句型来帮助表达和自我梳理：**“对不起，我可能没办法…因为…如果我这么做了，我肯定会产生某种感受。我不想拥有这种感受，所以希望你能理解。”**

严格地说，讨好型人格不算是一个心理学词汇，学术上比较相近的应该叫做“取悦症”，也叫做“看管型人格障碍”，就是说习惯于取悦他人者对拒绝和敌意有着根深蒂固的焦虑和抗拒，所以只考虑他人而忽略自己。

希望这节课的内容能帮助你建立更健康、更平等的人际关系。
