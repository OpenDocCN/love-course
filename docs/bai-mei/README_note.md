# 白魅教程

## 概述
在本节课中，我们将学习如何使用白魅进行编程。白魅是一种基于Python的编程语言，它简单易学，适合初学者。

## 安装白魅
首先，您需要安装白魅。以下是安装步骤：

1. 下载白魅安装包：[白魅安装包下载链接](https://www.example.com/download)
2. 解压安装包
3. 运行安装程序

## 基本语法
以下是白魅的基本语法：

```python
# 定义变量
name = "白魅"

# 输出变量
print(name)
```

## 数据类型
白魅支持多种数据类型，包括：

- **字符串**：用于存储文本，例如 `"Hello, World!"`
- **整数**：用于存储数字，例如 `42`
- **浮点数**：用于存储小数，例如 `3.14`
- **布尔值**：用于表示真或假，例如 `True` 或 `False`

## 控制流
白魅支持条件语句和循环语句。

### 条件语句
```python
# 如果条件为真，则执行代码块
if name == "白魅":
    print("欢迎来到白魅的世界！")
```

### 循环语句
```python
# 循环执行代码块
for i in range(5):
    print(i)
```

## 函数
函数是代码块，可以重复调用。

```python
# 定义函数
def say_hello():
    print("Hello, World!")

# 调用函数
say_hello()
```

## 总结
本节课中，我们一起学习了白魅的基本语法、数据类型、控制流和函数。希望您已经掌握了这些基础知识。
