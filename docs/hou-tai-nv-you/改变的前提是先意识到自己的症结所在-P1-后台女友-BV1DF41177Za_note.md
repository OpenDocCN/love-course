在本节课中，我们将要学习如何识别阻碍个人成长与改变的几个核心心理特征。清晰地认识这些“症结”，是迈向积极改变的第一步。

## 识别五大核心特征 🔍

上一节我们介绍了自我认知的重要性，本节中我们来看看具体需要关注的五个特征。以下是五个常见的、可能阻碍你改变的心理与行为模式：

1.  **迎合他人，掩饰自我**
    第一个特征是为了迎合他人的喜好，来掩饰自己真实的需求。

2.  **过度在意他人评价**
    第二点是内心敏感脆弱，非常在意他人对自己的看法。

3.  **缺乏界限，难以拒绝**
    第三点是缺乏界限感，很少说不，不懂得如何拒绝别人。

4.  **依赖外界认可获取能量**
    第四点是自身无法产生能量。需要靠外界的认可来获得自信。

5.  **进行无效的横向对比**
    第五点是对自己的要求标准，不是来自于和过去的自己对比，而是来自于和外界的对比。

## 行动触发点与决心 💪

了解了这些特征后，我们来看看如何将其转化为行动指南。

以下是判断是否需要立即行动的标准：
*   如果你满足了上述三个以上的特征，就需要决心寻求变化了。

决心意味着坚定地说“不”，拒绝旧有模式的干扰，并为自己的改变负责。其核心行动可以概括为：**`assert(“No to old patterns”)`**
