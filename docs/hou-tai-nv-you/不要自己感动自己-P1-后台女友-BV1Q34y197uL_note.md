## 📖 概述

在本节课中，我们将要学习如何区分“深情”与“讨好”这两种行为模式。理解它们的本质差异，对于建立健康、平等的人际关系至关重要。

## 🔍 核心概念解析

上一节我们介绍了本课程的主题，本节中我们来看看这两个核心概念的具体定义。

深情是在维护自我尊严的前提下，真诚地对他人好。其核心公式可表示为：

**深情 = 真诚的付出 + 自我尊严的保持**

讨好则是放弃自我尊严与底线，一味地去迎合和满足他人的喜好与要求。其行为模式可描述为：

```python
# 讨好的行为模式
while 对方有要求：
    无条件满足()
    自我尊严 -= 1
```

## ⚖️ 两种行为的结果对比

理解了定义之后，我们来看看这两种不同行为模式会带来怎样截然不同的结果。

以下是两种行为模式的后果对比：

*   **深情的行为**：能够换来对方平等的爱与尊重。
*   **讨好的行为**：只会换来一个既看不起你，又想利用你的人。

## 💡 错误归因与正确认知

看到不同的结果，我们很容易产生困惑。许多人最终变得愤懑不平，责怪“深情换不来真情”。

然而，问题的根源往往在于最初的混淆。这其实是一开始就错误地将“讨好”当成了“深情”去使用，方法错了，自然无法得到期望的结果，这怪不得别人。

## 🎯 总结

本节课中我们一起学习了“深情”与“讨好”的关键区别。请记住，真正的深情伴随着自尊，而失去底线的讨好只会损害关系。认清这一点，是走向平等健康关系的第一步。
