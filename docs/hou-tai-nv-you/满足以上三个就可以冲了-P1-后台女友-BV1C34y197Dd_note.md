# 课程名称：识别好感信号 🎯 - 课程编号：P1

在本节课中，我们将学习如何识别他人对你可能怀有好感的五个关键非语言信号。理解这些信号有助于你在社交互动中把握时机。

## 识别好感信号 🕵️♂️

上一节我们介绍了课程目标，本节中我们来看看具体的识别方法。当一个人对你有好感时，其行为模式常会发生变化，并释放出一些可观察的信号。

以下是五个核心的好感信号列表：

1.  **频繁的眼神接触**：对方的目光会经常落在你的身上。
2.  **保持近距离**：对方在你身边时，会一直保持在社交距离范围之内。这是一种希望亲近的信号。
3.  **处于你的视线内**：对方会故意站在你轻易就能看到的地方，不会离开你的视线太久。
4.  **整理仪容与夸张动作**：对方可能会摆弄头发、整理衣服，或做一些稍显夸张的动作，旨在吸引你的注意。
5.  **提高互动可见度**：对方与朋友交谈时，会提高音量、更爱笑，并且很在意你是否听到以及作何反应。

## 信号背后的含义与行动建议 💡

上一节我们列举了具体信号，本节中我们来分析其含义。通常，如果对方对你有好感，他/她主动直接搭讪的概率可能较小。

但是，对方会通过给出上述反应和信号，来**邀请你主动接近**。

如果在同一空间里，女生对你展现出这些信号，那就代表你可以尝试着去接近她了。

---

本节课中我们一起学习了识别他人好感的五个非语言信号：频繁眼神接触、保持近距离、处于视线范围内、整理仪容与夸张动作、提高互动可见度。理解这些信号能帮助你更好地解读社交意图，并在适当时机采取行动。
