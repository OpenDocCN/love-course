在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如数字、文字）。你给盒子起的名字就是**变量名**，而盒子里装的东西就是**变量值**。

在Python中，你使用等号（`=`）来给变量赋值。其核心操作可以表示为以下公式：

**`变量名 = 变量值`**

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：

```python
message = "你好，世界！"
```

## 基本数据类型 🔢

理解了如何创建变量后，我们需要知道变量里可以存放哪些不同类型的数据。Python有一些内置的基本数据类型，最常见的有以下几种：

以下是三种最常用的基本数据类型：

1.  **整数（int）**
    代表没有小数部分的数字，可以是正数、负数或零。
    例如：`age = 25` 或 `temperature = -10`

2.  **浮点数（float）**
    代表带有小数点的数字。
    例如：`price = 19.99` 或 `pi = 3.14159`

3.  **字符串（str）**
    代表文本信息。在Python中，文本必须用单引号（`'`）或双引号（`"`）括起来。
    例如：`name = "小明"` 或 `greeting = 'Hello'`

## 查看与使用变量 👀

创建了变量并存放了数据之后，我们如何查看和使用它们呢？

你可以直接使用 `print()` 函数来打印出变量的值。同时，变量之间可以进行基本的运算。

以下是查看和使用变量的方法：

*   使用 `print(变量名)` 来输出变量的值。
*   对数值型变量（整数、浮点数）进行加（`+`）、减（`-`）、乘（`*`）、除（`/`）等数学运算。
*   对字符串变量使用加号（`+`）进行拼接。

让我们通过一段代码来实践一下：

```python
# 定义几个变量
name = "李雷"
score_math = 95
score_english = 88.5

# 打印变量
print(name)               # 输出：李雷
print(score_math)         # 输出：95

# 进行数学运算
total_score = score_math + score_english
print(total_score)        # 输出：183.5

# 进行字符串拼接
introduction = "我是" + name
print(introduction)       # 输出：我是李雷
```
