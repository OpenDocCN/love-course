在上一阶段的约会课程中，我们了解到约会是提升两人关系流程的最佳时机。约会相处之后，还有一个关键性的技巧可以运用，那就是**肢体接触**。你们需要开始进行“肉体的碰撞”。

我们把肢体接触分为五个层次：
1.  非敏感接触
2.  接触
3.  牵手
4.  接吻
5.  纳杀（发生亲密关系）

接下来，我将为大家演示这些阶段的具体做法。

### 第一阶段：非敏感接触 👐

首先，我们要进行的是**非敏感接触**。这指的是尝试碰触女生的非敏感部位，例如头发。

以下是操作方法：

你需要找到一个小道具，比如一个小纸团或一根小杂草。这样的小东西随处可得。

然后，你可以对女生进行试探：
> “你头上有个东西，我帮你拿下来。”

这时可能有两种反应：
*   **第一种是拒绝**：她拒绝你碰触她的头发。
*   **第二种是接受**：她允许你帮她拿掉东西。

如果她接受了，就证明你通过了这次非敏感试探，可以进入真正的接触阶段。

---

### 第二阶段：接触 ✋

上一节我们介绍了非敏感接触，本节中我们来看看真正的**接触**。这指的是碰触她较为敏感的部位，例如脸、手或腰。

这里教大家一个小技巧：**看手相**。

你可以对她说：
> “你的掌纹好像有点奇特，让我看看……你看，你的命多好。”

如果女生愿意把手递给你，就证明她通过了接触测试，你们可以进行下一步。如果她没有递出手，你就需要回到上一步重新开始。

---

### 第三阶段：牵手 🤝

通常，牵手发生在带女生散步或经过某些场合时。

以下是推荐的牵手方式：

当你们遇到不太好的路面，例如马路、楼梯或电梯时，可以采取这个动作：

然后，你拉起女生的**手腕**带她过来，接着做一个自然的转换：把她的手从你的右手放到左手里，让她**挽着你的胳膊**。

这样做的好处是，她的手是自由的。如果她感到压力或不舒适，可以随时拿开。如果她喜欢并愿意进行肢体接触，她就会一直挽着。

---

在我们和女生已经完成牵手这一情侣举动之后，可以尝试下一步：**接吻**。

我教给大家两种方法：

1.  把她搂在怀里。
2.  用手指轻轻抚摸她的嘴唇。
3.  如果她没有抗拒，就可以吻下去。

**第二种方法**是**三角对视法**：
1.  先看女生的左眼。
2.  再看她的右眼。
3.  最后将视线移到她的嘴唇。
4.  如果对视超过5秒且她没有扭开头避开，就可以吻下去。

关于“纳杀”阶段，实在无法在此演示。但可以明确的是，在你成功完成牵手和接吻之后，你们已经是很好的情侣关系。此时，你只需要一个合理的私密空间（如旅行、旅馆或家中），关系自然可以水到渠成。

### 核心原则与总结 📝

本节课中我们一起学习了肢体接触的五个层次。你应该已经明白，维持和女生关系并提高成功率的最大方式，就是通过各种小的试探，**循序渐进**。

其核心逻辑可以概括为一个简单的**流程公式**：
```
试探 → [接受] → 进入下一阶段

```

这样做可以扩大成功率，避免因过于着急和唐突而导致女生反感或不适。同时，你也应该知道，和女生的肢体接触只是关系升级的环节，而不是最终目的。
