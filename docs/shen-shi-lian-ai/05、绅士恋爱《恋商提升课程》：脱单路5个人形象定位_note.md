# 05、绅士恋爱《恋商提升课程》：脱单路5：个人形象定位 👔

在本节课中，我们将要学习个人形象定位的重要性，以及如何通过明确的风格定位，在众多男性中脱颖而出，给女性留下深刻的第一印象。

上一节我们探讨了如何建立初步联系，本节中我们来看看如何通过外在形象巩固你的独特吸引力。

首先，想大家一个问题：你知道全世界最高的山峰叫什么吗？没错，都知道是珠穆朗玛峰。那第二高呢？可能许多人已经不知道了，是乔戈里峰。那第三高呢？估计已经没什么人知道了。

那么再问一个类似的问题：你认识的最漂亮的女生是谁？第二漂亮的呢？第三呢？你只能记住你身边最漂亮的女生、身材最好的女生、最温和的女生、最有个性的女生。但是你是记不住第二、第三、第四的人。

同样在女生眼里，看男生也是一样的。她们只会记住最帅的、最有钱的、最风趣的、最温和的、感觉最好的那几个人。并且形象是第一印象的来源。所以你们需要找到一个独特的个人形象定位，成为男生中的“最”，才有可能得到女生的关注。

今天我会推荐一些形象的定位给大家。

以下是三种核心的形象定位元素，每种都对应不同的性格与生活场景。

**1. 商务精英元素**
*   **适合人群**：已工作、性格认真稳重、处于适婚期的男生。
*   **关键元素**：合身精致的西装、纯色衬衫、各色领带、腰带、大衣、优质皮鞋，以及头发简单整齐。
*   **核心要点**：西装是男人的武器，是小男生成长为男人的标志。重点在于**合身**。
*   **合身标准公式**：`合身 = 正确的袖长 + 合适的肩膀 + 恰当的腰围 + 适中的衣长/裤长/中档`
*   **颜色建议**：标准商务色——黑色、藏青色、灰色。竖条纹款式比较显瘦。
*   **款式选择**：单排扣只选择一粒扣或两粒扣（只扣上面一粒）。双排扣左下角的扣子不要扣。
*   **价位选择**：工作族不考虑千元以下西装，易变形、褪色、难保养、寿命短。腰带选头层牛皮，百元左右无logo即可。普通消耗皮鞋建议200-300元，品牌可淘宝搜索“狼地围”。走精英路线建议选择700-1000元能穿几年的优质手工皮鞋，品牌建议“上海旗艾订制店”。
*   **发型选择**：偏分油头、背头、短发等根据脸型设计的无刘海简单发型。

**2. 潮牌时尚元素**
*   **适合人群**：学生、喜欢户外、性格张扬的年轻男生。
*   **关键元素**：logo字母外套、棒球服、大图案T恤、皮质马丁靴、破洞衣服、潮鞋，以及张扬的发型。
*   **核心要点**：年轻的态度就是张扬。潮牌代表的就是这种性格，要求的是前卫和个性，而非严格的合身与搭配。
*   **颜色与款式**：随意，建议直接购买成套搭配。
*   **价位选择**：从几十到上万元不等，根据自己消费能力选择。淘宝搜索关键词“潮牌”。
*   **发型选择**：染色以及层次感分明的Undercut发型是首选。

**3. 简约日常元素**
*   **适合人群**：追求简约、性格内向温和的男生日常穿着。
*   **关键元素**：纯色亚麻衬衫、开衫毛衣、套头卫衣、卡其裤、牛仔衣以及小白鞋。
*   **核心要点**：**简单**和**干净**是核心，能营造给女生非常愿意接近的感觉。
*   **颜色与款式**：建议选择纯色的基本款。
*   **价位与店铺**：推荐基础店铺“优衣库”（淘宝和实体店都有），价位通常在300元以下。
*   **发型选择**：中长发和分头。维持一致性非常重要。

以上三种元素足以满足任何男生在工作与日常中的穿着需求。但形象定位需要维持你形象的一致性。

例如，如果你定位是**商务精英元素**，那么你的朋友圈日常应多展示工作、商务洽谈、酒会、人生规划等内容。而**简约日常元素**就可以多展示咖啡厅、下午茶、拍照等内容。这些都有利于增加对女生的吸引力。

现在来给大家布置本节课的作业：
1.  确定个人的形象定位。
2.  购买符合形象定位的服装。
3.  记笔记，想象一下自己在打造形象定位的过程中，应该要注意到的问题。
