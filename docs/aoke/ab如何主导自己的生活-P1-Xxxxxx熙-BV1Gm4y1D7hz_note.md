# 课程一：如何主导自己的生活 🎯

在本节课中，我们将学习如何有效管理个人时间与精力，摆脱拖延，从而主导自己的生活。课程内容基于个人经验，总结了四个实用技巧。

记得有一次在台北开讲座，一位观众问我：“AB你一天至少有48个小时，才能一次学那么多的才艺，练这么多的硬价值。”当时这个问题让我很惊讶。

我自己深知我相当会拖延，并且为这个问题纠结了很久。毕竟刚从为公司工作，到离职开始创业、经营自己的频道时，生活形态是一个巨大的转变。因为没有人是你的老板了，你必须管好你自己。

你必须为自己的每日行为和行动后果负责。因此，今天我想与你分享主导自己生活的四个技巧。

## 技巧一：砍掉所有分心 🪓

很多时候当我在拖延、纠结，无法高效产出时，我可能同时在做好多事情：看美剧，或者学习其他什么东西。在那个生活过程中，我一定在做很多事。但每当我真正开始能专注时，我就会很果断地把一些不重要、无关的东西全部砍掉。

这种“砍掉分心”其实分大的和小的东西。

以下是具体分类：

*   **大的东西**：指我会砍掉一些宏大的学习格局。例如，当时我正在学弹钢琴，或者学跳舞，或是其他才艺。这是一个很大的学习项目，我会把它砍掉，如果我发现我当前专注的东西是至关重要的。
*   **小的东西**：例如当天与朋友的聚会，或是你当天想完成的一些小事项、要上的课程、要玩的游戏、要看的电影。无论是娱乐还是自我提升，我都会把这些不重要的分心项砍掉。

有趣的是，每当我砍掉这些东西，无论大小，我的心里就会进入一种状态：觉得我现在终于可以专心一致解决我要做的事情。我发现我必须把一些东西都砍光，只留下重要的去执行，我才能够持续专注。

## 技巧二：牛顿第一运动定律 🚀

上一节我们介绍了如何清除干扰，本节中我们来看看如何启动任务。很多时候你要做一件事，会开始拖延，因为这个项目实在太庞大，你觉得做起来很烦，于是就被眼前庞大的工作量震慑住了。

通常在这时，我会告诉自己：好，我没必要先想着要把它做完、做到完美，但至少我可以先开始做。无论这件事多复杂，我只要能开始做五分钟就可以了。

通常有两种方式去想这件事：

*   **时间法**：就像我刚讲的“五分钟”。
*   **步骤法**：你把这件事简化到只需执行的第一步。

有趣的是，牛顿第一运动定律讲：静者恒静，动者恒动。当你开始做的时候，十之八九很大的几率，你就会持续做下去。而当你不做的时候，你就必须花很大力气去启动它。所以只要你启动那一瞬间，确实会有一个惯性，让你持续往前把剩下的事情做完。

当然，如果最后你没把事情完成，没关系。你要了解，很多东西其实难在不能开始，而非不能完成。

## 技巧三：每日MVP 🏆

我们已经讨论了如何开始，接下来看如何确保每日都有核心产出。第三个技巧叫做“每日MVP”。意思是，我会为这一天设定：我只要完成一个最重要的事项就可以了。

我今天的核心目标就是完成这个事项。其他的东西，能完成就完成，不能完成就算了。但就是有一件事，是我今天绝对要完成的。

这个技巧其实很好。因为很多时候你一天想着要做很多很多事情，当你有太多事情想完成时，你其实会进入一种混乱状态，每件事都做一下，结果你就会开始拖延。

然而，当你想着“我今天只要完成这一件事”的时候，有趣的事情就发生了。因为你起床时，无论刷牙、吃饭、做任何事，你都会开始调整，因为你知道我今天就是要完成这件事。你所有的事情都会朝这个方向调整。

这样的心理状态和行为，会让你更有效率地完成那件事。就像刚讲的牛顿第一运动定律，这个惯性有可能让你持续把其他事情也做完。但无论如何，当你每天都能完成一件你觉得很重要的事，累积下来就非常可观。

所以这个方法我到现在还每天使用。我一天都会决定一个MVP，比如“今天就要把这个影片拍完”。那么其他事情，比如学语言、运动等，都排在这件事的次要位置。

## 技巧四：营造舒适有趣的环境 🌿

最后，我们来探讨环境对效率的影响。最后一个技巧是：我会让我周遭的环境非常舒适以及有趣。

因为当你在做事时，如果你的心情比较愉悦、比较自愿，你做起来会比较有效率。所以我习惯放Spotify，放音乐。我比较喜欢爵士乐和古典乐。虽然有研究指出，放这些音乐会影响你的专注力，让你有些分心，但这样的牺牲我愿意接受，因为它可以让我的心情更加愉悦。

包含我的书桌或屏幕，我都会选择让我看起来比较舒适、能让我工作更有效率的。甚至有时候，我可能想去咖啡店工作，我也愿意花这点时间，只为了让我周遭的环境更舒适，来换取效率。

你要了解的是，当你开始要主导自己的生活，已经没有老板的时候，如果你还用过去那种别人鞭策、逼迫你的方式来主导生活，其实会遇到很大困境。

要从这些混乱中找到秩序与平衡，最基本的，就是要可以主导自己的生活。你不需要很有效率、很有产出地做完所有事情。你只要每天持续进步，完成那些对你来说很重要的事情。那么长久累积下来，你最终就会找回主导自己生活的节奏。

## 总结 📝

本节课中，我们一起学习了主导自己生活的四个核心技巧：

1.  **砍掉所有分心**：果断移除大小干扰，只专注关键任务。
2.  **应用牛顿第一定律**：克服拖延的关键在于“开始”，利用惯性推动进程。
3.  **践行每日MVP**：每天锁定并完成一件最重要的事，积累可观成果。
4.  **营造舒适环境**：通过调整物理与心理环境，提升愉悦感与效率。

掌握这些方法，并持续实践，你将能逐步建立起对自己生活的有效主导权。
