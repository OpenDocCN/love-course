最近网络上流行一个理论，称九成的男人是没有吸引力的。刚听到时，我怀疑这件事的真实性。

我回想起自己念中小学的时候，那时大家都有发禁，还要穿制服。印象很深刻，当时很多人会想把裤子改窄一点以求好看，但因此会被教官或教务主任训斥甚至记过，认为这是坏学生的代表。如果你想当好学生，甚至会觉得自己不应该注重外表，乖乖读书才会被夸奖。

等到上班工作忙碌起来，就开始觉得赚钱才是正事。所以，我仔细思考后认为，如果大部分男人缺乏吸引力，主要有以下几个原因。

## 1. 忽视身体健康 🏋️♂️

第一个原因是，放任自己的身体状态变差。

我经常收到一些关于生活不快乐或交不到女朋友等问题的咨询。每当遇到这类问题，我首先会问对方：“你有先把自己的身体照顾好吗？”

我知道，很多人（包括以前理工科出身的我）会因忙于工作、熬夜等，认为去健身房来回太浪费时间。但如果你真的把时间空出来，优先去做这件事，生活会发生巨大的转变。

**核心转变公式：**
`投入时间锻炼` → `体格与气场改善` → `获得他人“另眼相看”`

你的整个气场、体格和睾酮水平等都会改变。你会发现周围的人开始用不一样的眼光看你，觉得你真的很不一样。这是一个非常实际且有效的“生物暴力解法”。

## 2. 软弱的心理强度与缺乏纪律 🧠

第二个原因是心理软弱，放任自己毫无纪律。

大部分人是懒惰的，坚持一件事很痛苦，很多人都想走捷径，不愿按部就班。那么，如何保持纪律呢？

我最初培养的心态是：意识到**最终你人生的所有问题，都只能靠你自己解决**。如果没有这层觉悟，你做很多事情都是做给别人看的。如果你的动力和耐心都依赖于外部认可，那么这种动力会消失得很快。

维持纪律的关键，不在于你充满动力的时候，因为动力非常短暂。**重点不是你想做的时候才去做。**

**纪律的核心代码：**
```python
if motivation_level == LOW: # 当你不想做的时候
    still_complete_task() # 但仍然去完成它
```

当你健身或练拳击时，每一个动作都是当下短暂痛苦的累积，也是成长的累积。你知道今晚回家睡觉时，会感受到一种满足感和骄傲。除了提升纪律，我还非常推荐大家学习如何掌控情绪。

例如，不要让愤怒掌控你，导致完全失控。有些人会破口大骂，有些人选择逃避或抱怨。第一步，你要意识到：**当我不能控制情绪时，这件事本身是软弱且没有吸引力的。**

不要觉得被情绪掌控行为是很有男子气概的表现。你不需要通过失控的情绪来让人感到可怕，那是学生时代的小把戏。当你能控制好情绪和行为，并专注于解决问题时，这是一种非常强大且有吸引力的心态。

**但需要补充：** 控制情绪不代表你要忍气吞声，这是两件完全不同的事。你可以控制好行为，同时非常冷静或果断地解决眼前的问题。

## 3. 无法独立主导自己的生活 🏠

第三点，无法独立主导自己的生活。

我并不认为你一定要非常有钱才有吸引力，但**能否独立主导自己的生活**却非常关键。想想看，如果你和室友同住，共用客厅和浴室，因为这些不是你自己独有的，你可能就不会去清理，觉得别人也会用，何必帮他清理。

这会导致因为你没有完全的权利，也就不想尽义务，生活越来越散乱。我认为，即使你当下的金钱能力不足以买房买车，也应该用现有的经济能力，想办法住在自己可以负担得起的地方。

**独立生活的价值：**
*   **驱动赚钱欲望**：如果你能一个人住，这会驱动你想要赚更多钱。
*   **训练责任感**：你必须为自己空间的一切负责。
*   **核心原则**：**你所做的一切，都是为你自己而做。**

## 4. 缺乏社交能力与社交地位 🤝

第四点，缺乏社交能力与社交地位。

作为社会的一员，我们从小并没有被很好地教育如何提升社交圈和社交自信。你希望有自信，但不想变得霸道；希望谦虚，但不想自我贬低。这中间确实需要很多拿捏。

所以，基本上你需要了解如何去做“主导”。主导意味着拥有比较自信、有魄力的“阿尔法”思维。当然，除了这种社交主导力，你还要懂得如何做“社交微调”。

但我认为，你应该先专注于建立主导力和自信，**先做过头**，然后再学习如何做社交微调，这是一个更好的方式。如果一开始就想谦虚、怕东怕西，其实很难变得自信。

在拥有基础的社交自信和技巧后，你可以开始学习更进阶的：如何与人协同合作、如何谈判、如何观察非常隐性的信息。这会是更高阶的社交能力。

如果你的社交能力不行，你就是孤立的一个人。周遭没有一群所谓的“阿尔法”或高价值男人时，你的吸引力和能力会相对较低。请记住，**让你的生活圈拥有更多有价值的人，这件事的威力非常强大。**

## 5. 除了工作，没有任何其他追求 💎

最后一点，除了工作以外没有任何其他追求。

意思是，如果你真的只看重钱，其他事情都不管，你的生活就会陷入死局。就像有人说“穷得只剩下钱”，大概就是这个概念。所以我经常说：钱很重要，但钱不够好。

**人生价值分层模型：**
1.  **生存层**：当你没有钱时，当然要专心赚钱。
2.  **生活层**：当你有维生能力时，应开始有更大的格局。
3.  **生命层**：追求钱无法直接兑换的价值。

当你开始有能力赚钱时，我鼓励你开始有更大的格局。这可能包括学习任何技能、培养兴趣或才艺。例如，我会弹钢琴、练大提琴，这对我而言是生活里除钱以外的另一种价值。我不可能靠钱、但不练习，就买到这种文艺属性的能力。

人生的冒险和体验也是一样。你也许可以花钱坐游艇、住很贵的旅馆，但很多人生的体验，你必须亲自走过。例如，与他人的深度交流、去某个地方、建立某种合作关系、探索未知的国家等。这些都不是把钱丢出去就一定有的，你必须要有冒险精神去探索未知，突破舒适圈。

那么，有什么是比钱更重要的东西呢？这个思维叫做：**打造你的帝国**——打造你觉得很有价值的东西。

例如，你希望建立一个很棒的车友社群，大家一起研究跑车或越野车。当你在打造这样一个有价值的东西时，你可能就不会斤斤计较是否亏了一点钱，因为你在换取**钱无法直接兑换的价值**。

想想看，如果你现在没有金钱问题，你希望打造出什么东西？如果你觉得“我不知道，有钱再说吧”，那是不够的。等到你有钱才开始打造，那时你可能年纪已大，不知从何做起。**你必须从现在就开始培养这种能力。**

## 总结与行动号召 📝

本节课中，我们一起学习了导致男性缺乏吸引力的五个核心原因：忽视身体健康、心理软弱缺乏纪律、无法独立主导生活、缺乏社交能力与地位、以及除了工作外没有其他追求。

请记住，也许你现在觉得自己很弱、很普通，但是**开始锻炼自己、提升自己永远不嫌晚**。我自己的力量属性也是这几年才开始训练的。我不会抱怨说“我都这个年纪了，所以不想学这个”。我认为，所有的属性和自我提升，都可以从现在开始。

也许你今天开始提升时没有任何感觉，但过了一个月、半年、一年、两三年，就会差非常多。甚至十年后，你的整个生活可能已经完全改变。那时，你再看到别人（包括异性）对你的反应改变时，你就会发现真的有差。虽然本质上你可能还是当初的那个你，但 somehow，一切好像都不一样了，你认知的现实也已经改变。这时，你会获得一种巨大的满足感。

**行动的第一步，往往是最难的，但也是最重要的。** 从上述五个方面中，选择一个你最有感触的点，制定一个微小的、可持续的计划，并立即开始执行。
