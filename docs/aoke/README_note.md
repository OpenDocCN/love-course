# 🌟 奥客教程 🌟

## 概述

在本节课中，我们将学习如何使用奥客（Outlook）进行邮件管理。我们将介绍如何创建账户、发送邮件、接收邮件以及管理联系人。

## 创建奥客账户

首先，您需要创建一个奥客账户。以下是步骤：

1. 打开奥客官网：[https://outlook.com](https://outlook.com)
2. 点击“创建免费账户”
3. 按照提示填写信息，包括用户名、密码和邮箱地址
4. 完成验证步骤

## 发送邮件

以下是发送邮件的步骤：

1. 登录奥客账户
2. 点击“新建邮件”
3. 在“收件人”栏输入收件人邮箱地址
4. 在“主题”栏输入邮件主题
5. 在正文栏输入邮件内容
6. 点击“发送”

## 接收邮件

以下是接收邮件的步骤：

1. 登录奥客账户
2. 在左侧菜单中点击“收件箱”
3. 您将看到所有收到的邮件

## 管理联系人

以下是管理联系人的步骤：

1. 登录奥客账户
2. 在左侧菜单中点击“联系人”
3. 点击“新建联系人”
4. 填写联系人信息，包括姓名、电话、邮箱等
5. 点击“保存”

## 总结

本节课中，我们一起学习了如何创建奥客账户、发送邮件、接收邮件以及管理联系人。希望您已经掌握了这些基本操作。
