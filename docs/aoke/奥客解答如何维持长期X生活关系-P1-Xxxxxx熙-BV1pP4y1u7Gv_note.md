在讨论了身体管理后，一个常见的问题是：经济状况是否会影响关系和自我认知？本节我们来探讨财务与生活品质的平衡。

有学员提问：“之前我收入少时很焦虑，现在收入恢复了焦虑就没了。但我既没有养家或房贷的压力，我应该配合世俗规范，努力月入三万或更多，还是赚得够用就好？”

这是一个有意思的问题。钱的多与少，不完全由绝对数字决定，而是由你当前的生活方式决定。如果你的约会路线规划、约会处理方式，包括开房间的档次、吃饭的档次等都能负担得起，那么钱不一定需要很多。

所谓“够用”，就是买东西不用看价钱，吃饭不用看菜单后面的价格。我认为人生达到这种程度就很足够了。所以，如果你现在交往的女生，你们去的地方都不会让你经济紧张，那在某种程度上就是一种财务自由。

例如，我认为人绝对不能三餐吃泡面生活。不需要天天山珍海味，但至少要吃人吃的食物。问题在于，要能三餐都吃人吃的食物，月薪三万很难办到。现在的食物高度工业化，变得很便宜，但我们却越吃越胖，越吃越不健康。

所以，你需要考虑这一点。你需要知道什么样的生活是“人过的生活”，然后推算需要每月赚多少钱才能支撑这样的生活。我认为这种思维方式更为可取。
