# 身高不够焦虑怎么办？拿到联系方

## 📚 课程概述

在本节课中，我们将要学习两个在社交互动中常见的问题：如何克服因身高不足而产生的自信焦虑，以及如何处理搭讪后对方冷淡或失联的状况。我们将拆解核心心态，并提供具体、可操作的调整方法。

## 🧠 第一部分：如何克服身高不足带来的自信焦虑

上一节我们概述了课程内容，本节中我们来看看如何具体调整因身高问题产生的自信焦虑。

自身有健身习惯，但受限于身高不高，面对女性时难免有不自信的状况，请问该如何调适？这一块我觉得，第一个你自己不能先“跪”。你自己不能先觉得，好像身高不够，然后就感到气馁。

一旦你具备这样的念头，就糟了。女性绝对可以察觉到，这个人可能气势上不太够，可能自信不太够，她们一定可以发现你的弱点。所以你要先忘掉它，你要先忘掉这件事情。

那么要说忘掉也没那么容易，因为它毕竟是根植在蛮多人心里面的一个疙瘩。我建议就是，你不妨这样做。

以下是具体的调整步骤：

1.  **练好身材**。当你的体格比一般人要好时，你一定会有一个时刻是，女性盯着你胸肌看的时候。
2.  **建立积极联想**。她们可能会打量你的身材，可能会打量你的体格。那以后只要你想说，身高不够怎么样怎么样的，你就回想起之前被女性打量胸肌、打量体格的那个眼神。
3.  **替换焦虑**。那你就可以大幅度地把你对于身高不够这件事情的担忧，给降下来。

**核心心态公式**：**自信 = 突出优势 - 聚焦短板**。通过强化你的身材优势（突出优势），并转移对身高的过度关注（减少聚焦短板），来重建整体自信。

## 💬 第二部分：搭讪后对方冷淡或失联如何处理

在学会了如何调整自身心态后，我们来看看外部互动中一个棘手的问题：搭讪后的跟进。

搭讪过程中，女性也感觉有意愿，也有推进互动，包含肢体触碰和调情，但回去之后，关系冷掉或LINE不回或封锁，是否因为当时当下没有直接带走进行即时约会？

很多人都把搭讪的本意搞错了。搭讪不是为了要跟对方拿联络方式，搭讪的主要目的是要冲即时约会。

以下是关于搭讪目的的正确理解：

*   搭讪的主要目的是争取即时约会，可能是喝咖啡，可能是喝饮料。
*   虽然即时约会难度很高，但也不是没有机会发生。
*   你不能够指望说拿到那个LINE就一定会得到回应。

说穿了就是这个样子。甚至于说她给你LINE，可能只是为了让你离开，反正她之后可以封锁掉。所以搭讪的目的性你要搞清楚。

**核心行动准则**：**搭讪首要目标 = 即时约会**。联络方式是次要的，是即时约会互动愉快后的自然延伸。

搭讪不是为了要跟对方有后续的联络，而是当下就冲即时约会。如果可以的话，你是因为即时约会跟对方互动愉快之后，可能聊个LINE有后续，应该是这个情况。但绝对不是说，你搭讪，对方给LINE，后面就会有后续，机会没有很高，这个你要明白。

## ✅ 课程总结

本节课中我们一起学习了两个关键社交技巧。

首先，我们探讨了如何克服身高带来的自信焦虑。关键在于**停止自我否定**，并通过打造并**聚焦于自身其他优势**（如健硕的体格）来转移注意力，从而建立稳固的自信。

其次，我们厘清了搭讪的核心目的。搭讪的**首要目标不是获取联系方式**，而是尽可能创造**即时约会**的机会。联系方式应是即时约会愉快互动后的副产品，而非保证后续发展的凭证。

记住，强大的心态和清晰的目标是有效社交的基础。
