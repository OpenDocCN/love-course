**公式**：Lea Kai = 快速变形艺术家 + 热爱时尚 + 梦想成为美术家

**公式**：艺术旅程 = 遇到Froyan + 参加天才秀 + 第一次上台表演

**公式**：挑战与支持 = 紧张的挑战 + 观众的反应 + 热情融合 + Froyan的支持
