# 恋爱课堂 P3：舔的受害者是你 🎯

在本节课中，我们将深入探讨“跪舔式”付出的核心危害。我们将分析这种心态如何破坏人际关系、扭曲自我认知，并最终导致失败。课程将拆解其背后的错误逻辑，帮助你建立健康、平等的互动方式。

---

## 跪舔行为的四大危害

上一节我们介绍了“跪舔”行为的定义，本节中我们来看看这种行为具体会带来哪些危害。

### 1. 无法获得他人尊重

当你通过过度付出去维系关系时，你基本不可能得到别人的尊重。这种与外界的互动方式，会导致他人瞧不起你。

当一个人瞧不起另一个人时，就容易释放恶意。因为他用不着在乎你的感受。

**核心概念**：`尊重缺失 -> 恶意释放`

### 2. 成为恶意释放的对象

因为你得不到尊重，对方就容易对你释放恶意。例如，我看不起你，所以我可以随意释放对你的恶意，因为我不在乎你的感受。

我认为你就是个垃圾，所以我完全不尊重你，也不是为了你好，就是看不惯你。只有当一个人在你心目中地位很低时，你才不会去在乎他的感受。

**核心概念**：`低价值感知 -> 感受被忽视`

以下是很多人遇到的困境：
*   女孩子利用你却不给你回馈。
*   对方不懂感恩你的好。
*   根本原因在于对方不尊重你。

### 3. 对人际关系失去信心

如果你经常这样与人交往，你也会对人际关系失去信心。

例如，很多人去学习所谓的“套路”，其实就是因为你们对男女关系失去信心了。你自己已经失去信心，对所有女人都失去信心，才会去想技巧和套路。

如果你对女孩子有信心，你绝对不会用这种技巧和套路。最归根结底的问题，是你一开始就错了，导致你对人际关系失去信心。

### 4. 内心产生怨气，陷入对抗思维

发展下去，你的内心就会产生怨气，进而认为人与人之间的交往就是吃亏或占便宜的关系。

例如，你们觉得跟女的交往，不是她花钱就是你花钱，不是你吃亏就是你占便宜。你们会觉得为女孩子花了钱就是吃亏，女孩子给你花了钱就是占便宜，还值得炫耀。

这是一种心灵扭曲的想法。你很容易陷入与人对抗的境地，制造和激化与女孩子之间的矛盾。你们那样只会让矛盾越来越大，因为你本身就已经陷入对抗的境地了。

## 健康付出 vs. 跪舔式付出

上一节我们剖析了跪舔的危害，本节中我们来对比一下健康的付出心态是怎样的。

我与女孩子出去从来都是我花钱，而且花钱很大方。因为我的想法不同：我觉得出去就干脆花钱，我就是为了追你，所以花钱。我告诉女孩子，我觉得男女在一起，女的不该花一分钱，就应该男的花钱。

所以我不会让女孩子觉得欠了我的，或者我请她吃饭她就应该回馈我什么。这是我本来的想法，对方就觉得吃我这顿饭心安理得，吃得舒服。

**核心概念**：`健康付出：出于本心，不求即时回报`

而炫耀女的给自己花钱，是最没有尊严的人才会做的事。一个要脸的人，哪好意思用女人的钱？你要脸的话，你好意思用女人的钱吗？这属于一种价值观的问题，我们觉得很丢脸的事情，你们却觉得是骄傲。

我与谁出去花钱，是一种“连接性”的想法。因为我喜欢你，我就愿意为你花钱。哪怕为你花再多钱，我也情愿，因为我喜欢你，我愉快。我天天跟你出来花这些钱，没觉得亏，也没觉得难受。

**核心概念**：`连接性付出：我喜欢你 -> 我自愿付出 -> 我感到愉快`

你们很多时候跟女孩子出去不愿意花钱，是因为你们本来就抠门，说难听点就是因为没钱，才会斤斤计较这个事情。如果你有钱，又喜欢这个女孩子，你给她买好穿的好吃的，都会觉得愿意。

例如，我年初手机坏了，想干脆把我和我女朋友的手机一起换了。我花一万多块钱买手机，眼睛都没眨一下，觉得这很正常。我没说这是送她的礼物，就是想我换了那我们一起换吧，顺便给你买一个。对方得了这个东西，心里会对你有一种认可，觉得你真的把她当回事，也觉得你这个人比较大气体面。

## 错误思维模式的根源：失败者特征

我们已经看到了两种付出心态的巨大差异，那么，导致跪舔思维的根源是什么呢？本节将揭示其背后的失败者逻辑。

你们学的那套东西，就是“炮学”的核心价值观，换了个名字而已，挂羊头卖狗肉。不管叫什么名字，就是你们那种价值观和思维方式。

你们那个群体，是这个社会上能力最差、层次最低的一群，胆子还小。胆子大一点，也不会去搞“炮学”这种东西。你可以看到你们很多人，不是做厨师的，就是做保安的，是社会最底层的乌合之众，没能力没本事，只会去学这种东西，想着怎么去骗人、占别人便宜、走捷径。

**核心概念**：`失败者思维 = 寻找捷径 + 占人便宜`

你们可以看到，所有失败者身上都有这种特征：
*   第一个是我怎么去占你便宜。
*   第二个是我有哪些捷径可以走。

越失败的人，越有这样的特征。这是因为你们的思维方式和三观决定了你们的失败。你们不是没有智商和能力，你们就是不愿意去吃苦，不愿意去认真地想问题。

## 过度付出的错误逻辑链

理解了失败者思维，我们再来具体拆解“过度付出”这种行为背后的错误逻辑链。以下是其典型的四步心理活动：

很多人是这么想的：我是为了得到你的爱，而做了一些事情。然后我觉得你会因此而感动，从而决定接受我。

也就是说，你们最熟悉的那句话：“只要我默默对你好，你总有一天会感动，然后会接受我。” 还把自己想得特别伟大。

**核心公式**：`错误逻辑：我的付出 -> 你的感动 -> 你的接受`

你一开始会觉得“我为了爱她，为了得到她的爱而做一些事情”。因为这是你自己的需求，你就只会站在自己的角度考虑问题：我是不是要请她吃饭？我是不是要给她买衣服？我是不是要送她手机？我对她这么好，她应该会感动，应该会接受我吧？

这会给对方什么感觉呢？你只是“为了得到我”而这么做。

---

## “为得到”与“为喜欢”的本质区别

这个区别至关重要。为了从你身上得到什么而对你好的人，和真的对你好的人，有本质区别。

为了得到你的什么而对你好的人，他根本不会用心。你会觉得这个人根本不走心，不会考虑你的感受。因为他只会考虑他的目的。这种对好，属于一种“贿赂”。

**核心概念**：`贿赂性付出（目的驱动）` vs. `真心付出（情感驱动）`

举个例子：一个朋友对你好是为了跟你借钱，另一个朋友对你好是真心把你当朋友，第三个对你好是觉得你有钱。你的感觉会一样吗？

那个为了得到你而对你好的男人，女孩会觉得他并不了解自己，好像也不是认真的，只是为了追到她而这么做。女孩会觉得：“你对我很好，但我就是觉得哪里不对。因为我觉得你只是沉浸在自己的世界里，根本没有关心我。”

这就导致：你很难得到对方情感上的响应。对方会觉得：“你确实对我挺好，也花钱，但我很难心安理得地接受。因为我觉得你不是出自本意，不是真的喜欢我才这么做，而是为了得到我才这么做。”

## 本节课总结

在本节课中，我们一起学习了“跪舔式”付出的多方面危害及其根源。

我们分析了它如何导致**失去尊重**、**遭受恶意**、**摧毁人际信心**并滋生**怨气与对抗思维**。通过对比健康的“连接性付出”，我们明确了**真心喜欢**与**企图得到**的本质区别。最后，我们揭示了这种模式背后深藏的**失败者思维**——即热衷于寻找捷径和占人便宜。

记住，任何建立在自我贬低和功利计算之上的关系都无法长久，更无法带来真正的幸福。健康的关系始于平等的姿态和真诚的意愿。
