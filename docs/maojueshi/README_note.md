# 猫爵士的编程课 第1课：变量与赋值 🐱💻

在本节课中，我们将要学习编程中最基础也是最重要的两个概念：**变量**和**赋值**。理解它们是如何存储和操作信息的，是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了本课的主题，本节中我们来看看什么是变量。你可以把变量想象成一个**带标签的盒子**。这个盒子（变量）可以存放各种东西，比如数字、文字或更复杂的数据。盒子上的标签就是**变量名**，我们通过这个名字来找到并使用盒子里的内容。

在编程中，变量用于存储程序需要使用的数据。例如，一个叫 `score` 的变量可以用来存放玩家的游戏分数。

## 如何给变量赋值？🔄

知道了变量是存储数据的盒子，那么如何把数据放进去呢？这个过程就叫**赋值**。我们使用等号 `=` 来完成赋值操作，这可以理解为“把右边的值，放进左边的变量里”。

其核心公式是：
**`变量名 = 值`**

以下是赋值操作的具体步骤和例子：
1.  **创建变量并赋值**：当第一次给一个名字赋值时，就创建了这个变量。
    ```python
    age = 3
    ```
2.  **改变变量的值**：之后可以给同一个变量赋予新的值，旧的值会被替换掉。
    ```python
    age = 4
    ```
3.  **将其他变量的值赋给它**：也可以把一个变量里存放的值，复制到另一个变量里。
    ```python
    my_age = age
    ```

## 变量命名规则 🏷️

为了让我们和计算机都能清晰地理解变量的用途，给变量起名需要遵循一些简单的规则。以下是主要的命名规范：
*   变量名只能包含**字母、数字和下划线**。
*   变量名不能以数字开头。
*   变量名是**区分大小写**的，`Cat` 和 `cat` 是两个不同的变量。
*   应选择有意义的名称，例如用 `user_name` 而不是 `a`。

## 让我们动手试试 ✍️

理论需要结合实践。下面我们通过一个简单的例子，将以上概念串联起来。我们将创建几个变量，并进行赋值和修改操作。

```python
# 创建变量并赋值
pet_name = "Tom"
pet_age = 5

# 打印变量
print("宠物名字：", pet_name)
print("宠物年龄：", pet_age)

# 修改变量的值
pet_age = 6
print("修改后的年龄：", pet_age)

# 进行一个计算
human_age = pet_age * 7
print("相当于人类的年龄：", human_age)
```

运行这段代码，你会看到变量如何存储数据，以及数据如何被改变和用于计算。
