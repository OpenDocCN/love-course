# 恋爱心理课 01：展现价值，而非“利用价值” 💡

在本节课中，我们将探讨在恋爱与人际关系中一个核心的区别：如何展现真正的个人价值，而非仅仅展现“利用价值”。理解这一点，对于建立健康、长久的关系至关重要。

## 缺乏热情的困境

上一节我们提到了核心概念，本节中我们来看看缺乏热情可能带来的问题。生活中，很多人对工作或事情缺乏热情。这样的人很可能变成只谈利益的人。即使最初有热情，也可能逐渐变得圆滑世故。

大公司内部往往存在小团体。关键在于你属于哪个小团体。在大公司的不同部门中，你在自己领导和老板眼中的形象非常重要。你需要分辨：你是一个有价值的人，还是只有利用价值的人？

## 识别“只有利用价值”的人

以下是几种典型的、只会让人感到有“利用价值”的人。了解这些特征，有助于我们反思自身，并识别他人。

**1. 带有强烈利益动机的人**

当我们察觉到某人接近我们带有强烈的利益动机时，例如一个女生明显是图你的钱，我们很容易产生不好的想法。我们会觉得：“这个人对我没安好心，他图的不是感情或未来，只是我的钱。” 这种心态会导致我们只想利用对方，占对方便宜，而不会真心对待。这解释了为什么有些人会成为“舔狗”——因为对方清楚地知道你图谋不轨，从而毫无道德负担地利用你。

**2. 能力不足且斤斤计较的人**

这类人并非图谋大利益，但总想从别人身上占小便宜。例如，作为老板，遇到那种没钱却拼命讨价还价、付款不痛快、总想多拿赠品的顾客，会非常厌烦。与这种人打交道耗费精力，且没有价值，通常只想做一次性交易，不愿再有往来。

**3. 人品有问题、私心过重的人**

这类人将个人利益置于集体责任之上。例如，公司有紧急订单需要加班完成，但个别员工以“公司不能强制加班”为由，拒绝完成工作，导致公司蒙受损失。即使有权益纠纷，也应先完成紧急事务，事后再协商。这种不顾大局、关键时刻“掉链子”的行为，是人品问题。一旦传开，没人敢托付重要事情给这样的人。

**4. 能力强但自我定位过高的人**

这种人能力出众，但过于自我，太把自己当回事。老板不敢把重要任务交给他们，因为风险太大。他们可能恃才傲物，觉得事情都是自己做起来的，老板不算什么，从而用核心业务来要挟老板，坐地起价，造成“尾大不掉”的局面。

**5. 能力强但不忠诚的人**

这类似于恋爱中遇到的“海王”或“渣女”。对方条件很好（颜值高、身材好），但情感经历混乱、不忠诚。面对这样的人，你很难想与其建立长期恋爱关系，只会担心“头上长草”。因此，你只会计算短期投入与回报，想着如何“低成本高回报”地与其交往，而不会付出真心。

## 核心总结

本节课中我们一起学习了如何在关系中区分“价值”与“利用价值”。真正的价值源于你的热情、人品、忠诚度和为他人着想的格局，而不仅仅是你的能力或可被榨取的利益。无论是职场还是情场，做一个有价值的人，才能吸引真正尊重你、愿意与你长期同行的人；若只展现“利用价值”，则只会换来算计和短期的利益交换。请审视自身，努力成为前者。
