# 推进亲密关系其实不难-P2：是否敢于进取，推进亲密关系-猫

在本节课中，我们将要学习“进取心”在建立和推进亲密关系中的核心作用。我们会探讨缺乏进取心的表现、成因及其对人际关系的影响，并理解如何通过提升进取心，来改善与他人的连接。

---

上一节我们讨论了与外界的连接减少带来的问题。本节中我们来看看，这种状态如何具体影响我们的“进取心”。

当你减少与外界的连接时，一个明显的后果是你会对自己失去要求。你可以随心所欲，想怎样就怎样。观察许多宅男，他们可能不修边幅、衣着随意、发型凌乱。

如果你每天都需要去见女孩、认识新朋友或与人会面，你还会这样做吗？显然不会。当你急剧地缩小自己的生活圈时，人就特别容易变得懒散，失去进取心。

这种状态直接导致的结果就是自尊心降低。**自尊心降低 = 对自己没有要求**。你对自己的要求越低，你的自尊心就越低。

许多社交障碍或恋爱恐惧，都与这种低自尊有很大关系。这类人在交往中，往往不是去纠缠别人，就是会跪舔。因为他们自尊心低，潜意识里认为无论自己怎么做，别人都会给予不好的反馈。

例如，很多人不愿意打扮自己，根源想法是：“**我不管怎么打扮，女孩子一定会拒绝我**。” 如果你持有这种想法，你如何改变自己？如何让别人接纳你？这本质上阻碍了你与他人发展亲密关系。

亲密关系的发展，本质是一种正向反馈的循环：**我对你有好的反馈 → 你对我有好的反馈 → 我再给你好的反馈**。通过这种循环，两人关系才会越拉越近。但如果你一开始就让人感到厌恶、鄙夷或看不上，这个循环就无法启动。

---

除了外表，缺乏进取心在言行上也有明显表现。以下是其常见的几种表现形式：

1.  **说话、做事不动脑筋**：在社交场合说话随意、不考虑他人感受，容易让人产生负面印象。
2.  **常有强烈的悲观情绪**：因为认为事情无法改变。例如，女孩不理你，就会立刻觉得“我是不是没戏了”。而有进取心的想法是：“**她现在不理我，我想办法让她理我**。”
3.  **努力却焦虑**：这种努力只是“看起来努力”，是为了给自己一个交代，而非为了真正达成目的。努力方向是盲目的。
4.  **在自卑与自负间挣扎**：自卑表现为内心敏感多疑；自负则表现为听不进批评、不能面对客观事实，以及对外界过度依赖和索求。
5.  **交往姿态失衡**：由于内心急切又怕担责，在交往中要么过于卑微（太想得到好反馈），要么过于装逼（容易激化矛盾）。例如，一上来就打压别人，看似强势，实则破坏了关系基础。

这类人容易陷入“对人不对事”的逻辑怪圈，将精力全用在琢磨“人”而非“事”上，导致交往不自然，让人感觉情商低。

---

那么，缺乏进取心的深层原因是什么？上一节我们提到了表现，现在我们来探究其根源。

核心问题在于：**自身的世界过于狭小**，且**缺乏对他人的真正兴趣和喜爱**。

由于不与外界产生深度连接，当面对陌生人和事物时，他们的第一反应是回避。如果无法回避，则往往不知所措，不知道如何回应。因为他们对他人没有真正的兴趣，交往的目的更多是“只想得到”。

想想很多人去搭讪或追求女孩时的心理：你真的对那个人有兴趣、有喜爱吗？还是说，只是觉得对方漂亮，目的仅仅是“啪啪啪”？女孩并不傻，她能感受到你是否只想占便宜。

**缺乏对他人的兴趣 → 无法表达自身真情实感 → 难以建立深度连接**。

这就是为什么很多“约炮高手”无法谈恋爱。他们并非不想，而是没有能力维持长期关系。因为他们对别人不是真的感兴趣，所有的关系都停留在短期，无法发现和把握与别人之间真正的联系与距离。

---

让我们来梳理一下整个逻辑链条：

1.  **与外界缺乏联系感** → 降低对自己的要求 → **自尊心降低**。
2.  **自尊心低** → 对自己没有要求 → **丧失进取心**。
3.  **缺乏进取心** → 在交往中表现为悲观、姿态失衡、目的性强（只想得到）→ **无法获得他人好的反馈**。
4.  **无法获得好反馈** → 进一步验证自己的低价值感 → **加深低自尊和行为模式**，形成恶性循环。
5.  同时，**自身世界狭小且对他人无真实兴趣**，导致情感表达和相处能力差，无法建立真正的亲密感。

---

本节课中我们一起学习了“进取心”对于推进亲密关系的关键作用。我们了解到，缺乏进取心不仅表现为外在不修边幅和言行随意，更深层的原因是自尊心低、自身世界封闭以及对他人缺乏真诚的兴趣。这会导致我们在人际关系中陷入恶性循环，难以获得正向反馈，从而无法建立真正的亲密连接。要打破这个循环，起点在于敢于打开自己，培养对世界和他人的真实兴趣，并从小事开始对自己有所要求，重拾进取心。
