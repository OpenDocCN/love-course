## 📖 什么是“切后排”？

“切后排”是一个战术术语，主要用于描述在团队对抗中，优先锁定并击败敌方阵型后方脆弱但高价值目标的行动。

上一节我们介绍了“切后排”的基本定义，本节中我们来看看这一战术的核心思想。

### 核心思想

其核心思想在于**以最小代价换取最大战果**。通过精准打击敌方的核心输出（如射手、法师）或关键辅助（如治疗者），能在短时间内瓦解敌方的整体战斗力，从而赢得团战胜利。

这个思想可以用一个简单的公式来概括：

**战术收益 = (敌方核心单位价值) / (我方切入成本)**

当预估的战术收益大于1时，执行“切后排”通常是明智的选择。

## 🔧 如何执行“切后排”？

理解了核心思想后，接下来我们看看执行“切后排”的具体步骤。以下是执行此战术的关键环节。

1.  **目标选择**：识别敌方阵型中价值最高、生存能力最弱的单位。通常是主要伤害来源或关键治疗者。
2.  **时机把握**：等待敌方关键控制技能进入冷却，或我方创造出一个有利的切入时机（例如敌方阵型脱节）。
3.  **路径规划**：利用地形或技能，选择一条能避开敌方前排干扰、直接抵达目标身边的路线。
4.  **技能协同**：与队友沟通，确保控制链与爆发伤害能衔接，以瞬间秒杀目标。
5.  **撤离或转火**：成功击杀后，根据战场情况选择安全撤离或转而攻击其他敌人。

## ⚠️ 注意事项与常见误区

执行“切后排”并非无脑冲锋，需要注意以下常见问题。以下是执行过程中需要警惕的几个方面。

*   **孤立无援**：切勿在未与队友沟通的情况下独自深入敌阵，这极易导致被集火秒杀。
*   **目标错误**：错误地攻击了敌方坦克或战士，浪费了关键技能和时机。
*   **时机不当**：在敌方控制技能齐全、阵型完整时强行切入，成功率极低。
*   **忽视生存**：只考虑击杀，不考虑击杀后如何存活，导致一换一甚至白给，有时并不划算。
