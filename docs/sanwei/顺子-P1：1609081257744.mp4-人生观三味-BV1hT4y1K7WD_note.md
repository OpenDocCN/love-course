## 视频播放状态 🎥

上一节我们介绍了本教程的背景，本节中我们来看看如何描述一个正在进行的活动。

文本中反复强调了“我正在看电影”这一动作。这表示一个持续进行的状态。在编程或脚本中，我们常用循环或状态标志来表示这种持续性。

例如，一个简单的伪代码循环可以表示为：
```
while (isWatchingMovie) {
    print("我正在看电影");
}
```

这个循环会持续输出状态，直到条件 `isWatchingMovie` 变为假。

2.  **执行操作**：自动运行解除音效或安装程序。

