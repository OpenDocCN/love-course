**生活中有趣的事情**：首先你可以回想记录一些旅游的事。比如说你可以在野外蹦极、你去野外烧烤，或者是你在旅游中的一些故事。比如说横店影视城，或者你当年去凤凰古城、丽江的这么一些故事。比如说你的故事是：去野外探险时迷路，最后靠着不断探寻走出来。你可以这么说：记得我大二的时候，我组织了同学一起去参加野外探险活动。在一座深山老林里，我们野炊、烧烤，每个人都玩得很开心。到了下午五点左右，我们准备回去了。我们往回走，可是却发现越走越不对劲，因为我们感觉已经不是当时那条路了。我们加快速度前进，路上有些女同学已经开始害怕了，甚至都要哭出来了。大家脸色都很沉重。因为已经快到晚上了，如果再不出去，我们一行人就可能会被困在这里。白天玩了一天，手机都没有电了，也没人带手电筒。森林里漆黑一片，肯定是非常危险的。我们转啊转啊，却发现怎么也转不出来。这时候已经是七点了，我们足足在森林里转了快两个小时。大家又累又渴，手机又没有电，眼看着天就要完全黑下来了，我们却怎么走也走不出来。当时体会到的是那种面临着恐惧、面临着团队内部的焦躁不安、面临着有生命危险的那种感觉。如果这时候是你带领的团队去探险，你会怎么办？我当时大声跟大家说：大家不要怕，我记得这条路，来过好几次，前面再走一会儿就是出口了。其实当时我心里没底，因为我根本不认识这里，没有来过。为什么我要这么说？因为我需要给大家一个希望，给团队继续前进的力量。如果一个团队因为外在的恐惧，内部没有人在把持，那么肯定会崩溃。时间到了晚上的七点半，我们终于发现了前面有一些灯火的影子。我们这群人就好像看到了生命的曙光，大家加速冲了过去，终于从这片森林出来。这一刻，我们真的好像的重生了一样，大家都欢呼雀跃，笑容又回到了脸上。看到了吗？一个小故事，你可以把它修饰得非常丰满，让听故事的人心情随着你的叙述而起波动，而且这个故事当中镶嵌了我的高价值。所以我们一定要在讲故事之前，把我们之前经历过的旅行的故事，回忆出来，整理一番，最后用嘴巴说出来。一旦我们讲故事讲的多了，我们就能够做到信手拈来，事先不一定要做大量的准备，就可以现场叙说。上面的是旅游这一方面的。你还可以通过你的爱好技能这一方面，来给对方讲解。比如说你在学习音乐、学习舞蹈、学习茶艺、学习烹饪的故事。或者是你在玩音乐、学舞蹈、修电脑的故事，甚至在做饭的故事。这些故事都可以说。我会经常跟女生说这样一个故事，就是我烹饪的故事。每次快到圣诞节了，我就想好好做一顿饭，来犒劳一下自己。因为之前一直都在外面饭店吃的饭。我觉得一个好男人，必须要能做好饭。所以那天我一大早就醒来了，到菜市场去买菜。我很久没体会到菜市场上那种吵闹的感觉了。你知道吗？那种感觉特不一样。你从一个高楼林立的、车水马龙的城市里，突然一下子去到喧闹、充满人情味的、充满人与人之间和谐相处的菜市场。市场里到处充满了叫卖声，你会发现，感觉和平时非常不一样。从职场中人与人的尔虞我诈，转换到阿姨大妈的笑脸迎人。我很喜欢这种感觉，感觉很放松。买了菜之后，我就一个人忙活了一下午，我感觉特别的充实。下午的时候，我就可以吃到我喜欢吃的菜。这个过程中女人可能会问我：你还会做菜？我说对啊，我最擅长做三道菜。你知道是什么吗？女人就好奇地问：是什么呀？我说很简单，就是煎鸡蛋、煮鸡蛋、炒鸡蛋。一般这个时候女人就会笑我，还跟我开玩笑。但是呢，她知道我是在跟她开玩笑，她也明白，其实我是一个会做饭的男人。她心里对我的感觉会不一样。你看我这样一个故事，体现出了我对生活的一种乐观、向上的、积极的想要去磨练新技能的生活态度。然后我还会让她参与进来。她问我会不会做菜啊，我就说煎鸡蛋、煮鸡蛋、炒鸡蛋，这就是幽默。你看，一个故事当中可以同时体现你的两种，甚至三种品质。只要你事先把你的故事设计好，以后你遇到了女人，或者是在你想说的场景时，你随时随地都能够拿出来使用。这就是技能这方面的话题。另外还有宠物这个话题。基本上绝大多数女人都是喜欢宠物的，所以你可以从宠物这方面下手。比如说小狗、小猫。但是你千万不要说那种让女人恶心、害怕的动物，比如说蛇、蜥蜴那一类。你可以说：我今天看到了一只狗，它让我想起了我家的华华。我在五年前养了一只狗，叫华华。虽然它不是一只名贵的狗，只是一只普通的萨摩，但是它非常的聪明。聪明到什么程度，你知道吗？它甚至都知道一加一等于几。你知道它怎么表示它知道吗？比如你向它伸一只手指，它就“汪”一声。你跟它伸两只手指，它就“汪汪”两声。你左右分别伸出一只手指，它就“汪汪汪”叫三声。自从我的华华去世之后，我就再也不敢养狗了。因为每一次养狗，我都会对它们产生非常强烈的感情。我会把它当做我的朋友，甚至是我的亲人去对待。而一旦它们再去世，我感觉我很难接受这种悲痛。你看这段故事它体现了什么？它体现了我的一个生活状态，以及对待小动物的爱心。首先介绍了我以前那只狗的情况，然后我说我以后再也不会养狗了。她就问为什么。我说狗是人类的朋友，我把它当做亲人一样，我不愿意承受这种感觉。这就体现了我对于情感的珍惜。那么，你可以从我上面说的这几个方面入手，去回忆记录诉说你的有趣的故事。其次，你可以多看书。你可以在书中看到某些观点，在跟别人聊起来的时候，
