在本节课中，我们将要学习一种名为“上推下切”的核心沟通技巧。这是一种语言的归类方法，能帮助你拓展对话、深化理解，并让交流永远有话可聊。我们将通过简单的概念和实例，让你快速掌握这项技术。

## 什么是上推下切？

上推下切本质上是语言的归类方法。语言归类分为**向上归类**、**向下归类**和**横向归类**。

**向上归类**就是“上堆”，是一种总结概括。**向下归类**就是“下切”，是一种细化分类。在使用上推下切这种方法时，它主要是和**关键词法**一起使用，即对对方话语中的关键词进行上堆或下切操作。

这项技术的核心目标是让你在对话中永远有话可聊。

## 横向归类：寻找相似元素

横向归类是为了找到同类别的、或者是具有相似关系的事物和元素。其主要目的是为了让对话的选择更加灵活。

以下是横向归类的应用实例：
*   **对方关键词**：海边。
*   **横向归类回复**：“这样感觉很惬意，跟我曾经组织队员一起在瀑布边露宿的感觉很像，你不会想到有好多玩。”

当你这样回复后，你就有了选择权：可以继续询问他在海边的事情（下切），或者引导他聊聊瀑布边露营的经历（横向引导）。

## 向上归类（上堆）：概括与共识

向上归类是将沟通的要素概括到更大的系统中。它把一个陈述或判断的要素概括为更大的分类，从而创造出对其所表达含义的更丰富、更深入的理解。

上堆能帮你找到更多话题。另一方面，它让人们从更广阔的视角、更系统的层面思考，能够更快速地察觉对方话语背后的正面动机，发现新的突破方向，从而化解冲突和矛盾。

每当双方沟通出现意见分歧时，运用向上归类可以起到非常好的建立共识的作用。

以下是上堆的应用实例：
*   **对方陈述**：“我喜欢无聊的时候出去走走。”
*   **关键词**：无聊。
*   **上堆回复**：“这是一种想要寻求变化的感觉。大海的广阔和丰富能让你很享受那种自由。”

## 向下归类（下切）：细化与澄清

下切是将整体的、虚幻的概念分解为局部的、更清晰的或者可以量化的范畴。它把一个陈述分解为很小、很具体的部分。下切操作可以结合 **`5W+1H`** （何时、何地、何人、何事、为何、如何）提问法来进行。

在下切的时候，常用的开放式问题包括：
1.  你希望怎样？
2.  发生了什么？
3.  还有呢？
4.  你计划如何去做？

以下是下切的应用实例：
*   **对方陈述**：“我喜欢无聊的时候去海边。”
*   **下切提问1（了解细节）**：“在海边时，能说说你通常会碰到什么好玩的吗？”
*   **下切提问2（了解需求）**：“为什么特别喜欢在‘无聊’的时候去海边呢？”
