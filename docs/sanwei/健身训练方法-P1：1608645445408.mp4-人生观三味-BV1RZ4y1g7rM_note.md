# 健身训练方法 - 第一课：基础拳击与手臂练习 🥊💪

在本节课中，我们将学习两个基础健身动作：转身手臂推举与基础拳击姿势。这些动作有助于增强上肢力量、协调性与核心稳定性。

---

## 转身手臂推举

上一节我们介绍了课程概述，本节中我们来看看第一个具体练习：转身手臂推举。这个动作主要锻炼肩部与手臂肌肉。

**动作要领**：转身后，将手臂稍微抬起，把手臂举到侧边直到与肩膀平行。稍微停顿，吸气，然后将手臂反向推回起始位置。重复这个练习8到12次。

**核心公式**：`动作重复次数 = 8 到 12 次`

---

## 基础拳击姿势

完成了手臂推举的练习后，接下来我们学习一个基础拳击姿势，这能提升你的协调性与爆发力。

**动作要领**：双脚与肩同宽站好，一脚在前，一脚在后，膝盖稍微弯曲。将一只手臂向前伸展，并弯曲另一只手臂置于脸颊旁进行防护。

以下是拳击姿势的步骤分解：

1.  保持双脚稳定，重心均匀分布。
2.  前手拳向前刺出，后手拳护住下颌。
3.  出拳时转动腰胯，利用身体力量。

重复交换手脚的姿势，模拟出拳动作，充分发力但注意控制幅度。

**核心代码**：`while (练习中) { 保持核心收紧； 流畅交换姿势； }`

---

## 课程总结

本节课中我们一起学习了两个基础训练动作：转身手臂推举与基础拳击姿势。坚持练习可以有效提升上肢力量与身体协调性。
