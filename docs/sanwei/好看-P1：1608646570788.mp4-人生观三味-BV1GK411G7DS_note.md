# 课程名称：城市观光小火车指南 - P1 🚂

在本节课中，我们将学习如何在一个城市中，利用观光小火车作为交通工具进行游览。我们将了解其基本运作方式、费用以及如何规划行程。

---

上一节我们介绍了课程的主题，本节中我们来看看具体的事件经过和观察到的细节。

在一次前往市中心的步行途中，我中途停下来休息。休息结束后，我继续出发，随后看到了城市中运行的观光小火车。

以下是关于这种观光小火车的具体信息：

*   这种小火车是收费的。
*   费用根据目的地不同而变化，**公式**为：`票价 = 10元 或 20元`。
*   它的路线覆盖了市中心以及酒店附近的区域。
*   乘客可以在沿途的站点随时上车或下车。

---

本节课中我们一起学习了城市观光小火车的基本使用信息。我们了解到它是一种连接市中心与酒店区的收费交通工具，票价在10至20元之间，并且乘坐方式灵活，可随时上下车。
