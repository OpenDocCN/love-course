# 形象改造课程 - P1 - 人生观三味 🎯

在本节课中，我们将要学习如何通过有效的外形改造，提升个人整体形象与吸引力。我们将从核心理念入手，逐步讲解从发型、面部修饰到身材、气质乃至服装搭配的具体方法。无论你的起点如何，掌握这些原则都能帮助你展现出更好的自己。

## 核心理念：外形不等于五官 🧠

在我们出门时，拥有一个良好的外形非常重要。很多时候，女性会直接被一个人的整体外形所吸引。

这里所说的外形，并非指五官多么精致或脸庞多么帅气，而是指整个人所透露出来的感觉。

电影《中国合伙人》的主角黄晓明在片中给人的感觉是：留着三七分发型，戴着厚重眼镜，穿着不合身的衬衫，弯腰驼背，说话缺乏自信。如果在生活中看到这样的人，你不会认为他是帅哥，也不会觉得他具有魅力或吸引力。

然而，我们在娱乐节目中看到的黄晓明，形象是帅气、阳光、有精神且充满魅力的。同样是黄晓明，因为穿着、肢体语言和气质的不同，给人的感觉天差地别。

有些特质很难改变，例如五官和身高。本教程的目的不是让你通过整形去改变这些，事实上这些也无需改变。我们可以改变的是气质、身材、发型、服装搭配和生活方式等。

“吸引家”意味着一种改变：改变过去所有不好的地方，修正言行缺陷，提升不吸引人的方面，从而达到充满魅力的状态。

## 关于外形的两种误区 ❌

在外形方面，男性通常有两种错误的意见。

第一种是认为自己不够帅，不可能有美女喜欢，认为美女只喜欢帅哥。这是一种非常错误的想法。正如黄晓明的例子所示，他的魅力并非由脸庞定义，而是通过不同的打扮、造型和肢体语言来体现的。这说明，一个人帅不帅，关键在于整体给人的感觉，而非单纯的五官或身高。一个人需要经过整体性、综合性的打扮，才能打造出理想的形象。

第二种意见是认为外貌不重要，内在才是关键，女性不会因为长相而结婚，所以无需在乎外表。持这种观点的男性通常不重视外在，整天邋里邋遢，自然给人感觉不好。

在中国这样男多女少的国家，你想在马路或聚会上吸引心仪女性的注意，第一印象至关重要。女性会根据第一印象对你做出初步判断。因此，男性的外形非常重要。

假设有十个男人同时追求一个女人，而她对这十人都只是第一次见面。她如何判断谁更优秀？很简单，通过外形。从男人的打扮可以看出他是成熟还是幼稚，温和还是偏激。所以我们一定要注意自己的形象。

此外，请思考：女性在与男性约会前，通常会花两个小时精心打扮。如果你与她约会，连两分钟都不愿意花在打扮上，你觉得这是尊重她吗？她会喜欢这样的你吗？

## 身体修饰：从头开始 ✂️

相信大家已经认识到外形的重要性了。下面，我们将探讨如何改造不满意的自己。首先从身体修饰说起。

### 发型设计

发型非常重要。当你与一个人见面时，假设你的总分是100分，那么发型就占了**41%**的分数。人们可以从发型判断一个人的性格。

发型设计最重要的是适合自己。你可以去一家比较好的理发店。什么样的理发店不错？通常是那些连锁店较多、装修看起来较高档的。找一个不错的理发师，与他沟通你想要的感觉，让他帮你设计。

**注意**：切忌将自己偶像的发型生搬硬套到自己头上，这可能造成反效果。例如，一个脸型方阔、身材高大威猛的朋友，偶像却是罗志祥。他拿着罗志祥长发的照片让理发师剪同款，结果却不伦不类。罗志祥脸型尖长、气质斯文，适合港台偶像风；而那位朋友身材粗犷豪放，应走成熟刚毅风格。因此，盲目模仿会导致不协调。

有些兄弟可能会说，自己去某些理发店时，理发师不好好剪，但给朋友剪得却很好看。以前我也遇到过同样的问题，后来发现了解决方法。

**解决方法**：以后去剪头发时，一定要穿上你觉得最帅气的衣服。因为有些人剪发时处于轻松状态，穿着T恤、短裤和人字拖。这样发型师无法判断你日常的状态，会认为你平常就是这样打扮，从而下意识地设计适合你当下打扮的发型。此外，人在为精英人群服务时更有激情，对普通人则可能懒散。所以，你要表现出自己是一个高标准的人，这样他才会尽心为你设计好发型。

**记住**：穿得帅气一些再去剪头发。

### 面部修饰

接下来是脸部的状态。

**眼睛**：核心是戴眼镜的问题。在我看来，中国**93%**的男性不适合戴眼镜。一旦戴上眼镜，就很难充分展现气质。眼睛是心灵的窗户，我们最好不要在窗户上装一块玻璃，阻碍人与人之间的交流。有些女性可能不喜欢戴眼镜的男性，但绝不会有不喜欢不戴眼镜男性的女性。因此，无论如何，尽量不戴眼镜。近视的兄弟需要想办法解决这个问题，无论是戴隐形眼镜还是做手术。我个人就做了眼睛手术，现在状态非常好。

**鼻子**：关键是检查是否有鼻毛露出。如果你与女性互动时有一撮鼻毛露出来，很可能会引起反感。鼻毛需要定期修剪。

**胡子**：胡子因人而异，取决于体毛是否旺盛。体毛旺盛的人需要定期打理，修剪出适合你的胡子形状。有些人毛发不旺盛，不怎么长胡子，这会显得比较年轻稚嫩，这类男生可以考虑往韩国偶像型风格发展。

如果你想促进胡须生长，以营造更成熟的感觉，可以尝试以下方法：
1.  购买单刀片刮胡刀。
2.  每晚用刮胡刀刮你想要长胡子的地方。
3.  刮完后，用切开的生姜涂抹刚刮过的区域。
4.  每天一次，持续两周，毛发会逐渐变得浓密。

原理是生姜中的元素具有促进毛发生长的功能。

**皮肤**：男性对皮肤通常没有太多讲究。唯一需要注意的是，有些男生脸上长痘后会留下痘印。如果痘印严重，需要找医生治疗；如果不严重，尽量保持正常作息，维持内分泌平衡。对于皮肤特别干燥、状态差的哥们，可以购买面膜护理，例如屈臣氏的蚕丝面膜就不错。

### 身材管理

身材非常重要。中国男性普遍偏瘦弱。如果身材特别瘦弱，就很难搭配衣服，因为无法把衣服撑起来，选择衣服时会非常苦恼。

对于身材特别瘦小的男生，建议：
1.  多吃一点东西。
2.  多锻炼，例如做俯卧撑。
3.  确保每周有两天可以去健身。

持续一个月后，身材会慢慢健壮起来，这对穿衣选择很有好处。

身材过胖的人选衣服也是一个困扰，建议在饮食方面加以控制，将身材调节到协调的状态。

关于身材，有一句话是：**穿衣显瘦，脱衣有肉**。如果我们能做到这种状态，女性通常会比较喜欢。

## 气质修炼：举止定乾坤 🧘

很多男性给人感觉不好，问题其实出在气质上。当你与女性接触时，映入她眼帘的就是你的气质。从你走路的姿势，到聊天时的坐姿、站姿，都能反映出你的气质，显示出你是自信还是自卑。

你可以这么认为：你举手投足之间展示出来的东西，就是你气质的表现。有些人，你第一眼就觉得他高贵；有些人，你第一眼就不想打交道；有些人，你第一眼就觉得他和善好相处。即使你从未跟他说过话，这些信息也是通过他的肢体语言透露出来的内心状态所传达的。

如果你的姿势做得不好，就会给人留下不良印象。现在，我教你如何调整肢体姿势，让自己看起来充满气质。

**站姿练习**：
1.  背部贴着墙壁站好。
2.  确保脚跟、臀部、肩膀和后脑勺都触及墙壁。
3.  保持这个姿势站一会儿。
4.  伸手到背后，确认下背部与墙壁没有离太远。如果有，收紧小腹让背部靠墙。
5.  现在离开墙壁，不要改变姿势，在房间里走一圈。
6.  整个过程，想象有根绳子牵着你的头，始终把你稍稍往上拉。
7.  把这个抬头挺胸的姿势牢牢记住。
8.  如果可以，每天练习一次。从现在起定期检查自己的姿势，如果发现驼背要立即改正。

**坐姿**：
1.  上半身保持与站姿相同的感觉（被向上牵引）。
2.  坐在凳子上时，尽量不要整个屁股都坐上去，只坐一半。
3.  身体稍稍往侧后方靠。
4.  时刻记得调整自己的状态，因为这是你气质的来源。

当你坚持调整一段时间后，身体就会记忆这种状态。以后无需刻意调整，你都能保持这种有气质的感觉。

## 服装搭配：重头戏登场 👔

讲完身体修饰，我们来讲重头戏——服装搭配。

市面上所见到的衣服都是由厂家生产的。厂家生产衣服的标准是让大多数人都适合穿。因此，当你过胖或偏瘦时，穿厂家生产的衣服效果就会变差。

对于衣服的选择，我们有两种方式：
1.  去定做，例如衬衫、西服、马甲等。
2.  （我比较推荐）去购物网站搜索你想要的服饰，例如“衬衫”，再加入你想要的风格关键词。这里我比较推荐“韩版”，因为大家都是亚洲人，身型比较相似。那么你就搜索“衬衫 韩版”。

搜索出来的结果会有很多模特穿着不同样子的衬衫，他们的搭配通常都比较不错。你可以选择与你身型、气质相似的模特，模仿他穿的衣服、裤子和配饰。这样可以弥补你在服装搭配知识上的欠缺，还能节省大量时间。

平时你也可以多看看时尚杂志，例如《男人装》，多逛一些像TOPMAN这样的店铺。看得多了，你的“眼力”就练出来了，也就更懂得如何搭配了。当然，也要结合下面我要讲的专业知识。

### 搭配原则与技巧

当我们穿的衣服偏大或偏小时，不管五官多好看，给人的感觉都不好。另外，“挫男”通常总是穿运动装，因为他不会搭配，也从不去想怎么搭配，这是不行的。

一个男人不管什么时候，都要给自己准备几套适用于不同场合的衣服，例如：
*   正式场合的
*   商务的
*   运动的
*   休闲的

至少我们要准备这四套衣服。

**衬衫**是男人必备的装备，不管是休闲还是正式场合都能搭配。男人需要对衬衫有一定的了解。衬衫分两种：修身版型和宽松版型。我们亚洲人一般适合穿比较修身的，因为亚洲人身型相对瘦小，修身衬衫穿起来会比较好看。黄晓明在《中国合伙人》中形象不佳，很大原因就是因为他穿了那件很大、很不合身的衬衫。

**裤子**方面可以选择一些休闲款式，例如锥形裤、铅笔裤，能显示腿形的裤子会比较性感。

**配饰**方面：
*   当你穿圆领衣服时，用一个较长的吊坠，在胸前形成三角形，整体有拉长效果，会让脸显得比较小。
*   如果穿低领衣服，就戴一个较短的链子，短链子在胸前形成一个圆形，有一种对称的效果。
*   选择链子时不要选择黄色的，因为我们本身就是黄种人，选择金黄色会显得不太好看。可以选择银色、黑色的链子。链子不在乎大小，但要上档次、有质感。女性非常看重这些细节。
*   穿短袖衣服露出手臂时，手上光溜溜的不太好看。如果戴上手链或手表，就会显得好看多了。手链可以在网上买到质感不错的。手表一般要戴比较好的，因为手表是男人身份的象征。如果经济能力有限，戴手链就好。如果有一定的经济能力，可以购买一些比较好的手表。

### 核心原则：呼应原则

很多人说自己不会搭配衣服。现在教大家一个原则，以后按照这个原则去搭配，保证效果不会差。这个原则就是**呼应原则**。

简单来说，就是你身上的服装颜色必须相互呼应。例如：
*   你穿一件粉色的衣服（亮色），要突出粉色，就要选择一件暗色、朴素的裤子，例如黑色或深蓝色。鞋子则可以选择亮色，例如棕色。这样穿出来的效果就是相互呼应。
*   如果你穿商务套装，主要显露部分是深色（如深色西装、深色裤子），那么在选择其他部位搭配时，就可以选择亮色，例如穿一件亮色的衬衫（粉色、蓝色、白色都行）。这时最好不要选择暗色衬衫（如黑色），不然会显得沉闷。然后，棕色鞋子、棕色皮带、棕色手表都可以，这都是相互呼应。

你只要按照“色调统一，一身一显”的思路去呼应着搭配，就会比较好看。

男人身上吸引人的地方，在于他精致的生活方式。男人不需要花费太多时间在打扮上，但如果你连最基本的打扮都不愿意做，那你和女性就会缺乏共同话题。
