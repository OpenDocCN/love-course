# 烹饪教程 P1：家常豆豉蒸鱼 🐟

在本节课中，我们将学习一道家常菜——豆豉蒸鱼的制作方法。这道菜风味浓郁，制作过程简单，适合厨房新手尝试。我们将从准备配料开始，逐步讲解烹饪的每一个步骤。

## 准备与炒制配料

上一节我们介绍了课程内容，本节中我们来看看具体的准备工作。首先需要处理基础配料，并通过炒制激发其香气。

以下是炒制豆豉配料的关键步骤：
*   下入豆豉，在锅中炒出香味。
*   放入切好的姜末和蒜末。
*   加入适量的小米辣。
*   整个炒制过程持续约50秒即可。

## 烹煮与蒸制鱼肉

配料炒香后，便可以开始烹煮鱼肉。这个阶段主要目的是让鱼肉吸收汤汁的味道并完全蒸熟。

以下是烹煮与蒸制的步骤：
*   在锅中加入少许酱油。
*   待锅中水烧开后，放入准备好的鱼。
*   盖上锅盖进行蒸制。
*   鱼肉蒸制约10分钟后已基本成熟。
*   若要追求更加入味和软烂的口感，可以总计蒸制30分钟。

本节课中我们一起学习了豆豉蒸鱼从备料到完成的完整流程。关键在于先将豆豉等配料炒香，再通过蒸制让鱼肉充分吸收风味。希望你能成功复刻这道美味佳肴。
