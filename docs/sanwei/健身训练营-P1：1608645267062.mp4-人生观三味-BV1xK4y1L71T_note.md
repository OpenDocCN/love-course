在本节课中，我们将要学习两个基础的肩部训练动作：哑铃前平举和哑铃侧平举。这些动作能有效锻炼肩部前束和侧束的肌肉，帮助你塑造更宽阔、更有型的肩膀。

## 动作一：哑铃前平举

上一节我们介绍了课程概述，本节中我们来看看第一个具体动作——哑铃前平举。这个动作主要针对肩部前侧的三角肌前束。

**起始姿势**：身体站直，双手各持一个哑铃，掌心朝向自己，将哑铃置于大腿前侧。

**动作执行**：
以下是哑铃前平举的分解步骤：

1.  保持手肘微屈且固定，将哑铃向前上方举起，直至达到下巴高度。
2.  在举起过程中，将掌心旋转180度，使其最终朝向面部。
3.  在动作顶端稍作停顿，感受肩部前束的收缩。
4.  有控制地将哑铃沿原路径下放，同时将掌心旋转回起始方向。

**训练安排**：
该动作的建议训练量为：**4组，每组10到12次**。

## 动作二：哑铃侧平举

完成了前平举的学习后，我们接下来看看如何锻炼肩部侧束，这需要用到哑铃侧平举这个动作。

**起始姿势**：身体站直，双手各持一个哑铃，自然垂于身体两侧，掌心相对。

**动作执行**：
以下是哑铃侧平举的分解步骤：

1.  保持手肘微屈，将哑铃向身体两侧平举，直至手臂与地面平行。
2.  在动作顶端，想象你在用肘部引导向上，并感受肩部侧束的强烈挤压感。
3.  有控制地将哑铃下放回起始位置。

**训练安排**：
该动作的建议训练量同样为：**4组，每组10到12次**。
