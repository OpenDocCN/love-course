## 概述 📋

我们将创建一个程序来管理鞋子的库存信息。这个程序的核心是使用一个**字典**来存储数据，并通过**函数**来执行添加和查询操作。通过这个例子，你将理解数据结构与函数如何协同工作。

---

## 核心数据结构

首先，我们需要一种方式来存储每双鞋的信息。我们将使用Python的**字典**来实现。每双鞋的信息包括品牌、尺码和颜色。

以下是定义一双鞋信息的代码示例：
```python
shoe = {
    "brand": "Nike",
    "size": 42,
    "color": "Black"
}
```

---

## 创建库存列表

单一的字典只能存储一双鞋的信息。为了管理多双鞋，我们需要一个列表来存放所有这些字典。

上一节我们介绍了如何表示单件商品，本节中我们来看看如何管理整个库存。

我们将创建一个空列表作为我们的库存：
```python
inventory = []
```

---

## 添加商品到库存

有了库存列表后，我们需要一个方法向其中添加新的鞋子。我们将创建一个函数来完成这个任务。

以下是添加新鞋到库存的步骤：
1.  创建一个新的字典来存放新鞋的信息。
2.  从用户输入中获取鞋子的品牌、尺码和颜色。
3.  将这个新字典添加到库存列表中。

```python
def add_shoe():
    brand = input("输入品牌: ")
    size = int(input("输入尺码: "))
    color = input("输入颜色: ")

    new_shoe = {"brand": brand, "size": size, "color": color}
    inventory.append(new_shoe)
    print("鞋子已添加到库存。")
```

---

## 查询库存信息

仅仅添加商品还不够，我们还需要能够查看库存里有什么。我们将创建另一个函数来查询并显示所有库存的鞋子。

以下是查询并显示所有库存的代码：
```python
def show_inventory():
    print("\n当前库存：")
    for index, shoe in enumerate(inventory):
        print(f"{index+1}. 品牌：{shoe['brand']}, 尺码：{shoe['size']}, 颜色：{shoe['color']}")
```

---

## 构建主程序循环

现在，我们已经有了添加和显示库存的功能。我们需要一个主程序循环让用户可以选择执行哪个操作。

我们将使用一个`while`循环来持续显示菜单，直到用户选择退出。

以下是主程序循环的结构：
```python
while True:
    print("\n--- 靓鞋库存管理系统 ---")
    print("1. 添加新鞋")
    print("2. 显示所有库存")
    print("3. 退出")

    choice = input("请选择操作 (1/2/3): ")

    if choice == '1':
        add_shoe()
    elif choice == '2':
        show_inventory()
    elif choice == '3':
        print("感谢使用，再见！")
        break
    else:
        print("无效选择，请重试。")
```
