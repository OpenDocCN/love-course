# 麻将基础教程 - 第1课：认识“顺子” 🀄

在本节课中，我们将要学习麻将中的一个核心概念——“顺子”。理解“顺子”是学习麻将牌型组合的基础，对于后续掌握听牌、胡牌至关重要。

## 什么是顺子？

上一节我们提到了学习“顺子”的重要性，本节中我们来看看它的具体定义。

在麻将中，“顺子”指的是由**同一种花色**的、**数字连续**的三张牌所组成的牌型组合。它只存在于“万”、“条”、“筒”这三种数牌中。

其核心概念可以用一个简单的公式表示：

**顺子 = 同花色且数字连续的三张牌**

例如，“二万、三万、四万”或“五筒、六筒、七筒”都是标准的顺子。

## 如何形成顺子？

理解了定义后，我们来看看顺子在实战中是如何形成的。这通常涉及到摸牌、吃牌等操作。

以下是形成顺子的几种常见方式：

*   **自己摸牌凑成**：通过自己从牌墙中摸进所需的牌，与手中已有的两张牌组成顺子。
*   **吃上家的牌**：当你的上家打出一张牌，而这张牌恰好能与你手中的两张牌组成顺子时，你可以声明“吃”，从而组成一个顺子。这是从其他玩家处获得牌以组成顺子的主要方式。

## 一个实战场景分析

现在，让我们结合一个具体的例子来加深理解。

（开启电源），（家里没下轿，上家捡了个便宜），（在家里没下车，上家捡了个便宜）。

这段描述模拟了一个实战片段：
1.  场景开始（开启电源）。
2.  描述者（“家里”）可能未能及时“吃”牌组成顺子（没下轿/没下车）。
3.  因此，他的上家获得了机会，通过这张牌获得了利益（捡了个便宜）。这生动说明了在游戏中，能否及时利用牌来组成顺子，会影响局势的走向。
