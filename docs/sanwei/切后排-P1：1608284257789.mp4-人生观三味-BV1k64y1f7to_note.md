# 切后排 - 课程P1：核心概念与开局分析 🎯

在本节课中，我们将学习如何分析一场游戏的开局，理解关键单位的角色与作用，并掌握“切后排”战术的基本思路。我们将通过一个具体的对局案例，拆解双方的初始布局与核心策略。

---

上一节我们介绍了课程的整体目标，本节中我们来看看本局对战的初始阵容与关键单位。

双方阵营的核心单位几乎同时出现。一方是“大牛”与“种子”，另一方则是两个属国“Snowbee”和“胡老大”。

以下是双方开局的关键单位分析：

*   **独门长城**：此单位属于“类可育龙”阵营。其特点是**坦克单位占比非常高**。
*   **小兔子**：由于坦克比例高，“击下”任务通常由“小兔子”承担。核心逻辑是：**一旦有坦克可攻击，就贴走一个坦克进行射击**。
*   **属国方**：本局中是“胡老大”和“Snowbee”两家。“胡老大”目前处于明牌状态，其行为模式是：**每次发动属国攻击，都执行一次“滴滴射”操作**。

目前，“胡老大”的具体表现尚不确定，有待观察。在接下来的课程中，我们将持续分析其战术展开。

（敲铃声）

---

本节课中，我们一起学习了如何解析游戏开局。我们认识了“独门长城”与“小兔子”的搭配逻辑，其核心在于**高坦克比例**与**针对坦克的集火原则**。同时，我们也了解了属国“胡老大”的**明牌“滴滴射”** 行为模式。这些是理解后续战术发展的基础。
