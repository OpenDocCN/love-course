# 31.《PIMP：2.第二部分：内核模块：3.流动感（同义：理所当然感、授权：流动感模块-视频：3.

在本节课中，我们将要学习“流动感”模块，并分析一个错误的互动示范。理解这种“理所当然”或“授权”的感觉，对于建立自然、顺畅的社交互动至关重要。

---

上一节我们介绍了内核模块的基本概念，本节中我们来看看“流动感”的具体体现。流动感是指在互动中保持连贯、自信的状态，避免生硬和犹豫。

以下是错误示范的关键问题分析：

*   **行为缺乏连贯性**：互动过程是断断续续的。
*   **指令生硬**：交流方式像是在呼来喝去。
*   **节奏混乱**：行动没有稳定的节奏，一会儿做这个，一会儿做那个。

---

本节课中我们一起学习了“流动感”模块的核心反面教材。错误的示范表现为互动不连贯、指令生硬且节奏混乱，这破坏了自然流畅的互动感觉。掌握流动感的关键在于让行为变得连贯且理所当然。
