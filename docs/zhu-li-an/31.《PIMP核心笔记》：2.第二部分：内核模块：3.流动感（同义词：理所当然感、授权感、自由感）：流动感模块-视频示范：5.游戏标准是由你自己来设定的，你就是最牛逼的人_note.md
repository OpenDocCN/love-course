# 课程31：《PIMP 核心笔记》第二部分：内核模块3 - 流动感 🌀

在本节课中，我们将要学习“流动感”模块的核心概念。这个模块也被称为“理所当然感”、“授权感”或“自由感”。其核心在于，由你个人来设定互动的标准和游戏规则，从而建立强大的内在框架和吸引力。

---

## 模块核心：设定你自己的游戏标准 🎯

上一节我们介绍了内核模块的基本概念，本节中我们来看看“流动感”的具体实践。其根本原则是：**游戏标准由你自己设定，你就是最权威的人**。

你需要完全接纳自己，包括接纳自己是一个“怪人”或与众不同的人。必须停止取悦他人、迎合外部标准或为他人的眼光而活。

例如，如果你认为当下需要改变发型来显得更酷，就立刻冲动地去改变，这种行为是不可取的。

---

## 建立你的现实框架 💪

当与女孩互动时，关键在于确信。如果你确信自己的状态，女孩就会被拉入你的现实感、你的框架和你的游戏规则之中。

无论你外表如何，你都是权威，因为游戏规则由你制定。

以下是基于此原则的具体心态示例：

*   **关于发型**：如果你是短发，那么短发就是最酷的，留长发的则是“死宅男”或“基佬”。如果你是长发，那么短发的人就是“基佬”，并且是“拿着剃须刀互相给彼此的男朋友剃短的”。这就是你应拥有的感觉。
*   **关于互动**：如果你去搭讪一个女孩却不知道说什么，那么“不知道说什么”这件事本身就是最牛逼的。那些事先准备好台词的人反而很烂、很垃圾。
*   **关于拒绝**：如果你被女孩拒绝，这恰恰是你牛逼的证明，因为你很真实。那些从不被女人拒绝的人才是垃圾，因为他们迎合别人、缺乏个性、试图讨好所有人。

**核心公式：你的主观确信 -> 你的强大框架 -> 他人的感知与认同**

## 实践应用与案例 🌟

这完全取决于你的主观决定。如果你自我感觉强大，并拥有强大的框架，女孩就会感受到，并认为你很酷、很强。

举个例子：即使你是一个很“宅”的人，只要你有强大的框架（例如坚信“这就是我喜欢的东西”），女孩也会被你吸入你的框架之中。

相处一段时间后，女孩甚至可能开始觉得，电脑游戏是世界上最牛逼的东西，就像你认为你的工作一样。

以下是一些实践中的观察：

*   我交往过的每个女朋友，最终都会觉得“把妹”是一件很牛逼的事情。
*   以路易为例，他的发际线高、头发少，喜欢把妹和讲YY课，声音大，模仿女人有时特别像。但这些都不重要，这就是**设定标准**。你会看到女孩最终都会喜欢他。
*   关键就在于，一旦你确信了，对方就会被吸入你的框架。

## 总结与行动要点 ✅

本节课中，我们一起学习了“流动感”模块的精髓。

记住核心：**我是一个男人，我拥抱我所拥有的一切。标准由我制定。**

这种心态会给你带来积极的预设，赋予你理所当然的自信感，让你在互动中完全放松下来。

**最终行动指南**：
1.  完全接纳独特的自我。
2.  停止为他人改变或寻求认可。
3.  在任何互动中，由你内心确信的标准来定义价值。
4.  拥抱你现有的一切，并将其作为你强大框架的基石。
