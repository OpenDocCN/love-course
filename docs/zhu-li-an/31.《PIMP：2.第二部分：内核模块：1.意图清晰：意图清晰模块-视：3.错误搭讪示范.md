# 31.《PIMP：2.第二部分：内核模块：1.意图清晰：意图清晰模块-视：3.错误搭讪示范

在本节课中，我们将通过一个具体的视频示范，学习在社交互动中一个典型的错误搭讪方式。我们将分析其问题所在，并理解为何这种方式无法有效传达清晰的意图。

上一节我们介绍了意图清晰的重要性，本节中我们来看看一个反面的实际案例。

## 错误示范分析

以下是一个典型的错误搭讪开场白示例：

```
你去和他说嗨，你不能这样，嗨。
```

这段对话包含了几个关键问题。

## 核心问题列表

以下是这个错误示范中存在的几个主要问题：

1.  **指令模糊**：话语以“你去和他说嗨”开头，这是一个对第三方或自己的命令，而非直接与目标对象进行的交流，导致意图传递对象错位。
2.  **自我否定**：紧接着的“你不能这样”构成了对前一个指令或行为的即时否定，制造了矛盾与犹豫的信号。
3.  **内容空洞**：重复的“嗨”缺乏任何实质性的、能开启对话或表达兴趣的内容，显得敷衍且目的不明确。

## 问题公式化表达

我们可以将这种错误模式简化为一个**公式**：

**无效搭讪 ≈ 模糊指令 + 自我矛盾 + 内容空洞**

这个公式概括了导致沟通失败的核心要素。

## 正确做法启示

本节课中我们一起学习了错误搭讪的一个反面教材。其核心教训在于：直接、肯定且带有实质内容的开场，远比迂回、自我否定且空洞的方式更能传达清晰意图。在后续课程中，我们将对比学习正确的表达方式。
