# 31.《PIMP：2.第二部分：内核模块：1.意图清晰：意图清晰模块-视：6.意图不清晰示

在本节课中，我们将通过一个视频示范，具体分析一个“意图不清晰”的互动案例。我们将拆解这个案例中的言语和行为，理解为何它缺乏吸引力，并明确“意图清晰”所需的核心要素。

## 概述

上一节我们探讨了“意图清晰”的理论基础。本节我们将观看一个实际示范，直观地感受当意图表达模糊时，互动会呈现出怎样的状态。

这个示范模拟了一种常见的、但效果不佳的打招呼方式。

## 示范案例分析

以下是视频中展示的意图不清晰的互动范例：

```
男：“嗨。”
（短暂停顿后）
男：“嗨，你好，我是米亚。”
```

或者，如同许多人在初次接触女性时的典型方式：

```
男：“嗨。”
（然后就没有更多有效交流）
```

## 关键问题解析

这种互动方式的核心问题在于：**言语背后缺乏力量**。

“嗨”只是一个开启对话的信号，但如果后续没有承载明确意图或价值的跟进，它就显得空洞、犹豫且缺乏吸引力。它没有传递出你为何要与对方交谈，也没有展示出任何自信或独特性。

## 核心要点总结

本节课我们一起分析了一个意图不清晰的示范。我们看到了一个简单的“嗨”如果没有后续有力内容的支撑，会显得多么苍白无力。关键在于，你的**每一句话**，尤其是开场白，都应该直接或间接地传递出你的自信、目的和独特性。模糊的意图会导致模糊的结果。

在接下来的课程中，我们将对比学习意图清晰的正面示范，看看如何将力量与目的注入你的言行之中。
