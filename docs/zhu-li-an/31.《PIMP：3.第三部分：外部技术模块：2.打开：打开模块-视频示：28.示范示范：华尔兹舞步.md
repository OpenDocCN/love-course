上一节我们介绍了外部技术模块的基本框架，本节中我们来看看一个具体的动作示范：“华尔兹舞步”。

这个动作示范包含了一系列连贯的步骤和身体协调要求。以下是其执行过程的分解说明。

**步骤分解：**
1.  起始姿势：双脚并拢，身体保持直立平衡。
2.  第一步：右脚向侧方迈出，同时身体重心右移。
3.  第二步：左脚向右脚并拢，恢复平衡。
4.  第三步：左脚向侧方迈出，同时身体重心左移。
5.  第四步：右脚向左脚并拢，完成一个循环。

其移动模式可以用一个简单的**代码**序列表示：
```python
步骤 = [“右移”, “并拢”, “左移”, “并拢”]
for 动作 in 步骤:
    执行(动作)
```

在理解基础步骤后，需要注意动作的流畅性与节奏。步伐应与内在节拍同步，通常以三拍为一个循环单元，其节奏模式可表示为 **公式：1-2-3 | 1-2-3**。
