# 31.《PIMP：3.第三部分：外部技术模块：3.上钩：上钩模块-视频示：41.高能量+高

在本节课中，我们将学习“上钩”模块的一个具体应用：如何运用**高能量**与**高社交强度**来开启对话。我们将通过一个视频示范的文本分析，来理解这种互动方式的核心要素与表达技巧。

---

上一节我们介绍了上钩模块的基本概念，本节中我们来看看一个结合了高能量与高社交强度的具体示范。

对话示范的原文如下：
> 嘿你好 哇你舞跳太棒了，我覺得你是全場最耀眼的那個，你是瀋陽姑娘嗎。

这段开场白包含了几个关键组成部分。以下是其核心要点的分解：

1.  **热情问候与即时赞美**：以“嘿你好”快速建立联系，并立刻用“哇你舞跳太棒了”给出具体、强烈的正面反馈。
2.  **升级评价**：将赞美从具体行为提升到个人比较层面，“我覺得你是全場最耀眼的那個”使用了最高级表述。
3.  **提出轻松话题**：最后以“你是瀋陽姑娘嗎”提出一个关于对方背景的、低压力的问题，将对话导向轻松交流。

值得注意的是，尽管表达方式充满活力，但讲述者强调：**我的講話的背後依然有一種深度強度**。这意味着，外在的高能量与热情，需要由内在的自信、专注和意图来支撑，而非流于表面的浮夸。

---

本节课中我们一起学习了如何运用高能量与高社交强度进行对话切入。其核心模式可总结为：**热情开场 → 具体且强烈的赞美 → 提出开放式轻松问题**。同时，我们理解了外在表现需要内在的“深度强度”作为基础，这样才能确保互动既富有感染力，又具备足够的吸引力和引导性。
