# 课程31：《PIMP 核心笔记》第三部分：外部技术模块 - 4. 创造气氛、情绪共振：视频示范 - 5. 角色扮演——把女孩当做自己的女朋友 👫

在本节课中，我们将要学习一种名为“角色扮演”的互动技巧，特别是如何通过将女孩称为“女朋友”来快速建立亲密感和专属氛围。

## 方法概述

下一个方法叫做角色扮演。我喜欢角色扮演的原因是，女孩可以借着扮演的角色表达出很多内容，又无需对自己表达的任何内容承担责任。

## 核心技巧：称呼对方为“女朋友”

我个人最喜欢的角色扮演的方法很简单，就是把女孩叫做我的女朋友。

**示例对话**：
> “小白，我觉得我们爱情走到了尽头，我觉得不能再一块玩耍了。小白，我们分手吧，喝完这杯咖啡我们就分手，各回各家各找各妈。来，当我最后五分钟的女朋友吧。”

这个女孩很明显知道你们并不是男女朋友关系，但是女孩不会说“啊真的吗我们是男女朋友吗？真的假的？”，因为女孩知道这是一个角色扮演。

## 角色扮演的作用

那么，这样做有什么用呢？

以下是角色扮演的核心作用：

1.  **强化二人合一的感觉**：它创造出“嘿，你看现在就我们两个人，我们是男女朋友”的专属情境。
2.  **聚焦双方注意力**：女孩的注意力完全被拉到了你和女孩之间。

## 与共谋理论的对比

上一节我们介绍了共谋理论，这里可以做一个清晰的对比。与共谋理论不同的是，共谋把你和女孩的注意力，拉到了其他人身上。而角色扮演把注意力完全拉到了你和女孩，你们两个人这个小圈子里。

## 总结

本节课中，我们一起学习了“角色扮演”这一技巧，特别是通过将女孩称为“女朋友”来营造亲密氛围。这种方法的核心在于，它允许双方在一种安全、有趣的“扮演”框架下表达情感，从而**强化二人合一的专属感**，并将所有注意力聚焦在你们两人构成的小世界内部。
