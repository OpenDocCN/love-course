# 课程31：《PIMP 核心笔记》第三部分：外部技术模块 - 打开模块：视频示范11 - 嗨FIVE示范，最安全的肢体碰触 👋

在本节课中，我们将学习一种被称为“嗨FIVE”的社交互动技巧。这是一种非常安全且有效的肢体接触方式，旨在通过轻松、无压力的击掌动作，在互动初期自然地拉近与对方的距离。

## 概述

“嗨FIVE”技巧的核心在于，它通过一个看似随意、甚至有些“傻气”的理由，来发起一次击掌。这种方式能够巧妙地绕过直接肢体接触可能带来的尴尬或冒犯感，让互动氛围变得轻松有趣。

## 技巧原理与应用场景

上一节我们介绍了肢体接触在社交互动中的重要性。本节中，我们来看看如何以一种最安全、最不具威胁性的方式开启第一次接触。

其核心逻辑是：**寻找或创造一个无关紧要的共同点或小事件，并将其作为击掌庆祝的理由**。这个理由本身不需要多么严肃或合理，其目的是为肢体接触提供一个自然的“借口”。

以下是一个具体的应用实例：

在我身上发生过这样的事情。我和女孩刚开始聊天的时候，没有第一时间去触碰她。我也不想非常粗鲁冒失地，马上去触碰这个女孩。

于是，我就会说：“嘿，你是沈阳本地人吗？”
女孩说：“是啊。”
我接着说：“真的吗？我真的特别喜欢沈阳女孩。耶！你是沈阳本地人吗？我不是沈阳的。好，谢天谢地。耶！”

任何很傻的原因，都可以让你去和女孩high five。

## 关键步骤解析

以下是实施“嗨FIVE”技巧的几个关键点：

1.  **发起话题**：以一个简单、轻松的问题开始对话，例如询问对方的家乡、对某件事的看法等。
2.  **获取回应**：等待对方给出肯定的或任何可回应的答案。
3.  **制造理由**：将对方的回应与一个积极、庆祝性的情绪联系起来，哪怕这个联系看起来有点“傻”或牵强。
4.  **主动邀请**：在说出庆祝语（如“耶！”）的同时，自然而然地伸出手，做出准备击掌的姿势。

公式可以概括为：**简单问题 + 对方回应 + 庆祝性解读 = 嗨FIVE邀请**

## 总结

本节课中，我们一起学习了“嗨FIVE”这一安全肢体接触技巧。它通过一个轻松的话题和一个小小的庆祝理由，为击掌创造了一个完美的时机。记住，理由不必完美，关键在于传递出轻松、友好的态度，从而自然过渡到第一次肢体互动。掌握这个方法，可以帮助你在社交开场时更顺畅地建立连接。
