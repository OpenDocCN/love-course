# 31.《PIMP：3.第三部分：外部技术模块：1.外部技术模块：外部技术模块介绍：4.施压，减压实

在本节课中，我们将通过一个实战视频示范，学习外部技术模块中的核心互动技巧：**施压**与**减压**。你将看到这两种技术在实际对话中是如何被运用的。

上一节我们介绍了外部技术模块的基本概念，本节中我们来看看它们在真实场景下的具体应用。

## 视频示范解析

以下视频片段展示了一段包含施压与减压技术的互动对话。请注意对话中的情绪波动和语言策略的转换。

> 人物A：有Wi-Fi对不对？
>
> 人物B：有啊，忘了，干嘛呀，你干嘛呀？
>
> 人物A：我给你加Wi-Fi啊。
>
> 人物B：你走你走我没转儿子，你别过来，你最好别过来。
>
> 人物A：告诉你妈吧。
>
> 人物B：我不告诉你，不是不是不是这个，来来，忘了，我帮你加我帮你加，你过来呀。
>
> 人物A：哈哈哈，你很，你过来，你过来宝宝，我想让你过来，你过来，你哪有忘了呀，你别过来呀，你加不加？

## 核心技巧分解

通过分析以上对话，我们可以提炼出两个核心的外部技术操作。

以下是本示范中运用的关键技巧列表：

1.  **施压**
    *   **定义**：通过语言或行动，主动制造紧张感、挑战或不确定性，以测试对方反应或推动互动进程。
    *   **实例**：人物A说“告诉你妈吧”和“你加不加？”，这是在施加压力，将对方置于一个需要做出选择或回应的境地。

2.  **减压**
    *   **定义**：在施压后或气氛紧张时，通过幽默、让步或转换话题，主动缓解紧张气氛，使互动回归轻松或可控状态。
    *   **实例**：人物B说“我帮你加我帮你加”和人物A说“哈哈哈，你很…你过来宝宝”，这些语句旨在缓和之前“施压”造成的对抗感。

## 技巧运用要点

你会发现，在实际互动中，**施压**和**减压**并非孤立存在，而是紧密交织、快速转换的。有效的互动往往是在施加一定压力后，适时地给予减压，从而引导对话的节奏和方向。其核心循环可以简化为一个公式：

**互动流程 ≈ 施压 → [观察反应] → 减压 → [建立舒适感] → 再次施压…**

本节课中我们一起学习了外部技术模块中的“施压”与“减压”实战应用。我们通过分析一段对话，明确了施压是制造推动力，而减压是提供舒适区，两者交替使用是掌控互动节奏的关键。掌握这种动态平衡，是提升社交互动能力的重要一步。
