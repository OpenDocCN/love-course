# 课程31：《PIMP 核心笔记》第三部分：外部技术模块5：收场 - 视频范例75：带领方法之“切断对话” 🎯

在本节课中，我们将学习一个关键的互动技巧：如何主动切断无益的对话，并引导话题走向，从而掌握互动的主动权。

---

上一节我们讨论了互动中的话题选择，本节中我们来看看一个具体的带领方法：**切断女孩的对话**。这个方法的核心在于，你无需被动地跟随对方的所有话题，尤其是那些对增进你们关系无益的内容。

如果女孩开启了一个不会让你们感到开心、或对关系发展没有帮助的话题，你没有义务一直倾听。不必让她随心所欲地主导所有谈话内容，也无需一直跟随她的带领。

以下是两个典型的需要被切断的对话例子：

*   **关于宠物的悲伤话题**：女孩对你说：“我家的小泰迪上週得病去世了，他叫Nao Nao。我一直把他當作我最好的朋友，每天我都陪他在一起。”
*   **关于前任的怀旧话题**：女孩说：“這個音樂又讓我想起了前男友，本來還想和他一直一直在一起的，結果他就那樣無情的走了。”

此时，如果你的真实想法是：“你的泰迪去世了和我有什麼關係？”或“你前男友和我有什麼關係？我憑什麼在這一直聽？”，那么实话是，你无需听她没完没了地讲下去。

你应该做的，是**立刻切断这个对话**，然后**引入一个你喜欢的新话题**。

以下是切换话题的两种话术示例：

1.  **在娱乐场所**：你可以说：“好了，不要想不開心的了。我們在酒吧/夜店，我們來玩個新遊戲，特別好玩的遊戲。我們要開心起來。”
2.  **在休闲场所**：你可以说：“不要想了，我們在咖啡廳，我們在享受我們的下午時光。對了，最近的那個電影你看了嗎？”

**关键在于一致性**：你不必言不由衷。如果你内心根本不想表达同情，就无需强迫自己说：“天哪，你的小狗好可憐，你一定特別傷心……”或“他怎麼可以這麼對你，如果是我的話……”。

记住，你要活在你自己的现实感里。**一切规则由你决定**。如果中途出现的事情会让气氛变得低落，无法推进两人关系，那么就立刻切断对话。

这样做能强化一个核心事实：**你是带领者（老板）**。你需要有这样的心态：
`if (topic.isBoringOrNegative()) { cutOff(); introduceNewTopic(); }`

你要知道，在这个框架下，是女孩在向你证明自己，试图给你留下好印象。这实质上是在让她追逐你。此时，女孩的思维模式应该是：“天哪，我得足够好才行。我这样做可以吗？我说这个可以吗？”——她在对你做出反应，她在证明自己。

这种情况对于建立吸引是非常有利的。

---

本节课我们一起学习了“切断对话”这一带领方法。其核心是：主动过滤无益话题，果断引导至积极方向，从而建立你作为互动带领者的框架，让对方进入追逐你的模式。
