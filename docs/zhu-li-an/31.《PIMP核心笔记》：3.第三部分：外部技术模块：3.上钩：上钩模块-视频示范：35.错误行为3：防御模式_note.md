# 课程31：《PIMP 核心笔记》第三部分：外部技术模块 - 上钩模块视频示范：错误行为3：防御模式 🛡️

在本节课中，我们将学习一种被称为“防御模式”的常见错误反应形式，并通过一个具体示例来理解其表现。

---

上一节我们介绍了其他错误行为，本节中我们来看看“防御模式”。

防御模式是一种在社交互动中，个体因感到威胁或不适而表现出的戒备和疏离状态。其核心特征是通过言语或非言语信号，提前阻断或冷却可能的深入交流。

以下是一个展示防御模式的具体对话示例：

*   “嘿，对不起，我只是过来打个招呼。”
*   “嘿，你好。”

在这个简短的交流中，第一句话“嘿，对不起，我只是过来打个招呼”清晰地体现了防御模式。发言者在一开始就使用了“对不起”这样的道歉语，并立即用“只是”来限定自己行为的目的和范围，表明其意图仅限于最表层的寒暄，潜意识里是在设立边界，防止对话向更私人或深入的领域发展。这种预先的“免责声明”是防御模式的典型标志。

---

本节课中我们一起学习了“防御模式”这一错误行为。我们了解到，防御模式的核心在于通过提前道歉和限定互动范围来建立心理防线，这会无形中阻碍自然、流畅的社交连接。识别并避免这种模式，是进行有效外部互动的重要一步。
