## 3. 练习：练习7—保持理智 🧠

在本节课中，我们将要学习一个名为“保持理智”的练习。这个练习的核心是培养一种基础能力：无论外界发生什么，你都能保持专注和稳定，不被干扰。我们将通过一个30秒的对话练习来实现这个目标。

### 练习概述

这个练习要求两人一组进行。在30秒内，其中一人需要持续说出自己今天学到的课程内容。另一人的任务则是通过言语或动作制造干扰和紧张感。说话者的目标是**不受干扰**，保持自己的节奏，持续陈述学习内容，仿佛干扰者不存在一样。

### 练习步骤详解

以下是进行此练习的具体步骤。

1.  **分组与角色分配**：两人组成一队。一人担任“陈述者”，另一人担任“干扰者”。
2.  **设定时间**：练习时长为30秒。
3.  **陈述者任务**：在30秒内，持续、大声地说出你今天学到的任何课程内容。**没有对错之分**，想到什么就说什么。
4.  **干扰者任务**：你的工作是制造紧张感，试图打断或影响陈述者。你可以说任何话、做任何动作（除了物理接触），例如质疑、嘲笑、提出无关问题等。
5.  **核心要求**：无论干扰者说什么或做什么，陈述者都**不能回应**。必须忽略所有干扰，仿佛干扰者不存在，只专注于复述“我学到了……”这类内容。

### 练习示范与要点解析

上一节我们介绍了练习的步骤，本节中我们来看看练习中的关键行为和心理要点。

在示范中，干扰者可能会说：“你手上动作很奇怪”、“这课程真笨”、“你害怕吗？”等等。而陈述者应始终如一台复读机，只重复类似“我学到了这个”、“我学到了那个”的话。

这个练习的核心在于训练你的**自我意识**和**注意力控制**。当干扰者让你发笑或感到震惊时，你的注意力会瞬间被吸引过去。练习的目的就是学会将注意力从这些“震惊”或干扰中剥离，重新聚焦于自己的任务。

**关键技巧**是设定并保持自己的节奏。例如，你可以规律地说：“分点一，……；分点二，……；分点三，……”，以此建立稳定的输出节奏，帮助自己屏蔽外界干扰。

### 练习目的与实际应用

这个练习看似简单，实则模拟了高压社交环境。在实际生活中，例如在酒吧、餐厅或当众演讲时，环境可能很嘈杂，节奏很快，他人会给你带来压力。

通过这个练习，你训练的是**保持自己的节奏和立场**，不被外界环境的快节奏或他人的反应带偏。无论周围发生什么，你都能像扎根在地面一样稳定，专注于自己的表达和目标。这不是为了去干扰或对抗他人，而是为了**强化你自身的稳定性和专注力**。

### 总结与开始练习

本节课中，我们一起学习了“保持理智”这个练习。它通过一个高干扰的对话场景，训练你在压力下保持专注、控制自我意识和维持个人节奏的能力。
