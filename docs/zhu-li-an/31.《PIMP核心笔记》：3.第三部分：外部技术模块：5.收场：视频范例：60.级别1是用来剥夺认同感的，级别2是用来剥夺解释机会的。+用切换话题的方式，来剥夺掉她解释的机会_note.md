# 课程31：《PIMP 核心笔记》第三部分：外部技术模块5：收场 - 视频范例 🎬

在本节课中，我们将学习一个关于沟通中“挑战”技术的具体范例。我们将重点分析两个不同级别的挑战如何运作，并理解如何通过特定技巧来引导对话，从而在互动中创造心理张力。

上一节我们介绍了外部技术模块中的基本挑战概念，本节中我们来看看一个具体的视频范例，它展示了如何将挑战升级并剥夺对方的解释机会。

## 核心概念解析：两级挑战

这个范例的核心在于区分两种级别的挑战，并运用话题切换技术。

*   **级别1的挑战**：其核心目的是**剥夺对方的认同感**。你通过一个轻微的否定或质疑，让对方感到需要重新赢得你的认可。
*   **级别2的挑战**：在级别1的基础上叠加，其核心目的是**剥夺对方的解释机会**。你不给对方空间去辩解或证明自己，从而将那种“未被满足的认同需求”锁定在她心里。

## 技术应用：如何剥夺解释机会

以下是实现级别2挑战、剥夺对方解释机会的具体操作方法。关键在于快速转移对话焦点。

1.  **发起挑战**：首先，提出一个直接的、略带否定的观察或评价。例如，你说：“其实你的稳气还挺一般的。”
2.  **立即切换话题**：在对方还未来得及对上述挑战做出反应或解释时，立刻、无缝地切入一个完全无关的新话题。例如，紧接着问：“对了，你多高啊？”
3.  **聚焦新节奏**：将全部的沟通节奏和注意力，都牢牢锁定在新话题上。反复追问：“哎，你多高啊？你多高啊？” 引导对话进入你设定的新轨道。

## 心理效应与结果

通过上述操作，会产生特定的心理互动效果。

*   女孩会被你强行带到下一个话题中。
*   然而，她内心会保留着关于上一个挑战的、未被处理的情绪：即那个“小小的没有被满足的认同感”和“恢复你对她认同”的需求。
*   她会因此产生内部心理活动：“他竟然敢说我稳气烂，我还没机会和他解释证明自己呢，怎么办？天哪！” 这种焦灼和想要“弥补”或“澄清”的冲动，就构成了所谓的**追逐效应**。
