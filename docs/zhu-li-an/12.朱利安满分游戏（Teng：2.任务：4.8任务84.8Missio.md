# 课程12：朱利安满分游戏 - 任务8 🎯

在本节课中，我们将学习朱利安满分游戏的第八个任务。这个任务旨在通过一系列社交互动练习，帮助你提升自信和社交技巧。我们将重点关注每日外出、组合互动、带领女生以及克服着装不适感这四个核心部分。

---

## 第一部分：每日外出练习 🚶‍♂️

上一节我们介绍了任务的整体框架，本节中我们来看看第一个具体要求。

你需要在下个七天里，每天外出20至30分钟。这是最后一段集中练习时间，下周起你可以逐渐调整。由于已经坚持了七天，这是第八天，过程不应太辛苦。每天外出不应被视为一件大事，你应当感到轻松。

**核心行动公式**：`连续7天 × 每天外出20-30分钟`

---

## 第二部分：完成五个组合互动 🤝

完成了每日的基础练习后，本节我们将深入到一个关键挑战：组合互动。

在上次任务的视频中，我们提到了在合作时间内完成“大规模接近挑战”。在这个挑战中，你需要完成一系列组合互动。具体来说，你需要介绍两个互不相识的女生认识。

以下是执行步骤：

1.  接近第一个女生并开始交谈。
2.  在交流中，不要只是聊完就结束。像人猿泰山一样，从一个互动摆荡到另一个。
3.  对她说：“来见我的朋友”，然后带她去见你刚才交谈的第二个女生。
4.  介绍她们认识。例如：“这是Suzi”。
5.  目标是在一次外出中，不能只和同一个女生互动，而要完成**五个**这样的介绍组合。

如果中间有组合互动不成功，就重新开始计数，但请努力尝试完成五个。这能锻炼你作为社交连接者的能力。

**代码描述核心流程**：
```python
while 已完成的组合数 < 5:
    女生A = 开始交谈()
    女生B = 开始交谈()
    介绍(女生A, 女生B) # 执行“来见我的朋友”
    已完成的组合数 += 1
```

## 第三部分：带领三个女生转场 🎪

掌握了组合互动后，我们需要进一步提升带领能力。

许多人会对“带领女生”感到困扰。因此，除了组合练习，本周你还需要练习带领三个女生转场，例如从当前位置带去酒吧或其他娱乐场所。

这并不意味着你必须带她们去玩乐。关键在于你作为发起者和带领者。你可以使用各种说辞，例如：
*   “我朋友在前面，去见我朋友。”
*   “我们去找点新朋友。”
*   “我们往前走走看。”

这是一个挑战。无论如何，请尝试带领三个女生进行场景转换。当你做这件事时，记得保持玩乐心态。如果女生坐着，你可以说“站起来”，并伸出手引导。你会惊讶地发现，只要你笑着带领，成功的机会很大。

**核心行动清单**：
*   目标：带领3个女生转场（如去酒吧）。
*   关键：主动伸手引导，并说“我们走”。
*   心态：保持微笑与玩乐感。

---

## 第四部分：穿着挑战 👔

最后，我们将通过一个特别的练习来突破心理舒适区。

这是关乎个人安全感的一项任务。你需要选择你最尴尬、最不想穿的衣服，或者一种让你感觉不安全的穿着方式。

例如：
*   你觉得站起来别人会嘲笑你的衣服。
*   你穿着感觉不舒服。
*   不要做头发造型（如果你平时用发胶或发蜡，这次不要用）。

穿上这些让你觉得不安全或尴尬的服装，然后外出活动。你会发现，这件事本身并没有那么重要。这个小挑战能帮助你解决对“尴尬着装”的恐惧，之后你会感到更自在。

**总结**：
1.  连续七天，每天外出20-30分钟。
2.  完成五个介绍组合（混合套件）。
3.  带领三个女生转场。
4.  选择一天，穿上让你感到尴尬的衣服，并外出以克服这种不适感。
