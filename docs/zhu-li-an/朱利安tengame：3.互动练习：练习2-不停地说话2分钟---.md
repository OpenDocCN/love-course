# 朱利安tengame课程3：互动练习：练习2-不停地说话2分钟 🗣️⏱️

在本节课中，我们将学习一个名为“两分钟聊天”的互动练习。这个练习旨在训练我们思维的流畅性和表达的即时性，通过持续不断地说话来克服内心的犹豫和评判。

## 练习规则与目标

上一节我们介绍了课程的基本框架，本节中我们来看看“两分钟聊天”的具体规则。这个练习的核心目标是：**在两分钟内，不间断地、快速地说话**。

以下是练习的具体规则：
*   两人一组进行。
*   一人将手轻拍在另一人的背上，示意开始。
*   在接下来的两分钟内，说话者必须持续不断地表达，不能有任何一秒的停顿。
*   说话内容不限，可以讲故事、提问、描述任何事物。

## 练习过程示范

理解了规则后，我们通过一个示范来具体看看如何操作。想象我现在正在轻拍你的背，练习开始。

“我现在在讲你的背上，我在做这个例子。我想去夜晚，去夜晚的，去夜晚的，去夜晚的…（持续不断地变换话题和内容）”

就像这样，在两分钟内，完全没有休息。倾听的伙伴需要确保说话者没有停顿。如果出现停顿，则需要重新开始。决定谁先开始后，就可以准备练习了。

## 练习核心要点与心态

完成了练习示范，接下来我们探讨完成这个练习所需的关键心态。这个练习的难点在于，两分钟不间断说话，感觉会非常漫长。

唯一能让你持续下去的方法就是：**关闭你的思维评判**。如果你心中有任何关于自我形象或说话质量的顾虑，你将无法快速流畅地表达。

你可以慢慢地组织语言，但若想达到快速、不间断的目标，你就根本不能去考虑或分析你正在说的内容。你必须让话语快速流动起来。因为如果你试图理解自己说的每一句话，并思考“这合理吗？”或“这说得够好吗？”，你将永远无法达到快速流动的状态。

这个练习的秘诀可以用一个代码块来描述其核心思想：
```plaintext
while (time < 2分钟) {
    说话();
    // 禁止执行以下判断
    // if (内容是否合理?) { 停止; }
    // if (内容是否够好?) { 停止; }
}
```

## 进阶练习与日常应用

掌握了基础练习后，我们可以尝试更具挑战性的进阶训练。这个练习可以每天进行。我建议你每天醒来后，就对着镜子练习。

开始时，可以不必一直盯着镜子。当你能轻松完成两分钟后，就尝试延长到五分钟、十分钟，甚至十五分钟。只要你坚持每天不间断地练习15分钟，你的表达流畅度将会大幅提升。结果会显而易见。

现在，让我们交换位置，再尝试一次两分钟的快速表达。就像说唱歌手一样，让话语自然流淌。3、2、1，开始！
