# 朱利安游戏法则课程 1：法则 2 - 提供价值 💎

在本节课中，我们将学习游戏十大法则中的第二条：**提供价值**。这条法则的核心在于，将你的注意力从“我能得到什么”转向“我能给予什么”。掌握这一心态转变，能让你在人际互动中更加自信、顺畅，并自然建立吸引力。

上一节我们介绍了游戏法则的框架，本节中我们来看看第二条核心法则的具体内容。

专注于“提供价值”是最容易取得结果、并能主动持续去做的行为。这并非制造隔阂，而是关于给予。

许多人陷入“索取”模式，甚至在心理上为此洋洋自得。你渴望某种特定结果，排斥另一种。每次你感到紧张、害怕或遇到阻碍时，通常是因为你只想着“拿到”某种东西。

因此，你可以时常问自己：如果你感觉不好，你能“拿到”什么？可能只是一种脆弱的自我形象或强调的自我。你能“拿到”什么？不如思考你想“给予”什么？你能“给予”什么？

想象一个场景：你向某人索要100元。这个过程充满紧张和对抗。但如果你换一种方式，顺畅、轻松地向对方提出，你便走出了自己的固有模式。这涉及到竞争与合作的感觉。每次你试图“索取”，你都会感到压力。

你还好吗？站在前面，你感觉如何？让我们进入这个“给予”的练习。
