# 课程31：《PIMP 核心笔记》第二部分：内核模块1：意图清晰模块 - 视频示范1：意图不清晰——抱女孩 🎬

在本节课中，我们将通过一个具体的视频示范，来学习“意图不清晰”在互动中的表现。我们将分析一个“抱女孩”的场景，看看当意图被隐藏时，行为会呈现出怎样的状态。

## 概述

上一节我们介绍了“意图清晰”作为内核模块的重要性。本节中，我们来看看一个“意图不清晰”的反面示范。通过剖析这个具体案例，我们可以更直观地理解清晰表达意图与隐藏意图所带来的行为差异。

## 视频示范分析

在示范视频中，当行为者意图不清晰时，其“抱女孩”的动作呈现出以下特点：

以下是动作的具体表现描述：

*   她会慢慢地接近。
*   她的动作显得犹豫和不确定。
*   她会通过语言进行试探，例如询问：“你喜欢这样吗？”或“你喜欢吗？”

这些行为共同构成了一个意图模糊、缺乏自信和主导性的互动模式。

## 核心概念总结

本节课中我们一起学习了“意图不清晰”的一个行为范本。其核心模式可以概括为：**行动迟缓 + 态度犹豫 + 语言试探**。这种模式与清晰、直接、坚定的“意图清晰”模块形成了鲜明对比。理解这种差异是调整自身行为、向“意图清晰”迈进的第一步。
