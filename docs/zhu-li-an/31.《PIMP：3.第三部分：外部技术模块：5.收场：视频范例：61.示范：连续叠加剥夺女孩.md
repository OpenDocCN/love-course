# 31.《PIMP：3.第三部分：外部技术模块：5.收场：视频范例：61.示范：连续叠加剥夺女孩

在本节课中，我们将学习一种名为“连续叠加剥夺认同感”的互动技术。该技术的核心在于，通过有策略地切断话题并叠加挑战，逐步剥夺对方从你这里获得认同感的机会，从而引导对方反过来追逐你的认同。

---

上一节我们介绍了快速切换话题的方法，本节中我们来看看一种更强烈的互动方式：直接切断当前话题。

这种方法的核心是剥夺对方重新从你身上获得认同感的机会。具体操作方式是，在对话中加入一个“推”（即否定或挑战性的陈述），从而叠加挑战，持续剥夺对方的认同感。

以下是该技术的一个具体应用范例：

1.  首先，你可以提出一个否定性的陈述。例如，你说：“其实你吻技还挺糟糕的。”
2.  接着，你可以追问对方的感受，施加压力。例如，你说：“我说你吻技烂，你心里什么感觉？”
3.  此时，对方可能会急切地想要证明自己。例如，女孩可能回复：“你还是第一个说我吻技不好的。以前和我在一起的男生都很喜欢和我亲亲，我真的就……”
4.  面对对方的辩解，你的状态应保持“怀疑”。
5.  对方可能会继续辩解，例如说：“我没骗你，……”
6.  此时，你可以叠加新的挑战，继续剥夺认同感。例如，你可以说：“其实我挺喜欢你这样的女孩，但是你真的没那么好，而且你年龄也不小了。”
7.  通过这样“叠加、叠加、叠加”地剥夺认同感，对方便会开始“追逐、追逐、追逐”你的认同感。

---

本节课中，我们一起学习了“连续叠加剥夺认同感”的技术。该技术通过 **切断话题 -> 加入挑战（推）-> 保持怀疑态度 -> 叠加新的否定** 这一系列操作，逐步剥夺对方的认同感，最终引导对方转而寻求并追逐你的认同。关键在于有控制地施加压力并保持框架，而非进行人身攻击。
