# 朱利安搭讪游戏课程：4. 任务二 🎯

在本节课中，我们将学习任务二的具体内容。任务二建立在任务一的基础上，要求你继续坚持外出互动，并引入新的挑战，以帮助你提升自信、适应不同场景并掌握互动中的关键信息获取技巧。

---

上一节我们介绍了任务一的核心要求，即连续七天、每天至少外出30分钟进行互动。本节中我们来看看任务二的具体内容与新增挑战。

任务二要求你继续坚持外出七天，每天至少30分钟。这意味着从任务一开始，你将累计完成**连续14天**的每日外出挑战。

以下是任务二的具体要求列表：

*   **坚持每日外出**：继续执行**连续7天**，每天**至少30分钟**的外出互动。
*   **混合日间与夜间搭讪**：本周你必须同时进行日间搭讪和夜间搭讪。如果你习惯夜间活动，本周需加入日间搭讪；如果你习惯日间活动，则需加入夜间搭讪。目标是让你在任何时间、任何环境下都能自如地开启对话。
*   **禁止自我否定**：在接下来的七天里，完全停止否定自己。无论互动结果如何——被拒绝、未达到预期行动量——都不允许在内心或报告中批评自己。你必须专注于寻找积极面，认可自己的每一次尝试。
*   **获取关键信息**：每天，你需要从至少一次互动中，获取对方对以下五个后勤问题的答案：
    1.  你和谁一起来的？
    2.  你住在哪个区域？
    3.  你怎么回家？
    4.  你有室友吗？
    5.  你今晚/明天有什么安排？
*   **使用特定开场白**：本周的三次开场必须使用“**Daddy‘s home**”（爸爸回家了）这句话。这个开场白要求你充满自信地表达意图，同时能够自我解嘲，避免显得怪异。

---

接下来，我们将对上述要求中的关键点进行详细说明。

**关于禁止自我否定**：这意味着你需要转变思维模式。例如，当你被拒绝时，不应想“我真失败”，而应想“我有勇气上前搭讪，这本身就值得称赞”。在你的每日报告里，只允许出现积极、自我鼓励的内容。

**关于获取后勤信息**：这需要你在互动中平衡情感连接与逻辑信息收集。直接按顺序提问会像审问，建议你回看课程第九条法则（玩转悖论）来学习如何自然地融入这些问题。

**关于特定开场白“Daddy‘s home”**：使用这个开场白时，关键在于**自信地表达**并**保持轻松幽默**。你不能怯懦或过于严肃。说完后，需自然地继续对话，而不是等待对方反应或寻求认可。公式如下：

```
走向目标 -> 自信地说出“Daddy‘s home” -> 立即衔接后续对话
```

---

本节课中我们一起学习了任务二的全部内容。核心是延续每日外出的习惯，并新增了混合场景搭讪、培养积极自我评价、学习获取互动中的关键后勤信息以及使用一个高展示度的特定开场白等挑战。请专注于你的每一次进步，在报告中只分享你的成功与收获。祝你好运！
