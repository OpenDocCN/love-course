# 课程31：《PIMP 核心笔记》第三部分：外部技术模块 - 收场：视频范例 - 方法三：强迫投资 💪

在本节课中，我们将学习收场环节的第三个方法：“强迫投资”。这是一种通过带有轻微命令感的互动，引导对方进行语言或行为上的小额投入，从而建立吸引力的技巧。

---

上一节我们介绍了其他收场方法，本节中我们来看看如何通过“强迫”来获取投资。

“强迫投资”的核心在于，当你在互动中向对方提出一个要求（例如“打电话给我”），而对方反问“为什么”时，你不直接解释，而是以略带命令的口吻要求对方执行。例如，你可以说：“说出来。” 或 “打电话给我。”

这句话本身可能没有实际意义，但关键在于对方因为你的要求而说出了“打电话给我”这句话。这本身就是一种微小的服从性投资。有时，你甚至可以重复这个过程来强化效果，例如在她说完后，再次命令：“再说一遍，打电话给我。”

---

以下是实施“强迫投资”的一个常用话术范例：“注意言辞”。

例如，当你问女孩“你想我没？”，她回答“没有”时，你可以使用这个方法。你可以带着命令感和自我娱乐的态度对她说：
```
注意言辞，再说一遍，你想我没？
```
这种方式既传达了你的主导框架，又以一种轻松、戏谑的方式进行了互动，避免了真正的冲突或尴尬。

---

本节课中我们一起学习了“强迫投资”这一收场技巧。它的要点是：在互动中通过带有轻微命令感的要求，引导对方进行语言上的小额投资，同时结合“注意言辞”这类话术，以自信、戏谑的态度来推进关系。记住，核心在于创造服从性和有趣的张力，而非真正的强迫。
