# 31.《PIMP：3.第三部分：外部技术模块：3.上钩：上钩模块-视频示：12.你可以用热

在本节课中，我们将学习如何通过传递自身的热情与兴奋感，将互动对象自然地“吸入”你所营造的氛围框架中。这是一种强大的情绪引导技巧。

上一节我们介绍了框架建立的基础，本节中我们来看看如何通过情绪感染来强化它。

核心操作是：**主动分享你的热情与兴奋，并引导对方关注同一焦点**。这能瞬间转移对方的注意力，并将其情绪状态与你同步。

以下是一个具体的应用实例：

*   “你看这个景色，天呐，简直太棒了！”
*   对方的典型反应会是：“啊？什么？”（注意力被成功吸引）
*   此时，对方便瞬间被**吸入**了你的情绪框架，即你的热情与兴奋感所营造的氛围中。

关键在于，你的表达必须是真诚且自发的，而非机械背诵。你分享的是真实的感受，对方回应的则是你对当下时刻的投入状态。

本节课中我们一起学习了“情绪吸入”技巧。通过主动、真诚地表达你的热情与兴奋，你可以有效地将对方的注意力与情绪引导至你的框架内，从而创造一种强烈且共享的互动氛围。
