# 12.朱利安满分游戏（Tengame皮卡团队内部提供）：3.练习：练习6-从内部绘制状态_note

在本节课中，我们将学习一个名为“从内部绘制状态”的表演练习。这个练习旨在训练演员专注于内心的画面和情感反应，并通过特定的规则来强化即兴互动中的专注力与情绪控制。

## 概述 📖

这个练习的核心是让两位演员在严格的规则下进行问答互动。一位演员扮演提问者，必须始终保持积极快乐的情绪；另一位演员扮演回答者，但只给出负面或拒绝性的答案。练习的目标是让提问者学会在持续面对负面反馈时，依然从内心调动并维持指定的积极状态。

## 练习规则与流程 🔄

以下是本练习的具体规则和步骤说明。

### 角色与基本设定

练习需要两人一组，分别扮演“男生”和“女生”角色。我们将进行一次角色互换，所以每位参与者都会体验两个角色。
*   **第一轮**：你是“女生”，我是“男生”。
*   **第二轮**：角色互换，你是“男生”，我是“女生”。

在每一轮中，“男生”的角色是**提问者**，“女生”的角色是**回答者**。每一轮的互动时间为30秒。

### 提问者（“男生”）的行为准则

作为提问者，你必须严格遵守以下三条规则，这些规则是练习的核心：

1.  **只能提问**：你所说的每一句话都必须是疑问句。你不能进行任何陈述、评价或命令。
2.  **保持快乐**：每次提出问题后，你必须**大笑**，表现出非常开心、积极的状态。
3.  **等待与更新**：每次提问后，等待对方回答。每一个问题都必须是**全新的**，不能基于上一个答案进行追问。

**公式：提问者行为 = 提出新问题 + 大笑 + 等待回答**

### 回答者（“女生”）的行为准则

作为回答者，你的规则相对简单，但至关重要：

*   你对所有问题的回答都必须是**负面的、拒绝的或对抗性的**。例如，你可以说“不”、“滚出去”、“我不懂”等。
*   你的回答没有其他限制，目的是持续给提问者制造情绪上的挑战。

## 练习目的与核心挑战 💡

上一节我们明确了练习的规则，本节中我们来看看这个练习的设计目的和演员面临的核心挑战。

这个练习的难点在于制造一种情绪冲突：提问者在外在行为上被要求保持极度快乐，但接收到的所有言语反馈都是负面的。这要求演员：

*   **从内部绘制状态**：你的快乐不能依赖于外部（对方的回答），而必须完全从内心想象和调动出来。
*   **隔离情绪**：练习“你的反应”和“你的表演”是两回事。内心可能感到挫败，但外在表现必须符合角色要求。

如果提问者因为对方的负面回答而动摇，失去了笑容或快乐的状态，那么回答者（“女生”）有权象征性地“打”提问者（“男生”）一下，以示其“从内部绘制的状态”崩溃了。

## 示例对话 🗣️

为了更好地理解互动过程，以下是一段模拟的对话示例。请注意提问者如何遵守规则，以及回答者如何持续给出负面反馈。

```
提问者（男生，大笑）：“你叫什么名字？”
回答者（女生）：“这不关你的事。”

提问者（男生，大笑）：“你从哪里来？”
回答者（女生）：“我从哪里来和你没关系。”

提问者（男生，大笑）：“你喜欢看电影吗？”
回答者（女生）：“不，我很讨厌。”

提问者（男生，大笑）：“你今天开心吗？”
回答者（女生）：“快滚出去。”
```

（对话可以如此持续30秒，提问者不断更换问题并保持大笑。）

## 关键提示与总结 🎯

本节课中我们一起学习了“从内部绘制状态”这个高难度练习。它的精髓在于：

*   **核心训练点**：它强迫演员**不依赖外部刺激**，而是**从自身内部创造并维持一个特定的情感状态**（这里是“快乐”）。这是一种强大的心理技巧。
*   **关于反应**：在生活中，我们的反应常由外界触发。但在表演中，我们需要控制反应。这个练习中，**“正确的反应”是保持快乐**，无论对方说什么。如果你因对方的话而生气或沮丧（并表现出来），那就是“错误的反应”，意味着你被外部带走了。
*   **实验性质**：这是一个“实验”。目的是让参与者习惯在压力下，依然能让情感“从内心流露出来”。当你扮演“女生”时，你可以尽情做“世界上最坏的女生”，全力挑战对方；当你扮演“男生”时，你需要调动全部内在力量来守住你的“快乐状态”。

通过这个练习，你将更深刻地理解如何掌控并运用自身的情感，为角色塑造打下坚实基础。
