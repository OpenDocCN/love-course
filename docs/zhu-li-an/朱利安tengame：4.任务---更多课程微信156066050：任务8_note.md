## 任务二：合并社交组合 🔗

在上一节的任务视频中，我提到了这个概念。本周我们将重点练习合并社交组合。

合并社交组合本质上是将两个互不相识的群体介绍给对方。例如，介绍两个女孩认识。我经常这样做，尤其是在夜晚开始时，用以让自己进入高度社交的状态。

以下是本周的基本练习步骤：

1.  接近第一个女孩。例如：“嗨，你好吗？”
2.  不要只停留在第一个女孩这里并结束互动。相反，要像人猿泰山一样，从一根树枝荡到另一根树枝，从一个女孩到另一个女孩。
3.  与第一个女孩交谈后，对她说：“来见见我的朋友。”然后带着她去接近下一个组合。
4.  向第二个组合介绍她。当你进入下一个组合后，你可以选择留在她身边，完全专注于新组合，或者回到她那里。

**任务核心规则**：你不能在没有带着上一个女孩的情况下去接近下一个女孩。尝试至少连续合并5个组合。

**流程示例**：
```
向第一个女孩说：“嗨，你好吗？”
“来见见我的朋友。”
接近下一个组合：“嗨，这是某某某。”
然后带着那个组合中的女孩，去往下一个组合：“来见见我的朋友。”
```
依此类推，完成5个组合。如果你进行到3个组合时中断了，身边没有女孩了，请从头开始，再次尝试完成5个“分支”。

在合并组合时，你可以让它变得有趣。例如，你可以说：“来见见我的女朋友，这是我女朋友。”只要它是乐趣的一部分，怎么说都行。

务必保持主导。我常看到你们在合并组合时犯的一个大错误：你们会说“来见见我的朋友”，然后介绍另一个女孩，“见见我的朋友”。对方通常会露出“发生了什么？我们认识吗？”的表情。如果你只是说“你们认识一下”，然后就闭嘴不说话，互动就会冷场。

所以要面带微笑地主导，告诉她们该做什么。当我合并组合时，我几乎像在训练营里一样指挥：“嘿，眼神交流，握手，眼神交流。”我会做这类事情，并且让它变得夸张。我甚至会做一些小角色扮演，比如：“嘿，你知道这是苏西吗？我跟你提过的，她是那个服务员……”随便说些无厘头的话让它有趣。

同时也要明白，即使组合没有真正“粘合”在一起也没关系。你不必期望她们接下来会聊上30分钟。有时她们只是打个招呼就冷场了，然后你必须选择是留在第一个女孩那里，还是跳到下一个。即使冷场了，你也可以在两个组合之间来回穿梭。

也不要过度推销。只需想着“面带微笑地主导”，并且这不是什么大事。如果你表现得像“嘿，来见见我的朋友”是件天大的事，她们会想：“搞什么鬼？有什么企图？”你不会想做得太过。你要意识到这根本没什么大不了的。“嘿，来见见我的朋友，过来这边就行。”

即使她不愿意去见朋友，你也可以做的是：直接走到下一个组合，轻轻拍她说“嘿，来见见我的朋友”，然后把她介绍给第一个女孩。

但本周我希望你开始扮演介绍者的角色，并且至少连续完成5次介绍。

**核心要求**：`本周至少完成5次连续的组合合并。如果中断，请从头开始。`

## 任务三：移动女孩 🚪

这个任务与合并组合相辅相成。很多人在移动女孩方面有困难，所以我希望你在合并组合时练习这一点，但同时我也要你专门练习将女孩移出酒吧或俱乐部。

本周你至少需要有一个晚上外出，基本任务就是带女孩走出俱乐部。

你的任务是：带3个女孩走出俱乐部或酒吧。这并不意味着你必须把她们“带走”或发生什么，我只是想让你积累一些移动她们离开场所的参考经验。

你可以说任何你想说的话。例如：“嘿，我朋友在前面，来见见我朋友。”或者说：“我们出去透透气吧。”你甚至可以直接说：“嘿，你能陪我走到门口吗？然后我们可以再走回来。”这只是一个小挑战。

无论用什么方法，尝试带3个女孩走出酒吧或俱乐部。在做这件事时，请记住“小步推进”原则，并注意你要求的顺从程度。

如果一个女孩正坐着，你直接说“走出俱乐部”，你要求的是很多。但如果你说：“嘿，握个手好吗？站起来一下。我能看看这个吗？嘿，往这边走一步。”小步推进，你成功的几率会高得多。

你会惊讶地发现，只是先和女孩握手，然后说“过来一下，过来一下”，如果你只是这样做，几乎就能把她带到门口了。只要面带微笑。

**核心要求**：`将3个女孩带出酒吧、俱乐部或场所。`

## 任务四：克服表面不安全感 👕

这是最后一个任务。在其中的某一天，我希望你这样做：穿上你最令人尴尬的一套衣服。或者可能是你有一些旧衣服，你真的很不想穿。无论是什么让你穿上后感到不安全的打扮，或者任何让你觉得“哦，糟了，我会很显眼，人们会嘲笑我，我穿这个不舒服”的穿着。

完全不要做头发。如果你平时用发胶或发蜡，那天不许用。如果你平时不用，那天就必须用很多。基本上，我希望你触及所有那些表面的、外貌上的不安全感。

然后，就以这种打扮外出并进行搭讪。你会意识到，这根本无关紧要。这只是一个小挑战，用以克服那些愚蠢的小不安全感，过一段时间你就会明白：“哦，这从一开始就根本不是大事。”

**核心要求**：`选择一天，以让你感到尴尬或不安全的穿着打扮外出练习。`
