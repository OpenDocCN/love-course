# 31.《PIMP：3.第三部分：外部技术模块：5.收场：视频范例：29.示范：把大步骤切分成一

在本节课中，我们将学习一个关键的社交策略：如何通过将一个大目标分解为一系列小步骤，来有效地引导互动走向最终目的。这个方法的核心在于降低每一步的决策压力，逐步推进。

如果你刚到夜店、酒吧或聚会，就直接对一个女孩说之后去你家开个小派对，那么女孩接受的可能性很低。反之，在夜店或酒吧活动进行到后半段时，这种可能性会极大提升。

你可能会遭遇新的拒绝。那么，你要解决的将是女孩提出的**新的拒绝借口**。同时你要明白：女孩跟你出了夜店，意味着她对你又进行了一点点“投资”，并且她离你的最终目的地更近了一步。

接下来你该怎么做呢？答案当然是：朝着你家的方向，**更近一步，更近一步，再更近一步**，最终抵达你的性爱地点。
