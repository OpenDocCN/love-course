# 31.《PIMP：3.第三部分：外部技术模块：3.上钩：上钩模块-视频示：7.正确的提供更

## 概述
在本节课中，我们将学习“上钩”模块中的一个关键环节：如何通过正确的对话示范，自然地引导互动并获取更多信息（即“标本”）。我们将通过分析一段具体的视频示范，拆解其核心步骤和话术技巧。

---

上一节我们介绍了“上钩”模块的基本概念，本节中我们来看看一个具体的视频示范，学习如何在实际互动中正确操作。

这段示范的核心在于，通过主导对话节奏、切换话题与情绪，并在推拉互动中，自然地获取对方的信息。示范者并非直接索要信息，而是将问题融入轻松、多变的聊天过程中。

以下是这段互动中可拆解的关键步骤列表：

1.  **开启多样话题**：主动聊各种各样的话题，避免对话陷入单调。
2.  **切换情绪状态**：在不同的情绪中不断切换，使互动显得生动、不可预测。
3.  **运用推拉技巧**：交替使用表达兴趣（拉）和制造轻微挑战（推）的语句。例如：“诶等等你多大了，18岁，你对我来说太老了。”
4.  **设立沟通节奏**：通过提问、调侃、陈述，不断主导对话的走向与速度。
5.  **提供合理理由**：为搭讪或提问提供一个轻松、自然的理由。例如：“没有没有我就是觉得你挺萌的，所以说想过来打个招呼。”
6.  **植入未来情景**：在对话中自然提及后续可能的事件或地点，测试对方反应。例如：“啊我要去太原街，你要往那边走吗？”
7.  **确认背景信息**：在互动中穿插对对方背景的确认，这既是话题也是信息收集。例如：“诶你是沈阳姑娘吗？”
8.  **主动结束互动**：在气氛合适时，主动、明确地结束当前回合的对话，为后续联系留有余地。例如：“我们下集再见咯。”

通过以上步骤，示范者完成了一次包含信息收集、情绪调动和关系铺垫的完整互动。整个过程行云流水，没有直接和生硬的索取感。

---

## 总结
本节课中，我们一起学习了“正确提供更多标本”的对话示范。关键在于**主导节奏**、**融合推拉**、**切换话题与情绪**，并将信息收集巧妙地隐藏在自然的社交互动中，而非直接提问。记住，目标是让对话变得有趣且由你引导，从而让对方在舒适的节奏下愿意透露更多信息。
