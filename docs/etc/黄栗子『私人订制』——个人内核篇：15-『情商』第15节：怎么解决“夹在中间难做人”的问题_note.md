# 黄栗子『私人订制』——个人内核篇：15-『情商』第15节：怎么解决“夹在中间难做人”的问题_note

在本节课中，我们将要学习一个常见的人际沟通难题：如何在传达坏消息时，避免自己“夹在中间难做人”。我们将分析常见的沟通误区，并掌握一套行之有效的沟通策略，帮助你在履行义务的同时，保护自己不被负面情绪波及。

## 概述：为何会“夹在中间难做人”？

在生活中，每个人都有自己的角色定位。作为医生，需要向病患家属传达噩耗；作为人事主管，需要向员工传达辞退通知；作为银行职员，需要告知客户贷款未获批的消息。

在传达这类坏消息时，我们很容易陷入“夹在中间难做人”的困境。虽然错不在我们身上，但传达消息的人却总容易遭受池鱼之殃。这是由于当人听到坏消息时，情绪往往变得伤心或愤怒，而传达消息者近在眼前，自然就成了最直接的情绪发泄对象。

那么，如何避免在人际交往时陷入这样的困境呢？首先，我们要学会识别并避开几个关键的沟通误区。

## 需要避开的三个沟通误区

以下是三个在传达坏消息时常见的错误做法，理解它们能帮助我们防患于未然。

**1. 误区一：表现出过度的“感同身受”**

例如，在医院里，如果医生向病患家属报告不理想的检查结果时，脸上流露出“这种结果我也接受不了”的表情，就很容易出问题。原本医生只是一个传达消息的中间人，不必承担任何责任。但如果表现得过于同情，就容易给对方一个迁怒于你的情绪出口。对方可能会想：“你既然也觉得难以接受，那你为什么要让这种事发生？”如此一来，明明不关你的事，你却可能因此受到牵连。

**2. 误区二：混淆“善意”与“愧疚”**

在传达坏消息时，为了表示自己没有恶意，很多人开口第一句话就是道歉：“对不起，……”。但事实是，你的道歉基本不可能被对方体谅。从心理学角度看，人在遇到坏事时，总想找到责任人。这时候，谁离得越近谁就越危险。因此，在表达善意时，如果混杂着愧疚的情绪，就很容易被对方反将一军。对方会觉得：“如果你没有错，你在道什么歉呢？”即使事情本身与你无关，也可能被对方当成是你的错。

**3. 误区三：试图替对方解决问题或过度安慰**

有些人在传达坏消息后，为了安抚对方，会试图替对方出谋划策，例如说“没事的，想开点，说不定反倒是好事呢？”这看似是在安慰，实则是把自己带进了坑里。一旦你开始安慰对方，你的角色就从“不承担责任的信息传达者”转变成了“要承担责任的行动方”。如果对方听了你的安慰还是难受，回头继续找你，你是否要接着安慰？如果你给对方想了办法，他回头再来详细咨询，你是否还要帮他善后？最后，无论结果如何，你都可能被卷入其中，劳心费神。

在传达消息时，本来没有你的事，可是一旦触及这些沟通误区，就容易多出一部分需要你负责的事，处理不好，反倒会引火烧身。

## 核心策略：保持中立，给予恰当陪伴

那么，当我们“夹在中间难做人”时，到底该如何沟通，才能既尽到义务，又不会引火烧身呢？最行之有效的方法就是：**保持中立的态度，给予恰当的陪伴**。

在传达坏消息时，我们应该展现出中立的姿态，既不过分亲热，又不过分冷漠。这样做才能既准确无误地传达消息，又不会进一步刺激对方的情绪。

上一节我们介绍了需要避开的误区，本节中我们来看看具体该如何操作。

### 1. 展现专业姿态，避免过分亲热

在“不过分亲热”这方面，没有什么比专业人士的中立姿态更稳当。因此，在医患沟通准则里，总会提醒医生注意塑造专业形象，并尽量在专业环境中与患者沟通。对其他专业人士来说，有制服的需穿戴整齐，没有制服的，在穿着、表情管理上也要符合专业规范。

具体做法可以概括为以下代码所描述的“专业姿态公式”：

```
专业姿态 = 清晰有力的言语 + 直视有神的眼神 + 规范得体的外表
```

说话要清晰有力，不要支支吾吾；眼神要直视有神，不要躲躲闪闪。这些细节都是在暗示对方：“我来告诉你这些消息，是因为我的职责所在，但事情并非由我造成。”无论是医生、人事主管还是银行职员，都需要经常练习这种专业性。做到既不拖泥带水，又没有个人情绪掺杂其中，才是最恰当的尺度。

### 2. 表达“遗憾”而非“道歉”，区分善意与责任

其次，在与人交往时，人人都有同理心。所以在面对坏消息时，会产生不忍心的情绪，由此容易生出“我应该帮帮你”的心态。但一定要注意一个关键细节：我们应该表达的是“愿意帮助的善意”，而非“责任在我的愧疚”。

请大家体会以下两句话的区别：
*   **“对不起，这件事没办好。”**
*   **“很遗憾，这件事没成功。”**

“遗憾”是因为对方会伤心难过，所以我们出于人道主义也觉得不开心。“对不起”则意味着这件事没办成，我们也有责任。说出“对不起，这件事没办好”，对方很容易抓住话柄反问你：“你现在说对不起有什么用？你要怎么弥补？”而说出“很遗憾，这件事没成功”，对方则更可能顺着语境问：“那我该怎么办？”两句话带来的效果天差地别。

因此要记住：**不管你有多么善解人意，都只是来传达消息的，而不是来帮忙解决问题的，更不是来道歉的。** 当面临“夹在中间难做人”的时候，注意这一点言语上的差别，很多不必要的麻烦就可以避免。

### 3. 给予陪伴时间，但避免主动安慰

前面我们强调，传达坏消息时不要过分亲热。那是否意味着传达完就要赶紧离开，防止被对方找麻烦呢？离开当然要离开，但离开得太快也容易出错。在沟通完后快速离开，代表着不尊重，对方也很可能对你产生不满情绪。

因此，传达坏消息后，最好能找个地方和对方坐下来，给对方传达一种态度：“发生这样的事，你肯定有很多疑问，我会陪着你，帮你解决疑惑。”但这里要注意一个关键点，那就是：**可以陪伴，但不要主动安慰**。

不管是说一些体贴的话，还是做出握住对方的手、拍拍对方后背等肢体动作，这些安慰行为都不可取。要时刻记住，传达坏消息的角色本身就不讨好，因此“不被迁怒”就已经是成功了，不要再奢求对方能对你感恩戴德。你做的任何额外行为都可能节外生枝。

## 总结

本节课中，我们一起学习了如何解决“夹在中间难做人”的沟通困境。

为人处事，难免会陷入需要传达坏消息的境地。这时候最好的应对方式就是**保持中立的立场**。要明白，事情变成这样并非由你造成，你只是一个信息传递者。就好比病人生病不是医生的责任，公司裁员不是人事主管的原因，银行贷不了款也不是银行职员的问题。

所以我们才要记住核心原则：**要专业而不要同情，要善意而不要愧疚，要陪伴而不是主动安慰。**

最后请注意，以上策略适用于“你只是信息传递者”的情况。**但如果事情本身就是因你而起，是你造成的错误，那就完全不同了。** 这个时候你要做的第一步是学会如何正确道歉，这在本专栏的第十三节课程中有详细讲解。
