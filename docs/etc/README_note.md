# 课程名称：Python基础入门 - 第01课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等）。标签就是变量的名字，我们通过这个名字来找到并使用盒子里的数据。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。

```python
my_name = "小明"
my_age = 25
```

这里，我们创建了两个“盒子”：一个叫 `my_name`，里面存放了文字“小明”；另一个叫 `my_age`，里面存放了数字25。

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有一些内置的基本数据类型，最常用的有三种。

以下是三种核心的数据类型介绍：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数、负数或零。
    *   代码示例：`count = 10`，`temperature = -5`

2.  **浮点数**
    *   表示带有小数点的数字，用于处理更精确的数值。
    *   代码示例：`price = 19.99`，`pi = 3.14159`

3.  **字符串**
    *   表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   代码示例：`greeting = “你好，世界！”`，`name = ‘Alice’`

## 如何查看和转换类型？ 🔄

我们知道了数据类型，那么如何查看一个变量到底是什么类型呢？又如何在不同的类型之间转换呢？

Python提供了 `type()` 函数来查看变量的数据类型。

```python
height = 180
print(type(height))  # 输出：<class 'int'>
```

有时，我们需要将一种数据类型转换为另一种。例如，将数字转换成文字以便拼接，或者将用户输入的文字转换成数字进行计算。

以下是常用的类型转换函数：

*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

```python
# 将字符串转换为整数进行计算
num_str = “100”
real_num = int(num_str) + 50  # real_num 现在是 150

# 将数字转换为字符串进行拼接
age = 30
message = “我今年” + str(age) + “岁。”  # message 现在是 “我今年30岁。”
```

## 总结 📝

本节课中我们一起学习了Python的变量与数据类型。

我们首先将**变量**理解为一个存放数据的命名容器。然后，我们认识了三种**基本数据类型**：用于整数的 `int`，用于小数的 `float` 和用于文本的 `str`。最后，我们掌握了使用 `type()` 函数检查类型，以及使用 `int()`, `float()`, `str()` 函数在类型间进行转换的方法。

记住：**变量是数据的盒子，数据类型决定了盒子里装的是什么以及能做什么**。这是你Python之旅的坚实第一步。
