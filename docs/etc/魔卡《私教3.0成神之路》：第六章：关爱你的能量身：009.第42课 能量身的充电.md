# 魔卡《私教3.0成神之路》：第六章：关爱你的能量身：009.第42课 能量身的充电-书写 📝

在本节课中，我们将要学习一种安静而有效的能量身充电方式：书写。我们将探讨如何通过不同的书写练习来滋养能量、提升振动频率，并改善与自我的关系。

上一节我们介绍了“进攻”对于能量积累和恢复的作用。本节中，我们来看看一种更为内敛的补充方式。

## 书写作为能量充电方式

书写是一种效果显著的能量层充电方式。课程开始时布置的四项常规作业包括健身、瑜伽、日记和冥想。大家应根据自己的时间安排选择一项或多项进行。

一方面，不要贪多求快，不顾实际时间安排，造成过大的心理负担。另一方面，也不要完全无视这四项常规作业。这四样作业中最简单容易上手的就是日记。

## 日记：连接自我与小我

有些同学可能觉得写日记是小学生的事情，认为自己不需要提高文学素养，也没有闲情逸致记录每一天。这些想法可以理解。

作为老师和引路人，只能尽职尽责地告知做这些事情的好处。至于做不做、什么时候做、做到什么程度，完全看大家自己的选择。

日记这件事最重要的不是写多长、多细或多优美。最重要的是连续地写，每天都写。每天5分钟都是可以的，想写什么就写什么，不需要考虑主题、文体或辞藻。

因为我们的目的是滋养能量，拉近自我与小我的距离，改善自己与自己的关系。

当你在写日记时，你无法确定那些流淌出来的文字，到底是你的自我在写，还是你的小我在写。很多时候，几天后重新翻看之前的日记，你可能会很惊讶当时写下的内容，甚至感到莫名其妙。

如果有这种感觉，就可以明确确定你现在的自我理性意识，发现了之前写出这部分日记的是你的小我。读取小我的表达是觉察自我关系、拉近自我关系的一个非常重要的方法。

前面的课程中提到，小我会用情绪和痛苦作为语言来提示自我理性的人生主线任务。现在，当你学到这一课并开始养成写日记的习惯后，可以直接从日记中读出小我的表达。

所以，去尝试养成这个新习惯吧，你会从中收获很多。

## 抄经：连接高维能量

第二种方式是抄经，即抄写中华文化中的那些经典。这些经典都带着极高的能量，可以穿越历史和时间，保持旺盛的生命力，为无数人的生命和生活做加持。

以下是推荐的经典：
*   **心经**：共260个字，字字珠玑。去年抄写三十几遍后，便可自然默写。如果每天睡前抄一遍心经，持续一年，你的能量振动频率可以直接跃迁一个级别。
*   **金刚经**：5000多字。
*   **道德经**：5000多字。
*   **六祖坛经**。

心经抄完、领悟并得到实惠后，你会自然而然地想去抄或看金刚经。这是不用思考和努力就自然而然的事情。古语说得好：读杂书万卷，不如读经典一藏。

去年是把这两件事情一起做的：默写一遍心经，然后开始写日记。

## 书法：修身养性与提升价值

第三个方式是书法。书法的门槛会比前两者稍高一点，因为需要购买笔墨纸砚，甚至可能需要找老师报名学习。

但毫无疑问，它能够提供修身养性的作用。如果爱上了书法，它也会成为一项自身的价值，在工作、社交和恋爱中都会自然而然地为你提供高价值。

另外需要强调的是，如果养成了抄经和书法的习惯，假以时日，你整个人的气质会发生毫无疑问的改变。

如果问我为什么气质会变好，为什么气场会变得有吸引力，答案很简单：因为你的能量身振动频率提升了。

也许你依旧不理解、想不明白其中的科学逻辑。其实也可以放松下来去释怀这个疑惑。

因为能量身的运作逻辑本来就和精神身不一样，和物质身的逻辑就更不一样了。当我们用自己的、算力为**120比特/秒**的自我理性意识，去试图思考与解释这个世界的一切时，毫无疑问，你的算力是严重不足的。

就像我们肉眼的可见光，它们只是宇宙频谱中非常非常窄小的一段光谱。绝大多数的频率和能量形式我们根本看不见，但它们都存在在那里。

所以，你要试着松开你的自我理性设限，保持开放，去体验、去经历、去感受。这样，你的能量振动频率和生活才会越来越好。

至于做不做，什么时候做，做到什么程度，这其实是完全看大家自己的选择。

## 总结

本节课中，我们一起学习了通过书写为能量身充电的三种方式：**日记**、**抄经**和**书法**。这些练习的核心在于持续行动与开放体验，而非追求形式与结果，它们能有效帮助我们提升能量振动频率，改善与自我的关系，并潜移默化地改变个人气质与生活状态。
