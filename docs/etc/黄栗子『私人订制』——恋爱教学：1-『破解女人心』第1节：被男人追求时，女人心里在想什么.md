# 黄栗子『私人订制』——恋爱教学：1-『破解女人心』第1节：被男人追求时，女人心里在想什么

在本节课中，我们将要学习如何理解女性在感情初期的心理活动。通过分析女性在相处时的常见反应和话语背后的含义，帮助大家更好地把握两性交往中的关键信号。

全球有近76亿人，中国约占14亿。与我们年龄相仿的人群约有9000万，两个人相爱的概率是**4500万分之一**。遇到一个令你心动的女性并不容易，因此我们一旦遇到便不愿轻易放手。

为了解决这个问题，我研究了大量案例并收集了各方面资料，将所有精华汇总成《读懂女人心》专栏，旨在帮助各位男性读懂女性心理，做到知人知面更知心。相信本专栏一定物超所值，让你收获颇丰。

对本专栏的介绍到此结束。下面正式进入《读懂女人心》第一期课程。想要破解女人心，首先要明白女性心理究竟在想什么。为此，我采访了不同年龄层、不同性格、不同职业的多位女性，聊了她们难以直言的故事。最后我发现：

无论是活泼的还是安静的，无论是情窦初开的少女还是经验丰富的成熟女性，在两性关系中，她们都存在一个共同点：**所有女性都喜欢“以小见大”**。男性有时随口一说或不经意的一个动作，就能让女性陷入思考。

在两性交往中，男性常常误解女性的意思，女性也很无奈，觉得男性不懂自己。想知道女性真正在想什么吗？下面我将分享四种最常见的情况，看看隐藏在她心里的话。

以下是四种常见情境及其背后的女性心理：

1.  **“我要的是感觉”**  
    许多男性疑惑：自己在相处中很规矩，从不轻浮，经常送礼物，每天按时发早安晚安，天气变化及时关心，但女性却无动于衷。女性心里的话是：“他一开始方向就错了，自私地赋予了我某些属性，自发做了很多事，但感动的只有他自己。”由此可见，女性不注重表面功夫，她在意的是你这个人本身，以及你能否带给她“感觉”，而不是程式化的关心。

2.  **“你再主动一点”**

4.  **“好感是被持续消耗掉的”**

虽说“女人心，海底针”，但仍然有规律可循。女性会以小见大，也常在一句话后隐藏另一层意思，表面行为可能掩盖真实目的。因此，在与女性相处时，男性需要根据她说话的语气、神态、气氛和环境综合判断她的行为。完全读懂女人心任重道远，但做到“听其言，观其色”，就能解决大部分心口不一的问题。
