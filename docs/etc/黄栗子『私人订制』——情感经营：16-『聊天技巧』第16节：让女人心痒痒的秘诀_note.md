## 一、 什么是情绪波动？

情绪波动是指由事件引发的心情起伏变化。这种起伏能让人对经历记忆深刻。

**公式：事件A（负面）→ 情绪低落 + 事件B（反转）→ 情绪高涨 = 深刻记忆**

例如，早上丢失钱包让你一整天沮丧，晚上发现钱包就在家中，心情瞬间由阴转晴。你可能会忘记具体琐事，但“失而复得”的强烈情绪会长久留存。

在两性关系中，如果不能激起对方心中的涟漪，就很难在她心中占据重要地位。
