# 魅男《全套合集》课程二：强筛术 🎯

## 概述
在本节课中，我们将学习“强筛术”。上一节我们介绍了如何收集资源，本节我们将重点讲解在社交软件上，如何快速筛选出有恋爱或速约意向的有效目标，避免在不合适的人身上浪费时间。

## 核心概念：强筛术
强筛术的核心是**快速表达兴趣，筛选意向**。其逻辑基于一个简单公式：
**有效关系 = 你的意向 × 对方的意向**
如果任何一方的意向为0（即无恋爱/速约意图），结果就为0。因此，我们需要快速识别并聚焦于意向匹配的目标。

## 操作前的准备：展示面设置
在开始筛选前，你的个人资料（展示面）是基础。展示面的质量直接决定了你筛选的“直接”程度和成功率。

以下是设置展示面的关键点：
*   **照片**：必须精修，展示高价值（如才艺、宠物、身材、优质生活）。
*   **昵称与签名**：根据展示面强度调整。若展示面强，可稍显露骨；若展示面普通，则应保持正常，以获取更多初始流量。
*   **个人信息**：不必完全真实，应以“塑造吸引力”和“以结果为导向”进行优化。例如，填写能提升吸引力的职业、收入、学历等信息。
*   **标签**：选择能传递价值的标签，如“会下厨”、“有腹肌”、“有趣”等。

**代码示例：展示面设置原则**
```plaintext
IF 展示面照片质量高 THEN
    资料可以更直接、露骨
ELSE
    资料应保持正常、温和
END IF
```

## 强筛术的两种策略
根据你拥有的资源量（匹配人数）和展示面强度，筛选策略分为直接与间接两种。

### 1. 直接强筛
*   **适用条件**：展示面非常强，每天有大量匹配资源。
*   **策略**：开场直接表明来意，例如使用“谈恋爱吗？”或“你喜欢直接还是委婉？”等话术。即使大量被拒绝也无所谓，旨在从海量资源中快速过滤出有意向的少数人。
*   **核心公式**：**筛选效率 = 资源基数 × 筛选直接度**

### 2. 间接强筛（报假性IOI）
*   **适用条件**：展示面一般或资源量较少。
*   **策略**：在正常聊天中，通过“表达兴趣”进行筛选，而非直接表明目的。这更委婉，目的是在不吓跑潜在目标的前提下进行测试。
*   **核心方法**：在聊天早期，自然融入表达好感的语句，观察对方反应。

**关键过渡**：了解了两大策略后，我们来看看具体如何操作。间接强筛是更普适、更安全的方法，下面将用我的实际案例进行演示。

## 实战演示：间接强筛话术与应用
以下是间接强筛的核心话术模式，其本质是发出一个兴趣指标（IOI），测试对方态度。

**常用话术列表**：
*   “第一眼感觉是喜欢的类型。”
*   “感觉还不错，挺有眼缘的。”
*   “你笑起来的样子很漂亮。”（针对具体照片）
*   “小猫挺可爱的，当然，你也很好看。”（声东击西）
*   “看了资料，感觉是我喜欢的类型。”

**操作流程**：
1.  匹配后，简单打招呼。
2.  迅速使用上述一句强筛话术。
3.  观察对方反应：
    *   **积极回应**：继续聊天，并尽快引导至交换微信。
    *   **无回应或消极回应**：放弃，转向下一个目标。
4.  核心目的：**将有效资源快速转移到微信**进行后续维护，避免因账号被封而丢失资源。

## 无效目标的识别与处理
并非所有匹配的人都值得花费时间。以下是需要立即排除的无效目标特征：

**无效目标列表**：
*   **非真实聊天者**：一上来就索要微信，或一直引导你进行特定消费（如刷礼物、去某酒吧）。
*   **养鱼者**：主播、营销号等，其目的是让你付出金钱或时间，而非发展关系。
*   **无意图者**：资料或聊天显示对方无恋爱或速约意图。
*   **直接索要微信者**：在社交软件上一匹配就发来微信号的人，多为无效或诈骗资源。

**处理方式**：一旦识别为无效目标，立即停止互动并拉黑/删除，不要有任何犹豫。

## 软件设置技巧
正确的软件设置能帮你获得更多初始流量。

**关键设置列表**：
*   **探探**：
    *   “发现”设置中，不要开启“不让太近的人看到”。
    *   “性别”选择“不限”。初期让更多人（包括男性）右滑你，可以提高账号活跃度，从而获得更多系统推荐。
*   **牵手等婚恋软件**：
    *   认真填写资料，参照“以结果为导向”的原则优化所有字段。
    *   **目的导向公式**：**吸引力 = 资料展示价值 - 用户心理阈值**。你的目标是让资料价值远超对方的心理阈值。

## 总结
本节课我们一起学习了“强筛术”的核心与操作。

1.  **核心**：强筛术旨在快速筛选出有共同意向的目标，核心在于**主动表达兴趣，测试对方反馈**。
2.  **基础**：你的**展示面质量**决定了筛选策略的激进程度。
3.  **策略**：资源多则用**直接强筛**，资源少则用**间接强筛（报假性IOI）**。
4.  **操作**：使用标准化话术（如“第一眼感觉是喜欢的类型”）快速测试，积极回应则引导至微信，消极则果断放弃。
5.  **排雷**：学会识别并立即剔除**无效目标**，如营销号、养鱼者。
6.  **设置**：优化软件设置（如性别选“不限”），以获取最大初始流量。

记住，所有操作都应服务于最终目的：**高效积累有效资源，并转移到微信进行长期维护和关系升级**。下节课，我们将讲解如何在这些有效资源的基础上进行吸引和关系推进。
