# 黑猫门徒核心课程：49.M3转性框架速推完整案例：铺垫居家下厨的筛选，会在合适的

## 📖 课程概述

在本节课中，我们将学习如何通过“居家下厨”这类话题进行筛选和铺垫，并在对方释放出合适的兴趣指标（IOI）时，自然引出私密空间的邀约。我们将分析一个完整的聊天案例，拆解其中的框架操作、指标识别与应对策略。

---

## 🎯 核心概念与框架铺垫

上一节我们讨论了话题的选择，本节中我们来看看如何具体操作。核心在于建立并维持“奖品框架”，即让对方感觉需要争取你的认可，而非相反。

“居家下厨”是一个优质话题，因为它能同时达成多个目标：
*   **建立安全感与人设**：展示你稳定、居家的生活面，降低对方的防备。
*   **方便筛选**：通过讨论口味、厨艺等，可以测试对方的配合度与兴趣。
*   **为私密邀约埋伏笔**：自然地与“来我家做饭/吃饭”这类邀约建立联系。

你的角色是“筛选者”，而非“追求者”。你需要搭建好一个“笼子”（即舒适且有吸引力的框架），当对方想“钻进来”时，邀约的条条框框已经在那里了。

## 🔍 案例实战：指标识别与框架攻防

以下是聊天记录中的关键回合分析，我们将重点关注如何识别兴趣指标（IOI）、无兴趣指标（IOD）和服从性测试（CT），并进行相应的框架维护。

### 回合一：话题开启与初步筛选

我通过“做饭”话题开启聊天，进行人设铺垫和初步筛选。对方如果接话，说明有一定兴趣。

**关键操作**：主动离场，控制节奏。聊天中敢于主动结束话题，能避免陷入对方的节奏，保持框架高度。

### 回合二：高风险操作解析（不推荐模仿）

这里案例中提到了一个去“密室”的高风险邀约操作。这需要满足严格的前提条件：
1.  当前框架是你高对方低。
2.  邀约地点是对方感兴趣并主动提出的。
3.  该场合能为你增加社交光环（例如，是多人社交局，而非一对一约会）。

**核心公式**：`高价值邀约 = 自身高框架 + 对方兴趣 + 社交认证加成`
除非你非常有把握，否则不建议初学者模仿这种操作。

### 回合三：细节暴露与框架测试

对方提到“你是不是要给我改备注了”，这暴露了一个关键点：**她非常在意“改备注”这件事**。

**框架分析**：
*   在意“备注”，意味着她期待双方关系有“特殊性”。
*   这本身是一个隐性的兴趣指标（IOI），表明她在投资情绪。
*   我的应对是维护框架并进行筛选：“这还记不住吗？为什么要改备注？”

### 回合四：合格台词与兴趣指标

对方说“我们才认识多久，你就这样”，这是典型的“合格台词”（一种看似拒绝实则调情的IOI）。

**识别与应对**：
*   **识别**：我没提关系进展，她却主动提及，这是害羞和兴趣的表现。
*   **应对**：我选择不直接接茬，假装没听到。这制造了“猫绳效应”，让对方产生“他为什么不接我话”的疑惑，反而可能放大她的需求感。

### 回合五：探索性指标与价值展示

对方说“我还不知道你叫什么”，这是一个探索性指标（IOI），表明她想了解更多。

**操作与失误处理**：
*   我尝试冷读（猜测她的姓），但猜错了。
*   **应对原则**：冷读有风险，猜对加分，猜错减分。但如果你本身框架优势明显，小失误无需慌张，坦然承认即可，保持姿态。
*   失误后，对方立刻抛出筛选：“那你觉得我姓什么？” 我选择了冷处理，并开启新话题（密室照片）来转移焦点，剥夺她起势头的机会。

### 回合六：软框架与反筛选

对方发穿搭照片并说“把自己拍得胖胖的”，这是一个“软框架”圈套，意在引导你去夸她。

**拆解框架**：
*   普通男生会回应：“你不胖，很瘦。”
*   我的应对是：不评价胖瘦，而是从其他点调侃（“不是胖，是没站直”），不进入她的框架。
*   进而抓住核心：“你怎么这么在意胖瘦？” 将圈套反抛回去，对方开始解释，框架被拆解。

### 回合七：模糊邀约与节奏控制

对方提出“以后见面你看我是不是真胖”，这是一个包裹在调侃（CT）里的模糊邀约（IOI）。

**应对策略**：
*   我没有接招，而是敷衍回应。因为对方仍带有“上位者姿态”。
*   **原理**：不接招，让对方的需求感打在棉花上，反而可能放大她的需求。但注意，**前期关系未稳时慎用**，容易掐灭火苗。当对方已投入足够多沉默成本时，可以选择性回应IOI。

### 回合八：角色调转与框架确立

当我敷衍后，对方开始主动围绕我的话题（密室）展开，并说“你太聪明了…（话没说完）”。

**框架胜利的标志**：
*   对方开始从你身上找话题，顺着你的框架走。
*   说话变得犹豫、欲言又止（这是害羞和服从的表现）。
*   这意味着角色已经调转：**你成为了被追逐方**。

### 回合九：重启话题与社交认证

第二天，我自然地重启话题。关键在于“连续性”，每天聊天后的重启不会显得突兀。

**价值展示**：我分享了一张在轰趴馆当“狼人杀法官”的俯瞰视角照片。
*   **社交认证**：在社交游戏中担任核心角色（法官或高手），能展现领导力和社交光环。
*   这为个人魅力增加了筹码。

### 回合十：情绪推拉与最终铺垫

经过多轮调侃打压（推）和展示价值（拉），对方情绪被充分调动。

**关键回合**：围绕“吃”的互筛
1.  对方说“不吃怎么长肉肉”，是引导投资的筛选。
2.  我调侃打压，否定她的框架。
3.  对方试图证明自己（“看来我不太胖”），我指出她前后矛盾，拆解小心机。
4.  我抛出被动态框架：“你离我远点，别啃我。” 对方进入框架：“够你吃两顿”。
5.  **最终，我引出铺垫好的邀约筛选**：“那你得自己下厨”。对方立刻跳步：“好的，我去你家给你做吧。”

**面对跳步的应对**：
*   不能立刻答应，那会暴露需求感。
*   必须把节奏慢下来，增加筛选：“你来我家买菜？这有点问题。” 保持矜持，节奏放慢对我们更有利。
*   对方转移话题后，我实施“惩罚”——也转移话题，不接做饭的茬，保持态度一致性。

## ✅ 核心要点总结

本节课我们一起学习了如何通过“居家下厨”话题进行有效铺垫和筛选，并在实战中维护框架。

1.  **人设铺垫**：利用安全、居家的話題建立舒适感和吸引力。
2.  **指标化应对**：坚持只对兴趣指标（IOI）、服从性测试（CT）做出反应，忽略或打压无兴趣指标（IOD）和合格台词。不要猜测女生多变的情绪，只看可衡量的行为指标。
3.  **框架至上**：永远优先维护自己的“奖品框架”和态度一致性。不因对方的单次IOI而剧烈改变态度。
4.  **控制节奏**：敢于主动离场、冷处理，并在对方跳步时放慢节奏，增加筛选。节奏控制者掌握关系主导权。
5.  **动态平衡**：聊天是动态的，应对方式并非唯一。需要在保持框架一致性和给予对方适当回旋空间之间找到平衡。

记住，整个过程的本质是 **“搭建舞台，让对方主动走来”** 。你只需做好筛选，并在合适的时机，亮出早已准备好的邀约路径。
