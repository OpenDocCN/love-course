## 学员提问：公众演讲的心态波动问题 🎭

学员描述了一个关于公众演讲的心态问题：两三年前开始，在演讲或汇报前会出现紧张状态，如气短、心跳加速。今年2月参加线下课后，有三四个月时间完全没有这种感觉，演讲非常轻松。但到7、8月份，这种轻松状态又消失了，现在又回到了紧张的状态，等待上台时会心跳加速，上台后看到很多人会气短、声音改变。学员尝试回想2月份导致心态变化的具体原因，但想不起来。因此想请教摩卡老师，从3.0课程的角度应如何理解，以及是否有方法能一旦觉察到紧张状态，就能迅速跳出，让自己保持轻松愉快。

## 核心解答：接受与内核稳定 🤝

首先，这个问题的内核在于“接受我与大家的区别”。许多真实、客观、自然的发生和存在，大家是不接受的。学员当前问题的深层次原因就是一种“不接受”。因为希望状态一直好，这个想法往往会导致状态不好。这是一个辩证且矛盾的现象。

在生活中，大多数人都在追求好的结果，如发财、顺利、升职等。但如果不具备稳定强大的内核，大多数情况下都达不到预设的“好”的标准。而实际发生的事本身可能并不差，只是因为它低于“好”的预期，所以显得差了。主观上快速贴上“不好”的标签，但客观事实未必如此。我们的感受会迅速过去，但情况已被扭曲。核心在于：事情的“不好”，是被你预设的“好”的预期衬托出来的。

回到演讲问题。状态时好时坏是自然而然的。希望它一直好下去的方式，并非寻找一种让问题永不发生的心态或理论。真正的方式是：**在不好的时候，接受它**。紧张、焦虑时，诚实地面对自己的小我，启动沟通程序。即使沟通后仍有些紧张，也接受它。上台后，可以将紧张诚实地释放出来，例如说：“我现在有点紧张，我也不知道为什么。” 释放后，通常在几分钟内就能进入状态。

这与搭讪焦虑类似。真正的搭讪焦虑只存在于看到目标到说出第一句话之间的时段。一旦开始行动，焦虑就大幅降低或消失。演讲前的等待时段也是最紧张的，想得越多，焦虑反而越大。这与人脑的一个特性有关：**自我（120比特/秒的算力）想得越多，焦虑不是减小，而是增大**。因此，在公众演讲前，过度思考对演讲质量的正面影响不大，负面影响可能很大。演讲应是一个自然的输出。

开启与小我的沟通模式后，如果仍有紧张，但其能量已降到可以处理的程度，那么上台第一件事就是“诚实释放”。**态度大于内容**。如果听众不认可你的态度，就不会认真听内容。诚实地表达紧张，是自我与小我贴近后，对外部的自然投射。这种坦诚能拉近与听众的距离。反之，试图用自我（120比特/秒）去管理小我（1600万比特/秒）产生的细微反应，是极其困难的。

至于状态为何波动（以前不紧张 -> 后来紧张 -> 上课后不紧张 -> 现在又紧张），这本身不构成问题。因为**接受“存在即有理”**。它是自然发生，背后一定有原因，但知不知道原因不重要，重要的是尊重既成事实。尊重事实就是尊重自然。中国传统文化讲“人法地，地法天，天法道，道法自然”。生活中大多数事情都是自然发生的。痛苦常源于对抗这些自然发生的事。越抗拒，越持续。

例如，喜欢一个女生而对方不喜欢你，这是自然发生的。但很多人不接受，试图用各种方法扭转，这个过程无论成功与否，大多伴随痛苦。因为需要用极大的能量去对抗自然。一件事本身无所谓好坏，好坏标签是站在不同利益角度贴上的。如果能通过3.0课程的学习，理解并做到“因无所住而生其心”，生活就会顺利很多，心力和幸福感会大幅提升。因为原本用于对抗、消耗的能量，可以转而用于建设性的事情。

## 关于“十二字真言”与“随他去”的深入探讨 🧘

学员联想到线下课提到的“十二字真言”（尽全力，随他去），询问“随他去”是否就是“接受”，是否与本次解答的核心相似。

“十二字真言”中最难做到的就是“随他去”。因为从出生至今，我们的教育和经历都在不断建立、强化和壮大“自我”。自我膨胀，就会抓取越来越多的东西（我的名字、面子、房子等）。自我感越强、越膨胀的人，越难做到“随他去”，因为这需要放下、弱化或松动自我感。

对于那些没有太多经历、未曾跌倒、仍迷信“我命由我不由天”的人，对他们说“随他去”，他们可能无法理解甚至不认可。往往是在生活中重重跌倒过的人（如经历失恋、离婚、失业、创业失败），才不再迷信自我的力量，懂得谦卑，风水才能真正好起来。

要理解，任何事情的成败，自我能做的只是其中一部分。很多事情是多种因素、能量综合平衡后的结果。有句话叫：**任何自然而然发生的事情，无论多小，都是全宇宙都同意的结果**。例如，从南京到广州参加线下课并能顺利返回，这么一件小事，也需要所有相关环节（国家政策、交通、家人、公司、摩卡的身体等）都同意才能实现。

深刻理解这一点，才能慢慢做到“随他去”。这之前是“尽全力做”，而“尽全力做”之前是“观因缘”（可理解为“看窗口”）。努力的前提是有窗口，没有窗口的努力是无效的。努力只是必要条件，而非充要条件。例如，扫大街的环卫工人很努力，但很难因此成为百万富翁。所以，“十二字真言”的逻辑是：先“观因缘”（看窗口），再“尽全力做”，最后“随他去”。

## 关于送礼物的场景与情商 🎁

有学员提到群内关于相亲见面是否要送礼物的讨论。摩卡老师当时的建议是：把菜品规格安排好，预期不要太高，否则会锁住能量影响发挥。

这与今天讲的内核稳定（不要拉高预期）相关。但还有一个不同点：在相亲（特别是由介绍人、家人陪同的第一次见面）场合，**不适合送礼物**。这会给双方带来压力和尴尬感。相亲严格来说不算约会，更像是认识。结束后如果感觉不错，在第二次单独见面时送个小礼物是合适的。第一次相亲就送贵重礼物，尤其会增加压力，可能让整个气氛变得奇怪。这是一个常识和情商问题。

## 关于“从未送过礼物”的澄清与成长 🔄

有同学提到摩卡老师早期说过“从未给女生送过礼物”。那是在很早（约2014-2015年），处于纯PUA思维笼罩下的状态。在健康良性的人际关系中，礼物是表达情感、重视对方的心意载体。无论是朋友、女朋友还是家人，礼尚往来是正常的。一个在意健康的关系中完全没有礼物，反而是不健康的指标。

今天再看，当时处于盲目执行国外PUA理论的阶段，是缺乏独立思考的。现在作为一个在情感、人际关系上相对成熟的个体，认为在健康的关系中，礼物和礼金是自然存在的。提醒大家：学习1.0、2.0、3.0课程，**不要陷入某一个单一的概念或一句话中**，要着眼整体和系统，抓取精髓，建立大局观和整体观。

## 关于“双向觉察”的练习 🧠

“双向觉察”需要练习，在初期会消耗脑力和注意力，有点累，就像学习任何新技能一样。任何技能在养成习惯前，都会经历一个有点累、有点痛苦的阶段。就像小时候学走路会摔很多跤，但今天走路已是自然而然的事。摩卡老师计划将来为自己的孩子记录学走路摔跤的次数，让孩子明白，今天看似简单的事，当初也经历了大量练习和挫折，从而在面对困难时获得勇气。

心态稳定强大，真的可以淘汰掉90%甚至更多的人。根据统计数据，中国月收入低于5000元的人仍有十几亿。我们常常活在自己的感知圈里，离客观事实很远。

## 学员案例：居住环境与能量状态 🏠

学员明一分享了自己的情况：在深圳租住的房子住了5年，因为便宜、性价比高，但采光和通风差，家具多是别人送的旧物。在深圳感觉疲惫、睡不好，回老家休养一个多月后，整个人充满激情和活力，作息规律（早睡早起），听歌都有感觉。但回到深圳后，状态又蔫了。他怀疑是否与作息或房子有关。

摩卡老师指出：
1.  **核心问题**：学员在老家与父母相处消耗了能量，导致回深圳后疲惫。状态与城市无关，与自身生命力状态有关。
2.  **作息影响**：**同样睡8小时，不同时间睡觉（如顺应地球日月规律早睡早起 vs. 熬夜）对能量状态的影响完全不同**。
3.  **居住环境**：学员不愿待在家里是**结果**，原因是环境（采光、通风、家具、能量场）让他不舒服。他为了省房租，支付了更高昂的代价（能量、运势、创造力受影响）。**采光和通风是居住环境中最重要的因素**。他忽略了最重要的东西（西瓜），捡了芝麻。
4.  **深层模式**：这种行为模式源于潜意识里的“**我不配拥有好生活**”。用别人扔掉的东西，还引以为傲。这种模式导致他5年来省吃俭用，却省出了负债，生活质量低下。
5.  **改变之道**：首先要“看见”这个模式。看见即疗愈。然后，**要对自己好一点**，使用好的东西，吃好的东西，住进明亮、通风、让自己舒服的房子。越对自己好，生命力越强，创造力越强，结果越好，从而进入正循环。

学员后来掉线，但摩卡老师总结：他能意识到自己能量振动频率低，是很好的开始。一旦看见并改变模式，调整能量频率，就会越来越好。就像从FM交通台调到AM音乐台，接收的信息和创造的结果会完全不同。
