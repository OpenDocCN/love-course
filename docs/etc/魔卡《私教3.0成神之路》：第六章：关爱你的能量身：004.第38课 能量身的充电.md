# 魔卡《私教3.0成神之路》：第六章：关爱你的能量身：004.第38课 能量身的充电
## 004. 第38课 能量身的充电：做计划的意识调频作用（上）⚡  

在本节课中，我们将要学习如何通过“做计划”这一行为，来为我们的能量身进行充电和调频。我们将探讨计划如何对抗焦虑、建立生活秩序，并初步了解其可视化的重要性。

---

上一节我们介绍了通过睡眠和饮食来提升能量的具体方法。本节中，我们来看看当日常生活异常忙碌时，另一种关键的充电方式：**做计划的意识调频作用**。

有的同学可能会说，每天忙到没有时间吃饭，半夜回家疲惫不堪。感觉一天中属于自己的时间，只有睡前刷短视频的一小会儿。虽然常常控制不住刷到凌晨两三点，但并非本意。也想早睡早起，吃得健康，只吃六分饱，但做不到。

记住，如果你脑子里有这样的想法，你要觉察到这是**NPC的页力自动化系统在表达**，是意识完全被锁在三维相里的陈述，是意识低频能量振动时会产生的想法。一旦能量振动频率提升，你自然不会再这么想。你也会自然地发现时间不仅没有减少，反而多出来了。

因为能量提升后，效率随之提升，你会正确管理时间，懂得断舍离。

不知道看课程的同学当中，有多少人有做每日计划安排的习惯。这其中，又有多少人会正确地做自己的每日计划安排？

以下是做计划的两个核心作用：

1.  **建立生活的节奏感**：节奏感和焦虑感是一对敌人。有节奏就没有焦虑，有焦虑就没有节奏。
2.  **实现可视化**：可视化非常重要。把你的愿望、计划、欲望、情绪可视化，对于我们建立自己的意识世界秩序至关重要。

因为我们从小到大，绝大多数时间都在学习物质世界的相关知识，对于意识世界的东西知之甚少。所以现在物质生活虽然极大丰富，但内心却越来越焦虑和痛苦，这就不足为奇了。

前段时间，我去北京拜访了一位朋友。他是中国最大的团体心理咨询平台机构的负责人。他告诉了我一组数据。

现在全国小学生的抑郁比例已达到23.9%，即平均4个小学生中有1个抑郁。中学生的抑郁比例已达到49.2%，即平均2个中学生中有1个抑郁。

这个数字说明，全社会的压力、焦虑和不幸福的负能量，已经穿透成年家长，流向了未成年孩子。我们越来越有钱，但越来越不幸福。这个问题值得大家花时间思考。毕竟我们以后大多要为人父母，生儿育女。

你可能会脱口而出，这是因为老师给的课业压力太大，或学校的校园霸凌太多。其实不是，这些现象是结果，不是原因。

原因恰恰是，几乎所有家庭的所有家长，都已基本处于极度焦虑的压力状态。而这种负能量，绝大部分是从家庭流向孩子的。

可能有的同学觉得，现在才30岁，结婚生娃还早，到了那时再说也不迟。但这里我想提醒你觉察以下几点：

*   觉察你现在的生活幸福指数。
*   觉察你现在对于时间分配使用的焦虑程度。
*   觉察你十年前的愿望，现在还记得多少。
*   觉察你今年年初许下的愿望，又达成了多少。

第32课的作业是畅想五年后生活的样子。你有没有认真对待自己的畅想？有没有把愿望可视化？有没有至少一个大致框架，分几步走？每年的分目标是什么？这五个分目标之间有没有联系和递进的关系？

如果你只是交完作业就忘了理想，如果你只是元旦一过就忘了新年愿望，如果你现在疲于奔命、精疲力尽、很不快乐，还认为没时间静下来思考，并幻想过两年情况就会自然变好。

那我想告诉你，这都是你的妄念。这种把希望寄托于未来而不是现在的念头，会让你远离自己，远离幸福。

你现在如果没有时间分配给自己，以后也一样没有时间分配给伴侣，未来也一样没有时间分配给孩子。你必然会陷入一种低频能量的生活状态：每天很累但不开心；为家庭努力工作，伴侣和孩子却离你越来越远；你觉得委屈痛苦，抱怨身边人不理解你、不和你亲密。

其实，你自己和自己也不亲密。你的自我和小我的距离很远。而身边人和你的关系，只不过是你自我与小我关系模式的投射而已。

本节课重点：做计划很重要。一个科学的计划，是将你脑子里的安排可视化出来。一个科学的计划，能够给你提供节奏感。最后一个科学的计划，能够给你提供完成感。

关于“完成感”，我们将会在下节课做详细介绍。

本节课中，我们一起学习了做计划对于意识调频和能量提升的基础作用，理解了计划如何通过建立节奏感和实现可视化，来对抗焦虑并构建内心秩序。下一课，我们将深入探讨计划带来的“完成感”及其强大能量。
