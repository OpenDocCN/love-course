**核心公式**：`提出小要求 -> 观察服从度 -> 判断好感度 -> 调整关系进度`

通过她对每个小要求的服从程度，可以判断好感度，最终在合适时机发出约会邀请，成功率会高很多。

## 如何正确运用服从测试？ 🛠️

了解了概念和例子后，我们来看看在两次相处中具体如何操作。

首先，通过提出小的要求来测试。例如，下午工作时请她帮你点一杯奶茶。如果她照做，可以进行下一步；如果拒绝，则需自然转移话题，切勿纠缠。

### 阶段二：下达测试指令的技巧
运用服从测试时，指令的表述方式至关重要。

以下是下达指令的注意事项：
*   **使用肯定句，而非疑问句**：不要给女生选择拒绝的余地。
*   **示例对比**：

    *   **正确（肯定句）**：“这里人多，我拉着你的手。”（同时伸出手）
    *   **错误（疑问句）**：“你要穿我的外套吗？”
    *   **正确（肯定句）**：“穿上，不然着凉了，我可不想带你去医院。”

当女生通过服从测试后，需要给予适当的正面回馈，以示奖励和兴趣。例如，约会时帮她拿包，或送一份小礼物，同时配合深情的眼神。

**关键点**：反馈要恰到好处，过度则会降低自身价值。

如果女生频繁拒绝你的服从测试，则需要采取“冷冻”策略。

2.  **适度拒绝**：当她提出要求时，可以适当拒绝。
3.  **制造轻微挫败感**：目的是激发她的好胜心和挑战欲，让她重新关注你。

“冷冻”一两天后，需完全忘记之前的拒绝，以平常心继续互动，并尝试提出新的、要求更低的服从测试（例如，从“拥抱”降级为“帮我整理一下衣领”），重新判断好感度。有时，退而求其次也是一种前进。

随着关系升温，服从测试的难度应逐步增加，形成一个递进链条：**打电话叫醒 -> 答应约会 -> 为你整理衣领 -> …**

`你提出服从测试 -> 她完成并投入 -> 你给予正面反馈 -> 她获得满足并愿意继续投入 -> 你提出更进一步的测试 -> …`
