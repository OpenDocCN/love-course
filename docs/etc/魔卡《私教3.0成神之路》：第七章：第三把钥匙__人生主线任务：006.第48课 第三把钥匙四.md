# 魔卡《私教3.0成神之路》：第七章：第三把钥匙__人生主线任务：006.第48课 第三把钥匙四

## 概述
在本节课中，我们将要学习关于“生命频道”的知识。我们将从日常生活中的电视频道比喻入手，探讨人类意识如何存在于不同的能量频道中，并重点分析第一种也是最普遍的频道——“受害者频道”的特征与表现。

## 从电视频道到生命频道
上一节我们介绍了寻找人生方向的钥匙，本节中我们来看看决定我们体验的“频道”。

在我童年时，家中只有一台电视机，频道数量很少。当时主要收看中央电视台，流行的电视剧是《渴望》。人们的精神生活和意识形态比较统一。

现在，我们可以通过电视、手机、网络和卫星接收全球各种频道与信息。人们的想法和意识形态也变得各不相同，由此延伸出各种风格、文化圈层和生活方式。

提及此事，是因为人类的意识也存在类似的分频道现象。不同的人具有不同的能量振动频率，相同频率的人会活在同一个频道里。然而，人生频道的切换并不像按遥控器换台那样简单。

这主要有两个原因：第一，很多人根本不知道还有其他频道可供选择；第二，人们难以摆脱原有频道的内容。

## 能量、振动与频率的核心
近代最伟大的科学家之一尼古拉·特斯拉曾再三强调，宇宙一切问题的核心在于三点：**能量、振动和频率**。

同样，人一生所遇各种问题的物理与智慧核心，也是**能量、振动和频率**。

人群中约有90%的人，自然地活在同一个能量振动频率很低的频道中，这个频道被称为“受害者频道”。

## 受害者频道的特征
活在这个频道里的人，可被称为NPC（非玩家角色），意味着他们的玩家意识不在线，生活缺乏自主意识。

你或许不认同此说法，认为每个人都有大脑和意识，都在努力生活、追求理想。我这样说的依据在于，这个频道的人具有三个共同特征。

以下是这三个核心特征的详细说明：

**第一，永远向外求取。**
他们的欲望没有止境。无论是财富、美色、地位还是名望，获得多少都不会满足。努力进取是好的，但不知意义的进取就失去了生命力。这种无意义的努力，类似于三恶道中的“饿鬼道”，永远在抓取，却永远感到饥饿。为了获利，他们可以抛弃幸福、感情、道德、底线、亲人、爱人乃至健康。

**第二，强迫性重复。**
在向外求取的路上，他们进行着强迫性重复。无论是否得到，结果都无效。得到时，像服用止痛药，药效过后痛苦依旧；未得到时，则持续处于痛苦、挫败与绝望中。他们不断经历无聊、虚无、挫败和绝望，最终陷入无意义感。这类似于“畜生道”的动物，终生执行性欲、食欲和求生育的机械程序。

**第三，长时间心烦意乱。**
无止境的外求和强迫性重复，必然导致大量问题与障碍。由于意识处于受害者频道，他们总感觉不顺，觉得所有人都在与自己作对，所有事都不如己意。他们的内心能量无序且矛盾，活在“总有刁民想害朕”的幻觉中，生活充满无休止的斗争与冲突，几乎没有良好的滋养性关系。这样的人如同活在地狱，每日承受痛苦折磨。

## 受害者频道的人生准则
受害者频道的人是推卸责任的高手、抱怨的行家和制造负能量的大师。

他们无意识地奉行几条人生准则：
*   凡是让我不舒服的就是不对的。
*   凡是让我不理解的就是错误的。
*   凡是让我不喜欢的就是不好的。

## 总结
本节课中，我们一起学习了“生命频道”的概念。我们了解到，大部分人不自觉地活在“受害者频道”中，其特征表现为**永远向外求取**、**强迫性重复**和**长时间心烦意乱**。生活在这个频道的人，如同苦海上一条没有船长的空船，随波逐流，无论风向如何，对他们而言都是逆风。认识到自己所在的频道，是做出改变的第一步。
