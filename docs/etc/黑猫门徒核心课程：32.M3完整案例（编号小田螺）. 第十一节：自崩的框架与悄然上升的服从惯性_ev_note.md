# 黑猫门徒核心课程：32.M3完整案例（编号小田螺）. 第十一节：自崩的框架与悄然上升的服从惯性 📈

在本节课中，我们将通过一个完整的聊天案例，深入剖析在互动中如何应对对方的挑战、维持自身框架，并观察服从惯性如何逐步建立。我们将重点关注指标识别、框架博弈以及情绪能量的管理。

## 概述：框架博弈的核心

上一节我们探讨了如何识别兴趣指标。本节中，我们来看看当对方不服从时，如何通过一系列操作稳住自身框架，并引导对方形成服从惯性。

关键在于不害怕将对方的情绪指标“憋回去”。如果对方出现A3指标（即尝试确立自身框架的指标），但经过博弈后并未服从你，那么你绝不能先服从。**把对方的情绪憋回去并不可怕**。第二天，着急的会是对方，而你保持镇定。这比拼的是心态。

所以，这时我可以嘲笑对方。我越是嘲笑，对方可能就越上火，心态越不稳定，甚至会觉得自己做什么都是错。

## 关键词的语境与框架预设

“可爱”这个词，在这里肯定不是好话。所以，词语的含义要看语境，并非所有词都一定是正向或负向的。“可爱”在这里的潜台词是：“你真傻”。

当女生已经适应了我常规的负面表达方式后，她自然不会认为这是一个正面词汇。同理，如果你长期给予对方正面反馈，如夸奖和赞扬，那么即使你偶尔说一些中性词，别人也会往正面去理解。

**因此，最初形成的互动框架至关重要。**

当然，这是因为对方做的是一个**服从性测试（Compliance Test，简称CT）**。我拆解之后，这里也是一个变种操作：我又做了一个**反思机会**，我说“做梦了嘛”。这利用了之前已有的梗。对方之前曾用做梦来做个CT，仅此而已。

## 应对不服从与施加惩罚

然后，对方没有直接回答我。这里我将其视为不服从，因为她完全没有往我的话题上靠，还在说自己的事。

所以，我扔出了一串“狗头”表情（一种IOD，即无兴趣指标）加上惩罚。这是一个不屑的态度。结果，对方反过来赶紧服从了。

我们讲过，她在IOD之后再次服从，相当于发起了一个新的CT（比如“碰脑我干嘛了？”）。好，接下来她这回服从了。所以我的态度，相对于上一个回合就好很多。

但是我也没有给出一个特别明显的奖励。这只是相对于上一回合的态度好一些，在观感上就算是一种奖励了。

## 重启话题与应对假意示格

紧接着，我又重启了一个新的**高价值展示（DHV）**，以拉开距离。这是一个很标准的操作。

接下来她的两句话就很有需求感了。越接近A3阶段（即A2后期），对方就越容易暴露需求感，但我们还是要保持稳定，并进行筛选。

然后紧接着，对方做了一个**假意示格（False Disqualifier）**。注意看：“你看窗外吧，别陪我了”。为什么是假意示格？这里解释一下：假意示格是**先设定一个前提关系，然后又主动破坏这个关系**。例如“分手吧”的前提是“你们在一起”。在这里，“别陪我了”的潜台词是“你之前是在陪我”。所以，这里本来我只准备执行一个“钓鱼”操作，扔出一个假性IOI（兴趣指标）。这种技能在后期可以使用。

结果后来被她破坏掉了，但破坏掉也没关系。我们只要后续保持技能，拉开距离就没问题。你看，我一旦松口（刚扔出钓鱼的话），本来想着她可能会贴上来，结果没有。她开始做CT了，看到了吗？你扔出IOI，人家就做CT。那好吧，我下面只好变招。

## 调整策略与保持框架

我还是保持了一个钓鱼的思路，但是，我再找一个推开释放的点。钓鱼如果在后期使用，就不容易被别人认为是舔狗。就像我们刚才讲的，你前期已经形成了一个框架：我是负面的，我是奖品，我跟你保持距离。所以我说这种话，她也不会判定为我要去拉近关系或追求她。她甚至敢做这个CT都算很大胆了，其他女生可能都不敢，只会试探性地小心询问。

当然，这个妹子段位确实不低，至少直觉很强，给我扔了一个CT。

**接CD（冷读）是什么？** 往前翻一翻，大概就是猜测她是什么样的人，诸如此类。那我呢，我这里还是琢磨着怎么把这个钓鱼给做完。所以，我不着急去释放，先顺着她的框架往下讲，保持一下自己的框架，然后立刻说“我觉得挺可爱的吧”。紧接着，释放掉“傻傻的”那种感觉。

## 提前释放与应对强框架对手

这里为什么我要提前释放，不等她上钩才释放呢？因为我觉得心里不稳。这妹子的直觉太强了，我稍微一松口，CT就扔过来了。不像其他女生的反应都是“哎呀，这么好呀”。她框架很强，那我就不敢跟她接着绕弯子，尽快把这个能量释放掉，这样我比较稳。虽然没法达到在情绪能量最高点释放的效果，但最起码不会丢分，赶紧释放掉。因为她接下来不知道会打出什么牌，打乱我的操作。还是赶紧回到我们的框架里比较好。

## 复合框架与二选一陷阱

好，接下来就很有意思了。我释放掉之后呢，她并没有继续追加太多情绪，不像其他女生那样用同样的情绪反馈于我。她来了一个提问：“问什么傻傻的”，这又是**融入我们框架的一个节奏**。

但是呢，接下来的操作（当然我当时不知道），当我问她为什么傻傻的时候，我只能判定这是一个兴趣指标，她进入了我的框架。我就做了一个简短的CT。结果她接下来的行为是给我一个**复合框架（Compound Frame）**。

这个复合框架很有意思，就是我们之前在核心课里讲过的一个操作：**让人二选一，就能看出对方的倾向**。但她不知道的是，在这里她把“喜欢对一个人的喜欢”放进来，这本身就代表了她的倾向。我前面一点也没有表现出对她的喜欢，没有任何暴露需求感指标的地方。说白了，她有点着急了，想引导我，同时给我复合框架。

一般来说，其他男生可能在这里就觉得女生都说到这个地步了，那我肯定选第二个（喜欢）了。但以我们这样稳定的框架来说，我肯定不会选二。她想让我二选一，然后让我自己暴露倾向。我可以选择不选，或者选第一个（不喜欢），但其实不选是最好的。因为为什么？说白了，**她这个行为暴露了她自己的倾向和需求感**。这种情况下，可以视为一个隐形IOI或IOI。

我们要是选了第一个的话，相当于是做成一个负面服从，不太好。但是这里她又给我复合框架，我不去拆也不行。所以我就尽量不去造成这样的对抗（即绝对性质的负面服从），我直接选择了跟她倾向性完全相反的第一个选项。当然，如果你真的觉得稳操胜券，对她的情绪沉默成本掌控得很好，你当然可以选第一个。但最稳妥的是不选，就最好了。不选，让她自己去胡思乱想，让她暴露更多的倾向，以便我们做更多的CT。

所以我这里只能去轻微地拆了一下她的复合框架，带点小自恋，而没有去做出一个对抗性的选一。当然了，要是她框架很强，以及你觉得对方沉默成本很高，动不动就给你表白的情况，我当然可以选第二个逗她玩，反正你知道她会回来的。

所以你看到，我并没有在对方负面服从的时候发作。其实相当于是我的框架在这里没有那么强，她还敢继续向我提出CT。所以我这里才强调了我的行为是：**不解释，你自己想去吧**。我把CT扔回去，同时表明了我的框架——我不选，你自己想。

## 逻辑博弈与情绪能级管理

哎，她倒好，这里框架拆得非常有意思，利用我给他的复合框架，然后来拆我的框架，拆得很棒。那我这里只能强词夺理了。有的时候，你们聊天不是一定要逻辑那么严谨。我这句话就完全没有逻辑：“傻傻的才有可能想出来”。女生也经常会给你说这种毫无逻辑的话，让你去琢磨到底什么意思。其实可能这句话只是她当时搪塞你的话而已。我这里就是搪塞她，我实在想不出来什么话，把这句话给他怼回去了。毕竟她用的是我的话拆我的框架。

所以接下来，对方给我甩了一个IOD，看到了吧？为什么？因为**我没服从，她给了惩罚**。我也回应了IOD，同时拆她框架：“什么叫作弊玩呢？你来回一会儿傻一会儿不傻，到底傻还是不傻？”

后续对方发出这个表情。在IOD之后，我回以IOD加一个否定。同时，**字数、情绪、能级下降**，对方就回落了一个IOI，试图又把我的情绪拉回来，回应了三个表情。但是呢，相对前面一个回合而言，他这个能级……看IOI、IOD，有时候不光看字面，很多时候看的是前后回合的能级对比。

前一回合给我甩IOD，这一回合又扔IOI（扔表情拉回来）。如果说对方之前都是一个兴趣指标，或者说比较好的能级情况下，这时候扔出这样一个表情包，我不会认为它是一个很强的IOI。因为它相对之前能级并没有上升多少，而且这个兴趣指标并不是很明确，没有一个具体的东西在里面，没有借口。

但是，**相对前一个回合是IOD的情况，同时是因为你甩IOD，不是你自己舔回来的**——一定要记住。你甩了IOD，让他自己舔着脸回来了。他原来是在跟你耍脾气，马上自己又堆着笑回来了。那这时候他肯定不可能真的舔着脸说“哎呀怎么怎么好喜欢你”，那很少数。他能把能级从负数变到正数，但我不认为这是个很强的IOI，就筛选就可以了。但如果他本来是一个负3变到正2或正3，那这个跨度就大了，那你不可能指望他一下子爆出很强烈的IOI，人家也要面子。

这样比较来说，这块我可以做个CT，但也很简短，不要复杂，能级不要高，简单就行，不要有很强需求感。我就顺着他的表情来：“来抱抱”，他不是放抱抱表情吗？那你来抱抱。啊，这就补了一个“我”。因为我后来反应过来一个事儿，他总是让我去抱抱他，对吧？我就说“来抱抱”，他说“那你抱我”，我这补了“我”。结果对方没理我。

## 延迟回应与指标真伪判断

过了一会儿，她又回我一句话：“你干嘛呢？”这里你们可以判定前面的兴趣指标是假的，我没做这个CT，不应该释放，都可以。判断为一个操作失误，也可以。判断成她不服从也可以。不管是如何，你CT既然做出去了，那**对方服从就给奖励，不服从就给惩罚是必须的**。

那对方这个，我不管他装傻还是真傻，反正他没服从，而且也没在这个话题中继续（比如提出条件等等）。那提出条件，我可能可以继续去强行强调这个CT。但是现在他连条件都没提，直接跳出了框架，那我就直接给惩罚。那对方显然是没反应过来，那说明前面这个兴趣指标有可能是假的，只是他为了圆场扔出来的。我是不确定的。

然后呢，我也没理他，照常操作。直到他说出下一句话：“也许是想你了”。你看这个表情，他解释了自己上面的行为。OK，这又回到了一个标准流程：**我扔CT -> 对方兴趣指标 -> 我扔CT -> 对方没服从 -> 我惩罚 -> 惩罚完后，对方出现兴趣指标 -> 那我们就能做新的CT**。

我这个CT其实是延续了上面，但是其实是个新话题：“我让你抱抱我了”，我要重新做一遍。看到了吗？标准操作。所以还是那句话：**看指标，看指标，看指标**。

## 应对敷衍服从与奖励设置

啊，有时候聊天不用那么着急回。我有时候看女生发单表情或者只发一句“你干嘛呢？”，我会等等，看她后面会不会续一句话。有时候女生真的会给你续一句话，尤其是在进入A2后期之后，她会给你续，这很有意思。坏了，天下无人不舔狗。每个人都有舔狗的一面，女生也有。投入太多，她就会有舔狗的心态在里面。所以这个东西我们是不用去担心的。你心态一崩，被她抓到这个痛脚，那你就不用指望出现这种情况，你永远看不到，你会越来越自我怀疑。

这里很有意思，对吧？首先我是给了个奖励，但是她服从得很敷衍，所以我就给得也很敷衍，同时还扔了个狗头，对吧？抵消一下，对冲一下，让这个奖励显得没有那么真。哎，对方继续向我索取。这本身也是一个CT的感觉，向我索取奖励，但没有明确要什么。

然后我这里呢，等于是先甩一串IOD（“哈哈哈哈”），表达说你这个行为出乎我意料，很可笑。紧接着，**反CT**。这个潜台词是：“我奖励你见我，你要不要？”为什么我要奖励这个？首先，我们这个操作不能当做常规操作使用。她是在女生多次提出要见我的背景下（往前面看聊天记录），同时我们聊天一直是连续的，情绪保持连续。这种情况下，“要见我”已经是一个长期存在的梗。就是长期地她要见我，同时我用一个语序的方式设置“奖励你见我”，把我自己暗示成了奖品——必须是奖励的情况下，你才能见到我。这里面暗藏的玄机比较多，别看就短短的几句话。

它不能作为常规使用的原因我前面讲了，不是说这时候我感觉是我主动邀约了。女生长期提出要见你，多次有各种借口要见面，我只是延续了她这个梗。所以不要断章取义，拿我这句话去邀约女生。明白吗？特殊情况特殊对待，具体问题具体分析。

## 保持低需求感与框架一致性

好，我们接着往下看。结果对方：“嗯哈哈好啊”。这个服从得很敷衍，我奖励的也就很敷衍：“真乖，哈哈哈哈”。双方给的感觉都是没有当真，这就对了。先讲为什么他是这么一个敷衍的态度，因为他觉得我说这句话就很敷衍，没有说真的要见你怎么着，是在开玩笑的性质下说的。那他呢，又长期去提出要见我，那他不可能去拒绝，他顶多作为筛选。那他作为一个比较敷衍的服从呢，只能视为说他的服从惯性导致了这已经是最好的结局之一了。当然他如果真的是说已经很有服从惯性，那“好呀，什么时候见面啊”这是最好的。接下来我们可以做筛选，甚至可以去植入一些预期在里面。

但是在他判断前，他可能觉得我也是在开玩笑，所以他也开玩笑式地服从，我有开玩笑式的奖励。然后大家都没提这话题了，又重启别的话题去了。看到了吗？这还是一句话：**不要有需求感**。不要好像邀约了，或者说好了什么，你就一直盯着这件事。我在开玩笑，我感觉像开玩笑。如果对方认真了，我可能才会认真。那我开玩笑，对方也开玩笑，你这时候反而着急地认真了，那怎么回事？那你前面都是假的呗，装的呗，那你框架就崩了，一致性就失效了。

## 识别隐形兴趣指标与博弈

这里这个CT我为什么做呢？肯定没有看到明显的IOI，还是一个问题：**情绪的延续**。我们都已经带离话题了，他又拉回来了，想尝试。这是我刚刚前面讲的：我开玩笑他也开玩笑，结果我认真了，对吧？这里就变成了：我开玩笑，他开玩笑，我开玩笑，结果他认真了。我带离话题之后，他又试图回到邀约这个话题来。哎，这是什么？**隐形兴趣指标**。他做了一些你该做但没做的事情——本该继续往下讲话题，他跑回去了，这就是个隐形兴趣指标。

所以呢，我也做了一个很浅很浅的CT。然后我也没有很明确，你看我这CT不会立刻又怼上去了。我说“啊那等我眼睛好”，我也没说等我眼睛好是什么。你说得很隐晦，我也就说得很隐晦。你看这就是一个博弈的状态：我开玩笑，你开玩笑，我认真，那我就输了；我开玩笑，你开玩笑，我开玩笑，哎，结果他认真了。哎，对吧？我们后面又取得一波优势。

当然了他就给了一个表情，那种服从。我要给奖励，也肯定只给一个很简短的奖励就可以了，不用那么认真，或者情绪那么浓重，能级那么强的奖励没意义。根据情绪的起伏去判定一些东西。当然这里我连奖励都没给，因为我可能就当时觉得这个服从太敷衍了，越来越敷衍，那我就不会再给你奖励了。前面就已经很敷衍了，IOD加吐槽。潜台词是：你干嘛这么开心，我也不是认真说的。当然他可能没听懂。

## 复合框架的运用与对抗

这里就很简短的，没什么东西。这个图直接就往后看。调个男压、叠锤叠吹这些东西。我这里的意图其实是在给她复合框架，结果呢，她的“好玩”，利用这副哥又开始玩软框架：“哎呀保护我啊”，对吧？也是个CT。真的很有意思，她很会利用女人那个身份做CT。那我就反CT呗，我不接你茬，对吧？我扔了个别的CT给你。不服从，惩罚。对方继续抛出兴趣指标。啊，我就继续DHV呗，你看到吗？都是这样。因为这个兴趣指标我觉得太轻了，我就筛选DHV，筛选DHV。

这里又给我植入新框架，对吧？你看到了吗？我不植入新框架，她就会给我植入新框架。所以有的时候啊，碰到有些女生，你真的不用着急植入新框架。你发现偶尔植入点新框架，她不吃招的话，你不管她，反而她知道你可能会有点“污”的东西。有点“污”的东西是情绪，但一直“污”可能就是猥琐了。尤其在对方不接招的情况下，你就变成了需求感。
