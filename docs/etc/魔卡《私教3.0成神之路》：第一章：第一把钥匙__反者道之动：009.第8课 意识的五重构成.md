# 魔卡《私教3.0成神之路》：第一章：第一把钥匙__反者道之动：009.第8课 意识的五重构成 👁️

在本节课中，我们将要学习意识的五重构成模型。这个模型将帮助我们更深入地理解“真正的我”，并解释我们内在的冲突与感受。

---

上节课我们提到了“真正的我”是我的意识。本节中，我们来看看这个意识具体是如何构成的。

我们从小到大都认为自己有一个能被触摸和看到的身体。同时，我们也知道自己拥有精神层面的思维意识。尽管在解剖人体时，我们从未发现一个名为“思维意识”的实体，但我们确信它存在。法国哲学家、科学家笛卡尔也说过：“我思故我在。”

心理学家告诉我们，我们的意识分为“意识”和“潜意识”。对心理学了解更深入的同学还会知道，在潜意识之下，还存在一个“集体潜意识”。这些都是目前常见的对思维和意识的解释体系。

摩卡曾经历一年半的抑郁症，无法正常生活与工作，长期处于情绪低迷状态。至今，我已持续进行了五年每周一次的心理咨询，阅读了大量心理学书籍，并参加了许多心理学培训。心理学工作者的主要工作领域就在我们的潜意识。因为我们虽然能意识到生活中的许多现实问题，却常常感到除了痛苦之外，根本无从解决。原因在于，问题的根源往往深藏在我们的潜意识之中。我们自己能意识到的应对方法，基本上都是无效的。

但是，今天摩卡不会使用心理学的解释体系来为你书写意识的使用说明书。接下来，让我们开始本课正式的内容。

首先，我要告诉大家一个结论：我们每个人其实拥有五重身体。

以下是这五重身体的详细说明：

1.  **物质身**：即我们的肉体。
2.  **精神身**：你可以将其理解为你的意识。
3.  **能量身**：你可以将其理解为人的气场。有的人充满正能量，有的人则浑身负能量，这里指的就是能量身。
4.  **以太身**：物理学告诉我们物质有三种形态：固体、液体和气体。但实际上，在气体之上，还存在一种形态，称为“以太体”。
5.  **本我身**：你也可以将其理解为“元神”或“灵魂”。

这五重身体分别对应五种属性：
*   物质身属 **土**
*   精神身属 **水**
*   能量身属 **火**
*   以太身属 **风**
*   本我身属 **光**

我们常说“人性的光芒”，或形容一个人的“灵魂很高洁”，其实就是因为我们看到了这个人活出了他的本我身。他的言行举止都流露出了爱的光芒，让我们发自内心地爱戴和喜欢。

---

本节课中，我们一起学习了意识的五重构成模型，包括物质身、精神身、能量身、以太身和本我身，并了解了它们各自对应的属性。
