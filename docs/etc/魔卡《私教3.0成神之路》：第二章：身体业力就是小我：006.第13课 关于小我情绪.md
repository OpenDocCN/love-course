# 魔卡《私教3.0成神之路》：第二章：身体业力就是小我：006.第13课 关于小我情绪

## 概述

在本节课中，我们将要学习情绪表达的第四层，也是最高级的认知维度——四维人的情绪表达。我们将了解四维人是如何从根本上以积极视角看待情绪，并学会如何与内在的小我建立良性沟通，从而转化能量，改善内外人际关系。

## 从三维到四维：情绪认知的根本转变

上一节我们介绍了情绪表达的前三个层次。本节中，我们来看看最高级的第四维度。

四维人看待情绪的根本出发点与三维人完全不同。三维人无论处于哪个情绪表达层，首先都将情绪定义为一种负面事物，区别仅在于处理方式不同。然而，四维人对情绪持有**正面定义**。

大家还记得我们在第十一课中提到的核心概念吗？**情绪和痛苦是小我提示我们人生主线任务的方式和语言**。理解这一点是认知转变的关键。

## 转化沟通：与小我建立良性循环

一旦你理解了小我的善意出发点，并“转码”了它与你沟通的语言，所有原本的负面能量振动会快速、甚至瞬间转化为正能量的振动调频。

你不再需要向外宣泄情绪，也无须向内压抑，更不必战战兢兢地尝试“平和”表达。你只需要在四维世界——即投影源内部——与你的小我进行沟通即可。

这样的内部循环模式一旦建立，你与小我的关系会越来越好，越来越亲密。你的能量内耗将变得极少，而能量恢复与累积的速度会非常快。

## 内在模式的外部投射

相应的，当你在外部三维世界处理事务时，在三维人看来，你将成为一个效率极高、如有神助、非常厉害的人。

更重要的是，当你在四维世界内在，能够习惯性地看见并良性沟通你的小我时，这个模式会被投射到外部三维世界。你将具备看见他人小我的能力，以及与他人小我良性沟通的能力。自然，你与他人的关系也会越来越好。

以下是其根本原因：

*   绝大多数三维人的真正意识处于沉睡状态。
*   他们清醒时，往往只是他们的小我在运作身体。
*   因此，三维世界常规意义上的沟通，往往是**小我与小我的沟通**。

无论是夫妻、同事、上下级、朋友、同学，皆是如此。这就是冲突如此之多，良好关系如此难以获取的根本原因。

## 俯视感与根本转变

当你开始练习并建立这种内在沟通模式，达到一定阶段后，你会自然产生一种俯视感与居高临下感。同时，这种感觉中会充满善意，甚至包含很大程度的慈悲。

这就像是一个正常人，看着吸毒者犯毒瘾时痛苦、过着悲惨可怜生活的样子，会自然而然地想为他做点什么，希望他好起来。

一旦这种体验在你内在真实发生，那么无论你做什么，诸如跪舔、需求感、匮乏感这些感觉，都会从根本上离你而去。

## 核心比喻：投影源与影像

这里需要再次提醒大家一个核心世界观：我们所生活的三维世界只是一个被投影出来的**影像**，而**投影源**在四维世界，也就是我们的内心。所有你看到、经历到的一切人、事、物，都是先在你内心被构造出来，然后投影到三维世界的。一定要深刻记住这一点。

例如，我们用手机投屏在家里的电视上看短视频。电视屏幕上的成像是一个镜像，你在屏幕上滑动或遥控是无效的。一切有效控制都在我们的手机——即**投影源**上进行。

在这个比喻中：
*   **手机（投影源）** = 我们内心的四维世界。
*   **电视屏幕（影像）** = 我们生活的三维世界。

## 四维世界人际关系关键总结

最后，总结一个关于四维世界人际关系的关键知识点：
**除你以外的所有其他人，都是你人生游戏中的NPC。他们的出现，是为了帮你完成你的人生主线任务。**

## 本节课总结

本节课中，我们一起学习了情绪表达的四维认知。我们明白了四维人如何正面定义情绪，学会了通过内在沟通将负面能量转化为正能量，并理解了良性内在模式如何改善外部人际关系。关键在于认识到三维世界是投影的影像，而真正的改变始于四维的投影源——我们的内心。
