# 59、私人订制3.0：最佳互动话题---你真自恋【加

在本节课中，我们将学习一个用于拉近关系的互动话题技巧。这个技巧的核心是通过一种轻松、略带调侃的方式开启对话，旨在创造有趣的互动氛围。

## 概述

上一节我们探讨了开启对话的重要性，本节中我们来看看一个具体的、被称为“你真自恋”的互动话题。这个方法通过一个预设的情景提问，引导对方参与猜测，从而自然过渡到对对方的赞美。

## 技巧详解

这个技巧的操作分为几个明确的步骤。以下是具体实施过程：

1.  发起提问：向对方提出一个预设情景的问题。
    公式为：**“我最近喜欢上了一个女孩，你猜她漂不漂亮？”**

2.  等待回应：无论对方回答“漂亮”或“不漂亮”，都为后续对话留下了空间。

3.  给出答案并赞美：在对方猜测后，揭示答案并表达赞美。
    回应模板为：**“漂亮。（稍作停顿）美女都这么自恋的吗？学会了。”**

## 技巧原理

这个技巧之所以有效，是因为它融合了多个沟通元素。它通过提问制造悬念和互动，最终的答案巧妙地将赞美对象指向对方。用“自恋”一词进行轻度调侃，能缓和直接赞美的尴尬，让对话显得轻松、有趣。

## 总结

本节课我们一起学习了“你真自恋”这一互动话题技巧。我们分析了它的具体操作步骤，从发起提问到最终给出幽默的赞美回应，并理解了其背后制造互动和轻松氛围的原理。掌握这个技巧，可以帮助你在沟通中更自然地拉近关系。
