## 第二部分：身体的感知与绽放 💃

心态建设为我们打下了内在基础，本节中我们将把焦点转向外在，学习如何让身体变得柔软、敏感且充满表现力。身体的绽放始于感知。

### 核心练习：呼吸与链接

1.  **与空气链接**：通过深长的呼吸带动身体。吸气时，感受气息充满身体，带动躯干微微伸展；呼气时，缓慢释放，感受身体的放松与下沉。呼吸是连接内在与外在的桥梁。
2.  **与物品链接**：如果感知空气有困难，可以借助实体物品，如墙面、镜子或椅子。轻轻用手或身体部位触碰它们，想象在与一个有生命的对象互动，传递情感。
