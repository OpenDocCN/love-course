# 魔卡《私教3.0成神之路》：第一章：第一把钥匙__反者道之动：008.第7课 我是谁？_ev 🧠

## 概述
在本节课中，我们将深入探讨“我是谁”这个核心问题。我们将通过分析演员周星驰的例子，区分“角色”与“玩家”的概念，并最终揭示“真正的我”其实是我们的意识。理解这一点，是学习“意识使用说明书”的第一步。

## 角色与玩家的回顾
上一节我们介绍了角色和玩家的概念，并留下了作业。现在，我们来回顾一下。

周星驰在《喜剧之王》、《长江7号》、《少林足球》中扮演的是演员角色。在《西游降魔篇》中，他扮演的是导演角色。在《月光宝盒》中，他扮演的也是演员角色。

在演员角色内部，周星驰在不同电影中又演绎了不同的具体身份：
*   《月光宝盒》中的至尊宝。
*   《长江7号》中的周铁（父亲）。
*   《少林足球》中的“大力金刚腿”阿星。
*   《喜剧之王》中的尹天仇。

这些是周星驰所有的角色身份。

而玩家始终只有一个，那就是周星驰本人。无论周星驰在不同电影中演绎得多么令人印象深刻，我们都会说“这是周星驰演的”。后来他不再演戏，转而做导演并且很成功，我们也会说“这部电影是周星驰导演的”。

## 身份的延伸思考
如果将来周星驰离开电影行业，转行去做厨师或开餐厅，他的角色身份就会变成厨师。我们依然会说“这是周星驰开的餐厅”，而不会说“这是至尊宝开的餐厅”。

好，现在我们用同样的逻辑再深入思考一层。

## 意识与身体的本质
如果周星驰和魔卡互换了身体。周星驰的意识进入了我的身体，我的意识进入了周星驰的身体。那么，你会认为谁是周星驰，谁是魔卡呢？

描述类似情形的电影有很多，例如开心麻花出品的《羞羞的铁拳》。

所以，**真正的“我”其实是我们的意识**，而物质身体只是一台生物学的容器，其作用是承载我们的意识。这个原理与人开车类似：车只是一台机器，其作用是运载人去想去的地方。两者在本质上没有区别。

## 探寻“真我”
当别人问“你是谁”时，我会回答“我是魔卡”。但“魔卡”只是我的网络ID，并非我的真名。如果我愿意，我可以叫白卡、蓝卡、绿卡、红卡或黑卡。它只是一个代号。

我过去是学生，后来是打工者，再后来是情感教练，现在我的身份又变成了个人成长导师。因此，我的名字、职业、身体都不是本质的。**只有我的意识才是真正本质的“我”**。

屏幕前的你也是一样。所有人都是如此。

## 意识的未知领域
然而，我们对于这个最本质的“我”——我们的意识——其构成和运作逻辑几乎一无所知。没有人给过我们一份**意识使用说明书**。我们只能在生活中一点点体会和摸索，常常不得要领：不知道自己为何如此，不知道为何人际关系不佳，也不知道为何没做什么却总感到疲惫。

接下来的内容，你将正式开始学习你的**意识使用说明书**。

## 总结
本节课中，我们一起学习了如何区分“角色”与“玩家”。我们通过周星驰的例子认识到，名字、职业、身体都是可变的角色，而**意识才是恒定不变的“玩家”和真正的“我”**。理解“我是我的意识”，是开启自我认知与成长大门的第一把钥匙。
