## 第一节：核心站姿与胯部发力 🦵

稳定的站姿是所有动作的基础。本节我们将学习如何通过正确的脚部与胯部姿势，为后续的肢体表达打下坚实基础。

**核心站姿公式**：
1.  双脚自然站立，使大腿中间呈现一条缝隙。
2.  将一只脚（如右脚）向斜前方迈出约一步距离。
3.  绷直脚背，用脚尖点地支撑。
4.  重心落在后方脚上，前方脚作为辅助和发力点。

以下是胯部发力的关键步骤：
*   **找到发力点**：注意力集中在胯骨两侧。
*   **U型运动轨迹**：想象用胯部画一个水平的“U”字或“八”字。不是左右平移，而是两侧胯骨交替向上顶起。
*   **保持连贯**：在练习所有上肢动作时，胯部的摆动应尽可能保持不间断，这是产生动态美感的关键。

上一节我们建立了稳定的下盘基础，本节中我们来看看如何让双手也“活”起来。

## 第二节：手部动作与道具运用 ✋

手是表达情感的重要工具，而道具能有效解决“手不知该放哪里”的尴尬，并增加表演的层次感。

**核心原则**：手中有物，心中不慌。道具是手的延伸和参照物。

以下是常见道具的使用思路：
*   **扇子**：可用于遮挡、扇风、引导视线。例如，从下巴缓缓上移遮住半张脸。
*   **纸巾（替代品）**：捏在手指间，随手腕转动而飘动，模拟手帕或丝巾的效果。
*   **手机/抓夹**：可作为临时道具，握在手中或作为头发装饰，在动作中自然使用。

**基础手部动作代码**（以扇子为例）：
```
动作序列 = [“持扇于胸前”, “从下巴缓缓上移遮脸”, “向侧方平展打开”, “收扇轻点肩头”]
for 动作 in 动作序列:
    执行(动作)
    保持(胯部摆动) # 关键：胯部动作永不停止
```

掌握了手部的灵动后，我们需要将视觉焦点提升——眼神是灵魂的窗户。

## 第三节：眼神训练与表情管理 👀

眼神传递着情绪和力量。坚定、有故事的眼神能瞬间抓住对方的注意力。

以下是眼神力量的训练方法：
*   **定点凝视练习**：选择一个小目标（如自己的指甲盖、笔尖），集中注意力凝视，努力延长不眨眼的时间（从15秒开始练习）。
*   **方向移动练习**：按“上看→下看→左看→右看”的顺序缓慢移动眼球，每个方向停留片刻，保持头部不动。
*   **情感注入**：练习在凝视时带入微妙的情绪，如“好奇”、“神秘”或“温柔”。

**表情管理要点**：
*   避免频繁眨眼或眼神飘忽。
*   配合动作做出相应表情，如用扇子遮脸时，可配合略带神秘或俏皮的微笑。
*   沉浸于角色，忘记“自己在表演”的紧张感。

当肢体、手部与眼神都准备就绪后，我们将把它们融合到完整的表演情境中。

## 第四节：情境融入与完整流程演练 🎶

学习技巧是为了应用。本节我们将把前面所学的所有元素，融入到一个具体的生活情境中，例如“沐浴后”。

以下是“沐浴后”情境的演绎流程建议：
1.  **氛围铺垫**：在沐浴时就开始播放符合角色氛围的背景音乐。
2.  **角色入场**：披着浴袍或穿着睡衣走出浴室，此时已进入“艺妓”或预设的角色状态。
3.  **自然动作**：对着镜子，边用毛巾擦拭头发，边加入练习过的胯部摆动。可以随手拿起梳妆台上的抓夹，在摆动中自然地将头发盘起。
4.  **道具衔接**：拿起准备好的扇子或替代道具，跟随音乐节奏，将眼神、站姿、胯部与手部动作串联起来。
5.  **即兴发挥**：记住核心动作组合，不必死记硬背顺序。根据音乐和当下感觉，重复和组合这些基础动作。

**核心要义**：自然与沉浸。你的目的是享受这个展示自我的过程，而不是完成一套标准体操。对方感受到的是你散发出的自信和魅力，而非动作的难度。
