## 秘籍一：学会真实夸奖

真诚的赞美能迅速拉近人与人之间的距离。试想，你身边有两个人：一个总能发现你的优点并给予肯定，另一个则总是挑剔你的缺点。你自然会更亲近前者。当然，我们并非提倡毫无原则的奉承。

真实的夸奖才能打动人心。这里提供一个有效的夸奖公式：**感受 + 事实 + 对比**。运用这个公式，可以减少目的性，让赞美显得更自然。

例如，当女性换了一个新发型时。
*   如果你只是说“真好看”，这只传递了一个宽泛的感受，难以在她心中激起波澜。
*   但如果你这样说：“这个新发型真适合你（感受）。它很好地修饰了你的脸型（事实）。我见过别人剪类似的发型效果一般，没想到被你驾驭得这么好（对比）。”这样的夸奖就具体、真诚得多。

部分男性可能会担心：这样夸奖会不会显得自己姿态很低？关键在于心态。夸奖女性本质上是欣赏她，不要功利地认为“我是为了追求她才这么做”。怀着“我只是想让你开心”的心态去赞美，对方也会越来越喜欢与你相处。

在两性关系中，你的竞争力很大程度上取决于你能带给对方怎样的情感体验。想让女性主动，就要让她感到与你在一起是轻松、愉快的。

## 秘籍二：学会主动挑战

在感情中主动争取是值得鼓励的，但部分男性容易陷入过度付出、失去原则的误区。主动不等于廉价。**主动是我愿意给予，而廉价是你索取我就给**。前者有框架，能产生吸引力；后者无底线，往往只会收获“好人卡”。

请记住，女性是被吸引来的。除了对她好，持续的、势均力敌的“挑战”更能激发她的兴趣。

举个例子，当你追求的女生问你：“我和某某某谁更漂亮？”常见的回答是“当然是你最漂亮”。但这种答案往往在对方预料之中，缺乏新鲜感。

更好的回答可以是：“当然是我喜欢的那个更漂亮。”这个回答巧妙地将问题抛回给对方，能瞬间引发她的脸红与思考，制造了情绪波动。

建立挑战关系的前提，是避免无底线的付出。我们要懂得给予，也要懂得合理索取。

例如，女生几次约饭都希望你去她家附近，但你们距离较远。你可以提议选择一个折中的地点。如果她坚持要你去她那边，你可以说：“要不你给我讲个笑话吧，如果我笑了就马上过来。”这是一个很小的要求，如果她答应，互动就增加了趣味性；如果她不答应，你可以退一步，让她在点餐时多点一些你喜欢的菜作为补偿。

核心在于引导对方也为这段关系付出。健康的关系永远是双向的付出与收获，而非单方面的奉献。

爱情有时像一场博弈，真心是筹码，但懂得控制场面才是关键。追求方往往处于心理劣势。有些人在求爱失败后会放弃，有些人则会越挫越勇。后者看似痴情，但若不加控制，不断追加投入后，让你无法自拔的可能不再是爱情，而是不甘心。
