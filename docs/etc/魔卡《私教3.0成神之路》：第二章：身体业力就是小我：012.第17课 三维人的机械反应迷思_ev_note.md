# 魔卡《私教3.0成神之路》：第二章：身体业力就是小我：012.第17课 三维人的机械反应迷思 🎬

## 概述
在本节课中，我们将要学习三维世界中人的机械反应模式。我们将回顾情绪如何像滤镜一样扭曲现实，并揭示冲突产生的内在闭环机制。最后，我们会探讨如何通过觉察打破这种机械循环，从而改写命运。

## 课程内容

前两节课我们介绍了多巴胺、催产素、内啡肽和血清素，这些是小我决定我们日常感受的核心化学物质。现在，我们已经对此有了理性的认知。

在更早的课程中，我们也对小我的情绪表达及其意义和作用，进行了系统性的了解。本节中，我们来回顾一下原有的生活。

绝大多数事情都坏在了情绪上。情绪一来，便排山倒海，使我们无法理性、专注地处理眼前事务。

情绪就像一个有色滤镜，结合了我们曾经的记忆和经历。当这几样东西混合在一起，便在我们眼前蒙上了一层厚厚的浓雾。它扭曲了我们当下看到的事实，遮蔽了我们的理性视觉。我们的行动开始偏离初衷。

在情绪的驱动下，我们早已不知被带往何处。

这里摩卡要提醒大家，所有三维世界的冲突之所以成为冲突，原因在于：当你的内在情绪涌起、提示出现时，你没有识别到小我的语言。你迷失在了外界三维世界具体的人和事的表象之中。

以下是冲突形成的闭环过程：
1.  冲突首先在你内心形成。
2.  冲突被投射到外部的三维世界。
3.  那些NPC（外界的人与事）开始配合你的投射，执行你投射中所需的角色指令。
4.  冲突再被你感觉到，反噬回你的内心。

这是一个速度惊人的闭环，你根本反应不过来。

如果你跟随摩卡在3.0课程中进行提升与学习，这个连贯且快速转动的命运闭环，其转速就会慢慢降下来。

这有点像放电影。我们觉得电影是连续生动的，那只是一种错觉。电影其实是一帧一帧的画面高速拼接而成，只是速度太快，欺骗了我们的眼睛。

又或者，我们坐地铁时，有时会在隧道里看到商家的广告。那些广告是画在隧道墙壁上的一张张静止画面。当地铁高速运动起来，那些静止画面就被我们看成了会动的广告，所以叫做“动画片”。

我们的人生游戏也是如此。每一个当下，其实都是打破原有机械命运循环的机会。

**关键在于：当你觉察身体、进入四维时，就等同于在播放页上按下了暂停键。改写命运的机会，便由此出现。**

但很可惜，所有三维人都在全力以赴地跟着这部三维动画往前狂奔。他们一刻也不愿暂停，生怕跑慢一点就会失去更多，哪怕已意识到自己非常落后。

他们机械地认为，只有加快速度才有机会反超。于是，他们每天带着无比焦虑，疯狂地想做各种事、很多事，结果越做越焦虑，越来越无法集中注意力去做真正有用的事。

他们把自己活成了一只没头的苍蝇，在三维世界里到处碰壁，掉队越来越多。他们几乎没有真正有意识地做过任何对自己人生有建设性意义的事情，一直只是在原地进行梦游般的机械式反应。

渐渐地，他们觉得这就是自己的悲惨命运，然后便放弃了。

但事实根本就不是这样。

## 总结
本节课中，我们一起学习了三维人的机械反应迷思。我们认识到情绪如何制造冲突闭环，并理解了人生如同高速动画，每一个当下都是按下暂停键、改写命运的机会。打破机械循环的关键，在于从觉察身体和进入更高维度开始。
