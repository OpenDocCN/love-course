## 第二步：如何赞美——三个实用技巧

### 技巧一：FFC法则

FFC法则包含三个元素：**感受（Feeling）**、**事实（Fact）** 和 **对比（Compare）**。这三个元素组合起来会让你的夸赞听起来真实可信。

**公式：** **有效赞美 = 感受 + 事实 + 对比**

**示例：**
你想夸赞女生的外套。
*   **苍白赞美：** “你这外套不错啊。”（只有感受）
*   **运用FFC法则：** “你这外套不错啊（**感受**）。最近天气又冷还刮风，这外套不仅保暖还挺好看（**事实**）。比起街上其他女生穿的羽绒服，保暖是保暖了，但一点也不好看，还是你身上这套有一种独特的气质（**对比**）。”

这样的赞美更加具体、真实，容易触动女生的内心。

细节赞美不仅仅是赞美对方的一个小细节，更高级的方式是：**细节 + 这个细节产生的影响**，再结合恰当的时机。

**公式：** **高级细节赞美 = 具体细节 + 细节产生的影响/感受**

**示例：**
*   **初识期：** 可以关注她的衣着或配饰。“你的围巾挺好看的（**细节**）。刚才离你还有段距离，就让我一下子注意到你了（**影响**）。”这能减轻初识的尴尬。
*   **熟悉期：** 可以关注她的独特风格或气质。“跟你在一起时，我总是有一种如沐春风的感觉，让人很舒适（**细节+感受**）。”这样的赞美胜过一万句“你好温柔”。
