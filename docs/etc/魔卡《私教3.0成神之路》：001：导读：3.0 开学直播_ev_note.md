## 3.0课程的核心本质

上一节我们了解了学员的多样性，本节中我们来看看3.0课程的核心定位。

**3.0课程与1.0、2.0有本质区别。**
*   **1.0与2.0** 专注于**个人魅力与两性吸引力**的方法论。
*   **3.0** 则专注于**个人生命力与创造力**的激发。

你可以将个人魅力理解为生命力的一种外在表现。一个缺乏生命力的人，很难拥有真正的吸引力。

**3.0课程的核心目标是引导你的思维从三维世界升维到四维世界。** 目前你可能对此感到陌生，这很正常。随着课程推进，你将逐渐理解并体验到思维升维带来的巨大价值——那种做事如有神助、结果截然不同的感觉。

我（摩卡）的个人目标，是用**6个月**的时间，帮助大家建立起完整的四维思维框架体系。这套系统无论应用于事业、职场、社交还是情场，其威力都是“核弹级别”的。现在你或许将信将疑，这完全可以接受，我们可以一起向前验证。

## 课程设置与学习方式

明确了课程的核心后，我们来看看具体如何学习。

课程设置旨在减轻大家的学习压力，便于长期坚持。以下是具体安排：

**每周学习流程：**
1.  **每日知识点（课件）：** 每周会发布7个约5分钟的知识点视频，每天学习一个。你可以在通勤、休息等碎片时间轻松完成。
2.  **每周直播答疑：** 每周二和周四晚8点，各有一次45分钟的直播（共90分钟）。**直播主要用于答疑，而非讲授新课。** 答疑内容围绕当周7个知识点的理解与应用展开，旨在校正你的思维，确保学习方向正确。

这种“轻量每日学习+深度每周答疑”的模式，能让你在不过多影响日常工作生活的前提下，持续进步，最终将知识内化。

## 常规作业与核心原则

了解了学什么和怎么学之后，我们来看看需要配合完成哪些日常练习。

以下是常规作业，请根据自身时间、精力和条件，**任选其中一项或多项**执行，关键是**每日坚持**且**不构成心理压力**。

**常规作业清单：**
*   **日记：** 睡前用5-10分钟记录，内容不限，关键是每日进行。
*   **冥想：** 时间长短自定（如5、10、15分钟），以无压力为准。
*   **瑜伽：** 可线上跟练或线下报班，量力而行。
*   **健身：** 根据个人日常节奏安排。

**作业两大核心原则：**
1.  **不能变成压力：** 一旦感到压力，立即调整（如减少时间或项目）。
2.  **重在每日坚持：** 不追求时长与质量，哪怕每天只做一分钟，持续进行也比中断更强。

如果四项全做，每日约需3小时，请务必科学安排，避免虎头蛇尾。

此外，每日知识点视频下方设有作业提交端口，同样遵循“无压力”原则，可长可短。
