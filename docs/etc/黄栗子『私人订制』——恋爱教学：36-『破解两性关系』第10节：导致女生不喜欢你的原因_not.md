## 一、恋商过低：不懂建立亲密关系 🧠

上一节我们提到了“恋商”这个概念。所谓**恋商**，是指一个人与异性建立并保持亲密关系的能力。在我们的教育体系中，并不注重恋商的提升。很多人的爱情启蒙只能依靠影视剧或各类小说故事，这也导致他们存在认知偏差，对真实世界的感情所知甚少。

下面栗子就具体说一说恋商低的几种表现。

### 1. 过分暴露需求感

需求感是指你让一个人感受到被需要的程度。而过度的需求感则会让人感到压力。很多男同胞在追求爱情时，会在对方还没有对自己产生好感前，就表达出自己强烈的需求感。例如嘘寒问暖、频繁送礼、没事找对方聊天，甚至认识没两天就告白。

要知道，喜欢并接受一个人的前提是能够被他所吸引。在还没有吸引到对方时，就让对方接受自己，这显然是不现实的。因此，在未确定吸引之前，必须控制好自己的需求感。

这里我们可以运用**镜像法则**，也就是对方怎么对待我们，我们也以相似的频率和方式回应他。

**代码示例（互动频率）：**
```python
# 假设女生回复消息的间隔时间较长
her_reply_interval = “long”

# 应用镜像法则：保持相似的互动频率
my_reply_interval = her_reply_interval

if her_reply_interval == “long”:
    # 先忙自己的事，过段时间再回复
    focus_on_my_task()
    reply_after_a_while()
```
保持与对方相同的频率互动，就不会显得需求感过强。

### 2. 不懂换位思考

男女之间存在很大的思维差异。男人偏理性，女人偏感性；男人喜欢用逻辑解决问题，女人喜欢用情绪表达感受。这种本质上的不同，会让很多男同胞在跟女生聊天时出现沟通不畅的问题。

举个例子，女生抱怨：“我最近工作太累了，天天加班，老板还不给加工资。” 男生的理性回应可能是：“那你给老板反映一下，让他多招几个人或者给你加薪。” 然而，女生这样说的目的常常是为了发泄情绪，得到安慰。男生的应对方式是在帮她解决问题，双方不在一个频道，女生自然会觉得“他不懂我”。

因此，异性之间想要相处融洽，就得多培养共情能力，学会换位思考。接上面的例子，男生其实可以回应情绪：“你们老板也太不人道了，天天让加班，要是我可能也受不了。” 先回应对方的情绪，女生就会觉得你很懂她。如果想给出建议，最好等到女生情绪平复之后。

### 3. 没有框架

有些男同胞对爱的认知就是“对她好”，认为自己只要迁就对方、想尽办法对对方好，就能证明自己很爱她。确实，爱一个人，对他好一点没有错，但这不意味着要降低底线去迁就对方。

记住一个道理：**你的一退再退，永远满足不了他的得寸进尺**。你这次允许他约会迟到，那之后约会他就可能都迟到，甚至直接放你鸽子。你这次答应无条件借钱给他，下一次他就可能敢直接找你要钱。

所以说，男人在感情中一定要保持好自身框架。那要如何保持框架呢？栗子希望大家记住以下三点原则：

以下是保持框架的三点核心原则：
1.  **尊重对方，但同时不要贬低自己。**
2.  **可以付出，但对方必须给予回馈。**
3.  **可以退步，但不能改变已有底线。**

如果你在与他相处过程中发现自己打破了其中某条原则，那就该意识到自己的求爱方式存在问题了。丢失框架的男人只会让女人有恃无恐，越来越不知满足。如此一来，在这段关系中你就处在了低位，或许对方会被你感动，但这种感动绝不是爱情。

好，恋商低的情况就讲完了。它无关乎你的个人价值，但你必须相信，它能够影响你的整个人生。好在恋商是能够通过后天培养提升的。努力成为一个恋商高的人，不仅能收获美满的爱情，更能在人际交往过程中如鱼得水。

## 二、个人价值低：缺乏吸引力 💎

说完了恋商，接下来我们看看因个人价值低所导致的求爱失败情况。毋庸置疑，当你的个人价值太低时，就很难对人产生吸引。除了个人财富和外形条件这两类最直白的价值体现外，还有几类是我们不常注意的。

### 1. 潜在价值不足

作为男人，展现潜在价值特别重要。所谓**潜在价值**，是指未来可能发挥出来的价值。比如你现在虽然财富积累比较少，但正在努力打拼事业，这就属于有潜在价值。大多数女生并不介意男生年轻时没有巨大成就，但如果他安于现状、不思进取，这就属于没有潜力，会让人瞧不上。

### 2. 内在价值欠缺

一个人的特长、各项技能、人格魅力都属于内在价值。因此，平时多培养一些兴趣爱好，掌握一技之长，就能成为一个很有价值的人。除此之外，也得注重自身人格魅力的塑造。比如你虽然不善交际，但为人谦和、做事踏实务实，这就属于你的人格魅力。

可能有些男同胞会认为，与女生相处时表现出绅士风度，能显得更具人格魅力。这种想法没错，但不能把绅士风度跟没有主见搞混了。你约会点菜让女生拿主意，吃完去哪儿也让女生做决定，这看似是在尊重女生意见，实则是种没有主见的表现。

其实最能吸引人的魅力叫做**领袖气质**。你能在亲密关系中带领女生，就是对他最大的吸引。但也要注意，领袖气质不等于大男子主义。与女生相处时，不要总是固执己见。在让对方信服你的情况下领导对方，这才叫真正的领袖气质。

那么如何拿捏和女生交往时的尺度，既显得自己有主见，又不会让对方觉得太大男子主义呢？很简单，男同胞们可以在做一件事情之前，先准备好完整的计划，在具体实施的时候再询问一下对方的意见即可。

**举例：**
你计划与他约会，流程是先吃饭再逛街，晚上8点再和朋友一起吃宵夜。那么吃完饭后，你就可以对女生说：“等会我们去逛逛街。8点我跟朋友约了一起吃宵夜，你没什么事的话就一起去吧？” 这样的询问既告知了你的安排，也尊重了女生的意见。如果女生不愿意吃宵夜，那你直接送她回去即可。

### 3. 负能量爆棚

最后，栗子再说一种会有损个人价值的东西，那就是**负能量**。某些人平日里就特别爱抱怨，今天手机被偷，明天工作不顺心，后天生病挂点滴，从他嘴里听不到什么好消息。不可否认，人都有负面情绪，但如果总是向旁人暴露自己消极负能量的一面，势必会给人造成压力。

偶尔一两次抱怨，对方可能会安慰你；但如果总是对外散发负能量，那肯定会让人躲得远远的。就算你现在腰缠万贯，别人也会怕你明天就破产。作为成年人，调节自我情绪是基本能力。既然选择生活在这个社会上，肯定得认可社会的主流价值观。悲观厌世解决不了问题。如果无法自我调节情绪，可以寻求专业心理咨询师的帮助。总之，不要让自己处在负能量爆棚的状态。
