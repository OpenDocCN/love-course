# 魔卡《私教3.0成神之路》：第三章：第七感莫那识觉察力：007.第25课 觉察力的练习要义 📝

在本节课中，我们将要学习在初步练习第七感觉察力时，可能遇到的几种常见情况及应对要义。我们将重点探讨如何通过“双向觉察”这一核心方法，将意识稳定地锚定在当下，从而有效管理注意力与能量。

上一节我们介绍了如何开启觉察力以及“双向觉察”的实例。本节中，我们来看看在刚开始练习时，会有哪些常见的情况。

## 常见情况一：注意力完全被事情占据

我们很容易完全把注意力放在事情上面。

我们会很自然地认为，所谓的全神贯注、聚精会神、心流体验，就是百分之百的注意力全部放在我们所做的事情上面。大家要注意，这是一个结果，而不是原因。这种情况往往不受我们控制。

它有时会莫名其妙地自己出现，但也可能很久不再出现一次。你完全控制不了它何时来、何时再来。这样一来，有意识的训练就无从下手。而且，聚精会神本就不是一件容易的事情。

我们经常开小差、神游、胡思乱想。做着眼前的事，但注意力早已变成想象力或担忧焦虑，跑到别的人、别的事上面去了。你根本意识不到，也察觉不到。往往过了很久之后，你才反应过来，然后开始自责。

以下是“双向觉察”作为解决方案的核心描述：

*   **核心概念**：双向觉察是一个可操作的、可练习的、可以意识化的第七感练习窍门。
*   **运作机制**：通过同时意识到**所做的事情**（A点），以及**做事的自己**（B点），就像同时拉紧了绳子的两端。
*   **效果显现**：第七感的“觉察之神”就在我们的意识里显化了出来。仿佛它能够被你感觉到。一旦哪一端松掉了，不管是事情端还是自我端，一旦“觉察之神”掉下来，你的意识会很快察觉到，然后重新将它接起，重新进入觉察状态。

这样一来，你的意识能量就完全处于你的有序管理之下了。那种踏踏实实的感觉，那种和自己待在一起的感觉，是非常真实和美好的。

## 常见情况二：意识分配不均

在开始练习双向觉察的时候，你会发现要么意识在自己身上，忽略掉了做的事情；要么意识在做的事情上面，忽略掉了自己。好像不那么容易做到将意识分配在两个点上。

不用担心，这是正常的。慢慢地练习熟练之后就好了。但是，这里有一件事情已经在这个过程中神奇地发生了：你已经很容易地待在当下了，你不走神了。你的注意力不会漫无边际地胡思乱想、到处神游、心不在焉了。

光这一点，对于绝大多数普通人来说，已经是意识形态、生活状态、能量磁场上的一个巨大飞跃了。

## 给冥想练习者的建议

有开始冥想练习的同学，可以更好更快地发展双向觉察的第七感能力。

你们可以像冥想时一样，把B点的注意力放在自己的呼吸上面，把A点的注意力放在你眼下正在做的事情上面就可以了。一边关注着自己的呼吸，觉察着自己的存在；一边关注着所做的事情，觉察着事情的进展。“觉察之神”拉满，注意力高度集中。

在日常生活当中，无时无刻的任何场景，做任何事情的时候，你都可以开展双向觉察的能力练习。一段时间之后，你一定会庆幸自己发展了这项精神力。

## 练习心态与节奏

刚开始练习的时候，想到即可开始练习。但是很快，“觉察之神”可能会掉落甚至消失。

这都没有关系。接受自己目前的意识能力还处于一个较低的发展水平这个事实。不要操之过急，不要拔苗助长，不要去责怪自己，更不要进行自我攻击。这些行为都会让你的能量调频愈加负面。

以下是具体的练习心态指导：

*   **接纳当下**：几秒钟有觉察就几秒钟，几分钟有觉察就几分钟。给自己一点时间，慢慢来比较快。
*   **随时练习**：想起来就马上用眼前的事情开始练习。练一会儿是一会儿，能持续多长就多长。
*   **保持耐心**：绳子掉了或者绳子没了都没关系，下次想起来再来。慢慢地，你就可以从一天几秒钟，到几分钟，到几十分钟，甚至几个小时。

我期待那一天尽快在你的身上到来。因为那一天到来的时候，你已经基本走完成神之路了。
