# 📸 私教课 58：拍照与修图基础教程

在本节课中，我们将学习如何拍摄一张构图得当的照片，并使用后期修图软件进行美化。课程内容涵盖从拍摄姿势、构图技巧到使用美图秀秀和VSCO进行人像精修与色调调整的全过程。

## 一、拍摄构图技巧

上一节我们介绍了课程概述，本节中我们来看看具体的拍摄构图方法。构图的核心是让人物在画面中处于合适的位置。

**核心构图公式**：`人物中轴线 ≈ 画面中心线`

以下是具体的拍摄步骤与要点：

1.  **人物位置**：人物在相框中的占比约为三分之一。拍摄者需要蹲下，并将手机侧倾一定角度。
2.  **手机角度**：手机不能过于端正，需要倾斜一点。也可以尝试将手机倒过来拍摄，这样会显得人物更高。
3.  **画面平衡**：需要注意人物与背景景物的平衡。例如，如果左侧墙壁过多会影响美观，但墙壁太少又会导致人物不在画面中心。
4.  **姿势指导**：被拍摄者的肩膀需要放松并放平，不要耸肩。手臂不要架着，保持自然下垂。
5.  **眼神与互动**：被拍摄者不要直视镜头，可以自然地看着手机或手中的物品（如戒指），营造随意感。
6.  **利用九宫格**：使用手机的九宫格辅助线，让人物占据其中两格即可。
7.  **场景选择**：可以选择五星级酒店大堂、商场等公共空间作为拍摄地点，这些地方通常环境不错且免费。

## 二、人像精修（美图秀秀）

掌握了基本拍摄后，我们进入修图环节。首先使用美图秀秀进行人像精修，主要调整面部和身体轮廓。

以下是使用“瘦脸瘦身”功能时的核心调整部位：

1.  **面部轮廓**：推平脸上不平整或凸起的地方，如下巴、颧骨。如果鼻子较塌，可以适当挺立。
2.  **头发**：如果头发不够蓬松，可以向上推拉发顶，使其显得更饱满。鬓角可以向下拉一点，增加成熟感。
3.  **脖子与肩膀**：如果脖子较粗或肩膀线条不好看，可以适当修瘦。对于太瘦的人，反而需要将胳膊等部位推粗一点，使其看起来协调。
4.  **身体与腿**：腿太细可以适当加粗。使用“增高塑形”功能不仅可以拉长腿部，还可以拉长上半身和脖子，前者能增加成熟感，后者能提升气质。
5.  **细节处理**：使用“去皱”功能淡化法令纹和黑眼圈。使用“祛痘”功能去除痘印，让脸部更干净。

**代码示例（修图逻辑）**：
```python
if 脸部不饱满:
    推平凹陷处
if 头发扁塌:
    向上推拉发顶
if 脖子显短:
    使用“增高塑形”拉长脖子
if 腿过细:
    适当加粗腿部轮廓
```

## 三、色调调整（VSCO）

人像精修完成后，我们使用VSCO进行色调调整，赋予照片统一的风格和高级感。

以下是VSCO调色的核心参数与步骤：

1.  **曝光**：增加画面整体亮度。根据原图明暗度决定是否调整。
2.  **对比度**：增加对比度可以让雾蒙蒙的照片更清晰。但不宜过度。
3.  **锐化**：这是必调项，能显著增加照片的质感和清晰度。通常可以调至较高数值。
4.  **饱和度**：**降低饱和度是营造高级感的关键**。高饱和度显得土气，低饱和度则显得冷静、高级。
5.  **白平衡**：调整色温。偏冷（蓝色）色调显得高冷，偏暖（黄色）色调则有复古或温馨感。
6.  **色调分离**：这是最重要的滤镜功能。推荐将“阴影”色调设置为蓝色，这能决定你照片的整体风格。“高光”色调可设为黄色或蓝色，根据照片氛围选择。
7.  **HSL（色调）**：可以单独调整画面中某种颜色的色度、饱和度和亮度。例如，让背景中的黄色更亮。

**调色公式**：
`高级感 ≈ 低饱和度 + 冷色调（白平衡）+ 蓝色阴影（色调分离）+ 高锐化`

## 四、配图处理

主图修好后，也需要对同一组照片中的配图（场景图、细节图）进行调色，以保持朋友圈或相册风格的统一。

配图处理非常简单：

1.  直接使用VSCO，套用与主图人物相似的调色参数。
2.  重点调整**锐化**和**饱和度**，确保画质清晰且色调统一。
3.  无需使用美图秀秀进行精修。
4.  不要添加边框，以保持简洁。

## 总结

本节课中我们一起学习了从拍摄到成片的完整流程。我们首先探讨了如何通过蹲拍、侧拍和利用九宫格来获得更好的构图。接着，我们使用美图秀秀的“瘦脸瘦身”等功能对人像的面部、身体轮廓进行精细化调整。最后，我们通过VSCO软件，通过降低饱和度、调整白平衡和使用色调分离滤镜，为照片赋予了统一、高级的色调风格。记住，构图是基础，精修优化细节，色调决定风格，三者结合才能产出优质照片。
