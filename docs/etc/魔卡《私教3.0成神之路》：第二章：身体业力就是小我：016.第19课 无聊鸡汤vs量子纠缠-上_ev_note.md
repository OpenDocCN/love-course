## 量子纠缠：一个科学的入口

今年以来，“量子纠缠”一词在网络上非常流行。接下来，我们通过一段视频的解释来理解它。

以下是视频内容的转述：

我将用我的名片来讲解量子纠缠。我的名片有两面：中文面和英文面。当我把中文面翻到背面时，英文面就同时翻到了正面。反之亦然。这没有问题。

关键在于：当中文面翻到背面时，是中文面先动，还是英文面先翻到正面？你会回答是“同时的”，因为它们本就是一张名片的两面。这类似于中国哲学中的“阴阳”：阳去便是阴生，它们本是一体。

神奇的事情即将发生。假设我有两张名片，左手一张，右手一张。我把左手名片的中文面翻到背面，右手名片的中文面会同时翻过去吗？不会，因为它们是两张独立的名片。

现在，把这两张名片紧紧贴在一起。当我把前面名片的中文面翻到背面时，后面名片的中文面也会到背面，因为它们靠得特别近。

再把它们分开。此时，翻转左手名片，右手名片不会动，因为有了距离。当然，我可以先看到左手状态，再指挥右手模仿，但这需要时间传递信息。

然而，假设这两张名片是两个处于“纠缠态”的物理粒子。当把它们分开后，改变左边名片的状态，右边名片的状态也会**同时**改变。这就是量子纠缠。

为什么？玻尔的解释是：别看这两个粒子（名片）分得很远，它们依然是一个整体。分开很远与紧紧贴在一起是一回事。所以改变一边，另一边同时改变。

量子纠缠挑战了我们的常识：
1.  **挑战一（整体性）**：两个粒子在空间上分离，为何还是一个整体？
2.  **挑战二（超距作用）**：实验中，两个粒子悬空，无任何操纵。左边改变，右边同时改变，这意味着可能存在超光速的信号传递，这与爱因斯坦相对论中“光速最快”的定律相悖。

总结现状：两个处于纠缠态的粒子，一个状态改变，另一个同时改变。要么接受玻尔的解释（它们无论多远都是一个整体），要么接受存在超光速的信息传递（违背相对论）。当今主流科学支持玻尔的解释。
