## 什么是情感？💡

情感是人类对外部刺激和内部状态产生的复杂心理与生理反应。它不仅仅是感觉，更包含认知评价和生理唤醒。

上一节我们介绍了课程主题，本节中我们来看看情感的具体定义与构成。

情感由三个核心部分组成：
*   **主观体验**：个人内心的感受，如快乐、悲伤。
*   **生理反应**：身体伴随情感发生的变化，如心跳加速、面部潮红。
*   **行为表达**：情感的外在表现，如微笑、哭泣。

## 情感的核心功能 🛠️

情感并非无用的副产品，它在生存和社会交往中扮演着关键角色。

以下是情感的几个主要功能：

*   **适应功能**：情感帮助机体适应环境。例如，恐惧促使我们避开危险。
*   **动机功能**：情感能激发行为。兴趣驱动探索，爱驱动关怀。
*   **组织功能**：情感影响认知过程。积极情感拓宽思维，消极情感窄化注意力。
*   **信号功能**：情感通过表情、语调传递信息，促进社会沟通。

## 如何学习与运用情感？🎯

理解情感后，我们需要掌握识别与运用情感的基本方法。

以下是几个关键步骤：

1.  **自我觉察**：定期内省，识别并命名自己的情绪。可以使用“情绪日记”记录。
2.  **观察他人**：注意他人的非语言信号，如微表情、肢体语言和语调变化。
3.  **共情练习**：尝试从对方视角理解其感受。公式可简化为：**共情 = 认知理解 + 情感共鸣**。
4.  **适度表达**：根据情境和对象，选择合适的方-式表达情感。例如，在代码中，一个条件判断可以模拟情感反应逻辑：

    ```python
        expression = "温和鼓励"
    elif situation == "出现冲突":
        expression = "冷静陈述"
    ```
