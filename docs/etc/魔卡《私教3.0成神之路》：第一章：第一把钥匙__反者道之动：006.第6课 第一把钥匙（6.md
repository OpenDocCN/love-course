# 魔卡《私教3.0成神之路》：第一章：第一把钥匙__反者道之动：006. 第6课 第一把钥匙（6_6）玩家与角色 👤🎮

在本节课中，我们将要学习一个核心概念：区分“玩家”与“角色”。这是理解我们如何从人生游戏的困境中解脱出来的关键一步。我们将通过游戏比喻来阐明这个概念，并探讨它如何影响我们的情绪、决策和内在能量。

上一节我们介绍了要从游戏内部跳脱到游戏外部。本节中我们来看看，在人生这场游戏中，“外部”究竟对应着什么。

我们大家都玩过游戏。回想我们玩过的所有游戏，无论是电子游戏还是童年时的过家家，我们首先做的第一件事就是选择一个游戏里的英雄人物。

例如，摩卡小时候在街头的电子游戏厅喜欢玩《三国志》，他会选择赵云这个角色。和小朋友玩过家家时，他喜欢扮演带刀侍卫，而不是国王。后来玩《星际争霸》时喜欢用神族，玩《帝国时代》时喜欢选蒙古。

在这些例子中，游戏里的英雄是一个“角色”，而玩游戏的我们是一个“玩家”。游戏里的角色会根据剧情一路向前拼杀。我们作为游戏之外的玩家，也会跟随这个角色和剧情获得游戏体验。

但我们都知道，角色是角色，我们是我们。我们作为玩家不会深度认同这个角色。

我们不会觉得赵云在游戏里吃了一个包子，我们自己的肚子就饱了。我们只会知道“我的这个角色赵云补血了，他更扛打了”。如果赵云被大Boss击败，我们可能会气馁，然后重新开一局，或者干脆去做别的事。我们不会觉得自己也死了。

那么在人生这场游戏当中，我们选择的这个英雄角色其实就是我们自己。或高或矮，或胖或瘦，或富或贵，或贫或穷，都经历着跌宕起伏的人生剧本。

和我们玩游戏时唯一的差别就是：我们不再认为自己是“玩家”了。我们深度认同了“角色”。我们认为自己就是角色，角色在人生游戏里的经历，深度地影响着我们（玩家）的操控。

这是完全错误的。这也是厉害的人和普通的人，一个根本的心理分水岭。

我们眼中看到的那些各行各业真正厉害的人，他们一定都是情绪稳定、思路清晰、方法正确、内心强大、能量满满。

他们是怎么做到的呢？我们怎么就做不到呢？根本原因有两点。

以下是第一点原因：

**第一，他们很早就意识到了“角色是角色，玩家是玩家”。他们的意识牢牢地扎在“玩家”这个位置上，和“角色”保持着一定的心理距离。**

换句话说，他们在自己的心理意识，和他们在生活中遭遇的各种事情之间，有一段距离，有一个防火墙。所以很多生活中的打击，根本无法穿透这道防火墙去伤害他们的内心。

反观我们普通人，我们没有“玩家”的概念。我们只有“角色”的概念，或者说，我们的“玩家”和“角色”这两重意识是完全重叠的，中间没有距离，也没有防火墙。所以我们特别容易被生活中的情感、事业、学业等各种事情和不顺所打击，然后造成伤害。

以下是第二点原因：

**第二，他们的玩家意识和角色意识是合一的，没有内耗。想做什么就可以很顺利的做好。**

而我们普通人呢？“角色”和“玩家”意识总是对抗、内耗。最常见的就是大家都体会过的：很多时候我们的脑子里都有两个声音。

一个说去抽烟，一个说要戒烟。
一个说去旅游，一个说要工作。
一个说去写作业，一个说再玩一会儿吧。
一个说离婚，一个说凑合过吧。
一个说要努力，一个说还是躺平吧。

是不是很熟悉？所有的高手不内耗，普通人大量的内耗。高手的能量都用来做外部的创造，普通人的能量都在心里被自己内耗掉了。

那为什么会这样呢？心理学家弗洛伊德说“我”的构成有自我、本我和超我。道家说的是精、气和神。佛家说的是身、心和灵。

消除内耗，也就是摩卡要带大家进入四维世界、成为一个四维人要学习的第一课：**如何终结内耗？**

本节课中我们一起学习了“玩家”与“角色”的区分。我们明白了，将自我意识定位于“玩家”，并与“角色”保持心理距离，是情绪稳定、减少内耗、汇聚能量的关键。这为我们后续学习如何具体操作奠定了基础。
