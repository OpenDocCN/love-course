# 魔卡《私教3.0成神之路》：第二章：身体业力就是小我：009.第14课 小我是什么？_ev_note

在本节课中，我们将要学习“小我”的本质。我们将明确小我的定义、作用、语言方式及其力量的来源，帮助你建立对自身更深刻的理解。

上一节我们从四维世界介绍了小我情绪表达的价值。本节中，我们来看看小我究竟是什么。

首先揭晓谜底。你可以理解为小我就是我们的身体，即我们的物质身。人体解剖打开后能看到的一切，都是小我的构成部分。显微镜下的细胞、基因、神经元也是其构成部分。

肌肉、皮肤、骨骼、脏器，以及呼吸、消化、泌尿、血液循环、免疫等所有系统，都是小我的一部分。它们都是活的，全部拥有智能和记忆。因此，过去我们认为智能仅存在于大脑、想法和思考中，现在需要对人体智能的概念进行扩容。

**每个细胞都有智能，它们都会表达自己。** 它们表达想法的语言不是中文或任何人类语言，也没有声音和眼神。它们的语言就是你能感知到的**情绪和痛苦**。

每一种情绪和痛苦都有原因。真正的原因一定在你的身心内在，绝对不是你原来以为的外在三维世界的人和事。记住这一点，这是我们3.0课程非常重要的结论性核心知识。你将在生活中慢慢思考和验证它的正确性。

所以，当你再听到有人说小我如何不好、如何妨碍你、需要被消除和压制时，你就知道这种认知有问题。这种认知本身就是对自身某部分的不接纳和对抗。这种认知本身的振动频率是负面的，必然引起一系列能量内耗与负面后果。

接下来，我郑重给出关于小我的定义、作用、语言方式及其力量来源的说明。

以下是关于小我的几个核心要点：

1.  **小我是我们最亲密的伙伴。**
2.  **小我一直恪尽职守地提示你，你的人生主线任务是什么。**
3.  **小我提示你的语言是情绪和痛苦。**
4.  **小我的力量来自于全身每一个细胞的记忆库。** 这包括你从小到大所有经历留存下来的记忆，以及基因中来自整个家族谱系的记忆。这是相当于几千万GB信息文件的庞大隐形力量，是你区区**120比特/秒**算力的理性思维根本无法抗衡的。

所以请记住，永远不要和你的小我正面硬刚。请你看到他，读懂他，尊重他，爱护他。这样你才能在他的积极配合下，完成任何你想要达成的目标。

请允许我用一句改编的广告词来结束本课内容：**你的能量超乎你想象。** 当然，不是通过喝饮料，而是通过照顾好你的小我。

本节课中，我们一起学习了小我的本质。我们明确了小我就是我们的物质身体，它由所有活着的、拥有智能和记忆的细胞与系统构成。小我通过情绪和痛苦与我们沟通，其力量源于庞大的身心记忆库。理解并尊重小我，而非对抗它，是发挥自身潜能、完成人生目标的关键。
