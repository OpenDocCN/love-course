## 答疑环节与核心概念梳理

我之前讲过情绪的公式，不知道是否有人记得？情绪的公式是：**吸引 → 正极情绪 → 负极情绪 → 引导投资**。引导投资即指向长期关系。

*   做饭：“有空一块做饭。”
