# 15、《恋爱必修课》：Ｃ吸引升温篇０２聊天禁忌（上）_note

在本节课中，我们将要学习线上聊天中至关重要的吸引力法则。我们将探讨如何通过避免常见的沟通陷阱，来维持并提升你在对方眼中的吸引力。核心在于理解人性，并据此调整你的聊天方式。

## 概述：吸引力与人性博弈

上一节我们介绍了吸引力的重要性，本节中我们来看看如何在聊天中具体应用。很多人搞砸一段关系，仅仅是因为“正常发挥”。他们自然的语言习惯和面对喜欢的人时所呈现的状态，反而会降低吸引力。许多塑造吸引力的行为是反直觉的。

例如，你买衣服时讨价还价。如果与老板来回拉锯20分钟，最终以800元成交，你会感到满意。但如果老板直接同意你500元的报价，你反而会怀疑自己出价过高，感到不快。这就是人性：我们并非永远用绝对收益衡量事物，而是更在乎过程中的体验。

因此，基于人性的特点，保持有吸引力的沟通至关重要。在聊天中，存在几个必须避免的禁忌。

## 禁忌一：文字必须口语化

首先，我们要确保聊天文字的口语化，避免使用生硬的书面语。线上聊天多为文字沟通，但许多人会不自觉地使用正式语言，或因过于重视对方而导致文字死板。从事严肃工作的人也可能有此问题。

你需要明确，你们的目标是建立亲密关系。刻板、官方、客套的文字会让互动状态生硬，难以建立轻松感和亲密感。

以下是实现口语化的方法：

你可以想象对方就坐在身边，将你想说的话真实地说出来，然后把说出的内容打成文字发送。口语化不仅关乎措辞，也涉及句子结构。正常对话不会主谓宾齐全。

例如：
*   **口语化**：`“去哪了今天？”` -> `“人民公园逛了一圈。”`
*   **书面化**：`“你今天去哪里了？”` -> `“我今天去了人民公园。”`

在口语情境中，我们会默认沟通对象，因此常省略“你”、“我”等主语。

如果你不确定自己的文字是否口语化，可以尝试以下方法：

> 找一段与异性20句以上的聊天记录，请一位亲近且不会嘲笑你的朋友，一起朗读这段对话。如果感觉不尴尬、不滑稽，那就没有问题。

其次，注意标点符号的使用。不是完全不能用，而是避免每句话都使用标准的逗号、句号。至少不要频繁使用句号，逗号用于间隔尚可，也可以用空格代替。同时，减少使用问号、感叹号等过于醒目的符号。

例如，问`“今天去哪儿了？”`，直接说`“去哪了”`即可，无需加问号。若要表达疑问或感叹，可以用表情符号代替语气。

总之，口语化的表达能消除疏远感，让人感觉更亲近，这是关系发展的基础。

## 禁忌二：保持互动的一致性

上一节我们强调了口语化的重要性，本节中我们来看看互动节奏的掌控。第二个禁忌是：保持互动的频率、内容比例和能量层级与对方一致，避免需求感失衡。

具体而言：

**1. 频率一致**
如果对方回复很慢（十几二十分钟或几小时一次），说明对方优先处理自己的事务，并未处于集中精力与你互动的状态。此时，无论你的话术多精彩，都效果有限。你需要做的是也去忙自己的事，与对方保持“有一搭没一搭”的回复节奏。
反之，如果对方秒回，而你也有时间并希望关系积极发展，你也应该秒回，以提高互动热度。只有频率同步，对话的能量和话题深度才有可能推进，关系才可能更进一步。

**2. 内容比例持平**
理想的聊天内容比例是对方占六成，你占四成。因为男生理应更主动一些。如果你的聊天内容占比达到六成甚至以上，基本可以断定你的需求感过强。

**3. 能量层级同步**
互动能量从低到高依次为：**文字 < 图片 < 语音 < 短视频 < 音频通话 < 视频通话**。

在互动初期，建议尽量控制在前三种方式（文字、图片、语音）以内。约会一两次后，可以尝试语音电话。关系比较亲密后，再使用视频通话。切忌认识没多久就每晚进行“电话粥”，这会导致关系能量过早消耗殆尽，后续发展缺乏上升空间。

同时，如果你尝试提升能量层级时对方未接招，应立即退回原层级。
*   **例如**：你们在文字聊天，你发了一条语音，但对方仍用文字回复。那么你的下一条信息也应回归文字，不要出现对方发文字、你持续发语音的不协调情况。
*   **再如**：你发了一张照片作为话题，但如果后续聊天中对方从未分享过照片，你也应停止分享，回归纯文字聊天。

保持互动频率、内容和能量与对方一致，能有效控制你的需求感，避免显得过于主动。对女生而言，过于主动比男生更为不利，因为主动本就是男生该做的事。男生的“主动”与“过于主动”有区别，而女生在大多数情况下应处于“被动”，“主动”很容易就变成“过于主动”。过于主动意味着需求感爆炸，会显得你价值低、缺乏吸引力。

此外，女生的主动若拿捏不好，容易给男生留下轻浮的印象，从而削弱对方与你长期发展的意愿，只激起短期关系的兴趣。同时，男生的被动也会压抑其主动征服的天性。因此，不建议不擅长把握分寸的女生轻易尝试过于主动。

## 总结

本节课我们一起学习了线上聊天中两大核心禁忌：**避免使用非口语化的书面语**，以及**保持互动频率、内容与能量层级的一致性**。掌握这两点，能帮助你在聊天初期建立舒适感，有效控制需求感，为吸引力的塑造和关系的升温打下坚实基础。下节课，我们将继续探讨其他的聊天禁忌。
