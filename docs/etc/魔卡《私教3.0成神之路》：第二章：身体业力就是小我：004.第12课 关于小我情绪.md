# 魔卡《私教3.0成神之路》：第二章：身体业力就是小我：004.第12课 关于小我情绪

在本节课中，我们将要学习情绪在四维世界中的正确认知，并深入理解应对情绪问题的四个不同层次。我们将从最无效的方式开始，逐步探讨如何更健康地处理情绪。

上一节我们介绍了情绪需要被表达，而非情绪化地表达。本节中，我们来看看情绪在更高维度下的本质，并系统性地认识处理情绪的四个层面。

首先，我给出本课的重点：情绪问题应对的四个维度。

以下是四个应对层次的详细说明：

*   **最差的第四层：压抑层**。这种方式是委屈自己。它是最无效、最无意义，对人际关系和自我生命力伤害最大的一种方式。当然，它表面上看起来是最“和谐”的。

*   **较差的第三层：爆发层**。我们可以理解为向外攻击，把情绪的力量在短时间内猛烈地喷发出去。采用这种方式的人，当时往往有一种宣泄感、力量感和无畏感。但事后通常需要收拾残局，并付出较大的代价。

*   **较好的第二层：表达层**。使用这层方式的人已是人群中的少数。他们的内在与外在关系往往不错，自我与环境的天平相对平衡，能量和情感情绪是流通的。他们的长期关系通常处于一种真实的和谐状态，并且会随时间越来越和谐。

心理学家武志红老师曾说过：**亲密关系的前提不是“好”，而是“真实”**。如果我内心不满，但脸上堆笑、嘴上说好，这就是一种虚伪的“好”。我们以为这样关系就能融洽和谐，但慢慢地会发现，生活很快就会给我们上一课。

说到表达情绪，可能有些同学对这个行为感到陌生，仿佛从小到大都活在第三层和第四层里。那么，我来举个例子。

假设我正在给大家讲解情绪知识，讲了好几遍，同学们仍一脸困惑。这时，我心里产生了一些烦躁情绪。

**表达情绪**是这样的：“我现在感觉到自己心里有一些烦躁。我给大家很努力地讲解了半天，但好像大家还是不明白。我觉得自己有些没用，有一种失败感。我想先休息5分钟调整一下，我们再说后面的内容，好吗？”

这就叫表达情绪，也是良性真诚沟通的一种重要形式。大家发现了吗？它是**以一种没有情绪的方式，在表达情绪的内容**。

相反，我们大多数人日常的**情绪化表达**是这样的：“我都说了这么多遍了，你们怎么还是听不懂？不就是**表达情绪，不要情绪化表达**吗？有那么难理解吗？你们是傻吗？用点心不行吗？不讲了，下课！😡”

熟悉吗？生活中，我们遇到这样情绪化表达的机会太多了。情绪化表达通常带着攻击性，对关系具有破坏力。最致命的是，表达者往往自己不知道在做什么，因为他们已被情绪控制。而听者只会感到恐惧或反感。

为什么呢？因为负面情绪作为一种能量，从我的身体里，经由我的语言，流到了你的身体里，被你感知到。我内心对于自己教学无能的恐惧，对于学生不断用困惑的眼神和问题提醒我“看到自己教学无能”的反感……这是一个负能量的振动频率，由我发出，然后共振了你。

或者，我可以换一种更简单的方式解释：你可以把**语言信息**理解为一个低频震动，能量较弱；把**情绪信息**理解为一个高频震动，能量较强、比较有冲击力。当这两个信息同时发出时，通常双方都无法有效接收低频的语言信息，而只能接收到高频的情绪信息。这就是很多家庭所谓的“讲道理沟通”但效果很差的本质。因为他们双方都未察觉，自己说出的每一句话都在进行情绪宣泄。情绪的频**率完全屏蔽了语言内容的信息**，除了破坏关系，毫无意义。

那么，最后一层，也是最好的层次，就是**转换层**。这一层基本上只有四维人能够做到。具体如何操作以及有何好处，我们将在下节课揭晓答案。

**本课作业**：请大家结合第十一课和第十二课的内容，回想一个自己曾经印象深刻的情绪场景，并按照本课的表达层次进行分类，定义一下当时的你处于第几层。

本节课中，我们一起学习了情绪应对的四个层次：压抑、爆发、表达与转换。我们明白了情绪化表达的危害，并初步了解了健康表达情绪的方式。理解这些层次，是迈向情绪成熟和建立真实关系的重要一步。
