# 魔卡《私教3.0成神之路》：第二章：身体业力就是小我：001.第10课 小我和自我内耗会带来什么后果？ 😰

在本节课中，我们将要学习当“小我”与“自我”发生内耗时，会带来哪些具体的负面后果。我们将通过几个生动的例子来理解这种内耗如何影响我们的表现和幸福感，并学习正确的应对路径。

上一节我们介绍了“小我”、“自我”和“大我”这三个概念，也知道了“自我”和“小我”的算力特别悬殊。本节中，我们来看看当它们开始内耗，会产生什么结果。

## 内耗的典型场景 🎤

以下是两个常见的、因小我与自我内耗而导致表现失常的例子。

**场景一：在KTV遇到心仪对象**
本来你和熟悉的兄弟在KTV开心自在地唱歌，气氛很好。你唱得很投入，也很忘我。突然间，推门进来一个陌生的女人。再定睛一看，这个女人和你的梦中情人一样美丽。接下来，你无法像刚才一样忘我投入地唱歌了。你控制不住地开始紧张、别扭、不自在。你突然感觉不能做自己了。

于是你在心里拼命给自己打气，告诉自己不要慌，好好唱歌，好好表现，争取博得她的青睐。但是你发现，你越是这么说，你越唱不好。直到一个破音或者走调，你感觉后背一阵冷汗，心里咯噔一声“完了”。你回过头尴尬一笑，下意识地说：“这歌好久没唱了，今天发挥失常哈。”大家不以为然地敷衍性鼓鼓掌，而那位美女心里对你留下了一个负面印象。是你不会唱歌吗？是，又好像不是。

**场景二：登台演讲**
平时私下里你能说会道，和朋友们在路边撸串时指点江山的那个你仿佛消失了。取而代之的是紧张、流汗、忘词、手脚哆嗦、语无伦次。感觉自己就像一个傻逼一样在台上展览。是你不会表达吗？是，又好像不是。

## 内耗的原因：小我“跑”了 🏃‍♂️

原因其实很简单：**你的一部分离开了你，跑到大美女和台下的观众那里去了**。哪部分跑了呢？就是那个顽皮的“小我”。它带着**1500万比特**的算力离开了你，而你根本控制不住它。

也许描述让你脑补不出画面。那么接下来我们来看一个视频的比喻。

> 🎼 哎哎哎哎哎… 🎼 就是…哎… 🎼 呀要呀呀哎呀… 🎼 哎呦，我这个方向盘真回到你们手了。

在这个比喻中，“小我”很害怕、很紧张，它一直通过情绪和身体的反应告诉理性的“自我”：“我害怕，我紧张，我不舒服。”但是理性的“自我”没有听见，也没有看见，继续自顾自地“开车”，结果就变成了视频当中这样失控的局面。

## 错误路径：压抑与无视 ❌

绝大多数普通人没有“自我”和“小我”的明确意识。他们的行为路径都是**无视小我、背离小我、压抑小我**。表现出来就是内外不一致，感觉“自我”在极力想把“小我”封在里面，不让它出来。

但是内在的压力和情绪张力却越来越大，导致整个人根本无法聚焦眼前要做的事情，最后就崩盘了，以失败收场。其核心公式可以概括为：
**无视小我情绪 → 内在压力积累 → 表现失常/崩盘**

## 正确路径：看见与合一 ✅

那么正确的路径做法是什么呢？他们会**看见小我的情绪表达**，会首先去做**顺应小我**的事情，会去**安抚小我、释放小我**。然后内外就一致了起来，没有了内部越来越强的压力，就能够平静、理性地聚焦眼前的事情，依照自己的水平做到应有的水准。

例如，有的人当众说话时也紧张，但他会表达出来。他会说：“对不起大家，我现在真的很紧张。能不能请大家给我鼓鼓掌，给我一些鼓励，让我的状态更饱满呢？谢谢大家。”我们所有人都会鼓掌，我们不但不会看不起他，还会觉得这个人很真实、很真诚、不装逼。

其实他的这句表达，只是他内部“自我”和“小我”之间对话的一个外部显化。过程如下：
1.  **看见与接受**：当身体里的“小我”表现出紧张情绪时，他的“自我”看见了，并且接受了。
2.  **顺应与安抚**：他的“自我”没有去压抑和无视“小我”的情绪表达，而是去顺应了“小我”，安慰了“小我”。
3.  **释放与合一**：然后这个人的紧张情绪就消退了，内心没有了干扰，内外就一致了起来。

这个过程在里面完成后，显化到外面，就是我们听到的那段真诚的请求。这体现了人生的一个基本原则：**对自己诚实，尊重自己**。即：
**感觉紧张 → 看到紧张 → 承认紧张 → 表达紧张**
**感觉不愿意 → 看到不愿意 → 承认不愿意 → 表达不愿意**

这种“自我”与“小我”的**合一**，是我们整个生命力、创造力和幸福感的构成地基。

## 内耗的长期后果 🏚️

但很不幸的是，绝大多数人每天都在无数次的拆毁自己的生命力地基。
*   他们心里一万个不愿意，脸上却陪着笑脸，嘴上却说“好啊好啊”。
*   他们心里明明紧张得要死，脸上还要装出云淡风轻的样子，嘴里还说着“没事没事”。

他们以为别人看不出来，感觉不到。他们迷信自己“自我”理性的那**可怜巴巴的120比特**的算力。他们根本不知道，“小我”的**1500万比特**的算力，会从他们浑身上下的每一个毛孔向外界传递着**相反的信息**。

最后，他们日积月累地感觉到不堪重负的内耗，越来越体会不到幸福和快乐。而他们期待中的那些美好人际关系，就如没有地基的空中楼阁一样，从一开始就是一个梦幻泡影。

## 总结 📝

本节课中，我们一起学习了“小我”与“自我”内耗带来的后果。我们看到，当“自我”无视或压抑“小我”的情绪时，会导致内在压力剧增、表现失常。而正确的路径是看见、承认并顺应“小我”的情绪，达到内外合一，从而稳定发挥，并建立起幸福感的根基。
