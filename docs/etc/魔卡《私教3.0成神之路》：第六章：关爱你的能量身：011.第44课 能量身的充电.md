# 魔卡《私教3.0成神之路》：第六章：关爱你的能量身：011.第44课 能量身的充电-冥想的做法 🧘

在本节课中，我们将要学习如何正确地进行冥想，掌握从身体姿势到呼吸方法的各项核心要点，帮助你获得更深层次的冥想体验。

上一节我们介绍了冥想的诸多好处，本节中我们来看看具体实践的注意事项。

## 一、坐姿的选择 🪑

冥想时可以采用多种坐姿。以下是常见的几种盘腿方式：

*   **散盘**：最容易，但稳定性最差。
*   **单盘**：难度适中，稳定性较好。
*   **双盘**：最难，但稳定性最高。

你也可以选择站着、躺着或像平常一样坐着冥想，但这些姿势通常只能进入较浅的冥想层次，难以获得深度体验。如果你无法盘腿，甚至散盘也做不到，这可能是身体在提醒你需要通过站桩或练习瑜伽来疏通气血。一旦气血通畅，双盘会变得轻松。

## 二、脊柱的姿势 ⚡

保持脊柱正直是冥想姿势的核心。整条脊柱必须摆正、摆直，不能塌腰、驼背或耷拉着脑袋。

在保持脊柱正直的基础上，冥想时间应遵循身体感受，能坚持多久就多久，并自然延长。这一点类似于健身：姿势不对，努力白费。如果姿势在几分钟后就开始变形，那么后续的冥想时间意义不大。**冥想时间的长度应建立在姿势正确的前提下。**

你需要感受身体的信号。身体会用酸、麻、胀、痛等感觉告诉你何时该结束。如果无视这些信号，仅靠意志力强行坚持，反而会拉远自我与身体（小我）的距离。

如果感到疼痛但姿势未变形，你可以尝试与这种感觉共处。以下是具体做法：

1.  将注意力觉察到疼痛的位置。
2.  尝试与这种酸麻胀痛的体验待在一起。
3.  只要姿势不变形，就尽可能长时间地保持觉察、感受和陪伴，直至穿越这种感觉。

## 三、头部与舌位的细节 🧠

除了腿和脊柱，头部姿势也至关重要。你需要微收下巴，让鼻孔垂直于地面，这类似于拍照时摄影师常提醒的“收下巴”。同时，脊柱正直，下颌微收，嘴角自然上扬，双目自然下垂，找到自然放松的感觉。

当以上姿势习惯后，可以加上“舌抵上颚”，即冥想时将舌头抬起，轻轻贴住口腔上颚。这样做可以使任脉与督脉处于连通状态。随着时间的推移，口腔中会自然分泌出宝贵的津液（养生补品）。当感觉津液满溢时，可分三次缓慢咽下。

## 四、呼吸的方法 🌬️

冥想时应使用**腹式呼吸**。其要领是：吸气时腹部隆起，呼气时腹部收回。

大多数人的日常呼吸是较为短浅的胸式呼吸。而腹式呼吸气息更深、更足，可以气沉丹田，再上行至头顶，最后呼出体外。在冥想中，呼吸是需要特别留意和觉察的重点。

你可以将注意力放在“数息”上。具体方法是：缓慢、深长地吸气，再缓慢、深长地吐气。呼吸频率可以循序渐进地调整：

*   初始目标：每分钟 **6到5次** 呼吸。
*   进阶目标：每分钟 **4到3次** 呼吸。
*   高阶目标：每分钟 **2到1次** 呼吸。

**注意**：所有练习都应建立在身体机能自然配合的基础上，循序渐进，切忌急功近利。

当以上所有要点都能毫不费力地自然完成时，可以在每次吸气时轻轻做一个“提肛”动作（收紧肛门肌肉），并在每次呼气时，随着气息缓慢呼出而放松该处肌肉。

## 五、手部姿势与心态调整 🙏

手部可以自然放置，没有固定要求。以下是常见选项：

*   自然放在膝盖上。
*   双手叠加，放在盘起的腿上。
*   采用“握固”手势。
*   结特定的手印。

在冥想初期，手部姿势并不太重要。核心是确保身体姿势正确，并留意全身是否处于放松状态。很多人会不自觉地耸肩、塌腰或低头，这些都需要在过程中持续觉察和调整。

当姿势无误后，只需将注意力放在呼吸上即可。如果杂念纷飞，无需评判或攻击自己，只需温和地将注意力重新拉回到呼吸上。注意力可能会反复被带走，这很正常，只需反复将其拉回即可。如此往复练习，便能渐入佳境。

**最后强调**：冥想不是专注地思考，更不是胡思乱想，而是要让表层的意识思维逐渐静止，从而进入入定的可能。

本节课中我们一起学习了冥想的完整做法，从坐姿、脊柱、头部、呼吸到手位与心态。记住道家的一句口诀作为结尾：**识神不死，元神不生**。你可以试着去觉察疼痛的位置，把注意力放到那个地方，开始你的练习。
