# 课程08：速约五术 🎯 第一术：撒网术

在本节课中，我们将要学习“速约五术”中的核心基础——撒网术。这一术的核心目标是**高效收集潜在目标的联系方式**，为后续的深入互动打下基础。

## 概述：撒网术的核心逻辑

撒网术的核心思想是：在社交软件上的初期互动，其主要目的并非深入聊天，而是**以最快的速度获取对方的联系方式**（通常是微信）。这是因为在社交软件上建立的浅层连接，转移到微信后才意味着真正互动的开始。因此，提升效率的关键在于缩短“收号”过程。

上一节我们概述了速约五术的框架，本节中我们来看看第一术“撒网术”的具体操作理念。

## 核心理念：效率优先

在社交软件上，初始阶段的核心任务是**收号**。这意味着你使用这些软件的主要目的是将潜在目标添加到你的微信联系人中，构建你的“大鱼塘”，以便进行后续的维护和培养。

因此，若想提升整体效率，你必须**以最快的速度完成收号**。许多时候，在社交软件上花费大量时间聊天，但加到微信后，互动又需从头开始。这表明，在软件上过度聊天的作用有限。

我们的策略是：尽可能压缩在社交软件上的聊天过程，将核心动作聚焦于**快速获取联系方式**。这可以总结为一个简单的公式：

**核心目标 = 最小化软件聊天时间 + 最大化成功收号率**

以下是执行撒网术时需要明确的几个要点：

*   **目标明确**：每次互动的直接目的就是获得微信。
*   **过程精简**：聊天内容应服务于“建立基本信任”和“提出加微信请求”这两个节点。
*   **避免纠缠**：不在软件上展开长篇、深入的对话或解决复杂问题。

## 总结

本节课中我们一起学习了“撒网术”的基本原理。我们明确了在社交软件初期阶段，**“收号”是高于“聊天”的第一优先级任务**。通过将互动目标简化为快速获取联系方式，我们可以显著提升前期效率，为后续的“养鱼”和深入互动积累足够的潜在资源。

记住，撒网是为了捕捞，而不是在海中与每一条鱼嬉戏。高效地撒下你的网，才能为接下来的步骤储备充足的“鱼苗”。
