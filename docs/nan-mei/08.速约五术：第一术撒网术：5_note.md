# 08.速约五术：第一术撒网术：5_note

在本节课中，我们将要学习“撒网术”中关于展示面拍摄的核心转变。我们将重点理解为何“颜值”已取代“价值”成为展示面的首要因素，并掌握通过角度与后期处理提升照片吸引力的具体方法。

## 从“价值优先”到“颜值优先” 🔄

上一节我们介绍了展示面的基础概念，本节中我们来看看其核心标准的重大变化。过去，拍摄展示面时，“价值”是排在第一位的要素。但是，今后你们拍摄展示面时，“颜值”必须排在第一位。

## 提升照片颜值的具体方法 ✨

一个男生只要稍微化个妆，并找到自己最好的拍摄角度，就能显著提升照片效果。每个人一定存在一个最佳角度，找到那个角度进行拍摄，是成功的第一步。

以下是结合后期处理进一步优化照片的关键点：

*   **角度挖掘**：专注于寻找并利用个人最佳拍摄角度。
*   **后期处理**：拍摄后通过软件进行适当调整与优化。

经过角度选择与后期处理，最终呈现的照片基本都会非常帅。你们见过我也知道，我本人并不帅，但拍出来的照片就能给人感觉很帅。

## 照片为何如此关键？ 💡

因此，在照片上一定要多花心思。很多时候，一个女生喜欢你，她对你的初始兴趣点（上钩点）可能就是你的某一张照片。所以说，展示面照片是非常关键且重要的。

**本节课总结**：本节课我们一起学习了展示面拍摄从“价值优先”到“颜值优先”的核心理念转变。我们掌握了通过寻找最佳拍摄角度和结合后期处理来有效提升个人照片颜值的方法，并深刻理解了高质量展示面照片在吸引异性初始兴趣中的关键作用。
