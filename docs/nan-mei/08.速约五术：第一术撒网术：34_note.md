# 速约五术 08：第一术·撒网术 🎣

在本节课中，我们将要学习速约五术中的第一术——撒网术。其核心目标是快速、大量地积累初始社交资源，为后续的筛选与深入互动奠定基础。没有足够的资源基数，后续所有技巧都将难以展开。

## 核心概念与执行原则

撒网术的核心在于 **“大量行动”** 与 **“严格执行”**。其成功不依赖于复杂的技巧，而取决于是否能够量化并完成基础的积累动作。你可以将其理解为以下公式：

**资源池总量 = 每日接触量 × 执行天数**

这个公式表明，资源的积累是一个通过持续、简单的行动，不断累加的过程。

## 关键执行步骤

以下是实施撒网术需要遵循的具体步骤，每一步都至关重要。

1.  **设定明确的每日目标**：例如，每日主动向一定数量的新朋友发送打招呼信息。
2.  **简化开场话术**：初期无需复杂话术，使用简单、友好的通用开场白即可，重点是完成“接触”这个动作。
3.  **保持持续执行**：将撒网行动变为每日习惯，避免中断。量的积累是此阶段唯一重要的指标。
4.  **不进行深度筛选**：此阶段唯一目的是扩大资源池，因此不应在初期花费过多时间进行个别深度聊天或筛选。
5.  **记录与复盘**：简单记录每日的行动数据，确保执行过程不偏离量化目标。

## 与后续课程的衔接

上一节我们介绍了撒网术的基础执行框架，本节中我们来看看它为何是后续所有步骤的基石。撒网术是整套方法的第一步，它为后续的筛选、开场与引导互动提供了必需的“原材料”。

如果没有执行好撒网术，导致初始资源池匮乏，那么无论后续的聊天技巧多么高超，都将面临“无米之炊”的困境。因为你没有足够的样本进行筛选，也无法实践更精细的互动策略。

## 总结与行动号召

本节课中我们一起学习了速约五术的第一术——撒网术。它的核心是**通过严格、量化、持续的执行，快速积累初始社交资源**。这个方法本身非常简单，其效果直接取决于你的执行力度。

请务必先专注于完成本阶段目标，打好资源基础。在后续课程中，我们将逐步讲解如何利用这些资源，进行有效的开场、引导和深入交流。
