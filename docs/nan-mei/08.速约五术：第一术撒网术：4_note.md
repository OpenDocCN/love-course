# 08.速约五术：第一术撒网术：4_note

在本节课中，我们将要学习如何重构你社交展示照片的价值排序。我们将分析从过去到现在的展示重点变化，并为你提供一个清晰、可操作的现代照片价值框架。

---

上一节我们讨论了照片的初步筛选，本节中我们来看看如何定义单张照片的核心价值。

一张照片必然包含其内在价值。我们可以观察到，从前与现在的拍摄焦点存在差异。从前的照片可能以**展示价值**为主。

然而，现在的氛围已经改变。

以下是现代社交展示中，照片价值的核心排序列表：

1.  **颜值**
    *   首先你需要长得帅。你的照片必须传递出“你很帅”的感觉，营造一种“小鲜肉”的视觉印象。
    *   **核心公式**：`照片首要任务 = 展示高颜值`

2.  **身材**
    *   其次，你需要展现良好的身材。这是构成吸引力的重要视觉组成部分。

3.  **价值与品味**
    *   最后才是展示你的生活价值、经济实力或个人品味。在当前的社交环境下，价值可以排在最后考虑。

综上所述，从前的价值排序是第一位，而现在的把妹氛围已转变为上述顺序。

---

本节课中我们一起学习了现代社交展示照片的全新价值排序。关键在于理解当前环境下的优先级：**颜值 > 身材 > 价值**。掌握这个核心框架，将帮助你更有针对性地选择和制作你的展示面照片。
