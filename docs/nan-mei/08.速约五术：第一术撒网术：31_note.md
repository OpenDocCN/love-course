# 课程08：速约五术·第一术撒网术 🎣

在本节课中，我们将要学习“速约五术”中的第一术——“撒网术”。我们将重点探讨如何通过打造个人展示面来有效吸引目标对象，为后续的互动打下坚实基础。

## 展示面的核心作用

上一节我们介绍了“撒网术”的概念，本节中我们来看看其核心执行环节：构建展示面。展示面是你在社交平台上的个人形象窗口，其质量直接决定了你能否成功吸引他人并获取联系方式。

一个优质的展示面需要传递积极、真实且有吸引力的生活状态。

以下是构建展示面时需要关注的几个关键点：

*   **展示积极生活**：照片和动态应体现你的兴趣爱好、社交活动或个人成就。
*   **确保真实自然**：避免过度修饰，展现自然状态下的自己。
*   **突出个人特质**：通过内容展现你的性格、品味或独特之处。

只要将展示面这一基础工作做好，在初始阶段成功获取对方的联系方式通常不会有问题。
