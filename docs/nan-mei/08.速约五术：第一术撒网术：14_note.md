# 08.速约五术：第一术撒网术：14_note

在本节课中，我们将要学习“速约五术”中的第一术——撒网术。撒网术的核心是收集潜在约会资源的一种方法。我们将详细探讨这种方法的适用人群、核心逻辑以及效率评估，帮助你判断自己是否适合采用此策略。

## 撒网术的核心逻辑与适用性分析

上一节我们介绍了撒网术是一种资源收集手段。本节中，我们来看看这种方法对实践者的具体要求。

接到（指主动搭讪结识）是收集资源的一种形式。你需要理解，这是收集资源的一种手段。

如果你的身高不是很高，那么不太建议采用接到的方式。因为接到需要后续的互动和发展。

从大量学员的实践经验中可以总结出，身高较高的人采用接到方式具有优势。这个方法适合他们，因为效率比较高。

身高较矮的人则不太适合接到。例如，如果采用接到，后续的成功率会特别低。

这里定义的成功率，是指我们是否能与对方建立亲密关系或发生亲密关系。这才是成功的标准。

你会发现，采用接到方式达成此目标的概率是比较低的。因此，不太建议采用接到。

## 核心要点总结

以下是关于撒网术（接到）的关键考量点：

*   **方法本质**：一种主动收集潜在约会资源的**手段**。
*   **关键评估指标**：最终能否建立**亲密关系**。
*   **重要影响因素**：实践者的**身高**条件。
*   **适用建议**：此方法对身高较高者**效率更高**，对身高不具优势者**成功率较低**，需谨慎采用。

## 课程总结

本节课中，我们一起学习了“撒网术”的基本概念。我们明确了它作为一种资源收集手段的定位，并重点分析了身高条件对这种方法成功率的影响。核心结论是，撒网术（接到）更适用于身高具有优势的实践者，以追求更高的效率；而对于身高不具优势者，其后续达成亲密关系的概率较低，因此需要慎重选择。理解这一点，是有效运用后续技巧的基础。
