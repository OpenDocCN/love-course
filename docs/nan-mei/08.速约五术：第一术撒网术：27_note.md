# 08.速约五术：第一术撒网术：27_note

在本节课中，我们将要学习“速约五术”中的核心基础——撒网术。掌握此术的关键在于理解并有效构建你的“展示面”，这是后续所有互动与发展的基石。

## 展示面的重要性

上一节我们介绍了速约五术的整体框架，本节中我们来看看第一术“撒网术”的核心：展示面。展示面是个人形象的线上呈现，无论目的是短期吸引还是长期维护关系，都需要依赖展示面。如果长时间不更新展示面，你收集到的大部分联系人可能会逐渐将你遗忘。

因此，维护一个活跃且优质的展示面至关重要。

## 展示面的维护策略

为了保持展示面的有效性和吸引力，需要遵循特定的更新与质量标准。

以下是关于展示面更新频率与质量要求的具体策略：

1.  **更新频率**：展示面的更新频率应控制在**每两天一次**。这能保持你在他人视野中的存在感，又不会造成信息过载。
2.  **质量标准**：每一张用作展示面的照片都必须经过精心修饰。这意味着你拍摄的每一张照片都需要进行后期处理，以确保最佳视觉效果。
3.  **质量检验**：在最终发布前，可以将照片发到可靠的社交群组中征求意见，以获得外部反馈。
