# 08.速约五术：第一术撒网术：7_note

在本节课中，我们将要学习“撒网术”中关于“养号”期限的核心要点，并了解为何“探探”是实施此策略的最佳平台。

## 养号期限

上一节我们介绍了撒网术的基本概念，本节中我们来看看具体的操作时间。一般养号的期限设定为**7天**。养号7天之后，基本就可以开始进行下一步操作。

## 平台选择与策略

养号完成后，接下来的策略是只匹配女性用户。选择“探探”作为主要平台，是因为如果你们的目标是快速约会，那么探探是目前最好的平台。

以下是选择探探而非其他平台的具体原因：

*   **用户质量高**：像“陌陌”等其他软件，目前不太容易实现快速约会，因为平台上的营销账号过多。
*   **管理严格**：探探平台管理非常严格，因此上面的用户基本都是正常、真实的活跃用户，几乎没有营销账号。
*   **用户意图明确**：在探探上活跃的用户，其目的大多非常明确，不是寻找恋爱关系，就是寻找约会机会。

因此，在探探平台上进行速约操作是比较可靠的。
