# 08.速约五术：第一术撒网术：17_note

在本节课中，我们将学习“撒网术”中关于实际搭讪（接搭）环节的核心技巧。我们将重点关注搭讪时的身体姿态与移动方式，这些细节对于建立良好的第一印象至关重要。

上一节我们介绍了搭讪前的准备工作，本节中我们来看看如何在实际接近时表现得体。

开始搭讪时，你的身体姿态是传递自信的关键。许多初学者容易犯一个错误：身体不自觉地左右摇晃。这会给人一种紧张和不自信的感觉。因此，走向目标时，步伐应稍稳，站定时也要站稳。

---

当对方表现出拒绝信号时，我们的反应同样重要。人的本能反应是试图更靠近对方以说服或拉近距离。但这实际上会产生反效果。

以下是应对拒绝时的正确身体语言原则：

如果发现对方拒绝你，你应该立即向后稍微靠一点。这个细微的后撤动作能有效避免给对方造成压迫感，为互动留下舒适的空间。

---

本节课中我们一起学习了搭讪时的身体控制技巧。关键点在于：**接近时要平稳以展现自信**；**遭遇拒绝时应后撤以避免压迫**。掌握这些身体语言的基本原则，能为你的搭讪开一个好头。
