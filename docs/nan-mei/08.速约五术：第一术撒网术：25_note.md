# 08.速约五术：第一术撒网术：25_note

在本节课中，我们将要学习“速约五术”中的第一术——“撒网术”。我们将探讨如何构建个人价值体系，并了解在初期能力不足时，应如何通过持续行动来培养和展示这些价值。

## 构建你的价值体系 🧱

你的价值体系是你吸引他人的基础。这个体系可以包含多种元素，例如你的身材、外貌、颜值、才艺或厨艺。这些都是构成你个人价值体系的重要组成部分。

## 如何培养价值体系 🌱

如果你在开始时感到能力不足，不知道如何构建这个体系，那么解决方法就是去培养它。这是一个需要主动投入的过程。

以下是具体的行动思路：

*   如果身材不够好，就去健身。
*   这是一个长期的过程，需要耐心和坚持。

## 执行与过渡 🔄

在培养个人价值的过程中，你可以开始执行“撒网术”这一整套方法。上一节我们介绍了价值体系的构成，本节中我们来看看在行动中需要注意什么。

关键在于立即开始并持续行动。即使价值尚未完全展现，行动本身也是展示诚意与潜力的开始。

---

本节课中我们一起学习了“撒网术”的核心前提——构建与培养个人价值体系。我们明确了价值可以来源于多个方面，并且知道了在起点不足时，应通过如健身等长期行动来主动培养。这是展开有效社交吸引的第一步基础。
