# 课程08：速约五术·第一术撒网术 🎣

在本节课中，我们将要学习“撒网术”的核心原理与操作流程。这是一种通过社交媒体动态吸引潜在目标，并高效筛选出有互动意愿对象的方法。

## 撒网术的核心流程

上一节我们介绍了课程主题，本节中我们来看看“撒网术”的具体执行步骤。其核心在于发布动态后，利用互动行为进行高效筛选。

以下是“撒网术”的基本操作过程：

1.  在特定时间（例如晚上九点半）发布社交媒体动态（如朋友圈）。
2.  发布后，会有人对你的动态进行点赞或评论。
3.  此时，主动与这些点赞和评论的人开启私聊。

通过这个过程，你所能接触到的潜在资源会逐渐增多。

## 累积互动窗口

理解了基本流程后，我们进一步探讨其背后的逻辑。每一次发布动态并跟进互动，实际上是在累积与他人的“窗口”。

每次发布朋友圈后，有人点赞或评论，你去切入私聊，因为这些互动行为本身就暗示了对方可能有交流的意愿（即“有窗口”）。因此，长期坚持此方法，你累积的潜在有效联系人会越来越多。

## 高效筛选的关键

然而，并非所有互动都值得投入大量时间。为了提升效率，关键在于从互动者中快速识别出最有价值的对象。

如果你想高效地实践这个方法，就必须用最快的方式筛选出**需求最强烈**的客户或对象。

---

**本节课总结**

本节课中，我们一起学习了“撒网术”。我们首先了解了其发布动态、跟进互动的基本流程。接着，我们明白了该方法能有效累积潜在交流窗口的原理。最后，我们强调了高效执行的核心在于从互动者中**快速筛选出最有需求的目标**。掌握这一术，是进行高效社交拓展的第一步。
