# 08.速约五术：第一术撒网术：2_note

在本节课中，我们将继续学习“撒网术”的具体操作步骤，重点讲解如何准备和设计你的社交软件资料，特别是名字的选取策略。这是成功吸引目标、高效收集资源的基础。

上一节我们介绍了“撒网术”的基本概念，本节中我们来看看如何具体搭建你的“渔网”——即社交软件的个人资料。

## 1. 准备工作：下载软件

开始收集资源前，首先需要下载相关的社交软件。以下是常见的软件选择：

*   探探
*   默默
*   Soul

请务必先下载并安装这些软件。

## 2. 资料设计：以探探为例

软件准备就绪后，下一步是精心设计你的个人资料。我们首先以“探探”为例，讲解名字的设计技巧。

名字是你给人的第一印象，设计得当可以显著提高匹配率。

以下是几种取名策略：

*   **使用英文名**：你可以取一个英文名。
*   **使用暗示性强的名字**：例如“走肾诱华”这类名字，筛选意向目标的强度很高，但缺点是容易导致账号被封禁。
*   **使用引发好奇的名字**：例如我的名字“别诱华危险”，这类名字能勾起他人的好奇心。
*   **搜索现成英文名**：你也可以直接在百度上，搜索并选用那些常见的英文名。

总之，在探探上，名字一定要取一些暗示性较强的。
