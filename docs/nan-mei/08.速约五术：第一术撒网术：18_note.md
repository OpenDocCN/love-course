# 08.速约五术：第一术撒网术：18_note

在本节课中，我们将学习“撒网术”中的一个核心技巧：如何根据女生的实时反馈来动态调整你的互动行为。掌握这一点，能让你在社交互动中更加游刃有余。

---

上一节我们介绍了撒网术的基本概念，本节中我们来看看如何具体应用。

在搭讪互动过程中，你需要观察女生的反应。如果女生对你的反应比较好，你就可以将互动推进得更积极一些。反之，如果女生反应不佳，你就应该将互动节奏放慢，适当后撤。

**核心原则**：根据对方的反馈（Feedback）来调整你的行为（Action）。这可以用一个简单的公式表示：

**你的下一步行动 = 女生的当前反应**

当女生反应积极时，你可以进行“进挪”，即拉近物理或关系距离。

以下是几种常见的进挪方式：

*   向她靠近一点，缩短物理空间。
*   尝试与她握手。
*   在合适的时机给予拥抱。

所有这些行为都必须建立在女生给出良好反应的基础上，不可冒进。根据女生的反应来释放相应的行为，这个原则至关重要。

因此，在搭讪时，你必须密切关注这些互动细节。

这些细节是决定互动成败的关键。
