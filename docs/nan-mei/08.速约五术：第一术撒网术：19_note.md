# 课程08：速约五术 - 第一术：撒网术 🎣

在本节课中，我们将要学习“速约五术”中的第一项核心技巧——“撒网术”。这是一种在特定场景下快速开启对话并获取联系方式的标准化方法。我们将通过分解具体的话术步骤，让你能够清晰、直接地应用。

## 概述

“撒网术”的核心在于利用一个简单、自然的场景作为开场白，快速建立初步联系并导向交换联系方式的目标。其流程直接，易于执行。

## 话术执行步骤

以下是“撒网术”在具体场景（例如在商场或大厦）中的标准操作流程。

1.  **开场白**
    根据当前场景，直接上前说：“嘿，我刚在那边逛街，看到你感觉挺友善的，过来打个招呼。”

2.  **处理回应**
    说完开场白后，如果对方保持沉默或没有表现出明确拒绝，便进入下一步。

3.  **提出收号请求**
    紧接着说：“我朋友马上就过来，我们加个微信吧。”

## 核心逻辑与应用

上一节我们介绍了具体的执行步骤，本节中我们来看看其背后的核心逻辑与应用要点。

此术式的关键在于**快速与直接**。它省略了复杂的寒暄，通过一个合理的理由（感觉友善）开启对话，并立即以“朋友马上到”为理由，自然过渡到交换联系方式的请求。

其核心公式可以概括为：
**自然开场 + 简单理由 + 直接请求 = 收号**

你只需要反复执行这套标准化的话术，就可以在不同场合尝试获取联系方式。

## 总结

本节课中我们一起学习了“速约五术”的第一术——撒网术。我们掌握了其基于特定场景的标准化话术步骤，理解了其追求高效直接的核心逻辑。记住，熟练应用的关键在于清晰表达与自然执行。
