# 08.速约五术：第一术撒网术：1_note

在本节课中，我们将要学习速约五术的第一术——撒网术的第一部分。核心内容是**收集资源**。无论采取何种后续行动，获取足够的潜在联系人资源是首要且基础的一步。许多人在此环节遇到困难，导致后续流程无法顺利展开。本节课将重点介绍如何通过多个主流渠道有效收集资源，并简要说明个人展示面的构建思路。

## 资源收集的重要性

上一段我们明确了本节课的主题。本节中我们来看看为何资源收集如此关键。

大部分男性在尝试与异性建立联系时，常常卡在“没有资源”这个起点上。没有足够的潜在联系人，便无法开启对话、进行练习，这容易导致心态受挫，进而使得后续的所有技巧和方法都难以付诸实践。因此，系统性地收集资源是开启整个流程的基石。

## 主流资源收集渠道

理解了资源的重要性后，以下是当前几种主流且有效的资源收集渠道。

*   **探探**：一款基于地理位置和滑动匹配的社交应用，是快速获取大量初步联系人的传统渠道之一。
*   **陌陌**：同样基于地理位置的社交平台，功能多样，可用于发现附近的用户并建立联系。
*   **街头搭讪**：线下直接与陌生人开启对话的方式，能最直接地锻炼即时互动能力。
*   **新兴平台**：如**抖音**、**小红书**等，这些平台内容属性强，可以通过共同兴趣切入，进行更自然的互动。

## 展示面的初步构建

在通过各种渠道收集资源的同时，我们还需要注意个人形象的展示。一个良好的展示面能显著提高他人接受你联系的意愿。

展示面主要指你在社交平台上的个人资料，包括头像、相册、个人简介等。你需要有意识地排列和展示能够体现你生活品质、兴趣爱好和个人特质的内容，以吸引他人的注意并建立初步好感。

本节课中我们一起学习了速约五术中撒网术的第一步——资源收集。我们明确了资源是后续所有互动的基础，列举了如探探、陌陌、街头搭讪及抖音、小红书等主流收集渠道，并强调了同步构建优质个人展示面的必要性。掌握这些是迈出成功第一步的关键。
