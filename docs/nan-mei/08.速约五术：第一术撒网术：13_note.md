# 课程08：速约五术 - 第一术 撒网术 🎣

在本节课中，我们将要学习“速约五术”中的第一术——撒网术。我们将通过一个具体的案例，分析在特定社交软件上识别和筛选目标时需要注意的关键点，并总结出实用的操作原则。

上一节我们介绍了“速约五术”的整体框架，本节中我们来看看第一术“撒网术”的具体应用场景与风险规避。

## 案例分析：特定社交软件的用户画像

以“默默”这款软件为例，其用户群体存在明显的职业倾向。根据观察，该平台上存在大量酒吧营销人员（俗称“酒托”）以及会所按摩服务者。这两类职业的女性用户在深圳地区尤为集中。

相比之下，其他地区（例如合肥）的此类用户比例则较低，情况相对良好。因此，在深圳地区使用该软件时，需要格外注意。

基于以上用户特征，在来到深圳后，我便大幅减少了在该软件上的使用频率。

## 核心操作原则

以下是基于案例分析总结出的“撒网术”核心操作原则：

1.  **地域差异原则**：不同城市的同一社交平台，其用户构成与质量可能存在显著差异。需要根据本地情况调整策略。
2.  **职业识别原则**：对平台上常见的职业类型（如酒吧营销、会所服务等）保持敏感，这有助于快速筛选并避免无效接触。
3.  **效率优先原则**：当某个平台在特定地区的目标用户密度过低或干扰信息过多时，应果断减少投入，转向更高效的平台或渠道。

本节课中我们一起学习了“撒网术”的初步应用。关键在于理解**目标用户密度 = 有效用户数 / 总用户数**这一核心概念，并通过分析地域和职业特征来评估和选择最适合的“撒网”平台，从而提升社交效率。在后续课程中，我们将继续深入其他技巧。
