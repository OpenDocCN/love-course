# 恋爱心态建设课程 P1：放下抱怨，准备成长 💔

在本节课中，我们将要学习在遭遇情感挫折时，如何建设健康的心态。核心在于放下抱怨，将注意力转向自我成长，从而避免陷入负面情绪的循环。

---

在追求真爱的道路上，遇到挫折、与不合适的人产生纠葛甚至分手是常见的情况。有些人会因此感到深受伤害。

情感体验深刻的人，会体会到刻骨铭心、荡气回肠的感受。自古以来，描述失恋后复杂心情的词语很多，例如撕心裂肺、世界灰暗。更有才子登峰造极，通过诗词来表达。

代表人物有李商隐、李煜、李清照等，他们皆有佳作传世。以李煜为例，他一生从帝王沦为囚徒，爱情经历大起大落。我们暂不展开其故事细节，仅欣赏其词作。他的词意境深刻。

其词风清新脱俗，艺术造诣与美感引发了无数人内心的共鸣。当然，也有人不喜此类婉约风格，认为其过于缠绵，是病态美学。但我认为这并非病态，而是非常正常的情感表达，因为他们的感情很真实。

因为真实所以感人，因为感人所以经典，因为经典所以不朽。下面我们朗诵几句以供欣赏：“无言独上西楼，月如钩。寂寞梧桐深院锁清秋。剪不断，理还乱，是离愁。别是一般滋味在心头。” 还有：“别时容易见时难，流水落花春去也，天上人间。” 以及：“问君能有几多愁，恰似一江春水向东流。” 最后一首：“一重山，两重山。山远天高烟水寒，相思枫叶丹。菊花开，菊花残。塞雁高飞人未还，一帘风月闲。” 这展现了古人对待失恋的一种态度与情怀。

准确地说，这是一部分古人的情感表达方式，并非全部。原因有二：第一，他们属于才子佳人阶层，并非普通百姓，是极少数群体。第二，即使是古代的才子佳人，态度也并非只有这一种，还存在其他很多流派。

例如豪放派。那么现代人是什么态度呢？现代人在失恋、闹矛盾或情感不顺时，我们举一个比较极端的例子，比如遭遇背叛。我观察到的态度大致有两种。第一种是较为理性的方式，例如那句著名的话：

**“恋爱虽易，婚姻不易，且行且珍惜。”**

还有一种则是感性的方式，即发泄情绪、指责抱怨。现实中这种情况似乎更多见。人们抱怨对方薄情寡义、见利忘义、有始无终，指责对方有眼无珠、为何看不上自己、见异思迁、移情别恋。

我们不去辩论这些指责是否正确，姑且假设它们都是对的。那么，上述这些话的潜台词是什么？意思只有一个：**我没有错，我是个受害者，所有的错都是对方的错。**

如果一个人只会以这种方式看待和思考问题，那么这才是悲剧的根源。是什么悲剧呢？这不但损人不利己，还会耽误甚至降低自己，而非促进成长。为什么这么说？

第一点，在既定事实面前，只会一味指责别人、怨天尤人，却又拿不出切实改变现状的办法和行动。这种思维方式在心理学上被称为 **“外归因”** 。这本身就是一种低价值的表现。

```text
思维模式 = 外归因 (External Attribution)
```
这种模式有什么意义？它解决不了任何实际问题，并且还会显得自己特别廉价，不值得被爱和珍惜。因此我们不推荐这样做。

并且，这样也不能让自己的生活或心情变得更好。或许有人会反驳说：“我说出来就痛快了。” 然而，抱怨吐槽带来的情绪释放很短暂。长期来看，依然没有任何幸福感可言。关键还在于，这种事也不适合对外人诉说。

俗话说，家丑不可外扬，好事不出门，坏事传千里，没有不透风的墙。不管你们曾经关系多好，是闺蜜还是老铁，万一以后闹翻了呢？这些都可能成为别人笑话、鄙视甚至攻击你的证据。

即使你们关系一直不错，这些事在别人眼中也是给你减分的。这是人性使然。这对你的人生成长与幸福毫无益处。你想想是不是这样？

那该怎么办呢？高情商的做法，可以用一句话总结：**“把快乐留给别人，把悲伤留给自己。”** 有些事自己消化掉就结束了。如何消化呢？具体思路我们下期再分享。

---

上一节我们探讨了外归因思维的弊端，本节中我们来看看与之相对的另一种模式。

第二点，就是有些人在感情中遇到任何问题，都只会指责别人（即外归因思维），而不知道反省自己（这在心理学上称为 **“内归因”** 思维）。

```text
思维模式 = 内归因 (Internal Attribution)
```
那么这样做的结果是什么呢？结果就是这个人永远看不到自己的不足和缺点。这就等于这个人失去了成长性，也叫作失去了进步的空间。这跟“骄傲使人退步，谦虚使人进步”是同一个道理。

本来，每个人都有这样那样的缺点和不足，这很正常。但是，假如一个人有很多缺点，同时他还没有了进步的空间和成长性，这就可悲了。尤其是当你的另一半在不断追求进步和成长时。

两个人之间的差距就会不断被拉大。这也是许多表面感情矛盾形成的内在原因、深层原因和本质原因。我们也要看到，并非所有人都只追求外在的东西，如功名利禄、声色犬马。

有很多人把追求内在成长看得很重要。什么是内在成长？指的是精神的境界、心灵的体验、人生的拓展等等。当然，我们再说一遍，不是所有人都这样。

在两个人刚开始谈恋爱时，这种人生价值观的差距往往容易被忽略。等到真正在一起后，它就会慢慢发酵出各种各样的矛盾。一旦感情出现问题，人们往往只会指责那些由价值观差异衍生出来的表面问题，而忽略了价值观差异这个深层问题。

也就是说，**一个人有成长性，另一个人因各种原因没有成长性，这才是产生各种矛盾的本质。**

为什么有人没有成长性？有两个原因。第一是主观原因：过于自负、自满、傲娇任性，没有空杯心态。这种人肯定没有成长性，他认为自己什么都懂、什么都会。

第二是客观原因：一部分人为生活所迫，工作太忙、平时太累；或个人运气不好，生活中不如意的事情太多，耗尽了用于成长的时间、精力和好心情。日常被这些事占满。

有人说，不能给自己拒绝成长找客观理由，不能为他们开脱，那些人就是懒。“只要想成长，谁都挡不住”、“有志者事竟成”、“苦心人天不负”，你没见过凿壁偷光的故事吗？这些话到底是鸡汤还是毒鸡汤，各位自行分辨。今天我也不想表达自己的观点，这个以后再说。
