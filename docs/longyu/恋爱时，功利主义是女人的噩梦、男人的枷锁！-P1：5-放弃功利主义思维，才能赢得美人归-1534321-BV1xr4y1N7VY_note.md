# 恋爱课程 05：放弃功利主义思维，才能赢得美人归 💔➡️💖

在本节课中，我们将要学习功利主义思维如何成为恋爱中的障碍，并探讨如何通过培养“娱乐精神”来化解被动局面，从而建立更自然、更美好的恋爱关系。

## 回顾：娱乐精神的优势

上一节我们介绍了展示娱乐精神的好处。本节中我们来看看它的第三个核心优势：**在交往过程中保持主动，避免陷入被动**。

所谓“被动”，不同的人会遇到不同的具体情况，每个人感受到被动的原因也各不相同。但这并不重要，重要的是化解被动的方法就那么几种，我们掌握即可。

那么，为什么展示娱乐精神能化解被动呢？因为娱乐精神能让你的追求行为变得**灵活**。它以一种真真假假、嬉皮笑脸的开玩笑方式，为你和对方都留有余地和退路。这能避免把天聊死，也不至于把关系搞僵，让你始终拥有回旋的空间。

## 深入理解“娱乐精神”

有一点我们需要强调：娱乐精神不只是通俗理解的幽默感或讲笑话。它是一种宝贵的**品质**和强大的**社交心态**与**恋爱心态**。它能反映出你超然洒脱的人生态度。如果仅仅将其理解为幽默或耍贫嘴，那就过于狭隘了。

## 功利主义：娱乐精神的最大阻碍

听到这里，你可能会问：娱乐精神如此重要，我们该如何提高？为什么很多人没有这种精神？原因有很多，但主要原因是长期受**功利主义**的思维习惯和行为方式所束缚。

我们在前面的课程中反复批判过功利主义，称其为“爱情的杀手”和“恋爱的大忌”。具体来说，功利主义思维有两个明显特点：
1.  **行为目的性太强**
2.  **患得患失**

这种思维方式就像一根无形的绳子，或者像《西游记》里捆住孙悟空的“捆仙绳”和“紧箍咒”，束缚了你的恋爱天赋、生命活力和表现力。

以下是功利主义在恋爱中的两种具体表现：

### 1. 急功近利型
在恋爱过程中，目的性太强，只在乎结果，忽视了过程本身的美好。缺乏享受过程的能力，总想“走捷径”。

**典型心态**：`“最好刚认识就确立关系，赶紧答应做我女朋友，否则我投入这么多，最后谈不成岂不是很亏？”`

这就是急功近利、患得患失的表现，很不好。

### 2. 谨小慎微型
这部分人的特点是：在恋爱中姿态太低，总怕出错。其根源在于有许多根深蒂固的错误、片面或不真实的恋爱认知。

**导致的结果**：把与喜欢的人聊天、约会这种美好的事情，变成一种负担和包袱。或者，将吃饭、逛街、看电影等本应享受的过程，变成机械的“走流程”，毫无快乐可言。

如果这种谨小慎微的状态不改变，即使学习再多的恋爱技巧也无用武之地。你个人的优点（如经济条件、外貌）也会被这种无形的绳子束缚，你的男性魅力和天性会受到压抑，在约会或聊天时无法施展个人魅力。

## 功利主义的深远危害

这种状态的影响远不止于恋爱。它对你拓展社交、与人交往也会产生很大负面影响。处处精打细算以求成功，结果往往适得其反，导致失败。正如《红楼梦》所言：**“机关算尽太聪明，反误了卿卿性命。”**

此外，你自身也会感觉非常糟糕，生活得别扭、压抑、不痛快。即使外在条件优越，内心的幸福感和心情也会大打折扣。这非常划不来。这种哲学层面的思辨，值得大家好好体会。
