# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要一个地方来存放它，这就是**变量**。同时，数据本身有不同的种类，比如数字和文字，这些种类就是**数据类型**。掌握如何创建变量并了解其数据类型，是编程的基石。

上一节我们概述了本课程的目标，本节中我们来看看什么是变量。

## 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如一个数字、一段文字）。标签就是变量的名字，通过这个名字，我们可以在程序中的任何地方找到并使用盒子里的数据。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。

```python
my_name = "龙宇"
age = 25
```

以上代码创建了两个变量：一个名为 `my_name` 的“盒子”里存放了文字“龙宇”；另一个名为 `age` 的“盒子”里存放了数字 `25`。

接下来，我们看看这些数据具体有哪些类型。

## 基本数据类型 🔢

Python有一些内置的基本数据类型，用于表示不同种类的数据。以下是三种最常用的类型：

1.  **整数**
    整数是不带小数点的数字，可以是正数、负数或零。
    ```python
    count = 10
    temperature = -5
    ```

2.  **浮点数**
    浮点数是带有小数点的数字，用于表示更精确的数值。
    ```python
    price = 19.99
    pi = 3.14159
    ```

3.  **字符串**
    字符串是由一系列字符组成的文本数据，需要用单引号 `'` 或双引号 `"` 括起来。
    ```python
    greeting = "你好，世界！"
    author = '龙宇'
    ```

了解数据类型后，你可能会问，我们如何知道一个变量里存放的是什么类型的数据呢？

## 使用 `type()` 函数检查类型 🔍

Python提供了一个内置函数 `type()`，它可以返回任何变量的数据类型。这在调试和了解代码时非常有用。

```python
print(type(my_name))  # 输出：<class 'str'>
print(type(age))      # 输出：<class 'int'>
print(type(price))    # 输出：<class 'float'>
```

从输出可以看到，`my_name` 的类型是 `str`（字符串），`age` 是 `int`（整数），`price` 是 `float`（浮点数）。

现在我们已经学会了创建变量和识别类型，最后我们来探讨一个常见的操作：如何改变变量的内容。

## 变量的重新赋值 ✏️

变量的值不是一成不变的。你可以随时给一个已经存在的变量赋予一个新的值，这称为“重新赋值”。

```python
score = 95
print(score)  # 输出：95

score = 100   # 将变量 score 的值改为 100
print(score)  # 输出：100
```

需要注意的是，当你给变量重新赋值时，新值的数据类型可以和之前完全不同。

```python
value = 123      # value 最初是一个整数
print(type(value))  # 输出：<class 'int'>

value = "现在我是字符串了"  # 现在 value 变成了一个字符串
print(type(value))  # 输出：<class 'str'>
```

## 总结

本节课中我们一起学习了Python编程的起点：**变量**与**数据类型**。

*   我们知道了**变量**就像一个命名的存储盒，用 `=` 来赋值。
*   我们认识了三种**基本数据类型**：用于整数的 `int`，用于小数的 `float`，以及用于文本的 `str`。
*   我们学会了使用 `type()` 函数来检查变量的类型。
*   我们理解了变量可以通过重新赋值来改变其内容和类型。
