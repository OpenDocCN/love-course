# 爱情启示录 第1期：妨碍男人获得爱情的九十九个原因·由来 🧩

在本节课中，我们将要学习本系列课程的由来和背景。我们将探讨为何要系统性地分析男性在恋爱中遇到的障碍，并了解本系列内容的结构与目的。

---

大家好，我是龙宇。首先向大家致歉，因为今天的内容比较特殊。特殊之处在于它一反我们平时每期视频干货多的特点。今天我们不讲干货，内容比较水。

这主要是为我们后面这几期节目做一下铺垫。后面这几期节目是一个系列。主题就是分析作为男人在谈恋爱这件事上失败或者叫还没有成功的一些原因。这是一个系列。

我们为什么要花几天时间讲这个系列呢？首先，我认为这确实是一个比较严肃的问题。弄清楚这些有助于我们更好地完善自己，少犯错误少走弯路。还有一个原因，就是作为爱情的一个狂热信徒，我这一直以来，就二十几年以来，对恋爱学这个领域很多问题很多现象都特别感兴趣。

都说兴趣是最好的老师，确实是这样。在强烈的好奇心和求知欲的驱动下，我愿意花时间和精力去做这种比较深入的、系统的探索、研究、分类和总结。恋爱学要探讨的问题很多，包罗万象。

比较宏观的问题有，比如什么是爱情观，什么是正确的爱情观，哪些是错误的。爱情中有哪些陷阱，如何防范那些谈恋爱目的不纯的人，他们都有哪些手段方法技巧，如何去识别和应对。这些都是宏观的。

还有吸引力的本质是什么，如何让自己变得更加有吸引力，如何展示。恋爱过程分几个阶段，有什么流程和顺序吗，以及为什么怎么做。不是三段论吗，是什么为什么怎么做。都有哪些恋爱技巧等等这些。

这其中就有一个很重要的主题，或者叫研究课题，那就是我们开篇提到的：到底是什么原因阻止一个人获得爱情。我们说得再详细一点，都有哪些原因在妨碍着一个人（我们主要说男人）追求爱情、获得爱情。也可以把这个问题再换一种说法：导致一个男人情感受挫、很难找到真爱、谈恋爱总是谈不成的原因都有哪些。

除了上面说的我自己个人对恋爱学领域这些问题都很好奇这个因素之外（这个因素属于主观因素），它还有一个客观因素也在驱使着、鞭策着、鼓励着我把这些问题搞清楚。

客观什么因素呢？它是这样的：有的亲人们可能知道，我教恋爱学、讲课、做培训、情感咨询也有十几年了，线上线下的。跟学员接触的时间很长，很多就成了好朋友。大家都相信我的人品（我比较靠谱仗义，不胡来），也认可我的教学理念（我们以人为本，理性细致，我们干货比较多——除了今天——我们鸡汤少）。同时，兄弟们也认可我的敬业精神（那就是专注、认真、负责、不敷衍、不忽悠这种精神）。

所以，我和很多兄弟交流得比较深入。他们问我的问题就比较多，或者说他们更愿意跟我探讨问题。有很多问题甚至都超出了情感的领域，像什么人生态度、人生观、家庭琐事、什么事业方面的、工作、职场、心理等等这些都会探讨。

我们就说情感领域，他们把自己亲身经历的和女生交往的一些过程细节跟我一起交流，向我咨询问题。在这一方面也给我提供了大量的现实的一些案例。再加上我本身又有一个好为人师的坏毛病。这就导致学员他们用数不尽的问题倒逼老师成长。

因为对于我来说，没有经过我自己认真思考、深入思考过的问题，在别人问的时候我讲不出来。我不愿意把一些道听途说的、没有经过自己大脑的、我自己还拿不准的答案随便分享给别人。亲人们，我不是一个随便的人。

好了，我们回到正题。就是前面我们做了一些铺垫，算是把今天的题审清楚了。因为接触的学员多了，我就发现很多单身的、脱单困难的、情感受挫的、不会谈恋爱的亲人们，本身条件不错。你说身高、长相、颜值、收入、学历、品位、包括见识、人品、生活习惯、个人素质等方方面面的都不错，各种条件不像大家印象当中的那么差，根本不是。

有些兄弟（当然不是全部）可以说他们条件很好，非常好。就拿学历来说，那有清华的、北大的、人大的、北理工的、上海交大的、研究生，这些学历是没问题的。再说职业，有外科医生、律师、企业高管，还有创业者。大家想想，这些兄弟们收入他们能低得了吗？还有富二代、民营私营的老板、在一二线城市有好几套房产的、财务都自由了（虽然是少数，但是也有）。你还别不信，事实它就是这样。

有些亲人在自己的专业领域，他们事业都做得很成功。但唯独在跟女生交往、谈恋爱这件事上经常受挫，表现得迷茫困惑，包括经常犯一些错误，不知所措，不知道怎么突破，没有恋爱框架。没有恋爱框架，给人感觉好像是情商低、不会聊天。总之，出各种各样的问题。

这个就让我想起木桶理论说的那样：就是你这个桶，只要其中有一块短板，它就会把里面的水漏出一部分去。漏出去的水就是幸福，它就是爱情。而组成短板的要素，那些因素，我就发现有很多。

当然了，具体到每个人，他不可能把这些要素都占全。但是我们讲课是针对所有的人，所有的想通过学习恋爱学提高自己的情感认知和恋爱技巧的兄弟。所以我们要把统计出来的这些因素都得讲出来。大家就可以对号入座，有则改之，无则加勉。

如果您有成长之心，能听到心里去那就最好。如果您一向比较孤傲，内心比较敏感，不愿意去触碰自己的软肋，那就当听评书听故事，也无不可。如果是您听着听着，有点感觉了，心中有那么一点点共鸣，并且听完之后你若有所思、怅然若失、感觉有启发，从此你开始五日三省五申，我们这一系列的节目就算没有白做，我就感觉到很欣慰了。如果您一高兴，起身抬手给视频点个赞，那就皆大欢喜了。

谢谢大家。好，谈高级恋爱找龙宇情感，我们下集再见。
