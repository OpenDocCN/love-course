# 课程名称：魏子哥的编程入门课 第1课

在本节课中，我们将要学习编程中最基础但至关重要的概念——变量。理解变量是编写任何程序的第一步。

## 什么是变量？🧠

上一节我们介绍了课程目标，本节中我们来看看变量的本质。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据，比如数字、文字或更复杂的信息。标签就是变量的名字，通过这个名字，我们可以在程序中的任何地方找到并使用盒子里的数据。

在大多数编程语言中，创建一个变量（即准备好一个盒子并贴上标签）的过程叫做“声明”。给变量放入一个具体的值，则叫做“赋值”。

## 如何声明与使用变量？📦

理解了变量的比喻后，我们来看看在代码中如何具体操作。以下是一个在类似Python的语言中声明和赋值变量的基本示例：

```python
# 声明一个名为‘age’的变量，并赋值为25
age = 25
# 声明一个名为‘name’的变量，并赋值为“魏子哥”
name = "魏子哥"
```

代码中，`age`和`name`就是变量名（盒子上的标签），等号`=`是赋值操作符，意思是将右边的值放入左边的变量中。

## 变量的命名规则与最佳实践 ✏️

给变量起名字并非随心所欲，需要遵循一些规则和约定，这能让你的代码更易读、更专业。

以下是命名变量时需要遵守的核心规则和推荐做法：

1.  **可以包含的字符**：变量名通常可以包含字母（A-Z, a-z）、数字（0-9）和下划线（_）。
2.  **不能以数字开头**：例如 `1variable` 是无效的，但 `variable1` 是有效的。
3.  **区分大小写**：`Age`、`AGE` 和 `age` 是三个不同的变量。
4.  **避免使用保留字**：不能使用编程语言已经占用的关键词，如 `if`、`for`、`while` 等作为变量名。
5.  **追求清晰有意义**：变量名应清晰描述其用途，例如用 `user_age` 而非 `a`，用 `file_path` 而非 `fp`。

## 变量的类型 🏷️

变量不仅可以存储数据，还可以存储不同类型的数据。就像盒子可以专门用来装水果、书籍或工具一样，变量也有“类型”。常见的类型有：

*   **整数**：存储没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数**：存储带有小数的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：存储文本信息，在代码中用引号包围，如 `"你好"`, `"abc123"`。
*   **布尔值**：存储逻辑判断的结果，只有 `True`（真）或 `False`（假）两种值。

了解类型很重要，因为它决定了你能对这个变量进行何种操作。例如，数字可以进行加减乘除，而字符串可以进行拼接。
