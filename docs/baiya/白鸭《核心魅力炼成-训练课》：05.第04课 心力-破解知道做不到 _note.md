# 白鸭《核心魅力炼成-训练课》：05.第04课 心力-破解知道做不到 🎼

## 概述

在本节课中，我们将探讨**心力**这一概念，它是指我们内核的力量，帮助我们抵御外界的刺激，并更好地掌控我们的身体。我们将学习如何通过训练提升心力，以及如何将高维度的认知应用于我们的日常生活中。

## 心力的定义与作用

**心力**是指我们内核的力量，它有两个主要作用：

1. **抵挡外界刺激**：当外界刺激过于强烈时，心力可以帮助我们保持内核的稳定，避免内核被打乱。
2. **决策执行**：心力可以帮助我们执行决策，控制我们的身体，使其按照我们的意愿行动。

### 核心概念

**心力** = 内核力量

## 心力的训练方法

心力可以通过训练来提升，其核心原则是**抗阻做功**。

1. **从小重量开始**：就像训练肌肉一样，心力训练也需要从小重量开始，逐渐增加难度。
2. **克服外界刺激**：例如，在等电梯时克服焦躁情绪，或在面对压力时保持冷静。

### 核心概念

**抗阻做功** = 通过克服外界刺激来提升心力

## 心力训练的三个重点

1. **提升心力**：通过抗阻做功，不断克服外界刺激，提升心力。
2. **提升觉察能力**：及时发现并处理内核的晃动和不良念头。
3. **提升认知维度**：学习正确的游戏规则和计分规则，站在更高维度看待问题。

### 核心概念

**心力训练三个重点** = 提升心力、提升觉察能力、提升认知维度

## 高维度认知的应用

在心力训练中，我们需要将高维度的认知应用于日常生活中。

1. **关注内核稳定**：在与人交往时，关注自己的内核是否稳定，是否输出自信、沉稳、强大的行为。
2. **设定高维度目标**：将目标设定为提升心力、觉察能力和认知维度，而不是追求低维度的目标。

### 核心概念

**高维度认知** = 从更高维度看待问题，设定更高目标

## 总结

本节课中，我们学习了心力的概念、作用、训练方法以及高维度认知的应用。通过训练心力，我们可以更好地抵御外界刺激，掌控自己的身体，并提升个人魅力。
