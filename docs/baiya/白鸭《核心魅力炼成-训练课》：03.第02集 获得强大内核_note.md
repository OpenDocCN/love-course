# 白鸭《核心魅力炼成-训练课》：03.第02集 获得强大内核_note

在本节课中，我们将要学习“强大内核”的构成与工作原理。我们将深入探讨内核的三个层次，并理解它是如何驱动我们的行为，最终塑造我们的魅力。

## 内核的构成

上一节我们介绍了魅力源于高价值的行为表现，而本节中我们来看看这些行为背后的驱动力——强大内核。内核主要由三个层次构成。

以下是内核的三个组成部分：

1.  **认知层**：这是最上层，存储着我们对于世界的理解。世界的游戏规则、评分规则都记录在这一层里。
2.  **内心世界**：这是中间层，包含我们内心的想法和感受。例如自我认知、存在感、安全感、满足感等。这一层深受上层认知的影响。
3.  **决策层**：这是最下层，存储着经过认知和内心世界处理后，对身体形成的指令信号。

## 内核的工作流程

了解了内核的构成后，我们来看看它是如何运作的。这是一个从接收信号到做出反应的完整流程。

当外界对我们产生一个刺激（例如，有人对我们竖起大拇指），这个信号会进入我们的**认知层**。我们会根据已有的认知进行判断（例如，“他是在夸我”）。

接着，这个判断会进入**内心世界**，引发相应的感受（例如，开心、满足、对对方有好感）。

然后，我们会产生一个想法（例如，“我要给他一个好的回应”）。这个想法会在**决策层**形成思维决策，并发出指令给身体。

最终，身体执行指令，做出相应的行为表现（例如，同样竖起大拇指并表现开心）。

这套流程 `外界刺激 -> 认知层 -> 内心世界 -> 决策层 -> 行为表现` 在我们的人生中不断运行。

## 认知匮乏的后果

如果我们的认知层知识匮乏，对外界的理解维度就会很低，这会导致一系列错误的反应。

以下是认知匮乏可能导致的三种情况：

*   **显性规则误解**：例如，小时候看跳水比赛，以为水花越大得分越高，并因此为水花大的选手欢呼。这种行为在了解规则（水花越小得分越高）的人看来会很傻。这种规则相对容易通过观察倒推出来。
*   **隐性规则困境**：例如，完全不懂麻将规则的人与熟知规则的人打牌，可能错误地以为“9万管1万”。这种隐性规则难以悟透，需要踩很多坑、交很多“学费”才能摸索清楚。
*   **滞后性危害**：这是最可怕的一种。某些行为的影响是滞后的。例如，不了解健康饮食知识，认为炸鸡好吃就天天吃。快乐和满足感是即时的，但身材臃肿、健康问题却是在很久之后才会显现，届时修复需要付出巨大代价。

我们生命中很多人际关系问题，都是这种既隐性又滞后的。例如，在感情中拼命对对方好，初期可能效果不错，但长期会导致自己越来越卑微；或者通过“作闹”来获取关注，短期内对方可能会哄你，但到达某个时间点，对方可能会离开。这时再想修复感情会非常困难。

因此，在不了解游戏规则时，我们的理解、感受、想法都可能是错误的，最终导致的行为也是减分、没有魅力的。

## 搭建正确的认知体系

只有掌握了清晰正确的规则，把事情做对，我们才能搭建起强大的认知体系，从而对自己越来越有信心。

所以，我们训练方法的第一步，就是通过听课、抄笔记、抄心法，将这些全新的、有魅力的认知灌输给你，替换掉以前那些错误的、不自信的认知。

例如：
*   **错误/不自信的认知**：喜欢一个人，我一定要让他看到我的诚意，一定要对他好。
*   **正确/有魅力的认知**：喜欢一个人，我对他说什么根本不重要。我有没有魅力，能否用魅力去征服和吸引他，这才重要。

这个世界的法则是：**讲话的人是谁，比讲什么更重要**。如果一个人没有魅力，把心掏出来也没用；如果一个人特别有魅力、特别强大，身边的坏人都可能为他变成好人。

通过十二堂课不断输入新的认知，替换旧的认知，我们内核的基础就搭建好了。

## 从“知道”到“做到”

然而，有了正确的认知，并不代表我们立刻就能拥有魅力。知道和做到之间存在差距，我们的内核还需要有力量。

举例来说，你有了要表现得自信沉稳的认知。面对一个条件普通的相亲对象，你或许能做到从容自信。但面对一个极其优秀的相亲对象时，你可能会不自觉地变得紧张、花痴，女神范儿荡然无存。对方看到你的样子，兴趣可能就消失了。

所以，在搭建好内核的理论基础后，我们还需要通过训练不断强化它，让内核越来越有力量，能够更好地把控自己。

人的思维有惯性，即使知道了正确的做法，做事时仍会不自觉地跑回旧的不自信模式中。这时，就需要训练来让新认知顺畅运转，让高价值行为能持续输出。

## 核心训练工具：觉察、贴标签、搬轨道

这就需要用到我们的核心训练方法——魅力工具：**觉察、贴标签、搬轨道**。

以下是这三个工具的使用方法：

1.  **觉察**：当心中冒出一个由旧认知产生的不自信想法，或内核开始晃动时，第一时间发现它。
2.  **贴标签**：给这个冒出来的想法、情绪或行为一个明确的定义。例如，男生不回信息，你感到“焦躁”；帅哥来搭话，你行为“慌张”。贴上标签，就是将它从潜意识中拉到意识层面。
3.  **搬轨道**：将旧的认知“轨道”切换到新的认知“轨道”上。让新的、有魅力的想法顺利替代旧想法，接管身体，发出决策指令，从而做出高价值、有魅力的行为表现。

如果旧的认知和想法不能被第一时间觉察，我们就会被下意识带着走，做出没有魅力的行为。但一旦给它打上标签，新的认知就会开始接管，引导所有后续行为和想法向有魅力的方向进行。

除了上课输入知识，我们每天都要运用这三个魅力工具，不断重复“觉察->贴标签->搬轨道”的过程，从而让自己心力越来越强，让有魅力的内核工作得越来越顺畅、自然。

## 总结与展望

本节课中我们一起学习了强大内核的构成与工作原理。

总结一下：我们通过上课不断输入新认知，替换旧认知，让不自信的想法变少，有魅力的想法变多。然后通过日常训练（觉察、贴标签、搬轨道），让有魅力的认知能顺利接管身体，产生有魅力的行为。当有魅力的行为越来越多，周围人的积极反馈会让你越来越坚定。当内核完全强大时，你会发现，**你说什么话都是正确的**，因为你的魅力足以支撑你所说的任何内容。无论你回答“可以”还是“不可以”，别人都会珍惜你、好好对你。到那时，你做什么都是有魅力的。
