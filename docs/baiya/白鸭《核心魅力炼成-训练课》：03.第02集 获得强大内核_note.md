# 白鸭《核心魅力炼成-训练课》：03.第02集 获得强大内核

🎼大家好，我是白鸭。上节课我们探讨了人为什么会有魅力，以及魅力是如何被感知的。我们了解到，魅力源于我们在与他人交往时展现出的高价值、自信、沉稳和强大的行为。

那么，我们为何会展现出这些高价值的行为呢？这是因为我们内心拥有一个强大的内核，就像钢铁侠的核反应堆一样，源源不断地为我们提供能量，使我们能够展现出这些优秀的品质。

今天，我们就来深入探讨一下这个内核的原理。😊，Yeah。Yeah。🎼它主要由三个部分组成：

1. **认知层**：这一层存储着我们对这个世界的理解，包括世界的运作方式、各种事物的游戏规则和评分规则等。
2. **内心世界层**：这一层包含我们的想法和感受，如自我认知、存在感、安全感、满足感等。这一层的内容深受认知层的影响。
3. **决策层**：这一层是经过认知和想法感受两层的处理后，对身体形成的指令信号。

当外界对我们产生刺激或信号时，例如有人对我们竖起大拇指，这个信号会进入我们的认知层。我们会根据对人和人之间关系的理解或对世界游戏规则的理解进行判断。然后，我们的内心世界会对这个信号产生反应，例如感到开心和满足。紧接着，我们会产生一个想法，即要给对方一个积极的回应。这个想法会形成思维决策，并在决策层产生指令，最终到达我们的身体，驱动我们做出相应的行为。

这个过程在我们的人生中每天都在不断运行。🎼

当我们认知层知识匮乏时，我们对外界的理解维度非常低下，对外界给予的很多反应都会是错误的。🎼例如，小时候看跳水比赛，我们可能认为水花越大，跳得越好，分数越高。但实际上，水花越小，得分越高。🎼

有些事物比较复杂，例如麻将规则，需要我们不断摸索才能理解。🎼还有一些事物的影响是滞后的，例如健康饮食。🎼如果我们不了解某些事情的游戏规则和评分标准，我们对很多事情的理解、感受和想法都可能是错误的，导致我们做出的行为没有魅力。

因此，我们需要搭建一个强大的认知体系，让我们对自己越来越有信心。🎼

我们的训练方法的第一步是，通过听课、抄笔记、抄心法等方法，将这些全新的知识认知灌输给你，替换掉你以前那些错误的不自信的认知。🎼

然而，仅仅拥有这些认知还不够，我们还需要让内核拥有力量。🎼例如，你知道要表现得自信、沉稳和强大，但当你遇到不同的人时，你的内核可能会动摇。🎼因此，我们需要通过训练方法，让内核越来越有力，越来越能够掌控自己。

我们的训练方法包括觉察、贴标签和搬轨道。🎼

1. **觉察**：当你心中有一个不自信的想法冒出来时，你需要第一时间发现它。
2. **贴标签**：给这个想法、情绪和行为贴上标签，例如“焦躁”或“慌张”。
3. **搬轨道**：将旧的认知搬到新的认知上，让新的想法顺利替代旧的想法。

通过不断运用这些魅力工具，我们可以让自己的心力越来越强，有魅力的系统越来越顺畅。🎼

总结一下，我们通过不断输入新的认知，替换旧的认知，让你越来越自信。🎼然后通过训练，让我们有魅力的想法和认知能够顺利接管我们的身体，产生有魅力的行为表现。🎼当你有魅力的行为表现越来越多时，周围人对你的反馈会越来越坚定，越来越相信自己的认知是正确的。🎼

当你拥有了强大的内核时，你会有这样的概念：🎼例如，一个优秀的男生向你表白，你说“可以，我觉得你还不错”或“不可以，我觉得你表白的太随意了”，这两个答案都是正确的。🎼因为你的魅力可以支撑你说的任何话。🎼
