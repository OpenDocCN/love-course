## 独自拍摄演示

以下是关键操作步骤：
1.  在行走过程中，务必使用**连拍**功能。
2.  即使没有他人协助，也可独立完成。本次演示使用了一个包作为支架。

接下来，我们尝试在无人协助的情况下进行拍摄。我将手机反转过来，设置**十秒定时**拍摄。
```python
# 操作类比：设置相机定时器
camera.set_timer(seconds=10)
```
我先确认拍摄位置，然后摆好姿势。姿势的把握非常重要。

当前的拍摄效果基本符合预期。如果机位能再稍高一点会更理想。独自拍照通常需要多次尝试，这很正常，关键在于是否有拍摄的意愿。

我现在尝试第二次十秒定时拍摄。这次我将**对焦点锁定在画面上方**，然后开始拍摄。

第二次的效果比第一次好很多。但画面有些曝光过度，因为无人协助调整光线。与第一次对焦完全在下方不同，我们需要重新拍摄。

反复失败并不重要，它能带来更多成长经验。

## 设备放置与光线控制

现在，我调整设备位置，将包垫得更高一些。具体如何垫高由你自行决定。不要认为自己无法完成。

尽量减少将手机竖直放置，因为这样摄像头位置较高。如果将手机反转，摄像头会朝向下方，更贴近地面。毕竟很少有人会直接趴在地上取景。

记住在对焦时，尽量点击画面中**光线最亮**的区域进行测光和对焦，以降低整体曝光。
```python
# 操作核心：对高光区域点测光
metering_mode = “spot”
camera.set_metering(metering_mode, target=“brightest_area”)
```
点击后锁定对焦与曝光，然后开始拍摄。确定好这个位置后，完成拍摄。

这次拍摄本身基本没有问题，但我的脚部姿势没有摆好，不如前两张照片的效果。因此我们需要再次重拍。

反复重拍是很好的体验，不要觉得烦躁。如果能有伙伴协助，效果会更佳。

## 与伙伴配合拍摄

最后我演示一张认真的拍摄，不做讲解。在之前的讲解中，我忽略了如何摆动作和做好行为管理。如果有伙伴协助，则会方便很多。

首先，我演示几种姿势供伙伴拍摄参考：
1.  **站直**的感觉。
2.  **微侧身**的感觉。
3.  **行走**中的步伐感觉。

我将拍摄这三种感觉。

拍摄这类照片时，尽量选择**趴下**或**蹲下**的低角度机位，这样视角更好。为了视觉效果，我可能会趴下。如果介意地面脏，蹲着拍摄也可以。

伙伴在这个有光的位置拍摄即可。有伙伴协助时，他能随时帮你找到最佳角度。

你需要明确告诉伙伴你想要的效果。例如，可以尝试拍摄双脚的构图，但记住重点是有人帮你调整。

也可以拍摄单脚的特写，这种角度很贴近地面。对焦点应在鞋子上。

再拍一张对焦在地面，让鞋子产生虚化效果的照片。稍微调整一下角度。

如果有路灯，光线效果会更棒。

我先为你拍一张，你的脚刚好在这个位置。我演示如何利用这个光线进行拍摄调整。

尝试让光线落在不同位置。即使有车辆经过也无妨，可以先拍出来看效果。
