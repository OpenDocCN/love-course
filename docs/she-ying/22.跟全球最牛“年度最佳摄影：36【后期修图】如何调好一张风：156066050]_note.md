# 课程22：跟全球最牛“年度最佳摄影师”学摄影！（完结）：36【后期修图】如何调好一张风景照片 🏞️

在本节课中，我们将学习如何对风景照片进行后期处理。核心原则是：**风景照片不宜过度修饰**，否则会显得不自然。我们将通过几个具体案例，学习使用Snapseed和VSCO等软件进行简单有效的调整。

---

## 概述

风景照片的后期处理，关键在于**适度**。过度调整色彩、对比度或添加夸张的滤镜，往往会破坏照片原有的自然美感。本节课将通过分析几张不同场景的风景原图，演示如何通过微调色彩、光影等参数，让照片在保持真实感的同时更具视觉吸引力。

---

## 第一节：手机拍摄风景照的调整思路

上一节我们概述了风景照后期的核心原则。本节中，我们来看看用手机拍摄的风景照具体该如何处理。

手机摄影的宽容度通常不如单反相机，因此后期调整空间相对有限。处理手机风景照时，重点应放在**色彩饱和度**和**整体氛围**的微调上，避免大范围的参数改动。

以下是用Snapseed调整一张新疆奎屯大峡谷（安集海大峡谷）照片的步骤，这张照片由小米Max3拍摄，未经任何修图。

1.  **调整图片**：进入Snapseed的“调整图片”工具。
2.  **饱和度**：轻微增加饱和度（+10至+20），让色彩更鲜明，但不过度。
3.  **氛围**：适当增加氛围（+20左右），可以提升照片的通透感和立体感。
4.  **对比度**：可轻微增加对比度（+20左右），强化明暗对比，使画面更通透。
5.  **暖色调**：根据个人喜好，微调暖色调（例如+5），让画面偏暖或偏冷。
6.  **突出细节**：可适当增加“结构”（+20左右），让细节更清晰。“锐化”在此例中意义不大，可省略。
7.  **白平衡**：使用“白平衡”工具，微调“色温”（例如-6）和“着色”（+3或+4），让天空色彩更真实、通透。

经过以上几步微调后，照片已经得到改善。**过度调整**反而会失去自然感。

## 第二节：单反级风景照的精细调整

上一节我们处理了手机拍摄的风景照。本节我们来看一张用更高端设备（Redmi Note 7）拍摄的雪山日落照片，学习如何更精细地处理光影细节。

这张拍摄于冷嘎措的贡嘎雪山日落照片，虽然设备价位不高，但在光线充足时画质尚可。后期目标是**呈现更多暗部细节**，同时保持高光区域的层次，实现自然的明暗过渡。

以下是调整步骤：

1.  **调整图片**：
    *   **亮度**：轻微提亮（+30），让整体曝光更舒适。
    *   **饱和度**：加强饱和度，让“日照金山”的效果更金黄，天空更蓝。
    *   **氛围**：增加氛围（+40），提亮暗部，增强通透感。
    *   **阴影**：适当增加阴影（+20），让前景枯草的细节略微显现，避免死黑。
    *   **暖色调**：向正数微调（+5），让整体色调更温暖。
2.  **白平衡**：
    *   增加**色温**和**着色**（均+8左右），可以让日落时的山体更偏向金黄透红的感觉，增强“金山”的视觉效果。
3.  **滤镜选择**：
    *   可尝试添加“粗粒胶片”滤镜。**先将滤镜强度滑到0**，再选择喜欢的样式（如A03）。这能为照片增加一些艺术化的胶片质感。
    *   **慎用**“HDR景观”、“魅力光晕”、“色调对比度”等效果强烈的滤镜，它们容易让照片显得“重口味”和不自然。

**核心要点**：风景照的调整应尽量**保留暗部细节**，避免一片死黑；同时**控制高光**，避免过曝，追求自然的光影过渡。

## 第三节：特殊场景（水面倒影）的调整

上一节我们处理了有复杂光影的雪山照片。本节我们来看一张在印度科钦拍摄的、带有水面模糊效果的日落渔网照片，学习如何处理特殊场景。

这张照片使用iPhone的“实况”模式拍摄，让水面产生了动态模糊效果。后期思路是**校正构图**并**强化色彩**，同时注意保持水面的模糊感。

调整步骤如下：

1.  **旋转**：首先使用“旋转”工具，将不水平的海平面调整至水平。
2.  **调整图片**：
    *   **饱和度**：显著增加饱和度（+40），强化日落色彩。
    *   **氛围**：**注意**：对于“实况”模式拍摄的模糊水面，增加“氛围”可能会让水变清晰，从而削弱动态效果。此例中可选择不加或只加一点点。
    *   **暖色调**：向负数微调（-5），让画面色调偏冷一点，与暖色的日落形成对比。
3.  **突出细节**：
    *   **结构**：可以向负数微调（-20），让水面**更模糊一些**，强化动态感。
    *   **锐化**：可轻微增加，让静态的渔网结构更清晰。
4.  **白平衡**：
    *   增加**色温**和**着色**（均+8），让日落色彩更浓郁，偏向红紫色调。
5.  **滤镜**：
    *   可尝试“粗粒胶片”滤镜，但并非必需。有时经过以上调整，色彩已经足够。

**关键点**：后期调整需要**考虑前期拍摄手法**（如这里的动态模糊）。调整“结构”和“氛围”时，要思考是否与你想保留的拍摄效果相冲突。

## 第四节：VSCO与Snapseed的对比与应用

前面三节我们都在使用Snapseed。本节我们来看看另一款流行软件VSCO，并比较它与Snapseed的不同特点。

VSCO拥有非常丰富的胶片滤镜库，风格多样，但参数调整不如Snapseed精细。它更适合快速赋予照片某种特定的胶片风格。

以下是使用VSCO的简单流程：

1.  **选择滤镜**：首先在大量的滤镜中选择一个喜欢的风格（如F系列）。
2.  **微调参数**：然后进入调整页面，对曝光、对比度、饱和度、清晰度（类似结构）、色调（高光/阴影补偿）等进行微调。
3.  **白平衡与色调分离**：VSCO的白平衡调整相对简单。其特色功能是“色调分离”，可以分别给高光和阴影区域添加不同的颜色倾向。
4.  **暗角**：可以添加暗角，突出画面中心。

**VSCO 与 Snapseed 的核心区别**：
*   **Snapseed**：优势在于**精细的局部调整和强大的控制工具**（如“调整图片”中的“氛围”、“HDR景观”等），更适合对照片进行精确、深入的修饰。
*   **VSCO**：优势在于**丰富的预设滤镜和独特的胶片色彩科学**，更适合快速调色和统一照片风格，培养审美。

**建议**：初学者可以**以Snapseed为主**，掌握基础的光影色彩调整。当需要特定胶片风格或想快速出片时，可以**辅助使用VSCO**。也可以先用Snapseed精细调整，再导入VSCO添加滤镜。

## 总结

本节课中，我们一起学习了风景照片后期处理的核心理念与实操技巧。

我们首先明确了 **“适度调整”** 的原则，避免过度修饰破坏自然感。随后，我们通过三个案例，学习了如何使用Snapseed针对不同场景（手机广角、雪山日落、动态水面）进行有针对性的调整，重点在于**平衡色彩、恢复细节、保持通透**。最后，我们介绍了VSCO软件，并将其与Snapseed进行了对比，让大家了解如何根据需求选择合适的工具。

记住，**前期拍摄永远比后期更重要**。后期只是弥补前期的不足或强化个人风格的手段。掌握一两款软件（如Snapseed），并理解其核心功能，远比盲目尝试大量滤镜和软件要有效得多。
