## 概述

“坐电梯”是一种动态的记录方式，它模拟电梯上升或下降的视角来捕捉画面。这种方式强调**连续性**和**节奏感**，常用于表现时间流逝或空间转换。

---

## 核心概念与公式

这种记录方式的核心在于控制画面的**垂直移动速度**与**帧率**的匹配。一个基础的参考公式如下：

**速度 (v) = 层高 (h) / 时间间隔 (t)**

在代码中，我们可以这样模拟其关键参数：

```python
# 模拟电梯拍摄参数
frame_rate = 24  # 帧率，单位：fps
floor_height = 3  # 假设每层楼高3米
transition_time = 2  # 层与层之间过渡时间，单位：秒

# 计算垂直移动速度
vertical_speed = floor_height / transition_time  # 单位：米/秒
print(f“建议垂直移动速度：{vertical_speed} 米/秒”)
```

上一节我们介绍了核心的速度概念，本节中我们来看看具体需要准备哪些工具。

---

## 所需工具与材料

以下是实现“坐电梯”记录方式所需要的基本工具列表：

*   **摄像机**：具备手动控制曝光和帧率功能的设备。
*   **稳定器**：如三脚架或手持云台，确保垂直运动平滑。
*   **标记物**：用于在场景中标记每层的“停留点”。
*   **计时器**：精确控制每层的拍摄和过渡时间。

准备好工具后，接下来我们进入具体的操作流程。

---

## 操作步骤详解

以下是“坐电梯”记录方式的标准操作步骤：

1.  **场景勘察**：选择具有多层结构的场景，如楼梯间、书架或高楼立面。
2.  **参数设置**：依据上述公式，在摄像机上设置好**帧率**、**快门速度**和**对焦模式**（通常使用手动对焦或小光圈以保持景深）。
3.  **标记点位**：在计划停留的每一层高度做好标记。
4.  **开始拍摄**：启动摄像机，按照预定的匀速在标记点之间垂直移动设备。
5.  **保持节奏**：在每一层标记点短暂停留，模拟电梯停靠，然后再继续移动。

理解了操作步骤后，我们还需要从一些经典作品中汲取灵感。
