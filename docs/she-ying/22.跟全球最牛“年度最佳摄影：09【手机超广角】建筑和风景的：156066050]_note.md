在本节课中，我们将学习手机摄影中一个非常实用的功能——超广角。我们将了解它的工作原理、适用场景以及如何操作，帮助你轻松拍出更具视觉冲击力的建筑和风景照片。

## 什么是手机超广角？

上一节我们介绍了手机全景模式的用法。本节中，我们来看看当你不使用全景模式时，另一个强大的工具：超广角功能。

早期的手机通常是单个摄像头，这个镜头是**广角镜头**，但并非**超广角镜头**。如今，许多手机配备了多个摄像头（例如三个或四个），其中就包含了专门的超广角摄像头。以小米9为例，它拥有三个摄像头，分别覆盖**超广角**、**长焦**和**普通焦段**。

超广角镜头的视角比普通镜头更宽广，能让你在有限的距离内捕捉到更多的场景内容。其核心优势在于，现代手机的超广角镜头**畸变控制**做得很好，能确保建筑线条横平竖直，不会拍得歪歪斜斜。

## 如何开启与使用超广角？

使用超广角功能非常简单。通常在手机的原生相机应用中，你可以在拍摄界面下方直接找到切换按钮（例如 **1x**、**0.5x** 或带有“广角”标识的图标）。点击切换到超广角模式即可。

**操作代码示意（以常见相机UI为例）：**
```
打开相机App -> 在变焦选项中选择“0.5x”或点击“广角”图标 -> 构图并拍摄
```

## 超广角的适用场景

以下是超广角功能最能大显身手的几种情况，我们将通过具体案例来说明。

### 1. 拍摄宏伟建筑

当你站在建筑内部或前方，发现普通镜头无法拍全整个结构时，超广角是你的最佳选择。

**案例演示：** 在一个有圆形穹顶的建筑内，使用普通焦段只能拍到局部，缺乏整体表现力。切换到超广角模式后，能将整个穹顶结构完整纳入画面，视觉效果立刻变得震撼且富有张力。即使站在穹顶正下方，超广角也能轻松覆盖整个圆形，而普通焦段则无法做到。

### 2. 展现广阔风景与室内空间

在体育场馆、音乐厅等宽阔场景，或者面对壮丽的自然风光时，超广角能帮助你记录下更完整的画面。

**案例演示：** 在体育场馆内，普通镜头无法同时拍到左右两侧的观众席。使用全景模式容易失败或变形。此时，切换到超广角，可以轻松将整个场馆顶部的壮观结构全部拍下，画面充满表现力。同样，在拍摄深圳音乐厅等大型建筑夜景时，超广角能让建筑显得更加宏伟震撼。

### 3. 创造独特视觉透视

超广角镜头近大远小的透视特性较强，可以拍出具有冲击力的照片。例如，靠近前景拍摄，可以突出前景的视觉引导作用。虽然边缘物体会产生一些拉伸变形，但这种变形常常是可以接受甚至有益的。例如，用超广角拍摄人像时，可以适度拉长腿部线条。

**重要原则：** 无论使用何种镜头，拍摄建筑时都应尽量保证画面**横平竖直**，这是建筑摄影美感的基础。

## 给单摄像头手机用户的建议

如果你的手机只有一个摄像头，也想体验超广角效果，可以考虑购买外接的超广角附加镜头。

以下是选择外接镜头需要注意的事项：
*   这类镜头通常通过夹子固定在手机原摄像头上。
*   价格较高的镜头（约200-300元）**畸变控制**较好，能保证建筑线条笔直。
*   价格低廉的镜头（几十到一百多元）很可能产生明显的画面弯曲和畸变。
*   长期来看，直接更换一部配备多摄像头和原生超广角的手机更为方便和经济。一部3000元左右的手机通常已具备此功能。

## 本节课总结

本节课中我们一起学习了手机超广角功能。我们了解到，超广角是拍摄建筑和风景的得力助手，它能帮助我们轻松捕捉更宽广的画面，且现代手机的原生超广角镜头畸变控制出色。操作上，只需在相机应用中切换模式即可。对于单摄像头手机用户，可以通过外接镜头实现类似效果，但更换多摄像头手机是更便捷的长期解决方案。
