# 春社CT发麻摄影2.0：第二十九集、平常中的不平常 📸

在本节课中，我们将学习如何捕捉日常场景中的非凡瞬间。课程将涵盖拍摄时长控制、镜头选择、内容规划以及运镜技巧，帮助你提升视频拍摄与剪辑的效率与表现力。

上一节我们介绍了课程主题，本节中我们来看看具体的拍摄时长建议。

拍摄时，每条镜头的时长应相对较短。拍到三到五秒即可，无需一直录制很长时间。

以下是关于剪辑时长的具体建议：
*   在剪辑时可能会纠结如何选择素材，因此不建议拍摄很长的片段。
*   避免拍摄几十秒甚至一分钟的镜头。
*   尽量将每个镜头控制在十秒左右，这样最有利于后期剪辑。

接下来，我们探讨第二个镜头的拍摄方法。

第二个镜头可以拍摄人物的侧脸，然后捕捉他回眸一眼的感觉。

但是人物仍需保持行走状态。

在规划拍摄内容时，常会遇到不知该拍什么的情况。

当你不知道拍什么时，可以多拍摄一些人文风景。

以下是人文风景的具体内容：
*   人文风景指的是周围的**路人**。
*   人文风景也指周围的**环境**。

这些元素在视频中都非常重要。

现在，让我们学习运镜时的关键技巧。

在摇动镜头时，例如这样摇过去，千万不要急促。

你必须懂得缓慢移动。

缓慢移动能为你的镜头增添许多灵感。

例如，什么样的灵感呢？你可以去跟随后期制作的音乐节奏，一步步地卡住节拍。

最后，我们来看一个简单的动作示范。

你靠在这里，然后我说“回头”。

你只需要正常地回头，向上方看去即可。

本节课中我们一起学习了如何高效规划拍摄：控制单个镜头时长在十秒左右利于剪辑；通过侧脸回眸等镜头捕捉动态瞬间；在缺乏灵感时多关注人文与风景素材；以及运用缓慢、跟随音乐节奏的运镜方式来提升画面质感。掌握这些要点，你就能更好地从平常生活中发掘并记录不平常的视觉故事。
