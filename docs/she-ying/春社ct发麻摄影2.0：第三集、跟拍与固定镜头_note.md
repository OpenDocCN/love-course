# 春社CT发麻摄影2.0：第三集、跟拍与固定镜头 📹

在本节课中，我们将学习两种基础的视频拍摄手法：跟拍与固定镜头。理解并掌握这两种技巧，是提升视频叙事能力的关键。

上一节我们介绍了构图与景别的基础知识，本节中我们来看看如何通过镜头运动来讲故事。

## 跟拍镜头 🚶

跟拍是指摄像机跟随被摄主体一起移动的拍摄方式。它能创造强烈的临场感和连续性，常用于表现人物的行动路径或展示环境。

以下是实现平稳跟拍的几个要点：

*   **稳定优先**：尽量使用稳定器或借助身体缓步移动，避免画面剧烈抖动。
*   **保持距离**：与被摄主体保持相对固定的距离，除非有特殊的叙事需求。
*   **焦点跟随**：确保自动对焦锁定主体，或手动跟焦，防止主体变模糊。
*   **路径预判**：提前规划好移动路径，避开障碍物，使运动流畅。

## 固定镜头 🎬

固定镜头是指摄像机机位、光轴、焦距均固定不变的拍摄方式。它强调画面内的主体变化，能传递稳定、客观或压抑的情绪。

固定镜头的核心在于**构图**和**等待**。公式可以简单表示为：
**优质固定镜头 = 精心构图 + 耐心等待决定性瞬间**

以下是有效运用固定镜头的建议：

*   **构图严谨**：按照之前学习的构图法则（如三分法）精心安排画面元素。
*   **利用动静对比**：在静止的画面中，捕捉人物或物体的运动，能突出主体。
*   **承载情绪**：长时间静止的镜头，可以积累情绪，营造特定的氛围。
*   **作为转场**：一组固定镜头剪辑在一起，可以形成干净利落的节奏。

## 两种手法的结合运用 🔄

理解了各自的特点后，在实际拍摄中常常需要混合使用跟拍与固定镜头。

例如，可以用一个固定镜头作为场景的开始，交代环境；然后切入一个跟拍镜头，引入人物并跟随其动作；最后再用固定镜头作为场景的结束，留下余韵。这种组合能使视频段落富有节奏和呼吸感。

本节课中我们一起学习了跟拍与固定镜头这两种核心拍摄手法。跟拍带来动感和引导，固定镜头强调静态和观察。掌握它们并灵活组合，是让你的视频叙事更专业、更生动的第一步。
