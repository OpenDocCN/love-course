## 拍摄前的准备工作

上一节我们介绍了核心概念，本节中我们来看看具体的拍摄前准备。充分的准备是成功抓拍的基础。

以下是拍摄前必须完成的四个检查步骤：

1.  **检查相机设置**：确保使用高速快门（如1/500秒以上）以凝固瞬间，并采用连续自动对焦模式。
2.  **观察光线方向**：让模特侧对或面向主光源，使发丝能勾勒出明亮的轮廓光。
3.  **与模特沟通**：引导模特放松，并告知其你将要捕捉的动作，避免摆拍感。
4.  **寻找简洁背景**：选择干净、不杂乱的背景，使人物主体更加突出。

## 捕捉瞬间的实战技巧

准备工作就绪后，接下来进入实战环节。捕捉撩头发的精髓在于预判和连拍。

以下是三个关键的实战技巧：

*   **预判动作起点**：在模特的手即将触到头发时就开始半按快门跟踪对焦。
*   **使用连拍模式**：启动相机的高速连拍功能，从动作开始到结束进行全程捕捉。
*   **聚焦眼神与表情**：确保对焦点始终锁定在模特的眼睛上，情绪是照片的灵魂。
