# 春社CT发麻摄影2.0：第九集、沙滩与人 🌊📸

在本节课中，我们将学习如何在沙滩环境中进行人像摄影。我们将探讨光线运用、构图技巧以及与拍摄对象的互动方法，以捕捉自然生动的人物瞬间。

上一节我们介绍了城市街拍，本节中我们来看看如何在海滩这一特殊场景中进行创作。

## 光线与曝光控制

沙滩环境光线强烈，反射率高。正确控制曝光是成功的关键。

以下是曝光控制的核心要点：

*   **避免过曝**：沙滩和海水反光强烈，相机自动测光容易使人物脸部曝光不足。建议使用**曝光补偿（+0.7至+1.3 EV）** 或对准人物面部进行点测光。
*   **利用黄金时段**：日出后和日落前一小时的光线柔和，色彩温暖，是拍摄的最佳时间。
*   **巧用逆光**：让太阳位于人物侧后方，可以勾勒出发丝光，营造浪漫氛围。需用反光板或闪光灯为人物面部补光。

掌握了光线控制后，我们需要考虑如何安排画面元素。

## 构图与场景运用

好的构图能让照片主题突出，富有故事感。

以下是沙滩人像的构图技巧：

*   **利用引导线**：海岸线、波浪的痕迹可以作为视觉引导线，将观众视线引向人物。
*   **简化背景**：以干净的天空、大海或沙滩为背景，避免杂乱的游客和船只干扰主体。
*   **尝试不同景别**：结合远景（人物与大海）、中景（半身像）和特写（表情、细节），丰富一组照片的节奏。

构图让画面有了骨架，而人物的姿态与互动则赋予照片灵魂。

## 人物引导与互动

让被拍摄者在镜头前放松自然，才能捕捉到真实的情感。

以下是引导拍摄对象的建议：

*   **设定简单动作**：让模特行走、回头、撩头发或玩水，进行抓拍而非摆拍。
*   **互动与鼓励**：在拍摄过程中给予积极反馈，例如“这个感觉很好”、“刚才的动作非常自然”。
*   **捕捉瞬间**：关注人物大笑、凝视远方或与海浪互动的自然瞬间。

本节课中我们一起学习了沙滩人像摄影的核心技巧：通过**曝光补偿**控制强烈光线，利用海岸线等元素进行构图，并通过引导捕捉人物自然瞬间。记住，技术的最终目的是服务于情感的表达。多练习这些方法，你就能在沙滩上拍出充满活力与美感的人像作品。
