# 📸 春社ct发麻摄影2.0：第二十八集、高空固定镜头拍摄

在本节课中，我们将学习如何独自一人利用三脚架，在高空场景（如天台）完成一条过的慢动作视频拍摄。课程将涵盖拍摄前的准备、动作设计要点以及简化拍摄流程的技巧。

上一节我们介绍了高空拍摄的场景选择，本节中我们来看看如何具体执行一次独立的固定镜头拍摄。

当只有你一个人和一个三脚架时，最佳策略是拍摄一条过的镜头。将视频与音乐搭配，并处理成慢动作效果最为理想。

以下是关于动作幅度的具体建议：
*   动作幅度不宜过大。例如，在冬天可以轻轻甩动围巾，或脱、穿外套。
*   这类小幅度的动作配合慢效果，感觉最佳。
*   若在天台进行大幅度动作，会令单人拍摄变得非常困难。

因此，我在此尽量选择拍摄一条过，时长大约10秒左右。

这种拍摄方式相对简单。你只需要用三脚架固定好机位，对准预定位置。

以下是拍摄前的准备步骤：
*   调整三脚架位置，对准你计划活动的区域。
*   自己预先演练两到三遍。
*   不要认为自己非常有天赋，多练习几次没有坏处。

我们的动作基本上保持一致。你也可以自己尝试变换几个新动作，慢慢切换即可。

本节课中我们一起学习了单人高空固定镜头拍摄的核心方法：选择一条过的拍摄方式，搭配音乐使用慢动作，设计小幅度的优雅动作，并通过提前演练和三脚架辅助来简化拍摄流程，从而独立完成高质量的镜头。
