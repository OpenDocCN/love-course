# 22.跟全球最牛“年度最佳摄影：35【后期修图】用好蒙版，别人：156066050]_note

在本节课中，我们将学习摄影后期处理中一个非常强大且有趣的功能——**蒙版**。蒙版能让你对照片进行精细的局部调整，例如让画面中只有特定部分保留色彩，而其余部分变为黑白。我们将通过两个具体案例，手把手教你如何使用蒙版，让你的照片更具创意和专业感。

---

## 🎨 第一部分：创意色彩分离——让人物“跳”出黑白背景

上一节我们概述了蒙版的作用，本节中我们来看看如何用它实现一个经典效果：让彩色人物置身于黑白背景中。

首先，我们有一张已经初步修好的彩色照片，画面中有一位穿红色衣服的人物，背景建筑色彩丰富。

**第一步：将整张照片转为黑白**

1.  在修图工具中，找到并选择“黑白”滤镜。
2.  滤镜下通常有多种预设（如对比、昏暗、中性等），你可以根据喜好选择，例如选择“对比”。
3.  应用后，整张照片会从彩色变为黑白。

**第二步：进入蒙版编辑界面**

1.  点击右上角的返回箭头图标。
2.  在弹出的菜单中，选择“查看修改内容”。
3.  此时你会看到修图步骤列表，其中包含“原图”和“黑白”两步。这代表你只进行了一步“黑白”调整。

**第三步：使用蒙版“擦”出彩色人物**

我们的目标是只让穿红衣服的人物恢复彩色。

1.  在步骤列表中，点击“黑白”这一步。右侧会出现三个小图标：垃圾桶（删除效果）、画笔（蒙版）、三条杠（更多选项）。
2.  **点击中间的画笔图标**，进入蒙版编辑模式。此时照片可能会暂时恢复彩色。
3.  点击画笔图标旁边的“反相”图标（通常是一个圆形被分成两半的图案），将其激活（通常变为蓝色）。这时你会看到整张照片覆盖了一层半透明的红色薄膜，这层“红膜”就是**蒙版可视化效果**，它代表黑白效果所作用的区域。
4.  点击“红膜”旁边的“眼睛”图标，可以切换显示或隐藏这层蒙版视觉效果。**建议先关闭“眼睛”（变为灰色）**，以便直接观察照片的黑白效果。
5.  蒙版参数的核心是**效果强度**。在屏幕下方，你会看到“黑白”二字和一个强度滑块（数值为0-100）。
    *   `黑白 = 100`：表示该区域为100%黑白效果。
    *   `黑白 = 0`：表示该区域完全无黑白效果，即恢复原始彩色。
6.  **关键操作**：将“黑白”强度滑块调到 **`0`**。然后用手指在屏幕上**涂抹你想恢复彩色的区域——即那位红衣人物**。你会发现，被你涂抹过的地方，颜色回来了。
7.  **精细涂抹技巧**：由于手指较粗，容易涂到背景。这时，**用两根手指在屏幕上做“放大”手势**，将照片局部放大，再进行涂抹，可以精确控制范围。
8.  **纠错方法**：如果不小心将背景也涂上了颜色，只需将“黑白”强度调回 **`100`**，再去涂抹那些错误区域，即可将其恢复为黑白。
9.  建议先精细处理人物边缘，确保轮廓清晰，然后再处理内部大块区域，这样效率更高。

**原理总结**：蒙版就像一块有颜色的玻璃纸（红色薄膜），盖在照片上。我们用画笔（强度为0）在这层“玻璃纸”上把人物部分“擦掉”，于是底下照片原本的色彩就从“洞”里显露出来了。关闭蒙版可视化（眼睛图标）后，我们看到的最终效果就是：人物彩色，背景黑白。

完成涂抹后，点击确认（✓），再点击保存，一张具有创意色彩分离效果的照片就处理完成了。

## 🖼️ 第二部分：局部精细调整——控制不同区域的效果强度

蒙版不仅能做“有/无”的创意效果，更能对任何修图步骤进行**局部强度控制**。上一节我们实现了整体效果的变化，本节中我们来看看如何对照片的不同部分应用不同强度的滤镜。

假设我们有一张风景自拍原图，并已经进行了四步基础调整：`调整图片` -> `突出细节` -> `白平衡` -> `添加“处理胶片”滤镜`。

**目标**：我们希望山体保持“处理胶片”滤镜的100%强度，但天空部分的滤镜强度减弱到25%，让天空颜色更自然。

**操作步骤**：

1.  点击右上角返回箭头，选择“查看修改内容”。在右侧步骤列表中，你可以看到 `原图`、`调整图片`、`突出细节`、`白平衡`、`处理胶片` 这五个步骤。
2.  点击“处理胶片”这一步，然后再次点击中间的**画笔图标**，进入该步骤的蒙版编辑界面。
3.  激活反相图标（变为蓝色），并关闭蒙版可视化（眼睛图标变灰），以便直接观察照片。
4.  此时，下方的强度滑块对应的是“处理胶片”的强度。
    *   `强度 = 100`：表示该区域应用100%的“处理胶片”效果。
    *   `强度 = 0`：表示该区域完全不应用此效果。
5.  **实现局部控制**：
    *   首先，确保强度为 **`100`**，然后涂抹**山体**部分。这样山体就获得了完整的滤镜效果。
    *   接着，将强度滑块调到 **`25`**，然后去涂抹**天空**部分。这样天空就只应用了25%强度的滤镜，效果更淡。
6.  你可以随时切换不同的强度值（如50%、75%），对不同区域进行微调，实现非常细腻的局部控制。
7.  此方法同样适用于“白平衡”、“突出细节”等其他任何修图步骤。你可以为照片的每个局部“定制”修图参数。

完成所有调整后，点击确认并保存即可。通过蒙版，你不再是给整张照片一个统一的“处方”，而是成为了照片的“局部医生”，可以精确地强化或减弱每一处的效果。

## 📝 本节课总结

在本节课中，我们一起深入学习了蒙版这一高级后期技能：

1.  **蒙版是什么**：它是一个局部编辑工具，允许你对照片的特定区域应用或移除修图效果，而不影响其他区域。可视化状态下，它像一层有色薄膜。
2.  **蒙版的核心操作**：通过**画笔**和**强度滑块** (`0` - `100`) 的组合，控制效果在局部区域的有无及强弱。
3.  **蒙版的两大应用**：
    *   **创意效果**：如“色彩分离”，让主体突出。
    *   **局部精细调整**：对画面的不同部分应用不同强度的滤镜或调整参数，使后期处理更具层次感和专业性。
4.  **操作技巧**：善用**双指放大**进行精细涂抹，利用**强度滑块**和**反相功能**灵活控制与修改。

掌握蒙版，意味着你掌握了从“整体调色”到“局部精修”的关键飞跃，能够创造出更具个性和视觉冲击力的作品。
