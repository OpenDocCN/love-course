# 摄影课程 22：跟全球最牛“年度最佳摄影师”学摄影！（完结） 🏆

## 14【练就摄影眼】摄影视觉构成二 👁️

在本节课中，我们将学习摄影观察的第二个核心概念：**将事物看作线条**。我们将探讨如何通过识别和拍摄线条，来构建更具美感和引导性的画面。

上一节我们介绍了“图形感”的概念，即把事物看作图形来观察和拍摄。本节中，我们来看看如何将事物看作**有线条感的物体**，并掌握拍摄线条的方法。

### 什么是“线条感”？

“线条感”是指忽略物体的具体材质、功能等属性，仅从其形状和轮廓中抽象出线条元素进行观察。许多日常事物都可以被看作线条。

以下是生活中常见的、可被视为线条的物体示例：
*   公路与斑马线
*   电线杆
*   窗帘的褶皱
*   叶子的脉络

当我们学会以“线条”的眼光看待这些物体时，我们的拍摄思路就会发生转变，从记录事物本身转向构建画面的形式美感。

### 如何拍摄线条？

拍摄线条的核心要点是**追求延伸感，避免断裂感**。延伸感能让观众的视线沿着线条移动，使画面更舒展、更有深度。

以下是拍摄线条时需要遵循的几个关键原则：

1.  **保持横平竖直**：当线条本身是水平或垂直时（如建筑围栏），拍摄时应利用手机的九宫格辅助线，确保线条在画面中也是水平或垂直的。这能带来稳定、规整的视觉效果。
    *   **操作提示**：`打开手机相机设置中的“网格”功能。`

2.  **利用对角线构图**：让线条从画面的一角延伸至对角。这种构图方式能最大化地利用画面对角线长度，创造出强烈的动感和延伸感。
    *   **视觉公式**：`线条路径 ≈ 画面对角线`

3.  **确保线条完整**：拍摄时，尽量让重要的线条在画面内保持完整，不要被边框切断。断裂的线条会让人感到压抑，并中断视觉的流畅性。
    *   **核心思想**：`完整线条 > 断裂线条`

4.  **简化画面**：拍摄线条时，应尽量避开杂乱元素（如无关的树木、行人）。简洁的背景能让线条主体更加突出。
    *   **拍摄逻辑**：`线条主体 + 简洁背景 = 有力画面`

5.  **尝试黑白滤镜**：对于光影形成的线条（如栏杆的影子），使用黑白滤镜可以剥离色彩干扰，让线条的形态、对比和质感更加突出。

### 实践案例分析

让我们通过具体案例，来理解上述原则如何应用。

**案例一：安集海大峡谷**
在拍摄这张风景照时，观察的步骤是：
1.  将整座山看作**不规则的图形**。
2.  将盘山公路、河床、山脊褶皱都看作**线条**。
拍摄的关键在于确保**公路这条核心线条的完整性与延伸感**。如果公路在画面中被截断，画面的纵深感和引导性就会大大减弱。网络上的“S型构图”本质也是为了营造这种蜿蜒的延伸感。

**案例二：怒江七十二拐**
使用全景模式拍摄时，目的是将每一个转弯的路径都完整纳入画面。未经训练的拍摄者很容易在画面中“切断”这些道路线条。经过“线条观察”训练后，你会自然地将山脊、梯田、公路都视为线条，并本能地追求其完整与延伸。

**案例三：城市旋转楼梯**
拍摄旋转楼梯时，不应只拍局部。要让镜头展现楼梯**旋转上升或下降的过程**，利用其曲线形成视觉延伸，让观者感到舒适流畅，而非突兀压抑。

### 核心概念总结与作业

本节课我们一起学习了摄影观察的第二个核心概念：**将物体看作线条**。其拍摄要义是**追求延伸感，避免断裂感**。

请记住，构图技巧是死板的，而**观察事物的本质**——比如将其抽象为图形或线条——才是摄影思维的基础。先学会“看”，才能学会“拍”。

**本节课的作业是观察练习：**
请你在生活中主动寻找具有“线条感”的物体，例如电线、叶脉、桌布纹路、建筑轮廓等。不需要立即拍摄，而是用“摄影眼”去观察，尝试在脑海中将它们抽象为线条。这是培养你观察能力的重要一步。
