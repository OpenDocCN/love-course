# 22.跟全球最牛“年度最佳摄影：33【后期修图】学会剪裁，废片：156066050]_note

在本节课中，我们将学习摄影后期中一个非常实用的技巧——剪裁。通过剪裁，我们可以重新构图，突出照片主体，让原本平淡甚至杂乱的画面变得简洁有力。即使拍摄时受到设备或环境的限制，合理的剪裁也能拯救一张照片。

---

## 剪裁的核心作用

上一节我们介绍了建筑修图的基本方法，本节中我们来看看如何通过剪裁进行“第二次构图”。

剪裁的核心作用是**在不损失单位面积像素质量的前提下，重新选取和突出画面中的重点部分**。这与直接使用数码放大拍摄有本质区别：

*   **数码放大拍摄**：在拍摄时通过手机软件放大画面，会导致像素损失，画质下降。公式可理解为：`最终画质 = 原始像素 / 放大倍数`，放大倍数越大，画质损失越严重。
*   **后期剪裁**：先以最高像素拍摄完整画面，后期再截取所需部分。这个过程是**选择**而非“拉伸”，因此能最大程度保留选中区域的原始画质。

**结论是：在拍摄时，应尽量避免使用数码放大；若构图不理想，宁愿先拍全，再通过后期剪裁来优化。**

## 剪裁实战：突出框架中的主体

以下我们通过具体案例来学习剪裁的操作与思路。

### 案例一：广州塔与雕塑框架

我拍摄了一张广州塔位于雕塑框架中的照片。原始画面包含天空、树木和其他杂乱元素，导致主体“小蛮腰”不够突出。

**操作步骤如下：**

1.  在修图软件（如Snapseed）中，点击【工具】选项。
2.  选择第一排第五个【剪裁】工具。
3.  选择剪裁比例。**不建议初学者直接使用“自由”比例**，因为难以控制画面美感。最常用的是 **4:3** 或 **1:1**（正方形）。
4.  我尝试了1:1比例，将画面剪裁为正方形，使主体更突出。
5.  但我觉得还不够，于是撤销后，选择4:3比例进行更激进的剪裁。
6.  我不断缩小剪裁框，最终只保留了雕塑框架和内部的广州塔。这样，主体变得极其鲜明，画面非常简洁。

**核心要点：** 剪裁时，要敢于做减法，只保留最核心、最能表达主题的元素。

## 剪裁与多倍光学变焦的对比

上一节我们完成了对单一主体的剪裁，本节中我们来对比剪裁与手机多倍光学变焦的效果。

我使用具备2倍光学变焦的小米9拍摄了同一场景。对比发现：
*   **直接拍摄后剪裁**：画面简洁，但经过大幅剪裁后，细节有一定损失。
*   **使用2倍光学变焦拍摄**：直接获得了简洁构图，画质更清晰，细节更丰富。

**这是因为光学变焦是通过镜头物理结构实现放大，而非软件算法插值。**

因此，**如果您的手机配备多倍光学变焦镜头，应优先使用它进行拍摄，这比后期大幅剪裁能获得更好的画质。** 剪裁更适用于拍摄时无法变焦或变焦后构图仍不理想的情况。

## 剪裁比例的选择艺术

学会了基本操作后，我们来看看如何为不同的照片选择合适的剪裁比例。

### 案例二：印度街头的黄色房子

我拍摄了一面在阳光下非常漂亮的黄色墙壁，与悬挂的衣物形成对比。原图为4:3，但直接剪裁效果不佳。

以下是尝试不同比例的过程：

1.  首先使用【透视】或【视角】工具将建筑拉正。
2.  尝试4:3比例剪裁，发现上下多余部分（如铁门）破坏了画面的色彩纯粹感。
3.  观察画面，黄色色块更接近正方形。于是切换为 **1:1**（正方形）比例。
4.  移动剪裁框，精确框选最纯粹的黄色墙面区域，去除顶部和左侧的干扰物。

最终，这张照片仅通过剪裁就达到了很好的效果，几乎无需其他调色。**这说明剪裁比例没有定式，需根据画面主体的形状和美感来决定。** 常用的比例有1:1、4:3、16:9等，需多加尝试。

## 创意剪裁：旋转与倒影

剪裁不仅能修正构图，还能创造新的视觉效果。

### 案例三：乌代布尔湖面倒影

我拍摄了一张包含城堡、鸽群及其湖面倒影的照片。原图水平线歪斜，且画面元素较多。

**创意剪裁步骤如下：**

1.  使用【旋转】工具，软件自动将水平线拉直。
2.  进行 **180度旋转**，将倒影部分置于上方。这样，真实的景物在下，倒影在上，形成一种超现实的趣味感。
3.  使用 **1:1** 比例进行剪裁。舍弃大部分天空和地面，只保留鸽子腿部与倒影连接的部分，以及一个人物倒影。
4.  这样的画面会引发观者思考：“哪边是真？哪边是假？”，增加了照片的创意和故事性。

**这个案例表明，剪裁结合旋转，可以化平凡为神奇，将普通的倒影照片转变为充满想象力的作品。**

## 培养剪裁的直觉

最后，我们通过一个随手拍案例，学习如何通过剪裁拯救废片。

### 案例四：水波纹与建筑倒影

这是一张非常普通的随手拍，画面歪斜，主体不明。我想表现的是日落时分建筑的倒影与水波纹结合的意境。

**拯救步骤：**

1.  **旋转与拉直**：先用【旋转】和【透视】工具将建筑调正。
2.  **尝试剪裁**：先尝试1:1比例，保留水波纹和倒影，效果尚可。
3.  **进一步精简**：感觉两侧建筑仍是干扰。改用4:3比例，**大幅剪裁**，只保留单个建筑及其在水中最具抽象感的倒影部分。
4.  **后期增强**：由于剪裁范围很大，像素有损失，因此需适当增加【锐化】和【清晰度】。最后套用一个合适的滤镜，强化光影和色彩氛围。

最终，一张杂乱的随手拍变成了一幅具有抽象意境的画面。**这需要我们在日常中多观察、多练习，逐渐培养“哪些该留，哪些该舍”的视觉直觉。**

## 本节课总结

在本节课中，我们一起学习了摄影后期中至关重要的剪裁技巧：

1.  **剪裁的本质**：是“第二次构图”，能在不损失画质的前提下优化画面。
2.  **核心原则**：敢于做减法，突出主体，追求简洁。
3.  **操作关键**：慎用自由比例，多尝试1:1、4:3等固定比例。
4.  **与变焦的关系**：优先使用光学变焦拍摄，剪裁作为补救和创意手段。
5.  **创意延伸**：剪裁可与旋转等工具结合，创造出新颖的视觉效果。
6.  **直觉培养**：剪裁能力来源于大量的观察和实践，要明确每张照片的表达重点。

记住，**剪裁的目的不是掩饰拍摄失误，而是主动地、创造性地强化你的视觉表达。** 拿起你的照片，现在就开始剪裁练习吧！
