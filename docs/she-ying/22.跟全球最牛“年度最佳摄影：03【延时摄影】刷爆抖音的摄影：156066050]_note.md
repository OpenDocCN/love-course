## 什么是自动曝光/自动对焦锁定？

自动曝光与自动对焦锁定是一项隐藏功能。它的作用是，当你长按屏幕上的某个对焦点时，手机会固定当前的曝光值和对焦距离。这意味着，即使你移动手机或场景光线变化，相机也不会重新测光或对焦。

**核心操作公式**：
`长按屏幕对焦点 → 触发“自动曝光/自动对焦锁定”`

以下是锁定功能的基本演示步骤：

1.  打开手机相机，对准一个包含远近不同物体的场景（例如，近处的胶卷和远处的公仔）。
2.  点击屏幕上近处的物体（如胶卷）进行对焦和测光。
3.  **用手指长按**该对焦点，直到屏幕上方出现“自动曝光/自动对焦锁定”或类似提示（如一个锁形图标）。
4.  锁定后，即使大幅度移动手机，对焦点和曝光值也会保持不变。

## 锁定功能解决了什么问题？

在旅行或抓拍时，我们常使用连拍来捕捉飞鸟或人物转瞬即逝的瞬间。如果每拍一张都要重新点击对焦、拉动曝光滑块，效率会非常低。

锁定功能的核心价值在于**提升拍摄效率与一致性**。以下是它的两个主要优势：

*   **保持曝光一致**：锁定后，即使拍摄多张照片，曝光参数完全相同，适合需要统一色调的系列拍摄。
*   **固定对焦距离**：锁定的是清晰的物距范围，而非具体物体。只要主体在这个距离范围内移动（如行人走过），都能保持清晰，便于进行高速连拍抓拍。

## 不同手机品牌的锁定功能差异

需要注意的是，此功能在不同品牌手机上的实现方式有所不同。

**iPhone、小米、OPPO、vivo、三星等**：
操作逻辑一致，即`长按对焦点锁定`。锁定后可以连续拍摄多张参数一致的照片，实用性很强。

**华为、努比亚等**：
它们提供的是“对焦与测光分离”功能。长按对焦点会出现两个可独立拖动的圆圈：
*   **外圈（对焦点）**：决定画面哪里清晰。
*   **内圈（测光点）**：决定画面哪里曝光准确。

然而，这种分离功能在**拍完一张照片后会自动解除**，无法像iPhone那样在锁定状态下连续拍摄，因此在抓拍等需要连续性的场景中实用性较弱。

## 锁定功能的实战应用场景

掌握了基础原理后，我们来看看这个功能在具体拍摄中如何大显身手。

### 应用一：拍摄唯美光斑效果

想将远处的灯光拍成朦胧浪漫的光斑吗？锁定功能是关键。

**操作步骤**：
1.  将手机靠近一个近处物体（如你的手），距离大约10厘米。
2.  点击屏幕对焦在近处物体上，并**长按锁定**对焦和曝光。
3.  将手机移向远处的灯光场景。由于对焦距离被锁定在近处，远处的灯光自然会虚化成光斑。
4.  根据光斑亮度，适当向下拉动曝光滑块（小太阳），防止过曝，然后拍摄。

### 应用二：街头人文抓拍

在街头预见一个有趣的场景框架时，锁定功能能帮你从容等待“决定性瞬间”。

**拍摄思路**：
1.  预先观察并构好图（例如，一个颜色鲜艳的挂毯作为框架）。
2.  对焦在画面中你期望人物出现的位置，并**长按锁定**对焦和曝光。
3.  保持姿势，耐心等待有趣的人物进入你的构图框架。
4.  使用连拍功能，捕捉人物最自然或最具特征的瞬间。
