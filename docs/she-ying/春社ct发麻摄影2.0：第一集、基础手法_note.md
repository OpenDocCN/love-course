# 📸 春社CT发麻摄影2.0：第一集、基础手法

在本节课中，我们将要学习摄影的基础手法。这些手法是构建优秀照片的基石，掌握它们能帮助你更好地表达创意，捕捉动人瞬间。

上一节我们介绍了课程概述，本节中我们来看看具体的基础摄影手法。

## 核心概念：曝光三角

摄影的基础在于控制光线进入相机的量，这主要由三个参数决定，它们的关系可以用一个核心公式来描述：

**曝光量 = 光圈 × 快门速度 × ISO感光度**

这个公式意味着，要获得一张曝光正确的照片，你需要平衡光圈大小、快门速度和ISO感光度这三个要素。

## 基础手法详解

以下是构成摄影基础的几个关键手法。

1.  **光圈控制**
    光圈是镜头内控制进光量的孔洞大小。它用f值表示，如f/1.8、f/8。**f值越小，光圈越大，进光量越多，背景虚化效果越强（景深浅）**。反之，f值越大，光圈越小，进光量越少，背景更清晰（景深深）。

2.  **快门速度**
    快门速度是相机感光元件曝光的时间长短。它用分数或秒表示，如1/500s、2s。**快门速度越快，越能凝固瞬间动作，进光量越少**。快门速度越慢，则能记录运动轨迹，进光量越多，但需要三脚架防止画面模糊。

3.  **ISO感光度**
    ISO感光度是相机感光元件对光线的敏感程度。**ISO值越高，对光越敏感，能在暗光下拍摄，但画面噪点会增加**。ISO值越低，画质越纯净，但需要更多光线。

4.  **构图原则**
    构图是安排画面元素的方式。常见的法则有三分法、引导线、对称构图等。目的是让画面更平衡，主体更突出，引导观众视线。

5.  **对焦清晰**
    确保拍摄主体清晰是对焦的目的。你可以使用相机的自动对焦功能，或手动调整镜头对焦环，使主体轮廓分明。

了解了各项基础手法后，我们需要知道如何将它们结合起来运用。

## 实践应用：模式选择

在相机上，你可以通过不同的模式来应用这些手法：

*   **光圈优先模式（A/Av）**：你设定光圈值，相机自动匹配快门速度。适用于控制景深，如拍摄人像（大光圈）或风景（小光圈）。
*   **快门优先模式（S/Tv）**：你设定快门速度，相机自动匹配光圈值。适用于捕捉运动（高速快门）或拍摄流光轨迹（慢速快门）。
*   **手动模式（M）**：完全由你控制光圈、快门和ISO。适用于复杂光线或追求特定创意效果。
