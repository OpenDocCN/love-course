# 📸 春社CT发麻摄影2.0：第三十八集、天台建筑风景

在本节课中，我们将学习如何在天台环境中进行建筑与风景摄影。课程将涵盖从观察环境、构图技巧到安全拍摄的完整流程，帮助你利用简单的设备捕捉富有故事感的画面。

上一节我们介绍了环境观察的重要性，本节中我们来看看具体的拍摄步骤与构图方法。

首先，你需要观察天台四周，寻找可以拍摄的画面。例如，围栏上的结构或物体可以作为拍摄元素。周围环境可能显得杂乱，但这并不重要。你可以在后期处理中将照片调整为黑白，杂乱的元素会因此转变为具有美感的画面。

接下来是具体的拍摄操作指南。

以下是拍摄时需要执行的步骤列表：

1.  第一个镜头：你可以让模特用手扶着或拉着围栏等结构。
2.  第二个镜头：尽量营造出一种类似偷拍的感觉。拍摄时务必注意天台安全。
3.  设备使用技巧：如果你的手机无法放置在理想位置进行取景，可以切换到前置摄像头。将手机直接放置过去，就能通过前置摄像头看到对方的画面。

通过以上步骤，你可以在保证安全的前提下，完成在天台环境中的创意拍摄。

本节课中我们一起学习了天台摄影的观察方法、构图思路以及具体的拍摄与设备操作技巧。关键在于发现环境中的结构之美，并通过视角与后期处理将其强化，最终创作出富有情绪和故事感的作品。
