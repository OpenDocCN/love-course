# 艾伦聊天方法《傻瓜聊天课》：012.案例四成品.mp4

🎼大家好，今天我们将通过一个聊天案例来讲解如何在微信聊天中吸引女神。首先，让我们来认识一下女主角，她是通过搭讪认识的一位女生。

**搭讪与初次接触**
- 我通过搭讪认识了她，但由于微信好友众多，一直未能与她聊天。
- 一天凌晨2点42分，她给我发消息询问为何没有出来，并发送了一个表情包。

**闺蜜的介入**
- 她提到自己曾与另一个女生一起被邀请去酒吧，但被我拒绝。
- 她的闺蜜（女主角）因此开始主动与我聊天。

**女生的需求与态度**
- 她询问我是否玩得开心，我回应说忘了我们是如何认识的。
- 我表示在找猫的铲屎官，并邀请她有空一起玩。

**高价值男人的策略**
- 我没有急于见面，因为高价值男人不会因为一个女生的突然邀约而慌张。
- 我回应她的邀约，表示有空再一起玩。

**女生的态度转变**
- 两天后，她给我发了一个表情包，我回复了一个问号。
- 她的态度从热情转变为冷淡，并表达了对被忽视的不满。

**废物测试与回应**
- 她说最恨别人发信息后不再回复，这实际上是一种废物测试。
- 我没有道歉，而是以幽默的方式回应，避免了降低自己的价值。

**调情与话题开启**
- 我们开始调情，并从她的朋友圈照片展开话题。
- 她分享了一些其他照片，但我没有立即回应。

**邀约与距离问题**
- 我邀请她喝酒，她询问是否需要带身份证。
- 她通过提问来测试我的态度，我以同样的问题回应，使她软化。

**直播与活动**
- 我提到要直播，并询问她的位置。
- 由于距离较远，我决定不赴约。

**高价值男人的聊天技巧**
- 我通过询问她是否经常在郑州活动，展示了自己的高价值。
- 我回应说“跟我很搭”，这是一种高价值男人的聊天技巧。

**总结**
- 本节课中，我们学习了如何通过聊天技巧吸引女生，并展示了高价值男人的沟通方式。
