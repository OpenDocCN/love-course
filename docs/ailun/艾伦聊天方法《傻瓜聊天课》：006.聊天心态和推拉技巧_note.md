# 艾伦聊天方法《傻瓜聊天课》：006.聊天心态和推拉技巧

🎼好，兄弟们，今天微信聊天的第六节课主要教授正确的聊天心态以及高阶的聊天技巧。在跟女生聊天前，了解并调整好心态至关重要。

**心态调整的重要性**

为什么难以调整心态？简单来说，传统观念中女生被视为稀缺资源，许多人认为自己不够优秀，尤其是面对漂亮的女生时。这种心态导致在与女生聊天时，我们的姿态低于对方，从而陷入迎合讨好的误区。

**自信的重要性**

女生喜欢有态度的男人，而非只会讨好的人。许多人迎合女生，是因为希望得到积极的反馈。然而，这种心态阻碍了与女生建立良好的互动。

**如何提升自信**

首先，接受并包容自己。不要因为收入低、外貌不佳而自卑。大多数男人长相相似，自信源于自我接纳。

**平等对待女生**

与女生聊天时，应保持平等，避免低价值的心态。想象自己是一个优秀的人，与女生分享生活状态，而非只关注她的反应。

**情圣思维**

运用情圣思维，想象自己是高价值的人，与女生聊天时不过分讨好，让女生围绕自己。

**女生的心态**

女生希望找比自己优秀或同阶层的伴侣。过分讨好会让她觉得你低价值。

**推拉技巧**

推拉技巧基于自信，可调动女生情绪。例如，先拉后推：询问星座，然后说星座不准；先推后拉：发朋友圈截图，然后说看了好几遍。

**作业**

思考如何与女生进行微拉。
