# 艾伦聊天方法《傻瓜聊天课》：006.聊天心态和推拉技巧_note

在本节课中，我们将要学习如何建立正确的聊天心态，并掌握高阶的推拉技巧，以提升与女生互动的质量。

## 一、摆正聊天心态

在开始与女生聊天之前，必须摆正自己的态度，拿捏好心态。

许多人在聊天时姿态过低，表现为迎合、讨好甚至跪舔。这源于一个根本错误观念：认为女生是稀缺资源，尤其是漂亮的女生，并因此认为自己不够优秀。

当你以低于对方的姿态去交流时，就丧失了自我框架。女生喜欢的是有态度、有原则的男人，而非只会讨好的人。

这种讨好行为往往源于渴望从对方那里获得“好的回应”。例如，在发送消息前，会担心对方不回复或删除自己。这种心态会阻碍你建立自然、良好的互动。

如果你的聊天过于平淡或迎合，女生会觉得与你聊天很无聊。

## 二、如何建立自信心态

以下是建立自信心态的关键步骤。

首先，要对自己足够接纳和包容。有些兄弟可能因为月薪不高、长相普通或没车而感到自卑，并认为配不上漂亮的女生。

这种不自信会在聊天中流露出来，让女生感知到你是一个低价值的男人。

增强自信最直接的方法就是完全接纳自己。从客观角度看，大多数男性的外貌基础相差无几。如果有女生评价你不够帅，那可能是她的审美问题，而非你的问题。

当你能够充分接纳和包容自己后，自信便会油然而生。一个自信的男人在聊天时，不会过分渴望女生的即时回复或积极反馈。

## 三、享受聊天过程

与女生聊天，最重要的是享受过程本身，而非仅仅关注文字表面的信息传递。

如果只停留在字面意思，聊天时会感到彷徨和迷茫。例如，当女生分享她的生活状态时，许多兄弟会不知所措，思考如何帮她解决问题。

之所以会这样想，是因为心态处于较低位置。必须明白，聊天双方是平等的。

如果你对女生说：“像你这样的女生能跟你聊天，我就感觉很幸福了。”这种姿态会让女生认为你是低价值男人，觉得你配不上她。

因此，需要培养“情圣思维”：想象自己是像吴彦祖或王思聪那样优秀、高价值的人去聊天。

## 四、高手与新手的区别

高手与新手聊天最显著的区别在于心态。

新手总想迎合、讨好女生，特别害怕得到不好的反应。而高手聊天时，会让女生围绕自己转，不会过分讨好对方。

女生能清晰感知到一个自信男人与一个不自信男人之间的差别。

记住：你把女人当女神，她就把你当屌丝；你把女人当普通人，她才会把你当男神。

## 五、理解女生的需求

一个漂亮的女生寻找男朋友时，通常希望对方比自己优秀，或至少处于同一阶层。

如果你在聊天中过分讨好、迎合，将她地位放得很高，女生会感受到你的不自信，并在潜意识里判定你为低价值男人。

高手聊天时，不会特别在意女生的即时反应。我们始终让对话围绕自身展开，经常向女生分享自己的生活状态。

女生能感受到你是一个自信、优质的高价值男人。因此，聊天一定不要围着女人转。

如果不知道如何回复某句话，就不要再延续那个话题。如果女生分享了她的状态，你不必围绕她的话题展开，可以同样分享一张照片或你的状态。

有兄弟问：如果女生反应不好怎么办？反应不好通常不是因为你说了某句话，而是因为她对你这个人不感兴趣。

例如，如果你的朋友圈照片质量差、个人形象糟糕，女生通过这些内容判定你是低价值男人，那么她对你发送的任何消息都可能不屑一顾。

问题出在你这个人整体的展示上，而非某一句聊天内容。

## 六、推拉技巧详解

在讲解推拉技巧前，必须再次强调前提：自信。只有自信的男生，才敢于调动女生的情绪，勇敢地进行调侃。

良好的推拉技巧能让女生觉得聊天很有趣，使她的内心像坐过山车一样产生跌宕起伏的情绪。

**推拉**的核心是制造情绪波动：先表达欣赏或拉近（拉），再表达调侃或推开（推），或者反之。

以下是两个具体例子，帮助理解此概念。

**例子1：先拉后推**
当聊到星座时，你问女生是什么星座。她回答是白羊座。
你可以说：“我是天秤座，天秤座和白羊座很配哦。”（这是“拉”）
女生可能会说“谢谢”。
接着你可以说：“不过一般星座都不太准。”（这是“推”）
这就完成了一个简单的先拉后推。

**例子2：先推后拉**
你截取女生朋友圈的图片发给她，并说：“怎么朋友圈里全是自拍，真臭美。”（这是“推”）
女生可能会生气，回复“不想看可以不看”。
此时你再说：“害得我每一张都看了好几遍。”（这是“拉”）
这就完成了一个典型的先推后拉。

以上例子旨在打开思维。这种推拉式的聊天方式，在日常生活中很容易找到应用场景。
