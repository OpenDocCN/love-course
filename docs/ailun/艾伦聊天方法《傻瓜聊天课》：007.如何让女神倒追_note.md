# 艾伦聊天方法《傻瓜聊天课》：007.如何让女神倒追_note

在本节课中，我们将要学习如何在与高价值女性互动时，建立并保持自己的框架，从而吸引对方，甚至让她对你产生主动的兴趣。核心在于理解并运用“框架”这一概念，通过自信的态度和正确的应对策略，扭转互动的态势。

## 理解“框架”的力量

上一节我们介绍了自信的重要性，本节中我们来看看什么是“框架”。框架是你在互动中持有的根本信念、态度和立场。它决定了你如何看待自己，以及你允许别人如何对待你。

一个漂亮、高收入的女性在与男性聊天时，她的框架通常是：“我很优秀，你们应该来追求和讨好我。”这就是她的强势框架。

许多男性在与此类女性聊天时，自身没有框架，或者框架不够强硬，导致自己的立场被对方的框架吞噬。结果就是，他们会不自觉地开始迎合和讨好对方。

## 建立你的强势框架

那么，一个高价值的、受过训练的男性，面对漂亮女性时应持有怎样的框架呢？关键在于以下三点：

以下是构建强势框架的三个核心要点：

1.  **自信**：坚信自我价值。你不过是一副好看的皮囊，而我是一个自信的男人。
2.  **平常心**：视对方为普通女性。她除了外貌，并无特别之处。
3.  **无需求感**：不害怕失去。`即使你反应冷淡，我还有其他选择。`

当我们去面对这些非常漂亮、非常有框架的女生的时候，我们要保持的就是这样的框架。这源于你的自信，而自信又源于你自身的优秀和丰富的选择。女生会在潜意识里认为你是一个高价值男人。

相反，一个一直在讨好女生的男人，会被认为是低价值的。一个有态度的男人，在女生看来，意味着他平时选择很多。

## 从框架到吸引：让女生倒追的前提

如何让女生来倒追你的前提，可以总结为以下四点：**有态度、有框架、要自信、不害怕失去**。

当你具备了这些，再去跟女生互动，效果就会截然不同。例如，当你想邀约女生，她回复“你是不是想睡我”时，很多兄弟会陷入纠结，思考“是”或“不是”哪个答案更好。

当你在思考该发什么样的话才能得到好的反应时，其实你就已经掉入了女生的思维陷阱。最好的方式是把问题反抛给对方：**“怎么了？你觉得我约你就是想睡你吗？”**

## 展示标准：框架的主动表达

在拥有了框架之后，该如何让女生来倒追你呢？框架的另一个重要表现形式，在于你要让女生知道你是一个**有标准的男人**。

以下是让女生知道你拥有标准的方法：

*   在聊天中主动询问：**“你会做饭吗？如果我们谈恋爱，你愿意做饭给我吃吗？”**
*   提出你的期待：**“你会跳舞吗？你会弹钢琴吗？”**

这样的聊天会让女生产生一种她在追逐你的感觉。当女生连续问你几个问题时，你可以反问：**“你问我这个问题，是不是喜欢我？”** 或 **“你准备好追我了吗？”**

核心是保持自己的标准和框架，不落入对方的节奏。

## 应对“废物测试”

女生为什么会提出“废物测试”呢？要知道，当我们试图展示高价值和自信，并想让女生倒追时，女生就会想测试这个男生是否真的值得、是否真的自信。

例如，女生问：**“你谈过几个女朋友？”** 如果你开始纠结答案，甚至求助朋友，隔了很久才回复“三个”，那么即使答案本身没问题，女生也能感受到你的不自信。因为她会觉得，一个简单问题都需要思考这么久，你是一个低价值的男人。

女人想要的往往不是具体的答案，而是测试你面对问题时的**态度**。只要你的回复是自信、笃定、勇敢的，就足够了。

以下是几个应对废物测试的万能回复示例：

*   **问题**：为什么跟前女友分手？
    *   **回复**：这是一个悲伤的故事，有机会再讲给你听。
*   **问题**：你经常撩妹吗？
    *   **回复**：没有呀，通常我才是被撩的那一个。
*   **问题**：你喜欢我什么？
    *   **回复**：可能是因为时机吧。喜欢一个人有时难以形容，就像你无法解释一个笑话为什么好笑，可能只是一个眼神或一次心跳。
*   **特别提醒**：当女生说 **“我给你介绍个女朋友吧”** ，千万注意。这通常是强烈的嫉妒心和测试。她并非真心想介绍，而是在测试你是否对她感兴趣。一定不要相信，并巧妙化解。

## 废物测试应对策略盘点

最后，我们来整理和盘点面对女生废物测试时的核心策略。

以下是两种核心应对方法：

1.  **忽视法**：当你不知道如何回复时，直接忽略这个问题，假装没看到，开启下一个话题。
2.  **反问法**：把问题抛回给对方。
    *   例如：“你是不是想睡我？” -> **“你觉得我跟你聊天，就是为了想睡你吗？”**
    *   例如：“你谈过几个女朋友？” -> **“你觉得我谈过几个？你猜猜看。”**

我们一定要给对方传递出轻松、自信、幽默的态度，而不是把注意力纠结在问题本身。
