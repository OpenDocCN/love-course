# 艾伦聊天方法《傻瓜聊天课》：007.如何让女神倒追

🎼好，兄弟们，今天第七节课，我们来探讨如何让女神倒追你。是不是很兴奋，想要了解接下来的内容？在讲解今天的内容之前，先分享一个小秘密。

**为什么高手能吸引女生？**

高手与女生聊天时，背后有一个强大的框架。什么是框架？比如一个漂亮、收入高、精致的女生在与男生聊天时。

**女生的框架**

这个女生的心态可能是：“我很漂亮，腿很长，五官精致，收入也很高。你们男人应该赶紧追求我，讨好我。”她会认为自己是一个高价值的女人。

**你的框架**

很多兄弟在与这样的女生聊天时，没有自己的框架，容易被对方影响。所以，我们要自信，保持强硬的框架。

**自信与框架**

自信的男人在面对高价值或框架强的女生时，会认为对方只是外表好看。我们要自信，看透本质，知道对方只是普通女生。

**如何让女生倒追你？**

1. 自信
2. 强硬的框架
3. 不害怕失去对方

**应对女生的废物测试**

女生提出废物测试，是为了测试你的价值、自信程度。以下是一些应对废物测试的答案：

1. **分手原因**：这是一个悲伤的故事，有机会再讲给你听。
2. **撩妹**：没有呀，一般我通常是被撩的那一个。
3. **喜欢你的原因**：可能是因为时机吧，我觉得有的时候喜欢一个人，她可能难以用言语去形容，就像你无法去形容一个笑话，为什么好笑，可能只是因为一个眼神或者是一次心跳。

**总结**

本节课中，我们学习了如何让女神倒追你。关键在于自信、强硬的框架和不怕失去对方。希望对你有所帮助！
