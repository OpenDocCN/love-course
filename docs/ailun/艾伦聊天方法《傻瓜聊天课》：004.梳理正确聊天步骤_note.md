# 艾伦聊天方法《傻瓜聊天课》：004.梳理正确聊天步骤

🎼今天第四节课我们将探讨与女生聊天时常见的五大话题分类。许多人在听完前三节课后，仍然缺乏清晰的与女生聊天的思路。

因此，接下来的内容将帮助你更清晰地梳理聊天思路。首先，你需要了解在与女生聊天时，每个阶段应该做什么，以及应该聊什么。在你刚与女生添加微信的阶段，你不可能去邀约，也不可能立即与她调情。为什么？因为你之前没有做好铺垫，没有建立好基础。

所以，如果你想立即邀约，是无法达到预期效果的。那么，在添加女生后，通常第一个阶段要做什么呢？进行基本信息交换。首先，你需要让女生知道你是谁，为什么要与她搭讪，你对她的感受是什么。

## 基本信息交换

在第二个阶段，也就是你完成了信息交换和足够的铺垫之后，我们该如何让女生对你产生兴趣呢？其实就是在建立联系感。在你完成了足够的信息交换之后，如何进一步拉近与女生的关系呢？

首先，你需要保持与她的联系。因为一个漂亮的女生每天在微信上收到许多男性的问候。然而，大多数男性都像许多兄弟一样，总是急于拉近关系，过于急躁。

所以，大多数男性一开始就想追求她，但她的反应并不好，或者并没有服从那些男性之后，那些男性就不会再与她聊天了。

## 建立联系感

当你与女生有了足够的信息交换和联系感之后，接下来该做什么呢？那就是拉近关系。我们可以从表达感受开始。例如，你可以对女生说：“我觉得跟你聊天很开心，你是个很会照顾人的女生吗？”

这种表达都是在表达感受。当你对女生表达完足够的感受之后，接下来该做什么呢？那就是调情。如何与女生调情呢？许多学生总是认为调情就是耍流氓，其实并非如此。

与女生调情时，你需要知道一个漂亮的女生也非常渴望遇到一个有趣的男生，与她互动。她希望遇到一个有趣的灵魂，而不是一个总是色眯眯地说一些不正经事情的人。

正确与女生调情的方式我们将在后面的课程中讲解。当一个女生开始与你调情时，你们之间的关系就会迅速拉近。这时，你们可以很快地从普通朋友进阶为恋人。

## 调情与邀约

当你与女生产生过调情之后，你需要知道，此时你应该做的事情不是继续陪她聊天，而是进入第五个阶段——邀约。为什么要邀约呢？因为无论是对你还是对女生来说，你们在微信上聊得越多，如果不邀约，就只能说明你们在打嘴炮。

所以，你需要将她邀约出来。因为在微信上聊得再多都没有意义，你得不到任何东西。

## 总结

今天这一节课帮助大家梳理了聊天的正确步骤。我们再来总结一下：

1. 交换信息：让女生知道你是谁，让她对你产生安全感。
2. 展开话题：增加联系感，让她对你有一个基本的第一印象，让她感到与你聊天是愉快和轻松的。
3. 拉近关系：调侃。
4. 邀约：将她邀约出来。

这就是正确的聊天思路。第一，交换信息。第二，展开话题。第三，调情拉伸。第四，邀约。
