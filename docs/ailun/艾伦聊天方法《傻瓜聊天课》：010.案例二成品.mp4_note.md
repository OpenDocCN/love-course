# 艾伦聊天方法《傻瓜聊天课》：010.案例二成品.mp4

🎼大家好，今天我们将通过一个案例来讲解如何快速与女生从陌生到熟悉再到邀约的全过程。首先，让我们来认识一下今晚的女主角。

**女主角介绍**
这位女生是在社交软件上认识的，我在社交软件上与她交流不多，而是迅速转到了微信。加了她微信后，我先发送了自己的名字，并表达了无目的性、无需求感。

**开场白**
女生回复后，我表达了我们离得很近，暗示可以约她，但没有直接说。接着，我询问她在做什么，她表示在床上休息。我称赞她的眼睛很精致，并询问她是否是成都人。她回答是四川人，不是本地人。

**信息交换**
我告诉她我是江苏人，在这边玩几天，她休假几天，所以现在在床上休息。我询问她的身高，她回答160厘米，我接着说：“那你亲我的时候，我们要坐下。”这里通过身高话题快速拉近关系，并表达了我的直接和果断。

**调情**
我通过身高话题进一步调情，并询问她是否可以摸一下她的小兔子（她朋友圈的照片中戴着小兔子帽子）。她同意后，我暗示要邀约她见面。

**邀约**
我通过委婉的方式表达了对她的兴趣，并询问她是否可以做饭。她表示会，我邀请她品尝我做的牛排。她表示准备起床，我要求她发位置给我，并最终要到了她的手机号。

**结束**
聊天结束后，我给她打电话，告诉她从我的位置到她那里大约需要20分钟，我需要先忙一些事情，然后会在某个时间出发，让她准备一下。

**总结**
这个案例展示了如何通过快速调情、邀约和拉近关系，从陌生到见面。关键在于前期的展示和聊天的节奏。
