## 正确认识邀约与被拒绝

上一节我们介绍了课程概述，本节中我们来看看关于邀约的基本心态。邀约女生时被拒绝是非常正常的事情。不要害怕失败，也不要害怕被拒绝。至少你去邀约她时，你让她知道你是一个可以勇敢邀约的男人。

女人可能会比较矜持。你去约她，她会想：如果第一次约我，我很轻松就答应跟你见面了，会不会显得我很廉价，会不会显得我很随便。所以说，邀约女生被拒绝是很正常的事情。

如果第一次去邀约女生时被拒绝了，你要知道下一次你再去邀约她的时候，也许会更顺畅。他会觉得：我上一次都拒绝你了，那这一次我就答应你吧。

**核心要点**：**邀约被拒是正常过程，是展示勇气的机会，并为下次邀约铺垫。**

## 邀约的最佳时机

了解了正确心态后，我们来看看何时发出邀约最合适。如果你去跟女生聊天，第一轮你就可以去邀约她。但是通常我们会在第二轮或者是第三轮聊天的时候进行邀约。

在第一轮聊天的结尾，你可以跟他先模糊邀约一下。先跟他说：“那有机会一起玩吧，我要休息了。”这样的结尾其实就是在模糊邀约。

如果你跟女生聊天，你只去聊但又不去约的话，时间久了，会让这个女生觉得你是一个没有勇气的男人。

**核心公式**：**邀约时机 ≈ 第1轮（模糊邀约）或 第2/3轮（正式邀约）**

## 如何发出明确的邀约

明确了时机，接下来我们学习如何具体表达。去邀约女生的时候，该跟她发哪些东西呢？其实非常简单。你邀约的目的一定要明确。你要让这个妹子知道你真的是要约她，而不是在跟她打嘴炮。

所以如果你跟女生聊天前去邀约她，你不要跟他发：“你什么时候有时间呀，一起喝东西。”你可以说明：“明天下午有时间吗？我请你喝东西。”这样的话就让这个妹子知道，你现在是真的要约她，而不是在跟她打嘴炮。

以下是几种有效的邀约话术示例：

*   **基础版**：你跟她说：“明天下午有时间吗？我知道有一家还不错的咖啡厅，明天带你去尝一下。”
*   **场景描述版**：我们可以去加一些场景的描述。比如说：“我朋友前两天给我推荐了一个非常棒的秘密基地，在这里可以喝咖啡，而且还可以欣赏到这个城市最美的夜景，明天有时间吗？我带你去尝尝。”

## 应对模棱两可的回应

发出邀约后，对方可能不会立刻答应。如果你去邀约女生的时候，你给她发了邀约的话，这个女生对你的反应是模棱两可。比如说女生说“我想一想要不要去”，或者是妹子说“听起来很诱人，但是呢我得考虑一下”，又或者是你去邀约女生的时候，妹子跟你说“明天再说吧，到时候再说吧”。这个时候我们该怎么办呢？

这个时候一定要记住，千万不能着急。我们在前面的课程里面已经讲到了，在跟女生聊天的时候，我们可以进两步退一步。去邀约女生的时候，妹子说“我考虑考虑吧，明天再说吧”，这个时候呢你可以先退回来，适当的放一放，再聊一些其他的，然后再重新穿插这个话题，让她知道我是真的要约你。

那么还有一种方法，你跟妹子聊天的时候，你跟她说：“明天下午有时间嘛，我请你喝东西。我知道有一家还不错的咖啡厅。”妹子可能会跟你说：“我明天再看吧，明天看看时间再说。”我们可以跟她这样回复，你就说：

```text
那就暂定明天下午见。如果来不了的话，提前跟我说一声。
```

那么很多时候一个女生她们跟你出来见面，对于女生来说成本非常高，她也在犹豫要不要跟你出来见面。这个时候就需要我们作为男人，我们要替她来做选择。所以说我们跟他说：“那就暂定明天下午见。如果你来不了的话，你再跟我说。”这个叫**假定成交**。

那么假定成交有一个好处是什么呢？好处就是这个妹子在明天的时候，她就会想：哎，如果那我已经答应跟你见面了。如果我不去的话，会不会不太合适。那么这个时候呢她就会自己说服自己来跟你见面。

就像是你买了1个5000块钱的手机，但是你支付了1个200块钱定金一样。如果你不去支付这个定金，那你可能就不会买她了。但是你一旦向商家支付定金了，你就会不断的去说服自己，哎呀，定金都掏了，买就买吧。

## 被明确拒绝后的策略

当然，邀约也可能被明确拒绝。如果你问她“明天下午你有时间嘛，我请你喝东西”，妹子跟你说“没有”，那么通常女生拒绝约你呢有两种原因。

第一种是她真的没有时间。比如说他可能会给你发一句跟你解释，他会告诉你：“哦，明天我约了朋友一起去爬山。”“明天下午我约了朋友去打高尔夫。”那么通常女生在跟你解释明天她为什么没有时间的时候，就说明妹子是愿意跟你约会了，只不过这个契机不对。所以说你可以再追发一句，比如说“那后天呢？”或者是“那下周一呢？”

还有一种情况就是你去问妹子：“明天下午有时间吗？我请你喝东西。”妹子说“没有”。然后呢，你说“那后天呢”，妹子也说“没有”。或者说他根本就不回复你，或者说妹子可能直接给你发一句“再说吧，到时候再看吧”。那么，你要去判断一下这个妹子她是否对你感兴趣。

那么说到这里，我们要再讲一点，就是邀约的前提是什么。首先，你要知道邀约的前提是你通过前面跟他的聊天来判断出他对你是感兴趣的。你是在对妹子有一个吸引的情况下，这个时候你去邀约，妹子会愿意跟你出来约会。

如果这个妹子都对你没有任何兴趣，那么这个时候你去邀约她，无非是向她暴露你的需求感。所以说，邀约的前提一定是判断出这个女生对你感不感兴趣。

如果你邀约妹子被她拒绝了，要么是妹子没有时间，要么是她对你不感兴趣。她之所以对你不感兴趣，最重要的原因是你没有一个很好的吸引。所以说我们要适当的放一放。

又或者是为什么这个时候你去邀约她，她不给你出来约会呢？也许这个妹子她现在身边有比你更优秀的男人，或者是她已经有男朋友了。所以说她不是对你不感兴趣，只不过是她遇到了一个比你更感兴趣的男人。

所以说，你去邀约一个妹子被拒绝，没有关系，放一放。也许一个月以后，3个月以后，或者是半年以后，这个妹子会重新为你释放窗口。那这个时候我们就一定要把握住了。所以不管是妹子没有时间，还是对你不感兴趣，我们都要适当的放一放。

如果妹子对你不感兴趣，你约她他说没时间，这个时候呢你再去追发：“为什么没有时间呀？我请你喝东西呀。怎么不出来呀？”你这样去追发，只会让妹子觉得你需求感很强。

**核心策略**：**判断拒绝原因（真忙/无兴趣） → 若解释则改期再邀 → 若无兴趣则暂停吸引 → 避免追问暴露需求感**

## 展现确定性与诚意，防止被放鸽子

最后，我们讲一讲邀约时非常重要的，就是你要让妹子看到你的确定性以及你的诚意。什么是确定性呢？我举一个例子，很多兄弟刚加女生的微信，上来就去跟他说：“一起出来喝东西吗？一起出来玩吗？我请你喝酒呀。”这个妹子会觉得这个男人很随便，他没有诚意。他跟我说，刚加微信就跟我说，一起出来喝酒嘛？那么我不能去。

如果你去邀约妹子，妹子说了“好”。你说明天下午有时间吗？我们一起出来喝东西，妹子说了“好”。那这个时候我们一定要记住：**快速确认时间、地点、手机号**。

我们为什么要确认时间、地点、手机号呢？就是告诉这个妹子，我是真的要约你。然后这个妹子知道你跟他见面的重要性。我们向大家确定时间、地点、手机号最重要的目的，还有就是防止妹子放你鸽子。

以下是确认邀约的具体步骤：

1.  **确认时间地点**：比如说你约了妹子，明天下午见，妹子说“好”。这个时候你可以跟他说：“那明天下午5点钟在德济广场负一楼星巴克见吧。”
2.  **索要手机号**：然后妹子这个时候跟你说“好”，我们还要找他要一样东西，就是要她的手机号。当妹子把她的手机号发给你之后，他会更加的确定我明天要跟你见面。因为她把自己的手机号都发给你了。如果她不来跟你见面的话，会显得她非常没有礼貌。
3.  **次日提醒**：当你跟女生已经确定好了明天下午要见面了，时间地点，手机号拿到之后，到第二天上午的时候，你可以给妹子打一个电话，你说：“别忘了我们下午要见面哦。”你这样去跟妹子打一个电话，防止妹子放你鸽子。你提前打这样一个电话，最重要的原因是你要在第二天提醒这个妹子，我是真的要见你，我们下午就要见面了。因为现在每个人都是很忙的。
