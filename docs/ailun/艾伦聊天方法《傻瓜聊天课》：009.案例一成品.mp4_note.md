# 艾伦聊天方法《傻瓜聊天课》：009.案例一成品.mp4

🎼大家好，今天我们将通过一个微信聊天案例，展示舒富哥如何与女生搭讪并最终邀约成功。

## 案例背景

舒富哥在成都太古里搭讪认识了一位名叫王莹的女生，并在晚上6点46分通过微信加了她。

## 聊天过程

### 开场破冰

- 舒富哥：菊雨飞，你呢？
- 王莹：我叫王莹，英文名克里斯塔尔。

### 潜移默化邀约

- 舒富哥：要不是今天太累了，就约你喝咖啡了。
- 王莹：哈哈你是要微信要累了吗？

### 话题转移

- 舒富哥：哈哈你是学什么的？感觉你像是很会跳舞的女生。
- 王莹：天哪，你是第五个说我像跳舞的。

### 交换信息

- 舒富哥：我是做情感咨询，在跟朋友一起创业。
- 王莹：情感咨询，就你对我的职业很感兴趣。

### 拉近距离

- 舒富哥：我不是微商，我在太古里的被捕上班。
- 王莹：嗯我准备考二级心理咨询师。

### 询问兴趣爱好

- 舒富哥：成都今天雾霾很大。
- 王莹：嗯对呀，今年好多了，去年比较严重。
- 舒富哥：你平时都喜欢做什么呀？
- 王莹：比较喜欢音乐和看电影吧。

### 模糊邀约

- 舒富哥：想爬成都某天台拍高空来着，你知道市中心那哪栋楼能够爬天台上面去拍照嘛？
- 王莹：不知道啊，我没有爬过。
- 舒富哥：哈哈记得前任三是在成都拍的。你看过了吗？
- 王莹：嗯，对，我说想再看一遍。
- 舒富哥：想再看一遍你呢其实我在说这句话的时候，已经在潜意识里面想要去约他了。
- 王莹：我也是还想看最近上映的勇敢者游戏。
- 舒富哥：好呀，有机会可以一起。

### 确定邀约

- 舒富哥：那就去电视塔吧，我过去接你。
- 王莹：哦哦，要不然在电视塔见你来接我太麻烦了。
- 舒富哥：没关系啊，我滴滴过去比较方便。
- 王莹：那你那就到博社吧，我就在博社后面住。
- 舒富哥：嗯发一个位置。

## 总结

本节课中，我们学习了如何通过微信聊天与女生建立联系，并通过话题转移、信息交换、模糊邀约等方式，最终成功邀约女生。关键在于保持聊天节奏，把握时机，并降低女生的成本。
