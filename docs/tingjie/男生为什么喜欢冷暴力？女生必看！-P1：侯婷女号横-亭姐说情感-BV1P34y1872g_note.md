## 什么是冷暴力？

冷暴力指的是一种在关系中不明确、不回应、不拒绝也不接受，并强行阻断一切亲密互动的行为模式。

其核心表现可概括为：**不沟通、不解决、不负责**。

## 冷暴力使用者的四个心理特点

上一节我们定义了冷暴力，本节中我们来看看习惯使用冷暴力的人通常具备哪些心理特点。以下是四个关键特征：

1.  **极度自私与自我**
    这类人的思维逻辑是：“我现在不想说，我就是不说。我不会在意你的感受，我只在意自己的感受。” 你的感受在他看来是次要的。

2.  **缺乏责任感**
    他不明白一段关系中的喜怒哀乐都需要双方共同解决和经营。由于没有责任感，他自然不会考虑你是否难受、是否整夜无法入睡。只要能达到让你难受的目的，他就认为自己的目的达到了。

3.  **毫无共情能力**
    在日常生活中，我们看到陌生人哭泣都会于心不忍，更何况是对待最亲密的伴侣。最令人难受的是，他连“我现在心情不好，想一个人待会儿”这样的话都不愿说，因为他的目的不是为了缓和关系，而是让你心寒。

4.  **追求绝对自由，抗拒约束**
    这种人的逻辑是：“你怎么又提让我烦的事？你怎么不能给我一点空间？” 他无法给予你最基本的尊重，反而对除你之外的所有人都很有礼貌。他明明知道你在等待回应，但你越是等待，他就越会将冷暴力的时间拉长。
