# 情感修复课 01：分手后重建关系的两个核心原则 💔

在本节课中，我们将要学习分手后若希望重归于好，必须牢记并践行的两个核心原则。这两个原则旨在帮助你调整心态、规范行为，从而为关系的修复创造可能性。

## 认清现实：关系决定行为 🧘‍♀️

上一节我们介绍了课程目标，本节中我们来看看第一个核心原则：“屁股决定脑袋”。这句话的含义是，你必须认清分手已成事实的现状。你与他当下的关系状态，决定了你应该做什么事，而不应越界。

在一起时都未曾如此关心对方，分开后反而陷入过度殷勤的状态，这不合规矩。这种行为不仅让你讨不到任何好处，同时也会让对方觉得，这种冷淡处理的方式对你好使，足以消耗你的精力与期待。

## 专注行动：用事实代替言语 🛠️

理解了行为边界的重要性后，我们进入第二个原则：“事实比成果重要”。你说得天花乱坠都没有用，因为对方在决定分手前，必定已经默默给过你很多次机会。你也不应总是去逼问分手的原因。

以下是践行此原则的具体行动指南：

*   **弥补不足**：努力补偿之前自己做得不足的地方。
*   **打破刻板印象**：通过行动打破对方心里对你的原有负面印象。
*   **抵消负面情绪**：做一些事去抵消他之前累积的负面情绪。

你只需踏实做好这些事，不必纠结于即刻的结果，自己要保持清醒，不要糊里糊涂。

---

本节课中我们一起学习了分手后寻求复合的两个关键原则：一是认清关系现状，规范自身行为不越界；二是停止空谈，用切实的行动弥补不足、改善印象。记住，但行好事，莫问前程，清醒而踏实的努力才是改变关系的基础。
