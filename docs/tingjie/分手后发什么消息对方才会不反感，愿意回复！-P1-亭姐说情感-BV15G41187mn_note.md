# 情感修复课程01：分手后如何发送对方不反感且愿意回复的消息 💌

在本节课中，我们将学习分手后如何与前任进行首次沟通。核心目标是发送让对方不反感、并愿意回复的消息，从而重建联系，为后续步骤打下基础。

## 收起恋爱脑，避免情绪绑架

上一节我们介绍了课程目标，本节中我们来看看成功沟通的首要原则：避免情绪化。

很多人分手后难以与前任成功对话，无论发送什么消息，对方都不回复或反应冷淡。此时，首要任务是收起恋爱脑，不要用情绪绑架对方。

这一点对大多数人而言非常困难。若能收起恋爱脑，在挽回道路上就已成功一大半。因为恋爱脑是人的本能，在情感遭受冲击后，自然会**产生崩溃、痛苦、焦虑等情绪**。

在这些情绪驱使下，人会做出许多不理智行为。

以下是情绪驱使下的典型错误行为列表：
*   信息轰炸
*   围追堵截
*   疯狂道歉
*   拼命求和
*   卑微纠缠

这些情绪和行为虽然正常，但任何成功都不是靠情绪和胡闹得来的。成功者往往比你想象中更加沉着、冷静、清醒和智慧。

> 情绪是人的一生之敌。

因此，联系前任时，第一条消息必须仔细斟酌。

## 识别并避免反感信息

上一节我们强调了情绪管理的重要性，本节中我们来看看哪些具体信息会让对方反感。

我们需要思考：哪些话是对方反感的？哪些信息会让对方看到就不想理你？哪些信息可能让对方误以为你在纠缠，从而将你拉黑？

这并不难找，我们可以进行换位思考。如果你主动提出分手，当下你最讨厌接收的信息是什么？

以下是常见的、会引起反感的开场白列表：
*   “在干嘛？”、“吃了没？”、“睡了没？”（显得无聊且无意义）
*   “你是不是外面有人了？”、“你是不是早就想分手了？”（引发反感和抵触）
*   “我对你这么好，你为什么离开我？”、“我到底哪里不好？”（消耗对方耐心）
*   “再给我一次机会吧，我以后再也不这样了。”（缺乏实质诚意）

这些信息一旦发出，只会消耗对方所剩无几的耐心。因此，必须避免发送此类消息。

## 学习发送正确的消息

了解了应避免的内容后，本节我们来看看应该如何正确发送消息。

核心原则是：传递正面情绪，不让对方感到压力或反感。如果实在不会说，保持沉默也比胡乱发送要好。

分手后每一句话看似随意，实则句句都应有其含义。不要不经过思考就脱口而出。

以下是几个正确消息的示例：
*   **减轻压力型**：“你现在不想回复我也没关系，等你情绪好点再说也行。” （不给对方压力，可能促使回复）
*   **简单夸赞型**：“你最近好像变得更好看了。” （传递正面情绪）
*   **价值吸引型**：说一些能够吸引对方兴趣的内容。

即使对方不回复，也很正常。不要因此失落或难过。隔一段时间后，再寻找新话题开启聊天即可。

总之，前期联系的目的在于**重建联系**。只要能够恢复正常沟通，就可以逐步规划后续推进步骤。

记住，不要急躁。一步登天只会让对方更加抵触。

---

本节课中我们一起学习了分手后首次沟通的核心技巧：**管理自身情绪，避免发送引起反感的信息，转而发送能传递正面价值、减轻对方压力的消息**，以此重建沟通桥梁，为后续可能的修复奠定基础。
