# 情感修复课程 01：分手后未拉黑删除，复合成功率分析 🧩

在本节课中，我们将探讨分手后若未被对方拉黑或删除联系方式，应如何理解这一信号，并学习通过实质性改变来有效提升复合成功率的核心方法。

上一节我们介绍了课程主题，本节中我们来看看复合的核心基础。

分手后想要成功复合，做好以下几点，你离目标就不远了。在讲解重点之前，需要明确：任何人都无法对你的快乐和幸福负责。那么，分手后复合的核心到底是什么？答案是：**改变**。

首先，我们需要知道，改变是一个需要沉淀的过程。这种改变是自身不受外界任何因素影响的、一种本质上的、发自内心的改变，这样才是真正的改变。

以下是关于“真正改变”的一个简单例证：

*   当你做某件事情时，很多人都在否定你。
*   选项A：用真正做出这件事情的**结果**去证明自己。
*   选项B：不厌其烦地去告诉别人，你在做这件事的过程中付出了多少努力和辛苦。

答案很明显。改变永远不是用嘴说出来的，而是要靠拿出结果让人信服的。必须让你的改变，之后以最优秀的样子站在对方面前，这才是更好的说服力。

当然，在你改变以后，是否还愿意选择他，那是你自己的事情。你变好以后，选择别人或选择更好的人，也没有任何人能怪你，因为你的价值变得更高了。

但是，如果说你为了挽回前任而做出那些改变，脑子里一直想着“我到底要怎么样才能让这个人看到我的改变”，我直接告诉你，这都称不上是本质的改变。这只是因为你失去了一个恋人，非常痛苦的情绪促使你为了挽回而去改变，是为了挽回而刻意营造的人设。

也许你能够骗过你的前任，甚至能够骗过自己，能够假装一阵子。但是你真的能够假装一辈子吗？难道你可以一直长期地装下去吗？装不了，你总会露出马脚的。

所以，提醒大家：只有真正地沉淀下来，真正地去做出改变，在你去挽回对方时，才能让对方看到不一样的自己。只有你发自肺腑、带着诚意去改变，然后站在他的面前，才会让他觉得有一种眼前一亮的感觉，挽回也会变得非常容易。

本节课中我们一起学习了：分手后未被拉黑是一个积极信号，但复合的真正核心在于个人由内而外的、实质性的**改变**。这种改变不是为了表演给对方看，而是为了提升自我价值。当你以更好的面貌出现时，无论是挽回旧爱还是开启新篇章，都将拥有更大的主动权。
