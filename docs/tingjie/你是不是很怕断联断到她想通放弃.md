# 你是不是很怕断联断到她想通放弃

在本节课中，我们将学习如何应对分手后“断联”过程中的焦虑情绪，并理解其根源在于自我价值感的缺失。我们将探讨如何停止自我否定，建立自信，从而打破“害怕被彻底放弃”的恶性循环。

## 断联焦虑的根源 🤔

上一节我们概述了本课主题，本节中我们来看看断联焦虑的具体表现与核心原因。

你是不是也很怕断联断到他想通放弃你。在断联这个过程中，很多人就是才一周没有联系就开始慌张了。真的是五花八门的焦虑，怕有新欢怕被忘记。真的就怕他想通了把你忘记了。

那么你为什么会这么想呢。事实就是你自己都不够认可自己的价值。觉得人家会通过想通而放弃你，就好像是看到了最真实的你。**不自信价值感又不高**。

## 解决问题的关键：自我增值 🔑

既然我们明白了焦虑源于低价值感，本节中我们来看看解决问题的根本方向。

如果说你还想复合的话，那么在价值感不高的情况之下呢，那就要好好的去做，做你该做的事情。就是如何给自己增值。

在某种程度上呢，你越担心什么就会越来什么。反之呢，如果说你价值足够高的情况下，你俩相比较，你还胜出的时候，你有什么可担心的呢。你还会担心吗。哪怕他气得要死的时候想明白了，一会儿呢就又想不通了，跟你一样的情绪反反复复。

## 低价值感的行为表现与恶性循环 🔄

理解了方向后，我们需要识别低价值感在挽回过程中的具体有害行为。

很多人呢怕被分手以后，沉迷于对方说的那些狠话、跳过的刺。然后呢去进行各种反思道歉，卑微的像极了是你最有硬德的样子。这种思想的本质依然是你不够自信。

我无数次提你们为什么分手，分手的原因是什么。你还在纠结你的性格你的脾气，你爸怎么样，你妈说了什么。你越说什么，他就会对你越失望，然后把你放下。

不自信的原因有很多，会让你在相处的时候，多疑敏感焦虑，想太多很别扭。后期呢还会有大起大落的情绪，发脾气很在意对方的看法，特别怕失去。

还会在分开以后呢，你无法客观的分析原因，让你觉得是因为你自己不好，导致的分手。然后不自觉的去把它给美化了，觉得他真的承担了很多。不自觉的呢就觉得他有更好的生活和发展。

因为你是被抛弃的那一个，所以你的失去感就会特别重。所以呢你就是更痛苦的那一个。那么呢你就是在变相的去否认自己。**这就是一个恶性循环**。

## 如何打破循环：停止幻想，正视现实 💡

认识到恶性循环后，本节我们将学习打破它的具体心态调整方法。

所以分手以后一定要对症下药。你怕的太多呀就越颓废。不要自己吓自己。他没有你想象的那么强，你也没有那么弱。

你仔细想想看，你当初看他的样子，是不是当时也不觉得他像今天那么好、那么高大呢。其实都是不自信加失去感在作祟。
