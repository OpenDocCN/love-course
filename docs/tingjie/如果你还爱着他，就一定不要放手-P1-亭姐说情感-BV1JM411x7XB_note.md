# 情感修复教程 第1课：如果你还爱着他，就一定不要放手 💔

在本节课中，我们将要学习分手后，如果内心依然深爱对方，为何以及如何坚持不放弃。我们将探讨感情的本质、磨合的重要性，并理解纠缠背后深刻的爱意。

## 不要轻易放手

分手后，如果你还爱着他，就不要去放手。如果你还爱着，就一定不要放弃。错过一辆车可以再等，没有钱也可以再挣，可是错过了一个人，那就是一辈子了。

## 认清内心现状

你可能也知道，跟他再也回不到过去了。但这并不是你想要的结局。你可能还是很喜欢他，很爱他。你仍然会想到他，听到他的名字时，心头还是会一颤。你担心他过得不好，虽然他现在与你已无关系，你依然会感到心疼。

上一节我们探讨了内心的真实感受，本节中我们来看看感情中的责任与磨合。

## 感情需要共同经营

我知道你们分开，他也有错，他也有很多缺点。可是，难道你就没有吗？你现在改了吗？我告诉你们，没有哪段感情是顺顺利利的。感情都需要经过不断的磨合。

你说你谈个恋爱，跟这个人相处有点矛盾，你就开始打退堂鼓了。你只要遇到问题就想逃避。那你跟谁都谈不好。

以下是关于“合适”的误解：

*   你可能会说，是因为跟这个人不合适。
*   但我想说，**世界上没有完全合适的两个人**。

## 家庭与陪伴的价值

女人如果没有了家庭，就像路边的野花，人人都喜欢，但是却没有人去呵护。男人如果没有了家庭，就像路边的一艘船，过海的时候有很多乘客，可是到了岸边，却只剩下他自己。

现实生活告诉我们：
> 不如住别墅，不如恋人手牵手过马路。金窝银窝，不如相濡以沫的小窝。珍惜枕边人，愿得一人心，白首不分离。

## 理解“纠缠”背后的爱

真正爱过的两个人分手以后，都会有一方苦苦纠缠，不愿意放弃。其实你知道吗，越是卑微纠缠你的人，一定是爱你如命的人。

他用一次又一次的各种方式去联系你，不管你态度有多冷漠，他也不会放低姿态去做这一切的事情。这都是因为他太爱你，他离不开你。

两个人能够相知、相遇、相爱，真的需要很大的缘分。也许这辈子，只有为了你，他才会这样做。
