# 前任总是在你和新欢之间不停徘徊：侯婷女横2-亭姐说情感-BV1

## 一、 发现前任有新欢时的首要反应

上一节我们概述了课程主题，本节中我们来看看当得知前任已有新欢时，你的第一反应应该是什么。

当你知道你的前任有了新欢，你当下就应该断开所有联系。

别人在谈着恋爱的同时，不管你是纠缠也好，做朋友也好，这都不合适。

这些行为都是在给他制造渣你的权利。

## 二、 评估自身处境与做出选择

理解了首要原则后，接下来你需要冷静地评估自己的处境。

其次你要考虑你能不能接受。

不能接受就不要浪费时间了。

## 三、 剖析前任徘徊行为背后的真相

当你发现前任总是在你和新欢之间徘徊，这其实也说明了几个问题。

前任有新欢还来撩你，不是渣，就是你和新欢都有好与不好的地方。

你知道这是什么行为吗？别人在选妃。

其实你和新欢都是驴马，没找到更好的之前先记着再说。

## 四、 破除常见错误心态与借口

可能道理你们都懂，但是就是做不到，总是要给自己找不能放下的理由。

以下是几种典型的错误心态：

*   **心态一：** “要是和他失去了联系，他们感情越来越好了该怎么办？”
*   **心态二：** “如果他不来找我，是不是没有机会了？”

其实你越这样想就越没机会，因为你的这些行为在别人眼中一文不值。

不要为自己的无能找借口找理由。

也不要因为不甘心还要跟新欢做对比。

相信我，新欢也是可怜人。

## 五、 警惕空洞承诺并果断行动

我相信有这样遭遇的人一定听过一句话：“你给我时间我会处理好。”

然后所谓的处理就是一拖再拖。

不要再当傻子了好吗。
