# 情感修复教程 01：七步黄金复合法 💔➡️❤️

在本节课中，我们将学习一套系统性的“七步黄金复合法”。这套方法旨在帮助那些面临情感破裂、对方态度决绝的情况，通过理性、有序的步骤，重新建立联系并修复关系。整个过程强调自我反思、策略性行动和耐心推进，而非情绪化的纠缠。

---

上一段我们概述了本课程的目标，接下来，我们将详细拆解这七个核心步骤。每一步都至关重要，必须严格按照顺序执行。

以下是七个必须按顺序执行的步骤：

1.  **认可分手决定，以朋友身份相处**
    首先，你需要完全认可对方提出分手的决定。停止争辩与道歉，主动退回到朋友的位置。这能极大降低对方的防御心理，为后续联系保留可能性。

2.  **保持弱联系，避免信息轰炸**
    在对方不拉黑你的前提下，保持低频、轻松的联络。切忌频繁发消息或打电话，目的是维持一个基本的沟通渠道，而不是施加压力。

3.  **复盘关系，找出核心问题**
    你需要理性复盘过去的相处细节。将导致分手的问题逐一列出，并**自己**分析原因、思考解决方案。公式可以表示为：**问题识别 + 原因分析 + 解决方案 = 有效复盘**。

4.  **切实改变，并展示改变**
    针对上一步找出的问题，开始切实地改变自己。无论是性格、习惯还是沟通方式，并且要通过朋友圈等间接方式，让对方能够看到你的变化。

5.  **强化自身吸引力**
    回顾最初吸引对方的特质（如幽默、上进、体贴），将这些优点进一步放大和提升。重新建立你的个人魅力。

6.  **引导对方倾诉，学会倾听**
    即使已同意分手，也可以找机会开启轻松聊天。在对话中，多引导对方表达感受和想法，你则扮演倾听者的角色。代码逻辑类似于：`while(聊天进行中){ 多提问； 少谈论自己； }`

7.  **重建线下联系，谨慎解释过去**
    当线上互动变得愉快自然后，可以尝试约线下见面。见面时像朋友一样相处，避免谈论沉重的过去。只在合适时机，简单、真诚地为你过去的不当之处道歉。

---

本节课中，我们一起学习了“七步黄金复合法”的完整流程。从**接纳现状**开始，经过**自我反思与改变**，最终到**重新建立舒适的联系**。请记住，关键在于顺序、耐心与真诚的行动，而非急于求成的情绪化表达。把握每个细节，过程会逐渐变得轻松。
