## 理解低位状态与分手原因

上一节我们介绍了课程主题，本节中我们来看看导致分手的常见心理模式。

有一类人总是在感情中不自觉处于低位状态。他们任何事情都迁就对方、包容对方。这类人被分手往往不是因为具体矛盾，而是因为他们更害怕被对方抛弃，害怕对方说出“分手”二字。

他们不断通过讨好行为去满足对方。这类人其实过于心软。人们常说势均力敌的爱情才能长久。过度迁就一个人，只会使亲密关系的天平越来越倾斜，直到最后崩塌。

这类人往往共情能力较强，善良、正直、心软。所以哪怕对方做得再过分，他们也不愿意放手。这种男生在和女朋友闹别扭时，即使非常生气，表露出来的也只是表面一点点情绪。

因此对方根本不可能把他们的情绪放在心上。他们被分手的理由通常是：“我对你没有感觉了”或“我们真的不合适”。

---

## 高段位挽回三步法

如果你面临上述情况，应该怎么办呢？以下是三个核心步骤。

### 第一步：重塑核心心态

首先你要明白一点：在亲密关系中，你的感受应该放在第一位。分手后你更应该先考虑自己，不要再把自己摆在一个比他低很多的位置上。

回想恋爱时你事事迁就对方，低位的做法依旧阻止不了他离开你，分手后更是如此。所以你千万不要再去死缠烂打，别再卑微求和。你现在哪怕低到尘埃里，他都是看不到你的。

你应该怎么做呢？首先你需要强化自己的态度，要与以前的性格形成鲜明对比。

**心态公式：** `新自我 = 坚定态度 + 自我优先`

### 第二步：设计精彩开场

你需要一个精彩的开场，例如展示给对方不一样的生活状态。

假如你是一个比较沉稳、内敛、日常居家的人，你的社交动态就可以多发一些外出游玩、健身、跑步等内容。通过展示生活的变化，间接传递你已改变的信号。

### 第三步：营造聊天舒适区

具体做法是：与他聊天时不要聊感情，保证聊天内容充实丰富，但又不要过于殷勤。

目标是让他慢慢觉得：“现在跟你聊天感觉特别有趣”，让他感受到你变了。这时他一定会重新燃起对你的兴趣。

---

## 巩固成果与掌握主动权

进入挽回阶段后，切记不可再回到恋爱时的低位状态。

你应该呈现一个全新的自我，用新的态度释放自己的魅力。只有让他觉得你不一样了，引起他的兴趣了，恋爱的主动权才能重新回到你手里。

**关系主动权代码：**
```python
if 展示新自我 == True and 引起兴趣 == True:
    主动权 = 回归
else:
    主动权 = 仍在他方
```

而困难在于，如果你一直陷入情绪和心态的怪圈，再多技巧和套路都救不了你。
