# 明明很喜欢，但是偏偏要刻意疏远

在本节课中，我们将要学习一种常见的情感心理模式——回避型依恋。我们将解析其核心特征、心理成因，并理解其行为背后的矛盾逻辑。

## 概述

回避型依恋是一种典型的心理模式，其根源往往与个体成长过程中安全感的缺失有关。这种模式显著影响个体在亲密关系中的态度与行为。

## 回避型依恋的七个显著特征

以下是回避型依恋模式在亲密关系中常见的七个行为与心理特征。

1.  **刻意疏远亲密关系**
    即使对方内心存有喜爱之情，也会刻意与伴侣保持距离，避免过度亲密。

2.  **逃避或缺乏沟通能力**
    倾向于回避深入沟通，或时常将沟通演变为争吵，以此掩盖自身在情感交流上的弱点。

3.  **表达能力弱**
    不善于直接表达情感，常通过说反话或做出与内心相反的行为来传递感情。

4.  **缺乏安全感**
    对感情的稳固性持悲观态度，不相信任何关系能够长久稳定，总觉得感情不堪一击。

5.  **鄙视亲密与依赖**
    将深情投入视为“恋爱脑”，并认为这是一种侮辱。他们害怕被感情束缚，从而失去自我。

6.  **感情中缺乏信心**
    表面看似在关系中占据高位，实则内心处于低位。对自己要求苛刻，不允许自己犯错或显露脆弱，认为只有“强者”才值得被爱。

7.  **将克制与痛苦等同于幸福**
    并非不想被爱，而是认为必须伴随着逃避与克制，这份爱才会让自己感到安全。痛苦成为了感知幸福的某种支撑。

通过以上特征，我们可以发现回避型依恋者对亲密关系的整体态度并非真正的厌恶，而是充满矛盾。

## 核心矛盾：渴望与恐惧的并存

上一节我们介绍了回避型依恋的具体行为特征，本节中我们来看看驱动这些行为的核心心理矛盾。

其心理逻辑可以概括为：**“想亲近 + 怕受伤 => 选择逃离”**。

*   **正常依恋模式**：喜欢 → 所以我接近。
*   **回避型依恋模式**：喜欢 → 但喜欢可能带来伤害 → 所以我必须逃离。

这种“渴望被爱”与“恐惧受伤”并存的矛盾，导致了他们在关系中的疏离行为。疏离的背后，反而是极度渴望被爱护和接纳。归根结底，他们心中有爱，只是尚未学会或以安全的方式去接受和表达这份爱。

## 总结

本节课中我们一起学习了回避型依恋的心理模式。我们了解到，这种行为并非源于不爱，而是源于早期安全感缺失所导致的矛盾心理：既渴望亲密，又害怕因此受伤，从而采取疏远来保护自己。理解这一点，是改善与这类人群相处方式的第一步。
