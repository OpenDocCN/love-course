## 核心判断标准：成就 vs. 消耗

上一节我们介绍了课程的核心主题，本节中我们来看看具体的判断标准。一段关系的好坏，根本上取决于它是相互促进还是单方面索取。

**核心公式**：**健康的关系 ≈ 相互成就 > 单方面消耗**

以下是“正缘”（成就型关系）与“孽缘”（消耗型关系）的关键行为对比：

*   **正缘（成就你）**：对方会以真诚的态度邀请你进入他的生活，提供关心、陪伴和支持。同时，他懂得珍惜并回报你的付出。
*   **孽缘（消耗你）**：对方会不断榨取你的好意和爱意，一边占你便宜，一边挑剔你，甚至嫌弃你付出得还不够多。

---

## 健康关系的基石：相互性与共赢

理解了基本判断标准后，我们需要深入其背后的原则。任何持久的关系都建立在互惠的基础上。

请记住：**任何关系都讲究相互成就**。单方面的持续付出，往往只会导致对方感到厌烦，觉得你过于粘人或管得太多。

**核心代码描述**：
```python
if 关系 == “相互成就”:
    print(“这是健康的感情：合作共赢”)
else:
    print(“这是不健康的关系：一方拖垮另一方”)
```

真正好的感情模式是“合作共赢”，而不是一方拖垮另一方。

## 选择伴侣的关键品质

那么，一个值得托付的“正缘”伴侣具备哪些特质呢？这些内在品质是关系长久稳定的保障。

一个真正值得交心、共度人生的伴侣，通常具备以下三个关键品质：

1.  **有人品**：行事有底线，待人真诚。
2.  **有责任**：对自己的承诺和双方的关系负责。
3.  **懂感恩**：珍惜对方的付出，并会给予回应。

因此，即使你非常渴望开始一段感情，也需要擦亮眼睛，以上述标准进行观察和判断。
