# 和前任分手后他还会不会关注自己-P1-亭姐说情感-BV1m14y1f7nb_note

在本节课中，我们将要学习分手后前任是否会继续关注你的心理机制。我们将探讨情感的复杂性、关联的延续性，以及如何理解并应对前任的关注行为。

## 🌫️ 情感的灰色地带

人的感情是复杂的。这个世界永远不是非黑即白的两极分化状态。我们更多的状态是生活中一种比较灰色混沌的状态。感情其实也是一样的。

一旦两个人有了感情，就会产生关联。

## 🔗 关联的延续性

这个关联不是说你分手了，它就会消失掉。它是消失不掉的。而且就算你们在相爱的时候，也不可能做到彼此百分百地爱对方。这是做不到的。

在分手之后，其实也不会做到完全不爱了。可能能够说把这份爱埋藏在心底里，不把它拿出来而已。

## 👀 本能的关注行为

所以说就算他想要放下你，对于人性来说，只要你还能够在他面前刷存在感，他就会本能的时不时的过来看你一眼，看你过得好不好。

以下是关于前任关注心理的具体分析：

*   **痛苦源于比较**：对于大部分的人来说，他看到自己的前任过得很好的时候，他就会觉得很痛苦。
*   **心理根源**：因为前任的幸福跟自己无关，他就会有失落感和失去感。

## 🛠️ 行动建议：利用心理铺垫

所以呢，这个时候如果说你想去挽回的话，可以利用前任的这个心理去做铺垫。

如果说你不知道该怎么去做的话，你也可以私信“复合”，我让你爱的人重新爱上你。
