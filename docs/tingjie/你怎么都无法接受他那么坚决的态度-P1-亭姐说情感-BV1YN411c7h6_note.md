在本节课中，我们将要学习如何解读伴侣坚决、冷漠态度背后的真实心理，并理解为何卑微讨好往往无效。掌握这些核心概念，是调整相处模式、修复关系的第一步。

你怎么都无法接受他那么坚决的态度。你开始做一些比较卑微甚至是极端的事情，就是为了让他感受到你的爱。可是他还是不为所动。其实这些都是不理智的。

上一节我们提到了不理智的挽回行为，本节中我们来看看男人行为背后的逻辑。男人不会轻易拒绝一个爱了很久的女生。他之所以那么坚决，是他害怕给你希望，却觉得跟你在一起很累很压抑。

**核心公式：坚决态度 ≈ 恐惧给予希望 + 感到关系压抑**

明白这一点很重要。你千万不要再去卑微的讨好，做感动自己的事情了。

理解了原因后，我们需要重新审视他的“冷漠”。男人他选择冷漠变得狠心，并不是说不在意你，而是你们的相处模式让他觉得累了。他不想让自己长期陷入这种压抑的感情，所以他才会这么狠心。

以下是两种关键心态的对比，帮助你清晰区分：

*   **无效心态：** 他认为“只要我继续讨好、证明我的爱，他就会回心转意。”
*   **有效认知：** 他感到“**当前的相处模式**让我感到疲惫和压抑，我需要空间或改变。”

如果说你能调整你们之间的相处模式，其实让他接受你并不难。
