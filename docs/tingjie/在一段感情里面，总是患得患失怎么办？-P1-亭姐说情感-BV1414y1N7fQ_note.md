# 情感课程01：如何摆脱感情中的患得患失？💔

在本节课中，我们将要学习如何在一段感情关系中减少患得患失的情绪，并理解这种情绪背后的本质。我们将探讨为何会过度期待他人回复，以及如何将安全感和价值感的建立回归自身。

## 理解患得患失的本质

上一节我们介绍了课程主题，本节中我们来看看患得患失的具体表现。许多人都有过这种经历：你的心情随着对方的举动而波动，随着对方的行为而起伏。对方没有给予你回复，你就会感到无比焦躁。

这种感觉就像自己被对方控制。对方手里仿佛拿着一把遥控器，可以随时掌控你的情绪开关。但这个情况的本质，不是别人在掌控你，而是你在想着掌控对方。

## 剖析期待回复的动机

理解了表象后，我们进一步剖析行为背后的动机。期待对方回复信息的动机和需求通常有以下几点：

以下是三种常见的心理需求：
1.  **安全感需求**：对方不回复你的信息，会引发你的焦躁不安。对方的回复会给你带来安全感。
2.  **被在乎的需求**：对方的回复和热情会让你感受到被在乎、被重视。
3.  **价值感需求**：你的生活可能缺少意义和价值感。与他保持联系这件事，会让你觉得被关注、被肯定，从而获得价值感。

所以你会发现，表象是“我很爱你，我很在乎你”，但深层次的原因是：与他发信息这件事，可以满足你自身的一种需求。

## 停止控制不可控之事

当我们把动机剖析清楚后，就需要面对一个核心问题：控制欲。如果我们非要去期待别人用自己想要的方式来爱自己、满足自己，那就等于强行控制那些不可控的事情。

这就像我们要控制明天的天气一样，是自讨苦吃。你拼命想掌控的，其实最终都掌控了你。我们把安全感、自我价值感、被肯定这些事物建立在别人身上，就等于亲手把遥控器交给了别人。这就是一切糟糕情绪的开始。

## 审视付出的真正动机

有人可能会说：“我付出了那么多感情、金钱和精力，为什么还会这样？” 接下来，我们再来谈一谈“付出”的复合动机。

例如，你发一条消息的动机是“我很想你”。那么，在消息发出去的那一刻，你表达思念的这个目的就已经达成了。至于对方回复与否、回复内容是什么，已经不重要了。

如果你的动机总是后者——即为了索取对方的回应和关注——那么你必然会有很多不开心。你的心完全依赖别人，这种过度的控制本质上是一种自私。所以，当一些人用自己的付出来“绑架”别人时，对方反而会逃得更快。因为对方会更加确定：你付出的动机，就是为了索取更多昂贵的爱和回报。

## 回归自我成长的核心

你明白了吗？你的生活可能缺少一些东西，比如价值感、爱和安全感。你总是希望另一个人出现来拯救你，这本身就是一种贪心。

所以，当自己能力不足，就像杯子里的水不满的时候，应该想办法让自己的能量充盈起来。而不是去祈求别人，把他杯子里的水施舍给你。记住一点：在自我成长这条路上没有捷径可走，感情更不能作为捷径。

## 课程总结

本节课中，我们一起学习了感情中患得患失情绪的根源。我们认识到，这种情绪源于将安全感和自我价值建立在他人反馈之上。关键在于，要把关注的焦点从“控制他人”转回到“充盈自我”，通过自我成长来获得内心的稳定与力量，而不是向外索取。
