# 💔 情感修复课 01：分手时把话说绝了，还能挽回吗？

在本节课中，我们将要学习当双方在分手时都说了非常决绝的话之后，关系是否还有挽回的可能。我们将分析说狠话背后的心理，并提供三个明确的步骤来帮助你理清思路，找到修复关系的突破口。

---

两个人把话都说绝了，还能和好吗？首先，我们需要理解为什么会说狠话。那些推开对方的话，其本意往往并非真想结束关系，而是为了发泄情绪，并希望对方能察觉到你的情绪，进而照顾你的感受。

你真正渴望的是对方的关心。你希望吵架之后，他能主动来哄你。然而，狠话说出口的同时，你也伤害了对方，并且可能忽略了对方的感受，甚至自己事后也会感到后悔。

把话说绝后，最让人在意的是：有些话一旦说出就无法收回。这常常成为阻碍双方和好的最大因素之一。本节中我们来看看一个关键认知：有时把话说绝了，情况并没有你想象的那么严重，对方也可能没有你预想的那么难受。

若想与他重新和好，以下三点你必须明白。

以下是三个需要你明确的核心要点：

1.  **行为模式评估**：你是否经常把话说绝，或做出推开对方的举动？例如，频繁提出分手。
2.  **后果与反思**：每次你说狠话后，对方是何反应？关系是如何一步步恶化的？你必须反思并真切体会对方的感受，才能做到共情，找到突破口。
3.  **情感基础回顾**：你们长期的感情中，有哪些值得回忆和留恋的美好时刻？他当初喜欢你什么？想清楚这些，并思考后续如何照顾他的情绪。让他重新看到这些，他才会对你重拾信心。

明确了这三点之后，你才能知道该如何去挽回他，以及如何经营好你们之间的感情。

---

本节课中我们一起学习了，分手时说绝话并不一定意味着关系终结。关键在于理解行为背后的情绪需求，并通过评估自身行为模式、反思对方感受以及回顾积极的情感基础这三个步骤，找到修复关系的路径。理清这些，是迈向挽回的第一步。
