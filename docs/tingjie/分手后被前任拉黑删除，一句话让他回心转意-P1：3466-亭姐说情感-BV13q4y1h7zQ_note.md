# 分手后被前任拉黑删除，一句话让他回心转意 - P1：3466 💔

在本节课中，我们将要学习当分手后被对方拉黑删除时，如何通过一条精心设计的信息来缓和关系，并可能让对方重新考虑。核心在于理解对方的心理状态，并采取正确的沟通策略。

## 理解拉黑删除背后的情绪 🧠

上一节我们介绍了课程的主题，本节中我们来看看对方行为背后的含义。

如果对方把你拉黑删除了，说明他对你还有很强的抵触情绪。

## 采取行动前的准备 ⏳

理解了对方的抵触心理后，采取行动前需要充分的准备。

你要先冷静个半个月。

这个冷静期（**冷静期 = 15天**）的目的是让双方的情绪平复，降低对方的防御心理。

## 关键信息的内容与发送 📝

在度过了必要的冷静期后，就可以发送关键信息了。以下是这条信息的具体内容和结构分析。

最近还好吗？被你删除拉黑了，我才发现给你制造了这么多的伤害。真的是抱歉，我现在才明白，想走的人不应该强留，你的决定是正确的，要不然我永远都不会长大。联系方式可以加过来吗？我不会再打扰你了，因为我不想失去一个朋友。最后呢还是希望我们都能够好好的。

这条信息的核心逻辑可以概括为：**道歉 + 理解 + 尊重决定 + 提出低压力请求 + 表达祝福**。

## 信息发送后的心态与后续 🔄

发送信息只是第一步，保持正确的心态至关重要。

我是小婷老师。如果说你也遇到情感问题，私信“复合”，我让你爱的人重新爱上你。
