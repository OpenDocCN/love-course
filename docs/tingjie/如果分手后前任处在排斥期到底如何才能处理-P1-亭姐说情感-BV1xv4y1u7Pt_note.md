## 排斥期的核心表现与心理机制

上一节我们介绍了课程主题，本节中我们来看看排斥期的具体表现。分手后，双方的心理状态截然不同。

被分手的一方，脑海中充满对方离开后带来的恐惧、恐慌、不甘、难过等情绪，同时也会反复回忆过去感情中的美好，以及对未来的期待和遗憾。

主动提出分手的一方则相反。他脑海中想的，基本上都是在感情中所感受到的不公、委屈、伤害和爱的缺失等负面情绪。正是这些负面情绪支撑着他下定决心离开，并开始新生活。

## 排斥期的外在行为与错误应对

基于上述心理，排斥期会出现一些典型的外在行为。以下是常见的表现形式：

*   改变外在形象和生活状态。
*   例如，购买新衣服、做新发型、旅游。
*   或与朋友聚会、认识新的异性、努力工作等。

需要注意的是，这也是新恋情的高发期。但无论对方做什么，这些行为的本质都是在淡化其自身的负面情绪。大多数人会在这个阶段选择纠缠以求复合。但请思考：对方正处于寻求开心快乐的阶段，怎么可能愿意与你复合，并回到之前那种令他不快乐的生活中去呢？

## 排斥期的正确行动指南

既然纠缠无效，那么正确的做法是什么？以下是此阶段你仅需专注的两点核心行动：

1.  **为过去的行为道歉，并承认分手的事实。** 公式表达为：**真诚道歉 + 接受现状**。
2.  **减少联系频率，但保持联系渠道。** 同时向对方展示你的积极改变与适度的难过。代码逻辑可描述为：
    ```python
    if 状态 == “排斥期”:
        行动 = 减少主动沟通(keep_channel_open=True)
        展示 = 自我改变 + 适度情感流露
    ```

## 排斥期的过渡与后续关键

当你妥善处理了排斥期后，双方的情绪会逐渐平复。我们将迎来分手后的第二个时期——冷静期。那个阶段才是复合的真正关键时期。绝大部分分手能否成功修复，取决于你在冷静期如何行动。
