上一节我们介绍了课程主题，本节中我们来看看第一种心理状态。

## 1. 悲伤 😢

主动提分手的女生可能比你更难过。她在感情上非常舍不得，但在理性上认为该做的努力都已做完，分手是必然结果，却又感到无可奈何。面对这种撕裂的情绪，她必然会感到悲伤。

如果你想复合，在这个阶段就必须老老实实闭上嘴巴，做好倾听和认可的角色。千万不要跳起来与她争辩，或一味地为自己做解释，那将犯下大错。

如果这个阶段处理得当，复合的几率至少能提高 **50%**。

理解了悲伤状态后，我们接下来看看第二种关键心理。

## 2. 期望 ✨

与男生的结果导向思维不同，女生更偏向过程导向。分手之后，她会不由自主地想起你们共同的美好过往和你的优点。

这些美好回忆与当下的处境形成强烈反差，会使她在痛苦中产生一个念头：**“我要不要再试一次？要不要最后给他一次机会？”**

所以，我之前强调过，面对主动分手的女生，除非因过度纠缠而需要短暂断联来缓和关系，其他情况切忌盲目断联。主动提分手的女生与男生不同，**越断联，她死心得越快，关系冷却得越迅速**。

以下是处理“期望”状态时的行动清单：

*   **接受朋友身份**：以朋友的身份陪伴在她身边。
*   **保持稳定存在**：确保当她产生复合念头时，你就在那里，并且情绪和状态稳定，甚至积极向上。
*   **避免消极表现**：分手后持续颓废会打消她复合的念头。

当我们妥善处理了对方的期望，接下来将面临复合路上最大的障碍。

## 3. 恐惧 😨

当她产生复合念头时，恐惧感会随之而来。她会思考：**“如果复合了会不会再次分手？他能改变吗？我还适合和他谈恋爱吗？”**

此时，她的心理防御机制很可能启动。

当你发现原本能正常聊天的她，突然回复变得非常冷淡，甚至直接拒绝你时，不必担心。这仅仅意味着她进入了“恐惧”的心理状态。

这时一定要反其道而行之，去安抚她。可以告诉她“慢慢来，线下做朋友也挺好，不要多想”，同时依旧不慌不忙地陪她聊天、逗她开心。

不要因为她的拒绝而方寸大乱，也不要因此害怕与她交流。只要你的情绪保持平稳，她也会慢慢平静下来，不再恐惧。你给予她足够的安全感，她才愿意向你靠近。
