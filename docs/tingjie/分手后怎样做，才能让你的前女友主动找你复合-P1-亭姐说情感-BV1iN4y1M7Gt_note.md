# 💔 分手复合指南：如何让前女友主动回头 - 第1课

在本节课中，我们将探讨一个核心问题：为什么在分手后，你的付出和示好反而会让前任感到压力并拒绝你？我们将从人性的底层逻辑出发，分析“基础项”与“加分项”在感情中的作用，并为你指明有效复合的正确方向。

你可能听过前任说这样的话：“你不要对我这么好，我很有压力。”或者“我知道你对我很好，但我们真的没可能了。”“我知道你真的很爱我，但我们有缘无分。”“你不要这样对我好了。”等等。

类似的话总会在你努力付出、试图复合的时候，给你当头一棒。

我知道你很伤心也很难过。但我问你一个问题：为什么他会对你如此绝情？

众所皆知，人性贪婪，本性就是趋利避害。没有人会拒绝对自己更有利的人生，更没有人会主动拒绝爱。

大家都想找一个对自己好的、可以和自己共度余生的人。对于前任而言，对他好、爱他的，不正是你自己吗？

所以你到底有没有想过，为什么他会反人性地去抗拒你、拒绝你呢？

接下来要讲的内容可能会颠覆你的认知，但请你仔细听。

**证明你爱不爱他、对不对他好，对于复合甚至爱情的建立，没有半毛钱关系。**

在爱情当中，没有人会拒绝爱他、对他好的人，这是人性，没有错。

但也正是因为这是人性，所以“爱他、对他好”只是爱情的**基础**。有才正常，没有才不正常。

你不爱他，你不对他好，你不为他付出，他怎么可能跟你在一起呢？

决定他们选择的，不是这些**基础项**，而是**加分项**。

你拼了老命地去付出一切去复合，到头来挽回的全部都是“基础项”。他为什么要跟你复合？他为什么不选择那个既有基础项又有加分项的人呢？

所以你还在纠结要不要给他送个礼物，你还在自责上一次跟他吵架没有顺着他的意思。这一切都是没有用的。

你要想一想，最开始你们在一起的时候，他是**因为什么**爱上你的？

虽然爱是无条件的，但**爱上一个人总是有原因的**。

*   肤浅一点，比如：`你的颜值高`、`你的身材好`。
*   深度一点，比如：`你的三观`、`你的幽默感`、`你的性格`。

事实上，很多人分手就是因为这个“加分项”在相处的过程中慢慢地流失掉了。

*   有很多男生在刚开始谈恋爱的时候，包容宽广，像是一个可以为女生解决一切的男人。一段时间之后呢？小气计较，对女生的要求视而不见。
*   有多少女生刚开始恋爱的时候，像一个永远温柔的天使。在恋爱了一段时间之后呢？暴躁易怒，一言不合就把“分手”挂在嘴边。

所以，你要去找到那些**曾经让他痴迷，但在爱情中流失掉的“加分项”**，然后重新呈现给他。

告诉他，他所爱的特质一直都在。

只有将这些曾经拥有和现在创造出来的“加分项”展现给他，你才能成为他眼中那个不只“爱他、对他好”，还是那个**值得他为你付出**的人。

他才有可能重新爱上你。
