# 情感修复课程 P1：真性分手复合六步法 💔➡️❤️

在本节课中，我们将学习一套针对“真性分手”情况的系统化复合步骤。真性分手通常意味着矛盾深刻，触及底线或形象严重恶化。我们将通过六个清晰的阶段，从降低抵触感到最终重建关系，逐步讲解如何有效挽回。

## 第一步：断联

上一节我们提到了真性分手的严峻性，本节中我们来看看第一步——断联。出现真性分手局面，原因通常是严重触及对方底线，或违背道德原则。另一种可能是过度纠缠导致个人形象彻底恶化。

此时，无论言语多么动听或行为多么讨好，对方都接收不到爱的信号。因为对方对你的抵触情绪正处于峰值，无法进行任何有效沟通。

你唯一需要做的，是用最有效的方式降低对方的抵触感和防备心理。断联正是为此服务。

## 第二步：打造展示面

在通过断联为关系降温后，本节我们聚焦于自我重建。两个相爱的人走到分手，根本原因在于一方或双方价值严重流失，导致彼此不满。

没有人会推开一个自己深爱的人，除非真的无法忍受。因此，挽回的核心是挽回那个曾经不够好的自己。

你需要把严重流失的个人价值，通过社交展示面的方式重新呈现出来。展示面做得好坏，是真性分手能否成功复合的最关键筹码。

## 第三步：复联

经过前两步的铺垫，对方抵触感已有所降低。本节我们将进入重新建立联系的阶段——复联。

以下是提升复联成功率的一个小技巧：**服从性测试**。

具体做法是，结合你对他的了解和现实生活情况，从他擅长的领域切入。你可以问他一个小问题请他解答，或求助他帮一个小忙。

**公式：对方回应程度 ∝ 1 / 当下抵触感**

对方的回应越积极，说明当下对你的抵触感越低。这种自然的复联方式，也能为后续更有针对性的操作做好铺垫。

## 第四步：建立聊天舒适感

成功复联后，对方可能重新对你产生兴趣，但更多是防范和怀疑。本节我们来学习如何通过聊天建立舒适感。

你需要控制好聊天频率，避免暴露强烈的需求感和目的性。

以下是选择话题的建议方向：
*   从互动性强的话题入手。
*   选择能输出高价值的话题。

通过这样的交流，可以进一步建立信任感，强化对方对你的兴趣指标。

## 第五步：邀约见面

当线上聊天建立起一定舒适感后，关系需要向线下推进。本节我们探讨如何通过邀约见面升级关系。

这个阶段的核心是升级暧昧关系，切忌因对方态度缓和而急于求和，否则容易导致“表白死”。

行动指南是：少说多做，不要主动提及感情。在约会过程中，通过自然的肢体互动来升级关系，是最舒适有效的方式。

## 第六步：确定关系

经历了线下约会的升温，关系终于来到最后一步。本节也是我们课程的终点——确定关系。

这一步无需多言，当感情通过前五步自然回流并加深，重新确定关系便是水到渠成。

---

本节课中，我们一起学习了真性分手后复合的六个核心步骤：**断联 → 打造展示面 → 复联 → 建立聊天舒适感 → 邀约见面 → 确定关系**。这套方法的本质是**先处理情绪，再重建吸引，最后自然升级关系**。请记住，挽回的终点不是对方，而是成为一个更好、更有价值的自己。
