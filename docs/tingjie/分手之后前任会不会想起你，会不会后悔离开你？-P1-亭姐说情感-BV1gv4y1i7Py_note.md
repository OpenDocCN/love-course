# 分手之后前任会不会想起你，会不会后悔离开你？-P1-亭姐说情感-BV1gv4y1i7Py_note

在本节课中，我们将要学习分手后前任的心理活动规律，并理解成功复合所必需的核心基础——降低需求感。掌握这些原理，是开启有效情感修复的第一步。

分手后，他还会想起你。答案是肯定的。他会不断想起你。他不仅会想念你的坏，还会想念你的好。

这种相思之苦，如果能受得了，他就不会找你复合。那他受不了，就一定会找你复合吗？那也不一定。

如果你抱有**过高的需求感**，他一般都不会找你复合。这就是为什么我一再强调，复合首先就是要降低需求感。不要再去窥探对方的生活，不要再去纠缠。他想要复合，你也想要复合，但只是因为你暴露了过多的需求感，他就不想复合了。

你可能会觉得这挺不公平的。确实，这个世界本身就是不公平的。感情本身没有对错，也没有公平可言。**谁爱得多，谁就弱势**。爱得多就说明你需要得多，无论是时间、身体还是金钱。

谁越在乎，谁就越放不下。越放不下，就越没有办法过好自己的人生，结果就越吃亏。对方的一举一动，都更加占据你的内心，直到占据你的全部世界。

当你的世界只剩下他，他就不得不选择离开。因为当你的世界里只有他，他还怎么走进你的世界呢？感情的世界没有对错与公平，只有谁更在乎谁、谁更放不下谁、谁更想占据谁的内心、谁更想依赖谁。

那么，分手后，他对你还有爱吗？爱你之前，他也是不爱的。由爱到不爱，由不爱到爱，这都是很正常的。

我很喜欢一句话：**爱不是大风刮来的**。爱是在一个个不起眼的小细节里，逐渐累积起来的。是在漫长的岁月里，一点一点蜂拥起来的恩与情。

感情就像事业，也像养宠物，需要我们认真对待、坦荡经营。不能一味索取，或一味享受。如果你一味索取、一味享受感情，关系指定不会长久，因为这本身就是不健康的。

如果你正在经历一些情感困惑，也可以寻求帮助，学习如何让你爱的人重新爱上你。
