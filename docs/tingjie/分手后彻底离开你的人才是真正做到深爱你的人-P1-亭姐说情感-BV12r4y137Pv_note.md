# 情感课程 P1：分手后彻底离开的人与深沉的爱 💔

在本节课中，我们将要探讨一种情感观点：分手后选择彻底消失的人，其爱意可能更为深沉和高级。我们将分析这种行为的心理动因，并与分手后纠缠不清的行为进行对比。

## 深沉之爱的表现

上一节我们介绍了课程的主题，本节中我们来看看这种深沉之爱的具体表现。

那种分手之后就彻底消失的人，他们的爱是很深沉很高级的。这里排除那种短暂交往后玩消失的人。前提是他之前真的爱过。分手之后他就彻底消失了，你再也看不到他的任何动态。

## 行为背后的心理动因

理解了行为表现后，我们进一步探讨其背后的心理原因。

一个人得多爱你多重视你，才会把这段感情当成了自己的全部。这个人是得多伤心多难过多失望，才会从你的整个世界里面消失。忍住思念不搭理，相比那种分手之后还纠缠不清、互相利用价值的行为，这种爱是要深沉很多的。

以下是支持这一观点的几个核心心理动因：
*   **极度的重视**：将感情视为全部，因此无法接受其破碎后的残局。
*   **深刻的伤痛**：巨大的伤心、难过与失望，驱使ta选择完全抽离。
*   **克制的尊重**：以“不打扰”作为最后的温柔和尊重，即便需要忍受思念。
