# 情感修复课01：三步打破心理防御，重建沟通桥梁 🔓

在本节课中，我们将要学习如何识别并打破对方在感情破裂后建立起的心理防御。当对方对你充满排斥和抵触时，任何沟通都难以进行。我们将通过三个清晰、可操作的步骤，帮助你化解对方的负面情绪，为关系修复创造机会。

## 识别防御信号：冷漠与回避

你有没有遇上过这种情况呢，不管你跟他说什么，他都态度冷冰冰的，要么敷衍的回你一两个字，要么就根本就不搭理你，工作和生活里面也尽量避开你，不想跟你有任何的牵扯。

这说明他对你的排斥和抵触情绪非常大。这是对方建立心理防御的明确信号，意味着常规的沟通方式已经失效。

## 打破防御的三步策略

但是别担心，今天我教你打破对方的心理防御，其实也没有你们想象中那么麻烦，只需要三步就足够了。

上一节我们识别了防御信号，本节中我们来看看如何一步步瓦解它。

### 第一步：弥补缺失的需求

分开就说明你们在过去的相处过程中，他有部分需求没有被满足。

以下是弥补需求的两个核心方向：

*   **陪伴需求**：如果他需要陪伴。你就要得让他知道，他需要人陪的时候你一直都在，你永远比其他人陪他的时间要长。
*   **空间需求**：他需要更多的个人空间的话，你就告诉他，你不会一直缠着他，也不会逼他做一些他不想做的事。

### 第二步：疏导负面情绪与合理归因

你要让他知道，他可以不顾忌的跟你表达他所有的情绪，因为你都能包容和理解他。

同时呢在沟通的过程当中，你要选择合理归因。以下是关键的表达范式，将指责转化为理解：

*   **关于“纠缠”**：为什么我一直烦你，让你感受到厌恶，不是因为我自私，而是因为我需要你。但是呢我不知道怎么来表达我的这一份需要。
*   **关于“错误”**：为什么我会走到今天这一步，不是因为我不爱你，而是因为我不知道怎么才算是珍惜。我让你失望了，但是我现在知道如何去爱人，我才意识到我自己之前的所作所为是多么的幼稚。

### 第三步：提高对方的自尊感

分开之后你不知道跟对方说什么，或者是担心对方是不是还在抵触你，那这个时候表达感谢就是一个很好的方式。

你可以这样说：
> 感谢过去这段时间你对我的包容，真的很感谢你。我第一次感受到了被接纳，我看到这段感情中你真的是牺牲了很多，之前是我忽视了你。

这种发自肺腑的感谢呀，真的是能瞬间让一个人不再去防御，停止抱怨。如果这时候他能够自然的看到你的理解和包容的话，他自然就会愿意去重新审视和你的这段感情。

## 总结与机会

至少呢他不会再那么排斥你了，这样你的机会不就来了吗。

本节课中我们一起学习了打破心理防御的三步法：**弥补需求、疏导情绪、提升自尊**。通过满足对方曾被忽视的需求，接纳其情绪并进行合理归因，最后给予真诚的感谢来提升其价值感，可以有效降低对方的防御心理，为关系修复打开一扇门。
