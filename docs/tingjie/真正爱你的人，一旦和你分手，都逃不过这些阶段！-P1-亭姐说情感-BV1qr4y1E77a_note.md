## 阶段一：纠缠与崩溃的边缘 🤯

上一节我们介绍了课程概述，本节中我们来看看第一个阶段。这个阶段的核心特征是情绪在极端之间剧烈摇摆。

真正爱你的人在分手初期，会陷入拼命纠缠对方的冲动中，同时又会产生干脆放弃的想法。这种矛盾的心理状态，导致当事人总是在情绪崩溃的边缘徘徊。

**核心矛盾公式**：`纠缠的冲动 + 放弃的念头 ≈ 情绪崩溃的边缘`

## 阶段二：失控的思念与深夜煎熬 🌃

理解了第一阶段激烈的矛盾后，我们进入第二阶段。这一阶段的痛苦更倾向于内化，表现为理智与情感的持续拉锯。

当事人清楚地知道不应该再去打扰对方的生活，但思念的情绪完全不受控制。这种无法掌控自己想法的状态，直接导致在一个又一个失眠的夜里辗转反侧。

以下是此阶段的典型心理活动列表：
*   理智上：知道不应打扰。
*   情感上：忍不住想联系。
*   结果：思想失控，严重失眠。

## 阶段三：矛盾心理的终极拉扯 ⚖️

经历了内耗的第二阶段，最终会进入更为持久和折磨的第三阶段。这是理智与感性之间一场没有硝烟的战争。

理智层面要求自己彻底消失在对方的世界里，以寻求解脱。然而，感性层面却被“万一他回头”的微小希望所控制。这种“舍不得又放不下”的矛盾心理，会持续地折腾当事人。

**心理状态代码描述**：
```python
while True: # 持续的心理循环
    if rational_thinking: # 理性思考
        decision = "彻底消失"
    else: # 感性思考
        decision = "等待回头"
    internal_conflict = True # 内心冲突恒为真
```

如果你发现自己正在经历类似的情感困扰，这恰恰说明用情至深。理解这些阶段是自我觉察和情绪管理的第一步。在后续的课程中，我们将基于此认知，探讨如何采取建设性行动，让你爱的人重新爱上你。
