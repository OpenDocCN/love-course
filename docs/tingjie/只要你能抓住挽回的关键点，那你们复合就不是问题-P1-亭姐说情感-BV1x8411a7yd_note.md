# 情感修复课程 01：理解“断联”的核心 🧩

在本节课中，我们将要学习分手后挽回的一个关键概念——“断联”。我们将深入探讨其定义、必要性以及如何正确执行，帮助你理解为何急于联系对方往往适得其反。

---

成功复合分手后，如果你想要挽回，通常有一个跨不过去的坎，叫做“断联”。顾名思义，就是要切断和对方的所有联系。不要时不时就去问对方“你为什么跟我分手”、“我到底是哪里做的不好”、“我改还不行吗”这类话。

很多人明明知道有时候需要断联，但生活中很少有人能真正做到。他们动不动就会想：“不行，我等不了了，我现在就要去跟他掰扯清楚，能在一起就在一起，不能在一起大不了就拜拜吧。”如果你也是这种心态，那我劝你别去了。你百分之百会得到“拜拜”的结果。

而且我还要补充一点：拜拜之后你也洒脱不了。如果你真有“不行就拉倒”的决心，你就一定要忍住。在这种特殊时候不要去找他，你啥也不说，直接当作已经分手了。说不定对方反而会开始回头。

记住，挽回的第一步，不是拼命联系对方去认错，也不是去对方公司门口堵着道歉，而是要去判断对方目前对你的**好感值**到底是在**正值状态**还是**负值状态**。也就是说，要看他现在对你到底还有没有负面的情绪。

如果他对你仍有负面情绪，那么你现在任何的联系和努力，对他来说都是雪上加霜，都是“烦不胜烦”。因此，你的行为都是在朝着“希望复合”的终点逆向行驶、背道而驰。

为什么呢？因为只有你让他对你的负面情绪**消减归零**了，你之后的付出和联系才有可能有用。**消除负面情绪，这才是我们说的“断联”这个操作真正的含义**。

---

本节课中我们一起学习了“断联”在情感挽回中的核心作用。我们明确了它的定义是**切断联系**，其根本目的并非惩罚或放弃，而是为了**让对方的负面情绪归零**。只有当好感值从负值回到零点甚至正值时，后续的挽回行动才会有效。记住，克制当下的冲动，是为了赢得未来的可能。
