# 分手后如何让女生主动找你复合呢？-P1-亭姐说情感-BV1Wg411h7iL_note

在本节课中，我们将要学习一个在情感修复中常见的困惑：为什么全心全意的付出与示好，有时反而会将对方推得更远？我们将剖析人性底层逻辑，明确“基础项”与“加分项”的区别，为你指明有效的复合方向。

你一定听过前任说过这样的话：“你别再对我好了，让我很有压力。”“我知道你对我好，但是我们真的没有可能了。”“我知道你爱我，但是我们真的没有办法在一起了。”这些话总是在你努力付出、努力复合的时候给你当头一棒。

这些话让你流泪，甚至让你产生崩溃的情绪。你可能会想问一个问题：为什么他会对你如此绝情？众所周知，人性趋利避害。人的本性就是趋利避害，没有人会拒绝对他有利的人生，更没有人会拒绝爱。

世界上不存在一个想要找一个对他不好的人谈恋爱。反而大家都在说：“我要找一个对我好的、爱我的、可以跟我共度余生的人。”而对于前任而言，对他好、爱他的那个人不就是你吗？他明明是心知肚明的。

所以，你到底有没有想过，他为什么会反人性地拒绝你？接下来的话可能会颠覆你的认知，但是你们一定要去听。证明你爱不爱他、对不对他好，对于复合甚至是爱情的建立，根本没有半毛钱的作用。

在爱情当中，没有人会拒绝爱他、对他好的人，这是人性没有错。但也正是因为人性，你爱他、对他好，这也只是爱情的基础。有才正常，没有才不正常。你不对他好、不为他付出，他怎么可能会跟你在一起呢？

你拼了老命地去付出一切去复合，到头来拿出的都是“基础项”。他为什么要跟你复合呢？他为什么不选择那个既有“基础项”，又有“加分项”的人呢？所以，你还在纠结“他马上就要过生日了，要不要给他送个礼物”？

你还在自责“你上次跟他吵架，你没有顺着他的意思”。说到底，他会不会爱上你，不是看你有多爱他，而是看你有多值得被他爱。所以，你要呈现一定的，不是这些基础项，而是你的“加分项”。你要想一想最开始你们在一起的时候，他是因为什么爱上你的。

虽然说爱是无条件的，但爱上一个人总是有原因的。肤浅一点的比如说：颜值好、身材好。深度一点呢？因为你的三观、你的幽默、你的性格。你身上一定存在着对他而言，只有你拥有的、无可替代的“加分项”。

事实上，很多人分手就是因为这个“加分项”在你们相处的过程中，慢慢地流失掉了。你要将这些曾经拥有和现在创造出来的“加分项”展示给他，你才能够成为他眼中的那个高分的伴侣，他才有可能重新爱上你、重新靠近你。

---

本节课中我们一起学习了情感修复中的一个关键误区：单纯依靠“付出”和“示好”这类“基础项”无法有效促成复合。核心在于，你必须重新发掘并展示你独特的、吸引对方的“加分项”，重建自身不可替代的吸引力。这是让对方重新考虑你的根本。
