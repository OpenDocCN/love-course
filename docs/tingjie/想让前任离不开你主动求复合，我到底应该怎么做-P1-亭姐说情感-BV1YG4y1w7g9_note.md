# 情感修复教程 第1课：重建吸引力与自我价值 🧠

在本节课中，我们将要学习亲密关系破裂的核心原因，并掌握两个关键原则，帮助你停止被随意对待，甚至让对方害怕失去你。核心在于理解人性规律并提升自我价值。

前任选择离开，无一例外，都是在两人拉扯的过程中，对方突然意识到一个问题：原来无需对你那么好也可以，你似乎更加离不开他。感情的微妙之处就在于此。一旦对方发现你只能通过发脾气来回应，你便失去了所有可以反击的武器。你甚至没有自己的底线和原则框架，这会导致他的行为越来越离谱。

上一节我们探讨了关系失衡的表现，本节中我们来看看如何改变这一局面。想让对方不敢随意对待你，甚至害怕离开你，其实只需要做到两点：**放弃幻想**与**尊重人性**。

人性是趋利避害的。一旦他“拥有你”的成本，远远大于他在这段感情中能够获得的收益，他就会迅速离开。这可以用一个简单的公式来理解：

**对方离开的倾向 ∝ (拥有你的成本 / 从关系中获得的收益)**

关系“高开低走”的关键原因在于，你将他最初的高配合度当成了常态，这是非常致命的。没有人可以因为爱一个人而完全放弃自尊，这也是人性。如果一个人任劳任怨，任你打骂也没有脾气，遇到这种人时，你需要警惕：他是否想从你身上得到其他东西？

以下是实现改变必须学会的另一点：弱化对爱情的滤镜，敢于筛选。爱情当然美好，但如果一个人只能在恋爱这个维度上获得快乐，那么这份快乐的价值就变得非常单薄。一份美好的感情只能是锦上添花。

这个世界上有意义的事情太多了。只有你不断发展自我潜能，才能让自己手握最大的选择权。所有你能想到的优秀特质，都是你筛选别人的筹码。只要你的价值足够大，偶尔的“作”也可能被看作是任性的可爱。

如果你一无所有，甚至连及格线都达不到，他人对你的忍耐值就会变得非常低。就像许多家庭主妇，其实并没有平等对话的权利。90%的两性烦恼都源于想要的太多，而自己能够匹配的太少。

换句话来说，就是被拿捏住了短处却不敢反抗。如果你正在经历一些情感困惑，可以寻求专业指导，学习如何让你爱的人重新爱上你。

本节课中我们一起学习了关系破裂的人性根源，并掌握了通过**放弃幻想**、**尊重人性**和**提升自我价值**来重建吸引力的核心原则。记住，强大的自我才是所有关系的基石。
