# 情感识别课程 01：识别潜在情感变化的迹象 🕵️♀️

在本节课中，我们将学习如何识别女生在情感上可能发生变化的几种关键行为表现。了解这些迹象有助于及时察觉关系中的潜在问题。

上一段我们介绍了课程主题，本节中我们来看看具体的行为表现。当一段感情出现变化时，对方的行为模式往往会发生一些微妙的转变。

以下是四种需要留意的具体行为表现：

1.  **信息回复变慢**：她回复消息的速度越来越慢，经常以“在忙”作为理由。实际情况可能并非忙碌，而是你对她而言不再那么重要。
2.  **态度转变**：她不再像过去那样对你撒娇，反而更容易发脾气，并时常表现出对你的厌烦。
3.  **沟通减少**：她不再主动与你沟通聊天。即使你主动发起对话，她也只是敷衍了事。
4.  **缺乏关心**：她不再关心你的心情、生活的好坏或情绪的起伏。她更宁愿独自玩手机。

如果女生出现以上四种情况，核心在于**情感投入度**的降低，可以抽象为：`她对你的关注度（Attention） < 关系维持所需阈值（Threshold）`。这时需要及时了解她产生情绪变化的原因，并有针对性地解决问题，而不是等到面临分手时才试图挽回。

本节课中我们一起学习了识别情感变化的四个关键行为指标：回复变慢、态度转变、沟通减少和缺乏关心。及时关注这些信号是维护健康关系的重要一步。
