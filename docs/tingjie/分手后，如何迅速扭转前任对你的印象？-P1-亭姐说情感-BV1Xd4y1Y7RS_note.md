# 情感修复课程 P1：扭转前任印象的核心理念 💡

在本节课中，我们将学习情感修复的核心理念。修复关系不仅是挽回感情，更是帮助双方重塑自我认知、建立平等健康关系的过程。我们将重点探讨如何通过自我表述打破失衡局面，引导共同成长。

## 情感修复的本质

上一节我们提到了课程主题，本节中我们来看看情感修复的根本目的。修复感情不仅仅是修复感情这么简单，还是帮助对方去重塑自我的一个认知体系。

而且在这个过程当中我们需要做到自我表述，众所周知两个人的分手呢一定不是一个人的原因造成的，你固然有自己的错误。

## 关系中的责任与平等

理解了修复的本质后，我们需明确关系中的责任归属。而对方也有对方不足之处，情感修复修复的是双方，帮助的是彼此。

在舒适感铺定的前提下，该大胆提出对方问题的时候，就一定要大胆的去表达说实话，舒适感铺垫的过渡之后呢，对方的潜意识当中会默认自己是高过于你的。

## 打破失衡局面的方法

认识到可能出现的失衡状态后，我们需要掌握打破局面的具体方法。而且还会在不知不觉的依赖于你，对你招之则来挥之则去，所以一开始呢这个局面是不可避免的。

为了打破这个局面你一定要去大胆表达，行事大胆一点，主动进行自我描述去改变对方的这个态度就好了。

以下是实施自我描述、建立平等关系的具体行动示例：

*   **回绝对方的某些要求**
*   **指出对方在相处过程当中的不足**
*   **强调这段关系当中的平等**
*   **阐述好的感情是需要相互支撑，彼此之间相互妥协的**

不要害怕，只要舒适感到位了氛围对了的话，对方是能听进去的。

## 引导共同反思与成长

通过主动表达建立平等沟通后，关系将进入新阶段。复合呢是共同成长的过程，我们在反思自己的同时也要引导对方同时也能够反思自己。

而且这个世界上其实呢没有什么完美的情人的，我们要学会跟对方的缺点进行去相处。

## 课程总结

本节课中我们一起学习了情感修复的核心理念。我们明确了修复是双向的成长，需要勇敢进行自我表述以打破关系失衡，并最终引导双方共同反思，学会接纳彼此的不完美。
