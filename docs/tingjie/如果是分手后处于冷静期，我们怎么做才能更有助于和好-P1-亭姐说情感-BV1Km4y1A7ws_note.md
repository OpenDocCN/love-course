# 情感修复课程 01：分手冷静期分析与行动指南 🧠

在本节课中，我们将要学习分手后“冷静期”的核心概念与关键行动。理解这一时期双方的心理状态，并采取正确策略，是决定感情能否成功修复的重要基础。

## 冷静期的定义与重要性

上一节我们介绍了分手后的不同阶段，本节中我们来看看最关键的“冷静期”。

分手后的第一个时期叫“排斥期”。主动提分手的一方会狠心放下一切想要逃离，而被分手的一方会疯狂地去求复合。当两个人经历了情绪拉扯期之后，就会进入分手之后最关键的时期——**冷静期**。

在两个人进入冷静期之后，双方都会为你们分手的原因，还有分手后的各种行为进行复盘。

## 冷静期双方的心理活动

以下是冷静期时，双方内心可能经历的思考过程：

*   **挽回方会思考**：之前感情中哪里不好、哪里不对、哪里该去改掉。如果在排斥期去求复合被反复拒绝之后，你也会想，对方是不是真的不爱你了，你是不是应该放弃了。
*   **提出分手方会思考**：你的前任经历了情绪的爆发期之后，他也会去回想你们之间的感情，是不是他也有错，是不是对你要求太多了，是不是没有给到你足够的耐心和理解等等。

很多挽回者其实都是在冷静期这个阶段选择了放弃，从而导致本来能复合的感情成为了遗憾。因为此时，提出分手的一方会进入一个很纠结的状态，他会开始考虑要不要复合。

## 窗口期的识别与应对

当冷静期发展到一定阶段，会出现一个关键转折点——“窗口期”。

他会测试你的态度。这个时候会出现一些表面行为，比如加固你的联系方式、观察你的动态、或者是突然打你的电话、用分割物品来测试你的反应，看看你的态度。

很多人这个时候就会去想，对方是不是也想复合，然后又开始了各种错误操作，从而让前任更加坚定了要和你分手。其实这只是分手后的第三个时期——**窗口期**。

这个时期前任从完全拒绝你的状态，变成了愿意重新去考虑这段关系的状态。这段感情能不能复合，也是这个时期去决定的。
