# 情感课程 P1：删除与拉黑的心理分析 🔪

在本节课中，我们将要学习“删除”与“拉黑”这两种行为在情感关系结束后的心理差异。我们将分析它们各自带来的影响，并理解为什么其中一种可能被视为更绝情。

---

很多人都说删除和拉黑的人绝情。但删除其实比拉黑更绝情。

拉黑之后，自己看不到对方，对方也搜索不到自己。这避免了在通讯录中看到对方的名字，也避免了在醉酒时拨通对方的电话。

上一节我们介绍了拉黑的基本效果，本节中我们来看看拉黑背后的情感动机。

拉黑让一切变得简单明了。因此，拉黑不算绝情，反而是深爱的表现。当事人害怕抑制不住思念，害怕因回忆心酸而重蹈覆辙、再次伤害对方。所以选择拉黑，让一切结束。

接下来，我们对比一下“删除”行为。

删除对方时，对方并不知情。你的信息会像陌生人一样，静置在对方的联系人列表中。对方可能无意中刷到你，再次看到时，会想起过去共同经历的点点滴滴。

以下是删除行为可能引发的一系列心理反应：
*   勾起的回忆让人眼角含泪，但过去已无法回去。
*   删除者被认为狠心，因为未作任何通知。
*   对方可能试图联系，发送信息后却收到“**对方已开启好友验证**”的回复。
*   对方可能不服气，尝试拨打电话。接通后，你问“**你找哪位？**”，这会让对方感觉如五雷轰顶。
*   对方会怨恨自己为何仍有期待，并懊恼自己不是先删除的那一方。
*   对方只能敷衍一句“打错了”，然后挂断电话。

你的态度显得风轻云淡、十分冷漠。这种冷漠给仍在爱你、期待你的人上了人生中最深刻的一课。冷漠才是插在人心口的一把钢刀。很多人说，删除只需一瞬，却足以了结一段人情。

一段感情无法继续，存在许多客观原因，不一定是不爱了。曾经在一起的两个人选择分开，放弃了欢声笑语和共同经历的山川湖海，或许是因为今生缘浅，又或许是因为在一起的痛苦大于快乐，最终两人选择结束。

---

本节课中我们一起学习了“删除”与“拉黑”在情感终结时的不同心理意义。我们了解到，拉黑可能源于保护与不舍，而删除带来的未知与冷漠，往往对另一方造成更深的心理冲击。处理关系结束的方式，本身也传递着最后的信息。
