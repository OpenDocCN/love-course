# 正在遭遇情感危机、冷战分手甚至

在本节课中，我们将学习一个为期30天的情感挽回计划。该计划分为三个阶段，旨在通过特定的行为策略，重新吸引对方并制造情感波动，为关系修复创造机会。

你是否正在遭遇情感危机、冷战分手甚至是闹离婚。不要怕，这里介绍一个方法，让你在30天之内尝试挽回。你需要分为三个阶段去使用这个方法。

## 第一阶段：彻底消失期（第1-10天）🌫️

上一节我们介绍了计划的整体框架，本节中我们来看看第一阶段的具体操作。

第一阶段，你需要在分手之后的头10天内彻底消失。

以下是本阶段的核心行动指南：
*   **不更新动态**：不要在社交平台发布任何新内容。
*   **不删除拉黑**：保留对方的联系方式，不要删除或拉黑。
*   **不回复消息**：如果对方发来消息，切记不要回复。

这个阶段的目的是**制造神秘感**，增加感情中的**不确定因素**。这样对方会认为你真的放手了，从而可能产生恐慌感，这有助于重新**制造你的吸引力**。

## 第二阶段：间接展示期（第11-20天）📱

完成了第一阶段的“消失”后，第二阶段我们将开始进行有限的、积极的自我展示。

从分手的第10天到第20天，你需要开始更新你的动态。

以下是本阶段的核心行动指南：
*   **发布积极内容**：发布一些积极向上且与情感无关的内容。
*   **展示精致自拍**：可以发布你精致的个人照片。
*   **不主动联系**：仍然不要主动去联系对方。
*   **适度互动**：可以时常去给对方的朋友圈点一个赞。

此阶段的核心公式是：**积极展示 + 无需求感 = 引发好奇**。通过展示更好的自己，同时保持距离，来引发对方的关注和好奇。

## 第三阶段：主动接触期（第21-30天）🤝

这是整个计划中最重要的阶段，我们将从间接展示转向一次精心设计的直接接触。

第三个阶段，也就是最重要的一个阶段，在你们分手的第21天到第30天，你可以发消息给对方。

以下是发起接触和见面的具体步骤：
1.  **发送见面邀约消息**：
    *   **情况A（曾同居）**：消息内容可为：“我现在路过你们家楼下。”
    *   **情况B（未同居）**：消息内容可为：“你过来给我送点东西。”
2.  **见面时注意事项**：见面时，不要提起任何有关情感的内容。
3.  **临别时的关键动作**：在你临走的时候，要去轻轻地拥抱一下他，告诉他：“我想你了，见到你真好。”
4.  **果断离开**：然后头也不回地继续走掉。

这个行为设计的目的是**让他的情绪产生波动**。你的行为会让他的情绪像过山车一样，**造成他的情绪落差**。但是这个时候你千万要沉住气，不要回他的任何消息。

完成了第三阶段的主动接触后，计划的固定部分就结束了，后续需要根据实际情况灵活应对。
