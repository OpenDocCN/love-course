# 情感修复课程 01：停止纠缠，开启复联 🚫➡️🕊️

在本节课中，我们将学习如何在分手后停止无效的纠缠行为，并通过一段关键信息来扭转对方对你的负面印象，为可能的复合创造机会。

说了一千遍一万遍不要纠缠。被你捧作神明的人不会低头看尘埃里的你。你过度纠缠之后再来问应该如何和好。你已经放下了自尊和面子，但他还是不肯回头。

上一节我们指出了纠缠的弊端，本节中我们来看看一个具体的方法来扭转局面。

今天教你一个有效的方法，可以迅速扭转他对你的负面影响。把以下这段话记下来，收藏，并直接转发给他。

以下是需要你发送的关键信息文本：

**“这些天我冷静下来了，是时候跟你说再见，和过去告别了。曾经我们都是彼此相爱，现在我们是彼此伤害。而一切好像都是我亲手造成的。我后悔对你的纠缠，后悔自己的自以为是，后悔我高估了我在你心里的位置。现在我真的有点想通了，也醒悟过来了。因为我发现爱真的不可以勉强。其实我的内心还是非常谢谢你，谢谢你曾经给我的陪伴，也谢谢你教会了我放下。所以现在跟你说再见，因为我打算重新振作，开启新的生活。”**

直接发给他，然后什么都不要管。什么都不要说，也不要去解释。直接开始断联。

无论他看到这条消息之后，是回你还是不回你，你都不要回复。耐心等待一段时间，让子弹飞一会儿。

如果你发完这段话，还有什么不懂的，可以在后续课程中寻找答案。

本节课中我们一起学习了停止纠缠的核心逻辑，并掌握了一段用于传递冷静、醒悟与告别态度的关键信息。记住，发送后保持断联是让这段信息生效的关键步骤。
