# 情感挽回课程 P1：如何掌握关系主导权？💪

在本节课中，我们将要学习在分手后挽回前任时，如何避免常见错误，并逐步建立自身在关系中的主导地位。核心在于控制情绪、提升自我价值，并通过策略性行为重建吸引力。

## 避免心浮气躁，认清现状

在挽回感情的过程中，一个大忌是心浮气躁。这表现为认不清自己当前在感情中的状态，盲目着急。心里虽然焦虑，却没有具体的解决方法。

人在着急的情况下，理智会变得不清晰。这会导致做出的行为和决定更加错误。

## 过激行为的后果

原本情况可能并不太糟糕，只要操作正确，就存在挽回的希望。但是，如果因为焦虑而做出一些过激行为，反而会加剧双方之间的矛盾。

这种行为只会让对方提防你、逃避你，不愿意与你进行更多交流。最终，你会在感情中处于一个十分被动的状态。

## 依赖性是问题的根源

造成这种被动局面的根本原因，是你过于依赖对方了。你把自身的全身心、精神情绪等都放在了次要位置，只需要对方给你情绪反馈。

公式可以概括为：**你的情绪状态 = f(对方的关注度)**。

只要他多花时间陪你聊天、多关心你一点，你整个人就像获得了能量。反之，他对你稍有冷落，你就会感到着急和不安，甚至发脾气或情绪崩溃。

## 掌握主导权的核心：降低对方警惕

男女之间的相处就像一场攻防战。你想要占领感情中的主导地位，就必须学会放低对方的警惕心。这需要你学会忍耐和控制自己的情绪。

如果你坚持相信“你已经吸引了他”，那么你就真的已经开始吸引他了。胡思乱想只会影响你的发挥，而坚信这一点，往往能让你真正地吸引到他。

## 利用“不联系”创造好奇

在挽回过程中，为了看到效果，你总是忍不住要去联系对方。你担心自己态度不够热情，对方就会以为你另寻新欢，从而放弃对这段感情的希望。

但是，你需要清楚一个策略：你不去找他，他反而会对你更加好奇，会忍不住去留意你的动态。这降低了他对你的戒备心。

在此之后，你的操作才会更加顺利。利用好冷静期来控制自己的情绪，能帮助你更好地掌握主导地位，并增加你的吸引力。

## 冷静期内的自我提升

在挽回中，双方必定会经历一个情感的冷静期。在这个期间内，你不应该只是“冷冻”关系，不做任何措施，这是不合理的。

在此期间，你需要完成以下关键任务：

以下是冷静期内自我提升的三个步骤：

1.  **找出自身新的亮点**：客观分析自己的优缺点，找到可以展示的新特质。
2.  **针对性地强化**：围绕找到的新亮点进行强化，而不是盲目提升。例如，你认为漂亮的服饰不一定适合你。
3.  **进行有效改造**：可以关注当下的时尚热点来改造形象，提升衣着品味。同时，尝试丰富自己的生活，提高生活品质，目的是带给对方耳目一新的感觉。
