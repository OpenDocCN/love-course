# 情感课程01：理解分手后女生的决绝心理 🧠

上一节我们提到了“考察期”的概念，本节中我们来看看处于这个时期的女生，其心理和行为会有哪些具体特点。主动提出分手的女生，往往已因种种原因开始打算放弃这段感情，并会表现出以下几个关键特征。
