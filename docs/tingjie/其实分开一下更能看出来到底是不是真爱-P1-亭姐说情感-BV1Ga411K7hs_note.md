# 情感分析课程 01：识别生命中不可或缺的真爱 💔

在本节课中，我们将学习如何通过分手后的具体心理与行为反应，来科学地判断一段感情是否属于“真爱”。我们将拆解四个核心的观察维度，帮助你进行清晰的自我诊断。

---

上一段我们明确了本课的目标，本节中我们来详细看看第一个关键判断维度。

**第一，睡眠规律出现显著紊乱。**
你每晚都很难入睡，到了早晨你又不愿意醒来。因为只要一醒来，你就必须得去面对已经分手的事实，所以你不愿意醒。在半梦半醒间还能骗一骗自己，痛感也就没有那么强。

---

了解了睡眠方面的表现后，我们接下来看看你在社交状态上的矛盾体现。

**第二，外在表现与内心感受严重割裂。**
你总是装作让自己看上去很洒脱的样子，但是一个人的时候，或者是说跟朋友在一起的时候，你就完全绷不住了。也可能会稍微保持一些理智，但是眼泪和心痛的感觉，止不住地向外流淌。

---

除了情绪和社交，行为模式也会发生改变。以下是关于“关注行为”的观察点。

**第三，产生无法自控的关注行为。**
你会经常地翻看对方的动态，不管是不是被删除了，你都会习惯性地去点开他的头像。慢慢地你就会发现，他不但动态没有更新，也渐渐地从三天可见变成一条红线。

---

最后，我们将探讨一种更深层的心理感受，这是判断真爱的核心体验之一。

**第四，持续体验到心理上的“空洞感”。**
你心里总是空荡荡的，时常有一种被掏空的感觉。也会经常回忆起曾经一起吃饭的餐厅，一起压过的马路，一起逛公园。但是一回过神来，却发现早都已经是物是人非了。

---

**总结与诊断**

本节课中我们一起学习了判断真爱的四个关键行为与心理指标。我们可以将其归纳为一个简单的判断公式：

**真爱可能性 = 观察(睡眠紊乱 + 表里不一 + 持续关注 + 内心空洞)**

如果说满足以上三点，毫无疑问，这就是你的真爱，千万不要轻易放手。如果说你现在正在经历一些情感的问题，也可以私信复合，我让你爱的人重新爱上你。
