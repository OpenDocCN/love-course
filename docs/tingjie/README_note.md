# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是赋值，公式如下：
**`变量名 = 值`**

例如，执行 `name = "婷姐"` 这行代码，就意味着创建了一个名为 `name` 的“盒子”，并在里面存放了“婷姐”这个数据。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的种类，这就是数据类型。Python有一些内置的基本数据类型，最常见的有以下三种。

以下是三种核心的基本数据类型介绍：

1.  **整数**
    整数是不带小数点的数字，可以是正数、负数或零。
    例如：`age = 18`

2.  **浮点数**
    浮点数是带有小数点的数字，用于表示实数。
    例如：`price = 9.99`

3.  **字符串**
    字符串是由零个或多个字符组成的文本数据，需要用单引号（`'`）或双引号（`"`）括起来。
    例如：`message = "Hello, World!"`

## 如何查看和使用变量？ 👀

知道了如何创建变量和其类型后，我们来看看如何查看和使用它们。在Python中，使用 `print()` 函数可以将变量的内容输出到屏幕上。

代码示例如下：
```python
# 定义变量
name = "婷姐"
age = 18

# 打印变量
print(name)
print(age)

# 打印组合信息
print(name, "今年", age, "岁。")
```

运行这段代码，你将在屏幕上看到：
```
婷姐
18
婷姐 今年 18 岁。
```

## 总结 📝

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。

我们首先将变量理解为一个存储数据的命名容器，并通过 **`变量名 = 值`** 的公式进行赋值。接着，我们认识了三种基本数据类型：用于整数的**整数**、用于小数的**浮点数**以及用于文本的**字符串**。最后，我们掌握了使用 `print()` 函数来查看和输出变量值的方法。

理解这些概念是后续学习更复杂操作（如计算、判断、循环）的基础。请务必动手尝试文中的代码示例，加深理解。
