**真正的复合**意味着：在感情破裂、生活与人际关系陷入混乱后，**其中一方能够主动站出来，推翻所有积累的负面情绪**。

**主动修复 = 承担情绪责任 + 引导关系转向**

*   **思维模式的转变**：你们学会了换位思考，开始习惯站在对方的角度考虑问题。
*   **冲突解决与模式建立**：你们能够解决彼此间的矛盾，并共同寻找到让两个人都感到舒适的相处模式。

当新的相处模式建立起来后，关系中的沟通与安全感会得到质的提升。本节我们来看看健康复合关系中的沟通与情感保障。

复合后的感情中，双方会建立起更坦诚、更有安全感的沟通方式。

*   **直面历史**：双方会坦然讨论过去的过错，不再逃避问题。
*   **情绪管理**：双方不会再因情绪上头而无法面对矛盾。
*   **开放分享**：任何开心或不开心的感受，都可以彼此分享。
