# 分手后放大他的愧疚感，他就能回头找你复合 - P1：333 🎭

在本节课中，我们将要学习如何通过放大对方的愧疚感，来促使他主动回头寻求复合。我们将探讨愧疚感在分手后的作用机制，并介绍一个核心的操作方法。

## 愧疚感的作用机制 💡

上一节我们提到了课程的核心目标，本节中我们来看看愧疚感为何能产生如此大的影响。

愧疚之心人人有之。当他对你提出分手时，内心通常会伴随一定的愧疚感。他的愧疚感越强，就越舍不得这段感情。他的潜意识会时刻地提醒他自己，仍然需要你。

## 如何扩大愧疚感：核心方法 🛠️

了解了愧疚感的重要性后，我们接下来看看具体如何操作才能有效地扩大这种感受。

最让对方愧疚的情况，一定是被抛弃的那个人，选择不吵不闹，并默默地答应分手。

以下是两种不同回应的对比及其效果：

*   **有效方法（默默接受）**：这种行为会让他预期的冲突和辩解落空，从而使他内心的愧疚感无法被抵消或转移，持续累积。
*   **无效方法（纠缠指责）**：如果你去纠缠和指责他，那么在他看来，你的行为像是在宣泄情绪。那么他既承受了你的宣泄，他对你的愧疚感，就减少了很多。
