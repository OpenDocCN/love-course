# 不要把自己搞得很卑微很低位，分

在本节课中，我们将要学习分手后如何避免陷入卑微纠缠的误区，并通过理解人性与心理，建立平等姿态，从而掌握感情复合的主动权。核心在于价值提升与心理博弈，而非单方面的乞求。

## 走出卑微纠缠的误区

分手后想复合，依靠的绝不是卑微的纠缠。通过卑微纠缠得来的感情，在后续的相处过程中会让你变得患得患失、小心翼翼，生怕自己做错事。这种情况在分手后前任疑似有新欢时，尤为令人难以承受。

但现实是残酷的。如果想和好，就必须忍受这个过程。因此，对于那些想要逆袭、想要翻身的小伙伴，接下来要认真听。

## 理解感情的可琢磨性

在人与人的交往中，一个人做了什么事，就会给你带来什么样的情绪体验。所以，感情其实是一种可以琢磨的东西。

例如，一个人一直以来的性格、潜意识、行为都是有迹可循的。因此，稍微有点经验、懂点人性的人，就能够推断出他的处事方式。既然两个人曾经相爱过，就意味着你们是旗鼓相当的。

## 挽回前的两个核心准备

在采取挽回行动之前，你需要先确定两件事。

以下是需要评估的两个关键方面：
1.  **价值评估**：你的软硬价值（如外貌、经济、性格、情商等）是否可以与前任均衡地较量。
2.  **心理洞察**：你能否读懂并把握前任的心理想法。

如果目前你们的状态是真性分手，分手后你尝尽了各种方法，把自己搞得很卑微、很低位，甚至前任已经开始接触新的人，那么接下来才是真正的较量。

## 框架概述与抗压测试

篇幅所限，今天先讲一个大概的框架。后续会分享真实案例。首先，我们讲一个考验自身抗压能力的例子，这个问题几乎每个人都会做错。

例如，当你开始表现得很自然，让前任认为你已经走出失恋、准备接触新生活时，前任可能突然拉黑删除你，或者说一些祝福你的话，如“你会遇到更好的人”、“我不值得”等。这其实都是**反向试探**。

## 识破试探与保持姿态

他就是通过一系列打压你的动作，来观察你是否有应激反应。对于那些沉不住气、急眼了跑去质问的，或者赶紧各种示好的人，那就破功了。你的前任也就放下心来，因为他知道你这些都是伪装的。

那么他就会继续吊着你，搞你的心态。如果你想逆袭翻身，就必须学会引导前任。

## 引导的关键：制造心理反差

至于如何去引导，说白了就是让他难受，让他想不明白，让他去猜。因此，你不能顺着他，不能听他的安排。

我们一直要酝酿一种**抽离的姿态**。这种姿态等同于宣告：你既然不害怕失去我，那我也不会害怕失去你。

假如在这个过程中，你自身的价值（无论是内在还是外在）都有所提升，这就会让前任产生巨大的心理反差，从而可能过来反扑你。

## 总结与心态调整

本节课中，我们一起学习了挽回中避免卑微心态的重要性。其实挽回这件事，没有想象中那么复杂。在感情里，你越是紧张，就越难达到预期。往往心态放平，把事情简单化，好运也就来了。

**核心公式**：**高价值 + 平稳心态 + 心理引导 > 卑微纠缠**

如果你正在经历情感问题，也可以寻求进一步指导。记住，目标是让爱的人重新爱上**有价值的你**，而非怜悯你。
