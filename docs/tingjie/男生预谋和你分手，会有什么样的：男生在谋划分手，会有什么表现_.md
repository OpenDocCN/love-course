# 男生预谋和你分手，会有什么样的：男生在谋划分手，会有什么表现_

在本节课中，我们将学习如何识别男性在预谋分手时可能出现的几种关键行为表现。了解这些信号有助于你及时察觉关系的变化，并思考应对策略。

上一节我们介绍了课程主题，本节中我们来看看具体的表现有哪些。

如果男性出现以下几种行为，可能代表他正在预谋分手，你需要开始留意。

以下是四种常见的预谋分手表现：

1.  **他不再主动联系你。** 这表示他可能试图通过冷战的方式，迫使你先提出分手。
2.  **他对你忽冷忽热。** 他只会在自己感到无聊时才会想起你，而不会主动关心和寻找你。
3.  **他永远都在忙。** 他忙到没有时间联系你，也没有时间陪伴你。
4.  **他对你不闻不问。** 无论你做什么或说什么，他都不再在意。即使你半夜和朋友外出，他也不会过多询问。

遇到表现出上述行为的男性，他很可能正在预谋与你分手。
