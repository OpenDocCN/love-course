同样是“作”，每个女孩的动机却不尽相同。有人是为了得到更多的爱，有人是被宠坏了，也有人只是单纯的脾气不好、作而不自知。虽然她们最终都得到了分手的结果，但想要挽回时，针对不同的类型也应该有不同的解决方案。

上一节我们概述了“作”的不同动机，本节中我们来看看第一种具体类型及其挽回方法。

## 第一类：缺乏安全感型 🔒

缺乏安全感的女孩在感情中非常不自信，甚至是自卑的。她们潜意识里不相信会有人爱她，认为自己不值得被爱，总结为**极度缺乏安全感**。

为了获得内心世界的安逸，她们会疯狂地向另一半索取爱，希望对方的一举一动都在自己的掌控之下。每天都会去问对方：“你爱我吗？”“你去哪儿？”“去多久？”“回来还会爱我吗？”严重者还会限制对方的社交，提出很多过分的要求。

如果对方不照做，就是不爱自己，并企图以各种极端的方式让对方证明他有多爱她。造成这种心理的缘由有很多，家庭关系的影响、过去感情的创伤，都有可能让一个人走进这样的失控情绪中。

而对方选择与你分手的原因，也一定是因为你过界的爱压得他喘不过气了。

**核心概念：安全感与信任的平衡**
感情的基础是信任。如果你不能试着去信任她，那么你的安全感就是填不满的无底洞。公式可以表示为：
**安全感 ≠ 无限索取与控制**
**安全感 = 自我价值感 + 相互信任**

在这种情况下，你要学会给对方空间。你要清楚，就算他对你再好、再爱你，如果你无法建立信任，你也得不到真正的满足。挽回的关键在于重建信任感，并提升自我价值。

上一节我们探讨了因安全感缺失而“作”的情况，接下来我们看看第二种因被偏爱而产生的行为模式。

## 第二类：恃宠而骄型 👑

在感情里，被偏爱是一件幸运的事。这意味着对方把一把刀递到了你的手里，给了你可以随时伤害他的机会。而很多女生也就是仗着自己有这样的机会，所以对别人的伤口视而不见。

她们享受着对方的爱和好，把这一切都当做是对方应该做的，是理所应当的。直到有一天对方遍体鳞伤地离开，才追悔莫及。

如果你也是一段感情中被爱的那一个，那想要挽回对方的时候，你就应该好好反思一下自己了。

以下是几个需要自省的关键问题：
*   你是不是真的爱她？
*   你有没有在不知不觉间伤害过她？
*   有没有不把对方的付出放在眼里？
*   有没有觉得她对你的好都是理所应当的？

如果答案是有，那么**道歉，真诚地去道歉**。这是挽回的第一步，也是重建关系平等尊重的基石。

分析了因被偏爱而“作”的行为后，我们最后来看一种当事人可能并未察觉的情况。

## 第三类：作而不自知型 😶

其实有很多女孩子，直到和对方分手的那一刻，都没有意识到自己其实很“作”这件事。她们甚至会以为这只是一次寻常的吵架、寻常的分手，等自己气消了，对方也冷静下来了，两个人还是会和好如初。

直到这一次，对方再也没有来求你和好，甚至对她主动的示好视而不见。

喜欢闹分手绝对是恋爱中的大忌，因为男人和女人对于分手的认知不同。在男人眼中，分手了就应该了断这段感情。而女生则是把这些当气话，并不影响她的爱。

**核心概念：对“分手”的认知差异**
代码可以描述这种思维差异：
```python
# 女性常见思维 (简化模型)
if feeling_upset == True:
    say(“我们分手吧！”) # 意图：表达情绪，寻求关注
    actual_intention = “希望对方挽回和安抚”

# 男性常见思维 (简化模型)
if hear(“我们分手吧”) == True:
    process_as(“关系终止的正式声明”)
    response = start_moving_on() # 开始疏离，准备结束
```

所以，如果你经常对你爱的人发脾气、闹分手，那么这一点绝对是要改的。在挽回这样一段感情时，就不要再想着去赌气，期待对方还会像以前一样先来找你。尽快主动地去给对方一个台阶才是上策。

如果对方打定主意不接你的台阶，那么你就只能下点功夫，好好去哄一哄了。男生也是有脾气的，很多时候也是需要被哄的。
