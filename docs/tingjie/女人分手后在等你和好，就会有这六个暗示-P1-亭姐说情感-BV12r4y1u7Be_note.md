# 情感修复课程 01：识别她等待复合的六个关键暗示 🔍

在本节课中，我们将学习如何识别女性在分手后可能仍在等待关系修复的六个关键行为暗示。理解这些信号是迈向有效沟通和潜在和解的第一步。

---

上一段我们明确了本课程的目标，接下来我们将逐一解析这些具体的暗示行为。以下是女性可能仍在等待复合的六个常见表现：

1.  **分手后态度冷淡，言辞决绝**
    她可能表现出冷漠或说出非常绝情的话。

2.  **拒绝接听电话与回复信息**
    她不接你的电话，也不回复你的任何信息。

3.  **对你的所有动态选择无视**
    她对你的任何消息都采取视而不见的态度。

4.  **回避任何当面沟通的机会**
    当你试图约她见面交谈时，她总会找各种理由推脱。

5.  **切断你们的直接联系渠道**
    分开后，她拉黑了你们之间的联系方式。

6.  **在短期内开始新的恋情**
    她迅速有了新欢，以此让你知难而退。

---

我们刚刚列出了六个具体的暗示。这些行为的核心逻辑在于：**她正在逃避**。

你可以思考一下，如果她不是真的**放不下**，又怎么会持续不断地选择**逃避**呢？这种回避行为本身，往往反映了内心未完全割舍的情感纠葛。
