# 情感修复课程 01：掌握四个阶段，稳定复合心态 🧠

在本节课中，我们将学习如何在尝试复合的过程中，有效管理自己的情绪，避免被前任的行为过度影响。核心在于理解分手后对方心理变化的四个典型阶段，并据此调整自己的心态和策略。

---

大部分尝试复合的人，在此过程中都极易被前任的行为影响自身心态和情绪。网络上虽有众多关于心态建设的建议，但往往未能触及根本。根本问题在于，你始终无法准确知晓对方的真实想法。

言语无法作为准确判断的依据。当对方对你抱有负面情绪时，他完全可以说反话。因此，你依然会被他的情绪和行为带偏方向。本节课程旨在将这个问题彻底讲透。

上一节我们指出了问题的核心，本节中我们来看看具体的解决方案。理解对方分手后的心理变化规律是关键。

前任在分开后通常会经历四个心理阶段：排斥期、空白期、思考期和怀念期。每个阶段中，对方的心理和情绪特征都不同。只有透彻理解这四个阶段的情绪与行为特征，并明白这些都是对方合理的情绪反应，我们才能最大限度地调整自身情绪，建设稳定心态。

以下是四个阶段的详细特征分析：

*   **排斥期**：典型特征是对方极度排斥，对你的任何举动视而不见、听而不闻，其核心心理可概括为 **“看见你都烦”**。
*   **空白期**：分开后，对方会进入一段对你无感的状态，其核心想法是 **“你什么样子跟我没有关系”**。
*   **思考期**：对方会开始测试你，并在你回应后态度忽冷忽热。更甚者，可能会反复将你拉黑或删除。
*   **怀念期**：在此阶段，对方内心开始怀念你的好，产生复合的想法会变得非常强烈，表现为经常主动找你聊天、关心你。

了解了这四个阶段的行为情绪特征后，我们必须做好清晰的思想准备。从决定挽回开始，你就要接受分手的事实。

复合这件事，本质上需要耐心和理智。你必须做好打持久战的准备。

---

本节课中，我们一起学习了分手后对方心理变化的四个阶段：排斥期、空白期、思考期和怀念期。理解这些阶段的特征，能帮助我们预见对方的行为，从而不被其轻易影响，更好地稳定自身心态，为理性的复合行动打下坚实基础。
