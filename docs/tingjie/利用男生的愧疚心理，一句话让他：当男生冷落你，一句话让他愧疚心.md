# 利用男生的愧疚心理，一句话让他：当男生冷落你，一句话让他愧疚心

在本节课中，我们将要学习一种在感情中被冷落时的沟通方法。当对方开始忽视你、态度冷淡时，如何通过一句话引发对方的愧疚感，从而为关系改善创造机会。

任何情侣在一起时间久了，男生都难免会产生倦怠的心理。因此，他开始忽视你，冷淡你，甚至可能想逼着你先提分手。你可能会感觉自己变得无所谓了，感觉自己的真心被人践踏了，但又舍不得这段感情。

接下来，我将教你一句话。这句话的目的是让他内心产生愧疚，从而再次心软并靠近你。

以下是这句话的具体内容：

> 是不是我只要选择放手，我们之间就结束了？我花光所有的运气才遇到你，但是现在却不得不放弃。我感觉你的世界已经不需要我了，我不想再打扰你了。或许只有这样子，我们才能够各自安好。

说完这句话后，如果对方回复了，你**一定不要回复**。你需要冷着他，让他感到忐忑，促使他进行反思。
