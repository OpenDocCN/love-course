## 概述：愧疚感的巨大能量 💥

上一段我们提到了课程主题，本节中我们来看看为什么愧疚感有如此大的杀伤力。

愧疚是最大的情绪能量。分手后，对方的愧疚感具有巨大杀伤力。主动提出分手的人并非如释重负，转身后可能心如刀割。很多时候，主动分手的人会更痛苦，因为他们是做出“壮士断腕”决定的一方。双方在关系中都投入了时间和青春。大部分主动分手的人，通常是攒够了失望，曾经有多爱，分开就有多痛。

一段感情破裂，主动分手的人在痛苦基础上，还需承担一种与愤怒同样伤人的情绪，这种负能量就是愧疚感。在亲密关系中，愧疚感的作用显而易见，它可以缓和双方矛盾。前任的愧疚感越强，就越舍不得这段感情，越忘不掉你，从而越想珍惜你并找回你。

## 激发愧疚感的核心策略 🛡️

上一节我们理解了愧疚感的力量，本节中我们来看看如何具体激发这种情绪。

以下是几个关键点，教你如何激发前任的愧疚感并利用它来挽回。

### 1. 避免错误行为，不给对方消除愧疚的机会

不要做不理智的事情，不要给对方消除愧疚感的机会。错误行为包括：疯狂发微信求复合；拉黑删除前任的家人或朋友并昭告天下；在朋友圈散发负能量，给旁人提供笑料；丢掉前任的物品泄私愤；找前任的家人或朋友控诉他。

分手极端状况下，很多人会做出失智行为。这相当于一顿操作猛如虎，事后恨自己是二百五。本来对方正愁无法摆脱内心愧疚感，你的这些行为直接给人递了梯子，简直是助人为乐的“中国好前任”。相反，如果他做错事，你有罪，你选择接受甚至以德报怨，那么他的愧疚感就会达到顶峰。

### 2. 安静接受分手，优雅还对方自由

这个点不做详细阐述，最主要的是要理解：**爱而不得是人生的常态**。你什么都没有做错，表现得越优雅，你就越占理。你占理了，他就理亏了，心里一定会犯嘀咕。在这种情况下，他会产生一种认知失调，在这种状态里不断反思或疑惑“分手是错的”，这个种子由此埋下。

### 3. 专注自我提升，让他觉得失去你是损失

这个点最重要的是，分手后一定要把关注点放到自己身上。去做那些你之前一直想做但不敢做、或没有时间做的事情。纵然最终挽回不了也不亏，毕竟你变美了、变强了，自己才是最大的受益者。

### 4. 时机成熟时，技巧性展开复联

时间成熟、对方愧疚感爆棚时，可以适当展开复联。但要记住，复联也要讲究技巧。经过一段时间分开和情绪沉淀，你们的关系已经破冰回春，他实在没有拒绝你的理由。但是，如果你的借口烂、演技差，对方一眼看出你是故意的，便失去了挽回的先机。

### 5. 提高不可得性，不轻易答应复合

千万不要轻易答应他的复合请求。说白了，就是让对方觉得你很难得到，提高自己的不可得性。我们都知道，求挽回的人在情感中本就处于低位，容易被对方压着一头。人性本贱，越是轻而易举得到的东西，越不珍惜。
