## 概述：复合是一个有阶段的递进过程

从分手到成功复合，通常会经历八个明确的情绪与互动阶段，同时关系性质会跨越六个层次逐步升级。理解这些阶段与关系层，能帮助你避免错误操作，系统性地推进修复。

## 八个复合阶段详解

上一节我们概述了整体框架，本节中我们来详细看看每一个阶段的具体表现和核心任务。

### 第一阶段：分手与冷淡期
对方提出分手，明确表示不愿继续关系。行为上开始对你冷淡，甚至不回复消息。

### 第二阶段：纠缠与关系恶化期
你因无法接受分手而开始纠缠、道歉、认错、做出保证。这一系列操作往往导致对方更加想远离你，甚至可能删除或拉黑你的联系方式。

### 第三阶段：冷静与关系转折期
你开始冷静下来，明白需要采取正确行动。停止纠缠，进行“冷处理”。此时，双方关系从敌对状态转向陌生人状态。

### 第四阶段：尴尬与试探期
经历一段冷冻期后，你们进入尴尬的陌生人时期。当你尝试重新接触时，很可能再次碰壁，因为对方平静的生活被打破，对你的负面印象仍有残留，初期反应会比较冷淡。

### 第五阶段：自我提升与关系回暖期
你开始研究并选择正确的话题进行联系，降低了联系频率，增加了对话的舒适感和满足度。同时，你的自我提升初见成效，并被对方察觉。对方开始犹豫是否要再给你一次机会，但同时也担心重蹈覆辙，怀疑你的改变只是暂时伪装。

### 第六阶段：线下破冰与印象重塑期
你尝试邀约见面以突破关系。初次见面时，你保持了情绪稳定和良好表现，给对方留下了积极印象。但你依然不会主动提出复合，因为你明白对方虽有惊喜，但仍有犹豫。

### 第七阶段：顾虑消除与关系升级期
经过一段时间沉淀，你在线上联系和线下邀约中表现出一致性，基本打消了对方的顾虑。前任开始尝试与你暧昧，愿意配合你，并开始怀念你的好。关系从普通朋友进阶为亲密朋友，并进入暧昧期。

### 第八阶段：契机把握与复合达成期
随着暧昧期频繁接触，会出现某个关键契机。你抓住这个契机，运用技巧（如刺激对方占有欲）或突然后撤（制造失去感），使对方在情绪驱动下主动提出复合。

## 六层关系递进模型

理解了阶段后，我们来看看关系是如何一层层递进的。以下是关系演变的六个层次：

1.  **敌对关系**：对应第二阶段，因纠缠导致关系恶化。
2.  **陌生人关系**：对应第三、四阶段，停止纠缠后关系归零。
3.  **普通朋友关系**：对应第五阶段后期，舒适感初步建立。
4.  **亲密朋友关系**：对应第七阶段初期，互动加深，顾虑减少。
5.  **暧昧关系**：对应第七阶段后期，互动充满情感张力。
6.  **复合关系**：对应第八阶段，成功重新建立恋爱关系。

## 总结与自测

本节课中，我们一起学习了分手后走向复合的八个具体阶段和六层关系递进模型。你可以根据描述，自测当前与前任所处的关系阶段。理解这个路线图，能帮助你在情感修复过程中保持耐心和方向，避免因焦虑而犯错。

**核心行动公式**：`停止错误纠缠 -> 自我提升与冷处理 -> 重建舒适感 -> 线下破冰 -> 展示一致性 -> 抓住契机推动复合`
