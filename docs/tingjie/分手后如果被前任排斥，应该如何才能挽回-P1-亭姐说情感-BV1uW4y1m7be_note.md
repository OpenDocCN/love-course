在本节课中，我们将要学习当前任表现出排斥态度时，如何通过把握四个关键时期来有效挽回关系。课程内容将拆解每个阶段的核心策略与行动要点。

---

当他排斥你并且你非常想挽回时，必须采取正确的方法。我们一定要把握好以下四个时期。

## 1. 排斥期 🚫

在排斥期，他会非常反感你。因此，我们一定要避开谈论感情，一定不要去纠缠，让他平复自己的情绪。

以下是此阶段的核心行动准则：
*   不要谈论感情话题。
*   不要进行任何形式的纠缠。
*   避免谈论过去或翻旧账。

上一节我们介绍了需要完全避免接触的排斥期，接下来我们看看关系开始解冻的阶段。

## 2. 缓和期 🤝

进入缓和期，他已经开始慢慢接受你，能和你正常交流。此时，可以和他适当增加一些聊天的频率。

了解了关系缓和的信号后，我们需要进一步重建情感连接。

## 3. 吸引期（情感唤起） 💭

在吸引期的第一阶段，目标是唤起美好记忆。你可以通过一些共情，或者回忆你们之间美好的点点滴滴，让他重新想起你的好。

成功唤起对方的情感记忆后，我们需要展示切实的改变来巩固吸引力。

## 4. 吸引期（价值展示） ✨

在吸引期的第二阶段，重点是让他看到你的变化。公式可以概括为：**价值展示 = 外在提升 + 内在改进**。
无论是你外观穿着的改变，还是你以前的一些缺点现在得到了很好的改善，都需要被看到。

---

本节课中我们一起学习了挽回的四个关键时期：**排斥期**、**缓和期**、**吸引期（情感唤起）** 与 **吸引期（价值展示）**。核心在于根据对方的不同心理阶段，采取从“避开冲突”到“展示改变”的渐进策略。
