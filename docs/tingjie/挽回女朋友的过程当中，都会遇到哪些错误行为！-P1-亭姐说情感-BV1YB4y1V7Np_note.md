## 错误一：分手后哀求对方复合

上一节我们介绍了本课程的目标，本节中我们来看看第一种错误行为。

当对方向你提出分手时，代表着他已经对你的一切都加以否定。因此，分手后千万不要向对方表现出一副楚楚可怜的样子，去低三下四地求对方复合，甚至去纠缠对方。否则只会徒增对方的心理压力，对你更加鄙视，更加合理化自己的分手决定。

**核心逻辑**：低姿态的哀求 = 强化对方的负面印象 + 验证其分手决定的正确性。

## 错误二：给出过多的承诺

了解了哀求的弊端后，我们再来分析第二种常见的错误做法。

以下是这些承诺通常无效的原因：
*   这些话语会让对方感觉你在敷衍，没有真正思考自身的错误。
*   对方会认为你根本没有考虑过如何具体改正。
*   即使对方暂时因承诺回到你身边，当你无法兑现时，只会让对方更加坚决地离开。

**核心要点**：空头承诺 ≈ 敷衍。无法兑现的承诺会导致二次伤害，让关系彻底破裂。

## 错误三：以一成不变的形象出现

前面我们讨论了语言上的错误，最后，我们来看看行为模式上的关键误区。

分手后的挽回，最有效的方法之一是改变自我、提升自己。倘若在分手后你没有反省自身的问题，没有去改变自己的缺点，就一味按照自己的想法去挽回，这样做只会对你的挽回毫无作用。

**核心公式**：**旧自我 + 旧方法 = 失败挽回**。改变是打破僵局的关键。
