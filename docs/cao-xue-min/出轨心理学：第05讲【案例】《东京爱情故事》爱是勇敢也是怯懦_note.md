# 出轨心理学：第05讲【案例】《东京爱情故事》爱是勇敢也是怯懦

## 概述
在本节课中，我们将通过分析电视剧《东京爱情故事》中的案例，探讨爱情中的勇敢与怯懦，以及安全感和刺激感在人际关系中的作用。

## 案例分析
### 故事背景
- **《东京爱情故事》**：1991年首播的经典日剧，讲述了东京女孩丽湘、小城姑娘李美和摇摆不定的男主丸志之间的爱情纠葛。
- **主要人物**：
  - **丽湘**：勇敢、独立、大都市女孩。
  - **李美**：温柔、内敛、小城姑娘。
  - **丸志**：在丽湘和李美之间摇摆不定。

### 案例分析
- **丸志的摇摆**：丸志对丽湘的勇敢和活力感到心动，但同时也被李美的温柔和安全感所吸引。
- **安全感和刺激感**：丽湘代表刺激感，李美代表安全感。丸志在追求刺激感的同时，也渴望安全感。
- **最终选择**：丸志最终选择了李美，因为他认为李美能给他带来安全感。

## 心理学解释
### 安全感和刺激感的需求
- **安全感的定义**：知道自己可以信任对方，对方不会伤害自己。
- **刺激感的定义**：对方身上有自己不了解的一面，对方有能力离开自己，但依然因为爱而留在他身边。
- **案例分析**：丸志在追求刺激感的同时，也渴望安全感。他最终选择了李美，因为她能给他带来安全感。

## 如何避免出轨
### 保持平衡
- **同时兼顾安全感和刺激感**：在感情关系中，既要追求安全感，也要追求刺激感。
- **主动创造**：通过自己的行动，创造安全感和刺激感。

## 总结
本节课中，我们通过分析《东京爱情故事》中的案例，探讨了爱情中的勇敢与怯懦，以及安全感和刺激感在人际关系中的作用。在爱情中，我们需要同时兼顾安全感和刺激感，通过自己的行动，创造一个平衡的关系。
