# 出轨心理学：第12讲【问答】重建的过程中，有必要见第三者吗？

## 概述
在本节课中，我们将探讨在出轨关系重建过程中，是否需要与第三者见面，以及如何处理与孩子、伴侣、新恋情的关系。

## 问题一：如何缓解与孩子的关系？
**解答：**
1. 与原伴侣重建合作关系，作为孩子的养育者。
2. 在伴侣情绪稳定后，请求其安抚孩子，改变孩子对出轨的看法。
3. 面对孩子的指责和远离，做好心理准备，可能需要多次尝试和时间。

## 问题二：如何相信伴侣不再出轨？
**解答：**
1. 询问伴侣，若未发现出轨，是否会主动切断关系。
2. 忠诚不仅是态度和语言，更是信念、能力和行动。
3. 了解伴侣出轨背后的需求和缺乏的能力，如独处能力、情绪管理等。
4. 增加出轨的代价，如签订协议，设定出轨后果。
5. 努力经营感情关系，增加独立生活能力。

## 问题三：是否需要见第三者？
**解答：**
1. 根据目标决定是否见面，如好奇、保护孩子、收集信息等。
2. 注意：见面可能增加痛苦，不如关注伴侣的需求和出轨原因。
3. 将注意力放在重建关系上。

## 问题四：如何与新男友相处？
**解答：**
1. 花时间消化过去经历，探究前任出轨原因和自身成长。
2. 与现任男友沟通信任规则和亲密关系目标。
3. 增加亲密行为，如提问、约会等。
4. 预防出轨，设定出轨后果，增加风险和成本。

## 问题五：如何调整信任感？
**解答：**
1. 重建需要时间，至少一两年甚至更久。
2. 了解妻子努力，寻找增加信任感的行动。
3. 培养信任对方的能力，关注关系中的美好一面。
4. 承担信任错负的能力，接纳不确定性和风险。

## 行动环节
1. 发现信任行动，如收入管理、朋友圈介绍、告知行踪等。
2. 分享彼此的信任行动，寻找代表信任的行动。

## 总结
本节课中，我们学习了在出轨关系重建过程中，如何处理与孩子、伴侣、新恋情的关系，以及如何调整信任感。希望这些内容能帮助你更好地应对出轨关系重建的挑战。
