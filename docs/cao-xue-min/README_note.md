# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要将数据存储在计算机的内存中，并给这些存储位置起一个名字，这就是**变量**。同时，数据有不同的种类，例如数字和文本，这些种类被称为**数据类型**。掌握如何创建变量以及了解不同的数据类型，是开始编程的基石。

## 1. 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。标签就是变量的名字，而盒子里存放的东西就是数据。在Python中，创建一个变量并给它赋值非常简单。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如，我们创建一个名为 `message` 的变量，并在其中存放一段文本“Hello World”：
```python
message = "Hello World"
```
现在，`message` 这个“盒子”里就装好了“Hello World”这段文本。之后在程序中，我们使用 `message` 这个名字，就相当于在使用“Hello World”这个数据。

上一节我们介绍了变量的比喻，本节中我们来看看Python中常见的数据类型。

## 2. 基本数据类型 🔢

Python中的数据有多种类型，最常用的有以下几种：

以下是几种核心的数据类型及其示例：

*   **整数 (int)**： 表示没有小数部分的数字。
    ```python
    age = 25
    count = -10
    ```

*   **浮点数 (float)**： 表示带有小数点的数字。
    ```python
    price = 19.99
    pi = 3.14159
    ```

*   **字符串 (str)**： 表示文本信息，需要用单引号 `'` 或双引号 `"` 括起来。
    ```python
    name = "Alice"
    greeting = '你好'
    ```

*   **布尔值 (bool)**： 只有两个值，`True`（真）或 `False`（假），常用于条件判断。
    ```python
    is_student = True
    is_raining = False
    ```

了解了数据类型后，你可能会问，我们如何知道一个变量里存的是什么类型的数据呢？

## 3. 使用 `type()` 函数查看类型 🔍

Python提供了一个内置函数 `type()`，它可以返回任何变量或值的数据类型。这对于调试和确认数据非常有用。

以下是使用 `type()` 函数的示例：
```python
height = 180
print(type(height))  # 输出：<class 'int'>

temperature = 36.5
print(type(temperature))  # 输出：<class 'float'>

city = "Beijing"
print(type(city))  # 输出：<class 'str'>
```

## 总结

本节课中我们一起学习了Python编程的起点：**变量**与**数据类型**。

我们首先将变量比作一个带标签的盒子，用于存储数据。然后，我们认识了四种基本数据类型：用于整数的**int**、用于小数的**float**、用于文本的**str**以及用于真假的**bool**。最后，我们学会了使用 `type()` 函数来查看变量的具体类型。

记住，**变量名 = 值** 是赋值的基本公式，而清楚数据的类型是进行正确运算和操作的前提。在接下来的课程中，我们将学习如何让这些不同类型的数据相互作用。
