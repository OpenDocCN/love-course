## 应对生理症状：厌食与失眠

身心是一体的。越是在心灵受伤的时候，越是要好好对待自己的身体。我们先来看厌食问题。

遭遇出轨后，很多人会因为过度的难过和哀伤吃不下东西，严重的会厌食。你的身体在经受痛苦后出现这些反应是正常的。它在告诉你，你正在经历的事情对你非常重要。

然而，长时间厌食对身体非常有害，会导致皮肤变差、免疫力下降并滋生消极情绪。当你刚出现厌食问题时，就需要及时调整。

以下是应对厌食的几个步骤：

*   **保证基本摄入**：一定要吃东西。可以从流食开始，如果汁、米糊或果泥。选择那些无需咀嚼也能咽下的食物。
*   **寻找安慰食物**：在脑海里回想过去充满温暖记忆的食物，例如外婆做的红烧肉或妈妈做的绿豆粥。寻找这类食物来吃。
*   **唤醒食欲**：吃之前闻一闻食物香味，在脑海里回忆食物的口感，主动地人为唤醒食欲。
*   **选择有益情绪的食物**：一些食物有助于情绪好转。例如：
    *   富含**维生素B族**的食物：谷物、香蕉。
    *   富含**钙**的食物：牛奶、花生、虾，可以稳定情绪。
    *   **巧克力**和**糖**：脂肪和糖的组合能让大脑获得原始的愉悦感。
*   **补充基础营养**：如果实在吃不下，可以喝葡萄糖或服用营养补充剂，以保证基本的能量和营养。公式表示为：`身体能量充足 → 心灵状态改善 → 食欲恢复`。

身体有了力气，你的精神和情绪也会好起来。

除了厌食，很多人还会经历失眠。被出轨后，整夜失眠是焦虑和抑郁情绪带来的常见反应。

首先要明白，这种应激性的短暂失眠通常不会对健康造成大的冲击，所以请坦然接受这个状态。

以下是改善失眠的方法：

*   **保证营养与运动**：白天保证营养，并适当做一些运动。运动能带来疲劳感，并促进大脑分泌**内啡肽**，这种神经递质可以缓解焦虑、改善睡眠。
*   **多晒太阳**：尽可能保证自己晒太阳。阳光可以刺激大脑分泌**血清素**以改善情绪，同时帮助合成**褪黑素**，让我们在夜里更容易入睡。此外，阳光还能帮助身体合成**维生素D**，有助于形成新的记忆。
*   **接纳噩梦**：如果做与出轨有关的噩梦，不要恐慌。噩梦是大脑释放强烈情绪、处理恐惧的正常方式。试着与这些感受平静共处，像安抚孩子一样安抚自己。睡觉时抱着玩偶也能提供一些安全感和力量。

关键是要坚持正常生活，关爱自己，并相信痛苦会过去。

## 管理心理困扰：反刍思维与抑郁倾向

除了生理上的厌食和失眠，很多人还会不断反刍出轨中经历的痛苦和自责。

经历被出轨后，你可能会不自觉地对事件进行反复思考和回忆，甚至难以控制。这在心理学上叫做**反刍思维**。

反刍思维有其积极面：善于反思的人希望收获经验教训，获得改变结果的主动权。但如果只停留在空想，就可能产生极端自责，阻碍真正的行动，甚至诱发抑郁。

当你察觉到自己陷入反刍时，需要及时停止。

以下是停止反刍思维的方法：

*   **书写宣泄**：把自己所有想的、自责的、过不去的坎、后悔和愧疚都写下来。书写是与自己对话的过程，能帮助你跳出自我，以旁观者视角冷静看待问题。
*   **设定“反刍时间”**：训练自己在一个固定时间（如睡前洗澡时）专心想这些事情。安排时间去想，可以避免24小时不间断地被侵蚀。
*   **采取行动**：在思考时问自己：**什么样的行动能带来真正的改变？** 例如：
    *   学习新东西。
    *   看一场电影。
    *   尝试一家新餐厅。
    *   做一些能获得全新体验的事情。
*   **改变环境线索**：触景生情会引发反刍。可以给家里换装饰、去新地方旅行、剪个新发型。新鲜感会迫使大脑重新认识环境，减少对过去的回忆。代码逻辑可表示为：
    ```python
    if 接触到熟悉线索（如旧物、旧地）:
        触发反刍思维
    else:
        大脑处理新信息，减少回忆过去
    ```

失眠和反刍多是应激状态，通过时间和努力可能自然好转。但如果你对所有事物都丧失兴趣，什么都不想做，这意味着你可能已处在**抑郁状态**，丧失了活力。这时必须警惕，不能任其发展成抑郁症。

如果出现抑郁状态，我们之前提到的吃东西、运动、晒太阳都是自救方式。但自救需要靠信念去坚持。你必须对自己的生活负起责任，因为身体、感受和未来生活都是你自己的。

以下是增强自救信念的方法：

*   **观看励志影片**：观看如《相助》、《永不妥协》、《当幸福来敲门》等电影。它们展现的真实苦难与拼搏精神，能让你对痛苦感同身受，并被主角在困境中怀有希望、排除万难的精神所激励。
*   **重温童年动画**：看小时候爱看的动画片，可以让你暂时逃离现状，唤起童年的单纯、梦想和憧憬。

这些方法能帮助你不断形成新的积极想法。

关于厌食、失眠、反刍和抑郁，最后需要强调的是：**如果上述状态持续一个月以上毫无缓解，或你曾有抑郁症史且症状持续一两周，请务必及时就医或寻求专业心理咨询。** 越早获得帮助，恢复的可能性和速度就越大。

## 防止痛苦演变为心理创伤

最后要讲的是，遭遇出轨后的痛苦若处理不当，可能形成**心理创伤**。创伤的主要特点是“回到过去”，即你会重复体验当初的痛苦。

并非所有痛苦都会形成创伤。只要在痛苦发生时及时处理，就能有效避免。

避免痛苦形成创伤有两个关键点：**一是增强对意外事件的掌控感，二是增加与外界的连接以获得支持感。**

**1. 增强掌控感**
出轨打破了你对关系的可控感。你不知道关系如何变化，对未来感到恐惧。重新建立掌控感至关重要。
*   **理解原因**：了解出轨的原因和关系的变化过程（如本课程前面所讲），能帮助你找到解释，明白事情是如何发生的，从而在未来更有信心避免类似情况。公式表示为：`理解（Why） → 掌控感 ↑ → 恐慌 ↓`
*   **学习处理方法**：当你知道如何处理关系中的问题（后续课程会涉及），并且不害怕问题发生时，你的掌控感会更高。

**2. 增加连接与支持感**
美国芝加哥精神分析研究所教授彼得指出，你需要找到痛苦的“见证者”，敞开心扉吐露体验。如果孤独的痛苦未被分享和见证，即使获得掌控感，仍可能出现孤独无助感，这在未来容易被轻微相似场景激发。

因此，在情绪相对平静后，请务必增加对外连接：
*   寻找可以求助和分享的对象，如专业咨询师或朋友。
*   通过写日记、在可信的社交平台小号倾诉等方式对外分享。

当**掌控感**和**支持感**重新建立后，痛苦可以只是痛苦，而不会变成影响你多年的创伤。
