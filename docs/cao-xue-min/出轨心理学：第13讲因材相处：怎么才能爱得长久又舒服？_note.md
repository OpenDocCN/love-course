# 出轨心理学：第13讲因材相处：怎么才能爱得长久又舒服？_note

在本节课中，我们将探讨如何通过满足核心需求与适配相处模式，来构建高质量、持久且舒适的亲密关系。这不仅是预防出轨的有效方式，更是让双方都愿意主动维系关系的核心。

上一模块我们系统分析了出轨的原因、应对、选择与重建。本模块我们将聚焦于“创伤后成长”，探讨如何从这些经历中获得积极改变。本节课，我们将首先从**关系**的角度入手。

许多听众，无论是否经历过出轨，都关心如何预防出轨。此前课程已提供了一些方法，例如了解对方的忠诚信念、规则，评估出轨概率，明确出轨代价等。然而，对于一段关系而言，最佳的预防策略并非时刻“防备”，而是**建立一段高质量的关系**，让双方都发自内心地渴望停留其中、彼此拥有。

一段高质量的关系，通常让双方都感到“值得”且“舒服”。“值得”关乎核心需求是否被满足；“舒服”则关乎双方的相处方式。因此，本节课将分为两部分：
1.  如何满足彼此的核心需求。
2.  如何找到最适合彼此的相处方式。

## 一、 满足核心需求：让关系“值得”

许多作品强调“相爱最重要”。爱情固然关键，但其维系并非凭空而来，背后深受**需求**的影响。需求有大小之分：有些需求未被满足，尚可忍受；但另一些需求若被满足，你会深感这段关系“太值得了”，甚至能包容其他未被满足的需求。这些便是**核心需求**。

一段持久亲密的关系，往往并非所有需求都被满足，而是**核心需求得到了强烈满足**。反之，若核心需求缺失，即使其他需求被满足，个人仍会感到空虚。

**案例一**：一位来访者的妻子出轨了。来访者工作家庭料理得当，对妻子温柔体贴。但其妻子的核心需求是**事业发展**，出轨对象均为业内资深前辈。此前，妻子曾劝说丈夫一同创业或更求上进，但丈夫未重视，最终出轨发生。

**案例二**：一位来访者的丈夫与第三者有了私生子，但声明不爱对方，只是年过四十强烈**渴望拥有孩子**。丈夫曾劝说妻子生育，但被拒绝。对孩子（或事业发展）的追求，是许多人关系中的核心需求。

那么，人的核心需求有哪些？我们又该如何发现呢？

加拿大社会学家约翰·李的爱情风格理论，总结了六种类型，对应着六类典型的**核心需求**：

以下是六种爱情风格及其核心需求：

1.  **激情型**
    *   **核心需求**：激情、性、浪漫。
    *   **特点**：喜欢制造浪漫与性愉悦。若伴侣魅力下降或态度冷淡，使其情感与身体需求得不到满足，便会觉得关系失去价值。

2.  **游戏型**
    *   **核心需求**：轻松感、自由感、有趣、好玩。
    *   **特点**：难以忍受关系中的责任与约束，视自由与快乐为最重要，希望与伴侣游戏人生、无拘无束。

3.  **友爱型**
    *   **核心需求**：陪伴、稳定的心情、适度的空间。
    *   **特点**：偏爱平静、距离合适、长久的感情状态，非常注重稳定感与踏实的陪伴。

4.  **实用型**
    *   **核心需求**：事业发展、生儿育女、相互照顾（自己与老人）。
    *   **特点**：非常实际，可能会对关系进行成本收益分析，看重关系的实用性。这未必是自私，因为他们也可能为对方提供这些实用价值。

5.  **占有型**
    *   **核心需求**：占有、支配。
    *   **特点**：希望完全拥有甚至支配对方，需要对方极度亲近、持续给予爱的保证并无条件服从。否则会觉得关系无价值。**注意**：占有型常伴有控制欲，并非健康的关系模式，许多PUA（搭讪艺术家）行为便源于此。

6.  **利他型**
    *   **核心需求**：对方对爱的接收，以及感动、感激等回应。
    *   **特点**：享受为对方奉献，甚至将爱人利益置于自身之上。但若付出得不到回应或甚至被拒绝，便会觉得关系无价值。

现实中，一个人可能兼具多种核心需求。例如，日剧《咒怨》中的角色，其需求可能混合了**实用型**与**友爱型**。另一个案例中，来访者与出轨前女友复合，因双方都追求无拘无束与新鲜刺激，满足了彼此**游戏型**与**激情型**的需求。

**如何发现核心需求？**
1.  **对话法**：与伴侣坐下来聊聊，做什么事会让彼此觉得关系“特别有价值”，并将其归类，找出背后的核心需求。共同商讨满足彼此需求的方法。
2.  **观察法**：
    *   **看过去经历**：观察对方如何选择前任（因激情、实用还是游戏？）。注意，需求会变，重点看当前互动。
    *   **看资源投入**：观察对方大部分时间、精力、金钱投向何处（关系内：生活实用还是浪漫？关系外：如事业，则关系至少不应妨碍其事业）。
    *   **看矛盾行为**：例如，某人声称爱赚钱，却突然热衷一件不赚钱的事，这事可能隐藏着其核心需求。思考关系能否满足此需求。

**核心需求会变化**：人是变化的，核心需求亦然。例如《我的前半生》中，罗子君丈夫的需求从**激情**（貌美如花）转变为**友爱与实用**（分担压力、理解关心）。需特别关注：逢十的年龄阶段、重大丧失（如失业、亲人去世）、自觉魅力丧失等**脆弱期与变动期**。建议在这些节点，更好地了解彼此，通过行动满足自己与对方变化后的核心需求。

## 二、 适配相处模式：让关系“舒服”

然而，有时即使核心需求互相满足，相处仍可能“不舒服”。

**案例**：一对情侣均认为关系“值得”，但都感到“太累”。女生受不了男生在感情热烈时突然冷淡、失联；男生则受不了吵架时女生不停纠缠、不让他躲开。双方都爱对方，却不知问题出在哪，怀疑爱已被消耗殆尽。

其实，这些行为与“爱”关系不大，更多与双方的**依恋类型**有关。依恋类型由心理学家玛丽·安斯沃斯提出，指婴幼儿时期与养育者互动形成的反应模式，主要分为三种：**安全型**、**焦虑型**、**回避型**。这种模式会内化为成年后的人际互动方式，直接影响亲密关系的相处。

若想相处舒服，了解彼此的依恋类型是关键窗口。以下是三种类型的特征与相处建议：

**1. 安全型依恋**
*   **特征**：易于亲近，自在，乐于求助，不惧拒绝，不担心被抛弃或暂时疏远。
*   **相处建议**：与安全型相处较轻松，包容性强，做自己即可。

**2. 焦虑型依恋**
*   **特征**：常担忧对方是否真心爱自己、愿在一起，渴望极度亲近，总觉不够，易给对方压力。
*   **相处建议**：
    *   明确、直接地表达爱意。
    *   如需分开，明确告知忙碌时长与下次联系时间，勿突然消失。
    *   当对方因小事指责时，理解这是其表达爱渴求的本能反应，非针对你。避免对抗，勿让其独处“冷静”，应陪伴安抚，给予安全感后再沟通。

**3. 回避型依恋**
*   **特征**：既怕对方消失，又会在对方靠近时感到羞耻，故常强装不在乎。难以完全信任或依赖他人，不喜直接表达爱，心烦时偏好独处，冲突时本能想逃离。
*   **相处建议**：
    *   充分了解并尊重其所需的自由空间。
    *   当对方远离或不领情时，可能仅是因关系过近而紧张，并非不爱你。
    *   此时应坚定表达爱意，告知你会在某处等待，然后给予其所需的时间与空间。

**回顾案例**：男生属**回避型**，故在感情热烈时本能逃避、不表达爱；女生属**焦虑型**，需要密切互动与情感确认。若不调整，易陷入“男逃女追”的疲惫循环。

依恋类型并非非黑即白，一个人可能同时具备两种类型的特质。例如，假设依恋类型总分为10分，一个人可能是**焦虑型7分 + 回避型3分**。此时需观察：何种场景激发焦虑反应？何种场景激发回避反应？进而分场景制定新的互动方式。

## 总结与行动

本节课我们探讨了构建高质量关系的两大支柱：
1.  **满足核心需求**：通过爱情风格理论识别并满足彼此的核心需求，让关系感到“值得”。
2.  **适配相处模式**：通过了解彼此的依恋类型（安全型、焦虑型、回避型），调整互动方式，让关系感到“舒服”。

这正如教育中的“因材施教”，经营关系也需遵循“因材相处”的原则。在“我爱你”三个字中，“我”和“你”才是关键。让爱的表达与方式适合独特的彼此，让双方在爱中获得价值感，方能建立互相适合、互相成就的关系。
