# 出轨心理学：第13讲【问答】什么样的伴侣更好相处？ 👫

在本节课中，我们将通过回答六个典型问题，深入探讨亲密关系中的核心议题，包括需求满足、依恋类型匹配、关系改变以及冲突处理等。我们将学习如何识别并应对关系中的挑战，旨在帮助你构建更健康、更稳固的伴侣关系。

## 问题一：重建关系会助长出軌者的自私吗？

上一节我们介绍了出轨是满足需求的错误方式。本节中我们来看看关系重建过程中的需求满足问题。

选择重建关系后，出轨者同样需要非常努力地去满足被出轨者的需求。通常，出轨者需要付出更多努力。重建的第一步是出轨者反思错误，并通过行动道歉和弥补。完成道歉与宽恕后，双方身份重新转变为伴侣，这意味着需要互相满足对方的需求。双方都需要努力。

你在最后提到了自私。自私的定义是：**用伤害他人的行为来满足自己的需求**。但本课程始终强调，应**用不伤害、且最好是共赢的方式来满足双方的需求**。重建过程也需遵循此原则。

尤其需要告知被出轨者：在互相满足的过程中，切勿以伤害自己为代价去满足对方。**先爱自己，再爱他人**。

## 问题二：相同依恋类型的人在一起更合适吗？

并非如此，相同依恋类型组合有时可能更糟糕。依恋类型决定了个体在亲密关系中的本能感受与行为。当这种本能行为可能伤害关系时，两个人都采用相同行为，只会让关系恶化。

以下是三种依恋类型两两相同匹配时的相处情况分析：

*   **安全型 + 安全型**：这是最稳定的组合，双方都会感到舒适。因为彼此都能坦诚表达爱与想法，保持既独立又亲密的状态。面对冲突时，也更能互相包容。
*   **焦虑型 + 焦虑型**：相处时可能非常热烈，但发生矛盾时易产生激烈冲突和大量“抗议行为”（如拼命发短信、打电话、指责等）。真正的需求（如渴望安慰）常被隐藏在这些抗议之下，最终演变为互相指责和要求。因此，这种组合冲突多，且每次冲突都会积累伤害。
*   **回避型 + 回避型**：短期相处可能无问题，但很少能长久。即使结婚，也可能仅是形式在一起，难以建立真正亲密。更糟的是，他们可能向外（如通过出轨）寻求情感满足。因为回避型虽不擅长紧密联结，仍有情感需求，而出轨关系提供的距离感反而让他们舒适。对回避型而言，焦虑型或安全型伴侣反而更可能满足其情感需求。

需要注意的是，上述匹配结果是学术统计结论，并非每个个体都如此。我们可以通过了解彼此依恋类型来提升相处质量，运用课程中的方法减少消极影响，避免误会积累，以更好的行动增进亲密。

## 问题三：非安全型依恋可以改变吗？

可以改变。童年时期在与父母等养育者的关系中形成的依恋类型，其改变同样需要建立一段高质量、深刻的关系。

为什么必须在关系中改变？因为只有在关系中，你依恋类型所引发的本能反应才会被激发。被激发后，关系中全新的、更能给你安全感的互动方式，才有可能改变那些被本能唤起的情绪与感受。**依恋类型的改变需要在关系互动的实践中完成**。

因此，若想改变依恋类型，可以尝试建立一段深刻的咨询关系，或开始一段高质量的恋爱。你需要在关系中展现真实完整的自我，并从互动中获得爱与安全的体验，从而成长。

一个更实用的建议是：恋爱时尽可能选择**安全型依恋**的伴侣，这样你更有可能调整自己的非安全型依恋。

## 问题四：核心需求矛盾（如生育观不同）如何化解？

如果你指的“继续下去”是继续恋爱，那么只要有感情，短期内问题不大。但长期相处难免产生问题。

从现实层面看，生育和家庭对女性的影响通常更大。虽然有些男性出于爱会让步，但风险在于，随着年龄增长，男性很可能强烈想要孩子（女性也可能，但届时女性的生育机会更少）。冷冻卵子或许是一个选择，但技术层面仍存在许多不确定性。

不要幻想对方会改变。应该思考的是：在彼此想法不变的前提下，你们计划怎么做？

首先，你需要自己想清楚以下问题：
*   你准备永远不生育吗？
*   甚至永远不结婚吗？
*   还是说，在满足某些条件后，你也会考虑结婚生育？

想明白后，再与伴侣沟通：
*   如果确定不婚不育，对方有何打算？
*   如果是满足条件才婚育，双方是否可能共同努力去创造这些条件？

总之，你需要先深入思考自己的决定，多了解没有孩子和家庭的人生是怎样的。如果了解后仍坚持己见，那就坚定地按照自己的想法，主动、积极地规划人生。

恋爱与婚姻虽有白头偕老的愿望，但更主要的是在这个过程中不断了解自己和对方，成就彼此的人生。因此，当你确实发现无法或不愿妥协时，坦诚面对现实并告知对方，让对方也有机会面对真实并做出选择。此时，选择放手也是一种负责任、有担当的表现。

## 问题五：非安全型必须找安全型伴侣才能拥有好爱情吗？

谈不上“必须”。这首先取决于你对“好爱情”的定义。

*   如果你渴望**激情**，那么**焦虑型依恋**的伴侣可能更能带来激情，因为他们表现得更热烈。
*   如果你渴望**亲密与安稳**，那么**安全型依恋**的伴侣确实更合适。因为在亲密关系中，安全型伴侣更愿意做出包容和让步，也更善于自然地表达爱与需求。

## 问题六：焦虑型与回避型组合，冲突时如何改变？

坦白说，这种组合有些棘手。焦虑型与回避型是依恋类型组合中最折磨人的一种，但在生活中（尤其受社会文化影响的异性恋中，女性焦虑型+男性回避型）非常常见。

焦虑型常伴有怀疑和自卑，通常处于弱势；回避型则显得独立理性，通常处于强势（但这并非真正的独立理性，因为真正的独立理性会直面伴侣感受和关系问题）。在这种组合下，双方会**互相强化对方的自我感觉和形象**：焦虑型在对方回避时会更加焦虑，回避型在对方激烈反应时会更加回避，从而陷入恶性循环。

尤其需要注意的是，这种关系对焦虑型一方的伤害更大，可能逐渐损害其自尊，因为对方的回避是一种忽视，会加剧其自我怀疑。

我的首要建议仍然是：**考虑寻找一位安全型、愿意认真直面你需求的伴侣**。

若想改变你们现有的关系，或许可以尝试**伴侣咨询**，借助专业力量。仅靠自身努力会比较困难。但你们可以学习依恋类型知识，并约定：在本能反应过后，能否坐下来回顾当时发生了什么？根据课程所学，识别哪些反应是依恋类型导致的，然后重新表达当时的感受、想法、爱与需求，共同商量出一个**共赢的满足方式**。

### 行动环节：发现你的爱情风格

通过上节课和本节课的问答，你已经了解到，每个人的爱情风格中都隐藏着其核心需求。满足这些核心需求，会让人感到关系有价值；反之，关系则可能陷入“食之无味，弃之可惜”的境地。

发现一段感情中的爱情风格，能更好地识别彼此的核心需求，让你更理解这段关系，也更有方向地去稳固它。

例如，对我而言，我的爱情风格属于**激情、有爱和实用**的综合体。这意味着我的核心需求包括：感情与身体的满足、稳定的安全感、关系有助于我个人目标达成等。因此，我的伴侣需要保持个人魅力、维持感情浓度、保持忠诚、在生活与事业上能协助我等。这样，我会觉得这段关系很有价值。

对我的伴侣而言，他的爱情风格属于**激情、有爱和利他**的综合体。不同之处在于“利他”，他的核心需求是**接受并回应他的爱**。因此，我需要关注并回应他对我的各种付出。

那么，你的爱情风格是什么？背后有哪些核心需求？
进一步思考，你的伴侣（或过去的伴侣），他的爱情风格和核心需求又是什么？
你们是如何彼此满足这些核心需求的？

期待你的发现与分享。让我们每天都更懂自己一些，也更懂对方一些。
