# 出轨心理学：第03讲【问答】不婚主义者是高概率出轨者吗？

## 概述
在本节课中，我们将探讨关于不婚主义者、挑剔的女朋友、前女友的影响、激情之爱以及忠诚信念确认等问题。

## 不婚主义者是高概率出轨者吗？
**问题**：男朋友说他是不婚主义者。不婚主义者算是规则否认者吗？他出轨的概率会不会很大？

**解答**：
- 不一定。如果他认可一对一的亲密关系，就不属于规则否认者。
- 确认他是不想和你结婚，还是不想和任何人结婚。
- 如果他是主张一对一的不婚主义者，关键在于你想要什么样的关系。
- 如果你希望结婚，考虑是否接受没有婚姻的关系。

## 女朋友挑剔，是否不喜欢我？
**问题**：我女朋友很挑剔，我做什么她都不满意。她是不是不喜欢我，或者她是不是另外有喜欢的人？

**解答**：
- 直接询问她的不满之处。
- 了解她挑剔的原因，是否因为其他原因未表达出不满。
- 如果她可能是最大化者，她可能不是针对你，而是面对焦虑。

## 上一段关系出轨，下一段关系会出轨吗？
**问题**：如果对方在上一段关系中间出过轨，我要怎么办？她会不会在和我的这段关系里也出轨呢？

**解答**：
- 分析出轨背后的原因，是否是人的问题或关系问题。
- 聊聊她当时为什么出轨，对忠诚规则的理解。
- 判断她是否是高概率出轨的人，是否有意愿改变。
- 如果不是高概率出轨的人，找到感情关系处理的问题。

## 男朋友前女友求助，是愚亲未了还是怎么回事？
**问题**：我发现男朋友的前女友每次一示弱求助，男朋友就会去帮他。这是愚亲未了还是怎么回事？我应该怎么办？

**解答**：
- 判断男友是否只在示弱时互动，还是日常有密切联系。
- 如果男友有英雄主义倾向，他可能喜欢做别人的英雄。
- 询问他更想做谁的英雄，并关注彼此的需求。
- 如果他忽视了你的感受，考虑是否继续在一起。

## 男朋友坦白爱上别人，如何说服他回心转意？
**问题**：男朋友坦白说他爱上了别人，说那个人能给他不一样的感觉，但我一看就知道那不是真爱。我要怎么做才能说服前男友呢？

**解答**：
- 激情之爱可能随着阻碍消失而递减。
- 直接离开对方，让他发现激情的短暂性。
- 确认他是否有明确的分手想法。
- 如果他愿意努力，可以通过咨询和互动增进关系。

## 忠诚信念确认
**问题**：在一段关系开始之前，两个人做忠诚信念的确认很重要。

**解答**：
- 制定自己的忠诚规则，包括聊天、肢体接触和社交场合。
- 确认自己对忠诚信念的理解，并记录在评论区。
- 了解每个人对忠诚信念的理解都不一样，关键在于了解自己。
