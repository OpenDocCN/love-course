# 出轨心理学：00关于背叛，我们一无所知

## 概述
在本节课中，我们将探讨出轨的心理学，了解其本质和影响，并学习如何应对和修复由此带来的伤害。

## 讲师介绍
我是曹学敏，国家二级心理咨询师，复旦大学社会心理学博士生。我拥有十多年的专业心理咨询经验，擅长家庭组织、亲密关系和个人成长。

## 出轨的普遍性
* 出轨比我们想象的要普遍得多。
* 中国已婚和同居人群中，男性出轨率从2000年的11.8%上升到2015年的34.8%。
* 每三个丈夫和每7.5个妻子中，就有一个曾经出轨。

## 出轨的影响
* 出轨会破坏一个人的安全感、自我价值感和对爱情、婚姻的希望。
* 会影响到孩子对亲密关系的看法。
* 会让人对爱情产生怀疑。

## 应对出轨
* 这门课程将帮助你从社会心理学、认知行为疗法和ACT疗法等角度理解出轨。
* 帮助你从怀疑中得到洞察和勇气。
* 提供应对出轨发生后的情绪危机的方法。
* 教你如何重建亲密关系。
