# 出轨心理学：第08讲 情绪急救：如何处理情绪危机？

## 概述
在本节课中，我们将学习如何处理出轨带来的情绪危机，特别是愤怒和羞耻感。

## 情绪急救
### 处理愤怒
- **愤怒是正常的反应**：被出轨后，愤怒是一种正常的情绪反应。
- **表达愤怒**：可以通过摔东西或想象报复来宣泄愤怒。
- **避免报复**：报复可能会加剧痛苦，并阻碍个人成长。

### 处理羞耻感
- **羞耻感的影响**：羞耻感可能会让人感到孤立和无助。
- **隔离负面评价**：避免与嘲笑和落井下石的人交往。
- **寻找支持**：与支持你的人相处，找回自我价值感。

## 总结
本节课中我们一起学习了如何处理出轨后的情绪危机，特别是愤怒和羞耻感。通过表达情绪、隔离负面评价和寻找支持，我们可以更好地应对这些情绪，并促进个人成长。
