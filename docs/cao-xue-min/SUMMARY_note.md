# 出轨心理学：第01讲 出轨：在错的地方找想要的东西

## 概述

在本节课中，我们将探讨出轨的心理原因，以及为什么人们在错误的地方寻找满足感。

## 核心概念

**公式**：出轨 = 心理需求未满足 + 错误的满足途径

## 出轨的原因

以下是导致出轨的一些常见原因：

### 1. 心理需求未满足

*   **情感需求**：缺乏关注、理解和支持。
*   **性需求**：性需求未得到满足或寻求新鲜感。
*   **自我价值感**：缺乏自信和自尊。

### 2. 错误的满足途径

*   **逃避现实**：通过出轨来逃避现实生活中的问题。
*   **寻求刺激**：追求新鲜感和刺激感。
*   **满足虚荣心**：通过出轨来满足虚荣心。

## 出轨的影响

出轨会对个人、伴侣和家庭产生严重影响：

### 1. 个人

*   **心理创伤**：自尊心受损、信任感丧失。
*   **情感困扰**：焦虑、抑郁、愤怒。

### 2. 伴侣

*   **信任破裂**：信任感丧失、情感疏远。
*   **心理创伤**：自尊心受损、愤怒。

### 3. 家庭

*   **关系破裂**：婚姻破裂、家庭破碎。
*   **子女受到影响**：子女心理创伤、行为问题。

## 总结

本节课中，我们学习了出轨的心理原因和影响。了解这些原因和影响有助于我们更好地预防和应对出轨问题。
