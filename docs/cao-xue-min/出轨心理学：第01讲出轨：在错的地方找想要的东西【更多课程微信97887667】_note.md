# 出轨心理学：第01讲 出轨：在错的地方找想要的东西

## 概述
在本节课中，我们将探讨出轨的定义、影响以及背后的心理原因。

## 出轨的定义
**以下是关于出轨的定义：**
- 伴侣是否知道并接受多重关系。
- 是否存在情感依恋。
- 身体接触的程度。

**公式：**
- 出轨 = 知情且接受多重关系 + 情感依恋 + 身体接触

## 出轨的影响
**以下是出轨可能带来的影响：**
- 对被出轨者的伤害：信任感、安全感、自我价值感、对爱情的希望。
- 对孩子的看法：对亲密关系的看法、对信任的怀疑、对爱和婚姻的怀疑。
- 对出轨者的影响：心理折磨和痛苦。

## 出轨的原因
**以下是出轨可能的原因：**
- 关系裂痕未修复。
- 工作压力增大。
- 内心不接受忠诚原则。
- 环境影响。

## 30天觉察日记
**以下是30天觉察日记的第一天主题：**
- 看见自己的需求。

**行动：**
- 想象你和伴侣在一起最幸福的三个时刻。
- 分析这些时刻满足了你的哪些需求。

## 总结
本节课中，我们学习了出轨的定义、影响和原因。在下一讲中，我们将继续探讨出轨背后的心理原因以及如何应对出轨带来的伤害。
