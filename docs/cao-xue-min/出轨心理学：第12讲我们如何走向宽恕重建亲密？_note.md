# 出轨心理学：第12讲 我们如何走向宽恕与重建亲密？💔➡️❤️

在本节课中，我们将要学习在关系经历出轨创伤后，如何超越“翻篇”，真正地走向宽恕、重建信任并重获亲密。这是一个面向未来的行动指南。

在上一讲中，我们探讨了如何面对出轨的真相并进行建设性争吵，这帮助我们学会了如何直面与放下过去。然而，如果重建关系仅仅停留在“翻篇”这一步，是远远不够的。

我曾接待过一对伴侣，经过近一年的咨询，他们直面了出轨，看清了背后的真相与问题，不再为此争吵。但他们紧接着遇到了新问题：两人不知道该如何相处了。因为过去的那份信任感和亲密感已经发生了变化，而这两者对任何希望长期维持的关系而言，都是不可或缺的。总之，对于选择重建的你们来说，翻篇只是重建的过程，而非最终目的。

重建关系的最终目标，是指向一个更美好的当下与未来。今天这一讲，我将从三个部分来探讨面向未来的重建该如何进行：**第一，做到真正的宽恕；第二，重建信任；第三，重获亲密**。

## 一、如何做到真正的宽恕？🤝

出轨是一种非常严重的伤害。对于这份过去，双方都需要学会真正的道歉与宽恕。这既是对过去的放下，也是对未来的承诺与展望。

对出轨者而言，要做的承诺是：**“我不会再做出同样的伤害，我要和你重新开始，一起共建未来。”**
对被出轨者而言，要做的承诺是：**“我不再把你当做一个‘出轨的人’，不会再以‘被出轨者’的身份审视你、要求你。我会把你当做爱人，去重新和你相处。”**

听到这里，很多人会问：道歉难道不是一开始就要做的吗？为什么是这个时候？的确，出轨者只有道歉并取得原谅后，重建才会开始。但在出轨这件事上，**什么时候的道歉更有价值？** 是在他完全明白自己错在哪、究竟给对方造成了何种伤害之后。在此之前，压力之下的道歉可能只是一种应激反应，他未必真正意识到自己的过错与造成的伤害。那时的道歉，哪怕真诚，也很难让被出轨者发自内心地接受与宽恕，因为他会觉得出轨者根本不懂自己犯了什么错，也无法确信未来出轨是否会卷土重来。这些想法对重建非常不利。

因此，在双方都平静下来、认真探索了出轨事件之后，出轨者需要**再一次**向被出轨者做一次真正的、深刻的道歉。

### 具体如何操作？

以下是进行有效道歉与宽恕的步骤：

1.  **出轨者的深度反思**：在正式道歉前，出轨者需要花时间认真思考有关这次出轨的各种问题，最好将思考写下来。例如：
    *   你具体做了什么？
    *   这么做是想满足你的哪些需求？
    *   这件事对伴侣和关系造成了哪些伤害？
    *   回过头看，原本可以怎么做来避免这件事？
    *   未来你会怎么做来避免再次发生？
    *   未来你希望两个人的关系能达到哪种程度？

    这个过程可能艰难，需要数天甚至数周，可能需要多次修改。但深入的反思有利于认识自己，你的诚意也更能打动伴侣。

2.  **被出轨者的平静回顾**：在出轨者反思的同时，被出轨者也需让自己平静下来，重新回顾伤痛，思考以下问题：
    *   你现在对出轨这件事的理解是什么样的？
    *   这件事给你和你们的关系带来了哪些伤害？
    *   你希望这段关系接下来会怎么样？

3.  **举行道歉仪式**：当双方都经过认真思考后，接下来要举行一个具有象征意义的道歉仪式。仪式意味着状态的转换，从彼此伤害转向彼此宽恕。
    *   **具体做法**：请你们将上述问题的思考写成一封信，然后**面对面地念给对方听**。人对声音和场景的记忆更深刻，面对面的诉说能生成新的、有温度的记忆，增强信念感，帮助对抗未来的不确定性。其作用与婚礼誓言类似。

4.  **被出轨者的回应**：当出轨者发起原谅的请求后，被出轨者可以这样回应：
    > “我会试着原谅你。但我要告诉你，我原谅你，不代表我接受了这些事情的发生，也不代表我的痛苦已经消失了。我原谅你，只是因为我希望我和你之间，还有机会去拥有更好的生活。”

    这几句话很重要。真正的宽恕后，痛苦可能依然存在，但宽恕能让痛苦消失得更快。研究显示，能宽恕伴侣的人能享受到更多的幸福，包括更高的自尊、更少的烦恼与紧张、更满意的生活。

5.  **出轨者的理解与后续行动**：出轨者要明白，伴侣选择宽恕，代表他愿意放下仇恨与指责，这背后是对你的希望与爱。在之后的日子里，你依然要始终关心对方的痛苦，并试着安慰对方。

6.  **仪式的结束**：仪式的最后，给彼此一个温暖的拥抱。如果目前做不到，可以用握手等让自己舒服的方式结束。

希望这个道歉仪式能帮助你们达成和解，迈向信任与亲密的下一阶段。

## 二、如何重建信任？🔐

经历了道歉与宽恕，接下来要重建信任感。一段好的关系离不开合作与付出，但只有彼此建立起信任感，合作与付出才能良好运转。因此，信任感的建立对重建关系至关重要。

如果说出轨让信任变成了“负数”，那重建信任首先要做的是让“负数归零”，然后再一点一点积累成“正数”。

在信任感建立的初期，最重要的是**减少被出轨者对当下的不确定性**。因此，被出轨者需要知道更多信息，拥有更多知情权。在这个过程中，出轨者需要更加主动。

### 重建信任的行动指南

以下是重建信任的具体方法：

1.  **共同制定信任行动计划**：
    *   被出轨者问自己：**“这段时间对方怎么做，能够让你更加安心？”** 最好能列一个行动清单。
    *   列完后与对方商量，询问对方是否愿意做这些事。如果不愿意，给予调整空间，共同商定出一个明确的行动方案。

2.  **增加透明度与报备**：最常见的方式包括：
    *   出轨者每天报备行踪。
    *   手机响时，主动告知是谁来电。
    *   晚回家时，主动说明原因。
    *   总之，出轨者应努力让自己的24小时变得“透明”，以安抚被出轨者本能的不安情绪。

3.  **彻底处理第三者事宜**：关于第三者的任何情况，出轨者都需要向被出轨者报备，例如是否已彻底切断联系、对方是否尝试再联系、如何回应等。这是在之后很长一段时间里都需要坦白的事情，旨在消除被出轨者对当下和未来的恐慌。

4.  **应对第三者再次联系的情况**：即便出轨者已表明立场，第三者仍可能联系。此时，**最好的做法是在收到消息的那一刻，就公开告知伴侣**。隐瞒的风险极大，一旦被发现，信任可能再次崩塌。

5.  **避免信任重建的误区**：重建一段时间后，双方容易陷入一个误区：把“信任行动有没有结束”当成了“信任感有没有建立”的标准。
    *   出轨者可能觉得：“你怎么还看手机？不信任我！”
    *   被出轨者可能强忍不安：“我不能问，问了就代表我不信任他。”
    *   **正确做法**：将信任行动视为双方约定的日常规则。先确定行动计划的时间（例如半年），在此期间坚持履行。到期后再商量是否延期或结束。

6.  **关注积极互动与表达感谢**：信任行动很重要，但更重要的是关注两人之间的积极互动。可以每天睡前回顾：
    *   今天对方为你做了什么？满足了哪些需求？维护了哪些利益？
    *   今天对方为这段关系、这个家做了什么？
    *   完成回顾后，及时表达感谢。如果说不出口，至少告诉对方：“我看到了你的努力。”当心中积累越来越多积极回忆时，信任感也会逐渐稳固。

7.  **升华对信任的理解**：经历过破碎后，我们需要升华对信任的理解。要认识到未来充满不确定性，每个人身上都有未知之处。
    *   **真正的信任感**不是：“因为过去的你一直做到，所以我相信你。”
    *   而是：**“我知道人性是复杂的，但我愿意和你一起去承担这份风险。”**

当两个人重新建立起信任感后，你们就有机会去重新获得亲密。

## 三、如何重获亲密？💑

在亲密这件事上，著名心理治疗师哈里斯认为，两性亲密体现在三个方面：**情感亲密、心理亲密和身体亲密**。

*   **情感亲密**：你们互相了解对方的感受。即“我关心你”。
*   **心理亲密**：你们互相了解对方的想法。即“我懂你”。
*   **身体亲密**：你们能够互相了解对方的身体感受，互相触碰对方的身体。

你们可以按这三种类型来回顾：过去你们在哪方面做得特别好？把它们写下来，每种亲密写三件事。接着，写下未来你们在这三种亲密上的目标是什么，相应的行动计划又是什么。这一步记录是在传递一种信念：过去我们曾拥有幸福的亲密时刻，相信未来我们依然可以。

### 重建亲密的具体方法

1.  **建立情感与心理亲密的共通方法：共情对话**
    *   在对话中，学会在听完对方的话后，**重复对方的感受和想法**。
    *   用好奇心追问更多细节。
    *   对话中有沉默时，不要急着救场，给对方思考时间，耐心听他把话说完。
    *   如果缺乏表达感受的能力，可以借助网上的“情绪词汇表”，在有情绪时找到对应词汇，并主动描述自己及对方的情绪。

2.  **重建身体亲密**
    *   身体亲密的重建通常更难，尤其是出轨涉及性行为和情感依恋的情况。
    *   **理解与尊重**：被出轨方可能本能抗拒，身心遗留恶心感和排斥感。出轨者要理解，任何接触都要**先询问对方意愿**，尊重“不”的回答。
    *   **从按摩开始**：按摩既不是特别亲密的接触，但确有接触，能帮助建立放松和舒服的感受，体现互相的关心。因此，按摩甚至可以比牵手和拥抱更早进行。
    *   **恢复性行为**：一个推荐做法是离开原有环境，在旅行中发生（如温泉旅行，带有“洗涤”的洁净感）。如果在家进行，试着增加一些仪式感，象征这段新关系取得了重大进展。

## 总结与行动环节 📝

本节课中，我们一起学习了在放下过去后，如何通过真正的宽恕、系统的信任重建以及多维度的亲密重获，来面向未来，重新建立和拥有真正亲密的关系。

我想对所有选择重建的伴侣说：重建并不仅仅是修补损坏的关系，它更是**你们互相重新选择对方的一个过程，是关系全新的开始**。而且，重建不只是态度和感受，在关系中，它更多是一种**行动**。你们始终需要通过行动来表达歉意、表达信任和表达爱。只有态度而没有行动，重建是做不起来的。这个过程非常艰难，需要很久，需要互相配合才能坚持下去。

当然，在重建之后，如果你们平静下来，发现还是想要分手或离婚，这也是一种选择。相信这是深思熟虑后的决定，也能走向更好的未来。而且，即使是告别，相信这也是不再怨恨的告别。
