# 出轨心理学：第12讲 我们如何走向宽恕重建亲密？

## 概述
在本节课中，我们将探讨如何通过宽恕、重建信任和重获亲密来走向出轨后的关系重建。

## 真正的宽恕
**核心概念**： 宽恕是对过去的放下，也是对未来的承诺和展望。

1. **道歉的重要性**： 出轨者应在完全理解错误和伤害后，向被出轨者进行真正的道歉。
2. **道歉前的准备**： 出轨者需认真思考出轨的原因、对关系造成的伤害以及如何避免未来再犯。
3. **道歉仪式**： 通过写信和面对面朗读，加深记忆，增强信念感。
4. **被出轨者的回应**： 表达原谅的意愿，但强调原谅不代表接受伤害，而是希望拥有更好的未来。

## 重建信任
**核心概念**： 信任感的建立是重建关系的关键。

1. **行动计划**： 制定增加信任感的行动计划，如报备行踪、告知来电等。
2. **知情权**： 被出轨者有权了解更多信息，以减少不确定性。
3. **信任行动的误区**： 不要将信任行动与信任感的建立等同起来。
4. **积极互动**： 通过积极的互动和表达感谢，增强信任感。

## 重获亲密
**核心概念**： 亲密关系包括情感亲密、心理亲密和身体亲密。

1. **回顾和目标**： 回顾过去在亲密关系中的优点，设定未来目标。
2. **情感亲密**： 通过共情和对话，增进了解和关心。
3. **心理亲密**： 通过倾听和追问，了解对方的想法。
4. **身体亲密**： 从按摩开始，逐渐增加身体接触，如拥抱和性行为。

## 总结
本节课中，我们学习了如何通过宽恕、重建信任和重获亲密来走向出轨后的关系重建。重建关系是一个艰难的过程，需要双方共同努力。希望这节课能帮助你们在关系中找到新的方向。
