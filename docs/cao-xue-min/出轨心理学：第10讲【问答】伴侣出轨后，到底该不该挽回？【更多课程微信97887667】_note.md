# 出轨心理学：第10讲【问答】伴侣出轨后，到底该不该挽回？

## 概述
在本节课中，我们将探讨伴侣出轨后是否应该挽回的问题，并分析不同情况下的应对策略。

## 问答环节

### 问题一：原谅出轨后的心态调整
**问题**：男朋友出轨后，虽然我选择了原谅，但还是常常会觉得心里不平衡，总觉得便宜了他。请问老师，我这样的心态正常吗？我该怎么做才能调整这样的心态呢？

**解答**：
- 刚开始选择原谅出轨时，有这种心态是非常正常的。
- 这种不平衡的感觉是因为关系平衡被打破，对方违背了契约，而伤害结果由你承担。
- 长期不平衡会导致关系问题，双方都会感到不舒服。
- 调整心态的方法：
  - 让对方用行动证明歉意和重建关系的真诚。
  - 询问自己有哪些事情可以既不破坏关系，又能让你觉得没有便宜他。
  - 例如，让对方增加对关系的投入，如时间、精力或金钱。

### 问题二：结束关系后仍关注前任
**问题**：前任出轨后，我选择了结束这段关系，但还是忍不住关注他，看他发的每一条微博和朋友圈，我还是放不下他。我该怎么办呢？

**解答**：
- 放不下前任是很普遍的现象，也很正常。
- 放不下的可能并不是前任本人，而是与他在一起构成的记忆。
- 可以一边放不下，一边继续好好生活，等新生活起来后，有时自然会放下。
- 如果放不下已经干扰了新生活，可以尝试以下方法：
  - 心理学上的暴露疗法：增加关注前任的时间，直到感到负担和无聊。
  - 增加阻碍：例如，看完微博后卸载，下次想看时需要重新下载和登录。

### 问题三：原谅后对方热情下降
**问题**：我曾经试图原谅男朋友，但是还是忍不住去想他背叛我的事情，我最后还是放弃了这段关系。因为我能感受到在他原谅我以后，他对我的热情还是在下降，他还是在跟第三者联系，为什么以心相待得不到好的结局呢？我真的很失望，我该怎么走出这种失望呢？

**解答**：
- 爱情和感情关系是双人物，结局需要两个人共同的努力。
- 不是以心相待得不到好的结局，而是错的人使你得不到好的结局。
- 他与第三者保持联系，说明他不是一个对的人，也没有尊重你们的契约和你的感受。
- 真正的宽恕需要出轨者真正的潜意识和巨大的诚意去重建关系。
- 离开他是正确的，可以通过这门课重新找到投入去爱的勇气和方法。

### 问题四：发现老公出轨后的困扰
**问题**：孩子半岁的时候，我发现老公出轨，当时真的超级伤心，老公也认错了，说为了孩子不要离婚，现在已经过去了半年，我还是会时不时想起他出轨这件事情，感觉永远都过不去了，我很痛苦，不离婚我很难受。可是孩子还这么小，如果离婚的话，我又怕对孩子不好，不知道该怎么办。

**解答**：
- 感到痛苦和难以过去是很正常的。
- 在兼顾妻子和妈妈这两个角色时，你有担当的表现。
- 关于是否为了孩子维持婚姻，建议你：
  - 询问自己担心离婚对孩子造成的消极影响有哪些。
  - 思考如何既离婚又减少消极影响。
  - 寻找离婚后还能养好孩子的积极案例，学习借鉴。

### 问题五：是否挽回出轨的伴侣
**问题**：对方出轨后，我要不要挽回？

**解答**：
- 不建议挽回，因为挽回容易引发感情关系上的权力不平等。
- 挽回可能会让你越来越痛苦，追求平等的感情关系更重要。
- 在好的平等的感情关系中投入，你的投入才会被珍惜，你才能获得成长和幸福。

## 行动环节
- 记录爱的行动：关注伴侣对你的感受，并询问其他可以为你做的事情。
- 分享伴侣为你做过的爱的行动，以及感受。
