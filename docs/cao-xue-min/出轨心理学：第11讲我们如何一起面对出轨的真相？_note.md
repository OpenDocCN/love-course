# 出轨心理学：第11讲 如何一起面对出轨的真相？

**概述**： 本节课将探讨在出轨事件后，如何进行关系的重建，包括心理准备、面对真相、减少破坏性争吵等。

## 重建的心理准备

**以下是重建过程中需要考虑的几个关键点**：

- **出轨者的误区**： 出轨者往往低估重建信任的难度，高估自己承受压力的能力。
- **被出轨者的误区**： 被出轨者可能会将重建视为对出轨者的惩罚，或不断制造新的伤害。
- **关键共识**：
  - 重建是两个人的事，而非单方面的努力。
  - 重建是一个曲折的过程，需要面对争吵和失败。

## 面对出轨的真相

**以下是面对出轨真相时需要注意的几个方面**：

- **出轨者需要主动披露真相**： 包括出轨的原因、过程、影响等。
- **避免放大镜问题**： 不要过分关注细节，而是关注背后的动机和原因。
- **X光式问题**： 挖掘出轨者出轨的动机，了解其个人特质和需求。

## 减少破坏性争吵

**以下是减少破坏性争吵的方法**：

- **避免戳痛点**： 不要在争吵中提及对方的痛点。
- **避免翻旧账**： 不要反复提及过去的问题。
- **避免找队友**： 不要在争吵中寻求外界的支持。

## 总结

**本节课我们学习了重建的三个重点**：

- 重建的心理预期。
- 面对出轨真相时，将放大镜问题转化为X光问题。
- 在重建过程中，减少破坏性争吵。
