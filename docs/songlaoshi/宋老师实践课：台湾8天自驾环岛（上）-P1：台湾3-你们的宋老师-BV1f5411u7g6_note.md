## 行程起点与租车

旅程从抵达台湾并办理租车开始。自驾是探索台湾灵活且深入的方式，选择合适的车辆和租车公司至关重要。

上一节我们介绍了课程概述，本节中我们来看看行程的具体起点。

以下是租车时需注意的核心事项：
*   确认租车所需证件：通常需要**有效的驾驶执照**、**护照**以及**国际驾照认证件（IDP）**。
*   检查车辆状况：取车时务必环绕车辆检查一遍，并用手机拍照或录像记录下已有的划痕和损伤。
*   了解保险条款：明确租赁费用中包含的保险类型以及可能需要自费的额外保险项目。
*   熟悉当地交规：台湾靠右行驶，需特别注意摩托车众多，转弯和变道时要格外小心。

## 首日行程与发现

完成租车后，正式开启环岛之旅。首日的行程会带领我们初步感受台湾的风情。

以下是首日行程中的一个关键发现：
*   在某个地点，后来发现这里是免税的。这意味着在此处购买商品可以享受免缴税费的优惠，对于购物者而言是一个重要信息。
*   同样在免税地点，后来发现这里也是免税的。这进一步确认了该区域的免税属性。
*   这边是免税的。直接陈述了该区域的免税性质。
*   有点饱。这是在行程中个人的状态描述，可能与在免税区或之前地点的饮食体验有关。
*   后续多次重复提到“后来发现这里是免税的”，这反复强调了该发现，并可能表示在行程中多次确认或在不同区域都遇到了免税情况。

## 核心概念说明

在自驾旅行中，涉及费用计算时，了解免税概念很有帮助。

其核心计算公式可以简化为：
`最终支付价格 = 商品标价 × (1 - 免税税率)`
但更常见的情况是，免税商品直接以不含税价格销售，省去了计算的步骤。

在程序中，如果需要判断一个地点是否免税，可能会用到类似的条件判断结构：
```python
if location.is_duty_free == True:
    print("此处购物免税")
else:
    print("此处商品含税")
```
