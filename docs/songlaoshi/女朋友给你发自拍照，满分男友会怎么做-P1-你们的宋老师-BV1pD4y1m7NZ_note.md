# 课程名称：满分男友自拍照回应指南 - P1 🧑‍🏫

在本节课中，我们将学习当女朋友发来自拍照时，如何做出恰当的回应。我们将分析其背后的心理，并掌握一套从保存图片到组织语言的完整流程。

---

女朋友发来一张自拍照，应该怎么办。这是一件看图说话的小事。我们先来分析一下，女朋友给你发自拍照的心理动态。她是想让你夸她。那么第一步该干嘛呢。是夸她吗。错了。

第一步是保存原图。保存到手机相册。

接着，可以对照片有一个整体的印象。把原图从上到下，从左到右，从里到外，从表面到本质地观察一遍。先把握整体，再放入细节。

然后，需要运用熟练的语言技巧。可以运用比喻、夸张等多种手法进行描述。

以下是基本功的设置。请记住这个结构：整体肯定，加细节表现，再加情境设定。我来给同学们示范一下。可以说：“我宝宝真好看，嘴巴笑起来好甜。” 如果同学一发，所有男生都说他的女朋友漂亮。

但是，很多同学以为在这里就把话题结束了。这只能算作基础分。

请注意关键点。自拍说明什么。说明女朋友自己一个人。她可能感到孤独、无聊、没有人陪伴。

所以要干嘛呢。需要立刻把分析照片的思绪，以最快的速度组织成语言。把想说的话列在后面。

最后一句话，可以说：“给你的晚安。” 然后可以接着展开，营造浪漫的氛围。例如提到玫瑰、牛奶等温馨事物。

---

本节课中我们一起学习了回应女友自拍照的步骤。我们从分析心理动机开始，明确了第一步是保存图片而非直接夸奖。接着，我们掌握了“整体观察-细节描述-情境设定”的回应结构，并理解了自拍背后可能隐含的陪伴需求，从而给出了更贴心、更完整的回应范例。
