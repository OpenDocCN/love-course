# 课程：条件语句入门 - 01 🧠

在本节课中，我们将通过一个生动的生活场景，来学习编程中一个非常基础且重要的概念——**条件语句**。我们将看到，如何根据不同的“条件”或“选择”，来决定程序执行不同的操作。

---

## 场景引入：商场购物的选择

今天在商场，我看到一条好看的裙子。它有两种颜色：红色和黄色。两种颜色都很好看，这让我面临一个选择。

以下是几个可能的选项：
*   **A**：买红色的。
*   **B**：买黄色的。
*   **C**：两条都买。

在编程中，这种“根据情况做出不同决定”的逻辑，就是通过**条件语句**来实现的。

---

## 条件语句的基本逻辑

上一节我们引入了一个选择场景。本节中，我们来看看如何用编程的思维来模拟这个决策过程。

在大多数编程语言中，最基本的条件语句是 **`if...else`** 结构。它的逻辑是：**如果**某个条件成立，就做A事；**否则**，就做B事。

用我们购物的例子来类比：
```python
if 选择 == “A”:
    购买红色裙子()
else:
    购买黄色裙子()
```
这段代码的意思是：如果选择是A，就执行“购买红色裙子”这个操作；否则（即选择不是A），就执行“购买黄色裙子”这个操作。

---

## 处理多种可能：`elif` 的运用

然而，现实中的选择往往不止两个。就像我们的例子，除了A和B，还有选项C。这时，我们就需要引入 **`elif`**（“否则如果”的缩写）来处理更多条件。

以下是使用 `if...elif...else` 结构来模拟完整购物决策的代码：
```python
if 选择 == “A”:
    购买红色裙子()
elif 选择 == “B”:
    购买黄色裙子()
elif 选择 == “C”:
    购买红色裙子()
    购买黄色裙子()
else:
    print(“无效的选择”)
```
这段代码清晰地列出了所有可能的情况及其对应的操作：选A买红色，选B买黄色，选C则两件都买，如果输入了ABC之外的选项，则提示选择无效。

---

## 总结与升华

本节课中，我们一起学习了**条件语句**的核心概念。我们从一个简单的购物选择出发，逐步理解了：
1.  **`if...else`** 语句用于处理“二选一”的基本决策。
2.  **`elif`** 语句用于扩展条件，处理“多选一”的复杂场景。

通过 `if`、`elif`、`else` 的组合，我们可以让程序像人一样，根据不同的输入和条件，灵活地做出不同的反应，这是让程序拥有“智能”的第一步。记住这个基础结构，它将是你在编程世界中构建更复杂逻辑的基石。
