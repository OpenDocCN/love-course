# 当女朋友问你哪张照片好看？-P：女朋友问你哪张照片好看1-你们

在本节课中，我们将学习一个实用的沟通公式，用于当他人（例如你的女朋友）询问你对多张照片的看法时，如何给出具体、真诚且有建设性的反馈，避免笼统和敷衍的回答。

上一节我们介绍了笼统回答的弊端，本节中我们来看看一个结构化的反馈公式。

这个公式的核心结构由四个部分组成，依次对两张照片进行评论，并为每张照片提供一个具体的使用建议。公式如下：

**评论照片1 + 使用方法1 + 评论照片2 + 使用方法2**

以下是该公式的具体应用步骤：

1.  **评论第一张照片**：首先对第一张照片给出一个具体的正面评价。
2.  **建议使用方法一**：紧接着，为这张照片设想一个具体的用途或场景。
3.  **评论第二张照片**：然后，对第二张照片给出另一个具体的正面评价。
4.  **建议使用方法二**：最后，为第二张照片设想一个不同的具体用途。

让我们通过一个实例来理解这个公式的运用。

例如，当被问及哪张照片好看时，可以这样回答：
“**这一张照片的光线很柔和，我要把它设成我的手机壁纸。哇，这一张的构图好有创意，我要把它发到朋友圈分享。**”

本节课中我们一起学习了一个高效的照片评价沟通公式。通过“具体评论”结合“使用建议”的方式，你的反馈将显得更加用心、真诚且具有互动性，能有效提升沟通质量与对方的好感度。
