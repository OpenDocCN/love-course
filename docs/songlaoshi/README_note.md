# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

变量可以理解为一个存储信息的“盒子”。在Python中，你可以给这个盒子起一个名字，然后把数据放进去。之后，你就可以通过这个名字来使用盒子里的数据。

创建一个变量非常简单，只需要使用等号 `=` 进行赋值即可。等号左边是变量名，右边是你想存储的值。

**代码示例：**
```python
my_name = "小明"
my_age = 18
```

上一节我们介绍了变量的基本概念，本节中我们来看看如何给变量命名。给变量起名字需要遵循一些规则。

以下是命名变量的主要规则：
*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如，`age` 和 `Age` 是两个不同的变量）。
*   应避免使用Python中的关键字（如 `if`, `for`, `print`）作为变量名。

## 认识数据类型 🔢

变量里可以存放不同类型的数据。Python有一些内置的基本数据类型，最常见的有三种。

以下是三种核心的数据类型：
1.  **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。在代码中直接书写。
2.  **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
3.  **字符串**：由一系列字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `“你好世界”`, `‘Python’`。

**代码示例：**
```python
# 整数
score = 100

# 浮点数
price = 9.99

# 字符串
greeting = “早上好”
```

理解了不同类型后，你可能会想知道一个变量具体是什么类型。Python提供了一个非常方便的函数 `type()` 来查看。

**代码示例：**
```python
print(type(score))    # 输出：<class ‘int’>
print(type(price))    # 输出：<class ‘float’>
print(type(greeting)) # 输出：<class ‘str’>
```

## 一个简单的实践 ✨

现在，让我们把以上知识结合起来，写一个简单的程序。这个程序会创建一个变量来存储你的名字，并打印一条问候语。

**代码示例：**
```python
# 1. 创建一个字符串变量存储名字
user_name = “宋老师”

# 2. 使用print函数打印出包含该变量的信息
print(“你好，” + user_name + “！欢迎学习Python。”)

# 运行结果：你好，宋老师！欢迎学习Python。
```
在这个例子中，我们使用了加号 `+` 来连接多个字符串，组合成最终要输出的句子。

## 总结 📝

本节课中我们一起学习了Python编程的基石：
*   **变量**是用于存储数据的命名容器，通过赋值操作符 `=` 来创建和修改。
*   **数据类型**定义了数据的种类，我们重点介绍了**整数**、**浮点数**和**字符串**这三种基本类型。
*   可以使用 `type()` 函数来检查任何变量或值的数据类型。

掌握变量和数据类型是后续学习更复杂操作（如数学运算、列表、条件判断）的基础。请务必动手尝试创建自己的变量，并观察它们的行为。
