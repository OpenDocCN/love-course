## 理解问题的本质

上一节我们介绍了课程的目标，本节中我们来看看这个问题的真实意图。当伴侣询问“你喜欢我什么”时，其含义通常超越字面。

她需要安全感。她不确定你是否真心。她不清楚你是否真正了解她。她渴望一次真诚的情感表白。

## 构建回答的核心策略

理解了问题的深层需求后，我们来看看如何构建回答。关键在于不要只回答表面问题。仅列举你喜欢她的具体点是不够的。必须将女性普遍欣赏的优点都融入回答中才能获得高分。

因此，核心策略是赞美，尽情地、广泛地赞美。

你需要赞美她身上已有的许多优点。甚至可以将她身上没有但你欣赏的优点也真诚地表达出来。把你能想到的所有美好词汇都讲述出来。

以下是构建回答的具体公式：

**夸奖点1 + 细节1，夸奖点2 + 细节2，……，一直延续到夸奖点N + 细节N。**

最后，用一个真诚的表白来收尾。

## 回答示范

根据上述公式，以下是一个回答范例：

“你太温柔了，每次我看到你如水的眼睛，我都觉得自己何德何能。你太可爱了，当你生气时嘟着嘴的样子，让我愈发喜爱。”

以此类推，不断添加具体的夸奖和细节。

最后进行表白：“我爱你，你愿意嫁给我吗？”

回答时态度必须十分诚恳，并且要迅速回应。这是一个万能的回答框架。
