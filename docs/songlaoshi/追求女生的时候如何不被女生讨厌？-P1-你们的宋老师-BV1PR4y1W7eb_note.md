# 恋爱基础课 01：如何避免在追求女生时被讨厌？🚫

在本节课中，我们将学习追求女生过程中的一个核心原则：保持“普通”。许多男生因为行为“自以为是”而招致反感，我们将通过具体例子来剖析这些行为，并学习如何摆正心态。

上一节我们介绍了与女生聊天的“三不三要”口诀，本节中我们来看看如何在实际追求中应用“要普通”这一原则。

## 什么是“普通”？

“普通”意味着以平常心对待女生，避免因过度自我感觉良好而做出令人生厌的行为。许多男生在追求过程中容易陷入“自以为是”的误区。

为了更清晰地理解，我们将从女生的视角，通过几个典型例子来说明什么是“自以为是”的行为。

以下是女生眼中几种常见的、令人反感的“自以为是”行为：

1.  **过度自信，轻视对方**  
    有些男生盲目自信，认为女生对自己必然感兴趣。例如，他们会说：“你发生什么意外我都养着你”或“别多看了，再看就爱上我了”。这类言语显得轻浮且不尊重。

2.  **炫耀自我，意图明显**  
    有些男生急于展示自身价值。例如，刚加上好友就发送自己的健身或旅行照片，然后问女生：“想不想跟我一起去？”这种行为无论自身条件如何，都容易显得目的性过强且肤浅。

3.  **强行幽默，越界玩笑**  
    有些男生以朋友自居，却开不合时宜的玩笑。例如，对女生说：“你素颜的时候特别像我前女友”或“走，带你去吃好吃的”。这种玩笑并非真正的幽默，反而会让人感到被冒犯。

可以看到，这些行为的共同点在于，男生将自己放在了过高的位置。

## 核心心态：摆正位置

追求喜欢的女生时，关键在于摆正心态。不要认为自己特别帅气、幽默或了不起。

**核心公式：平常心 + 尊重 = 良好的初始印象**

这种过度自我感觉良好的心态，会让人显得 **浮夸**、 **不稳重** 且 **不真诚**。理解并践行“普通”这一基本原则至关重要。

本节课中我们一起学习了在追求女生时应避免的“自以为是”行为，并理解了保持“普通”心态的重要性。下节课，我们将深入探讨如何具体建立舒适自然的互动。
