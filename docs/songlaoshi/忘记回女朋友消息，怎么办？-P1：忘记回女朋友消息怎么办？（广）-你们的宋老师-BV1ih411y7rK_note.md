# 课程：忘记回女朋友消息怎么办？ - P1 🆘

在本节课中，我们将学习一个常见但棘手的问题：忘记回复女朋友消息后，如何妥善应对。我们将分析问题的核心，并提供一个结构化的回复公式，帮助你化解危机，甚至增进感情。

忘记回复女朋友消息十分钟，才发现自己没回，怎么办？这是一个常规的回复题。当你准备回复时，首先问她“干嘛”是错误的。直接道歉也是错误的。

但是，如果你承认自己错了，她就会追问“错哪了”。如果你答不出来，就要被“教育”。如果你答出来了，她又会问“那你该怎么改”。这等于为自己设下了苛刻的条件。此时女生的想法是：你不理她的这段时间，干嘛去了？

她可能会怀疑：你是不是和别的女生聊天？是不是不重视她、忘记她了？关键在于，她关心的是**你干嘛去了**，以及**为什么不回消息**。因此，我们的回答必须让她知道，你心里一直有她，你很爱她。虽然没回消息，但我们要扭转局面。

我们的目标不是让她更生气，而是通过回答让她心情变好。那么，回答的公式是什么呢？哪位同学记得？公式是：**体质细壮 + 自我感知 + 温情 + 霸道 + 温馨，最后加“心”**。

下面我来举个例子：
> “刚才没回你，是因为我去上网，突然看到一个包包，觉得很好看，就想着买给你。不准拒绝，只是送你而已。我爱你。”

有的同学可能会说：“宋老师，我就忘记回十分钟消息，就要去买一个包包，是不是太亏了？不想花钱。” 不想花钱也是可以的，还是用这套公式。

我再举一个例子：
> “刚才没理你，因为我在QQ浏览器看小说频道，看小说太入迷没看到消息。我正在追一本爱情小说，老婆，这主角好可爱，我突然想到了你和我幸福的过往。别怀疑，就是你和我。爱情，我爱你。”

刚才我提到的是爱情小说。看爱情小说，能瞬间拉近距离，还能让女朋友有安全感。因为女朋友会知道，你没回她的时候，心里想的依然是她。而且女朋友一听，一个大男人居然在看言情小说，这不就是想和她有共同话题、准备和她长久发展吗？所以说爱和了，女朋友往往就心软了。
