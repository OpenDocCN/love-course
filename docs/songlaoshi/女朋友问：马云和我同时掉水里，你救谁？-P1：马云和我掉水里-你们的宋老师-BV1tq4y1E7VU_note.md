# 课程名称：如何巧妙应对“假设性提问” - 第1课 🧠💡

在本节课中，我们将学习如何应对生活中常见的“假设性提问”，例如经典的“如果我和你妈妈同时掉水里，你先救谁？”的变体。我们将通过一个具体案例，分析问题本质，并掌握一个通用的回答公式，帮助你在类似情境中既能表达情感，又能巧妙化解难题。

## 女朋友问：马云和我同时掉水里，你救谁？ 💦

女朋友提问：如果马云和我同时掉水里，救马云他会给你500万，救我，我会嫁给你，你选择救谁？

这道题乍看之下，和“我和你妈同时掉水里”属于同一类问题。但这道题更具操作性，想象空间也更大。

## 问题本质分析 🔍

所以这道题的答案相对比较开放。如果回答救女朋友，女朋友可能会觉得你没有上进心，连500万都不要。如果回答救马云，女朋友则会认为你对她不是真爱，根本没想过要娶她。

这道题其实和我们上节课讲的问题属于同一类型。什么类型呢？**假设性提问**。

## 应对策略与核心公式 ✨

面对一道假设性提问，我们的回答技巧是一定要正面避开问题本身。

以下是核心回答公式：

**浪漫 + 转移**

我来拆解这个公式：你可以回答“我当然是先救你了”，然后立刻将话题转向马云——“然后看看马云会不会加价”。

---

本节课中，我们一起学习了“假设性提问”的特点与本质。我们了解到，这类问题往往没有标准答案，直接回答任一选项都可能陷入困境。关键在于使用 **“浪漫+转移”** 的公式，即先给予情感上的肯定（浪漫），再幽默或巧妙地将问题焦点转移开，从而化解两难局面。掌握这个技巧，能让你在人际沟通中更加从容。
