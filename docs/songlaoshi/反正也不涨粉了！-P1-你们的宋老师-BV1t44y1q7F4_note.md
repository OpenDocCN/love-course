# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要一个地方来存放它，并需要知道我们存放的是什么类型的东西。这就是变量和数据类型的用武之地。本节将引导你理解如何命名一个“盒子”（变量）来存储数据，以及如何区分盒子中存放的不同种类的“物品”（数据类型）。

## 1. 什么是变量？

你可以把变量想象成一个贴有标签的盒子。标签就是变量的名字，盒子里面存放的就是数据。在Python中，我们使用等号 `=` 来给变量赋值，这就像把一件物品放进一个盒子里并贴上标签。

**公式/代码示例：**
```python
my_name = "宋老师"
```
在这行代码中，我们创建了一个名为 `my_name` 的变量，并将文本“宋老师”存储在其中。

上一节我们介绍了变量的比喻，本节中我们来看看创建和使用变量时需要遵循的具体规则。

以下是定义变量时需要记住的几个关键点：
*   **变量名**只能包含字母、数字和下划线，且不能以数字开头。
*   **变量名**是区分大小写的，例如 `age` 和 `Age` 是两个不同的变量。
*   应选择有意义的**变量名**，使其能清晰表达所存储数据的含义。

## 2. 基本数据类型

数据类型定义了变量中可以存储哪种数据。Python有一些内置的基本类型，最常用的有三种：用于文本的字符串、用于整数的整数型和用于小数的浮点型。

**公式/代码示例：**
```python
# 字符串 (str)，用于表示文本
name = "Super Idol"
# 整数 (int)，用于表示没有小数部分的数字
age = 100
# 浮点数 (float)，用于表示有小数部分的数字
temperature = 36.5
```

理解了变量的容器作用后，我们需要了解容器里能放什么“物品”。下面我们来详细看看这几种基本类型。

以下是三种基本数据类型的简要说明：
*   **字符串**：由引号（单引号 `'` 或双引号 `"`）包围的任何内容。它用于存储名字、句子等文本信息。
*   **整数**：不包含小数点的正数或负数。它用于计数、索引等。
*   **浮点数**：包含小数点的正数或负数。它用于需要更高精度的计算，如测量、科学计算。

## 3. 查看与类型转换

有时，我们需要知道一个变量里到底存了什么类型的数据，或者需要将一种类型转换为另一种类型以满足计算需求。

**公式/代码示例：**
```python
# 使用 type() 函数查看数据类型
print(type(age))  # 输出：<class 'int'>

# 类型转换
number_str = "123"
number_int = int(number_str)  # 将字符串"123"转换为整数 123
```
`type()` 函数就像给变量做了一次“体检”，告诉我们它的具体类型。而 `int()`, `str()`, `float()` 等函数则是“转换器”，能改变数据的类型。

掌握了基本类型后，你可能会想知道如何确认数据的类型，以及如何在不同类型间进行转换。以下是两个常用的操作。

以下是相关的操作说明：
*   使用 **`type(变量名)`** 可以查看该变量的数据类型。
*   使用 **`int()`**、**`float()`**、**`str()`** 等函数，可以将数据转换为对应的整数、浮点数或字符串类型。
