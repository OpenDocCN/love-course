## 选项分析

上一节我们介绍了课程目标，本节中我们来看看具体的选项。以下是四个需要判断的朋友圈类型描述。

**A. 朋友圈都是自拍和肉麻情话**

这种男生的朋友圈通常频繁发布自拍。例如，一张模糊的自拍配上一段文案：“叫你宝贝没别的意思，只是忘记你的名字了。” 或是坐在车上望着窗外的自拍，配文：“我走了，和你的故事，暂停了。”

这种男生多半比较自恋，活在自己的世界里。他们总认为自己是被感情伤害的苦命男孩，总是一副在苦苦等待真爱的样子。实际上，他们特别自我且自私。朋友圈的自拍，无论怎么拍，都是千篇一律的角度和动作。发朋友圈的目的，是试图营造一个深情的人设来引诱女孩上钩。

**结论**：A是典型的伤感型渣男。因此，A是渣男，排除。

**B. 定位都是酒吧、夜店、沙鸭、太風**

经常出没于酒吧、夜店等灯红酒绿场所的男生，一般玩心都特别重。玩心重的人，责任心通常不太强，也就不会对一个人太过忠诚。再加上灯红酒绿的世界诱惑比较大，很难抗拒。因此，B是渣男的可能性非常大。

**结论**：B不可能是“非渣男”选项，排除。

**C. 全是健身、运动、腹肌的强光自拍**

健身、运动、阳光这些特质本身没有问题，不代表他是渣男。但是，请注意这里提到了“腹肌的自拍”。

总是在朋友圈发自己腹肌的行为，是一种求偶表现。准确地说，这是“广泛撒网，集中捕捞”的求偶表现，试图用自己的身材来迷惑小女生。这种男生身边确实有很多被他身材所迷倒的小女孩。正因为选择太多，他一般不太会去珍惜某一段关系。

**结论**：C多半是渣男，排除。

**D. 朋友圈都是脏话和不文明发言**

很多同学会疑惑：朋友圈都是脏话和不文明发言，不是渣男，难道他是好男儿吗？请注意，题目问的是“谁不是渣男”，而不是说“他不是渣男他就是好男儿”。

第一，朋友圈都是脏话和不文明发言的，这属于“游历男”。那么游历男是渣男吗？他不是。根据2022年新颁布的渣男标准，新一代的渣男需要有一定的文化、素质和品味才能称为“渣男”。达不到这个条件的，即使身边有很多女生，也只能叫做“游历”。

**结论**：D是正确选项，他不是渣男，是游历男。

## 核心概念总结

本节课我们一起学习了通过朋友圈内容初步判断男性行为模式的逻辑：

1.  **伤感表演型（A）**：通过**自拍+深情文案**营造人设，属于渣男。
2.  **玩心放纵型（B）**：**常定位于娱乐场所**，责任心弱，属于渣男。
3.  **外貌展示型（C）**：通过**展示身材（如腹肌）**进行广泛求偶，属于渣男。
4.  **低素质粗鲁型（D）**：**内容充满脏话和不文明用语**，不符合当代“渣男”的素质门槛，属于游历男而非渣男。

**最终答案**：根据以上分析，**D选项**所描述的男性“不是渣男”。
