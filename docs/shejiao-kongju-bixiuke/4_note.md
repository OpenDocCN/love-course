# 社交恐惧必修课 · 第四节：社交前不再焦虑的方法，哪怕要见的是你的女神 😌

在本节课中，我们将要学习如何应对社交前的预期焦虑。我们将探讨什么是预期焦虑，它为何会产生，以及如何通过一个简单的公式来有效管理它，从而显著减轻社交恐惧。

## 人人都有预期的焦虑 😟

上一节我们探讨了社交恐惧的表现形式，本节中我们来看看一个核心概念：预期焦虑。这是一种普遍存在的情绪。

预期的焦虑指的是：由于过去的不良经历，导致对未来即将发生的事情的担忧。一般人多少都会有预期的焦虑。对自己要求越高的人，或者越没有安全感的人，预期的焦虑就越严重。而预期焦虑越严重的人，就越容易表现得糟糕。

以下是预期焦虑的一个具体例子：
> 许久未见的高中同学组织的同学会明天就要聚会了，我现在就开始担忧，担心自己会表现得紧张丢脸。因为我大学期间有了社恐，而高中时我给同学的印象是比较热情开朗的。我非常害怕让他们发现我变成了一个自卑胆小的人。这种预期的焦虑导致我整个晚上睡不好觉。精神状态不好，加上不断地思考这个事情，就让我更容易表现得不自然。所以第二天果然表现得紧张。

### 预期焦虑的特点

以下是预期焦虑的两个主要特点：

1.  **有过不良的经历，担心未来可能再次发生**。我们不会无缘无故地对聚会聚餐产生害怕。一定是在过去类似的场合中，出现过焦虑、紧张或恐惧的情况。因此，当再次面临时，就会非常害怕。
2.  **对自己要求越高的人，越容易产生预期的焦虑**。对自己要求越高，或者越没有安全感的人，预期的焦虑就越严重。

### 预期焦虑的本质：自我保护机制

预期焦虑看似让我们不舒服，但实际上它有存在的价值。它的本质是让我们预见危险，并规避危险。

我们下意识反应没有逻辑，它只会根据你遇到过危险或对危险的理解，来判断当下是否需要躲避。因此，预期焦虑的产生，是希望让我们远离危险。它的意义是让我们能够更健康、更安全地活下去。

## 正常的 vs. 过分的预期焦虑 ⚖️

理解了预期焦虑的普遍性和价值后，我们需要区分正常和过分的预期焦虑。

正常的预期焦虑是为了规避危险。人人都有预期焦虑，如果没有，反而无法预见危险，容易陷入险境。对于一般人来说，预期的焦虑有其正面的价值和意义。而且，一旦我们为预期的焦虑做出应对方案，焦虑就会减轻或消失。

**公式：预见危险 -> 制定方案 -> 焦虑减轻**

例如，面对即将到来的面试产生预期焦虑，担心失败。于是我们做了充分准备，准备了公司材料、面试问题等。在做充分准备之后，我们的预期焦虑也就会消失。

然而，社恐人容易出现的是**过分的预期焦虑**。过分的预期焦虑不仅仅无法规避风险，反而会影响我们的正常生活。它会有高度的焦虑感，并且我们很容易用错误的方法去对待它，反而让我们容易沉浸其中无法自拔。

以下是过分的预期焦虑的一个例子：
> 我即将跟心目中的女神约会，提前一周就开始焦虑。我担心表现得不好。每每想到要约会，我就非常紧张，吃饭都没胃口。但我没有把精力放在准备合适的衣服、聊天话题或约会项目上，而是把所有精力都放在担心约会那天会紧张不自然，以及女神会如何失望上。我实在太焦虑了，于是找了个借口取消了约会。取消之后又非常后悔和自责。

### 两者的核心区别

以下是正常与过分预期焦虑的核心区别：

*   **焦虑程度**：正常焦虑程度较低；过分焦虑程度极高。
*   **精力投向**：正常焦虑会想办法去解决问题；过分焦虑则把大量精力放在对未来的担忧上，而非解决问题本身。

我们要解决过分的预期焦虑，关键就是把精力放在**解决问题本身**，或者放在**我们正在做的事情**上。当我们把精力放在正确的地方时，焦虑程度不但会降低，行动能力也会变得更强。

## 解决预期焦虑 = 解决一半社恐 🎯

在探讨了预期焦虑的区分后，我们来看为什么解决它如此重要。

**解决预期的焦虑等于解决一半社恐**，这毫不夸张。预期焦虑是社恐能够持续存在的能量来源之一。

社交恐惧的形成，往往是**先回忆，后预期**。例如，有一次上台回答问题表现不好，被嘲笑。此后便开始反复回忆和纠结这件事（回忆），然后每当这位老师上课时，就开始担忧再次被叫上台（预期）。反复多次之后，就形成了社交恐惧的症状。

持续的预期和社交之后的回忆，总是在不断地强化自己社交时紧张不自然的感受。每一次陷入回忆和预期，就是在不断地关注症状，给症状提供源源不断的养料。

很多同学有一个误解，认为只要在社交时不紧张才是最重要的。当然这很重要，但最开始要处理的一定是**预期的焦虑**和**社交后的回忆**。如果这两个部分没有解决好，再怎么解决社交时的恐惧感，效果都很有限。

甚至可以说，社交时紧不紧张其实**一点都不重要**。只要你能够把预期的焦虑和社交后的回忆处理好了，社交时的紧张自然会减轻。而且，解决预期焦虑远比解决社交时不紧张来得简单，因为它的焦虑程度相对较低，也更容易从症状中跳出来。

所以，我们的重点应该放在解决社交前的预期和社交后的回忆上。

## 解决预期焦虑的公式 📝

既然预期焦虑如此关键，那么如何解决它呢？我们提供一个可实践的公式。

预期焦虑不可被完全“解决”，因为它是一种自我保护机制。但我们可以**无限地减小**它对我们的影响。我们的目标是将其减小到正常水平。

**解决预期焦虑的公式：识别 -> 承认 -> 专注当下 -> 忍受**

### 1. 识别预期的焦虑
首先，要能察觉到自已陷入了预期焦虑。判断标准有两点：
*   是否在**反复担忧**一件未来可能发生的事情？
*   是否**陷入到焦虑的情绪**中？

只要发现自己出现这种状态，就可以告诉自己：“我现在可能陷入到预期的焦虑了。”

### 2. 承认其存在的意义
预期焦虑的存在本身是有意义的，是为了**规避风险**。只是我们有社恐，安全感极低，才陷入了过分的预期焦虑。承认它的意义，本质是理解它，不去排斥和对抗它。我们知道：“它出现是因为我害怕丢脸、害怕让人失望，这是我的下意识在试图保护我。”

### 3. 专注于当下（切断关注）
负面情绪持续存在，是因为我们的**关注**。如果不关注它，它就会消失。专注当下，就是把注意力拉回到此时此刻所做的每一件事情上（工作、学习、休息等）。记住一句话：**未来的事情未来办**。除非是为未来做实际准备（如准备面试材料），否则不需要现在处理症状带来的担忧。

### 4. 忍受焦虑
我们并非能完全投入当下，内心会有不安。我们需要**带着担忧去投入当下**。要相信，只要持续不关注负面情绪，它迟早会消失。记住：**不安常在，带着不安去做事情**。忍受当下的不适感，坚持做该做的事，是正确的路径。

## 公式应用案例 🧑‍💻

让我们通过一个案例，看看如何具体运用这个公式。

**场景**：今天是周三，周五下午我要作为项目组长，在百人大会上做工作汇报。我听到消息后非常担忧，害怕上台紧张、丢脸，怕辜负领导的期望。

**应用公式**：
1.  **识别**：我感到心跳加速、坐立不安，并且反复在想“上台丢脸怎么办？”。我意识到：“我陷入了预期的焦虑。”
2.  **承认**：我理解这种焦虑的产生。因为我缺乏安全感，害怕危险（丢脸、让人失望），我的下意识在用焦虑提醒我。这很正常，我不对抗它。
3.  **专注当下**：我知道反复担忧不能解决问题，只会让状态更糟。今天是周三，我该工作就继续工作。同时，我可以为汇报做**实际准备**：完善演讲稿、演练内容。我的注意力放在“准备”和“完成手头工作”上。
4.  **忍受焦虑**：准备过程中，不安感依然存在。我接受这种不舒服，但不理会它。我继续做我的事，相信只要我持续投入当下，这种焦虑感会逐渐减弱。
