# 恋爱方法：高阶篇 — 如何让女生更深入的爱你（第二集）

🎼大家好，我是乐天。欢迎来到恋爱方法。今天我们将探讨如何让女生更深入地爱你。首先，我们需要了解一个概念——投资。

**投资的概念**：当你和女朋友分手时，你感到伤心，这并非因为感情的结束，而是因为你在这段感情中投入了时间、金钱和物质等。

**投资的五大块**：

1. **时间投资**：指你们共同投入的时间，以及一方为另一方所做事情的时间。
2. **情感投资**：指你们为这段感情所投入的情感。
3. **物质投资**：指你们为对方所花费的金钱或其他物质，如买衣服、买房等。
4. **身体投资**：指你们为了吸引对方而做的身体方面的投资，如整容、性方面的投资等。
5. **心理投资**：指你们为对方做事情，甚至为了对方而为难自己，如为了和你在一起而放弃和家人团聚。

**投资的两种类型**：

1. **主动投资**：对方主动为你做的事情，如主动买衣服、为你做事等。
2. **引导投资**：引导对方为你做事情，如你口渴了，让对方帮你买水。

**总结**：在整个投资过程中，你不能只让对方投资，你也需要投资对方，这样才能建立良好的感情。
