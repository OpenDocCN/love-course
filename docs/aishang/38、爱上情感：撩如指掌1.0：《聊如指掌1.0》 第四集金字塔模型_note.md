## 聊天金字塔模型
聊天金字塔模型将聊天分为五个层次：

1. **陌生层**：初次见面，话题如问候、天气等。
2. **熟悉层**：关系进一步，可以聊个人兴趣、工作等。
3. **暧昧层**：关系更加亲密，可以聊情感、约会等。
4. **情人层**：关系稳定，可以聊亲密话题、未来规划等。
5. **确立关系层**：正式确立情侣关系。

### 陌生层
- **话题**：你好呀，天气不错，你在加班吗？
- **目的**：建立初步联系。

### 熟悉层
- **话题**：你多大了，你在干嘛呀，你忙不忙？
- **目的**：加深了解。

### 暧昧层
- **话题**：想我了吗？宝宝，亲一下抱一下么么哒。
- **目的**：拉近关系。

### 情人层
- **话题**：我搂着你，你会亲我吗？你爱不爱我？
- **目的**：确认关系。

### 确立关系层
- **话题**：叫老公，确立情侣关系。
- **目的**：正式确立关系。

## 案例分析
以下为两个错误的聊天案例：

1. **案例一**：两人认识多年，但话题始终停留在第一层和第二层，无法升级关系。
2. **案例二**：两人聊天内容停留在第一层和第二层，无法有效推进关系。

## 正确运用金字塔模型
以下为正确运用金字塔模型的聊天案例：

1. **案例一**：通过展现共同兴趣，快速进入第二层，然后逐步进入暧昧层。
2. **案例二**：直接称呼对方为“小宝宝”，快速进入暧昧层，然后逐步进入情人层。
