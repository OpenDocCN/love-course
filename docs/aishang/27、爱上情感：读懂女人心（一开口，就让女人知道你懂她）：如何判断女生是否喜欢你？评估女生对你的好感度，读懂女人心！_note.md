要掌握社交直觉，你需要具备以下两种基础能力：

以下是社交直觉的两个核心组成部分：

**公式**：**社交直觉 ≈ 情商 = 精准表达 + 准确理解**

没有社交直觉，与女生聊天就容易陷入“尬聊”，双方各说各话，无法建立有效连接，更难以让对方喜欢上你。
