# 40、爱上情感：魅力形象：15.《魅力形象》之发朋友圈的注意事项

🎼 hello，大家好，我是乐天。本节课我们将探讨发朋友圈时的一些注意事项。

### 1. 发朋友圈的频率

**朋友圈不要天天都发，几天发一次即可。不发或发一些有意义的。**

**公式**：朋友圈发布频率 = 有意义内容 / 总发布次数

### 2. 照片数量

**不要每次都发很多张，除非照片质量高且不重复。1-2张即可。**

### 3. 修图风格

**朋友圈照片修图风格最好统一，避免杂乱。**

**公式**：朋友圈修图风格统一度 = 统一风格照片数 / 总照片数

### 4. 自拍与他人视角

**不要全是自己自拍，使用他人视角拍摄的照片是高品质朋友圈不可或缺的。**

**公式**：他人视角照片占比 = 他人视角照片数 / 总照片数

### 总结

本节课中我们一起学习了发朋友圈时需要注意的几个要点，包括发布频率、照片数量、修图风格以及自拍与他人视角的平衡。希望这些内容能帮助大家更好地展示自己的魅力形象。
