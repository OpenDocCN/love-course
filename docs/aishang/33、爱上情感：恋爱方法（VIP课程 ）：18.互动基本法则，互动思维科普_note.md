# 爱上情感：恋爱方法（VIP课程）18.互动基本法则，互动思维科普

## 概述

在本节课中，我们将学习互动基本法则和互动思维科普，包括：

* 不要为了聊天而聊天
* 提供价值与索取价值
* 真实
* 过于逗逼
* 主导话题
* 正经话题与不正经话题的比例
* 带入现实感

## 1. 不要为了聊天而聊天

**公式**：聊天目的 = 提升关系 + 建立联系感

* 不要漫无目的的聊天，要明确聊天目的。
* 不要为了聊天而聊天，要结合实际情况和课程内容。

## 2. 提供价值与索取价值

**公式**：提供价值 = 站在对方角度思考 + 展示真实自我

* 提供价值要站在对方角度思考，而不是单方面索取。
* 判断是否在索取价值的方法：是否为了达成目的而做某些事情。

## 3. 真实

* 真实可以化解废物测试，因为真实的人很少。
* 表达真实自我可以建立人格吸引力。
* 不要戴着面具社交，要真实表达自己的想法。

## 4. 过于逗逼

* 过于逗逼会降低自己在女生中的吸引力。
* 幽默只是加分项，不是必选项。
* 要根据实际情况调整自己的状态，该正经就正经，该逗逼就逗逼。

## 5. 主导话题

* 不要任由话题发展，要控制话题方向。
* 要有流程图，明确下一步怎么做。
* 要引导话题发展，顺应关系正常发展。

## 6. 正经话题与不正经话题的比例

* 正经话题用于建立联系感，不正经话题用于升级关系。
* 两个话题相辅相成，螺旋式上升。
* 要根据实际情况调整话题比例。

## 7. 带入现实感

* 要让女孩感受你的真实生活，真实人格三观。
* 要让女孩感受到你的真实样子，而不是停留在网络上。
* 要让女孩感受到你的安全感，而不是担心你。

## 总结

本节课中，我们学习了互动基本法则和互动思维科普，包括不要为了聊天而聊天、提供价值与索取价值、真实、过于逗逼、主导话题、正经话题与不正经话题的比例、带入现实感等。希望同学们能够将这些知识应用到实际生活中，提升自己的恋爱技巧。
