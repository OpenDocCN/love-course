# 36、爱上情感：聊天达人（从小白快速变身聊天高手）：和女生聊天如何传递情绪，什么是情绪聊天？

## 概述
在本节课中，我们将学习如何在与女生聊天时传递情绪，以及什么是情绪聊天。我们将探讨如何通过情绪聊天吸引女生，并了解情绪在聊天中的重要性。

## 什么是情绪聊天？
情绪聊天是指通过语言、语气、表情等方式传递情绪，使对方感受到你的情感。情绪聊天不仅仅是文字的交流，更是情感的沟通。

## 情绪聊天的重要性
在与人沟通时，语言只占7%，语气占38%，肢体语言占55%。因此，情绪在聊天中占据了非常重要的地位。

## 传递情绪的方法
以下是四种女生喜欢的情绪语言模式：

### 1. 日常用语
使用日常用语，如“靓仔”、“小姐姐”等，可以迅速拉近与女生的距离，传递出亲切感。

### 2. 网络用语
在聊天过程中，适当使用网络用语，如“老铁666”、“小奶狗”等，可以增加聊天的趣味性，传递出轻松愉快的氛围。

### 3. 非逻辑式调情
通过非逻辑的方式表达情感，如“今天的天气好难，我想起了你”，可以打破常规，增加聊天的趣味性。

### 4. 表情
使用表情，如“😊”、“😂”等，可以更直观地表达情绪，使聊天更加生动有趣。

## 总结
本节课中，我们学习了如何在与女生聊天时传递情绪，以及情绪聊天的重要性。通过掌握四种情绪语言模式，我们可以更好地与女生沟通，吸引她们的注意。记住，情绪是聊天的灵魂，聊天只是一个载体。
