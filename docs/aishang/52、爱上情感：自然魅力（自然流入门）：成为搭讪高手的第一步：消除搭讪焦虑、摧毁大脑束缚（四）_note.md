# 52、爱上情感：自然魅力（自然流入门）：成为搭讪高手的第一步：消除搭讪焦虑、摧毁大脑束缚（四）

在本节课中，我们将探讨如何消除搭讪焦虑和摧毁大脑束缚，以成为搭讪高手。

## 消除身份感的阻碍

这是一个改变行动预期的重要方法。以下是一些常见问题：

*   **自我怀疑**：许多人认为他们没有资格搭讪，内心不允许，不敢接近。
*   **身份认同**：认为自己不适合搭讪，更愿意保持内向、绅士、安静的形象。

### 身份感与行为模式

*   **身份感**：认为自己没有资格，不是那种人。
*   **行为模式**：导致不受女生欢迎的行为。

### 改变身份感

*   **设定新身份**：想要随心所欲地接近喜欢的女孩子。
*   **行动**：用新身份行动，接近女孩子。
*   **情绪调整**：情绪会抵抗，但需要通过行动逐渐对齐。

## 恐惧与改变

*   **恐惧变化**：人类本能上讨厌改变，讨厌挑战舒适区。
*   **大脑机制**：大脑倾向于阻止未知、复杂的事情。
*   **行动**：克服恐惧，采取行动。

## 逃避现实与行动

*   **逃避现实**：逃避搭讪，寻找借口。
*   **行动**：克服逃避，采取行动。

## 期待高光时刻

*   **高光时刻**：大脑想象中的美好时刻。
*   **现实**：需要行动，而不是等待。

## 生命的价值

*   **行动**：改变需要行动，而不是等待。
*   **价值**：通过行动获得真正的价值。

## 总结

本节课中，我们学习了如何消除搭讪焦虑和摧毁大脑束缚。要成为搭讪高手，需要改变身份感，克服恐惧，采取行动。不要期待高光时刻，而是通过行动获得生命的价值。
