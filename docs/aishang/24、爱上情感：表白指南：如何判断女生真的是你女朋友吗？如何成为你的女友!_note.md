# 24、爱上情感：表白指南：如何判断女生真的是你女朋友吗？如何成为你的女友!

在本节课中，我们将探讨如何判断女生是否真的是你的女朋友，以及如何成为她的男友。

## 名义女朋友与实质女朋友

**什么是名义女朋友？**
- 与对方聊天或表白，对方未拒绝但未答应，你就觉得这是你的女朋友。
- 实际上，这只是对方出于礼貌或未理解你的意思。

**什么是实质女朋友？**
- 你们在生活中发生亲密行为，做情侣之间应该做的事情。
- 你们能够在一起亲密，女孩会主动找你，不想和你分开。

## 男生如何获得女朋友

以下是一些男生获得女朋友的常见方式：

1. **见面次数少，聊天不多**：男生和女生见面次数少，聊天内容主要是第三方话题，如学习、工作等。
2. **女生给予普通信号**：女生对你微笑或打招呼，你就以为她喜欢你，然后表白。
3. **约会形式普通**：你们出来约会，就像普通朋友一样，没有亲密行为。

## 吸引与魅力价值

**价值产生吸引**：吸引等于价值。

**什么是价值？**
- 广义的价值：生存价值、繁衍价值、金钱价值、颜值价值等。
- 狭义的价值：魅力价值，即行为线索。

**魅力价值**：
- 强支配力、底气、肯定感、不在乎他人看法、好情绪来源、理所当然感、决断力、清晰的界限感等。

## 广义价值与狭义价值

**广义价值**：
- 短期价值：视觉线索、魅力价值。
- 长期价值：金钱价值、颜值价值、帮忙价值、备胎价值等。

**狭义价值**：
- 魅力价值，即行为线索。

## 情人与供养者

**情人类型**：
- 男性拥有魅力价值，女性会自然喜欢他们。
- 他们把女性当做女人看待，而不是女神。

**供养者类型**：
- 男性拥有备胎价值，把女性当做女神。
- 他们需要给女性买礼物、表白等。

本节课中，我们学习了如何判断女生是否真的是你的女朋友，以及如何成为她的男友。关键在于吸引和魅力价值，以及如何展现你的价值。希望这节课对你有所帮助。
