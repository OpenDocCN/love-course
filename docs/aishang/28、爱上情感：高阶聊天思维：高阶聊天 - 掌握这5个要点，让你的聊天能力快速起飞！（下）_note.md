## 1. 以你为中心，而非以女生为中心

**核心概念**：主导关系

女生的社交通常较为被动，她们即使喜欢某个男生，也很少会主动表达。因此，作为男生，你需要学会把握机会，主动出击，而不是被动等待。

**公式**：主动出击 > 被动等待

**代码**：`#主动出击 #被动等待`

## 2. 控制需求感

**核心概念**：需求感控制

在聊天过程中，你需要控制自己的需求感，使其小于等于女生的需求感。

**公式**：你的需求感 ≤ 女生的需求感

**代码**：`#需求感控制`

## 3. 以你为中心的互动

**核心概念**：以你为中心

在互动过程中，所有话题和活动都应以你为中心，让女生感受你的魅力。

**公式**：以你为中心 > 以女生为中心

**代码**：`#以你为中心`

## 4. 说什么不重要，重要的是谁说的

**核心概念**：一致性

在聊天过程中，文字表面的内容占比很小，重要的是传递出的吸引力和一致性。

**公式**：吸引力 + 一致性 > 文字内容

**代码**：`#吸引力 #一致性`

## 5. 改变情绪，而非逻辑

**核心概念**：情绪体验

在聊天过程中，你需要关注女生的情绪体验，而不是逻辑。

**公式**：情绪体验 > 逻辑

**代码**：`#情绪体验`
