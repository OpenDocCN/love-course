# 33、爱上情感：恋爱方法（VIP课程 ）：资源收集篇 — 搭讪技巧讲解（第四集）_note

在本节课中，我们将要学习搭讪过程中的几个核心技巧，包括如何有效截停对方、沟通时的态度以及收号时的策略。这些技巧旨在帮助你更顺畅地开启一段对话并建立初步联系。

上一集我们介绍了搭讪的开场白和一些基础技巧。本节中，我们来看看如何通过具体的行动步骤来提升搭讪的成功率。

## 1. 有效截停对方 🛑

搭讪的第一步是让女生停下来与你交流。如果对方不停下，你便无法展开对话。

以下是截停女生的建议方法：
*   **角度与方式**：建议从女生的侧边，以大约**45度角**的方向接近并截停。可以说“你好，请等一下”，目的是让她看到你。
*   **成功标志**：如果对方停下来，你的成功几率就已达到**50%**。如果对方继续前行，成功几率则很低。

因此，让女生停下是至关重要的一步。如果对此仍有疑问，建议参考相关的搭讪示范视频。

## 2. 保持友善的沟通态度 😊

成功截停后，接下来需要与对方交谈。此时，保持友善的态度非常重要。

与女生交流时，语气应尽量友善。因为你是去认识对方，如果表现得很凶，女生可能会认为你脾气不好。

## 3. 展现真诚的意图 🤝

除了态度友善，展现真诚也同样关键。

你需要真诚地上去打招呼并认识对方。可以告诉女生：“我是很真诚地过来认识你的。” 真诚是建立信任的基础。

## 4. 收号时的选择技巧 📱

当对话进行到一定程度，你想添加对方微信时，可以使用一个小技巧来促成。

在最后阶段，你可以给女生一个选择题：“**是我加你，还是你扫我？**” 无论她选择哪一种方式，你都算成功了。这是一个收号的常用惯例。

## 5. 应对拒绝与调整心态 🔄

即使使用了技巧，也可能面临拒绝。这时，调整好心态并妥善应对非常重要。

如果女生拒绝，你可以尝试再努力一次。例如，可以说：“如果你后面觉得我是坏人，可以随时把我微信删掉。” 看看对方是否愿意改变主意。

如果她依然拒绝，建议不要纠缠，选择放弃即可。

有时女生会说“我有男朋友了”或“这样不好”。你可以用简单的话应对，例如回应“我也有（男朋友），我只是单纯想认识你一下”，然后继续尝试添加微信。
