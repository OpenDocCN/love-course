# 28、爱上情感：高阶聊天思维：高阶聊天 - 被正妹删除拉黑，如何重新挽回让她喜欢我（案例讲解 上）

## 案例背景

本案例讲述了一位男士在社交软件上认识一位女士，经过一段时间的聊天后，女士将他删除。后来，男士通过其他途径重新联系上女士，并尝试重新建立联系，最终成功挽回女士的故事。

## 聊天破冰

**以下是聊天破冰的步骤：**

1. **社交软件转微信**：如果你不知道如何在社交软件上转微信，可以咨询客服。
2. **自我介绍**：直接告诉对方你是谁，例如：“我是小叔叔。”
3. **备注对方**：将对方的名字打出来，并备注。
4. **描述自己状态**：以自己为中心，描述自己的状态，例如：“醒来就跑来公司加班审稿子了。”
5. **提供话题**：提供有趣的话题，例如：“我估计我的肾要被加班给搞坏了。”
6. **回应对方**：根据对方的回应，给予适当的回复。

## 聊天技巧

**以下是聊天技巧：**

1. **保持社交地位**：不要跪舔对方，保持社交地位。
2. **提供价值**：在聊天中提供价值，让对方感受到你的价值。
3. **关注对方**：关注对方的需求和感受，让对方感受到你的关心。
4. **情绪体验**：在聊天中创造情绪体验，让对方感受到快乐、悲伤等情绪。
5. **话题拓展**：根据对方的话题，拓展话题，让聊天更加深入。
6. **避免逻辑聊天**：避免与对方进行逻辑聊天，以免聊天变得无聊。
7. **关注兴趣指标**：关注对方的兴趣指标，例如：解释、发长信息等。
8. **不要删除对方**：不要删除对方，以免失去了解对方的机会。

## 案例分析

**以下是案例分析：**

1. **男士通过社交软件认识女士，并通过聊天建立了联系。**
2. **女士将男士删除，但男士通过其他途径重新联系上女士。**
3. **男士通过聊天技巧，重新建立了与女士的联系，并成功挽回女士。**

## 总结

本节课中，我们学习了如何通过聊天技巧，重新挽回被删除的女士。希望同学们能够将所学知识应用到实际生活中，提升自己的聊天能力。
