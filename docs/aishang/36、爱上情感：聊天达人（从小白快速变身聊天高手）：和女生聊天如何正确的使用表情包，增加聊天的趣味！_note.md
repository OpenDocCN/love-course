# 36、爱上情感：聊天达人（从小白快速变身聊天高手）：和女生聊天如何正确使用表情包，增加聊天的趣味！

在本节课中，我们将学习如何在与女生聊天时正确使用表情包，以增加聊天的趣味性。

## 表情包的作用

表情包的主要作用是**缓和气氛**，使聊天更加幽默、舒适，减少需求感，拉近与女生的距离。

## 如何找到合适的表情包

1. **积累表情**：平时留意别人发的表情，觉得好的就保存下来。
2. **微信群**：加入一些微信群，保存群里有趣的表情。
3. **聊天软件**：使用带有表情功能的聊天软件，如搜狗打字。

## 正确使用表情包

1. **开场表情**：使用可爱、有趣的表情进行开场。
2. **互怼表情**：使用能挑起女生情绪的表情，如“爸爸来教你做人”。
3. **夸赞表情**：使用赞美女生的表情，如“你怎么这么招人喜欢”。
4. **安慰表情**：女生情绪不好时，使用安慰表情，如“一不要？没有人没有人要我要你啊”。
5. **高冷表情**：使用高冷表情调逗女生，如“像我这么高冷的人是不会理人的啊”。
6. **调情表情**：使用调情表情拉近关系，如“我不信除非你亲我一下”。
7. **卖萌表情**：使用卖萌表情，如“哎呀，别别用就别说话，用心去感受”。

## 表情包只是聊天的一种延伸

表情包只是聊天的一种延伸，真正的关键在于你的聊天能力。提升聊天能力，可以参考我们的聊天技巧课程。

本节课中，我们学习了如何正确使用表情包，以增加聊天的趣味性。记住，表情包只是聊天的一种延伸，提升聊天能力才是关键。
