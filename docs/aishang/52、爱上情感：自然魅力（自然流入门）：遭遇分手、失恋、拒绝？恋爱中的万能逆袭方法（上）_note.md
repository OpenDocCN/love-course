# 52、爱上情感：自然魅力（自然流入门）：遭遇分手、失恋、拒绝？恋爱中的万能逆袭方法（上）_note

在本节课中，我们将要学习如何面对恋爱中的挫折、拒绝与失恋。我们将探讨什么是你能掌控的，什么是你无法掌控的，并学习如何为自己设立正确的目标。核心在于理解挫折与能力的关系，并掌握一种能够实现自我成长与逆袭的思维方式。

## 概述：掌控与无法掌控

在恋爱中，有一部分事情你可以完全掌控，另一部分则完全无法掌控。本节课将讲解如何面对这些问题，以及如何为自己设立正确的目标。我们将深入探讨挫折的本质以及能力的含义。

## 自我：一切问题的根源

上一节我们提到了掌控与无法掌控的区分，本节中我们来看看问题的核心——自我。在我从事恋爱领域研究的这些年里，我越发发现，研究恋爱之道的终极门槛，其实就是研究我们自己。

**我们是“因”，女生是“果”。** 因为你做了某些事情、展示了某些特质，女生才被你吸引。这是一个因果关系。许多人感觉与女生聊天没有话题、面对女生感到不适、搭讪时情绪紧张，这些问题都属于内部阻碍，根源都在你自己身上。

例如，当你面对心仪的女生没有话题时，并不是女生导致你无话可说，而是你遇到她时，内心出了问题，感觉自己没有话题。因此，研究恋爱之道，最终就是研究我们自己。

**自我是一切问题的开始，也必将是一切问题的终结。** 我们的主题将回归到如何改变自我、研究自我。我们不能推卸责任。很多人可能会抱怨女人拜金、不懂珍惜好男人、势力、喜欢伤害好男人等。虽然推卸责任会让我们感觉轻松，仿佛问题不在自身（例如，归咎于身高、财富、房产），但这样做的同时，你也放弃了解决问题的权利。

对于解决恋爱问题和自我提升，最主要的阻碍就是关于自我的内部阻碍。为了追求美好的爱情，我们必须不断完善自我，承担相应的责任。你必须拥有能量。例如，如果你想提升情商、认识更多女生，就必须打破原有的社交圈，这过程中可能需要承担被拒绝、被误解的风险。这是成长的必经之路。

你必须为自己的命运负责，没有退路。没有人会替你解决单身或被拒的问题。除非你主动行动。我们的外部课程虽然是免费的，但总有人不愿去听、不愿去解决。对于这部分尚未准备好承担责任的人，我们爱莫能助。

## 两种心态：推销者与购买者

上一节我们探讨了自我责任的重要性，本节中我们来分析两种关键的恋爱心态。不要试图用女人来填补你的空虚和不开心，不要总想着通过向外索取来让自己圆满。许多人因为单身而不开心，认为只有心仪的女生和自己在一起才能快乐，这都属于自我不圆满的状态。这种不断索取的心态，会让你沦为一个没有吸引力的“推销者”。

我们可以把心仪的女生比作看电影。只有电影吸引观众，而不是观众吸引电影。明星在台上，也是我们被明星吸引。在与女生相处时，你要成为电影的“导演”，输出价值来吸引她，而不是等着看女生如何“表演”。这种思维方式必须转变。

记住一点：**你本身应该是开心、圆满、有能量的。** 你需要用你的能量和开心去吸引女生来到你身边。

在心态上，可以区分两种角色：
*   **推销者心态**：你认为自己需要从对方那里索取某些东西（如电话号码、机会）。这种心态会导致你感到紧张、自我怀疑，阻碍你的行动。
*   **购买者/提供者心态**：你是一个提供价值的人。想象一下，如果你在街上随机向人发放人民币，对方拒绝你，你还会痛苦吗？显然不会。这就是心态改变的力量。

以下是两种心态的核心区别：
*   **推销者**：`心态 = “我能从她那里得到什么？”` 行为表现为紧张、害怕拒绝。
*   **购买者/提供者**：`心态 = “我能给她带来什么价值或体验？”` 行为表现为从容、无惧拒绝。

## 自我的边界与世界的碰撞

我们继续深入自我的概念。学者三本耀司（注：此处应为山本耀司，根据常见引用修正）说过，自我是看不见也摸不着的。只有撞上一些别的、强大的东西，反弹回来，你才会了解自己。

一个两三岁的小孩可能认为自己是世界的主宰，哭闹就能得到一切，他的“自我”就是整个世界。然而，真实世界非常复杂且庞大。

请看下图（想象一张图）：你眼中的世界很小，而真实世界很大，并且充满“暗礁”。在自我成长的过程中，你会触礁——即碰到一些高水准的、你未曾解决过的问题。就像很多男生从小缺乏恋爱教育，在与女生相处时一定会遇到挫折，这些挫折就是暗礁。它们会刺痛我们，让我们感到痛苦，这就是挫折。

随着成长，比如一个6岁的孩子和其他小朋友抢玩具，他发现别人不会轻易把玩具给他，这时他才形成了自我的边界感，明白世界不是围着自己转的。

## 面对暗礁：退缩与成长

对于普通人来说，当触碰到超出自己认知范围、水准较高的“暗礁”时，他们可能会选择退缩和逃避。这样，他们的世界就会变得更加狭隘。

如果一个人遇到阻碍（如表白被嘲笑、被当作供养者、被背叛）后选择退缩，他的世界就会变小。反之，如果他迎难而上，将这些未曾经历过的问题内化为自己的一部分，他就获得了成长。前者叫“退缩”，后者叫“成长”。那些被你内化的“暗礁”，就变成了你的能力。

我们每个人都活在一个自以为很了解的狭小舒适区里。你认为所有女生都像你妈妈一样照顾你，或者认为只要真诚就能获得爱情。但这只是你想象中的世界。

真实世界要复杂得多。一旦你在现实中与女生相处、表白、付出，却发现被拒绝、被发好人卡，你就“触礁”了。你进入了一个头脑空白的领域，这时面临两个选择：退缩或成长。

*   **选择成长**：困扰你的问题被内化，你的世界变大，能力变强。
*   **选择退缩**：你的世界收缩，变得更加狭隘。

## 挫折的两种模式：欧皇与非酋

上一节我们了解了面对“暗礁”的两种选择，本节中我们来看看由此衍生的两种心理模式。人在面对挫折时，会开启两种模式，类似于游戏中的“欧皇模式”和“非酋模式”。

*   **欧皇（欧洲皇帝）模式**：代表富饶、运气好、积极。
*   **非酋（非洲酋长）模式**：代表点背、运气差、消极。

请看下图（想象一张趋势图，横轴是时间/经历，纵轴是能力水平）：
*   **欧皇路径**：遇到障碍（红线） -> 进入“损失区”（能力暂时低谷） -> 积极应对，内化挫折 -> 能力上涨到比之前更高的水平。
*   **非酋路径**：遇到障碍 -> 进入“损失区” -> 抱怨、退缩 -> 能力下降到比之前更弱的水平。

“损失区”就是我们面对挫折、处于人生低谷的时期。关键区别在于心态。

**欧皇与非酋的核心区别：**
1.  **预期不同**：
    *   **欧皇**：关注美好预期和已获得的成长。例如，搭讪5个女生成功1个，他会想：“太好了，我认识了一个心仪的女生！”
    *   **非酋**：关注糟糕预期和痛苦。同样情况，他会想：“看吧，我就是不行，被拒绝4次，好难受。”
2.  **在“损失区”的忍耐力不同**：你愿意在挫折中待多久？越不急于立刻摆脱困境，越能积蓄力量，把握翻盘机会。
    *   **反面例子**：失恋后无法忍受孤独，急于找新女友填补空虚，这样找到合适的人或挽回的概率极低。
    *   **正面例子**：失恋后冷静反思，总结教训，提升自我，不急于开始新恋情，反而会大大提高下一段感情的质量。

非酋模式的人会抱怨运气不好、底牌差（如没有高富帅的条件），并寻找现实中无法解决的客观理由（如“等我赚够100万才行”）来当挡箭牌，证明自己做不到。这是一种“受害者心态”。

## 正向循环与恶性循环

基于不同的选择与心态，会形成两种截然不同的循环。

**欧皇的正向循环**：
自信 -> 有动力处理问题，冷静思考学习 -> 掌握恋爱能力 -> 获得女生正向反馈（如变帅、会聊天） -> 进一步促进自信 -> 行动力、创造力更强 -> 更有能力面对挫折 -> 脱离损失区后能力变得更强。
`公式：自信 → 积极行动 → 能力提升 → 正向反馈 → 更强自信 (良性循环)`

**非酋的恶性循环**：
遇到挫折 -> 痛苦、颓废、只想挽回不想提升 -> 无精打采、稀缺心态、匮乏感 -> 自我否定、怀疑 -> 认为自己无吸引力 -> 没有勇气面对女生 -> 用糟糕预期相处 -> 被拒绝 -> 更加颓废。
`公式：退缩 → 消极情绪 → 能力停滞/下降 → 负面反馈 → 更加不自信 (恶性循环)`

如果没有外力打破，处于恶性循环的人可能沦为彻底的“受害者”，每天都在抱怨外界。

这两个循环的关键变量是**自信**。自信让你积极改善自我，而不自信则将你拖入深渊。

关于自信与自卑，有两句有意思的话：
*   **自卑的人总是正确的**：因为他们总能找到“证明”自己不行、外界有问题的“正确”理由。
*   **自信的人总能翻盘**：因为他们关注成长和方法，相信问题可以解决。

例如在搭讪时：
*   **自信者**会想：“被拒绝可能是方法问题，我下次可以做得更好。”
*   **不自信者**会想：“别人行，我就不行，果然是我这个人有问题。”
