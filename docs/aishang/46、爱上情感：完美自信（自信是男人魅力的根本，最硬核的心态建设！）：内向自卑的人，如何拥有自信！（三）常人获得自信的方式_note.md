# 46、爱上情感：完美自信（自信是男人魅力的根本，最硬核的心态建设！）：内向自卑的人，如何拥有自信！（三）常人获得自信的方式

🎼 本节课我们将探讨内向自卑的人如何获得自信，重点介绍常人获取自信的四种方式。

### 一、提升自己的世俗价值

**核心概念**： 通过提升自己的世俗价值，如财富、外貌、技能等，来获得自信。

**具体方法**：

* **形象改造**： 换新衣服、改变发型等，提升外在形象。
* **提升技能**： 学习新技能或提升现有技能，如语言、音乐、运动等。
* **增加财富**： 通过工作、投资等方式增加收入。

### 二、通过小团体抱团取暖

**核心概念**： 通过加入小团体，与他人建立联系，获得归属感和安全感。

**具体方法**：

* **结交朋友**： 扩大社交圈，结识新朋友。
* **加入社团或组织**： 参与集体活动，与他人共同进步。
* **寻求支持**： 在困难时向朋友或家人寻求帮助。

### 三、依赖某种特长或技能

**核心概念**： 通过在某个领域拥有特长或技能，获得成就感和自信。

**具体方法**：

* **发展爱好**： 选择自己感兴趣的领域，如音乐、绘画、运动等。
* **学习新技能**： 报名参加培训课程，学习新技能。
* **展示才华**： 在朋友或公众面前展示自己的特长。

### 四、依赖特定的场合中的特定角色

**核心概念**： 在特定的场合或角色中，获得自信和存在感。

**具体方法**：

* **参与活动**： 参加自己擅长的活动，如演讲、比赛等。
* **担任领导角色**： 在团队中担任领导角色，发挥领导力。
* **成为专家**： 在某个领域成为专家，获得认可。

### 总结

本节课介绍了常人获取自信的四种方式，包括提升世俗价值、小团体抱团取暖、依赖特长或技能、依赖特定场合中的特定角色。通过学习和实践这些方法，内向自卑的人可以逐步建立自信，成为一个更加自信和魅力的人。
