# 52、爱上情感：自然魅力（自然流入门）：搭讪总是失败、被拒绝？改变搭讪结果的神奇奥秘！

🎼 在本节课中，我们将探讨如何通过改变心态来提升搭讪的成功率。很多人在搭讪时总是失败，被拒绝，原因可能就在于他们的心态。

### 美好预期与糟糕预期

**公式**： 美好预期 → 积极心态 → 良好结果
**公式**： 糟糕预期 → 负面心态 → 恶劣结果

**以下是关于美好预期与糟糕预期的要点**：

- **美好预期**： 在搭讪前，抱有积极的想法，相信自己会获得好的结果。
- **糟糕预期**： 在搭讪前，抱有消极的想法，担心被拒绝或受到负面反馈。

### 自我预言的实现

**案例**： 一位学员多次搭讪，但从未获得过任何女孩子的联系方式。原因在于他有一个自我预言：他接近任何女孩子都不会得到号码。

### 社交的三个层面

1. **文字表面**： 说什么。
2. **表现层面**： 怎么说，肢体语言，表情。
3. **意图层面**： 想要什么，动机。

**以下是关于社交三个层面的要点**：

- 女生不会只看你的文字表面，更会关注你的意图和表现。
- 如果你怀揣着糟糕预期，女生会感觉到你的负面情绪，从而拒绝你。

### 改变预期，提升搭讪成功率

**方法**：

1. **降低预期**： 不要一开始就设定过高的目标，从小目标开始，逐步提升。
2. **积累正反馈**： 每次搭讪后，无论结果如何，都要给自己一些正面的反馈，增强自信心。
3. **突破舒适区**： 尝试在新的环境和情境下搭讪，挑战自己。

### 总结

本节课中，我们学习了如何通过改变心态来提升搭讪成功率。改变预期，积累正反馈，突破舒适区，都是提升搭讪成功率的关键。希望你在今后的搭讪中，能够运用这些方法，获得更好的结果。
