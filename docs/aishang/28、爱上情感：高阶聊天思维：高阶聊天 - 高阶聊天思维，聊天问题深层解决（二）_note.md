# 28、爱上情感：高阶聊天思维：高阶聊天 - 高阶聊天思维，聊天问题深层解决（二）

**概述**： 本节课我们将探讨高阶聊天思维中的两个重要转变：从稀缺心态到丰富形态，以及从仰望姿态到男神姿态。

### 一、从稀缺心态到丰富形态的转变

**核心概念**： 稀缺心态 → 丰富形态

1. **稀缺心态**： 过分关注眼前目标，忽视自身成长，导致焦虑和患得患失。
2. **丰富形态**： 关注自身成长，提升恋爱技能，以更成熟的心态面对感情。

**示例**： 
- **稀缺心态**： “我只有这一次机会，如果聊砸了，就要单身很多年。”
- **丰富形态**： “这次机会很重要，但我也会努力提升自己，为未来做好准备。”

### 二、从仰望姿态到男神姿态的转变

**核心概念**： 仰望姿态 → 男神姿态

1. **仰望姿态**： 将女生视为女神，过度讨好，导致失去自我。
2. **男神姿态**： 保持自信，尊重女生，以平等的姿态交流。

**示例**：
- **仰望姿态**： “你这么漂亮，我都不敢跟你说话。”
- **男神姿态**： “很高兴认识你，我们可以聊聊彼此的兴趣爱好。”

### 三、从关注文字表面到关注潜在沟通的变化

**核心概念**： 关注文字表面 → 关注潜在沟通

1. **关注文字表面**： 只看文字内容，忽视语境和情绪。
2. **关注潜在沟通**： 结合上下文和情绪，理解对方真实意图。

**示例**：
- **关注文字表面**： “你真讨厌。”
- **关注潜在沟通**： 根据语气、表情和情境，判断对方是开玩笑还是真的生气。
