# 27、爱上情感：读懂女人心（一开口，就让女人知道你懂她）：如何判断女生真的是你女朋友吗？如何成为你的女友!

## 概述
在本节课中，我们将学习如何判断女生是否真的是你的女朋友，以及如何成为她的男友。

## 名义女朋友与实质女朋友
### 名义女朋友
**定义**：与女生聊天或表白，对方未拒绝但未答应，就认为她是你的女朋友。

**行为表现**：
- 朋友间的肢体接触，如牵手、拥抱等。
- 礼貌性的聊天，如询问工作、学习等。

### 实质女朋友
**定义**：与女生发生亲密行为，做情侣间该做的事情。

**行为表现**：
- 亲密行为，如亲吻、拥抱等。
- 想要腻在一起，主动找对方。

## 名义女朋友的来源
1. 与女生见面次数少，聊天后认为她是女朋友。
2. 聊天内容多为第三方话题，如学习、工作等。
3. 女生给予普通信号，如微笑、打招呼等，就认为她喜欢你。
4. 约会时像朋友一样，没有亲密行为。

## 吸引与魅力价值
### 吸引
**公式**：价值产生吸引，吸引产生感觉，感觉等于喜欢。

### 魅力价值
**定义**：男人身上吸引女人的价值，如支配力、底气、肯定感等。

### 行为线索
**定义**：女人从男人身上看到的吸引她的行为，如支配力、底气、肯定感等。

## 广义价值与狭义价值
### 广义价值
**定义**：包括魅力价值和长期价值，如金钱、颜值、帮忙等。

### 狭义价值
**定义**：魅力价值，即行为线索。

## 情人与供养者
### 情人
**定义**：拥有魅力价值，女人会自然喜欢他们。

### 供养者
**定义**：拥有备胎价值，把女人当成女神，需要给女人买礼物、表白等。

## 总结
本节课中，我们学习了如何判断女生是否真的是你的女朋友，以及如何成为她的男友。要吸引女生，你需要拥有魅力价值，并学会正确地与她聊天、约会。
