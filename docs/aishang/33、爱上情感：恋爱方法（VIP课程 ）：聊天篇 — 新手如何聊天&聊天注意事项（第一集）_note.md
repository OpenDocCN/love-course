# 爱上情感：恋爱方法（VIP课程）——聊天篇：新手如何聊天&聊天注意事项（第一集）

🎼大家好，欢迎来到练爱方法5.0。我是乐天。本节课我们将探讨新手如何进行聊天以及聊天时应注意的事项。

### 不会聊天的原因

以下是导致不会聊天的两个主要原因：

1. **原始吸引力不足**：如果你的朋友圈展示不佳，如全是转发、文字或模糊图片，女生可能不会对你产生兴趣。
2. **缺乏话题**：如果你生活单调，缺乏阅历，应多读书、看节目，学习导师的聊天记录。

### 聊天注意事项

以下是聊天时应注意的几个要点：

1. **心态**：不要过度讨好或害怕失去对方，应保持自信，将对方视为追求者。
2. **平衡**：在字数和时间上与对方保持平衡，避免一方过于主动。
3. **避免低价值行为**：不要为了讨好对方而说笑话或贬低自己。

### 聊天能力

以下是一些重要的聊天能力：

1. **主导话题**：引导对话方向，避免被对方带跑。
2. **目的性聊天**：确保每句话都有助于推进关系。
3. **扩展话题**：从对方的话题出发，扩展对话内容。
4. **重启话题**：在对方结束对话时，重新开启话题。
5. **切断话题**：在对话达到高潮时适时结束，避免过度聊天。
