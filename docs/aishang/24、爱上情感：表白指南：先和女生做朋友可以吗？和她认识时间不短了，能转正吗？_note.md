以下是导致这种想法的几个原因：

3. **传递了错误的标签**：传递了朋友、供养者或备胎的标签，而不是情人的标签。
4. **无法跳出友谊区**：一旦进入友谊区，很难再跳出来。

1. **传递正确的潜沟通**：一开始就传递出男人对女人的感觉，而不是朋友的感觉。

4. **敢于拉升关系**：在聊天和约会中，敢于升级关系。
