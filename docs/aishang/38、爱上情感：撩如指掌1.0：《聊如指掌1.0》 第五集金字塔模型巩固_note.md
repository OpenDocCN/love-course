# 38、爱上情感：撩如指掌1.0：《聊如指掌1.0》 第五集金字塔模型巩固

## 概述
在本节课中，我们将继续巩固昨天所学的聊天金字塔模型，这是一个原创的聊天框架，有助于提升聊天效果。

## 金字塔模型介绍
金字塔模型分为五层，每一层代表聊天关系的不同阶段。

### 第一层：陌生
- **定义**：你和对方刚刚认识，处于陌生阶段。
- **话题**：通常涉及自我介绍、问候等。

### 第二层：熟悉
- **定义**：双方已经熟悉，可以像朋友一样聊天。
- **话题**：询问对方年龄、工作、兴趣爱好等。

### 第三层：暧昧
- **定义**：关系进一步发展，可以聊一些暧昧话题。
- **话题**：表达关心、暗示等。

### 第四层：情人
- **定义**：关系更加亲密，可以聊一些私密话题。
- **话题**：表达爱意、分享私密想法等。

### 第五层：确立关系
- **定义**：关系已经确立，可以深入交流。
- **话题**：探讨未来、规划生活等。

## 金字塔模型应用案例
以下是一个聊天案例，展示了如何应用金字塔模型：

1. **陌生阶段**：女孩发来英文，你回复“你群发？”
2. **熟悉阶段**：询问“你是济南的吗？”
3. **暧昧阶段**：表达关心，如“原来济南还有你这个小可爱？”
4. **情人阶段**：暗示关系，如“我们适合在一起吗？”
5. **确立关系**：深入交流，如“你单身是因为一般人可能你看不上。”

## 总结
本节课中，我们学习了聊天金字塔模型，并分析了如何在不同阶段运用该模型。通过掌握金字塔模型，你可以更好地提升聊天效果，建立更亲密的关系。
