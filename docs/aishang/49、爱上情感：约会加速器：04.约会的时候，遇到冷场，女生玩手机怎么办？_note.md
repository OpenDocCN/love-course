# 49、爱上情感：约会加速器：04.约会的时候，遇到冷场，女生玩手机怎么办？_note

在本节课中，我们将学习如何在约会中应对冷场，以及当女生一直玩手机时，应该如何妥善处理。掌握这些技巧，能让你的约会过程更加顺畅自然。

---

约会中出现冷场是正常现象，不必因此感到紧张或焦虑。适当的冷场有助于判断女生对你的兴趣程度。如果女生对你感兴趣，她通常会主动寻找话题。

在约会时，如果女生总是低头玩手机，通常意味着她对本次约会不感兴趣。或者，因为冷场出现，女生感到无措而拿起手机。此时，你只需要化解冷场即可。

上一节我们介绍了冷场的普遍性，本节中我们来看看如何避免和处理冷场。

以下是避免约会冷场的几个技巧：

1.  **主导约会进程**：约会时不能茫然不知所措。如果你不知道做什么，就容易造成冷场。你需要主导约会，做你想做的，聊你想聊的。约会时适当的冷场也没有关系。
2.  **重视破冰与场所选择**：约会冷场不一定是你聊天的问题，更多时候是约会初期的“破冰”没有处理好，导致气氛尴尬。如果你不是一个健谈的人，约会场所的选择就非常重要。例如，如果一开始就约在咖啡厅，双方除了喝茶就得说话，一旦无话可说就会冷场。
3.  **选择互动性强的约会场所**：选择一些即使彼此不说话，也不会尴尬的场所。以下是几个推荐：
    *   一起抓娃娃
    *   一起看电影
    *   一起玩蹦床
    *   一起开卡丁车
    在这些场所互动熟悉后，再坐下来聊天，效果会更好。
4.  **学会及时转场**：不要在一个约会场所停留太长时间。适时转换场景可以带来新鲜感，避免无聊和冷场。

在约会冷场时，千万记住不要做以下这几件事情：

*   **不要刻意寻找话题**：例如生硬地问“你在玩什么呀？”
*   **不要夺命连环问**：连续追问对方的工作、身高、家庭住址等私人问题，会给对方带来压力。
*   **你自己不要总是看手机**：这会加剧冷场，显得你也不投入。

---

了解了需要避免的事项后，下面我们来看看几个化解约会冷场的小技巧。

**技巧一：直面尴尬，用游戏破冰**
你可以直接戳破尴尬的局面。例如，对女生说：“我们现在是不是冷场了？我觉得是。那我带你玩个游戏吧——拇指大战。”接着解释规则：“谁把对方的拇指压在下面就算赢，输了的人请看电影。”通过一个小游戏，能自然打破僵局，并创造新的约会环节。

**技巧二：应对女生玩手机的情况**
如果女生一直玩手机，你可以直接但有趣地提出：“我们别玩手机了。这样吧，如果现在谁有消息进来，就可以指定对方做一件事。”用游戏化的规则来约束手机使用，让约会保持互动。

当女生在约会中一直玩手机，通常有两种可能：一是她真的有事；二是她感到非常无聊。对应有两种解决方案：
1.  采用上述技巧中的互动游戏来吸引她的注意力。
2.  及时转场，前往下一个约会地点，用新的环境刺激兴趣。

---

**总结**
本节课中我们一起学习了约会冷场的处理方法。记住，约会时冷场一点都不可怕，陌生人在一起出现冷场是正常的。熟悉的朋友即使不说话也不会尴尬。当女生一直低头玩手机时，你需要先判断她是真的有事还是对你没兴趣，再运用课程中的技巧灵活应对。
