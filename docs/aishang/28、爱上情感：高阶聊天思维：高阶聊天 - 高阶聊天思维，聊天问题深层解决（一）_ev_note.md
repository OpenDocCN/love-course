# 28、爱上情感：高阶聊天思维：高阶聊天 - 高阶聊天思维，聊天问题深层解决（一）_ev

## 概述

在本节课中，我们将学习高阶聊天思维，特别是从索取价值到提供价值思维转变的重要性，以及如何从漫无目的式聊天转变为主导话题，并探讨从逻辑研讨到情绪刺激的转变。

## 从索取价值到提供价值思维转变

**核心概念**：

* **价值**：一切有利于生存和繁衍的东西。
* **提供价值**：展示或提供对对方有益的东西。

**步骤**：

1. **识别价值**：认识到价值可以是情绪、知识、经验等。
2. **展示价值**：通过行为、言语或行动展示你的价值。
3. **避免索取**：不要总是想着从对方身上获得什么，而是思考你能提供什么。

## 从漫无目的式聊天到主导话题

**核心概念**：

* **主导话题**：控制对话的方向和内容。
* **兴趣指标**：判断对方的兴趣和反应。

**步骤**：

1. **判断兴趣指标**：观察对方的反应，了解其兴趣。
2. **决定话题**：根据对方的兴趣选择合适的话题。
3. **控制节奏**：决定何时开始、何时结束话题。

## 从逻辑研讨到情绪刺激

**核心概念**：

* **情绪体验**：通过对话传递情绪，引发对方的共鸣。
* **短期情绪**：追求短暂的快乐，可能不利于长期发展。

**步骤**：

1. **关注情绪**：在对话中关注对方的情绪反应。
2. **传递情绪**：通过言语、行为或情境传递积极的情绪。
3. **避免逻辑**：不要陷入无休止的逻辑讨论，而是关注情绪体验。

## 总结

本节课中，我们学习了从索取价值到提供价值思维转变、从漫无目的式聊天到主导话题的转变，以及从逻辑研讨到情绪刺激的转变。这些高阶聊天思维将帮助你更好地与对方沟通，建立更深入的联系。
