## 概述
在本节课中，我们将学习如何拍摄出高质量的照片，包括设备选择、光线运用、构图技巧等。

## 设备选择
以下是拍摄照片的常用设备：

- **手机和单反相机**：单反相机画质更清晰，效果更好，但相对较重不便携。手机则更便携，像素也足够满足基本需求。

**公式**：清晰度 = 设备像素 / 拍摄距离

## 光线运用
拍摄时，光线至关重要：

- **自然光**：白天利用自然光，晚上利用灯光。
- **光线方向**：避免直射光，寻找柔和的光线。

**代码**：
```python
def find_good_light(source_light):
    if source_light == 'natural':
        return 'daytime'
    elif source_light == 'artificial':
        return 'soft'
    else:
        return 'avoid'
```

## 构图技巧
构图和角度影响照片效果：

- **角度**：合理利用角度，可改变身高视觉效果。
- **构图**：多看优秀作品，模仿并练习。

**代码**：
```python
def compose_photo(angle, composition):
    if angle == 'low' and composition == 'wide':
        return 'flattering'
    else:
        return 'average'
```

## 照片焦点
焦点决定视觉重点：

- **自动对焦**：手机自动对焦功能。
- **手动对焦**：手动调整焦点。

**代码**：
```python
def focus_photo(auto_focus, manual_focus):
    if auto_focus:
        return 'auto'
    else:
        return 'manual'
```

## 照片背景
背景影响照片整体效果：

- **背景选择**：干净整洁的背景，如咖啡厅或餐厅。

## 拍摄技巧
以下是一些拍摄技巧：

- **朋友帮忙**：朋友可提供好的角度和拍摄效果。
- **后期处理**：后期处理可提升照片质量。
