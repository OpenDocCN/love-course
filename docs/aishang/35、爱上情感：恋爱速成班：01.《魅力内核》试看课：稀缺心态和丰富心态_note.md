# 35、爱上情感：恋爱速成班：01.《魅力内核》试看课：稀缺心态和丰富心态

🎉欢迎来到魅力内核课程。今天我们将探讨稀缺心态和丰富心态。许多人在学习课程后进步缓慢，原因可能如下：

1. **急功近利**：不注重自身提升，总是急于求成。
2. **不敢实战**：容易退缩，缺乏实战经验。
3. **懒惰缺乏行动力**：喜欢幻想而不付诸行动。
4. **狙击或挽回心态**：总是想着狙击或挽回，而非积极发展。

**稀缺心态**：当你社交圈子狭窄，接触女生机会少，缺乏社交和恋爱经验时，你会觉得女生很神秘、珍贵，容易产生需求感，这种心态称为稀缺心态。

**丰富心态**：当你拥有丰富的社交资源，相信女生资源丰富，有能力去认识和发展感情时，你便拥有了丰富心态。

以下是稀缺心态和丰富心态的对比：

| **稀缺心态** | **丰富心态** |
| --- | --- |
| 社交圈子狭窄 | 社交圈子广泛 |
| 接触女生机会少 | 接触女生机会多 |
| 缺乏社交和恋爱经验 | 拥有丰富的社交和恋爱经验 |
| 需求感强 | 需求感低 |
| 焦虑 | 轻松自在 |
| 不自信 | 自信 |

**稀缺心态的危害**：

1. **反应模式**：在稀缺心态下，你容易陷入反应模式，无法做自己，行为下意识迎合女生。
2. **失去吸引力**：在稀缺心态下，你容易紧张、患得患失，甚至跪舔乞求，导致失去吸引力。
3. **沉默成本**：不断投入时间、精力、感情和金钱，却成为沉默成本。

**如何培养丰富心态**：

1. **收集资源**：通过社交软件、街头搭讪等方式，大量接触女生，积累经验。
2. **提升自己**：不断学习恋爱技巧和知识，提升社交能力。
3. **相信自己**：相信女生资源丰富，有能力去认识和发展感情。

**本节课总结**：

本节课我们学习了稀缺心态和丰富心态的区别，以及如何培养丰富心态。在接下来的课程中，我们将学习如何克服社交恐惧、拥有行动力、处理负面反馈，以及正确认识社交和恋爱，让你变得开心自信，从而更好地建立起丰富心态。
