## 美图秀秀修图教程

首先，我们以美图秀秀为例，教大家如何修图。

1. 打开美图秀秀，点击“人像美容”。
2. 选择自拍图片，打开它。
3. 点击“磨皮美白”，调整亮度至合适。
4. 点击“祛痘去斑”，自动或手动处理。
5. 点击“瘦脸瘦身”，调整脸型。
6. 点击“眼睛放大”和“亮眼”，使眼睛更有神。
7. 点击“去美化”，选择滤镜，调整图片质感。
8. 点击“保存”，将图片保存到相册。

## ins软件修图教程

接下来，我们使用ins软件修展示面图片。

1. 打开ins软件，选择展示面图片。
2. 点击“笔”工具，选择“戏剧效果”滤镜。
3. 点击“剪裁”，选择4比3比例。
4. 点击“保存”，将图片保存到相册。

## 加白边软件修图教程

最后，我们使用加白边软件修展示面图片。

1. 打开加白边软件，选择修好的展示面图片。
2. 调整图片位置，添加白边。
3. 点击“保存”，将图片保存到相册。
