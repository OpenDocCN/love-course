在本节课中，我们将要学习如何对自拍和展示面图片进行后期修图处理。修图是提升图片视觉效果、增强个人魅力的关键步骤。我们将介绍几款必备的修图软件，并一步步演示具体的操作流程。

## 必备修图软件介绍

上一节我们介绍了如何拍摄展示面，本节中我们来看看如何通过修图让图片效果更佳。首先，你需要准备以下四款手机修图软件。

以下是四款必不可少的修图软件：
*   **美图秀秀**
*   **Instagram (Ins)**
*   **PicsArt**
*   **Max**

## 自拍图片修图教程

下面，我们以一张自拍图片为例，开始第一步的修图工作。我们将使用美图秀秀进行基础的人像美化。

### 第一步：使用美图秀秀进行基础美化

打开美图秀秀，选择“人像美容”功能，并导入你的自拍图片。

以下是基础美化的具体步骤：
1.  **磨皮美白**：点击“磨皮美白”。系统会自动处理。建议将“磨皮”强度调整至**10**左右，“美白”强度调整至**10**以内（如**7**或**8**）。完成后点击确认。
2.  **祛痘祛斑**：点击“祛痘祛斑”。先点击“自动”让系统处理。若仍有瑕疵，点击“手动”，放大图片后仔细点除剩余痘痘或斑点。完成后点击确认。
3.  **瘦脸瘦身**：点击“瘦脸瘦身”。选择“自动”模式让系统微调脸型，使脸部看起来更瘦。完成后点击确认。
4.  **眼睛放大**：点击“眼睛放大”。选择“自动”模式让眼睛显得更大。完成后点击确认。
5.  **亮眼**：点击“亮眼”。选择“自动”模式让眼睛更有神采。完成后点击确认。

完成以上步骤后，初步的人像修图就结束了。接下来，我们需要为图片添加滤镜，提升整体质感。

### 第二步：添加滤镜

在美图秀秀中，点击“去美化”，然后选择“特效”或“滤镜”选项。

以下是添加滤镜的方法：
*   在滤镜列表中逐一尝试，选择最适合当前自拍图片的一款。
*   例如，“壁波”滤镜可能是一个不错的选择，它能增强皮肤质感，让图片整体效果更佳。
*   选定合适的滤镜后，点击确认。

至此，图片处理已完成约50%。点击“保存”，将这张初步修好的自拍图片保存到相册。

### 第三步：使用Instagram添加白边

初步修图完成后，我们使用第二款软件Instagram来优化图片构图。打开Instagram，导入刚才保存的自拍图片。

以下是添加白边的步骤：
1.  通过双指捏合手势，将图片缩小。
2.  调整图片在画布中的位置，例如将其置于右侧角落。
3.  此时图片周围会出现白色边框，形成更具设计感的构图。
4.  点击保存，将最终版自拍图片保存至相册。

自拍图片的修图流程到此结束。你可以反复练习并探索这些软件的其他功能。

## 展示面图片修图教程

接下来，我们学习如何修整展示面图片。我们将使用PicsArt和Instagram两款软件。

### 第一步：使用PicsArt添加滤镜与裁剪

打开PicsArt，导入你需要修整的展示面图片。

以下是PicsArt的处理步骤：
1.  **添加滤镜**：点击底部工具栏的“效果”。在众多滤镜中，“戏剧效果”是一个常用且能快速改变图片氛围的选择。尝试应用它，并对比效果。
2.  **裁剪图片**：点击“裁剪”工具。为了突出人物主体，建议选择**4:3**等比例进行裁剪，使构图更紧凑。完成后点击确认。
3.  **保存图片**：点击保存，将处理好的图片存至相册。

### 第二步：使用Instagram添加白边

与处理自拍图片一样，我们再次打开Instagram，导入刚才用PicsArt修好的展示面图片。

重复添加白边的操作：
*   缩小图片，调整位置，为其添加白色边框。
*   点击保存，获得最终版的展示面图片。

## 最终效果与总结

经过以上步骤，我们得到了最终修图后的自拍与展示面图片。对比原图，修图后的图片在质感、美观度和个人魅力展现上都有显著提升。
