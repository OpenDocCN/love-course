# 38、爱上情感：撩如指掌1.0：《聊如指掌1.0》 第七集：通过表情判断兴趣

## 概述
在本节课中，我们将学习如何通过微信表情来判断对方的兴趣。

## 表情含义解析
以下是微信常用表情的绿灯和红灯分类：

### 绿灯表情
- 😊：开心
- 😂：大笑
- 😍：喜欢
- 😘：亲吻
- 😁：偷笑
- 😛：调皮
- 😜：眨眼
- 😎：酷
- 😏：坏笑
- 😐：平淡

### 红灯表情
- 😠：愤怒
- 😡：生气
- 😩：哭泣
- 😖：失望
- 😞：难过
- 😟：困惑
- 😐：不悦

### 黄灯表情
- 😐：平淡
- 😑：不悦
- 😕：困惑
- 😖：失望
- 😞：难过

## 表情包判断
表情包通常被视为绿灯，除非内容过于激烈。

## 表情测试
某些表情，如害羞的表情，可能具有双重含义，需要根据聊天上下文来判断。

## 总结
本节课中我们一起学习了如何通过微信表情来判断对方的兴趣。通过识别表情的绿灯、红灯和黄灯含义，我们可以更好地理解对方的情绪和兴趣。
