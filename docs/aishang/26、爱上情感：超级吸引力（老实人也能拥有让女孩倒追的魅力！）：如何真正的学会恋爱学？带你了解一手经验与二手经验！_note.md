## 一手经验与二手经验
### 一手经验
一手经验是指个人亲身经历和体验获得的经验。例如，亲自去约会、表白等。

**公式**：一手经验 = 亲身经历 + 体验

### 二手经验
二手经验是指从他人或媒体那里获得的经验。例如，听别人讲述他们的恋爱经历，或阅读恋爱技巧书籍。

**公式**：二手经验 = 他人的经验 + 媒体信息

## 参考经验库
参考经验库是指个人积累的恋爱经验，包括成功和失败的经历。

**代码**：
```python
class ReferenceExperience:
    def __init__(self):
        self.experiences = []

    def add_experience(self, experience):
        self.experiences.append(experience)

    def get_experience(self):
        return self.experiences
```

## 建立一手经验
### 1. 主动尝试
主动尝试新的恋爱方式，如约会、表白等。

### 2. 学习与实践
学习恋爱技巧，并将其应用于实际生活中。

### 3. 反思与总结
在实践过程中，反思自己的行为和结果，总结经验教训。
