# 38、爱上情感：撩如指掌1.0：《聊如指掌1.0》 第三集 聊天吸引力模型_ev

## 概述
在本节课中，我们将学习如何通过聊天建立吸引力，提升聊天技巧。

## 聊天的作用
聊天的作用主要分为两部分：
1. **展现自己**：通过聊天展示自己的个性和价值，吸引对方。
2. **拉升关系**：在吸引对方的基础上，不断升级关系，达到亲密的目的。

## 聊天的目的
聊天的最终目的是将对方约出来，实现现实中的互动。

## 展现自己的方法
以下是通过聊天展现自己的几个方面：

### 社会地位
- 展示自己的职业、收入等社会地位。
- 例如：通过聊天提及自己的工作、收入等。

### 收入
- 展示自己的消费能力、生活品质等。
- 例如：提及自己购买的高档商品、享受的优质服务。

### 阅历
- 展示自己的经历、知识等。
- 例如：分享自己的旅行经历、阅读过的书籍等。

### 圈子
- 展示自己所在的社交圈子、朋友等。
- 例如：提及自己参加的社交活动、结识的朋友。

### 预选
- 展示自己身边吸引人的女性资源。
- 例如：提及自己认识的女性朋友、追求者等。

### 家庭
- 展示自己的家庭背景、家庭教育等。
- 例如：提及自己的父母、家庭氛围等。

### 兴趣爱好
- 展示自己的兴趣爱好、特长等。
- 例如：提及自己喜欢运动、擅长烹饪等。

### 价值观
- 展示自己的价值观、人生观等。
- 例如：分享自己的观点、信仰等。

## 总结
本节课中，我们学习了如何通过聊天建立吸引力，提升聊天技巧。希望大家能够将所学知识应用到实际生活中，提升自己的社交能力。
