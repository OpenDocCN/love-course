## 第一部分：什么是清晰的界限感？🚧

上一节我们概述了课程内容，本节中我们来看看“态度”的具体表现——界限感。所谓界限感，就是你的个人标准、原则和底线。它可以分解为以下四个核心问题，用于审视自己的行为准则：

以下是界定个人界限感的四个维度：

1.  **我自己做什么样的行为，我可以接受。**
    *   **示例**：作为一个男生，我可以接受自己化妆、打扮时尚、修眉。我可以接受自己请女生吃西餐或小龙虾。

2.  **我自己做什么样的行为，我不能接受。**
    *   **示例**：我不能接受穿着脏衣服出门。我不能接受女生要求我发红包时就照做。

3.  **别人对我做什么样的行为，我可以接受。**
    *   **示例**：我可以接受长辈在饭桌上给我夹菜。我可以接受朋友开一些不过火的玩笑。

4.  **别人对我做什么样的行为，我不能接受。**
    *   **示例**：我不能接受别人开很过火的玩笑。我不能接受女生要求我购买超出我经济能力的礼物（如手机、名牌包）。我不能接受女生随意查看我的手机。

**核心公式：清晰的个人界限感 = 明确的接受清单 + 明确的拒绝清单**

如果你对以上四点没有清晰的答案，意味着你缺乏个人原则和标准。你可能会变成一个“怎么都行”的人，像橡皮泥一样任人揉捏，这在恋爱学中就是“没有态度”的表现。

## 第三部分：“跪舔”行为的根源 🕳️

为什么你会陷入“跪舔”模式？主要根源有两个。

以下是导致“跪舔”的两个核心原因：
