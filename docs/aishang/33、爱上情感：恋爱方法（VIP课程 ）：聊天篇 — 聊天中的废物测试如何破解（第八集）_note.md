# 恋爱方法VIP课程：聊天篇 — 聊天中的废物测试如何破解（第八集）

🎼大家好，欢迎来到恋爱方法5.0，我是乐天。今天我们将探讨如何应对女生在聊天中的废物测试。所谓废物测试，是指女生通过提问来测试你对她的兴趣和诚意。以下是三种应对方法。

### 应对废物测试的三种方法

1. **忽略**：当女生问你“你是不是想约炮呀？”时，你可以转移话题，比如：“哎，我都肚子疼了，你怎么也不关心我一下？”
2. **推回**：同样的问题，你可以回答：“你觉得呢？你猜。”将问题推回给女生。
3. **保持自我**：如果你确实想约女生，直接回答“是的”，如果你不想，直接说“不是”。注意，使用这种方法时，回答要简洁，保持真实自我，不要解释。

### 实战案例

🎼以下是一些实战案例：

- 女生问：“你是不是想跟我约炮呀？”
  - 回答：“你觉得我是想跟你约炮吗？”

- 女生问：“你是不是跟每个女人都这样去说呀？”
  - 回答：“你觉得你跟其他的女人都一样吗？”

这些回答都是通过推回方法来应对废物测试。

### 新手建议

对于新手来说，建议使用推回方法来应对废物测试。以下是一些万能的推回方法：

- “你觉得呢？”
- “你猜”
- “对吧？”
