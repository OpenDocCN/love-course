# 恋爱方法 33：高阶篇 — 把妹中千万不能做的行为（第一集）🚫

在本节课中，我们将要学习在追求心仪对象时，必须严格避免的几种错误行为。理解这些禁忌，能帮助你避免不必要的失误，提升交往的成功率。

上一节我们介绍了课程概述，本节中我们来看看具体哪些行为是“雷区”。

## 一、避免说教对方

很多新手喜欢用自认为正确的观念去教育对方，并且认为这是为了对方好。姑且不论对方是否需要，这种态度本身就是错误的。下次再遇到你认为错误的事情时，不妨先反思自己：你没有自己想象的那么厉害，对方也没有你想象的那么傻。

## 二、避免过度装逼

有些人在追求女生时，最初可能处处讨好（跪舔）。在接触恋爱学后，他们知道了“框架”的概念，却误解了其含义，认为就是要让自己拽起来，态度要强硬。这种过度装逼的行为，经常导致被女生拉黑。

## 三、避免态度猥琐

长期缺乏与异性接触的人，可能对任何与女性相关的信号都过度敏感。一旦获得与女生接触的机会，便迫不及待地将话题引向性方面，但又害怕引起反感，于是表现得遮遮掩掩。这种想撩又不敢撩的态度，恰恰暴露了你的猥琐。

## 四、避免自我设限（自我射线）

有时问题不在于对方对你不好，而在于你总把对方往最坏的方向去想。有句话叫：**一个人想他是什么样子，他便是什么样子**。这不仅适用于把妹，也适用于所有方面。因为自卑，觉得自己得不到对方，所以把对方想得很坏，这是典型的“吃不到葡萄说葡萄酸”，是面对高分对象时常见的心理。

## 五、避免过早表白或邀约

这是常见的错误。一上来就去邀约或表白，即使对方颜值普通，也大概率会被无视。过早暴露需求感会让对方感到压力。

## 六、避免跪舔

跪舔当然也有可能成功，但这是一种备受诟病的择偶方式，且成功率极低，属于小概率事件。请停止你跪舔的行为。

## 七、避免查户口式聊天

一问一答的对话模式，俗称为“查户口”。在恋爱学流行后，这种现象已减少。聊天无法推进，大多不是内容问题，而是心态问题。因为你太在乎得失，所以不敢主动与女生聊天，也不敢寻找话题。

## 八、避免过分炫耀虚假价值

随着恋爱学流行，很多人开始建设展示面。但有些人喜欢展示虚假的高价值（DHV），跟女孩聊天时总是吹嘘自己多么厉害。虚假的高价值展示只能吸引爱慕虚荣的女孩，真正有价值的女生一眼就能看破。

以上就是在把妹过程中不可以做的一些行为。了解禁忌后，我们来看看哪些正面行为可以为你加分。

## 九、善用技巧与方法

只要我们使用技巧和方法时张弛有度，就能在互动中加分。例如，面对女生的“废物测试”：
*   **女生问**：“我是你的第几个女人？”
*   **普通回答**：“你猜。” （仅仅忽略了测试，未最大化拉近关系）
*   **高阶解法**：“你表现好点，可能成为最后一个。” （运用了逆向思维，破解测试的同时引导女生追逐，并拉近了关系）

这就是学好技巧后，可以为你加分的行为。

## 十、巧妙营造环境

环境能极大影响成功率。合理利用环境，可以为你加分。

以下是几个例子：
*   在头等舱搭讪的成功率，通常高于在经济舱。
*   在高端酒店电梯里搭讪女生，成功率高于在大街上。
*   如果你是医生，在医院环境中搭讪，会优于在其他环境。

本节课中我们一起学习了在追求过程中必须避免的八大错误行为：说教、过度装逼、态度猥琐、自我设限、过早表白、跪舔、查户口式聊天以及炫耀虚假价值。同时，我们也探讨了两个能为你加分的正面行为：善用沟通技巧和巧妙营造有利环境。理解并规避这些禁忌，运用正确的方法，能让你在情感道路上更加顺利。我们下期再见。
