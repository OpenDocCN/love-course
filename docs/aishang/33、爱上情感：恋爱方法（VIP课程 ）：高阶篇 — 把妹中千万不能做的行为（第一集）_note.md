# 恋爱方法：高阶篇 — 把妹中千万不能做的行为（第一集）

🎼大家好，我是老天，欢迎来到恋爱方法。本期我们将探讨在追求女生过程中，一些绝对不能出现的行为。

### 1. 教育对方
很多新手喜欢用自己认为正确的观念去教育对方，并认为这是为了对方。但即使对方不需要你的态度，这种做法也是错误的。下次遇到你觉得是错误的事情时，先反思自己，不要高估自己，也不要低估对方。

### 2. 过度装逼
屌丝在追求女生时，往往没有一个正确的态度，处处跪舔女生。但在接触恋爱学后，误以为框架就是要让自己拽起来，态度强硬。这种做法很容易被女生拉黑。

### 3. 猥琐行为
屌丝男渴望很久后，看到路上一个红色的塑料袋都可以联想到女生的内裤。好不容易有机会接触女生，却迫不及待地想扯到这方面，但又怕女生反感，遮遮掩掩。这种猥琐的态度会暴露出来。

### 4. 自我射线
有时候不是对方对你不好，而是你总把对方往最坏的方向去想。有一句话：“一个人一想他是什么样子，他便是什么样子。”这句话不仅适用于把妹，适用于所有方面。因为自卑，觉得自己得不到对方，所以把对方往最坏的方向去想，典型的吃不到葡萄说不到酸。

### 5. 表白错误
屌丝常见的错误是一上来就邀约或表白，即使对方颜值低。一上来就表白也会被无视。

### 6. 跪舔
跪舔当然也可以把倒妹。但现在的人择偶方式都让人诟病，屌丝们也别再说自己一点机会都没有了。但跪舔把妹是小概率事件，请停止跪舔的行为。

### 7. 茶不口
一问一答的对话模式，俗称为茶不口。这种现象在恋爱学流行起来后已经很少见了。大多数情况下，不是聊天内容的原因，而是心态的原因。因为你太在乎得失，所以不敢跟女生聊天，也不敢找话题。

### 8. 过分炫耀
现在连演学流行起来，很多人开始建设自己的展示面，都喜欢把一些虚假的高价值展示面放在上面，跟女孩聊天时也说自己多么厉害。但虚假的高价值展示只能吸引那些爱慕虚荣的女孩，真正有价值的女生一眼就能看破。
