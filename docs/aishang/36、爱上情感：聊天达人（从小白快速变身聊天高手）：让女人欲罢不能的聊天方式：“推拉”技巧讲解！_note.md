**推拉**是一种在聊天或互动过程中，通过积极的语言或行为模式肯定对方，同时采取消极的语言或行为模式调侃、制造阻碍、挑战对方的技巧。

**公式**：

推拉 = 积极语言/行为 + 消极语言/行为
