# 33、爱上情感：恋爱方法（VIP课程 ）：展示面篇 — 如何自拍&自拍攻略（第三集）_note

在本节课中，我们将要学习如何通过高质量的自拍来提升个人展示面。我们将重点探讨自拍的场景选择、背景要求以及最关键的拍摄角度与技巧。

在上一集，我们介绍了适合拍摄展示面的各类场所。可能有朋友会问，如果周围没有朋友帮忙拍摄，该怎么办？其实，你也可以独自前往这些场所进行自拍。

下面我们来看一些自拍实例。

**第一张图片**拍摄于星巴克。在拍摄前，我整理了自己的形象，并手持星巴克咖啡杯。我故意让星巴克的logo露出约四分之一。这样无需文字说明，观看者也能识别出地点。

**第二张图片**是在家中的一张小摇椅上拍摄的，环境舒适宜人。

**第三张图片**是我穿着正装工作时拍摄的，背景是简洁的纯白色。

**第四张图片**是我与朋友在书店的一张自拍，背景具有美感。

**第五张图片**是我在吃早餐时拍摄的自拍。

以上这些自拍图片，我们可以总结出一些共同特点。

以下是自拍的核心要点：
1.  **形象管理**：拍摄前需整理个人形象。
2.  **背景选择**：背景必须优质，不能杂乱。背景需要看起来舒服、干净整洁，或略显高端。

这就是自拍时需要注意的基本事项。接下来，我们详细说明如何进行自拍。

我发现很多朋友自拍时喜欢将相机从下往上拍。这种方式会使脸显得特别大且宽，效果不佳。也有些朋友让手机与脸部平行拍摄，这样拍出的脸型也并非最佳。

那么正确的拍摄方法是什么呢？我们应该将手机举过头部，采用**由上而下的角度**进行拍摄。这样会使脸型显得更瘦、更小，拍出的自拍才好看。

**核心技巧公式：最佳自拍角度 = 手机位置 > 头部高度**

我们可以回顾之前的几张图片，都是采用由上而下的角度拍摄的。这样既能拍到环境，又能让脸型显瘦，这才是正确的自拍方式。

现在，我们来实际演示一下如何自拍。

我们拿起手机，采用由上而下的角度，并**选择一个最适合你的角度**。选择角度的目的是为了**扬长避短**。

例如，如果你的左脸比右脸好看，就应多拍摄左侧面，避免拍摄右侧面。我们可以将脸稍微偏向好看的一侧，然后由上而下拍摄。关键是要展现好看的一面，隐藏不好看的一面。

这是自拍时需要注意的核心事项。最重要的一点是：在由上而下找到适合的角度后，尽可能将脸拍得瘦一些。

以下是自拍操作步骤：
1.  将手机举过头部。
2.  调整手机，找到能最好展现你优势脸型的角度。
3.  保持自然表情，由上而下拍摄。

这样我们就完成了一张自拍图片。大家可以观看屏幕上的成片效果。这就是自拍场合需要注意的一些关键事项。

本节课中，我们一起学习了自拍的核心要点：选择优质简洁的背景，并掌握由上而下的拍摄角度来优化脸型。记住**扬长避短**的原则，你也能拍出出色的展示面自拍。
