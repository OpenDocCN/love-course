# 爱上情感：恋爱方法（VIP课程）- 展示面篇 — 如何自拍&自拍攻略（第三集）

🎼大家好，欢迎来到5.0，我是乐天。这一集我们将探讨如何进行自拍。在上一集中，我们介绍了拍摄展示面的场所。可能有些朋友表示周围没有朋友帮忙拍摄，那怎么办呢？当然，你也可以自己去这些场所自拍。接下来，我会展示一些我的自拍图片给大家参考。

**以下是自拍图片展示：**

1. **星巴克自拍**：背景是星巴克，我手持咖啡杯，露出星巴克logo的一部分，既展示了地点，又保持了隐私。
2. **家中摇椅自拍**：家中环境舒适，我坐在摇椅上自拍，背景干净整洁。
3. **正装工作自拍**：背景纯白色，展现专业形象。
4. **书店与朋友自拍**：背景是书店，与朋友互动，氛围温馨。
5. **早餐自拍**：背景是早餐，生活气息浓厚。

**总结自拍特点：**

1. **形象整洁**：每次自拍前都整理好形象。
2. **背景干净**：选择干净整洁或美观的背景。
3. **避免杂乱**：确保背景不是杂乱的。

**如何进行自拍：**

1. **避免从下往上拍摄**：这样会使脸部显得大且宽。
2. **从上往下拍摄**：将手机举过头顶，由上而下拍摄，使脸部显得瘦小。
3. **选择角度**：选择最适合你的角度，扬长避短。
4. **展现美好一面**：将好看的一面展现出来，隐藏不好看的一面。
