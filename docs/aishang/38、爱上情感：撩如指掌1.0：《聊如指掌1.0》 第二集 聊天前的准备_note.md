# 38、爱上情感：撩如指掌1.0：《聊如指掌1.0》 第二集 聊天前的准备

## 概述
在本节课中，我们将学习如何为聊天做好准备，提升个人吸引力。

## 核心概念
**吸引力公式**：形象 + 头像 + 朋友圈 + 资源 = 吸引力

### 第一部分：形象的重要性
**1. 好的形象**
- **重要性**：形象在聊天和约会过程中至关重要。
- **实践**：提升个人形象，包括服装、发型等。

### 第二部分：优质头像
**2. 优质头像**
- **重要性**：头像是聊天窗口的第一印象。
- **实践**：使用个人照片作为头像，避免非主流或模糊图片。

### 第三部分：优质朋友圈
**3. 优质朋友圈**
- **重要性**：朋友圈展示个人品味和生活。
- **实践**：发布高质量照片，如下午茶、餐厅、超市、工作场景等。

### 第四部分：收集资源
**4. 收集资源**
- **重要性**：拥有多个女性资源提升自信心。
- **实践**：在微信中至少保持10个女性资源。

## 具体操作
以下是具体操作步骤：

**1. 改造形象**
- **方法**：提升个人形象，包括服装、发型等。
- **资源**：可报名私教课程或参考相关教程。

**2. 更换头像**
- **方法**：使用个人照片作为头像。
- **注意事项**：避免使用非主流或模糊图片。

**3. 删除脏乱差朋友圈**
- **方法**：删除低质量朋友圈，发布高质量照片。
- **注意事项**：保持朋友圈精致，避免垃圾信息。

**4. 收集女性资源**
- **方法**：在微信中添加至少10个女性资源。
- **资源**：可通过社交活动、朋友介绍等方式获取。

## 总结
本节课中，我们学习了如何为聊天做好准备，提升个人吸引力。通过改造形象、更换头像、发布优质朋友圈和收集女性资源，我们可以更好地吸引他人，为后续的聊天打下良好基础。
