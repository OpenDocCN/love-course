# 课程38：爱上情感：撩如指掌1.0 🎯

## 第二集：聊天前的准备 📝

在本节课中，我们将要学习聊天前需要做的准备工作。这些准备关乎你个人是否具备吸引力，是成功开启对话的基础。我们将详细讲解四个核心要点，帮助你从认识女生的第一刻起就建立吸引力。

---

上一节我们介绍了课程的重要性，本节中我们来看看聊天前需要做的具体准备。

聊天准备需要准备以下四点。

以下是四点核心准备事项：

1.  **拥有一个好的形象。**
    这一点在昨天的课程中已经提及。一个好的形象至关重要。我们曾展示学员改造前后的对比。改造前形象可能只能打**1分**，而改造后形象则大幅提升。形象改进能直接带来信心，这是自信积累的过程。形象在整个互动过程中起到决定性作用。我们不要求穿着奢侈品牌，但要求形象必须过**合格线**，让对方觉得你值得了解。如果你的形象不佳，即使在线上聊得很好，线下约会时也可能“见光死”，导致前功尽弃。

2.  **设置一个优质的头像。**
    我发现很多学员的头像过于非主流或不合适。聊天窗口最直观的展示就是双方的头像。一个优质的头像非常重要。如果女生的头像很丑，你也不愿意和她聊天。因此，我们建议头像使用**你自己的清晰照片**，让女生知道在和谁交流。头像和聊天内容共同勾勒出你鲜活的形象，这是建立吸引的关键。请注意，拥有好形象（第一点）是拍出优质头像照片的前提。

3.  **拥有至少三组以上优质的朋友圈内容。**
    很多学员的朋友圈充斥着转发、微商广告或无意义的文字，这些内容无法提升吸引力。我们希望将你准备成一个鲜活的人，让对方在聊天前就能产生兴趣。你的朋友圈代表你的品味和生活状态。品味低，发出的内容就低端。你需要发布能展现积极生活状态的内容。

    以下是优质朋友圈内容的例子（可举一反三）：
    *   喝下午茶（约**100元**）。
    *   在音乐餐厅用餐（约**100元**，多人可AA）。
    *   逛超市（可**不花钱**，仅拍照展示生活）。
    *   工作环境展示。

    拥有至少三组这样的朋友圈，能让女生在聊天前就对你形成良好印象，觉得你是一个积极向上、懂得享受生活的人。

4.  **收集10个以上的女性资源。**
    这一点与昨天讲的“强大心态”相关。强大心态的第一条是“我不缺”。收集10个以上可以互动的女性资源，能让你真实地拥有这种心态，而不是空谈。当你的微信里有足够多的聊天对象时，你的内心才会真正变得强大。同时，做好前三点（形象、头像、朋友圈）也能在实践中不断积累自信，比如收到对方对你外形或生活的夸奖。

---

以上四点是对学员的最低要求。具体操作方法（如如何改造形象、拍摄照片）本课程无法详尽展开，因为每一块都需要大量时间讲解。大家可以自行尝试解决，或报名我们更高级的课程获得一对一指导。

接下来，我们提供一些拍摄高质量朋友圈的思路和场地建议。

以下是适合拍摄高质量朋友圈的场地思路（可通过“大众点评”等APP搜索和预览）：
*   **西餐厅**：通常装修有格调，价格不贵（可使用团购）。
*   **游泳馆**：适合夏季，体验不同生活，创造聊天话题。
*   **健身房**：建议选择环境较好的进行体验（通常有体验课）。
*   **高端办公室**：如果你有此类资源，可以加以利用。
*   **茶馆**：许多茶馆装修雅致，适合拍照。
*   **商场**：逛商场不花钱，内部常有适合拍照的背景墙。
*   **温泉**：适合冬季，可拍摄场景细节。
*   **其他任何高逼格场地**：关键在于体验生活，积累真实话题。

话题来源于生活。没有丰富的生活体验，就很难有话题与女生交流。以上场地体验都能为你提供自然的聊天切入点。

---

最后，我们通过对比来强调朋友圈质量的重要性。

左边是普通粉丝的朋友圈，内容杂乱、低价值，让人没有聊天的欲望。右边是导师的朋友圈，形象好、照片优质、内容精致。如果两人同时添加一个女生，且聊天水平相当，右边朋友圈的男生吸引女生主动开启话题并成功的几率**远大于左边**。这就是做好聊天准备的威力。
