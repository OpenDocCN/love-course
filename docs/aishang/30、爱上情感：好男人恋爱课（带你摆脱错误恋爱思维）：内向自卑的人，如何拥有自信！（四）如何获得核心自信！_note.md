# 30、爱上情感：好男人恋爱课（带你摆脱错误恋爱思维）：内向自卑的人，如何拥有自信！（四）如何获得核心自信！

**概述**： 本节课将探讨内向自卑的人如何获得核心自信，帮助大家摆脱错误恋爱思维，提升自我价值。

### 一、场景自信与核心自信

**1. 场景自信**

*   **定义**： 在特定场景下，因自身价值得到认可而获得的自信。
*   **特点**：
    *   在熟悉场景中，深知自己能提供价值。
    *   在熟悉场景中，获得特殊地位，无法在其他场景中获得。
*   **例子**： 在专业领域、熟悉场所、擅长领域等。

**2. 核心自信**

*   **定义**： 在任何场景下，都能感受到的自信。
*   **特点**： 不受特定场景限制，源于内在价值。
*   **例子**： 拥有核心自信的人，在任何场合都能自如应对。

### 二、场景自信的危害

*   **限制自我提升**： 陷入舒适区，不愿离开熟悉场景。
*   **阻碍成长**： 过度依赖场景自信，难以实现自我突破。

### 三、如何获得核心自信

*   **摆脱场景自信**： 逐步剔除依赖，培养内在价值。
*   **提升自我认知**： 了解自身优势，明确目标。
*   **持续学习**： 不断充实自己，提升综合素质。

### 四、案例分析

**1. 个人经历**

*   **爱好带来的场景自信**： 通过爱好获得自信，但受限于场景。
*   **形象改造带来的场景自信**： 通过改变形象获得自信，但受限于形象。
*   **摆脱场景自信，获得核心自信**： 通过不断努力，摆脱场景限制，实现核心自信。

**2. 朋友案例**

*   **足球队长**： 因足球获得场景自信，但受限于场景，难以适应社会。

### 五、总结

**本节课介绍了场景自信与核心自信的区别，以及如何获得核心自信。希望大家通过学习，摆脱错误恋爱思维，提升自我价值，成为更好的自己**。
