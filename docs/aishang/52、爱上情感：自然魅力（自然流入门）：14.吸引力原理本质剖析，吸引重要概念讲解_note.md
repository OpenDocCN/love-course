# 52、爱上情感：自然魅力（自然流入门）：14.吸引力原理本质剖析，吸引重要概念讲解

## 概述

在本节课中，我们将深入探讨吸引力的本质，分析吸引力的核心工作原理，并讲解一些重要的吸引概念。

## 吸引力不是一种选择

**核心概念**： 吸引力不是一种选择，而是基因设定的自然反应。

**解释**： 我们不是完美的生物，而是进化的产物。我们的行为很大程度上受到基因的设定，包括对异性的吸引。这种吸引是基因设定的自然反应，而非我们的选择。

## 三脑工作原理

**核心概念**： 大脑分为核心脑、情绪脑和大脑皮层，吸引力的产生是这三个部分共同作用的结果。

**解释**： 核心脑捕捉到视觉信号，情绪脑产生吸引或排斥的感觉，大脑皮层则对这种感觉进行合理化。

## 逆向合理化

**核心概念**： 为了使情绪决策感觉良好，我们的逻辑会为情绪行为找一个正当的理由。

**解释**： 当我们的情绪产生某种行为时，我们的逻辑会为这种行为找一个合理的理由，以保持我们的想法和行动一致。

## 吸引力的肤浅性

**核心概念**： 吸引力很肤浅，它通过一些指标和行为线索来体现。

**解释**： 如果一个人没有吸引力的指标和行为线索，那么在初次见面时，他们很难被吸引。

## 择偶的代价

**核心概念**： 择偶是有代价的，挑剔的一方在繁衍和抚养后代上投入更多。

**解释**： 在自然界中，雌性在选择配偶时比雄性更加挑剔，因为她们在繁衍和抚养后代上投入更多。

## 择偶的策略

**核心概念**： 择偶策略包括雄性展示自己的吸引力，雌性进行废物测试。

**解释**： 雄性通过展示自己的吸引力来吸引雌性，而雌性则通过废物测试来评估雄性的价值。

本节课中，我们学习了吸引力的本质、三脑工作原理、逆向合理化、吸引力的肤浅性、择偶的代价和择偶的策略。这些知识有助于我们更好地理解吸引力的机制，并提高我们的吸引力。
