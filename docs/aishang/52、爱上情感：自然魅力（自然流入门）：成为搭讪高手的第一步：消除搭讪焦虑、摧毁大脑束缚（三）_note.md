# 52、爱上情感：自然魅力（自然流入门）：成为搭讪高手的第一步：消除搭讪焦虑、摧毁大脑束缚（三）

## 概述
在本节课中，我们将探讨如何通过改变行动预期来消除搭讪焦虑，并摧毁大脑中的束缚。

## 消除搭讪焦虑的方法
以下是消除搭讪焦虑的三个方法：

### 1. 改变行动预期
**定义**：行动预期是指在做任何事情时，大脑中都会有一个预期。

**原理**：预期会影响女生对你的反应，以及你接近女生的最终结果。

**案例**：
- **美好预期**：认为女生会因为你的接近而开心，会接纳你。
- **糟糕预期**：认为自己是不会被女生喜欢的，预期非常糟糕。

**结论**：美好预期更容易获得好的结果，而糟糕预期则可能导致失败。

### 2. 设定合理的目标
**原理**：过大的目标会给你过大的压力，从而难以出手。

**建议**：
- 细化目标，例如：与5个女生进行眼神交流。
- 不要以结果为导向，关注自身的提高和突破。

### 3. 提供价值
**定义**：提供价值是一种合作思维，你是在给对方提供价值，而不是试图从对方身上索取东西。

**案例**：
- **索取价值**：试图从女生身上获得好的回应、号码或后续机会。
- **提供价值**：认为女生会因为你的出现而变得开心，变得有价值。

**结论**：提供价值更容易获得女生的认可，并消除自我束缚。

## 总结
本节课中，我们学习了如何通过改变行动预期、设定合理的目标和提供价值来消除搭讪焦虑。希望这些方法能帮助你成为搭讪高手。
