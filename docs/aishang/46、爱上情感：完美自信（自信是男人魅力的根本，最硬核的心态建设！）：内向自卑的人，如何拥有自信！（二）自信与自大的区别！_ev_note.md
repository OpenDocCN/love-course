## 自信与自大的区别

### 1. 自信的定义

**公式**：自信 = 真正相信自己的能力

**代码**：`confidence = believe_in_my_abilities`

自信的人相信自己的能力，不会因为别人的评价而动摇。

### 2. 自大的定义

**公式**：自大 = 无时无刻不在与他人比较

**代码**：`arrogance = constant_comparison_with_others`

自大的人总是与他人比较，寻求超越他人以获得存在感。

### 3. 区别举例

**例子**：当别人说“你连西餐都没吃过，太low了”时，

* **自信的人**：可能会回应“哦，好吧。”
* **自大的人**：可能会回应“我虽然没吃过西餐，但我昨天刚吃的日料呢，你吃过吗？”

### 4. 内心强大

**公式**：内心强大 = 不在乎别人的评价

**代码**：`inner_strength = not_caring_what_others_say`

内心强大的人不会因为别人的评价而动摇，他们有自己的价值观和信念。
