# 52、爱上情感：自然魅力（自然流入门）：成为搭讪高手的第一步：消除搭讪焦虑、摧毁大脑束缚（一）

## 概述

在本节课中，我们将学习如何消除搭讪焦虑，摧毁大脑束缚，成为搭讪高手。

## 搭讪焦虑的来源

以下是搭讪焦虑的常见来源：

* **内部阻碍**：自我否定、焦虑紧张、感觉没有资格。
* **外部阻碍**：方法不当、女生有男朋友、女生不理睬。
* **大脑束缚**：恐惧被拒绝、尴尬、嘲笑。

## 消除搭讪焦虑的方法

以下是消除搭讪焦虑的方法：

### 1. 缩短反应时间

* 在看到心仪的女生后，3秒内必须采取行动。
* 缩短大脑反应时间，降低焦虑。

### 2. 保持大脑空白

* 屏蔽思维困扰，避免负面想法。
* 直接开场，绕过焦虑发生时机。

## 总结

本节课中，我们学习了如何消除搭讪焦虑，摧毁大脑束缚。通过缩短反应时间和保持大脑空白，我们可以克服搭讪焦虑，成为搭讪高手。
