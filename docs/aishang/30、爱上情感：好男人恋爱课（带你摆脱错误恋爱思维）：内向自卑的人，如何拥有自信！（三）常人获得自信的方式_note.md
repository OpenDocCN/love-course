# 30、爱上情感：好男人恋爱课（带你摆脱错误恋爱思维）：内向自卑的人，如何拥有自信！（三）常人获得自信的方式

**概述**： 本节课将探讨常人获得自信的四种方式，帮助内向自卑的人提升自信。

**一、提升自己的世俗价值**

* **核心概念**： 通过提升自己的世俗价值，如财富、外貌、地位等，可以短暂获得自信。
* **公式**： 自信 = 世俗价值
* **示例**： 通过购买新衣服、改变发型等方式提升自己的形象，从而获得自信。

**二、通过小团体抱团取暖**

* **核心概念**： 通过加入小团体，与他人一起活动，可以获得自信。
* **公式**： 自信 = 小团体 + 他人
* **示例**： 与朋友一起出去玩、参加社交活动等，可以获得自信。

**三、依赖某种特长或技能**

* **核心概念**： 通过掌握某种特长或技能，如唱歌、跳舞、摄影等，可以获得自信。
* **公式**： 自信 = 特长或技能
* **示例**： 通过学习唱歌、跳舞等技能，可以在表演时获得自信。

**四、依赖特定的场合中的特定角色**

* **核心概念**： 在特定的场合中扮演特定的角色，可以获得自信。
* **公式**： 自信 = 特定场合 + 特定角色
* **示例**： 在公司担任领导、在酒吧担任VIP等，可以获得自信。

**总结**： 本节课介绍了常人获得自信的四种方式，帮助内向自卑的人提升自信。通过提升自己的世俗价值、加入小团体、掌握特长或技能、扮演特定角色等方式，可以逐步建立自信，摆脱内向自卑的状态。
