# 35、爱上情感：恋爱速成班：能不能和女生先做朋友，再追她？

## 概述

在本节课中，我们将探讨一个常见的问题：是否应该先和女生成为朋友，然后再追求她。我们将分析这种做法的优缺点，并提供一些实用的建议。

## 核心概念

**公式**：建立男人对女人的感觉，而非朋友对朋友的感觉。

## 删除朋友关系的风险

以下是删除朋友关系可能带来的风险：

- **难以转变印象**：一旦成为朋友，很难将这种印象转变为情人的印象。
- **成功率低**：从朋友到恋人的转变几率非常小。
- **失去挑战感**：表白或过度关心可能使女生觉得你缺乏挑战性。

## 表白的误区

- **表白过早**：刚认识女生就表白，可能会让她感到不舒服。
- **失去主动权**：表白后，女生有权利拒绝你。

## 正确的相处方式

- **参加内部课程**：例如男生计划，学习男女交往的本质和流程。
- **避免表白**：通过自然的方式吸引女生。
- **了解女生心理**：避免有目的性的接近。

## 总结

本节课中我们一起学习了如何避免与女生成为朋友后再追求她的误区，并探讨了正确的相处方式。希望这些内容能帮助你更好地理解女生心理，提升你的恋爱技巧。
