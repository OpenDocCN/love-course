# 社交焦虑管理教程 - 第一课：理解与应对焦虑根源 🧠

在本节课中，我们将要学习如何识别社交焦虑的根源，并掌握一套系统的方法来应对它。我们将重点理解焦虑触发时的反应机制，并学习如何通过特定的步骤来管理这些反应，而不是被它们控制。

---

## 理解焦虑触发时的反应

当你被焦虑触发时，你会不自觉地发出声音或做出反应。理想情况下，你应该意识到这个触发点，而不是直接做出反应。但最终的结果往往是，你在社交焦虑中持续发出“假情绪”。

当你因社交焦虑而发出声音时，你会发现自己的反应与现实情况不符。这种发出声音的行为，就是你会达到的最终结果。你甚至可以定制发出这种反应的时间，当然，你也可以计时发出声音。

例如，你可能会想：“我应该什么时候发出声音？当我感到焦虑的时候。”但这恰恰是你最不想发出声音的时刻。通常，当你感到焦虑时，你会试图逃离，或者人们会尝试离开。当他们感到焦虑时，他们会试图逃离或自我沉淀。例如，当你非常生气时，你会想：“如果我大叫，它就会消失；如果我喝很多酒，它就会沉淀然后消失；或者我吃很多东西、看电视、试图逃离，它就会消失。”

## 建立应对系统：行动、发出、重复

你会把脚放下，决定不逃离，而是选择“发出”反应。因此，有一个方法：**行动、发出、重复**。这就是你应对社交焦虑挑战的方式，也是你应该利用的方法。

当你出去寻找参考，想要解决那些虚假的社交焦虑并达到目标时，请记住这个流程：当你“发出”时，遵循 **行动 -> 发出 -> 重复**。

我认为社交焦虑的挑战很棒。例如，进行“走楼梯”挑战，或者你可能会决定：“下周，我会和三个人说话。”这可能就是你的“行动”。现在，你有了一个系统，因此你有了步骤，知道如何以适合的、双赢的、可调整的方式去做到。但你仍然可能感到很多焦虑，所以你可能离开了你的位置，然后突然，焦虑感会击中你。

好，你完成了“行动”，你启动了“引擎”，进行了“发出”，然后“重复”。就这些，再加上时间和重复。你会开始放弃，释放自己。这正是真正解决问题的开始。

## 方法的细节与常见误区

如果你执行了这些步骤：**行动 -> 引擎 -> 发出 -> 重复**，每次启动引擎并发出，效果就会很好。但这里有一些细节需要注意。

我知道这听起来不够灵活，但这就是我们想要的。你可能会想：“那些小技巧怎么样？”不，你认为你能就这样解决吗？比如“击倒恐惧”那样的小技巧？那只会让你回到原点。

现在来看一些细节：调整和再调整。当你使用社交焦虑挑战时，有正确的方式和错误的方式。

错误的方式是：当你使用那些挑战或采取行动时，试图去抵制自己的感受。可能很多人到现在还在这样做。我刚开始的时候也做了很多。这就是你能知道自己是否在做这些事的方式：当你外出时，你可能会感到恐惧和焦虑，而你所做的就是依靠你所有的参考经验。你会说：“我做过这些，我可以做这些。”然后，你会想要对抗它，就像是“打破”它。你会想：“噢，我有一点担心，来吧，来做，来打破这些。”你甚至可能会去你的“快乐的地方”。你只是在试图抵制你感受到的焦虑：“让我去我的快乐的地方，然后打破它，去做。”

## 抵制策略的局限性

这是抵制。你试图抵制自己的感受。这非常有用，因为我说过，这很简单，简单到让你去做。然后人们会想：“嘿，我做了这些，我尽力去抗拒，去了我的快乐的地方，最后我完全抵制了自己，对吗？”

可惜答案是：虽然这会给你带来暂时的结果，但这不是你真正想要的结果。它只是暂时的，不会达到最终目标。就像我一直说的，它是暂时的，不会持续。你可能会在抵制的过程中感受到一点提升，但这就是会发生的情况：你必须一直行动，保持在快乐的地方，不要回复。这就是你所知道的，不是你想要的结果。当你害怕回复，而你可能会回复时，它不是永恒的。

再次强调，我一直在开玩笑。这是你的策略：如果你去抵制，你的策略就是“我会继续做到我80岁、90岁”。不，你不应该依靠任何东西来把自己放在那里。依靠“进步的抵制”，就像依靠喝酒一样：“我需要30分钟的行动才能不再感到紧张”和“我需要30分钟的喝酒才能不再感到紧张”是一样的东西。虽然两者都可能给你暂时的提升，但这只是成功的幻想。

## 重新建立基础：接受与重建

对于你，如果你一直在做这些，猜猜会怎样？就像你一直在打，比如说，这是起点，你能够很快地跳到这里，但你被困住了，你不能再往前走。所以你必须学会后退一步，然后重新开始。你可能会感到抵制，在做这件事时，就像你一直在尝试你的社交技巧，你一直在喝酒，你被困住了。

而真正的成功，你必须停止喝酒。但当你不再喝酒时，你会感觉到结果下降，你会回到起点。你必须接受“自我击中”，然后把它建立在一个稳定的基础上，然后再去打破它。同理，如果你一直在抵制，你必须重新开始抵制，对吧？抵挡着去到你快乐的地方，抵挡着冲突，重新联系那些在抵制之前吓到你的东西，然后从那里开始工作。

所以，接受“自我击中”，这就是重新联系。你可以试探一下自己是否在离开那些激烈的感觉，还是向着它们前进。比如，去跟那个人打招呼，去跟那个组织打招呼，这是你的“行动”，是你去做的。让我们把它塞进去，让我们去除掉它，对吧？或者让我们在这里生活，而不是感受是在离开、抵挡你感受的东西，还是通过它。你可以感觉到是在紧张还是在放松你感受的东西。

## 实践练习：冷却挑战

好，再一次，一个重要的练习，我给我的其他教练布置了很多寒假作业，就是：做一个“冷却挑战”。比如一个月，你走的方式不是想要抗冻，这是重要的。不要做一个自我帮助的方式，比如说：“走，冷，就像这样。”看看你如何在冷却中放松，因为那会是一种当水打到你的身体时，那种冷却的感觉。我们的立即反应是去抵挡。你可能也会有社交紧张的感觉，立即的感觉：“噢，噢，刺穿。”对吧？所以，抵挡那种忧心，看看你能不能放松每一块肌肉，进入冷却，然后完全放松，不会有刺激的感觉。当那种感觉打到你的身体，那会是一个你必须做的方式，当它来到“我被紧张的感觉，放松”。

## 深入探索：智慧路线法

所以，再一次，这一切都是关于自我帮助：感受那些感觉，并与之相互放松，而不是从外部塞进去或逃离。你也可以作为补充，跟随“智慧的路线”。这是一个更有精神层面的方式，但加上放松和内在工作，可以创造奇迹。

你问自己，例如，让我说：“去跟那个人打招呼，你会害怕。”你可以问自己：“我为什么害怕？”这叫做跟随智慧的路线。最终问自己：“为什么？我为什么害怕？‘因为我可以被拒绝。’”所以你从“我害怕跟这个人打招呼”开始，问“为什么？”——“那我可以被拒绝”。再问“为什么？为什么害怕被拒绝？”——“如果我被拒绝，人们会看到，我可能不被环境接受。”再问“那你不再接受，你会被拒绝，为什么？这更可怕。”——“那我会孤独。”这是一个例子。

好，所以害怕孤独。通过做这种智慧的路线，最终你可以说继续走，但是你能去多深入？它能够让你去深入感受。你可以说，我们有不同的表面恐惧，还有不同的核心恐惧。如果你放弃，你只想到表面的经验：害怕说“嗨”，害怕说“嗨”，害怕说“嗨”。这很好，但是如果你深入地工作，比如说，害怕孤独，害怕孤独，害怕孤独，你会更多地接触，因为这会引起你更多的影响，而任何其他外界的情况，都是与“害怕孤独”同时一起工作的。

## 结合方法与持续练习

所以我喜欢配合，让它走。你不需要经常做这些，但是，让我看看，我能不能再深入一点？为什么？好，另一个问题是：“我第一次经历这些是什么时候？”好，所以，你可以说：“噢，我害怕跟这个人打招呼。你还记得你第一次经历过这种感觉吗？你还记得你第一次……你不是这样出生的，这到底是什么时候？”这可以带给你一些更多的根本经历，可能会形成那些原始的伤痛。

现在，当你跟随了智慧的旅程，问自己：“当时是什么时候？”不要太过犹豫，就像我说的，不要认为你需要记住第一次工作。第一次可能是你生命中的第一次，你甚至没有记忆，你甚至不能形成记忆。所以，这不是说第一次记住，因为即使如此，也不会做得很好，对吧？就像第一次，让我说，你能够识别：“第一次是，你知道，在幼儿园，当这个人开我玩笑，那是第一次，我经历了社交焦虑。”酷，好意识，但是，这知识会改变任何东西吗？不。

所以，这就是问题：跟随智慧旅程，“当时是第一次”，它就会让你更有意识，但不解决。所以你不必记住所有的东西，但是这样做，两种方法，会让你可以说，让你更有感觉。所以而不是“我怕被拒绝”，问“为什么？”——“怕自己孤独”，可能会带来更多。一样的，“我现在感觉到这些，当时是第一次”，可能会带来更多。

所以目标是，这种更有精神的方法，是为了让你能够更多地感觉，就这样。这不是想要记住，这是在感觉层次上做的事情。真正的工作，是在感觉层次上做的，就是放弃，不在这里，不在记忆中，放弃。再说一次，这不是知道你问题的一切，而是解决它。你不必记住第一次来解决，就像我说的，经常你不能。

## 最终建议：行动与内在工作结合

最后，闭嘴，继续放弃。意思是，如果有些东西一直在你身上，你一生，就像社会的焦虑，就像你一生，你会说：“让我做一个冥想，它已经没了吗？”回到地球：时间，和练习。时间，和练习。

好，你一生在这个模式里，可能会花点时间去解决。这不意味着你不能做到，你应该还在做，开始放弃。不要用“哦，解决问题是内在工作”作为不做事情的理由。把你脑袋放出去，去做事，然后配合放弃的行为，和被困在解决问题的行为。好，这就是你解决问题的方式。可能会滑倒。
