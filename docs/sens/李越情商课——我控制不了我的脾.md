# 李越情商课——我控制不了我的脾

在本节课中，我们将要学习如何理解和管理自己的脾气。我们将探讨情绪产生的根源，并学习如何通过觉察情绪产生的“节点”，来更好地与自己的情绪相处，而不是试图在爆发时强行控制它。

## 概述：情绪无法被“控制”

上一节我们介绍了情绪的基本概念，本节中我们来看看如何处理“容易生气”的问题。许多人会问：“我特别容易生气怎么办？”或者“别人一点我就炸，怎么控制自己？”。

实际上，情绪的产生是无法被掌控的。我们之前课程中一直强调，当你意识到自己在生气时，就可以告诉自己“不要去生气”。问题的核心往往在于，在理性层面上，我们缺乏控制情绪的**意愿**。

一旦你认为“生气后需要控制”，就容易陷入第一个误区：将情绪的发作或持续视为理所当然。他们会觉得“自己生气很正常，生气过后要控制住”。但这根本无法解决问题，因为在情绪上来时，我们很多时候是无法控制它的。例如，愤怒作为一种自我保护机制，一旦发作就很难控制。

所以，如果你经常出现这个问题，关键不在于“生气后怎么办”，而在于**觉察情绪产生的原因**。情绪产生后，持续性地发作对于我们思考和解决问题意义不大。重要的事情是，我们要去思考“为什么会产生这些情绪”。

## 情绪产生的根源：期望与现实的冲突

上一节我们提到了情绪的作用，本节中我们来深入探讨情绪，特别是负面情绪是如何产生的。

人产生情绪是因为本能，如生存、繁衍或提高认知。而被人们深刻记住的情绪，往往是负面情绪。这些负面情绪产生的核心原因，是**期望与现实的不符**。

**公式：负面情绪 ≈ 主观期望 - 客观现实**

*   **主观世界**：是我们对世界、人事物的规律和规矩的想象。我们认为世界“应该”按照某种方式运行。
*   **客观世界**：是现实实际发生的情况。

当主观认知的世界和客观现实的世界发生冲突时，就会产生情绪。例如，你认为“我对男朋友非常好，无微不至，他就应该珍惜我、对我好”。这只是一个主观认知。现实中，可能发生“你对他好，他却出轨了”的情况。这时，你的期望与现实严重不符，就会产生强烈的痛苦、愤怒等负面情绪。

产生这种情绪的本质原因，是为了让你去**认知客观现实的规律，去接受现实**。人生中的所有苦痛和情绪，本质上都是为了帮助和提醒我们去认清现实。

## 识别情绪的“节点”：过往经历与未来期望

理解了情绪产生的根源后，我们需要识别哪些具体情境会触发我们的情绪。这些触发点通常与两个因素有关：**过往经历**和**未来期望**。

以下是两种主要触发情绪的情况：

1.  **过往经历的“节点”**：成长过程中，那些让你无法释怀、期望与现实严重不符的经历，会形成情绪记忆的“节点”。当类似情境再次出现，这个“节点”就会被触发。
    *   **例子**：小时候因父母外出被独自留在家，感到孤独和被抛弃。长大后，当伴侣要单独和朋友聚会而不带你时，这种“被抛弃”的孤独感就会被唤起，导致你产生负面情绪。

2.  **未来期望的落空**：你对未来某事抱有特定幻想或期望，但现实并未按此发生。
    *   **例子**：你期望伴侣在生日或节日一定会给你准备惊喜，以此作为“爱”的证明。但当对方忘记或没有准备时，期望落空，你就会产生失望、生气等情绪。

**核心要点**：你需要知道，自己可能会因为特定的过往经历或对未来的特定期望，而在某些情境下产生特定的情绪。**觉察到这一点本身，就是管理情绪的第一步**。

## 如何找到你的情绪“节点”？

上一节我们了解了节点的来源，本节中我们来看看如何具体找到属于自己的情绪“节点”。

在人际交往中，别人做的某件事或说的某句话让你极度不舒服，往往不是因为这个人本身，而是他作为一个“导火索”，诱发了你内心深藏已久的、由过往“节点”带来的痛苦。

以下是几种常见的关键人生节点：

*   **被忽视**：成长中经历过被忽视（如家人更关注其他孩子），可能导致长大后再次被忽视时，产生憎恨、嫉妒、失落或讨好的情绪。
*   **被误解**：曾被冤枉或误解（如被错误指责），可能导致对“被误解”极度敏感，容易产生愤怒和隐忍报复的心理。
*   **被打压**：表达想法或展示自己时遭到否定和冷嘲热讽（如父母嘲笑你的梦想），可能导致你羞于表达真实自我，害怕再次被打压而产生羞耻、尴尬的情绪。
*   **被嘲笑**：有过被公开嘲笑的难堪经历，这会使你对“被嘲笑”相关的情境产生强烈的防御或愤怒情绪。
*   **被拒绝/被欺骗**：类似的负面经历都会形成情绪记忆的节点。

**行动建议**：课后请认真思考，你会因为哪些具体的事情或情境而生气、难过？尝试将这些事情与你童年的类似经历联系起来。找到这些“节点”，是理解自己情绪模式的关键。

## 总结与下节预告

本节课中我们一起学习了：

1.  情绪爆发时难以控制，管理的核心在于**觉察情绪产生的原因**，而非事后压制。
2.  负面情绪主要源于**主观期望与客观现实的冲突**。
3.  我们的情绪反应常由**过往经历形成的“节点”**和**对未来期望的设定**所触发。
4.  通过回顾人生，识别如**被忽视、被误解、被打压**等关键节点，可以预先知道自己会在何种情境下产生何种情绪。

**重点重申**：我们觉知情绪的目的，是为了在下次类似情况出现时，**不被情绪完全控制**，而不是去解决“这一次如何把情绪压下去”。知道自己会因为什么而生气，本身就是一种巨大的进步。
