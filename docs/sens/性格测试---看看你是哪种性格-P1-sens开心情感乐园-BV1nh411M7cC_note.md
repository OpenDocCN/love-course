## 概述：为什么需要了解性格？

上一节我们介绍了情绪表达的重要性，本节中我们来看看如何因人而异地沟通。沟通不能使用同一种方式对待所有人，因为不同性格的人需求点不同。学习性格知识，能帮助我们更好地与不同人交流。

## 性格测试初体验：你的选择是什么？

以下是两个情境选择，请根据你的直觉做出决定：

*   情境一：在狮虎山与狮子老虎共处24小时，无装备、无救援，九死一生。
*   情境二：在充满无数无毒蛇的蛇窟里待24小时，蛇会缠绕你但无生命危险。

**必须二选一**。这个选择能初步反映你的性格倾向：选择狮虎山（选项1）的人可能更在乎过程和感受；选择蛇窟（选项2）的人可能更以结果为导向。选择不同，是因为性格不同，我们在乎的东西也不同。

## 引入PDP性格测试

PDP，全称 Professional Dynamic Programs，是一个行为特质动态衡量系统。它能有效帮助我们协调不同性格之间的关系，理解对方的需求点。

在学习具体类型前，请先看看以下四种动物，你觉得自己最像哪一种？

*   老虎
*   孔雀
*   猫头鹰
*   考拉

## 详解老虎型（力量型）性格 🐯

老虎给人的印象是凶猛、强悍。接下来，我们通过几个问题来剖析老虎型性格的特征。

### 特征一：结果导向，现实理性

老虎是肉食动物。这象征着老虎型人格非常注重**结果**，非常**现实**。他们只关注看得见的利益和目标，方向明确，以结果为导向。他们常常会忽略情绪和感受，直奔主题而去。

**核心公式**：**目标 > 感受**

### 特征二：掌控欲强，强势主导

老虎占山为王，领地意识极强。这反映了老虎型人格有很强的**掌控欲**和**征服欲**。在生活中，他们表现得非常强势，喜欢命令和控制，常以自己的目标和意愿为主，不太在乎别人的感受或意见。

**行为模式**：命令、控制、要求服从。

### 特征三：行动迅速，坚忍不拔

老虎捕猎的特点是快、准、狠，但成功率并不高。这对应了老虎型人格**行动力极强**、雷厉风行，但可能因准备不足而显得冲动鲁莽。同时，他们**坚忍不拔**，对认定的事情和目标一往无前，抗压能力很强，不达目的誓不罢休。

**核心代码**：
```python
while not goal_achieved:
    take_action() # 立即行动
    if failed:
        persist() # 坚持，不放弃
```

### 特征四：喜欢独行，不易受影响

老虎是独居动物。这意味着老虎型人格**特立独行**，喜欢独来独往。他们不容易受他人影响，很难听取别人的意见，常常坚持自己的决定。

## 典型代表

讲到这些特征，我们很容易联想到一位典型的企业家代表：**董明珠女士**。她的管理风格以强势、果断、目标明确著称，是老虎型性格的典范。例如，在会议中她可能直接下达指令，而非讨论，强调效率与服从。

本节课中，我们一起学习了PDP性格测试中的第一种类型：**老虎型（力量型）**。我们了解到老虎型人格的核心特征是：**结果导向、强势掌控、行动迅速、坚忍不拔且喜欢独行**。理解这种性格，有助于我们在沟通时识别对方需求，避免冲突，实现更有效的交流。
