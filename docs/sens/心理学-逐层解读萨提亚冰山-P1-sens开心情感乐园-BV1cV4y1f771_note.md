# 心理学课程-01：逐层解读萨提亚冰山模型 🧊

在本节课中，我们将学习心理学中的萨提亚冰山模型。这是一个帮助我们理解人类行为与内在心理过程之间关系的工具。我们将从最表层的行为开始，逐层深入，探讨感受、情绪与生命力之间的内在联系。

## 行为层：改变的起点

我们首先解读冰山的第一层——行为。例如，一个人可能总是无精打采，对谁都感到沮丧，缺乏活力，不愿说话。此时，刻意改变外在行为，例如早晨增加一些运动，确实可以影响整体的生命力状态。

因为心理学指出，身体的变化会直接影响心理状态。然而，这种改变往往不持久。

## 为何行为改变难以持久？

行为改变不持久的原因在于，我们可能并不清楚为何要改变，并且内在的自我根本没有发生变化。仅仅改变外在行为，其效果永远是有限的。

**真正的、无限的力量源于内在的改变。** 因此，当行为改变难以实现或维持时，我们就需要向冰山的更深处探索。

## 潜入水下：探索感受层

上一节我们讨论了行为改变的局限性，本节中我们来看看行为背后的驱动层——感受。当无法理解一个人为何总是表现出某种行为时，我们就需要“潜水”，去探索水面之下的冰山。

首先需要探索的是他的**感受**。例如：发生争吵时，他的感受是什么？为何他一冲进办公室就对我发火？为何回到家会对家人生气？每一个行为背后，都有一系列冰山层次在运作。

### 感受与情绪的关系

那么，感受是什么？感受与情绪是同一个东西吗？答案是否定的。它们之间存在区别与联系。

一些书籍会将情绪和感受连在一起讲。原则上，感受和情绪本质是同一事物，只是情绪是外在可见的，而感受是内在的、不可见的。当两者一致时，它们就是一回事。

**举例说明：**
> 我心里非常愤怒，因为刚刚有人侵犯了我的权益。但作为一名教师，为了维持良好的授课状态，我表面掩饰自己，表现得平静。此时，外在的“情绪”是平静的，内在的“感受”却是愤怒的。这时，情绪与感受就分裂了。

这种状态称为“身心分裂”——“身”是外在可观察的，“心”是内在真实的。当身心一致时，情绪与感受就是一回事，例如开心就哈哈大笑，生气就表现出来。

在冰山理论中，我们常常需要在对方风平浪静的外表下，去探索其可能波涛汹涌的内在。学会区分情绪与感受至关重要。

## 感受的分类：舒服与不舒服

以下是对感受的分类。我们通常将感受分为两类：

一类是让我们**舒服的感受**，通常被称为正面情绪；另一类是让我们**不舒服的感受**，被称为负面情绪。

*   **正面情绪示例**：开心、喜悦、幸福、快乐。
    *   （注：骄傲常被视为一种状态，其背后的核心感受可能是满足感。）

*   **负面情绪示例**：沮丧、委屈、愤怒、焦虑、惊慌。

人们通常不喜欢负面情绪，原因很简单：它们让人感觉不舒服。我们从小就被潜移默化地教会：让我们舒服的是“好”的，让我们不舒服的是“坏”的。但让“我”不舒服的人或情绪，就一定代表“负能量”吗？答案是不一定。

## 感受如何影响生命力

为什么要详细区分感受呢？因为一个人的感受状态，会直接影响其整个冰山系统的运作。

**核心机制如下：**
> 当你的感受常常是**舒服的**（如兴奋、感动、喜悦、平静、放松），这些感受会自动滋养你的内在**生命力**。生命力得到滋养，其核心部分（如爱与被爱、连接感、安全感、价值感等）就会茁壮成长。

一旦生命力被滋养，它就会反过来有力地**支持外在行为**。你会更有活力、更有动力去行动。

## 从感受到目标：行为的动力链条

我们活在世上，无论是个人成长、创业还是家庭生活，最终都希望实现某些人生价值或目标。人生目标依靠什么实现？**依靠行为。**

但你是否遇到过这种情况：明明知道该做什么，就是提不起劲去做？或者看到团队成员缺乏活力？这往往是因为，支持其行为的内在生命力已经枯竭。

**逻辑链条可以逆向推导：**
1.  行为缺乏动力。
2.  原因是内在生命力枯竭，无法支持行为。
3.  生命力枯竭，则意味着个体的感受**长期处于不舒服的状态**。

因此，为你的冰山系统负责，**最首要的一步就是设法让你的感受好起来**。因为只有感受变好，才能滋养生命力；生命力充沛，才能为行为注入活力；有活力的行为，才能真正创造出我们想要的人生目标。如果生命力得不到滋养，无论计划多么宏伟，都难以有效执行。

---

**本节课总结：**
我们一起学习了萨提亚冰山模型的前几个层次。我们从最表层的**行为**开始，认识到单纯改变行为的局限性。接着，我们潜入了水下，探讨了驱动行为的**感受**层，厘清了感受与情绪的区别与联系。然后，我们将感受分为舒服与不舒服两类，并揭示了**感受如何直接滋养或损耗我们的内在生命力**。最后，我们理解了“感受 → 生命力 → 行为 → 目标”这一动力链条。管理好我们的感受，是激活整个系统、实现个人目标的基石。
