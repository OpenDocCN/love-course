# 情感行业揭秘课程 - P1 🕵️♂️

在本节课中，我们将一起了解情感咨询与培训行业的一些内部见闻与运作逻辑。我们将探讨该行业的核心商业模式、关键人物以及一些普遍存在的现象。

## 行业本质：培训驱动的商业模式 💼

情感行业是一个依靠培训来维持存在的行业。

那么，为什么需要培训呢？与其他行业类似，这个行业也需要对导师进行包装。例如，包装导师的资历，声称他是某某大学教授。

但是，在这个行业中，导师的权威性必须建立在实战经验的基础之上。只要你的实战案例足够多，你就能招到学员。别人会因此相信你，认为你能够帮助他解决情感问题。

这就是该行业简单粗暴的逻辑。它不看重你的理论或头衔，只看重所谓的“实战”成果。

## “实战”光环与教学效果的疑问 ❓

上一节我们介绍了行业看重“实战”的逻辑，本节中我们来看看这个标准是否合理。

因此，行业内特别鄙视“键盘侠”。大家都强调自己是实战派。至少主流声音是这么做的。

但实际上，“实战经验丰富”是否就意味着他教学教得好呢？这一点实际上要打一个问号。情感方面的东西，不能用如此单一的标准去衡量。还有很多其他因素也需要考虑。

## 行业代表人物与变迁 🔄

接下来，我们来看看行业内的一些关键人物及其发展轨迹，这部分内容你们可能会相对比较感兴趣。

以下是几位具有代表性的导师及其故事：

*   **浪迹**：他是后期具有极大影响力的人物，我们稍后会详细讲到。
*   **冷爱**：他很早就转型去做女性情感教育了。我认为他是一个充满智慧的人。在好几年前，PUA行业还没有像现在这样受到打击和下滑时，他就已经转型，并且做得很好。
*   **花镇情感**：这个品牌虽然中间也有波折，但总体而言，女性情感市场的风险比男性情感市场小太多，而且他现在也做得很好。
*   **墨菲斯**：他大概现在有40几岁了。当年他参加电视节目，在节目里与女嘉宾互动，闹出过一些笑话。我个人当时觉得他还行，但后来听别人说，他实际上是一个“比较大的剧本”（意指套路深）。我听说的，我没见过他。他似乎比较喜欢创造一些概念，故意显得自己很厉害。
*   **摩卡、Tango等**：这批人包括坏男孩团队的Tango。其实这些历史都已经过去了。现在你去搜索这些名字，他们的资料还会被一些卖盗版的人拿出来卖。我觉得这很可笑，因为他们已经很多年不干这个了，看那些过时的资料没什么用。

## 平台崛起与“浪迹”团队 📈

再往后，就是以浪迹为代表的一批人崛起的时代。他们是在“坏男孩”这个团队里成长起来的。

“坏男孩”当时成功的原因，是将“恋爱巴士”做成了一个类似淘宝的交易平台。平台突然爆发，月销售额增长非常快。

就在这个时候，浪迹团队在那个平台上表现突出，销售额做得很大，成为了第一名。之后他们便单独出来，成立了公司。

我个人觉得，浪迹团队应该算是这个行业里最有影响力的一个公司。我也曾在那里工作过一段时间。

## 行业生态：管理与从业者素质 🏢

那么，这家公司有什么可讲的呢？它的公司在这个行业里算是比较正规的。当然，如果与其他管理完善的互联网公司相比，还远远不够。但在这个行业中，已经可以了。

我为什么这么说？因为这个行业实际上比较乌烟瘴气。

首先，我们说从业者素质。你看那些导师，他们的头衔通常是什么？比如“我写了一个什么样的案例”或“我写了一本什么样的电子书”。实际上，他们没有一个特别牛的正式头衔，也没有很硬的社会关系背书。再加上他们普遍比较年轻，所以你也不能过分责怪。

他们都喜欢展示自己的实战成果。例如，当年经常被同行抨击的台湾导师“成真”，他就从来没有展示过实战的东西。我们行业里的人以前都喜欢批评他，因为他只做视频，只搞理论。

他的理论非常多。他做视频时，背景里放两本非常厚的书。谈个简单的恋爱，有必要搞那么复杂吗？当然，人需要学习，但不见得要在恋爱这一块学习如此复杂的理论。因为没必要，每个人都是不一样的存在。这一点我后面的视频会详细聊到。

## “三驾马车”的案例 🚗

最后，我们再看一个比较有影响力的例子，比如浪迹教育旗下的“十大导师”，其中就有所谓的“三驾马车”。

“三驾马车”喜欢在一起，因为他们都是北方人。他们在成都时就住在一起。后来他们去深圳，这也是他们与浪迹教育分开的契机。我当时刚好在他们公司，负责监督他们的一些事务，比如财务和学员招生。

我去看的时候（在这个视频里我可能会附一些照片），发现他们起步其实挺困难的。三个人在深圳，各种成本和开销非常大，管理也充满了问题。都是男性在一起，很多时候效率并没有那么高。这就是“三驾马车”包括飞哥他们的故事。
