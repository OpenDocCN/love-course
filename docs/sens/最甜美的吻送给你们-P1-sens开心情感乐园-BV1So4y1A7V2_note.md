## 歌词叙事结构解析 📝

了解了核心情感后，我们进一步分析歌词的叙事结构。它并非平铺直叙，而是通过场景的切换和情绪的循环，模拟了现实中难以启齿的爱恋心理。

歌词的叙事遵循一个循环结构，可以用以下伪代码表示：

```python
while 爱意存在:
    产生表白的冲动
    if 感到畏惧:
        将话语咽下 # “无法说出口的话”
        陷入痛苦与想象 # “在梦里” / “眼泪流淌”
    else:
        尝试表达 # “我想要说出口的话”
```

以下是歌词中体现叙事逻辑的几个层次：

*   **内心独白**：大量运用第一人称“我”，直接倾诉内心活动。

*   **重复与强调**：通过重复“说出口的话”、“我愛你”等短语，强化了情感的迫切感和受阻感。
