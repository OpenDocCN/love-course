# 李越情绪价值课——男女聊天中，

在本节课中，我们将学习在男女聊天中，如何避免错误并提供高质量的情绪价值。课程将首先回顾聊天的禁忌，然后深入讲解七种具体的夸奖技巧，帮助你成为更受欢迎的沟通者。

## 回顾：聊天的六大禁忌 ⚠️

上一节我们介绍了课程的整体框架，本节中我们先来回顾聊天的核心禁忌。避免犯错比做对更重要。

以下是聊天的六大禁忌：

1.  **代入感过强**：在暧昧期，不要自动代入男女朋友角色，要求对方付出。记住双方“什么关系都不是”。
2.  **交浅言深**：关系未到深度时，避免谈论过于深入的话题（如原生家庭创伤），否则会显得没有边界感。
3.  **传递负能量**：不要向他人宣泄对生活、工作或家庭的不满。负能量会让人感觉你是一个“能量黑洞”。
4.  **需求感过强**：不要时刻抓着对方聊天、希望对方排解你的孤独。这会给对方带来捆绑感和压力。
5.  **态度卑微**：不要因对方条件好就自动成为“舔狗”。心态处于劣势会让你不自信，对方也可能瞧不起你。

聊天一旦踩中这些雷区，关系基本就结束了。

## 核心技巧：如何提供极致的情绪体验 ✨

了解了不该做什么之后，我们来看看具体该怎么说话。核心是“嘴要甜”，但夸奖需要技巧，避免尴尬和流于表面。

### 1. 反其道而行：夸“缺点” 🔄

两个人放下偶像包袱，才可能坦诚相处。面对对方的缺点，不要说“我不介意”，这反而是在承认缺点。

**正确做法是：将缺点在他那里变成优点。**

*   **公式**：`你的[缺点]，在懂得欣赏的人眼里，其实是[优点/特质]的体现。`
*   **举例**：
    *   对身材圆润的女生说：“狗才喜欢骨头，有质感的男生都喜欢有些肉感的女孩。”
    *   对父母离异的男生说：“有这样的经历，你现在才比任何人都更明白家庭的重要性。”
    *   对头发少的男生说：“现在这是变强的象征，证明你很厉害。”

当对方发现他最介意的缺点在你这里成了优点，自然会卸下防备。

### 2. 夸细节：从细微处见真章 🔍

大部分人希望赞美是具体、可信的。从细节入手，并看到细节背后的品质。

**方法是：观察一个具体细节，并引申出背后的个人特质。**

*   **公式**：`我注意到你[具体细节]，这一定说明你是一个[某种品质]的人。`
*   **举例**：
    *   “你连手指甲都修剪得这么整齐，一定是个对生活很有讲究的人。”
    *   “你的T恤图案好独特，是学艺术的吗？果然学艺术的人感觉就是不一样。”
    *   “你的车里好整洁，果然低调的理工男都是宝藏男孩，细节控。”

越是细节的夸奖，越能展现你的真诚和观察力。

### 3. 夸他愿意花时间的地方 ⏳

一个人把时间花在哪里，哪里就是他重视并想获得收获的地方。

**做法是：认可他投入时间的爱好或事业，这本身就是对他这个人的认可。**

*   **公式**：`你专注在[他的爱好/工作]上的样子，整个人都在发光。`
*   **举例**：
    *   他喜欢拼乐高或弹钢琴：“你全神贯注的样子真的好迷人，一定会为拥有这样的爱好而自豪。”
    *   他坚持健身：“你对抗压力的样子，真的很帅。”

### 4. 夸过程与努力，给正向预期 🌱

当对方热爱某事但尚未取得明显成果时，要肯定他的过程，并给予对结果的信任。

**核心是：忽略当前不完美的结果，强调他努力过程中的品质，并表达“我看好你”。**

*   **公式**：`虽然现在结果还不明显，但我知道你一直在努力。[描述他努力的样子]，[表达信任]。`
*   **举例**：
    *   “这样的硬骨头也只有你才能啃得下来。”
    *   “过程虽然辛苦，但你在过程中努力的样子真让人佩服，你身上写满了坚毅。”
    *   “我看好你，你一定可以。”

不断给予肯定，他会知道你在这件事上是支持他的人，从而更愿意与你分享。

### 5. 夸他带给你的影响（对高价值对象奇效）💎

对于条件较好的男性，他们需要更高阶的情绪价值：被信赖、被认可能成就他人。

**方法是：真诚地向他请教，并给予实际行动后的积极反馈。**

*   **步骤**：
    1.  就你不擅长的事（如工作方法、时间管理）向他请教。
    2.  **关键**：按照他的建议去执行。
    3.  一段时间后，给予具体反馈：`用了你的方法，我[取得了什么进步/解决了什么问题]，真的非常感谢你。`
*   **举例**：“用了你教的时间管理法，我的工作效率大大提升，整个人都松弛了，谢谢你。”
*   **注意**：请教时只陈述问题，不要附带抱怨情绪。反馈要真诚，而非故意套近乎或占便宜。

你的反馈让他感到被信赖、能影响他人，满足了他的“英雄梦”。

### 6. 夸正向行为与证据，引导关系走向 👍

在任何互动中，忽略“不爱”的证据，寻找并强化“爱”的证据。

**心法是：定义对方爱你，并夸奖他爱你的行为，他会更愿意重复这个行为。**

*   **公式**：`你做了[具体行为]，这说明你[关心/爱我]，我好开心。`
*   **正面举例**：
    *   他忙了一天晚上才回消息关心你：“亲爱的忙完了还想到我，关心我吃什么，这么爱我，好开心。”
    *   他出差买了不合适的包给你：“宝贝出差还记得给我买礼物，太感动了！你送的我都喜欢。”（同时委婉告知喜好）
*   **反面警示**：如果指责“你怎么一天都不回消息”，对方会感到挫败，不再想做类似的事。

### 7. 给予认可与信任的“偏爱句式” 💖

以下是几种能极大提供情绪价值的表达句式：

*   **“这是我的第一次...”**：`我第一次主动约男生看电影。` 意味着深度认可和区别对待。
*   **“我从来没...”**：`这事我从来没跟别人说过。` 表达了极高的信任，将对方视为最亲密的人。
*   **“只为你”**：对方感谢时，用轻松语气说 `Only for you`。表达了偏愛，又不施加压力。
*   **“只有你可以”**：在对方受挫时使用。`我相信这件事只有你才能做到。` 给予极大的肯定和信任。
*   **“因为你我才...”**：`因为认识你，我才变得比以前更平和。` **注意**：后面必须接**正向**改变，而非抱怨或绑架。

## 心法与总结 🧠

本节课我们一起学习了如何通过聊天提供直击灵魂的情绪价值。

**最后两条核心心法：**

1.  **真诚可以“演”**：别人看不见你的心，只能看见你的行为。说话时直视对方、适时点头，结合真诚的话语，就能传递出足够的诚意。
2.  **反向学习**：当别人夸你让你非常受用时，记住那种感觉，并分析对方在什么场景、用什么方式夸了你。记录下来，作为你的学习素材。

将这些技巧内化为技能，你将在人际交往中无往不利。在渴望被关注、被认可的社会里，提供独特的情绪价值会为你的人生带来诸多便利。

**总结**：避免六大禁忌，灵活运用七种夸奖技巧（夸缺点、夸细节、夸投入、夸过程、夸影响、夸正向行为、用偏爱句式），并辅以真诚的态度和反向学习的心法，你就能成为一个高情绪价值的供给者。
