# 恋爱与武器 第1课：从“追求”到“吸引”的思维转变 🎯

在本节课中，我们将要学习一个核心观念：为什么在现代社会，吸引女生比追求女生更有效。我们将通过分析时代变迁、信息环境和个人包装三个方面，帮助你理解并掌握“吸引”的底层逻辑。

## 一、时代已经改变：从“有限选择”到“无限可能”

上一节我们介绍了课程的主题，本节中我们来看看时代背景的根本性变化。

从前的社会环境是“车马很慢，书信很远，一生只够爱一个人”。这听起来浪漫，但其本质是**选择有限**的结果。你的母亲在年轻时，村里年龄相仿的男性可能只有张三、李四、王五和你父亲这几个人。即便她没有完全看上你父亲，但与其他人的条件（如两头猪、两亩地）相差无几。在没有更多比较的情况下，谁对她更好，她就可能选择谁。那个年代的婚姻大多如此，离婚率低，通过“追求”确实可能成功。

但那个时代已经过去。

你的母亲一生可能只认识几位男性，而**现在的女性一生中可能会遇到上千万个男性**，这还不包括互联网上的搭讪。如果你在社交平台注册一个女性账号，无论照片如何，一天内主动打招呼的男性可能数以千计。这就是现实。

当年，你父亲请你母亲看电影、帮忙干农活、时常送水果，这些行为在当时是新颖且难以拒绝的关怀。然而，**拿过去追求女生的方法用在现代女性身上，无异于“拿着明朝的尚方宝剑斩清朝的官”**。

你向她表白，但她可能从幼儿园起就没少收过情书。当**她收到的情书比你写过的作业还多**时，她还会珍惜你的心意吗？你必须明白，你喜欢的女生只要形象尚可，就绝不止你一个人在追求。即便是条件普通的女生，也会有其他男性去讨好。

那些形象好、气质佳的女生，你越是刻意追求，对方可能越看不上你。因为你的讨好和表白，如果缺乏独特性，在她眼中就会显得廉价，并与她遇到的其他“追求者”无异。**你的讨好和表白，若没有高价值作为支撑，就会显得很廉价。**

## 二、拥抱变化：在互联网时代主动营销自己

上一节我们探讨了时代变迁带来的挑战，本节中我们来看看如何适应这个新时代。

有些兄弟不喜欢用社交软件，认为从身边人入手更真诚，或觉得社交软件上没有真正的爱情。这让我想起一个故事：一位企业老总炫耀自己只用小灵通，不用智能手机。我提到，百丽集团的CEO也从不使用智能手机甚至电脑，而结果是百丽关停了上千家门店并最终退市。**停止抱怨，拥抱变化才是唯一的出路。**

固步自封，会让你错过每一个时代风口。现在是互联网时代，一个信息过剩的时代。

你是否在公司见过颜值出众、身着职业装的人来找你的老板？她们很多是基金经理。当你的资产达到一定量级，就会有专业人士来帮你管理财富。你说你没钱，接触不到？这没关系。电影《华尔街之狼》中，乔丹·贝尔福特将垃圾股卖给穷人而暴富。他的太太问为何不卖给富人，他说：“富人都很精明，怎么会相信我？穷人虽然穷，但又傻又懒，还很贪婪。”

**不同阶层的人，在这个时代都会被对应的产品找上门。** 你昨天浏览了某个网站，今天可能就会收到相关的商品推荐。信息太过剩，产品如果找不到目标用户就会滞销。

**你可能就是那个“产品”。** 如果你还执着于“在一棵树上吊死”，不去寻找属于你的“用户群体”，那么你这款“产品”迟早会滞销倒闭。

### 如何让你成为“畅销产品”？

以下是成为“畅销产品”的关键步骤：

1.  **停止无效标榜**：不要再只是标榜自己是个“好人”。别人不知道，也不关心。不要再空等“缘分”，**新中国以后就没有神仙妖怪了**。你有那个时间，不如去经营自己。
2.  **进行专业包装**：一款畅销产品离不开专业的营销和包装。想成为受女生欢迎的男人，同样需要。这就是**展示面**的重要性。在互联网时代，没有线上展示面，你就像一个网络透明人。
3.  **打造互联网名片**：微信创始人张小龙曾透露，有上亿用户设置朋友圈“仅三天可见”。你是否是其中之一？**什么样的女生会愿意和一个网络透明人聊天甚至谈恋爱？** 这恰恰是你在网上容易遇到骗子的原因——因为正常的女生不会愿意与一个没有身份标识的人深入社交。

试想，一个头像非本人、朋友圈一片空白的人约你去酒店，你会去吗？你会警惕。同理，**女生凭什么喜欢上一个透明人？**

现在大家时间碎片化，生活节奏快，都很“懒”。对于没有任何包装的透明人，女生**压根就懒得搭理**。她们更愿意选择那些**看起来**有品位、有学识的男人。

女生喜欢拆盲盒，是因为盲盒有**华丽的包装和抓人的标语**，让她们觉得其中有惊喜。一个没有头像、没有生活展示的男人，就像路边的破纸盒，即便内有宝藏，也只会被偶然的拾荒者捡走。

### 课后作业：打造你的第一张互联网名片

以下是你的第一个实践任务：

**给自己拍摄一张合格的微信头像。**

**建议：**
*   **不要只拍一张大脸**：如果你的脸不足以像明星一样具有吸引力，那么单纯的脸部自拍无法征服女生。
*   **用环境加分**：自拍应透露出你所在的环境。传达的信息是：“我虽然不是超级帅哥，但我很有生活品质，我出入的场合是整洁、有格调的。” 例如，在整洁的咖啡厅、书店或有设计感的背景前拍摄。
*   **巧用镜子**：对镜自拍可以自然展现半身像，囊括更多环境信息，更直观地展示你的整体形象和穿搭。

示例描述（想象以下画面）：
*   坐在一家有格调的咖啡馆窗边，手边有一本书或一杯咖啡。
*   在健身房镜子前，展现健康活力的状态。
*   在一个艺术展或设计简洁的公共空间。

## 三、形象是第一生产力：这是一个看脸的时代

上一节我们学习了如何打造线上展示面，本节中我们来看看形象在现实中的决定性作用。

一个社会实验：一群模特混入应届毕业生参加招聘会，尽管他们对专业问题一无所知，回答“不知道”，但面试官却认为他们“真诚”，并全部录用。而真正的毕业生若回答“不知道”，则会被直接淘汰。

如果谁还在告诉你“这个世界不靠形象”、“不以貌取人”，那他一定是在欺骗你。这可能令人难以接受，但**这就是事实**。

想一想，你喜欢的女生，有几个是邋遢、不打扮、不化妆的？你自己也不会喜欢形象不佳的异性。凤姐即便做慈善，你会因此喜欢她吗？不会。

再看“犀利哥”，他当年因颜值和独特气质爆红网络。他的脸是帅的，但为什么没有女生扑上去？因为他的**整体形象是邋遢、落魄的**。

**长得丑没关系**。“没有丑女人，只有懒女人”，这句话同样适用于男人。换一个合适的发型，刮干净胡子，修整眉毛，换一身得体的穿搭。你不一定瞬间变成明星，但至少不会依然邋遢。

在我的导师所传授的更高级课程《恋爱与武器》中，足足用了**四分之一的篇幅**来讲解个人形象改造。这不仅有助于恋爱，更能让你的生活和工作左右逢源。

无论你是太高、太瘦、太胖、太矮、太黑，还是显得太幼稚，都可以通过包装来改善。这能让一个月薪五千的人，活出月薪一万的质感；也能让月薪两万的程序员，拥有月薪十万的气场。

**案例对比：**
1.  **一位年轻程序员**：学习前是标准的“格子衫”打扮，追求一个女生三年无果。学习《恋爱与武器》进行形象改造后，曾经对他爱答不理的女生主动来找他，但他已看不上对方了。
2.  **一位41岁的销售经理**：学习前发型凌乱、胡茬邋遢、穿着无型的宅男格子衫和外套，生活环境破旧，看起来缺乏价值。学习两个月后，形象气质巨变，宛如销售总监。
3.  **我的导师熊导**：你能想象他半年约会超过200个女生吗？平均一天见一个以上。这绝非他学习前的形象所能达到的。正是通过十个月的学习改造，他才轻松完成了高强度的社交考核目标。

他们中有底子好的，但大部分单身的兄弟起点都差不多。**只要你愿意，就可以像他们一样完成蜕变，成为型男，成为受女生欢迎的男人。**

看看现在的自己：是否仍然没有一张像样的“互联网名片”？这就像**不写简历就去应聘一样荒谬**。醒醒吧，别再天真了。行动起来，从提升自我形象开始，改变你的人生轨迹。

**恋爱与武器，是男人一辈子都应该学习的课程。**
