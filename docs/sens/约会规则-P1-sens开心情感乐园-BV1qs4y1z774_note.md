# 约会规则 - P1 - 当断则断：把握关系升级的关键时机 ⏳

在本节课中，我们将学习在约会中如何识别并果断把握关系升级的关键时机。核心在于理解“当断不断，必受其乱”的原则，学会在尊重对方的前提下，大胆而聪明地推进关系。

上一节我们介绍了约会中的基本互动原则，本节中我们来看看如何在实际互动中识别并行动。

所谓“当断则断”，是指在约会中，当女生表现出快乐和开心的瞬间，你不要因为犹豫不决、唯唯诺诺或不够大胆，而丧失推进关系的机会。机会往往转瞬即逝。一旦错过那个关键点，女生可能就不再想将关系升温。但如果你在那个时刻把握住了，成功迈上新的台阶，那么你和女生的关系就将开启一个全新的阶段。

然而，你的大胆进攻和主动激起，必须建立在一个核心原则之上：**尊重女生**。很多学员在听到这个观点后，会变得非常大胆积极地去推进关系，例如想着“要牵手就去牵”，结果一牵手就被女生甩开。有些人会非常固执，认为“必须当断则断”而继续尝试，这反而会让感觉变差。

关键在于，你是在女生开心的瞬间做出判断，用推进关系来试探女生的感觉。例如，你去牵她的手，如果她不愿意，那么**马上尊重女生**的选择。退回来，继续维持开心快乐的氛围。这说明她当前的“窗口”并不适合进行这一步。

如果女生的拒绝并不坚决，那又说明什么呢？这时你也不应坚持。这说明她可能更适合接受暗示。例如，你尝试牵手，她说“不要这样”，你可以马上退回来说：“对了，之前提到的那家餐厅，我们周五有空去看看吗？” 这便是一种聪明的迂回，换用暗示的方式继续互动。

当今女性的思想观念普遍比较独立。因此，你不妨学会更直接地表达自己的一些想法。很多男生喜欢沿用古时候那一套，两个人互相暗示、暗送秋波。但现在的年轻人，尤其是00后，往往更加直接。很多时候，感情正是在彼此的试探和矜持中慢慢降温的。

感情互动有一个有趣的现象：如果你抱着一种积极主动的态度，对方很可能感受到并回应以积极；如果你抱着消极被动的情绪，对方也可能变得消极。因此，作为男生，在把握好机会、识别出对方需求时，要**大胆进攻，勇往直前**；而在遇到抵抗时，则要**聪明地迂回转折，尊重对方**。

做到以上几点，恋爱便没有想象中那么难谈了。
