# 短期关系攻略（一）：核心认知与破冰话题 🚀

在本节课中，我们将要学习短期关系的核心认知，理解其独特的“快”的本质，并掌握建立初步联系与安全感的关键话题技巧。

## 概述：什么是短期的“快”？

上一节我们介绍了课程主题，本节中我们来看看短期关系的核心特点。短期的“快”和大家理解的快不一样。短期想要玩好，一定要保持自己的不上头。

你自己处于一个不上头的状态比什么都重要。因为你只有在不上头的状态，你才能够清晰地判断你们所处的阶段。你只有在不上头的阶段，你才能够真正意义上实现“慢便是快”。目前来讲，往往短期比较厉害的人，其实是不会把骚话这些东西挂在嘴边的，也不会去提前给女生打预期。甚至玩短期厉害的人，她不会去很兴奋很亢奋，而是一个很自然而然的状态，就像水这样流下来那么自然。绝对没有一些应急操作和我们网上看到那些精彩的操作。

那些都是搞笑的。能够在网上那样操作还存活下来的女生，要么这个女生的质量很低（不管是她的社会地位还是她的颜值身材都比较低），要么就是这个女生真的窗口特别大（可能刚结束了一年的恋爱长跑，然后已经单身半年了）。只有这两种情况才能用网上那些东西奏效。所以为什么你聊没用，别人聊有用，是因为别人找到了这样窗口大，或者说质量低的女生。所以你们不必羡慕。你看到这样的聊天记录，你又要知道，这个女生的质量要么不高，要么就是窗口很大、运气好。

## 短期“快”的本质：抓住心理变化

那我们来继续讲什么叫短期的“快”。短期的快，它主要是体现在一种自然的情况下。那么想要快，最重要的是抓住女生心理的一个变化。主要有两个阶段，我给大家讲清楚，大家就明白了。

以下是建立短期关系必须经历的两个核心心理阶段：

1.  **安全感和舒适感的建立**：想要玩短期，一定是对你这个人有一个模糊的大概的了解，并且建立一个认同。为什么我说安全感后面要加一个舒适感？就是我跟你相处很舒服，我跟你相处很自在，我跟你相处很轻松。光是自在和轻松这个点就是大部分兄弟需要去研究的。就怎么样让对方处于一个舒服的状态，处于一个自在轻松的状态。因为只有这样他才能够去打开自己。只有打开了自己，你才会成为短期的那个预选项。就是你能够从普通人、从陌生人、从网友这些身份上，跳到我们说的短期预选对象。那你只有把这第一点做好了才可以。

2.  **关系升级的试探与推进**：在建立了初步的舒适区后，需要通过特定话题进行试探，获取关键反馈，从而判断窗口并决定是否及如何推进关系。

## 如何建立安全感与舒适感？

上一节我们介绍了短期关系的两个核心心理阶段，本节中我们来看看如何具体建立安全感与舒适感。

我们就讲两个点：安全感和舒适感。

*   **安全感**：安全感就是你的朋友圈和你自己的自我介绍。就是我要知道你大概是一个什么样的人，什么样的工作。当然这些东西不要深度聊，因为本来短期就没有必要聊那么多。所以说短期要聊的其实很简单，就是建立一个大概的人物画像。就好像我们对女生一样，因为她喜欢穿黑色，工作是老师，那我大概率就可以判断：我对她的安全感来自于，她是一个老师，大概率是从事正当行业的，那么她肯定不会做酒托这样的一个行为，肯定也不会做仙人跳这样的一个行为。那一米七穿黑丝，那我瞬间我就会觉得是个优秀的女生，我会把她放在我的备选项里。你要考虑这个点，这个是安全感的问题。你得去提供一些信息。这种信息最好不要去聊，因为短期的窗口稍纵即逝。所以说你有一个好的展示面是有助于你去玩短期的。这个是第一个点。

*   **舒适感**：第二个点就是舒适感。那舒适感这个东西想要去建立，那这个就比较困难。就是首先你自己要处于一个自在的状态。就比如说你能不能够做到在线下约会的时候，很轻松地跟对方聊到性话题上，而且不是那种很刻意很套路的去聊，而是很轻松很自在的去聊。就是你自己要把自己合理化这个工作做好。就是你自己要把自己合理化成一个“这个话题是很轻松平常的”人。只有当你自己把这个事情当得很轻松平常，对方才会轻松平常。如果你都觉得这是一个特别的事，对方也会这样觉得。

## 短期关系的生死线：见面与约会话题

最重要的部分来了。就是当你建立了对方对你的了解，有了一定的安全感，也建立了这样一个吸引（比如你身上有一些优点，生活比较不错，你住五星级酒店，这些都是加分项。五星级酒店也不贵，三四百块钱，你去一次打一次卡就行了。然后你平常你该住什么地方住什么地方，你带女生去的时候稍微花一点血本，花个三四百块钱，没问题），这里我们就要讲到整个短期最重要的一个环节：见面和约会话题。这个是决定了生死的重要点。

以下是短期约会话题展开的三个步骤：

1.  **变熟悉**：先变熟。想要去变熟就是加强联系。比如：“你今天坐什么车来的？”“想喝咖啡吗，还是想喝奶茶？”“能不能喝热的还是喝冷的？”“最近忙不忙？”“我最近怎么样怎么样，跟你出来见面挺开心的，你本人比照片好看太多了。”这些话都是在拉近距离。拉近距离，就是先去给线下的一个环境做一个铺垫。就是你自己要像我刚才的状态一样，很自然，很轻松。这是第一个点“熟”这个阶段的话题，大概就是这些东西。你也可以加入一些自己的理解，或者说你想象一下你跟朋友见面怎么聊的。

2.  **做试探**：第二个阶段就是试探。那这里就是情感话题。这个重要的地方是**持动**。这里的情感话题可以聊很多，比如感情经历、对感情的看法、对于长期短期的看法、对于结婚的看法，以及说是对于我的一些看法、对于未来关系的一些期待，这些东西都可以聊。但是这个时候你要注意他的潜台词，不要把它误判成长期了。那这个地方最重要的是持动。当你去聊到了，如果对方回应很积极，你可以去聊更深的话题。例如：“我前几天看了一个报道，他们说在中国，大部分女生在性生活里，她是在伪装的，就是她并不快乐并不开心，其实都是在迎合男生，甚至有的时候她去发出声音也是在假装，我不知道你怎么看？我觉得我身边朋友倒还没有怎么样，但是我不知道他们是不是装的，我看起来他们好像很幸福。”就比如说去聊这些话题试探。重点不是要通过这些事情得到结果。很多人会觉得我做了这件事情之后，我就能取到什么样的结果，不要去这样想。你做这件事情就很简单：得到他的一个信息，得到他的一个反馈。比如他很开放地跟你聊关于性生活这件事情，那你就要知道他其实是持有一个开明态度的，那你就要知道接下来就很有戏。如果说他是聊“这个我不太知道啊”，或者说拒绝沟通，那你就要知道，要么他对你没兴趣，要么他就不是短期的目标。他会觉得在这个时间点聊这个太超前了。你重点是获得反馈。
