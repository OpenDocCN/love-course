# 道长的思考力-层级分析如何拆解并理解一套复杂事物 - P1 🧠

在本节课中，我们将要学习一种强大的思维工具——**层级结构**。我们将了解它的定义、三种核心形态，并通过具体场景学习如何应用它来拆解和理解复杂事物。

## 什么是层级结构？ 🏗️

上一节我们介绍了本训练营的四个基础结构，本节中我们来看看第一个：**层级结构**。

层级结构的主要用途是理解复杂概念，或向他人解释一个复杂领域。这一讲的核心任务是让大家知道什么是层级结构并学会使用。

如何理解呢？我们先来看一个场景。你的孩子小明刚上初中，有一次生物课上，老师给他讲到大脑，但说了一堆复杂的概念，小明很难记住老师的内容。他沮丧地回到家。问题是，你如何帮助他理解什么是大脑？

大脑就是一个非常复杂的概念。人类五千多年历史，有无数人试图掌握大脑的秘密。直到上世纪50年代，美国神经科学家保罗·麦克莱恩提出一套大脑层级假说，才真正让公众了解了大脑结构。这就是著名的三脑假说理论。

在这套假说中，大脑被分为三个层级。

以下是大脑的三个层级：

1.  **第一层叫爬行脑**，也就是图上红色部分。这部分大脑主要由脑干、小脑和延髓等部分构成，基本负责生物体的呼吸、心跳、遭遇危险、欲望等最基础的生物本能。这张图就是青蛙解剖后的大脑，几乎大多数爬行动物的大脑就只有这个部分，所以叫做爬行脑。
2.  **第二层也叫边缘系统**。它大体由负责释放紧张情绪的杏仁核、释放生长激素的脑垂体、负责短暂记忆的海马体、过滤所有感知的丘脑等等部分组成。边缘系统也被称为哺乳脑，主要负责动物的情绪、记忆和感知。
3.  **第三层则是前额叶**，也就是绿色的部分。人体大脑的主要部分，就是额叶、枕叶、颞叶和顶叶四个部分组成。前额叶则是负责思考和理性的部分。这是少数灵长类动物才有的部分，也直接导致了人类和其他动物不同的原因。

这三个层级基本按年代不同而进化。爬行脑大致产生于十亿年前，那时候地球是爬行动物的天下。而在两亿年前，更多哺乳动物逐渐进化出了边缘系统。前额叶的诞生大致发生在一千万年前。

把三层大脑从具象到抽象，可以得到一个倒立的三角形层级结构。它们分别负责**本能、情感和思考**。

所以，用层级结构可以很快解释一个复杂概念。

## 为什么需要层级结构？ 📈

但为什么你一定需要用层级结构去进行理解？

我们在纸上画一条横轴，这是时间方向。然后再画一条纵轴，这是复杂度的方向。这个世界有很多领域，都是从简单到复杂再到超级复杂的过程，它们成指数级的增长。比如物种从单细胞到复杂人体的进化、人类社会的形成、科技的进步、一家公司的扩张、一个人人生的变化，都遵循这个规律。

这时候一个局外人，想要理解超级复杂的领域，就会形成巨大的信息落差。想要缩小落差，你并不需要去理解所有信息，而应该掌握事物从简单到超级复杂的形成规律。这个形成规律正是**层级结构**。

但从简单到复杂的进化规律，则包括三种不同形态。

以下是层级结构的三种核心形态：

1.  **第一是叠加形态**。
2.  **第二是构成形态**，也是涌现的形态。
3.  **第三是分化形态**。

我们分别来看。

**第一，叠加形态**，也就是刚刚说到的大脑进化规律。大脑的进化并非是一个物体，一旧换新的过程，而是不同年代的层层叠加过程。叠加形态通常指新事物直接叠加到旧事物之上。

大名鼎鼎的马斯洛需求金字塔，也就是一个叠加形态的层级结构。按马斯洛的观点，每个人需要满足下层需求，才会更多去满足更上层的需求。

**第二，构成形态**。比如几个人能组合为家庭，很多家庭能组合成不同地区社会，而更多的社会又组合成一个国家。构成形态就是指更小的部分，组合成更大的部分。

人体的进化形成也可以看作是一个构成形态。很多细胞组合成生物大分子，从而构成不同的脏器。而不同的脏器也会组合成人体的八大生命系统，比如胃部和肠道的组合形成消化系统，大脑和身体各个神经又组合为神经系统。八大生命系统最后又组合成人类。

**第三，分化形态**。比如一家公司的设定，最开始只有一个创始人，他什么都能干。但后来一个人分化出产品部、市场部和财务部。而产品部则可能分化成技术部、产品设计和供应链的部门。分化形态是指事物从一个部分，逐渐分化和分裂出很多部分。

大家最习惯使用的思维导图，就是一种分化形态。

## 如何使用层级结构？ 🛠️

那么我们如何去使用层级结构？你需要掌握的核心，当然就是确定这个事物形成的具体形态是什么。

我们用两个场景详细说明你如何使用层级结构。

### 场景一：清晰的自我介绍

**场景一**，你第一次参加一个线下读书会，在一家好看的咖啡馆，现场大致十位成员。主持人要求每一位成员都做一个自我介绍。问题是，你如何做一次清晰的自我介绍？

每个人做自我介绍，当然就是弥补对方和你之间的信息差过程。直接介绍姓名加职位，就会显得平淡无奇。而你的核心正是应该去展示自我层级的形成。

比如你可以重复自己的某类特质、某个惊人的成果，或一个夸张的目标。它们刚好对应着三种不同的形态。

以下是三种自我介绍的方法：

1.  **第一，重复自己的某类特质**。比如这就是我第一次应聘咨询公司时的自我介绍：“我今年23岁，是一个很平凡的人，不是富二代，没有耀眼的学历，也没有去过出名的500强公司。所以我唯一值得依靠的东西，只有努力。我的努力主要指一个方面：大量研究。比如我母亲在我12岁时因为抑郁症自杀，这导致我一直陷入不正常的成长环境中。但我从大一开始，就每天去图书馆大量研究抑郁症和心理学的相关理论。再比如我研究数据，虽然大学从来没有学过数学，但还是花了半年时间，从国外网站上一点一点学习概率论。后来我为了积累Excel的技能，花下班时间主动帮所有同事制作Excel报表。心理学，概率论，Excel技能，以及形成的积极探索的性格，让我进入了一家500强企业。”有时候即使表达无需深刻的逻辑，层层叠加就能形成很好的效果。
2.  **第二，讲一个惊人的成果**。由于公众对一个惊人的成果通常都很好奇，而好奇心驱使他们更想知道这个成果是由哪些部分组成，这就更多体现了构成形态。比如：“我写作出版了一本书，上市一年后也就卖了1000多本，但从去年5月开始销量突然10万+。大家一定很关心，这些都是什么原因而构成呢？”乔布斯在第一次发布iPhone的时候，也是使用了这个结构：“iPod，iPhone，你懂吗？这不是三个不同的设备，这是一个设备，我们称之为iPhone。”
3.  **第三，一个夸张的目标**。说出一个看上去不太容易完成的目标，接下来大家则关心你如何分解。比如，埃隆·马斯克的目标是把100万人送上火星。按照他的商业计划，这个目标就需要两部分做支撑：第一是交通，也就是如何把100万人送上火星；第二是居住，也就是人类到了火星以后该如何生存繁衍。而在交通方面，问题又集中在火箭技术和新能源两方面。所以你会看到，马斯克这十年一直在研究火箭回收技术，以及致力于寻找更好的能源方案，以替代效率低效的化石能源。而在火星生存方面，首先需要改善的就是火星环境，特别是温度。最好的几个方案，就是搬运甲烷以形成温室效应。当然马斯克也说，在火星南北极引爆核弹，从理论上来讲，也可以形成更好的温室效应。尽管目标非常夸张，但如果你能做到层层分化，也能形成很好的效果。

### 场景二：搭建心理学知识结构

我们再来看第二个场景。

**场景二**，你的母亲最近退休并步入更年期，时常情绪崩溃。由于多次吵架，她和家人的关系也越发紧张。而你一直想系统学习心理学，这次更想用心理学帮助母亲走出危机。那么问题就是，你如何搭建心理学结构呢？

心理学当然可以有很多层级结构搭建，但要解决母亲焦虑情绪这个目标，首先你需要深刻理解什么是焦虑。

一个人很多时候都需要生活在心理边界内。比如在空间上，你习惯去熟悉的咖啡馆，回到家躺在舒适的角落。在信息处理上，你也更喜欢熟悉的朋友，聊熟悉的话题，看自己感觉舒适的或是非常熟悉的视频。

但有时候我们又不得不处在心理边界外，这时候你就来到焦虑区。比如上级让你陪她去一个陌生的地方、公司让你处理很多很难完成的任务、你即将学习一些比较难的课程，这些都会让你处于焦虑区。

焦虑区也有边界，那就是认知边界。你对一件事有认知且能搞定，那就是在舒适区。你对一件事有认知但暂时不能搞定，那就处于焦虑区。所以这个时候一个人会拼命想去搞定某件事，获得控制感。你对一件事没有认知且无法搞定，那就处于恐慌区。这时候一个人会选择逃避或彻底屏蔽掉外物。

回到你母亲进入更年期这个案例，其本质在于，由于社会角色的转变、社会快速进步对于自身控制感的不断降低，这会导致很多老人时常处于焦虑区和恐慌区中。

在焦虑区的典型特征是不断追求控制感，所以他会变得非常无理，会打着关心关怀的旗帜，更多去干涉其他人的生活。而在恐慌区的典型特征则是完全封闭，这是很多老人变得固执的根本原因。

这时候你需要做的就是尽量让他回到舒适区之内。比如多让母亲分享年轻时候的故事，特别是很有成就感的故事。多让母亲待在熟悉的环境，而非让她住在复杂的城市中。

所以，**舒适区 -> 焦虑区 -> 恐慌区**，这就是一个叠加形态的层级结构。

当然如果你想更深入了解心理学，则需要把这个层级结构不断展开。比如在深入了解心理边界是如何产生、什么是情绪、如何通过心理对话技术，让对方回归到舒适感之中。

## 总结 📝

本节课中我们一起学习了**层级结构**。

*   它的主要用途是**理解复杂概念**或向他人解释复杂领域。
*   层级结构有三种核心形态：**叠加形态**、**构成形态**和**分化形态**。
*   应用层级结构的关键在于**确定事物形成的具体形态**，并据此进行层层拆解或构建。
