紧张时产生的多余小动作，会在潜意识沟通中暴露低价值，导致吸引力消失。

    *   **错误**：长时间将手插在口袋里。

    *   **错误**：为了用肢体语言而胡乱、快速地做很多手势。

这些多余动作多源于紧张。需要有意识地关注，一旦发现立即停止。通过多次有意识的纠正，问题自然会得到解决。
