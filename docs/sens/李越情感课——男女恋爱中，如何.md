# 李越情感课 P1：男女恋爱中，如何说话不伤人？ 💬

在本节课中，我们将学习如何通过“漂亮的说话”艺术，在与异性沟通时，既能表达真实想法，又不会伤害对方感情，甚至能促进双方共同成长。核心在于掌握一个包含五个步骤的沟通公式。

上一堂课我们探讨了如何与异性沟通交流。现实中，我们遇到的并非总是满意的对象。这时，我们需要通过语言沟通，检验对方是否具备成长性，并引导其共同进步。因此，语言表达方式至关重要。

说话本质上是一门艺术。要达到这种艺术效果，需要了解世界上每个人的表达方式不同，但能被广泛接受的方式是相通的。我们的目标就是提炼并运用这种通用的沟通方式。

现实中有许多女生说话过于直接。例如，可能对追求者说“你长得太丑了，但如果你变帅我们可以谈恋爱”。这种表达方式极易伤害对方自尊，引发抵触和怨恨，从而错失一个潜在可发展的对象。

因此，我们需要学习如何说出可能伤害对方的话，同时让对方感到得体、体面，而非受到伤害。这有一个具体的公式。

## 核心沟通公式 📝

说这类话有一个公式，包含以下五个要点：

1.  **先给对方打预防针**
2.  **给对方戴高帽子**
3.  **表现出自己的决绝和坚定**
4.  **说出你想说的话、道理，并指出对方的缺点和不足**
5.  **让对方自己做选择（关于双方关系）**

接下来，我们将详细拆解这五个步骤。

### 第一步：先打预防针 💉

在沟通中，我们常因害怕对方生气、难过或失望而不敢直言。这种恐惧对应着对方可能产生的、我们不愿面对的行为。

所谓“打预防针”，就是提前将对方可能产生的、我们无法接受的行为告知对方。也就是“预判对方的预判”。

例如，当你想反驳一个人的观点时，不要直接说“你说错了”。可以这样说：“我心里有些不同看法，但不知该不该说，我特别害怕你听了会生气或觉得我在装腔作势。”

在邀请别人时也可以使用：“其实我早就想约你吃饭了，但一直害怕被你拒绝。”这样说，对方通常难以拒绝，因为你已经预判了他的反应。

常见的口头禅如“说句不好听的话，你别生气”、“我接下来说的话可能会让你难过”，都是在打预防针。提前打好预防针，可以避免对方出现你无法接受的激烈反应。

人有一种奇怪的心理：一旦内心被人猜中，往往就不会按照对方预期的糟糕方式去行动了。

### 第二步：给对方戴高帽子 🎩

打完预防针后，接下来是“戴高帽子”。既然预判了对方可能会有不好的反应，为什么还要说呢？因为在你心中，认为他是更优秀的人。

具体做法是列举对方的一系列优点。例如：“我觉得你是一个宽容、成熟、稳重的人。”

结合第一步的案例，可以这样说：“你讲的观点，前面部分我觉得很有道理。但我后面有些不同看法，又不知该怎么说，怕你生气。不过，正因为我觉得你是一个有思想、成熟稳重、能理性思考的人，不会像别人一样一听不同意见就吵架，所以我才想跟你分享我的看法。”

这就是说话的艺术：给对方戴高帽子，把他捧到一个高度，让他自己在心中树立起一个“通情达理”的人设。人们都对自己的优点有不切实际的乐观，喜欢被认可。这样做，他会觉得你非常懂他。

### 第三步：坚定地表达自己 ✊

在打完预防针和戴过高帽子后，需要坚定地表达自己的意图。

可以这样说：“正因为你是这样（通情达理）的人，所以我决定一定要把我的观点说出来。我不想埋藏在心里，我认为人与人之间的沟通需要真诚。而且和你聊天很愉快，总能激发我分享一些平时不会说的想法。”

这部分的核心是传递出：因为信任对方的为人，所以你才选择坦诚。

### 第四步：说出核心内容与道理 🗣️

这是表达的核心部分，说出你想说的话、相关的道理，并指出对方的缺点或不足。

例如，在拒绝一个外形不佳的追求者时，可以这样说：“直白地说，我不接受你的原因，仅仅是你的外形不是我喜欢的类型。（当然，我知道这话很容易伤害男生的自尊心，请你千万别生气。）事实上，你的五官基础不错，只是在穿衣打扮、发型等方面有待提升。但你的品行、条件、言谈举止，我都很欣赏。”

请注意，在指出缺点时，可以再次嵌入“打预防针”（如括号内内容），以缓和冲击力。

### 第五步：让对方做出选择 🤝

最后，将关系的选择权交给对方。

可以这样说：“我非常愿意帮助你，可以陪你挑选衣服、换发型，帮你提升外在形象。在这个过程中，如果你进步了，我依然没有喜欢上你，那你也会因为变优秀而吸引其他女生，我也可以把我身边不错的姑娘介绍给你。如果我们最终能互相喜欢，那皆大欢喜。”
“所以，如果你真的喜欢我，并愿意进步，不认为我是在打击你，我希望我们可以先以朋友的身份相处，共同成长。我说这些，不像其他女生那样只是敷衍地发‘好人卡’，因为我觉得你非常棒，值得听到实话，也值得变得更好。”
“当然，你是否愿意和我以这样的方式相处，需要你自己认真思考后做出选择。”

以上就是完整的公式应用。整个过程严格遵循了：打预防针 -> 戴高帽子 -> 坚定表达 -> 说出核心内容（含道理与不足）-> 给予选择权。

## 公式的深层应用与价值 💎

上一节我们介绍了完整的沟通公式，本节中我们来看看如何将其应用于具体场景，并理解其背后的深远价值。

假设你需要拒绝一位外形一般但人不错的追求者，并希望保持友好关系，可以综合运用上述公式。

**以下是结合公式的拒绝话术示例：**

> “我知道你喜欢我，有些话想对你说，但又特别害怕说完后你会生气，觉得我肤浅或是个坏人。（打预防针）”
>
> “但相处下来，我觉得你是个非常好的男生，人品、教养各方面都很好，我很欣赏你。（戴高帽子）”
>
> “所以，即便你可能生气，我还是想告诉你实话，我不想像其他女生那样随便发‘好人卡’敷衍你。（坚定表达）”
>
> “直白说，我不接受的原因主要是外形不是我喜欢的类型。当然，我知道这话很伤自尊，请你别生气。事实上，你五官不错，主要是穿衣打扮、发型可以提升。你的为人品行我真的很欣赏。（说出核心内容与不足）”
>
> “我愿意帮你找发型师、搭配衣服，陪你逛街改造形象。如果你进步后我还是没喜欢上你，你也会更吸引其他女生，我可以帮你介绍。如果我们能互相喜欢，那就更好。所以，如果你愿意，我们可以先以朋友身份共同成长。这一切选择权在你。（让对方选择）”

### 为何这样做：提供价值与“种树” 🌱

这与我们上一堂课讲的“培养男人”或“种树”能力一脉相承。通过主动提出帮助对方改造形象（如决定发型、衣着），你实际上在提供价值。

在这个过程中，因为你是为他好，而他愿意接受，他的“服从性”会增强，同时也能感受到你的善意。真正的“养备胎”不是消耗对方，而是帮助对方成长，使其因你的存在而获得正向反馈，变得更好。这样，他自然不会对你产生怨恨。

### 长远的眼光 🔭

男人的事业黄金期多在30岁以后。在他20多岁青涩、不被看好的时候，你若能给予真诚的帮助和引导，而非消耗或吊着他，他会铭记于心。

即使最终未成恋人，他也会将你视为重要的朋友。未来当他飞黄腾达时，在你需要帮助时，他很可能不会袖手旁观。因为你在他低谷时给予了别人未曾给予的尊重和帮助。

**以下是需要牢记的原则：**

*   **筛选与淘汰**：如果对方在沟通后表现出不可控、不听话或非常急躁，这恰恰说明他不具备所需的成长性，应及时止损。
*   **真诚行动**：如果你提出了帮助，就需要拿出实际行动，否则会显得虚伪。
*   **放眼未来**：人际关系不要只看眼前利益，长远的、相互成就的关系更加稳固。

## 总结 📚

本节课中，我们一起学习了“如何漂亮地说话”这一沟通艺术。

我们首先认识到，在与可能不满意的追求者沟通时，直接粗暴的表达会伤害关系，而艺术化的表达则能引导关系向积极方向发展。

**核心在于掌握一个五步沟通公式：**
1.  **打预防针**：预判并说出对方可能有的负面反应。
2.  **戴高帽子**：真诚赞扬对方的优点，将其置于“通情达理”的位置。
3.  **坚定表达**：表明自己出于信任和真诚才选择直言。
4.  **说出核心**：委婉但清晰地表达观点、道理及对方不足。
5.  **给予选择**：将关系的决定权交给对方，展现尊重。

通过这套方法，我们不仅能更得体地处理棘手对话，避免伤人，还能在实践中为对方提供提升价值的帮助，实现“种树”而非“摘果”的长远关系建设。记住，最高明的关系策略是互利共赢，共同成长。
