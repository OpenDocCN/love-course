## 表白的三种类型 📝

以下是表白的三种主要形式，它们适用于不同的情境和关系阶段。

*   **行为表白**：通过行动而非语言表达好感。
*   **口头表白**：直接通过语言说出心意。
*   **书面表白**：通过文字（如信件、消息）传达情感。
