## 咖啡厅搭讪 ☕

咖啡厅的特点是人们通常坐着不动，因此有较充裕的聊天时间，且环境相对安静。在这种安静环境中，最大的挑战是**社交压力**，因为你的搭讪行为可能会引起周围人的注意。

在咖啡厅搭讪时，一定要简单明了，并告知对方你即将离开，这样可以有效减轻对方的压力。
