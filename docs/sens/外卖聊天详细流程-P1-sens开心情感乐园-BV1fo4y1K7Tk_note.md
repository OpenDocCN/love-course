## 第一步：打招呼测试 👋

上一节我们概述了第一轮聊天的目标，本节中我们来看看如何开始。第一步是“打招呼测试”。

无论使用何种开场白（如一个表情 😊 或简单问候），目的都是为了测试女生当下的反应和对你是否有基本窗口。开场白本身没有绝对的好坏之分。

`女生回应积极` -> `存在窗口` -> `可进入下一步`
`女生无回应` -> `窗口较小或暂无` -> `需暂停或调整策略`

当我们通过第一步测试，确认女生存在基本窗口后，便可以进入第二步：右脑思维开场。

通过右脑思维开场，可以快速将对话从简单的寒暄，引导至更个人化、更易产生共鸣的方向。

## 第四步：聊天四项线做吸引 💬

当对话进入平行、融洽的状态后，就可以运用“聊天四项线”来系统性地加深吸引了。

聊天四项线是一个循环的聊天模式，包含四个要素：
1.  **正经问问题**：获取对方信息，表达关心。
2.  **正经展示价值**：在回答中自然植入自身优点或生活状态。
3.  **调侃自己**：制造轻松感，避免显得炫耀。
4.  **调侃对方**：在合适的时机进行轻微、友好的调侃，增加暧昧张力。

**公式：**
`提问 -> 对方回答 -> 展示价值 + 调侃 -> 建立连接与吸引`

通过这个循环，女生会在舒适的氛围中与你交换信息，同时感受到你的价值、幽默感和独特性。通常进行20-30句这样的对话，就能建立起较强的吸引。
