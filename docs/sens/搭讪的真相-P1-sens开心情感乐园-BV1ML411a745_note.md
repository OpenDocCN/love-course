## 核心矛盾：搭讪的重要性与“低价值”属性

上一节我们介绍了搭讪的必要性，本节中我们来看看搭讪行为本身的一个内在矛盾。

搭讪在本质上是一个价值框架较低的行为。一个有高价值的男性或女性，通常是被动接受他人的结识，他们只需要进行筛选，而无需主动出击。主动出击的一方往往处于劣势。

当你明白了搭讪的目标和本质后，接下来我们看看实现这个目标需要经历的三个阶段。

当你外貌条件不错，或搭讪成功率（如收号率）达到30%-50%时，可以进入第二阶段。此时，仅靠勇气和冲动已无法实现更高跨越。
