# 恋爱秘籍新手课 P1：为什么要正视自己的欲望 💡

在本节课中，我们将要学习一个核心心态：真诚地直面自己的目的与欲望。这是建立健康恋爱关系的第一步，也是筛选合适伴侣的关键。

## 课程概述

我们都谈论“筛选”，但什么是筛选？如果你研究过约会高手的心态，会发现他们有一个共同特点：**得之我幸，失之我命**。想要高效获得你想要的，最直接的方式就是直面你的目的，这样，不适合的人自然会被筛选掉。

上一节我们介绍了课程框架，本节中我们来看看“直面欲望”为何如此重要。

## 直面目的：高效筛选的开始

我经常在开场时说：“如果你不想认识我，我马上就走。”如果对方真心不想认识你，你何必费尽心思让他认识你呢？即便认识了，你们会相处愉快吗？

并不会。因为对方从一开始就没有认识你的意图。所以，你究竟是在享受约会的感觉，还是在编织一个谎言强迫别人爱上你？你想做一个情圣，还是做一个情兽？

说得更深一点，你学习的恋爱技巧，究竟是丰富人生的**工具**，还是限制人生的**桎梏**？

## 做真实的自己，而非他人的复制品

曾经在夜店，我精确执行教程中的每一步，毫不隐藏意图。但那时，我并不是真正的我，我只是别人方法的复制品。那样的情况下，我如何能收获想要的幸福？

所以问题来了：使用别人的方法和心态，到最后，那还是别人。我们都掩饰真实的自己，只是因为我们不敢真正接受内心，不敢让真实的自己去面对社会和世界。这非常可悲，并且可能会伤害更多人。

以下是关于诚实的几个要点：

*   如果你寻找长期关系，就告诉他。
*   如果你只为了一夜情，也让他知道。
*   如果你觉得她漂亮想亲她，也要告诉她。

这没什么好害羞的。在这个虚伪的社会中，你应该诚实。

## 在寻找真爱的路上保持清醒

或许你现在怀疑自己能否达到这种心态。不要着急，在课程介绍中我讲过，我会帮你养成一系列习惯来真正达到这种心态。目前，你只需在理论上接受它。

记住，我们要经历很多失败，才能找到“对的人”。不过要专门提醒一下：在寻找真爱的路上，你会遇到很多“夜壶”。她们或许外表闪闪发光，但内在的缺陷无法让你真正幸福。

如果一个美女天生将男女关系对立，把男人当动物对待，或天天教别人如何耍男人……当你遇到这类女人，如果被她闪闪发光的外表迷惑，并自我合理化地说服自己找到了真爱，那你只是在欺骗自己。

她们也在骗你，告诉你“你是条狗，你才是好男人”。那是她们的可耻，但如果你上当受骗，那就是你的可悲。

## 了解本质，享受成长过程

我们不需要内心不完整的女生，我们需要的是能与我们完美契合的那一个。所以，永远不要被表面东西欺骗，更不要骗自己。

对于生命中的每一个女生，你都应该真正去了解她，寻找她的真相。如果她不合适，就应该继续寻找。享受并感激这段寻找的过程，因为你在过程中不断成长，变得更加优秀。

就像我以前在课堂中讲过的：分手、失恋这些失败，往往不是结局，而是一个新的开始。它会让你遇到一个全新的、更好的、更优秀的自己。

## 回顾核心行动框架

我希望你们在练习《恋爱秘籍入门》时，任何时候都回来看看、听听上面这段话。它会让你持续走在正确的道路上。

在我之前讲到的步骤中，以下是构建吸引力的完整行动框架：

1.  **准备**：做好心态和外在准备。
2.  **开场**：掌握认识大量女生的核心方法。
3.  **吸引**：通过互动让女生对你产生兴趣。
4.  **示好**：展示兴趣，拉近关系。
5.  **邀约**：掌握正确方法，让约会更高效。
6.  **升级**：学习快速建立肢体亲密接触的方法。
7.  **应变**：学会针对不同女生调整方法，而非一成不变。
8.  **确立**：掌握确定恋爱关系与正确相处的制度。

在吸引一个女生之前，你必须满足上述一系列条件。仔细看一下这八个步骤，哪一个是你现在真正需要的？

*   你是否不会聊天？
*   你是否不会展示兴趣？
*   你和女生聊天是否总是平平淡淡、唯唯诺诺？
*   你是否能把她约出来，但关系毫无进展？

仔细看一下，到底哪个阶段是你最需要的。这些东西缺一不可，需要一步步达成最终结果。
