# 聊天技巧课程 01：情绪调动聊天法 🎭

在本节课中，我们将要学习一种核心的聊天技巧——情绪调动聊天法。我们将探讨情绪在交流中的重要性，并学习如何通过特定的语言策略来有效调动对方的情绪，从而营造更好的聊天氛围。

## 概述：为什么情绪如此重要？

聊天技巧非常重要，而情绪调动聊天法是其中的关键。人们聊天时，情绪扮演着至关重要的角色。尤其是在两性相处的过程中，男生和女生的聊天本质上主要是在交流情绪。情绪背后所表达的含义或潜台词，会极大地影响双方交流时的氛围。

为了帮助大家理解“聊情绪”为何如此重要，我们来看一个简单的例子。

## 案例分析：情绪 vs. 内容

我们和自己身边的好哥们（例如树根和狗哥）聊天，可以作为一个正面案例。我们两个人可能聊上半个小时，最后总结出来的聊天内容，足以写满两张A4纸。

假设若姨和诚哥两人去逛街。他们从下午一点逛到五六点，虽然天气很热，但两人（作为女生）逛得非常开心。当他们回来时，你问若姨：“今天下午你们都聊了什么？”她很可能不记得具体内容了。

她只会说：“我也不知道，我就感觉我们俩人今天下午逛得很开心。”实际上，女生在和男生聊天的过程中，更注重的是情绪上的调动。

换句话说，当你和一个女生聊天十分钟或二十分钟后，请仔细想一想：过了半个月或二十天，那个女生还会记得你们聊天的具体内容吗？通常不会。除非聊天的内容非常有记忆点。如果只是平常的闲聊，她根本记不住当时聊的具体内容。

但是，她会记得当天和这个男生聊天时感觉很开心。她会记得聊天中间有一个非常开心的点，虽然可能忘了具体是什么，但会记得“和他聊天很开心”这个感觉。

这个例子说明，在和对方聊天的过程中，聊天的具体内容可能并没有那么重要。更重要的是我们所说的“情绪的调动”。

## 如何调动情绪？🔄

在情绪调动的过程中，我们应该如何巧妙地应用这个方法呢？其实，我们在上一节课中聊到的“俏皮话开场”，就是用一个小的套路或技巧来开启话题。同样地，调动情绪也可以使用类似的方法。或者说，“俏皮话开场”本身就是开场过程中调动情绪的一种方法。

在和对方聊天的过程中，情绪是一成不变的吗？当然不是。我们和对方聊天时，情绪需要有高有低。我们需要让对方的情绪升起来，再降下去，然后再升起来。

换句话说，我们和他聊天时，不可以一味地称赞他，不可以一味地让他感觉我们在交流过程中是在“舔”着他。情绪有上升也有下降，这才叫做“调动”。

再换句话说，我们可以怎么做呢？就像老话常说的：**给一个巴掌，再给一颗枣**。也就是“给个巴掌给个枣，给个巴掌给个枣”，最后“枣升柜子”。

我们可以先尝试聊一些看起来像是会贬低他的话，但后面话锋一转，其实我们的目的是夸他。又或者，刚开始我们其实是夸他，但转过头来，我们又贬低一个不那么重要的“小底”来中和这个效果。

## 情绪调动的三大目的 🎯

我们这么做，真正想要达到什么效果呢？主要有以下三种目的：

以下是情绪调动聊天法的三个主要目的：

1.  **隐藏需求感**：当我们想夸一个女生时，如果找不到好的切入点，用这种方法可以避免过多暴露我们的需求感。不会让对方觉得“你就是在夸我，我身边那么多人夸我，你也在夸我，你这个男生真奇怪”。
2.  **打破尴尬，重新破冰**：如果之前的聊天比较尴尬，我们可以用这样的方法来重新破冰。
3.  **制造暧昧，用于调情**：这种方法可以起到暧昧和调情的作用。

## 实战案例解析 📝

如果只是单纯地聊理论，朋友们可能不太好理解。我们和之前一样，拿一些我们这边学员的真实案例来和大家具体分析一下。

### 案例一：破冰与重启对话

这是一个用于重新开启话题、进行破冰的方法。他们两人机缘巧合，在大润发有一次小小的偶遇。他就借助这个话题，重新开启了两个人的交流。

以下是具体的聊天步骤：

*   **第一步：确认事实，建立连接**
    *   男生说：“你昨天在大润发看到我了吧，哈哈。”
    *   对方给予认同的回答，证明两人没有看错。
*   **第二步：制造情绪波动（“给巴掌”）**
    *   男生接着说：“我看到的是狗哥，不是一个奇怪的小狗子。”对方说：“对啊，我去买零食。”
    *   然后男生进行了一个看似“贬低”的举动：“先通过说两人照片不太一样，在一个这样的挖鼻孔，迷惑的好像是有一点贬低意思的这样一个内容。”
    *   当我们发出这样一个信息时，对方会怎么想？她会想：“哎，你这么说想怎么？你想说我长得丑吗？你想说我P图P得太过分吗？”这会让对方产生负面情绪，或者让情绪“荡下来”。
*   **第三步：情绪反转与提升（“给枣”）**
    *   这个时候话锋一转，他说：“哎，其实你本人好看多了。你本人要比照片好看多了，你个照片杀手照片骗子。你为什么把照片拍得那么丑？”
    *   他用类似这样的方法，将前面的话题圆回来，带来情绪的上升。这就是脱口秀和相声里常说的 **“恐惧解除”** 过程。
*   **第四步：效果达成**
    *   对方马上明白了：“哦，原来你前面说的并不是说我长得丑，而是说我长得比照片好看。”这会给予对方一个正面的反馈。
    *   这个时候对方的情绪已经得到了调动。她后面的回复就像看完一个惊恐画面后松了一口气的样子，比如：“哎呀，你刚刚差点，你在这个红色感叹号的边缘徘徊。”
    *   这样一来，我们聊天的思路就有了发挥的空间。

### 案例二：低需求表达好感

如果我们想对对方表达好感，这种低需求的情绪调动方法也可以做到。大家可以看一下这个聊天记录。

以下是具体的对话分析：

*   **背景**：男生先问：“你明天忙完要准备去哪？”对方说：“我要去我亲戚家吃饭，我被我妈念了很多遍了。”两人在正常闲聊。
*   **情绪调动尝试**：男生接着说：“那要不，哎，我来拯救你吧，明天带你吃大餐。”对方提出了障碍：“我爸来接，那你怎么拯救呢？”
*   **关键回应**：这时男生说：“你不会说那个凶神恶煞的叔叔吗？”
*   **分析**：正常情况下，男生可能会想：“哇，对方的亲人，如果我说的不太好，会不会马上搞砸？”所以可能会好好表达一下。但此时做一个小小的情绪调动，表达了我们并不像其他人那样，需要通过不断示好或“舔”的行为来接近对方。我们也有自己的主见。同时，这也做了一个小测试，因为对方的言下之意是：“如果不是因为这个原因（爸爸来接），其实可以考虑跟你出来。”

## 核心逻辑与总结 📚

以上是两个学员的聊天案例，它们各有特色，但最终都实现了情绪的传递。让女生能在冷冰冰的文字中（因为微信交流很难通过文字直接表达情绪），感受到你或让你感受到她的情绪。这能整体提升我们聊天的气氛。

我们在和对方聊天的过程中，可以沿用这个方法：不断地调动情绪，让对方有更好的情绪波动。目的是什么呢？目的其实之前聊过：**交换价值，调动情绪，升级关系**。我们必须先调动情绪，才能进一步升级关系。

很多朋友在和异性交流时存在一个问题：可能只是单方面想给对方留下好印象，让对方觉得这个男生聊天很礼貌、说话好听。但如果仅仅是这样，最后得到的效果往往不尽人意。

第二点，我们不得不谈到在和异性交流过程中需要考虑的一条逻辑链。现在大家可以拿出笔和纸记一下：

**情绪 -> 潜台词 -> 反馈**

我们在和异性交流的过程中，**情绪**是一个点，**潜台词**是一个点。我们需要根据对方的情绪和潜台词，来给出相应的反馈。而在潜台词中，有一个非常重要的工具，就是我们过一会会讲的——**关键词聊天法**。

本节课中，我们一起学习了情绪调动聊天法的核心概念、应用目的以及实战案例。记住，聊天的内容会遗忘，但聊天时产生的愉快情绪会被长久记住。学会先“给巴掌”再“给枣”，制造情绪的起伏，是让对话变得生动有趣的关键。下一节课，我们将深入探讨如何捕捉和利用聊天中的“潜台词”。
