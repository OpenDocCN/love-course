# 07.约会攻略：带入私密空间的理由-P1-sens开心情感乐园-BV1Jj411g732_note

在本节课中，我们将要学习如何构建一个合适的理由，将约会对象带入私密空间。理解并运用合适的理由，是提升约会成功率的关键一步。

上一节我们介绍了私密空间的种类，本节中我们来看看如何为进入这些空间找到一个自然且让对方安心的理由。

你可能会看到一些案例，其中带入私密空间的理由非常直接。甚至像“我今晚要XX你”这样非常直白的表达。但是，这种直接的方式通常显得很暴力。

我要告诉大家的是，你不能只看到这一句话，而是应该看到具体的案例背景。一般来说，如果对方的服从性越高，你带入的理由就可以越直白。如果说对方的服从性越低，你带入的理由就要越隐晦。

我再说一遍，就是对方的服从性越高，你带入的理由就可以越直白。如果说对方的服从性越低，你需要带入私密空间的理由就更隐晦。明白这个道理吗？这是一个很简单的道理。

就是如果说对方愿意跟你走的话，你有什么理由都可以跟你走。如果说对方不是那么愿意走，你就需要稍微有一些借口，给对方一些台阶去把对方带走。

以下是具体的一些例子：

例如，你跟对方约了晚上十点钟的电影，地点在你家附近。假设你们九点钟见面，见面之后你就可以跟她说：“我忘了我家里要遛狗，要不你陪我先回去遛一下狗，我们遛过狗之后再回来去看电影。” 这就是一个很直接的、带入你私密空间的理由。因为你们接下来还有约定的电影时间限制，你现在要回去先遛个狗，然后再出来看电影，这个理由可以让对方感到安心。

还比如说，你们决定“我们去吃宵夜吧”。到了目的地却发现那个地方没开门。这时你可以说：“那不如我们上去吃，或者我带你回家吃点东西再送你回去。” 你带入私密空间的理由，要尽量让对方安心，那你带入私密空间的成功率就越大。

大家明白这个道理吗？
