# 课程名称：情感框架构建指南 - P1 🧱

在本节课中，我们将要学习人际交往中“框架”的核心概念。框架是你与他人互动的边界和规则，掌握它对于建立健康、有吸引力的关系至关重要。我们将从三个核心组成部分——标准、一致性和筛选——来深入解析如何构建并运用你的个人框架。

## 什么是框架？🤔

上一节我们介绍了课程主题，本节中我们来看看框架的具体定义。框架的本质是树立你与其他人交往的边界。它决定了互动中的主导权、规则和底线。要想有效掌握并运用框架，我们需要从以下三个核心点出发。

## 框架的三大支柱

以下是构建稳固情感框架的三个关键组成部分。

### 1. 标准：明确的边界与弹性

框架的基石在于拥有清晰的标准。标准的核心在于**明确的喜好与厌恶**，避免模棱两可。例如，你喜欢自信的人还是自卑的人，必须有明确的答案。你的标准越模糊，你的框架就越软弱。

**为什么需要标准？** 这可以与“符格”（合理化解释）结合。标准是你“做什么”，符格是“为什么这么做”。例如：
*   **标准**：我认为男生为女生花钱是爱的体现。
*   **符格**：因为花钱的行为代表了愿意付出的态度，这能让我感受到被爱和安全感。

关于标准的第二点，是理解 **“外圆内方”** 的概念。想象一个球体（圆）内部包裹着一个立方体（方）。
*   **方形**：代表绝对不可逾越的原则和底线，**对事不对人**。例如，无论关系多亲近，你都不会在身上纹对方的名字，因为感觉被物化。
*   **圆与方之间的区域**：代表有弹性的边界，**对人不对事**。这里的规则可以根据关系的亲疏远近适当调整。例如，“我只会为我的女朋友花钱”。对方目前不是女朋友，所以规则不适用；但当对方满足条件（成为女朋友）时，规则就会弹性变化。

**核心公式**：**标准 = 明确规则 + 合理符格**。框架不是死板的墙，而是一部分坚固、一部分有弹性的边界。

### 2. 一致性：言行合一的奖惩机制

上一节我们定义了标准，本节中我们来看看如何维护它，这就需要一致性。一致性最简单理解是：**积极奖励积极，消极惩罚消极**。

以下是维护一致性的两个要点：

1.  **奖励积极**：当对方展现出迎合或兴趣时，及时给予正面反馈（奖励），并提供让对方继续付出的空间。例如，对方说“我想你了”，你应回应以热情，而非冷淡。
2.  **惩罚消极**：当对方犯错或挑战你的底线时，要及时指出并给予改正的机会。如果对方屡教不改，你需要明确告知后果：“你这样做，可能会失去我。” 这传达出你是一个即使失去对方也不会委屈自己的人。

**核心代码**：
```python
if 对方行为 == 积极:
    给予奖励()
elif 对方行为 == 消极且非故意:
    给予警告并预留改正空间()
else: # 对方行为 == 消极且故意/屡犯
    执行惩罚(可能包括离开)
```
丧失一致性（如嘴上说“只给女朋友花钱”，行动上却为普通朋友花钱）会让他人觉得你口是心非，框架软弱，从而不再尊重你的边界。

### 3. 筛选：基于价值的审慎评估

掌握了标准和一致性后，我们来看看如何运用框架去选择合适的人，这就是筛选。筛选的核心是：**处在“考虑”状态，而非“到手”状态，去深入了解对方的价值以判断是否满足你的要求**。

这两种状态的区别如下：

*   **考虑状态**：你像一位谨慎的消费者。例如，你想报舞蹈班，会去了解不同机构的师资、价格、环境等信息，再做出选择。在人际中，这意味着你真诚地想了解对方的性格、价值观、生活态度，以判断他/她是否适合你。
*   **到手状态**：你内心早已认定目标，所谓的“筛选”只是走过场。就像一进店就决定办卡，不再关心其他信息。在人际中表现为“查户口式”提问（你多大了？做什么工作？），而非深入交流。这本质上放弃了筛选权，因为无论对方如何，你都已经“决定购买”。

**核心要点**：有效的筛选要求你始终保持“考虑”心态，通过深入交流评估对方，并敢于将不符合你标准的人筛选掉。你的提问应旨在探索对方的内在价值，而非仅仅填充信息表格。

## 总结 📝

本节课中我们一起学习了构建情感框架的三大支柱：
1.  **标准**：确立明确且带有合理弹性的个人边界（外圆内方）。
2.  **一致性**：通过“积极奖励积极，消极惩罚消极”的机制，坚定不移地维护你的标准。
3.  **筛选**：保持在“考虑状态”，基于深入的价值评估来选择交往对象，而非被冲动驱使进入“到手状态”。

掌握好这三点，你就能建立起清晰、有力且灵活的个人框架，在人际交往中占据主动，建立更健康、更令人尊重的关系。
