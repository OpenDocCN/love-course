# 为什么他总说自己是好男人，却从

在本节课中，我们将要学习探索核心需求的第三个思路：分析一个人所在家庭的文化传承。我们将了解这些传承的理念如何固化个人的行为模式，以及它们可能掩盖了哪些深层需求。

上一节我们介绍了从个人经历探索核心需求，本节中我们来看看家庭文化这个更宏观的背景。

文化传承是心理学表达性治疗的专业术语，即我们通常所说的家庭教育的核心理念。这些核心理念会固化每个人的“超我”。“超我”即我们每个人道德约束的部分。

公式表示：**人格结构 ≈ 本我 + 自我 + 超我**

当“超我”过于强烈后，“本我”（即人性、兽性、本能的部分）就会被“超我”和道德约束所压抑。一个人的心理发展就会因此失衡。

以下是几个具体的家庭文化传承案例及其影响：

**案例一：在“匮乏”教育下成长的男孩**

一个从小家境贫寒的男孩，父母不断强调：“我们家条件不好，你未来一定要努力赚很多钱，你要节约，不能调皮。”

在这样的教育理念下长大的男孩，大家通常认为他未来会很“匮乏”和“贪婪”。原因在于，家庭不断灌输“我们家没钱，很缺”的观念时，孩子的潜意识会接收到两个信号。

1.  **父母的无能**：当孩子认为自己的父母都无法应对生存问题时，他很难相信自己可以搞定环境、生活和未来。即使长大后赚到很多钱，这些钱对他而言只是数字，他内心对生活永远有无能为力的感觉，永远觉得“不够”，并且不相信自己可以比父母更厉害，无法相信自己能掌控环境和未来。
2.  **环境充满危险**：家庭可能随时垮掉，危险永远存在。不管自己多厉害、赚多少钱，永远都不能停。这种教育理念让他习惯战战兢兢地与环境对抗，永远觉得世界很危险，不相信自己可以“留下”和“活下来”。

这样的孩子无论有钱与否，都会永远担心生存安全问题。他的核心需求是**极度渴望生存的安全感**，而非金钱本身。

他不会爱上特别会赚钱的女孩，因为这样的女孩会加剧他的焦虑，让他感觉像在逃命时遇到了另一个逃命的人，从而觉得环境更恐怖。赚钱是他的表层需要，**安全才是他的核心需求**。

即使他因为安全感找了一个会赚钱的女孩，也并非出于喜欢和欣赏。一个时刻担心生存安全的人，缺乏喜欢一个人的动力。最后两人可能一起赚钱，但无人停下来了解对方的**核心需求**，这样的关系很容易走向分裂。

此外，当一个男孩被不断教育“很缺、不安全”时，他固化的“超我”会持续压抑“本我”（即“我不能放肆，必须保持清醒赚钱”）。这使他时常无法释放人的本能需求，因此这类男人很难专一，因为紧张之后只想要偷偷放纵。如何满足这种安全感，我们将在后续课程详细讲解。

**案例二：在“规则与独立”教育下成长的女孩**

一个出身高知、规则感强的家庭，总是强调“坐有坐相，站有站相”，要遵纪守法懂礼貌。家人之间的互动礼貌大于情绪。对女孩的教育则强调：“女孩子一定要自强、独立，靠别人都会跑，一定要靠自己，不要贪图享乐，要做正义的人。”

过于强调孩子要独立、靠自己的教育理念，传递了一个潜在信号：**“你其实很弱”**。因为如果你不弱，为什么要强调你一定要强？这就像有人让你减肥，你听到的是他觉得你胖的意思。

当这个“要独立、要强大”的“超我”一旦形成，即使长大后没有父母强调，女孩依然会用这套规则约束自己。在自我约束时，潜意识接收到的“我很弱”的信号会被反复强调，告诫自己千万不能依赖任何人，因为依赖意味着毁灭，所以必须刚强。

因此，许多女孩一边接受独立教育，一边潜意识强调“我很弱，一旦依靠、一旦走进亲密关系就完蛋了”。即使她名牌大学毕业、学历高、收入不菲，也很难真正融入一段感情，因为她始终戴着“我要刚强”的面具。

建立亲密关系需要分享脆弱、敞开心扉、交流感受。但在这种教育下的女孩认为分享脆弱是危险的，她们不敢表露真实想法、做真实的自己，只敢做坚强独立的“虚假自己”。

这样的女孩渴望什么？被父母不断强调“你很弱，你要刚强”的她，极度渴望**自己的内在被肯定**。她渴望允许自己柔软下来、享受生活，可以撒娇、发脾气、攀比、不规矩、拥有脆弱。因为只有允许自己脆弱，才是真正的强大。

这样的女孩可能自己都无法意识到这份渴望。如果她想“撒泼”（本能），但总被教育要规矩做人，本能被约束会让她难受，从而启动心理防御机制安抚自己。她可能会告诉自己：“我不需要撒泼，规矩做人就是我自己。”

由于长期习惯被约束的氛围，如果让她放松、释放，她反而会产生愧疚感。她极可能不停地为自己制造被约束的环境，例如：
*   找一个同样有强烈规则感的男人，两人一起自我束缚，又自我难受。
*   找一个与自己完全相反、特别释放、没有规则感的男人。这个男人的放肆让她爽快，又让她紧张，这同样是在为自己制造一种不舒服的约束感。

生活中很多人不允许自己舒服，他们习惯了不舒服的感觉，因此不允许自己快乐。许多人会与不喜欢的人结婚，过上并不开心的生活，在充满矛盾和抱怨中度过一生。

如果没有客观地觉察自己的行为模式，他们就无法意识到自己的无意识行为，只会不停地找理由：“因为父母催婚，我才选了这个人”、“因为当时还小不懂事”。从精神分析角度看，这其实是潜意识喜欢这种不舒服的感觉。因为习惯了被压抑，就无法拒绝根本不喜欢的男人，也无法拒绝一段不开心的生活。因此，许多夫妻在互相折磨中度过一生。

**案例三：在“男人不可靠”观念下成长的女孩**

一个女孩父母离婚，她跟随母亲。母亲一直教导她：“男人都不可靠，人心很不坚定。”或者母亲离婚后，向女孩呈现了表面强硬、实则懦弱悲催、麻木隔绝感情的样子。

即使父母没有离婚，但在他们的婚姻中，母亲总是被伤害，总是争吵而父亲无动于衷，母亲总是拿父亲没办法的样子，会让小女孩的潜意识接收到信号：**感情很伤人，妈妈搞不定爸爸，我当然也很难搞定男人**。

因为妈妈说男人不可靠，小女孩的潜意识会不断加强，让她反复强调：“我是一个不值得被爱的人，因为妈妈用行动告诉我感情不可靠。”

当这个潜意识一直存在时，女孩未来在感情中极易莫名其妙地担心男人出轨，即使对方什么都没做。她可能总希望对方必须按她的要求来对她好，总想去改变对方，或者总在向对方传递“我搞不定男人”的信号。

即使男孩最初什么都没做，当他接收到“这个女孩不值得被爱、搞不定感情”的信号时，也会无意识地做出一些伤害女孩的举动，让女孩自我怀疑：“为什么我老是遇到渣男？”

因为这些“渣男”，某种程度上是这个女孩创造出来的。她营造了一种感情必然走向分裂的氛围。这就是大家所说的“吸渣体质”，也是心理学上讲的“场”（即氛围）。不同的人带着不同的信念，会创造出不同的氛围，在不同的氛围下就会产生不同的互动关系。

被教育“男人不可靠，你搞不定男人”的女孩渴望什么？她极度渴望**自己和妈妈不一样**，渴望自己和另一半的相处方式与父母不同，渴望能够与另一半并肩而行，渴望拥有幸福的婚姻，被对方珍惜和尊重。

但在现实中，这种环境下长大的女孩往往不相信感情。因为当渴望反复无法满足时，女孩就会斩断自己的渴望，潜意识告诉自己：“不要太看重感情，因为男人不是好东西。”当她的渴望因得不到而不被相信时，这就回到了课程最初提到的：核心需求因为没有得到过，也没有感受过父母相爱且同时爱自己的感觉，所以一旦有人说爱她，她无法相信。
