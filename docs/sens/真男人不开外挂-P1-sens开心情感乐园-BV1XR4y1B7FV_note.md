# 真男人不开外挂 - 课程 P1 🚫

在本节课中，我们将要学习“外挂”这一概念。我们将探讨什么是外挂，为什么依赖外挂无法建立深度吸引力，以及如何避免这种行为，从而走向真正的核心自信。

## 概述：什么是“外挂”？🤔

上一节课我们介绍了“自主觉醒”。本节中，我们来看看“外挂”。

外挂是自信背后的虚假支撑。它指的不是软件作弊工具，而是指一个人依赖外在的物质、身份或标签来获得自信和吸引他人的行为。

以下是几个具体的例子：

*   你拥有劳斯莱斯、法拉利等豪车，你认为只有开着这些车时才有自信。
*   你佩戴百达翡丽、爱彼、劳力士等名表时，才感觉自己充满自信。
*   你在出席重要场合时，必须全身穿戴奢侈品名牌，否则就感觉不自在。
*   你认为穿优衣库就没有自信。
*   你通过强调“我爸爸是做什么的”、“我家有多少资产”、“我是哪个名校毕业的”来获取自信。

所有这些行为都叫做“调用外挂”。

## 为什么依赖“外挂”行不通？⚠️

有一句话是这么说的：当你开外挂时，你确实可以吸引到异性，但你一定进入不到深度吸引。

许多团队教人拍摄展示面、包装自己、建立不属于自己的人设。他们生搬硬套别人的东西，向女生进行高价值展示，但自身并无相应价值。在输出这些价值观后，他们会感到浑身不舒服，感觉与自己不兼容。

有的人甚至会购买高仿A货，例如香奈儿、爱马仕等带有大Logo的仿品穿在身上。他可能认为这获得了自信，但一旦将这些假货依附于身，这就不再是自信，而是调用外挂。如果你穿着A货去到太古里，可能会浑身不自在，感觉衣服带刺，生怕被人识破。

我不支持大家走以上两条路，因为这背后没有任何真实的东西支撑。这些行为是希望走捷径、开外挂。一旦没有这个外挂，见面时就会“一致性断裂”，甚至感到浑身不舒服。例如，有人觉得必须开保时捷、法拉利才有自信，一旦没有这些，就被“打回原形”，失去自信。

这也是为什么我们会在一些城市之外的地方开设线下课程，就是希望你们不要调用外挂，能用最真实的自己去吸引异性。

## 深入解析：开外挂为何让人不适？🔍

下面我们往深入讲解一下，为什么开外挂的人会不舒服。

首先，我们知道了开挂是什么。开外挂会强烈地提醒我们真相的存在——你到底是什么样的人，以及你为什么要做这件事。

例如，我和女生约会时说：“我是上海人，我住在上海的汤臣一品。” 说出这话确实可以吸引女生。但开外挂也绑定了一个真相：你知道自己是用钱和物质吸引了对方，因此你必须持续输出高价值才能维持吸引力。在我看来，这不叫核心自信。

无论你多有钱，经历多丰富，家庭背景多好，开外挂都会让你感觉到“外挂的重量”。这就像身上背负了一笔永远的债务，因为你总想展示自己好的一面，并为此额外消耗心力。**公式：心力消耗 = 维持人设所需的持续能量输出**。

这样只会让你内在的“垃圾”越来越多，无法做最真实的自己，你的吸引力就会被遮蔽。如果这样的“债务”太多，只会让你心力交瘁，因为你时刻都想呈现最好的状态。

所以，开外挂意味着你需要为太多东西提前准备，相当于给自己挖了一个坑，构建了一个不属于你的世界。你生活其中，生怕打破自己一手建立的脆弱围墙，害怕暴露内心的不自信。

## 真正的自信与建议 💎

我相信有些人真的有钱。但请记住，有钱自己知道就好，不用说出来。因为自信是在不经意间展示的，女人能感受到。

如果你有过开外挂的经历，你也会有种不堪重负的感觉。站在我的角度衡量，我觉得你完全没必要这样为难自己。假设你真的很有钱、很有价值，也没必要告诉别人：“我有价值，你看到了吗？”

为什么说必须做真实的自己最好？所谓的“自己”，是真正的自己，不是你周围的人构建出来的你。这是我2022年希望给大家带来的真实感受：我希望你们能真正拥有核心自信。**核心自信 = 真实的自我认知与接纳 - 对外在标签的依赖**。

自信这个内容框架很大，我有机会很喜欢跟大家分享。

## 总结 📝

本节课中，我们一起学习了“外挂”的概念。我们明确了外挂是指依赖外在物质或标签获取虚假自信的行为，分析了它无法带来深度吸引的原因，并深入探讨了开外挂会让人背负心理债务、消耗心力、远离真实自我的弊端。最后，我们强调了真正的自信源于内心，是做最真实、不依赖外挂的自己。
