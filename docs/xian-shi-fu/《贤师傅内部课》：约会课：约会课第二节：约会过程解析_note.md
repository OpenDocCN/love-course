# 《贤师傅内部课》：约会课：约会课第二节：约会过程解析_note

在本节课中，我们将通过复盘电影《黄金大劫案》中的约会情节，深入解析约会过程中的核心心态与技巧。我们将重点关注男主角“小东北”的行为模式，学习如何在实际约会中运用“第三方媒介”、“无需求感”等关键概念。

---

## 课程内容概述

上一节我们介绍了约会的核心心态。本节中，我们来看看这些心态在具体情节中是如何体现和应用的。我们将跟随电影剧情，逐一拆解小东北与女主角互动的关键节点。

以下是本节课将详细分析的几个核心阶段。

### 第一阶段：初次印象与无目的状态

他们第一次见面是在女主角家的晚会上。小东北当时在偷叉子。在女主角看来，他就是一个小偷，首次印象非常不好。

对于小东北而言，他的状态是无所谓。他认为自己和女主角不是同一个世界的人，未来不会有交集。他既没有需求感，也没有特定目的。

### 第二阶段：目的出现，需求感仍无

第二次见面是在街上。小东北找小女孩买花，同时计划用砖头抢钱。这个细节很重要：他此刻的核心目的是抢钱，心里完全没有想过要和女主角在一起。

他的心态是：**过程大于结果**。他不在乎能否和女主角在一起，只关注当下抢钱的目标。此时，他的行为模式转变为 **有目的，无需求**。

### 第三阶段：利用“第三方媒介”进行肢体接触

接下来发生了轰炸。小东北决定救人，直接牵着女主角的手逃跑。

这个牵手动作自然吗？女主角会觉得突兀吗？并不会。原因在于“第三方媒介”的作用——当时他们的注意力都在逃命和天上的飞机上。女生的注意力被转移，从而不会刻意关注肢体接触本身，身体反而会逐渐习惯。

他们躲到角落时距离很近，这也很合理，因为是在躲避空袭。这再次运用了“第三方媒介”：我是为了和你一起躲空袭才靠这么近，并非刻意接近。

### 第四阶段：情绪调动与“不按套路出牌”

在躲避时，女生说：“你别对着我喘气。”小东北的回应是：“我拖了头猪跑我能不喘吗？”这是一种轻微的“打压”。

随后，女生再次要求他别喘气，他却故意又对着她喘了几口。这就是“不按套路出牌”，用以调动对方情绪。

**核心要点**：激发情绪不一定是制造开心，让对方产生一点小生气也是一种情绪波动。关键在于把握度，不要过度。

### 第五阶段：制造“投资”与心态合理化

小东北被砖拍晕，被女生带回家照顾。此时，女生因为“他救了我”而产生了照顾他的责任感，这是人之常情。

小东北醒来后，第一反应是“讹钱”，甚至伸手去摘女生的项链。他的心态依然是：**我跟你肢体接触是为了讹钱，而不是为了亲近你**。他依然没有需求感。

随后，他发现女生的房子很熟悉，于是产生了新的、更大的目的——偷地图。**偷地图比男女感情重要得多**。这让他保持了“有目的，无需求”的状态。

我们来回顾一下女生的心理变化：

1.  初次印象：他是个偷叉子的小偷，很穷。
2.  第二次：他当街抢钱，但危险时救了我。
3.  获救后：他因救我而受伤，我出于人情必须照顾他。
4.  开始投资：女生将他带回家、让他睡自己的床、为他找吃的。这就是女生在投资 **时间、精力、情感**。

### 第六阶段：暴露需求感与“合理化”

小东北偷画地图时被女生发现。他立刻“取景”（转移焦点），说：“我偷的不是地图，是偷你的照片。”并进一步说：“我偷你照片还能是什么意思？不就是稀罕你吗？”

这时他似乎在暴露需求感，但为何显得自然？因为此时关系已经推进（女生已让他睡在自己床上），可以适度暴露。更重要的是他的“合理化”能力。

当女生给出负面反馈：“你喜欢我还打人？”小东北的回应是：“哪个大老爷们不打媳妇的？”这句话同时完成了 **合理化** 与 **关系升高**。

**心态核心**：我所做的一切行为都是合理的，我不焦虑。我不会因你的负面反馈而动摇。

### 第七阶段：强吻与再次利用“第三方媒介”

小东北强吻了女生。女生给出负面反馈后，他并没有纠结，而是用叉子扭出了一朵“花”送给女生。

这又是一个经典的“第三方媒介”运用：用折花这个新奇行为，瞬间转移女生对强吻的注意力。该做的事情已经做了（肢体接触），而对方的身体已经开始习惯。

**核心原理**：当你想要进行肢体接触时，通过“第三方媒介”转移对方注意力，能让接触变得顺理成章。

### 第八阶段：关系推进与带领

此后，女生开始主动找小东北（例如送药），因为她已经进行了大量投资。关系进入良性循环。

约会时应展示自己，并带领对方。带领的核心是 **制造情绪起伏**，同时自己保持 **无核心需求感** 的心态。

## 核心技巧总结与运用

以下是我们可以从小东北身上学到的、并能自行创造技巧的核心思维：

1.  **目的与需求分离**：约会时可以带有目的（如推进关系），但内心不执着于结果（无需求感）。
2.  **善用“第三方媒介”**：为肢体接触或敏感行为创造一个外部理由，使其合理化。例如：
    *   “车来了，快走。”（顺势牵手）
    *   “我们玩石头剪刀布，谁输了谁请喝咖啡。”（创造互动机会）
3.  **合理化一切行为**：对自己的一切行为充满信心，不因对方负面反馈而焦虑。常用话术如：
    *   “我亲都亲了，要不你还回来？”
    *   “我前面请你吃饭了，现在轮到你请我喝东西了，礼尚往来嘛。”
4.  **调动情绪，不按套路出牌**：通过轻微的打压、调侃或反其道而行之，让对方产生情绪波动。
5.  **引导对方“投资”**：让女生在时间、精力、情感或金钱上有所付出，这是关系深入的关键。
