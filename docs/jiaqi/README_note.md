# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。每个变量都有一个名字（即标签），我们可以通过这个名字来使用或修改盒子里存放的数据。

创建一个变量非常简单，只需使用等号（`=`）进行赋值即可。等号左边是变量名，右边是你想存储的值。

**核心公式：**
`变量名 = 值`

例如：
```python
name = "嘉琪"
age = 25
```

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们还需要知道盒子里装的是哪种类型的东西。Python中常见的基本数据类型有以下几种。

以下是几种最常用的数据类型：

*   **整数（int）**： 表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数（float）**： 表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**： 表示文本信息，需要用单引号（`'`）或双引号（`"`）括起来，如 `"Hello"`, `'嘉琪'`。
*   **布尔值（bool）**： 表示逻辑判断的真假，只有两个值：`True`（真） 和 `False`（假）。

## 如何查看与使用？ 👀

知道了如何创建变量和数据的类型后，我们来看看如何查看它们并在程序中使用。

以下是基本的操作示例：

1.  **打印变量**： 使用 `print()` 函数可以将变量的内容显示在屏幕上。
    ```python
    print(name) # 输出：嘉琪
    ```

2.  **查看类型**： 使用 `type()` 函数可以查看一个变量的数据类型。
    ```python
    print(type(age)) # 输出：<class 'int'>
    ```

3.  **进行运算**： 变量可以参与各种数学或逻辑运算。
    ```python
    x = 10
    y = 3
    print(x + y) # 输出：13
    print(x > y) # 输出：True
    ```

## 动态类型特性 ✨

Python有一个重要特性叫“动态类型”。这意味着你不需要提前声明变量的类型，同一个变量名甚至可以重新赋值为不同类型的数据。

**代码示例：**
```python
my_var = 100      # 此时 my_var 是整数（int）
print(type(my_var)) # 输出：<class 'int'>

my_var = "现在我是字符串" # 重新赋值为字符串（str）
print(type(my_var)) # 输出：<class 'str'>
```

本节课中我们一起学习了Python编程的基石：变量与数据类型。我们了解了变量就像一个命名的存储盒，并通过赋值来存放数据。我们认识了整数、浮点数、字符串和布尔值这几种基本数据类型，并学会了如何使用`print()`和`type()`函数来操作和检查它们。最后，我们还了解了Python动态类型的特性，它为编码提供了灵活性。掌握这些概念是后续学习更复杂编程知识的关键。
