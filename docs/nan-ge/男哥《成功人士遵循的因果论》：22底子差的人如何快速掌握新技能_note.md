# 男哥《成功人士遵循的因果论》第22课：底子差的人如何快速掌握新技能 🚀

在本节课中，我们将要学习底子薄弱的人如何通过高效阅读与思考，快速掌握一项新技能。核心在于理解并应用“踩在巨人肩膀上”的方法，将庞大复杂的知识体系分解为可执行的具体步骤，从而克服学习障碍，实现快速提升。

## 高效阅读与思考能力的重要性

上一节我们介绍了课程主题，本节中我们来看看高效学习的基础。成功需要高效的阅读与思考能力。阅读不仅指读书，也包含课程学习、与人交流、获取资讯等多种形式。

面对同样的信息，有人看不懂或一知半解，无法实操。而有人却能快速理解、落地实践，并且具备辩证思维。所谓辩证思维，即举一反三的能力。例如，你只学会 `1 + 1 = 2`，而别人却能推导出 `3 + 4 = 7`。当遇到 `7 + 8 = 15` 的新问题时，如果你只会套用 `1 + 1 = 2` 的方法，缺乏变通，就无法有效解决。本节课将帮助你改变这种状况。

聚焦，就是将一件看似困难的事情，细化成一个具体的点。只要解决这个点，就能获得理想的结果。今天要讲的方法共分三步。

## 第一步：如何踩在巨人的肩膀上高效学习

以下是实现“踩在巨人肩膀上”这一概念的具体方法。

首先，以学习本课程或任何一项新技能、进修、创业为例。我们需要思考一个新手在学习中常遇到的问题。

*   **问题一：学不下去、读不懂。** 学习者会因为看不懂内容而自我怀疑。必须承认，要想在某个领域成功，必须学习该领域内专业且系统的学科知识。例如，学习心理学或社交动力学。
*   **问题二：专业内容枯燥难懂。** 专业领域的原文和理论往往枯燥无味，难以理解。如果死磕教材，读不懂就无法指导实践。尤其是一些专业术语，在时间有限且需解决生存与发展压力时，学习会变得低效并引发焦虑。

此时，就需要运用“踩在巨人肩膀上”的方法。真正优秀的人在学习专业原文、名著或系统知识时，擅长同时利用该领域的“周边产品”。

### 什么是“周边产品”？

例如，学习《乔梁心理学》或 **Python语言** 这类系统课程。课程可能非常繁琐，包含240个章节。读到第100章时，你可能会因大量专业名词而放弃。你无法判断前100章是全书最难的部分，还是后面更难，缺乏整体把握。

这时，利用以下工具会非常有效。我们必须承认，阅读年代久远、语言迂回晦涩的内容时，除了枯燥，最大的问题就是不理解。本质上，别人能读懂而你不能，可能只是他掌握了更好的方法。

当你无从下手时，建议采用以下方法：利用 **知乎、豆瓣** 等平台。抖音平台娱乐性较强，权威性不如前两者。在这些平台上，你可以寻找相关线索，这就是“站在巨人的肩膀上”。因为这里存在许多该领域的“大神”。

例如，搜索 **Python语言** 或 **订单流交易**。你会找到许多实战派分享的评价，或对某个知识章节的理解。这等于你踩在了他们的肩膀上。在你尚未完整学习整个项目前，你已经看到了成功者或失败者是如何看待和讨论这些内容的。

你可能会说，这些线索有的有用，有的会误导人。但这总好过你独自在知识的海洋中盲目挣扎，像无头苍蝇一样在原始森林里寻找答案。大多数读过、学过、实战过的人都会分享体会。一旦他们谈论体会，你就能抓住重点。

假设掌握一项技能需要15个步骤，分享者可能会指出第7步才是关键，前6步大致了解即可。这种方法特别适用于解决晦涩难懂的知识点或内容。这样你就不会感到无助，你不是一个人在经历从0到1最艰难的过程，而是借鉴了过来人的理解。

作为一个新手，你可能会严格按照书本、课程或高人的理解来学习。但过来人可能会告诉你，某一段内容并非重点。你需要通读后建立整体概念，才能抓住最有用的核心。在一个领域里，你可能会发现真正的过来人说，只需要详细掌握第三点和第六点，其余部分可以快速浏览。这对你整体学习效率是极大的提升。

### 核心要点总结

因此，你必须学会以下重点：**踩着别人的肩膀**，就是依赖他人的解读和评价，来解决晦涩难懂的问题。当然，你所依赖的对象必须是“实战派”，而非没有经验的人。这并不难判断，在浏览大量内容后，通过对比你就能识别出谁是实战派。

做一个延伸：有时你接触某个领域却不能明理，就在于知识太专业或举例太少。但如果你踩在巨人的肩膀上，去查阅资料，会发现很多人分享了日常实战的经验、体会，以及对学科某个单元的理解。如果你底子差、生活经历少，这将是让你实现飞跃的关键，远胜于独自苦思。
