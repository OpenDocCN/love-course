# 男哥《教你学会和女生聊天》课程05：调动对方情绪奥义初级版 🎭

在本节课中，我们将要学习如何调动女生的情绪，这是让聊天变得有趣和吸引人的关键。我们将重点介绍“侧面高价值展示”这一核心方法，并初步探讨“反差感”的运用，帮助你通过聊天自然地吸引对方。

## 概述：调动情绪的核心

调动对方情绪的核心奥义，在于如何通过聊天让女生产生情绪波动。例如，即使谈论一碗面，也能引出“一个人吃吗？没有小姐妹陪吗？”等带有情绪的话题。实现这一点的首要方法，就是学会侧面高价值展示自己。

## 第一点：侧面高价值展示自己

如果你想让对方在聊天中产生情绪，必须学会侧面展示自己的高价值。女生会通过聊天观察和判断你是否具有吸引力。在聊天中传递出你优秀、不缺乏异性关注的信息，就能吸引女生。

同样，在聊天中传递出你不缺女生的状态，本身也是一种侧面高价值展示，因为它暗示了你自身的优秀。

**例如**，当女生说：“感觉这两天和你聊天挺开心的，但你这样的应该不缺女生吧？为什么还单身？”

这其实离成功很近。我们可以运用之前课程学过的“废物测试”应对方法。你可以这样回答：

> “我一直都不缺女生，但成熟后我发现，宁缺毋滥，定会得偿所愿；饥不择食，准会悔不当初。”

这句话一出，效果就不同了。这不是你在迎合对方、证明自己，而是让对方主动去发掘你的特质。这句话也表达了：我从来不缺机会，但我有宁缺毋滥的原则，你是否珍惜？不珍惜也没关系，可能是你想多了。我并没有要求你珍惜。

这一系列的交流都能侧面抬高你的价值。

再**比如**，当女生问：“你在干嘛？”

你不能回答：“我在抠脚丫子”或“我在打游戏”。一个优秀的男生，即使真的在打游戏，也不会直接这么说。

他可能会说：“在单手开法拉利”（意指正在做一件有意义的事）。不要认为优质、不缺女生的男生整天都很闲。所以，一个优质的男生在下午时分可能会说：“刚开完会，嗓子有点疼。”

这个例子传递的信息是：你在上班，很忙碌；你不是一个普通的打工者，而是需要负责开会发言的领导；你很有上进心。如果你说“我刚下班回家睡觉”，则无法体现任何价值。

学习本课程的目的是找到更好的自己。你要明白女生喜欢怎样的聊天方式，但不能靠背诵话术或虚假伪装。人生短暂，你本就应该活得有目标、有意义。

试想，如果你回答“在抠脚”或“打游戏”，并认为“打游戏有什么不对，要女朋友干嘛？”，女生对你的兴趣就会大大降低。

兄弟们要明白：在聊天中，如果你能侧面展示自己是个领导、前任女友条件不差，或者通过晒宠物照片来展示你很有爱心、对家人非常保护，这些都是很好的高价值展示。关键在于不能直接表达或迎合对方。

不要说：“我对女朋友一直很好，给她花很多钱。”高手通常会这样说：

> “我前女友当时很多人追，条件也不错，个子高，长发。但我们因为某个特别的原因在一起了。不过这些都是过去的事了，我也不想再提，不想有更多瓜葛。”

请思考，哪种方式展示高价值的效果更好？

有的兄弟或内部学员可能会想：“南哥，我想做真实的自己，不能那么装，不能侧面装逼。难道我要说我在开飞机、单手开法拉利吗？”

首先，你必须明白，侧面展示（即使略带夸张）在这个世界里往往最有效。大俗即大雅，大雅即大俗。

但更重要的是，师傅不仅要教你技巧（花拳绣腿），更要指引你成为一个更好男人的方向，帮助你找到更优秀的自己。

这句话如何理解？你是否发现，身边有些条件不如你的人，感情生活却比你顺利？比如，你给女生买了iPhone、iPad，三个月连手都没牵到；但这个女生三天后却和一个穷小子在一起了，还在朋友圈秀恩爱。

那个穷小子装逼了吗？他没有。所以，真正的情感高手是如何在自身条件（子弹）有限的情况下，让女生觉得他比别人更优秀的呢？无非是让女生参与进来，通过展示让女生自己去发掘，然后让女生自我否定：“哎呀，我怎么能这么想，原来他是这样的人。”

因此，真正的侧面高价值展示，并非单纯要你去“侧面装逼”（尽管这很有效）。你需要明白那些“穷小子”是怎么做的：他们善于利用一种感觉——反差感。

## 第二点：利用反差感

反差感是我接下来要重点讲解的利器，希望兄弟们认真听、认真学。这一点也是本核心课程的精华，我在大量直播课中也会运用。

其核心是：如果你是一个60分的人，如何让女生珍惜你？而不是你做到100分，女生却说：“你给我100分了吗？这100分跟我有什么关系？你给了100分，我就要200分。”因为你的过度付出反而提升了她的欲望阈值。

本节，我将透露我内部课程“南哥内核320”的冰山一角：**反差感**。

反差感这个词源于内容创作，常用于电影、剧本创作中，为了让观众记住一个平凡的角色而使用的手法。

你可能会问，这与我们的现实生活有关吗？戏剧手法会不会离生活太远？我举个例子你就明白了。

你是否看过很多古装剧或民国剧？其中一条常见主线是：女主角和男主角。起初，女主角觉得男主角是个小混混，而她自己可能要和一个上流社会的人进行商业联姻，她认为这是买卖婚姻。

随后，在某种特定的偶然条件下，女主角认识了男主角。起初她看不上男主角，甚至觉得他就是个小混混、小瘪三、小地痞流氓。

但是，突然发生了变故，比如遇到炸弹袭击或日本人来袭。这时，男主角下意识地挺身而出保护了女主角。在那个瞬间，女主角感受到了男生的保护欲。

紧接着，男生可能说：“刚才的事你别多想，我是个渣男。”然后转身离开。经过几次这样的轮番刺激后，女生就对男生动心了。

你可以回想一下《黄金大劫案》或《泰坦尼克号》。在《泰坦尼克号》里，富豪男友又帅又有钱、衣品好，还送她巨大的钻石，但女主角说“我不要这一切”。她选择了杰克，一个不修边幅、头发不洗、胡子拉碴的穷小子。

杰克是怎么做的？当女主角说不想去三等舱跳舞时，杰克是怎么引导的？杰克是计划跳完舞就和她在车里发生关系吗？显然不是。

所以，后者更懂得扬长避短。如果我想展现幽默，我会选择先不幽默，然后突然说句幽默的话，你就会觉得特别开心。如果我想表达我是个正经人，比如我说：“南哥我就是一个很正经的男生。”那么我前面可能会加一些不正经的表情或玩笑，然后突然说一句很正经（立体）的话。

同样，如果你想展示自己孝顺，不要直接说。而是前面说一些话，让女生觉得你像个浪子，然后再通过反差来体现。

**你可以这样设计聊天节奏**：三天内不怎么聊天，聊得也不深。但通过一些话语，让女生觉得她对你形成了某种“人设”判断。然后，你突然通过一个行动或一句话制造反差，让女生觉得：“哎呀，原来是我错怪你了！”这种让她自己说服自己、然后认可你的过程，远比那些直接给她金山银山的人，更能让她记住你的独特特质。

## 总结

本节课我们一起学习了调动女生情绪的核心奥义初级版。重点掌握了**侧面高价值展示**的方法，即通过间接方式在聊天中传递自身优点和吸引力，而非直接炫耀或迎合。同时，我们初步探讨了**反差感**这一高级技巧，通过制造预期与现实的落差，让女生主动发现并认可你的特质，从而产生更深刻的情感印象。记住，聊天不是背诵话术，而是结合技巧，引导对方参与并发现更真实的你。
