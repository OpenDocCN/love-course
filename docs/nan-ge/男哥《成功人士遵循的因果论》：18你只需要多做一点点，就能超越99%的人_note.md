# 男哥《成功人士遵循的因果论》第18课：🚀 你只需要多做一点点，就能超越99%的人

在本节课中，我们将要学习如何通过巧妙利用外部环境与他人的力量，来有效克服拖延并实现高效自律。核心在于，将个人目标与他人进行“捆绑”，借助外部监督和契约关系来驱动自己前进。

---

上一节我们介绍了隔离不良环境的重要性，本节中我们来看看如何更进一步，主动构建一个能促进自律的“人力环境”。解决拖延和实现高效自律的关键，不仅在于物理环境的塑造，更在于学习过程中**动用周围的人**来帮助你。

其核心含义是：将你任务或项目相关的所有人，都纳入到你的工作计划中。通过赋予他们角色，将你的进度与他们进行**捆绑**，让他们来监督和帮助你。因为仅靠个人意志，往往不足以时刻保持警醒。

以下是两种关于“公开目标”的常见认知：

*   **传统认知**：主张人要低调，事情成功前不宜宣扬。
*   **现代认知**：认为公开承诺（“吹个牛逼”）能形成压力，更有利于成功。

我认为后者更具实战性，更符合现代成功标准。例如，我计划下周一与某人讨论项目思路。为了让讨论有基础，我就必须在周一前完成相应的准备。即使我自己想拖延，但因为已将他人纳入计划，我需要考虑对他人时间的尊重。如果毫无进展，我将面临他人的批评甚至失望。这种压力会促使我赶紧行动。

作为一个自称有拖延症的人，我这些年能不断提升并取得成绩，核心就在于**善于动用群体效应**。具体方法是，将很多人纳入我的项目，并与之建立时间约束和契约。

相信理解这一点后，你会恍然大悟：原来自己过去总是在闭门造车，独自苦苦坚持，最后动力耗尽。何不尝试把别人也拉进来呢？

以下是几种可行的实践方法：

*   **共同学习**：例如，叫上你的女朋友一起听课。
*   **提升技能**：例如，学习领导力时，邀请一位同事共同参与。
*   **向上管理**：例如，向领导承诺：“下周三我要完成演讲/任务，我想在周二提前完成并向您复盘。”

想一想，当你把别人拉进你的计划后，你需要承担可能令他们失望的后果。这种压力是否会促使你立刻行动？因此，本课程提供的全是实战方法，没有空洞的理论。这些都是经过反复验证的有效策略。

在谈论自律人生时，我们必须认清：自律自觉是一种高级品质，且**大部分人都缺乏自律**，因为它反人性。许多成年人依然熬夜刷剧、打游戏、喝酒，这些习惯如何能通向成功和高度自律？因此，你需要借助外部力量。

让别人提醒你、督促你，效果是**立竿见影**的。

同时，这一点也引出一个反向思考：你的不自律，是否也源于周围环境中有“弱者”在拖拽你？他们可能会说“学这有什么用”，消耗你的能量，增加你的拖延。

所以，你要思考：你的不自律，是否在无意中给了某些人一种“拖你后腿”的角色，而那些人本身就带有这种负能量。

---

本节课中我们一起学习了克服拖延、实现高效自律的一个关键实战策略：**主动构建外部监督系统**。通过将你的目标与他人进行“进度捆绑”，并建立时间契约，你可以利用人际压力有效驱动自己。同时，也要警惕环境中可能消耗你能量的负面因素。记住，成功往往就藏在那“多做的一点点”里——即主动设计并利用好你周围的人力环境。
