# 男哥《教你学会和女生聊天》：06调动对方情绪初级版补充版_note

在本节课中，我们将学习调动对方情绪核心奥义的补充内容。除了之前介绍的反差感和侧面高价值展示，我们将深入探讨一个名为“双全对决”的独家理论，并学习如何通过控制聊天节奏和高点退出来掌握主动权。这些方法将帮助你理解聊天互动的底层逻辑，而不仅仅是寻找话题。

## 核心思路：超越话题本身

上一节我们介绍了制造反差和侧面展示高价值。本节中我们来看看，除了这些，还能做什么来有效调动情绪。

聊天话题本身可以通过各种渠道获取，但为什么常常效果不佳？原因在于你不知道如何组合这些“工序”以产生特定效果。如果你能明确，先做A再做B是为了制造反差，先做C再做D是为了侧面展示高价值，那么你的聊天策略将会有效。

## 独家理论：双全对决 🥊

现在，再提供一个超值的核心思路，这是我初级版课程的冰山一角：**双全对决理论**。

什么是双全对决？想象一下，你和朋友用拳头对顶。只要对方一用力，你就会站不稳；为了站稳，你会施加一个相同的力。最终两力平衡，两个人都能站稳。此时，如果对方突然撤去拳头，你会因为反向的力而摔倒。

再举一个例子：我们背靠背，我不断挤你，你为了靠住也会不断挤我，达到平衡。如果我突然不发力了，你会不会落空摔倒？这就是双全对决理论。这一点非常实用，是关系转变的关键第一步。

### 理论应用实例

以下是双全对决理论在聊天中的具体应用场景：

*   **场景一：打破固有印象**。如果女生将你定义为“钱包”（认为你只会花钱），你突然停止花钱并保持沉默。她会感到“财产”受损，从而主动探寻。错误做法是：虽然不花钱了，但嘴上仍随叫随到。当她想确认你是否还在时，你立刻回应“我还在”，这就前功尽弃。
*   **场景二：改变情感定位**。如果女生将你视为“情感宣泄桶”（开心时找别人，失恋时找你陪伴）。你陪伴两天后，第三天突然消失。她会想：“怎么不回我？”第四天继续消失，第五天你展示和别的女生吃火锅。她会反思：“我是不是太过分了？”人都会反思。
*   **场景三：制造惯性落差**。这类似于父母每天陪伴，突然离开你会不适应。如果你每天找女生聊天，形成惯性，然后在第三、四天突然消失。女生习惯了你的陪伴，当晚没等到你，就会开始反思：“是不是我说错话了？”“他是不是喜欢别人了？”当她胡思乱想时，调动情绪的目的就达到了。第五天，在她非常想聊天时出现，效果最佳。

所以，聊天的节奏需要设计。例如：**聊三天，停两天；聊两天，停一天**。好奇心和对人好，哪个更有效？有时对别人太好，对方反而会恃宠而骄。

## 关键技巧：高点退出 🎢

接下来，我们探讨第二个关键技巧：高点退出。

在你们聊天情绪最高涨、最嗨的时候，主动提出结束：“我有点事，等会再聊。”而不是每次都等到无话可聊、气氛下降时才被动结束。

当情绪展开并到达一个高点，你预感到再继续可能就要走下坡路、陷入尬聊时，能否主动退出？这正是女生常用以保持吸引力的方法之一：一方面聊得很嗨，另一方面又主动先退出，不配合到底。

如果你能预感到接下来可能冷场或尬聊，却还要硬聊二三十分钟直到无话可说，这是很危险的。同样，如果女生对你忽冷忽热，你明显感觉到她“热”的阶段即将过去，你能否比她更先“抽离”？

请思考：你聊天的目的是为了获得关系主动权，还是仅仅为了证明自己是她联系人列表中价值最高的人？你究竟想要哪一种？这也是让对方快速识别你“情人属性”而非“供养者属性”的最有效点。

你会发现，那些对你高冷的女生，可能只是对你如此；那些你认为的“渣女”，对我的学员而言也可能是好女友、好老婆。所以，要思考：是她们不对，还是你的策略激发了她们对你采用“索取和供养”的模式？

有时，我们甚至应该感谢那些曾“坑”过你的女生。正因为她们，你才遇到我，才得以成长。你并不是还对她付出不够，也不是多么专一。请对着镜子看看自己：这些年来，你让自己受了多少罪？你从未好好爱过自己。

报名我的课程，就是进步的第一步。对任何人付出都有风险，但投资自己，永远没有风险——除非你今天投资自己提升了，明天却失忆了。
