# 《爱商训练营》：08正确认知，快速走出聊天误区_ev_note

在本节课中，我们将要学习如何正确理解聊天在恋爱中的作用，并揭示单纯学习聊天技巧的局限性。我们将探讨吸引力的本质，并指出提升个人展示与包装才是解决聊天困境的根本途径。

## 理解聊天的局限性

言归正传，今天我们来讲一讲聊天。一个男生如果会聊天，就会体现其高情商，而高情商当然会使其更受女生欢迎。但是我想说的是，全网都在教聊天，然而只学聊天其实是没有用的。所以你今天一定要明白，只想通过学习聊天来解决聊天问题，是行不通的。

只学聊天，没用。你不信的话，可以尝试用一个山水画、一个小猪猪或者一个卡通人物的头像，去跟一个优质的女生聊天。我相信不只是她不会跟你开场，可能她连跟你匹配都不会。所以，很多导师过度地强调聊天技巧。

在我看来，就这一点而言，他们一定不能解决自己的情感问题，他们也一定没有遇到过高分的女生。在我看来，你想要聊天好，就要明白整个恋爱是什么。我相信你也是为了弄明白整个恋爱是什么而来的。因为有时你聊天可能比我还“会撩”。

有时你又不会。你没有高手那样的稳定性和持续输出能力。你没有在自己疑惑的时候，拥有那种坚定的心。你不知道核心是什么。没有核心指导你去实践，你就会出现能力忽高忽低的情况。你就会出现同样一句话，有时好使有时不好使的现象。

## 恋爱的本质：自我推销

那么，我要告诉你恋爱的本质。吸引优质女生，吸引任何你喜欢的女生的本质，就是**把你自己推销出去**。假设你是一个销售，你要卖一款产品。思考一下，你怎么才能让你的业绩好呢？

以下是达成良好业绩的两个关键步骤：

1.  **你需要客源**。因为你没有客户的话，你是没有办法成交的。
2.  **有了客源之后**，你需要思考如何能够提升成交率。关键不在于如何提升销售能力，而在于**如何能够有一个好的广告和好的产品包装**。

我相信你会发现，这个世界上任何一款产品，除了其功效和优质本身，大家普遍是从包装、广告和展示开始关注的。正所谓“始于颜值，终于才华，陷于人品”。很多人都喜欢跟女生过度地展现才华，或者表现自己是一个多么优质的男生。

我不是说你的注意力都应该放在外表包装上，只有颜值而没有才华和人品。但是你要明白，**形象是第一个敲门砖**。如果你的包装不OK，其实没有人会去关注你是不是一个好产品。

## 聊天在销售流程中的位置

你聊天就相当于销售环节。你们可能都在思考“我如何跟用户去交流”。但是我想告诉你的是，如果你没有一个很好的广告，没有一个很好的包装，你的用户是不足的。这些用户很可能现在不想买你的产品。

或者说，你没有足够的用户样本，你是无法提升你的聊天技能的。试想一下，你没有办法在第一时间内让用户关注你，那么你说什么都没有用。你有没有想过，这就是聊天的本质？

我这里给大家举一个例子。比如你们经常会在评论区里说：“这个女生上来就不喜欢我，学什么聊天技巧都白搭，聊什么呀，这个女生都看不上我，白扯。”其实说这话的兄弟，我觉得你是实践过的，你是实战过的。你明白，其实光学聊天是没有用的。

你明白自己是没有展示和包装的，但是你又回避自己如何提升包装和展示。第一种人认为：“哎呀那是虚假的。”第二种人认为：“我一个男人不去赚钱，我展示什么呀。”我觉得这两种想法都是错误的。

## 提升维度：从展示与包装入手

如果你想要真正聊天好、约会好，你首先要明白第一个道理，这个道理跟聊天约会没有直接关系。这个世界是一个IP的世界，是一个全民网红的世界。你无法在最短时间内，从视觉方面快速吸引女生，那么你后面做的都是白搭。

而我们就要探究一下，如何能够快速地改变你展示的现状，这才是你聊天的重点。没有形象，女生看不上你，你根本就没有聊天的机会。你离高手其实就差一步之遥，但是却被全网的理论派说成“聊天就是可以吸引女生的，一定要学习交流”。

而实际上，吸引女生分为：**包装、客源、销售话术和产品优质**。那么你想一下，聊天要好，你要先明白大量女生都喜欢什么样的头像，愿意给什么样的头像、朋友圈、照片留下时间。而这些都属于我们“客源”环节中的“广告”部分。

话再说回来。如果你都发现单纯的靠聊天技术没有用，那你要学会提升维度去思考一下。解决这个问题的最本质是什么？就是**如何能够解决你形象的问题**。

一个男生的形象，可以从他的发型、适合他的衣品、适合他的身材这些方面入手。从这些方面入手，绝对可以快速地提高你跟女生聊天的困境。

一定要记住：**你是什么人，就撩什么样的妹**。你想一下，一个200斤的胖子，或者一个1米4、头上三根毛的女生，对你说“我在酒店里等你”，或者对你说“天上的星是明月，地上的心是我爱的人，小哥哥你来不来”，你会不会去？

觉得我说的更实在，更为结果而导向，觉得我是个有结果的人，觉得这个人一定牛的，你就继续看下去。
