# 男哥《成功人士遵循的因果论》05：轻松战胜诱惑之物理隔绝法 🛡️

在本节课中，我们将要学习如何通过顺应人性，而非对抗人性，来培养专注力并高效达成目标。核心在于理解并运用“物理隔绝法”，创造一个无法分心的环境，从而让专注变得自然而然。

---

成功人士通常具备乐观、高效和精力充沛的特质。我周围的同事和朋友经常对我说，南哥，我发现你总是精力充沛。南哥，我发现无论公司还是生活中遇到任何事情，你总能乐观面对，时刻保持激情与冷静。南哥，我觉得你做任何事情效率都很高。

那么，为什么能达成这样的结果呢？我通常这样回答：这就是命。那些有钱或更容易成功的人有一个特质，因为钱是客观的。我可能体能好，所以你看那些创业或开公司的人精力充沛，是命中注定的精力充沛。

今天，我将为大家揭晓：有命，也有运。如何改运呢？其实就是改变环境，就能改变你的命运。除了命不好以外，还有没有机会实现自己的目标、人生理想、脱单等愿望呢？我的答案是可以做到。这需要运用以下几种方法。

首先，我要讲一个人性的关键点。你会发现我的课程以探究和洞察人性本质为主。因为我始终发现，所谓“不要挑战人性”，就是要顺着人性的思维走，不要轻易地挑战人性。通过我的课程，你会更清晰地明白，那些成功的人、赚到钱的人、各行各业的领袖、艺术家等，除了具备我上节课说的那些点以外，他们是如何在现实生活中实操应对人性弱点的。

我经常提到一件事：要高效。高效的定义是什么？首先要明白，一个人埋头苦干是不会成功的。如果埋头苦干能成功，那些天还没亮就起来扫街的阿姨们，她们的工作虽然值得敬佩，但为什么她们花了这么多时间专注于此，却未能成功？答案就是不高效。

高效的定义应该是**事半功倍**，而不是**事倍功半**。想要高效、乐观、随时随地都有精神头，第一个考验就是：你能不能专注？我经常跟公司的人讲，也这样告诫自己：做一件事一定要有匠人精神、工匠精神，一定要专注不分心。

但这其实是一个误区。你可能认为，只要具备了全神贯注的精神，人人都可以特别好地专注。但我想说，当你真正想要专注做一件事时，这与你的**环境、训练和方法**密切相关，而不在于主观上要求自己。

你会发现有些人进入工作或学习状态非常快，并且很少中途分心，一点就透，事半功倍。而另一些人，在学习情感课或做事业时，却费力不讨好，事倍功半。

我总结了四种方法，让我们能够快速解决现状。第一个方法叫做**物理隔绝法**。

你一定知道什么事情会阻碍你专注。你也一定会明白，人生的任何目标都需要专注。专注的反面是什么？我们要知道，这个世界充满诱惑。对人性的正确认知和判断是：**人是抵挡不住诱惑的**。

因此，我们要把那些已知的、会诱惑我们的事情**阻挡并隔离起来**，这样我们才能真正专注。如果你说，我要逆着人性来，我要克服诱惑、战胜诱惑，那我觉得这是反人性的。

我经常在群里说，有些学员做得非常好，夸他们做笔记。有些人呢，我就知道你是躺在床上听的课，或者在刷牙时听的。你要明白，当你的意志力无法支撑自己专注时，你就要让环境来帮你。

比如，你知道牵扯你精力、诱惑你的是电脑、手机、抖音、零食（所以你不去健身），或是某个软件。那么，就算你想分心，如果没有这些东西，你还能干什么呢？

所以我经常要求公司的人和私教：把你所有认为会让你分心的事情、APP、图文，在你做正事的时候，**全部删除或隔离掉**。这个时候你就会想：我什么都没有了，那除了做好手头的事，我还能干嘛呢？

兄弟们一定要理解：不是那些成功的人比你更能专注，比你更能挑战人性。有时候，那些成功的人更经不起诱惑。但在面向成功的目标之前，他们学会了**经得起诱惑之阻碍诱惑、隔离诱惑**。

如果你在学我的课时，好好想一想：此时此刻，你是不是有时还在回信息？那么麻烦你，把这个软件先退出登录。或者，你是不是在学课时又在刷抖音？那么麻烦你，先把抖音屏蔽掉。又或者，你是用电脑记笔记，但电脑一会儿弹广告，一会儿你又想查点有趣的事。那么麻烦你，把电脑的Wi-Fi关掉。没有网，我除了学习，还能干嘛呢？

如果你知道躺在床上容易睡着，那就不要认为用课余时间充电就应该这样。人是没有办法抗拒诱惑的。那麻烦你，把你的书桌清理干净，只留一个能听我课程的媒介，再加上你的笔记本。

我告诉你，那些真正成功的人，他们不会去思考“如何才能更专注”、“如何能够抵挡诱惑”，这些都是背道而驰的问题。所以我觉得我们应该更实战、更简单粗暴地解决。你一定要解决的就是：把所有牵扯你精力的事情全部隔离掉。让你发现，不是你不再分心了，而是你**没有任何事情可以让你分心**。

我相信通过这样的交流，你会发现，原来在过往中，自己曾几何时离成功很近。

我还要在这里给兄弟们说一点：这个方法有一个前提，在于你要**遵从人性，顺势而为，不要去挑战人性**。你首先要承认：我自己是管不住自己的，我自己的专注能力还没有锻炼成熟，不要去挑战它。

---

**本节课总结**

本节课中，我们一起学习了培养高效专注力的核心方法——**物理隔绝法**。关键在于承认人性的弱点，即我们难以抵抗诱惑。因此，最有效的方式不是依靠脆弱的意志力去对抗，而是主动改造环境，将所有已知的分心源（如手机、无关软件、零食、网络等）物理性地移除或隔离。通过创造一个“无心可分”的纯净环境，让专注成为唯一且自然的选择，从而实现事半功倍的高效状态。记住，成功者往往更懂得如何顺应并善用人性，而非挑战它。
