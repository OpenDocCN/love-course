# 男哥《成功人士遵循的因果论》完：41职场99%的人错误思维（三

在本节课中，我们将要学习一种在职场中极为普遍的错误思维模式：将一切未完成的事情都归咎于自己“意志不坚定”。我们将剖析这种思维的局限性，并探讨更有效的成功路径。

## 核心错误：将失败归因于意志力

第三种错误思维，是将一切未完成的事情，都归咎于“我的意志不坚定”。“自我意志不坚定”是我们常给自己在职场中不能获得成功所下的一个定义，也是一种错误的潜意识。我们从小就被灌输一种观念，似乎所有的成功都需要依靠一种叫做“意志”的品质去执行。

我们不得不承认，那些能够坚持一件事情、有强大意志力的人确实更容易成功。但关键在于，我们每个人在所有时刻、所有事情中，都能具备同样坚定的意志吗？

因此，这句话的核心含义是：你要能够认可自己，**并非任何时候都能保持坚定**。

## 解决方案：从依赖意志到营造专注

上一节我们指出了单纯依赖意志力的问题，本节中我们来看看什么是更可行的替代方案。

我们真正应该学会的是 **“聚焦”** 。当我们能够聚焦于某件事情时，发挥主要作用的并非你的“意志”，而是你**专注地沉浸其中**的状态，这使你免受过多外部声音的干扰。

以下是两个生活中的例子：

*   **健身/跑步**：你能做好它，是因为让自己的身体和意识沉浸于运动过程中，不受外界干扰。你享受这种沉浸的状态，而非仅仅依靠“咬牙坚持”的意志力。
*   **工作**：工作同样需要专注。当我们进入专注状态时，所谓的“意志力”只是一个非常微小的支撑点。

所以，这又回到了前面课程的观点：不要总是责怪自己“不专注”，而是要学会**给自己营造出一个能够专注的环境**，并培养能让自己沉浸进去的专注品质。

## 环境与多任务处理的陷阱

拥有一个能让自己沉浸其中的环境至关重要。反之，那些身处错综复杂、充满各种干扰声音的环境，并且同时处理N件事情的人，其结果往往事与愿违。

这正应了 **《孙子兵法》** 中的智慧：
> **“无所不备，则无所不寡。”**
> （意指：处处设防，则处处兵力薄弱。）

试图面面俱到、同时应对所有事情，最终会导致每一件事都做不好，精力分散，无法深入。

## 总结与行动指南

本节课我们一起学习了职场中第三种错误思维：将失败简单归咎于个人意志力薄弱。我们认识到，**意志力并非随时可用的稳定资源**，真正的关键在于**学会聚焦，并主动为自己营造沉浸式的专注环境**。

最后想对大家说，认知再多，不去实践也依然过不好这一生。因此需要大家多多实战、执行和思考。作为我的学员，起码要做到一点：**千万不要在职场中，再持有这三种狭隘的错误思维**。

从今天起，请停止苛责自己的意志力，转而开始设计和优化你的工作环境与流程，让专注自然发生。
