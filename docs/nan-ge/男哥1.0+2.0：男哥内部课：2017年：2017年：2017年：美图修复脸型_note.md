# 课程一：美图秀秀基础修图教程 🖼️

在本节课中，我们将学习如何使用美图秀秀软件进行基础的人像修图，以改善展示面照片的效果。课程将涵盖从拍照技巧到软件操作的全过程。

## 概述

本节课的核心是学习如何通过简单的拍照和修图技巧，有效提升个人展示面照片的质量。我们将使用美图秀秀软件，重点讲解美白、眼睛调整、鼻子修饰和下巴优化等关键步骤。

## 拍照基础

上一节我们介绍了课程目标，本节中我们来看看如何拍出更好的原始照片。

正确的拍照姿势能大幅提升原图质量。以下是关键要点：

*   **角度**：手机镜头应向下倾斜约30度，并从较高位置拍摄。
*   **场景**：选择如咖啡厅等环境良好的背景。
*   **细节**：注意仪容，如剃净胡子、整理发型。

## 软件准备与导入

在掌握拍照技巧后，我们需要准备好修图工具。

进行修图前，需在电脑上安装美图秀秀软件。操作步骤如下：

1.  打开美图秀秀软件。
2.  点击“人像美容”功能。
3.  导入需要修饰的照片。

## 核心修图步骤

以下是使用美图秀秀进行人像修饰的核心操作流程。

### 1. 基础调整与美白

首先对照片进行旋转校正，并改善肤色。

*   **旋转**：使用旋转功能调整照片至正确角度。
*   **局部美白**：使用“皮肤美白”功能，选择“局部美白”模式，涂抹皮肤区域以提亮肤色，避免整体画面过白。

### 2. 眼部修饰

眼睛是修饰的重点，目标是使其更有神。

*   **放大眼睛**：使用“眼睛放大”功能，适度调整，避免过度夸张。
*   **开眼角**：使用“瘦脸”或“微雕”功能中的局部调整，轻微拉动眼角，使眼型更饱满。

### 3. 鼻子修饰

通过修饰使鼻型更挺拔。

*   **缩小鼻头**：使用“瘦脸”功能，向内轻微推动鼻翼两侧。
*   **提亮鼻梁**：使用“局部美白”功能，在鼻梁处进行涂抹，创造高光效果，使鼻子更挺。
*   **磨皮平滑**：使用“局部磨皮”功能，处理鼻子周围的皮肤，使其更平滑。

### 4. 下巴与面部轮廓

修饰下巴和面部线条，优化脸型。

*   **突出下巴**：使用“局部美白”功能在下巴处提亮，视觉上使其更前凸。
*   **磨皮融合**：使用“局部磨皮”功能对下巴区域进行打磨，使修饰痕迹更自然。
*   **瘦脸**：若脸颊较宽，可使用“瘦脸”功能轻微向内推挤两侧腮部。

### 5. 细节处理

最后处理皮肤瑕疵，完成精修。

*   **去除眼袋/黑眼圈**：使用“消除黑眼圈”功能，小心涂抹下眼睑暗沉区域，注意保留卧蚕。
*   **平滑法令纹**：使用“局部磨皮”功能，对法令纹区域进行轻柔打磨。
*   **整体磨皮**：最后使用“自然磨皮”功能进行整体皮肤柔和处理。

## 成果保存与对比

所有修饰步骤完成后，需要保存成果。

点击“保存”按钮将修好的图片存储到电脑。通过对比修图前后的照片，可以清晰看到面部肤色、轮廓和细节的改善效果。

## 总结

本节课中我们一起学习了展示面修图的完整流程。从前期的拍照技巧，到使用美图秀秀进行逐步的肤色美白、眼部放大、鼻子修饰、下巴优化以及细节处理。关键在于局部功能的精细使用和力度的适度控制。通过练习掌握这些技巧，可以有效提升照片视觉效果。
