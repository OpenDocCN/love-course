本节课中我们一起学习了：
1.  **识别自身角色与对方投资**：明确自己是“情人”还是“供养者”属性，并观察对方是否付出。
2.  **控制对话节奏与探测需求**：通过“状态+感受”开启对话，避免话多暴露需求，学会推拉。
3.  **进行情绪互动与框架保持**：用调侃、冷读或状态分享来互动，不轻易被对方情绪牵制，保持自身框架。
4.  **运用审判者框架与区分关系阶段**：在长期关系中，适时切换角色，并理解恋爱阶段与吸引阶段思维的不同。
5.  **理解感性与理智思维的应用**：在两性互动中，过度依赖逻辑说教往往无效，需要融合感性表达。

核心在于，互动是情绪的博弈，而非道理的辩论。掌握框架，灵活运用推拉，并理解不同关系阶段的核心需求，才能有效引导对话与关系走向。
