# 夜的第二章《约会全能达人》：第07课 - KTV聚会（下集）🎤

在本节课中，我们将学习在KTV聚会中如何通过多种互动游戏，自然地与心仪对象增进关系，并掌握转场和氛围控制的技巧。

上一节我们介绍了适合开场破冰的“抓手指”游戏。本节中，我们来看看更多适合在KTV中与女生互动、逐步升温暧昧气氛的游戏。

## 1. 猜拳喝酒游戏 🎲

这个游戏的规则非常简单。大家一起石头剪刀布，出拳时喊出一个杯数，例如“半杯”、“一杯”。杯数可以逐渐累加，最后输的人喝下当前喊出的杯数。

但需要注意以下要点：
*   酒的度数不能太高，杯子不能太大，否则容易有人喝醉。
*   女生喝醉或让对方感觉被灌酒都不合适。
*   自己喝醉也会耽误正事。

## 2. 玩骰子游戏 🎯

玩骰子规则简单，但玩好并不容易，过程会比较刺激。骰子中的“1”点可以当作任何数字。南北方的具体玩法（如“加摘”或“不加摘”）可以自行搜索练习。

本节课中，我运用了“虚张声势”的方法。故作镇定，让女生觉得我有很多她所喊的数字，诱使她不断加高杯数，等她喊多时我再“开”她。这个游戏需要提前与朋友多加练习，熟能生巧。

## 3. “我爱你/不要脸”游戏 💬

这个游戏的规则也特别简单。一圈人互相传话，按顺序说“我爱你”，逆序转话时则说“不要脸”。

规则虽然简单，但玩起来特别容易晕，一不小心就会说错。和女伴坐在一起玩这个游戏时，能够利用游戏作为第三方媒介，传递你想表达的意思，更容易产生暧昧情绪，而对方又无法指责，毕竟只是在玩游戏。

## 4. 转盘/酒瓶惩罚游戏 🔄

现在KTV通常自带各种道具小转盘。如果没有，也可以用酒瓶或酒杯旋转来代替。指针指向谁，谁就接受惩罚。

但需要注意，总是以喝酒作为惩罚会显得无趣。除了喝酒，“真心话”和“大冒险”也是不错的选择。
*   如果你的女伴选择“大冒险”，僚机可以让她与你拥抱或亲一下脸，这都是很好的互动方式。
*   如果你的女伴选择“真心话”，可以互相问一些比较私密的问题，例如“第一次恋爱是多大？”、“谈过几个男朋友？”。

无论是肢体接触升级，还是真心话提问，都应该由浅入深，由不敏感话题到敏感话题，否则容易让人感到不适。

## 5. 吸纸牌游戏 😉

这个游戏极度容易产生暧昧氛围，但需要注意，放在聚会后期玩比较合适。那时大家已经放得比较开，也喝得差不多了，不想再喝酒。

因为游戏规则是如果不用嘴接住传递的纸牌，就要喝酒。所以为了避免喝酒，你和她只能选择让嘴唇越来越近。这个游戏的要点是，你们要尽量靠后去接纸牌，不要主动提前告知对方，否则可能弄巧成拙，为他人做了嫁衣。

## 6. 适时转场 🚪

当现场情绪和气氛都到位之后，你们就可以考虑转场离开KTV了。转场的方法有很多种。

例如，可以邀请她：“陪我去个洗手间吧？”只要前面的互动都很好，她一般不会拒绝。KTV聚会的核心简单来说，就是要动起来，多利用周围的人、场景、道具以及转场，来保持整个约会过程的能量不低落。

至于之后是否要一起吃饭，真的要看你对当时“窗口”（对方好感度）的识别能力。请记住：过程大于结果。一次好的KTV聚会体验，能给对方留下深刻印象，也为后续的聊天互动激发更多话题。
