# 《爱商训练营》课程03：千人斩恋爱高手对废物性测试的理解 📚💡

在本节课中，我们将要学习恋爱聊天中的一个核心概念——“含量”，并深入理解“废物性测试”的本质。我们将探讨如何通过把握关键对话节点，而非机械背诵话术，来建立真实的吸引力。

## 聊天中的“含量”与节奏 🎯

上一节我们介绍了聊天的基础框架，本节中我们来看看聊天的“含量”问题。我认为，聊天不应是背诵固定话术或进行虚假伪装，说一些不符合自身人设的话。

学习的目的是找到更好的自己、自己的节奏和表达方式，而非简单模仿他人。模仿往往无效，因为每句话都依赖于特定的语言环境、聊天阶段和特定人设。

你需要找到与自己匹配的说话方式。例如，如果你的衣品沉稳，说话方式也应相应沉稳。若你是企业人士，你的发型和穿衣搭配需符合身份。如果你想接触年轻女生，且自身年龄也不大，说话的节奏就应做出调整。

**核心是：每个人都要找到自己的节奏，而不是学成别人的样子，最后连自己是谁都不知道。**

## 跳过约会的吸引力：情人属性 💘

那么，为何有些聊天能跳过约会阶段直接建立吸引力？我的大部分成功聊天案例正是如此。

这证明在聊天中传递了正确的**情人属性**，从而引发了女生正向的情感策略回应。情人属性的一个最大特质是：在百句对话中，你可以随意回复大部分内容。

你无需纠结第一句必须接第二句，第二句必须接第三句。只需注重**二八定律**——把握住那最关键、占对话总量约20%的核心内容即可。其余80%的日常对话可以轻松应对。

**公式：吸引力 ≈ 把握20%关键对话节点**

例如：
*   女生问“吃什么了？”，你可以轻松回答“喝了碗粥”。
*   女生问“在干嘛？”，你可以如实说“看电视/打游戏/刷抖音”。

要真实。如果你不是富二代，就不要说“在开法拉利，很爽”。真实是可贵的。

## 关键节点：识别“废物性测试” 🚨

那20%的人做到了80%的人做不到的事，他们察觉到了一个高手通常不会明说的秘密：在聊天中存在一些必须注意的关键交流点。

这些关键点绝非星座八卦或兴趣爱好等能聊得热火朝天的话题。我给大家举个例子：

有没有女生问过你：“你喜欢什么类型的女生？”
如果你回答“我喜欢可爱的”，她可能会说：“那我不可爱”、“是不是有比我更可爱的女生你还要去找？”
如果你回答“宝贝我只要你啊”，她可能会说：“我不信”。

你会发现，当女生聊得特别嗨时，可能会突然冷淡一下。在那次聊天中，只要你按照她预设的节奏去回应，就会失去感情的主动权。

这就是为什么有时聊得很好却没有下文。因为在聊得好的过程中，她在某一分钟的几十秒里，抛出了一个测试。你觉得自己应该那么回，她哈哈一笑继续聊，但实际上她已经通过这个测试将你筛选掉了——她认为你属于“投资供养型”。

**投资供养型**的典型回应模式是试图解释和讨好。例如：“你喜欢我什么？”“宝贝我喜欢你可爱。”“我不可爱。”“不能啊，你这就很可爱。”“那有比我更可爱的怎么办？”你会发现这类对话逻辑上容易陷入死循环。

再举一例：假设女生是气质型。
你说：“宝贝，我觉得你的发型配这身衣服很有气质。”
她可能说：“那要是遇到比我更有气质的人，你是不是就会喜欢她？”“你是不是只想短期跟我在一起？”

你会发现，这些问题才是最核心的。它们的共同特征是：**无论你回答A还是B，似乎都不对**。这时你就需要警惕，对方的话绝非字面意思。

另一个常见例子：在暧昧或恋爱阶段，女朋友问：“老公，这件衣服好看吗？”
你说“不好看”，她会不高兴。
你说“好看”，她又可能说你“敷衍”。

**这才是聊天中“二八定律”里最重要的关键点。** 只要这些点处理对了，剩下的日常对话你可以随意发挥。你会发现，你无需背诵华丽话术。一句简单的“想你了”和一篇50字的华丽情话，可能让她同样开心，因为你已经通过了前期的关键测试。
