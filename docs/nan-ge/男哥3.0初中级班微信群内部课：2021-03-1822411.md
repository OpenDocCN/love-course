# 男哥3.0初中级班微信群内部课：2021-03-1822411

在本节课中，我们将学习如何通过三款核心软件（美图秀秀、Snapseed、Facetune）来优化个人照片，提升整体视觉质感与吸引力。课程将涵盖软件功能解析、修图顺序逻辑以及具体的参数调整技巧。

## 软件准备与核心作用 🛠️

首先，你需要下载三款软件：美图秀秀、Snapseed 和 Facetune。它们的核心作用各不相同。

以下是每款软件的主要功能：

*   **美图秀秀**：主要用于调整**身形**与**五官**。其核心功能包括拉长小腿、调整肩宽、修瘦身形，以及通过“3D”功能智能优化五官。
*   **Snapseed**：核心作用是**调整光线与整体质感**。它能让照片的色彩更鲜亮，线条更突出，是提升照片“画质”的关键工具。
*   **Facetune**：属于**可选**工具，主要用于局部精细处理，例如平滑皮肤、提亮衣物或强调某个局部细节。如果觉得付费较贵，可以寻找替代方案。

## 核心修图流程与顺序 🔄

上一节我们介绍了三款软件的分工，本节中我们来看看它们的使用顺序。正确的顺序能避免反复调整，达到最佳效果。

基本流程是：**先用 Snapseed 调光，再用美图秀秀修身形和五官，最后根据需要决定是否使用 Facetune 进行局部微调**。如果顺序颠倒，先修脸再调光，可能会导致光线过度或效果不佳。

## Snapseed 调光详解 💡

现在，我们深入讲解如何使用 Snapseed 进行第一步的全局光线调整。目标是让照片达到“鲜亮、干净、清新”的视觉感受。

打开 Snapseed 的“工具”菜单，选择“调整图片”。以下是各项参数的核心调整思路：

1.  **亮度**：这是首要调整项。根据原图是白天或夜晚拍摄，通常增加 `30` 到 `60` 的数值，让整体画面变亮。
2.  **对比度**与**饱和度**：在提亮亮度后，为了不让照片过于刺眼或艳俗，通常需要将这两项**略微降低** `5` 到 `10` 个数值，以营造清新淡雅的感觉。
3.  **氛围**：此项可调可不调，建议数值在 `0` 到 `10` 之间微调。它会影响整体色调融合，使用需谨慎。
4.  **高光**与**阴影**：高光可适当增加以突出亮部；阴影则参照脸部，确保脸部不过暗，同时能增加背景黑色部分的质感。
5.  **暖色调**：此项通常不调整，除非原图色温明显偏蓝或偏黄，可做轻微校正。

> **核心要点**：亮度着重加，对比度和饱和度轻微减，其他参数视情况微调。不要死记数值，多尝试以培养感觉。

## Snapseed 局部工具妙用 🎯

完成全局调整后，我们利用“局部”工具来强化照片的视觉焦点。这个工具能让你的照片在信息流中第一时间抓住眼球。

以下是“局部”工具的几个关键应用场景：

*   **强化光源**：圈出天空、灯光等自然或人造光源，提高其**亮度**，让画面更有层次和吸引力。
*   **突出线条与结构**：圈出建筑边缘、砖墙、衣物纹理等任何线条，增加**结构**值（`+20` 到 `+50`），能让画面更清晰、有质感。
*   **提亮主体**：圈出脸部，适当增加亮度（`+10` 到 `+20`），并轻微降低对比度（`-5` 到 `-10`），可以在不磨皮的情况下让脸部更干净。
*   **强调细节**：圈出鞋、背包、品牌 Logo 等具辨识度的物品，通过提高亮度或结构来突出它们。

> **操作提示**：双指在屏幕上可缩放局部调整的影响范围，确保红色覆盖区精准对应你想修改的区域。

## 美图秀秀：五官与身形精修 ✨

当照片的整体光感和结构通过 Snapseed 优化好后，观众才会注意到你本人。这时，我们使用美图秀秀进行第二步的精修。

进入“人像美容”下的“面部重塑”功能。以下是面部调整的核心思路：

*   **眼睛**：避免过小或过大。如果眼睛无神，优先尝试“提高”功能，其次才是“放大眼睛”。
*   **嘴巴**：根据脸型调整大小，通常稍微调小会更精致。
*   **脸型**：根据你是圆脸、方脸还是鹅蛋脸，适度使用“瘦脸”功能。例如，方脸可稍瘦，鹅蛋脸则少动。
*   **发际线与鬓角**：可用于修饰额头形状或填补鬓角空缺。
*   **身形调整**：使用“瘦身瘦腿”功能。
    *   **拉长小腿**：只针对膝盖到脚踝的部分进行拉长，这是显高的自然关键。
    *   **调整肩宽**：适当加宽肩膀，可以优化头身比。注意背后背景不要有大量直线条，否则拉伸时会导致背景扭曲。

## 课程总结与行动指南 🚀

本节课我们一起学习了从“软件认知”到“流程实践”的完整修图入门知识。

我们来总结一下核心链条：
1.  **审美先行**：追求干净、鲜亮、有线条感的整体照片质感。
2.  **流程固化**：Snapseed 调光（全局+局部）→ 美图秀秀修容（五官+身形）→ Facetune 微调（可选）。
3.  **参数活用**：所有数值都是变量，不要死记，关键在于理解“亮度加、对比减、结构提”的感觉。
4.  **拍摄是基础**：修图的前提是拍出有光、有线条、有内容（如 Logo、特定场景）的原图。避免在杂乱或无特征的背景前拍摄。

> **最后也是最重要的**：知识需要实践。请立刻开始拍照，应用本节课的方法进行修图，并将成果分享出来以获得反馈。你的形象提升之旅，始于按下快门和动手修图的第一刻。
