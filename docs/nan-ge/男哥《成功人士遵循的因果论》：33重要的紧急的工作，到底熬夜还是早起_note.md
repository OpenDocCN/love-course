# 男哥《成功人士遵循的因果论》第33课：重要的紧急工作，到底熬夜还是早起？⏰

在本节课中，我们将要学习一个关于时间管理的重要决策：面对一项重要且紧急的工作时，我们究竟应该选择熬夜完成，还是选择早起处理。我们将通过对比两种方式的效率与影响，帮助你做出更明智、更健康的选择。

## 核心决策：熬夜 vs. 早起

上一节我们讨论了睡眠的重要性，本节中我们来看看一个具体的应用场景。当你面临一项必须在短时间内完成的重要紧急工作时，在“熬夜”和“早起”之间，我建议选择早起。

例如，一项需要**3小时**完成的工作：
*   **熬夜方案**：从晚上11点工作到凌晨2点。
*   **早起方案**：从早上5点工作到7点半。

如果你发现某项工作需要熬夜数小时才能完成，那么选择早起通常会更加高效。

## 个人实践与观察

我本人是早起方案的受益者。我的工作有时需要赶项目或做总结，很容易就持续到凌晨四五点甚至更晚。但我发现，每天晚上9点、10点之后，我的工作效率会变得极低。

因此，我调整了策略：
1.  我在晚上11点或12点获得高质量的睡眠。
2.  我选择早起工作。

我发现，一个上午完成的工作量，可以媲美甚至超过整个下午加晚上的产出。我周围一些事业有成的朋友也采用类似方法。

以下是他们的典型做法：
当面临需要推进大量项目时，他们会选择在晚上8、9点就入睡休息。然后，根据工作量预估，在凌晨4点到5点起床，之后吃饭、工作、写报告、阅读学习，一气呵成地工作到早上8、9点。

## 效率对比分析

这种方法的好处是双重的：
*   **完成了工作**：他们完成了原本需要熬夜才能做完的事情。
*   **避免了熬夜**：到凌晨4点时，他们已经睡了6到7个小时，精力充沛。

相比之下，如果选择熬夜，到了深夜一两点，人会因为疲惫和缺乏休息而效率低下。这就是为什么我一开始就说：千万不要认为对自己狠、能熬夜就能成功。你必须学会做聪明人，找到事半功倍的方法。

我不希望大家打疲劳战。凌晨12点之后，头脑昏沉，那种状态下即使你忙于项目、审阅合同或处理任务，也很可能漏洞百出。我建议每位朋友在12点前上床休息，然后在5点起床，继续你原本打算熬夜完成的事情。

这时，天亮了，你的大脑经过几个小时的睡眠休息也清醒了，效率会极高。

## 学会补充能量：为身体“充电”

第一点中的第三小点，是你要学会在适当的时候为自己补充能量、加油充电。这就像吃东西一样，我们每日三餐，但饿的时候也需要加餐。

我将其视为一些原则。平常因为工作和生活节奏紧张，我们可能无法完全控制，有些点做不到（比如有时晚上不得不熬一下）。但是，周末的时间是可以规划的。

**把“睡觉补觉”安排进我的周末日程，其重要程度与完成一个项目等同，必须严格执行。** 因为我始终相信能量守恒，欠下的睡眠债，必须找时间补上。即使是天塌下来的大事，你也应该保证基本的精神状态和睡眠时间。

做一个比喻：如果你是一个电子设备（比如iPad），周一到周五电量已被消耗殆尽。睡眠不足就像只用充电宝充一会儿电，电量从未满格。那么周末，你就必须用高效的充电器把电充满。

很多时候，我们不能出人头地、缺乏执行力、效率低下的原因，就在于不断消耗自己、内耗自己，电量从未充满过，这种情况可能持续了好几年。这就是聪明人、有能力的人与普通人的区别。

普通人可能会怀疑：如此重视睡眠，是不是不够努力、是一种懒惰、不够能吃苦？也许成功人士真的不需要太多睡眠？但我想告诉你：**想成功，先睡足7个小时**。

士兵打仗，平时训练极其艰苦，但会不会让他好几年睡不好、伙食跟不上呢？打仗时可能会处于极端痛苦的环境，无法睡觉。但高强度训练之后，要不要给他补充体能和休息呢？

**请相信一句话：“一个人必须工作到精疲力尽才能成功”是一个错误的概念和错觉，这样做的代价极高。** 如果你将职业生涯看作是30年，而不仅仅是一两年，这种每天把精力耗尽用于工作的方式，会极大损害健康，从长期看将严重影响你的生产力，最终影响你的成果。

**请再相信一句话：你人生中犯过的所有错误，都可以追溯到某个选择或情绪失误。而选择与情绪的背后，是饮食、作息、睡眠、休息和疲惫。身体的疲惫让我们的大脑无法正常运转，从而做出错误的选择和产生负面情绪。**

根据美国兰德公司的研究，仅在美国、日本、德国、英国和加拿大这五个国家，每年因睡眠不足导致的经济损失就高达**6800亿美元**。当你发现自己的效率、创造力、记忆力、学习能力和决策力在近期都不高时，睡眠与这些能力有着深远而直接的关联。

你的人生，拼的就是效率、记忆力、工作态度、工作能力和学习能力。如今，世界顶尖的投行高盛、顶尖的商业咨询公司麦肯锡，都开始聘请睡眠专家，帮助员工获得充足、高效、优质的睡眠。

所以你要明白，为什么我把“高效体能”作为第一点，并把“睡眠”作为高效体能的首要部分来讲。我希望用我的亲身经历让你明白，不要再相信那些宣扬不睡觉、少睡觉的“毒鸡汤”。我不相信有任何成功学可以颠覆人体的自然规律。**你只有先睡得好，才有精力去攻克人生中想要实现的目标。**

## 本节总结

本节课中，我们一起学习了如何在高强度工作需求下管理时间与精力。

好，我们总结一下第一点（高效体能）中关于睡眠的部分：
1.  **提高睡眠质量的基础**：养成好习惯，卧室不放电子设备，调整呼吸和肌肉放松意识，投资高质量的床上用品。
2.  **保证睡眠时长**：理想状态是每天睡够7小时。
3.  **学会科学补觉**：如果睡眠不足，要学会补觉。例如熬夜后，第二天下午3点饭困时，或在傍晚5点到8点之间，给自己一个补觉的机会。或者等到周末，好好睡上半天。
4.  **核心原则**：**宁可早起，绝不熬夜。**
