# 男哥聊天记录 课程1.0+2.0：第十节【修图黑科技】🖼️

在本节课中，我们将要学习如何通过后期修图，将你的照片提升到能够吸引他人的水准。修图是展示面打造的最后一步，也是至关重要的一步，它决定了你最终呈现在社交平台上的形象。

上一节我们讨论了拍摄技巧，本节中我们来看看如何通过后期处理，让照片达到最佳效果。

## 概述：什么是“帅哥”标准？🤵

在开始修图之前，我们必须明确一个核心的审美标准：什么样的男性照片会被认为是“帅”的？这决定了我们修图的方向和目标。

以下是构成“帅哥”形象的核心要素，请按照这个标准来塑造你的照片：

1.  **高**：视觉上的高挑感。这不仅仅是拉长腿，更需要注重细节：
    *   **腿长**
    *   **脖子长**
    *   **手指长**
    妹子会通过观察这些身体部位的比例来判断真实身高，因此修图时需全面调整。

2.  **瘦**：整体必须显瘦。视觉效果上，身高1.78米-1.9米不要超过140斤（最好130斤左右），1.7米-1.78米在120斤左右。**照片中一定要瘦**。

3.  **白嫩**：皮肤质感要好。“嫩”指皮肤状态显年轻、保养好，没有粗糙或非自然衰老的痕迹。这比单纯的白更重要。

4.  **五官端正**：按照重要性排序：
    *   **鼻子**：必须高挺。需要修整**山根**、**鼻头**和**鼻翼**三个部位。
    *   **脸型**：必须瘦，最好有清晰的下颌线。
    *   **下巴**：适当修长。
    *   **眼睛**：适当增大、增亮即可。
    *   **眉毛**：加粗、加重，要有型。

5.  **头小脸小**：头小、脸小能极大提升气质。

6.  **肩膀宽**：肩膀要修得宽阔，但需平直自然，不要修成尖角。

## 核心修图流程🔧

明确了标准后，我们进入实战环节。以下是必须严格遵守的修图步骤，全程仅使用**美图秀秀**一款软件。

### 第一步：基础美化

首先进行整体皮肤的基础处理。

*   **操作**：使用“一键美颜”功能。
*   **参数**：选择“低”档，强度调整为**50%**。

### 第二步：清洁皮肤

这一步的目的是让脸部皮肤光滑无瑕。

以下是需要使用的功能和目标：

*   **功能**：“磨皮”与“去痘”功能。
*   **目标**：去除脸上所有的痘痘、痘印、疤痕、斑点、粗大毛孔等任何瑕疵。需要放大图片仔细处理每一个细节，直到皮肤像“剥了壳的鸡蛋”一样光滑。

### 第三步：精细修整五官

这是修图的核心，需耐心处理。

以下是针对五官各个部位的修整方法：

1.  **鼻子**：
    *   **功能**：“瘦脸瘦身”工具。
    *   **操作**：推高**山根**和**鼻头**，缩小**鼻翼**。拍照时最好侧拍，便于修出立体感。

2.  **脸型与头部**：
    *   **功能**：先使用“小头”功能，再使用“瘦脸瘦身”工具。
    *   **操作**：先将头部整体缩小，再用“瘦脸瘦身”把脸颊的肉向内推，**瘦到不能再瘦为止**。

3.  **眼睛**：
    *   **功能**：“眼睛放大”和“亮眼”功能。
    *   **操作**：每只眼睛用“眼睛放大”点**10下**左右，再用“亮眼”功能提亮眼神。

4.  **眉毛**：
    *   **功能**：“亮眼”功能。
    *   **操作**：在眉毛上涂抹，使其加粗、颜色加重。

5.  **嘴唇**：
    *   **功能**：“瘦脸瘦身”工具。
    *   **操作**：如果嘴唇较厚，特别是下唇，将其修薄，会显得更清爽。

6.  **下巴**：
    *   **功能**：“瘦脸瘦身”工具。
    *   **操作**：将下巴修得**长一点**，方向是**向下并微微向前**，与高鼻梁形成协调轮廓。

### 第四步：调整全身比例

让身材在照片中显得更完美。

以下是全身各部位的调整策略：

*   **腿**：使用“增高塑形”功能，拉长**15%至45%**。注意调整范围是从**口袋位置到脚踝**。
*   **上半身**：同样使用“增高塑形”，拉长**15%**左右。
*   **脖子**：拉长**3%到5%**。
*   **肩膀**：使用“瘦脸瘦身”工具，向两侧拉宽，注意保持**平直**。
*   **整体瘦身**：使用“瘦脸瘦身”工具，将**头、腿、胳膊、手指、衣服**等所有部位修瘦，确保整体线条纤细。

### 第五步：修正变形（归位）

修图可能导致背景物体（如门窗、线条）变形，需要修复。

*   **操作**：仔细检查照片，使用“瘦脸瘦身”工具将因修图而变形的背景物体拉回原状，**最小化修图痕迹**。

### 第六步：裁剪构图

裁剪能突出主体，提升照片质感。

以下是裁剪的核心原则：

*   **比例**：优先裁剪成**正方形**，最显高。
*   **内容**：
    1.  照片最底部应是你的**脚底**。
    2.  裁掉画面中所有**低端、杂乱、无关**的物品（如灭火器、价签、不相干的人、杂物）。
    3.  裁掉**拉低照片整体逼格**的物品。
    4.  通过裁剪让人物在画面中占比更大，突出主体。

### 第七步：调色

调色让照片质感更高级。主要调整四个参数：

1.  **明暗**：根据原图光线调整亮度。环境光暗则调亮，过曝则调暗。
2.  **对比度**：控制画面的“颗粒感”或清晰度。对比度低，画面柔和像老电影；对比度高，细节锐利。皮肤瑕疵多可适当调高对比度来弱化。
3.  **饱和度**：调整照片色彩的“冷暖”。调向冷色调（蓝、白）显得清冷，调向暖色调（黄、红）显得温暖。根据照片环境调整。
4.  **滤镜**：**最好不用滤镜**，容易显得假。如果非要用，避免粉红、黑白、胶片、素描等夸张滤镜。可选“咖啡”、“电影”、“日落”等自然系关键词的滤镜。
