# 男哥《成功人士遵循的因果论》03：🎯视觉目标法让成功可视化（曝光效应：激励自己的情绪与斗志）

在本节课中，我们将要学习一种名为“视觉目标法”的强大工具。该方法通过将抽象目标转化为直观的图像，并利用“物理曝光”效应，持续激励我们的情绪与斗志，从而让成功变得清晰可见、触手可及。

## 🖼️ 什么是视觉目标法？

上一节我们探讨了如何设定可量化的目标，本节中我们来看看如何将这些目标“可视化”。视觉目标法，其核心含义是将需要解决的各项过程性目标，转化为图像形式。

具体如何操作呢？我们可以借鉴数学中常用的**X-Y坐标轴**方法。

*   画一个横坐标（X轴），代表**日期**。
*   画一个纵坐标（Y轴），代表**你需要完成的事项**。

通过这种方式，我们可以快速利用图像感官和颜色、面积等视觉元素，来刺激自己关注那些未完成的、结果导向的过程性指标。

以下是具体操作步骤：
1.  **列出目标**：将你的各项目标写在坐标轴围起来的区域里，方便随时提醒自己。
2.  **规划事项**：例如，除了直播任务，我可能还有分阶段的功课、会议安排、高校合作等事项需要科学分配。我可以把这些事项和设定的截止日期，以同样的方式在X-Y轴上体现出来。
3.  **纳入生活**：在我的视觉图表中，我还会加入用于放松的时间，比如周末的游泳、购物、旅游等活动安排。

这种视觉感官目标法，在我个人的实战经历中效果显著。它直接帮助我从2017年初创公司，走到了2018年的辉煌阶段。

你会发现这方法非常神奇，因为它就像一个**行动指南**。我们不仅拥有了数据感，更通过不同的颜色和面积区块，清晰地了解到整个目标是由哪些轻重缓急的事项组成的。

## 📍 第三要点：物理曝光法

在将目标可视化之后，我们需要让它持续出现在视线中。这就是第三个要点：物理曝光法。

具体做法是，把我们可实操的这些过程目标，写下来并放置在日常生活中高频出现的地方。

以下是我常用的曝光位置：
*   卧室的墙上
*   厕所的镜子上
*   一进门就能看到的墙面
*   设置为手机屏保，每次使用手机时都能看到

你可能会问：多看目标真的有用吗？这里涉及一个重要的心理学原理。

二十世纪六十年代，心理学家做过一个实验。他向参与者展示各种照片，有些照片出现了二十多次，有的十几次，有的只出现一两次。之后，请参与者评价对照片的喜爱程度。结果发现：人们看到某张照片的次数越多，就越喜欢这张照片。这就是心理学上常说的**曝光效应**。

所以，“心想事成”并非纯粹的唯心主义。失败者常常认为，自己没有经历更多，却非常固执地坚守现有认知，并坚定不移地相信自己“永远不行”，因为“之前努力了也没用”。物理曝光法运用了同样的原理：多看目标，多提醒自己目标的存在，看得越多，就越容易实现。

任何让你烦恼、觉得解决不了的事情，先写下来，就已经解决一半了。因为我们是在不断刺激自己的情绪和斗志。

## 💪 掌控情绪与斗志

千万不要认为做成一件事，失败仅仅是因为缺少某个客观条件。成功者与弱者的一个关键区别在于思维模式。

*   **弱者思维**：认为困难总比方法多。他们纯粹客观行事，认为解决问题必须满足一、二、三所有条件，缺一不可，否则就会失败。
*   **成功者思维**：他们是主客观相结合的人。他们不仅会把目标有效拆分、使之可执行、可量化统计并不断调整，还会把目标变成一种**信念**。

很多事情我们做不成，但成功者可以。可能一件事需要四个条件，但他只有两个条件就敢去做。他敢做的原因，不是天天空喊“我能行”，而是通过视觉方式，每天向自己“告解”目标。

这种方式，能够潜移默化地让你在遇到相关事物时，获得正向反馈。例如，你设定今年要赚一百万，并且清楚这一百万由哪些部分组成。这个目标每时每刻都在提醒你。当你遇到消耗精力的事情，比如不必要的应酬，或是按部就班却影响情绪的事务时，你都能通过视觉目标，获得正向反馈，明确自己真正的目标是什么。

人与机器、程序的区别，就在于我们拥有**斗志和情绪**。掌控情绪，就是掌控成功的关键。回顾历史，我们国家的胜利，并非仅仅依靠先进的装备，其中震撼世界的，正是那种强大的、不可战胜的斗志。好好想一想，斗志是否正是你有时失败所真正缺少的东西？

## 📝 方法总结与组合使用

本节课我们一起学习了视觉目标法与物理曝光效应。其实，以上方法不仅在我身上使用，我身边那些比我优秀的人、我的老师，乃至一些身家上亿的人，他们的工作场地或家里，也常常能看到这种记录着现阶段具体实操目标的小白板。

讲到这里，你可能会发现，这三个方法是组合起来使用的：

1.  **目标设定**：首先，确定你的目标不是空泛的愿景，而是**可实现、可量化**的具体事物。
2.  **计划分解**：然后，将目标分解到月、周、天去执行，并结合时间，制定一个**有数据、有数字**的详细计划。
3.  **视觉曝光**：最终，把心之所向的目标，放在最容易看到的地方，每天看一遍，提醒自己。

最后，我想分享一句我非常喜欢的话：不要觉得你的目标遥不可及，有时候，只是因为用错了方法，或对目标理解有误。如果你用对方法，正确理解，并通过那些有实操、有实战、有结果的人的经验来指导自己，如果你能够忠于目标，每天在心中构想它，你一定会抵达那片真正属于你的天空。
