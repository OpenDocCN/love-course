# 男哥1.0+2.0：2017年课程：案例：案例：案例：微信指导大卫案例12.12_note

在本节课中，我们将通过一个具体的微信指导案例，学习如何建立强势框架、展示高价值，并理解两性关系中的“共谋”概念。课程内容将拆解为几个核心部分，帮助你掌握沟通与关系推进的关键技巧。

## 一、建立强势框架与展示高价值 💪

上一部分我们了解了课程概述，本节中我们来看看如何在实际互动中建立主导地位。

案例中，当对方当着我的面打游戏时，我采取了直接而强势的回应。我要求他放下手机，并在他质疑时，强化了我的框架：“我难道是来看你打游戏的吗？”

紧接着，我通过分享个人经历来展示高价值。我表示自己大学时也玩游戏，但后来认为**游戏耗时，不如将时间用于更有意义的事情**。这种表达能瞬间提升对方对你的认知层次。

以下是展示高价值的核心要点：
*   **分享升华后的观点**：从自身经历出发，引出更积极、更建设性的价值观。
*   **提供替代方案**：指出时间的更好用途，例如“不如多联系几个客户”。
*   **避免说教与安慰**：目的不是批评或同情对方，而是传递一种更高的生活态度。

## 二、理解“共谋”：关系升级的核心 🔗

建立了初步的框架和价值展示后，关系如何深入发展呢？本节我们来探讨“共谋”这一核心概念。

强连接关系的处理，关键在于**共谋**。共谋是指双方一起谋划某事，产生共同期待，并共同努力去实现。

**关系发展公式可以简化为：共谋 -> 共同实现 -> 产生希望 -> 持续共谋**

这个循环能不断推进关系。生活由**回忆、希望和当下**组成。共谋为你们创造了共同的“希望”（未来规划）和“回忆”（一起完成的事情），而“当下”便是你们在一起相处的时光。因此，填充共同的希望至关重要。

## 三、实战话术与肢体语言解析 🗣️

理解了核心理念，我们来看看实战中的具体应用，特别是搭讪时的话术与肢体语言。

案例中提到“大善”的话术。当对方说“不用了”之后，我追加了一句：“我又不是坏人，也不是做销售的”，这给了对方一个安全合理的解释，并争取了更多时间。

然而，比话术更重要的的是你的**整体肢体语言**，它决定了你给人的第一印象。

以下是搭讪时需要注意的关键肢体语言要素：
*   **妆发与衣着**：你的外在形象。
*   **眼神**：坚定、友好的眼神接触。
*   **身体姿态**：开放、放松，不具侵略性。
*   **距离感**：保持让对方感到安全的社交距离。

即使你的开场白只是简单的“能认识一下吗？”，如果肢体语言传递出自信与安全感，对方仍可能愿意继续交谈，这便表明对方有“窗口”（兴趣指标）。

## 四、课程总结 📚

本节课中，我们一起学习了从具体案例中提炼出的关系构建技巧。

我们首先学习了如何通过**强势表达**和**高价值展示**建立初步吸引。然后，我们深入探讨了关系深入的核心——**共谋**，即通过共同规划和实现目标来创造持续的连接感。最后，我们分析了实战中的**话术策略**与至关重要的**肢体语言**。

记住，有效的互动是价值展示、共同目标建立与得体外在表现的综合体现。
