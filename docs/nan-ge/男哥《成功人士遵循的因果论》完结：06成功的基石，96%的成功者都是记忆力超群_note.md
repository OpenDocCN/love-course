# 男哥《成功人士遵循的因果论》完结：06 成功的基石，96%的成功者都是记忆力超群 🧠

在本节课中，我们将要学习记忆力对于个人成功的重要性，并探讨如何理解与提升这一关键能力。记忆力不仅是储存信息的工具，更是构建知识网络、提升决策效率的基石。

## 记忆力与成功的关系

上一节我们介绍了课程的整体框架，本节中我们来看看记忆力为何被视作成功的基石。根据《纽约时报》的一份调查，所谓的成功者中，有**96%**的人记忆力都非常好。这提示我们，当我们在抱怨环境或缺乏机遇时，其实我们自身就蕴藏着强大的潜能。

记忆力是反映一个人综合素质高低的重要指标。历史上许多杰出人物都拥有非凡的记忆力。

以下是两个著名的例证：
*   相传法国拿破仑将军能准确地记住设置在法国海岸的大炮的种类和位置。如果部下报告错误，他也能及时跟进并纠正。
*   前苏联领袖列宁的记忆力同样出色，他能准确记住国民经济统计等繁杂资料，并常常指导助手到特定书籍的特定页码去查证，助手一查果然如此。

## 记忆力在现实中的应用

理解了历史范例后，我们再将视角拉回现实。在我的观察中，无论是在情感关系处理还是赚钱能力方面，表现出色的人记忆力通常都非常好。

你可以回想身边那些优秀、出人头地或值得你学习的人，他们是否也是如此？记忆力超群，是高效工作并取得良好结果的重要组成部分。

## 记忆力的核心作用

那么，记忆力具体是如何发挥作用的呢？你记住的东西，构成了你大脑中的信息网络。你记得越快、越多、越牢固，你脑海中的世界就越丰富。**记住是使用的前提**，这也是许多实战派和有成果的人经常强调的观点。

然而，我也了解到，很多朋友正深陷记不住的烦恼之中，最明显的体现就是在许多关键节点上，最后却想不起来。
