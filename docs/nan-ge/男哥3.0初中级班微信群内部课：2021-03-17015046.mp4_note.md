以下是核心的修图功能列表：
*   **磨皮**：用于涂抹全脸，消除浅层的痘印、胡茬和皱纹。注意避开鼻子侧面、眼睛和嘴巴等需要保留细节的部位。

Facetune 的优势在于可以精细控制，避免像“美图秀秀”等软件自动磨皮后导致的整体失真或不一致。

## 五官细节的审美与调整标准 👁️👃👄

完成基础磨皮后，我们进入五官调整环节。这需要你对自己的五官有清晰的认知。调整的目标是自然、有亲和力，而非修成一个“怪物”。

以下是各部位的具体调整思路：

**1. 下巴**
标准下巴不应过尖。调整技巧在于修饰边缘，使其更有弧度，而非单纯拉长。如果嘴部较凸，建议拍摄侧面15-30度角，并可将下巴向前下方微调，以优化整体轮廓。

**2. 鼻子**
男性的鼻子宜大不宜过小、过精致，这与某些审美认知相关。调整时需谨慎：
*   **鼻尖上提**：男性较少使用，除非是鹰钩鼻。
*   **鼻尖收小**：可轻微调整，但需保持鼻头的存在感。
*   **鼻翼**：如果鼻翼过宽，可以适当收窄。
*   **鼻梁**：可使用“鼻影”功能增加立体感，但切忌修得过细过挺，会显得虚假。鼻梁塌的话，用推拉工具向前微调更自然。

**3. 脸颊（腮帮子）**
好的腮帮子应有弧度，而非刀削般尖锐或过于圆润。调整时：
*   先用“瘦脸”功能微调脸部整体宽度。
*   再针对突出的“咬肌”部位向内轻推，塑造自然的弧度。

**4. 眼睛**
调整目的不是单纯变大，而是变得“有神”。
*   **眼距**：男性的眼距通常比女性近一些，稍调近眼距能增加神采。
*   **眼睛角度**：调整眼睛的倾斜度，改善“吊梢眼”或“下垂眼”的问题。
*   **眼睛高度**：重点在于提拉上眼皮，使眼睛更有力，而非单纯放大瞳孔。

**5. 额头与发际线**
额头与太阳穴共同构成发际线。如果发际线后移或太阳穴凹陷，可通过微调额头和太阳穴的轮廓来改善。理想的比例是“发际线到眉毛：眉毛到鼻尖：鼻尖到下巴”接近1:1:1。

**6. 嘴巴**
男性嘴唇宜有一定肉感和厚度，显得性感。调整重点在于 **宽度**，而非让嘴变小。嘴宽过大会显得不够精致。其次可调整 **人中长度**，过长的“人中”会影响美观。可以将嘴巴整体向鼻子的方向轻微上提。

## 修图后的重要步骤：朋友圈管理 🔄

第一个案例修图完成后，你需要记住一个关键点：**朋友圈的一致性**。

不要修好一张照片就立刻发布。你需要积累一批质量统一的照片后再更新朋友圈。可以逐步设置“三天可见”→“一周可见”→“一个月可见”。

新加的或已有的女生看到你优质的新照片，会产生进一步了解你的兴趣。但如果她翻看你之前质量参差不齐的朋友圈，这种好感就会被破坏。因此，旧的低质量朋友圈要么隐藏、删除，要么考虑使用新的微信号用于社交展示。

保持一致性，否则所有修图的努力都可能前功尽弃。
