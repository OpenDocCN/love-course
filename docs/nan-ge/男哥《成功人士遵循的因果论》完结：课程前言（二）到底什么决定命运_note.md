# 男哥《成功人士遵循的因果论》完结：课程前言（二）到底什么决定命运_note

在本节课中，我们将要探讨一个核心问题：究竟是什么真正决定了一个人的命运？我们将逐一分析常见的误解，并揭示那个拉开人与人之间差距的关键因素。

---

很多人认为性格决定成败。**我不同意这个观点**。每一个成功人士的性格都各不相同。因此，决定命运的因素与性格无关。

那么，是否与努力有关呢？

**我也想告诉你，与努力也没有关系**。凌晨扫马路的阿姨比我们更努力。我并非贬低这个职业，只是想说明，仅仅努力是无效的。

那么，最正确的答案是什么？**决定一个人命运的是心智**。

心智才是拉开人与人之间距离的核心因素。提升心智分为五个阶段。

以下是心智提升的五个阶段：

**第一阶段：动物本能阶段**
*   这个人对自己毫无察觉和反思。
*   行为完全依托本能，认为“我觉得就应该这样”、“我认为就这么做吧”。

这个阶段的人如同动物。猪不知道自己是猪，所以它才是猪。如果猪意识到自己是猪，它绝不会甘愿做猪，哪怕钻地洞也会逃跑。

上一节我们介绍了无意识的动物本能阶段，本节中我们来看看觉醒的开始。

**第二阶段：反思阶段**
*   他开始关注自己行为背后的原因。
*   开始思考“我为什么会这么做”。
*   处于这个阶段的人，抱怨和指责会越来越少，开始全方位反思自己。

正如列宁所说：“当一个人开始反思自己的时候，他将进入一个伟大的阶段。”许多跟随我几年的兄弟，你们正在做的正是这件事：学会安静下来，学会独处，学会独立思考。因为“高处不胜寒”，当你进入第二阶段开始思考时，很多人无法理解你，因此你可能会感到孤独。

从反思行为到管理内心，是心智成长的又一步。

**第三阶段：情绪控制阶段**
*   逐渐开始接纳自己。
*   减少内耗，心态趋于平和稳定，变得非常理性。

我的课程有三节核心内容，并非教你不顾一切地蛮干或消耗自己，而是**先帮你算清楚自己有哪些内耗**。例如，你天天思考一件事却从不行动，这种思考本身就是一种内耗。在此阶段，外界评价和嘲笑对他的干扰会越来越少。

当情绪稳定后，强大的行动力便随之而来。

**第四阶段：知行合一阶段**
*   行动力开始越来越强。
*   思考时能用大脑有效控制行为，非常专注。
*   会用行动带来的变化去反哺和修正思想。
*   即使内心支离破碎，脚步也绝不会停歇。

这就是我常说的：成功者形成了一个特质——**他们构建了一套属于自己的、能够不断吸收新知并落地实践、最终产生良好效果的闭环思想体系**。他情绪再糟，内心再破碎，都不影响他做好每一件事并取得好结果，因为他已经有了自己的体系。

最终，心智将进入一种极致的专注状态。

**第五阶段：极致专注阶段**
*   对要达到的目标和结果进入完全痴迷的状态。

我们常看到一些曾经与我们水平相当的人，做出一些让我们当时无法理解、甚至觉得“傻”或“神经病”的痴迷行为。但一两年后，他们与我们已截然不同。面对同一件事，他们显得异常违心、痴迷。这种状态让当时的我们困惑，但他们做事的结果通常不会差。当然，他们同样会失去一些东西。你觉得失去的是什么呢？

---

本节课中我们一起学习了决定命运的核心因素——心智，并详细剖析了心智成长的五个阶段：从**动物本能**，到**反思觉醒**，再到**情绪控制**，进而达到**知行合一**，最终进入**极致专注**。请认真思考一下，你现在正处于哪个阶段？
