# 男哥1.0+2.0：2017年课程：案例：案例：案例：学委修图6.13_note

在本节课中，我们将学习如何仅使用手机软件，将一张背景杂乱、人物皮肤有瑕疵的普通照片，处理成背景虚化、人物突出、质感高级的精修图。我们将通过一个具体案例，分步骤讲解三个核心软件的使用方法。

## 📸 案例原图问题分析

上一节我们概述了本节课的目标，本节中我们来看看需要处理的案例原图存在哪些具体问题。

这是一张浩然导师的照片。原图放大后可以看到，背景中有行人和车辆，环境显得杂乱。建筑物上有空调等杂物，整体感觉不够简洁高端。人物面部存在油光、痘印等皮肤问题。

我们的修图目标是将这张原图处理为背景虚化、人物皮肤干净、整体色调高级的照片。

## 🛠️ 准备工作：认识三大核心软件

在开始修图前，需要准备好以下三个手机应用。它们各自负责修图流程中的不同环节。

以下是本次教程将用到的三个软件及其核心功能：

1.  **Tadaa SLR**：主要用于**抠图**和**背景虚化**。它能智能识别人物边缘，并模拟出类似单反相机的大光圈虚化效果。
2.  **Facetune 2**：主要用于**人物面部精修**。功能包括磨皮、去油光、祛痘、调整五官形状（如眼睛大小、鼻梁高度）等。
3.  **VSCO**：主要用于**滤镜调色**。拥有丰富的滤镜库，并能进行细致的参数调整（如曝光、对比度、饱和度、色调等），赋予照片独特的风格。

**提示**：部分软件可能需要付费。对于VSCO，可以通过电商平台以很低的价格获取其全部滤镜。

## 🔍 第一步：抠图与背景虚化 (使用 Tadaa SLR)

我们首先解决背景杂乱的问题。本节将使用Tadaa SLR软件，将人物主体从复杂背景中分离并虚化背景。

1.  **导入照片**：打开Tadaa SLR，点击右下角按钮，从相册导入需要处理的原始照片。
2.  **智能抠图**：
    *   点击软件中的“手指”图标，开启**边缘识别**功能。
    *   软件会自动识别人物轮廓。用手指涂抹人物主体，软件会将其标记出来（通常显示为绿色）。
    *   对于自动识别不准确的地方，可以手动精细涂抹或使用橡皮擦工具修正。
3.  **应用虚化**：
    *   抠图完成后，点击右上角的“下一步”。
    *   在虚化效果界面，建议选择 **“线性”** 模式，这样虚化效果会由近及远渐变，显得更自然。
    *   通过滑动滑块调整虚化强度，通常调到 **3-4** 的数值即可，过度虚化会显得虚假。
4.  **保存图片**：调整满意后，点击右上角的“储存”按钮，将处理后的图片保存到相册。

**核心操作总结**：`导入图片 -> 智能抠图（修正边缘）-> 选择“线性”虚化 -> 调整强度至3-4 -> 保存`。

## ✨ 第二步：面部精修 (使用 Facetune 2)

背景处理完成后，我们接下来聚焦于人物面部的美化。本节将使用Facetune 2来解决皮肤油光、痘印和五官细节问题。

1.  **导入与基础磨皮**：
    *   在Facetune 2中打开上一步保存的图片。
    *   点击下方“修图”选项，选择 **“更磨滑”** 工具。
    *   在面部有油光、痘印和皱纹的区域进行涂抹。涂抹时需注意：
        *   **避开鼻梁和眉毛**，否则会失去立体感，显得很假。
        *   通过下方滑块控制磨皮强度，以自然为准。
2.  **局部瑕疵修复**：
    *   对于明显的痘痘或斑点，可以使用 **“修复”** 工具进行点对点清除，效果更精准。
3.  **五官细节调整**（进阶）：
    *   点击下方“工具”选项，选择 **“面部”** 功能。
    *   软件会自动识别面部结构，你可以调整：
        *   `眼睛大小`：让眼睛更有神。
        *   `面部宽度`、`下巴形状`：微调脸型。
        *   `鼻子`：调整鼻子大小、宽度，甚至提升鼻梁高度。
    *   **切记**：所有调整务必适度，以自然为第一准则。
4.  **增强细节**：
    *   使用 **“细节”** 工具，涂抹眉毛、睫毛、眼球，可以使其颜色加深、更有神。
    *   同样使用“细节”工具涂抹首饰（如项链、手表），能让其更闪亮突出。
5.  **保存图片**：处理完成后，点击右上角保存。

**核心操作总结**：`磨皮去油光（避开鼻眉）-> 修复斑点 -> (可选)微调五官 -> 增强眉毛/眼睛/首饰细节 -> 保存`。

## 🎨 第三步：滤镜调色 (使用 VSCO)

完成面部精修后，照片的“硬件”部分已经达标。本节我们将通过VSCO进行最后的“软件”调色，为照片注入风格和氛围。

1.  **导入照片**：打开VSCO，导入上一步保存的精修图。
2.  **选择与调整滤镜**：
    *   可以直接在滤镜库中选择喜欢的风格应用。如果购买了全套滤镜，会有更多选择。
    *   **更推荐的方式是手动调整参数**，点击第二个图标进入编辑界面。
3.  **常用参数调整思路**：
    *   `曝光`：微调，一般不建议大幅改动。
    *   `对比度`：**适当降低**（如-1到-2），可使画面更柔和。
    *   `锐化`、`清晰度`：**适当增加**（如+1到+2），让轮廓更清晰。
    *   `饱和度`：**适当降低**，容易营造高级、沉静的感觉，黑白效果可直接将饱和度降至最低。
    *   `色温`：向蓝色方向微调，可营造冷色调高级感；向黄色方向调则显温暖。
    *   `肤色`：如果调冷色温后皮肤发青，可在此选项中向暖色（橙色）补偿。
    *   `暗角`：轻微增加，有助于突出画面中心人物。
    *   `颗粒`：轻微增加，可模拟胶片质感。
4.  **保存与导出**：调整满意后，点击保存，照片会自动导出到系统相册。

**核心操作总结**：`导入 -> 手动调整参数（降对比/饱和度，微调色温/肤色，加锐化/暗角/颗粒）-> 导出`。

## 📖 总结与练习建议

本节课中，我们一起学习了如何利用三部手机完成专业级人像修图。

我们分三步走：首先用 **Tadaa SLR** 完成抠图和背景虚化，解决环境杂乱问题；然后用 **Facetune 2** 进行深度面部精修，改善皮肤和五官细节；最后用 **VSCO** 进行风格化调色，提升照片整体质感。这三个软件的组合，基本覆盖了手机人像修图的核心需求。

**课后练习建议**：
1.  在您的手机上下载或找到功能类似的这三个软件。
2.  选择一张自己或朋友的照片作为练习素材，严格按照教程步骤操作一遍。
3.  重点关注每个步骤中的“度”的把握，尤其是磨皮强度和五官调整，目标是“自然好看”而非“判若两人”。
4.  多尝试VSCO中不同的参数组合，找到自己喜欢的色调风格。

熟能生巧，反复练习几次后，你就能快速掌握这套高效的手机修图流程。
