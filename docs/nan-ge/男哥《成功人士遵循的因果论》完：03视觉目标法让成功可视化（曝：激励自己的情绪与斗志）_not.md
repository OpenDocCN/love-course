# 男哥《成功人士遵循的因果论》完结：03 视觉目标法让成功可视化（曝光效应：激励自己的情绪与斗志）🎯

在本节课中，我们将要学习一种强大的目标管理方法——视觉目标法。我们将探讨如何将抽象的目标转化为直观的图像，并利用“物理曝光”效应持续激励自己，从而让成功的过程变得清晰可见、触手可及。

## 什么是视觉目标法？📊

上一节我们介绍了如何将目标拆解为可执行的步骤。本节中，我们来看看如何将这些步骤“可视化”。

视觉目标法，指的是将各种需要解决的过程性目标，转化为图像的形式。具体做法可以借鉴数学中常用的**X-Y坐标轴**。

你可以画一个横坐标（X轴）和一个纵坐标（Y轴）。其中，**横坐标代表日期**，**纵坐标（Y轴）则代表你需要完成的事项**。

通过这种方式，我们可以快速利用图像感官和颜色、面积等视觉元素，来刺激自己关注那些未完成的、结果导向的过程性指标。

## 如何构建视觉目标图？✍️

以下是构建个人视觉目标图的具体步骤：

1.  **列出各项目标**：将你的各项核心目标，写在坐标轴围起来的区域里，方便随时提醒自己。
2.  **分解阶段任务**：例如，除了日常直播，你可能还有分阶段的功课、会议安排、高校事务分配等。将这些任务连同设定的截止日期，以同样的方式在X-Y轴上体现出来。
3.  **纳入生活平衡**：在我的视觉图表中，我还加入了用于放松的时间，比如周末的游泳、购物、旅游等活动。这确保了目标与生活的平衡。

这种视觉感官目标法，在我个人的实战经历中效果显著。它直接帮助我从2017年初创公司，走到了2018年的辉煌阶段。

你会发现这方法非常神奇，因为它就像一个**行动指南**。我们不仅拥有了数据感，更能通过不同的颜色和面积区块，清晰地了解整个目标由哪些轻重缓急的事项组成。

## 第三要点：物理曝光法🔍

除了制作图表，我们还需要让目标持续出现在视线中，这就是物理曝光法。

具体操作是，把我们可实操的这些过程目标，写下来并放置在日常生活中高频出现的地方。我经常把它们挂在卧室、厕所，以及一进门就能看到的墙上。我还会将其设成手机屏保，这样每次使用手机时都能看到。

你可能会问：多看目标真的有用吗？

这里涉及一个广为人知的概念——**“心想事成”大法**。无论是冥想、某些宗教，还是佛教，都非常认可一件事：将你的目标清晰地写下来，放在枕边和日常能经常看到的地方，这件事就更容易实现。

这听起来或许不可思议，但事实就是如此。这就是成功人士比你强的一个关键点。

我们想一想，你生活中可能有很多问题需要解决。弱者通常会认为“困难总比方法多”，因为他们总是在**客观**地做事。所谓客观，就是认为一件事要解决，就必须满足一、二、三……所有条件，少一个就会失败。

但你会发现，任何领域的成功者，都是**主观与客观相结合**的人。他们不仅会把目标有效拆分、使之可执行、可量化统计并不断调整，还会把目标变成一种**信念**。

很多事情我们做不成，但他们能做成。有时一件事需要满足四个条件，但他们可能只满足两个就敢去做。他们敢做的原因，并非天天喊“我能行”，而是因为他们**每天通过视觉方式告诫自己目标所在**。

这种方式能够“力甘健饮”地（意指潜移默化、持续滋养），让你在遇到相关事物时，给自己正向的反馈。

例如，你今年的目标是赚一百万，并且清楚这一百万由哪些部分组成。这个目标每时每刻都在告诫你。当你遇到消耗精力的事情，比如不必要的酒局，或是工作中按部就班却影响情绪的事务时，你都能通过视觉提醒，获得正向反馈，明确自己的核心目标是什么。

## 曝光效应的科学依据🧠

此外，我想跟大家分享一个心理学实验。

在二十世纪六十年代，心理学家罗伯特·扎伊翁茨（Robert Zajonc）进行了一项实验。他向参与者展示各种照片，有些照片出现了二十多次，有的十几次，有的只出现一两次。之后，请参与者评价他们对照片的喜爱程度。

结果发现：**人们看到某张照片的次数越多，就越喜欢这张照片**。这就是心理学上著名的 **“曝光效应”**（Mere Exposure Effect）。

所以，你说“心想事成”大法是唯心主义吗？或许有太多争论。但我想说的是另一件事：失败者往往认为自己没有经历更多内容和文化，却非常固执地坚守现有认知，并“义不容辞、身不由己、坚定不移”地坚信自己永远不行，因为“之前努力了也没用”。

其实本节提到的物理曝光法，也是同样的道理。**多看目标，多提醒自己目标的存在，越看就越容易实现**。这个道理无需过多阐述。

包括这一点：只要是你烦恼的、觉得解决不了的事情，**先写下来，就已经解决一半了**。因为我们不断在刺激自己的情绪和斗志。

千万不要认为做不成一件事，仅仅是因为当初某个条件不具备而失败了。人的成功有时确实有偶然性，但那个人成功了，在你看来为何偶然？就是因为他除了有清晰、量化的过程指标服务于结果之外，他还明白：**人与动物、电脑、程序的区别，就在于拥有斗志和情绪**。这就叫掌控情绪。

我想告诉大家的是，这么多年来我就是使用这些方法。后来我还将这个方法提炼出来，不仅自己用，也分享给身边同等价值观的朋友。

我建议大家可以这样做：我一般会买四块白板。

以下是布置曝光环境的建议：

1.  **多场景布置**：你可以将白板挂在墙上、冰箱上、衣柜上。
2.  **覆盖行动路径**：我经常在下午工作到深夜的动线上，包括工作区、回家必经处、厕所甚至客厅，都会放一块白板。
3.  **清晰书写目标**：在白板上清晰地写上我可实操的、结果导向的过程目标。

每天看一看，目标真的离你那么近。

我们想一想，我们的国家是如何取得胜利的？真的是因为当时的自动化设备很强、武装很先进吗？全世界研究军事的人都会震撼并思考：我们现有的国家建立，为什么当时的人们会有那样强大的斗志？

好好想一想，**斗志**，是不是你有时失败时，真正缺少的东西？

## 总结与共勉🌟

本节课中我们一起学习了视觉目标法与物理曝光法。

其实以上方法，不仅仅在我身上使用。我身边那些比我优秀的人，甚至我的老师，以及那些身家上亿的人，有一次我去他们的工作场地或家里，都看到了这种小白板，上面记录着他们现阶段某个模块的具体实操内容。

讲到这里，兄弟们可能会发现，其实三个方法是组合起来使用的：

1.  **目标具体化**：先确定你的目标不是空泛的愿景，而是可实现、可量化的东西。
2.  **计划拆解**：然后将其变成月、周、日的计划去执行。
3.  **数据化与视觉化**：结合时间，做一个有数据、有数字、少用形容词、多用名词和数量词的计划。
4.  **持续曝光**：最终，我们要把心之所向的目标，放在最容易看到的地方，每天看一遍，提醒自己。

最后，我想跟大家分享一句我非常喜欢的话：

**不要觉得你的目标遥不可及，有时是因为你用错了方法，或对目标理解有误。如果你用对方法，正确理解，并通过那些有实操、有实战、有结果的人传授经验，同时能够忠于你的目标，每天“心想事成”你的目标，你一定会碰到那片真正属于你的天空。**
