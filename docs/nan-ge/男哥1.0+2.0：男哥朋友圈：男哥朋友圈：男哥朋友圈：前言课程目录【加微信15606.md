## 概述

Python是一种易于学习且功能强大的编程语言。本节将介绍Python的基本概念，为后续学习打下基础。

## 变量与数据类型

上一节我们概述了Python，本节中我们来看看变量和数据类型。变量用于存储信息，数据类型定义了信息的种类。

Python有以下几种基本数据类型：
*   **整数**：表示没有小数部分的数字，例如 `10`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`。
*   **字符串**：表示文本信息，用单引号或双引号包围，例如 `"Hello"`。
*   **布尔值**：表示真或假，即 `True` 或 `False`。

定义一个变量的代码如下：
```python
name = "小明"  # 字符串变量
age = 25       # 整数变量
height = 1.75  # 浮点数变量
```

## 基本运算

了解了如何存储数据后，我们来看看如何操作它们。Python支持多种基本运算。

以下是常见的运算符：
*   **算术运算符**：用于数学计算，如 `+`（加）、`-`（减）、`*`（乘）、`/`（除）。
*   **比较运算符**：用于比较值，如 `==`（等于）、`>`（大于）、`<`（小于）。
*   **逻辑运算符**：用于组合条件，如 `and`（与）、`or`（或）、`not`（非）。

使用算术运算符的示例：
```python
a = 10
b = 3
sum_result = a + b  # 结果为13
div_result = a / b  # 结果为3.333...
```

## 控制流程：条件判断

程序需要根据不同情况做出决策，这通过条件判断实现。

`if` 语句的基本结构如下：
```python
if 条件:
    # 条件为真时执行的代码
else:
    # 条件为假时执行的代码
```

例如，判断一个数字是否为正数：
```python
num = 5
if num > 0:
    print("这是一个正数")
else:
    print("这不是一个正数")
```
