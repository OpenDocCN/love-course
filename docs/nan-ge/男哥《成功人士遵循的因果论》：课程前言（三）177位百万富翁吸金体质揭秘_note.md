# 男哥《成功人士遵循的因果论》课程前言（三）🔍：177位百万富翁吸金体质揭秘

在本节课中，我们将要学习一项基于177位百万富翁习惯的研究成果。我们将揭秘他们普遍具备的九大核心特质，这些特质共同构成了他们的“吸金体质”。理解并实践这些原则，将为你迈向成功提供清晰的路径。

上一节我们介绍了成功的基本心态，本节中我们来看看那些已经取得成就的人具体在做些什么。通过对大量成功人士的观察，我们可以总结出一些共性的、可学习的行为模式。

---

## 1. 📚 坚持为知识而阅读

研究发现，80%的富人每天都会阅读至少30分钟。他们的阅读主要以自学和自我提升为目的，而非为了娱乐。

富人阅读是为了获取知识。他们倾向于阅读以下三类书籍：
*   **成功人士的自传**
*   **个人修养与发展类书籍**
*   **历史类书籍**

我的课程将着重教你如何选择对的内容。同时，如何提高在短时间内的高效阅读能力，是成功人士应具备的知识。不要总是抱怨没有机遇或条件。你需要思考的是，那些成功人士在底层时，也和你一样。他们善于利用现有的环境资源和心智。

## 2. 🏃‍♂️ 坚持体育锻炼

76%的富人坚持每天进行30分钟以上的有氧运动，例如跑步、快走、骑自行车等。有氧运动不仅对身体有益，对大脑也很有帮助。

在我职业生涯初期的复盘中也曾提到，当我发现一个问题在短期内无法解决，用尽所有人脉、工作方法甚至组织力量都无效时，我记住一个信念：**生命在于运动**。我依然每天工作12小时，但会调整时间进行健身。因为我知道，乐观的态度取决于我的运动。结果在第二、三个月，问题往往被莫名解决，并带来新的突破。

## 3. 🤝 结交成功人士

你会和你经常来往的人一样成功。富人总是寻找目标明确、乐观热情、心态积极的人做朋友。

如果你的室友或周围朋友充满负能量，你没必要为了合群而听从他们。同时，也要避免与负能量的人或环境接触。这就是我课程中提到的：你要了解人性。人无法身处负能量的环境和人群中而不受影响。既然知道人性有此弱点，就不要去挑战它。

## 4. 🎯 追求自己的梦想与目标

追求自己的梦想和目标能产生长久的幸福感。你的幸福指数可能是事业，也可能是美好的婚姻，那就去追求它，不要人云亦云。这最终会转化为对你有帮助的财富。

不要追逐别人的梦想。富人不断修正自己的目标，并坚韧且充满激情地去追求。激情为你提供了克服失败、错误和拒绝的能力，以及坚持的定力。

## 5. ⚖️ 努力工作，尽情玩乐

不要认为每个富人都对自己非常苛刻。实际上，富人比你更懂得玩耍。他们更会在努力之后奖励自己，实现正向反馈。

富人很少因为骄奢淫逸而失败，他们通常是因为某笔投资失误。所以不要学习那些错误的认知。你努力工作的目的是为了更好地、有质感地活着，去追求更高的幸福指数。**自律是为了能更好地玩乐**，而不是为了“有意义”而天天苦行。

## 6. 💰 拥有多种收入来源

一定要明白“一专一副多涉猎”。对任何事情都保持乐观的好奇心去了解，但要有一个主要的收入来源。

在我的研究中，65%的富人在赚到第一个100万时，已经拥有至少三种收入来源。额外的收入来源有很多种，特别是在当今时代。

## 7. 👨‍🏫 拥有自己的导师

我常说“选择大于努力”。你需要在红尘中历练，同时要选择一位正确的导师。成功的导师不仅会对你的生活产生实际影响，告诉你什么该做、什么不该做，还能定期活跃在你成功的道路上。

他们与你分享的是有价值的成功经验，而非空洞理论。这些经验来自他们的实战、他们的导师，或来自曾经的失败打击。所以，你一定要学会**站在巨人的肩膀上**，而不是闭门造车。

## 8. 😊 保持积极的人生态度

只有当你拥有积极的精神状态时，才能获得长期的成功。积极向上是所有白手起家的百万富翁的标志。

问题在于，大部分人无法确定自己的想法是积极还是消极。所以，需要停下来倾听自己的思想，感受它们的存在。我认为冥想是一个很好的方法，它能让你意识到这些消极思想的存在。正确面对消极情绪和自身缺陷，减少内耗，意识到它们的存在是关键。

## 9. 🚶‍♂️ 不从众，创造自己的圈子

我们总是希望融入社会，因此尽力不让自己过于突出。但“融入”其实是一个错误的方向，它无法让你与他人区分开来，这正是大部分人无法获得成功的原因。

你可能会想：“难怪我这么做不太好，因为没人这么做过。” 无论是在开场、互动还是其他事情上。成功的人创造的是自己的圈子，并把他人拉进来。你需要做的是突破自己，然后创造你自己的圈子，让别人来加入。
