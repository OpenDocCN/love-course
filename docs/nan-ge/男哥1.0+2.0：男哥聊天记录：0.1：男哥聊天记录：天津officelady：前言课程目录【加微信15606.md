# 课程1：Python基础入门 🐍

在本课程中，我们将学习Python编程语言的基础知识，包括变量、数据类型和基本操作。

## 概述

Python是一种简单易学的编程语言。本节将介绍其核心概念，为后续学习打下基础。

## 变量与数据类型

上一节我们概述了Python，本节中我们来看看如何存储信息。

变量用于存储数据。Python有几种基本数据类型。

以下是主要的数据类型介绍：
*   **整数**：表示没有小数部分的数字，例如 `age = 25`。
*   **浮点数**：表示带有小数的数字，例如 `price = 19.99`。
*   **字符串**：表示文本信息，用引号包围，例如 `name = “张三”`。
*   **布尔值**：表示真或假，即 `True` 或 `False`。

## 基本运算

了解了如何存储数据后，我们来看看如何操作它们。

Python支持多种数学和逻辑运算。

以下是常见的运算符：
*   **算术运算符**：用于数学计算，如加（`+`）、减（`-`）、乘（`*`）、除（`/`）。
*   **比较运算符**：用于比较值，如等于（`==`）、大于（`>`），结果返回布尔值。
*   **逻辑运算符**：用于组合条件，如与（`and`）、或（`or`）。

例如，计算两个数的和可以用公式 `sum = a + b` 表示。

## 控制流程

程序并非总是直线执行。控制流程语句允许我们根据条件改变代码的执行路径。

`if` 语句是最基本的条件判断结构。

其基本语法用代码描述如下：
```python
if 条件:
    # 如果条件为真，执行这里的代码
else:
    # 如果条件为假，执行这里的代码
```

## 总结

本节课中我们一起学习了Python的基础知识。我们介绍了变量与数据类型、基本运算以及控制流程中的条件判断。掌握这些概念是编写Python程序的第一步。
