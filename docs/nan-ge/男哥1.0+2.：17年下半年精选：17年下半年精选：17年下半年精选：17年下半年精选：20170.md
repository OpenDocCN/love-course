### 修图实战案例：人物肖像精修

我们将通过一张原图，分步骤演示如何修出质感人物照。

**第一步：背景虚化与环境净化**
使用 **Tadaa SLR** 应用。
1.  导入原图。原图问题：背景杂乱，有无关人物和车辆，主体不突出。
2.  使用“抠图”功能，用画笔将人物主体大致勾勒出来。软件会自动识别边缘，细微处可手动调整。
3.  将人物全部涂满后，点击下一步。选择“线性”虚化模式以使效果更自然。
4.  调节虚化强度（通常3-4即可），使杂乱背景模糊，突出人物主体。
5.  保存图片。

**第二步：面部精修与细节提升**
使用 **Facetune 2** 应用。
1.  导入上一步保存的图片。
2.  **磨皮去痘**：使用“修图”中的“更平滑”功能，涂抹面部油光、痘印、皱纹区域。注意避开鼻梁和眉毛轮廓，以保持自然。
3.  **去除瑕疵**：使用“修复”功能，点掉明显的斑点或痘痘。
4.  **调整五官（可选）**：在“工具”中选择“面部”，可微调眼睛大小、鼻子形状和高度、脸部宽度等，使面容更精致。
5.  **强化细节**：使用“细节”功能，涂抹眉毛、眼线、项链、手表等部位，使其颜色加深、更突出有神。
6.  **修饰眉毛**：如果眉毛稀疏，使用“上色”功能，选取原有眉色进行填补，画出自然眉形。
7.  保存图片。

**第三步：滤镜调色与风格定调**
使用 **VSCO** 应用。
1.  导入精修后的图片。
2.  推荐自行调整参数以获得独特风格：
    *   **曝光**：微调，避免过度。
    *   **对比度**：适当降低，营造柔和感。
    *   **锐化/清晰度**：轻微增加，提升质感。
    *   **饱和度**：降低，可营造高级灰或黑白感。
    *   **色温**：向蓝色（冷色调）微调，显得干净。
    *   **肤色**：向黄色微调，补偿冷色调下的肤色。
    *   **暗角**：轻微增加，突出中心主体。
    *   **颗粒**：轻微增加，增添胶片质感。
3.  也可直接选用VSCO内置滤镜（可通过淘宝以极低成本购买全套滤镜）。
4.  导出最终成片。
