# 男哥的Python入门课 #1：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础、最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等），而标签（变量名）则帮助我们记住盒子里装的是什么。

在Python中，我们使用等号 `=` 来给变量赋值。其基本语法是：

**`变量名 = 值`**

例如：
```python
name = "男哥"
age = 30
```
这里，我们创建了两个“盒子”：一个叫 `name`，里面存放了文字“男哥”；另一个叫 `age`，里面存放了数字30。

以下是给变量命名的几个关键规则：
*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如 `Age` 和 `age` 是两个不同的变量）。
*   应避免使用Python的保留关键字（如 `if`、`for`、`while`等）作为变量名。

## 认识基本数据类型 🔢

我们已经学会了如何用变量来“装”东西，那么Python的“盒子”里都能装哪些类型的东西呢？这就是数据类型要回答的问题。Python有一些内置的基本数据类型，最常用的有三种。

### 1. 整数与浮点数

这类数据用于表示数字。
*   **整数**：没有小数部分的数字，例如 `-10`, `0`, `100`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。

在代码中，你可以直接使用它们：
```python
my_int = 100    # 这是一个整数
my_float = 3.14 # 这是一个浮点数
```

### 2. 字符串

字符串用于表示文本信息。在Python中，你需要用单引号 `'` 或双引号 `"` 将文字括起来。

```python
greeting = "Hello, World!"
author = '男哥'
```
一个有用的技巧是，你可以使用加号 `+` 来将多个字符串连接在一起：
```python
full_message = greeting + " I am " + author
print(full_message) # 输出：Hello, World! I am 男哥
```

### 3. 布尔值

布尔值只有两种可能：**真** 或 **假**。在Python中，它们用 `True` 和 `False` 来表示（注意首字母大写）。这种类型常用于逻辑判断。

```python
is_learning = True
game_over = False
```

## 如何查看数据类型？ 🔍

有时候，你可能不确定一个变量里装的是什么类型的数据。别担心，Python提供了一个非常方便的函数 `type()` 来帮你查看。

其使用格式为：

**`type(变量名或数据)`**

让我们通过一个例子来实践一下：
```python
height = 180
weight = 70.5
nickname = "大壮"

print(type(height))    # 输出：<class 'int'>
print(type(weight))    # 输出：<class 'float'>
print(type(nickname))  # 输出：<class 'str'>
print(type(True))      # 输出：<class 'bool'>
```
`<class ‘int’>` 表示“整数类”，告诉我们 `height` 是一个整数。同理，`‘float’` 是浮点数，`‘str’` 是字符串，`‘bool’` 是布尔值。
