# 男哥《亲密关系》课程10：智慧沟通的两个基本功 💬

在本节课中，我们将要学习智慧沟通的两个核心基本功。掌握这些基础技能，是后续学习具体沟通方法和技巧的前提。课程的核心在于，我们必须先打好基础，才能进行有效的练习和提升。

## 概述

本节课将介绍智慧沟通所依赖的两个最重要的基本能力。在讲解具体的练习方法和技术之前，我们必须先理解和掌握这两个根基。

## 核心基本功一：准确表达情绪

上一节我们概述了基本功的重要性，本节中我们来看看第一个基本功：准确表达情绪。

笼统地表达“不开心”或“不舒服”是不够的。我们需要将情绪表述准确。例如，“不舒服”可能包含很多种具体的情绪。

以下是需要准确区分的情绪示例：

*   **负面情绪**：伤心、尴尬、内疚、生气、委屈、焦虑、慌张、紧张、迷茫、孤独、遗憾。
*   **正面情绪**：高兴、自豪、开心、自信、感激、温暖、幸福、满足、欣慰。

把自己的正向情绪或者负面情绪用准确的词汇表达出来，这是非常重要的。语言表达能力是智慧沟通的基础之一。

## 核心基本功二：积累情绪词汇

那么，语言表达能力是如何练习的呢？这依赖于平时的积累。

我们需要多去看、多积累这些描绘情绪和感受的词汇。丰富的词汇库是进行准确表达的前提。这是我们要学会智慧沟通的第二个最重要的基本功。

## 总结与行动

本节课中我们一起学习了智慧沟通的两个基本功：**准确表达情绪**和**积累情绪词汇**。

听完这节课之后，我们需要在与伴侣沟通时，有意识地去注意并运用这两个基本细节。在做到这两点之后，下节课将会讲解如何通过一个简单的“六句话”方法来练习表达能力。

接下来的关键在于我们自己去实战，改正过去一些错误的、不恰当的表达方式。
