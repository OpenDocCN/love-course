# 《爱商训练营》第13课：KTV聚会游戏指南（下集）🎤

在本节课中，我们将继续学习KTV聚会中的互动游戏。这些游戏旨在逐步提升现场氛围，创造与你心仪对象自然互动的机会。我们将介绍几种经典游戏的规则、策略与注意事项，帮助你成为聚会中的全能达人。

## 热身游戏：抓手指👆

上一节我们介绍了适合开场的热身游戏“抓手指”。当大家彼此还不熟悉时，这个游戏能有效提升现场情绪与气氛。

然而，仅玩一个游戏可能导致场面枯燥。因此，我们需要准备更多游戏，让暧昧氛围逐步升温。

## 进阶游戏一：杯数猜拳🥂

以下是“杯数猜拳”的游戏规则。

*   所有参与者一起进行石头剪刀布。
*   出拳时需同时喊出一个杯数，例如“半杯”、“一杯”。
*   杯数通常按“半杯、一杯、一杯半、两杯……”的顺序逐级增加。
*   最终输掉猜拳的人，需喝下当前喊出的杯数的酒。

这个游戏需要注意以下事项。

*   酒的度数不宜过高，杯子不宜过大，以防有人过快喝醉。
*   尤其需避免女生喝醉，或让对方感觉你在刻意灌酒。
*   若你或同伴喝醉，将无人协助聊天，更可能错过后续机会。

因此，选择合适的酒和杯子大小至关重要。

## 进阶游戏二：骰子游戏🎲

玩骰子是一个规则简单但富有策略的游戏，过程比较刺激。

其核心规则是：骰子点数“1”可以代表任何数字。南北方的具体玩法（如“加摘”或“不加摘”）略有不同，你可以通过简单搜索进行了解和学习。

在本课程中，我运用了“虚张声势”的策略。

**策略示例：** 故作镇定，让女生误以为我持有大量她所喊的数字，诱使她不断加高喊数，直至我选择“开”她的骰盅。

这个游戏需要你与朋友多加练习，方能熟能生巧。

## 进阶游戏三：我爱你/不要脸💖

“我爱你/不要脸”是一个规则特别简单但极易出错的传话游戏。

以下是游戏规则。

*   参与者围成一圈，按顺序向旁边的人说一句话。
*   正向传递时说“我爱你”。
*   逆向传递（即改变传递方向）时说“不要脸”。

虽然规则简单，但玩起来特别容易晕头转向而导致说错。

当与你心仪的女伴坐在一起玩此游戏时，它能成为一个极佳的第三方媒介。你可以利用游戏规则，向她传递你想表达的情感，从而更容易营造暧昧氛围。由于处在游戏语境中，对方通常不会明确拒绝或说破。

## 进阶游戏四：转盘/酒瓶惩罚游戏🔄

现代KTV通常备有各种小道具，如转盘。若无道具，可用酒瓶或酒杯代替。

游戏规则是：指针或瓶口转向谁，谁就接受惩罚。输家接着转动道具指向下一个人。

但需注意，惩罚若总是喝酒会显得乏味。除了喝酒，真心话大冒险是绝佳的选择。

当你的女伴输了游戏，需要你的僚机配合时，可以遵循以下策略。

*   若她选择“大冒险”，僚机可提议让她与你拥抱或亲吻脸颊，这都是良好的互动升级方式。
*   若她选择“真心话”，你们可以互相询问一些较为私密的问题，例如“初恋在什么时候？”或“谈过几个男朋友？”。

无论是肢体接触还是言语交流，都应遵循“由浅入深、由不敏感到敏感”的原则，否则容易引起对方不适。

## 压轴游戏：纸巾传递😘

这个游戏无需过多解释，它能极度容易地产生暧昧氛围。

但请注意，此游戏更适合在聚会后期进行。此时大家已更加放松，酒也喝得差不多，不再想继续饮酒。

游戏规则是：用嘴传递纸巾，若未接住纸巾则需罚酒。因此，为了避免喝酒，你与她的距离会因纸巾越来越小而被迫越来越近。

游戏要点是：尽量向后仰头去接纸巾，而不是向前凑。否则可能适得其反。

## 情绪高点与转场策略🚪

当现场情绪与气氛达到高点后，你们便可以计划转场离开。

转场的方法有多种，例如邀请她“陪我去趟洗手间”。只要之前的互动良好，她通常不会拒绝。

KTV聚会的核心要点可以总结为：**要走动起来，多利用周围的人、场景、道具以及转场的机会，保持整个约会过程的能量不低落。**

至于聚会后是否要一起吃饭，真的取决于你对当时“窗口”（对方好感度）的识别能力。请记住：**过程大于结果**。在KTV给对方留下良好印象，能为后续的聊天激发更多话题和互动。
