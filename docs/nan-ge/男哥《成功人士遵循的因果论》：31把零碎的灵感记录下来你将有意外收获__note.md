# 男哥《成功人士遵循的因果论》：31把零碎的灵感记录下来你将有意外收获__note

在本节课中，我们将要学习如何利用日常生活中那些看似零碎、无意识的“碎片时间”，来捕捉灵感、激发创造力，并最终将这些零散的“珍珠”串联成有价值的成果。这是一种高效工作与节省时间的核心方法。

## 认识“无意识时间” 🧠

上一节我们介绍了灵感的重要性，本节中我们来看看捕捉灵感的最佳时机——“无意识时间”。

所谓“无意识时间”，定义非常广泛。基本上，当你没有在进行高强度的工作、学习时，都可以算作无意识时间。你不能在专注学习我的课程时，思维跑偏去想其他事情。除了高强度工作学习以外的时间，都属于无意识时间。

例如：
*   你跑步时可以在思考。
*   你遛弯时可以在思考。
*   你品尝美食时可以在思考。
*   你躺在床上时可以在思考。
*   你排队时、在地铁里、等飞机时可以在思考。
*   你聊微信时，思维也可能迸发灵感。

你可能会听说一种方法，教你用这些时间去重复记忆知识点或回顾内容。然而，那些让你感到棘手、即使花费大量专注时间也无法解决的、需要创造力的事情，恰恰需要利用这种碎片化的、无意识的“珍珠”，最后将它们穿成线。

这是一种非常好的、高效工作并节省时间的办法。

## 捕捉灵感的三个实用方法 🛠️

你可能觉得这方法靠谱吗？但我认为这是一个非常重要的宝藏，并且我会持续这样做。以下是具体操作方法。

### 方法一：善用手机备忘录

我随手想到任何碎片化的事情和灵感，都会记录在手机的备忘录里。例如，我今天提到的某个概念（WRH），或一次有启发的交流，都要立刻记录下来。我的备忘录里，充满了各种碎片化、不同结构的记录。

我曾经看过我那位融资了千万的老板的手机备忘录。他说：“小易你看一下我的备忘录。”里面全是他学习的心得、在路途中的思考，并且不断往上补充。我告诉你，真正有钱人的备忘录，如果你有机会去看一下，里面记录的都是他们在路程上、喝咖啡时、甚至看病时思考的问题。别人问我：“南哥你想啥呢？”我说我在思考、记录心得。这看起来很像一个摘抄本。

所以，手机备忘录，你们到底有多少人用起来了？

### 方法二：巧妙运用微信文件传输助手

因为有时候，你记录不方便，或者没时间记录。当灵感涌现，我来不及写，急得冒汗时，我就先把它变成语音，用微信文件传输助手发给自己。可能就说三分钟，我也不去思考这段话是否完美，因为前面的课程也学过，我不去纠结。先把想法说出来，记录下来。

之后，在我复盘的时候，再看这三句话能不能展开，哪里不通顺再优化。所以，当灵感来得太快，写字打字都来不及，就需要第二个工具——微信文件传输助手。它是你自己对自己说话的工具，可以解决你随时需要记录但打字不便的问题。

### 方法三：大胆分享，借助他人视角

如果你身边有别人，比如在饭桌上，或者在微信群里，千万不要吝啬你的想法。趁机就可以跟朋友去分享、讨论出来。这也是我前面说过的，让别人的角色带入到你的工作思考中。

在这个过程中你会发现，他们作为听众，会给出他们的疑惑和感触，这可以帮助你更高效地精益求精。所以我特别讨厌一句话：“在事情做成做大之前要闷不吭声。”你去看一看，哪个老板是闷不吭声坐在那的？他都是不停地跟人交流、路演，问别人“你觉得这个怎么看？我是这个想法，要不要加入？”他每天都在自己质问自己。我没听说过闭门造车、跟个鞋匠似的呆着就能成功的。

所以有些话你要辩证地去听。比如你的斗志——“我要开法拉利，我要赚多少钱”——这种连目标都不算，你也学过的，这只算自己的一种志向。你怕别人觉得你吹牛，那可以不说。但我觉得，有时候你真的要把别人带入角色里，这我就不多讲了，因为这不是今天课程的重点。

## 无意识时间的科学原理与价值 💎

总结一下，其实无意识时间的创造能力，是你刻意去创造时效率的三倍，而且这是有科学依据的。

在专注的工作、思考和学习的“意识时间”里，我们的大脑处于紧绷状态。这种状态特别适合做你擅长的、流程化的东西。此时大脑的活动就像一束**强光**：明确、直接、强烈、亮眼。但也许很多信息都被这束强光挡在了外面。

你应该能认可我。比如，现在你想做生意，不想创造那是扯淡，你必须创造。你看那些画家、毕业写生的大学生，有谁是天天关在家里创作的？就是这个道理。

而在“无意识时间”里，我们的大脑是放松的。那种刻板的、勒着你的感觉消失了，取而代之的是**横纵交错、发散的“弥散光”**，照亮了你整个大脑。如果你好好利用这个活跃的状态，并把其中的灵感提炼出来，我认为你的时间利用相对而言是非常高效的。

所以你千万别认为，那些有钱人、有能力的人天天过得跟苦逼似的。他们其实活得很精彩。就算活得不精彩的时候，他们也乐观：“我今天又进步了。只要我每天这样进步下去，我一定成。”而且还可能有意外收获。这真的不是成功学。

## 给初学者的实践指南 🧭

我给你举个例子。你学我的课程，就不能用无意识的时间去学。你需要找一个书桌，泡好一壶茶，打开我的文档，专注听写。因为**吸收知识是一个需要专注的过程**。

但是，涉及**创造力**的事情，你每天都要多问自己几句：我真的要这么干吗？为什么这么干？有没有更好的方式？更好的方式是什么？那就是创造力的事。

我希望你能够很好地运用好时间，做一个有心人。一旦稍微有一点感悟，就抓下来、记录下来。我相信有很多兄弟是我的情感学员，你看3.0课程中，是不是聊到要感知自己的“天地人”状态？我今天还遇到一个兄弟说，难怪我不知道什么叫“天地人”状态感受，能给我解释解释吗？我说，你不是不知道，是你从来没有感悟过，或者记录下来。

我说，“天”就是今天的天气、气候对你的感受；“地”就是你所在的地域，比如我从北京到深圳，深圳什么样的地域、湿度，我适应不适应，感觉了什么；“人”就是当地的人文是热情还是怎样。你没有感悟，你从来都认为，那些零星的、放松的时间是最宝贵的，是最能够有感悟和创造的。并且，就算你有感悟了，你也没记录下来。**这就是你一直没进步的原因。**

你一定要让你的“产量”大大提升，同时你的“质量”也要大大提升。老天给每个人灵光一闪的机会都差不多，有些人就让灵光飘过，有些人则拼命记录下那个灵感。你不放过无意识的时间，不放过那一闪而过的灵光，那个灵光就经常来眷顾你。那就是好运，就是好的想法，你就能晋升，就能当一把手、二把手。

## 核心总结与祝福 ✨

本节课中，我们一起学习了如何将“无意识时间”转化为创造力的源泉。

我经常说，时间在表面上看对人人平等，但其实并不平等。因为大部分人对此无能为力：一方面有消耗你的事情，一方面自己不会做选择，选择是极端的。还有就是，不知道时间其实是分成不同品质的。

所以要让你时间更有价值，就要：
1.  **选择有价值的事情去做**，同时从 **`现实价值`** 和 **`情绪价值`** 两个维度来判断，只做高价值的事情。
2.  涉及到与别人合作沟通时，一定要利用我教的降低沟通成本的方式：**`提前打预防针、定基调 -> 沟通 -> 迅速总结`**，让你沟通的每一句话都不白说。

最后我要告诉你，生活中那些看似零碎、无意义的时间，是最重要、大有可为的。不要放弃这些线索、信息和灵感。
