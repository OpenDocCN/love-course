# 男哥《成功人士遵循的因果论》：课程前言（一）你将获得什么？💡

在本节课中，我们将要学习这门课程的核心价值与你能获得的收获。课程创始人南哥将分享他从社会底层到成功创业者的实战经验，并揭示成功人士在从零到一的过程中，与普通人在认知、习惯和执行层面的关键区别。

大家好，我是南哥。首先做一个自我介绍。我是南哥恋爱教育的创始人兼董事长。大部分粉丝应该了解到，我从2016年起，在300多位导师中，已经是销量第一、服务第一、收入第一的导师。当时我所在的平台融资了千万美金，我是该平台的一哥。南哥恋爱教育从2016年开始，也一直是全国排名前三的、营收百万级的情感机构。同时，我帮助了超过千位的高分女生，被行业誉为“高分妹杀手”。

在这个理论派盛行的年代，实战派其实很少。在实战派当中，撩普通女生也是撩，撩没有见过世面的女生也是撩，但撩富二代、官二代，撩二线明星也是撩。所以“高分妹杀手”这个称谓，相当于行业对我的认可。

在我的这个课程当中，我将会讲述我是如何从零到一的。我也会去讲，一个社会底层的人，应该具备什么样的品质、信念和工作方法，才能够实现从零到一。目前为止，我的公司帮助了超过4万名学员。除了营收百万级的情感公司以外，我还是英国自营机构的高级交易员，拥有7年衍生品交易、现货交易和芝加哥商品期货交易的经验，当日最大操作上限为100万美金。你会看到我的交易截图，包括出金记录和每日盈利。

包括使用英国伦敦的交易方法，还有美国华尔街的一套期货交易方法。包括使用伦敦交易机构的Level2订单流交易、真正的做庄思维、主笔订单交易方法、做市商交易方法、能量分布、价格分布、市场深度洞察。

通过我的课程，我将会告诉你，一个真正的成功人士、一个真正高效的人，是如何高效利用时间的，是如何做到玩得开心、工作开心的。我经常说的理念是 **`work hard, play hard`**。我不会去讲那些你有钱之后应该怎么做。我也不会讲那些已经融资、有足够现金流的企业应该怎么做。我更愿意告诉你，一个没有融资、没有现金流的公司，如何从零到一。一个身处底层、没有外界帮助的人，如何利用现有环境，调整心态、工作方法、战略和目标，实现从零到一。

在我的课程当中，你将会获得有钱人与你在认知上的几大区别。你会明白为什么你感到迷茫，而他在同样从零到一时却不迷茫。同样都是人，都有人性的缺陷，他是如何利用人性的缺陷实现自我价值的？同样都是人，都有偏好和欲望。那些能赚大钱的人，普遍都有一个很好的身体素质。这些人在没钱的时候，是如何调整自己的作息和饮食习惯，让自己每天比别人都精力充沛，遇到任何事情都相对更乐观的？

同时，我也会讲述，我从零到赚取第一个100万，最需要的是吸收、学习、内化、实战各种各样的内容。那么，在有效的时间内，如何能够高效阅读、高效理解，并高效地将全世界的知识内容，制定出符合自己节奏的计划？计划应该如何实施和快速落地？在落地过程中，如果出现不一致，应该如何修改自己的计划？

同时，我将会讲到，那些有钱人在从零到一的阶段，为什么拥有很强的执行力。与此同时，那些真正优秀、成功的人，从一个普通人翻身，他们就没有拖延症吗？还是他们也一样有拖延症和懒惰？他们是怎么面对这些事情的？

我希望在这短短几日的课程中，从认知到实战到实操，能够让你明白：以你现有的条件和环境，你如何能够破圈晋升，如何能够从零到一。我经常思考华为为什么成功，但我的思考点可能跟别人不一样。这些年来，我在企业管理、股权激励、自主运营方面的学习，让我一直在探究。其实人们最关心的，并不是阿里怎么成功，华为有什么制度，华为为什么优秀。因为这些大公司能够不断融资，他们做的任何制度在当下都是对的。我们应该关注的是，在他们没钱、没有融资，只有几十人、十几人甚至几个人的时候，他们是怎么从零到一融到钱的？他们是怎么从零到一，建立起自己整个生态和产业链的？

人生也是一样。你有了第一桶金，后面翻十倍二十倍，相对来说就简单容易得多。相信我的这些课程，你一定会大有收获。并且它不只是一个认知课程，每节课都有可实操、可落地的方法。只要你是一个地球人，就一定会有收获。我要求我的助理对学习进行跟进和考核，同时为你提供工具包和相应的作业。希望这次学习，是你人生的一个新开始和一个新开端。
