## 纠正核心心态：关于“逆袭”的误解

首先，我想纠正一个观点。大家是否听说过一句话，叫“屌丝逆袭白富美”？前两年网上挺流行这句话。

既然大家都听说过，我现在问第二个问题：你们觉得“屌丝逆袭白富美”这种说法正确吗？

刚才认为不正确的各位，请说一下理由。有兄弟说，“屌丝改变了就可以逆袭”。没错，但“屌丝逆袭白富美”这句话是错的。为什么？因为这句话说错了对象，需要修正一下才正确。

修正什么？屌丝可以逆袭自己。逆袭自己之后，他有可能吸引到白富美。只要他努力，这是有可能的。因此，无论如何，逆袭或提升的都是自身，而不是通过努力让别人喜欢你。这就搞错了目标，明白吗？

一个人很多时候会出现一种情况：你越在意一个东西，越得不到它。对不对？我们想一下，女生是不是就是这样？你越在意她，越有需求感，越得不到她。

人生中很多事情都是这样。你越在意它，越得不到它。这是一个循环。

例如赚钱也是如此。你越是天天想着钱，掉进钱眼里，做什么事都在乎自己有没有收入，这样的人可能赚到钱吗？不可能。因为他太在乎得失，无法投入，对吗？因此他就没有回报。

这是很典型的例子。在感情上也是一样，你越是在乎结果，越是没有结果。因此，我们不应该把目标定为“我要找到一个怎样的女朋友”。目标应该定为“我提升自己”。

提升自己之后，自然而然可以吸引到很多更优质的女生。对吗？这是大家首先要摆正的心态。
