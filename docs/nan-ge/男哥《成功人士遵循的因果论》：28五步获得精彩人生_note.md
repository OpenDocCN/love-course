# 男哥《成功人士遵循的因果论》第28课：五步获得精彩人生 🧭

在本节课中，我们将学习一种名为“表格罗列法”的决策工具。这个方法旨在帮助你清晰地梳理生活中的各项事务，通过评估其现实价值与情绪价值，从而做出更明智的选择，规划出更精彩的人生路径。

## 第一步：理解决策的核心——价值二分法

上一节我们介绍了课程主题，本节中我们来看看决策的核心逻辑。人做选择时常常感到困难并事后后悔。要克服这一点，首先需要理解驱动人行动的两大核心价值：**现实价值**与**情绪价值**。

*   **现实价值**：指事情能带来的实际利益，例如提升能力、增加收入、改善健康、拓展人脉等。其核心公式可概括为：**现实价值 = 物质收益 + 能力成长**。
*   **情绪价值**：指事情能带来的情感满足，例如快乐、成就感、放松感、被需要感等。

人倾向于追求利益。明确每件事能“图什么”，能帮助你在感到痛苦时依然坚持。例如，工作是为了提升能力并赚更多钱；运动是为了获得好身体和好状态。

## 第二步：构建你的决策表格 📊

理解了价值分类后，我们需要一个工具来具象化分析。以下是创建决策表格的具体步骤。

1.  **列出事项**：在表格第一列，写下你近期（如一天、一周或一个月）计划或面临的所有事项。
2.  **评估现实价值**：在第二列，分析每件事对你的现实价值（升职、赚钱、提升认知等）。用“高”、“中”、“低”、“无”来标记。
3.  **评估情绪价值**：在第三列，分析每件事带来的情绪价值（是否让你开心、满足）。同样用“高”、“中”、“低”来标记。
4.  **写明理由**：在第四列，简要写下你给出上述价值判断的理由。
5.  **做出结论**：最后一列，根据后续的原则，初步判断“做”或“不做”。

## 第三步：运用原则进行筛选与排序 ⚖️

有了填好的表格，我们来看看如何根据原则对事项进行筛选和排序。以下是决策时需要遵循的四大核心原则。

*   **原则一：双高价值优先**。如果一件事同时具备高现实价值和高情绪价值，则必须做、认真做，并思考是否能持续做。这常是能将爱好、特长与事业结合的方向。
*   **原则二：单高价值可选**。如果一件事现实价值**或**情绪价值其中一项绝对的高，也可以选择去做。二者之间可以互相转化和弥补。
*   **原则三：现实价值优先**。当现实价值与情绪价值冲突时，情绪价值要让位于高现实价值的事项。让位不是完全不做情绪价值高的事，而是在排序上优先保障高现实价值的事项。
*   **原则四：剔除低价值事项**。不要将“鸡毛蒜皮”、可做可不做、价值不高的事情列入表格。人的精力有限，必须聚焦于达到一定标准以上的高价值事项，避免在次要问题上过度发散。对于低价值事项，原则就是“不做”。

## 第四步：实践案例解析 💡

为了让你更好地理解上述步骤和原则，我们来看几个具体的例子。

*   **案例A：做交易**
    *   **现实价值**：高（能带来收入）。
    *   **情绪价值**：中（本身也喜欢）。
    *   **理由**：符合兴趣且能创造财富。
    *   **结论**：做（符合原则一或二）。

*   **案例B：指导公司员工**
    *   **现实价值**：低（技能已掌握，主要帮助他人提升）。
    *   **情绪价值**：高（帮助他人，获得成就感）。
    *   **理由**：利于团队成长与公司运转。
    *   **结论**：可选做（符合原则二）。

*   **案例C：课程研发**
    *   **现实价值**：高（提升用户能力，为自己带来收入）。
    *   **情绪价值**：高（享受分享知识的过程）。
    *   **理由**：创造价值并获得满足感。
    *   **结论**：必须做（符合原则一）。

## 第五步：整合执行与持续优化 🔄

最后，让我们将这一切整合起来，形成可持续的行动模式。将表格中标记为“做”的事项，根据价值高低进行排序并执行。遵循“Work hard, play hard”的理念：高现实价值的事努力做，高情绪价值的事则用来奖励自己、恢复精力，从而更高效地投入下一轮工作。定期（如每周）更新此表格，根据实际情况调整事项与评估，使决策持续优化。

---

本节课中我们一起学习了“表格罗列法”这一决策工具。人生由选择构成，精彩的人生源于持续选择那些**既有用又快乐**的事。当你面临选择困境时，不妨列出表格，用**现实价值**与**情绪价值**两把尺子进行衡量，并遵循四大筛选原则。坚持这个方法，你将能更清晰地聚焦重点，减少内耗，从而活出更高效、更精彩的人生。
