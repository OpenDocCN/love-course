# 《爱商训练营》：12告别遇到喜欢的人就紧张_note

在本节课中，我们将要学习为什么遇到喜欢的人时会感到紧张，并探讨其背后的核心逻辑。我们将分析紧张情绪产生的根本原因，并提供一种根本性的解决方案，帮助你建立从容的心态。

## 紧张情绪的底层逻辑

上一节我们介绍了课程的主题，本节中我们来看看紧张情绪产生的根本原因。

遇到喜欢的人就紧张，其底层逻辑并非源于你多么爱对方，也不是因为你天生容易紧张。核心原因在于你“没得选”。

假设你的手机里同时有20位女性，她们颜值、身材、学历背景都与你当前喜欢的女生相同，并且都愿意与你聊天。在这种情况下，你还会担心自己说错话吗？你还会反复分析对方的想法吗？你还会刻意思考如何回复每一条信息吗？你还会感到无话可说吗？

核心问题并非紧张本身，而是因为你没有其他选择。“没得选”导致你害怕失去，因为一旦失去就意味着归零。这种心态会让你患得患失，追求完美表现，从而变得畏首畏尾。

## 解决方案：让自己“有的选”

理解了问题的根源，那么如何解决呢？根本方法是**让自己“有的选”**，增加你的选择权。

经验丰富的人为何不紧张、为何潇洒？并非因为他们强行自信或假装洒脱，而是因为他们深刻理解人性本质：没得选就会紧张。因此，他们避免让自己陷入焦虑的境地，选择主动扩充自己的“联系人列表”。因为他们拥有选择，不会在一棵树上吊死，而是拥有“一整片森林”。这与古代皇帝翻牌子的心态类似，与一位聊得不开心，便可以换另一位交流。

那些仅仅劝你保持平常心、调整心态的说法是错误的。只有当你真正“有的选”时，心态自然会变好。

以下是实现“有的选”的具体思路：

*   **量化目标**：例如，我的初级班内部课程会教授如何实现一天认识五到十个女生，一个月积累三百位符合你喜好的女性联系人（颜值高、身材好、气质佳）。拥有如此大量的潜在选择，将直接解决你的紧张心态。

## 用销售思维理解恋爱

为了更清晰地阐述这个逻辑，我们可以用一个类比来理解。

你的整个恋爱过程，本质上相当于将自己的“产品”销售出去，其流程与商业销售完全一致。

你经常声称：“我是一个好人，一个好男人，一个有责任心的人。”我们可以将你的这些特质称为“产品”与“产品品质”。然而，在没有任何“用户”（即潜在交往对象）知晓你的前提下，品质再好，能否达成“成交”（建立关系）呢？答案是不能。

我想说明的是，产品（自我提升）确实重要，我也致力于帮助你成为更好的自己。但若想“成交”，你必须先理解那些成功者的思路。

以下是成功者与普通人的关键思维差异：

*   **成功者思维**：他们优先解决“用户流量”问题，拥有大量可选择的对象，心态因而从容。
*   **普通人误区**：过于聚焦于“产品优化”（自我纠结），在没有“用户”基础时空谈平常心。

因此，首先要与成功者思维“同频”。在没有大量潜在对象（用户）的情况下，空谈保持平常心是无效的。谁能让你最快速地拥有大量意向选择，例如一个月内认识几百位女性，谁就能让你直接建立起良好的心态。

## 课程总结

本节课中我们一起学习了遇到喜欢的人时紧张的根本原因——“没得选”的心态，以及根本的解决方法——通过增加选择权来建立从容心态。我们还将恋爱过程类比为销售，指出在自我提升的同时，积累潜在选择对象是稳定心态、提升成功率的关键。记住，拥有选择，才能拥有淡定。
