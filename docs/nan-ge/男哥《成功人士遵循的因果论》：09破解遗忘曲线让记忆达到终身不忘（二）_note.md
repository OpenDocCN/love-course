# 男哥《成功人士遵循的因果论》：09破解遗忘曲线让记忆达到终身不忘（二）_note

在本节课中，我们将要学习第三种高效记忆方法——基于艾宾浩斯遗忘曲线的科学复习法。我们将了解记忆的周期性规律，并掌握一套具体的复习时间表，以将短期记忆有效转化为长期甚至终身记忆。

上一节我们介绍了情绪刺激法，本节中我们来看看如何利用科学规律来对抗遗忘。

## 理解遗忘的周期性

研究发现，人类对于信息的遗忘存在周期性规律。如果能在即将遗忘的关键时间点进行重复记忆，其效果远胜于毫无规律的死记硬背。

记忆分为**短期记忆**和**长期记忆**两种。短期记忆向长期记忆转化的过程，存在特定的时间窗口。

## 科学的复习周期表

以下是基于艾宾浩斯遗忘曲线理论总结出的八个关键复习周期。按照此周期进行复习，可以最大化记忆效率。

**核心周期公式：**
`复习时间点 = [5分钟， 30分钟， 12小时， 1天， 2天， 4天， 7天， 15天]`

*   **第一个周期：5分钟**
    *   学习新内容20-30分钟后，应暂停。间隔5分钟后，立即回顾一遍所学内容的核心要点与笔记。

*   **第二个周期：30分钟**
    *   在5分钟复习完成后，间隔30分钟。在此期间可以从事其他活动，故意让记忆模糊，然后在30分钟时间点进行第二次复习。

*   **第三个周期：12小时**
    *   完成30分钟复习后，间隔约12小时（例如，晚上学习，次日早晨复习）进行第三次复习。要认识到遗忘是正常过程。

*   **第四个周期：1天**
    *   从初次学习算起，间隔一天后进行第四次复习。

*   **第五个周期：2天**
    *   间隔两天后进行第五次复习。可以将新旧内容的学习与复习日程滚动安排，例如学习新内容的一天，正好是复习旧内容的日子。

*   **第六个周期：4天**
    *   间隔四天后进行第六次复习。

*   **第七个周期：7天**
    *   间隔一周后进行第七次复习。

*   **第八个周期：15天**
    *   间隔十五天后进行第八次复习，此时记忆已趋向稳固。

## 方法融合与核心理念

掌握以上八个周期并应用得当，可以极大提高学习效率，避免在短时间内进行低效、疯狂的重复。

我希望大家能将本课介绍的三种方法融会贯通，结合使用：
1.  **强迫运用法**：侧重于从记忆事项的“意图”出发，主动创造使用场景。
2.  **情绪刺激法**：侧重于利用情绪作为记忆的“媒介”，将知识点与生活中的喜怒哀乐事件关联。
3.  **科学复习法（本课重点）**：建立在对人性遗忘规律了解的基础上，进行科学、有计划的重复。

复习、重复与复盘，远比不停地记录新内容更为重要。高效的复盘并非在短时间内高频进行，而是遵循科学的周期。按照上述周期安排复习，每日效率会很高，最终能达到近乎终身不忘的效果。
