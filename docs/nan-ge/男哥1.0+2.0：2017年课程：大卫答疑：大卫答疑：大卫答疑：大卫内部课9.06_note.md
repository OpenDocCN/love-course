# 课程名称：展示面拍摄与修图核心技巧 🎬
## 课程编号：大卫内部课 9.06

在本节课中，我们将要学习如何拍摄和修饰高质量的展示面照片。我们将深入探讨社交认证的概念、拍摄时的环境与人物选择、常见的错误示范，以及实用的修图技巧。通过学习，你将能够打造出真实、自然且具有吸引力的个人展示面。

### 社交认证与展示面核心 🧑‍🤝‍🧑

上一节我们介绍了课程概述，本节中我们来看看社交认证的核心概念。

工作是一种社交认证。你不仅可以拍摄一个人工作的场景，也可以拍摄你和别人一起工作的场景。例如，拍摄团队协作的画面就是一种工作展示面，它同时也是一种社交认证。

与名人合影是高价值社交认证的一种形式。这当然算，因为这也属于工作或社交活动的一部分。接下来，我们看看生活类展示面是什么样的。生活展示面通常展现你与他人一起玩乐的场景，例如一起吃烧烤或一起工作交流。

因此，社交认证类的工作和生活展示面应该这样拍摄，它能营造一种现代感。同时，你需要注意展示面中出现的人物对象。

我的社交认证对象都是年轻人。展示面中的人物应尽量选择年轻一些的。现在我们来看一位学员的展示面，你能看出问题吗？

他展示面中的人物年龄偏大。如果你的社交圈是年龄偏大的人，那么你的展示面吸引的也可能是年龄偏大的群体。所以，这种社交认证效果可能不太好。但它可能会吸引40多岁的女性。背景颜色是次要的，与具体人物如王健林关系不大。

### 展示面拍摄两大禁忌 🚫

上一节我们讨论了社交认证的对象选择，本节中我们来看看拍摄展示面时必须避免的两个严重错误。

第一个禁忌是**抠图**。有学员曾将我的照片中的人物抠下来，换成自己的脸。这是一个忌讳。不要把别人的展示面照片抠下来换上自己的脸。一定不要抠图，这种行为非常低级。抠出来的图片会显得很假。

第二个禁忌是**过度修图**。有些学员在使用修图软件时，将磨皮等功能参数调到极致，导致皮肤看起来像蜡像一样不真实。修图是为了改善瑕疵，而不是创造一张假脸。过度修图会失去真实感，见面时容易“见光死”。

---

### 环境与人物姿态的关键作用 🌆

上一节我们强调了不要抠图和过度修图，本节中我们来看看构成一张优秀展示面的其他关键要素：环境与人物姿态。

拍摄展示面时，第一要注意你的动作和姿态。第二要注意周围的环境。我们以我的一张照片为例。这张照片是在一个类似创意园区或别墅区的环境拍摄的，旁边停着车，整体环境感觉不错。

有学员模仿了这张照片的动作和角度，穿着也类似。但他唯一没注意到的是环境。他的拍摄背景看起来像村头，导致最终效果不佳。所以，展示面不仅仅是模仿动作，环境同样至关重要。即使在胡同里拍摄，也需要注意环境氛围。

另外，关于修图，我上次课程教过一些方法，但有人使用得有点过度。例如，有位学员把脸部磨皮磨得太过，看起来像塑料脸。修图时，应该保留皮肤的一些自然纹理，避免变成假人。

### 专业修图软件与技巧详解 🛠️

上一节我们指出了修图过度的危害，本节中我们来详细学习如何使用专业软件进行自然修图。

我将演示如何使用 **FaceTune 2** 这款软件进行修图。我们以一张有痘印和皮肤不平整的照片为例。

以下是修图的核心步骤：

1.  **使用“修复”功能**：这个功能用于去除痘痘或明显的斑点。你可以放大图片，在痘痘上涂抹即可去除。
2.  **使用“更薄滑”（磨皮）功能**：这是关键步骤。不要将滑杆直接推到100%。过度磨皮会让皮肤失去纹理，像蜡像。我通常只推到40%-50%，这样既能改善肤质，又能保留真实感。
3.  **对比效果**：修图后，与原图对比，你会发现皮肤变好了，但人看起来依然真实自然。

**核心公式**：`适度磨皮 (40-50%) = 改善肤质 + 保持真实感`

过度修图就像给自己戴上了一张完美的面具，虽然照片好看，但失去了与人真实连接的根基。

### 身形修饰与增高技巧 📏

上一节我们学习了面部修图，本节中我们来看看如何修饰身形和身高，这对于展示面同样重要。

以前大家可能常用美图秀秀的“瘦脸瘦身”功能。这个功能可以用于手动推挤脸部或身体轮廓来显瘦。但在有直线条（如墙壁接缝）的背景前使用要谨慎，容易穿帮。

今天我推荐一个更专业的软件：**Spring**。它专门用于自然瘦身和增高。

以下是使用Spring的核心方法：

1.  **瘦身功能**：软件界面通常有两条竖线和一个圆圈。将圆圈放在你不想改变的部位（如男性的肩膀），将两条竖线分别放在身体两侧。然后调整强度，一般10%-20%就能达到自然瘦身效果，切忌拉得过高。
2.  **增高功能**：软件内有几条横线。按照提示，将最下面的线放在脚踝处，中间的线放在臀部（胯骨），最上面的线放在肩膀处。然后调整强度，通常30%左右就能自然拉长腿部比例，让人显得更高。

**核心技巧**：使用任何修图软件，**“适度”是关键**。调整幅度太小没效果，太大则假。多尝试，找到最自然的效果。

### 优质展示面八大推荐场景 📸

上一节我们掌握了修图技巧，本节中我们来看看可以去哪些地方拍摄高质量的展示面。以下是八个推荐场景，它们构成了一个合格展示面的基础。

以下是八个推荐的高价值展示面拍摄地点：

1.  **高级酒店下午茶/咖啡厅**：人均消费不一定很高（几十到百元），但环境通常优雅，适合拍摄。
2.  **高逼格西餐厅或自助餐**：用餐环境也是展示生活品味的好场景。
3.  **健身房**：展示健康、自律的生活状态。
4.  **旅游景点**：选择适合年轻人的短途或长途旅行地点，避免老年旅行团常去的地方。
5.  **工作场景**：可以是真实的办公，也可以是精心摆拍的“工作状态”，展现事业心。
6.  **与宠物互动**：这是杀伤力很大的场景。**关键点**：不要刻意摆拍，最好让他人抓拍你与宠物自然互动的瞬间。
7.  **展示厨艺**：可以是在家或户外，注意环境、穿着和表情的自然协调。
8.  **其他高价值活动**：如高尔夫、卡丁车等，根据自身情况选择。

**核心思路**：这八个场景是基础模板。拍摄时，务必注意**环境、自身穿着打扮、动作姿态和表情**的和谐统一。要么不做，要做就尽量做到100分。

### 常见展示面问题即时点评 🎯

上一节我们列出了优质场景，本节中我们结合学员实例，即时点评一些常见的展示面问题，帮助大家深化理解。

以下是一些学员展示面照片的集中点评与修正建议：

*   **问题一：脸部比例过大**：很多自拍照喜欢把脸怼满屏幕，这暴露缺点且显得死板。**解决方法**：手臂伸长一些，让脸在画面中的比例减小，采用斜上45度角拍摄。
*   **问题二：环境与穿着不搭**：例如，穿着昂贵的潮流服装（如Gucci、SLP）去爬山或在简陋车库拍照，会显得突兀、装逼。**解决方法**：在什么环境，穿符合该环境的衣服。想突出穿着，就选择与之匹配的时尚街区或艺术园区作为背景。
*   **问题三：姿势僵硬刻意**：摆拍痕迹过重，表情不自然。**解决方法**：多尝试抓拍，或让拍摄者引导你进行一些自然的活动（如走路、回头、与物品互动），然后进行连拍。
*   **问题四：构图死板**：人物总是放在画面正中央。**解决方法**：学习一些基础构图法，如三分法、对角线构图，让人物偏离中心，画面会更生动。
*   **问题五：背景杂乱或低价值**：在杂乱卧室、普通教室、昏暗KTV拍照。**解决方法**：有意识地选择我们推荐的八大场景，或至少背景干净、整洁、光线良好的地方。
*   **问题六：滥用修图特效**：过度使用虚化、夸张滤镜，导致照片失真。**解决方法**：滤镜用以微调色调和氛围，切忌改变照片本质。修图以“自然改善”为第一准则。

### 聊天思维与展示面的结合 💬

上一节我们点评了展示面，本节中我们简要探讨一下展示面如何影响聊天，以及聊天中的核心思维。

展示面是你在社交软件上的“第一印象”。优秀的展示面能为你打开聊天窗口，提供高价值展示。当女生在聊天中给你“负反馈”或“框架测试”时（例如嫌你回复慢、调侃你），正确的思维不是解释或讨好。

**核心思维**：**保持高姿态，将自己视为奖品，引导对方投资。**

**错误示范**：
女：“你回复好慢哦。”
男：“不好意思，刚才在忙。”（解释，低姿态）

**正确示范**：
女：“你回复好慢哦。”
男：“创业小青年，工作太忙了。”（转化话题，展示高价值）
或
女：“原諒你了。”
男：“嗯？我做什么了需要你原谅？”（反弹框架，保持姿态）

你的展示面塑造了你的价值，聊天中的每一句话都应维护和强化这种价值，而不是削弱它。
