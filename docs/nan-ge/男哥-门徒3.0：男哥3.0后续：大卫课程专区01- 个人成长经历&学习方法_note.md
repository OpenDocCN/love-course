# 男哥-门徒3.0：导师专区01 - 大卫的个人成长与学习方法 📚

在本节课中，我们将跟随大卫老师的分享，了解他从一个情感领域的“萌新”成长为导师的个人经历，并学习他高效、扎实的学习方法。他的故事将告诉我们，正确的学习路径和强大的执行力是取得进步的关键。

## 课程背景与初衷 🎯

上一节我们介绍了导师专区的设立原因，本节中我们来看看大卫老师的课程。

设立导师专区是为了让学习更有逻辑和体系。每位导师都有自己的风格和调性。将所有导师的内容都放在3.0课程中，可能会让学员感到杂乱。因此，我们让每位导师将自己的感悟和想法总结成课程，以专区的形式进行讲授。

接下来，我将为大家介绍大卫老师。

## 个人成长起点：从“理工男”到情感学员 🔄

大卫老师最初是一名理工科专业的学生，逻辑思维能力很强。他从大学开始学习工程学。他不太擅长与女孩交流，认为喜欢女生就是要追求、要对她们好。这与很多学员报课前的想法一样。

大卫老师的基础甚至比一些学员还要差。他当时就是这样的形象。后来他有些发胖。一张自拍显示了他毕业后的样子。他毕业时学有所成，学习成绩比较好。他后来参与的科研项目成果也很好，属于全球比较顶尖的科研项目。目前中国的微生物工程界，还没有研究到那个顶尖的技术阶段。学业方面他不再展开说明。

大卫老师当时为什么来北京呢？因为他喜欢一个在北京上学的女生。为了她，大卫老师放弃了一份去欧洲发展的前程，或者说一个攻读博士学位的科研机会。他选择来北京找她。

因为他和那个女生聊天时，对方态度很暧昧。作为一个“萌新”理工男，大卫老师以为对方那样说话、甚至称呼“亲爱的”，就是喜欢他。他觉得自己应该来北京。因为他生活中不认识那么好看的女生，而他当时特别喜欢她。

现在在他看来，那个女生也就那么回事。但是学过物理的人都知道，选择的参考系不同，看到的东西就不同。她当时的状态就不一样。大卫老师当时所处的参考系，就是他当时的层次、人脉和朋友圈子。在他当时的圈子里，那个女生简直就是仙女。但事实上她的分数真的很高吗？可能有一些，但现在看来也不算是特别高分的。

## 痛苦经历：付出与失衡的关系 💔

大卫老师当时为那个女生花了不少钱。他刚来北京时，第一份工作实习期一个月包吃住，只有500块钱。因为他留过学，英语比较好，周末就去58同城上找一些英语家教之类的零工赚钱。他攒钱给那个女生买了一套奢侈化妆品CPB。

但那个女生一直吊着他。大卫老师当时比较“萌新”，觉得这个女生应该是喜欢他的。但有时对方会突然生气，几天不理他。那时他就特别焦虑，心里想是不是自己对她还不够好。

大卫老师还送过一个限量版的雕塑给那个女生，因为她特别喜欢相关的电影。送给她之后，对方只是说了谢谢，后来也没有后续。

再到后来，大卫老师发现那个女生好像喜欢另一个男生。那个男生家里没钱，年龄比她小，也在北京上学。大卫老师想不通，因为那个男生长得也不是很帅。但大卫老师一直对她很好。

大卫老师甚至用自己办的信用卡帮那个女生租车，因为她上学没有收入办不了卡。租完车后，对方说去接几个朋友。结果第一个接上的就是那个男生。最后演变成大卫老师开车送他们俩去一个酒店。后来那个女生又找了她自己的朋友，然后让大卫老师先回去，把车留下。

大卫老师后来才知道，她和那个男生是情侣关系，而且认识时间比大卫老师晚。他不明白，为什么自己和她聊了那么长时间，那个男生才认识她几天，他们就能在一起。

后来大卫老师从一个共同朋友那里知道，那个女生对所有男生都采用“吊着”的方式，对别人好，又突然推开，再给点甜头。她用这种方式“养”了一堆男生，今天有人帮她租车，明天有人帮她做事，后天又有人送她礼物。

大卫老师当时感到很崩溃。这件事发生在2015年。这件事过去大概四五个月之后，一次阴差阳错的机会，大卫老师知道了“某男孩”这个平台。他上去看看男生怎么追女生，上面有各种干货和导师直播。

他无意间点开一篇帖子，说男生不能对女生太好，对她好反而会让她远离你。大卫老师越看越觉得那就是自己的写照。帖子最后有一个绿色的条幅，点开是加助教微信。他就加了，于是联系到了男哥。

这一下打开了他新世界的大门。他才知道原来谈恋爱是可以学习的。他想要是早学了，就不至于犯那么多错了。因为在男哥的帖子里，他看到了自己犯的一些原则性错误，每一步都非常吻合。他产生了强烈的共鸣。

## 学习方法论：从笨功夫到开悟 🧠

于是大卫老师决定报课学习。作为一个新学员，进群后他很紧张。他问的第一个问题，直接被一个老学员说“傻逼，一边看着去”。他没办法，只能先看着，天天看男哥分析案例，进行学习。

当时男哥处于1.0阶段，课程在某男孩平台上。大卫老师跟着男哥开始学。他记得第一节课听的是“五大聊天利器”：取解、推拉、画面感植入、表演性人格和冷读。这五种互动方式让他非常震惊，因为他从来不知道跟女孩还能这么聊天。学完后他马上就去尝试。

当时他还不认识探探，只能用“默默”去跟周围的人打招呼，一个个聊。他开始拼命学习。

当时的学习条件比较艰苦。男哥的课程在那个平台自己的APP上，不方便，每次听课也没有画面，只能听语音。他只能去记语音里的内容。看案例图片也只能看一遍，回放时没有图片，只有语音。所以他学得特别拼，动力来自于之前那段痛苦经历。

以下是大卫老师当时的学习方法：

*   **详细记录**：他展示了2016年7月的笔记，那是他报课一个多月后记的。每节课都有笔记。他把男哥所有的案例都记下来。当时男哥每周有两节内部课，还有一个黄金档的公开课。只要男哥上课，他无论如何都会去听。只有一次因为约会提前请假，其他课全勤。
*   **抄写案例**：因为回放只有录音没有图片，案例只能看一遍。所以在听直播课时，他直接把对话案例抄下来，包括男哥和女生的每一句对话。他还在后面加入自己听课时的感悟和引导。
*   **背诵记忆**：他用的是最笨但似乎最有效的方法：把聊天内容全背下来。他甚至能记住几年前的具体案例细节，例如一个关于“百万收入模特”的案例，其中男哥用了“从小想做一个超人英雄…后来只想要一只猫、一只狗和一个爱的人”等高价值展示的话术，以及后续的互动。他熟悉到几年后还能复述。

所以现在回过头看，很多学员课也不听，说了也不来，这样是学不到东西的。很多人羡慕大卫老师，他说只要先做到他当年学习劲头的50%，就可以很厉害了。

开始学习后，大卫老师的心态首先有了质的飞跃。他还没有实战约会，但因为听男哥的课，开始做展示面，玩探探（当时积木还没出），认识了一些之前在工程领域接触不到的、分数还可以的女生。因为他当时工作借调到了西直门，附近有北京外国语学院、北京师范大学等学校。

心态产生飞跃是因为他找到了解决问题的方法。他之前不知道谈恋爱可以学习，现在知道了，并且找到了对的导师（男哥）。接下来要做的就是刻苦学习、付出时间积累。虽然他觉得自己比别人笨一点，但他像阿甘一样，不在乎别人骂他傻逼，只管学。他当时完全没想到今后会做导师，只是想解决个人感情问题。因为受过之前的刺激，他决心真正付出行动努力。

## 关键转折：实践、反思与能量反转 ⚡

大卫老师曾做过一件标志性的事。他之前送那个女生的限量版雕塑，自己很喜欢，买不着了。他纠结要不要要回来，就去问男哥。男哥说：“那你要他干啥？”大卫老师解释后，男哥建议他直接去要。

他就按男哥说的做了，发消息问那个女生：“我是不是还有个猫男爵雕塑在你那？你还我吧。”这是大卫老师第一次公开这个聊天截图。

从这段聊天能看出，那个女生一开始还是端着，用“绿茶”的方式说话（如“工作哟”），给一点窗口。但这时的大卫老师不会吃亏上当了，直接了当进入主题。对方还是有点端着，问“还需要还你啥一次说清”。但此时的大卫老师已经不是过去的他了。过去他会焦虑，担心对方看不起自己。现在，从他说话的能量和态度就能看出地位发生了反转。虽然当时他还不像现在这么厉害，但进步已经很大。他说雕塑是限量版买不着了，所以要回来，其他的就当是做慈善。“做慈善”的意思就是把她当要饭的。这时已经开始能量反转。

经历这件事后，大卫老师更有信心了。因为他现在已经可以看不起那个曾经让他焦虑的人了。他能看出对方说话的“绿茶”德性，知道对方在给他窗口、想让他往上贴。他至少能够识别出这样的人，只不过实战经验还差一些。于是他继续努力学习。

## 学习进阶：从1.0技术到2.0思维 🚀

大卫老师是2016年6月4日报的名。学到7月份时，他已经可以有一些实战聊天，展示面也在跟着做。

他展示了当时做的第一个探探展示面，是当年流行的九宫格。他的执行力非常强大，说拍就去拍，说找就去找。现在回头看，当时很多照片都是游客照，拿吉他的那张特别装逼、摆拍，拿宠物的那张发型也不好看。但他强调：先做再说，做了之后可以改进。现在很多学员总纠结发型、胖瘦等问题，而不敢行动，这让他很生气。

他经常给男哥发自己的聊天案例让男哥点评。他提问有一个特点：几乎从来不直接问“这句话怎么回”。因为他明白，如果男哥帮他回了这一句，下一句他还是不会。与其学这一句，不如学习背后的思维。所以他提的问题始终关注更高维度的东西，关注方法。例如他当时问：“默默上女生打招呼回复，聊了几句，她不顺着我往下接，只会表情怎么破？”他先给出自己的思路，觉得是个死循环。现在他明白了，问题的核心在于当时的展示面不够优质，吸引不了对方，对方只是在敷衍。但即使当时不明白，他也不会去问具体怎么回，因为他想学到一种方法或思维，能让自己知道下一句该回什么。

他这样学到8月份时，感到非常痛苦，因为他“中毒”了。跟女生交流时，他脑子里浮现的全是男哥的话术，因为他背了太多案例。他跳不出这个思维，永远在男哥的回复“数据库”里搜索数据去回复。这样他没有自己的调性和特点，女生会觉得他奇怪。

大概在8月底9月初，他“中毒”情况非常严重，觉得必须找男哥聊聊。刚好当时平台有一个活动，优秀学员可以和导师进行40分钟一对一通话。他果断用了这次机会。

男哥用这40分钟给他讲了2.0的东西：**七步法**。这为他打开了新世界的大门。学会七步法后，因为他之前的基本功（各种话术）非常扎实，他一下子就上了一个台阶。七步法具体是：
1.  激发对方回复欲望
2.  基本信息交换
3.  互动
4.  升高
5.  揭露对方，塑造自己
6.  预期
7.  邀约

学完这个之后，他就开始了更多的实战。

## 灵活运用与风格形成：找到自己的路 🛣️

2016年年底，大卫老师还在按七步法老实聊天。但他发现一个问题：不是每个女生用七步法都能聊成功，成功率还是比较低。他于是开始思考，是不是存在更高维度的方法。他开始自己尝试，并在过程中不断与男哥交流。

他发现不是每个女生都适合用完整的七步。有的女生反馈好，可以适当直接邀约；有的反馈不好，可能需要放几天再聊。

到2017年春节前，他聊死过200多个女生。因为他比较挑，一般只匹配高分女生。2017年春节时发生了一件事，他在Merry U上认识一个女生，一开始对方不愿加他微信，觉得他不行。但他通过有事没事、保持节奏频率一致地聊天，最后把对方“聊活”了。对方说之前误解了他。这是他第一次“聊活”一个女生，非常开心。

从那之后，他对七步法的运用更加灵活，社交直觉也更强了。但有些高维度的东西，需要自己不断琢磨和修正。而且每个人的调性不一样。

大卫老师最近还在和男哥交流风格问题。他对妹子的态度有时会特别强硬（如说“滚”），而男哥可能会选择卖个萌。他们分析后发现：男哥平时给人的感觉是“屌儿郎当”、有时比较严厉；而大卫老师平时给人的感觉是特别老实、好人的角色，初期交流也很有礼貌（闷骚）。所以他表现出的强硬反差反而让女生觉得有吸引力。而男哥如果那么说，别人就该骂他了。这说明每个人风格不同，需要找到属于自己的路和调性。

2017年6月1日，大卫老师来到男哥的公司，成为一名实战导师。做导师后提升更快，因为不仅要解决自己的问题，还要帮助别人解决问题，看问题的维度也更高了。

到最后的3.0阶段，他听到“开悟片”时感觉非常对，因为很多经历他自己体会过。3.0把最高维度的东西提前讲给了学员，课程也比以前更体系、更干货。他当年可没人讲过这些神秘课程，都是自己体会的。

## 总结与寄语：期待3.0的大神诞生 🌟

本节课中我们一起学习了大卫老师的成长经历与学习方法。

大卫老师发现，学员中能像他当年那样学习的人很少，佳琦老师算一个。他是1.0学员中的代表（千里挑一），佳琦老师是2.0学员中的代表。他非常期待3.0学员中也能出现这样的人物。但这取决于听课的各位。

他送给大家一张图，意思是看问题的角度不同，看到的东西就不同。关于3.0的学习方法，他不推荐大家都照搬他的模式，因为可能会“中毒”、学得不对。真正的学习方法应该是每个人自己去总结。他的经历可以起到抛砖引玉的作用。

他是逻辑性特别强、思维严密的人，而佳琦老师是思维跳跃、天马行空的人，他们的学习方法就有出入。大家可以去听佳琦老师专区的分享。听完后，应该做的是找到适合自己的方向，找到自己的东西。

按照金字塔模型，顶尖的人永远是少数。他特别期待看到3.0学员中能出现一个大神。第一步是找到自己的学习方法。导师已经把最核心的东西都教给大家了。
