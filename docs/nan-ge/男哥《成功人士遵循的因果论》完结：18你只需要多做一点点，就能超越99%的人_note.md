# 男哥《成功人士遵循的因果论》完结：18你只需要多做一点点，就能超越99%的人_note

在本节课中，我们将探讨如何通过巧妙利用外部环境与他人的力量，来有效克服拖延并实现高效自律。核心在于，将个人目标与他人进行“捆绑”，借助外部监督和契约关系来驱动自己前进。

上一节我们介绍了塑造有利物理环境的重要性，本节中我们来看看如何将“人”这一要素纳入你的自律系统。

## 利用“人力环境”克服拖延

这里所说的环境，并非指物理空间，而是指你周围的人际网络。解决拖延和实现高效自律的关键策略是：**在学习或工作过程中，主动动用周围的人力资源来帮助你保持自律**。

具体做法是：将与你任务或项目相关的人，全部纳入你的工作计划中。通过赋予他们角色，将你的进度与他们进行捆绑，让他们来监督和帮助你。其背后的逻辑是，仅靠个人意志力难以时刻自我提醒，因此需要主动引入外部监督机制。

## 两种认知模式的对比

关于目标公开与否，存在两种截然不同的认知：

1.  **传统认知**：主张低调行事，事情成功前不宜声张。
2.  **现代实战认知**：**“我成功，是因为我当时吹了个牛逼”**。即通过公开承诺来倒逼自己行动。

哪一种更符合现代成功标准？我认为是后者。例如，我计划下周一与某人讨论项目思路。为了让讨论有基础，我就必须在周一前完成相应的准备。即使我自己想拖延，但因为已将他人纳入计划，我需要考虑对他人时间的尊重和可能面对的批评。这种“面子”和契约压力，会有效驱使我赶紧行动。

## 实战方法：将他人纳入你的计划

以下是将此策略付诸实践的具体方法：

*   **学习伙伴**：邀请你的伴侣或朋友一起学习这门课程，并互相交流。
*   **同事监督**：如果你想提升领导力，可以邀请一位同事作为你的“复盘伙伴”。
*   **向上管理**：向你的领导承诺：“下周三我要完成演讲/任务，我想在周二提前完成并向您汇报。”这样就将领导纳入了你的进度监督体系。

当你把别人拉进你的计划后，你将面临他人的期待和可能的失望。这种社会压力会成为你立即行动的巨大动力。

## 核心总结与反思

本节课我们一起学习了如何通过**建立外部监督契约**来克服人性中的惰性。你需要明白，高度自律是反人性的高级品质，大部分人都难以仅靠自觉做到。因此，善用“群体效应”和“契约压力”是实战且有效的方法。

同时，这一点也引出一个反向思考：你的不自律，是否也源于周围环境给了他人类似的“拖后腿”的角色？有些朋友可能会无意间消耗你的能量，增加你的拖延。因此，主动选择并塑造你的“人力环境”，与能推动你前进的人结盟，至关重要。

**核心公式**：
`高效行动 = 个人目标 + 外部监督角色 + 社会契约压力`

记住，成功往往就藏在那“多做的一点点”里——这一点点，可能就是主动为自己构建一个不让自己轻易退缩的环境。
