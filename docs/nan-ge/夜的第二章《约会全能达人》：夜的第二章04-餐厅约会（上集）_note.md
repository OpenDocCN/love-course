# 夜的第二章《约会全能达人》课程04：餐厅约会（上集）🍽️

在本节课中，我们将要学习餐厅约会的前半部分核心技巧。我们将探讨如何选择合适的餐厅、点餐时的注意事项，以及如何通过对话营造舒适的约会氛围。这些知识将帮助你避免常见错误，提升约会体验。

上一节我们介绍了咖啡厅约会的完整方案，本节中我们来看看餐厅约会的场景。餐厅是许多恋爱故事开始的地方，它与咖啡厅有相似之处，也是一个增加彼此联系的重要场合。然而，餐厅约会的时间周期和互动节奏更难把握。学习完本节内容，你将收获颇多。

## 1. 选择合适的餐厅场景

约会如果选择吃饭，要选择合适场景。并不是所有餐厅都适合约会。

以下是选择餐厅的核心原则：
*   尽量不选择油腻、油烟重或口味很重的地方。
*   宽大的四人台优于狭小的两人台。

其实餐厅场合的选择，不用刻意思考是否高端大气、是否要吃牛排。约会的核心是为了见人，而不是为了吃饭。试想一下，如果对方因为场地不够高端就不来，那建议你没有必要进行这次约会。这符合“南哥内核”和“山甲林男神框架”：对方应该是为了你这个人而来，而不是为了一顿饭。

## 2. 点餐时的带领与关怀

点餐过程的核心在于带领女生。女生通常喜欢被带领的感觉，不会喜欢一个纠结犹豫的男人。

以下是点餐时的关键步骤：
1.  **提前询问忌口**：例如，“有什么忌口吗？” 这样既能照顾对方，又能显得强势、有男人味。
2.  **主动承担点餐责任**：可以说“这次我来点”，展现决断力。

## 3. 饮品与甜点的巧妙选择

在点饮品时，需要注意两个细节。

以下是需要避免的错误和正确的做法：
*   **错误做法**：很多男生在约会中试图灌女生喝酒，认为这样自己会有机可乘。但这会让对方感到不适，第一次吃饭就劝酒容易暴露不良意图。
*   **正确做法**：不要劝对方喝酒。可以询问“要不要喝点饮料啊，吃點小甜品啊？”，这样反而能让对方产生好感。很多女生喜欢颜值高的甜点，她们可能喜欢拍照，也会因此很开心。

## 4. 创造话题与营造氛围

对话时不要一直谈论高价值、三观等正经话题，有时也需要一些幽默感。

以下是一个创造话题的实例：
你可以说：“你知道我为什么带你来这家餐厅吗？因为这是一家网红餐厅，有很多明星关注。我前两天看到你的朋友圈，你不是蛮喜欢吴亦凡吗？吴亦凡也来过这家店。” 随后可以迅速降低预期，开个玩笑：“骗你的啦，讨厌。” 接着可以回归真实：“其实我之前在这边工作，经常光顾这家店，跟厨师也挺熟。”

这种方法先抬高对方预期，再迅速降低，创造一种反差感，能为你们制造更多话题和轻松气氛。同时，通过提及和厨师熟悉，也在侧面展示自己朋友多、善于社交的高价值。
