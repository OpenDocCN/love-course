# 男哥《成功人士遵循的因果论》：19高效推进项目，快速提升领导力的万能钥匙__note

在本节课中，我们将学习一种高效推进项目、快速提升领导力的实战方法。其核心在于巧妙利用外部环境和人际关系，将个人任务转化为受监督的公共承诺，从而克服拖延，实现高效自律。

## 克服拖延的实战案例

上一节我们探讨了因果论的基本概念，本节中我们来看看如何将其应用于具体场景。当你面对一项需要细致整理但感到困难、不愿启动的任务时，可以采取以下方法。

首先，主动与你的领导或同事约定一个具体的讨论时间。你甚至可以为此付出一些成本，例如请客吃饭或承诺分享部分业绩。通过付出，你邀请他人加入并确定了时限，比如一小时或三小时后进行交流。

此时，由于他人已经准备好并期待与你交流，你今晚便别无选择，必须为了明天的会议而整理好所有材料。这种方法迫使你立即行动。

## 提升领导力的行动策略

想要练习管理能力和领导力却迟迟没有行动，不知从何学起？解决方法同样直接。

每当有项目机会出现时，例如岗位竞选或新项目启动，立即报名并争取成为负责人。当你成为“第一个吃螃蟹的人”，并且项目下周就必须启动时，你本周就只能全力以赴。

以下是接下来你必须紧急推进的事项：
*   赶制项目计划。
*   落实各个关键节点。
*   完成整体预算核算。
*   搜集必要资料。
*   准备演示或录制材料。
*   进行反复练习。

否则，你将成为全公司最丢脸的人。许多条件或许不如你但最终成功的人，往往无意中运用了这一点：他们习惯利用周围环境和人来介入并督促自己。

## 方法的核心：洞察与利用人性

你需要深刻理解人性。人的天性是懒惰的，这并非意味着某人高尚而另一人低级，懒惰是普遍常态。自律本质上是反人性的。

除了懒惰，人还“要面子”、重视声誉。你可以利用人们“要脸”的心理，来克服自身不自律的弱点，这非常具有实战价值。

我总结一下，这个方法的本质是 **将截止日期提前化**。

例如，一个任务若没有明确截止日期，就容易被无限搁置。当你为任务附加了“人设”（对他人的承诺），截止日期就被提前了。同时，**将未能及时完成的不良后果即刻化、现实化**，让你因不敢面对而无法拖延，从而实现极致的自律。

再举一个生活中的例子：你觉得健身太麻烦，即使报了班也不去。单纯请私教督促可能无效。但如果你去追求一位欣赏好身材的异性，并向对方吹嘘自己健身很棒，并约定下周一起运动，那么你看你这周会不会拼命训练。

## 关键认知：工作是社会化的行为

我认为，凡事要有认知，更要通过实战来总结，这才是最有效的。工作是一种社会化的行为，而非个体活动。

因此，你的同事、合伙人、上司、客户都能成为推动你的外部力量。你要学会调动这些力量来督促自己。在这种情境下，稍有拖延就会影响整个团队进度，成为拖后腿的人。

相比之下，许多人喜欢闭门造车。这使得任务变成了纯粹的私人事务，拖延的后果只由自己缓慢承受，不会立刻影响他人对你的评价。由于没有立即可见、可观的严重后果，你便缺乏紧迫感。

所以，你必须主动发动周围的人和力量。去思考哪些人可以参与到你的任务和工作中，让一个人的工作变得有多方监管。无论是别人对你能力的质疑（“你说的我才不信”），还是对你项目的期许（“我愿意提拔你，但是……”），这都形成了 **牵一发而动全身** 的局面，让你不敢拖延。

## 总结：实现简单高效的自律

本节课中，我们一起学习了一种将外部监督与个人承诺相结合的高效工作法。

通过这种方法，你能够实现真正的自律，成为令人羡慕的模样。对他人而言，自律是反人性且痛苦的；但围绕本节所讲的方法，你的自律会变得**简单、高效又快乐**。

试想一下：你既完成了项目和任务，又获得了他人的赞许。当别人问“我们都做不到，你怎么做到的？”时，你心中明白：其实我也做不到，我只是更了解人性，并懂得如何掌控人性。
