# 《爱商训练营》第07课：清晰聊天的正确目的，成为受欢迎的人 👥

在本节课中，我们将要学习聊天的核心目的。许多人在聊天中感到困惑和挫败，往往是因为目标设定错误。我们将明确聊天的正确方向，并理解如何通过引导对方提供话题，来建立真正的吸引力和舒适感，从而成为更受欢迎的人。

## 回顾与问题根源

上一节我们探讨了为何学习聊天总是失败。其根本原因在于，许多人被过时或缺乏实战经验的理论所误导。这导致了一个严重问题：你不仅没有获得理想的社交结果，甚至可能因为错误的学习，失去了原本自然的交流能力。

如果你发现自己学完后反而更不会聊天了，那么问题很可能出在学习源头上。

## 形象包装是基础

上一节课我们介绍了通过头像和朋友圈搭建人设的重要性。一个良好的人设能为你赢得对方的注意和时间。关于如何具体构建，在我们的内部课程中有详细的一对一指导，能帮助你在短时间内焕然一新。

形象改造至关重要，但关键在于向谁学习。你的学习对象，将直接影响你能否吸引更多异性的好感。

## 恋爱的本质是“销售”

其实，恋爱的本质类似于如何将产品成功销售出去。第一件事是**包装**，好的包装能带来更多“客源”。当你拥有了潜在对象，才会面临第二个问题：如何通过聊天“销售”自己。

你的形象展示向谁学习，是像易烊千玺还是赵本山，这取决于你自己的选择。但核心是，有效的展示是成功聊天的前提。

## 聊天技巧的核心目的

如果你的形象展示是合格的，对方愿意进一步了解你，那么我们就进入第二点：如何把自己“销售”出去，也就是聊天技巧。

**聊天技巧的核心，是在最短时间内让对方对你有好感。** 但请注意，更关键的一步是：

**如何让对方为你提供话题，让对方愿意陪着你聊。**

你必须记住这一点：**你的聊天目的是为了让对方给你提供话题。** 如果你的目的错了，结果必然也是错的。现实中，很多人的聊天总是自己在找话题，对方从未主动提供，聊天从未围绕你展开。任何教你这样做的课程，本质上都是错误的。

在我们的内部课程中，会详细讲解如何让女生围绕你聊天。即使你只回复一个表情或“哈哈”，对方也愿意追着你聊。跟什么样的人学习，就会成为什么样的人。

## 驳斥常见的错误目的

在聊天初期，唯一正确的目的是：**思考如何让聊天内容与自己相关，如何让对方提供整场聊天的话题。**

你可能会觉得这不可能。但世界上有很多你未曾见过、认为不可能的事情。我们的目标是引导你建立正确的思维，提升认知，从根本上解决“不会聊天”的问题。

许多导师自己都未曾接触过优质女性，他们的形象和谈吐缺乏吸引力。以下是几种常见的错误聊天目的论：

以下是几种需要警惕的错误聊天目的论：

*   **聊天是为了邀约**：如果你只想着邀约，往往会越约越约不出来。
*   **聊天是为了吸引对方**：那么“吸引”到底是什么？**吸引就是激发对方在你身上探索的欲望，吸引就是让对方围绕你聊天，吸引就是让对方提供话题。** 很多人空谈吸引，却不明白这个道理。
*   **聊天就是为了升高关系**：持这种观点的人也是错误的。单方面的一厢情愿去升高关系，很容易遭到拒绝。很多兄弟因为害怕被拒而迟迟不敢行动，导致关系停滞不前。

## 聊天的正确本质

在我看来，聊天不是为了升高关系，也不是为了说情话。

**聊天的本质就是：引导对方提供话题。**

你应该思考的是“怎么让他提供话题”，以及“怎么让他围绕我聊天”。为什么？因为一旦你建立了这个认知并朝此努力，你就会明白：

**聊天的本质是让女生喜欢你。只要她喜欢你，你邀约她就会出来；只要她喜欢你，你说什么她都开心。**

因此，聊天如何让女生喜欢你的问题，就转化为了**如何激发对方在你身上探索的欲望**。

## 实践与展望

在我后续的课程中，有大量的实战案例。你会看到女生从对我一知半解，到快速地在三到五分钟内提供大量话题。有的女生甚至会主动“开车”。你会看到一个原本不理我、看不起我、甚至对我有偏见的女生，如何快速地转变为主动提供话题、主动升高关系、甚至主动邀约。

如果这种情况在我身上大量出现，那么你向这样的人学习，是否离你的成功更近了呢？
