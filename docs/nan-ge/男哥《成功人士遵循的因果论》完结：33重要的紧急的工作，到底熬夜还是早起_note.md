# 课程一：高效体能管理 - 睡眠与工作节奏 🛌⚡

在本节课中，我们将要学习如何通过优化睡眠和工作安排来提升效率，避免无效的疲劳战。核心在于理解“宁可早起，绝不熬夜”的原则，并学会通过科学的休息为高效工作赋能。

## 1. 熬夜与早起的抉择 ⏰

上一节我们介绍了睡眠的重要性，本节中我们来看看当面临紧急工作时，应如何在熬夜与早起之间做出选择。

如果有一项重要且紧急的工作，必须在三小时内完成，你面临两个选择：从晚上11点工作到凌晨2点，或者从早上5点工作到7点半。建议选择早起完成工作。

当你发现需要熬夜数小时才能完成任务时，选择早起通常更高效。这是我的亲身经验。我过去常因赶项目或总结工作而熬夜至凌晨四五点，甚至六七点。但我发现，每天晚上9点或10点之后，我的效率极低。

因此我调整了策略：在晚上11点或12点获得优质睡眠，并选择早起。我发现一上午完成的工作量，相当于整个下午加晚上的总和。我周围一些成功的伙伴也采用这种方法。

以下是他们处理紧急项目的典型做法：
*   当近期有大量项目需要推进时，他们会在晚上8点或9点就入睡休息。
*   根据工作量预估，他们选择在凌晨4点到5点起床。
*   起床后，他们吃饭、工作、写报告、阅读学习，一气呵成地工作到早上8点或9点。

这样，他们既完成了原本需要熬夜做的事，又避免了熬夜。因为在凌晨4点时，他们已经睡了6到7个小时，精力充沛。反之，如果选择熬夜，到深夜一两点时会因疲惫和休息不足而导致效率反而更低。

## 2. 避免疲劳战，为大脑充电 🔋

这就是为什么我开头强调，不要认为熬夜、对自己狠就能成功。必须学会做聪明人，找到事半功倍的方法。我不希望大家打疲劳战。

凌晨12点之后，头脑容易昏沉。那种状态下，即使你在忙项目、看合同或处理任务，也可能漏洞百出。我建议每个人在12点前上床休息，然后在5点起床，继续你原本打算熬夜完成的事情。

这时天已亮，你的大脑经过几小时睡眠休息后也变得清醒，效率极高。

## 3. 学会适时“补教”与能量守恒 ⚖️

第一点中的第三小点，是学会在适当的时候为自己补充能量、加油充电。这就像吃东西，每日三餐是基础，但饿的时候是否需要加餐？我认为这是一些原则。

由于工作和生活节奏紧张，我们可能无法完全控制。以上有些点或许你暂时做不到，例如有时晚上必须加班。但你可以在周末安排时间进行“睡眠补教”。

我把周末补觉安排得如同完成一个项目一样重要，并严格执行。因为我始终明白能量守恒的道理：欠下的睡眠债，必须找时间自行调整补上。即使是天塌下来的大事，也应保证基本的精神状态和睡眠时间。

可以将自己想象成一个电子设备，比如iPad。从周一到周五，电量已被消耗殆尽。睡眠不足就像只用充电宝短暂充电，电量从未充满。周末时，你就必须用高效的充电器把电充满。

很多时候，我们不能出人头地、缺乏执行力或效率低下的原因，就在于不断消耗自己、内耗自己。这个“电量”可能已经好几年没有充满过了。这就是聪明人、有能力的人与普通人的区别。

普通人可能会怀疑：如此重视睡眠，是不是不够努力？是一种懒惰？不够能吃苦？也许成功的人真的不需要太多睡眠？但我想告诉你：**想成功，先睡足7个小时**。

士兵在平时训练中极其艰苦，但会不会让他连续好几年睡不好、伙食跟不上呢？不得不承认，打仗时可能会处于极端痛苦的环境，无法睡觉。但在高强度训练后，要不要给他补充体能和休息呢？

请相信一句话：**“一个人必须工作到精疲力尽才能成功”是一个错误的概念和错觉。这种做法代价极高。**

如果你的职业生涯是30年，而不仅仅是一两年，这种每天将精力耗尽用于工作的方式，会极大损害健康。从长期看，将严重影响你的生产力，并最终影响结果。

## 4. 睡眠不足的代价与科学依据 📉

请再相信一句话：你人生中犯过的所有错误，都可以追溯到一个源头。你说当时选择错了或情绪错了，但再往前追溯是什么？是饮食、作息、睡眠、休息和疲惫。

身体的疲惫让我们的大脑无法正常运转，从而做出错误的选择和产生负面情绪。

根据美国兰德公司的研究，仅在美国、日本、德国、英国和加拿大这五个国家，每年因睡眠不足导致的经济损失就高达**6800亿美金**。

当你发现自己的效率、创造力、记忆力、学习能力和决策力在近期自我总结中不高时，请注意，睡眠对这些能力有着非常深远且直接的关联。

你的人生拼的就是效率、记忆力、工作态度、工作能力和学习能力。如今，世界顶尖的投行高盛和顶尖的商业咨询公司麦肯锡，都已开始雇用睡眠专家，以帮助员工获得充足、高效、优质的睡眠。

## 5. 核心原则总结 📝

所以你应明白，为什么我把“高效体能，让时间做加法”这第一点的首要位置，用来讲睡眠。我希望用我的亲身经历让你明白，不要再相信那些宣扬不睡觉、少睡觉的“毒鸡汤”。我不相信有任何成功学可以颠覆人体的自然规律。

**你只有先睡得好，才有精力去攻克人生中想要实现的目标。**

好，我们总结一下第一点：提高睡眠质量。

首先，需要养成基本好习惯：
*   所有电子设备都不应放在卧室。
*   调整呼吸、放松肌肉、保持平静意识。
*   使用高质量的床上用品。即使房子差一点，卧室、床和床品也一定要有品质。

理想状态是每天睡眠够7个小时。如果实在不够，必须学会补觉：
*   熬夜后的第二天下午，饭后容易困倦，在3点左右或5点到8点之间，给自己一个补觉入睡的好时机。
*   或者等到周末，对自己好一点，不要胡吃海喝，拿出半天时间好好睡觉。

还有最重要的一点，重申我们的核心公式：**宁可早起 > 绝不熬夜**。
