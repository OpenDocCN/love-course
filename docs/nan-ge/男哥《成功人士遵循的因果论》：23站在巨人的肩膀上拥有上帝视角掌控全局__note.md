# 男哥《成功人士遵循的因果论》第23课：站在巨人的肩膀上 👣 拥有上帝视角掌控全局

在本节课中，我们将要学习一种高效的学习与认知方法——“站在巨人的肩膀上”。这种方法旨在帮助初学者攻克那些看似艰深、枯燥的专业领域，通过借鉴前人的经验和智慧，快速建立全局观和实战计划。

## 概述：成功人士的学习共性

你会发现，大部分成功人士在学生时代通常具备三个特点：第一，学习成绩好；第二，学习方法好；第三，很会“耍小聪明”。这里的“耍小聪明”指的是能够迅速领悟他人点拨，并借此获取更多可用的“工具包”。

## 核心方法：借助“工具包”顺藤摸瓜

“工具包”是指搜集线索、顺藤摸瓜的能力。具体来说，就是围绕你要学习的书籍或课程，去探索其相关的知识点和周边内容。这就像“抛砖引玉”，目的是找到那些已经走过这条路的前人的理解和经验。

**公式：高效学习 = 目标知识 + 前人经验（线索与周边）**

因此，如果你觉得学习某样东西太累、学不懂，我强烈推荐你使用“踩着巨人肩膀”这个方法。

## 方法应用实例

以下是该方法在两个不同场景下的具体应用：

**实例一：学习抽象的情感课程**
有的人觉得情感课程枯燥晦涩，这与个人经历有限有关。这时，你可以考虑组建或加入社群，与有丰富经历的人交流。你可以询问：“对这件事你是怎么理解的？你有没有疗程记录可以给我看看？”他们往往会指出重点与非重点，让你事半功倍。

**实例二：研究“性格是否决定成败”**
假如你想改善性格，去知乎搜索“性格是不是决定成败”。你会发现存在不同观点，例如，有的观点与我的看法一致：**性格不决定成败，掌控性格与情绪者得天下**。这时你就明白了，在阅读《性格决定成败》这类书籍时，需要持有辩证的看法。作者的**观点并不一定是正确的**。在阅读时，你就要思考：哪些内容可为己所用？哪些不行？你可以通过豆瓣、知乎等平台，查阅许多成功人士对此的认知，甚至包括那些“三年挣了6000万”的人的看法。这能为你提供多角度的参考。

## 方法的核心优势

我推荐给你的第一个“工具包”，特别适合处理艰深、晦涩的专业领域。借鉴此法对你将非常有效，原因如下：

1.  **降低难度**：事情变得不再困难，因为前人已大致探索过一遍。
2.  **提升格局**：在阅读和学习时，你能对整体局面有一个更具实战性的计划性认知和布局掌控。
3.  **建立门槛**：任何晦涩、难懂但结果导向的实战技能，此法都是你的“敲门砖”，是他人难以超越的门槛。当大家都觉得晦涩时，你因获得了提示和不同视角，事情就不再神秘，你也就有了方向。

我们比拼的其实就是**求知能力**。我比你强，你比他强。我希望你能在求知路上，踩着巨人的肩膀，变得更高效、更有收获。

## 过渡与预告

当然，南哥所讲的明理、理解与实操能力，仅此一个维度吗？不是的，它是**三维一体**的。下节课，我将为兄弟们讲解另外两个维度。

我认为，当你将这三个维度结合运用，去学习你所需的任何领域——无论是情绪管理、领导力提升，还是后续我将推出的众多实操课程，甚至是学习伦敦投行的金融交易方式——其底层逻辑都是相通的。

## 类比总结

这就好比你想去旅游，比如去马尔代夫（尽管目前疫情下可能无法成行）。你应该怎么做？去查看关于马尔代夫消费和流程的攻略，同时看看**小白、中等收入和高收入人群**分别是如何看待这次旅行的。你一定会有所收获。

请记住，当你感到困惑时，你首先应该解决的是 **“如何能够高效”** 这个问题，而不是认为“困惑是应该的”。从“学不会”到“学会”，本可以不必如此艰难。其实，优秀的人比你更懂得化繁为简，他们善于做减法，并且特别热衷于做一件事：**走捷径**。
