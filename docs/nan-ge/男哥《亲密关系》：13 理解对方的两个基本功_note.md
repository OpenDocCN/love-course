# 男哥《亲密关系》：13 理解对方的两个基本功_note

在本节课中，我们将要学习如何真正理解对方。理解是建立和维护亲密关系的基石，而要做到这一点，我们需要首先练习好两个核心的基本功。这两个基本功与我们之前课程中提到的“表达自己”有相似之处，但侧重点在于如何接收和解读对方的信息。

## 概述：为何需要理解对方？

在生活中，我们常常遇到特别理智的人。例如，当朋友非常伤心、独自哭泣时，一个理智的人可能会劝解：“哭有什么用？难过有什么用？生活不相信眼泪。你现在不应该难过，应该想怎么解决问题。”从道理上讲，这些话没有错，甚至说的是实话。但这种方式不近人情，尤其在情侣之间。

当一个人需要发泄情绪时，他必须将情绪发泄出来。此时，简单地拥抱对方，说“没事，哭出来就好了”，比逻辑地解释问题更有效。如果你变得过于逻辑，对方反而会觉得你不懂他、不理解他。

为了说明理解的重要性，我们来看一个案例。

## 案例：一对需要被理解的母女

一位母亲在年轻时因丈夫出轨而离婚，独自抚养女儿。女儿长大后特别叛逆，但学习成绩优秀，从名牌大学毕业，外貌出众。然而，女儿坚持要嫁给一个比自己大十几岁、没有学历和地位的男人。母亲对此非常费解，无法接受女儿的婚姻，也不搭理女婿。

对于这位母亲而言，年轻时离婚、丈夫出轨、女儿不听话，这些都是至亲之人的背叛，她感到伤心。她内心的痛苦得不到理解，也没有释放的出口，因此她会固执己见，拒绝听取他人意见。她需要的是被理解，她的情绪需要有一个宣泄的出口。只有这样，她才有可能接受女儿的婚姻和女婿。

同时，女儿也需要被理解。一个各方面条件都不差的女生，为何要选择这样一个伴侣？原因在于原生家庭的缺失。她从小父母离异，缺乏父爱，因此对父亲角色有一种向往。父母离婚的经历让她对“门当户对”的同龄优秀男性缺乏安全感。相反，一个年纪较大、条件一般但全心全意对她好的男人，能让她感到安心，不用担心对方出轨。对这个中年男人而言，能有一个年轻漂亮的伴侣也别无他求。因此，女儿的选择是她母亲婚姻失败的受害者心态的体现。

在这个案例中，母亲和女儿都需要被理解，都需要情绪的宣泄出口。这个家庭的关系和心理状态存在一些问题。因此，帮助他人释放负面情绪显得至关重要。这引出了我们今天的第一个基本功。

上一节我们通过案例看到了理解与情绪宣泄的重要性，本节中我们来看看第一个基本功的具体内容。

## 第一个基本功：接纳对方的负面情绪 🧘‍♂️

第一个基本功是：**我们需要练习自己接纳对方负面情绪的能力**。这意味着我们要么自己看清问题的本质，要么帮助对方看清问题的本质。

### 案例：一段因忽略情绪而结束的婚姻

举一个例子：一位三十出头、已离婚的女士。她的第一段婚姻是闪婚，她本人单纯善良，对丈夫很好。但后来她发现丈夫出轨，两人最终离婚。她一直感到委屈和憋屈，认为自己对丈夫那么好，对方却做出这种事，因此对前夫怀有仇恨心态。

我与她分析：并不是丈夫出轨导致婚姻出现问题，而是婚姻中先出现了问题和裂痕，才导致了出轨。我引导她回忆婚姻中是否发生过什么改变。她最终想起来，前夫曾经历约半年的工作低谷期，被原单位辞退，且长时间找不到新工作。那段时间，一个男人经受这样的打击，情绪非常低迷。

然而，那时恰好是她自己新工作刚开始的时期，她忙得焦头烂额，根本没有时间关注前夫的心情。她复盘时间线后发现，正是那段时间，她前夫在网上认识了一位女网友，对方经常开导他，最终导致了出轨的结果。问题的核心并非她不够善良，而在于她忽略了前夫的情绪，将注意力完全放在了自己的新工作上。

这个故事说明，在亲密关系中，忽略伴侣的心情可能造成严重后果。因此，练习接纳对方负面情绪的基本功非常重要。

### 如何练习接纳情绪？

以下是具体的练习方法：

准备一张纸，将其划分为小块区域。每个区域记录以下五个要素：
1.  **人物**：情绪的主体（如：妻子、同事、父母）。
2.  **时间**：观察到情绪的时间点。
3.  **地点**：观察到情绪的具体地点。
4.  **情绪**：你观察或感知到的情绪状态（如：心烦、焦躁）。
5.  **求证**：向对方核实你的观察。

**记录格式示例**：
```
1. 人物：妻子
   时间：晚上7点半
   地点：客厅
   情绪：似乎有些心烦、焦躁
   求证：询问“你现在是不是有点心烦？”
```

**练习要点**：
*   **对象**：主要记录伴侣的情绪。如果单身或异地，可以记录每天接触的人，如同事、父母，以训练感知他人情绪的能力。
*   **求证的作用**：求证时，对方可能否定你的观察（例如，回答“没有不开心”）。这没关系，因为即使被否定，对方也能感受到你的关心，这对感情仍有积极作用。如果对方否定，不要追问，给予对方空间，等他愿意沟通时再交流。

这个练习的核心目的是培养你主动关注和接纳他人情绪的习惯。

掌握了接纳情绪的基本功后，我们还需要能够精准地识别情绪。这就是我们要介绍的第二个基本功。

## 第二个基本功：准确描述对方的感受 🎯

第二个基本功是：**我们要能够准确描述出对方内心的感受**。这与我们之前“表达自己”课程中提到的“准确描述自我感受”是相通的。

当对方有负面情绪时，你需要判断清楚究竟是哪一种或哪几种情绪：是**生气**、**绝望**、**担心**、**委屈**、**失望**，还是**尴尬**、**失落**、**后悔**、**愤怒**等？要用准确的词汇说出对方的心理状态。

### 为何准确描述如此有效？

心理学规律表明：当一个人体验到某种情绪时，如果有关心他的人能**越准确地描述**出这种情绪，对情绪释放的帮助就越大。对方会觉得你真正理解了他，为他的情绪提供了一个出口，从而产生“你很懂我”的感觉。

### 处理复杂情绪

当对方同时存在多种复杂情绪时（例如，既有**后悔**又有**生气**），你应该尝试将**每一种情绪都分别描述出来**。比如：“我看你现在好像有些后悔之前的选择，同时也在为这件事生气。”这样，对方后悔部分的情绪可能得到宣泄，而生气的部分仍然存在，但你已经为他打开了一个理解的窗口。

这要求我们设身处地地为对方着想，仔细体察其复杂的内心感受。

## 总结与练习建议 📝

本节课我们一起学习了理解对方的两个核心基本功：
1.  **练习接纳对方的负面情绪**：通过日常观察、记录和求证，培养关注并接纳他人情绪的习惯。
2.  **练习准确描述对方的感受**：使用精确的词汇识别并表达出对方的情绪状态，尤其是复杂情绪。

这两个基本功说起来容易，做起来需要长期的练习和积累。它们是建立深层理解的基础。下节课我们将讲解四个理解对方的具体方法，但这些方法都建立在今天这两个基本功之上。因此，请务必重视今天的练习，将其融入日常生活。
