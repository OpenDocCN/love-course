### 人的两种思维模式

人的思维模式一般分为两种，这两种模式反映了一个人对自己学习能力的判断。

以下是两种思维模式的具体描述：

*   **保守型思维模式**：这种模式倾向于总结自己**现有**的特点和特质。例如，当被问及“你是一个怎样的人”时，可能会回答“我数学很好”、“我乐观开朗”。这种思维基于过去已具备的特征给自己下定论。
*   **成长型思维模式**：这种模式关注自身的**潜力**和**未来发展**。同样的问题，成长型思维的回答可能是“我是一个正在努力健身的人”或“我正在努力学习提升自己”。这种思维将自己视为一个不断变化、越变越好的过程。

我们应当用成长型思维模式看待自己，因为人的特质是可能变化的。过去的优点不一定能持续，过去的缺点也未必无法改正。永远不要给自己下一个确定的、局限性的定论。

### 如何提升学习动力：批判性思维

除了转变自我认知的思维模式，提升学习动力的另一个关键是学会如何找到问题，这需要用到批判性思维。

批判性思维不等于简单的批判或抬杠，它是一种理性的评估信息的方式。在中国以应试教育为主的环境下，这种思维较少被强调，但它对于独立思考和有效学习至关重要。

以下是养成批判性思维的四个关键问题，每当接收到一条新的外界信息时，都应思考：

1.  **信息的发布者是谁？** 这个人的身份是什么？
2.  **他发布这条信息的目的是什么？**
3.  **这条信息是否公正？**
4.  **信息中的数据来源或道理是否科学合理？**

掌握了这四个问题，你就能辨别信息的可靠性，吸收其中有用的部分，同时保持思维的开放性，避免思维固化。当你学会对已知信息提出疑问时，你就开启了主动学习和成长的循环。

### 高效学习模式

结合以上内容，最高效的学习模式不是埋头苦读，也不是边学边实践。

**最高效的学习模式 = 学习 + 实践 + 总结**

这个模式能让你以最快的速度提升。

### 课程总结

2.  **批判性思维**：通过四个关键问题（发布者、目的、公正性、科学性）来评估信息，从而发现问题、保持学习动力。
3.  **高效学习模式**：最有效的提升方式是结合学习、实践与总结。
