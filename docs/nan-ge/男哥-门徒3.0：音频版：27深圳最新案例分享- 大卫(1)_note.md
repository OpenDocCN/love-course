# 男哥-门徒3.0：音频版：第27课 深圳最新案例分享-大卫(1) 💬

在本节课中，我们将通过一个真实的深圳聊天案例，学习如何在实际互动中运用核心心态与技巧。我们将重点分析如何应对女生的拒绝、犹豫和负能量，并掌握将线上窗口转化为线下邀约的关键步骤。

## 核心心态：不把问题当问题 🧠

上一节我们介绍了课程概述，本节中我们来看看案例中贯穿始终的核心心态。

当对方说“不加（微信）”时，在我看来这完全不是问题。我无所谓，而不是像有的人可能遇到这句话就不知所措。思考一下这句话：**不要把这个事当做一个问题**。

那么它就不会成为你的障碍。在实际中，我是这样运用这个思维的：我根本不焦虑。你可以从我的回复中看出来，我并没有把这件事当回事，也没有把这个问题看作她对我的阻碍。

我只是说“好，那完事再问你”。在我看来无所谓，你迟早都会给我。当我不把这件事当作自己的难题，当我看得比较开时，她的态度就缓和下来了。

## 识别窗口与保持节奏 🎯

上一节我们学习了核心心态，本节中我们来看看如何在实际对话中识别兴趣并保持自己的节奏。

你的目的是推进关系。其实能看出来她对我有兴趣，但可能缺乏安全感。如果你根据对方的回复去造句，那么显然就已经聊到头了，因为“哦哦”没办法往下接。

我有自己的节奏感和方向。我能看出来她又在犹豫。她发的表情很多都透露出犹豫的感觉。所以我说“你这种纠结的真的是天秤了”，因为天秤座比较明显的特点就是纠结。

她又说“加了聊不到一起都懒得删”。我说“所以说呢，我晚一点再问你一遍”。我依然没把这当回事，在我看来你迟早都会加，只不过是时间早晚问题。我还加了一句“傻瓜”，这带有打情骂俏的感觉。

## 分析社交信号：IOI与IOD 🔍

上一节我们讨论了如何保持节奏，本节中我们深入分析一个具体的社交信号。

她接着说“快服了你了”。这是一个需要判断社交直觉的时刻：大家觉得“快服了你了”是一个IOI（兴趣指标）还是IOD（无兴趣指标）？

在我看来，“快服了你了”是一个略带轻微打击的IOI，是她想跟我互动的表现。如果她说“我服你了”，那可能就是IOD。汉语博大精深，“我他妈真服你”可能就是一种无话可说的状态。但她先发了一堆笑脸，再来一个“我快服了你了”，感觉是无奈，但同时又有窗口。

我们仍然不能根据她的回复来造句。她说“我快服了你了”，你怎么接？在我看来，她的闭合话术并不影响我。因为她都开始说快服了我了，我认为这时是展现自己的好机会。

所以我跟她说：“我这个人呢特别简单，不喜欢套路。看我ID就知道，我ID叫‘少一点套路’。”意思是我自己尽量少用套路，也告诉别人不要套路我，这营造出一种高手的感觉。

## 灵活沟通与填充安全感 💬

上一节我们分析了社交信号，本节中我们看看如何通过灵活沟通来建立舒适感。

我们一定要学习一种灵活的沟通方式。不是说你夸了对方，都得说“那你是不是也该夸夸我”。我可以换非常灵活的方式。比如她下一句接的是“你是为什么又夸我的”，其实这句话跟“你是不是也该夸我”是一回事。

然后她开始跟我说，觉得我照片上的形象不错。下面还有一些语音，分享她生活的一些细节片段。这些内容其实不重要，但她跟我分享生活，比如她有一个“美人帮”闺蜜群。

我也给她分享我的生活状态：我刚吃了一个肠粉。我很明白，她第一次拒绝加微信是觉得聊得不够多，这通常是缺乏安全感。怎么填充安全感？就需要多分享生活日常点滴，这非常好用。

以下是关于聊天频率的关键点：
*   不需要随时待命一样去跟人聊天。
*   一个真正有价值的人有自己的生活、工作和其他社交。
*   我偶尔发一句，可能很长时间不回。后来八点多我说“下午刚忙完”，她说“也在忙”，我说“晚饭还没吃”。这些都是填充安全感的点。

## 应对负能量与高价值共鸣 ⚡

上一节我们学习了如何建立安全感，本节中我们来看看当对方状态不佳时该如何应对。

这时聊天开始变得有点走心。但你的社交直觉需要足够敏锐。她说了一句话：“谢谢什么光鲜亮丽，你来试试我的背后啊。”

首先，不要安慰她。屌丝才会去安慰，说“没事，抱抱”之类的。我不会安慰。第二，我明白今晚不适合聊太多，因为她状态不OK。这是我意识到的第二点，一定要有社交直觉。

回到案例中。尽管是走心的聊天方式，但我不会在这个时候跟她走太多心。因为显然她情绪不OK，这是一种负能量。我没有安慰或跪舔，而是告诉她“哎，我也一样”。

我们现在的节奏频率就一致了，而且我的价值显得比她更高。我展现出了比她更好的心态，并且能引发共鸣。当女生抛出负能量时，你要思考如何高价值地表达，保持同一节奏频率，并激发共鸣。比如她钱丢了，你不要说“我再给你钱”，而可以说“我手机前段时间也丢了”，去表达节奏一致的感觉。

## 处理消极反馈与及时止损 🛑

上一节我们讨论了如何应对负能量，本节中我们看看当方法无效、收到消极反馈时该如何处理。

这时她回了一个“呵呵”。怎么办？思考一下这个“呵呵”你会怎么回？当你遇到一个问题，不把它当回事，应该用什么样的语气和态度去回？不要曲解。

我前面的聊天没有任何问题。“呵呵”什么？你喝你妹呢？所以这时还是在于“不要把这个事情当回事”。她状态情绪都不OK，负能量爆棚时，我就不要跟她交流太多。直接“好吧，晚安”就行了。

有人说什么曲解，把“呵呵”拆成“口渴”，然后回“给你水”。这种回复方式太套路了。你回“给你水”，对方肯定会问“什么意思”，然后你又去解释。对方回一个省略号。只要你说了一句话还需要去解释其中的幽默感，那就不是幽默，是傻逼。所以我直接晚安，第二天接着聊。任何人都会有情绪低谷的时候。

## 把握时机与直接邀约 🎉

上一节我们学会了及时止损，本节中我们看看如何在对方状态恢复后把握机会推进关系。

第二天继续聊。我说“早”，她说“不早了”，我说“刚起来”，她说“舒服”。节奏频率完全一致，注意能量守恒。她没有给我太高能量，我也不会给她太高能量。

第二天遇到一件事：我们深圳线下课有导师带兄弟们去夜店蹦迪的环节，叫了很多女生，刚好碰上了。而且当天的卡座是我们开的，所以我直接说了一句：“请你请你去喝酒啊。”

结果她问：“什么意思？”已经很明白了，我就是请你喝酒。再用一下刚才的思路：当你不把这个问题当做问题时，应该是什么语气和态度？所以直接一句：“就是叫你一起来酒吧玩啊，来不来嘛？”你爱来不来，无所谓。

## 化解疑虑与展现真诚 💎

上一节我们进行了直接邀约，本节中我们看看如何化解对方的后续疑虑。

她问我要具体信息，包括位置。在我看来，她还是缺乏安全感。关键是我和他之间的事，而不是这个局有多少人。这完全不是问题。感觉好，感觉对，为什么不见面？感觉对为什么不一起玩？感觉这种东西无法具体解释，逻辑经不起推敲，而感觉最靠谱。

我说：“感觉跟你还蛮聊得来，所以想见一面聊聊天玩一玩喽。”然后她又抛出一个难题。我发现这个女生真的不停地有难题抛给我。

再给大家一次机会思考如何解决。不会？你为什么要把这个东西当作一个问题？你为什么要把这看作困难？

接下来我的操作是：他说“OK”，我说“我没那么多套路啊，想吃什么就去吃，想见什么就主动说”。意思是，我还是对你有窗口，不想错过机会，但没有直说，比较含蓄隐晦。

## 识别隐藏窗口与引导变现 🚀

上一节我们通过真诚化解了部分疑虑，本节中我们学习如何识别对方隐藏的兴趣并引导见面。

虽然她回了一个“OK”，看似今天不会见面。但仔细看，她为什么要写作文？这种超过三行字就是写作文行为。如果一个人不是特别想见，没有多大窗口，干嘛打这么多字？我还是能看出她隐藏的窗口。但这个窗口需要一个正确的引导才能“变现”。有好感没用，得让她出来见面、互动。

这时操作来了：她写了五六行字作文，我就回三个字：“我懂啊。”引导她说更多东西。她还给一个闭合话术，聊天很难受，各种难题。但在我看来无所谓，你爱来不来，我开的卡，这是我的心态。

我根本不焦虑，因为我的做法没问题。我在说她的问题：“感觉你怎么这么焦虑呀？”她给我负反馈，我不仅自己不焦虑，还在说她的问题。想一下自己聊天时是否缺少这样一种感觉。

## 案例总结与线下课推广 📈

上一节我们完成了从线上到线下的引导，本节中我们对整个案例进行总结。

这就是一个快速的邀约，持续两天的聊天，第一天加，第二天邀约。后续按照线下课里讲的夜店操作流程，基本没问题。最后我自己并没有和女生发生故事，因为线下课以学员为主，在带学员，该约的会都没去，该见的人都没见。线下课挺忙的。

我说的是心态上“爱来不来”，但如果你把这话说出来，对方绝对拉黑你。我不是让你真的说这话，而是要有这种感觉。

这次线下课学员收获非常大，拍出了非常棒的展示面，每个人蜕变都很彻底。线下成长比线上快得多，七天相当于自己学一年。我们帮你拍展示面，你能迅速看到喜欢人数增加，一天能认识20-30个女生。展示面只是其中一个环节，还有夜店操作等内容。更重要的是感官体验，让你真正“坐上飞机”体验飞行的感觉，学得很快。
