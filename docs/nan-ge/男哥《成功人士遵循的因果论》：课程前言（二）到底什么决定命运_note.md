# 男哥《成功人士遵循的因果论》：课程前言（二）到底什么决定命运 🧭

在本节课中，我们将要探讨一个核心问题：究竟是什么真正决定了一个人的命运？我们将逐一分析常见的误解，并揭示那个拉开人与人之间差距的关键因素。

很多人认为性格决定成败，但我不这么认为。每一个成功人士的性格都各不相同，因此，决定命运的关键并非性格。

那么，是不是努力决定了命运呢？

我也想告诉你，努力同样不是决定性因素。凌晨扫马路的阿姨比我们许多人更努力，我并非轻视这份职业，只是想说明，单凭努力是不够的。

所以，最正确的答案是什么？决定一个人命运的是**心智**。

心智才是拉开人与人之间距离的核心因素。提升心智分为五个阶段，以下是这五个阶段的详细说明。

第一个阶段是**动物本能阶段**。处在这个阶段的人对自己毫无察觉和反思，行为完全依托本能。他们的思维模式是：“我觉得就应该这样，我认为就这么做吧。”

猪不知道自己是猪，所以它才是猪。如果一头猪发现自己是一头猪，它绝不会甘愿做一头猪，哪怕钻地洞也会逃跑。

上一节我们介绍了懵懂的本能阶段，本节中我们来看看觉醒的开始。

第二个阶段叫做**反思阶段**。人们开始关注自己行为背后的原因，思考“我为什么会这么做”。处在这个阶段的人，抱怨和指责会越来越少，开始全方位地反思自己。

就像列宁所说：“当一个人开始反思自己的时候，他将进入一个伟大的阶段。”许多跟随我几年的兄弟们，你们正在做的事情就是学会安静下来，学会独处，学会独立思考。因为高处不胜寒，当你进入第二阶段开始思考时，很多人可能不理解你，因此这个阶段可能会感到孤独。

在学会反思之后，下一步便是管理内心的波澜。

第三个阶段是**情绪控制阶段**。人们逐渐开始接纳自己，减少内耗，心态趋于平和稳定，并变得非常理性。我的课程中有那么三四节，重点不是教大家拼命蛮干或消耗自己，而是先帮助大家算清楚自己有哪些内耗。

例如，你天天在思考一件事却没有行动，这种思考本身就是一种内耗。处在这个阶段，外界评价和嘲笑对他产生的干扰会越来越少。

控制好情绪，我们才能将目光投向更长远的目标。

第四个阶段是**知行合一阶段**。人们开始进行长期的自我观察和关注，行动力变得越来越强。这个阶段的人，思考时是用大脑控制行为，会非常专注，并善于用行动带来的变化去反哺和修正思想。

即使内心支离破碎，脚步也依然不会停下来。这就是我常说的，成功人士往往形成了一个**自有的、不断吸收新鲜事物并能落地实践、形成良好效果的闭环思想体系**。他情绪再糟糕，内心再破碎，都不影响他做好每一件事，因为他已经拥有了自己的体系。

当知行合一成为习惯，人便会进入一种极致的状态。

第五个阶段是**稳定的专注阶段**。处在这个阶段的人，对自己要达到的目标和结果进入了完全痴迷的状态。我们常常看到，当初与我们水平相当的人，我们会觉得他这么做是“傻逼”、“神经病”、“痴迷了”。

但过两年，他就和我们不一样了。面对同样的事情，他显得非常违心、非常痴迷，甚至让我们无法理解。但这样的人，做事的结果一般不会差。当然，他同样会失去很多，你觉得失去的是什么呢？

你现在处于哪个阶段？我希望你好好想一下。

本节课中我们一起学习了决定命运的核心因素是心智，并详细剖析了心智成长的五个阶段：从**动物本能阶段**，到**反思阶段**，再到**情绪控制阶段**，进而达到**知行合一阶段**，最终进入**稳定的专注阶段**。理解自己所处的阶段，是迈向更高心智层次的第一步。
