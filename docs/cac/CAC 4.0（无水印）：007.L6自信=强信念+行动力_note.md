# CAC 4.0（无水印）：007.L6 自信=强信念+行动力

## 概述
在本节课中，我们将探讨如何获得真实的自信，并理解自信与自恋之间的区别。

## 自信的定义
首先，我们需要明确自信的定义。自信是指始终认为自己有机会，并且会采取行动。它包括两个核心要素：

1. **正面感觉**：认为自己有机会。
2. **行动力**：采取实际行动。

### 公式表示：
自信 = 强信念 + 行动力

## 自信与不自信的区别
自信与不自信的区别在于情绪和行动：

- **自信**：正面情绪主导，采取行动。
- **不自信**：负面情绪和视角主导，行动受阻。

### 列表：
以下是自信与不自信的区别：

- **自信**：
  - 正面情绪
  - 采取行动
- **不自信**：
  - 负面情绪
  - 行动受阻

## 自恋的定义
自恋是指认为自己不可能失败，无法接受不完美。自恋者往往对自身要求过高，对他人要求完美，容易产生拖延、借口和退缩等行为。

### 列表：
以下是自恋的典型特征：

- 认为自己不可能失败
- 无法接受不完美
- 对自身要求过高
- 对他人要求完美
- 容易产生拖延、借口和退缩等行为

## 总结
本节课中，我们一起学习了自信的定义、自信与不自信的区别以及自恋的定义。希望这些知识能帮助你更好地理解自信，并在生活中获得更多的成功。
