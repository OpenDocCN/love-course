# CAC 3.0：L19答疑：一些被忽略的细节 📚

在本节课中，我们将探讨一些在与人交往中经常被忽视的细节，这些细节往往决定了交往的成功与否。

## 过度热情

**定义**：回复信息过多，态度过于热情，回复速度过快。

**影响**：
- 给人不安的感觉，如同黄鼠狼给鸡拜年。
- 增加对方压力，迫使对方做决定。
- 给人带来压力感，可能导致对方想要逃避。

**原则**：
1. **自述对等原则**：回复信息量与对方相当。
2. **态度对等原则**：对方冷淡时，你也保持冷淡；对方热情时，你才热情。
3. **回复时间对等原则**：避免总是秒回，适当延迟回复。

## 心虚

**表现**：自我解释，寻求认可。

**影响**：暴露需求感，降低自身价值。

**建议**：
- 不要过度解释。
- 学会建立安全感。
- 不要过度寻求认可。

## 追求舒适

**表现**：成为女生的情绪垃圾桶，不敢挑战，不敢调侃。

**影响**：
- 降低自身价值。
- 导致女生被其他男生吸引。

## 聊天过多

**目的**：邀约。

**建议**：
- 避免过度聊天。
- 高频聊天适用于异地恋。
- 低频聊天适用于短期框架。

## Too Much Game

**定义**：过度使用技巧。

**建议**：
- 先做正常人，再进行操作。
- 注意操作密度，避免每句话都在操作。

## 总结

本节课中，我们学习了五个容易被忽视的细节：过度热情、心虚、追求舒适、聊天过多和Too Much Game。这些细节可能会影响交往的成功与否，希望大家能够注意并改进。
