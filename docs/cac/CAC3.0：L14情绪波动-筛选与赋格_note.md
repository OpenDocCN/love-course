# CAC 3.0：L14 情绪波动-筛选与赋格 🎯

## 概述

在本节课中，我们将深入探讨如何通过筛选和赋格技巧来提升与女性的互动，从而建立更稳固的关系。

## 一、情绪波动与筛选

### 1. 情绪波动

在20到39阶段，给女性情绪上的波动是关键。这可以通过调侃、挑战和筛选三种方式实现。

### 2. 筛选

筛选并非意味着对女性狠心，而是对彼此负责。它可以帮助你保持权威感和高位框架。

**公式**：筛选 = 有意图的表达好感 + 保留的表达好感

### 3. 筛选实例

- 外表方面：短发、平胸、男孩子气
- 性格方面：听话、懂事、善良、孝顺、努力
- 爱好方面：艺术、酷、有自己的想法
- 生活习惯：自律、健身、读书
- 价值观：爱情观、家庭观念

## 二、赋格

### 1. 赋格的定义

赋格是一种奖励机制，当女性符合你的筛选标准时，你可以通过赋格来鼓励她。

**公式**：赋格 = 赋予想要的性格 + 赋予资格

### 2. 赋格的方向

- 有利于恋爱的方向：将赋格引向恋爱话题，初步植入男女氛围。
- 短期/长期方向：根据你的意图选择合适的赋格方向。

### 3. 失格

失格是指让对方没有资格与你在一起，通过制造阻碍来推远关系。

**公式**：失格 = 制造阻碍
