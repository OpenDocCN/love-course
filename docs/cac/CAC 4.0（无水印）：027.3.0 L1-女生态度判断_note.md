# CAC 4.0（无水印）：027.3.0 L1-女生态度判断

## 概述
在本节课中，我们将学习如何判断女生态度，从无感到依恋的五个阶段，以及每个阶段的特点和判断标准。

## 无感阶段（0-19）
### 特点
- 女生对你无感，把你当成普通人或路人。
- 女生不反感你，但也不感兴趣。
- 女生对你没有安全感，可能认为你有目的性。

### 判断标准
- **敷衍**：女生只回复嗯嗯哦哦等，或只发表情包。
- **顺着你的话往后说**：女生重复你的话，没有输出自己的信息。
- **经常不回消息**：女生可能真的懒得理你，或用不回消息的方式暗示你。

## 好奇阶段（20-39）
### 特点
- 女生对你开始感兴趣，但只对某些话题感兴趣。
- 女生会对你进行信息输出和情绪流露。
- 女生可能愿意跟你见面。

### 判断标准
- **信息输出**：女生分享自己的经历、想法和观念。
- **情绪流露**：女生表达开心、兴奋、高兴或伤感焦虑等情绪。
- **被动**：女生不主动找你，只有你找她才会回复。

## 兴趣阶段（40-59）
### 特点
- 女生对你感兴趣，想要了解更多关于你的信息。
- 女生会主动找你，给你发消息，找话题。
- 女生会向你提问，夸赞你，迎合你。

### 判断标准
- **提问**：女生问你各种问题，如星座、工作、爱好等。
- **夸赞**：女生认可你，夸奖你。
- **迎合**：女生配合你的话题，往你身上靠。
- **主动**：女生主动找你，给你发消息。
- **解释自己**：女生向你解释自己的想法和感受。
- **抛出压力测试**：女生测试你是否真诚，是否是伪装的。

## 好感阶段（60-79）
### 特点
- 女生对你有好感，想要和你建立亲密关系。
- 女生会向你索要安全感，想要和你见面。
- 女生会向你索要时间，想要和你在一起。

### 判断标准
- **索要安全感**：女生问你在干嘛，为什么没回消息。
- **索要时间**：女生想要和你见面，吃醋你和其他人在一起。

## 总结
本节课中，我们学习了如何判断女生态度，从无感到依恋的五个阶段，以及每个阶段的特点和判断标准。通过学习这些内容，我们可以更好地了解女生态度，从而更好地与女生交往。
