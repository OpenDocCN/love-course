- **情绪波动 (Tension)**: 类似弹簧的张力，通过制造压力和释放压力来维持对话的动态。
- **编辑效用递减**: 同一操作的效果随时间推移逐渐减弱。

## 情绪波动与张力
- **情绪波动**: 通过夸赞、认可等方式提升女生的心情。
- **张力**: 类似弹簧的张力，通过制造压力和释放压力来维持对话的动态。

## 给女生压力与张力
- 必须在20分以上的好奇阶段才能给女生压力和张力。

1. **调侃**: 通过幽默的方式制造压力，然后通过笑点释放压力。
2. **挑战**: 否定对方的观点和想法，但要注意不要侮辱对方。
3. **筛选**: 表达你的择偶标准，让对方知道你只喜欢符合标准的人。

- **调侃的方向**:

- **挑战的方向**:
  1. 当女生触犯你的原则和底线时。
  2. 当你掌握权威的道理时。
