# 02、CAC2.0700分钟：从把妹到亲密关系：8.关系里的话外音

## 概述
在本节课中，我们将探讨长期关系中的依恋风格，特别是回避型依恋和迷恋型依恋，以及如何应对这些情况。

## 依恋阶段的判断逻辑
**公式**： 依恋阶段判断逻辑 = 情感连接 + 回应 + 唯一性 + 纯粹情感关系

- 情感连接：与对方在一起时开心，分离时不舍。
- 回应：回应对方的话语，记住对方的话。
- 唯一性：对方不与其他人接触，只与你建立关系。
- 纯粹情感关系：对方因为你是你而爱你，接受你的缺点。

## 依恋风格
**代码**： 依恋风格 = 安全型 + 回避型 + 迷恋型 + 矛盾型 + 混乱型

- 安全型：符合所有依恋阶段判断指标。
- 回避型：害怕恋爱，忽冷忽热，不愿意建立承诺，不承认亲密关系。
- 迷恋型：害怕被抛弃，不停索要安全感，控制欲强。
- 矛盾型：回避型和迷恋型的混合体。
- 混乱型：爱恨交织，难以捉摸。

## 回避型依恋的应对方式
**步骤**：

1. **避免破防和强迫**：不要因为对方的回避行为而情绪崩溃或强迫对方。
2. **耐心等待**：依恋风格不是两天就能改变的，需要时间。
3. **主导和代理**：不要询问对方的意见，自己决定并执行。
4. **保持高位**：不要因为对方的回避行为而降低自己的价值。

## 迷恋型依恋的应对方式
**步骤**：

1. **理解对方的恐惧**：对方害怕被抛弃，需要安全感。
2. **保持高位**：不要因为对方的迷恋行为而降低自己的价值。
3. **沟通**：告诉对方你需要空间，并解释为什么。
4. **考虑是否合适**：如果对方无法改变，考虑是否继续这段关系。

## 总结
本节课中，我们学习了依恋阶段的判断逻辑、依恋风格以及如何应对回避型依恋和迷恋型依恋。记住，保持高位和沟通是关键。
