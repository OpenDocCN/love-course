# 02、CAC2.0700分钟：从把妹到亲密关系：7.依恋：黏人vs回避

## 概述
在本节课中，我们将探讨长期关系与短期关系的核心区别，重点关注依恋的概念及其在关系中的重要性。

## 核心概念
**依恋**：指个体从小到大与父母或养育者建立的一种依赖、信任和接纳的关系。

## 依恋的类型
以下是五种常见的依恋类型：

1. **安全型依恋**：个体在关系中感到安全和被接纳，能够积极表达自己的需求。
2. **回避型依恋**：个体在关系中感到不配得到爱，倾向于避免亲密关系。
3. **矛盾型依恋**：个体在关系中忽冷忽热，既渴望亲密关系又害怕受伤。
4. **迷恋型依恋**：个体在关系中极度缺乏安全感，害怕被抛弃，不断索要关注和确认。
5. **混乱型依恋**：个体在关系中表现出矛盾和混乱，既喜欢又恨对方。

## 判断依恋类型
以下是一些判断依恋类型的标准：

1. **强烈的情感连接**：个体在关系中感到强烈的情感连接，害怕失去对方。
2. **互相回应**：个体在关系中能够积极回应对方，愿意分享自己的感受和想法。
3. **唯一性**：个体希望与对方建立一对一的亲密关系，排斥与其他人接触。
4. **纯粹的情感关系**：个体在关系中注重情感连接，而非利益交换。

## 维护依恋
在长期关系中，以下是一些维护依恋的建议：

1. **主导带领**：在关系中保持主导地位，积极引导对方，避免低位框架。
2. **高位框架下的推拉**：在保持高位框架的同时，进行适当的推拉，让对方感受到你的价值。
3. **高位下的复格**：在高位框架下进行复格，给予对方认可和肯定。
4. **保持一致性**：在关系中保持一致性，避免忽冷忽热，让对方感到安心。

## 总结
本节课中，我们学习了依恋的概念及其在关系中的重要性，以及不同依恋类型的特点和判断方法。在长期关系中，维护依恋是至关重要的，需要双方共同努力。
