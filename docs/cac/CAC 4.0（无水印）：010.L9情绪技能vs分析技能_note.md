# CAC 4.0（无水印）课程 010：L9 情绪技能 vs 分析技能 🧠💬

在本节课中，我们将要学习一个核心概念：为什么你学了很多沟通技巧却用不出来。我们将深入探讨大脑的两个系统，并区分“情绪技能”与“分析技能”，帮助你理解并掌握真正有效的沟通基础。

## 为什么技巧用不出来？🤔

上一节我们提到了学习技巧的普遍困境，本节中我们来看看其根本原因。你无法应用所学技巧，是因为你的大脑不允许你执行那些步骤。

我们的大脑有两个重要系统。第一个是**边缘系统**。这是一个古老系统，许多动物也有，它掌管我们的**情绪**和**本能**。例如，众所周知的“战斗或逃跑”反应就源于此。当遇到威胁时，你想战斗或逃跑，这都是边缘系统驱动的。因此，那些能触发、激发或激怒你的模式，都发生在这个系统，你无法用理性去控制它。

更重要的是，人类会优先将注意力、资源、精力和思考能力分配给边缘系统。你所学的推拉、赋格、框架等语言技巧，以及废物测试应答等方法，都发生在理性大脑区域，即**前额叶皮层**。这是一个非常耗能的区域。

你是否经历过学习或深入研究时比平时更容易饿？或者感到“头好疼，要长脑子了”？这正是因为理性思考极其耗能。人的本能是优先避免理性思考。

举一个政治不正确的例子：为什么看似更有钱、地位更高的人更理性，而经济状况较差、生活环境更艰难的人似乎更冲动，难以理性决策（例如在投资或金钱决定上）？为什么赌徒中很少见到那样的富豪？因为前者的理性区域被激活得更多，甚至可能发育得更大。理性思考的过程是**反本能**的。

以CAC为例。当你看到女生发来某条消息时，本能是什么？本能是尽快回复，生怕她不高兴。或者本能是凭直觉回消息。而什么是理性？理性是：我知道你很急，但你先别急。根据CAC分数分析当前情况，思考该说什么，然后再回复。这就是在**克制本能**。你是否觉得用这种方式聊天会感到累和疲劳？因为它确实更耗能。

这就像电脑有两块显卡：一块是基础集成显卡，另一块是用于游戏的独立GPU。运行简单任务时，它用集成显卡（边缘系统）。只有运行大型游戏时，才会动用独立GPU（理性系统），但也会导致过热。为了防止大脑“烧坏”，有时它会**关闭**你的思考。这就是我们在搭讪时说不出话、约会时脑子一片空白、面试时懵掉、甚至考试时紧张到什么都想不起来的原因。

所以，学了那么多却用不出来，第一个原因是上午提到的：那些技巧不是真实的你。第二个原因是，你必须先解决情绪层面的沟通，才能开始尝试理性层面的沟通。

## 沟通的核心：情绪技能 🎭

上一节我们分析了技巧失效的根源，本节中我们来看看什么是真正的沟通核心。我们日常的沟通几乎全是**情绪的沟通**。因此，我们真正需要提升的是**情绪技能**。

我们人有两类技能：第一类是**分析技能**，即更理性的思考；第二类是**情绪技能**。你是否听说唱或看过《中国新说唱》？知道Freestyle（即兴说唱）吗？例如，“为什么看着你穿着一身黑，因为等不会……”这就是情绪技能，它更多基于本能。如果你说不出来，说明你的社会过滤器或大脑屏障把它卡住了。在这种情况下，你想调动理性思维去即兴说唱下一句歌词，肯定做不到。只有在放松、允许自己“胡言乱语”、不苛求押韵时，你才可能反过来做到押韵。

那么，占我们日常沟通80%以上的情绪技能具体包含哪些功能呢？以下是情绪技能的关键组成部分：

**状态管理**
什么是状态管理？有时你会觉得自己心情不好、状态不佳，不在一个开放、放松的社交状态。这时，你能否让自己恢复到一个放松的中立状态，或者从糟糕状态恢复到中立？有的人能做到，有的人完全做不到，就需要借助物质滥用，例如吃助眠药、止痛药，在国外有人服用Adderall（专注达）来集中注意力，或者喝酒甚至使用其他物质。这是因为他们无法靠自己进行状态管理，必须依赖外界。

**自由表达**
我在与你们交流的过程中，大约百分之五六十到七十的情况都是在自我表达，即随便瞎聊。就像现在这样，如果我不允许自己“瞎扯”，就不可能连续讲几个小时。如果每一句台词都是背下来的，那我就不用讲课了，可以直接去新东方了。

**自信**
自信就是我们早上讨论的内容，让自己保持积极状态，同时拥有持续行动的动力。

**扎根**
这对你们来说可能是一个新概念。扎根是一个比喻。如果你是一棵植物，根扎得有多深？如果根不深，风一吹就倒了。就像小树苗存活率低，因为风一吹就倒。我们都知道，树根扎得越深，树越容易存活。一棵4米高的树，地下的根可能有7-8米。如果你种一棵2米高的小树苗，它的根可能只有30厘米，风一吹就倒了，根本活不了。

这与什么模式相似？就在你经受压力测试、废物测试，或感到有压力时。当有人不停对你唠叨、逼迫你，让你感到不舒服时，你能否保持扎根？因为你完全建立在自己的土壤上，所以不会剧烈晃动。风很大时你当然会晃动，但不会晃太多。然而，如果扎根不够深，风一来你就直接倒了、断了。所以，关于扎根，你可以理解为：在很多情况下，尤其是在面对废物测试时，你那种“破防”的感觉，就是扎根问题。

**松弛感**
抖音上很多人谈论松弛感。什么是松弛感？就是感觉自己是放松的还是紧张的。例如，我此刻是以放松的状态跟你们说话。当你在跟女孩子讲话、上台发言或任何场合时，松弛感是你能不紧张并开始调动分析技能的基础。如果你紧张到快要窒息、僵硬甚至抽搐过去，就不可能理性思考该说什么更漂亮的话。

**合作型框架（而非对抗型框架）**
以聊天为例。有的人很容易跟女孩子杠起来、吵起来。为什么他总跟人杠、总跟人吵？这是因为他看待问题的视角以及对他人感受的认知，总是觉得别人在攻击他，因此他需要站出来维护自己，甚至比别人更先一步去对抗。所以，你看到很多人的聊天中存在这种对抗机制。

那么，如何将对抗机制转变为合作型框架？这就像打太极拳，对方打来一拳，我将其“转”出去，既不伤他，也不伤我，这就是最好的方式。别人会觉得舒服、情商高。有些主持人很会说话，例如伊能静（注：原文为“一力尽”，根据上下文推断可能指伊能静或类似风格的主持人），她为了节目效果会怼人，让人破防。但情商高的人会绕开。例如，当伊能静提到周杰（饰演尔康的演员）并故意说些难听的话（如“亲嘴”事件）时，周杰可能选择绕开，这就是高情商，也是一种合作型框架。即，我不会觉得一切都是在攻击我，我可以像水一样将其“转”出去。

**感知他人情绪**
有些人情商低，是因为在沟通中无法理解对方是开心还是不满意，无法感知他人的情绪和心理状态。与这种人打交道，我们会觉得很累。有时你在社交中或大街上看到一些人，他们就是这样：对方明确表示反感，他自己却好像不觉得，还自我感觉良好（按北京话说“觉得自己是一个美差”），完全不去考虑别人的不舒服。这就是没有情商的表现。

我们常说讨好型人格可能为别人想得太多，但至少证明你具备感知他人情绪的能力，这是一件好事。极端情况下，有些人无法识别别人的表情，这可能涉及心理问题层面。但是，当你非常紧张、焦躁、焦虑时，你也无法准确识别他人的真实感受。就像我昨天和今天一直在跟你们分析的：当你接近一个人时，他的感觉是什么？而你当时无法思考和感受到，第一个是视角和经验问题，第二个就是你当时懵了，已经无法感受对方，只能感受自己。

**自娱自乐**
即你能否营造开心的氛围。在我讲课的过程中，我是否享受当下？还是觉得这是一个任务？我能否自己逗自己笑？我有时讲笑话，你们可能不接梗，但我觉得挺好玩的，比如刚才说唱的那个梗。所以，这就是自娱自乐。当我开心时，你们即使没笑，至少不会觉得不舒服，也知道我是放松的状态。因此，自娱自乐特别重要。在线下或约会时，如果你处于开心的状态，别人也会受到你情绪的感染，这是我们后面会讲到的内容。

## 分析技能是什么？🔍

上一节我们详细拆解了情绪技能，本节中我们来看看与之相对的分析技能。分析技能非常简单，主要包括以下两点：

**理性分析与推理**
即理性地分析、推理别人当时是怎么想的。例如，在文字聊天时，你可能会思考很久。或者领导说了一句模棱两可的话，你回家琢磨了半天，这都是分析技能，需要你的理性介入：领导大概是什么意思？他是什么性格的人？今天说这话是不是在暗示什么？

**语言艺术**
即我们讨论过的所有聊天技巧、沟通的具体技巧。例如，我昨天告诉你的技巧，在紧张情况下你能说得出来吗？我觉得不能完全呈现，因为对方的反应是随机的刺激。但可以缓解一定程度的紧张。没错，对方的反应是随机的，但你应该去感知到。感知到以后，你就会自动知道该怎样去回应。所以，结果就是：我告诉你们开场白怎么说，为你们编写好所谓的“罐装话术”（Canned Game），然后你去做，依然做不到。因为你去背，结果不对；你知道要随机应变，却又完全应对不了。正是因为情绪技能的问题没有解决，现在让我们来解决它。

## 核心概念总结与课程意义 🎯

让我们再解释一下“情绪技能”这个词。情绪技能大致对应 **`emotional skills`**。可以理解为我们今天早上提到的与潜意识相关的内容。例如，我现在的快乐、我现在的疲惫感、我想这样做，都是潜意识在支配，而不是我思考后的结果。如果我去想“现在到底是左腿在上还是右腿在上，这样会不会不好”，那我就已经忘了要说什么，自己的沟通流程也就断掉了。

所以，今天我们讲的所有内容有一个很重要的核心概念：你所有的课程都在教你去学分析技能、学沟通技巧，但用不出来。如果能用出来，我觉得CAC 1.0的内容已经够用了。这也是我最近半年突然意识到的一点。对我来说，我一直都学会并使用这些技巧了，因为我是一个比较擅长学习的人，善于总结规律，有点像工程师型思维方式：捕捉规律，然后应用规律。所以我一直都会用。但我一直有一个困惑：我看那些在社交或与异性交流中很享受的人，我一点都不享受。我觉得我一直在跟自己斗争，从未享受过。但现在我可以享受了。原因就是：对我来说，有了自娱自乐；对我来说，我不再主要使用分析技能；对我来说，沟通变成了以我自己为核心。这就是区别。

本节课中，我们一起学习了沟通技巧难以应用的根本原因，深入探讨了大脑的边缘系统与前额叶皮层，并清晰区分了情绪技能与分析技能。我们详细拆解了情绪技能的八个关键组成部分：状态管理、自由表达、自信、扎根、松弛感、合作型框架、感知他人情绪以及自娱自乐。理解并优先培养这些情绪技能，是让你所学沟通技巧得以流畅、自然应用的基础。记住，真正的沟通始于内在的放松与自我接纳。
