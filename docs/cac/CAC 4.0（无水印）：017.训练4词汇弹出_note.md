# CAC 4.0（无水印）：017. 训练4词汇弹出

## 概述
在本节课中，我们将学习如何进行词汇自由联想训练，这是一种帮助消除社会过滤器的有效方法。

## 自由联想训练规则
以下是进行自由联想训练的规则：

1. 放空自己的大脑，不要刻意思考。
2. 想到什么词就说什么词，不要限制自己的思维。
3. 不能就地取材，连续列举同一类型的词。
4. 不要使用不恰当的词汇。

## 训练步骤
1. 放空大脑，等待第一个词出现。
2. 当没有想法时，不要着急，等待下一个想法的出现。
3. 说出自己的想法，不要限制自己的思维。

## 移除社会过滤器
通过自由联想训练，我们可以消除自己的社会过滤器，从而更加自由地表达自己的想法。

## 总结
本节课中，我们一起学习了如何进行词汇自由联想训练，这是一种帮助消除社会过滤器的有效方法。通过这种训练，我们可以更加自由地表达自己的想法，提高沟通能力。
