# CAC 4.0 课程：017. 训练4 - 词汇弹出 🧠💬

在本节课中，我们将要学习一种名为“词汇弹出”的自由联想训练。这项训练的核心目标是帮助我们移除内心的“社会过滤器”，让思维和表达更加自由流畅，从而改善在社交场合（如搭讪、约会）中因自我审查而产生的卡壳现象。

## 训练规则与常见误区 ⚠️

上一节我们介绍了课程目标，本节中我们来看看训练的具体规则和需要避免的误区。

这项训练的规则很简单：放空大脑，等待脑中自然“弹出”的第一个词汇，然后直接说出来。

以下是两种必须避免的错误做法：

1.  **不能就地取材**：例如，看着周围环境说“房顶、灯、摄像机、地毯、鞋子、瓶子、手机”。这不是自由联想，只是对眼前事物的列举。
2.  **不能连续列举同类词**：例如，不停地说“葡萄、杏、李子、橙子、西瓜、菠萝”。这只是在罗列“水果”这一类别下的词汇，并非自由的联想。

这两种做法都违背了“等待词汇自然弹出”的核心原则，而是主动进行了逻辑归类或观察描述。

## 正确示范与核心要点 🎯

理解了规则后，我们通过一个示范来感受正确过程。关键在于**不评判、不强迫**，只是如实说出脑中浮现的词。

示范过程如下：
> 纸尿裤。飞轮海。强奸犯。地毯。包子。（等待）汽车。MMA。宇宙。水管子。地下通道。无限火力。Shengchao。雨伞。凿壁偷光。作爱。写到。杀人犯。窒息。Feiji。😔，宇智播做猪。大疆无人机。

在这个过程中，你可能会觉得某个词很傻、很冒昧或不雅（例如“作爱”、“宇智播做猪”）。**说出这些词恰恰是训练的关键**。如果你因为觉得不妥而停住不说，那就是你的“社会过滤器”在起作用。本训练的目的正是要移除这个过滤器。

## 训练目的：移除社会过滤器 🛡️➡️🚫

那么，移除这个“社会过滤器”有什么实际意义呢？它直接关系到我们社交的流畅度。

你与好朋友聊天通常不会卡壳，因为你觉得“说什么都可以”。但在搭讪或约会时，你可能会想：“这个话题不合适”、“这句话不该说”。例如，一个略显粗俗但真实的生活趣事，你可能只愿与哥们分享，而对约会对象却难以启齿。

**核心公式：社交流畅度 ≈ 1 / 社会过滤器强度**

即使说出一些看似“傻”或“不雅”的内容，也远好过因自我审查而完全僵住、无话可说。移除过滤器能让你更自由地开启和延续对话。

## 实践练习指南 🏋️

最后，我们来一起进行实践练习。你可以选择睁眼或闭眼进行，如果闭眼有助于放松和排除干扰，可以先闭眼练习一段时间再睁开。

我们将进行大约1到2分钟的练习。请务必遵守核心要求：

*   **不列举**：不要主动列举眼前物品或同类词汇。
*   **不强迫**：无需逼迫自己思考下一个词，只需**等待单词自动弹出**。
*   **说出来**：将脑中弹出的词直接说出（或默念）。

记住这个**核心代码**所描述的状态：
```plaintext
while (training) {
    word = wait_for_pop_up(); // 等待词汇弹出
    say(word); // 说出词汇，不经过 filter()
}
```

**本节课总结**：我们一起学习了“词汇弹出”训练。我们明确了其规则是等待词汇自然浮现并说出，避免了两大误区（就地取材和列举同类词）。我们理解了训练的核心目的是移除“社会过滤器”，并通过示范看到了不评判、不强迫的关键心态。最后，我们掌握了实践练习的方法，即放松等待、如实表达。坚持此训练，将有助于你在社交中表达更自由、更流畅。
