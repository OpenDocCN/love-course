# 课程名称：CAC 🧮

在本节课中，我们将要学习CAC（Cumulative Average Cost，累计平均成本）的概念、计算方法及其应用场景。累计平均成本是一种用于计算库存商品单位成本的方法，尤其在持续采购且价格波动的情况下非常有用。

## 什么是CAC？📊

累计平均成本法是一种库存计价方法。它通过将一段时间内所有采购的总成本除以总采购数量，来计算库存商品的单位平均成本。这种方法的核心在于“累计”，意味着每次新的采购都会与现有库存合并，重新计算一个新的平均成本。

其核心计算公式为：
**累计平均成本 = (期初库存成本 + 本期采购总成本) / (期初库存数量 + 本期采购总数量)**

上一节我们介绍了CAC的基本定义，本节中我们来看看它的具体计算过程。

## 如何计算CAC？🧮

计算CAC需要遵循清晰的步骤。以下是计算累计平均成本的详细流程：

1.  **确定期初库存**：记录计算周期开始时的库存商品数量和总成本。
2.  **汇总本期采购**：记录在本周期内所有批次采购的商品数量和对应的成本。
3.  **计算总成本与总量**：将期初库存成本与所有本期采购成本相加，得到累计总成本。同时，将期初库存数量与所有本期采购数量相加，得到累计总数量。
4.  **应用公式**：使用上述公式，用累计总成本除以累计总数量，得出新的累计平均成本。

为了更直观地理解，我们来看一个简单的代码示例，模拟一次CAC计算：

```python
# 假设期初库存：100件，成本500元
initial_quantity = 100
initial_cost = 500.0

# 本期第一次采购：50件，单价6元
purchase1_quantity = 50
purchase1_unit_price = 6.0
purchase1_cost = purchase1_quantity * purchase1_unit_price

# 计算第一次采购后的累计平均成本
cumulative_quantity = initial_quantity + purchase1_quantity
cumulative_cost = initial_cost + purchase1_cost
cac_after_purchase1 = cumulative_cost / cumulative_quantity

print(f"第一次采购后累计平均成本为：{cac_after_purchase1:.2f} 元/件")
```

## CAC的应用与特点 🏷️

理解了计算方法后，我们需要了解这种方法在实务中的应用及其优缺点。累计平均成本法常见于永续盘存制下的存货管理。

以下是CAC方法的主要特点：

*   **平滑价格波动**：由于每次采购都重新计算平均成本，它能有效平滑因市场价格波动带来的成本剧烈变化。
*   **计算相对简便**：相较于先进先出（FIFO）或后进先出（LIFO）需要跟踪具体批次，CAC只需维护一个动态的平均成本值。
*   **反映平均成本趋势**：计算出的成本反映了企业在一段时期内获取存货的平均水平，便于进行整体的成本分析和定价决策。

然而，它也可能无法精确反映某一时刻库存的实际采购成本。

本节课中我们一起学习了累计平均成本法。我们首先明确了CAC是一种通过累计总成本与总量来计算动态平均单位成本的库存计价方法。接着，我们逐步拆解了其计算公式和计算步骤，并通过代码示例进行了演示。最后，我们探讨了该方法在平滑成本、简化计算方面的应用特点。掌握CAC有助于在库存管理和产品成本核算中做出更合理的决策。
