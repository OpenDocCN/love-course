# CAC 3.0：L7 买家心态 vs 卖家心态 🤔

## 概述

在本节课中，我们将探讨买家心态与卖家心态在情感交往中的应用，并通过实战聊天记录分析如何建立高位框架。

## 社交定位点

在两性交往中，男性是进攻者，女性是守门员。男性拥有无限繁衍的可能性，而女性则有限。因此，女性在择偶上更谨慎，更输不起。

## 高位框架与低位框架

**高位框架**：

*   提供价值，分享经历、情绪和生活片段。
*   掌控情绪，不被对方掌控。
*   高位狂假，有枯改情人的特质。

**低位框架**：

*   索取价值，不断提问，寻求对方注意力。
*   被对方掌控情绪。
*   卖家思维，不断促销自己。

## 买家心态与卖家心态

**买家心态**：

*   给出筛选标准，考虑犹豫。
*   压价，反对对方的装逼。
*   不轻易给认可。

**卖家心态**：

*   不断促销自己，展示自己空缺。
*   丧失底线，迎合对方。
*   炫耀，心虚的表现。
