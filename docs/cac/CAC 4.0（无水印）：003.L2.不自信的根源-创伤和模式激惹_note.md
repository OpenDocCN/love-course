# CAC 4.0（无水印）：003.L2.不自信的根源-创伤和模式激惹

## 概述

在本节课中，我们将探讨不自信的根源，特别是创伤和模式激惹对自信的影响。

## 教程内容

### 1. 教育因素

**原因**：教育环境对自信的影响。

**例子**：国外长大的孩子往往比国内长大的孩子更自信，因为他们有更多的机会展示自己。

**公式**：教育环境 → 自信程度

### 2. 创伤

**定义**：创伤是指非常重大的负面事件，如战争、绑架等。

**例子**：战争经历、家庭暴力等。

**公式**：创伤事件 → 情绪反应

### 3. 模式激惹

**定义**：模式激惹是指潜意识中的模式与当前情境相符，导致情绪反应。

**例子**：小时候的经历导致长大后对类似情境产生恐惧。

**公式**：潜意识模式 → 情绪反应

### 4. 潜意识

**定义**：潜意识是意识之下的部分，包括内隐记忆和模式。

**例子**：小时候的经历被潜意识记住，导致长大后对类似情境产生恐惧。

**公式**：潜意识 → 情绪反应

## 总结

本节课中，我们学习了不自信的根源，包括教育因素、创伤和模式激惹。这些因素共同作用于我们的潜意识，导致情绪反应，影响我们的自信。
