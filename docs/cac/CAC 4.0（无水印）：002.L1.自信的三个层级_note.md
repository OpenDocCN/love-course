# CAC 4.0（无水印）：002.L1.自信的三个层级 🌟

## 概述
在本节课中，我们将探讨自信的三个层级，帮助大家理解自信的真正含义，并学会如何建立真实的自信。

## 第一层级：拥有什么而自信（Having）

以下是拥有什么而自信的例子：

- 穿着增高的鞋让自己看起来更高。
- 做了一个新发型，看起来很帅。
- 穿着限量版球鞋。
- 坐头等舱出行。
- 有比其他人更漂亮的女朋友。
- 付得起其他人付不起的旅行费用。
- 拥有某种结果或身份。
- 拥有牛逼的工作。

**公式**：自信 = 拥有（物质/身份/关系等）

**原因**：
1. 从小被灌输的观念。
2. 自卑的人寻找借口。

## 第二层级：做到什么而自信（Doing）

以下是做到什么而自信的例子：

- 使用推拉技巧。
- 使用复合技巧。
- 筛选伴侣。
- 使用高价值的语言体系。
- 获得尊重和好态度。
- 保持清醒和良好的睡眠。
- 操作得当，结果良好。

**公式**：自信 = 能做到（技能/技巧/行为等）

**问题**：
1. 力量来源于外界，不稳定。
2. 过于依赖状态和运气。
3. 过于依赖技巧，容易感到疲劳。

## 第三层级：真实自信（Being）

以下是真实自信的例子：

- 认为自己的存在本身就是自信。
- 认为自己有权有力量。
- 不依赖外界因素。
- 获得稳定的状态。
- 不感到疲劳。

**公式**：自信 = 我有权有力量

**核心概念**：
- 解决自卑问题，从外界回归自身。
- 建立真实的自信，不依赖外界因素。

## 总结
本节课中，我们学习了自信的三个层级：拥有什么而自信、做到什么而自信和真实自信。通过理解这三个层级，我们可以更好地认识自信的真正含义，并学会如何建立真实的自信。
