# 🌟 CAC 2.0 教程：从把妹到亲密关系 🌟

## 概述

在本节课中，我们将学习如何从短期框架转变为长期框架，建立并维护一段健康的亲密关系。

## 1. 长期框架 vs 短期框架

### 1.1 长期框架

长期框架关注的是建立一段稳定、健康的亲密关系。它强调相互理解、尊重和信任。

**公式**：长期框架 = 相互理解 + 尊重 + 信任

### 1.2 短期框架

短期框架关注的是快速建立吸引力和亲密感。它更注重外在表现和短期效果。

**公式**：短期框架 = 吸引力 + 亲密感

## 2. 社交碾压心态

社交碾压心态是指在与他人交往时，试图通过贬低对方来提升自己的地位。

**代码**：社交碾压心态 = 贬低对方 + 提升自己

## 3. 叙事高阶

叙事高阶是指通过讲述故事来展示自己的价值观、人生观和世界观。

**代码**：叙事高阶 = 故事 + 价值观 + 人生观 + 世界观

## 4. 两性压力

两性压力是指男性和女性在交往过程中所面临的压力。

**公式**：两性压力 = 男性压力 + 女性压力

## 5. 赋格和反向赋格

赋格是指通过改变话题或角度来引导对话。

**代码**：赋格 = 改变话题 + 引导对话

反向赋格是指通过提问来引导对方思考。

**代码**：反向赋格 = 提问 + 引导思考

## 6. 竞速升级

竞速升级是指在交往过程中，通过不断升级关系来保持新鲜感。

**代码**：竞速升级 = 不断升级关系 + 保持新鲜感

## 7. 依恋：黏人 vs 回避

依恋是指个体在亲密关系中的依赖程度。

**公式**：依恋 = 黏人 + 回避

## 8. 关系里的话外音

关系里的话外音是指在交往过程中，通过非言语行为传达的信息。

**代码**：关系里的话外音 = 非言语行为 + 信息

## 9. 关系的动态视角

关系的动态视角是指从发展的角度看待亲密关系。

**代码**：关系的动态视角 = 发展 + 视角

## 总结

本节课中，我们一起学习了如何从短期框架转变为长期框架，建立并维护一段健康的亲密关系。希望这些知识能帮助你更好地理解亲密关系，并在现实生活中取得成功。
