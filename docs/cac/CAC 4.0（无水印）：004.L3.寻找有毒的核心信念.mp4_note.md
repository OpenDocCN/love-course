# CAC 4.0（无水印）：004.L3.寻找有毒的核心信念.mp4 教程

## 概述

在本节课中，我们将学习如何识别和应对有毒的核心信念。这些信念可能源于我们的早期经历，并影响我们的行为和情绪。

## 核心概念

**核心信念**：在为人处事的过程中，一些被视为公理的、无法被质疑的信念。

**公式**：核心信念 = 早期经历 + 情绪反应

## 核心信念的来源

以下是核心信念的常见来源：

- **早期经历**：家庭环境、教育方式、社会文化等。
- **情绪反应**：痛苦、恐惧、愤怒等。

## 有毒的核心信念

以下是一些常见的有毒核心信念：

- **做事不能一次性做好的话，那我就是傻逼，或者说就会被抛弃，就会有灾难发生**。
- **做事情只有很快做好才才行。如果你做的很慢，就算你未来能做好做的很慢，那也是傻逼，没有意义**。
- **别人认为我牛逼，别人给我这个credit，给我这个认可的时候，我才牛逼。而且就算我不牛逼，只要别人认可我，我也牛逼。而就算我自己觉得自己还行，如果别人不认可我，那他妈的我就是SB，我就是垃圾**。
- **人如果不能够落落大方的交流，如果不能擅长社交，如果不是擅长泡妞，如果不是很有魅力的一个人，那就垃圾就没**。
- **如果我让别人不开心了，我就是十恶不赦的罪人。我就开千刀万剐。我就会被这个社会所排斥**。
- **如果我不牺牲自己的话，就没有人会喜欢我，没有人会接受我。如果我不牺牲自己，那么我就会被排斥掉**。
- **好人一定会有好报，所以我对你好，你就一定会在某一个时刻，你会喜欢上我，你会爱上我，或者说啊我对你们这么好，你们在将来会知道回报我作为这个公司里的一员，或者你们会知道是吧，我为你们付出这么多**。
- **如果我穿的很丑，如果我不够高，如果不够帅，我就是怎么样的那就会。有不好的事情发生**。

## 如何寻找有毒的核心信念

1. **回顾早期经历**：思考哪些经历让你感到痛苦、恐惧、愤怒等。
2. **识别情绪反应**：分析这些情绪反应背后的信念。
3. **反思行为模式**：观察这些信念如何影响你的行为。

## 总结

本节课中，我们学习了如何识别和应对有毒的核心信念。通过回顾早期经历、识别情绪反应和反思行为模式，我们可以逐步改变这些信念，从而改善我们的情绪和行为。
