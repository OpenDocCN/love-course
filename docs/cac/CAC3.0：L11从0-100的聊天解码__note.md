## 第一阶段：无感 (0-19分)

上一节我们介绍了态度分段的整体框架，本节中我们来看看无感阶段的具体操作。

女生处于这个阶段，通常是因为没有看到你身上的价值，或者担心你是个坏人。因此，她对你不产生任何感受、情绪或感觉，只将你视为普通朋友，你的存在与否对她没有影响。针对这两个原因，我们需要采取以下操作。

以下是该阶段需要完成的两项核心操作：

1.  **展示价值**
    *   展示价值并非炫耀或装腔作势，而是通过分享你的趣事、经历和想法，向对方呈现你的价值。
    *   更直白地说，就是展示你的生活片段和性格片段。当女生看到你的生活片段，她会了解你大致是怎样的人；看到你的性格片段，她会知道你的性格特点，并对与你相处的预期有所判断。
    *   因此，你需要进行展示，让女生对与你相处这件事建立初步的预期。

2.  **表达共情**
    *   共情是指你能体会对方的感受，替对方思考，并说出她内心的顾虑。
    *   例如，如果你在搭讪时认识一个女生，可以说：“有点突然，不好意思，不是想吓到你。” 这就是一种共情，表达了理解她可能因突然接触而产生的担忧。
    *   再比如，如果女生在聊天中感到有压力，你可以说：“不好意思，我不是很擅长聊天，如果让你感到尴尬，我很抱歉。”
    *   当你把对方可能感到的不适主动提出来时，女生会感觉到被理解和体谅，从而消除不适感。反之，如果你不提，女生可能会认为你“做贼心虚”，加深她的不安全感。
    *   因此，第二项操作的本质是告诉她你不是坏人，为她提供安全感和舒适感。

## 第二阶段：好奇 (20-39分)

了解了无感阶段的操作后，我们接着来看好奇阶段。

在这个阶段，女生开始对你身上的某些小点产生兴趣，但还不是对你整个人感兴趣。此时，女生认为你能提供的东西有限。因此，我们需要提供更多的东西：**情绪波动**和**感觉**。

那么，如何调动情绪呢？我们后续课程会详细讲解，今天先总结几个要点：

以下是调动情绪的几种方法：

*   **表达你的筛选标准**：这能让你与其他没有原则的“舔狗”区分开来，表明你处于高位。
*   **尝试开玩笑和适度冒犯**：敢于开女生的玩笑，说一些挑战她的话。就像脱口秀是“冒犯的艺术”一样，适度的、有趣的冒犯能挑动心弦，产生幽默感。
*   **进行“赋格”**：当你展示了筛选标准和情绪价值后，女生如果觉得你很有价值，就会慢慢想要迎合你，以获得跟你接触的资格。“赋格”就是赋予她这种资格。关于“赋格”，我们后面会专门讲解。
*   **树立男女相处的氛围**：这是很多兄弟会忽略的一点。你可能很擅长聊天，和女生相处得很好，但如果没有提前植入男女相处的氛围，她可能只把你当成一个有趣的朋友。当你突然表白、拉手或亲吻时，会吓到她。你需要从最开始就慢慢渗透这种氛围，让她逐渐习惯以男女的方式与你相处。

这个阶段是大约70%的兄弟容易卡住的阶段。如果你处理得好，很快就会进入下一个阶段。

处理好奇阶段后，部分人会顺利进入兴趣及后续阶段。

这个阶段的核心操作是：

1.  **强化男女氛围**

记住：永远不要只听女人说什么，而要看她做什么。如果她口头承诺是你的女朋友，但现实中连牵手都不愿意，那显然关系并未到位。因此，这个阶段最重要的是强化男女之间的感觉。
