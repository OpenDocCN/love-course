# CAC 4.0（无水印）：016. 训练3 话题延续采访

## 概述

在本节课中，我们将学习如何进行话题延续采访，这是一种通过即兴发挥和拓展技巧来引导对话的方法。

## 采访准备

以下是采访准备的相关步骤：

1. **确定采访对象**：选择一个愿意参与话题延续采访的对象。
2. **准备采访问题**：准备一系列开放性问题，鼓励对方进行拓展和即兴发挥。
3. **设定规则**：告知对方不允许否定你的问题或提示，只能以“是”作为回答，并在此基础上进行拓展。

## 采访过程

以下是采访过程中的关键点：

1. **引导对话**：通过提问引导对方进行拓展和即兴发挥。
2. **胡扯八道**：在对话过程中，你可以故意提出一些看似不合理的问题，以激发对方的思维。
3. **目标明确**：作为采访者，你的目标是引导对方进行拓展，而不是让对方陷入困境。

## 训练目的

以下是进行话题延续采访训练的目的：

1. **拓展思维**：通过即兴发挥和拓展技巧，培养你的思维拓展能力。
2. **突破逻辑限制**：学会在对话中胡扯八道，避免被逻辑所限制。
3. **提高沟通能力**：通过引导对方进行拓展，提高你的沟通能力。

## 实例分析

以下是一个话题延续采访的实例：

**采访者**：我想拿一根笔向记者。

**被采访者**：是，那你想用这支笔做什么呢？

**采访者**：为什么明明你是个红绿色盲，你还要发明一个红色的瓶子？

**被采访者**：是，因为红色和绿色在我眼里都是平等的，我不能假定它是一个红色的瓶子。

## 总结

本节课中，我们一起学习了如何进行话题延续采访。通过即兴发挥和拓展技巧，我们可以提高沟通能力，拓展思维，并突破逻辑限制。希望你在今后的实践中能够运用所学，提升自己的沟通技巧。
