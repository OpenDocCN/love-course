# CAC 3.0：L21 潜沟通

## 概述
在本节课中，我们将探讨潜沟通在约会和社交中的重要性，以及如何通过眼神、声调、肢体语言和面部表情等非语言方式来传递信息。

## 眼神接触
**核心概念**：眼神接触可以展示你的社会地位和自信程度。

- **眼神对视**：在自然演化中，眼神对视被视为一种挑衅，地位较低的人会避免眼神对视。
- **创造张力**：通过眼神对视，可以创造张力，展示你的自信和吸引力。
- **释放张力**：当对方感到不舒服时，可以通过眼神挪开释放张力。

## 声调
**核心概念**：声调可以传递你的情绪和态度。

- **错误的声调**：声调上扬（寻求和谐）和过度平淡（不健康）都是错误的。
- **正确的声调**：声音下降，语气坚定，可以制造压力感，吸引对方。

## 肢体语言
**核心概念**：肢体语言可以展示你的自信和舒适度。

- **错误的肢体语言**：怂肩、身体总是转向对方、重心倾斜、拘谨的坐姿、并排走路的距离过近或过远。
- **正确的肢体语言**：放松的坐姿、重心靠后的站姿、自然的手势。

## 面部表情
**核心概念**：面部表情可以传递你的情绪和态度。

- **错误的表情**：过度善笑或完全面无表情。
- **正确的表情**：微笑、做你自己、自然、自信。

## 总结
本节课中，我们学习了潜沟通在约会和社交中的重要性，以及如何通过眼神、声调、肢体语言和面部表情等非语言方式来传递信息。通过掌握这些技巧，你可以更好地展示自己的自信和吸引力，从而在约会和社交中取得成功。
