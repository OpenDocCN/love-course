# CAC 4.0（无水印）：014.训练1 张力对视_note

在本节课中，我们将学习一项核心的社交情绪技能训练：张力对视。这项训练旨在帮助你提升在社交互动中承受、管理并释放紧张感的能力，适用于与任何人沟通的场景。我们将通过一个简单的对视练习，让你亲身体验并理解人际距离与情绪张力之间的关系。

## 训练目标与概述 🎯

本节训练将覆盖你在社交环节中，无论是与异性还是同性沟通时所需用到的关键情绪技能。这个练习与你生活中的许多互动模式非常相似。

## 第一步：建立对视环境

现在，我们开始第一个训练。这个训练的目的是培养大家承受张力以及释放张力的能力。

以下是具体的准备步骤：

1.  **分组与站位**：请两人一组，面对面站立。
2.  **调整距离**：双方应站得足够近，近到如果其中一人稍微晃动肩膀，就会碰到对方的肩膀。这是一个属于社交亲密距离以内的状态。
3.  **姿势要求**：在这个距离下，两人开始对视。

## 第二步：对视的行为准则

在对视过程中，有一些关键点需要注意。以下是需要避免的行为清单：

*   **不要说话**：保持安静，不要交谈。
*   **不要防御**：不要将手背在身后或做出环抱自己等防御性动作。
*   **不要移开视线**：过程中可能会想笑或想把眼睛挪开，请尽量避免。
*   **不要眼神飘忽**：不要去看对方眼睛以外的其他地方，要看着对方的眼睛。
*   **不要放空或发呆**：眼神交流应是一种互动形式，而不是眼神空洞地“假装”在看，或者自己走神去想别的事情。

## 第三步：深化练习与调整

理解了基本规则后，我们可以进一步深化练习。你可以尝试将距离拉得更近一些。

当距离近到超过某个界限时（例如，近到可以伸手搂住对方手腕的位置），张力感会发生变化。我们将以小组形式同时进行这个练习，时长为两分钟。这两分钟内请保持安静。

准备开始时，请确保距离合适（如果太远请再靠近一些），双手自然放松地放在身体两侧，不做任何多余动作。

## 第四步：训练感受与复盘

训练结束后，你可能会产生一些新的感觉，例如感到特别友好、柔和。这通常意味着最初的紧张感已经消失，或者被你很好地接纳了。

如果你对比训练前后的感受，可能会发现变化：距离更近时，产生的张力可能会更强。这种张力感受与对方的身高、身材等因素有关，可能某一瞬间会让人更紧张。

这个训练的核心目的是让你亲身体验张力产生和变化的过程。一个非常重要的启示是：在与他人（例如女性）打交道时，我们常常站得太远，甚至站在侧面或背面。你以为你获得了对方的注意力，但实际上可能根本没有。关键是要站到足够近的距离——当然不是紧贴上去，而是一个手臂刚好能碰到对方的距离，这样就对了。

## 课程总结 📝

本节课中，我们一起学习了“张力对视”训练。通过这个练习，你亲身体验了社交距离如何影响互动中的情绪张力，并练习了在近距离对视中保持专注、开放和非防御状态的能力。记住，在社交中保持恰当的距离是有效沟通和传递自信的重要基础。
