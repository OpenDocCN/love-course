# CAC 4.0（无水印）：014.训练1 张力对视

## 概述
在本节课中，我们将学习如何通过张力对视训练提升社交中的情绪技能。

## 训练目标
- 培养承受和释放张力的能力
- 提升眼神交流技巧

## 训练步骤

### 1. 组队与站位
- 两人一组，站立
- 保持一定距离，确保肩膀可以相碰

### 2. 对视练习
- 双方对视，不说话，不分开手
- 保持眼神交流，避免笑或分心
- 避免防御性动作，如背手或交叉双臂

### 3. 调整距离
- 逐渐缩短距离，直至手臂可以相碰
- 保持两分钟，保持安静

### 4. 反思与对比
- 比较不同距离下的感受
- 分析张力变化与距离的关系

## 训练要点

以下是训练过程中需要注意的要点：

- **保持眼神交流**：用眼神进行交流，而非分散注意力
- **避免防御性动作**：保持自然姿势，避免背手或交叉双臂
- **调整距离**：逐渐缩短距离，感受张力变化

## 总结
本节课中，我们一起学习了张力对视训练，通过练习提升眼神交流和承受张力的能力。希望这次训练能帮助大家在社交场合中更加自信和自如。
