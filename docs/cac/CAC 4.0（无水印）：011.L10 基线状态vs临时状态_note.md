# CAC 4.0（无水印）：011.L10 基线状态vs临时状态 🎯

## 概述

在本节课中，我们将探讨基线状态和临时状态的概念，并了解它们在社交和自我认知中的重要性。

## 基线状态

**定义**：基线状态是指一个人在平静状态下的默认状态，即每天起床后的恢复状态。

**特征**：
- **数值满值**：类似于游戏角色，无论前一天如何，经过一夜的休息，所有数值都会恢复到满值。
- **内向倾向**：倾向于回避社交，如直播等消耗精力的事情。
- **能量消耗**：与社交互动消耗大量精力。

**公式**：基线状态 = 平静状态 + 默认恢复值

## 临时状态

**定义**：临时状态是指一个人在特定情境下展现出的状态，如社交场合或特定活动。

**特征**：
- **社交能力**：在社交场合中表现出较强的社交能力，能够唬住一些人。
- **人格面具**：在社交场合中，可能会戴上人格面具，表现出与平时不同的性格。
- **能量消耗**：与基线状态相比，临时状态需要更多的能量和精力。

**公式**：临时状态 = 基线状态 + 社交能力 + 人格面具

## 两种状态的体验

**基线状态**：
- 轻松
- 持久
- 开心
- 稳定
- 自尊和自我效能感提升

**临时状态**：
- 累
- 不持久
- 不享受
- 稳定性差
- 自我攻击和自我怀疑

## 总结

本节课中，我们一起学习了基线状态和临时状态的概念，并了解了它们在社交和自我认知中的重要性。了解自己的两种状态，有助于我们更好地应对社交场合，并保持良好的心态。
