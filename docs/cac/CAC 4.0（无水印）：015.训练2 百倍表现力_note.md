# CAC 4.0（无水印）：015.训练2 百倍表现力

## 概述

在本节课中，我们将学习如何提升表现力，使其更具吸引力。通过分享个人经历，我们将学习如何运用不同的表现力，从而突破自我限制，展现更丰富的个性。

## 一、什么是表现力？

**定义**：表现力是指个体在表达情感、思想或行为时所展现出的能力。

**公式**：表现力 = 情感表达 + 行为展现

## 二、如何提升表现力？

1. **分享个人经历**：选择一个简单的经历，如吃披萨、坐地铁等，用30秒的时间进行分享。
2. **运用不同的表现力**：尝试运用高亢、低沉、放松、变态、莫名其妙等不同的表现力，突破自我限制。
3. **突破表现力界限**：在安全的环境中，尝试表达自己平时不敢表达的情感，如悲伤、愤怒、性等。

## 三、案例分析

**案例**：一位学员分享了自己在公交车上演讲的经历。

**分析**：这位学员在开始时感到紧张，但随着时间的推移，他逐渐放开了自己，展现出了自信和激情。

## 四、练习方法

1. **选择一个场景**：选择一个你熟悉且容易引起共鸣的场景。
2. **设定目标**：设定一个具体的目标，如表达悲伤、愤怒、性等。
3. **尝试突破**：在安全的环境中，尝试表达你的目标情感。
4. **反馈与调整**：在群里分享你的练习过程和感受，与其他学员交流，并根据反馈进行调整。

## 五、总结

本节课中，我们学习了如何提升表现力，使其更具吸引力。通过分享个人经历，运用不同的表现力，并突破自我限制，我们可以展现更丰富的个性。希望你在今后的生活中，能够运用所学，提升自己的表现力，成为更自信、更精彩的自己。
