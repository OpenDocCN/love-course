## CC逻辑

### CC含义

- **C1：Check（检查判断）**：判断女生对你的态度。
- **C2：Action（操作）**：根据判断采取相应行动。
- **C3：Cycle（循环）**：循环操作，提升好感度。

### 套路与实战

- **套路**：指一套话术体系，但无法适用于所有人。
- **实战**：通过大量实战积累经验，了解不同文化背景下的社交差异。

## 女生好感度阶段

### 五个阶段

1. **无感**：对男生没有感觉。
2. **好奇**：对男生产生好奇。
3. **兴趣**：对男生产生兴趣。
4. **依恋**：对男生产生依恋。
