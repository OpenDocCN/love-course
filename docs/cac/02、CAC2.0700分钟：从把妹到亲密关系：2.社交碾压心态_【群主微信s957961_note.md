# 02、CAC2.0700分钟：从把妹到亲密关系：2.社交碾压心态

## 概述
在本节课中，我们将学习如何通过社交定位点来提升自己在社交中的地位，从而吸引女生并建立亲密关系。

## 社交定位点
社交定位点是指我们在社交互动中所处的位置，它决定了我们在互动中的行为和态度。

### 核心概念
**公式**： 社交定位点 = 价值提供者 - 价值索取者

### 重要性
- 男人是进攻者，女人是防御者。
- 男人是性发起者，女人是性选择者。
- 女生在情感中更输不起，因此更倾向于站在高位进行筛选。

## 提升社交定位点的技巧
以下是提升社交定位点的五个关键点：

### 1. 价值提供者而非索取者
- 不要只关注女生，也要展示自己的价值。
- 不要连续提问，避免采访模式。

### 2. 掌控情绪
- 不要因为女生的负面反馈而破防。
- 保持情绪稳定，不要被女生牵扯走。

### 3. 富裕心态
- 不要害怕犯错，敢于表达自己。
- 不要害怕失去，相信自己有很多选择。

### 4. 自娱自乐
- 放开自己，做自己喜欢的事情。
- 不要拘谨，展现真实的自己。

### 5. 买家思维
- 不要像卖家一样推销自己。
- 根据女生的表现决定是否“购买”。

## 买家思维与卖家思维
- **卖家思维**： 炫耀、解释说服、迎合。
- **买家思维**： 筛选、考虑、压价。

## 案例分析
通过分析聊天记录，我们可以看到如何运用这些技巧来提升社交定位点。

### 案例一
- 男生在聊天中不断提问，没有提供任何价值，属于价值索取者。
- 男生在女生拒绝后表现出不高兴的情绪，暴露了需求感。

### 案例二
- 男生在聊天中不断展示自己的价值，并保持情绪稳定，属于价值提供者。
- 男生在女生提出要求时，没有立即满足，而是进行了压价。

## 一致性
- 在女生拒绝前后保持一致。
- 在收尾前后保持一致。
- 在建立关系前后保持一致。

## 总结
通过提升社交定位点，我们可以更好地吸引女生，建立亲密关系。在本节课中，我们学习了如何成为一个价值提供者，掌控情绪，培养富裕心态，以及运用买家思维。希望这些技巧能够帮助你在社交中取得成功。
