- 经常不回消息。

- 情绪流露：表达开心、兴奋等。

**定义**：在这个阶段，女生对你产生了真正的兴趣，想要了解更多关于你的信息。

**判断标准**：

- 主动：主动发消息、找话题等。
- 解释自己：解释自己的行为、想法等。

### 5. 依恋阶段 (80-100)

**定义**：在这个阶段，女生对你产生了深厚的感情，愿意与你建立长期关系。

**原因**：
- 她对你产生了强烈的依赖感。
