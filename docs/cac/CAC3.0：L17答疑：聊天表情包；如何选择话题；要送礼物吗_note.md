# CAC 3.0：L17答疑：聊天表情包；如何选择话题；要送礼物吗

## 概述
在本节课中，我们将探讨如何使用聊天表情包、如何选择合适的话题以及是否应该送礼物。

## 聊天表情包
以下是关于聊天表情包的详细说明：

### 表情包类型
- **舒适感表情包**：温和的表情包，如开心的小胖妞，用于提供温馨感，减少给对方的压力。
- **压力或张力表情包**：不认可对方、挑衅或挑战对方，如坏笑的小女孩、拿刀的老太太。

### 使用时机
- **舒适感表情包**：女生不安全感或还不熟悉你时，偶尔使用调节气氛。
- **压力或张力表情包**：有趣、好玩、自娱自乐，不害怕冒犯对方。

## 如何选择话题
以下是关于选择话题的详细说明：

### 原则
- 将对方想象成你的女朋友，自然地聊天。
- 深入话题，树立男女氛围，筛选、调侃、复格。
- 聊第三方话题时，及时联想到自己和对方的内在性格。

### 话题选择
- **感情话题**：感情经历、感情观。
- **深刻话题**：深入话题，树立男女氛围。
- **第三方话题**：聊太多时，及时联想到自己和对方的内在性格。

### 聊天初期
- 70%聊自己。
- 不要陪女生聊闺蜜话题。

## 是否送礼物
以下是关于送礼物的详细说明：

### 礼物作用
- 代表真诚度，展示长期属性。
- 表达诚意，但不能弥补操作上的缺失。

### 礼物逻辑
- 避免用物质金钱直接碾压女生。
- 展示什么就吸引什么，避免制造交易身份。

### 误区
- 过度送礼物，磨灭内在力量。
- 将自己放在交易身份上。

## 总结
本节课中，我们学习了如何使用聊天表情包、如何选择合适的话题以及是否应该送礼物。希望这些内容能帮助你在与女生相处时更加得心应手。
