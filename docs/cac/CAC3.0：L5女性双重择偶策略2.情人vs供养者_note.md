# CAC 3.0：L5女性双重择偶策略2.情人vs供养者

大家好，欢迎来到CAC 3.0的第五节课。在上节课中，我们提出了一个问题：男人眼中怎样的女人是渣女？女人眼中怎样的男人是渣男？男女对“渣”有不同的看法和理解。

### 男人眼中的渣女

男人认为的渣女，简单来说就是付出多而回报少的对象。例如：

- **时间、精力、金钱的付出**：女生接受礼物，但不发生关系，吊着男人，口头承诺恋爱关系，但实质上没有进展。
- **财产占有**：结婚后，女方试图独占共同财产或男方个人财产。

### 女人眼中的渣男

女人认为的渣男通常是不付出或出轨的男人：

- **不付出**：不花钱，不关心，甚至骗钱。
- **出轨**：花心，家暴。

### 女性的双重择偶策略

女人也有两套择偶策略：

1. **获取优质基因**：选择基因强大的男人，提供情绪上的快感，安全感。
2. **掌控关系**：选择可以掌控的男人，长期照顾自己和后代。

### 情人属性与供养者属性

- **情人属性**：提供情绪上的快感，安全感，但可能不注重长期关系。
- **供养者属性**：照顾家人，顾家，但可能缺乏吸引力。

本节课中，我们学习了女性双重择偶策略，以及情人属性与供养者属性的区别。了解这些有助于我们更好地理解女性的择偶心理，从而在恋爱中取得成功。
