# CAC 4.0（无水印）：031.3.0 L5-让女生主动迎合你：筛选 赋格 🤧

## 概述

在本节课中，我们将学习如何通过筛选和赋格技巧让女生主动迎合你。

## 核心概念

**筛选**：在表达好感时，有所保留，表达自己的标准。

**赋格**：对符合筛选标准的女生进行正面的评价和鼓励。

## 筛选

### 筛选的重要性

- 让女生知道你有标准，有所保留，有所质疑。
- 让女生感觉到你是买家，是富裕心态，是高位框架。
- 保持你的权威感，保持你的高位框架。

### 筛选的实例

- 外表方面：喜欢短头发的女生、喜欢平胸的女生、喜欢男孩子系的女生等。
- 性格方面：不作、情绪稳定、愿意沟通、乐观等。
- 爱好方面：喜欢运动、喜欢读书、有爱心等。
- 生活习惯：自律、健身、读书等。
- 价值观：对家庭父母的观念等。

### 筛选的注意事项

- 不要使用歧视性的特征进行筛选。
- 不要将筛选标准表达得过于明显，以免让对方感到不舒服。

## 赋格

### 赋格的作用

- 对符合筛选标准的女生进行正面的评价和鼓励。
- 奖励女生，鼓励她继续表现出好的特质。
- 帮助女生意识到你有筛选标准，从而更加努力地满足你的要求。

### 赋格的实例

- 女生分享做饭照片：感觉你是会照顾人的类型。
- 女生救助小野猫：没发现你还有这么有爱心的一面。

### 赋格的方向

- 有利于恋爱的方向：将赋格引向恋爱话题，保持男女氛围。
- 短期/长期复合：根据你的意图选择合适的赋格方向。

## 失格

### 失格的作用

- 对不符合筛选标准的女生进行负面评价和批评。
- 制造障碍，让对方感到不舒服，从而推远关系。

### 失格的实例

- 发现女生不符合你的要求：我们当不了朋友，我们玩不了一块去。

## 总结

本节课中，我们学习了筛选和赋格技巧，通过这些技巧，你可以让女生主动迎合你，更加珍惜你的好，并尊重你。记住，筛选和赋格的意识必须要有，才能更好地运用这些技巧。
