# CAC 3.0：L10 为什么网上的聊天课都是错的

兄弟们，从这节课开始，我们正式进入聊天的实战环节。因为上一课已经教给大家如何判断女生在不同分数阶段下的态度，所以从后面的课程开始，我们会教给大家不同阶段对应的具体操作。

## 本节课重点

### 1. 空杯心态

在学习CAC聊天方法时，保持空杯心态非常重要。这意味着你需要放下之前学过的聊天思维和技巧，包括你自己的聊天方法。就像你手里已经有一杯咖啡，现在想学白酒，你需要先倒掉咖啡，才能更好地学习白酒。

### 2. 语气的重要性

在聊天中，语气比话题和内容更重要。女生通过你的语气、用词、态度和框架高低等细节，来判断你是一个什么样的人，以及是否被你吸引。

### 3. 聊天的目的

聊天的目的是邀约和筛选女生是否符合你的标准。当你发现女生的情绪较高时，可以尝试邀约或模糊邀约。当女生出现明显的性取指标时，再进行确定邀约。

### 4. 避免过度聊天

不要过度聊天，见好就邀约。在现实中，一周最多聊两次天，就应该进行邀约。

## 总结

本节课介绍了聊天的几个重要点，包括空杯心态、语气的重要性、聊天的目的和避免过度聊天。希望这些内容能帮助大家更好地进行聊天实战。
