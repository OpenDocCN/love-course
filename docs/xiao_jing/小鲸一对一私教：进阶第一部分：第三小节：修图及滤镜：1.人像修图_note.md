# 小鲸一对一私教：进阶第一部分：第三小节：修图及滤镜：1.人像修图 👨‍🎨

在本节课中，我们将要学习如何通过修图软件，对人像照片进行美化处理，使人物看起来更帅、更有型。我们将从理解“帅”的标准开始，逐步讲解使用美图秀秀等软件进行面部轮廓、身材比例调整的具体步骤，并介绍一些实用的“黑科技”功能。

## 理解“帅”的标准 🎯

上一节我们介绍了修图的目的，本节中我们来看看什么是“帅”的标准。修图的关键前提是知道什么样的脸型和五官是好看的。

我们可以参考公认的帅哥，例如吴亦凡。观察他的照片可以发现几个特点：
*   他的下巴是尖尖的。
*   他的下颚骨是向上倾斜的。
*   他的鼻子是挺挺的。
*   他的额头比较饱满。

在修自己的照片时，目标就是让自己的脸型和五官尽可能接近这些帅气的特征。需要重点关注的三个部位是：**鼻子、下巴和下颚骨**。此外，拍照时尽量采用四分之三侧脸的角度，避免完全正对镜头，这样面部会更立体，也便于后期修饰。

## 核心修图软件与步骤 📱

接下来，我将为大家分别演示使用什么软件，以及通过哪些步骤来实现我们想要的效果。

但凡涉及人物的照片，我们第一步都是先使用**美图秀秀**这款软件。其“人像美容”模块提供了丰富的工具。

以下是修图时最常用、必须掌握的几个核心功能：

### 1. 瘦脸瘦身工具

这个工具用于调整面部轮廓和身材。进入“瘦脸瘦身”功能后，可以滑动屏幕放大照片，便于精细操作。下方的工具条可以调整画笔圆圈的大小。

**使用技巧：**
*   建议选择稍大一点的画笔，避免局部调整导致轮廓畸形。
*   调整时，推动画笔的方向就是让该部位变瘦或变形的方向。
*   每调整完一个部位（如下巴、下颚骨、鼻子），可以点击右上角的“对比原图”按钮查看效果，确保比例协调。
*   调整面部时，如果带动了背景或其他部位（如脖子、手臂），需要一并微调，以保持整体自然。

### 2. 祛斑祛痘工具

这个工具用于去除脸上的痘痘、痘印、痣等瑕疵。选择“祛斑祛痘”功能，将工具圆圈完全覆盖住瑕疵点，松开手指即可消除。

**使用技巧：**
*   左上角有放大镜，可以辅助精准定位。
*   对于男生，无需将脸部处理得完全光滑无瑕，保留一些自然的痕迹（如胡渣）反而更有男人味。

### 3. 面部重塑工具

此功能主要适用于正脸或接近正脸的照片，可以系统性地调整五官。工具内包含脸型、眉毛、眼睛、鼻子、嘴唇等细分选项。

**使用建议：**
*   此工具适合进行微调，例如微调脸宽、额头高度。
*   对于追求精致效果，更推荐使用“瘦脸瘦身”工具进行DIY式的调整，自由度更高。
*   谨慎调整眼睛、眉毛等细节，不熟悉五官比例容易修得不像本人。

### 4. 祛皱工具

用于去除抬头纹、法令纹、鱼尾纹等。选择“祛皱”功能，将画笔大小调整到能覆盖皱纹的宽度，然后沿着皱纹的走向涂抹即可。

### 5. 增高塑形工具

此功能对于全身照尤其重要，可以拉长腿部，打造“大长腿”效果。选择“美腿”功能，用两根参考线框住腿部区域，然后滑动“增高”工具条即可。

**注意事项：**
*   拉长幅度要适中，避免比例失调。
*   此功能可能导致背景（如地面瓷砖、建筑物）变形。

## 解决全身照变形的“黑科技”软件 🦵

上一节提到美图秀秀的“增高”功能可能导致背景变形。为了解决这个问题，我们可以使用另一款软件：**Spring**。

这款软件可以让人物整体变瘦或拉高，同时保持背景相对自然，不易产生明显畸变。以下是主要功能：

*   **整体变瘦**：使用“可乐瓶”图标功能，将圆圈对准头部，调整两侧约束线包裹身体，然后滑动工具条即可整体瘦身。
*   **整体拉高**：使用“弹簧”图标功能，同样方法操作，可以实现整体增高，尤其适合构图带角度的照片，能有效减少背景变形。

## 高级效果营造技巧 🚗

除了基础修饰，还有一些“黑科技”可以提升照片的逼格。

### 1. 营造动态模糊效果

想拍出背景车辆移动模糊的感觉，但拍照技术有限？可以使用**美颜相机**的“高级柔焦”功能。
1.  在“图片精修”中打开照片，进入“编辑”->“高级柔焦”。
2.  选择“手动”模式。
3.  使用“涂抹”工具，仔细涂抹需要模糊的车辆区域。
4.  如有误涂，使用“擦除”工具修正。
5.  点击确认，车辆背景就会变得模糊，营造动态感。

### 2. “制造”肌肉线条

对于想展示更好身材的兄弟，可以使用**有型**这款软件来增强肌肉线条。
1.  打开软件，导入照片。
2.  选择“胸肌”或“腹肌”效果（免费款式足够使用）。
3.  将肌肉模板对准身体相应部位。
4.  点击“魔法棒”图标应用效果，并可调整强度、颜色使其更自然。
5.  保存图片（后期可通过裁剪去除软件水印）。
