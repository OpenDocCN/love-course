## 什么是“打压”？🤔

上一节我们介绍了课程主题，本节中我们来看看“打压”的定义。

打压是一种在恋爱中常见的互动技巧。打压的目的是为了和女生推进关系。从本质上讲，它是一种以退为进的技巧。如果你打压以后双方的关系恶化了，那很明显这就不是打压，而是侮辱了。

## 核心原理：模仿高价值行为 💎

打压其实是一种对高价值男生行为的模仿。因为高价值的男生，不会因为女生长得好看，就接受她的一些无理要求。在美女出现一些不恰当的言论或行为时，他们会给予否定、质疑或批评。在高价值男生的眼中，美女并没有区别于其他人的特权。但是在“屌丝”的世界里，美女因为美就可以获得一切特权。

**公式：有效打压 = 对不当行为的否定 + 无需求感的态度**

## 正误案例对比 📊

以下是两个生活中的例子，用以区分什么是正确的“打压”，什么是错误的“侮辱”。

**正确案例（打压）：**
> 比如我们去买东西，你去买东西时会挑这些东西的毛病（“这个衣服怎么还有线头呀”、“这个面料看起来很容易起皱呀”），但店家不会生气，反而会解释原因，因为他知道其实你是想买这个东西的。

**错误案例（侮辱）：**

以下是关于使用打压技巧时必须注意的几个核心要点。

3.  **自身状态**：如果你自己已经流露出强烈的需求感或“输球感”，就不要再去打压，否则会显得别扭。
4.  **程度把控**：打压的程度要视对方性格和当时情况而定，切忌过度。
