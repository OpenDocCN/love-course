## 什么是未来展望？ 🤔

上一节我们介绍了课程主题，本节中我们来看看“未来展望”的具体定义。

如果说聊天的三个阶段是三个相互碰撞的部分，那么未来展望就是当中的润滑剂。因为它可以在任何一个阶段使用。

**未来展望**的核心是：你在女生面前，为她描绘一个虚拟的画面。这个画面的主角必须是你和她，只有你们两个人。这样做的目的是让女生提前幻想你们见面后，或未来生活在一起的场景，想象你们会说什么话、做什么事。从而让你们在实际见面后，能够轻易地升高关系，却不会显得唐突。

## 未来展望的三种运用模式 🛠️

了解了基本概念后，我们来看看如何具体运用。未来展望主要分为三种运用模式。

这一种模式趋向于发展长期关系。在这个过程中，我们描绘未来和女生在一起的美好画面，让女生的内心充满安全感，感觉未来和你在一起会很浪漫、很美好。以下是具体方法：

第三种模式是让女人坚定这段长期关系，核心在于你对事业心的展望和对未来的规划。这涉及我们聊天中的第三个阶段——勾心阶段。这一点威力巨大，甚至能让价值比你高的女生死心塌地地跟着你。因此，出于安全考虑，这部分更深入的内容我们放在线下面授课程中详细讲解。如果你有兴趣，特别是想追求高分女生但自身价值暂时不够，可以联系我们了解详情。
