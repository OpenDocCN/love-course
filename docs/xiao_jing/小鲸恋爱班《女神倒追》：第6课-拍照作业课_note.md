## 核心点评维度

一张照片的好坏，可以从以下几个核心维度进行判断。以下是这些维度的具体说明：

1.  **形象打理**：这是拍照的基础。包括发型、眉毛、面部清洁和整体精神面貌。如果形象没有打理好，其他方面再出色也无济于事。
    *   **不合格示例**：头发凌乱、眉毛未修、面部油光。
    *   **合格示例**：发型整洁有型，面部干净清爽。

2.  **服装搭配**：服装传递着你的品味和价值感。避免过于普通或邋遢的搭配，选择能体现个人风格和质感的衣物。
    *   **不合格示例**：普通T恤搭配休闲裤，缺乏亮点。
    *   **合格示例**：有设计感的潮牌服饰，搭配协调。

3.  **背景与取景**：背景用于衬托人物，而非主角。应选择简洁、有质感或有设计感的场景，避免杂乱、低档或过于“游客照”式的背景。
    *   **不合格示例**：颜色杂乱的书架、纯白墙壁、喧宾夺主的风景。
    *   **合格示例**：有格调的咖啡厅、设计感强的建筑局部、开阔的海边。

4.  **姿势与表情**：姿势要自然、放松，避免僵硬和做作。表情管理同样重要，要找到自己最上镜的角度和神态。
    *   **不合格示例**：身体僵硬、表情尴尬、刻意摆拍。
    *   **合格示例**：姿态放松、表情自然、与环境有互动。

5.  **构图**：人物应是画面的主体。一个基础的合格构图是：人物位于画面中央区域，且上下左右留有适当的空间。
    *   **核心公式**：`合格构图 ≈ 人物居中 + 四周留白均衡`
    *   **常见错误**：人物太小、头顶或脚下留白过多、画面被无关元素填满。

6.  **光线与焦点**：光线决定了照片的质感和氛围。拍摄时需要通过点击屏幕调整焦点，以确保人物面部曝光正确，背景不过曝或过暗。
    *   **操作提示**：拍照时点击屏幕上的小圆圈，将其对准人脸或画面较亮处，以调整整体曝光。
    *   **不合格示例**：人脸过暗、背景过曝（一片死白）。

7.  **颜色与后期**：照片的整体色调影响着高级感。即使原图尚可，低级的颜色也会让照片大打折扣。这需要通过后期修图来优化。

## 作业实例分析

现在，让我们结合上述维度，具体分析一些学员作业。通过对比，你可以更清晰地理解好照片的标准。

### 不合格案例解析

以下是几个典型的不合格案例及其主要问题：

*   **案例A（书房照）**：
    *   **形象**：头发未打理，显得凌乱。
    *   **背景**：书架颜色搭配显得低档。
    *   **服装**：搭配普通，缺乏价值感。
    *   **结论**：基础的形象和背景出现问题，导致整张照片质感很差。

*   **案例B（自拍照）**：
    *   **形象**：眉毛未修，发型随意。
    *   **构图**：距离镜头太近，显得脸大且不自然。
    *   **背景**：杂乱无章。
    *   **结论**：自拍更需注重细节，此照片在多个维度上均不合格。

*   **案例C（游客式站姿照）**：
    *   **构图**：人物太小，背景喧宾夺主。
    *   **姿势**：典型的僵硬游客照姿势。
    *   **角度**：正面拍摄，暴露了面部缺陷。
    *   **结论**：未能突出人物主体，且姿势和角度选择失误。

### 合格案例解析

相比之下，合格的案例在多个维度上都处理得较好：

*   **案例D（时尚街拍）**：
    *   **形象**：发型、眉毛精心打理，佩戴墨镜。
    *   **服装**：穿搭时尚有层次，体现品味。
    *   **背景**：选择有设计感的建筑或街道，质感强。
    *   **姿势**：姿态放松自然，与环境有互动感。
    *   **构图**：人物位于画面黄金位置，四周留白恰当。
    *   **结论**：各方面均衡，整体感觉高级、有型。

*   **案例E（与宠物合影）**：
    *   **情感互动**：与宠物互动，画面富有情感和活力。
    *   **形象**：形象打理干净。
    *   **背景**：背景简洁，不杂乱。
    *   **结论**：通过引入情感元素（宠物），即使背景简单，也能让照片生动、有吸引力。

## 实用技巧与总结

在本节课的最后，我们总结几个能立即提升拍照水平的实用技巧：

1.  **视线方向留白法则**：当人物看向某一侧时，应在其视线方向预留更多空间。这会使构图更大气、舒适。
    *   **代码式描述**：`if (人物视线方向 == 左侧) { 左侧留白 > 右侧留白 }`

2.  **规避面部缺陷**：如果知道自己正脸或某个角度不好看，拍摄时应有意识地侧脸、低头或利用道具（如杯子、手机）进行局部遮挡。

3.  **多角度尝试**：不要站在一个位置只拍一张。应前后左右、高低不同角度多拍几张，然后从中选出效果最好的。

4.  **检查清单**：每次拍完照，立即用我们讲的**七个维度**（形象、服装、背景、姿势、构图、光线、颜色）快速检查一遍，发现问题当场重拍。
