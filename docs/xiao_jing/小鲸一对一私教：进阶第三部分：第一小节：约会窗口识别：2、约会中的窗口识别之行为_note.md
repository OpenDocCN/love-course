# 小鲸一对一私教：进阶第三部分 第一小节：约会窗口识别（2）行为篇 👀💃

在本节课中，我们将学习如何在现场单人约会中，通过观察女生的行为和举止来识别她对你的兴趣窗口。上一节我们介绍了女生“说了什么”代表有窗口，本节我们将重点分析女生“做了什么”。

女生在行为举止上表现出窗口，其说服力远大于言语。因为言语可能存在“打嘴炮”的成分，而行为是更真实的第一表现。我们将优先识别行为，再结合说辞进行综合判断。

以下是女生在约会中表现出真实兴趣的十种关键行为。

眼睛是心灵的窗户。眼神接触是判断窗口的重要指标。

**公式**：`窗口大小 ∝ 朝向你的身体部位数量`

女生会故意提及自己优秀、靠谱的地方（类似于男生的DHV - 展示高价值行为）。

这通常是她希望在你心中建立良好形象的表现。判断时需结合基本社交常识，分辨是正常的分享还是过度的炫耀。
