# 小鲸一对一私教：进阶第一部分：第三小节：修图及滤镜：2.照片调色及滤镜处理_note

在本节课中，我们将学习如何使用专业的调色和滤镜软件，将一张普通的照片提升质感，使其看起来更加高级和舒适。我们将重点介绍两款核心软件：Snapseed和VSCO，并讲解基础的调色原理与滤镜应用技巧。

上一节我们介绍了如何使用美图秀秀等软件进行人物轮廓和五官的修饰。本节中，我们来看看如何通过调整颜色和添加滤镜，来提升照片的整体“逼格”。

## 核心软件介绍

市面上有许多调色软件，例如Snapseed、VSCO、Mix、Instagram等。本节课我们将主要讲解**Snapseed**（专业调色）和**VSCO**（丰富滤镜）这两款软件。掌握这两款，你的照片质感将得到显著提升。

## 第一步：使用Snapseed进行基础调色

我们首先打开Snapseed软件（图标为一片小树叶）。导入一张已经用美图秀秀处理好的照片。

一张好照片通常具备干净、明亮的特点。原始相机拍出的照片往往有些灰暗。我们的调色目标就是让它变得明亮、干净，避免“脏兮兮”的感觉。

以下是调色的核心步骤：

1.  **打开工具菜单**：点击底部的“工具”按钮。
2.  **选择“调整图片”**：这是最基础的全局调整工具。
3.  **开始逐项调整**：在打开的界面中，上下滑动屏幕即可选择不同的调整参数。以下是各项参数的作用：
    *   **亮度**：让整张照片变得更亮或更暗。通常我们向右滑动增加亮度。
    *   **对比度**：调整照片明暗部分的差异。向左滑动降低对比度，照片会偏向“日系”或“性冷淡”风格，带点灰调；向右滑动增加对比度，明暗对比会更强烈。
    *   **饱和度**：控制颜色的鲜艳程度。如果人脸过红或颜色过于鲜艳，可以向左滑动降低饱和度。拍摄食物时，可以适当增加饱和度以增强食欲感。
    *   **氛围**：这是一个综合调整参数，可以同时影响高光和阴影。向左滑动会营造灰调风格，向右滑动则让画面轮廓更清晰。
    *   **高光**：仅调整照片中明亮的部分（如天空、皮肤高光）。
    *   **阴影**：仅调整照片中暗部的细节。向右滑动可以让暗部变亮，细节更清晰。
    *   **暖色调**：调整照片整体的色温，偏蓝（冷）或偏黄（暖）。

**调色要点**：调色没有固定数值，它取决于照片本身的状况和你想要的风格（如港风、工业风、小清新）。多观察优秀照片的色调感觉，并在此工具条上进行尝试，是学习调色的最佳途径。

调整完毕后，点击右下角的“√”确认。你可以长按屏幕对比调整前后的效果。

## 第二步：使用Snapseed进行细节增强

基础调色完成后，我们可以进一步强化细节。

1.  再次点击“工具”。
2.  选择“突出细节”。
3.  这里有两个参数：
    *   **结构**：增加它会让画面中所有物体的轮廓变得更清晰、更有质感。适合风景或全身照。
    *   **锐化**：效果比“结构”更柔和，适合用于半身照或人脸特写，让皮肤细节更清晰。

根据照片类型选择合适的参数进行调整，然后点击“√”确认。此时照片已经比原图舒服很多，可以直接使用了。

## 第三步：使用VSCO添加并微调滤镜

为了让照片风格更统一或更具艺术感，我们推荐使用VSCO添加滤镜。

首先需要在淘宝花费几元钱购买VSCO的全套滤镜并完成安装。打开VSCO，导入刚才在Snapseed中调好色的照片。

以下是添加滤镜的步骤：

1.  点击照片进入编辑界面。
2.  点击底部的滤镜图标，浏览并选择一个你喜欢的滤镜风格。
3.  **关键步骤**：选择滤镜后，**务必再次点击该滤镜**，会出现一个强度滑块。将滤镜强度适当降低（例如降到50%左右），让效果更自然，避免过于突兀。
4.  点击“√”确认。
5.  最后点击右上角的“...”菜单，选择“保存到相册”，尺寸选择“实际尺寸”。

## 高级技巧：Snapseed局部调整

如果遇到逆光拍摄导致脸部过黑，或身体局部肤色不均的情况，可以使用Snapseed的“局部”功能进行精准修复。

1.  在Snapseed中点击“工具”，选择“局部”。
2.  将画面放大，在需要调整的区域（如脸部）点击一下，会出现一个“亮”字标记。
3.  用双指在屏幕上做开合手势，可以控制调整范围的大小（红色区域）。将范围精确控制在需要调整的部位。
4.  左右滑动屏幕，即可单独调整该区域的亮度、对比度等参数。
5.  如果需要调整多个区域（如脸和脖子），可以再次点击添加新的调整点。

这个功能可以非常有效地解决局部曝光或色彩问题。

## 总结

本节课中我们一起学习了照片调色与滤镜处理的核心流程。我们首先使用**Snapseed**进行基础的亮度、对比度、饱和度等全局调整，并利用其“局部”功能解决特定问题；然后使用**VSCO**添加风格化滤镜，并记得降低强度以保持自然感。

修图看似步骤固定，但难点在于根据每张照片的光线和内容，培养出对色彩和风格的审美与感觉。这需要通过**大量的练习和观察**来积累经验。请记住，统一你朋友圈的照片风格（要么都加滤镜，要么都不加），会让你的个人展示更具质感。多拍、多修、多总结，你的修图功力一定会稳步提升。
