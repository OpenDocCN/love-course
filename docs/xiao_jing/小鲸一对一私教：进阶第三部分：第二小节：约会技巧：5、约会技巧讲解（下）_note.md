## 🔍 筛选：确立你的标准

上一节我们介绍了吸引力的基础，本节中我们来看看如何通过筛选让对方更重视你。

以下是你可以用来筛选的一些标准示例：

*   **生活技能**：例如，“你会做饭吗？”
*   **生活习惯**：例如，“你的生活作息规律吗？”
*   **个人经历**：例如，“你人生中有没有什么特别的冒险经历？”

你可以通过聊天自然探讨这些话题。如果女生在某些方面不符合你的标准，这本身就是一种筛选。

若女生明确表示不符合（例如，“我不会做饭”），可能有两种情况：一是你误判了她的兴趣指标；二是她本身价值可能未达你的预期。此时，你可以转向她可能具备的其他特质进行筛选，例如，“我看你朋友圈常分享读书，你喜欢看什么类型的书？”

**核心逻辑**：你设立的筛选标准，本质上是象征性的，目的是让她能够顺利通过，从而让你后续表达的兴趣显得合情合理。公式可以概括为：**她通过筛选 → 你表达兴趣 = 合理升级关系**。

## 🎬 现场实操与转场

我们回到约会现场，看看如何具体应用。

这样，你们就自然地变成了并排坐，手臂和大腿可能轻微接触，试探进入便已完成。

之后，可以利用拍照进一步进挪：
1.  “你照片挺好看，但我觉得本人更美。来，我教你个拍照角度。”

### 转场的重要性

转场是衔接整个约会、分散进挪步骤的关键。典型的转场路线是：**咖啡厅 → 餐厅 → 商场/游戏厅 → 酒吧 → 私密空间**。

进挪应分散在不同转场中：
*   在咖啡厅完成牵手。
*   前往餐厅的路上搂腰。
*   餐厅饭后，在街角接吻。
*   在酒吧互动后舌吻。
*   最终转场至私密空间**TD**。

**务必遵循进挪顺序**：`手 → 腰 → 抱 → 亲 → 摸`。切忌在短时间内跳跃式进行，例如刚搂腰就立刻尝试接吻，极易被拒绝。
