# 恋爱方法课程 05：追求阶段详解 🎯

在本节课中，我们将要学习恋爱关系发展中的“追求阶段”。这是继认识、熟悉和吸引之后的关键一步，其核心在于如何恰当地表达好感，并推动关系向恋爱方向发展。我们将重点探讨开始追求的时机、具体步骤以及其中的核心原则。

上一节我们介绍了建立吸引的重要性，本节中我们来看看如何将吸引转化为明确的追求行动。

## 何时开始追求？⏰

最重要的一点是：**必须在与女生建立起相互吸引之后，才能进入追求阶段**。这是一个核心分水岭。

*   **无吸引时追求**：在缺乏吸引力的前提下表达好感，会给对方带来压力，甚至可能引起反感。过早暴露需求感会降低你的价值，削弱前期建立的吸引。公式可表示为：`无吸引 + 暴露需求 = 价值降低`。
*   **有吸引后追求**：在吸引力已经存在的基础上表达好感，是对关系的正面推进。此时暴露需求感是加分的。公式可表示为：`有吸引 + 暴露需求 = 关系升级`。

因此，吸引是决定追求行为效果的关键前提。

## 如何追求？三步走策略 🚶‍♂️🚶‍♂️🚶‍♂️

明确了时机后，追求过程本身可以系统地分为三个步骤。以下是这三个核心步骤的详细介绍。

### 1. 筛选

筛选的目的是引导女生向你展示她的高价值，并给予她认可。这既是对女生的一种测试，也是一种肯定。

*   具体做法：通过提问或创造机会，让她展现自己的优点，例如厨艺、才艺、性格闪光点等。
*   你的回应：对她的展示给予真诚的欣赏和认可。代码逻辑可理解为：`if (女生展示优点) { 给予正面反馈(赞美、认同) }`。

### 2. 可得性

在筛选之后，你需要让她明白“你为什么喜欢她”。这是将你的好感与她具体的优点联系起来，让她感到被喜欢是合理的、具体的。

*   具体做法：明确告诉她，正是因为她所展现的那些特质（例如善良、有趣、独立），你才对她产生了好感。公式可表示为：`你的优点 (A+B+C) => 我的喜欢`。

### 3. 表白

表白是追求阶段的收尾仪式。它通常在双方关系已经心照不宣时进行，用以正式确认恋爱关系。

*   核心要点：表白应是“临门一脚”，而非“开球”。它宣告追求阶段的结束和恋人关系的开始。

> 关于“筛选”与“可得性”的详细技巧，我们会在后续专题课程中深入讲解。“表白”同样有独立的课程进行剖析。

## 理解女生的行为模式与应对策略 🧩

在追求过程中，理解对方的心理和行为模式至关重要。女生通常表现得更为矜持和谨慎，这种行为背后有多重原因。

*   **延长考察期**：为了有足够时间判断男生是否真诚、表里如一。
*   **确认自我感受**：确保自己是真正被吸引，而非被技巧迷惑。
*   **增加男生投入**：根据“投入越多，越难放弃”的原则，这有助于关系的长期稳固。
*   **规避社会压力**：被动接受追求可以避免被贴上“随意”、“不自爱”等负面标签。

因此，在追求阶段，男生需要承担主动推进的责任。女生的“主动”往往体现在对男生“主动”行为的积极回应和配合上。

同时，这个阶段女生可能会提出更多“废物测试”，例如“你为什么喜欢我？”、“你和你前女友为什么分手？”。这些测试旨在探测你的真诚度和一致性。关于如何应对“废物测试”，艾伦老师的课程会有专门讲解。

当然，现代社会也有很多女生在建立吸引后会主动出击。即使遇到这种情况，我们也不应跳过上述的“筛选”和“可得性”步骤。缺少这些环节，即使在关系确立后也可能埋下隐患。

本节课中我们一起学习了恋爱“追求阶段”的核心框架。我们明确了追求必须在**吸引建立之后**开始，并系统拆解了追求的**三个步骤：筛选、可得性与表白**。同时，我们也分析了女生行为背后的逻辑，强调了男生主动推进与理解对方的重要性。掌握这些原则，能帮助你有策略、有分寸地表达好感，顺利跨越从朋友到恋人的关键一步。
