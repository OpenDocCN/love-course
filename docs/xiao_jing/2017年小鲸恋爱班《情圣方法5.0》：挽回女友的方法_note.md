## 分手原因分析

分手的情况大致可以分为以下几种。理解这些类型有助于我们定位自身问题。

以下是四种主要的分手原因类型：

1.  **性格过于被动**：太懵懂、太内向、没有主见、不会调情、不够有吸引力。
2.  **性格过于主动**：太外向、太轻浮、太会调情、太“坏”，给人感觉你只有性需求，像个渣男。
3.  **情绪与性格问题**：情绪化、不自信、没有安全感、占有欲强、多疑、对女生限制过多。
4.  **生活状态消极**：消极、抱怨、负能量、一事无成，在对方身上看不到希望。

## 挽回的核心原则与框架

上一节我们分析了分手的常见原因，本节中我们来看看挽回时应遵循的核心原则。挽回需要根据具体情况、具体的人和具体事件来制定策略。但这里可以介绍一个通用的大框架。

**核心公式：有效挽回 = 停止错误行为 + 深刻自我反思 + 切实自我建设**

### 第一步：立即停止纠缠

女生提出分手时，我们应该做的是冷静下来，不要主动去联系她。然而，我们往往却信息、电话不断轰炸。这种行为会让女生极度反感，甚至拉黑你。她们不想接电话的目的，其实就是暂时不想跟你接触。所以这个时候，我们就不要去找她。

我们常有的错误想法是：“我要和她好好沟通一次，把话说清楚。”但沟通是建立在双方都愿意的基础上的。如果没有这个前提，我们的沟通其实就是骚扰。即使建立了沟通，也是无效沟通。我们想做的本是让对方重新喜欢你，但我们却在做让她讨厌的事情。如果你连自己都控制不了，又怎么能控制好别人呢？

### 第二步：进行深度自我反思

分开以后，我们其实要反思一个问题：她以前喜欢你，为什么现在不喜欢你了？也许你照照镜子，自己也能明白：你还是不是之前的那个样子？

*   以前的你可能是：**自信、阳光、乐观、神采飞扬、自由、淡定、洒脱**。
*   现在的你可能是：**自卑、抑郁、无精打采、焦虑、无奈、不知所措**。

### 第三步：专注自我建设与展示

认识到问题后，就要冷静下来，先把自己现在的生活，恢复成你曾经的那个样子，然后变本加厉地去做自我提升。

在朋友圈等社交平台，让别人看到不一样的、正在变好的你。这个过程就像搭讪：我们以为搭讪认识的是女生，其实认识的是我们自己。挽回也是同理，我们以为挽回的是前女友，但其实挽回的是那个**失去的、更好的自己**。你的目标是改变并恢复成以前有魅力的样子。

## 关于“第三者”问题的特别说明

我们经常还会遇到一种情况，就是分手时出现了第三者。但外因都是通过内因起作用的。如果你跟这个女生的关系很好，基本上是不会出现第三者的，女生一般都比较专一。

这个“内因”其实就是你自己。感情出现问题，负有最大责任的一定是自己。要多从自己身上找原因，让自己的感情成熟起来。这个时候如果我们去找第三者麻烦，或者纠缠他们两人，都不能改变你自身的问题。我们要发现自身的问题，然后去改变它，这才是我们应该做的。

建议大家参照我们前面讲的长期关系课程，去发现我们在交往过程中到底出现了哪些问题，然后有针对性地改变它。
