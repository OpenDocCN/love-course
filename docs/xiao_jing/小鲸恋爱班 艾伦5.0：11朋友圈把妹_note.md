讲完按等级划分邀约的方法，我们再来看一个可以运用的小技巧：学会应用备注来快速筛选信息。

可以备注的信息包括：
*   身高
*   年龄
*   星座
*   一起做过的事情
*   胸围
*   通过什么方式认识
*   是否约会过，约过几次
*   是否发生过关系，几次
