# 课程一：Kevan的自我介绍 🎤

在本节课中，我们将跟随情感导师Kevan的视角，回顾他从学生时代到成年后的情感经历与观念演变。通过他的个人故事，我们将理解实践经验与自我反思在情感学习中的重要性。

## 早年经历：学生时代的探索与挫折

我是小鲸恋爱班的情感导师Kevan。学生时代，我是一个特别调皮的学生。中学生活有两条主线：一是打架，学校里不能有比我强势的人，否则我就要动手；二是追求女生，学校里最好看的女生应该是我的，即使我追不上，别人也不能追。现在回想起来，这种逻辑很幼稚，但当时这就是我的行事准则。

那时我追求女生厉害吗？答案是不厉害。

小学四年级时，我喜欢一个低年级的女生。为了和她同班，我告诉父亲自己成绩不好需要留级，于是留级到了她的班级，坐在她后面。我们每天一起放学回家，那段时光很美好。升入初中后，我们仍在同班，座位不远。但她似乎喜欢上了另一个班的男生，对方也是英语课代表。我经历了人生中最灰暗的一段时光。

进入初中后，我不敢和她说话。和别的女生可以开玩笑、打闹，但面对她时，我表现得非常呆板。记得初一有一天放学，我写了一张只写了四个字的小纸条。趁没人的时候，我叫住正在回家的她，说我有东西要给她。她可能猜到我的意图，表示不要。我问她确定不看一眼吗？她扭头说了一句拒绝的话，然后离开了。那个画面一直清晰地印在我脑海里，当时我没忍住掉下了眼泪。后来，我把那个英语课代表打了一顿。从那以后，我再也没有和那个女生说过一句话。

初中时，我也追求过其他女生。曾骑自行车到女生家附近找她玩，被家长发现后非常尴尬，被指责初中就开始谈恋爱。高中时，恋爱有失败也有成功：有我追不到的女生，也有喜欢我但我不理会的女生。毕竟，女生不喜欢我，我也没办法。但我有一个优点：女性朋友很多，能和女生打成一片。比如请女生帮我抄家庭作业、和女生聊天、了解谁喜欢谁、哪个女生暗恋谁等。有时我也会翻女生的书包，看看有没有带课外书。

## 大学时期：恋爱经历的丰富与转变

上大学后，我学习服装设计，身边有很多美女，学校里美女也很多。别看我现在的样子，上大学时长得还不错。在我的观念里，大学只要不挂科就好，主要精力都放在谈恋爱上。我交往过校花、系花、国际班副班长、舞蹈团团长等，身边的女生一直在换。

我讲述其中一段恋爱经历：刚上大一时，我和舞蹈团团长的故事。那时学校舞蹈团招新，她上大二，是我的学姐，艺术系学习室内设计（当时叫装潢设计）。她是学校舞蹈团团长。招新那天，我们去参加面试，一位教授带着她（后来我们都称这位教授为“老头子”，现已过世）。她负责示范动作，老头子负责选人。看到她示范动作时，我很惊讶，觉得怎么会有这么好看的人，身材特别好。我至今记得当时的画面：她扎着辫子，用彩色绳子绑着头发，穿着裤脚收紧的裤子，显得腿部线条特别好看。皮肤很白，身高大约一米六六，身材很好。招新那天，老头子特别喜欢我，于是我进入了舞蹈团，认识了她。后来发现，她居然是我的老乡。

那时，我虽然在新大一新生和班级军训中很受欢迎，但从未想过会和她有交集。我们的开始是她追求我。有一天，她经过我住的寝室一楼门口（可能是中秋节），我问她要去哪里，晚上有什么安排。我当时在门口拿着一个大杯子喝水。她说晚上没安排，等我约她。我很懵，不知道如何应对。晚上她给我发消息，问我有女朋友吗？我回复还没有。她问要给我介绍女朋友吗？我说好。她问我喜欢什么类型？我说不知道。她问：“你看我这样的行不行？”我说当然行。她说那就这么定了，于是我们开始交往。

因为她的缘故，后来学校里很多人认识我，说我是某某的男朋友。她在我们学校应该算排名第二的美女（因为还有一个身高一米七五、读大四的女生更漂亮）。我一直觉得这个女生才是我的初恋，因为和她在一起的感觉是以前从未有过的，很甜蜜。我们交往了一年多后分手。

## 成年之后：社交拓展与观念成熟

毕业后，我从事服装设计，接触到一些艺人和网红。加上我比较爱玩，社交能力不错，认识了很多演艺圈的人，也交往过几个艺人和网红。同时，我学习了PUA（搭讪艺术），因此吸引力较高。很多人对明星的私生活感到好奇，看到明星会很激动，认为明星不是常人，无法用正常的眼光去评价。例如，有些人觉得明星是最帅最美的，认为普通人长得像他们的偶像是一种侮辱。但其实不是这样。明星确实好看，但在生活中，很多人也能长得那么好看。我见过很多知名明星，和他们一起喝酒、吃饭。我最深的感受是：他们也是常人。我现在可以用常人的眼光看待他们，这是我自己转变的地方。

下面讲述我和一个艺人的恋爱经历。她是双子座，开始并不喜欢我，因为我们的条件不吻合。她很漂亮，也有名气，追求者会送她车、打款，送999朵玫瑰（999朵玫瑰体积很大）。如果她想嫁一个有地位的人，对她来说很简单。但我有我的优势：我组织各种聚会，如去酒吧玩、唱歌、郊游等，认识很多玩得很嗨的朋友，而且很会哄女孩子开心。很多女生喜欢和我一起玩，我认识的朋友也很多。她的一个闺蜜和我的关系也很好。

有一次我约她喝下午茶，她本来不打算来。但当时她正在演一个孕妇的角色，家里有孕妇假肚子道具。她的闺蜜劝她带着假肚子出去，说自己怀孕了，看看我的反应。她想凑热闹，取笑我，于是答应了。她挺着大肚子出来，穿着针织裹身衣，显得肚子更大。我看到后吓了一跳，心想这下完了，很尴尬，不知道说什么。我说前段时间见她还没有肚子，怎么突然挺着大肚子出来了。她是演员，眼泪很逼真，我看不出来。她说前段时间不明显，穿外套看不出来。下午，我安慰她，劝她把孩子打掉，告诉她当单亲妈妈多么辛苦，但她不同意。回去后，她和闺蜜描述我是如何被骗、如何傻的。双子座的特点是爱玩，对熟悉的朋友特别调皮，但对不熟悉的人特别高冷。后来我发现被骗了，但内心很开心，因为又有机会了。经过这件事，我们的关系好了很多，经常和她及她闺蜜一起玩。

有一次组织郊游，几个朋友一起去。那晚灯光暧昧，音乐诱惑，我比较主动，有机会睡在一起。但她跟我说，睡在一起可以不动她吗？我说可以，只要她把持得住就行。她说她肯定可以。我说那就好。当然，后来我没忍住，发生了关系。但我们两个脾气都很大，发生关系后经常吵架，差点打架。交往一年多后，我们分手了。

## 对PUA的见解与课程理念

现在，我已经记不清具体谈过多少次恋爱了。不要相信那些PUA自称谈过多少多少次恋爱，真正到一定数量是记不住的。我会记住某些事情，但具体数字真的不记得。我现在所教授的内容，并不是PUA的方法或技巧。PUA是几十年前出现的概念，讲的是如何在夜店搭讪，通过特定环境和开场白吸引平时不可能吸引到的女生，并快速发生关系。但国外的情况与中国并不吻合，这些方法在中国基本不适用。不过，PUA也总结了一些有道理的内容，比如从基因进化层面分析（我在《基因的奴隶》一课中讲到），以及M3模型（将追求女生的过程步骤化）。它的意义在于第一次总结了一些基本规律，但如果你没有大量恋爱经验，可以不用看。这些内容不太适应今天的中国。不说国外与国内的差距，就说八九十年代中国女性与现在女性的差距，拿九十年代追女生的套路追现在的女生，结果会很惨。

PUA传到中国时，我看了很多团队。现在中国有很多大大小小的团队，基本上是一些完全不会搭讪的人，把这些东西神化，然后教别人搭讪，基本上都是胡扯，像传销一样，卖给你一个PUA的梦，但他们自己都不是PUA。在国外，PUA也只是小众文化，因为一本畅销书《把妹达人》（美国记者写的）将它推到了公众视野。

我们现在做的是将理论转化为适合中国的理论，结合大量案例进行教学。我看了无数大大小小团队的内容，把其中最好的、有用的东西总结出来，用简单易懂的方式让你理解，知其然并知其所以然。结合自身和学员的大量案例，提炼出在实践中真实有效的技巧和方法。

## 初衷与意义：情感教育的重要性

最后，我讲讲为什么做这个。我自己有这么多经验，通过这些经验，我知道追求女生、谈恋爱确实有技巧和方法。而另一半和未来的家庭，除了事业以外，是人生最重要的事情。比如王宝强、马云的事件，还有前段时间跳楼自杀的程序员，他被一个女生骗了很多钱，最后被逼自杀。他是一个很成功的程序员，开发的应用在中东有3000万人使用。但在现实生活中，很多人对这方面一无所知。

去年，国家将“圣女”一词搬上春晚；今年开春，党组织开始极力宣传恋爱结婚。我也算是响应国家号召，为国家贡献绵薄之力（开玩笑）。我觉得做这件事很有意义，因为这确实有技巧。我希望那些不懂、不注重这方面技巧的人能多注重一点，因为你的家庭和另一半真的很重要。所以，我认为这是一件有意义的事情。

本节课中，我们一起回顾了Kevan从学生时代到成年后的情感经历，了解了实践经验与自我反思的重要性，并探讨了PUA的局限性与情感教育的真正意义。希望这些内容对你有所启发。
