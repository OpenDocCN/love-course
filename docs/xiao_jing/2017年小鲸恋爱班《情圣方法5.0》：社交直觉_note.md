# 2017年小鲸恋爱班《情圣方法5.0》：社交直觉_note

在本节课中，我们将要学习社交直觉的核心概念。社交直觉是情商的一个分支，其核心在于判断女生对你的好感或反感窗口。这种能力需要在与女生的交往中逐步培养和加强。

## 什么是社交直觉？

社交直觉的核心是判断女生对你的好感或反感窗口。这需要我们在交往中不断观察和总结。

上一节我们介绍了社交直觉的定义，本节中我们来看看好感窗口的具体表现。好感窗口分为主动和被动两种类型。

## 主动好感窗口

主动好感窗口指女生主动表现出的靠近或兴趣信号。以下是常见的主动好感窗口示例：

*   女生主动与我们进行眼神接触，例如在你未注意时深情注视，当你回看时她又移开目光。
*   女生迎面走来时会主动微笑或进行调侃、挑逗。
*   交流时，女生的身体会倾向我们，说话时会凑近。
*   遇到麻烦时，女生会首先寻求我们的帮助，即使身边有其他人可选。
*   女生会主动询问问题以表达好奇，并对我们感兴趣的事物表现出浓厚兴趣。例如，我们发了一张骑马的朋友圈，她可能会回复：“我喜欢骑马，这是在哪里？你会骑马吗？可以教我吗？”
*   在对话偶尔沉默时，她会主动开启新话题。
*   对话过程中，她听得认真，回应积极。
*   在微信聊天时，她回复速度快，字数多，并常使用开心的表情。
*   女生会模仿我们的行为或喜好，例如我们常喝红牛，她最近也开始喝；或使用我们常说的口头禅。
*   与闺蜜同行，闺蜜离开后她仍留下。
*   在KTV等场合，她会选择坐在我们身边，即使暂时离开（如上厕所、点歌），回来后仍坐回原处。或者最初不坐在旁边，但回来后坐得离我们很近。
*   与我们在一起时，她不怎么玩手机或不接电话。
*   交流时，她心情愉悦，享受互动，并且膝盖、胸膛、脚尖等身体部位会朝向我们。
*   她会主动问起关于前女友、是否单身等问题，例如：“很多女孩子喜欢你吧？”、“你为什么单身？”、“你喜欢什么样的女生？”。

## 被动好感窗口

大多数女生比较矜持，因此被动好感窗口更为常见。被动好感窗口指女生虽不主动，但会积极回应我们的靠近。

以下是其表现：

*   女生从不主动找你，但对你的联系有求必应。你找她，她就会回应；你约她，她就会出来。
*   简单说，女生的态度是：**不主动，不拒绝**。
*   她们不会主动靠近我们，但当我们靠近时，她们不拒绝，并且会配合。

## 反感窗口

反感窗口与好感窗口相反。当女生表现出与上述好感窗口相反的行为时，即代表她对我们没有好感或持有反感。

例如：
*   回避眼神接触。
*   身体后倾或转向别处。
*   回复消息很慢或字数很少。
*   经常看手机或找借口结束互动。
*   拒绝我们的靠近或邀请。

与女生交往时，社交直觉是必备技能。它能帮助我们避免贸然行动，使自己陷入尴尬处境。所谓的“直男癌”，其本质就是社交直觉较低。

因此，我们需要不断提高自己的社交直觉，从而在与女生交往时，能准确判断她的反馈和好感度。这一点特别重要。

本节课中我们一起学习了社交直觉的概念，以及如何通过主动、被动的好感窗口和反感窗口来判断女生的兴趣。掌握这些信号，能让你在交往中更加从容和有效。
