## 什么是预期？

**核心公式**：植入预期 = 描绘未来共同经历的画面 + 引导女生产生期待

4.  **打破焦虑**：对于不擅长现场发挥的男生，提前植入预期能提供清晰的行动指南，缓解焦虑。
