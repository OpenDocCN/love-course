# 波本酒制作教程 🍷

## 概述
在本节课中，我们将学习如何制作波本酒。波本酒是一种以玉米为主要原料的美国蒸馏酒，具有独特的风味和口感。

## 材料与工具
以下是制作波本酒所需的材料和工具：

**材料：**
- 玉米（至少50%）
- 大麦麦芽
- 水
- 酵母

**工具：**
- 麦芽粉碎机
- 麦芽糖化锅
- 发酵桶
- 蒸馏器
- 储存桶

## 制作步骤

### 1. 玉米处理
**步骤：**
- 将玉米磨成粉。
- 将玉米粉与水混合，制成玉米浆。

**公式：**
玉米粉 + 水 → 玉米浆

### 2. 麦芽糖化
**步骤：**
- 将大麦麦芽磨成粉。
- 将麦芽粉与水混合，加热至60-65°C。
- 保持温度，让麦芽糖化。

**公式：**
麦芽粉 + 水 → 麦芽糖

### 3. 发酵
**步骤：**
- 将玉米浆和麦芽糖混合。
- 加入酵母，搅拌均匀。
- 将混合物放入发酵桶中，保持温度在20-25°C。
- 发酵约5-7天。

**代码：**
```python
def ferment(mixture, temperature, days):
    # 发酵过程
    for day in range(days):
        # 每天检查温度
        if temperature < 20 or temperature > 25:
            print("温度不在适宜范围内，发酵失败。")
            return False
    return True
```

### 4. 蒸馏
**步骤：**
- 将发酵好的混合物放入蒸馏器中。
- 加热蒸馏器，收集蒸馏出的酒精。

**公式：**
发酵混合物 → 酒精

### 5. 储存
**步骤：**
- 将蒸馏出的酒精倒入储存桶中。
- 储存一段时间，让酒精成熟。

**代码：**

## 总结
本节课中，我们一起学习了如何制作波本酒。通过玉米和麦芽的糖化、发酵、蒸馏和储存，我们得到了美味的波本酒。希望这节课能帮助你了解波本酒的制作过程。
