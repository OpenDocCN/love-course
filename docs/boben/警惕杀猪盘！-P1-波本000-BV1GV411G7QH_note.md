## 什么是“杀猪盘”？🐖

“杀猪盘”是一种结合了情感诈骗和金融诈骗的复合型网络骗局。骗子将建立信任的过程比喻为“养猪”，最终实施诈骗敛财则称为“杀猪”。

上一节我们介绍了课程主题，本节中我们来看看其具体定义。

这种骗局通常通过社交网络或交友平台寻找目标（即“选猪”），然后通过长期的情感交流建立亲密关系（即“养猪”），最后引诱受害者在虚假投资平台或赌博网站投入大量资金（即“杀猪”）。

## “杀猪盘”的运作模式 ⚙️

了解其运作模式是识破骗局的关键。整个过程可以被系统地拆解为几个阶段。

以下是“杀猪盘”骗局的典型步骤：

1.  **寻找目标（选猪）**：诈骗者在婚恋网站、社交平台寻找情感空虚、有一定经济基础的潜在受害者。
2.  **建立关系（养猪）**：伪造成功人士身份，通过日常关怀、情感共鸣等手段，与受害者建立恋爱或亲密朋友关系。
3.  **诱导投资（铺垫）**：在信任建立后，不经意间透露自己通过某投资平台或内幕消息获利，激发受害者兴趣。
4.  **小额试水（诱饵）**：引导受害者在其控制的虚假平台进行小额投资，并让受害者顺利提现，以消除疑虑。
5.  **大额投入（杀猪）**：以“机会难得”、“为未来共同生活”等理由，诱使受害者投入所有积蓄甚至贷款。一旦大额资金进入，平台便无法提现，骗子随之失联。

## 核心话术与心理学原理 🧠

骗子的话术经过精心设计，旨在利用人类的情感弱点。理解这些常见话术能帮助我们保持警惕。

以下是诈骗中常用的核心话术及其对应的心理操控点：

*   **“我在用舅舅/朋友公司的内部平台，稳赚不赔。”**
    *   **原理**：利用“内幕消息”的稀缺性和权威性，制造不公优势的假象。
*   **“我想和你有未来，我们一起多赚点钱吧。”**
    *   **原理**：捆绑共同未来与经济利益，利用受害者对关系的承诺感。
*   **“你先试试，能提现的。我怎么会骗你？”**
    *   **原理**：通过可验证的小利（小额提现）建立全面信任，为后续大额诈骗铺路。
*   **“快追加资金，马上要涨了/要封盘了！”**
    *   **原理**：制造紧迫感和贪婪，促使受害者在非理性状态下匆忙决策。

## 如何识别与防范？🛡️

识别骗局需要结合理性判断和常识。我们可以通过一些具体特征来辨别。

以下是关键的识别点和防范措施：

1.  **核实身份**：对方拒绝视频通话或线下见面，所有个人信息仅停留在网络。
2.  **检查平台**：对方推荐的投资或赌博平台无法在官方应用商店找到，或需通过特殊链接下载。
3.  **警惕高回报**：承诺“稳赚不赔”、“超高回报率”的项目都违背基本金融规律。记住一个简单公式：**过高的收益率 = 高风险或骗局**。
4.  **保护隐私**：切勿向网友透露自己的财务详情、账户密码及验证码。
5.  **咨询他人**：在投入大笔资金前，务必向身边可信赖的亲友或专业人士征求意见。
6.  **立即止损**：一旦怀疑，立即停止转账并尝试提现所有资金。若失败，保留证据（聊天记录、转账凭证）并立即报警。
