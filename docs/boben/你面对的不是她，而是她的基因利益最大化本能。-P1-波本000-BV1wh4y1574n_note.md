## 基因利益最大化本能：核心概念

上一节我们概述了本课程的主题，本节中我们来看看“基因利益最大化本能”这个短语的具体含义。它指的是，生物个体（包括人类）的许多行为，在无意识层面上，是由其基因寻求被复制和传递到下一代的本能所驱动的。

从进化生物学的角度看，个体的生存与繁殖成功，是其基因得以延续的载体。因此，**个体的行为策略往往倾向于最大化其基因的长期传播概率**。这可以理解为一个优化问题：

**公式：** 行为策略 ≈ argmax(基因传递的成功率)
