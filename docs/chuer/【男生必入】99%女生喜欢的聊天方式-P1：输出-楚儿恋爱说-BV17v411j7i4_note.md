# 恋爱聊天技巧课 - P1：核心输出思维 🧠

在本节课中，我们将要学习那些能让聊天变得轻松、吸引人的核心思维方式。掌握这些思维，你也能像那些“会聊天”的人一样，与女生自然流畅地交流。

---

聊天对于不擅长的人来说，常常令人头疼：不知道聊什么、怎么聊、冷场了怎么办。然而，许多人都见过那些特别会聊天的男生，他们似乎从不担心这些问题，和谁都能聊得来。

你可能会认为他们天生外向、善于言辞。天生外向或许是真的，但能做到“会聊天”并且把话说到女生心坎里，这往往需要技巧。那么，他们究竟是如何做到的呢？解决问题要从源头出发。接下来，我们将剖析那些会聊天的男生所具备的思维方式。

## 1. 学会换位思考 🔄

换位思考是每个人都应该学习的重要课程。在与女生交往时，你也需要多运用这种思维，否则对方如何能了解到你其实是个不错的人呢？

例如，女生常说喜欢帅气、幽默的男生。很多男生一看，觉得自己既不帅也不幽默，立刻就泄气了。其实不必如此。你应该这样想：谁不喜欢好看又幽默的人呢？男生同样也喜欢美女。

**核心思维公式：** `站在对方角度思考问题 → 理解普遍偏好 → 减轻自我压力`

只要你学会站在对方的角度考虑问题，你的心情都会好很多。女生不仅喜欢帅气幽默的男生，也同样欣赏情绪稳定、脾气好的男生。

上一节我们介绍了换位思考的重要性，本节中我们来看看如何在实际互动中给予对方良好的感受。

## 2. 积极给予正反馈 👍

没有人喜欢被反驳或否定。但很巧的是，我身边就有这样一位朋友。虽然我们关系很好，但每次聊到某些话题，总会因为观点不同而像要展开一场辩论赛。如果不是因为他最后总能好脾气地收场，我们恐怕早就闹翻了。

人们在聊天时，总是希望收到正反馈。什么是正反馈？举个例子，就像男朋友最“恐惧”的问题：“我今天好看吗？”、“这件裙子我穿好看吗？”、“这对耳环适合我吗？”。

**核心应对代码：**
```plaintext
if (女朋友询问外观评价) {
    response = “好看，适合，你穿/戴什么都好看”;
}
```

这听起来可能有些“无理取闹”，但女生就是喜欢听到这样的回答，她会因此开心，然后对你更加温柔。当然，在还不是男女朋友关系时，没有必要如此刻意地“哄”，女生也能分辨出来。但即便如此，在聊天中多给予正面的反馈仍然是必要的。

学会了提供情绪价值，接下来我们要探讨聊天的核心目的——关注对方。

## 3. 将关注点放在对方身上 👀

“你好像不怎么发朋友圈啊？有没有照片啊？放我看看呗。没有？那现在拍一张吧。” 我原以为这只是网络段子，谁知艺术源于生活。这种情况，甚至可能发生在并非未曾谋面的人之间。不知道为什么，有些人就是执着于要看照片。

这种行为是典型的反面教材，可以直接判定出局。这里要跟兄弟们强调一点：聊天是为了什么？是为了拉近彼此的关系。像上面那样聊天，不知道的还以为你是来选美的。

与女生聊天，最忌讳的就是一厢情愿，比如：“我要帮她，我一定要帮她”。有些兄弟在聊天时，总是一味专注于表达自己，完全听不出女生已经不耐烦了。要么就是在对方说话时，只是敷衍地“嗯”、“啊”，既不感兴趣也不关心。

那么，女生是在跟你聊天，还是在说单口相声呢？看看那些会聊天的男生，为什么他们能把女生逗得哈哈大笑？为什么他们能让女生产生心动的感觉？因为他们明白，女生需要的是**关注**和**情绪的共鸣**。

**核心概念：** 聊天是 `双向互动`，而不是 `单向演讲 + 单向倾听`。

因此，聊天时要多观察女生对你谈论的话题是否感兴趣。如果她明显接不上话，或者显得兴致缺缺，那就应该赶紧切换话题。如果她开启的话题你不熟悉，也可以从中找到一个切入点，巧妙地转移到你更了解或更感兴趣的领域。
