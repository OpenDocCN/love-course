# 恋爱教程 01：掌握“以牙还牙”法则，建立平等关系 💔➡️❤️

在本节课中，我们将学习一个关键的恋爱法则——“以牙还牙”。通过一个真实案例，你将理解如何在追求过程中建立平等、尊重的互动关系，避免陷入单方面付出的困境。

很多男性在追求女性时，会因为对方偶尔的主动而欣喜若狂，也会因为自己发送大量消息得不到回应而借酒消愁。然而，第二天他们仍会装作无事发生，继续发送没有回应的问候。

这种方式显然无法成功追到心仪的对象。追求需要讲究方法。接下来，我们将通过一个具体案例，拆解“以牙还牙”这个万能公式的应用。

## 案例背景：不对等的付出

一位36岁的牙医兄弟找到我，讲述了他的困境。他喜欢上一位常来看牙的女性，对方追求者众多。为了脱颖而出，他尝试了各种网上流行的追求方式，但关系始终停滞不前。对方只在有需要时才会主动联系他。

最近，女生看上了一枚卡地亚戒指，他立刻下单购买。朋友却提醒他可能被当成了“备胎”。他感到困惑，因为在他生病时，女生曾请假照顾他，还亲手为他做生日蛋糕。他迷茫于这是否代表喜欢，以及自己该如何行动。

通过沟通，我能感受到这位兄弟在感情中紧绷的状态。过于喜欢一个人，容易让自己的一切情绪都被对方牵动，每天付出大量时间精力，却在不确定的关系中痛苦挣扎。

## 第一步：停止妥协，建立边界

幸运的是，他找到了我。接手后，我做的第一件事是让他退掉戒指。他起初很惊讶且不敢，但我指出，当前这种不对等的关系，根源在于他自己忘记了也需要被爱和回应。

**核心问题**：你总是“呼之即来，挥之即去”，一次次妥协让步，让对方有恃无恐。你的付出太轻易，对方自然不愿投入自己的时间和精力来经营这段关系。

我告诉他，我的规划是为了他未来的幸福。过程或许会让他和对方感到不适，但结果是导向幸福。他听从建议，退掉了戒指。

## 第二步：等待时机，实施“以牙还牙”

我们首先要做的就是等待。既然对方以往只在有需要时才找他，那我们就静候这个时机。

两天后，女生主动发来消息：“戒指到了吗？”
我们回复：“我退了。”
女生有些生气：“退了？为什么？那不是送我的吗？”

在女生的惯性思维里，兄弟的付出是理所当然的。我们的一次拒绝就引发了她的不满，仿佛我们夺走了本属于她的东西。此时，我们的首要目标是扭转她这种想法。

我们回复：“戒指的意义太神圣了，不该存在于我们这种不明不白的关系里。”
这句话直接点明了两人关系的现状，不给对方回避问题的机会。

女生回复：“这话说得太严重了吧？你不想买就算了。”
我们表明态度：“你要的东西我送不起。”
这传达了一个信息：你可以随时收回你的好，我同样可以。这就是 **“以牙还牙”**。

我们通过感同身受告诉她：我的感情不会给一个不值得的人。
女生回复：“这就是你说的喜欢？真是一个戒指就把你看清了。”

有的兄弟可能会疑惑，对方态度为何突然强硬。这很正常，这是她“拿捏”人的策略：软硬兼施。一种方法不管用，就换另一种。这也是兄弟过去得不到回应却仍卑微付出的原因之一。

兄弟态度的转变让对方琢磨不透，不敢再用旧模式对待他。这为我们创造了绝佳的进攻时机。

## 第三步：让对方体会“不被在意”的感觉

接下来，我们让她也体会一下不被在意的感觉。
我让兄弟回复：“随你怎么想吧。”
我们表现出生气，对方才会意识到兄弟过去对她有多么用心和呵护备至。而这样一个对她好的人，可能就要被她亲手推开了。

过了很久，女生发来消息：“我那时候在气头上，说话有点重，你不会生气吧？”
转折点来了。对方态度软化是好的证明，但也不能高兴太早。她看似在道歉，实则并未真正意识到错误，只是想通过服软哄回兄弟。

遗憾的是，兄弟已不再是过去的他，这招已经不管用了。
我们直接回复：“没生气。还有事吗？”

五分钟后，女生问：“你到底怎么了呀？怎么突然就这样了？”
我让兄弟回复：“没什么。我只是突然发现，你的不知好歹，不过是我让你恃宠而骄罢了。所以我只是把你对我的傲慢，轻轻添了一笔还给你而已。原来不用在意别人的心情，真的很轻松。”

**核心对比**：没有对比就没有伤害。想让对方真正明白你的付出和珍贵，你就要拿出“以牙还牙”的姿态，与过去的自己形成对比。对方才会明白你曾经多么体贴入微，才会真正反思是否忽略了你。

女生回复：“我没有不在意你。”
她看似示弱，但依然没有正视过去的错误。我们便再给她一点时间思考。

十分钟后，女生回复：“我有时候可能是回消息慢了一点，但大家都是因为有事才那样的。我也知道你喜欢我、对我很好。可就是因为这样，我才会无视你的心意，就是怕闹翻之后会真的失去你。都说朋友比恋人更长久，我只是想你永远都能陪着我而已。”

兄弟们请注意，这里对方看起来非常真心，似乎之前的一切都是为了两人好。这其实是对方因兄弟的改变而慌了阵脚，只能以退为进，试图解决问题。如果是以前的兄弟，可能就感动了。

但我告诉兄弟：想要得到真正想要的，就不能被眼前的甜头迷惑。
我让兄弟回复：“那你有没有想过，我为什么从来都不曾用你对我的方式对你？因为我怕你生气，怕你难过，所以连一句重话都不敢说。但你的肆无忌惮让我明白了，所有的快乐、痛苦和幸福感，都是我一个人的感受。不过还是感谢你的陪伴。我确实放不下，但我不会再奔向你了。”

对方说是为了兄弟好，那我们就告诉她什么是真正的好：是为了让你开心而委曲求全，而不是打着“为你好”的名义做着伤害你的事。同时，我们也告诉对方：我们累了，不再沉溺于这样的关系。

女生回复：“所以你现在是要跟我划清界限吗？这么久的感情你说放弃就放弃吗？”
我们回复：“我以前也觉得放弃一段真心付出的感情会很遗憾，所以试图站在你的角度和你共情。可我越为你考虑，就越为自己感到难过。有很多个睡不着的夜晚，我都在想，你明明看见了我双手捧上的真心，怎么还舍得把它一次又一次地践踏在地上？真心到底能换来什么？原来只有失望。”

面对对方的挽留，我们告诉她：虽然舍不得，但这段感情我看不到尽头了，所以不愿再走下去。

## 第四步：坚持与收获

之后的好几天，对方都没有回复。兄弟每天都很焦虑，但我有把握对方肯定会再找他，于是安慰他交给时间。

突然有一天，在我们沟通时，对方发来一条长消息：“这么久以来，我一直欠你一句对不起。对你忽冷忽热，还心安理得地享受你对我的好。你很失望吧？以前我没有认真对待你的感情，也没有看清自己的内心。到现在你真的要离开了，我才发现，原来我早就喜欢上你了。如果你对我还有那么一点点好感，能不能再给我一个机会？这次换我来追你。”

至此，案例基本结束。对方认清了自己的感情，并践行了追求兄弟的誓言。兄弟后来春风得意，天天向我报备进展。没过多久，两人便确定了情侣关系。

## 总结与核心法则

本节课中，我们一起学习了“以牙还牙”法则在恋爱关系中的应用。

**核心公式**：
**健康关系 = 平等付出 + 明确边界 + 自我尊重**

想要获得真正的幸福，你只能走99步，剩下的那一步，应该用于引领和带动对方勇敢踏出。通过建立边界、停止单方面过度付出，并让对方体会到同等的感受，你才能扭转不平衡的关系，赢得真正的尊重与爱情。

记住，你的真心很珍贵，应该留给懂得珍惜的人。
