# 【脱单干货】怂男注定单身，哪些

在本节课中，我们将要学习为什么“胆怯”是脱单路上的主要障碍，以及如何通过转变心态和行动来抓住稍纵即逝的机会。我们将探讨三个核心观点，帮助你理解感情中的机会本质。

上一段我们提到了胆怯对脱单的影响，接下来，我们具体看看调查中揭示的真相。

我曾进行一项调查，在微信群中采访不同女生，了解她们不欢迎哪种男生。采访对象主要是陌生女性。

开始我以为是不高、不帅、没钱、没事业的男生，因为很多兄弟常这样抱怨。

事实是，我采访了许多情侣，发现条件普通的男生非常多。我们大部分都是普通人。

这一点不影响我们去爱别人。

也不影响我们被别人爱。

一位小姐姐的话让我印象深刻。她说，如果有人对你说要等到某个时候再谈恋爱，那么放心，这个人一时半会脱不了单，而且绝对是脱单困难户。

我不清楚这位女生的经历，但这段话让我反思。

原来注定单身的男生，并非那些长得丑或没钱的，而是那些没有机会出现在女生生活中的男生。胆怯的人永远在做准备。

他们最难脱单，因为女生对你的兴趣窗口总是稍纵即逝。

## 核心观点一：机会不等人 ⏳

上一节我们介绍了胆怯的本质，本节中我们来看看机会的特性。

世界上任何机会都是短暂且转瞬即逝的。它像一个行色匆匆的人站在你面前问：“你要一起走吗？”

它不会问你有没有做好准备。

不会因为你是否优秀而多等你一段时间。

只要你稍有犹豫，它扭头就走。

每个女孩的青春都短暂，她们不得不为自己的未来考虑。她们也害怕碰不到真命天子。

她们会想，为什么没有男生来追我。

为什么他迟迟不行动。

她们也会读一些鸡汤，例如“从不主动的男生就是不够喜欢你”，或“别浪费太多时间在一个人身上”。

胆怯且永远在做准备的人，一直在埋头苦干。

却错过一个个路过的机会，或可怜兮兮地对机会的背影讨价还价：“再给我一点时间好不好。” 对不起，没有时间。

## 核心观点二：实践出真知 🛠️

理解了机会的紧迫性后，我们来看看如何有效“准备”。任何闭门造车的准备都没有意义。

你在感情中出现的问题，只有真正去谈恋爱才能发现。

也只有经历感情的人才知道自己哪里做得不好。

他们说自己没做好准备，说自己还需要努力。

但他们从未问过别人：“你希望找什么样的男朋友？”或“你想和我过什么样的生活？”

他们一直努力成为自己心目中那个“完美”的男朋友：要有钱、有学历、有房有车、有八块腹肌。

与其说你在做准备，不如说你在逃避。因为面对任何机会，只要萌生“让我准备一下，不着急”的想法，你就一定会错过它。

进入社会后，目标一直在变，但目的只有一个：弥补内心的匮乏感，让自己更喜欢自己。例如：
*   我要考上好大学。
*   我要找好工作。
*   我要月入过万。
*   我要买房。
*   我要还清贷款。
*   我要有自己的事业。

一生都在弥补匮乏感的路上，没有尽头。我们永远对自己不满意，永远不认可自己，永远觉得自己没准备好开始一段感情。

这样的人怎么可能轻松开始一段感情？每个你要花百万迎娶的新娘，第一次约会可能都在不过百元的小旅馆。

## 核心观点三：窗口期有限 ❤️🔥

既然闭门准备无效，那么我们应该如何看待感情中的时机呢？没有人会一直等着你。

我们喜欢一个人时，并没有想象中那么“深情”。我们都有窗口期，在这段时间里特别喜欢你。

但如果从你身上得不到积极回馈，我就会心灰意冷地去找别人。

曾有女生问我，喜欢的男生说现在没能力给她更好生活，等将来有能力再来找她，她该等吗？

我的回答是：离开他。因为我们找的是可以共度难关的伴侣，而不是一直在做准备的懦夫。

每个人进入社会后压力会越来越大，麻烦会越来越多。因为未来你需要成为领导、丈夫、爸爸、好儿子。

成长和成熟是每个人必经的过程。

每个在你身边停留的时间都是有限的。如果你抓不住那个人，就要做好永远失去他的准备。

没有经历过，你永远不知道爱情的样子。要想看清人，最起码要多经历。恋爱的段位、心理博弈的基础，不去经历感情，你怎么会懂？

很多人自以为懂，但感情是感性的。一旦感情上来，再多的理论不去实践都是空谈。

**爱情中，输家永远是胆怯的一方。**

没有人可以逼你，只有你自己可以选择想要什么样的生活。人生茫茫，别留遗憾。别等伤过了才追悔，别等错过了才珍惜。
