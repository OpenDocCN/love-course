## 2. 异性朋友众多的女生 👥

接下来，我们看看第二种类型：异性朋友很多的女生。那些看起来很好相处、交友广泛的女生，其实极其不好追。

*   谈恋爱需要顾及对方感受，不如与朋友相处自在。

因此，拥有大量异性朋友的女生，追求起来挑战更大。
