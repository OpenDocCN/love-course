# 恋爱教程 第1课：避免“讨好式”追求 🚫

在本节课中，我们将要学习一个常见的追求误区——“讨好式”追求为何会失败，并了解建立吸引力的正确逻辑。

一位外形帅气、家境优渥的男生，在家人的安排下与一位女生相亲。女生本人比照片更好看，且双方初次见面就聊得很投机，彼此都产生了好感。之后几天，女生会主动发消息，并且回复迅速。

然而，这位男生缺乏恋爱经验，虽然感觉越来越喜欢对方，但两人的关系始终停留在朋友阶段，聊天内容也仅限于日常，无法更进一步。为此，他向表姐请教。

以下是表姐给出的建议：
*   要对女生好，用真诚打动她。
*   要给女生买礼物，送她喜欢的东西。
*   脸皮要厚，每天主动找她聊天。

男生认为这些建议很有道理，并开始执行。他经常送礼物，并每天定时发送早安、晚安。

起初感觉尚可，但后来情况发生了变化。女生发来信息，表示不希望他再发送程式化的问候，并希望退还礼物的钱。女生最后明确表示，自己目前不想谈恋爱，并婉拒了他。

男生感到困惑，他不明白为何最初对自己有好感的女生，态度会突然转变。他自身条件优秀，开局也占据了“天时地利人和”，但最终却遭到了拒绝。

上一节我们介绍了案例背景，本节中我们来看看问题核心。

原因其实很简单：他从一开始就对“如何吸引女生”存在错误认知。他试图通过**单方面讨好**的方式来追求女生。这种不平衡的关系极为廉价，极易导致恋爱关系失败，或使自己沦为“舔狗”。

我举一个非常简单的例子。假设你在某个场合认识一位对你有好感的女生，你打算如何追求她？如果你去询问女性朋友，很可能会得到这样的标准答案：“你要对她好，要真诚，脸皮要厚，天天送早餐、送礼物。”

但如果你真的照做，结果往往是：你在女生心中的初始吸引力假设为 **`80`** 分，每讨好一次，吸引力值可能就会下降一点，例如 **`吸引力 -= 10`**。当吸引力降至 **`50`** 分或更低时，女生就会说出那句经典的话：“你人很好，但我们不合适。”

很多男生想不明白为什么会这样。所有女生都会说“我喜欢对我好的男人”，但当真的出现一个对自己唯命是从的男人时，女生又会说“我对他没感觉，他对我再好也没用”。

追女生的正确逻辑其实很简单。你只需要与她进行良性的**情绪互动**，让她觉得和你在一起很轻松、愉快，然后自然地把她带入你的生活。这样，她才会自然而然地喜欢你。如果你做不到这一点，无论你对她多好、送多少礼物，都无济于事。

本节课中我们一起学习了“讨好式”追求的弊端。其核心在于，单方面的付出无法建立真正的吸引力，反而会降低自身价值。建立吸引的关键在于平等的情绪互动与舒适的相处体验。
