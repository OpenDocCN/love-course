# 恋爱避坑指南 01：警惕成为感情中的“供养者” 💸

在本节课中，我们将要学习一个在恋爱关系中需要警惕的角色——“供养者”。我们将剖析其定义、特征、危害，并探讨如何避免陷入这种不平等的相处模式。

## 什么是“供养者”？🤔

上一节我们提到了“供养者”这个概念，本节中我们来详细看看它的定义。

所谓“供养者”，是指在恋爱关系中，主要通过为女方花钱、提供物质资源来维系双方关系的一方。

这种“供养关系”通常具备两个核心特征：
1.  女方会或明或暗地向你索取钱财。
2.  你感受不到女方对你的真正喜欢。

## “供养者”的真实案例 📖

为了更清晰地理解，以下是两个典型的“供养者”案例。

**案例一：**
一位月薪约5万元的男士，其女友是一名刚毕业的大学生。男士非常喜欢这位女生，但女生并不喜欢他。于是，男士通过持续为女生花钱来维持关系，包括购买各种礼物、频繁转账，甚至为女生偿还花呗和信用卡。他每月5万元的收入，有4.5万元都花在了女友身上。

**案例二：**
一位40多岁在海外打工的男士，娶了一位离异带娃的女性。男士将每月的工资卡上交，但其妻子将所有钱都花在自己和女儿身上，从不给他花。男士在外工作辛苦，吃穿用度却很差。本质上，在这位妻子心中，他只是一台赚钱的机器。

以上便是典型的“供养者”处境。

## 为何“供养关系”难以长久？⚠️

了解了“供养者”的表现后，我们来看看这种关系为何注定不稳定。主要有以下三个原因：

**1. 缺乏情感基础**
女方从根本上并不喜欢“供养者”，只是将其视为临时的提款机来获取金钱。当捞取的利益差不多时，便会找理由分手。

**2. 性价比极低**
这类似于一种不对等的交易。例如，有男生在6个月内为女生花费10万元，期间同居10次，相当于每次花费1万元。这好比租车，市场价格是500元一天，但因你特别喜爱这辆车，对方便坐地起价至1万元一天。时间长了，你必然无法接受。更何况，你是按月“交钱”，却享受不了几次“服务”。

**3. 期望与回报严重失衡**
在这段关系中，你喜欢对方，但对方不喜欢你。你投入大量金钱，自然期望获得情感回报，如对方的关心和体贴。然而，对方不仅不会给予，反而会越来越冷淡。最终，你看不到希望，只能主动放弃，而之前的所有花费都已付诸东流。

## 如何避免成为“供养者”？🛡️

前面我们分析了“供养关系”的弊端，那么如何避免陷入这种困境呢？核心在于**掌握价值的主动权**。

众所周知，女性和男性一样，都喜欢钱。但**将钱给予对方，并不等同于能吸引对方爱上你**。

关键在于，你的钱和资源应牢牢掌握在自己手中，并以此展示吸引力，而非直接赠予。例如：
*   你购买了一套房子，女方与你在一起时能够居住。一旦分手，房子与她再无关系。
*   你购买了一辆车，女方与你在一起时能够使用。一旦分开，车也与她毫无瓜葛。

反之，如果你花钱为女方买房、买车，这无疑是极不明智的。结果可能是，女方住着你买的房、开着你买的车、背着你送的包、用着你送的口红，却去与别人谈恋爱。

**价值 = 掌握在自己手中的资源**
这个道理很简单：当你手里拿着包子时，狗会一直跟着你。但如果你把包子给了狗，它吃完马上就会跑掉。
