# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：
```python
message = "你好，世界！"
```
这个变量名 `message` 就是盒子的标签，之后我们可以通过这个名字来使用盒子里的数据。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的类型。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其说明：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由一系列字符组成的文本，需要用单引号或双引号包围，例如 `‘Python’`, `“学习”`。
*   **布尔值**：只有两种取值，`True`（真）或 `False`（假），常用于逻辑判断。

## 如何查看与转换类型？ 🔄

在编程时，有时我们需要知道一个变量的类型，或者在不同类型之间进行转换。

我们可以使用 `type()` 函数来查看任何变量或值的数据类型。例如，`type(10)` 会返回 `<class ‘int’>`，表示这是一个整数。

以下是两种常见的类型转换操作：

*   **转换为整数**：使用 `int()` 函数。例如，`int(3.9)` 的结果是 `3`（注意，这是直接去掉小数部分，并非四舍五入）。
*   **转换为字符串**：使用 `str()` 函数。例如，`str(100)` 会将数字100转换为字符串 `‘100’`。

## 动手试一试 ✍️

理论需要结合实践。现在，让我们在Python交互环境中尝试定义变量并进行类型转换。

请打开你的Python解释器或开发环境，输入以下代码：
```python
# 定义变量
name = “楚儿”
age = 3
price = 9.9
is_learning = True

# 查看类型
print(type(name))
print(type(age))

# 类型转换
age_str = str(age) # 将整数年龄转换为字符串
print(“我的名字是” + name + “，我” + age_str + “岁了。”)
```
运行这些代码，观察输出结果，这将帮助你巩固对变量和数据类型的理解。
