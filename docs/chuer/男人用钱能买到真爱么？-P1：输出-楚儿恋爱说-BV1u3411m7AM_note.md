# 💰 男人用钱能买到真爱吗？ - 课程01：输出篇

在本节课中，我们将通过一个真实的故事，探讨金钱在感情中的作用，并分析其背后的深层逻辑。

---

给你200万去追一个30岁、月薪2000的普通女生，你能追到吗？先别急着回答。这里有一个真实的故事。小李出身县城，20来岁来到大城市打工，认识了小几岁的果果。由于两人的背景条件相似，他们很能聊到一块去。下班后，他们经常一起去吃路边摊、吃宵夜。时间久了，小李开始喜欢果果，但他觉得自己条件不好、没什么钱，于是将这份感情隐藏在了心底。后来，他们各自找了其他工作，开启了不同的人生。

两人的交集越来越少，直到后来彻底断了联系。一晃十年过去，小李总算熬出了头，赶上了好机遇，摇身一变成为身家千万的小老板，也算是扬眉吐气了。一天，他突然收到了果果的一条群发短信，内容是请求帮忙“砍一刀”之类的网购助力。于是，小李便和果果聊了起来。聊天中，他发现果果这些年过得并不好，工作一般，而且还没结婚。得知小李如此成功后，果果先是一番恭维，然后开始倾诉自己过得如何不如意。

这样一来二去，两人又重新熟络起来。时间仿佛回到了十年前，只不过这次小李已不再是那个只能吃路边摊的穷小子。又过了一段时间，小李借口出差做生意，顺便到外地看望果果。两人把酒言欢，聊起过往。小李当初对果果的那份感情，又被重新点燃了。当果果说出自己这么多年连一套属于自己的房子都没有时，小李直接给她打了100万，说：“买房子去吧。”随后，他又给了果果一张存有100万的卡，说：“拿去随便花。”

小李本以为这样果果肯定会投怀送抱。结果，他什么都没换来。小李回到家后，看着手机上不停发来的消费短信，越想越生气，越想越憋屈，觉得自己就是一个冤大头。于是，他找到我说：“这是个坏女人，我怎么样才能把钱要回来？”

许多年轻人总是抱怨女人太现实、只认钱，认为“如果我有钱了，就能如何如何”。在他们眼中，自己之所以没人喜欢、没有自信，就是因为没有钱。确实，**钱是英雄胆**。但他们从未想过，自己身上除了钱，还有没有其他地方值得女生去爱。

所以，幸亏你现在还没钱。如果有钱，也可能被你挥霍光。这样的例子还少吗？小会计挪用公款，疯狂打赏女主播，最后锒铛入狱；创业者成功后网络相亲，最终被骗自杀；彩民中了大奖，反而妻离子散、家破人亡；拆迁户吃喝嫖赌，败光所有拆迁款。

一个人真正的成熟，来自于内心世界的成长。一个真正爱你的人，是先爱你这个人，然后才爱你的其他条件。光想着有钱就能获得真爱的人，就像一个孩子，觉得“我长到1米8了，那我就是大人了”。但即便你真的长到1米8，你的内在也远远支撑不起这个高度。你还需要花更多时间去学习、去磨砺、去碰壁、去真正地自我成长。

总有一天你会明白，真正让你被爱、能留住人心的，不是你的1米8身高，也不是你的200万，而是你**人格魅力的万丈光芒**。
