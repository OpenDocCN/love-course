## 第一点：避免过度讨好，保持关系平衡⚖️

上一节我们介绍了课程概述，本节中我们来看看追求初期的首要原则。

追求初期可以表现出青涩或木讷，但切忌跪舔或过度讨好。恋爱包含博弈成分，过度放低自己会导致感情天平失衡。公式如下：

**失衡关系 = 过度付出 + 自我价值降低**

这种行为会让女生将你的付出视为理所当然。即使最终在一起，不平等的关系也可能持续，这是必须避免的大忌。

## 第二点：保持自我，展现真实人格🌟

在接触初期，应尽可能做真实的自己。

以下是具体建议：

*   不必展示100%的自我，但展现的部分应体现个人意志与三观。
*   避免得失心过重，唯唯诺诺并不吸引人。
*   有思想、有主见的男生更具吸引力。
*   行为举止需体现素养，尊重彼此想法。
*   以平常心看待关系，克服紧张情绪。

## 第三点：有的放矢，及时止损🎯

恋爱需要明确目标并懂得适时停止。

你需要主导关系走向，同时大方自然地做自己。核心在于：

**关系健康度 = 尊重对方思想 + 尊重自我想法 + 保持理智**

应考察双方三观是否相似。若发现确实不合适或对方无心交往，需为自己设置停损点，及时退出。

## 第四点：线上互动：分享感受，避免问答式聊天💬

见不到面时，线上互动至关重要。

互动应生活化、自然化，以营造舒适感。例如：

```python
# 良好互动示例
if see_scene_reminds_of_her:
    send_photo_with_message("看到你最喜欢的花，就想到了你。")
```

互动以分享感受为主，避免评论对方观点。应先倾听，再就相关话题展开。

## 第五点：线下推进：创造机会，升级关系🚀

线下约会是将关系推向深入的关键。

以下是推进关系的步骤列表：

1.  **形象管理**：保持干净整洁，提升内外形象以增加自信。
2.  **发挥优势**：如果不擅言辞，就发挥个人特质（如运动、设计、博学等）来增强吸引力。
3.  **规划约会**：首次约会后两周内，计划2-4次约会以推进好感。
4.  **关系测试**：通过互动小游戏（如比手掌大小）测试对方好感度。
5.  **肢体接触**：暧昧到一定程度后，可从手指、手臂等部位开始自然接触（如过马路时牵手）。
6.  **最终确认**：当肢体测试顺利后，在浪漫氛围下通过拥抱、接吻升级关系，随后表白。

女生肢体上的认可比口头认可更重要。她更在意你的真诚、喜欢她的态度、内在担当与上进心。

你需要把握约会机会引导关系升级，并让女生投入参与，这才是“攻心为上”的关键。
