上一段我们提到了人口比例，本节中我们来看看具体的社会现象如何形成。

*   **要求与现实脱节**：部分女性要求同龄男生在结婚时必须拥有房、车等物质条件。

这就形成了第一波滞留。

这就导致了第二波滞留。
