## 女生的择偶倾向

影响女生择偶倾向主要有两个辨别因素：**长期关系**与**短期关系**，以及男生自身的**资源和品质竞争力**。

所谓**长期关系**，通俗讲就是以结婚为目的的恋爱关系。

对应的**短期关系**，则是指“火包友”或随便玩玩的那种关系。对于女生而言，在短期关系中，最吸引她们的男性特征通常是“长得帅”、“有钱就行”。相反，如果一位女生足够善良和长情，想认认真真谈一段恋爱，她们会更倾向于男生的另一些特质。

“奶”说白了就是适当的撒娇，而且“奶”和“硬”刚好互补。“硬”是常态，“奶”就是偶尔的甜点。

最后一个特质就是“放得开”，所谓就是自信。
