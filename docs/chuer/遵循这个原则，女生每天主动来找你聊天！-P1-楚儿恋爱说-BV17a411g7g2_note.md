## 理性聊天的陷阱：为何你的话题总是冷场？🤔

上一节我们提到了聊天中的常见困境，本节中我们来看看导致这种困境的核心原因——“理性聊天”。

所谓理性聊天，是指带有强烈目的性、以解决问题或询问信息为主的对话方式。这种聊天**枯燥乏味**，缺乏舒适感，无法产生情绪价值。女生的感受是越聊越无趣，甚至感到烦躁。绝大多数尴尬收场或聊不下去的情况，根源都在于此。

以下是理性聊天的典型表现：
*   以询问信息为目的，例如：“你在干嘛？”
*   采用一问一答的模式，例如：“看什么电视？”
*   对话缺乏情绪起伏和趣味性。
