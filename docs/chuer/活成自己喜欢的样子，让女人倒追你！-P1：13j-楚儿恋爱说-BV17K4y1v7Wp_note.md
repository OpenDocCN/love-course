在本节课中，我们将要学习如何客观地认识自己在感情中的现状，并理解阻碍你获得理想关系的核心问题。我们将剖析常见的思维误区，并为你指明自我提升的起点。

你18岁时放学回家，村里有好几个妇女在你家门口聊天。她们扯着大嗓门，声音一个比一个大，从东村的单身汉聊到西村的寡妇。从婆媳之间的不和，谈到子女的种种琐事。你关上门戴上耳机，一句话也不讲。你在心里想着，烦死了一群怨妇。

28岁时，你每天上着朝九晚五的班，拿着固定的微薄工资，没有任何工作的激情。朋友都成家立业了，你还是一个单身狗，被家里逼着去参加相亲，碰壁碰得灰头土脸。周末就在家睡觉，饿了就叫外卖，吃饱了继续玩手机。这个时候的你，敢让你未来的女人看到吗？你喜欢现在的自己吗？

上一节我们描绘了一种可能存在的消极状态，本节中我们来看看导致这种状态的三个核心误区。

以下是三个阻碍你获得理想感情的核心误区：

1.  **误区一：将问题归咎于外部条件**
    很多兄弟理直气壮地说：“现在的女人都太物质了，找什么狗屁爱情，单身也挺好。”“我到底做错了什么？比我丑的一大堆都成家了，我连个女朋友都没有。”“那哥们儿家里有钱，还有女的跟他，怪不得我爹没本事。”
    你不是富二代，也不是大帅哥。很多兄弟认为有钱了才能有女生喜欢，长得帅才能吸引小姑娘。可是你有钱吗？未来一定会发达吗？你很帅吗？打算整容吗？很多普普通通的兄弟处境并不好，你们没有做到想象中的那样迎娶白富美、当上CEO、登上人生巅峰。一旦放假，就想把自己关起来，打一局游戏找找激情，不然会无聊死。冷静下来想一想，空虚吗？但凡有点良心，觉得对得起自己吗？想想那个刚入社会的你，是这样想的吗？
    **核心公式：吸引力 ≠ 财富 + 外貌**
    正是因为我们不高、不帅、也没钱，所以才更需要去学习怎么跟女生相处、怎么谈恋爱。我们提供不了物质价值，难道还不能提供情绪价值吗？不然人家为什么要选择跟你在一起？你去找一个女生问问，一个男生相处起来舒服值多少钱？会哄女生值多少钱？能给她带来快乐又值多少钱？

2.  **误区二：无意中给自己营造了消极的圈子**
    我接触的人很多。一部分是我在做情感导师这五年中接触到的形形色色的人，另一部分是我深入交情的粉丝和学员。很讽刺的是，成天喊着“等有钱了就谈恋爱”的兄弟，往往还是没有女朋友，也没什么钱。感情中那些喜欢抱怨的人，往往层次都很低。你抱怨完了，问题还在那里，并没有消失。
    并且抱怨越多，现在过得越差。那些“忙到没有时间谈对象”的兄弟，往往有时间躺在家里睡觉、点外卖、打游戏、喝点小酒、吃点花生米、跟兄弟们吹吹牛逼、吐吐槽。你在无意中给自己营造了一个很差的圈子，身边是一帮不思进取、爱抱怨爱吐槽的兄弟。在这样的氛围之下，还找什么女朋友？找了也没结果。
    可是那些烦人的问题真的消失了吗？并没有。它只是在找一个合适的时机出现，给你一记狠狠的报复，毁掉你的最后一次斗志。

3.  **误区三：对待爱情的态度既渴望又嫌弃**
    我知道你为什么要跟我抱怨女人的物质，抱怨那个兄弟出身好、娶了白富美。因为你只想要好的，不能接受坏的。你也想要美好的爱情，可是你不想学怎么去获取女孩子的心。你也想要超级高的工资，可是你不愿意加班、不愿意超额付出。你也馋人家的身子，你还嫌人家很麻烦。
    总而言之，你没弄明白任何事情都有两面性，每个人也都有每个人的烦恼。你只是还不够成熟罢了。王宝强现在有钱，照样被欺骗、被背叛，一样也有感情烦恼。贾乃亮长得又高又帅，一样也被背叛，离婚收场。
    **核心代码逻辑：**
    ```plaintext
    if (只索取，不付出):
        result = 感情失败
    elif (既渴望，又嫌弃):
        result = 原地踏步
    else: # 正视问题，主动学习与付出
        result = 开始成长
    ```
    你虽然是个普通人，但是你不能输给你自己。
