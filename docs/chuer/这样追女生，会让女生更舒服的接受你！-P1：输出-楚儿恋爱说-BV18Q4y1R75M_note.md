1.  首先提出一个高难度请求（例如，邀请对方进行一场长途旅行）。
2.  预料之中地被拒绝后，再提出你真正的目标（例如，“那一起吃顿便饭总可以吧？”）。
3.  此时，对方因已拒绝过一次，出于补偿心理，接受这顿“便饭”邀约的可能性会大大增加。
