## 把握共性：从共同点切入对话

上一段提到了开启对话的常见误区，本节中我们来看看如何避免“查户口”式的尬聊。关键在于找到你与对方都感兴趣的区域，从双方熟知的话题入手。这样既能让你聊得自然，也能让对方更愿意回应。

对于刚认识的两个人，创造共性最简单的方法是从**共同好友**入手。

以下是利用共同好友开启对话的一个例子：

```text

女生说：“嗯？他怎么会突然和你提起我呀？他怎么说的？”
男生说：“哎，没有没有。是你们上次出去玩啊，不是遇到一件有意思的事儿吗？正好呢，他就跟我提起来了。”
女生说：“哦，这事儿啊！我跟你讲，当时是这样的……”
```

从共同好友切入话题，首先不易引起对方反感，其次能激发对方的好奇心——人们通常想知道别人如何评价自己。这样，对话便能自然地展开。

但需注意，共同好友话题仅是开启聊天的契机。我们的目的是恋爱而非单纯交友。因此，在几个回合的交流后，应自然地将话题引导到你们两人身上，进行更深入的了解。

其实，任何相识场景都能找到共性话题。例如，搭讪认识可以从当时场景聊起；聚会认识则可以聊聊聚会上的趣事。

**总结：共性话题千千万，聊对方向是关键。**

## 寻找个性：围绕她的兴趣展开对话

在建立了初步联系后，我们需要让对话持续深入。本节我们将探讨如何通过关注对方的个人兴趣来达成这一目标。这里的“个性”并非指展示你特立独行的一面，而是指通过女生的微博、朋友圈等社交平台，了解她的兴趣点，并由此入手展开聊天。

人们总是更关注与自己相关的内容。如果你一直围绕自己聊天，对方很容易失去兴趣。因此，聊天时应**把话题引到对方身上**，并创造互动。

以下是一个围绕对方兴趣展开对话的例子：

```text
女生说：“是啊，怎么你也喜欢吗？”
男生说：“一直知道他但是没怎么听过他的歌。你有没有什么推荐的歌呀？”
女生说：“啊，我这有歌单可以推荐给你啊！”
男生说：“诶，最近好像有他的演唱会诶，要不要一起去看看呀？”
```

女生通常很乐意与人分享并“安利”自己喜欢的事物。聊天时，围绕她喜欢的内容进行，即使你了解不深也没关系。关键在于引导她滔滔不绝地讲述，这样对话就不会陷入僵局。

不过，运用此方法时话题一定要适度。如果你只懂皮毛却夸夸其谈，在她面前可能会显得班门弄斧。
