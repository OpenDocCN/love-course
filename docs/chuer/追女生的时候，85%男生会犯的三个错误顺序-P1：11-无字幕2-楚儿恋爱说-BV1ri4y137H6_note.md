# 恋爱教程 01：理解吸引力开关与关系的正确阶段 🧲

在本节课中，我们将要学习两性吸引力的核心机制，以及建立恋爱关系的正确阶段顺序。许多人在追求过程中犯错，往往是因为颠倒了这些阶段的顺序。

## 吸引力开关：男女的差异 🔄

地球上每个男人与女人的择偶机制，都存在着一些可以量化的吸引力开关。当你看到时装杂志上面的泳装模特时，会产生出喜欢的感觉，并且有生理反应，就是因为它触动了你的吸引力开关。

同样，男人也可以去触动女生的吸引力开关，让她们感受到你的男性魅力。但是，男女双方之间的吸引力开关是不一样的。

*   **男生的吸引力开关**：男生大多数对女生的**繁殖价值**有反应，只有少数对**生存价值**有反应。
*   **女生的吸引力开关**：女生大多数对男生的**生存价值**感兴趣，只有极少的一部分对男生的**繁衍价值**感兴趣。

举个例子，男生都喜欢脸蛋好看、身材好看的女生。即便你知道很多好看的小姐姐浑身上下都是做的，可是更多情况下依然是“卑微相近大女种”。否则，现在的医美整形行业也就不会这么吃香了。

女生会对男生哪些特质产生吸引呢？健康身材、腹肌、身高、财富、社会地位等等。其实很多情况下，大家忽略了一个强度很高的吸引力开关，就是**预选**。

这个开关不仅存在于人类的择偶机制中，也同样适用于很多的动物。有人曾经做过一个非常有趣的实验：落单的公鸡很难吸引母鸡，但是在它的身边放上一个假的母鸡，其他母鸡就会迅速地进入到它的地盘当中。

同样的，如果一个女生认为某个男生对其他女生具有极强的男性魅力，可能会立刻觉得被他吸引。

## 恋爱关系的三个阶段 📈

上一节我们介绍了吸引力的来源，本节中我们来看看关系发展的必经之路。两性情感当中，两个人在一起确定关系之前，要经过三个阶段：

1.  **吸引阶段**
2.  **熟悉感与舒适感阶段**
3.  **诱惑阶段**

你要能够吸引到人，才有可能发生后续的故事。不过在诱惑被你吸引的女生之前，有一个很重要的附加阶段，那就是建立舒适感。正常的人类都会选择和自己气场相近、感到舒服的人在一起，无论男女。

下面为了更容易让大家清楚明白，我把每个阶段进行细分。

当两个人加上微信，无论是你去搭讪、社交软件还是其他方式途径认识女生，最开始的阶段就是**吸引阶段**。人与人之间刚认识的时候，也是建立吸引的最佳时期，因为第一印象很重要。当你对一个人的第一印象已经定型的时候，那么后期想要扭转这种印象，是一件很困难的事情。

后续呢，你们两个人之间的交流、经历过程中在一起经历过的事情，这个过程就是**建立舒适感的阶段**。

最终，两个人关系进一步的升高，确定关系在一起之前，所称之为**诱惑阶段**。而诱惑是互相的，通俗一点讲就是互相的提供价值，为彼此的投入和付出，比如时间成本、金钱成本，甚至是感情成本等等。

如果你可以和女生建立了足够的舒适感，能让她在中期阶段愿意和你一个人在房间里独处，那么她就可能会在结尾阶段同意和你在一起，甚至可能会诱惑你。所以，当你努力制造机会让她诱惑你，就不必太花心思在诱惑本身。

但是，这一切的前提呢，都是在具有足够吸引力的前提下才会发生的事情。如果没有吸引力的存在，女生当然会拒绝我们苦心经营的舒适感，拒绝我们的示好。

其实呢，当一些好看一点的女生从小到大，经历过无数的“好好先生”以及“舔狗”的追求与奉承，已经社会化习以为常了。

**拿出你们的小本本记下来划重点：如果你太早的去寻求舒适感、盲目的升高关系，只会降低吸引力，甚至结果更糟糕。**

两个人在一起的正确顺序是：从认识女生建立吸引，到中期的建立熟悉感、舒适感和联系感，最后的阶段才是营造被诱惑的机会。也就是说：

**吸引力 + 舒适感 = 诱惑机会**

但很多新手同学们本末倒置搞错顺序，只想着能够一步到位。

## 四种常见的错误顺序 🚫

以下是新手在追求过程中最容易弄错的四种阶段顺序。

### 错误一：先诱惑（从结尾开始）

新手兄弟们最常犯的错误，就是在还没有吸引到女生之前，就集中精力来诱惑她。这就好比把马车放在马面前。

抢先诱惑，尤其是对于高分好看的女生来说，就相当于是在说：“我们虽然不是很熟，但你愿意和我在一起吗？慢慢的就熟了。”很多男生误从结尾开始，就像普通人一样，完全无视女生的舒适感。

即便有一部分的男生长得很帅，对于女生来说有一定的吸引力，也抵挡不了女生的不舒适感。因为他们必须去应付那些既不熟悉、又不信任的人所施展的积极追求。

**这里建议大家：千万不要在结尾阶段之前就精虫上脑、色迷心窍。要记住，欲速则不达。**

### 错误二：先寻求舒适感（从中间开始）

很多接触过一些情感知识，甚至稍微聪明一点的男生，都会了解抢先诱惑会让女生感觉到极度的不舒适。所以他们的做法是先集中精力去建立舒适感。

可是这些人不懂的是，好看的小姐姐们天天被那些所谓的“好好先生”们疲劳轰炸，搞不好都要被烦死了。她们得应付太多粘上来的“舔狗”，实在是没有空去一而再、再而三的耐心回答那些老套的对话了。

“好好先生”们虽然没有前者那么讨人嫌，但是同样不会招女生的喜欢。如果一个男生一开始就没有吸引力，只会说“嗨你好，我是XXX，你叫什么名字？”，很大概率会被像前一种“好人”那样被忽视。

原因在于：一个对你完全不感兴趣的女生，干嘛在乎或者记住你的名字？难道只因为你的发问，她就一定要透露自己的个人资料信息吗？

如果女生对你不感兴趣，企图建立舒适感只会让女生感觉到更加的无聊。向她进行自我介绍、引导女生进行自我介绍、太早的开口称赞她的行为，都是在企图建立舒适感。

设想一下，你们和一个自己完全不感兴趣的人聊天，而且这种情况会出现很多次甚至几年，你们就会感觉到有多别扭，有多理解女生的心情了。

### 错误三：吸引而没有舒适感（从起点开始，跳过中间段直接到结尾）

每个起点到终点之间都有一个中间的过渡存在。在情侣相处交往的过程中，总会有从建立吸引到诱惑的契机存在，但前提是你要先营造出建立舒适感的机会。

很多男生条件其实不错，也能够获得女生青睐，但他们往往会出现这三个问题导致无法继续：

1.  无法把互相吸引正常化（急于推进）。
2.  忽略了舒适感的存在（让人感到不安）。
3.  女生出现反悔的情况（因不适而后撤）。

### 错误四：只有吸引与舒适感（从起点开始，直到困在中段）

相信有很多大兄弟听到过这样一句话：“我觉得你人挺好的，但我们还是做朋友吧。”或者“你对我就像哥哥一样好。”

“做朋友”这个词听起来真的很讽刺。在我看来，其含义是指“害怕向花了时间去相处的女人表达爱慕之意的男人”。

如果你无法在舒适感阶段向女生透露出你是一个潜在的爱慕者，那有些女生可能没有那么强的直觉能发现你。在舒适感阶段，如果你没能牵到她的手或者是进一步升高关系，可能会永远困在这个阶段。

事实上，女生会希望你一直和她处于这样的舒适感阶段，因为她们不需要承担太多的压力和心理负担。以至于当你过渡到诱惑阶段时，会听到她说“我们还是做朋友就好”。这也就是说明了你困在了友谊区。

## 总结 📝

本节课中我们一起学习了：

1.  **吸引力开关**：男女被吸引的核心价值不同（男重繁衍价值，女重生存价值），并引入了**预选**这一强力开关。
2.  **关系的三个阶段**：**吸引** -> **舒适感** -> **诱惑**，这是不可颠倒的正确顺序。核心公式为：**吸引力 + 舒适感 = 诱惑机会**。
3.  **四种常见错误**：切勿（1）未吸引先诱惑，（2）未吸引先求舒适感，（3）有吸引却跳过舒适感，（4）停留在舒适感无法进入诱惑阶段。这些错误都会导致关系失败或陷入“友谊区”。

如果你在感情面前总是受伤失败，你想提高自己的恋爱技巧、摆脱单身或者是挽回的话，对于恋爱即便你现在一无所知、迷茫困惑，那么也不要恐惧，因为学习可以让你成长。
