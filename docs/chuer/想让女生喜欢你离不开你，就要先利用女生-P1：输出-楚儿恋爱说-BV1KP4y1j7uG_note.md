# 恋爱技巧课 P1：利用“付出”建立吸引 👫

在本节课中，我们将要学习一种建立情感连接的策略：通过引导对方为你付出，来增加她对你的好感和依赖。这种方法的核心在于创造互动和“亏欠感”，而非单方面讨好。

上一节我们介绍了课程主题，本节中我们来看看这一策略的具体应用案例。

## 学员的实践案例 📖

以下是学员通过引导女生付出，成功建立关系的四个具体例子。

1.  **学霸女友**
    *   学员在大学时喜欢一位学霸女生。
    *   他故意挂科一科，以请教学习问题为名接近她。
    *   每次请教后，他会给女生买杯奶茶作为答谢。
    *   补考通过后，他邀请学霸女生吃大餐庆祝。
    *   最终两人成为情侣。

2.  **非主流女友**
    *   学员在网吧玩《英雄联盟》时，注意到一位游戏技术很好的非主流女生。
    *   他主动请求女生教他玩游戏。
    *   作为答谢，他给女生的游戏账号购买了一个皮肤。
    *   两人经常一起玩游戏，关系逐渐升温。

3.  **女同事**
    *   学员的同事负责商务文案工作。
    *   他常请这位女同事帮忙排版和检查方案。
    *   他看见女同事吃包子时说好吃，并请她次日为自己带两个。
    *   完成一个大项目后，他以答谢为由邀请女同事吃饭。
    *   最终两人发展为恋爱关系。

4.  **最终的老婆**
    *   学员在一次线下活动中认识了一位爱追剧的女生。
    *   他先与女生一起追剧、讨论剧情。
    *   随后，他借口要加班出差，请女生先看，并拜托女生在他忙完后为他讲解落下的剧情。
    *   之后他邀请女生吃饭答谢，两人最终步入婚姻。

看完了学员的例子，我们再来看看这一方法在其他人身上的应用。

## 核心原理与总结 💡

本节课中我们一起学习了通过引导付出建立吸引的方法。
