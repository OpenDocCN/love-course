## 核心方法：“杯子效应”原理与操作

上一节我们介绍了常见的兴趣信号，本节中我们来看看一个更主动的试探技巧——“杯子效应”。

你可以通过无意中拉近彼此杯子的距离来观察反应。从心理学角度看，杯子与嘴唇关系密切，杯子的距离能反映人际关系的亲疏。关系越亲密，杯子的距离通常会越近。
