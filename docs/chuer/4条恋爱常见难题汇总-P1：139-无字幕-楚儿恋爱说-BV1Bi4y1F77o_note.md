上一节我们介绍了课程的整体目标，本节中我们来看看第一种常见游戏：“钓胃口”。

## 1. 游戏一：钓胃口 🎣

**应对法则**：核心在于**不让她感觉占据了主动权**，并表现出她的“调情”对你没有“钓”的作用。

*   **保持冷静**：即使对方很有吸引力，也要装作认同她的任何理由或拒绝。
*   **引发好奇**：你的平淡反应会让她不习惯，可能反而激发她的征服欲。

了解了如何应对“钓胃口”，接下来我们看看第二种测试你底线的方式：“假装吃醋”。

## 2. 游戏二：假装吃醋 😠

在这种模式下，她会在你面前与其他男性表现暧昧。目的可能是让你感到不安，或测试你在遇到“竞争”时是否会退缩。本质上，**她是在观察你的反应**。

**应对法则**：反应必须**坚定且自信**，传达出“我不是弱者”的信号。

*   **坦诚沟通**：将她带到一旁，认真沟通此事。

处理了外在的“竞争”测试后，我们进入第三种更复杂的游戏：“口是心非”。

---

最后，我们来剖析一种通过控制沟通节奏来施加影响的游戏：“欲擒故纵”。

## 4. 游戏四：欲擒故纵 ⏳

具体表现为：不立即回复你的电话或信息，而是延迟回复。目的是**避免显得过于渴望或需求感过强**，她可能有意，但不想吓跑你。

**应对法则**：**镜像她的行为节奏**，并进行**最终测试**。

以下是具体操作步骤：

*   **镜像法则**：她延迟回复，你也适当延迟回复，不要显得急迫。
*   **终极测试（阴智法）**：如果她长时间不回复，几天后可以再联系一次，询问是否收到信息。
*   **果断决策**：如果她依然不回复，**公式**可总结为：`无回应 = 无兴趣`。此时应果断放弃，寻找下一个目标。

本节课中我们一起学习了四种常见的女性心理游戏及其应对策略：
1.  **钓胃口**：保持冷静，不露需求感。
2.  **假装吃醋**：坦诚沟通，设立底线。
3.  **口是心非**：明确兴趣，展示高价值与选择权。
