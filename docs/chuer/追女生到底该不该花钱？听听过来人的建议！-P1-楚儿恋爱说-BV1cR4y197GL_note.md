## 原则一：为共同体验投资 🍽️🎡

这种花费的问题在于，你完全将钱花在了她个人身上，自身缺乏参与感。长期如此，容易破坏关系的平等性。

**公式**：`单向物质付出 → 关系失衡 → 你变得卑微，她变得高傲`
