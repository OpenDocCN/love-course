# 和女生聊天没话题尬聊，宁愿先不

在本节课中，我们将要学习当与心仪女生聊天陷入“无话可说”的尴尬境地时，如何正确应对。我们将从两性心理的角度分析问题根源，并提供三种实用的方法来转变话题、保持吸引力，从而让聊天自然流畅地继续下去。

## 错误认知：无话可聊不等于不会聊天

首先，我们需要纠正一个常见的错误认知。与女生“无话可聊”，并不单纯意味着你不会聊天或聊天内容无趣。这种状态是每一段关系发展过程中都可能出现的一种情况。

只不过有的关系出现得比较早，有的出现得晚；有的在问题爆发后才出现，而有的则在关系进入厌烦阶段前就得到了改善。

## 识别危险信号：交流动力下降

上一节我们介绍了“无话可聊”的本质，本节中我们来看看如何识别关系中的危险信号。当你们发现双方失去交流动力时，危险信号就已经出现。

例如，平时很主动的女生突然不再主动发起聊天，或者每次都需要你主动寻找话题，而你突然也不想再主动找话题了。如果不加以改变，问题会继续恶化。

那么，你的内心或女生的内心，就会感到这段交流“没什么意思”，甚至产生孤独感。结果要么是双方陷入沉默，觉得没有东西可交流；要么是一方挣扎着硬拉另一方聊天，从而导致对方产生厌烦情绪。

## 核心策略：在饱和前转变话题

为了让女生不感到厌烦，甚至越来越喜欢与你聊天，我们必须掌握一个核心策略：在聊天话题开始饱和之前，主动进行转变。我们需要用新的内容重新激发女生的好奇心，提供新的话题。

以下是三种实现话题转变的有效方法，请记好笔记。

### 方法一：增加关联性话题

聊新话题固然会带来新鲜感，但如果在转换话题时没有注意**关联性**，就会导致话题转变过快，女生同样会迅速进入厌烦期。

**举例说明：**
假设前十分钟你们在聊“前任”这个话题，已经聊得有些烦了。你觉得应该换话题，但没有任何铺垫，就直接跳到“意大利面要配上惠灵顿牛排”。女生会觉得完全接不上你的话题，于是索性不接话。即使接了，也可能会敷衍地说“你说的对”。

### 方法二：交出聊天主动权

把聊天的主动权交出去，引导她去聊自己感兴趣的事情。最常见的方式就是提问。

**提问示例：**
*   你喜欢小动物吗？
*   你平时喜欢吃什么东西？
*   你最近都在看什么？

或者，聊一些与对方直接相关的话题，让对方自己找到聊天的动力。这样做有一个明显优势：即使聊天进入了平淡期，对方也不会因此厌烦你，因为“问题”是出在她自己身上（她对自己感兴趣）。

### 方法三：主动学会“聊废话”

“聊废话”是一种高情商的聊天方式，它几乎不会消耗对方的耐心，反而能让女生越聊越开心。

**对比示例：**
女生发了一张自拍问你：“我这身衣服好看吗？”
*   **正常聊天（终结式）：** “好看” 或 “不好看”。
*   **“聊废话”（互动式）：** “你想听真话还是听假话？”

女生可能会说“先听假话”。这时，我们就可以先夸一下：“假话是，你穿什么都好看。” 听到这个夸奖，女生会很开心，但知道是假话，她只会开心一半，从而对“真话”更感兴趣。

此时，我们再给出真话：“当然是越性感的越好看。” 兄弟们可以感受一下，明明一句话就能聊完的无聊话题，经过一些小小的“作对”和互动，就会变得很有意思。

## 最后提醒：适时保持距离

最后提醒一下兄弟们，如果实在无话可聊，保持距离、暂时冷淡也是一种不错的选择。当你感觉到无法推进关系时，停一停、休息一下，然后再去“进攻”，效果反而会更好。
