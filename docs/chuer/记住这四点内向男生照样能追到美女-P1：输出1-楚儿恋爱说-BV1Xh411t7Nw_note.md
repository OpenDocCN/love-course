## 第一点：摒弃无效观念 🚫

上一节我们介绍了课程概述，本节中我们来看看需要摒弃的第一个错误观念。

不要相信“只要好好挣钱，自然就会有女朋友”这种说法。这种话听起来有道理，但实际上缺乏实用性。它无法指导你如何具体致富，反而可能让你逃避恋爱，最终导致财富和爱情双双落空。

说这种话的人，其逻辑等同于告诉你：**只要本钱足够多，什么生意都能做**。这显然是废话，因为如果本钱足够多，做什么都可以，甚至存钱吃利息就足够了。成功的创业者往往白手起家，以小博大。

## 第二点：恋爱是一种能力 ⚙️

我们已经清除了一个思维障碍，接下来需要认识到恋爱的本质。

恋爱是一种与赚钱类似的能力。你无法赚到超出你认知范围的钱，同样，你也追不到超出你恋爱能力范围的女生。恋爱能力存在**段位**差异。

*   **段位高者向下兼容**：如果你的恋爱能力高于女生，即使她更有钱，也可能愿意听从你。
*   **段位低者被利用**：如果你的恋爱能力低于女生，即使你更有钱，也可能只会被她利用。
