## 建立健康的情感账户

如何避免自我感动，建立健康的关系呢？我们可以引入“情感账户”的概念。

想象你们共同拥有一个情感账户。双方为关系做出的积极付出（如理解、陪伴、帮助）都是在向这个账户**存钱**。而索取、伤害、冷漠则是在**取钱**。

以下是维护情感账户的关键点：

*   **默默存钱，必有回报**：你的付出即使对方暂时没察觉，也增加了账户总额，最终会回馈到关系质量上。
*   **避免斤斤计较**：如果每做一件事都立刻要求对方认可或回报，关系会变得很累。计较付出多少，往往得不偿失。
*   **把握大方向**：只要双方都在为账户做净存入，大方向是正向的，细节上的多少不必过于苛求。
*   **做真实的自己**：爱情的真谛在于相互吸引和志趣相投的同行，而非单方面的追逐和依附。真实的人格魅力才是持久吸引力的源泉。
