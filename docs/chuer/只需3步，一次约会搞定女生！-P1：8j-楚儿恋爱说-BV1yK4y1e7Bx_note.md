## 第一步：试探 👣

人与人的安全距离在半米左右。在两性关系中，能否打破安全距离，是决定双方是普通朋友，还是能迅速升高关系的关键。作为男性，需要承担打破安全距离的责任，因此必须进行试探，目的是测试对方对你的亲密程度。

在刚见面、女生对你好感度不明确时，你必须进行测试。

*   **距离测试**：走路时尝试靠近女生。若她没有刻意拉远距离，则测试通过。

在成功完成第一步的试探后，意味着你们已经打破了初步的安全距离。接下来，我们需要将关系推向更明确的阶段。

当你顺利通过了前两步的测试和进挪，你们的关系已经非常亲密。最后一步将决定你们能否正式确定关系。
