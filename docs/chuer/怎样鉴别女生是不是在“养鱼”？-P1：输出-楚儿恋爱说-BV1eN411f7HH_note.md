上一节我们概述了本课的目标，本节中我们来看看具体的鉴别方法。以下是“备胎”与“理想男友”的七个核心特质对比：

*   **备胎**：女生通常只在有需求（如抱怨、求助）时主动联系。
*   **男友**：喜欢你的女生会变得粘人，乐于分享生活琐事。

**5. 关注范围：漠不关心 vs 在意你的全部**
*   **备胎**：女生不会关心你身边的人和事。

*   **备胎**：只能得到表面的关心。

**7. 重视程度：敷衍了事 vs 牢记于心**
*   **备胎**：女生不会在乎你说过什么或做过什么。
