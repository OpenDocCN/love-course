# 这三种女人不值得付出，男人要学：输出-楚儿恋爱说-BV15b4

在本节课中，我们将学习如何识别三种不值得男性付出时间、精力和情感的女生类型。通过逆向思维，帮助你避开潜在的情感陷阱，将有限的精力投入到值得的关系中。

## 1. 带有目的性接近你的女生 💰

上一节我们介绍了课程目标，本节中我们来看看第一种需要警惕的类型：带有目的性接近你的女生。这类女生最常见的表现是贪图钱财。

以下是识别此类女生的关键行为：

*   **直接索取**：例如，在节日前后以“其他女生都有礼物”为由，直接要求你购买特定物品（如“我看上了一个包包，要不然你送给我吧”）。
*   **间接暗示**：在共同逛街时，对某件商品表现出特别喜欢，随后又失落地说“最近手头不太宽裕，还是算了吧”，以此引发你的赠予行为。

真正喜欢你的女生，不会舍得让你独自承担经济压力来满足她的私欲。所有健康的人际关系都讲究有来有往，爱情也不例外。追求这类女生，即使你对她有求必应，她也未必会动心，因为**为钱而来的女生，最终也会为钱而去**。当你遇到一个总是让你为她花钱的女生时，务必保持警惕。

## 2. 有“公主病”且瞧不起你的女生 👑

了解了物质导向的女生后，我们接着分析第二种类型：有“公主病”且瞧不起你的女生。这类女生通常以自我为中心，认为所有人都应该围着她转。

以下是此类女生的典型特征：

*   **习惯指使他人**：经常命令别人做事，且说话不顾及他人感受。
*   **认为一切理所当然**：将别人对她的好视为理所应当，缺乏感恩之心。
*   **情感关系混乱**：可能同时与多个男生保持暧昧，但内心认为无人配得上自己，享受被追捧的感觉而不会真正与人交往。

不要误以为“征服”这样的女生会带来成就感。她们往往任性、自私且缺乏责任感。与她们相处，你可能会感到疲惫且不被尊重。这种心智不成熟的女生，不值得你去追求。

## 3. 永远把自己当成受害者的女生 😔

认识了以自我为中心的女生，最后我们来剖析第三种类型：永远把自己当成受害者的女生。这类女生习惯将问题归咎于外界和他人。

以下是与此类女生交往时需要注意的信号：

*   **拒绝自我反思**：例如，工作中被领导批评后，第一反应不是检讨自身问题，而是向你抱怨自己受了多大委屈。
*   **持续散发负能量**：频繁地找你倾诉，内容多是抱怨他人不好或生活不公。
*   **在关系中逃避责任**：如果你们交往，未来发生任何矛盾，她很可能都将错误全部归咎于你。

适度的倾诉是信任的表现，也能缓解痛苦。但凡事皆有度，如果对方**总是**隔三差五地传播负能量，你就要注意了。人生本已不易，何必再为自己找一个持续制造不快的伴侣。
