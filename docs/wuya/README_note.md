## 概述

“乌鸦”是一种基于群体智能的优化算法。它模拟乌鸦藏匿和寻找食物的行为来解决复杂问题。算法通过迭代更新一群“乌鸦”的位置来寻找最优解。

上一节我们介绍了课程的整体目标，本节中我们来看看“乌鸦”算法的具体运作机制。

---

## 核心概念

算法的核心在于每只乌鸦都会藏匿自己的食物（即一个潜在解决方案），同时也会试图窃取其他乌鸦的食物。这个过程通过记忆和随机探索来平衡。

位置更新是算法的关键，可以用以下**公式**描述：
`新位置 = 旧位置 + 随机数 * 飞行长度 * (记忆中的最佳位置 - 当前位置)`

或者，用伪**代码**描述其核心循环：
```python
for 每只乌鸦 in 乌鸦群体:
    随机选择另一只乌鸦
    if 随机概率 > 警觉概率:
        新位置 = 当前位置 + 随机飞行 * (被选乌鸦的记忆位置 - 当前位置)
    else:
        新位置 = 随机位置
    评估新位置的适应度
    更新乌鸦的记忆（如果新位置更好）
```

## 算法步骤

以下是“乌鸦”算法执行过程的具体步骤。

1.  **初始化**：随机生成一群乌鸦，并为每只乌鸦分配一个随机位置和记忆（初始记忆即为初始位置）。
2.  **评估适应度**：根据目标函数计算每只乌鸦当前位置的适应度值。
3.  **位置更新**：对于每只乌鸦，随机选择群体中的另一只乌鸦进行跟随或探索。
4.  **更新记忆**：如果新位置的适应度优于记忆中位置的适应度，则用新位置更新记忆。
5.  **迭代**：重复步骤2至4，直到满足终止条件（如达到最大迭代次数或找到满意解）。
