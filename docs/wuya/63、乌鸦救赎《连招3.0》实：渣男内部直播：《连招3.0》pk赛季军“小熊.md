## 第一部分：前价值——社交的入场券 🎫

上一节我们概述了课程目标，本节中我们来看看连招的第一招：**前价值**。这是所有社交互动开始前，对方能直观感受到的你的价值总和。

### 前价值的核心：价值匹配

首先你需要明白一个核心概念：**价值匹配**。在认识女人以前，最重要的就是价值匹配。

**价值匹配公式**：
```
我们倾向于结交价值 ≥ 我们自身价值的人。
```

简单来说，我们都不愿意去结交那些比我们价值低很多的人。我们希望结交价值与我们差不多，但最好高于我们的人。如果你一开始无法满足别人的价值匹配需求，对方就会将你“左滑”（忽略），即使你“超级喜欢”对方，也可能无法开启对话。

价值匹配需求，翻译成人话就是“势利眼”。因为了解一个人成本太高，所以大脑会通过**价值信号**快速判断，这是一种“以偏概全”、“以貌取人”的高效方式。

对于优越的外表，最直接的想法是“看着舒服，心情好，喜欢”。我们也会自然而然赋予他很多优秀的内在定义和正面评价，比如心地善良、聪明、正直等。

**关键点**：如果你无法在短时间内证明自己的价值，女人就不会与你进行下一次接触。在陌生人阶段，凡是能证明自己“强大”的信号，都是有效的。

### 前价值的展现形式

通过前价值去证明自己的强大，满足人们的价值匹配需求。前价值的展现分为线上和线下。

**线上前价值（社交软件）**：
*   你的照片
*   你的名字
*   你的个人简介
**线下前价值**：
*   **社交认证**：例如，你与身边人相处融洽，一堆人围着你聊天，旁人会觉得你受欢迎。
*   **行为举止**：例如，礼貌对待服务员，说“谢谢”，旁人会觉得你有礼貌。
*   **外形穿着**：穿着得体，别人也会更想认识你。

这些都属于前价值，也常被称为“展示面”。

### 什么是社交高手？

社交高手的状态是：我还没有出现在社交场合，就已经有人想要认识我。在软件上，女孩子还没看到我的全部信息，就已经在思考如何跟我打招呼。或者，配对后你可以出于礼貌主动说“嗨”，不必总是等待对方。

### 打造展示面：克服心理障碍

展示面通俗易懂。看看女生的朋友圈、小红书或抖音，很多女孩子都拍得很漂亮。

许多男人害怕“见光死”，担心照片拍得太好、太高级，或者用了不属于自己的东西（如租车拍照）会被认为是骗子。但有一句话很重要：**假的也比没有好**。

**关键心态**：你顾虑太多了。换位思考，女孩子都喜欢美、喜欢玩。你的生活建设（如健身、听livehouse）也能吸引人。照片拍得漂亮、有高级感，女孩子会认为你懂得让生活更高级，并愿意向你学习。

书中有个例子：康师傅方便面包裝上有肉，但泡出来没有。你不用太在意“包装”与“实物”的细微差距。很多时候，不是我们“见光死”，而是女孩子。见面时，你可能会发现女孩和照片差很多，但男性大脑会自动进行“美化修正”。最厉害的“美图秀秀”其实是人的大脑。

### 展示面实操指南

以下是打造展示面的具体建议：

**1. 发型与穿搭**
*   **发型**：头发留长一点，在美团或大众点评上找一家评分高、价格在80-150元左右的理发店，让发型师简单设计一下。
*   **穿搭拍照**：如果不懂，可以在小红书等平台请一位“约拍”摄影师（通常是学生，价格一两百元）。可以请他帮忙去服装店挑选衣服，然后当天购买、当天拍摄。

**2. 照片的核心：社交认证**
最好的展示面不是单人照，而是**聚会照**。聚会照传递**社交认证**，表示你拥有良好的人际关系，已被其他人选择，对方会因此认为你很有价值。

**3. 名字与签名**
*   **名字**：在社交软件上避免使用英文名（可能显得装逼）。建议使用好玩、有趣的中文名，例如：“告诉我中午吃啥”、“全网寻找王阿姨”、“爱情故事收购”。有趣的名字本身就能开启话题。
*   **签名**：可以真诚地介绍自己，包括社交目的、兴趣爱好、性格等。最好能直接表明择偶目的。书中有很多案例可以参考。

### 侧面展示：高级的价值呈现技巧

展示价值还有一个高级方法：**侧面展示**。

**原理**：你越想证明自己厉害，反而越容易弄巧成拙。舆论是叛逆的，会反着来。你越逞强，就越可能受屈辱。

**方法**：把真正想展示的价值放在**主体的侧面**，而在**主题中聊其他东西**。
*   **例子**：在颁奖典礼上，有人说自己是冠军，你可能不服。但如果他旁边坐着一位真正的美女模特，你反而会服气。他看似在领奖、讲道理，实则想让大家看到旁边的模特。
*   **应用**：比如我今天穿了一件3000块钱的衣服，如果直接问你们“这衣服好看吗”，可能很多人会挑刺。但如果我在讲课，有人主动说“小熊这衣服挺好看”，效果就完全不同。

**核心要记住**：强者不强。越是想要逞强的人，越会受到屈辱。你很强，但不能自己说，得让对方自己发现。只要留下一点“小尾巴”（证据）让对方看到即可。

在心理学上，这叫做**暗示**。书中有很多例子，需要多看几遍才能领悟。

## 第二部分：意愿锁定——掌握关系主动权 🔒

上一节我们详细介绍了如何构建前价值，本节我们来看看如何利用前价值开启对话，并通过第二招**意愿锁定**，在聊天第一分钟就锁定对方的恋爱意愿，掌握主动权。

### 意愿锁定的定义与时机

**意愿锁定**很简单：利用女生“右滑”（喜欢）你的行为作为证据，让她承认对你有兴趣。如果她对你不感兴趣，为什么要右滑你？

**关键点**：必须及时进行意愿锁定。时间久了，对方可能会找借口不承认“右滑”行为，或者干脆忘记。

**最佳时机**：匹配成功后立刻进行。因为她刚刚右滑喜欢你，此时没有其他借口反驳。如果她反驳，就违背了一致性原则。如果真的反驳，那可能就是真的划错了。

### 意愿锁定的目的与前置步骤

意愿锁定的目的是为了达成**状态同步**：彼此都知道自己喜欢对方，也知道对方喜欢自己。

匹配完成后，最好先上一个“保险”——**寒暄**。这让你看起来更自然，像个正常人。
*   **寒暄示例**：“我刚在拍照，你在干嘛？”、“我刚撸完猫，你呢？”、“这么晚还不睡？”。
*   **作用**：类似于日常和朋友、亲戚的客套话，是正常人的行为举止，避免显得唐突。

### 强制认同：巩固同步状态

寒暄之后，不要直接意愿锁定，先问一个问题：**“你看我资料了吗？”**

大部分人会回答“看了”。这意味着她匹配你时，无法假装不知道你的价值。彼此是因为知道价值而匹配的，这更能证明她是因为喜欢而右滑你，彼此是状态同步的。

**状态同步举例**：你想拉朋友一起去拍展示面，但他觉得没用。你需要费力说服他。但如果你拉一个同样买了《连招3.0》的兄弟，你们都认同展示面的重要性，沟通效率就会高很多。这就是状态同步。

### 意愿锁定的完整流程与做实

以下是意愿锁定的完整对话流程：

1.  男：嗨。
2.  女：嗨。
3.  男：我刚刚吃饭了，你吃了吗？（寒暄）
4.  女：吃了。
5.  男：你看我资料了吗？（强制认同）
6.  女：看了。
7.  男：问个问题，我是不是你喜欢的类型？（意愿锁定/强切）
8.  女：是的，不然我匹配你干嘛？

通过这个流程，我们让对方暴露了需求感。至此，我们成了掌握主动权的一方。

但这还没结束，我们需要“做实”，把模糊的恋爱状态变成实实在在的、即将恋爱的认知。

**做实的方法**：继续说：“那我们聊聊天吧。我们聊得好的话，就见面相处一下。如果见面相处合适的话，我们就在一起。你觉得呢？”

这是一个在中国公认的、非常正常的恋爱流程。你们只是把它摊开讲明白了。如果对方此时还墨迹（例如“会不会太快了”），就不用理会，大不了换一个。

做完意愿锁定并“做实”后，就可以自然地索要微信了。

### 意愿锁定的三大好处

意愿锁定技术的好处非常多，主要有三点：

**1. 破除对方防护罩，掌握主动权**
意愿锁定相当于破掉了对方的“防护罩”，对方暴露了需求感，很多“作”的花招就无法使用，会乖乖和你推进关系。如果对方没有暴露需求感，而你暴露了，你就会陷入被动，被迫跟着对方节奏走，甚至面临“废物测试”（例如：“你是不是对很多女生都说过这话？”、“你是不是渣男？”）。

**2. 过滤高心态对象，节省时间**
意愿锁定能把那些愿意大方承认喜欢你的人过滤出来。这类女孩子天生心态比较好，对男孩子没有不良目的。对于那些墨迹、犹豫的对象，但凡她牙崩半个字，就直接放弃。因为即使勉强在一起，后续也会非常内耗。连招3.0的速度很快，不要纠结于个别对象。

**3. 植入潜在恋爱对象标签，脱颖而出**
意愿锁定后，你在对方潜意识中被植入了“潜在恋爱对象”的标签，产生了唯一性连接。社交软件上，每个人都在和多个异性聊天。当你的竞争对手还在问“在吗？”“吃了吗？”的时候，你们已经聊到了“相处合适就在一起”的阶段。你已经脱颖而出，竞争对手几乎没有活路了。

## 总结与常见问题解答 💡

本节课我们一起学习了《连招3.0》中最核心的两个基础招式：**前价值**与**意愿锁定**。

*   **前价值**是你的社交入场券，通过线上展示面（照片、名字、签名）和线下形象（社交认证、行为举止）来满足初期的价值匹配需求。高级的技巧是**侧面展示**，避免直接逞强。
*   **意愿锁定**是对话开启后的第一把钥匙，通过**寒暄**、**强制认同**和**强切**，在第一时间让对方承认兴趣，达成**状态同步**，从而让你牢牢掌握关系推进的主动权。

### 直播中提及的常见问题解答：

**1. 关于展示面心理建设**
*   **问**：如何克服觉得自己丑、不敢用原相机拍照的心理？
*   **答**：必须克服这个障碍。不要用美颜相机，用**原相机**拍摄，像素更高，经得起后期修图。美颜相机拍出的照片再经过平台压缩，质量会很差。你自己觉得丑的地方，别人可能并不那么关注。修图时（如用美图秀秀），重点修饰自己的缺点即可。

**2. 关于意愿锁定后的聊天**
*   **问**：意愿锁定后聊什么？会不会太快？
*   **答**：意愿锁定后，就进入了相互了解内在的阶段。可以聊彼此的性格、生活、兴趣爱好等。这些在连招的第三、四、五招中会有详细讲解。如果想要一段稳定的关系，建议慢慢经营，给彼此安全感。速度太快，后续需要投入的精力可能更大。

**3. 关于“撞车”（对方也学过类似方法）**
*   **答**：无需担心。这一套流程符合人性与正常的恋爱逻辑，就像一个高效的“相亲模式”。一个有恋爱经验、懂得经营关系的人反而更有魅力。撞车说明对方也可能在寻求一段认真的关系。

**4. 关于寻找“僚机”**
*   **答**：僚机（志同道合的学习伙伴）非常重要，是情感基础建设。可以在学习群（如连招3.0的学员群）中寻找同城或价值观一致的兄弟。僚机可以相互监督、一起拍照、讨论情感问题，实现群体决策，加速成长。选择僚机时，需注意人品和经济实力尽量对等。

**5. 关于“状态同步”**
*   **答**：状态同步可以简单理解为“同频”。在情感中有三种状态：**状态靠前**（需求感更强）、**状态靠后**（需求感更弱）、**状态同步**。通常，状态靠后的一方更有控制权。意愿锁定就是通过“强切”问题，让对方承认兴趣，从而使其状态靠前，让你获得控制权。如果双方状态不同步（如吵架时一方在生气，另一方讲道理），沟通就会无效。
