在本节课中，我们将要学习一个核心心态调整方法：当状态不佳时，如何通过“承认”与“接纳”来有效化解困境。这是改善情绪与提升行动力的第一步。

## 承认状态不佳是第一步

当我们自身状态不够好时，唯一需要做的事情就是承认自己的状态不好。然而，许多人在此情况下不愿意承认这一点。他们反而会思考：“我现在应该做些什么，才能改变这个不好的状态？”

## 接纳带来心境转变

其实，当你的状态本身不好时，你却试图通过做很多事情来改变它。这本质上是不愿意承认状态不好的表现。**先承认你的状态不好，并接纳它**。这样做之后，你才会发现自己整个心境已经发生了改变。

上一节我们介绍了承认的重要性，本节中我们来看看为何接纳是关键。

## 从潜意识理解接纳

如果你立刻去寻找各种方法来解决状态不好的问题，从潜意识的角度来讲，这正表明你不承认自己的状态不好。当我们承认状态不好时，内心深处就在告诉自己：“我接纳了自己状态不好这个现实。”

以下是接纳带来的两个核心转变：
*   **状态不好，我接受。**
*   **状态好，我也接受。**

## 接纳如何提升视野

以这样的心态面对，我们自己的思维和心胸就会变得开阔。当你真正接纳这一切时，你就自然而然地上升了一个台阶。

**站得更高，就会看得更远。**

在这个新的高度上，你能够自然而然地想到办法，去解决和化解那种状态不好的情况。

## 总结与核心公式

本节课中我们一起学习了化解不良状态的核心路径。其过程可以总结为以下简单公式：

**承认状态不佳 → 全然接纳现实 → 视野提升 → 自然找到解决方案**

因此，化解不好状态的第一步永远是：先承认，先接纳。在不接纳的情况下，费心去寻找其他方法都将是徒劳的。
