在本节课中，我们将探讨一个有趣的心理状态——“被主动”。我们将分析它的含义、判断方法，并理解为何接纳这种状态比强行追求“纯粹主动”更为健康。

你有没有发现一个很有意思的现象？我们在生活中很多时候去做一些事情，总处于一种“被主动”的状态。

## 什么是“被主动”？🤔

“被主动”如何去理解，又如何去判断呢？想象一下，我们去做某件事情，真的是你主观意愿非常想去做吗？

大多数情况下，这是一种“不得已”。不得已，就是一种**被动的主动**。在这种“被主动”的状态下做事，很多人会感到讨厌。

## 对“被主动”的常见误解 ❌

许多人讨厌这种状态。他们想积极、想努力，想让自己变得积极向上。他们认为：**我一定要什么事情，都主动、凭主观意愿去做。这样主动去做，才是好的。**

当你有这个想法时，你其实会慢慢变得糟糕。

## 重新审视“被动主动” 🔄

为什么这样讲？被动做一件事情，听上去好像是一种不情愿的状态。但是，“被动主动”地去做一件事，它是一种**顺其自然**的状态。

核心原则是：
```
该主动时就主动，不该主动时就不主动。
该“被动主动”时，我们就“被动主动”。
```

我们需要接受这样的现实，没有刻意去要求的必要。不必刻意要求自己非得主动，也不必认为不主动或“被动主动”就是不好的。

## 接纳与顺其自然 🌱

不要有“被动主动不好”的观念。我们去接纳任何一种状态。而且，做任何一件事情，我们是顺其自然的，按照自然规律进行。

该你去承担、该你去做的时候，你就应该去做，不要找其他理由去搪塞。

## 按规律办事的好处 ✨

按照规律来办事情，那我们自己就会活得轻松一点，而不会去刻意做一些事情。

我们发现，往往刻意做一些事情，会让自己变得不高兴，会让自己变得越来越糊涂。很多时候，刻意为之就是一种“得过且过”的状态。

## 总结 📝

本节课中，我们一起学习了“被主动”状态。生活当中很多事情其实很简单，只是我们很多时候太过刻意了。关键在于**接纳顺其自然的状态**，该主动时主动，该“被动主动”时则坦然接受，这样我们才能活得更轻松、更通透。
