# 高手都是谋定而后动 - 课程 P1 🎯

在本节课中，我们将要学习“谋定而后动”这一核心思维模式。我们将探讨高手与普通人在决策和执行上的关键区别，并理解深思熟虑对于成功的重要性。

## 高手与普通人的思维差异 🤔

上一节我们介绍了课程主题，本节中我们来看看高手与普通人在行动逻辑上的根本不同。

真正的高手是看清楚了，想明白了，然后再去执行。而很多人是知道了，刚刚看到了某个东西，然后立马就去执行。这就是高手和普通人之间的差距。

高手永远是三思的，永远是要想清楚的。而大多数普通人都是根据自我感觉来去做事情的。

## 如何实践“谋定而后动” 💡

理解了差异后，我们来看看如何将这种思维应用到具体事情中。

任何一件事情，都**有利有弊**。高手会看清楚自己想要的是什么。他会综合判断做这个事情会给自己带来什么，他自己能不能够承受。他会想很多事情，综合考虑过后，看清楚了，想明白了，才去执行，才去做这件事情。

所以，如果你想要成为高手，做任何事情之前，请三思。认真地去想一想，认真地思考一番。

以下是实践这一原则的具体建议：

*   在我们不了解的情况下，不要妄动。
*   在你不了解的情况下，自己去做一些事情都是错误的，会让自己误入歧途。

## 盲目行动的后果与总结 ⚠️

最后，我们来审视一下未经思考就行动的潜在风险。

而当你陷入那种困境的时候，你才感觉到后悔，那就是无用功了，那就是你走到弯路了。

**所以，真正的高手是先谋定而后动。**
