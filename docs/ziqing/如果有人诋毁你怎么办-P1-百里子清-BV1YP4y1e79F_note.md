# 课程名称：如何应对诋毁 - P1 🛡️

在本节课中，我们将要学习当面对来自同事、朋友等身边人的诋毁时，应如何理性应对。我们将分析一种常见的冲动反应及其弊端，并探讨一种更为明智和有效的处理策略。

## 常见的冲动反应与弊端 🤔

上一节我们介绍了课程主题，本节中我们来看看一种常见的应对方式及其问题。

当身边的人如同事或朋友诋毁你时，许多人会采取直接回怼并与之争论的做法。他们的目标是必须把事情争论清楚，一定要告诉对方，并一定要让对方意识到是他错了。他们认为对方不应该诋毁自己。

然而，这样做过后，其实解决不了问题。这种行为还会让彼此之间的感情受到伤害，并且你自己的状态会变得越来越糟糕。

## 理解诋毁者的逻辑 🧠

从某种角度来讲，我们要意识到一点：这种诋毁你的人，他有自己的一套逻辑。在他的逻辑里，就已经把你定义了。你处于一种百口莫辩的境地，你说什么都是错，你做什么都是错。

## 明智的应对策略：选择远离 🚶‍♂️

理解了对方的逻辑后，本节中我们来看看在这种情况下最明智的做法。

这种情况下，我们最明智的一个做法就是远离。具体行动包括：
*   我不和你争论。
*   我不做任何举动。

我选择远离，因为你识毁我。从某种角度来讲，你已经把我定义成不好的、不OK的状态。那么你就是一个不值得交往的人。之前我和你有一定的交集，那是之前。但是直到现在，你要识毁我，那就是你的问题。

## 远离带来的积极影响 🌱

那么，选择远离会带来什么好处呢？

选择远离这样子的人，我就会得到一种平静。我自己利用这些时间，去修炼自己，让自己过得越来越好。这样我就会慢慢地、逐渐地提升。

以下是核心的行动建议：
*   遇到识毁你的人，最好的一个方式就是远离他。

---

本节课中我们一起学习了应对诋毁的策略。我们分析了直接争论的弊端，理解了诋毁者的固化逻辑，并明确了“选择远离”这一核心应对方法。远离消耗你的人际关系，将精力专注于自我提升，是获得平静与成长的有效途径。
