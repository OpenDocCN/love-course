## 原则一：初期尽量少花钱 💸

上一节我们提到了基本原则的重要性，本节中我们来看看第一个具体原则：追求初期应尽量控制花费。

在关系尚未明确的初期，双方并不熟悉。此时可以承担约会中的共同开销，但务必控制在自身能力范围内。

以下是关于此原则的具体说明：

*   **区分消费类型**：约会、游玩等共同活动产生的花销，可以适当承担。例如，一起吃饭、看电影的费用。
*   **不承担个人消费**：如果对方在商场购买个人物品，这不属于你的责任范围。为这类消费买单是错误的。
*   **避免设立不良先例**：如果此次为其个人购物买单，未来遇到同样情况，你将陷入“给或不给”的两难境地，给则可能成为沉重负担，不给则可能影响关系进展。

## 原则二：核心是量力而行 ⚖️

理解了初期控制花费的必要性后，我们来看最重要的核心原则：一切消费必须量力而行。

追求行为应根据自身实际经济状况进行，不应让自己陷入入不敷出的困境。

以下是关于“量力而行”的实践要点：

*   **公式**：**你的恋爱支出 ≤ 你的可支配恋爱预算**
