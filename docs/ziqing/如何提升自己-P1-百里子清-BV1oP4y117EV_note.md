# 课程：如何提升自己 - P1 🧠

在本节课中，我们将探讨一个核心问题：如何有效地提升自己。我们将分析常见的思维误区，并引入“道”与“术”的概念，帮助你理解真正持久的成长源于内在的根本性转变。

## 放下“提升”的执念

有位粉丝问我：“怎么才能提升自己？” 我的回答是：“先放下这种想法。” 他接着问：“如果放下了，还怎么去提升自己呢？” 我没有作答。我明白他内心想要的是什么。大多数人想得到的答案，无非就是一种快速见效的方式和方法。

他们想要的是具体的方法。他们的潜意识里期待的答案是：你直接告诉我，我到底该怎么做，第一步做什么，第二步做什么，我才能提升自己。然而，如果顺着这个思路去寻找，你是永远也无法真正提升自己的。

## “道”与“术”的启示

讲到这里，我想起了商鞅变法和申不害的变法。申不害和商鞅都是法家的代表人物，但为什么商鞅最终能成功，申不害却失败了呢？关键在于，申不害的变法**有术无道**，而商鞅的变法则是**有道有术**。

所以，我们大多数人想去提升，其实想要的只是这种“术”——即具体的方法和技巧。但是，仅有“术”是走不远的。我们需要去做到“有道”。我们需要去研究，去思考：我该如何从根源上去改变？

## 根源在于思维转变

我们的根源，就是思想与思维方式的改变。这种“道”的提升，才能带动我们整体的、根本性的提升。而很多人想要改变自己，就是想很快得到效果，这恰恰是“欲速则不达”。

你越是执着于找到一种方法去快速提升自己，你就越会进入一种迷茫的状态。越是持有这种执念，越会让自己迷惑，从而看不清事物的本质。

## 如何看清本质并真正提升

因此，我们只有先放下，先放下“急于提升”这种想法。让自己处于一种“空杯”的状态。这个时候，你才能看清事物的本质。

你看清本质过后，就会明白：我如果要提升自己，应该提升自己内在的东西。要做到**有道有术**。这样的一种提升才是长远的，才是可持续的发展。

---

**本节课总结**：我们一起学习了“道”与“术”的区别。真正的、持久的自我提升，并非源于对具体方法（术）的执着追寻，而是始于放下急功近利的心态，通过转变根本的思维方式（道），达到内在的成长，最终实现有道有术的全面发展。
