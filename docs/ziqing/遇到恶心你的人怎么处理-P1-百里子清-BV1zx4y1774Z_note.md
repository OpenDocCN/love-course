生活当中总会有那种让我们恶心的人。你遇到这样子的人，心里会不会有这样子的想法，就是很想去回怼。但是呢，如果越是去回怼，好像他越起劲，他越是恶心你。然后这个样子让我自己不舒服，然后好像我就和他是一样。

上一段我们描述了面对令人反感者时常见的矛盾心理。接下来，我们来看看这种困境的本质。

我也变成恶心的人了。但是如果我不去回怼，不做出任何一个回应吧，他也会得寸进尺。这种情况下好像是左右为难的。

其实呢这种情况下我们可以这样去想。他来恶心你，他其实也是在磨练你，磨练你的耐心。

以上就是我们面临的典型困境。下面，我们将介绍第一种核心处理策略。

## 策略一：视为心性磨练 🪨

我就是要有这样子的心性，我不去理会他，你不去搭理他。你这样就是锻炼了你控制自己情绪的能力。就把他当成一个磨刀石。其实从这个角度来讲，他还有一点好处。

**核心公式**：`将对方的挑衅行为 = 锻炼情绪控制力的磨刀石`

当我们能够运用上述策略看待问题时，通常能有效化解大部分日常困扰。然而，有些情况可能超出忍耐的极限。接下来，我们探讨第二种策略。
