在本节课中，我们将学习当遇到他人不理解自己时，应如何调整心态与处理方法。我们将通过对比过去与现在的应对方式，探讨为何不必强求他人理解，以及如何专注于自我接纳与前行。

## 过去的应对方式：解释与内耗

上一节我们介绍了课程主题，本节中我们来看看一种常见的初期应对模式。

如果遇到别人的不理解，你会怎么处理呢。我是这样处理的，来看看和你的处理方式有什么不一样。之前的我，遇到这样子的状态。你比方说我在工作当中，和同事之间发生了一些误会，同事他不理解我。然后这个时候，我内心深处像一块石头一样压着。我一直在想，我要通过什么样子的方式去给他解释一下，然后让他理解我，告诉他这是无心之事，这只是一个误会而已。脑袋里还会组织一些语言，找一个比较恰当的时机，去找对方说这件事情。

说了过后，我也没有明显的感觉到他就理解我了，然后这件事情就不了了之了。但是又过了一段时间我才发现，其实他没有理解我。

## 心态的转变：允许与接纳

经历过这种事情过后，我的处理方式就发生了改变。上一节我们看到，试图解释可能效果有限，本节中我们来看看更有效的核心心态。

我允许别人不理解我。他不理解那是他的事情，我也不必要去做一些解释。那么既然是不理解，那我允许他不理解我。然后这个时候，我自己也允许自己接纳这种状态，然后去做自己该做的事情，把这个事情就此放下。

以下是支撑这一心态的两个关键理由：

1.  **频道不同，解释无效**：因为别人不理解你，那么他的这种思维方式，或者说和你的这种频道，不在同一个频道之上。你去做更多的一个解释，人家也没有办法理解你。我们自己心里清楚这一点就行了。
2.  **无需所有人理解**：也不要指望着所有的人都能理解你。

## 深入思考：理解的价值

从另外一个层面来讲，他理解你又能怎么样，他不理解你又能怎么样。你得到他的理解，你又能得到什么。那我们把这些问题认真去想一想，你就能够明白了。

是否能够得到别人的理解，有的时候其实并没有那么重要。那么很多时候我们为什么要渴望得到别人的这种理解呢。无外乎就是想证明自己的这种价值，以为得到别人的理解，得到别人的认可就是一种成功。而这种成功是我们真的想要的吗。

## 核心原则：不强求

别人不理解你非得让别人理解，那就是强人所难，那别人就没有办法理解你这样子的一个做法。

**核心公式可以概括为：**
**自我接纳 + 停止解释 + 专注己事 = 内心平和**
