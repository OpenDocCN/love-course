# 课程名称：人际相处策略 第一讲 - 如何应对让你不悦的人 👥

在本节课中，我们将要学习当生活中遇到那些让你感觉不愉快、难以沟通的人时，应如何调整心态并采取具体的相处策略。核心原则是接受人际差异的客观性，并优先关注自身的感受与边界。

## 理解现状与核心原则

生活中，我们时常会遇到一些让自己感觉不悦的人。这些人可能难以沟通、其行为方式令人不适，并且有时我们无法完全避免与他们接触，例如对方可能是你的同事或亲友。

面对这种情况，需要把握一个核心原则：**非必要，不接触**。

## 具体应用策略

以下是基于“非必要，不接触”原则的具体行动指南。

*   **工作场景**：如果对方是同事，除工作必需的合作外，尽量减少其他接触。
*   **亲友场景**：如果对方是亲朋好友，仅在无法回避的场合（如家庭聚会）进行必要接触，其余时间选择保持距离。

试图强行与让你感觉不悦的人搞好关系、寻求其认可或营造良好感觉，往往是违背人际交往自然规律的。人与人的相处存在“物以类聚，人以群分”的客观现象，有些人之间就是难以融洽相处。

## 调整个人心态

上一节我们介绍了具体的行为策略，本节中我们来看看如何调整内在心态，这对于执行上述策略至关重要。

当你决定减少与不悦之人的接触后，无需过度思虑他人的看法。例如，不必担心亲人会批评你不合群。此时，你应当优先考虑自己的感受。

因为人生中有许多事情无法完全掌控，我们只能顺应现实。强行去融合那些本质上合不来的人，反而会成为困扰自己的问题。我们需要尊重人际交往中的自然规律。

## 尊重自然规律

人际间的亲疏合拍，本身也是一种自然规律。从某种角度看，“合不来”的状态有其自身存在的规律性。我们不应违背这种规律。

## 课程总结

本节课中，我们一起学习了应对令自己不悦之人的方法。关键在于接受“并非所有人都能融洽相处”这一现实，并贯彻“非必要，不接触”的行动原则。同时，我们应优先关照自身感受，不必过度在意他人评价，并学会尊重人际交往中存在的客观规律。
