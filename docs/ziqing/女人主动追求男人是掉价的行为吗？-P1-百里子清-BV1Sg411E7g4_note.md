## ⚖️ 什么才是真正的“掉价行为”？

上一节我们明确了追求行为的本质，本节中我们来看看哪些具体行为才可能构成“掉价”。

关键在于区分“主动追求”和“不当行为”。追求是表达好感、争取机会；而“掉价”往往体现在追求过程中的姿态与方式上。

以下是几种在追求过程中可能被视为“掉价”的具体行为模式：

*   **过度讨好（跪舔）**：表现为放弃自我原则，一味迎合对方。
*   **非他不可的执念**：认为对方是唯一选择，将自己置于没有退路的境地。
*   **无回应的纠缠**：在对方已明确缺乏兴趣或拒绝后，仍持续打扰，不愿放手。

**公式：掉价行为 ≈ 失去自我 + 忽视对方反馈 + 突破合理边界**

而**主动、大方、尊重彼此的追求，是自信和清醒的体现，绝非掉价**。
