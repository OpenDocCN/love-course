# 分手后心理调适指南 💔 - 课程01

在本节课中，我们将要学习分手后如何调整心态，处理情绪，并专注于自我成长。我们将探讨几个关键的心理建设步骤，帮助你平稳度过这个阶段。

上一节我们介绍了课程的整体目标，本节中我们来看看分手后需要明确的第一个核心认知。

分手后无需过度悲伤，也不必追问对方是否爱过。因为**爱过与否**的答案，对于已结束的关系并无实际影响。当你开始新的恋情时，自然会逐渐释怀，那时的痛苦感将大大减轻。

感情的本质便是如此，不同的人出现在生命里扮演着不同的角色。

以下是几种常见的情感关系模式：
*   有些人是为了与你体验恋爱过程。
*   有些人是为了促使你获得成长。
*   有些人则是为了与你共度余生，走向婚姻。

上一节我们梳理了感情中不同人的意义，本节中我们来看看如何正确处理与前任的关系。

分手后，对方的生活状态已与你无关。对方过得好，是**他/她的事**，无需羡慕；对方过得不好，同样是**他/她的事**，也不必幸灾乐祸。

你的首要任务是专注于经营好自己的生活。

让自己活得开心，才是最重要的事。

本节课中我们一起学习了分手后的基本心态调整方法：接受关系结束的事实，理解感情的不同阶段意义，将注意力从前任身上收回，并最终聚焦于自我的幸福与成长。记住，过好自己的日子才是正事。
