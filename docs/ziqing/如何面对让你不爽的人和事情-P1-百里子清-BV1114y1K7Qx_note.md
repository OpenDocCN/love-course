## 核心策略：选择远离

在生活中遇到让我们不爽的人时，最直接有效的方法是选择远离。许多人遇到这种情况时，会反复思考如何与对方相处、如何调和关系。实际上，这种状态往往难以调和。

我们应尽量远离。不与对方接触，便不会发生令你不快的事情。如果执着于思考“他为什么让我不爽”、“他为何这样对我”，你就会陷入对方的情绪状态中。

这是一种无用的内耗，它会分散你的心力。这意味着你没有专注于自己的事情，而是在思考他人的行为。无论原因为何，面对让你不爽的人或事，选择远离即可。尽量远离是首要原则。

## 处理无法完全远离的情况

以下是针对无法完全远离的场景（如亲友、同事）的应对方法。

有人会提出，在日常生活、工作或学习中，有些人无法彻底远离。例如，有血缘关系的亲朋好友，他们的某些特质或行为可能让你感到不快。

在这种情况下，我们依然可以做到**尽量远离**。即，尽量减少接触，做到非必要不接触。通过这种方式，你依然行使了自己的选择权。

举例来说，如果是你的亲人，且你确实感到与对方相处不舒服，你可以尽量远离，非必要不接触。在必须接触时完成接触，然后尽量减少其他时间的交集。其余时间，则专注于自己的事情。

## 专注自身，获得平静

上一节我们讨论了如何减少接触，本节中我们来看看远离之后应该做什么。

核心是专注于自身的成长。关注自己，去做那些能让你感到平和、平静与舒适的事情。主动远离那些糟心的事，这是你可以做出的选择。

你会发现，通过这种方式，你的状态会越来越好。这是一个渐进向好的过程。

## 总结与核心要点

本节课中我们一起学习了面对不快人与事的基本心法。

请记住一个根本道理：**选择权始终在你手中**。任何时候，你都有选择的权利。不要忘记这项权利。如果你不去选择，而是陷入纠结，那便是你自己的问题了。

**核心公式**：
```
你的情绪状态 ≈ 你投入的注意力 - 你接触的负面源
```
