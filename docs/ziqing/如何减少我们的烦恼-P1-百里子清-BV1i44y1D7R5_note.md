# 课程一：如何减少我们的烦恼 - P1 🧘

在本节课中，我们将要学习烦恼产生的核心根源，并探讨一种通过转变关注点来减少烦恼的思维方法。我们将分析过度关注“结果”所带来的问题，并理解关注“原因”或“过程”本身的重要性。

## 烦恼的根源：过度关注结果

大家是否思考过，我们每日烦恼的根源在何处？答案其实很简单：我们过于注重事情的结果了。

例如，在日常工作中，今天去做一件事，若结果不理想，我们就会感到糟糕。

心情会非常不舒服，并因此结果产生许多负面情绪。我们会不断推演：如果之前不那样做，是否就不会出现这个结果？我好像在某处做错了，当时不应该那样做。我们会一直顺着这个思路想下去。

因为这个不好的结果，我们顺着这些思路去回想之前的事情，脑海中不断盘旋着一个念头：要是这件事没发生该多好，要是换种做法，就会出现另一个局面。我们总是进行这样的假设。

这种想法的出现，正是因为我们太看重结果了。实际上，当出现不利局面时，我们只需接受它、承认它即可。接受“我没有做好”这个事实，不去纠结不好的结果，而是将心思用于后续的事情上。

## 沉浸于结果的代价

上一节我们介绍了烦恼源于对结果的执着，本节中我们来看看这种执着会带来哪些具体代价。

将时间和精力投入到后续的事情上，把后面的事情做好，就能避免再次陷入不利局面。但若你此时总想着已发生的结果，除了浪费时间，还会消耗心情，让自己越来越不舒服。

除了导致上述局面，过度关注结果还会带来其他影响。再举一个例子：有人做一件事，取得了好结果，他便非常高兴，沉浸在“我终于成功了”的喜悦中，想要炫耀，向他人证明自己的能力。

他沉浸在这种喜悦里，却没有察觉周遭已在悄然改变。因为你的时间和精力都耗费在喜悦上，便没有余力去应对接下来发生的事情。

然而，抛开我们个人的想法与感受，世界有其运行规则，其他事物都在不断变化。我们周围的环境已经改变，而我们却仍沉浸在之前的结果中。请想一想。

我们就与现实脱节了。这种脱节会带来什么后果呢？它会带来烦恼。因为你无法做到随机应变，无法跟随变化的节奏去顺应自然规律。那么我们之后做任何事情，不就容易陷入混乱了吗？

## 解决方案：关注“因”而非“果”

上一节我们看到了沉浸于结果的危害，本节中我们来看看解决问题的核心方法。

因此，我们若过于注重结果，无论这结果是好的还是坏的，你都会陷入一种糟糕的状态，从而产生更多烦恼。反之，如果我们不注重这个“果”。

我们去关注“因”，更注重问题本身，注重事情的起因而非结果。从事情的“因”出发去做事，就相当于遵循并顺应自然的规律。

那么，无论出现何种局面，我都能冷静、冷静、再冷静地去应对。这样我便能做到随机应变。如此一来，顺着规律而行，烦恼自然就会越来越少。

## 总结

本节课中我们一起学习了烦恼的一个重要根源——对“结果”的过度执着。我们分析了无论结果好坏，沉浸其中都会导致与现实脱节并产生更多烦恼。最后，我们探讨了解决方案：将关注点从“果”转移到“因”，注重过程与规律本身，从而能够冷静应对变化，减少烦恼。核心在于：**接受已发生的结果，将精力专注于当下与未来的行动**。
