在本节课中，我们将要学习一个简单而有效的方法，用于判断他人是真心为你好，还是仅仅在看你笑话。核心在于转变我们看待问题的角度，通过消除“分别心”来掌握人际交往中的主动权。

## 核心方法：一视同仁

上一节我们介绍了课程的主题，本节中我们来看看具体的解决方法。关键在于**不进行区分**，将所有人置于同一种状态下来对待。

你可以选择将他们**一视同仁地视为都是为你好**。当你采取这种心态时，无论对方初衷如何，你只需接受这份“好意”。问题便得到了简化。

或者，你也可以选择将他们**一视同仁地视为都是来看笑话的**。同样，不做区分，笼统地以此心态面对。你会发现，问题同样迎刃而解。

很多时候，烦恼源于我们自身。我们执着于区分“真心为好”与“虚情假意”，这种分别心导致了区别对待。

## 区别对待的困境

当我们试图区别对待时，看似在厘清问题，实则让情况变得复杂。区别对待意味着我们被他人的态度所牵引。

公式表示为：
**你的情绪 = f(他人的态度)**
此时，你的情绪状态取决于他人的行为，丧失了自主性。

反之，若保持一视同仁，问题则简单许多。例如，认定“所有人都是为我好”，我便欣然接受这一切。

再例如，认定“所有人都是看笑话”，我也能接受。因为最终的决定权与如何看待此事，掌握在我自己手中。

## 掌握主动权：转变观念

关键在于观念的转变。如果执着于区分，你便顺应了他人的思维框架。

代码逻辑描述如下：
```python
if 试图区分他人意图：
    状态 = “被他人影响”
else: # 选择一视同仁
    状态 = “掌握主动权”
```

当我选择不区分好坏时，我便能以自我为主导，活出真实的自己。我依照自己的意愿来看待事情。

这样，我始终占据主导地位。这是一个更佳的状态，它能让我包容更多，减少计较。

以下是这种心态带来的具体好处：
*   不会因他人“真心为好”而过度高兴。
*   不会因他人“看笑话”而感到悲伤。
*   从而能始终保持一个平和、良好的自我状态。

这难道不是一种更好的生活方式吗？很多时候，仅仅转变一个观念，你看待问题的角度便会不同。

随之而来的，是你的个人状态乃至整个生活状态都会发生积极的变化。
