# 课程：远离那些试图改变你的人 - P1 🚫

在本节课中，我们将探讨为何需要远离那些试图改变我们的人。我们将分析这种行为背后的逻辑，并强调保持个人独立性的重要性。

## 远离试图改变你的人的核心理由

上一节我们介绍了课程的主题，本节中我们来看看具体原因。我们要远离那些试图改变你的人，不管他是谁，就算是你的亲朋好友，甚至是父母，我们都要这样做。

**为什么必须这样做？** 因为他们这样做的背后逻辑，其实就是一种控制。他们想你要根据他的这种想法来。

**他为什么要改变你？** 就是他一定要让你按照他的方式，按照他的这一套来。这样子的話，才能达到他控制你的目的。他需要你作为他的附庸。

## 人际交往的对等原则

但是，他们忘了一点。人和人之间的交往，就算是自己和自己父母之间的交往，我们都是一个对等的状态。

以下是关于个体独立性的核心要点：

*   首先我们是独立的个体。我们来到这个世界上，我们就是独立的那么一个个体。
*   独立的个体，我们和人交往的时候，他就必须建立在对等的状态之下。
*   如果你试图去改变别人，那你自己想一想，那你就是把自己的意志强加给别人。

## 坚持独立，活出自我

希望别人按照你的这一套来，那这样子人家也不会愿意做你的附庸。这个世界上任何人都不希望自己成为别人的附庸。

所以，对于那种试图想改变我们的人，不管他说得天花乱坠，不管他给你多少的承诺，不管他把这个东西描述得有多么的美好，我们都要远离他。

因为他的出发点他的目的都是不好的。而我们生活在这个世界上，首先得保持自己的独立性。你自己这种独立性才是你独特的魅力，才能活出我们人生的精彩。

这个样子的話我自己的人生，才是我自己说了算。所以说我们要朝着这个方向去努力，远离那种试图改变你的人。

---

本节课中我们一起学习了远离试图改变你之人的重要性。我们认识到这种行为本质上是控制，违背了人际交往的对等原则。保持个人的独立性和自主权，是活出精彩人生的基础。因此，我们必须有意识地识别并远离这类人，捍卫自己人生的主导权。
