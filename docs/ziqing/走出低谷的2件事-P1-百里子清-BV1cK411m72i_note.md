# 走出低谷的两件事 - 课程 P1 🧗‍♂️

在本节课中，我们将要学习当处于人生低谷时，两种最有效的应对策略。这些方法的核心在于接纳现状、减少消耗并积蓄能量，从而为走出低谷奠定基础。

## 接纳现状，积蓄能量 🌱

上一节我们提到了处于低谷时的基本心态。本节中我们来看看具体该如何操作。

在低谷期，最好的做法是接纳。这意味着接纳自己的状态，承认现实，避免胡乱行动。这并非消极摆烂，而是为了慢慢积蓄能量。此时，最重要的有两件事。

以下是第一件关键事项：

*   **不要产生新的消耗**。我曾有过这样的经历：在低谷期仍要求自己做很多事，用高标准苛责自己，强迫自己坚强。这些做法都在消耗本已不足的能量，属于无用功。它们只会让你在低谷状态中持续更久，难以走出。

## 稳定与提升能量 🔋

了解了避免消耗的重要性后，我们来看看如何积极地为自身补充能量。

第二件关键事项是**稳定并提升自己的能量**。

以下是具体可行的做法：

*   **多做让自己开心的事**。例如，我会通过静坐让心情恢复平静。我也会阅读经典的人物传记，感受作者的人生起伏。在这个过程中，我能慢慢领悟道理，心情也随之开朗起来。
*   **从经历中寻找经验**。你或许还能从中找到一些有助于走出低谷的经验和方法。这些都是积蓄能量的行为。
*   **照顾好自己的身体**。身体是革命的本钱。失去了健康，就真的失去了翻盘的机会。

这些点点滴滴的积累，最终会积少成多，实现质的飞跃。

## 走上坡路的开始 🚀

当你开始实践上述方法，身心逐步放松时，你就是在慢慢积蓄能量。事实上，此时你已经悄然开始走上坡路了。

接下来需要做的，就是不断重复这些积蓄能量的事情，形成正向循环。

本节课中，我们一起学习了走出低谷的两大核心：**避免新的消耗**与**稳定提升能量**。关键在于接纳现状，并通过具体的、滋养身心的行动来逐步积累力量。如果你此刻正身处低谷，不妨尝试一下这个方法。
