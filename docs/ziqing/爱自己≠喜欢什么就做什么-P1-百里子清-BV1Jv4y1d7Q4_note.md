# 爱自己≠喜欢什么就做什么 - 课程P1 🧠

在本节课中，我们将要学习一个关于“爱自己”的重要概念。很多人误以为爱自己就是随心所欲，只做喜欢的事。我们将探讨这种想法的局限性，并了解什么才是真正有益于自身成长的“爱自己”。

## 核心概念的误区

很多人认为，爱自己等同于“我喜欢做的事情我就去做，我不喜欢做的事情我就不去做”。他们认为这种做法是在让自己成长，是在有选择性地为自己而活。

## 深入剖析喜好标准

然而，你会发现，“喜欢的”与“不喜欢的”标准，完全基于你自身当下的感受和偏好。渐渐地，你会根据这种喜好走得越来越偏。

真正的爱自己，是**该吃苦的时候就要吃苦**。对于那些你不喜欢、但必须去面对和完成的事情，我们就要去做。

## 行动的依据：需要而非喜好

我们做一些事情，是因为**它必须去做，应该去做**，而不是取决于你喜欢与否。

例如，用代码来描述这个选择逻辑：
```python
if task.is_necessary():  # 判断事情是否必要
    do(task)             # 必须做就执行
# 而不是：
if i_like(task):         # 仅凭喜好判断
    do(task)
```

你不喜欢吃某种苦的食物，而喜欢吃甜的。如果完全依照喜好，只吃甜食，请想一想，你的身体状态会发生什么样的改变。

## 长期后果与自然规律

长期这样下去，会发生什么样的变化？很多时候，“我喜欢干嘛就干嘛”这种想法，并不是爱自己，也不是为了自己好。

我们应该做什么、不应该做什么，应根据**自然的规律**和**生活的需要**。你该去面对的东西就去面对，不该面对的东西，才选择放弃。

## 通往更好的自己

遵循这样的规则，就能慢慢地让自己变得越来越好。

---

本节课中，我们一起学习了“爱自己”的真谛。它不在于放纵喜好，而在于勇敢承担必要的责任，依据客观规律和真实需要来行动，这才是通往持久成长和幸福的正确路径。
