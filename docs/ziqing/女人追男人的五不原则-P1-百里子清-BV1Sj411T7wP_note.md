## 原则一：切勿卑微祈求 🙅‍♀️

上一节我们介绍了课程概述，本节中我们来看看第一个原则。追求男性时，首要原则是避免卑微地祈求对方的感情。

## 原则二：坚守个人原则与底线 ⚖️

在明确了平等姿态的重要性后，我们进入第二个原则：维护自我。

## 原则三：屏蔽外界闲言碎语 🧏‍♀️

当你开始坚守自我时，可能会面临外界的干扰。因此，第三个原则是关于应对外部声音。

*   倾听这些言论会打乱你的追求节奏，影响心情。
