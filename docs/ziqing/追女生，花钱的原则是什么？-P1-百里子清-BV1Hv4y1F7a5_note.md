## 核心原则

以下是需要把握好的几个核心原则。

### 1. 量力而行原则 💪

有多大能力，就在能力范围内进行消费。不要伪装成经济实力远超自己的人。合理开销是关键。

**公式：** `你的恋爱开销 ≤ 你的经济承受能力`

例如，你月收入约为6000-7000元，那么在追求女生时，每月为此花费5000-6000元是不可行的。这会严重影响你的基本生活。因此，所有花费都应在你的承受范围之内。

上一节我们明确了恋爱需要开销，本节中我们来看看如何控制开销的尺度。量力而行是首要原则。

### 2. 区分大小开销原则 ⚖️

小钱可以花，但涉及金额较大的项目则需谨慎。

这一点很好理解。在追求阶段，双方关系尚未确定，即便之后成为男女朋友，对于某些大额消费也应保持理性。

例如，如果女生要求你为她购买一部昂贵的手机，而这超出了你的支付能力，那么就不应该为此买单。特别是在追求阶段，这类大额开销不应由你承担，这属于她个人的消费。

平常约会中的小额开销，如吃饭、喝咖啡等，可以由你支付。核心原则是：**大开销不给，小开销可以给**。

### 3. 应对“闺蜜局”原则 👭

在追求初期，女生有时会带着闺蜜或朋友一同赴约。

如果是初次见面，简单的吃饭等开销，你可以承担。这是一种礼貌。

但是，如果女生是为了在朋友面前炫耀或撑面子，从而产生了远高于正常水平的消费（例如去非常昂贵的场所），那么这笔钱就不应该由你支付。

如果她想要面子，应该自己承担成本。若她要求你来承担这种非必要的“面子消费”，从某种角度说明她不够尊重你。追求这样的女生，意义可能不大。

我们追求女生，并非舍不得花钱，而是不应该花这种“冤枉钱”。

### 4. 处理对方经济困难的原则 🤝

在追求过程中，如果对方遇到紧急的经济困难向你求助。

这种情况下，你可以考虑将钱**借给**她，但必须根据实际情况判断。

这种事情不常发生，但如果真的遇到，需要你三思而后行。

**判断逻辑：**
*   如果你们感情基础较好，对方回应积极，可以适当提供帮助。
*   如果你觉得风险很大（例如，对方可能不还钱或动机不纯），那么可以选择不借。

把握好“救急不救穷，且需评估风险”的原则即可。
