# 课程名称：如何面对他人的负面评价 - P1 🧘‍♂️

在本节课中，我们将学习如何有效应对来自他人的负面评价。核心在于转变我们看待和处理这些评价的思维方式，从而减少其对我们情绪和生活的干扰。

## 理解我们的第一反应 🤔

当我们收到一个负面评价时，大多数人的第一反应是寻找方法去化解它。我们通常认为，只要成功化解，这个负面评价对我们的影响就会变小。

## 化解行为的悖论 🔄

上一节我们介绍了面对负面评价的本能反应，本节中我们来看看试图“化解”行为可能带来的问题。

如果我们采取行动让负面评价消失或离开视线，反而会发现脑海中更频繁地想起它，并刻意记起这件事。这意味着，你越是试图化解它，就越是发现自己无法真正化解。

以下是导致这种现象的原因：

*   **过度投入精力**：我们将过多精力和注意力集中在“化解”这件事上。
*   **期望值过高**：我们误以为上述行为能达到消除影响的目标，这实际上是对结果抱有过高期望。
*   **行为过当**：我们在这个事情上做得有点过了，反而强化了它的存在感。

## 重新定义“评价”的视角 👁️

前面我们探讨了强行化解评价的困境，现在让我们换个角度审视“负面评价”本身。

从另一个角度看，所谓的“负面评价”只是你的第一反应，你认为它是负面的。但对于做出评价的人而言，他可能认为自己的评价是正向的，只是在表达个人观点。每个人的思维方式始终是不同的。

因此，当我们不去定义何为正面或负面评价，而是选择接纳它，仅仅将其视为一种评价、一种看法、一种理解或一种观点时，情况就会发生变化。

以下是视角转变后的关键改变：

*   **停止刻意行动**：我不会再刻意去做一些事情来消除这种“负面评价”。
*   **接纳存在**：因为我接纳这种评价的存在。
*   **内心归于平静**：我自己不再为此揪心，状态回归平静。
*   **转移关注点**：我会把注意力和关注点用在其他更有价值的方面。

## 接纳带来的格局改变 🌊

上一节我们学会了接纳评价，本节中我们来感受这种接纳带来的深远影响。

当你开始接纳并接受这一切时，你会发现自己的整个格局都变得不同。这种情况下，你会感受到一种积极的自我改变，格局也随之提升。此时，我们就像水一样，能够包容万物。

当你拥有如此大的度量时，你会发现自己在看待任何事情时，都会有新的领悟和感受。你对这些事情的看法，也将融入不一样的理解。

---

**本节课总结**：本节课中我们一起学习了应对他人负面评价的新路径。关键在于停止将其标签化为“负面”，并放弃“化解”的执念，转而练习**全然接纳**——将其视为一种普通的观点表达。通过这种视角转换，我们能减少内心冲突，将精力重新导向更有益的方向，从而实现个人格局的成长与内心平静。
