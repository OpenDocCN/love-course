# 别人的认可重要吗？-P1-百里子清-BV1JR4y1m7d9_note

在本节课中，我们将探讨一个普遍存在的心理议题：我们是否应该过度追求他人的认可。我们将分析这种追求背后的心理动因及其可能带来的负面影响，并引导你转向更健康、更可控的自我认可。

---

很多人总是挖空心思，想要获得别人的认可。

实际上，你越是想获得别人的认可，你得到的往往是不认可与失望。你把更多的时间和希望寄托在别人身上，从而忽略了自己。

但是，我们每个人真正应该获得的，是自己的认可。

获得别人的认可是一个**未知变量**，而获得自己的认可，则是自己可以掌握和提升的。追求他人的认可，就如同将希望寄托于一个未知数 `X` 之上。

那么，请你想一想，如果你依赖于此，你每天都会在担惊受怕和焦虑中度过。这难道不是在浪费自己的时间吗？

所以说，获得别人的认可，并没有我们想象的那么重要。别人认可你或不认可你，你依然是你。

如果你能想明白这个事情，你就会明白：获得别人的认可，根本就不是一件非做不可的事情。它没有必要去做，本质上是一种无用功。

当你冷静下来，认真思考一下，你自然就能发现这个问题了。

然后你就会知道：我不会再去在意别人的认可。

---

本节课中，我们一起学习了过度追求他人认可的弊端。关键在于认识到，自我认可才是可控且可持续的内心力量之源，而将他人的评价视为生活的重心，只会带来焦虑与消耗。希望你能将关注点回归自身，建立稳固的自我价值体系。
