# 回归本我 - 课程 P1 🧘

在本节课中，我们将要学习如何应对外界的负面评价，并探讨为何将时间和精力回归自身是更明智的选择。核心在于理解：**他人的评价并不定义你，你的自我关注和成长才是关键**。

---

当有人对你说坏话时，记住不要向他解释。许多人遇到这种情况，会去找这个人辩解，声称自己并非对方口中的样子，或者自己没有那么坏。也有人会选择回怼，甚至与对方争吵。

其实这没有必要。别人说我的坏话，我就当作没听见。即便听见了，我也装聋作哑。你越是与他较劲，他就越会认为自己说得对，从而在他心中，你就被固化为那个形象。我不解释，也没有必要向你解释。你说什么就是什么。

很多时候，我们应该把更多的时间和精力用在自己身上，少去理会这些事情。因为在这个世界上，有人会说你好，也有人说你不好。说你好与说你不好，其实都没有太大的关系。有时，有人说你好，也未必是真的为你好，或者真的认可你的好。有人说你坏，他也不一定是真的觉得你坏。

因此，很多时候没必要去搞清楚他们究竟为何要这样说。我只需要关注我自己就足够了。我关注我自己的生活，不去和他们计较太多。把更多的时间和精力回归到自己身上，将这些资源都用在自己身上，去做自己喜欢做的事情。你会发现，这个世界就会变得越来越好。

---

本节课中，我们一起学习了应对外界负面评价的心态与方法。核心在于**不辩解、不纠缠，将注意力回归自我**。通过关注自身的生活与成长，而非他人的言论，我们能够获得更平和的心境和更积极的人生。
