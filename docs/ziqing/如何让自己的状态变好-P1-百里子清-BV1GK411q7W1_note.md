# 课程：如何让自己的状态变好 - P1 🧠

在本节课中，我们将要学习一个关于改善个人状态的核心心态。许多人在状态不佳时，会陷入抗拒和焦虑，反而让情况变得更糟。我们将探讨如何通过接纳现状来开启积极的转变过程。

## 接纳现状：转变的第一步

上一节我们提到了状态不佳时的常见困境，本节中我们来看看解决问题的第一步：接纳。

当我们状态不好时，首先需要允许自己处于这个状态。很多人在状态不好时，总是否认这一点。他们强迫自己必须立刻变好，认为只有状态好了才能开始改变。

其实，想让状态变好，必须先承认当前状态不好。这背后的逻辑是：只有正视问题，才能开始解决问题。

## 逻辑解析：以行动打破僵局

为了理解这个逻辑，我们可以看一个具体的例子。

例如，如果我想拍一个视频，但觉得自己现在拍不好，于是选择不去拍。如果永远不去尝试，就永远不知道拍出来的实际效果如何。

我感觉状态不好，但有时这种过度在意和感觉，反而会让状态越来越差。此时，如果我能不顾虑这些，直接去做这件事，允许自己以不好的状态去表现，就会发现：

将这种不好的状态表现出来，本身就是一种好的开始。因此，当我们状态不好时，学会承认这一点，状态才会慢慢变好。这是一个过程，也是一种转变。

## 核心规律：波动与转变

任何事情都遵循这个规律：不好和好之间，并非固定不变。

它呈现为一种波动的形状。当我们承认现状并开始行动时，转变就发生了。之后，状态便会慢慢地变得越来越好。

---

本节课中我们一起学习了改善状态的关键心态：**先接纳，后转变**。通过承认并允许自己状态不佳，并在此基础上开始行动，我们就能打破负面循环，引导状态向积极的方向波动和发展。记住，转变是一个过程，始于对现状的坦然接受。
