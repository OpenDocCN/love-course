# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如数字、文字）。你给盒子起的名字就是**变量名**，而盒子里装的东西就是**变量值**。

在Python中，你使用等号 `=` 来给变量赋值。其核心操作可以表示为以下公式：

**`变量名 = 变量值`**

例如，我们写 `name = “子清”`，就意味着创建了一个名为 `name` 的“盒子”，并在里面存放了“子清”这个文字。

## 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的种类，这就是**数据类型**。Python有一些内置的基本数据类型，最常见的有以下三种。

以下是三种最基础的数据类型介绍：

1.  **整数**
    *   **含义**：没有小数部分的数字，可以是正数、负数或零。
    *   **代码示例**：`age = 25` 或 `temperature = -10`

2.  **浮点数**
    *   **含义**：带有小数点的数字，用于表示更精确的数值。
    *   **代码示例**：`price = 19.99` 或 `pi = 3.14159`

3.  **字符串**
    *   **含义**：由一系列字符组成的文本。在Python中，必须用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   **代码示例**：`name = “Alice”` 或 `greeting = ‘Hello World!’`

## 如何查看和转换类型？ 🔄

理解了不同类型后，你可能会问：我怎么知道一个变量是什么类型？以及，我能否改变它的类型？

Python提供了两个有用的内置函数来解决这些问题。

以下是相关的操作说明：

*   **使用 `type()` 函数查看类型**
    *   这个函数会返回变量的数据类型。
    *   **代码示例**：
        ```python
        height = 180
        print(type(height)) # 输出：<class ‘int’>
        ```

*   **使用类型转换函数改变类型**
    *   你可以将一种数据类型转换为另一种，例如把字符串“123”转换成整数123。常用的函数有 `int()`, `float()`, `str()`。
    *   **代码示例**：
        ```python
        number_str = “100”
        number_int = int(number_str) # 将字符串转换为整数
        print(number_int + 50) # 输出：150
        ```

## 动手试一试 ✍️

现在，让我们将以上知识结合起来，完成一个简单的练习。

请尝试在Python环境中输入以下代码，并观察运行结果：

```python
# 创建几个变量
my_name = “子清”
my_age = 20
my_height = 1.75

# 打印它们
print(“我的名字是：”, my_name)
print(“我的年龄是：”, my_age, “岁”)
print(“我的身高是：”, my_height, “米”)

# 查看类型
print(type(my_name))
print(type(my_age))
print(type(my_height))

# 尝试一个类型转换
age_next_year = my_age + 1
print(“明年我就”, age_next_year, “岁了。”)
```
