# 如何看待自己的优点和缺点：不把优点当优点，不把缺点当缺点

在本节课中，我们将学习一种独特的视角：如何通过不刻意区分优点与缺点，来获得对自我和世界的全新领悟。这种方法的核心在于放下固有的评判，以更开放的心态去感受和接纳。

---

上一段我们提到了放下对优缺点的执着，本节中我们来具体看看这种观念如何运作。

当你不再将自己的优点视为优点，也不再将自己的缺点视为缺点时，一种领悟便会悄然产生。此时，一种潜移默化的改变开始发生，你会发现自己进入了一种不同的状态。

这种状态非常奇妙。同时，你也会感受到，自己所感知到的世界和事物变得与以往不同。

实际上，所谓的优点和缺点，常常是一种动态的状态。例如，你认为自己拥有的某项技能或特质是一种优势，但在他人眼中，它可能恰恰是一个缺点。

这背后的原因在于，他人看问题的角度与你不同。我们所感知的社会与世界，本质上是我们个人主观的感受。

当你放弃给事物贴上“优点”或“缺点”的标签，仅仅去感受、感悟并全然接纳一切时，你会发现改变不仅发生在自己身上。

你周围的世界似乎也随之发生了变化。这是因为，正如我们刚才所说，每个人所感知的世界，才是他真正“看到”和“知道”的世界，而每个人的感知都是独一无二的。

因此，当你开始接纳一切，一种悄然的改变便会发生。这种变化非常微妙，而这个过程本身就是一种领悟。

你会发现，自己变得和以前不一样了。我们对于事物的看法，以及基于这些看法所做的后续选择，都会发生改变。

这种改变是一种经历和过程。它会循序渐进地引导我们回归到一种平静、自然、顺其自如的状态。

这样一来，我们的感悟和生活品质便有可能发生积极的转变。而这种转变，或许正是我们所追求的那份平静与幸福。

---

本节课中，我们一起学习了通过超越“优点”与“缺点”的二元对立来认识自我的方法。关键在于放下主观评判，全心接纳，从而在动态的感知中获得个人成长与内心的平静。
