# 做什么都要准备好了才开始？是一

## 概述

在本节课中，我们将要学习一种常见的心理障碍——“凡事都要准备好了才开始”。我们将剖析这种心态的本质，理解它为何会阻碍我们的行动，并探讨如何通过立即行动来打破这个循环，从而真正开始并完成我们想做的事情。

## 什么是“准备陷阱”？🤔

上一节我们概述了课程主题，本节中我们来看看这个问题的具体表现。

凡事都要准备好了才开始，往往做不成一件事。什么都要准备好，说白了就是给自己找一个借口，就是不敢开始，害怕失败。脑海里总是出现“我得准备充分了”、“我得考虑周全了”、“我得三思了”、“我不能轻易去做这件事了”这些声音。

## “准备陷阱”带来的后果

过去，我也曾陷入这种心态。后来我发现自己永远也没有办法完全准备好。这种心态会导致几种典型的困境：

以下是几种常见的困境表现：

*   **错过时机**：有些时候认为自己准备好了去做的时候，却发现这个时机已经过了。
*   **内心抗拒**：有的时候好不容易下定决心要去做的时候，却发现自己内心深处又冒出一个声音，告诉自己还没有准备好。
*   **动力耗尽**：还有的时候你会发现自己耗费了很多时间和精力去准备，当自己真正想要去做的时候，却发现自己不想去做这件事了。

这种情况真的很尴尬。因为**凡事都有风险**，光想着好事不愿意承担风险，这怎么可能成功呢？

## 如何打破“准备陷阱”？⚡️

理解了问题的根源和后果后，本节我们来看看解决方案。

“当我准备好了，我才开始去做某件事情”，这一种想法本身就是一种内耗，是一种自我消耗。现在我不这样做了，如果我想做一件事情，我就会立马去做。

其核心行动公式可以概括为：
**想法 -> 立即行动**

例如，我现在想拍一个视频表达一些观点，我就会立马去拍。你想要得到什么，首先得付出行动。没有开始就不会有结果。总是在想，总是在准备，没有去行动，永遠也不會有結果。

## 总结

本节课中我们一起学习了“准备陷阱”这个概念。我们认识到，过度追求“完全准备好”往往是一种逃避开始的借口，它会让我们错过时机、消耗动力并最终一事无成。关键在于将“想法 -> 准备 -> 行动...”的漫长循环，转变为“想法 -> 立即行动”的简洁模式。记住，**开始行动是通往任何结果的唯一起点**。
