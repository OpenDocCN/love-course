在本节课中，我们将要学习一个核心思维模式：如何避免因过度担忧未知结果而影响行动，从而更顺利地达成目标。我们将探讨“以结果为导向”的思维如何成为阻碍，并学习如何转换心态，专注于过程本身。

## 为何“以结果为导向”会成为阻碍

上一节我们介绍了课程的主题，本节中我们来看看问题的根源。我们很多时候受很多事情影响，就是因为“以结果为导向”的思维方式。

去做一件事情之前，我们总会想东想西。例如，会思考：**如果我这样去做，会造成什么样的结果？** 如果出现了那个结果，我又该怎么去承受？

这种以结果为导向的思维会滋生出很多问题，让我们联想出许多尚未发生的状况。而这些结果和状况都是未知的，我们其实并不知道它们是否会真实发生，但我们却提前进行了联想。

## 未知联想如何影响行动

联想这些未知的结果，就会对我们执行当前这件事造成直接影响。这种以结果为导向的担忧，会影响到当下的我们。

受到影响后，我们可能就不会开始做这件事，或者在做事的过程中打折扣。这里的“打折扣”可以理解为：**行动效率 = 预期投入 × 心态系数（当心态受担忧影响时，系数 < 1）**。

在打折扣的过程中，我们的状态会变得越来越不好。在这样一个恶性循环中，自身的状态持续下滑，最终可能导致彻底放弃或失败。

## 解决方案：专注于过程而非结果

那么，如何不被其他东西影响？关键在于改变我们的关注点。

去做一些事情的时候，不要以结果为导向。不注重结果，结果就不会提前影响你。你就不会被这些未知的联想所控制或影响。

反过来，以坦然的姿态去面对这一切。你可以用代码来模拟这种心态的转变：

```python
# 旧模式：被结果困扰
def old_mindset(action):
    unknown_result = imagine_outcome(action)  # 联想未知结果
    if is_negative(unknown_result):           # 判断结果好坏
        motivation = 0                         # 失去动力
    return motivation

# 新模式：专注于行动本身
def new_mindset(action):
    focus_on_process(action)                  # 专注过程
    motivation = 1                            # 保持动力
    return motivation
```

## 转变心态带来的积极力量

当你不再被结果束缚，你还可以反过来影响其他事物。你能够影响很多东西，你的影响会成为一种正向的力量。

这种正向力量会逐渐为我们积蓄能量，增加我们的能量值。用公式表示可以是：**个人能量增长 = 正向行动 × 时间**。

我们自己的状态就会因此变得越来越好，形成一个积极的增强回路。

## 核心要点总结

以下是本节课的核心要点列表：

*   **问题根源**：以未知的、想象中的结果为导向，会引发担忧并阻碍行动。
*   **影响机制**：对未知的联想消耗当下能量，导致行动打折或放弃。
*   **解决关键**：将注意力从“未知结果”转移到“当下过程”。
*   **心态代码**：执行 `focus_on_process(action)`，而非 `imagine_outcome(action)`。
*   **最终收益**：摆脱控制后，个人状态进入正向循环，能量得以积累。
