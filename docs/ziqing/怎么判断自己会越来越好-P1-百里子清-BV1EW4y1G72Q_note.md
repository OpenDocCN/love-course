# 课程名称：如何判断个人成长趋势 - P1 🧭

在本节课中，我们将学习一个简单而有效的方法，用于判断个人状态是否正在向积极方向发展。我们将通过核心概念和具体迹象来识别成长趋势，并讨论如何在顺境中保持警惕，以实现持续进步。

## 领悟与顿悟：成长的核心标志 🌱

上一节我们介绍了课程目标，本节中我们来看看成长的核心标志。判断自己是否变得越来越好，有一个简单的判定标准：观察自己是否有所领悟或顿悟。

例如，你之前思考某件事，一直想不通，处于死胡同和纠结状态。突然你发现自己领悟了，能想明白了。能放下的就放下，不能放下的也释然了。当你处于这种比较轻松的状态时，你就会发现自己正在变得越来越好，朝着好的方向发展。

**核心状态公式**：`个人状态 = 领悟程度 + 情绪释然度`

## 积极状态与消极迹象的对比 🔄

当你感觉自己比较轻松、比较OK，做任何事情都很自然，而不是处于纠结或扭曲状态时，这就是积极的迹象。

相反，有时我们做一件事，总感觉哪里不对劲，但又说不出来。这种模糊的不适感，就是状态可能变糟糕的迹象。当你做事情变得越来越顺利时，你就会发现，这正是你变得越来越好的标志。

以下是积极状态与消极迹象的关键区别列表：

*   **积极迹象**：做事顺畅、心态轻松、想法自然通透。
*   **消极迹象**：感到莫名别扭、思路陷入死胡同、持续纠结。

## 顺境中的警惕与长期策略 📈

当然，在这种状态非常好的时候，我们需要警惕。因为“否极泰来”，当你发现状态极佳时，就要注意可能的下坡路。人生中任何事情都处于波动状态。

我们要学会在波段的高峰时，注意向下的节奏，及时“刹车”；而在上升时，步伐慢一点，同时留意向上的迹象。这样，我们就能顺着人生的波路或波纹，让自己的人生慢慢变得越来越好，活得别样精彩。

**人生波动策略代码描述**：
```python
if current_state == "peak": # 处于高峰时
    action = "brake_and_prepare" # 刹车并准备
elif current_state == "rising": # 处于上升期时
    action = "pace_slowly_and_observe" # 放缓步伐并观察
else:
    action = "keep_going_and_adjust" # 持续前进并调整
```

本节课中，我们一起学习了如何通过**领悟感**和**做事顺畅度**来判断个人成长趋势。我们对比了积极与消极的迹象，并强调了在顺境中保持警惕、遵循人生波动规律的重要性，以实现持续而稳健的进步。
