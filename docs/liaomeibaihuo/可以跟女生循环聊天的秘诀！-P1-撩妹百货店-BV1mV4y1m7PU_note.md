## 概述 📖

循环聊天法的核心在于构建一个可持续的对话结构。它不仅仅是寻找话题，更是关于如何将对方的话语转化为新话题，从而让对话像滚雪球一样不断进行下去。

上一节我们介绍了课程的目标，本节中我们来看看循环聊天法的基本模型。

## 核心模型：话题循环公式 🔄

该方法的核心可以总结为一个简单的公式：

**`新话题 = 对方回应 + 延伸/关联 + 开放性问题`**

这个公式意味着，你需要从对方的上一句话中提取关键信息，将其与你已知的信息或普遍经历关联起来，然后抛出一个需要对方详细回答的问题，从而开启新一轮的对话循环。

理解了核心公式后，接下来我们看看具体如何操作。

## 实施步骤详解 🛠️

以下是应用循环聊天法的三个关键步骤。

1.  **倾听与抓取关键词**
    全神贯注地倾听对方的回答，从中识别出可以深入探讨的“关键词”。这些词通常是名词、动词或表达感受的形容词。

2.  **关联与延伸**
    将抓取到的关键词与你自己的经历、知识、或一个相关的小故事进行关联。这一步的目的是展示共鸣，并为提问搭建桥梁。

3.  **提出开放性问题**
    基于你的延伸，提出一个以“如何”、“为什么”、“你觉得”等开头的问题。避免使用只需“是”或“否”回答的封闭式问题。

掌握了步骤，我们通过一个实例来加深理解。

## 实战演练示例 💬

假设对方说：“最近工作好累，一直在加班。”

*   **步骤1：抓取关键词**：“工作”、“累”、“加班”。
*   **步骤2：关联延伸**：“听起来项目很紧张啊。我上个月也有段类似经历，连续加班后周末简直想彻底躺平。”
*   **步骤3：提出开放性问题**：“你们团队是为了赶什么新项目吗？/ 你一般加班后会做什么来放松自己？”

通过这个例子，你可以看到对话如何从一个简单的陈述，自然地导向更深入、更个人的交流。
