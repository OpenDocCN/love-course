在本节课中，我们将学习如何通过一个简单的“撩妹百货店”游戏机制来模拟互动。这个机制的核心是使用一个随机函数来生成不同的对话选项，从而影响游戏进程。

---

## 核心机制：随机对话生成 🎲

上一节我们介绍了课程主题，本节中我们来看看实现互动的核心代码逻辑。其核心是使用 `Math.random()` 函数来随机选择一个回应。

以下是核心代码公式：
```javascript
let responseIndex = Math.floor(Math.random() * responses.length);
let selectedResponse = responses[responseIndex];
```
这段代码会从预设的回应数组 `responses` 中随机选取一个元素。

---

## 构建回应列表 📝

理解了随机选择机制后，我们需要一个可供选择的对话内容库。这些内容构成了互动的素材。

以下是构建回应数组的示例：
```javascript
const responses = [
    "你今天看起来真精神。",
    "这家店的咖啡味道不错。",
    "我好像在哪里见过你。",
    "天气真好，适合出来走走。"
];
```
数组中的每一个字符串代表一个可能的对话选项。

---

## 流程控制与分支 🌳

有了对话库和选择机制，接下来需要根据玩家的不同选择来推进故事。这通常通过 `if...else` 或 `switch` 语句来实现。

以下是处理不同回应的逻辑结构：
```javascript
if (selectedResponse === responses[0]) {
    // 触发剧情分支A
} else if (selectedResponse === responses[1]) {
    // 触发剧情分支B
}
// ... 其他分支
```
根据随机选中的回应，游戏将进入不同的叙事路径。
