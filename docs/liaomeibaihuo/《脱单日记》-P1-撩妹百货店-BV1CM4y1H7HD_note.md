# 《脱单日记》课程 P1 - 撩妹百货店 🏪

在本节课中，我们将学习如何通过一个简单的“撩妹百货店”游戏来理解编程中的基本概念，特别是**变量**和**条件判断**。我们将使用一个模拟对话的场景，来演示如何根据不同的输入做出不同的回应。

---

## 概述

本节课的核心是模拟一个商店对话系统。顾客进入商店后，店员会根据顾客想购买的商品，给出不同的回应。这个简单的互动包含了编程中最基础的两个要素：存储信息的**变量**和根据信息做决定的**条件判断**。

## 从对话到代码

上一节我们概述了本节课的目标，本节中我们来看看如何将一段自然对话转化为计算机可以理解的逻辑。首先，我们分析一段店员与顾客的对话：

顾客说：“我想买点东西。”
店员问：“你想买什么？”
顾客回答：“我想买一个苹果。”

在这个对话中，“顾客想买的物品”是一个会变化的信息。在编程中，我们使用**变量**来存储这类信息。

我们可以这样定义一个变量：
```python
item = "苹果"
```
这里，`item`就是一个变量，它存储了字符串值`"苹果"`。

## 核心概念：条件判断

仅仅知道顾客想买什么还不够，店员需要根据具体的商品来回应。这就需要用到**条件判断**。

以下是条件判断的基本逻辑结构：
```python
if item == "苹果":
    print("苹果在左边的货架上。")
elif item == "香蕉":
    print("香蕉今天特价。")
else:
    print("抱歉，本店没有此商品。")
```
这段代码的意思是：**如果**变量`item`的值是“苹果”，就执行第一条打印语句；**否则如果**它是“香蕉”，就执行第二条；**否则**（即以上都不满足），执行最后一条。

## 构建“撩妹百货店”程序

现在，我们将变量和条件判断组合起来，构建一个完整的简单程序。

程序需要完成以下几步：
1.  提示顾客输入。
2.  将输入的值保存到变量中。
3.  判断这个变量是什么，并给出相应的回答。

以下是这个程序的一个示例代码框架：
```python
# 1. 获取顾客想买的商品
customer_want = input("你想买什么？")

# 2. 根据商品进行判断并回应
if customer_want == "苹果":
    print("苹果在左边的货架上。")
elif customer_want == "香蕉":
    print("香蕉今天特价。")
elif customer_want == "巧克力":
    print("巧克力是送给她的好礼物。")
else:
    print("抱歉，本店没有" + customer_want + "。")
```

## 关键要点总结

本节课中我们一起学习了编程的两个基石：
1.  **变量**：用于存储程序运行中可能变化的数据，例如`item = "苹果"`。
2.  **条件判断**：使用`if`, `elif`, `else`语句让程序根据不同情况执行不同操作。

通过“撩妹百货店”这个生动例子，我们可以看到，即使是最复杂的软件，也是由这些基本的“询问-存储-判断-回应”逻辑构建而成的。理解了这个流程，就迈出了学习编程的第一步。
