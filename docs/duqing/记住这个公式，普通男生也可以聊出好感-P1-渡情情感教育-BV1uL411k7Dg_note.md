## 公式应用解析
让我们通过一个例子来解析这个公式。假如女生给你发消息说：“最近工作好忙，烦死了。” 你应该怎么回复？

大部分脱不了单或者情商低的兄弟，第一反应是试图解决问题。他们会回复：“你怎么突然忙起来了？需不需要我帮你做些什么？” 总是想着帮女生解决问题，这很容易被女生当成工具人。

其实，女生在跟你分享自己的状态，只是为了排解负面情绪。你需要做的是接纳她的情绪。

## 公式实战演练
我让那位学员这样回复女生：“我最近很轻松，公司来了个实习生，帮我分担了很多工作。羡慕吧。”

以下是这个回复的拆解：
*   **“很轻松”** 是 **分享状态**。
*   **“实习生分担工作”** 是 **描述行为**。
*   **“羡慕吧”** 是 **表达感受**。

女生可以从任意一个角度切入话题。于是女生回复：“我要是能像你一样就好了。” 这是很常见的回复。

我们继续通用公式。学员回复：“不用羡慕我，我天天闲得发慌，哪有你过得那么充实。” 这时，女生不会觉得你闲，反而会觉得你在凡尔赛。

女生回复：“你就凡尔赛吧，来分分钟内啪你。” 可以看到，女生的情绪已经被调动起来了。
