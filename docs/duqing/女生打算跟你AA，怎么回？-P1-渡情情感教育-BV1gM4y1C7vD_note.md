## 一、 女生提出AA的两种意图

2.  **故意曲解**：将“AA”从单纯的“金钱分摊”曲解为“时间与金钱的共同分摊”，为后续互动埋下伏笔。

**策略分析**：

**核心技巧**：抓住机会，大胆但幽默地植入暧昧感。很多男生把握不住这样的时机。
