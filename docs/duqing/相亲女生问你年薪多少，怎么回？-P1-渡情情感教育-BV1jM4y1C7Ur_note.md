# 相亲攻略课 01：当被问及年薪时，如何巧妙回应？💰

在本节课中，我们将要学习在相亲场景下，当女生直接询问你的年薪时，如何避免陷入查户口式的尴尬对话，并通过建立情感共鸣和巧妙转换框架，给出一个既真诚又吸引人的回答。

---

很多男生在相亲时遇到女生询问年薪，会感到不知所措。一种常见的错误是过于直白地回答，例如“我年薪6万左右”。这种查户口式的回答，会让本身就像查户口的对话氛围变得更加枯燥。如果你年薪不高，这种回答很难为你加分。

另一种错误是试图用玩笑搪塞过去，比如“我一秒钟上下几十万”。这种夸张的玩笑缺乏诚意，可能会让一些别有用心的人觉得你是在炫耀，反而将你视为潜在的“提款机”。

那么，女生问收入真的只在乎具体数字吗？其实不然。她的重点往往在于评估你未来的付出意愿和慷慨程度。如果你表现得斤斤计较，即使身家过亿，也可能不被考虑。

有些男生会选择模糊应对，例如当被问“你年薪多少啊？”时，回答“你猜啊”。在女生追问后，又说“以后我们在一起你就知道了”。这种方式会让女生感到困惑甚至生气，她可能会认为你缺乏自信或不诚实，从而让原本的好感消失。

尤其是在相亲初期，这种不流畅的沟通很容易让女生感到反感，无论她最初对你印象如何。因此，关键在于跳出问题本身，与女生的情绪和潜在需求建立共鸣。

如果对话一直干巴巴地停留在事实层面，女生很难提起兴趣。女性通常是情绪导向的，我们需要将这个“死亡问题”转化为展示高情商和建立吸引力的机会。

以下是核心的回应策略：**通过表达当下感受和转换对话框架来巧妙回应**。我们来看一个成功案例。

同样是女生问“年薪多少”，这位男生回答道：“你问的是有对象的时候，还是没对象的时候？”

这句话首先说出了自己的真实感受——被直接问收入有点突兀。它没有回避问题，但巧妙地设置了前提：我的收入状况取决于我的感情状态。这瞬间反客为主，掌握了对话主动权。

女生很可能会接着问：“那区别大吗？”

此时，男生可以坦诚地回答单身时的收入，例如：“没对象的时候，直白点说，8万左右。” 这展示了诚实。

当女生好奇“有对象的时候呢？”，男生可以幽默而巧妙地将她带入情境：“这你不该问问你自己吗？你把我工资卡放哪了？”

这种调侃的回应，瞬间将女生带入“女朋友”的角色，用幽默化解了直接谈钱的尴尬，同时暗示了愿意为伴侣付出的态度。关键在于**建立并主导对话的框架**，将她的难题转化为双方互动的游戏。

大家需要学习的正是这种**树立对话框架的思维**。具体话术可以千变万化，但其核心是**共情、幽默和框架转换**。
