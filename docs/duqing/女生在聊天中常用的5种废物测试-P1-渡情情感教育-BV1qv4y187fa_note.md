## 1. 打压测试 🛡️

上一节我们介绍了课程概述，本节中我们来看看第一种常见测试：打压测试。女生可能会通过贬低你的身高、体重或工作来测试你的自信和框架。

面对这种测试，关键在于保持自信，坚持自我框架。如果女生一两句话就能打压到你，那么无论你付出多少，都难以获得她的青睐。

**正确应对公式**：
当女生说“你太胖了”时，你可以回复：

## 2. 服从型测试 ⚖️

了解了如何应对打压，接下来我们分析第二种：服从型测试。这种测试表现为女生将你当作“工具人”，例如让你帮忙搬家或做文件。

想让女生心动，你需要做的不是一味听话和顺从，而是进行等价交换并调动她的情绪。

以下是应对方法：
*   **核心策略**：等价交换 + 情绪调动。
*   **示例对话**：
    *   女生：“你可以帮我搬家吗？”
    *   男生：“帮你没问题。不过我很口渴，你先帮我把蜜雪冰城搬回家吧。”

在成功应对了前两种测试后，当女生对你有好感、情绪被调动起来时，可能会进行花心测试。例如，她会问：“你平时都是这么撩女生的吗？”或“你对每一个女生都这么好吗？”

**正确应对代码**：

if 女生问：“你平时都是这么撩女生的吗？”

## 4. 好人卡测试 🃏

第四种测试是好人卡测试。女生可能会说“你对我真好”或“我一直把你当哥哥”。这通常分为两种情况，需要区别应对。

以下是两种情况的处理方式：
*   **情况一（试探）**：女生不确定你是否喜欢她。你可以用撒娇的方式引导双向奔赴，例如回复：“我也觉得我对你挺好的，可是你对我一点都不好。”
*   **情况二（委婉拒绝）**：女生知道你喜欢她，想拒绝。此时切忌死缠烂打，应以退为进，用幽默化解。例如回复：“那妹妹以后要听话，不然家法伺候。”

## 5. 自我否定测试 🪞

最后，我们来看自我否定测试。女生常会说“我又胖了”或“我都没以前漂亮了”。这同样需要根据她的真实意图来回应。

这个测试也分两种情况，应对策略不同：
*   **情况一（凡尔赛）**：女生其实很自信，只是想被夸。你可以用调皮的方式调动情绪，例如说：“那你就多吃点吧。”
