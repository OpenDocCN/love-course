掌握了制造情绪的技巧后，接下来是课程的重中之重，请务必仔细聆听。

很多人都说“真诚是必杀技”。这句话本身没有错，但很少有人告诉你，**真诚本身也可以是一种方法和刻意营造的形象**。

在典型“直男”思维中，真诚意味着：
**真诚 = 我对你好 + 我给你买礼物发红包 + 我鞍前马后 + 我让你开心**

他们认为：“这就是我的真心，所以你理应和我在一起。” 这种行为本质上是站在**索取对方价值**的立场上，是一种受害者心态和穷人思维，因为它没有考虑对方是否需要这种形式的“真诚”。

而女人所需要的真诚，是站在**她的角度**去考虑。

我举个例子：假设你想请一个女人吃饭。

**错误示范（自我感动式）：**
> “明天咱们一起去吃火锅。为了和你见面，我专门推掉了客户的饭局。”

你以为这展示了真诚和付出，但女生可能会想：“又不是我让你推掉的。你推不掉客户饭局，跟我也没有关系。” 这反而会给她一种被胁迫、被道德绑架的感觉，好像必须和你约会。

**正确示范（需求对接式）：**
> “我看你朋友圈最近挺想吃火锅的。我知道一家非常小众但味道很正宗的店。我记得你周三晚上通常不加班？如果时间合适，我就预定位置了。”

根据她的需求（想吃火锅），展示你了解她的细节（周三不加班）。这样她才会感到舒服、轻松，甚至有些小感动。这才是她能感受到的“真诚”。
