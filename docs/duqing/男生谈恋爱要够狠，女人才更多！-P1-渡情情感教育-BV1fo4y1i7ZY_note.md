## 核心原则：不害怕失去 🚫

上一节我们介绍了课程的核心目标，本节中我们来深入探讨其根本原则。这个原则可以用一个简单的公式来概括：**自我尊重 + 设立底线 = 健康的关系**。简单来说，就是在恋爱中不要害怕失去对方。

当发生争吵时，如果你讲道理对方不听，那么可以选择停止争论，让双方冷静。如果对方对你呼来喝去，你可以选择不搭理，甚至反向提出你的要求。当对方态度恶劣时，你可以选择“晾着”他，无论他说什么都暂时不予回应。否则，之前建立框架的努力就会前功尽弃。

这就好比盖房子，地基没有打牢，房子就经不起风吹雨打。这个道理大家应该都明白。
