# 情感课程01：掌握两个核心感觉，轻松推进关系 🎯

在本节课中，我们将学习如何通过聊天有效吸引女生，并建立两种核心感觉，从而顺利推进关系。课程内容基于一个具体场景展开：当女生在社交软件上主动问你“在干嘛”时，如何高情商地回应。

---

上一节我们提到了女生主动开启对话是一个重要信号。本节中，我们来看看大多数男生的错误回复方式，并理解其问题所在。

以下是两种常见的错误回复示例：

1.  **“在想你啊。”**
    *   这种回复过于油腻，容易引起女生的反感。

2.  **“我在工作/打游戏呢。”**
    *   这种是封闭式回答，直接终结了话题，让女生无话可接，错失了聊天的机会。

这些回复的问题在于，没有从女生的心理出发。女生主动问“在干嘛”，本身就是一个想和你聊天的借口，意味着她对你有一定的兴趣。你的回复决定了是能把握机会升级关系，还是让对话变得乏味。

---

理解了常见错误后，我们来看一个学员的真实失败案例，并分析其根本原因。

学员当时回复：“吃饭了，你呢？”女生说：“我刚下班。”学员接着回：“嗯，回家注意安全。”之后女生便没有再回复。

这个对话看似礼貌，但毫无情绪波动，属于无效沟通。长期进行这种没有情绪的聊天，最终关系只会停留在朋友阶段，甚至变成陌生人。核心原因就是**没有提供情绪价值**。一旦出现能调动女生情绪的竞争者，你就会被淘汰。

---

那么，如何正确回复才能提供情绪价值呢？接下来，我们将介绍一个简单有效的回复公式。

记住这个核心公式：**表达行为 + 表达状态 + 表达感受**。其中，**表达当下的感受最为关键**。

我们通过一个成功案例来演示这个公式的运用。男生在社交软件上认识了一个女生，第二天女生主动问：“在干嘛呢？”

我指导男生套用公式回复：“**看到你问我在干嘛，我心情还不错。**”
*   “看到你问我在干嘛”是**行为**。
*   “心情还不错”是**感受**。
*   这句话直接、积极地表达了因她而起的愉悦情绪，能感染对方。

女生果然被调动了好奇心，回复：“为什么呀？”

我继续让男生深化：“**我刚才正准备打开手机找你，就突然弹出你的消息了，感觉有点默契。**”
*   “准备打开手机找你”是**行为**和**状态**。
*   “感觉有点默契”是**感受**。
*   这句话巧妙地将巧合转化为“默契”，创造了积极的情感链接。

通过这样的对话，女生会感觉和你聊天很舒服、很开心，从而从对你有好感逐渐变得依赖和更喜欢你。

---

本节课中我们一起学习了如何通过高情商的聊天思维吸引女生。关键在于让她拥有两种核心感觉：**舒适感**和**情绪上的愉悦感/默契感**。

只要你能在聊天中持续提供这种情绪价值，让女生拥有这两种感觉，关系推进便会水到渠成。最后再重申一遍，这不是套路，而是一种**高情商的思维方式**。熟练掌握“**表达行为+状态+感受**”这个公式，并在关键时刻灵活运用，就能让你在聊天中占据主动。

**公式总结：**
```
有效回复 = 表达当前行为 + 描述自身状态 + 传达内心感受
（重点：突出积极、与对方相关的感受）
```
