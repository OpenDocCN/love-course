理解了评估的概念后，我们来看看如何主动使用它。在一个两性交往互动的过程中，评估分为两个阶段。我们先讲比较明显的第二阶段评估。

一个完整的第二阶段评估遵循一个核心公式：

**问题 + 命题 + 评价 = 完整的评估**

让我用一个例子来解释这个结构：
1.  **问题**：你喜欢冒险吗？
2.  **命题**：我觉得冒险就是当你想去云南玩的时候，不定行程、不打行李，说走就走。我喜欢这种随性自由的感觉，也希望对方能跟上我的节奏。
3.  **评价**：（当女方回应后）看来你也是个喜欢刺激的人，这点我们挺合拍。

在这个结构中，命题部分是将你的问题与你所看重的品质（如随性、自由）联系起来，本质上是设立你的标准。

1.  **评估时女方表示“不配”或达不到标准怎么办？**
