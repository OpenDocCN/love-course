# 13-安小妖《好男人，你也要学会撩妹》：151-200：187.长期关系的核心机密——一个永不吵架分手的框架

## 概述
在本节课中，我们将学习如何维系长期关系，避免争吵和分手。

## 核心概念
**框架**：在关系中，确定各方的优先级和界限。

### 1. 了解女生特征
女生喜欢挑战你的框架来确认你在她心中的地位。

### 2. 制定框架
- 第一等级：家人
- 第二等级：事业
- 第三等级：兄弟、朋友
- 第四等级：女人

### 3. 框架的重要性
不按照框架执行会导致关系错位，引发争吵和控制。

### 4. 长期关系准则
1. **包扁结合**：甜蜜语与批评相结合，避免甜蜜语失效。
2. **决策力**：男生应扮演领导者角色，承担决策责任。
3. **不可预测**：给对方惊喜，避免预测到的好。
4. **保持独立**：不要受控制，展现自信。
5. **松紧有度**：给对方空间，避免过度控制。
6. **幼儿理论**：以小朋友的方式对待对方，激发童心和母爱。

## 总结
本节课中我们一起学习了如何维系长期关系，避免争吵和分手。希望这些技巧能帮助你建立更健康、稳定的关系。
