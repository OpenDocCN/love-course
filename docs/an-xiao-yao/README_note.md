### 1. 核心概念与定义

上一节我们介绍了课程目标，本节中我们来看看“安小妖”是什么。

“安小妖”是一个**虚拟角色**。它由一系列**属性**和**行为规则**定义。其核心目的是与用户进行**交互**。

我们可以用一个简单的**公式**来描述其本质：
**安小妖 = 属性集合 + 行为函数**

---

### 2. 核心属性拆解

了解了核心定义后，接下来我们详细拆解构成“安小妖”的几个关键属性。

以下是“安小妖”必须具备的三个核心属性：

*   **名称 (Name)**: 角色的标识符，固定为“安小妖”。
*   **性格 (Personality)**: 一套描述角色行为倾向的文本标签，例如“友好”、“好奇”、“乐于助人”。
*   **知识库 (Knowledge Base)**: 一个存储信息的数据结构，用于回答用户问题。

---

### 3. 构建流程

现在我们已经明确了“安小妖”的组成部分，本节中我们来看看如何一步步将其构建出来。

以下是创建“安小妖”的四个主要步骤：

1.  **初始化角色**：设定角色的名称和初始性格参数。
2.  **构建知识库**：向知识库中添加结构化的数据或文本信息。
3.  **定义交互逻辑**：编写函数，使角色能够根据用户输入和自身知识库生成回应。
4.  **测试与优化**：通过模拟对话测试角色行为，并根据反馈调整性格参数或知识库内容。

---

### 4. 一个简单的代码示例

理论需要实践来巩固。让我们通过一个高度简化的代码示例，直观地理解上述概念。

```python
# 定义一个简单的安小妖类
class AnXiaoYao:
    def __init__(self):
        self.name = “安小妖”  # 属性：名称
        self.personality = [“友好”, “好奇”]  # 属性：性格标签
        self.knowledge = {  # 属性：简易知识库
            “问候”: “你好！我是安小妖。”,
            “天气”: “今天天气不错哦。”
        }

    # 行为函数：回应
    def respond_to(self, user_input):
        if “你好” in user_input:
            return self.knowledge[“问候”]
        elif “天气” in user_input:
            return self.knowledge[“天气”]
        else:
            return “我不太明白你的意思。”

# 创建实例并交互
bot = AnXiaoYao()
print(bot.respond_to(“你好”))
```
