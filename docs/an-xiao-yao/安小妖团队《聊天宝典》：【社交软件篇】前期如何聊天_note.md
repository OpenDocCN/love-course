# 安小妖团队《聊天宝典》：【社交软件篇】前期如何聊天

大家好，我是唐唐。在社交软件上快速吸引女生，聊天需具备融合感、联系感、舒适感和安全感。

## 融合感

融合感是指让对方感觉像老朋友一样。例如，看到句子可以随意评论，如“嗨，懒猪，起床了，太阳晒屁股了”。这样可以让对方顺着你的思路聊下去，产生融洽感。

## 联系感

联系感是指让对方记住你。例如，给对方起一个有趣的外号，寻找共同点，共同记忆等。

## 舒适感

舒适感是指去除高价值照片，分享接地气的东西，如小时候的故事。舒适感可以平衡吸引，让女生愿意与你深入交流。

## 安全感

安全感是指展示你的稳定工作、爱好等，让对方知道你是一个可靠的人。

## 如何聊天

1. **时据防线**：与女生聊天时，必须经过时据话才能达到真正聊天的状态。
2. **索取价值**：从对方那里获取信息，开启话题。
3. **三维空间聊天**：用幽默的方式询问对方的状态，如“你现在是不是正在抠着脚丫，吃着薯片看电影”。
4. **复格**：赞美对方，告诉她你喜欢她。
