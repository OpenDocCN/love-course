# 1-260节聊天课：183.对你没好感的女孩该怎么追

## 概述
在本节课中，我们将学习如何通过调情来吸引对你没有好感的女孩。我们将探讨如何通过制造冲突和互动来增强彼此之间的亲密感。

## 调情与冲突
**调情**是指在两个人互相感兴趣之前，通过语言上的互动来关注对方。**冲突**则是在特定情境下对某人的形象、品质或能力进行轻微的调侃。

### 制造冲突
以下是一些制造冲突的方法：

- **轻微的调侃**：例如，“你个笨蛋，还是我来吧。”
- **单方面的冲突**：例如，你调侃女孩，但女孩没有回应。
- **你来我往的冲突**：例如，你调侃女孩，女孩也调侃你。

### 接住女孩的话
为了扩大暧昧感，你需要学会接住女孩的话，并借助她主动制造的小冲突。

### 调情的方式
- **高度关注**：在语言上对对方进行高度关注。
- **调侃和否定**：通过调侃和否定来表达关注。
- **SM**：调情可以看作是一种语言上的SM。

### 判断调情状态
- **只聊你和我**：如果两个人只谈论彼此，那么他们很可能处于调情状态。
- **调侃彼此**：通过调侃彼此来增强亲密感。

## 例子
以下是一个制造冲突的例子：

- **情境**：你和女孩在吃东西时面对面。
- **女孩**：你老看我干嘛？
- **你**：那我不看你我看谁啊，你是今天晚上这家餐厅里面最好看的女孩，当然要看你。

## 总结
本节课中，我们学习了如何通过调情来吸引对你没有好感的女孩。我们探讨了制造冲突和互动的重要性，以及如何通过语言上的关注和调侃来增强彼此之间的亲密感。希望这些技巧能帮助你更好地与心仪的女孩交流。
