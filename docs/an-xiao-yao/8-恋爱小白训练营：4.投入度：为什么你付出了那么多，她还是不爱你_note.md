# 8-恋爱小白训练营：4.投入度：为什么你付出了那么多，她还是不爱你

## 概述
在本节课中，我们将探讨恋爱中的投入度问题，分析为什么即使你付出了很多，对方可能仍然不爱你。

## 核心概念
**公式**： 投入度 = 情感投入 + 时间投入 + 金钱投入 + 精力投入

## 投入度失衡
以下是关于投入度失衡的几个关键点：

- **单方面投入**：许多人用尽全力对伴侣好，却得不到相应的回馈，导致关系失衡。
- **恶性循环**：单方面的付出可能导致恶性循环，让人陷入痛苦中无法自拔。
- **失衡关系**：在一段健康的关系中，双方的感情地位应该是平等的，建立在动态均衡的基础之上。

## 投入的三种类型
以下是衡量女生为你投入的三种类型：

1. **精力投资**：包括时间、心思和注意力等精神层面的投入。
2. **身体投资**：包括接吻、牵手和亲密关系等身体接触。
3. **金钱投资**：包括买礼物、请吃饭等金钱方面的投入。

## 维持平衡
以下是如何维持投入平衡的建议：

- **天平理论**：在关系中，双方应该像天平一样，保持平衡。
- **引导投入**：通过自己的投入，引导对方也投入。
- **心理警戒线**：设置心理警戒线，避免过度投入。

## 总结
本节课中我们一起学习了投入度的重要性，以及如何维持投入平衡。只有双方都愿意投入，才能建立一段健康、稳定的关系。
