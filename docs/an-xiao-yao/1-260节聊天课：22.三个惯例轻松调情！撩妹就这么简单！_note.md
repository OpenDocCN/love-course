# 1-260节聊天课：22.三个惯例轻松调情！撩妹就这么简单！

## 概述
在本节课中，我们将学习如何通过三个简单的聊天惯例来轻松调情，营造暧昧氛围。

## 惯例一：选择我
**公式**：如果全世界只剩下两个男人，一个丑得像猪，另一个是我，你会选择谁？

1. 向女生提出这个问题。
2. 如果女生选择丑男，你可以调侃她：“哈哈，那太棒了，你嫁给丑男，其他女人就只能选择我了。”
3. 如果女生选择你，你可以反转：“我还真不一定能娶你，但我没说全世界就剩你一个女人。”

## 惯例二：养猪的时光
**公式**：你总能让我回忆起那段养猪的时光，尤其是你撅着嘴的时候，特别像可爱的麦兜。

1. 跟女生说：“有时候你让我觉得特别亲切。”
2. 如果女生逗你说：“你不会是喜欢上我了吧？”
3. 你可以说：“不是啦，是你总能让我回忆起那段养猪的时光，尤其是你撅着嘴的时候，特别像很可爱的麦兜。”
4. 女生会假装生气：“切，你才是猪呢。”
5. 你可以继续调侃她：“嗯，看在你这只麦兜这么可爱的份上，就允许你吻我一下吧。”

## 惯例三：不可思议的事情
**公式**：你都快跟我聊了一个小时了，居然还没有向我表白，真的很不可思议。

1. 跟女生说：“我发现了一件非常不可思议的事情。”
2. 女生会问：“什么事情呀？”
3. 你说：“你都快跟我聊了一个小时了，居然还没有向我表白，真的很不可思议。”
4. 女生会抗拒：“欠，或者说你真讨厌。”
5. 你继续说：“哦，这算是你的表白方式吗？那还真的蛮特别的，既然这样的话，我就代表全世界接受你了。”

## 总结
本节课中我们一起学习了三个简单的聊天惯例，通过这些惯例，我们可以轻松调情，营造暧昧氛围。希望这些方法能帮助你更好地与女生交流。
