# 1-260节聊天课：190.如何跟女孩子实现情感上的共鸣

大家好，我是夏洛。今天我们来探讨一种常见的聊天错误——随意评价。这种错误常导致女孩感到烦恼，认为你笨拙、不会说话、情商低。

### 什么是随意评价？

随意评价是对别人的性格、状态、情绪、能力等进行毫无根据的、主观性推断、推测和评论。例如，你问女孩“你在干嘛呢”，她回答“在外面吃东西”，而你却评论“吃货的日子总是这么幸福”，这就是随意评价。

### 随意评价的危害

随意评价会让对方感到不舒服，认为你自以为是、自作多情。例如，你问女孩“你有没有出过轨”，她会因为你的提问方式而不愿回答真实情况。

### 如何避免随意评价？

1. **客观描述事实**：先描述一个客观事实或现象，再询问对方的看法。例如，你可以说“现在很多甜品品牌口味越来越好”，然后询问女孩的看法。
2. **避免主观评价**：不要用主观判断忽视对方的客观条件。例如，不要用“吃货”等概念化词汇给对方扣帽子。
3. **尊重对方**：在聊天中尊重对方的感受，避免过于主观的评价。

### 总结

本节课我们学习了如何避免随意评价，希望你在以后与女孩聊天时能够运用这些技巧，实现情感上的共鸣。
