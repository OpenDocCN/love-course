# 【安小妖】恋爱达人速成班：魅力工程-第8课：建设正确的聊天心态！

在本节课中，我们将学习如何建立正确的聊天心态，以提升与异性的交流效果。

## 1. 聊天心态的重要性

如果你总是等待对方回复，只会问一些无聊的问题，担心没有话题，害怕犯错，试图讨好对方，那么问题不在于你的聊天技巧，而在于你的心态。

**公式**： 正确的聊天心态 = 自信 + 接受 + 主动

## 2. 选择权与心态的关系

想象一下，在舞会上，如果你只有一个舞伴，当别人邀请她离开时，你会感到失落和害怕。但如果你有多个舞伴，即使一个离开，你也不会慌张。这是因为你有选择权。

**代码**： 选择权 = 自信 + 主动

## 3. 建立独特唯一性

对方希望成为你的独特唯一，这意味着你在众多人中选择了她，而不是只有她一个选择。这其中的差别很大。

**公式**： 独特唯一性 = 选择 + 价值

## 4. 控制欲望，保持心态

当你能控制自己的欲望，面对异性时，你会拥有一个良好的心态。你不需要讨好对方，因为你足够优秀。

**代码**： 控制欲望 = 自信 + 自我价值

## 5. 紧张与社交恐惧症

当你急于索取对方的价值时，你会变得紧张，担心说错话。这可能导致社交恐惧症。

**公式**： 紧张 = 紧迫感 + 担心

## 6. 患得患失

患得患失是指自己限制了自己。例如，当你搭讪女生被拒绝时，你会认为自己的行为是错误的。

**代码**： 患得患失 = 自我限制

## 7. 关系与聊天内容

与女人聊天时，重要的是你的身份，而不是你说了什么。了解你们的关系，可以避免犯错。

**公式**： 关系 = 聊天内容

## 8. 男性在两性关系中的角色

男性应该掌握主动权，而不是被女性左右。这样你才能拥有更多的话语权。

**代码**： 主动权 = 话语权
