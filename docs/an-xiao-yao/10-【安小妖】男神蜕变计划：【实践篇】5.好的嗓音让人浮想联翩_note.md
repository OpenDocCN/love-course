# 课程10：【实践篇】5. 好的嗓音让人浮想联翩 🎤

在本节课中，我们将要学习如何通过声音提升个人魅力。好的嗓音能让人感觉沉稳自信，使沟通更舒适有效。我们将从基础表达、声音控制到高级幽默感进行系统练习。

---

## 概述

声音是沟通的重要工具。说话过快、含糊或缺乏感情，都会影响信息传递和他人感受。本节课程将指导你如何锻炼出清晰、舒适且富有吸引力的嗓音。

---

## 初级阶段练习

上一节我们概述了声音的重要性，本节中我们来看看如何打好声音基础。以下是六个初级练习要点，帮助你建立良好的说话习惯。

1.  **将意思表达清楚**
    这是基础。如果你不能准确地向目标传达你的意思，那么再多的技巧都是没有用的。对方根本不知道你想要说什么。

2.  **营造舒服的感觉**
    舒服是一种什么感觉呢？声音太小会让人感觉到不舒服。说话太快会不舒服。声音太大也会不舒服。

3.  **有感情**
    能够准确地表达自己的意思之后，可以学着讲故事。先把故事讲清楚，然后再适当地加入感情。如果一个故事你觉得很有趣，先讲给自己听，然后润色完善再带着感情去讲给别人，这样就有进步了。如果你自己都觉得没有意思，不可能让别人觉得有趣。

4.  **避免不良习惯**
    多余的废话、口头禅、脏话最好不要。常见的废话有“然后”、“那什么”、“那啥”、“嗯”、“这个”。废话太多或停顿太久，长时间下来会发出不好的声音，让人反感。当然，和对方关系好到了一定程度时，可以偶尔说一些脏话，但要注意分寸，不能人身攻击。

5.  **说话时要饱满圆润**
    尽量让每一个字、每一个词发音饱满圆润。不要把每个字的音节吞掉，一定要让对方听得清楚。

6.  **注意说话时的韵律**
    当你学会上面的内容时，就要开始关注说话时的韵律。换我们中国古人的方法讲，叫“一扬正措”。一句话中，一部分说得轻快且上扬，一部分缓慢且柔和，一部分气势低沉，一部分声高气扬。如果你现在不知道跟别人聊天时聊什么，可以先试着调整好聊天的声音或语气。就像歌手练嗓子一样，当你能够控制自己的语气语调时，自己聊天搭讪的勇气也会变得充足。

## 磁性嗓音的操作方法

掌握了基础之后，我们可以进一步练习，让声音更具磁性和穿透力。以下是两个核心的气息控制练习方法。

**一、深吸慢呼气息控制延长练习**
其要领是要先学会“蓄气”。先压一下气，把废气排出，然后用鼻和舌尖间隙像闻花香一样，自然松畅地轻轻吸，吸得要饱满。然后气沉丹田，慢慢地放松胸肋，使气如细水长流般慢慢呼出。注意呼气一定要均匀，控制的时间越长越好。

**二、深吸慢呼数字练习**
我们把第一个步骤称为“吸提推送”。“吸提”是气息向里向上，“推送”是气息向下向外。在推送的同时做气息延长练习。这里推荐一个方法：**数枣练习**。要数到气尽为止，看你一次性能数多少个“枣”。

```
示例：出东门，过大桥，大桥底下一树枣，拿着竿子去打枣……
```

## 幽默感训练

声音的质感很重要，但谈话的内容与表达方式更能吸引人。幽默感是关键。以下是培养幽默感的五个操作方法。

1.  **每天找一条笑话，用不同的语气把它读出来**，尽可能讲得幽默风趣。

2.  **每天务必在工作生活中，找到一些有趣的事情幽默一下**，最好就是当众幽默。实在不行，可以通过自嘲的方法来补救。坚持一个月，你会发现自己的心境完全不同了。

3.  **自我暗示**
    如果你想改变自己的讲话，提升自信心，一定要学会自我暗示。想象自己是成功的，乐观地面对生活，放松情绪。这个心态在人际交往中非常重要。

4.  **形体练习**
    正常人讲话时都会做出相应的肢体动作。每天可以坚持练习五分钟，在镜子面前向自己微笑，展示自己的手势形态，培养微笑的习惯。要笑得灿烂、真诚和有亲和力。

5.  **话题信息的收集**
    平时看电视、看报纸、看杂志、看书、交谈和观察时，可以有意识地扩展话题源泉。这样在与人聊天时，就有了源源不断的话题。

## 呼吸与姿势的要点

好口才与坏口才的区别，其中一个因素是呼吸和姿势。讲话时深呼吸让肺部充满空气，声音会更有力量。

**检查呼吸方法**：深呼吸时，如果你的胸腔张开，表示呼吸太浅。正确的方法是把手放在胃部，每次吸气时腹部会隆起，这就叫**腹式呼吸**。

**姿势要点**：不良姿势会限制你的横隔膜与呼吸，大量消减你讲话的力量。讲话时要确保上半身保持正直。可以想象有一条线从你的脊椎一直延伸到你的头顶，把它拉紧，但别太紧张，身体要保持舒适和放松。

## 声音投射练习

面对声音太尖或是太软的问题，可以通过以下练习改善。

1.  找一个开放的空间，在室内或室外，与一位朋友相隔约三大步距离。
2.  用腹式呼吸，憋住气然后慢慢吐气，重复两遍。
3.  再吸一次气，吐气时用平常说话的音量（让对方听见即可），问朋友听起来如何。
4.  回到原位，继续说相同的话。这次不要面向朋友，而是瞄准他上方2-3公分的地方发声。**想象你的声音像橄榄球一样，经过抛物线射门得分**。
5.  检查结果并改进。然后多跨三步，重复相同的话，试着稍微提高音量，但不要让喊叫改变你的音调。
6.  再跨大三步，记得用“高抛物线法”发声。问你的朋友评估你的声音投射方式。
7.  看你能站多远，且不用喊叫就能让对方听见你的声音。一直练习到你可以轻松放大音量说话，而不改变音调为止。

在这个过程中，你会发现你的声音越来越清晰、有力。

## 总结

本节课中，我们一起学习了塑造良好嗓音的全过程。我们从**清晰表达、营造舒适感、注入感情**的基础练习开始，然后学习了通过**气息控制（深吸慢呼）** 来增加声音的磁性，并探讨了通过**幽默感训练、自我暗示和形体练习**来提升整体表达魅力。最后，我们明确了**正确的腹式呼吸与身体姿势**的重要性，并通过**声音投射练习**来实际改善音质与穿透力。

希望你能坚持练习，培养出一副清晰、自信且富有吸引力的好嗓音。
