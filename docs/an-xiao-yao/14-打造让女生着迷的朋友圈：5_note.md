# 14-打造让女生着迷的朋友圈：5

Hello 大家好，我是大力。这节课主要讲解如何打造一个好的展示面，让你的朋友圈更具魅力，吸引异性。

### 1. 头像的重要性

**公式**： 头像质量 ≈ 70% 的吸引力

- 尽量让头像好看、引人注意。
- 颜值不够，可通过发型、拍摄角度提升。
- 头像背景要干净整洁，避免杂乱。
- 头像照片以头像或半身照为主，避免全身照。
- 拍摄角度：侧面或45度角，避免正面。

### 2. 朋友圈核心：价值

**公式**： 朋友圈价值 = 高生活品质 + 时尚感

- 自拍：展现型男、时尚、会打扮的一面。
- 服装搭配、场景选择、后期修图都很重要。
- 发型：找专业理发店设计适合你的发型。

### 3. 高品质朋友圈内容

**列表**：

- 生活类：展示丰富多彩的业余生活，小资情调。
- 美食类：展示高生活品质，美食照片要拍出美感。
- 工作类：展示认真做事的魅力，避免刻意摆拍。
- 娱乐活动：展示热爱生活、身体健康的一面。
- 与女生合照：展示女人缘，避免暧昧。
- 社交类：展示地位、受人尊重的一面。

### 4. 修图与配文

- 使用修图软件优化照片，如美图秀秀。
- 配文要避开照片中想要表达的高价值，避免装逼。
- 避免过度装，以适度为原则。

### 5. 避免误区

- 避免八卦、乱七八糟的链接。
- 避免过度装，以适度为原则。
- 避免伪文艺青年气息的诗句、歌词。
