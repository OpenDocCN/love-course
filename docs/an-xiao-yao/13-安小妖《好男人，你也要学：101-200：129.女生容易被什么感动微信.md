# 课程13-安小妖《好男人，你也要学会撩妹》：129. 女生容易被什么感动？ 💖

在本节课中，我们将要学习如何理解“感动”在情感互动中的真正含义，并探讨如何通过真诚而非功利的方式去触动女生的内心。

---

很多时候，你认为自己做的事情或说的话足以让女生感动，但事实上并未达到预期效果。进一步说，即使成功感动了一个人，也并不意味着对方会因此喜欢你。

感动之所以复杂，是因为它具有极大的个体差异性。就像男生无法理解痛经的感觉，女生也无法理解蛋疼的感觉一样。

每个人的痛点不同，感动点也不同。只有深入了解一个女生，才更容易找到能感动她的那个点。

现在很多男生抱怨社会浮躁、女生势利，认为女生越来越难以被感动。说这种话的男生，其实并不真正了解女生，至少了解得不够深入。

他们没有将女生视为独立的个体，而是更多地当作猎物。他们更关心的是使用何种手段去俘获对方，以及征服后如何维持关系。

这种做法恰恰忽略了最重要的一点：女生和他们一样，拥有人类共通的情感需求。

如果女生喜欢喝可乐，你买十箱放在家里，此时再给她一瓶，她不会感动。如果女生非常喜欢吃苹果，你却给她一个橘子，她也不会感动，因为再好的橘子也吃不出苹果的味道。

如果女生喜欢陈奕迅，你费尽心血弄来两张他的演唱会门票，她可能会感动。如果女生做PPT遇到困难，你熬夜帮她解决问题并做出出色的PPT，她也可能会感动。

因此，你要给予女生的，必须是**她想要的东西**，或者是**她意想不到的东西**。同时，正如开头提到的，我们还需注意：感动仅仅是感动吗？它是否意味着被感动的人就会喜欢你呢？

很多时候，同一件事情，如果你做了，女生毫无反应；但别人做了，女生却可能感到喜欢和感动。这时你需要明白，问题有时并非出在你身上，而是出在女生身上。

相比于男生，女生更容易进行“逆向合理化”。如果她喜欢一个男生，那么这个男生做的一切都是对的；即使明显做错了，她也能找出无数理由欺骗自己。

因此你会发现一个现象：例如，一个经常打女生的男生，如果连续几天没有打她，女生可能会觉得美滋滋，甚至对姐妹说：“他好像变好了，前几天我生病还给我做饭，我特别感动。”

反之，如果换成一个她不喜欢的人，即便你跑前跑后地伺候她，她也可能毫无反应。

有些人见识了某些现象，便说女生都是势利的、见钱眼开，认为能否征服女生无非是钱多钱少的问题。这种说法有失公允。

**人都是缺什么才会想要什么**。对于缺钙的人，你给他补心，他当然不稀罕。对于缺钱的女生，你给她很多爱，她也不会稀罕，因为她想要的是钱。同样，对于缺爱而不缺钱的女生，你给她很多钱，她也不稀罕。

这两种情况并无高下之分，只因她们本就不同。

所以，如果你没有钱，就不要去找缺钱的女生。你口袋空空，怎能怪小卖部的大妈势利眼，不舍得给你一瓶农夫山泉？如果你既没钱，也没颜值和才华，还不努力奋斗，只知道抱怨，那么你注定会被淘汰。

回想一下，在你生命之初，你已经淘汰了亿万兄弟才来到这个世界。你已经走到了前面，但若想走得更远，就需要将抱怨的时间用于学习和思考。这样才能继续走在前面，成为人生赢家。

让女生感动的方法成千上万，事实上更是不计其数。只要你用心挖掘，总会找到。

很多人从小就懂得付出，也会关心、体谅他人，事事为他人着想。出去旅游会给喜欢的人带礼物，也常把他人挂在心上，做力所能及的事。人心都是肉长的，没有那么多真正无情无义的人。

当我们为一个人付出、挂念，哪怕只做一点点事，对方都可能很感动，比如收到一份生日礼物。因此，如果一个人仅仅想感动别人，那确实很简单。

但很多人在这背后，藏着其他想法和目的。他们极度渴望一个结果，等不起，也不想等。他们无比渴望找到一种方法，能让自己快点成功，至少看到一点希望。

所以，这样的人希望知道“女生最容易被什么感动”，这成了他们的救命稻草、最后的希望与坚持。

此时我必须指出：你想要去感动这个人，这是刻意的行为，带有阴谋和不可告人的目的，甚至可以说是一种欺骗。这种功利性容易让人走上歧途。

或许有人会说：“我就是单纯想感动她，没有其他想法。”在我看来，这是一句天大的笑话。人做事情总是有目的的。如果你说“我想锻炼感动女人的能力，所以要去感动她”，这倒可以。但真有几个人是这么想的呢？

真正能感动别人的，是你内在那颗**真正爱她的心**。我相信很多人看到这句话会想：“爱她又怎么样？她又不会和我在一起，这不是扯淡吗？”

这恰恰说明你是无知的，说明你不懂得爱护别人，说明你内在修养不够，还需要修炼。我见过很多人刻意做了很多事，女生也确实被感动了。

但就因为女生没有选择和他在一起，这个人最后就像脱缰的疯狗一样，胡乱撕扯，说什么绿茶、备胎、欺骗。这只能说明他本身就不是那样的人，所做的事也并非发自内心。

如果他本身就是一个有爱的人、浪漫的人、懂得关心体贴的人、有着高尚道德情操的人，那么他不会像疯狗一样。

所以，何必总想着要去感动别人呢？**做好自己，其实最能感动别人**。不刻意、不做作、无为而为、真真实实，这就是生活。因为生活中并没有那么多惊喜和设计，正因如此，感动才显得格外简单。

不经意间流露出来的、不带目的性、不带功利心的善意，才是最能打动女生的东西。

如果说女生真的容易被什么感动，那其实是她们需要的**安全感**——能够保护她，即使她是错的，你依然站在她的一方。如果你真的这样做了，她会在内心感谢你很久很久。

---

本节课中，我们一起学习了“感动”的本质。关键在于**深入了解对方的需求**，并**以真诚而非功利的心态去付出**。感动源于不经意间的善意与安全感，而非刻意的设计与索取。做好自己，真诚相待，才是情感互动中最动人的部分。
