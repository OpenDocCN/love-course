在聊天时，模仿对方的打字语言习惯，让女生感觉你们是一类人。例如，模仿女生喜欢在句子末尾加波浪号或使用特定语气词。

在邀请女生约会时，不要直接问“我们明天去吃饭怎么样？”而是提供两个选择，如“我们去吃披萨还是牛排？”这样女生就不太可能拒绝。

### 线上线下聊天技巧
