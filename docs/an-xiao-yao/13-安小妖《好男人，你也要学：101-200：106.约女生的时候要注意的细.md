## 邀约的提前规划与灵活应对

上一节我们介绍了沟通的基本心态，本节中我们来看看具体邀约时的安排。提前规划邀约至关重要。临时邀约可能导致对方因计划冲突而无法赴约。

即使邀约被拒，也不应就此终止对话。可将此次邀约尝试视为一次普通的聊天契机，自然地转向其他话题。

```

```
或者：
```
我：“真羡慕你睡眠质量这么好，我总是失眠。”
```
通过分享自身状态，可以延续对话。

**初期吸引力 ≈ 后续沟通顺畅度**
