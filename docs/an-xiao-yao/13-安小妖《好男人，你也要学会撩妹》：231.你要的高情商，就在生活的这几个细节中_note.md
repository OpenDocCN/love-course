# 13-安小妖《好男人，你也要学会撩妹》：231.你要的高情商，就在生活的这几个细节中

## 概述
在本节课中，我们将学习如何通过生活中的细节提升情商，以维护长期关系。

## 核心概念
**公式**：高情商 = 找到问题根源 + 少说多做 + 解决冲突

## 教程内容

### 1. 维持关系中的注意事项
- 在关系的初期，不要过早地完全依赖对方，以免失去吸引力。

### 2. 应对吵架
- 吵架会侵蚀感情，宣泄情绪只是表面，深层积压更多情绪。
- 不要在吵架时说出伤害对方的话，避免给对方压力。
- 吵架时，一方先停止宣泄，另一方情绪也会缓和。
- 用轻松的方式岔开话题，避免情绪升级。

### 3. 情伤处理
- 当对方宣泄情绪时，找到根源，满足其核心需求。
- 高情商的处理方式是解决问题，而非直接说服。

### 4. 不要过度干涉对方生活
- 尊重对方的兴趣和热情，即使你不理解。
- 接受对方的不完美，不要强加个人标准。

### 5. 站在对方角度考虑问题
- 不要将自己的意识强加给对方，克服惰性，内化尊重和礼貌的习惯。

### 6. 真正的生活智慧
- 保持无条件尊重，无论关系如何，都要尊重对方。
- 真正的生活智慧是意识到并做到这一点。

## 总结
本节课中我们一起学习了如何通过生活中的细节提升情商，以维护长期关系。希望这些方法能帮助你更好地理解和处理人际关系。
