# 10-【安小妖】男神蜕变计划：【实践篇】2.好的发型让你成功一半_note

在本节课中，我们将要学习男生如何选择并打理适合自己的发型。发型的选择不应盲目模仿，而应基于对自身条件的了解。掌握正确的方法，找到属于你的发型并不困难。

## 了解自己的脸型

上一节我们介绍了课程主题，本节中我们来看看如何确定自己的脸型。首先，你需要了解自己属于哪种脸型，可以通过以下方法进行测量。

以下是测量脸型的三个步骤：
1.  将前额刘海全部撩至脑后，完全露出前额发际线。如果是长发，请同时露出双耳。
2.  正面面对镜子，使用皮尺测量四个数据：脸部的长度、额头的最大宽度、颧骨的最大宽度以及下巴的宽度。
3.  将测量得到的数据与标准脸型数据图进行比对，从而判断自己所属的脸型类别。

## 了解自己的发质

确定脸型后，下一步是了解自己的发质。不同的发质适合不同的发型和打理方式。

以下是几种常见的发质类型及其特点：
*   **油性发质**：建议选择短发，便于日常清洁和打理。
*   **细软发质**：适合通过烫发或留长发来增加视觉上的发量和蓬松感。若选择烫发，需注意加强护理以修复损伤。
*   **粗硬发质**：发质坚硬，不易塑形或烫卷，但一旦定型则持久。建议尝试大卷烫发或较短的发型，日常需勤于打理。
*   **自然卷发质**：头发容易显得杂乱，可通过烫发来达到理想的定型效果。

## 男生发型风格分类

在了解自身条件的基础上，我们可以来看看男生发型的主要分类。通常，我们可以按长度将发型分为短、中、长三种。

以下是按长度划分的发型示例：
*   **短发**：包括 Undercut、莫西干、寸头等。
*   **中发**：包括齐刘海、三七分、复古油头等。
*   **长发**：包括斜刘海、中分等。

## 选择喜欢的风格

除了长度，风格也是选择发型时的重要考量。根据不同的气质和喜好，男生风格大致可分为几种类型。

以下是五种主要的男生风格类型：
1.  **阳光型风格**
2.  **儒雅型风格**
3.  **时尚型风格**
4.  **复古型风格**
5.  **硬朗型风格**

你可以根据自己的喜好尝试不同的风格。

## 如何选择与沟通理发师

选定了心仪的发型风格后，找到一位合适的理发师并与之有效沟通至关重要。选择理发师主要有两种途径。

以下是两种选择理发师的方法：
1.  **询问朋友**：向你欣赏其发型的朋友直接询问理发师信息。
2.  **自行寻找**：建议选择有完善培训和晋升机制的连锁理发店，通常理发师水平更有保障。进店后，可根据价位（通常选择中间价位）进行选择。

选定理发师后，清晰的沟通能确保最终效果。沟通时需注意以下几点。

以下是与理发师沟通的关键点：
*   展示多角度的目标发型图片。
*   详细说明自身头型和发质情况，以便理发师调整。例如：“我的中庭较短，希望发型能视觉上拉长中庭。”或“我不擅长使用造型工具，希望吹干后就有型。”
*   建议定期（如一年）更换理发师，以避免发型陷入固定模式，缺乏新意。

## 日常发型打理技巧

从理发店出来的完美造型难以维持？本节我们将学习日常打理技巧。打理发型通常只需用到吹风机、造型品（发蜡/发泥）、梳子和定型喷雾。

以下是两款常见发型的打理方法：
*   **背头（如Undercut）**
    *   **条件**：适合发质柔软者，对脸型要求不苛刻，但脸型不宜过长。
    *   **打理步骤**：
        1.  吹头发时，用手或大梳子辅助将头发整体向后吹。
        2.  吹至半干时，取适量发蜡在掌心揉开，然后将头发向后抓梳定型。
        3.  最后取少量发蜡整理两侧发际线边缘。
*   **发饰短发**
    *   **特点**：适合多数脸型，尤其适合发量不多的男生，易于打理。
    *   **打理技巧**：此发型几乎无需使用强力定型产品，自然成型即可。平均每2-3周修剪一次以保持最佳状态。
*   **中分发型**
    *   **打理步骤**：
        1.  先将头发完全吹干，防止凌乱。
        2.  使用卷发棒分区将头发烫出卷度。
        3.  用手指调整出自然的纹理。
        4.  最后使用定型喷雾固定造型，延长保持时间。
