## 低情商的十个表现

以下是低情商的十个表现：

1. **不懂得基本的社交礼仪**：总是喜欢麻烦别人，如群发求赞、群发测试清理好友等。
2. **言语上胜过别人**：在表达自己的观点之前先把别人贬低一番，以此来抬高自己。
3. **面子看得比天还大**：不懂得低头，自尊心过高，害怕失败，不敢迈出第一步，也不开口求助。
4. **不能平衡自己的状态**：情绪大起大落，要么自我感觉良好，要么自卑得不行。
5. **不懂得换位思考，自我反省**：错的永远是别人，听不懂别人的拒绝，也无法领会别人话你的意思。
6. **在公众场所对女人评头论足**：开有颜色的玩笑，或者是对女人动手动脚。
7. **道德绑架**：不尊重他人的意愿，强迫他人接受自己的观点。
8. **键盘侠**：用恶毒的语言攻击他人，如“好看的女人都是婊子”、“有钱的女人都是被包养的”等。
9. **不懂得自己的身份和地位**：总是在不恰当的场合表达不合适，或者是不应该由她来表达的观点和看法。
10. **不会平等爱人**：在爱情里只会付出，不会索取，要么辜负别人，要么被辜负。

## 为什么低情商的人，讲话做事容易伤及别人的感受？

1. **感知他人情感的能力差**：缺乏该说什么，该做什么，有什么影响的判断力。
2. **缺乏关注他人情感的意愿**：不在乎，但处理不好。
3. **容易感觉被伤害**：把别人无意识的行为解读为对自己的攻击。
4. **不愿意接受意图与结果的差距**：爱找借口，推卸责任，怪罪他人。
5. **缺乏正面影响他人情感的能力**：却常常无意识地给别人带来负面的情绪。
