# 13-安小妖《好男人，你也要学会撩妹》：151-200：179.聊妹外挂，让你妙语连珠大招放不停

欢迎来到本期《好男人，你也要学会撩妹》。我是猪酿人吃一点鲜生。在现实生活中，我们发现有些男生与女生聊天时能谈笑风生，而有些男生则在与心仪的女生聊天时感到无话可说。

**聊天能力的重要性**

- 与他人聊天时，无需承担责任，聊得好坏只是闲聊。
- 与心仪的女生聊天时，带有目的性，希望让她开心、觉得你有趣。

**聊天高手的三大能力**

1. **倾听**：学会倾听，就像训练场上的士兵先学会挨打一样。女生讨厌滔滔不绝、自我膨胀的男生。
2. **发问**：运用开放式提问引导对方多说话，了解她的想法和看法。
3. **讲故事**：适时发表独到见解，以讲故事的方式描述，激发对方思绪。

**如何克服聊天困难**

- **倾听**：闭上嘴巴，竖起耳朵，听清楚对方说了什么，背后的含义是什么。
- **发问**：运用开放式提问引导对方多说话，了解她的想法和看法。
- **讲故事**：适时发表独到见解，以讲故事的方式描述，激发对方思绪。

**聊天氛围的重要性**

- 聊天氛围比话题更重要，让女生感到舒适，没有压迫感。
- 不必局限于特定话题，让聊天自然流畅。

**总结**

本节课中，我们学习了如何克服聊天困难，成为一个话题制造机。希望这些技巧能帮助你在与心仪的女生聊天时更加自信、流畅。下期再见！
