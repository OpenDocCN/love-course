# 安小妖团队《聊天宝典》课程03：【技巧篇】一个惯例 ☕️

在本节课中，我们将要学习一个实用的聊天技巧——“咖啡惯例”。这个技巧旨在帮助你测试女生的反应，并自然地将聊天导向最终目的：邀约。

很多男生在成功开场、完成基本信息交换并开启聊天话题后，会感到迷茫，不知道接下来该聊什么，也不清楚应该将聊天的方向引向何处。

我们必须时刻记住，聊天的核心目的只有一个，那就是邀约。所有的对话都应以此为目标，层层递进，最终达成目的。为了实现这一点，我们首先需要测试女生的反应。接下来介绍的“咖啡惯例”就是一个有效的测试工具。

## 惯例准备与执行

上一节我们明确了聊天的核心导向，本节中我们来看看如何具体操作“咖啡惯例”。这个惯例需要你提前做一些简单的准备。

当你某次去喝咖啡时，记得为你的咖啡拍一张美观的照片并保存下来。这张照片在未来会起到重要作用。

以下是具体的执行步骤：
*   在工作日的上午，将这张咖啡照片发送给女生。
*   同时附上文字：“我知道你早上很累，送一杯咖啡给你。”

## 惯例的意图与效果分析

我们已经了解了“咖啡惯例”的具体做法，现在我们来深入分析其设计意图和可能带来的效果。

这里的关键并非真正送一杯咖啡，而是发送照片。这个行为传递了关心与趣味性。

大部分时候，女生收到这样的信息会认为你既贴心又有趣。你可以借此机会判断以下几个关键点：
*   女生是否愿意与你继续交流。
*   她对你的感觉如何。
*   她当前是否有空闲时间。

## 对比测试与直接邀约

通过“咖啡惯例”测试反应，其优势在于避免了唐突感。这与直接邀约的方式形成了鲜明对比。

如果你直接对女生说“美女，我们一起去吃饭吧”，她极大概率会拒绝你，因为这种方式非常突兀。

但是，如果你先以“咖啡惯例”这样的话题进行测试，就可以根据女生的反应来决定后续的聊天话题、方向，以及判断是否应该发起一个真实的邀约。

本节课中我们一起学习了“咖啡惯例”这个实用技巧。我们明确了它的准备方法、执行步骤，并分析了其通过测试女生反应来避免唐突、自然导向邀约的核心作用。记住，有效的聊天需要策略和测试，而不是盲目地直奔主题。
