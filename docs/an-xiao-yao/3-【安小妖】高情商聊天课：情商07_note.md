# 3-【安小妖】高情商聊天课：情商07

## 概述
在本节课中，我们将探讨如何通过幽默来应对敌意与攻击，并学习如何提升自己的情商。

## 幽默感是什么？
**幽默感**是一种提升他人与自己相处时愉悦程度的能力。它不仅仅是搞笑，而是一种智慧的表现。

## 黄渤的幽默智慧
黄渤以其幽默和智慧著称，以下是他的一些经典例子：

- **应对调侃**：在第50届台湾金马奖上，黄渤被调侃穿着睡衣出席，他机智地回应：“金马奖我老来，回家嘛，当然穿睡衣啦。”
- **巧妙夸人**：当被问及是否经常被错认成王宝强时，黄渤回答：“替王宝强感到无比的骄傲。”

## 提升情商的方法
以下是提升情商的一些方法：

### 巧妙地夸人
- **避免拍马屁**：高情商的人在不贬低自己的情况下，也能让别人开心。
- **例子**：当女生问“我这件衣服好看吗？”可以回答：“这件衣服都被你穿了，能不好看吗？”

### 机智地面对调侃
- **用幽默回应**：当面对调侃时，用轻松幽默的方式回应，可以缓和气氛。
- **例子**：当女生说“你好矮啊”，可以回答：“我也觉得诶，其实再矮一点就好了，这样我就能买到半价票了。”

## 高情商的幽默感
**高情商的幽默感**是为了化解尴尬，不让人尴尬。

## 提升讲话吸引力的方法
- **记录有趣的事情**：将生活中有趣的事情记录下来，在聚会或约会时分享。
- **加入肢体语言和表情**：在讲述时加入肢体语言和面部表情，使故事更加生动。

## 总结
本节课中我们一起学习了如何通过幽默提升情商，以及如何巧妙地夸人和面对调侃。希望这些方法能帮助你成为一个更受欢迎的人。
