# 5-【安小妖】极速吸引方程式：14.什么是舒适感_note

在本节课中，我们将要学习亲密关系发展中的一个核心概念——**舒适感**。我们将探讨为何单纯地频繁聊天无法推进关系，并详细介绍如何在交往初期有效地建立舒适感，为关系的升级打下坚实基础。

大多数同学存在一个问题：遇到喜欢的女生后，恨不得24小时与她待在一起，而不考虑女生的心情或时间。结果到最后，女生会突然变得很冷漠。

很多男生不明白为何会这样，觉得明明聊得很好。但聊天质量不应以时间长短来衡量。聊天的目的是让关系变得暧昧，而非证明情投意合。如果你时时刻刻想着和她聊天，即使你笑点足、话题多，女生可能会想：你究竟有多无聊才会一直闲聊？你难道没有正经事做吗？一个不知上进的男人，如何让人放心交往？这样聊到最后，你会绞尽脑汁想话题，直到筋疲力尽。

退一步说，即使女生神经大条未察觉，或你们生活都枯燥，也需考虑女生拥有自己的个人时间。当然，也存在一种令人满意的情况：你喜欢的女生也喜欢和你聊天。此时更应提高警惕，学会欲擒故纵，控制住急切的心情，先吊足她的胃口。

举个例子：如果你和女生聊天到半夜，且聊得很嗨，双方都想继续，你会怎么办？你可能因感到女生对自己有兴趣而不想结束话题，选择继续聊，直到她困了说要去睡觉。首先，聊得太晚会导致精力不充沛，后半场聊天会因困倦而思维迟钝，内容越来越空洞。其次，你向女生暴露了离不开她的小心思。在关系未明确前，最好不要暴露这些心思。最后，一天就把想聊的都聊完了，之后靠什么维持吸引力？

正确的做法是：趁双方情绪都非常高涨时，主动终止对话。例如说：“今天和你聊天挺开心的，但我明天要早起上班，你早点睡吧。”这样不仅隐藏了自身需求，也让女生更加期待下一次聊天。

单纯的吸引不足以让故事发生。吴彦祖很吸引我，但我会死心塌地等他来谈恋爱吗？不会。要想有故事，需要进入下一个阶段：开始建立她对你的**舒适感**。

那什么是舒适感呢？舒适感不是简单的聊骚或套近乎，不是每天问“在干嘛呢？吃了吗？睡了吗？晚安”。这些有时不叫联系，对某些人而言是单方面的骚扰。

**舒适感**是指对方对你产生的信任与安全感，愿意与你亲密而不感到别扭。然而，这种信任不同于她对同学、同事或知己的信任，因为后者缺少特殊的亲密感。这种安全感也不同于她对家人的安全感，因为与家人的安全感难以脱离血缘关系。

那么，如何在交往初期就给女生建立舒适感呢？以下是几个方法。

## 1. 共谋

共谋就是与她一起做同一件事。当此事发生时，她会在潜意识里觉得你们像一对情侣。你与她在相同的物理空间做同一件事，称为**现场现实共谋**。其实，你与她约会做的每件事都算共谋，但应努力做一些特别的事。普通的吃饭聊天经历太多，难以调动双方情绪，容易无聊。因此，我们要做的事应能引发她特别的情绪，例如紧张、激动或纠结。

**例如**：
*   人行道绿灯最后十秒，带着她跑到街对面（**注意**：务必在无车情况下，有车切勿尝试）。
*   在餐厅吃完饭，偷偷结账后告诉她：“我们这次逃单。”然后不由分说拉着她跑出去，跑一段再说：“完蛋，餐厅好像有监控，明天我们要上新闻了。”

操作时需千万注意分寸。如果对方性格内向、害羞且敏感，你把她拉到舞台上跳舞献唱，她可能之后再也不理你。应根据不同的人做出不同的共谋。

## 2. 条件反射

初中生物课上，我们都学过一项著名的生物学和心理学实验。简单来说：给一条狗摇铃铛然后喂食，持续一段时间后，它只要听到铃声就会分泌唾液。这称为**巴普洛夫效应**或**条件反射**。铃声是**条件**，分泌唾液是条件触发的**反射**。

在这个实验中，铃铛、狗、分泌唾液与我们要讲的升级调情有何关系？很简单：**条件**（铃铛声）是我们要创造的，而**反射**（分泌唾液）就是我们与对方建立的舒适感，即暧昧氛围的营造。（此处仅为举例，并无任何不尊重女性之意。）

当你们一起玩，她情绪高涨或与你情意绵绵时，要让一个东西反复出现在她视线里。例如，你带她玩抓娃娃机，抓到后送给她。之后只要她看到这个娃娃，就会想起你。这个东西会成为她反射的**条件**。

再比如，送女生礼物时，与其花大价钱买爱马仕包包，不如送她每天乐于使用的东西，例如口红、香水，均价在500到1000元之间。

## 3. 分享秘密，展示脆弱

信任是组成舒适感的重要方面。想要获取别人的信任，光靠说很难做到。在日常生活中，我们想获取信任，必须表现好、口碑好，可能还需要主动向对方投资，且这也不能保证获得信任。有句话说：“无事献殷勤，非奸即盗。”所以还需要一定的时间跨度。这些都说明信任建立非常困难，因而显得珍贵。

在与女生相处时，有了一定时间积累，你还需要一些“投资”才能获得相当的信任。这项投资可以是告诉她一个你的秘密。注意，要装出煞有介事的样子，要求她一定不能说出去，这样你的投资才显得有价值。

例如，你可以对她说，感谢她这次约会的陪伴，为此透露一个秘密。内容可自行把控，不一定是惊天秘密（真秘密可能吓走对方），可以是一件较普通的事。但你可以告诉对方：“我不会和其他人分享这件事，因为其他人不理解，或会据此评判我。”你甚至可以要求她也分享一个秘密，这样更为公平亲密。你们都是听过对方秘密的人了，这是革命情谊。

展现你的脆弱，也是一个屡试不爽的招式，微信聊天或面对面沟通都适用。当然，这建立在彼此有好感的基础上。切记，必须是双方互有兴趣，否则效果只会适得其反。

在你和女生的聊天过程中，通过一个不经意的话题引导至你的脆弱面。例如，你可以说：“我有时候也会没有安全感，会想要逃离，有时候也希望能找个人靠一靠。”这会激起女性的母性关怀，女生也会觉得你们有相似感受，能够互相理解，关系自然更上一层楼。
