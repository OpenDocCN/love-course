# 1-260节聊天课：154.什么样的肢体语言会让你加分？

## 概述
在本节课中，我们将探讨如何通过使用多重对话脉络来增强聊天效果，从而在男女交往中加分。

## 对话脉络的定义
**对话脉络**是指聊天时话题的延续性和流畅性。

## 单一对话脉络与多重对话脉络
- **单一对话脉络**：话题单一，如询问工作情况。
- **多重对话脉络**：话题多样，如从衣服聊到朋友圈。

## 使用多重对话脉络的优势
- **制造熟悉感**：通过多样化的话题，让人感觉更熟悉。
- **建立信念**：随着时间推移，信任感逐渐建立。

## 如何使用多重对话脉络
- **主导互动**：主动引导话题，避免线性聊天。
- **切换话题**：根据情况灵活切换话题。

## 切断当前话题脉络
- **分享感受**：如果对方不感兴趣，及时切换话题。
- **对方讲话**：当对方讲话无话题时，及时切断话题，开启新话题。

## 总结
本节课中我们一起学习了如何使用多重对话脉络来增强聊天效果，从而在男女交往中加分。希望这些技巧能帮助你在聊天中更加自信和迷人。
