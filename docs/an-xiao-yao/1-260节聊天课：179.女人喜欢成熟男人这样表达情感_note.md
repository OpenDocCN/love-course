# 1-260节聊天课：179.女人喜欢成熟男人这样表达情感_note

在本节课中，我们将学习两个社交中普遍存在的现象，帮助你理解为何与女生聊天时常常失败、尴尬或缺乏良好氛围。我们将从一个常见误区开始，逐步分析如何通过调整表达方式来提升社交吸引力。

## 从“幽默”的误区开始

如果你问女生喜欢什么样的男生，“幽默”往往是条件之一。幽默确实是吸引女生的重要价值，但关键在于理解幽默的本质。

在社交动力学中，有一个概念叫做**外并**。它指的是营造和维护与他人交流时的社交气氛。我们可以通过一个场景来理解：在聚会中，你讲述自己的趣事，还没讲到最有趣的部分，大家就已经开始笑了。这种自发的笑声表明你们之间存在良好的社交外并。

相反，社交智商不高的人讲完笑话后，往往会自己先笑。你可以留意一下，当别人讲完笑话自己先笑时，你内心的感受如何？你原本可能想笑，但对方先笑反而让笑话显得不那么好笑。

这种现象的原因是：笑容本质上是为缓解紧张和社交压力。笑话的核心技巧是人为制造社交压力，例如相声中的“抖包袱”。讲完笑话后的短暂沉默（压力空档）有助于引发笑声。如果你自己先笑，就提前释放了压力，削弱了笑话的效果。

自己先笑往往是担心无人发笑会尴尬，因此用笑声填补沉默。这种行为反映出**缺乏自信**和**社交地位较低**。相比之下，公司老板讲笑话时，员工容易发笑（即使不好笑也会假笑），因为老板的社交地位高。

女生能通过这种细微举动判断男人的社交价值和地位。回想郭德纲的相声，最令人发笑的地方往往是他一本正经地“胡说八道”。因此，当你试图让女生觉得有趣时，应避免流露出“我要讲笑话了”或准备大笑的姿态。相反，保持平稳，留出空档，让女生自己体会笑点。

## 对话中的思维定势与情绪价值

接下来，我们通过对话比较来探讨另一个知识点：思维定势如何影响对话效果。

我们平时说话常受思维定势影响，导致用词模式化。例如，假设你有一件开心事要与朋友分享：

**场景一：**
- 你对哥们说：“你绝对猜不到我这个鼠标买得多合算。”
- 哥们说：“我猜五百块。”
- 你说：“我才花了二百块。”
- 哥们说：“那你可真是厉害了！”

这是一个良好的谈话状态。

**场景二：**
- 你对朋友说：“你绝对猜不到我这件衣服多合算。”
- 朋友说：“那你一定捡到便宜了，一百块？”
- 你感到尴尬，只能说：“其实我花了二百块。”
- 朋友只好说：“那也不错了。”

两种对话的效果和情绪差异显而易见。第二种回应不经意间透露出朋友内心的不安全感。他的潜意识思维是：“我要表现给你看，证明我的聪明。”表面上这种对话显得愚蠢，但根本原因是朋友争强好胜、缺乏安全感。

这种心理导致他错失了营造良好聊天氛围的机会。朋友间对话的目的是建立共同兴奋感，提升彼此情绪，分享快乐时刻。说完后两人情绪高涨，社交气氛很棒。但第二种情况破坏了气氛，因为朋友将你的话视为测试，导致交流变得无趣。

这就是为什么有人会说“和你讲话真没劲”——指因不良外并导致的糟糕聊天氛围。内心安全感强的人意识到你捡了便宜，仍会说出一个正常（不过低）的价格，只要高于实际价格，就能带给你良好情绪，营造“来劲”的感觉。

社交直觉正常的人都知道，你这么说一定是捡到便宜了，因此无需在朋友面前展示聪明。否则只会显得社交智商不高。如果你能给对方带来好情绪，对方就会喜欢与你相处。这就是**情绪价值**，它能吸引别人。

另一种情况也能带来情绪价值：如果朋友知道某个地方能买到更便宜的相同物品，并愿意带你去买，他可以说出更低的价格。因为他的行为提供了实际价值，这种价值带来好情绪，抵消扫兴感。最终你能退掉较贵物品，以更便宜价格获得所需，这同样能提升兴致，让双方觉得“玩得不错”。

无论哪种方式，关键是为对方提供**能带来价值的对话**。只有做到这一点，才能营造良好的聊天氛围。

## 总结与提升建议

本节课我们一起学习了两个社交细节：幽默的真正应用和对话中的情绪价值。这些习惯往往源于内心状态，如果不自知，可能导致说话出现问题。

今后你可以在某些情况下有意识地控制自己，调整心态，避免第一时间说出无法带来正面价值和情绪的话。这样才能避免氛围和关系变得尴尬冷淡。

希望你能逐步提升情商，越来越懂得如何与人相处，如何让他人感到快乐，喜欢与你待在一起。
