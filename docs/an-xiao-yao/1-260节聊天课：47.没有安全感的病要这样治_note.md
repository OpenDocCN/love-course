# 1-260节聊天课：47.没有安全感的病要这样治

大家好，我是夏洛，这里是撩妹聊天大师的第47节课。今天我们将探讨如何在感情中获得安全感。感情中缺乏安全感就像感冒一样，是一种常见的病，男女皆难以幸免。

**缺乏安全感的根本原因**

通常，我们提到安全感时会说：“谁能给我安全感？”这其实是你缺乏安全感的根本原因，因为你把安全感的获得寄托在了别人身上。安全感是自我给予的。

**如何给自己安全感**

1. **保持清醒的自我对话**：跳出自我，以第三方的视角与自己对话，避免过度纠结和负面情绪。
2. **处理与外界的关系**：亲情和友情是获取安全感的重要方式，需要处理好与亲人、朋友的关系。
3. **增大资源储备和开发能力**：财富、异性、努力程度、创意、外向型人格和社交倾向等都是资源，储备充足的资源可以维系吸引力，减少焦虑。
4. **构建选择伴侣的标准**：明确自己想要的，避免感情透支和焦虑。

**总结**

本节课中，我们学习了如何在感情中获得安全感。通过保持清醒的自我对话、处理与外界的关系、增大资源储备和开发能力以及构建选择伴侣的标准，我们可以更好地应对感情中的不安全感。希望对大家有所帮助。
