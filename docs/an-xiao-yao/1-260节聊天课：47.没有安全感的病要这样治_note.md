# 1-260节聊天课：47.没有安全感的病要这样治_note

在本节课中，我们将学习如何在感情中获得安全感。安全感缺失是感情中一种普遍存在的问题，如同感冒一样常见，无论男女都可能经历。我们将探讨其根源，并提供具体的方法来建立和巩固自身的安全感。

## 概述：安全感缺失的普遍现象

在感情中缺乏安全感是一种非常流行的常见问题。随着经历增多，你会发现许多关系难以界定，既非纯粹友谊，也非明确恋情。学生时代，对方主动找你聊天可能就被视为喜欢的信号。但成年后，即使发生了亲密关系，也不一定代表对方真正喜欢你。

从公众人物到身边朋友，充满激情却短暂的关系越来越多，长久专一的感情似乎越来越少。许多人因此被裹挟，习惯于一种只能同甘、不能共苦的交往模式，觉得关系结束也就结束了，懒得去挽留。

然而，内心深处，每个人都渴望一段踏实稳定的感情。我们脑海中都有一幅理想的画面：一个心灵相通的伴侣，一个眼神就能明白彼此，两人可以共同回忆数十年的点点滴滴。如果说成熟意味着在感情中小心翼翼保护自己，不再全心全意付出，那么这种成熟我们宁可不要。

## 缺乏安全感的根本原因

上一节我们描述了安全感缺失的普遍性，本节中我们来看看其根本原因。

通常，人们谈论安全感时的口吻是：“某某能不能给我安全感？”这恰恰是问题的根源——**你将安全感的获得寄托在了他人身上**。**安全感永远只能自己给自己**。如果你不能自己制造安全感，就会经常处于安全感匮乏的状态。

## 建立安全感的四个方法

理解了问题的根源后，以下是建立自身安全感的四个核心方法。

### 1. 与自己建立积极对话

首先，你要学会保持清醒，与自己进行积极对话。常言道“当局者迷，旁观者清”，这是一个哲学难题：人很难跳出自我，以第三方视角客观审视自己。这会导致许多纠结和自我评价的不公允。

你可能会总是关注那些过得比你好、身份比你高的人，并紧抓自己的小缺点不放。这会导致失望和自我否定，产生无尽焦虑，并加剧对安全感的渴望。

因此，你需要进行自我保护，让内心的对话变得积极起来，屏蔽那些导致安全感缺失的负面心声。作家渡边淳一提出的“钝感力”概念正是此意：**让自己不再对外在的财富、身份、伴侣条件等过于执着，对外界刺激的反应也不再过于敏感**。这样就能在很大程度上获得安全感。

### 2. 维护稳定的外部支持系统

如果说第一点是处理好与自己的关系，那么第二点就是要处理好你与外界的关系。

能够带来稳定关怀并分担风险的亲情与友情，是你获取安全感的重要途径。**友情和亲情，是与爱情进行风险对冲的工具**。你在爱情中遇到的问题，可以通过亲情和友情得到很大程度的缓解。

但这里存在一个问题：亲情和友情也可能出现困难与挫折。
*   **亲情方面**：随着年龄增长，家人对你的预期会变化。如果你未能满足他们对家庭贡献、关怀或责任的期待，亲情就可能产生嫌隙。
*   **友情方面**：如果你的思维、收入未能与朋友保持大致同步的增长，出现较大落差，你们之间交往的自由空间会变小，禁区会变大。

因此，你需要努力保持自己与亲人、朋友的预期一致，处理好这两类至关重要的人际关系。

### 3. 增强自身资源与能力

第三点，要增大自己的资源储备以及开发资源的能力。

*   **资源** 主要指两个方面：你的**财富**，以及你**结识的异性资源（社交圈）**。
*   **能力** 则指你的**努力程度**、**创造财富的创意**、**外向型人格**和**社交倾向**。

我们常以为安全感缺失是因为无法经营稳定关系，其实不然。真正让你缺乏安全感的，是想到未来可能失去青春、吸引力、财富，再也无法轻易获得异性青睐时产生的**焦虑**。

所以，你要做的是储备充足资源来维系自身吸引力。即：**努力赚钱，并建立更丰富多元的社交生活**。这样，你就不再感到焦虑，也无需迫切向外寻求安全感。

### 4. 明确择偶标准

第四点，你要构建自己选择伴侣的标准。

说白了，你心里要清楚自己想要什么样的人。否则，对每个遇到的人都全力以赴，会透支你的情感账户，导致焦虑和恐慌。选择不当的后果是举步维艰，即使咬牙坚持很久，最终也可能一场空。这种内心创伤正是造成安全感匮乏的重要原因。

因此，心里一定要有明确的谱。

## 总结

本节课中，我们一起学习了感情中安全感缺失的根源与解决方法。我们认识到，安全感源于自身，而非他人。通过**与自己建立积极对话**、**维护稳定的亲情与友情支持系统**、**增强个人资源与能力**以及**明确自身择偶标准**这四个步骤，你可以逐步建立起坚实的内在安全感，从而更从容地面对感情关系。
