# 11、【安小妖】脱单12技：第11课高分闺蜜策略

女生的价值与我们相当或更低，这种闺蜜容易相处。本策略强调的是高分闺蜜策略，即女生的价值社交圈高于或优于我们。

**如何让闺蜜为我们所用，与我们建立密切关系**？

首先，我们需要了解什么是闺蜜。以下是一个例子：

**闺蜜案例**：

一位形象良好、参加选美比赛、身材棒、皮肤好、会穿着的女生。她是我的众多闺蜜之一。我们是在今年四月份的深圳时装周认识的，当时我去参加一个女性朋友（前女友）的时装发布会。活动结束后，我在大厅看到她们，觉得她们长相不错，气质佳，于是主动搭讪，并邀请她们一起合影。

**本章要点**：

1. **闺蜜定义**：现代社会中，闺蜜通常指女性的密友。但本策略中的闺蜜是指男闺蜜，有利于我们与女生建立关系。
2. **正常闺蜜关系**：与女生成为朋友，但可能只是情感垃圾桶或玩伴。
3. **成为闺蜜的原因**：错过与心仪女生在一起的机会，成为兄弟或哥们。
4. **闺蜜的作用**：把妹助手、女性心理导师、社交圈和资源。

**高分闺蜜策略**：

1. **以闺蜜身份接近女生**：先与女生建立友谊，再逐步提升关系。
2. **成为男闺蜜的条件**：外在形象、共同兴趣、闪光点、条件匹配。
3. **提供价值**：情感价值、帮助追求心仪男生。
4. **得到最终性能**：建立信任、建立友谊、语言与身体的艺术、触发关系、突破点。

**总结**：

本节课中，我们学习了高分闺蜜策略，包括闺蜜定义、作用、成为男闺蜜的条件、提供价值、得到最终性能等。希望对大家有所帮助。
