## 声音的四大要素

那么，我们该如何训练自己的声音，让它变得更有魅力呢？在此之前，我们需要先了解声音的四大要素：音色、音量、音高和音长。它们相辅相成，也互相制约。

### 1. 音色：声音的特色

音色，即声音的特色和个性。它由你的口腔形状、发音时舌头的位置以及先天的声带发育特点决定。

**音色纯正**的第一标准是吐字清晰、明亮、圆润。用通俗的话说，就是说话能让别人听得清楚。

事实上，很多人说话时含糊不清，声音很大但粘连严重，让人听不清楚内容。有些人说话很快，像机关枪一样，也容易让人抓不到有效信息。

**圆润**的意思是说话声音不能太尖。比如有的女生嗓音很细，说话声音大一点就容易往头腔走，造成声音听起来不舒服。还有的人天生声音很低、很沙哑，即所谓的“公鸭嗓”。

通常情况下，音色明亮会给人活泼开朗的感觉，而音色低沉会给人比较成熟的感觉。

如果你的音色比较虚，并且喜欢用尾音上扬，会让人觉得你没有自信。如果你的音色像很多女生那样比较“嗲”，就会给人一种装腔作势的感觉。

因此，声音能够传递非常丰富的信息。对我们的指导意义在于：在任何情况下沟通交流时，一定要确保自己的状态良好、亢奋，不要让声音显得疲惫、低沉、虚弱，这会给人带来不好的感觉。你应该让声音时刻保持饱满、明亮的状态。

### 2. 音量：声音的大小

音量，顾名思义就是声音的大小。

有的人描述一件事情时听起来很有趣，是因为在讲述过程中，声音的音量轻重缓急、忽高忽低，富有节奏和韵律感。

但有的人描述同样一件事情，就像白开水一样平淡，再有趣的事从他嘴里讲出来都像在听天气预报，很没劲。这就是音量控制的问题。

还有的人为了让别人听到自己的声音，讲话时音量很高，这其实并不是非常好的选择。无论在什么场合，音量过高都是一种不成熟、低素质的表现。

你看那些擅长演讲、擅长讲话的人，他们在讲话时都不会刻意放大声音，而是保持一种平稳、得体的状态。

声音太高有一种喧宾夺主、哗众取宠的姿态，是一种没有安全感的表现，说明内心极度渴望被人关注。

生活中也是如此。例如，在餐厅需要服务员服务时，不必大声喊叫让每个人都知道。包括跟朋友分享有趣的事情或购物经历，也没有必要把声音提得很高让所有人都听见，这是一种不好的习惯。

在音量方面，我们需要注意的是：当你在描述一件事态的发展时，要找到它的重心和重点。例如，在类似“但是”、“其实他内心是这么想”这样的转折或强调处，当你想表达重要信息时，要有意加重音量，让大家注意到你接下来要说的内容。这是一种在音量上应该注意的表达策略。

### 3. 音高：声音的声调

音高，就是声音声调的高低，即声音的“Key”。它取决于你的声带在一定时间内振动的频率。

这里所说的高与低，并非一个绝对的客观标准，而是针对你个人而言的音调高低之分。

音调较高时，表示你处在一种比较兴奋、活泼的状态，会传达出高兴、愉悦的心情。

而音调比较低沉时，则会向别人传达出你比较忧郁、阴沉或平静的心理状态信息。

例如，同样说“啊”：
*   **“啊……”**（发音低沉、绵长）是一种比较低沉的状态。
*   **“啊！”**（发音短促、有力）则是一种比较高亢的状态。

你会发现这两者的区别在于发音位置的高度不同。

当你感觉发音位置非常下沉时，发出的音调就比较低。而当你人为地感觉把整个胸腔打开，呼吸道打开，整个人挺直，头微微上扬时再去发音，这时发音位置就比较高，发出的音调也较高。

因此，给我们的启示是：**你可以通过调整自己发音时的身体状态，来调整发音的高度。**

### 4. 音长：音节的长短

音长，指的是每个音节发音的时长是否干脆利落。

就像弹钢琴时，每个音之间是有一个明显的断联，还是有一种流畅的连奏感。

例如，在白天安慰一个朋友时，你可能会说：“我觉得你这样肯定是不对的，你这是种不负责任的行为。” 这是一种音长较短、比较干脆的说话方式。

而到了深夜，两个人喝了点酒，在一种很温情的气氛中互相安慰时，可能就不一样了：“我觉得呢，说实话，其实这件事情，并不能全怪你，因为没有任何一个矛盾的出现，是由一个人造成的……” 像这样的话，就是人为地把声音拖得比较长，所表达的感情状态也非常不同。

对你来说，应该训练自己驾驭不同说话感觉的能力，用不同的状态去尝试发音和练习。这样，你就能够在不同的场景下，通过声音表达出不同的感情状态。仅仅是通过声音，再配上相应的内容，就会让你的表达变得很有魅力。
