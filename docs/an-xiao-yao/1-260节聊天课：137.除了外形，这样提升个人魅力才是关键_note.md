# 1-260节聊天课：137.除了外形，这样提升个人魅力才是关键

## 概述
在本节课中，我们将学习如何提升个人魅力，特别是通过改变性格和行为习惯。

## 核心概念
**公式**：个人魅力 = 自我认知 + 行为改变 + 积极心态

### 了解自己
1. **明确性格特征**：每年花时间静心练习性格，列出优点和不太确信的优点。
2. **列出性格缺陷**：识别阻碍进步的拦路虎，如羞怯、暴躁、主观臆断等。

### 改变性格
1. **标记难以改变的性格特点**：划线标记。
2. **标记想要改变或加强的性格特点**：旁边标记小星星。

### 行为改变
1. **选择目标**：一次只选一个目标，如改变内向的性格。
2. **列出行为**：列出与他人相处时展现的内向行为，如提前离席、不敢插话等。
3. **反其道而行之**：逼迫自己参加更多活动，接受邀约。
4. **模仿榜样**：以欣赏的人为榜样，模仿他们的行为。

### 坚持与心态
1. **保持积极态度**：以积极的心态面对自己，保持自信和决心。
2. **接受新鲜事物**：加入新的组织，拓展视野。
3. **善待自己**：给自己时间和空间，放大自己的改变，给予自己正反馈。

## 总结
本节课中我们一起学习了如何提升个人魅力，通过了解自己、改变性格和行为习惯，以及保持积极心态。希望这些方法能帮助你成为一个更加魅力四射的人。
