## 追求大龄女生的要点
### 1. 洗脑工作
- 给女生灌输思想：你比她的心理年龄更大
- 清理掉女生认为小男生不成熟的偏见

### 2. 展示成熟
- 成熟更多是心态方面的展示和表现
- 成熟和幽默不矛盾
- 展示沉稳、冷静、善良、宽厚、仁爱、温柔、坚强、慷慨、自立、目标明确、雄有大志等特点
- 首先展示坚强、果断、心胸宽广、有责任感、有担当等特点
- 通过分享经历展示坚毅、成熟的一面

### 3. 破除预期
- 破除女生认为年纪小的男生情感状态不稳定的预期
- 聊聊你的情感态度，表达对精神契合伴侣的期望
- 感情重在经营，不抱有不切实际的幻想

### 4. 采取不同姿态
- 针对不同类型的女生，采取不同的相处模式
- 对比自己老到、见多识广的女生：踏实靠谱，展示真实自然，偶尔调情
- 对心智模式幼稚的小女孩：强势姿态，带她接触新鲜事物
- 注意：外貌年龄不是决定因素，重要的是一开始在女生心中塑造的形象

### 5. 注意事项
- 更加果断：在小事上拿捏不定，放大到大事上也会拿捏不定
- 心胸宽广、有包容心：在待人接物、面对错误时保持谦和、淡定
- 有担当：买单、计划、安排、照顾女生
- 扮演不同角色：朋友、爸爸、喜剧演员、上司，展示阅历和成熟
