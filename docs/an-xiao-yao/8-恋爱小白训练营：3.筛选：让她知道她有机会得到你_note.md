# 8-恋爱小白训练营：3.筛选：让她知道她有机会得到你

## 概述
在本节课中，我们将学习如何通过筛选过程，让女生知道她有机会得到你，从而增强你们关系的稳定性。

## 筛选的重要性
**筛选的本质**是赋予可得性，这是长期关系中非常重要的稳定剂。**公式**：可得性 = 兴趣来源 × 筛选标准。

### 筛选的步骤
以下是筛选女生的具体步骤：

1. **明确筛选标准**：例如，你喜欢自然美，可以与她讨论对整容的看法。
2. **避免肤浅条件**：不要只关注外表，要挖掘内在价值。
3. **给予认可和赞美**：明确指出她的特殊之处，让她知道你在乎她的独特性。

### 可得性的赋予
1. **注意给予方式**：不要一次性给予太多，以免她不珍惜。
2. **在吸引力的情境下**：筛选是赋予可得性，在没有吸引力的情境下，筛选会暴露需求感。

### 女生的疑问
1. **合不合适**：考虑价值匹配和生活模式是否相同。
2. **为什么喜欢我**：考虑你的兴趣来源是否真心。

### 总结
通过筛选，你可以让女生知道她有机会得到你，从而增强你们关系的稳定性。下次女生再问你“你喜欢我什么”的时候，认真告诉她你的理由，而不是随便敷衍她。
