# 1-260节聊天课：73.社交圈对于男女交往意味着什么_note

在本节课中，我们将探讨社交圈在人类交往，特别是两性关系中的核心作用。我们将从进化心理学的角度，理解人类如何本能地构建社交圈、评价彼此价值，以及这些机制如何深刻影响男女的交往策略与行为模式。

## 社交圈的本质与价值评估 🌐

上一节我们介绍了沟通中的情绪价值，本节中我们来看看社交环境如何塑造我们的关系。人类被设计去形成一个社交圈。我们会倾向于和那些能够提高我们生存与繁衍几率的人在一起形成社交圈。

我们并不会去直接评价一个人的绝对价值。正相反，我们会评价我们自身的价值如何被自己的圈子所影响。正是因为这样，我们才会自发地去选择同那些最合适的对象来做朋友。

这就意味着即使一个人的价值有一些绝对标准，但是他也常常会被相对地评价。这涉及到一种叫做**价值差别**的东西。换句话说，我真正关心的是你的、可能影响我自身价值的这种“相对价值”。

以下是更简单的例子：

*   我可能会更愿意跟一个很有钱的老板成为朋友，而不是愿意跟一个乞丐成为朋友。
*   区别就在于他们对于我们自身价值的影响是不一样的。前者是更加有利于我们生存的。

这就是先天被植入在我们意识中的一种价值评价方式。

## 大脑的信息处理与社交智慧 🧠

基于这种价值评估程序，我们的大脑会在意识中进行评价，并且更广泛、更经常地在我们的无意识中进行。我们的大脑无时无刻不在删除、曲解、归纳和过滤信息。

同时，我们的注意力、想法、情绪和合理化机制，都会准确地去关注那些一般来说最能潜在影响我们生存和繁衍的事物。同样的机制甚至扩展到我们的记忆。情绪不但强烈影响我们的行为，也会影响学习行为。更强烈的情绪会产生更优先的记忆。

如果你曾经在童年时期经历过真正的痛苦，那么相较于童年的其他记忆，你可能会更加清楚地记住这些痛苦的经历。

处于社交圈子中，免不了就会出现社交智慧的区分。社交智慧也是一种价值。那些能更准确地判断价值的人，就能够更好地保证自己拥有一个有利的圈子。

因此，我们的社交智慧、感受共振的能力、价值判断的能力、解读群体力学的能力等等，也是我们价值的组成部分，因为它带给我们竞争优势。简单来说：

**公式：更高的社交技能 ≈ 更高的社交价值**

## 两性对社交信号的不同解读 ⚖️

我们评价相对价值的能力，多数来自于我们对发生在周围的社交信号的敏感度。例如，那些观察到“预选择”社交信号并被吸引的女人，就能够比对手更有效地传播自己的基因。她们识别并且回应准确指标的倾向，能够帮助她们获得更高价值的对象。

同时，男人并不被“预选择”所吸引，因为这样的信息对女人有用，但对男人就不是这样了。男人通过视觉就可以评价出女人的许多价值。所以男人从未进化出“配偶复制”行为的压力。

正相反，男人在进化中偏好的指标是年轻、健康、生育能力、身材匀称、忠诚等。可以归纳为：

**公式：男人更易被外表吸引，女人更易被“游戏”（社交地位、预选择等信号）吸引。**

当然，这不是教条的黑与白真理，相反我们说的是一种倾向和概率。

## 关系作为投资：风险与策略 💰

如果我们把两个人之间的亲密关系形容成为一种“联盟”，那么这就会涉及到许多在时间和精力上的投资。如同任何投资那样，必须考虑风险和机会成本。

一个潜在对象的吸引力只是在结合时的一个指标。例如，一个吸引人但留不住男人的女人就处在劣势，相对于那些能够吸引并且留住他的女人。能够获得更高进化成功率的，当然是那些能够最好地选择并且维持其投资的人。

当选择配偶时，对方可能有高价值，但也意味着高风险。更大的价值差异也意味着更大的风险。当事人必须考虑风险，可能需要降低自己的标准。

一个男人可能愿意跟一个女人发生短期关系，但并不意味着他也愿意交付自己的长期资源（例如，冒风险怀孕）。在许多例子中，他的程序也在下意识地计算是否值得冒险出轨。他的潜意识交配策略会基于当前环境，计算潜在的收益与风险。

## 两性的不同出轨策略 🔄

两性其实有时都会因为误入歧途而出轨。很自然地，他们会有不同的内在策略。

女性的策略一般倾向于：喜欢一夫一妻制，和具有最高价值的男性建立长期关系以获得承诺。她对于这个男人会非常挑剔。一旦她得到了他的承诺价值，她的出轨策略可能是为了获得高价值的基因。当她的情绪指标显示值得冒险时，她可能行动。

**代码逻辑示例（简化）：**
```
if (当前伴侣价值不足 && 遇到高基因价值对象 && 风险评估通过) {
    可能触发出轨策略;
}
```

值得注意的是，每个月女人最容易受孕的那几天，也是她们可能比平时更偏爱男性化特征（通常与基因质量相关）的时期。
