# 1-260节聊天课：73. 社交圈对于男女交往意味着什么 🤝

## 概述

在本节课中，我们将探讨社交圈对男女交往的影响，以及人类如何通过社交圈来评估和选择伴侣。

## 社交圈的形成

**人类被设计去形成一个社交圈，我们会倾向于和那些能够提高我们生存与繁衍纪律的人，去在一起形成社交圈。**

**公式**： 社交圈 = 生存与繁衍纪律 + 个体价值

## 价值评价

**我们并不会去直接的评价一个人的价值，正相反，我们会评价我们自身的价值如何被自己的圈子所影响。**

**公式**： 个体价值 = 圈子影响

## 价值差异

**那么这就意味着即使一个人的价值有一些绝对标准，但是他也常常会被相对的评价，有一种叫做价值差别的东西。**

**公式**： 价值差异 = 绝对价值 - 相对价值

## 社交智慧

**那些能更准确的去判断价值的人，就能够更好的去保证自己拥有一个有利的圈子。**

**公式**： 社交智慧 = 价值判断 + 圈子选择

## 男女差异

**男人被外表吸引，女人被游戏吸引，但是当然这不是教条的黑与白的真理，相反我们说的是一种倾向和概率。**

**公式**： 男性偏好 = 外表 + 年轻 + 健康 + 生育能力
**女性偏好 = 游戏性 + 价值 + 承诺**

## 亲密关系

**如果呢我们把两个人之间的亲密关系去形容成为一种，联盟的话，那么这就会涉及到许多在时间和经历上的投资。**

**公式**： 亲密关系 = 投资回报 + 风险
