# 1-260节聊天课：39.可以无限复制的幽默小技巧_note

在本节课中，我们将学习两种简单实用的幽默聊天技巧：“假否认”与“假附和”。这两种技巧的核心在于制造语言上的反差，从而化解尴尬、应对测试，并营造轻松有趣的交流氛围。

## 假否认：表面否认，实则承认

上一节我们介绍了课程主题，本节中我们来看看第一个技巧“假否认”。这个技巧的核心是，当对方（尤其是女生）对你提出指责或质问时，你表面上进行否认，但实际内容却是在承认对方的话。这种表面与实质的反差能制造出幽默效果。

它的应用场景主要是应对女生对男生的“废物测试”。应对得好，既能化解测试，又能展现幽默感；应对不好，则容易显得自身价值很低。

**核心公式**：**表面否认 + 实质承认（常带夸张）= 幽默效果**

以下是“假否认”的几个应用实例：

*   **场景一：被问是否经常搭讪**
    *   女生问：“你是不是经常在大街上搭讪认识女生啊？”
    *   低价值回答：“是”或“不是”。
    *   幽默回答（假否认）：“不经常啊，一天就认识十几个吧。”

*   **场景二：被问是否常去某个场所**
    *   女生问：“你经常来这玩吗？是不是在这约过很多姑娘？”
    *   低价值回答：“经常来”或“我第一次来”。
    *   幽默回答（假否认）：“不经常啊，一天只来一次。”

*   **场景三：被问是否常带女生回家**
    *   女生问：“你是不是经常带女生回家啊？”
    *   幽默回答（假否认）：“没有啊，我经常带男人回家。”

关键在于，女生进行废物测试时，往往不在意答案本身，而是在观察你应对问题的态度和能力。使用“假否认”技巧，能巧妙避开直接承认或否认的陷阱，以幽默化解尴尬，让女生觉得你反应机智、心态轻松。

## 假附和：先附和，再衍生或夸大

学会了“假否认”来应对质问，接下来我们看看在需要表达不同意见时，如何运用“假附和”。当女生表达某个观点，你内心不同意但又不便直接反对时，可以先假装附和她的观点，然后通过衍生、夸大的方式，巧妙地表达你的不赞同。

**核心公式**：**表面附和 + 夸大/衍生 = 委婉表达不同意见**

以下是“假附和”的几个应用实例：

*   **场景一：应对不喜欢的观点**
    *   女生说：“我其实挺讨厌男生抽烟的。”（而你抽烟）
    *   不佳回答：“我从来不抽”（讨好）或“我们很早就抽了”（顶撞）。
    *   幽默回答（假附和）：“其实抽烟应该是女生的爱好。”（先附和“讨厌男生抽烟”的感觉，然后衍生出一个新奇观点引发好奇）

*   **场景二：委婉表达不满**
    *   女生约会迟到却说：“我还挺准时的吧？”
    *   直接表达不满可能破坏气氛。
    *   幽默回答（假附和）：“你真的是挺准时的，你每次跟我约会，从来都不迟到超过一个小时。”（先附和“准时”，再通过夸大事实委婉指出问题）

*   **场景三：日常趣味互动**
    *   女生问：“我这个新衣服/口红漂亮吗？”
    *   普通回答：“漂亮。”
    *   幽默回答（假附和）：“嗯，非常漂亮。”（短暂沉默后，假装刚反应过来）“诶，你刚说什么漂亮？”（通过制造一个小转折增加互动趣味）

这种方法能在不直接冲突的前提下，表达你的态度，并为对话增添情绪起伏和趣味性。

## 课程总结

本节课中，我们一起学习了两种制造幽默的聊天技巧：
1.  **假否认**：用于应对质问或测试，通过“表面否认，实质承认”的反差制造幽默，化解尴尬。
2.  **假附和**：用于委婉表达不同意见，通过“先附和，再夸大或衍生”的方式，巧妙传递态度，增加谈话趣味。

两者的核心都在于创造“表面态度”与“实际内容”之间的反差。掌握这两种技巧，能帮助你在社交中更从容地应对各种情境，展现幽默与智慧。
