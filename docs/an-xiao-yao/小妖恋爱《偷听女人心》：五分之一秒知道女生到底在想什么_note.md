## 微表情概述
微表情是指人在无意识中表现出的面部表情，它反映了人的真实情绪和想法。以下是一些常见的微表情及其含义：

### 瞳孔扩大
当一个人遇到喜欢的人或开心的事情时，瞳孔会不自觉地扩大。**公式**：瞳孔扩大 → 感兴趣或喜欢。

### 上眼皮松弛
当一个人对某件事情感兴趣时，上眼皮会出现松弛或轻微的褶皱。**公式**：上眼皮松弛 → 专注或感兴趣。

### 眉毛平缓
当一个人感到疲惫或对某件事情不感兴趣时，眉毛会变得平缓，眉间距离会变小。**公式**：眉毛平缓 → 疲惫或不感兴趣。

### 第一次见面
在第一次见面时，观察对方的瞳孔、身体姿态和表情，可以判断对方是否对你感兴趣。

### 讲故事时
在讲故事时，观察对方的表情和身体姿态，可以判断对方是否对你所说的事情感兴趣。

### 拉手时
在拉手时，观察对方的眼神和面部表情，可以判断对方是否愿意升级关系。
