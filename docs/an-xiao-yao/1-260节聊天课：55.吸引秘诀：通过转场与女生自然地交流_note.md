# 聊天课 1-260：55. 吸引秘诀：通过转场与女生自然地交流 🎯

在本节课中，我们将要学习约会中的一个重要技巧：转场。转场是指在约会过程中，主动更换到另一个场地或空间与女生相处和交流。掌握这一技巧，能帮助你们建立更自然、舒适的互动关系。

## 什么是转场？🔄

转场，即换到另外一个场地来跟女生相处，换到另外一个空间来跟女生进行交流。

上一节我们介绍了转场的定义，本节中我们来看看为什么要进行转场。

## 转场的意义与目的 🧠

两个人需要一些很自然的相处，这样女生才能渐渐把注意力放在你身上。或者说，渐渐产生那种说不清道不明的东西，这个东西就是情感。情感不是通过理性的决策说“我要产生情感”就能够产生的，而是在不经意中被唤醒、被调动起来的一种心理能量。

很多时候，你刻意跟女生讲很多话，刻意表现自己，说了一堆，其实这时女生对你的注意力都是一种被动的注意力。这不是一种自然的力量。

你需要的一种社交的注意力。就是当你们两个很放松，比如在逛街时，突然旁边有一只很可爱的小狗过来。这时你很自然地蹲下去，跟它互动，摸一摸它。很可能你在那边玩狗，女生在你身后看着你，就会在心里觉得这个男生还挺可爱、挺有耐心，对小宠物很关爱。这时就会有某个瞬间，女生内心的情感被激发出来。

这就是一种非常自然的注意力，这个状态也是最好、最有效率的。它比你一直跟她讲很多话题要强很多，后者甚至会引起女生的厌烦。

所以，大家要有这样一种意识：要通过转场来跟女生有很多自然的相处，让女生有一个在心里面渐渐接受你的过程。而不是带着很强的目的性，无时无刻不把注意力放在她身上。

比如不管到哪儿，时刻想着“她累不累呀”、“她是不是不开心啊”、“她饿不饿呀”。这样反而会让女生有很大压力，让她觉得反感。你倒不如反过来，忘记她的存在，不去注意她。

比如还是一起去逛街，逛一个小商店。你不要总是跟在女生后面，她看什么你看什么，这样就没什么劲了。你完全可以放飞自我，去看一些你自己喜欢的东西。这样才会让女生觉得很舒服。

而不是让女生有一种她在吃饭、她在吃一道菜，然后你一直盯着她看的那种很不爽的感觉。也就是说，你的注意力要来回地跳跃。

要从女生身上跳跃到周边的事物上，然后再跳回来，然后再跳离开。有这样的反复，就能够有一疏一密，能够给女生有放松的时间。这样才是比较合适的一种相处方式。

所以，我们一定要学会去分散注意力，能够学会去自然的相处。

## 何时以及为何需要转场？⏰

可能你在某一个地方跟女生聊得非常开心，但这时你也一定要有意识去结束当前这个地方的互动，一定要去一个新的地方。否则，在一个地方待久了就会有一种焦虑感。

为什么会焦虑呢？这其实跟我们上课是一样的。你在一个教室里面听一个老师讲一节课，你最高能承受的极限就是45分钟一堂课。让你待久了，即使这个老师再有个人魅力，他讲的知识再精彩，你都不愿意去听了。这是人的一种本能、一种本性：你对一个地方的注意力，最多只能持续45分钟。

在约会中也是一样的。当你在一个地方觉得跟女生聊天已经度过了聊天的高潮、高峰时期之后，你觉得两个人开始好像不知道该说什么了，开始出现一些空荡冷场了，那么这时就应该有意识选择下一个地方，进行下一次约会了。

## 转场带来的额外好处 🤝

当你学会去转场，学会换不同的地方来跟女生相处时，你会发现无形中，你们之间的熟悉感会大大增强。

因为只有你们两个人共同经历了由一个地方到另外一个陌生的地方这样一个过程时，你们之间才会形成那样一种心灵上的默契感，一种同甘共苦的感觉。这是一种非常宝贵的心理上的体验。
