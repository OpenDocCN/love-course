有效的调情不单靠语言完成，而是因为你真正扮演了那个角色。你的手势、面部表情、说话语调都在扮演这个角色。就像观看喜剧，乐趣不仅来自语言本身，更来自表演者生动的形象与话语的有机结合。这才是真正有趣的部分。如果表演痕迹过于明显，则会索然无味。
