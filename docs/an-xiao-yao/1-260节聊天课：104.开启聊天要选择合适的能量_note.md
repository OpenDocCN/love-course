# 🎤 1-260节聊天课：104.开启聊天要选择合适的能量

在本节课中，我们将学习如何在认识女生时，通过控制表达和交流的能量等级来开启一段成功的聊天。

## 1. 能量等级的重要性

**公式**：能量等级 = 表达方式 × 环境能量

以下是能量等级的三个级别：

* **一级**：最低能量等级，适用于安静的环境，如星巴克。使用轻柔的声音和低调的声线。
* **二级**：适用于街道或商场等嘈杂环境。使用较高的能量等级来吸引对方的注意。
* **三级**：适用于酒吧等非常嘈杂的环境。需要使用更高的能量等级来确保对方能够听到你的话。

## 2. 如何截停女生

* **走路节奏**：通过调整自己的走路节奏，引导对方放慢脚步，最终停下来。
* **身体语言**：在接近对方时，保持45度角的身体朝向，避免给对方带来压迫感。

## 3. 捕捉兴趣指标

* **停下来听你讲话**
* **整理头发或注意形象**
* **询问问题**
* **身体微微靠近你**

## 4. 说话态度和用词

* **礼貌尊重**：用礼貌和尊重的语言与对方交流，展现你的诚意。
* **避免轻浮**：避免使用轻浮的词汇，如“美女，一起出来玩”。
* **使用“我们”**：尽量使用“我们”而不是“你”或“我”，展现你的团队意识。

## 总结

本节课中，我们学习了如何通过控制能量等级、走路节奏、身体语言、兴趣指标和说话态度来开启一段成功的聊天。希望这些技巧能够帮助你更好地与女生交流。
