# 1-260节聊天课：104.开启聊天要选择合适的能量_note

在本节课中，我们将学习如何在与女生初次交流时，控制你的表达方式和能量等级。合适的能量等级能显著影响对方的回应，是成功开启对话的关键。

## 理解能量等级

表达方式多种多样。有的人表达时静如止水，非常柔和清淡。有的人讲话时则手脚并用，情绪高涨。

当你认识一个女生时，第一句话或第一个字的能量等级若未把握好，得到的回应会有很大差别。

我们可以将能量等级分为三级。

*   **一级**代表最低能量等级。你可以用最轻柔的声音和最低调的声线说话。
*   **二级**是中等能量等级。
*   **三级**是最高能量等级。

## 根据环境选择能量

上一节我们介绍了能量等级的概念，本节中我们来看看如何根据环境选择匹配的能量。

选择能量等级的首要依据是**环境**。你需要使自己的能量与环境能量相匹配。

以下是不同环境下的能量选择建议：

*   **在星巴克等安静场所**：环境能量等级为1。你需使用等级为1的说话方式开场。在此环境下，无需大声说话对方也能听清。若使用过高能量，例如大声说“嗨”，会吸引周围人目光，给女生带来社交压力，增加被拒绝的可能性。
*   **在街上或商场**：环境能量等级为2。此时若声音太小，对方可能听不到或未意识到你在与她讲话。
*   **在酒吧或吵闹场所**：环境能量等级为3。你需要用很高的能量开场。否则，过于低调可能让人误以为你行为猥琐。

## 根据对方状态调整能量

除了环境，能量等级的选择还需观察**对方当时的状态**。

*   若女生在沉思、走得慢，用高能量方式说话显得不合适。
*   若一对闺蜜正开心说笑，而你接近时能量低沉沉闷，则会对你的第一印象产生负面影响。

因此，控制好能量等级，说好第一句话，是至关重要的关键。

## 如何自然地截停对方

控制好开场能量后，下一步是让对方愿意停下来与你交流。有一点是毋庸置疑的：若想获得联系方式，女生必须停下脚步与你沟通。她不可能一边向前走一边告诉你号码。

能让对方停下脚步的关键，在于**你自身走路的节奏**。你要让她在你的带动下，愿意跟随你的脚步放慢并最终停止。

以下是具体做法：

*   **理想情况**：出现在对方面前第一步时，就尽量截停她。
*   **如果未停**：可先陪她走一小段路，然后**让你自身的脚步慢慢放慢、逐渐停止**，同时带动她也放慢脚步停下来。

## 捕捉兴趣指标（IOI）

在搭讪过程中，捕捉对方反馈的**兴趣指标**至关重要。很多男生将注意力全放在自己身上，忽略了互动的双向性。

若只有你在策划整个过程，对方毫无参与，那仅是单方面互动。好的搭讪中，女生会给出许多反应，你需要从中挑选微妙的信号予以回应，互动才能升级。

以下是一些微妙的兴趣指标示例：

*   她停下来听你讲话。
*   她整理自己的头发或注意形象。
*   她向你提问。
*   她的身体微微向前靠近你。

## 注意身体朝向与态度

开始交流时，你的身体朝向应与对方成**45度角**。这个角度能掩饰你的需求感，且不会给女生带来压迫感。

如果交流开始后，对方转为正面面对你，此时你也**一定要将身体朝向她的正面**，这是礼貌的做法。

下一个要点是**说话时的态度和用词**。态度和情绪贯穿整个互动过程，尤其在开始时尤为重要。

去搭讪本身已是展现勇气的酷炫行为。在这种状态下，你应尽量使用**礼貌和尊重的语言**来调和。

*   若态度很酷，语言却很轻佻，会招人讨厌。
*   若态度很酷，语言礼貌尊重，则大多数女生更容易接受，因为你表现出了尊重。

用词方面也需注意。避免使用“美女，一起出来玩”这类听起来轻佻的词汇。

可以更严肃地说：“希望下次有机会可以出来喝杯咖啡，好好认识一下。”

同时，尽量使用“我们”，减少使用“你”或“我”。例如：

*   `“我能不能要你的号码？”` （效果较差）
*   `“我们两个留个联系方式吧。”` （效果更好）

这两种表达方式的效果截然不同。
