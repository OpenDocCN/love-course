# 撩妹聊天大师 第97课：男人这样的诚实，女人都超级喜欢 💖

在本节课中，我们将要学习如何通过“诚实”与“筛选”的心态，在与女性互动中建立强大的吸引力。核心在于转变思维，从“讨好”变为“选择”，并通过直接、真诚的沟通方式瓦解对方的心理防御。

---

许多男生过于在意给对方留下深刻印象。其实，关键在于让女生感觉到**你正在筛选她**。起初你可以有意伪装这种心态。

当你习惯这种心态后，你便能真正做到筛选女生，判断她是否符合你的标准。与女生聊天时，我的内心状态是：**她必须告诉我任何我想知道的事情**。只有在此基础上，我才会考虑是否要使用一些技巧或进行诱惑。

“诱惑”一词容易引发负面联想。我们不妨将其替换为更积极的理念：**诚实、信任与尊重**。

以下是你可以直接询问女生的一些问题，用以了解她的内在：
*   什么事情让你感到高兴？
*   小时候你通常做什么？
*   你最好的朋友是谁？
*   你做过哪些让自己骄傲的事？为什么？
*   你有过尴尬的时刻吗？
*   你养过宠物吗？给它起了什么名字？

提问时，**必须保持眼神接触**。这能传递出你对她的内在真正感兴趣。如果你回避眼神接触，她很难对你产生兴趣。

你必须让女人感受到你强大的心态，而眼神接触至关重要。她能从你的眼神中感知到很多信息。

我认为这样注视是理所应当的，是对她的一种尊重。对女人感兴趣这件事本身没有错。

很多男生会问：得到女生联系方式后，何时联系她比较合适？

我的答案是：**任何时候都可以**。只要你想，随时可以打电话或发信息。即使你正和一位女士相处，也可以给另一位女士打电话，告诉她：“哎，我想起你了。”

为什么要这样做？因为我们诚实。我怎么想，就怎么说。而且，女生通常也愿意听到这样的实话。这未尝不是一种浪漫的表达。

女人不希望你害怕她们。她们需要的是被照顾、被供养、被保护。她们需要感受到浪漫，需要觉得你比她们更强。

我很喜欢在第一次约会时问女人想去哪里。这能帮助我判断她的类型。如果她想去高级餐厅，我就带她去大排档。即使她抱怨，我也能提前知道她对我感觉不佳，从而尽早结束约会。

女人喜欢诚实的男人。虽然她们不期待你诚实，但一旦你展现出诚实，她的心理防御便会自然瓦解。而防御的瓦解，就是对你产生感觉的开始。

如果你和女生聊了十分钟，还不知道如何推进关系，你可以直接告诉她：
`“我已经和你谈了十分钟了，我现在正在想，怎么样能够把你带到那棵树下，然后深深地吻你。”`

女人都需要一个主动的男人，但过于主动又会让她们害怕。尊重有两种：源于害怕，或源于崇拜。你能带来哪一种，你就是哪一种。
