# 1-260节聊天课：89. 暧昧的五种基本套路 🤔

## 概述
在本节课中，我们将探讨如何通过“推拉”技巧轻松进入暧昧状态。

## 推拉技巧
推拉是一种通过拉进和推开对方来吸引对方注意力的技巧。

### 推拉的定义
- **推**：将对方推开，给予负面反馈。
- **拉**：给予正面反馈，拉近与对方的距离。

### 推拉的重要性
- 推拉可以制造情绪上的波动，吸引对方。
- 推拉可以增加不可预测性和悬疑感。

### 推拉的应用
- **拉**：通过暗示、认可或肯定来吸引对方。
- **推**：通过暗示、拒绝或批评来推开对方。

## 五种推拉技巧
以下是五种不同的推拉技巧：

### 1. 后悔感
- 当对方表现出兴趣，然后突然退缩，让对方感到后悔。
- 例如：对方主动示好，然后突然拒绝。

### 2. 情感过山车
- 通过快速变化情绪，让对方感到兴奋和好奇。
- 例如：先热情，然后冷淡，再热情。

### 3. 显示与隐藏
- 有意地展示某些方面，同时隐藏其他方面。
- 例如：展示幽默感，同时隐藏脆弱。

### 4. 好警察，坏警察
- 使用两种截然不同的态度，让对方感到困惑。
- 例如：一方面关心对方，另一方面又批评对方。

### 5. 故意推翻
- 故意说出与事实相反的话，让对方感到惊讶。
- 例如：对方说天气好，你却故意说天气糟糕。

## 总结
本节课中，我们学习了如何通过推拉技巧进入暧昧状态。通过掌握这五种推拉技巧，你可以更好地吸引和吸引对方。
