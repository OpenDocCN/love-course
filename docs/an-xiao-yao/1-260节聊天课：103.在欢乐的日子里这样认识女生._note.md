# 1-260节聊天课：103.在欢乐的日子里这样认识女生

大家好，我是夏洛，这里是撩美聊天大师的第103课。年底了，各种娱乐活动丰富多彩，认识喜欢的女生变得尤为重要。因为这些活动中，大家心态轻松，更容易接纳新朋友。

## 搭讪技巧

**1. 隐藏需求感**

搭讪时，你的需求感会通过身体语言表露，所以要尽量隐藏。肢体语言撒谎困难，所以保持自然。

**2. 减少小动作**

紧张时，小动作会增多，让对方感到紧张。避免用身体或手挡住对方前进路线，以免造成侵犯感。

**3. 礼貌接近**

礼貌的接近更容易被接受。带着欣赏和认可的心态，而非游戏心态。

**4. 切入角度**

在对方一侧的前中后位置切入，前方1-1.5米处用弧线运动轨迹接近。

**5. 身体语言**

身体后倾和朝向45度，避免侵略性。放慢语言速度，清晰表达。

**6. 递联系方式**

在对话结束后递手机，将社交压力转给对方，让对方更愿意留下号码。

**7. 互动时间**

保持3-5分钟互动时间，让对方记住互动情景，提高后续互动可能性。

## 心态调整

**1. 享受活动**

将心态转换为享受活动，而非刻意寻找目标。

**2. 微笑放松**

时刻微笑，避免在人群中搜索目标，动作和眼神要放松。

**3. 合适时机**

选择合适的时机移动想认识的女生，如提出一起喝东西的要求。

## 总结

本节课中，我们学习了如何在欢乐的日子里认识女生。搭讪时要注意隐藏需求感、礼貌接近、选择合适的切入角度和身体语言。同时，调整心态，享受活动，保持微笑和放松。希望这些技巧能帮助你在欢乐的日子里结识心仪的女生。
