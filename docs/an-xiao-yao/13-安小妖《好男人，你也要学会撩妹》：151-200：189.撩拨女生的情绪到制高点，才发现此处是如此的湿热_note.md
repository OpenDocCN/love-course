**公式**：感觉 = 情绪 + 期待

*   了解女生对“感觉”的理解，有助于我们更好地制造这种感觉。

### 2. 制造感觉的方法

*   **公式**：新鲜感 = 持续的惊喜 + 保留
*   通过保留一部分特质，并持续地给予惊喜，保持新鲜感。

#### 方法四：崇拜法则
*   **公式**：崇拜 = 独立思维 + 独特认知
