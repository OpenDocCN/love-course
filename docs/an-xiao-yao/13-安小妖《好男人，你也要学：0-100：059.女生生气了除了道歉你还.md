## 判断生气类型：真生气还是傲娇？ 🤔

上一节我们介绍了课程概述，本节中我们来看看如何判断女生是真正生气还是在傲娇。首先，你需要区分这两种情绪状态。

在以下情况下，女生可能会傲娇：
*   你开玩笑说她“最近圆润了”或“这件衣服不适合你”。
*   她以轻微的情绪表达不满，但并非真正动怒。

面对傲娇，你的应对方式是及时找补并夸奖她。例如：
```
“我就喜欢你这样，一点都不胖。”

```
