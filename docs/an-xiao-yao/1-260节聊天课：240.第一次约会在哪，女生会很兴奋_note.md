# 1-260节聊天课：240.第一次约会在哪，女生会很兴奋

## 概述
在本节课中，我们将学习如何在半固定场所搭讪女生，并了解如何通过巧妙的方法留下深刻印象，最终获得联系方式。

## 半固定场所的定义
半固定场所是指双方或其中一方在特定地点有固定活动，但并非完全固定。以下是一些例子：

- **第一种情况**：对方固定而你不固定
  - 护士、售楼部导购、商场前台、超市收银员、咖啡厅老板等。

- **第二种情况**：双方都不固定，只是停留
  - 公园、河边散步、电影院、酒吧等。

## 行动方案
以下是一个简单的四个步骤的行动方案：

### 第一步：多制造偶遇
在心仪女生的工作场所多出现，买买东西去那里转转，保证你在她面前的曝光率。

### 第二步：留下印象
通过行为或交谈方式给她留下特殊印象。例如，在健身房喜欢上前台女生，但每次接触只是递健身卡，走的时候再拿回。

### 第三步：要联系方式
当你和她见过三次以上，可以要联系方式。通过谈话引导，使用思维风暴聊天法挖掘共同喜好。

### 第四步：深入聊天
避免聊关于工作的事，尽快解除她的工作状态。可以通过调侃和转移话题来实现。

## 举例说明
以下是一个具体的例子：

- **场景**：你在医院认识了一个护士，出现三次，聊了几句。
- **问题**：她还是和你讲医院里的事。

- **目的**：通过这种方式，你可以把话题引到其他方面，解除她的工作状态。
