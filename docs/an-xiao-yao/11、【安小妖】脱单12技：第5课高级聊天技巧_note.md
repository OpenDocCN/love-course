# 11、【安小妖】脱单12技：第5课高级聊天技巧

## 概述
在本节课中，我们将学习如何运用高级聊天技巧来提升与女生的互动效果，包括聊天的目的、识别女生的聊天性质、聊天流程以及如何破解废物测试和ASD。

## 聊天的目的
1. **铺垫邀约**：通过聊天为邀约做准备，想象约会场景，提前铺垫。
2. **加速现场游戏**：通过聊天加速现场游戏的进展。
3. **测试女生**：通过聊天判断女生对你的吸引程度，了解关系阶段。

## 识别女生的聊天性质
1. **IOI（兴趣指标）**：女生对你感兴趣，主动提供话题或询问个人信息。
2. **IOD（无兴趣指标）**：女生对你不感兴趣，可能通过表情或话题表达。
3. **ASD（反弹机制）**：女生认为你对她不感兴趣，可能通过负面评价或行为表达。
4. **废物测试**：女生通过提问或行为测试你的价值。

## 聊天流程
1. **提供话题**：根据女生信息或朋友圈提供话题。
2. **建立框架与推拉**：在聊天过程中，通过话题引导和表情表达，与女生进行推拉互动。
3. **调侃与奖励**：女生反应好时，进行调侃并奖励升级关系。
4. **测试窗口与邀约**：测试女生是否真的对你感兴趣，进行邀约。
5. **推开与重新开启话题**：女生反应不好时，主动推开并开启新话题。

## 破解废物测试与ASD
1. **废物测试**：通过侧面回答、反问或否定来破解。
2. **ASD**：给女生台阶，降低预期，并给予IOD回馈。
