# 1-260节聊天课：85.养成这种说话习惯，做一个活好还不粘人的男人_note

在本节课中，我们将学习如何借鉴猫的特质，在男女交往中塑造一种轻松、有魅力且不粘人的沟通方式。核心在于不主动索取情绪价值，同时营造有趣、无压力的互动氛围。

## 课程核心：猫的启示 🐾

上一段提到的“猫令人主动讨好的能力，来自于它从来不主动地讨好人”这句话很有深意。为什么猫招人喜欢？因为它可爱，但从不主动理人、不黏人、不抓着人不放。你找它，它就玩耍；你不找它，它也不会表现出过度的留恋纠缠。

这一点正是它魅力的来源。更精确地说，它从不向别人索要情绪价值，同时让你和它相处起来没有任何压力。这就是它最大的魅力所在。

## 应用于人际交往 🤝

将这个模型套用在男女交往上，道理是一样的。最有魅力的人具有这些特征：你和TA相处起来很舒服、没有压力、感觉非常好，可以很亲密热络。但是，TA从来不会主动找你分享弱点或小心思，一直保持一种酷酷的存在。

所以，我们要学会做这样的人。“不粘人”相对容易做到，只要你能克制自己，不对喜欢的人说无关痛痒、无病呻吟的话，不用没有意义的情绪去干扰对方。

## 如何营造有趣氛围？ 🎭

那么，如何让你和女生接触时，让她觉得氛围更好、更有趣呢？你需要养成一种特定的说话习惯。

以下是几个具体的例子和表达方式：

*   **语言调侃示例**：你可以问女生：“你会英文之外的其他外语吗？”如果她冷漠地回答“不会”，你可以接着说：“那我们之间就有差距了。不过没关系，我人品这么好，可以教你。” 这是一种亲切、友好、带点小调侃的对话方式，能表现出可爱、好玩的特质，同时对她有小小的调侃。这就是我们昨天聊到的“自大有趣”聊天法。

*   **带领框架示例**：带女生去餐厅或某个地方玩时，你可以说：“这个地方你来过吗？没来过吧？那我就在你心里插上一根小旗子了，这个地方是我带你来的。如果你表现好，以后可以带你去更多好玩的地方。” 这种表达方式能在无形中植入你强势的框架，同时带领她，并给她一种好玩、软化、亲切的体验。

*   **浪漫预期示例**：和女生出去玩时，你可能会带一个拍立得（撩妹小武器）。女生看到后可能会说：“你还带这个？” 你可以大方地说：“这是我的撩妹神器。今天，我打算为你拍三张照片。回去后你可以放在书本里或贴在墙上，我觉得会很好看、很可爱。如果你表现好，以后我们就有更多相处机会，我能为你拍更多照片。到时候你就会拥有一整面墙的照片，都是一个很可爱很萌的你，都是我眼中的你。怎么样，是不是很浪漫？”

很多时候，就是要像这样给女生一个很强的框架，给她一种无法拒绝的浪漫、情调和体验。这是一种非常有效、能快速释放吸引力、打开双方情绪开关的交流状态和方法。

## 总结与练习 📝

本节课我们一起学习了如何养成不粘人且有魅力的说话习惯。核心是**不主动索取情绪价值**，并通过**有趣、带框架的调侃**来营造轻松愉快的互动氛围。

你可以经常练习对女生说：“如果你表现得更好，我会……”，这相当于强行植入一个奖励机制，诱导和引导她给你更好的回应和反馈。

这种表达方式、感觉和习惯，值得你去学习、练习和尝试。
