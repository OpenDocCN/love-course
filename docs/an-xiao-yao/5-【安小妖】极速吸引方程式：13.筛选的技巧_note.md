# 5-【安小妖】极速吸引方程式：13.筛选的技巧_note

在本节课中，我们将学习如何运用筛选的技巧，通过温和或生硬的台词，引导女性展现其内在品质，从而建立吸引并自然过渡到舒适感阶段。

---

上一节我们介绍了筛选的基本概念，本节中我们来看看具体的筛选技巧。筛选台词是任何能鼓励或帮助女性认识到自身良好品质的话语。声音的筛选台词如同音频面试，你需明确表现出正在寻求某些特质，并顺着对方的话语进行评价。

温和的筛选台词应是微妙且难以察觉的，这能使整个过程显得自然。你的筛选台词需温和，同时能让女性展现其吸引人的特质，并让她感受到她在努力赢得你的兴趣。毕竟没有人喜欢面试。

温和的筛选台词并不难，你只需提出一些能让她描述自身个性或给出吸引你理由的话题。

以下是常见的话题示例：
*   旅行
*   职业
*   教育
*   兴趣与爱好
*   宠物
*   烹饪
*   “如果……会怎样”类问题（例如：如果你能变成任何动物，你会选择什么？）

这些话题可以通过提问或陈述的方式提出。以旅行为例，你可以问：“你喜欢旅行吗？”随后可跟进提问：“你去过哪里？”、“你最喜欢什么地方？”或“你最想去哪个地方？”当然，不要问遍所有问题，否则她会觉得自己在参加面试。

如果使用陈述方式，你可以说：“我很喜欢旅行。”然后观察她的回应。如果她已被你吸引，她会很愿意与你谈论彼此的共同点。

要把陈述和提问结合起来，不要只依赖其中一种。你可以问她一个她可能感兴趣的话题，这样即使让筛选台词深入一些也没有关系，例如：“那你的故事呢？”或“那么，你是谁？”

当她开始讲述时，你要使用过滤提问，或用你自己的提问意识引导她展现积极特质。事先想好你的后续回应，以便发现你们之间的共同点。这样做，你可以比局限于指定话题找到更多内容。这种方法也会促使她更动脑筋。

如果你对一个女人感兴趣，你不能让她不假思索地通过你的筛选问题。你让她投入更多思考，会增强她对你的联系感。

最生硬的筛选台词需承担冒失和尴尬的风险。但当你发现女性对你感兴趣，且需要她努力赢取你时，生硬的筛选台词也能发挥作用。

以下是一组生硬的筛选台词示例：
1.  那你认为你有什么优点呢？
2.  为什么我应该要你的电话号码？（或：为什么我应该打电话给你？）

有的女性会很自然、很容易地对生硬的筛选台词进行回应，另一些则慢热一些，需要哄一哄。但从温和的筛选台词开始练习会比较好，让她习惯于“努力赢取你的认可”这种想法，并且真的是通过她的个性而非外貌来赢取。

当女性告诉你一些让你对她更感兴趣的事情时，奖励她。毕竟这是你希望她做的事情。认同她就是一个很好的奖励，但这种认同必须让她感觉到真诚。通过深入谈论她提供的话题，来显示你真的感兴趣。

当你能够利用她对筛选问题的回应，作为探索彼此共性的催化剂时，你就能从筛选过程进阶到舒适感阶段了。即使她没有对某个难题提供有用信息，只要她努力回答了，你也应该奖励她。

记住，将谈话内容局限在类似话题上是不理智的。如果她尝试回答了你的筛选问题，对她好一点，换个话题，过段时间再尝试另一个温和的筛选问题。

同样，一个有要求的女性不会在你只展现个性某一面时，就对你表示强烈兴趣。你不应该只在她身上发现一个感兴趣的特征后就结束筛选。你应该从几个不同方面筛选她，花时间把这些特性发掘出来。

在筛选问题之间，用一些朋友间的正常谈话连接起来。这个过程越自然越好。如果她们没有感受到你的真诚，或认为你正根据某张检查清单来筛选，她们就不会认真对待你。

一个成功的筛选流程看起来应如下所示：
**筛选问题 -> 她回应 -> 根据回应探索话题 -> 认同她 -> 推开她 -> 开启下一个话题 -> 提出下一个筛选问题**

这里涉及一个概念：**推开**。过多的认同可能会让她感到不适，降低她认为你具有挑战性的感觉，或让你重新落入“她在筛选你”的框架，又得回到吸引阶段。而推开就是停止认同她，改变你正在对她显露兴趣的谈话框架。

推开应以一种轻微开玩笑的方式进行，绝不应使用打压的方式。尽管推开时可能包含一些无兴趣指标的肢体语言，例如稍稍把身体转开。大多数时候，推开是“很可惜你是……”后面跟一些孩子气的戏弄。

**例如：**
*   `很可惜你太笨了。`
*   `很可惜我们年龄差太远了。（如果年龄接近则可用）`
*   `很可惜你是某某星座的。`
*   `很可惜我并不喜欢你。（此时要坏笑并以开玩笑的口吻说）`

当她已经足够努力赢得你的兴趣时，让她知道这个事实尤为重要。给她一个全面的认同，而不是基于她个性某个方面的认同，或者直接告诉她你对她感兴趣或好奇。

在这个全面的认同中，要保持你一直以来互动的一致性。例如，如果之前你曾戏弄过她，你可以用如下全面认同：
`你知道吗，当我第一次遇见你的时候，我不确定你是否是一个我感兴趣的人。但现在我们之间已经有很多共同点了，这真的太棒了。`
或者简单地说一句：`你太令我惊讶了。`

现在，你们可以进阶到舒适感的阶段了。
