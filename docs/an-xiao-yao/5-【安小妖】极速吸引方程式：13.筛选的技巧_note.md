# 5-【安小妖】极速吸引方程式：13.筛选的技巧 🎯

## 概述
在本节课中，我们将学习如何运用筛选技巧来吸引女性。筛选台词可以是鼓励或帮助女性认识到自身良好品质的话语，类似于音频面试。

## 筛选台词的技巧

### 1. 温和的筛选台词
*   **定义**：温和的筛选台词应该是微妙的，难以察觉的，使整个程序自然流畅。
*   **目的**：让女性展现吸引人的特质，并感受到她在努力赢得你的兴趣。
*   **方法**：
    *   提出话题，如旅行、职业、教育等，引导女性描述自己的个性或吸引你的理由。
    *   结合提问和陈述的方式，例如：“你喜欢旅行吗？你最喜欢哪里？”或“我很喜欢旅行，你呢？”
    *   使用过滤提问或引导性问题，让她展现积极的特质。
    *   事先想好后续回应，以便发现共同点。

### 2. 生硬的筛选台词
*   **定义**：生硬的筛选台词可能会冒失和尴尬，但有时也有效。
*   **方法**：
    *   直接提问，例如：“你认为你有什么优点？”或“为什么我应该要你的电话号码？”
    *   根据女性的反应，调整筛选方式。

### 3. 推开
*   **定义**：推开是指停止认同她，改变谈话框架，让她感受到挑战性。
*   **方法**：
    *   以轻微的开玩笑方式，例如：“很可惜你太笨了”或“很可惜我们年龄差太远了”。
    *   使用无兴趣指标的肢体语言，如稍微转开身体。
    *   保持一致性，例如在之前戏弄过她，可以用全面认同的方式。

### 4. 全面认同
*   **定义**：在全面认同中，要让她知道你已经足够努力去了解她，并欣赏她的特质。
*   **方法**：
    *   保持一致性，例如：“当我第一次遇见你的时候，我不确定你是否是一个我感兴趣的人。但现在我们之间已经有很多共同点了，这真的太棒了！”
    *   直接告诉她你对她感兴趣或好奇。

## 总结
本节课中，我们学习了如何运用筛选技巧来吸引女性。通过温和和生硬的筛选台词，以及推开和全面认同，我们可以更好地了解女性，并建立更深入的联系。记住，真诚和一致性是关键。
