# 安小妖团队《聊天宝典》：【基础篇】如何打造一个好的展示面 📚

## 概述
在本节课中，我们将学习如何打造一个吸引人的展示面，特别是如何在朋友圈中展示自己，以吸引女生的注意。

## 朋友圈内容
以下是朋友圈内容的关键点：

### 1. 照片类型
- **领袖类照片**：与朋友合影，展示社交地位和认可。
- **工作类照片**：展示热爱工作、有潜力的形象。
  - 学生：奖学金、学生会主席、课题研究。
  - 工作/创业：演讲、办公环境、专业成就（奖状、年会照片）。
- **生活类照片**：展示丰富多彩的业余生活，如咖啡厅、旅行、健身。
- **感悟类照片**：展示对个人有积极影响的物品、路线或人。

### 2. 朋友圈内容原则
- **避免负面情绪**：不抱怨加班、不指责他人。
- **内容质量**：明确主题，言之有物。
- **简洁配文**：传递最重要信息，避免啰嗦。
- **自然风格**：不造作，不无病呻吟。
- **适度修图**：美化但不过度，尊重他人。

### 3. 展示状态
- **我很忙但很好**：展示忙碌但积极的生活。
- **我很强且有用**：展示专业能力和成就。
- **我很有趣**：展示个性化和有趣的一面。

## 基础朋友圈及格标准
以下是基础朋友圈建设标准：

1. **发布频率**：每隔三天左右发布一条朋友圈。
2. **照片比例**：每两到三条朋友圈中有一张带自己形象的照片。
3. **照片数量**：一条朋友圈的照片尽量不要超过四张。
4. **照片类型**：避免大量自拍照和裸露肌肉照。
5. **修图**：不要不修图也不要过度修图。

## 总结
本节课中我们一起学习了如何打造一个吸引人的朋友圈，通过展示自己的多面性和积极形象，吸引女生的注意。希望这些技巧能够帮助你提升自己的展示面，吸引更多关注。
