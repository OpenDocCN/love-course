# 1-260节聊天课：92.说错话也可以让妹子更喜欢_note

在本节课中，我们将要学习“推拉”技巧中的一个重要模型——“好警察与坏警察”。这个模型的核心在于通过交替展现热情与冷漠的态度，来激发女生的情感波动和追逐欲。

我们都会有这样的遭遇，被一些很优质的女孩拒绝或不理会。你分享的话语，她们完全不在意。奇怪的是，你越有这样的遭遇，就越期望得到她，越渴望追逐她，而不是果断地远离她。

因为在她刚认识你时，曾对你表示过一定程度的兴趣。这种兴趣在你的心里建立了一种潜意识：她可能会接受你。于是，你开始非常渴望她能接受你，并开始不断地追逐。

你开始担心自己哪里做得不好，变得小心翼翼，甚至有些可怜。这就是今天要说的技巧“好警察和坏警察”所要解决的问题。你有时对她表现冷漠，有时又对她很热情。这和所有的推拉技巧原理相同。

你冷漠之后，必须表现热情。热情之后，又需要表现冷漠。女生经常会对男生做评价，例如：“你挺无聊的，但是人很好。”如果一个男人听了这样的话，他的反应会是不断向女人证明自己不只是人好。

他会努力证明自己具备她所说的其他品质，努力获得她的认可。那么，这种技巧我们怎么来用呢？

## 技巧一：冷读

以下是使用“冷读”技巧的步骤：

1.  **陈述一个观察**：你可以说：“你知道吗，你看起来像是个很热情的人。”
2.  **提出一个相反的印象**：接着说：“但事实上，很多时候别人会觉得其实你挺难接近的，甚至觉得你可能是不愿意相信任何人、比较封闭的人。”
3.  **表达个人看法并转折**：然后说：“但我个人觉得，其实你并不是一个很外向的人。我也不太喜欢这样性格的女生。”
4.  **进行“拉”的操作**：最后给出一个积极的例外：“不过你的笑容倒是很治愈，所以我还是想能约你一起出去玩。我甚至觉得你会不会是我评价标准中的一个例外。”

当你使用这样的冷读时，女生就会想要纠正你的认知。她觉得哪些地方不妥，就会来证明她自己，从而获得让她满意的你的认可。当她们这么做时，就已经在情感层面开始追逐你了。

## 技巧二：故意推翻

上一节我们介绍了通过冷读引发对方证明欲的方法，本节中我们来看看另一个推拉技巧——“故意推翻”。这个技巧显得在逻辑上有些跳跃，即你先做某件事或说某句话，但之后又用另一件事或话完全推翻前者。

运用这种技巧需要你很好地控制表达。你要能在某个时刻表现得非常真诚，在另一个时刻又突然表现得很“虚伪”，实现情绪上的快速转换。

以下是“故意推翻”的一个应用示例：推翻称赞。

1.  **真诚地赞美（拉）**：首先，非常真诚地去赞美女生，尤其是赞美她努力让自己被认可的事情。在真诚称赞后，甚至可以适当夸张。
    *   **示例代码/公式**：`赞美 = 真诚 + 具体细节 (+ 适当夸张)`
2.  **话锋一转进行推翻（推）**：通常“拉”是表现她在某方面很特别、是给你最佳体验的唯一存在。这时，你可以突然表现出迟疑，然后说：
    *   **示例话语**：“不过呢，其实你给我的感觉，跟几年前那个让我最难以忘记的女孩的感觉一样的深刻。在这几年中，除了她之外，再给我这样感觉的就是你了。”

这就是一种“推拉”，把之前建立的她的“唯一性”部分抹杀掉，制造情感上的起伏。
