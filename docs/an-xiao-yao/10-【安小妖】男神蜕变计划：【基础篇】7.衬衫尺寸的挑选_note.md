# 10-【安小妖】男神蜕变计划：【基础篇】7.衬衫尺寸的挑选

## 概述
在本节课中我们将要学习如何挑选合适的衬衫尺寸，包括领围、袖长、版型以及颜色等方面，帮助男士们找到最适合自己的衬衫。

## 选择衬衫尺寸的要点

### 领围选择
- **领围测量**：衬衫领围从38厘米到43厘米不等，购买时告诉营业员你的领围是多少，就可以挑选到尺寸合适的衬衫。
- **塞入手指验证**：在穿着时领围处，可以塞进你的一个手指，也可以验证衬衫是否合身。
- **领高**：衬衫领的高度应该高于西装领的1.5至2厘米。

### 袖长选择
- **袖长验证**：衬衫袖与衬衫顶的验证方法类似，也应该是比西装袖长出1厘米左右。

### 版型选择
- **体型差异**：偏胖或偏瘦的人一般适合买领较为大，但异常合适的衬衫。
- **颜色选择**：颜色上最好选择深色系或者冷色系购买。
- **图案选择**：如果想选择条纹衬衫最好是竖条纹走向的。

### 肤色差异
- **肤色较暗**：肤色较暗的男士可以选择一些比较鲜亮的色彩。
- **肤色较白皙**：皮肤较白皙的男士则可以依照环境的不同，进行良好的自我选择。

### 脸型差异
- **标准衬衫**：标准衬衫适用于各种脸型。
- **脸型窄而长**：脸型窄而长的男士比较适合宽角领。
- **脸型较瘦**：脸型较瘦 颧骨突出的男士非常适合尖领。

## 总结
本节课中我们一起学习了如何挑选合适的衬衫尺寸，包括领围、袖长、版型以及颜色等方面，希望每位男士都能找到最适合自己的衬衫。
