# 1-260节聊天课：141. 学会这招，再也不担心约会冷场尴尬

大家好，我是夏洛，这里是撩美聊天大师的第142节课。我发现现在越来越多的男生把聊天当成一个大坑，一提起聊天就会变得小心翼翼，生怕自己出现什么问题，说错的话。其实，到底该说什么样的话，没有标准答案，不同的情况有不同的选择。而我今天要教给你的，就是让你有自己的一个框架，让你自己心里有这样一个概念：在什么阶段，该用什么样的感觉去说话。

### 什么是不同情况下用不同的方式说话？

比如，同样的问题，你可以把你和女生的关系分为吸引力阶段和舒适感阶段。在吸引力阶段，你就要表现出对女生的不感兴趣，适当的去推开她，从而让她觉得你是一个自信的，有自己标准和框架的男人。而如果是在舒适感阶段，那么就要表现出你对于女生的热情。

### 举例说明

- **女孩问你，你是真的喜欢我吗**？

  - **吸引力阶段**：我喜欢你，不过不要想歪了，只是作为一个普通朋友那样的喜欢。

  - **舒适感阶段**：你在我心中，永远是我遇见你的那个下午，公园里的那只粉色的气球，看到你就会让我想起那天的美好。

- **女孩跟你说，我有男朋友了**？

  - **吸引力阶段**：那很好啊，那他是不是很有钱啊，如果是的话，那下次我们两个吃饭就一定要叫他一起吃。也就是说，要去表现出一种不care，一种轻松，一种幽默的感觉。

  - **舒适感阶段**：我知道你有男朋友，可是我还是无可救药地爱上了你，并且我依然会用我们都能够接受的方式去呵护你，照顾你，关心你。

### 总结

这就是今天我想要教给大家的，在不同的情况下，用不同的感觉来讲话。以后，就不需要再为自己不知道该说什么样的话而感到担心和忧虑了。
