## 场景分析与解读

**场景一：邀约晚餐**
你邀请女生：“明天晚上下班一起吃个饭呗。”
女生回答：“我下班要晚上七八点，会不会太晚了？”

*   **错误解读**：你认为女生只是陈述事实，并乐观地表示愿意调整：“没关系，我去你下班的地方接你。”

**场景二：约会后询问**
约会后你问女生：“你接下来有什么事情吗？”
女生回答：“我打算去逛逛街，你有事情要办吗？”

*   **潜在困惑**：她是希望你陪她，还是暗示你该离开了？

## 核心原理：肯定回答即真实意图

为什么会出现上述误解？关键在于提问方式本身包含了提问者的倾向。

所有提问的**肯定回答**，通常反映了提问者内心的**真实期望**。这是一种普遍的语言习惯。

**公式：提问者的真实意图 ≈ 其问题的肯定回答**

让我们用代码逻辑来理解这个核心概念：

```python
# 女生提问：“我下班要晚上七八点，会不会太晚了？”
# 这个问题的“肯定回答”是：“是的，太晚了。”
if question == "会不会太晚了？" and affirmative_answer == "是的，太晚了":
    her_real_intention = "拒绝邀约，或希望改期"
    # 她已将拒绝的意愿藏在问题里

# 女生提问：“我下班要晚上七八点，对你不会太晚吧？”
# 这个问题的“肯定回答”是：“是的，不晚。”
elif question == "对你不会太晚吧？" and affirmative_answer == "是的，不晚":
    her_real_intention = "愿意赴约，但顾及你的时间"
    # 她已将接受的意愿藏在问题里
```

> 任何人提问时，都会下意识地将自己期望的答案嵌入问题中。当对方给出肯定回答时，便符合了提问者的心意。
