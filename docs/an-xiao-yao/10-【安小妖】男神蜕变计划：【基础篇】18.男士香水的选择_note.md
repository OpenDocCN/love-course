# 10-【安小妖】男神蜕变计划：【基础篇】18.男士香水的选择

## 概述
在本节课中，我们将学习如何选择男士香水，了解不同类型的香水及其使用方法。

## 男士香水类型
以下是男士香水的几种类型：

### 古龙水
**公式**：古龙水含有4%到8%的精油。
- 特点：淡香，以柑橘类清香为主，适合约会使用。

### 淡香水
**公式**：淡香水香精含量7%到12%，持续时间3到4小时。
- 特点：最常见和广泛使用的香水，适合日常使用。

### 传统男人香
**公式**：香根草、烟草、皮革等香气。
- 特点：粗犷、古典，适合成熟男性。

## 男士香水使用方法
以下是男士香水使用的一些技巧：

### 使用位置
- 脉搏跳动的位置，如手腕、脚踝、膝后、脖子、耳后、手肘内侧等。

### 使用禁忌
1. 抽烟者不宜使用香水。
2. 香水不宜过浓过多。
3. 避免使用女士香水。
4. 不宜光脚穿凉鞋。
5. 避免使用过多发油或发胶。
6. 保持香水原味，避免使用补香。
