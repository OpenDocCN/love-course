# 6、【安小妖】恋爱达人速成班：魅力工程-第20课-表白被拒你该怎么办

## 概述
在本节课中，我们将学习如何应对表白被拒的情况，以及如何通过正确的方法挽回或继续发展关系。

## 表白被拒的原因
**以下是表白失败的主要原因：**
1. **心态问题**：长期相处时心态不佳，节奏不对。
2. **缺乏攻势**：过分在乎，显得犹豫不决，缺乏自信。
3. **逃避挫败感**：不主动，面对感情时犹豫不决。

## 表白被拒后的应对方法
**以下是针对不同情况的应对方法：**
1. **直接拉黑**：暂时放弃，等待机会重新加回微信。
2. **冷淡拒绝**：迅速抽离，将关系定位为朋友，提升自己。
3. **委婉拒绝**：以雷霆手段迅速升级关系，调动情绪。
4. **热情回应**：提升自己，建立框架，学会拒绝。

## 挽回关系的方法
**以下是挽回关系的步骤：**
1. **抽离止损**：降低防备心，停止对感情的投资。
2. **二次吸引**：变得更好，用新的自己去吸引对方。
3. **保持平常心**：不要把表白当成严重的事情。
4. **创造机会**：在自我提升后，创造见面机会。
5. **建立吸引**：学会诱惑对方，抓住机会打破舒适区。
6. **做好预选**：让对方感觉到你有其他选择。

## 表白先决条件
**以下是建立表白先决条件的要点：**
1. **吸引**：增强吸引力，让对方倒追你。
2. **关系暗示**：通过关系暗示，成为男女朋友。
3. **肢体接触**：不经意地进行肢体接触。
4. **社交压迫**：利用社交压迫，降低表白所需的吸引力。
5. **制造预期**：暗示要表白，但不立即表白，提高成功率。

## 总结
本节课中我们一起学习了如何应对表白被拒的情况，以及如何通过正确的方法挽回或继续发展关系。希望这些方法能帮助你更好地处理感情问题。
