# 1-260节聊天课：119.调动起女生情绪的聊天才是好的聊天_note

在本节课中，我们将要学习如何通过聊天有效调动女生的情绪。这是建立吸引力的核心，也是让对话变得有趣、深入的关键。

很多男生在与女生聊天时，感觉对话氛围不佳。虽然开启了许多话题，但聊天始终无法热烈起来。这通常是因为没有把握住聊天的关键——调动情绪。

调动情绪是吸引中非常重要的元素。情绪调动充分，女生就会对你有感觉。情绪调动不足，对女生的吸引力就会下降，女生回复你的频率也会降低。在两性关系中，所谓的吸引，本质就是给女人造成的情绪波动。女人对你的情绪越平淡，你对她就越没有吸引力。

以下是几种调动女生情绪的具体方法。

## 一、使用亲密称呼 🏷️

在对话中加入亲密的“昵称”，可以将平淡的对话瞬间变得暧昧。

例如，当女生向你抱怨不开心或身体不适时，你可以发一个“抱抱”的表情，然后说：“哎哟，小可怜，安慰你一下。” 或者在开始聊天时说：“在干嘛呢，小仙女？” 这比单纯问“在干嘛”更有趣，更能吸引女生回复。

## 二、运用冷读术 🔮

上一节我们介绍了通过称呼拉近距离，本节中我们来看看如何运用冷读术引起好奇。冷读术指在对方没有防备或充分了解时，看透其心思，从而引发好奇。

例如，将普通问句“你是不是在国外留过学？”转换为冷读陈述句：“你一定有在国外留过学吧。” 再如，将“你觉得自己的性格怎么样？”转换为：“如果我没看错的话，你的人缘一定不错吧？”

普通问句可能只得到“是”或“不是”的简单回答。而冷读式的陈述句，则可能引发对方多种反应，如：“你是怎么知道的？”或“没有，但为什么你会这么说？” 冷读必须使用陈述句，它是一种猜测。既然是猜测就可能不准确，但这并不意味着失败。冷读的目的并非完全猜中信息，而是通过这种方式让对方敞开心扉，主动提供更多信息。

## 三、调侃与幽默 😄

幽默是人际交往中不可缺少的元素。擅长幽默和调侃的男生通常更容易打动女生。

在微信聊天中，表情是传达情绪的重要因素。同样的文字搭配不同的表情，可以产生截然不同的效果。
