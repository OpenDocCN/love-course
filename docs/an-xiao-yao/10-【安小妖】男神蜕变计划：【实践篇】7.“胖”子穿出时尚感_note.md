# 10-【安小妖】男神蜕变计划：【实践篇】7. “胖”子穿出时尚感

## 概述
在本节课中，我们将学习如何让胖壮男生通过服装搭配展现出时尚感。

## 胖壮身材搭配原则
胖壮男生的身材比较宽，因此在搭配时，可以在黑白灰基础上呈现出倒三角，以实现视觉平衡。

### 倒三角原则
- **公式**：倒三角原则 = 宽松上衣 + 窄裤
- **代码**：`top: loose; bottom: narrow`

## 避免显胖的搭配禁忌
1. **发型**：避免长发，选择短发以保持身体比例协调。
2. **材质**：选择不透光的材质，避免太薄或太透的单品。
3. **尺寸**：选择合适的尺寸，避免太松或太紧。

## 搭配建议
1. **宽松基本款**：选择宽松的T恤、亚麻裤或牛仔短裤。
2. **双色系黑白灰**：以黑白灰为基础，搭配双色系。
3. **亚麻材质**：选择亚麻面料的衣裤，营造日系感。
4. **极简短袖衬衫**：搭配休闲裤，营造成熟感。
5. **竖条纹衬衫**：搭配宽松休闲裤或西装裤，达到成熟时尚风。
6. **直筒牛仔裤**：搭配暗灰色系T恤，达到倒三角视觉效果。
7. **宽松阔腿裤**：搭配宽松T恤，营造硬汉感觉。

## 总结
本节课中，我们学习了胖壮男生如何通过服装搭配展现出时尚感。通过选择合适的颜色、材质和款式，以及遵循倒三角原则，胖壮男生也能穿出时尚感。
