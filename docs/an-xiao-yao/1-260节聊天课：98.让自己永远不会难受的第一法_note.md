# 1-260节聊天课：98.让自己永远不会难受的第一法_note

在本节课中，我们将探讨一个核心原则：如何通过明确自我需求和坚定选择，来掌控自己的情感生活，从而避免陷入难受的境地。关键在于下定决心，追求自己想要的生活。

---

生活由一连串选择构成。许多人并不清楚自己真正想要什么，只是随遇而安，被动接受遇到的一切。这就像你是一个单词，是选择成为明确的 **“yes”**，还是模糊的 **“maybe”**。大多数人都是“maybe”，这使得他们难以顺畅地打理自己的感情生活。

今天过得不错，并不保证明天也能如此。当我曾作为一个很花心的男人时，我很享受那种生活方式。但后来，我的态度和价值取向改变了，我开始追求其他目标，例如沉心静气地钻研某门知识。于是，我选择停止先前的生活方式。这一切都在我的掌控之中，因为我爱自己。

有时我会这样描述自己：我有很好的幽默感，态度非常明确，我很聪明，很迷人，在其他领域也做得非常出色。但此刻我不想讨论这些。当需要向女性传达某些事情时，最好的方法是运用幽默。幽默能展现出你拥有良好的态度、聪明和迷人特质，这并非自以为是。

你为什么不给自己一些积极的评价呢？面试工作时，你不会说：“我是个无赖，很不诚实，很懒惰，经常把事情搞砸。” 你绝不会这么说。我从未刻意对任何人造成主观上的伤害，我不需要那样做。我一直用我的诚实、信任和尊重来与人相处交往，我认为这超级有用。

你的能力会带给你自信。许多人试图用各种技巧来增加自信，但事实上，除非你同时学会了新本领、具备了足够的能力，否则这些技巧根本没什么用。

为了获得真正的信心，我要给你两条规则。

以下是两条核心规则：

1.  **谁在乎女生在想什么？你才是一段关系中最重要的人。** 如果你不在乎女人在想什么，你的恐惧就会消失。如果你真正相信自己是最重要的那个人，你就会开始对自己的生活负责。这条规则能使你变得强壮、有决断力且诚实。你需要下定决心做好自己。不在乎女人说什么，并不代表你粗鲁；而是由你自己决定想要什么，然后去努力，这对你才最有意义。
2.  **回到尊重问题上。** 我非常尊重我的女人。如果我把手放在她的腰上，她说“不”，我会立刻把手拿开。如果她不想我那样做，完全可以。如果她说不想和我在一起，我就不会继续与她交往。离开是世界上最容易的事，我只需动动脚就能做到。

当你有一天学会了真实地对待自己，你就永远不需要再读任何关于“如何吸引好女孩”的书了。

在与女人相处时，她们有时会抱怨：“为什么总要按你说的做？” 这时我会回答：“我很高兴你记得我的规则。这是我生命中的一个方向。我不在乎女人说什么，我知道我想要的生活是什么，我比别人更了解自己。我的生活里没有民主。” 总之，你必须控制与女人交往的每一个过程和每一句话。
