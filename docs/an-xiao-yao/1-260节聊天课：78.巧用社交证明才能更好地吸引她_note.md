# 1-260节聊天课：78.巧用社交证明才能更好地吸引她

### 社交证明的吸引力

你是否注意到，生活中有一种现象，就是当你和朋友一起出去玩时，你可能会不由自主地看向朋友注视的方向。这是因为环境中的社交信号激发了你的反应，这就是社交证明的一个典型案例。

**公式**：社交证明 = 社交信号 + 反应

情绪程序被设计来回应社交信号，越多人倾听、关注你，与你达到情绪共鸣，你就获得更多的社交证明，这是一个滚雪球的过程。

### 产生社交证明的方法

1. **群体理论**：赋予我们进入新的社交互动的力量，并战胜整个群体，随心所欲地创造社交证明。
2. **带随从**：带朋友或伴侣出现，通过聊天、大笑、跳舞、游戏、拍照等方式创造社交证明。
3. **时尚打扮**：精心设计的开场白、时尚的打扮、得体的穿着等，都能在社交场合中吸引更多关注，传递出更多的高价值。

### 半孔雀行为

半孔雀行为是指利用自己的吸引力，在社交压力下展现行为的价值。这种行为能够强化所有人对你的反应，一个能掌控全局的男生比一个普通男生更具吸引力。

### 吸引力的传递

通过个人魅力刺激对方的情绪，引起共同的笑声，情绪共振，把自己打造成一团火焰，吸引更多人进入你的空间，通过故事传递你的价值，获得认可和社交证明。

### 注意力与价值

注意力总是与价值相关，刺激行动、光亮、声音、价值，人们自然会对接收到的信息产生兴趣。

### 避免过度追求关注

为了博得关注而忽略自己展示的价值，会降低你的价值。要自然地获得关注，而不是过分追求。

### 总结

本节课中我们一起学习了社交证明的吸引力、产生社交证明的方法、半孔雀行为以及如何通过个人魅力吸引他人。希望这些内容能帮助你更好地吸引他人。
