# 安小妖团队《聊天宝典》课程三：【社交软件篇】转换平台 📱➡️💬

在本节课中，我们将要学习如何将社交软件上的互动关系，平稳地转移到更常用的沟通平台——微信上。这是提升沟通效率、深化关系的关键一步。

---

社交软件功能再强大，通常也不是女生最常用的核心沟通工具。因此，在社交软件上建立初步联系后，必须进行的一步操作是**转换平台**，即将女生引导至微信。

很多男生会使用一个直接的方法：简单互动几个回合后，就对女生说“这个软件我不常上，加个微信吧，我的微信号是XXX”。客观来说，这个方法有一定成功率，但成功率不高。那些毫不犹豫就同意加微信的女生，多数可能是推销人员，只有极少数女生可能因为无聊且觉得你的展示面不错而同意转移。大部分情况是，女生看到你经常在线，你却说不常上，她可能会下意识地拒绝你。因此，建议少用这个方法。

在尝试与女生交换微信之前，需要满足几个前提条件：
1.  你的个人展示面是及格的。
2.  你保持了无需求感，双方的互动是良性的。
3.  你们之间已经进行了10句以上的有效聊天。

即使你的展示面合格，女生也可能不给你微信。这就像即使你是一个帅哥，在街上随便找一位女生说“美女，加个微信呗”，对方大概率也会拒绝。所以，方法至关重要。

---

上一节我们介绍了转换平台的必要性和前提条件，本节中我们来看看两个具体、有效的方法。

**第一个方法是兴趣爱好转移法。**
我们可以根据对方的兴趣爱好和生活习惯找到合适的话题，再通过这个话题自然地引导至加微信。

以下是具体操作示例：
*   **示例1：** 如果女生展示出明确的兴趣爱好（比如游戏），你可以说“看你挺厉害的，来一把？”，以此为借口自然地加到微信。
*   **示例2：** 如果她喜欢看电影，你可以先和她聊聊近期上映的电影。当她愿意深入交谈后，你可以说：“我现在有点事要忙，留个微信吧，待会再聊。” 这种方式基本不会被拒绝。

**第二个方法是情绪法。**
此方法的核心是先通过一个话题调动起对方的情绪，然后顺着这个情绪自然地转移到微信上。

以下是具体操作示例：
*   **案例：** 女生的ID名字比较酷。男生可以先从她的ID开启话题，用特别的方式激起她的兴趣。接着，可以顺着聊到女生的职业，然后故意说“工作不是生活的全部，恋爱还是要谈的”，从而顺利加到微信。

---

其实在社交软件上，你匹配的每一个女生，几乎都有机会加到微信，除非你在这个过程中犯了错误，或者将对话拖延得太久。要抓住一切机会去转移平台。如果一个女生始终不愿意，你只需要反思自己的聊天方式是否存在问题。如果没有太大问题，就不要在一个看不到希望的事情上耽误太多时间。毕竟，我们在社交软件篇第一节课就强调过，社交软件的特质是**高效**。
