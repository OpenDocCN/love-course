# 13-安小妖《好男人，你也要学会撩妹》：151-200：173.框架如何维持？我来教你

## 教程

### 概述

在本节课中，我们将学习如何维持框架，以及如何通过自身建设来提升自己的吸引力。

### 框架的本质

框架是指在与女生相处的过程中，保持自己的态度和安排，不被女生带节奏，引导女生顺从自己的态度。

**公式**：框架 = 自身建设 + 外形着装 + 生活细节 + 自我认同

### 提升自己的好习惯

以下是几个提升自己的好习惯：

1. **养成嫉妒自己生活的习惯**：经常复盘自己的生活，设定目标，量化生活，让自己随时间增值。
2. **多表达**：遇到困难时，及时与朋友、父母沟通，寻求关怀和温暖。
3. **及时执行自己的想法**：想到什么就马上做，避免拖延，保持高效。

### 总结

本节课中我们一起学习了如何维持框架，以及如何通过自身建设来提升自己的吸引力。希望这些方法能帮助你过上更加高大上、体面的生活。
