# 13-安小妖《好男人，你也要学会撩妹》：101-200：124.如何判断女生对你的兴趣

## 概述
在本节课中，我们将学习如何通过女生的行为来判断她对你是否有兴趣。

## 判断女生兴趣的指标

### 1. 难以见面
- **公式**：女生难以与你见面 → 对你没有兴趣
- **描述**：即使你是她的同学或同事，她也会尽量避免与你见面。

### 2. 缩短交流时间
- **公式**：女生缩短与你交流的时间 → 对你没有兴趣
- **描述**：在电话、微信或QQ上，女生只回答是或否，不分享生活，不表示关心。

### 3. 约不出来
- **公式**：多次约会都推脱 → 对你没有兴趣
- **描述**：无论活动多么有趣，她总是以忙碌为借口推脱。

### 4. 约会时带朋友
- **公式**：约会时总带朋友 → 对你没有兴趣
- **描述**：她带着朋友或让你带朋友，以减少暧昧机会。

### 5. 约会后失联
- **公式**：约会后突然失联 → 对你没有兴趣
- **描述**：她可能觉得你不够优秀，或对你没有感觉。

### 6. 不允许暧昧行为
- **公式**：不允许暧昧行为 → 对你没有兴趣
- **描述**：她不给你任何暧昧的机会，如牵手、拥抱等。

### 7. 保持安全距离
- **公式**：保持安全距离 → 对你没有兴趣
- **描述**：她总是与你保持一定的距离，不让你靠近。

## 总结
本节课中，我们一起学习了如何通过女生的行为来判断她对你是否有兴趣。希望这些知识能帮助你更好地了解女生的心思。
