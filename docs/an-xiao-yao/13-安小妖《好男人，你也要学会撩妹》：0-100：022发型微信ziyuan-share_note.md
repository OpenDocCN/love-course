## 教程：如何选择合适的发型

### 概述
在本节课中，我们将学习如何根据脸型和发质选择合适的发型，提升个人形象。

### 脸型测量
以下是测量脸型所需的数据：
1. **额头高度**：测量眉峰到发际线的距离。
2. **颧骨宽度**：测量两个外眼角之间的距离。
3. **脸长**：测量发际线中心和下巴肩部之间的距离。
4. **下巴长度**：测量下巴肩部到下唇中间的距离。

### 常见脸型及发型推荐
以下是几种常见脸型及其推荐的发型：

**1. 椭圆形脸**
- 特点：脸长大于颧骨宽度，额头大于下巴，脸角圆形。
- 推荐：经典短发，两边剪短，顶部头发稍长。

**2. 方形脸**
- 特点：颌骨和下巴比例完美，轮廓清晰。
- 推荐：经典短发或长发型，保持渐变长度，可留短胡茬。

**3. 长方形脸**
- 特点：脸长大于颧骨宽度，脸型较长。
- 推荐：避免两边头发剪得太短，保持一定长度，可留短胡茬。

**4. 圆形脸**
- 特点：额头和下巴较大，脸角圆润。
- 推荐：平顶造型或方形胡须，增加下巴线条。

**5. 棱角形脸（钻石脸）**
- 特点：下巴、眉骨、两颊有清晰痕迹。
- 推荐：增加额头和下巴宽度，使用条纹刘海，保留耳后头发。

**6. 心形脸**
- 特点：额头到下巴逐渐缩小。
- 推荐：避免剪短发型，选择中等长度，可留胡须。

### 发质问题
- **硬质发质**：使用护发素，使头发柔顺。
- **软质发质**：使用定型发胶，增加发型精致度。

### 鬓角类型
- **锥形鬓角**：适合方形脸或三角脸，修剪成尖三角形。
- **直鬓角**：垂直流出的直线型，可修剪成90度直角。
- **细直角**：适合毛发硬的男生，显得斯文优雅。

### 发型推荐
- **平刘海**：阳光、干净、利落，对女生产生吸引力。
- **三七分发**：显示头部立体感，可随时改变造型。
- **斜旁克**：适合心形脸，自然立体，刘海遮过眉毛。
- **中分**：复古帅气，阳刚之气，清爽搭配。
- **undercut**：简洁易打理，凸显男性气质。
