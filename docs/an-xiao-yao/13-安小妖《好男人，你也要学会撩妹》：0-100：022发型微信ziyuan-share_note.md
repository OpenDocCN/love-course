# 课程13：男士发型指南 💇‍♂️

在本节课中，我们将学习如何根据脸型选择适合自己的发型。通过测量几个关键数据，你可以判断自己的脸型，并了解不同脸型最适合的发型风格。此外，我们还将探讨发质护理、鬓角修剪以及一些经典发型的名称与特点。

## 测量脸型数据 📏

上一节我们介绍了课程目标，本节中我们来看看如何确定自己的脸型。首先，你需要测量四个关键数据。

以下是测量步骤：

1.  测量额头高度：测量眉峰（眉毛最高点）到发际线的垂直距离。
2.  测量颧骨宽度：测量两个外眼角之间的水平距离。
3.  测量脸长：测量发际线中心点到下巴尖端的垂直距离。
4.  测量下巴长度：测量下巴尖端到下唇中心的垂直距离。

记录下这四个数据：**额头**、**颧骨**、**脸长**、**下巴**。

## 常见脸型与发型建议 👨

现在我们已经获得了脸型数据，接下来根据这些数据判断脸型，并了解相应的发型建议。

以下是六种常见脸型及其特点：

*   **椭圆形脸**
    *   **特点**：脸长 > 颧骨宽；额头 > 下巴；下颌线条圆润。
    *   **发型建议**：这是较完美的脸型，适合大多数发型。最推荐经典短发，两侧剪短，顶部留长并打造蓬松感。搭配短胡茬效果更佳。

*   **方形脸**
    *   **特点**：颌骨和下巴轮廓清晰，面部比例理想。
    *   **发型建议**：适合多种短发或略长发。推荐经典利落的发型，侧面采用渐变长度以增加层次感。下巴处可留短胡茬。

*   **长方形脸**
    *   **特点**：脸长是四个数据中最大的。
    *   **发型建议**：避免两侧头发过短，顶部需保持一定长度。可尝试将顶部头发偏向一侧。需经常修剪胡子，避免留长胡子。

*   **圆形脸**
    *   **特点**：额头和下巴较宽，面部线条圆润，缺乏明显角度。
    *   **发型建议**：需要通过发型创造轮廓感。平顶发型可以突出下巴形状，方形胡子有助于明确下巴轮廓。

*   **棱角形脸（钻石脸）**
    *   **特点**：下巴、眉骨、两颊轮廓清晰，有如钻石切割。
    *   **发型建议**：适合在额头和下巴增加宽度的发型。条纹刘海可增加质感，耳后保留头发以弱化骨骼结构。避免剪短两侧头发，应追求有层次感的发型，如偏分。

*   **心形脸**
    *   **特点**：从额头到下巴逐渐变窄，比例美观。
    *   **发型建议**：避免过短发型，以免突出宽额头和尖下巴。中等长度头发较安全，可遮盖额头或增加顶部发量。适当留胡子可使下巴线条更柔和。

## 发质护理与定型 💆‍♂️

了解脸型与发型搭配后，我们还需要关注发质，因为发质是支撑发型的基础。

以下是针对不同发质的护理建议：

*   **发质硬**：需使用护发素使头发变柔顺。
*   **发质软**：需使用定型发胶来维持发型。想拥有精致造型，发蜡是必需品。

## 鬓角的修剪 ✂️

除了整体发型，鬓角的修剪也至关重要，它直接影响精神面貌。

以下是两种常见的鬓角类型：

*   **锥形鬓角**：鬓角底端修剪成尖三角形。适合发量稀疏或气质斯文的男生，方形脸或三角形脸较适合。
*   **直鬓角**：沿发际线垂直修剪，线条锐利。粗直鬓角显阳刚，细直鬓角显斯文。适合发质硬、希望展现男子气概的男生，锥角型脸较适合。

## 经典发型名称介绍 💇

最后，我们来认识一些经典的发型名称及其特点。

以下是几种流行的男士发型：

*   **平刘海**：如宋仲基造型，显得阳光、干净、利落，对女生有吸引力。
*   **三七分发**：能显示头部立体感，呈现五官。刘海较长，便于用定型产品改变造型。
*   **斜庞克发型**：适合心形脸。刘海稍过眉，顶部蓬松，形成上大下小的视觉效果。
*   **中分**：经典不过时，兼具复古帅气与阳刚之气，搭配清爽穿着更显帅气。
*   **Undercut**：两侧削薄剪短，中间长度适中，干净利落，凸显男性气质，易于打理。

本节课中我们一起学习了如何通过测量判断脸型，并根据不同脸型选择合适的发型。同时，我们也了解了发质护理的重要性、鬓角的修剪技巧，以及几种经典发型的名称与特点。打理好发型是提升个人形象的关键一步。
