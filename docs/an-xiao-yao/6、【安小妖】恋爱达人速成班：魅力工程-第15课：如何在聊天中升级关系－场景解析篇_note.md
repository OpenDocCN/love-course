### 1. 共谋

共谋就是跟他一起做同一件事，有这种事的发生，他会潜意识里，觉得你们是一对情侣。

**公式**： 共谋 = 一起做同一件事

**代码**： `共谋 = 一起行动`

### 2. 现场现实共谋

你和他在相同物理空间上，做同一件事，叫做现场现实共谋。

**公式**： 现场现实共谋 = 相同物理空间 + 一起行动

**代码**： `现场现实共谋 = 相同地点 + 一起行动`

### 3. 非现场共谋

你和对方没有在一起，比如微信上，幻想在一起做同一件事情，叫做非现场共谋。

**公式**： 非现场共谋 = 虚拟空间 + 一起幻想行动

**代码**： `非现场共谋 = 虚拟空间 + 一起幻想行动`

### 4. 条件反射

初中生物课上，我们学过一项，特别著名的生物学和心理学实验，简单的来说，是给一条狗咬铃铛，然后给它喂吃的，持续一段时间。它以后一听到铃铛，就分泌唾液，这叫做巴卜洛夫效应，也叫做条件反射。

**公式**： 条件反射 = 条件刺激 + 反射行为

**代码**： `条件反射 = 条件刺激 + 反射行为`

### 5. 分享秘密

信任是组成舒适感的一个重要方面，要想获取别人的信任，不是说说就可以做到的，这是非常难的。

**公式**： 信任 = 分享秘密 + 保密承诺

**代码**： `信任 = 分享秘密 + 保密承诺`

我们小的时候，没有觉得童年多么快乐，但是长大之后，见了太多残酷肮脏的东西，会觉得小时候，是特别有意思的，并且认为童年的记忆，十分珍贵。

**公式**： 回忆童年 = 共同回忆过去 + 展望未来
