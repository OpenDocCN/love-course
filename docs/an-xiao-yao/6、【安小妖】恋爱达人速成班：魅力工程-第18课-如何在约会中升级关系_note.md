# 6、【安小妖】恋爱达人速成班：魅力工程-第18课-如何在约会中升级关系

## 概述
在本节课中，我们将学习如何在约会中升级关系，避免浪费彼此的时间和精力。

## 关键步骤
**升级关系的步骤应该是：**
1. 建立信任与舒适感
2. 肢体触碰
3. 行为升级
4. 亲密关系
5. 可有可无的表白

## 微信升级的弊端
1. 没有见过真人，不要为陌生人投入太多感情
2. 见面后仍需重新升级
3. 给女人过高预期，可能导致约会推进困难

## 聊天目的
- 约会
- 升级关系

## 升级方法
1. **勇往直前**：克服恐惧，赞美对方，展现自信。
2. **暧昧气氛**：通过提问和观察，了解对方的感受。
3. **行动升级**：通过礼貌性、奖励式和游戏式接触，增加亲密感。

## 升级被拒怎么办
1. **退回原关系**：给出合理的理由，保持无需求的态度。
2. **保持正常关系**：缓解对方的压力，减少焦虑感。
3. **提高价值**：提升自身价值，增强自信。
4. **下一次升级**：更加自信和坚定。

## 触碰她的方法
### 绅士版
1. 拍照时帮她摆姿势
2. 进餐厅时帮她脱外套
3. 为她披上外套
4. 打开车门时扶她
5. 上楼梯时走在她前面
6. 过马路时拉起她的手
7. 挤公交或地铁时离她近一些

### 进阶版
1. 开玩笑时摸摸她的头
2. 把胳膊给她挽
3. 刮一下她的鼻子
4. 看着她的眼睛叹气，弹额头
5. 吃饭时优雅地微笑，指自己的脸
6. 轻轻捏捏她的脸蛋，说“你真可爱”
7. 抚摸她的头发，开玩笑说“你头上好像有一只小虫子”
8. 走到坡地或陡峭的路，先登顶，然后拉她上来

### 情圣版
1. 以西方礼仪亲吻她的手背
2. 把围巾围在她的脖子上
3. 把她的手放在口袋里
4. 趴在她耳边说话，吹气
5. 抚摸她的头发，放在她的脸上，对视几秒，然后放开
