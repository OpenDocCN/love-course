# 13-安小妖《好男人，你也要学会撩妹》：218.巧用多看效应让她习惯有你，从此离不开你

## 概述
在本节课中，我们将学习如何利用多看效应，让心仪的人习惯你的存在，从而更加依赖你。

## 多看效应的定义
**多看效应**，也称为单纯曝光效应或重复曝光效应，指的是我们对某个事物因熟悉而产生的好感。这种现象适用于各种事物，如文字、图画、照片、声音等。

### 公式
\[ \text{好感} = \text{熟悉度} \times \text{曝光次数} \]

## 多看效应的实验证明
心理学家扎荣兹进行了一项实验，让一群人观看某个大学班级的毕业相册。结果显示，出现次数越多的照片，人们对其中的照片人物的评价越高。

## 如何应用多看效应
1. **适度曝光**：避免过度曝光，以免引起厌烦。
2. **保留神秘感**：在初次接触时，不要接触时间过长，以保持神秘感。
3. **曝光次数**：一般而言，曝光次数在10到20次之间时，能够达到最大的喜欢程度。

## 注意事项
1. **避免过度曝光**：过度曝光会引起厌烦。
2. **初次接触**：保持神秘感，避免接触时间过长。
