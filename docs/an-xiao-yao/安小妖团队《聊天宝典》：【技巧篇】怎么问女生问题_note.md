# 安小妖团队《聊天宝典》：【技巧篇】怎么问女生问题_note

在本节课中，我们将要学习在社交软件上与女生聊天时，如何有效地提问和回答问题。我们将重点分析常见的“查户口”式聊天为何不受欢迎，并介绍更高级的沟通技巧，帮助你建立有趣、自然的对话。

## 认识“查户口”式聊天 🚫

上一节我们概述了课程目标，本节中我们来看看需要避免的典型错误——“查户口”式聊天。

“查户口”这种聊天方式，是在两个人刚刚认识的初期联系感阶段，女生最讨厌的一种聊天交流方式。之所以很多男生会不自觉地启动“查户口”的聊天模式，是因为他们和女孩子互相还不熟悉，刚刚认识，甚至谈不上认识。

他们不知道聊什么，同时又很想和这个女孩子交流，所以他们就用一个又一个的问题，来机械地延长聊天的时间。他们以为交流的时间越长越好，但其实“查户口”模式的交流时间越长，女生对你的差评就越多。

女生会觉得越反感越讨厌。很多男生其实也感觉得到，对方的情绪和氛围慢慢地在走低，但是没有办法。

## “查户口”的两种不良后果 📉

了解了“查户口”的定义后，我们来看看它通常引发的两种不良互动模式。

以下是两种常见的负面反应：

1.  **直接表达反感型**：有些女孩子性格更霸气一点，像“女汉子”，有不爽的就直接表达出来。
2.  **委婉回避型**：案例中的另一种女孩子，则会一直退让，觉得跟你实在没什么好聊的了，最后以“我还有点事，我们回头再聊”结束对话。

通常来说，女孩子会选择前面一种做法，但是也有女孩子会选择后面一种做法。我们都知道，这两种都是“查户口”引发的不良结果。

## 正确的提问方式：诱导式提问 🎣

既然直接提问效果不佳，本节中我们来看看正确的做法——诱导式提问。

比如，当你想知道对方是哪里人时，错误的直接提问是：“你是哪里人？”。而一个比较好的回应可以是：“你看起来像南方姑娘。”

这种问法就是一个**诱导式的提问**。问题从来都不会直接提出来，而是采用迂回的办法来提问，或者说直接陈述。因为直接提问容易使对方提高警觉，觉得“我的家庭什么样，关你什么事？”，或者在还不熟悉的情况下，觉得不需要别人过多打探自己的事情。

人们对于别人说错的话，通常有一种纠正的心态。所以，这样给对方设定一个（可能错误的）答案，让对方来更正，他就会把注意力转移到“更正你的错误”上，而不是去警惕“你到底要做什么”。而对方更正的，恰恰是你想要的答案。

**公式：陈述一个猜测（可能错误） -> 引导对方纠正 -> 获得真实信息**

无论你猜对没有，你都能达到自己的目的——开启一个话题并获取信息。这个方法比较好。

## 如何回答女生的提问 💡

当我们自身引起女生兴趣时，她们也会主动提问。本节中我们学习如何有趣地回答这些问题。

比如，女生觉得你长得挺帅，看了你的照片，可能会主动问你的问题。比如：“你是做什么工作的呀？”

其实你会发现，在跟女孩子的互动当中，我们并没有直接给出答案，而是每次都有一个小互动，每次都是一个小的情绪波动。这样效果往往更好。

**示例对话流程：**
> 女：“你是做什么工作的呀？”
> 男：“你猜猜看？”
> 女：“我猜不到，快告诉我。”
> 男：“告诉你可以，但你想听真话还是假话？”
> 女：“我問你你當然是要聽真話了。”
> 男：“我是魔术师吗？等你见到我的时候，我帮你完成一个心愿。”
> 女：“什么心愿啊？”
> 男：“那就要问你了。”

在这个地方你会发现，到最后你也没有直接告诉她你是做什么工作的，但是你跟她的互动充满了情绪的波动，会挺有意思的，同时自然地安排了一个约会，预设了你们要见面。这是一个比较好的回答女性提问的方式。

## 节奏把控：平衡聊天氛围 ⚖️

掌握了提问和回答的技巧后，最后我们需要注意聊天的整体节奏。

当然，我们要注意的就是，你不能一直开启逗逼模式，一直无厘头；也不能一直是很平淡的正常聊天。这两个要交叉进行，是最好的。在有趣和正经之间找到平衡，才能让对话持续吸引人。
