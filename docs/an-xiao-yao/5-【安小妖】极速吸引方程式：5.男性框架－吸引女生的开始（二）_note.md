# 5-【安小妖】极速吸引方程式：5.男性框架－吸引女生的开始（二）

(音乐)，信念体系建立后需要的就是不断去尝试，尝试不只是在原有基础上尝试，想要获得更大的收益就要走出自己的舒适区。

**为什么走出舒适区这么难？**

心理舒适区本质上是熟悉的应对方式。

**什么是心理舒适区？**

心理舒适区并不一定意味着舒适，比如有人在小城市找了一份安逸的工作，工作没有太大的成就但比较舒适，所以就不想再挑战自己。

**心理舒适区的本质**

1. 心理舒适区并不一定意味着舒适。
2. 心理舒适区也不意味着熟悉的环境。

**如何扩大你的舒适区，提高经验值？**

1. **打破你的生活习惯**：克服懒惰，打破日常习惯。
2. **勇于接受挑战**：克服恐惧，接受新的挑战。
3. **做计划**：精确到小时的小计划，以及以十年为目标的大计划。
4. **管理时间**：合理利用时间，避免无效消耗。
5. **执行力**：设定奖励，提高执行力。

**如何确认目标？**

思考一下你想过怎样的人生，想要怎样的婚姻和怎样的爱情。

**如何拆分目标？**

确认目标后，制定实施计划，拆分成月度或周度执行计划。

**如何排除逐步实施的计划，以及完成的节点？**

梳理解决可能遇到的困难，请求帮助，合理调配资源。

**如何管理时间？**

刘比杰夫的时间管理方法：记录、统计、分析、反馈。

**如何提高执行力？**

设定奖励，提高执行力，不断观察和调整执行方案。

**如何激励自己？**

1. 社会激励：与他人比较，激发动力。
2. 及时奖励：设定短期目标，及时奖励自己。
3. 记住监控：记录进步，监控变化。

**总结**

本节课中我们一起学习了如何走出舒适区，扩大舒适区，提高经验值，以及如何管理时间和提高执行力。希望这些方法能帮助你更好地实现目标，提升吸引力。
