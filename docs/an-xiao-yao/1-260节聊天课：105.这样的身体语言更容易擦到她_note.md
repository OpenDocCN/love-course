# 1-260节聊天课：105.这样的身体语言更容易擦到她

## 概述
在本节课中，我们将学习如何通过适当的身体语言来吸引和搭讪陌生女生。

## 核心概念
**身体语言**：比语言更有影响力，能传达你的状态和心理。

### 身体语言注意事项
1. **减少不自然动作**：避免紧张导致的来回跺脚、捏紧拳头或身体晃动等。
2. **身体后倾**：将重心放在离女生更远的脚上，保持视觉距离。
3. **手臂位置**：手臂不要高于胸部，避免显得不自信或侵略性。
4. **眼神交流**：保持一半左右时间的对视。
5. **收号时不要急于拿手机**：等对方答应后再递手机，避免需求感过强。

### 不同情况下的搭讪技巧
1. **女生与朋友在一起**：
   - 首先得到其他女生的认可，表现出对朋友的尊重。
   - 询问朋友是否可以聊几分钟。
2. **要联系方式**：
   - 使用自然、不显得需求感强的用语，如“我可以留下你的联系方式吗？”或“我们可以留个联系方式，下次再好好认识一下”。

### 完整话术
1. **交代搭讪原因**：例如，在等朋友时看到她，觉得她很吸引人。
2. **表达欣赏**：不仅仅是外貌，还可以是她的性格或不自信的地方。
3. **进行简单冷读**：建立联系感，如猜测她的家庭背景或星座。
4. **话题接续**：如果不知道说什么，可以坦白表示紧张。

## 总结
本节课中我们一起学习了如何通过适当的身体语言和搭讪技巧来吸引和搭讪陌生女生。希望这些技巧能帮助你更好地与人交流。
