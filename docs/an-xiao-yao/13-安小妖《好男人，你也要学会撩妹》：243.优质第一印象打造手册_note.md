## 概述
在本节课中，我们将学习如何通过服装搭配提升个人形象，给女生留下良好的第一印象。

## 第一印象的重要性
**公式**：第一印象 = 外形 + 风度 + 精神风貌

女生看到男生的第一印象最重要的是整体的**风度**和**精神风貌**。第一印象对女生来说非常重要。

## 服装搭配的重要性
**公式**：服装搭配 = 颜色搭配 + 剪裁搭配

对于男人来说，观察第一印象时，最重要的是全身的颜色服装搭配，给人的感觉。

## 颜色搭配
**代码**：
```python
基础色系：黑白
暖色系：红橙黄绿青蓝紫
冷色系：红橙黄绿青蓝紫
```

黑白是百搭色，其他大色系要如何搭配呢？

- 同种色系搭配：暖色亮，冷色暗沉，单调。
- 撞色搭配：冷暖撞色为最佳搭配方式，有亮色突出，用暗色掩盖多余亮点。

## 身体部位颜色分布
**代码**：
```python
头部：脸部颜色
身体：冷色调为主，如深蓝、牛仔蓝、米白、浅灰色、咖啡色、黑色
腿部：简单剪裁，直筒型裤子
鞋子：与服装风格相匹配
```

脸部颜色决定了身体部位穿着颜色的种类。

- 偏黑肤色：以冷色调为主，如深蓝、牛仔蓝、米白、浅灰色、咖啡色、黑色。
- 偏白肤色：暖色为主，如红色、粉色、亮紫色等。
- 偏黄肤色：选择中间系颜色，如绿、青、蓝等。
