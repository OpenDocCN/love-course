## 什么是微反应？

上一节我们介绍了课程目标，本节中我们来看看微反应的具体定义。

微反应是指人在受到有效刺激的一刹那，不由自主表现出的、不受思维控制的瞬间真实反应。在国外，微反应技术多用于司法侦查、商业谈判和心理辅导等领域。许多学校也开设相关课程，教授学生如何识别谎言，并通过神态和肢体动作揣摩对方心理。

如今，这项技术更多地应用于日常人际交流。了解对方的心思，有助于你们更好地沟通互动，在合适的时机表达观点，提升人际交往能力。

**核心公式**：`有效刺激 → 瞬间真实反应（微反应）`
