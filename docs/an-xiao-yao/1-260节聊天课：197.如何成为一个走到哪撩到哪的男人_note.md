# 1-260节聊天课：197.如何成为一个走到哪撩到哪的男人

大家好，我是夏洛，今天这节课我要来教大家如何去应对一些难以回答的问题。

## 应对难以回答的问题

想象一下，你看到一个不错的女孩，跑过去想认识她。女孩问你：“为什么你想要认识我？”这是一个很明确指向你的问题。

对于很多男生来说，面对这种尴尬的情况，会感到紧张。这时，你可能会沿着女孩的逻辑路径回答，比如：“我觉得你很可爱，对你有感觉。”但这样的回答并不能引起女生继续与你互动的热情。

## 如何回答才能调动女生情绪

那么，如何回答才能充分调动女生的情绪，让她觉得你很有趣，愿意与你更多互动呢？

**核心概念**：跳出自己的逻辑圈，将问题转化为对方的问题。

例如，你可以回答：“因为你不像坏人啊。”这样的回答显得更加巧妙。

**公式**：难以应对的问题 → 跳出自己 → 转化为对方的问题

当你接受到难以应对的问题时，如果试图用“我怎么样怎么样”来回答，只会让人觉得你很弱势。因为“我”指向自己，当对方明确指向你提问时，你如果还指向自己，就会给人一种防守的感觉，显得你在辩解，从而显得你很虚弱。

但如果你能跳出这个圈子，把问题变成对方的问题，就不会觉得尴尬，也不会觉得难以回答了。

## 总结

本节课中，我们学习了如何应对难以回答的问题。通过跳出自己的逻辑圈，将问题转化为对方的问题，可以让你显得更加自信，从而引起女生的兴趣。
