# 13-安小妖《好男人，你也要学会撩妹》：201.最牛心理画像，告诉你看不到的爱情的模样(下）

## 概述
在本节课中，我们将继续探讨爱情的十个层次，从第六层到第十层，深入了解爱的不同阶段和表现形式。

## 爱的层次

### 第六层：关注
关注一个女生是因为她能给我带来快乐，依恋是为了让她天天守着我，而占有就是不允许她和别人好，交易就是我总得做点什么，以便于说她对我更好。

**公式**：关注 = 快乐 + 依恋 + 占有 + 交易

### 第七层：共情
共情包括理解、揣摩和牵挂。我们要关注对方的一举一动，了解他的内心感受，包括他的不开心、烦恼和需要帮助等。

**公式**：共情 = 理解 + 揣摩 + 牵挂

### 第八层：支持
支持包括帮助、优质服务和热情。我们要知道对方需要什么，提供最优质的服务，帮助对方开心。

**公式**：支持 = 帮助 + 优质服务 + 热情

### 第九层：奉献
奉献包括尊重、信任、包容、接纳、自我牺牲和爱。我们要勇于自我牺牲，尊重对方，相信对方有能力处理好自己的事情。

**公式**：奉献 = 尊重 + 信任 + 包容 + 接纳 + 自我牺牲 + 爱

### 第十层：和谐
和谐包括一致、真诚、自然和坚定。我们要与对方保持一致性，真诚地对待对方，自然地表达自己，坚定地支持对方。

**公式**：和谐 = 一致 + 真诚 + 自然 + 坚定

## 总结
本节课中，我们学习了爱情的十个层次，从关注到和谐，每个层次都有其独特的表现形式和内涵。希望大家能够通过学习，更好地理解爱情，找到属于自己的幸福。
