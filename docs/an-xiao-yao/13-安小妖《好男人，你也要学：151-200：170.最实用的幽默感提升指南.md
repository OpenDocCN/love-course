# 13-安小妖《好男人，你也要学：151-200：170.最实用的幽默感提升指南

在本节课中，我们将学习如何有效提升幽默感。幽默感是人际交往中的润滑剂，能让你在聊天中更受欢迎，尤其在和异性互动时显得尤为重要。我们将从幽默者的共同特征入手，逐步介绍具体的训练方法，帮助你摆脱聊天尴尬，成为有趣的人。

## 幽默者的共同特征

上一节我们介绍了幽默感的重要性，本节中我们来看看那些有幽默感的人通常具备哪些共同特征。理解这些特征，有助于我们明确学习方向。

以下是幽默者常见的五个特征：

1.  **语言能力强**：幽默者对话题的把控感强，能清晰把握谈话的方向与节奏。他们思维敏捷，关键时刻的一句话常如神来之笔，引人发笑。
2.  **富有想象力**：这种想象力并非天马行空，而是指对细节和有趣语境的敏锐捕捉与巧妙“客串”能力。
3.  **生活状态放松**：幽默者通常真实不做作，生活态度轻松。例如，一位52岁的心理科主任风趣幽默，享受生活与工作，状态松弛。
4.  **善于理解与开玩笑**：他们未必阅历丰富，但理解力好，善于开恰如其分的玩笑，并能精准把握他人的笑点。
5.  **保持阅读与思考**：幽默者通常保持一定的阅读量，勤于思考，对事物有独到见解。

## 如何训练获得幽默感

了解了幽默者的特征后，本节我们将具体探讨如何通过训练来获得幽默感。以下是五个核心的训练方法。

### 1. 广泛阅读与积累

幽默需要素材的积累。现代人生活压力大，但阅读不必局限于工具书。

以下是推荐的积累途径：
*   阅读趣味性强的书籍，如王森的《就想开间小小咖啡馆》，既能放松又能增广见闻。
*   多浏览微博、微信、知乎等平台的经典吐槽和段子。
*   留意书籍、电影中的经典幽默桥段，并尝试在生活中灵活运用。

### 2. 丰富生活阅历与观察力

幽默源于生活。一个生活单调的人很难拥有幽默感。

以下是丰富生活与观察的方法：
*   积极体验生活，关心国家大事，也理会家庭琐事。
*   培养细致的观察力，留意身边人与事物的点滴变化。
*   培养个人爱好，例如逛花鸟市场，观察世间百态。
*   多与朋友参加活动，丰富见闻。这样在与女生聊天时，话题自然源源不断。

### 3. 善于自嘲与开放沟通

封闭的内心无法产生幽默。幽默需要信息的交流与情绪的流动。

以下是练习自嘲与沟通的建议：
*   不要将所有事憋在心里，学会与朋友沟通、倾诉。
*   适时自嘲，不仅能舒缓自身情绪，还能展现风趣，赢得好感。
*   例如，朋友生病抽血后发朋友圈调侃：“看着抽的两管血心里非常难受，这要是两管脂肪那该多好。”这就是一种幽默的自嘲。

### 4. 区分幽默与搞笑

很多人误将搞笑等同于幽默，结果适得其反。幽默不是夸张的肢体语言或怪异行为。

以下是关键区分点：
*   **幽默**：话语经得起品味，让人会心一笑，并能领悟其中意味。
*   **搞笑**：可能只是突兀、奇怪的行为，未必能赢得好感，甚至令人反感。
*   例如，同学用夸张动作捡起小女孩掉在地上的棒棒糖并自己吃掉，这仅是搞笑而非幽默，并未给其暗恋的女生留下好印象。

### 5. 保持乐观与笑容

幽默感的底层是一种乐观豁达的人生态度。

以下是培养此心态的核心：
*   无论性格如何，都尝试用乐观的笑容面对世界。
*   幽默感的关键在于是否具备“笑看人生”的态度。
*   就像有人提醒的那样，让嘴角时常像月牙一样上扬。对生活保持微笑，幽默感便会无处不在。
