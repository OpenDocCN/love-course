# 11、【安小妖】脱单12技：05.作业_note

在本节课中，我们将学习如何完成第五课的课后作业。作业分为两个核心部分：首先是提交课堂笔记，以深化对课程内容的理解；其次是进行实际的聊天练习，运用课程提供的模板与异性互动。完成这些作业是巩固知识、获得个性化指导并快速进步的关键。

## 提交课堂笔记 📓

上一节我们介绍了第五课的核心内容，本节中我们来看看如何通过课堂笔记进行内化与总结。

观看完第五课后，你需要向导师提交一份课堂笔记。这份笔记的重点在于记录你对课程的个人总结与理解。

笔记应是一个经过内化和思考的产物，而非边看课边机械记录。未经主动思考的笔记收效甚微。

以下是完成笔记的步骤：
1.  **回顾与思考**：完整观看课程后，脱离视频，凭记忆和理解梳理核心要点。
2.  **总结与书写**：用自己的语言写下课程带来的启发、对关键概念的理解以及可能的疑问。
3.  **提交导师**：将完成的笔记提交给导师审阅。

提交笔记后，你便可以开始进行作业一。作业一完成后，也必须先提交给导师，才能查看参考答案。

向导师提交作业有诸多好处。导师会针对你作业中的不足之处进行点评，指出问题所在并提供优化建议。

同时，导师能更准确地评估你当前的情感经验水平，从而为你后续的课程学习提供更个性化的定制方案。部分学员报课后疏于完成作业，最终学习效果大打折扣。

这并非我们期望的结果。我们最欣赏的是那些报课后努力学习、积极向导师提问的学员。我们鼓励你在看完课程后提出额外的疑问，提问越多越受欢迎，因为这是快速进步的途径。

我们有学员仅用10天就解决了他人一两年都无法突破的聊天难题，这证明了课程与导师辅导的有效性。导师的辅导旨在突出重点、传授干货，但前提是你需要自问：课前准备是否已做好？

## 进行聊天实践练习 💬

完成课堂笔记和第一项作业后，我们将开始第二项作业——基于第五课内容的延伸实践。

第二项作业要求你运用第五课的聊天模板，进行实际的异性聊天练习。

该模板提供了一个从打招呼（如“你头像不错”）到获取联系方式的完整流程结构。在学习初期，你尚不具备自主构建完整对话的能力，因此最佳方式是模仿。此聊天模板经过内部数百人乃至外部社群数千人的实践验证，被证明是一个可持续、稳定、可复制性强且容错率高的有效方法。

**公式：有效聊天 ≈ 运用成熟模板 + 反复练习**

如果你能熟练掌握第五课的模板，你的聊天能力将远超周围多数人，无论他们是否有恋爱经验。但在前期，你必须进行反复练习。

因此，第二项作业要求你：根据第五课模板，至少与十位女性完成完整的聊天流程。

在练习过程中，你可能会遇到各种问题。例如，如果对方完全不理睬，你需要思考是否是自身吸引力不足。吸引力不足可能涉及外在形象、朋友圈展示面或个人微信的价值建设，这些方面都需要提升。

如果对方回应良好，但你无法将对话顺利推进到底，则需注意问题所在：是不会引导话题，还是不会组织语言？此时，你应联系导师或助教寻求帮助，以解决问题并继续进步。

对于许多学员而言，一开始完成十份完整的聊天记录可能颇具挑战。

如果你觉得困难，请在每完成一次聊天尝试后，先进行自我思考与复盘。当你确实无法独立完成时，再将聊天记录截图。

请注意，截图需为长图形式：将每次屏幕截图后，使用拼图软件（如“美图秀秀”）拼接成长图，然后发到学习群中，与同学和导师共同探讨。

这种方式能确保你先经过独立思考，再寻求外部反馈，这样的成长才是扎实有效的。切忌刚聊一句遇到困难就立刻向他人求助，这不利于培养思考能力，学习效果也会大打折扣。
