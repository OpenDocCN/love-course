# 【安小妖】脱单12技：05.作业

在本节课中，我们将学习如何完成课后作业，以及如何通过作业提升自己的聊天技巧。

## 1. 课堂笔记

看完第五课后，首先需要提交一份课堂笔记。笔记的重点是总结和理解课程内容。

**重点**：内化思考，而非简单记录。

## 2. 作业一

完成笔记后，开始做作业一。作业一完成后，提交给导师，然后才能查看答案。

**好处**：导师会根据你的不足之处进行指导，并评估你的情感经验，以便定制后续课程。

## 3. 作业二

作业二要求根据第五课的模板与异性聊天，至少聊十份。

**模板**：从打招呼到收电话的流程。

**目的**：适应良好的聊天结构，提升聊天技巧。

## 4. 遇到问题

在聊天过程中，可能会遇到以下问题：

* **女生不理你**：考虑提升外在形象、朋友圈照片和个人价值。
* **无法完成聊天**：联系导师或助教，学习引导话题和造句。

## 5. 完成作业

完成十份聊天记录后，思考问题，并将聊天记录截成长条图，与同学和导师探讨。

**注意**：先思考再提问，避免依赖他人。

## 总结

本节课中，我们学习了如何完成课后作业，以及如何通过作业提升自己的聊天技巧。希望同学们能够认真完成作业，并在实践中不断进步。
