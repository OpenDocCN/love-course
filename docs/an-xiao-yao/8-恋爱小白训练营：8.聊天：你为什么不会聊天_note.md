# 8-恋爱小白训练营：8.聊天：你为什么不会聊天_note

在本节课中，我们将要学习阻碍你与女生顺畅聊天的核心心态问题。许多初学者感到聊天困难，其根源往往不在于技巧或话题，而在于内心深处的错误认知。纠正这些心态，是学会真正聊天的第一步。

## 概述：聊天的核心障碍

很多同学表示不擅长或不知道如何与女生聊天。如果你也存在这样的问题，说明你缺乏和女生聊天的正确精力。持续下去，很容易陷入“不敢聊导致不会聊，不会聊又更不敢聊”的恶性循环。

## 阻碍聊天的四种错误心态

以下是四种常见的错误心态，它们是你聊天路上的绊脚石。纠正这些心态，你才能建立平等、自信的沟通基础。

### 1. 严重稀缺心态

在与女性交往的过程中，稀缺心态让男人迷住了一个女人，这是最糟糕的一种心态。

**稀缺心态的定义是**：一旦出现一个你喜欢的女孩，你就会说服自己：
*   **A.** 她是一段时间之内自己唯一喜欢的女孩。如果她不喜欢自己，要等待很久下一个自己喜欢的女孩才会出现。于是你开始过分在意结果。
*   **B.** 除了她，你不会再和其他女生产生类似的连结感。

这种心态非常常见，好莱坞电影和媒体将其充斥在你的大脑里。如果你去听情歌，会发现几乎所有的情歌都把你浸泡在这样的心态当中。

你见过一个自己喜欢的女孩，回家后不停想起她。请问这个女孩回家后，她会特意花时间想你吗？会的，但是她想的不是你。她会想除了你以外的人和事。

如果在和女生聊天中你存在这样的稀缺心态，那么注定你是没有办法吸引女人的。

### 2. 神坛上的女神

上一节我们介绍了稀缺心态，本节中我们来看看另一种物化女性的心态。

大多数男人常常会把女人放在神坛上当做女神。但是你们觉得女人会愿意让你把她当做女神吗？女人可能会要求你把她当做女神，但当你照做时，你会立即失去对她的吸引力，而且女生也会开始变得不尊重你。

女人不想让你把她当做女神。她们可能享受被当成女神的感觉，然而她们不会因为你把她当做女神而喜欢你。因为女人希望被当做人来对待。

当你和女生聊天前，如果你思前想后不敢开口，生怕自己做错事说错话，那么你就是在把她当做一个物体，试图从中榨取价值。这就是女人不愿意被当做女神的原因。

如果你想掌握怎么和女人聊天，那么请尊重自己，把自己和女人放在平等的位置之上。

### 3. 存在过分解读

多数男人和女生聊天的时候，都会过分解读女生的言行，喜欢多想。一个姑娘稍微和自己说两句话，自己回家之后能想好几个月。女生给自己买了一个钥匙链、借了一根圆珠笔、微笑了一次，心里就会过度解读这件事。

这就是为什么很多女人选择对大多数男人冷漠。因为女人心里清楚，这些男人会过分解读，然后言行变得奇怪。

一旦一个女人发现，你不会过度解读你们之间的沟通，你很酷、你无所谓、你享受当下、你很幽默，她会对你很友善。但如果她觉得你会过分解读，回家后脑中反复思考，慢慢地，她就会对你变得冷淡。

**核心原则**：不要去在乎结果，不要去过分解读，你将会得到更加友善的回应。

### 4. 尝试逻辑说服

和女生聊天的时候，很多男人都在尝试通过逻辑说服女人，让女人喜欢上自己。

他们认为自己可以向女人展示：给女人买礼物、带她去很棒的地方、自己有别墅等等。他们觉得女人会被这些东西所吸引。于是在聊天中说：
*   “我刚买房不久，车也是新的，我们一起开车去我家吧。”
*   “我知道有家服饰店不错，我给你买几件衣服吧。”
*   “我们在一起吧，我会对你非常好的，我会让你成为这个世界上最幸福的女人。”

这些一律都是在尝试通过逻辑去说服女人。除了极少数物质的女人之外，女人不会对任何形式的逻辑说服做出反应。

**关键结论**：永远不要试图通过大量的逻辑说服女人喜欢自己。让女人喜欢自己的唯一方式，是让她对你产生感觉，即你作为一个人所拥有的整体感觉，包括自信、幽默等等。

## 聊天的本质思考

以上四个心态，是“怎么和女生聊天”的基石。如果你在这四个心态上存在问题，不论你的话术有多厉害，不论你的话题有多么生动有趣，都不会让女生真正喜欢上你。

说完了四个心态，现在我想让你思考两个本质问题。

### 问题一：聊什么真的重要吗？

女生聊天并不是看她对话题感不感兴趣，而是看她对聊天这个人感不感兴趣。

试想一下，你在车站等车，一个满脸油光、有口臭、体型不佳的人过来找你聊天，你脑海中的第一个想法是“别靠近我”，接着你想让她赶紧从你面前消失。如果她很认真地跟你聊你最喜欢的话题，比如游泳或足球，你会配合她聊下去吗？我相信你不会，因为你唯一想的就是如何让她赶紧消失。

再试想一下，如果一个身材高挑、妆容精致、身上散发着淡淡香味的美女走过来向你问路，你会怎么想？你会觉得“好养眼，好享受”，甚至可能没听清她说什么。假如她问的地方你并不认识，你也愿意帮她查地图，尽可能多地交流，因为你就是想和她多待一会儿。

所以很多时候，我们觉得聊天难以进行，不是话题的原因，而是聊天对象的问题。话题其实并不难找，关键是对面的女生想不想跟你聊。

**核心结论**：聊天的关键不是找话题，而是建立吸引。假如我们的自身建设已经很到位，那么开口聊天就不再是问题。

### 问题二：我们到底聊的是什么？

紧接着我们来说一下第二个问题。

其实，你跟别人聊的，恰恰是真实的自己。很多人觉得没有话题聊，因为你的生活本来就没有什么有趣的事情，你自己的内心世界本来就很匮乏。你怎么可能有话题聊呢？其实不是聊天无聊，是你这个人无聊。

我记得小时候班级里，总有那些特别能说会道的男孩子，女孩下课就围着他。为什么呢？因为他见识多，说的东西很新奇，大家听他说话就像看电影一样有意思。所以大家对他说的话特别期待，觉得这个人特别有魅力。

有一句古话叫做：“问渠那得清如许，为有源头活水来。” 要想让自己聊天有话题、有内容可聊，我们就先要让自己的生活丰富起来。

**生活与聊天的关系公式**：
`丰富的生活体验` -> `充实的内心世界` -> `自然流露的聊天内容`

打个比方：我们的生活好比“吃草”，跟女生聊天或社交就像“挤奶”。如果你的草吃得不够，或者消化得不好，那么你的奶是挤不出来多少的。

走出自己的社交圈，多参加各种活动，你就可以把自己的生活维度打开，发现很多有意思的事情。然后再把这些有意思的事情分享给女生，就很好了。

除了找同学聚餐，还可以主动去认识一些陌生人。比如现在网络上有很多同城活动，你可以挑一些自己感兴趣的去参加。

一旦你的生活变得有意思了，你就像一棵树得到了充足的养分，开出的花、结出的果自然是又香又甜的。

## 总结

本节课中我们一起学习了阻碍聊天的四种核心心态：**稀缺心态**、**将女性神化**、**过分解读言行**以及**试图逻辑说服**。这些心态是聊天困难的根本原因，纠正它们是第一步。

随后，我们探讨了聊天的两个本质：**聊天的关键在于建立吸引，而非寻找话题**；**聊天内容源于你真实而丰富的生活体验**。让自己成为一个有趣、充实的人，才是畅聊不息的不二法门。
