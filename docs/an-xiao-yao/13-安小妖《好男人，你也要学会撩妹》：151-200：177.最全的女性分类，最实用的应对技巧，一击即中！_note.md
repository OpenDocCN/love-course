## 教程内容

### 1. 理智型女生

**特点**： 充满智慧的气质，望而生畏。

**应对技巧**： 直接表达爱意，用感情战胜理智。

**公式**： 直接表达爱意 → 吸引她 → 用感情战胜她的理智。

### 2. 内向型女生

**特点**： 不爱表达情感，容易产生压抑感。

**应对技巧**： 多理解、关心和帮助，提供情感宣泄机会。

**公式**： 多理解、关心和帮助 → 让她在心情不好时倾诉 → 提供情感宣泄机会。

### 3. 自尊心强女生

**特点**： 对自己的容貌或其他外在条件很有自信心。

**应对技巧**： 巧留余地，抓住时机委婉表达爱意。

**公式**： 巧留余地 → 给她自尊心满足 → 抓住时机委婉表达爱意。

### 4. 表面不在意女生

**特点**： 内心充满矛盾，不愿被拒绝。

**应对技巧**： 以静制动，把握好节奏。

**公式**： 以静制动 → 把握好节奏 → 猎取芳心。

### 5. 喜欢直率表达女生

**特点**： 喜欢直出来说话，讨厌拐弯抹角。

**应对技巧**： 单刀直入，直率邀约。

**公式**： 单刀直入 → 直率邀约 → 让她感到无法拒绝。

### 6. 喜欢旁敲侧击女生

**特点**： 喜欢用言语掩盖本意，不喜欢被一语道破。

**应对技巧**： 注意说话艺术，察言观色。

**公式**： 注意说话艺术 → 察言观色 → 避免一语道破。

### 7. 喜欢潇洒从容男生

**特点**： 不喜欢呆若木鸡或少言寡语的人，也不喜欢巧舌如簧的人。

**应对技巧**： 保持潇洒从容，真诚自然。

**公式**： 保持潇洒从容 → 真诚自然 → 消除戒备心理。

### 8. 喜欢果断自强男生

**特点**： 喜欢刚毅、果敢、热情、爽朗、敢于承担责任的人。

**应对技巧**： 收起妈宝男和巨婴的一面。

**公式**： 收起妈宝男和巨婴的一面 → 成为一个果断自强的男生。

### 9. 喜欢巧迫常规男生

**特点**： 喜欢生活有所变化，不喜欢刻板、墨守成规的人。

**应对技巧**： 打破平淡生活，带来惊喜。

**公式**： 打破平淡生活 → 带来惊喜 → 满足她的求变心理。

### 10. 喜欢先入角色男生

**特点**： 喜欢说话的人的情绪感染力。

**应对技巧**： 通过语言表达个人感受，提高关心程度。

**公式**： 通过语言表达个人感受 → 提高关心程度 → 打动她的芳心。
