# 1-260节聊天课：142.聪明男人才懂得的聊天策略

大家好，我是夏洛，这里是撩妹聊天大师的第143节课。今天我们来探讨一个重要话题：如何通过聊天策略吸引女性。

## 搭讪后的思考

当你成功搭讪并获取到女生的联系方式后，你可能会想象到很多画面。但事实是，仅仅因为女生给了你联系方式，并不意味着她对你有深入了解或兴趣。

## 了解的重要性

女人可能会被你吸引，但这并不意味着她有必要深入了解你。因为被吸引很容易，而缺乏了解可能导致误解。例如，女生可能会认为你只是想与她发生关系。

## 赋予人格（负格）

为了建立信任，你需要学会“赋予人格”，即展示你选择她的原因。这对于外表吸引人的女性尤为重要，因为她们可能已经习惯了被众多男性追求。

## 如何负格

以下是一种负格的方法：

1. 喝点东西，装作有点醉。
2. 突然说：“你知不知道你有时候挺烦人的？”
3. 表扬她的外貌和性格，然后说：“我认识你这么久，就没见你对谁发过脾气。”
4. 提出你的不同之处：“我就想问问你，你说你把自己弄得那么完美，你不累吗？”
5. 建议她在你面前放松一点，偶尔说脏话。

## 负格的效果

通过这种方式，你可以：

1. 让她感受到你的诚意，希望她放松。
2. 帮助她找到自己与众不同的地方。
3. 通过互动交流，证明她的独特之处。

## 总结

本节课中，我们学习了如何通过聊天策略吸引女性。了解、赋予人格和互动交流是关键。希望这些技巧能帮助你更好地与女性沟通。下节课再见，我是夏洛，拜拜。
