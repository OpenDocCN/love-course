# 1-260节聊天课：195.聊天的三个基本功，学完稳稳上高地

大家好，我是夏洛，今天这节课咱们来讲调情。调情，听起来神秘，常觉得无从下手。今天一步步教你简单易学的调情方式。

**调情基础**：

*   调情需先有情，情是良好关系，建立需赞美、提供价值、带出去玩等。
*   但不能跪舔，需调整不平衡状态，让她觉得接受是合理的，不带来压力。

**调情方法**：

1.  **赞美中带批评**：如“看你那副可爱的样子，不忍心批评你总是粗心的丢东西。”
2.  **夸她捧自己**：如“没想到你不仅长得可爱还很聪明，我就说嘛我看人的眼光一向很准。”
3.  **抱怨调整态度**：如“多怪你长得那么可爱，我正好放下那些需要整理的资料，周末出来找你玩的。”
4.  **自嘲关注自己**：如“你长得这么可爱，性格一定也很好吧，唉我就会以貌取人，没有办法。”

**聊天准则**：

*   你等于关注，等于感兴趣，我等于自信。
*   灵活运用“我和你”，表达兴趣和自信。

**调情总结**：

*   赞美后关注自己，显得自信。
*   在直白表达背后加上其他内容，分散注意力。

**提出要求调整**：

*   **买单**：如“这次我先请你，下次再给你为我买单的机会。”
*   **情绪变化**：如“我聪明吧，不管你要干嘛，我都能随机应变，还不快夸夸我。”
*   **约玩**：如“听说你胆子蛮大的啊，喜欢刺激的运动，巧了我也是，我们去挑一下比比啊。”

**调情核心**：

*   在直白表达背后加上其他内容，分散注意力，让她关注更有趣、槽点、想了解的东西。

**总结**：

本节课我们学习了调情的基本概念和方法，通过赞美、批评、自嘲等方式调整态度，让聊天更有趣、更有槽点，从而吸引女生。希望对大家有所帮助。
