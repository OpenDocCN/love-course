# 8-恋爱小白训练营：9.升级：如何在聊天中升级关系_note

在本节课中，我们将学习如何在聊天中有效地升级你与对方的关系。我们将首先分析两个常见的升级误区，然后深入探讨建立舒适感的具体方法，包括共谋、条件反射和分享秘密。掌握这些技巧，能帮助你更自然、更顺利地推进关系。

## 误区一：在微信上告白或确认关系

上一节我们介绍了课程概述，本节中我们来看看关系升级的第一个重灾区：在微信上告白或确认关系。

在未见面或现实中无肢体接触的前提下，于微信上升级关系，绝大多数情况下会导致关系崩盘。没有现实生活中的接触与升级就在微信上确认关系，且不能马上见面，会导致女人缺乏安全感与信任度。女人潜意识里会觉得你对待这份感情不认真，她也很难真正投入这段感情。一段不被信任、缺乏投入的感情势必会出问题。

另一方面，一旦在微信上确认关系（例如互称老公、老婆、亲爱的），但尚未见面，会导致女生对你的期望过高。她会因感情因素将你的形象主观美化。一旦线下见面，如果你的真实形象与她的预期有丝毫偏差，女生就会觉得你是骗子，从而导致关系崩盘。

**核心概念**：告白是胜利的凯歌，而不是冲锋的号角。关系升级完成才需要表白。

## 误区二：不断试图展示高价值

说完了第一个误区，我们再来看看第二个常见错误：不断试图向女人展示你的高价值。

在你的说法里这是展示高价值，但在女人眼里，这往往是“装逼”。在与女生相处过程中，向对方适当展示自己的脆弱与悲伤有何不可？难道必须是完美无缺、永远积极乐观的人才值得被爱吗？如果你在女人面前无法做真实的自己，双方该如何相处？她又可以喜欢你什么呢？所以，适时展示脆弱面，在恰当时候能起到超乎想象的作用，这点我们稍后会详细讲解。

女人并非对吃喝玩乐或生活趣事不感兴趣，而是可能对你这个人不感兴趣，或对你的表达方式不感兴趣。如果你们的相处模式是你整天“装逼”展示所谓高价值，每天找她说话分享生活，并用生硬尴尬的直男方式调情，那么任何女人都难以喜欢并给出积极回馈。如果你真的喜欢她，就别再伪装成高高在上、完美无缺的人。

高价值与优质生活的展示，无需靠嘴巴说出来。对待喜欢的女人，在未见面前说一万句调情的话，都比不上想办法见一面，然后牵着她的手给她买一个冰淇淋来得实在。如果你们暂时无法见面，就不要过早表达感情。

## 如何在聊天中升级关系：建立舒适感

好了，澄清了两个误区后，我们正式进入核心环节：探讨如何在聊天中升级关系。单纯的吸引不足以让故事发生，这点我们在之前的课程中已讨论过。例如，吴彦祖很吸引我，但我并不会死心塌地等着和他谈恋爱。所以，要想有故事发生，我们需要进入下一阶段：**建立舒适感**。

那么，什么是舒适感？舒适感并非简单的聊骚或套近乎，也不是每天问“在干嘛”、“吃了吗”、“晚安”。对许多女生而言，这甚至是单方面的骚扰。舒适感是指对方对你的信任与安全感，愿意与你亲密而不感到别扭。然而，这种信任不同于对同事、同学或知己的信任，因为后者缺少特殊的亲密感；这种安全感也不同于对家人的安全感，因为家人间的安全感难以脱离血缘关系。

在交往初期，如何为女生建立舒适感呢？以下是几个有效方法。

### 方法一：共谋

共谋，即与对方共同完成同一件事。这类事情的发生，会让她潜意识里觉得你们像一对情侣。以下是具体操作方式：

*   **现场现实共谋**：你与她在相同物理空间做同一件事。其实约会中的每件事都算共谋，但应努力做些特别的事。普通的吃饭聊天经历太多，难以调动情绪，容易无聊。我们要做的是让她产生特别情绪，例如紧张、激动或纠结。
    *   **举例**：人行道绿灯最后十秒时，带她跑到街对面（确保无车）。
    *   **举例**：在餐厅吃完饭，偷偷结账后告诉她“我们这次逃单”，不由分说拉她跑出去，跑一段再说“完蛋，餐厅好像有监控，明天要上新闻了”（其实你已买过单）。

**操作注意**：需根据对方性格把握分寸。如果对方内向、害羞且敏感，你突然拉她上台跳舞献唱，她可能再也不理你。因此，要根据不同性格设计不同的共谋活动。

### 方法二：条件反射

接下来我们看看如何利用条件反射原理来建立舒适感。初中生物课我们学过著名的巴普洛夫实验：给狗摇铃铛然后喂食，持续一段时间后，狗听到铃声就会分泌唾液。铃声是条件，分泌唾液是条件触发的反射。

在这个实验中：
*   **条件（铃声）**：是我们要创造的。
*   **反射（分泌唾液）**：是我们与对方建立的舒适感，即营造暧昧。

**核心公式**：`条件（我们创造） -> 反射（舒适感/暧昧）`

当你们一起玩耍、她情绪高涨或与你情意绵绵时，让一个物品反复出现在她视线内。例如：
*   带她去抓娃娃机，抓到后送给她。之后她只要看到娃娃就会想起你。
*   送她一个每天都会用到的东西，例如口红或香水（均价200-500元）。以后她每次使用就会想起你。这比花大价钱买名牌包更有效。

### 方法三：分享秘密与展现脆弱面

最后，我们探讨通过分享秘密和展现脆弱面来建立信任。信任是舒适感的重要组成部分，但获得他人信任非常困难。在日常生活中，我们需要表现好、口碑佳、主动投资，并经过时间累积，才可能获得信任，正所谓“无事献殷勤，非奸即盗”。

在与女人相处一段时间后，你需要通过“投资”来获取信任。这个“投资”就是告诉她一个你的秘密。

**操作要点**：
1.  要摆出煞有介事的样子，要求她一定保密，这样你的“投资”才显得有价值。
2.  秘密内容由你把控，不一定是惊天秘密（以免吓跑对方）。可以是一件普通但你不会与他人分享的事，因为“其他人可能不理解”。
3.  你甚至可以要求她也分享一个秘密，这样更公平亲密，建立“革命情谊”。

**展现脆弱面**：这也是一个有效方法，适用于微信聊天和面对面沟通，但前提是彼此都有好感。在聊天后期，偶尔聊聊你感到自卑的点，分享你的压力、恐惧、窘迫、遗憾或伤痛。例如，你有时也会没有安全感，想逃离，或希望找个依靠。

**注意事项**：这种情感分享要适度，不能过多。切勿在前期女生对你表达兴趣前就过早分享。在后期适当分享，能激起女性的母性关怀，让她觉得你们有相似感受，能互相理解，从而使关系更上一层楼。

## 聊天前期操作建议

在聊天前期，你可以分享你们经历过的故事、身边朋友的故事或听过的故事。总而言之，主动提供谈资让女生接话，对她而言更容易。在聊天前期，我们应提供90%以上的内容，因为这是你吸引力的来源。通过分享，你也可以寻找女生感兴趣的点，从而延伸话题，这是一个相对可靠的方法。
