# 5-【安小妖】极速吸引方程式：12.筛选的意义

## 概述
在本节课中，我们将探讨在情感发展过程中，筛选的意义及其在吸引过程中的作用。

## 山选过程回顾
以下是山选在情感发展模式中的顺序位置：
1. **开场白**：对话开始，双方未产生吸引。
2. **过渡**：正常谈话，双方未互相吸引。
3. **吸引**：一方开始被另一方吸引，产生好奇。
4. **山选**：另一方开始被吸引，产生好奇。
5. **舒适感**：双方情绪受对方影响，有肢体亲密动作。

## 山选的意义
山选过程是吸引的相反过程。以下是山选的几个重要意义：

### 1. 显示非外貌吸引
山选过程给了你一个理由，来显示你不是因为她的外貌或身材而被吸引。

### 2. 避免快速更换对象
如果你已经被山选，你不能再见到另一个有吸引力的女人后，转身就和另一个女人交往，这样会让她以为你只是对她的外貌或身材感兴趣。

### 3. 满足女性的竞争心理
大多数女性喜欢去赢得有要求的男性的注意，调情对她们来说是一件有趣的事情。

### 4. 建立平等关系
筛选巩固了你对她的吸引，使你们可以在一个平等的条件下开始舒适阶段，专注于建立彼此之间的联系感。

### 5. 投资心理学原理
人们为了某一个目标，投资越多，就越想达到这个目标。山选过程越真实，你在不符合你标准的女人身上花的时间就越少，而对于那些符合你标准的女人，你对她的兴趣和好感就会越来越真诚。

## 筛选技巧
以下是几个筛选技巧：

1. 寻找除了美貌之外的特性，如音乐喜好、理想、爱好等。
2. 不要让筛选过程太简单，让她相信她就是你要找的那种女人。
3. 高标准的女性可能不善于筛选，但她们通常在所有场合中保持光彩照人。
