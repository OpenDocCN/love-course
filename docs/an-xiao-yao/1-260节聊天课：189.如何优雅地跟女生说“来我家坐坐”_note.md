1. 不想上班就别上了呗，辞职。
2. 不上班，你怎么挣钱养活你自己啊，今天想买这个明天想买那个的。

5. 哎呀我也是，我比你还困，比你还不想上班。

## 分析
当别人向我们表达感受、态度或状态时，他们通常希望我们能够：

1. 认真倾听。
2. 给予反馈。
3. 达成某种一致。

## 正确回复方式
正确的回复方式应该是：

1. 让对方明白我们非常愿意了解他们的状态和处境。
2. 设身处地为他们着想。
3. 表达理解，因为能够感同身受，有过类似的经历。
4. 简单地说，就是我们在听，而且我们懂。

以下是对上述七种回复方式的分析：

1. 提供解决问题的方法：这种回答方式看似有用，但实际上没有任何用，对方心里还会有怨念。
2. 警示意味：这种回答方式给人一种不在乎对方感受的感觉，对方可能会觉得不舒服。
3. 挑衅：这种回答方式可能会引起争吵，不利于建立良好的关系。
4. 比较和忽视：这种回答方式会让对方感觉被忽视，不利于建立情感共鸣。
5. 转移话题：这种回答方式会让对方感觉被忽视，不利于建立情感共鸣。
6. 过度提及自己：这种回答方式会让对方感觉被忽视，不利于建立情感共鸣。

与女生产生情感共鸣的方法：

1. 不要评价对方的感受和状态。
2. 通过表达与对方相同的行动来表示认可。
