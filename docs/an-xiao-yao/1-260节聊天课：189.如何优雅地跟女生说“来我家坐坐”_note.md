# 1-260节聊天课：189.如何优雅地跟女生说“来我家坐坐”_note

在本节课中，我们将学习如何与女孩建立情感上的连接。我们将从一个常见的生活场景入手，分析不同回应方式背后的情商逻辑，并总结出达成情感共鸣的核心方法。

## 情商测试：冬日清晨的抱怨

我们从一道情商测试题开始。冬天非常冷，你和几个朋友住在一起。早上大家都不愿离开温暖的被窝去上班，但又迫不得已。这时，你的室友揉着眼睛，睡眼惺忪地抱怨了一句：“哎呀真想睡懒觉不想去上班。”

此时，你会如何回应？以下是六种常见的回答选项：

1.  不想上班就别上了呗，辞职。
2.  不上班，你怎么挣钱养活你自己啊？今天想买这个明天想买那个的。
3.  嗨，你跟我说有什么用吗？有本事跟你们老板说去。
4.  哎呀我也是，我比你还困，比你还不想上班。
5.  你呀睡得太晚了，今天早点睡，明天就起得来了。
6.  每天早起确实挺辛苦的，谁让咱们要来北京闯荡呢。

你会选择哪一个？请先思考一下。

## 分析：高情商回应的核心

在通常情况下，当别人向我们表达感受、态度或状态时，他首先希望我们能认真倾听。如果还能得到一些反馈，甚至能达成某种一致，他会更加开心。这样的交流能让双方都满意，并增进彼此的关系和感情。

因此，正确的回复方式应该是：首先让对方明白，我们愿意了解他的状态和处境，并愿意设身处地为他着想。在此基础上，表达出“我理解你，我能感同身受，我认可你的情绪”。简单说，就是传递“我在听，而且我懂”的信号。

有了这个共识，我们再来逐一分析上述六种回答。

## 逐一解析六种回应方式

以下是针对每种回答方式的具体分析：

**第一种和第五种：急于提供解决方案**
这两种回答（“不想上就别上呗，那就辞职”和“你今天早点睡，明天就起得来了”）都是在提供解决问题的方法。但如果对方能轻易做到（如辞职或早睡），他早就做了，无需他人提醒。这种听起来有用的方案其实并无实际帮助，反而可能让对方产生怨念，觉得你“站着说话不腰疼”。因此，这两种是低情商的回答。

**第二种：用现实进行警示**
“不上班，你怎么挣钱养活你自己啊？”这句话带有警示意味，用血淋淋的现实告诉对方别傻了。这种回答给听者的感觉是：我不在乎你的感受，也不在乎你幼稚的情绪发泄，我只希望你成熟点。对方会感到不舒服，以后便不愿再与你分享感受，因为他的感受在这里得不到肯定，只得到否定和压抑。

**第三种：挑衅与无效回应**
“你跟我说有什么用啊？跟你们老板说去。”这种回应近乎挑衅，毫无意义，容易引发争吵，显然也不可取。

综合来看，以上几种回答都没有与对方产生情绪上的共鸣。

**第四种：陷入“比惨”式比较**
“我呀比你还困，我也不想上班。”这听起来似乎有共鸣，但其实存在弊端。弊端在于，你的回答是在进行比较。言外之意是：“你这算什么？我比你还惨。”这依然是对对方情绪的一种忽视，潜台词是：“你不应该抱怨。”因此，这种比较的方式也是不对的。

与“比较”类似的一种情况是**偷偷转移话题**。例如，听到对方说“我很困很累不想上班”，你回应说：“哎呦你让我想起我刚来北京的时候……”当你不断谈论自己、强调自身感受时，其实也是对对方感受的一种忽视。

因此，当对方与我们分享某种感受时，我们不应过度提及自己。例如，女孩说“今天加班好累呀”，你不应回答“我前段时间也老加班，确实特别累”，这样的话意义不大。

## 如何达成情感共鸣：方法与示例

经过以上分析，我们可以得出结论：要与女孩产生情感上的共鸣，首先不要原原本本地评价对方的感受和状态。

例如，女孩说：“刚刚陪朋友看完电影回家，感觉好累。”如果你只是说：“在外面玩了这么久，确实很累呀。”这话虽然表示理解，但完全是阐述式的，只是把她的话换种方式又说了一遍。这种关注和理解意义不大，对方感受不到你的诚意。

如果你想与对方达到共情，最好的方式是通过表达与对方一致的行动，来表示你对她的认可。更进一步，如果你想更好地达成情绪共振，可以适当运用一点夸张的技巧。

**示例：**
女孩说：“哎呀这两天身体不舒服，都不想吃东西了。”
你可以这样回应：“那我最近也吃得简单点，就不再在朋友圈秀美食了，免得给你造成更大的心理创伤。等你恢复了，再带你一起去把好吃的餐厅都吃一遍。”

这样的表达在某种程度上做到了很好的情绪共鸣，同时给对方带来了很高的情绪价值。

## 课程总结

本节课中，我们一起学习了如何与女孩建立情感连接。我们从一个常见的情商测试入手，分析了六种不同回应方式的优劣，并揭示了高情商回应的核心在于传递“我在听，而且我懂”的信号。关键在于避免说教、比较或转移话题，而是通过表达理解、认可乃至一致的行动来达成情感共鸣。记住，真诚的倾听与共情的回应，是拉近彼此距离的优雅方式。
