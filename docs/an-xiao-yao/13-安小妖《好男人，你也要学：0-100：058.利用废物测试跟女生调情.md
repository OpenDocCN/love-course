## 什么是废物测试？ 🤔

上一节我们介绍了课程主题，本节中我们来看看废物测试的具体定义。

废物测试是一个专有名词。它的含义是女生用来检验男生是否有足够能力保护自己、是否表里如一、是否花心、能否提供安全感。出于这些目的，女生会提出一些测试性问题。

很多男生觉得回答这些问题非常困难。因为回答稍有不慎就会暴露出自己的需求感，或展现出自己的低价值，导致应对尴尬。其实应对女生抛出的废物测试并不像想象中那么难。

## 废物测试的意义 🔍

女生的废物测试实际上是测试男生内外的一致性。这种测试很多时候并非特意进行，而是人类进化过程中的一种自然反应。其目的是为了筛选测试目标是否适合自己。因此，男生构建一套行之有效的应对系统很重要。

废物测试对男生也并非坏事。女生对男生抛出废物测试问题时，代表她已经与男生建立了一定吸引。女生不会对自己完全不感兴趣的男生抛出废物测试。所以当女生抛出废物测试、释放兴趣指标时，你要看清形势，通过合理回答安全度过筛选。

以下是三种常见的废物测试类型及应对方法。

### 1. 劳力型废物测试 👜

第一种废物测试是劳力型废物测试。“劳力”指让你出苦力、帮忙劳动，例如让你帮忙拿包、拿衣服、拿重物。

按照传统思维，女生让男生帮忙拿包或重物，男生通常会乐呵呵地殷勤帮忙，心里想这正是展现体贴和力量的机会。但如果女生使用命令语气而非撒娇语气，就等于抛出了一个废物测试。

```
“一分钟10块钱，我先帮你记账上了哈。”
```
或者，拿到包包后可以说：
```
“哪有那么便宜的事，你得请我吃饭啊。”
```
说这些话时要与女生目光对视并微笑，切勿摆出臭脸。更酷的做法是趁伸手接包时顺势牵住女生的手，然后微笑着说：
```
“对于我而言，你可比你的包包贵重多了。”
```
这样就将废物测试巧妙逆转成了服从性测试，并完成了肢体接触。

第三种废物测试是反语寻求认同型。不少女生喜欢通过反语形式提出废物测试。

综合以上应对方式可以看出，应对废物测试并不难。只要在回答时跳出女生的框架，利用推拉、打压、调侃、戏谑等方式应对即可。

最简单的例子是，在微信等聊天软件中刚认识几天的女生让你帮她充话费。或者刚认识几天逛街时，她看中很多东西并对你说：“我今天钱没带够，你帮我买了吧。”这些都是典型的金钱型废物测试。

```
“好啊，我家正好有些脏衣服没人洗呢，你帮我洗了我就给你买。”
```
