# 1-260节聊天课：100.送她礼物之前必学的撩妹套路

大家好，我是夏洛。这里是撩妹聊天大师的第100节课。圣诞节过后紧接着是元旦节，之后是情人节和春节，这是一个充满欢乐的时段。

在这个时期，我们相信每个人对情感的渴望都被充分激发。因此，利用这个欢乐的节日氛围，给喜欢的女生送小礼物，准备小惊喜和小浪漫，是一件任何时候都不过分的事情。至于送什么，之前已经讲过很多了，无论是送宠物、旅行、食物还是生活用品。

今天，我们不讨论送什么，而是讲解如何给她准备惊喜，用什么样的场景制造良好的情绪体验。

### 第一种方法

1. 准备一件小礼物。
2. 约她出来吃饭或玩耍。
3. 在路上装作不舒服，比如停车路边，捂着胸口说心脏不舒服。
4. 她帮你找药时，将礼物藏在手套箱里。
5. 告诉她药在里面，让她打开礼物。
6. 故作轻松地说现在感觉好多了。
7. 她会疑惑礼物怎么出现在箱子里，你可以说不是特意准备的，是运气好找到的。

这种方法的**关键**在于让她在不知情的情况下发现你为她准备的礼物，给她带来意外和惊喜。之后，你可以说这不是特意准备的，只是运气好找到的，这样可以弱化你为她准备礼物的行为和动机，增加趣味性。

### 其他方法

1. 她认为你应该准备礼物时，你可以说太忙忘了，然后在路上买糖果给她，实际上糖果里藏有礼物。
2. 发消息或打电话说心情不好，想出去走走，说服她出来。
3. 带她去爬山或城市制高点，告诉她不知道送什么，最后决定送她最美好的景色。

### 总结

本节课中，我们学习了如何给女生送礼物的套路，包括制造惊喜、弱化动机和增加趣味性。希望这些方法能帮助你更好地表达情感，给女生带来快乐。
