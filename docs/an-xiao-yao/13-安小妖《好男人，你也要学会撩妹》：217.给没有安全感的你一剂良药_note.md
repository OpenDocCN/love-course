# 13-安小妖《好男人，你也要学会撩妹》：217.给没有安全感的你一剂良药

## 概述
在本节课中，我们将探讨如何在感情中获得安全感。安全感是许多人面临的常见问题，尤其是在经历了各种感情之后。

## 缺乏安全感的原因
以下是导致缺乏安全感的原因：

- **寄托于他人**：将安全感的获得寄托在他人身上，而非自己。
- **自我评价失衡**：过于关注他人，忽视自我，导致自我评价失衡。
- **对外界反应敏感**：对外界反应过于敏感，容易产生焦虑。

## 如何获得安全感
以下是获得安全感的几个方法：

### 1. 自我维护
- **保持清醒的对话**：跳出自我，以第三方的视角与自己对话。
- **屏蔽负面情绪**：屏蔽那些让自己缺乏安全感的心声。

### 2. 处理好与外界的关系
- **亲情与友情**：亲情与友情是获得安全感的重要方式。
- **保持预期一致**：与亲人朋友的预期保持一致，避免嫌隙。

### 3. 增大资源储备
- **财富**：努力赚钱，建立更丰富多元的社交生活。
- **能力**：挖掘财富的创意，培养外向型人格，提高社交倾向。

### 4. 构建选择伴侣的标准
- **明确标准**：明确自己想要的伴侣类型，避免感情透支。

## 总结
本节课中，我们学习了如何在感情中获得安全感。通过自我维护、处理好与外界的关系、增大资源储备和构建选择伴侣的标准，我们可以更好地应对感情中的安全感问题。
