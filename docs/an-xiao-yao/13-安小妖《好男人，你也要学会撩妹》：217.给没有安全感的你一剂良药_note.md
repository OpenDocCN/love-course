# 13-安小妖《好男人，你也要学会撩妹》：217.给没有安全感的你一剂良药_note

在本节课中，我们将要学习如何在感情中建立和维持安全感。安全感是稳定感情的基石，缺乏安全感就像情感上的“感冒”，非常普遍。我们将通过四个核心步骤，帮助你从自身出发，构建坚实的安全感。

## 为什么我们会缺乏安全感？

上一节我们提到了安全感的重要性，本节中我们来看看缺乏安全感的根本原因。通常，人们认为安全感是他人给予的，例如“他能不能给我安全感”。这恰恰是问题的核心：**你将安全感的获得寄托在了别人身上**。安全感永远只能自己给自己。如果你无法自己制造安全感，就会经常处于匮乏状态。

## 第一步：与自己积极对话 🗣️

要给自己安全感，首先需要保持清醒，与自己进行积极对话。古人云“当局者迷，旁观者清”，这是一个哲学难题：我们很难跳出自我，以第三方视角客观评价自己。这会导致自我评价失衡，例如总是盯着比自己优秀的人，并抓住自己的小缺点不放，从而产生失望和焦虑。

以下是进行自我维护的关键方法：

*   **屏蔽负面心声**：主动过滤那些让你缺乏安全感的内心声音。
*   **培养“钝感力”**：这个概念由作家杜鞭纯一提出，意指**对外在的财富、身份、伴侣颜值等不过于执着，对外界反应不过于敏感**。公式化表达可以是：**安全感 ∝ 1 / 敏感度**。降低敏感度，能在很大程度上提升安全感。

如果说处理好与自己的关系是内在基础，那么下一步就是处理与外界的关系。

## 第二步：维护稳定的外部支持系统 👨‍👩‍👧‍👦

能够带来稳定关怀并分担风险的亲情与友情，是获得安全感的重要方式。它们可以与爱情形成“风险对冲”：你在爱情中遇到的问题，可以通过亲情和友情得到很大缓解。

然而，这些关系也可能出现问题：

以下是需要关注的两个问题及应对方法：

1.  **预期管理**：随着年龄增长，亲人对你的贡献预期会变化。如果未能满足，亲情可能出现嫌隙。朋友亦然，如果你们的思维、收入增长出现较大落差，交往的自由空间会变小。
2.  **保持同步**：因此，你需要努力保持自己与亲人、朋友的预期和成长步伐大体一致。处理好这两块重要的人际关系，你的安全感就有了着落。

建立了内外部的支持体系后，我们还需要提升自身的“硬实力”。

## 第三步：增强自身资源与能力储备 💪

我们常以为安全感缺失源于无法经营稳定关系，其实不然。真正让你焦虑的，是对未来丧失青春、吸引力和财富的恐惧，担心无法再获得青睐。

所以，核心是储备充足资源来维系吸引力：

*   **资源**：指**财富**和**结识的异性（社交广度）**。
*   **能力**：指**努力程度**、**挖掘财富的创意**、**外向型人格**和**社交倾向**。

**说白了，努力赚钱并建立丰富多元的社交生活，能有效减少焦虑，从而不再迫切寻求外在的安全感。** 代码式表达这一逻辑可以是：
```python
if 财富储备充足 and 社交生活多元:
    焦虑感降低
    对外在安全感的依赖减少
```

当自身变得强大，选择伴侣时也会更加从容和有底气。

## 第四步：建立清晰的择偶标准 ✅

最后，你必须构建自己选择伴侣的标准。心里要清楚自己想要什么样的人。否则，对每个遇到的人都全力以赴，会透支你的情感账户，导致焦虑和恐慌。

选择不当的后果是举步维艰，长期坚持可能仍是一场空，这种内心创伤是造成安全感匮乏的重要原因。所以，心里一定要有谱。

## 总结

本节课中，我们一起学习了如何在感情中构建安全感的四个步骤：**首先，通过积极自我对话和培养钝感力处理好与自己的关系；其次，通过管理预期和维护同步，经营好亲情与友情的外部支持系统；再次，通过积累财富和拓展社交来增强自身资源与能力；最后，建立清晰的择偶标准以避免情感透支。** 记住，安全感源于稳定的内心和强大的自身，而非他人。
