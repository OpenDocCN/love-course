# 乌鸦救赎《终极火箭班》第1课：目标的重要性 🎯

在本节课中，我们将学习乌鸦经典语录中关于“目标”的核心思想。理解并设定清晰的目标，是推动一切行动与改变的首要步骤。

有些时候你会感到无法推进，感觉再也推不动了。这时你需要采用“以终为始”的思维。

## 以终为始：先看到结果

上一节我们提到了前进的困境，本节中我们来看看解决方案：**以终为始**。这意味着在行动之前，你必须先在自己的脑海中清晰地“看到”最终想要达成的结果。

你必须明确知道自己要什么。这一点至关重要。

## 明确“你要啥”

以下是明确目标的具体方法，你需要向自己提出一系列问题，并给出清晰的答案。

当别人问你“你要什么”这个问题时，你必须能够非常准确、毫不犹豫地回答出来。

*   **人生目标**：我要成为什么样的人？
*   **生活目标**：我想要什么样的生活？
*   **工作目标**：我想要什么样的工作状态？
*   **收入目标**：我想要达到什么样的收入水平？

对于每一个问题，你都必须能给出非常明确、非常具体的答案。

## 建立自己的标准

你的答案必须非常清晰、非常坚定。无论外界如何变化或他人有何看法，你都需要建立并坚守自己的内在标准。

**核心公式：清晰目标 = 具体领域 + 明确标准**

---

本节课中我们一起学习了目标设定的核心：以终为始。关键在于，你必须无比清晰地知道自己究竟“要啥”，并在人生、生活、工作和收入等核心领域，为自己建立起明确、具体、个人化的标准。这是所有行动与成长的起点。
