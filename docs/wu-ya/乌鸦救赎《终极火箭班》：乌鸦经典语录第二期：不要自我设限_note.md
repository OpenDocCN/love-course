# 乌鸦救赎《终极火箭班》：乌鸦经典语录第二期：不要自我设限_note

在本节课中，我们将要学习一种至关重要的思维方式：如何打破“自我设限”的枷锁。我们将通过具体的例子和分析，理解限制性思维与创造性思维的区别，并学习如何将后者内化为自己的行动指南。

## 核心问题：两种不同的思路

面对同一个问题，人们通常有两种截然不同的思路。

以下是绝大多数人（约70%）的思路：

*   面对高消费场所（如夜店），首先想到的是“去不起”。
*   将高消费视为无法逾越的障碍，并为此寻找理由和借口。
*   最终结论是“等以后有钱了再说”，选择放弃或无限期推迟。

然而，成功者的思路则完全不同。他们的思维模式里根本没有“找理由”这个选项。

以下是成功者的思路：

*   面对“夜店需要一千块”这个事实，他们的第一反应是：“我怎样才能赚到这一千块？”
*   立即转向寻找解决方案，例如：“我一个人拿一百，找十个人不就一千了吗？”
*   核心信念是：总会有方法。关键在于你是否愿意去寻找。

## 彻底改写你的思维体系

上一节我们看到了两种思路的对比，本节中我们来看看如何将成功者的思路变成你自己的。

你必须彻底改写自己的思路。这种思维模式的转化需要时间和练习，你需要花精力将其内化，构建成自己的一套思维体系。

当这套体系建立后，你将发现：

*   面对问题时，你的大脑没有时间去思考消极的、做不到的理由。
*   你会坚信：任何问题都有解决方法。所谓“解决不了的问题”，全都是人类自我设限的结果。
*   人的可能性是无限的，因为世界本身就是一个可以被“假设”和创造的世界。

## 信念的力量：从想法到现实

理解了思维模式的重要性后，我们来看看信念能产生何种具体的力量。

一个人的信念强大到一定程度，它就会从脑海中“跳出来”成为现实。没有不可能，一切都可以进行“假设”并实现。

例如，我们提出的“戒指”想法，在三四个月前还纯粹是脑海中的一个概念。但通过坚定的信念和行动，半年后它已经成为戴在学员手上的实物。这证明了：**信念 → 行动 → 现实**。

## 设定目标：先有目标，后有方法

为了让信念落地，我们需要一个清晰的载体，那就是目标。

你必须时刻提醒自己坚持目标。一个有效的方法是将目标写在纸上、贴在墙上，每天抬头就能看见。

在设定目标时，最关键的是要**避免自我设限**。例如，本想设定“26岁开上牧马人”，却因自我怀疑改成了“28岁”。这种在设定目标时就打折的行为，本身就是设限。

关于目标设定，有一个核心公式：

**`目标` 决定 `想法` 的边界。**

*   当你定下“挣10块钱”的目标，你就会产生如何挣10块钱的想法。
*   当你定下“一年挣100万”的目标，你就一定会找到一年挣100万的方法。
*   方法永远存在，关键在于你敢不敢设定那个目标。

## 敢于冒险：改写人生的钥匙

设定了高远的目标，接下来就需要有与之匹配的勇气去执行，这常常意味着冒险。

永远不要自我设限，不要质疑自己能否做到。历史由人创造，成就源于自身的假设、选择和一系列行动。核心在于：**敢想，然后敢做**。

许多人连“想”都不敢想。敢想的人中，又有一大批人败在“不敢做”上，因为害怕风险。

你必须把“冒险”也变成你的思维电路之一，做一个敢于冒险的人。原因如下：

*   **冒险是最好的话题**：它创造了独一无二的体验和故事。
*   **女人喜欢有冒险精神的人**：因为冒险精神代表着活力、勇气和可能性。
*   **冒险能改写人生价值**：每一次冒险都有50%的成功率。一旦成功，你的人生价值可能就此全部改写，如同“鲤鱼跳龙门”。

## 立刻行动：没有“准备好”这回事

最后，也是最重要的一步，就是将一切付诸行动。

不要自我设限，更不要等待。不要等“有钱了”再去做什么。例如，学员“幽郎”17岁，通过穷游（吃馒头咸菜、住帐篷）去过了很多地方。关键在于他想，并且立刻去做了。

“准备好”是一个伪命题，它会将你限制至死。回顾我的许多第一次——第一次讲课、第一次去夜店、第一次接待客户——没有一次是“准备好”的。都是“昏昏碌碌就上来了”，但最终，也都做到了，成了。

**行动公式是：`想` → `做` → `成`。** 等待“准备好”只会让第一步永远无法开始。

**本节课总结**

本节课中，我们一起学习了“不要自我设限”的核心思维。

1.  我们首先**识别了问题**：对比了找借口的限制性思维与找方法的创造性思维。
2.  接着我们明确了**解决方案**：必须彻底改写并内化创造性的思维体系。
3.  我们认识了**信念的力量**：强大的信念能将假设变为现实。
4.  我们掌握了**设定目标的秘诀**：先定目标，方法自来；设定时要坚决，避免打折。
5.  我们鼓起了**行动的勇气**：要敢于冒险，冒险是突破和话题的来源。
6.  最后我们坚定了**行动准则**：立刻行动，永远不要等待所谓的“准备好”。

记住，你的可能性是无限的，限制只存在于你的脑海中。从今天起，停止设限，开始创造。
