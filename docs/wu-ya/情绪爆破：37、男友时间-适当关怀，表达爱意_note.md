# 情绪爆破：37、男友时间-适当关怀，表达爱意_note

在本节课中，我们将要学习情绪爆破系列课程的最后一个核心技巧：如何在长期关系中通过适当的关怀和表达爱意来经营感情。这是维持“男友时间”吸引力的关键。

上一节我们介绍了长期关系维护的重要性，本节中我们来看看具体如何操作。

## 核心原则：张弛有度

虽然对方已是你的女朋友，但关怀必须适度。关怀过度，她会感到被监管；关怀不足，她会觉得你不在乎。因此，掌握“适当”的度至关重要。

## 每日联系规范

为了帮助你掌握这个度，以下是一个基于多年经验总结出的具体操作指南：

*   **核心规则**：每日主动发送的信息不宜超过**20条**。
*   **原理**：超过此数量，你的需求感会暴露得过于明显，无论对方是否是你的女友，过度暴露需求感都可能让关系变得有压力。
*   **例外情况**：如果对方主动关怀、主动找你，则不受此限。这反而是她爱意的表现。
*   **换位思考**：试想如果她每天给你发超过20条信息，你的感受如何？恋爱需要给予彼此呼吸的空间。

## 实践方法

表达爱意和适当的关怀，都应包含在上述的每日20条信息之内。那么，如何维持日常的联系和吸引力呢？

关键在于，使用“无意义的废话”来维持你们之间的连接和吸引力。这些看似无关紧要的日常分享，正是维系感情温度的重要纽带。
