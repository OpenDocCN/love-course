# 乌鸦救赎海峰情绪爆破课：16、安全时间-收集信息 📝

在本节课中，我们将要学习在“安全时间”阶段需要做的第二件核心事情：**收集信息**。我们将探讨两种常见的错误聊天模式，分析其弊端，并学习如何通过有效收集信息来建立联系感、避免后续的升级障碍。

## 两种错误的聊天模式

上一节我们介绍了安全时间的概念，本节中我们来看看在此阶段常见的两种错误沟通方式。理解这些误区是掌握正确方法的第一步。

以下是两种典型的错误聊天模式：

1.  **查户口式**：这种模式通过连续提问来收集信息，例如“你在干嘛？”、“你多高？”、“你多大？”。它的问题在于提问过多，缺乏互动，无法建立吸引。
2.  **过度耍贫嘴式**：这种模式聊天内容特别油嘴滑舌、显得轻浮。它用“耍贫嘴”来代替真诚的交流。

这两种形式都不可取，都聊不好天。其根本原因在于没有掌握真正有效的收集信息手段和方法。

## 正确的信息收集方法

了解了错误模式后，我们来看看应该如何正确地收集信息。关键在于，收集信息后必须立即加入一个**互动**。

具体方法是：在通过提问（例如“升高”）收集到一条信息后，紧接着加入一个基于该信息的互动评论。

例如，在对方透露某个信息后，你可以回应：“这么长的腿可以玩很久。”这就是一个简单的互动。这个步骤将单纯的问答转变为有来有往的交流。

## 为何收集信息至关重要

你可能会问，为什么在安全时间必须持续做收集信息这件事？因为收集信息是建立**联系感**的重要一环。

如果没有在安全时间充分收集信息，会导致一种常见情况：对方可能看似对你有好感（有信用），但会在事后反悔。

这里涉及一个核心概念：**ASD**。ASD，即**反荡妇机制**，通常在最后你试图升级关系时出现。它出现的主要原因，就是你在安全时间阶段并不真正了解对方。

因此，如果严格按照课程步骤执行，在此阶段收集足够多的信息，对对方有充分了解，那么你基本不会遇到ASD的情况。公式可以简单理解为：

**充分的信息收集 → 深厚的联系感 → 有效降低ASD出现的概率**
