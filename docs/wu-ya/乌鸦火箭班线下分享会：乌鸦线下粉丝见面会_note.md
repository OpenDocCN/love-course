# 乌鸦火箭班线下分享会：乌鸦线下粉丝见面会 🚀

## 概述
在本节课中，我们将学习如何从“经营话术”转向“经营价值”，探讨物质力量在社交与人生中的核心作用，并构建一套以资产和技术为核心的成长战略。我们将深入理解人性、需求与价值创造之间的关系，帮助你找到并优化自己的人生模式。

---

## 第一部分：从话术到价值——两种策略的本质区别

在社交与个人发展中，存在两种截然不同的策略路径。上一节我们概述了课程核心，本节中我们来看看这两种策略的具体区别。

一位早期通过微商赚取上亿财富的年轻朋友曾与我探讨。他提出一个核心观点：我所经营的策略，最终落地方案都在“说话”。他承认，他的策略与我的有相似之处，但根本区别在于：**我用策略去经营某种“话术”，而他用策略去经营某种“物质”**。

例如，做微商的人非常注重“门面”——豪车、海景别墅、为团队经理提车等。这些物质展示所释放的力量，远大于任何精妙的言语。其中心思想是：**你应该用策略去经营一种物质力量，而不是仅仅经营你的说话技巧**。

如果只经营话术，最终可能陷入空虚。就像公开课常讲的：如果四五十岁还在玩弄说话技巧，会显得很低级。年轻时运用技巧获取认可无可厚非，但年纪增长后，核心应转向价值积累。否则，可能像村口闲聊的老大爷，停留在浅层社交，人生难以突破。

很多人将“物质”与“金钱”简单挂钩，并产生排斥。但以我个人经历而言，出身平凡甚至低于平均水平，只要敢于折腾、不放弃，在中国依然有很多机会获得自己想要的东西。这位90后朋友基础并不雄厚，但他通过策略经营“价值”，而我们过去则通过策略经营“话术”。这就是根本区别：**你经营的东西如果是虚的（话术），他经营的东西是实的（价值）。实的东西可以积累，虚的东西则可能随时间贬值**。

## 第二部分：核心战略框架——成为时间的主人 ⏳

理解了策略的转向，我们需要一个清晰的框架来指导行动。本节我们将构建一个以“资产”和“技术”为核心的战略体系。

### 1. 人生只有一次：时间的稀缺性与精力管理

“人生只有一次”这句话常被提及，意在提醒珍惜时间。我是一个深感时间不够用的人。我每日的例行安排包括：
*   自制健身餐（控制饮食）
*   健身
*   护肤
*   学习摄影（已进入PS阶段）
*   计划学习素描与音乐

这些安排源于一个认知：我们是玩社交的，未来工作必然涉及社交与资源整合。一个只会谈生意的“油腻大叔”形象是乏味的。你需要才艺和“附加值”来丰富社交氛围，提升个人魅力。因此，艺术修养是规划的一部分。

但学习这些需要大量时间。与此同时，为了达成更高的人生成就，我还需要学习战略、股权等商业知识。这些领域的复杂程度不亚于情感社交技巧。

**核心矛盾在于：时间有限，且每日高效精力（黄金时间）仅约3-4小时。** 过了这个时段，注意力难以集中。因此，必须极度珍惜时间，避免将其浪费在如刷抖音这类碎片化、浅层的信息消费上。任何领域的精通都需要**系统性学习**和老师引领。

年龄增长也带来紧迫感。我年近三十，常感恐慌，因为距离预期目标仍远。**人生只有一次，且高效时间窗口正在收窄。**

### 2. 人生是一场自我认识的旅途 🧭

奋斗、学习、塑造价值，但终有一天可能发现一切成空。我在亲生父亲去世时，捧着骨灰顿悟：人奋斗积累的房子、车子等，在生命终结时都失去了关联。那一刻，我真实体验到何为“人生如梦”。

科学推论显示，人类与地球的出现是偶然，所谓“人生意义”或许都是自我构建的阶段性目标。认识到这一点曾让我苦闷，觉得奋斗终将归零。

但后来我明白了：**人生是一场自我认识的旅途**。真正的意义不在于外在目标，而在于内在的发现与成长。这并非易事，需要内在觉醒。

以下是我自我认识过程中的几个关键层面：

*   **信心层面**：我曾非常自卑，总在搞砸事情。早期创业时，合作伙伴（小吉）在网站即将引流时失联，卷走了粉丝集资的几千元（在当时是巨款）。我陷入困境，但强大的“不能死在这里”的信念驱使我去寻找他。最终在一个偏僻小店找到他，并接手了烂摊子。**那是我人生第一次独立面对并解决麻烦，建立了“我能平事”的核心自信**。这种能力后来不断强化，能解决更复杂的人际和资金问题。
*   **思维层面**：后期我才意识到自己掌握了许多独特的思维方式。社交中，我能快速预判对方的预判，并制定应对策略。这是一种历练出的**社交直觉**。认识人性，首先要承认自己也有“混蛋”和阴暗的一面。只有认清自己，才能理解并预防他人可能的行为。
*   **情绪与欲望层面**：
    *   **情绪**：我曾脾气暴躁，管理方式简单粗暴（靠骂）。认识到自己愤怒的弊端但难以改变后，我选择聘请一位更有耐心的副总来负责管理。**这是基于自我认知（自知之明）而采取的高效解决方案**。
    *   **欲望**：曾认为贪财好色是错误状态。但现在认识到，**欲望（如对车、成功的渴望）是前进的根本动力**。就像某足球明星为买车而努力踢球一样。保持欲望、保持战斗状态很重要。我父亲曾很强势，但晚年因健康问题（肌肉流失导致睾酮素下降）变得怯懦。这让我明白，**战斗勇气与身体激素水平相关**。保持雄性特征（对异性吸引力、好胜心）是重要的生存本能。

**公式：自我成长 = 发现“我不能” -> 证明“我可以” -> 巩固并拓展能力边界**

综上所述，人生的旅途本质是不断认识自己：认识自己的思维模式、情绪弱点、欲望动力，并找到与之相处或利用之道。认识世界是为了更好地认识自己。

### 3. 用物质认识灵魂：重新定义物质与人性

在中国，“物质”常被视为贬义词。但以我的经验，**在所有社交策略中，物质是最稳定、最有效的杠杆**。它的效果是“一对N”的，而话术往往是“一对一”的。

**核心观点：用物质去认识灵魂（人性）。**

*   **物质过滤低效社交**：有一定经济实力的人，通常更不在意琐碎得失（如谁请吃饭），相处更顺畅。他们不傻，遵循“天下没有白嫖”的规则。你若没钱，就必须成为“高分”（高价值者），变得优秀，融入圈子。拥有高质量朋友圈后，人生麻烦会减少。
*   **物质揭示人性层次**：不同物质层次，你会遇到不同的人性和麻烦。在低纬度，你看到的重复于“吸引与否”、“是否暧昧”。只有当你自身价值提升，进入更高圈层，才能见识更复杂、更深刻的人性博弈（例如涉及巨大利益时的冲突）。**物质是打开不同层次人性世界的钥匙**。
*   **摒弃传统偏见**：中国历史上“士农工商”的等级观念（抑商）深植人心，导致对“露富”等的恐惧。但对于像我这样的个体户，上面没有直接管辖的“大哥”，适度展示实力（如开好车）并未带来麻烦，反而常有意外惊喜。**不要活在幻想的恐惧中，要用勇气打破思想枷锁**。我们的原则是：不幻想敌人多强，先交手试探。打不过就认输或加入，这反而可能是建立联系的契机。
*   **总结**：通过提升物质价值，你会遇到更大的麻烦，但也因此能认识更深刻的人性。当前的许多人际困扰（如背后坏话）相比之下微不足道。真正的挑战（如涉及千万利益的纠纷）需要你在达到相应层次后才能面对和处理。

### 4. 做人性的朋友：利用规律驱动自己

认识到人性的规律后，关键是如何利用它，尤其是驱动自己。

*   **接受“花钱解决问题”**：早期我也希望低成本办大事。但后来发现，这会导致你永远在低层次打转。**能花钱解决的事，就不动脑（指纠结）**。这会产生成本，但也倒逼你去赚钱。就像C罗想买豪车，所以努力踢球赚钱。当你认同“花钱是最好的办法”时，你才会更努力地赚钱，思维变得更现实。这是人性的驱动循环。
*   **吸引力法则与自我逼迫**：当我意识到需要为健身、美容等付费时，我就知道自己必须学习商业赚钱。**脑中有了“花钱-赚钱”的概念，注意力就会自动过滤相关信息，找到机会**。就像想买牧马人时，满街都能看到它。
*   **利用惰性，倒逼成长**：人性有惰性。健身时，自己练容易偷懒，请教练则能在力竭时助推，迫使身体突破极限，生长肌肉。**这个逻辑普遍适用：人需要时常将自己置于“绝境”**。例如，把钱花完，逼自己赚钱。每天鼓励自己没用，濒临“绝境”才能激发真正潜能。**最厉害的套路应用在自己身上**：理解别人（或销售）如何被激励，反过来用在自己身上，倒逼自己行动。

## 第三部分：构建个人模式——从学习到创造 🔧

掌握了核心战略，我们需要将其转化为个人可执行的系统。本节我们将探讨如何找到并优化属于你自己的人生算法。

### 1. 时间的主人是“资产”

巴菲特的核心投资逻辑是：做时间的朋友。但更准确地说，**时间的主人是“资产”**。在时间长河中，唯一不易贬值的是资产（或贬值较慢）。外表、话术也是资产，但贬值速度更快。

**关键是要分清：哪些东西随时间增值（或保值），哪些随时间消逝。** 青春是不可再生资源，必须珍惜。塑造价值的方向也有讲究，应聚焦于那些“越战越强”的价值，例如商业能力、知识资产，而非仅注重短期外表。

**战略与战术的关系**：选对战略点（如“应许之地”），任何战术（大情圣、连招）都可能成功。**战略正确后，战术可以多样化**。在我们的体系中，**塑造那些随时间增值的价值就是正确战略**。商业能力可以积累，而把妹技巧则有年限。**资产是时间的主人，拥有增值资产，你就是时间的主人**。结果将是：岁月让你更成熟，而非更落魄。

### 2. 物质不应被唾弃：一种科学主义视角

我们对物质的推崇并非拜金主义，而是基于观察和验证的**科学主义**。在探究人性过程中，物质是效力最广泛、最持久的杠杆。我学过催眠，但它像钢琴，要求高且需对方配合；而进化心理学、迷男方法等，不要求对方配合，更能达成目标。

宇宙由物质构成，这是最高级的自然规律（天道）告诉我们的。**追求物质效能最大化，是符合“天道”的智慧**。物质不仅指金钱，也包括用金钱转化的形象、技能等。

关于分享：不必逢人便讲此理念。实际结果成功的人（如大老板）未必理解得比我们深刻。在社交展示上，当我摄影技术提升后，我发现**高质量照片本身就有吸引力，配文（套路）反而不再重要**。这再次印证：**物质（高质量展示）到位，套路可省**。

### 3. 用物质驾驭人性：出牌顺序的转变

我那位微商朋友的核心思想是：**用物质替换话术，并以此决定你的出牌顺序**。

例如，与一个女孩聊天，我直接分享我拍的高质量照片，引发对方极大兴趣和邀约。我出的牌是“摄影技术”这项物质价值。未来，我还可以出“吉他”、“素描”、“商业认知”等牌。**思考重心从“我该说什么话”转变为“我该出哪张价值牌”**。

对于初学者，问题可能是“我没有牌”。**答案：塑造价值就是你的功课**。要经常性塑造价值，偶尔性打牌。高级别的社交关系（如我现在的设计师女友），绝不可能是单纯靠话术拿下的，它是我价值提升的“验证点”。这会激励你塑造更多“牌面”。

学习是常态，但很累。建议：
1.  **先学习“如何学习”**：阅读《如何阅读一本书》、《刻意练习》等，提升学习效率。
2.  **用欲望牵引学习**：例如，学摄影就找美女模特，激发潜意识的动力。理解自己的潜意识（欲望），并顺应它来设计学习路径。
3.  **保持精力**：学习消耗大，需要冥想等方式快速恢复精力。

当你拥有多项价值指标（颜值、身材、摄影、商业认知等），与人互动时，你不再依赖套路，而是根据对方需求打出相应的价值牌。**你变成了稀缺资源，是别人想方设法来搞定你**。这就是 **“用物质驾驭人心”** 的终极状态。

至此，对人性的理解可以简化为：**人性永远追随物质。因此，只需专注于理解和驾驭物质。**

### 4. 战略：聚焦资产与技术

“战略”指拿下某个关键点后，全局会发生根本性改变（从底层到高层，从追逐到被追逐）。

我们已透彻讨论：**你所有的奋斗战略点，都应设立在“物质”上**。而驾驭物质靠的是**技术**。世界上一切有价值的东西都由技术构成。

**公式：塑造价值 = 学习技术 -> 取得资产**

因此，未来的人生游戏规则是：**以学习技术为常态，以取得资产为目标**。社交将变为“验证点”——验证你价值高低的考场。搞定了，说明你行；搞不定，说明还需进步。

我的亲身经历：随着我自身价值（公司、资产）提升，我的社交圈自动从同学层，升级到打工层、老板层、投资人层，甚至政府官员层。**我什么都没做（指刻意社交），只是不断垫高自己，是他们来找我**。这印证了战略的正确性。

### 5. 找到自己的人生模式

每个人都有自己的成功“模式”，它像量身定制的算法。**没有“神之一手”**，不存在学会某一招就天下无敌。成功是多种能力（跨领域技术）的综合运作。

爬到金字塔尖极其困难（例如在深圳，拥有两套房的人数远少于拥有一套房的）。你需要不断拆解、学习、整合。**关键是要自信，并敢于出招**，尤其是敢于通过“烧钱”测试物质策略的效果。

我曾花近百万（房租+装修）租豪宅做私人会所，测试高端社交一年。虽然直接项目合作不多，但我验证了“通过物质展示整合资源”的有效性。灵感来源于卡萨诺瓦的故事。这次经历让我明白：**要告别单纯用话术出招，学会用价值（物质）出招**。

**花钱需要练习和境界**。不经历“败家”，就学不会“花钱”。早期创业者烧投资人的钱，也是一种驾驭资本、创造价值的练习。要有“花资本的钱，创造更大价值”的胆识和思维。

**建立个人模式的第一步是：相信自己的想法，并去测试它**。遇到挑战，先交手，在实战中调整。

### 6. 学习模式 -> 掌握模式 -> 创造模式

这是个人进化的三个阶段：
1.  **学习模式**：通过看书、找名师、观察他人，学习各种经营价值的策略。
2.  **掌握模式**：通过训练，将所学内化，化繁为简。
3.  **创造模式**：最终形成自己独特的一套打法。

**你的模式就是你的全世界**。我在情感行业觉得推广很难，但跨界到形象设计领域后，发现推广很简单。**打破自身模式，才能看到新世界**。这要求保持谦虚和好奇心，对新领域、新模式持开放态度。世界太大，未知太多。

### 7. 价值的核心：发现与满足需求

我们强调“创造价值”，但价值的背后是**需求**。**发现需求就是发现价值**。

*   **微观层面（人际）**：洞察他人个性化需求并满足，就是创造高价值。例如，发现女友虚荣，就在办公室送花；发现女孩喜欢家庭温馨，就为她做一顿饭。这带来“默契”和“深交”感，是高级的情商。
*   **宏观层面（商业）**：发现一个群体的需求，并提供完整的解决方案，就构成了一个价值体系。

**发现需求**和**提供满足方式**，这两者本身就是可以深耕的技术。掌握它们，你可以在众多领域畅行无阻。

### 8. 技术多元化：未来比拼的维度

对于已知的、已形成产业的技术（如PS、摄影、穿搭、连招），要**不遗余力地学习**。这是现成的价值创造工具。

未来，我们团队内部比拼的，不再是泡妞技巧，而是**谁掌握更多元化的技术**，更像一个“全才”。当然，要有重点：**以商业和赚钱能力为根本**，因为未来需要驾驭巨额资金。

当你价值足够高时，与其他仍停留在技巧层面的人交流，他们会无地自容。这才是我们追求的降维打击状态。

## 总结 🎯

本节课中，我们一起学习了从“经营话术”到“经营价值”的思维转变。核心要点包括：

1.  **战略转向**：用物质力量替代话术作为核心策略，经营可积累的资产。
2.  **核心框架**：人生是自我认识的旅途，需通过物质认识更深层的人性，并做“人性的朋友”来驱动自己。
3.  **行动纲领**：成为时间的主人，聚焦于随时间增值的资产和技术。物质是科学有效的杠杆，应用它来驾驭社交。
4.  **个人进化**：找到并不断优化自己的人生模式，经历学习、掌握到创造的过程。保持谦虚，打破模式边界。
5.  **价值本质**：价值的核心是发现并满足需求。要多元化学习已知技术，并以商业能力为根基。

**人生不易，马不停蹄**。要学的东西太多，每一种技术都需要时间攻克。认清方向，珍惜时间，即刻行动。
