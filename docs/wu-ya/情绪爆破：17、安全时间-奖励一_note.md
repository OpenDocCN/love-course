# 情绪爆破课程 17：安全时间-奖励一 🎁

在本节课中，我们将学习一个在沟通中非常经典且实用的技巧——“安全时间”中的“奖励”。这个技巧的核心在于通过给予对方认同和肯定，来强化积极的互动行为。

## 什么是奖励？ 🏆

奖励，简单来说，就是给予对方一个正向的回馈。这可以是夸赞对方，让对方感到开心，或者认同对方。有一个关键概念需要记住：**认同是最高级别的奖励**。理解这一点对掌握后续技巧至关重要。

## 为何要使用奖励？ 🤔

在深入探讨如何使用奖励之前，我们首先需要理解这样做的益处。我们可以通过一个生活场景来理解其重要性。

想象一下，当一个女生，或者任何人，为你做了一件她认为对你有好处的事情。例如，你的母亲看了天气预报，发现气温是15度。于是，在你出门时，她直接扔了一件很厚的羽绒服给你。作为一个20来岁、血气方刚的年轻人，你并不觉得冷，穿太多反而会感到难受，因此你并不想穿这件衣服。

此时，如果你不认同你母亲的这个行为，即你否定了她的好意。那么，当她下次再遇到类似情况时，她很可能就不会再这样做了，因为她从你这里得到了“不需要”的反馈。当她寻求你的认同时，你给予了否定，她便会停止这种行为。

## 如何实施奖励？ 📝

以下是实施奖励时可以参考的几个要点：

*   **及时给予正面反馈**：当对方做出你希望看到的行为时，应立即表达认可。
*   **具体化你的夸赞**：不要笼统地说“你真棒”，而是指出具体哪里做得好，例如“你这个想法很周到”。
*   **表达真诚的认同**：奖励必须发自内心，敷衍的夸赞反而可能产生反效果。

## 课程总结 📚

本节课我们一起学习了“安全时间”技巧中的“奖励”。我们明确了奖励的本质是给予认同，并理解了**认同是最高级别的奖励**这一核心观点。通过生活化的例子，我们看到了给予肯定如何能鼓励对方持续做出积极行为，而否定则会使其停止。在下一节课中，我们将继续探讨奖励的另一个重要技巧。
