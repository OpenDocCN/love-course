# 乌鸦救赎《终极火箭班》课程笔记 01：第一念练习 🧠

在本节课中，我们将学习“第一念练习”的核心概念。这个练习旨在帮助你识别并突破在采取行动前，内心产生的第一个、通常是阻碍性的念头。通过一个简单的实验，我们将直观地理解“第一念”如何影响我们的行为。

---

## 课程实验：突破“第一念”的束缚

现在我们可以做一个实验。请先确认你所在的位置：是在家里、网吧、宿舍还是其他地方？如果你所处环境中有其他人，请在心中默认为“有人”状态。

以下是实验步骤：

现在，请大家跟随我的指令进行测试。请喊出一声，不要用打字代替，需要你真的发出声音。不必在意周围是否有人。

有人表示不敢或害怕。这正是当你想做一件事时，你的“第一念”将自己困住的感觉，即一种受困于心的感觉。

关于公共环境的顾虑，其实无需在意。我当年在公交车站喊叫时，周围人都像看傻瓜一样看我，而我笑着离开，觉得他们才是傻瓜。

如果你喊了但声音很小，或者仍有犹豫，这说明你已经错过了最佳行动时机。刚才我发出指令时，若能瞬间喊出，那非常厉害。犹豫半天，意味着你内心的障碍没有突破。

被骂也无所谓。最关键的是，喊完之后，是否发生了你想象中的那些糟糕后果？即便最坏的情况发生，比如被骂了，又能怎样？你会因此而死吗？不会。这仅仅是喊一声而已。

试想，如果让你去搭讪呢？这就是你的“第一念”被控制住了，导致你无法行动。你会犹豫：“万一被拒绝怎么办？”但即使被拒绝，你也不会死。骂你是傻瓜？也没事。

最关键的是，我想让你亲眼看看最坏的结果究竟什么样。你会发现，其实也没事。你就这样去做了。

还有一个关键点：即便你按照他人的期望行事，别人也未必就不骂你是傻瓜。你做事非常有策略，但你的大脑无法计算世界上发生的所有事。即使在一个小范围内，你仍会出错，仍会有人骂你。但最关键的一点是，你自在吗？

刚才在有人环境下成功喊出来的人，请在心里为自己扣“1”。

## 核心概念解析

本节中我们介绍了“第一念”的现象，现在我们来深入看看其核心逻辑。

“第一念”通常是一个**阻碍性念头**，其公式可概括为：
**行动意图 → 第一念（阻碍：“如果…怎么办？”） → 犹豫/不行动**

这个念头的本质是**对负面结果的灾难化想象**，它夸大了行动的代价（如“会被嘲笑”、“很丢脸”），同时低估了你的承受能力（“无法忍受”）。

练习的目标是**切断“第一念”与“行动”之间的默认链接**。方法是通过实验验证，证明你恐惧的后果（用代码表示其虚幻性）：
```python
if 执行行动():
    实际后果 = 获取真实反馈()
    想象后果 = 第一念.预测的后果()
    assert 实际后果.严重程度 << 想象后果.严重程度  # 断言：实际后果远轻于想象
```

## 如何应用：日常练习指南

理解了概念后，以下是你可以立即开始的练习步骤，旨在将理论转化为习惯。

1.  **识别**：当想做某事（如发言、搭讪、尝试新事物）却感到阻力时，立刻捕捉那个最先冒出的念头（例如：“别人会怎么看我？”）。
2.  **评估**：快速问自己：“这个念头所预言的最坏结果是什么？它发生的概率有多高？即使发生，我真的无法承受吗？”
3.  **行动**：给自己一个极短的决策窗口（例如3秒），无视第一念，直接执行计划中的微小行动（如先开口说一个字、先迈出一步）。
4.  **复盘**：行动后观察结果。记录下实际发生的情况与你“第一念”的预测有何不同。通常你会发现，**现实远没有想象那么糟糕**。

## 本节总结

本节课中，我们一起学习了“第一念练习”。我们通过一个当众发声的实验，亲身体验了“第一念”如何阻碍行动，并分析了其作为“灾难化想象”的本质。核心在于，我们必须通过**直接行动**来验证并打破这种虚幻的恐惧链接。记住，行动的关键不是消除恐惧，而是带着恐惧依然前行。真正的自由，来自于意识到“即使被骂是傻瓜，你也依然完好无损”之后所获得的内在自在感。
