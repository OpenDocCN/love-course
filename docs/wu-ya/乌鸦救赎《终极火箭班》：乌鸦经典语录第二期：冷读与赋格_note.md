# 乌鸦救赎《终极火箭班》：乌鸦经典语录第二期：冷读与赋格_note

在本节课中，我们将要学习“冷读”与“赋格”这两个核心社交技巧。它们主要用于建立舒适感、深化联系，并引导对方形成你所期望的人格特质。理解并运用这两个概念，可以有效提升你在人际互动中的影响力。

## 冷读的作用

上一节我们介绍了课程主题，本节中我们来看看冷读的具体作用。冷读最大的作用是能够建立舒适感。A3阶段的末尾，正是开始建立舒适感的关键节点。

冷读的作用是让对方感觉你非常了解他。人际关系往往从分享秘密开始，冷读能建立一种唯一性，让对方认为只有你真正理解他，而其他人并未认真了解过他。这会使你显得洞察力极强。

冷读还能让陌生人迅速产生一种你们已经认识很久的熟悉感与联系感。这是建立舒适感的一个必要阶段。

## 什么是赋格

理解了冷读如何建立初步连接后，接下来我们需要巩固并引导这种关系，这时就需要用到赋格。赋格是“佩服效用”的体现，其核心是赋予对方一种人格。

以下是赋格的具体做法：
*   你是一个温柔善良的女孩。
*   你是一个孝顺的女孩。
*   你是一个对男朋友好的女孩。
*   你是一个愿意听话的女孩。
*   你是一个乖乖女儿。

进行赋格时，必须选择对自己有利的特质。例如，“你是一个非常开放的女孩”。假设的能力是无限的，当你持续进行某种假设时，事情的发展往往会朝你假设的方向演变。

通过赋格，你可以塑造对方。本来他不是这样的人，但经过反复赋格后，他可能真的会成为这样的人。这类似于量子力学中的“观察者效应”：被观察的事物会随着观察者的心态而转变。研究员的心态会影响事物本身的变化。

例如，许多原本很乖的女孩，通过赋格可以被塑造成非常开放、甚至愿意尝试SM的人。在正常交往中，她可能像女神一样令人不敢亵渎，但当你给她贴上特定的人格标签后，她真的可能朝那个方向转变。

## 冷读与赋格的结合运用

我们已经分别了解了冷读和赋格，现在来看看如何将两者结合，发挥最大效力。成功的冷读可以作为赋格的证据，使你的赋格更具说服力和影响力。

其逻辑链条是：**冷读建立信任 -> 信任作为证据 -> 证据支撑赋格**。

例如：
1.  冷读：“我感觉你是一个很愿意冒险、具有尝试新鲜事物能力的女孩。”（对方认可）
2.  赋格：“那么，我感觉你这个人特别容易跟别人发生一夜情。虽然你现在可能还不是，但我感觉你已经有这个趋向。”

此时，冷读的结论（“愿意冒险”）成为了后续赋格（“容易发生一夜情”）的有利证据。

你可以配合一些肢体语言进行暗示，比如在说到“发生一夜情”时，手指轻轻拂过自己的胸口。这种暗示积累时间长了会特别有效。

同样，你可以赋格：“我感觉你在找寻一个男神来征服你。”同时配合指向自己的暗示。长此以往，对方每当想到相关问题时，首先联想到的就会是你。

本节课中我们一起学习了冷读与赋格。冷读用于快速建立了解与舒适感，而赋格则是在此基础上，为对方赋予有利于你的人格标签。将冷读作为赋格的证据，可以使你的引导更加自然有力。记住，假设和暗示拥有塑造现实的力量。
