# 情绪爆破：34、爆破时间-引导投资_note

在本节课中，我们将要学习情绪爆破系列的最后一项核心技巧——引导投资。我们将探讨如何在吸引力达到顶峰后，通过引导对方进行投资来有效维持吸引力，从而稳固长期关系。

上一节我们介绍了如何在爆破时间将吸引力推至顶峰。本节中我们来看看，当吸引力达到饱和后，如何防止其“蒸发”并长期维持。

吸引力达到顶峰后，便无法继续增加。此时所有后续操作的目标都是维持吸引力。若维持不当，吸引力便会下降。这就像一杯水，装满后无法再注入更多，只会慢慢蒸发。引导投资正是维持吸引力的经典且高效的方法，堪称“必发技”。

当对方已成为你的伴侣时，需要开始引导其为这段关系投资。只有在对方付出后，吸引力才能得以保持。这听起来或许有些策略性，但为了关系的长久稳固，这是必要的步骤。若不引导投资，不促使对方发自内心地爱你，关系便难以持续。

以下是引导投资的具体方法：

例如，在520这类节日，许多男性会给女性发红包。但更有效的方法是进行双向引导。一位优秀学员的做法是：他先给女生发一个52元的红包，然后问：“我有礼物吗？”结果女生回赠了他一个520元的红包。通过类似互动，他当日收到了上千元。这是一种简单有效的金钱投资引导。

如果不好意思直接引导金钱投资，可以引导对方投入精力或时间。同样在特定日子，可以要求对方：“今天多陪陪我，我们一整天都在一起。”这便是一种时间投资。

此外，日常生活中的小事也可以成为引导投资的契机。例如：
*   让对方帮你倒杯水。
*   让对方帮忙打扫家里。
*   让对方为你做一顿饭。
*   让对方帮你挑选衣服。

当这类引导投资的行为积累到一定数量后，你的吸引力便能得到长期维持。对方会产生“得失心”，因为已经投入了太多（时间、精力、金钱、情感），从而变得难以离开你。

本节课中我们一起学习了引导投资这一维持吸引力的关键技巧。我们明白了在吸引力饱和后，需要通过引导对方在金钱、时间、精力等方面进行投资，来制造对方的“得失心”，从而长期稳固关系，防止吸引力流失。
