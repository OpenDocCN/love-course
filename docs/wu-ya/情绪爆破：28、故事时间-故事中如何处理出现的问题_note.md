# 情绪爆破：第28课 - 故事时间：如何应对故事中的问题 🧩

在本节课中，我们将学习在分享情感故事时，如何处理对方可能提出的、看似否定或刁难的问题。我们将看到，这些问题并非障碍，反而是推进关系和深入情感话题的绝佳机会。

上一节我们介绍了分享情感故事的基本概念，本节中我们来看看当故事分享过程中遇到特定类型的回复时，应如何巧妙应对。

有些情感故事的切入点并不明显，难以分辨。这通常是因为缺乏足够的聊天经验和能力，导致无法判断某个话题是否可以顺利引入情感故事。例如，对方可能会给出一些类似于“兴趣声明”的反馈。

以下是几种常见的、看似否定实则蕴含机会的回复示例：
*   “你是不是经常叫别人宝贝？”
*   “你这个渣男。”
*   “你是不是个海王？你一看就是个海王。”

这类对方给予的反馈，表面上是一种否定。但实际情况是，每一次否定或所谓的“废物测试”，都是一个极好的、能够引导出情感话题并推进双方关系的“引子”。

正如前文强调的“引子”概念，上述那些话，本质上就是一个可以证明自己、并开启分享情感话题的入口。关键在于如何利用这个否定作为引子。

如果我遇到上述类型的回复，我会这样利用其作为开启情感话题的引子。例如，当对方说“你这个渣男”时，我的回应可以是：

> “跟我那个该死的前女友一样的口吻。”

请仔细体会这个回应。它是否成功地引出了“前女友”这个话题？这充分运用了“万能前女友”这个我经常分享的观念。接下来，如果对方对此感兴趣，自然会追问：“你前女友怎么了？” 从而顺利开启更深层的情感故事分享。

本节课中我们一起学习了如何将对话中看似负面的回复转化为开启情感故事的契机。关键在于识别“引子”，并运用如“前女友故事”等技巧，将话题自然导向情感分享，从而深化连接。下节课我们将继续探讨相关技巧。
