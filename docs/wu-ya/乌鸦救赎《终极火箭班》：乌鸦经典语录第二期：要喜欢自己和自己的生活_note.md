# 乌鸦救赎《终极火箭班》第2课：要喜欢自己和自己的生活 🚀

在本节课中，我们将学习乌鸦关于自我认同与生活态度的核心观点。核心思想是：对自身和生活的积极感受，是驱动个人改变与成长的内在动力。

上一节我们介绍了建立积极心态的重要性，本节中我们来看看如何具体地“喜欢自己和自己的生活”。

当你讨厌自己的生活时，你就不愿意去感受和体验生活。思考一下，一个人在什么情况下才愿意分享某样东西？如果连我的微信名片你们都不愿意分享出去。

这是因为你觉得分享这件事物不会让你感到自豪，反而可能让你担心别人认为你很猥琐。但如果你感受到某样东西非常棒，并且你有强烈的意愿去分享它，那么请思考，你的生活是否也是如此。

**核心逻辑**可以表述为：
**强烈的分享意愿 ≈ 对生活的高度认同**

如果你的生活确实很出色，并且你有强烈的分享意愿，那么积极的自我引导就会自然形成。我们每个人的基本心理机制其实都相似：不要讨厌自己。人不可能通过讨厌自己而真正改变，除非你先开始喜欢上自己。

这就像圣经中“马太效应”所阐述的道理：穷的会更穷，富的会更富。难道这个天平就无法被打破和控制吗？关键在于你从何时开始感觉到自己是“牛逼”的。

当你开始感觉自己很棒时，你会发现生活进入了一个正向循环。你的生活轨迹本质上是由你的思维塑造的。马太效应确实存在，但世上本没有天生的“屌丝”。

以下是基于上述观点整理出的行动要点：

1.  **停止否定**：立即停止对自己和生活的讨厌与抱怨。
2.  **寻找亮点**：主动发现生活中让你感到自豪或“牛逼”的细节，无论多小。
3.  **练习分享**：从分享这些小的积极感受开始，逐步建立自信。
4.  **思维引导**：有意识地用“我喜欢……”的句式替代“我讨厌……”，引导思维转向。
5.  **接纳过程**：改变需要时间，接纳当下的自己，是走向理想自己的第一步。

本节课中我们一起学习了喜欢自己和自己的生活的重要性。我们明白了，积极的自尊和分享欲是启动正向人生循环的关键开关。改变始于自我接纳，当你开始欣赏自己，你的世界才会随之改变。记住，你的思维塑造了你的现实。
