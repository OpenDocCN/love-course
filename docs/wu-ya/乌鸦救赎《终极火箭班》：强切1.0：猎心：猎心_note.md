## 概述

“猎心”技巧的核心在于，通过重复和引导性的语言模式，在对方表现出犹豫或拒绝时，持续施加温和的吸引力，并最终将对话导向你设定的目标框架中。这并非简单的坚持，而是一种有策略的对话推进方式。

上一节我们介绍了课程的整体目标，本节中我们来看看“猎心”技巧的具体对话实例。

## 对话范例分析

以下是一段展示了“猎心”技巧的典型对话。请注意对话的推进节奏和语言模式。

```
A: 你真的不想和我一起去？
B: 不想。
A: 但我真的想和你一起去，我們去哪裡？
B: 去你家，到處走走，走…
A: 我真的想和你一起去，但我…
A: 我真的想和你一起去，走…
A: 我真的想和你一起去，走…
（后续为“我真的想和你一起去，走…”的多次重复）
```

## 核心技巧拆解

现在，我们来拆解这段对话中运用的关键技巧。

### 1. 接受拒绝并转换框架
对话始于一个可能被拒绝的问题“你真的不想和我一起去？”。当对方回答“不想”时，操作者没有争辩或放弃，而是立刻用“但”进行转折，接上“我真的想和你一起去”，将话题从“是否同意”转换到了“去哪里”的行动框架上。其公式可以概括为：

**接受否定 + 表达自我意愿 + 提出新问题**

### 2. 使用重复进行引导
在对方给出一个模糊的回应“去你家，到处走走，走…”之后，操作者开始重复核心句“我真的想和你一起去，走…”。这种重复起到了以下作用：
*   **强化意愿**：不断强调“我想”，传递出坚定和自信。
*   **填充对话**：避免冷场，保持对话的动能。
*   **施加潜意识影响**：通过重复，让“一起去”这个想法在对方脑海中不断加深。

以下是重复模式的核心代码逻辑：

```python
while not agreement: # 当未获得同意时
    print("我真的想和你一起去，走…") # 重复核心引导句
    # 持续施加温和的压力并等待对方反应
```

### 3. 引导行动指令
在重复的语句中，始终伴随着“走…”这个简单的行动指令。它非常具体且易于执行，不断暗示对方做出“移动”或“行动”的决策，将对话从语言层面推向实际行动层面。

## 技巧要点总结

以下是运用“猎心”技巧时需要注意的几个要点：

*   **保持语气平和**：整个过程应保持轻松、自信的语气，而非乞求或强迫。
*   **忽略次要抵抗**：对于对方犹豫的“但我…”等话语，不必深入纠缠，继续推进你的核心引导句。
*   **专注最终目标**：所有对话都应服务于将对方引导至你预设的“行动”或“地点”目标上。

## 总结

本节课中我们一起学习了“强切1.0：猎心”技巧。我们了解到，它的关键不在于说服，而在于**引导**。通过“接受拒绝-表达意愿-重复引导-发出指令”这一系列步骤，可以有效地掌控对话节奏，将互动导向你想要的方向。记住，自信的重复和明确的行动指引是此技巧的核心。
