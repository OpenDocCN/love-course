# 乌鸦救赎《终极火箭班》第1课：自我引导训练 🚀

在本节课中，我们将学习“乌鸦经典语录”中的核心概念——自我引导训练。我们将探讨为何需要重新学习说话，以及如何通过改变习惯来建立自信，从而在与异性互动中获得更好的反馈。

## 重新学习说话的必要性

你可能真的是在重新学习说话这件事情。

如果你一旦入了“连招”这个门派，那么泡妞是假门派的。如果你一旦入了“连招”这个门派，那么你可能面临的一个问题是要重新进行说话的学习。

我真的就是在一开始时候不要去想太多。我真的没有想过一个人看完连招之后，他立马就可以要去说话非常自如，在女人面前想说什么就说什么。那个是需要花时间来磨练的。

**核心概念**：掌握“连招”不是一蹴而就的，它需要时间和练习来内化。

---

## 改变说话习惯的第一步

所以第一步你先要改变自己的说话，需要变成什么？就**从以前的提问变成不断陈述自己**。

我知道兄弟这个事情会很别扭，因为我在改变你的习惯，我在改变你的说话习惯。这个事情会让你感觉很别扭，但是这是必要做的一件事情。

你只有先往这个方向去走，然后你才能够说是往这个方向走的非常漂亮。我们先得敢走，敢做比会做是更重要的一件事情。因为有很多人是想要先会做然后才敢做，这个理由是不成立的。很多时候其实你都是没有准备的。

**核心公式**：**行为改变 = 停止提问 + 开始陈述**

上一节我们探讨了重新学习说话的必要性，本节中我们来看看具体如何迈出第一步。

以下是改变初期需要理解的两个关键点：

*   **从提问到陈述**：将对话模式从寻求答案（提问）转变为表达自我（陈述）。
*   **敢做优于会做**：在完全准备好之前就开始行动，行动本身比等待完美的准备更重要。

## 应对初期的负面反馈

所以在一开始的时候，你会接受到大量的负面信息和负面反馈，女人给你不好的回应。

**核心代码**：
```plaintext
初始阶段 -> 行动 -> 收到负面反馈 -> 调整并坚持 -> 逐步获得正面反馈
```

## 总结

本节课中我们一起学习了自我引导训练的基础。我们认识到，进入“连招”门派意味着需要重新构建我们的沟通方式，核心是从被动的提问者转变为主动的陈述者。这个过程初期可能会伴随不适和负面反馈，但“敢做”比“会做”更重要。只有通过持续的练习和调整，才能逐步建立自信，实现流畅自如的交流。
