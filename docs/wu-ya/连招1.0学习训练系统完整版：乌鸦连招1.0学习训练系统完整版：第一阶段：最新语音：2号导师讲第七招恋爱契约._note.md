在本节课中，我们将要学习“恋爱契约”这一招。这一招主要用于在成功确立恋爱关系后，巩固双方感情，为未来的稳定发展打下基础。我们将重点讲解如何通过规划未来、安排约会以及锁定关系进度，来增强女生的安全感和对这段关系的信心。

## 恋爱契约的作用

上一节我们介绍了中断结纱（确立关系），本节中我们来看看如何巩固关系。恋爱契约是在双方成为男女朋友后使用的招式。

它的核心作用是补充女生在恋爱中的安全感。这种安全感与第五招“建立联系感”所提供的不同。第五招的安全感是让女生知道你是一个没有恶意、值得信赖的朋友，过程侧重于相互了解和信息交换。

这区别于初中生那种儿戏的告白（例如：“我好喜欢你，当我女朋友好不好”）。我们要让女生感受到，我们是真心想和她共度未来的每一天。

让女生感觉到你的真心，她才会以真心回应。这样双方才能拥有一段美好而长久的恋爱关系。如果在确立关系后还玩弄套路和感情，关系难以长久，内心也会不安。

这一招除了表明真心，还要让女生知道你的未来规划中有她的存在。只要她相信你的未来生活里有她，她就会觉得当前关系非常稳定。

你可以和女生谈论未来，例如十年后一起买房买车。虽然你没有直接说“我爱你”，但她会因此感到安心，觉得和你在一起有未来。

同时，女生也会认为你是一个有上进心的男人。没有人愿意跟随一个整天抽烟、喝酒、打游戏、没有未来的男人。

**核心公式**：`恋爱契约 = 表明真心 + 规划共同未来 + 增强安全感`

因此，恋爱契约就是让女生知道，我们当下的所有努力和言行，都是为了彼此共同的未来。这就是第七招的核心作用。

规划相处模式的核心，是商量未来双方各自扮演的角色。例如，你希望她主内，你主外。当然，有些女生可能想当“女强人”。这时你可以幽默回应：“吃软饭虽然不好，但能吃你的软饭是我的荣幸。”这只是一个互动过程。

恋爱规划的作用就是给女生制造美好憧憬，同时让她确信，你的未来里有她的存在。

## 具体操作：约会规划

因为你已经领悟了恋爱规划的精髓，并且双方已成为情侣，女生会非常信任你。这时，她可能会产生强烈的见面冲动。

如果女生没有主动提出约会，这很正常。因为中国女性通常比较传统和含蓄，不善于直接表达。她们可能使用潜台词，而不是直接说“我想见你”。

由于你们已经是情侣关系，主动邀约就变得非常简单。在中国传统的交往模式中，成为男女朋友后约会是一件顺理成章的事。

以下是约会规划的四个核心方面：

1.  **确定约会地点**
2.  **确定约会时间**
3.  **确定碰面方式**
4.  **确定约会项目**

我们将逐一讲解。

### 关键点：让女生参与规划

在进行约会规划时，有一个关键点：**必须让女生参与进来**。

如果全程由你独自策划，女生只是被动同意，那么她放你鸽子的几率会很大。因为整场约会与她无关，她没有参与感，也就没有责任。

如果时间、地点、项目都由她参与决定，那么她放鸽子的可能性就很小。因为这里面有她的责任和承诺。即使她真的因故不能来，也会感到非常抱歉，并主动商量改期。

因此，规划时要尽量配合女生的时间，询问她的喜好，综合双方意见来做决定。

### 1. 约会地点

约会地点的选择很多，例如：公园、咖啡厅、游乐场、图书馆、超市等。

我曾经最特别的一次碰面地点，是约在一家成人用品店门口。因为那就在她家楼下。这虽然有点尴尬，但也能起到一定的性暗示作用，增加趣味性。

地点没有严格限制，可以根据双方喜好和方便程度选择。

### 2. 约会时间

约会时间应选择双方都有空的时候，通常是周末。可以先询问女生：“宝贝，你什么时候有空？周六还是周日？” 确定日期后，再精确到上午或下午，最后确定具体钟点。

例如：“那就定周六下午3点，在XX地方见，可以吗？” 得到确认后，时间就锁定了。

### 3. 碰面方式

碰面方式主要有三种：
1.  女生过来找你。
2.  你过去接她。
3.  双方自行前往约会地点。

通常建议选择第三种方式。如果你去接她，她可能会因为化妆、打扮而让你久等。约定各自前往，她会更有时间观念，准时赴约。

### 4. 约会项目

约会项目比较灵活，主要分为A类和B类。

**A类项目**的特点是：**能自然提供话题，无需你刻意寻找话聊**。这类地点适合聊天技巧尚不熟练的新手。例如：游乐场、真人CS、射箭、骑马、私人影院、密室逃脱等。在这些场所，丰富的环境和活动本身就能消耗时间、创造共同体验，即使聊天不多，女生也会觉得开心。

**B类项目**的特点是：**需要你通过聊天来主导和推进约会**。例如：咖啡厅、逛公园等。这类约会让双方有更多深入的言语交流，可以分享经历、交换信息，能让关系迅速升温。但这要求你有较好的聊天能力。

选择A类还是B类，取决于你的当前水平和想要达到的效果。

## 进阶技巧：首次约会的“放鸽子”策略

在首次约会前，有一个进阶技巧：可以策略性地“放鸽子”。这并非不礼貌，而是一种心理学应用。

它的作用是：**让女生更加期待下一次约会**。当女生满心欢喜地准备赴约，你却告知临时有事不能来，她的情绪会从高点跌落（情绪波动）。这反而会加深她对你的情绪投资，让她更想见到你。

**核心原理**：人们对于“得不到”和“失而复得”的东西往往更加渴望。

**重要前提**：这个技巧必须在**你已对女生建立足够吸引力**的基础上使用。如果吸引力不足，放鸽子会导致关系崩盘，让她觉得你不尊重人、不守信用。

**折中方案（推荐）**：先告知女生“临时有事，约会取消”，让她产生失落感。过一段时间后，再告诉她“事情解决了，约会照常”。这种“失而复得”的喜悦会极大地刺激她的情绪，让她更珍惜这次约会。

此外，放鸽子也能**测试女生对你的兴趣浓度**。
*   如果她反应平淡（如：“没关系”、“知道了”），可能说明她对约会或对你的兴趣一般。
*   如果她表现出明显的失落、撒娇甚至小抱怨（如：“我都准备好了，你怎么能不来”），则说明她非常在意你和这次约会。

通过观察她的反应，你可以判断自己在她心中的位置。

## 具体操作：进度锁定

*   关于接吻：“到时候我可能会‘壁咚’你哦，你小心别靠墙走~”（以开玩笑的语气）。

关键是要用轻松、开玩笑的口吻说出来，让女生觉得这只是一个有趣的提议，而不是严肃的要求或压力。如果她接受或默许，那么约会时你就可以自然地进行；如果她表现出抗拒，那就当作玩笑过去，不要强求。

千万不要用严肃、强迫的语气，否则会让女生觉得你目的性太强、很“low”。进度锁定只是为后续的亲密接触做一个心理铺垫和缓冲。

## 本阶段聊天注意事项

当你们已经成为情侣，并打完恋爱契约后，聊天可以更加随意和真诚。不必再拘泥于套路、打压或应对废物测试。

“我爱你”是三个非常沉重的字，代表了一种严肃的承诺，几乎等同于“我要娶你”。如果你不打算娶她，就不要轻易说出口。如果女生问起，你可以解释：“‘我爱你’是一个承诺，我说出口就代表我要娶你。所以我不会轻易说。” 这样反而会让她觉得你更可靠、更浪漫，并可能为了得到这三个字而更加努力。

此外，成为情侣后，可以正常进行信息交换，例如询问对方的身高、家庭情况、老家在哪里等。这不再是暴露需求感，而是情侣间必要的了解。不要害怕问这些问题。
