## 活在当下（灵在）

“灵在”就是佛教所说的“当下”。当下很简单，就是**吃饭、睡觉、喝水**。

但我们在喝水时，脑子里想的是女人、工作；吃饭时，想的也不是吃饭。我们的注意力并不在正在做的事情上。

我建议大家尝试我的生活方式：吃饭时，就认真观察筷子夹起食物、送入口中、咀嚼的感觉；洗手时，就认真听水声、闻香皂味。这就是**活在当下**。

当下没有情绪，也没有思维。一开始你可能感觉不到什么，但如果你能在这种状态下持续五分钟，就会开启新的大门。你会发现饭更香了，生活更真实了。因为以前你从未真正进入生活。

睡觉时就安心睡觉，排除思维，放空自己。**放空就是高度集中注意力**。一个注意力涣散的人，无法做好任何事情。

你可以尝试“行走冥想”，散步时将注意力完全放在脚步上。坚持五分钟，你会进入一种轻松愉悦的状态，感觉树木更绿，世界更明亮。

## 接纳的过程与应用

“接纳”是指，将一个让你害怕的事物，逐渐变得平常，如同家人。这个过程就是接纳。

以下是接纳的过程描述：

2.  接触与了解：通过交换信息，逐渐了解该事物。
3.  心态转变：恐惧感消退，感觉变得平常。
4.  最终状态：能够平静、自在地与之相处。
