# 连招1.0学习训：乌鸦连招1.0学：第三阶段：心态：心态第7课内观音

在本节课中，我们将学习“内观音频”练习。这是一种通过引导式冥想，结合呼吸与观想，来放松身心、清理负面情绪并积聚正面能量的方法。整个过程旨在帮助你达到深度的放松与内在和谐。

---

首先，请采用一个舒适的姿势，坐着或躺着。全身放松。放松你的头皮，放松你的眉毛。

内观不是催眠。内观是一种与能量相关的功法，跟随引导进行即可。放松，非常地放松。脸颊两旁的肌肉也放松了，下巴也放松。

脖子放松，肩膀也放松。

现在，将你的注意力放到呼吸上。每一次吸气，你都吸入一种带有麻醉效果的宇宙能量。这种能量从鼻腔进入额头，**能量 = 麻麻的感觉**。这种麻麻的感觉一直蔓延到你的气管、支气管和肺部。

每一次呼气，你都将自己体内的不愉快、负面情绪和疾病呼出体外。保持这样呼吸的观想。

此时，你全身已沉入非常放松的状态。只有你、你的呼吸和引导者的声音存在。放松。

每一次吸气时，吸气通过你的鼻腔和额窦。**额窦 = 额头内部的空腔**。吸气要从鼻腔进入额窦，再进入气管。能量在不断进入身体，通过肺部“解压”到身体的每一个部分。

吸气。现在你的肺部、肝脏和心脏都放松了，都被能量包围着。这个能量还在不断向身体的其他地方扩散，到达身体的每一个地方、每一个细微的角落里。所有的负面情绪都被排出了体外。

你现在感到自己轻飘飘的，非常舒服，非常放松。你感觉不到身体的存在，现在只有你和引导者的声音存在着。

源源不断的能量在冲刷着你身体的每一个部分。你的身体变得非常透明，像一个玻璃人。因为所有的杂质、所有的负面情绪都离开了你。你就像一束光。非常好。

做得很好。

上一节我们完成了身体的放松与能量灌注，本节中我们来看看如何与内在自我连接。

现在，想象自己站在自己的面前，就像看着镜子一样。在心中去描述你内心的自己。你不需要回答，只需要观察。

以下是几个观察的指引点：
*   他是一个小孩吗？
*   他的身体是什么样的？
*   他在哭还是在笑？
*   你对这个内在的自己是什么颜色的？
*   你对这个内在的自己是什么样的感觉？你喜欢他吗？讨厌他？

慢慢地观察。现在，去拥抱你内在的自己。就这样，拥抱自己。把你的能量传输给内在的自己，让你们合二为一。就是这样。

现在，你自己已经得到了内在自我的能量。你感到整个身体都飘起来了，非常舒服，非常喜悦和放松。很好，全然地放松自己。

刚才我们与内在自我进行了融合，接下来，我们将学习如何具体地运用和转化能量。

现在，感觉所有的能量都在自己的下腹部集中，并在你的下腹部转动。这个能量代表你过往经历中最喜悦、最高兴、最让你回味无穷的事情。所有的能量都在你的下腹部汇集。

现在，赋予它颜色。你觉得它是什么颜色的呢？在心里回答，不要去想，只是让它自然地出现色彩，自然地变形。

这个喜悦的能量是什么符号？它是一棵树吗？它是一个动物吗？或者说它是一个象形的符号吗？这些都没有关系。

不管是什么，你让它从你的丹田部位，往你的全身流动。你想让它到哪里去，就让它到哪里去。很好。

现在，让它变得更大、更亮，让它充斥着我们身体的每一个部分。每一个细微的地方都充满了这种快乐和喜悦的能量。

现在，让这个喜悦的能量离开你的身体。对，让它在你的体表运动，每一个地方都滚过。它现在是什么颜色的？它现在是什么形状？非常好。

现在，让这个能量发射到引导者这里来。我们所有人的快乐和喜悦，都连接在一起了。非常好。

处理完正面能量后，我们同样需要面对和释放负面情绪。

现在，回到自己的内在。想象一件事情，回想一下你从小到大，对自己最悲哀、最愤怒、最不舒服的东西所带来的那种情绪。比如被打的恐惧、被抛弃的恐惧、那种不甘、那种愤怒。

让这些负面的能量，在自己的腹部汇集。它是什么颜色的？它是什么形状？

让它变大。现在，让这个负面的能量再变小。随着吸气，这个能量变大；随着呼气，这个能量变小。**吸气 -> 能量变大；呼气 -> 能量变小**。

感受一下这种负面的东西，让它实质化。现在，把它放到你的胸口处。将手放到胸口，把这个能量给拉出来，让它在你的手里面变大、变小。看到它的形状了吗？让它变形。

现在，我们做一个抛洒的动作，把所有的烦恼和愤怒、所有的负面情绪都抛得远远的。引导者数三声，说“丢出去”，你就把它丢出去。

一，二，三，丢出去。

现在，你整个身体都非常轻松了，非常舒服。很好，你整个人很轻松、很愉快。

请你记住这种轻松、愉快、充满能量的感觉。在你睁开眼睛之后，你会把这种感觉带到你今后的生活中去。你是充满能量、充满喜悦和爱的。等一下你睡觉的时候会睡得非常好，明天早上起来你将精神百倍。

引导者等一下要数三声，然后打响指。在打响指之后，你就睁开眼睛，回到现实中来。

一，慢慢地醒来。
二，把眼睛睁开。
三，动一动身体吧。

本节课中，我们一起学习了“内观音频”的完整练习。我们从身体放松和呼吸观想开始，逐步与内在自我连接，处理并转化了正面与负面的能量，最终达到深度放松与内在净化的状态。请记住这种充满能量与喜悦的感觉，并将其融入日常生活。
