# 情绪爆破课程 14：什么是安全时间 ⏳

在本节课中，我们将要学习情绪爆破五个时间法则中的第二个——安全时间法则。我们将了解它的定义、核心作用，以及为何它在建立吸引力的过程中至关重要。

上一节我们介绍了建立初步联系的“表达时间”，本节中我们来看看关系如何进入下一个更稳固的阶段。

## 什么是安全时间法则？

安全时间法则是指，在通过“表达时间”建立了初步吸引力和熟悉感之后，双方进入的一个相对安全的聊天阶段。这个时间节点的核心特征是，双方不再是陌生人，彼此间已经建立了某种羁绊和联系感。

## 安全时间的核心特征

进入安全时间后，关系会呈现出一个关键状态：**一般不会因为某些话题而突然触怒对方，导致被拉黑的情况发生**。这是因为此时的关系已经通过了“是否安全”的初步评判。正因在此阶段不易被对方疏远，所以它被称为“安全时间”。

## 安全时间的主要任务：情绪刺激

判断进入安全时间后，我们的核心任务就变得明确起来。我们可以开始聊许多能够制造强烈情绪波动的话题。

情绪爆破的核心在于制造**情绪波动**。而大量运用情绪波动技巧的最佳时机，正是在安全时间节点。因为在此阶段，关系基底相对安全，我们无需过度担心某些话题会立刻触及对方底线。

因此，在安全时间，我们需要大量刺激对方的情绪，以制造更强烈的情绪张力。公式可以简单表示为：
**情绪波动 → 情绪张力 → 吸引力增强**

所以，安全时间我们要做的唯一一件事情就是：**情绪刺激**。

## 本章小结

本节课我们一起学习了安全时间法则。我们明确了它的定义是关系进入相对稳固、安全的阶段；理解了在此阶段的核心优势是不易因话题冒犯对方；并最终确定了安全时间的核心任务是进行**情绪刺激**，以制造情绪波动，从而增强吸引力。
