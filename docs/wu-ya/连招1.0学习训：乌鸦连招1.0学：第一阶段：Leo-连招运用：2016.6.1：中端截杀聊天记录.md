# 连招1.0学习训：乌鸦连招1.0学：第一阶段：Leo-连招运用：2016.6.1：中端截杀聊天记录

在本节课中，我们将要学习连招系统的第六招——“中断截杀”。这一招的核心目的是与目标快速建立恋爱关系，从而跳过从陌生人到熟悉人的常规步骤，为后续的约会和关系发展铺平道路。

## 概述：什么是中断截杀？

中断截杀的核心是与妹子建立明确的关系。当妹子从内心认同你是她的男朋友时，约会过程就会变得非常容易。这是一种变革性的技巧，能省去大量前期铺垫。

其形式多样，例如“七天情侣”、“以爱之名”、“书记选小老婆”或“皇上选妃子”。但形式只是外壳，真正的核心是**建立关系**。

## 第一阶段：聊天记录解析与技巧拆解

上一节我们介绍了中断截杀的概念，本节中我们来看看如何在实际聊天中运用。我们将解析一份从连招第二招快速推进到第六招的完整聊天记录。

这份聊天记录中的妹子是一名学舞蹈的大学生。记录展示了如何快速进行语言升级，以及在截杀过程中如何处理女生的刁难和废物测试。

### 1. 第二招的运用：制造好奇与互动

以下是运用第二招“原境螺”（制造好奇）的关键对话和技巧分析。

**对话示例：**
> 我：不许笑，再笑逼东里。
> 妹子：什么意思？
> 我：我会三十六种“升文”。
> 妹子：都是些啥？

**技巧解析：**
*   **说一半藏一半**：当我说“我会三十六种升文”时，并没有具体说明是什么，这成功引起了妹子的好奇和提问。
*   **避免把话说满**：一次性把信息说完，容易导致对话终结（妹子可能只回“好吧”、“嗯嗯”）。适当的留白能为互动创造空间。
*   **引导投资**：当妹子产生兴趣后，不要立刻满足她。可以设置一个小门槛，让她为你做点什么（例如“叫声好听的”），这是挑战性测试的运用。

**过渡**：通过制造好奇和互动，我们成功吸引了妹子的注意力并让她产生了投入。接下来，当妹子开始主动提问或服从测试时，就为我们进行关系升级创造了窗口。

### 2. 推进关系：暗示、筛选与框架控制

在互动良好的基础上，可以开始进行关系升级的试探。

**关键操作点：**
1.  **暗示邀约与关系**：
    *   **代码示例**：`“算了吧，下次试给你看。”`
    *   这句话暗示了未来见面的可能性，同时也在试探妹子的反应。
2.  **处理IOD（无兴趣指标）**：
    *   当妹子说“不用试了，告诉我就好”时，这是一个轻微的IOD。
    *   **应对方法**：用无攻击性的IOD反弹。例如：“对哦，你又不是我女朋友，不能告诉你。” 这既保持了框架，又以玩笑方式推进了关系话题。
3.  **抛出挑战性测试**：
    *   当妹子撒娇说“不要嘛告诉我”时，立即抛出测试：“叫声好听的”。
    *   妹子服从后，给予奖励并再次暗示：“哈哈好吧，如果我们有缘能见面的话，就告诉你吧。”
4.  **进行筛选**：
    *   当关系热度足够，妹子问“那我还有机会吗？”时，开启筛选。
    *   **筛选示例**：
        *   “你声音怎么样？说一句我听一听。”（考察声音）
        *   “你平时喷香水吗？如果男朋友喜欢你会喷吗？”（考察服从度和体贴度）
    *   筛选通过后，给予奖励：“你的声音挺好听的。”

**过渡**：通过一系列的暗示、测试和筛选，我们不仅升高了关系，也收集了妹子的兴趣指标。当妹子表现出足够的好感时，便是发起“中断截杀”的最佳时机。

### 3. 中断截杀实战：强势带领与破解测试

这是最关键的阶段，需要结合情绪判断、强势带领和破解废物测试。

**截杀发起：**
1.  **判断窗口**：当妹子说“就喜欢智商高的”时，确认其兴趣高涨。
2.  **发起截杀**：使用高框架话术。
    *   **低框架示例**：`“那你做我女朋友好吗？”`（把决定权交给对方，姿态低）
    *   **高框架示例**：`“那你想要我做你男朋友吗？”`（把“对方想要”作为前提，姿态高，且保留自己拒绝的权利）
3.  **处理犹豫与废物测试**：妹子通常会因安全感不足而犹豫或测试。
    *   **测试1**：“这样的女生很多。”
        *   **破解**：直接点明并回答：“你是在问我还会对其他女生这样吗？不会啊。”
    *   **测试2**：“只有我吗？可是为什么呢？”
        *   **破解**：抬高自身价值并制造缘分感：“因为不是每一个时刻我都想恋爱。”（暗示：你刚好出现在我想恋爱的时候，很幸运。）
    *   **测试3**：“没见面就要试吗？”
        *   **窗口判断**：这不是拒绝，而是犹豫。语气不确定表明窗口很大。
        *   **应对**：强势带领：“试了见一样嘛。行还是不行，给我答案吧，在我改变主意之前。”
4.  **完成仪式与最终确认**：
    *   妹子答应后，立即给予奖励（如“哈哈么么哒”），强化她的行为。
    *   引导完成关系仪式，如改称“老公/老婆”。
    *   **面对终极安全感测试**（如“你花心吗？”、“我不想被伤”）：
        *   **核心原则**：提供安全感，但保持框架。
        *   **示例回答**：“花。但这不妨碍我爱一个人。” → “喜欢你自信一点，这样我会更爱你。” → “谢谢你认真，听着还有点感动。” → “我的答案有用吗？你不如看我怎么做好了。如果你觉得不合适，我们可以随时中止，反正我不会流泪的。”
        *   最终用浪漫强势的话术收尾：“我不会哄人，但我会用嘴让你不哭。”

## 核心技巧总结 🧠

本节课中我们一起学习了中断截杀的全过程，以下是核心要点的总结：

1.  **目的**：快速建立明确恋爱关系，简化后续步骤。
2.  **前提**：确保吸引足够，妹子情绪投资高。
3.  **核心技巧**：
    *   **公式**：`制造好奇 → 引导投资 → 暗示升级 → 进行筛选 → 判断窗口 → 强势截杀 → 破解测试 → 完成仪式`
    *   **框架控制**：始终保持“我是奖品”的高姿态（例如：“那你想要我做你男朋友吗？”）。
    *   **废物测试破解**：识别测试背后的安全感需求，结合“提供安全感”和“保持自我框架”来回答。
    *   **情绪带领**：在关键时刻（如对方犹豫时）要敢于强势带领，替对方做决定。
4.  **心态**：女生废物测试越多，往往代表她越认真。耐心、诚恳地应对，是建立长期关系的基础。

通过以上步骤和心态，你便能有效地运用“中断截杀”，与目标建立起稳固的恋爱关系。
