## 概述：什么是IOI与假性IOI？

IOI，即兴趣指标，指对方对你表现出好感的信号，例如眼神交流、主动搭话等。然而，并非所有IOI都发自真心，有些可能是“假性IOI”，背后隐藏着其他目的。盲目对假性IOI采取行动，例如直接表白，往往会导致失败。

上一节我们介绍了社交互动的基本概念，本节中我们来看看如何分辨兴趣指标的真伪。

## 核心策略：服从性测试

面对女性发出的IOI，正确的做法不是立刻升级关系或表白，而是进行“服从性测试”。CT是Compliance Test的缩写，即服从性测试。

**核心公式：收到IOI → 下达CT → 根据反馈判断IOI真伪**

服从性测试主要有两个作用：
1.  引导对方为关系进行投资，加深联系。
2.  检验对方发出的IOI是否真诚。

如果对方通过了服从性测试，说明兴趣可能是真实的；如果拒绝或抵触，则之前的IOI很可能属于假性。

2.  **下达测试**：使用服从性测试或挑战性测试来检验对方。
3.  **评估反馈**：根据对方是否服从或接受挑战，来判断IOI的真伪。
4.  **放弃表白**：在关系早期，表白通常是高风险的低效策略，应优先使用其他互动技巧推进关系。
