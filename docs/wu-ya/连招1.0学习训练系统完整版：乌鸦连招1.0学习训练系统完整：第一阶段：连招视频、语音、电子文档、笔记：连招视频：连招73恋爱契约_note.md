## 概述：什么是性期待？

上一节我们介绍了连招的四个基本模块：吸引、安全感、接杀和潦草。本节中我们来看看如何通过“性期待”来高效推进关系，避免在不必要的阶段浪费时间。

## 聊骚的核心原则：间接与试探

首先你要知道，聊骚绝对不可以太过认真和刻意。你需要以一种间接、擦边的方式进行试探。

**核心公式：聊骚 = 间接性擦边 + 试探**

所谓间接，就是试探。试探是一种美德。如果你直接进攻，例如说“老婆你喜欢这个姿势吗？”，对方会感到尴尬。你需要让她在你面前一点点放松下来。

以下是开启聊骚的两种实用方法：

### 方法一：从接吻话题切入

你可以从接吻这个话题开始，逐步向性方面靠拢。

1.  **发起话题**：你可以问她：“你一般接吻的时候，喜欢舌吻，还是喜欢轻轻碰一下？”
2.  **引导评分**：不管她回答什么，接着问：“如果给你的舌吻打分，满分10分，你觉得能打几分？”
3.  **植入期待**：如果她打5分以上，你可以说：“那属于中上游了，我下次一定要跟你试试看。”如果分数低，你可以用调侃的语气说：“这可是作为女朋友必学的一项本事，下次我可以教教你。”

**关键点**：不要生搬硬套这些话术。每次聊天的具体方式可以不同，跟随自己的感觉，但原理都是从温和话题逐步深入。

### 方法二：利用“真心话大冒险”

这是一种更游戏化的间接方式。

1.  **建立规则**：提议玩“真心话”，规则是每人问对方五个问题，不能重复，必须诚实回答。
2.  **铺垫普通问题**：先问几个正经问题，例如“你谈过几次恋爱？”、“你最喜欢我哪一点？”。
3.  **插入关键问题**：轮到你的最后一个问题时，可以假装思考后说：“实在没得问了，我问个稍微‘骚’一点的吧。”然后抛出核心问题，例如：“你一晚上最多能承受几次？”或“你做爱时最喜欢哪种姿势？”

**关键点**：问完关键问题后，不要傻傻地停住。可以顺势再聊两三句，然后**主动中断**性话题，转回普通聊天。

## 高级技巧：节奏控制与画面植入

聊骚不是持续不断地进行，而是要有节奏地穿插。

**聊骚节奏模型**：
```
普通聊天 — [聊骚片段] — 普通聊天 — [聊骚片段] — 普通聊天
```
中间的连线代表聊骚片段，你需要穿插进行。如果一直聊，会让对方觉得你目的性太强，反而暴露意图，破坏安全感。

### 如何主动中断并再次拉回？

主动中断性话题是个好习惯。中断后，你可以随时找个借口再拉回来。这需要练习。

例如，当对方说“最近工作好累”，你可以回应：
> “我最近也有点累。不过我以前学过一些按摩技巧，手法非常不错，有空可以给你试试。”
> “不过我认为，做爱这件事后戏更重要。我和以前的女朋友做完后，都会给她按摩，让她非常放松舒服。”

**核心操作**：你这样相当于在给她植入一个画面。人理解信息需要依靠大脑中已有的画面或经验。你通过描述按摩、精油、后戏，让她在脑海中构建出与你亲密接触后的温馨场景。

接着，你可以更深入一点，问她：
> “你以前和男朋友时，比较喜欢什么姿势？”或“喜欢在哪个时间段？”

当你获取这些信息后，第一次见面就可以尝试带她去开房了。

## 必须避免的致命错误

在聊骚过程中，有几个关键错误必须避免。

以下是需要警惕的几点：

1.  **忌聊成“黄色小说”**：如果你们无法马上见面，切忌无休止地进行文字性爱（文爱）。女人是一种需要内在情感感受多于外在身体感受的动物。如果你让她在幻想中充分感受到了高潮，她可能就不再需要真实的你了。每次想深入聊骚时，问自己：我是想就此结束，还是想拥有她无数个夜晚？
2.  **忌强行推进**：如果女孩表现出不愿意聊性，不要强行继续。你只需要以擦边的方式，暗示这种可能性即可。例如，可以说：“我不会放弃任何一个把你睡了的机会。”然后就此打住。这种暗示会埋在她心里，反复发酵。
3.  **忌目的单一暴露**：不要让对方觉得你只对她的身体感兴趣。最好的方法是遵循这个原则：**“我爱你的性格，同时也迷恋你的身体。”**

**核心心法**：真实的男人学会矛盾地表达自己。这需要练习。例如，语言上推开（“你这身衣服真难看”），但动作上拉近（搂过来亲一口）。在聊天中，也是一会儿照顾她的情感和浪漫需求，一会儿又进行性感的挑逗。这会让她产生一种在你这里可以**安全地体验性**的感觉。
