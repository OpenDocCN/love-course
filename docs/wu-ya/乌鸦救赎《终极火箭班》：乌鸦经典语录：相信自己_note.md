# 乌鸦救赎《终极火箭班》第1课：相信自己 🚀

在本节课中，我们将学习“相信自己”这一核心概念。我们将探讨为何依赖外部标准或“正确方法”可能是一种迷思，并理解转向内心寻求力量的重要性。

## 核心概念解析

上一节我们介绍了课程主题，本节中我们来看看“相信自己”的具体含义。

其核心在于，当你希望他人相信某件事时，只要你勇敢地表达出来，就一定会有人相信。由此你会发现，并不存在一个需要去追寻的、绝对“真实”的答案。**真实答案** 在于我们做事情的过程本身。

例如，当你想学习一种方法时，你总会认为客观上存在一个相对更科学的方法。但实际上，这个所谓的“相对科学的方法”也是通过实践摸索找到的，它并非天然存在。

## 实践案例：创业探索

以下是关于创业探索的具体说明。

*   **迷思**：在最初创办公司时，我总在探索公司运营是否存在一个特定的、现成的成功招数。
*   **真相**：其实并不存在。你的想法就是方法。关键在于你要相信自己的做法，必须相信自己的做法。

## 如何建立自信

那么，人何时才能变得自信呢？

当你不**再向外界寻求力量**，而是**转向自己的内心**开始寻求力量时，自信便会产生。当你能够不再依赖外力，转而相信**自己内心的力量**来支撑自己继续前行，就意味着你理解了这一点。

**结论公式**：**自信 = 停止向外寻求 + 开始向内探索**

不存在客观存在的唯一正确方法。正确的方法**要靠自己内心去探索**。

---

本节课中，我们一起学习了“相信自己”的深刻内涵。我们明白了不存在绝对的外部标准答案，真正的力量和正确的方法源于内心的探索与坚信。关键在于停止向外寻求认可，转而依靠并践行自己内心的想法与判断。
