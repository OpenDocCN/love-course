# 🐦 乌鸦自信聊天宝典：2.2 常用聊天技巧的定义

在本节课中，我们将学习几种在和女生相处时非常实用的聊天技巧。这些技巧的核心在于调动女生的情绪，并让她对你产生好感。我们将逐一解析推拉、角色扮演、附格、筛选以及骄傲风趣这五个核心技巧的定义与基础用法。

## 🔄 推拉

上一节我们介绍了课程的整体目标，本节中我们来看看第一个核心技巧：推拉。推拉，顾名思义，包含“推”和“拉”两个动作。这种说话方式不仅在恋爱学中常见，在许多相声和搞笑节目里也广泛使用。通俗来讲，就是“先给一巴掌，再给一颗糖”。

推拉主要分为两种：常用推拉和关系远近推拉。

以下是常用推拉的一个例子：
1.  **推（损）**：我会先说“你最近又胖了”。（此时女生可能会生气）
2.  **拉（夸）**：然后我会接着说“但是更加符合我心目中女神的形象了”。

这就是先推后拉。还有一种方式是“拉推”，即先夸后损。

以下是拉推的一个例子：
1.  **拉（夸）**：我会说“你今天的衣服穿得很漂亮”。（对方可能会说谢谢）
2.  **推（损）**：然后我会说“但是你的围巾好像搭得不是特别好，跟这件衣服不太配”。

这里需要注意的是，推拉的尺度与你跟女生的熟悉程度密切相关。很多学员在使用推拉时，因为尺度把握不当，直接将女生“推走”了。例如，过早地攻击女生的长相或触及她的核心利益点。

因此，在最初和女生聊天时，最好不要过早使用推拉。很多人掌握不好这个度。例如，曾有学员先夸女生漂亮，女生很高兴；但他接着损道：“漂亮有什么用，下巴都尖成那个样子了，像个蛇精一样。”这直接从调侃变成了人身攻击，导致女生彻底翻脸。

所以，推拉技巧更适合在彼此熟悉之后使用，并且你必须清楚地区分推拉和人身攻击。

关系远近推拉的原理与常用推拉相同。一个经典的例子是“皇后惯例”：
1.  你叫女生“爱妃”。
2.  女生可能会说：“我什么时候成你的爱妃了？”
3.  你接着说：“对呀，你不是爱妃，你是皇后。”（此时女生会开心）
4.  然后你可以说：“皇后不就是又老又丑的那个吗？”

这同样是一种先拉后推的技巧。

**核心公式**：如果你想夸一个人，就采用 **先推（损）后拉（夸）**；如果你想损一个人，就采用 **先拉（夸）后推（损）**。最终表达的重点是后面那个动作。

推拉技巧还可以衍生出其他策略，例如“欲擒故纵”。这个技巧是指，前期你和女生关系很好，但突然消失或不再联系她，让她产生情绪上的冷落感（相当于“推”）。当她因此产生巨大情绪波动并需要你安慰时，可能会主动来找你。不过，这个策略尤其不推荐新手使用。

## 🎭 角色扮演

理解了如何通过推拉制造情绪波动后，我们来看看如何让聊天变得更有趣。角色扮演就是一个非常有效的方法。它非常好理解，即你为自己和女生设定一个虚拟的角色关系进行互动。

例如：
*   **场景**：你在上班。
*   **女生说**：“这么忙啊？”
*   **你说**：“对啊，朕日理万机。爱妃，过来给朕捶捶背。”

在这个例子中，你为自己设定了“皇帝”的角色，为女生设定了“爱妃”的角色。如果女生愿意接你的话，进入这个角色，就说明她愿意和你玩，这是一种兴趣指标。

角色扮演的威力在于它能带入“框架”。所谓框架，即互动中双方默认的权力关系。例如，即使女生年龄比你大，只要你让她叫你“欧巴”（哥哥），你们之间就建立了“哥哥和妹妹”的关系框架。在这个框架下，哥哥通常是更强势、更有保护欲的一方。

你可以利用这个框架展示男性魅力。例如，当以“欧巴”的角色关心生病的“妹妹”时，你可以严肃地要求她吃药，甚至直接买药送过去。这种基于角色的关心，往往比直白的关心更自然、更有力，能有效推动关系升级。

**核心概念**：任何带有特定称呼的互动，如“老公/老婆”，都属于角色扮演。它通过建立虚拟关系框架，让你能更自然地说出一些平时不便直接表达的话（例如，“好困啊，好想抱着爱妃睡觉”），从而增加暧昧感和趣味性。

## 🪄 附格

掌握了通过角色扮演增加趣味性的方法后，我们来看一个更高级的技巧：附格。这个技巧源自灵修和催眠领域，非常强大。

附格的核心是：**将你希望对方拥有的特质，“赋予”对方，并给予正向反馈（奖励）**。

例如，如果一个女生性格大大咧咧，但你希望她对你温柔。你可以这样说：
> “虽然你看起来有点大大咧咧，像个女汉子，但我感觉你对我特别温柔。你的这种温柔让我感觉特别好，如沐春风。”

这个过程中，你做了两件事：
1.  **赋予特质**：你指出“你对我很温柔”（即使她目前并非如此）。
2.  **给予奖励**：你表达“我很喜欢你的这种温柔”。

这个技巧的厉害之处在于，它可以“塑造”一个你理想中的伴侣。你可以对任何你喜欢的特质进行附格，比如时尚、可爱、开朗等。但关键在于，一定要加上“我喜欢”这样的正向反馈。

其原理类似于“正向强化”。举个例子：如果你讲笑话时，总有朋友捧场大笑（奖励），你就会越来越喜欢讲笑话，越来越擅长搞笑。对女生也是如此，当她表现出某种你喜欢的特质（或被你赋予该特质）时，你及时给予欣赏和奖励，她就会更倾向于保持或强化这个特质。

## 🎯 筛选

在学会了如何互动和塑造印象后，我们需要建立一个重要的心态：筛选。很多男生不具备筛选能力，因为他们内心没有明确的择偶标准。

**筛选是一个高价值行为**。试想，什么样的人才有资格筛选？是那些有选择权的人。当你对女生进行筛选时，潜台词是：“除了你，我还有别的选择。”女生能从这个行为中感知到你的高价值，就像淘宝上销量最高的商品，人人想要，反而更具吸引力。

筛选的另一个重要好处是，它能帮你规避掉不合适的对象，最终目的是为了获得幸福的长期关系。如果没有筛选，仓促开始一段感情，可能会后患无穷。

**如何建立筛选标准？** 你需要明确自己的择偶标准。以下是一个示例框架：
*   **外在**：身高165cm以上，喜欢短发，眼睛大，鼻子挺，嘴巴小巧。
*   **内在**：性格温柔体贴但不失活泼，理解我的工作，孝顺父母。

有了清晰的标准，你才能有理有据地进行筛选，而不是被动接受。

## 😎 骄傲风趣

最后，我们来看一个综合性的态度技巧：骄傲风趣。它要求你呈现出一种既骄傲自信，又幽默风趣的状态。

例如，你可以说：
> “每次我来深圳，天气都会为我变好。”

这是一句非常“臭美”、自恋的话，但它传递出强烈的自信。然而，单纯的骄傲可能显得狂妄。因此，需要加入幽默感来调和，让自信显得可爱而非讨厌。骄傲风趣就是让你在展示高度自信的同时，用幽默弱化可能带来的攻击性，从而显得更有魅力。

关于这个技巧，有一本专门的书叫《骄傲风趣法》，里面有更系统的阐述。它是一种更深层的态度和状态建设。

## 📚 本节课总结

本节课我们一起学习了五个核心的聊天技巧：
1.  **推拉**：通过先损后夸或先夸后损的方式，制造女生的情绪波动。核心是把握尺度，熟悉后使用。
2.  **角色扮演**：通过建立虚拟的角色关系框架，让聊天更有趣、更暧昧，并能自然地说出一些拉升关系的话。
3.  **附格**：通过赋予对方你喜欢的特质并给予正向奖励，来潜移默化地“塑造”和吸引对方。
4.  **筛选**：基于清晰的个人标准选择女生，这是一个展示高价值、并确保长期关系幸福的重要心态和行为。
5.  **骄傲风趣**：一种融合了高度自信与幽默感的个人状态，能让你显得更具魅力。

这些技巧是构建吸引力的重要工具，但需要勤加练习并灵活运用。记住，真诚与尊重是所有技巧的基石。
