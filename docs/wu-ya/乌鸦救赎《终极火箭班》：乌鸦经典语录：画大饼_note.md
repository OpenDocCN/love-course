# 乌鸦救赎《终极火箭班》第X课：画大饼的艺术 🎯

在本节课中，我们将要学习“画大饼”这一沟通技巧的核心思想。这是一种通过描绘美好未来图景，来激发他人兴趣和行动欲望的方法，尤其适用于推动那些“反人性”的共同目标。

## 共同兴趣：话题的自然源泉 🌱

上一节我们介绍了课程主题，本节中我们来看看“画大饼”的第一个应用场景：基于共同兴趣的相处。

大家每天都会打游戏，通常只打两三把，不会太多。正是因为游戏这个共同项目，两人之间的话题自然会很多。当你们拥有共同参与的项目时，话题本身已经不再重要，因为项目会自带话题。

你要玩游戏，肯定会涉及“上路推塔”、“你跟我来”、“不要抢我人头”这类交流。天天就是在喊这些，话题自然就产生了。两人共同经历的事情多了，感情自然就会变好。你会发现，很多时候当你把一件事情经营到正确轨道上时，会感觉好像什么都没做，仿佛事情本就该如此。

但是当你把事情做糟糕时，你会发现到处都是问题。而当你做对时，别人也不会觉得你特别有聪明才智，只会认为这个事情就应该这样。做对的时候，就会有这种本该如此的感觉。

当有了共同的兴趣爱好并能一起参与时，就完全不用再去担心没有话题了。

## 应对“反人性”目标：激发内在欲望 🔥

然而，有一些活动属于“反人性”的范畴，例如健身以及学习。这些听起来很美好，但本质上与人性中追求即时舒适的特性相悖。

健身和学习这种东西属于反人性的活动。你很难强行将其强加于对方，因为对方可能会直接产生抗拒，例如“我不想去健身”。因此，你需要激发他对于这件事的内在欲望。

这时，你可以通过“吹嘘”健身带来的好处来实现。例如，你可以描绘这样的画面：“如果咱俩都变成型男酷女，你有马甲线，我有大胸肌，然后我们一起拍个照。这很酷。以后朋友们看到我们，都会觉得我们这段情侣很酷。”你可以问：“老婆，咱俩要不要做一段很酷的情侣？”

当他对此产生兴趣后，就会开始主动查健身房的资料，到处查询，查好了就可能会一起去。但如果你突然直接拉女朋友去健身，她第一时间联想到的会是那些沉重的哑铃和器械，她会觉得很累，会想“为什么要去？不如去吃吃喝喝玩玩乐乐”。吃吃喝喝是顺应人性发展的东西，而学习这类活动是反人性的。

因此，如果要推动这种反人性的活动，就必须先勾起对方的欲望。

## “画大饼”实践：描绘与引导 🎨

那么，具体如何操作呢？关键在于提前描绘美好蓝图，而非直接提出要求。

就我们俩为了共同健身这个事情，我一开始都没有直接跟他去说这件事，只是给他看各种健身博主的账号。然后告诉他“哎，这样怎么样，那样怎么样”。接着，如果我俩给他描绘一个具体的画面。

我天天跟他说：“我说咱俩要是变成那种型男酷女，很棒对吧。两个人都开公司，又年轻，又有钱，长得又帅又好看，而且身材又棒。我觉得这样活着挺潇洒。”我每天就在那里构想，并跟他聊天讨论。

然后他就开始对这件事情产生兴趣，他觉得这个事情很美好。说白了，这就是“吹牛逼”。接着，我的女朋友也觉得这个事情很美好。她会说：“老公你这么一说，我也觉得挺开心的，确实不错。那要么我们去办个健身卡？”

我说：“可以啊，那你去看看附近有什么健身房吗？我们一起去。”当然，这一切的前提是拥有一定的经济基础，能够为这样的生活选择买单。没有钱，什么都难以实现。幸运的是，我们有钱，我们可以为自己的生活买单，没有问题。

本节课中，我们一起学习了“画大饼”这一沟通技巧。其核心在于：**对于自带话题的共同兴趣，顺其自然即可；对于“反人性”的进阶目标，则需要先通过描绘诱人的未来蓝图（画大饼）来激发对方的欲望，从而引导对方主动产生行动意愿，而不是强行推动。** 关键在于将“我要你做”转化为“他自己想做”。
