# 乌鸦救赎社交光环：社交光环：第三节_note

在本节课中，我们将要学习“社交光环”这一核心概念。社交光环是一种强大的社交现象，它能让他人跳过常规的价值评估过程，直接认可你的高价值。我们将通过定义解释和真实故事，帮助你理解并掌握这一概念。

## 什么是社交光环？

社交光环的专业定义是：**你和A社交的画面被B看到，B跳过了所有审核你价值的环节，直接认定你是一个高价值的人，并愿意与你交往。**

用更通俗的话来说：**当某人看到你很受欢迎时，他会直接选择喜欢你。你相当于用A作为了你的价值担保人。**

这类似于“免检产品”或“口碑产品”。一个典型的例子是《水浒传》中的宋江。许多好汉一听说是宋江，便不再与他为敌，反而请他喝酒吃肉。这是因为宋江帮助兄弟的故事已通过第三方传递开来，所有人都知道他是好人。因此，当他们遇到宋江时，无需再进行社交博弈、展现高价值或使用话术。兄弟们因为听过他的故事，便直接选择追随他。

这就是社交光环的核心作用。

## 社交光环实战故事

理解了定义后，我们来看看社交光环在现实生活中的应用。以下是两个关于我和我朋友的真实故事。

### 故事一：为朋友牵线

我有一个发小，名叫白狼。他今年30岁，是一名医生，在父亲的医院工作，收入不错，有房有车。我们从小认识，关系很好。前段时间，我认识了一位来中国学习和投资的韩国女孩，她三十出头，家庭背景和自身素质都不错。我们聊得来，建立了友谊。

后来，这位女孩表示她想结婚。我立刻想到了白狼。但我没有简单地直接介绍两人认识，因为那样往往效果不佳。我设计了一个方案。

我花了三天时间，向女孩讲述白狼的故事。以下是讲述的主要内容：

*   **感情故事**：我讲述了白狼和他前女友认识的奇葩经历。当初他女友为了气前男友，找白狼假扮现男友，结果假戏真做，两人在一起三年，期间发生了许多故事，我都是见证者。
*   **为人处世**：我描述了白狼如何慷慨地帮助兄弟、照顾兄弟的感情和感受，以及在面对矛盾时如何处理。他的性格有点像“及时雨”宋江，对哥们非常讲义气。
*   **性格匹配分析**：我分析了女孩和白狼的性格。女孩是缺乏安全感的女强人，而白狼聪明、智慧、情绪稳定，对未来没有太大的野心，乐于做普通人。我指出，这两种性格非常互补：一个在前冲，另一个能提供稳定的情感支持。

女孩像读小说一样看完了我几万字的描述，对白狼的期待值变得非常高，主动索要联系方式。但我没有立刻给，而是继续铺垫，强化他们“天作之合”的印象。这时，女孩可以说已经“爱上”了白狼——尽管他们还未曾交流过一字。

然后，我将聊天截图和女孩照片发给白狼，白狼表示非常满意。至此，我才将双方的微信互推。他们加上好友后，因为女孩已对白狼有了极大的好感，对话进行得非常顺利。聊了大约三天，两人便确定了关系，并开始规划见面和未来生活。

这个故事展示了通过第三方塑造社交光环的巨大威力：一个从未谋面的人，仅凭故事就产生强烈好感，迅速建立关系。
