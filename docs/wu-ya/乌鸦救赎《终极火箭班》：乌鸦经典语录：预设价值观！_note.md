# 乌鸦救赎《终极火箭班》第三课：预设价值观！🧠

在本节课中，我们将要学习一个深刻影响你情感关系的内在原因——预设价值观。这是一种心理学概念，指你内心深处对自己和世界的根本看法。这些看法会像剧本一样，引导你的人生走向，尤其是在与女性的互动中。

上一节我们探讨了其他可能导致关系破裂的因素，本节中我们来看看这个更为内在的“预设价值观”是如何起作用的。

## 什么是预设价值观？

预设价值观是你对自己身份和能力的核心信念。它像你为自己人生写下的一个基础剧本。

**核心公式：预设价值观 → 自我行为 → 他人反馈 → 现实结果**

如果你的预设价值观认定“我不是一个有魅力的、能吸引女性的人”，那么你的生活就会朝这个方向发展。这并非玄学，而是一种自我实现的预言。你将自己描述成一个自卑、缺乏魅力的人，你就会不自觉地按照这个角色去“表演”。

## 预设价值观如何影响关系？

以下是预设价值观在情感关系中的具体体现和影响。

### 1. 自卑的剧本

如果你预设自己“没有魅力”，你就会在互动中表现出不自信。面对一个自卑的男人，他人的自然生物性反应就是轻视。这不是吸引力法则，而是简单的角色匹配：你扮演“弱者”，就容易吸引来“强者”或“欺凌者”的对待。

> 友好，你一旦演好人，你一定遇坏人。如果你标榜道德，你一定遇到流氓。如果你标榜善良，你一定遇到坏人。如果你标榜自己是个混蛋，你一定能够遇到被你欺压的百姓。这就是一个角色扮演，什么人遇什么人。

你把自己设定成一个不被喜欢的人，那么被女性疏远或抛弃，就是你“赢得”的、符合剧本的自然结果。

### 2. “女神”陷阱

一个常见的错误预设是把心仪的女性“神化”。当你把一个女人当作“女神”时，你们之间就失去了平等。

真正的进步在于认知的转变：认识到所有人都是平等的众生。激情过后，你会看到她脱妆的样子，看到她费力穿上牛仔裤的瞬间。你会发现，她和所有人一样，有普通的一面。我眼中没有“女神”，只有平等的个体，甚至有时我会把对方看作需要引导的“小妹妹”。心态不同，互动方式就完全不同。

> 你老把人家当女神，那人家就一定会把你当做屌丝来对待了。

### 3. 金钱的匮乏感

预设价值观也体现在你对金钱的态度上。很多人对钱有强烈的匮乏感和饥渴感，认为“攒钱”是正道。

但真正掌控财富的人，只把钱视为工具。他们不成为金钱的奴隶，钱来了又去，用于做事。如果你预设“攒钱很艰难、钱很重要”，那么你的赚钱过程就会真的变得艰难而痛苦。事物的奖赏感，往往取决于获取它的难度。

> 就像打王者荣耀一样，顺风局五分钟解决战斗很没意思。打了快一个小时，团结一致险胜，赢的那一刻手都在抖，感觉像公司上市一样开心。

### 4. 关系中的不安全感

在恋爱中，如果你的预设价值观包含“我可能被出轨”、“我不够优秀”，就会产生强烈的不安全感。

这种不安全感会驱使你做出查手机、盘问行踪、突击检查等行为。这些行为毫无魅力，只会让对方厌烦，或者 ironically（讽刺地），促使对方真的做出符合你预设的事情——出轨。

> 人生是一场戏，我极力配合你的演出。你想办案，好，那就让你有个案子办。

关键不是改变“不查”这个习惯，而是改变底层的预设价值观。当你真正相信自己是充满魅力、值得被爱的人时，你根本不会产生这些不安全的念头和行为。

## 如何建立健康的预设价值观？

以下是建立积极预设价值观的几个关键点。

1.  **自我认可**：在潜意识里坚信自己是有魅力的、有价值的人。认可自己，无需向外寻求过度验证。
2.  **平等视角**：视所有人为平等的个体，去除“神化”或“贬低”的滤镜。你们都是芸芸众生中的挣扎者，没有区别。
3.  **放下掌控**：停止通过查岗、盘问来获取安全感。这种行为暴露并强化了你的不自信。
4.  **聚焦自身**：将注意力从“看住对方”转移到“提升自己”上。当你充满自信、生活精彩时，反而会形成吸引力。

当你不再害怕失去，对方反而可能害怕失去你。这是人性：人一旦发现你不想抓紧她，她就会开始抓紧你。对方的不在乎会引发你的好奇和投资，最终使你陷入更深的关注。

## 总结

本节课中我们一起学习了“预设价值观”的核心概念。你的内在剧本——你对自己是谁、配得什么的根本信念——会外化为你的行为，并最终吸引来与之匹配的现实。在情感关系中，自卑的预设会招致轻视，神化对方会导致不平等，对金钱或关系的匮乏感会制造障碍和痛苦。破解之道在于从内心建立强大、平等的自我认知，停止扮演受害者或追逐者的角色，转而成为自己人生的主导者。当你改变内心的预设，你外在的世界和关系才会随之改变。

**核心要义：改变现实，先从改写你内心的剧本开始。**
