# 乌鸦香火传承：扬升六芒星核心自信2.0：11、扬升六芒星-核心自信：2.语言模式2.0_note

在本节课中，我们将深入探讨“扬升六芒星”体系中关于核心自信构建的关键部分：语言模式2.0与身份认同。我们将学习如何通过调整语言和思维框架，在人际互动尤其是两性关系中掌握主动权，并理解“身份”如何成为个人力量的源泉。

## 📝 概述：语言模式2.0的核心——逆向筛选

上一节我们介绍了资源整合的方法论，本节中我们来看看其核心应用之一：逆向筛选。资源整合的连招版本，其本质是一套从认识到建立关系的完整流程。关键在于，你绝不能陷入被对方逆向筛选的被动局面。

逆向筛选是指：当你向十家公司投递简历，九家都录取你，唯独一家（比如华为）没有录取你。此时，你反而会觉得那家拒绝你的公司最有价值。这就是你被逆向筛选淘汰了。

因此，在互动中（无论是面试还是追求心仪对象），正确的策略是表现出“我很有市场”。你需要传递的信息是：其他选择都在接纳我，我在进行挑选，我对你并没有特别强烈的需求。这种语言模式是一种“推”的艺术。

面对让你心动、具有高潜在价值的对象时，你越要表现得“并非没有你不行”。你必须具备这样的胆识：将“冒着失去一个人的风险，才能真正得到她”内化为你的预设价值观。只有当你处于“可以不合作”的心理优势地位时，合作才有可能发生；急切地讨好只会让你失去机会。

如果对方真的不选择合作，那就坦然接受。如果你做不到这一点，你就会被对方逆向筛选淘汰。你的目标应该是：**逆向筛选对方**。即使是你更想得到对方，也要通过语言和姿态营造出“是她更想得到你”的局面。这样你才能被重视，而不是被轻视。

## 🎯 行动策略：自我引导与价值展示

以下是实施逆向筛选的关键步骤，其核心在于先进行自我引导和价值展示。

1.  **优先进行自我引导**：在互动伊始，不要急于去筛选对方。筛选是向对方展示你自身价值的机会。如果你先筛选对方，对方就可能暴露其价值，使你陷入被动，你的后续行为都像是在证明自己。因此，在不明对方底细时，你首先要做的是“吹嘘”自己，进行强有力的自我引导。
2.  **将自我引导变为性格习惯**：这不是一种临时套路，而应成为你的性格特质。就像许多成功的老板，他们见到任何人都会首先谈论自己的事情和规划。这是一种需要培养的习惯。如果你不能在一开始就框住对方，反而被对方框住，那么在整个互动链条中你会越来越被动。
3.  **展示价值前置**：无论如何，你首先需要展示自己的价值。自我引导永远应该放在筛选对方之前。这是掌控局面的起点。

## 📖 案例解析：王牧江的故事——吹嘘的边界

为了更生动地理解，这里有一个商业领域的极端案例，说明了“吹嘘”策略的双面性。

这是一个关于“逆向筛选”的商业故事。主人公王牧江因情感受挫，决定“豪赌”人生。他变卖资产，租赁豪车并雇佣外国秘书，向地方政府吹嘘要建设“全国最大的贸易中心”。政府被其描绘的蓝图（类似万达广场的概念）说服，甚至应其要求炸掉了公安局大楼以腾出地块。最终，项目因缺乏实际支撑而败露，王牧江被捕。

这个故事说明了“吹嘘”（即高价值展示）本身没有错，甚至是必要的。王牧江的问题在于**吹嘘的逻辑无法自洽，证据链不完整**。他讲述着马云级别的事业，但个人细节和生活层面无法支撑这个形象，让人感觉他并非这个段位的人。这属于“无中生有”的诈骗。

正确的做法是：**在自身真实价值的基础上进行合理润色与放大**。例如，我本人的逻辑是自洽的，我的生活、资产、经历都经得起查验，因此只需适当放大即可。核心方法是：先自信地展示价值（吹嘘），然后逆向筛选对方，使原本高价值的对方感觉你是更高价值的一方。

## 🔑 核心概念：身份认同——力量的根本

讲完了语言模式，我们自然要探讨其内在支撑：身份认同。身份，即你对自己当下角色的定位，它是你力量的根本，并依托于信念。

### 什么是身份？

身份是你所扮演的各种社会角色。例如，我同时是父母的儿子、弟弟的哥哥、女友们的伴侣、学员们的导师、朋友的谋士等。我会在不同角色间切换：作为CEO时，我思考公司战略与利益分配；作为导师时，我思考如何将复杂逻辑通俗易懂地传授出去。

我能做好这些，是因为我**坚定不移地认同这些身份**。我深信自己就是一名情感导师，也是一名合格的CEO。当你明确知道自己是谁（身份），你才知道自己该做什么、不该做什么，才会产生责任感和担当。

### 身份的力量来源：认同感

身份的力量与身份的社会价值高低**无关**，它完全来源于**你对自己的认同程度**。

*   **公式**：**身份的力量 = 对身份的认同程度**
*   **示例**：一个认同自己“农民”身份的人，可以悠然自得地喂猪、享受劳作；而一个不认同此身份的人去做同样的事，则会浑身不自在、格格不入。同理，如果你不认同自己“情感导师”的身份，即使硬着头皮讲课，你的状态也会出卖你，听众会感觉你不专业。

你必须做到极致认同。哪怕只是“假装”自己是这个领域最专业的人。**骗人先骗己**，当你自己都深信不疑时，你的外在表现就会像那么回事。久而久之，装久了，你就真的成为了那个人。我从年轻时开始讲课，最初也慌张，但通过不断实践和获得反馈，强化了这份身份认同，如今便能游刃有余。

### 如何获得身份认同：快速切换与强制认同

聪明人可以快速在不同身份间切换。我能够迅速进入新项目、新角色，因为我能在半天内厘清：客户是谁、我们提供什么、我的职位、我的职责范围。这就像一名演员，刚演完皇帝，下一秒就能演乞丐。关键是可以**训练自己强制认同某个身份**，而不必等待自然而然的感受。

一个深度认同自己的人，不会在语言中寻求外界认可。他的语气是确定的：“你们听我说完，一定会认同我。”反之，如果一个人不认同自己的身份，他就会眼神游移、寻求他人反馈，这种不自信会立刻被他人感知。

## 💎 进阶探讨：核心自信的现代构成

有人问及如何看待“情景自信”。我现在依然认为核心自信至关重要，它主要解决的是：如何在与价值高于或低于自己的人互动时，都能保持自由、不被逆向筛选。

我现在的核心自信主要来源于两部分：

1.  **对“未来自我”的信念**：我深信自己未来会成为某个样子，并且有过实现目标的经历，这让我更容易相信下一个目标也能实现。
2.  **“此处不留爷，自有留爷处”的底气**：这源于清晰的自我定位。例如，在合作中，我上来就定位为“不可替代的合伙人”，而非雇员。我让对方明白“你不能没有我”，同时传达“我可以没有你”的姿态。这种底气来自于：**我愿意冒着失去这段关系或搞砸项目的风险，来换取真正的合作**。在谈判和恋爱中，我都实践过这一点。

真正的狠招是“走对方的路，让对方无路可走”。当对方用“分手”或“终止合作”来要挟你时，如果你能率先表达出“我可以离开”的态度，对方就无计可施。当然，这有时会导致关系真的结束，但你必须能承受这种风险。天底下最糟糕的事莫过于“你不想跟我在一起/合作了”，如果你连这都不怕，并敢于率先亮出底牌，就没人能拿捏你。

## ⚔️ 博弈思维：零和博弈与制胜关键

上述策略的底层逻辑涉及博弈论。现实中的合作大多属于**零和博弈**（利益总和固定，你多我就少）。“正和博弈”（双赢）很难自发产生，通常只在“纳什均衡”下出现——即双方势均力敌，谁也无法单独成事，合作是唯一最优解。

零和博弈的胜负关键在于**前置**，即所有规则、条款、筹码必须在合作开始前就谈妥并锁定。就像决斗，胜负在拔刀前就已由战前准备（情报、装备、状态）决定。如果前期没有建立优势，后期就会极其被动。

因此，在人际和商业互动中，**邪恶与否不是关键，有效才是关键**。为了有效，有时需要采取强势、前置的策略。人性是自私的，如果有机会独占利益，很少有人会主动选择分享。所谓的“分钱”、“双赢”，往往是出于不得已（纳什均衡），或是为了更长远的统治（如刘邦）。

## 👑 领导者智慧：缺陷的妙用与分钱艺术

以刘邦和项羽为例。项羽个人能力太强（“大树底下寸草不生”），导致他无法容忍和留住韩信这样的人才（嫉贤妒能）。而刘邦恰恰因为自己“啥都不会”（不会打仗、不懂内政），反而能吸引萧何、韩信等顶尖人才来辅佐，并懂得将钱财、权力分给他们。

这对我们的启示是：**“好城必有一缺”**。一个看似完美无缺的人或方案，反而让人无从下手、难以合作。适当暴露自己的“缺陷”（不擅长的领域），能吸引那些在此领域擅长的人来弥补，从而形成强大团队。顶级领导者的艺术不在于事事亲为，而在于**识人、分钱（利益共享）和不嫉贤妒能**。如果你什么都会，反而没人需要帮你了。

关于识人，并无神奇相术。最有效的方法就是**通过实际事务进行筛选**——让他讲、让他做，一听一做，便知有无真才实学。这需要付出一定的筛选成本（如工资），但为了找到对的人，这是值得的。

## 📚 本节课总结

本节课我们一起学习了“语言模式2.0”与“身份认同”两大核心模块。

1.  **语言模式上**，我们掌握了“逆向筛选”的心法：通过优先进行自我引导和价值展示，营造高价值形象，避免陷入被动。关键在于敢于冒着失去的风险，并运用“推”的艺术。
2.  **身份认同上**，我们理解了力量的根本来源于对自我角色的深度认同，而非角色本身的高低。通过快速切换和强制认同，我们可以获得在任何场合下的从容与自信。
3.  **策略底层**，我们触及了博弈论的思维：在多为零和博弈的现实世界中，制胜关键在于前置布局，敢于在合作开始前确立规则和优势。
4.  **领导者视角**，我们学到了“缺陷”的妙用和“分钱”的艺术，明白真正的强大在于整合人才而非事必躬亲。

记住，这一切的核心始于内心：**确信自己是谁，并敢于以此为基础去行动和表达**。
