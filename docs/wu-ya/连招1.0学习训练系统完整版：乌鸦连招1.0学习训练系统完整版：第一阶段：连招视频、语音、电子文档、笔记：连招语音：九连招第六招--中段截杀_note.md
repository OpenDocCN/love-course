上一节我们介绍了如何建立舒适感和联系感，本节中我们来看看如何利用“中段截杀”技巧，将处于暧昧阶段的关系正式确立下来。

中段截杀属于连招的第六招。在第五招建立联系感之后，双方已经相互吸引，处于信任和玩暧昧的阶段。此时关系突破就像一层窗户纸，但很多人因为错误表白或追求方式导致关系破裂。中段截杀的核心是**带领对方，而不是把选择权交给对方**。你要给出有限的选择（例如二选一），引导关系向你想要的方向发展。

这个阶段其实可以跳过。在过去的策略中，聊到舒适感阶段后就直接约会，结果具有很大不确定性，像是在赌博。对方可能接受约会，但关系并未明确，你的投入可能没有回报。

连招系统的设计是连贯的。通过陌陌、微信或世纪佳缘等渠道吸引妹子并聊天建立关系后，在中段截杀阶段还不宜直接邀约。**最高效的邀约时机是在完成“性期待”铺垫之后**。如果你熟练掌握连招，甚至可以实现第一次见面就发生亲密关系，这在过去被认为是神话。因为一旦通过中段截杀确立了恋人关系，双方在精神层面已经接受彼此，后续进挪（如牵手、拥抱、接吻）就没有障碍了。

当然，对于高手，也可以挑战不用中段截杀就直接约会并推进关系，但这属于更高难度的玩法。

以下是中段截杀最常用且高效的惯例：“七天情侣”。

**使用“七天情侣”惯例的步骤：**

1.  **铺垫与试探**：首先对她说：“我感觉你这个姑娘挺不错的，跟你聊天各方面都很舒服。”（此时可以适当暴露需求感）。然后接着说：“所以我突然很想知道，跟你在一块会是一种什么样的感觉。我有个想法。” **说完这句话后，停止回复，等待她的反应。** 这是一种试探，看她是否好奇并愿意接话。

2.  **抛出惯例**：如果她问“什么想法？”，说明她已猜到你的意图并愿意深入。此时，你可以说：“我想和你做七天的情侣，看看我们在一块会是一种什么样的感觉。如果合适，我们就续约；如果不合适，那我们就还是朋友。”

3.  **处理回应**：
    *   **顺利答应**：如果她爽快答应，例如说“好吧，我们试试看”。截杀成功。之后可以进行“善后工作”，例如通过“改口仪式”让彼此称呼“老公/老婆”，让关系感更强。
    *   **委婉拒绝**：如果她说“感觉还是做朋友比较好”等。这时，你要**立刻表现出无所谓的态度（IOD）**，跳过这个话题，回到自我引导，聊其他内容。但要注意，她之后可能会主动把话题绕回来。如果她绕回来，你可以再次尝试抛出“七天情侣”的提议，第二次成功率很高。如果她没有绕回，你需要重新加强舒适感和安全感，然后再一次尝试截杀。

**关键技巧：忽略前提话术**
这类似于一种陷阱设置。例如，在关系即将突破时，问“今天我们可以不带套吗？”，无论她回答“可以”还是“不行”，都意味着她**已经接受了“会发生关系”这个前提**。中段截杀的铺垫也运用了类似原理，让她提前进入“考虑恋爱关系”的框架。

除了“七天情侣”，还有其他可行的中段截杀惯例，以下是“书记选配偶”惯例。

这个惯例更具游戏感和戏剧性，能有效调动情绪。

**“书记选配偶”惯例步骤：**

1.  对她说：“经过组织上的一段考察，一致认为你这个同志非常不错。现决定，你成为吕书记（替换成你的姓）的媳妇候选人。请问你愿意接受组织上更深层次的考核和筛选吗？”
2.  她通常会说“愿意”。如果说不愿意，则退回重新建立吸引和舒适感。
3.  她同意后，进行一种“儿戏化筛选”：“你愿意为书记天天做饭吗？”“愿意陪书记熬夜并端茶吗？”等等。

5.  她回答“我愿意”后，你就可以说：“好的，老婆，现在可以改口了。”自然过渡到确立称呼的环节。

这个惯例通过半真半假、类似游戏的方式，让女性在轻松的情绪中接受关系的升级。

**蓝图规划示例：**
