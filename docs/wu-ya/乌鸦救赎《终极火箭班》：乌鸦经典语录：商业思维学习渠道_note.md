# 乌鸦救赎《终极火箭班》：乌鸦经典语录：商业思维学习渠道_note

在本节课中，我们将学习如何正确获取商业思维与赚钱方法。课程将澄清常见的错误观念，并指明系统学习商业知识的正规渠道。

---

许多人误以为赚钱可以靠运气或捷径。这种想法并不现实。赚钱是一门系统的学科，需要通过学习来掌握。

上一节我们澄清了关于赚钱的常见误解，本节中我们来看看如何系统地学习商业思维。

学习商业思维需要寻找正规、系统的教育渠道。你不能指望在大街上捡到钱，也不能依赖所谓的“江湖大师”。挣钱有特定的路径和方法。

以下是获取商业知识的几种正规渠道：

*   攻读**EMBA**（高级管理人员工商管理硕士）课程。
*   参加**工商管理**相关的培训班或课程。

通过这些渠道，你可以搞清楚商业是如何运作的。商业运作本身存在模型，但这个模型比简单的“连招”模型要复杂得多，因为它包含的要素太多了。

因此，你不要总想着如何在短时间内快速提高收入。那种“什么都不想管，什么都不想做，收入就想高起来”的思维是一种不切实际的思维。

赚钱存在科学的方法。如果你想提高收入，必须进行方法论上的学习。当然，我们这门课并不专门教授具体的赚钱方法。

如果你想学习这些具体方法，可以转向其他专业平台。以下是你可以进行自学的一些方向：

*   在**网上**搜索并学习正规**商学院**的公开课程或资料。
*   使用如**得到APP**这类知识服务平台，其中包含许多关于赚钱方法与商业思维的课程。

你可以去这些地方学习，了解赚钱的思维逻辑。商业的本质及其运作规律，是另一门独立的学科。它就像武术或医学一样，赚钱本身也是一个需要深入研究的学科。

---

本节课中我们一起学习了：建立商业思维不能依靠幻想和捷径，而应通过EMBA、工商管理课程等正规渠道进行系统学习。同时，我们指出了将赚钱视为一门复杂学科的正确心态，并推荐了得到APP等自学平台作为补充。记住，提升收入的关键在于方法论的学习。
