# 连招1.0学习训练系统完整版：第六课 - 中断截杀 🎯

在本节课中，我们将要学习连招系统中的第六招：中断截杀。这一招是关系升级的关键步骤，旨在将双方关系从普通联系感直接推进到恋人阶段。我们将详细解析其核心概念、操作技巧以及注意事项。

## 概述

中断截杀是连招中至关重要的一环。它能在建立足够的吸引和联系感之后，将男女关系升级为恋人关系，从而降低后续肢体接触的难度。成功的截杀建立在扎实的前期基础之上。

## 核心概念：试探与合理化借口

上一节我们介绍了关系升级的目标，本节中我们来看看实现升级的核心方法：**试探**与寻找**合理化借口**。

直接提出升级要求（例如“做我女朋友”）通常会导致拒绝。因此，我们需要一个间接的、能让双方都接受的“借口”来创造机会。

**公式：关系升级 = 合理化借口 + 逐步试探**

这个借口为双方行为提供了理由，降低了女生的心理防备。例如，邀请女生到家里“尝尝你的手艺”，比直接提出亲密要求更容易被接受。

## 关键技巧：试探的艺术

以下是运用试探技巧的步骤和原则：

1.  **试探的本质**：试探是以退为进的行为模式。通过间接询问对方对某事的看法或感受，来评估升级关系的可能性，而不会直接暴露最终目的。
2.  **处理反馈**：根据试探得到的反馈决定下一步行动。
    *   **积极反馈**：可以继续推进，逐步深入话题。
    *   **消极反馈**：应立即后撤，切换话题，避免施加压力。稍后可以再次尝试。
3.  **与死缠烂打的区别**：试探在遭拒后会后撤并择机再试；死缠烂打则是在被拒后持续纠缠，这会引起反感。关键在于让对方逐步习惯并“逆向合理化”你的行为模式。

## 三种截杀惯例详解

掌握了核心技巧后，我们来看看三种具体的截杀操作惯例。

### 1. 七天情侣惯例

这是一个经典但需谨慎使用的惯例。

*   **操作方式**：以“玩个游戏”为借口，提出尝试“七天情侣”。约定七天内体验情侣互动，七天后可续约或解约。
*   **注意事项**：此方法使用广泛，容易“撞招”，可能暴露需求感。应偶尔使用，而非作为唯一手段。
*   **应对推诿**：如果女生表现出犹豫而非明确拒绝，可以轻描淡写地强调“这只是个游戏”，并施加轻微压力（如“不玩我就找别人啦”），促使对方接受。

### 2. 以爱之名（恋爱筛选框架）

这种方法更为灵活多变，通过框架植入让女生入套。

*   **操作核心**：使用“**假如你是我的女朋友，你会...**”的句式，构建一个恋爱筛选框架。例如：“假如你是我女朋友，你会陪我一起做饭吗？”
*   **推进原则**：从暗示性较弱的问题开始，逐步增强。通过多次的框架筛选，让女生逐渐接受“作为你女朋友”这个设定，并自行合理化。

### 3. 包办式婚姻（角色扮演）

利用角色扮演在轻松愉快的情绪中植入关系。

*   **操作原理**：人在情绪欢乐、放松时更容易做出决定。通过共同进入一个虚构的角色扮演情景（如“组织考察”、“皇上选妃”），在互动中自然植入关系确认。
*   **关键步骤**：先通过**改变称呼**让对方认可扮演的身份（如“爱妃”、“同志”）。称呼是强化身份认同最强有力的手段，能为关系升级铺平道路。

## 关系确认与改口

无论使用哪种惯例，最终都需要通过“改口”来确认关系。

*   **改口的意义**：让对方使用亲密称呼（如“老公”），是关系确立的标志。
*   **操作策略**：可以引导或要求对方改口，但若对方强烈抗拒，不应强迫。可以退而求其次，使用“宝贝”、“亲爱的”等称呼。避免因强迫改口导致整个截杀失败。

## 截杀失败的处理

截杀结果只有成功或失败两种。

*   **成功**：继续推进后续连招（第七、八、九招）。
*   **失败**：意味着前期的吸引或安全感构建不足（跳连）。此时需要回到第一招“自我引导”进行二次吸引。
*   **重要提醒**：二次吸引比初次吸引更困难、耗时更久。需要权衡投入产出比，有时转向新目标可能是更有效率的选择。
