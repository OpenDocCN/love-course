# 乌鸦救赎《终极火箭班》第2课：预设价值观 🧠

在本节课中，我们将学习“预设价值观”这一核心概念。我们将探讨它如何影响你在关系中的行为和心态，并学习如何建立并贯彻一个强大、一致的预设价值观。

---

上一节我们介绍了预设价值观的重要性，本节中我们来看看如何具体填充和运用它。

预设价值观是你内心深处的核心信念，它决定了你在关系中的角色和行为模式。如果你的预设价值观不清晰或不强大，你的行动就会充满犹豫和障碍。

一个生动的比喻是：满汉全席听着很香，但你吃不着，那它就不如能立刻吃到的方便面有价值。同理，一个听起来美好但无法践行的价值观，远不如一个你能坚定执行的简单信念。

因此，在心态层面，你必须主动填充自己的预设价值观。你必须下定决心，选择并填充一个特定的预设价值观。

以下是需要你下定决心填充的核心预设价值观：

*   **我是领导者（Leader）。**
*   **我是关系的负责人。**
*   **这段关系由我来指挥和发展。**

具体到亲密关系中，这个预设价值观就是：**我是我女朋友的领导者。我要成为一个像父亲一样具有引导力的男朋友。这段关系由我来带领。**

当你能够在你的女人面前，真正做出并秉持这种预设价值观时，你会发现自己的行为障碍将大大减少。

以下是当你的预设价值观不匹配时，会出现的具体问题：

*   在下达指令时，你会感到存在障碍。
*   在对她发火时，你会害怕她离开。
*   在为她规划未来时，你会觉得自己很傻。

这些问题出现的根本原因，是你的预设价值观并非“领导者”。你的预设价值观可能是想要向她证明自己，或者认为两人关系完全平等，甚至觉得她比你更优秀、更强大。

这时，当你试图做出“领导者”的行为（如下指令）时，你的方法与内心信念是相互违背的。你整个人处于一种错位和拧巴的状态。就像你的大脑和你的手在往相反的方向用力。

因此，你必须完成“履职”。履职意味着，你必须将“我是领导者”这个预设价值观，形成一整套牢固的、内在的信念体系。

你不能只学习表面上的技巧和方法。你必须在预设价值观的层面彻底扭转过来，坚信**你是领导者，你是负责人**。

当这个坚实的预设价值观建立起来之后，你会发现自己所做的决策和行动都是自然且正确的，之前的那种内在障碍感和不协调感将不复存在。

---

本节课中，我们一起学习了预设价值观的填充与贯彻。核心在于，你必须下定决心，将 **“我是关系的领导者与负责人”** 这一信念内化为你行动的基石。只有价值观与方法统一，你才能摆脱内在冲突，在关系中顺畅地引领方向。
