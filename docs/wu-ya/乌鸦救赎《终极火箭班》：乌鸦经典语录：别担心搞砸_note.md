# 乌鸦救赎《终极火箭班》：乌鸦经典语录：别担心搞砸_note

在本节课中，我们将学习乌鸦老师关于行动与心态的核心观点：“别担心搞砸”。我们将深入探讨过度思考的危害，并学习如何通过行动和反思来打破焦虑循环，实现更有效的个人成长。

## 核心观点解析

上一节我们介绍了本课的主题，本节中我们来详细解析乌鸦老师语录的核心含义。

与其在担心搞砸时承受心态上的痛苦，不如直接去行动，哪怕彻底搞砸。行动后反思，可能反而不会搞砸。在担心失败、犹豫不决的焦虑中痛苦，不如彻底去尝试。搞砸之后，再思考做错了什么，这样更实际。至少行动能带来经验。直接去做，不要想太多。

很多人水平不高的原因，就是想得太多，做得太少。

## 过度思考的危害

理解了核心观点后，我们来看看“想太多”具体会带来哪些问题。

以下是过度思考的几个主要表现和后果：
*   想得永远比做得多。
*   因为想太多而不敢说话。
*   害怕说错话，担心伤害到别人。

你太高估自己了。以你的份量，说句话就能伤害到别人吗？别反而被别人一句话就怼到哭。你太小看别人的心理承受能力了。说句话还怕伤害别人？你怎么知道别人的心态是怎样的？你怎么确定你的话就会伤害到别人？更可能的情况是，你被别人伤害。

## 行动与反思的路径

认识到过度思考的危害后，本节我们来看看正确的行动路径应该是什么。

与其在担心和焦虑中痛苦，不如彻底去尝试，哪怕搞砸。失败之后再去反思，说不定反而不会搞砸。很多人就是因为想得太多。很多人的心智已经失灵了。每天大脑里的声音不停说话、反思、空想，想象一大堆事情，却什么都没做。自己坐在沙发里，感觉像做了一场噩梦。

本节课中，我们一起学习了乌鸦老师“别担心搞砸”的行动哲学。核心在于**用“行动-反思”的循环替代“空想-焦虑”的循环**。关键在于停止内耗，直接去做，哪怕结果不完美，其价值也远胜于无尽的空想与焦虑。记住，完成胜过完美，经验源于实践。
