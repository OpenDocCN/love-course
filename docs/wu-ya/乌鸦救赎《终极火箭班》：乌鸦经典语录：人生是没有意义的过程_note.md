# 乌鸦救赎《终极火箭班》第X课：🚀 人生是没有意义的过程

在本节课中，我们将要学习乌鸦经典语录中关于“人生是没有意义的过程”这一核心观点。我们将探讨如何通过调整对结果的看法，来改善我们在人际关系，特别是两性关系中的心态和行为模式。

---

上一节我们介绍了调整心态的几种方式，本节中我们来看看第三种具体方法：**缩小对结果赋予的意义**。

核心观点是：人生本身是一个没有意义的过程。你来到这个世界时什么也没有带来，离开时什么也带不走。因此，许多我们当下认为至关重要的事情，其实并没有那么重大的意义。

基于这个观点，你不需要认为“搞定一个女人”或“让一个女孩成为你的女朋友”是一件至关重要、不容有失的大事。这样的事情并不存在。

以下是这种心态在实践中的应用方式：

*   **发展多个选择**：你可以同时与多个对象发展暧昧关系。女性通常也是如此，她们身边一般也会有三五个可供选择的目标。
*   **自然筛选**：她们会顺其自然地观察，看谁表现更好，与谁的关系发展更快，然后选择与谁谈恋爱。
*   **避免过早定位**：不要一开始就在心里定位“这个女人就是我未来的女朋友”或“这个女人就是我未来的媳妇”。这种定位方式是不明智的。

如果你对目标产生了过高的预期，会导致什么后果呢？

后果是：你会变得非常害怕自己把事情搞砸。这种恐惧会导致你**畏手畏脚**，在行动时犹豫不决，反而不知道该如何正常表现和推进关系了。

---

本节课中我们一起学习了“人生无意义”视角下的实践心态。核心在于**降低对单一结果的过度重视**，通过**发展多个潜在选择**来减轻压力，并以**自然筛选**代替**孤注一掷**的目标定位。记住这个公式：**高预期 = 恐惧 = 发挥失常**。下一节，我们将探讨在这种心态下具体该如何行动。
