## 🧠 第一课：核心话术与执行流程

上一节我们概述了课程内容，本节中我们来看看“大情圣”技术的核心执行流程。

### 核心话术拆解
“大情圣”技术的核心在于一套标准化的四句话术，它能快速引导对话方向。

以下是标准话术流程：

1.  **开场与筛选**
    *   **你的话术**：`你喜欢直接还是委婉？`
    *   **作用**：这是一个关键筛选。绝大多数人会选择“直接”，这为后续快速推进奠定了基础。

2.  **意图声明与逆向合理化**
    *   **你的话术**：`那我直接来了。我觉得男人和女人之间没有友谊可言，要么在一起，要么是路人。女人提供繁衍，男人提供生存。有人可能会说有友谊，但我觉得那也是一些想搞定对方却假装朋友的人。`
    *   **作用**：声明你的直接意图，并为你的行为提供一个合理的解释（逆向合理化），降低对方的抵触感。

3.  **提出封闭式选择**
    *   **你的话术**：`我觉得男人和女人之间只有两条路可以走。`
    *   **战术停顿**：**此处必须停顿**，等待对方回应“哪两条路？”。这是引导对方进入你框架的关键一步。

4.  **双重绑定与最终选择**
    *   **你的话术**：`一个是约，一个是恋爱。直接了当，你想和我走哪个？选一个，或者都不选我消失。`
    *   **作用**：给出两个明确选项（双重绑定），并将“不选”的后果也明确给出，降低对方的压迫感，同时迫使对方做出选择。

### 执行要点与心得
在理解了核心话术后，我们需要注意以下执行细节，以确保流程顺畅。

以下是关键执行要点：

*   **节奏控制**：话术应分段发送，模拟真人聊天节奏。尤其在“只有两条路可以走”之后必须停顿，等待对方回应。
*   **平台规避**：在某些社交软件（如探探）上，直接使用“约”字可能触发风控。可以用拼音“yue”或与前后文字空一格来规避。
*   **应对“委婉”**：如果对方选择“委婉”，只需忽略“那我直接来了”这句话，直接开始逆向合理化部分即可，话术依然有效。
*   **心态管理**：此技术可批量操作。不要纠结于单个目标的得失，有人不选或失败，直接换下一个。你的核心优势是速度和数量。

## ⚡ 第二课：强化版技术与实战深化

上一节我们掌握了基础话术，本节中我们来看看如何通过“需求引导”技术进行强化，并处理后续互动。

### 强化版核心：需求引导
强化版“大情圣2.0”融入了“需求引导”技术。其核心原理是：通过让对方先承认对你有好感（需求感），来逆转框架，使后续推进更加顺畅。

它分为两个层次：

1.  **感受认同**：引导对方承认对你有“感觉”。例如：`我是你有感觉的类型吗？`
2.  **行为认同**：引导对方承认想与你发生特定“行为”。例如：`想躺在我怀里吗？`

**执行顺序**：应先从“感受认同”开始，逐步升级到“行为认同”。过早进行行为认同容易被拒绝。

### 实战流程：从筛选到邀约
让我们结合一个实战案例，看强化版话术如何与基础话术结合，完成从聊天到邀约的闭环。

以下是结合需求引导的实战步骤：

1.  **前置需求引导**：在发起标准话术前，先进行浅层需求引导。例如：`我是你有感觉的类型吗？` -> `（对方回应后）那就是对我的身体有感觉了？` 这为后续直接话术做了铺垫。
2.  **执行标准话术**：在氛围预热后，无缝接入上一课的标准“大情圣”话术流程。
3.  **处理“中奖”（对方选“约”）**：当对方选择“约”时，迅速切换到私密聊天环境（如微信），并开始进行“打消顾虑”的对话。
4.  **打消顾虑**：主动预判并解决对方的担忧（安全、照片真实性、交通等）。例如：`我们可以先视频一下。` `如果你过来，我帮你叫好车，明天早上也会帮你安排好。` 提供安全感是促成邀约的关键。
5.  **完成邀约**：在顾虑基本消除后，直接、清晰地提出邀约并确认细节。

### 后续互动：改口与关系定位
无论对方选择“恋爱”还是“约”，都可以通过“改口”来快速锚定关系。

以下是改口的具体方法：

*   **方法**：在对方选择“恋爱”后，跳过确认关系的问题，直接问：`你怎么称呼自己男朋友？` 然后根据对方回答，顺势说：`那我叫你媳妇/宝宝，你叫我老公。`
*   **原理**：改口是一种“标记”行为，像动物标记领地一样，能在心理上快速做实关系，减少对方的犹豫。
*   **备用方案**：如果对方害羞，可以使用“仪式感”话术：`网恋也得有个开始仪式，像拜天地。你先叫一次老公，以后可以不叫。` 通常对方会配合。

3.  **实战闭环**：我们分析了从开场、筛选、邀约到打消顾虑的完整实战流程，以及通过“改口”技术快速锚定关系的技巧。
4.  **核心心态**：这项技术强调效率和批量操作。不要害怕失败或拒绝，快速切换目标是其成功的关键。
