# 乌鸦救赎《恋商聊天课程》新1.0：6中段截杀._note

在本节课中，我们将要学习连招的第六招——中段截杀。这一招是关系升级的关键，用于在建立吸引和舒适感之后，将暧昧关系正式确立为恋爱关系。掌握此招，能有效避免错误表白导致的关系倒退，并大幅提升关系推进的效率。

## 课程概述

上一节我们介绍了如何建立舒适感和联系感，那是连招的第五招。当双方已经相互吸引、彼此信任，处于俗称的“玩暧昧”阶段时，关系就像一层窗户纸，一捅就破。本节中我们来看看如何优雅地“捅破”这层纸，即通过“中段截杀”技巧，安全高效地确立恋爱关系。

这个阶段的核心在于**带领**，而非把选择权交给对方。就像销售过程中，给客户太多选择会导致盲目，我们需要帮对方做出决定。在恋爱中也是如此，你需要给出明确的、对你有利的选项来带领关系前进。

## 中段截杀的重要性与可跳过性

中段截杀是一个可以跳过的阶段。在过去没有这个技巧时，人们聊到舒适感阶段后直接约会，结果充满不确定性，像是在“赌”。你可能请客吃饭、出去玩了一天，但尝试突破关系时却被拒绝，投资白白浪费。

为了避免这种尴尬和低效，我们设计了中段截杀。它的好处在于，**在见面之前就通过网络聊天确立恋人关系**。这样，双方在精神层面已经接受了彼此，见面后的进挪（如牵手、拥抱、接吻）将几乎没有障碍。熟练掌握后，甚至可以实现第一次见面就发生亲密关系，这已被证明是可行的。

对于高阶学员，当你们觉得无聊时，可以挑战更高难度：**不使用中段截杀，不确立关系，直接约会并达成亲密关系**。这为自己增加了游戏难度，也是向更高境界的迈进。

## 核心技巧一：七天情侣惯例 👫

这是我们目前使用频率最高、效果极佳的黄金惯例。以下是具体操作步骤：

**首先，需要进行铺垫和试探。**

你需要向对方暴露约20%的意图，让对方有所察觉但又不觉唐突。例如，你可以说：
```
我感觉你这个姑娘挺不错的，跟你聊天各方面都很舒服。所以我突然很想知道，跟你在一块会是一种什么样的感觉。我有个想法。
```
说完“我有个想法”后，**停止回复，等待对方反应**。这是一种试探。通常，对方会好奇地问“什么想法？”，这表明她已猜到你的意图并愿意继续沟通。

**接着，使用“忽略前提话术”抛出惯例。**

当对方问“什么想法？”时，你可以说：
```
（再次强调）我有个想法。我想和你做七天的情侣，看看我们在一块会是一种什么样的感觉。如果合适，我们就续约；如果不合适，OK，那我们还是朋友。
```
这运用了“忽略前提话术”，即跳过“是否要建立关系”这个前提，直接讨论“建立关系后如何”。对方无论回答“可以”还是讨论细节，都已落入“同意尝试”的框架。

**最后，进行关系确立的“仪式”和善后。**

如果对方愉快地答应了，你需要巩固关系。可以说：
```
但我们这样就算确立关系了吗？我总觉得好像什么都没变。我们来个仪式吧。
```
对方问什么仪式，你可以引导：
```
改口。叫我老公吧。
```
如果对方对“老公”称呼感到别扭，可以折中让其叫“亲爱的”。然后你可以问：“你喜欢我叫你老婆，还是媳妇？”让对方选择一个称呼。至此，恋人关系正式确立。

**如果对方第一次拒绝怎么办？**

如果对方委婉拒绝（例如“还是做朋友比较好”），切勿纠缠。要表现出**IOD（无兴趣指标）**，无所谓地略过此话题，回到自我引导，重新建立吸引和舒适感（通常是不安全感未消除）。之后可以再次尝试截杀，或者等待对方主动将话题绕回（这种情况也常见），第二次截杀成功率很高。

## 核心技巧二：书记选配偶惯例 📝

这是另一个有效的截杀惯例，更具戏谑和角色扮演色彩，能有效调动情绪。

**开场白可以这样说：**
```
感觉经过组织上的一系列考察，你这个姑娘挺不错的。经过组织一致性决定，批准你成为【你的姓氏，如吕】书记的媳妇候选人。请问你愿意接受组织上更深层次的考核和筛选吗？
```
**对方同意后，进行儿戏化的筛选：**
以开玩笑的口吻提问，例如：“你愿意为吕书记天天做饭吗？”“愿意在吕书记熬夜时端杯茶吗？”
**最后，进入“宣誓”环节：**
```
那么，我们来个仪式。你愿意无论贫穷、疾病、战争，还是（加入笑料）不孕不育，都愿意跟你的爱人吕书记在一起吗？
```
当对方说“我愿意”后，自然过渡到改口叫“老公/老婆”的环节。这种半真半假的游戏状态，降低了对方的心理防御，使关系确立更加顺畅。

## 截杀成功后的蓝图规划与性期待铺垫 🗺️

成功确立关系并改口后，工作并未结束。接下来需要进行**蓝图规划**，这仍在巩固舒适感和联系感。

**你可以规划未来，特别是第一次约会：**
例如：“我发现一家很棒的美食店，我们第一次约会就去那里吧。”“我们来规划一下第一次见面做什么吧，去密室逃脱还是吃西餐？” 让对方参与进来，共同想象和期待见面。

**紧接着，为后续的“性期待”话题做铺垫：**
有两种常见方式：
1.  **游戏切入**：提议玩互相提问的游戏，每人问对方五个问题，必须回答。前期问普通问题，最后自然过渡到：“我没问题了，问个骚点的吧…你最喜欢什么样的做爱姿势？”
2.  **直接提问切入**：例如，“老婆，我问你个问题…你是不是处女？” 无论对方回答是或否，都可以展开讨论。如果对方说不是，你可以说“我讨厌处女”，引起好奇，然后解释“处女不懂事、不会照顾人”，从而自然聊到性经历、喜好等话题。

**这些聊“性”的内容，是为后续邀约和见面后的亲密进挪做铺垫**。当双方能以恋人身份坦然讨论这些时，见面后的肢体升级将水到渠成。

## 关键原则总结

1.  **带领而非乞求**：永远掌握关系的主动权，给出选择（最好是二选一），带领对方前进，而不是把决定权交给对方。
2.  **持续试探**：在关系升级的每一步，都进行轻度试探，暴露部分意图，让对方心理上有所准备，避免感到唐突。
3.  **无所谓心态**：面对拒绝时，立刻展现IOD，撤回并重建吸引，保持姿态。
4.  **创造合理借口**：人是需要借口的动物。无论是确立关系的惯例，还是聊性话题，都是在创造一个合理、有趣、低压力的“借口”，让对方能够顺理成章地接受关系升级。
5.  **保持自然流畅**：整个流程追求平淡、自然、顺畅地推进，而非制造戏剧性的“惊喜”。这才是稳定和智慧的策略。
