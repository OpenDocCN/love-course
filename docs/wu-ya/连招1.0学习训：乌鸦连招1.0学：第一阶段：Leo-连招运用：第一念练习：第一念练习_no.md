在本节课中，我们将要学习连招心态模型中一个至关重要的部分——**第一念练习**。我们将探讨第一念的概念、它如何体现勇气，以及如何通过实践来突破思维中的限制。

## 什么是第一念？

上一节我们介绍了课程主题，本节中我们来看看第一念的具体含义。

第一念代表的是**勇气**。它指的是当你想做一件事时，脑海中浮现的第一个、最直接的冲动或想法。这个念头往往纯粹而有力，但紧随其后的，通常是怀疑、恐惧和借口，这些后续念头会筑成一堵“思维中的墙”，阻止你将想法付诸行动。

许多人都有这样的体验：想法很多，但实现的很少。总觉得有种无形的力量在限制自己，有些事情就是“不敢去做”。这种堵着你的感觉，就是我们需要识别和突破的障碍。

## 第一念的发现：一个亲身经历

为了理解第一念如何运作，我将分享自己发现它的过程。这段经历展示了从被“墙”阻挡到亲手将其推倒的完整心理路径。

当时我躺在住所，已经准备睡觉。我突然产生一个念头：想出去看看半夜的楼房。这个“想出去看看”就是**第一念**——一个简单的冲动。

紧接着，第二念（方法）开始出现：需要起床、穿衣服、走到门外。
然后，第三念（怀疑）立刻涌上来：这很傻，为什么要折腾自己？快睡觉吧。
最后，各种恐惧和借口（第四念及以后）将念头彻底搅乱，让人放弃行动。

但那次，我产生了叛逆心理，决定跟随第一念行动。我穿上衣服走到小区，看到了半夜的楼房。这个普通的场景却让我产生了强烈的反差感：原来思想和现实可以如此之近。如果只是躺着空想，那种“做不到”的憋屈感会不断堆积。

## 核心练习：用行动冲破“思维之墙”

理解了第一念的概念后，我们通过一个具体练习来体验如何用勇气冲破那堵“墙”。这个练习旨在让你清晰地感知到从“不敢”到“敢”的瞬间突破。

我进行的第二次实践更为极端。我给自己下了一个决心：要在一个人多的地方大声喊一句话——“我要成为亿万富翁”。在下班路上，我体验着接近焦虑的折磨感，就像准备搭讪却不敢上前时的挣扎。那种恐惧让人头皮发麻。

我给自己下了最后通牒：如果喊不出去，我这辈子都是个懦夫。我走到一个有二三十人的公交车站，在站中间喊出了那句话。那一刻，外在什么都没变，但内心感觉撕破了一片天，获得了巨大的**自由感**。那堵无形的墙被硬生生推倒了。

喊完之后，我感到前所未有的自由，并放声大笑。我理解了别人理解不到的东西：别人看我是傻逼，我看别人也是傻逼。这正是“别人笑我太疯癫，我笑他人看不穿”的写照。

以下是你可以尝试的练习步骤，用于亲身体验这种突破：

1.  **选择一个场景**：找一个有人的环境，如家里（有家人在）、公园、街道或车站。
2.  **确定你的“一句话”**：内容不重要，可以是任何你想喊的，如“我能行！”或“今天很棒！”。
3.  **感受“墙”的存在**：在准备行动时，仔细体会那种焦虑、恐惧和“不敢”的感觉，这就是思维之墙。
4.  **全力行动，不打折扣**：鼓起全部勇气，大声地喊出来。关键是要投入、彻底，而不是扭捏捏捏、打了折扣地去做。
5.  **体会突破后的感受**：完成后，感受内心的变化，体验那种冲破限制后的自由和力量。

如果你在做的时候感到扭捏或打折，做完后可能会更尴尬。只有全力以赴、发自内心地去完成，才能获得真正的突破感。

## 第一念与后续念头的模型

通过上面的练习，我们体验了行动的力量。现在，我们系统地解析一下行动过程中不同阶段的念头，这能帮助我们更好地理解自己的心理过程。

这个模型描述了从想法产生到最终放弃或行动的心理链条：

*   **第一念**：**勇气**。最初的冲动或想法 (`想法 = “去做X”`)。
*   **第二念**：**方法**。思考如何实现 (`计划 = “通过步骤A、B、C去做X”`)。
*   **第三念**：**怀疑**。恐惧和借口开始出现 (`怀疑 = “别人会怎么看我？这很傻吧？”`)。
*   **第四念及以后**：**混乱与放弃**。各种负面想法交织，最终说服自己放弃 (`放弃 = “算了，还是别做了”`)。

**勇气**和**懦弱**是相对存在的。平淡的生活中没有展现勇气的机会。只有当你感受到害怕、懦弱，并选择冲过去时，勇气才得以现身。看不见懦弱，就没有勇气现身的机会。

类似的练习，如在分手后到大街上大声唱歌，或者主动去搭讪，都能起到宣泄情绪、提升内在力量的作用。它们都是**第一念练习**的变体，核心都是**用行动克服恐惧**。

## 总结

本节课中我们一起学习了**第一念练习**的核心内容。

我们首先定义了**第一念**是代表**勇气**的最初冲动。然后，通过我的个人经历，大家看到了“思维之墙”如何阻碍行动，以及用极端练习（当众喊话）冲破这堵墙后所带来的巨大**自由感**。我们分析了从第一念到最终放弃的念头模型，并提供了可操作的练习步骤。记住，**勇气大于策略**。真正的成长始于识别你的“第一念”，然后凝聚全部勇气，不打折扣地付诸行动。

当你下次感到“不敢”时，就能清晰地看到那堵墙的存在。而推倒它的唯一方法，就是行动。
