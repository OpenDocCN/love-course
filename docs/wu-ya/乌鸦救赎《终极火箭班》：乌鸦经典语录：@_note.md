# 乌鸦救赎《终极火箭班》第1课：批判性谈话核心原则 🧠

在本节课中，我们将学习“乌鸦救赎”关于批判性谈话的核心原则。这些原则旨在帮助我们在讨论中保持逻辑清晰，避免无效的争执，专注于问题本身。

上一节我们介绍了课程主题，本节中我们来看看具体的核心语录与解析。

## 核心原则解析

**核心原则一：对事不对人，拒绝“身份”预设**
批判性谈话中，将对方说成“小孩”或类似身份预设是错误的。这不是关于对方是否成熟的问题。你可以继续批判他的观点，但不应采用“大人对小孩”这种不平等的对话姿态。有理不在身高，有志不在年高。不应以年龄或身份作为论据，而应就事论事。

**核心原则二：坚守逻辑阵地，拒绝人身攻击**
你需要在逻辑上说服我，不应将辩论上升为人身攻击。如果对话演变为人身攻击，我将终止谈话。需要理解的是，终止谈话的原因并非理亏，而是因为对方“谈话的电路有问题”——即对话方式与逻辑基础存在根本缺陷，使得有意义的交流无法进行。在这种情况下，你有权反过来批判他这种无效的沟通方式。

以下是基于以上原则整理出的行动要点列表：

*   **聚焦论点**：始终围绕讨论的具体事件或观点本身展开。
*   **剥离身份**：主动忽略并拒绝讨论双方的年龄、地位等与论点无关的身份标签。
*   **逻辑优先**：所有论证应建立在事实与逻辑推理之上，公式可概括为：**有效论证 = 事实依据 + 逻辑推导**。
*   **设立边界**：一旦对方开始人身攻击，明确指出的同时，有权暂停或结束对话。这可以用一个简单的条件判断来描述：
    ```python
    if 对话方式 == “人身攻击”:
        回应(“你的谈话电路有问题，我拒绝继续。”)
        终止对话()
    else:
        继续逻辑辩论()
    ```
*   **反批判权**：当对方使用无效的沟通方式时，你拥有直接批判其“沟通方式”的权利，这本身是批判性谈话的一部分。

## 课程总结

本节课中我们一起学习了“乌鸦救赎”批判性谈话的两大核心原则。首先，我们必须**对事不对人**，摒弃以身份压人的思维，专注于议题本身。其次，我们必须**坚守逻辑阵地**，明确拒绝人身攻击，并有权终止“电路有问题”的无效交流。掌握这些原则是进行有效、理性对话的基础。
