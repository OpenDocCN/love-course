# 乌鸦救赎《终极火箭班》：乌鸦经典语录：预设价值观_note

在本节课中，我们将要学习“预设价值观”这一核心概念。这是乌鸦经典语录中的关键思想，旨在帮助我们理解并克服行动中的心理障碍。

上一节我们介绍了课程背景，本节中我们来看看“预设价值观”的具体内涵。

## 核心概念解析

预设价值观指的是，我们内心预先设定的、关于“何种情况更糟糕”的潜在判断标准。它常常在无形中影响我们的决策与行动。

以下是该概念的核心要点：

*   比起搞砸事情所带来的丢人感，你自己因为什么都没做而产生的自责感，才会让你更痛苦。

这个要点可以通过一个简单的**公式**来理解：

**预设的价值观 = 认为“不作为的自责” > “行动后搞砸的丢脸”**

当你的内心预设持有此价值观时，它会推动你优先选择行动，而非因害怕失败而停滞不前。

## 如何应用

理解了这一概念后，关键在于将其应用于实际生活。这意味着在面临选择时，主动意识到并调用这一预设：**宁愿承受尝试后可能的不完美，也不要忍受因怯懦而毫无作为的悔恨**。

本节课中我们一起学习了“预设价值观”。它指出，内在将“自责感”置于“丢人感”之上的心理预设，能有效促使我们采取行动，减少因畏惧失败而产生的拖延与退缩。掌握这一观念，有助于我们在决策时克服恐惧， prioritise action over inaction。
