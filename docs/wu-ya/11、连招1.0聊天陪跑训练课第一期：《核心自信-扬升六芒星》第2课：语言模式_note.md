# 课程11：连招1.0聊天陪跑训练课第一期：《核心自信-扬升六芒星》第2课：语言模式 🗣️

在本节课中，我们将要学习如何通过语言模式来展现核心自信。上一节我们介绍了行为模式，本节中我们来看看语言模式。自信是一种气场，它由外部可观察的肢体语言、声调和语言模式共同构成。语言模式是让他人直接感受到你自信的重要途径。

## 语言模式的定义

以下是关于语言模式的几个核心定义，帮助我们理解其本质和作用。

1.  **语言是传递个人情绪和生存处境的直接途径。**
    我们可以通过语言表达当下的感受（如开心或不开心），也可以描述自己的生存状况（如讲述一个故事）。这是行为模式无法直接做到的。

2.  **语言可以违背实际心理状态。**
    行为模式很难违背真实心理，但语言可以。例如，即使内心难过，我们也可以用语言说“我很好”。这意味着语言具有“撒谎”或“润色”的可能性。

3.  **语言可以带领人们的意识离开当下，进行非现实的抽象探讨。**
    这是人类独有的能力。语言允许我们进行抽象化描述，探讨未来或概念性事物（如商业计划、人生理想）。这种能力是“讲故事”或“描绘蓝图”的基础。

4.  **通过措辞手段可以反映一个人的实际地位。**
    一个人使用的语言技巧（如推拉、引经据典、高价值展示）能够反映出他所在的圈层和社交地位。

5.  **系统化的情绪语言加数据佐证，可以呈现一个人的强大魅力。**
    将富有感染力的情绪化表达与具体事实（数据）结合，能极大增强个人魅力。这背后涉及“边际效用”等概念。

6.  **语言的潜在含义，影响远大过语言的本意。**
    人们往往更相信他们自己从话语中“解读”出来的含义，而非说话者直接陈述的本意。因此，利用“潜台词”进行沟通往往效果更佳。

## 让你看起来自信的语言模式

理解了语言的定义后，我们来看看具体哪些语言模式能让你显得更自信。以下是十种有效的模式。

**多用陈述句，少用疑问句。**
这会让你看起来毫无需求感。疑问句过多像是在索取价值，而陈述句则常用于展示价值。例如，把“你多大了？”换成“看面相你应该27了吧。”

**使用封闭式提问。**
这让你看起来连问问题都早有准备，自信满满。封闭式提问是给出选项（如A或B），而非开放问题。例如，把“我可以请你吃饭吗？”换成“我们是吃川菜还是日料？”

**用半拒绝的方式邀请别人，用无所谓的方式面对拒绝。**
例如，邀请时说“周末有空的话可以见面，如果不方便就算了。”如果被拒绝，则表现得很淡然。这能有效预防尴尬，并止损价值感。

**用推拉的方式调整情绪。**
这让你的说话看起来非常具有戏剧性和控制力。推拉即“说话大喘气”，先制造紧张（推），再释放放松（拉），能强烈刺激对方情绪。

**敢直接承认自己的目的和缺点，然后进行逆向合理化。**
这让你既真实又不显得低价值。例如，承认“我看到你第一眼就想和你亲密”，但紧接着解释“因为我是一个真实、不虚伪的人”。

**善用比喻和举例。**
用对方熟悉的事物来解释抽象概念，让你的话更容易理解，更有穿透力。

**没事干就筛选一下别人。**
“筛选”是把“对方要不要”的问题，偷换概念为“对方配不配”的问题。经常使用会让你不知不觉拿到社交框架。

**敢于下指令才是真正的自信。**
学会清晰、坚定地发号施令。初期可以采取“合作”口吻（如“我来做A，你来做B”），熟练后可以直接下指令。

**原则问题哪怕会得罪人也要拒绝。**
这样做之后，你反而会赢得尊重，更有魅力。拒绝时可以进行简单的逆向合理化。

**聊嗨了就是撤的时候了。**
在对话情绪最高点时主动结束，能让你的印象定格在最美好的瞬间。这是反人性的，但效果极佳。

## 让你看起来自卑的语言模式

了解了该做什么，也要知道应避免什么。以下是几种会让你显得不自信的语言模式。

**不断用提问来维持话题。**
这很low，应该用高价值故事来填充对话，而非不断提问。

**前硬后软，没有一致性。**
要么刚到底，要么别刚。前后不一致会让你成为“怂包”。

**别人拒绝你，你还在热情。**
这会让你迅速流失价值，看起来像是在讨好对方。正确的做法是快速用IOD（兴趣指标）回应或直接撤离。

**永远不要把自己丢人的事情告诉别人。**
这与“分享脆弱面”不同。分享脆弱面是展示逆境中的成长，而分享纯粹的丢人事只会损害你的形象。

**在别人表达想要之前，就先表达自己的想要。**
或者总是比别人更热情。这会使你陷入被动。正确的能量状态是：呈现自己时像火焰般热情，回应别人时像幽灵般稍冷。

**认为道歉有用。**
道歉只会让你看起来很low。如果想缓和关系，就用实际行动补偿（如买礼物、请吃饭），这比说“对不起”实在得多。

**说话充满攻击性。**
强势不是好事，但弱势也不是。社交是综合打法，需要在强势与温柔、谦虚与自信之间找到平衡。

**嫉妒别人的优秀。**
对于别人的优秀，要懂得欣赏和鼓掌。这展现的是你的心胸和格局，是一种差异化的高价值。

## 总结

本节课中我们一起学习了语言模式如何塑造自信形象。我们从语言的定义入手，理解了其传递情绪、抽象描述和蕴含潜台词的能力。接着，我们掌握了十种能彰显自信的具体语言模式，如多用陈述句、封闭式提问、推拉技巧等。同时，我们也明确了八种应避免的、会显得自卑的语言习惯。记住，自信的语言是内在价值的自然流露，也是可以通过刻意练习掌握的社交艺术。关键在于综合运用，灵活把握，并在实战中不断复盘和调整。
