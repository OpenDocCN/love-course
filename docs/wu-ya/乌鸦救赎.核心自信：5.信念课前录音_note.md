# 乌鸦救赎.核心自信：5.信念课 📜💡

在本节课中，我们将要学习“信念”这一核心概念。我们将通过分析一位学员的困惑，深入探讨预设价值观、技术与本心之间的关系，并理解如何建立和运用自己的信念系统。课程内容将帮助你厘清学习的目的，区分“术”与“道”，并找到个人成长的真正方向。

## 概述：学员的困惑与核心问题

一位跟随学习四年的学员提出了他的困惑。他通过学习掌握了处理两性关系与社交的方法，并成功找到了女朋友，对此他表示感谢。

然而，他开始质疑：爱一个人应该是发自内心的付出，希望对方过得好，并为当下和未来做计划。他感觉自己学习了“泡学”（指追求异性的技巧）后，很多时候在隐藏自己的需求感，对女性不再抱有强烈的需求感，反而更多地追求技巧层面的、非真实的自我表达。

他开始觉得，“泡学”所带来的影响，反而让他忘记了男人的担当——应该是带领自己的女人过上好的生活，而不是每天虚度光阴。他形容自己是一个“俗人”，一个“不成熟的幼稚”的人。

因此，他开始质疑所学的东西到底是为了满足生理需求，还是为了成为更好的自己。他预设的价值观是：男人应该对身边的每一个女人负责到底，而非玩弄感情。

他现在觉得，和女朋友在一起，可能是过于追求“术”而忘记了初心，整天浑浑噩噩，忘记了最初的那份担当。他今年22岁，感觉自己被无形的力量包围，认为爱人不应该是这样，对生活应该抱有更高的希望。

他提到曾两次学习“励志课程”，但感到思维困难，意识在慢慢恢复，对事物的认识维度在变化。

简而言之，他的核心问题是：**内心预设的“负责”价值观，与所学的“技巧”产生了冲突，导致他迷失了方向，质疑学习的根本意义。**

## 分析与解答：预设价值观与技术的本质

上一节我们概述了学员因价值观冲突而产生的困惑。本节中，我们来看看如何理解并解决这种冲突。

首先，这是**预设价值观之间的冲突**。你预设了“男人必须负责”的价值观，这本身没有问题。如果你认为这是对的，完全可以按照这个想法来设计你的行为。

关键在于，你忽略了一点：**技术（“术”）本身并无善恶，它取决于使用者的动机和方式**。如果你与一个女孩建立关系后，觉得自己应该对她负责到底，这是你的个人选择，没有人强迫你。

我（乌鸦）从未倡导过“不负责”的方向。如果你觉得对女友用了很多“术”，这源于两个原因：

1.  **你对“术”的驾驭感不强**。真正的复制，是复制一个人的精神或灵魂，而不是机械地复制他的技术。技术只是我内在灵魂与思维方式的外在体现。我通过教授具体的技术，是希望你们能看见技术背后的灵魂和态度。如果直接谈论灵魂，你们可能觉得空洞、难以复制。但若只停留在表面技术，不去深思其背后的精神源头，便是本末倒置。就像学习《道德经》，不应只背诵句子，而应去感悟老子在何种精神状态下写出这些话，为何他能想到这些。

2.  **善良的人同样可以运用策略**。如果你的初心是善的，你依然可以运用计谋和技巧。技术无罪，关键在于你如何使用它。难道善良的人就应该毫无策略，只靠所谓的“真诚”吗？如果只有真诚而没有智慧，那可能是愚蠢的。我也很真诚，但真诚需要智慧的引导。

## 核心概念解析：什么是预设价值观？🤔

上一节我们分析了问题根源在于价值观与技术的混淆。本节中，我们深入探讨“预设价值观”这个核心概念。

学员进一步问：预设价值观是什么？怎么实际操作？怎么创造？为什么明知有些事情（如早起、学习）有用，却还是拖延？如何改变预设价值观？

首先，**预设价值观（Preset Values）** 是什么？我在上一堂课已经讲得很清楚。你可以吸纳任何语句、思想，作为支撑你逻辑的预设价值观，甚至可以从电影（如《教父》）的台词中汲取。

**预设价值观不是具体的方法论（“连招”），而是你面对一件事物的根本态度和感受。**

我们可以用一个公式来理解事物、认知与预设价值观的关系：

```
事物本身 (Thing) + 预设价值观 (Preset Values) = 你所认知的世界 (Your Perceived World)
```

或者更直白地说：**`世界 = 事物 × 预设价值观`**

让我们通过一个例子来理解：

*   一颗**枸杞**，在外国人眼中，只是一种红色的果实。
*   但在许多中国人眼中，枸杞关联着“壮阳”、“补肾”等概念和画面。

枸杞本身没有变，变的是观察者带入的**预设价值观**。**事物本身无法决定它是什么，是你的理解（即你的预设价值观）决定了事物对你而言的意义。**

一块石头，它无法决定自己是“石头”。你之所以认为它是石头，是因为你被灌输了“这是石头”的概念。**你看到的世界，其实是你内心预设价值观的投射。**

因此，预设价值观有两个关键作用：
1.  你可以主动选择或创造自己的预设价值观。
2.  预设价值观塑造了你所看到的“真实”世界。

## 如何获取与运用预设价值观？🔍

上一节我们明确了预设价值观的定义和力量。本节中，我们来看看从哪里获取以及如何运用它。

关于从哪里吸收两性关系的预设价值观，答案就在“连招”里。“连招”中包含了大量的态度和心法，这些就是预设价值观的体现。你需要从中提炼的，不是步骤，而是背后的精神和看待问题的角度。

要改变“知道却做不到”（如拖延）的问题，关键在于提升你的意识层级。这涉及到“活在当下”的练习。

**“活在当下”是一种通过觉知来休息心智、连接内在的方法。** 它并不神秘。例如，当你专注于观察手掌的纹路时，你只是在“看”，没有加入任何预设价值观（如“这纹路代表命运”）或情绪。这时，你便处于一种“觉知”状态。

*   **好处1：心智得到休息**。思维和情绪停止消耗你的精力。
*   **好处2：与内在自我保持连接**。你不再被纷乱的思绪（想女人、想工作、想理论）带走，而是能时刻感知到自己内在的状态。

通过这种练习，你能在情绪（如愤怒、懒惰）刚萌芽时就觉察到它，而不是等情绪爆发后才后知后觉。因为你始终与内在保持联系，就像时刻在监控自己的内心状态。这样，你就能在负面预设价值观或情绪产生巨大影响之前，化解它们。

## 总结：信念的纯粹力量 💪

本节课中，我们一起学习了“信念”的核心内涵。

我们通过一位学员的困惑入手，探讨了**预设价值观**与**技术（术）** 的关系。我们明白了：
*   技术是工具，其价值由使用者的初心和信念决定。
*   **真正的学习是复制精神，而非机械模仿技术。**
*   **预设价值观是你解读世界、做出反应的底层代码**，它决定了事物对你的意义。
*   你可以主动从各种来源（如课程、电影、经历）中吸收和创造积极的预设价值观。
*   通过练习“活在当下”的觉知，可以更好地管理情绪，保持与内在自我的连接，从而让信念稳定而清晰。

信念本身很纯粹，它就是你所坚信并践行的根本原则。建立强大的核心自信，始于厘清并坚守对你而言真正重要的信念。希望本课能帮助你找到属于自己的方向，不再迷失于技术与本心的冲突之中。
