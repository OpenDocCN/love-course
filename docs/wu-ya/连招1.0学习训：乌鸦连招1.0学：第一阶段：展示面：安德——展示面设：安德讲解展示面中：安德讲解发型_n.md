# 发型设计基础教程 🧑💇♂️

在本节课中，我们将要学习发型设计的基础知识，包括如何根据发质打理造型、如何认识不同脸型以及如何选择与之匹配的发型。课程内容将涵盖从基础造型技巧到发型与脸型搭配的完整流程。

## 第一节：基础造型技巧

上一节我们介绍了课程概述，本节中我们来看看针对头发偏软、不易定型的发质，如何进行有效造型。

以下是详细的造型步骤：

1.  **吹干头发**：低下头，使头发自然下垂。使用吹风机的最大热风档，边吹边用手来回拨动头发，确保热风均匀吹拂，避免集中一点损伤发根。
2.  **初步定型**：在头发仍处于低头状态时，薄薄喷上一层干胶（定型喷雾）。喷完后不要立刻抬头。
3.  **梳理发胶**：抬起头，用宽齿梳将头发上已喷的干胶快速梳开，使其恢复柔软状态。
4.  **涂抹发泥**：取适量发泥在掌心均匀揉开，然后涂抹在头发上。反复揉搓，确保每根发丝都沾到发泥，使头发呈现蓬松、凌乱的状态。
5.  **塑形与最终定型**：用手或吹风机简单调整出想要的发型走向（如向左、向右）。最后，喷上两到三层干胶进行最终定型。

**核心概念**：此方法的关键在于利用干胶的初步固定和发泥的塑形能力，通过分层处理解决头发软塌的问题。公式可概括为：**吹干 + 薄胶固定 + 发泥塑形 + 干胶最终定型**。

## 第二节：造型产品解析

上一节我们介绍了具体的造型步骤，本节中我们来了解一下常用的造型产品及其区别。

以下是三种主要造型产品的特点：

*   **干胶（发胶/定型喷雾）**：呈空气状喷雾，使用后清爽不油腻，定型效果强，能使头发变硬。
*   **发泥**：呈固体、非透明膏状。质感偏自然，不油腻，适合发质偏软、发量中等或较少的人群使用。
*   **发蜡**：呈透明或果冻状。光泽感强，偏油亮，适合头发较干或发量较多的人群使用。

**核心概念**：选择产品的关键在于发质和 desired look（想要的效果）。**发质软/少 → 发泥；发质干/多 → 发蜡；最终定型 → 干胶**。

## 第三节：脸型分类与特点

上一节我们了解了造型工具，本节中我们来看看发型设计的一个重要参考维度——脸型。认识脸型是选择合适发型的基础。

常见的脸型主要有以下七种：

1.  **椭圆形脸（鹅蛋脸）**：被视为标准脸型，适合绝大多数发型。
2.  **圆形脸**：给人温柔可爱的感觉。发型选择较多，需注意修饰两侧头发，避免让脸型显得更圆。短发长度不宜过短。
3.  **长方形脸**：应避免完全露出额头，适合留有刘海以缩短脸型视觉长度。不宜留长直发。
4.  **方形脸（国字脸）**：轮廓刚毅，缺乏柔和感。适合留有柔和线条的较长发型，避免留长鬓角，两侧头发剪短更显精神。
5.  **正三角形脸**：额头较窄，下巴宽。刘海宜薄，长度齐眉或以上，不宜超过眉毛。
6.  **倒三角形脸**：额头宽，下巴尖。头顶头发可稍长，适合西装头、莫西干等发型，两侧头发宜短。
7.  **菱形脸**：颧骨宽，额头和下巴窄。两侧头发不宜过短，以免突出颧骨和耳朵。

**核心概念**：发型的作用在于修饰和平衡脸型轮廓。例如，**圆形脸用两侧头发修饰宽度，长形脸用刘海修饰长度**。

## 第四节：发型与脸型搭配实例

上一节我们认识了各种脸型，本节中我们结合具体发型，来看看如何为不同脸型选择合适的发型。

以下是针对部分脸型的发型搭配建议：

*   **倒三角形脸**：适合头顶较长、两侧较短的发型，如西装头、莫西干或凌乱的刘海发，可以使脸型比例更均匀。
*   **方形脸**：适合短发，以突出刚毅特征。侧背头、复古背头是不错的选择，但应避免四六分发型，以免强调下颌宽度。
*   **菱形脸**：避免两侧头发过短。可选择长度适中、有一定体积感的发型来平衡突出的颧骨。
*   **长形脸**：发型重点不在遮盖，而在自然过渡。适合长度适中、两侧不过分夸张的发型，如侧分短发。
*   **椭圆形脸**：几乎可以尝试任何发型和长度，选择范围最广。

**核心概念**：搭配的原则是**扬长避短**。例如，方形脸用短发和侧背头**强化**阳刚之气，倒三角脸用头顶发量**平衡**上下宽度。

## 第五节：流行发型名称与风格

上一节我们探讨了脸型与发型的搭配，本节中我们来熟悉一下常见的男士发型名称及其风格，帮助大家更好地与发型师沟通或寻找参考。

以下是部分常见的男士发型类别：

1.  **板寸**：极短的短发，由平头演变而来，干净利落。
2.  **毛寸**：比板寸略长，头发常竖起，有层次感和精神。
3.  **碎发**：将头发修剪出层次和凌乱感，长短皆可。
4.  **分头**：头发有明确分路，如三七分、四六分，常见于职场，时尚感强。
5.  **背头**：将前额及头顶头发向后梳理，整齐成熟。
6.  **纹理烫/卷发**：通过烫发制造卷曲纹理，需要打理以呈现自然弧度。
7.  **朋克发型**：部分或全部头发向上竖起，个性鲜明。

**核心概念**：发型名称对应着特定的**视觉风格**和**打理方式**。例如，**背头**代表**整齐、复古**，**纹理烫**代表**蓬松、时尚**，需要更多日常打理。

## 第六节：商务发型与造型态度

上一节我们了解了各种发型名称，本节中我们聚焦于适合职场环境的商务发型，并分享关于发型选择的个人态度。

商务发型通常要求干净、整洁、得体，体现专业与自信。常见的商务发型包括板寸、毛寸、碎发、分头和背头等。侧分背头、四六分短发等都是兼具时尚与稳重的选择。

**核心概念**：商务发型的核心是 **“前不遮眉，侧不遮耳，后不及领”** ，保持定期修剪（约20天一次），避免过长或过于夸张的染烫。

在选择发型时，不必过分拘泥于“脸型必须配某种发型”的规则。发型可以尝试和改变，重要的是找到能让自己感到自信和精神的造型。头发是可再生的，大胆尝试是发现最适合自己发型的最好方式。
