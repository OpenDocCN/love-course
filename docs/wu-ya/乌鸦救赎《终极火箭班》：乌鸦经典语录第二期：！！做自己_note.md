# 乌鸦救赎《终极火箭班》：乌鸦经典语录第二期：！！做自己_note

在本节课中，我们将学习如何化解自身情绪，专注于当下，并真正地“做自己”。课程将通过一个简单的互动练习，揭示我们内心阻碍的来源，并解释“做自己”的核心含义。

上一节我们介绍了课程的主题，本节中我们来看看如何通过具体行动理解“活在当下”。

现在，请大家一起做一个动作。请模仿我，非常大声地喊一声，让所有人都能听见。

以下是练习后的观察与互动：
*   敢于喊出来的人，请在聊天区扣“1”。
*   喊完之后，出事了吗？
*   有人把你当作神经病吗？

最多的情况可能是你的父母问你“你有病吗？”。但他们有没有因此对你采取实质行动？

通过这个练习，我们可以看到许多人的困境所在。有多少人是活在自己的幻想中，自己吓唬自己？我们常常用未来可能发生的事情来恐吓自己。

你看到问题的界限在哪里了吗？它不在外界。所有的阻力永远都不在外界。

我是通过这个非常小的例子来说明这件事：所有的阻力都不在外界，而是在你自己的脑海当中。你自己给自己编织了一张网，让你自己跳不出去。

没错，所有的阻碍都是在心里。比如，你担心“我妈说我神经病”。

那么，什么叫做“做自己”？做自己就是不去在意别人的看法。因为别人的回应是未来的事。

过去的挫败会让你变得不自信，而未来的担心会让你变得恐惧。负担源于过往的经历，恐惧源于对未来的猜想。

什么叫做活出自己？那就是“喊了又怎么了？”。我就是我自己，我做我想做的事情，这一刻我就是我自己。

你们去喊的时候，可能身边有人，可能是在网吧。我记得有一次讲公开课，说到这个点时，发现很多听众就在网吧。

自卑源于过去的创伤，是内心自己留下的陈旧伤痕。而恐惧则源于对于未来的猜想，害怕别人会如何回应你。这些恐惧都指向未来。

自卑是因为什么？初生牛犊不怕虎。如果一个人没有受过伤，没有经历过挫败和自我否定，他就不会自卑，不会在别人面前抬不起头。这都是因为过去有陈旧伤痕。

那可能是在某个特定时刻，做了某件事，然后被别人嘲讽或挖苦过，从此便形成了“我不行，我做不到”的念头。

本节课中我们一起学习了情绪与自我阻碍的根源。我们通过一个简单的呼喊练习，认识到阻力往往源于内心对过去创伤的负担（`自卑 = 过往挫败`）和对未来回应的恐惧（`恐惧 = 未来猜想`）。“做自己”的核心在于**专注于当下，不被他人的看法（未来回应）所束缚**，打破自我编织的心理牢笼。
