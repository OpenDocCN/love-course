# 乌鸦救赎《终极火箭班》：乌鸦经典语录：别在意别人怎么看_note

在本节课中，我们将学习乌鸦救赎的核心观点之一：如何摆脱他人眼光的束缚，专注于自身的策略与自在。

## 核心理念阐述

上一节我们介绍了课程主题，本节中我们来深入理解核心理念。

当你依据他人的想法行事时，结果也未必如人所愿。即便你采取了非常有策略的行动，你的思维也无法精确计算世界上发生的所有事情。

**核心公式**：`他人认可 ≠ 预期结果`

## 策略的局限性

理解了核心理念后，我们来看看策略本身的局限性。

即便将问题限定在一个很小的范围内，你依然无法进行完美计算。错误仍然会出现，你依然可能被他人指责。

以下是策略局限性的具体表现：
*   思维无法处理全局复杂性。
*   在有限条件下仍会出错。
*   无法避免他人的负面评价。

## 关键收获：自在的价值

认识到策略的局限后，最关键的一点在于转向内在的平衡。

但是最关键的一点在于，你是否感到自在。

**核心代码**：`if (focusedOnSelf) { state = "自在" }`

本节课中我们一起学习了乌鸦救赎的重要观点：过度关注他人看法和追求完美计算并不可取。真正的关键在于采取有策略的行动，同时接纳不完美与批评，最终实现内心的自在。记住，你的价值不应由他人的评价来定义。
