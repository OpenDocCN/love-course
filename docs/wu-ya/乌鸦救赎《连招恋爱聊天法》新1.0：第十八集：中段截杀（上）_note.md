# 乌鸦救赎《连招恋爱聊天法》新1.0：第十八集：中段截杀（上）💘

在本节课中，我们将要学习连招恋爱聊天法的第六招——中段截杀。这一招的核心在于，如何在双方已建立吸引和暧昧关系后，高效地确立恋爱关系，避免因错误操作（如盲目表白）而导致关系倒退或失败。

## 课程概述

上一节我们介绍了舒适感与联系感阶段。当完成第五招建立联系感后，双方已产生相互吸引和心动感觉，彼此信任，关系进入俗称的“暧昧”阶段。此时，关系突破就像一层窗户纸，非常容易。但许多人因操作不当，如贸然表白，反而破坏了关系。本节将详细讲解如何利用“中段截杀”技巧，安全、高效地将暧昧关系升级为确定的恋爱关系。

## 中段截杀的意义与可跳过性

这个阶段的核心是带领对方，而非把选择权完全交给对方。就像销售过程中引导客户做决定一样，你需要给对方提供有限的选择（例如二选一），带领关系向前发展。

值得注意的是，中段截杀这个阶段本身是可以跳过的。在早期的技巧中，并没有这个环节。那时，人们聊到舒适感阶段后便直接邀约，现场能推进到什么程度就到什么程度，这很大程度上是在“赌”运气。你不敢表白，关系未定，只是暧昧，一次约会可能毫无进展，投入的时间精力可能白费。

这里有一个生动的例子：一个男孩约女孩出去，买了大量零食看电影，中途忍不住表白被拒。女孩随后尴尬地问：“但我还能吃吗？”这个笑话揭示了在关系未确立时进行模糊邀约的风险——你可能会沦为单纯的“饭票”或“陪伴者”。

## 确立关系后邀约的优势

我们的连招设计是一脉相承的。通过陌陌、微信或世纪佳缘等渠道吸引女生，聊天建立吸引，再到中段截杀确立关系。**在信息期待（即双方对见面有强烈期待）时进行邀约，是效率最高的。**

如果你熟练掌握连招，甚至可以实现 **第一次见面就发生亲密关系**。这在过去被视为神话，但现在已成为事实。因为一旦通过“中段截杀”确立了恋爱关系，你们在名义上和情感认知上已经是恋人。此时进行身体接触的障碍会小很多。

例如，在代聊时，可以直接说：“第一次见面时，我要牵你的手，到时候你千万不要拒绝我。”对方同意后，见面牵手、拥抱、接吻都会很自然。因为对方在精神层面已经接受了你作为恋人的身份，肉体上只是需要一点习惯的过程。流程上只需稍加引导，比如使用“一分钟牵手惯例”，在私密空间逐步升级亲密行为。

## 给高阶学员的挑战与人生规划

对于学习一段时间后感到“无聊”的高阶学员，可以尝试增加难度：**不使用中段截杀技巧，不确立关系就直接邀约，并尝试首次约会就发生亲密关系。** 这更具挑战性，可能需要两次约会。当找对象对你而言已完全不是问题时，你可以为自己设计更丰富的人生，例如经营长期关系、挑战更高难度的社交玩法等。

**你的人生由你自己规划。** 你的担心和期待都可能变成现实。将注意力集中在如何做成自己想做的事情上，不必活在他人的评价和框架中。`5 + 4 = 9`，`6 + 3` 也等于 `9`，人生有无数种精彩的活法。

## 核心技巧：七天情侣惯例

现在，我们言归正传，回到中段截杀的具体技巧。目前我们社群中使用频率最高的惯例是 **“七天情侣”**。这是一个王牌级的黄金惯例，具有改革意义，效果卓著。

以下是使用“七天情侣”惯例的具体步骤和注意事项：

**首先，需要进行铺垫。** 铺垫的目的是让对方感知到你20%的意图，避免提议显得过于唐突。

具体操作时，可以先给对方一个赞美：
> “我感觉你这个姑娘挺不错的，跟你聊天各方面都很舒服。”

**在这个阶段，可以适当暴露需求感。** 因为吸引已经存在，且处于暧昧期，适度的需求感表达是安全的。接着，可以引出核心提议：
> “所以，我突然就很想知道，跟你在一块会是一种什么样的感觉。”

这句话为后续提出“七天情侣”的试交往提议，做好了自然的情绪铺垫。

## 关于社交软件与展示面的补充

工欲善其事，必先利其器。使用如世纪佳缘等社交软件时，**高质量的展示面（个人资料照片）** 至关重要，能显著吸引女生主动联系。考虑开通**钻石会员**可以解除发信限制，极大提升效率。但请注意，一个城市的用户资源有限，不要一次性匹配过多导致精力分散，聊一个成一个才是关键。
