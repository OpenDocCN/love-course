# 乌鸦救赎《终极火箭班》第二课：语言侵略话术 🚀

在本节课中，我们将要学习“语言侵略话术”的核心概念与应用。这是一种通过提前使用下一阶段关系中的语言来突破当前关系界限的沟通技巧。

## 概念解析：什么是语言侵略？

上一节我们介绍了暗示技巧，本节中我们来看看语言侵略。语言侵略的核心是**提前把下一个阶段才会出现的话语和场景，在当前关系中说出来**，以此试探并推动关系发展。

其核心公式可以概括为：
**语言侵略 = 当前关系 + 下一阶段关系的话语**

## 实战案例剖析

我将通过一个具体对话案例，来展示语言侵略话术的实际应用。

我与一位女性讨论“男女复刻”的问题。我判断她存在复刻方面的毛病。当时我们并非恋爱关系，但我直接使用了语言侵略。

我对她说：“你有那个复刻病，赶紧去看一下。如果以后生不了孩子可不要你。”

她回复道：“不要你别要。”

## 话术的应对与圆场策略

当对方做出拒绝或负面回应时，你需要有应对策略。以下是两种核心的圆场方法。

一旦被对方拒绝，你有两种处理办法。

*   **有话题可圆回**：如果存在逻辑上可以衔接和解释的话题，就顺势圆回去。例如，我当时的回应是：“不要你不是我不要你，是你老公不要你，你想啥呢？” 这就是利用“未来老公”这个下一阶段的概念，将话题圆了回来。
*   **通用话题：开玩笑**：如果实在无法圆回，则使用“开玩笑”作为通用话题。你可以说：“随便跟你开个玩笑而已。” 但需注意，使用这种通用话题时，效果通常不是最佳。

## 技巧总结与核心要点

本节课中我们一起学习了语言侵略话术的精髓。

语言侵略，即我们刚才所讲的暗示的进阶应用，其本质在于**突破关系**。通过大胆假设并说出未来关系中的话语，来测试对方反应并创造推进关系的契机。

记住，关键在于自然地将“未来场景”植入“当前对话”，并根据对方的反馈灵活运用圆场策略。
