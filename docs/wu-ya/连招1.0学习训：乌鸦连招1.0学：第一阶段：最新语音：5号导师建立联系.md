# 连招1.0学习训：乌鸦连招1.0学：第一阶段：最新语音：5号导师建立联系

在本节课中，我们将要学习如何与女生建立“联系感”。这是恋爱关系发展中的一个关键阶段，旨在让女生感受到你的真诚和建立关系的诚意，从而为后续发展打下坚实基础。

## 为什么要建立联系感？

上一节我们介绍了吸引力的重要性，本节中我们来看看为什么在吸引之后需要建立联系感。

建立联系感的主要作用是向女生展现真实的自我，并让她感受到你认真对待这段关系的诚意。在早期与女生相处的模式中，常见做法是送礼物、夸赞，但这往往导致被贴上“好人”标签。前四招建立了吸引力，让女生认可你的魅力并产生进一步了解的倾向。此时，若你继续仅以玩笑或玩闹的方式互动，女生会认为你并非真心想谈恋爱。

因此，联系感的核心目的是：**让女生感受到我们展现真实自我的意愿，以及与她认真交往的诚意。**

## 如何建立联系感：四个信息交换点

建立联系感的第一步，是与女生深入交流，让她更了解你。这需要通过交换四个层面的信息来实现，从而构建安全感和相互了解。

以下是建立联系感的四个核心信息点：

1.  **家庭信息**
    这是相对外围的信息，与恋爱无直接关系。做法是简单了解彼此父母的工作等基本情况。例如：`“你父母是做什么工作的？”` 同时分享自己家庭的信息。完成这一步，家庭信息交换即告完成。

2.  **社交圈信息**
    此部分比重也较小。主要目的是让彼此了解对方的朋友圈和相处模式。你可以分享与朋友的互动，例如：`“今天和朋友健身很开心，之后还一起去玩了。”` 这侧面展示了你的社交生活和价值观。分享时，应侧重朋友积极、有价值的一面。

3.  **个人生活信息**
    这是比重较大的关键点，涉及恋爱中更重要的方面。内容包括：**工作、学习、饮食起居、个人偏好、性格特点**。分享这些能让女生立体地了解你。对于不知如何开启话题的学员，可以从这些方面进行自我引导，将良好的生活状态展示出来。

4.  **恋爱经历信息**
    此点涉及彼此过去的恋爱经历，例如恋爱次数、深刻体会。分享时可以使用“故事惯例”。例如“省钱惯例”：讲述一个过去恋爱中，因对方只把自己当“饭票”而醒悟的经历，并引申出现在约会时会观察对方是否懂得互相付出的观点。这个故事能传递你的恋爱观，并无形中影响对方后续的行为模式。

在与女生交流这四个信息点时，方式可以相对随意，通过提问或自我引导进行自然交换。

## 适度的关心与矛盾表达

仅仅交换信息是不够的。要让女生真切感受到恋爱的诚意，还需要付出关心的行动。但关心需要把握尺度，避免过度。

如何把握关心的“度”？关键在于 **“像父亲一样矛盾地表达”**，即结合“推”（拒绝、调侃）和“拉”（接纳、关心）的技巧。

例如，女生说：`“今晚一起去吃小龙虾吧。”`
你可以先“推”：`“不行，你脸上痘痘这么多还吃小龙虾？”`
过一段时间后，再“拉”：`“走，带你去个神秘地方。”` 然后带她去小龙虾店。

这种矛盾的表达，既体现了关心（担心她长痘），又展现了主见和魅力（最终带她去），避免了过度讨好。其核心是 **“霸道又温柔”**，嘴上开玩笑拒绝，行动上却强势地满足对方。

## 创造共鸣感

在通过信息交换和适度关心建立了基本安全感后，下一步是深化连接，创造共鸣。

创造共鸣的第一个方法是寻找 **相似性**。当发现彼此来自同一城市、有相同爱好（如健身、音乐、旅游）或相似人生经历时，会迅速产生熟悉和亲切感。“老乡见老乡，两眼泪汪汪”就是相似性的体现。关键在于**共同的兴趣爱好和人生经历**。

然而，在整个连招体系中，快速创造深层共鸣的核心点是：**分享秘密与脆弱面**。

*   **秘密**：分享那些让你感到羞愧、从未告诉别人的事情。
*   **脆弱面**：分享过去让你深感伤痛或触动的经历（例如之前“省钱惯例”中提到的失恋痛苦）。

展现脆弱面能让女生看到你坚强外表下的柔软内心，极大地增强情感连接。

## 共同行动与共谋

共同完成一件事能极大地强化联系感。例如：**一起计划并完成一次旅行、共同看一场电影、商议为彼此父母购买礼物**。

这种“共谋”行为，就像当家人与外界发生矛盾时，你会毫不犹豫地站在家人一边。与女生共同谋划和完成事情，能使你们成为“自己人”，感受彼此依靠，从而让连接感更加深刻和牢固。

## 课程总结

本节课中我们一起学习了建立“联系感”的两个核心部分：

1.  **建立安全感**：通过交换**家庭信息、社交圈信息、个人生活信息、恋爱经历信息**这四个层面的信息，并结合**适度的走心关心**以及**像父亲般矛盾的表达**（推拉结合），让女生感受到你的真诚和交往诚意。
2.  **创造共鸣感**：通过发现**相似性**，并关键性地**分享秘密与脆弱面**来快速建立深层情感连接。最后，通过**共同行动与共谋**（如一起旅行、规划事情），将彼此关系紧密联结。

掌握这些要点，你就能有效地与女生建立真诚而深入的联系感。
