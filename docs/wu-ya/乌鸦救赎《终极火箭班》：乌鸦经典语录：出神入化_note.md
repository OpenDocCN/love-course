# 乌鸦救赎《终极火箭班》第1课：出神入化 🚀

在本节课中，我们将学习“出神入化”这一核心学习状态。我们将探讨其表现、价值以及如何珍惜和利用这种状态来加速个人成长与知识积累。

## 学习中最忌惮的“着魔”

学习中最忌惮的是“着魔”。因为一旦着魔，这个人必定会成为专家。正如一句老话所说：不疯魔，不成活。当你研究某个领域达到非常专业、精深的程度时，你会发现自己的反应变得机械化。



## 机械化反应的体验

上一节我们提到了专业状态下的机械化反应，本节中我们来看看具体的表现。达到那种状态时，会产生一种独特的心理体验。我本人曾经历过这种“不疯魔不成活”的状态。你会发现，自己面对他人以及对他人的反应，都变成了一种预设的、机械化的模式。



例如，人类普遍存在一种“好纠错”的心理。只要我说错一句话，立刻就会有一群人出来纠正我的错误。比如，数一数，一、二、三、四、五、六、七、八、九，九个人。这种反应就是典型的机械化反应。达到这个境界会非常有意思。



## “出神入化”状态的价值与珍惜

当你一旦达到“出神入化”的境界时，你学习新知识的速度会变得特别快。你能够从中反思和领悟出许多道理。



你必须珍惜这种时光。这种状态非常稀有，可能只维持几个月，有些人可能只有几天，有些人则可能维持得更久一些，比如半年或六个月。



一定要珍惜这段时光，因为你最初的知识积累，往往都是在这种状态下完成的。在这种状态下，你的学习效率和取得的成就会呈几何倍数增长。这其实就是进入了一种“心流”体验。如果你不了解“心流”是什么，可以去查询一下。
