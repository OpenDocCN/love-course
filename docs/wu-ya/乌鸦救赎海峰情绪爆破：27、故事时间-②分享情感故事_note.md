# 乌鸦救赎海峰情绪爆破：27、故事时间-②分享情感故事_note

在本节课中，我们将要学习“故事时间”的第二个核心技巧：分享情感故事。我们将探讨分享情感故事的时机、常见契机以及如何利用这些契机进行深入交流。

上一节我们介绍了故事时间的基本概念，本节中我们来看看如何具体地分享情感故事。

分享情感故事并非必须在特定时刻进行，但通常的流程是在传达信息并找到契合点之后。这不是一个必须严格遵守的规则，你可以在感觉安全的任何时间分享，但标准的互动流程往往将其安排在这个阶段。

故事时间中最常用的方式就是分享情感故事。基本上，每一个对我感兴趣的女生，在互动中的某些时刻，都会问到类似的问题。

以下是女生在聊天时可能提出的、标志进入情感故事分享契机的典型问题：
*   **你交往过几个女朋友？**
*   **你是怎么和前任分手的？**
*   **你有多久没谈恋爱了？**

这些问题都是非常明显的、能够自然切入情感故事的契机。只要你们的关系朝着情侣方向发展，对方几乎必然会问到这些问题。它们可能现在问，也可能之后问，但迟早会出现。

因此，当出现上述类似话题时，你就需要意识到，这是一个可以展开长篇分享的绝佳机会。

情感故事的分享通常以“小时”为单位计算，而非简单的几句话。如果对方对你真正感兴趣，你们可以就彼此的情感故事聊上整整一个晚上。这不仅能提供大量交流内容，更能成为深化彼此理解和连接的强大契合点。

本节课中我们一起学习了分享情感故事的时机与关键契机。核心在于识别那些关于恋爱经历、分手原因等个人情感问题，并将其视为深入分享的入口，从而建立更深层次的情感连接。
