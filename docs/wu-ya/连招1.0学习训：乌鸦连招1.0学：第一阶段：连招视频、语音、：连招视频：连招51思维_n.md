# 连招1.0学习训练系统完整版 第五讲：心态 🧠

在本节课中，我们将要学习“连招”体系中的第五个核心部分：心态。上一讲我们探讨了思维模式，本节我们将深入分析心态的本质、其不稳定性，以及如何通过调整内在能量和认知来建立并维持积极、稳定的心态。

心态是一种状态。这种状态并不稳定，因为它没有固定的“钥匙”可以随时开启或关闭。当你进入这种状态时，你会感觉自己的表达和思维都变得非常流畅、通顺；但当你不在这种状态里时，就会感觉处处受阻。

## 思维、恐惧与能量

思维等于恐惧。思维的活跃度越高，恐惧感和不安全感就越强。这是因为思维源于“小我”，而“小我”服务于生存本能，会不断制造担忧和恐惧。

**公式：思维强度 ∝ 恐惧感**

当你进入一种特定的积极状态（早期理论称之为“情圣意识”）时，你实际上是与一种更高层次的意识进行了连接。这很大程度上是一种自我催眠，能让你的话语和观念变得鲜活。但这种状态极不稳定。

## 肢体语言与内在能量

恐惧是有“味道”的，它会通过你的肢体语言和能量场传递出去。你的内在能量决定了你的外在表现。

*   **积极能量**：对应展开式的肢体语言，如笑容、开放的姿态，会感觉地心引力变轻。
*   **消极能量**：对应收缩式的肢体语言，如皱眉、弯腰、驼背，会感觉地心引力变重。

单纯模仿积极的肢体语言（如“强颜欢笑”）是无效且消耗能量的，因为内在与外在的能量不匹配，会被识别为“皮笑肉不笑”。人们本能地会远离消极的人，因为消极能量意味着更低的生存可能性，而吸引力本质上源于价值，即“能生活得更好”的潜力。

## 选择你的现实：平行宇宙的视角

生活很大程度上不存在绝对的真相，它是被你描述和认知出来的。你对事情的看法决定了它对你的影响。

我们可以用一个“平行宇宙”的假设来理解：在每一瞬间，都存在无数个已经发生的、包含所有可能性的宇宙。这些宇宙就像一帧帧静止的画面。

**核心比喻**：就像快速翻动书角上的连环画会产生动画效果一样，是我们的“意识”这只手在选择翻到哪一页，即选择去经历哪一个宇宙。

你不是在创造现实，而是在选择经历哪一个已经存在的可能性。你的“想法”和“行动”是同步的。如果你持续想着换工作，很可能不久后就会真的换工作。因此，没有什么是不可逆转的，关键在于你的意识选择去往哪里。

## 从感受到思想的转变路径

当陷入痛苦或消极状态时，试图从“思想”上硬生生想通，往往效率低下且过程漫长。更有效的方法是先从“感受”层面入手。

以下是具体步骤：

1.  **接纳感受，停止抵触**：痛苦本身并非源于事件，而是源于你对事件的抵触。当你全然地接纳某种感受（即使是疼痛或负面情绪），不再试图逃避或解决它时，抵触消失，痛苦感就会大幅减弱甚至转化。
2.  **感受引导思想**：当你完全接纳某种情绪的瞬间，解决该问题的灵感和积极思想自然会涌现。这比纯粹的逻辑思考要直接和有效得多。

这源于禅宗的思想：不与生活抵触。你可以通过一句积极的自我陈述来引导心态，例如：“我所经历的一切，都是为了让我变得更好。”

## 总结

本节课我们一起学习了心态的奥秘。我们了解到心态是一种不稳定的状态，与思维带来的恐惧感相关。内在能量决定外在表现，积极能量更能产生吸引力。更重要的是，我们探讨了“现实是选择的结果”这一视角，并掌握了通过“先接纳感受，再转变思想”的高效路径来调整心态。记住，**苦本身并不苦，是你与苦的抵触让它变得难以忍受**。选择接纳，就能选择进入更积极的“宇宙”。
