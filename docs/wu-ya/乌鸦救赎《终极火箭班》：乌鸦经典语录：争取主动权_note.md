# 乌鸦救赎《终极火箭班》第1课：争取主动权 🎯

在本节课中，我们将学习在恋爱吸引阶段及后续关系中，如何通过避免暴露需求感和进行服从性测试来争取主动权，从而避免关系演变为不平等的控制游戏。

## 吸引阶段的核心原则

上一段我们提到了吸引阶段的目标，本节中我们来看看具体需要遵守的两个核心原则。

第一，在吸引阶段，你必须保证自己的需求感不暴露。

第二，你需要能够控制局面，以获得最多的控制权。要尽可能避免让对方成为需要你服从的“姑奶奶”。

## 理解关系的本质：控制游戏

在探讨具体方法前，我们需要先理解一个核心概念：公平的恋爱关系并不存在。世间任何事情都没有绝对的公平。

在我看来，真正的爱情只存在于彼此相爱但尚未确立关系的时刻。一旦突破关系进入恋爱，爱情就会转变为一个**控制游戏**。

这个游戏的规则是：**谁获得更多的主控权，谁就占据优势；而控制权较少、更喜欢对方的一方，则会处于被动**。关系的味道因此改变，不再纯粹是喜欢或爱，而是演变为一场控制权的博弈。

## 争取主动权的关键方法：服从性测试

尽管关系会演变为控制游戏，但我们依然需要进入并维系长期关系。因此，你需要掌握争取主动权的方法。

关键方法是：经常给对方做一些服从性测试，让对方为你做一些事情。

以下是进行服从性测试的要点：
*   从暧昧阶段就可以开始。
*   即使进入关系中期，也不要客气或停止。
*   你需要明确地告诉对方你的要求。

## 课程总结

本节课我们一起学习了乌鸦救赎关于恋爱主动权的重要观点。我们认识到，确立关系后的互动在某种程度上是一种控制权的博弈。为了在其中占据有利位置，我们需要在吸引阶段隐藏需求感，并在整个关系中持续、明确地进行服从性测试，以巩固自己的主动权。
