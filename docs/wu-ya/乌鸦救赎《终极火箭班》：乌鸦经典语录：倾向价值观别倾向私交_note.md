# 乌鸦救赎《终极火箭班》：乌鸦经典语录：倾向价值观别倾向私交_note

在本节课中，我们将要学习一个核心的人际关系与决策原则：**倾向价值观，而非倾向私交**。我们将深入探讨“重感情”与“正直”之间的区别，并理解为何基于价值观的决策比基于私人交情的决策更为稳固和明智。

## 核心概念解析

上一节我们介绍了人际关系中的一些基本误区，本节中我们来看看“正直”与“感情用事”的本质区别。

乌鸦指出：“仗义是表面正直，后面是会歪掉的。” 这句话揭示了仅凭一时义气行事的内在风险。表面上的仗义之举，可能缺乏坚实的价值基础，最终容易偏离正轨。

## 关键原则：重感情不等于正直

许多人存在一个认知误区，即将“重感情”等同于“为人正直”。这是一个需要纠正的预设价值观。

**核心公式：重感情 ≠ 正直**

原因在于，重感情更多是**倾向私交**，即因为与某人的私人关系而做出选择或判断。而正直则根植于**倾向价值观**，即依据一套普适的、正确的原则和是非观来行事。

以下是关于这一原则的详细阐述：

*   **误区**：重感情的人都以为自己是正直的人。
*   **纠正**：必须从预设的价值观中删除“重感情等于正直”这条关联。
*   **原因**：正如课程核心所强调的，我们应该“倾向价值观而不倾向（私人）思维（交情）”。基于私人关系的决策往往带有偏颇和不确定性，而基于共同价值观的决策则更加一致和可靠。

## 总结

本节课中我们一起学习了乌鸦经典语录的精髓：**倾向价值观，别倾向私交**。我们明确了“重感情”并不自动意味着“正直”，因为前者可能源于不稳定的私人关系，而后者必须建立在稳固的、正确的价值观之上。做出判断和选择时，应以价值观为准绳，而非为私交所左右，这样才能确保行为的长期正确性和一致性。
