在本节课中，我们将学习“进挪窗口”这一核心概念。理解窗口的开启与关闭时机，是避免被拒绝、顺利升级关系的关键。

上一节我们介绍了关系推进的基本概念，本节中我们来看看如何识别和把握关键的“窗口期”。

## 核心概念：进挪窗口

“进挪窗口”指的是女性在互动中，愿意接受关系升级的短暂信号或时机。这个窗口的开关与对方的**情绪**直接相关。

## 窗口开启的信号

当女性希望与你建立更亲密的关系时，她会给出暗示。识别这些信号至关重要。

以下是窗口可能开启的具体表现：

*   **深情凝视**：在互动中，尤其是接近接吻时，她会长时间地、深情地望着你。
*   **沉默期待**：她保持沉默，只是看着你，仿佛在期待你采取行动。

**公式：窗口开启 = 特定信号（如深情凝视）+ 高情绪状态**

## 错过窗口的后果

如果你自身的社交直觉不够敏锐，未能发现窗口已经开启，就会错过机会。

**代码逻辑描述：**
```python
if 窗口开启 == True and 采取行动 == False:
    后果 = “错过升级时机”
elif 窗口关闭 == True and 采取行动 == True:
    后果 = “吃到IOD（无兴趣指标）”
```

对方的情绪会回落，窗口随之关闭。此时若你再尝试升级关系，必然会遭到拒绝（吃到IOD）。

## 关键要点总结

本节课中我们一起学习了“进挪窗口”的原理与应用。

1.  **窗口与情绪挂钩**：窗口的开启与关闭，紧随对方情绪的起伏。
2.  **识别信号是关键**：需敏锐捕捉如深情凝视等非语言暗示。
3.  **果断行动**：一旦确认窗口开启，应果断进行关系进挪，犹豫就会错过。
4.  **核心原则**：不在对方窗口关闭时强行升级关系。

时刻注意窗口的关闭与开启，是关系进阶中必须掌握的技能。
