# 乌鸦救赎《终极火箭班》：乌鸦经典语录：搞定闺蜜_note

在本节课中，我们将学习如何处理伴侣与闺蜜之间关系过密的问题。核心在于通过强势沟通，明确关系边界，让伴侣认清谁才是她生活的“主角”。

---

上一节我们介绍了关系中的潜在挑战，本节中我们来看看一个具体案例：当伴侣过于听从闺蜜意见时，该如何应对。

我曾遇到一个情况，伴侣的闺蜜不愿听从我的建议。我选择与她直接、坦诚地沟通。但在矛盾激化到当前程度之前，我已经与她谈过。我的核心观点是：**闺蜜是闺蜜，老公是老公，必须分清孰轻孰重**。

你需要运用策略去说服一个人斩断某些关系。我的用词通常非常严厉，目的是让对方产生一种感觉：**不听我的建议是愚蠢的**。我能塑造出这种氛围。本质上，是要让对方明白**谁才是她应该首要考虑的人**。

我对她说了大意如下的话：“你是跟我生活，还是跟你闺蜜生活？你是要和我过日子，还是和她过？” 经过这样一场谈话后，她的行为改善了很多，不再事事都与闺蜜深入讨论。我强调：“有些事情只属于我们两人，不应全部透露给外人。”

以下是处理此问题的关键步骤：

如果你觉得自己无法掌控这段恋情，恋爱中的所有问题都需要找那个闺蜜来解决，那不如我们三个人一起生活。或者，干脆让她来直接与我谈。你夹在中间做什么？我究竟是在与你谈恋爱，还是在与她谈？

“不如我们三个一起过算了，多好、多舒坦。” 大致经过我这样一番讽刺和敲打之后，这个问题就基本解决了。
