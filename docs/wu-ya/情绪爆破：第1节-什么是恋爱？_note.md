# 情绪爆破：第1节-什么是恋爱？_note

在本节课中，我们将学习恋爱的核心理论框架。理解这个框架是学习后续所有技巧和方法的基础。我们将探讨恋爱的本质、吸引力的来源，以及关系发展的不同阶段。

## 概述：明确学习目标

在深入学习“情绪爆破”技巧之前，我们必须先搞清楚恋爱究竟是什么，以及我们该做些什么。这就像设定一个目标。在没有明确目标之前，盲目学习各种方法和技巧是无效的。

因为学习方向可能出现偏差。所以，本节课我们要确立一个明确的信念和目标。这是我们必须做的第一步。

## 第一部分：恋爱的本质是吸引

那么，第一个问题就是：恋爱究竟是什么？

恋爱本身是一个非常抽象的概念。简单诠释的话，它就是异性之间的正常吸引。一个男人喜欢一个女人，是因为这个女人吸引了他。同理，这个女人能否喜欢他，也取决于他能否吸引到她。

这个道理很简单。但其中有一个至关重要的点，这个词就是**吸引**。我们所有人每天都在谈论这个话题。

那么问题来了：吸引又源自于什么？吸引从何而来？既然一段恋情、一段恋爱关系就是彼此的互相吸引，那么关键就在于“吸引”这个词。那么，吸引的本质又是什么？它从何而来？

我直接告诉你们答案：**情绪活动**。

**情绪活动产生吸引**。这是吸引的本质。这四个字，我希望你们在今后一生中都牢牢铭记。它们太重要了。这就是吸引最本质的东西。

## 第二部分：理解“情绪活动产生吸引”

有些朋友可能不太理解这个意思。我们举一个例子：为什么是情绪活动产生了吸引，而不是其他东西？

1.  **例子一：金钱**
    假设出现一个特别爱钱的女生。钱能带给她大量的吸引。这时，一个老板丢了一百万在她面前。你们觉得这个女生当时会产生什么样的心理活动？这种心理活动就叫做**情绪波动**。
    这种活动造就了什么情况呢？当我把这一百万丢在她面前，说“你拿了这一百万，就跟我处对象”，你们想想，这个造就了她大量情绪波动的“钱”，能不能让她跟我在一起？答案是：可以。
    同时，我们再举一个延伸例子：这时又出现另一个更有钱的老板，丢了一千万在她面前。她会选择跟哪一个人谈恋爱？答案不言而喻，她肯定会选择丢一千万给她的那个人。因为一千万给她的情绪波动比一百万要大。
    那么，吸引这个女生的根本原因是因为钱，还是因为这个老板？肯定是因为钱。因为她爱的是钱，能让她产生情绪波动的是钱。

通过这个例子，我们可以总结出一句话：**一切吸引的起源是情绪波动**。

这是一个信念。这个信念对于我们的“情绪爆破”课程极为重要。它会伴随你一生。我希望你们能做到这种程度：以后但凡出现任何吸引不足的问题，都可以用“情绪波动”这个根本原因去分析和解决。没有比这更根本的东西了。

## 第三部分：情绪波动的多种来源

刚才举了关于金钱能产生吸引的例子。那么，除了金钱，我们还能不能举出其他能吸引女生的论点呢？可以。

以下是几个例子：

*   **幽默风趣**：如果一个女孩子特别喜欢幽默、讲段子、能逗她开心的男生。那么，当一个幽默风趣的男生出现在她面前时，女生在接收到他幽默风趣的信号后，能不能产生吸引和大量情绪波动？答案是：能。

所以，回顾我们曾经经历过的所有恋爱，试着回想一下：你在跟一个女生第一眼相遇，甚至刚接触聊了几句话的时候，是什么东西撩起了你内心的情绪波动？这样你就明白了，为什么一切吸引的起源源自于情绪波动。

因此，我们以后所使用的所有方法、技巧，都建立在一个前提条件下：**制造情绪波动**。

以下是几个具体技巧及其根本目的：

*   **打压**：根本目的是为了制造情绪波动。
*   **自我引导**：根本目的是为了制造情绪波动。
*   **展示面/秀肌肉/展示生活圈价值**：根本目的还是制造情绪波动。

因为只有她心里产生了情绪波动这个活动之后，我们才能产生吸引。所以，情绪波动是一切吸引的起源。我们所做的一切，其根本唯一目的就是：**通过情绪波动产生吸引**。

## 第四部分：情绪波动的局限性

下一个问题来了。首先，我们必须把“情绪波动产生吸引”这个观念变成一种信念，刻在骨髓里。

那么，一切吸引力不足的原因就可以解释为：你没有给她带来足够的情绪波动。

那么第二个问题是：既然一切吸引的起源源自于情绪波动，那么情绪波动能让我们跟女生走到哪一步？能走到一生吗？能走到结婚吗？

思考一下：情绪波动是不是我们能够跟一个女人在一起时，永远利用、并走完一生一世的东西？答案是不行。

那么，情绪波动能让我们跟女生走到恋爱总体环节中的哪一个阶段？答案是：走到**热恋期退却之前**。在热恋期结束、热恋本事消失之前，情绪波动一直伴随着我们。

如何解释呢？结合自己的恋爱经历想一想：我们在刚认识一个女生时，每天特别想她，渴望拥抱她，渴望在一起。女生在热恋期也有同样想法。那时，我们是不是有大量的情绪波动产生？所以有吸引。那时的吸引源自于情绪波动。

那么热恋期结束后，为什么没有情绪波动产生了？再结合实际想一想：当你跟一个女生谈了一两年、两三年，甚至更久，你有没有一种感觉：牵着她的手好像牵着自己的左右手，甚至不想发生关系，但又离不开她？这已经变成一种习惯。但这时，你内心还有那种渴望、热情和激情吗？没有了。

所以，情绪波动只能帮我们走到热恋期结束之前。那么热恋期结束之后，我们又靠什么东西去吸引女生？这就是一个问题了。

## 第五部分：长期吸引的源泉——需求

情绪波动产生吸引，一切吸引的起源源自于情绪波动。但当热恋期退却后，情绪波动已经不能帮助我们持续吸引这个女人了。因为那时内心已经很平淡，没有那么多激情了。

那么，这时还有吸引吗？既然情绪波动没了，是不是就没有吸引了呢？我告诉你们：有吸引。吸引是永恒不变的定律。它永远摆在那里，永远不会变。

为什么？因为现实生活中有这样的例子：两个人认识后结合在一起，一生一世永远在一起，直到死都深深相爱。这个定律就冲击了我们刚才说的“情绪波动产生吸引”。如果情绪波动消失，照道理就该分手了，但事实并非如此。

所以我才说，**起源**源自于情绪波动。那么，是什么在持续地吸引这个女人呢？已经不是情绪波动了。后面的那个东西叫做：**需求**。

当情绪波动不再能提升源源不断的吸引力时，另一个东西接手了，这个东西叫做需求。**情绪波动是一切吸引的起源，需求是一切吸引持续发展的动力**。

吸引分为两块：
1.  **短期关系**：由情绪波动驱动。
2.  **长期关系**：由需求匹配驱动。

一段短期关系加上一段长期关系，中间有个加号，就等于整个恋爱的步骤。

## 第六部分：理解“需求”与关系成败

那么，需求是什么？我举一个例子：假设我需要我的另一半是一个有钱、温暖、体贴、关心的人，但有时又会给我一点家暴（这是我的特殊需求，我喜欢被虐打）。当这些需求能够百分之百匹配到我这个人时，她所拥有的一切都是我想要的。

试想，当你生命中出现这么一个人：你要她多漂亮她就有多漂亮，你要她多完美她就多完美，你要她有什么事业她就有什么事业，你要她早上起来给你做饭她就给你做饭……当这种需求，你不用说她就能完美做到时，你愿意跟这样的人在一起一生一世吗？答案很明显。

前提条件是**需求百分之百匹配**。当需求达到百分之百时，如果你说不喜欢乖巧的，那她就是一个不听话的人，但她依然完美符合你心目中所有需要的东西。这叫百分之百的需求匹配。

那么，为什么我们在恋爱关系中经常出现分手？下图完美解释了在一起和分手的过程：

```
在一起：因情绪波动产生吸引 -> 开始关系
分手：情绪波动消失 + 需求不匹配 -> 关系结束
```

我解释一下：我们因为彼此产生大量情绪波动而在一起。但我们分手，是因为在相处过程中，情绪波动消失（热恋期退却），吸引力减弱。此时需要需求来持续吸引，但需求不匹配。

例如：这个女生满足了我百分之百的需求，我爱她爱到死。但我只满足了这个女生百分之十的需求（比如她需要幽默风趣、温柔、会赚钱的男人，而我都不是）。这些需求不匹配她时，她为什么要跟我在一起？分手后，我会不断追逐她（我的百分之百），但她会不断拒绝我，因为她已经看清我不是她想要的男人。

**挽回的本质**就是这么一回事：因为女神满足了我大量需求，但她已经看不上我了。在了解我之后，她知道我只有百分之十的需求能匹配她（因为情绪波动已帮不上忙）。

## 第七部分：短期与长期的抉择：一个关键问题

这里有一个关键的理论问题：假设出现两个男人（A男和B男）和一个女生。
*   A男满足了这个女生**百分之百的需求**。
*   B男给了这个女生**大量的情绪波动**。

请问，在刚认识、彼此不了解的情况下，这个女生会选择跟谁在一起交往？

答案是：她**一定会跟B男在一起**。

为什么？回到我们今天的中心思想：因为**情绪波动才产生吸引，一切吸引的起源源自于情绪波动**，而不是需求。在刚认识的时候，我需要的东西是情绪波动。我不知道你是什么样的人，但你能带给我大量情绪波动，我就能跟你在一起。因为情绪波动产生了吸引。

需求是要在漫长的相处关系中去慢慢发现的。你需要时间去了解一个人的根本本质。

## 总结与核心信念

今天的课就到这里。最后我们再来强调一点：在这节课程中，你必须搞清楚，并把“情绪波动产生短期关系”这个理念植入骨髓。这尤为重要。

**核心公式：**
```
恋爱 = 短期关系 (情绪波动) + 长期关系 (需求匹配)
```

**核心信念：**
1.  **一切吸引的起源源自于情绪波动。**
2.  **经过漫长相处后，接手并维持吸引的结果是需求匹配。**

当植入这个信念后，我们的工作就变得简单了。吸引女生的工作很简单：我们暂时抛开需求部分（需求是你的人格特质和魅力，非短时间能养成）。我们只需要把**情绪波动**的技巧、方法和套路弄好。

随着你在短期关系中的经验累积和人格重塑，只要你把短期关系做好了，你的需求和特质会产生变化，它会反过来影响你的长期吸引力源泉。

所以，我们的注意力应全部集中在**短期关系（情绪波动）** 当中。我们唯一的目的就是制造情绪波动。能制造情绪波动的技巧实在太多了。
