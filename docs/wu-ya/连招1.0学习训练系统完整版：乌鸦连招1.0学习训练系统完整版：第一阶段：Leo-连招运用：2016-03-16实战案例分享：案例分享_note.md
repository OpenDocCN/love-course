# 连招1.0学习训练系统完整版：第一阶段：Leo-连招运用 🎯

## 课程概述

在本节课中，我们将通过一个2016年3月16日的实战聊天案例，完整解析“连招1.0”系统的运用过程。我们将重点关注如何从破冰开始，逐步运用自我引导、识别兴趣指标等核心技巧，最终实现关系的快速升级。案例中的对话将作为我们分析的蓝本。

## 第一阶段：破冰与开场

上一节我们概述了课程内容，本节中我们来看看聊天的第一步：破冰与开场。开场的目标是建立初步联系，氛围轻松即可。

案例中，导师先发了一个“？”。对方（昵称“超人婆婆”）回应“你好超人婆婆”。导师随即回应：“知道了，你的昵称真是最了。”这是一个简单的破冰，核心在于自然、不刻意。

**核心操作**：开场无需复杂话术，自然打招呼并轻微评价对方（如昵称）即可破冰。

## 第二阶段：初期互动与话题带领

在破冰之后，聊天进入初期互动阶段。此时对方可能回应冷淡，需要由男方主动带领话题。

导师发现对方朋友圈内容很少，便说：“而且你的朋友圈是我什么也看不到啊。”对方回应：“我没什么朋友圈啊。”之后，导师说：“你真是与众不同的姑娘啊。”对方冷淡回应：“是吗，好像还好啊。”

当对方回应冷淡时，导师没有纠缠，而是主动开启新话题：“我们同龄对吗？”以此引导出对方的年龄信息。对方回答“88年”，导师说“我属兔子”。这时，对方问：“你87年的对吧？”

**核心概念**：**黄金IOI（兴趣指标）**。当对方以“你”开头的疑问句询问你的个人信息时，即为黄金IOI，表明对方对你产生了初步兴趣。公式表示为：`对方对话 = “你” + [关于你的问题] + “?”`

## 第三阶段：识别与回应兴趣指标

识别黄金IOI后，需要合理回应，既不过度热情，也不冷漠无视，保持对话平衡。

对方提出黄金IOI“你87年的对吧？”之后，导师简单确认：“是啊。”紧接着，对方抛出第二个黄金IOI：“什么星座？”导师回答：“双鱼。”并反问：“你信星座吗？”对方巧妙回避：“没什么信不信的，就是随便看看。”

此时，导师开始运用**自我引导**技巧，主动展示自己：“集十二星座优点于一身。”这句话幽默且展示了高价值。对方果然产生兴趣，追问：“有什么优点啊？”（又一个黄金IOI）。导师没有直接列举，而是幽默回应：“太多了。”

**核心技巧**：**自我引导**。在对话中自然引入关于自身特质、生活或价值观的内容，以展示吸引力。代码式描述为：`对话内容.insert(关于自我的积极描述)`

以下是此阶段的关键点列表：
*   对话字数需保持对等，避免一方长篇大论。
*   对方释放黄金IOI时，是展示价值或推进关系的好时机。
*   自我引导要自然，源于真实生活。

## 第四阶段：深入互动与关系升温

随着自我引导的进行，对方兴趣提升，对话内容增多，关系开始升温。

对方追问“最重要的优点，我听听”，导师回答“太善良”。此时，对方也开始自我引导，提到“我在加班，眼睛都花了”。导师顺势接话：“我看你头像眼睛也花。”并自然询问：“你做什么工作，这么晚下班？”

对方发来一张办公桌的电脑图片。这是一个重要的**IOI**，因为发送图片意味着对方愿意付出更多精力让你了解她的状态。接着对方反问：“难道你也没下班？”（IOI）。导师肯定并夸奖：“是的…聪明。”

当对方进一步关心：“你怎么还没下班？”并抱怨“我的工作是全世界最烂的工作”时，导师没有直接回应抱怨，而是以昵称“婆婆”称呼对方，并说明时间已晚（晚上11点50），对话暂告段落。

**核心概念**：**IOI（兴趣指标）** 的多样化形式。包括：主动发送图片、打听你的生活状态、关心你的情况、解释自己的行为（如下文）等。

## 第五阶段：联系感建立与自我引导交换

第二天，对方主动开启对话，解释“昨天晚上睡着了”。这又是一个IOI（解释行为）。对话进入更深入的阶段，双方开始交换更多个人信息和生活经历，建立联系感。

导师通过朋友圈了解到对方可能喜欢健身，便说：“你喜欢健身？…以前大学是每天健身都八块腹肌。”对方回应：“我是运动员出身”，并分享了运动员时期的照片和成就（自我引导/DHV）。导师给予积极回应（奖励IOI），引导对方更多分享。

对方讲述了作为运动员的经历、伤病、求学和工作变迁。导师也相应地进行了自我引导，分享了自己从国企到公务员再到辞职创业的经历。并找到共同点：“你一个人在H市？…我们是一类人…离开家不服输想追求自己想要的生活。”这建立了**共谋**感。

**核心技巧**：**联系感与共谋**。通过分享相似经历、价值观或状态，制造“我们是一类人”的感觉，拉近心理距离。

## 第六阶段：价值观探讨与情感共鸣

在建立联系感后，对话可以深入到价值观、理想和情感层面，为关系升级做铺垫。

双方聊到理想职业。导师开玩笑说“家庭主妇”，对方配合。之后对方认真谈及理想是体育记者。导师表示“记者会很累”，对方说“记者可以全世界到处走”。导师评价：“你是个喜欢自由冒险的女人。”

接着，导师分享：“我的终极理想是周游世界。”对方回应：“你走的时候可以把我装箱一起带走吗？”对话进入**人格筛选**阶段。导师幽默回应：“安检不能带宠物…你会做饭吗？”双方围绕生活技能、做饭等话题轻松互动。

随后，话题自然过渡到感情观。对方表达了寻找“对的人”的困难和对爱情的迷茫。导师问：“你还相信爱情吗？”对方肯定，并倾诉了更多内心感受，甚至主动分享了自己写的一大段关于理想爱情和伴侣的内心独白（强烈的IOI和窗口）。

**核心概念**：**人格筛选**。通过谈论理想、生活态度、价值观等，无形中筛选出与自己匹配的伴侣，同时展示自身特质。

## 第七阶段：关系升级与确立

当对方释放出极强的兴趣和情感窗口时，应果断抓住机会，进行关系升级。

在听完对方深刻的内心独白后，导师深受触动，直接表达了感受：“你感染到了我的情绪…想恋爱了。”对方确认：“真的吗？”导师没有犹豫，将对方的独白内容转化为对她的承诺，描述了一幅共同未来的温暖画面。

接着，导师直接提出：“我想知道我们在一起的感觉。”对方回应：“我也想，否则我不会给你发信息的。”导师果断**劫杀**：“从这一刻开始…做我女朋友。”对方经过短暂确认（“你想好了吗？”），最终同意（“我愿意”）。尽管对方说“我连你叫什么都不知道”，但关系已经确立。

**核心操作**：**劫杀**。在情感共鸣达到高点、窗口明确时，大胆、真诚地提出确立关系的请求。关键在于时机把握和前期铺垫的扎实。

## 课程总结

本节课中，我们一起学习了一个完整的聊天案例，它展示了“连招1.0”系统从破冰到确立关系的全过程。我们重点分析了：
1.  **破冰**：自然开场，无需压力。
2.  **话题带领**：在对方冷淡时主动引导。
3.  **兴趣指标识别**：准确识别黄金IOI及各种IOI，并合理回应。
4.  **自我引导**：自然展示自身高价值和生活方式。
5.  **联系感建立**：通过分享经历和寻找共同点拉近距离。
6.  **深入共鸣**：探讨价值观和情感，建立深层连接。
7.  **关系升级**：抓住窗口，果断进行关系劫杀。

整个流程的核心在于**自然、真诚**地互动，所有技巧都建立在真实的生活和情感基础上。通过学习和内化这些步骤，你可以更有结构地推进关系，避免迷茫和失误。
