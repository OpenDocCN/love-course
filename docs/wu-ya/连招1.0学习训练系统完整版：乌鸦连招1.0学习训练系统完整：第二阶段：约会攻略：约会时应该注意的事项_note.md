## 四、情绪带领模型图解 📊

让我们用模型来理解情绪互动。

想象两个箭头：
*   **左边箭头**：代表你的情绪（正向能量）。
*   **右边箭头**：代表女生的情绪（可能是负向的紧张能量）。

**初始状态**：箭头相对。如果女生的紧张情绪（箭头）比你的正向情绪更强，你就容易被她的负能量带领。
