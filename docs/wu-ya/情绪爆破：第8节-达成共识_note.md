# 情绪爆破：第8节 - 达成共识 🤝

在本节课中，我们将要学习聊天组成的最后一个核心环节：如何达成共识。这是任何完整对话都必须经历的步骤，它为交流画上圆满的句号。

## 概述

达成共识是聊天组成的第五步，也是最终步骤。无论与谁交流，一个完整的对话都需要以此收尾。缺少这一步，对话便不完整，可能导致不愉快的结果。

## 共识的普遍性

上一节我们介绍了互动技巧，本节中我们来看看如何收尾。达成共识适用于所有聊天场景。

*   与上级沟通，需要达成意识的统一。
*   与异性聊天，希望关系升级，同样需要达成共识。

只有当前面四个步骤（打招呼、收集信息、传达信息、互动）都完成得很好，我们才能进入这最终画上句号的步骤。

## 达成共识的技巧

那么，如何有效地达成共识呢？有很多技巧可以使用。这里我们用一个最经典的连招技巧——“七天情侣”技巧来举例说明。

**“七天情侣”技巧**的核心是提出一个低压力、可试错的提议。其逻辑类似于：
`昂贵产品 = 分期付款 + 无理由退款承诺`
这个技巧为对方提供了充分的台阶，减少了“必须立刻做决定”的压迫感。

当你完成前几步，但吸引力积累尚不充足、底气不足时，可以使用此类技巧来尝试达成关系进阶的共识。如果对方同意，那么这次聊天便取得了完整的成功。

## 聊天组成的核心框架

纵观整个聊天组成，其顺序绝对不能错。我所讲的框架主要针对异性社交聊天，但其中多个步骤具有通用性。

以下是适用于几乎所有交流场景的核心步骤：

1.  **打招呼**：对话的起始。
2.  **信息交换**：在“收集信息”与“传达信息”中至少完成一项。
3.  **达成共识**：对话的收尾。

需要注意的是，“互动”这一步并非所有场景都必须。例如，在上下级的任务安排沟通中，可能只需要“传达信息”和“达成共识”即可。

## 严格遵守框架的重要性

在平等态度的沟通中（尤其是社交吸引），聊天组成的五个部分必须严格按照以下顺序执行，不容有错：

1.  打招呼
2.  收集信息
3.  传达信息
4.  互动
5.  达成共识

唯一可以灵活调整的是：在“收集信息”时，可以稍微“传达”一些自己的信息。除此之外，整个结构应原样体现在你的聊天框架中。因为这就是聊天的本质与核心组成，缺少任何一环，对话质量都会大打折扣。
