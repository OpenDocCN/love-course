# 乌鸦救赎《终极火箭班》第1课：认知 🧠

在本节课中，我们将学习乌鸦关于“认知”的核心观点，重点探讨如何看待感情与“吃亏”行为，并建立更理性的处事思维。

## 感情与理性的权衡

上一节我们介绍了课程主题，本节中我们来看看乌鸦对感情的看法。感情往往使人陷入痛苦，因此不应过于看重感情，这没有意义。处事应从大局出发，舍弃个人小情绪。重视感情的人常自认为是正直的人，但这种想法是愚昧、愚蠢且不明智的。这类人也不值得交往。

以下是这类人可能带来的问题：
*   他们最初可能对你很好。
*   但后续可能会损害你的利益。

## 关于“吃亏”的认知误区

在理解了感情的角色后，我们进一步分析一种常见的认知误区：主动牺牲利益。那些习惯于阉割自身需求、甘吃哑巴亏的人，也总认为自己是在坚守正直。他们无论事情大小都选择吃亏，在任何该吃亏的场合都冲在前面，并以此自比雷锋。

乌鸦对此的评价是：这是愚蠢的行为。“吃亏是福”这句话本身正确，但其道理并非如此理解。所有通过自我阉割来吃傻亏并自诩正直的人，其认知都是错误的。

## 核心原则总结

本节课中我们一起学习了乌鸦认知论中的两个关键点。第一，过度重视感情是愚昧的，应以理性大局观为主导。第二，无原则、不自知的“吃亏”并非美德，而是认知不清的愚蠢表现。真正的智慧在于明辨何时应该坚持，何时可以退让，这其中的分寸才是需要掌握的真正道理。
