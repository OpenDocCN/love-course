# 乌鸦救赎海峰情绪爆破：19、安全时间-奖励三_note

在本节课中，我们将深入探讨“奖励”这一情绪刺激工具的正确使用时机与核心原理。上一节我们介绍了“认同”与“不认同”的不同效果，本节中我们来看看如何通过“奖励”来制造强烈的情绪波动。

## 课程概述

奖励是一种强大的情绪刺激工具，但其使用并非没有条件。错误地滥用奖励，例如频繁而无缘由地夸赞对方，不仅无效，反而可能适得其反。奖励的核心在于：**必须在对方先做出某个特定行为后，才能给予夸张的、超出行为本身的正面反馈**。

## 奖励的前提条件

奖励的使用有一个明确的前提条件。这个前提条件是：**对方必须先主动做出某一件事情**。

这件事情不一定直接针对你，也可能是针对其他事物或情境。关键在于，这个行为提供了一个可以被你“捕捉”并放大的机会。

## 核心原理与示例

为了让你更清晰地理解，我们来看一个具体的例子。

假设我正在讲课，旁边有一位助手。她主动帮我把黑板擦干净了。此时，我如何给予奖励呢？

以下是奖励的示范过程：

1.  捕捉行为：“哇，我没想到你居然是这么勤快的人。”
2.  关联情境：“你看我在这边忙，你还主动帮我擦黑板。”
3.  升级评价：“你是这么善良、这么优秀的女生吗？”
4.  表达感受：“我真的感觉你特别暖心，特别温暖。”
5.  夸张承诺：“就因为你这么一个行为，我就爱了。我今天晚上要请你好好出去玩一玩，我们要去三里屯喝一杯，我连孩子的名字都想好了。”

这个回应听起来可能很夸张甚至像个笑话，但这恰恰是奖励原理的体现。

## 奖励的核心公式

奖励的核心原理可以总结为以下公式：

**奖励 = 对方的行为 × 十倍以上的夸张认同**

当对方做了一件可以被你用来表达认同的事情时，你需要给出远超该行为常规价值的正面反馈。有多夸张都不为过。这种强烈的反差，才构成了有效的情绪波动和情绪刺激。

在情绪刺激中，我们需要做到态度鲜明：认同时就彻底、夸张地认同；不认同时则要坚决地表达立场。

## 课程总结

本节课我们一起学习了“奖励”工具的关键要点。我们明确了奖励必须建立在**对方先有主动行为**的前提之上，其核心在于给予**十倍以上的夸张认同**来制造情绪波动。请牢记，无条件的赞美不是奖励，基于具体行为的夸张反馈才是。
