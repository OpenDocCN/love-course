# 乌鸦救赎《终极火箭班》课程笔记 02：净信 - 纯粹的信念 🚀

在本节课中，我们将学习“净信”这一核心概念。这是一种纯粹、毫无杂念的信念状态，被认为是实现目标的关键心理基础。我们将通过具体事例和理论依据来理解其内涵与应用。

上一节我们介绍了课程背景，本节中我们来看看“净信”的具体含义。

泰勒提出了一个经典论点：当你足够自信、足够坚定地去做某件事时，你必然能够做成。这其中不能掺杂任何一丝一毫的怀疑。

为了说明这一点，可以分享一个个人经历。在我十八岁、月收入仅三千元时，我与我的师父（即心态老师）进行了一次对话。我告诉他某件事可以去做，因为这件事旨在突破个人的底层意识和集体意识。我表示愿意去做。

我做出这个决定的依据，正是当时看到的泰勒的一个视频。泰勒在视频中阐述，当你以绝对的自信和坚定去行动时，你一定能成功。视频中，泰勒说完这句话后，直接去亲吻了一位女孩，而那位女孩接受了。

这个场景让我当时愣住了。

后来，我找到了支持这一观点的其他理论依据。例如，佛教中提到的“净信”，意指干净、纯粹的信念。

以下是“净信”状态的具体描述：

当你对某事的信念坚定到一定程度，这种执念应如同你下楼去买一卷卫生纸时那般自然和确定。你毫不怀疑自己能买到它。

将这种信念应用于事业或目标上：你必须相信到自己“绝对可以做到”的程度。拥有这种纯粹、无杂念的信念，你必然会成功。

因为人是受意念支配的动物。一旦你想到失败，失败就可能降临。反之，如果你持续不断地思考成功，没有时间去想消极的事情，那么成功也会随之而来。

本节课中我们一起学习了“净信”的概念。它强调的是一种纯粹、坚定、毫无自我怀疑的信念状态，这种状态被视作驱动行动和达成目标的核心心理力量。通过将信念强化到日常小事般的确定程度，并排除所有杂念，个人便能更有效地导向成功。
