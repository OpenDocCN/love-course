## 为什么要进行人格筛选

在开始学习具体连招之前，一个根本性的问题是：为什么要进行人格筛选？人格筛选是整个互动流程的起点，其目的在于快速识别并吸引那些与我们价值观、生活状态相匹配的潜在对象。它并非简单的评判，而是一种高效的初步连接策略。

## 人格筛选第一招：暗度

上一节我们提出了人格筛选的必要性，本节中我们来看看人格筛选的具体起始动作。

人格筛选的第一招，其核心概念是 **“暗度”**。

“暗度”并非直接展示或询问，而是通过间接的、不经意的自我状态流露，来实现初步的筛选与吸引。其核心公式可以概括为：

**暗度 = 自然呈现生活状态 + 幽默自我引导**

这个公式意味着，你需要以一种轻松、自信且幽默的方式，展示自己真实而积极的生活面貌，而不是生硬地推销自己。

## “暗度”招式的具体操作步骤

理解了“暗度”的核心概念后，以下是执行这一招式的具体操作流程。整个过程是一个自然的对话推进。

### 步骤一：自我状态呈现

当别人询问你的近况时（例如：“你现在在干嘛呢？”），这是启动“暗度”的绝佳时机。

你的回应不应是简单的“在工作”或“没干嘛”。相反，你应该引导对话，以幽默且自信的方式呈现自己积极的生活状态。

**示例代码（对话模拟）：**
```
他人提问：“你现在在干嘛呢？”
你的回应：“我自己开公司，在创业呀。”
```

### 步骤二：幽默自我引导与价值建立

在抛出关键信息后，对方可能会产生好奇或追问（例如：“什么学历？都已经开公司了？”）。此时，需要运用幽默进行自我引导，进一步强化你的独特性和价值感。

**示例代码（对话模拟）：**
```
他人追问：“什么学历？都已经开公司了？”
你的回应：“我初中学历啊（或中专学历）。我那帮初中同学现在都在念大学呢，他们还没出来工作，我都已经工作好多年了。这算是改变命运吧，哈哈。”
```
通过这种对比和幽默的自嘲，你实际上传达了几个关键信息：自信、行动力、社会经验丰富以及不循规蹈矩的独特性格。这能有效激发对方的好奇与吸引。

### 步骤三：吸引形成与后续铺垫

当以上两步完成后，吸引便自然产生。此时，互动的氛围已经从普通寒暄转变为对方对你产生兴趣。

那么，吸引来了之后我们可以怎么做？这时就为后续的互动技巧（如边线带领、挑战性测试）做好了铺垫。这些技巧的目的是**堆砌吸引状态**，即将对方刚刚产生的兴趣和吸引感放大、加深。

## 什么是挑战性测试

在“暗度”成功建立吸引之后，我们提到了可以使用“挑战性测试”。这里对其进行简要说明。

**挑战性测试**，是一种互动技巧，旨在通过轻微、有趣的挑战或调侃，测试对方对你的兴趣程度，并进一步激发对方的情绪投入，从而“勾出”并增强对方的吸引状态。它就像在已经点燃的火苗上扇风，让火势更旺。

**核心要点**：挑战性测试必须建立在初步吸引已经形成的基础上，且态度应该是 playful（好玩）的，而非真正的冒犯或否定。
