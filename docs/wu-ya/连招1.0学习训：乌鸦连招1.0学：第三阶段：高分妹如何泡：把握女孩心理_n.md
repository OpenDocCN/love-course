这个作业的作用是：学会细致观察自己，将注意力放回自身，从而越来越喜欢自己，散发自信能量。

以下是具体操作方法：

2.  **内容**：对着镜子，找到一个自己真实的优点。可以是外形的（如“我的眼睫毛很长”、“皮肤很细腻”），也可以是性格的（如“我很有耐心”）。
3.  **关键**：必须具体、真实。哪怕是很小的优点。不要关注缺点（如痘印），只关注好的地方。
4.  **强化**：在当天感到负面情绪（焦虑、愤怒、沮丧）时，在心里默念这个优点，将注意力转移回来。
5.  **目标**：坚持一段时间后，你会不再需要刻意寻找优点，因为你从内到外都接纳并喜爱自己，形成稳定的自信气场。

这个作业训练你的社交直觉、观察力，以及不带目的性与女性交流的气场。
