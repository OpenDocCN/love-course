## 人与宇宙的三种关系

我们都知道人和宇宙的关系，它有三个关系：**人和自然的关系**，**人和社会的关系**，**人和自己的关系**。

以下是这三种关系的具体内涵：

*   **人和自然的关系**：即我们所说的自然科学。
*   **人和社会的关系**：即我们平时所说的社会科学。
*   **人和自己的关系**：这就是生命科学。

我们现在的文化全球都是一体化，普遍是西方文化。西方文化的特点在于，在人和自然的关系、人和社会的关系方面非常发达，即社会科学和自然科学方面，一直到现在的量子物理学，都非常发达。

但是在人和自己的关系，即生命科学这一块，西方文明是欠缺的，它对人自身的更深层了解，是完全没有的。这方面在东方，像中国的儒家文明、印度的各个宗教，还有很多其他古老民族的文化，反而对自身的研究更透彻，对自己了解得更多。

## 科学的局限与“空性”的发现

在社会科学和自然科学非常发达的现代，我们把一切都通称为科学。但是等科学发展到现代的时候，我们开始认识到，自己最难理解的是我们自身。

当我们用科学的方式、用科学的眼光来看我们自己，就会发现其实完全看不透。用西方科学的研究方法来研究自己，会研究出很多笑话。就像达尔文说人是猴子变的，后来又有人推论人是鱼变的，甚至还有说法说人是章鱼变的。用这种外界思维的方法来研究自己，就会产生各种非常荒唐的说法。

西方科学发展到量子物理学，它把组成世界的物质越分越小，到最底层的时候就发现，这个物质其实是空的，是完全没有任何东西的。你往里面分就是无尽的真空。

这就和我们一直以来想象的“所有物质都由一个最基本的粒子（如原子）组成”不同。科学发现原子往里分，还有质子、中子、电子，再往内部去看，但到最后，你看到的全都是空的。

这个时候我们发现，**空间可能并不存在**。我们再回头去看东方，佛陀在2500年以前就说宇宙是空的，任何东西都是空的。现在看来，确实如此。

所有的物质，人类自己的身体，还有所有的电脑、房间、桌椅，这些东西全部都是空的。它只是由一种粒子的关系组成，就好像是映射在屏幕上的画面一样，都是空的，而且都是由不同的频率组成。

## 宇宙的“闪动”与心态的重要性

频率就是，虽然我们看人体是一个坚固的东西，但实际上如果你用一个非常细微的方式来观察，就会发现我们这个宇宙在不断地、非常快地闪动。

这个闪动在佛教里有一个概念叫“刹那”，每一个刹那有无数个生灭，整个宇宙每一个刹那就重新组建起来。这些宇宙是空的。

为什么说心态非常重要呢？因为这个宇宙实际上是非常空的。这个宇宙是由什么组成的？**其实就是由我们的心态、我们的意识组成的**。

你当然感觉到什么都是实在的，因为你本身就在宇宙之内。但是在我们内心最深的地方，有一个源头，这个源头在不断放射出意识流。这个意识流通过它以前的一些记忆和惯性记忆来创造这个宇宙。

宇宙是空性，在空性之前还有一个“一体性”。所谓的一体性是什么意思呢？就是说你和我，还有我们所有的大家，都只有一个灵魂。

也就是说，女人和你也是一体的。只是因为我们的整个世界是空的，只有一个灵魂，就好像在梦里面一样，我们是梦里面不同的人事物，但这些人事物都是一体的。

这是一种心态，是人生观，和我们平时所受的教育不同。你必须要有这方面的基础，因为我们的课程是一年的，你必须要在这一年里，心态有一个基础。

这个不好理解没关系，你只要愿意去理解，愿意来听，就能够得到很多东西。这不光是一个知识，这是一种能量，有很大的能量。

这个人并不是你妈妈生出来的，他是凭空出现的。他出现之后，整个宇宙就填补了关于他的记忆：你的母亲就会有从小到大的记忆，哥哥的班主任就会想起还有一个特别调皮的学生，他的同桌也会有相关的记忆。
