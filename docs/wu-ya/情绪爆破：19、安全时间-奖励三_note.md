# 情绪爆破：19、安全时间-奖励三_note

在本节课中，我们将深入探讨“奖励”这一情绪互动工具的正确使用时机与核心原则。理解并掌握这一节点，能有效提升你制造情绪波动的能力。

上一节我们介绍了“认同”与“不认同”的不同效果，本节中我们来看看如何通过“奖励”将认同的效果放大，从而制造强烈的情绪刺激。

## 奖励的使用前提

奖励不能随意使用。它有一个关键的前提条件：对方必须先主动做出了某一件事情。

这件事可能并非直接针对你，也可能是对某件事物或环境的行为。例如，我正在讲课，旁边的助手主动帮我擦干净了黑板。

## 奖励的核心原则：十倍认同

当对方做出了一个可以被认同的行为时，我们给出的奖励应该是**十倍以上的认同**。这种夸张的幅度是制造情绪波动的关键。

以下是奖励的夸张表达示例：

> 我没想到你居然是这么勤快的人。你看我在这边讲课，你主动帮我擦黑板。你是这么善良、这么优秀的女生吗？我真的感觉你特别暖心、特别温暖。就因为你这么一个行为，我就爱了。我今天晚上要请你好好出去玩一玩，我们要去散野烂漫处，我连孩子的名字都想好了。

这个例子听起来可能像笑话，但它阐释了奖励的原理：**有多夸张都不为过**。通过极度放大的正面反馈，让对方感受到强烈的情绪波动和价值认可。

## 情绪波动的制造

这才叫做情绪波动，这才叫做情绪刺激。其核心模式是：

*   **认同（奖励）时**：给予**夸张的、十倍以上的**正面反馈。
*   **不认同时**：态度坚决，明确划清界限。
