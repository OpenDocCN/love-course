# 乌鸦救赎海峰情绪爆破课：24、安全时间-无意义陈述句 🧩

在本节课中，我们将要学习安全时间流程中的最后一个，也是最重要的一个技巧：无意义陈述句。我们将探讨它为何是聊天的“主菜”，以及如何运用它来有效推动和收集话题。

上一节我们介绍了安全时间中的其他技巧，本节中我们来看看作为流程收尾的核心部分——无意义陈述句。

许多朋友可能已经注意到，这个技巧在之前的课程中曾出现过。这并非偶然，而是因为无意义陈述句在安全时间流程中扮演着不可替代的关键角色。记住，安全时间流程需要以无意义陈述句作为收集话题的核心。

所有其他的技巧，都扮演着“调味品”的角色。那么什么是“主菜”呢？主菜就是**无意义陈述句**。它是聊天的主菜，这一点尤为重要。在安全时间的结尾，无意义陈述句的核心作用是提供和开启新话题。

以下是关于无意义陈述句的一个具体应用示例：

*   我曾发送过这样一条信息：“我买了一套精神小伙的套装，穿上后走了一段精神步伐。” 这就是一个无意义陈述句。
*   随后我配上文字：“帅吗？” 这就构成了一个完整的话题引子。

无论我与女生的关系推进到何种程度，即使关系已经非常亲密，我仍然需要使用无意义陈述句来建立日常话题。例如，我发一张图片并问她“帅吗？”，她可能会回复“神经病，谁会买那种衣服穿”。这时，我可以接着回应：“我准备拿来当睡衣。” 她可能说：“那你穿的时候别碰我。” 我则可以继续推进：“我要让你感受一下被精神小伙侵犯的乐趣。” 这个过程就轻松地延续了两天或更久的对话。

因此，在安全时间的结尾，我们以无意义陈述句为核心来推动话题，其重要性不言而喻。

至此，关于安全时间的全部五个技巧就讲解完毕了。我们已经掌握了完整的流程。
