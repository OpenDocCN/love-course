# 乌鸦救赎《灌顶计划3.0》课程二：销售身份聊天记录解析 🎯

在本节课中，我们将一起学习并解析一套以“销售身份”为背景的聊天记录。我们将深入拆解其中的每一个步骤，理解其背后的核心原理和操作技巧，帮助你掌握快速建立吸引、推进关系的核心方法。

## 一、开场与初步互动

上一节我们介绍了课程的整体框架，本节中我们来看看如何开启一段对话。开场的目标是建立初步联系，并快速测试对方的兴趣度。

**开场白：**
```
嗨。
```
女孩回应：
```
嗨老伴。
```
我接着问：
```
看我的资料了吗？
```
她说：
```
看了怎么啦？
```
我确认：
```
我确定你看了就OK。
```
然后直接提问：
```
我问你个问题，我是不是你喜欢的类型？
```
她回应（试图建立框架）：
```
你可以变成我喜欢的类型吗？
```
我给出IOD（无兴趣声明）：
```
我喜欢做自己。
```
她说：
```
直男吧？不过我喜欢。
```
我推进：
```
那不就得了。这样吧我们聊聊看，聊得好的话就见面约会，约会相处合适就在一起吧，你觉得怎么样？
```
她同意：
```
可以的。
```
我引导至微信：
```
乖，那微信聊吧。
```

**核心要点：**
*   开场需简洁直接。
*   当对方试图建立框架（如“你可以变成我喜欢的类型吗？”）时，可使用 **IOD（表达无兴趣）** 来维持自身框架，例如“我喜欢做自己”。
*   在对方展示兴趣后，可快速提出“约会-相处-在一起”的**恋爱框架**进行试探和锁定。

## 二、微信深入交流与筛选

成功转到微信后，对话进入深度互动阶段。由于对方已初步接受恋爱框架，我们可以直接开始筛选和建立舒适感。

我开场：
```
我们相互了解一下吧，既然是以恋爱为目的的。
```
她说：
```
好的。
```
我开始筛选：
```
你对男朋友怎么样？会疼人吗？
```
她回答（寻求公平）：
```
相互的吧。
```
我追问：
```
是的，所以你会疼人吗？
```
她确认：
```
当然会了。
```
我提供价值并描述画面：
```
那就好。其实我很疼女朋友，尤其大姨妈，红糖水暖宝宝爱的抱抱。
```
她积极回应：
```
哈哈哈，希望我能够体会到你的疼爱呢。
```
我给予框架：
```
有机会的，你乖就会有机会。
```

**以下是筛选环节的常见问题与回应策略：**

1.  **关于生活技能（如做饭）的筛选：**
    *   我问：`你会做饭吗？`
    *   她可能回避：`可以不会吗？`
    *   退让并维护框架：`哈哈可以不会，但是要打下手。` 接着提供价值：`我挺喜欢做饭的。`
    *   她态度软化：`但是可以陪着你做饭。我会帮倒忙吗？`
    *   肯定态度：`没事，陪着就好了。态度很重要。`
    *   她接受：`哈哈好。`

2.  **关于兴趣与信息交换：**
    *   我问：`乖，你平时喜欢做什么？`
    *   她答：`平时喜欢玩。你看我朋友圈了吗？`
    *   我答：`嗯，大概看了一下，应该可以玩到一起去。`
    *   **身份输出**：她问：`看出来我是做什么工作的吗？` 我答：`我是做销售的。你是做什么的？`
    *   她答：`我是做早教老师的。`
    *   我评价并连接：`哦哦，和小孩子在一起挺好的，应该很会照顾人。`

**核心要点：**
*   **筛选**是了解对方是否符合自己需求的过程，问题需具体（如“会疼人吗？”、“会做饭吗？”）。
*   当对方对筛选问题表现出犹豫或抗拒时，可以进行**合理退让**（如“可以不会，但是要打下手”），同时强调态度的重要性，这能有效维护框架并避免冲突。
*   **身份输出**可以在交流中自然带出，例如当对方询问或话题相关时。如果职位普通，可以说行业而非具体职位，以提升感知价值。
*   快速推进的关键在于使用了**意愿锁定（强切）**，提前打碎了对方的心理防御，使双方能以“潜在恋人”状态直接沟通。

## 三、处理废物测试与分享脆弱面

在交流中，对方可能会进行“废物测试”，即提出一些挑战性问题来测试你的自信和框架。同时，深度的信任建立往往伴随着脆弱面的分享。

她进行废物测试：
```
你喜欢撒娇吗？哈哈你想听我撒娇吗？
```
我回应（接受但附带打压）：
```
会啊，我喜欢嘤嘤怪。
```
她接话：
```
我努力学习。
```
我切入更私密话题：
```
你谈过几次啊？
```
她答：
```
两个。
```
我对比并自我披露（控制数字）：
```
这么少啊，我谈过四五个。
```
她分享脆弱面：
```
可能是我太憨憨了，谈恋爱只想着好好对待对方，可是对方不珍惜，然后我就主动说分手了。
```
我表达共情并连接：
```
这样啊，心疼你。那你对男朋友应该很好。我是前女友太多了，实在受不了。
```
她追问：
```
太多不都是被宠出来的，是吧？
```
我具体化问题（避免抽象）：
```
不是，她脑子里面有剧本，比较烦，喜欢按照电视剧那么来，妈的搞死我了。
```
我反问以筛选：
```
你作吗？会不会没事故意证明一下自己的男朋友不爱你吗？
```
她回应：
```
为什么要证明不爱？不应该是证明爱吗？
```
我表达感受：
```
谁知道，他妈的服了，搞得我都抑郁了。
```
她给予安抚：
```
哈哈没关系，我不会作你的。
```

**核心要点：**
*   应对**废物测试**时，可以表面接受，但通过调侃（如“嘤嘤怪”）等方式进行轻微打压，保持自身姿态。
*   谈论恋爱史时，**数字需合理控制**（例如，30岁左右说“四五个”），过多会显得花心，过少则不够真实。
*   当对方**主动分享脆弱面**（如感情受伤经历），是信任度提升的标志。应给予共情（如“心疼你”），并连接回自身（如“那你对男朋友应该很好”），同时可进行自我披露以增进亲密感。
*   对于“作”或“有剧本”（喜欢戏剧化）的女生，可以在早期通过开玩笑的方式“崩掉”她的剧本，让互动接地气。

## 四、截杀（关系升级）的时机与执行

当感觉到吸引建立且对方无明显顾虑时，便是尝试进行“截杀”（即关系升级，如确立临时情侣关系）的时机。关键在于判断氛围和对方的接纳程度。

我铺垫：
```
你看着还挺可爱的。我有个想法，想不想听？
```
她说：
```
想。
```
我提出“七天情侣”惯例：
```
我们这样吧，这样聊也不知道合适不合适。我们做七天情侣吧，模拟一下。如果七天后感觉好我们就来真的，如果不合适我们就做朋友。七天之内我们像朋友一样相处，感受一下我们彼此的状态，你觉得呢？
```
她同意：
```
好啊，还有这样的玩法，不错的玩法。
```
我提议仪式感以强化同步：
```
好啊，那我们走个仪式嘛，不然什么都没改变。
```
她问：
```
什么仪式？
```
我具体化：
```
改口吧，我叫你老婆你叫我老公，标志我们关系的开始。
```
她犹豫：
```
绝。可以换一个吗？龙哥或者宝贝？
```
我先接纳再坚持（合理化）：
```
哈哈也行吧。但是第一次嘛，仪式。以后可以不叫。第一次做个仪式，你说呢？媳妇还是老婆你喜欢哪个？
```
她仍犹豫：
```
更喜欢宝宝或者宝贝吧，可以吗？
```
我继续合理化：
```
可以啊。我是说第一次嘛仪式感，叫老公。以后可以不叫，这是我们关系的开始。
```
我示范并坚持：
```
那我先做个示范。我先来，老婆还是媳妇，你选一个。
```
她妥协：
```
就宝贝吧，好不好？
```
我最终同意并巩固：
```
以后宝贝。这不是标志我们关系的开始嘛。
```
她最终配合：
```
好的。
```
我完成仪式：
```
恩呢，媳妇。
```
她回应：
```
老公。
```
我确认并结束对话：
```
乖了媳妇。媳妇你还不睡吗？
```
她答：
```
哈哈准备睡了，明天要开始上班了呢。
```
我结束：
```
晚安。
```

**核心要点：**
*   **截杀时机**：关键在于感知对方是否已对你产生信任，主要标志是**无明显顾虑**（如不再频繁废物测试或调查你）。
*   **截杀方法**：使用“**七天情侣**”等惯例，其本质是**状态同步**的一种委婉形式。通过设定一个临时、低风险的关系角色，让双方提前体验恋爱状态。
*   **坚持与合理化**：当对方犹豫时，通过“**仪式感**”、“**这是关系的开始**”等说法进行**逆向合理化**，为同步行为提供理由。可以适当坚持，但也要在对方强烈抗拒时灵活退让，保持互动流畅。
*   **原理深度**：所有关系推进的核心原理都是 **`前价值` + `状态同步`** 。“七天情侣”、牵手、接吻等都是实现状态同步的不同“思路”或“模具”。理解原理后，可以灵活创造各种方法。

## 五、课程总结与核心心法

本节课中，我们一起学习并解析了一套完整的快速吸引聊天记录。从开场破冰、微信筛选、处理测试，到最终的关系截杀，我们看到了如何将“**意愿锁定（强切）**”和“**状态同步**”的原理贯穿始终。

**核心总结如下：**
1.  **快速开场与框架建立**：直接、简洁，在兴趣窗口期快速植入恋爱框架。
2.  **筛选与舒适感并行**：通过具体问题筛选，同时通过共情、自我披露和提供价值（如描述照顾场景）建立舒适感。
3.  **巧妙应对挑战**：对废物测试进行友好打压，对分享脆弱面给予共情连接。
4.  **把握时机进行截杀**：在感知到足够吸引和低顾虑时，使用“七天情侣”等惯例进行关系升级，其本质是**状态同步**。
5.  **核心心法**：社交的终极奥秘在于 **`前价值`** 和 **`状态同步`** 。所有技巧都是为实现“同步”寻找一个合理、委婉的“理由”或“角度”。同时，面对高价值对象时，保持心态：“凡与我无关的资源，皆无意义”，维持内在自信。

**给初学者的建议**：自用时，可将整个流程拉长至1-2天完成，尤其在分享脆弱面建立信任的环节给予更多时间，这会让关系更稳固，避免因过快而产生的“不真实感”或风险感。
