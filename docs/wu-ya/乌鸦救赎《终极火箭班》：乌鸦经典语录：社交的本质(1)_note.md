# 乌鸦救赎《终极火箭班》第1课：社交的本质 🎯

在本节课中，我们将学习乌鸦关于“社交本质”的核心观点。我们将探讨社交的真实目的，并理解它如何服务于个人目标，而非单纯的情感满足或交友。

## 概述

社交的本质并非让他人感到愉悦或不适。社交的目的也从来不是为了交友而去社交。理解这一点是建立有效社交策略的基础。

## 社交的核心目的

上一节我们概述了课程主题，本节中我们来看看社交的真实目的。社交的本质不是为了交朋友，而是为了筛选出与你合适、匹配的人，并与之组成联盟。

我们进行社交不是为了交朋友。没有人会单纯因为“喜欢交朋友”而进行社交。因为个人要完成某些事情，仅凭一己之力往往无法实现。

因此我们需要联盟。这时我们才进行社交，去认识那些能提供帮助的人、认可我们理念的人，或是能共同做事的人。我们没有闲情逸致仅仅为了和睦相处而与一群人社交。

## 社交中的原则与态度

理解了社交的筛选目的后，我们来看看在社交互动中应秉持的态度。社交并非为了追求和睦相处。

如果没有这种预设，那么当遇到让你不爽的情况时，你就应该回击。如果对方的话你不认可，或严重伤害了你的自尊，你就应当予以回敬。从来没有人规定社交必须是为了和睦相处、让大家相安无事。

否则，你就会遇到一些让你非常不爽的混蛋，那时你将陷入被动。

以下是处理社交冲突时的关键点：
*   当对方言行让你不悦时，应果断回击。
*   若对方言论不被你认可或伤害自尊，必须予以回敬。
*   社交不预设“必须和睦相处”的规则。

## 总结

本节课中，我们一起学习了乌鸦关于社交本质的深刻见解。我们明确了社交的核心是**筛选盟友、组建联盟**，而非追求情感愉悦或广泛交友。同时，我们认识到在社交中应保持原则，**不畏惧冲突**，及时回击不当言行，以保护自身利益和尊严。这为建立高效、目标导向的社交关系奠定了思想基础。
