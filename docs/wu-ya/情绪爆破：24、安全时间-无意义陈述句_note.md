# 情绪爆破：24、安全时间-无意义陈述句_note

在本节课中，我们将要学习安全时间流程中的最后一个核心技巧：无意义陈述句。我们将探讨它为何是聊天互动的“主菜”，以及如何运用它来自然地建立和推动话题。

上一节我们介绍了安全时间中的其他技巧，本节中我们来看看作为话题核心的“无意义陈述句”。

## 无意义陈述句是聊天的主菜

之前的技巧都扮演着调味品的角色。那么什么是主菜呢？主菜是叫做**无意义陈述句**，它是聊天的主菜，这个尤为重要。

在安全时间的结尾，我们需要以无意义陈述句为收集话题的核心。所有技巧最终都服务于引出这个核心内容。

## 无意义陈述句的作用

无意义陈述句在此时的作用是提供话题。它通常是一段分享自身状态或趣事的描述，本身不寻求具体答案，而是开启轻松对话。

例如，我经常会发一些信息出去。以下是一个具体例子：

> 我买了一个精神小伙子的套装，然后我穿上之后，来了一个精神的步伐。帅吗？

这段陈述就是无意义陈述句。它分享了一个生活片段，并以一个开放式问题（“帅吗？”）结尾，旨在引发对方的反应和评论。

## 无意义陈述句的运用实例

无论关系推进到什么程度，都可能需要使用无意义陈述句去建立话题。以下是一个完整的对话示例，展示了如何运用：

1.  **发起话题**：我发一张图，问他“帅吗？”
2.  **对方反应**：他说：“神经病。谁会买这种精神小伙子的衣服穿？”
3.  **延续话题**：我说：“我准备拿来当睡衣。”
4.  **互动升级**：他说：“那你穿这个睡衣的时候你不要碰我。”
5.  **幽默收尾**：我说：“我不碰，我要让你感受一下你被精神小伙侵犯的这种乐趣。”

这整个互动过程就是聊天的主菜。它从一段无意义的分享开始，自然地衍生出幽默、调侃和情感互动。

## 为何在安全时间结尾使用

在安全时间的结尾，以无意义陈述句为核心去推动话题，是因为它能最自然地衔接起双方的情绪和注意力，将之前的“安全”氛围转化为有趣、可延续的对话。

本节课中我们一起学习了安全时间流程的最后一个技巧——无意义陈述句。我们明确了它是聊天互动的主菜，理解了它提供话题、建立联系的核心作用，并通过实例看到了如何在实际对话中运用它来创造轻松、有趣的交流氛围。掌握这个技巧，能让你更流畅地引导对话走向。
