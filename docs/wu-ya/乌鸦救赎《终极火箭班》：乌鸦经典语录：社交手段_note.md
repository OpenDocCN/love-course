# 乌鸦救赎《终极火箭班》：乌鸦经典语录：社交手段_note

在本节课中，我们将学习乌鸦关于社交的核心心法与手段。核心在于理解社交的本质是价值交换与精神力量的准备，而非攻击与对抗。我们将探讨如何通过心态调整和具体行为，建立有效且积极的社交关系。

## 心态武装：精神力量的准备 💪

上一节我们介绍了课程主题，本节中我们来看看社交的基础——心态准备。否定是一种精神力量，而精神力量需要用精神力量来解决。你必须让自己的心态强大起来。很多时候，你无法在心态层面上做到随机应变。如果你出门时毫无准备，就必然会遭遇挫折。

因此，精神或心态上的准备，很多时候需要通过自我暗示来完成。例如，出门前进行心理暗示：**“我今天会碰到认可我的人，也会碰到不认可我的人，但这就是事实本身，没有什么可抗拒的。”**

以下是基于此心态的行动准则：
*   努力与那些认可我的人建立联系。
*   忽略掉不认可我的人，他们并不重要。
*   不认可我的人不是我的朋友，也不是我生命的主要组成部分。
*   没有必要在他们身上浪费时间，只与朋友维系关系。

这些都是出门前的必要准备。很多时候，你需要想通一些关键的事情。虽然本节课不深入展开，但必须在精神力量上进行准备或武装。

## 社交准则：从攻击者到价值提供者 🔄

上一节我们探讨了心态建设，本节中我们来看看具体的社交行为准则。首先，自己不要成为一个喜欢攻击别人的人。

因为年少轻狂时，可能看了一本书就自以为知晓了整个世界。但很多人看了一千多本书，反而更加沉默，因为全懂之后会觉得无话可说。所以，不要那样做。

如果你不想被别人攻击，就不要成为喜欢攻击别人的人。这是一种后手策略。如果有人严厉攻击你，你可以回敬，但你自己不要主动成为攻击者。这是一个非常坏的毛病。

社交本是为了交朋友和开心，而不是给自己找不痛快。如果你喜欢攻击别人，就别怪别人为什么老打压你。有些人自己行为不当，四处招惹是非，最后被反击了还想不通是怎么回事。自己就不要成为那样的人。

## 核心策略：成为价值提供者 🎯

在社交过程中，我倾向于成为那个提供价值的人。我常思考的是：**“在每一次社交互动中，我能为这个人做点什么？”** 或者换个角度：**“我如何做事，才能让这个人觉得我很厉害？”**

我不会立刻去想如何怼这个人，并把他怼到颜面无存。那是我20岁之前的想法。那时出门，听到别人说些厉害的理论，我自认为也看了不少书，就会想：**“哼，你在这跟我扯理论？我该怎么用两句话把你怼到颜面无存？”**

但我从未成功过。为什么？因为即便这个人读书没我多，他也有自尊。你怼别人，别人一定会反过来怼你，这很正常。一旦他不认可你，你说得再有道理他也听不进去，他一定会回击。

所以，不要给自己找这种不痛快。你首先应该想的是：**如何为对方提供价值？如何让对方觉得你厉害，并愿意跟随你？**

如果你想征服他人，想在外收小弟、收小妹，就要想清楚：大哥从来都是愿意分享、割舍利益的人。即 **`我能将我的资源价值分享给你，别人才会臣服于你`**。

你把别人打压一顿，然后指望别人给你当小弟，这是不可能的。你怼我，就是不认可我的价值；你不认可我的价值，我为什么要与你建立亲切的联系？这在逻辑上不成立。

---

**本节课总结**：我们一起学习了乌鸦社交手段的核心要点。关键在于 **心态武装**，通过自我暗示做好精神准备；行为上要 **避免成为攻击者**，转而 **成为价值的提供者**。社交的本质是吸引与共赢，通过分享资源和价值来建立稳固的关系，而非通过打压和对抗。记住，强大的心态和利他的行为，才是有效社交的基石。
