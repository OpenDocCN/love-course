# 情绪爆破：26、故事时间-①传达信息 📖

在本节课中，我们将要学习如何通过“传达信息”这一核心技巧，来制造聊天中的“故事时间”与“契合点”。这是开启高质量对话的关键一步。

## 概述

“故事时间”是指双方进入一种高度契合、有聊不完话题的聊天状态。这种状态的出现，往往是因为某个话题恰好契合了对方的需求或兴趣。为了主动寻找并制造这种契合点，我们需要掌握“传达信息”的技巧。

上一节我们介绍了“故事时间”的概念，本节中我们来看看如何具体地“传达信息”。

## 核心技巧：无意义的陈述句

在“传达信息”中，一个非常有效的方法是使用**无意义的陈述句**。这个技巧我们在“安全时间”部分已经接触过。它的核心作用是：在不施加压力的情况下抛出话题，试探对方的兴趣点，从而寻找可以深入聊天的“契合点”。

对于初学者而言，由于可能缺乏丰富的社交经验，很难立刻找到能引起对方强烈共鸣的话题。因此，不需要思考复杂的花招，最直接有效的方法就是反复使用这个核心技巧。

**核心操作公式**：
```
不断使用【陈述句】去试探 → 寻找【话题契合点】 → 进入【故事时间】
```

## 操作步骤

以下是使用“传达信息”技巧进入故事时间的具体步骤：

1.  **持续抛出陈述句**：不要等待完美的时机，而是主动、频繁地用简单的陈述句开启或延续话题。这些陈述句可以是对周围环境的观察、对某件事的感受，或一个简单的自我状态描述。

2.  **观察反馈**：每抛出一个陈述句后，注意观察对方的反应。如果对方对这个话题表现出兴趣（例如，回应积极、追问细节、分享自己的相关经历），那么就找到了一个潜在的“契合点”。

3.  **抓住契合点深入**：一旦发现对方对某个话题有反馈，立刻停止“轰炸”，转而围绕这个契合点展开深入聊天。这时，对话就自然进入了“故事时间”。

4.  **简化思维**：在整个过程中，不要纠结于自己说的话是否“有意义”或“足够精彩”。你的核心任务是通过大量简单的陈述句进行筛选，直到击中对方的兴趣点。

## 总结

本节课中我们一起学习了“传达信息”这一关键技巧。它的核心在于，通过持续使用**无意义的陈述句**进行话题试探，来寻找与对方的聊天契合点。找到契合点后，即可顺利开启双方都感兴趣的“故事时间”。对于初学者，这是最直接、最不需要复杂社交直觉的入门方法。记住，行动比完美思考更重要。
