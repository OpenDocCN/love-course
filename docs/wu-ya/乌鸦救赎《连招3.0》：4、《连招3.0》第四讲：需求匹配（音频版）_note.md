## 课程概述

上一节我们介绍了身份输出，完成了外在价值的展示。本节中我们来看看如何通过“需求匹配”来展示内在价值，并推动关系进入相互吸引的阶段。需求匹配的核心，是在身份输出后直接进行筛选，其本质是给予双方展示价值的机会。

## 需求匹配的原理与时机

身份输出结束后，可以直接进行需求匹配。用探讨情感需求的方式来展开话题。需求匹配就是在做完身份输出以后，直接告诉对方你的身份背景，例如“我是自己创业，创业十几年”。这是一个身份输出。

做完身份输出以后，可以直接进行筛选。需求匹配其实就是筛选，通过筛选的方式去探讨两人的需求。这个过程会比以前快很多。

之所以可以如此直接，是因为前面已经做了意愿锁定。身份输出也满足了对方的价值匹配需求，所以就可以直接开始筛选。

这次我们不叫“人格筛选”而叫“需求匹配”，其实是一个意思，但“需求匹配”会更准确一些。

这一块内容与之前的人格筛选变化不大，只是现在可以直接拿出来用了。做完身份输出之后直接进入筛选。

需求匹配有这么一个特征，也就是达成相互吸引的共识。这样再去推动关系，就会感觉轻松很多。

连招一直避免剃头挑子一头热，总是让女人的意愿和我们的意愿处于同步状态。很多时候，连招更喜欢让女人的意愿先于我们，先让女人表示出她对我们的喜欢。

筛选的目的和意义跟以前基本没有变化，只是现在可以直接拿出来用。身份输出以后直接拿出来就好，不需要再去过渡别的东西。直接把挑战性测试等环节都去掉，一样可以用。

## 如何进行需求匹配：方法与示例

以下是进行需求匹配时可以采用的一些问题示例，其核心是借提问展示自身观点。

*   **示例一：关于照顾与依靠**
    *   **筛选问题**：你对男朋友怎么样，会不会心疼男朋友？
    *   **价值展示**：不管女生回答什么，你可以发表观点：“我觉得两个人在一起就是找一个依靠。比如女朋友大姨妈的时候一定要好好照顾好，情绪不好、身体不舒服的时候，正是需要男朋友的时候，否则谈恋爱是为了什么？”

*   **示例二：关于安全感与“作”**
    *   **筛选问题**：你谈恋爱的时候会“作”吗？
    *   **价值展示**：你可以说：“我不喜欢女生太‘作’，因为我觉得‘作’的女生大多没安全感。没有安全感的人往往会通过一些破坏性行为来确定男朋友对她的态度，但这会伤感情。其实女朋友‘作’也是因为没有安全感，所以我会提前把她的安全感给够。”

*   **示例三：关于信任与空间**
    *   **筛选问题**：你会查男朋友的手机吗？
    *   **价值展示**：你可以回应：“我是不会查女朋友手机的。因为我认为这是一种不自信的表现，而且每个人都有自己的空间。什么都想插手，其实是一种不成熟幼稚的表现，反而会让女朋友觉得你很幼稚。”

*   **示例四：关于面子与尊重**
    *   **筛选问题**：你会在人前给男朋友面子吗？
    *   **价值展示**：你可以表达：“我觉得两个人在外人面前是要相互捧的。两个人如果相互贬低，外人只会更加看不起两个人。我觉得在外人面前不给女朋友面子的男人，都是烂泥扶不上墙。”

*   **示例五：关于家庭与亲情**
    *   **筛选问题**：你和父母的感情如何？
    *   **价值展示**：你可以分享：“我跟父母的感情很好。和父母感情不好的人是最不靠谱的，对至亲都是这个态度，你跟她结交能有什么好下场？”

*   **示例六：关于孩子与责任**
    *   **筛选问题**：你喜欢小孩吗？
    *   **价值展示**：你可以说：“我以前挺不喜欢小孩的，觉得太闹。但现在上了年纪，觉得小孩特别可爱。而且一看到小孩，就会去联想如果这是我的孩子，我该怎么去教育他。”

**关键操作要点**：
1.  筛选问题要偏向内在需求和价值，避免外在或争议性话题。
2.  不需要很多问题，四到五个就够了。
3.  **必须避免连续发问**，不要像查户口。
4.  把每一个需求匹配的话题都展开聊得深入一些，聊几分钟后再开启下一个话题。
5.  其核心是 **`抛砖引玉`** ，借由问题引出自己的观点，完成内在价值展示。

## 需求匹配在连招3.0中的特殊意义

在连招3.0中，需求匹配的意义有所变化。因为通过前置的意愿锁定，整个对话从一开始就笼罩在谈恋爱的氛围中，类似于一种“彼此有感的有效相亲”。框架被锁死，游戏难度降低。

因此，需求匹配虽然仍有给予女生证明价值机会的作用，但已不是最重要。它更多地变成了一个**加强版的价值展示环节**，侧重于通过抛砖引玉的方式，侧面展示自己的内在价值、情感电路价值。

在连招3.0的流程中，意愿锁定和身份输出（前价值模块）展示了外在价值。需求匹配则负责展示内在价值。至此，吸引阶段就全部完成。这个过程可以非常快，可能几十分钟就能完成。

## 注意事项与后续步骤

完成需求匹配后，女生会感受到强大的吸引力，但又不易发动废测（因为前置的意愿锁定破坏了一致性）。此时，你看起来可能“太像个渣男”，因为过程过于丝滑和精准，反而可能让女生产生不安全感或怀疑。

因此，吸引完成后，**必须进入下一阶段：建立安全感和信任**。如果缺少这一步，关系很容易失败。下一讲我们将详细讲解如何建立信任。
