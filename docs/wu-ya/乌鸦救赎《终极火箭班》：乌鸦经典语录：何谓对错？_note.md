# 乌鸦救赎《终极火箭班》第1课：何谓对错？🤔

在本节课中，我们将深入探讨乌鸦救赎提出的一个核心概念：如何判断事物的对与错。这个概念是理解后续许多策略与行动的基础。

## 概念阐述

上一段我们提到了需要建立一个判断标准，本节中我们来看看这个标准的具体内涵。

“对你不利的东西就是错的。” 这是判断对错的根本原则。一件事或一个选择，无论其表面看起来多么合理或高尚，只要最终结果对你不利，那么对你而言，它就是错的。

**核心公式**：`结果对自身有利 = 对`；`结果对自身不利 = 错`

这个标准将判断的锚点从抽象的道德或他人的看法，转移到了具体且可衡量的**自身利益**上。

## 判断与应用

理解了核心定义后，以下是应用这一原则的三个关键步骤：

1.  **识别结果**：首先，需要清晰地分析一个决定或行动可能带来的各种结果，尤其是长期结果。
2.  **评估利害**：然后，客观评估这些结果中，哪些对你有益（如提升能力、获得资源、改善关系），哪些对你有害（如损耗精力、损失机会、损害健康）。
3.  **做出判断**：最后，根据“利大于弊”还是“弊大于利”来做出“对”或“错”的判断。

## 核心要点总结

本节课中我们一起学习了乌鸦救赎关于“对错”的界定标准。

总结起来，其核心在于**以自身利益为终极判准**。这要求我们摆脱外界杂音的干扰，培养独立、理性的分析能力，专注于行动和选择所带来的实际后果，并坚定不移地维护和增进自身的根本利益。掌握这一原则，是做出明智决策的第一步。
