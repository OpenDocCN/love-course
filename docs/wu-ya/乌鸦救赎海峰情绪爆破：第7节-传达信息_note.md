# 乌鸦救赎海峰情绪爆破：第7节-传达信息_note

在本节课中，我们将要学习聊天组成部分的第四个环节：**传达信息**。这个环节对于建立深度连接至关重要，它能确保双方在互动中达到平衡的了解。

上一节我们重点介绍了互动和收集信息的重要性。本节中，我们来看看如何有效地进行自我展示。

很多人卡在互动环节，导致聊天无法继续。实际上，传达信息也极为重要。如果前面环节都做好了，但没有经历传达信息，会产生一种情况。

请思考一个问题：如果你做好了收集信息和互动，但没有传达信息，会产生什么情况？这种情况其实很简单。女生不会了解你。

她不知道你是一个什么样的人，因为你没有把自己展示出来。你在不断提问时，可能有些女生不擅长聊天。她不会问你的工作、家庭情况或你的三观。

她问不出这些问题。这会造成一种后果。后果就是，你很了解她，知道了她的很多事。因为我们做了收集信息，也做了互动，并用互动撩动了她的情绪。记住，互动是由无数技巧组成的环节，专门用于撩动女生情绪波动。

这个地方加入了很多技巧。这些叫做技巧。传达信息我们也必须去做。原因在于，如果你使用了大量技巧了解她，但她不了解你，就会造成一种后果。

她觉得还不够了解你，你们还不能在一起。那么，传达信息具体怎么做？做些什么呢？通常我会刷脆弱面。

刷脆弱面。什么是刷脆弱面？传达信息一般可以在五个时间法则里进行。它等于进入了哪个时间法则？这是一个问题。

看完这节课后，请把这个问题在群里发给我。传达信息如果到达这个步骤，你觉得在五个时间法则里，它属于哪一个时间法则？

有一部分新来的朋友可能不知道情绪爆破的五个时间法则。在讲完聊天主谈部分后，我会精讲五个时间法则中我们该做的事情。

所以不用着急。那么，传达信息是什么意思呢？也就是把我自己的东西传达给你。

我的故事、我的想法、我所有的真诚部分都可以表达给你，让你知道清楚。从而起到一个效果，从而起到让你了解我，知道我是一个什么样的人，我值不值得你去爱的效果。

我们可以在传达信息时分享一些脆弱的东西。为什么要分享脆弱的东西？因为人性很奇怪。

人性在此时，当他看到脆弱面时，你要记住，我们是在这个步骤去传达脆弱面。但在前面的打招呼、收集信息和互动时，我们不要展示脆弱的东西出去。

而是在这个地方我们传达。分享脆弱的地方，他会感觉认识了真正的你。他知道真正的你是什么样，是别人看不到的。他会有这种感觉，他觉得你呈现了一个真正的我给他。

所以这很能打动人。这就是传达信息我们要做的一件事：尽量分享自己的脆弱之处，以达到让别人真正认识你的效果。这很棒。
