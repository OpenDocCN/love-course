# 15、预期式求爱聊天：第七讲：用预期搞定社交与谈判（音频版）_note

在本节课中，我们将学习如何将预期理论应用于社交与谈判领域。我们将探讨社交的本质、常见的误区，并学习一套从基础到高级的社交策略，帮助你建立高质量的人脉网络，保护自己免受伤害，并最终通过“势能”和“圈子”的力量达成目标。

---

## 一、社交的本质与常见误区

上一讲我们探讨了预期在亲密关系中的应用。本节中，我们来看看如何将同样的原理应用于更广泛的社交场景。

首先，请思考你对社交的理解是什么？在没有接触我的理念之前，你认为社交应该如何进行？你希望通过社交解决什么问题？

以下是常见的社交理解：
*   相互帮助。
*   交朋友。
*   一起玩乐、聊天、吃饭、打游戏。
*   认识厉害的人，寻找贵人。
*   获得归属感和安全感。

然而，现实中的社交往往与预期严重不符。你遭遇过哪些违背预期的事情？
*   费力不讨好。
*   遭遇背叛、被朋友坑害。
*   对方自私，只在乎自己。
*   有求不应。
*   相互打压，遇到小事也帮不上忙。
*   自己总是付出，收获却不对等。
*   有钱一起赚，有难各自飞，甚至落井下石。
*   难以融入某些圈子。
*   社交浪费时间，无法获得想要的资源。

这种普遍的失望感源于**需求来自于限制性条件**。很多人想学社交，正是因为现实社交常常违背预期，无法做到“有事真上”。虽然存在真诚互助的朋友，但大部分社交都难以达到互利互惠、公平对等的理想状态。

一个关键的人性规律是：**付出者会高估自己的付出，而被赠予者会低估对方的赠予**。这导致了期望与感知的错位。

## 二、基础社交：价值展示与吸引

上一节我们分析了社交的痛点，本节中我们来学习解决这些问题的第一个核心步骤：建立吸引。

建立朋友关系的方式与建立恋爱关系在原理上完全一致。核心流程可以简化为：**吸引 -> 信任**。朋友关系通常是自然形成的，不需要特定“仪式”，关键在于共谋和自然的心意相通。

那么，如何建立吸引？核心是 **“喷”**，即主动进行价值展示。

### 价值展示的核心原则

在陌生社交场合，你必须在**前二十分钟**内完成首次吸引。你需要养成习惯，一旦有陌生人在场，就主动进行一轮精彩的自我介绍，亮明自己的价值。

**公式：社交吸引力 ≈ 高价值展示的速度与精准度**

以下是具体操作方法：

1.  **准备精炼的自我介绍**：这不是介绍姓名年龄，而是展示你**最光鲜的成就、合作过的关键人物、做过的核心事情**。例如：“我叫吴亚，专注情感领域，目前是中国男性情感咨询的头部从业者，全网拥有数百万粉丝，创业十余年，也投资了几家公司……”

2.  **根据场景灵活调整**：
    *   **已知对方需求**：针对其潜在需求进行垂直展示。例如，在艺术家聚会中，可以介绍自己在艺术品收藏和拍卖领域的背景。
    *   **未知对方需求**：进行“无差别攻击”，把自己最突出的价值点都展示一遍，总有一点可能击中对方。

3.  **克服“低调”误区**：在陌生社交中，“多听少说”是害人的。如果你错过最初的吸引窗口，后续再展示自己会显得像在攀附他人。必须要有 **“舍我其谁”** 的气势，即使面对地位很高的人，也要自信地展示自己的独特价值或能力。

4.  **利用好“自我介绍”环节**：这是你抛砖引玉、展示价值的黄金机会。避免说无关信息，直接输出事业和成就。

价值展示（“喷”）的目的是建立初步吸引，为后续建立信任打下基础。如果二十分钟内没有形成吸引，后续很可能沦为无效社交。

## 三、建立信任与筛选盟友

上一节我们学会了如何通过价值展示打开局面，本节中我们来看看吸引之后如何建立信任，并筛选出真正有价值的社交伙伴。

建立信任的方式与恋爱类似：
*   **多见面**：聚会、转场，增加接触频率。
*   **聊私密话题**：交换信息，故意抛出一些自己的小秘密。
*   **建立共谋**：聊对方遇到的麻烦，并帮助解决。

如果线上加了微信后不知如何推进，可以尝试以下路径：
1.  **观察朋友圈**：判断其兴趣爱好和垂直需求。
2.  **投其所好**：从兴趣爱好切入破冰（如游戏、品酒、艺术），这比直接谈事业更让人没有防备。
3.  **主动邀约**：以“聊得开心，一起坐坐/喝一杯”为由发出邀请。酒精能快速拉近距离。
4.  **在互动中展示事业价值**：在建立联系的过程中，自然展示你事业上的能力和资源。

### 关键心法：向上社交与筛选

社交的核心是**向上社交**，至少是平级交往。与普通人交往十年获得的成长，可能不如与高价值人士交往一次获得的启发或机会。

**如何筛选高价值盟友？**
一个重要的认知是：**存量资源越高的人，人品通常越好，越愿意付出；存量资源越低的人，越可能因自保而做出损害他人的行为**。因此，要结交善良且有一定经济基础的人。

建立高质量人脉的通道主要有：
*   **付费学习班/总裁班**：价格越高，筛选出的人群质量通常也越高。
*   **高端俱乐部**（如高尔夫、车友会、艺术协会）：基于共同兴趣的社交场。
*   **朋友转介绍**：通过现有靠谱的朋友认识新朋友。
*   **通过异性（需谨慎）**：风险较高，关系可能随恋情结束而终止。

**切记**：不要依赖命运随机安排的关系（如同学、同事）。主动通过上述渠道走出去，构建自己的人脉网络。

## 四、进阶策略：背书式社交

上一节我们讨论了如何一对一地建立关系，本节中我们升级玩法，看看如何利用“势能”高效社交。

社交与谈判是一体的。谈判需要话语权，而话语权来自于**势能**。“势”即“人众为事”，你身边支持你、认可你的人越多，你的势能就越强。

**背书式社交**的核心是：**与你的社交盟友组队，在第三方面前互相捧场，为彼此的价值背书**。

### 操作流程与核心

1.  **找到可靠的社交盟友**：这是最大的难点。你需要一个值得信任、懂行情、愿意配合你，并且彼此有时间相处磨合的人。
2.  **共同出场，相互背书**：在见第三方时，互相介绍对方的成就和厉害之处。例如：“这位是李总，刚用一个月时间通过一个小项目赚了一百多万。”“这位是直播营销专家，师从XXX，一场活动能带货几千万。”
3.  **效果**：这种“三人成虎”的场面能快速建立信任和吸引。第三方会看到你拥有高质量的友谊和团队，从而产生羡慕感和加入欲。这比独自一人费力证明自己高效得多。

### 背书式社交的威力

它的核心威力不在于聊天技巧，而在于**展示了一段令人羡慕的高质量关系**。就像刘备有关羽张飞，唐僧有孙悟空猪八戒，这种“兄弟同心”的画面本身就有巨大的吸引力。人们渴望加入这样的团队，获得这样的友谊。

**如何找到第一个盟友？** 从你通过正常社交认识的人中筛选：善良、有一定经济基础（不差钱）、有付出意识、志向相投。花时间磨合，一旦找到第一个，形成“团队”的雏形，吸引第二、第三个盟友就会容易很多。

## 五、高级组织术：趋势式社交与圈子建设

上一节我们学会了利用盟友进行背书，本节中我们来看最高阶的玩法：构建圈子，利用趋势的力量。

这种玩法已经超越了单纯社交，进入了**组织术**的范畴。其核心流程是：

1.  **广泛连接**：围绕一个标的物（如一个创业项目），广泛接触和筛选对此感兴趣的人。
2.  **相互介绍，形成圈子**：将这些潜在伙伴相互介绍认识，让他们彼此交流，围绕标的物进行讨论。
3.  **自然筛选，形成趋势**：在讨论中，兴趣不足的人会自然脱落。留下的人通过不断交流强化共识，形成“众口铄金”的趋势。你作为核心连接点，自然成为圈子的中心。
4.  **利用趋势，达成目标**：当圈子形成势能后，无论是推进项目、融资还是销售，都会变得非常容易。

### 圈子的双重价值

1.  **进攻性价值（做局）**：人多势众，可以更容易地说服圈外人，搞定项目。
2.  **防御性价值（保护）**：这是防止在社交中受伤害的关键。**伤害成本低是你被坑的主要原因**。

**公式：个人受害风险 ∝ 1 / 社交圈子的互联程度**

如果你总是与某人单线联系，他伤害你的成本很低。但如果你把他引入你的圈子，让他与你的其他朋友相互认识，他的违约成本就大大增加。在高度互联的圈子里，做坏事意味着社会性死亡。因此，**经营圈子是保护自己资产和情感的最佳方式**。

### 重要心法总结

*   **项目不成，圈子已成**：即使最终项目没做成，你也已经构建了一个有势能的优质圈子。
*   **先认可人，再认可事**：必须先通过社交取得对方对你个人的认可，才有机会合作项目。
*   **分阶段练习**：先掌握正常社交（价值展示），再练习背书式社交，最后尝试构建圈子。
*   **保持距离与神秘感**：即使是社交盟友，也要保持适当距离，不要像对亲人一样全盘托出，以防未来关系生变。
*   **让对的人自己跳出来**：不要费力说服所有人。真正靠谱的、愿意参与的人，会在你营造的趋势中自己主动站出来。

一个有趣的规律是：**最后被你“搞定”的人，往往不是你的初始目标，而是那个说要和你一起去“搞定别人”的人**。这本质上是“共谋成交法”，将目标对象转化为合伙人。

## 课程总结

本节课中，我们一起学习了如何将预期管理应用于社交与谈判：

1.  **认清现实**：理解了社交中期望与现实不符的普遍性及人性根源。
2.  **打好基础**：学会了通过精准、自信的“价值展示”（喷）在陌生场合快速建立吸引。
3.  **建立连接**：掌握了通过共同兴趣建立信任，并明确了**向上社交**和筛选高价值（善良、有资源）盟友的原则。
4.  **利用势能**：学习了**背书式社交**，通过与盟友互相捧场，高效建立信任和吸引。
5.  **构建壁垒**：掌握了最高阶的**趋势式社交与圈子建设**方法，不仅能主动“做局”达成目标，更能通过增加他人的违约成本来**有效保护自己**，杜绝无效和伤害性社交。

社交的本质是价值的识别、展示与交换。从一对一的真诚沟通，到利用关系网络形成势能，每一步都需要练习和领悟。记住，你的目标不是讨好所有人，而是构建一个能滋养你、保护你并助推你成长的优质生态系统。
