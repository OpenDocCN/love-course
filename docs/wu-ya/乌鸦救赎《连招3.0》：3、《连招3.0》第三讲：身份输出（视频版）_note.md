# 乌鸦救赎《连招3.0》第三讲：身份输出 🎯

在本节课中，我们将要学习《连招3.0》的第三个核心环节——身份输出。我们将探讨如何通过简洁有效的方式，快速完成价值匹配，从而建立吸引力，为后续的互动铺平道路。

## 课程概述

上一节我们介绍了意愿锁定，本节中我们来看看如何在建立初步联系后，高效地展示自身价值。我们将深入理解人性在社交中的认知机制，并学习如何利用“身份输出”这一技巧，替代传统复杂的自我引导过程。

## 从探探到微信的过渡问题

许多学员反馈，从探探匹配成功并转到微信后，聊天氛围会突然变冷，感觉对方像换了一个人。这通常是因为对话软件和界面颜色的改变，给女生造成了“重新认识”的错觉。

**核心要点**：从探探加到微信后，**千万不要重新打招呼**（例如发“嗨”）。这会暗示双方关系回到了起点。

### 解决方案：强制认同

为了解决这个问题，我们需要使用“强制认同”技巧。以下是具体操作步骤：

1.  **话术示例**：
    ```
    男：我们先相互看一下朋友圈了解一下，然后5分钟以后再聊，不然不熟悉也没啥话题。
    女：好的。
    ```

2.  **执行后**：双方去看对方朋友圈，5分钟后再开始聊天。

### 强制认同的三个好处

以下是强制认同策略带来的三个核心优势：

1.  **确保信息查看**：强制对方查看你的朋友圈，展示你的价值。很多人加了微信后并不会主动去看。
2.  **建立共识**：将“对方看过你朋友圈”这件事从“未知”变为“已知的共识”，让对方无法假装没被吸引。
3.  **完成初步信息交换**：双方都知道彼此查看了朋友圈，相当于完成了一次初步的信息交换，有助于建立安全感和信任感。

**特殊情况处理**：如果对方朋友圈是“三天可见”，可以直接提出：“你朋友圈是三天可见，打开一下让我看看。” 这是一个合理的请求，对方通常会接受。

完成强制认同后，关系的结构就被加固了。你成为了对方候选人中，唯一一个完成了彼此信息交换、知根知底的男性，这能有效区分你与其他竞争者。

## 理解社交博弈：短期 vs 长期

在深入身份输出之前，我们需要理解陌生人社交的本质是一种**短期博弈**。

*   **短期博弈特征**：功利性强，不讲人情，人们更容易采取对抗或欺骗策略，因为不害怕失去未来的合作机会。
*   **长期博弈特征**：基于未来多次合作的预期，人们才会采取友好、合作的态度。

在短期博弈中，胜利取决于两个条件：**1. 高价值** 和 **2. 无需求感**。谁先展现出这两点，谁就赢了。反之，先展现低价值或高需求感的一方就会处于劣势。

**吸引阶段的本质**，就是将对方从“短期博弈”的对抗心态，转变为“长期博弈”的合作心态。而“推开”或制造“失去感”，是激活对方长期博弈态度的关键。

## 身份输出的原理与操作

完成了强制认同和初步的博弈心态调整后，就要进入本节课的核心——**身份输出**。

在《连招3.0》中，**身份输出直接取代了传统复杂的“自我引导”**。自我引导是通过侧面暗示来展示价值，而身份输出则更为直接高效。

### 为什么身份输出有效？

这基于对人性的深刻洞察：

1.  **价值匹配需求**：人在社交中本能地需要快速判断对方的价值，以决定是否继续投入。
2.  **信号制认知**：人脑为了节省认知资源，倾向于通过简单的**标签**或**信号**来快速分类他人，而不是深入了解全部细节。例如，我们更可能记住“他是个创业者”，而不是他创业的具体细节。
3.  **认知资源有限**：人们更关注自身需求，分配给陌生人的认知资源非常有限，一个鲜明的价值标签就足够了。

因此，与其等待对方来询问和定义你，不如主动出击，提供一个高价值的身份标签。

### 身份输出的具体方法

身份输出的核心是：**用一句话，展示一个高价值的身份标签**。

**公式**：`我是 [高价值身份标签] 。`

*   **示例1（身份本身高价值）**：“我是自己创业的，做互联网自媒体这块十几年了。”（创业者标签）
*   **示例2（职业不突出，则说行业）**：不要说“我是送快递的”，而要说“我是做物流行业的”。**记住：不要回职业，要回行业。**
*   **示例3（利用爱好）**：如果你的职业不具优势，可以将有价值的爱好作为身份。例如，喜欢摄影就说“我是个摄影师”，喜欢健身就说“我是健身教练/达人”。
*   **示例4（绑定前沿趋势）**：将身份与有潜力的领域绑定。例如，“我是做区块链开发的，主要研究Web3.0方向。”

**关键**：呈现方式决定价值感。你需要找到自己最有价值、最亮眼的那个角度，以身份的形式简洁输出。一句话即可，多说反而可能暴露需求感。

## 课程总结

本节课中我们一起学习了《连招3.0》的第三讲——身份输出。

1.  **解决了过渡问题**：使用“强制认同”技巧，避免从探探转到微信后氛围变冷，并初步建立信任。
2.  **理解了社交博弈**：认识到陌生人社交是短期博弈，吸引就是将其转化为长期博弈。高价值和无需求感是短期博弈中的致胜关键。
3.  **掌握了身份输出**：基于人性“信号制认知”的特点，我们无需复杂自我引导。只需用一句话，主动输出一个高价值的身份标签（职业、行业或爱好），即可快速满足对方的价值匹配需求，完成吸引阶段。

身份输出是化繁为简的关键一步，它让整个吸引过程变得高效、可复制。请根据自身情况，设计出最适合你的那一句身份输出话术。
