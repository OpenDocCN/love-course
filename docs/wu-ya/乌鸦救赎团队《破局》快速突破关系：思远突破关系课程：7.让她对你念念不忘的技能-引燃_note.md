“引燃”这个名称来源于游戏《英雄联盟》中的召唤师技能。其特点是造成持续性的真实伤害，无视防御。

在聊天中应用“引燃”，意味着在双方兴趣达到峰值时突然抽离，不再回复。这会给女生心理造成持续影响，让她不断思考你为何与众不同，从而在你心中占据更重要的位置。

3.  **果断抽离**：说完借口后便不再回复，将这次高峰体验像“存档”一样留在对方记忆中。
4.  **再次开启**：下次聊天时，女生会带着上次的“存档”记忆，反应通常会更热情。
