这其实不是为了残暴而残暴，而是为了不被别人残暴，我们才去“残暴”别人。人为刀俎，我为鱼肉。就是说我们如果不想成为别人的鱼肉，不想被别人按在案板上面去被别人欺负的话，我们就得在别人欺负我们之前把他给“欺负”了，把他的气焰给打压下去，这就是很好的保护我们自己。这就是个借口，我们就是为了保护自己。

任何事情都可以找出借口，我们可以利用这些借口去说一些事情，让别人能够理解我们。如果我们直接说出我们这些观点的话，别人也许不会同意，不会理解。

你觉得你是屌丝呢，很大程度上屌丝和那个好男人只有一线之隔。差别在哪？就是因为屌丝没有自己的原则，没有自己的底线，所以没有对这个世界的一个自己的理解。所以我们现在要做的就是：如果你们想跟着我这个属性走，那就先得走这个“好人属性”，就是得把自己好好的作为一个人，先把自己“人”的这一面给做好。就是要把自己原则和底线这些东西都给想明白了。

我们得一个巴掌一个枣，这种起伏就是我所创立的这个东西。前两个属性和后两个属性是对立的，但是这样的东西对立着，所以才能相生相克，互相牵制，你这个人才能处于一种平和状态。不然的话，光是把一种属性发挥到极致，你这个人是不完美的，性格上还是有缺陷的，别人还是能找到你缺陷在哪能攻破你。

所以说，任何一种属性（约炮属性也是）都是人的属性，女人也是可以接受的。我觉得我们都是要经历一遍的。当然了，我们不一定需要经历那么多，就是我们可以先把一种属性研究到一种极致境界，然后其他属性自然而然的，我们就能理解了。

就像练气功，不是说任督二脉吗，两个脉只要打通了，那身上奇经八脉就会打通，身上的百脉都会打通。就是说这些道理都是大道，大的道理都是相通的。宇宙中的这些规则，因为我们能够遵守，所以我们才能进化到现在。

所以我觉得如果我们想作为一个无懈可击的、完美的、成熟的人，作为一个“完整”的人，我们必须要经历过这些东西。就像人必须要经历过苦难，才会理解不苦难的日子是多么美好。
