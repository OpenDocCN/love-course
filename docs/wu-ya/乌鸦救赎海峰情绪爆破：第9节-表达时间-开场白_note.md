# 乌鸦救赎海峰情绪爆破：第9节 - 表达时间·开场白 🕐

在本节课中，我们将要学习情绪爆破聊天方法中“表达时间”的第一步：开场白。我们将探讨如何通过观察来构建独特且有效的开场方式，从而顺利开启对话。

上一节我们介绍了聊天的组成部分，本节中我们来看看聊天的具体流程与步骤。情绪爆破方法包含五个时间法则，本节课聚焦于第一个法则“表达时间”中的开场白环节。

开场白的核心在于观察。情绪爆破的开场方式并非固定话术，而是基于对聊天对象的细致观察。

以下是可用于观察并构建开场白的几个关键点：

*   **签名**：对方在社交平台或资料中留下的个性签名。
*   **样貌**：对方的外貌特征或给人的第一印象。
*   **身材**：对方的体型、健身痕迹等身体特征。
*   **穿衣搭配**：对方的着装风格、配饰等细节。
*   **其他细节**：任何能引起你注意的独特之处。

观察这些信息后，你可以将你的第一印象转化为开场白。例如，若对方身材健美，开场白可以是：“你看起来很结实。”这种开场源于具体观察，因此独特且真诚，能有效引起对方注意。

不要被固定的开场话术束缚。通过观察得到的开场方式更为灵活和个性化，是情绪爆破方法的精髓。它比以往课程中提到的任何固定开场方式都更有效。

本节课中我们一起学习了如何通过观察来打造情绪爆破式的开场白。关键在于先观察，再开口，从对方的签名、样貌、身材、穿着等细节中寻找开场灵感。下一节课，我们将继续“表达时间”的第二步：学习如何在开场后立即进行真诚表达。
