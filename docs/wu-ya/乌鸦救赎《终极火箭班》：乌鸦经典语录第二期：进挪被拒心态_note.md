# 乌鸦救赎《终极火箭班》课程笔记 02：进挪被拒心态 🧠

在本节课中，我们将学习在社交互动中，当“进挪”（即试探性肢体接触）被拒绝时，应如何调整心态。核心在于理解拒绝本身并不可怕，可怕的是因拒绝而产生的纠结心态会破坏后续互动。

## 核心心态：淡化目的性 🎯

上一节我们介绍了进挪的基本概念，本节中我们来看看执行时的心态关键。进行进挪时，目的性不能太强。你需要营造一种“我只是不经意碰了你一下”的感觉，或者让对方觉得，在你看来，进行进挪是一件很自然、无所谓的事情。

**核心心态公式**：`进挪行为 = 自然 + 不经意`

## 纠结心态的连锁反应 ⛓️

理解了正确心态后，我们必须警惕错误心态带来的危害。如果你的心态因被拒绝而受到影响、产生纠结，那么这种心态会直接影响你接下来的行为。

以下是心态影响行为的连锁反应步骤：

1.  **心态受影响**：进挪被拒后，内心开始纠结。
2.  **行为变怂**：纠结的心态导致后续行为变得畏缩、不自信。
3.  **框架被扭转**：你气场的减弱会改变双方互动的权力平衡（框架）。

## 关键规则：拒绝与纠结的博弈 ⚖️

现在，我们来深入探讨一个在社交动力学中至关重要但常被忽略的规则。这条规则关乎“他的拒绝”和“你的纠结”之间的博弈。

例如，你们之前聊得很好，对方也表现出兴趣，愿意与你互动、开玩笑。但在你尝试进挪被拒绝后，你因为对方的拒绝而自己陷入纠结。这种情况会彻底扭转你们之间的框架。很多学员都有过类似体验：进挪被拒后自己一纠结，后续的相处就突然变得不顺畅，感觉整个关系都变了味。

**核心规则**：`他的拒绝 + 你的纠结 = 框架被扭转（不利方向）`

## 总结与行动要点 ✅

本节课中，我们一起学习了应对进挪被拒的核心心态。

*   首要原则是保持行为自然，淡化目的性。
*   关键在于理解，拒绝本身不是问题，因拒绝而产生的纠结心态才是需要避免的。
*   务必记住“拒绝-纠结”博弈规则，避免因个人心态波动而破坏已有的良好互动框架。

**最终行动指令**：**千万不要去纠结。**
