# 乌鸦救赎《男神特殊时期集训营》第八课：如何面对情感中的失败 💔

在本节课中，我们将要学习如何面对和处理情感关系中的挫折与失败。我们将探讨分手后的心态调整、如何应对社交中的负面互动，并深入理解情感需求与个人价值的关系。掌握这些核心概念，将帮助你更从容地应对情感挑战，实现自我成长。

## 一、 面对分手：心态资源的积累与保护

上一节我们介绍了课程的整体框架，本节中我们来看看情感失败中最常见的情况：被分手。被分手后的感受，通常会经历一个急速下滑期。

男人被甩后，会先隐藏自己的需求，潜意识会抑制这种需求。但哪里有压迫，哪里就有反抗。大约在三四个月后，可能会爆发式地增加思念。当然，有些人可能很快就会出现悲伤难过的感觉。

进入那种悲伤状态后，几乎任何心态调整方法都会暂时失效。除非你拥有常年积累的心态资源。一个人的心态是否强悍，并不取决于某种单一的思维模式。

例如，你被女人甩了，告诉自己“没事，下一个人会更好”。但如果你的心态资源不够充沛，这句单薄的话无法将你从悲伤中拉出来。

心态很像存款。你需要通过经常冥想、建立预设价值观、及时解决负面情绪、进行自我沟通与疏导，以及不断去做自己害怕的事情（反脆弱）等方式来积攒心态资源。

当你真的面对重大难关时，你会发现没有那么难受，很容易就扛过去了。但如果你平时不进行心态资源的积累，当遇到重大心态问题时，就很难挺过来。

此外，保护心态资源同样重要。心态资源可能会以脉冲式的方式被释放。例如，今天和妹子聊天回应不好，进而对骂，这会非常伤害你的心态资源。愤怒是一种特别消耗心态资源的方式。

如果你能及时止损，避开损耗你心态资源的人，并通过冥想、预设价值观、自我沟通、反脆弱等方式不断训练自己，你就能获得更多“心态肌肉”。

当你真的面对分手或失恋时，你会发现可能只难受五分钟，然后该刷牙刷牙，该洗脸洗脸，完全不会干涉正常生活。人与人之间的不同，往往就在于心态资源储备的不对等。

不要相信世界上有什么“神奇一招”或一句话能让你瞬间从悲伤中走出来。那是不可能的。你在抖音上看到的心灵鸡汤之所以有时触动你，是因为你已经在相关方面积累了很多心态能量。

那句鸡汤只是起到了引爆和加成的作用。真正的功劳来自你长期的积累。如果你的整个人处于非常悲伤的状态，那种话是救不了你的。

**核心心态公式：**
`强大的心态 = 长期积累的心态资源 - 不必要的消耗`

如果分手后可以挽回，那就去挽回。如果真的很喜欢这个女孩，该挽回就去挽回，最后的结果并不重要。放不下就是放不下。

如果技术或能力不足，挽回不了，如何处理自己的心态问题呢？有一个非常简单的办法：忘记一个人最快的办法，是爱上一个新的人。

去找一个新的姑娘。你可以说这是放纵或滥情，但总比在伤心的失恋中颓废要好很多。这一招屡试不爽，新的恋爱会让你很快把前女友忘到九霄云外。

## 二、 人生的取舍：放下与获得的关系

在情感乃至人生的道路上，取舍是永恒的主题。我们这一辈子会错过很多东西。你会发现人生发展的脉络图，就是一个“舍与得”的过程。

砍掉大腿换汽车，卖掉汽车换飞机。人这一辈子都是这样走过来，有舍必有得。如果你想成为一个经历很多女人的魅力先生，你需要放下很多东西。

你可能需要放下对固定爱情的执着，甚至放弃传统的婚姻生活方式。如果你不放下这些，可能就无法拿起新的东西。这就像练《葵花宝典》，“欲练神功，必先自宫”。

这辈子要想往上走，必须砍掉点什么。而且，砍掉的部分，在当时看来一定是对你最重要的东西。每一次都是直击要害。

人跟人最后差的境界和段位，就差在有些人能够放下，有些人放不下。放不下的，就过一种固定的、或自认为安定的生活。但如果你有太多东西放不下，生活就会被拴住。

你会发现过了五六年、十几年，生活好像没有太大变化，因为你无法迎接新的可能。有时你要放下的东西是肉眼可见的。

例如，开公司想壮大，一开始你可能100%持股。但就像马云，股份已经稀释到很少。他不断卖掉股份，稀释给员工，通过一次次的放弃、割舍，才换来现在的规模。如果你不愿意放弃，可能公司也就止步于此了。

有时，你可能要花很长时间才能意识到，要想得到某样东西，需要放弃的是什么。例如健身，如果你真的想把健身这件事做好，最直接的影响就是“健身没朋友”。

你需要放弃很多让你感觉舒服的社交，放弃不健康的饮食。好的身材意味着你要放弃很多即时享乐的诱惑。

这引出了一个重要的原则：**非淡泊无以明志，非宁静无以致远**。

“淡泊”意味着简化生活，聚焦目标。就像真正的资本家，可能身价几十亿，但生活极其简单，将赚到的钱视为游戏道具和筹码，投入下一轮生产，而非用于享受消费。

“宁静”意味着要能沉得住气，守得住寂寞。生命由随机性和稳固性组成，但更重要的是那部分固化的东西。例如坚持读书，或花几年时间默默练习一项技能。

大家都在庸庸碌碌，忙于社交、交际、吃喝享受。而你花了几年时间专注做一件事，突然有一天出现在他们面前，他们会很惊讶你为什么走得这么远。

这就是“非宁静无以致远”。你要能够愿意花几年甚至几十年的时间，练就一身本事，只为有朝一日能够全负荷运转。能力的雕刻，都是以年为单位的。

五年一变都算是比较快的了。锻造一个绝世高手，没有十年时间根本锻造不出来。聊天、实战、理论、境界、心态，每一项都需要深耕，才能达到全面且精深的程度。

## 三、 应对社交挫折：保护心态资源与翻译情绪

在情感社交中，另一个容易崩溃的点是被怼（被言语攻击）的情况。处理这种情况，最核心的一点是学会保护自己的心态资源。

我以前是一个很爱聊闲气（为小事争执）的人。这不是因为我吵不过别人，相反，我吵架、讲道理、骂人的功夫都很深。但这样做的结果往往没什么用。

当然，并非所有的“怼”都没用。如果某人触碰到你的底线，并且具备一定影响力在抹黑你，那你肯定要回怼回去，这是在捍卫自己。我说的是生活中那些无关紧要的闲气。

例如，在跟女孩聊天时，刚好碰到一个心情不好的女人。你知道可以怼她，也怼得过，但怼完的结果肯定是互相拉黑，不可能有任何结果。这种情况，你必须训练自己避开这种无效的、没有意义的吵架。

直接不聊了，如果怕自己忍不住，直接把那个人删除或解除匹配就可以了。你必须学会保护自己的心态资源，因为这是人最根本的一种资源。

钱丢了可以再赚，面子丢了可以再争回来，唯独一旦心死了，就没得救了。历史上这种案例比比皆是，比如项羽，他的死亡在我看来纯粹是心态问题。

刘邦败了多少次都没事，项羽败了一次就自刎乌江，心态很糟糕。丁元英（电视剧《天道》角色）吃面时老板多收一块钱也不计较，因为他认为这是低效率的事情，不值得花费时间。

所以，丢任何东西都可以，但千万别把心态丢了。不管你是想建功立业、搞定更高级的妹子、自我提升，还是东山再起，最需要的首先不是钱，而是那股能支撑你站起来的信心和力量。

这完全是一种防守策略。不想让自己处于一种因为早上聊了点闲气，就生一天气，总想迁怒于人的状态。生活中一个芝麻大的事情，可能导致你一整天状态都不好。

所以，要懂得保护心态资源，降低心态资源流失的风险。这玩意很难攒，流失了很可惜。就像你去健身房练得很辛苦，晚上一顿火锅可能就白干了。好不容易攒的心态资源，因为几句争吵就消耗殆尽，很不值得。

**核心策略：**
`遇到无意义的挑衅 -> 识别为心态资源消耗 -> 主动避开或终止互动`

## 四、 搞不定的异性：价值、需求与自我提升

我们经常会碰到搞不定的妹子。很多时候，兄弟们会陷入一种错觉：搞不定一个妹子，是因为自身的价值不够高。其实有时候还真不是。

例如，我认识一个女孩，条件很普通，在深圳没房没车，只是经营一家小美容院，收入可能相当于高级打工者。但她很傲气。

后来我搞清楚了，她的价值不高，但追她的人价值高，所以她被“顶高”了。她因为工作原因，结识了很多高价值的客户（公司白领、富二代），这些客户的闺蜜圈又带她认识了很多高价值的男人。

所以，虽然她自身的生存段位可能不如你，但她身边不乏高价值的追求者，你或许只是她众多选择中排名靠后的一个，她自然对你傲气。

如果她是你的选择范围中最好的那个，或者你是她仅有的最好选择，她一定会对你很热情甚至倒贴。就像我身边总有很多美女，自然对一些分数不高但对我很热情的女孩没什么感觉。

所以，搞不定一个女人，有时并非因为你的价值比她低，而是因为你在她的追求者序列中不占优势。当然，你也会碰到自身价值就碾压你的女人，比如对方有车有房，而你还没有。

但你要明白一件事：**你错过的，永远是你该错过的。没有任何错过是偶然的，都是理所当然的。**

你自认为的错过，其实是内心潜意识认为应该搞定，但客观上没有搞定，从而感到遗憾。但如果你的价值足够高，那个“碗”根本摔不烂，掉到地上蹦几下还是完整的，你还能捡回来。

就像赵家（前女友）甩了我七八次，我每次都能捡回来。我没错过，她只是在通过删除微信、提分手的方式来验证我是否喜欢她，而不是真的想分手。

不该错过的，一定错不过。如果你认为错过了，那一定是该错过。所以，当你投入很多还是没有搞定一个女人，你就要明白这意味着什么。

这意味着你需要自我提升了。自我提升有三个维度：要么是价值不匹配，要么是聊天能力不匹配，要么是心态不匹配。反正你是错过了。

例如，我有一次在飞机头等舱认识一个女孩，聊得很好，后来知道她是某老总的女儿，非常优秀。但最后关系还是冷淡了，感觉错过了，很遗憾。

后来我把她的微信删掉了，删之前心里默默说了一声谢谢。谢谢她让我明白了，这个世界上还有这种能让我觉得自己应该继续努力提升的女人。

那时我刚创业成功，开始坐头等舱，觉得自己很牛逼，意气风发。但正是那个女人让我明白，我刚来到“千万的世界”，外面还有“亿的世界”、“上百亿的世界”，里面也有人在居住。

她让我明白，我还需要再提升很大一截。所以我这辈子都不会放弃自我发展和提升，因为总会山外有山，人外有人。

面对这种情况，你应该怎么做？你应该像我一样，对那个女人说一声谢谢。她就像一面镜子，照出你的不足。在你曾经的低维度里打遍天下无敌手，误以为自己天下无敌时，她出现在你的生命里给你上了一课。

男人的进步史一定跟耻辱感有关。你感到了羞耻、惭愧、自己的不足和弱项，这反而是强者的正常知觉。就像健身的人会越练越觉得自己小，总觉得缺块肉。但在别人眼里，你已经很壮了。

如果你内心感觉自己很弱小，这反而是强者的正常感觉，因为这会驱动你前进。如果你内心感觉自己无比强大，这可能说明你即将开始变得弱小，或者你正活在幻觉中。

## 五、 情绪的智慧：翻译负面情绪的密码

今天所讲的内容，都跟负面情绪有关：悲伤、愤怒、恐惧、不甘、焦虑等。我早年也和别人一样，讨厌所有负面感受，希望天天活在快乐中，不想要这些负面情绪。

但后来我慢慢明白了一个道理：**存在即合理**。如果一个东西存在，它一定有它的道理。从健身中，我深刻明白了情绪的其他作用。

你可以看看你的手臂。肱二头肌和肱三头肌是相互对立的肌肉。当你弯曲手臂，肱二头肌鼓起时，肱三头肌就会被拉直。这么做的目的是防止你过度弯曲，并且为再次伸展做准备。

当你伸直手臂时，肱三头肌发力鼓起，肱二头肌被拉直，防止你过度伸展。你的核心肌群、胸部、腹部和背部的关系也是如此。它们相互对立而统一，彼此存在都有其价值和依据。

在自然界中，任何系统都存在“油门”和“刹车”，不是只有油门到底才是最好的。一辆车开到时速300公里，没有刹车就很容易报废。

情绪系统也一样。为什么你有喜怒悲恐惊？它们都有其意义。快乐的情绪往往是为了奖励你，给你正反馈，让你继续努力。我们今天主要看看负面情绪。

我最先领悟的是关于愤怒的哲学思考。早年我在公司可以称之为“暴君”，看到员工没按我的意思办，上去就是一顿骂。但花了三年时间，我发现没什么用，该发生的事还是会发生。

比如，我因为员工穿拖鞋上班而生气。后来我尝试用制度解决：把“不许穿拖鞋”写进公司条例。但还是没用。后来我发现，光有制度不行，还需要培训。

于是，我改造了模式，让每个新员工在前七天接受公司管理制度培训，其中包含不许穿拖鞋的规定。后来，这个问题就消失了。

这是我第一次领悟到：**所有的情绪问题都是模式问题**。你没有找到一个好的方法去解决，情绪是在通过一种感觉提醒你：你有一件事没有解决掉，或者解决得并不高明。

你必须找到更好的模式来解决，让它永远不再发生。我一直践行这个标准，利用自己的情绪反馈，找到问题点，然后思考在更高维度上如何管理，而不是直接发泄情绪。

所以，我的公司运行非常健康，模式做得很好，甚至我可以两三个月不去公司，公司也没太大问题。这就是模式的力量，也是通过情绪领悟到的东西。

我开始明白，所有的情绪都像一个语言能力很差的小孩。它没办法直接给我一个逻辑告诉我哪里有问题，只能通过哭、喊、骂来告诉我“这有问题”。

我以前不懂翻译这种“情绪语言”，只是感受到情绪后就立刻发泄出去。但现在我懂得静下心来翻译：为什么我现在很愤怒？为什么会发生这种情况？我应该怎么解决？有没有更好的模式？

我几乎每次都这样思考问题，包括产生的悲伤感、被背叛感。每一种感觉都会让我增强某种模式，然后我变得越来越强大。

**核心方法：**
`产生强烈负面情绪 -> 暂停发泄 -> 自我提问（为什么？是什么问题？） -> 寻找更高维度的解决方案（模式）`

分解情绪，翻译它到底想告诉你什么。它一定是想告诉你一些东西，只是表达方式比较抽象。你的大脑就是用来翻译情绪的，而不是用来发泄情绪的。

一个很简单的例子：当你搞不定一个女生时，你会悲伤、蹉跎、焦虑、自暴自弃。如果你能把这些情绪翻译成：“啊，谢谢你让我看到了自己的弱小，我需要自我提升。” 如果你能独立做到这一点，非常厉害。

这表示你具备了跟情绪沟通的智慧。这种能力让你拥有了一个无比强大的帮手——你的潜意识。我们的潜意识比主观意识强大数百万倍，能感知到更多问题，但它的“bug”是“茶壶煮饺子——有货倒不出”。

当你掌握了翻译情绪的方法后，你就有了一个随时在身边、精准知道你所有问题、且比你想得更深远的智囊团。它只是在等你跟它沟通。

如何锻炼这一点？不断去观察。每一种情绪，尽可能去翻译它到底代表什么问题。比如同事得罪了你，你产生了情绪，就去翻译：问题到底是什么？答案又是什么？你的潜意识都会告诉你。

这就像照顾一个全身瘫痪但非常聪明的老头。你猜他的想法，猜对了，他会通过眨眼或其他方式给你反馈。跟情绪沟通也是如此，当你猜对的时候，你会感知到一种愉快的情绪，一种“猜对了”的幸福感觉。

当然，不是所有情绪都需要翻译。比如雨天听了一首伤感的歌，想起过去，这种情绪没必要深究。但如果在工作生活中，某个问题老是困扰你，没完没了地出现，你经常为此产生某种情绪，你就应该去翻译它。

一次翻译不出来，就两次、三次。当你真的翻译成功几次，并且它真的给出了问题和答案时，你就会知道这种力量有多强大。你会感觉自己脑子里住了一个“诸葛亮”，什么都知道，只是说不出来。

随着默契度提高，你甚至会产生一种“他心通”的能力。如果你有微表情或行为心理学基础，这种能力可以外延。例如，跟人沟通时，我死盯着对方的眼睛看，因为情绪变化从眼神开始。

我无法猜到对方脑子里具体在想什么，但我可以通过对方对某件事的情绪反馈，猜到他的态度。比如我给员工画饼，说今年要造就几个百万富翁，有人撇嘴，我就知道他对这个说法持否定态度。

然后我会换一种方式说，看到他露出兴奋表情，我就知道说到点子上了。这就是“他心通”：通过对方语言反馈出的情绪，猜测他的想法。人的情绪往往是真实的，尤其是通过表情和语气读出的情绪。

当你说的话和你的情绪不相符，别人就能知道你的话是假的。这种能力是可以练出来的。而且，你觉知自己情绪的能力越强，觉知他人情绪的能力也会越强。你要先拿自己练手。

例如，为什么我能猜到有人“聊天没动力”？因为我翻译过自己的这个情绪。我也出现过聊着聊着不想聊的情况，后来自己翻译发现，是因为最近聊的妹子都不太好看。如果要给我个八分妹，我能聊到飞起。

所以，若想知人，先要知己。因为人性不变，规律就那些点，剩下的现象都是围绕核心产生的变化。

## 六、 核心认知：需求创造价值

最后，我们来讲一个最核心、最具有指导性意义的概念：**价值与需求**。

有些人会被常规价值所迷惑，认为车、房、钱、美貌、肌肉等就是价值。其实这种思维是被社会上固定的价值排序所影响。如果你研究过人类大历史，会发现最有价值的东西一直在变，以前可能是好的马匹或马车。

并不是价值决定需求，而是**需求创造价值**。很多人以为我有高价值（有车有钱有房），所以我就有高价值。这种思维说不上错，但很肤浅。

例如，我们的课程。并不是因为我有一套泡妞技术，它就像金子一样发光，所有人看到就产生需求来报课。**如果你不想学习泡妞，我这个东西一文不值。是你的需求创造了我的价值。**

再比如，如果全世界的人都想要苹果，而你是全世界唯一或少数手里有苹果的人，你的苹果才会有价值，紧接着你也会变得有价值。**是人们的需求让你变成了金子，而不是金子自己会发光。** 金子本身不会发光，它只是反射光。

口罩在平时没人关注，为什么疫情期间价格飞涨？是因为人们产生了大量的口罩需求。**是需求创造了价值，而非价值创造需求。**

一定要搞通这一点。普通人的理解是“价值创造需求”，高手的理解是“需求创造价值”。如果我可以创造、制造或摸索到某种需求，那么意味着我已经掌握了价值本身。
