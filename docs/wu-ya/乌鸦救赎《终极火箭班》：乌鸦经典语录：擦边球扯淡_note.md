# 乌鸦救赎《终极火箭班》课程笔记 01：核心阶段与吸引 🚀

在本节课中，我们将学习乌鸦救赎《终极火箭班》中关于恋爱关系发展的核心框架。课程将恋爱关系划分为几个关键阶段，并重点讲解初始的“吸引阶段”。理解这些阶段是建立有效互动的基础。

## 关系发展的核心阶段

课程明确指出，恋爱关系的发展可以分为几个连续的阶段。理解这些阶段有助于我们明确在不同时期应该采取的策略。

以下是关系发展的三个主要阶段：

1.  **吸引阶段**
2.  **舒适感阶段**
3.  **诱惑阶段**

## 聚焦：吸引阶段

上一节我们介绍了关系的整体框架，本节中我们重点来看看第一个阶段——**吸引阶段**。这个阶段的核心目标是：**如何能够吸引到异性**。

课程强调，当两人开始建立联系、关系逐渐缓和时，工作重点并不仅仅是维持吸引。此时，需要引导对方为关系进行“投资”。

**关系投资** 是指让对方为两人关系的提升做出贡献。这可以理解为一种情感或行动上的承诺。

## 课程总结

本节课中，我们一起学习了乌鸦救赎《终极火箭班》关于恋爱关系阶段划分的核心概念。我们明确了关系发展会经历 **吸引、舒适感、诱惑** 三个阶段，并深入探讨了 **吸引阶段** 的目标不仅是建立吸引，更要引导对方进行 **关系投资**，为关系的深入发展奠定基础。
