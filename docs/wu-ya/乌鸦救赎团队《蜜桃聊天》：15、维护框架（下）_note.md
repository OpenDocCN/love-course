# 乌鸦救赎团队《蜜桃聊天》课程15：维护框架（下）🔧

在本节课中，我们将要学习在原则性问题上维护聊天框架的具体方法。上一节我们介绍了维护框架的重要性，本节中我们来看看四种实用的维护技巧。

## 核心原则：在原则性问题上必须维护框架

在涉及原则的问题上，框架必须得到维护。以下是四种有效的方法。

### 1. 方法一：平衡

当对方提出要求时，避免单方面付出，应寻求双方付出的平衡。

**示例场景**：女生常要求男生帮忙，如“你帮我拿个快递吧”、“你帮我修个电脑吧”。若全部照做，可能沦为免费劳动力；若直接拒绝，则显得不够绅士。

**正确做法**：提出平衡条件。
> **公式**：同意请求 + 提出对等交换条件

**示例回复**：“好啊，那你得请我吃好吃的。”
这种方法确保了双方都有付出，而非单方面索取。

### 2. 方法二：调侃

当对方用调侃或略带冒犯的言语测试你时，可以用幽默的调侃进行还击，以维持轻松氛围并守住框架。

**示例场景**：女生说：“哎你这个人挺有意思的，就是长得不咋地。”

**错误反应**：生气或陷入关于外貌的争论。

**正确做法**：用幽默调侃回应。
> **代码式描述**：`回应 = 夸张反应 + 幽默元素`

**示例回复**：“啊你敢说我丑，我报警了。”
这种方式既能化解尴尬，又能保持框架，使聊天气氛缓和。

### 3. 方法三：无视

对于在关系未到位时便提出越界要求的人，最佳策略是无视或切断联系。

**示例场景**：刚认识或还不熟的女生提出：“你给我发个红包吧”、“你帮我叫个外卖吧”。

**核心判断**：这类要求表明对方缺乏诚意，仅将你视为工具（如ATM），且不愿提供任何情绪价值。

**正确做法**：选择无视或直接拉黑。这本身就是一种强有力的框架维护。

### 4. 方法四：表达不良感受

当对方的言语让你感到不适时，应直接、大方地表达你的不良感受，明确告知对方你的底线。

**重要性**：若不及时告知，对方可能会变本加厉。这是许多人的血泪教训。

**示例场景**：女生说：“你能告诉我你为什么这么傻逼吗？”或类似令人不快的言语。

**正确做法**：严肃但理性地表达感受。
> **公式**：表明友好意愿 + 指出问题 + 表达感受 + 设定边界

**示例回复**：“虽然我很乐意跟你开玩笑，但我认为人与人之间起码的尊重是要有的。我不喜欢这样。”
你无需担心得罪对方，因为是对方先出言不逊。直接告知你的不开心，能让对方明确知道你的尺度。

---

本节课中我们一起学习了在原则性问题上维护聊天框架的四种核心方法：**平衡**、**调侃**、**无视**与**表达不良感受**。记住，在原则问题上坚守框架至关重要。希望你能灵活运用这些方法，建立健康、平等的互动关系。
