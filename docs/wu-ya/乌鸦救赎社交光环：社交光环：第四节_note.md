# 乌鸦救赎社交光环教程：第四节 🏢 公司里的故事与社交光环的力量

在本节课中，我们将通过一个真实的故事，深入理解“社交光环”如何在现实社交场景中自然生效，并学习如何低成本、高效地展示自身价值。

上一节我们探讨了社交光环的基本概念，本节中我们来看看一个具体的应用实例。

---

我曾多次拜访一位朋友的公司。这位朋友在广州，他的公司主营营销业务。营销工作常常需要运用客户心理学，因此与心理学领域关联紧密。营销与心理学可谓一脉相承，所以我们之间的交流比较多。

有一次，他的公司正在策划一场微信社群营销活动。他将我邀请进了活动的一个核心群。

当时，群里正在讨论一些心理学观点。我觉得这些讨论很合我的胃口，便展示了一些自己的见解和理论。我一讲起理论来，有时会收不住，讲的内容就比较多。

我延伸出了很多观点。这个时候，我在整个群里吸引到了很多人的注意。要知道，做营销不分男女客户，大家都是为了赚钱。因此，很多女性群成员开始关注我，并在群里向我提问。

她们与我交流心理学观点，最后话题甚至延伸到了情感领域，聊了很多内容。当时，有一个女孩反应特别积极，对我表现出极大的兴趣。由于我的微信设置了禁止通过群聊添加，她就在群里一再要求我开放权限，想要添加我。

最后，因为群里很多人都在要求，且聊得也很开心，我不好不给面子，就允许他们添加了。

那个女孩就这样加到了我的微信。加上之后，她没事就找我聊天，交流心理学和营销上的事情。聊着聊着，话题总是偏向情感领域，无事献殷勤，目的不言而喻。

我其实不太愿意接触这类话题。每次我讲完一些理论、概念或故事，我的那位朋友都会非常捧场。他总是出来为我做价值背书，比如“孟宏的心理学功底很深厚”、“他自己开公司很厉害”等等。

有一次，我去广州，正好去了他的公司，碰巧遇到了那天在群里要加我微信的那个女孩。

当时，我们先交流了一些关于营销或心理学的观点，就是普通聊天。我的那位做营销的朋友是个非常捧场的人，他不断地在帮我做价值背书，类似于“唱白脸”那种方式。实际上整个过程中我没说太多话，一直是他主导。

因为我们之前沟通很多，他就用他的口吻来讲述我的理论，并且说“这是孟宏讲过的……”，非常推崇我。上午交流完后，我们就一起去吃饭了。

吃饭时，自然不会再谈业务上的事情，大家放松下来闲聊。我注意到，那个女孩对我的兴趣指标非常多，一直在问我各种问题。但其实整个过程中，我并没有做太多特别的事情。

回来之后，她开始在微信上主动找我，没事就会问我一些心理学问题，频率比以前更高了，而且还问了很多私人问题。到最后，她甚至直接表白了。但我其实并没有给予任何明确的反馈。

直到现在，她还处于一种类似在朦胧追求我的状态。实际上，在整个过程中，我没有对她使用任何技巧。

所以说，社交光环是一种低成本且效果非常好的展示价值的手段。如果你学会这招，起码能在学习如何吸引异性这件事上，扫清四到五年的弯路。

很多时候，你只需要一张好照片。只是现在的照片传递的不再是传统的高价值信息，而是**高社交光环的信息**。同时，你也需要一些好的朋友。

说得直白一点，你需要有能一起“装逼”的伙伴。我们需要的是“逼友”——当然这个描述可能不太准确——但你需要一帮好兄弟。你需要去经营自己的兄弟关系。

以前我们可能会认为，所有男性都是来和我们“抢肉吃”、会把我们的妹子抢走的竞争对手，所以必须把他们全都赶走。但实际上，你现在不必这样思考问题。

你会发现，你的男性同胞、男性朋友们，可以变成你非常棒的伙伴和“僚机”，帮你呈现价值。这可能需要一些沟通和训练，但比起一个人单打独斗，效率要高出太多倍。

你会发现，社交光环的操作方法，无论是从学习的上手难易度，还是其门槛，都比学习聊天技巧或展示硬性价值要友好得多。它更适合新手。

但是，为什么很多人在学习吸引力时，会选择像“聊天”这类技巧呢？因为人在学习时都存在一种“选择性吸收”的问题。

**选择性吸收**指的是：很多时候你读书是因为遇到了人生困惑，想去寻找答案。于是大脑会设定一个问题，你只会在书里看到与这个问题相关的内容。而与问题无关的内容，你会选择忽略。

可能我以前也讲过关于社交光环的内容，但很多人没有看到。为什么？因为你的大脑认为这是你不需要的东西，所以选择忽略掉了。你会发现人是一种很偏执的动物，只会选择去吸收和学习自认为对自己有帮助的东西。

我相信有很多兄弟跟我学习了很长时间，我可以用我的人格担保：社交光环这个东西，从某种程度上说，它虽然不算是“捷径”，但确实能起到捷径的效果。

它能让你的学习速度倍增，直接帮你省下很多弯路。最关键的是，一旦你开始选择使用社交光环这样的技巧，你就必须去结交一帮好兄弟。

你会发现，你的人际交往能力也会得到改善。你会开始思考兄弟们的感受，思考如何获得一群朋友，而不是天天只想着如何搞定一个女人。实际上，那种想法太单一，也有点“不上道”。

**本节课总结**

本节课中，我们一起学习了一个通过“社交光环”自然吸引异性的完整案例。我们了解到，社交光环的核心在于通过**你所处的社交环境**和**他人对你的评价**来间接展示高价值，这远比个人直接展示更为有效和轻松。关键在于经营高质量的社交圈，让朋友成为你的价值背书者。这种方法门槛低、易上手，并能同步改善你的人际关系，是新手快速提升吸引力的高效路径。
