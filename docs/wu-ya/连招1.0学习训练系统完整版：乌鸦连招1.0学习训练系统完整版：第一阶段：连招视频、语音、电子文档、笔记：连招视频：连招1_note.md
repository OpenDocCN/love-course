# 连招1.0学习训练系统完整版：第一阶段：自我引导 🧭

在本节课中，我们将要学习“连招”系统的核心起点——自我引导。我们将深入探讨其定义、核心原理，以及如何通过具体的方法将其应用到你的生活和互动中，从而成为一个更具吸引力的个体。

## 课程概述

自我引导是“连招”体系的第一块基石。它并非简单的聊天技巧，而是一种更深层的能力：**牵引自己的生活向更好方向发展的能力**。本节课将为你拆解自我引导的本质，并指导你如何从内到外构建这种能力。

## 什么是自我引导？

上一节我们提到了自我引导是连招的起点。本节中我们来看看它的准确定义。

自我引导的定义，并非指一个人能完全了解自己。它指的是一个人拥有**自我牵引的能力**，能够主动推动自己向前进步。你会发现，自我引导更偏向于生活化的建设：即你如何将自己的生活经营得越来越好。

许多知名的PUA（搭讪艺术家）不愿深入谈论这一点。例如，有些高级课程命名为“出神入化”，其核心内容正是心态与生活建设。我们都明白，当你的生活质量提升后，你吸引异性的难度会自然下降。

当你自身状态很好时，比如穿着得体、举止自信，你出门就会吸引他人的目光。当别人被你的价值吸引时，你开启对话所受到的阻力会小很多。他们会觉得与你交流充满可能性，能提升他们的社交圈或生存状态。与你相处是有益的。

因此，我们所说的自我引导，并非聊天时引导话题到自己身上，而是指一个人**拥有将自己生活变得越来越好的牵引力**。这种能力是核心。有些人打工，有些人当老板，区别往往就在于此。没有目标和想法的人需要别人带领，而能带领别人的人之所以有吸引力，正是因为他们拥有这种将生活导向正轨的能力。

自我引导的核心就在于此。它有时看似像成功学，但成功学最初是从催眠领域发展出来的，并非完全是糟粕，只是有时过于功利。如今，泡学也出现了类似问题，变得急功近利。但我们的方向是：当你自己成为一个足够优秀的男人时，你和谁相处都会很顺利。

我们的目标是把自己变成一个“万能扳手”，能够应对各种类型的人。所以，自我引导最核心的并非聊天内容。很多人纠结于“不知道聊什么”，但重点不在这里。学习话术有帮助，但如果你生活本身不精彩，你每天大部分时间在床上、在无聊的办公室或电脑前度过，你确实没有内容可聊。

吸引的本质是什么？连招中有句话：**吸引便是缘起**。这句话有一定道理。我们不应带有仇富心理。早期有些理论提倡“穷逼把妹”，但如果我们有能力让自己过得更好，为什么不呢？泡学领域后来也逐渐转向“打造一个优秀的男人”，因为当你生活真正变好时，把妹会变得很简单。

## 自我引导的核心：建设生活

上一节我们明确了自我引导是关于生活建设的能力。本节中我们来看看为什么这是核心，以及如何理解它。

关键在于，你要先把话术这些东西放下，真正把自己的生活搞好。这不是空谈。例如，我认识一位纹身师，他艺术气息浓厚，生活状态很好，月收入一两万，在市中心有自己的工作室。他并没有学过任何理论，但他自然散发的状态，恰恰符合理论描述：他总能不知不觉地将话题锁定在自己身上，幽默地展示自己的生活。他生活得很OK，因此总能吸引到别人，包括我。

他达到了泡妞的四个境界中的“泡生活”和“泡自己”。他真的搞定了自己的生活。与他相处，你能感受到他身上的能量，他总是有新奇的想法，并能解释清楚。他始终能把话题引导到自己擅长和热爱的领域。

我们在自我引导环节最关键的是：**你真的需要先把自己的生活搞定**。价值吸引人，但价值只是能量的外显。当你对生活有看法、有想法，并按照自己的想法去建设生活时，价值会从你生活的各个角落自然散发出来，无需刻意装逼。

过去学习的一些理论过于极端，只集中在“价值展示”，导致很多人活得很累，不停装逼。但真正的高价值状态是：你不需要吹牛，别人通过你的言行举止就能看出，你就是那样一个人，你的生活就是由高格局、高格调、高品质组成的。

有句话叫“骗人先骗己”，说得低俗但有理：你需要自己先进入那种生活状态。你不需要总想着如何展示价值才能吸引别人。你要做的是提升自己的生活品质。

我们后期会制作一张表格，让你每天记录参与了哪些生活体验。你需要在墙上贴目标纸，完成一些小目标。例如，没去过星巴克，就去一次；没去过西餐厅、密室逃脱，就去体验。很多人根本没玩过这些。

首先要想明白：我们活着绝非仅仅为了有钱或出人头地。如果这样定位人生，你一辈子都是活给别人看的。“出人头地”意味着要向别人证明自己。如果你的生活真的很OK，你随手拍一张在西餐厅或旅游地的照片，那是你生活的自然流露，能量自然流淌，就能吸引人。但很多人为了展示而展示，甚至模仿他人的照片，这非常不好。

## 关于“没钱”的疑虑与信念

上一节我们谈到建设生活，可能有人会感到心虚：老师你说得轻松，但我没钱啊。本节中我们来专门探讨这个普遍的疑虑。

很多人会有这种感觉：你说要进入高品质生活状态，心态、思维、语言、行为自然会改变，但我一个月生活费就一两千，怎么办？

有时，真的是思维格局受限了。我举个例子：我们团队有个叫安小妖的女孩。她身上有种随遇而安、说走就走的浪漫情怀。她没钱，真的没钱。她曾去西部支教一年，没有工资，只有饭钱，每天走很远的路，在大山里坚持了一年。她还去过西藏，后来在丽江餐厅打工赚旅费。这个人活得非常写意，非常有吸引力。

不是做不到，而是想不到。当你发自内心想做一件事时，你会发现世界会配合你，整个宇宙会营造条件。钱不是绝对问题。当你真心渴望时，问题总能找到解决灵感，或有人来帮你。这是一种同频共振。

我本人也是逆袭而来，从收入几千到现在带领团队。凭的不是体力或运气，而是那股**一定要活成某种人**的信念。当你真正那样定位自己时，想法和思维同步，你必然会走上那条路。这无需怀疑。

信念课我们会在周日专门讲（阳神流氓星）。当你内心有这套东西运行时，就不要再用算计的方式生活，而是跟随那种“真的想干什么，一切都会让路”的状态。这就像蹦极，纵身一跃后，发现一切都很平常。你必须要有这种精神。

我看过一个电影，里面一个很有魅力的男人有个本事：三个水杯下有一个放钉子，女孩摆好后，他能坚定地一掌拍对。女孩问秘诀，他说：**不要在乎选哪个，因为世上没有完全正确的方法，看你要去哪。你要明白自己要去哪，正确的方法自然会出现。**

每个人的成功方式都不一样，我的东西你只能借鉴，但你要活出自己的路。我知道你们心里压抑了很多想法没有实现，导致人生感觉悲催，产生强烈的自我否定。我们有个“第一念练习”：想到什么就去做什么，没有所谓的正确模板，干就行了。

即便你谨小慎微或随心所欲地度过一生，你会发现一路上经历的困难、挫折和否定是一样多的。无论如何活，世界给你的反馈总量是差不多的。所以，既然如此，为什么不跟着自己的心走呢？

在我20多岁前，我总想把每件事做对，寻找平衡生活与社交的方法。但后来我发现，没有那种方法。我总结出一句话：**不管你做什么，总有一半人认为你对，一半人认为你错。与其如此，不如跟着自己的心走。**

## 自我引导的话术应用

上一节我们从心态和信念层面探讨了自我引导。本节中我们回到具体互动，看看如何在话术中运用自我引导。

自我引导在话术上也能解释通，有一语双关的作用。首先，你要把话题锁定在自己身上。为什么这能产生吸引？因为我在聊我自己感受到的东西，话题始终扣准在自己身上。

最简单的比喻：互动中，谁是电影，谁是观众？观众坐在椅子上看电影释放的能量、看法和思想，只能被动接收、被洗脑。我们的很多概念都是被别人洗出来的。我们要做的是**影响别人的人**，而不是被影响的人。

在互动中，我不是让你极端地只聊自己，那会变成怪胎。而是在和女人聊天时，更多地把话题**引导到自己身上**。你才是吸引方，是释放能量的一方。

如果你一开始就问对方：你做什么？多大？今天干嘛？找过几个男朋友？这就是在查户口。一开始不能这样。因为当你查户口时，你在让对方释放能量。伪自信的产生方式，就是通过释放能量并获得外界良好回应（IOI）来维持。如果你总问问题，就是在给对方贡献能量，助长她的自信和自我膨胀，她会越来越不把你当回事。

反之，如果一个妹子问你很多问题，你也会感觉很好，觉得对方很好聊。所以，在聊天过程中，我们不能给女人贡献太多能量。能量就是IOI（兴趣指标）。伪自信靠吞噬外界IOI维持小我的存在感。比如你进房间打招呼没人理，你会很难受。核心自信则不依赖这些，它自我供养，只向自己证明，因此受外界影响很小。

首先要明白，自我引导要把话题扣在自己身上，这是基础。那么，然后要嵌入什么东西？

有一个比喻：一根稻草不值钱。但在沿海城市，每年卖成千上万吨螃蟹，用稻草绳绑住螃蟹，稻草就能卖出螃蟹的价钱。我们人就像一根稻草，为什么有些人值钱，有些人不值钱？区别不在于你自己，而在于**你绑定了什么**。稻草和螃蟹绑定，就是螃蟹的价钱。

在自我引导中，你就是那根稻草，你要去绑定高价值的东西。例如，我下午饿了去西餐厅吃意面、喝咖啡。我本想拍张丰盛的展示面，结果吃完了才拍，只能幽默地说“猜猜我今晚吃了什么”。这嵌入了幽默、意面、咖啡、西餐厅等元素。在社会价值认同中，这些属于高价值。我（稻草）与这些高价值事物绑定，我就实现了“升值”。

再教你一个技巧：当女人问你“你前女友是做什么的？”，你可以说“我上个前女友是车模/模特”，或者“是个小明星/歌剧演员”。当你去吸引现在这个妞时，这会变得很简单。这是一种“作弊”方法。

这类似于职场：如果你在百度等大公司做过高管，再去小公司面试，对方听到你的经历，会觉得庙小配不上你，无需你多展示，就认为你很OK。同样，你说前女友是车模、演员，会立刻启动女人的**反向预选机制**，她会觉得你做她男朋友很OK，没有逻辑判断，直接感觉行。这是一个最近发现的技术点，很好用。

## 自我引导的具体元素与技巧

上一节我们介绍了话术中绑定高价值的概念。本节中我们系统性地拆解自我引导话术的具体构成元素。

自我引导分为两部分：**自我生活引导**和**自己擅长领域的引导**。男人擅长的领域（如心理学、台球）通常比较抽象逻辑化，直接讲可能枯燥。所以不要一上来就聊擅长领域，要先进行**生活上的引导**，聊聊你的生活起居。

以下是自我引导话术中需要嵌入的三个核心元素：

### 1. 幽默（骄傲风趣法）

首先，我们需要幽默。笑是统一情绪的第一步。如果你想和女人达到精神连接，情绪必须先统一。那么，该如何开玩笑？

这里涉及“行为基准线”理论。当对方不熟悉你时，你最好不要开打压式玩笑，她可能认为你在攻击她。同时，也不能开自嘲式玩笑，那会让自己显得Low，是在取悦别人、牺牲形象。

那么，我们该如何开玩笑？答案是：**骄傲风趣法**。即开自己的玩笑，但要以一种骄傲、夸张的方式。

例如：
- 女：“感觉你挺帅的。”
- 你：“是啊，没错，这么多年依然这么潇洒。”
- 女：“你好自恋啊。”
- 你：“没错，如果自恋可以当饭吃，我想我可以养活半个中国。”

她知道你在开玩笑。这满足了两个点：第一，她知道是玩笑，笑统一了情绪；第二，幽默本身代表很强的生存能力。只有当一个人处理信息从容不迫、能力过剩时，才能展现出幽默。幽默就像手表、戒指，是你生存能力富裕后的附加值。

### 2. 高价值展示（绑定高价值元素）

第二个元素是高价值展示。即前面说的“稻草绑定螃蟹”。你在进行自我生活引导时，要嵌入高价值元素。例如：我今天去了游乐场、北京、上海、丽江，并拍了展示面。

**图片（展示面）是最直接的**，因为文字传达信息慢且不易记忆。图像记忆更直观。朋友圈展示面就有这个功能，直接展现你的生活。你站在丽江，穿着得体，一张图片就说明了你是什么样的人，非常快速。

你需要与社会价值观认同的高价值元素（名牌、优质体验）绑定，并以幽默的方式表达。

### 3. 断面（说一半藏一半）

第三个元素是断面。即**说一半，藏一半**，培养女人对你发问的习惯。

例如：
- “我今天出门走了很远，本来想去西餐厅，结果发现自己穿了个像睡衣的内裤就出来了。都快进餐厅点餐了……”（到此停下）
- 女人会问：“后来呢？怎么样了？”
- 你再说：“我就穿了个大裤衩去餐厅了呀。”

或者说：
- 女：“你是做什么的？”
- 你：“修理工。”
- 女：“修什么的？”
- 你：“修人的。”
- 女：“修人？那是大夫啊？”
- 你：“是啊，在妇产科，全国著名的妇产科大夫。”

这种方式有问有答，很有意思，而且很容易引爆对方的IOI（兴趣指标）。你要培养她释放IOI的习惯。当累积到一定程度（如书中所说的六个黄金IOI），就可以进行下一步测试。

**关键在于展现的形式，而非内容**。很多时候，吸引人的不是你说话的内容，而是你说话时的状态和方式。内容反而是辅助。

## 自我引导的终极支撑：能量与生活

上一节我们列出了自我引导话术的三个技巧元素。本节中我们来探讨这一切的底层支撑是什么，以及如何获得它。

所有这些技巧的核心支撑点，是你的**真实生活**。你必须背后有一个非常好的生活状态作为支撑。你确实进入了那种生活状态。你需要给自己定一些小目标，去体验各种状态，把自己带进去，然后和女人聊。你只是把自己的生活聊出来，吸引就自然产生了。

吸引就是这么简单。当你足够爱自己，让能量流向自己，想方设法让自己活得更好、更舒服、更像一个男人时，你的生活就开始有价值、有质量。这时，**吸引便是缘起**。

以前你可能处于消极状态，感觉自己什么都不行，把自己封闭在小圈子里，打游戏、吃泡面，日复一日。这种状态无法展现强大的生存能力。只有当你动心起念，想让跟了自己20多年的肉体过上幸福生活时，你就会想办法。

例如，定制西装3000块很贵，但你可以攒两个月工资去体验。当你穿上定制西装，所展现的能量会自然吸引别人。这里还有一个秘诀：**这个过程可以滚雪球**。

当你穿上5000块的定制西装，那些穿得起定制西装的人与你沟通时不会拒绝你，因为他们觉得你们价值在同一层面。与这些人交流，你会获得无数可能性：投资项目、工作机会、社交关系，甚至得到女神。你的生活像滚雪球一样越滚越大。

最初的本钱很低，就是**爱自己**。没钱时，可以把自己收拾干净；攒点钱，投资好的衣服。别人看到你很棒，也愿意接近你。就像天使投资人，看你现在做得不错，才愿意投钱。我们的本钱就是爱自己，就是不想再吃泡面，想吃意面；就是活这么大没穿过定制西装，想体验一下。这是我吸引天使投资人过程中的一小步，一点一点滚雪球滚到现在的。

自我引导，其实就是展现高价值（DHV）。但高价值是什么？我来解释一下这个概念。

意念是一种能量，是你的意志力。如果我要拿起这张纸，我首先得“想”，这个“想”是能量。而“拿起纸”这个动作是外显。高价值就是通过一系列动作展现出来的**证据**，证明你确实处于一种很OK的状态。

核心是能量。但过去很多人只教你“拿纸”这个动作（外显），却没教你为什么拿、怎么拿，以及如何进入那种状态。比如，高价值的人坐姿自然挺直，不需要刻意强调。如果你内心能量颓废，对生活无望，你的状态就是驼背的。当你对生活充满希望时，你说话会斩钉截铁，动作有力。这种状态凭借的是内在能量。

关键是你要爱自己，要让生活变得更好，要有正向的三观。能量到了，外显自然就到了。

最后，想想为什么有人有钱有人没钱？前两天我悟到一句话：PUA（把妹达人）和英雄联盟玩家，本质都是“屌丝”，但注意力不同。

- **英雄联盟屌丝**：把钱花在买英雄、皮肤上；学习攻略、看电竞导师视频；下的功夫是练习微操、抢人头、秀战绩。
- **PUA屌丝**：把钱花在自己身上，买衬衫、裤子；学习如何生活、把妹；下的功夫是建设生活、拍展示面、秀生活。

两个人下的功夫一样多。最后，在各自领域，一个游戏玩得好，一个把妹能力强。区别就在于**注意力在哪里，能量就在哪里显现**。你们都是“屌丝”，只是要决定当哪种“屌丝”。走的路不同。

生活本身也是一个游戏，而且你逃不掉。这个游戏一样可以练习出很高的“微操”，只是它的规则更复杂，缺少新手指引。我们选择玩游戏是因为它简单，规则明确。但生活的技能，需要你自己去磨练出来。
