# 乌鸦救赎《终极火箭班》第X课：是非观 🧠

在本节课中，我们将学习乌鸦关于“是非观”的核心论述。我们将探讨如何跳出集体意识的束缚，建立以自我利益为核心的判断标准。

---

上一节我们介绍了课程主题，本节中我们来详细拆解乌鸦的论述。其核心观点是：判断对错不应依赖外界共识，而应基于对自身是否有利。

以下是核心要点的分步解析：

1.  **错误的本质**
    对你不利的事情就是错的。判断标准应内化，而非外求。

2.  **警惕集体意识陷阱**
    不要总是在集体意识中寻找对错标准。例如，妻子出轨被认为是错的，往往是因为社会共识如此。

3.  **缺乏共识的假设**
    假如没有社交媒体和公众人物不断强调“出轨是错的”这一观念，人们可能无法立刻意识到这是对自己不利的错误行为。

4.  **荒谬的后果**
    在这种情况下，一个人甚至可能只会温和地请求妻子“以后不要这样了”，而无法坚定地维护自身利益。这种局面是荒诞且痛苦的。

5.  **确立自己的判断准则**
    因此，定夺对错的方式应是：**对你不利即是错，对你有利即是对**。你需要建立清晰的自我意识。

---

本节课中我们一起学习了乌鸦的“是非观”。其核心在于将判断对错的权力从外部集体意识收回，转而聚焦于**自身利益**这个根本标尺。记住这个简单的原则：**对你不利的事，就是错的**。
