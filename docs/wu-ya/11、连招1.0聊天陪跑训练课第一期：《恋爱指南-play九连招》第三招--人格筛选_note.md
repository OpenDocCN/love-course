# 11、连招1.0聊天陪跑训练课第一期：《恋爱指南-play九连招》第三招--人格筛选_note

在本节课中，我们将要学习“连招1.0”聊天陪跑训练课的核心内容之一——《恋爱指南-play九连招》的第三招：人格筛选。我们将深入探讨筛选背后的核心概念“标准”，并学习如何通过筛选行为，在人际交往中建立吸引、表达兴趣并合理化关系升级。课程内容将尽可能简单直白，让初学者能够看懂。

## 课程概述

人格筛选是建立吸引和推进关系的关键环节。它并非简单的流程话术，其背后是个人“标准”的体现。本节课程将首先讲解“标准”的重要性及其在生活各方面的应用，然后解析“筛选”行为的底层逻辑、意义和具体操作方法。

## 第一部分：理解“标准”——筛选的基石

上一节我们介绍了吸引的建立，本节中我们来看看如何通过“标准”来筛选合适的人。标准是你对生活、朋友、伴侣的一种要求和态度，它是你个人框架的重要组成部分。标准可以折中，但绝不妥协。

### 什么是标准
标准即是你对生活各方面的明确要求。例如，对于居住环境，你的标准可能是保持整洁。这会影响你的情绪和效率。

### 标准的重要性
拥有标准能让你清晰地知道自己要什么、不要什么。在人际交往中，尤其是在寻找伴侣时，标准能帮助你快速筛选掉不合适的人，吸引并留下符合你期望的人。

以下是标准在几个方面的具体体现：

*   **生活标准**：例如，每天起床后先打扫房间，保持环境整洁芳香，这能提升整体生活品质和情绪。
*   **交友标准**：选择朋友时，应看重对方的能力和品性，如忠诚度、是否孝顺。一次不忠，终身不用。
*   **消费标准**：清楚自己的喜好，不轻易被他人意见左右。例如，换车时只认准某个品牌。
*   **择偶标准**：这是人格筛选的核心。你需要明确自己对伴侣在外形、性格、价值观等方面的具体要求。

## 第二部分：如何建立你的择偶标准

根据你的性格和需求，建立清晰的择偶标准是有效筛选的前提。最好的伴侣模板是：**喜好相同，性格相反**。

### 制定你的标准
你需要静下心来，认真思考并写下你对理想伴侣的具体要求。

以下是你可以考虑的维度：

*   **外形**：例如，皮肤要求、颜值几分。
*   **性格**：例如，温顺、贤惠、不干涉对方、有共同语言。
*   **价值观**：例如，孝顺父母、对待感情忠诚。
*   **兴趣爱好**：需要有交集，可以培养，这是产生共同话题和“共谋”心理的基础。

### 性格搭配原则
*   **性格相反**：一个强势，一个温柔；一个快，一个慢。这样的搭配更稳定，能形成互补。如果两人都强势，容易产生冲突；如果两人都弱势，则可能无法共同应对外部压力。
*   **标准对撞的处理**：当双方标准冲突时，目标是达到“双赢”。即先坚持自己的立场，然后进行折中处理，采纳双方合理的部分，在尊重对方的前提下做自己。公式可以理解为：**最终方案 = (我的观点 - 对方不认同的部分) + (对方的观点 - 我不认同的部分)**。

## 第三部分：人格筛选的底层逻辑与话术应用

明确了标准后，我们进入实操环节——人格筛选。这不仅仅是一套话术，更是一种高价值的行为模式。

### 筛选的意义
1.  **间接表达兴趣**：在对方对你产生兴趣（IOI）后，筛选是一种不暴露需求感的、间接表达你也对她有好感的方式。
2.  **展示高价值**：筛选行为本身是一种高价值信号。在社会价值排序中，通常是价值较高的一方筛选价值较低的一方。主动筛选能传递出你自身价值不菲、选择权多的信息。
3.  **合理化关系升级**：它为双方关系从吸引阶段升级到更亲密阶段提供了合理化的台阶。就像销售过程中客户需要了解产品细节才会安心购买一样，让对方通过展示内在价值（如会做饭、有爱心）来“通过”你的筛选，会使确立关系的过程更自然、稳固，避免“事后反悔”。
4.  **聚焦内在价值**：筛选应针对内在品质（生存价值），如性格、厨艺、孝顺，而非单纯的外表（繁殖价值）。这传递出你想建立长期关系的诚意，而非仅仅短期的生理吸引。

### 筛选的话术与衔接
筛选需要融入自然的聊天中，避免生硬。

以下是一个话术示例和衔接要点：

1.  **开启筛选（结合暗度）**：在筛选前，可先植入心锚，例如说“我感觉你好像喜欢我”。这能预先在对方心里种下你们之间可能存在浪漫可能的种子。
2.  **自然切入**：以聊天的风格进行筛选。例如，聊到生活时，可以问：“你平时在家会自己做饭吗？”
3.  **深入互动**：如果对方回答“会”，接着问：“现在会做饭的女生可真不多。你最拿手的是什么菜？” 然后可以围绕这个话题展开讨论，分享你自己的经历或喜好。
4.  **接入联系感**：筛选后，自然过渡到分享个人故事和价值观，从而建立联系感。例如，当谈到孝顺话题时，你可以说：“我觉得孝顺父母特别重要，如果一个人连对家人都不好，那对别人恐怕也很难真心。”

**关键点**：整个流程应如行云流水，筛选、有力描述（“你是不是想多了解我一点？”）、建立联系感等环节环环相扣。

## 总结与作业

本节课中我们一起学习了人格筛选的核心——建立个人标准，并掌握了筛选行为的底层逻辑与话术衔接方法。记住，筛选是为了找到那个与你匹配的“螺丝帽”，让你的人际关系，尤其是亲密关系，更加顺畅和高质量。
