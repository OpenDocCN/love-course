# 梵公子课程01：基础概念入门 🎯

在本节课中，我们将要学习一系列基础但至关重要的概念。这些概念是后续所有学习的基石，理解它们将帮助你更轻松地掌握更复杂的知识。

## 什么是变量？📦

上一节我们介绍了课程的整体目标，本节中我们来看看第一个核心概念：变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在程序中，我们通过变量名来访问这个“盒子”里的内容。

以下是定义一个变量的基本方法：
```python
my_variable = 10
```
在这个例子中，`my_variable` 是变量名，`=` 是赋值符号，`10` 是存储在这个变量中的值。

## 理解数据类型 🧱

变量可以存放不同类型的数据，就像盒子可以装书本、水果或玩具一样。常见的数据类型决定了数据可以进行的操作。

以下是几种基本的数据类型：
*   **整数**：没有小数部分的数字，例如 `-5`, `0`, `42`。
*   **浮点数**：包含小数点的数字，例如 `3.14`, `-0.001`。
*   **字符串**：由字符组成的文本，需要用引号包围，例如 `"你好，世界！"`。
*   **布尔值**：只有两种可能的值，`True`（真）或 `False`（假）。

## 掌握基本运算 ➕➖✖️➗

有了数据和变量，我们就可以对它们进行运算。运算不仅限于数学计算，也包括对文本的处理和逻辑判断。

以下是几种基本的运算类型：
*   **算术运算**：包括加法`+`、减法`-`、乘法`*`、除法`/`等。
*   **比较运算**：用于比较两个值，结果为布尔值，例如等于`==`、大于`>`。
*   **逻辑运算**：用于组合多个布尔条件，例如与`and`、或`or`、非`not`。

## 控制流程：让程序做决定 🔀

程序并非总是从上到下逐行执行。通过控制流程语句，我们可以让程序根据条件选择不同的执行路径，或者重复执行某些代码块。

以下是两种基本的控制流程：
*   **条件判断**：使用 `if`、`elif`、`else` 关键字。只有当条件满足时，对应的代码块才会执行。
    ```python
    if 温度 > 30:
        print("天气很热。")
    else:
        print("天气不算很热。")
    ```
*   **循环**：使用 `for` 或 `while` 关键字。可以重复执行一段代码多次，直到满足退出条件。

---

本节课中我们一起学习了编程的四个基础概念：**变量**是存储数据的容器，**数据类型**定义了数据的种类与操作，**基本运算**让我们能够处理这些数据，而**控制流程**则赋予了程序逻辑判断和重复执行的能力。理解这些是编写有效程序的第一步。
