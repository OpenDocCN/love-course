## 一、如何选择拍摄场景

上一节我们介绍了课程的整体框架，本节中我们来看看拍照的第一步：如何选择能让你“变帅”的拍摄场地。很多摄影师懂得美学，但不懂如何把人拍得“受女人喜欢”，关键就在于场景选择。

以下是选择优质拍摄场景的三个核心步骤：

1.  **初步信息收集**：通过三个渠道收集潜在拍摄地点。
    *   **小红书**：搜索“城市名 + 探店/拍照打卡地”。
    *   **抖音**：刷同城内容，观察本地美女高频出现的拍摄场景。
    *   **正妹朋友圈**：观察你已添加的、会拍照的当地女生朋友圈，找出她们重复定位的场所。

2.  **精准场景筛选**：收集地点后，需用以下三个标准进行严格筛选。
    *   **同一场景出片率**：在小红书等平台搜索该地点，看是否有大量博主在此成功出片。数量多代表场景经过验证，可靠度高。
    *   **同一博主对比**：点开在该场景拍得好看的博主主页，对比她在其他场景的照片。如果该场景下的照片是她所有照片中最好看的，说明场景加分；反之则说明场景可能把人拍丑，应淘汰。
    *   **排除客观干扰**：分析一张场景样片好看的原因，是因为模特本身颜值高、穿搭好、摄影师强，还是特定光线（如午后阳光、夜晚灯光）造就的？要选择那些在不同条件下都能把人拍好看的“硬实力”场景。

3.  **核心逻辑总结**：一个好的场景能弥补个人外形的不足，一个差的场景则会让帅哥美女也失色。选择大于努力。

## 二、如何根据场景搭配服装

选好了场景，接下来需要为场景匹配合适的服装。错误的穿搭会瞬间破坏照片的氛围感。

以下是不同场景下的服装搭配核心原则：

1.  **高逼格室内场景**：如高端西餐厅、法餐、五星级酒店走廊、行政酒廊、高空酒吧等。
    *   **搭配原则**：穿着**偏成熟的正式服装**。西装、衬衫等正装能提升气场和质感，与场景的高级感匹配。
    *   **公式**：`高端室内场景 = 正式/成熟着装`

2.  **户外或休闲场景**：如露天咖啡厅、公园、草地、街拍、郊游等。
    *   **搭配原则**：以**休闲、宽松**的服装为主，颜色尽量选择**浅色系**（如白、卡其、浅蓝）。浅色在自然光下更显阳光、青春。
    *   **公式**：`户外/休闲场景 = 浅色系 + 休闲着装`

3.  **主题性场景**：如机车、卡丁车、健身房、高尔夫球场、派对等。
    *   **搭配原则**：服装必须**高度贴合场景主题**。骑机车穿皮衣，打篮球穿球服，健身穿健身服。这是营造“氛围感”的关键，让人一眼就觉得“像那么回事”。
    *   **代码逻辑**：`if 场景 == 主题场景: 服装 = 主题服装`

## 四、拍摄构图与角度实战技巧

场景和服装准备就绪，现在进入实战拍摄环节。如何构图和寻找角度，决定了照片的最终效果。

以下是拍摄时需要执行的构图与角度检查清单：

1.  **竖屏与三分之二构图**：为适配短视频平台，一律使用竖屏拍摄。构图时，让人物占据画面约三分之二的面积，头部位于九宫格左侧或右侧的竖线上。`构图 ≈ 人物占画面2/3 + 头部靠左/右`
2.  **聚焦人物，简化背景**：选择背景干净、线条简单、杂物少的画面，让视觉焦点牢牢锁定在人物身上。
3.  **背景避免复杂线条**：确保头部后方没有明显的直线条（如门框、柱子），以便后期修图时不影响背景。
4.  **规避顶光与底光**：寻找“顺光”角度，让光线均匀打在脸上。避免从头顶或脚底直射的“鬼光”。
5.  **脚底贴边，显高技巧**：拍摄全身或大半身照时，让画面底部刚好卡在脚底，手机可微微上扬，能视觉上拉长身高。
6.  **拍摄大半身**：对于身高不占优势者，优先拍摄膝盖以上的大半身照，更容易聚焦和塑造质感。
7.  **培养“网感”**：多浏览优质照片和短视频，积累审美，模仿电影海报、专辑封面等有质感的构图和氛围。
8.  **先帅后自然**：在颜值提升初期，优先追求拍出最帅的角度和表情，不必过度追求“自然”。帅是基础。
9.  **脸部对焦清晰**：拍摄前务必点击屏幕中的人脸进行对焦，确保面部清晰，为后期修图留下空间。

## 五、与被拍摄者/摄影师的沟通要领

很多人苦恼于“找不到人拍照”，其实关键在于明确分工和有效沟通。

以下是关于拍摄沟通的核心要点：

1.  **摄影师可以是任何人**：只要你提前完成选景、构图、找光等所有准备工作，对方只需要按快门。路人、朋友、服务员甚至三脚架都可以胜任。
2.  **明确告知最佳角度**：了解自己最帅的（如45度侧脸）和最丑的（如仰拍鼻孔）角度，并明确告诉拍摄者。
3.  **善用实况(Live)照片和闪光灯**：打开实况功能，便于后期选取表情神态最完美的一帧。无论是白天还是夜晚，开启闪光灯常亮模式能让照片更清晰、有质感。
4.  **控制眼神与嘴巴**：作为被拍者，注意管理表情。避免瞪眼、翻白眼（眼神难修）以及嘴巴紧闭或张大（嘴型难修）。
5.  **做真实动作进行抓拍**：摆拍时动作幅度要小，贴近真实状态。通过连续抓拍，从中选取最自然好看的瞬间。

## 六、如何筛选可修图的原始照片

拍摄结束后，如何从海量照片中选出最具“潜力”的进行后期修图，是成败的关键一步。

以下是选择原始照片的黄金法则，**必须避开以下“无法修复”的硬伤**：

*   **核心筛选标准**：选择**修图空间最大**的照片，而非原图看起来最顺眼的照片。
*   **一票否决项（直接删除）**：
    1.  **眼神不佳**：翻白眼、眼神涣散、过于凶狠。
    2.  **嘴巴状态错误**：嘴巴紧闭或张开过大。
    3.  **没有下颌线**：面部与脖子连成一片，缺乏轮廓。
    4.  **面部浮肿肉感多**：脸上赘肉多，即使推脸也会导致比例失调。
    5.  **正面鼻孔外翻**：后期难以修正。
    6.  **脖子肉堆积**：缺乏清晰的脖子线条。

**理想原片特征**：`消瘦的脸型 + 清晰的下颌线 + 端正的五官基础`。满足这些条件的照片，经过后期才能产生“脱胎换骨”的效果。
