## 话题选择与延伸

在建立了初步联系后，我们需要选择合适的话题来延续对话。以下是推进聊天的核心方法：

核心方法是**聊对方**或**聊自己**，并将话题与当下场景联系起来。例如，在健身房认识，就可以聊健身话题。

*   **聊对方**：称赞对方的特点，如“感觉你的身材练得挺好的，标准S级曲线”。
