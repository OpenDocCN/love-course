# 感情不是生活的全部 - 课程 P1 🧠💡

在本节课中，我们将要学习如何建立健康的自我价值认知，并理解个人幸福不应完全依赖于他人（尤其是异性）的反馈。我们将探讨两种不同的思维模式，并学习如何将关注点从他人转移到自我成长上。

## 核心问题：过度依赖他人反馈

上一节我们介绍了课程主题，本节中我们来看看一个核心问题。许多人将自我价值感建立在他人（如心仪的女生）的反馈上。这是一种不健康的思维模式。

你的核心问题在于，你过于看重女生对你的反馈。这属于你没有认识到自身价值的问题。

我的价值并不取决于女生是否理我，也不取决于你是否回我消息。我的价值取决于我自己本身是否开心。这个女生没有认可我的价值，不代表我就没有价值。我是自我认可的人，我不需要这个女生的认可。

以下是两种思维模式的区别：
*   **你的模式**：认可取决于别人。
*   **我的模式**：认可来源于我自己。

## 关键心态：区分问题归属

这是一种需要培养的思维模式。她不理我又怎么样呢？她不理我，是**这个女生的问题**，不是我的问题，因为我聊天没有出错。

这一点很重要。就像我原来也常常特别希望女生回我消息。但很多时候你需要明白，她为什么不回你消息？是因为她比你有更好的人选，并不是因为你不行。这才是核心。

## 行动核心：关注自我成长

好，第二个核心是，你需要关注到自己身上。我需要在乎自己内心的真实反馈。

我现在如何看待和女生聊天？其实我不愿意和女生聊天，因为我觉得很浪费时间。看你的聊天记录，你和她天天聊却没时间提升自己。有这个时间，我应该尽可能投资在自己身上。

以下是我不赞同的行为示例：
*   我不会随随便便给一个女生花时间陪她聊天。
*   我从来不会去聊那些“挑礼貌”之类的内容，我觉得这很消耗我。

所以，我的想法始终放在我自己身上：我自己怎么变得更好？我自己才是核心。而你总是关注在别人身上。这一点很重要。

## 总结与回顾

本节课中，我们一起学习了建立独立自我价值的重要性。我们分析了过度依赖他人反馈的问题，学会了区分问题的归属（是她的问题，不是你的问题），并最终将行动核心锚定在**关注自我成长**上。记住，你的价值来源于自我认可，你的时间应投资于让自己变得更好。
