## 开场：判断沟通意愿

上一节我们介绍了课程概述，本节中我们来看看如何迈出聊天的第一步。

我通过社交软件认识了一位在南京工作的女生，她各方面条件优秀，但刚开始聊天时态度比较高冷。加上微信时已是凌晨一点，我不确定她是否愿意继续聊。

我的开场白是：“还以为是睡了。” 女生在约15分钟后回复：“要睡了，回头再聊。” 我则回复了“安安”（晚安的意思）。

这里我想说明的第一个要点是：无论你通过何种方式认识一位女生，聊天的首要方向是**判断对方当下的沟通意愿**。你需要观察对方是否愿意与你交流，而不是自以为是地认为对方有时间或感兴趣。

如果对方没有时间或意愿，此时你应该少说话，而不是多说话。否则，女生可能会觉得你烦人、黏人，或者不懂她。

以下是判断沟通意愿的核心思路：

*   **主动方心态**：你是希望认识对方、接触对方、寻找女朋友的一方。
*   **被动方心态**：对方可能只是通过社交或相亲认识不同的人进行对比，看是否符合其条件。如果不符合，她自然不感兴趣。
*   **关键策略**：在当今环境下，单纯的“陪伴”或“对女生好”已不稀缺，**吸引才是关键**。我们需要通过聊天方式吸引女生，让她对我们产生好奇。

## 推进：试探与话题引导

上一节我们学习了如何开场并判断意愿，本节中我们来看看如何通过试探来推进对话。

我发了“安安”后，对方没有回复“晚安”。我在试探她第二天是否会主动联系我，比如发个“早”或表情包。但直到下午五点她都没有动静。

我忙完后，主动给女生发了一个表情包，目的是再次试探她当天的沟通意愿。

此时有两种可能的情况：

1.  女生继续不理我，态度冷淡。这说明她可能只是用社交软件打发时间，并非真心想恋爱或接触男生。我们无需在此类女生身上浪费时间。
2.  女生愿意回复。但因为她前一天没有主动发消息，说明她可能比较慢热或对我兴趣一般。接下来的思路应以**增加她对我的熟悉度**为主，推进关系需谨慎，避免使用可能让对方觉得油腻的“土味情话”。

我发完表情包后，女生回了一个“？”。这表明她对我的状态很一般。此时不用着急延续话题，所以我根据她社交软件的头像（一只鸭子），说了一句：“鸭鸭。” 这相当于一句“废话”，目的是轻松破冰。

女生回复：“回来了？我好像在出差。” 我顺势延续话题：“对，昨天晚上到了12点。” 女生接着回了一句与我的话题无关的话：“今天温度还蛮舒服的。”

## 核心：把握聊天主线

上一节我们看到女生将话题引向了日常闲聊，本节中我们来学习如何把握聊天的主线。

当我说“昨天晚上12点到的”，女生并没有关心我，只是说了一句关于天气的日常废话。很多兄弟的思路此时可能会被女生带偏，去聊天气，这就错了。

聊天很多时候需要**围绕你们两人之间**，而不是说一些与双方无关的话题。例如，刚认识就聊七大姑八大姨的事，女生肯定不想听，因为这与她无关。

我当时是这么说的：“南京合肥应该都一样，你长得蛮温柔的。” 我开始将话题转移到对方身上，因为聊天气对我们俩的感情推进没有帮助。

她回复：“嗯。” 这也是很多兄弟会遇到的问题：你夸她，她却反馈冷淡。很多人会觉得太难，不想聊了。

我想告诉大家的是：**聊天前期，你需要有自己的主線，不要被女生影响**。很多兄弟遇到一个“嗯”就不会回了。正确的方式是不要被女生影响，坚持自己的节奏。

我当时说：“你这个词，应该是对上面这两句话的认可。”（这其实也是一句废话，因为“嗯”确实不好接）。或者，可以尝试更细致地夸赞，因为夸女生“好看”、“温柔”的人很多，这些都是宽泛的词。可以夸细节，比如“眼妆很精致”、“某张照片很有感觉”。

接着看，女生开始“刁难”：“是（对）第一句，我并不温柔。” 怎么办？夸好看不行，说丑更不行。

很多兄弟的问题在于抓不住聊天精髓，聊不好就容易放弃。但根本问题不解决，未来依然会困扰你。当时我是这么说的：“所以是感觉，头发长长的。你温不温柔我其实不知道，我只是看照片的感觉。你自己照片拍出这种感觉，你能怪我吗？我只是凭我的感觉去说这件事。”

结果女生继续不配合：“没有吧，不長。” 她一直在强调自己不温柔、头发不长。

我说：“可能昨天晚上，看之前的照片。（附上一个笑脸表情）”

从以上三段对话，我想告诉兄弟们三点：

1.  **夸人要谨慎**：随便夸可能“翻车”。
2.  **夸人要细致**：更细心的夸赞会让女生觉得你贴心，不是群发，她才会更关注你这个人。
3.  **灵活应对拒绝**：当女生拒绝或否定时，需要灵活变换思维，不要被女生影响，而是要有自己的主線，延续自己的话题，说出自己的感受。哪怕女生不赞成，也没必要去纠结或争论。

## 深入：交换信息与展示自我

上一节我们学习了如何应对冷淡和把握主线，本节中我们来看看如何通过交换信息和展示自我来深化了解。

遇到女生纠结头发长短，其实不必继续纠缠，因为前面的对话主要是判断沟通意愿。既然愿意沟通，接下来就该交流基本信息，趁着女生有了解我们的动力。

我说：“N（南京的简称？或指代？），我叫吴真。” 她说：“叫我小奶就行了，我朋友这么叫。” 我说：“挺可爱的，小奶。” 她反问：“哪里可爱了？”

我们能发现，这位女生聊天时比较较真。因此之后与她聊天，需要更注意言辞，更谨慎一些。

我说：“显得奶哩奶气的感觉。” 同时，我发了一张我自己和我家狗狗的合照。因为前面一直在围绕对方聊，接下来我要说说自己了，否则你会觉得很难聊。这时我分享自己的生活，等待女生回应。

有人会问：万一女生不回消息怎么办？那就暂时不聊。因为你的聊天没有犯错，分享自己的生活没错，只不过这个女生可能不想了解你。我们没必要在不感兴趣的人身上浪费时间。

她问：“你的狗吗？” 我说：“对，你看她的表情。” 她问：“她？（意思是我的狗是女生吗？）” 我说：“对，小公主，就长得有点丑。”

## 关键：控制聊天节奏

上一节我们开始展示自我并得到了回应，本节中我们来聊聊控制聊天节奏的重要性。

这里我想和兄弟们聊一下**聊天的节奏**。很多兄弟聊天时经常一次性打很多字，比如女生问“你的狗？”，你可能回复：“对，可爱吧？它叫XX，已经X岁了，特别乖……” 如果你说很多，万一女生不回你，或者回得字数很少，你就会很尴尬，需要想新话题。

我们可以把话题拆分开，**让女生不停地向我们提问，然后我们不停地回答**。这样反而能让你占据主动。并且，女生不断向你提问，侧面也是在增加她对你的“投入”。

例如，很多人想了解女生时，会不停地问对方：“你做什么工作？”“你什么星座？”等等。而我们通过说自己、展示自己的方式，引发女生对我们的好奇和提问，也是变相地让女生增加对我们的投入，从而越来越喜欢我们。
