# 长期关系核心课 P1：博弈的力量 💪

在本节课中，我们将要学习长期亲密关系中一个至关重要的概念——**博弈**。理解并掌握博弈，是避免在感情中陷入被动、维系健康平等关系的关键。

## 什么是关系中的博弈？🤔

上一节我们提到了博弈的重要性，本节中我们来看看它的具体含义。情感关系中的博弈，本质上是人与人相处的智慧。其最底层的逻辑，是**你自己的自身框架**与**对方的框架**相互碰撞和磨合的过程。

许多男生在关系初期可能表现得有些“直男”，但进入恋爱后，会发现伴侣开始提出各种不满。例如，抱怨你没送花、嫌弃你工资少、指责你不懂哄人等等。

## 框架碰撞与被动处境 🥊

以下是女生常用的一些策略，其目的是将你拉入她的框架：

*   **抱怨与指责**：指出你的不足，与他人比较。
*   **撒娇与任性**：通过情绪化方式，让你迁就她的需求。
*   **明示与暗示**：引导你为她花钱、付出更多关心和体贴。

如果你不断陷入这些问题却不知如何应对，核心原因只有一个：**你不懂得博弈，自身框架太弱**。你正在被对方的节奏带着走，这在感情中意味着被动，长期如此，关系便岌岌可危。

## 如何增强博弈能力？⚖️

我们已经了解了陷入被动的原因，那么，如何扭转局面，增强自己在关系中的博弈能力呢？关键在于两点：**强化自身框架**与**增加对方投入**。

如果一段关系中总是你在单方面付出，而对方只提要求却不付出，你的地位就会非常危险。因此，引导对方投入至关重要。投入可以分为四个层次：

1.  **物质投入**：金钱、礼物等实际付出。
2.  **精力投入**：为你操心、处理事务所花费的心力。
3.  **时间投入**：陪伴你的时间。
4.  **情绪投入**：为你产生喜怒哀乐等情绪波动。

**操作核心**：根据对方情况，引导其投入。如果对方物质不丰富，就引导她投入时间、精力和情绪。公式可以简化为：**你的话语权强度 ∝ 对方的总投入**。

只要你能有效增加对方在关系中的投入，你的博弈能力和话语权就会随之增强。相反，如果对方从未为你付出，只是一味享受你的好，那么你在这段关系中将处于极其不利的位置。

## 总结 📝

本节课中，我们一起学习了长期关系的核心——**博弈**。我们明白了博弈是双方框架的碰撞，软弱框架会导致被动与妥协。而增强博弈能力的根本方法，在于**坚守自我框架**并**巧妙增加对方的全方位投入**。记住，健康的感情是一场动态平衡的合作，而非单方面的奉献。
