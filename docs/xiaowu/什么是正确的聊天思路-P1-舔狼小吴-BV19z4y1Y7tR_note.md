# 聊天基础教程 P1：建立正确的聊天思路 💬

在本节课中，我们将通过分析一段聊天记录，学习如何建立流畅、自然的聊天思路。我们将重点关注如何避免常见错误，并理解聊天的核心在于延续对话，而非追求华丽的辞藻。

## 概述：从一段问题聊天记录开始

我们首先从一段聊天记录中分析过程中存在的问题。

**示例对话：**
```
A: Hello
B: 你也说Hello
A: 今天真冷啊
B: Yes，刚回家冻得跟冰棒似的，哈哈
B: 好
```

你这里是不是在聊自己？是的。聊完自己之后该怎么办？我可以聊聊对方吗？或者我可以延续一些第三方话题吗？是不是都可以？我就可以说我这里的天气，或者说你们那块地方的天气总是这样，就很烦呀。可不可以加点自己的感受进去？可以。那我也可以问问对方，我说“你在干嘛”，是不是都行。

## 分析：如何回应对方的谦虚

好，我们看后面。

**示例对话：**
```
A: 话说你这穿搭有点帅
B: 还好吧
```

这时你去聊对方，他说“还好吧”，那你是不是不知道到底该怎么接了？对，怎么接呢？他说“还好吧”，你就可以比如说说他的某个特点比较吸引我，可不可以？可以。那我也可以怎么说吗？我说“什么时候向你学习”，不也可以吗？

你把聊天想成什么了？不懂。关键在于，当对方表现出谦虚时，你可以具体化你的赞美或提出一个轻松的学习请求，这都能让对话继续。

## 核心问题：不要追求华丽的语言

看第二张示例。

**示例对话：**
```
A: Hi
B: Hello，你这个名字缺个酱
A: 不缺
B: 我口袋不会回了
```

这个开场白整体很单调。但问题是，你不要在聊天中去追求非常华丽的语言，你不需要。你只要把聊天延续下去就可以了。明白吗？你又不是艺术家，又不是要写诗，为什么非要追求每句话都华丽呢？你不会聊的话就不要强行那样聊。

如果你觉得对方某些点比较好，你可以借此吸引注意，但你没有必要像每个教学案例一样。案例也只是故意展现出来了，显得他很会说话，但你达不到他这种层次。就像你觉得我有的时侯聊得很自然，我有时也是故意表现，可能10个案例里只有这一个案例表现出来了，所以你误以为聊天一定要那样。明白了吗？他有时那样聊，女生也接不下去。你看到的不一定是真实的全貌，可能背后只是极少数的情况。

懂了。就这么简单，你没有必要追求每个回合都很华丽。华丽的背后通常也需要对方比较配合你。

## 错误示范：话题跳跃与不专注

你刚才的例子：“你说你这个名字缺个酱，他不接话，然后你说那吃薯条估计都不蘸酱”，这不就相当于扯远了吗？就是很奇怪。万一女生给你发省略号呢？那你是不是就觉得这句话接得不好？你会那么想，对呀。

再看另一个例子。

**示例对话：**
```
A: Hello hi，你这红色感叹号蛮好看的
B: 豆沙色口红吗？
A: 不是，就是很普通的承诺，同学送的
B: 早呀不知不觉昨天睡香了
```

那我可能会怎么回？我说“适合你才是最好”，是不是也可以？

好，我们再看这个，你后面两个人的对话。

**示例对话：**
```
A: 哎下雨了，窗户上有雨滴（然后你发个表情包）
B: 咋的呢，刚填饱肚子
A: 他吃的不错呀，可不你吃午饭了没
B: 没呢等你请我吃呢
A: 他还吃泡面吗，我看行，不然你叫我什么，叫我弟弟就行
```

他说“没呢，等你请我吃”，那我要去思考之前聊了什么内容，或者说女生对我够不够了解。他说“等你请我吃呢”，那我就可以说“有空约一下”可不可以？我说“有空约一下”，我又没说我一定要去跟他一起吃饭，对不对？如果女生接着问“哎呀那什么时候去吃呀”，那就说明这个女生是主动送上门了，你就没有必要去这么说。所以你这里也可以客套一下，你说“好呀，那下次到时候可以一起吃哟”，是不是也可以啊？

## 识别兴趣信号与切换话题

好看后面。

**示例对话：**

B: 好吧
```

他就不好接嘛，对吧？这时就可以问对方“你在干嘛呢”。所以这女生可能也说明对你没兴趣。对，对呀，就这么简单。

我看下一个。

**示例对话：**
```
A: 哈喽嗨，减肥的女生身材真不错，哈哈，现在减肥快成功了吗
B: 啊？我说没看懂吗？你接着延续呗，加班呢，你呢
A: 好不刚搬回家，跟室友一起住了
B: 我哪有室友，之前不是合租吗，跟谁呀，我咋不知道（他发个懵的表情）

```

你这不就离谱吗？我搞错人了。“现在有个合租的”，他就说记错了呗。他觉得你好奇怪呀。

**示例对话续：**
```

你这里其实算一个模糊的邀约。

**示例对话续：**
```

A: 我不在酒店啊，又换工作了

```

你这聊天给人的感觉就是不上心。他觉得你总说错话，所以他不找你聊天。我当时的想法就是，怎么聊我，怎么聊他，一直在想这个，感觉这么久都没啥进步。是，那种感觉。那没啥进步，你要好好反思自己。

## 总结你的核心问题

那我们看看你的问题所在。以下是总结出的几个关键点：

*   **思路僵化**：不会正确地在“聊他”和“聊我”之间切换。
*   **对话单调**：整体聊天缺乏情绪，像是“你一句，我一句”的面试，没有话题感。
*   **延续困难**：无法自然地延续话题，经常卡住。

你下次再去找一个女生，在对方配合度尚可的情况下，去找找聊天的感觉。这个我教不了你，帮不了你。我怎么跟你讲？我现在去高级教育，你聊天就会了吗？你不会的。对，对呀。那还是得说，需要你接下来在聊天过程中，我帮你复盘，你才知道问题在哪。懂吗？

你聊天觉得困难，很有可能是因为这个女生对你没有什么兴趣，所以你很难进行下去。这也是一个比较核心的原因。比如你问“我叫你啥，你可以叫我弟弟，我也可以说我自己的名字”，但是不知道怎么延伸话题，觉得不知道下一步怎么走。

## 解决方案：明确聊天思路与任务

那下一步怎么走，这就是聊天的思路。你看当前这个话题到底能不能延续，不能延续就跳转到其他话题。就这么简单。你看女生给不给你话题，不给你话题你就切换。

你脑子里还是没有准确的“聊天地图”。那你就去看，好好反思我之前课程里给你的东西，你并没有掌握。你回去好好把课程里的内容看看。

关于开场白容易冷场的问题，我觉得你那几个开场其实都还好，也是我理解的问题。当时我理解是有点冷场，然后听老师这么一说，感觉也没有太大问题。

你提到的“不知道何时接话题”这一点，和第一点是一个道理。你要通过复盘自己的聊天，才知道到底什么时候该找新话题。这个东西不是我能告诉你的，是你自己能不能从自己的聊天方式中找出来。

那么按我说的，关于“什么时候接话题，什么时候不接话题”，你就要去看你的**主线任务**和**支线任务**。你当下聊的是主线任务还是支线任务？如果是支线任务，我就知道了，我不要去延续它了，我去聊主线任务上的话题。那我就可以看主线话题到底可以聊哪些。就这么简单。

如何判断该不该切换话题，如何延续话题，你这相当于还是这个点。第一、五、六点相当于你还是不会聊天。那你就把之前发给你的关于主线任务、支线任务的内容，以及我发给你的案例，好好看一下。观察我在聊天过程中到底聊了哪些话题，哪些话题没聊，什么时候去延续，什么时候不该延续。
