# 课程一：理解变量与赋值 🧠

在本节课中，我们将要学习编程中最基础也最重要的两个概念：**变量**和**赋值**。它们是所有程序的基石，就像盖房子需要砖块一样。

## 什么是变量？

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个**贴有标签的盒子**。这个盒子可以用来存放数据，比如数字、文字或更复杂的信息。标签就是变量的名字，我们通过这个名字来找到并使用盒子里的数据。

在编程中，变量有三大特征：
*   **名称**：用于标识和引用变量。
*   **值**：存储在变量中的数据。
*   **类型**：变量所存储数据的种类（例如整数、文本）。

## 如何进行赋值？

理解了变量是“带名字的盒子”后，我们需要知道如何把数据放进盒子里。这个过程就叫**赋值**。赋值操作使用等号 `=`，但它不是数学中的“等于”，而是表示“将右边的值存入左边的变量”。

其核心公式是：
**`变量名 = 值`**

例如，在Python中：
```python
age = 25
name = "小吴"
```
第一行代码创建了一个名为 `age` 的变量，并将数值 `25` 放入其中。第二行创建了变量 `name`，并存储了文本“小吴”。

以下是赋值时需要注意的几个关键点：
*   等号 `=` 是赋值运算符，意为“赋予”。
*   变量在使用前通常需要先赋值。
*   变量的值可以通过再次赋值来改变。

## 一个简单的实践例子

让我们通过一个具体例子来串联以上概念。假设我们要记录小吴的年龄，并在一年后更新它。

```python
# 创建变量并赋值
xiao_wu_age = 25
print("小吴今年的年龄是：", xiao_wu_age)

# 更新变量的值（重新赋值）
xiao_wu_age = 26
print("小吴明年的年龄将是：", xiao_wu_age)
```
运行这段代码，你会看到输出先后显示25和26。这演示了变量如何存储数据，以及其值如何被改变。
