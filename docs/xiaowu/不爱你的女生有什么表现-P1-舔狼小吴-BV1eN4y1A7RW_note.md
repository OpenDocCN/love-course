# 课程名称：识别不真诚关系的表现 - P1 🧐

在本节课中，我们将学习如何识别一段关系中，对方可能并非真心投入的几个关键表现。这些迹象有助于你更好地判断关系的健康程度。

上一节我们介绍了课程的主题，本节中我们来看看具体的行为表现。

以下是只想玩玩、并非真心爱你的女生可能展现的四种主要表现：

1.  **漠视你的感受**
    她根本不会在乎你的感受。你的喜怒哀乐与她无关，她更不会包容你。当你带着情绪与她交流时，她会表现得比你还生气。

2.  **拒绝主动和解**
    你们吵架后，她明知道你伤心痛苦，但从来不会向你低头，表现得像个陌生人。每次都是你妥协、主动求和。

3.  **忽视你的付出**
    在恋爱关系中，她只看到自己的付出，并且认为你的付出是理所应当的，是你应该做的。

4.  **单方面索取照顾**
    她一直希望你照顾她，只接受你的付出，却不想自己受一点点委屈。

本节课中我们一起学习了识别不真诚关系的四个核心表现：**漠视感受**、**拒绝和解**、**忽视付出**和**单方面索取**。你可以对照观察，你身边的那个女生是否也有类似表现。
