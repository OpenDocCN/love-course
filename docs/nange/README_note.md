# 男哥的Python入门课 #1：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础、最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如：
```python
name = "男哥"
age = 30
```
这里，`name` 是一个变量，它存储了字符串“男哥”；`age` 是另一个变量，它存储了数字30。

## 核心数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道“盒子”里装的是什么类型的东西。Python有几种基本的数据类型，它们决定了数据可以进行的操作。

以下是几种最常见的数据类型：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由文本组成的数据，需要用单引号或双引号包围，例如 `"Hello"`, `'Python'`。
*   **布尔值**：只有两种取值，`True`（真）或 `False`（假），用于逻辑判断。

## 如何查看与理解类型？ 🔍

在编程时，了解一个变量当前是什么类型非常有用。Python提供了一个内置函数 `type()` 来帮助我们。

我们可以通过以下方式使用它：
```python
height = 1.75
print(type(height))  # 输出：<class 'float'>
```
这段代码会告诉我们变量 `height` 的类型是浮点数。

## 变量的命名规则 🏷️

不是任何单词都能作为变量名。为了写出清晰且正确的代码，我们需要遵守一些规则。

以下是命名变量时需要遵循的主要规则：

*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名是区分大小写的（`age` 和 `Age` 是两个不同的变量）。
*   应避免使用Python的保留关键字（如 `if`, `for`, `while`）。

一个良好的习惯是使用描述性的名字，比如用 `user_name` 而不是简单的 `n`。

## 一个简单的实践示例 💻

现在，让我们把以上知识结合起来，写几行简单的代码。

```python
# 定义几个变量
course_name = "Python入门"
student_count = 25
is_online = True

# 打印它们的值和类型
print("课程名称：", course_name, "， 类型：", type(course_name))
print("学生人数：", student_count, "， 类型：", type(student_count))
print("是否在线课：", is_online, "， 类型：", type(is_online))
```
运行这段代码，你会看到每个变量存储的值以及它对应的数据类型。
