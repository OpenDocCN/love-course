在本节课中，我们将探讨一个关于男性心理与两性关系的重要话题：自信。我们将分析社会观念对男性自信的影响，并讨论如何在复杂的社会声音中建立并保持内心的坚定。

首先需要声明，今天讨论的内容并非为了博取眼球。请大家先看一段视频中提出的现象：为什么有些女人总爱生闷气不说话？她们的想法是：说了你又不听，听了你又不懂，懂了你又不做，做了还错，错了又不改，改又改不好。而一旦说“你改不好”，你就生气，被指责为“无理取闹”。那么，她还能说什么呢？

实际上，对于网络上某位老师发布的这类视频内容，我并无意见。因为他的粉丝群体主要是女性，他必须获得女性的支持才能进行带货，这很正常。

但是你们知道吗？我国有五千万的兄弟，连女人的手都没有碰过。比激进的女权主义更可怕的，是一种**“女本位的传统主义”**。这种观念意味着，我们从小到大接受的教育都在传达：只有给女人大把花钱的男人才算成功；努力读书工作是为了挣钱娶媳妇；男人必须做家里的顶梁柱，不出去干活就是废物；而女人则应该负责貌美如花。

这些思想表面上看似乎没有错。然而，网络上那些拥有话语权的人，总喜欢站在道德制高点，去谴责我们这些男生。这导致我们的小男孩从小就缺乏自信。

试想一下，我们有些兄弟连基本生存都成问题，你跟他谈什么情商，谈什么格局？我时常感觉，我们这个民族在某些方面被这种女性视角的叙事压制住了。

但你知道吗？真正到了生死存亡的战争或灾难时刻，生死关头，冲在前面的永远是男人。我们的社会精英在关键时刻，做出的选择永远是**牺牲男人以保护女人**。

女人有时赌的是幸福，而男人更多时候是在赌命。女人往往只看到自己未曾获得男性所拥有的权利，因此去抗争。但她们未必看到了男性所付出的牺牲与承担的责任。

当然，我明白以上言论有些偏激。我的核心目的，只是希望兄弟们能变得更加自信。

你们不需要去听信女人对你们的贬低。因为她们可能也只是听到网上那些“厉害”的老师怎么说，就跟风认为你“没有用”。事实上，女人很少有自己的独立思想，她们只喜欢听那些偏向自己立场的话。

因此，你们也不必去跟女人讲道理。更不要真的认为自己没有用。

**你应该永远保持自信和向前的勇气。**

关键在于：**多关注自己，把自己变强**。当你自身强大起来，这个妹子自然就会听你的话。而当你获得了她的倾听与跟随，你自然也会开始心疼她、保护她。

这便是一个**矛盾的对立和统一**的过程。
