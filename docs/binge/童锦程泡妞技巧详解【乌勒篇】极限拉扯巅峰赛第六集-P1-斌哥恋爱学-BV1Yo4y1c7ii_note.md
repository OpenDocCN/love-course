# 童锦程泡妞技巧详解【乌勒篇】极限拉扯巅峰赛 第六集 🎬

在本节课中，我们将要学习“极限拉扯”互动中的高级技巧，特别是如何通过推拉、转移话题和应对突发状况来维持吸引力并推进关系。我们将通过分析一段经典对话，拆解其中的关键步骤和核心心法。

---

上一节我们介绍了需求感暴露的“度”需要在推拉中产生。本节中我们来看看，如何在具体的对话交锋中运用“极限拉扯”来共建吸引力。

极限拉扯的魅力在于，双方都在张弛有度的互动过程中，共同构建吸引力。此时，双方的感情反而更容易推进。

以下是对话中的一个关键片段及其分析：

*   **场景：** 双方喝酒，讨论换地方。
*   **对话：** “我们先喝完这一瓶吧。”“那我先喝。”“行，你真干了。”“我也不想干，可是你刚才要对我吐舌头。”
*   **分析：** 即便对对方有好感，女性通常也不愿让对方觉得自己随便。遇到对方的直接试探（吐舌头），一般会抗拒或害羞。

我们来看看乌勒如何应对。她并没有直接回答，而是进行了一个带有引诱意味的转移。

*   **动作：** “是嘴巴干了。”“哪里干了，嘴巴。哪里？这里。”
*   **分析：** 乌勒虽然害羞，但做了一个引诱动作（指出是嘴巴干），随即又把主动权往回拉了一点。

上一集说过，极限拉扯的规则是“谁先下手谁输”。所以，当乌勒把嘴唇凑上来时，童锦程便后退一步，同时做出绅士般的关心动作。

乌勒见状，立刻再次发起进攻。“你为什么老在乎我几点下班？”童锦程不给机会，再次做出嫌弃般的防守。

一般此时，女性的内心会受挫。让我们看看乌勒怎么办。

“我想跟你聊的东西就少儿不宜了。”乌勒完全不慌，面带笑容，坚持不懈地再次发起进攻，并且这次进攻带有强暗示。

“你能坐这吗？我喜欢在上面。”乌勒这一招很致命。童锦程立刻飘了，口出狂言：“哎呀，我喜翻上面一点。什么东西少儿不宜？就聊一下平平常常的事情，比如**成人教育**。”

乌勒见童锦程上钩后，马上急转枪头，做了一个退让。“**成人教育**就是成人什么专升本啦……成人大学，对。”童锦辰发现自己被戏弄，立刻调整战术转移话题。

“到时候可以咨询一下。**美瞳**，我看看，什么颜色？”“是你喜欢的颜色。”乌勒继续乘胜追击。被怼得哑口无言后，童锦程再次调整战术转移话题。

“你是内蒙古人吗？那你会骑马吗？”毕竟是“祖师爷”，一句“你会骑马吗”立刻让战局峰回路转。

兄弟们看到这里，应该能发现规律：**每次在形势不利时，就可以转移话题。而这个新话题中，要带着陷阱和侵略性。**

例如，童锦程在陷入被动后，通过“你会骑马吗”这种带有暗示性的话题重新掌握节奏。“你看你又吐舌头，你能不吐舌头吗？你这样我真的忍不住想要下班。”

当乌勒反问“你说什么乌苏（酒）啊？”时，童锦程通过夸张的玩笑和反转来应对：“哎哎哎，我就说说……哎呀，我开玩笑的。”“我当真了。”

此时，乌勒也没有手下留情，通过质疑酒精度数进行反击：“这度数也不高啊……老板你得喝度数多高的？”童锦程回答：“**64度**，就那种打火机都会烧着的那种。”

但没想到高兴得太早，撞到了枪口上。乌勒反问：“你那么能喝吗？”童锦程说：“没有啊，就看跟谁喝。”乌勒见形势大好，马上反攻：“我**现在**就有点多了，我也有点晕。你这家里带过多少个妹妹了？”

童锦程回答：“嘎嘎是第一个，你是第二个。”乌勒说：“我信你。你看啊，真的我哥……好心，你跟我哥干杯？哥哥喝完吗？喝完！行，哥哥讲什么不行。”

这个地方可能很多兄弟看不懂，我解释一下：童锦程这个过激的谎言（“你是第二个”）可能惹怒了乌勒，但她没有直接发飙，而是选择了更高级的方式——**阴阳怪气**。这也是天蝎座惯用的招式。
