# **《恋爱学：如何赢得心仪之人》P1 - 赢取Q币和追求爱情的秘诀** 🌹

## 概述
在本节课中，我们将学习如何通过坚定的内心和厚脸皮的态度去赢得心仪之人，无论是爱情还是事业。

## 核心概念
**公式**：成功 = 厚脸皮 + 坚持 + 不顾面子

## 教程内容

### 以下是关于如何赢得心仪之人的关键点：

1. **美女的归宿**
   - 大部分美女最终嫁给的是那些脸皮厚、死缠烂打的人。
   - 这些人通常不按常规出牌，不顾及面子。

2. **脸皮厚的人的特点**
   - 他们直接表达自己的感情，不顾及对方的拒绝。
   - 他们不等待对方主动追求，而是主动出击。

3. **面对拒绝的态度**
   - 脸皮厚的人不会因为一次拒绝就放弃，他们会持续追求。
   - 他们会利用各种机会接近对方，即使对方心情不好。

4. **案例分享**
   - 创业初期，作者为了拿到一个订单，多次拜访一位老板，最终成功。
   - 即使面对拒绝和尴尬，作者依然坚持，最终赢得了机会。

5. **结论**
   - 这个世界上的好东西都是通过努力和坚持获得的。
   - 要赢得爱情和事业，需要一颗强大的内心，放下面子，坚持不懈。

## 总结
本节课中我们一起学习了如何通过厚脸皮、坚持和不顾面子来赢得心仪之人。记住，成功需要付出努力和坚持，不要害怕失败，因为每一次失败都是向成功迈进的一步。
