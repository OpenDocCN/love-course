# 恋爱与财富第一课：主动争取的力量 🎯

在本节课中，我们将探讨一个核心观点：无论是获取财富还是赢得爱情，许多美好的事物都需要通过积极主动、坚持不懈的“争取”来获得。我们将通过具体的故事和分析，理解“厚脸皮”背后的积极含义与行动策略。

上一节我们介绍了主动争取的核心概念，本节中我们来看看这一现象在现实中的具体体现。

你们知道大部分的美女最后都嫁给谁了吗。不是有钱的，也不是长得帅的。根据我30多年的观察和总结，他们大部分都嫁给了那种脸皮厚的死缠烂打的人。这些人的脸皮是真的厚。他们不讲武德。

他们上来就一顿操作猛如虎，软磨硬泡知道吧。根本不按套路出牌。当别的小伙子还在考虑对方会不会拒绝我，要怎么开口。有的还傻乎乎的去问这个女人，我可以追你吗。更傻的是，他还等着那个美女主动的去追求他。

而那些脸皮厚的人呢，他不管那么多。他冲上去就说，**“我喜欢你，我爱你，没有你，我活不下去。”** 根本不要脸知道吧。他们根本不在乎什么面子。今天不行就明天，明天不行就后天，后天不行就下个月。等到你哪一天。

然后心情不好的时候，遇到不开心的事情，然后你就软下来了。突然就来了一句，你今天请我喝酒吧。我跟你们讲，这种人以后是不可能没有钱的。

理解了追求爱情中的主动模式后，我们来看看同样的原则如何应用于商业和财富积累。

我刚开始创业的时候，为了拿到一个订单，我拜访了一个老板。老板的价值非常大。

他拒绝了我六次。到第七次的时候，我又追到他的办公室。这个大哥呢，看到我就很烦。本来是一个看上去很斯文文雅的一个人。他突然就对我说了一个字。当时我真的是很没有面子，恨不得找一个地方就直接挖个洞钻进去。

我也说了一句话，**“大哥对不起，我刚刚从对面的那一个楼滚过来，你能不能让我在这里休息三分钟。”** 这个大哥哈哈一笑，他就给了我这个机会。

基于以上两个领域的例子，我们可以总结出普适性的成功法则。

所以说这个世界所有的好东西都是抢来的。客户是抢来的，你的人也是抢来的。

钱也是从别人的口袋里面抢过来的。你想要得到的东西，他都需要一颗强大的内心，放下你的面子。这个世界不属于80后，不属于90后，也不属于00后。这个世界属于那种脸皮厚的人。

以下是追求成功（无论是财富还是爱情）的三个具体行动要点。

你们想要追女孩子很简单。只要做到三点。

*   **第一不要脸。**
*   **第二坚持。**
*   **第三坚持不要脸。**

本节课中我们一起学习了主动争取的重要性。核心在于培养强大的内心，放下对“面子”的过度执着，通过**“不要脸 + 坚持 + 坚持不要脸”**的行动公式，在爱情和事业中更有效地达成目标。记住，许多机会不是等来的，而是通过持续、勇敢的尝试“争取”来的。
