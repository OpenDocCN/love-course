在本节课中，我们将要学习一个关键的心理学概念——“沉没成本效应”，并探讨它如何影响我们在感情中的判断和行为，导致我们对某个人产生难以自拔的“上瘾”感。

## 核心概念：沉没成本效应 💡

上一节我们介绍了课程主题，本节中我们来看看其背后的核心原理。

你之所以对一个人“上瘾”，并非因为对方有多么优秀，而是你的头脑被情绪操控，陷入了一种心理陷阱。这种心理陷阱在心理学中被称为 **沉没成本效应**。

**沉没成本效应** 是指：人们为了避免承认损失所带来的负面情绪，而沉溺于过去的付出中，从而选择非理性行为方式的一种心理倾向。人性天生厌恶损失的感觉，因此会做出非理性的行为来自我欺骗。

## 生活中的沉没成本案例 📊

理解了核心概念后，我们来看看它在日常生活中是如何体现的。

以下是几个常见的例子，帮助你更好地理解：

*   **股票投资**：你买错了股票导致亏损。理性做法是及时止损（`理性决策：卖出股票`），但你可能反而投入更多资金买入，因为停止投入就等于承认了自己当初的愚蠢。
*   **观看电影**：你买票后发现电影很烂。理性来说，离开影院去做更有价值的事对你更有利，但很多人会选择忍着看完，试图“挽回”那张电影票的成本。

在这些案例中，最好的选择本应是着眼于未来能带来更好体验的事情，而非执着于弥补过去的损失，但人们往往难以做到。

## 沉没成本在感情中的运作 🫀

现在，我们将这个概念应用到感情关系中，看看它是如何让你“上瘾”的。

沉没成本不仅指金钱，还包括付出的时间、精力和情感。许多男性所处的感情关系并非他们想要的，甚至带来痛苦。但由于在一起久了，已经为这段关系投入了大量时间、金钱和感情，沉没成本会阻碍他们做出理性判断。

他们天真地认为，只要自己再努力一点、再改变一些，关系就有救。于是投入更多来“救场”，导致沉没成本越来越重，最终深陷痛苦无法自拔。

## “上瘾”的心理过程：从幻想到赌徒心态 🎲

上一节我们分析了沉没成本如何困住一段关系，本节中我们来看看它对单方面“上瘾”的具体影响。

对一个女性“上瘾”的过程通常遵循以下模式：

1.  **初期幻想**：你开始对她产生兴趣，聊了几句后便开始幻想“她是不是喜欢我”，并不断寻找她喜欢你的证据（例如：“她这句话什么意思？”“她为什么对我笑？”）。
2.  **陷入焦虑**：你会因为她的态度变化而焦虑（例如：“她今天为什么不热情？是不是有别人追她了？”）。
3.  **未来畅想**：晚上独自躺在床上，你会开始幻想未来的种种场景，比如带她见朋友、旅游、见父母、结婚生子。此时，你潜意识里已经告诉自己“我爱上她了”。
4.  **非理性付出**：尽管理性上知道关系尚早，但你已无法自控，试图通过满足她的一切来换取她的爱，以满足自己内心的期待。从这一刻起，你很可能已走向过度付出的道路。
5.  **赌徒心态**：在你一次次投入后，若她对你的热情反而减少，你会感到慌张。你想不通为何自己如此爱她、投入这么多，绝不允许自己放弃。于是，你**加大投入，想一把捞回本**，开始疯狂付出，结果越陷越深。这就是你“上头”的根本原因。

## 总结与建议 📝

本节课中，我们一起学习了**沉没成本效应**如何操控我们的感情。

*   **核心要点**：你对一个人“上瘾”，往往不是因为对方的价值，而是因为你已投入的成本（时间、精力、情感）让你难以割舍，陷入了“继续投入或许能挽回”的赌徒心态。
*   **关键公式**：**情感“上瘾” ≈ 初期兴趣 + 自我幻想 + 持续投入 + 害怕损失（沉没成本）**
*   **理性做法**：学会识别沉没成本陷阱。当一段关系或单方面付出主要带来痛苦而非快乐时，**及时止损**（`理性决策：停止追加投入，重新评估关系`）是更健康的选择。你的时间、精力和情感是宝贵的，应该投资给未来能带来积极回报的人和事。
