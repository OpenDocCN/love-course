## 概述
在本节课中，我们将要学习编程中最基础但至关重要的概念：变量。理解变量是编写任何程序的第一步。

## 什么是变量？🧠
上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。

变量是计算机内存中一个存储数据的“盒子”。你可以给这个盒子起一个名字，并向里面放入数据。之后，你可以通过这个名字来读取或修改盒子里的数据。

## 如何声明变量？📝
理解了变量的概念后，接下来我们学习如何创建，或者说“声明”一个变量。

在大多数编程语言中，声明一个变量需要指定两个关键信息：**变量名**和**（可选的）初始值**。其基本语法通常如下：
`数据类型 变量名 = 初始值;`

以下是声明变量的三个核心步骤：
1.  **选择数据类型**：决定变量要存储什么类型的数据，例如整数、文本等。
2.  **命名变量**：为变量起一个有意义的名字，以便在代码中引用它。
3.  **赋值**：使用等号 `=` 将数据存入变量。

## 变量的命名规则🔤
声明变量时，命名需要遵循特定规则。本节将详细说明这些规则。

变量名通常由字母、数字和下划线组成，且不能以数字开头。此外，变量名是区分大小写的，这意味着 `myVar` 和 `myvar` 是两个不同的变量。

以下是命名变量时需要遵守的具体规则列表：
*   变量名可以包含字母、数字和下划线 `_`。
*   变量名不能以数字开头。
*   变量名不能是编程语言预留的“关键字”（如 `if`, `for`, `int`）。
*   建议使用有描述性的名字，例如用 `userAge` 而不是 `a`。

## 变量的基本操作：赋值与使用⚙️
我们已经学会了如何声明和命名变量，现在来看看如何实际使用它们。

对变量的两个最基本操作是**赋值**和**取值**。赋值使用等号 `=`，将右边的值存入左边的变量。取值则直接在表达式中使用变量名。

请看以下代码示例，它演示了变量的声明、赋值和取值过程：
```python
# 声明一个名为 ‘count’ 的变量，并赋初始值为 10
count = 10

# 对变量进行赋值：将 count 的值改为 20
count = 20

# 使用变量：将 count 的值（20）加上 5，结果（25）存入新的变量 total
total = count + 5

# 使用变量：打印 total 的值
print(total) # 输出：25
```

## 总结
本节课中我们一起学习了编程的基础——变量。我们了解了变量是存储数据的容器，掌握了声明变量的语法与命名规则，并实践了变量的赋值与取值操作。理解这些概念是后续学习更复杂编程知识的坚实基石。
