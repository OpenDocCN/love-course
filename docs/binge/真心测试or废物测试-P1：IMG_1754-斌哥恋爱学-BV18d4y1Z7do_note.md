# 💖 真心测试与废物测试 - 课程P1

在本节课中，我们将要学习恋爱关系中的两个核心概念：“真心测试”与“废物测试”。我们将探讨它们的定义、运作机制以及如何应对，帮助你理解女性在亲密关系中的潜在心理动态。

## 什么是真心测试与废物测试？

上一节我们介绍了课程主题，本节中我们来详细看看这两个概念的具体含义。

**真心测试**，是指对方试图验证你对她的感情是否真诚、是否足够深刻。她希望得到一个肯定的答案，以确认你的心意。

然而，在这个寻求答案的过程中，往往隐藏着一个**废物测试**。其核心逻辑是：**如果你应对不佳，她会认为你软弱、容易被掌控**。一旦她发现你缺乏自我掌控力，她就会瞧不起你。

## 女性的潜在心理动态

理解了测试的基本定义后，我们来看看这些测试背后反映了怎样的心理。

女性的心理模式有时显得颇为奇特。其典型过程如下：**女性最初因为崇拜你而选择与你在一起**。在关系确立后，她会逐渐尝试将你从你原有的“阶层”拉下来，拉到与她相近的层次。

紧接着，她会运用自己更为丰富的（情感）经验来“打败”你。这个过程可能表现为对你进行**蹂躏、控制、折磨**。然而，在完成这一系列行为后，她反而会看不上已被她“击败”的你，最终选择分手。

以下是这一心理动态的典型步骤：
1.  因崇拜而开始关系。
2.  试图将对方拉低至与自己同等的心理或情感层面。
3.  利用经验优势在互动中占据主导。
4.  在主导和控制后产生轻视。
5.  最终结束关系。

需要说明的是，这通常是比较幼稚、或社会阅历与心智成熟度较低的女性可能持有的思路。

## 总结

本节课中我们一起学习了“真心测试”与“废物测试”的核心区别与联系。我们了解到，真心测试是对方寻求情感确认的表层诉求，而废物测试则是隐藏其下的、对你个人价值与框架的考验。同时，我们也剖析了关系中一种不健康的心理动态模式，即通过“拉低-控制-轻视”的循环来获取安全感，这有助于你识别并避免陷入此类消耗型关系。理解这些概念是建立健康、平等亲密关系的第一步。
