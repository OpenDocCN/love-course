# 恋爱极限拉扯详解•第三集•童锦程VS呆妹 🎬💘

上一节我们介绍了极限拉扯的基本概念，本节中我们来看看这些理论在具体对话中是如何应用的。

## 核心话术解析：“谁先动心谁是狗” 🐕

对话以“谁先动心谁是狗”这句话展开。这句话是一个极其有效的策略，其作用是多层面的。

以下是这句话的三个核心作用：

1.  **隐藏自身需求感**：通过将“动心”定义为一场竞赛的失败，主动将自己置于看似被动或不在乎的位置，从而避免暴露过强的目的性。
2.  **为对方植入心锚**：在对方心中种下一个“不能先动心”的规则，引导对方的行为和思维框架。
3.  **将互动游戏化**：在两人之间设置了一场“谁先动心谁就输”的游戏，这能有效激发女生的好奇心和胜负欲，让平淡的交流变得更有张力。

然而，需要注意的是，这个技巧如今已被广泛使用，其效果可能因此打折扣。

## 高情商应对“借钱”考验 💰🛡️

当童锦程以“有难处”为借口联系呆妹时，呆妹的回应堪称教科书级别。她没有直接揭穿，而是顺势设置了一个难题。

以下是呆妹回应的精妙之处：

*   **预先表明态度**：在对方具体说明“难处”前，呆妹先说“哥们帮你解决的，只要你开口就行啊，我跟你什么关系呢”。这招“先礼后兵”提前亮明了乐于助人的姿态。
*   **制造心理预期**：此回应会让对方（及观众）隐约感觉“难处”可能是借钱，从而为后续互动埋下伏笔。
*   **提供防借钱技巧**：即便后续需要拒绝，因为已有“慷慨”表态在前，双方关系也不至于尴尬。这是一个实用的社交防护技巧。

当童锦程真的提出“先借个三万吧”时，呆妹的临场反应展现了高情商。

1.  **直播情境的利用**：童锦程听到呆妹在直播后，立刻改口说“该借”，并试图将对话拉回正轨。这是因为在公开直播中“借钱”多半是节目效果，他的改口既能配合氛围，又能彰显自己大方。
2.  **快速关系修复**：在呆妹抱怨“你终于把我拉黑干净了”时，童锦程立刻用“呆妹老婆”的亲昵称呼进行安抚，有效修补了因“拉黑”可能产生的情感裂痕。

## 设立框架与反制：关于“纯真爱情”的博弈 💑⚔️

在关系修复后，童锦程顺势抛出了一个关键框架：“我这一直在找一个纯真的爱情，没有经济利益的。”

*   **核心要点**：这句话必须以开玩笑的口吻说出。如果过于严肃，会让对方觉得你吝啬或算计。
*   **战略价值**：一旦这句话成功传达，就等于设立了一个“我们的关系不应掺杂金钱”的隐形规则，对方日后便难以随意向你索取财物，因为她会明白你对此有清晰的边界。

对此，呆妹进行了精彩的反制。她运用了心理学中的“锚定效应”。

*   **操作手法**：她先连续开出“3万”、“2万5”、“2万”的高价，最后突然抛出一个极低的“1万5”。这种从高到低的巨大落差，会让人更容易接受最后的报价，这是销售中常用的策略。
*   **对话还原**：
    > “你给我转3万，我就可以给你纯真的爱情。”
    > “你给我转2万5……我就可以给你纯真的爱情。”
    > “我给你转1万5，我就可以给你纯真的爱情。”
    > “那你转给我吧。”

最终，童锦程以“挂断语音”的方式结束了这一轮博弈，并评论“他如果说真的不喜欢，他就不会直接把我拉黑了”，这体现了对对方行为背后心理的洞察。
