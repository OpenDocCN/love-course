# 【情商思维提升】最牛的人一定是没有感情的 - P1 - 斌哥恋爱学 - BV1c14y1n7zA

## 概述
在本节课中，我们将探讨情商思维在商业成功中的重要性，特别是如何运用非情感化的思维方式来达成目标。

## 核心概念
- **棋子思维**：将每个人视为棋子，根据其能力进行利用。
- **层次思维**：理解并运用层次思维，区分不同人的能力和价值。
- **猎杀心态**：将客户和员工视为猎物，追求利益最大化。
- **帮助心态**：以成就员工和客户为目标，实现双赢。

## 教程内容

### 1. 棋子思维
**公式**：将每个人视为棋子，根据其能力进行利用。

**内容**：
如果你想要赚大钱，不要把人当人看，把每个人都看作一个棋子。这是最牛逼的人所采用的思维方式，他们会利用感情来操纵他人。

### 2. 层次思维
**公式**：理解并运用层次思维，区分不同人的能力和价值。

**内容**：
层次比较高的人能够理解这种思维方式。大家都是棋子，有能力就上，没能力就下去。创业时，请务必克服这种心理障碍，因为很多人过不了这一关，他们还活在那贫穷的环境中。

### 3. 猎杀心态
**公式**：将客户和员工视为猎物，追求利益最大化。

**内容**：
很多人理解不了这种东西，比如“顾客就是上帝”。这句话的道理很深，有些话在不同的场景中，说出来的目的是不一样的。如果你把顾客当上帝，那这个生意还做不做呢？这里有两种思路：一种是猎杀心态，一种是帮助心态。

### 4. 帮助心态
**公式**：以成就员工和客户为目标，实现双赢。

**内容**：
帮助心态就是把客户和员工当作猎物，但需要非常聪明。很多成功的商人都是这样的，他们把很多人当作猎物，可以随时猎杀掉。但现在我们讲双赢，就是你随时都要想到成就你的员工和客户，用这种心态去做事，你的言行很容易感动人，他们会真心诚意地为你付出。

### 5. 性格与策略
**公式**：根据你的性格选择合适的方法。

**内容**：
你是哪种性格，就选择哪种方法。如果你要帮助别人，成就别人，但心里又很难过，那你的内心境界还没有达到这个层次，就别玩这种套路。如果你是自私自利的，心狠手辣的，其实你也可以当老板，像老虎一样的老板。

## 总结
本节课中我们一起学习了情商思维在商业成功中的重要性，特别是如何运用非情感化的思维方式来达成目标。希望这些内容能够帮助你更好地理解并运用情商思维，实现个人和事业的成长。
