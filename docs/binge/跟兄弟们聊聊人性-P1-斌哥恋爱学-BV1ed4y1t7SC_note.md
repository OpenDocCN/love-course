# 跟兄弟们聊聊人性 - P1 - 斌哥恋爱学 - BV1ed4y1t7SC

## 概述
在本节课中，我们将探讨斌哥关于人性的观点，特别是关于金钱和亲情的关系。

## 核心概念
**公式**：金钱 = 权力，亲情 = 关系

## 教程内容

### 1. 金钱的重要性
**斌哥说**：等你们年龄大了，你们就知道，就是你荷包里面的钱有用，然后再有用的呢，就是你家里的有血缘关系的那几个人。

**解释**：随着年龄的增长，人们会逐渐认识到金钱的重要性，尤其是在关键时刻，金钱可以带来安全感和便利。

### 2. 亲情的力量
**斌哥说**：知道了，你外面的那些朋友都没有用了，然后我再跟你们讲几句更狠的话。

**解释**：亲情是人生中不可或缺的一部分，尤其是在困难时刻，家人的支持尤为重要。

### 3. 金钱与亲情的关系
**斌哥说**：等你到了年龄大了之后，你如果没有钱，你没有赚钱的能力，那跟你有血缘关系的那几个人，他们也不太喜欢你。

**解释**：金钱可以影响人际关系，包括亲情。如果一个人没有经济能力，可能会受到亲人的冷落。

### 4. 亲情与金钱的平衡
**斌哥说**：你们现在还年轻，我知道你们会觉得我养了儿子，我养了女儿，很可靠，可靠得很。

**解释**：在年轻时，人们可能会依赖父母的经济支持，但随着年龄的增长，应该学会独立，并承担起自己的责任。

### 5. 人性的黑暗面
**斌哥说**：我告诉你，你要把我这几句话听懂，你现在赚的钱再多，等你到了年龄大了之后，你会哭的。

**解释**：斌哥提醒人们，金钱并不能带来永恒的幸福，而亲情和人际关系才是最重要的。

### 6. 亲情与金钱的取舍
**斌哥说**：那你还是听不懂我说的什么，你知道吗，你并不明白这些道理，你以后倒霉，那跟我没有屁的关系。

**解释**：斌哥强调，人们应该学会在金钱和亲情之间找到平衡，不要过分追求金钱而忽视了亲情。

### 7. 亲情与金钱的实践
**斌哥说**：我早就告诉过你们，你们非要不相信，对不对，你非要去相信你那些所谓的感情，所谓的亲戚朋友。

**解释**：斌哥提醒人们，要警惕人性的黑暗面，不要过分依赖所谓的感情和亲戚朋友。

### 8. 亲情与金钱的总结
**斌哥说**：但是我告诉你，这就是人性，你总觉得那种不孝顺的儿子女儿，不是别人家的。

**解释**：斌哥指出，人性的黑暗面无处不在，我们应该学会面对和应对。
