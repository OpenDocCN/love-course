# 恋爱吸引力法则教程 - 第1课：核心能量法则 🔮

在本节课中，我们将要学习恋爱吸引力法则中几条关键的、反直觉的定律。这些定律的核心在于个人能量与磁场，掌握它们有助于你理解并主导情感关系。

上一段我们提到了吸引力法则的基础是个人能量，本节中我们来看看几条具体的反人性定律。

第一，你越清醒，你的内在核心越稳定，你的磁场中心就越强。对方就越容易对你着迷。

第二，你的情绪越稳定，你发出的“电磁波”就越稳定、越有秩序。对方反而会因此变得患得患失。

第三，如果你的磁场发生偏离，越在意对方，对方就可能越冷漠。你会从一个能量的“吸食者”转变为能量的“供养者”。

第四，你越爱自己，就越能吸收对方身上的能量，对方反而会越爱你。所有关系匹配的本质，都是能量的相互吸引。让人离不开你的是你自身的高能量特质，而不是你拱手奉上、用于投喂别人的能量。

你的每一次讨好、顺从和迁就，都相当于从你自己的电磁场中分出一部分能量给了别人。

---

理解了核心定律后，以下是几条具体的提升吸引力的方法，你一定要记下来。

第一，情感高手永远只筛选，不做改变。因为改变他人是一件极度消耗自身能量的行为，甚至可能改变因果，让自己背负不好的业力。强者不会对他人抱有不合理的期待，不把自己是否开心建立在他人之上，如此便能掌控一段关系的主动权。

第二，情感高手懂得在感情中保持“光环效应”。因为他们经营的根本不是感情，而是能量。所以他们从不遮盖自己的才华与优点，让对方对自己时刻保持高度的兴趣与暧昧。长久的亲密关系是靠吸引来的。

第三，好的感情不是刻意逢迎，更不是苦心经营。只要你自身的能量足够强大，就能吸引许多人围绕在你身边。此时的你就像一个太阳，当你无限量地增加自身能量和吸引力时，别人就会主动将能量奉上。你的能量越强，吸引力就越强，也就会越来越得到异性的青睐。

---

本节课中我们一起学习了恋爱吸引力法则的核心：**个人能量磁场**。关键点在于保持自我内核稳定、情绪稳定，并专注于提升自身能量（爱自己），而非向外索取或供养能量（讨好他人）。通过筛选而非改变、保持光环效应、以及成为高能量个体，你就能从根本上提升自己的吸引力。
