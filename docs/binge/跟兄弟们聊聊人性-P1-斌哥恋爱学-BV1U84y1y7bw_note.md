# 跟兄弟们聊聊人性 - P1 - 斌哥恋爱学 📚

在本节课中，我们将探讨一个现实而深刻的话题：人性、金钱与亲情的关系。我们将剖析个人经济能力如何影响你与周围人，包括至亲之间的互动，并理解专注于提升自身价值的重要性。

## 核心观点阐述 💡

上一段我们介绍了本节课的主题，本节中我们来具体看看其中的核心论述。

随着年龄增长，你会认识到，你钱包里的钱非常有用。
此外，真正有用的还有你家里那些与你有血缘关系的少数几个人。
你外面的那些朋友最终都没有太大用处。

## 关于亲情的现实考量 👨👩👧👦

上一节我们提到了血缘关系的重要性，本节中我们来看看关于这一点的更深入阐述。

你听了可能不相信。
当你年龄大了之后，如果你没有钱，也没有赚钱的能力，那么即使是与你有血缘关系的那几个人，他们也可能不太喜欢你。
你们现在可能不相信，因为你们还年轻。
你们可能会觉得，自己养育的儿子或女儿非常可靠。
各位不要认为这是危言耸听。
你要真正理解我这些话。
你现在赚的钱再多，等你年龄大了之后，你可能会后悔。
你们如果明白，年龄大了也可以很幸福。

上一部分谈论了亲情与金钱的联系，本节中我们回应一些可能的误解并重申核心。

## 行动指南与总结 🎯

前面我们讨论了人性与现实的诸多方面，最后我们来看看具体的行动建议。

以下是给你的核心建议：
*   **专注赚钱**：老老实实赚钱吧。
*   **精深一事**：人只干一件事，把一件事做透彻。
*   **自然获得尊重**：当你做到以上两点，你到哪里都会有人车接车送。
*   **赢得主动权**：你到哪里别人都会把饭做好了请你吃，到哪里别人都会听你的话。

当你自身强大、拥有价值时，很多事情都会变得简单，包括恋爱。
