# **斌哥恋爱学 - P1 - 看人不要看容貌，而是看神态**

## 概述
在本节课中，我们将学习斌哥恋爱学中的“看人不要看容貌，而是看神态”这一重要观点。

## 教程内容

### 1. 準到嚇人的六個死人數
以下是斌哥总结的六个观察人的要点：

1. **行为和语言相女性化的男生**：这类人多半是市井之徒，可能有些小气，斤斤計較。
2. **说话伤人者**：并非因为情伤低，而是不尊重对方。
3. **重要场合下的表现**：在重要的人面前，一个人的真实面目会暴露出来。
4. **过度完美的人**：不能做朋友，因为他们没有把你当作自己人。
5. **单身很久的人**：一般比较老实。
6. **识人的关键**：不是长相，而是神态。

### 2. 神态的重要性
以下是关于神态的详细解释：

- **貌美神态差的**：这类人容易随波逐流，难如意。
- **貌丑而神态安详中气十足者**：这类人不可小觑。
- **貌丑而神态失和心绪难平者**：这类人容易随波逐流，难如意。

### 3. 衣着与神态的关系
以下是关于衣着与神态关系的解释：

- **随便穿穿的人**：在规则之上。
- **衣冠楚楚的人**：在规则之下。
- **有钱人**：穿得越舒适，看上去越随意。
- **没钱人**：穿得越谨慎，连丝巾都不落下。

## 总结
本节课中我们一起学习了斌哥恋爱学中的“看人不要看容貌，而是看神态”这一重要观点。通过观察一个人的行为、语言、神态和衣着，我们可以更好地了解一个人。
