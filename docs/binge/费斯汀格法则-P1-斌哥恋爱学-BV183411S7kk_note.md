## 费斯汀格法则简介
费斯汀格法则指出，在生活中的10%不可控事件中，有90%的事件是我们可以通过选择去掌控的。

## 案例分析
**案例**：当你和女朋友吵架时，你砸了家里的杯子，导致母亲受伤，你迁怒于女朋友，女朋友感到委屈，并告诉了闺蜜。闺蜜安慰你，并建议晚上一起去喝酒，第二天再好好谈谈。你发现女朋友去了酒吧，告诉了母亲，母亲认为你受伤时她还出去，最终你们分手。

**分析**：
1. **不可控事件**：吵架本身是不可控的。
2. **可控事件**：砸杯子、迁怒、女朋友告诉闺蜜、闺蜜安慰、你告诉母亲等。

## 费斯汀格法则应用
**公式**：不可控事件 × 10% + 可控事件 × 90% = 生活

**代码**：
```python
# 定义不可控事件和可控事件
uncontrollable_event = 10
controllable_event = 90

# 计算生活
life = uncontrollable_event * 0.1 + controllable_event * 0.9
print("生活 = {}%不可控事件 + {}%可控事件".format(uncontrollable_event, controllable_event))
```
