## 核心逻辑：两种价值的交换

*   **繁衍价值**：指与吸引异性、传递基因相关的特质与资源，如外貌、健康、魅力、社交地位等。
*   **生存价值**：指保障个体生存与生活的资源与能力，如金钱、财富、权力、生存技能等。

## 三种价值交换模式

理解了核心概念后，我们来看看人们如何操作这两种价值的交换。根据交换方式的不同，可以分为三个层级。
