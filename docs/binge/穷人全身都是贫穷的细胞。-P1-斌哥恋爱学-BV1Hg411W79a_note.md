# 穷人全身都是贫穷的细胞。-P1-斌哥恋爱学-BV1Hg411W79a_note

在本节课中，我们将要学习一个核心概念：“贫穷的细胞”。这个概念旨在帮助我们理解，为何有些人似乎永远无法摆脱经济与生活的困境。我们将剖析其背后的思维与行为模式，并探讨如何避免自身陷入这种状态。

## 社会现实与二八法则

上一节我们介绍了课程的主题，本节中我们来看看我们所处的社会现实。

你还爱这个世界吗？你还相信这个世界吗？这个社会存在一个二八法则。**80%的人**在社会上始终处于被动的境地。**80%的人**一生都可能买不起房、看不起病、结不起婚、生不起小孩、养不起家庭。

他们甚至在父母生病时，都无力承担医疗费用。这就是活得非常痛苦的一群人。

## 如何定义“穷人”？

了解了宏观的社会结构后，我们接下来具体定义一下我们所讨论的“穷人”。

我们对穷人的定义很简单：**穷人就是懒惰的人**。他们不愿意付出努力，缺乏耐心，做事不踏实。例如，有些兄弟经济拮据，跑来询问如何赚钱。

但当你给他具体的项目去做时，他并不会行动。相反，他可能会直接开始辱骂和批评。这就是典型的穷人思维。我们遇到这样的人，通常选择直接拉黑。

## 核心：贫穷的细胞

那么，穷人赚不到钱的根本原因是什么呢？其本质原因在于：

**他全身都是“贫穷的细胞”。**

这是一种根植于思维和行为习惯的贫困模式，它阻碍了任何改变的可能。

## 我们的应对策略

认识到这种模式后，我们应该采取何种态度呢？

对于我们而言，目标是跳过这个层面的人。我们不应与他们一般见识。我们认为这样的人是不可理喻的，纠缠其中只会消耗我们自身的能量。

---

**本节课总结**

本节课中，我们一起学习了“贫穷的细胞”这一概念。我们认识到，经济上的贫困往往与懒惰、缺乏耐心和不切实际的思维模式紧密相连。真正的改变始于识别并远离这些“贫穷的细胞”，将精力专注于自我提升与切实的行动之上。
