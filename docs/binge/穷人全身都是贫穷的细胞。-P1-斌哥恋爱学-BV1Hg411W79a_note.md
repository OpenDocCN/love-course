## 概述
在本节课中，我们将探讨斌哥恋爱学中关于“穷人全身都是贫穷的细胞”这一观点，并分析其背后的社会现象和核心概念。

## 社会现象分析
### 二八法则
这个社会存在一个二八法则，即80%的人在社会上处于被蹂躏的地位，他们一辈子都买不起房、看不起病、结不起婚、生不起小孩、养不起老婆。甚至当父母生病时，他们都无法给予足够的金钱支持。

### 穷人的定义
**公式**：穷人 = 懒惰的人 + 不愿意付出 + 没有耐心 + 不老实

例如，有些兄弟虽然很穷，但当他们寻求赚钱的方法时，却不愿意付出努力，甚至开始抱怨和批评。这样的人通常被认为是典型的穷人。

## 核心概念
### 穷人赚不到钱的本质原因
**公式**：穷人赚不到钱 = 全身都是穷的细胞

这意味着，穷人之所以无法赚钱，是因为他们内心深处存在着懒惰、不愿意付出、没有耐心和不老实等负面特质。

## 如何应对穷人
### 跳过这个层面的人
我们不应该与穷人一般见识，而是要跳过这个层面，避免与他们产生不必要的争执。

### 不可理喻的人
对于那些不可理喻的穷人，我们应该保持距离，避免与他们产生冲突。
