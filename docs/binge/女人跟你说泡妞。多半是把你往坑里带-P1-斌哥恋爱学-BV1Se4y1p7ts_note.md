```

```
其中，**硬价值**包括经济能力、社会地位等；**软价值**包括情商、领导力等；**自我框架强度**指坚持自身原则和边界的能力。

*   **主导思维**：关注你能创造什么价值、带来什么体验，而不是不断猜测和满足对方的所有需求。

你就永遠不會吃虧和傷了。
