# 跟女人相处要有无所谓心态 🧘‍♂️ - P1 - 斌哥恋爱学

在本节课中，我们将要学习一种与女性相处时至关重要的心态——“无所谓”心态。掌握这种心态，能帮助你摆脱患得患失的情绪，提升个人魅力和心理素质。

## 摒弃无效思考 🤔

上一节我们介绍了课程主题，本节中我们来看看需要摒弃哪些错误思维。

与女性相处时，你需要具备无所谓的心态。你不应去思考“有没有戏”、“她懂不懂”这类问题。你也不应总是询问女性到底是怎么想的。

这些都是缺乏恋爱经验，甚至可以说智力上有些欠缺的人，喜欢纠结的问题。

## 聚焦行动，创造事实 🚀

了解了错误思维后，我们来看看聪明人的做法是什么。

聪明的人会直接去创造事实。例如，如果你想约会，就先发出邀请再说。成大事者不纠结。真正做事的人没有那么多时间和精力浪费在纠结上，他们都是先行动了再说。

此外，我们的一些兄弟总是把这个女人看得过重。

## 核心心态：无所谓 💡

聚焦行动是方法，而“无所谓”则是支撑这一切的核心心态。

我告诉你们三个字：**无所谓**。当你对什么事情都感到无所谓时，你的情商、心态和心理素质都会变得很高。你越在意，就越得不到。因此，不要去在意。

没有什么大不了的，兄弟们。
