## 信息的价值密度 💎

语言的价值可以通过其“信息密度”来衡量。一条信息包含的有效价值越高，其吸引力越强。

我们可以用以下方式理解：

**高价值信息** = 包含情感、观点、邀约或价值的文字
**低价值信息** = 空洞、无实质内容的寒暄

例如，当一个男人发信息只说“你好”，我们就能判断他可能缺乏社交价值。因为这条信息是空洞的，没有提供任何情绪价值或实质信息。

富人或高价值者的每一句话都力求有价值，他们不浪费时间。那些没有价值的表达方式，会在长期的社会经验中被逐渐淘汰出他们的语言系统。

## 掌握系统的关键 🔑

理解了语言系统代表阶层、思维决定语言、以及信息需要价值密度这些核心概念，你就能掌握强大的吸引力工具。持续优化你的表达，淘汰低价值话语，你就能变得特别厉害。
