## 低情商者的核心特征：热衷争论

上一段我们提到了课程主题，本节中我们来看看低情商者的一个普遍行为模式。

情商低的人都有一个特征，那就是喜欢去争。他们常常陷入争论，但争到的往往都是些没用的东西。

## 反思“胜利”的真正价值

那么，你赢了又如何？你们要听懂我这句话。很多人比较短视，没有深入思考“胜利”带来的实际后果。

你需要思考一下这个问题：你通过争论想要获得的那个东西，真的值得吗？

以下是关于“争论胜利”的几个思考角度：

*   **代价高昂的胜利**：例如，我跟我的女朋友吵架，我赢了又如何？**胜利的结果 = 她生气 + 她拒绝合作**。
*   **真正的目标是什么**：赢了就是她生气，赢了她就不跟我做我想做的事情了。
*   **策略性让步**：那么就应该让她赢。我只需要在一个关键点上赢下来，那就是高手。

## 成为情感高手的思维转换

你要想透彻这一点，你要想透彻你到底想要什么。这需要将思维从“赢得辩论”转向“实现最终目标”。

想透彻哪些东西你不应该去争论，因为争论了就没有意义。这要求我们区分**原则问题**与**无谓琐事**，并在大多数非核心问题上保持灵活与包容。
