# **斌哥恋爱学 - 情商低者争斗心理解析**

## 概述
在本节课中，我们将探讨情商低者倾向于争斗的心理特征，并分析争斗的无效性以及如何避免无意义的争论。

## 核心概念
**公式**：情商低者 → 倾向于争斗 → 争斗无效 → 无意义争论

## 争斗心理特征
### 以下是争斗心理特征：
- 情商低者喜欢争斗。
- 争斗往往得到无用的结果。
- 争斗可能导致对方生气，从而影响双方关系。

## 争斗的无效性
### 以下是争斗无效性的分析：
- 争斗可能导致对方生气，从而影响双方关系。
- 争斗无法解决问题，反而可能加剧矛盾。

## 如何避免无意义的争论
### 以下是避免无意义争论的建议：
- 想透哪些事情不值得争论。
- 在争论前思考争斗的潜在后果。
- 尝试从对方的角度理解问题。

## 总结
本节课中我们一起学习了情商低者争斗心理的特征及其无效性，以及如何避免无意义的争论。希望这些内容能帮助你更好地处理人际关系。
