# 恋爱心理学 01：识别情感操控中的“成瘾”陷阱 🔍

在本节课中，我们将剖析一种特定的情感互动模式，了解它是如何通过一系列心理步骤，让人在关系中逐渐“上瘾”并难以自拔的。我们将拆解其核心的三个触发条件，帮助你识别并理解背后的心理机制。

## 触发条件一：示弱以获取掌控感

上一段我们概述了课程目标，本节中我们来看看第一个关键步骤。在相处过程中，对方会有意地让自己处于弱势的一方。许多缺乏经验的人难以察觉这一点，因为他们正渴望与对方建立关系。当对方表现得非常脆弱时，他们反而会感到得意。

这样做的核心目的是：**让你觉得她可以被轻易掌控**。面对比你强的人，你会自然警觉；而面对比你弱的人，你则容易放下戒心。当你确信自己已经完全掌控局面时，就会为下一个条件埋下伏笔。

## 触发条件二：制造失控与投入

当你认为一切尽在掌握时，第二个条件便被触发——让你体验失控。对方会突然提出分手或开始保持距离，采取冷暴力。此时你会产生疑问：“她明明比我弱，为何主动提分手？” 你自信能够挽回她。

为了重新获得那种掌控感，你会开始对她投入更多，甚至可能放弃原有的底线。以下是这一阶段常见的表现：

*   不断加大情感、时间或物质上的投入。
*   对方却对你保持忽冷忽热的态度。
*   你逐渐进入一种自我怀疑的状态。

这种不断追加“沉没成本”的过程，为最终的上瘾铺平了道路。

## 触发条件三：给予奖励并固化成瘾

经历了自我怀疑和持续投入后，第三个条件将触发，使你彻底上瘾。当你用尽全部真诚去挽留，对方会故意回到你身边。这时，你重新获得了掌控感。

这种“失而复得”的快乐，对大脑而言是一种强烈的**奖励机制**。其心理公式可以简化为：**“失控焦虑 -> 加倍投入 -> 重获关系 -> 获得快感”**。你会迷恋上这种感觉。

经过几次这样的“推拉”循环后，你就会对这段关系彻底上瘾。此时，无论身边有多少人劝阻，几乎都难以产生效果。

## 课程总结

本节课我们一起学习了情感操控中诱导“成瘾”的三个核心阶段：从**示弱建立虚假掌控感**，到**制造失控迫使你加倍投入**，最后通过**给予复合作为奖励**来固化这种成瘾循环。理解这些步骤是识别不健康关系模式、保护自身情感健康的第一步。
