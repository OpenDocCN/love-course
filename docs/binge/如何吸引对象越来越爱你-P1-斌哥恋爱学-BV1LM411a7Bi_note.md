# 如何吸引对象越来越爱你 - P1 - 斌哥恋爱学 - BV1LM411a7Bi

## 概述
在本节课中，我们将学习如何通过改变自己的心态和行为来吸引对象，使其越来越爱你。

## 第一部分：忘掉对对方的期待
**核心概念**：忘掉对对方的期待，因为期待意味着对方不符合你的理想形象，你需要通过自己的努力去创造理想的关系。

- 忘掉对对方的期待，因为你在向对方提要求。
- 期待证明对方不是你想要的样子。
- 你需要靠自己去创造一段关系。

## 第二部分：不要抱怨
**核心概念**：不要抱怨，因为抱怨会潜移默化地影响对方，使其变成你抱怨的样子。

- 不要抱怨你现在的生活状态。
- 抱怨是强大的咒语，会变成现实。
- 如果抱怨无法解决，尝试改变自己。

## 第三部分：大胆想象
**核心概念**：大胆想象你渴望的关系，并保持这种想象。

- 大胆想象你渴望的关系是什么样子。
- 让思想和感觉保持在拥有这种关系的状态中。
- 清楚地知道你想要的到底是什么。

## 第四部分：吸引你想要的关系
**核心概念**：不需要解决眼前的问题，只需要吸引你想要的关系。

- 不需要解决眼前的人或复杂的事情。
- 吸引你想要的关系，命运会安排符合你标准的人。
- 恋爱关系只是一场合作，体验比圆满更重要。

## 第五部分：培养爱的能力
**核心概念**：提升自己的能量和思维，培养爱的能力。

- 人与人之间的交往是能量的覆盖。
- 提升能量和思维的方法。

## 总结
本节课中我们一起学习了如何通过改变自己的心态和行为来吸引对象，使其越来越爱你。希望你能将这些方法应用到实际生活中，找到属于你的幸福。
