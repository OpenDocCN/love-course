# 【赚钱和泡妞思维】男人如何拥有大格局？ - P1 - 斌哥恋爱学 - BV1kD4y1s7LE

## 概述
在本节课中，我们将探讨如何通过增加个人财富和资源来提升个人格局。

## 核心概念
**公式**：格局 = 财富 / 资源

### 1. 财富与格局的关系
如果你的口袋里有1万块钱，损失1万会非常难过。拥有100万时，损失1万仍会难过，但不会特别难过。若拥有1000万，损失1万则毫不在意。这说明，随着财富的增加，对金钱的敏感度降低，格局也随之提升。

### 2. 资源与格局的关系
如果你活了20几年，手上只有一个妹子，分手会生不如死。若谈了10个女生，分手后可能难过半天，但很快恢复。若手上有50个妹子，分手后心中会暗爽。这说明，资源越多，对失去的承受能力越强，格局越大。

### 3. 人性与格局
**代码**：痛苦 = 财富 / 资源
快乐 = 资源 / 财富

痛苦源于财富和资源的匮乏，快乐则源于资源的丰富。因此，要提升格局，就要努力搞钱搞资源。

## 总结
本节课中我们一起学习了如何通过增加个人财富和资源来提升个人格局。希望你能从中获得启发，努力提升自己。
