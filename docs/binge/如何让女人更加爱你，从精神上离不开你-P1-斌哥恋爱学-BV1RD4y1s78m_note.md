## 核心二：利人性夸赞与安抚 🤝

了解了情绪稳定的表现后，我们进入第二个核心技巧：利人性的夸赞与安抚。这要求你采取主动的低姿态。正确的夸赞应以自我为中心。

具体方法是：少夸她本人（如漂亮、优秀），多夸她对你好的行为。因为她的优秀可能与你无关，而应强调她的行为带给你的感受。

**公式：夸赞 = “你做了[具体行为]，让我感到[你的感受]”**

例如，不说“你真漂亮”，而说“你今天特意为我下厨，让我觉得特别幸福”。

最后，我们探讨第三个核心技巧：拿捏式推拉。这指的是在关系中主动创造张力和吸引力，让她的情绪因你而波动。

例如，在你们最热恋的时候，你突然暂时抽离，独自去旅行，这可能会让对方更加在意你。高段位的男人能让女人又哭又笑，对她既纠结又让她觉得自己特别爱你。

实施此技巧需坚守一个原则：**舍得让你爱的人吃“心理上的苦”**。

以下是执行此原则的三个要点：

1.  **设立反馈机制**：当对方行为出格时，你必须给予有原则的反饋，绝不姑息。
2.  **果断决策**：如果真到了该分手的时候，绝不拖拉。不进行“精准扶贫”，不抱有侥幸心理。
3.  **及时止损**：这是最重要的行动准则。如果你为了爱不顾一切，失去标准和原则，那么最终的结果也需要自己承担。
