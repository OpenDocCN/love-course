上一节我们介绍了课程的主题，本节中我们来看看核心观点的具体阐述。

兄弟千万不要去在意女人的信用跟承诺。女人是感性思维。例如，我相处了一个妹子，我给她几千块钱，我让她去给我买一个东西，她没有买，她还把我这几千块钱给黑了，变成她自己用了。或者说她答应了你的约会，过几天她就反悔了，这很正常。

你不要去生气。不讲信用就不讲信用，你没有办法改变她一个多年的习惯。每一个人都会付出代价，每一个人都会为她的错误而买单。不过这就是更高级的智慧了。

她拿走你的钱，她一定会为此付出代价。不管是在你这个地方付出代价，还是在她的人生中付出代价，这都无所谓。你付出的金钱看似亏了，实际上呢你学会了很多经验。你知道下一次呢应该怎么样子去应付了。

我其实跟这个女人在一起之后，我就学会了很多东西。例如，我学会了容忍她的错误，我去装傻，我不去撕破脸皮损失自己的利益。

我以前跟很多兄弟在一起的时候，我老是喜欢去改变他们，管理他们。当我跟这个女人谈过恋爱之后，我刚开始也是一个这样的思维，去跟女人沟通想去改变她，顺便呢去讲一下自己的道理。

随着我的阅历越来越丰富，我已经不这样去做了。她骗我的钱我就笑一笑，当面或者背地里面欺骗我，我也笑一笑，或者呢我就装傻。她不讲信用，你也微笑就行了。

而女人她有一个特征，就是她明明犯了错，她还可以跟你吵架。大家是不是想不通？她想把这个错误放在你的头上。你也不用去搭理她，让她放肆，让她的这个心里舒服。这个就是你跟这个女人相处，要付出的一个代价。任何人都是有代价的，这就是高明的男人。

我有一段时间经常去惯着女人，把她惯出一堆毛病来。这一堆毛病呢，只有在我这个地方才能让她生存，才能被容忍。在其他地方是没有办法生存的，也没有人惯着她，对不对？那么她就离不开你了。

女人有时候像小孩子，她的这个行为逻辑，智力就跟几岁的小孩子一样。她根本不知道这个社会有多残酷，所以说你去保护她。

其实我刚开始讲的一段话，是非常重要的，就是看你们能够领悟到多少了。
