## 语言艺术：一句话完成推拉 💬

在心理博弈的基础上，高超的语言技巧能让效果倍增。

面对高质量的女生，策略是**先推后拉**。“推”是制造轻微的距离或否定，“拉”是表达欣赏或靠近。高手能在一句话中同时完成这两步。

例如，评价女生“长得就还行吧，满分100分的话大概98分”。这句话里：
*   **言语语气（推）**：像是在调侃或挑剔（“就还行吧”）。
*   **语言内容（拉）**：实际上给出了极高的评价（“98分”）。

这种组合会同时触发对方大脑的**直觉系统**（感受到被怼）和**理性系统**（意识到被夸），制造出强烈的情绪波动和吸引力。

**代码示例：**
```python
# 推拉语句结构
def push_pull_sentence():
    push = "言语上的轻微否定或调侃"  # 例如：“你长得就还行吧”
    pull = "内容上的高度赞赏或肯定"  # 例如：“满分100分你能拿98分”
    return push + "，" + pull  # 组合输出
```
