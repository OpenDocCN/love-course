# 出轨学 - P1 - 斌哥恋爱学 📊

在本节课中，我们将基于真实调查数据与逻辑分析，探讨影响婚内出轨行为的十二个关键变量。课程旨在帮助大家理解出轨现象背后的数据规律，内容简单直白，适合初学者。

---

## 概述 📋

本节课我们将学习一份2003年的调查数据，该数据显示，当时中国男性的婚内出轨率为20.9%，女性为5.6%。这一数据符合生物学上关于男性天性的部分定义。我们将从这份涉及多个变量的调查中，提取并分析十二个核心因素。

---

## 十二个关键变量分析

上一节我们概述了课程目标与基础数据，本节中我们来看看具体影响出轨率的十二个变量。以下是每个变量的详细分析：

**第一，城乡差距。**
城市男性的出轨率是农村男性的1.62倍。然而，女性的出轨率在城乡之间没有显著差异。

**第二，年龄。**
年龄对出轨率的影响呈倒U形曲线。不论男女，出轨率都会在某个特定年龄达到峰值，随后逐渐下降。

**第三，外表吸引力。**
这一项数据颇具意味。外表帅气的男性，其出轨率比相貌普通的男性高出83%。而长相好看的女性，其出轨率比相貌普通的女性高出478.6%。

**第四，对婚内爱情的满意度。**
爱情满意度每上升一个单位，男性出轨的概率下降35.7%，女性出轨的概率则下降63.9%。

**第五，受教育年限。**
受教育年限对男性出轨概率基本没有影响。但对于女性，每多接受一年教育，其出轨概率上升12.4%。调查中女性平均受教育年限为9.5年（约高中水平）。这意味着，高中文化以上的女性，每多上一年学，出轨概率就增加12.4%。

**第六，社交时间。**
男性的社交时间每增加一个单位，出轨概率提升23.3%。女性方面则呈现相反趋势：出轨率最高的群体是家庭主妇。女性的社交时间越低，出轨率反而越高。无业女性的出轨率是女性打工者的538.4%，自由职业女性的出轨率是女性打工者的207.6%。

**第七，收入。**
男性收入每增加一个单位，出轨概率增加19.9%。

**第八，行业。**
从事第三产业（服务业等）的男性，其出轨率比从事第一产业（农业）和第二产业（工业）的男性高出一倍。

**第九，社会地位。**
拥有高管身份的男性，其出轨概率是普通男性的4.018倍。

**第十，婚内性行为频率。**
该频率每上升一个单位，男性出轨的概率下降17.3%。但此因素对女性出轨率毫无影响。

**第十一，有无子女。**
有子女的男性，其出轨率是无子女男性的1.88倍。然而，有子女的女性，其出轨率仅为无子女女性的33%。

**第十二，对出轨的态度。**
对出轨持开放态度的男性，其出轨率是不持开放态度男性的四倍。但女性的出轨率，基本不受其自身对出轨态度（开放与否）的影响。

## 总结与思考 💡

本节课中，我们一起学习了基于2003年数据的十二个影响出轨率的关键变量。从城乡差距、年龄、外表吸引力，到满意度、教育、社交、收入、行业、地位、性频率、子女状况及个人态度，每个因素都揭示了不同群体在出轨行为上的差异。

数据表明，许多因素对男性和女性的影响截然不同。理解这些规律，有助于我们更理性地看待亲密关系中的潜在风险。在这个信息纷杂的时代，保持清醒的认知并积极经营自己的生活，才是最重要的。
