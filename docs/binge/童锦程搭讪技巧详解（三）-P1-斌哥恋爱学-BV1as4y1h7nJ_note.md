“我知道有个地方能进去……不会是你家吧？”

未完成的事件，比如一支没买到的口红，会成为女生心中的一个念想。她甚至会将这支口红与你进行记忆捆绑，之后很可能会再次向你提起。

“口红店关门了，说明我们没有缘。”

“没事我看你们打完车我就走了。”

同时，要牢记找到合适的机会，就要进行肢体接触。

“什么牌子的口红最贵？”
