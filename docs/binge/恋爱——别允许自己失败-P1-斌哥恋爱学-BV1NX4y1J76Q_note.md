# 恋爱——别允许自己失败 - P1 - 斌哥恋爱学 - BV1NX4y1J76Q

## 概述
在本节课中，我们将探讨恋爱中的失败态度，以及如何培养强者思维以避免失败。

## 核心概念
**公式**：失败 ≠ 成功之母

## 一、失败的本质
很多兄弟在谈恋爱中允许自己失败，这是一种弱者思维。一次感情失败可能意味着权限的崩盘。赚钱也是如此，你以为失败是成功之母，但实际上这只是规律和规则。

## 二、强者思维
你可以尊重他人，但你要成为一个强者。你的潜意识中应形成一套观念：失败从来不是成功之母，失败只会导致更失败，只有成功才能让你更成功。

## 三、时间与努力
一次感情的失败，一次生意的失败，你可能需要花三五年的时间才能填补这个坑。但如果你这次干得很成功，那么下次就会干得更成功。

## 四、不输的心
你一定要有一颗不输的心，这样你才能慢慢赢得胜利。

## 五、失败的原因
很多人过得差，生意失败，感情失败，是因为他们认为输了他无所谓。陈有量没有干过朱元璋，项羽没有干过刘邦，所以他们死了。我们之所以在谈恋爱和赚钱上允许自己失败，是因为我们总觉得自己有退路。

## 六、温水煮青蛙
绝大多数人感觉不到自己是在走向灭亡。我并没有想去贩卖焦虑，你也可以每天活得很快乐。但等你老了，你就会发现即便你焦虑也不再怎么管用了，你只能欣然接受自己的悲惨命运。

## 七、强者思维的重要性
我讲的这个东西是强者思维，你可能会觉得它很虚。但当你想明白了，你就会觉得这个才是真实的，而你现在认为的那些实实在在的东西，可能等你醒悟过来之后，才发现那些才是虚的。

## 总结
本节课中我们一起学习了恋爱中的失败态度以及如何培养强者思维以避免失败。希望你能从中受益，成为一个更强大的自己。
