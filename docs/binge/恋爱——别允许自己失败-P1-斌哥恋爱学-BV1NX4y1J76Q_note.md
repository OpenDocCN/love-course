# 恋爱教程 01：摒弃“允许失败”的弱者思维 💔➡️💪

在本节课中，我们将要学习一种关键的思维转变：在恋爱（以及事业）中，为何必须摒弃“允许自己失败”的心态。我们将探讨这种心态的危害，并理解“强者思维”是如何运作的。

很多人在谈恋爱时，能够允许自己失败。这种心态本质上是弱者思维。一次感情失败，可能意味着全局的崩盘。赚钱也是同样道理。

你以为失败是成功之母。但我要告诉你，这只是规律和规则。你可以尊重它，但你要变成一个强者。你的潜意识里必须形成一套观念：失败从来不是成功之母。

失败只可能导致你更加失败。只有成功才能让你更加成功。一次感情失败，或一次生意失败，你可能要花三五年时间才能填补这个坑。但如果你这次干得很成功，那么你下次就会干得更成功。

因此，你必须有一颗“不输”的心。这样，你慢慢就能赢了。很多人过得差，生意失败、感情失败，就是因为他们认为输了也无所谓。

陈友谅没有战胜朱元璋，项羽没有战胜刘邦，所以陈友谅和项羽死了。我们在谈恋爱和赚钱上允许自己失败，是因为总觉得自己有退路，反正饿不死，对吗？

所以很容易陷入“温水煮青蛙”的境地。绝大多数人感觉不到自己正在走向灭亡。我并非贩卖焦虑。当然，你可以每天活得很快乐，反正脑子里有泡面，有王者峡谷。

但等你老了就会发现，即便你焦虑也不再管用。你只能欣然接受自己悲惨的命运。我讲的是强者思维。你可能会觉得它很虚。

但你以为的“虚”，一旦想明白，你就会觉得这才是真实的。而你当下认为的那些实实在在的东西，可能等你醒悟过来后，才发现那些才是虚的。

---

本节课中，我们一起学习了“强者思维”与“弱者思维”的核心区别。关键在于，必须从潜意识里根除“允许失败”的心态，认识到失败只会导向更多失败，而成功才能孕育成功。在恋爱与人生的道路上，树立“只许成功，不许失败”的决心，是走向胜利的第一步。
