“原来你也穿高跟鞋啊。”“我不懂他穿高跟鞋的。”“他就开着的。”

看到没有，一进门就自然地开启对话。这也说明童在这个时候是比较自然和放松的。当两个人相处时比较轻松自然，这个时候是比较容易擦出恋爱火花的。

**核心概念：自然放松的状态是关系升温的催化剂。**
