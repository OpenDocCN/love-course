# **斌哥恋爱学 - P1 - 这个社会的本质**

## 概述
在本节课中，我们将探讨斌哥关于社会本质的观点，特别是男人在恋爱中的自信和胆量来源。

## 男人的自信和胆量来源
斌哥指出，男人的自信和胆量实际上来源于他们的经济实力。以下是他与一位寻求帮助的兄弟的对话：

- **斌哥**：“你有钱吗？”
- **兄弟**：“没有。”
- **斌哥**：“你有好的工作吗？”
- **兄弟**：“我的工作也不怎么好。”

斌哥解释说，这个女人离开他是正常的，说明她的头脑是清醒的。他强调，如果男人买不起女人，她就不会和他在一起。

## 购买力的概念
斌哥提出了“购买力”的概念，并将其分为以下几类：

- **颜值购买力**
- **金钱购买力**
- **内涵购买力**
- **家庭背景**

## 提升自己的指标
斌哥建议，男人应该努力提升自己的各项指标，包括：

- 锻炼身体
- 多阅读
- 多听斌哥的课程和直播
- 练习演讲
- 穿着得体

通过这些方式，男人可以实现他们想要的一切。

## 内功修炼
斌哥强调，如果男人不去修炼内功，只是来询问如何聊天，这些都是没有用的。

## 成为一个厉害的人
斌哥提到，他的课程不仅适用于男女关系，还可以应用到生活的各个方面。通过学习，人们可以成为一个很厉害的人。

## 总结
本节课中，我们一起学习了斌哥关于社会本质的观点，特别是男人在恋爱中的自信和胆量来源。希望这些观点能够帮助大家更好地理解自己和周围的世界。
