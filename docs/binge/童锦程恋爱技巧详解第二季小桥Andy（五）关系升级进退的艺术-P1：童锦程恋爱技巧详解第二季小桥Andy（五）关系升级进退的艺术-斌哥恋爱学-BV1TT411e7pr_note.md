# 童锦程恋爱技巧详解 第二季 小桥Andy（五）关系升级 进退的艺术 - P1

## 概述
在本节课中，我们将学习童锦程在恋爱关系中如何维护长期关系，以及进退的艺术。

## 关系维护
**搭讪只是恋爱关系中的第一步，长期关系的维护才是关键。**

### 维护技巧
1. **引导对方付出**：高手会引导妹子付出，而不是单纯地花钱。
2. **提出要求**：短时间内连续向妹子提出两个要求，第二个要求往往会在第一个要求被接受后默许。

### 进退策略
1. **反复进攻**：在对方没有发火的情况下，反复进攻，拒绝后稍微退一步，然后继续进攻。
2. **找到机会反击**：在对方拒绝时，找到机会以关爱的方式反击，夺回主动权。

## 心理学技巧
1. **选择心理**：如果妹子想了一下选择了你，说明她懂得照顾你的情绪；如果她直接选择你，说明她心里有你，很喜欢你。
2. **专情测试**：表面上很忠心，实际上制造一个竞争对手。如果表现得很专情，她会觉得你是废物；如果不当废物，她会觉得你是渣男。

## 总结
本节课中我们一起学习了童锦程在恋爱关系中如何维护长期关系，以及进退的艺术。希望这些技巧能帮助你更好地维护恋爱关系。
