上一段我们提到了恋爱中常见的困惑，本节中我们来看看具体有哪些核心原则和方法。

首先，你需要从深层次理解一个概念：**对方爱不爱你，完全取决于她当下的行为反馈**。因为爱意是流动的，它是一个变量。公式可以表示为：
**爱意 = f(当前行为)**
今天可能是满分，明天可能就不及格。当对方对你好时，接受她的好并相信她是爱你的，给予正向反馈。如果对方送你礼物，就大方接受并感谢。当对方赞美你时，说“谢谢你的赞美”。当对方怠慢你时，则相信她此时就是不爱你，并采取相应措施（如沟通、暂时断联等）。关键在于内心不要过于纠结，根据对方的行为“打分”即可。分数高时加分，分数低时减分。许多人的错误在于，感受到爱时胡乱猜疑，受委屈时却又过度讨好。

上一节我们介绍了根据行为反馈调整心态，本节中我们来看看恋爱中的互动本质。

其次，要明白谈恋爱是一个相互影响的过程，看谁能更有效地传达自己的框架。不要盲目相信女朋友口头声称喜欢的类型（例如喜欢为她花钱的、听话的）。你需要进行反向引导。切记不要向现任女友抱怨前女友有多差劲。在她爱你时，她或许会心疼你；但当感情出现问题时，这只会成为她攻击你的理由。真正聪明的男人从不贬低前女友，相反，他会说前女友很好、很漂亮、很会心疼人。因为女性天生带有征服欲和胜负欲，这能激发她的竞争心理。

在理解了相互影响的本质后，我们接下来探讨如何管理情绪。

在感情中，想避免被伤害，就应该把情绪当作自己的武器。谈恋爱不是找一个人发泄情绪，而是要知道什么事情该生气，什么事情不该生气。许多人在小事上斤斤计较、情绪激动，但当伴侣真正犯错时，却因为害怕对方离开而不敢表达立场，变得怯懦。这需要想清楚自己的底线。

除了个人情绪，外部环境的影响也不容忽视。

第三，朋友圈子非常重要。有些人的感情不好，可能源于身边朋友的价值观不正，总是给出馊主意。同时，你女朋友身边的圈子也值得你注意。人都有从众心理，大部分行为会与他人做对比。例如，如果一个女人的闺蜜都很拜金，她即使没有频繁索要礼物，也可能觉得自己已经对你足够好。反之，如果她的闺蜜都是体贴、懂事、老实的女孩，那么她在跟你闹情绪时，内心多少会有些愧疚感。

我们讨论了外部环境，现在来谈谈恋爱的核心驱动力。

女人要的是“感觉”，而不是沉重的“感情”。感情太沉重，会让人想逃离；而感觉则轻松得多。在这点上，你需要自私一点。做你自己想做的事，比如想睡懒觉就睡懒觉，自己开心就好。最坏的结果无非是分手，然后开始新的关系。你可以感谢对方把美好的青春给了你，甚至在她结婚时送上祝福。

最后，我们来探讨关系稳定的根本保障。

男人只要拼命去赚钱，就不必总是担心女人会离开。只要你处于努力赚钱的状态中，让女人感受到你的上进心，聪明的女人自己会去权衡利弊。不要花太多精力去刻意讨好。如果你有赚钱的本事，就强势一点，再对她好一些。如果你暂时赚钱能力不足，就多“画大饼”——经常描绘你们未来的美好画面，并且用切实的行动去创造这个未来。她听得多了，并且看到了你的行动，也会慢慢相信。
