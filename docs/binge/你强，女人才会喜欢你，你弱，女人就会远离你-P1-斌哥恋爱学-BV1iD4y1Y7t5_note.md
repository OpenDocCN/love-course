其实这只是一个表象。重要的是你是否有**人格魅力**，你的信心是不是很足，是不是有**领袖气质**，是不是有**雄性气质**。这就是女人为什么会喜欢那种有钱或长得帅的男人的原因，因为这些外在条件可以让他们展现出显得很强的样子。

*   **雄性气质**：通常指果敢、有担当、有保护力的阳刚特质。
