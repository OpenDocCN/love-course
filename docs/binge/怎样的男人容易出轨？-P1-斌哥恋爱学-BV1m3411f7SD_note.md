## 核心影响因素分析

伴侣是否出轨，主要取决于两个核心因素：**个人特质**与**外部机会**。其关系可以概括为：

**出轨可能性 ≈ 个人出轨特质 × 外部机会**

接下来，我们将逐一分析构成“个人出轨特质”的几个关键方面。

上一节我们概述了核心分析框架，本节中我们来看看具体有哪些个人特质会提高出轨的概率。

以下是基于观察总结的六类易出轨男性特质：

1.  **原生家庭影响**
    如果他的父亲曾有出轨或拥有婚外情人的历史，那么该男性未来出轨的概率会显著增大。家庭环境对个人行为模式有深远影响。

2.  **社交圈层价值导向**
    群体内共同的价值观传导，会使个人的道德观念逐渐模糊。在一个将出轨视作风流韵事、作为朋友间谈资，且无人因此感到羞愧的圈子里，个人久而久之也大概率会认同并效仿这种行为。

3.  **低自我认同与情感泛滥**
    自我认同感低、拒绝能力差且多愁善感的男性，其情感通常较为泛滥，容易见一个爱一个，难以维持稳定专一的情感关系。

4.  **缺乏分析能力与过度单纯**
    这类男性往往缺乏判断力，比较单纯，极易被别有用心者（如俗称的“绿茶”）迷惑并操控。即使旁人提醒，他们也可能会认为提醒者小题大做。

5.  **性别歧视与不尊重女性**
    包括那些脾气暴躁、持有严重大男子主义观念的男性。他们通常认为只有具备挣钱能力才有家庭地位。与这类男性交往需格外谨慎，即便结婚，婚后也极易出现关系破裂。

6.  **富有但闲暇时间过多**
    例如家境富裕，或工作无需投入大量时间、缺乏上进心的男性。充裕的金钱与时间若缺乏正向引导，可能增加寻求婚外刺激的机会。
