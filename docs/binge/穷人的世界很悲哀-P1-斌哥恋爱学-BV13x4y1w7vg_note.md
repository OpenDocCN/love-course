## 经济匮乏者的婚恋困境

一个残酷的事实是，经济条件匮乏者的处境往往很悲哀。为了组建家庭，他需要耗尽自己的积蓄，同时也会耗尽父母的积蓄，甚至可能耗尽祖上三代人的全部积蓄。其典型表现为：支付一套房子的首付，购买一辆汽车后却舍不得加油。**公式**可以简化为：
**个人婚恋总成本 ≈ 个人积蓄 + 父母积蓄 + 家族历史积蓄**

上一节我们提到了经济匮乏者的整体困境，本节中我们来看看这种困境如何具体限制他们的选择范围。

## 选择范围的限制与风险

付出了三代人的代价后，他们在选择伴侣时仍不敢追求外貌条件更出众的对象。因为外貌出众的女性身边通常存在大量追求者。当下的社会流行一种观念：“结婚了又怎样？结婚了不是也可以离婚吗？” 这种观念进一步加剧了婚恋市场的不稳定性。

由此可见，社会现实在某种程度上是为难经济匮乏者的。你越是贫穷，所面临的挑战和欺压就越多。

## 财富作为社会地位的信号

那么，为什么会出现“越穷越被欺负”的现象呢？因为拥有财富通常向外界传递出一个强有力的信号。**核心概念**可以表述为：
**财富 = f(个人能力， 家族实力)**
即，财富是个人智慧、能力或家族实力的函数。这向外界表明你或你的家族很厉害，因此一般人不敢轻易欺负拥有财富的人。这就是社会血淋淋的残酷法则。

理解了财富的信号作用后，我们再来看看它在具体关系中的体现。

## 财富对关系动态的重塑

当你处于经济匮乏状态时，你可能会为某个女人爱得死去活来，并需要付出极高代价。然而，在富裕者的世界里，你会发现与同样女性的互动成本可能极低，甚至在某些语境下是“免费”的。

更进一步的现实是，当你拥有充足财富时，你甚至可以使用女性的金钱来支付共同消费（例如开房），并且她会为此感到高兴。在这种关系动态下，她不敢向你索要房子或车子等重大资产，她只需要确认你具备持续赚钱的能力就足够了。
