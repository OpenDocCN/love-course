# 倪《女神解码》第二单元：辅课程01 - 发型打理篇：美式油头 💇‍♂️

在本节课中，我们将学习如何打理一款经典的美式油头发型。我们将从所需工具开始介绍，逐步讲解造型步骤，帮助你快速掌握打造利落、有光泽发型的技巧。

## 工具准备 🛠️

以下是打造美式油头所需的核心工具清单。

1.  **专用发油**：一款来自澳洲拳击手品牌的经典美式油头发油。它与发蜡的区别在于，发油具有光泽感和湿润度，允许反复造型，而发蜡质地较干，通常只能做一次性定型。
2.  **染色发蜡（奶奶灰）**：一款日本的奶奶灰色染色发蜡。它在塑形的同时，能为头发染上灰色，实现造型与染色的双重效果。
3.  **造型梳**：一把管式折叠梳，用于精确分界和梳理头发，是塑造整齐油头的必备工具。

## 造型步骤详解 ✨

上一节我们准备好了工具，本节中我们来看看具体的造型流程。

### 第一步：吹干并吹立发根

首先，需要将头发吹干，并将发根吹得蓬松立起。这样做是为了方便后续的塑型。如果头发被吹得塌软，后续造型会比较困难。

### 第二步：涂抹发油

接下来，我们使用准备好的发油。取大约一个指节量的发油，在掌心揉搓均匀。然后，将发油均匀地涂抹在头发上，并用手抓出大致的形状。

**核心概念**：本节课使用的发油是**水溶性发油**。其特点是遇水后会溶解，这为后续的精细梳理和调整提供了便利。

### 第三步：精细梳理与定型

涂抹发油后，头发会显得比较油亮。此时，打开管式折叠梳，在梳齿上沾取少量清水。接着，用沾湿的梳子进行精细梳理。例如，塑造一个三七分发型，就沿着分界线仔细梳开、梳顺。

完成以上步骤后，一款光泽有型的美式油头就打理好了。整个过程大约需要十到十五分钟。

## 补充：染色发蜡的使用 🎨

除了基础的发油造型，我们之前还介绍了奶奶灰染色发蜡。它的质地比发油偏干，但在塑形的同时具备染色功能。使用方法与发油类似，均匀涂抹并梳理即可，可以打造出独特的灰发造型。

## 总结 📝

本节课中，我们一起学习了美式油头的打理方法。我们首先认识了**专用发油**、**染色发蜡**和**造型梳**这三样核心工具，并理解了水溶性发油可反复造型的特性。随后，我们逐步实践了**吹立发根**、**涂抹发油**和**沾水梳理**这三个关键步骤。掌握这些技巧，你就能在日常生活中快速打造出帅气有型的发型。
