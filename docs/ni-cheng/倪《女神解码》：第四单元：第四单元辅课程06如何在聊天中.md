# 倪《女神解码》：第四单元：第四单元辅课程06如何在聊天中

在本节课中，我们将要学习一个关键的聊天技巧：如何在与女生互动时，提前植入“预期”。掌握这个方法，能让你们的线下约会更加顺利自然。

## 课程概述

预期的作用是让对方有心理准备。回想以往约会经历，当你尝试靠近或接触对方时，她若表现出抗拒，原因往往是她没有心理准备，觉得你的行为突然，从而感到尴尬或紧张。植入预期就是为了避免这种情况。

## 如何植入预期

植入预期的方法其实很简单。你只需要提前在微信聊天时，从侧面或正面告诉她，见面时可能会发生什么。

例如，如果你原本只说“我们一起去看电影”，她的预期就仅限于看电影。但如果你说“我们手拉手去看电影，坐最后一排”，她的心理预期就会包含“可能会牵手”。

以下是你可以植入的不同层级的预期示例：

*   **初级预期**：例如提到“手拉手”。
*   **高级预期**：例如说“你见到我会害羞吗？”、“你见到我会脸红吗？”、“我怕你见到我会情不自禁”、“如果我抢你女朋友会怎么样？”。

当你表达了这些预期后，如果她仍然愿意出来见面，通常意味着她对你有好感，并且同意了这些潜在互动，同时也有了相应的心理准备。

## 植入预期的前提条件

上一节我们介绍了植入预期的具体方法，本节中我们来看看使用这个技巧的重要前提。

这种预期植入，必须建立在你和她已有良好互动、她已经对你产生吸引的基础之上。只有这样操作，她才可能接受。否则，她可能会直接拒绝你。

## 预期的作用实例

为了让你更直观地理解预期的作用，这里分享一个真实聊天案例。

我曾对一个女孩说：“我们见面时，你得搂着我看电影，不许非礼我。”她回复“哈哈哈”并开了个玩笑。后来我们见面时，她真的搂着我了。这就是提前植入预期所达成的效果。
