# 倪《9.0女神解码》：第三单元：第三单元辅课程07P图攻略_note

在本节课中，我们将学习如何快速美化照片，使其适合在社交平台上展示。教程将分为两个部分：人物照片处理和风景照片处理，使用两款常见的手机应用完成。

## 概述

我们将使用“美图秀秀”和“InstaSize”两款软件，通过简单的步骤，让照片的肤色更均匀、画面更有质感。整个过程注重自然效果，避免过度修饰。

## 人物照片处理

上一节我们介绍了本课的目标，本节中我们来看看如何处理一张人物照片。

首先，打开“美图秀秀”应用。选择“美化图片”功能，而不是“人像美容”。导入需要处理的人物照片。

以下是处理人物照片的核心步骤：
1.  点击“智能优化”选项。
2.  在优化类型中选择“人物”。
3.  完成上述操作后，照片中人物的黑眼圈会减淡，肤色会变得均匀白皙。此时即可保存照片。

接着，打开第二款应用“InstaSize”。导入刚才在美图秀秀中保存好的照片。

这款应用主要用于调整颜色和质感。以下是具体操作：
1.  在滤镜库中，选择一个名为 **Cellsense** 的滤镜，这个滤镜能营造欧美风格色调。
2.  点击“调整”工具，找到“锐度”选项。
3.  将锐度值调整到 **2** 或 **3** 左右。增加适量锐度可以让皮肤看起来更有质感，避免虚假感。

完成这两步后，照片效果会得到显著提升。我们可以对比处理前后的照片，感受其中的变化。

## 风景照片处理

处理完人物照，我们来看看风景照片的处理方法。其流程与人物处理类似。

首先，同样在“美图秀秀”的“美化图片”功能中，导入风景照片。选择“自动优化”功能，进行微调后保存。

然后，在“InstaSize”中打开这张照片。许多好看的照片效果并非预先设定，而是通过不断尝试和调整对颜色的感知得来的。

以下是尝试和调整的过程：
1.  尝试不同的滤镜，直到找到最合适的一款。例如，**Kayak** 滤镜就是一个不错的选择。
2.  再次进入“调整”选项。这里有多种参数可供尝试。
3.  我个人偏好调整“锐度”，因为它能有效增强照片的质感。

此外，如果你希望照片看起来更干净或更具个人风格，可以尝试裁剪画面，增加留白比例。因为干净的构图往往更受欢迎。

## 总结

本节课我们一起学习了使用美图秀秀和InstaSize进行照片美化的基本流程。关键步骤包括：用美图秀秀进行基础肤色优化，再用InstaSize添加滤镜并调整锐度以增强质感。对于风景照，方法同样适用，多尝试不同滤镜是关键。照片美化是一个探索的过程，不必追求一步到位。
