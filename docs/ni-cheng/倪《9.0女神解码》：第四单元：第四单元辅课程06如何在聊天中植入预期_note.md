# 倪《9.0女神解码》：第四单元辅课程06：如何在聊天中植入预期 💬

在本节课中，我们将要学习一个关键的聊天技巧：如何在与女生交流时，提前植入“预期”。这个技巧能帮助你在实际约会中减少对方的抗拒感，让互动更加自然顺畅。

## 课程概述

预期的作用，是让对方在事情发生前有心理准备。回想你过去的约会经历，当你试图靠近或接触对方时，她常常会反抗。原因在于她觉得突然，没有心理准备，从而感到尴尬和紧张。植入预期，就是为了避免这种突如其来的尴尬。

## 什么是植入预期？

植入预期，是指在见面之前，通过聊天从侧面或正面告诉对方，这次见面可能会发生什么。这样，当她同意赴约时，内心已经对即将发生的事有所准备。

## 如何植入预期？

方法其实很简单。你只需要在微信聊天时，提前进行暗示。

以下是植入预期的基本思路：

*   你原本的邀约是：“我们一起去看电影吧。”这时她的预期仅限于“看电影”。
*   植入预期后，你可以说：“我们手拉手去看电影，坐最后一排。”这时，她的心理预期就包含了“可能会拉手”。

## 预期的层级与示例

预期可以分为不同的层级，从初级到高级。

以下是不同层级的预期示例：

*   **初级预期**：例如“抱抱”。
*   **高级预期**：例如“你见到我会害羞吗？”、“你见到我会脸红吗？”、“我怕你见到我情不自禁。”、“如果我强迫你，你会怎么样呢？”

当你说了这些话之后，如果她仍然愿意出来见面，通常代表她喜欢你、同意你的暗示，并且已经有了相应的心理准备。

## 植入预期的前提条件

上一节我们介绍了预期的具体话术，但必须注意，这种技巧的使用需要前提。

植入预期必须建立在你和对方已有良好互动、你已经吸引到她的基础上。否则，她可能会直接拒绝你。

## 实战案例分享

为了让你更直观地理解，这里分享一个我之前的聊天案例。

我对那个女孩说：“我们见面时，你搂着我看电影吧，你别不理我啊。”她回复“哈哈哈”并开了个玩笑。后来我们见面时，她真的搂着我了。这就是成功植入预期的作用。
