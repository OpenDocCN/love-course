# 倪《女神解码》：第三单元：第三单元辅课程05拍照基础攻略

在本节课中，我们将学习如何拍摄一张优秀的展示面照片。展示面与普通头像或证件照的核心区别在于，它旨在展示你的生活品质与个人吸引力。

## 概述：什么是展示面？

展示面并非单纯的人物肖像。人是展示面的一部分，但仅占约**30%** 的权重。另外**70%** 则由照片的背景、构图与布局决定。一张成功的展示面照片，需要系统地构建视觉吸引力。

上一节我们明确了展示面的定义，本节中我们来看看构成优秀展示面的具体原则。

## 优秀展示面的五大核心原则

以下是拍摄展示面必须满足的五个基本原则：

1.  **人物状态**
    如果照片中包含人物，人物的穿着、发型、皮肤状态必须得体、美观。

2.  **画面清晰度**
    照片的像素不能过低，必须保证清晰。公式表示为：**画面质量 = 高分辨率 + 高锐度**。

3.  **背景环境**
    背景不能显得低端。环境必须看起来干净、大气、有格调。

4.  **构图与角度**
    拍摄角度需主次分明，构图需有层次感。我们将在后续实战案例中详细讲解。

5.  **元素规避**
    必须规避不应出现在展示面中的元素，例如无关人员或显得低端的物品。

理解了基本原则后，我们将在下一节课程中，进入实战拍摄环节。

## 总结

本节课我们一起学习了展示面照片的核心概念与五大拍摄原则。记住，展示面的重点是**通过整体画面（30%人物 + 70%环境与构图）来传递生活品质与吸引力**。掌握这些基础后，你便能更有方向地规划自己的展示面拍摄。
