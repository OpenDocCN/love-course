# 倪情感《女神聊法》：《女神聊法》——7、避开聊天雷区_note

在本节课中，我们将要学习如何通过避免常见的错误聊天方式来提升沟通效果，从而更好地与女生建立吸引力。

上一节我们介绍了如何判断女生是否对你感兴趣。如果你顺利抓住了窗口期，那么接下来就进入了建立吸引力的阶段。如何通过聊天来建立吸引力，是许多男生迫切需要解决的问题。许多男生在与女生交流时，常常纠结于如何措辞、如何回复以及如何解读对方的话语。久而久之，这种习惯可能导致关系越聊越僵。

与女生愉快聊天其实并不难，关键在于掌握两点：第一，避开聊天雷区；第二，掌握有趣的聊天技巧。今天，我们先来探讨第一个要点——避开聊天雷区。

以下是男生在聊天中经常踩到的几个雷区：

**1. 提问透支（查户口式聊天）**
许多男生聊天时总喜欢不断提问。在正常情况下，为了了解对方，适当提问以展开话题是必要的。但切忌连续不断地提问，这会透支基本的社交礼貌。

例如：
*   男生问：“你在干嘛？”
*   女生答：“在吃饭。”
*   男生又问：“吃了什么？”
*   女生答：“清蒸鱼。”
*   男生再问：“好吃吗？”
*   女生答：“还可以。”

这种连续性的提问既无聊又无趣。其令人反感的根本原因在于没有提供价值，反而在不断向对方索取价值。你不停提问，对方就需要不停回答。你们是在聊天，不是在面试。一场本该愉快的交流变成了职场问答，女生很难对这样的对话产生好感。

有的男生可能会说，提问是为了了解对方，不知道女生喜欢什么，只能通过提问来了解。这个出发点没有错，但连续的提问会透支对方的社交礼貌和耐心。当一个女生回答了十个问题后，她的耐心可能就已耗尽，不再回复或仅用“嗯”、“哦”敷衍。如果你在聊天中经常收到此类回复，就需要反思是否犯了提问透支的错误。

**2. 展示透支（过度自我展示）**
我们常提醒男生，聊天不要一直围绕女生转，也要适当展示自己。于是，有些男生便一发不可收拾，沉迷于自我展示。

聊天时全部以“我……”开头，全程都在讲自己的事情，忽略了女生的情绪。这会让女生觉得这个男生很自我，甚至有炫耀和装腔作势的嫌疑。

例如：
*   女生说：“每天都在纠结是坐车还是坐地铁。”
*   男生说：“我基本不纠结，每天都开车。”
*   女生问：“那限号怎么办？”
*   男生说：“难道我只有一辆车吗？”

此时，女生心里可能会想：“你装什么？” 因此，这种展示透支的方式非常不可取。女生会认为你是一个非常自恋的人，沟通起来很累。而且，过度展示本身就是一种低价值行为。如果你不断展示自己，说明你缺少倾听。当女生对对话没有进行任何投入时，她就会失去聊天的兴趣。

**3. 感受透支（用情过度）**
可能很多男生不太明白感受透支的含义。其实，它指的是用情过度，或者不分情况、不分阶段地全情投入。

例如，刚认识没聊多久，你就对女生说：“我觉得我们太默契了”、“我觉得我们上辈子肯定认识”、“你是我见过最漂亮的女生”等等。这种自以为真诚的沟通方式，在女生看来可能很滑稽。女生的情感需要一层层累积。一开始就急切地表达强烈感受，并期望得到同等反馈，这种希望几乎总会落空。

用这种方式聊天，传递的情感已经超越了你们当前的关系阶段，女生肯定会主动拉开距离，之前好不容易维系的好感也可能被消耗殆尽。

再举一个简单例子：很多男生聊着聊着就直接称呼“宝贝”、“亲爱的”。当女生尚未明确表现出兴趣时，这样的称呼会让她觉得你极度不正经。有时，男生还会犯另一个错误：洋洋洒洒写几百字发给女生，自己被文笔感动，但女生看到如此长的表达，下意识会觉得这个男生非常喜欢她，非常想获得她的认可。这种过分强烈的感受表达，只会让她认为你是一个不成熟的男人。

**4. 打压透支（过度打压对方）**
过度打压也是一种非常错误的聊天方式。有些男生觉得女生价值比自己高，想通过打压来拉平双方地位。但实际情况是，当女生对你的吸引力还不够，或者你本身不具备高价值时，突然进行打压只会让女生觉得你这个人有问题。

她心里可能会想：“你算什么？凭什么跟我说这些？” 很多男生学了一些打压技巧就迫不及待地想使用。

例如：
*   “这套衣服真好看，如果你能穿进S号就更好了。”
*   “你挺可爱的，不过我担心以后跟你面对面聊天，一直低着头会得颈椎病。”

这些都是错误的方式。如果对方确实在某些方面（如身材、身高）有不足，男生刻意针对这些缺点进行打压，这已经不是好的打压方式，而是人身攻击了，会让人觉得你极度不礼貌。

本节课中，我们一起学习了四种需要避开的聊天雷区：**提问透支**、**展示透支**、**感受透支**和**打压透支**。在与女生聊天的过程中，尽量避免这些错误方式，就可以达到更好的沟通效果。下一节，我们将探讨如何掌握有趣的聊天技巧。
