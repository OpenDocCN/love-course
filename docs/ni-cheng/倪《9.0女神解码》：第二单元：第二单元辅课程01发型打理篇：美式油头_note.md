# 倪《9.0女神解码》：第二单元：第二单元辅课程01发型打理篇：美式油头_note

在本节课中，我们将要学习如何打理一款经典的美式油头发型。我们将从所需工具开始介绍，然后详细讲解造型的每一个步骤，确保即使是初学者也能轻松掌握。

## 概述

美式油头是一种经典、利落的发型，打理得当能显著提升个人形象。本节课将指导你使用正确的产品和工具，完成从吹干到最终定型的全过程。

## 所需工具介绍

在开始造型前，你需要准备以下核心工具。以下是具体清单：

*   **美式发油**：这是打理油头或背头的专用产品。它与发蜡的区别在于，发油具有**光泽感**和**湿润度**，并且可以**反复造型**。公式表示为：`发油特性 = 光泽感 + 湿润度 + 可重复塑性`。
*   **染色发蜡（如奶奶灰）**：这是一种在塑形的同时能改变头发颜色的产品。其特性是**偏干**，但具备**染色效果**。
*   **梳子**：用于精确分界和梳理头发，是塑造整齐发型的关键工具。

## 造型步骤详解

上一节我们介绍了所需的工具，本节中我们来看看具体的造型步骤。整个过程分为吹干、涂抹发油和梳理定型三个阶段。

### 第一步：吹干头发

首先，需要将头发吹干，并将发根吹得立起来。这样做是为了让头发变得蓬松，为后续的塑形打下基础。如果头发是塌下来的，后续造型会比较困难。

### 第二步：涂抹发油

接下来，我们使用发油进行塑形。以下是具体操作方法：

1.  取大约一个指节量的发油。
2.  在掌心均匀揉开。
3.  将发油均匀地涂抹在头发上，并用手抓出大致的形状。

> 本节课使用的发油是水溶性的，其特点是遇水会溶解。这为后续的湿梳定型提供了便利。

### 第三步：梳理定型

在涂抹好发油后，头发会显得比较油亮。此时可以进行最后的精细梳理。

1.  打开水龙头，将梳子稍微沾湿。
2.  按照你设定的分界线（例如三七分）进行梳理。用沾湿的梳子可以轻松地将头发梳理得整齐、服帖且有光泽。

## 其他产品补充说明

除了发油，染色发蜡（如奶奶灰）也是打造个性造型的选择。它的使用方法与发油类似，但质地更干，主要特点是在塑形的同时能赋予头发新的颜色。

## 总结

本节课中我们一起学习了如何打理美式油头。我们首先认识了核心工具——发油、染色发蜡和梳子。然后，我们一步步实践了造型流程：先将头发吹蓬松，接着均匀涂抹发油进行初步塑形，最后用沾湿的梳子梳理出整齐、光亮的最终发型。每天只需花费十分钟左右，你就能拥有一款帅气利落的发型出门了。
