# 倪团队《极致聊天：倪团队《极致聊天：赠品《极致聊天》：《极致聊天》——

在本节课中，我们将学习如何组织语言，使你说出的话更具画面感和吸引力，从而让女生更愿意与你持续聊天。我们将探讨如何用具体的名词和动词代替抽象的形容词，以及如何通过构建场景来增强对方的代入感。

---

我们都是听故事长大的，至今依然喜爱听故事。这是因为故事能让我们保持好奇心，沉浸其中，不自觉地代入主角，体验情节。好故事不仅能带来愉悦，还能传递新知。

如果说你能让女生感觉和你聊天就像在听故事，那么你就能轻易地让她喜欢上你。许多初学者可能需要长时间练习才能讲好一个完整的故事，但要想把天聊好，你只需掌握讲故事中的一项核心能力即可。

这项能力就是：说出具有画面感的话。例如，一般人夸女生好看时可能会说：“你好漂亮。”而我会说：“看你的第一眼，我还以为是网红呢。”这两句话有何不同？

前一句“你好漂亮”使用了抽象的形容词。后一句“看你的第一眼，我还以为是网红呢”则运用了动词和名词，营造出更具体的画面，让对方更精准地理解我想表达的感觉。

如果你想让你说出的话更具画面感，我的建议是：在组织语言时，尽量使用动词和名词，减少使用形容词。如果必须使用形容词，可以用比喻句来修饰其意境。

例如，当你想说自己吃得很饱时，可以在后面加上：“现在肚子就像一个被加满气的气球一样。”或者：“我现在的肚子跟一个小孕妇一样。”

我个人在聊天时，经常会交代清楚四个元素：**环境、人物、感受、动作**。因为当你说出的话包含这四个元素时，对方更容易产生代入感。

例如，当女生问“在干嘛”时，我会回答：“我躺在家里的沙发上看电影，挺舒服的。”这句话中：
- **环境**：家里的沙发上
- **人物**：我
- **感受**：舒服，跟度假似的
- **动作**：在看电影

当女生听完这些具有画面感的话，她会自然脑补出画面。画面越具体，她就越容易回复你。

她可能会回复：“看什么？”、“你一个人住啊？”、“怎么还跟度假似的呢？”或“真羡慕你可以那么轻松。”总之，你提供的画面感越强，她越容易与你聊下去。

如果有人对你说：“诶，我昨晚梦见你了，然后好像发生了什么……”听完这句话，你是不是会很想听下去？因为你会本能地想了解接下来发生了什么。

如果你能在聊天中灵活地做到“说话只说一半”，那么与你聊天的女生就会一直保持较强的好奇心，愿意继续聊下去。

在实际聊天中，你可以先说一句具有画面感的话，然后隐藏其中一个细节，或修改后再发给对方，从而达到“说话只说一半”的效果。

例如，你想对对方说：“我今天在公司忙了几个小时，现在就想去按摩。”你可以把“去按摩”改成“去干大事”。于是，对方就会好奇你接下来要干什么大事。
