# 倪《9.0女神解码》：第三单元：第三单元辅课程04朋友圈如何配文_note

在本节课中，我们将学习如何为朋友圈照片搭配文字。核心在于理解文字与照片的关系，并掌握一种简单、安全的配文方法。

上一节我们介绍了照片拍摄，本节中我们来看看如何为照片配上合适的文字。

很多人容易陷入一种“用力过猛”的情况。所谓“用力过猛”，是指故意撰写很长的故事来配合图片。实际上，**展示面比文字更重要**。女性更相信眼睛看到的画面，而非你的文字描述。

因此，我们的核心建议是：配文尽量简单，尽量少带情绪。这意味着，只需用简单、不带情绪的文字去描述事实，这是最保险的方式。当然，如果你文采飞扬、创意十足，也不排除能写出精彩的配文。但我们的建议是，在最保守的情况下，应确保照片本身完美，而文字则**言简意赅**。

例如，针对下午茶的照片，可以简单地配文：“下午茶”或“和好友下午茶”。无需使用过分夸张的表情或修饰。

我们推荐这种方法，是因为不希望教授过于复杂的技巧。复杂的技巧若被误解，可能导致一些极端且不好的效果。

以下是你可以参考的最简单、最保守的配文步骤：

*   **核心原则**：展示面 > 文字。
*   **文字风格**：简单、直接、少情绪。
*   **操作示例**：描述事实，如“下午茶”。
*   **避坑指南**：避免长故事和夸张表情。
