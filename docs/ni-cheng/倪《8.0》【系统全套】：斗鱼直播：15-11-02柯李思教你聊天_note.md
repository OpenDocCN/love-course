## 第一步：平台选择与资源收集 🎣

上一节我们介绍了课程的整体目标，本节中我们来看看实际操作的第一步：选择合适的平台并开始收集联系人。

探探是一款在2015年兴起的社交软件，用户以年轻人为主。其早期通过街拍团队推广，吸引了大量用户，但如今用户质量变得参差不齐。

操作的核心是“资源收集”，即大量添加潜在联系人。以下是具体操作步骤：

1.  **打开探探**：浏览并右滑喜欢的用户照片进行匹配。
2.  **发送初始消息**：匹配成功后，发送一个简单的表情作为开场。
3.  **引导至微信**：当对方回复后，直接提出“微信聊”的请求，并将自己的微信号发送给对方。
4.  **等待通过**：在微信上等待对方添加好友，并做好备注（例如：探探-丹燕）。

**核心概念公式**：
```
资源收集效率 = 匹配数量 × 转化至微信的成功率
```
关键在于短时间内接触大量潜在对象，并快速筛选。

## 第二步：微信初期聊天框架建立 🤝

成功将联系人转移到微信后，就进入了关键的初期聊天阶段。本节我们将学习如何建立有效的聊天框架。

初期聊天的目的是筛选对方兴趣度，并建立轻松、有趣的互动氛围。避免过早暴露需求感。

以下是初期聊天的标准步骤：

1.  **交换基本信息**：作为社交礼仪，首先礼貌地交换姓名。
2.  **进行初步筛选**：使用幽默的方式询问对方是否单身。例如：“耍朋友了吗？”（意为“谈恋爱了吗？”）。
3.  **给予轻微夸赞**：夸赞对方的某个特点，但随即加入一个“筛选器”。例如：“我觉得你眼睛挺好看的，但是胸有一点小。” 这能测试对方的反应和服从度。
4.  **测试兴趣指标**：如果对方对你的话题表现出兴趣（如追问或解释），则是一个积极的兴趣指标。

**核心聊天代码框架**：
```plaintext
if 对方回复积极:
    进行下一步关系拉伸
else:
    暂时搁置，继续其他对话或资源收集
```

当判断对方兴趣度（假设满分为10分）达到6分左右时，即可进入下一步。

## 第三步：关系拉伸与服从性测试 🎭

当确认对方有足够兴趣后，我们需要通过“关系拉伸”和“服从性测试”来推进关系。本节我们来看看具体如何操作。

关系拉伸是指使用更亲密的称呼或提出略带暧昧的要求。服从性测试是提出一个小要求，观察对方是否愿意服从，这是判断对方窗口大小和推进关系的关键。

操作步骤如下：

1.  **使用亲密称呼**：在兴趣度足够时，突然称呼对方为“宝宝”等亲密用语。
2.  **提出小要求**：例如，“你会做饭吗？”或“抱一下”。
3.  **观察与应对**：
    *   **如果对方服从**：给予夸奖，巩固关系。
    *   **如果对方拒绝**：不要纠缠，可以幽默地后退一步，给对方台阶下。例如：“谢谢，你是个好人。” 然后暂时转移话题。
4.  **循环测试**：通过一系列从小到大的服从性测试，逐步升级关系。

**核心概念**：整个关系推进过程，可以看作是一系列**服从性测试**的累积，最终导向确立关系。

## 第四步：话题控制与框架保持 🧠

在聊天中，保持自己的话题框架至关重要，不要被对方带入她不擅长的领域。本节我们学习如何掌控聊天方向。

当对方试图将你拉入她的逻辑框架（例如，反复讨论你不愿深入的话题）时，需要巧妙转移。

应对策略如下：

*   **转移话题**：当对方纠结于某个不利话题时（例如反复问“你觉得多大才算大？”），直接忽略，开启一个新话题。例如：“亲爱的你什么星座？”
*   **保持神秘与跳跃**：聊天思维可以适当跳跃，这能增加神秘感和趣味性，避免陷入枯燥的问答模式。
*   **使用套路化回复**：许多开场白和回复可以标准化，提高聊天效率。
