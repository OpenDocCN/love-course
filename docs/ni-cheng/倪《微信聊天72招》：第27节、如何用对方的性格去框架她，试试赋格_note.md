# 倪《微信聊天72招》：第27节、如何用对方的性格去框架她，试试赋格 🎭

在本节课中，我们将要学习一个名为“赋格”的重要聊天技巧。赋格能帮助你通过赋予对方特定的性格特质，来建立自己的标准和框架，从而在互动中脱颖而出。

上一节我们介绍了如何拉伸关系以加速发展。本节中，我们来看看如何运用“赋格”来深化互动，并展示你的独特性。

## 什么是赋格？

赋格是一个专业术语，其核心含义是：**主动赋予对方一种性格特质**。

例如，在聊天时，女生发了一个很可爱的表情，这时你可以对她说：“你是个可爱的姑娘，我就喜欢这样可爱的女生。” 这样，你就赋予了对方“可爱”这个特质。

同时，赋格也是我们用来**框架对方**并**展示自己标准**的方式之一。只要时机得当，对方会自然地认为你是一个优秀的男人，因为只有优秀的男人才会有自己的标准。如果对方对你有好感，她往往会为了符合你的标准而调整自己的行为。

## 框架的作用是什么？

框架对方，意味着你为她设定了一个行为或性格的参考范围。

例如，你说她可爱，几乎没有女生会反驳说“不，我不可爱”。这意味着，她可能会下意识地朝着你设定的“可爱”框架去表现自己。这就是框架引导行为的作用。

## 为何赋格比单纯赞美更有效？

请先思考这个问题：如果你只是因为她漂亮才和她聊天，那么众多觉得她漂亮的男人中，她为何一定要选择你？

越漂亮的女生，往往越少主动展示自己独特的个性。因为通常不需要她展示这些内在价值，男人们就已经被她的外貌所吸引。

我们在聊天中，需要去发现女生某种符合我们标准的特质，并因为这种特质而喜欢她，而不是仅仅因为外貌。

以下是赋格带来的核心优势：

*   **展示独特性**：你展示出与其他男人不同的一面。
*   **建立高标准**：你表达了自己对女人的标准和经验。
*   **超越外表**：让女生觉得你并非只看重她的美貌，从而觉得你与众不同。

## 如何在实战中运用赋格？

在聊天中，可以不经意地赞美她的某个内在点，并将其与你的喜好关联。

例如，你可以问：“你是哪里人？” 她回答是成都人。这时你可以说：“天呐，你是四川妹子，我喜欢四川的山水和四川人给人的亲切感。”

这样一来，你就和那些只关注她外表的男人区别开了。你是因为她给你的整体感觉（亲切）而与她聊到一起。

以下是实战中常用的赋格语句示例：

*   “你一看就是温柔善良的乖乖女。”
*   “我发现你还真是个善良靠谱的好姑娘，这一点我挺喜欢的。”

这些语句都完成了赋格，并同时表达了你的兴趣和标准。

## 成功赋格的关键能力

这需要你具备一种**挖掘女性独特内在品质的能力**。许多女性都有多面性，你需要主动去发现她们身上与众不同的地方，然后针对这些独特的性格特质进行赞美和赋格。

如果你能很好地做到这一点，你就会发现许多其他男人未曾留意的东西。女生也会因此感觉你是一个非常懂她、能看到她独特价值的男人。
