上一节我们列出了改造的总体框架，本节中我们首先来看看**服装单品**的选择。

冬季兼顾保暖与帅气的内搭首选是高领暗色系毛衣。`颜色选择：黑色、灰色、藏青色`。这些颜色通常不会出错。

外套可选择长款呢大衣或短款羽绒服。想叠穿可选择`羊毛开衫 + 呢大衣`的组合，温暖又优雅。

鞋子可选中高帮、皮质或麂皮材质的牛筋底靴子。这种鞋子好看、保暖且防滑。

了解了基础单品，接下来我们进入**造型搭配**环节。

*   **若体型偏胖**：勿选肥大衣服，尤其冬季穿肥大羽绒服会显臃肿。尽量选择修身款大衣或外套。

服饰搭配完成后，我们来看看提升精致度的关键——**发型设计**。

发型并非流行就适合，找到适合自己的最重要。如无必要，尽量不剪平头。一是太普通，二是平头对颜值要求极高。

建议选择优质理发店，与理发师充分沟通，请其为你设计发型。可告知理发师发型用于约会或重要场合，对方会更认真对待。若不提要求，理发师可能随意处理。

设计好发型后，务必请理发师教你日常打理方法。例如`美式油头`或`The Crew Cut`等发型，不打理会难看。最好养成每日打理头发的习惯。

除了发型，**皮肤护理**同样不可忽视。

第二，皮肤补水很重要。若想省时省力，可购买成套护肤品，水乳搭配使用。皮肤不仅不再油腻，冬季也不会干燥起皮。品牌尽量避开过于常见的开架品牌，可选择像`资生堂`等系列，价格适中。女性若知你使用这些护肤品，会认为你追求生活品质。
