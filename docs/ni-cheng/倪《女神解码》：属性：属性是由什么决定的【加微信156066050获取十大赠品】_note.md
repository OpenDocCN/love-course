# 倪《女神解码》课程一：属性由什么决定？🔍

在本节课中，我们将要学习一个核心概念：你的“属性”究竟由什么决定。很多人误以为这完全取决于外貌，但事实并非如此。我们将通过分析具体例子，揭示决定属性的真正关键因素。

## 属性并非由外貌决定

很多男性认为，属性是由长相决定的。例如，有些男性被认为长了一张“约炮脸”，而有些则被认为长了一张“跪舔脸”。

然而，这种看法并不准确。真正的决定因素并非如此简单。

## 通过明星案例理解属性

为了更好地理解，我们来看三个广为人知的明星例子：陈冠希、周杰伦和陈建州（黑人，范玮琪的丈夫）。

思考一下，他们三人分别具备什么样的属性？

答案很明显：
*   陈冠希具备**短期关系属性**。
*   陈建州（黑人）具备**长期关系属性**。
*   周杰伦则让女生觉得**适合恋爱**。

## 行为模式决定属性

为什么他们会形成不同的属性？我们来逐一分析。

陈冠希之所以具备短期属性，是因为他的**行为模式**：坏坏的笑容、略带不负责任的感觉、活在当下、风流倜傥、热爱冒险、行为勇敢无畏。这些行为共同塑造了他的短期关系属性。

周杰伦之所以让人觉得适合恋爱，是因为他的**行为模式**：浪漫、酷帅、才华横溢、带有冷幽默，并且整体感觉比较靠谱。女生会觉得和他谈恋爱会非常开心，他是非常适合小女生的恋爱对象。

陈建州（黑人）之所以被认为适合长期关系，是因为网上流传的许多关于他的**行为模式**：如何对范玮琪好、为她做早餐、接她下班等等。这些行为讲述了他是一个可靠、体贴的男人，因此更适合作为长期伴侣或丈夫。

## 核心结论

综上所述，决定一个人属性的，并非外貌，而是其**行为模式**。

**属性 = 行为模式**

具体来说，是你的言行举止、眼神、肢体语言、喜好和具体行为，共同决定了你属于何种属性。这也最终决定了女性如何定位你，从而影响你的情感生活。

我们无需讨论这些明星的真实生活，关键在于要弄清楚：哪些行为模式会导向哪些属性。

例如，短期关系属性应该对应哪些行为？长期关系属性又应该有哪些行为？如何通过行为让女性将你定位为长期发展对象？

这正是我们下一节课要详细探讨的内容。
