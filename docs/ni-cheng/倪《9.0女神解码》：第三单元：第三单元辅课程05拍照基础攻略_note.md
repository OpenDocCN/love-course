# 倪《9.0女神解码》：第三单元辅课程05 📸 拍照基础攻略

在本节课中，我们将学习如何拍摄一张优秀的展示面照片。展示面与普通头像或证件照的核心区别在于，其目的是展示个人的生活品质与吸引力。因此，人物虽然是照片的一部分，但并非全部。

上一节我们明确了展示面的定义，本节中我们来看看构成一张优秀展示面的五个核心原则。人物在展示面中大约只占**30%** 的重要性，剩余的**70%** 则由照片的背景、布局等元素决定。

以下是构建优质展示面必须满足的五个基本原则：

1.  **人物状态**：若照片中包含自己，则人物的穿着、发型、皮肤状态必须得体、整洁，形象需保持良好。
2.  **画面清晰度**：照片的像素不能过低，必须保证清晰。
3.  **背景环境**：背景不能位于显得低端的场所，应选择看起来干净、大气的环境。
4.  **构图角度**：拍摄时必须分清主体与陪体，明确主次关系。
5.  **元素规避**：必须避免一些不应出现在展示面中的事物，例如无关人员或显得低端的物品。

我们已经了解了展示面的五大基础原则。在下一集课程中，我将通过实战案例，具体演示如何应用这些原则拍出一张好的展示面。

本节课中，我们一起学习了展示面摄影的五个基础原则，理解了人物、清晰度、背景、构图和元素筛选在塑造高质量展示面中的关键作用。
