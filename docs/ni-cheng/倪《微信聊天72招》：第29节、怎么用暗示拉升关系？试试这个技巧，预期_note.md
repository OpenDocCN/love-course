# 倪《微信聊天72招》：第29节、怎么用暗示拉升关系？试试这个技巧，预期_note

在本节课中，我们将要学习一个在聊天中用于拉升关系的核心技巧——“预期”。我们将了解它的定义、作用、最佳使用时机以及具体操作方法。

上一节我们介绍了“属性”的概念，它帮助我们根据目标向女性展示相应的特质。本节中，我们来看看如何通过“预期”技巧，为关系的进一步拉伸铺平道路。

## 什么是预期？

预期，是在提出邀约后，给予女性关于未来关系可能发展的暗示。

**核心公式**：邀约 + 未来关系行为暗示 = 预期

例如，告诉对方在约会时可能会牵她的手。这样做的目的是让女性提前对关系拉升做好心理准备，甚至默认这种发展。

## 预期的作用

以下是预期技巧的两个主要作用：

1.  **测试窗口**：在聊天阶段就能测试对方对你的兴趣程度，了解她愿意将关系发展到哪一步。
2.  **铺垫约会**：能让后续的约会更加顺利。因为对方心里已有所期待，更容易接受你在现场做出的、拉升关系的具体动作。

预期本身也是一种测试和拉伸。但与其他技巧一样，它不能滥用。

## 何时使用预期？

释放预期的最佳时机，是当双方已进入暧昧或调情阶段，并且有了一定的关系基础，到了该提出邀约的时候。

**错误示范**：如果双方关系尚未暧昧，此时使用预期会显得突兀。
**正确示范**：在暧昧阶段，你说：“我想搂着你看电影，像情侣一样。” 这个预期会让对方脑海中浮现相关画面，从而产生期待。

## 使用预期的注意事项

使用预期时，力度必须合适，并注意对方的反馈。以下是几个关键点：

*   **力度匹配**：预期必须与当前关系阶段相符。例如，当前关系只能牵手，你的预期就应止步于此，而非暗示更亲密的行为。
*   **观察反馈**：
    *   如果对方只是轻微抵抗或害羞（例如回复“不要啦”、“坏人”或害羞表情），可以根据情况选择是否进一步释放预期。
    *   如果对方强烈抵抗（例如说“你想多了”、“不可能”），则说明预期力度过大，或关系确实未到位。
*   **应对抵抗**：当对方强烈抵抗时，最佳做法是暂时撤回，继续建立吸引和展示价值，而非执着于本次邀约。

## 预期的实际案例

你可以在微信聊天中，于适当时机使用以下预期话术：

*   **示例1**：“我想手牵着手跟你一起去看电影。”
*   **示例2**：“要是我跟你约会的时候，忍不住想亲你怎么办？”

无论女生是拒绝还是同意，预期都已发挥作用——她脑海中已经预演了关系可能被拉伸的场景。
