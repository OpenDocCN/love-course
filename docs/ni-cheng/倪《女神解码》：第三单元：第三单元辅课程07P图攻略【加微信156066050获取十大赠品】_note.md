# 倪《女神解码》：第三单元辅课程07：P图攻略 🖼️

在本节课中，我们将学习如何快速处理一张可用于社交展示的照片。教程将分为两个部分：人物照片处理与风景照片处理。我们将使用两款常见的手机应用，通过简单的步骤提升照片的观感。

## 一、人物照片处理

上一节我们概述了课程内容，本节中我们来看看如何具体处理一张人物照片。

首先，打开“美图秀秀”应用。请确保进入“美化图片”功能，而不是“人像美容”功能。

以下是处理人物照片的具体步骤：

1.  导入目标人物照片。
2.  点击“智能优化”选项。
3.  在优化类型中选择“人物”。

完成此操作后，照片中人物的黑眼圈会减轻，肤色也会变得更白皙。但这并非最终成品，请先保存当前效果。

接下来，我们需要打开第二个应用“Instasize”进行进一步调整。这个应用主要用于调色和增强质感。

以下是使用Instasize的步骤：

1.  在Instasize中导入刚才保存的照片。
2.  在滤镜库中，选择一个名为“Cellseal”（或类似风格）的滤镜，这个滤镜色调偏欧美风。
3.  点击“调整”工具，找到“锐度”选项。
4.  将锐度值调整到 **2.3** 左右。

增加锐度可以让人物皮肤显得更有质感，避免过度美化带来的虚假感。经过这两个软件的处理，你的照片会变得更具吸引力，甚至拥有类似网络红人照片的质感。我们可以通过对比处理前后的照片来观察效果。

## 二、风景照片处理

处理完人物照片，我们再来看看风景照的处理方法。其流程与人物处理大致相似。

首先，同样在美图秀秀的“美化图片”功能中打开风景照。

以下是初步处理步骤：

1.  选择“智能优化”或“自动”功能进行微调。
2.  保存初步优化后的照片。

然后，再次打开Instasize应用进行精修。需要说明的是，优秀的修图效果往往不是预设的，而是通过不断尝试和调整获得的。

以下是进一步的调整步骤：

1.  在Instasize中导入风景照。
2.  尝试不同的滤镜，例如“Kayak”滤镜可能是一个不错的选择。
3.  进入“调整”选项，尝试各种参数。通常，将“锐度”适当调高可以增加照片的质感。

此外，如果你希望照片看起来更干净或更具个人风格，可以尝试裁剪画面。

例如，通过裁剪增加画面中的留白区域。因为从女性视角来看，她们通常更喜欢干净、简洁的画面。画面中白色区域越多，往往意味着照片给人的感觉越干净。
