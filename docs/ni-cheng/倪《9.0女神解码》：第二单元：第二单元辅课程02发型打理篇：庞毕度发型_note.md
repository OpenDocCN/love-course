# 倪《9.0女神解码》课程笔记：第二单元辅课程02 - 发型打理篇：庞毕度发型 💇‍♂️

在本节课中，我们将学习如何打理一款经典的发型——庞毕度发型。我们将从准备工作开始，逐步讲解使用发蜡塑形和定型喷雾固定的完整流程，帮助你轻松掌握打造持久有型发型的技巧。

## 准备工作：吹干头发

打造任何发型的第一步，都是确保头发处于合适的状态。对于庞毕度发型，你需要先将头发完全吹干。

上一节我们介绍了准备工作，本节中我们来看看核心的塑形步骤。

## 核心步骤：使用发蜡塑形

头发吹干后，接下来需要使用发蜡来塑造发型。我使用的是丝滑扣品牌的发蜡，当然你也可以根据个人喜好选择其他品牌。

以下是使用发蜡塑形的具体步骤：

1.  **取用发蜡**：用手指挖取适量发蜡，大约一枚硬币大小。
2.  **均匀揉搓**：将发蜡在双手手掌和指尖间充分揉搓均匀，使其微微发热软化。
3.  **抓塑发型**：将揉搓均匀的发蜡从发根到发梢均匀涂抹并抓理头发，塑造出你想要的发型轮廓和纹理。发蜡的核心作用是 **`塑形 + 纹理`**。

完成发蜡塑形后，为了应对户外环境并让发型更持久，我们需要进行最后一步。

## 最终定型：使用定型喷雾

为了防止头发被风吹乱，并让塑造好的形状保持更长时间，我们需要使用定型喷雾进行最终固定。

操作非常简单：在距离头发约20-30厘米处，均匀喷洒定型喷雾，重点照顾发型的外轮廓和容易散乱的部位。

## 课程总结

本节课我们一起学习了庞毕度发型的完整打理流程。我们首先将头发吹干作为准备，然后使用发蜡进行核心的塑形与纹理创造，最后使用定型喷雾来固定发型，确保其持久有型。
