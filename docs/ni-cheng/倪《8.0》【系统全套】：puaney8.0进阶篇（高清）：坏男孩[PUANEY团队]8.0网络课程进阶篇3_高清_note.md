# 课程8.0：聊天中的吸引力构建 🎯

在本节课中，我们将要学习如何通过聊天对话来构建对女生的吸引力。我们将剖析传统观念的误区，并深入探讨吸引力的核心本质及其在语言交流中的具体应用方法。

## 吸引力核心：从“打压”到“展示” 🔄

上一节我们介绍了吸引力的常见误区。本节中我们来看看吸引力的真正核心。

曾经有一种观点认为，打压女生会产生吸引。这句话曾误导了许多人多年。因为这句话并未指出吸引力的核心本质到底是什么。

之前我们讨论过，其他一些书籍或帖子可能会告诉你，需要通过打造朋友圈、进行形象改造或更换发型来产生吸引。更多的人不会告诉你聊天本身是可以产生吸引的。今天我们将解释并剖析如何在聊天中产生吸引。

## 常见错误聊天模式 ❌

以下是许多人在聊天中常犯的错误。

第一点，很多人的沟通模式是不断提问。例如，我们曾有一位线下学员，他与女生聊了四句话，全是问题：第一句问“吃饭了没有”，第二句女生回复内容不重要，第三句问“最近店里的生意如何”（女生是开店的），第四句问“之前的发票的事情搞定了没有”。他问我这样的聊天有什么问题。实际上，这种聊天本身可能没有问题。因为很多人会告诉你这是“夺命连环问”，但连环提问并非问题的本质。真正问题的本质在于，你在聊天中没有构建吸引力。

## 吸引力的本质：展示 🌟

在聊天中产生吸引力的核心，可以用两个字概括：**展示**。

如果你看过我们的5.0课程，你会知道男人吸引女人的本质就是展示吸引力。多种品质可以提升你的吸引力，例如：
*   **幽默**
*   **财富**
*   **外表帅气**
*   **爱心**
*   **孝顺**

这些都属于吸引力的品质。那么如何产生这些吸引力呢？唯一的方式就是通过**展示**。

举一个简单的例子：当一只雄性孔雀想要吸引雌性孔雀时，它不会跑过去为对方倒水或拿食物。它的做法是**孔雀开屏**。这就是展示自己。

因此，在聊天中，我们通过**语言的展示**来达到吸引的结果。

## 展示中的问题：平淡与低价值展示 ⚠️

接下来，我们来看看在展示中常见的问题。

大部分人不擅长展示自己的品质。他们的聊天平淡无奇。例如，当女生问他“你今天做了什么呢”，他可能回答：“我今天去打了一盘游戏，然后吃了一份外卖，工作了大概8个小时，写了一篇文章。” 这些描述没有进行有效的语言展示，因此不会对女生产生吸引力。但这还算中规中矩，至少没有犯错。

更糟糕的情况是，很多兄弟在聊天过程中，不仅没有展示高价值或优秀特质来产生吸引，反而做了许多**DLV**（低价值展示）。例如，当女生问他“在做什么呢”或聊天开始时，他第一句话就说：“今天在家里面打了一天的游戏，好累啊。” 这展示了一种很低端、很宅、很单一的生活品质，这是我们需要避免的。

你需要识别出女生不喜欢的男性品质，例如：
*   生活很单一
*   脾气很暴躁
*   情绪不稳定，像小孩子（例如，“今天我又跟爸爸吵架了，我又跟妈妈吵架了”）

这些负面情绪都属于低价值内容，我们不应在聊天中向女生展示这些。

## 正确的展示内容 ✅

那么，我们应该展示什么呢？

你应该展示积极、阳光的内容。例如：
*   展示你对工作的上进心。
*   展示你的社交圈子、你的朋友（这本身也是一种预选）。
*   展示你高品质的生活方式。
*   展示你的爱心和孝顺。
*   展示你精彩的生活方式。

这些是我们在聊天中应该展示的内容。

## 展示的艺术：避免粗暴，结合“台阶” 🎭

然而，新的问题出现了。很多兄弟说：“我也展示了，我告诉女生说，今天我的LV钱包丢了。” 这种展示是非常错误的。

我们称之为**粗暴的展示**。粗暴的展示会让女生觉得你这个人太装，或者给人一种很不舒服的感觉。很简单，当有一个哥们对你说：“兄弟，我最近买了一套别墅。” 你可能会觉得他无礼，心想：“跟我说干嘛？” 人天生就不喜欢直接粗暴的展示，因为这给人一种炫耀的感觉。

因此，最后我要告诉大家展示的一个正确核心知识：展示离不开另外一个词语——**台阶**。

**展示只有与台阶结合在一起，才会成为一个完整的展示。** 我们将其总结为一个永恒的公式：**展示 + 台阶**。

## “展示+台阶”公式应用实例 📝

在举例之前，我先说明一个规律：你所展示的吸引点（例如，我们用高价值社会地位举例），永远是你展示中的辅助，是**绿叶**；而**台阶**应该是展示中的**主体**。

**实例一：展示社会地位**
前几天我与一个女生聊天。我想展示我的社会地位，想告诉她我是一个有经济实力的老板。我是这样开始的：
> 我说：“我的脾气特别差，我的缺点特别多。”
> 她问：“怎么了？”
> 我说：“我脾气特别火爆。最近一个月我把两个员工都骂跑了。所以有时候特别为自己的脾气感到懊恼。”

明白了吗？你的主体、你的台阶，是“脾气很差”。只不过你在无意中植入了一个高价值信息：“我把员工骂跑了”。这样她就能感觉到你是一个社会地位较高的人。你是无意间将其穿插在对话中的。

如果你换一种方式，比如直接说：“我最近开了一家公司，我是一个老板。” 女生可能会想：“傻吗？老板我见多了。” 她一定会产生抵触情绪。

**实例二：展示精彩生活**
我想展示我的生活很精彩。我会跟女生这么说：
> “最近有两件事情特别想去做。第一件事情呢，我特别想跟我的一个女生朋友去徒步旅行。第二件事情呢，我特别想完成我小时候的一个梦想，就是去学钢琴。正好我有一个朋友是教钢琴的，我跟她报了一个一对一课程。”

这也是在展示你的生活很精彩。

## 思维模式的转变：从发问到展示 💡

你会发现，这种聊天方式与之前很多兄弟的聊天方式有什么不同吗？就是我更多地会去说关于我自己的事情。

**展示天然地要求你多谈论关于自己的事情。** 这就是很多兄弟需要首先改变的思维模式：你一定要**减少提问**，**增加展示**。

之前我们也说过，我在这里重复一个聊天模式：我们可以问问题吗？可以。千万别因为我讲了这些，你就害怕去问女生问题。其实你可以问。

但是，很多人问问题的方式是：问 -> 女生答 -> 继续问 -> 女生再答。这样两三个回合下来，女生就会觉得非常无趣，或者觉得跟你聊天没有乐趣。

正确的模式应该是：**问 -> 女生答 -> 你给予反馈**。在这个反馈当中，你就可以多进行自我展示。

## 展示的基础与切入方法 🛠️

关于这一点，我还要奉劝兄弟们一句：你的展示，很简单，**你得有东西可以展示**。你需要知道如何从自己的生活经历中提取素材。你想要展示财富、展示预选，你首先得拥有这些东西，得真正经历过这些生活。

所以，最终还是要送大家一句话：**你一定要去行动，去做事**。因为一个真正有魅力的男人，一定是做了很多事，而不是吹了很多事。

很多兄弟会说：“那我没有什么东西，我不知道怎么切入展示。” 刚才说了一个方法是“问答反馈”模式。

最后，还有一个方法就是**下钩子**。例如，在开启话题时，你就跟她说：“今天太倒霉了。” 女生就会很好奇，问：“怎么了？” 于是你可以用我们一个非常经典的例子：
> “车被刮，钱包丢了，还被人说不帅。” （加上两个哭的表情）

这一句就是典型的“展示加台阶”公式。那么，什么是展示？什么是台阶？
*   **展示**：车被刮、钱包丢了（暗示了一定的经济基础和生活状态）。
*   **台阶**：还被人说不帅（一个可以调侃、容易引发女生互动的话题）。

女生肯定会揪着你的“台阶”（被人说不帅）来跟你聊天，而不会直接问“哎，什么车？”（如果她这么问，可能有点不接地气了）。她可能会说“你本来就不帅”，或者说“哎哟，别人说你不帅？”等等。“台阶”永远是给女生提供与你聊天的切入点，而“展示”则是你聊天中那些隐性地吸引她的部分，当然大家都心照不宣。
