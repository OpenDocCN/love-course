## 第二轮聊天的开启与心态调整

上一节我们分析了首轮聊天的破冰技巧，本节中我们来看看两天后如何开启第二轮对话。

在开启第二轮聊天前，需要先思考聊什么以及如何切入。这里有一个特别重要的点：许多人在思考与女生聊天时，第一印象是如何把聊天变得有趣，如何逗对方开心。

这是一个错误的目的。你首先要调整好心态，即采用试探的心态。不带有任何特定目的，不想着刻意逗对方开心，只是保持自己较好的状态，见一步走一步。

这才是正确的心态。在女生没有明显释放好感信号时，依然应该用试探的态度去开启话题。

开场白不推崇过于炫酷的方式，那会显得用力过猛。因此，这里依然使用了一个表情包——那个可爱的小男孩宋明国表情包，非常适合调节气氛。

**代码示例：开场消息**
```
你：[表情包：小男孩说“咦 宝哥”]
她：哈哈 明天不上体能就能喝起来
```

这里依然将问题转为陈述，没有说“明天一起喝酒吗”，而是采用一种强势、拽拽的说话方式。

## 话题的接续与技巧运用

妹子的回复中提到“明天不上体能就能喝起来”，因为她是一名练习生，有体能课。她的话里没有特别明显、好接的话题梗。

从女生的话里可以发现，她虽然顺从，但没有太多主动示好的感觉。这很正常。

以下是回复的思路：

首先，使用了一个曲解技巧。我说“体能是谁，你为什么要上”，曲解了“上”字的含义。

然后，如果只有这句话，妹子很难回复。因此需要加一句话题分岔，这里使用了一个叫“感受认同”的技巧，即表达出女生未说出的感受。

她每天要上各种训练课，年纪轻轻，一定会感到枯燥。我们青春期时也会觉得枯燥。因此我回复：“感觉你们每天也蛮辛苦和枯燥的”，目的是进行简单的情绪共鸣。

这里也运用了上一节课简单提到的“双线理论”聊天技巧：一句正经，一句不正经。这样的聊天方式会让女生很喜欢。

**公式：双线理论**
`聊天内容 = 正经话题 + 不正经话题`

女生回复：“我也不想，确实好累”。这时我没有继续接这个话题，因为前期需要多切换话题。

很多人会逮住这个机会，想和妹子大聊特聊训练、追逐梦想等。一定要记住，微信聊妹的前期要浅尝辄止，不断切换话题，不要聊得太深，让妹子感觉聊天很轻松。

## 切换话题与应对挑战

这时，我另起了一个话题：“话说你在干嘛”。她回复：“在看电视，你呢。没出去喝酒吗”。

这个妹子对“我”的印象是爱玩、喜欢喝酒，即被贴上了标签。被贴标签当然不好。

我回复她：“我大部分时间也是不喝酒的，我也是优秀的年轻人，徐同学”。这里的“优秀”运用了自恋技巧。

兄弟们需要学会把一句普通的话说得稍微有趣一点的能力。

本来我试图将聊天节奏带入平缓，因为刚才已经进行了情感共鸣。结果这个妹子来劲了，她说：“大部分时间不喝，因为天天喝”。

这时不能顺从她。如果顺从就显得太听话，或者只是尴尬地“哈哈”一笑，妹子会觉得聊天没有挑战性。

接下来这一大段是与妹子互相挑战的聊天，体现你的挑战性。这是很多兄弟欠缺的，因为他们怕得罪妹子，容易掉进女生的框架，被主导节奏。

当然不能屈服。我说：“说话怎么这么欠，你啥心做的，欠做的”。妹子反应很快：“那你是酒瓶做的”。她一直强调爱喝酒。

这时两个人有点掐起来了。有兄弟会担心一直这样下去不利于拉近关系。

接下来，我要告诉大家，在双方观点完全不同时，如何既不搞砸，又不显得认输，还能保持自己的框架。不是那种一面对冲突就很怂、很讨好的男人。

聊天看起来是话术，其背后体现的是你是一个什么样的男人。

方法是求同存异，合并框架。看我是怎么回的：“酒瓶做碰上欠做，可以来掰个靶子”。“掰个靶子”这句话，幽默又温柔地化解了僵持的关系。

这样一轮下来，至少你没有输。

妹子笑了，说：“哈哈僵还是宝哥啦”。这句话不需要去接，不要继续硬杠。

继续切换话题。我回复：“你调皮眼，明天要跟你玩骰子”。妹子说：“我要先让你三杯吗”。

你会发现，其实已经在聊邀约的事情了，已经在描绘一个邀约的场景。这样邀约还会难吗？

我回复：“为什么”。她说：“这样显得我比较厉害，先震慑你”。这个妹子很可爱。

我抓住了“震慑你”这个关键词，说：“你震慑我的方法有很多，比如说在酒吧玩倒立”。这是调侃的文字，“在酒吧玩倒立”很有画面感和喜感。

妹子回复：“不知道说什么好，我先劈个岔吧”。这也是一个段子，可以明显看出妹子进入到了你的节奏。

这时，差不多又可以主动结束对话了。昨天我们讲过，不要做说最后一句话的那个人，要在最高潮的地方结束对话，让女生产生期待感和意犹未尽的感觉。

于是我回复：“别闹，明天跟你联系”。她回复：“OK OK”，加上一个老实人的表情。服从度还是蛮高的。
