# 倪《微信聊妹术》：第二集：聊天时，如何建设强大心态_note

在本节课中，我们将要学习在与女生进行微信聊天前，如何建立正确且强大的心态。心态是成功沟通的基础，一个良好的心态能让你在互动中更加从容、自信，并有效提升吸引力。

## 概述：心态的重要性

许多兄弟知道聊天需要自信和坦然，但具体如何做到却感到迷茫。单纯告诉自己“要自信”往往没有效果。本节课将为你拆解三个核心心态，它们并非空洞的自我暗示，而是基于实际行动和思维模式的转变。

## 核心心态一：Do It 心态 🛠️

上一节我们提到了心态建设的重要性，本节中我们首先来看看最根本的 **Do It 心态**。

传统的心态建设方法，如自我暗示（例如不断告诉自己“我很优秀”），往往效果有限。因为世界的普遍价值观（例如女生可能天生偏好高富帅）不会因个人的自我暗示而改变。你能做的是接受现实，并着手改变自己。把妹的核心不在于你如何看待世界，而在于女生如何看待世界。

因此，你需要的是 **Do It 心态**。这种心态强调通过**实际行动**来获得真实的自信，而非空想。自信来源于你切实变好的过程与结果。例如，通过形象改造、健身、阅读、广泛社交等具体行动，让自己真正变得优秀，并因此获得他人（包括女生）的正面反馈。这种基于成就的自信才是坚实可靠的。

**公式：自信 = 实际的自我提升 (Do It) + 外界的正面反馈**

唯有踏踏实实地去做（Do It），才是建立自信的核心与正道。

## 核心心态二：提供价值的心态 🎁

在理解了 **Do It 心态** 是根本之后，由此衍生出的第一个重要聊天心态是 **提供价值的心态**。

在与妹子聊天前，你需要明确一个核心原则：尽量多地去**提供价值**，而非**索取价值**。请记住以下公式：

**公式：提供价值 = 产生感觉**

只要你提供的价值足够多，女生对你的感觉就会更好。提供价值主要分为两类：情绪价值和实际价值。

以下是提供价值的具体分类与方法：

**1. 情绪价值**
情绪价值并非指你自己傻乐，而是指你能**影响她的情绪**，制造情绪的波动（如让她笑、紧张、期待、失落）。可以制造情绪波动的聊天技巧主要有三个：
*   **推开拉近（推拉）**：先赞美（拉近），后调侃或否定（推开）。例如：“你的眼睛好特别…像流氓兔。”
*   **骄傲风趣**：以有趣、放松的语气进行幽默的自恋或自我赞美。例如：“像我这样帅气又单纯的男人，你千万不要套路我。”
*   **挑刺和赞美**：先故意挑一个小毛病（挑刺），再进行赞美。例如：“你这件衣服有点像我奶奶家的窗帘…但是怎么穿在你身上就这么合适呢？”

**2. 实际价值**
实际价值指女生可以直接用眼睛看到的、能展示你生活状态的价值。例如，通过朋友圈或私聊分享高质量的生活瞬间：在精致的餐厅吃饭、展示厨艺、进行有趣的休闲活动等。关键在于分享的内容本身**必须具有吸引力**，避免在脏乱差的环境下拍照。

相反，不断追问“在干嘛？”“吃了吗？”等行为，则属于**索取价值**，会降低吸引力。

## 核心心态三：游戏者心态 🎮

上一节我们介绍了如何通过提供价值来吸引对方，本节中我们来看看能让你保持轻松状态的 **游戏者心态**。

太多兄弟在聊天中过早“较真”，投入过多情感，导致需求感过强、得失心重。**游戏者心态**旨在帮助你保持轻松和主动权，它包含两种具体心态：

以下是游戏者心态的两种实践方式：

**1. 一边玩一边撩的心态**
与妹子聊天时，不应放下手中所有事情、全程捧着手机等待。正确的做法是像打游戏一样：收到信息后，可以忙完手头的事情再回复。这有助于降低你的需求感，避免因过度重视而产生焦虑。

**2. 有选择性的心态**
你不能只和一个妹子聊天。如果资源过于单一，你会将所有精力和情绪都寄托于一人，对方的一举一动都会让你紧张不已。拥有选择权是一种心态，更是一种实际状态。这种状态需要通过积累更多的社交资源来获得。就像游戏中有多条命时，你的心态会更放松，表现也会更好。

2.  **提供价值的心态**：牢记“提供价值=产生感觉”，多分享情绪价值和实际价值，避免索取价值。
