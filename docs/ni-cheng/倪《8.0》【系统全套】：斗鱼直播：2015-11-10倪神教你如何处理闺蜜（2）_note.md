# 倪神社交教程 8.0：如何处理闺蜜（2）👭

在本节课中，我们将学习在社交互动中，如何通过恰当的言行维护团队氛围，避免因不当的玩笑或打压行为而降低自身价值。

上一节我们介绍了社交互动的基本框架，本节中我们来看看一个具体的言行原则。

## 核心原则：维护队友，而非打压

在社交场合，与你同行的朋友或“闺蜜”是你的队友。核心原则是维护他们，而不是通过打压他们来彰显自己。即使是以开玩笑的方式，直接打击队友的外貌或价值也是不明智的。

**公式：社交价值 ≠ 打压队友**

以下是具体的行为解析：

当你的女性朋友被要求喝酒时，你可以用轻松的方式为她化解，而不是附和他人去指责她。例如，对方说“女生你过来先喝一杯”，她问“干嘛”，你不应该说“你长得那么丑你先喝”。

**关键点：** 永远不要打击你的队友。你或许可以开队友的玩笑，但目的应是增进融洽，而非贬低。

## 原理阐述：为何不能打压队友？

你知道为什么要这么做吗？这并非意味着你完全不能开队友的玩笑，而是需要把握分寸。

人们通常会形成一种直觉：如果一个男人通过打压自己人来获取关注或优越感，会让人觉得这个人格调很低。

## 类比说明：老板与员工的例子

为了更清晰地理解这一点，我们可以举一个简单的例子。

设想一个场景：一位老板和他的员工一同外出。

以下是基于这个场景的推演：

如果这位老板在公开场合不断贬低或取笑自己的员工，旁观者不会觉得这位老板很厉害，反而会认为他缺乏风度和领导力。

这个道理在朋友间的社交互动中同样适用。维护你的朋友，就是在维护你自己的社交形象。
