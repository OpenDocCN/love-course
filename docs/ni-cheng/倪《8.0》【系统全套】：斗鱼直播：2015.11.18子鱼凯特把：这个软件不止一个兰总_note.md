上一节我们概述了课程目标，本节中我们来看看展示面的核心——头像选择。第一印象至关重要，头像是你展示面的门面。

**核心要点**：你用作头像的第一张照片必须是最帅、最好看的一张。如果第一张照片很差，对方根本不会查看你后续的内容。

了解了头像的重要性后，我们来看看其他照片选择的常见错误。展示面需要一致性，反差过大会让人困惑。

对方有一张照片其实挺好，但为什么不用作头像？他后面还放了一张和女生的合影，看起来太像情侣了。

以下是照片选择的两个主要禁忌：

**核心要点**：**展示面的一致性**是建立可信形象的关键。巨大的反差会摧毁你精心营造的第一印象。

## 第五部分：实战聊天记录分析（上）- 开场与兴趣测试

现在，我们进入实战分析环节，通过一段真实的聊天记录，学习如何开局。案例发生在11月15日，从探探匹配开始。

匹配后，我发送了一个笑脸表情。当时是凌晨1点，我发送了开场白：“还不睡”。对方回复了一个伸舌头的调皮表情。

**技巧解析：兴趣指标测试**
在发送表情对方回复后，这本身就是一个**兴趣指标测试**。对方回复你，表明她对你有初步的兴趣，愿意和你聊天。这个表情可以看作是一个**IOI**。

测试通过后，聊天就会更容易进行。开场白“还不睡”在凌晨1点这个时间点，是合理的，也表明对方当时可能有较大的聊天窗口。

**代码式流程**：
```
匹配成功 -> 发送友好表情 -> 收到回复（IOI）-> 进行简单开场（如“还不睡”）-> 观察对方回应积极性
```

## 第六部分：实战聊天记录分析（中）- 拉升关系与解读回应

开场顺利后，下一步是尝试拉近关系。对方回复“刚吃完嘻嘻”，这是一个非常积极的信号，表明聊天窗口在扩大。

我回复了“诱惑我”。这句话的目的是为了**拉近彼此距离**，制造一种熟人之间开玩笑的氛围。

对方回复了一个“？”。站在女生角度，回复问号可能有多种含义：
1.  询问“你想干嘛？”
2.  兴趣度不高。
3.  可能当初是误划（匹配错了）。

相比回复表情，问号的态度通常更冷淡一些。但案例中的女生回复的是表情，说明兴趣度不错。

**核心要点**：聊天时，**观察女生的反应比纠结自己说什么更重要**。对方回复“刚吃完嘻嘻”并附表情，窗口很大；如果只回“？”，则需谨慎。

## 第七部分：实战聊天记录分析（下）- 话题延伸与框架维持

当女生展示出高兴趣窗口时，如何延续话题？案例中，我通过个人资料里的“超能力”设置（写了“希望能喝酒”）自然地延续了话题。

随后，女生问“你是来这里带走别人的吗？”，这可以视为一种**废物测试**，她在试探你的意图。对此，我没有直接回答，而是用“我是拒绝的”和表情来应对，保持轻松框架。

**关键策略**：
*   **利用资料制造话题**：个人资料的细节可以成为聊天的切入点。
*   **幽默与暗示**：用“怕被带走”这类话术，可以微妙地展示高价值。
*   **应对测试**：面对女生的试探（废物测试），不必过于严肃地拆解或迎合，保持轻松、不置可否的态度往往更好。

**总结应对模式**：**展示价值 -> 接收IOI -> 幽默互动 -> 轻松应对测试 -> 引导至微信**
