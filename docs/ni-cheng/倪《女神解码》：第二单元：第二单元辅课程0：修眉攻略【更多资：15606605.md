## 修眉工具介绍

以下是修眉所需的两个核心工具及其用途：

*   **眉刀**：一个小刀。它是用来刮掉眉毛边上长出的一些小碎毛。
*   **眉剪**：这是用来把长得过长的眉毛修剪整齐的工具。

其实，我们有这两个道具就足够了。
