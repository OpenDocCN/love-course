## 六、核心前提：成为有能力提供价值的男人

在进入正题前，我想强调一个核心前提：**要成为一个有能力提供价值的男人**。
请注意，是“有能力提供”，而不是“一直去提供”。这意味着：当我想让女生开心时，我有办法逗她开心；当我想让她体验更好的生活时，我有能力做到；当我想调动她的情绪时，我能做到收放自如。这才叫提供价值的能力。
这适用于所有社交关系（父母、朋友、伴侣）。当你成为一个有能力提供价值的人时，你就会成为一个极具吸引力的人。

## 七、第一课核心：聊天思维与心态建设

很多兄弟会想：怎么又是讲思维心态？能不能直接讲干货？
我想说，这就像前戏，必须做足，体验才能最好。思维和心态是地基，很多兄弟在这方面问题严重。本节课程的目的有两个：
1.  **掌握微信聊天的正确思维**：很多人聊天没有目的和方向，拿到妹子不知道往哪里聊。
2.  **修炼强大的内心，为实战做准备**：很多人面对高价值女生会紧张、害怕对方生气、对方不回信息就焦躁不安，这都是内心不够强大的表现。

### 7.1 分析现状：你的竞争对手真的很强大吗？

在社交网络不发达的年代（短信时代），妹子社交圈窄，竞争对手少，按照“三次约会”等原则很容易成功。
但现在不同了。互联网连接了所有人，一个妹子可能刚和你约会完，就在回家的路上和北京、甚至美国的高富帅聊天。你的竞争对手从同城变成了全国乃至全球。
这听起来很绝望，但我想说，我刚才举的例子（全球高富帅）只占男人的10%。而且，我们的优势在于：**大部分男人的聊天都非常无聊**。
现在的女人就像被反复垂钓的鱼，对普通的告白、弹琴唱情歌等老套方法已经完全免疫。但很多男人还在用这些方法，因为他们受老一辈电视剧的影响。

### 7.2 微信聊天的本质

**微信聊天就像一支笔，在用文字、图片、语音描绘你的自画像给女生看。**
聊天不好的人，画出的自画像丑陋、烦人、显得跪舔；聊天好的人，画出的自画像帅气、有趣，女生一看到他的信息就愿意点开回复。
你时刻都在通过聊天影响着自己的形象。

### 7.3 让自画像变丑的罪魁祸首：暴露需求感

很多兄弟经常暴露需求感而不自知。例如，加到一个胸大的妹子，手脚发麻，脑子里开始幻想开房细节，这时说出来的话味道就变了；或者妹子不回信息，你就变得奇怪，去追问。
**为什么不能暴露需求感？**
1.  **欲望会被控制**：别人一旦看到你的欲望（小尾巴），就可以通过它来控制你（例如让你花钱、讨好）。**无欲则刚**，当别人看不到你的欲望时，就无法控制你。
2.  **女生会“耍贱”**：你越想要，她越不给，以此来显得自己矜持、价值高。
3.  **显得低价值**：一个被下半身（欲望）控制的男人，是低价值的。高价值男人不会轻易被欲望控制。

**暴露需求感的具体表现有哪些？**
1.  **不断问问题**：这是男性解决问题式的思维，但恋爱不是解决问题。
2.  **回复过快和过剩**：秒回、回复字数太多。同样是解决问题思维，但恋爱是享受相处，而非快速解决问题。
3.  **过早表达兴趣和讨好**：比如不过脑的赞美。
4.  **无病呻吟**：说一堆与女生无关的话题，会让女生觉得“你跟我说这些干嘛？”，唯一的解释就是“你想泡我”，从而暴露需求感。

### 7.4 女生喜欢男人在微信上表现成什么样？

大家可能都能说出一些关键词：幽默有趣、展现价值、有态度、很酷、有自己的事情做……
但问题是：**知道很多道理，却依然不知道怎么做**。
这里我引出我们整个聊天课程的核心框架——**四大框架**。后续所有聊天技巧都将围绕这四大框架展开：
1.  **绅士礼貌**
2.  **有趣吸引**
3.  **浪漫贴心**
4.  **真实流氓**

这四大框架为你提供了清晰的聊天方向和自我塑造的目标。我们会在后续课程中，将开场、过渡、调情等具体技巧植入这些框架中进行讲解。

## 八、建设强大的内心

### 8.1 常见心态问题：面对高价值女生就懵逼

为什么看到漂亮或高价值的妹子就不敢说话？
1.  **过往经验误导**：过去的经验告诉你“她们要求很高，只喜欢高富帅”。
2.  **潜意识自我否定**：认为自己配不上对方。
3.  **被结果驱动，导致过程痛苦**：你一直在想象和她在一起或开房的结果，导致当下的聊天过程变得痛苦和焦虑。你应该**专注于享受聊天本身**，而不是被想象中的结果所驱使。

**重点：你可以否定你现在的方法，但不能否定你人的本身。**
我曾帮助一位非常普通、不自信的学员。通过用他的身份和展示面去聊天，我们成功吸引了不错的女生，甚至有一次三人行的机会。这件事彻底颠覆了他的认知。他明白了：**不是他这个人不行，而是他的方法不对**。
如果你也有“等我有钱了/开跑车了，就能搞定一切”的想法，这是错误的寄托。你应该专注于提升自己的技巧和能力。

### 8.2 如何内心强大？—— “不缺”与“不屑”

内心强大的外在表现可以归结为两点：
*   **我不缺**：不是指你拥有很多，而是指你有能力、有选择。如果你能轻松找到其他女生，就不会对某一个过分执着。
*   **我不屑**：对无谓的挑衅、女生的威胁（如“不来找我就删了你”）保持淡然。真正强大的男人有自己的标准和原则。

### 8.3 了解关于女人的事实

要内心强大，你需要了解一些关于女人的事实：
1.  **她们也是普通人**：也要吃喝拉撒，会打呼、放屁，有非常不雅和屌丝的一面。了解这一点，能帮你祛魅，不再把她们神化。
2.  **她的价值来源于你怎么看待她**：你把她当女神，她就是女神；你把她当普通女孩，她就是普通女孩。
3.  **她们遵从“膝跳反射”**：你可以找到刺激她们反应的点（如物质、幽默、被带领），只要方法对，完全可以应付得来。
4.  **女人最终属于性格和外在都强大的男人**：这提醒你要成为一个真正强大的男人。

### 8.4 要做哪些事来让自己内心强大？

1.  **做一个有选择的男人**：增加你的资源（认识更多女生）。当你别无选择时，容易做出不理智的傻事。人的终极目标之一就是成为有选择权的人。
2.  **持续地自身建设**：包括提升衣着品味、健身、锻炼口才、增长见识、阅读、培养善良品质等。**自信来源于你做了多少事，获得了多少正面反馈**，而不是听了多少道理。
3.  **持续地练习和训练把妹技术**：包括话术、聊天艺术等。我本人也一直在坚持这样做。
