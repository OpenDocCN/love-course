# 倪《长期迷恋》课程第十一集：如何正确给女友道歉？——承担责任的正确方法 💔

在本节课中，我们将要学习如何向女友进行正确的道歉。这与之前课程中提到的“哄女友”技巧不同，本节重点在于当我们确实做错事、伤害了对方时，应如何承担责任并有效道歉。

上一节我们介绍了处理女友情绪的方法，本节中我们来看看当错误确实在我们时，该如何行动。

首先需要明确，本节讨论的道歉有其特定前提。有些男性的道歉非常廉价，只要女友生气就不分对错地说“对不起”。这种道歉只会让自身价值降低，在女性眼中显得懦弱，且无法真正解决问题。我们所说的道歉，适用于你**确实做错并伤害了对方**的情况。此时你必须敢于道歉，而非在没错时也道歉。

例如，你答应共度纪念日却因故爽约，这就需要道歉。但道歉不等于反复说“对不起，下次不会了，再犯就如何如何”。这种道歉本质是试图用未来承诺平息对方当下怒火，是**不愿承担当下责任**的表现。

在我看来，真正的道歉包含三个核心步骤。

以下是道歉的三个核心步骤：

1.  **主动接受事实**
    不要掩饰或辩解。例如你迟到，就说：“我迟到了，让你等了50分钟。”这叫承认事实。而不是说“至于吗？不就等了一会儿？”这类试图掩饰的话只会让对方更恼火。

2.  **承担后果**
    接受对方因此产生的所有情绪。对方生气、抱怨，你都应接受，因为这是你的问题导致的。你需要承担一切可能出现的后果。避免说“你怎么还生气？我不是解释了吗？我也很难过，你别这样。”这些话的本质是回避责任。承担责任是男人成熟的体现。回避责任，即使能言善辩或外表出众，在女生心中也依然是个没长大的孩子。

3.  **用行为弥补**
    我不喜欢用空口承诺未来，那只是“打嘴炮”。造成伤害和损失，就直接进行弥补。例如，对她说：“走，我带你去吃最好的烤鸭作为弥补，再看场电影，还可以答应你一个要求，你觉得怎么样？”这就叫用行为弥补。而不是说“以后我不会再这样了”。**以后再犯，再次弥补即可**。

如果你始终坚持用行为弥补，就不会出现女友在吵架时翻旧账的情况，因为“旧账”都已清偿。你们的关系会因此变得干净、纯粹。

听到这里，你是否发现，这里没有奇技淫巧，只有最本质、最朴实的道理。这是“大道”，是坚持做正确的事。恋爱需要做正确的事，做人亦然。如此，一切自会水到渠成。

本节课中我们一起学习了真正有效的道歉三步骤：**主动接受事实、承担后果、用行为弥补**。关键在于摒弃廉价的口头道歉，敢于为错误承担责任，并通过实际行动进行修复。这就是如何向女友正确道歉的全部内容。
