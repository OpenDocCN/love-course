# 倪《女神解码》课程笔记：第二单元辅课程02 - 发型打理篇：庞毕度发型 💇‍♂️

在本节课中，我们将学习如何打理一款经典的发型——庞毕度发型。我们将从准备工作开始，逐步讲解造型产品的选择与使用，最终完成一个持久有型的发型。

## 准备工作：吹干头发

打理发型的第一步是确保头发完全干燥。请使用吹风机将头发彻底吹干，为后续造型步骤做好准备。

## 造型产品：发蜡的选择与使用

上一节我们介绍了准备工作，本节中我们来看看核心造型产品——发蜡。发蜡用于塑造和固定发型的基本形状。

*   **产品选择**：视频演示中使用的品牌是“湿滑扣”。你可以根据个人喜好选择任何你信赖或喜欢的发蜡品牌。
*   **使用方法**：
    1.  取适量发蜡于掌心。
    2.  在双掌间均匀搓开，使其软化并均匀覆盖手掌。
    3.  将发蜡涂抹于干燥的头发上，并抓出你想要的发型轮廓和纹理。

## 最终定型：使用定型喷雾

使用发蜡塑造出基本形状后，为了对抗外部因素（如风）并让发型保持更长时间，我们需要进行最后一步——定型。

以下是使用定型喷雾的步骤：
1.  在完成发蜡造型后，保持约20-30厘米的距离。
2.  将定型喷雾均匀喷洒在头发上。
3.  喷洒后无需再次整理，让喷雾自然风干定型即可。

至此，庞毕度发型就打理完成了。

## 课程总结

本节课中我们一起学习了庞毕度发型的完整打理流程。关键步骤包括：先将头发吹干，然后使用发蜡（如**湿滑扣**品牌）抓出理想形状，最后使用定型喷雾进行持久固定。掌握这些步骤，你就能轻松打造出一款有型且持久的发型。
