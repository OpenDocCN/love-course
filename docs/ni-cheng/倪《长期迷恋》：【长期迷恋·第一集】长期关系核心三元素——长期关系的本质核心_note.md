# 倪《长期迷恋》课程第一集：长期关系核心三元素 🎯

在本节课中，我们将要学习长期关系的本质核心，即“绑定”的概念。我们将深入探讨构成这种绑定的三个关键元素，帮助你理解如何建立并维护一段健康、平等的长期关系。

## 概述：长期关系的基石

上一节我们介绍了课程背景，本节中我们来看看长期关系最底线的要求。我认为长期关系最基础的一个要求就是平等尊重。这是所有良好长期关系赖以生存的土壤。所有好的长期关系都是建立在这个基础的土壤之上的。否则你学的技巧都是徒劳。

但是，很多兄弟的长期关系并不是建立在平等尊重的前提下。他们的长期关系存在一些问题。

以下是几种常见的不平等关系模式：

*   **模式一**：因为这段关系是跪舔而来的，在长期相处中就各种委曲求全，违背自己的意愿。
*   **模式二**：因为关系不平等，女生总是盛气凌人，站在高点颐指气使。即便有的时候是对方的错，那也是你的错。
*   **模式三**：一旦关系出现危机，永远是男生去修复关系。而女生就是一副毫不在乎这种感情的样子。

如果你的关系是这样的，你的第一步不是要学着更多的相处技巧，而是要把你的关系恢复到平等和尊重的状态下。她都不尊重你，她都觉得你低她一等，你学的所有的相处技巧都像是在讨好她。有一句话说“弱国无外交”，恋爱中也是如此。

那么如何去恢复这个关系呢？我其实在撩法的长期关系三部曲中讲过，就是你需要去引发冲突，然后打一场“三关战争”。既然是打仗，那么就有可能会有伤亡。情感中打三关战争，也有可能会让你失去这种关系。你必须有一个信念：那种不平等的恋爱关系，我宁可不要。我只向往那种美好的平等尊重的情感关系。否则你永远会深陷其中痛苦不堪，最后你甚至开始都会去讨厌自己。

## 长期关系的核心：绑定

那么接下来我们看一下长期关系最重要的核心到底是什么。这个核心就是**绑定**。你想要有更长久的长期关系，你必须跟女生有深层的关系绑定。

那为什么是绑定呢？我们跟任何人在一起，都是因为我们的关系被绑定在一起。

以下是几种常见的绑定关系示例：

*   **同事关系**：因为工作而绑定在一起，所以每天都会见面。
*   **朋友关系**：因为彼此欣赏，在一起舒服而绑定在一起，所以经常会见面。
*   **客户关系**：因为价值的流动，而把关系绑定在一起，所以从完全没有交集的人，产生了联系。

那么我们要长期跟一个女生在一起，我们也需要被绑定在一起。不管是客观条件绑定，还是主观情感的绑定。举个例子，你跟妹子莫名其妙的漂流到了一个岛上，只有你们两个人，你们别无他选，那这也是一种绑定。很多人也会试图用婚姻去绑定一段关系。

## 绑定的三大元素

决定你们能否绑定在一起的因素有哪些呢？绑定一共有三个元素。

以下是绑定的三个核心元素：

1.  **通过彼此投入去绑定**
2.  **通过关系链绑定**
3.  **通过多重需求去绑定**

可能你还不知道是什么意思，没关系，我们继续。

### 元素一：通过彼此投入绑定

我们先看第一个元素，通过彼此投入来绑定。这是很多年轻人恋爱觉得最重要的一种绑定，也就是彼此是否愿意为对方付出。

那么有意思的是，很多哥们的关系是男人付出远远大于女生对他的付出。甚至可以说他们的关系就像是一个人在供养着另一个人。那有兄弟会说，人家妹子也付出也投入了，人家付出了身体。性是一件愉悦的事情，其实男女是平等的，不存在谁付出更多。如果你觉得你赚了，你这个就是交易思维，是所谓的嫖客思维。这种思维是不会有好的长期关系的。

**付出要遵守一个对等的原则**。意思就是说，你跟对方的付出要对等。对等不是说要一样多，根据彼此的经济实力、实际情况等等可以有所浮动。

**公式示例**：
`你的付出 / 你的能力 ≈ 对方的付出 / 对方的能力`

比如说你月收入2万，情人节你送对方一个3000的包，那对方月收入5000，送你一个1000块钱的钢笔，这个也是合理的。但是双方都必须有所行动。另外如果女生单方面为你付出，比如说她不断的给你买礼物，那么你也需要同时反馈给对方，去时刻的保持这样一个投入平衡的状态。否则只要有一方失衡，一定会出问题。始终记住一句话：世界上没有白吃的午餐。

这里有兄弟就会说，谈个恋爱这么计算算计好累。你可以这样子想，任何一段关系都需要磨合的过程。就好像你开一台新车，一开始也需要时间去熟悉，去驾驭一下。比如你喜欢方向盘松一点，但是这个车的方向盘很紧，那么你要么改方向盘，要么你改变习惯。那么在这个磨合的过程，是会有一些不舒服的。当你和女生都认同了彼此付出要平衡这件事情，然后也磨合过了之后，那么接下来你们的关系就会很舒适，很有默契。后面的相处就会更轻松。

而不至于像很多兄弟，一恋爱了就上头，然后不断付出。付出了之后，还会用一句话去催眠自己：“男人嘛就该为女人付出”。那我当然不反对这句话，但是记住一句话：**在关系里，付出者会更爱被付出者**。所以我们不是要女人给我们投入，而是为了让她更爱你，更深度的绑定在一起。我们主动给女生投入可以无所谓，但相应的对方也要给我们付出。

OK这是课程的第一板块，也是绑定的第一个元素：投入。到这里很多人会有一个疑问：我投入了，但是女生不会主动给我们投入，那么我们应该如何去引导对方，为我们投入，以达到投入平衡呢？下一节课，我们会单独讲解这个主题。
