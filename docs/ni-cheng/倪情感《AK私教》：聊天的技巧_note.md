高价值的男人才会挑剔。指出女生不足，让她在你面前不自信（激发她的“孔雀心理”，促使她解释，这是一种投资）。
*   **纯挑刺**：“我发现你鼻子上的黑头有点多。” （可能让对方不开心）
*   **组合挑刺**：“你鼻子上长了颗痘痘，没事说明你还年轻，哈哈。” / “你走路怎么是歪的？好萌啊，跟只小企鹅一样。” （挑刺后逗她，让她害羞）

**5. 展示标准**
框架对方，让她下意识迎合你。
*   **示例**：“我觉得你挺好看的，但是漂亮的女孩都不乖，我喜欢乖的。” / “你身材好瘦，但是我喜欢肉一点的抱着舒服。”

**6. 暧昧与拉伸关系**
当有了基本了解和吸引后，需要合理拉近关系。
*   **可暧昧的话题**：做饭、看电影、看海、逛街、吃冰淇淋等情侣常做的事。
