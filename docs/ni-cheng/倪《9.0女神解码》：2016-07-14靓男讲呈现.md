# 倪《9.0女神解码》：2016-07-14靓男讲呈现

在本节课中，我们将学习如何构建有效的聊天框架，并掌握“推拉”这一核心互动技巧。我们将从盖房子的比喻开始，理解聊天蓝图的重要性，然后深入探讨如何运用逆向思维和推拉技巧，让对话变得有趣且充满吸引力。

## 聊天需要蓝图 🗺️

很多人在聊天时没有明确的框架，就像盖房子没有蓝图一样。没有蓝图，你可能会找来各种材料胡乱堆砌，房子永远盖不起来，或者盖到一半就塌了。把妹聊天也是如此，如果没有一个清晰的思路和结构，你就会东拉西扯，永远无法推进关系。

上一节我们强调了框架的重要性，本节中我们来看看如何构建这个“聊天蓝图”。

## 两种思维模式：理性与感性 🧠❤️

世界上最会把妹的人，往往是诗人、作家、歌手、艺术家。他们擅长把妹，是因为他们和女人一样，拥有**感性思维**。女人是感觉动物，不总是理性的。而最不会把妹的，往往是程序员、科学家等**理性思维**主导的人。

因此，你必须学会建立两种思维：
1.  **逆向思维**：学会抖机灵，说出人意料的话。
2.  **感性思维**：能体会歌词、电影中的情感，产生共鸣和感悟。

## 核心技巧一：逆向思维与幽默感 😄

什么东西会让人发笑？一个核心原理是：**事情的发生与你预料的相反**。

例如，在周星驰的电影或《屌丝男士》中，很多笑点都源于情节发展完全出乎观众的意料。这种“意料之外”制造了幽默感。

培养这种思维，你需要天马行空的想象力，摆脱常规的逻辑束缚。

以下是培养逆向思维的方法：
*   多关注微博上的“点开全文君”，学习里面的段子。
*   多看脑筋急转弯。
*   关注一些幽默的博主（如提到的徐思谦）。

## 核心技巧二：推拉技巧 ⚖️

“推拉”是互动中制造情绪波动的关键技巧。“推”是轻微地打压、推开；“拉”是赞美、拉近。一推一拉，能让对方情绪起伏，对你产生兴趣。

**推拉公式**：`赞美（拉） -> 轻微打压（推）` 或 `打压（推） -> 赞美（拉）`

例如：
*   “你今天穿得真好看。（拉）……别高兴太早，是人比衣服好看。（推后拉）”
*   “你性格好傻逼。（推）……不过我喜欢。（拉）”

关键在于自然衔接，让对话充满趣味性和挑战性，而不是一味的讨好。

## 实战案例解析 💡

让我们通过一个实际聊天记录片段，看看如何应用上述技巧。

**案例背景**：女生开场说“网红好”，带有轻微打压意味。

1.  **展示态度，逆向回应**：
    *   女生：“网红好。”
    *   回应：“我不是网红，你才是。”（逆向否定，同时把话题抛回）
2.  **推拉结合**：
    *   女生否认后，可以说：“那多好，我就不喜欢网红。”（表达自身态度，是“推”的一种形式）
    *   随后转折：“但是我公司就是培养网红的呀。”（制造矛盾点，引发好奇，是“拉”）
3.  **转移话题，展示价值**：
    *   当对话陷入“你是不是网红”的纠缠时，果断转移话题到生活邀约（如“去喝酒吗？”），并展示对本地生活的了解（如区分不同酒吧档次），间接展示了自己的见识和生活圈。

这个案例中，通过保持框架、运用推拉、展示价值，即使中间有波折，后续女生仍会主动联系。这说明了**个人整体吸引力（展示面、生活方式）比单纯的聊天技巧更重要**。

## 格局与心态：超越技巧 🚀

本节课最重要的升华在于：聊天技巧是“术”，个人格局才是“道”。

*   **妹子是跟你的人走，不是跟你的话术走**：真正的高分妹，最终会被你是一个什么样的人所吸引，而不是你多会撩。你的视野、事业、圈子、谈吐构成了你的基本盘。
*   **提升自身价值**：与其沉迷于钻研话术套路，不如花时间提升自己的真实价值——管理能力、运营思维、事业成就。当你的格局达到一定高度时，很多“技巧”层面的问题会自然消失。
*   **拒绝“屌丝思维”**：不要用自己有限的认知去揣测他人（例如认为别人都在用假货、P图）。这种思维会限制你的成长和吸引力。

## 总结 📝

本节课我们一起学习了：
1.  **聊天需要像盖房子一样有蓝图**，避免杂乱无章。
2.  **建立感性思维和逆向思维**，让对话有趣、出人意料。
3.  **掌握“推拉”核心技巧**，通过一推一拉制造情绪波动。
4.  **认识到个人格局和真实价值的决定性作用**，技巧是辅助，自身建设才是根本。

记住，成为一个有魅力的男人，远比成为一个只会聊天的男人更重要。将精力专注于提升自身，你的吸引力会自然流露。
