台阶是一个**合理化的理由**，让女生的每一个服从行为都显得自然，避免她产生社交压力或尴尬。

*   **在酒店房间**：

    *   `台阶2`：“我有点渴了，帮我看看房间里有水吗？”（服从性测试：找水）

    *   `台阶5`：“我怕电视里跳出僵尸，快保护我。”（进挪的台阶：拥抱）
