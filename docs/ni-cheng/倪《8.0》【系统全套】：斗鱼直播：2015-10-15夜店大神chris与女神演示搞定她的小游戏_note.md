# 夜场社交游戏教程 第8.0课：活跃气氛与升高关系的实用技巧 🎲

在本节课中，我们将学习在夜场（如酒吧、KTV）社交中，如何通过游戏达成三个核心目的：灌酒、活跃气氛以及升高与异性的关系。课程内容将分为三个主要部分，每部分介绍相应的游戏玩法与策略，旨在帮助初学者理解游戏背后的社交逻辑，并能在实际场合中流畅运用。

很多人去夜场时，不清楚自己究竟要做什么。例如，在酒吧或KTV的聚会中，常见的行为有三种：第一种人不停地点歌；第二种人在那里玩手机；第三种人虽然不停地喝酒玩游戏，但并未抓住游戏的核心目的。

玩游戏的真正核心并非为了娱乐而娱乐，而是为了把妹。真正的核心目的只有三个。

## 第一部分：灌酒游戏 🍺

上一节我们提到了游戏的三个核心目的，本节中我们来看看第一个目的：灌酒。古人说酒是一种“狂药”，妹子喝酒后更利于你与她进行更疯狂、更刺激的互动，从而进一步拉近关系。

灌酒的核心是让妹子在游戏中自然地饮酒。以下是两个简单有效的灌酒游戏。

### 游戏一：350

这个游戏需要使用七个骰子。以下是具体规则：

1.  每人可以摇三次骰盅。
2.  第一次摇完后，打开骰盅，将点数为1和5的骰子取出。
    *   点数1代表100分。
    *   点数5代表50分。
3.  用剩余的骰子进行第二次摇动，再次取出1和5。
4.  进行第三次摇动。
5.  游戏目标是在三次机会内，使取出的骰子总分数达到或超过350分。
    *   公式：**总分数 = (1的个数 × 100) + (5的个数 × 50)**
6.  如果玩家达到350分，则无需喝酒。超过350分的部分（每100分为一杯酒，50分为半杯酒）可以累积并传递给下一位未达标的玩家。最终未达到350分的玩家需喝掉所有累积的酒。

这个游戏可能让运气不佳的玩家一次性喝下好几杯酒。

### 游戏二：猜数游戏

这是一个适合多人参与的游戏。以下是玩法：

1.  准备一堆骰子（例如8-10个）。
2.  玩家轮流摇一次骰盅，并在摇之前喊出一个1-6之间的数字。
3.  摇完后，如果骰子中有该数字，则将所有该数字的骰子取出，游戏继续。
4.  如果骰子中没有该数字，则玩家需喝酒。
5.  随着游戏进行，骰子数量减少，玩家喝酒的概率会逐渐增加。

为了让妹子一次喝更多酒，可以引入“三次机会”规则：当骰子很少时，提议让妹子摇三次，若三次均未猜中，则需喝掉一整杯酒。你还可以运用心理战术，例如在妹子摇完后（她未知结果时），声称“这不是你喊的数字”，诱使她放弃已中的结果而选择重摇，从而增加其喝酒的概率。

## 第二部分：活跃气氛游戏 🎉

在完成灌酒游戏后，我们来看看如何活跃气氛。在社交场合，有时会因大家不熟而冷场，存在“社交黑洞”（如一直玩手机的人）影响整体情绪。这时需要通过游戏快速破冰，调动所有人参与。

以下是几个简单易学、能有效活跃气氛的游戏。

### 游戏一：抽手

所有玩家将一只手背朝上叠放在一起。游戏规则如下：

1.  玩家需按照数字顺序（1, 2, 3...）依次抽回自己的手。
2.  抽手时，必须大声喊出对应的数字。
3.  若多人同时喊同一个数字，或喊的数字顺序错误（如跳过数字），则需喝酒。
4.  最后一名将手抽回的玩家也需要喝酒。

这个游戏考验反应速度，紧张感能迅速带动气氛。

### 游戏二：抓手指

选择一名玩家作为“抓捕者”。游戏规则如下：

1.  抓捕者掌心向下平伸出手。
2.  其他玩家各伸出一根手指，轻轻触碰抓捕者的掌心。
3.  抓捕者数到“3”时，迅速握拳抓取手指。
4.  被抓住的玩家需喝酒。若抓捕者未抓住任何人，则抓捕者自己喝酒。
5.  被罚酒的玩家成为下一轮的抓捕者。

### 游戏三：手指1520

这是经典“十五二十”游戏的变体，使用手指进行。游戏规则如下：

1.  所有玩家伸出右手，握拳后围成一圈。
2.  拳头代表数字“0”。
3.  从一名玩家开始，喊出一个0-15之间的数字（即所有玩家拳头数之和的可能范围）。
4.  如果场上玩家拳头数之和等于所喊数字，则所有玩家必须迅速将手拍向圈中央。
5.  反应最慢、最后一名拍手的玩家需喝酒。
6.  若所喊数字与拳头总和不符，则轮到下一位玩家喊数。

## 第三部分：升高关系游戏 💑

在通过游戏灌酒并活跃了气氛之后，我们进入最终环节：如何利用游戏升高与妹子的关系。这包括语言和肢体上的升级。

以下是两个非常适合在暧昧阶段推进关系的游戏。

### 游戏一：真心话大冒险（骰子版）

这个游戏通常使用“大话骰”（吹牛骰）的规则进行，但将惩罚替换为真心话或大冒险。

1.  **大话骰基本规则**：每人摇骰后，按顺序猜测所有玩家骰盅中某个点数的总个数。1点可以充当任何数字。玩家可以“开盅”质疑上家的叫数，若质疑成功，则被质疑者输；反之，质疑者输。
2.  **真心话大冒险规则**：输家不再喝酒，而是选择“真心话”或“大冒险”。赢家为其出题。
    *   **策略**：如果妹子对你有好感，可能会选择“大冒险”，这为你提供了肢体接触（如拥抱、亲吻）的绝佳机会。若选择“真心话”，则可以询问一些私密问题，进行语言上的关系升级。问题应从普通（如“你多大了？”）逐步过渡到暧昧（如“你喜欢什么姿势？”）。

### 游戏二：我没有

这个游戏适合多人参与，能巧妙地获取妹子的隐私信息。游戏规则如下：

1.  每人开始时拥有“5条命”（可用5根手指表示）。
2.  玩家轮流说出一件“我没有XXX”的事情（例如：“我没有涂口红”）。
3.  在座所有符合该描述（即“没有”该事物）的玩家，必须放下一根手指（失去一条命）。
4.  说过的话题不能重复。
5.  当一名玩家的5根手指全部放下时，需喝一杯酒，然后手指重置为5，游戏继续。
6.  **核心技巧**：通过设计问题来试探妹子。例如，问“没有男朋友的”，观察她是否放下手指，从而判断其感情状态。问题可以逐步深入，如“没有穿紫色内裤的”、“没有来大姨妈的”，以此了解当晚进一步发展的可能性。
