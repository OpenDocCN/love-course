# 倪《8.0》【系统全套】课程笔记 第1课：预选与展示面核心 🎯

在本节课中，我们将要学习社交吸引中的两个核心概念：**预选**与**展示面**。我们将解析它们如何影响你在异性眼中的形象，并理解其背后的原理。

## 预选：降低游戏难度的关键 🔑

上一段对话提到了“预选”。预选是什么意思？给大家解释一下，预选其实就是让女生知道，你身边的女人都很漂亮。它的核心目的是传递一个信息：你并非缺乏异性资源，你的身边不乏优秀的女性。

这会产生一个重要的心理效应。如果女生看到你身边都是长得很一般、气质普通的女性，她来到你面前可能会很自信，甚至表现出高傲的态度。相反，预选能让她觉得“你身边有这么多漂亮的女生，那我是不是不会被你轻易看重？”，从而无形中增加了你的吸引力和选择权。

为什么要运用预选？翻这个预选其实就是，让你在整个社交互动中，把吸引的过程变成一个游戏，并把这个游戏的难度，调成简单模式。它让你更容易操作，建立初始优势。

## 展示面：效果差异的根源 📸
