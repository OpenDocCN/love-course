## 核心原则：渐进与舒适 🧩

所有近挪行为都必须围绕一个核心原则：**确保女生的体验良好**。这意味着每一次接触升级都应以对方的舒适度为前提。

## 近挪三步法 🚶‍♂️➡️🤝

以下是执行近挪的三个具体步骤，它们共同构成了一个流畅且尊重对方的过程。

### 1. 循序渐进

接触必须从安全的、非私密的区域开始，然后逐步向更私密的区域升级。
*   **公式**：`安全区接触` → `试探性升级` → `私密区接触`
*   **示例**：从触碰指尖或手背开始，逐渐过渡到手臂、肩膀。

### 2. 转移注意力

当你要触碰女生某个部位时，不要将注意力集中在触碰点上，而是通过其他动作或话题来分散对方的注意力。
*   **代码描述**：
    ```python
    if action == “将手搭在女生肩上”:
        focus = “指向远处的某个事物”  # 转移注意力
        look_at(“指向的事物”)
        # 而不是 look_at(“女生的肩膀”)
    ```

### 3. 注意释放

每次接触后，要敏锐察觉女生的反馈。一旦对方表现出任何不适，应立即停止接触并自然地将手收回。
*   **核心**：在女生感到不舒服**之前**主动释放，而非等到对方明确抗拒。
