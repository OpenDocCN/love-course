# 倪《撩妹兵法》：03【撩法】第三集——心态之内心强大的阴阳心态（正式课）_note

在本节课中，我们将要学习两种极具魅力的男性心态：阴心态与阳心态。我们将深入探讨其核心概念，并提供具体、可操作的修炼方法，帮助你构建强大的内心，从而在情感互动中更加从容自信。

## 课程概述

心态建设由“心态”与“建设”两个词组成。心态指内心的状态，建设则强调行动与实践。调整心态的主要目的是消除面对异性时的恐惧与茫然。当我们了解恋爱的本质并知道该做哪些正确的事情时，恐惧感自然会消失。恐惧往往源于未知，一旦知晓，内心便会踏实。

然而，仅仅知道理论是不够的。心态的变好需要执行力，需要你去实践。知道与做到之间存在巨大鸿沟。接下来，我们将正式进入两种核心心态的学习。

## 阴阳心态：两种魅力模型

上一节我们介绍了心态建设的基本概念，本节中我们来看看具体的两种心态模型：阴心态与阳心态。这个术语引自国外的一本书，它非常精准地描述了两种魅力状态。

简单来说：
*   **阴** 就是 **不受人影响**。
*   **阳** 就是 **影响他人**。

为了便于理解，我们可以参考两位NBA巨星：
*   **凯文·杜兰特** 是“阴心态”的极致体现。他在关键时刻能完全屏蔽外界干扰（如客场球迷的嘘声），表情毫无波澜，专注完成投篮。这种极致的“不受影响”能给予身边人强大的安全感。
*   **勒布朗·詹姆斯** 是“阳心态”的典范。他外放、张扬，充满领袖气质，通过动作、情绪感染全场，影响队友与球迷，主动输出价值与能量。

这两种都是极具魅力的状态。我们的目标就是将这两种“补丁”安装到你的行为系统中。

## 如何修炼“阴心态”：不受外界影响

理解了阴心态的概念后，我们来看看如何具体修炼它。许多人容易受他人影响，尤其是受心仪女生的影响，这会导致情绪波动和行为失措。

以下是修炼阴心态必须做到的三点核心：

1.  **尊重自己的标准**
    在互动中，你必须拥有并坚守自己的底线。例如，标准可以是“绝不低声下气讨好女生”。一切违背此标准的事都不做。当你无比尊重自己的标准时，他人反而会更尊重你。

2.  **拥有选择的能力**
    这意味着，即使失去当前这位女生，你也有能力和信心去认识下一个。这种能力源于你已打造的吸引力和社交能力。没有选择权的人生是悲哀的，它会让你因为害怕失去而委曲求全。

3.  **接纳自己，突破障碍**
    每个人都不完美。阴心态要求你完全接纳自己的缺点，甚至缺陷。当别人攻击你的弱点时（例如身高），无需解释或掩饰，简单的承认（如“对呀”）然后继续做你该做的事即可。真正的强大在于无视那些你无法改变的部分。

**核心公式：阴心态 = 尊重自我标准 + 拥有选择权 + 完全自我接纳**

修炼阴心态没有捷径，需要你像打磨最重要的作品一样，持续地爱惜和打磨自己。

## 如何修炼“阳心态”：影响与辐射他人

掌握了内在稳定的阴心态后，我们转向对外辐射能量的阳心态。阳心态是一种主动输出价值、影响他人、掌控局面的状态。

我曾在一个生日局上，尽管不是主人，但我主动照顾在场的每一个人，打破沉闷气氛，组织大家玩耍，让所有人都感到被关注和快乐。这种主人翁式的付出，让我成为了全场的焦点，吸引力剧增。

想要培养阳心态，你需要做到以下两点：

1.  **乐于分享价值**
    你必须坚信：**分享越多，收获越多**。要主动提供价值，无论是情绪价值（如带动气氛）还是实际价值（如分享资源）。即使你目前觉得自己“一无所有”，从关心他人、倒茶水、切蛋糕等小事做起，也能培养这种利他思维。与之相反的是“索取思维”，这只会让人远离你。

2.  **保持精力充沛的状态**
    当你出现在人前时，必须保持热情、积极、充满能量。避免抱怨、背后说人坏话或情绪低落。如果你自己能量不足，应该独自处理（例如听振奋的音乐调整），而不是将负能量传染给他人。一个阳性的男人，应该把快乐带给别人。

**核心代码：阳心态 = 主动分享价值 + 持续输出高能量**

你不必像某些极端例子那样夜夜笙歌，但至少在与朋友相处时，要能展现出充沛的精力与积极的状态。

## 阴阳心态的融合与选择

需要强调的是，阴与阳是两种并不冲突的心态模式。你不需要两者都达到极致。

*   有些人天性更偏向内敛、稳定，将 **阴心态** 修炼到极致就非常有魅力。
*   有些人则天性外放、喜欢影响他人，将 **阳心态** 发挥到极致也同样吸引人。
*   你也可以根据场合和自身状态，在不同程度上调用这两种心态。

关键在于，选择最适合你本性的那种模式，然后终身去打磨它。

## 课程总结

本节课中，我们一起学习了构建内心强大的两种核心心态：

*   **阴心态**：一种 **不受外界影响** 的稳定状态。修炼方法是：**尊重自己的标准**、**拥有选择的能力**、**接纳自己突破障碍**。
*   **阳心态**：一种 **主动影响他人、输出价值** 的辐射状态。修炼方法是：**乐于分享价值**、**保持精力充沛**。

心态建设非一日之功，需要你在知晓道理后，坚定地去实践、去经历、去修正。祝愿各位都能修炼出强大的心态，从容面对情感世界。
