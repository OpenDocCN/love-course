## 核心一：试探 🕵️♂️

上一节我们介绍了课程的整体框架，本节中我们来看看第一个核心：试探。试探是指在聊天初期，双方互相探测对方兴趣度和状态的过程，而不是一上来就猛烈进攻。

你跟刚加的女生需要互相试探对方对自己的兴趣度。而不是一开始就进行密集的聊天。很多人会“用力过猛”。

以下是“用力过猛”的常见表现：

*   **过早暴露目的**：比如第一次聊天就表达好感、询问对方是否有男朋友、或第一次聊天就让对方发照片。
*   **聊不恰当的话题**：例如在初次聊天时就涉及性话题。
*   **过分讨好**：只会赞美和认同对方，不敢表达不同意见，表现出“老好人”姿态。

而正确的态度是试探。你跟女生刚刚认识，互相并不了解。这就好比**`摸石头过河`**，需要一步步试探。如果你不试探，一上来就“放大招”，就像在拳击比赛中不了解对手就胡乱出拳，会过早暴露自己，使局面变得被动。

聊天是一场博弈。你暴露自己越多，失败的可能性就越大。女生每天会接触很多男性，其中大部分人都表现得一腔热血、非常躁动。然而，试探在聊天初期才是核心。

当你用试探的方式聊天，你会获得有利信息。

以下是试探能带来的四个好处：

1.  **试探出女生对你的好感度和兴趣度**。
2.  **试探出对方当前的状态和聊天欲望**。
3.  **试探出对方的性格和脾气**。
4.  **试探出对方喜欢的聊天方式**（例如，喜欢正经还是有趣，能否开玩笑）。

在聊天初期，了解这些信息对你非常有利。这就像销售产品前，需要先试探客户的购买意向，而不是一上来就推销产品。

要做到有效试探，你必须做好两件事。

### 1. 保持克制

首先，需要保持克制。克制主要体现在两个方面。

**第一，克制情绪。** 不要表现得过于兴奋。即使内心激动，也要表现稳妥。例如，避免使用“啊呀，哈哈，很开心认识你啊”这类过于兴奋的表达。之前例子中男生说“哇，昨天是不是睡得很晚啊，睡了这么久”，这也是一种直男式的兴奋表现。

**第二，克制表达欲望。** 你可能有很多话想说，或急于了解对方，但请务必克制。落实到具体操作，就是每个话题不要过于深入。第一次聊天无需对一个话题刨根问底。

例如，聊到旅行话题时，简单交流两三句即可：
```
你：喜欢去冷的地方还是阳光沙滩？
女生：喜欢阳光沙滩。
你：喜欢阳光沙滩的，都是对自己身材很自信的，哈哈。
```
至此便可打住，无需继续深入分享自己的沙滩经历、炫耀知识或追问对方细节。

保持试探的态度至关重要。如果你一开始就无比兴奋，在女生面前就几乎是透明的。记住，**情绪平稳本身也是一种高价值和魅力的体现**。懂得克制的男人，同样具有吸引力。

### 2. 切换话题

很多兄弟可能会问：保持克制后，本来话就不多，岂不是没得聊了？于是，你需要掌握第二个关键：**多线切换话题**。

这就像钓鱼时，需要在水里投放多种诱饵，才能知道鱼喜欢吃什么。如果你只聊一两个话题，就像只撒一种鱼饵，万一鱼不喜欢，你就一无所获。多线切换话题才能做到真正的试探。

有兄弟会问：切换话题不是和“克制表达欲望”冲突吗？并不冲突。**克制表达欲望是克制不要对一个话题深入展开聊**，而**切换话题是指在不同话题间进行浅层、跳跃式的交流**。这两点在前期必须清晰区分。

切换话题主要有两个作用：

1.  **了解女生对哪个话题更感兴趣**（即“撒鱼饵”的作用）。
2.  **让女生感觉聊天很轻松**。每个人都喜欢被带领，切换话题就意味着你在带领对话。被带领意味着无需自己费力思考，感觉轻松无压力。有些兄弟总问女生“你觉得呢？”“你喜欢做什么？”，这会让女生觉得很累。不断切换话题则会让女生产生一种“跟你很聊得来”的感觉。

方法是：说完一个话题（大约两三句），马上就切换到新的话题。

有兄弟可能会说：可是我不知道该怎么切换话题。不必着急，在后续第二板块的聊天技巧中，我们会讲解如何自然、不留痕迹地无限打开话题。

其实，不仅是你在试探，女生同样也在试探你。这是双方的互动。你见过哪个女生一见到男生，不是表现得很矜持、很克制吗？即便她很喜欢那个男生，也会采用试探的态度。这是两个陌生人刚认识时的正常表现。聊天是一个慢慢互动的过程，表现正常一点、克制一点，女生反而更能接受。

## 核心二：平衡 ⚖️

上一节我们详细探讨了“试探”的核心，本节中我们来看看与之环环相扣的第二大核心：平衡。

什么是平衡？**你与女生的聊天投入度应保持均衡**。你们的关系应该像玩跷跷板，双方的投入在总体上必须保持平衡。如果有一方失衡，情感就会偏向另一方，导致一方陷入被动。现实中，多数情况是很多兄弟自己先失衡了。

要想保持跷跷板的平衡，双方都不能急着“往前走”（即过度投入），否则对方就会把你“撬起来”。

这里说的“投入度”主要指两个方面：
1.  **花在对方身上的时间**。
2.  **花在对方身上的心思**。

例如，上大学时，我曾追求一个低年级的女生。我常常一下课就跑到寝室，坐在床上盯着手机，思考发什么信息给她，连室友叫吃饭都不去。而她呢，总是在参加社团活动、聚会或唱歌。很明显，**我对她投入的时间和精力远大于她对我投入的**。这种不平衡的关系，最终自然没有结果。

投入度的平衡具体体现在以下几个点：

### 1. 兴趣度的平衡

简单说，就是两个人在聊天中表达好感时要平衡。很多人一开始聊天，没说几句就会说“小姐姐你好可爱呀”、“看到你有一种触电般的感觉”之类的话，直接暴露目的。这会导致关系失衡。

插一句题外话：你和女生聊天，女生当然知道你的目的，也知道你对她有意思。所以，你一开始就挑明，纯属多余。女生恰恰喜欢这种朦胧的感觉。一开始就暴露目的，相当于打牌时直接亮明了底牌。对方看清你的牌面，就能轻松应对；如果对方只是猜测，游戏反而会更有趣。这一点非常重要。

### 2. 聊天字数的平衡

这是双方投入度最直观的体现。具体看微信对话框，**你发出的绿色部分（你的发言）和女生发出的部分，在字数上应大致相当**。如果你的绿色部分太多，表明你的需求感过强，没有形成和谐的聊天氛围，还会给女生带来很大压力。我看过很多兄弟的聊天记录，往往是绿压压一片，而女生只回复几个字，这就是严重的失衡。

### 3. 回复时间的平衡

**不要秒回女生信息**。秒回的行为，就像我之前那个例子，抱着手机苦苦等待，而对方却在潇洒快活。回复时间的平衡，不是让你刻意憋着不回（当然你也可以这么做），而是**你确实要有自己的事情做**，不要一门心思只想着这个女生。你需要有自己的精彩生活、兴趣和追求，这才能在本质上吸引女生。如果你整天无所事事，总在问“在干嘛”，吸引力就会大打折扣。

具体操作上，如果女生大约五分钟回复你，你也应该在大约五分钟后回复她。这就是回复时间的平衡。**所有的秒回，都会暗示你是一个无所事事的男人，并潜在地讨好对方**。
