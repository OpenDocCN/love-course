# 倪情感《AK私教》第一课：完美关系的第一步 🚫

在本节课中，我们将要学习一个在亲密关系中非常普遍但极具破坏性的行为模式——**讨好**。我们将深入探讨为什么过度讨好反而会把对方推远，分析其背后的心理学原理，并为你提供改变这一模式、建立健康平等关系的具体路径。

## 概述：为什么“好女孩”反而被冷落？

我们身边常有这样的女孩：在朋友、同事、家人眼中，她性格好、体贴、真诚、善良，宁愿自己吃亏也不愿拒绝别人。拥有这样的伴侣，理应感到幸运。

然而现实往往并非如此理想。在咨询中，我遇到过不少这样的女孩，她们困惑于为何自己努力迁就，男友或心仪对象却越来越冷淡。问题的核心，恰恰在于这种无底线的“好”。

上一节我们概述了“好女孩”的困境，本节中我们来看看其背后的深层心理机制。

## 第一节：亲密关系中的“公平理论” ⚖️

经过与这些女孩的深入交流，我发现她们有一个共同点：在处理情感问题时，因为害怕对方生气或不开心，而选择忍气吞声、一味顺从，极少表达自己的不满。她们在用一种小心翼翼、近乎讨好的方式守护感情。

按照常理，这种乖巧温和的女孩本应得到更好的呵护，但为什么事实恰恰相反？大多数人会为她们打抱不平，认为是对方不懂珍惜。但从情感科学的角度看，人际交往中存在一个核心概念：**公平理论**。

**公平理论**指的是，在人际交往尤其是亲密关系中，人们并非总想以最小付出换取最大回报。双方更在意关系的公平性。当两人付出的成本和获得的收益基本相当时，关系会更快乐、更稳定。

用公式可以表示为：
**`己方收益 / 己方投入 ≈ 对方收益 / 对方投入`**

反之，如果关系中持续存在不公平，一方感到自己**过度获益**，而另一方在**过度付出**，那么双方都会感到不安，并潜意识里想要改变这种状态。

## 第二节：过度付出的反作用：内疚与疏远 😔

作为女性，我们常理所当然地认为，过度付出的一方更渴望改变不平衡的关系。但为什么获益的男性也会感到不舒服并想要逃离？

这是因为在男性思维中，**公平**是一个强有力的社会标准。当一个人获得的回报超过其应得时，他会本能地感到**焦虑和内疚**。试想，如果一个你并不喜欢的低价值男性疯狂跪舔你，送你贵重礼物，你是否会感到不安并想拒绝？

因此，在情感关系中，获得过度回报的一方承受着不小的隐形压力。为了减轻内心的**愧疚感**，他们往往会选择后撤或疏远对方。没有人喜欢长期被内疚情绪包围。

所以，过度讨好和迁就，会在不经意间挑起对方的**内疚感**，最终导致对方疏远，以期心理平衡。

## 第三节：认识“讨好型人格” 🎭

讨好，本质上是一种心理疾病。心理学家哈丽雅特·布拉克在《取悦症：不懂拒绝的老好人》一书中指出，一心想要当好人是一种有害的心理疾病。

这种刻意讨好源于自信心的缺失，渴望通过为别人做事来换取外界的肯定。这种渴望一旦成为心理定势，就会严重降低判断力和自控力，变成一种如同上瘾的**习惯和依赖**。你会习惯性地在所有人际关系，尤其是恋爱中讨好甚至跪舔对方。

以下是书中归纳的三种讨好类型：

1.  **认知型讨好**：你需要并争取让身边每个人都喜欢你。你衡量自我价值的依据是能为别人付出多少。
2.  **习惯型讨好**：你被迫牺牲自己的需求去照顾别人的需要。例如，自己很不开心却怕对方不高兴而强颜欢笑。你几乎从不说“不”，也极少请人帮忙，对亲近之人的请求无力招架。
3.  **情感逃避型讨好**：你为了逃避令人害怕或不安的情感（如冲突、被否定）而选择讨好别人。

关键在于分清：你在感情中过于考虑别人，是真心想为对方做，还是基于以上三种原因，为了获得认可？

## 第四节：讨好模式的根源与形成 🧸

从原生家庭的角度分析，每一个在情感中习惯讨好的人，其行为模式都深受童年影响。

习惯性讨好者，在小时候被灌输了这样一种思想：**爱是有条件的**。只有当你的表现符合父母（最初的重要他人）的期望时，你才能感受到爱，才不会遭到拒绝或批评。

例如，做客时活泼有礼会得到“懂事”的夸奖；若因害羞不语，则可能被训斥。长此以往，你脑中会形成这样的观念：只有做让亲密的人满意的事，才能得到奖励和爱；否则就可能失去认可。

为了获得父母有条件的爱，避免被否定，你只能隐藏自己真实的需求和感受，努力去实现并内化父母的期望，用讨好来迎合他们，做一个“好孩子”。这副“好孩子”的人格面具会伴随你进入成年，成为性格中难以磨灭的一部分。

即使日后独立生活，这种幼年习得的讨好行为模式，也会自动操纵你在所有人际关系中的表现，包括与恋人相处。

## 第五节：如何改变讨好型模式？ ✨

在恋爱中，许多女孩一边羡慕那些无需过度付出就能获得爱的同性，一边又无法从讨好模式中走出来。改变根深蒂固的习惯是困难的，它意味着要打破多年的思维定势，需要坚定的决心和循序渐进的过程。

改变的核心在于两点：

**1. 学会拒绝，表达真我**
不再讨好，并非走向另一个极端——对他人无理由地冷淡和敌对。而是不再逆来顺受、曲意逢迎。当你开心时就笑，该生气时就怒，该争辩和拒绝时，一定要勇敢表达出来。

**核心行动准则：**
```text
if (对方的要求让你感到不适):
    表达你的真实感受 + 温和而坚定地拒绝
else:
    欣然接受或合作
```

当你以清醒的头脑和适当的方式，在亲密关系中塑造出有原则、有姿态的自我时，你反而更能获得对方的关注、理解和真正认同。

**2. 强大内心，提升自我**
讨好往往源于内在的无价值感。内心的空虚无法永远靠外界力量填补。你需要通过不断学习、增长见识、丰富阅历来滋养内心，构建强大的精神世界。

同时，也需注重外在提升（如得体的打扮、朋友圈建设等）。只有当你自己足够强大、内心充盈时，“讨好”才会自然走开，健康平等的爱才会主动到来。

## 总结

本节课我们一起学习了“讨好型人格”如何破坏亲密关系。我们探讨了其背后的**公平理论**心理机制，认识到过度付出会引发对方的内疚与疏远。我们分析了讨好行为的三种类型及其**原生家庭**的深层根源。

最重要的是，我们明确了改变的方向：**勇敢表达真实感受，学会拒绝，并持续投资和强大自我**。请记住，试图取悦全世界，唯独取悦不了自己。真正的完美关系，始于你珍视并坚定地做自己。

所有的好梦，都需要穿过黑夜，付诸行动。改变就从现在开始。
