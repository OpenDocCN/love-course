## 什么是否定？

上一节我们介绍了社交互动中的基本概念，本节中我们来看看“否定”这一具体技巧。否定并非字面意义上的批评或贬低，而是一种通过情绪平淡、态度无所谓的方式，来回应对方过度展示自身价值的行为。

例如，当对方炫耀自身优势时，普通回应可能是恭维或讨好，而否定则是淡淡回应，显得不在意。这种态度能传递出你并非在刻意讨好对方。

---

## 否定的正确使用方式

以下是正确使用否定技巧的几个关键点：

1.  **态度平淡**：否定时应保持情绪平稳，避免表现出刻意或敌意。
2.  **力度适中**：否定力度不宜过重，否则容易引起对方反感。
3.  **非刻意性**：否定应显得自然，而非精心设计的挖苦或讽刺。

例如，在代码中，我们可以这样表示否定的态度：

```python
response = "哦，那挺好的。"
```

这种回应既不热烈也不冷漠，恰到好处。

## 否定的适用场景

否定技巧并非适用于所有情况。它主要适用于那些自信过度、盛气凌人的对象。对于自卑或内向的对象，使用否定可能适得其反。

例如，当对方不断展示自身高价值时，适当的否定可以显得你自身价值更高，从而吸引对方注意。

## 否定的作用与效果

否定技巧虽然看似违背常理，却能产生以下效果：

1.  **降低对方防备**：让对方觉得你并非有意讨好，从而放下戒心。
2.  **突出自身独特性**：与大多数恭维讨好的人形成对比，让你显得与众不同。
3.  **展示高价值**：通过平淡回应对方的高价值展示，间接体现自身的高姿态。

例如，用公式表示否定的效果：

\[
\text{吸引力} = \frac{\text{自身价值}}{\text{对方预期}}
\]

当对方预期你恭维时，你的平淡回应会提升自身吸引力。
