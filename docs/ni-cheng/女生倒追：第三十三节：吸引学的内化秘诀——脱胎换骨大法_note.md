# 女生倒追：第三十三节：吸引学的内化秘诀——脱胎换骨大法_note

在本节课中，我们将要学习吸引学的核心内化秘诀。我们将探讨如何从内心深处建立自信，并让这种自信自然流露，从而真正提升你对异性的吸引力。课程将分析三种提升吸引力的途径，并重点讲解如何通过“内化”过程，实现持久的魅力转变。

吸引力是自我实现的。当你相信对方应该喜欢你时，你对待他的言行举止就会像对待一个追求者。结果就是他真的会不知不觉地被你吸引。

如果你认为女孩子不会喜欢上你，需要借助金钱、地位或技巧来博得关注，那么你的言行举止也会从潜沟通中流露出这种不自信。对方会不知不觉地感受到，从而不会喜欢上你。即使对方最初对你有好感，也会渐渐失去。

这里的关键在于你内心深处的“一杆秤”。对方能感知到你潜意识里的真实想法，而不是你口头告诉自己的。许多人表面上认为自己很优秀，只是眼界高，但这并非内心最深处的认同。我说的是你潜意识里的那杆秤。

我认识一个朋友，上学时有个叫小莉的女生喜欢他，但他不喜欢小莉，他喜欢班里的“班花”。结果他见到小莉和见到班花时，言行举止完全不同。对小莉，他的潜沟通是“你喜欢我”；对班花，则变成了“我喜欢你”。结果就是自己喜欢的人不喜欢自己，喜欢自己的人自己又不喜欢。

这种情况在很多男性身上都发生过，而且不止一次。据我了解，几乎每个人都遇到过类似情况。

当你从内心深处觉得对方应该喜欢你，就像面对曾经追求过你的女孩时，你根本不会担心她是否喜欢你，因为她的追求行为给了你自信。也就是说，对于小莉，你认同自己拥有比她更高的社交价值身份。

但当你看到那位美女或“公主”时，问题就变成了：如何实现高社会地位价值的身份认同？如何提高自己在女性面前的吸引力？

以下是实现高社会地位身份认同、提升吸引力的三种途径：

**第一，纯粹依靠外部因素。**
即通过拥有更多金钱、更高职位等来提高自身价值。目前很多男性都在为此奔波，抱有“挣大钱、买房买车就不怕没有女人”的思想。然而，这种吸引力并不长久。因为女性喜欢的可能不是你，而是你的钱。一旦你的外部条件发生变化，比如没钱了，她可能很快就会离开。

不仅如此，依靠外部物质吸引来的女性，往往是特别追求物质的人。她们并非真正喜欢你。更重要的是，即使是对这些追求物质的女性，如果你完全依赖外部条件而自身缺乏吸引力，她们在相处一段时间后也会觉得无趣而离开。因为对她们而言，找到另一个既有吸引力又能提供物质的男性并非难事。

**第二，操纵你传递给女性的潜沟通信息。**
这就是学习技巧。绝大部分吸引技巧的作用就是操纵你传递给女性的潜沟通信息。因为女性判断是否被你吸引，主要是通过潜意识分析你的潜沟通。

假设一个男性的社交价值低于某位女性，即他内心是低价值身份认同。那么女性就会从他的言行举止中，通过潜意识处理，得出“这是个低价值男人”的结论，从而不喜欢他。

如果这位男性学习并熟练运用了许多吸引技巧，即使他的内心仍是低价值身份，女性也可能因为这些技巧而误认为他是高价值男人，从而被他吸引。当然，如果你的技巧运用不当，在过程中被女性的第六感（即潜沟通）捕捉到低价值信息，吸引力就会失效。

这就是为什么有些人发现，用了很多技巧后，一开始似乎有效，但确立关系后好像又没用了，随后可能会遇到对方不接电话、不回信息等问题。

**第三，实现高社会地位的价值及身份认同，让你自然流露出具有吸引力的状态。**
这样你就不用担心“露馅”，也不必时刻思考该用什么技巧、该说什么话。你自然的行为举止就能让女性产生吸引。就像现在的我，哪怕一句最普通的“你好”，也能对女性产生吸引力，因为我从内心认同自己是一个高价值的身份。

这时，你就可以做最真实的自己。在任何时刻，你想说什么就说什么，想做什么就做什么，你都会对女性具有吸引力。

记得以前在学校时，我问过一个女同学怎样才能吸引女生。她告诉我：“做你自己。”但问题在于，当时的我内心仍是低社会价值的身份认同。无论我如何告诉自己、试图说服自己“我很厉害、女人应该喜欢我”，只要我内心仍觉得自己是低价值男人，我的潜沟通就在告诉女性“不要喜欢我”。

所以，当我采纳她的建议，做我自己、想干嘛就干嘛时，结果非常失败。不是失恋的问题，而是根本谈不了恋爱，好几个女性都不喜欢我。当时的我一心想着干大事、挣很多钱，认为只有有钱了，才会有女性喜欢我、愿意和我在一起。

那段时间，我常常感到寂寞。当时我也坚信，就算做自己也没用。

之后发生的事情，你可能已经知道了。通过吸引学，我的人生彻底发生了变化。我依然吃着六七块钱的盖浇饭，兜里只有十几二十块钱，但美女却像蜜蜂围着花瓣一样缠着我。

有好几年的时间，我一直在尝试用技巧等手段去操纵女性对我的印象和判断。我为自己低价值的身份认同，暂时披上了一层薄薄的高价值身份外皮，让女性以为我是一个高社会地位的男人。这给了我很多接触和了解她们的机会。说白了，就是“披着狼皮的羊”。

但这是必经之路。一开始我还“披”不好，这张“羊皮”总会掉，或者这里露一块、那里露一块。随着经验增加，披的功夫越来越好，渐渐做到滴水不漏。后来我慢慢发现，这层高价值身份的外皮越来越厚，感觉不再仅仅是一层皮了。它开始和我身上的“肉”结合在一起。

渐渐地，这高价值身份认同的“皮”，成了我身体的一部分，而不再是被披着的外衣。这就是“内化”的过程。

在学习吸引学时，无论是心态还是外部技巧，内化都是把原本不会自觉去做、去说的东西，通过实践、练习以及内心心态的调整，逐渐变成属于自己的一部分。它会自然流露，而不需要刻意提醒或去做。

这就是为什么每次参加线下课程的兄弟能够快速改变。在线下，我会让他们一下子接触很多女性、做很多事情，并且立刻发现问题、立刻纠正、立刻改变。这相当于在线下几天内，他们可能比过去20多年做的尝试还要多。

从一个很小的技巧，到整个身份的认同，都存在内化的过程。人生和世界上的许多事物，都有一个必然的过程。好比我们国家近30年的经济发展非常快速，但也不可能一口吃成胖子，需要一个过程。
