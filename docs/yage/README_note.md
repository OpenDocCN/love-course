# 课程名称：Python基础入门 - 第1课：变量与数据类型 🦆

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。这个标签就是变量名，盒子里的东西就是变量保存的值。

在Python中，使用等号 `=` 来为变量赋值。其基本语法是一个简单的公式：

**`变量名 = 值`**

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存储在其中。

```python
message = "你好，世界！"
```

## 基本数据类型 🔤

我们已经知道如何用变量来“装东西”，接下来需要了解我们能存放哪些“类型”的东西。Python中一些最基本的数据类型包括整数、浮点数、字符串和布尔值。

以下是几种核心数据类型的定义和示例：

*   **整数**：不带小数点的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由引号包围的文本，例如 `"Hello"`, `'Python'`, `"123"`。
*   **布尔值**：只有 `True` 和 `False` 两个值，代表真和假。

## 如何查看和修改变量？ 🔧

知道了变量的类型，我们就可以查看它并使用它。使用 `print()` 函数可以将变量的值输出到屏幕。而修改变量的值就像更换盒子里的物品一样简单，只需再次使用赋值语句即可。

以下是相关的操作示例：

*   **打印变量**：使用 `print(变量名)`。
*   **修改变量**：对已存在的变量名赋予一个新值。

让我们看一段代码，它演示了创建、打印和修改变量的完整过程：

```python
# 创建一个变量并赋值
age = 20
print(age)  # 输出：20

# 修改变量的值
age = 21
print(age)  # 输出：21
```

## 总结 📝

本节课中我们一起学习了Python编程的基石。我们首先将变量理解为一个存储数据的命名容器，并通过 `变量名 = 值` 的公式进行赋值。接着，我们认识了整数、浮点数、字符串和布尔值这几种基本数据类型。最后，我们掌握了使用 `print()` 函数查看变量值以及通过重新赋值来修改变量的方法。掌握这些概念是您后续学习更复杂编程知识的关键一步。
