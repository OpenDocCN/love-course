# 恋爱脱单课程 P1：如何面对喜欢的人保持良好心态 💖

在本节课中，我们将学习面对心仪对象时，如何建立并保持一个健康、强大的心态。这是掌握恋爱主动权的关键基础。

我们每个人都听过一句话：遇到喜欢的人，抱着“无所谓”的态度，就能掌握主动权。这句话只说了一半，另一半没人告诉你。试问，有谁在面对真正喜欢的人时，能做到完全“无所谓”？

古往今来，我们听过最多的就是“英雄难过美人关”这类典故。所以，心态问题至关重要。以下干货对男女都适用。很多人讲不清楚如何保持好心态，如何做到“无所谓”。耐心看完，你一定会大有收获。

## 心态的核心：安全感

首先，谈到心态，必然会提到“安全感”。那些在恋爱中唯唯诺诺的人，其实都源于缺乏安全感。他们总试图从对方身上得到反馈，来证明自己是安全的、是被爱的。这是大错特错的。

**安全感从来不是别人给的，是自己给自己的。**

那么具体该怎么做呢？我们不谈具体的聊天公式，而是把视角拉高。

## 建立良好心态的两个核心观点

以下是建立强大心态的两个关键观点。

### 1. 丰富阅历，宠辱不惊

第一点是：自己看得起自己，也要让对方看得起你。答案是**丰富自己的阅历，达到宠辱不惊**。

你是不是那种对方给一点甜头，就感动得不行的人？对方主动一点，你就加倍奉还、火力全开？这种心态其实是在告诉对方：**你现在给我的，就是我能够遇到的最好待遇（天花板）**。这会在对方脑海中形成一个对你的固定认知。

当有一天他不想珍惜你了，这个认知就会成为对方内心贬低你的一个理由。我们从小被教育要“学会感恩”，这没错。但没人告诉我们后面的道理：**没有无缘无故的好**。别人对你好，是因为你身上有他想要的东西（无论是物质还是情绪价值）。

所以，别人对你的好，你就坦然接受。但关键是，你要配得上这份好，让对方觉得你值得拥有更好。这才是健康的恋爱相处状态。

### 2. 扮演好“自己”这个角色

上一节我们讨论了如何通过提升自我来赢得尊重，本节中我们来看看关系中自我的定位。

你这一辈子最该扮演好的角色是**你自己**，而不是“谁的女朋友”或“谁的老公”。否则，你就会不由自主地想要做一些事来迎合这个标签。

谁规定男朋友一定要帮女朋友搬家？谁说女朋友一定要断绝和其他异性朋友的联系？没人说男人必须养着女人，也没人规定女人必须在家带孩子。但是太多人用这些标签束缚自己，对对方俯首称臣，一切以对方不生气为原则。

所以要敢于说“不”。你要让对方明白一件事：**我和你过去不认识，只是现在在一起。如果因为和你在一起，耽误了我一直坚定想要成为的自己，那我们的关系可能就无法继续了。**

你要知道，那些不被珍惜的人，往往把自己活成了对方的附属品。只有让对方意识到，你是一个独立的、随时可以离开的人，但同时**愿意为对方做出一定程度的牺牲**，对方才会懂得珍惜。

## 总结：什么是真正的“无所谓”

本节课中，我们一起学习了建立恋爱良好心态的核心。

爱情中最好的心态就是：**我们相遇是一场缘分。你投桃报李，我心存感激；你若无动于衷，我便拂袖而去。** 这才是真正的“无所谓”。
