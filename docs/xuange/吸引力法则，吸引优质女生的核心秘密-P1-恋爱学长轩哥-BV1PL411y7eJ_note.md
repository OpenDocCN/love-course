# 恋爱课程 01：吸引力法则核心秘密 🧲

在本节课中，我们将要学习吸引力法则的核心秘密，特别是如何在与优质对象互动时，构建健康的心理位置关系。理解并应用这些原则，能帮助你在社交和情感关系中建立更强的吸引力。

## 核心前提：理解人类的“牧场属性” 🧬

上一节我们介绍了课程概述，本节中我们来看看一个关键的心理前提。你必须了解人类的“牧场属性”。这是人类进化以来刻在DNA里的选择，很难被改变。动物界的领袖、民族的斗争崛起，都摆脱不了牧场的特质。放到现如今，它不仅仅体现在物质方面追求依赖，更体现在拥有物质层面以后，对于精神层面的向往。归根结底，我们都无法摆脱变得越来越优秀的想法。

在情感中，“牧场属性”是打开你心理开关的钥匙。只有当你的心理位置和对方一样，或者更高的时候，才会被激活，才会有我们说的爱慕和魅力。相反，比对方低的时候，最多只会产生喜欢。这常表现为“你很好，可以当朋友，但不适合成为情侣”。

**核心公式：吸引力 ∝ 心理位置高度差**

举个例子你就明白了。我们大部分人喜欢上一个人的时候，总会以付出示好来得到对方的重视。即使对方同意了和你的交往，但你心理位置依然是下位者，是不会对你产生爱慕属性的。对方随时随地都会有离开你的可能，而且他也不会觉得有什么不对。

## 核心策略一：不彰显行为 🤫

了解了“牧场属性”这个心理前提以后，我们该如何去做呢？紧接着来看第二点：不要彰显自己的行为。

我们都知道，喜欢一个人，是不可能完全藏住需求感的。此时你就要记住一个“比大小原则”：即使付出了，也是在心理位置的上方。简单来讲，就是当你的行为和对方比较起来，倾向于上位者时，那你的吸引力就是加分的。相反，吸引力是减分的。

以下是聊天中的错误行为示例：

*   **不断追问**：女生没有回复你，你就不断地发消息。
*   **过度夸赞**：不断夸奖对方“你好优秀”、“你好漂亮”、“你是我的女神”。

因为这些行为是在隐藏自己在社交中的卑微，所以你才会不断强化同一个行为，最终心理位置就处于下位者。

所以说，高情商的策略就是不彰显自己的行为。说简单一些，**不要依靠对方来满足自己的情感需求**，这是一种很 low 的心理。比如聊天，我们就是在平等的沟通，我不会期望你关心我，更不会期待你的秒回。但如果你有主动分享自己的状态，那我也会及时接住。

## 核心策略二：合理的冷淡 🧊

当然，你也会遇到对方没有任何回应，或者不是你想要的感受。所以还需要了解第三个策略：合理的冷淡。

这里就可以记住一个“三次原则”。还是拿聊天举例子，在没有得到回应，或者对方的回应不是很热情的时候，选择及时撤离。当然，我不是让你直接扭头就走，愤怒地告诉对方“我们以后再也不联系了”。这就像一个小孩子不开心在闹脾气一样，这不是一个上位者的态度。

所以我们要用一种平静的行为告诉对方：我冷淡了。

以下是“合理冷淡”的行为示例：

*   **延迟回复**：女生上午跟你回消息，你下午或者第二天再回复。
*   **减少主动**：你以前总是提供话题，现在开始你只回答不分享。
*   **简洁解释**：如果对方问原因，你就直接说“最近有些忙”，不要多做解释。

此时，对方会意识到可能要失去你了。这种平静的冷淡，意味着你的心理位置是高于对方的。因为只有上位者和平等关系，才容易让人失去，下位者是不容易失去的。

## 总结与回顾 📝

本节课中，我们一起学习了吸引力法则的核心秘密。

1.  **核心前提**：我们首先探讨了人类的“牧场属性”，理解了心理位置的高低是产生爱慕与吸引力的关键。
2.  **核心策略一**：接着，我们学习了“不彰显行为”的原则，即避免通过卑微的付出来索取关注，保持平等甚至上位的心理姿态。
3.  **核心策略二**：最后，我们掌握了“合理的冷淡”这一技巧，通过平静地后撤来调整关系动力，维护心理位置的高度。

上面分享的两种方式——“不邀功的行为”和“平静的离开”，都属于上位者和平等的心理位置。也只有这种位置，你才有可能打开对方“牧场属性”的开关。它和物质没有太大关联，但却能让你在一段社交关系中，拿捏自己的位置，让自己变得越来越有吸引力。

最后，分享的任何内容，只是为了让双方的沟通更加顺畅。真心才能获得真爱。希望这些内容对你有帮助。
