# 【恋爱脱单】吸引力法则教你吸引

在本节课中，我们将要学习吸引力法则中的一个核心概念，它关乎如何在社交互动中构建健康的心理位置，从而有效吸引他人。这个概念尤其对已经步入社会的成年人具有指导意义。

## 核心理念：构建心理位置

首先需要搞明白一件事。你们从认识的那一刻开始，不是在聊天，不是在吃饭，不是在谈生意，而是在构建彼此之间的心理位置高低。

上一节我们介绍了课程的核心主题，本节中我们来深入了解其背后的前提。

## 前提：人类的慕强属性

第一点前提，你必须了解人类的慕强属性。这是人类进化以来刻在DNA里的选择，很难被改变。动物界的领袖，民族的斗争崛起，都摆脱不了慕强的特质。

放到现如今，它不仅仅体现在物质方面追求依赖。更体现在拥有物质层面以后，对于精神层面的向往。回根结底，我们都无法摆脱变得越来越优秀的想法。

那放到情感中，慕强属性是打开你心理开关的钥匙。只有当你的心理位置和他一样，或者更高的时候，才会被激活，才会有我们说的爱慕和魅力。

相反，比对方低的时候，最多只会产生喜欢。也就是男生经常听的，你很好，可以当朋友，但不适合成为情侣。

举个例子你就明白了。我们大部分人喜欢上一个人的时候，总会以付出示好来得到对方的重视。即使对方同意了和你的交往，但你心理位置依然是下位者，是不会对你产生爱慕属性的。对方随时随地都会有离开你的可能，而且他也不会觉得有什么不对。

所以在以前的视频中，我曾经提到，你从聊天到见面，无时不刻都要展示自己的价值，同时对方也会不断的评估你。

了解了这个心理前提以后，我们该如何去做呢？紧接着来看第二点。

## 策略一：不彰显需求感的行为

不要彰显自己的行为。我们都知道，喜欢一个人，是不可能完全藏住需求感的。

此时你就要记住一个比大小原则：**即使付出了，也是在心理位置的上方**。简单来讲，就是当你的行为和对方比较起来，倾向于上位者时，那你的吸引力就是加分的。相反，吸引力是减分的。

以下是聊天中的具体表现：

就拿聊天举例子。当你刚认识一个喜欢的人，你主动找他聊天，这个行为从一开始，双方的心理位置其实是平等的。但让你慢慢变成下位者的原因就在于，你之后的行为比对方越来越低。

比如以往案例中我经常说的，女生没有回复你，你就不断的发消息。再比如不断的夸奖对方，你好优秀，你好漂亮，你是我的女神。因为你的行为是在隐藏自己，在社交中的卑微，所以你才会不断的强化自己同一个行为，最终心理位置就处于下位者。

所以说高情商的策略，就是不彰显自己的行为。说简单一些，不要依靠对方来满足自己的情感需求，这是一种很low的心理。

比如聊天，我们就是在平等的沟通，我不会期望你关心我，更不会期待你的秒回。但如果你有主动分享自己的状态，那我也会及时接住。所以其中会有不同的高情商沟通方式，但核心都是为了保持心理位置的平衡。

当然你也会遇到对方没有任何回应，或者不是你想要的感受。所以还需要了解第三个策略。

## 策略二：合理的冷淡

这里就可以记住一个三次原则。还是拿聊天举例子，在没有得到回应，或者对方的回应不是很热情的时候，选择及时撤离。

当然我不是让你直接扭头就走，愤怒的告诉女生，我们以后再也不联系了。这就好像一个小孩子不开心，在闹脾气一样，这不是一个上位者的态度。

所以我们要用一种平静的行为告诉对方，我现在对你冷淡了。

以下是具体操作方法：

*   **延迟回复**：比如女生上午跟你回消息，你下午或者第二天再回复。
*   **减少主动**：你以前总是提供话题，现在开始你只回答不分享。
*   **简洁解释**：如果对方问你原因，你就直接说，最近有些忙，也不要多做解释。

此时对方肯定会知道，他要失去你了。这种平静的冷淡，你的心理位置就是高于对方的。因为只有上位者和平等关系，才容易让人失去，下位者是不容易失去的。

## 课程总结

本节课中我们一起学习了吸引力法则的核心应用。

上面分享的两个方式，一个是不邀功的行为，一个是平静的离场，都属于上位者和平等的心理位置。也只有这种位置，你才有可能打开慕强的开关。

它和物质没有太大关联，但却能让你在一段社交关系中，拿捏自己的位置，让自己变得越来越有吸引力。
