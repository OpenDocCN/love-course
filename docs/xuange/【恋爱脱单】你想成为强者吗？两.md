# 【恋爱脱单】你想成为强者吗？两

在本节课中，我们将学习如何在感情中建立“强者思维”。这种思维模式能帮助你摆脱患得患失的心态，掌握关系中的主动权，从而建立更健康、更平等的情感关系。我们将深入探讨强者思维与弱者思维的核心区别，并通过两个关键建议，帮助你将其应用到实际相处中。

## 强者思维与弱者思维的核心区别

上一段我们提到了课程的目标，本节中我们来看看两种思维模式的本质差异。

弱者思维的核心是**恐惧**。拥有这种思维的人，其关注点永远在对方身上：担心对方是否喜欢自己、是否会讨厌自己、是否会不理自己。这种心态最终会导致他们将对方置于关系的主导地位。

强者思维的核心是**反思和行动**。拥有这种思维的人，首先关注自身，不会轻易为了对方而降低自己的底线和原则。虽然听起来有些冷酷，但往往是这样的人才更具魅力和吸引力。

## 如何培养强者思维：两点核心建议

理解了两种思维的区别后，我们来看看如何具体培养强者思维。以下是两个至关重要的实践建议。

### 第一点：掌控感

掌控感是指无论外界如何变化，自身情绪和行为都不受其干扰的能力。拥有强者思维的关键，在于实现对**自我**的掌控。

在社交中，从你与任何人接触的第一秒开始，双方就进入了权力争夺期。有强者思维的人不会因外界变化而动摇。此时，对自身的掌控就变得至关重要。如果你无法控制自己，在相处中一再妥协，甚至不断降低底线，那么你就只能被对方掌控。

例如，你们认识不到一星期，对方暗示要礼物，或第一次吃饭坚持要带闺蜜，你不乐意他就指责你小气、不是真心。意志不坚定者很容易妥协，从而陷入恶性循环。

因此，拥有掌控感的关键在于**敢于拒绝**。面对上述情况，你可以明确表态：“我只给女朋友送礼物”，或直接拒绝带闺蜜吃饭的要求。请注意，这并非教你事事拒绝，而是拥有这种敢于维护原则的态度，你才能被平等对待。这种掌控感不仅对情感有益，对工作和学习同样大有帮助。

### 第二点：主导力

强者在一段关系中通常是主导者。但这里的主导，并非指强势地控制所有细节，而是**主导关系朝着更好的方向发展**。

相反，弱者往往喜欢控制一些与大方向无关紧要的琐事。我发现很多男生在这方面存在误区。

例如，一个男生第一次和女生见面吃饭，女生说想吃烤肉，男生却偏要说“我昨天刚吃过，不想吃”，尽管他自己也很久没吃了。我问他原因，他说这是为了“显示自己的态度”，不去顺从女生。这个理由令人无语。请记住，所谓的“有态度”，应体现在**原则问题上不妥协**，而非在诸如“吃什么”这种小事上较劲。只有弱者才会在意这些容易做到的小事，因为真正的大事需要更多能力和资源去支撑，他们往往做不到。

如果你还不明白，可以想想职场中的情况：有些领导为了体现权威，每天揪着考勤、甚至上厕所时间这类小事不放；而真正有能力的领导，思考的是如何合理安排工作、提高公司整体效率。

那么，如何拥有这种主导力呢？记住一个原则：**抓大放小**。

“抓大放小”是什么意思？如果你每天纠结于女生是否秒回消息、是否说早安晚安、是否主动问你在干嘛，那你就是在揪着小事不放。你需要关注的应该是“大”的、原则性的方面，例如：对方人品是否好、是否善良、生活是否独立。这些才是让一段关系变好，并能支撑你们走到最后的关键因素。

当你关注这些大方向时，再回头看那些秒回信息之类的事情，就会发现它们不过是无关紧要的小事。哪个女生没有点小个性呢？作为男生，你只需要主导关系的大方向就足够了。过分关注小事，反而会降低你自己的价值，这正是弱者思维的开端。

## 总结与回顾

本节课中我们一起学习了如何在感情中建立“强者思维”。

最后需要强调的是，以上两点并非独立存在。**主导力**和**掌控感**是相辅相成、互补的关系。如果你也想拥有强者思维，建议将本课内容反复理解。虽然这不像具体的案例技巧那么直接，但心态建设往往更为根本和重要。

记住，好的感情是双向奔赴，唯有真心才能获得真爱。掌握强者思维，是为了让你在保持真心的同时，也能保护好自己，赢得应有的尊重和主动权。
