## 理解人际距离与边界感

上一节我们介绍了课程主题，本节中我们来看看人际交往中的距离理论。美国科学家爱德华博士提出的人际距离理论，将人与人之间的关系分为四种距离：

*   **公共距离**：例如与陌生人打招呼。
*   **社交距离**：例如普通朋友和同事关系。
*   **个人距离**：例如与关系很好的朋友和家人。
*   **亲密距离**：例如产生了肢体接触的亲密关系。

你需要明白，你与对方的关系程度，决定了你应该与之保持什么样的距离。我常说的“边界感”，实质上就是懂得区别对待不同的交往对象。

明白了人际距离的基础概念后，下面我将分享给你三个需要牢记的核心原则。

## 必须牢记的三个核心原则

以下是三个能帮助你避免冲动性社交、提升吸引力的关键原则。

### 1. 过于热情是一种压力

你可能会觉得与人相处很麻烦：太疏远不行，太过热情也不好。因此，在两者之间找到一个合适的度，才是最好的相处方式。

分享一个学员的故事：一位男生与喜欢的女生在周三晚上第一次约会，直到晚上12点半才各自回家。女生对男生有好感，男生也感受到了这份心意。男生回家后不到20分钟就能入睡，但女生卸妆护肤后可能要到凌晨两点才能休息。男生未考虑到这一点，次日（周四）晚上又邀约女生见面，这让女生感到为难：拒绝怕对方多想，不拒绝自己又非常疲惫。这就是典型的冲动型社交，只会把喜欢的人越推越远。

**核心公式：关系进展速度 ≠ 关系质量**
因此，任何事情都不要推进得太快，来日方长才能细水长流。

### 2. 话说一半是保护自己

人性有一个弱点：见不得别人比自己好，除非他的好与我有关。因此，无论在职场还是情感中，说话说一半就好，这样你反而会更有神秘感。越是毫无保留的“傻白甜”，越容易吃亏上当。

举例来说，双方认识不久，你就主动透露自己情感经历不丰富、接触异性不多，或者告知对方自己的具体工作和收入。你自认为这是真诚，但我告诉你，这绝非好事。

### 3. 指手画脚容易累积思想障碍

每个人活了二十多年，都有自己的思维方式和立场。尤其是在你们刚刚认识、还不清楚对方思想的时候，不要轻易评头论足。

再分享一个故事：男生和女生第一次约会，路过一家服装店时，女生觉得这家店的衣服太花哨，直接说了出来，甚至带有批评口吻。但她没想到，男生其实很喜欢这种风格。于是男生会思考：“我以后见这个女生，还要不要穿这种风格的衣服？”这在无形中就形成了一个思想障碍。这类障碍累积多了，好感就会下降。

所以，请克制自己想要对别人指手画脚的欲望。对方不会因为你的一句话就改变，你们的关系也不会因为一句话就加深。

记住了以上三个原则后，我们应该如何去具体改变呢？接下来提供两个实用的思路。

## 两个实用的改善思路

以下是两个可以帮助你系统性避免冲动性社交、提升交往掌控感的方法。

### 思路一：内心建立评级系统

每接触一个人，就为其建立一个新的评级系统。对方的行为达到某个评级后，你再去做相应的下一步，为进入下一个评级而付出。如果对方没有达到你的评级标准，那你就不进不退，甚至可以考虑减分。

**代码描述评级逻辑：**
```python
if 对方行为符合当前关系等级:
    推进到下一等级，并付出相应行动
else:
    保持现状或降低等级
```
灵活运用这个系统，就能在每次认识新朋友时，有意识地为你们的感情“控分”，冲动性社交自然就避免了。

### 思路二：养成被动思维

这里的“被动”不是指行为上的被动，而是在与人交往中，等待对方先向你表达出分享自己的欲望之后，你再分享自己。

这样做的好处是：你不会过度琢磨对方每句话的含义，也不会去干预对方的想法。当你采取被动姿态后，你会本能地把更多关注点放在观察对方的状态上，就像狩猎者静静匍匐在草丛中，根据外界变化来选择自己的下一步行动。
