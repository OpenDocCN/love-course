# 恋爱脱单课 01：应对女性冷淡的三大策略 🎯

在本节课中，我们将要学习当女性突然对你态度冷淡时，如何通过三个核心步骤扭转局面，重新建立吸引力，甚至引导对方主动联系你。

## 第一步：停止纠缠，保持距离 🤚

上一节我们介绍了课程目标，本节中我们来看看第一个关键策略。当女性对你冷淡或不理睬时，切忌死缠烂打。不要进行电话轰炸，也不要信息追问个不停。

1.  对你失去了兴趣，你对她不再具备吸引力。
2.  有其他男人捷足先登。

无论是哪一种原因，你的纠缠都没有任何作用。这种行为只会让她看不起你。这就像手捧着沙，你握得越紧，沙流失得越快。

理解了停止纠缠的重要性后，我们进入第二步。你需要转移你的注意力目标，并扩大你的社交圈。

**心态稳定性 ∝ 社交选择权**
