上一节我们介绍了课程概述，本节中我们来看看第一种需要警惕的类型。

## 1. “三不”女生：不主动、不拒绝、不负责

为什么把这种类型放在第一条？因为根据许多案例观察，大部分问题并非源于男生自身。这类女生在互动中会回应聊天、接受陪伴，甚至允许牵手拥抱等亲密举动。然而，一旦面临正式表白，她们便会立刻撇清关系，表明“我们毫无关系”。无需怀疑，这种行为就是在吊着你。

她们通常会接受所有追求者的好意，并从中进行筛选，直到找到心仪的对象，最后无情地将你淘汰。如果你认为自己可能是那个“天选之子”并试图拯救她，那么成功的概率微乎其微。

## 2. 同性缘差但异性缘好

以下是这类女生的一个显著特征：你会发现她们身边总是围绕着一群男生，以各种身份满足其不同方面的需求，例如所谓的“男闺蜜”、“饭友”、“酒友”。她们能与每个人都保持着既亲近又暧昧的关系，并熟练地掌控进度，游刃有余。

但恰恰是这种行为，会导致她们失去许多同性朋友。无论是出于嫉妒，还是害怕其真实面目被揭穿，女生通常能一眼看出群体中谁在扮演“白莲花的角色。因此，同性朋友少是她们的典型特征。然而在面对男生时，她们又会伪装起来，欺骗你的感情。

## 3. 带有目的性的利己主义者

举几个常见的例子，你可以看看自己是否遇到过：
*   你不找她，她绝不找你。一旦出现，必定是有事需要你帮忙。
*   在节假日等特定时间点，突然对你表现出热情。

这类女生对你并不带有任何真实感情，即使你付出再多努力也毫无意义。从本质上讲，她们属于精致的利己主义者。如果你无法满足其需求，被删除好友是最基本的操作。她们还常常用各种流行的网络话术来绑架你，例如：“判断一个男人喜不喜欢你，就得看他愿不愿意花钱。” 她们永远呈现一种索取的姿态，一旦你开始付出，只会越陷越深。如果遇到这种情况，请赶紧远离。

## 4. 喜欢无病呻吟的“公主病”

“公主病”在这类女生身上表现得像一种不治之症。你能感受到满满的矫情和做作。当她们发现这种方式能获得更多关注时，便会变本加厉：无中生有自己的心情，添油加醋自己的经历，把自己塑造得很可怜、很孤独。但这其实都是为了引起其他男生的关注和关心。

这种行为无疑也是一种“广撒网”。如果你遇到这样的女生，无论对方其他条件多好，都不要被其牵动。因为她们仿佛失去了共情能力，只会以自己的情绪为中心，不断影响你的心情。这也是我们常说的情感操控（PUA）。

## 5. 与前任纠缠不清

这一点无需多言，大家应该都能理解。其实，女生的一些小毛病、小问题通常都可以接受。但唯独最不能接受的，就是和前任藕断丝连。因为从某个角度来看，这种行为就是“吃着碗里的，看着锅里的”。

可能是由于自己作闹或其他原因导致不清不楚地分手，关系并未完全结束。分开后才又想起前任的各种好，一旦和你吵架，就会去前任那里寻求温暖。最好一辈子都不要碰到这类女生，因为这不光是身体上的折磨，精神上的折磨更是双倍打击。经历一次，便足以悔恨终生。
