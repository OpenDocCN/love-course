# 恋爱脱单课程 01：如何高情商应对敷衍不主动的女生 💬

在本节课中，我们将学习在聊天中遇到女生回复敷衍、不主动时，如何通过高情商的方式表明态度、扭转局面，从而建立平等、互相尊重的沟通关系。

相信大家在聊天中都遇到过“嗯嗯”、“对呀”、“是的”、“没有”等敷衍的回复。说实话，当时的你是否绞尽脑汁地换话题，打了字又悄悄删掉，或者直接不知道说什么。

这种情况并非你的问题，而是对方不够礼貌和尊重。因为现在“舔狗”太多，无论对方如何敷衍，总有一大群男生会拉着她继续聊天。从此刻开始，我们要让你变得不一样。

所以，请先暂停你此前各种花里胡哨的想法。我们要回归根本，表明自己的态度，但必须是高情商的。别着急，本期内容将结合一个真实案例来告诉你具体怎么做。

## 案例背景与分析

我的一个朋友在网上认识了一个女生。开头是很正常的打招呼。其实开头普通打招呼就足够，不必模仿网上花里胡哨的方式，那会显得你缺乏价值。

之后，我的朋友主动找了一个话题：“你是空乘吗？”（因为此前看过对方个人页面的照片）。这个开场没有问题。

过了一会儿，对方回复：“对呀。”这就是一开始提到的敷衍回复。但在此时还算正常，只能说对方不积极，毕竟有人一开始聊天确实比较被动，男生再主动些就好。

我的朋友选择继续深入话题：“是不是最近都没有国际航班了？”这是一个非常好的**封闭式提问**。在社交开始阶段，简单的封闭式提问更容易调动对方积极性，且话题可拓展性强。

到这里，对方本可以有很多话可说，例如“是啊，现在都没了”或“不是，我现在……”。但对方又只回复了两个字：“是的。”

我朋友接着说：“那每天也挺辛苦的，回家就不想动了。”他选择了继续话题，并尝试与对方共情。结果对方再次回复两个字：“是啊。”

就在这时候，我朋友找到了我。我认为他的聊天没有任何问题：聊天时间在晚上、从照片找话题、用封闭式提问降低回复难度、尝试共情分享情绪。这些细节反而显得他情商很高，且没有因对方敷衍而气急败坏或影响心态。

那为什么会这样？我的初步判断是：女生可能习惯了这种敷衍，因为身边肯定有很多人拉着她聊天、嘘寒问暖。

## 核心应对策略：高情商表明态度

兄弟们，重点来了。此刻，我让我的朋友停止一切聊天话题，并回复：
**“你是手机键盘坏了吗，只能回复两个字？哈哈。”**

这句话用开玩笑的语气表达了自己的不满情绪。切记不要直接说出来，**笑里藏刀才是高情商的表現**。

为什么要这样说？因为在现实中，很多男生不敢表明自己的态度，即使对方确实有问题，也会选择忽视、原谅，甚至继续讨好。既然都这样了，为什么还要忍气吞声？直接表明自己不滿的态度，可以立刻让你与“讨好型”男生区分开来。

## 策略效果与后续发展

后来，对方的回复态度直接大转弯：“哈哈没有啊，最近太累了。”这里的“哈哈”并非真的在笑，而是对方在掩饰尴尬。她明白我这个朋友和其他男生不一样，所以才会在后面解释自己。

我让朋友回复：“累了就早点休息吧。”后来对方说：“我好多行李没收拾。”此刻，我告诉朋友：**无论对方再说什么，晚上都不要再回复了**。并且，第二天对方大概率会再来找你。

果不其然，第二天中午，女生主动找我朋友聊天：“怎么还没有起床啊？”事实证明，对方也是会正常聊天的。那为什么一开始要敷衍呢？屏幕前的你也都懂。

后面的故事就不多说了，双方一起吃饭、看电影。最关键的转折点就在于：**我朋友把自己和“讨好男”彻彻底底地区分开了**。

## 课程总结与核心要点

本节课中，我们一起学习了如何应对聊天中敷衍不主动的女生。

好的感情是双向奔赴。男生也要勇于表达自己的态度。聊天不求秒回，但起码要互相尊重。实在不行，那就算了，起码没有丢失自我。

最后留一个思考题：你知道为什么当晚不回复，第二天对方还会主动来找我朋友吗？

希望大家能勇敢表达自己的态度。多一个敢于表达态度的人，“嗯嗯怪”就会少一个。

**课程核心公式：**
**高情商应对 = 停止讨好 + 用玩笑语气表达不满（“笑里藏刀”）+ 适时切断对话以制造心理落差**
