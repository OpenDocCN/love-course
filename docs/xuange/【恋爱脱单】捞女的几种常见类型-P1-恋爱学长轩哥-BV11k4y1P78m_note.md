## 第一种类型：直接索取型 💰

这是最常见且最低级的一种类型。这类女性的特点是交往目的明确，即直接获取金钱或物质利益。

以下是其常见行为模式：

*   **添加好友需付费**：例如，某些主播会要求刷礼物或“飞机”才同意添加好友。
*   **日常开销索取**：出门、约会等场合会直接向你要钱。
*   **巧立名目要红包**：例如，以生日派对等各种理由，要求你发红包或转账。
*   **索取小额利益**：甚至可能让你帮忙点外卖。

## 第二种类型：情感拉扯型 🎭

上一节我们介绍了直接索取型，本节中我们来看看第二种更高级的类型。这类女性会运用情感技巧，制造暧昧和不确定性，让你在患得患失中主动付出。

以下是其典型策略：

*   **忽冷忽热**：她会给你一种好像喜欢你，但又不够喜欢的感觉，通过态度反复来操控你的情绪。
*   **给予微小甜头**：例如在相处时故意靠近你，制造对你有好感的假象。
*   **诱导表白后拒绝**：当你按捺不住表白时，她会拒绝，但会接受你送的礼物或给予的物质好处。

其核心模式可以用一个简单的循环来描述：
```
对你热情 -> 你升温并付出 -> 她开始冷淡 -> 你困惑并试图挽回（付出更多）-> 她再次给予微弱希望 -> 循环继续
```
