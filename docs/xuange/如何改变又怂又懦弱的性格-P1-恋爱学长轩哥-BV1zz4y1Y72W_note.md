# 课程：如何改变又怂又懦弱的性格 - P1 🧠

在本节课中，我们将要学习如何识别并改变“又怂又懦弱”的性格特质。这种性格常表现为讨好型自卑人格，在情感和工作中容易感到胆怯。我们将通过四个核心要点，帮助你建立自信，学会表达自我，不再轻易受他人影响。

## 概述：性格成因分析

上一节我们介绍了课程目标，本节中我们来看看这种心态的根源。造成这种心态的原因，大多数与从小的家庭环境有关。

例如从小家教严格，或经常受到打压。遇到问题心软，会慢慢地养成对事情缺乏信心、过分在意他人想法的讨好型自卑人格。

## 改变方法：四个核心要点

以下是改变这种性格的四个具体方法。

### 第一点：明白“不会有人在乎你”

首先，你需要明白一个道理：在日常社交中，真的不会有人在乎你想什么、说什么、做什么。

只要你没有切身地影响到其他人的利益，你所有好的表现，别人只会说你情商高。差的表现，也只会被别人当一个笑话，一笑了之。而唯独在乎是否出丑、会给别人留下什么印象的，只有你自己。

所以就算你说错了什么话，过不了三天别人都会忘掉。哪怕你当场“社死”，别人也只会看到你身上的价值之后，再做考量。

**核心公式：他人关注度 ≈ 你对他人利益的直接影响**

举一个例子：当你参加一个聚会，遇到一个很好看的女生。你一开始不敢要联系方式，但喝多后在朋友怂恿下上前，结果女生给了你联系方式。你觉得是酒精带给你的力量，实际上是你学会了放开自己。纠缠你的，永远是你自己。

有句话说得好：只要你不尴尬，尴尬的就是别人。

### 第二点：多接触“恶心的人”以警醒自己

为了让道理更深刻，必须清楚地告诉大家：讨好型人格的人，应该多碰到一些“恶心的人”来警醒自己，给自己上一堂生动的教育课。

为什么这么说呢？因为你只有多碰到这些不懂感恩、不断压榨和过度索取的人之后，才会明白，你的习惯性讨好真的一文不值，而且容易被烂人利用。

这就像一个屠龙少年变成恶龙的故事。只有付出沉重代价，才会有改变自己的想法。当你把别人的套路都摸清后，就没什么可怕的了。

讨好型人格不在于你在哪个圈子，也不是互相理解就能解决。人与人之间的平等，需要相互付出来平衡。如果付出的价值不平等，你再怎么讨好或换人，都是没有用的。

### 第三点：别把自尊看得太重

有一部分自卑的人，可能因日常受忽视太多，导致把自尊看得太重，太把自己当回事。然后做什么事都小心翼翼，话也不敢说，事也不敢做。

**代码描述的心态陷阱：**
```
if (对方没有回应) {
    判定为“对方不喜欢我”;
}
```

例如，一个学员在聚会上和一个女生搭话，当时女生正和别人说话，忽视了他，结果他就认为对方对他没有好感。这种想法非常极端。

自卑其实是另一种形式的自负。改变性格或习惯意义不大，因为那只是在否定过去的自己。真正的改变始于认清并接纳自己。

看看身边自信的人，他们有的方面可能比你更邋遢、更阴暗，只是他们认清了自己。有人喜欢你就必定有人讨厌你，想让所有人都喜欢的结果，就是自己难受。

所以想要自信，更应做到的是自爱，多照顾自己的感受，而不是揣测别人的感受。

### 第四点：认识到“怂和懦弱不是你的错”

最后，明白这一点就够了：怂和懦弱从来不是天生的，而是成长环境、家庭和社会多方面压力所造成的。

想通这一点就能知道，这并不是你自己的错误。

例如，工作中领导打压你，不是你的原因，而是他的资历和职场环境带来的底气。情感中，女生对你冷漠或打压，是因为她身边有不少追求者，这才是她有底气的根源。

所以想改变，就要尽可能去提升自己，找到最适合自己的路，一步一步往上爬，才能改变目前所处的局面。

短时间的怂和懦弱不可怕，可怕的是你待在这样的环境中任人宰割，还觉得自己是一个忍辱负重的人。

## 总结与延伸应用

本节课中我们一起学习了改变怂懦性格的四个要点：明白他人并不那么关注你、通过接触负面人物警醒自己、不过度看重自尊、以及认识到问题的根源不在自身。

你可能会觉得这些内容和情感太相关，但这些内容才是很多情感问题的根源——即心态问题。无论你追女生时卑微讨好，还是聊天总是很平淡，基本上都与自己的心态息息相关。

当然，这些道理放到工作和生活中同样如此。

如果你觉得本课对你有帮助，可以分享给有需要的朋友。改变始于认知，终于行动。祝你早日获得属于自己的力量。
