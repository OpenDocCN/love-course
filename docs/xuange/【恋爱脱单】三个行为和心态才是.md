# 【恋爱脱单】三个行为和心态才是

在本节课中，我们将通过一个真实案例，剖析在追求过程中容易被贴上“猥琐”标签的三种核心心态与行为。理解并避免这些误区，是建立健康恋爱关系的第一步。

不久前，一位粉丝被人评价为“猥琐”。“猥琐”是一个彻底的贬义词，一旦与这个词挂钩，通常不会有好事发生。

本视频结合一个案例，详细说明是哪三件事和心态，会让对方感觉你很猥琐。这位粉丝说：“沛沛你好，我有一位情感经历很丰富的好兄弟。前段时间他第一次和女生见面，过了几天就被传成了猥琐男。但我这个朋友社交能力强，情商也很高。他不好意思，就拜托我来问问你。”

无论这位兄弟说的是真朋友还是假朋友，下文提及的都将指代案例中的男生。他的案例就因为做了以下三件事，直接被归类为猥琐男，非常有代表性。

看完这个案例你就明白了。

## 1. 得失心太重

我当时反问：“为什么你会觉得这个朋友的社交能力和情商都非常高？”他给我的大概回复是：这个朋友在街上遇到喜欢的女生从来不会犹豫，大胆上去认识，经历很丰富。交朋友非常直接，从来没有一个人说他猥琐。行就行，不行就拉倒，所以他不觉得自己有问题。

实际上，不论男女，有这种想法都很正常。毕竟每个人的时间和精力都有限。能在一起当然最好，如果不行，直接说明不要吊着对方。

这种心态放在男生身上会更加明显。我接触过这么多案例，大部分男生觉得，只要对方好看、身材好，其他什么都不重要，他都愿意去和女生接触，并且希望迅速得到结果。

但是这类男生忽略了一件事：**恋爱不是买卖**。不是你付出了，就可以当场得到回报。尤其是女生，更加注重过程和感觉的推进。这一点进度的快慢，取决于女生的性格特点和过往经历，这里就不细说了。

像案例中的男生，也许就遇到了和以往社交经历不同的情况。说白了，就是对方的反应和以往接触的人不同，于是心态就崩了。

## 2. 目的性太强

我们接着来看一小段聊天记录。
女生说：“我感觉我们不是很适合，可能当好兄弟会更好。”
男生反问道：“那为什么之前，你还单独和我吃饭看电影，还和我聊天聊到很晚？”

很多人都会有这种心急的心态，觉得女生和自己单独出门、聊天互道晚安，那就是喜欢自己。甚至有人在第二次见面时就动手动脚。

像案例中，男生直接把自己最真实的想法表达了出来。女生过了一会儿说：“难道我和你吃饭看电影、聊天到深夜，就代表我喜欢你？”
男生说：“我对你还是很有感觉的，我们现在不就在接触吗？我们先试一试，慢慢磨合。”

这句话非常经典。男生话里话外的意思就是在说：**只要我们能在一起，其他的都不重要**。但在女生眼里，她并不觉得你是真喜欢她，反而会觉得你很猥琐。因为她觉得男生没用“上半身”思考问题，这在社交中是非常低价值的。“你和动物有什么区别？”

兄弟们要知道，女生愿意和你单独见面、愿意和你聊天，绝对不等于男女之间的喜欢。只能说男生当前有一些点吸引到了她，她愿意和你继续接触。这是一个好事。

你有想亲亲抱抱的想法也很正常，毕竟我们都是人类。但千万不要有案例中男生的心态：“你都出来了装什么装”，甚至不顾对方感受强行升级关系。那女生肯定会觉得你很猥琐，因为你忘记了社交中最重要的一点：**彼此尊重**。

所以在追求阶段，男生需要提供的是舒适感。我在以前的视频里也说过：**吸引力**是让对方对你有好感，**舒适感**是让对方觉得你人很靠谱、有安全感。只有这两个点达到平衡后，双方才会顺理成章地牵手。

## 3. 买家心态

这种心态最直接的想法，就是把物质和感情划等号。你肯定不止一次在评论区看到：“追女生不就是钱吗？”这会造成两种结果：第一种是通过物质得到一个人；第二种是强行付出，对方不接受就开始诋毁对方。

像案例中这位兄弟就觉得，自己遇到了渣女或拜金女。他觉得女生不想和他在一起，为什么叫他吃饭还要来？送香水为什么还要收下？最后因为没有得到，开始气急败坏地说这个女生“又当又立”，开始诋毁女生。

这种行为其实就是典型的买家心态。他想通过自己的付出来得到相应的结果，如果没有得到，就会觉得对方有问题。

但正确的逻辑应该是：如果男生一意孤行地付出，得不到结果，就不要埋怨别人。所以我常说，付出是互相的，而且要在自己的承受范围内。如果会让自己后悔，那就减少付出。既然做了，就不要气急败坏地后悔，这才是男人。

除非对方有意引导你付出，或者对你的所有好都不拒绝，那就另当别论了。这位兄弟的案例，是付出没有得到结果，然后开始诋毁。

我再分享一个相反的案例。之前有一位学员谈了一段恋爱，每次都是他跪求和好。后来在疫情期间，这位兄弟靠物流代理赚了不少钱。从那以后，女朋友对他的态度发生了翻天覆地的变化，给他买东西不说，还总是去看望他的父母。这反而让他有了分手的念头，觉得是因为自己有钱了，这个女生对他的态度才变好的。如果自己现在还是和以前一样，可能他的女朋友早就跟别人跑了。闹到最后，女生提出要一笔分手费。

这个案例是通过物质得到了想要的爱情，那为什么也没有好结果呢？归根结底，就是买家心态，把情感给物化了。没人喜欢，就觉得对方不识抬举；自己吃不到葡萄，就说葡萄酸。等真有人喜欢自己了，又会觉得对方是不是图自己什么，自己会不会吃亏。

这种把感情物化的买家心态也好，又或者是得失心和目的心，本质上都是不够坦荡和真诚。当一个男人自己都不能对感情有一个清晰正确的认知时，女生又怎么可能被你吸引？那当然不可能获得长期的真爱了。

像案例中的这位兄弟，双方最终以拉黑删除画上了句号。

也许说到“猥琐”这个词，所有人想到的都是电视剧中鬼鬼祟祟、看着就不像好人的人。但实际上，在你和女生的社交中，一旦出现我上面说的三种行为和心态，就会被女生贴上“猥琐”的标签。拉黑只是时间问题，而且你会离真爱越来越远。希望各位不要和案例一模一样。

最后，我分享的任何内容，只是为了让双方的沟通更加顺畅。**真心才能获得真爱**。我是柯可，还有什么问题可以一起聊聊。
