## 🧐 特征一：社交与兴趣模式

上一节我们介绍了课程概述，本节中我们来看看第一个特征，主要涉及个人兴趣与社交活动。

长期单身的男生在兴趣爱好和社交频率上往往表现出特定模式。

以下是具体的表现：

*   没什么爱好，平时很少出门。

## 🤝 特征二：人际互动方式

了解了社交兴趣特征后，我们进一步观察他们在人际交往中的行为模式。

这一特征主要体现在与异性或他人互动时的主动性与边界感。

以下是具体的表现：

*   心地善良，从来不会主动追求，也不擅长拒绝。

## 🧘 特征三：内在欲望与心态

除了外在行为，他们的内在心态和欲望也会发生变化。

这种心态趋向于平和与低欲望，其核心公式可以概括为：**欲望水平 ≈ 0**。

以下是具体的表现：

*   想要的东西越来越少，越来越佛系。

## 😔 特征四：面对心仪对象时的心理

当特定的心动对象出现时，他们的心理活动会呈现出一种自我限制的模式。

这种心理的核心代码逻辑类似于：
```python
if 遇到喜欢的女生:
    自我评估 = “我肯定追不到”
    采取行动 = 说服自己放弃
```

以下是具体的表现：

*   看到自己喜欢的女生，总觉得自己追不到，尝试说服自己放弃。

## 😶 特征五：情感投入状态

最后，我们来观察他们在广义人际关系中的情感投入程度。

他们对人的普遍态度友好，但对具体事物的情感卷入度在降低。

以下是具体的表现：

*   对人都很好，在乎的事越来越少。
