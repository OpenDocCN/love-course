# 课程名称：Python基础入门 - 第01课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如数字、文字等）。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是：
`变量名 = 值`

例如：
```python
name = "轩哥"
age = 25
```

这里，我们创建了两个“盒子”。一个名叫`name`的盒子里存放了文字“轩哥”，另一个名叫`age`的盒子里存放了数字`25`。

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道盒子里装的是哪种类型的东西。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其示例：

1.  **整数**
    整数是不带小数点的数字，可以是正数、负数或零。
    ```python
    my_score = 100
    temperature = -5
    ```

2.  **浮点数**
    浮点数是带有小数点的数字，用于表示更精确的数值。
    ```python
    pi = 3.14159
    price = 19.99
    ```

3.  **字符串**
    字符串是由一系列字符组成的文本，需要用单引号(`'`)或双引号(`"`)括起来。
    ```python
    greeting = "你好，世界！"
    author = '轩哥'
    ```

4.  **布尔值**
    布尔值只有两种可能：`True`（真）或`False`（假）。它常用于条件判断。
    ```python
    is_learning = True
    is_completed = False
    ```

## 如何查看与使用变量？ 👀

创建了变量之后，我们如何查看和使用它们呢？最直接的方法是使用`print()`函数。

例如，运行以下代码：
```python
name = "轩哥"
print(name)
```
屏幕上就会显示出：
```
轩哥
```

你也可以在`print`中组合文字和变量：
```python
name = "轩哥"
age = 25
print(name, "今年", age, "岁。")
```
输出结果为：
```
轩哥 今年 25 岁。
```

## 变量的命名规则 📝

给变量起名字不是随意的，需要遵循一些简单的规则。

以下是命名变量时需要遵守的主要规则：

*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名是区分大小写的（`age`和`Age`是两个不同的变量）。
*   不能使用Python的保留字（如`print`, `if`, `for`等）作为变量名。

良好的命名习惯能让代码更易读，例如使用`user_name`而不是`a`。
