# 恋爱课程 P1：如何分辨女生是真的保守还是假的保守 🔍

在本节课中，我们将学习如何通过女生的具体行为，分辨她是真的保守、对你没有兴趣，还是在“假保守”、实则对你有好感。掌握这些关键表现，能帮助你避免错过推进关系的良机。

## 核心概念：真拒绝 vs 假保守

在互动中，女生的反应可归为两类：**真拒绝**（无兴趣）与**假保守**（有兴趣但矜持）。判断的关键在于**观察行动而非只听言语**。

---

## 三个关键表现

以下是三个帮助你判断的核心行为表现。

### 1. 对邀约的回应

邀约时，如果女生没有秒回，也未直接拒绝，最终在你的坚持下同意出门，这属于**假保守**。公式可概括为：
```
假保守 = 不秒回 + 不直接拒绝 + 最终同意
```
反之，如果她直接说“没空”或找理由推脱，则是**真拒绝**。

上一节我们介绍了核心概念，本节中我们来看看第一个具体表现——邀约回应。

### 2. 对肢体接触的反应

当你有轻微肢体接触（如碰触手臂）时，如果女生表现害羞但并未完全抗拒动作，这属于**假保守**。代码逻辑可表示为：
```python
if 反应 == "害羞" and 动作 != "完全抗拒":
    类型 = "假保守"
```
反之，若她立刻避开并在后续互动中下意识保持距离，则是**真拒绝**。

### 3. 聊天中的互动模式

女生若自称“慢热”，但会主动提供话题，并在未秒回时做出解释，这属于**假保守**。其行为模式为：
```
假保守 = 自称慢热 + 主动提供话题 + 为延迟回复解释
```
反之，如果她总是不秒回，且回复冷淡、不拓展话题，则是**真拒绝**。

本节课中，我们一起学习了通过**邀约回应**、**肢体接触反应**和**聊天模式**这三个方面，分辨女生是真拒绝还是假保守。

如果女生符合“假保守”的表现，说明她对你有好感，只是出于矜持或试探而稍作保留。此时，男生应**更关注她的行动而非言语**，并可以适时、尊重地推进关系。
