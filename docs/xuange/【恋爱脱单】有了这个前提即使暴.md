## 理解需求感与博弈

上一节我们介绍了课程的主题，本节中我们来看看什么是需求感以及它为何会产生问题。

暴露需求感也可以变成吸引。这种说法并不违背常理，因为人不是机器，遇到喜欢的人不可能完全隐藏好感。但前提是需要明白什么是需求感。

许多男生都遇到过这种情况：越对一个人上头，对方反而离你越远。当你不对一个女生上心时，她反而会主动接近你。

这种情况的出现，是因为凡事只要有人存在，就会有博弈。

**举例说明**：
> 你走进一家车行，销售发现你一进门就笔直走向一辆车，仔细查看，对它的性能比销售还了解。等到谈价格时，你就很难把价格谈下来。因为销售知道你非常喜欢这辆车。如果这家店处于垄断地位，你就没有任何挣扎的余地，因为销售知道已经吃定你了。

这揭示了人性的一个弱点：每个人都是逐利的。

## 情感市场中的博弈

理解了基本的人性博弈后，我们将其应用到情感市场中。

在情感市场里，如果你对一个女生每天嘘寒问暖、不断发消息、发小作文表达爱慕，她的潜意识就会告诉她：“你已经这么喜欢我了，我不能让你这么容易得到我。我要设计一些考验，也许我会收获更多。”她身边的闺蜜也可能给出类似建议。

那么，为什么你会被拿捏呢？

**核心公式**：
```
被拿捏 ≠ 暴露需求感
被拿捏 = 社交地位处于下风
```

就好比你和销售聊天。初级销售会不断让利，让你觉得自己捡了便宜从而购买。高级销售则会展示产品价值，让你认可产品从而购买。前者态度献媚，后者平等互利，结果完全不同。

当你和销售聊天时，你已经知道他的需求是让你掏钱。所以，被拿捏的核心并不在于暴露需求感，而在于你的社交地位处于了下风，是你自己主动降低了姿态。

## 核心技巧：保持社交地位平衡

上一节我们分析了问题的根源，本节中我们来看看解决之道——保持社交地位的平衡。

放到情感中，你要记住：时刻保持社交地位的平衡。这样，你的需求感也能产生吸引力。因为当你主动加女生微信、主动开启话题时，女生也知道你的需求是想了解她。

以下是实现平衡的两个关键点：

### 1. 保持行为稳定

比如你想和女生聊天，必然需要主动找话题。此时你的需求就是和对方聊天。但是，当她没有主动分享或态度冷漠时，你不应通过“献殷勤”来推进关系。

**行为模式代码**：
```python
if 对方反应冷淡:
    保持稳定行为 # 例如：正常结束话题，不追问，不讨好
else:
    继续自然互动
```

你的行为是稳定的，没有任何献媚和迎合的态度，这就不会降低自己的社交地位。

### 2. 合理暴露需求感

记住，在三个不同阶段，你可以适度深入行为，暴露不同层次的需求感。

以下是分阶段暴露需求感的指南：

*   **初识阶段**：暴露“认同”的需求感。
    *   *示例*：你们聊到音乐，她喜欢听说唱。你可以说：“我最近也常听说唱，很喜欢某位歌手。”这会让对方感到亲切。

*   **朋友阶段**：暴露“欣赏”的需求感。
    *   *示例*：你可以对她说：“你的性格很好，和你相处很自然、很开心。”

*   **暧昧阶段**：暴露“情感”的需求感。
    *   *示例*：你可以说：“我每天工作很累，但和你见面时，整个人都感觉轻松了很多。”

了解了以上两点，你们的社交地位就会处于平衡状态。此时，你暴露的需求感反而会让关系越来越好。
