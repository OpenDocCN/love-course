# 4.3.3聊天技巧：通过框架传递社交地位高低的信号

在本节课中，我们将要学习一个核心的聊天概念——“框架”。我们将了解什么是框架，它如何在不经意间传递出社交地位高低的信号，并学习如何在两性互动中建立和维持一个强有力的框架，从而引导关系向有利的方向发展。

## 什么是框架？

上一节我们介绍了聊天中的基本概念，本节中我们来看看“框架”这个核心工具。

框架这个词来源于神经语言程序学。它的基本意思是指一个人言行举止背后的潜台词，是言语之外的含义。它是一种不自觉的暗示。也就是说，虽然很多事情你没有明确说出来，也没有直接表达态度，但你的言行举止会“出卖”你。

例如，虽然你没有向一个女人表白，但你经常主动找她聊天，在她面前表现得唯唯诺诺。这种态度就已经传递了低价值的信号，女人一样能感知到你对她的企图。在销售中，销售员也经常使用框架。比如他打电话时可能会说：“王哥，你看咱们这个合同是周三签还是周末签，您比较方便？” 这句话的框架就是：**“你已经打算签合同了。”** 你可能并没有明确答应，但他会用这样的话来“框住”你。

如果你的框架足够强，那么不论你说什么、做什么，在对方看来都是对的，别人都会倾向于按照你的规则来行事。

## 框架在两性关系中的力量

理解了框架的基本定义后，我们来看看它在两性互动中的具体应用。

特别是在两性关系中，如果你设立了正确的框架，那么女人就会倾向于服从你的引导。你经常会听到女孩说：“我要一个对我好的男人。” 她设立的框架就是：**“想追我的男人，不管条件如何，都得学会对我好。”**

在你和女孩的互动中，女孩会经常用这种框架来暗示你。如果你真的照做了，那么你就落入了她的框架，展现出了低价值。我们需要学会识别并跳出这种框架。

以下是识别和应对框架的一个例子：

有一次我和一个女孩在网上聊天。我说：“你看着挺小的，好像还是学生。” 她说：“你猜猜看。” 我回复：“那猜对了有什么奖励呢？” 她说：“奖励你可以给我打个电话。”

请注意“奖励你可以给我打个电话”这句话。女孩这句话的框架和潜台词是：**“我觉得你喜欢我，你想了解我、靠近我。既然你想追我，那么你就要好好表现。如果你的表现让我开心（猜对了），我可以‘奖励’你（允许你给我打电话）。”** 这时，她展现的姿态比我高，框架是“我在追她”。

我们的回复肯定不能落入这种框架。所以我当时调侃地回复她：“我觉得你还挺适合创业的嘛，毕竟你已经到了盲目自信的阶段了。” 这句话的意思是：**“我什么时候说过我喜欢你了？你想多了，你太自信了（而且是盲目自信）。”**

## 如何建立高价值框架？

既然跳出对方的框架如此重要，那么我们应该如何主动建立自己的高价值框架呢？

撩妹高手经常对女人设定的框架是：**“我是万人迷，我有很多选择。虽然觉得你不错，但我不会讨好你，更不会听你安排。应该是你喜欢我，我可能会给你机会，但你需要符合我的标准。”**

这种态度听起来可能有点“欠揍”，但确实非常实用。如果你一直用这种方式和女孩交流，女孩就会逐渐接受你的框架，认为这就是实际情况。你的气场和整体状态就能影响到她，她会觉得应该努力才能获得你的认可，甚至会主动来“追求”你。

## 一致性测试与框架游戏

在展现框架时，你可能会遇到挑战。女孩们不会轻易接受你的设定，她们会进行测试。

虽然你在展现自己框架时，表现可能很像真的，但大部分女孩会通过各种方式来挑战你的框架，测试你是否前后一致。这就是“一致性测试”。如果女孩通过测试，感受到了你的不自信或前后矛盾（发现你是伪装出来的），那么她可能就会拒绝你。所以，你需要准备好应对女人的一致性测试。

同时，你还要注意，女孩经常会和男人玩一些“框架游戏”，看看你会不会落入她们的“圈套”。

以下是女孩常用的几种框架游戏圈套：

1.  **表达好感圈套**：她先对你表达好感，让你自以为有机会，看看你会不会反过来追求她。应对方法：不要轻易向女孩表达好感，要通过“筛选”来回应。
2.  **夸奖炫耀圈套**：她故意夸你很厉害，然后看看你会不会向她炫耀、证明自己。如果你这么做了，说明你很在意她的看法，落入了圈套。
3.  **打击测试圈套**：她可能故意打击你，说你这里不好、那里不好，然后看看你会不会解释、担心，在意她的看法。
4.  **无理要求圈套**：她会要求你做一些（你）做不到的事情，看看你会不会去解释或道歉，这同样是引导你向她证明自己。

如果你帮女孩做事、去追她、向她炫耀、向她道歉或解释自己，那么你就落入了这些圈套。这说明你在乎她，你想让她高看你一眼。

## 如何应对圈套并反制？

如果你跳进她的圈套，基本上会发生两件事：第一，她会觉得很爽，因为她觉得征服了你；第二，她会对你的兴趣下降。她喜欢这种征服的感觉，并不表示她喜欢你这个人，也不表示你有机会“搞定”她。

所以，当女人给出圈套时，普通男人会迫不及待地跳进去，以为可以趁机表达在乎，以为这样很浪漫。但你千万不要跳进去。

你应该原封不动地“奉还”回去，也就是说，你要设下一个新圈套让她来跳。甚至，可以完全不理会她。这个时候，沉默往往就是最好的应对方式。

关于圈套，还有一个有趣的现象：**看起来越像是圈套，别人跳进去的几率就越低。**

例如：
*   如果有人说：“帅哥，你出去的时候，可不可以帮我带杯水？” 这是一个很合理的请求，大部分人不会拒绝。
*   但如果女孩说：“傻逼，你给我站起来，像个娘们一样，到厨房去给我拿一杯水！” 这时，很少有人会落入这个圈套，因为接受就等于承认自己是“傻逼”和“娘们”。

因此，你的圈套要从小处入手。你要引诱目标，让她无意间进入一连串的小圈套。比如你可以跟她说：“哎，你能不能帮我拿一下饮料？谢谢。” 久而久之，当她落入你的框架时就会毫无感觉。你的圈套也就可以变得更大、更频繁。很快，她可能就会开开心心地帮你按摩、做饭。

## 使用框架的时机与心法

最后，也是最重要的一点：使用强框架是聊天第三阶段的事情。

如果你过早地展现自己的强框架，女孩是不会搭理你的。只有女孩觉得你有价值的时候，你展现框架，她才会服气。否则，她都觉得你无所谓，怎么可能服从你的框架呢？

这个道理可以用一个经典的“四等人”说法来完美解释：

1.  **第一等人：有本事，没脾气。**（完美）
2.  **第二等人：有本事，有脾气。**（恃才傲物，可以接受）
3.  **第三等人：没本事，没脾气。**（老好人，普通人）
4.  **第四等人：没本事，有脾气。**（最惹人厌）

框架一定要到聊天第三阶段才可以用。如果我们在聊天第二阶段已经展现了价值，女孩已经知道我们是有本事的人了，那么这个阶段我们再展现自己强硬的框架和态度，是可以接受的。女孩会认为你是“第二等人”（有本事有脾气）。

但如果你过早地展示强框架和强硬态度，那么女孩就会觉得你是“第四等人”（没本事有脾气），是个“垃圾”。所以一定要注意时机。

同时，框架还有一个重要作用：**引导女孩来主动追逐我们。**

我们的框架必须展示出“我们很受女孩欢迎，是她喜欢我们、追逐我们”。如果你不去这样做，女孩就会不停地给你设置圈套，借着每件小事来设定“她才是主角”。我们不能落入这个框架，而是要抓住并再利用这个框架。

## 实战话术与心态

那么，具体在聊天中如何营造“我才是主角”的框架呢？这需要借助每一句话的潜台词。

我们要注意每一句话的潜台词，营造一种“你才是主角，你才是被追的人”的氛围。当我们和女孩接触时，可以借由每一句话，引导女孩来“追逐”你。

以下是几个调侃的话术例子，用于营造框架：

*   **当女孩碰你的时候**，你可以说：
    *   “天哪，你怎么这么主动？”
    *   “哎，别乱摸，这个不是免费的。”
    *   “哎呀，你怎么这么猴急？”
*   **当女孩问你“你觉得我们在一起合适吗？”**，你可以说：
    *   “啊，我觉得你是个好人。”（给她发“好人卡”）
    *   “我想多了解你一些。”（潜台词：我对你有点兴趣，但你说合不合适？想太远了，你太主动了。）
*   **当女孩对你好、关心你的时候**，你可以说：
    *   “别对我这么好，要不然我很容易感动的。”（潜台词：你再对我这么好，我可能就被你追到了，你在主动追我哦。）

要注意的是，我们说这些话时，都是用**调侃的口吻**，而不是特别正经的态度。当我们调侃她太过主动或想“骗”我们时，并不表示这是事实，只是因为我们就是那种“坏坏的”男人，在故意歪曲状况。但如果你的框架够强，她会被你吸引，并煞有介事地和你打情骂俏。

女人知道我们是在开玩笑，所以她们会愿意和我们进行这种轻松的打闹。但要记住：**女人只会回应有价值的男人。** 所以再次强调，这些是在聊天第三阶段，当你已经建立一定吸引后才可以做的事情。

这些话术本身并不重要，重要的是它们所隐含的框架的内在力量。如果你的心态越来越强大，技巧越来越熟练，正确的话术自然会从你嘴里冒出来。但如果你内在还不够强大，技巧也不够熟练，那么多背一些这样的台词，在和女孩互动时偶尔说一些，也能帮助你营造出“你才是主角”的感觉。
