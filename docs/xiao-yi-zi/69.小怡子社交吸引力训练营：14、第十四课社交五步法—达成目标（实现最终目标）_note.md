# 课程69：社交吸引力训练营 🎯 第十四课 社交五步法—达成目标（实现最终目标）

在本节课中，我们将学习社交五步法的最后一步——**达成一致**。这是整个流程的终点，旨在帮助你顺利实现社交目标。

## 概述：达成一致是顺水推舟

上一节我们介绍了欣赏环节，本节中我们来看看如何完成最后一步。社交五步法中，前四步是构建关系的核心，而**达成一致**是顺水推舟的事情，无需花费太大精力。

这就像踢足球：前四步帮助你盘带、过人，甚至绕过门将；最后一步只需临门一脚，几乎不可能失败。因为前四步已打破陌生关系，让对方对你产生兴趣，并确认了兴趣的真实性，你也回应了同等兴趣。此时，你只需做一件事：**提出要求**。

这个要求是五步法的终点，是你流程下来想要达到的目标。但请注意，目标不一定是最终目标，也可能是阶段性目标。

以下是不同社交场景的目标示例：

*   **情感关系**：目标可能是确立男女朋友关系，也可能只是完成一次邀约。
*   **面试场景**：目标可能是谈妥薪水并入职，也可能只是通过初试获得复试机会。
*   **商务洽谈**：目标可能是直接成交、签订协议，也可能只是获取进一步深入沟通的机会。

总之，目标有深有浅，可以是最终目标，也可以是阶段性目标。

## 如何提出合适的目标

具体提出何种目标，取决于前面四个环节的能量强度，尤其是**吸引环节**。

还记得筛选环节吗？对方对你兴趣越强，你筛选的强度就可以越强。同理，如果对方通过了更强的筛选，你就可以给出能量更强的欣赏，从而获得很深的**结盟感**。在此前提下，你就可以提出能量更强、更接近最终目标的要求。

反之，如果在吸引环节，对方表现出的兴趣一般，你给出的筛选也不强，那么最后提出的目标最好只是**阶段性目标**。

在两性关系发展中，这个目标往往是**邀约**。因为关系大多从网络聊天开始，而成为男女朋友的关键是先见面，从线上转到线下，真实地了解彼此，再进一步发展关系。

## 实例解析：健身房女教练

我们以健身房女教练为例，回顾整个流程并完成最后一步。

1.  **破冰与了解**：女教练接待你，介绍设施课程，你们聊到职业、兴趣爱好。
2.  **吸引**：她称赞你工作能力强、成熟稳重，并主动加微信，表达兴趣。
3.  **筛选**：你提出“技能互换”（你指导工作，她教你训练）。
4.  **欣赏**：她通过筛选，你表达欣赏：“有事业心的美女不多见，我得好好指点你。”她给予正面回应。
5.  **达成一致**：感受到结盟感后，你直接提出邀约：“不如咱们就从这个周末开始吧。”这就是提出要求，完成邀约。

**达成一致**就是这么简单。

## 社交五步法的完整应用

学完整个流程后，更重要的是深入理解并开始练习。社交五步法不仅适用于情感关系，在任何社交场景中都能通用，帮助你有效达成目标。

以下是社交五步法在其他两个场景的应用示例。

### 应用场景一：面试求职 🏢

在面试中与面试官或HR打交道时，可以这样应用：

**破冰**：见面后先打招呼、善意微笑，并聊即兴话题（如公司装修、楼宇特点），以拉近关系，消除陌生感。

**了解（建立吸引）**：进行自我介绍，展示教育背景、工作经验、能力特长和职业规划，吸引HR产生兴趣。同时注意捕捉对方的正面或负面反馈。

**筛选**：如果感受到对方有较强兴趣，可提出标准进行筛选。例如：“我提前说明，我不太喜欢喝酒，担心公司的酒文化。不知贵公司是否如此？”观察对方是否迎合你的标准。

**欣赏**：若筛选通过，表达对公司和岗位的看好与期待，展示强烈加入意愿。

**达成一致**：在面试场景下，达成一致通常是谈薪水、确定入职细节，或推动下一轮复试。

如果不按此流程，可能会犯错误。例如，过早表达欣赏可能导致对方在了解阶段考核更严、压榨薪水；或在对方有兴趣时未筛选就直接谈薪水，往往只能得到“等通知”的模糊答复。学习此流程后，你就能明白过程原委并避免这些问题。

### 应用场景二：商务洽谈 💼

假设你是广告销售，约见客户公司的市场部负责人，希望对方投放广告。

**破冰**：先将合作事宜放一边，通过夸赞拉近关系。例如：“王总，业内都说您是营销专家，我除了谈合作，也想借此机会向您学习。”或称赞办公室环境。

**了解（建立吸引）**：提前准备，展示价值。例如：“王总，我根据贵公司定位做了份营销方案，针对淡季、旺季和年底活动有些想法……您是专家，请指点。”通过免费提供方案，了解对方公司和个人的想法。

**筛选**：如果对方对方案感兴趣或想法一致，可进行筛选。例如：“王总，您的意见很好。我冒昧问一句，如果方案让您满意，咱今年还有多少广告预算？”根据对方反应（如“有预算”或“可申请”）判断兴趣真实性。

**欣赏**：根据交流细节，表达对对方或此次合作的欣赏与期待。

**达成一致**：可能是约定再次详谈方案，或私下吃饭深入沟通，推动合作更进一步。

若不按此流程，业务新手常犯的错误是：急于求成，一味谈产品、价格，恳求机会，结果常被拒绝；或对方有兴趣时未筛选，回去埋头改方案，最终被告知“没预算”，努力白费。学习此流程后，你就能理解并避免这些情况。

本节课中，我们一起学习了社交五步法的最后一步——**达成一致**。通过情感关系、面试和商务洽谈的实例，我们看到了如何根据前期互动能量提出合适目标，从而顺利完结整个社交流程。

记住，以上例子的核心是掌握**原理**，而非死记话术。在生活中多种社交场景应用此五步法，能让你少走弯路。
