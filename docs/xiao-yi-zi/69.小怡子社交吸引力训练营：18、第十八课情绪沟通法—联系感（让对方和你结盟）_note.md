# 69.小怡子社交吸引力训练营：18、第十八课情绪沟通法—联系感（让对方和你结盟）_note

在本节课中，我们将要学习情绪沟通法的第四个核心情绪：联系感。我们将探讨为什么在建立吸引后不能立即邀约，以及如何通过建立联系感让对方感觉与你结盟，从而为后续互动打下坚实基础。

上一节我们介绍了吸引情绪的作用，本节中我们来看看如何将吸引转化为更深层次的情感连接。

## 为什么吸引后需要联系感？

很多人在社交过程中存在一种错觉：一旦产生吸引，对方就会轻易配合。实际情况并非如此。

吸引情绪能让对方对你产生兴趣并向你靠拢，但吸引本身往往不足以成为对方赴约的筹码。这是因为你在吸引阶段展示了高价值，但对方可能会认为这些价值与他无关，你或许只是在炫耀。

从对方视角思考：如果你认识了一位非常优秀的异性，你不会立即开始追求。通常，你会在感觉有机会得到对方时才开始付出。这是因为此时对方的优秀与你的努力产生了联系。

同理，在社交中，仅有吸引时，对方会觉得你的价值与他无关。你需要让他感觉有机会获得你的价值，你们之间才可能结盟。

例如面试场景：你展示了技能后，还需表达对公司的兴趣，面试官才会觉得你的才能与企业相关，此时谈薪资会更顺畅。

因此，我们需要进入联系感阶段，让关系更紧密，加速结盟。

## 什么是联系感？

两个人能从陌生走到一起，完全是因为联系感。但需注意一个常见误区：常联系不等于有联系感。

例如，两个人有相似的生活轨迹和品味，但若没有情感交集，他们可能只是认识多年的普通朋友。

联系感的本质是**以情感纽带为基础的联系方式**。它不仅是普通联系，更是精神层面的沟通。这种联系依赖于情感的代入，而非简单的话题内容。

**联系感 = 情感纽带**

## 如何建立联系感？

以下是四种建立联系感的有效方法。

### 1. 欣赏

在吸引阶段后，表达对对方的欣赏能产生彼此认同的感觉。这种认同感能满足对方自我实现的需求，是拉近关系的强大纽带。

欣赏带来的情绪是**认同感**或**自我实现感**。被有价值的人认可，能极大满足对方的精神需求。

具体表达欣赏的方法，可回顾社交步伐中关于欣赏的课程。

### 2. 真实

展示真实的一面能有效建立联系感。在日常生活中，人们常戴面具社交，因此遇到一个敞开心扉的人会感到格外珍贵。

如果对方已认为你是高价值个体，你的真实会让他感觉被当作自己人，从而建立深厚的情感纽带。

**真实 = 展示不遮掩的自我**

### 3. 分享脆弱

分享脆弱是“真实中的真实”，能瞬间让你的形象立体化。脆弱面是人们最深的秘密，只愿与信任的人分享。

当你主动分享脆弱时，对方会感到被特殊信任，从而增强你们之间的联系感。

但需注意：**分享脆弱必须在建立吸引之后进行**。若过早暴露脆弱，可能被视为低价值，反而削弱吸引力。

如何分享脆弱？最好的方式是**讲故事**，描述高价值背后的敏感与内心需求。

例如在面试中，展示能力后可以补充：“虽然积累了一些能力，但很多时候是独自面对。我其实渴望一个靠谱的团队，一起合作成就更大的事情。”

**分享脆弱 = 展示高价值背后的情感需求**

### 4. 共谋

共谋是行为上的联系感，指建立只有你们两人懂的小秘密或小套路。

例如，为共同的朋友起一个只有你们知道的外号。这种共谋越多，联系感越强。因为每当看到相关事物，对方就会想起你，形成“瞬间想念”。

共谋可以很简单，例如给食物起外号。这种共享的“秘密语言”能持续强化你们之间的独特连接。

**共谋 = 共享的秘密或内部玩笑**

## 总结

本节课中我们一起学习了联系感的重要性及其建立方法。

联系感是情绪沟通法的关键环节，它能让对方感觉与你的价值相关，从而愿意结盟。我们介绍了四种建立联系感的方法：欣赏、真实、分享脆弱和共谋。

在实际应用中，无需使用所有方法，应根据互动节奏灵活选择。通常，适当的欣赏与认可已足够。

当你成功建立吸引并进入联系感后，就可以准备进入情绪沟通法的最后一个情绪：达成一致。我们将在下节课详细讲解。

**核心公式回顾：**
*   **联系感** = 情感纽带
*   **分享脆弱** = 展示高价值背后的情感需求（需在吸引后使用）
*   **共谋** = 共享的秘密或内部玩笑
