## 核心心态：放下你的面子

学习方法是外在框架，而内在心态才是成功的引擎。在开始实践吸引技巧前，最重要的一条心态原则是：**放下你的面子**。在女性面前，面子不值钱。执着于面子会导致两种致命心态。

### 受害者心态

受害者心态源于不自信和过强的面子观念。拥有这种心态的人常怀疑一切，当遇到超出其原有价值观理解范围的事情时（例如快速发生关系），他们会站在道德制高点进行批判。其根源在于他们的现实认知有限，无法理解更广阔的可能性。

**公式：受害者心态 = 不自信 + 面子薄 + 怀疑论 + 道德批判**

正确的做法是放下所有成见和面子，先思考新方法、新世界观是否也有其道理。

### 优越感心态

优越感心态同样源于脆弱的面子。这种人自尊心极薄，害怕被冒犯，容易将他人的言行解读为敌意，容易嫉妒。他们可能依靠家境、外表等外在条件支撑虚假的优越感，无法正视真实的自己。在社交中，他们表现得非常“强硬”或喜欢辩论，但这并非强大，而是内心脆弱的体现，如同害怕蟑螂却过度拍打它的人。

**核心概念**：在社交中，没有什么是绝对“强硬”的。社交的目的是沟通，前提是对方能接受并理解你的话。你需要让对方感觉你们是同一阵营，而非对立状态。
