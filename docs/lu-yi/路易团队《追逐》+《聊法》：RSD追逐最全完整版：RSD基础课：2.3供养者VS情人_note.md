# 路易团队《追逐》+《聊法》：RSD基础课 2.3 🧠 供养者 VS 情人

在本节课中，我们将要学习RSD理论中一个非常核心且有趣的概念：**供养者**与**情人**。这是理解女性潜意识如何对男性进行分类的关键，也是决定你与女性互动走向的基础。

## 概念起源：两种类型的男人

上一节我们介绍了吸引力的基础，本节中我们来看看女性潜意识中的分类机制。在女性看来，存在两种类型的男人。女性可能不会在逻辑上明确地将男人分类，但她的潜意识和基因中写入了这样的信息。她会下意识地、无意识地对任何有潜在亲密可能性的陌生男性进行分类。

以下是女性分类的两种基本类型：

*   **情人**：这是女性常说的“有感觉”的类型。她们希望与这类男人谈恋爱，觉得特别有感觉。这类男人通常被认为特别会与女人交往，甚至有些“坏”。
*   **供养者**：女性可能会这样描述这类男人：“他可能不太适合谈恋爱，但是结婚的类型”。跟他在一起没有那么多激情或感觉，但他可能挺有钱，或者被认为是“适合当男朋友但不太适合当情人”的类型。

## 生物学与社会学类比

为了理解这两种角色的来源，我们可以从社会学和生物学角度分析，这与动物行为很相似。

以鸟类为例：许多雄性鸟类会筑一个精致的巢，并收集亮晶晶的东西（类似于人类女性佩戴首饰）来吸引雌鸟交配。交配后，它们会产下后代。此时，筑巢的雄鸟需要外出捕食，回来喂养雌鸟和孵出的幼鸟。

在这个过程中，筑巢和捕食的雄鸟在生物学上起到的作用是 **`供给资源以繁殖后代`**。这类似于人类社会中，男性需要提供房子（筑巢）和赚钱养家（捕食）的角色。对于女性来说，这种供养对其生存和生活质量至关重要。

然而，故事还有另一面：当那只“供养型”雄鸟外出捕食时，可能会有另一只拥有更优基因、更健康、对雌鸟更有吸引力的雄鸟（即“情人”）偷偷前来与雌鸟交配。结果，雌鸟生下的后代可能都是这位“情人”的。而那只“供养者”雄鸟回来后，会不知情地喂养所有幼鸟。

我们发现这两类雄鸟功能截然不同：
1.  **供给资源**：筑巢（买房）和捕食（赚钱）。
2.  **提供优质基因**：作为有感觉的“情人”，雌鸟愿意与其交配，以生下更健康、更有生存优势的后代。

人类社会与此一模一样。新闻中常出现男性抚养多年后发现孩子非亲生的案例，其内在逻辑与此相通。

## 核心机制：分类与双重标准

女人在见到任何男人时，都会下意识地将其归类为**供养者**或**情人**，几乎没有模糊地带。一旦被分类，就很难改变。

一个关键点在于：**一旦你做出任何供养者类型的行为，就会立刻被女人划分为供养者**。

以下是供养者类型行为的例子：

*   买礼物、送东西给女人。
*   没有原则地、不停地为女人花钱。
*   总是带女人去最高档的餐厅、VIP影院。
*   进行所有传统约会模式：逛街、吃饭、看电影，并在过程中为女人大量购物。

根据许多人的失败经验，这样做通常无效。你越陷入供养者的行为和角色，女性将你判定为供养者后，你们的关系发展就会越困难。

相反，**情人**则是女性一见到就会想“我会和他上床”的类型。女性对情人的言行态度也完全不同。情人的特质通常与“雄性领袖”相关，拥有更好的基因。这里**更好的基因**并非指长相，而是指：

*   强大的决断力和控制力。
*   情绪稳定，难以被挑战。
*   拥有自由感和主导力。
*   能给女人带来幽默、开心等积极情绪。

## 双重标准的影响与“解法”

弄清了分类，我们必须明白：**女人对供养者和情人奉行完全不同的双重标准**。

对待情人，女性的容忍度和接受度极高。情人搂抱她，她会觉得理所当然、很酷。因此情人与女性从认识到发生关系可能非常迅速（几小时或一两天）。

*   “你把我当什么人了？”
*   “我是这种女孩吗？”

这是因为在女性看来，供养者的肢体接触是“别扭的、有目的的”。因此，真正的解决方法是：**想尽一切办法避免所有供养者行为，努力成为那个情人**。

## 正确的关系发展路径

那么，供养者和情人的正确关系应该如何构建？

在RSD中有一个核心概念：**想要成为一个女孩的男朋友，最强大的位置是从认识她开始，以最快速度发生关系，然后主动从情人角色转换为包含供养者特质的男朋友角色**。

**错误路径**：
*   `认识 -> 送礼/讨好（供养者行为）-> 被归类为供养者 -> 关系停滞或遭遇羞耻感 -> 难以发生关系`。

成为情人后，你可以在长期关系中既保持情人的吸引力特质，又真心实意地付出。这才是健康的关系模式。
