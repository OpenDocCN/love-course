# 《聊法》：第9节：怎么聊——聊天的情绪表达_note

在本节课中，我们将要学习如何将普通的聊天话题转化为富有情绪色彩的对话。上一节我们介绍了如何寻找聊天话题，本节中我们来看看如何通过情绪表达，让对话变得生动有趣，从而更好地吸引对方。

## 为什么需要情绪表达？

大部分男生在与女生聊天时，倾向于使用直男思维，无法理解女生的情感需求。例如，当女生生病或不开心时，男生可能会说“多喝点水”，但女生真正需要的是关心和安慰。

女人在工作中遇到不顺心的事情并向男生抱怨时，很多男生会选择讲道理、分析对错。然而，女人此时需要的是一位倾听者，需要的是附和与共情，而不是指责或批评。

如果你想与女人进行良好的沟通，就不能仅仅从理性和逻辑出发。相反，你应该从自身感受、女生的感受以及双方互动的感受出发，培养一种新的思维方式。

## 培养发散性思维

在与他人，特别是与女生聊天时，我们的大部分话题都属于“第三方话题”。这类话题与第一人称的“我们”和第二人称的“你”都没有直接关系。因此，在聊这些话题时，我们需要培养发散性思维，尝试将话题与自身、对方以及双方的关系联系起来。

以下是培养发散性思维的步骤：

1.  **联想与话题本身相关的内容**
2.  **联想与自身相关的内容**
3.  **联想与对方相关的内容**
4.  **联想与双方关系相关的内容**

我们以“工作”这个话题为例。当你问女生“在干嘛？”，女生回答“我在上班”。大部分男生可能会说“那好好工作吧”，这样就切断了话题。为了延续对话，你可以进行联想。

例如，你可以说：
*   `你上班玩手机不会被老板揍吗？`
*   `你上班玩手机我要给你老板打小报告。`

请注意，不要照搬这些话术，而是理解其背后的思维模式。

## 实践：如何将话题聊出情绪

接下来，我们通过一个具体例子来练习如何将话题聊出情绪。假设你问女生“做什么工作的？”，她回答“我是幼儿园老师”。

### 1. 联想与话题本身相关的内容

女生的回答是“我是幼儿园老师”。与这个话题本身相关的回复可能包括：
*   那些小孩闹不闹？
*   你是不是带出了一群熊孩子？
*   分享你自己或亲戚家小孩的趣事。
*   分享童年搞笑的经历。

这些回复可以帮助你延续话题，但绝大多数男性只停留在这个层面。

### 2. 联想与自身相关的内容

如何将“幼儿园老师”与你本人联系起来？以下是一些例子：
*   `老师好！`（加上一个坏笑的表情）
*   `原来我们是半个同行。`（假设你是培训师）
*   `我们分手吧，我从小最害怕老师了。`
*   `我的小侄子特别闹，你会管吗？`
*   `你在哪个幼儿园？我把妹妹也送过去。`

通过将无聊的第三方话题与自身关联，话题就不再枯燥了。

### 3. 联想与对方相关的内容

每个人，尤其是女生，都更关注自身。将话题引向对方，能让她更感兴趣。例如：
*   `以后你也要生娃，现在当幼师就当提前预习当妈妈了。`
*   `原来你是个孩子王啊！`

### 4. 联想与双方关系相关的内容

如何将“幼儿园老师”与你们的关系联系起来？例如：
*   `假如将来找你当老婆，肯定特别轻松，因为你能分分钟把孩子哄好。`
*   `以后我们的孩子你会不会也这样带？`（此回复适用于聊天中后期，有一定暧昧度时，前期使用需注意方式，避免引起反感。）

以上就是根据一个旧话题，联想出新话题的四种方向。通过这种方式，你可以锻炼自己的发散性思维。

## 培养审视聊天目标的习惯

我们都希望通过聊天吸引喜欢的女生。因此，我们的对话应尽可能以拉升关系、升温暧昧为主要目的。你需要养成一个习惯：时刻审视自己的聊天内容，看看你的回复是否有助于推进你和女生的关系。

回顾你过往的聊天记录，特别是那些聊不下去的、女生兴趣逐渐减弱的、或反应很差的对话，你会发现，大部分情况下你只停留在第三方话题层面。

我并非说聊第三方话题或逻辑性话题不好。在适合的时候，比如分享生活态度或爱好时，你应该聊这些正经话题。但如果你通篇都只聊第三方话题，没有任何关系拉升和情绪注入，女生最终不可能喜欢上你。

就像心电图一样，如果始终是一条直线，人就失去了生命迹象，你们的关系也会死掉。女人是情绪化的动物，她们对你是否感兴趣，取决于你能否在聊天中不断拉升关系、注入情绪，制造升温的感觉。

这就是为什么我们要教你与女生调情。只有不断升温、推拉，使用各种技巧让女生对你产生更大兴趣，达到某个临界点后，你才能得到想要的结果。

总有男生抱怨：“为什么我和她聊了这么多话题，她还是不喜欢我？” 试想，即使你们聊了10086个话题，但如果全是第三方话题，没有情绪、没有调情，那么聊到第一万个话题时，女生对你的感觉和聊第一个话题时没有任何区别——就像一条水平的心电图，她不可能喜欢你。

也有人说：“我和她把该聊的话题都聊完了，为什么她对我兴趣越来越小？” 话题是有限的，如果你只是机械地罗列话题，而不去延伸、不加入即兴发挥和调情情绪，那么几分钟就能聊完所有话题，女生依然不会喜欢你。

当然，我不是教你对女生全程油嘴滑舌。你应在该正经时正经，该调情时调情，做到**情绪线与逻辑线相结合**。这需要你不断实践，摸索出适合自己的分寸。我们之前讲过的“社交信号”能帮助你在聊天中更好地判断何时该如何聊。

## 培养多模式回复的思维

如果你之前是“话题终结者”，总是把天聊死，那么我建议你在和女生聊天时这样锻炼：针对女生的每一句话或问题，尽可能想出**三种不同风格**的回复，然后从中挑选出对当下聊天和关系推进**最有帮助**的那一句作为你的回复。

注意：不是选最逗逼、最色情或最能炫耀你价值的回复，而是选择对**当下聊天情境和关系发展最有利**的那一句。

如果当下适合逻辑探讨，那就正经聊；如果适合调情开玩笑，那就用暧昧的方式聊。总之，选择对当下关系帮助最大的回复。

### 举例说明

假设你正在和一个对你有基础好感但还未特别暧昧的女生聊天。你开了句玩笑调戏她，她回复说：“不要欺负我这种又年轻又好看的人。”

我们可以抓住这句话中的不同关键词，联想出多种回复风格（**以下仅为示例，切勿照抄**）：

*   **抓住“欺负”这个词**：
    *   `我那么爱你，怎么会欺负你呢？`

    *   `像你这么年轻的宝宝，我怎么舍得欺负呢？`
    *   `你不知道吗？我就专门喜欢欺负小仙女。`

*   **抓住“年轻”这个词**：
    *   `宝宝你真不要脸，你都是奔三的人了，还自称年轻呢。`

*   **抓住“好看”这个词**：
    *   `哎呀，你如果真的好看的话，我怎么会瞎了眼爱上你呢？`

*   **抓住“人”这个词**：
    *   `你还是人吗？`（如果女生反应好或继续开玩笑，可以补一句：`你不是天天自称是小仙女，靠喝仙露度日吗？`）

通过抓住对方回复中的关键词，你可以联想出各种不同风格的回复，让对话充满变化和情绪。

## 本节课总结 📝

本节课我们一起学习了如何通过情绪表达来提升聊天质量：

1.  **运用发散性思维**：根据对方的回复，联想出与**话题本身**、**与你自身**、**与对方**以及**与你们双方关系**相关的新话题。
2.  **培养目标审视习惯**：聊天应以**拉升关系、升温暧昧、增进了解**为主要目的，避免通篇逻辑或通篇油滑，做到**情绪线与逻辑线相结合**。
3.  **建立多模式回复思维**：针对女生的每句话，练习构思多种风格回复，并选择对**当下关系推进最有利**的一句进行回应。

掌握这些方法，并加以练习，你将能更好地在聊天中表达情绪，建立更深入、更有吸引力的连接。
