## 什么是认同感？

首先，我们需要弄清楚“认同感”这个概念。所谓认同感，可以用一个最简单的场景来解释：当你追求一个女生时，你的行为模式通常是“我要追你”。

**具体表现如下：**
*   我要给你买礼物。
*   我要求你了，我送你这么多东西。
*   我对你好，我每天给你送吃的送喝的。
*   你得病了吗？我快点买东西去。

在潜沟通中，这些行为向女人传递了一个核心概念：**我的价值不如你**。我需要通过买东西、在你生病时买药、在你饿时买吃的、为你充话费等方式，来获得你的认同。

这本质上是在说：“求求你了，我都为你做了这么多，求求你认同我吧，求求你喜歡我吧。”这些行为都在表达一个概念：**我需要你的认同，我在跪下来求你认同我**。

祈求女人的认同，这种行为就叫做 **“寻求女人的认同感”**。用正常的人类语言翻译，就是“我在追女人”。然而，在RSD理论中，一个非常重要的概念是：**你必须让女人来追逐你**。

## 什么是剥夺认同感？

理解了寻求认同感的弊端后，我们自然会问：那女人为什么会来追逐你呢？答案的关键在于 **“剥夺认同感”**。

**但事实恰恰相反。**

这就好比一个食物：好的鸡蛋对你有营养价值，你愿意吃；但鸡蛋变坏发臭了，对你没有价值，你就会想把它扔进垃圾桶。对女人而言，男人的逻辑也是如此。

**一个典型的场景是：**
