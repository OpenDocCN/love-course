上一节我们探讨了情绪共振与赋予资格，为互动奠定了良好的情感基础。然而，仅仅拥有吸引力和好情绪是不够的。本节中，我们来看看如何通过“确定物流”来主导整个互动进程，确保你能根据实际情况做出最有效的决策。

但假如你不清楚女孩当下的情况是怎么样的，假如你不知道该如何去处理下一步，如果你不知道女孩的物流情况是怎么样的，你就不清楚你和她两个人的关系，如何拉升到下一步。

## 如何确定女孩的物流？

以下是两种确定女孩物流状况的核心方法。

你要养成一个习惯，也就是要在每次和女孩搭讪的时候，问下面的这样的问题。

*   你是和朋友一起来的吗？

*   你是合租还是自己住？
*   现在和你在一起的就是你的室友吗？
*   你们是坐什么车来的？
*   你一会怎么回家？
*   你明天要干什么？

你越快发现女孩当下的状况，事情就会对你越有利。从和女孩对话的一开始，你就要去确定物流。

避免过于直接露骨的问题，如“你在哪个小区住？”。可以换成更自然的问法，如“你在沈阳哪个位置住啊？”或“假如今晚上你回去晚了，你爸妈不会训你吧？”，从而间接获得信息。

*   **“哎，你们去哪开party啊，好玩吗？”** -> 她有兴趣，可以继续引导。
*   **“啊，我是去不了了，我明天还得早起上班呢。”** -> 她有时间限制。

你给出了一个很合理的小借口，女孩的回应要么会问细节（表明兴趣），要么会说不行并给出具体的物流信息作为理由。

## 如何确定自己的物流？

有趣的是，很多男人搞定了女孩的物流，却忽略了自己的物流。你必须很实际的去考虑这个问题。

每次出门前，你都需要问自己以下问题：

*   **交通方式：** 我是走着去还是坐车去？坐什么车？一会儿怎么回来？
*   **时间规划：** 今晚我最晚能玩到几点？（例如，去夜店要考虑第二天上班）

提前弄清楚这些，你才能在现场从容地规划如何把女孩带回私密空间，或进行有效的转场。

最后一个你随时都该问自己的问题是：你眼前的这个女孩，她的下载进度条到了百分之多少了？

也就是说，你需要评估：

*   **情绪状态：** 她是还有些拘谨，还是已经沉浸在你的好情绪里？
