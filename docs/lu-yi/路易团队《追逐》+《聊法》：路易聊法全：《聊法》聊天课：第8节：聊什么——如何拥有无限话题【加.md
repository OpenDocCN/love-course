# 路易聊法全《聊法》聊天课：第8节：聊什么——如何拥有无限话题 💬

在本节课中，我们将要学习聊天的核心目的，并深入探讨如何构建和延续话题，让你在与他人交流时永远不愁没话可说。我们将分析常见的沟通误区，并学习将“骨架”话题转化为“血肉”丰满对话的实用技巧。

## 聊天的目的与误区

首先，我们需要明确聊天的目的。很多人认为聊天是为了约会或让对方喜欢自己。这引出了一个哲学问题：什么是目的，什么是结果？

例如，吃火锅的目的是享受美食，结果是吃饱了，生命得以延续。喝咖啡的目的是约姑娘见面、消磨时光，结果可能是晚上失眠。

那么，和女生聊天的目的是什么？目的是让姑娘喜欢你、对你感兴趣。结果是，她真的喜欢你了，并想和你见面。很多人第一步就走错了：刚和女生聊几句，就不断邀约。即使被拒绝，仍反复邀约。即使你提出请客吃昂贵的大餐，女生也不太可能见你。即便见了，你也可能只是“买单侠”，请了一顿饭而已。

## 沟通的本质与非正常模式

人类最本质、最纯粹的交流模式是：我问一个问题，你回答后反问我；我回答后，再问你一个新问题，如此循环往复。这非常简单、纯粹，是沟通的本质。

然而，很多男生在和女孩聊天时，会陷入两种非正常沟通模式。

以下是第一种非正常沟通模式：

*   **极端自我（查户口/夺命连环问）**：一直问问题，丝毫不考虑对方的反馈。聊天时一个接一个问题，像连珠炮一样。请注意，聊天时问问题是好事，但不应在不考虑对方任何反馈的情况下不断提问。有时你也会遇到喜欢查户口、夺命连环问的女生。这未必是好事，她可能想了解你，也可能是个极端自我、不考虑他人感受的人，甚至可能是骗子，通过询问职业、收入来推销原油期货或股票。你需要分辨她是真的感兴趣，还是在夺命连环问。

以下是第二种非正常沟通模式：

*   **极端被动**：字面意思就是不会主动。对大部分女生来说，社交初期不太主动、不太会问过多问题，这很正常。因此，你更需要学习如何让聊天更有趣、更有吸引力，带领女生，主动提供话题。但如果作为男人，你非常被动，不敢主动带领，怎么可能得到想要的结果呢？

## 聊天的骨架与血肉

本节我们要搞清楚聊天中“骨架”和“血肉”的联系。

**首先是聊天中的骨架**。骨架就是聊天中的话题，是那些具体、现实、有真实含义的“干货”，没有水分。例如你问女生的问题、她问你的问题、你们互相讲述的事情。

很多兄弟会问：“到底和女生聊什么？” 请记住：**你想了解什么，就去问什么；你想让她知道什么，就去表达什么**。如果你真的不知道聊什么，那就先别聊。但既然我们在学习，我们会教你具体的话题和培养话题延续能力的方法。

**其次是聊天中的血肉**。血肉是由上述具体话题引发出的讨论、联想和想法。仅仅建立骨架（问答）是不够的，你需要填充即兴发挥的内容，将话题延续、铺开、展开，延伸出新的内容。否则，聊天会变成枯燥的一问一答，令人厌烦。

上一节我们介绍了沟通的本质与误区，本节中我们来看看如何通过具体例子，为骨架话题填充血肉。

## 话题延伸实例分析

以下是几个将骨架话题延伸出血肉内容的例子：

*   **年龄话题**：问完年龄并互相告知后，话题不应结束。如果女生比你小，你可以说：“原来你是个宝宝。” 或 “你要叫我叔叔了。” 如果女生比你大，可以说：“女大三，抱金砖。” 或 “在我眼里你永远18岁。” 不去填充即兴发挥的内容，努力延续话题，聊天就会变得枯燥。

*   **美食话题**：问女生“平时喜欢干嘛”，她回答“喜欢美食”后，你可以说：“就知道你是个小吃货。” 如果她说正在吃，你可以说：“多吃点，宝宝正在长身体。” 在聊天中后期，有一定吸引后，可以说得更暧昧：“多吃点，宝宝还在发育的时候。” 你还可以借此聊喜欢的餐厅，植入未来一起吃饭的邀约；聊到厨艺，植入一起做饭的邀约；甚至聊到体重、减肥等暧昧话题。

*   **身高话题**：互相告知身高后，可以自由发挥。例如联想到“最萌身高差”，这个差值可以由你定义。还可以聊女生是否喜欢穿高跟鞋。在聊天中后期有一定暧昧后，可以聊“这个身高需要踮脚才能接吻”等。甚至有学员在已有吸引的基础上聊到：“如果身高差很多，有个体位（69）是做不了的。” **切记，在聊天前期千万不能这么聊。** 我知道即使这样提醒，仍会有人不分时间、场合、对象地胡乱套用。话已说清。

以下是常用话题列表：

*   **陈述状态**：例如“我正在公司加班”。可以丰满为：“我正在公司苦逼地加班，感觉自己睡眠好少，老得好快。”

*   **爱好/喜欢干什么**：另一个最容易延伸话题的点。
*   **喜欢看什么剧**：能在细节中找到共同话题。
*   **喜欢看什么书**：同上。
*   **喜欢听什么歌**：同上。
*   **喜欢吃什么美食**：绝大部分女生都是吃货，此话题可排第一。

最后，我想教你一个锻炼无限延伸话题能力的方法。这是我们路易团队线下私密课的指定内容，效果拔群。

**“此与联想”游戏规则如下**：

最好有个朋友在旁边。首先，你说出一句话，信息量要适中（不能太短如“我饿了”，也不能太长像几个句子拼凑）。例如：“我今天下午打了一下午球，感觉肚子好饿。” 或 “昨天电视上那个飞机失事的新闻，你听说了吗？”

然后，你的朋友从你这句话里挑出一个词（不能是开头或结尾的词），你必须在下一句话中用到它。例如，从“我还挺喜欢刚下完雨之后，在外面散步的感觉”中，关键词不能是“我”或“感觉”，可以是“雨”、“外面”或“散步”。

朋友指定新关键词后，你用这个词联想出下一个句子。朋友再从新句中挑出新关键词，以此类推。

**示例**：
A: 我还挺喜欢刚下完雨之后，在外面散步的感觉。
B: （指定关键词）散步。
A: 好久没有和大咖散步了，感觉还挺想念他的。
B: （指定关键词）大咖。
A: 我从来没看过《百变大咖秀》，也不理解为什么女孩们这么喜欢看。
B: （指定关键词）女孩。
A: 如果我是个女孩的话，我肯定会把自己弄得很漂亮。……（继续）
