社交直觉这个词听起来非常抽象、空洞和虚无。但它有一个明确、逻辑清晰的结构可以解释。

我们之前讲过，一个女人的社交直觉是五十个男人的总和。我个人通常会把社交直觉分成多个部分，RSD理论中也会如此划分。

社交直觉的核心是一种能力：**判断女人给出的兴趣是真是假的能力**。首先，你必须拥有这个能力。

有些男人会瞬间失去吸引力。他们是这样做的：女孩说了一句话，这个男人就以为女孩是在表达兴趣。他真以为是表达兴趣，然后开始自夸、自吹自擂，觉得自己很厉害。结果就失败了，女人对他变得非常冷淡。

女人看到一个男人时，会希望自己未来的儿子能拥有眼前这个男人身上的基因。如果这个男人连这点东西都判断不出来，她就不会希望自己的儿子也缺乏这种社交直觉。那么，这个男人就被淘汰了。

所以，第一点，也是最重要的一点，请记下来：**判断假兴趣的能力**非常重要。你需要知道什么兴趣是真的，什么兴趣是假的。

这叫社交直觉，她给出的是假兴趣。正确的玩法是推开这种假兴趣。

例如，女孩说：“一会要跟我回家吗？”你可以说：“大姐，这样不太好吧，我跟你才认识三分钟，你别这么饥渴好不好。”随便跟她开个玩笑，哈哈过去就好了。

所以，判断假兴趣非常关键。
