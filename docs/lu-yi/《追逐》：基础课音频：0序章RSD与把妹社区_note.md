# 《追逐》基础课 0️⃣：序章 - RSD与把妹社区

在本节课中，我们将要学习把妹社区的历史演变，以及正宗社交力学（RSD）是如何从中脱颖而出，成为全球领先的约会与自我提升体系的。了解这段背景，能帮助我们更清晰地认识RSD的地位、核心方法与独特价值。

## 把妹社区的起源与演变

上一节我们介绍了课程概述，本节中我们来看看把妹社区的背景。把妹文化大约在2007至2008年间引入中国，但其源头在美国。

最初，有一本名为《把妹达人》的书，揭示了美国把妹社区的种种内幕。中美两地把妹社区的演变过程基本相似。

一开始，把妹社区如同一个地下组织，仅在网络上建立了一些论坛。许多男性因为无法成功吸引女性，便聚集起来研究各种方法、技巧和话术，以满足自身需求与欲望。

在外界看来，这样的社区显得猥琐且低等。

然而，随着社会主流媒体（包括真人秀和电视节目）的不断曝光，把妹社区的内容逐渐变得可以公开讨论，并为更多人所了解。

## 社区核心人物与理论分野

在深入了解社区演变时，出现了需要厘清的问题。以下是社区发展过程中的几位关键人物：

*   **尼尔·史特劳斯**：《把妹达人》的作者，曾是《纽约时报》记者，采访过布兰妮等名人。为学习把妹，他找到了**弥南**。
*   **弥南**：被视为PUA（把妹达人）文化的始祖，他自创了一套“M3模型”。他在洛杉矶好莱坞租下一栋大别墅开设课程。
*   **泰勒**：RSD的核心创始人。当时他也是慕名前往弥南别墅的学员之一。

泰勒入住后，情况发生了变化。泰勒自己也招募了许多学员，并开创了一套名为“正宗社交力学”的课程。而弥南的课程名为“社交力学”。

一个有趣的现象是，泰勒“正宗社交力学”课程的学员人数迅速超过了弥南，并在各地与弥南竞争开课。

## RSD的核心理念与崛起

上一节我们认识了关键人物，本节中我们来看看核心理念的冲突。传统PUA理论与RSD理论最大的分歧在于学习方法。

弥南代表的传统PUA理论，要求学员背诵所有台词、惯例、话术和方法，并让学员直接套用。

这导致了一个问题：在好莱坞，一群学员使用完全相同的开场白去搭讪女孩。女孩回应：“这话我听过了。” 学员们顿时不知所措。

泰勒是第一个提出不同思路的人：我们能否从自身出发提升魅力，不使用任何背诵的惯例台词，从而成为能自然说出吸引女性话语的人。

RSD（正宗社交力学）正是从这个角度出发的。

时至今日（指2015年11月），在整个全球PUA或把妹达人社区中，只有泰勒带领的团队持续运营至今。弥南早已停止了相关的培训、方法研发与实战。

泰勒的公司已成为全球最大的把妹达人及约会建议与自我提升公司，即RSD（正宗社交力学）。

## RSD的实践验证与核心主张

这个公司坚持了十多年的实战传统，融合了对社交学、心理学等多学科的研究，并通过大量实践进行验证。

最有力的是，他们通过**现场偷拍的一手实战视频**，展示女性最真实的现场反应，以此验证所有方法的有效性。视频中也会出现学员，用以解释为何有效，以及每个阶段该如何操作。

RSD用清晰的逻辑结构，阐明了男女交往的本质。

由此可见，RSD是如何从把妹社区中逐渐衍生并确立的。由于泰勒发展迅猛，他成了弥南的宿敌。在《把妹达人》书中，泰勒被描绘成反派，书中用各种言辞贬低他。

但实际情况是，泰勒用其团队和长时间的实践证明了，他拥有全球最强的把妹公司。

如今，全球几乎所有著名的把妹达人都出自RSD门下，曾是泰勒的学员。泰勒及其团队做过许多看似疯狂的事情，例如：

*   组织学员拄着拐杖搭讪并最终带女孩回家。
*   让学员坐着轮椅进入夜店，与漂亮女孩接吻。
*   让六七十岁的老人在夜店与女孩接吻。
*   泰勒及其核心导师剃光头，以证明**长相并非吸引女性的关键**。

这些实践旨在说明，男性吸引女性依靠的是**男性的特质**，这正是RSD的核心方法，也是本门课程要传授的内容。
