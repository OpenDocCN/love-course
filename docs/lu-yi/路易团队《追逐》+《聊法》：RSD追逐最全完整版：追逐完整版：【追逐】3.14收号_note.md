### 概述：什么是C计划？

C计划是当A计划和B计划都无法实施时的备选方案。它的核心是收集女孩的联系方式（如微信或电话），以便在物流条件更合适时再次邀约，完成最终目标。但请记住，收号永远是下策的下策，你的首要目标永远是尝试直接推进关系。

### 第一节：收号的正确心态 🧠

上一节我们介绍了C计划的基本概念，本节中我们来看看执行C计划前必须建立的核心心态。

你的终极目的应该是和女孩发生关系。微信和电话号只是连接两次见面的桥梁，是备选方案里的备选方案。永远不要以收号为最终目的。

有趣的是，当你努力执行A或B计划时，最好不要过早收号。因为一旦女孩在进入卧室前拿到了你的联系方式，她和你回家的动力就会减弱。她会想：“我现在有选择了，可以不去，反正之后还能再见。”如果你没有给她号码，她的想法则是：“如果我不去，可能就再也见不到他了。”

**核心心态公式：**
```
首要目标 = 推进到最后一步关系
收号 = if (物流不允许) then 备用桥梁 else 避免过早执行
```

收号也不应成为你认同感的来源。不要因为收到一个号码就沾沾自喜，向朋友炫耀。号码只是一座桥，连接现在即将分别的两个人，让双方有再次见面的机会，仅此而已。

你要尝试一口气完成从认识到发生关系的整个过程。但如果物流不允许，就把这个过程分成两部分：第一次见面（收号），然后通过号码这座桥梁，进行第二次见面。

### 第二节：收号的最佳时机与策略 ⏰

在理解了正确心态后，我们来看看在什么场合、什么时间点执行C计划最有效。

最常见的收号场景发生在夜店或聚会的前半段。此时，女孩的“购买度”或“下载进度条”还处于很低的位置。她刚到场，情绪还在积累，通常没有准备好和任何人发生关系。

以下是此时执行C计划的策略：

*   **领先起跑线**：在夜店前半段，你可以广泛社交，收取大量号码。这能帮你建立自信和丰富的心态，体现你的性价值（清晰的意图、控制力、流动感和自我娱乐）。
*   **重点突破**：到了后半段，当女孩的情绪和“购买度”升高时，你再重点与某个女孩互动，尝试执行A或B计划。
*   **备用方案**：如果当晚A、B计划都失败了，你手上还有前半段收集的号码。你可以联系她们，看看是否有女孩当晚可以搞定，不让之前的努力白费。

即使暂时带不走女孩，至少可以收到号码，保留未来的机会。在夜场前半段，大家刚进场，先收号，之后还可以继续聊。

有时在夜店后半段，你遇到一个很喜欢的女孩，但她的物流条件极差（例如室友醉倒、男友出现等）。这时你要明白，女孩的漂亮程度和她的物流没有关系。你的心态应该是：“现场还有其他物流条件更好的漂亮姑娘。”对于这个女孩，收号即可。

**总结来说，大部分收号都发生在夜店的前半段，目的是建立联系储备，为后续创造机会。**

### 第三节：收号的五个核心步骤 🚀

现在，我们进入实战环节。以下是成功收号并确保后续邀约顺利的五个关键步骤。

#### 步骤一：确保意图清晰

你必须让女孩清楚知道你为什么要她的号码——因为你想再次见到她。你不是为了要号而要号，更不是一只祈求“糖果”或“骨头”的小狗。

**错误示范**：“我能加你微信吗？谢谢。”（语气不确定，目的模糊）
**正确示范**：“咱们加个微信吧。”（意图清晰，理所当然）

你的感觉应该是：“因为现在A计划和B计划都没法实施，所以我收号，是为了下次见面时再执行它们。”你要做一个没有什么可抱歉的男人，释放出清晰、坚定的意图。

对于初学者，要建立一个习惯：**如果你和一个女孩聊了两三分钟以上，在分开前，一定要把号码要到手。** 即使你觉得她不会给，也要去做。很多社交动态只有通过大量实践才能掌握。

#### 步骤二：确定下次见面的物流

既然收号将沟通一分为二，你就要尽量在当面时确定下次见面的细节，以最小化事后手机邀约的难度。

当你收号后，可以立即开启类似话题：
*   “我们之后什么时候再见？”
*   “今晚晚些时候我们再碰面？”
*   “周末一起出去喝杯咖啡？”

聊得越具体、越有现实感越好。例如：
```python
# 示例对话结构
if 收号成功:
    问：“明天下午一起喝咖啡？”
    答：“好啊。”
    跟进：“你明天下午具体几点有空？在XX地方见面方便吗？”
else:
    # 处理拒绝或模糊回应
```

如果女孩当天无法见面，就聊第二天、后天或周末。关键在于**尽快再次见面**，间隔时间越长，女孩放鸽子的可能性越大，你们之间的连接感也会消散。

#### 步骤三：扩大投资和赋予资格

这一步能极大降低女孩放鸽子的可能性。

*   **扩大投资**：在女孩同意邀约后，通过挑战性的口吻让她增加投入。
    *   *例句*：“你是真的想出来啊，还是就嘴上说说而已？不想的话千万别勉强自己哦。”（用开心、挑战的语气，而非缺乏安全感）
*   **赋予资格**：收号后表达你对她的好感和期待。
    *   *例句*：“说真的，我今天见到你特别开心，已经开始期待下次带你去我常去的那家很酷的咖啡厅了。”

#### 步骤四：过程要隐蔽

不要在收号时把手机晃来晃去，这会让女孩暴露在周围人的评判目光下，她可能会因为怕被朋友视为“随便”而拒绝你。侧过身，尽量背对女孩的朋友，低调地完成操作。

#### 步骤五：收号后不要立刻离开

收号后立刻跑掉，会传递出“你只在乎号码”的低价值信号。至少再随意聊上几十秒到一分钟，然后自然地告别离开。这显得你从容不迫，收号只是交流中一件很平常的小事。

### 第四节：号码的管理与后续跟进 📱

成功收号后，如何管理这些联系人并进行有效跟进，决定了C计划最终的成败。

**首先，养成整理号码的习惯。** 如果你向每个聊过的女孩要号，最终会积累大量联系人。很多人很难记住当晚认识的女孩具体是谁。建议使用微信分组功能，将当天聊得好或有可能再约的女孩拉到一个分组（例如命名为“A”或“Probably”）。

**如果忘记了女孩的名字，怎么办？** 直接、轻松地应对。
*   *例句*：“嘿，我叫什么名字你还记得吗？……其实我也忘了你叫啥了，哈哈。现在告诉我吧，你微信多少？”

**后续跟进的关键是“快”**。在女孩还记得你、对你有印象和投资感的时候，尽快发起邀约。等待时间越长，一切烟消云散的可能性就越大。

**在跟进时，可以提及当面聊过的细节**，这能唤起回忆，增强现实感。
*   *例句*：“嘿，我是昨晚在XX和你聊咖啡店的Hero。今天下午我突然有事，咖啡喝不了了，晚上你干嘛？”
