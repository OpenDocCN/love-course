# 《追逐》基础课 2.10：反应模式 🎯

在本节课中，我们将要学习一个在人际吸引中至关重要的概念——**反应模式**。理解并避免反应模式，是建立健康吸引力的关键一步。

## 什么是反应模式

上一节我们介绍了吸引力的基础，本节中我们来看看什么是反应模式。反应模式通常指，当你内心过分看重一个女孩，将她视为女神，并产生配不上她的感觉时，你会在她面前自然进入的一种状态。

这种状态就是反应模式。顾名思义，它指的是女孩的任何言行都会引发你非常激烈的反应。

以下是反应模式的一个具体例子：
*   假设女孩是你的同班同学，坐在你前面。她突然回头问你：“你有指甲刀吗？我想借一下。”
*   因为你内心视她为女神，你可能会立刻表现出过度热情：“我帮你借！这边这边，谁有指甲刀？”
*   这种过度迎合和激动的行为，就叫做反应模式。

## 反应模式的原理：RAS系统

要理解反应模式为何产生，我们需要了解一个概念。在RSD系统中，有一个名词叫**RAS**（网状激活系统）。这是一种神经生理系统，存在于每个人体内。

它的作用是让人将全部注意力集中在某一件对其而言**极具价值**的事物上。

以下是RAS作用的类比：
*   当你非常饥饿时，眼前突然出现一盘食物，你的目光会“啪”地一下被牢牢吸引。
*   同理，如果你认为一个女孩极其漂亮、极具价值，并感觉自己配不上她，你的RAS就会启动。
*   它会要求你把所有注意力都集中在这个女孩身上，并极度重视她。

## 反应模式的危害

在理解了原理后，我们来看看反应模式的具体危害。当你处于反应模式时，你会在与女孩的互动中，自然而然地通过言谈举止释放出“我配不上你”的信号。

这个信号是持续存在的，它会**完全扼杀你的吸引力**。女性希望感受到的是，一个男人不仅觉得自己配得上她，有时甚至能让她产生“我配不上他”的感觉。例如，一些自卑的女孩可能会对心仪的男性产生这种心理，这时男性想要吸引她会非常容易。

反之，如果你对女孩进入了反应模式，你的本能会驱使你将所有注意力放在她身上。女孩的一个眼神、一个动作或一句话，都会引起你强烈的情绪波动或反应。此时，你的吸引力便消失了。

## 如何陷入反应模式

那么，哪些行为会导致我们陷入反应模式呢？核心原因在于**过分在意**。

例如，你和朋友聊天时，不断谈论最近遇到的一个女孩：“我有个同事，她家境特别好，穿的衣服太可爱了，那小丝袜简直性感……我过两天要开始追她。”你在她背后与朋友反复讨论她，或者回家后不断想她的事情。

当你再次见到这个女孩时，你几乎必然会进入反应模式。因为你投入了过多的注意力。而这个女孩回家后，她可能根本不会想你，她想的是自己的发型、要追的剧、明天的约会。你们双方投入的“想念时间”完全不平衡：你想她两小时，她一分钟也没想你。在这种情况下，你几乎不可能成功吸引她。

## 反应模式的具体表现

以下是反应模式的一些具体行为表现，它们都属于需要警惕的误区：
*   **过度关注女孩的动向**：女孩看向哪个方向，你就立刻跟着看过去，并且动作迅速、紧张。
*   **过度重视女孩的言行**：把女孩说的每句话、每个动作都看得太重。
*   **情绪极易被牵动**：女孩的任何小事都能立刻引起你强烈的注意力或情绪波动。
*   **随时准备提供帮助**：时刻准备好立刻去帮助女孩做任何事情。
*   **抱有“女神”心态**：内心觉得自己配不上对方，将对方置于高不可攀的位置。

这些表现都源于我们之前课程中提到的“性价值”里的**理所当然感**的缺失。当你缺乏与女孩相处时的理所当然感，你自然会缺乏主导力，完全被对方主导。表面上你可能和她聊得很多，但那只是因为你一直在附和、一直在对她做出反应，这并无法真正吸引女性，只会降低你的吸引力。
