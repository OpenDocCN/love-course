## 概述：什么是C计划？

## 核心心态：收号不是目的

收号不应是你的终极目的，也不应成为你认同感的来源。**微信或电话号只是一座桥**，连接现在即将分别的两个人，让双方有再次见面的机会，仅此而已。你的目标始终是**从认识女孩到发生关系**。如果物流不允许，你才将这个连续过程分成“第一次见面收号”和“第二次见面完成”两个部分。

**备选流程：** 认识 → (C计划收号) → 桥接 → 再次见面 → (A/B计划) → 发生关系
