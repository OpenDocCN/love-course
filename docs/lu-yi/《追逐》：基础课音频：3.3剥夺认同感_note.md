## 什么是认同感？

## 什么是剥夺认同感？

理解了寻求认同感的弊端，一个自然的问题是：女性为什么会来追逐你？答案的关键在于**剥夺认同感**。

尤其对于漂亮的女性，她们身边通常围绕着无数男性，这些男性不断地**承认她的价值**，给予她认同。例如，她只需在朋友圈发一张自拍或一段普通的心灵鸡汤，就会有一群男性点赞、评论。这些男性认为，点赞和收到回复意味着自己有机会。

以下是剥夺认同感的核心操作方法：
