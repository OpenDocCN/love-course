# 《聊法》：第12节、聊天中如何有控制力 🧭

在本节课中，我们将要学习聊天中的“控制力”这一核心特质。我们将探讨控制力的含义、缺乏控制力的表现，以及如何在实际聊天中培养和体现这种特质，从而让你成为关系中的主导者。

## 理解控制力：谁是原因，谁是结果？

上一节我们提到了聊天中的负面反馈，其中涉及了控制力的概念。本节中，我们来看看如何具体培养和体现这种特质。

路易团队的追逐课程中，提出了男性魅力的四个核心点：**意图清晰**、**控制力**、**流动感**和**自我娱乐**。任何有魅力的男性身上都能找到这四个特质。

首先需要明确一个问题：在聊天中，是谁在主导谁？我们之前的课程强调过，在与女性聊天和交往的过程中，需要去主导对方。但仅仅知道这个道理是不够的，许多人甚至无法判断在一段对话中，究竟是谁在主导。

要判断或成为能够主导关系的男性，可以关联以下几个概念来理解：
*   **谁是原因，谁是结果？**
*   **谁在对谁做出反应？**
*   **谁更在意这段关系？**
*   **谁更害怕失去这段关系？**

## 缺乏控制力的表现 🚫

以下是缺乏控制力的一些常见表现，如果你在聊天中有类似行为，说明你的情绪和行为可能正被对方主导。

1.  **过分在意对方说的话**：将对方每句话都看得过重。
2.  **追求完美回复**：当对方说了某句话后，绞尽脑汁去想一个特别完美的回复。
3.  **因对方的负面反应而紧张**：当对方不理你、对你有不好情绪或反应差时，你感到特别紧张。
4.  **行为以引起注意为目的**：开玩笑是希望对方给你好的反馈、能笑；说对方坏话或打压对方，是希望对方能注意到你。
5.  **因对方的好反应而过度解读**：当对方对你一旦有好的反应（如礼貌性夸奖或试探），你就把持不住，感觉对方爱上你了。
6.  **刻意炫耀**：例如，“我要去和乙方签合同了，回头聊”、“我刚提了一辆宝石捷”、“我爸很厉害”、“我曾经如何”。**关键不在于不能分享成就，而在于你抱着什么目的**。很多人的想法是：`我希望说完这些，对方就能感觉到我的厉害并马上喜欢上我。`

**案例说明**：
一个男生打招呼后，女生回应“你好”。刚聊两句，男生就说：“刚来南城找个客户，签了份合同，顺便逛逛。你呢？” 之后很快以“OK有空聊”主动结束对话。几天后他再次找女生聊天时，女生已记不起他是谁。这个案例说明，生硬的炫耀和刻意的高价值离场，若缺乏真正的互动和带领，是无效的。

上述表现只是缺乏控制力的冰山一角。如果在聊天中你有类似表现，就说明：**女人是原因，你是结果**；你在对女人做出反应，且远多于她对你的反应；你的情绪被女人主导。在这段社交中，女人处于高位，是被追逐者；你处于低位，是追逐者。

## 判断练习：识别原因与结果 🧠

为了加深对控制力的理解，我们通过几个例子来判断谁是原因（主导者），谁是结果（反应者）。

以下是几个场景，请判断谁是原因，谁是结果：

1.  你讲了一件小事，女人哈哈大笑。谁是原因？
2.  你讲了一个好玩的事，女人没觉得有趣，这让你很不爽。谁是原因？
3.  女人展示前男友多厉害，你并不在意，这让她很不爽。谁是原因？
4.  你约女生，她被放鸽子，你在微信上破口大骂。谁是原因？
5.  女人没回消息，你跑到兄弟群焦急地问：“她为什么不回我？我做错了什么？” 谁是原因？
6.  女人说了一句难回复的话，你跑到群里问：“兄弟们，帮我想个牛逼回复！” 谁是原因？
7.  约会中男生牵女生的手，被甩开。男人紧张地问：“我做错了什么？” 谁是原因？如果男人不在意，当做没发生，他有没有成为结果？

这些例子旨在帮助你更清晰地认识控制力、原因与结果的关系。

## 如何获得控制力：两大核心方法 🔑

理解了问题所在后，我们来看看如何真正获得控制力。

### 方法一：保持低情绪化

**低情绪化**是指：不会因为女人的负面反馈或不好的言辞而感到特别沮丧、难过或不爽；也不会因为女人对你略微示好或感兴趣，就感到欣喜若狂。一个词概括：**宠辱不惊**。

*   **前期应用**：在聊天前期，低情绪化至关重要。它能让你与其他“挫男”区分开，因为挫男总是努力寻求女人认同，过分在意女人感受却不尊重自己。真正的酷男人面对负面反馈时不会在意，这能让女人快速感受到你的性价值（魅力）。
*   **中后期调整**：当女生开始被你吸引，对你进行情绪投资后，你需要**适当做出反应**，让她感受到你的在意。例如，她故意给你看前男友照片，你确实不在意，但可以**假装吃醋**来让她感觉你在意她。如果你真的吃醋、真的在意，那就失去了控制力。
*   **警惕控制欲**：很多男生在恋爱中，因为在意、吃醋和嫉妒，会产生很强的控制欲，总想控制对方和谁出去、给谁发信息等，这都属于没有控制力的表现。
*   **重要区分**：**不反应不等于不回应**。
    *   `不反应`：指**情绪**没有太大波动，不会因对方挑战而产生负面情绪。
    *   `不回应`：指**行为上**不说话或只回几个字。错误的不回应（如只回“嗯”、“哦”）会让女生觉得你很高冷奇怪，可能把对方“冻跑”。

**核心**：你该怎么聊还是怎么聊，但保持情绪稳定。

### 方法二：抛弃完美主义

为什么总有男生问“怎么打招呼能让女生喜欢我”？本质上，他**害怕失败**，害怕因开场白不够好导致聊天无法继续，害怕暴露不完美的一面。

*   **开场白的作用**：开场白的作用仅仅是**开启对话**，把你扔到和女生交流的场景中。除此之外，没有神奇效果。
*   **想象实验**：
    *   男人A有一句“全世界最牛逼的开场白”，能让任何女人立刻觉得他酷。但后续聊天一塌糊涂。女人会因为他一句开场白就喜欢他吗？
    *   男人B开场白只会说“嗨”，效果平平。但他能通过聊天慢慢主导、带领女人，引导对方进行情绪投资。女人会因为他只说“嗨”就不愿和他约会吗？
*   **女人的判断逻辑**：女人是看**行为线索**的。初期只要不犯大错，按照课程中的正确方法聊天，女人就不会讨厌你。她会通过后续互动来了解你。即使你说错话、有些小瑕疵，女人往往并不在意。
*   **接受不完美**：每个人都是普通人，都会犯错、有缺点。世界上不存在完美。害怕被拒绝、害怕说错话导致反应不好，会让你什么话都说不出来。**如果一个女人从不给你任何拒绝或挫折，那才是不正常的。**

## 总结 📝

本节课中，我们一起学习了聊天中“控制力”的核心内涵。

我们首先明确了控制力意味着成为关系的“原因”而非“结果”。接着，我们列举了缺乏控制力的种种表现，例如过分在意对方、追求完美回复、因对方反应而情绪波动等。

然后，我们探讨了获得控制力的两个核心方法：**保持低情绪化**（宠辱不惊，区分不反应与不回应）和**抛弃完美主义**（接受不完美的自己，不因害怕失败而不敢行动）。

希望本节内容能帮助你打消不必要的顾虑，以更平和、从容的心态去面对所有聊天对象以及过程中出现的挫折，真正掌握对话的主导权。
