# 阮琦·沟通训练课：第六课：让沟通为我所用_note

在本节课中，我们将要学习沟通的核心目的——如何让沟通真正为我们所用，帮助我们办成事、解决问题，而不仅仅是用于情绪宣泄或争一时口舌之快。

## 沟通的核心目的：办成事

上一节我们探讨了沟通的多种面向，本节中我们来看看沟通最根本的用途。学习沟通最有用的目的，是让我们能够办成事。我们想做一件事情，别人去做可能做不成，但换了你去做，通过有效的沟通，别人愿意配合你，最终把事情完成。这才是最有价值的能力。

沟通在其中起了相当重要的作用。但这并非意味着仅凭好口才就能解决一切问题。沟通有时也需要所谓的“实力”作为基础和保证。

**公式：成功办成事 ≈ 实力（基础） + 沟通能力（催化剂）**

例如，你是一名公司业务员，公司派你去谈一笔生意。生意能否谈成，一部分取决于你的沟通能力，另一部分则取决于公司的实力为你提供的担保。公司有实力，对方认可你，你才有机会发挥口才。如果公司没有实力，或者你失去了职位，那么单靠沟通去敲门谈合作，通常是行不通的。

因此，沟通就像软件，是帮助我们办成事的重要条件，但并非有了它就无所不能。

## 实力是沟通的基石

我们之前在讨论两性关系时也讲过类似观点。有些男孩认为只要自己能说会道，就能追到任何女孩。这种思维在沟通中，就等同于认为只要沟通能力过关，就能说服任何人，让他们去做任何事，无论是搭讪、邀约还是表白。

那些所谓的“PUA”技巧，片面地拔高了沟通（尤其是语言话术）的能力，声称可以让人无所不能。但科学的看法是，仅靠招数和套路，用处非常有限。

常见的实力保证包括所谓的世俗标准：**有财、有钱、有地位**。拥有这些，再配合良好的沟通能力，就如虎添翼。如果没有这些，仅靠沟通能力，成就将非常有限。

另一种实力是**关系**。如果你没有显赫的财富或地位，想追求喜欢的人，靠的也不仅仅是油嘴滑舌，而是通过良好的相处，建立起一层层可靠的关系。最后再配合沟通能力，才有可能改变对方的行为，让对方接受你。

关系也是一种实力，它和金钱、地位一样，需要投入、培养和建立。如果你和交往对象建立了较好的关系，再去影响他的行为就会容易很多。当然，有了关系还需要配合良好的沟通能力。

生活中的很多事情也是如此。一个家族没落了，但关系网还在，聪明的人懂得利用这些关系东山再起。反之，如果不会利用关系，可能就永远没落下去。另一方面，如果一个草根出身的人，没有任何背景，即使他再聪明、再英俊，想快速崛起也极为困难，概率极低。

我们要理解，沟通能力（俗称能言善辩、会说话）是一种**软实力**。它要想把事情办成，需要依附在相应的**硬实力**基础上，无论是金钱、势力，还是人脉关系。结合起来，才能办成事。

这才是沟通最有用的地方，而不仅仅是怼人后自己感到痛快。高兴痛快之后，你的生活并没有发生根本性变化。我们学习有用的学问和能力，最终是为了让生活发生变化。

## 将沟通应用于解决问题

我们的课程已进入后半段，现在要树立更明确的方向：学沟通是为了什么？有了这个方向，我们可以进行更多针对性的自我训练。

说到用沟通来办成事或解决问题，我从自己的生活经验说起。我的生活相对简单，不像销售需要处理复杂的客户关系。我只能从身边的小事举例。

前两天，我带自己去医院做检查。前一天交了钱，医生让我约时间检查。后来我因事离京两个多月，回来后想起此事，下午三点给医院医疗部打电话，询问当天可否预约。对方说可以，并告知五点下班。我驱车一小时后赶到检查部门，却发现他们四点就已下班（虽然规定是五点）。我致电最初接电话的部门，护士表示她只负责告知时间，检查部门提前下班她也没办法，让我改天再来。

这种情况让人非常窝火。你找不到具体的人吵架，护士说她不负责，检查部门的人已经走了。

这时，沟通能力开始起作用。我回到接电话的医疗部，没有大吵大闹，而是说：“我三点十分打过电话（手机有记录），请问是哪位接的？我想找到这个人。为什么告诉我五点前可以来，我来了却无法办理？我想请这个人给我一个解释。”

我把事情落实到了具体的责任人。接待我的是护士长，我向他说明了情况，并表示对结果非常不满意。最后，护士长带着我到屋里，给各部门医生打电话，最终帮我成功预约，事情得以解决。

事后我思考，这个解决方式很重要：
1.  虽然生气，但我的目标不是撒气，而是不让这趟白跑。因此，态度上应该心平气和。
2.  按理说没人关心你的利益，但总需要有人为此负责。最可能负责的就是接电话的那个人，而不是习惯性提前下班的检查部门人员。
3.  找到这个人，让他意识到需要为自己的不严谨付出一点代价（无论多小）。为了自身利益不受损，他就会调动关系去帮忙解决问题。

在这个过程中，我不需要吵闹或指责，只需要找到相关人士，说明情况并表明要追究那个人的责任即可。后来护士长可能出于维护团队记录或帮助同事的考虑，妥善解决了问题。

这是我生活中不多见的、需要通过沟通解决问题的场景。由此我想，同学们在各行各业工作，学完沟通课后，应该有意识地思考：生活中是否遇到过事情卡住的情况？这种“卡住”不同于被人怼了心里憋屈，而是你想达成的目标无法完成。

这时，我们应该想一想，如何通过“行动加语言”的组合来解决问题。这才是沟通最有用的地方。

我记得以前在沟通课上讲过另一个例子：我在免税店买了一个旅行箱，回酒店一小时后发现箱子流出粘性胶水。我去换货，店员以“商品离店概不负责”的规定拒绝。后来我与经理沟通了一个小时，层层上报，最终成功换货。

很多时候，我们的环境有一个特点，叫“按闹分配”——似乎只有大吵大闹，问题才可能解决。我们从小受到一种扭曲的教育，好像讲道理没用，只有撒泼打滚才能维护利益。

但这其实是不好的。它让大家逐渐不重视良好的沟通能力，认为遇事只要胡闹、不要脸就行。然而，并不是所有人都能不要脸。那些要脸的人，利益受损时往往只好忍气吞声，因为他们看不到通过沟通成功解决问题的案例，认为只有撒泼打滚才是唯一选项。

这是不对的。我们应该都是要脸的人。从此以后，在生活中遇到问题或难题，就尝试开动脑筋，用沟通去解决，看看能否达成目标。当然，这也需要良好心态：达到了，我开心，维护了利益，提高了能力；达不到，这也是生活常态，并非事事都能如愿。我可以欣然接受。

这就像搭讪一样。搭讪是对沟通能力最立竿见影、也最残酷的检验。你在街上看到一个漂亮女孩，与她毫无交集，你走上前通过沟通，让她加了你的微信。在这几分钟内，你改变了一个陌生人的行为，满足了你的期望。这就是沟通能力的实际体现。

当然，这事也不是百分之百能成。有时也会失败、碰壁。但良好的沟通能力意味着，失败不会影响你的心态和情绪，你还能从容地尝试下一次。这就是沟通的本质：它不能解决一切，但可以让我们达成相当一部分目标；对于那些未能达成的，良好的沟通也不会破坏人际关系或影响我们的心态。

**标准：良好的沟通能力 ≠ 100% 成功，而是 = 能成功一部分 + 不成功时不破坏关系/形象。**

因此，撒泼打滚不是我推荐的方式。

## 经典案例：杜月生的沟通智慧

由此，我想到以前讲过的一个案例——杜月生的故事。

杜月生在发迹前，是上海黑帮大亨黄金荣家的一个小喽啰，一直不起眼。一次，黄金荣家发生盗窃案，一个手下偷了一袋大烟土跑了。当时家里没有管事的，只有黄金荣的姨太太和杜月生。姨太太便把找回烟土的任务交给了杜月生，并给了他一把枪。这是杜月生出头的机会。

杜月生在租界转悠，傍晚时远远看见一个黄包车过来，车夫拉着一个人，那人正是偷烟土的，脚下还有一个大麻袋。杜月生找到目标后，说了一番话，偷烟土的人便乖乖让车夫把车拉回了黄府，赃物得以归还。

杜月生说了什么？这段话充分体现了沟通能力。试想，如果你是杜月生，拿着枪，看见车夫拉着小偷过来，你会怎么说？也许会说“不许动”或“举起手来”。语言不受限制，但如何组织一段最合适的话，让听者乖乖配合，不反抗、不作对，最终按你期望的方式行动，达成你的目标？这是一种特别漂亮、厉害的能力。有这种能力的人，才能当真正的老大。

杜月生的话分两段：
1.  **先稳住车夫（司机）**：他举枪让对方看到有武器，然后对黄包车夫说：“我知道这事跟你没关系。你只要把车拉到黄府，赏你20块大洋。” 先撇清他的责任，并给予利益承诺，避免车夫因害怕或觉得倒霉而添乱。
2.  **再稳住小偷**：他对车上的人说：“想发横财你是不可能了，你已经跑不了了。但是命你能保住，黄家从来没在家里杀过人。我让你回黄府，你的命是能保住的。” 抓住了对方最关心的生存问题，消除了他鱼死网破的念头。

两段话，分别针对两人的核心关切（车夫：利益与责任；小偷：生命安全），用最精炼的语言稳住了双方，顺利将人和赃物押送回去。此事之后，姨太太非常信任他，事事交办，杜月生由此崛起。

由此可见，沟通能力最重要的不是跟人互怼，而是在明确目标后，如何组织语言，让需要合作的人最容易、最准确地按照你的意愿去行动。

## 如何锻炼沟通解决问题的能力

意识到这点后，我们在生活中不要怕困难。遇到事情，有意识地去锻炼自己面对、处理、解决。就算不成，也保持良好心态。这样才能真正提高沟通能力和为人处世的能力。

我又想起一个例子：古罗马的凯撒大帝。在他还未成为独裁者时，有一次需要说服拥护他的选民，选举他信任的一个人担任年度执政官。这是一个非常明确的任务：我希望你们选张三。

但在集会上，他最后说的话非常精妙。大意是：“我希望通过各位的努力，让张三成为我们的首席执政官。” 这句话非常准确、有力。

他不是说“我希望大家都投票给张三”。因为让张三当选，不止“投票给他”这一种方式。还包括说服朋友投票给他，或者分散其他竞争者的票数等。“通过各位的努力”这个说法，涵盖了所有可能的行为路径，比单纯要求“投票”更周全、更多样。

这就是一种能力：你想让别人做什么来达成你的目标？别人到底要做什么才能达成你的目标？你自己是否想清楚、想明白、想周全了？这种思考能力，最后体现在沟通表达上。

很多人不想这些事情。他们想让别人帮忙时，想法特别肤浅表面。对方更不会操心（因为不是他的事）。对方愿意配合就不错了，但具体怎么做，对方不会深入思考。所以，如果你只是说“帮我个忙，你往东”，他就往东，不会多想“我往东是为了什么”。只有你自己才知道，最终想达成什么目标。

聪明人会想明白：为了让这个人帮我达成目标，除了让他“往东”、“往西”之外，有没有一个更准确、更圆满的说法？这些都是沟通能力。

如何锻炼？如果单纯落实到表达能力上，就变成了文字游戏。但如果我们落实到生活场景中，落实到解决一个又一个具体问题上，就是一种非常好的锻炼方式。

那些创业者，如马云等，他们常说创业就是不断解决问题。这可以理解，因为创业者面对的困难和问题最多样、最不确定。我们大部分人的生活是固定的工作、程序和人，相对单调。而那些开拓事业的人，会遇到一个又一个问题，这时沟通能力就非常重要。结合现有实力与沟通能力，解决一个问题，实力（筹码）增加，沟通能力也得到锻炼，从而能解决更大、更多的问题。人生就是这样一个有成就感、充满挑战的过程。

对于我们学习者来说，应该树立一个意识：把提高沟通能力落实到具体的问题和场景中。我们的生活通常比较简单，一旦遇到问题，不要逃避，应该怀着“终于有机会锻炼自己”的心态去尝试。就算不成，也保持平和心态，不要撕破脸或自我怀疑。在这样的过程中不断提高自己。

## 沟通在生活中的综合应用

目前来说，锻炼沟通能力最直接的场景之一是恋爱、追女孩。因为这正是如何通过说话，让对方接纳我们、产生更多交流的场景。所以追女孩是我们开设各种课程的入口和契机。

但我们也要认识到，追女孩只是沟通应用的一个场景。生活中还有更大、更丰富的部分：工作、生活、处理各种问题。

最后我再举一个医院的例子（最近去医院比较多）。那天我带母亲去医院，医生让她做完检查后，直接回来，不用挂号，等医生看完当前病人后下一个就进去。我们检查完回来，在诊室外等候。当时里面有一个病人，医生让我们等。等的时候，突然来了一个中年男子，也拿着片子。医生让他在外面等。但他直接堵在了门口。

我意识到，如果等里面病人出来，他可能会顺势进去，到时我再叫住他说“我们先来的”，场面会尴尬。所以我想文明地先沟通一下。我对他说：“你是在这等进去吧？不好意思，我们是先来的，而且我母亲80多岁了，站着不合适，一会儿应该让我们先进去。” 我语气和缓地跟他讲理。

结果碰到一个特别自私的人，他说：“我就一句话，一句话就说完。”

这时，同学们你们会怎么办？我觉得，我跟他沟通的目的是避免门一开，两人同时往里挤的尴尬。但现在这个目的达不成了。所以，与这个人的沟通就此结束。后来我没理他，扭头不再说话。

等门再开时，我和他都站在门口。我把头伸进去问医生：“医生，请问下一个病人该是谁了？” 医生念了我母亲的名字。我便把门打开，扶母亲慢慢走进来坐下。那个人就在外面等着了。

这也是沟通能力的综合应用。有时你需要跟人讲道理，有时跟这人讲道理没用（比如论证“谁先来”、“你一句话能不能说完”没有意义），沟通就可以结束。但你的目的不是让他让你进去，而是你们能先进去看病。所以可以换个对象（医生）、换个场景（直接询问医生顺序），让医生决定谁先进去。

生活中时时刻刻都有这样的小场景。虽然我们普通老百姓没有那么多惊天动地的大事，但如果你善于观察生活、勤于思考、勇于锻炼，其实还是有不少机会的。

## 总结

本节课中我们一起学习了如何“让沟通为我所用”。大家现在应该明白了，我们要让沟通能力与解决生活中的实际问题结合起来。这样既能维护我们的利益，也能更好地提高我们的沟通能力。这比仅仅耍嘴皮子怼人两句，或哄女孩开心一下，要有用得多。

**核心要点总结：**
1.  **沟通的核心目的是办成事**，而不仅仅是情绪宣泄。
2.  沟通是**软实力**，需要依附于**硬实力**（如金钱、地位、关系）才能发挥最大效用。
3.  将沟通应用于**解决问题**，是最高效的学习和锻炼方式。
4.  面对问题，目标导向，**心平气和**地寻找关键责任人，往往比发泄情绪更有效。
5.  沟通不是万能的，要有“**尝试解决，坦然接受结果**”的良好心态。
6.  从**生活小场景**入手，不断观察、思考和练习，是提升沟通能力的有效途径。

通过有意识地在实际生活中运用和锤炼沟通技巧，我们才能真正掌握这门能力，让它为我们的生活和工作目标服务。
