# 阮琦·沟通训练课：第七课：沟通的剑与盾 🗡️🛡️

在本节课中，我们将要学习沟通中的“进攻”与“防守”策略。我们将通过一个真实案例，深入探讨当沟通场景中存在“裁判”（如摄像机背后的观众）时，如何调整策略以保护自己或达成目标。核心在于理解沟通不仅是内容的交锋，更是规则与舞台的博弈。

## 从一场真实对峙说起

上一节我们探讨了沟通的基本心态，本节中我们来看看一个极具冲突性的真实案例：罗永浩与方舟子的线下对峙。这个案例生动地展示了当沟通被置于公众视野下时，规则会发生何种根本性变化。

几年前，罗永浩因质疑方舟子某基金款项去向，携带摄像团队，在方舟子经常出现的写字楼下截住了他及其律师，要求当面对质。这对于方舟子而言是极其不利的局面：其一，事件本身他可能不占理；其二，其面对面口才与反应力远不及以口才著称的罗永浩。

面对此情境，方舟子的反应是：一言不发，扭头走向电梯，甚至在等电梯时低头玩手机，完全无视罗永浩的质问。这种“拒绝沟通”的姿态，就是我们分析沟通策略的起点。

## 第一条核心规则：有观众在场时，“不沟通”也是一种沟通

如果沟通是纯粹的两人私下交锋，选择不沟通或许只是一种回避。但当存在第三方观众（如摄像机背后的万千网民）时，情况就完全不同了。

**核心规则**：当有摄像机（即观众）对着你时，你的沉默、逃避同样会向观众传递明确信息——你不敢或不想沟通。这本身就是在沟通，并且通常是示弱的表现，会让你在观众心中的评价上“先输一步”。

方舟子采用类似“无可奉告”的方式应对，在观众眼中立刻显得“作贼心虚”，而罗永浩则占据了“义正辞严”的道德高地。方舟子的律师后续与罗永浩的交锋也完全处于下风。整个事件凸显了在公众注视下，错误应对策略带来的全面被动。

## 沟通的战场与规则：当裁判出现时

那么，当沟通的战场从私人领域扩展到公共领域，规则究竟发生了哪些变化？关键在于“裁判”的出现。

摄像机意味着观众，而观众扮演了裁判的角色。他们评判的不仅是**谁更有理**（事实），更是**谁讲理的方式更胜一筹**（表达与气势）。在公众舆论场，后者往往比前者更能影响裁判的“判决”。

这与法庭辩论有相似之处，但也有巨大不同。法庭为了尽可能屏蔽口才与情感的干扰，回归事实与理性，设立了关键规则：

*   **法庭规则**：控辩双方不直接争吵，而是分别向法官和陪审团（裁判）陈述。
*   **罗伯特议事规则**：在讨论分歧时，意见相左的双方不应直接试图说服对方，而应面向一个中立的仲裁方陈述己见。

这些规则的核心智慧在于：**将“人对人”的肉搏，转化为“人对规则/裁判”的陈述**。这为我们解决类似方舟子的困境提供了直接启发。

## 正确的防守策略：将“剑”化为“盾”

如果你是方舟子，面对扛着摄像机、口才占优的对手上门质问，正确的应对策略是什么？以下是关键步骤：

**首先，必须正面应对，不能逃避。** 只要出现在镜头前，就等于进入了竞技场，沉默即等于认输。

**其次，改变对话对象，利用规则保护自己。** 这是最核心的防守技巧。不要与对手进行面对面的纠缠式辩论，而应将摄像机后的观众作为你的陈述对象。

具体而言，你可以向对手（更是向镜头前的观众）提出合理的沟通条件：

> “罗永浩，如果你想讨论这件事，我们有两种选择。第一，我们关掉摄像机，私下找地方谈。第二，如果你坚持要公开讨论，那么镜头后的观众就是这场讨论的裁判。我们应该遵循文明的讨论规则，像法庭辩论一样，我们各自对着镜头（裁判）陈述自己的观点，而不是互相争吵。同时，鉴于你是突然拦截，我只能给出20分钟时间对着镜头把事情说清楚。”

提出这样的条件，有几点好处：
1.  **合情合理**：援引了公认的文明讨论规则（如法庭规则），占据理性高地。
2.  **分化观众**：即使对手不同意，也至少能赢得部分观众的认同与同情。
3.  **掌握主动**：为自己设定了边界（如时间限制），避免了无休止的纠缠。
4.  **削弱对手优势**：将对方擅长的“面对面快速交锋”，转化为更依赖逻辑准备的“分别陈述”，拉平了双方在“口才体力”上的差距。

这个技巧类似于“人质谈判”中的策略：谈判专家不会直接与歹徒讨价还价（“放两个人质，就给食物”），而是将条件建立在客观制约上（“我需要向我的上级证明我正在与能做主的人对话，如果你能释放一两个人质来证明你的主导权，我就更容易说服他们提供援助”）。这样提出的条件更容易被接受。

## 沟通的根本目的：是“争胜”还是“办事”？

通过上述案例，我们深入探讨了在公开冲突中如何防守。但沟通的“剑与盾”还有更深层的含义，这关乎我们沟通的根本目的。

**很多时候，我们误以为沟通是为了“说服”对方。** 我们潜意识里将对方视为“理性人”，认为分歧仅仅源于对方“对事实了解不全”。只要我们把事实讲清楚，对方就应该认同我们。

**但事实往往相反。** 人与人的分歧，深层原因在于**价值判断**的不同。而价值判断源于各自的生活经历和情感体验，难以通过简单的事实陈述改变。我们总是倾向于选取符合自身价值判断的事实来构建认知。因此，试图通过“讲道理”来说服对方，在多数一对一的情况下是无效的，反而容易导致关系破裂。

那么，有效的沟通是为了什么？

*   **将沟通作为“剑”**：目的是展示自己、压倒对手、赢得观众。这常见于公开辩论、网络论战，其成果是获得认同与影响力（如网红靠此变现）。**公式**：沟通（剑） → 展示口才/逻辑 → 赢得观众支持 → 获得影响力/利益。
*   **将沟通作为“盾”**：目的是在存在分歧的情况下，保护关系、促成合作、达成实际目标。这常见于商业合作、政治协商、日常协调。其核心不是争对错，而是寻求妥协与交易。**公式**：沟通（盾） → 承认分歧 → 寻求利益交换/妥协 → 促成合作/解决问题。

罗永浩作为网红，擅长将沟通作为“剑”，在粉丝面前犀利地碾压对手。但当他经营企业，需要与供应链上下游合作时，这种“剑”的模式就失效了——他把合作伙伴说得哑口无言，但对方可能就不给他供货了。而像雷军这样的企业家，更擅长将沟通作为“盾”，与价值观可能不同的人达成妥协与合作，最终推动事情办成。

## 总结

本节课中，我们一起学习了沟通中“剑”与“盾”的双重属性。

1.  **当沟通存在“观众”（裁判）时**，规则改变。不可逃避，而应利用规则。核心防守策略是：**将“人对人”的进攻，转化为“人对裁判”的陈述**，通过提出合理程序（如援引法庭规则、设定边界）来保护自己，削弱对手的锋芒。
2.  **认清沟通的根本目的**。若为展示与争胜（如公开辩论），可善用沟通的“剑”。若为解决问题与促成合作（如商务、人际），则应视沟通为“盾”。其核心在于**尊重价值判断的差异**，绕过无谓的“说服”，转向务实的**妥协与交易**。

沟通的艺术，在于审时度势，明确目标，灵活运用“剑”的锋芒与“盾”的守护，从而在复杂的互动中达成所愿。
