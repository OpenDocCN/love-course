# 贤师傅《进阶课程班》：025.展示面第五节：2022最新网红修图方式_ev_note

在本节课中，我们将要学习如何使用一款名为“醒图”的手机软件，高效地完成照片的后期处理。课程将分为自拍和他拍两个主要场景，详细讲解从基础调节到精细美化的完整修图流程，帮助你掌握微调技巧，让照片在保持自然的同时提升质感。

## 一、 修图软件选择 🛠️

上一节我们介绍了修图前的准备工作，本节中我们来看看应该选择什么工具。在过去，手机修图可能需要使用多个软件，流程繁琐且容易损失画质。如今，我们推荐使用“醒图”这款软件，它集成了修容、调节、加滤镜等全部功能，操作便捷且能保证画质。

## 二、 修图前的分类 📂

在开始具体操作前，我们需要明确修图的照片类型。照片主要分为两种：

*   **自拍**：自己拿起手机对着镜头拍摄。
*   **他拍**：由他人为你拍摄。

我们将分别以这两种类型为例进行讲解。

## 三、 通用第一步：基础调节 ⚙️

无论是自拍还是他拍，修图的第一步都应该是进行基础调节，而不是直接修脸。先调整好整体画面，能为后续细节处理打下良好基础。

1.  **导入照片**：在醒图中导入需要处理的图片。
2.  **构图裁剪**：点击“调节”中的“构图”功能。如果背景杂乱或人物太小，可以尝试调整比例或进行裁剪。原图比例合适则可跳过。
3.  **尝试智能优化**：点击“智能优化”，让系统自动调整。如果效果满意，可以在此基础上微调；如果不满意，则进行手动调节。
4.  **手动调节参数**：以下是核心调节参数的作用与调整建议：
    *   **亮度**：调整画面明暗。根据原图情况适量增加或减少。
    *   **对比度**：影响人物与背景的分离度。适当增加（通常10-30）可以让轮廓更清晰，但不宜过多。
    *   **饱和度**：控制色彩鲜艳程度。一般微调或不调，避免人物肤色失真。
    *   **HSL（重点）**：这是最常用的功能之一，可以针对画面中的特定颜色进行单独调整。例如：
        *   调整“肤色”的**色相**向冷色调拉，可以营造冷白皮效果。
        *   调整“红色”的**饱和度**，可以增加唇色气色。
        *   降低背景中杂乱颜色（如“黄色”）的**饱和度**，可以让画面更干净。
        *   公式表示：`HSL(色相, 饱和度, 明度) -> 针对特定颜色通道的调整`
    *   **锐度**：增加画面清晰度。通常固定增加30左右。
    *   **结构**：影响细节的虚实感。适当增加（如20）能让皮肤纹理更真实自然。
    *   **高光**：调整画面最亮部分。适当增加可以让人脸更突出。
    *   **色温**：控制画面冷暖色调。向冷色调拉（负值）显得通透，向暖色调拉（正值）显得温馨。
    *   **色调**：控制画面偏紫或偏黄。一般不动，如需调整幅度宜小。

**调节的核心原则是：每调整一个参数后，都使用对比按钮与原图对比，确保变化自然。**

## 四、 核心环节：人像美化 💁

完成基础调节后，我们进入人像美化环节。顺序必须是先调节后修脸，以避免调色后脸型产生变化。

### 1. 尝试自动美颜

首先可以点击“一键美化”或“自动美颜”，这是新手福音。系统会自动处理皮肤等问题。如果效果自然，可以在此基础上微调；如果效果夸张，则关闭并选择手动美化。

### 2. 面部重塑（懒人微调）

在“人像”中找到“面部重塑”功能，可以进行系统化的参数调整。以下是关键部位的调整建议：

*   **小头**：如果头身比不协调，可轻微调整（约20），注意避免背景扭曲。
*   **瘦脸**：根据脸型微调（10-15），切忌过度。
*   **发际线**：可调整额头高度，改善比例。
*   **眼睛**：可微调大小、眼高、眼距等（5-10），让眼睛更有神。
*   **鼻子**：重点调整“鼻翼”收窄（5-10），让鼻子更秀气。可轻微调整“鼻尖”增加立体感。
*   **嘴巴**：可调整大小、唇厚等。

**所有调整必须遵循“微调”原则，目标是改善而非换头。**

### 3. 瘦脸瘦身（手动精修）

“面部重塑”后，务必使用“瘦脸瘦身”工具进行手动精修，这是让效果自然的关键。

*   **放大修图**：双指放大图片，便于处理细节。
*   **局部推拉**：
    *   **鼻子**：将鼻梁推直，对称地调整鼻翼。
    *   **眼睛**：轻微推开眼角，使眼睛更有神。注意保持眼型弧度的自然流畅。
    *   **脸型与下巴**：推匀称脸部线条，如果脸圆可稍拉长下巴。
    *   **发际线与头发**：可向下拉头发修饰过高发际线，或向上拉头发制造蓬松感。**操作时务必注意背景线条，避免穿帮。**
    *   **身体**：全身照可适当收窄肩膀、胳膊、腰腹、腿型，让身材更匀称。

### 4. 手动美颜（细节处理）

自动美颜后，使用“手动美颜”进行精细化处理。

*   **磨皮**：选择“磨皮”笔刷，**必须放大图片**，只涂抹皮肤区域，避开五官轮廓线和头发，保留质感。
*   **祛痘**：使用“祛痘”功能点掉明显痘痘。
*   **祛法令纹/黑眼圈**：使用相应工具淡化。
*   **细节**：使用“细节”笔刷涂抹眼睛、眉毛、嘴唇轮廓、鼻梁（山根）和头发，能瞬间提升五官立体感和毛发质感。
    *   代码类比：`细节笔刷 -> 提高局部区域的锐度与清晰度`
*   **白牙**：如果牙齿偏黄，可用“白牙”笔刷涂抹。

### 5. 妆容与立体感

*   **五官立体**：使用“五官立体”功能，适当增加（如“全脸立体”），可以像打阴影一样让面部更立体，但不宜过重。
*   **美妆**：
    *   **眉毛**：在“美妆-眉毛”中选择与自身眉形契合的款式，将透明度调低（如50），达到“有妆胜无妆”的心机效果。
    *   **其他**：可根据需要尝试“卧蚕”、“美瞳”、“腮红”等，透明度务必调低，追求自然。

## 五、 画龙点睛：添加滤镜 🎨

所有美化步骤完成后，最后一步是添加滤镜。

*   **选择风格**：根据照片属性（如日常、复古、风景）选择滤镜分类。
*   **控制强度**：**任何滤镜都不要拉到100%**。通常30-40的强度即可，最多不超过50，以保证照片自然不夸张。
*   **常用推荐**：“质感”、“卡梅尔”、“胡桃木”、“克莱因”等都是不错的选择。

## 六、 他拍实例快速演示 🏞️

对于他拍（外景）照片，流程与自拍一致，但侧重点略有不同。

1.  **调节重点**：
    *   **HSL功能至关重要**：单独调整天空（蓝色）的饱和度使其更蓝，调整植物（绿色）的饱和度使其更绿，调整肤色（橙色）的色相和明度使其更白皙。
    *   **局部调整**：如果整体调色后人脸仍偏暗，使用“局部调整”功能，在脸部添加调整点，单独提高脸部的亮度、对比度，并降低色温（去黄）。
2.  **人像美化**：步骤与自拍相同，使用自动美颜结合手动瘦脸瘦身进行微调。
3.  **妆容与滤镜**：同样以自然为主，添加眉毛和低强度滤镜。

## 七、 补充技巧与总结 📝

*   **肤色笔**：在极暗光线下，如果肤色很黑，可使用“妆容笔”里的肤色笔（选最接近的两种颜色），**放大后**仔细涂抹皮肤区域，并大幅降低透明度（10-20），有“素颜霜”效果。
*   **最终检查**：导出前，再次检查整体效果，确保修饰自然，没有明显的修图痕迹。
