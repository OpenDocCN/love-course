# 贤师傅《进阶课程班》：024.展示面第四节：展示面拍摄， 构图技巧_ev_note

在本节课中，我们将要学习展示面拍摄的核心手法与构图技巧。理解这些原理，将帮助你不仅能够模仿优秀的展示面照片，更能自己创作出突出个人魅力的高质量照片。

## 概述：拍摄的核心目的

上一节我们探讨了如何寻找拍摄地点。本节中，我们来看看如何在实际拍摄中运用技巧。首先，必须明确一个核心原则：**展示面拍摄的唯一主体是人**。

所有拍摄都应围绕一个核心目的：**展示好看、有魅力的自己**。场景、道具、背景都只是服务于凸显“人”这个主体的工具。切忌让华丽的背景或杂乱的元素抢了风头。如果你想突出场景，就不要把自己拍进去。

## 第一部分：拍摄姿势与状态

理解了拍摄目的后，我们来看看如何通过姿势和状态来自然展现自己。核心原则是：**自然与舒适**。

以下是关于姿势的两个关键点：

1.  **拒绝僵硬摆拍，拥抱自然抓拍**：不要死记硬背特定姿势。最自然的状态往往来自抓拍。可以尝试在场景中随意走动、整理头发、低头思考或环顾四周，让拍摄者进行连续抓拍。
2.  **学习模特动态**：如果不知道如何动，可以搜索“平面模特街拍”或“模特抓拍姿势”等关键词进行观察。学习专业模特如何在镜头前自然切换姿态和角度，通常每1-2秒变换一个动作。

**技巧提示**：平时可以对着镜子练习，找到自己好看的角度和眼神，并记下来。

## 第二部分：需要避免的误区

在开始拍摄前，了解哪些事情不该做同样重要。以下是两个常见的拍摄误区：

1.  **避免刻意做作的行为**：例如，在图书馆刻意摆出看书的姿势会显得很不自然。更佳的选择是自然地倚靠书架思考，或手持书本走动时被抓拍。
2.  **避免错误的光线**：
    *   **不要逆光拍摄**：即光源（如太阳）在拍摄者正后方，会导致人物脸部过暗或完全看不清。
    *   **避免强光直射**：即使是补光灯，也不要直接正面打在脸上，会导致面部过曝、细节丢失。
    *   **最佳选择是侧光**：让光线从侧面或前侧方柔和地照过来，这样面部轮廓和质感最好。因此，尽量避免在正午阳光直射的室外拍摄。

## 第三部分：核心构图技巧

构图决定了照片的视觉重心和美感。对于展示面拍摄，掌握以下两种最常用、最易出片的构图法即可。

### 1. 三分构图法（九宫格构图）

当背景稍显杂乱，或画面中有明显线条（如柱子、地平线）时，推荐使用三分构图法。

**操作方法**：在相机设置中打开“网格线”（九宫格）。将人物主体放在网格线的四个交叉点附近，通常是左侧或右侧的交叉点。

**关键原则**：人物的视线方向应朝向画面更空旷的一侧，这样能形成视觉延伸感，让照片更有呼吸感和故事性。

```plaintext
示例：如果人物在右侧交叉点，那么他应该看向左侧的空白区域。
```

### 2. 中心构图法

当背景非常简洁、对称或均匀时（如一面纯色墙），可以将人物置于画面正中心。

**适用场景**：背景元素简单、左右对称，不会干扰主体。此时人物在中心能最大程度地吸引注意力。

### 全身照的构图要点

拍摄全身照时，有一个显高且突出人物的秘诀：

**脚贴底，头留空**：在构图时，让脚部贴近画面底部边缘，头部上方则留出一定的空间。即使拍摄时没做到，后期也可以通过裁剪调整。

**注意**：头部最好位于九宫格上方的横线或交叉点附近，不要顶满画面上缘。

## 第四部分：拍摄者的实用技巧

如果你作为拍摄者（或指导你的“工具人”朋友）进行拍摄，以下技巧能立刻提升成片效果：

以下是几个立竿见影的拍摄手法：

1.  **低角度拍摄**：拍摄站立全身照时，蹲下放低手机，从低处向上拍，能显著拉长腿部比例。
2.  **倾斜角度**：在低角度基础上，可以将手机稍微向内（朝向被拍者）倾斜，形成对角线构图，视觉上更显高且有动感。
3.  **主动沟通与示范**：如果你的“工具人”不太会拍，最有效的方法是：
    *   **先示范**：用手机为他拍一张符合构图要求的照片。
    *   **再讲解**：指着照片告诉他构图要点（如“我的头要在这个网格点”、“脚要贴底”）。
    *   **最后固定**：帮他摆好手机构图，让他保持不动，你回到位置后，让他连续抓拍即可。

记住，一次拍摄产出30-50张照片很正常，目的是从中筛选出最自然、构图最好的几张进行后期精修。

## 总结

本节课我们一起学习了展示面拍摄的核心心法与实践技巧。我们明确了**“人”是唯一主体**的原则，学会了通过**自然抓拍**来展现最佳状态，并规避了逆光、摆拍等常见误区。在构图上，我们掌握了应对杂乱背景的**三分构图法**和应对简洁背景的**中心构图法**，以及拍摄全身照的**“脚贴底，头留空”秘诀**。最后，我们还分享了如何指导拍摄者进行低角度、倾斜拍摄等实用技巧。

理解这些原理后，你将能更有效地指导他人为你拍照，或自己为他人拍摄，从而高效地产出高质量的展示面原片。下一节课，我们将进入关键环节——**修图**，学习如何让好的原片通过后期处理变得更具魅力。
