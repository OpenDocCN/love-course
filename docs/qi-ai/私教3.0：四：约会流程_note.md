## 约会的核心目的与“四次约会法则”

明确了问题所在，我们进入正题：约会。首先，你必须清楚约会的两个核心目的：

1.  **建立或放大吸引**。
2.  **升级关系**。

关系升级是循序渐进的流程：**牵手 → 接吻 → 发生关系（TD）**。请注意，**好感不等于吸引**，有了吸引才能进行肢体升级。

为了保证流程稳妥，我们采用“四次约会法则”。这是一个框架性设计，帮助你从认识到发生关系。每个约会都有其明确目的：

1.  **第一次约会**：目的是**建立好感**，而非急于肢体接触。
2.  **第二次约会**：目的是**推进关系并完成牵手**。
3.  **第三次约会**：目的是**推进关系并完成接吻**。
4.  **第四次约会**：目的是**进入私密空间并发生关系**。

> **重要提示**：此法则为灵活框架，非死板公式。需根据女生的反应活学活用。反应好可加速，反应一般则需放缓。核心原理在于满足女性在关系推进中所需的**安全感**。
