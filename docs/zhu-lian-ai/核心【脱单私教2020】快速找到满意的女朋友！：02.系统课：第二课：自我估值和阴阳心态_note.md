# 脱单私教2020：02. 自我估值与阴阳心态 🧠🔥

在本节课中，我们将要学习两个核心心态概念：**自我估值**与**阴阳心态**。理解并掌握它们，是建立强大内在自信、提升个人吸引力的基础。我们将通过清晰的步骤和实例，帮助你从内而外地改变。

## 什么是自我估值？

上一节我们介绍了课程的整体框架，本节中我们来看看**自我估值**。自我估值，简单来说，就是你如何看待自己价值的高低。这完全是你内心的标准，与外界的评价无关。

以搭讪为例，很多新手不敢行动，是因为脑海中充满了不自信的声音。这种情绪是潜意识的，没有逻辑可言。然而，一旦他鼓起勇气尝试，并获得了女生正常的、甚至积极的反馈，他的状态就会立刻改变。

他在接下来的互动中会变得更勇敢、更自信，成功率也随之上升。这并非因为他的技巧瞬间精进，而是因为**他内心的自我估值提升了**。他通过正反馈，改变了对自己的看法，认为自己“可能还不错”或“比普通男人更强”。

这种自我认知的转变至关重要。我们见过无数案例：起初非常胆怯的学员，在获得几次成功经验后，整个人变得积极主动，成功率也莫名提高。为什么？因为他的状态完全不同了。

**核心公式：自信状态 = 高自我估值**

一个猥琐的“嗨”和一个自信的“嗨”，在语气、姿态、眼神接触上截然不同。这本质上是由你的自我估值决定的。一个男人从“男孩”成长为“男人”的核心指标，就是拥有**核心自信**。拥有核心自信，你就有能力自然地向外输出价值，无需刻意寻找漂亮话。

你的自我估值，就是你如何看待自己。这比世界如何看待你重要得多。你只能为自己而活，活在自己的评价中，而非他人的眼光里。参考他人评价可以，但**自我评价必须高于外界评价**。这是获得真正自由的关键。

在无数实践中我们发现，**自我估值是后续一切吸引力手段的根基**。没有良好的自我估值，你的吸引力就像在沙滩上盖房子，不堪一击。

## 如何建立良好的自我估值？

理解了自我估值的重要性后，我们来看看如何具体地建立它。以下是四个核心步骤。

### 第一步：觉察不良的潜意识声音

首先，你需要学会听见内心那些阻止你的不良声音。心理学有句名言：“当你的潜意识没有进入你的意识时，那就是你的命运。”

这意味着，如果你没有意识到潜意识在控制你，它就会决定你的人生。有些潜意识是保护性的，但有些是**不良的、限制性的**，比如“我就是一个不自信的人”。

**行动指南：**
当你在街上看到心动的女生，内心出现“别去，你肯定会被拒绝”的声音时，你需要：
1.  有意识地**听见**这个声音。
2.  意识到：“看，那个不良的潜意识又来了。”
3.  点亮内心的“预警灯”，仅仅意识到它的存在，而不被它控制。

### 第二步：给自己积极的心理暗示

觉察到负面声音后，下一步是用积极的心理暗示替换它。不要认为这是“打鸡血”，科学证明积极暗示能切实改变生活。

**核心概念：信念塑造现实**
你的注意力投向哪里，你的世界就呈现什么样子。如果你不断暗示自己“我是优秀的，她们会喜欢我”，你的注意力就会自动寻找支持这个信念的证据（例如：她刚才看了我一眼）。找到的证据越多，你的信念就越强，形成良性循环。

反之，如果你总想着“她不喜欢我”，你就会找到无数“证据”来验证这个想法。

**积极心理暗示示例：**
*   “她一个人坐着，应该也挺无聊的。”
*   “我发现她偷偷看了我，应该对我有兴趣。”
*   “我系统学习过恋爱知识，三观正，懂得照顾人，一定很受欢迎。我只需要让她发现我的优点。”

### 第三步：持续建设更优秀的自己

仅有心理暗示是不够的，你必须在行为上真正投资自己，成为更优秀的人。这有两个关键点：

**1. 不要给自己设限**
不要用“我做不到”、“这不适合我”来限制自己。无论是搭讪不同类型的人，还是尝试新风格，都要勇敢去试。

**2. 持续投资与建设**
通过具体的行动，日积月累，让你从内到外真正改变。

**以下是建设自己的行为示例：**
*   舍得为自己花钱，投资在技能、体验上。
*   坚持健身，保持良好作息。
*   养成阅读习惯，不断提升认知。
*   在事业上保持进取心。
*   结交优秀的朋友。
*   用心打理外形，保持良好形象。

当你通过行动看到镜中越来越优秀的自己时，你的自我估值会自然提升。

### 第四步：用“未来的自己”战胜一切

最后，学会召唤“未来的自己”来帮助当下的你。每天晚上，在脑海中详细勾勒你未来成功的样子：衣着、身材、气质、伴侣……越细节越好。

**心理练习：视觉化未来**
1.  将想象的所有细节画面，整合成一张大的“海报”。
2.  让这张“海报”从头顶升起。
3.  想象它化成无数蕴含画面的雨滴，360度落下包围你。
4.  用手轻轻“触摸”这些雨滴，感受未来的能量。

**应用场景：**
当你觉得与心仪的女生差距较大时，不要胆怯。只要你正在执行第三步（持续建设自己），你就可以请出“未来的自己”与你联手。**当下的努力者 + 未来的成功者**，这个组合将给你无比的信心，让你觉得没有任何不匹配，因为辉煌的未来触手可及。

## 阴阳心态：幽灵与火焰 🕯️

在建立了坚实的自我估值基础后，我们再来补充一个高级心态工具：**阴阳心态**。这个概念有助于你在互动中保持平衡与影响力。

*   **阴 (幽灵)**：代表**不受影响**。像幽灵一样，外界的攻击、拒绝、抱怨穿过你，但不会对你造成伤害或激起负面情绪。这是**内在稳定**的基石。
*   **阳 (火焰)**：代表**影响他人**。像火焰一样，主动散发温暖、能量和积极情绪，去照亮和影响周围的人。

**核心应用：**
*   **当遭遇拒绝或负面反馈时**：启动“阴”的心态。成为幽灵，不被对方的情绪或行为影响，保持内心平静。
*   **当需要推进关系或提供价值时**：启动“阳”的心态。像火焰一样主动提供能量、幽默和温暖，让对方感受到你的魅力。

**示例：**
女生抱怨你时，先用“阴”的心态不被她的情绪带偏（火箭穿过幽灵），然后用“阳”的心态开个玩笑或转移话题，提供情绪价值，让氛围变好。

拥有控制力的基础就是“阴”——不被对方影响。同时，用“阳”去积极地影响局面。永远不要抱怨或散发负面情绪。

---

**本节课总结：**
我们一起学习了建立强大心态的两大支柱。首先，**自我估值**是你自信的根基，通过**觉察负面声音、给予积极暗示、持续行动建设、召唤未来自我**四个步骤可以稳步提升。其次，**阴阳心态**是你在互动中的高级心法，**像幽灵一样稳定内心，像火焰一样照亮他人**。请反复理解并实践这些概念，它们将从根本上改变你的状态和吸引力。
