## 投资测试维度解析

上一节我们介绍了女性双向分类系统的概念，本节中我们来看看其中一个重要的维度：投资测试。这个维度描述了女性在情感关系中，根据其过往的投资与回报经验，可能形成的不同心理模式。

那么这个属性是如何游离变化的呢？首先我们来看第一个维度：投资测试。

我举一个例子。比如一位女性投资者，她与一位男朋友相处了很长时间。她可能投入了很多，例如投入了青春、金钱以及大量精力。

她投资了许多经历。最后这位男性结束了关系，转而投向其他女性。

这位被留下的女性可能会感到非常伤心和难过。她可能会认为男性都不是好东西。

她可能会决定培养许多备胎。因为之前的择偶方式让她感到痛苦。

这种择偶方式让她做出了一次失败的选择，因此她会转而选择另一种择偶方式。这就像我们平时做事一样，如果用一种方法没有得到好结果，你肯定会尝试另一种方法。

女性也是如此。她投资了很多却没有得到好结果，于是她可能逐渐演变成一个倾向于测试男性的类型。她会学着培养备胎。

她会培养很多备胎。

以下是投资测试维度中，女性心态演变的关键步骤：
1.  **长期高额投资**：女性在一段关系中投入了大量时间、情感、金钱等资源。
2.  **投资失败**：关系以失败告终，且对方（被定义为“渣男”）是主动结束方。
3.  **认知重构**：女性形成“男性都不是好东西”的概括性负面认知。
4.  **策略转变**：为避免再次承受单一投资的巨大风险，转而采用“培养多个备胎”的策略。
5.  **行为固化**：新的择偶策略被学习和强化，逐渐成为其稳定的行为模式。

这个过程可以用一个简单的**公式**来概括其核心逻辑：
**重大情感投资失败 + 归因于男性群体 → 防御性择偶策略启动**
