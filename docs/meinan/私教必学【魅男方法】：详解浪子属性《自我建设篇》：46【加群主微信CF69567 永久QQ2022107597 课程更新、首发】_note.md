## 行为模式：过程导向 🔄

上一节我们介绍了课程主题，本节中我们来看看浪子行为模式的核心。浪子平时的行为模式是以过程为导向的。

以过程为导向意味着：很多时候我只在乎过程，不在乎结果。我做一件事，很多时候刚开始做的时候我会很开心、很兴奋、很众志成城、热情很高地去做。但是，做到一半就没心思、没兴趣继续做这件事了。

因此，浪子通常做事没有什么耐心。他们缺乏帝王属性中常见的坚忍。帝王做事往往坚持到底、做到最好。而浪子做事，很多时候依靠的是**三分钟热度**。

以下是关于行为模式的要点总结：
*   核心思维：**过程 > 结果**
*   动力特征：初始热情极高，但难以持久。
*   与帝王属性的对比：缺乏**坚忍**与**追求极致**的特质。
