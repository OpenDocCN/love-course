## 把握邀约时机 🎯

上一节我们介绍了如何建立吸引和安全感，本节中我们来看看如何将关系推进到线下见面阶段。

此时我们应该明确目标：以邀约为目的。前期所有关于吸引、人格属性建立以及安全感铺垫的工作均已完成后，直接提出邀约即可。

## 应对对方的“废测” 🛡️

当提出邀约时，对方可能会抛出“本以为你是正人君子”这类试探性话语，这通常被称为“废测”。

这时，我们需要用自己一贯的坚定态度来破解它。对方之所以进行废测，往往是因为对你有兴趣和好感。这个原理类似于：

**公式：兴趣 → 关注 → 测试**

以下是具体类比，帮助理解：

*   当你对某件商品感兴趣时，你才会询问它的细节。
*   例如，你想买空调，才会问“这台空调好不好？”。
*   你不会在购买电视机时，去询问空调的好坏。

意思是一样的。对方因为你而产生了兴趣，才会通过话语来测试你的反应和一致性。

## 课程总结 📝

本节课我们一起学习了关系进阶的关键一步：邀约。首先，在吸引与安全感建立完毕后，应果断以邀约为目标。其次，当对方进行废测时，应理解这是兴趣的体现，并用稳定的态度从容应对。核心在于识别窗口、把握时机，并坚持自己的框架。
