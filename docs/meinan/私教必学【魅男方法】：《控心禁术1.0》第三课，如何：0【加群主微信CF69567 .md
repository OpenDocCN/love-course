## 三、操作方法：如何展示“心疼”

刷心疼是一个技术活。在现实里，可能需要具备一定的表演成分。

## 四、心疼故事的分类与编撰

刷心疼的方式有两种：心理层面和生理层面。生理层面不太适合我们男生，男生刷心疼应尽量运用心理层面的。

**心理层面的心疼分两种：**

**A. 来自于自我（内心矛盾与疲惫感）**
比如你平时创业，每天做很多事情，在别人面前呈现一副很坚强强大的样子。但在夜深人静的时候，你才卸下自己的防备。这种冲击感对女生的杀伤力非常大。你要把心里的那种疲惫感描述出来。

**公式**：`现在的正向状态（果） + 过去的脆弱经历（因） = 立体真实的形象`

**案例模板（事业型）**：
“我创业很努力，生活很自律。为什么我要这么拼命赚钱？因为曾经我的女朋友为了钱离开了我。当时我和朋友合伙创业，朋友骗走了我的钱，我事业陷入低谷，欠了很多债。当时女朋友想买一个手机，几千块钱对当时的我来说非常多，没能给她买。后来她走的时候，我哭得像个孩子。正因为受过那样的伤，才塑造了今天拼命工作的我。每天工作16小时，但夜深人静卸下面具时，感觉非常疲惫，有时候真的很想有个人能抱抱我。”

**B. 来自于外界（被伤害后依然相信）**
1.  **重兄弟情谊**：你很相信朋友，但被朋友出卖，造成巨大损失。后来朋友有难，你依然相信他，却又被他欺骗。即使这样，你依然珍惜友情。
2.  **重感情**：你重感情，但曾经被感情伤得遍体鳞伤。即使这样，你依然相信感情。
3.  **亲情**：小时候父母忙于应酬没怎么管你，你经常被同学欺负却不知跟谁说，只能一个人躲在柜子里哭。这种经历造就了现在独立但内心渴望关怀的你。

**编撰核心**：一定要进入那种伤感的状态。先把自己感动，再让别人感动。回想整个事件，再把信息输送出去。

## 六、效果评估：心疼指数

刷心疼时，女生的反馈有心疼指数。心疼指数类似于兴趣指标，代表女生心理层面的反应。我们需要判断心动错觉达到了几，达到6就可以进行压迫表白。

**心疼指数判断标准：**

*   **指数 +1：关心**。例如：“你太累了，应该好好休息一下。”
*   **指数 +2：动容**。例如：“你真是个傻孩子，你怎么能这样？”
*   **指数 +3：产生保护欲**。例如：“我以后再也不会让你受到伤害了。” 或 “我以后会好好保护你的。”

**判断方法**：在聊天中敏感地察觉女生的反应。一条条往上加。如果心疼指数达到6到8，就可以直接压迫表白。

**重要原则**：刷了心疼之后，当天一定要连接压迫表白，不能耽搁，因为它具有时效性。心动错觉不会持续很久。
