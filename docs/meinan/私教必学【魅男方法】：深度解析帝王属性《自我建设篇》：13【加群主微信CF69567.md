## 以结果为导向的思维模式

上一节我们介绍了帝王属性的基本概念，本节中我们来看看其核心的思维模式。

帝王的一个思维模式，他们的一个做事风格就是以结果为导向的。很多时候我做一件事情，你有没有发现，浪子其实是以过程为导向的。他做很多事情，他对结果不负责。结果怎么样，他不管，他反正做得爽就可以了，这是浪子的一个思维模式。很多时候他不会去考虑结果。

那么帝王的话，很多时候他做事是会考虑结果的。而这个结果会怎么样，然后我们这个过程不应该这样。以结果为导向的去做事，是帝王的重要特征。

## 理性权衡与决策

理解了以结果为导向的出发点后，我们再来看看帝王是如何实现这一点的。

很多时候帝王是比较理性的。相对来说，他在平时做事的时候，是会去权衡利弊的。他是非常理性，并不感性。

以下是理性决策过程中的两个关键步骤：

1.  **明确目标结果**：在行动前，首先定义清晰、可衡量的预期结果。公式可以表示为：**目标结果 = 可衡量的成功标准**。
2.  **评估过程与利弊**：设计或选择行动路径时，系统性地分析每种选择的潜在收益与成本。这可以用一个简单的决策代码框架表示：
    ```python
    if 行动A的预期收益 > 行动A的潜在成本:
        考虑执行行动A
    else:
        重新评估或选择行动B
    ```
