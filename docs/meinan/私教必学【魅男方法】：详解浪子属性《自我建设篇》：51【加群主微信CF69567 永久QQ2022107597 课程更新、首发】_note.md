# 私教必学【魅男方法】详解浪子属性《自我建设篇》：51 🎭

在本节课中，我们将要学习“浪子”这一属性的核心特质，特别是其“脸皮厚”的行为表现。我们将分析不同经济状况下浪子的行为差异，并探讨这一属性与其他属性（如帝王属性）的潜在冲突。

## 脸皮厚的特质 🛡️

上一节我们介绍了课程的主题，本节中我们来看看浪子属性的核心表现：脸皮非常厚。

浪子的脸皮非常厚。

## 浪子的经济分层 💰

理解了脸皮厚是浪子的普遍特质后，我们进一步来看，根据经济状况，浪子可以大致分为两类。

浪子包括有钱的浪子和没钱的浪子。

有钱的浪子通常在很多事情上不在乎别人的眼光。他不在意别人的看法，觉得所有东西都无所谓。

## 经济弱势浪子的行为模式 🎪

与富裕浪子的“不在意”不同，经济状况不佳的浪子，其脸皮厚会通过更具体的行为表现出来。

比较穷的浪子可能会做出一些脸皮很厚的行为。

以下是日常生活中可能见到的、由这类浪子做出的行为示例：
*   蹭饭
*   借钱不还
*   各种耍无赖的行为

这些行为绝大部分都是浪子才会做出来的，体现了脸皮非常厚的特性。

## 与帝王属性的冲突 👑

浪子这种脸皮厚的行为模式，很容易与另一种常见属性——“帝王”属性——产生冲突。

作为帝王属性的人，很多时候会看不惯这种行为。他会很讨厌这种脸皮厚、借钱不还、蹭饭的人。

这两种属性非常冲突。因此我们需要理解这一点。
