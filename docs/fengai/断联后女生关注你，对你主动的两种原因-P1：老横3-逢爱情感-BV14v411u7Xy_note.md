# 情感课程 P1：断联后女生关注你的两种原因分析 🧐

在本节课中，我们将要学习分手后对方依然关注你的两种核心原因。理解这些原因，有助于你更好地把握对方心理，避免错误应对。

---

上一节我们介绍了课程主题，本节中我们来看看第一种原因：对方单纯想观察你的生活状态。

分手后，对方屏蔽了你的一切消息，但是还会关注你的空间、微博、朋友圈，看你的抖音甚至旁观你打游戏。  
有时候还会主动地撩你，然而你一纠缠，他又依然断联。

**原因一**：他单纯想看看你离开了他是不是真的可以过得好。  
你过得好了，他会感觉到失落甚至是生气，觉得分手对你来说居然还是一种解脱。  
要是你过得不好，他只会有一丁点的心疼，剩下的更多则是庆幸离开了你。

---

理解了第一种观察心理后，我们接下来分析第二种更为复杂的原因：对方内心矛盾，既不想你放下，又不敢快速复合。

**原因二**：他不想让你放下，又不想那么快地答应你。  
他的内心其实还爱着你，不想让你因为一时的受挫就真的放弃。  
所以即使屏蔽，也还是忍不住关心，并且丝毫不介意让你知道。  
但是你们的旧感情毕竟是不成功的，所以他也害怕太快答应你会重蹈覆辙，才继续对你冷漠。

---

以下是关于挽回本质的核心要点：

挽回的本质是去重新建立感情和培养信任。  
而这两点的关键都在于让他看到**不一样的你**。
