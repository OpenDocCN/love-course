# 💔 挽回爱情课程 - P1：对症下药，重塑自我

在本节课中，我们将要学习分手后挽回的核心技巧。核心思想是“对症下药”，即通过自我改变来展示诚意，从而重新吸引对方。

分手后的挽回需要运用特定技巧。总结起来，核心就是四个字：**对症下药**。

首先，你需要回顾你们之前的感情经历。重点是找出他曾对你特别不满意的事情。请认真听取建议：把这些不满意的事情总结并记录下来。

以下是具体需要执行的步骤列表：

*   如果他不喜欢你抽烟，你就不抽烟。
*   如果他不喜欢你喝酒，你就不喝酒。
*   如果他嫌弃你好高骛远，那你就踏实工作。

上一节我们介绍了找出问题并列出改变清单，本节中我们来看看如何执行并展示成果。等到你将所有他不满意的行为都改正之后，你需要带着全新的形象重新站在他面前。

这个过程的目的是让他亲眼看到你的改变。同时，让他切实感受到你的诚意和付出。

请相信，做到以上几点，挽回对方就不再是一件困难的事情。

本节课中我们一起学习了挽回的核心理念“对症下药”。我们通过回顾问题、列出改变清单、坚决执行并最终展示全新自我的完整流程，为成功挽回奠定了坚实的基础。记住，真诚的改变是挽回中最有力的语言。
