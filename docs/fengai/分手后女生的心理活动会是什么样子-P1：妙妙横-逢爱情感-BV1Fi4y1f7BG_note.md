# 分手后女生的心理活动解析 - 课程01：排斥期 🚫

在本节课中，我们将要学习分手后女生的心理活动，特别是分手初期（排斥期）的心理状态。我们将了解她为何会感到解脱，以及为何会抗拒前任的复合请求。

## 概述

分手后，双方的心理状态截然不同。提出分手的一方通常会感到释然，而被分手的一方则会感到焦虑和恐惧。这种差异是导致初期复合失败的关键原因。

## 分手初期的心理状态

上一节我们概述了双方的心理差异，本节中我们来看看具体表现。

当一段感情结束时，只要双方曾付出过真情实感，提出分手的一方在做出决定的瞬间，内心会获得**释然**和**解脱**。

与此同时，被分手的一方在接到分手消息时，内心会充满因爱人突然离开而产生的**焦虑**和**恐惧**。

## 被分手方的本能反应

了解了双方的心理状态后，我们来看看被分手方的典型反应。

被分手方的第一反应通常是去**纠缠**和**挽留**。

然而，在提出分手的一方看来，这种行为是在**阻碍她开始新的生活**，是试图将她重新拉回感情的泥潭。

因此，她的内心绝对会对前任产生**抗拒**。

## 复合的关键时机

基于以上分析，我们可以得出一个重要结论。

想要成功复合，必须等待这个**排斥期**过去之后才有可能。
