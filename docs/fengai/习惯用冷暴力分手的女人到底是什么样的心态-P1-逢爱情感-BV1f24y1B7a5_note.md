# 情感课程01：解析“冷暴力分手”背后的心态 🧊

在本节课中，我们将要学习“习惯使用冷暴力分手”的女性背后的心理机制。我们将拆解冷暴力的定义，分析其主要特点，并探讨面对这种情况时，个人应如何应对。

## 什么是冷暴力？

首先，我们需要了解什么是冷暴力。冷暴力指的是一种不明确、不回应、不拒绝也不接受的态度。其核心行为是**强行阻断一切亲密互动**。

理解了定义后，我们来看看习惯使用冷暴力的人通常具备哪些特点。

## 冷暴力者的四个核心特点

以下是习惯使用冷暴力的人最常见的四个心理与行为特征。

1.  **极度自私与自我**
    这类人的思维逻辑是：**“我现在不想说，我就是不说”**。他们只在意自己的感受，而将你的感受视为次要。

2.  **缺乏责任感**
    他们对关系中的“责任感”没有概念。他们不明白一段关系的喜怒哀乐需要共同经营与解决。由于缺乏责任感，他们不会考虑你是否难受，只要能达到让你痛苦的目的，他们就认为目的达成了。

3.  **没有共情能力**
    日常生活中，我们看电视剧都会随剧情哭笑或于心不忍。但冷暴力者作为你最亲密的人，却连 **“我现在心情不好，想一个人待会儿”** 这样的话都不愿说。因为他们的目的不是缓和关系，而是让你心寒。

4.  **追求绝对自由，抗拒约束**
    他们的逻辑是：**“你怎么又提让我烦心的事了？”** 或 **“你怎么不能给我一点空间？”** 他们无法给予你最基本的尊重，却可能对除你以外的所有人都很有礼貌。他们明知你在等待回应，却会故意拉长冷暴力的时间。

## 如何应对冷暴力？

上一节我们分析了冷暴力者的心态，本节中我们来看看面对这种情况，什么才是更重要的。

当面对冷暴力时，**自救远比想方设法去修补一段关系更为实在**。将关注点从“改变对方”转移到“保护自己”的内心感受与边界上，是更为健康的选择。

## 课程总结

本节课中，我们一起学习了冷暴力的定义，剖析了习惯使用冷暴力者的四个核心心态特点：极度自我、缺乏责任感、没有共情能力以及追求无约束的自由。最后，我们明确了面对冷暴力，自我关怀和建立边界比盲目修补关系更为重要。
