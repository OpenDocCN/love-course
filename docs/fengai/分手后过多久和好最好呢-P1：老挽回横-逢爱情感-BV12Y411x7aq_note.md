## 关于“真爱”的认知误区

在考虑复合时，我们需要先破除一个关于爱情的常见错误观念。

上一节我们分析了双方的情感时间差，本节中我们来看看影响复合决策的一个关键心理障碍。

短暂的分离有时是为了更好的重逢，但成功复合的前提是核心矛盾得到解决。
