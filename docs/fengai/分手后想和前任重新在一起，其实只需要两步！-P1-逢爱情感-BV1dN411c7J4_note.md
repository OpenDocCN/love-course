## 第一步：学会换位思考 🔄

上一节我们介绍了课程目标，本节中我们来看看第一步：换位思考。感情中最大的错误是自以为理解对方。例如，在对方不需要关心时强行展现体贴。既然你想挽回，就必须明白对方的真实需求。

在挽回初期，对方最不需要的是你不停的道歉，因为他此刻可能不想沟通。对方也不需要你急切表露真心，因为他可能只想独处。因此，在聊天时，你应尽量减少关心和道歉的内容。

你的对话应尽量减少需要他回复的内容，尽量不让他感到被打扰。这是关键。

以下是具体的沟通话术结构，应为：**道歉 + 关心 + 结束语**。

例如：
> 对不起之前一直打扰你。听说你最近有点焦头烂额，我会好好反省，也希望你尽快回到状态。这条你就不用回了，希望你工作加油。

这句话包含了道歉、关心和结束语。这是为他着想的沟通，核心在于减轻他的压力。

所有其他沟通技巧都万变不离其宗。这是挽回期间的核心沟通策略。

## 第二步：学会给予安全感 🛡️

上一节我们探讨了如何通过换位思考进行沟通，本节中我们来看看如何建立安全感。首先，请分析以下两句话：

1.  “你能不能正常和我说话？为什么非要这个样子？就不能好好聊一聊吗？”
2.  “我真的好喜欢你，我不能没有你。我知道错了，你再给我一次机会好不好？”

这两句话都包含了道歉、表白以及最后的指责（例如，“是你不愿意好好谈”）。你在沟通中无意识地表达了攻击性。

面对这种攻击性，对方本能地会感到不安全，感受到敌意。这使他更无法心平气和地交流，也更难体会你的真情实意。

一般情况下，如果你找他聊天他不理你，就不要再持续追问。你越是如此，他越不想回应。换作任何人，看到长篇的“小作文”都会感到烦躁。
