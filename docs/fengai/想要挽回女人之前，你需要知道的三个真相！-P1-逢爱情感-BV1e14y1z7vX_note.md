## 真相一：乞求只会让你彻底丧失吸引力

上一段我们概述了本课内容，本节中我们来看看第一个关键真相。

乞求行为会明显削弱你的个人吸引力。这个道理显而易见，但大部分人在情绪冷静之前很难认清这个事实。

在情绪平复之前，任何挽回前任的努力都是徒劳的，甚至只会起到反作用。情绪化是挽回过程中最大的敌人。目前，你和他都处于最糟糕的情绪状态。他的情绪表现为逃避和反感你，而你的情绪则表现为无法接受分手的事实。

情绪会将你变成脆弱不堪的动物，使你面对他时只剩下乞求。但是，乞求永远不会给你带来任何转机，也不会帮助你挽回他。

## 真相二：顾虑太多将永远难以挽回

了解了情绪化与乞求的危害后，我们接下来探讨第二个阻碍挽回成功的因素：过多的顾虑。

以下是挽回者常见的几种顾虑：
*   “我不联系他，对方彻底忘了我该怎么办？”
*   “他不愿意理我，我再不联系，岂不是没有挽回的机会了？”

这些都是典型的顾虑。顾虑太多是众多挽回者突出的表现特征。然而，在对方尚未放下戒备心之前，过多的顾虑都是徒劳的。

想要让对方彻底忘掉之前的不愉快，完全抛掉戒备心理，你能做的就是暂时断开联系。核心策略是：**`让时间帮助你化解对方的负面情绪`**。

## 真相三：改变自我是重建吸引力的前提

当我们通过断联化解了对方的负面情绪后，并不意味着就能自然挽回。本节我们将探讨重新建立联系的核心：改变自我。

断开彼此的联系，等到对方没有了戒备心，过了一段时间就可以重新开始互动了。当然，仅仅重新联系并不能直接获得吸引力。

这其实是一个非常大的话题。为了便于你的理解，这里简要说明其本质：重新获得吸引力，和吸引一个新伴侣的过程是不一样的。

对于吸引新伴侣，你只需要表现得足够优秀就行了。公式可以简化为：**`吸引力 ≈ 展示现有价值`**。

但对于重新吸引前任，则需要通过改变来打消对方的固有顾虑，从而引发他对你的新的好感。这个过程是：**`重建吸引力 = 实质性改变 + 打消固有负面印象`**。这才是一个有效的二次吸引过程。
