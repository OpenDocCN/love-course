# 情感修复课程 01：三种不建议复合的情况 🚫

在本节课中，我们将要学习在亲密关系结束后，三种明确不建议尝试复合的具体情况。理解这些情况有助于你保护自己，避免受到二次伤害。

上一节我们介绍了课程的主题，本节中我们来看看具体是哪三种情况。

## 情况一：建立在单方面付出的关系上

如果你们的关系建立在由你单方面持续付出的基础上，而对方只是单方面享受你的好，完全不顾及你的想法和感受。这通常意味着对方只将你视为备胎。

**核心逻辑**：`对方视你为备胎 → 遇到理想人选 → 你会被抛弃`

在这种情况下，一旦对方身边出现其理想中的人，他会立刻离开你。此时若选择挽回，未来极有可能遭受第二次同样的伤害。

## 情况二：分手后对方迅速开始新恋情

如果你们分开不到半个月，甚至不到一周，对方身边就已经出现了新的人。这种情况被称为“无缝衔接”。

以下是关于此情况的要点：
*   从法律和道德表层看，分手后开始新恋情是对方的自由。
*   但能在如此短的时间内接触并接纳新人，表明你对他而言并不重要。
*   若在此情况下仍试图挽回，你将承受巨大的心理压力和痛苦。

## 情况三：对方缺乏基本的尊重

当你已经竭尽全力去争取挽回时，或许你的方法并不恰当，但对方却连最基本的尊重都没有给予。

以下是对方不尊重的表现：
*   辱骂你。
*   羞辱你。
*   讽刺你。
*   践踏你的真心。

即使已经分开，但你们曾经在一起。对方能用这种方式伤害你，表明他已毫无尊重可言。你很难再期待对方在未来会给予你真正的爱。

本节课中我们一起学习了三种不建议复合的情况：单方面付出的关系、对方迅速开始新恋情，以及对方在挽回过程中表现出毫无尊重。如果你的关系符合以上任何一种情况，建议不要选择挽回，以保护自己免受进一步伤害。
