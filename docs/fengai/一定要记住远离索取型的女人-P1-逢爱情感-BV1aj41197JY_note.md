那么，这种索取型人格是如何形成的呢？本节我们来探讨其背后的心理根源。

这一类人往往源于原生家庭的不幸福。例如父母关系不融洽、从小父母离异。在单亲家庭长大的孩子，可能从小缺乏家庭温暖和父母的关爱。或者，有些人小时候家庭物质条件匮乏。

长大之后，她们就会疯狂索取情感和曾经缺失的东西，比如物质与金钱。
