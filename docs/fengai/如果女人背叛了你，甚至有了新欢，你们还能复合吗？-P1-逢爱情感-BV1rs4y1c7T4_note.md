# 情感修复课程 01：背叛后的复合可能性分析 💔

在本节课中，我们将要学习当伴侣（特别是女性伴侣）发生背叛并有了新欢后，双方是否还有可能复合。我们将从背叛者的心理、关系的动态变化以及被背叛者如何应对这几个方面进行分析。

---

有人提出疑问：背叛男人的女人会有好下场吗？答案是不会。她目前与新欢在一起可能显得开心，但将来必定会后悔。

因为她本质上只爱自己，缺乏真心关怀他人的能力。她不会珍惜任何真心对待她的人。

随着时间的推移，那些曾经真心待她的人并非愚钝，会逐渐收回对她的好。当她消耗尽所有真心待她的人之后，处境只会愈发悲惨。

她轻视那些真心爱她的人，这些人迟早都会离开她。最终，她会遇到一个在情感段位和能力上都高于她的人。

这个人将会完全掌控她。此时，当她生活不如意时，她会想起谁？她会想起你，并可能回来找你。

她误以为你还会像过去一样对她好，她误以为回来之后还能继续扮演你心中女神的角色。

这个时候，如果你缺乏原则和骨气，你可能会选择接受她。

那么她的生活将因为你的卑微讨好而再次变得舒适。但如果你是一个有原则、有骨气的人，你会借此机会教育她，让她明白践踏他人真心会带来怎样的后果。

我理解你此刻非常痛苦，但过一段时间再回看，你会发现她的生活已经出现了问题，已经不如意了。

此时，你只需要做一件事：决定是否要接受她，是否允许她回到你的生活中。

**决策路径如下：**
*   **选择接受**：你们的关系便可能和好。
*   **选择拒绝**：她将不得不独自面对生活的磨砺。

你能理解其中的含义吗？因此，在感情中，真心待人并非没有优势。

**你的真心付出** → **可能换来他人对你的真心对待**
