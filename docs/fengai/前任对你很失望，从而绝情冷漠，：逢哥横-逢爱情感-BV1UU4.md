# 前任对你很失望，从而绝情冷漠，：逢哥横-逢爱情感-BV1UU4

---

如果前任对你特别失望，甚至分手后对你越来越冷漠绝情，对你说“别再找我了，我们已经没有可能了，回不去了”，这并不意味着真的没有机会和好。

八成以上的人听到这些话就会感到绝望，想要直接放弃，但你其实理解错了他的意思。

这些话表面上看似绝情，但并不是否定了你们过去所有的感情。他只是不喜欢目前的相处状态。最起码你们过去的感情是存在的，只是现在在一起的时间久了，失去了新鲜感。

换一个角度来讲，前任对你越绝情，就说明他越在乎你。因为他太过在乎这段感情，太爱你了，所以当他积累了太多的失望和矛盾分歧时，就会导致感情缺失，进而想要逃离这段令人疲惫的关系。

正是因为他太过在乎，才承受不起失去的代价。

上一节我们分析了绝情态度背后的心理，本节中我们来看看如何通过具体的沟通来打破僵局。

今天教你一招，用以打破他对你的失望，重新建立起依赖感。你可以在分手十天之内，把下面这段话发给他：

> 如果有一天你能遇上一个人，能像我爱你那样去爱他，那你就能明白今天我是多么的痛苦和崩溃。没能让你坚定地选择我，说明是我做得不够好，是我不够优秀。不主动怕失去你，太主动怕自作多情，伸手怕犯错，缩手怕错过，多一点怕打扰，少一点怕遗憾。爱你这件事，我真的尽力了。如果当初我们刚遇见，你就告诉我我们会有今天的结果，那我一定不会陷得这么深。我可能很难再走出来了，你也别有太多的心理压力，我只是想跟你说说心里话。能与你相遇相爱，是我这辈子最幸运的事，很可惜没能留住你。

这段话的核心在于**表达理解、承认不足、释放情感而不施加压力**，其结构可以概括为：**真诚反思 + 情感倾诉 + 无压力结尾**。

将这段话发给他之后，三天内他很大概率会找你。但是，你必须克制住自己，不要回复他任何信息。

以下是此阶段的关键行动要点：

*   **保持沉默**：对方主动联系时，不回复是为了制造空间感，避免重回旧有互动模式。
*   **观察反应**：留意对方联系时的情绪和内容，为下一步行动提供依据。
*   **稳定心态**：利用这段时间调整自己的情绪，避免因焦虑而做出冲动行为。
