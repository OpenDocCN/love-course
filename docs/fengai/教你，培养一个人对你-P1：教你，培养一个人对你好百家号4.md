# 教你，培养一个人对你-P1：教你，培养一个人对你好百家号4

在本节课中，我们将学习如何通过行为引导，影响他人对待你的方式。核心在于通过一致的行为反馈，让对方明白何种行为能获得你的积极回应，从而引导其主动对你好并联系你。

上一节我们介绍了课程主题，本节中我们来看看具体的行为引导原则。

## 引导主动联系 🔄

如果你希望一个人主动联系你，需要在他不联系你时保持冷淡，前提是你确定对方对你有感情。当对方联系你时，则热情回应。这种行为模式旨在培养对方主动联系你的习惯。

以下是错误做法与正确引导的对比：

*   **错误做法**：对方不联系你时，你频繁联系对方；对方联系你时，你却态度冷淡。
*   **正确引导**：对方不联系时，你保持冷淡；对方联系时，你热情回应。

**核心公式**：`对方行为` → `你的反馈` → `引导未来行为`
*   期望行为（主动联系）→ 积极反馈（热情回应）→ 强化该行为
*   非期望行为（不联系）→ 消极/无反馈（冷淡）→ 减弱该行为

## 应对冷战策略 ❄️

当对方对你采取冷战态度时，你的应对方式将决定未来的互动模式。如果你在对方冷战时求和讨好，这等于告诉对方冷战能换来你的更多关注，从而会助长对方的冷战行为。

以下是应对冷战的两种方式：

*   **强化冷战的做法**：对方冷战时，你表现出求和与讨好。
*   **引导关系改善的做法**：对方冷战时，你表现出更不在乎；对方热情时，你回应得更热情。

**行为引导逻辑**：只有让对方明确感受到“对你好”才能换取“你的好”，他才会倾向于用积极的方式对待你。
