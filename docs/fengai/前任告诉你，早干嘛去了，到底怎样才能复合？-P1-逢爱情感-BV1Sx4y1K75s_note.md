# 前任告诉你，早干嘛去了，到底怎样才能复合？-P1-逢爱情感-BV1Sx4y1K75s_note

在本节课中，我们将学习如何解读前任话语中的深层情绪，特别是当对方说出“早干嘛去了”这类充满失望的言语时，应如何理解其心理并采取正确的沟通策略进行关系修复。

## 理解话语背后的情绪

上一节我们提到了核心问题，本节中我们来看看如何深度解读对方的话语。

当对方说出“你早干嘛去了，我之前需要你，但是现在我不需要了”时，这句话本身代表了她对你曾抱有期待，但期待最终落空了。

期待落空后，情绪转变为失望。可能是她需要你陪伴、鼓励或支持时，你未能满足她的需求。

这让她感觉你不在乎也不重视她。

多次期待落空后，她便不再对你抱有期待。

因此，她经历了从期望到失望，再到绝望的心理过程。

## 采取正确的沟通策略

明白了她的心理路程后，你就知道该如何着手修复关系了。

以下是你可以采用的沟通话术示例：

> 我明白了，是我之前让你失望的次数太多，没有体会到你的绝望。直到分手后我才理解你的痛苦。是我做得不够好，我明白得太晚了。很抱歉给你带来了这么不好的恋爱体验。总之都是我的不好。

这段话的目的是让对方明白你现在理解她的感受，并引导她主动与你沟通。

这属于积怨清理和化解的过程，运用了行为合理化与共情的技巧。

## 应对对方的后续反应

这段话发出后，对方可能会回应：“那你现在说这些还有什么意义呢？你要是早一点明白，我们现在也不会变成这样了。”

此时不要认为没有希望了。受过伤的人会下意识地保护自己，不会因为你的一番话就立刻同意复合。

这时不要着急或气馁，要学会一点点感化对方的内心。

对于因失望而导致的分手，坚持才是最重要的。
