# 前任非常理性，到底怎样做才有挽回的机会-P1-逢爱情感-BV1cV4y1u7bv_note

在本节课中，我们将要学习如何挽回一位性格倔强、固执且非常理性的前任。我们将分析其行为背后的底层逻辑，并提供具体的应对策略。

---

如果你的前任性格倔强固执，一旦做出决定就不轻易改变，这并不意味着你没有机会了。本节将揭示其行为背后的核心逻辑。

没有绝对无法挽回的人，关键在于缺乏一个足以让其改变主意的合理理由。这背后通常有两个原因：**一是他看不到任何希望；二是他盲目地认为分手决定是明智且正确的**。

那么我们能做什么呢？其实很简单：**第一，让他看到希望；第二，让他对自己的决定感到后悔**。我们需要引导他不断进行自我否定与怀疑，质疑自己分手的决定是否正确。

---

上一节我们介绍了理性分手背后的两个核心原因，本节中我们来看看我们自己是如何彻底放弃一件事物的，这有助于我们理解对方的心理。

我们彻底放弃某事物通常也基于两个层面：

以下是第一个层面：
*   该事物对我们而言已毫无意义，或在绝大多数情况下都弊大于利，因此我们不得不放弃。

以下是第二个层面：
*   该事物收益很大、极具诱惑力或吸引力，但我们需要为之付出的成本太高，或需做出巨大牺牲（包括精神、身体或物质上的）。即便如此，我们最终仍可能选择放弃。

这正是许多看似处于高位、自身价值很高的人依然被分手的原因——“你很好，可是我受不了”。以上两种情况均属于对方理性分手的范畴，你可以进行自我对照。

---

除了理性的分手，还有一种情况相对乐观，那就是情绪化的分手。

他突然脾气爆发、完全无法忍耐、对你大动干戈。你以为他做出了一个难以更改的决定，但这实际上可能是被当时局面或你的行为所逼迫，而做出的一种夸张反应。你只是被这种激烈的表象震慑住了。
