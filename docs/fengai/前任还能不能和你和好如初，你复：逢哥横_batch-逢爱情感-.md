# 前任还能不能和你和好如初，你复：逢哥横_batch-逢爱情感-

在本节课中，我们将要学习一个关于情感修复的核心观念：为什么将复合的希望寄托于“探测前任是否还在意你”的想法是错误的，以及如何建立正确的心态来掌握关系修复的主动权。

很多人会有一种想法：在尝试挽回之前，先想知道前任心里是否还有自己。如果前任心里还有自己，那就去复合；如果没有，那就不去找他了。这种想法是完全错误的。这么做等于把复合的主动权完全交给了前任。

让他来决定你们是否还有机会在一起。在挽回的过程中，你会变得非常在意他的想法，不断揣测自己做什么他会开心，做什么他会不开心。这样畏首畏尾，反而会错过复合的最佳时机。

那么，前任的想法是否重要呢？前任的想法是重要的。但它影响的并非复合的最终结果，而是你在复合时应该采用的具体方法。他是否对你余情未了，决定了你采取的方式会有所差异。

同时，前任的想法其实也并不重要。说不重要，是因为他的想法左右不了你的决定。因为你喜欢他，想要跟他在一起，所以你会选择放手一搏。不论最终输赢，只要自己努力过了，就能问心无愧。

**核心心态公式：坚定决心 + 主动行动 = 最大化复合几率**

拥有这样的心态，才能极大地增加挽回的成功概率。
