# 想挽回修复但是不知道怎么破冰应该怎么办-P1-逢爱情感-BV1iM4y1Y7XX_note

在本节课中，我们将要学习当感情出现裂痕、对方态度冷漠时，如何停止内耗、调整心态并迈出破冰的第一步。核心在于将关注点从对方转回自身，通过改变自身状态来影响关系走向。

许多人面临感情问题时希望修复，但遭遇对方冷漠的态度后便感到不知所措，不知如何破冰。无需纠结，也无需继续内耗。当你处于纠结不断、能量不稳定的状态时，不应做出任何决定。

## 停止纠结与内耗

上一段提到了能量不稳时不宜行动，本节中我们来看看如何停止无谓的纠结。你可能会问自己是否还要继续坚持，或因为对方态度明确而考虑放弃。你明白断舍很难，但不舍得又不知如何继续。此时，最佳选择是暂停一切行动，不让此事持续牵扯你的情绪。

继续纠结只会让你更加受伤。你应该专注于做好自己的事，享受生活，回归正常的生活轨道。不要自乱阵脚，要找到自己的生活节奏。

## 调整心态与降低期待

找到节奏后，心态的调整至关重要。感情中的许多矛盾，源于我们对伴侣的过高期待。

如果对方既不行动也不改变，而你替他完成了他该做的事，结果只会让你感到疲惫和心情低落。不如放松一些，主动降低对他的期待。在认真的感情关系中，两人的状态通常是互补的。

以下是常见的互补状态示例：
*   一方较为勤快，另一方可能相对懒散。
*   一方过于积极和强势，另一方可能变得消极被动。

因此，当你希望关系发生改变时，可以尝试用对方的态度和立场来生活。感到痛苦的一方自然会率先做出改变。

## 专注于自身，创造舒适感

关键在于放过自己。你要确认，最大的敌人往往是自己。自己想通了，事情就解决了。

核心行动准则可归纳为：**做好自己的事，少管别人的事，别担心老天的事**。当你让对方感到轻松，让这段感情让他觉得很舒服时，他自然就不会那么急于离开你。

本节课中我们一起学习了情感破冰的初级心法：停止内耗、调整心态、降低期待并专注于自我成长。记住，改变始于自身，当你变得稳定而松弛时，才能为关系的修复创造真正的空间。
