# 真爱验证指南 💖 - 课程 P1：冷静期测试法

在本节课中，我们将学习一种通过“冷静期”来验证双方是否是真爱的方法。这个方法的核心在于，通过分离后的情绪反应，来辨别是习惯依赖还是真正的爱。

很多人会困惑自己是否真的爱对方，或者分手后是否值得挽回。其实，分手后可以冷静一段时间来测试对方是否爱你。因为刚分开时，你的痛苦可能只是源于不习惯没有对方的生活。

上一节我们提到了冷静期的必要性，本节中我们来看看如何具体操作和判断。

以下是进行“冷静期测试”的具体步骤与判断标准：

1.  **双方进入冷静期**：分手后，主动或被动地与对方断绝联系一段时间，让双方情绪都平复下来。
2.  **观察自身情绪**：在冷静期间，留意自己的内心感受。重点观察是否会出现强烈的想念、难过或煎熬的情绪。
3.  **做出最终判断**：如果经过一段时间冷静后，你内心仍然持续感到**想念、难过和煎熬**，那么这很可能意味着对方是你的真爱。

如果测试结果显示对方是你的真爱，那么请务必珍惜这段感情。建议主动尝试复合。许多感情最终错过，往往是因为一方没有回头，而另一方也没有挽留。
