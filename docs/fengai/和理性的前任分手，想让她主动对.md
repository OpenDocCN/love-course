# 和理性的前任分手，想让她主动对

## 概述

在本节课中，我们将要学习如何与理性的前任相处，并理解他们提出分手的核心逻辑。掌握这些关键点，将有助于你找到复合的正确路径。

## 理性思维：一场开卷考试

为什么理性的人是最容易复合的呢？其实你去跟理性的人复合，就是一场开卷考试。他们想要什么，他们有什么需求，你去想办法提供就行了。

理性的人，他们的思维模式很简单，就是永远在权衡利弊。

## 理性人的权衡标准

与你在一起的时候，理性的人会评估一系列具体因素。以下是他们通常会考量的关键指标：

*   **情感体验**：跟你在一起的时候开不开心，愉不愉快。
*   **现实条件**：物质条件有没有达到。
*   **观念匹配**：三观是否匹配。
*   **外在条件**：你的相貌，你的外在条件是否能满足他。
*   **社会资源**：你的社会地位，你的家庭背景等等。

有的人就要问了，那跟爱没有关系。

## 爱在理性关系中的位置

怎么可能没有关系呢，主观上的爱也很重要，但是在理性的人心中，占比不是特别的大。

所以你会发现，跟理性的人在一起，他明明很喜欢你，但是他总觉得跟你在一起没有未来，然后找各种理由去放弃。

## 理性分手的表面与实质

有的理由你可能听着根本就不成立，那是为什么呢？因为他不想让你站在道德的制高点去制裁他。

他总不能说：
*   `我嫌你长得不好看`
*   `嫌弃你没有钱`
*   `嫌弃你工作能力不行`
*   `嫌弃你的家庭背景不行`

他只会说：**我跟你在一起不开心，不合适，三观不合**。他们总是找这些情绪上或者说性格上的矛盾跟你提出分手。

## 复合的核心策略：找出并解决问题

接下来说的就是重点了。你要做的就是，找出你的前任为什么跟你分手，他到底权衡利弊些什么，是哪些东西。

你把他给找出来，然后呢，一点一点的去解决掉。当然，你找到了原因，你就去改变。

这个时候，你要做的就是，把你的每一个指标都达到他所需要的那个合适度。这个时候的主观爱意，自然而然的也就回来了。

## 总结与鼓励

所以说，理性的人复合是非常的简单，就像是开卷考试一样。就是已经告诉你答案是什么了，你把答案死记硬背下来，然后在试卷上一笔一画的写出来，你自然而然的就能得到满分。

而且，真的有完全理性的人吗？只要你做的不是太差，你的爱人一般都是能够打破外在的条件的。

**公式：复合成功 = 找出关键不及格项 + 针对性提升至及格线以上**

一旦你能够突破你在前任心中那个不及格的点，那爱情就重新回来了。
