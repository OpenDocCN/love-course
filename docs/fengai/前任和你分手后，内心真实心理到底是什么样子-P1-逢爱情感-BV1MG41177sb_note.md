## 分手初期的心理状态与沟通误区

上一节我们概述了本课主题，本节中我们来看看分手后对方的心理状态。如果你们是真心分手，那么在分手后，就不要老是想着当面跟对方沟通和解释，没有用的。对方现在处在一个极度讨厌抗拒你的状态里，你去跟他说什么，他都会无动于衷。

而且呢，你们很多人的沟通根本就没有沟通到一个点上。你跟对方说那些山盟海誓，做那些承诺，看似好像很感动，又或者所谓的很真诚，但是对对方来说没有任何的用。他就是不抱希望的要跟你分手了。

## 复合的关键：建立共情

所以呀，分手后你想要去复合，最重要的就是先共情。你得让对方深刻的意识到，你现在是能够跟他感同身受的，你能够懂他，你能理解他。

**核心公式：有效沟通的前提 = 消除抵触 ≈ 成功共情**

只有这种状态的时候，对方才会没有对你那么的抵触。他不抵触你了，你才能够去沟通，去联系，重新修复关系。

## 无法共情的后果与正确心态

但如果说共情做不到，他一直都是反感、讨厌和抵触你的，那你们始终都无法达成平衡关系。

以下是此时你需要避免的行为：
*   不要问，也不要去猜他到底是怎么想的。
*   不要纠结他还爱不爱，他还喜不喜欢。

别傻了，能分手就是已经没有那么喜欢你了呀。如果说你想要去复合，你就不要再去纠结对方的想法。

## 复合的正确行动路径

你还不如好好的想想，从哪个方面入手，把你们的问题去解决，然后如何去跟对方建立联系，让他感受到你的改变。这才是真正意义上的能够帮你复合。

**行动代码框架：**
```plaintext
1. 停止纠结对方想法 -> 2. 分析并解决核心问题 -> 3. 策略性建立联系 -> 4. 通过行动展示改变
```
