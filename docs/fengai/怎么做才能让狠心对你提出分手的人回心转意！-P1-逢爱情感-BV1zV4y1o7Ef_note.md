# 情感修复教程 第1课：如何让提出分手的人回心转意？💔

在本节课中，我们将要学习一个关于情感修复的核心策略：如何通过调整自身行为与互动模式，来影响对方的态度。核心在于理解人性心理，避免过度付出，并学会引导对方投入，从而重建吸引力和关系主动权。

## 停止讨好行为

上一节我们介绍了课程目标，本节中我们来看看需要停止的第一个关键行为：讨好。

想让女人回心转意，就不要对她太好。讨好这个行为在女人眼里就是“舔”。一个特别残酷的真相是：一个男人无论多么优秀，只要他对女人过度关心和付出，为自己喜欢的女人做什么都愿意，这样往往不会被珍惜。

## 理解女性的核心需求

了解了不该做什么之后，我们需要明白对方在关系里真正需要什么。

80%的女人在关系里不需要你对她太好。她们在关系里渴望的是，你能够激起她们的征服欲，仅此而已。因此，你不断去讨好她，她就对你爱答不理。反而你能吊着她，她却对你欲罢不能。

你会发现，很多时候你对她越好，她反而觉得你很廉价。但是你对她高冷，或者说你对她爱答不理，她反而对你越来越好。

## 学会放大自身价值

明白了需求，接下来我们需要掌握提升自身吸引力的方法。

要拿捏女人，你要学会放大你的价值。在一段关系里面，你的价值有多高，在一定程度上不是取决于你本身的价值，而是取决于她给你投入了多少。

**核心公式：感知价值 ∝ 对方投入**

所以你要让女人持续的喜欢你，你就要引导她不断地给你投入时间、身体、精力等等。人性就是这样，投入得越多就越重视。

## 掌握“拉扯”技巧

价值需要通过互动来体现，以下是如何在互动中创造张力的方法。

学会来回的拉扯。有时候你对她关心让她上头了，让她觉得你很喜欢她，接着就迅速地撤离，不回她的信息，让她觉得你好像又没有那么喜欢她了。

```python
# 情感互动“拉扯”模型示例
interaction_cycle = ["表达关心", "迅速撤离", "引发猜测"]
for step in interaction_cycle:
    execute(step)
```

那么你到底喜不喜欢她呢？这样她就会陷入到猜测之中。所以你要掌握住主导权。

## 改掉暴露需求感的毛病

最后，为了巩固主导权，我们必须修正一个常见错误。

你想要获得主导权，首先要做的第一件事就是改掉你动不动就想要表达需求的这个毛病。你不要动不动的就去表达你有多么的爱她，你有多么的离不开她。

因为人性就是这样，暴露需求感只会降低你自身的价值。通过自身努力获得的东西，永远比别人给的要更加珍贵。
