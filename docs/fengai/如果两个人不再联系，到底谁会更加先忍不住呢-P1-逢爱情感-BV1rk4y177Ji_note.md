# 情感教程 01：分手后谁会更先忍不住联系？🤔

在本节课中，我们将要学习分手后双方的心理动态，特别是关于“谁会更先忍不住联系对方”这一常见问题。我们将抛开性别刻板印象，从情感伤害的深度与心理依赖的角度来分析这一现象，并探讨如何将精力用于真正有助于个人成长与关系修复的事情上。

## 核心逻辑：与性别无关

上一节我们介绍了课程的主题，本节中我们来看看问题的核心逻辑。两个人不联系后，男人和女人谁会先忍不住，这个问题其实跟男女关系并不大。因为每个人身上都会有一定比例的女性特质，反之亦然。

用男女来衡量谁先忍不住并不准确。能问出这个问题，说明你一直在用单一的思维来思考问题。但是感情这个东西，它往往并不是非黑即白的关系。

## 动态变化：感情的非线性

感情状态是动态变化的。可能很多人昨天还在说老死不相往来，没准今天就复合成功了。

那么分手后，谁会先忍不住联系对方呢？准确的来说，就是**受伤更深的人更能忍得住不去联系**。

以下是帮助你理解这一点的例子：

*   **受伤更深的一方**：你很爱很爱一个人，但是他持续地让你失望，持续地不自觉做出一些伤害你的事情。哪怕你很爱，你也会忍住更久不去联系。
*   **被偏爱的一方**：对方则像是一个被宠坏了的孩子，肆无忌惮地胡闹。但是你一旦抽身，他可能就慌了，不知道该怎么办了。于是乎他就会先受不了，先来主动找你。

哪怕他还没有意识到自己真正的问题，还是觉得是你狠心、是你不辞而别，他也会率先主动。

## 底层逻辑：主动联系与爱的深度无关

从底层逻辑来理解这件事情，你会发现：**是否主动去联系你，或者能不能忍得住去联系你，跟爱不爱关系真的不大**。

这与谁爱得更深也没有多大的关系，甚至与是否意识到了自己有错误也没有什么关系。

## 行动建议：关注真正重要的事

如果说你真的想挽回一段感情，我希望你先放下“谁会先忍不住”这个问题。因为你关心的这个问题本身，对你是否能挽回没有任何意义。

分手后本身的精力就有限，还有一部分注定会消耗在自我内耗上。所以，我们要将这个有限的精力，都放在对的地方。

以下是应该关注的核心事项列表：

*   关心那些真正对你修复关系有用的事情。
*   关心那些对你个人成长有用的事情。

## 总结与延伸

本节课中，我们一起学习了分手后双方的心理博弈。关键在于，更先联系的人往往是情感依赖更强或更不适应分离的一方，而非爱得更深或自知有错的一方。因此，将精力从猜测对方转向自我成长和有效行动，才是修复关系或开启新生活的正途。
