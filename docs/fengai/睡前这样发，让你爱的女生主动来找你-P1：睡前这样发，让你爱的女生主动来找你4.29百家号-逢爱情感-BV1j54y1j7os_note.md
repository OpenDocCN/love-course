# 睡前这样发，让你爱的女生主动来找你 - 课程01：爱情激素与人性本质 🧠💖

在本节课中，我们将要学习爱情背后的科学原理，了解影响我们情感体验的生理机制，并探讨如何基于这些知识构建更深刻的情感承诺。

## 爱情激素的作用原理

上一节我们介绍了课程主题，本节中我们来看看爱情背后的科学。据研究表明，人体在恋爱的时候会分泌很多种爱情激素。其中会让人意乱情迷产生愉悦的激素，叫做**苯基乙胺**。它会使我们迫切地想和对方在一起。

然而，苯基乙胺的浓度高峰通常只持续**6个月到3年**的时间，这大致对应一次恋爱的时间周期。

## 人性本质与爱情承诺

理解了激素的影响后，我们进一步探讨人性的本质。所以，人本身就不是长情的生物。至死不渝的爱情，本质上是违背天性的。

因此，当我在结婚的时候，不会说“无论贫穷富贵，我都不会离开你”这样的话。而是选择把双手放在**自私的基因**和**进化心理学**的著作上宣誓：**我将违背我的本能，忤逆我的天性，永远地爱你。**
