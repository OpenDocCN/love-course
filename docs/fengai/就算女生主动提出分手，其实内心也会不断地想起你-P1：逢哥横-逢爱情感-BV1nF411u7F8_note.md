# 爱情心理学课程 01：分手后女生的心理分析 🧠

在本节课中，我们将要学习女生在主动提出分手后的真实心理状态。我们将分析其行为背后的情感逻辑，并探讨如何理解与应对这种局面。

## 核心心理机制

主动提出分手的女生也会不断想起你。无论分手时两个人翻完所有的旧账，或是大吵出手之后立下誓言，无论她走得有多么坚决，你都一定要相信女人天生就是感性的动物。她们拥有更多敏锐的情思，以及更脆弱的防御体系。

## 两种典型反应模式

上一节我们介绍了女生分手后的核心心理状态，本节中我们来看看由此衍生出的两种典型行为模式。

正是因为其感性与防御脆弱，她们通常会表现出两种极端反应。

以下是两种具体的反应类型：

1.  **嘴硬心软型**：她们内心盼着你去挽留，以此逃避自己面对分离的焦虑。
2.  **决裂彻底型**：她们表现得非常决绝，无论你怎么挽留，得到的回应可能只有拉黑、删除，甚至是辱骂。

第一种女生很好挽回，只要持续地去哄她就可以。

那么第二种呢？其实也并不难，只是很多男生不理解。他们被这些坚决的言辞给吓住了。关键在于认识到一个核心事实：分手后，女生并没有她看起来那么冷血和不在乎。

哪怕是她们找了一个新的人来代替你，在相处过程当中，也会不断地将新人与你做对比。

**无论她现在对你表现得很冷漠，还是看起来已经有了新欢，都不要因为过度的慌张，而丢失了你在她心中原有的价值。**

## 复合的关键

好好去揣测对方此时的心理，激发她对你的感情和占有欲，复合也当然是水到渠成的事情。
