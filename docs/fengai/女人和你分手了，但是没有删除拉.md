上一节我们介绍了课程的主题，本节中我们来看看第一种常见原因：将你视为备胎。

## 1. 将你视为备胎 🛞

当女性在分手后未删除或拉黑你，可能意味着她将你视为一个可以随时联系的备胎。这种情况通常发生在她投入新恋情后未能获得幸福，或认为新伴侣不如你时。她会继续保持与你的联系，以便在需要时寻求依靠。

以下是具体表现：

*   她在新恋情中感到不幸福或不满时，会主动联系你。
*   如果她在新恋情中非常幸福，被伴侣疼爱，则可能会彻底删除你。
*   在未完全获得幸福前，她不会轻易断绝与你的联系。
*   她可能会继续与你聊天和来往，但不再投入感情。

**核心概念**：备胎关系是一种**单向的情感储备**，公式可表示为：
`她的联系意愿 = f(新恋情满意度)`
即，她对你的联系意愿与新恋情的满意度成反比。

了解了备胎心理后，我们接下来看看第二种可能性：假装做朋友以利用感情。

## 2. 假装做朋友以获取付出 🤝

如果女性在分手后未删除拉黑你，另一种可能是她想利用你的感情，希望你继续为她付出。这类女性通常会提出“即使做不了情侣，也可以做朋友”，以此作为维持关系的借口。

以下是常见的利用方式：

*   她会在获取你的信任后，提出各种要求。
*   她可能以朋友之名，让你赠送礼物或为其消费买单。
*   她会利用你的剩余感情，来解决自己的消费或情感需求。

**核心概念**：这种关系本质上是**情感剥削**。需要警惕“男女间纯友谊”在此情境下的真实性。代码逻辑可以简单描述为：
```python
if 她提出“做朋友” and 同时提出物质要求：
    动机可能是“利用”
```

上一节我们探讨了被利用的风险，本节中我们来看看最后一种相对积极的可能：她后悔了并给你机会。

## 3. 心里后悔并给予复合机会 💔

当女性真正不爱了，分手后往往会果断拉黑，不留任何机会。反之，如果她对你仍有眷恋或后悔分手，就可能不会删除拉黑，而是继续保持联系。

以下是这种行为背后的信号：

*   她离开后，发现你的优点，心生后悔。
*   她会通过给你发消息或其他方式暗示，等待你重新追求她。
*   她为你留出了重新建立联系和感情的空间。

**核心概念**：这种行为传递的是**潜在的复合窗口**。关键在于识别对方是给予机会，还是仅仅维持联系。关系状态可表示为：
`她的行为 = {不删除， 不拉黑， 主动联系} → 可能意图：后悔/给机会`
