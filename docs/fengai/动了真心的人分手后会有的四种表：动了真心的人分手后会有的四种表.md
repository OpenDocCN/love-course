# 动了真心的人分手后会有的四种表：动了真心的人分手后会有的四种表

在本节课中，我们将要学习当一个人在一段感情中投入了真情实感，分手后通常会出现的四种心理与行为表现。理解这些表现有助于我们更好地认识自己或他人的情感状态。

上一节我们介绍了本课程的主题，本节中我们来看看具体是哪四种表现。

以下是四种典型的表现：

1.  **无法做到不再联系，根本控制不住自己。**
    当事人会感到一种强烈的冲动去联系前任，即使理智上明白不应该这样做，情感上却难以自控。

2.  **在“继续纠缠”和“选择放手”之间反复挣扎。**
    内心充满矛盾，一方面渴望挽回关系或保持联结，另一方面又知道应该尊重对方的决定并让自己离开。

3.  **试图逐步退出对方的世界，同时又恐惧真正的失去。**
    可能会采取删除联系方式、避开共同朋友等行为来让自己疏远，但内心深处非常害怕这样做会导致两人彻底终结，再无关联。

4.  **每日思念对方，却畏惧主动去找他/她。**
    思念成为一种常态，但出于害怕被拒绝、担心造成打扰或自我尊严等原因，而不敢将思念付诸行动。
