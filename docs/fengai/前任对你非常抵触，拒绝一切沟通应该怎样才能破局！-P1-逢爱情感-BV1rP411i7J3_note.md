# 前任对你非常抵触，拒绝一切沟通应该怎样才能破局！-P1-逢爱情感-BV1rP411i7J3_note

在本节课中，我们将要学习当前任因分手纠缠而对你产生强烈抵触、甚至拒绝沟通时，应如何破局。核心在于理解情绪变化的规律，并采取正确的步骤来重建联系。

如果你们的分手经历过纠缠，对方对你非常抵触，甚至已将你删除或拉黑。这种情况下，不要认为仅通过解释、认错、道歉或要求再见一面，就能让对方回心转意或心软。这是无效的。

只要你表现出想与他重新在一起的意图，对方就一定会疏远你。那么，是否意味着这辈子都不可能，再也没有机会了？并非如此。无需担心，因为负面情绪可以被淡化，时间是治愈一切的良药。

这里我举一个例子。假设你小时候过得不太幸福，父母对你可能很严厉甚至不好。但当你长大后，回想起来还会有那种特别痛苦和不堪的感觉吗？通常不会了。你留下的记忆更多是父母对你的好。这与你的心态变化有很大关系。

你的认知体系随着成长而扩大，这扩展了你的包容度和共情能力。你开始体会到他们的不容易。另一个原因是，人都会下意识地回避那些感觉痛苦的回忆，而更愿意铭记那些带来开心、喜悦和幸福的回忆。

这解释了为什么很多人在分开一段时间后，又会重新想要和对方在一起。因为他们之间的冲突被时间淡化了。因此，在当下这个阶段，你的首要任务是给对方创造一个没有压力的环境，与对方建立一个“舒适圈”。

这个“舒适圈”即所谓的“做朋友”。只要对方习惯了以新的身份和关系与你相处，你们就可以像普通朋友一样，进行日常的沟通和交流。你可以给他分享生活，他也会开始对你的分享有所回应。

这与先前的抵触和厌恶感相比，局面就好太多了。

本节课中，我们一起学习了当前任因抵触而拒绝沟通时的核心应对策略。关键在于**停止施压，利用时间淡化负面情绪，并致力于以朋友身份重建无压力的舒适圈**。记住，修复关系是一个渐进的过程，耐心和正确的姿态是成功的基础。
