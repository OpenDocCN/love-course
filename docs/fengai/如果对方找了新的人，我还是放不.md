# 如果对方找了新的人，我还是放不

在本教程中，我们将学习当分手后对方已有新欢时，应如何调整心态并采取有效策略，以增加复合的可能性。我们将探讨核心原则、具体行动步骤以及重要的心理战术。

## 概述

分手后，对方开始新的恋情是常见情况。本节将分析此情境下的基本心态与定位。

分手后对方有新欢，这并非不道德行为。分手意味着双方恢复自由。这比无缝衔接的恋情更为坦诚。对方是怎样的人，你最为了解。无需他人分析，你心中自有衡量标准。

## 策略核心：定位与心态调整 🧘‍♂️

上一节我们概述了基本心态，本节中我们来看看具体的自我定位策略。

如果对方分手后有了新欢，你的策略是选择与他做朋友。你可以将他视为你的备选。带着这种心态行动，你会更从容。你可以隔三差五，依据自己的心情，自然地刷存在感。

以下是执行此策略的注意事项：
*   切记不要故意搞破坏。否则前任会将问题归咎于你，认为你在刻意影响他的新关系，甚至可能刻意逃离你。
*   请放心，恋爱中的人都很敏感。你的存在本身就会为他们制造更多问题。

你必须稳住心态。通常，这类快速开始的感情，消退得也很快。此时，你的核心注意力应放在自己身上。

## 自我提升与展示 🌟

上一节我们强调了稳住心态的重要性，本节我们将聚焦于如何通过自我提升来创造吸引力。

你需要努力提升自己。必须拓展社交圈，结识几位优秀的异性朋友。你要向更好的状态发展，并将这些动态展示出来。甚至可以让人看到你拥有追求者，且追求者的优秀程度可能远超你的前任。

人都有劣根性，你可以从中利用一种心理：就像一家饭馆有很多人排队，人们就会觉得它一定好吃。当他看到更优秀的人都在靠近你时，你的好才需要通过其他优秀的人的衬托来显现，而不是指望他自己回想起你的好。

这样他才会承认你的好与优秀。加之他与新欢的体验感可能越来越差，他内心的不平衡感就会增强。那么，他后悔并回头联系你的可能性就很大。

## 复合时机与心理博弈 ⏳

上一节我们讨论了如何创造吸引力，本节我们来看看当对方回头时，应如何把握时机。

切记，不要对方刚一回头就立刻同意和好。一定要让他付出努力、追求你一小段时间。这是为了防止他轻易得到和好后又轻易离开。这个心理机制很多人并不理解。

## 总结与重要提醒 📝

本节课中，我们一起学习了对方有新欢后的复合策略。

我们首先确立了“做朋友”的定位与平稳心态。接着，我们探讨了将注意力回归自身、通过提升与社交展示更高价值的方法，利用“竞争性嫉妒”心理。最后，我们强调了在对方回头时，需设置考验期以巩固关系。

仅依赖自我提升和等待对方自主后悔，过程通常很慢。人的成长有阶段性，过程缓慢。如果你只是傻傻站在原地，等待对方某天自主后悔、想起你的好、后悔分手，那几乎是痴人说梦。他若能自主后悔，当初就不会深思熟虑后选择离开你、去接触新的人。
