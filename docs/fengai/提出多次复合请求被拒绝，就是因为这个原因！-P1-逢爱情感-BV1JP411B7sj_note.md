# 情感修复课程 01：理解拒绝是修复关系的必经之路 💔

在本节课中，我们将要学习如何正确看待在情感修复过程中遭遇的拒绝。理解这一点是成功修复关系的第一步。

## 拒绝是正常现象

上一节我们介绍了课程主题，本节中我们来看看为什么拒绝是正常的。不要认为被拒绝就意味着没有希望。在修复关系的过程中，被拒绝是再正常不过的一件事。

回想一下，你每一次让对方失望和反感时，对方承受了多少心酸和委屈。他一次又一次地原谅你，一次又一次地给你机会。结果是你没有珍惜。最后对方实在无法忍受，选择分开。如果轻而易举就能哄回来，那便是假性分手。

## 人是独立的个体

以下是关于人的独立性的核心观点。

*   对方不是你养的小猫小狗。你打宠物一顿，它可能依旧对你不离不弃。
*   但人拥有独立的思想。你让他失望的次数多了，他就会离开你。
*   不要认为你伤害了他，还希望他能一笑而过、轻易原谅你。这绝对不可能。

## 对方已给过机会

在主动提出分手之前，对方其实已经给过你很多次机会。你依旧没有把握好。因此，他完全有资格离开你。

你没有资格站在道德的制高点上批判他，要求他必须原谅你。

## 如何应对指责与拒绝

以下是修复过程中可能遇到的正常情况。

*   只要对方不上升到攻击你的父母。
*   只要对方不对你的人身进行攻击。
*   只要对方不做出毁三观的事。

那么，他骂你、指责你，都是非常正常的现象。包括拒绝你，也很正常。想一想他之前承受的那些辛酸和委屈，这些是你应该承受的。如果连这点都忍受不了，那么你或许应该换一个人。

---

本节课中我们一起学习了：在情感修复初期，遭遇拒绝和指责是必经且正常的阶段。这源于对方长期积累的失望，以及其作为独立个体的选择权。正确理解并承受这个过程，是迈向关系修复的基础。
