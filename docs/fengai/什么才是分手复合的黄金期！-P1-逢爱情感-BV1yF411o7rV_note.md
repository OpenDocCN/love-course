# 分手复合指南：把握四个关键时期 💔 - 课程01

在本节课中，我们将学习分手后复合的四个关键时期。理解并把握这些时期，能为成功复合增加可能性。很多人误以为复合越早越好，但事实并非如此。对方在分手后的特定时间段内，确实可能产生强烈的和好意愿。

## 第一时期：抵触期 😠

上一节我们概述了四个时期，本节中我们来看看第一个时期——抵触期。

如果两人因矛盾分手，对方大概率会产生抵触情绪。无论过去多么甜蜜，此刻他可能只记得矛盾和你不好的一面。你需要明白，这是情绪冲动在作怪，并非事实。感情不会说断就断。

**核心概念**：`抵触情绪 ≠ 事实`。他的反感是暂时的情绪反应。

## 第二时期：平静期 😌

当对方情绪平复，从过去感情中走出来后，抵触会转变为不抵触。这是尝试恢复联系的时机。

以下是复联前需要做的准备：

*   **心态检查**：复联前，先问自己是否控制好了情感需求，是否会操之过急。
*   **尝试复联**：从简单的问候开始，保持低频率联系（例如每周1-2次）。**必须以普通朋友身份进行，而非恋人**。
*   **展示改变**：将你在断联期间做出的积极改变，通过社交动态等方式间接展示给他看。

## 第三时期：需求期 💘

当对方彻底走出失败感情后，会对新的感情产生需求，重新想要谈恋爱。此时你更需要沉住气，因为他需要感情，但不一定需要你的感情。

在这个阶段，你可以采取以下行动：

*   **持续展示改变**：加强自身正面形象的展示，增加他对复合未来的信心。
*   **适度增加联系**：可以适当增加联系频率，聊更多话题，但**依旧不能暴露过多需求感**。

## 第四时期：权衡期 ⚖️

顾名思义，在这个时期，对方会理性权衡各方面，仔细考量与你复合的利弊。

因此，你的目标是**增加“利”的方面，减少“弊”的方面**，通过你的持续改变和理性互动，让天平向复合倾斜。

---

本节课中，我们一起学习了分手复合的四个关键时期：**抵触期、平静期、需求期和权衡期**。每个时期都有其特点与应对策略，关键在于保持理性、展示积极改变，并逐步重建信任与吸引力。
