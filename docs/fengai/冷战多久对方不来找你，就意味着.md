# 冷战多久对方不来找你，就意味着

在本节课中，我们将探讨亲密关系中的冷战现象，分析其背后的心理动因，并思考如何打破僵局以维护感情。

很多情侣在出现矛盾时，倾向于采用冷战的方式应对。有时甚至抱持“你不找我，我也不找你”的态度。许多感情正是在这种持续的冷淡中逐渐消散。那么，冷战持续多久对方不主动联系，便可视为关系结束呢？

面对这个问题，我们或许可以反向思考：如果你担心感情结束，为何在冷战时不是由你主动联系对方呢？你在顾虑什么？是面子问题，还是在关系中的地位？你希望被哄、被重视，渴望对方持续主动。

你希望拥有被宠爱、被在乎的感觉，因此不愿自己显得低声下气地去联系对方。若果真如此，这并非自尊心作祟，而是自卑情结在影响你。因为你需要通过这种方式，来确认自我价值与重要性。

**核心概念：自我价值 = 内在确认，而非外在赋予**

你的重要性，根本上应由你自己确认，而非依赖他人告知。如果在恋爱中，你寻求的是被认同、被肯定、被需要的感觉，那便表明你的内心存在某种匮乏感，你总是需要从他人那里获取力量。

请仔细思考：是你的对错、面子更重要，还是这份感情本身更重要？明明两人心里都有对方，也期望和好，却固守“你不找我，我就不找你”的僵局。请认真想想，如果谁也不愿先迈出第一步，总认为对方不主动就是不爱自己。

那么，两人的感情该如何推进呢？即便对方总是愿意扮演“低头”的角色，关系总能恢复和睦，但你自己内心匮乏的问题解决了吗？若未解决，你自身无法获得成长，这段关系迟早也可能变凉。

**核心观点：爱情 ≠ 博弈**

爱情不是一场博弈，不应总计较输赢。因为决出胜负之日，往往也是感情终结之时。

---

本节课中，我们一起学习了冷战对感情的潜在危害，剖析了其背后可能存在的自卑心理与对“被爱”的过度外求。关键在于认识到，健康的感情需要双方基于内心丰盈的主动维护，而非陷入“谁先主动谁就输”的博弈思维。成长自己，勇敢沟通，才是感情长久之道。
