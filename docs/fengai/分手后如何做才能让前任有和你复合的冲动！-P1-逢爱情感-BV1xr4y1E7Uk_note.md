在本节课中，我们将要学习如何通过有效的聊天方式，打破与前任之间的冷淡僵局，为可能的复合创造积极条件。核心在于避免错误的沟通模式，并掌握能重新激发对方兴趣的对话技巧。

许多人在分手后尝试联系前任时，常常遭遇对方态度冷淡、回复敷衍的情况。你觉得自讨没趣，对方太敷衍，慢慢的两个人又进入到了冷场。这都是因为你不会聊天的结果。

上一节我们指出了常见的沟通困境，本节中我们来看看具体有哪些聊天技巧可以帮助改善关系。以下是三个核心的聊天技巧，旨在让对话变得轻松、有趣且富有吸引力。

**第一，少问封闭式的问题。**
“在忙吗”、“多喝热水”、“早点睡”这类句子根本勾不起前任的兴趣。其实就算是普通朋友，老是问“在忙吗”就已经是特别尴尬的事。对方一般都不敢轻易回复，更何况是前任这么特殊的身份。这个时期可以聊一些与你们无关的第三方的故事。可以是最近的新闻，也可以是八卦，又或者是最近经历的有趣的事。这样可以平缓稳定地进行聊天。

**第二，多引导对方说话。**
可以聊聊自己的所见所闻，认识了什么新朋友，学会了什么新技能（最好是对方喜欢的），还有去哪里旅行（最好是你们约定过要去的），又或者是内心获得了怎样的提升。总之，就是更多的去聊自己，而不是去问对方，避免陷入一问一答的冷场之中。你要想方设法让这一时期的互动变得轻松有趣，多一点观念性的展示和感受性的话题。当然，也可以鼓励对方谈谈他的看法。

**第三，不动声色地升温互动。**
这个时候，你可以慢慢聊一聊之前你们在一起的美好往事。可以感慨，但是不要谴责。要想找错，只能从自己的身上找。如果对方积极回应你，你可以适当地说：“如果有可能的话，我希望自己可以回到那段快乐的时光。我想我已经做好了准备，不想再让那些不好的事情发生了。”

本节课中我们一起学习了重建沟通的三个关键技巧：避免封闭式提问、多分享自我以引导对话、以及适时重温美好回忆来升温关系。掌握这些方法，可以帮助你与前任建立更舒适、积极的交流氛围。
