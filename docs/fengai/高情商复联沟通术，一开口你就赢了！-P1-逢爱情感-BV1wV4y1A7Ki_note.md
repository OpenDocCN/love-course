# 高情商复联沟通术，一开口你就赢了！ - 课程 P1

## 📖 课程概述

在本节课中，我们将要学习分手后如何与前任重新建立联系，即“复联”的核心沟通技巧。复联的目的不仅是降低过去留下的负面印象，更是为了建立一个全新的、积极的印象。其中，沟通是连接彼此、升温关系的关键环节。本节课将为你详细拆解复联聊天的具体法则与策略。

## 🗣️ 第一法则：三高一低聊天法则

上一节我们概述了复联沟通的重要性，本节中我们来看看第一个核心技巧：“三高一低”聊天法则。这个法则旨在通过聊天内容、情绪共鸣和沟通节奏的调整，重塑你的个人形象。

以下是“三高一低”法则的具体内容：

1.  **高质量内容**
    *   聊天内容需要具备价值，可以是新奇有趣的见闻，也可以是有思想深度的观点。
    *   公式：**聊天内容 ≠ “吃了吗/睡了吗/在干嘛”**。
    *   每个人都倾向于关注有价值的信息。当你提供有意义的内容时，对方对你的整体印象会随之发生积极改变。

2.  **高气和吸引**
    *   寻求与对方在价值观、兴趣爱好上的共鸣点。
    *   大多数人更喜欢与三观相合、爱好相近的人相处，因为这能促进相互理解和沟通。
    *   你可以主动了解对方的喜好，这有助于你理解他/她的内心世界和行为逻辑。

3.  **高情绪氛围**
    *   在聊天中主动引导积极向上的情绪氛围。
    *   你需要保持开心、积极、正能量的人设。多使用赞同和肯定的语气，并适时赞美对方。

4.  **低频率聊天**
    *   当你们交流顺畅或感觉聊天很愉快时，可以主动选择终止当前话题。
    *   这样做有两个好处：其一，任何话题都有聊尽的时候，主动结束可以避免陷入无话可说的冷场尴尬；其二，这能给对方制造一种“意犹未尽”的感觉，就像电视剧的“下集预告”，让他/她对下一次与你的交谈产生期待。

## ⚖️ 第二要点：保持自身框架

掌握了聊天法则后，我们还需要注意沟通中的姿态。本节中我们来看看如何保持“框架性”，避免在复联中陷入被动。

保持框架意味着维护自己的原则和底线。如果对方主动发消息你就秒回，对方提出任何要求（无论合理与否）你都立刻答应，这相当于完全让出了自己的底线。

当你失去框架时，对方也会失去继续征服你、探索你的欲望，这对于挽回关系是极为不利的。

## 🎯 第三要点：跳出对方思维框架

除了保持自身框架，我们还需要学会打破对方的预期。上一节我们讨论了自我坚守，本节中我们来看看如何主动出击，扰乱对方的心理节奏。

分手后，提出分手的一方往往会对被分手者有一个心理预期。例如，对方可能预期只要你态度强势一些，你就会卑微地回头。

如果你的言行举止完全在对方的预期之内，就无法激起他/她的好奇心。你需要打破这种预期，让对方明白你并非只会死缠烂打。

特别是当对方对你态度冷漠或进行指责时，无需直接反驳，也不必沉默以对。你可以换一种出其不意的方式回应，打乱他/她的节奏。

当对方琢磨不透你时，才会产生情绪波动，从而对你产生更多兴趣。

## 📝 课程总结

本节课中我们一起学习了高情商复联沟通的三个核心要点：
1.  运用“三高一低”聊天法则，通过提供有价值的内容、寻求共鸣、营造积极氛围并控制聊天节奏，来重塑个人印象。
2.  在沟通中保持自身框架，维护原则和底线，避免因过度迎合而失去吸引力。
3.  学会跳出对方的思维框架，通过打破其心理预期、给出意想不到的回应，来重新激发对方对你的好奇与兴趣。
