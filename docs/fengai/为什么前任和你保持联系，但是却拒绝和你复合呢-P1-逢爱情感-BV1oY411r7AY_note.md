# 课程名称：分手后联系的心理分析 🧠 - P1

在本节课中，我们将要学习如何理解分手后前任与你保持联系，却拒绝复合的复杂行为。我们将深入探讨其背后的心理动机，帮助你摆脱困惑，看清行为的本质。

## 如何看待分手后的联系行为

上一节我们介绍了课程的主题，本节中我们来看看如何客观看待分手后的联系行为。

分手后，如果对方还偶尔发来问候，例如祝你生日快乐，或者寒暄几句寻求帮助，比如帮忙代购。对方也可能询问一些问题，例如关于工作或股票。再或者，对方偶尔给你的社交动态点赞或评论。

当你以为你们还有可能复合的时候，对方却后撤了。对方甚至会表现出言辞激烈的拒绝行为，例如拉黑、删除、言语攻击，或者忽然间不再回复任何消息。

## 联系行为的核心目的

上一节我们描述了常见的联系行为，本节中我们来剖析这些行为背后的核心目的。

对方这些行为不代表他还爱你，也不代表他没有放下，更不代表他想复合。这只是你的主观印象。事实上，他做出这一类行为只有一个目的：确认他在你心目中是否还占据着位置。并且，他希望以此来继续影响你的情绪。

## 前任保持联系的两种心理动机

了解了核心目的后，我们进一步分析驱动这种行为的两种深层心理需求。

以下是两种关键的心理动机：

1.  **被需要感**
    毫无例外，人都有一种需要，这种需要叫做被关注、被接纳、被爱的需要。很多人认为前任不爱自己了，那么他必然也不需要自己继续来爱他。所以，当我说他来联系是为了确认你是否爱他，为了让你继续爱他时，这些人便会非常迷惑甚至不以为然。

    我举个常见的例子。大多数人都有过被追求的经历。但是，当他的追求者突然放弃他，跟别人在一起的时候，他就会觉得无所适从，会非常失落、难受，甚至是嫉妒。他为什么会出现这些情绪？因为他的潜在想法是：**“你必须继续来爱我，你要继续当我的粉丝。”**

    这就是人的需要。需要被接纳、被关注、被爱，跟饿了需要吃饭、渴了需要喝水是一样的道理。你的前任也是如此，他也有这样的需求。他希望继续被你爱，继续被你关注，继续做你心中的唯一，哪怕他不爱你。

2.  **被爱是一种强有力的自身价值肯定**
    当我们被抛弃时，往往会产生一些自我否定的想法。失恋本身就是对人的一种价值否定的行为。你被甩了，就意味着你被他否定了。于是，你会陷入到持续的低落情绪中，你会有无意义和自身无价值感。

    相反来说，被爱就是一种强有力的价值肯定行为。你被别人喜欢，证明你是有价值的，是漂亮、有能力、有吸引力的。这会强化和完善你的良性的自我概念。

---

本节课中我们一起学习了前任在分手后保持联系的心理动机。我们明白了，这些联系行为主要源于对方对“被需要感”和“价值肯定”的需求，而非为了复合。认清这一点，有助于你摆脱情感困扰，更理性地看待双方的关系。
