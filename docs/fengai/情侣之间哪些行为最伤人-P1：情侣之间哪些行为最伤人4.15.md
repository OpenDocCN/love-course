# 情侣之间哪些行为最伤人-P1：情侣之间哪些行为最伤人4.15

在本节课中，我们将要学习在亲密关系中，哪些具体行为会对感情造成严重的伤害。理解这些行为有助于我们避免无意中伤害伴侣，从而维护更健康、稳固的关系。

上一节我们介绍了课程的主题，本节中我们来看看具体有哪些伤人的行为。

以下是四种在情侣关系中常见且极具伤害性的行为：

1.  **对伴侣态度冷淡，从不主动联系。** 这种行为表现为长期忽视对方的情感需求，缺乏基本的关心和互动意愿，会让伴侣感到自己被遗忘和不被重视。
2.  **拒绝进行真诚沟通，不愿敞开心扉。** 这意味着遇到问题时选择沉默、逃避或敷衍，而不是通过交流来解决问题和增进理解，导致隔阂日益加深。
3.  **拒绝改正自身缺点，敷衍度日。** 明知自己的某些习惯或问题影响了关系，却不愿做出任何努力去调整或改善，采取一种消极、得过且过的态度。
4.  **固执己见，将问题抛给伴侣。** 其核心逻辑是 **“我就是这样，你能接受就继续，不能接受就离开”** 。这种态度完全推卸了维护关系的共同责任，是对伴侣感受的漠视。

本节课中我们一起学习了情侣间四种主要的伤人行为，包括冷淡忽视、拒绝沟通、拒绝改变以及固执己见。认识到这些行为的危害是改善关系的第一步。
