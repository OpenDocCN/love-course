# 情感修复课程 01：当对方说“不可能了”，还能挽回吗？ 💔

在本节课中，我们将要学习如何正确理解前任在分手时所说的“我们不可能了”这类话语背后的真实心理，并探讨挽回的可能性与核心行动方向。我们将从心理学角度分析对方的立场，并为你提供清晰的行动思路。

---

上一段我们提到了分手时常见的决绝话语，本节中我们来看看这些话语的本质是什么。

不要浪费你们和好的机会。严格来讲，机会并不是对方给你的，而是你自己争取的。对方在你们相处的时候，就默默给过了你很多机会，是你错过了。你一次次的忽略他的感受，一次次的不理不睬，他已经包容你很久了。

这一次他觉得真的受不了了。而他说的“我们不可能了”、“我再也不可能回头了”诸如此类的话，其实也属于一种情绪宣泄的方式。他现在确实认为，分开可能过得会更好，所以他才会这么说。这个就是他目前的一个立场。

---

理解了话语背后的情绪和立场后，我们来看看这种立场是否固定不变。

而立场和个人认知这种东西，则是很容易发生改变的。那么我们从心理学的角度来说，如果说他真的内心当中就完全确定，你们老死不相往来了，他其实连“我们不可能”这句话，他都不会跟你说。给你多说一句，那都是浪费。

都没有必要。其实他说完这句话的时候，也是在看你的态度，和接下来的表现，以及后续关系的发展。

最后，我们来总结核心的行动公式与人性规律。

**核心行动公式可以概括为：**
**无效行动（旧模式） → 关系恶化**
