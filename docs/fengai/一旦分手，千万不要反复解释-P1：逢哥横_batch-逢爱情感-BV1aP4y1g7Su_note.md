## 分手不是突然的决定

在一起那么久的感情，他为什么说不爱就不爱了。

你不停的给他发信息，去道歉，去解释，得到的却是一句“我们不可能了，不要再来纠缠我”。

你以为他之所以这么绝情，是因为分手前你做错的那件事。

其实你不知道的是，那只是压死骆驼的最后一根稻草。

## 分手的本质是长期积累

而在这之前，你已经错过了N件事了，而且并没有发生任何的改变。

**分手 = 日积月累的怨念和不满**

你曾经对他的那些好，已经不足以让他看到和你走下去的希望了。

他心里的笔记本记载着你全部的缺点。

## 反复解释的负面影响

上一节我们了解了分手的累积性，本节中我们来看看为什么反复解释会适得其反。

所以说当你去试着挽回他的时候，要做的并不是反复的去解释。

因为反复解释传递的信息就是“我没有错，是你错怪我了”。

**传递的信息：`我没有错 = 你在无理取闹`**

这样做只会加剧对方对你的反感。

## 正确的挽回第一步：冷静与反思

以下是挽回初期应该采取的核心步骤。

你要做的就是冷静思考下那些日积月累的矛盾点在哪里。

利用好你们仅有的联系通道，让他有机会看到你的努力和改变。

**行动公式：`自我反思 -> 识别矛盾点 -> 通过行动展示改变`**

你明白了吗？
