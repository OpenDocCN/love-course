## 🧠 理解线上沟通的核心

如果你们已经经历了一段时间的断联，现在需要重新去搭建关系了，那么该怎么去发消息呢？很多人会觉得，双方这么久不联系，对方应该没有抵触情绪了，于是就想直接约线下见面。

这样想就错了。如果线上沟通都做不到，线下见面只会起到副作用。所以，线上沟通尤为重要。

聊天聊的不是话题，而是一种**氛围**。只有让对方感觉与你聊天是**舒适的、轻松的、无压力的**，他才能够进入你开启的话题。

接下来，我将从几个方面告诉你具体该怎么聊天。

## 📝 复联初期的聊天要点

以下是复联初期发消息需要掌握的四个核心要点。

### 1. 学会试探

复联的初期，前两次的聊天你要做好充分的心理准备。对方的态度可能依旧会很冷淡，聊天氛围也会很陌生。

我们这次聊天的目的，只是去试探一下对方愿不愿意回复你。可以说，就是去刷一下存在感。

### 2. 学会寻找共性

你的话题需要结合你和对方之间的事情，而不是随便拼凑一些话。

正确的开场可以用以下三种方式：
*   **询问近况**：例如问他最近工作怎么样，业绩完成如何，最近忙不忙。或者从你自己的工作出发，主动分享。
*   **回忆共同经历**：例如，“刚才路过了我们之前吃过的那家火锅店，重新装修了，感觉生意比之前还好。”
*   **请求帮助或请教**：从对方擅长的地方去请教他，或者请他帮一个只有他能帮的忙。这需要结合你们的具体情况来找话题。

正常情况下，只要最近一段时间你没有去骚扰或纠缠对方，他大概率会回复你。

### 3. 明确身份

很多人只要和对方一说上话，立马就会进入到男女朋友的角色。请认清事实：你们已经分手了，你现在是前任。

所以，一定不要再去问对方：“你还爱我吗？”、“还能再给我们一次机会吗？”、“我们真的不可能了吗？”。

你们是因为感情问题分开的，此时再聊感情，只会让他再次想起之前的不愉快，并觉得你没有任何改变。

你要记住，你们现在只是**普通朋友**。话题应仅仅局限于**工作、生活和日常**。

### 4. 控制聊天频率

我知道你肯定想天天跟对方聊天，最好还能互道晚安。但现在一定不能心急。

这里记住一个原则：**“三三聊天法”**。
*   每隔两三天联系一次。
*   如果对方回应较好，可以多聊七八个回合。
*   如果对方态度一般，就只聊两三句。

说太多容易变得无聊，让对方反感，也更容易出错。复合需要一个过程。只有先在线上给对方轻松的感觉，让他直观感受到你的成长和变化，他才可能愿意与你线下见面。

## 📌 课程总结

本节课中，我们一起学习了与前任恢复联系时如何发送消息。核心要点是：**重视线上沟通、营造轻松氛围、从试探开始、寻找共性话题、摆正朋友身份、并运用“三三聊天法”控制频率**。记住，复合是循序渐进的过程，线上建立舒适感是走向线下见面的关键第一步。
