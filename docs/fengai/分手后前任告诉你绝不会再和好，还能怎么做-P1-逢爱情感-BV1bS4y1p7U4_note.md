# 分手挽回教程 第1课：当对方说“绝不回头”时，我们还能做什么？💔

在本节课中，我们将要学习如何理解前任在分手时所说的“绝不回头”等决绝话语背后的真实心理，并探讨在这种情况下，我们依然可以采取哪些有效的应对策略。

---

大多数人都认为，分手时说了“绝不回头”的人，就真的不会回头了。但其实真的不一定。你要知道，分手是一个过程，而不是一句话。

分手后说“绝不回头”、“我们没有结果了”等等这些话，基本上都是在刚有分手念头的时候说的。这个时候给你留的时间，就是让你去彻底解决问题的。如果在这段时间里，你能够给对方信念，不去纠缠，那么对方就不至于走到心累崩溃的地步。

上一节我们提到了分手是一个过程，本节中我们来看看真正想分手的人通常会经历的心理阶段。

真正想分手的人，都会经历三个阶段：**疏远 -> 试图拉近关系 -> 又疏远**。这就是他犹豫不决、优柔寡断的表现。

那为什么会这么说呢？因为双方在一段感情里都是投入的，投入的越多越难以割舍。所以说，说了“绝不回头”不可怕，可怕的是“心死”。“心死”才是决绝的。

对你彻底心死的人，会直接消失、全网拉黑，因为他承担不了这份爱了，让你再也找不到他。而真正能回头的人，你就一定能找到他，他偶尔还能回你的信息，哪怕只是一个“嗯”或一个表情，因为他对你还有余情。

理解了对方话语背后的心理状态后，我们来看看具体应该采取什么行动。以下是挽回初期需要遵循的核心原则。

我常说，分手后想去复合，首先第一件事情就是：**先去处理好自己的情绪，再处理你们的问题**。因为只有理性，才能对抗分手之后失控的状态。

---

本节课中我们一起学习了：分手时的决绝话语并不代表最终结局；分手是一个动态的心理过程，会经历疏远与拉锯；判断挽回可能性的关键在于对方是否“心死”；而挽回行动的第一步，永远是先处理好自己的情绪，保持理性。
