# 挽回的过程中，千万不要抱着卑微：lv_0_2021022315

在本节课中，我们将要学习在感情挽回过程中一个至关重要的心态：**避免以卑微的姿态乞求复合**。我们将通过一个真实案例，剖析卑微心态的负面影响，并探讨如何转向更理性、更有效的解决问题模式。

## 卑微乞求的后果

上一节我们提到了避免卑微心态的重要性，本节中我们来看看一个具体的案例，理解卑微行为如何将关系推向更糟的境地。

我的一位咨询者，在分手后经历了巨大的痛苦。他连续十几天的夜晚都在哭泣中醒来，哭累后才能再次入睡。每天早晨，他都不敢面对镜子中的自己。

他看到的是浮肿的双眼、憔悴的脸庞和清晰的泪痕。他时常感到自己可怜，却依然选择卑微地、苦苦地哀求他的前女友。

他恳求哪怕只见最后一面，好好说说话，或者哪怕不说话，只看一眼也行。然而，他的前女友始终躲避他，态度冰冷。

他表现得越卑微，他的前女友就越是反感，仿佛他犯了什么不可饶恕的错误。但实际上，他并没有。

## 卑微心态的实质与危害

在详细了解他们的情况后，我为他指明了方向。他们之间的问题，并非靠卑微能够解决。

你知道为什么你越卑微地乞求，对方反而越气愤吗？原因在于，你的卑微行为实际上将全部责任推到了对方身上。

具体来说，你的卑微祈求，是在逼迫对方继续去面对和忍受一段令其痛苦的关系。对方会想：“凭什么我要这样？凭什么我要去面对和解决这些问题？”

这甚至会导致对方特别看不起你，认为你总是摆出一副受害者的姿态。因此，**卑微换不来怜悯**。

卑微看似表明了你的态度，实际上却将情况推向更加极端和对立的局面。

## 正确的行动方向：发现问题与解决问题

那么，我们真正应该做什么呢？核心是：**不被情绪掌控，去发现问题，并解决问题**。

因为分手的发生，本身就意味着关系中存在问题。难道不是吗？每当我们真正解决一个问题时，对方才会切实地感受到关系的改善，从而可能向我们靠近。

以下是转向理性修复关系的几个关键点：

*   **停止情绪化纠缠**：首先需要平复自己的情绪，停止一切低姿态的、情绪化的联系和乞求。
*   **客观分析问题**：冷静下来后，理性回顾关系，找出导致分手的核心矛盾与问题。
*   **聚焦自我提升与改变**：将注意力从对方身上收回，专注于解决自身存在的问题和提升自我价值。
*   **通过行动展示改变**：当准备好后，可以通过适当的、非压迫性的方式，让对方看到你的切实改变，而非口头承诺。

一起加油吧，挽回之路需要的是智慧和策略，而非眼泪与卑微。
