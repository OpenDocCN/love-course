# 分手后如何应对狠心离开的女生？💔 课程 P1

在本节课中，我们将要学习当女生狠心离开后，如何调整心态与行为，以争取在感情中占据更有利的地位，并为可能的感情新开始创造条件。

## 停止无谓的纠缠

上一节我们介绍了课程的目标，本节中我们来看看首要的行动原则：停止跪舔与纠缠。

我没有办法保证对方绝对能够回心转意，但是最起码能让你在感情中地位更高一点。让你面对一个不在乎自己的人的时候，一定要做到停止跪舔。

你之所以纠缠不是因为你多爱对方，而是因为你很缺爱。你在希望他给你更多的爱，但是你的纠缠只能感动自己。

## 认清现实，放下执念

理解了停止纠缠的必要性后，我们需要进一步调整内心认知。

不要再抱有不切实际的希望了。放下你的不甘心，不高估自己的存在，也不低估对方的绝情。

你的世界不是非他不可，与其耿耿于怀，还不如的彻底放下。

## 为新的开始创造条件

当我们能够放下执念，事情便会出现转机。

放下才能给你们的感情一个新的开始。

以下是关于后续行动的建议：
*   **关注自我成长**：将精力回归自身，提升价值。
*   **保持适度距离**：给予双方冷静和思考的空间。
*   **重塑吸引力**：通过积极变化重新引起对方注意。
