# 情感分析课程 01：解读分手后前任的持续联系 🧐

在本节课中，我们将要学习如何分析和理解分手后前任仍然保持联系这一行为背后的潜在心理动机。我们将拆解几种常见情况，并提供清晰的判断方法与应对策略。

---

## 前任联系的心理动机

很多女人分手后仍然联系你，总是很关心你的样子，给你一种她想要复合的错觉，却是从来都不谈复合。这是为什么呢？

其实，她并不是想要复合，只是想要通过关心你来减轻自己内心的愧疚感。因为她知道在这段感情里是她先对不起你，所以，她主动来关心你，在一定程度上就能够弥补自己的不安而已。

但是，如果说你没看明白，把女人的这种不清不楚的关心当作是复合，到最后你只会抱着希望被她再一次的伤害。

上一节我们介绍了前任联系可能源于愧疚感，本节中我们来看看如何判断其真实意图。

## 如何判断联系的真实意图

为什么会发生你回应或者求和之后，她就消失了呢？因为她已经确认了信息，已经看到了你依然爱她的想法，她的目的就达到了，那么她就舒坦了。

那么，我们怎么判断她只是为了联系而联系，还是为了求和呢？其实，很简单。看她后续的行动。

以下是判断其意图的关键行为列表：

*   **她消失了，拒绝了**：那么毋庸置疑，她就是在试探你还爱不爱她。
*   **她持续联系并推进关系**：这可能意味着她有复合的意愿。

当然可能是你的前任有想要复合的一个心态，并且呢，正在试探的阶段。

## 不同情况下的应对策略

根据上一节的判断，我们可以采取不同的行动。以下是针对两种主要情况的选择：

**情况一：你已无感情或存在原则性问题**

*   **选择A**：如果你对她没有感情了，那么赶紧走人。
*   **选择B**：或者说你虽然还有感情，但是你们之间出现了原则性的问题，不能够再继续下去了，那么你就咬咬牙逼自己彻底的了断。

**情况二：你仍希望复合**

第二种情况就是你还喜欢她还放不下她，依然想要在一起。那么就要逆转两个人之间的关系呀。

我之前呢，就说过有关挽回的那些方法提高吸引力，让你对她重要起来，让她时刻关注着你的动态，而不是你离不开她。核心关系公式可以概括为：

**吸引力 > 需求感**

当你卑微讨好身处低位，你满足她的一切需求，那么她就不会珍惜你的。

## 关于“做朋友”的真相

还有一种情况呢，就是她真的想要跟你做朋友。有句话说呢，真正爱过的人是没有办法做朋友的。

其实，我观察到的就是分手后做不做朋友，主要的就是看和她维持社交关系的价值成本。关系决策模型可以描述为：

**IF 维持社交关系的成本 > 她提供的价值 THEN 不做朋友 ELSE 做朋友**

如果说和她维持社交关系的成本呢，大于她的价值，那么你就不会再跟她做朋友，反之呢，则会做朋友。
