# 分手挽回教程 第1课：当对方说“早干嘛去了”时，如何应对？💔

在本节课中，我们将要学习当你在挽回前任时，对方说出“早干嘛去了，我以前需要你，现在不需要了”这类话，应该如何理解和应对。我们将深入分析对方的心境，并提供具体的沟通策略。

## 理解对方的心境

上一节我们介绍了课程主题，本节中我们来看看如何理解对方说这句话时的心情。

对方这样讲的时候，内心往往带着复杂的情绪。你需要了解到他的心情和心境，因为只有你了解了，你才能够真正走到他的内心深处去。

他在讲这个话的时候，内心是带着期待的。例如：
*   我需要你的时候，我很期待你可以陪着我。
*   我期待你可以支持、理解我，可以改掉你的坏毛病。

但是后来他发现，只要对你还有期待，就会失望，失望之后就会受伤。他会发现你根本不在乎他，也并不重视他。受伤的次数变多，一次一次失望之后，他就慢慢学聪明了。

于是他不再对别人有所期待，学会了以后要独立，要靠自己，不要傻傻地再去相信别人，不要再傻傻地去等待。所以，其实在这个过程当中，他是经历了从失望到绝望，然后又到了受伤的过程。

## 如何有效回应

理解了对方的心路历程后，我们来看看如何组织语言进行回应。

当你了解到他的这个心境的时候，你后面的语言和整个人对他的理解，才能够更加深入地走到他的内心里面去。这时，你可以这样对他说：

> “我听到你这样讲的时候，我能够猜到以前的我让你失望的次数太多了，很抱歉。我明白得太晚了，直到分手后，我才体会到你内心真正的绝望。以前每次你有需要的时候，好像我都没有真正地注意到，我也没有关注到你。而且有时候我注意到了，可是我真的是无能为力。总之，让你产生了这种痛苦的感觉，给你带来了不好的爱情体验，是我没有做好。”

你这个回答就等于是在引导着对方告诉你说：“是的，你是了解我的心情的。”那么在这个过程里面，就属于积怨清理和化解的一个过程，用到了**行为合理化**和**共情**的技巧。

这等于说你慢慢地把对方的一个死心的状态，一点一点地去给他捂热。因为了解和懂得他，本来就是一个捂热的过程。

## 面对对方的抗拒与后续坚持

当然，对方也可能有进一步的负面反应，本节我们来看看如何应对。

对方也有可能会告诉你：“你现在说这些太晚了，你要早一点懂这些的话，咱俩就不会走到今天了。”对方一定会很抗拒，重新浇灭你的希望。这是再正常不过的，他一定会不断地把你赶走，然后打击你。他不会马上就告诉你“我对你有希望”，因为这就是人的**自我保护机制**。因为他失望得太久了，他就会本能地来保护他自己。

那这个时候我们应该去怎么做呢？不要着急，也不要气馁，坚持有温度地尝试着走到对方的内心。而对于失望性的分手，咱们就必须坚持。**坚持一贯的态度**是非常重要的，甚至比改变自己更加重要。

我刚刚只是给大家示范了一句话的小例子，但是像这种事情，咱们还是要多次尝试。这个方法是在离婚谈判里面一个很重要的关键点，就叫做**积怨化解**。坚持有温度地尝试，你才能够化解对方对你的怨气，走向复合。

## 课程总结

本节课中，我们一起学习了当挽回时对方说“早干嘛去了”的应对方法。核心要点是：首先，理解对方话语背后从期待到失望再到绝望的心路历程；其次，运用包含**共情**与**行为合理化**的言语进行回应，表达理解与歉意；最后，当对方表现出抗拒时，需要保持耐心，通过**坚持有温度的尝试**来逐步化解积怨，这是一个需要持续努力的过程。
