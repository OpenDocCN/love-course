上一节我们介绍了课程目标，本节中我们来看看第一个核心技巧：展现积极的生活状态。

## 1. 过得开心，至少表现得开心 😊

许多提出分手的女性会认为，男友当下的生活品质源于对自己的依赖以及自己的付出。她们认为，如果没有自己，对方的生活会变得一团糟。在这种心态下，她们希望看到男友离开自己后生活越来越糟糕。男生的不舍会让她们更加确信自己很好，而现在的男友已经配不上自己，从而认为分手是正确决定。

当实际情况与预期相反时，即男生并未因分手而变得糟糕，女生往往会感到不适。她们会觉得自己似乎没有想象中那么重要，并产生疑问：“他明明很爱我，为什么好像没受分手影响？” 这种对自身魅力的怀疑，反而会促使她想起你的好。

**核心公式**：`实际状态 > 对方预期状态` → `引发对方自我怀疑` → `促使对方怀念你的价值`

上一节我们探讨了通过积极状态引发对方心理落差，本节中我们来看看第二个技巧：控制你在对方面前的情绪表现。

## 2. 表现得毫不在乎 😐

真正在分手时感到不甘与不舍的人，不一定是主动提分手的一方，而往往是先转身离开的那个人。有时，明明是自己提的分手，但当对方直接答应时，放不下的反而成了自己。这就像分手是对方提出来的一样，这种现象源于“损失厌恶”心理。

真正会让前任后悔的男性，往往是分手后直接转身离开的人。这种果断的离开会让对方产生“这不是我想要的反应”的感觉，甚至让对方觉得被甩的是自己。结合“损失厌恶”心理，很多人在分手后会开始想起对方的好，甚至认为对方不可替代，后悔便自然产生。

**核心概念**：`损失厌恶心理`——人们面对同样数量的收益和损失时，认为损失更加令他们难以忍受。

上一节我们分析了利用“损失厌恶”心理制造反差，本节中我们来看看最后一个技巧：引入新的比较对象。

## 3. 寻找更优质的相处对象 👫

理想状态是找到一个各方面都比前任更优秀的交往对象，例如外貌更美、学历更高、身材更好。总之，形成全方位的碾压态势。

这会使前任产生困惑：“这么高价值的人居然会选择我的前任？原来我的前任这么棒！” 当吸引力再次形成，前任的价值在对方心中重新提升时，后悔的往往是那个当初提分手的人。

这种感觉类似于逛街时，你看不上一件物品，却有一个各方面比你优秀的人对它赞不绝口并抢着购买。此时，你不会再认为那件物品没有价值，而是会怀疑自己的眼光。

**核心逻辑**：`引入高价值参照系` → `抬升自身在对方心中的价值定位` → `引发对方的后悔情绪`
