# 如果女人绝情和你提出分手，你能

在本节课中，我们将要学习当面对伴侣绝情提出分手时，如何通过三个关键的心理调整策略来保护自己，并赢得情感上的主动权。这些策略的核心在于“狠心”，即果断的心理切割与自我重建。

上一节我们介绍了课程的主题，本节中我们来看看第一个关键策略。

## 第一：狠心放弃，重新开始 🚶‍♂️

当女人表现出绝情并冷落你时，许多人会为她寻找各种借口。例如，认为她太忙、心情不好或有难言之隐。然而，这些都不应成为对方持续冷落你的正当理由。

因为**真正的爱意味着在任何时候都相互依靠**。自始至终，你都应该是她的首选，而不是被她排在所有事情之后、偶尔才会想到的选项。因此，一旦女人开始冷落你，甚至长时间不联系，这本身就表明你在她心中已不再重要。

至少，你不值得她花费时间来维持关系。

了解了果断放弃的重要性后，接下来我们需要处理内心的期待。

## 第二：狠心放下期待，停止幻想 📉

真正放下一个人，并非指删除所有联系方式，而是指彻底放下任何关于复合的期待，让内心恢复平静。但很多男性难以做到这一点。尽管他们深知对方态度绝情，自己早该离开，却仍舍不得过去的记忆和已付出的努力。

他们还在期待两人之间能有全新的开始。

以下是执行此策略的关键点：
*   如果女人冷落你，而你也决定放弃，就不要再对双方关系抱有任何期待。
*   必须从心底快刀斩乱麻。
*   这样做才能为未来的美好腾出心理空间。

放下了对未来的期待，我们还需要处理对过去的执念。

## 第三：放下过往美好，停止自我强迫 🕰️

当一段感情走向结束时，最让人不舍的往往不是那个人本身，而是两人共同经历过的种种美好回忆。然而，你必须明白，这些美好会随着日复一日的回忆而渐渐褪色。

有时，正是因为你现在过得不好，才将过去看得过于重要。当你走出来，经历更多人与事，便会发现过往的美好也不过如此。

以下是需要认清的两个事实：
*   即使你今后再也无法经历同样的事情，但与此同时，你也不会再受到同样的伤害。
*   这个世界上，没有谁真正离不开谁，很多时候只是我们不敢尝试迈出那一步。

本节课中我们一起学习了应对绝情分手的三个“狠心”策略：狠心放弃并重新开始、狠心放下所有期待、以及狠心放下过往的美好。只要我们敢于向前迈出一步，前方便是海阔天空。若执意留在过去，所有的痛苦便成了自我的选择，因为你本可以离开。
