# 分手后感情依然很深，但关系很僵怎么办？ - 课程01：理解僵局与心态调整 🧠

在本节课中，我们将要学习分手后双方仍有感情但关系陷入僵局的核心原因，并探讨如何调整心态，为可能的复合创造健康的基础。

## 僵局为何产生？

分手后两人都还有感情，但关系很僵，不知道该怎么办，这其实很正常。这种情况通常源于沟通的破裂和情绪的对抗。

上一节我们介绍了僵局的普遍性，本节中我们来看看具体原因。

以下是导致这种僵局的常见原因：

*   可能是你们两人当中，某一个人或两个人都说了狠话。
*   也可能就是因为气不过，然后一方苦苦哀求，另一方就是不给机会。

## 关键：避免消耗最后的欲望

在这种情况下，双方既没有断联，也没有互删联系方式，那该怎样去争取复合呢？你一定要记住，不要乱来。

因为乱来很容易把你在前任心里的最后那点欲望也给消耗掉。你要明白，分手并不是结束，但是分手后你一系列的作死行为，真的有可能导致你们真正的结束。

## 负面纠缠的后果

对方需要时间，那你就给他时间。如果这个时候，你还是没有办法控制住自己的情绪，然后在分手的问题上不断的继续纠缠，甚至委曲求全。

以下是纠缠行为的具体表现及其传递的信号：

*   不断地道歉。
*   卑微地讨好。

那这个时候，你传递出的情绪价值就全部都是负面的。你可以换位思考一下，如果是你提出的分手，对方用这样的方式来纠缠你，你会怎么想呢？

**核心心态公式：卑微纠缠 → 传递负面价值 → 降低自身吸引力**

你是不是会觉得他没有你就是不行，然后你会有一种莫名的优越感。那如果情况是这样的话，你还会想跟这样的人继续在一起吗？你心里面关于他的形象，是不是又降低了很多呢？你会因为同情他、可怜他就跟他复合吗？并不会。

## 调整心态：拒绝卑微

所以说，如果你们两人还存在着感情，想要争取复合，就一定不要去卑微祈求。

我们在跟前任联系的时候，其实也不仅仅是表面问题，心态上面也很关键。其实那些所谓的高位的人，有的时候并不是真的高位，而是因为你太害怕失去了，把他们捧成了高位。

**心态调整代码：**
```python
# 错误的心态
my_status = “卑微的”、“害怕失去的”
their_status = “高位的”、“被仰望的”

# 健康的心态
my_status = “平和的”、“独立的”
their_status = “平等的”、“普通的”
```

说白了很简单的道理，没有人会愿意跟那些整天有负面情绪的人在一起的。

---

本节课中我们一起学习了分手后关系僵局的成因，明白了卑微纠缠只会消耗感情并降低自身吸引力。关键在于控制情绪，停止传递负面价值，并以平等、平和的心态去看待彼此关系，这是走向任何可能未来的第一步。
