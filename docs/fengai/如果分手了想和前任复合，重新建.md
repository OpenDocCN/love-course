# 复合教程 01：重建舒适感的聊天原则 💬

在本节课中，我们将要学习分手后如何通过聊天重新建立舒适感，这是复合过程中至关重要的一步。舒适感是关系升级的基础，而聊天则是建立舒适感的主要途径。我们将通过一系列具体原则，帮助你避免错误，掌握正确的沟通方法。

## 聊天的重要性与舒适感

分手后的联系，90%以上是通过聊天进行的。见面机会通常很少，因此聊天成为复合中最重要、最频繁的环节。

我们知道，挽回的基础是建立舒适感。舒适感主要通过聊天来建立。这一点非常关键。如果没有舒适感，对方会持续沉浸在分手前对你的不满情绪中。

这会导致对方无法与你进行更深入的交流和沟通。

## 聊天中的“三不聊”原则

以下是聊天中应避免的三大类话题。遵循这些原则，可以防止破坏刚刚建立的微弱联系。

1.  **别聊感情，别提过去**
    避免说“我爱你”、“你是我的唯一”、“这辈子非你不可”等类似的话。这类话语类似于讨好、承诺和发誓，在现阶段不宜提及。

2.  **别聊废话**
    避免发送“吃了吗”、“早安晚安”、“在干嘛”、“我想你了”等信息。你们已不是情侣关系，这类不痛不痒的话对方可能连看都不想看。

3.  **不要频繁道歉**
    “对不起”不是靠说出来的，而是要做出来的。对方已经对你不信任，过多的道歉只会增加他的厌恶感，让他觉得你缺乏诚意、不够真诚。

## 聊天中的“三聊”原则

上一节我们介绍了不该聊的内容，本节中我们来看看应该如何正确聊天。以下是三个核心的聊天原则。

**第一，找好时间，控制频率与长度**
当你们处于关系排斥期时，不要主动聊天。如果时间点可能影响对方的生活或工作，也不要聊。应选择对方忙完、一个人静下来的时间，例如晚上9点到11点，这个时段也更容易引发情感共鸣。同时，要关注对方的动态，用具体的事情开启聊天，这样内容不会空洞。聊天时不要恋战，不要滔滔不绝，更不要每天频繁骚扰。务必把握好聊天的时间和频率。

**第二，聊天内容要有目的性**
聊天内容应遵循一个核心原则：要有目的性。这个目的就是，在聊天中委婉、间接或侧面地展示你的变化。重点在于，千万不要直接从你嘴里说出来。你可以展示生活、个人性格、脾气、情商、工作、兴趣爱好等方面的积极变化。任何有利于对方改变对你看法、形成崭新印象的内容都可以。这样对方会觉得，之前的问题可能正在慢慢解决，你和分手前已经不一样了。这会促使他不断重新评估你，认为复合利大于弊，未来会更好，从而强化他对你的期待感。最后，在恰当时机便能一举成功。

**第三，聊对方感兴趣的话题**
你不能总是间接地聊自己的变化。你需要聊对方感兴趣的内容，关注并认可对方。找到他的兴趣点和需求，投其所好，聊他感兴趣的话题。这样才能逐步建立起良好的舒适感。

## 总结

本节课中，我们一起学习了分手后通过聊天重建舒适感的核心原则。我们首先明确了聊天在复合中的关键作用，然后详细阐述了“三不聊”（不聊感情、不聊废话、不频繁道歉）与“三聊”（控制节奏、展示变化、投其所好）的具体方法。记住，舒适感是关系升级的基石，而说合适的话、避免踩雷是建立舒适感的第一步。坚持这些原则，能为后续的复合行动打下坚实基础。
