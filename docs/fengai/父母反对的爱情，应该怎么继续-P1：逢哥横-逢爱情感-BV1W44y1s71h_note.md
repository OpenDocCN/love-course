父母反对的爱情是否应该继续？这个问题无需由他人给出答案，你可以通过向自己提问来寻找方向。

以下是几个需要你认真思考的问题：

第一，与他共度一生的人是你，而非你的父母。

第二，父母提供的是建议，并非命令。

第三，必须明确长辈反对的具体原因。他们是否因为某些**偏见**，或是由于对你**缺乏了解**，从而对未来感到担忧？

第四，你们的父母无疑是爱你们的，他们期望你们能获得幸福。然而，如果你们确信彼此**真心相爱**，那么必须向父母展现出你们共同面对未来的**决心**。
