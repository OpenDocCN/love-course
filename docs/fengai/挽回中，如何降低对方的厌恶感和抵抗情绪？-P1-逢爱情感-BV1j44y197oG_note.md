# 挽回教程 01：如何降低对方的厌恶感和抵抗情绪？ 💔

在本节课中，我们将要学习在感情挽回过程中，如何有效降低对方已经产生的厌恶感和抵抗情绪。我们将从理解厌恶感的成因开始，逐步探讨具体的应对策略和心态调整方法。

---

## 厌恶感的形成并非一日之寒

讨厌是突然才会出现的情绪吗？事实上并没有这么极端。除非你做了反差感极大的事情。人与人之间的交往，在开头就会进行第一轮的筛选。也就是说，喜不喜欢，第一眼就有感觉。只要没有必要的联系，一般也不会去刻意维系一段复杂的关系。

所以，恋人之间的厌烦情绪，大部分情况都是日积月累的。

---

## 剖析厌恶感的根源

对方为什么讨厌你？其实大部分原因，你心里多少都有数。透过现象看本质，剖析一下深层次的问题，你就能得到想要的答案。

如果你无法详知对方讨厌的原因，那么你可以问问身边的人对你在这段感情中表现的看法。那多半都是能够了解的。

**核心行动公式：**
`厌恶根源 = 自我反思 + 外部反馈`

---

## 第一步：冷却关系，暂停讨好

无论出于怎样的原因，你都需要先用最快的时间去冷却关系。这里的“冷却”不是说让你立刻就进入断联状态，而是立刻暂停讨好的行为。

你要明白，极力让一个人喜欢，那对方就越不属于你。不要反驳说“那只是好好跟对方说话，怎么算是讨好”。当你了解一个人讨厌你的时候，你还凑着往前去，这就是讨好。

正常人如果在路上有人骂了你，你要么置之不理，要么怒怼回去。对于远去的爱人，你也应该用正常的思维去修复感情，才能起到复合的效力。

**关键心态代码：**
```python
if 对方表现出厌恶:
    行动 = 暂停接触()  # 非断联，而是停止主动讨好
    心态 = 保持正常()
else:
    继续观察()
```

---

## 第二步：建立稳定的个人框架

也就是说，要有自知之明的同时，更要有稳定的框架。一个人的框架越稳定，他的边界意识才会更强。框架其实就是我们所说的底线和原则。

修复感情，不是钻到对方的脑子里说“我没有那么不堪”，或者打骂斥责对方“有眼无珠”。而是应该做到平和的理解和尊重。

以下是建立框架的三个要点：

*   **理解对方**：理解对方讨厌你的情绪是合理的。
*   **尊重选择**：尊重对方选择分离的决定。
*   **保持平和**：只有尊重对方的决策，对方才会善意地回忆起你们的过去，你也能够更好地深度反思自己。

---

## 第三步：理性评估与投入

讨厌我的这个人，值不值得我去重新追回？追回他的这段时间，我到底应该投入多少精力？

注意，投入时间的长短，不等于投入的程度。感情修复对于时间来说肯定是有要求的，过分计较短平快，反而会使你们的疏离感更强。

你的目标性越强，暴露的需求感就越深刻。而几乎所有人都害怕被索取与消耗，所以不可以暴露需求感。

情感里的拉扯不是硬性的拖拉硬拽，这样可能导致更深刻的厌倦。试想一下，如果你身上有一块美玉，来和你交朋友的人不是因为你的个人魅力，而是为了这块玉，那么你还想和他继续建立关系吗？相反的，对方越热情，你就想离他越远。

**投入评估清单：**
*   对方的核心价值是否依然吸引我？
*   导致关系破裂的问题，是否有解决的可能？
*   我准备投入的精力，是否会影响我的正常生活？
*   我的目标是修复关系，还是仅仅害怕失去？

---

## 总结

本节课中，我们一起学习了如何降低挽回中对方的厌恶感。

我们首先认识到厌恶感是日积月累形成的，需要冷静剖析根源。随后，我们明确了第一步是冷却关系并停止讨好行为，第二步是建立稳定、有边界的个人框架，通过理解和尊重来为沟通创造空间。最后，我们探讨了如何理性评估这段关系与自己的投入，避免因暴露过度需求感而将对方推得更远。

记住，挽回的基础是修复一个更好的自己，并创造一个让对方愿意重新靠近的环境，而非强行拉拽。
