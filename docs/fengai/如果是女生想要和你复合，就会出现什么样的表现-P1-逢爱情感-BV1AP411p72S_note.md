在本节课中，我们将学习如何识别女生在分手后可能仍想复合的潜意识表现。了解这些迹象，可以帮助你避免误判形势，错失修复关系的机会。

上一节我们介绍了课程主题，本节中我们来看看具体的表现特征。

## 1. 反复拒绝但期待转变

女生可能仍会不断拒绝你，表面行为是在推远你。但其内心是希望看到你的转变，期待你表现出不离不弃的态度。在双方的核心问题解决之前，她可能会选择躲避。

**核心认知**：问题终将被解决，她最终会重新面对你。你需要做的是耐心寻找问题根源，并等待时机。在此之前，对方表现出的所有绝情行为，大多受当下情绪影响，或可理解为一种对你的考验。

**关键点**：一时的拒绝并非真正的拒绝，可能只是对方不知如何面对。因此，你不应自暴自弃。记住公式：**真正的放弃 = 自我放弃**。

## 2. 埋怨与数落

以下是对方可能出现的言语表现：

对方会开始各种埋怨和数落你，指责你的种种不好，表面显得厌恶和烦躁。但这背后的逻辑类似于“嫌货才是买货人”。她指出你的不足甚至攻击你，往往是希望你做出改变，期待看到一个更好的你。

大多数女生在情感表达上存在口是心非的现象，无论是恋爱期间还是分手之后。

## 3. 删除或拉黑联系方式

很多男性在被删除或拉黑后会感到慌张，认为机会全无，进而可能开始纠缠、或陷入迷茫想要放弃。这反映出在情感认知上需要提升。

一个人选择删除或拉黑你，通常是因为她担心看到你的消息或动态时自己情绪失控。你的言行依然能影响她的情绪，这反而是一个积极信号。最糟糕的情况是对方对你无动于衷，你的任何行为都无法再引起她的情绪波动。

换言之，只有真正在乎你、爱你的人，才会想到用拉黑删除来保护自己或控制情绪。因此，不要轻易放弃。重复核心公式：**关系终结的唯一条件 = 自我放弃**。
