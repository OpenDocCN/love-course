## 感性思维与记忆留存

## 真心想分手后的心理状态

以下是这种情况下常见的感情发展路径：

3.  **好感消耗**：由于起初就没有很中意你，进入亲密关系后，你的缺点又逐渐暴露。所以导致好感度越来越低，她越来越看你不顺眼。
4.  **关系终结**：逐渐消耗掉心中仅存的好感后，她会觉得你做什么都是错的。当嫌弃大过于好感时，她就会主动提出分手，因为她已经处于想要逃脱感情的状态，不想继续下去了。

她心里很清楚，分手的本质原因就是她不爱了，不是你做的不够好。虽然你也有缺点，但是也是真心喜欢她的。只是她没有办法回应同等的感情，所以最终只能辜负了。

这种愧疚感过后，她就会愿意跟男方做朋友。同时发自内心的希望男方过得好与幸福。
