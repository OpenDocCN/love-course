# 分手复合教程 01：不互删是绝佳的复合窗口 💔➡️❤️

在本节课中，我们将要学习如何利用分手后未互删联系方式这一关键窗口，通过改变自我和重塑吸引力，科学有效地推进复合进程。

## 概述：未互删是绝佳的复合机会

分手后只要双方没有互相删除联系方式，复合的成功率就非常高。相比被删除或拉黑的情况，保留的联系方式是一个绝佳的沟通渠道、个人展示渠道，更是一个绝佳的复合窗口。很多人未能认识到这个机会的重要性，从而把握不住。

上一节我们概述了未互删窗口的重要性，本节中我们来看看为什么对方不删除你，以及这背后的心理。

## 对方不删除你的心理分析

我们换位思考一下。如果对方立场坚定，认为彼此就此别过、永远分手、绝无可能，并且是经过理智而坚决的思考后下的决心，那么他一定会断绝所有联系方式，让你根本无法找到他。

**核心逻辑**：`未拉黑删除 ≠ 心门完全关闭`

但凡没有拉黑删除，说明他的心门一定没有完全关上。他的内心多多少少对你仍有期待，期待着你的真诚回归。

那么，什么是对方期待的“真诚”呢？那就是**分手前他担心的问题不会再发生**，仅此而已。

然而，很多朋友不但没有解决对方担忧的问题，反而变本加厉地纠缠、打扰，甚至进行骚扰和破坏。这种行为模式大概率会导致你被删除拉黑。即使未被删除，对方也会对你冷漠绝情到冰点，让最后的希望近乎破灭。

了解了对方心理后，我们来看看如何利用这个窗口采取行动。以下是两种主要情况的应对策略。

## 复合行动指南：两种情况与策略

以下是针对“未删除”和“已删除”两种情况的复合核心策略。

### 情况一：未删除拉黑

通过保留的联系渠道，可以改善我们在对方心目中的刻板印象。你需要通过视觉效果（如朋友圈）、语言文字（如消息）来展示一个关键信息：**现在的你已经不会再出现分手前的问题，并且在其他方面也变得越来越好**。

**核心操作**：`持续展示积极改变 + 坚持 = 重建吸引力`

只要让对方持续看到这一点，并且坚持下去，和好是必然结果。这里的关键点在于**如何沟通**和**如何展示**，以及对节奏的把握。

### 情况二：已被拉黑删除

即使被拉黑删除，底层逻辑也是一样的，只不过所需时间可能更久。因为他刚拉黑删除后，对你形成了更大的负面刻板印象，短时间内对你的情绪全是负面的。

**核心策略**：`耐心等待 + 间接展示改变 = 重建联系窗口`

这其实很简单。你只需要耐心等待一段时间。无论他是否有新欢，只要你们真的有感情基础存在过，在此期间，你就需要通过间接的视觉效果（如共同朋友、社交平台）来展示你的变化。目的是让对方再次主动加回你，或者在你申请加回对方时，他能看到你的积极变化。这样对方才会重新产生期待，才有继续和你聊天的动力，才会同意恢复联系。

最后，我们来总结一下复合最根本的底层逻辑。

## 总结：复合的底层逻辑

本节课中我们一起学习了利用未互删窗口进行复合的策略。无论哪种情况，复合的底层逻辑始终是：**改变自己，重塑吸引力**。

**终极公式**：`复合 = 吸引回来 ≠ 乞求回来`

需要牢记的是，以上所有策略都有一个重要前提：**你们真的有感情基础，并且对方是真的爱过你**。
