## 第一点：面对现实，不纠结不纠缠

上一节我们介绍了课程的整体目标，本节中我们来看看具体的第一点：面对现实。

当女人突然不再联系你，通常有两种情况。第一种是她想吊你的胃口，让你主动。第二种是她已经受够了，打算离开。无论是哪一种情况，这样的女人都不值得你托付，也不值得你继续花费时间。

无论你多么爱她，都必须面对现实，并看淡这段关系。一个男人只有在感情上做到不纠结、不纠缠，才算是一个感情成熟的男人。这能让你在感情的道路上越走越顺，也越走越幸福。否则，你只会伤害自己，让自己痛苦。

你要明白一个道理：世界上两条腿的癞蛤蟆难找，但两条腿的活人多的是。不必在一棵树上吊死，何况这棵树可能还是一棵歪脖子树。

**核心心态公式：**
```
情感成熟度 = 不纠结 + 不纠缠
未来幸福概率 ↑
```

## 第二点：直面自我，不自轻不自贱

了解了如何面对现实后，我们接下来探讨如何面对自我。

当女人不联系你时，男人很容易想不开，打击也很大。因为男人也是感性的动物，一旦爱上女人，很容易沉迷其中，无法自拔，从而“沉醉不知归路”。如果男人陷入这种状态，显然是错误的。

高情商的男人面对女人的狠心离去，不会选择逃避或看轻自我，而是能够做到直面自我，不自轻不自贱。你要知道，出现这种不堪的结果，往往不是你的错，而是对方的错。何必拿别人的错误来惩罚自己？

所以，直面现实，看清对方，放过自己，才是最明智的选择。

## 第三点：不抛弃不放弃，拥抱新未来

在学会了直面自我之后，本节我们来看看如何积极面向未来。

面对那个选择抛弃你的女人，高情商的男人会先让自己冷静下来，仔细分析前因后果。如果认定是对方不想继续这段感情，那么就勇敢接受。同时，及时从这段感情的阴影中走出来，让自己继续勇往直前，而不是从此感到失落和逃避。

要做到不抛弃不放弃。对未来的感情保持开放，鼓起勇气重新拥抱属于自己的幸福。

以下是实现“不抛弃不放弃”的三个步骤：
1.  **冷静分析**：理性复盘关系结束的原因。
2.  **接受结果**：承认并接纳关系已结束的事实。
3.  **重新出发**：调整状态，积极迎接新的生活与可能。
