# 这种情况95%的人都复合了-P：这种情况95%的人都复合了4.

在本节课中，我们将要学习一种在情感咨询中观察到的现象：当分手后出现某些特定行为时，两人的复合概率非常高。我们将逐一解析这五种情况，并理解其背后的心理逻辑。

上一节我们介绍了本课程的主题，本节中我们来看看具体是哪五种情况。

以下是分手后复合概率较高的五种具体行为表现：

1.  拉黑或删除了你所有的联系方式。
2.  不接你的电话，也不回复你的消息。
3.  在短时间内快速开始新的恋情（有新欢）。
4.  故意躲避，不和你见面。
5.  分手后表现得非常绝情。

---

## 情况一：全面切断联系 🔌

当对方选择拉黑或删除你所有的联系方式时，这通常是一种强烈的情绪反应。这种行为背后，往往是为了避免自己心软或再次受到情绪波动的影响。这并不一定代表厌恶，而可能是一种**自我保护机制**。公式可以表示为：
**激烈行为 = 强烈情绪 + 防御心理**

## 情况二：拒绝一切沟通 📵

不接电话、不回消息是“全面切断联系”的延伸行为。这表明对方正在主动地制造距离感，试图让双方冷静下来。从另一个角度看，**需要用力回避的事物，通常也是内心在意的事物**。

## 情况三：快速建立新关系 💑

在分手后短时间内快速有新欢，是一种常见的“反弹式关系”。这通常是为了填补情感空缺、转移痛苦，或者向自己和外界证明“我过得很好”。这种行为恰恰暴露了对方并未真正从上一段感情中释怀。其心理动机可以概括为：
**新欢 = 情感替代品 + 证明工具**

## 情况四：刻意回避见面 🚷

故意躲避不见面，与切断线上联系同理，是物理空间上的隔离。这说明对方害怕见面会动摇自己的决定，或者引发不可控的情绪。**回避的强度，有时与在乎的程度成正比**。

## 情况五：表现出极度绝情 😶

分手后表现得非常绝情，说出狠话或做出冷漠行为，这常常是“理智”战胜“情感”的结果，或是为了让你更快死心。真正的绝情是平淡和无视，而非激烈的拒绝。代码可以类比为：
```python
if 表现 == “极度绝情”:
    真实情绪 = 可能为“强烈波动” # 用冷漠伪装
else:
    真实情绪 = 可能为“真正平静”
```

本节课中我们一起学习了分手后复合概率较高的五种典型行为：全面断联、拒绝沟通、快速新欢、躲避见面和表现绝情。理解这些行为背后的情绪动机和防御心理，是迈向有效情感修复的第一步。记住，这些激烈的表现往往意味着情绪并未终结，而是进入了另一种处理阶段。
