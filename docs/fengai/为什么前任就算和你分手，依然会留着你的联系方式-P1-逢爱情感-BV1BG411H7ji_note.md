# 情感修复教程 第1课：为什么分手后前任仍保留联系方式？ 💔

在本节课中，我们将要学习一个常见的情感困惑：为什么前任在分手后，依然会保留你的联系方式？我们将从人性本质和情感需求的角度，剖析这一行为背后的深层逻辑，并为你提供清晰的思考方向。

你可能听过前任说这样的话：“你不要对我这么好，我很有压力。”或者“我知道你对我很好，但是我们真的没有可能了。”“我知道你真的很爱我，但是我们有缘无分。”“你不要这样对我好了。”等等。

这些话总是在你努力付出、试图修复关系时，给你带来沉重的打击。

我知道你很伤心，也很难过。但我想要问你一个问题：为什么他会对你如此绝情？

众所周知，人性贪婪，本性就是趋利避害。没有人会拒绝对自己更有利的人生。

更没有人会主动拒绝爱。大家都想找一个对自己好的、可以和自己共度余生的人。

而对于前任而言，那个对他好、爱他的人不就是你吗？

所以你到底有没有想过，为什么他会反人性地去抗拒你、拒绝你呢？

接下来的话可能会颠覆你的认知，但请你仔细听。

**证明你爱不爱他、对不对他好，对于复合甚至爱情的建立，没有决定性的作用。**

在爱情中，没有人会拒绝爱他、对他好的人，这是人性，没有错。

但正是因为这是人性，所以“爱他、对他好”只是爱情的基础。有才正常，没有才不正常。

你不爱他，你不对他好，你不为他付出，他怎么可能会跟你在一起呢？

所以，决定他们选择的不是这些基础项，而是**加分项**。

你拼尽全力去付出一切、去试图复合，到头来换回的全是基础项。他为什么要跟你复合？

那么他为什么不选择那个既有基础项，又有加分项的人呢？

所以你还在纠结要不要给他送个礼物，你还在自责上次吵架没有顺从他的意思。这一切都没有用。

你要想一想，最开始你们在一起的时候，他是因为什么爱上你的？

虽然爱是无条件的，但爱上一个人总有原因。

肤浅一点，比如说：**颜值好、身材好**。

深度一点，比如：**你的三观、你的幽默、你的性格**。

事实上，很多人分手就是因为这个“加分项”在相处的过程中慢慢地流逝掉了。

有很多男生在刚开始恋爱的时候，包容宽广，像是一个可以为女生解决一切困难的超人。一段时间之后，却变得小气又计较，对女生的要求视而不见。

有多少女生刚开始恋爱的时候，像一个永远温柔的天使。在恋爱了一段时间之后，却作天作地，一言不合就把分手挂在嘴边。

所以，你要去找那些曾经让她痴迷、但在恋爱中流逝掉的“加分项”，然后重新呈现给她。

告诉她，她所爱的一直都在。

然后将这些曾经拥有和现在创造出来的“加分项”展现给她，你才能够成为她眼中那个不止爱她、对她好，还是那个值得她为你付出的人。

那么，她才有可能重新爱上你。

---

本节课中，我们一起学习了分手后前任保留联系方式背后的心理机制。核心在于理解：**基础的“对ta好”是关系的入场券，但真正决定吸引力和选择的，是那些独特的“加分项”**。修复关系的关键，在于找回并重塑那些曾经吸引对方、却在相处中遗失的闪光点。
