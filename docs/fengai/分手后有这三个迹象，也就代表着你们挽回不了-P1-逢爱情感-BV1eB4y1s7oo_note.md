在本节课中，我们将要学习如何识别一段分手后可能无法挽回的关系。通过分析三个关键迹象，帮助你认清现状，避免在无效的努力中消耗情感与时间。

## 第一个迹象：恐惧与犹豫 😨

上一节我们介绍了课程概述，本节中我们来看看第一个关键迹象——恐惧。

说个扎心的，注定挽回不了的三个迹象，第一个就是害怕。你打内心就害怕自己挽回不了，害怕永远失去他，自己又没信心。因为你害怕做错，害怕把情况变得更加糟糕，所以就一直拖着，一直耗着，不敢轻举妄动。

你明明知道这样下去只会渐行渐远，但你就是做不出来任何能够推进关系的操作。而且你也不知道该怎么做。到底能不能挽回，其实跟你的信念有很大关系。就最基本的就是，你一定要相信你的感情，相信你自己。

**核心概念可以概括为：**
**挽回可能性 ∝ 个人信念强度**
即，你的信念越坚定，采取有效行动的可能性就越大。

## 第二个迹象：全盘自我否定 😔

了解了恐惧如何阻碍行动后，我们接下来探讨第二个更具破坏性的迹象——自我否定。

第二个就是否定。否定自己，把分手的原因大包大揽的全怪自己身上，开始自我怀疑，甚至觉得自己就是个烂人，根本就配不上对方。同时也觉得改变提升又很难，又需要时间，就想放弃。

所以陷在“我很差劲”的情绪中无法自拔。然后对方有了新欢，就感觉自己哪哪都比不过新欢，更没希望了。那说实话，那他当初选择跟你在一起，你一定身上有吸引得到他的点，你也是很优秀的。有时候其实能不能挽回，靠的就是一口气。你整天泄气，这复合成功率再高也都被你给弄没了。

以下是陷入自我否定时的典型思维循环：
1.  将分手原因全部归咎于自身。
2.  产生强烈的自我怀疑与贬低。
3.  认为改变过于困难而想要放弃。
4.  与假想敌（如新欢）比较，进一步丧失信心。

## 第三个迹象：拒绝接纳新信息 🚫

面对自我否定，我们常常会固守旧有模式。这便引出了第三个迹象——拒绝接纳。

然后第三个就是拒绝。什么意思呢，就是人在面对超出自己认知外的信息，第一时间就是反抗，来维护自己的认知。就比如说我告诉你，不要给他发消息，你就会想，那我不给他发消息，他不就把我忘了吗。那我告诉你，你给他发什么消息，他都没有一个很好的反馈，那你就会觉得，发信息也没有用啊，我发这有什么意义。就是说很多时候，接纳也意味着进步。

**这种行为模式在代码逻辑中可以表示为：**
```python
if 接收到的建议 != 原有认知:
    反应 = “反抗” # 拒绝接受新信息
else:
    反应 = “接纳” # 停留在舒适区
```
