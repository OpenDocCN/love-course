# 情感修复教程 第1课：分手初期的心理机制与应对 🧠💔

在本节课中，我们将要学习分手初期常见的心理状态，理解其背后的心理学原理，并探讨为何冲动挽回往往适得其反。掌握这些知识，是迈向理性处理情感问题的第一步。

经历过分手的人都明白，分手的初期是很痛苦的。这种痛苦会促使你第一时间想要去挽回，而不是去好好想一想究竟哪里出了错，以及我们为什么要挽回。

## 理解“负与效应”与“损失厌恶” 🤔

上一段我们提到了分手初期的痛苦感，本节中我们来看看这种感受背后的心理学原理。其实在心理学中，这叫做**负与效应**。

这种效应体现的心态是：你没有得到一个东西的时候，你可能并没有那么想要得到。然而当你真正拥有后，突然将这个东西从你身边抽离了，你就会非常不舍。因为大多数人对损失和获得的敏感程度是不对称的。

面对损失的痛苦感要远远大于获得的愉悦感，在心理学上也叫做**损失厌恶**。

## 情感关系中的具体表现 💑

结合两性关系来说，就是“曾经有一份真挚的感情摆在我面前，我没有去珍惜，等到我失去的时候我才后悔莫及”。《大话西游》里的这句经典台词你们应该都听过。

那么恰恰大部分人他意识不到这种效应的存在，所以会被负与效应牵着鼻子走。以下是其具体的行为表现：

*   分手后在痛苦感的驱使下，第一时间会想要去挽回。
*   不会思考为什么你们会分手。
*   这些前因后果根本就不会去想。
*   也不会去从长计议。
*   只会在自己的情绪支配下做出一些不可避免的行为。

## 冲动行为的后果与正确心态 ⚠️

了解了常见的错误行为后，我们来看看这些行为会带来什么后果。就算是挽回，大家都知道，大家呢也都做过。你做出这种极端的行为只会增加反感，将对方越推越远。

当然了，也存在一部分人在分手初期会有一种如释负重的感觉，自身状态还不错。但是好景不长的是，分手的痛苦感会慢慢地占据他的大脑，就像喝醉之后醒来头痛。

因此，基于以上分析，我不建议分手后马上就尝试联系对方。因为这时候的你是被情绪所操控的，很难理性地去做出一些分析和判断。

**正确做法**：在采取任何行动前，先给自己一个冷静期，让情绪平复。
