# 逢爱情情感课程 P1：空窗期——对过往感情的基本尊重 💔

## 什么是空窗期？

上一节我们提到了“空窗期”这个概念，本节中我们来看看它的具体含义。空窗期特指在一段感情正式结束后，开始下一段感情之前，个人有意留出的一段情感空白与调整期。

## 为何空窗期至关重要？

理解了定义后，我们来看看空窗期为何如此重要。核心观点是：分手后必须经历空窗期。没有空窗期的分手，其本质是不负责任的。

在当今时代，分手是常见现象。但分手之后，个人必须留出空窗期。这段时期是对过往感情最基本的尊重。
