# 情感修复课程 01：分手后如何有效沟通？💔

在本节课中，我们将要学习分手后若希望挽回感情，为何以及如何通过面对面沟通来提升成功率。我们将重点探讨文字沟通的局限性，并详细讲解安排有效见面的策略。

## 文字沟通的局限性

上一节我们提到了课程主题，本节中我们来看看为何文字沟通在挽回感情时效果不佳。

分手后想挽回，必须记住四个字：**见面再说**。希望挽回感情，一定要争取见面沟通，这样成功率能在90%以上。

不要发送长篇大论的“小作文”，也不要依赖打字沟通。最次的选择是打电话或语音沟通。最好避免使用微信，以纯文字形式去关心对方或讨论复合话题。

因为你发送十篇文字信息，其效果也不如见面一次。文字沟通缺乏情绪传递。你给对方发送文字，只能表达问候和关心。你长篇大论地发送“小作文”，对方可能根本不看，甚至认为你为时已晚。

## 面对面沟通的核心优势

了解了文字沟通的不足后，我们来看看为何面对面交流如此有效。

挽回最好的方式是当面交谈。但在见面之前，你必须做好充分准备。你需要思考如何阐述你们之间的问题、如何解决、如何道歉并反省，以及未来该如何行动。这些问题必须通过见面聊才能产生效果。

打字沟通通常只会产生副作用。它无法反馈你认错的态度。相反，你发送的“小作文”会让对方觉得你是复制粘贴的。在微信上卑微求和，还会让你显得掉价，更让对方反感。见面交谈的成功率则非常高。

## 如何创造见面机会

既然面对面沟通如此关键，那么如果对方不愿见面，我们该怎么办呢？本节将探讨创造见面机会的策略。

如果你能把他约出来见面，那么基本上你已经成功了一半。有些人可能会说：“我能约出来，当初也不会分手了。我就是约不出来该怎么办？”没有别的办法，如果约不出来，你就必须想方设法创造见面的机会。

如果你想挽回对方，就必须当面说清楚。依靠文字沟通来挽回，基本上是不可能的。当然，想要见面肯定没那么简单。既然对方想分手，肯定不愿意再见你，甚至排斥你。因此你要做好长期努力的准备。

在初期阶段，可以通过文字进行日常的关心问候。同时，你可以寻找任何能与他见面的借口或理由。

以下是创造见面机会的一些方法：
*   以帮忙为由，请他吃顿饭。
*   声称他有东西落在你这里，提出给他送过去。
*   制造其他合理的、不具压迫性的见面契机。

然后通过见面交流的方式，一点一点地去挽回，这样效果就能事半功倍。
