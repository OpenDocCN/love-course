## 核心原则：不谈挽回

世界上没有绝对的事情，也没有挽回不了的人。你认为不可能的事，完全是因为你没有找到正确的方法。

想让前女友找你复合，首先不要跟女生谈挽回。女生在和你谈恋爱时，看到的是你身上的优点。分手时，一定是因为你的缺点。如果你一味地聊感情话题，对方就很难和你有进一步接触。你现在必须让她看到你的积极态度，以降低她对你的负面情绪。

上一节我们介绍了不谈挽回的核心原则，本节中我们来看看具体应该如何与对方互动。

## 具体行动步骤

以下是五个关键的行动步骤，帮助你逐步改善关系。

### 1. 像朋友一样相处

大多数女生不介意分手后和你做朋友。很多时候，采用以退为进的方式给对方减压，反而可以给自己一个复合的机会。

### 2. 与女生情绪保持一致

哪怕她说自己不好，你也应该跟她保持一致。这表示你理解并尊重她的感受，而不是急于反驳或解释。

### 3. 实施冷冻断联

如果女生已经不想跟你有任何往来，那么你必须采用冷冻断联。可能有些男生会担心，如果断联太久，不就彻底没戏了吗？但你可以仔细想一想，纠缠太多只会给她更多反感。时间一长，感情也就真的淡了。

### 4. 保持真诚

真诚才是最大的套路。你是什么样子，就照着这个核心去改变。真诚的核心就是改变。你要改变自己的行为和方式，不是因为你想挽回，而是经过你的反思，在你感情中就不应该犯这样的错误。
