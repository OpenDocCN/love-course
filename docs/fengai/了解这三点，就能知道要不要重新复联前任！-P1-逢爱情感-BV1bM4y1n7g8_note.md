# 情感修复课程 01：判断复联时机的三个关键信号 🔍

在本节课中，我们将学习如何科学地判断与前任重新联系的时机。核心在于识别对方情绪平复后流露出的“情感习惯”信号，避免在错误的时间采取行动导致关系恶化。

上一段我们提到了分手初期情绪爆发点的危险性，本节中我们来看看如何安全度过这个阶段，并捕捉关键的复联信号。

## 平稳度过情绪爆发期

分手后的女性处于情绪化阶段，这是一个情绪的爆发点。此时频繁联系可能引爆她的情绪，导致她将负面情绪全部发泄出来，甚至说出绝情的话。

**核心操作：冷处理**
```text
行动 = 不联系 + 不道歉
目的 = 让双方情绪降温 + 恢复平等关系
```
此时任何靠近行为都可能被视作骚扰。你需要调整心态，让双方情绪都得到缓和。这个动作至关重要，90%的男性难以做到。

当抵触情绪随时间下降后，对方的“情感习惯”便会开始浮现，她会不由自主地回忆过往。这时，重新联系的机会窗口就出现了。

## 判断复联时机的三个技巧

关键在于准确分析对方是否进入了“情感习惯”阶段。以下是三个用于判断的小技巧，你可以对号入座。

**以下是三个具体的判断方法：**

1.  **发送简单的试探信息**
    *   方法：给对方发送朋友间的简单问候，例如“最近忙吗？”。
    *   注意：切忌发送小作文或道歉，以免退回讨好姿态。
    *   判断标准：无论回复快慢，只要对方选择回复，即表明她尚未放下。

2.  **探查共同好友的反馈**
    *   方法：联系你们的共同好友，闲聊中了解对方是否曾提及你。
    *   判断标准：如果她从好友处打听或谈论过你，说明她仍然在意你。

3.  **观察社交平台的动态**
    *   方法：查看各大社交平台，留意她是否关注、访问或点赞你的动态。
    *   判断标准：若她决心彻底分手，通常不会反复查看你的动态。

**以上三点只要满足其中一点，你就可以考虑尝试复联，成功率会显著提高。**
