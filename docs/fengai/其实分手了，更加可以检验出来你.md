# 情感课程01：分手后的真爱检验指南 💔

在本节课中，我们将学习如何通过分手后的个人感受与行为，来检验一段感情是否属于真爱。我们将把原文内容整理成清晰的教程，帮助你识别那些深刻的情感信号。

---

上一节我们概述了本课程的目的，本节中我们来具体看看分手后可能出现的几种关键心理与行为迹象。以下是四种核心的检验指标：

1.  **睡眠规律紊乱**
    你每晚难以入睡，清晨不愿醒来。因为清醒意味着必须面对分手的事实，而在半梦半醒间，你还能暂时欺骗自己，从而减轻痛苦感。

2.  **情绪无法自控**
    你在人前努力装作洒脱，但独处或与朋友在一起时，情绪完全崩溃。你可能试图保持理智，但眼泪和心痛的感觉会不受控制地涌现。

3.  **持续关注对方**
    你会习惯性地、反复查看对方的社交动态。即使已被删除好友，你仍会点开其头像。随着时间的推移，你可能会发现对方不仅不再更新动态，其朋友圈也可能从“三天可见”逐渐变成了“一条横线”。

4.  **持续的失落与回忆**
    你心里总感到空荡，时常有被掏空的感觉。你会频繁回忆起共同经历的场景，如一起吃饭的餐厅、一起走过的路、一起逛过的公园。但回过神来，你会清醒地意识到早已物是人非。

---

如果以上情况你满足**三条或更多**，那么这强烈表明对方是你生命中真正爱的人。

**总结**
本节课中，我们一起学习了通过分手后的四种心理与行为反应——睡眠紊乱、情绪失控、持续关注和空虚回忆——来检验感情深度的方法。这些迹象是内心真实情感的映射，值得你认真对待。
