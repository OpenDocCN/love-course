# 挽回前任的正确方法 💔 - 课程01

在本节课中，我们将学习分手后三种不同情况下的正确复合策略。我们将逐一分析没有联系方式、有联系但回应冷漠、以及回应良好但回避和好这三种典型场景，并提供具体、可操作的行动指南。

## 概述

分手后的复合过程需要策略和耐心。盲目行动往往适得其反。本节课程将根据你与前任的联络状态，提供三种针对性的复合路径。

## 情况一：没有联系方式 🔒

上一节我们概述了课程内容，本节中我们来看看第一种情况：分手时比较绝情，你已经失去了所有直接联系方式。

此时，首要任务是寻找一个间接的联系渠道，但找到后切勿立即使用。例如，对方拉黑了你，但你知晓其社交账号。你可以在一段时间后尝试用小号联系。

**核心前提**：使用小号联系前，必须让双方关系有一段“冷却期”。切忌在刚被拉黑后，立刻切换账号进行骚扰。前任在分手后最反感的就是穷追不舍。

如果你已经进行过死缠烂打，现在正是你重建良好印象的时机。你可以着手梳理你们关系中的矛盾。

以下是梳理时需关注的重点：
*   他说过你的哪些缺点。
*   他褒奖过你哪些优点。
*   他说过哪些伤人的话。

梳理这些问题的原因在于，它们正是你复合路上需要攻克的核心难题。当前的你或许急于复合，担心对方有新欢，看不清矛盾之间的联系，也不知如何打破僵局。此时，进行理性梳理是最明智的选择。

## 情况二：有联系但回应冷漠 🥶

相比第一种情况，有联系方式但对方回应冷漠更令人煎熬。好在，这保留了一个复合的窗口。然而，许多人在此情况下容易自暴自弃。

尤其在对方毫无回应时，狠话容易脱口而出。本意是复合，最后却逼迫对方要么回应、要么拉黑，将自己置于非常被动的境地。

面对这种情况，我们需要从边缘性话题入手。

**沟通原则**：不要一开始就谈论感情，更不要用祈求的口吻沟通。他对你的冷漠源于恋爱期间积累的激烈矛盾。你越是祈求，他越会怀疑自己当初的选择。

我们可以寻找一些互动性较强的话题，或者进行自我披露。

**有效方法**：`自我披露你在感情中存在的一些问题`，是重新唤起对方回应的好方法。很多时候，前任会觉得在关系中“吃亏了”。你的自我披露会让他产生被认同的感觉，从心理上更容易接受你。

## 情况三：回应良好，但一提和好就回避 🏃‍♂️

回避型伴侣的典型表现是：平时沟通顺畅，一旦触及和好问题，就会躲回自己的“洞穴”。无论你把话说得多动听，他依然坚持分手决定。而当你放弃和好的目标，他又能像朋友一样与你正常聊天。

为什么他能和谐沟通，却难以重回情侣关系？这类前任有一个特点：`害怕表达攻击性`。他很难直接表现出不满，只能用回避来传达态度，因此你在复合时会感到格外憋屈。

在何种条件下，他会主动复合呢？回避型前任往往需要一些“戏剧性”来润色感情。

**关键策略**：`让他看到你们拥有不一样未来的可能性`，他才会主动走出洞穴，让复合水到渠成。

如何制造这种戏剧性？核心是`向他展示你身上的闪光点、潜力与价值`。当回避型前任看到你的成长与吸引力，自然会主动靠近。

## 总结

本节课我们一起学习了分手后三种联络状态下的复合策略。其实，无论哪种情况，复合的本质无非是你如何审视这段感情，以及如何实现自我提升。

牙齿有时会咬到舌头，睫毛偶尔会刺激眼睛。任何亲密关系都存在互相伤害的可能。然而，最终我们常常发现，即便互相伤害，我们依然无法舍弃彼此。你的感情是否也是如此呢？
