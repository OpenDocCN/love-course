# 情感修复课程 01：理解“绝情”背后的心理 🧠

在本节课中，我们将要学习如何解读女生在分手后表现出“绝情”行为背后的真实心理。理解这一点是修复关系的第一步。

## 绝情是放不下的伪装

上一段我们提到了课程主题，本节中我们来看看“绝情”行为的本质。一个表现得越冷漠、越绝情的女生，往往内心越是放不下这段感情。

**绝情 = 舍不得 + 内心挣扎的极端情绪表现**

她所表现出的不理你、不沟通、不解释，这些行为都是一种伪装。

## 绝情行为的内在动因

理解了绝情的本质后，我们来深入探讨其背后的原因。这些极端行为并非源于厌恶，而是源于深刻的情感矛盾。

以下是驱动“绝情”行为的核心心理：

*   **用推开你来测试你**：因为她非常喜欢你，但在这段关系中失去了信心和安全感。
*   **自我保护机制**：她害怕再次受伤，所以用“逼你离开”这种方式来保护自己，这实质上是内心脆弱和不确定的表现。

---

本节课中我们一起学习了“绝情”行为的心理密码。我们了解到，表面的冷漠和决绝往往是内心放不下、缺乏安全感的一种反向表达。关键在于识别这些信号背后的真实情感，而不是被表面的行为所吓退。在接下来的课程中，我们将基于此理解，探讨具体的应对策略。
