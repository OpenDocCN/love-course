# 如果你想跟对方重新在一起，你必

在本节课中，我们将学习如何在分手后调整自身状态，从被负面情绪支配的感性中走出来，变得理性，从而为重建关系奠定基础。核心在于理解对方的心理变化，并采取分阶段的策略性行动。

很多人分手后会陷入负面情绪，此时会变得非常感性。感性状态下做出的行为，往往让对方感到不解和压力。若想与对方重新在一起，你必须变得和对方一样理性。

唯有理性，你才能认真思考，才能让对方觉得你是一个能共同解决问题、并肩作战的人。以下内容将分三个阶段，阐述如何从负面情绪中走出，变得理性。

## 第一阶段：理解“安全范围”，增加感情浓度 🔒

上一段提到了理性思考的重要性，本节中我们来看看如何迈出第一步。你的前任在分手后，可能会将自己封闭起来，他有一个“安全范围”。

**安全范围**的定义是：对方不允许你提及感情。你提得越多，他的压力就越大，产生的负面情绪也越多。

因此，第一阶段你需要做的，是增加你与他之间的感情浓度。你需要明白，这个阶段他需要一个人好好思考。通常，一个人在决定分手后，并非有强烈的排斥情绪，更多是顺从自己的决定，并一次次强化该行为。

但在分手后，若他发现你的处理方式比他预想的更“离谱”，例如一哭二闹三上吊或不断做承诺，这些在他听来全是废话。因为他已经认定你是什么样的人，甚至可能觉得“果然分手见人品”。

那么，如何增加感情浓度呢？关键在于在分手后的第一阶段，你要变得十分理性。你也需要给自己一段时间思考你们的感情。

你可以对他说：“其实我觉得我们俩的感情本身没有问题，问题出在我们的相处方式上。可能我们之前从未真正心与心地沟通，没有找到让彼此舒适的相处模式，但我们的感情和爱本身没有错，对吧？”

同时，你可以提及几件记忆中深刻的美好往事。这样做会让他在后续的反思期，能回忆起与你在一起的甜蜜时光，这更有利于两人和好。

## 第二阶段：摆正位置，以“新身份”重建舒适感 🤝

理解了如何安全地增加感情浓度后，接下来我们需要重新定义彼此的关系。第二阶段的核心是摆清自己的位置，利用一个新的身份出发，去增加两人之间的舒适感。

这里所说的**新身份**，指的是重新成为朋友。很多人认为，爱过的人无法做朋友，不是仇人、陌生人，就是情侣。这种想法很极端。

事实上，分手后你表现得越淡定，成功的几率就越高。你需要接受分手的事实，接受现在彼此没有关系。若想重新在一起，必须从头开始一段新的关系。

我们可以用舒适的朋友关系，替代以往充满问题的亲密关系。这是重建连接的关键一步。

## 第三阶段：重建需求，激发二次吸引 💫

在建立了舒适的“朋友”关系后，关系的推进需要更深层的动力。第三阶段的目标是让他对你产生需求。人喜欢另一个人的核心原因，根本离不开“需求”。

你最初想和对方在一起，是因为他好看、有个性，或给了你与众不同的感觉。复合同样需要需求。在复合过程中，我们要重新培养对方对你的心理需求及情绪调节需求。

简而言之，你要让对方觉得你很懂他。人都有渴望被理解的需求。当你能在这个阶段讲出对方心里想说却不敢说的话时，你便站在了共情的角度与他聊天。

随后，找一个特定的氛围与他进行面对面沟通。这时他对你的感受一定会大有不同。当然，此时仍不能提复合，因为对方对你的需求可能只有一点点，并非全部。他只是想慢慢了解你。

在这个了解的过程中，你可以自然地散发你的优点，即**原始的吸引力**。你当初是如何吸引他的，现在可以再来一次。让他觉得你是一个有魅力的人，还有许多他未曾了解的地方。

如果你能做到这些，对方很可能重新被你吸引。

本节课中，我们一起学习了情感修复的三个理性阶段：首先，在对方的“安全范围”内理性增加感情浓度；其次，摆正位置以朋友身份重建舒适感；最后，通过重建需求和展现原始吸引力来推进关系。记住，从感性到理性的转变，是重建一切可能的基础。
