## “真性分手”的三大核心特征

上一节我们概述了课程目标，本节中我们来详细看看“真性分手”的具体表现。以下是判断“真性分手”的三个关键指标：

**1. 全面切断联系渠道**
分手时，对方可能会说“我太累了”、“你很好，但我们不合适”之类的话。然而，如果对方决意分手且不愿再给机会，其**核心行为**是：将全网所有能联系到的方式全部拉黑删除。这意味着你通过任何途径都无法找到他。即便存在一两个被遗漏的联系方式，一旦你试图联系，对方也会立即将你拉黑。其根本目的是不想再听你多说一句话，也不再给你任何解释或让其自我怀疑的机会。

**2. 拒绝任何形式的沟通与争论**
如果对方真心想分手，往往不会与你进行过多对话。他不会与你掰扯谁对谁错、彼此有什么问题、或者为什么会分手。他懒得解释，因为在他看来，既然决定分手，便无需再浪费精力。真正的决心分手，往往只是一句“分手”，然后整个人便如同人间蒸发。

**3. 快速回归个人生活并向前看**
当一个人真正不爱你且真正放下时，他能够快速回归自己的生活轨道，甚至开始新生活。他可能会怀念过去，也会因你的缺席而感到难过和不习惯，但生活总要继续。其**行为公式**可概括为：
```
生活状态 = 专注自我（工作、社交、接触新人） + 不再考虑你的感受
```
现在，他只需要为自己而活。
