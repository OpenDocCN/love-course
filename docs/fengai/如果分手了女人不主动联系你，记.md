# 情感课程 01：当她不联系你时，你需要做的三件事 💔

在本节课中，我们将要学习当感情关系出现冷淡迹象，特别是对方不再主动联系时，你应该如何调整心态和采取行动。核心在于停止讨好，通过三个“狠心”的步骤来重塑自我价值与情感边界，从而掌握关系的主动权。

上一节我们介绍了课程的主题，本节中我们来看看具体需要做到的三个核心步骤。

## 第一，狠心放下所有期待

真正的放下一个人，并非指删除对方所有的联系方式，而是指**放下所有的期待**，达到内心平静的状态。许多人难以做到这一点，往往是因为对已付出的感情成本难以割舍。

但不是所有的事情都要用付出和得到来衡量。就像你走了很远的路，但如果前方的路是错的，终究要回头，而且越早回头，浪费的成本就越少。

因此，如果对方冷落你，且你已决定放弃，就不要再对双方关系抱有任何幻想。你需要做的是从心底里**快刀斩乱麻**，停止期待任何故事的发生。

## 第二，狠心放弃，重新开始

有些男性倾向于为对方的冷落寻找借口，例如“她太忙了”、“她心情不好”或“她有苦难言”。然而，这些都不是对方冷落你的正当理由。

因为真正的爱意味着在任何时候都相互依靠，你自始至终都应该是她的首选，而非被排在所有事情之后才偶尔想起。

所以，当女人长时间不联系你，就表明你在她心中已不再重要。既然如此，你无需再将时间浪费于此。学会狠心放弃这段感情，有时结束未必是坏事。

以下是帮助你“狠心放弃”的几个思考方向：
*   停止为她找借口，正视“不被重视”的事实。
*   将注意力转移到自我提升和新生活上。
*   接受关系已经变化，并允许自己向前看。

## 第三，狠心放下过往美好

在感情结束时，最令人不舍的往往不是那个人，而是两人共同经历过的美好回忆。但你必须明白，这些美好早已在日复一日的失望中被消耗殆尽。

或许因为当下的你过于痛苦，才会将过去看得格外重要。而当你走出来，经历过更多更好的人和事时，便会发现过往的美好也不过如此。

记住，一个人最大的缺点就是偏执地爱上一个不爱自己的人。别再强迫自己沉溺于过去。

本节课中我们一起学习了当对方不再主动联系时，应遵循的三个原则：放下期待、放弃幻想、放下过去。关键在于停止讨好，将重心回归到自我成长与价值重建上。通过做到这三点，你不仅能赢得情感的主动权，也可能迎来心态的彻底转变，届时对方是否回头已不再重要。
