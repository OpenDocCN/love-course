# 怎样做才能正确利用人性，从而让

在本节课中，我们将要学习如何基于人性的普遍弱点，来影响前任的心理，从而让他对你产生“爱而不得”的感觉。核心在于理解“害怕失去”的心理，并学会制造“不可得性”。

## 理解人性的核心弱点：害怕失去

想让你的前任对你爱而不得，首先需要明白人性。人性都有一个弱点，就是害怕失去。你想要让对方回头也是基于这个心理，你害怕失去对方，害怕彻底失去这段感情。

那么其实在对方的心里也是一模一样的。当对方发现自己真的有可能失去那一刻，同样，他也会有害怕失去的这个心理。这个就是人性，是个人都逃脱不掉。不同的是你的这个心理就比较明显，而对方可能隐藏的比较深。

又或者说他自己到现在还没有意识到。那么这个时候你就需要去用外力去激发。

## 对方表现冷漠的原因

说到这里，可能很多人都会问一个问题：如果对方他根本就没有这种表现的话，对方看起来就像是恨不得马上就要离开我一样，而且分手之后，他对我也是相当的冷漠。

那原因其实很简单，大部分都是因为你做了一些事情，让对方对你产生了抵触和反感，让他对你完全没有了兴趣。

如果说你能够理解的再透彻一些，你就应该明白：对方他害怕失去你。更清楚的说，他是害怕失去你对他的好，你对他的在乎，你们这么多年的感情，又或者说是他自己在你身上付出的时间、精力、金钱以及投资。

## 引入核心策略：制造不可得性

当你深刻理解了我说的这个人性的弱点之后，你就可以学下去给对方制造一些不可得性。

什么叫做不可得性？就是让对方知道你不会一直在原地等着他。一旦他选择了彻底的分手，那也就意味着他不可能再轻易的得到你、靠近你。也就是说你要让他看到这个后果，这个后果就是彻底的失去你。

**公式：不可得性 = 你不再轻易能够被得到**

注意，制造不可得性，不是说让你跑去告诉你的前任：“我不会再爱你了，我不会再联系你了。”这种话呀，对方听到了，他都是没有分量的，他不会因为你说了这一两句狠话就重新的重视你。

## 如何正确制造不可得性

真正的去制造一个不可得性，是你们在断连了一段时间之后，对方再来找你，但是呢，你不在线，你不回复，或者说你不上他的当。

如果你们懂得废物测试就知道了，他过来测试你的时候，你就不要去给他提供他想要的一个反馈。

那么这个时候，他的内心才会产生落差，才会更加的想要主动的来找你。

以下是制造不可得性的关键步骤列表：

1.  **彻底断联**：给予双方时间和空间，停止一切主动联系。
2.  **保持边界**：当对方试探性联系时，不提供情感反馈或即时回应。
3.  **专注自我**：将精力投入到提升自己的生活和价值上。
4.  **间接展示**：通过社交平台等间接方式，展示你独立、充实的新生活。

## 总结与行动建议

本节课中，我们一起学习了利用人性“害怕失去”的弱点来影响前任心理的基础。核心是通过制造“不可得性”，让对方意识到彻底失去你的可能性，从而激发其内心的落差感和主动意愿。

记住，关键在于行动而非言语。通过冷静、有边界的行为来传递“你已不再轻易可得”的信息，远比任何狠话都更有力量。
