# 情感修复课程 P1：真性分手与假性分手的区别 🧩

在本节课中，我们将学习如何辨别真性分手与假性分手，并了解针对不同类型分手的初步应对策略。理解这一核心区别是制定有效挽回计划的第一步。

---

## 如何辨别真性分手与假性分手

关于真性分手和假性分手如何辨别，其实很简单。请记住这句话：**假性分手是分手发出信号，真性分手是分手后切断信号**。

说得直白一点，假性分手是对方只是想闹别扭，提出分手只是希望获得你的关注和安抚，并非真想结束关系。而真性分手，则是对方真的认为无法继续在一起，想要彻底分开。

看到这里，你无需紧张，也不必认为真性分手就无法复合。事实并非如此，只不过真性分手需要你付出更多的时间、精力以及耐心去挽回。女人的心是多变的，只要你能够抓住她的心理，掌握她的情绪变化，那么复合自然就是水到渠成的事。

上一节我们介绍了两种分手的核心区别，本节中我们来看看挽回初期应该如何掌握节奏。

## 挽回初期的节奏掌握：打破心理预期

首先，我们要学会打破对方的心理预期。对方提出分手时，他的一个心理预期是：你肯定会不同意、会不愿意、会纠缠吵闹。这是我们常人被分手后的正常反应。

这时，你需要反其道而行，大大方方地接受分手的事实。当然，这需要技巧，要使用“变焦话术”。

以下是运用“变焦话术”的两个核心步骤：

1.  **先肯定对方的价值**：认可对方分手的决定有其合理性。
2.  **再打感情牌**：基于过往的感情基础，为未来关系留有余地。

例如，你可以这样说：“**你说的也是对的，其实我早就想跟你这么说了，我也看出来咱俩真的不合适。但是我觉得咱俩相处这么久了，相处的也还挺不错，我们以后也没有必要闹得这么僵，之后还能当朋友，以后你有什么事情也可以找我帮忙。**”

看明白了吗？我们不死缠烂打，但核心目的是保留联系方式。当对方听到你说这些话时，他会瞬间感到困惑。他会觉得：“是我跟你分手了，怎么感觉好像是你把我给甩了似的？”本来应该情绪不稳定的是你，但现在因为你这句话，让他感觉被分手的人是他，情绪不稳定的也是他。

这样一来，情绪是否就被你拿捏了呢？你成功打破了对方的心理预期。他心里一定会产生一种想要抓住你的情绪。这时，他不但不会删除你的联系方式，相信我，接下来一段时间他还会持续关注你。即使嘴上不说，他也一定会暗中观察你。
