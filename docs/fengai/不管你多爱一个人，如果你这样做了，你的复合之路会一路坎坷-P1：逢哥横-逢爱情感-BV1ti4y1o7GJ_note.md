# 情感修复课程 01：🚫 停止无效联系，逆转复合困境

在本节课中，我们将要学习当对方停止联系时，一个常见的错误行为及其严重后果。理解并避免这种行为，是修复关系的第一步。

## 认清现实：停止自我欺骗

当女生不再联系你时，你心里其实已经明白了她的想法。然而，你不想放弃，还希望亲口听到她的拒绝。即便她的态度已经非常明确，痴情的人往往不到黄河心不死。

上一节我们认识了这种不愿面对现实的心理，本节中我们来看看这种行为的直接后果。

## 错误行为的代价：加剧对方反感

你必须知道，你越是主动联系女生，她就可能越讨厌你。你的纠缠和追问，不会唤起同情，反而会强化她的负面印象和逃离的决心。

## 核心策略：反其道而行之

面对这种情况，正确的做法是反其道而行。这意味着你需要停止施压和纠缠，采取截然不同的应对方式。

以下是停止无效联系后，你可以初步思考的几个方向：

*   **彻底断联**：给予双方时间和空间冷静。
*   **重塑自我**：将注意力放回自身，提升价值。
*   **间接展示**：通过社交平台等间接方式，展示积极变化。
