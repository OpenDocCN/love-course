## 停止纠缠 🚫

上一段感情的结束，往往意味着关系中存在长期且深刻的矛盾。本节中我们来看看为什么停止纠缠是复合的第一步。

分手后，许多人会立即选择道歉。这种行为通常表现为线上消息的轮番轰炸，或线下的苦苦等候。

以下是纠缠行为可能带来的后果：

*   如果对方同意和好，但核心矛盾未解决，关系很可能重蹈覆辙。
*   如果对方态度决绝，持续的纠缠只会让对方觉得你价值降低，将彼此推得更远。
*   你的纠缠与对方的抗拒会形成对立：你会疑惑“对方为何如此绝情”，而对方则反感“你为何还在纠缠”。

**核心要点**：走到分手这一步，说明存在长期矛盾。一味道歉无法解决问题，反而会加剧负面印象。因此，分手后首先要学会停止纠缠。

## 适度断联，降低负面情绪 ⏸️

上一节我们介绍了停止纠缠的重要性。本节中，我们来看看如何通过适度断联来为双方创造情绪缓冲的空间。

如果一个人并非戏弄你，那么提出分手通常是深思熟虑的结果。若彼此真心爱过，分手后双方内心都可能存有留恋。

此时，短暂的安静期非常必要。这既能给对方时间冷静衡量是否真要结束关系，也能让你自己反思感情中出现的问题。

以下是适度断联带来的两个关键转变：

*   **给对方制造认知判断**：短时间的断联会让对方认为你已经接受分手事实，并开始了自己的新生活。
*   **为自身姿态带来转变**：这能将你从“卑微祈求”的姿态，转变为“平等对话”的姿态。公式表示为：**姿态 = 断联前：卑微祈求 → 断联后：平等对话**。人不会尊重过于卑微的人，平等才可能让对方重新正视你。
