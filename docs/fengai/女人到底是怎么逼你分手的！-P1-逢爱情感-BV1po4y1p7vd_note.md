# 课程名称：冷暴力分手深度剖析 🧊 - 第1课

在本节课中，我们将要学习一种被称为“冷暴力分手”的情感伤害模式。我们将深入剖析其发展的不同阶段，帮助你理解其运作机制和背后的心理动因。

冷暴力分手是所有分手方式中最伤人的一种。它最痛苦的地方在于，会让你在很长一段时间里，情绪都跟着对方走，逐渐迷失自我。

## 第一阶段：热情消退期 🔥➡️❄️

上一节我们提到了冷暴力分手的整体伤害性，本节中我们来看看它的第一个具体阶段。

女生突然变得很忙，但她不会完全不理会你。她偶尔也会给你发短信或打电话，但你会发现对话中没有任何情绪上的波动，热情直接消失。这个阶段，你通常会开始质问她，问她最近怎么了。

以下是此阶段对方常见的回应模式：

*   她不会直接告诉你“我没那么爱你了”或“我爱上别人了”。
*   她们通常会说：“没有啊，我最近比较累，压力很大。”
*   然后会叮嘱你：“不要乱想。”

## 第二阶段：被动失联期 📵

经历了第一阶段的困惑后，关系会进入更明显的冷却期。

你不主动联系她，她也不会主动联系你。你发信息她可能不回，打电话她可能接，但只要没有急事，她绝不会主动联系你。此时，你会开始抓狂地思考一系列问题。

以下是此阶段你内心可能产生的疑问：

*   她到底在想什么？
*   她是不是不爱我了？
*   我们之间发生了什么？

这一连串的问号让你逐渐失去自我，你感觉到不对劲，但因为不舍得分手，于是开始试图寻找问题，想要维护这段感情。

## 第三阶段：挑剔与指责期 ⚖️

当你试图挽回时，对方的反应往往会将你推向第三个阶段。

她开始嫌弃你，一边觉得你管得太多，一边又觉得你不够爱她。一点小事就能让她生气，并动不动就以分手相威胁。

在这个阶段，你的心情会慢慢变差，开始小心翼翼地面对她，甚至刻意地讨好或迎合。她偶尔可能会不忍心，关心你几句，让你觉得她还在乎你，从而给你一丝虚假的希望。

## 第四阶段：反复与崩溃期 🎢

上一节我们看到关系在挑剔与偶尔的关心中反复，本节中我们来看看最终导致崩溃的阶段。

她可能会对你稍微好一些，但你会感觉这种“好”带有不自然或歉意。好景不长，她很快又会变回不再联系的状态，甚至更加严重。

以下是此阶段可能出现的具体行为：

*   你发过去的信息石沉大海。
*   你打过去的电话要么不接，要么关机。

当你终于忍不住去找她对质，询问到底怎么回事时，情况会急转直下。

*   如果你的态度好，她可能什么都不说，仿佛一切都没发生。
*   一旦你的态度变差，她就会开始把所有问题都推到你身上，并指责你。

此时，你的心态可能已经完全崩溃，彻底失去自我，陷入失眠和食欲不振的痛苦中。最终，你实在无法忍受，主动提出了分手。而这，恰恰正中了她的下怀。

> 其实，使用冷暴力分手的人，通常都是特别害怕承诺、自私且不懂得珍惜付出的人。他们没有提出分手的勇气，内心极度自卑。他们希望通过这种逃避的方式，让你默默退出，以此来减轻自己对这段感情的愧疚感。

本节课中，我们一起学习了冷暴力分手的四个典型阶段：从热情消退、被动失联，到挑剔指责，最后在反复拉扯中导致一方崩溃并提出分手。理解这一过程，有助于我们认清不健康的关系模式，更好地保护自己的情感健康。
