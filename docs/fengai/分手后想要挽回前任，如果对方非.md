# 分手后想要挽回前任，如果对方非

在本节课中，我们将学习分手后，当对方表现出强烈排斥情绪时，应如何正确处理。我们将分析排斥行为的表现、持续时间以及具体的应对策略。

分手后想挽回对方，但对方很排斥自己，该如何处理？排斥会有哪些表现？会持续多久？应该怎么做？这是你们关心的问题。耐心看完本教程，你会找到答案。

被分手的你们，脑海里肯定是对方离开后所带来的恐惧、恐慌、不甘心、难过等情绪。还有在过往的感情中，两个人相处的美好、对未来的期待和遗憾。而主动提分手的一方恰好相反。他脑海里想的，基本上都是他在你们这段感情中所受到的不公、委屈、伤害和爱的缺失等负面情绪。也是这些负面情绪支撑着他下定决心离开你，并且开始新的生活。

## 排斥期的表现与本质

这时，对方会产生一些表象行为。例如外在形象和生活状态的改变。女生会去买新衣服、做头发、旅游、和闺蜜聚会、认识新的异性、努力工作等。当然，这也是新欢的高发期。

但不管对方做什么，这些行为本质上都是在淡化他的负面情绪。大多数人在这个时间会去纠缠、去求复合。但你想想，对方正在开心快乐的时候，怎么可能要跟你和好？怎么可能答应你重新回到之前那种不开心的生活中去呢？

## 排斥期的核心应对策略

这时你要做的只有两点。

以下是具体操作步骤：
1.  **为过去的行为道歉，并承认分手事实。** 你需要为过去关系中造成伤害的行为诚恳道歉，并明确表示你接受分手这个决定。
2.  **减少沟通，但保持联系，同时展示改变与适当的难过。** 公式可以表示为：**联系频率降低 + 自我提升展示 + 适度情感流露**。这意味着不要频繁打扰，但可以通过间接方式（如社交媒体）展示你的积极变化和反思，同时不掩饰你的伤感，但不过度。

## 关键转折：冷静期

等到双方情绪都平复后，便会迎来分手后的第二个时期——冷静期。这个时候才是复合的最关键时期。绝大部分的分手能否复合，就要看这个阶段你怎么去做。
