因为在这个阶段，你的情绪并不能完全自控，所以此时如果有人能好好引导你、陪伴你，你会更快走出失恋的痛苦。

你需要做的就是不断地进行**印象植入**和**情感铺垫**。

**操作核心**：利用**共同经历**（地点、食物、习惯）触发**情感怀念**。

从而加速对方对你的情感怀念期。当他先一步进入情感怀念期时，恭喜你，你已经把他拉回来一半了。

他就会开始反思自己是否有问题。这个反思期也会让他重新确认，自己是真的很喜欢你、很爱你。
