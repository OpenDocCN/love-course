在本节课中，我们将要学习分手后试图挽回前任时，六种最让对方反感的行为。了解这些行为，可以帮助你避免在情绪驱使下做出错误的决定，从而更理性地处理分手后的关系。

上一节我们介绍了课程的主题，本节中我们来看看具体是哪六种行为。

## 1. 死缠烂打

第一种最令人反感的行为是死缠烂打。有的人不愿意接受分手的事实，不停地缠着对方。除了不停地打电话、发信息之外，还会在对方的必经之路，甚至是上班的地方守候，严重地影响到了对方的正常生活。

这样做会使对方感觉到压力很大，很怕再次见到你，甚至开始厌恶。

## 2. 博取同情

第二种行为是试图博取对方的同情。整日以泪洗面，借酒消愁，甚至连工作都放弃了，以此让对方感觉到愧疚。还有些人不吃不喝，把身体搞垮，希望对方因为不忍心而选择回心转意。

## 3. 动员朋友劝说

感情它本来就是两个人的事，分手这件事你知道的人越少越好。以下是关于此行为的详细说明：
*   如果你们有共同的好朋友，对你们的事情很了解，你们出现了问题，可以请这个朋友来做调解员。
*   但是如果你把分手这件事搞得人尽皆知，让所有朋友都参与进来，这不仅帮不上忙，还会让对方感觉到有很大的压力。
*   对方甚至会因为分手被很多人知道，而选择不再回头。

## 4. 长时间冷处理

有的人不想分手，但是又碍于情面，不敢向对方认错，就采取冷处理的方式，谁也不搭理谁。分手初期彼此冷静一下是好的，但是长时间不主动联系对方，即使对方不舍得分手，时间久了也会感觉到伤心失望，不愿意再回头。

## 5. 过分暴露需求感

虽然不采取极端的方式死缠烂打，但是让对方感觉到，没有他就没有办法生活下去，你时刻不能离开对方。这种情况你越是急于得到，对方就会越逃避，你们就会越来越远。

## 6. 空口承诺

第六种行为是说空话。可能是因为自身存在的一些缺点，对方要求你改正，一直没有重视，更没有改正，直到被分手才后悔不已。以下是此行为的典型表现：
*   分手后一再向对方保证，自己会改正错误，只要对方能够原谅你，肯定不会再继续错下去。
*   然而说归说，你还是原来的样子，仍然不会改变。
*   或许你的想法是，对方先原谅了你才会改正。其实对方希望看到的，是真正的改正了，才会考虑原谅你。

本节课中我们一起学习了分手后最让前任反感的六种挽回行为，包括死缠烂打、博取同情、动员朋友、冷处理、过度暴露需求以及空口承诺。理解这些行为的负面影响，是迈向理性处理和自我成长的第一步。
