# 情感修复课程 01：分手后的四个阶段与心理动态 🧠💔

在本节课中，我们将要学习一段亲密关系从终结到彻底结束所经历的四个典型心理阶段。理解这些阶段，有助于你更清晰地认识自己或对方在分手后的状态与行为。

一段感情走到终结，绝不是因为最后一次争吵或矛盾导致的。那只是压垮骆驼的最后一根稻草。真正的分手，从两个人开始沉默时就已经开始了。这个时期被称为“沉默期”。

## 第一阶段：沉默期 🤐

上一节我们提到了感情终结的起点，本节中我们来看看第一个具体阶段——“沉默期”的表现。

在这个阶段，双方会开始质疑对方的爱意，怀疑彼此是否合适，并不断挑剔对方的毛病，从而逐渐感到失望。可能曾试图改变，但最后发现徒劳无功。提前准备放弃的一方，内心已开始计划分手，等待的只是一次矛盾后的争吵。

## 第二阶段：爆发期 💥

了解了初始的沉默与失望后，关系会进入一个更激烈的阶段。

以下是“爆发期”的主要特征：
*   双方开始争论谁付出更多、谁对谁错。
*   主动提分手的人希望尽快抽离，开始新生活，甚至可能开始“无缝衔接”。
*   被分手的一方则开始拼命纠缠和挽回。

于是，双方完全站到了对立面，关系越来越僵，直至彻底拉黑所有联系方式，宣告关系终结。

## 第三阶段：冷静期 🧊

关系在爆发中彻底断裂后，双方会进入一个情绪沉淀的时期。

彻底失去对方后，各自会经历一段平静或发泄的时期，然后回归日常生活。但仍会感到不适应，或陷入回忆，想起过去的点点滴滴，在回忆中痛苦，在痛苦中反思和挣扎。

此时，主动提分手的人可能会产生一些后悔和遗憾。而被分手的人已开始接受现实，打算放弃并尝试新生活。但忘记一个爱的人确实很难，双方都会不时想起对方。

## 第四阶段：反复期 🔄

当情绪逐渐冷静，新的纠结和反复又会开始。

他会后悔当时的决定，但又拉不下脸主动找你。人往往如此：做错事容易，承认错误却很难。因此，他可能会开始间接性地接触你。

而被分手的一方呢？状态可以描述为：
```python
# 内心矛盾状态
while True:
    desire_to_move_on = True  # 想开始新生活
    stuck_in_pain = True       # 又沉浸在失恋痛苦中
    if desire_to_reconcile:    # 如果想争取复合
        recall_cold_response() # 却想起对方曾经的冷漠态度
```
就这样，在犹豫和反复中，两个人可能慢慢错过了彼此。
