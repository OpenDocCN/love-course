# 分手后是制造失去感还是做朋友相：分手后是制造失去感还是做朋友相

在本节课中，我们将要学习分手后如何选择正确的策略：是应该制造“失去感”，还是先以“朋友”身份相处。核心在于理解对方对你的“需求感”强弱，并据此采取行动。

## 策略选择的根本依据

分手后选择制造失去感还是做朋友相处，取决于你能给对方提供的价值。

具体来说，只要你们两人之间还存在重新建立需求关系的可能性，就可以考虑策略。对方提出分手，表明他当时对你产生了强烈的抵触情绪，但这通常只是一时的想法。

## 两种策略的适用场景分析

以下是两种不同情况下的策略选择：

**情况一：对方需求感强**
如果他对你的需求感依然强烈，你去制造失去感会非常有效。这意味着他突然意识到你的离开，从而可能重新审视你的价值。

**情况二：对方需求感弱**
如果他对你的需求感不强，你去制造失去感就完全没有作用。在这种情况下，挽回的几率会很低。

## 需求感薄弱时的核心策略

上一节我们分析了不同需求感下的策略，本节中我们来看看当对方需求感薄弱时，具体应该怎么做。

在对方没有需求或需求感薄弱的时候，首要任务不是施加压力，而是重新为对方创造价值。你需要先给对方建立需求，核心方法是提升自己的价值，让他重新对你产生依赖。

最终目标是，让对方在与你相处的时候，再次感到非常舒适和愉悦。当这一点实现时，挽回的成功率就会大大提高。

## 课程总结

本节课中我们一起学习了分手后策略选择的核心理念。关键在于评估对方对你的“需求感”：**需求感强时可制造失去感**，**需求感弱时则需先以朋友身份重建价值和舒适感**。记住，让对方重新感受到与你相处的“舒适感”，是挽回成功的重要基础。
