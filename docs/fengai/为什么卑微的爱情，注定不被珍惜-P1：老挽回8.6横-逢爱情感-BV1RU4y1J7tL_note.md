# 为什么卑微的爱情，注定不被珍惜-P1：老挽回8.6横-逢爱情感-BV1RU4y1J7tL_note

在本节课中，我们将要学习为什么在感情中过度卑微和单方面付出，往往无法获得珍惜与尊重。我们会分析这种心态的成因，并理解平衡在亲密关系中的重要性。

上一段我们提到了卑微爱情的核心问题，接下来我们具体看看这种心态通常有哪些表现。

以下是卑微爱情中常见的想法模式：

*   只要我对对方好，全心全意地付出，那么他也会对我好，为我付出。

如果你也是这样想的，那么你多半会失望。因为一个人付出再多，也没有办法换成两个人的幸福。

在感情里爱得太卑微，最终会让你失去它。无论男人还是女人，轻易得到的都不会被珍惜。如果爱得不对等，出问题那是迟早的事。

所有的事都讲究平衡，爱情更是这样。你的付出有上限，对方的贪心没有上限。

最后你会发现，你付出了所有，对方反而觉得你做的不够。时间久了，你会越来越累，而且你会感觉到对方对你越来越不重视。

为什么呢？因为你把对方捧得太高了。他反而对你的付出习以为常，并且仗着你的爱有恃无恐。这么一个不珍惜你的人，怎么可能跟你走完余生呢？

你之所以这样做，归根结底就是你内心的不自信。你想通过讨好获得对方的爱。

可是你知道吗？对方喜欢你和珍惜你的前提是觉得你好。当你自己都不自信的时候，很大程度上你都已经失去了这个“好”。你自己想想，越是打折的东西，你可能越是觉得它廉价，怎么可能再去珍惜呢？

所以记住一个核心原则：**千万不要爱一个人爱到尘埃里**。

本节课中我们一起学习了卑微爱情的动态：单方面过度付出会破坏关系平衡，导致自身价值感降低，从而无法获得对方的珍惜。健康的爱情需要建立在自信与相互尊重的基础之上。
