# 情感课程 P1：为什么在一起很久了，却还是没有安全感？🤔

在本节课中，我们将要学习一个常见的情感困惑：为什么在一段长期关系中，你依然会感到患得患失、缺乏安全感。我们将深入剖析其核心原因，帮助你理解关系中的心理动态。

## 核心原因剖析

上一段我们提出了问题，本节中我们来看看导致这种感受的根本原因。

出现这种感觉，最主要的原因就是：**关系中的双方投入程度不一致**。具体表现为，一个人已经非常坚定地决定在一起，而另一个人却还没有完全下定决心。

## 关系中的不对等状态

理解了核心原因后，我们进一步看看这种状态在日常中的体现。

这种心态的差异会导致行为上的不对等。在你掏心掏肺、全心投入的时候，对方可能还在进行理性的权衡，计较着关系中的利弊与得失。

## 自我认知与矛盾

认识到关系状态后，我们不可避免地会面对内心的矛盾。

其实你心里也明白这种不对等，可是依然无法轻易放下这段感情。这种认知与情感的冲突，加剧了不安全感。

---

以下是针对本课内容的总结：

*   **核心原因**：安全感缺失源于双方对关系承诺的**决心程度不同**。
*   **行为表现**：一方全力投入（**掏心掏肺**），另一方仍在评估（**权衡利弊**）。
*   **内心矛盾**：即使理性上明白问题所在，情感上仍难以割舍。

---

本节课中我们一起学习了长期关系中缺乏安全感的心理根源。关键在于识别关系中双方投入度的差异，并正视由此产生的内心矛盾。理解是解决问题的第一步。
