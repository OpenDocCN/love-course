# 💔 分手后她不删除你又不理你的原因 - 课程 P1

在本节课中，我们将要学习一个在分手后常见的困惑：为什么对方既不删除你的联系方式，也不回复你的消息。我们将深入分析其背后的心理动机，帮助你理解这种行为模式。

## 🔍 核心原因分析

上一节我们介绍了本课程的主题，本节中我们来看看具体的原因。

他不删除你又不理你，是因为他的心里还有对你的爱情，并没有完全放下你。

他之所以不回复你的消息，是怕回复之后又开始纠缠，对他现在的生活造成一定的影响。
