# 情感沟通课程 01：解读关系中的沉默应对 🧠

在本节课中，我们将要学习亲密关系中一种常见但极具伤害性的行为模式——沉默应对。我们将从心理学角度分析其本质，并理解它为何会对关系造成巨大破坏。

## 沉默是“无力”还是“攻击”？🤔

对方一遇到问题就选择沉默，直接选择摆烂，什么话都不说，什么都不做。很多时候你想要去沟通、去交流、去解决问题，但是对方却直接选择了沉默。这种情况真的会让人觉得，是不是自己做错了。

上一节我们提出了沉默应对的现象，本节中我们来看看其背后的心理动机。让对方觉得太累了才会这样。但事实上，想要沟通、想要表达情绪这样的诉求，在亲密关系里面，彼此之间是没有错的。因为在关系里面，你才是想要解决问题的那一方，而他是只顾着自己的舒服，顺其自然的态度来对待这段关系。

## 沉默的心理本质：一种被动攻击 🛡️

有的人可能觉得这种沉默代表着无力，代表着不知所措，代表着无可奈何，看上去是一种冷暴力的手段。但是从心理学的角度，这种沉默恰恰是一种反抗和攻击的方式。也许对方是回避性依恋，虽然他好像是没有生气。

但他的这种沉默就是在无视你所有的情感表达和情感需求。他在用一种被动示弱的方式来告诉你，你是加害者，而他是受害者。在心理学上，**无视就是对一个人最大的否定**，否定你的表达，否定你的感受，否定你的行为。

这种本质上都在否定你这个人全部的价值。他每次都不想要再跟你争，这就说明在他心里，你根本就不是一个值得沟通的对象，甚至他不想跟你沟通，也不屑跟你沟通。这其实是一个很严重的指责。

## 沉默对关系的核心伤害：剥夺“被看见”的需求 👁️

那么这种沉默的态度会给亲密关系带来多大的伤害呢？一个人最大的需求就是被看见的需求。我们都需要对方来看见我们的喜怒哀乐，来看见我们对爱的需求。被看见才有存在感，才有安全感，才有可能会幸福。

以下是沉默与正常争吵的对比，说明了为何沉默更具破坏性：

*   **正常争吵**：吵架本身并不会让人绝望。哪怕他反对你的观点，控诉你的问题，跟你争个对错，哪怕他的情绪非常上头，说出了一些伤人的话。但是这些都不会让人绝望。虽然你们的情绪波动非常大，但本质上你们还是在进行一种交流。你们能够看到对方的不满，也能够感受到对方的心情。
*   **沉默应对**：但是选择沉默的人呢？你和他摆事实讲道理，他也不讲话；你想要教他怎么和你沟通解决问题，他也不说；你和他表露情绪吧，甚至你崩溃大哭，只想要让他关心一下，他还是不说话。那么他的所有行为，只给你传递出一种信号，就是他无所谓，他不在乎你。

## 健康沟通与逃避沉默的区别 🔄

你要搞清楚包容和理解的含义。并不是说我全盘接受全是自己的问题，而是：
```text
我知道我们发生了矛盾，但是我也很愤怒。我没有办法和你好好的沟通，所以麻烦你给我一点时间来冷静。等到我情绪平复了以后，我们再来一起解决这个问题。
```
这才是两个人之间彼此的尊重，这才是想要解决问题的人应该有的表现。而那些一发生问题就直接消失、遇到矛盾就直接选择逃避的人，那你们的关系只会越来越糟糕。

---

本节课中我们一起学习了亲密关系中的“沉默应对”。我们明白了它并非简单的逃避，而是一种被动攻击，其核心伤害在于剥夺了伴侣“被看见”的基本需求，本质上是对对方价值的全盘否定。健康的沟通需要双方共同的意愿和努力，单方面的沉默只会让关系恶化。
