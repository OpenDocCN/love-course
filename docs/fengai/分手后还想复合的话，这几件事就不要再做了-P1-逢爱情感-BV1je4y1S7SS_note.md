# 分手后还想复合的话，这几件事就不要再做了-P1-逢爱情感-BV1je4y1S7SS_note

在本节课中，我们将要学习分手后若希望复合，必须避免的几种常见错误行为。这些行为往往会将对方推得更远，阻碍关系的修复。理解并避开这些陷阱，是迈向成功复合的第一步。

很多人分手后最容易做的一件事情就是死缠烂打。你需要解决的是对方对你的怨恨。任何负面情绪和死缠烂打对你们的感情毫无帮助。你只有成长自己，打破他对你的固有认知，才是正确的补救措施。

因此，在分手后，如果你还想继续这段感情，那么我接下来说的这几种行为，你千万不要去做。

以下是五种必须避免的行为：

1.  **避免“自我感动”式的仪式感**
    举一个最简单的例子：跟对方分开之前准备了一些礼物，在分开之后你还要想寄给对方，希望对方能够感受到你的爱意。又或者说你给对方发了一篇长篇的小作文，告诉对方你要怎样怎样、你要照顾好自己。其实大多数前任根本就不会在意你这些小动作。

2.  **避免空洞的、不停道歉**
    当然，我不是说不让你们去道歉。真正需要的是你意识到自己的问题和错误后再去道歉。很多人根本就不知道自己错在哪里了，然后就疯狂地去道歉，不停地向对方做出保证和承诺。而事实是，你们的分手只是因为相处中出现了问题。他想要看到的是你的改变，是你对这段感情认知上的改变。如果你永远都是只道歉而不知悔改，那么你就只会将他越推越远，让对方觉得和你这样的人分手，真的是一件非常正确的决定。

3.  **切忌死缠烂打**
    在分手后最忌讳的其实也就是这个。因为在分手之前，对方已经对你产生了负面情绪。本质上，如果你还想让对方能够回头，那就得先让他产生舒适的感觉。他如果觉得和你在一起舒适，又怎么会选择和你分手呢？那就一定是你给他带去了太多的负面情绪，导致他想要尽快地逃避和逃离你。所以，在这个阶段你去不停地骚扰对方、联系对方、去约对方，那么你这种死缠烂打、乞求对方的行为有意义吗？只会让他更加地厌恶和排斥你。

4.  **不要在社交动态上发布伤感文案**
    你在你的社交动态上发这些伤感的小文案，虽然说这些文案能够让你尽可能地看起来很难过、很伤心，但是你的小心思别人一看就看透了，你这不就是在博同情、博关注吗？那么引发的问题是什么呢？你会向你们的共同好友去倾诉你有多么难过、多么痛苦、他有多么无情。这种操作请你立刻停止。你要知道，你把分手这件事情告诉了第三个人，这个第三个人他是抱着一种什么样的心态去看这件事呢？跟你关系还不错，可能会安慰你一句、保持中立；而遇到一些墙头草，可能就把你所说的所有的话都给你的前任转述了。那你的前任就会破罐子破摔。

5.  **不要哀求，也不要纠结“是否还爱”**
    千万不要去求对方，也不要再去纠结对方到底还爱不爱你。无论他告诉你他爱你，或者是不爱你，还是说把你给删除了、拉黑了，这都是一种逃避的行为。这并不是问题的所在，恰好是你们的感情出现了问题，他才想要用这种逃避的方式去解决问题。你需要了解的是，复盘你的感情到底是哪里出现了问题，而这个问题能不能解决，这个才是关键点。而且也不要去求对方，大家都已经处于一种情绪失控的状态，根本就没有办法在合理的范围内控制好自己的情绪。这个时候你去求对方、你去找对方，问题不仅是说不清楚，也只会增加一些新的矛盾。

本节课中，我们一起学习了分手后希望复合时必须避免的五种行为：自我感动、空洞道歉、死缠烂打、发布伤感动态以及哀求与纠结。记住，复合的关键在于**解决核心问题**与**自我正向改变**，而非施加压力或情感绑架。停止这些错误行为，是为你和你们的未来关系创造新的可能性的第一步。
