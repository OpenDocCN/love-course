## 五种可能复合的表现分析

上一节我们概述了本课程的核心，本节中我们来看看具体的五种表现。以下是每种表现的详细解析：

1.  **不接电话也不回消息**
    这种行为常被理解为彻底的拒绝。然而，它也可能表明对方情绪波动很大，需要独处空间来平复心情，并非心意已决。

2.  **删除或拉黑所有联系方式**
    采取如此激烈的切割手段，往往是为了强迫自己停止留恋。这恰恰证明了对方仍在乎，需要用这种“物理隔绝”的方式来对抗情感。

3.  **故意逃避见面**
    对方选择回避直接接触，可能是因为害怕见面后情绪失控或决心动摇。这种“害怕”本身，就是情感仍在的折射。

4.  **短时间内开始新恋情**
    迅速投入一段新关系，有时并非移情别恋，而可能是一种“反弹式关系”，用以疗伤或证明自己。公式可表示为：**新恋情 = f(逃避旧伤)**。

5.  **态度表现得非常绝情**
    言语和行为极度冷漠、伤人，可能是出于强烈的自我保护，或想让你更快死心。其底层逻辑代码可能是：
    ```python
    if 内心痛苦:
        表现 = "绝情"  # 用外在的坚硬保护内在的脆弱
    ```
