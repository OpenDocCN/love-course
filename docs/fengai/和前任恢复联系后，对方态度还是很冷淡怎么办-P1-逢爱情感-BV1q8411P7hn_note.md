# 情感修复课程 01：如何应对复联后的冷淡态度？💔

在本节课中，我们将要学习与前任恢复联系后，如果对方态度依然冷淡，应当如何分析和应对。我们将拆解冷淡背后的原因，并提供具体的操作步骤，帮助你更有效地推进修复关系的过程。

大部分的挽回者在和前任初步取得联系时，说话都非常小心翼翼，生怕自己说错一个字或一句话惹对方不开心。然而，慢慢地你会发现，仅仅顺着对方的话来说，效果反而越来越差。即使出现了一些好的机会窗口，你也很难把握住。

为什么会产生这样的局面呢？

## 一、 对方态度冷淡的核心原因 🔍

上一节我们提到了效果不佳的现象，本节中我们来看看导致这种情况的两个根本原因。

1.  **负面印象尚未消除**：对方对你仍有抵触情绪。虽然你们已经成功恢复了联系，但对方态度冷淡是非常正常的。
2.  **复联操作出现问题**：你的复联过程无法对前任造成二次吸引。情感的挽回是一个系统的修复工程，离不开你自己的成长与变化，以及由此引发的对方的连带反应。

复联之后，由于个人的恋爱相处方式、对方性格和情感问题的深浅不同，不同的人展现出来的表现也截然不同。

## 二、 如何解决复联中的典型困难？🛠️

了解了原因后，我们来看看最常见的困难场景及其解决方案。在复联后，情绪不稳定、忽冷忽热的情况占据了大部分。

你在跟对方联系时，对方会回复，也没有表现出极度冷淡，但很多时候回复不及时，很久才会简单地回你一句，且不会说过多的话。

如果你面临的就是这样的情况，首先你要知道对方不愿意多回复你的原因是什么。

以下是两种可能的原因分析：
*   **原因一：仍有排斥，但留有机会**。对方对你还有排斥，因为心里的隔阂尚未彻底清除。然而，对方仍然对你有预期、给你留有机会，所以才选择了不积极回复信息，等待你自己去打动TA。
*   **原因二：正在考验你的诚意**。一开始，你发的信息可能缺乏吸引力，或者对方觉察不到你的任何变化和改变。因此，对方会觉得你还有待考验。当对方看到你的改变之后，才会考虑更积极地回复你的信息。

那么，当处于复联阶段发信息时，具体应该怎么做呢？

首先，最重要的就是你要能够吸引对方的注意力，而不是进行尴尬的闲聊。
