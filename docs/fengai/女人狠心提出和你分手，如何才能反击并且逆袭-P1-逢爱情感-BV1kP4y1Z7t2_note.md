# 情感逆袭课 P1：如何应对分手并重塑自我 🛡️

在本节课中，我们将学习当伴侣提出分手时，如何停止无效纠缠，将注意力转向自我成长，从而改变双方关系中的力量对比。核心在于停止祈求，开始自爱。

## 停止跪舔与纠缠

上一节我们介绍了课程目标，本节中我们来看看具体的第一步：停止跪舔和纠缠。面对已经不在乎你的人，继续纠缠只会让你处于更被动的位置。

你进行跪舔和纠缠，根源并非爱得深刻，而是你自身**缺爱**，你在**祈求对方给予你关注和情感**。她的离开让你内心感到空虚，急于寻求填补。但这种行为只能感动你自己，无法打动去意已决的人。

这类似于一个乞丐在街上死死抱住你的腿乞讨。你的反应除了害怕，更多的是厌恶。因此，你必须停止对她抱有回心转意的期望。

以下是实现停止纠缠的几个关键点：

*   **放下不甘心**：接受感情已经发生变化的现实。
*   **不高估自己的存在**：不要认为对方没你不行。
*   **不低估对方的绝情**：认清对方当前的态度和决定。

## 重塑你的情感世界

我们已经明白了停止纠缠的重要性，接下来，我们来探讨如何修复内心并转向更健康的关系。你的世界并非非她不可。

你内心的空洞，完全可以通过**结识新的人**或**投入其他有意义的关系**来填补。与其对过去耿耿于怀，不如将目光投向自己和你周围真正关心你的人。

把时间和情感留给那些值得的人，你才能真正走出低谷，开始新的生活。

## 总结

本节课中我们一起学习了应对分手的第一步：立即停止跪舔与纠缠，并认识到其根源是自身的“缺爱”状态。我们强调了放下执念、认清现实的重要性，并指出转向自我关注和健康关系是走出困境、实现情感逆袭的根本路径。
