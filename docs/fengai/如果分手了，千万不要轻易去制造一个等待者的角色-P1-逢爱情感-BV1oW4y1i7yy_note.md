# 情感修复课程 01：分手后，请避免成为“等待者” ⏳

在本节课中，我们将要学习一个分手后的关键心态：为什么不应该让自己成为前任眼中的“等待者”。这个角色会严重损害你的个人价值和复合的可能性。

---

说一千道一万，在分手后千万不要去制造一个“等待者”的角色。分手后，你是否也跟前任说过“我会一直等你”这种话？不管你们是否还能联系，或是已经断联，都不要去跟前任说“我会一直等着你，这段时间不会再打扰你”。

这种话一旦说出来，其实就是在向对方表明：你可以把我当做一个不打扰你的备胎，当做池塘里的鱼，移动的暖宝宝，需要我的时候可以随时来找我。那么在他的心里就会很明白，你随时都在。

这也在无形之中，把自己放在了一个很尴尬的位置上。其实，就是在给自己挖坑，让前任认为你是一个没有价值的人。

上一节我们分析了成为“等待者”的弊端，本节中我们来看看正确的应对策略是什么。

这个时候，你更应该反向操作。如果你们已经没有办法联系，是需要断联的情况。

那么，在断联之前就应该向对方表明：我也会向前走，我也会遇到更珍惜我的人。如果说你在这段关系当中是付出更多的一方，那么这样做就会让你之前的纠缠和之后的不打扰，形成一个反差。

以下是这个策略可能带来的积极效果：

*   有非常大的概率让前任对你产生情绪反扑。
*   能引发对方强烈的失去感。
*   避免你成为一个毫无价值的人或一个被动的等待者。

---

本节课中，我们一起学习了分手后避免成为“等待者”的重要性。核心在于通过表明“我也会向前走”的态度，来塑造自己的价值感和独立性，从而可能引发对方的反思与失去感，而不是将自己置于被动、低价值的境地。
