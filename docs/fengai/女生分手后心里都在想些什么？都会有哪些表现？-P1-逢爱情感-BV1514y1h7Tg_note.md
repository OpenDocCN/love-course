## 🧠 第一阶段：提出分手与情绪宣泄

以下是此阶段常见的表述或行为：
*   诉说关系中的疲惫感或指出双方问题。
*   强调“不合适”等概括性原因。
*   提出“希望还能做朋友”的提议。

## 😔 第二阶段：情感残留与矛盾回避

此阶段的核心心理状态可概括为：

*   认为**分开的痛苦 < 在一起的痛苦**。
*   会回复消息，但会拒绝任何和好的请求。
*   处于一种纠结与疏离的过渡状态。

以下是此阶段的典型想法：

1.  **初始期望**：和平分手，互不打扰。
2.  **应对纠缠**：态度冷淡，保持距离。
3.  **解决骚扰**：采取删除、拉黑、说狠话等方式彻底断绝联系。
