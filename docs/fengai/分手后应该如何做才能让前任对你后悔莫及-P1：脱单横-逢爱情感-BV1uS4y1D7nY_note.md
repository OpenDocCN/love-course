# 分手后应该如何做才能让前任对你后悔莫及-P1：脱单横-逢爱情感-BV1uS4y1D7nY_note

在本节课中，我们将要学习分手后如何通过真正的自我提升，让前任对你刮目相看甚至后悔。核心在于理解什么是感情中的“核心竞争力”，而非仅仅关注外在变化。

---

分开后如何让前任对你从刮目相看到后悔莫及？我发现很多粉丝对“分手后重点在于提升自己”有着片面的理解。绝大多数人分手后的提升都放在了“我要健身、我要变美、我要变有钱、我要变独立”等方面。

这些只是基础。什么叫基础呢？我认为这是一个人就算不谈恋爱，为了自己也必须做的事情。这是为了自己需要具备的条件，要活得漂亮精彩，让自己每天看着都赏心悦目。我不是为了取悦别人而变成这样，所以这些东西在感情中并非核心竞争力。它们是你们在社会中所处位置的核心竞争力。不要肤浅地认为我是为了爱情才需要把自己变得更好。不是的，你是为了自己在这个生活、这个社会中拥有更好的位置、更高的高度、更宽阔的生存空间才这样做。

很多人把那些外在变化、经济财富的积累，当成了让前任后悔的唯一方式。这些东西不光在感情中，在任何方面都非常重要。但是，能让你们长久走下去的并不是这些东西，而是你的**情绪价值、情感价值、性格价值**。综合起来，就是你能够给对方带来的**精神价值**。你们两个人在彼此相处的过程中，让彼此感觉到相处是舒服的。

人与人之间的磁场非常重要。即使今天你各方面的硬条件都非常不错，但你们两人的磁场不对，相处起来非常别扭，那一样无法让你们走在一起。

我说的“提升自己”，说白了呢？就是你自己跟别人相容的能力、适应能力，包括你们对感情的认知。我并不是说让你们把感情吃透，而是你们在自己面对对方的一些行为状态时，可以接收到一些信号传达的意义，以及可以随机应变的一些应对措施。提升自己的情商，或者说，你们两个人在和好之后，还会不会再面临重蹈覆辙的情况？就是说，在之后你们彼此的相处过程中，你们处理感情问题的能力，能不能圆滑到让各自都觉得很满意？

你要提升的是别人跟你在一起相处的**舒适度**，是让别人大脑兴奋、情绪愉悦的能力。

以下是两个具体的例子：

*   **女生案例**：如果女生特别爱哭，特别喜欢意气用事，那你分手后要改变的，就不是说你外表多么好看，而是**能否消除曾经感情中无理取闹的模样**。你要提升的是让自己变得温柔、善解人意，懂得体谅和关心别人。
*   **男生案例**：如果男生在感情中比较幼稚，给不了女生想要的安全感，那么就让自己变得成熟，像个爷们一样，**有担当、有事业心、稳重、有主见**，能让一个女孩子有一定的依靠。成熟的做人做事都是三思而后行，所以你要理智。

你们真正要提升的，就是你们曾经在情感中，真正最致命的、各自身上的问题。如果你把重心放在“我要变美、我要变漂亮、我多有钱”上，其实你的前任不会因为失去你而难受。因为这个世界上从来不缺有钱的、优秀的、漂亮的。而具备这些特质却处理不好情感问题、一身伤的人多的是。

你要明白的是，你在这段感情中的**弱点**到底是什么？你的**核心价值**又是什么？你要提升的绝对不仅仅只是外在这些东西。

---

本节课中，我们一起学习了分手后自我提升的真正方向。关键在于提升内在的**情绪价值**和相处**舒适度**，解决自身在感情中的核心问题，而非仅仅追求外在改变。这才是让前任后悔的深层原因。
