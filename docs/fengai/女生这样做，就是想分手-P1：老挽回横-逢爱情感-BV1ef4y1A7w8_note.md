## 识别“逼分手”的行为模式

上一节我们介绍了课程的主题，本节中我们来看看具体有哪些行为可能意味着对方在试图结束关系。

当一方不愿再为关系投入时，常会表现出一些特定行为，其核心目的是**加速关系的终结**，并可能将“提出分手”的责任转移给对方。

以下是几种典型的“逼分手”行为模式：

*   **行为反向强化**：她会故意重复做那些你明确表示过反感的事情。公式可简化为：`她知晓你的厌恶点A -> 她刻意强化行为A`。
*   **情感冷落与抽离**：她开始主动疏远你，减少互动，表现出冷漠的态度。
*   **沟通渠道关闭**：她拒绝进行有效沟通，当产生误会或矛盾时，完全不给你解释或讨论的机会。

这些行为的共同心理动机是：**通过制造持续的失望和不适感，迫使你主动提出分手**，从而使她在心理上避免承担“关系终结者”的责任。

## 基础应对思路：“老挽回横”

识别出信号只是第一步。面对这种情况，传统的错误挽回方式往往会加剧关系恶化，而“老挽回横”则提供了一种不同的基础思路。

“老挽回横”的核心在于**停止无效追逐，转变固有姿态**。它并非具体的技巧，而是一种应对心态的起点，强调避免在对方逼迫时陷入被动反应的恶性循环。

其基本原则可以概括为以下代码逻辑：
```python
if 对方出现“逼分手”行为：
    停止（纠缠、辩解、讨好） # 传统错误反应
    启动（情绪稳定、边界建立、自我聚焦） # “老挽回横”基础态
else：
    保持正常互动
```
