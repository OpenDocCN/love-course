# 课程名称：如何打消前任的防御机制（第一部分） 🧠

在本节课中，我们将要学习如何理解并应对前任在分手后产生的心理防御机制，从而降低他对你的排斥感，为后续可能的沟通创造空间。

## 停止联系，创造空间

上一节我们介绍了课程的目标，本节中我们来看看第一步具体该怎么做。要打消前任的防御机制，首先需要停止主动联系对方。这能为对方提供一个安静的思考环境。你需要明白他内心的情感诉求是暂时不想被打扰。既然他表现出不想见你，就不要反其道而行之。

断联这一行为会让对方产生强烈的失去感。这种失去感会引发他的一些失落情绪，而这种情绪对你们后续的复合是有帮助的。

## 克服对“新欢”的焦虑

以下是关于断联期间常见担忧的分析：

*   **担忧**：如果你担心在断联期间，他遇到了更好的人。
*   **分析一**：这种担心没有必要。一方面，即便他有了新欢，新欢也不会轻易替代你在他心中的原有位置。
*   **分析二**：另一方面，在短时间内找到的新欢关系，根本不会长久。

挽回本质上是一场心理博弈，其核心公式可以概括为：

**成功挽回 ≈ 理解对方心理 + 稳定自我心态**

因此，你需要专注做好自己的事情，不要把所有的精力都放在纠缠对方和过度解读对方的行为上。

## 应对极端排斥情况

如果说你之前做过一些纠缠对方的事情，导致他对你讨厌至极，甚至把你拉黑删除了，那也不是没有解决的办法。这种情况只是需要更长的时间和更正确的方法。

## 理解感情发展的阶段：从排斥期到适应期

当你平稳地度过感情排斥期之后，关系就会进入下一个阶段——感情的适应期。

感情适应期通常出现在分手后的两周左右。此时，你已经接受了分手的事实，开始适应一个人的生活，慢慢习惯没有他的问候和关心。

感情适应期不像排斥期那样情绪极端，但是他的内心还会残存一些负面情绪，不可能快速地转变心态。他内心多少也会对你产生一些埋怨。

## 适应期的联系策略

有些人在这个时期会想频繁去联系对方，以为双方经过一段时间的冷静，就可以平静地交流。但是你要知道，对方愿意理你并不代表原谅你了。

以下是适应期联系的核心原则：

*   **风险**：如果盲目挽回或者频繁交流，造成的后果可能就是关系退回到冰点，继续僵化。
*   **策略**：所以，哪怕对方愿意回复你的消息，你也要适当地保持距离。
*   **身份定位**：就算是联系，也要以普通朋友的身份进行。
*   **关键动作**：根据对方的回应情况和反馈，做到适可而止。

本节课中我们一起学习了如何通过断联来降低前任的防御心理，理解了感情从排斥期到适应期的变化，并掌握了在适应期进行安全、适度联系的基本原则。记住，耐心和正确的策略是这一阶段的关键。
