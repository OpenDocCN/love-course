# 情感修复课程 01：极端纠缠后的断联策略 🚫💔

在本节课中，我们将要学习为什么在极端纠缠后需要采取断联策略，以及断联如何通过心理学原理帮助你扭转被动局面，重新获得关系的主动权。

## 为何需要断联？🔍

上一节我们介绍了课程主题，本节中我们来看看断联的核心原因。极端纠缠后需要断联，因为断联可以收回“聚光灯效应”，并创造“不可得性”。

具体来说，当对方提出分手后，如果你表现出情绪崩溃甚至纠缠不休、祈求不要离开，对方表面可能抗拒，但潜意识会产生一种感觉：**他已经吃定你了**。

## 对方的潜在心理 🤔

基于“吃定你”的感觉，对方会产生一系列延伸心理。以下是其可能产生的具体想法：

*   既然已经吃定你，他不必立即答应复合，正好可以放松休息。
*   他不想再关注你的事情，也不想再迁就你的感受。
*   他现在想去看看，外面是否有比你更好的对象。
*   他认为你这么离不开他，即使他兜转一圈后没找到更好的再回来，你也会在原地等待。

## 断联的作用机制：关闭聚光灯 💡

在对方眼中，他感受到的都是你的聚焦，就像有一盏**聚光灯**打在他身上，仿佛他是你人生舞台的唯一主角。

当他对此深信不疑时，如果你突然关闭这盏聚光灯（即开始断联），他的心理会产生巨大落差。他会疑惑：“你不是离不开我吗？现在真的要分开了？” 以前那个处处珍惜他、以他为中心的人，现在居然开始独自生活，不再围着他转了。

## 激发损失厌恶情绪 😟

对你而言，断联使你不再是一个他能轻易得到的人。这种落差会激发他的**损失厌恶**情绪。这就像人突然失明后，才意识到世界原本的五颜六色。

正是因为这种损失厌恶，反而会刺激他反过来关注你和你的生活，想要一探究竟：“你为什么能这么快放下？” 这种情绪，即使对方已有新欢，也依然会被激发和存在。因为在人的潜意识里，都不喜欢失去已经拥有的东西。

## 总结与行动建议 📝

本节课中我们一起学习了断联策略的心理学基础。我们了解到，极端纠缠会强化对方的优越感和安全感，而果断断联则能收回你的情感能量，通过创造不可得性和激发对方的损失厌恶，促使他重新审视你和这段关系。

如果你不确定对方当下的心理状态，或想了解更多情感修复的具体方法，可以进一步关注相关课程或进行专业咨询。
