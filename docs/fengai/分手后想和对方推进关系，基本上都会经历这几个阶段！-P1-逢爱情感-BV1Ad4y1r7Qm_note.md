# 分手后关系推进指南 🧭 - 课程01

在本节课中，我们将要学习主动提分手一方在分手后的典型心路历程。了解这些阶段，有助于你理解对方的心理变化，并采取合适的策略来推进关系，避免踩雷。

## 阶段一：敌对期 ⚔️

刚分手时，对方会进入一个敌对阶段。此时，他对你充满负面情绪。

他的态度也是负面的。这意味着，**在他眼中，你完全是一个负面的存在**。你们之间的关系是敌对关系。

## 阶段二：陌生人期 🧑‍🤝‍🧑

上一节我们介绍了充满敌意的初期阶段，本节中我们来看看关系的缓和。当负面情绪逐渐降低，他会将你视作陌生人。

## 阶段三：普通朋友期 👋

由陌生人阶段，关系可以转化为普通朋友阶段。这通常是我们建议尝试“复联”的起点。

以下是复联初期的核心原则：
*   以朋友身份为基础进行相处、交流和接触。
*   核心目标是打消他对你的戒备心。

## 阶段四：关系深化期 🔄

从普通朋友开始，关系可以逐步推进。通过聊天的深入和互动，可以从普通朋友过渡到朋友，再发展到亲密的好朋友，最终推进到暧昧关系。

在这个过程中，有一个至关重要的原则需要遵守。只要你没有到达暧昧或亲密好友阶段，有些事情就不能做。

以下是此阶段必须避免的“雷区”：
*   **不要主动聊感情话题。**
*   **不要谈论过去的恋情。**

因为一旦开启这些话题，会立刻将他拖拽回对你不利的负面印象中。他会想起所有不愉快的事情。这会导致本来正向发展的关系瞬间“打回原形”，迫使你从头再来，且过程会更加艰难。

## 阶段五：复合窗口期 💖

最后一个阶段是大家最关心的：何时可以具体复合？当关系推进到一定程度，会出现一些明确的信号。

以下是复合关键窗口期到来的核心标志：
1.  **昵称重现**：曾经的暧昧昵称重新出现。
2.  **进入私密空间**：他允许你进入他的卧室或查看他的手机等私密领域。
3.  **接受肢体接触**：他不排斥与你的牵手、拥抱等肢体接触。
4.  **热情具有唯一性**：他对你的热情是专一的。
5.  **探讨过去问题**：他开始能够与你探讨曾经感情中出现的问题，这表示他开始想要接纳你。
6.  **接受单独约会**：他不排斥与你单独吃饭、看电影。
7.  **积极反馈**：对于你们的聊天，他能够给予积极有效的回应。

**只有当你观察到以上多数信号时，复合的关键窗口期才真正到来。** 此时，即使你不提，对方也可能主动提出复合。

---

本节课中，我们一起学习了分手后关系推进的五个阶段：从**敌对期**、**陌生人期**、**普通朋友期**，到逐步深化的**关系深化期**，最终抵达**复合窗口期**。理解这些阶段并遵守各阶段的互动原则，是成功修复关系的关键。
