# 致爱情中迷茫的你-P1：老挽回8.21-逢爱情感-BV1yg411j71V_note

在本节课中，我们将要学习如何在感情迷茫时调整心态，通过一系列积极的行动和思维转变来关爱自己，走出困境。核心在于**自我提升**与**心态调整**。

如果你正处于感情的迷茫当中，我想告诉你，不要为了任何人委屈自己。我们辛辛苦苦地来到这个世界上，总会经历一些事情。人生就像一段路程，沿途会有很美好的风景，也会有很遗憾的事情。所以当你遇到困难的时候，需要采取行动。

上一节我们提到了面对困难，本节中我们来看看具体的应对策略。以下是处理不如意结果的几个步骤：

*   如果不是你想要的结果，那就去改变。
*   如果改变不了，那就去适应。
*   如果适应不了，那就默默接受。
*   如果不想接受，唯有让自己变得更好，才能改变现状。

在明确了行动方向后，我们需要将关爱自己落到实处。以下是几个具体的生活准则：

*   你要好好吃饭，好好睡觉，好好挣钱，好好地爱自己。
*   不为不值得的人伤心难过，不为不值得的事彻夜失眠。

换个角度看世界，也许真的就不一样了。你会发现，阳光依旧灿烂，天空依旧蔚蓝。希望你每天都保持微笑，因为除了生死都是小事。

任何事情都不要钻牛角尖，为难自己。生命必须要有裂缝，阳光才能照射进来。这要求我们重新梳理人际关系。

*   去爱那些对你好的人。
*   忘掉那些不珍惜你的人。

最后，让我们聚焦于自我成长。做一个心态积极向上的人，用正能量激发自己，感染身边的每一个人。当你越来越优秀的时候，生活也会反馈给你无限的精彩。

本节课中我们一起学习了在情感低谷时如何通过**停止内耗 -> 采取行动 -> 关爱自身 -> 转换视角 -> 优化社交 -> 积极成长**的路径来实现自我拯救。核心公式是：**自我提升 + 积极心态 = 更好的生活反馈**。慈心挽回，旨在让你成为更好的自己。
