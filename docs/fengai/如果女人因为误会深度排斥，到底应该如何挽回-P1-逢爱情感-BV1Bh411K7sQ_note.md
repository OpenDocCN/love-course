## 理解排斥的根源：认知隔阂与防御心理

上一节我们介绍了课程的核心目标，本节中我们来看看排斥行为背后的心理机制。

所有的不沟通都可能是认知隔阂所导致的。这种认知隔阂会引发强烈的防御心理。当你每次去找她时，她内心排斥的想法实质是：“你能不能别再烦我了。”

她的内心活动通常基于过去的负面经验，形成了一种固化的认知。以下是几种典型的内心独白：

*   **对改变的怀疑**：“反反复复说来说去都是这些东西，他怎么可能会真的改呢？”
*   **对动机的质疑**：“还不是为了跟我在一起，那么只要跟我一和好就会得寸进尺。”
*   **对压力的逃避**：“烦不烦，幸好我跟他分了，再这样下去我直接拉黑算了。”

**核心公式：排斥行为 = 认知隔阂 + 防御心理**

因为对方害怕重复之前的伤害，并且想要逃避你给到的压力，所以她每次和你聊天时，都会用之前的认知隔阂来防御你。

## 破局关键：摆正聊天的方式与位置

理解了问题的根源后，我们来看看解决方案。在对方排斥之下，聊天的关键应该是放在**摆正聊天的方式和位置**来打破对方的认知隔阂。

如果你可以把握住对方的心理动机，给她一种**舒适的陪伴感**，并偶尔让她怀疑自己的固有认知，从而去否定那些成见，那么即使你们在这个新的关系位置上产生新的冲突，她也不会轻易地再次离开你。

**核心逻辑：** 当你从“给她伤害的前任”位置，成功转换到“相处还不错的朋友”位置时，她对你的包容度就会逐渐升高。

## 实践步骤：以朋友身份进行平等交流

理论需要付诸实践。那么，具体该如何聊天呢？首先，必须把你们的身份摆在**朋友的位置**上。

以下是执行此原则时需要避免的常见错误列表：

*   **不要急着去否定和解释你的动机与目的。**
*   **不能明显地让对方看到你小心翼翼、如履薄冰的样子。**

如果你表现出上述行为，就会强化对方的“受害者思维”。这样不仅会导致她不自在，你也会感到不自在。因为你们的聊天已经违背了朋友间**平等、包容**的原则。

此时的交流根本不是两个独立的灵魂在对话，而是双方都还停留在过去的情绪与观念里。因此，彼此都不知道该如何面对对方，对话自然无法顺利进行下去。
