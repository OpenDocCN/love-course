# 课程名称：如何建立不可替代性（第一部分） 🧠💖

在本节课中，我们将要学习如何在情感关系中建立“不可替代性”，从而让你在意的人难以离开你。核心在于创造独特的价值，而不仅仅是依赖外貌或常规互动。

## 概述：什么是不可替代性？

想让喜欢的人无法离开你的重点，是**你需要具备不可替代性**。

什么是不可替代的价值呢？举一个简单的例子：那些需要排队三四个小时的餐厅，它们排长队的原因是**它们有的，别人没有**。这点放在感情里也是一样的。外貌决定能不能在一起，而**不可替代价值决定你们可以在一起多久**。

上一节我们介绍了不可替代性的核心概念，本节中我们来看看如何具体创造这种价值。

## 创造不可替代价值的三种方法

以下是三种建立不可替代价值的具体途径。

### 1. 创造独特的共同经历与成长

首先，你可以去创造属于你们之间**特别的共同经历和成长**，从而建立起深厚的羁绊。这个经历一定要是特别的。

别总是说吃饭、看电影。做这些事，你跟其他人能有什么区别？以下是几个特别的例子：
*   你们一起奋斗过。
*   你们一起熬过很困难的时光。
*   两个人一起花了几个月时间，拼了一个特别复杂的乐高玩具。

核心是：**发挥你的认知，去尽可能让他跟你一起体验他从未体验过的事物**。这就会在很大程度上增加你的不可替代性。我一直相信付出会有回报，没有回报的话，那一定是你没有付出到点子上。

### 2. 从相处模式上建立价值

其次，你可以从相处模式上建立自己的不可替代价值。有一个小技巧是：**你需要与他保持一致性**，让他觉得你跟他就是一路人。

以下是具体做法：
*   浅层次模仿他的行为、语言习惯。
*   了解他的兴趣爱好。
*   使用他常用的表情包和口头禅。

当你开始研究他的逻辑习惯之后，还有一个好处：**你会发现你越来越懂他**。当你可以用他的逻辑去说服他的时候，你就会发现什么问题都会迎刃而解。用对方的思维跟对方去谈恋爱，你会觉得一切都好简单。

### 3. 建立互补性

第三点是互补性。一致性可以带来新鲜感和情绪价值，但若想长久在一起，**拥有互补性就显得格外重要**。

你弥补了他空缺的那一块，他会觉得和你是可以有未来的，因为你弥补了他性格和生活上的缺失。当然，如果你还能在工作上和经济上帮助到他，那么就是最好的互补。

## 总结

本节课中我们一起学习了建立情感不可替代性的三种核心方法：**创造独特共同经历、在相处模式上保持一致性、以及建立互补性**。如果你把这三点都掌握好，别说复合，他这一辈子都可能离不开你。
