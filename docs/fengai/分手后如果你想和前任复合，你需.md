# 分手后如果你想和前任复合，你需

在本节课中，我们将学习分手后如何建立正确的挽回心态与行动思路。核心在于调整自身状态，以理性、渐进的方式重建关系，而非盲目纠缠。

## 建立良好心态是第一步 🧠

上一节我们介绍了课程概述，本节中我们来看看复合的第一步：心态建设。在不同的心态下做同样的事情，结果必然不同。有时并非对方不爱了或毁灭了关系，而是你的心态过于着急。

我们可以问自己一个问题：我现在做的事情，到底是在满足自己的个人焦虑，还是真正在为对方做让他开心、舒适的事情？我们可以把分手简单理解为两个人走不下去了，可能因为各种争吵。

也就是说，在这段恋爱中的舒适感全无。如果一个人想跟另一个人重新开始或寻找新对象，他需要的正是舒适感。没有一个好的舒适感，对方不会跟你在一起。因此，心态建设非常重要。

你需要把自己代入对方的身份，与他互换位置。思考一下：如果是对方来挽回你，他做什么样的事情，能够让你真正意识到他是爱你、想跟你走下去的呢？思考这个问题就足够了。

## 以正常人的方式解决问题 🤝

在理解了心态的重要性后，我们来看看具体的行动原则。我们需要用正常人的方式来解决问题。什么叫做正常人的方式呢？因为我经常看到你们不断去纠缠对方，写一些过度承诺的话语，我认为这些是非常犯规的方式。

一个正常人分手后需要思考、需要疗伤、需要治愈自己、需要反复复盘这段感情，而不是没头没脑地去做复合这件事。这样做只会让你自身陷入死循环，对方也不会给你复合机会，所以你的情况会越来越差。

如果你现在正处于焦虑情绪，不妨把这件事暂缓一下。你先处理自己的焦虑，让自己真正开心起来，再去做这件事，那么你的结果就会好很多。

## 避免用力过猛，循序渐进 🔄

学会了理性处理问题后，我们还需要注意行动的节奏。我们不要去用力过猛。我们都知道，分手就是要经历戒断这个人的时期。

他在这个时期时疯狂地想要去找别人，就会做出非常多乞讨的事情，即为了感动对方。其实，你往往做一些感动对方的事情，他真的不会感动，反而把你自己感动了。你还觉得：“我做了好多呀，对方为什么不为所动？”

首先，他清楚地知道你是什么样的人，他知道你不会这么快就改变。然后你这个时候做了一通操作，每件事都挺“感动”，但你一次性做这么多事情又有什么用呢？

**真诚和真心都是循序渐进的**。一段感情从破裂到重新开始也是循序渐进的。我们在哪个角度、在哪个位置，做什么样的事情就可以了。

---

本节课中我们一起学习了复合所需的三个核心心态与行动原则：**建立良好心态**、**以理性方式解决问题**、**避免用力过猛并循序渐进**。记住，关系的修复是一个过程，找准自己的位置并采取合适的行动至关重要。
