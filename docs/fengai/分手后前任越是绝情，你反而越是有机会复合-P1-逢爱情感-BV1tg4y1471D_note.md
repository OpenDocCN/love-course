# 分手后复合指南：当对方绝情时，你反而更有机会 🔄 - 课程01

在本节课中，我们将要学习一个看似矛盾的现象：分手后，前任表现得越绝情、越冷漠，你成功复合的机会反而可能越大。我们将深入分析其背后的心理逻辑，并重点指出在复合过程中必须避免的几个常见禁忌。

## 核心观点：绝情背后是未消散的情绪

分手后对方越绝情，就越有机会复合。无论他对你说“这辈子我都不可能再跟你在一起了”，还是说“没有你我才是最幸福的”，只要他说得越狠，你就越有机会复合。

有句话叫做“无欲则刚”。如果他的心中连恨都没有了，那么你才是彻底没有机会了。

上一节我们介绍了绝情背后的机会，本节中我们来看看为什么很多人没有成功复合。

很多人面对对方越来越冷漠的态度后，就彻底丧失了信心。其实，你不用因此而感到绝望。你有可能只是触碰到了下面几个复合的禁忌。

以下是四个在复合过程中必须避免的关键禁忌：

1.  **禁忌一：乱用断联**
    你知道断联意味着什么吗？断联就是切断了跟前任所有的联系。如果在断联以前你什么准备工作都没有做，直接就断联了，那只能得到一个结果：彻彻底底的断联。我们断联的目的是要激发对方的复合欲望，盲目的断联只会让对方认为你已经接受了分手，进而也收拾好心情去迎接他美好的未来。

2.  **禁忌二：装可怜博同情**
    我知道分手以后，你确实很痛苦。你可以消沉一段时间，释放一下自己的负面情绪。但是，你千万不要将自己的落魄现状告诉你的前任。你得明白，永远都是好的状态在影响着差的状态。你给对方表现出自己伤心难过的状态，不是让他回来，而是在进行情感勒索。而且，你做得越过分，就越能成功地让前任想要远离你。

3.  **禁忌三：不认可对方**
    很多人不认可对方，试图想要改正对方。两个人都走到分手的这一步了，还想着要改正对方。总喜欢把对方一点点的小毛病放大了说事。说直白一些，就是不认可自己的伴侣。其中还透露出一种不信任和不尊重。你要记住，谁也不会跟一个不尊重自己的人在一起。要是你复合的过程当中这么去做了，那么你得到的就是他会越来越冷漠和无情的态度。

4.  **禁忌四：自以为是的爱**
    很多人在分手之后，行为模式就跟没有分手似的。有一句话是这样说的：“有一种冷，叫做你妈觉得你冷。”在爱情当中就叫做：**有一种爱，我觉得我爱你**。
    *   例如，明明对方晚上八点下班，晚饭都在公司解决完了，但是你非要每天做好饭，等他回家吃饭。
    *   例如，明明对方想要一个人静一静，你愣是恨不得24小时贴在人家的身边。

    你确实付出了很多，但是你这个付出都是为了满足你自己的付出欲，而不是基于对方的需要。明白了吗？

本节课中，我们一起学习了如何解读前任绝情行为背后的心理，并详细剖析了四个会严重阻碍复合进程的禁忌行为：乱用断联、装可怜博同情、不认可对方以及自以为是的爱。理解这些要点，是迈向成功复合的第一步。
