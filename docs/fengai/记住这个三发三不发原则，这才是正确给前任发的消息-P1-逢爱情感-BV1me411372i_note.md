# 复合沟通指南：消息三发三不发原则 📱

在本节课中，我们将学习在尝试与前任复合时，如何通过发送信息进行有效沟通。核心内容是“信息三可发三不可发”原则，掌握这些原则能帮助你避免雷区，建立舒适连接，为感情回暖创造机会。

你们在复合过程中可能无法见面，打电话对方也可能不接。此时，唯一能操作的就是发送信息。因此，如何发送信息变得特别重要。

## 三种不可发送的信息 🚫

以下是三种前任最不想看到的内容，发送这类信息极易引发反感。

1.  **不要谈论感情**：信息中千万不要去聊感情，一丁点暧昧的感觉都不要去触碰。
2.  **不要疯狂道歉**：他跟你分手无非就是不想再跟你继续了，想跟过去道别了。
3.  **不要卖惨诉苦**：不要说自己现在有多惨，多么的难受，多么的痛苦。你说的这些跟他一点关系也没有。不是他绝情，只是他现在固执地认为，你跟他在一起会更惨，会让他更痛苦。

## 三种可以发送的信息 ✅

上一节我们介绍了需要避免的三种信息，本节中我们来看看哪些信息是适合发送的。其中最后一点是关键。

1.  **能挑起他兴趣的话题**：你记着，并不是说他喜欢什么，你们就去聊什么。那叫投其所好，又进入到了“舔”的局面。
2.  **聊对他有好处的内容**：那什么叫做有好处呢？就是能在物质上或者是能在精神上让他能够感觉到对他有帮助。例如：`“听说你最近在忙XX项目，我刚好认识一个这方面的专家，需要的话可以介绍。”` 这个事情一定是重要且他难以解决的，不要说无关痛痒的话，否则就是刻意的献殷勤。
3.  **聊自己的改变（间接展示）**：聊自己的改变，但是不要刻意去聊，而是不经意间让对方感受到。再配合一些必要的社交动态来展示，去让对方发问你的变化，把它变成谈资的内容。机会如果不成熟，记得千万不要去主动说。等到对方对你的改变产生了一些兴趣的时候再去说。

有了这些舒服的连接，你们的感情才能够逐步的升温。

## 总结

本节课中，我们一起学习了与前任沟通时的“信息三发三不发”原则。核心是：**避免谈论感情、道歉和诉苦**；**多聊对方感兴趣、对对方有益处的话题，并间接展示自身积极改变**。遵循这些原则，可以帮助你建立更有效的沟通，为关系修复铺平道路。
