# 情感挽回教程 第1课：接受事实，先放他走 💔

在本节课中，我们将要学习分手后挽回的第一步，也是最关键的一步：如何正确面对分手事实，停止无效纠缠，为后续的挽回创造可能性。

分手后试图挽回的人，最常说的一句话是：我主动去找他了，但他的态度很坚决。我已经卑微到了极致，说了我有多么离不开他，可他依然无动于衷。他真的可以这么狠心吗？说这种话很幼稚。

分手，只代表你们从前的关系结束了。但你们的情绪、情感以及相互的认知并没有结束。有人可能会问，什么时候才算是真正的结束？其实，只要你还想坚持，那么对你而言，这段感情就没有结束。

然而，直接向对方展示你的“离不开”，这种做法无疑与直接伸手要钱没有区别。那么，怎样做才是正确的呢？接下来要说的内容并不复杂，但真正做到却不容易。核心方法是：**接受事实，先放他走**。

我知道很多人此刻会按捺不住。一个人的离开，就像心里缺了一块，带来剧烈的不安和难受，迫使你想要立刻填补这份空缺。因此，你会拼命去纠缠。但这种情况下的结果只会适得其反。

因为强烈的丧失感会让你的情绪变得不稳定。你会努力去讨好、去迎合。但对方的感受是：你带有强烈的目的性。他会因此产生抵触和逃避心理。

所以，在这段时间里，你应该做的是：给自己设定一个期限。在这个期限内，尽量不要去找他。

哪怕你正在经历戒断般的痛苦，但至少在这段时间里，你能够观察和感受自己情绪的起伏，尝试与自己的情绪相处。当你能够合理调节自己情绪的时候，你也能够慢慢看清他离开的真正原因，并慢慢学会爱自己。

不患得患失，不草木皆兵，你自然能够更客观地感受你和他之间的点点滴滴，并为自己营造出相对愉悦的情绪状态。走到这一步，自然会迎来修复从前关系的最佳时机。
