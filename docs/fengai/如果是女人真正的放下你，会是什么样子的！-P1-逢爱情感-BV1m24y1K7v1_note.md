> 人心都是慢慢变冷的，树叶是渐渐变黄的。

## 表现背后的原因：累积的失望

感情里所有的离开都是早有预谋的。女人在跟男人说分手之前，肯定已经给过男人无数次的机会。然而，女人一次又一次地给男人机会，对男人一次又一次地忍让，最终换来的也只不过是对男人一次又一次的失望。

以下是失望累积过程的简要描述：
*   被男人一次又一次无情的伤害。
*   男人不仅不懂得珍惜女人，而且还把女人对自己的爱与喜欢，用来当做自己犯错的保障。
*   男人还把女人对自己的包容与心软，用来当成自己犯错的筹码。

所以，男人根本就不明白，女人在下定决心跟男人分手之前，在选择离开男人的时候，那肯定是已经感受到了无数次的失望，历经过了无数次的伤害了。用公式来描述这种状态就是：

**失望累积总量 = ∑(每一次伤害 × 每一次期望落空)**

当这个总量超过个人承受的**情感阈值**时，离开的决定便会产生。

```python
if 失望次数 >= 承受阈值:
    情绪状态 = “平静” # 而非激烈的悲伤或愤怒
    行为表现 = “断绝联系” # 彻底而干净
```
