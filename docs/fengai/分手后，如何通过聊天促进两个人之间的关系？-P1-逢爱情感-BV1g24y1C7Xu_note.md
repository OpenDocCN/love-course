# 分手后关系修复指南 - 第1课：重启联系的艺术 💬

在本节课中，我们将学习分手后如何通过恰当的沟通来缓和关系，为后续可能的修复打下基础。核心在于**建立平等对话**，避免陷入纠缠或讨好的被动局面。

---

分手后重启联系的重点，是建立两个人平等对话的关系。一般来说，首先提分手的人会在这段关系中拿回主动权。而一个对你有强烈既定印象、且负面情绪处于顶峰的人，是不适合频繁联系的。

消息不回、打电话不接、道歉不听，这都是很常见的状态。这个时候如果你还去纠缠，就会完完全全地陷入被动。被动的一方会慢慢滑到卑微的角色里，能够做的事情就只剩讨好了。

但是，讨好在感情里是让人厌恶的，特别是在分手的时候。所以，这个时候你不要去联系，也不要去道歉，更不要去提你们的以后。你现在所做的一切行为，在对方看来都是不尊重他的。没有人想和只会纠缠他的人重归于好。

那么你要做的，是分手后拿出几天的时间完全不联系。目的是让对方的情绪降降温，给彼此一个情绪缓冲的时间，降低他对你的负面印象。这样你再去找他的时候，最起码他的态度是会好很多的。

如果他还提分手，那么我们可以把下面这段话发给他。

以下是建议发送的消息内容：

> 我这两天想了很多，我尊重你的决定。我也觉得应该把我们两个的事情先放一放。你做你的事情，我也做我的事情。

你就只说这么多，其他的什么都不要再说了。中间不要提你们的关系，也不要再提你的心情。这些都跟他没有任何的关系，他也不想关心你现在到底是什么样的状态和心情。

说这一句话，就是为了把你带到一个平等的位置上，表达你依然很在意这段感情，在意他。一个刚分手的人，就算他看起来特别冷静和绝情，冷静个四五天，情绪泄了劲，心理上剩的就只是情感的习惯了。

这种情感习惯爆发的时候，就是你最好的一个时期。

然后，你再去适当地发一些问候的消息。简单的问候就行了，不要再长篇大论。不要管他是什么时候回你的，一两个小时或者是隔天才回你。只要是他回你了，就是他的感性占了上风，那么就是你挽回的机会了。

最后，再去检查你的动态是不是被他看过了。一个跟你分手的人，还是会对你有好奇心，只是因为他想要了解你的状态。所以，我们要去看看他是否留下了这个蛛丝马迹。如果有，我们不要有什么反应。

分手后，两个人断联了一段时间，由于突然的情感缺失，再理智的人，也都会出现一些反复的反应。但是在他反复的前提下，你要很稳，没有什么动静。回复信息，简短干练，距离保持得很恰当。

只有你的淡定不纠缠，才能够让他产生后悔的心理。那么到了这一步，重启联系，就到位了。这个时候，你们就能够平等地进行对话了。

---

本节课中，我们一起学习了分手后重启联系的核心步骤：**断联降温 -> 发送尊重决定的消息 -> 简单问候 -> 保持淡定与距离**。关键在于通过行为将自己置于平等位置，而非纠缠讨好，从而为关系的缓和创造可能性。
