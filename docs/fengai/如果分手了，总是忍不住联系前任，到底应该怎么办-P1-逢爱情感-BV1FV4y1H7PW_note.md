# 分手后如何停止联系前任？ - 课程01

在本节课中，我们将学习如何应对分手后忍不住联系前任的冲动。我们将分析这种行为的心理根源，并提供一系列清晰、可操作的步骤，帮助你走出困境，专注于自我成长。

## 理解现状：为何对方不主动联系？

上一节我们介绍了课程目标，本节中我们来看看当前状况的本质。

既然对方没有主动找你，那么很可能他此刻心里没有你，或者他想找你的念头没有你那么强烈。如果他心里真的有你，他一定会主动联系你。

**核心公式**：`对方主动联系的概率 ≈ 对方对你的在意程度`

你在独自难过，对方可能在外一切如常。你之所以如此痛苦，是因为分手后，你只记住了曾经美好的回忆。

## 调整视角：回忆完整的过去

理解了现状后，我们需要调整看待过去的视角。

现在，请你想一想他是如何伤害你的。如果你们所有的回忆都是美好快乐的，你们也不至于走到分手这一步。

以下是帮助你调整视角的两个关键问题：

1.  你放不下的真的是他这个人吗？
2.  你真正放不下的，是不是有一个人陪伴的感觉？如果是，你应该去寻找一个新的人来陪伴你。

也有可能，你放不下的不是那个人，而是自己曾经对这份感情的付出。你不甘心付出了那么多，最后却毫无结果。你也不愿意承认自己当初看错了人。

## 采取行动：专注于自我提升

分析了心理根源后，本节我们将探讨具体的行动方案。

其实你无需不甘心，完全可以好好珍惜下一个机会。你现在无论是想和好，还是想开始新恋情，都应该先把精力放在如何过好自己的生活、如何提升自己上面。只有这样，你才能真正走出来。

**行动代码**：
```plaintext
if (feeling urge to contact ex):
    focus_on(self.improvement)
    engage_in(new.activities)
```

以下是你可以立即开始的行动步骤：

*   **转移注意力**：培养一个新的爱好或技能。
*   **设定目标**：为自己设定短期可实现的个人或职业目标。
*   **扩大社交圈**：与朋友、家人保持联系，认识新朋友。

## 课程总结

本节课中，我们一起学习了如何应对分手后联系前任的冲动。我们首先理解了对方不主动联系的含义，然后学习了通过回忆完整过去来调整心态，最后明确了将精力转向自我提升是走出来的关键。记住，向前看的第一步，永远是先照顾好自己。
