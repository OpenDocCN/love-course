# 情感课程 01：背叛的后果与自我价值 🧠

在本节课中，我们将探讨亲密关系中的背叛行为及其长期影响。我们将分析背叛者的心理动机、行为后果，并讨论被背叛者应如何维护自我价值与尊严。

---

上一节我们概述了课程主题，本节中我们来具体看看背叛者的行为逻辑与潜在结局。

背叛过你的人不会有好的结局。别看他当下与别人在一起似乎很开心，将来必定会后悔。原因在于他没有心，他只爱他自己。他不会珍惜任何真心待他的人。

时间久了，那些付出真心的人并非傻子，都会逐渐收回对他的好。当他消耗完所有真心待他的人后，处境只会越来越糟。

他轻视那些曾真心爱他的人，这些人迟早都会离开他。最终，他会遇到一个比他段位更高、能力更强的人。

这个人会彻底掌控他。那时，他过得不好会想起谁？他会想起你，并回来找你。他以为你还会像过去一样对他好，他以为回来仍能做你心中的女神。

但你需要明白，成年人在感情世界里没有无心之过。

**所有的背叛与外遇都是故意的。** 当他选择背叛的那一刻，内心就已非常清楚这必然对你造成伤害。他甚至清楚这种伤害可能毁掉你的一生，但他仍然选择了背叛。

因此，你应该认清现实：他在对你负责和满足私欲之间，选择了自私自利。

以下是面对背叛时，两种不同选择带来的结果：

*   **如果你没有骨气，选择原谅并接受他**：他会过得很好，你的妥协可能助长他的行为。
*   **如果你有骨气，选择离开并坚守原则**：你便用实际行动告诉他，践踏他人真心会付出何种代价。

我知道你现在很痛苦，但过段时间再看，你会发现他的生活已陷入困境。

此时，如果你想让他回来，你可以选择低头与复合。如果不想让他回来，那么生活自会磨平他的棱角。

---

在感情中，对他人好并非没有优势。

**优势在于：你的真心有机会换来另一颗真心的相待。** `真诚付出 -> 可能获得真诚回报`

---
