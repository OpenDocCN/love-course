# 情感课程01：为何付出不被珍惜？💔

在本节课中，我们将要探讨一个常见的情感困惑：为什么男生为女生付出很多，却往往不被对方珍惜。我们将分析其背后的根本原因，并提供清晰的思路。

## 付出过多的后果

上一节我们介绍了本课的核心问题，本节中我们来看看过度付出的具体表现。你以为对她好，她就会更加爱你。你以为把所有的爱都给了对方，她就再也不会让你离开。

**根本原因**往往是你付出太多。

## 关系失衡的原理

你以为只要忍让就可以留住她。你以为日久就可以生情。你以为痴心可以换来真情。

结果却换来一次又一次的失望。

以下是导致关系失衡的几个关键行为：

*   主动太多，把自己变得廉价，变得可有可无。
*   你越是低姿态，她就越是高姿态。
*   你越是无条件对她好，她越是把你的好当做理所当然。
