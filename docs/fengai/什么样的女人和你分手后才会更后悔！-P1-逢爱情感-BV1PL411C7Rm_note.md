## 核心原理：缺失感与挫败感的作用

上一节我们介绍了课程目标，本节中我们来看看其背后的心理学原理。

如何让女人后悔失去你？方法很简单：制造缺失感和挫败感。为什么这两种感觉会起作用？因为女性在拥有时，往往对拥有的美好感觉并不明显。许多女性内心有一个信念：她认为自己拥有的一切美好都是应得的。

因此，只有让她真切地感受到“失去”，她才会感到不适应。同时，她才会产生挫败感。进而，她才能回忆起你曾经对她的好，并真正地放不下你。

理解了核心原理后，我们来看看男女在情感触发点上的根本不同。

男人和女人在这一点上比较相反。对于男人，能刺激其情绪的是“得到”。对于女人，“失去”更能让她产生情绪。就像男人追求女人时动力十足，得到后非常兴奋。

于是，你越是努力，机会就越渺茫；越是努力，目标就越遥远。

明白了理论差异，接下来我们进入实践部分。以下是让前任产生后悔情绪的具体步骤。

你需要做的第一件事是停止关注对方。这是第一步。先放下你的不甘心，不管分手原因是什么。

我知道此时你总想做点什么，不做就很难受，甚至寝食难安。这很正常，直接进行到第二步：去对另外一个人好。并且，你要想方设法让你的前任知道这件事。关键点在于，你对现在这个新人的好，要比对前任更好。

## 心理反应与最终应对

执行了以上步骤后，对方会产生一系列心理变化。以下是其可能的心理活动及你的应对策略。

此时，你的前任心里会产生不平衡。她会越想越不平衡，并产生诸如“我跟你在一起时，你怎么没有这么对我？”“你还说爱我，我们在一起几年，难道比不上这个刚认识几天的人吗？”的想法。她心里会产生非常大的挫败感。

当然，她嘴上可能依然会说“这跟我有什么关系”，但心里会久久不能释怀，反而越来越放不下你。用不了多久，她就可能接受不了，主动过来试探你。

到这里还没有完全成功。你要记住，无论她怎么试探，你都要表现得风轻云淡。此时的心态应该是：不主动、不拒绝、不着急。
