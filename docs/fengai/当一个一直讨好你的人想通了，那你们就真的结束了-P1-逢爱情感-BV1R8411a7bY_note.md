# 情感关系课 01：当讨好者“想通”时，关系就真的结束了 💔

在本节课中，我们将要学习一段关于亲密关系结束过程的深刻剖析。我们将探讨当关系中一方停止“讨好”并真正“想通”后，双方的心理变化轨迹，以及这为何通常意味着关系的彻底终结。

## 核心建议：请听完这段话

我希望你能听完我接下来说的话。这是过来人给你们的一个建议。

## 反思分手的原因

我不知道你们分手的原因到底是什么。可能是他一次又一次地让你失望，或者你觉得你们两个人很难磨合，你想要及时止损了。再或者，是你身边出现了一个你觉得更加吸引你的人。

以下是你可以进行的反思步骤：
*   **暂停与思考**：你可以先暂停一下，好好地想一想这段感情。
*   **归因分析**：失望的责任在谁呢？如果你觉得主要的原因在于对方。
*   **评估对方态度**：那么他现在想要跟你沟通，想要解决问题吗？他的态度不够坚决，不够诚恳吗？

## 给关系留一个“观察窗口”

我知道你现在可能很讨厌他，嫌他烦，不想再被他打扰。你可以不理他，可以屏蔽他。

但是，我希望你至少可以给他留一个可以证明自己的窗口。你可以见证他慢慢变好的过程。这不是吊着他，我也不是在替他讲话。

**核心心理公式**：`当下的解脱感 ≠ 未来的无悔`

而是你真的很大概率会后悔现在的这个阶段。

## 心理角色的动态转换

你可能会觉得是解脱，如释重负，很轻松。你认为他应该每天都睡不着觉，一直回忆着之前在一起美好的画面，无心工作，整个人都废了。

但是，持续了一段时间之后，你们的角色会变换。

别不信，人都是先失去才知道珍惜的。其实，他也知道找一个新人比挽回前任更加容易。但是他更加的勇敢，更加的真诚，也更加的懂得珍惜。

**核心行为描述**：至始至终，他的手一直都是在这样伸着。伸得太久，他也会觉得很累，就会收回来。

## “想通”后的结局与你的后悔

等到他熬过这个阶段，他就会觉得一个人挺好。

而你，慢慢地就会开始想念他，开始愧疚自己当初那么的绝情冷漠。你会发现，好像你们两个人之间也没有什么不可调和的矛盾吧。你会后悔当初真的没有必要因为这些小事跟他发脾气、打吵打闹。

你会意识到，后来遇到的所有的人，都没有他那么的理解你、包容你，甚至爱你胜过爱自己。其实，换个人，相同的流程再走一遍，也挺没意思的。

**关系终结的判定式**：`当他想通并收回手 + 当你想回头 = 关系彻底结束`

那么，等你想通了，想回头了，你们之间真的就是彻底结束了。

## 总结与互动

本节课中，我们一起学习了亲密关系结束时，从“讨好”到“想通”这一心理转变过程如何不可逆地导致关系终结。我们分析了双方的心理动态，并强调了当下决定可能带来的未来悔意。
