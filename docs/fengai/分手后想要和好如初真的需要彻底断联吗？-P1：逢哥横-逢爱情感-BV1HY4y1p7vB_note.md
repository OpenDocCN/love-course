# 💔 分手挽回课程 P1：断联策略的适用性分析

在本节课中，我们将要学习分手后“断联”策略的适用性。我们将探讨断联并非适用于所有人和所有情况，并分析哪些性格类型和关系模式更适合或更不适合采用断联策略。

## 断联并非万能策略

分手后采取彻底断联的策略并不适合所有人。在某些情况下，断联可能产生反效果。

如果断联后，对方产生“你根本不在乎我”的想法，认为在一起时就被冷落，分手后长时间不联系更印证了其不重要，那么对方可能会彻底删除、拉黑你，并从你的世界消失。

因此，是否采用断联需要分情况讨论，必须依据双方的相处模式和彼此的性格特点来决定。

## 不适合断联的性格类型

上一节我们介绍了断联的潜在风险，本节中我们来看看哪些性格类型的人不适合作为断联对象。

如果你的分手对象是**非常理性**的人，平时喜欢思考，做事考虑周密，那么他/她决定一件事花费的时间较长，一旦决定就很难改变。

以下是此类性格的特点：
*   性格理性，善于深度思考。
*   做事周密，决策过程长。
*   决定后不易改变。

对于这种性格的人，断联并不适合。因为你断联后，对方会认为自己的决定是正确的——“你没有再联系他，那以后就这样吧”——从而导致分手更加彻底。

## 适合断联的性格类型

了解了不适合断联的情况后，我们再来看看哪种性格类型的人更适合作为断联策略的对象。

**冲动型、不理智**的**情绪失控型**人格更适合采用断联策略。

断联的目的在于：
1.  给你自己一段冷静的时间。
2.  让他/她也能够仔细思考。
3.  在断联期间，认真反思自身问题，努力提升和改变，为重新吸引和挽回对方做准备。

## 总结与后续

本节课中我们一起学习了断联策略的适用性。关键点在于：断联不是通用法则，需根据对方**理性深思型**或**冲动情绪型**的性格来判断是否适用。
