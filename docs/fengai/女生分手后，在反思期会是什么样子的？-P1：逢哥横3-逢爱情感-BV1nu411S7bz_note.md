## 反思期的心理状态 🔄

分手后一到两个月的时间是她的反思期。  
此时她能想到一万个理由去帮助你和她进行合理化。  
其目的是告诉自己，感情出现问题是两个人的错，不能只怪你。  
因此她会产生引领的愧疚感和后悔的意思。

一旦产生这种感觉，对你的怀念就会慢慢上来。  
这与一开始的否定倾向完全相反。  
她会想起你身上好的地方，以及你们曾经拥有的美好回忆。

## 反思期的核心行动策略 🛠️

在之前的阶段中，你已经铺垫了一些聊天的舒适感。  
本节中我们来看看，此阶段不能仅依靠口头交流，而需真正行动起来。

以下是本阶段必须执行的两项核心行动：

1.  **加大情感投资，创造安全感**  
    例如，在出差时送她一份礼物，让她知道你经常思念她。  
    安全感公式可概括为：**安全感 = 持续关注 + 具体行动**。

2.  **投入时间陪伴，刷存在感**  
    你再忙也要抽出时间陪伴她。  
    无论她情绪如何，坚持待在她身边。  
    代码逻辑可表示为：

女人心不可能不心软。  
当她一旦软下来，接下来就可以得寸进尺挽回升级关系了。

大部分的女生分手后都要经历这三个阶段。  
但时间不是死规定，而是因人而异的。
