# 💔 分手后应该隔多久找对方 - 课程 P1

在本节课中，我们将要学习关于分手后如何确定联系时机的基础知识。我们将重点探讨真性分手后的处理原则，帮助你理解“断联期”的意义，并分析在此期间可能产生的担忧。

## 核心原则：真性分手后的“断联期”

上一节我们介绍了课程主题，本节中我们来看看具体的操作原则。如果你们的分手属于真性分手，那么建议你在分手后的一个月内，都不要主动联系对方。

这样做的核心目的是：**消除对方在分手时对你积累的负面情绪和印象**。公式可以简单理解为：
**挽回成功率 ∝ 负面印象的消除程度**

## 关于“断联期”的常见担忧分析

理解了断联的必要性后，许多人会产生一些疑虑。以下是针对这些常见担忧的具体分析：

1.  **担心被遗忘**  
    无需担心对方会在这段时间忘记你。忘记一个人是非常困难的事情。

2.  **担心感情基础消失**  
    更何况你们曾经关系亲密，他不会轻易忘却这段共同经历。

3.  **担心对方寻找新欢**  
    也不必担心你不联系他，他就会转向他人。他的吸引力可能被你因为爱而主观放大。

## 极端情况下的价值判断

退一步分析，如果出现最坏的情况，也恰恰能帮助我们做出判断。

如果他真的在断联期内迅速开始新恋情，那只能说明他在你们尚未分手时，就可能已经与他人暧昧不清。

此时，你可以用这个逻辑进行判断：
```python
if 他在断联期迅速找到新欢:
    大概率存在过往暧昧
    挽回价值需要重新评估
```

对于这样的人，你需要认真考虑是否还有挽回的必要。
