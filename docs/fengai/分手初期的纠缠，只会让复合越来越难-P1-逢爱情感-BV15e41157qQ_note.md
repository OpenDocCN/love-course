# 分手初期的纠缠，只会让复合越来越难-P1-逢爱情感-BV15e41157qQ_note

在本节课中，我们将要学习一个在分手初期非常普遍的心理现象——“损失厌恶效应”。理解这个概念，是避免错误挽回行为、为未来可能的复合打下良好基础的第一步。

经历过分手的人都明白，分手初期是很痛苦的。这种痛苦会促使你第一时间想要去挽回，而不是好好想一想，究竟是哪里出了错误，为什么要挽回。

其实，这个心理叫做**损失厌恶效应**。这种效应体现的心态就是：你没有得到一个东西的时候，你可能并没有那么想要得到。然而当你真正拥有后，突然将这个东西从你身边抽离了，你就会非常的不舍得。

因为大多数人对损失和获得的敏感程度都是不对称的。面对损失的痛苦感，要远远大于获得的愉悦感。

那么，结合两性关系来说，就是“曾经有一份真挚的感情摆在了我的面前，但是我没有珍惜，等到失去了我才后悔莫及”。恰恰大部分人意识不到这种效应的存在，所以就会被这种感觉牵着鼻子走。

上一节我们介绍了损失厌恶效应的核心概念，本节中我们来看看它在分手后的具体表现。

以下是损失厌恶效应在分手后的典型行为表现：

*   在痛苦感的驱使下，第一时间想要去挽回。
*   不会考虑太多的前因后果，更不会去从长计议。
*   在情绪的支配下，不可避免地会出现纠缠式的挽回。
*   做出各种极端的行为，增加对方的反感，将对方越推越远。

当然，也存在一部分人在分手初期会有一种如释重负的感觉，自身状态还不错。但是好景不常的是，分手的痛苦感慢慢地会占据大脑，就像喝醉后醒来的头痛一样难受。同样很有可能去纠缠。

所以，我不建议在分手初期跟对方有太多的纠缠不清。因为这个时候你是被情绪操纵的，很难理性地去做一些分析和判断。同时你的自身状态也不是那么好，那么无疑就会给对方造成负面印象。

**挽回就是要一直去做加分项，而不是一个减分的过程。**

本节课中我们一起学习了“损失厌恶效应”及其在分手初期的表现。核心要点是：分手初期的痛苦和冲动是正常的心理反应，但被这种情绪驱使去纠缠对方，只会让复合变得更难。正确的第一步是认识到自己的情绪来源，并暂停纠缠，为理性的思考和自我调整留出空间。
