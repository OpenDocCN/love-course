# 情感分析课程 01：识别非真性分手的三种表现 💔

在本节课中，我们将学习如何通过女生的特定行为，判断其分手意图是否坚决。核心在于识别那些表面决绝、实则留有挽回余地的信号。

真正的离开往往是悄无声息的。如果一个女生分手时不哭不闹，安静地收拾行李离开，甚至能与你谈笑自若，那通常意味着她已真正放下。

然而，如果一个女生在分手后出现以下三种表现，则说明她可能并非真心想彻底断绝关系。

以下是三种关键的非真性分手表现：

1.  **会在你选择质问时大篇幅地指责你**：她会详细地数落你过去的种种不足，例如“你以前在**感情投入**和**日常关怀**上做得非常差劲”。
2.  **会在你去挽回时情绪激动地一遍遍说绝情的话**：当你尝试挽回时，她会情绪激动地反复说出“我们不可能了”、“别再找我”等绝情言论。
3.  **会各种拉黑删除你，但是还是会留一个联系方式**：她可能会删除或拉黑你的微信、QQ等主要社交账号，但总会保留一个如手机号或某个不常用平台的联系方式。

这些行为背后的心理动机值得深入分析。她斥责你，是希望你能真正**重视关系中出现的问题**；她说绝情的话，是为了刺激你做出更进一步的挽回行动；她偷偷保留一个联系方式，一方面是对你仍抱有期待，另一方面也是想观察你是否会通过这个渠道继续努力，以此验证你的爱意。
