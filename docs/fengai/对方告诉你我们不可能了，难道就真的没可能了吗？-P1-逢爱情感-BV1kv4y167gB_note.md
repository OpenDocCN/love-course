# 对方告诉你我们不可能了，难道就真的没可能了吗？-P1-逢爱情感-BV1kv4y167gB_note

在本节课中，我们将要学习一个核心情感认知：当伴侣在分手后坚决表示“我们不可能了”，这个决定背后的真相是什么，以及我们该如何正确理解和应对这种情况。

---

上一段我们提到了分手时常见的决绝表态，本节中我们来看看这种表态背后的本质。

人的决定其实是随着情绪流动而变化的。今天他说不爱你了，“不可能”是当下的过激情绪。他不爱此刻的你了，不可能和当下的你在一起而已。

他的态度可能会很坚决，甚至说了一些狠心的话，比如“再也不会回头了”。然后你就相信了。

那么我问你们，你们听过他信誓旦旦地说会爱你一辈子、爱到永久的话吗？他做到了吗？为什么现在他说不爱你了、不回头了，你就认为他做到了呢？

**核心概念**：人的情感承诺具有**流动性**，强烈情绪下的表态 ≠ 永恒不变的最终决定。

---

理解了决定的可变性后，我们来看看关系中的吸引力本质。

人就是这样的，不断地做决定，又不断地在打脸。谁都是一样的。其实你吸引他爱你的地方从来都没有消失，只是在一起时，一些琐碎的问题遮蔽了他的眼睛，他看不见了罢了。

所以，你现在该做的事情，就是让他重新地看见。

---

那么，如何让他重新看见你的吸引力呢？以下是具体的行动思路。

在这个过程当中，我觉得你应该要重新地保持距离。近距离你看不见爱我的原因，那么就把距离拉得远一些。

然后你再慢慢地重新提升自己。等到你在他的视线内慢慢变清晰的时候，他自然会重新看到爱你的原因。毕竟你已经成功吸引过他一次。

**行动公式**：**创造距离 → 自我提升 → 重新进入视线 → 唤醒旧有吸引力**

---

对于如何创造距离和联系，可能存在一些实操疑问，以下是针对这些问题的解答。

如果有的人说，对方不肯跟你在一起，保持一段没有交集的距离怎么办呢？我想说，作为曾经的恋人，你们之间的生活圈子、朋友、同事，不可能没有交集的。只不过说你没有制造出交集罢了。

只要你想要联系，那就一定有联系的窗口。而具体怎么把握这个联系的方式，那就要看你们两个人的具体原因了。

**关键点**：共同社交圈是天然的“联系窗口”，关键在于如何有策略地利用。

---

最后，我们探讨一个困扰许多人的根本问题：对方是否还爱着我？

至于那些还在纠结说，对方是否还爱着我的人，我只能说：爱你是真的，你们之间出现的问题也是真的。

如果说你真的爱他，希望重新在一起，那就不要再去纠结这个没有意义的答案。如果说能重新在一起，怎么可能不爱你呢？你说对不对。

**核心逻辑**：纠结“是否还爱”无助于解决问题。**行动指向复合 → 复合成功则爱自然存在**。
