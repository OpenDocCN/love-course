# 明明就很喜欢，却偏要刻意的疏远

在本节课中，我们将要学习一种常见的情感心理模式——回避型依恋。我们将探讨其核心特征、行为表现以及背后的心理动因，帮助你理解“明明喜欢却刻意疏远”这一矛盾行为。

## 概述：什么是回避型依恋？ 🤔

明明很喜欢一个人，却偏要刻意地疏远，这是一种典型的回避行为。这种行为模式往往源于个体从小缺乏安全感，从而形成了回避型依恋。在心理学上，回避型依恋具有一些显著的共同特征。

## 回避型依恋的七个显著特征 📋

以下是回避型依恋模式的七个核心特征，你可以对照观察，看看你的恋人是否具备以下特点。

1.  **刻意的疏远亲密关系**：即使对方非常喜欢自己，也会刻意保持距离。
2.  **逃避沟通或缺乏沟通能力**：常常将正常的沟通演变为争吵。
3.  **表达能力弱**：倾向于通过说反话或做与心意相反的事来表达感情。
4.  **极度缺乏安全感**：认为感情非常脆弱，不相信任何关系能够长久稳定。
5.  **鄙视亲密关系**：将恋爱视为一种“侮辱”，害怕自己被感情所束缚。
6.  **在感情中缺乏自信**：表面看似处于高位，实则内心自卑。对自己要求苛刻，不允许自己犯错或出丑，认为只有“强者”才值得被爱。
7.  **将克制与痛苦等同于幸福**：并非不想被爱，而是认为必须通过逃避这份爱，才能获得安全感。

通过以上特征我们不难发现，回避型依恋者对亲密关系的整体态度并非真正的厌恶，而是充满矛盾。他们渴望接近，却又不敢接近。

## 矛盾的核心：渴望与恐惧的拉锯战 ⚖️

上一节我们介绍了回避型依恋的具体特征，本节中我们来看看其行为背后的核心矛盾。

他们身披厚厚的“外壳”，看似疏离，但这层疏离的背后，隐藏着与常人无异的心理需求——**他们也需要被爱，也需要去爱，甚至比常人更需要他人的支持与温暖**。

两者的根本区别在于面对“喜欢”时的反应。我们可以用一个简单的公式来描述这种差异：

*   **常人模式**：`喜欢` → `接近`
*   **回避型依恋模式**：`喜欢` → `认为喜欢是可耻/无用/会带来伤害的` → `逃避`

对于回避型依恋者而言，问题不在于感受不到爱，而在于他们认为“喜欢”这种感觉本身是可耻的、无用的，或者会招致伤害，因此他们选择用逃避来应对。
