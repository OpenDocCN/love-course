分手后，对方依然频繁与你联系，这常常让人困惑其真实意图。他是否想复合？其行为背后通常隐藏着三种主要心理状态。

## 🤔 三种常见心理动机分析

以下是分手后对方保持联系的三种典型心理动机及其详细解读。

**1. 出于习惯与情感依赖**
对方只是习惯了你的存在，暂时无法适应没有你的生活。这种联系源于情感依赖，而非坚定的复合意愿。如果你在这种情况下选择复合，很可能在对方适应单身生活后再次被抛弃。其心理公式可概括为：
**联系动机 = 习惯 + 短暂的情感戒断困难**

**2. 强烈的占有欲作祟**
对方的占有欲很强。在他自己尚未开始新恋情之前，他也不希望你拥有新的伴侣。这种联系的目的是维持一种单方面的“所有权”宣示，而非出于爱意。核心逻辑可以用以下代码结构表示：
```python
if 自身未有新欢 == True:
    保持联系以阻止前任有新欢
```

**3. 心存爱意但顾虑重重**
对方心里确实还有你，并有复合的意愿，但同时又害怕两人会重蹈覆辙，再次经历同样的矛盾和痛苦。因此，其行为显得犹豫和矛盾。这种状态可以表示为：
**复合意愿 = 残留的爱意 - 对重蹈覆辙的恐惧**
