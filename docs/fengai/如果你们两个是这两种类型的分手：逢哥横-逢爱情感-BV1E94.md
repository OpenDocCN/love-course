# 如果你们两个是这两种类型的分手：逢哥横-逢爱情感-BV1E94

在本节课中，我们将要学习两种最容易导致分手、但也最容易复合的恋爱关系问题类型。理解这些类型有助于你识别自身关系中的核心矛盾，并找到修复的方向。

## 爆发性分手 💥

上一节我们概述了课程内容，本节中我们来看看第一种类型：爆发性分手。

这种分手通常发生在一种不平衡的相处模式中。其中一方长期扮演包容和忍让的角色，而另一方则持续表现出任性或“作”的行为。随着时间的推移，长期包容的一方情感消耗殆尽，最终无法承受而提出分手。

以下是爆发性分手的核心特征：
*   **关系模式**：一方持续包容，另一方持续任性。
*   **情感积累**：不满和委屈随时间默默积累，而非一次性爆发。
*   **分手决策者**：通常是长期包容的一方。
*   **关键公式**：**长期包容 + 持续任性 = 情感耗尽 → 分手**

## 疲惫性分手 😫

了解了因单方面消耗导致的爆发性分手后，我们再来看看第二种因双方互动产生的消耗类型：疲惫性分手。

这种分手源于双方在根本层面的不协调。两个人因为脾气性格不合、三观存在差异、或一方敏感多疑等问题，使得相处过程充满了摩擦与压力。这种持续的负面互动让彼此感到心力交瘁、压抑不堪，最终导致关系难以维系。

以下是疲惫性分手的核心特征：
*   **矛盾根源**：性格冲突、三观不合、敏感多疑等根本性差异。
*   **互动状态**：日常相处充满压力、争吵或冷战，令人感到疲惫。
*   **情感体验**：双方或一方长期感到“心累”和压抑。
*   **关键描述**：关系因持续的精神内耗而瓦解。
