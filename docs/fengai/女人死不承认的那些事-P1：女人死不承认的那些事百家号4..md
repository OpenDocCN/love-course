# 女人死不承认的那些事-P1：女人死不承认的那些事百家号4.

在本节课中，我们将要学习一种在情感关系中可能出现的沟通模式，即一方在缺乏直接证据时，倾向于完全否认某些行为或事实。我们将解析其具体表现和背后的逻辑，帮助你更清晰地理解这类互动场景。

上一节我们介绍了课程主题，本节中我们来看看这种行为模式的具体表现。

以下是该行为模式通常包含的几个核心步骤：

*   **信息缺失即否认**：对于未被你直接察觉或知晓的事情，她的立场是“不知道”。
*   **证据缺失即否定**：对于没有被你亲手抓到现行的事情，她的结论是“没做过”。
*   **证据不足即无视**：对于你缺乏实质性证据的事情，她会将其“当做没发生”。
*   **证据范围即承认边界**：她承认的程度，严格对应你掌握证据的多少。公式可表示为：**承认度 = f(证据强度与数量)**。
*   **无证据质疑引发冲突**：如果你在没有任何证据的情况下开始表示怀疑，她会做出情绪化反应。

当面对无证据的质疑时，她的典型反应如下：

1.  对你发脾气。
2.  指责你缺乏信任。
3.  甚至可能颠倒是非。
4.  将双方（包括她自己的）过错，全部归咎于你。

本节课中我们一起学习了一种特定的否认与应对模式，它描述了在缺乏确凿证据时，一方可能采取的从完全否认到激烈反驳的系列行为。理解这些模式是理性分析情感沟通的第一步。
