# 家庭关系第19讲：离婚后如何减少对孩子的伤害 👨‍👩‍👧‍👦

在本节课中，我们将探讨一个家庭框架发生变化后的重要议题：离婚。我们将重点分析离婚可能对孩子造成的伤害来源，并提供具体的行动要点，以帮助父母在分开后最大限度地保护孩子的情感世界。

上一讲我们讨论了家庭内部的各种难题。本节中，我们来看看当家庭这个框架本身解体时，我们面临的新挑战。

如今离婚已成为一种普遍的社会现象。传统观点常将离婚视为一种失败或问题。人们认为是因为夫妻无法经营好家庭才会离婚。然而，离婚也可以被看作是一种解决问题的办法。正是因为婚姻中存在无法解决的、持续的伤害，人们才选择离婚。离婚本身会带来伤害，但人们选择离婚，往往是因为不离婚所带来的伤害可能更大。

既然离婚是停止伤害的一种方式，我们就需要思考：在什么情况下，这个办法才能真正有效？关于离婚对成人的影响，我们在《亲密关系》课程中已有涉及。本节课，我们将重点讲解如何减轻离婚对孩子的伤害。

## 理解伤害：孩子经历了什么？

要弄清如何减轻伤害，首先要理解离婚对孩子的伤害究竟是什么。

家庭解体时，孩子受到的最直接冲击是“家没了”。几乎所有的孩子都会经历一个阶段：在家庭解体前，他们会非常努力地想让家维持下去。父母离婚后，家庭的残缺也很容易被孩子理解为是自己的残缺。因此，当看到别人有完整家庭而自己没有时，这些孩子常会觉得自己低人一等。

那些总是被父母卷入冲突的孩子，更会责怪自己，觉得是因为自己没做好才导致家庭解体。

如果处理得当，孩子也能慢慢从冲击中走出来。他们会觉得自己还是有家的，只是家庭成员少了一个。如果抚养者能给予足够的温暖和安全，孩子也能拥有安全感和归属感。孩子能否接纳这个特殊的家庭，很大程度上取决于家里大人的态度。如果大人觉得这是一件羞耻的事，孩子也会把它当作一个不可告人的秘密。如果大人自己的接纳程度高，孩子自然也会接纳。就像一个单亲家庭的孩子很自然地说：“我家只有我和妈妈，可是她是世界上最好的妈妈，我很幸福。”

即使孩子接纳了自己有一个特别的家，仍会有一些复杂的情感残留可能造成伤害。以下是三种常见的伤害来源：

**1. 被迫在父母间做出选择**
这个选择不仅是现实抚养权的选择，更是情感上依赖谁的选择。孩子通常对父母都有情感，让他们选择依赖一人而放弃另一人，是非常困难的事。更糟糕的情况是，一些孩子需要与原来的主要依恋对象分开，去跟另一方生活。例如，一个依赖妈妈的孩子，在父母离婚后只能跟爸爸生活。即使爸爸对她不错，她仍可能感到极大的不适应，经历艰难的心理调适期。

**2. 父母矛盾的波及**
孩子的情感会受到其依赖对象的影响。例如，当妈妈对爸爸怀有恨意时，孩子通常也会跟着一起恨爸爸。但另一方面，孩子内心又想去爱、想去依靠自己的父亲，这会使他陷入矛盾的情感。这种矛盾可能导致孩子产生极端的想法，例如想要剔除来自父亲一方的基因，这实质上是想消除内心的情感冲突，让自己变得一致。否则，他不仅无法接纳父亲的感情，甚至无法接纳自己作为“父亲的儿子”的那部分自我。

**3. 关系的纠缠**
在单亲家庭中，抚养者与孩子的关系可能变得异常紧密。抚养者容易把孩子当作情感寄托，孩子亦然。但这种亲近带来的不只有温暖，也可能有伤害。例如，一个由母亲独自抚养长大的孩子，可能对母亲的情绪极度敏感，将母亲的叹气理解为对自己的否定。双方可能陷入一种“相依为命”又彼此消耗的纠缠中。孩子甚至可能无意识地“把生活弄乱”，来为母亲提供“管教自己”的机会，以此满足母亲被需要的心理。当孩子到了该离家的阶段，这种纠缠会造成额外的冲突和伤害。

## 如何减轻伤害？三个行动要点

理解了伤害发生的原因，我们就拥有了减轻伤害的线索。以下是三个核心要点：

**1. 避免将父母矛盾延伸到孩子身上**
如果离婚后父母的矛盾仍在延续，孩子的内心就会变成父母情感的“战场”，孩子也很难有空间去发展自我。这并非要求两人离婚后一定要做朋友，而是要学会把伤害留在过去，把离婚当作一种结束。因为无论过去有多少爱恨，现在你自由了，孩子也应该拥有这种自由。

**2. 处理好养育上的协作与边界**
一对伴侣离婚后，他们不再是夫妻，但作为孩子的父母，仍有很多需要协作的地方。要让一对因协作不好而离婚的夫妻在养育上协作好，并不容易。关键在于划清彼此的边界。如果孩子跟随某一方生活，另一方对其养育做法再不认同，也不应随意发表意见、纠正或干预，除非对方愿意听取。相应的，一方也可以要求在自己陪伴孩子的时间里，让孩子只听自己的。这样，孩子才可能把双方的爱整合在一起，而不会在父母的矛盾中左右为难。

**3. 父母要有孩子以外的生活空间**
一些离婚者会觉得对不起孩子，因此不再寻找新的伴侣，认为只要和孩子过好就行。这种选择可以理解，但他们没意识到，如果孩子变成了父亲或母亲的全部，就意味着孩子被放在了要为父母一生负责的位置上，而孩子也会觉得应该为父母的一生负责。这种沉重的负担很容易让爱变成怨。那些离婚后仍然过得好的家庭，有一个共同特征：父母在孩子之外还有自己的生活空间。他们有工作、朋友、家人的支持。孩子也会慢慢发展自己的朋友圈。

**因此，减轻对孩子伤害的最大秘诀是：离了婚，你仍然要过好自己的生活。** 公式可以概括为：
**健康的单亲家庭 = 清晰的养育边界 + 父母独立的自我空间**

如果孩子既能从父母那里获得情感支持，又不用承担抚养者情感上的全部负担，他们就能更好地发展自己。

## 总结与思考

本节课中，我们一起学习了离婚可能对孩子造成的三类伤害：被迫选择、矛盾波及和关系纠缠。我们也掌握了减轻这些伤害的三个要点：隔离矛盾、协作养育并划清边界、以及父母保持独立的自我空间。

记住，离婚是解决问题的办法之一，而如何执行这个办法，决定了它最终是终止了伤害，还是转移了伤害。核心在于，将夫妻关系的结束，与父母身份的协作区分开来。
