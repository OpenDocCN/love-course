# 038.杨霞：青春期心理课：08.电子游戏给孩子带来的影响_note

在本节课中，我们将探讨电子游戏与网络小说对青春期孩子专注力、大脑发育及情绪行为的具体影响，并为家长提供科学的认知与应对视角。

## 概述：电子游戏与专注力问题

许多家长发现，越来越多的孩子沉迷于电子游戏和网络小说，甚至很小的孩子就开始接触iPad等设备。家长有时认为适度玩耍无害，且担心孩子不参与会失去朋友。然而，心理学研究揭示了其潜在危害。

## 电子游戏对大脑的单一通道刺激

上一节我们提到了家长的普遍担忧，本节中我们来看看电子游戏如何影响孩子的大脑运作。

心理学家研究发现，过去看电视时，孩子只有眼球在动，仅视觉单一通道在工作，其他感觉器官、肌肉和神经系统则处于类似“瘫痪”的闲置状态。这好比给人打上石膏固定在一个位置，只让他观看飞速变化的画面。长期如此，会导致“用进废退”——过度使用的通道疲劳，未使用的通道退化。

电子游戏的影响比电视更严重。游戏内容紧张，且常以简单、强烈的刺激为主，缺乏电视可能包含的情节与知识。孩子的大脑在游戏中处于**单一通道**反复接受紧张刺激的状态，并产生强烈兴奋。

## 对视觉与学习专注力的具体损害

理解了游戏对大脑的刺激模式后，我们进一步分析它如何直接损害孩子的学习能力。

孩子在游戏时看似全神贯注，但只是手指单一动作的震颤和眼睛死盯目标。当回到现实课堂或作业时，问题便会出现：

*   **视觉震动现象**：电子屏幕存在肉眼不可见的频闪。长时间注视后，眼睛再看书本或作业，容易出现视觉不稳定。
*   **专注力下降**：表现为无法专注阅读、书写和计算。
*   **错误率增加**：具体行为包括：
    *   丢字、落字、串行。
    *   书写时多一笔、少一笔。
    *   将数学符号看错，如加号看成减号。
    *   跳字阅读，例如将“井冈山”看成“井山”。

此外，如今高亮度的手机屏幕已导致儿科视力门诊量急剧上升。电子产品严重影响孩子的注意力、视力及大脑健康。

## 极端案例与情绪行为影响

除了常见的专注力问题，过度游戏还可能引发更严峻的身心健康危机。

一个真实案例是，一名七岁儿童因长期玩游戏患上脑胶质瘤。手术后，家长仍无法阻止孩子继续游戏。这凸显了严峻挑战：孩子是不成熟的个体，若成年人遵从孩子的不理智选择，后果无法承担。

电子游戏还会严重破坏孩子的情绪：

*   **虚拟对抗 vs 现实对抗**：足球、篮球等现实对抗运动允许孩子在奔跑、竞争中**宣泄能量与荷尔蒙**。游戏中的对抗虽引发激动和肾上腺素分泌，但能量只能通过手指动作释放，全身肌肉能量被压抑。
*   **情绪积压与行为变化**：能量未获释放导致许多孩子在游戏后变得**更加愤怒、烦躁**，易与家长对抗、发脾气、顶嘴、不讲理甚至说脏话（模仿游戏中的骂战）。情绪波动会直接破坏专注力，使其无法做好一件事。

## 关键期建议与健康社交引导

认识到危害后，我们来看看何时影响最大以及如何建立更健康的生活与社交方式。

心理学家研究发现，孩子接触电子游戏的年龄越小，对大脑的伤害越严重。相比18岁或25岁以后的成年人，**13岁、9岁甚至4-5岁之前**的早期接触影响尤为深远。

以下是促进孩子健康发展的建议：

*   **鼓励现实活动**：尽量让孩子在自然环境中进行踢球、打球、爬山等活动。
*   **采用传统阅读**：在自然光线下阅读纸质书籍。
*   **限制电子产品**：减少对电子产品的依赖，以保护视觉、注意力及专注力的正常发育。家长在此问题上不应退让。

关于社交的担忧，家长需明白：如果孩子的友谊仅建立在游戏上，那么一旦在游戏中受挫（如玩得不好、拖累队友或因故暂时不能玩），友谊可能轻易破裂。我们应希望孩子拥有**多元化的友情通道**，来自不同领域和活动的朋友交往，更有利于其学习、情绪和专注力的全面发展。

## 总结

本节课中，我们一起学习了电子游戏如何通过**单一通道刺激**损害孩子大脑发育，导致**视觉震动、专注力下降**及**学习错误率增加**。我们还探讨了其引发的**情绪积压、行为改变**等深层危害，并通过案例强调了早期接触的严重性。最后，我们明确了家长应坚持引导孩子参与现实活动与多元化社交，为孩子的身心健康奠定坚实基础。
