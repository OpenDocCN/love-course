# 课程078：人际交往2期 - 掌握沟通核心技巧 🗣️

在本节课中，我们将学习《说话就是生产力》一书的核心内容。这本书提供了一套系统的沟通训练方法，旨在帮助我们提升表达的准确性和影响力。课程内容将分为三个主要模块：四种核心说话技巧、一套八步沟通阶梯训练法以及两种实用的练习工具。

## 第一部分：四招核心说话技巧 💡

上一节我们概述了课程目标，本节中我们来看看提升沟通能力的四种立即可用的技巧。

### 第一招：垫子

垫子的作用是缓和紧张情绪并为自己赢得思考时间。它通过肯定提问者或问题本身来实现。

以下是运用垫子的三种主要方式：
1.  **赞扬提问的人**：例如，“您能发现这个问题，真是太细心了。”
2.  **赞扬提出的问题**：例如，“您这个问题非常专业，一下子问到关键了。”
3.  **展示自己的处境**：例如，“您这个问题很有挑战性，让我仔细想一想。”

### 第二招：迎合

迎合的目的是营造良好的谈话氛围，拉近双方距离，其核心是运用同理心。

以下是迎合的三种技巧：
1.  **当对方说出结论时，补充事实**：例如，对方说“今天真冷”，你可以回应“是的，气温都到零下十度了。”
2.  **当对方说出事实时，补充结论**：例如，对方说“这网五分钟断了三次”，你可以回应“没错，现在的网络质量确实不太稳定。”
3.  **当对方既有事实又有结论时，给予类似体验**：例如，对方说“我发烧到39度很难受”，你可以回应“我上次发烧也头晕得厉害，特别理解您的感受。”

### 第三招：制约

制约的技巧用于在沟通中掌握主动权，避免被对方牵着鼻子走。

以下是制约的三种方式：
1.  **抢先说出对方想说的话**：例如，客户问价格前，售货员先说：“这款手机价格有点高……”
2.  **提供不同的选择**：例如，“您是更看重价格，还是更看重性能呢？”
3.  **表现得比对方更加期待**：例如，客户要求降价时，你可以说：“其实我也希望能给您更优惠的价格，这样我也能多完成点业绩。”

### 第四招：主导

主导是引导谈话走向，让他人跟随你思路的技巧。

以下是掌握主导权的三种方法：
1.  **凡事分三点来说**：例如，“要解决这个问题，我们需要从三个方面考虑：第一……第二……第三……”
2.  **引用权威资料**：例如，“正如马云曾经说过的……”
3.  **超越话题，上升到更宏观的层面**：例如，从具体事件讨论引申到行业规律或普遍价值观。

## 第二部分：八步沟通阶梯训练法 🪜

掌握了四种基础技巧后，我们需要通过系统训练来内化能力。下面介绍一套从易到难的八步训练法。

### 第一步：讲述

讲述要求客观、有条理地陈述已知事实，不加入个人观点。

**讲述的核心公式是：`讲述 = 客观事实 + 清晰顺序 + 具体细节`**

例如，要求你“讲述一部电影的内容”，你只需说明时间、地点、人物和事件经过，而不应加入“我觉得这部电影很好”之类的评价。

### 第二步：叙述

叙述在讲述的基础上，加入了个人观点和看法，目的是说服他人。

**叙述的核心公式是：`叙述 = 客观事实 + 个人观点 + (经筛选的)支持性细节`**

例如，你想表达“某同事很负责”，你会选择他加班完成项目、多次检查细节等事实来证明你的观点，而忽略他偶尔迟到等无关事实。

### 第三步：表达

表达通过加入表演元素来提升影响力和说服力，让人印象深刻。

**表达的核心公式是：`表达 = 讲述 + 叙述 + 表演形式(肢体语言、语调、道具等)`**

例如，做自我介绍时，除了说出名字和背景（讲述+叙述），还可以配合手势、写在白板上，或展示作品集，使介绍更生动。

### 第四步：讨论

讨论是融入团队的重要方式，其关键在于通过提问来引导方向。

讨论中提问应聚焦于两点：
1.  **对方的逻辑**：“您是如何得出这个结论的？”
2.  **事实与细节**：“您这个判断是基于多少样本得出的？”“还有哪些相关细节？”

### 第五步：汇报

有效的工作汇报需要结构完整，为上级决策提供依据。

**汇报的核心三要素是：`事实 + 观点 + 建议`**

例如，不应只说“客户对方案不满意，合作可能失败”（事实+观点），而应加上“我建议我们可以准备A、B两套修订方案，再次与客户沟通”。

### 第六步：陈述

陈述追求出口成章、条理清晰的效果，需要通过书面练习来提升。

陈述有两个基本要求：
1.  **次序**：结构清楚，有先后顺序。
2.  **层次**：内容由浅入深，层层递进。

### 第七步：解释

解释旨在消除误会，需要确保信息被准确理解。

解释必须满足四个要求：
1.  听懂对方的真实意图。
2.  让对方知道你听懂了（例如，重复对方的问题）。
3.  用对方能理解的方式说明情况。
4.  确认对方确实听懂了你的解释。

### 第八步：回应

回应用于立场不同或面对挑衅时，需要进行有力反驳。

回应的步骤是：
1.  **剖析对方**：找出对方陈述中的事实、观点和逻辑。
2.  **找出漏洞**：识别对方故意忽略的事实或逻辑缺陷（如以偏概全）。
3.  **进行反击**：摆出被忽略的事实、对方未看到的事实或提供相反的事实，来证明其观点不成立。

## 第三部分：两种实战练习工具 🛠️

理论需要结合实践。以下是两个可以日常练习、提升说话能力的工具。

### 工具一：特殊扑克牌

制作一副54张的扑克牌，每张写上一个词汇（18个名词、18个动词、18个形容词，其中形容词包含9褒9贬）。

练习从易到难进行：
1.  **基础造句**：抽一张牌，用上面的词造句。
2.  **简单问答**：抽一张牌，造一个简单疑问句（如“你工作踏实吗？”），由同伴回答。
3.  **复杂对话**：抽一张牌，造一个复杂疑问句。同伴抽另一张牌，用牌上的词造句来回答你的问题。如此循环对话，可逐渐增加每次抽牌的数量（两张、三张）以提升难度。

### 工具二：看图说话

选择一幅图画，在规定时间内（如两分钟），针对图画提出尽可能多的复杂疑问句（而非简单的“是什么”）。例如，“图中人物的表情暗示了他怎样的情绪？”“事件发生的前因后果可能是什么？”。

这个练习能有效提升观察力、思维发散力和提问能力。

**最后的重要建议**：练习以上技巧和工具时，请尽量采用面对面交流的方式。只有面对面，你才能观察到对方的眼神、表情和情绪反馈，从而进行最有效的调整和提升。
