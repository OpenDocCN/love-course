# 课程007：识人物：赶在他还没有脱颖而出时 👁️‍🗨️

在本节课中，我们将要学习“识人物”这一进阶概念。我们将探讨如何在他人尚未取得显赫成就时，就识别出他们的潜力，并理解为何这比结识已成名者更有价值。

上一节我们介绍了“识人”的概念，本节中我们来看看它的进阶版——“识人物”。

## 什么是“识人物”？

“识人物”是一个动词，它并非字典中的标准词汇，而是“识人”的一个特例。它特指从人群中辨别并判断出那些未来的杰出之辈。

虽然我们常思考“别人为什么要认识你”，但许多人仍渴望结识杰出人士，哪怕只是点头之交。实际上，这条路通常行不通，且最终可能令人失望。道理很清晰：任何英雄豪杰也是凡人，一天只有24小时，分配给谁都需要理由。锦上添花的行为，通常不会让人太过在意。

然而，世间存在另一条更优、更具操作性的路径。那就是在“识人”的基础上更进一步，努力做到“识人物”：赶在别人尚未脱颖而出之前，及早发现对方的潜力，并给予关注与支持。此时，你的行为算是雪中送炭，远比锦上添花珍贵。

雪中送炭的好处是不需要排队，但坏处是变数太多，未来难以预见。从个人成长角度看，结识一个从长线看个人发展比你迅猛的人，也是一个绝佳的学习机会。与未来的国手下棋，未必能让自己成为九段高手，但一定能将原有水准提升一大截。

这些年来，我也结识了一批这样的人物。换言之，我很早就看好他们，相信他们必有脱颖而出的一天。算下来，我的预测准确率大约一半一半，称不上“慧眼识珠”。但即便只有一半人成功，也足够我学习和效仿了。

如果你不介意准确率问题，以下是我的一些粗浅心得。

## 识别“潜力人物”的四点心得

以下是识别潜在杰出人物的四个观察维度。

**第一点：他不应是张白板，必须曾崭露头角。**

这一点并非铁律，而是由我个人能力所限。我并无特殊能力，能在一群平平无奇的人中看出谁头顶有灵光闪耀。因此，我只能观察那些取得过一定成绩的人。成绩领域不限，可以是个人爱好，也可以是工作领域。关键是需要有闪光点，它无需耀眼，只需能将他与周围人区分开来。

区分的标准是什么？在同样的背景和外部条件下，他最终做出的成果要大大超过周围的人一截。天才或杰出人物不大可能一夜开悟，他们必定在很早时就展现过能力，只是那时光亮太微弱，易被忽视。

作为个人，判断标准其实很简单：无论对方成绩多小，假设自己在同样条件下，最终结果乘以10，能否达到对方的水准？**公式：`(我的预估成果 × 10) < 他的实际成果`**。如果不能，那么这个人很可能就是杰出之人。

**第二点：他应该比较执拗，拥有很强的专注力。**

从性格分析，这或许像在分析成功人士，有“事后诸葛亮”之嫌。但性格的确重要，且在一类人身上存在共性。我的出发点并非归纳总结，而是反向防御：我不知道他们最终成功的原因，因此只能找出那些易导致失败的性格特征，并反其道而行之，为我的选择加上一道保险。

一个能成长起来的人，性格不大可能很柔弱，否则易被他人的意见改变心意，导致前进时左右摇摆，错失机会。如果他心中有主见，不大听得进别人意见，在普通人看来便是“顽固”。同时，这种人常将主要精力用于思考，无暇向旁人解释想法，这就很容易从“顽固”上升为“执拗”。

但你会发现，这样的人耐得住寂寞，因为他们能长时间专注在一件事上，没有太多心思考虑情绪和感受。在我看来，专注力比聪明更重要，因为面对事物时，**专注力才有穿透力**。

**第三点：他对你讲过的道理，才是道理。**

这条标准可视为一根探针，用于检测对方的成色。每个人都懂一些道理，也听过一些道理，不同的道理带来不同层次的理解。

但潜在的杰出者不一样。他们讲的道理不会让你觉得脑子里多出了东西（即不会产生“丰富”或“富裕”的感觉），而是让你感觉到“清澈”。

自己懂的道理加上别人告诉你的道理，最多是告诉你一件事的几个面，让你看到更多。于是你觉得视角变得丰富，知识变得富裕，但你也知道，或许还有别的面，还有不确定的地方。

“清澈”则不同。清澈是几句话就把一件事情连根拔起，放在你面前，让你可以从任意角度观察，没有任何隐藏部分。在他们没对你讲道理前，你觉得自己知道得再多也差了点什么。但在他们讲过一番道理后，你知道，长久以来等待的答案终于出现了。事物就是那个样子，之前听到的其他道理都不再是道理。

无论是通过直觉还是思考所得，这类人就是有这样的本事。他们比一般人看得更深、更远、也更准。因此，等到他们下手时，手段和结果就会远超常人。你可以对比任何行业的资深专家，当他们深入浅出地讨论问题时，也有那份透彻与从容。困难的不是成为专家后如此，而是在很早以前就有这样的眼光。

**第四点：性格不重要。**

我知道许多人在选人时，喜欢讨论性格脾气，并套用一套“德艺双馨”的模板。但在我看来，性格并不重要。天才并不受世人标准束缚，人也不是为了让你感觉相处舒服才来到世间的。

所谓“杰出人士”或“脱颖而出”，意味着已经计算了性格因素可能带来的负面影响，但这依然不足以影响个人才华绽放的异彩。尤其在早期，这种人并不为周围人所认可，也很少能得到理解与支持。所以，性格脾气古怪一点很正常。

当然，我并非说一定要是个怪人。我的意思是，不要用性格作为重要的判定依据。对个人而言，标准降低到“彼此不至于相互冒犯和伤害”就足够了。也不要在他们偶然展现出极为爆裂的一面时太过失望。总之，对于潜在的天才，我们这些凡人要包容、要理解，不要用凡人的标准去衡量。

## 关键在于你自己

最后，我认为最关键的环节在于你自己。

如果你是喜欢热闹的人，那么你多半能找到擅长制造热闹的人。如果你是喜欢强力的人，那么你多半能找到脾气和行为都很强势的人。对我而言，我喜欢有创造力、有建树的人。

所以，从“崭露头角”开始，每个人的选择都不一样。也许你喜欢的是能造多大势、设立多少难题的对手。而我喜欢的是如何在有限条件下，用不一样的方法解决问题，追求简洁和优雅的人。

我认为，最好每个人都想清楚自己想要发现的是一个怎样的人。这样才可能让自己始终保持足够的耐心和宽容，与对方共同成长，等到他最终绽放的那一天。也只有这样，到了那天，你可以坦然地说：我的确能识人，识人中龙凤。

## 总结

本节课中，我们一起学习了“识人物”的概念与方法。我们探讨了如何通过观察对方是否曾“崭露头角”、是否具备“执拗与专注力”、其道理是否带来“清澈”之感，以及为何不应过分看重“性格”，来识别潜在的杰出人物。最重要的是，这一切的起点在于明确你自身欣赏与追求的特质。掌握“识人物”的能力，不仅能让你在人际交往中占据先机，更是个人成长道路上的一笔宝贵投资。
