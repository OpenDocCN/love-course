# 038.杨霞：青春期心理课：06.青春期情绪波动_note

在本节课中，我们将要学习青春期孩子的情绪波动特点，理解情绪如何影响专注力，并探讨帮助孩子平稳度过这一阶段的方法。

## 概述：情绪波动与专注力的关系

青春期的情绪波动会影响孩子的专注力。这个答案是肯定的。

青春期的孩子比童年期的孩子更容易情绪波动。因为此时情绪发展已进入暴风骤雨期。青春期的孩子情绪很容易一会高涨一会低落。情绪强度来得非常猛烈。

情绪波动时势必会影响专注力。情绪会像一匹烈马，把注意力拉得不知去向。因此在此阶段，我们需要注意协调情绪与思维之间的关系，或协调情绪与控制能力之间的关系。这能帮助孩子以更好的专注力完成学业。

## 影响情绪的因素

以下是影响孩子情绪的几个主要因素。

### 1. 生理变化 🌱

到了青春期，孩子生理上进一步成熟。第二性征开始发育。此时激素波动比童年期更加跌宕起伏。激素水平会大大提高。因此孩子更容易多愁善感、敏感、情感丰富。他看场电影产生的情感和小学生完全不同。遇到老师同样的批评，他产生的反应、联想和情绪反应也和小学生不同。他会产生很强烈的反应。

我们要注意青春期的孩子，在中学阶段情绪很容易波动，并且丰富多彩。因此要特别注意此阶段的特点。

### 2. 情感需求的变化 🤝

此阶段的情感不像小学生那样浮浅。他需要你与他有共同爱好，有共同语言，能解决他的一些问题，成为他的同盟。有时还需要你能理解他的情感，与他产生共情和通情。

作为老师或家长，我们首先要学会接受孩子的情绪。例如，一个中学生向妈妈抱怨新换的班主任不好。妈妈劝解孩子，说老师有道理、老师好。结果孩子很生气，说“既然你说老师好，那你去上学好了”，并拒绝上课。

在咨询中，我耐心询问孩子对新班主任的不满，像朋友一样听他诉说并出主意。首先是接受他的情绪，理解他的情绪，并建议他表达情绪。结果孩子反而说：“我们班主任也是第一次当班主任，不容易，他是临危受命。” 我继续站在他的角度说话。最终孩子自己说出了班主任的许多优点，并学会了忍受缺点。

在这个过程中，我没有正面要求孩子接受，而是站在理解、同情他的角度，甚至和他一起说班主任的不好。结果他反而理解了班主任。这里首先是情绪的作用。孩子在思维上完全明白老师是新手、没有经验、不是故意的。但孩子回家表达的是情绪。

家长此时一定要分清楚孩子表达的是情绪还是思维。其实孩子在思维上已经理解、原谅、接受了老师。我们需要在家里先接受孩子的情绪。人的情绪有一个特点：你对抗它时，它会越来越强烈，就像海浪，你挡住它，浪就会很高。但如果你像海绵一样接受它，浪就平息了。

对中学生的情绪，你要像海绵一样。首先学会接受，然后去理解，跟孩子站在同一战线。你会发现孩子就回到了理性的思维水平上与你对话。当孩子回到理性思维时，沟通就变得容易了。

### 3. 家庭环境的影响 🏠

我们不要让情绪破坏和孩子之间的专注力。我们的专注是希望孩子把注意力放在学习上，不希望他受到干扰。但在青春期，情绪会不断干扰。

例如，一个同学来咨询，他发现爸爸妈妈吵架，有时甚至闹离婚。这孩子就特别紧张。尤其是独生子女，没有兄弟姐妹，缺乏安全感。他特别怕爸爸妈妈离婚离开他，希望爸爸妈妈都在身边。此时爸爸妈妈吵架是孩子最焦虑、最紧张的时候。这种情绪肯定会影响到他的专注力。

我们都希望孩子在学习阶段把专注力放在学习上。但家长自己可能难免不能控制情绪，发生争执、意见不一致、因家庭琐事吵架。这就会影响孩子的情绪。家长的情绪影响孩子的情绪，导致孩子学习程度下降。他分心了，在担心爸爸妈妈。这也是情绪对孩子的影响。

### 4. 人际关系与同伴认可 👥

还有一些情绪可能包括早恋和人际关系。到了中学以后，孩子特别关注同伴的认可。在12岁之前，孩子最关注的是家长和老师的认可。到了中学阶段，他特别关注同伴和老师的认可。

一个人想要有良好情绪来保证稳定的学习状态，就需要具备良好的人际关系。人际关系需要愿意跟别人分享，愿意和别人友好相处，能够尊重别人，能把别人放在重要的位置上。如果脑子里只有自己，没有别人，人际关系就不会好。人际关系如果不好，情绪肯定也不好。情绪不好又影响到专注力。

## 总结与核心要点

本节课我们一起学习了哪些因素会影响孩子的情绪，而情绪又进一步影响专注力。专注力不好会再影响到学习效果。

请家长注意，孩子在青春期的情绪与专注力有密切关系。在青春期，我们应帮助孩子顺利度过这个情绪惊涛骇浪、暴风骤雨的时期，以保证孩子有更好的学习专注力和学习效率。

**核心公式**：`情绪波动 → 专注力下降 → 学习效果受影响`

**关键方法**：接受情绪 → 理解共情 → 引导理性思维
