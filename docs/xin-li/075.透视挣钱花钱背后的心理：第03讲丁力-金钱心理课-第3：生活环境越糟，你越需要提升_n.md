# 075.透视挣钱花钱背后的心理：第03讲丁力-金钱心理课-第3：生活环境越糟，你越需要提升_n

在本节课中，我们将要学习阻碍我们变得富有的核心心理因素，特别是**环境**如何深刻影响我们的金钱观和行为模式。我们将探讨如何识别并摆脱环境带来的限制性思维，从而为财富增长创造心理基础。

上一节我们介绍了内心观念对赚钱的影响，本节中我们来看看，究竟是哪些更深层的因素在持续阻碍我们变成有钱人。

如果你不知道自己为什么是穷人，就永远无法知道如何变成富人。你可能会认为富人天生幸运或依靠家庭，但事实上，阻碍我们变得富有的原因主要与三个核心有关。

以下是三个核心阻碍因素：

1.  **外界环境**：你出生在富裕家庭还是贫穷家庭，这有一定关系。
2.  **教育和知识**：很多人没有学过如何挣钱，大部分教育只教你通过简单交换（付出劳动换取金钱）来获得收入。他们不了解金钱并非简单交换，在使用过程中还有许多技能需要学习，例如提升财商、了解投资方式、规划金钱。一旦建立了这些知识体系，你就会明白，没有钱与“没有一个富爸爸”没有必然关系，因为后天可以改变。
3.  **自我**：既然讲金钱心理学，必然是通过内在部分去看外在变化的部分。你能从穷人变成有钱人，是因为你对内在有了足够的信任，在活动中积累了足够的知识，并且理解了外界环境与你的关系。

讲完这三点后，我们来看看今天课程最核心的部分——**自我**。但我们会把“自我”放到最后讲，因为这是一个循序渐进的过程。我们先讲第一个部分：**与环境的关系**。

说到环境，想想我们小时候的生活环境，与90后特别不一样。例如，90后在求职时，可能一言不合就辞职，而老一辈人则特别珍惜工作机会。这种差异源于生活环境的不同。

我们小时候，环境相对比较匮乏，缺乏安全感。但现在的年轻人相对富足，有十足的安全感，他们认为“只要有我在，我就能赚到钱”。因此，环境的影响确实很大。

在上一节课我们讲到，老一辈的思想和经验来自于他们当下的环境、政策及生存条件。到今天，环境已经大不相同。例如，关于房子，80后可能需要辛苦奋斗买房买车，但00后的孩子可能完全不焦虑，因为祖辈和父辈可能已留有资产。他们为之奋斗的东西，在下一代看来可能是理所应当的。

每个时代都有自己鲜明的部分，而我们的成长环境决定了我们对外界的很多互动模式都来自于上一代，而不是我们这一代。上一代对你的影响，有时你根本不自知，因为你完全生活在一个集体潜意识里。

当我们活在一种“匮乏”的集体潜意识里时，我们永远都在节省，永远相信“物比你贵”。宁可为了节省打车钱而去挤公交地铁，即使这可能让你中暑生病。这种思维方式让我们处于糟糕的状态，觉得所有东西都比自己贵。

但如果你看清现在的环境，你会发现**你比任何东西都贵**。就像90后认为工作不开心就可以离开，因为他们觉得自己的感受很重要。外界环境变了，我们的价值观就变了，对事情的判断标准也不一样了。

以前我们可能觉得领导说了算，现在则可能认为“领导说的可能对，但我的想法也有道理，如果你不尊重，我就不跟你玩了”。外界环境使得现在的年轻人不太在乎具体挣了多少钱，而在意能否实现多少价值。这种价值感，已不再单纯用金钱衡量，而是由“我能创造什么”以及“带给我的感受”来衡量。

有些时候，当我们看到环境正在发生变化时，请一定要提醒自己：你已不是生活在父母那个早年的年代了。请醒一醒，不要再沉睡在过去那种“你什么都没有，什么都比你好、都比你值钱”的感觉里。你现在一定要看清楚：**你比任何东西都值钱**。所以，投资自己来提升和改善环境，才是最重要的。

刚才我们说到环境，环境的影响除了来自父母的节俭观念和匮乏感，还有一种影响是大的经济环境带来的焦虑感。

例如，现在有些宝宝特别爱叫、大喊大叫。观察他们的父母，会发现很多是80后。80后生活在一个“一步赶不上，步步赶不上”的年代，政策始终在变，包括他们的父母（50后、60后）也对社会的剧烈变化体验深刻，且缺乏安全感。这种不安全感代代相传。

这就导致很多父母在与孩子交流时，总觉得孩子“听不见”，所以他们用吼叫的方式，认为这样才能让孩子接收到信息。如果你看到并满足孩子的需要，他会很温柔地跟你说话。这种互动模式，也反映了环境焦虑的传递。

此外，如果孩子在尝试分享或帮助时，总是得到“不用不用”、“不要乱花钱”这样的拒绝反馈，就会造就一种“不愿意接受世界馈赠和爱”的心理，内心永远是焦虑、慌张、不淡定的。如果一个人不淡定，周围环境给ta的是一种“你追我赶”的不淡定感觉，ta就很难变成有钱人。

为什么父母内在“不敢要”？是因为他们觉得自己“还不起”。很多父母怕欠人情，觉得如果得到别人的恩惠却不偿还，就会心里不安。这就是匮乏思想：你为什么觉得自己还不起？为什么觉得不配拥有更好的东西？当你一旦有了**配得感**，你就会发现，别人给你是因为爱你。你把它理解为爱，而不是金钱或其他东西时，你会觉得接受得理所当然，一句简单的“谢谢”就足以表达感谢。

所以，环境是什么？除了我们讲到的外在经济环境，还有一个就是**人文环境**。这个人文环境带给你的部分也很有关系：你每天的言行，有没有让自己相信，你是可以存在于一个丰盛的世界里，而不是匮乏的世界里？

以下是两种环境的对比：

*   **丰盛世界**：让你觉得富足、和平、美好、愉悦。
*   **匮乏世界**：充满焦虑、抑郁、纠结和各种各样的担心。

如果我们一旦能区分开，意识到“原来我可以生活在富足的状态里”，我们就给自己创造了一个美好的环境，让我们相信自己与环境的互动可以得到更多。而如果你把自己放在一个匮乏的环境里，你永远相信的是“不够好”、“不行”、“不配得”。你的环境，因为你的选择，变成了一个不断要从你身上透支的过程。匮乏环境就像吸血鬼，会不停地消耗你。而在富足的环境里，它会不断地滋养你。

本节课中我们一起学习了环境对我们金钱心理和行为的巨大影响。我们了解到，无论是家庭传承的匮乏感，还是时代经济环境带来的焦虑，都会内化为我们的限制性信念。关键在于，我们要清醒地认识到自己已身处新的时代，**你本身比任何外物都更有价值**。通过投资自己、提升配得感，并主动选择或创造一个丰盛的人文环境，我们可以摆脱环境吸血鬼的消耗，转向一个能持续滋养我们、助力财富增长的心理状态。下一讲，我们将继续深入另一个核心因素。
