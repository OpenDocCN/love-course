# 课程118：超级识人术 🧐 02、向算命先生一样精准洞悉对方性格

在本节课中，我们将要学习如何通过观察一个人的眉毛特征和细微动作，来洞悉其内在性格与即时情绪。课程将结合传统相学智慧与现代心理学，为你提供一套实用的“观眉识人”方法。

## 一、 眉型与性格：十种特征解析

知人知面更知心，360度看透一个人是社交不吃亏的秘密。中国相术是中华神秘文化中很神奇的存在。汉族相术起源可以追溯到三皇五帝时期。《大戴礼记》中记载：“视其所以，观其所由，察其所安。”随后从萌芽到不断发展，到了今天基本分为随机型和特征型两种。本节要讲的是后一种，即如何根据特征来洞悉他人性格。

有一句话叫“观眉毛识破人”。不同眉毛隐藏着不同性格。民间有种说法：一个人有钱没钱可以看他的鼻头，会不会存钱可以看他的鼻翼。不过想看一个人会不会用钱赚钱，或是和朋友共享利益和大家一起赚钱，就必须看他的眉毛。

眉毛是眼睛的华盖，面部的仪表。可以通过眉毛来推论人的潜能，看他一生的运势、人际关系、兄弟多寡、朋友好坏，甚至夫妻感情。心理学上也认为，眉毛的形状能够反映出一个人的性格。

不同的眉毛可以看出人的不同性格。下面结合中国文化相学典籍的精华智慧，为你细致讲解10种眉型透露出来的人物性格。

以下是十种常见眉型及其对应的性格特征：

1.  **聪明眉**
    *   **特征**：凡是细软、平直、宽长、清秀之眉毛。
    *   **性格**：此人必定聪明伶俐。聪明伶俐的人一般是情商比较高，视野较为开阔。和这样的人交往，你可以不用太注重小节，直抒胸臆即可。跟他的相处氛围较为融洽。

2.  **逆生眉**
    *   **特征**：首先需了解何为顺生眉。眉毛是多根生长在一起的。观察每一根眉毛，从眉头到眉尾向着同一个方向生长，就叫做顺生眉。一根根往上或往下生长都没关系。但如果眉头的部分向上生长，眉尾却有几根眉毛向眉头方向生长，这就叫做逆生眉。
    *   **性格**：两者相比，向上的顺生眉的人性格较为柔顺，处事比较圆滑，在待人接物方面能够考虑到对方感受，人际关系较好。逆生眉的人则较为反叛、叛逆，很容易因为不迎合与妥协而破罐子破摔。和逆生眉的人交往需要格外注意。

3.  **双连眉**
    *   **特征**：两眉头的部分延伸并连在一起。
    *   **性格**：这类人的性格比较坦坦荡荡，不善于遮掩，一是一二是二，可以说是很有机会能成大器。而且朋友比较多，很容易会得到贵人帮助。

4.  **凶恶眉**
    *   **特征**：如果一个人他的眉毛较为粗硬、浓密、逆生、散乱、短粗、卷缩，存在以上两种特征。
    *   **性格**：性格较为反叛，不拘传统教条，容易因为激情犯罪，因为一时的义气跟自我，做出不符合道德范畴的事情。

5.  **压眼眉**
    *   **特征**：凡是眉头小但是清秀，眉毛疏散不浓者。
    *   **性格**：个人意志不会太坚强，性格里含有悲观色彩，个性较为柔弱。不太重视人际交往，与六亲的关系也较为一般。

6.  **一字眉**
    *   **特征**：眉毛呈一字形。
    *   **性格**：一字眉较宽的人具有胆识。一字眉较窄的人比较固执跟缺乏耐力，难成大事。一字眉的人充满正能量，见不得别人受委屈，喜欢帮扶弱小。缺点是个性太过刚烈，不够柔软，一般不能听从别人的意见和建议，我行我素。有时候会由于太过直接而得罪人，或者跟人发生矛盾。

7.  **八字眉**
    *   **特征**：眉毛粗细一般，但眉尾延伸比较夸张，宛如八字。
    *   **性格**：长这种眉毛的人一般是刚正不阿，活泼有朝气。但是思虑浅，粗枝大叶，善于人情世故，却一生浮沉。如果其眉毛根部轮廓不够清楚，则在三十岁出头时工作会不顺利。此外，古书中曾提到，有八字眉或眉毛下垂的人，多有贪淫好色的倾向。

8.  **高相眉**
    *   **特征**：眉毛生得很高，离开眼睛的距离跟一般的常人比较远。
    *   **性格**：这种人前程很远大，非常有抱负。但也因为常常伴有好高骛远、不能脚踏实地行为特征。

9.  **柳叶眉**
    *   **特征**：眉形如柳叶状。
    *   **性格**：这种人的性格温柔，而且有智慧头脑。但是往往疏于沟通亲朋好友，婚姻感情波折多，得子较晚。不过亦得贵人相助，最终会事业有成。

## 二、 眉毛微动作与心理：解读无声信号

上一节我们介绍了如何通过静态眉型判断性格，本节中我们来看看动态的眉毛动作与心理的关系。根据细微的动作和表情可以帮助你读懂对方的所思所想。

眉毛的动作透露出诸多心理信号。美国社会心理学家得出一个结论：人们很难通过面部神情隐藏自己的性格，眉毛是最能表露一个人心理的。

以下是几种常见眉毛动作及其心理含义：

*   **眉毛向下靠近眼睛**：表示他对周围的人更热情，更愿意与人接近。
*   **眉毛上挑**：表示这个人需要尊重，需要更多时间适应现在的场合。如果你遇到的人将眉毛向上挑，这时你最好不要靠他太近，可以先与他握手，让对方主动接近你，以免让他感觉不舒服。

知道了这个联系，下次和人打交道时，就别忘了注意他的眉毛和眼睛。眉语之间的信息能够透露出人们解决问题的方法、关注细节的持久度以及是否能够做到实话实说等。

**轻抬眉毛**是一个典型且重要的微动作。这个动作是在距离稍远处向人们打招呼的姿势，从远古时代就被广泛使用。其含义不仅通行于全世界，连猴子和猩猩也会用这个动作打招呼。由此可见，眉毛轻抬的动作是天生的。

把眉毛快速地轻轻一抬，瞬间又恢复原位，这个动作是为了把别人的注意力引到自己的脸上，让人明白自己正在向他问好。**注意**：这个姿势在日本文化中是不被接受的。在日本人的眼里，这是一个不合理且被视为十分粗鲁的动作，甚至还被认为是含有明确的性暗示。

对迎面看到的人轻抬眉毛是一种下意识的反应。这就表示，你对出现在面前的这个人持有友善诚恳的态度。而且这个动作很可能与惊讶和高兴的情绪有关，仿佛在说：“见到您真是让我又惊又喜。”也可以理解为：“我非常敬畏您，并且对您有好感。”我们不会对擦肩而过的陌生人以及自己讨厌的人做出这个动作。

如果别人在见到你的一刹那没有对你轻抬眉毛，那就表示有可能他怀有挑衅的情绪。一个简单的实验会让你感受到这个动作的魔力：在酒店大堂的沙发上对陌生人轻抬眉毛，不仅所有人都会以相同的动作回应你，甚至还会有人主动走过来和你搭话。

> **黄金准则**：如果你喜欢他，就向他轻抬眉毛；如果你想让他喜欢你，还是向他轻抬眉毛。

眉毛的不同动作往往透露出千差万别的心理信号：

*   **眉毛上扬**：表示当事人知道自己问题的答案，是明知故问。
*   **眉毛外角提高**：会看起来很惊讶。
*   **眉毛内角高一点，离鼻子远一点**：看起来会变得很无奈忧愁。
*   **眉毛末梢若仅增加一点点眼角的细纹，仿佛眼睛在笑**：显现出快乐的感觉。
*   **眉毛位置降低，前额皱纹增加**：看起来像是在发脾气。
*   **眉毛倾斜**：表示悲伤。
*   **眉毛挑起，向中间靠近**：是害怕的表现。
*   **眉毛向下紧皱，上眼睑扬起，眼周绷紧**：表示行为人很可能准备袭击别人。
*   **眉毛上扬，并挤在一起**：是害怕、担忧和恐惧的表现。
*   **眉毛往下，眼皮上扬**：说明当事人是个好斗的人。
*   **眉毛向上拉紧**：是恐惧的表现。
