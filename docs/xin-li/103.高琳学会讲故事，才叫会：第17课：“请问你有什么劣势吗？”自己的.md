# 103.高琳学会讲故事，才叫会：第17课：“请问你有什么劣势吗？”自己的

在本节课中，我们将探讨面试中最具挑战性的问题之一：“你的弱点是什么？” 我们将学习如何通过讲述一个精心构建的故事，将面试官的关注点从你的弱点本身，转移到你如何认识并改进它的过程上，从而展现你的真诚、自省与成长潜力。

上一课我们介绍了如何在面试中讲述“我是谁”的故事，以展示能力、潜力以及与公司价值观的匹配。本节中，我们来看看当被问及自身劣势时，应如何巧妙回应。

许多人误以为这个问题可以敷衍了事，给出一些看似聪明实则空洞的答案。例如，声称自己“最大的缺点是追求完美”或“工作太努力”。这类回答在面试官看来缺乏诚意，因为没有人是完美的。若声称自己没有弱点，则可能暴露出两种问题：一是盲目自大，缺乏自我认知；二是内心脆弱，无法正视和改变缺点。在快速变化的职场中，企业需要的是**皮实**且**靠谱**的团队成员。

实际上，适当地暴露弱点并展示脆弱面，是建立信任与真诚关系的一种方式。当一个人能正视并讲述自己的某个缺点时，恰恰证明他接纳了这个问题，知晓解决方案，并且内心足够强大。

当然，并非所有弱点都适合在面试中分享。你所选择的弱点应满足以下条件：
*   **非致命性**：该弱点不应是职场通识中的严重缺陷（如情绪管理差、拒绝合作），也不应是应聘岗位的核心能力短板（如应聘数据分析师却对数字不敏感）。
*   **已改进或可控**：最好是一个你经过努力已经得到改善，或正在积极管理、可以控制的弱点。

那么，如何回答才能既真诚又出彩呢？核心方法是 **“换框”** 。即通过一个包含**冲突、解决冲突和完美结局**的故事，引导听众将焦点从“你的弱点是什么”，转换到“你如何解决这个弱点”的新框架中。

以下是运用“换框”法讲述弱点故事的具体步骤，我将以自己的真实经历为例进行说明：

1.  **提出弱点**：清晰指出你的一个具体弱点。例如：“我是一个主意特别多的人。”
2.  **描述冲突（弱点带来的负面影响）**：讲述这个弱点在特定情境下如何造成了问题或挑战。例如，作为创业者，主意多导致在招聘决策上反复无常，给执行团队带来困扰。
3.  **展示解决方案与行动**：详细说明你如何认识到问题，并采取了哪些具体步骤来改进。例如，我增加了一个“自我提问”的步骤：在提出新想法前，先区分这是“建议”（供讨论）还是“决策”（需说明理由）。
4.  **呈现改进结果（完美结局）**：说明行动带来的积极改变。例如，团队反馈我不再朝令夕改，这个弱点变得可控。
5.  **点明升华**：总结这个经历让你学到了什么，如何将弱点转化为资源。例如，主意多能为团队提供更多可能性，而通过建立过滤系统，它从决策干扰变成了创新助力。同时，这个过程也展现了我**懂得自省、善于倾听**的领导力。

请注意，讲故事时应保持真实感。弱点很难完全根除，强调“变得可控”比声称“彻底改变”更可信。例如，我至今仍主意多，但已能更好地管理它。

本节课中我们一起学习了在面试中回答“你的弱点是什么”这一难题的策略。关键在于避免自作聪明的套路，选择一个**非致命、已改进或可控**的弱点，并运用 **“换框”** 技巧，通过一个结构完整的故事（**冲突 → 解决 → 结局**），将考官的注意力引导至你解决问题的能力和成长历程上。任何弱点都是双刃剑，故事讲好了，就能让听众关注你挥舞这把剑的**途径与方法**，而非仅仅盯着剑刃本身。
