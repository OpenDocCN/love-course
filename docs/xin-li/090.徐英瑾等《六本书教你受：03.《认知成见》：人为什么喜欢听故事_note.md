# 090.徐英瑾等《六本书教你受：03.《认知成见》：人为什么喜欢听故事_note

在本节课中，我们将深入探讨一个具体的推理谬误——“合取谬误”，并分析其背后的心理学原理。我们将了解为何人们有时会违背概率原则，更倾向于相信一个更具体、更生动的描述，即使它在逻辑上发生的可能性更低。

上一节我们从宏观上介绍了认知成见，本节中我们来看看一个非常具体的推理谬误。

## 合取谬误：一个具体的推理陷阱

我们要讨论的这个具体推理谬误叫做“合取谬误”。“合取”即逻辑中的“且”关系。我们为何要讨论它？这背后有一个更大的背景。

这个背景就是概率推理。概率推理在日常生活中非常有用，因为现代科学技术重组了我们的世界，“概率”一词经常被提及。例如，某人可能会请同学分析他追上某人的概率是多少。

然而，我们在使用概率概念时真的理性吗？一些学者认为并非如此。这里要提到两位作者：特沃斯基和卡尼曼，他们是两位非常重要的心理学家，也在行为经济学上做出了许多贡献。

特沃斯基和卡尼曼先生做了一个实验，以加拿大英属哥伦比亚大学的88位大学生为被试。这个实验被很多人重复过，其结果应该是成立的。

他们问大家一个问题，其中虚构了一个叫“琳达”的人物。琳达姑娘今年31岁，单身、健谈、聪明，大学时学哲学，学生时代就很关心种族歧视问题和社会公义问题，也曾参与过反核示威游行。这个故事波澜不惊。

听完后，大家要回答问题：现在告诉你琳达的职业，你猜她更可能是一位银行出纳员，还是一位“身为女权主义者的银行出纳员”？

许多同学回答：“那应该是身为女权主义者的银行出纳员。”少数同学选A（银行出纳员），大多数同学选B（身为女权主义者的银行出纳员）。

这是一个概率问题。其核心原理是：**合取事件发生的概率，低于其中任一单个事件发生的概率**。

这句话如何理解？例如，假设你是一支被敌人包围的部队，要派侦察兵出去侦察。你不会只派一路侦察兵，因为这一路出意外就没消息了。你会派四路，东西南北各一路。你会想：不会那么倒霉吧，派出四路侦察兵都没消息，总有一路会回来。

这里用到的概率学原理是：四个事件（即四路侦察兵都完蛋）发生的概率相当低，低于单路侦察兵倒霉的概率。我们在日常生活中也常这样做，比如一些重大工作，我们都要搞备份方案，A计划、B计划。我们说“不能把所有鸡蛋都放在一个篮子里”。

如果我们按照这个思维去看琳达的案例，那么现在有两个命题：
1.  琳达是一个银行出纳员。
2.  琳达既是女权主义者，又是银行出纳员。

B事件显然是个合取事件，它的发生概率非常低，就像四路侦察兵全部完蛋一样。那么我们应该选A。为什么这么多同学都选B？

心理学的解释诉诸了另外一个理论：**心理原型论**。“原型”指原来的“原”，型号的“型”，英文叫Prototype。原型是指人类在进行类型归类时，每一个类型都有一个理想的范本。其他个体要靠近这个理想范本，才会被认为是这个类型的成员。

我随便举个例子。“家具”这个概念，桌子是家具吗？当然是。椅子是家具吗？是家具。壁炉是家具吗？很多人会愣住：壁炉？好吧，勉强算家具。有些东西就勉强了。

再说“颜色”。红色是颜色吗？是。蓝色是颜色吗？是。绿色是颜色吗？是。那么灰色是颜色吗？也算吧。但灰色不是很典型的颜色。“颜色”这个词首先给我们的感觉是很鲜艳的东西。

人也是这样。我们在判断人时，是按照典型性原则。对于好莱坞的女艺人来说，是“离婚四次或四次以上”更具有代表性，还是“投票给民主党”更具有代表性？大家肯定会认为，是离婚四次以上更具有代表性。

尽管实际上，投票给民主党的女艺人显然要比离婚四次以上的女艺人多得多。首先要知道，好莱坞几乎所有人都投民主党的票，只有极少数人（据我所知施瓦辛格是共和党）例外，大多数都是民主党。所以好莱坞是民主党的票仓，这我们都知道。

但是离婚四次以上的人，应该不常见吧？所以，投给民主党票的人，应该多于离婚四次以上的人。但是，离婚四次以上对好莱坞的女艺人来说，是有“代表性”的。这是因为虽然在好莱坞也不是人人都离婚四次以上，人数也占少数，但他们这个比例已经高于美国其他任何行业了，足以让我们觉得“这是你们的特征”。

请注意，离婚四次以上成为“你们的特征”，并不是因为这个特征在你们那里已经超过50%了，而是指你们这里所占的比例已经高于其他行当了。这已经构成了一个所谓的“诊断性特征”，能够让我判别你是什么特点。

大家会问：我说了这么多例子，这和前面所说的琳达案例有关系吗？有关系。

琳达案例前面告诉大家一个故事：琳达学哲学、关心公益活动、非常关心和平。这听上去好像是女权主义者的“标配”。这实际上已经勾勒了一个人物肖像，这个肖像与“女权主义者”相关。

正因为勾勒了这样一个人物肖像，现在给我两个答案：一、她是个银行出纳员。我的第一个想法是：银行出纳员和她上面说的这些信息有半毛钱关系？显然没关系。所以我们大脑对这个信息觉得很陌生。

你第二个信息是说：她是“身为一个女权主义者的银行出纳员”。好吧，我现在不管“银行出纳员”那一部分，“女权主义者”这个词我抓到了，这和前面的信息有关联。Bingo！抓第二个信息。这时候谁还管概率论啊？概率论在哪飞啊？

那么有人就会追问这样一个问题：我们在这里系统地违背概率性的要求，仅仅根据这个故事给我们的一个典型性的概念案例来进行匹配，这样做理性吗？

## 从进化论视角看“非理性”偏好

我来思考一个问题：如果你从进化论的角度来看待这个问题，就和概率论的要求不一样了。

概率论现在告诉我们一件事情：少说话。你说得越少，你错的概率也就越低。比如你说“她是银行出纳员”就够了，你非要说“她是身为女权主义者的银行出纳员”，你不就多说话了吗？这就是一个合取命题了，合取命题成立的概率就低了。

好吧，如果你真按照这个概率论的要求，你就别说话了，你不说话最安全。你说出越详细的一个东西，你就越“不行”。

但是，如果你站在进化论的要求，情况就不一样了。大家想想看，进化论的一个基本原理是什么？就是我们的这些行为要和外部的环境发生互动，然后来检测我们的行为是否适应于这个环境。

类似于：一只鸟看到这里火山要爆发了，“噗吱噗吱”就飞走了。你飞得慢，就死了。这很简单嘛，你一定要行动。

那么信念（英文的believe）和行动（action）之间的关系是什么呢？就是我们的行动是建立在这个信念的基础上。比如我有一个愿望：我现在渴了。但是我现在看到前面有两瓶水，一瓶是淡水，一瓶是海水。

如果我相信喝海水会让我倒霉，喝淡水是OK的，那么我就会舍弃海水去拿淡水，“咕咚咕咚”就喝了。你看，这个信念本身是正确的，我就活下来了。

你如果哪个家伙知识错误，相信喝海水也行，他喝了海水脱水挂掉了，死了。所以他的信念如果错了，就会导致行动也错了。行动错了会导致什么呢？他死了。这种“笨人”的基因就传播不下去。

也就是说，信念和行动有一种勾连，而行动又和你的适应性发生了勾连。更宏观的角度，就和基因的传播有关了。所以，产生正确的信念导致正确的行动。

但是，不是所有的正确信念都能够引发行动。比如空洞的、没有思想内容的这样一个信念，就没有办法导致行动。比如我说：“屁或者非屁，当中总有一个是真的。”好吧，你表达个逻辑规律，叫排中律。它变不成任何的行动，是不是？

你看哈姆雷特，他不知道是否应该杀死他的叔父时，他就进入了这样一种说空洞话的循环。“生存还是毁灭，这是一个问题。”他如果不停地问这个问题，他就没有办法复仇了。他最后复仇，是因为他从这个更抽象的哲学问题“下降”了。

所以呢，我们不但需要真的信念，而且我们还需要能够兑换成行动的信念。那么，你要变成一个兑换成行动的信念，这个信念就要具体。因为行动是具体的。

假设我是只狮子，我饿了。（老实说我在录节目的时候，我也饿了。）你就必须有一个具体的信念。我自己就有一个体会：我有一次在日本福冈旅游的时候，我沿着这条街找东西吃。我脑子里形成的东西是“我要在这条街上找东西吃”，但是我根本就不知道我应该去找哪一家。反正或者这家，或者那家，无非就这个样子。

然后我碰到一个问题：我走了整整半小时，我都数不清我走过多少家店了，我都不知道我应该确定哪一家。结果我肚子还是饿的。最后我吃了一家很普通的拉面馆。

我要通过这个案例告诉大家什么？如果我一开始形成的这个信念就是具体一点——不是“我要在这条街上吃东西”，而是“我要在街上吃拉面馆”——我的搜索范围将大大减小，是不是？我会跳过大概70%的店。“拉面，拉面，拉面”，我就找拉面就是了。然后在拉面里面找一家。或者我就选了中餐馆。

这就是我们空洞的想法难以兑换成行动，只有具体的想法它能兑换成行动。

动物世界也是这个样子。如果有一群斑马在前面跑，我是只猎豹，我说“我今天要抓一批斑马”。那么多斑马，你要抓哪一批啊？如果前面有一千匹，我编号ABCDEFG编到一千，一个巨大的西绪福斯（徒劳）任务，我就在这里“生存还是毁灭，这是个问题”。还有，我这样想，狮子这个物种要灭绝的。

它肯定是有个具体的方向，就是它要找……你看它的算法非常简单：找任何一个斑马群里面跑得最慢的斑马下手，小的、慢的斑马或者是羚羊下手，这是最容易上手的目标，非常非常精确。

所以呢，我们有可能，我们人类的进化机制已经培养出了一种对于特定信息的敏感性。我们并不是很喜欢处理非常抽象的信息。

所以从进化论的角度上来讲，你看这两条信息：一个“琳达就是一个出纳员”；另一个“哎呦，好生动哦，一个身为女权主义者的出纳员”。这显然就给我一种画面感了。你显然觉得第二个画面感更容易和你生活中的场景发生关联。第一个冷冰冰的，能够告诉我什么信息？

这样一个能够向实际信息兑换的可能性，就使得我们偏好那些更加具体的描述。而更加具体的描述从逻辑上看，恰恰很可能是加了很多形容词的描述，而这种描述恰恰是合取命题。这也就是为什么我们在日常生活中会犯合取谬误。

讲到这一步，我们就会发现，我们上面站在进化论角度对于合取谬误的这种“洗白”，好像给了我们一个更广泛的思考途径：理性到底是什么东西？我们要在怎样的理性观里面来看待合取命题的推理谬误？这也就是下一节的内容了。

## 本节总结

好，说到这里，我们这一节的内容就到此为止了。我们现在来做一个小结。

本节课中我们一起学习了：
*   **合取谬误**：在有些特定环境下，我们会认为合取命题的成真概率要比单个命题来得更高，比如像琳达案例这样的环境。
*   **心理原型论**：如果背景信息正好给出了某种心理原型，而我们要加以选择的两个选项当中，有一个选项恰好契合了这个心理原型，我们就会偏好于那个出现心理原型的选项。
*   **进化论视角**：我认为从进化论的角度，这种偏好是可以得到说明的。因为这种有心理原型加以支持的选项，它因为具有更丰富的内容，所以更容易被兑换成具体的行动，因此会在进化中被加以偏好。
