# 课程007：自省——控制“分寸感”的旋钮 🎛️

在本节课中，我们将学习“自省”这一核心概念。自省并非简单的自我批评，而是一种动态的自我观察与关系校准过程。我们将探讨为何自省至关重要，以及如何通过具体方法，像调节旋钮一样，掌控人际交往中的“分寸感”。

## 概述：自省是动词，不是名词

自省是一个**动词**。它意味着需要随时随地对自己的言谈举止及所处位置进行观察和监测。

如果你不主动进行自省，那么总会有人“帮助”你完成这件事。当你自我感觉良好，认为一切平顺时，这往往意味着周围正有人在忍耐你。这是从亲身经验中得出的个人教训。

## 从“同心圆”到“一个人”：我的教训

上一节我们介绍了自省的基本概念，本节中我们来看看一个因缺乏自省而陷入困境的具体案例。

曾经，我的个人社交很少。我误以为这是好事，因为人生苦短，我把时间都留给了亲密朋友，而非不相关的人。然而，人际关系是一种**同心圆结构**，自己位于中心，根据关系的亲疏远近一层层向外扩散，对不同的人应有不同的相处之道。

我却把本应是同心圆的人际关系，简化成了“一个人”。在朋友面前，我表现得过于随意；在朋友之外的人面前，我又显得过于冷漠。我认为这样正好，生活简单有序，没有麻烦。

但我并未意识到，朋友并不喜欢那个随意的我。他们觉得我的自我漫无边际，将别人逼迫到了角落。当我发现朋友们长期在默默忍受我时，我感到非常难过。一方面觉得对不起他们，另一方面也为自己难过，不明白自己为何变成了这样。

## 自我膨胀与边界失守

其实原因很简单：如果一个人生活过于平顺，就会失去自省能力，认为一切理所应当，觉得自己“一切都刚刚好”。

**公式：** 自我感觉“刚刚好” ≈ 自我溢出边界，侵占了他人领地。

通常情况下，人很难意识到这一点，看不清自我的真实模样。山本耀司曾说，自我必须与外界发生猛烈碰撞，一个人才能发现自我的存在。意识到朋友在忍耐我，就是一次猛烈的碰撞。我并非自己想象中那么好，在他人心中也并非我以为的样子。这让我认识到，我的自我对他人而言是多么令人讨厌的存在。

这固然令人沮丧，但也激发了我自省的勇气。

## 自省的核心：检查关系，确定边界

在我看来，自省并非简单地寻找自己的错处。既然谈到自我，就必然存在“他人”。自省更多的是**检查自我与他人之间的关系，确定彼此的边界在哪里**。

人们常用“亲密无间”形容紧密关系，但我的教训是：永远是“亲密有间”。人与人之间永远存在距离，控制好这个距离就是所谓的“分寸感”。

**公式：** 分寸感失衡 = 人际关系问题
*   分寸感太强 → 拒人于千里之外。
*   毫无分寸感 → 令人窒息，压制他人自我。

自省就是控制这个距离的**旋钮**。我的问题正是长期让这个旋钮失效，几乎在人际关系上“坠毁”。

## 自我的两面性与流体本质

人总有可亲可爱的一面，也有令人厌憎的一面。人们之所以能愉快相处，是因为彼此都用可亲可爱的一面相对。关系能更进一步，则是因为觉察到对方令人厌憎的一面后，依然选择接纳其整体。

另一方面，更重要的事是通过自省，**控制自己流露出令人厌憎的那一面，将其控制在他人可接受的范围内**。

当自我膨胀时，人会天真地认为自己只有一面，人人都爱自己。但这只是一种幻想。我们通常达不到那个人人喜爱的程度，能做到不让人讨厌已属不易。

在自省中，我发现自己最大的错误是误解了自我的状态。作为一名需要常年表达和输出的写作者，我误以为自我是一种**固体**，需要像金刚石一样坚硬，才能持久发声。

**代码描述（比喻）：**
```python
# 我曾以为的自我
my_self = “solid” # 固体，坚硬不变
# 实际上的自我
my_self = “fluid” # 流体，流动变化
```

但回顾十年前、二十年前的文章，我找不到那个所谓坚固的自我。我的想法和观念都在悄然改变。自我其实是一种**流体**，就像铁板烧台上的油。厨师必须飞快挥舞两把铁铲，才能让油集中而不四散流淌。只要没有约束，它就一定会扩张到无法继续为止。

**自省就是那两把铁铲**，需要一刻不停地将自我约束在有限空间内，约束在他人可以容忍的边界内。这就是为什么喝醉的人常令清醒者讨厌——酒精麻痹了理性，让平常被控制的那一面自然流露，而那一面往往不讨人喜欢。

## 九条自省方法

找到一个让自己和周围人都舒服的位置并不容易。一旦找到，又很容易忘记那只是暂时状态。我曾以为可以永远保持，但现实残酷，很快我就撞上“墙壁”，变成了令人厌烦的家伙。

幸好，我在大学的流体力学成绩不错，于是为自己总结了几条约束“流体自我”的自省方法。以下是具体清单：

1.  **谈话倾听率**：在每一次谈话中，我倾听别人说话的时间是否占到50%以上？
2.  **理解意图**：我和别人说的每一句话，是否建立在听懂对方意图的基础之上？
3.  **对话目的**：我和别人说的每一句话，是为了提供真实价值，还是为了表现自己的聪明才智？
4.  **对话方式**：我有没有用打断、嘲讽、反问的方式和他人对话？
5.  **边界意识**：我是不是越过了边界，对我不懂或不关心的事情发表了看法？
6.  **三思后言**：我在每一句话出口之前，是否先想过一遍，说出来会不会后悔？
7.  **状态觉察**：我此刻的状态究竟是单纯的感知，还是想要表达自己的某种欲望？
8.  **动机审视**：我此刻是否想从别人那里得到什么？
9.  **视角切换**：此刻别人眼里的我可能是什么样子？

在这九条中，我个人认为**第九条尤为重要**。

## 最重要的视角：他人眼中的自己

我们并非天然地成为自己。所谓的自我，其实是在人群中慢慢形成的。他人对我们的观感和期待塑造了自我——父母的期许、老师的肯定、朋友的赞美，让我们努力发展受肯定的那一面；同样的，他们的否定、排斥和批评，也让我们竭力控制受否定的那一面。于是，自我才能在人群中找到合适的容身之地。

我们每个人生下来就像一个圆，彼此拿着刀子在对方身上砍出一些缺口。最后，我们都变成了齿轮。也正因为我们都是齿轮，才有相对合适的位置，并且彼此紧紧咬合，共同转动，社会才得以正常运转。

因此，最重要的自省方式就是：**随时思考此刻别人眼中的自己，切换到他人的视角进行自我观察**。这种观察甚至要比他人真实的视角更为严厉，因为他人无法像自己一样清晰地认识自我的所有掩饰、伪装和小花招。

如果能持有这种诚恳的态度，就不容易沾沾自喜，或像我一样自我感觉良好，误以为一切平顺而无需关注自我模样。

## 总结与结语

本节课中，我们一起学习了自省的本质与实践。

**总结要点：**
1.  自省是动态的自我观察与关系校准，是控制人际“分寸感”的旋钮。
2.  自我并非坚固不变，而是像流体一样需要持续约束。
3.  自省的核心是检查自我与他人的关系边界。
4.  通过九条具体方法，特别是“切换至他人视角”，可以有效地进行自省。
5.  自省是一件需要每日进行、终生不懈的事。经验表明，稍不注意，人就会回到老路，自我再次“泛滥成灾”。

最后，自省需要每天进行，终生不可懈怠。根据我的经验，一段时间后只要稍不注意，人就会回到老路，自我又开始泛滥。于是，在他人眼中的自己，就会在可爱与可憎之间坐过山车。如果可能，请尽量让自我的轨道平滑一些，免得让人心惊肉跳，觉得无论远近都难以把握。
