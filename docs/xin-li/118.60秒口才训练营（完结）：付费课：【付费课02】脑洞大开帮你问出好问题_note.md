# 118.60秒口才训练营（完结）：付费课：【付费课02】脑洞大开帮你问出好问题_note

在本节课中，我们将学习如何通过提问来改善沟通。你将掌握两种提问方式，并学会一种名为“脑洞大开”的训练方法，帮助你在聊天中引导话题、让对方更愿意开口。

---

沟通时，你可能会遇到两种常见问题。第一种是你在滔滔不绝地说，对方只是被动倾听。第二种是对方开口后话题跑偏，你难以拉回正题。本节将探讨如何通过提问解决这些问题。

上一节我们介绍了沟通中的两大挑战，本节中我们来看看如何通过提问来应对。

**提问的第一原则是：你的问题必须承接对方上一个回答中的关键词。** 你不能随意开启全新话题。例如，如果对方提到“喝粥”，你可以接着问“你喜欢哪种粥？”，而不是突然问“你周末做什么？”。好的对话应如流水般自然，而非跳跃式的审问。

提问主要分为两种形式。

以下是两种提问形式的说明：

*   **封闭式问题**：答案通常是“是/否”或有限选项。例如：“你吃饭了吗？”或“你选A还是B？”。这类问题容易终结对话。
*   **开放式问题**：答案没有限制，能引导对方展开叙述。常使用“为什么”、“怎么样”等词语。例如：“你为什么这样想？”或“你觉得这个怎么样？”。在希望对方多聊时，开放式问题更有效。

那么，当对方话题跑偏时，如何巧妙地拉回来？关键在于**联想能力**。你需要找到当前话题与你目标话题之间的关联点，通过一个承接关键词的开放式问题，自然地将对话引导回来。

为了训练联想能力和提问技巧，我们引入“脑洞大开训练法”。

**脑洞大开训练法的核心是：用几个不相关的词语造句，并在结尾提出一个开放式问题。** 这能同步锻炼你的联想、表达和提问能力。

例如，给定三个词：**无聊**、**蚂蚁**、**电影**。

你可以这样组织语言并提问：“某个周末我感到很无聊，于是翻看手机找新电影，发现一部叫《蚂蚁》的电影。这让我很好奇，如果你是这部电影的导演，你会想表达什么主题呢？”

或者换一个场景：“我在电影院看电影时，突然感到手上有东西在爬，原来是一只大蚂蚁，吓了我一跳。如果你在电影院遇到这种情况，你会怎么办？”
