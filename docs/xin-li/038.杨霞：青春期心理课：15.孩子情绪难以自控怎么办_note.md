# 038.杨霞：青春期心理课：15.孩子情绪难以自控怎么办_note

在本节课中，我们将学习青春期孩子情绪波动的原因，并掌握一个核心的心理调节方法，帮助孩子建立情绪自控能力。

许多家长发现，孩子进入青春期后，情绪容易波动、敏感且冲动。这个阶段常被称为“暴风骤雨期”，情绪来得快、去得快，且非常猛烈。孩子发脾气时，情绪似乎完全失控。

家长也能感觉到孩子自身处于失控状态。童年时，孩子的喜好简单，可能只对吃喝感兴趣。到了中学阶段，孩子开始要求尊重、说话算数、注意隐私、不与他人比较。他们对家长的用词、方式方法变得敏感，拒绝命令，渴望得到及时指导而非控制。

家长感到与青春期孩子相处非常困难，人际关系难以处理。孩子既希望被理解，又不愿多交流，因为情绪像炮仗一样，不知何时会“点火”。

心理学家发现，在此阶段，孩子的思维尚未成熟，而情绪发展已超越童年水平。思维与情绪之间尚未建立良好沟通，孩子还不能经过理性思考来管理情绪，常处于冲动状态。

因此，培养孩子的协调性有助于打通思维与情绪的联系。大脑管理思维和情绪各有特定区域，当这两个区域能良好交换信息时，孩子会显得通情达理。“通情”指情绪，“达理”指思维。通情达理的孩子，其情绪反应与自身及他人的期望是合适且恰当的。

孩子在此阶段必然出现情绪与思维发展的不平衡。可能思维发展在前，情绪在后；也可能情绪在前，思维在后。情绪容易激动、不考虑后果的孩子，往往是情绪发展领先。

家长和老师都希望孩子能控制好情绪。但心理学研究发现，人对自己的思维和情绪基本难以直接控制。心理现象主要包括思维、情绪和行为。

人对行为是可以控制的。例如，孩子饿了会产生饥饿情绪和想吃东西的想法，但大部分孩子不会因此去偷窃。人能自觉控制行为，不做违法之事。

当你发现自己能做到不喝别人的水、不偷超市的包子、不摘花园里的花时，你就明白行为是可以被控制的。因此，可以不断训练自己控制行为。

例如，同学与你吵架或出言不逊，你感到生气（情绪），想揍他（想法），但你不能真的动手，因为后果不堪设想。情绪包含情感、冲动和态度。讨厌某人是一种情绪，随之可能产生挑剔、抱怨、骂人等想法。

但将这些想法付诸行动没有好结果。闲言碎语不会改变对方的讨厌，反而会让他人对你的品德产生质疑。修养会因此降低。

因此，我们要学会允许自己拥有情绪，接受情绪和想法，但管理好行动。这种方法称为“顺其自然，为所当为”，在心理疗法中属于森田疗法，遵循八字方针：**顺其自然，为所当为**。

“顺其自然”指对情绪和思维采取接受态度。当你心里出现讨厌、厌烦、愤怒、抱怨等情绪时，接受它，这些情绪通常在五分钟内就会消失。

例如，感到紧张时，就接受紧张；感到担心时，就接受担心。但同时，按部就班地拿起笔开始计划或行动。行动应遵循规范和平时的训练，而情绪则被接纳。当你接受紧张、恐惧、厌烦或愤怒的情绪时，它们很快会过去。

接受之后是忍受，接着“为所当为”，最终会达到享受的状态。否则，与情绪对立会导致难受。不要与情绪对着干。

另一种情况是“为所欲为”，即情绪来了就骂人、发火、摔东西。这种行为如同动物，无法使人成为有修养、有文化的成年人。长期为所欲为可能导致进监狱或医院。

还有一种方式是拼命抵抗情绪，例如紧张时告诉自己“别紧张”，生气时说“别生气”。但情绪控制有一个规律：**越克制，越逆反**。例如，越想睡好越睡不着，越想考好越考不好，越告诉自己不紧张反而更紧张。

因此，紧张情绪不可克制，但也不能为所欲为。例如，见到某人感到害羞，如果因为紧张而躲着对方，关系会变得更紧张。

正确的做法是接受自己的紧张、害羞和尴尬，同时“为所当为”。主动打招呼：“你好，最近怎么样？”对方可能会感到惊讶，原本紧张的关系可能因此缓和，甚至成为好朋友。

在处理交往、复杂关系或幼稚想法（如饿了想拿别人东西）时，都可运用此方法：**对我们的情绪和想法学会接受（顺其自然），对我们的行为学会约束（为所当为）**。这样才能成为一个成熟、有自控力的正常人。

本节课中，我们一起学习了青春期孩子情绪难以自控的原因，并掌握了“顺其自然，为所当为”这一核心心理调节方法。通过接受情绪、管理行为，可以帮助孩子更好地度过情绪暴风骤雨期，成长为通情达理、有自制力的人。
