# 088.【完】十点课堂即学即用：第2堂：沟通拒绝绕弯子，省时省力更高效

在本节课中，我们将学习如何摒弃含蓄与委婉，掌握直接、清晰、高效的沟通方式。我们将探讨直接表达的核心目的、常见误区、实用规范以及一个循序渐进的技巧模型，帮助你节省沟通成本，提升社交效率。

## 概述：为何需要直接表达？

沟通表达是人际关系的桥梁。我们东方民族常倾向于含蓄、委婉的表达方式，但这可能导致社交效率相对低下。

回想童年时期，我们的表达反而更直接有效。例如，摔倒时会直接说“我这里疼”。成年后，我们却常常掩饰感受，让对方猜测，例如回答“你觉得呢？”。直接表达内心感受和需求，往往能更快获得理解与帮助。

在职场上，高效的沟通尤其需要直接、明了、坦率的表达方式，以节省彼此时间，建立清晰的合作基础。但请注意，直接表达不等于口无遮拦的情绪发泄。

## 避免三种错误的“直接表达”

许多人误以为直接表达就是“想说什么就说什么”，并常以“我说话比较直”作为开场白。然而，缺乏技巧的直率可能适得其反。以下是三种需要规避的错误方式：

**1. 过于直接，不留余地**
东方文化中，赞美往往含蓄，批评却过于直接，容易引发对方抵触情绪。
> **反面例子**（妻子对做家务的丈夫）：“你脱的地根本没脱干净！你看，那都是头发丝呢！难得干一次家务，我夸奖还嫌弃？以后请你不要再做了！”

**2. 信息过多，容易混乱**
试图在一句话中塞入过多信息，会让对方抓不住重点。
> **反面例子**：“亲爱的，你带孩子到外面玩一下吧，记得给他穿上那个红色的毛衣，还有一个灰色的外套，还有一个那个粉红色的手套，都记得给它戴上啊。什么颜色的风衣？什么颜色的围巾？还有什么…听不懂吗？”

**3. 抓错重点，无效沟通**
将直接表达等同于情绪发泄，尤其在职场中，这并非有效的沟通方式。
> **反面例子**（下属对上司）：“我感觉这个事情让我很很受伤害，我很疲惫，你根本就不在乎我们！”（上司可能回应：“我发工资让你干活，你还来指责我？”）

以上方式虽是“直接表达”，却因未经训练，成了“杀敌一千，自损八百”的沟通，无法达成高效沟通的目的。

## 真正直接表达的目的与核心

上一节我们看到了错误的直接表达，那么真正的直接表达是怎样的呢？首先，我们需要明确其目的。

我们习惯间接委婉地表达心理和生理需求。但当我们的表达“七拐八绕”时，接收方也可能无法准确理解，从而造成无效沟通。

**直接表达的目的**是养成直接表达感受和需求的习惯。想要什么、不想要什么、喜欢或不喜欢，都可以直接说出来，从而在沟通中精准达成目标。

**直接表达的核心**可以用一个简单的公式概括：
> **直接表达 = 我的想法是什么 + 我的需求是什么**

其目标是让对方用最小的信息成本，理解你要表达的内容。

**举例对比：**
*   **不直接表达**：“我们家的卫生状况不是很好啊，所以你要多做几次打扫，这样子才能活得更健康嘛。”（对方可能困惑：“就我一个人打扫？你不用干活？”“我这不是没时间吗？”）
*   **直接表达**：“亲爱的，我感觉一个人整理家务好累啊，要是你能跟我一起分担就好了。”（对方可能回应：“老婆/老公你辛苦了，以后的家务我来陪你一起做。”）

直接表达尊重了彼此的沟通成本，能更有效地分享感受与想法，让对方立即收到精准信息，提升执行力与时效性。

**另一个例子：**
*   **不直接表达**：“你去买点水果回来。”（对方可能买回你不喜欢的火龙果，引发矛盾）
*   **直接表达**：“亲爱的，你去买点苹果或者葡萄回来吃吧。”

## 直接表达的四大规范与界限

学会了核心，我们还需要为直接表达设立规范与界限，以便在人际交往中既能保护自己，又能有效沟通。你需要掌握以下两条规范和两条界限：

### 规范一：沟通中要有逻辑
逻辑是事物客观运转的规律。沟通时，我们需要理清话语背后的逻辑。低效的沟通往往缺乏逻辑。
> **反面例子**：“哎，那个你把那个给我拿回来。”“哪个呀？”“那个！哎呀跟你说点啥事这么费劲的！”

高效的直接表达需要考虑三点：
1.  我希望向对方表达什么意思？（**我的意图**）
2.  对方在问我表达什么样的意思？（**对方的需求**）
3.  我跟他在哪些方面要达成共识？（**共同目标**）

> **正面例子（运用逻辑）**：
> 妈：“雅，把那个给我拿了。”
> 我：“妈您再说得仔细一点，我刚没听清楚呢。您再说一遍吧？”（**确认意图**）
> 妈：“哦，就是那个遥控器，在茶几上。”
> 我：“哦，妈，是遥控器啊，我马上去拿，您等我哈！”（**理解信息，达成共识**）

### 规范二：精确传达关键信息
在快节奏生活中，一句话内传达关键信息至关重要。信息缺失或冗长都可能导致误解。
关键信息的表达可以遵循以下公式：
> **关键信息 = 情况/结论 + 理由 + 具体行动**

> **例子**：“我们今天需要把各科作业带到学校去（**情况/结论**），因为我们要做期末考试（**理由**），今天就要带去（**具体行动**）。”

避免使用过于抽象的描述，如“给宝宝穿衣服”或“把表格拿过来”，而应说明“穿那件红色的毛衣”或“把上个月的销售数据表格拿过来”。

### 界限一：设置好语言规范
在表达“我的想法和需求”时，需考虑两点：别人是否会拒绝？我能承担表达后的结果吗？因此，需要保持平稳心态和彼此尊重。

> **反面例子**（对财务）：“怎么搞的？这个月奖金少算了这么多！什么意思吗？我做的不对吗？还是你们针对我？”
> **正面例子**（对财务）：“您好，我这个月奖金好像少算了几千块。我知道数据核算容易出错，特别希望您能帮我再仔细核对一下，谢谢！”

### 界限二：设置好尊重的界限
直接表达需把握分寸，目标是达成共识、促进关系。当对方排斥你的表达时，要及时止损，避免关系恶化。真正的界限能让你在任何情况下保护自己和关系。

## 直接表达的五阶技巧模型：金字塔 🗼

要想得体地进行直接表达，必须掌握说话技巧。下面介绍一个从低阶到高阶的“直接表达金字塔模型”，帮助你逐步提升。

### 1. 低阶：用“我”代替“你”（减少指责）
很多时候，用“你”开头的句子带有指责意味，不利于沟通。尝试用“我”来表达感受和需求。

| 指责式表达（用“你”） | 改进后表达（用“我”） |
| :--- | :--- |
| **A.** 你根本就没有参与到这课程的研发当中。 | **A.** 我感觉不能一个人来研发这个课程，有点力不从心。 |
| **B.** 你没有按时完成你的一部分工作。 | **B.** 我的时间有限，一直在等你的部分。 |
| **C.** 你妈妈怎么老是把垃圾捡回来？ | **C.** 我担心这些旧物会影响家里的卫生和健康。 |

### 2. 初阶：用“我们”代替“我”（促进合作）
在团队或家庭沟通中，一直用“我”可能显得逃避责任。用“我们”能强调共同目标，促进协作。

| 个人式表达（用“我”） | 合作式表达（用“我们”） |
| :--- | :--- |
| **A.** 我不能一个人来研发课程，感觉力不从心。 | **A.** **我们**不能一个人来研发课程，工作量很大，希望你也参与。 |
| **B.** 我的时间有限，一直在等你的部分。 | **B.** **我们**的项目进度有延误，一直在等你的部分。 |
| **C.** 我担心这些旧物会影响健康，但妈舍不得丢。 | **C.** **我们**的家庭健康是第一位的，这些旧物可能有影响，咱妈舍不得丢，我们一起和妈妈好好说说。 |

### 3. 进阶：从抽象描述到具体信息（线上沟通尤其重要）
线上沟通（如微信）容易因信息抽象产生歧义。表达时应包含：**情境/时间 + 具体对象（什么）+ 关键动作（干什么）+ 原因（为什么）**。

| 抽象表达 | 具体表达（公式应用） |
| :--- | :--- |
| 赞扬：你做得不错。 | **情境**：在时间紧任务重的情况下，**对象**：你为公司研发的天气APP测试工作，**动作**：完全达到了厂商和市场要求的规格。 |
| 请求：送女儿去幼儿园。 | **时间**：明天早上如果天气太冷，**动作**：请开车**对象**：送女儿去幼儿园。 |
| 指令：把表格打印出来。 | **时间**：今天下午，**动作**：把这份工商注册表格打印出20份并**放在我的桌子上**。 |

### 4. 高阶：平衡“事实”与“感受”（根据场景调整）
沟通中需注意事实与感受的配比。面对不同的人和场景，侧重点应不同。

*   **表达正面事件时**：**强调事实，弱化感受**。
    > **例子**：“你看，咱妈给我们做了炸排骨、清蒸鱼，都是我们爱吃的。”（强调事实）比“我感觉到幸福”（强调感受）更具体有力。
*   **表达负面事件或关切时**：**强调感受，弱化事实（尤其是带指责的事实）**。
    > **反面例子**（争论事实）：“你为什么对我生气？” “我没有。” “你就是生气了！你总板着脸！”
    > **正面例子**（强调感受）：“**我感觉**最近我们没那么亲密了，**我有点失落**。你最近好像很忙。”（对方可能回应：“我最近是比较忙，你多想了，等我忙完一定好好陪你。”）

### 5. 顶级：语言降维（极简高效）
在最高阶，我们需要对语言进行“降维”处理，直达核心，去除所有冗余。

| 冗余表达 | 降维后表达 |
| :--- | :--- |
| **A.** 原则上我同意你的感受，而且它也确实非常真实，但是我觉得我们似乎还有一点修改空间。 | **A.** 根据客户要求，这个设计还需要修改。 |
| **B.** 如果这个周末的某个下午，你有时间，又顺便在金茂附近带孩子的话，你可以帮我们推荐一下我们这个party。 | **B.** 周六下午三点，金茂大厦有个粉红之薇酒会，诚邀您参加并推荐。 |

## 总结

本节课中，我们一起学习了高效沟通的核心技巧之一——直接表达。

我们首先分析了含蓄表达可能带来的低效，并指出了三种需要避免的错误“直接表达”：过于直接不留余地、信息过多造成混乱、以及将直接表达等同于情绪发泄。

接着，我们明确了直接表达的核心目的是精准传达“我的想法”和“我的需求”，并为此设立了四大规范与界限：注重沟通逻辑、精确传达关键信息、遵守语言规范、以及设置尊重界限。

最后，我们掌握了一个实用的“金字塔模型”，通过五个阶梯式技巧——从用“我”代替“你”，到用“我们”促进合作，再到具体化信息、平衡事实与感受，最终实现语言降维——来逐步提升我们的直接表达能力。

掌握直接表达，就是掌握了一种高效、真诚的社交技巧。它能帮助你在各种场合清晰传达信息，节省彼此时间，建立更健康、更高效的人际关系。我是喜悦，我们下节课再见。
