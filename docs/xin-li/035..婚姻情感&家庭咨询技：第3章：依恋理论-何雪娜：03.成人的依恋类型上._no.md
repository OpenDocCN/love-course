# 婚姻情感&家庭咨询技能大全（完结）第035课：第3章 - 依恋理论 📚

## 第03讲：成人的依恋类型（上）🧑‍🤝‍🧑

### 概述

在本节课中，我们将要学习依恋理论在成人期的应用。我们将探讨婴儿期的依恋类型如何延续并影响成年后的心理状态与亲密关系模式，并介绍评估成人依恋的重要工具——成人依恋访谈（AAI）。

上一讲我们介绍了依恋的定义、起源、发展及其内涵，理解了为何依恋能帮助我们解读亲密关系。本节中，我们来看看成年期的依恋类型。

### 婴儿依恋类型与成人心理状态的关联

早先通过“陌生人情境试验”等方法，我们可以识别婴儿的依恋模式。同样，成年人的依恋心理状态也可以通过“成人依恋访谈”进行归类。下面我们来探讨婴儿期的依恋类型与其成年后心理状态之间的可能关联。

#### 安全型婴儿 → 安全型成人

安全型婴儿在“陌生人情境试验”中表现自信。他们能在陌生环境中自由玩耍，分离时会想念父母，与陌生人相处时不过度排斥，但更喜爱与父母在一起。重聚时，他们会表现出愉悦和欢迎。虽然预测到再次分离会难过，但在得到安抚后，仍能与陌生人互动。他们对于抚养者有信心，悲伤情绪平复后能继续探索。

他们成年后会成为安全型成人。与安全型成人恋爱，关系紧密且保有各自空间。

在成人依恋访谈中，他们的表现具有**言行的一致性**。他们能客观描述情境，无论喜好，评价不偏激。描述事件时**有因有果、逻辑清晰、主次分明**。他们会肯定依恋的价值，明白早年与父母建立稳固关系对成长至关重要。

由此可见，安全型婴儿与安全型成人之间存在显著相关性。

#### 回避型婴儿 → 冷漠型成人

回避型婴儿在“陌生人情境试验”中显得漠然。无论分离或重聚，情绪反应都很少。他们会主动回避父母，例如转身、不理睬或过度探索环境，似乎只在意玩具。他们忽视父母，不表达分离的难过，也不寻求肢体接触，同时也不表达愤怒。

他们成年后成为冷漠型成人，其成人依恋心理状态为冷漠型。

他们的语言**缺乏一致性**，常用**概述型方式**描述生活。例如，被问及与母亲关系时，可能回答“很好”，但追问细节时，语言**宽泛、空洞**，缺乏具体事件和情感连接。其描述常与定义矛盾，例如声称母亲“很好”，但后续描述中却包含母亲体罚或忽略他的事件。他们的反应简单概要，缺少事件支撑，显得前后矛盾。在亲密关系中，也可能出现类似的语言不一致现象。

#### 矛盾型婴儿 → 迷恋型成人

矛盾型婴儿在“陌生人情境试验”中，分离前就非常警觉，对分离敏感。他们会紧紧拽住照料者，显得无助恐慌，无法在照料者离开后探索环境。他们时刻关注父母举动，难以将精力放在学习上。这与许多孩子注意力不集中、行为问题的成因有关。当评估孩子学习问题时，或许可以考虑其依恋问题。

重聚时，他们表现出矛盾行为：要么愤怒，要么无助哭泣。抚养者的安抚很难让他们平静，因为他们似乎不相信照料者是稳定可靠的。哭泣后，他们可能显得非常无助。

他们成年后的依恋心理状态是迷恋型。

这类人在描述过去时，可能让听者感到“被淹没”。他们的描述**拉拉杂杂、缺乏前因后果和逻辑**，难以理解其真正想表达的内容。他们对依恋的需求**占满整个心智状态**，可能随时表达愤怒或无助。其叙述**句式冗长、语法混乱**，有时用古怪言语形容内心感受。与这类人打交道，会感到内在混乱，难以理清事件关联。

#### 混乱型婴儿 → 未解决/混乱型成人

混乱型依恋状态意味着内在有一个破碎的、不完整的自我影像。他们在父母在场时就开始表现出混乱和迷茫行为，例如呆住、神情恍惚、看向远方，或在父母离开时突然蜷缩在地。这些行为可能带有原始的灵长类动物特征，难以理解其意义。他们可能使劲哭，要求父母回来，但当被抱住时，一边挣脱一边又缠住父母，行为非常混乱。

混乱型依恋者并非时刻如此，也可能兼有矛盾型和回避型婴儿的某些行为举止。

成年后，这类人的依恋心理状态称为未解决型或混乱型。他们有大量未解决的内心心理问题。当讨论到虐待或丧失等情形时，他们的语言在**推理和逻辑上很混乱**，并有**令人吃惊的失误**。例如，描述已故亲戚时，会说“这个亲戚在表面上是活着的”，令人难以理解。他们可能陷入**长时间的沉默**，或**大段大段、毫无意义的歌功颂德**，这可能预示其内在的精神病性部分。

以上我们探讨了早年“陌生人情境试验”归类的婴儿类型，与其成年后依恋心理状态之间的关联。当然也存在复杂情形，需根据具体情境判断。

**切记**：在生活中不要随意给自己或他人贴标签。学习本课程是为了增加对人的内在理解，而非用于贴标签、相互攻击、批判或评价。随意贴标签不利于关系的修通。

### 什么是成人依恋访谈（AAI）？

接下来我们谈谈成人依恋访谈。这是一个非常有趣的访谈量表，现有简化版约20个条目。我们无法在此列出全部，但会分享几个条目，感受AAI如何进行施测。

AAI是**Adult Attachment Interview**（成人依恋访谈）的英文首字母缩写。它主要用于向父母或成人提问，让他们回想和反思自己与父母的关系，包括对丧失、拒绝和分离的体验。AAI最初设计是为了出其不意地捕获潜意识，可视为从特定角度为依恋系统提供情报。玛丽·梅恩设计它，主要是为了从成年人中提取其依恋品质。随后的更多研究搭建了从婴幼儿到成人的关联桥梁。

该访谈已被证明是评估成年期依恋的有效工具。与“陌生人情境试验”一样，它并非凭空设想，而是经过后续大量实验验证有效。对于归纳、总结或理解内在心理状态，AAI具有同等价值。

### 体验成人依恋访谈（AAI）题目

下面我们来感受几个AAI题目。如果你有兴趣，可以拿出纸笔，在听到题目时思考你会如何作答。想象一下，安全型、矛盾型或回避型的人可能会如何回答？如果你是临床工作者，也可以想象你的来访者会如何回答。

以下是几个示例题目：

1.  **首先，我想让你试着形容，当你还是孩子的时候你和父母的关系如何？从你能记得的最早时期开始说。**
    这个题目试图了解我们最早能从何时回忆起与父母的关联。在临床工作中，我们发现对于早年受损较严重、依恋不安全的人来说，他们可能会失去早先的记忆，很多人记不清六七岁甚至更早的事情。但从大脑结构看，即使记不清具体情境，杏仁核会记得那些体验。随着海马体发展和大脑皮层成熟，通常一个人至少能记得三四岁左右的事情。此时，我们可以去感受这个人对童年的体验和感受，以及他与父母的关系。想象一下，回避型的人可能简单回答“好”，但缺少后续内容；如果说“不好”，也可能缺乏支撑证据。

2.  **你能用五个形容词或短语来形容你和父母的关系吗？是什么样的记忆体验让你选择用这些词语呢？**
    你可能会用“紧密”、“开心”、“愉悦”、“痛苦”等形容词来描述内在状态。但注意，每个用词后面应有故事支撑，这些故事与你的记忆相连。如果你有“痛苦”的经验，那么与什么事件有关？这时我们可以观察一个人能否将内在情绪与事件本身连接，这是评判其是否具备**心智化**能力的标准之一。心智化意味着我们能描述自己的感受（从身体感受到情绪），并能连接事件与这些感受的关联，例如：“当那时我被批评的时候，我感觉心很凉，感到非常悲伤、委屈、愤怒。”更好的心智化能力还能连接双方的心理状态，例如在夫妻咨询中，识别出“当我越逼迫你，你就越后退；而你越后退，我就越生气”这样的负性循环。这有助于我们观察一个人能否理解情境与其内在的关联。

3.  **在童年时，当你痛苦的时候，你会做什么？然后会发生什么？**
    我们想知道一个人在痛苦时能得到何种支持。父母能理解孩子的痛苦吗？当孩子考得不好时，父母是批评指责，还是能想到孩子心里已经很难过，并及时给予安抚？大家可以回想与自己经历相关的事件。

4.  **在童年时，你觉得你被父母拒绝过吗？之后会发生什么？你父母意识到他们拒绝了你吗？**
    相信每个人都曾被拒绝过。例如，孩子想买玩具但父母不买。这个时刻很考验父母：孩子撒泼打滚，父母是批评大吼、转身离去，还是能理解孩子的内在情绪，给予反馈和安抚，直到情绪释放？亲密关系中也是如此：当我们有需求时，对方能否明白？当需求无法实现、需要拒绝时，你能否明白自己在拒绝别人？拒绝之后你会做什么？父母是否知道他们的拒绝带给孩子什么影响？

5.  **你的父母威胁过你吗？比如为了管教或者开玩笑。**
    这种情形特别多，例如“把你送给别人”、“再不乖就不要你了”、“大灰狼会来找你”等。不论父母是出于恐吓让你听话，还是开玩笑，你还能回忆起那些事件本身吗？我们可以观察自己是否感到安全，即父母是否更多地在心里支持我们，而非恐吓我们。

6.  **你现在和你的父母关系如何？**
    如果之前的问题都在询问一个人与父母的过去关系，那么这个问题问的是现在。但如果一个人早先描述父母对待他的方式非常糟糕，现在却说“很好”，我们可能会发现他**缺少一致性**，并感到困惑，不理解“好”在什么地方。

时间关系，不一一列举AAI的所有项目。但通过以上六个题目，大家可能多多少少了解了成人依恋访谈在做什么：它如何连接我们内在的潜意识（出其不意地捕获潜意识），以及如何为我们的依恋提供情报。

### 依恋的代际传承

从以上许多例子中，我们知道依恋关系具有**代际传承**性。这里稍微复习一下玛丽·梅恩的两个重大发现：
1.  婴儿在“陌生人情境试验”中的表现，与其儿童期的心理内在结构有关联性。
2.  依恋模式存在代际传承的关联。

父母和孩子会“共谋”，保留那些未解决的、僵化的心理空间和行为模式。强大的内在连接会使得父母（“邀请”自己的孩子做出与他们相同的行为。例如，如果父母曾是讨好型（因害怕被抛弃或忽略而压抑真实需求和情感如愤怒、委屈），他们成年后，也可能让他们的孩子形成类似的心理内在。这就是内在的**代际传递**。

鲍尔比曾说：“被细心呵护的孩子，最有可能成长为稳定、独立并信任他人的人。”要知道，信任是心理发展的基础。在埃里克森的人格发展八阶段理论中，第一个阶段就是建立**基本信任感**，并发展出**希望的品质**。如果第一阶段受损，人一生都很难构建真正的安全和希望。许多心理问题严重的人，几乎对生活失去信心、没有希望，也很难对他人建立信任感。

如何印证这句话？明尼苏达的苏劳夫（Sroufe）对于依恋的追踪调查，为我们提供了有力证据，证明了鲍尔比的观点，也让研究者对“陌生人情境试验”的成果深信不疑。

苏劳夫的研究：
1.  **第一组研究**：针对一些中产阶级的孩子，通过他们完成任务的反应方式了解其依恋类型。研究发现，安全型的孩子在**认知上发展更好**，**更容易探索和学习**，遇到挫折时**更能忍受**。而不安全型的孩子遇到困难时**非常焦躁**，**很难与他人合作**。
2.  **第二组研究**：更好地印证了代际传承。研究者扩展了范围，研究妈妈与孩子之间的代际关联。他们对一些平均年龄20多岁的年轻准妈妈进行了长达20年的追踪研究。这些母亲大多非常年轻、单身、意外怀孕，缺乏家人朋友照料，收入低，教育程度不高（有些高中辍学）。研究开始时，确认她们身体健康，无其他身心疾病。产后，要求护工对其照料婴儿的用心程度做出评判。随后，对这些母亲进行全方位评估（心理施测、访谈、问卷），以了解母亲与婴儿的关系。

研究很快见到成果：早先被护工反映对婴儿需求**不用心、不敏感**的妈妈，她们的孩子很容易被预测为**不安全型**依恋。这表明，依恋似乎与一个人的**先天气质或条件关系不大**（这些母亲身体都健康），而更多取决于**母亲照料孩子的品质**。因此，正如鲍尔比所言，婴儿出生后的气质类型（易于安抚型或难以安抚型）并非最关键因素，最关键的是**母婴之间的关系**和**互动品质**。这也呼应了我们最初讲到的：寻求与母亲的第一段关系品质，会反映出一生中的关系品质。

**补充观点**：在此也提一下我个人的一些看法。更多研究发现，在母婴互动中，母亲会**大量投射自己的内在感受**给婴儿。如果一个婴儿易于安抚，抚育过程可能相对顺利；但如果婴儿难以安抚（具有某些先天气质特质），母亲会如何将自己的想象和看法投射给孩子，这其实会影响照料品质。因此，理解不同的婴儿，也需要母亲有足够的**心智化能力**去观察和反思婴儿的内在世界。过于绝对的观点未必合适。

这个问题留给大家思考：在你们的感受和想象中，依恋的品质是否与我们**先天的气质有关联**？我们的实验模型证明很多时候没有关联。但如果我们从内在心理学，特别是精神动力学的角度（从内摄、投射及投射认同的角度）思考，也许会发现，不同的婴儿从出生开始，也许在与母婴互动的关联和关系上也会有些不一样。就像一个妈妈有两个孩子，一个好带一个不好带，妈妈可能会有很多评价（例如，好带的孩子像令人敬佩的父亲；爱哭闹的孩子像令人讨厌的弟弟）。我们会受到内部关系模式的影响。

### 总结

本节课中，我们一起学习了：
1.  婴儿期的四种主要依恋类型（安全型、回避型、矛盾型、混乱型）如何对应发展成成人期的依恋心理状态（安全型、冷漠型、迷恋型、未解决/混乱型）。
2.  评估成人依恋的重要工具——**成人依恋访谈（AAI）** 的基本原理和示例题目。
3.  依恋模式具有**代际传承**的特点，父母照料孩子的品质是影响孩子依恋安全性的关键因素，但婴儿的先天特质与母亲的投射互动也可能产生复杂影响。
4.  最重要的是，学习依恋理论旨在**增进对人的理解**，切忌用于随意**贴标签**，以免对关系造成损害。

理解这些模式，能帮助我们更深入地洞察自己与他人在亲密关系中的行为与情感根源。
