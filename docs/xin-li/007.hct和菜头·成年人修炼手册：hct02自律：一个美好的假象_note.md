# 007.hct和菜头·成年人修炼手册：hct02自律：一个美好的假象_note

在本节课中，我们将要学习“自律”这一概念的真相。我们将探讨自律为何常被误解，并通过一个真实的减重故事，揭示达成目标背后的真正动力——内心的自由，而非自我对抗。

## 概述

自律是一个被广泛使用，却常被误解的词汇。它常被用来评判他人或进行自我标榜。本节将解析自律的本质，并论证真正的、可持续的改变并非源于痛苦的自我约束。

上一节我们概述了自律的普遍误解，本节中我们来看看一个具体的个人经历，它如何挑战了我们对自律的传统认知。

## 一个关于体重的故事

我从小学三年级开始，体重便显著增加。在三十多年的时间里，我一直保持着超重的体型。

唯一幸运的是，我的外形在他人看来尚属“可爱”。因此，许多朋友都喜欢揉我的肚子。

但我也深知社会对肥胖者的普遍看法。有一个经典的说法是：**如果一个人连自己的身材都无法管理好，那么他还能做什么？**

这句话暗示所有肥胖者都因缺乏自律和自我放纵而失败，同时也暗含了说话者的自我夸耀。

2020年10月13日，我的体重达到了106.2公斤。从那天起，我决定控制体重。

在质疑声中，我研究了各种减肥方法，分析总结后发表了10篇相关研究。通过这个过程，我找到了适合自己的方式。

三个月零两周后，我的体重降至88.4公斤。我用16周时间，似乎赢得了“自律”的美名。

但请你先别忙着鼓掌，我的话还没有说完。

## 自律的假象

在这16周里，我压根没想过“自律”这件事。相反，我认为如果我真想着要自律，很可能早就放弃了。

原因在于，自律天然蕴含着**自我对抗**的意味。减肥时，你的“自我”想吃更多、想多休息，甚至你进化了数十万年的身体，也会本能地抵抗体重下降，视其为生存威胁。

要与如此强大的力量对抗，需要何等强度的自律？维持这种自律又需要什么？强大的意志力吗？而强大的意志力难道不需要肉体的能量支持吗？

简单地说，我没有选择与自己的身体和本能对抗。

既然我爱吃，我就改善食谱，寻找既爱吃又不易增脂的食物。既然我吃得多，我就减少每日餐数，这样无需每顿计算卡路里，但总热量并未超过以往。

因此，整个十六周我都是开心地吃，自然地减重。等到出现成效后，为了加快速度，我才自觉主动地增加运动。其实不运动也无非是进度快慢之别。

那么，请你告诉我，哪里有“自律”？我勉强自己顿顿吃草了吗？我勉强自己每天锻炼了吗？我勉强自己饿着肚子喝温开水了吗？实话告诉你，过去这些事我全都做过，最长的坚持过一年以上，但没有一样能持续下来。

整个过程最荒谬的一点是：当我毫无“自律”却成功减重时，大家夸我自律；而当我非常“自律”（痛苦坚持）却减不下体重时，人们认为我在自我放纵。

这让我对“自律”产生了极大怀疑。人们似乎是以**成败论英雄**。没有成效时，无论你多自律，人们不承认；有成效时，无论你是否真自律，人们都硬要说你自律。

这样看来，人们其实无法判断自律与否。当他们谈论自律时，是在谈论一件自己根本不懂的事。但这不妨碍他们用这两个字来评判你，让你产生愧疚，真心相信问题在于自己的“不自律”。

## 什么是真正的自律？

那么，什么是自律呢？

自律不是自我约束和自我压抑。自律并不真实存在。**自律只是一个人因为内心得到自由，而对外显现出来的样子。**

也就是说，所谓的“自律”只是看起来符合大众的想象罢了。

一个人能做到他人眼中的“自律”，原因在于他**内心自由**。

1.  **意志上的自由**：我决定减重，并非迫于社会压力、外界评论或亲友恳求，而是我自己想要去做，并非非做不可。
2.  **实质上的自由**：为此我做了研究，分析各种方法，学习人体能量代谢，找到了适合自己身体的方法。我有知识储备，对可能的变化心里有数，只是执行验证理论，不会迷惑怀疑。
3.  **行动上的自由**：在过程中，我可以随时选择放弃或继续，不会因这个选择产生压力。我选择继续，只是因为验证理论有效，肉眼可见体重下降、腰围缩小。

因此，我在意志、实质和行动上都是自由的，每一步都有选择权。事情变得很简单：我之所以那么做，只是因为**我可以这么做**。

我不需要外在压力、社会评价或自我谴责来产生“自律”，再用它去回应外界。当一个人内心自由时，只要他有一个目标，其内在力量就足以驱使他去达成。

就像我减肥的过程：我知道我要什么，也知道如何能做到，那么这就全然是我自己的事。我在其中感受到纯粹快乐，自然沉浸其中。于是，在外人看来，这就是自律的表现和坚持。

要知道，所有需要“坚持”的事情最后都会失败。所有需要“自律”的事情最后也都会被放弃。

中学时代，我曾自己编写物理学教案。这不是靠坚持或自律能做到的，只是因为我复习完了，在继续做题和编写教案之间，我觉得后者更有趣，于是从一页纸变成了一本小册子。

这看起来是我学习“自律”，但我知道这与自律无关。只因为当时我天真地想：如果想拿高分，对教材的掌握应该超过老师才行。虽然最终没超过老师，但已远超过去的自己。

## 总结与行动指南

本节课中我们一起学习了“自律”的真相。它并非痛苦的自我对抗，而是内心自由后的自然外显。

如果你还在为“自律”这个概念而自我折磨，是时候放下它了。比自律更重要的是寻找你内心的自由。

以下是寻找内心自由的行动指南：

*   **源自个人意志**：你的任何决定和行为，都应源自你的个人意志，而非外界或他人。
*   **寻找内在动力**：找到你做一件事时，内心能感受到快乐或意义的部分。
*   **放弃自我对抗**：与其强迫自己，不如寻找与自身欲望和特性合作的方法。

当你在心中全无“自律”之念，在别人眼里却满是“自律”时，那才是真正的自律。于是，才能以不坚持而坚持，以不追求成果而拥有成果。在达成目标时，也才有发自内心真正的快乐，而不是觉得“终于还完债了”。

最后，我希望你能找到自己内心的自由，尽早知道自己真正想要什么，才会有真正的人生改变和可持续的行动。如果你想不明白，那么迟早别人就会强塞给你一个目标——以自律或其他什么名义。
