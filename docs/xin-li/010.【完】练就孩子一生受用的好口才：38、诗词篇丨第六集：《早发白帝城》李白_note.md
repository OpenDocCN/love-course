# 课程010：诗词篇丨第六集《早发白帝城》李白 📜

在本节课中，我们将学习唐代诗人李白的经典诗作《早发白帝城》。我们将了解这首诗的创作背景，掌握朗读时应把握的核心情绪，并学习如何将情感融入诗歌的表达中。

## 诗歌背景与故事

上一节我们介绍了学习诗歌需要了解其背景，本节中我们来看看《早发白帝城》背后的故事。

这首诗的创作背景，可以概括为政治犯李白被大赦的故事。李白不仅是著名诗人，也曾是唐朝官员。他卷入了唐朝一次重大的政治事件——安史之乱，并因此受到牵连获罪。

当时李白被判以重刑，后经人求情，改为流放至偏远地区夜郎。在他途经四川，乘船行至白帝城时，传来了皇帝大赦天下的消息。这意味着他重获自由，无需继续流放。

## 核心情感分析

了解了故事背景后，我们来看看朗读这首诗时应把握的核心情感。

李白在失而复得自由的心情下，写下了《早发白帝城》。因此，朗读的基调应是**极度欢快、兴奋与畅快淋漓**的。这种情感类似于遭遇重大挫折后，突然迎来巨大转机时的惊喜与解脱。

为了帮助理解，可以设想一个类似情境：假如家人突遭贬职又旋即官复原职，其心情从低谷瞬间跃至顶峰，那种豁然开朗、喜悦至极的感受便是本诗的情感内核。

## 诗歌朗读与情绪把握

以下是结合具体诗句，分析如何通过语言传递情感：

**朝辞白帝彩云间**：清晨从白帝城出发，诗人眼中连云雾都成了彩云。这反映了其心情的明媚与喜悦。
**千里江陵一日还**：千里之遥一日即达，这并非写实，而是夸张地表达诗人渴望重返自由世界的急切与欢畅心情。
**两岸猿声啼不住**：连两岸猿猴的啼叫声，在诗人听来都显得悦耳动听，衬托出其内心的愉悦。
**轻舟已过万重山**：轻快的小舟已越过重重山峦。这句生动体现了诗人顺流而下、心情畅快淋漓的状态。

掌握诗歌的创作背景与核心情感后，朗读时便不必过分纠结于个别字词的高低轻重。只要整体情绪基调正确，在这个情感空间内的表达便是自由的。

## 课程总结

本节课中，我们一起学习了李白《早发白帝城》的创作背景与核心情感。我们了解到，这首诗诞生于诗人突获大赦、重获自由的特定时刻，因此朗读时应贯穿**惊喜、欢畅与解脱**的明朗情绪。请结合这种情感理解，多加练习这首诗的朗读。
