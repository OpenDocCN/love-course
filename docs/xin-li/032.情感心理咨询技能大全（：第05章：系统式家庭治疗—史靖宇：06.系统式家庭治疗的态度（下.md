# 情感心理咨询技能大全（完结）032：第05章 系统式家庭治疗—史靖宇：06. 系统式家庭治疗的态度（下）🧠

在本节课中，我们将继续深入学习系统式家庭治疗的核心工作态度。上一节我们介绍了部分基础态度，本节中我们将重点探讨如何通过拓展可能性、保持非专家姿态、运用假设、循环思维、保持中立与好奇以及资源取向等态度，与家庭进行有效工作。这些态度是治疗师与家庭建立联结、促成改变的重要基石。

## 拓展可能性 🌈

系统式家庭治疗师的首要态度之一是拓展可能性。家庭常常带着对问题的固定描述和担忧前来咨询。治疗师一方面需要理解他们当前的困境，但另一方面，不应过快接受他们对问题的单一描述。治疗师的工作是启发家庭，帮助他们寻找对问题的新的看法，扩大描述问题的空间，从而发现新的可能性。

以下是拓展可能性的一个经典案例，它源于催眠大师米尔顿·埃瑞克森对一位抑郁症年轻女性的治疗：

1.  **来访者描述**：一位二十多岁的女性感到绝望，认为自己形象、性格都很糟糕，没有组建幸福家庭的希望，甚至失去了活下去的勇气。她给治疗师三个月时间，声称若无改变就将结束生命。
2.  **挑战描述**：埃瑞克森没有接受她悲惨的故事版本，而是巧妙地引导她利用“死前”的时间进行尝试。
3.  **扩展行动**：他建议来访者取出存款，去美容院改变形象，并聘请服装搭配师选购新衣，将此视为“最后的飞跃”。
4.  **新的关注点**：当来访者第二次来访，形象焕然一新但仍抱怨自己门牙缝大时，埃瑞克森建议她练习从门牙缝中射水，并将其视为一项特殊本领。
5.  **互动与改变**：他进一步鼓励来访者向一位关注的男同事展示这个“本领”。来访者照做后，意外地开启了与男同事的互动，关系迅速升温，最终结婚生子。
6.  **态度核心**：这个故事提醒我们，不要过快认同来访者的悲惨叙事，而应致力于通过行动创造新的体验。只有获得了新的体验，来访者才有可能发展出新的故事描述，人生轨迹也可能因此改变。

## 非专家姿态 👥

第二个关键态度是，治疗师不将自己视为解决问题的专家，而是将家庭和来访者本身视为解决自身问题的专家。治疗师不替家庭做决定，也不预设家庭发展的“正确”方向。我们摒弃“对错”、“好坏”、“应该”等评判，完全尊重家庭和来访者自身的决定。

以下是这种态度的具体体现：

*   **不替家庭抉择**：例如，面对拒学的孩子，是努力返校还是选择其他路径？面对冲突激烈的夫妻，是分开还是继续生活？治疗师不提供答案。
*   **尊重与探索**：治疗师并非听之任之，而是尊重家庭的选择权，并在他们探索选择的过程中，帮助他们理解究竟发生了什么。
*   **家庭是专家**：最终，由家庭自己通过探索，做出最适合他们的决定。治疗师相信，只有家庭自己才知道什么对他们最有帮助。

## 运用假设 🧩

“假设”既是重要的治疗态度，也是一项核心技术。治疗师提供的所有信息、描述和评估，都应被视为一种假设或观点，而非绝对真理。这么做的目的是为家庭提供新的信息，拓展他们的视角，而不是为了找到“正确”的答案或做出诊断。

以下是运用假设的一个案例：

*   **问题呈现**：一位中年女性因总忘记关衣柜门而与丈夫频繁激烈争吵，深感苦恼。
*   **提出假设**：治疗师给出了一个意想不到的假设：建议来访者告诉丈夫，治疗师认为她“不关衣柜门”的行为，是无意识中在向丈夫发出“共度美好夜晚”的邀请。
*   **效果**：来访者和丈夫都觉得此假设不可思议，但它提供了一个全新的理解角度。此后，丈夫不再因此争吵，而是主动关上柜门，夫妻互动模式发生了积极变化。
*   **态度核心**：我们并非告知家庭“真相”，而是通过提供多样化的、新奇的假设，挑战他们固有的观念，引发思考，从而松动僵化的互动模式。

## 循环思维与互动视角 🔄

系统式家庭治疗强调循环的思维方式。当我们看到一位爱指责的丈夫或一位看似软弱的妻子时，我们需要思考：在家庭系统中，成员之间发生了什么样的相互影响，才促使某人表现出这样的行为？我们始终将个体行为放在互动的背景中来理解。

以下是循环思维的体现：

*   **相互塑造**：例如，一个“唠叨的妻子”可能与一个“回避的丈夫”共存，而丈夫的回避行为又可能加剧妻子的唠叨，形成一个相互影响的循环。
*   **不归咎个人**：有了这种循环的视角，我们就不会去指责某一个家庭成员，而是引导他们共同审视彼此的关系和互动中出现了什么问题。
*   **关注关系模式**：治疗的重点从“谁有问题”转向“你们的互动模式是怎样的”，从而为改变打开空间。

## 保持中立与好奇 ❓

在治疗中保持“中立”至关重要。这并非指没有立场，而是指治疗师可以持有多个不同的立场和观点，并能与每位家庭成员都建立良好的关系，同时不让任何人感到治疗师偏袒某一方。

以下是中立态度的具体内涵：

1.  **对家庭成员中立**：不因性别等因素同情或偏袒任何一方，对每位成员都给予同等的共情、友好与真诚。
2.  **对问题保持中立**：对于“上学与否”、“离婚与否”等问题，不预设好坏对错，保持开放。
3.  **对观点与解决方案中立**：尊重家庭认为的各种可能性及其合理性，不强行推荐自认为“正确”的解决方式。

与中立紧密相连的态度是“好奇”。好奇心通常在面对未知或感兴趣的事物时产生。始终保持对家庭的好奇，意味着治疗师不认为自己已掌握全部真相，而是带着探索未知的兴趣，与家庭一起拓展他们的故事。

以下是好奇心的作用：

*   **探索未知**：例如，不对“父亲经常不在家”的模式做价值判断，而是好奇这个家庭是如何形成并维持这一模式的。
*   **促进中立与尊重**：好奇心能帮助治疗师避免过早下结论，并通过不断提问与家庭工作，让家庭成员感受到被尊重。
*   **聚焦于人**：我们更重视“人”本身，而非他们所带来的固有观点。治疗师可以对那些僵化的描述“不屑一顾”，但绝对尊重和重视眼前这个人。许多治疗大师（如埃瑞克森邀请病人到家看电视，施佩琳邀请病人一同度假）都以实际行动体现了这种对人的重视。

## 资源取向 💎

资源取向是系统式家庭治疗的核心哲学与人性观。我们相信每个家庭、每个人、每个系统都已经具备了解决自身问题的全部资源，只是尚未察觉或激活。治疗师的任务是帮助家庭发现、看到并运用这些资源。

以下是资源取向的工作方式：

*   **关注资源**：治疗的重点不是寻找问题的根源，而是探索家庭已拥有但未被利用的优势、能力和可能性。
*   **理解问题的影响**：我们也会与家庭一起审视，当他们将精力过度聚焦于“问题”时，给生活带来了哪些影响（包括可能的“好处”和“限制”）。例如，母亲过度关注孩子的多动症，可能加强了夫妻合作，但也可能导致母亲辞职、社交减少。
*   **坚信改变能力**：无论问题多么棘手，治疗师始终坚信家庭有能力、有资源让生活变得不同。我们的工作是陪伴他们一起探索，将这些资源激活。

## 总结 📝

本节课中，我们一起深入学习了系统式家庭治疗下半部分的核心工作态度。我们探讨了如何通过**拓展可能性**来打破僵化的叙事，以**非专家姿态**尊重家庭的自主性，运用**假设**提供新视角，采用**循环思维**理解互动模式，始终保持**中立与好奇**以深入探索系统，并秉持**资源取向**的信念，激活家庭内在的改变力量。这些态度相互关联，共同构成了系统式家庭治疗师与家庭工作时的心法，是促成系统积极改变的重要基础。
