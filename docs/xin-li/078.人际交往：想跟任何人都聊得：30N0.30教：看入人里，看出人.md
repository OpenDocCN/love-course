# 078.人际交往：想跟任何人都聊得：30N0.30教：看入人里，看出人

在本节课中，我们将学习《沟通的艺术》这本经典著作的核心内容。这本书将帮助我们理解沟通的本质，掌握沟通前的准备、过程中的技巧以及如何处理人际关系。我们将通过简单明了的讲解，让初学者也能轻松掌握沟通的要领。

## 🧠 第一部分：看入人里 - 沟通前的准备工作

上一节我们介绍了课程的整体框架，本节中我们来看看沟通前需要做哪些准备工作。这些准备是有效沟通的基础，帮助我们更好地理解自己和他人。

### 1. 理解沟通的本质
沟通不仅仅是语言交流，非语言的方式如陪伴、身体接触等也在传递信息。例如，默默的陪伴传递了“你对我来说很重要”的信息。

### 2. 明确沟通的目的
沟通有两个主要目的：
*   **初级目的**：为了生存，满足基本需求。
*   **高级目的**：表达对彼此关系的重视。

### 3. 认识自我形象
了解别人如何看待你，直接影响沟通的方式。你可以定期询问亲友对你的看法，以获取更客观的自我认知。

### 4. 消除主观影响
使用“知觉检核”方法来客观理解他人，避免主观偏见。该方法分为三步：
    1.  描述你注意到的行为（客观事实）。
    2.  列出你对这个行为的至少两种解释。
    3.  请求对方澄清你的理解是否正确。

### 5. 管理个人情绪
情绪管理是沟通的关键。以下是管理情绪的步骤：
*   **认识情绪**：扩充情绪词汇，准确描述感受。
*   **辨别与评估情绪**：通过生理反应或行为变化识别情绪，并理性评估引发情绪的想法是否合理。
*   **改变想法**：质疑并调整引发负面情绪的非理性想法。

## 💬 第二部分：看出人外 - 沟通过程中的技巧

在做好了充分准备之后，本节我们将探讨在实际沟通过程中可以运用的具体方法和技巧。

### 1. 语言沟通：力求清晰具体
避免使用“尽快”、“可能”等模糊词汇。例如，与其说“尽快把文件给我”，不如说“今天下班前把文件给我”。

### 2. 非语言沟通：注意辅助信号
非语言沟通包括肢体语言、语调、触碰等，它没有固定语法规则，常作为语言的补充。但其解读可能不准确，需结合语境理解。

### 3. 积极倾听：给予有效反馈
倾听是沟通的重要组成部分。有效的倾听包含三个层次的反馈：
*   **简单回应**：如点头、说“嗯”。
*   **积极提问**：询问细节，验证理解。
*   **表达看法**：在理解对方后，分享自己的观点。

### 4. 营造友好氛围
创造良好的沟通氛围有两个方法：
*   **就事论事**：讨论焦点集中在解决问题上，而非评价个人。
*   **平等协商**：即使作为上级，也多用商量的语气提出建议。

### 5. 提出建设性批评
使用“行为-解释-感受-结果-意图”五步法则进行批评，可以避免直接指责引发的对抗。
    **批评公式示例**：描述行为 -> 提出解释 -> 表达感受 -> 说明结果 -> 陈述期望。

## 🤝 第三部分：看人之间 - 处理各种人际关系

掌握了沟通的技巧，我们还需要将其应用于具体的人际关系中。本节我们将学习如何处理亲密关系、应对冲突以及修复破损的关系。

### 1. 维护亲密关系
亲密关系（家人、朋友、爱人）同样需要沟通技巧。
*   **家人**：了解家庭的沟通模式（谈话取向或遵从取向），并努力与下一代建立更平等的沟通。
*   **朋友**：基于真实的社交需求和价值观选择朋友，无需刻意迎合。
*   **爱人**：可以通过五种行为增进感情：**表达肯定**、**共度有品质的时间**、**赠送礼物**、**服务行为**、**身体接触**。

### 2. 应对人际冲突
并非所有冲突都需要解决，应优先处理与关系密切的人和事相关的冲突。处理冲突的步骤如下：
    1.  确认自己的真实问题和需求。
    2.  与对方约定时间进行沟通。
    3.  在沟通中清晰描述问题，并理解对方的立场。
    4.  共同思考多种解决方案，并选出最合适的一个。
    5.  执行方案，并设立确认效果的量化标准和步骤。

### 3. 修补破损的关系
当关系出现裂痕时，可以运用一套道歉公式来尝试修复：
    **道歉公式**：表达后悔 -> 承担责任 -> 提出弥补 -> 真诚忏悔 -> 请求原谅。
需要注意的是，关系修复需要时间和持续的努力。
