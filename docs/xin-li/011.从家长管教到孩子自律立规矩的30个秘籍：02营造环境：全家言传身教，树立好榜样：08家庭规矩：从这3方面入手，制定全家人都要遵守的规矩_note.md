# 📘 课程011：从家长管教到孩子自律 立规矩的30个秘籍 - 02营造环境：全家言传身教，树立好榜样 - 08 家庭规矩：从这3方面入手，制定全家人都要遵守的规矩

在本节课中，我们将要学习如何通过制定全家人都需遵守的规矩，来营造一个言传身教的家庭环境，从而帮助孩子从被管教走向自律。

上一节我们介绍了营造环境的重要性，本节中我们来看看如何具体制定家庭规矩。家庭规矩不应只针对孩子，而应是全家共同遵守的行为准则。这能让孩子感受到公平，并更愿意主动遵循。

制定有效的家庭规矩，可以从以下三个核心方面入手。

以下是制定家庭规矩的三个具体方面：

1.  **生活作息与习惯**：规定全家人共同的作息时间与日常习惯。例如，设定统一的就寝时间、用餐时间，以及“玩具玩完后放回原处”等习惯。公式可以表示为：**家庭规矩 ⊇ {作息时间，日常习惯}**。
2.  **沟通与情绪管理**：建立家庭成员间的沟通规则。例如，“用平和语气表达不满”、“争吵后需冷静沟通解决”。这可以通过家庭会议来讨论和确立。
3.  **责任与家务分工**：明确每位家庭成员的责任，特别是家务的分担。根据年龄分配任务，如“孩子负责收拾自己的碗筷，父母负责做饭与清洁”。代码逻辑可简述为：`assign_chores(family_members) based on age_and_ability`。

通过从这三个方面共同制定规矩，能让规矩更全面、更易执行。全家人共同遵守这些规矩，是言传身教最直接的方式，为孩子树立了清晰的榜样。

本节课中我们一起学习了制定全家适用规矩的三个关键方面：生活作息、沟通方式与责任分工。当规矩成为全家人的共同承诺时，孩子便能在潜移默化中学会自律与合作。
