# 032.情感心理咨询技能大全（完结）：第03章：依恋理论-何雪娜：03.成人的依恋类型上_note

在本节课中，我们将要学习依恋理论在成人期的具体表现。我们将探讨婴儿期的依恋类型如何影响并关联到成人的依恋心理状态，并介绍评估成人依恋的重要工具——成人依恋访谈（AAI）。

## 课程概述

上一讲我们介绍了依恋的概念、起源、发展及其在理解亲密关系中的核心内涵。本节中，我们来看看成年期的依恋类型。除了婴儿期通过“陌生人情境试验”等方式评估依恋模式，成人的依恋类型同样可以通过玛丽·梅恩（Mary Main）的“成人依恋访谈”进行归类。下面，我们将探讨婴儿期的依恋类型与其成年后心理状态之间的关联。

## 安全型依恋的发展轨迹

首先，我们来看安全型婴儿。在“陌生人情境试验”中，安全型婴儿在与照料者分离时显得自信。他们能在陌生环境中自由玩耍，分离时会想念父母，与陌生人相处时不会过度排斥，但更喜爱与父母在一起。与父母重聚时，他们会表现出愉悦和欢迎。在预测到第二次分离时，他们可能会难过，但在得到足够安抚后，仍能与陌生人互动。重聚时，虽有悲伤，但因对抚养者有信心，他们能平静下来继续探索。

他们成年后会成为安全型成人。与安全型成人恋爱，关系紧密且保有各自空间。在“成人依恋访谈”中，他们表现出言行的一致性，能客观描述情境，评价不偏激，描述事件有因有果、逻辑清晰。他们肯定依恋的价值，理解早年与父母的稳固关系对成长至关重要。

由此可见，安全型婴儿与安全型成人之间存在显著相关性。

## 回避型依恋的发展轨迹

接下来，我们看一下回避型婴儿。在“陌生人情境试验”中，他们对与父母的分离和重聚都显得漠然，情绪反应很少。他们会主动回避父母，例如转身、不理睬或过度探索环境，似乎只在意玩具。他们忽视父母，分离时不表达难过，重聚时也没有亲近的肢体接触或愤怒表达。他们的注意力始终集中在玩具和环境上。

他们成长后，会成为冷漠型成人。在成人依恋的心理状态上，他们属于冷漠型，语言缺乏一致性。他们常用概述性、宽泛空洞的语言描述生活，例如“很好啊”，但缺乏具体事件和情感连接。其描述常前后矛盾，例如声称母亲“很好”，但后续描述中又提及母亲的体罚或忽略。在亲密关系中，也可能出现类似的语言与事实不一致的情况。

## 矛盾型与混乱型依恋的发展轨迹

以下是矛盾型婴儿的表现。在“陌生人情境试验”中，他们对分离异常警觉和敏感。分离前会紧紧拽住照料者，显得无助恐慌，无法在照料者离开后探索环境。他们时刻关注父母，难以专注于学习，这可能导致注意力不集中等行为问题。重聚时，他们表现出矛盾行为，如愤怒或无助哭泣，且抚养者的安抚很难让他们平静下来，因为他们似乎不相信照料者是稳定可靠的。哭泣后，他们可能显得非常无助。

他们成年后的依恋心理状态是迷恋型。这类人在描述过去时，信息可能冗杂、缺乏逻辑，让人难以理解核心。他们对依恋的需求占据了整个心智状态，可能随时表达愤怒或无助，语言冗长、语法混乱，有时使用古怪的言语形容内心感受，令倾听者感到混乱。

混乱型是更糟糕的状态。我们在第一讲提过，混乱型依恋意味着内在自我的破碎。在“陌生人情境试验”中，他们在父母在场时就开始表现出混乱和迷茫，例如呆住、神情恍惚、看向远方，或突然蜷缩在地。他们可能表现出一些原始的、令人费解的行为。当父母抱住他们时，他们可能一边挣脱一边缠绕，行为非常矛盾。混乱型婴儿并非时刻如此，也可能兼有矛盾型和回避型的一些行为。

成年后，这类人的依恋心理状态称为未解决型或混乱型。他们有大量未解决的内心问题。在讨论虐待或丧失等主题时，他们的语言推理和逻辑混乱，并有令人吃惊的失误（例如描述已故亲戚“在表面上是活着的”）。他们可能陷入长时间沉默，或进行大段无意义的、如同念经般的歌功颂德，这可能预示其内在的精神病性部分。

## 重要提醒：避免随意贴标签

以上我们探讨了婴儿期依恋类型与成年后心理状态之间的关联。当然，实际情形可能更复杂，需要根据具体情境判断。**请切记，不要在生活随意给自己或他人贴标签**。学习本课程的目的是增进对人的内在理解，而非用于相互攻击、批判或评价。随意贴标签（例如在婚姻中对伴侣说“我看你就是冷漠型”）可能引发更多矛盾，不利于关系的修通。

## 成人依恋访谈（AAI）简介

接下来，我们谈谈“成人依恋访谈”。这是一个非常有趣的访谈工具，现有简化版约20个条目。AAI是Adult Attachment Interview的缩写，主要用于向成人提问，让他们回想和反思自己与父母的关系，包括对丧失、拒绝和分离的体验。其最初设计是为了出其不意地捕获潜意识，为理解依恋系统提供一个情报视角。该访谈已被证明是评估成年期依恋的有效工具，其价值如同“陌生人情境试验”之于婴儿。

下面，我们来感受几个AAI的题目。如果你有兴趣，可以准备纸笔，思考如果是安全型、矛盾型或回避型的人，可能会如何回答。临床工作者也可以想象来访者会如何作答。

以下是几个示例题目：

1.  **首先，我想让你试着形容，当你还是孩子的时候你和父母的关系如何？从你能记得的最早时期开始说。**
    这个题目试图了解一个人最早能回忆起与父母的关联。需注意，早年受损较严重的不安全型个体，可能失去早年的记忆。

2.  **你能用五个形容词或短语来形容你和父母的关系吗？是什么样的记忆体验让你选择用这些词语呢？**
    回答时，每个形容词背后都应有具体的故事和记忆支撑。这能反映一个人能否将内在情绪与事件本身连接，是评判其“心智化”能力的标准之一。

3.  **在童年时当你痛苦的时候，你会做什么？然后会发生什么？**
    这个问题探索一个人在痛苦时能否得到支持，以及父母能否理解并安抚孩子的痛苦。

4.  **在童年时你觉得你被父母拒绝过吗？之后会发生什么？你父母意识到他们拒绝了你吗？**
    拒绝是常见体验。这个问题考察个体如何应对拒绝，以及父母是否意识到其拒绝行为对孩子的影响。

5.  **你的父母威胁过你吗？比如为了管教或者开玩笑。**
    威胁（如“再不乖就不要你了”）是否常见，以及个体是否感到安全。

6.  **你现在和你的父母关系如何？**
    这个问题连接过去与现在。如果一个人之前描述了父母的糟糕对待，现在却说关系“很好”，则可能缺乏一致性。

通过以上题目，大家可以大致了解AAI如何工作，以及它如何连接我们内在的潜意识，为理解依恋提供信息。

## 依恋的代际传承与研究证据

从以上例子可以看出，依恋关系具有代际传承的特点。玛丽·梅恩有两个重大发现：第一，婴儿在“陌生人情境试验”中的表现与儿童期的心理结构有关联；第二，发现了依恋的代际传承关联。父母和孩子会“共谋”，保留那些未解决的、僵化的心理和行为模式。强大的内在连接会使父母（无意识地）“邀请”孩子做出与自己相同的行为。

鲍尔比（Bowlby）曾说：“被细心呵护的孩子，最有可能成长为稳定、独立并信任他人的人。”信任是心理发展的基础。埃里克森人格发展八阶段理论的第一阶段，就是建立基本信任感，获得希望的品质。如果这一阶段受损，个体一生都难以构建真正的安全和希望。

明尼苏达大学的苏劳夫（Sroufe）的追踪研究为鲍尔比的观点提供了有力证据。他们的研究让人们对“陌生人情境试验”的成果深信不疑。

*   **第一项研究**：对中产阶级儿童实施任务，发现安全型儿童在认知上发展更好，更易于探索学习，更能忍受挫折；而不安全型儿童遇到困难时更焦躁，更难与他人合作。
*   **第二项研究**：更好地印证了代际传承。他们对一群平均20多岁、单身、意外怀孕、缺乏支持的年轻准妈妈进行了长达20年的追踪。研究发现，那些被护工评价为对婴儿需求不用心、不敏感的母亲，其孩子很容易被预测为不安全型。这表明，依恋品质与婴儿先天气质关系不大，更多取决于母亲照料孩子的品质，即母婴互动的质量。

当然，也有观点认为，婴儿的先天特质（如易于安抚或难以安抚）会影响母亲的投射和互动品质，从而间接影响依恋。这个问题留给大家思考：在你们的感受和想象中，依恋品质会与先天气质有关联吗？

## 课程总结

本节课中，我们一起学习了成人依恋类型的分类及其与婴儿期依恋的关联，包括安全型、回避型（成年后为冷漠型）、矛盾型（成年后为迷恋型）和混乱型（成年后为未解决型）的发展轨迹。我们强调了避免随意贴标签的重要性。接着，我们介绍了评估成人依恋的重要工具——成人依恋访谈（AAI），并通过示例题目了解了其工作方式。最后，我们探讨了依恋的代际传承现象，并介绍了相关研究证据。理解这些，有助于我们更深入地洞察自己与他人的关系模式。
