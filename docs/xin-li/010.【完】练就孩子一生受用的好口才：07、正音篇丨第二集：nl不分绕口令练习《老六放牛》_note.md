# 口才基础课 010：正音篇（二）n与l不分 🗣️

在本节课中，我们将学习如何准确区分和发音声母 **n** 和 **l**。这是许多南方地区朋友在说普通话时容易混淆的一对声母。我们将通过理解发音原理并进行针对性的绕口令练习，帮助你彻底攻克这个难题。

## 发音混淆的常见问题

许多人在发音时，经常将 **n** 和 **l** 混淆。例如，“拿来”会读成“拉来”，“连长”读成“年长”。这种混淆会导致表达不清，甚至闹出笑话。

以下是几个典型的错误发音例子：
*   “乌喇那拉氏” 被误读为 “乌啦啦拉四”。
*   “南汇” 被误读为 “兰汇”。
*   “鲶鱼” 被误读为 “连鱼”。

## n 和 l 的正确发音方法

上一节我们认识了问题，本节中我们来看看如何通过找准舌头位置来正确发音。**n** 和 **l** 的发音区别主要有两点：**舌头形态**和**气流路径**。

### 声母 n 的发音要领

发音 **n** 时，关键在于使用鼻腔共鸣。

1.  **舌头位置**：舌面放平，舌尖抵住上齿龈（上牙齿后背）。
2.  **气流路径**：发音前，气流先通过鼻腔，产生鼻音。可以先用鼻子发“嗯”的音，然后保持口型发出 **n**。
    > 发音公式：**嗯 -> n**

### 声母 l 的发音要领

发音 **l** 时，气流完全从口腔通过，动作轻巧。

1.  **舌头位置**：舌尖翘起，轻轻抵住上齿龈。
2.  **气流路径**：气流直接从口腔两侧流出，不经过鼻子。想象用舌尖像小棍一样轻巧地弹开。
    > 发音口诀：**舌尖轻弹，气流从口腔过**

### 核心区别总结

两者的核心区别可以归纳如下：
*   **n**：舌面平，**有鼻音**（气流走鼻腔）。
*   **l**：舌尖翘，**无鼻音**（气流走口腔）。

## 绕口令巩固练习：《老六放牛》

理解了理论后，我们需要通过大量练习来重塑肌肉记忆。以下是专为区分 **n** 和 **l** 设计的绕口令。

**请注意**：绕口令的目的是“读准”，而不是“读快”。务必先确保每个字的发音都正确无误。

> **《老六放牛》**
>
> 柳林镇，有个六号楼，
> 刘老六住在六号楼。
> 有一天，来了牛老六，牵了六只猴；
> 来了侯老六，拉了六头牛；
> 来了仇老六，提了六篓油；
> 来了油老六，背了六匹绸。
> 牛老六、侯老六、仇老六、油老六，
> 住上刘老六的六号楼。

**练习要求**：建议将此绕口令练习50-100遍，直到能够不假思索地准确发出每一个 **n** 和 **l**。

## 课程总结

本节课我们一起学习了如何区分声母 **n** 和 **l**。我们首先分析了混淆的常见情况，然后深入讲解了两者发音在**舌头形态**（平与翘）和**气流路径**（鼻腔与口腔）上的根本区别。最后，我们通过绕口令《老六放牛》进行了巩固练习。

攻克发音难点需要耐心和重复。只要你坚持用正确的方法练习，就一定能摆脱“n、l不分”的困扰，让你的普通话更加标准、动听。
