# 061.看理想·：像辩手一样思考：21.论证强度与：有人说你吃了两碗

在本节课中，我们将要学习论证的两个核心概念：**论证强度**与**论证责任**。我们将探讨不同类型的论证如何影响其说服力，以及在需要做出判断的场合，证明的责任应当如何分配。

***

## 回顾：演绎论证与归纳论证

上一节我们介绍了论证的基本分类。从逻辑学上讲，论证主要分为演绎论证和归纳论证。

演绎论证是一种逻辑推演。只要前提为真且逻辑形式有效，其结论就是100%保真的。结论中的信息并未超出前提给出的范围。其公式可以表示为：
**如果 P 且 (P → Q)，则 Q**。

例如：
*   已知：三班的班主任是潘老师。
*   已知：阿旁是三班的学生。
*   结论：阿旁的班主任是潘老师。

数学证明和科学中的假设-演绎模型也大量运用了演绎论证。

归纳论证则不同，它从已知经验出发，推论出超越已知信息的结论，因此结论是或然性的，无法保证100%为真。例如，“人性本善”或“寒门再难出贵子”这类全称判断，因为我们无法观测到每一个个体。

上节课我们介绍了归纳论证的几种主要方法。

以下是归纳论证的主要方法：
1.  **简单枚举法与科学归纳法**：简单枚举法仅适用于范围较小的结论；科学归纳法则结合了观察与科学原理，试图阐明现象背后的机理。
2.  **排除归纳法（穆勒五法）**：这是一种寻找因果关系的简化方法，包括求同法、求异法、求同求异并用法、共变法和剩余法。
3.  **类比推理**：根据两个对象的相似性，推论它们在其他方面也可能相似。
4.  **统计归纳法**：通过数据统计和分析来得出结论。

我们所做的摆事实、举数据、讲机理、引用权威等，都是服务于这些方法的应用。

## 论证强度的标准：因“损益比”而异

我们已经多次提到，无论是演绎还是归纳论证（尤其是归纳论证），其结论都不一定是绝对真理，它们只是以不同的**论证强度**支持某个观点。

例如，在论证一个因果关系时，能套用所有“穆勒五法”的论证，其强度就高于只套用“求同法”的论证。在统计归纳中，样本越大、数据越有代表性，论证强度通常也越高。

那么，论证到什么强度才算是“足够”呢？答案是：**没有统一的标准，它因领域、因具体情况而异**。决定这个标准的核心原则是 **“损益比”** ，即决策可能带来的收益与风险（损失）之比。

本节中我们来看看不同领域对论证强度的具体要求。

以下是不同领域对论证强度要求的例子：
*   **辩论赛**：标准是“比对方更强即可获胜”。评审会比较双方证据的代表性、样本的贴近程度等，论证强度更高的一方赢得比赛。
*   **重大政策制定**（如死刑存废、计划生育）：要求极高的论证强度。因为决策影响深远且可能不可逆，常需要高质量的研究甚至先行试点。
*   **日常小决策**（如领导是否慰问低保户）：论证强度要求较低，无需重大研究，因为风险较小。
*   **创业试错**：标准更低，讲究“快速试错”。在信息有限时，行动本身能产生新数据，过度的谨慎论证反而可能导致错失机会。
*   **法律诉讼**：
    *   **刑事诉讼**：要求“排除一切合理怀疑”，论证强度极高。因为后果是剥夺人的自由、财产乃至生命等基本权利。只要存在一种合理的无罪解释且无法被证伪，就不能定罪。
    *   **民事诉讼**：标准是“优势证据”或“高度盖然性”，即哪一方的故事更有超过50%的可能性为真，哪一方就胜诉。因为其后果通常是解决私人间的财产或纠纷，影响相对较轻。
*   **科学研究与应用**：
    *   **药物审批**：通常需要长达十数年、耗资巨大的临床试验，论证强度要求极高。
    *   **紧急情况**（如新冠疫苗）：因需求的紧急性和严重性，世卫组织可批准“紧急使用”，在未全面了解所有潜在副作用前投入使用，这降低了论证强度的即时要求。
    *   **工程设计**（如抗震级别）：采用统计学上的“置信度”（如95%）。根据过去数据预测未来风险，永远无法达到100%确信。根据建筑的重要性（损益比），来调整所需的置信度标准。

## 网络舆论中的论证责任：“谁主张，谁举证”

上述思路同样适用于网络舆论。近期一些事件表明，当有人提出指控时，我们不应立即要求被指控者“自证清白”。

难道今天但凡有个人跳出来说“你吃了两碗粉”，我们就有责任剖开肚子证明自己只吃了一碗吗？难道泼脏水的人说啥是啥，都不给人澄清和反驳的机会吗？

由此引出两个关键问题：第一，我们能基于多大可能性的指控去对待一个人？第二，**举证责任在谁**？

在法治社会，对一个人采取强制措施（如立案、搜查、拘留）需要与其嫌疑程度相匹配的论证强度，遵循“比例原则”。不能仅凭捕风捉影的质疑，就将人当作罪人对待。网暴在某种程度上就是滥用“私刑”。

这里的核心法律原则是 **“无罪推定”** ：任何人在未经依法审判证明有罪之前，应被视为无罪。简单说：**我是无辜的，直到你在法庭上证明我有罪；你不能假定我有罪，直到我去证明我无辜。**

与“推定”配套的概念是 **“举证责任”** 。在需要判定胜负的场合（如法庭、辩论赛），它的分配至关重要。

以下是举证责任的几条核心原则：
1.  **谁主张，谁举证**：提出积极主张的一方负有举证责任。如果证据不足，事实真伪不明，则负有举证责任的一方承担不利后果（如败诉）。
2.  **举证责任转移**：当一方完成初步举证（达到如“优势证据”标准）后，举证责任转移给对方。如果对方不能提出充分反证，则承担不利后果。
3.  **举证责任倒置**：在某些特殊情况下（如医疗侵权诉讼），法律出于公平和查明真相的需要，规定由被告方对某些事实承担举证责任。
4.  **证有不证无**：证明一个“存在”的积极事实，通常比证明一个“不存在”的消极事实更容易。因此，举证责任通常分配给主张积极事实的一方。例如，主张“你还钱了”的人，应提供转账记录；而主张“你没还钱”的人，则很难穷尽所有证据来证明“没收到钱”。

在辩论赛中，一种策略就是争夺“推定利益”和界定“举证责任”。例如，在“应否废除通奸罪”的辩论中，正方可以主张：法律默认保护人的自由权，若要动用刑法剥夺此项权利（保留通奸罪），反方必须论证其“必要性”和“比例原则”。如此一来，举证责任就被划给了反方。

## 总结与思考

本节课中我们一起学习了论证强度与论证责任。

对于**论证强度**的要求没有统一标准，它取决于具体情境下的 **“损益比”** 。后果越严重、风险越大的决策，所需的论证强度就越高。

在需要判定胜负的场合，当证据不足、事实真伪不明时，**担负举证责任的一方通常会落败**，即对方享受“推定利益”。划定举证责任最常见的原则是 **“谁主张，谁举证”** 和 **“证有不证无”** 。一方完成其举证责任后，责任才会转移。这些原则在实践中的应用，需要裁判者（如法官、辩论评审）做出专业判断，并且这种判断本身也可以被讨论和质疑。
