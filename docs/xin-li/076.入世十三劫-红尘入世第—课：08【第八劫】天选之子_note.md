# 076.入世十三劫-红尘入世第—课：08【第八劫】天选之子_note

在本节课中，我们将要学习如何区分并处理生活中的“大概率”与“小概率”事件。理解这一核心概念，能帮助我们在复杂多变的世界中做出更明智的决策，从而提升个人成功的可能性。

## 人与人的本质区别

上一节我们介绍了“天选之子”需经历考验的宏观背景，本节中我们来看看具体是什么造成了人与人之间的巨大差异。这种差异不仅体现在外貌、阶层或智商上，更体现在思维方式和进化程度上。

许多人依然在用最底层的线性思维来理解世界。世界是一个由复杂多变量因素导致的结果，而非简单的因果关系。就像很多人认为：
*   他考上好大学，**肯定**是找关系了。
*   他能演电视剧，**肯定**是和导演睡了。

这种思维把世界想象得过于简单。如果世界真如此简单，为何你送了礼领导却不要？为何你做了同样的事却得不到相同的结果？问题在于，他们未能理解世界的复杂性。

## 世界的动态系统与决策

这个世界是一个庞大且变化多端的动态系统。在这个系统中，所有参与者都会根据外界刺激做出反应，导致有千万种不同的思维在活动。因此，在这个动态系统中确立清晰的因果关系是非常困难的。

我们在这种动态系统中做决策，无法获得一个确定性的回复。凡是承诺“听了课一定能挣钱”或“做了项目一定能成功”的，都是在简化一个复杂的决策过程。**决策**本身包含许多变量，我们所能做的，并非追求绝对的确定性，而是**在不确定性中，通过方法提高找到确定性的概率**。

人缺乏安全感，总渴望确定性，但世界本身并非如此。所谓“运气”，往往是“实力”的体现。排除买彩票等极端情况，大部分的“好运”是因为当事人选择了一个**大概率可以成功**的赛道或方向，并通过叠加多个有利因素（大概率事件）来增加成功的可能性。

## 核心心法：为大概率坚持，为小概率预备

这是本节课最核心的行动指南。我们需要将精力和资源集中在能产生最大影响的大概率事件上，同时为可能发生的负面小概率事件做好预案。

### 理解大概率与小概率

以购买房产为例：
*   **决定资产价值的八成（大概率）因素**是：**人口流入、城市发展潜力、货币**政策。这三个因素的叠加，基本决定了房产长期的保值增值能力。
*   **影响价值的两成（小概率）因素**是：具体位置、物业、品牌、装修、楼层等。这些是建立在上述大概率基础之上的“加分项”。

如果你的决策模型（对人口、城市、货币的判断）正确，那么你**大概率**能通过房产增值获利。如果大概率的基础错了，小概率的加分项就如同无根之木，资产可能“有价无市”。这引出了一个重要公式：

**资产 = 能变现的“泡沫”**
**泡沫 = 不能变现的“资产”**

两者互为因果，没有绝对。

### 应用场景举例

以下是这一心法在不同场景下的应用：

**1. 择偶与婚恋**
寻找合适伴侣时，应关注对方的核心价值（大概率），而非次要优点（小概率）。
*   **对女性而言，大概率价值**可能包括：性吸引力、情感价值、生育价值、生存资料提供能力。具备这些，就具备了八成以上的成功概率。
*   **对男性而言，大概率价值**核心是提供**生存价值与养育价值**，即财富、社会资源与安全感。这在进化上是基于女性怀孕期间需要持续生存保障的现实。身材、长相、才艺等属于小概率的加分项。

如果没有前面的大概率价值作为基础，后面的小概率加分项将无处附着。**健康是1，其他是0**。对于男性，再帅身材再好，也不构成让女性托付终身的充分条件。

**2. 个人发展与健康**
很多人本末倒置，过度追求外在（小概率）而忽视根本（大概率）。
*   **“本”**：是你的五脏六腑、气血运行，是生存和健康的基础。
*   **“末”**：是指甲、发型、局部肌肉等外在修饰。

天天健身却从不做有氧、不注重养生调理，可能导致“身材好了，身体却差了”，这是愚蠢的。我们应把主要精力放在调理内在健康（大概率影响生命质量）上，外在修饰是锦上添花。

**3. 创业与择业**
创业本身就是在不确定性中寻找规律，没有 guaranteed 的成功路径。因此，选择赛道比盲目努力更重要。
*   **应坚持的大概率**：选择处于上升期的行业、去信息与资源更丰富的一线城市、向有成功经验的人学习并灵活运用其方法。
*   **应放弃或调整的小概率**：在朋友圈做微商（红利已过）、在没有退路的情况下“为坚持而坚持”。如果方向错了，及时调整而非欺骗自己。

### 警惕概率认知误区

在运用概率思维时，需警惕以下两个常见误区：

**1. 独立事件误区**
某些事件每次发生都是独立的，前次结果不影响下次概率。
*   **例子（掷骰子）**：连续掷出5次“1”，第6次掷出“5”的概率依然是 **1/6**，不会增大。每一次掷骰都是**独立事件**。
*   **代码模拟**：
    ```python
    import random
    def roll_dice():
        return random.randint(1, 6) # 每次掷骰，结果概率均等
    ```

**2. 混淆相关性与因果性**
两件事同时或先后发生，未必有因果关系，可能只是相关。
*   **错误因果**：“吃了我的产品，病就好了”。可能不吃药病也会在3天后自愈，吃药只是伴随事件。
*   **忽略前置条件**：“游泳运动员身材好，所以学游泳能让你手长脚长”。真实情况是：手长脚长的人更容易被选入游泳队，而非游泳让人手长脚长。这是把**相关性**当成了**因果性**。

## 行动指南与总结

本节课我们一起学习了如何运用“大概率与小概率”的思维模型来指导决策。

**最后的行动建议如下：**

*   **做出成功决策**：人和人的差距，是一个个成功决策叠加的结果。要主动寻找并叠加大概率成功的事情，例如提升核心价值、进入优势赛道、链接高势能人群。
*   **预防小概率风险**：对于可能造成重大负面影响的**小概率事件**，必须预防。例如购买重疾险预防大病，进行安全性行为预防性病。不要抱有侥幸心理，因为一旦发生，对你就是100%。
*   **关于教育子女的启示**：多生育（如果条件允许）可以增加培养出优秀子女的**概率**，并能在子女间引入良性竞争，避免因“唯一继承”而产生的懈怠。

**本节课重点总结：**
**为大概率的事情坚持做，为小概率的事情预备着。** 把握住能决定八成的核心要素，同时警惕那可能毁掉一切的二分风险，方能在红尘世中稳健前行。
