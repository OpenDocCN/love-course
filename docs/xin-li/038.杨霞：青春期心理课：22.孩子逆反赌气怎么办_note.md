# 青春期心理课 038：🔄 孩子逆反赌气怎么办

在本节课中，我们将学习当青春期孩子出现逆反、赌气不学习等行为时，家长应如何理解和应对。核心在于调整我们自身的态度与行为，通过“反向操作”的智慧方法，引导孩子做出积极选择。

## 理解逆反的动力

上一节我们介绍了青春期孩子的心理特点，本节中我们来看看孩子逆反行为背后的动力关系。孩子进入青春期后，常会表现出与家长“对着干”的倾向。

许多家长尚未意识到，这种“对着干”本身构成了一种动力关系。因为孩子意图反抗，所以家长需要主动走向另一个方向。

## 案例一：成功家长与“贪玩”孩子

以下是关于一对成功家长与其孩子案例的具体分析。

*   有一对家长，从小县城考入北大，毕业后创业，分别担任总经理和董事长，非常成功。
*   但他们上中学的孩子却非常厌学，且专门与父母对着干。
*   询问孩子为何不想像父母一样成功时，孩子回答：“成功就意味着不能玩吗？他俩从来没时间玩。如果成功就不玩，我趁现在不成功，赶紧先玩。”
*   家长此前一直不理解孩子为何贪玩、不爱学习，常感叹孩子不像自己当年那样刻苦。
*   给家长的建议是：有意识地在周末给自己放假，提高平日工作效率避免加班，并安排度假。
*   家长如此改变一年后，孩子也随之改变。
*   家长需改变过去每分秒都用于工作、并对孩子总是批评指责的状态。因为家长自身做得“够好”，所以更容易期望孩子“更好”。
*   采用“走到孩子反面”的方式，可以激发孩子成为一个“不一样的人”。

## 应对逆反的核心策略

因为青春期的孩子就想表现得与父母不同，父母勤奋他偏懒惰，父母好学他偏厌学。此时若进行过分的正面教育，他可能反而走向反面。

所以，我们应主动调整，策略性地“走到反面”。

## 案例二：应对“早恋”倾向

当孩子表达对异性的好感时，传统应对方式常适得其反。以下是具体分析。

*   例如，孩子对妈妈说“我喜欢班上的某某”。
*   以往家长可能会担心“早恋”，进而生气、担忧甚至限制孩子。
*   但结果往往相反。有孩子亲口说：家长越反对，我越觉得对方好；越说影响学习，我越不信；越让少接触，我越想多接触。
*   孩子的想法与成人不同，越强烈反对，他可能越想尝试。
*   因此，在此阶段需要“反着做”。

以下是具体操作方法。

1.  **保持镇静**：当孩子透露喜欢某人时，家长一定要保持冷静，避免激烈反应。
2.  **表示欣赏**：回应“好啊，他有什么特点吸引你？这么好的孩子，一定有些令人佩服的优点。”转而夸奖对方。
3.  **邀请与观察**：邀请该同学来家里玩或写作业，家长表现得殷勤热情。
4.  **引发孩子反思**：结果往往是，家长“越上杆子”，孩子自己反而会觉得“他没那么好”。当你站到他的对立面（即支持）时，他反而会比你更“通情达理”。

## 极端案例与原理深化

有时需要更极端的“反向操作”来化解危机。以下是相关案例。

*   一个高三孩子早恋，国庆节还与对象跑出去玩。家长非常生气。
*   在咨询中，孩子展示情书（七页纸写满“我爱你”），并声称不上大学了，要谈恋爱。
*   **应对策略**：咨询师表示支持，并“帮”他规划：不上大学，成年后租房、打工养活对方，并生两个爱情结晶，让爸爸抱孙子。家长配合点头。
*   **结果**：孩子立刻跳起来反对：“什么要孙子？我干嘛给你生孙子？我要上学！”
*   这个案例说明，当孩子逆反、赌气时，家长需冷静，分析并理解背后原因，真正尝试做孩子的“盟友”。

## 方法总结与实践要点

本节课中我们一起学习了应对孩子逆反赌气的核心方法。关键在于不直接对抗，而是通过理解和支持，让孩子自己做出正确选择。

以下是具体的行动要点。

*   **立场一致**：对孩子说“我站在你一边”，不直接对着干。像朋友一样支持他的想法（如抱怨老师、欣赏异性）。
*   **支持探索**：例如有高三孩子想放弃高考去网络写作，家长表示支持并认真帮他策划出版计划。
*   **引发自觉**：在具体规划写作字数、点子时，孩子自己提出：“我还是去读大学中文系吧，不然大学生活就没得写了。”
*   **保持智慧**：当孩子逆反、赌气时，家长首先要冷静思考。
*   **迂回引导**：运用迂回的智慧方法，而非与孩子置气、斗气或简单粗暴的正面说教。
*   **信任孩子**：相信孩子最终会做出正确的选择。

**总结**：面对青春期孩子的逆反，有效的方法不是正面交锋，而是通过理解、接纳甚至策略性的“反向支持”，化解对抗，让孩子在感到被尊重和理解后，自主走向积极的道路。
