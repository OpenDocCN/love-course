# 020.【完结】职场人际关系学：18堂处世绝学：第十六讲暴力沟通的四大陷阱，人

在本节课中，我们将要学习暴力沟通的四种常见形式。这些沟通方式是人际关系的致命伤，许多人职场发展受阻，往往源于无意中使用了这些方式，尤其是得罪了关键人物。我们将逐一剖析这四大陷阱，并提供避免的方法，以帮助大家在沟通中避免低级错误，维护和谐的人际关系。

## 第一种暴力沟通：道德评判 🏷️

上一节我们介绍了暴力沟通的总体危害，本节中我们来看看第一种具体形式：道德评判。这种方式是指以自我为中心，轻易对他人进行道德层面的指责，将行为问题上升为人品或道德问题。

以下是道德评判的具体表现与危害：

*   **表现**：当他人行为不符合自己的标准时，便指责对方“人品差”、“道德品质有问题”。这本质上是用自己的价值观去衡量和绑架他人。
*   **危害**：直接否定一个人的根本，伤害极大。一旦沟通上升到人格层面，矛盾极易激化且难以调和，因为对方感到面子和尊严受到了彻底伤害。
*   **核心逻辑**：`我的标准 == 道德标准`；`他人的行为 != 我的标准` => `他人道德有问题`。这是一个错误且有害的推论。

**案例说明**：
一位求职者面试销售岗位时，坦诚回答“想多赚钱以改善家庭生活”。这本是务实且积极的动机，尤其对于销售岗位而言，对收入的渴望是重要的驱动力。然而，一位面试官却因此评判其“道德品质有问题，太爱钱”，最终拒绝录用。该求职者后来在另一家公司凭借同样的动力成为销售冠军。这个案例表明，武断的道德评判不仅可能错失人才，评判者自身也可能因此声誉受损。

**过渡**：道德评判通过“贴道德标签”来伤害他人，而接下来要讲的第二种暴力沟通，则是通过“比较”来达到类似的效果。

## 第二种暴力沟通：进行比较 📊

除了直接进行道德指责，另一种常见的暴力沟通方式是习惯于将身边的人与他人进行比较。这等于公开地将你的朋友或家人划分等级。

以下是进行比较的危害：

*   **本质**：为了显示自己或达到某种教育目的，公开将对方与“更好”或“更差”的人进行比较。
*   **后果**：这种行为会对对方的自尊心造成巨大伤害。你无意中就得罪了人，导致他人对你产生反感和仇视，这对你未来的关系发展同样不利。
*   **关键点**：正如格林伯格所说：“如果真的想过上悲惨的生活，就去与他人比较。”比较剥夺了人的独特性和自我价值感。

**案例说明**：
在节目《少年说》中，一个女孩哭诉母亲总拿她与别的同学比较，从未看到她的努力。母亲却以“需要打击教育，防止骄傲”为由辩护。这种比较式的沟通，无论动机如何，都对孩子的自信心造成了“暴击”，是一种情感上的暴力。

**过渡**：比较是向外归因，伤害他人自尊。第三种暴力沟通则转向对内，表现为极力回避自身的责任。

## 第三种暴力沟通：回避责任 🤷

在人际关系中，推卸责任是一种典型的自私表现，也是重要的沟通陷阱。这种模式的特点是，将所有问题的责任都归咎于外部或他人。

以下是回避责任的表现与影响：

*   **常见句式**：使用“不得不”、“公司规定”、“领导说的”等表达，来规避自己的责任。例如：“这不是我的本意，是领导让我做的。”
*   **传递的信息**：这种沟通方式传递的信息是“我没有任何问题，都是别人的错”。这会使他人感到不被尊重，且不愿与你合作。
*   **核心原则**：必须认识到“我是一切的根源”。如果不能提升认知层次，改变抱怨和推诿的习惯，将会伤害身边所有的人。

**过渡**：回避责任是否认自己的能动性，而最后一种暴力沟通，则是用简单粗暴的定义，扼杀他人的可能性。

## 第四种暴力沟通：强加标签 🏷️

贴负面标签是生活中最常见、最不易察觉的语言暴力之一。它用静态、负面的词汇定义一个人或其行为，控制了沟通的氛围，并可能造成深远伤害。

以下是强加标签的表现与危害：

*   **典型场景**：对孩子说“你就是懒”、“你就是胆小”；对伴侣说“你就是没心没肺”。将一时的行为固化为永久的人格特质。
*   **巨大危害**：特别是对孩子，负面标签会产生“自我实现的预言”，强化你不希望看到的行为，并给孩子内心带来冲突和痛苦。例如，指责男孩“爱哭、软弱”，反而可能让他更不知道如何健康地宣泄情绪。
*   **正确认知**：行为不等于人。哭闹是一种情绪表达方式，本身没有对错。压抑情绪表达可能导致更大的心理问题。

**总结**：
本节课中我们一起学习了暴力沟通的四大陷阱：**道德评判**、**进行比较**、**回避责任**和**强加标签**。这四种沟通方式都会严重损害人际关系，是职场和生活中的致命伤。要建立和谐的人际关系，我们必须有意识地识别并戒除这些暴力沟通习惯，学会更客观、更负责任、更尊重他人的表达方式。记住，沟通的目标是连接和理解，而非评判与伤害。
