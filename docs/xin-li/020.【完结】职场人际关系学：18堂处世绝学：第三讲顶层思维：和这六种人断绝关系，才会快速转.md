# 020.【完结】职场人际关系学：18堂处世绝学：第三讲顶层思维：和这六种人断绝关系，才会快速转

在本节课中，我们将学习一种关键的顶层思维：识别并远离六种会阻碍你发展的人际关系类型。通过果断地与他们断绝往来，你可以更有效地提升个人价值，为事业和生活带来积极转变。

---

人际关系对我们的成功至关重要。与什么样的人交往，往往决定了我们自身的层次和发展方向。人到中年，更需要对过往进行深刻反思，总结成功经验与失败教训，尤其是在“人”的问题上。果断与以下六种人划清界限，是改变事业轨迹、避免陷入平庸的关键一步。

记住一个核心原则：**成功吸引成功，平庸吸引平庸**。你与谁为伍，最终很可能就会成为谁。

---

上一节我们强调了人际选择的重要性，本节中我们来看看具体需要远离的六种人。

以下是第一种需要远离的人：

1.  **只在借钱时才找你的人**
    这种人平时对你漠不关心，只在需要金钱援助时才会出现。一旦目的达成，便会立刻消失，毫无正常的人情往来。他们甚至将“借钱”视为一种生存手段。
    > **示例**：一位亲属首次小额借款后迅速归还，以建立“信用”，随即试图借取更大额度的款项，其真实目的是借后不还。这种人的行为模式是：`建立信任（小额快还）-> 提出大额请求 -> 借后消失`。

远离了利用你的“钱友”，接下来要警惕消耗你时间和精力的“玩伴”。

以下是第二种和第三种需要远离的人：

2.  **“蚂友”——总拉你打麻将的人**
    长期沉迷牌桌会使人上瘾，逐渐荒废事业。此类朋友应远离。
3.  **“酒友”——经常不醉不归的人**
    频繁的、无节制的饮酒不仅损害健康，影响家庭和睦，还会拖累事业进程。

除了消耗型的“玩伴”，那些消极度日、缺乏进取心的人同样需要警惕。

以下是第四种需要远离的人：

4.  **“闲友”——不思进取、闲混度日的人**
    这类人没有人生目标，安于现状，每天只知吃喝玩乐。与他们长期相处，你的斗志也会被消磨。**近墨者黑**，人的惰性很容易被环境影响。即使对方家境优渥可以挥霍，你也必须为自己的事业和未来负责。

比消极更危险的，是那些行走在危险边缘的人。

以下是第五种需要远离的人：

5.  **“烂友”——从事偏门、违法生意的人**
    这类人意志薄弱，可能因打击或诱惑而偏离正途。他们只顾眼前利益，缺乏道德与法律底线。与他们密切来往，一旦他们出事，你很可能会受到牵连。

最后，我们要警惕那些极端功利、只想索取不愿付出的人。

以下是第六种需要远离的人：

6.  **只想从你身上捞好处的人**
    这类人在职场或生活中，只在你对他有利用价值时出现。他们的核心逻辑是：`评估你的可利用价值 -> 提出请求 -> 利用完后消失`。当你需要帮助时，他们绝不会伸出援手。对于不擅拒绝的老实人而言，与这种人交往会吃大亏。

---

本节课中，我们一起学习了职场人际关系的顶层思维：**主动识别并远离六种消耗你、拖累你甚至危害你的人**。他们分别是：只在借钱时找你的人、总拉你打麻将的“蚂友”、不醉不归的“酒友”、不思进取的“闲友”、从事偏门的“烂友”，以及只想捞好处的功利者。

清理这些负面的人际关系，是为自己的成长扫清障碍、快速转运的重要一步。记住，优化你的社交圈，就是优化你的人生轨迹。
