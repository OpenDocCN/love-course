# 077.【完】打造个人影响力：第5节共情式沟通（下）丨行动：3步终结越聊越僵的恶性循环，让

## 概述

在本节课中，我们将学习如何将共情思维转化为具体行动。上节课我们探讨了共情是与生俱来的能力，以及合作思维的重要性。本节我们将深入三个具体步骤，帮助你停止沟通中的恶性循环，让他人理解并支持你。

---

## 第一步：停一停，放一放 🛑

上一节我们介绍了启动合作思维是共情的前提。本节中我们来看看如何在实际冲突中应用它。任何时刻，当你需要共情却感觉不到时，请按下内心的暂停键。

以下是具体做法：

*   **暂停惯性反应**：当冲突发生时，习惯性的评判和指责念头会涌现。例如，孩子因玩具在公共场合哭闹，你可能会感到丢脸、责怪他人。此时，你的目标可能变成了“消灭孩子的哭闹”。
*   **放下“唯一目标”**：当你死盯住自己的目标（如制止哭闹、快速完成任务），就会将对方的情绪视为障碍。这会将双方置于对立面。
*   **创造心理空间**：只有主动暂停并放下这些念头与单一目标，才能在心中腾出空间，将目光转向对方。

只有我们不把自己的需求当成唯一，才有可能看见并承认他人的感受与需求。这为进一步的沟通创造了可能。当我们放下那个所谓的“唯一目标”，彼此间的距离反而更近了。

我很喜欢龙应台在《亲爱的安德烈》中的一段回应。当18岁的儿子安德烈倾诉失恋和与老师的矛盾后，又自嘲自己的烦恼微不足道时，龙应台没有否定他的感受，而是说：“**我们心里的痛苦不会因为这个世界上有更大或者更值得的痛苦而变得微不足道。它对别人也许微不足道，但对我们自己，每一次痛苦都是绝对的，真实的，很重大，很痛。**”

这是一个教科书般的共情回应。她完全与孩子站在一起，看到了他的需求。试试看，停一停，把自己的需求放一放，不将其视为唯一中心，你才能看到别人的需要。这就是共情的开始。

## 第二步：拿起来，说出来 🫳➡️🫴

当我们学会了暂停，接下来就是主动靠近对方的感受。这一步的关键意向是“拿起来”——拿起对方的需求。

以下是具体做法：

*   **拿起对方的需求**：在心理上，尝试拿起并关注对方的状态。例如：“那个孩子哭得一身是汗，他真的好难过。”“我的合作伙伴语无伦次，他真的好想把这个方案做到完美。”
*   **将感受“言语化”**：“说出来”这个动作背后，是心理学中的“言语化”机制。感受最初是身体的主观体验，无法直接用理性处理。直到我们为它赋予语言符号（说出来），才能被思考、讨论和交流。这是一种高度适应性的情感加工机制。
*   **帮对方说出感受**：共情发生时，你可以尝试帮对方说出他的感受。例如：“**买不到玩具，你好难过，是不是？**” 或 “**方案总是不满意，你也很着急、压力很大吧？**”

当你的感受被对方准确说出时，你会体验到一种美妙的连接感：“他懂我。” 这会建立信任，促使你敞开心扉，进行心平气和的沟通。你也会相信，双方是合作关系。

你发现了吗？这是一个完美的合作循环：合作思维启动共情，共情让双方从对立站到一起，站到一起又进一步激发了共情。反复练习并将此固化为习惯，我们为人处事的方式就会改变。因为能共情对方，我们更能获得依赖、信任，进行有意义的交流。这个过程，就是影响力发挥的过程。

这里有两个小建议可以帮助我们更好地共情：

1.  **如果感觉不到对方情绪**：这不要紧。只要你有合作思维的意愿，就有了询问的勇气。你可以大胆承认并主动询问：“**我不知道你怎么了，但我很在乎你，你可以说一说吗？**” 这是很好的破冰方式。很多人做不到，是因为缺乏合作思维的意愿和动力。
2.  **避免“言语化”时的错误操作**：注意，不要在承认感受后立刻加上自己的评判或建议。例如：“你一定很难过吧？**但是你这样大哭大闹是不对的。**” 这会让共情效果大打折扣。我们常常一不小心就带出自己的目的。

更好的共情方式是：可以等对方发出邀请再给出建议，或者通过询问来确定自己的建议是否合适。

分享一个我常用的共情式表达结构：**描述问题 + 承认感受 + 询问办法**

*   **对孩子打针**：“要打针了（描述问题），你是不是很害怕（承认感受）？那你是想现在就赶紧打完，我们就买根棒棒糖吃，还是再等一会儿打（询问办法/给出选择）？”
*   **对完美主义合作方**：“整个方案修改了五次了，后天就要定稿了（描述问题），你肯定也急坏了吧，但是又不想放弃质量（承认感受）。那现在时间这么紧，我们还有什么更好的方案吗（询问办法）？”

如果方案是由对方自己说出来的，推进效果会比你直接给出建议好很多。你可以在生活中细心体会这种差别。

## 第三步：终极捷径——拥抱的力量 🤗

在回顾了“停一停”、“拿起来”这些理性方法后，我还想分享一个快速有效的方法：拥抱。

这不是玩笑。如果你想共情的人是爱人、重要的战友或你在乎的人，有时无需复杂步骤，直接去拥抱他。

为什么拥抱有如此大的魔力？心理学中的“身体优先理论”认为，我们先做出动作，情绪才会跟上，动作引导着情绪。

*   **对你而言**：当你和对方拥抱，你会在感情上更认同他，也更容易触发对他的共情。
*   **对对方而言**：他会在身体接触中感受到情绪的流动与支持。

科学研究也支持这一点：磁共振扫描显示，身体接触会刺激大脑皮层。其他研究表明，身体接触能降低心率、血压以及与压力相关的激素皮质醇水平。这些神经和生理层面的反应，会调节两个大脑，使它们在情绪上更加同步。

在《锅盖头》一书中，美国海军陆战队成员安东尼·斯沃福德描述了海湾战争前的一段经历：在即将出征对抗强敌、誓死而归的前夕，战友维尔蒂组织了一场“拥抱聚会”。书中写道：“我们已经做好了赴死的准备…多亏了维尔蒂，我们互相拥抱，觉得自己又像个人了…维尔蒂对我们敞开胸怀，让我们感受他的需要，我们也将自己的心敞开。这样，我们就不再是从前那帮粗鲁野蛮的傻大兵…”

这就是拥抱的力量。

## 总结

本节课中，我们一起学习了将共情思维落地的具体行动步骤：

1.  **停一停，放一放**：在冲突中暂停惯性反应，放下以自我为中心的“唯一目标”，为看见对方创造心理空间。
2.  **拿起来，说出来**：主动关注并拿起对方的需求，通过“言语化”帮对方说出感受，建立深度连接与信任。
3.  **利用拥抱的力量**：对于亲密或重要的人，直接的身体接触（拥抱）能快速引导情绪，促进情感同步，是共情的强大捷径。

共情是一种重要的能力，它根植于我们的脑神经机制。我们之所以有时会失去它，是被非合作思维阻挡了。通过练习这些方法，我们可以重建连接，发挥真正的影响力。

祝你有可以拥抱的人，祝你有属于自己的影响力。
