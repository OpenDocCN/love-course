# 青春期心理课 038 👁️：孩子频繁眨眼、抽动是怎么回事？

在本节课中，我们将探讨孩子出现频繁眨眼、身体抽动等行为背后的原因。这不是简单的眼部问题，而可能与一种名为“抽动症”的儿童神经官能症有关。我们将了解其成因，并学习一套以运动与家庭环境调整为核心的综合干预方法。

---

如果家长发现孩子频繁眨眼，首先可能想到的是结膜炎。

但如果医院检查显示眼睛没有问题，滴眼药水后孩子仍然眨眼，并且症状加重，从眨眼发展到耸鼻子、揪嘴巴，进而出现扭脖子、耸肩膀、动胳膊、鼓肚子甚至全身震动，就需要注意了。

这可能是抽动症。抽动症属于儿童神经官能症，是青少年容易出现的一种神经官能症。

它与成年人不同。同样属于神经官能症、神经衰弱，成年人可能更多表现为失眠，但孩子则可能出现抽动症。

症状加重后，还可能伴随出声，例如发出怪音、喉咙里不停清嗓子、不停咳嗽，甚至骂人。这被称为抽动秽语综合征，都属于儿童神经官能症。

儿童神经官能症是我们最近几期课程强调的重点。当孩子的神经系统出现失衡时，就会产生这类症状。

有的孩子失衡可能表现为抑郁，有的表现为焦虑，有的表现为抽搐，有的表现为失眠，而有的孩子则表现为抽动。

我们首先要了解，抽动症属于神经官能症。我曾接待过一位儿科医生的孩子，这个孩子出现了抽动症。

后来，这位家长先给孩子吃药，发现效果不好，又带孩子去住院，效果依然不理想。

他找到我时，希望我先看看孩子是什么问题，并寻找方法。我们首先排除了精神疾病和智力问题，确认属于抽动症。

后来我建议他在山脚下租一间房子，每天早晨带孩子去爬山，当时正好是暑假。

早晨爬山，上午回来睡午觉，下午带孩子游泳和打球。

经过两个星期的调整后，孩子从严重的抽动症变得只有轻微症状。

后来回家继续康复，情况就好多了。

我们来分析原因。这个孩子小时候是剖腹产，剖腹产孩子的神经系统通常比较敏感。

因此，剖腹产的孩子比顺产的孩子更容易因为压力和紧张而患上神经官能症。

其次，在小学时，家长经常强调学习的重要性，要求孩子必须保持第一。

等孩子上了中学，家长说了一句：“你一定要像小学那样得第一，好吗？”这句话之后，孩子就得了抽动症。

所以，家长过度的期望和期待给孩子造成了巨大压力和紧张。

因为神经系统紧张、劳累、疲劳，就会出现问题，只是表现形式不同。

有的孩子可能表现为失眠、头疼、犯困等，但有的孩子就会表现为抽动。每个人因人而异，应激反应不同，所以也不用过度紧张。

当孩子出现抽动症，如眨眼、耸肩膀或发出怪声时，家长首先应带孩子找心理医生做诊断。

诊断后，可以开始采取综合运动的方式进行干预。

这种综合运动方式可以包括一天内的多种活动，如爬山、游泳、打球、跑步，并减少造成早晨紧张的原因。

例如，如果原来每天都要学外语、打乒乓球，并因此引发了抽动，我们可以调整这些活动。

比如改为每周两次学外语、两次打乒乓球，其他时间可以去爬山，让生活方式变得更丰富、更多样。

外出游玩也有帮助，因为旅行面对的环境是放松的、多变的，而非单调的。

所以，家长遇到孩子有抽动症时，需要改变原来的学习方式。

我见过一位家长给孩子报了大约七个课外班，后来孩子出现了抽动症。

我建议他把七个课外班都暂停，只留下一个孩子喜欢且愿意做的，比如绘画。

暂停其他班后，孩子的症状就好了。原来是过重的课外班负担，叠加在学习任务之外，导致了抽动症。

这些都是可以帮助孩子解决问题的方法。

此外，还有感觉统合训练。

我们在小学阶段常对剖腹产的孩子使用感觉统合训练，帮助打通大脑左右半球的联系。

特别是交感神经系统和副交感神经系统之间的联系，让它们能够互相配合，而不是只有单一神经系统在工作。

例如，有的孩子紧张时只使用交感神经系统保持紧张状态好几个小时。

紧张过后，又没有及时通过锻炼和劳动来放松，就容易出现持续紧张，导致抽动。

因此，我们可以注意使用感觉统合训练来帮助孩子缓解抽动症状。

以下是感觉统合训练包含的一些项目：

*   **平衡能力训练**：例如走平衡木、爬山、荡秋千、旋转。
*   **协调性训练**：例如拍球、跳绳。
*   **触觉训练**：例如游泳、按摩、抚触。
*   **视觉与听觉能力训练**：例如打羽毛球、打篮球、打击乐。

这种综合了平衡、协调、触觉、视听觉等能力的训练，就称为感觉统合训练。

感觉统合训练的目的，是为了打通大脑各个功能区域之间的联系，打通它们的通道，让它们能够自动地互相配合。

我们还发现，患有抽动症的孩子的家长，尤其是妈妈，通常非常紧张。

家长本身的性格是高度紧张的，并且控制欲特别强。

我见过一位家长，她的孩子在我这里做抽动症训练。在我的指导下，家长几乎每天都要给我发微信，要求我调整这样那样的训练。

后来我说，你这样太让人紧张了。这一系列训练共20次，大约持续两个半月，你每天都要求我调整。

你的控制欲和紧张程度，连我都感觉到了，更何况你的孩子。

家长一开始辩解说没有给孩子压力，但我都感觉到了，因为你每天都在发信息要求改这改那。

我请家长一定要放松。所以，在对抽动症孩子进行训练、锻炼和调整时，家长同时也要改变自己。

需要改变自己的性格、控制欲和焦虑心态。

如果只想改变孩子，自己不做任何改变，那么孩子很可能还会复发，因为问题的根源还在。

我们说，青少年的问题，原因在父母，责任在自己。家长是有原因的。如果家长不做改变，根源仍在，过一段时间问题还会复发。

在家长和孩子共同配合之下，这个问题是很容易解决的。

---

本节课中，我们一起学习了孩子抽动症的表现与本质。它属于儿童神经官能症，常与压力、紧张及神经系统失衡有关。解决之道不在于单一的药物或说教，而在于**综合运动干预**、丰富生活体验、减轻过度负担，并结合**感觉统合训练**。最关键的是，家长需要同步调整自身的焦虑心态和控制欲，与孩子共同成长，才能从根本上解决问题。
