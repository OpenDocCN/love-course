# 032.情感心理咨询技能大全（完结）：第10章：恋爱中的咨询—安洁：09.婚前咨询的议题_note

在本节课中，我们将要学习婚前咨询。婚前咨询是指一对伴侣在经历了恋爱阶段、尚未步入婚姻时进行的咨询。它与恋爱咨询的不同之处在于，这对伴侣已经相处了一段时间，有了一定的磨合，对彼此有了更深的了解。我们将探讨促使他们走进咨询室的原因，以及婚前咨询在家庭生命周期中所处的独特位置——即新家庭成立之前的阶段。在这个阶段，伴侣关系已不仅仅是恋爱关系，而是形成了长期承诺。

## 婚前咨询中可能遇到的议题

上一节我们介绍了婚前咨询的基本概念，本节中我们来看看咨询师在婚前咨询中可能遇到的具体议题。以下是常见的几类议题。

### 1. 角色的转变

在恋爱过程中，许多人感觉恋爱是“慢慢恋，慢慢爱”，中间可能更换对象，更多是为了爱情，不一定需要承担太多责任。然而，一旦从恋爱状态进入婚姻状态，角色就发生了转变。

**公式：恋爱角色 -> 婚姻角色**

例如，女性可能要从“女孩”转变为“女人”、“妻子”和“儿媳”。男性则要从“大男孩”转变为“男人”、“丈夫”和“女婿”。这种跨越性的角色转变会给当下的年轻人带来困扰和不适。

来访者通常不会直接说“我要做太太了，很不适应”，而是通过生活中的具体事件呈现问题。例如，他们可能会抱怨：“我们已经商量要结婚，我也见过他父母了。但之后我要参加他家庭的很多聚会，如果我不去，他就会不高兴。” 这看似是小事，但其背后反映的是角色改变带来的关系议题：即从恋爱阶段的二人世界，转变为需要融入对方大家庭并承担新角色所带来的不适。

我们考量一对伴侣时，不能只看问题本身，还要看问题如何在关系中产生，以及如何受到系统影响。那么，是什么让当今的年轻人觉得角色转变如此难以适应呢？我们需要回到核心家庭的变化来看。

在少子化家庭增多的背景下，一个孩子在成长过程中，往往拥有自己的物理空间，在心理上也一直被当作小孩对待。许多年轻人到了二三十岁，也未必需要承担家庭中的很多责任。特别是65后或70年代初的父母，因经济条件较好，希望为孩子提供好的生活环境，孩子不需要很早“长大成人”，可以长时间享受“孩子”的角色。

同时，在少子化家庭的教养环境中，父母很早就给予孩子被尊重的互动模式，例如常问“你喜欢什么？”、“你自己愿意吗？”。因此，年轻人在过去二十多年的成长历程中，由于自己不太喜欢或不愿意，父母也未必会勉强他们参加家庭聚会或承担某些责任。长期在此情境中成长的年轻人，会更尊重自我感受、自我空间和自我边界。

突然面对从“大男孩/小女孩”到“男人/女人”、“丈夫/妻子”的角色转变，需要开始为生活做规划，并融入对方的原生家庭进行社交，这对当今的年轻人而言并非易事。

但在过去几十年，年轻人为何不觉得这么难呢？首先，过去年轻人离家时间较早，在较早离家的过程中，他们必须通过与外界的连接和人际交往来保证在社会中生存。而现在，年轻人可能越来越独立，“我跟喜欢的人在一起，做自己喜欢的事，可能不需要顾及他人太多感受”。这既有好的一面，也可能在角色改变上带来难题。

第二个层面是当今年轻人心理上成熟的年龄在变晚，这也与核心家庭的变化有关。过去社会没有“啃老”这个词，而现在这种现象并不少见。有些家庭经济条件很好，父母有体面的工作和积蓄，不需要孩子操心养老，甚至早为孩子准备好了教育投资、保险、房子或现金。因此，孩子某种程度上不再需要为现实生活花很大力气。这并不是责备父母，因为这一代父母（约65后）自己在成长过程中吃过苦，经历过物质匮乏和激烈竞争，所以他们希望孩子能生活得轻松些。

在这种互动关系中，孩子可以在家庭中很好地享受“孩子”的角色，不需要过早成熟去思考如何立业或离家。即便有些年轻人有很好的工作，但住的房子是父母提供的。因此，孩子在30岁左右，甚至在恋爱或刚结婚时，其角色可能仍停留在“我是一个孩子”的状态，两人更多像是在“搭伙过日子”，享受甜蜜情感。

对于两个刚开始组合、要建立新家庭的年轻人来说，心理上可能没有完成角色的转变。我们可能在咨询室中听到未婚妻抱怨：“我老公还像是个小朋友，我时时刻刻都要照顾他。” 或者“他总是打游戏到很晚，玩车模玩到废寝忘食，根本不记得我。我跟一个像大男孩的人结婚，怎么放得下心？” 对女孩子也是如此。过去几十年，女孩子可能被教育要孝敬婆婆，有“多年的媳妇熬成婆”的说法，但在当下文化中，这话可能也不太适用了。这些都是角色改变带来的冲突。

如果无法适应角色改变，即在做伴侣咨询时，双方都处在“儿童态”（一个小男孩和一个小女孩），他们可以玩得很愉悦，有共同爱好，一起旅行、做饭，但却无法像成年人一样生活。要知道，婚姻不是给小孩准备的，婚姻是给成年人准备的。小孩子在一起那叫做“过家家”的游戏。因此，突如其来的角色改变，若年轻人不能接受或适应，就可能带来关系冲突。

这种不适应在现实生活中有多种表现形式：
*   第一种：因为不想适应角色改变，两人之间可能会吵架。
*   第二种：因为角色改变带来了双方之间的陌生感、对未来的不信任感、或恐惧害怕。他们可能不确定“我跟这个人结婚，是否能把日子过下去”，从而出现对结婚的犹豫、怀疑和退缩。

所以，一对小夫妻来到咨询室，谈的可能都是琐事，比如“房子怎么装修”、“谁做饭”、“周末去谁父母家吃饭”等。这些听起来是生活琐事的冲突，但如果咨询师把精力都放在解决这些琐事上，就可能没有透过问题看到关系的本质——即因角色改变带来的冲突。如果只坐在解决问题上，咨询师会发现自己特别忙且累，因为不断在替他们解决问题，但这不应该是咨询师的主要工作。

在做家庭咨询时，我们常说咨询师的工作是“和伴侣/家庭在一起”，搅动家庭/伴侣系统，激活他们自己解决问题的能力，而不是咨询师去帮助他们或做判官（例如规定谁洗衣服、谁刷碗）。我们应该把力气花在问题背后的本质上，即透过症状看到他们关系的本质：也许他们的争执是因为“一个小男孩和一个小女孩玩不到一起而要一拍两散”；或者关系中一方是成年人，另一方还是“小男孩”或“小女孩”，出现了关系的不对等。这是我们要看到的。

### 2. 婚姻中需要开始承担责任

上一节我们探讨了角色转变带来的挑战，本节我们来看看婚前咨询中另一个核心议题：婚姻中需要开始承担的责任。第一个重要的责任是经济责任。

婚姻除了情感和性功能外，经济责任非常重要。钱怎么用、谁管钱、给原生家庭父母多少钱、原生家庭父母给多少钱，这些都是与经济相关的议题。而经济议题背后，又与个人的价值观以及在家庭中的权力大小有关。所以，钱不仅仅是钱的问题。

举个例子，有一对年轻夫妻，都是从中国到海外同一城市、同一大学接受教育，文化背景和教育经历完全一样。在求学过程中相爱，然后回国准备结婚。男孩选择了一家创业公司开始创业，女孩进入了一个相对稳定的单位工作。同居后，他们开始出现很多问题。

在女孩看来，这是经济问题。她是这样表述的：“以前在国外留学时，我们一起出去吃饭花钱都没问题，我不觉得我们有差异。但现在，比如我想周末出去吃顿好点的饭，度个周末，我老公就会觉得太贵，说‘你一个人去吃吧，我就不去了，不然又要花掉500块’。再有，我有假期时想出去旅行，想去好点的国家，一趟旅行费用要好几万。我老公不是不让我去，他说‘你去就可以了，但我就不要去了，我不去就可以省下来好几万的钱’。”

他们之间的争端就出现在经济问题上。我当时问她：“听起来你的先生其实挺不错，他虽然没陪你去，但并没有反对你去。也就是说，你有这样的生活方式和花钱方式，他并没有反对你。那你为什么会觉得经济是个问题呢？”

她跟我讲：“老师，你看我们两个人现在在用钱上是不一样的。那结婚之后，是不是也是这个样子呢？这让我觉得很绝望。” 绝望的原因有两点：第一，她能理解先生现在创业，公司很需要钱，所以他心疼钱，舍不得花几万块出国玩；第二，她回国后才了解到，双方的原生家庭有那么大的不同。

这个女孩来源于一个非常优渥的家庭，父母能给她提供很好的经济支持和援助，包括早早在城市里为她买了房。所以她只需要用心工作，没有任何后顾之忧。而这个男孩来源于中国一个三线城市的普通家庭，他是唯一一个依靠读书出国留学、回来光宗耀祖的男孩。他回国后不仅要创业拼搏，每个月还要反哺自己的家庭，给家里钱。同时，在那个阶段，他妈妈身体不好暂时不能工作，所以这个男孩承担着比较重的家庭压力。

因此，在他们的生活中，即便这对年轻伴侣在结婚时并不缺钱（两人的收入足够支付周末的好晚餐或假期的好旅行），但男孩子就是无法去享受这样的生活。所以，虽然看起来是经济问题，但其背后与很多问题联系在一起：包括两个家庭用钱方式不同、对孩子的期待不同。一个家庭是“你好好享受自己的人生就可以，甚至鼓励多出去走走看看”；另一个家庭则赋予男孩“必须不断努力、奋发图强、光耀门楣”的隐形责任。这也是中国文化很多时候给男士的一种影响或约束。

我们在做咨询时发现，那些拼命工作的男性，有时即使已达到很高成就，也无法让自己放松休息，觉得一休息就有罪恶感和焦虑感。因为在他的潜意识中，可能觉得“整个家族还有很多人指望着我，或者他们还过得很辛苦，我怎么可以在外面吃喝玩乐？” 这就是由于经济问题带来的冲突。

后来，帮他们梳理了各自不同的成长背景，以及经济冲突背后的实质后，这对年轻人最终没有走进婚姻，在婚前选择了分开。这个决定是他们自己做出的。我并不觉得这是一个特别不好的结果，也没有必要去批评这个女孩是否“嫌贫爱富”或“只知享乐”。如果咨询师持有这样的观点和偏见，那我们的立场就不中立了。

因为即便女孩在了解了男孩的成长史、用钱态度及当下生活压力后，她非常真诚地说：“我非常能够理解你。但对我而言，如果让我今后可能30年的婚姻生活都要面对这样的日子，我觉得有些害怕。我可能还没有准备好走进这样一段婚姻，去跟你一起过这样的日子。我害怕的不是穷或钱少，我害怕的是好像每用一次钱，身上都背负着很大的重任，或者如果自己不积极、不努力、不勤俭、不上进，就是对不起周边的人，对不起生我养我的父母。这个压力对我来讲太大了。对我来讲，我可能是需要好好享受我的生活的。接下来我会不会结婚、跟谁结婚，的确没考虑那么好。也许当我完全认识自己之后，我觉得我并不那么适合走进婚姻，去跟别人共享这么多生命历程。”

当然，这是他们两人互相认识对方之后的一个决定。所以，作为咨询师，我们要做的是梳理中间的过程，去看那个关系，搅动他们自己的系统，促使他们做出一些调整和改变。因此，千万不要在这对年轻人坐在你面前时，就有一个约定俗成的观点，比如“两个人在一起，劲就要往一处使”、“你不能看不起他”等等。如果是这样，我们就变成了批评和说教。所以我们一定要检查自己隐性的价值观，反思我们对来访者可以包容的到底有多少。这是婚姻中的经济责任。

婚姻中第二个层面的责任是性的责任。一旦成为夫妻，性在相互之间是有责任的。在恋爱期间，性很多时候是作为一种情感表达、爱的体现、增进爱的方式，或者完全是一种享受。在婚姻中，一旦走进婚姻，两人慢慢从很有激情的浪漫期，回归到理性和平期。在这个阶段，纯粹来源于身体吸引的性在降低，但又需要履行性的责任，这对年轻夫妇而言有时也有压力。

这种压力体现在咨询中可能有不同的呈现形式。例如，有的女性来访者说：“我睡觉比较早，11点前就睡了。但我先生睡觉比较晚，要搞到晚上一两点。我们在一起住之后就好像有时差。有时差之后，性这件事就好难找到一个时间去做。而且两个人好像没有像年轻时那么干柴烈火，上班回来也挺累，天天能见上面了，所以就变得平淡一些了。但后来我发现，我们俩躺在一起后，我先生在他睡觉后会去自慰。我最开始很紧张，也不敢出声，还要装睡，假装不知道这件事。” 当然，性方面的不同步也变成了他们之间的问题。

当谈到先生自慰时，作为年轻妻子，心中其实有各种各样的情绪。例如，她虽然在理性上知道这是男性用来自我满足的一种方式，或是宣泄白天压力的一种途径，但某种程度上也会觉得“他是不是对我已经没有兴趣了？我作为他的妻子，我的角色到底在哪里？” 这是我们在咨询中可能会遇到的议题。

第三个责任是处理彼此的关系和家庭关系。“处理彼此的关系”指的是年轻小夫妻之间的关系；“家庭关系”则包括了这对年轻夫妻与各自原生家庭的关系。

原生家庭的关系在结婚前就会有很多体现。例如：婚房是谁买的？车是由谁买？如果其中一方（特别是男方）买了房（在大城市需要花很多钱），买了房之后，另一方的名字加不加进来？或者买房后，另一方家里到底会买什么？这些看起来都是与婚姻相关的经济、聘礼、彩礼问题，但其背后其实是家庭关系。

比较常见的情况是，一方买房（如男方家庭付了首付），女方家庭可能没付。这时要不要加女方的名字？有的原生家庭父母会说：“反正以后你们自己还贷，那就把你太太的名字一起加上来，我们老夫妻的名字就不写了。” 这样可能争论会少一点。当然也会出现另一种情况：父母买的房，房上是父母和先生的名字，跟太太没有关系。也许在结婚前没有吵没有闹，但这件事可能会成为一个“梗”。梗的是什么？很多时候太太会讲：“我不是介意那个名字，我介意的是我跟他马上要结婚了，而他们家好像没有把我当成一家人。” 这也是容易出现的矛盾。

再比如，跟大宗财产相关（如全款买的房），可能在结婚前需要另一方写一个婚前协议并进行公证，公证如果情感出现问题，这房子跟他/她没关系。在这种情况下，我们也容易遇到矛盾。

这些其实都体现了小夫妻和原生家庭之间的关系出现了问题。当然还有其他的家庭关系问题，例如：两人决定结婚，但在结婚前有一段婚前同居时期。在这段时期，还是各回各家吃饭——“我回我父母家吃，你回你父母家吃”。如果双方都觉得OK，没问题。但有一方可能会觉得：“你没有带我去见过你的朋友，没有带我去你父母家吃饭，我会觉得不舒服。” 或者另一方会觉得：“我邀请你一起去我父母家吃晚饭，但你从来都不去，因此我觉得你还不像要跟我结婚的样子。”

这些其实都是在从角色到家庭关系中的一个大的改变。因为在这个家庭关系中，我们要面对的就不只是“你这个人”的关系了。正如前面所讲，婚姻是“一张床，六个人”，甚至是一个家族。你不可避免地要与这个家族、整个大家庭发生关系。那么，发生关系后，我们的边界在哪里？空间在哪里？交集又在哪里？这些都是在婚前需要不断磨合的。

还有一个在婚姻中需要开始承担的责任，是关于日常生活和娱乐的安排。这些相对容易明白一些，就是日常生活中，因为马上要走入婚姻，类似于一个“1:1版的婚后生活”，我们怎么样来安排日常生活？怎么样来安排旅行娱乐？这些也是我们需要承担的一些责任。

年轻的夫妻出现的这些议题，其实跟我们婚姻咨询课中讲到的婚姻议题有很高的重合度，都是跟金钱、性、原生家庭父母、日常生活、沟通、娱乐相关。唯独少了一条：跟孩子的教育相关。那么，它的不同点在哪里呢？

不同点在于，在婚前咨询的时候，与已经过了一段日子的夫妻咨询相比，不同点在于：这对小夫妻因为很年轻，具有的生命能量非常高（精力很旺盛），所以一旦发生冲突就很激烈，很容易吵架。因此，在非常激烈的争吵中，他们会发现冲突管理很难。而且，还是在婚前咨询，还没有那一张结婚证作为保障，也还没有一个孩子作为血缘的纽带，所以这个关系就显得特别脆弱。两个人就极有可能因为一些鸡毛蒜皮的事情吵起来，而且会吵得很厉害。吵完之后的结果就是：“我还不要跟你结婚了？”“我还不要跟这个人过下去了？”“我们这个婚姻还不要去领证了？”

这也是婚前咨询的一个特征：即这对年轻伴侣是有退路的。不像在结婚之后，结婚时间越长，离婚就越麻烦。即便是去换张离婚证，都不是一件那么容易的事情，分离是不容易的。但这年轻的小夫妻都还没有结婚，所以他们是有退路的。因此，这让关系的稳定性变差，让关系变得脆弱，让关系变得更加多变。所以我们常常说，可能在婚前的这个阶段，恋爱已经不错了，但是要走入婚姻了，在这个阶段当中，反悔的可能性是比较高的。

### 3. 对婚姻全能的想象

婚前咨询的第三类议题，是对婚姻全能的想象。在恋爱咨询中也有对于对方全能的想象，在婚姻咨询中也有。这就是对于婚姻有一个非常好的期待：我希望他爱我，他就应该是懂我的；他能够为我负责；他能够因为跟我结婚之后，所有的坏毛病都能改掉。

特别是在我们的成长历程中，有一类人特别喜欢过度负责，有时带有“拯救者心态”。这种心态在女性身上稍微多一点。特别看不得的就是那个男的过得特别惨或特别不好，然后就会对自己也有一个全能的想象：“我要去拯救他，我跟他在一起我可以改变他。” 我就曾经听一个女孩讲：“我们家的这个先生，他一直是在做一个小男生，什么都听他妈妈的，他妈妈从来都没让他做过主。我想我跟他结婚之后，我是可以改变这一点的。我会慢慢给他空间，慢慢让他为自己做决定。我想这是我可以做到的事情。”

但是，当然，当她来做咨询的时候，她发现自己陷入了巨大的挫败当中。实际情况是，她是很难去改变的。当我们去做一个过度负责的拯救者时，你常常也会发现，这样的一段关系会非常令你失望。你会发现其实要不然你很难改变什么，要不然你就会发现，你就变成了他的“第二个母亲”（打引号）。当然，也有可能是男士来做“第二个父亲”。所以，当你一开始对于对方有一个全能的想象，对于亲密关系觉得“两个人在一起了，我就完全拥有他，我就可以慢慢改造他”，有一个全能的想象，对于自我的能力有一个全能的想象的时候，那么在走进婚姻之前的这个落差就会越来越大。

那么在这个时候，咨询师可以怎么做呢？我们肯定是需要去调整他们的期待。例如，当你检查到你的案主可能有一个拯救者的心态，那么在此时最好是能够让他从拯救者的位置上能够出来。当然，我说起来是一句话，做起来是需要费一些功夫的。因为一个人成为拯救者，不是在一瞬间在那个位置上的，这还是跟他自己的个人成长经历有很大关系，也跟这个关系的互动有关。从拯救者的位置上出来，这是有可能去做的一个办法。

第二个办法是降低彼此对于婚姻以及对于对方全能的期待。降低期待，我们有几个办法可以去做：
*   第一个：只是把期待，比方说从10分降为8分，这是降低。
*   第二个：我们找一个替代品，去替代这样的一个期待。这些我们都是在处理期待的部分可以做的一些
