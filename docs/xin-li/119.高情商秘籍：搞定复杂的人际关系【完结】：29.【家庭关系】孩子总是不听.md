# 119.高情商秘籍：搞定复杂的人际关系【完结】：29.【家庭关系】孩子总是不听
## 29.【家庭关系】孩子总是不听我的话，怎么办？👨‍👩‍👧‍👦

在本节课中，我们将要学习当孩子“不听话”时，父母常见的心理动因、不同类型父母的反应，以及如何调整相处模式，将亲子关系从“控制与服从”转向“合作与成长”。

---

如果是一个淡定的父母，很少会问“孩子不听话怎么办”。他们通常尊重孩子，将其视为独立的个体，而非需要服从命令的士兵。当父母将孩子的行为定义为“不听话”时，往往意味着亲子关系已陷入“发号施令者”与“被动服从者”的模式。

为什么孩子不听话会让父母感到焦虑？这背后有三个心理学动因。

**第一，需要强烈的掌控感。** 生活中许多事让我们感到失控，例如工作不顺或对未来的担忧。我们希望孩子“听话”，成为生活这个**方程式**中一个恒定的参数。资源越少的家庭，父母往往越希望孩子按自己设计的轨迹发展，以弥补自身的失控感。

**第二，处于特别自恋的状态。** 有些人需要世界围着自己转，当无法实现时，便希望孩子来满足这种自恋感。孩子的“听话懂事”让他们体验到类似君王的感觉。

**第三，缺乏管理自己情绪的能力。** 这类父母无法妥善处理自身的恐惧、愤怒等情绪，于是将情绪投射到孩子身上。他们将孩子定义为“不听话”，实质是让孩子帮助处理大人自己的情绪。

了解了焦虑背后的动因，我们来看看四种动物类型的人面对孩子不听话时的表现。

以下是四种类型的父母及其典型反应：

*   **蜗牛型**：本身容易焦虑，依赖感强。他们依赖孩子，并想象孩子会按自己设定的完美样子成长。一旦现实不符，就会焦虑，并通过命令（如“你不许出去”）来试图控制，继而觉得孩子“不听话”。
*   **袋鼠型**：感到特别委屈，并以愤怒的方式表达。他们会说：“我为你付出这么多，你却这样，太让我失望了。”他们认为付出未得到回报，因而指责孩子。
*   **鸵鸟型**：第一反应是“自恋损伤”。他们心中有一个“完美孩子”的想象，并潜意识拒绝抚养现实中真实的孩子。当真实孩子与想象出现巨大差异时，他们会觉得丢脸、受损，可能做出过激反应。
*   **斑鸠型**：觉得孩子“造成了麻烦”，消耗了自己的时间和精力。他们心中涌起的是失望，甚至可能嫌弃自己的孩子。

在探讨了父母的反应后，我们需要从关系心理学角度理解两个核心原则。

**一，只有两种关系能产生绝对服从：暴君与臣子，或掌握生杀大权的狱警与犯人。** 亲子关系不应该是控制与服从，而应是**合作**关系。当孩子感到关系不适时，自然会反抗，表现为“不听话”。

**二，孩子的自我成长必然伴随“不听话”。** 孩子有两个关键自我成长期：
1.  **7-8岁**：开始独立思考和知识扩展，会提问、比较、维护自我。
2.  **青春期（小学高年级及初中）**：常被称为“叛逆期”，会挑战权威。**有研究显示，这个时期能叛逆的孩子，未来往往更有出息**；而一味乖巧听话的孩子，反而可能成长为“巨婴”。

在明白上述原则后，我们可以通过调整相处模式来改善亲子关系。

以下是四个具体建议：

1.  **认同这是必然发生的事**：孩子“不听话”是成长必然过程。如果孩子从未经历叛逆、绝对服从，反而可能意味着亲子关系已陷入病态。
2.  **接受自己的失落感**：当孩子开始质疑、辩论甚至超越你时，产生失落感是正常的。你需要接受，孩子已成长为一个**平等的独立个体**。
3.  **看到积极的一面**：孩子“不听话”说明他长大了，拥有了自我意识、思考力和创造力。只听命令、不思考的孩子，未来难有大的成就。
4.  **制定共同遵守的第三方规则**：改变相处模式，将孩子视为平等合作伙伴。当孩子提出“妈妈你要我这样，你自己也要做到”时，这是他希望改变关系的信号。此时，可以尝试共同制定规则，**大家一起遵守**。

---

本节课中我们一起学习了：
1.  父母觉得孩子“不听话”，常源于自身的焦虑，其背后有需要掌控感、自恋状态及情绪管理能力不足三大心理动因。
2.  蜗牛型、袋鼠型、鸵鸟型和斑鸠型父母面对此情境时，会有不同的情绪与行为表现。
3.  从关系心理学看，孩子“不听话”是自我成长的必然，且叛逆的孩子可能未来更有出息。
4.  当遇到孩子“不听话”时，可以通过认同必然性、接受失落感、看到积极意义以及制定共同规则四个方法来调整关系。

最后，祝你学完本课，能从焦虑中解脱，重新审视孩子“不听话”的积极意义，调整相处模式，从而培养出拥有独立思考能力、更具创造性的孩子。
