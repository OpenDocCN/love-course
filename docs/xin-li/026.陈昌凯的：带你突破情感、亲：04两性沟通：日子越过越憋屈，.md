# 课程026-04：两性沟通：日子越过越憋屈，是我选错人了吗？ 💔

在本节课中，我们将学习情绪在亲密关系中的核心作用，并掌握通过提升情商来避免冲突、实现有效沟通的具体方法。

我是陈昌凯，欢迎继续学习心理学课程。我充满好奇心，因此有一个特别的爱好：喜欢观察他人争吵。走在路上看到别人吵架，我常会驻足。这个爱好或许也影响了我选择成为心理咨询师。在咨询室里，我常常见证各种争吵，其中夫妻间的争吵最为常见。

无论争吵的起因是什么，战况如何，妻子往往会在最后发起“总攻”。她会指着丈夫的鼻子冷冷地说：“你不要以为你了不起，我们能过就过，不能过就离。”女性通常更偏感性。当她发现共同生活多年的丈夫此刻毫不退让、不懂她的需求，还在胡搅蛮缠时，会感到非常生气。于是用“能过就过，不能过就离”来威胁丈夫。

然而，男性从小被教育要“自强不息”，而“自强不息”的一个重要方面就是不能服软，所谓“士可杀，不可辱”。即使丈夫明白这只是威胁而非真想离婚，但面对此话，99%的丈夫都会说出那三个字：“离就离”，以示“我不怕你”。

当然，所有丈夫都知道说出“离就离”之后等待的是什么。妻子100%会回应：“好，明天就去办。”并可能加上“原来你早就想离婚了”等话语。这时，夫妻双方常感莫名其妙：明明开始讨论别的事，怎么最后吵到要离婚？

作为咨询师，我能从争吵内容中迅速发现冲突点，从而清晰了解他们如何走到要离婚的地步。其实过程很简单：**情绪升级**。妻子说“能过就过，不能过就离”时，情绪升了一级。丈夫没让情绪降下来，反而用“离就离”再升一级。至此，妻子无路可退，只能硬着头皮将情绪升至顶点——决定离婚。

幸好，夫妻通常晚上才有空来咨询，所以只能说“明天就去离”。即使我不做后续咨询，他们多半也不会真去。因为回家后争吵停止，各自休息，第二天早上互不理睬，不会立刻和好，但也多半不再提离婚。睡了一晚，情绪平复，理性回归。

如果两人在争吵时能稍加控制情绪，不让其不断升级，而是想办法降温，许多矛盾就不会如此复杂。这种让情绪降温的能力，就是今天常听到的心理学概念——**情商**。

情商指情绪智力商数，也称为**EQ**，是相对于智商**IQ**提出的概念。虽然如今媒体常提，但比起有一百多年历史的智商，情商只有三十多年历史。1987年，美国人比斯利在一篇文章中首次使用**EQ**概念。1988年，心理学家巴昂编制了首个标准化情绪智力量表。1990年，耶鲁大学的萨罗维和梅耶教授才系统定义了情绪智力。直到1995年，情商才真正进入大众视野。

1995年，哈佛大学心理学家丹尼尔·戈尔曼教授出版了名为《**EQ**》的书。该书迅速走红，进入各国畅销书排行榜，在全球掀起情商讨论热潮。书中，戈尔曼通过科学论证得出结论：情商是人类最重要的生存能力。人生成就最多20%可归因于智商，其余80%则受其他非智力因素影响，其中情商至关重要。

从这些研究可见，专家认为一个人的生活极大受情商影响。因此他们积极呼吁，必须从重视智商转向重视情商，并大力培养和提升青少年的情商。

具体来说，情商包含五个方面：
1.  认识自身情绪的能力。
2.  妥善管理情绪的能力。
3.  自我激励的能力。
4.  识别他人情绪的能力。
5.  人际关系的管理能力。

这五方面能力循序渐进。不了解自身情绪，自然无法识别他人情绪，更谈不上管理人际关系。因此，只有从了解自身情绪开始，逐步培养，才能逐一具备，最终成为高情商者。

后续课程将结合亲子教育，仔细分析这五个方面及如何培养孩子的情商。本节课，我们重点探讨如何妥善管理自身情绪，以帮助我们在人际交往中减少冲突，实现有效沟通。

讲到这，你可能会说：管理情绪还不简单，就是不发脾气。这可不一定。

我曾咨询过一对夫妻。刚结婚时家庭和睦，日子开心。后来妻子学会打麻将并上瘾，最大爱好就是一群人在一起打牌。但她常输，越输脾气越差，每次打完牌回家都和丈夫大吵。丈夫开始还对吵，后来吵累了，不再搭理她，连话都懒得说。结果两人同住屋檐下却不打招呼，不像夫妻，很快走到离婚边缘。

所以，无论是过度发泄情绪，还是过度压抑情绪，对我们和我们的生活都不是好事。

现在你可能觉得：情绪真麻烦，左也不是右也不是，干脆不要情绪好了。大家不吵不闹，做好自己的事不行吗？若真如此，你恐怕也不会满意。

如果你的丈夫非常理性，很少表露情绪，他可能擅长工作，但不擅交际。这都不是大事，但请想想：当你不开心时，由于他的理性，他很可能无法安慰你。要么躲开做自己的事，要么像前面课程说的，傻傻地问：“你是不是生气了？”你心里更不爽，因为你需要的是安慰而非质问，于是更加生气。但有什么办法？遇到一个没有情绪的人，你大概只能生闷气。因为冲他发火也没用，他可能一脸无辜地问：“你为什么要发火？这没道理。”是否感觉一拳打在豆腐上，非常无力、憋屈？婚姻仿佛成了你一个人的独角戏。

所以，千万不要认为情绪不好就想不要情绪，也千万别以为情绪不重要。因为情绪，特别是消极情绪，有着非常重要的功能。

首先，情绪帮助我们适应生活环境。婴儿生来不会说话，不舒服就通过哭来表达情绪。这是他们掌握的适应功能，以便让你知道他难受。当孩子摔跤跑到你面前哭时，他正通过悲伤情绪发出求救信号：“我现在很难过。”如果你抱抱他、安抚他，让情绪释放，他就会得到理解和陪伴。心理学研究告诉我们，能感受多种情绪的人比只有积极情绪的人更健康。因为生活本就复杂，既有积极也有消极。有人说自己从不伤心、沮丧，那很可能压抑了负面情绪。但情绪不会无故消失，其影响依然存在。回避面对这些情绪，反而会让身心受到更大伤害。

第二，情绪具有动机功能。例如，当我们迫切想解决某个问题时，常会感到焦虑。焦虑的出现是在提醒你快点处理，事情一旦解决，焦虑感也随之消失。情绪也能减少我们做某事的动机。例如，难题让我们痛苦，所以我们可能逃避它。同样，当一只凶恶的大狗扑来时，你一定会恐惧。这种恐惧才会调动身体躲开它。当你因丢了2000块钱难过、懊悔时，这种情绪会让你记住：以后要小心，钱不能乱放。

其实，消极情绪的存在是为了提醒我们遇到了威胁，需要做出改变：
*   **恐惧**提醒你遇到危险，要逃离。
*   **愤怒**提醒你受到侵犯，可能需要抵抗。
*   **伤心**提醒你可能面临失去，让你花更多心思去关注、珍惜，同时看看能否得到他人安慰。

第三，情绪具有社会功能。正因为绝大多数人的情绪在不同人内心激发的体验相似，所以情绪是人类共同的语言。大家通过这种语言识别对方感受，彼此传递情绪以达成社交目的。

例如，工作上你做得很好，我点头微笑表示赞赏。你看到微笑就知道我很开心、赞赏你，于是你也会回复微笑表示感谢。我们什么都没说，却有了心灵交流。当我看到你眼角挂泪、伤心难过时，我会给你做点好吃的、带你出去走走，或仅仅陪伴在身边，试图让你感到支持与理解，心情慢慢好起来。当你看到老板脸色难看、正在生气时，你就知道今天最好别招惹他，更别提加薪的事。

所以，情绪其实是我们的感受器，帮助我们识别生活中的“糖果”与“危险”。学会更好控制情绪，就是帮助我们更安全地面对生活。

无论是失控的情绪，还是受压抑或过度控制的情绪，都像大海中汹涌的波涛。你就像航行其中的小船，只能在巨大颠簸中艰难前行，一不小心就失去方向，甚至葬身大海。而妥善管理好的情绪就像平静的海面，让你既能平稳航行，又能欣赏海的美丽。

那么，我们究竟如何妥善管理情绪，让婚姻的小船平稳荡漾在幸福的海洋里呢？我从多年观察争吵的经历中找到了方法。

我发现所有人在争吵时，说的每句话几乎都以同一个字开头：“你”。“你不爱我了”、“你不努力上进”、“你就知道花钱”等等。当你说“你怎么怎么样”时，其实就是一种批评、指责的口吻。没人喜欢被批评指责。所以一开口，就已激起对方防御和对抗情绪，很容易造成冲突。

那么，既要表达情绪，又不能批评指责，我们该怎么做？方法很简单：把“你”换成另一个人称代词——“我”。

当我们批评别人时，比如“你不爱我了”，其实是因为内心有一个关于“爱应如何表现”的标准，例如应该体贴、照顾、陪伴等。你经过比较，发现对方行为未达标准，因此才说“你不爱我了”。所以不如反过来，直接说出内心关于爱的正向期待。

例如：
*   “你不爱我了”用“我”开头，变成：“**我希望你回家之后可以多陪陪我，不要总是工作。**”
*   “你不努力上进”用“我”开头，变成：“**我希望你工作能常有进步，不要每天花那么多时间去玩游戏。**”
*   “你就知道花钱”用“我”开头，变成：“**我希望你可以多买一些家里必需的东西，也为孩子将来的学习多存一点钱。**”

当你把“你”改成“我”后，就把批评指责变成了希望和期待。面对希望和期待，大多数人至少不会反感，也不会有太多对抗情绪，因此不易产生矛盾冲突。更重要的是，当你把正向期待告诉对方后，对方也知道该怎么做了。如果能做到，自然会立刻行动；如果做不到，至少也能在操作层面与你讨论。讨论时双方情绪会比较缓和，不至于发生情绪争吵。

当然，如果你的希望和期待值过高，超出正常范围，对方也会不高兴。所以期待还是要合理一点。

因此，这就是我们妥善管理情绪的第一个方法：**停止批评和指责，表达自己的正向期待，尽量在事实层面进行讨论。**

妥善管理情绪的第二个方法，是要敢于向对方说出你的情绪以及情绪的原因。

当你晚上加班到10点，饿着肚子、一身疲惫回到家，却发现孩子躺在地上打滚，玩具扔得到处都是。丈夫躺在沙发上玩游戏，桌上堆满零食袋、外卖盒。显然他没好好做饭，不知又给孩子吃了什么。家里一团糟，丈夫只顾玩，不管孩子也不收拾，一副所有事都等你回来的样子。想想自己加班辛苦、这么晚还饿着肚子，回家还要收拾烂摊子，你一定会非常生气。

这时，你可以板着脸生闷气，等丈夫在游戏间隙注意到你。但专心游戏的他很可能毫无觉察。你也可以冲丈夫大发脾气，指责他只知道玩、不管孩子不顾家。但估计丈夫会和你摆事实讲道理，告诉你他如何管孩子、管家，或干脆和你对吵。

所以，你不如用相对平和的语气，直接把你正在生气和生气的原因告诉丈夫。你可以对他说：“**我现在其实很生气，因为我加班到现在还没吃晚饭，本来就已经又累又饿。结果回家发现桌上地上到处都是乱糟糟的，孩子本来也该快睡觉了，结果到现在还没刷牙洗澡，这些都让我感到更累更生气了。**”

但请注意，说这些时一定要保持比较平静的状态，而非训斥口吻。因为这是在向丈夫解释和表达你的情绪，让他知道你为何生气。这样不仅能让他知道生气原因，也让他知道你希望他做些什么来改变状态。我猜即使丈夫没能力立刻做一顿热腾腾的晚饭，至少也会有所行动，赶紧停下游戏，把桌上地上收拾干净。

当然，如果你实在无法克制情绪，一开口就可能是批评指责，那你可以做第三件事：先不要急着开口说话，而先做三个又深又慢的呼吸。

我推荐你用**12秒呼吸法**：用**6秒**慢慢吸气，再花**6秒**慢慢吐气。这样三个深呼吸会让情绪有所缓和。然后你可以再告诉自己“生气解决不了问题”，或喝一杯温水。总之，等自己平静下来，再去和丈夫做必要沟通。

第四，如果对方情绪失控，那你更不必接他的招。可以勇敢告诉他：“**我知道你现在正在某种情绪当中，但这样的情绪并不能帮助我们解决问题。所以看起来现在我们不是讨论问题的时候，还是等你平静下来我们再沟通吧。**”这样既可以拒绝对方对你的消极影响，又做出了榜样，让对方知道何为有效的沟通方式。等到对方冷静下来，能和你相对平和对话时，你再使用前面说的三种方法，就可以顺畅沟通了。

本节课我们一起学习了情绪的重要意义。它不仅有适应功能，帮助我们适应生活环境；还有动机功能，促使我们采取行动；也有社会功能，是人际交流的重要工具。因此，过度压抑情绪和过度发泄情绪都会对亲密关系产生破坏作用。所以情商中提到的“妥善管理情绪”变得尤为重要。

那么，怎样才能做到妥善管理情绪呢？

首先，当你有情绪时，可以通过深呼吸等方法先让自己平静下来。

第二，要敢于向对方说出你的情绪以及情绪的原因。

第三，在沟通中不要带有强烈的批评和指责，而是表达自己的正向期待，尽量在事情层面进行探讨。

最后，要敢于拒绝对方用情绪来影响你，要促使对方也在相对平和的状态下与你沟通。

面对变幻莫测的情绪海洋，你其实无需惊慌失措。只要掌握好情商之舵，一定能帮助你乘风破浪。

今天就到这里，我们下次再见。
