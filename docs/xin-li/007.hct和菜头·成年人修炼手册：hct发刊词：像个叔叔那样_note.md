# 课程007：hct和菜头·成年人修炼手册：发刊词——像个叔叔那样 👨‍🏫

在本节课中，我们将一起学习何菜头老师《成年人修炼手册》专栏的发刊词。这篇文章回顾了作者开设专栏的初衷，并借由“叔叔”的角色，探讨了成人应如何与年轻一代进行平等、真诚且有深度的交流，以帮助他们更好地理解世界。

## 概述

五年前，何菜头老师开设了第一期专栏。当时，适逢罗振宇先生迎来一对双胞胎女儿。于是，那篇发刊词是写给这两位小朋友的，欢迎她们来到人间。

时光飞逝，如今两位小朋友已到了快上学的年纪。罗振宇先生再次找到何老师，询问他是否应该为此做点什么。

## 关于“叔叔”的思考

上一节我们了解了专栏重启的背景，本节中我们来看看何老师对于“叔叔”这一角色的思考。

何老师回忆起家乡的风俗：孩子上学前，叔叔们需上山寻找一种特别的藤条，处理后作为礼物送上以示祝贺。这种藤条抽打起来很疼却不伤皮肤，且经久耐用。

**公式：** `1981年的叔叔行为 ≠ 2021年的叔叔行为`

不过，那是1981年的叔叔们该做的事。2021年的叔叔若还这么做，恐怕会失去朋友。那么，现在的叔叔究竟能做什么呢？

## 记忆中的两位叔叔

何老师回放了自己的童年记忆。在所有叔父辈的人中，有两位令他印象最深。

以下是关于这两位叔叔的回忆：

*   **第一位叔叔**：经常给他买吃的，让他骑在脖子上爬山，并允许他摆弄自己的海鸥牌照相机。这类叔叔很受欢迎，但太费钱和体力。考虑到罗振宇家是双胞胎，开销需加倍，何老师自觉难以承受。
*   **第二位叔叔**：是何老师母亲的姐姐的丈夫的妹妹的丈夫，关系较远，暂称姑父。这位姑父年轻时一表人才，下班后做饭，还找小黑板给自家孩子补习，耐心温和。

何老师与这位姑父比较聊得来，喜欢他说话的语调和口气。

## 一次关键的对话

上一节我们认识了两位风格迥异的叔叔，本节将聚焦一次改变何老师认知的对话。

有一次，何老师买了一本《小学生数学手册》，自觉一次买齐了六年的数学知识，便去姑父家炫耀。

姑父接过书，仔细翻阅目录和部分内容后，将书还回并说道：“宁可少而精，不要大而全。这种大而全的东西不好，因为其中每一点都会讲得很浅。”

这句话何老师至今记忆犹新，原因如下：

以下是这次对话带来的三点启示：

1.  **对话态度**：姑父说话的态度，如同在与另一位成年人对话。
2.  **对话内容**：他在明确表达一个观点，并附带了论证，讨论的是事物的本质。
3.  **对话立场**：他完全不在意何老师听后是否会沮丧或生气，他在意的是是否说了实话，而非用好听的话哄骗。

何老师当时愣住了，一方面不太高兴，另一方面又觉得对方在讲道理。最关键的是，这种对话方式让他感到自己受到了尊重，并渴望进行更多探讨。

## “叔叔”角色的本质

很多年过去，何老师买书时仍会想起姑父的话。他承认姑父的观点未必完全正确，因为“博”与“精”本是个人发展的平行路径。

但是，当姑父在他面前掀起世界的一角，分享自己所认为的事物本质时，那一幕令他终生难忘。姑父打断了他的沾沾自喜，直接给予了一种成人对生活的领悟，迫使他停下来思考。

**代码：** `if (分享 == 事物本质) { 触发思考(); }`

这促使何老师在今后的成长中，也时常忍不住去掀起生活的地毯，在下面寻找自己认为的事物本质，同样不去管生活是否“高兴”。他觉得自己有可能成为姑父那样的叔叔。

## 专栏的使命与愿景

五年后，罗思思和罗维维也到了上学的年纪。学校生活是他们接触坚硬人生的开始，过程会越来越艰难和充满挑战。

何老师希望以“何菜头叔叔”的身份，通过文字在他们的成长道路上陪伴一段时光。他认为，在孩子的成长中，除了父母和老师，总需要一个类似叔叔或舅舅的角色。

这个角色身为成人，却能作为平等的朋友与孩子聊天，讨论学习生活之外的话题，尤其是未来成人世界的模样，为孩子的人生路做预先准备。

在孩子与成人之间，除了教育、命令与服从，还应存在另一种更轻松的相处方式。其中除了关切和诚恳，还应有平等和尊重。

以下是何老师希望通过专栏达成的三个核心目标：

1.  **理解成人思维**：希望孩子能因此理解成人如何思考，从而明白自己为何被如此对待。
2.  **理解世界运转**：希望孩子能因此理解成人世界如何运转，从而在未来避开内心与外界的致命打击。
3.  **培养宽容视角**：若能达成，更希望孩子能用更宽容的视角观察世界，而非过早吸收成见，将自己局限在狭小世界里。

如此，当这段人生旅程需要挥手道别时，他可以放心地看着孩子们投入自己的人生，知道他们能在未来寻找到自己，体会更多快乐与趣味，抵达他所不能抵达的地方。

## 礼物的准备与分享

当然，现在让两个五岁小女孩阅读这些文字可能为时过早。但何老师相信，她们会慢慢长大。在这一天到来之前，作为叔叔最好提前准备好礼物。这样，比她们年长的小朋友甚至大朋友可以先睹为快。

也许需要叔叔的并不只是她们两个。也许人人身边都需要一个像叔叔那样的人。也许对成人世界，有些人需要预习，而有些人需要复习。

所以此刻，何老师有一种酿酒师的心情。即便时光流逝，也不再那么令他畏惧。等到那一天，这些文字穿过漫长时光，加上读者的评论与讨论，最终抵达孩子们面前时，这将成为她们生命中最特别的礼物。

因为曾经有那么多的人、头脑和心灵，彼此紧密联系，共同创造出一份通向成人世界的指引，并如此耐心地等待两位小朋友长大成人，为她们修筑道路。

**引用：** “在这条道路上，能看到曾经有许多双脚掌留下的印记。那些印记指向远方，就像是我们当初沿着前人的足迹走入新世界一样。”

## 总结与启程

本节课中，我们一起学习了何菜头老师《成年人修炼手册》的发刊词。我们探讨了“叔叔”这一角色在引导下一代时的核心价值：真诚、平等、尊重，以及分享对事物本质的洞察。

这远比他任何一次创作都要艰难。但在多年的人生经历中，他也学会了一点：不妨先制定宏远的目标。最后即便无法完全抵达，也能多走出去很远的一段。后来的人因为这多出来的一段距离，最终可能真的抵达目的地。

那么，现在就让我们一起出发吧。这是一个叔叔应该干的事情。
