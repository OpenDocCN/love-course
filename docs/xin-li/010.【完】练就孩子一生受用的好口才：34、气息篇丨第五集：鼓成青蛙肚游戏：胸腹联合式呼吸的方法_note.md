# 口才基础课 010：掌握胸腹联合式呼吸 🐸

在本节课中，我们将要学习口语表达中至关重要的基础——**胸腹联合式呼吸**。我们将通过几个简单的小游戏，帮助你理解并掌握这种能让声音更有力量、更富感染力的正确呼吸方法。

---

## 课程概述

上一节我们介绍了单纯使用胸式或腹式呼吸的局限性。本节中，我们来看看如何将两者结合，形成更高效、更有力的呼吸方式。

胸腹联合式呼吸是优秀口语表达的基石。它能让气息更深、更稳，从而支撑起饱满、响亮且富有情感的声音。没有掌握这种呼吸方法，你的声音吸引力和感召力就会大打折扣。

---

## 核心呼吸法：胸腹联合式呼吸

其核心要领是：吸气时，**胸腔**（肋骨区域）与**腹腔**（腹部）**同时扩张**。这可以形象地理解为，让你的胸部和肚子像青蛙呼吸时一样一起鼓起来。

公式表示为：
**有效呼吸 = 胸腔扩张 + 腹腔扩张**

---

## 练习游戏：体会正确呼吸

以下是三个帮助你找到并巩固胸腹联合式呼吸感觉的小游戏。

### 游戏一：鼓成青蛙肚

首先，我们来玩“鼓成青蛙肚”的游戏。这个游戏的重点是同时关注腹部和胸腔的起伏。

*   **动作要领**：
    *   用鼻子深深吸气（不要用嘴）。
    *   感受腰带因腹部鼓起而变紧。
    *   **同时**，感受你的肋骨向两侧扩张，胸腔也微微鼓起。
    *   用嘴缓慢地将气吐出。
*   **关键指标**：必须同时满足“肚子鼓”和“胸腔（肋骨）扩张”。只鼓肚子是腹式呼吸，并非真正的胸腹联合式呼吸。

### 游戏二：闻花香

如果你对第一个游戏感觉抽象，可以试试“闻花香”。想象一朵香喷喷的花就在你面前。

*   **动作要领**：
    *   自然地用鼻子去深深闻花的香气。
    *   在闻的过程中，体会气息沉入腹部，同时胸腔打开的感觉。
    *   同样检查那两个指标：腹部是否鼓起？肋骨是否向外扩张？
*   **效果**：当你的胸腔和腹腔都充满气息时再发声，声音会格外悦耳。

### 游戏三：闻红烧排骨

不爱闻花？没关系，想象一盘刚出锅、香喷喷的红烧排骨就在你鼻尖前。

*   **动作要领**：
    *   像闻到极致美味一样，用鼻子深深吸气。
    *   专注于享受“香味”的同时，感受胸腹联合扩张的感觉。
    *   这本质上和闻花香是一样的，都是利用我们生活中本能的深呼吸动作来体会正确方法。

## 结合发声的巩固练习

通过上面的游戏，我们体会了静态的呼吸。接下来，我们将呼吸与简单的发声结合起来，进行巩固。

我们来练习发“哈”音，模仿京剧老生笑的感觉，目的是体会“气沉丹田”（气息沉到肚脐附近）的力量。

*   **练习步骤**：
    1.  坐直，全身放松，肩膀不要紧绷。
    2.  用鼻子进行**胸腹联合式吸气**，让腹部和胸腔扩张。
    3.  连续、有力地发出：“哈！哈！哈！哈！”
*   **关键感受**：
    *   将手放在肚脐（丹田）位置。
    *   发音时，你会感觉到腹部有节奏地“一鼓一瘪”。（吸气鼓，发“哈”时瘪）
    *   反复练习：`吸气鼓腹 -> 发“哈”收腹 -> 再吸气鼓腹 -> 再发“哈”`。
*   **代码式描述**：
    ```
    while 掌握呼吸法 == False:
        吸气(鼻子， 胸腹联合扩张)
        发声(“哈”， 腹部收缩)
        检查(腹部起伏， 胸腔扩张)
    ```

这种练习看似有些奇怪，但却是播音专业的基础训练。请放心练习，将其视为一个有趣的小游戏即可。
