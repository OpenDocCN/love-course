# 情感心理咨询技能大全（完结）032：第07章：策略式家庭治疗 — 09.策略治疗的家庭作业技术 🏠

在本节课中，我们将要学习策略式家庭治疗中的一项核心技术——家庭作业。我们将探讨其背后的哲学理念、布置原则以及几种具体且实用的家庭作业技术。

## 概述：家庭作业背后的哲学思考

上一节我们介绍了策略式家庭治疗的基本框架，本节中我们来看看其核心干预手段之一：家庭作业。在讲解具体技术之前，首先需要理解家庭作业背后的哲学思考，这涉及到策略治疗和系统治疗对于“改变”的基本看法。

传统的观念可能认为，心理治疗的关键改变是发生在治疗室当中的。持有这一观点的最经典治疗是精神动力学治疗。精神动力学治疗的频率会比较高，一般可能一周两到三次，高一点的话可能每天一次。之所以采用这样的设置，是因为精神动力学治疗认为，患者每一个细微的无意识感受和认知的改变，需要在治疗当中通过和治疗师的每一个细节的对话，一点一点去积累。

但是家庭治疗对于改变的观念不一样。家庭治疗里面有一句经典的话，那就是认为，关键的改变不是发生在治疗室里面，而是发生在生活当中。

家庭治疗为什么会这样认为呢？因为家庭治疗有一个核心的信念：家庭是一个完全不可能被人为控制的、有自主运行能力的系统。治疗师能做的事情，就是在治疗当中，去对家庭这个系统运作的规则进行一些探讨，然后实施一些必要的干预。这就好像治疗师是拿着一根棍子，家庭是一个水池，治疗师拿着棍子在这个水池里搅一搅，把水池原本的规则改变和扰动一下。那之后，这个家庭在生活当中会变成什么样，就让家庭自己去寻找适合他们的新方法。

为什么要这样做呢？举个例子，比如一对夫妻来做治疗，他们带来的问题是：“我们两个婚姻生活很不幸福，我们觉得我们根本不是一路人，我们需要改变。”谈到“改变”这个词时，有的治疗师可能就会认为，改变就是你们和好如初，于是把治疗目标定为让你们和好如初。但是这位治疗师可能会发现，他跟这个家庭谈得越深，越发现这对夫妻不是一路人。最后他可能就会怀疑：我们治疗的目标真的是要让你们和好如初吗？也许你们好聚好散也是一个选择。

这就提示我们，我们原本认为的合适规则，在不同的家庭身上不一定适用。也就是说，虽然我们的理论描述很好，觉得和睦的家庭就应该是夫妻彼此支持，孩子独立地发展自己，但是这样的一种改变，并不是在所有的家庭里面都能达成的。所以，这对我们咨询师的哲学思想提出了一个本质的挑战：我们能做到的工作有多少？我们是不是一定要要求家庭在咨询室里就发生改变？

因此，系统治疗和策略治疗提出：我们治疗的目的，不是让家庭在治疗室里就马上变成我们希望的样子。我们只需要去给他们布置作业，提供必要的干预。他们拿着这些干预带来的影响，自己回去寻找他们新的改变出路就可以了。而且家庭治疗师也相信，每个家庭都有自己改变的资源。家庭治疗师的目的，只是扰动和开启改变。

我曾经见过一个家庭。那个家庭刚来的时候，女儿看上去病得很重，17岁的小姑娘抽烟、喝酒、不上学，堵在学校门口欺负同校的女同学。爸爸妈妈都是很有教养、知识水平很高的国企干部和政府官员。爸爸非常焦虑和绝望，想不通女儿怎么会这样。最开始来做治疗时，他还拼命跟我要微信和手机联系方式。我坚持没有给他，跟他说：“我知道你很着急，但这个过程需要慢慢来，你们就按照我们的约定定期来见我就好了。”有时候治疗期间，女儿的情况出现反复，爸爸也会想方设法通过介绍人想找到我。但我的答复永远是：“情况我知道了，下次见面时我们再讨论。”同时也一再叮嘱那个家庭，在家里继续做我布置的作业。

后来你会发现，当你真正尊重并相信那个家庭有改变资源的时候，那个家庭的资源自己就出来了。后来那个爸爸想了个办法：他找了学校里女儿比较信任的几个高年级学长，和他们建立了很好的关系，然后请那几个学长站在不远不近的距离，去帮助和关心他的女儿。你看，每个家庭都有他们自己改变的资源。那个爸爸最开始想直接控制女儿，但发现没用。当你真正不去命令他的时候，那个家庭自己的资源会出现。

正是基于这样一些相信家庭存在资源、也认为治疗师的作用不是要去教家庭应该怎么改变的哲学理念，同时也是在认为我们没有办法去命令家庭的这样一个治疗性哲学观的前提下，才有了“家庭作业”这个治疗方法的产生。实际上，家庭作业最开始的起源是来源于认知行为治疗。家庭治疗在发展过程中，也受到认知行为治疗的影响。所以说，家庭治疗实际上是一个整合的心理治疗方式，基本上是以探讨家庭关系为核心，在这个过程中，哪些方法有用，我们就拿过来用。

## 布置家庭作业的原则

在了解了核心理念后，我们来看看布置家庭作业需要遵循哪些原则。家庭作业一般是我们留给家庭在两次咨询会谈的间隔期中，需要完成的一些行为任务。

以下是布置家庭作业时需要遵循的几个核心原则：

1.  **资源取向**：作业要关注家庭的资源和优点，而不是用批评、攻击、指责的方式来布置。例如，我们经常会布置家庭一个“记红账”的作业，让他们去记录彼此的优点和在治疗间歇期的改变。我们很少让他们去记录彼此的缺点，因为他们一般在家里已经记够了彼此的缺点。
2.  **形象、具体、有趣**：作业要比较形象、具体和有趣，不能布置那种文绉绉、枯燥无味的家庭作业。
3.  **针对元沟通层面**：作业一般要针对家庭的“元沟通”层面来开具。元沟通（关于沟通的沟通）是我们之前课程中讲过的概念。
4.  **基于评估**：家庭作业要建立在对家庭关系和规则评估的基础上来布置，不能什么都不知道，就给别人盲目地乱布置作业。
5.  **基于良好关系**：家庭作业要建立在和家庭比较良好的治疗关系的基础上。

## 常见的家庭作业技术

理解了原则之后，我们进入核心部分，看看策略治疗中有哪些具体、常见且有效的家庭作业技术。

### 1. 单双日作业 📅

单双日作业是一种行为作业。这种作业一般会用在这样的家庭身上：家庭表现出一种明显固化的生活方式。例如，爸爸妈妈面对16、17岁青春期的孩子时，特别想控制孩子，但孩子又特别反抗，双方都坚持自己的生活方式时，我们可以去布置这个作业。或者，有时我们讨论发现，孩子的“厌学”有一个好处是可以维持父母的婚姻。那我们也不去跟父母或孩子说“你要继续上学还是不上学”，而是布置一个单双日作业，让这个家庭体验一下，在不同的时间里，用不同的方式生活，他们的感受会有什么不一样。

这个作业一般我们会这样布置。比如，对于想控制青春期孩子的父母，我们可能会说：“你们回去以后，周一到周日，其中选三天（比如周一、三、五）。在这三天当中，爸爸妈妈你们对待你们的儿子，要像对待四、五岁的小宝宝那样对待他。他早上起来一睁眼，你们就要开始帮他穿衣服，要问他今天想吃什么饭，他不管去哪里玩，你们都要跟着他，告诉他你们很担心他，要为他的所有事负责。那余下的三天（比如周二、四、六），你们就把他真正地当做一个17岁的大男孩、大小伙子来看。同时你要跟那个大小伙子说，在‘小宝宝日’，你就要配合你的父母做一个小宝宝；在‘大人日’，他们真的把你当大人看的时候，你也要配合他们，把自己变成一个真正的大人，你要为自己未来的计划、学习、每天的生活负责，自己的衣服要自己洗等等。”你就让他们去体会在这两种不同的生活方式下面，他们的感觉和心情有什么不同。

然后剩下一天星期天，你可以跟他们说：“星期天呢，你们自己决定。如果这个小伙子你觉得当个小宝宝好，你就在星期天想尽办法表现得像个小孩子；你们父母如果星期天觉得把他当小宝宝看，你们也什么都不用说，就想尽办法通过你们的行动，把他变成一个小宝宝。你们都不用过多解释，只要行动就好。当然，如果星期天这个小伙子觉得当一个小宝宝太难受了，父母也觉得把儿子当小宝宝太憋屈了，那你们就把他当成一个大人看。”

这就是所谓的单双日作业。这个作业有一个作用：通过完成这个作业，可以促使家庭对两种截然相反的生活方式进行体会和比较，进而帮助他们对自己原本僵化、不合理的互动行为产生领悟，并且自行选择进步的方向。可能做完这个作业一对比，父母就觉得：“比起把他当小宝宝，还是把他当大人的生活更轻松一点。”孩子也会体会到：“原来我的一些行为，也在配合我的父母把我当小宝宝来看。”

### 2. 记红账 📖

记红账就是治疗师会要求参加治疗会谈的家庭成员，在回去以后各准备一个小笔记本，让他们在笔记本上面记录从这次见面结束到下次见面开始之间，他们看到彼此身上好的变化。我们会跟他们说：“你们回去准备那个小本本，可以刻意地提醒将要被记录的那个成员，跟他说：‘你记住哦，刘医生给我布置的作业是记录你好的地方，你注意一点。’”就是只准记录彼此好的地方，不能记录坏的地方。

这样的作业完成以后，我们可以在下一次治疗开始时，让家庭成员把他们的作业带过来。然后我们在治疗里面，和他们一条一条地去回顾作业的完成情况。你和他们成员去回顾这个作业完成情况的过程，就是帮助他们为互相点赞、认可彼此的过程。

我曾经见过这样一对夫妻。丈夫到了40多岁以后，莫名其妙地出现身体上的各种不舒服，去很多医院看了也没有任何问题。丈夫没办法上班，每天在家里当个“家庭主夫”。妻子对丈夫是一肚子怨言，觉得丈夫又生病又不赚钱，但正因为觉得丈夫生病，妻子的怨气又没办法直接表达，所以带来做治疗。第一次治疗后，我布置的作业就是记红账，让妻子和丈夫各自记对方身上20条好的变化。第二次来治疗时，我当着他们的面，把妻子写的丈夫的好处一一念出来。结果丈夫听完后非常感慨，他说：“以前我只觉得我老婆特别看不起我、鄙视我，现在看到她记的东西，我发现实际上在她心目中我还是蛮有价值的。”后来先生跟老婆说：“你知道吗？实际上我身体不舒服，很多时候就是当你否认我的时候，我就会特别不舒服。实际上我的不舒服是想跟你说，我希望你更多地认可我。”这就是记红账作业的作用。

### 3. 悖论处方 🔄

悖论处方的作业我们之前课程已经专门讲过。悖论处方作业有好几种。

*   **第一种**：治疗师会故意鼓励家庭中的某个成员保持或夸大他们的症状行为、问题以及当下有问题的互动模式。这种夸大可能会大到一种家庭自己都觉得荒谬的地步，进而促使家庭自发地对这些症状产生抵触和厌恶的情绪，从而达到让其改变的目的。
    *   **例如**：我曾见过一个男孩，他特别在意别人对他的看法，总觉得同学在背后议论他。我给他家庭布置了一个悖论作业：回去后，和父母共同商量制定一份调查问卷，目的是发给他的同学，问卷上要问的内容就是“究竟有多少同学那么在乎他这个人本身到底是好还是不好”。因为这位学生觉得所有人都在乎他的表现，所以他无法自如。我让他和父母去做这份问卷，印200份，周一上学时发给所有认识的同学，看看到底有多少人真的那么在意他。最后他真的去做了调查（只对同班同学），结果发现，好像并不是那么多人关注他，大家实际上都很忙。
*   **第二种**：让家庭在每天选择一个固定的时间，在这个固定的时间里，让他们的症状和问题集中出现。
    *   **例如**：我曾面对一个强迫症的小姑娘，她说自己控制不住症状，想摆脱但没办法。我跟她说：“你不用急着要摆脱你的强迫症状。你今天回去以后，每天选固定的一小时，在那一个小时里，你什么都别干，就想方设法把你强迫症的思维和洗手行为完全地表现出来。唯一的要求就是每天那一小时都要有，并且要在固定时间（如下午3点到4点）。即使某一天3点到4点你觉得不想让症状出来，不行，你也得想方设法让它出来。就算你自己没办法让它出来，你就打电话给你的父母，让他们帮你想办法把症状引出来。”后来她回去做了两个星期，回来跟我说症状反而减轻了。她说：“当我真正想去让那个强迫症状出现的时候，我发现实际上我越想让它出来它越不出来，我越不想让它出来它反而就越出来。”
*   **第三种**：让家庭回去以后讨论一下，他们的症状有什么好处。
    *   **例如**：对刚才那个强迫症的小姑娘，我给他们家庭布置了一个家庭会议作业：每两周当中每周选一个小时，一家三口坐下来讨论一下，这个强迫症状出现以后，给家庭带来了什么好处，要讨论出十条好处并写下来。后来他们真的做了讨论，总结出十条好处。其中有五条是：症状出现后，妈妈不再唠叨了，爸爸不再回避了，他们开始有意无意地要讨论“我”的病，我们家反而因为“我”的病有了新的沟通话题。另外五条涉及小姑娘自己：她觉得有了这个病以后，就不用天天去学校了（因为她不太喜欢那个学校），现在终于有了一个正当理由可以不去学校。这样布置作业的过程本身，也是帮家庭意识到症状的功能和意义。

悖论处方是把家庭放在一个“不会输”的位置上。也就是说，如果他按照你的处方去做作业，他的症状也得到了表达，并且是在规定时间内按照他的意志在表达，那就代表他对症状有控制能力。如果有的家庭听了这个作业觉得太愚蠢了不要做，那他不做的话，他的症状也就（因为被挑战而可能）发生变化。

当然，给别人布置了这样的作业以后，下一次再见他们时，你需要和他们去讨论他们做这个作业的过程、收获和感受。

### 4. 水枪射击与橡皮筋游戏 🔫

这个行为作业可以直接对家庭中一些成员不希望出现的行为进行中断处理。例如，一个家里，妈妈特别唠叨，爸爸又特别回避，他们都不希望看到对方这种行为。这时我就会布置一个作业。

我会跟那对夫妻说：“你们回去一个人买一把水枪，水枪里面装满水。在家的时候就把水枪别在腰上。如果丈夫你发现你的妻子回去以后唠叨又超过了五分钟，你就拿水枪往她脑门上射那么一枪。如果妻子你把水枪别在腰上，你发现你回去以后，相同的话说了两遍你的丈夫还是像没听见一样，你也拿水枪往他的脑门上来那么一枪。”这样的方式，是一种有趣的、温和的、开玩笑的方式，用射水枪来向其他成员传递你想表达的意图。

因为这对夫妻之前，妻子一看到丈夫回避，就拼命怒吼；她越怒吼，丈夫就越回避。他们原本是一追一逃。现在我们用这种水枪射击的游戏，让他们用一种有趣的方式，代替原本的怒吼和逃避来表达他们的需要。

大家注意到，我们布置的家庭作业都是比较好玩、有趣的。有时候天气冷，没办法射水枪，可以用橡皮筋。原理是一样的：如果妻子唠叨超过五分钟，丈夫就用橡皮筋弹她脑门一下；如果丈夫又回避了，妻子也弹丈夫脑门一下。这样的一种方式，就是用有趣的方式，对家庭的某些不合理行为进行扰动和挑战。

### 5. 书信与电话作业 ✉️

这种作业对于居住在外地、不容易规律参加治疗的家庭比较有用。例如，这个家庭来见你一次很不容易，你给他们布置了作业，或者见了他们几次以后，就让他们回去自己改变了。这时你可以和家庭做一个约定。

你可以说：“半年以后，比如女儿或者父母，你们可以给我写一封信，在信里说一下你们最近家庭的变化以及作业的完成情况，或者打个电话给我报告一下。”这个作业本身可以让家庭觉得他们和治疗师还有连接，同时也可以把治疗干预、治疗师在咨询里说过的话的影响，在无意识层面延伸到治疗室之外。那样的话，这个家庭心里面就留了一个印象，他们就知道：“不管我们做什么改变，最后都需要和某某治疗医生（比如刘亮医生）有一个总结。”

我曾经见过这样一个案例。一个抑郁症的小姑娘在治疗里说：“我的抑郁症就是有一个作用：如果我发现我爸不对我妈妈好，不懂得怎么去疼爱我妈妈的话，我的抑郁症是不会好的。”因为生病之前，她爸爸非常粗暴，经常对她妈不是骂就是动手，对她也是。后来她生病以后，爸爸就变温柔了。小姑娘说：“我不敢让我的病好，我特别担心我的病好以后，我爸爸又恢复原来那种粗暴的本性。”后来我跟他们谈过两次以后（因为是外地的），我就跟小姑娘说：“这样吧，你们回去三个月以后给我写封信，跟我说说你们家里的变化，再来说说你现在的决定是什么。你是决定继续要用生病来管理你的父母感情，还是说你发现他们现在可以自己管理感情了，你要让自己病好继续上学？”三个月以后，那个女孩给我发了一封电子邮件，她说：“现在虽然我的父母还经常吵架，但是我发现他们有一个很大的进步。以前我觉得他们是两个任性的小孩，一吵架就只会歇斯底里地发作。现在他们虽然有时候也会发作，但是也会去就事论事地讨论他们之间的一些问题。所以我做了一个决定，我决定把他们的冲突还给他们，我决定变回一个健康的正常女学生，我要去继续完成我的学业。”这就是家庭作业的力量。

### 6. 仪式化作业 🕯️

有时我们也会给家庭布置一些仪式。例如，我们会刻意针对她的症状，开具一些针对性的仪式，帮助家庭调整他们之间的互动规则。

*   **在缺乏情感支持的家庭**：我们可能会布置这样的作业：“你们回去以后，每周要用一个小时的时间坐下来，彼此成员分享一下最近各自的心情，以及自己对家庭里其他成员的期待。”通过这样的仪式可以促进家庭反思他们当下模式的合理性。
*   **针对长期患病者**：例如，我曾给一个长期患躯体形式障碍的患者布置仪式：“你这个病病了那么多年，也花了很多钱。所以你回去以后，到半年后你要举行一场派对。你要在派对上跟所有人分享，你生了这么多年的病有什么好处。分享完好处以后，你要当着所有人的面做一个决定：一个决定可以是‘这个病实在是病得好，本人以后要继续当个病人享受别人的关心’；另一个决定可以是‘这个病我不想要了，我要把它放一边去过一个正常人的生活’。”最后那个人真的在半年后搞了一场派对，在派对上还专门请了一个人来表演他的病，跟那个病做了一个告别。

当然，有时我们也会布置一些丧失告别和创伤告别的仪式给家庭，不过这不是本次课的重点。

### 7. 未来计划作业 🗺️

最后一个常用的作业，是针对那些处于家庭生命周期中要独立离家阶段的儿童青少年，或者对于那些对未来计划不明确的“空心病”学生。我们经常会布置这样的作业。

例如，很多厌学的孩子，他们之前把所有心思都放在父母的冲突上面。这时我就会给他们布置作业：“你想一想，未来五年内你想做的、一直想做但没机会做的、你觉得有意义的事情是什么？未来长大以后你自己想达成的三个目标是什么？”当这个来访者在做这个作业的时候，就是他作为一个独立的个体，在思考他个人未来人生计划的过程。这个作业也是促进家庭成员自我分化的作业。
