# 课程118：60秒口才训练营（完结）付费答疑05：如何引导孩子练习口才？👨‍👦

在本节课中，我们将探讨一个家长们普遍关心的问题：如何有效地影响并引导自己的孩子也开始练习口才。我们将分享具体、可操作的方法，帮助你在家庭环境中培养孩子的表达能力。

---

上一节我们讨论了成年人练习口才的挑战，本节中我们来看看如何将这种练习延伸到孩子身上。

我一直持有这样一个观点：口才若能从孩童时期开始培养，其效果远胜于成年后再练习。原因在于，孩子通常没有成年人那么多的顾虑，例如“爱面子”的心理。成年人常受时间和精力的限制，而孩子在这些方面则更为自由。然而，遗憾的是，我们的传统教育体系虽然教授语文多年，却很少系统训练公众讲话和流利表达的能力。

那么，作为家长，你该如何引导孩子从小练习口才呢？许多学员的第一反应是直接与孩子沟通，要求他们一起练习。但我建议不要这样做。

以下是第一个核心方法：身教大于言传。

*   **具体做法**：在你练习60秒口才时，选择孩子在家的时候进行。关键的一步是，**邀请你的孩子作为你练习的听众**。每当你完成一段60秒的讲述后，主动询问孩子：“作为听众，你觉得我刚才讲得怎么样？有什么建议可以给我吗？”
*   **核心理念**：这个行为的重点不在于孩子的反馈内容，而在于你主动放低姿态，与孩子建立平等沟通的关系。家长与孩子沟通出现问题，往往始于姿态不对——总是高高在上地指挥、命令或指责。没有人喜欢被命令和指责。当你以邀请和请教的态度与孩子互动时，会深刻影响亲子关系。在练习过程中的这种交流，不仅能改善关系，还会自然而然地激发孩子对“口才练习”这件事的好奇和兴趣。孩子天生对新鲜事物感兴趣，也倾向于模仿父母的行为。

除了上述的核心方法，我还有两个额外的建议，适用于不同年龄段的孩子。

以下是两个辅助性的引导策略：

1.  **转变睡前故事模式**：如果你的孩子年龄较小（例如五六岁），可以改变传统的“父母讲，孩子听”的睡前故事模式。尝试与孩子**互相讲故事**。你可以先请孩子给你讲一个故事（可以是书上的，也可以是学校发生的趣事），然后你再给他讲一个。不要小看讲故事，它是训练口脑协调能力最基础、最重要的练习之一。让孩子从小有机会表达，并充当TA忠实的、鼓励性的听众，至关重要。
2.  **创造“小主播”身份**：你可以帮助孩子在一个音频平台上注册一个账号，引导TA成为一名“小小主播”。初期可以录制“声音日记”，说说心里话。之后，可以分享TA学到的知识、读到的故事。虽然不专业，但“主播”这个身份能在同学中带来独特的自信。这种角色转变和公开分享的体验（即使设置为私密，对孩子而言也是“发布”），会显著提升TA的表达意愿和能力。

---

本节课中我们一起学习了引导孩子练习口才的三个递进式方法。

**总结如下**：
1.  **根本方法（身教）**：自己练习时，**邀请孩子作为听众并寻求反馈**，以平等姿态激发其兴趣。
2.  **基础训练（互动）**：通过**与孩子互相讲故事**，锻炼其最基础的口脑协调与表达能力。
3.  **动力激发（角色）**：帮助孩子打造一个“**小小主播**”的身份，通过分享获得自信，从而持续提升表达力。

引导孩子从小练好口才，是为了让TA在未来面对学习、工作与生活的各种挑战时，能够自信、清晰地表达自己，避免因表达能力不足而陷入困境。请记住，引导的前提是你自己先开始练习并乐在其中。
