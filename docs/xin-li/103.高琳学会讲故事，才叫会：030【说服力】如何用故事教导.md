# 课程103：📖 学会讲故事，才叫会沟通（完结）：030【说服力】如何用故事教导孩子下属，而不是讨厌的说教

在本节课中，我们将要学习如何运用讲故事的方法，有效地教导他人，避免陷入令人反感的直接说教。我们将探讨几种常见的无效教导方式，并学习两种高效的故事教导法。

## 常见的无效教导方式

上一节我们介绍了课程的目标，本节中我们来看看在教导他人时，哪些方式通常是无效的。以下是三种常见的、效果不佳的教导方式：

1.  **否定式教导**：当发现下属或孩子出错时，直接进行批评。这种方式会打击对方的自信，并常常伴随问题扩大化，例如使用“你怎么总是……”这类句式。

2.  **比较式教导**：常见的句式是“你看看人家谁谁谁……”。这种方式会严重挫伤对方的积极性，并造成无谓的比较伤害。

3.  **炫耀式教导**：讲述者沉浸于“想当年我如何成功”的故事中。这类故事往往起不到教导作用，更多是讲述者的自我陶醉，并可能引发对方的逆反心理。

## 为何故事是更好的教导工具

了解了无效的方式后，我们自然会问：那应该用什么方式来教导呢？让我们先看一个社会科学实验。

实验发现，当孩子直接告诉吸烟者“吸烟有害健康”时，遭到了拒绝。但当孩子先向吸烟者借火被拒，并由此引发关于吸烟危害的对话后，吸烟者反而更愿意接受戒烟帮助。

这个实验揭示了一个核心原理：很多时候，人们做得不好，**并非因为缺乏能力或知识，而是因为缺乏动力**。当动力不足时，强行灌输信息（说教）只会引发抵抗。

而故事是一个强大的工具。它不会强行说教，而是娓娓道来，引导对方透过故事反思自己的价值观、探索内在动力，从而自己找到答案。这遵循了行为改变的公式：

```
行为改变 = 能力 + 动力 - 阻力
```

当能力具备时，故事能有效激发 **动力**，并降低心理 **阻力**。

## 如何讲好教导故事

那么，具体该如何讲一个有效的教导故事呢？以下是两类非常实用的故事类型。

### 第一类：讲述自己失败的故事

在教导时，最应避免的是“想当年”式的炫耀故事。但有一种关于你自己的故事却非常有效，那就是分享自己“想当年失败”的经历。

**例如**：下属因被合作伙伴言语冒犯而情绪低落。作为领导，可以分享一个自己初入职场时类似的受挫经历：如何从愤怒、想要回击，到想起“职业精神”而克制，最后发现对方可能并非故意，并最终化解矛盾、达成良好合作。

通过分享自己的失败经历以及从中获得的教训，能让被教导者感到共鸣，并从中汲取经验，而不是感到被居高临下地教训。

### 第二类：讲述对方成功的故事

这听起来可能有些反常识，但用被教导者自己过往的成功故事来引导他，效果极佳。

**例如**：孩子学习怕苦怕累。可以对他讲述他过去成功的故事：“还记得你三年前参加游泳训练营吗？开始时也很抗拒、觉得辛苦，但坚持那个夏天后，你进步神速，进了校队，现在游得又快又好，并以此为傲。你看，你不是不懂得努力，也知道努力能带来什么。在数学或中文上，只要像游泳一样努力，你也可能做好并喜欢上它。”

讲述这种故事的核心好处在于 **赋能**。听者重温自己的成功，会再次感受到胜利的喜悦和自身的能力，从而获得信心和动力去面对当前的挑战。

---

**本节课总结**：
当我们想要教导他人时，最差的方式是打击、比较或炫耀式的说教。因为问题往往不在于对方的能力或认知，而在于动力。故事是激发动力的有效工具。要讲好一个教导故事，你可以选择分享自己相关的失败经历和教训，或者挖掘并讲述被教导者自己过往的成功故事，这两种方法都能引导对方自我反思，从而达到良好的教导效果。
