# 032.情感心理咨询技能大全（：第08章：体验式家庭治疗—姚玉红：06.治疗目标和规划_note

在本节课中，我们将要学习体验式家庭治疗（萨提亚模式）中治疗目标的设定与治疗规划的制定。我们将了解其四大核心总目标，并学习如何根据家庭系统的开放或封闭特征来制定具体、正向且具有体验性的治疗计划。

## 治疗目标的设定 🎯

上一节我们介绍了萨提亚模式的基本框架，本节中我们来看看其治疗目标的设定。萨提亚模式设定了四大总目标，这些目标是治疗工作的核心方向。

### 四大总目标

以下是萨提亚模式提出的四大治疗总目标：

1.  **提升自我价值**：目标是让每个家庭成员拥有更高的自尊和自我评价，让整个家庭重新拥有希望和梦想。技术层面，可以让家庭成员描述各自对理想家庭生活的想法，或通过摆放“理想全家福”的方式，将个体与家庭的梦想呈现出来，让成员感觉自己“值得拥有”最好的状态。

2.  **能够做选择和有取舍**：现代人特别需要这种能力。萨提亚治疗认为，人至少要考虑三个选择才叫真正有选择。当只有两个选择时，人容易陷入非此即彼的困境；当只有一个选择时，情况更糟，那其实是“没得选”。许多心理症状，正是个体感觉自己在适应和生存中“没得选”的结果。治疗目标是帮助个体看到，在任何情境下，世上都存在至少三个可能的选项，并练习如何做出取舍。

3.  **负责任**：治疗从不全是来访者的错开始，但最终要导向“个体拥有改变的可能性”。这包括两方面责任：
    *   **个体责任**：为个人的梦想和目标采取自救行为。
    *   **互动责任**：在家庭互动中，成员也需要为促进关系改善负起责任。例如，青少年不能只要求父母完美改变，也需要通过自身行为（如好好沟通）来帮助父母理解自己，最终让双方都更舒服。

4.  **和谐一致**：即实现一致性沟通，做到表里如一、心口如一。

在四大目标中，**提升自我价值**是最根本的。在拥有较高自我价值感的基础上，个体才能更好地表现为**可以做选择**、**可以负责任**和**可以和谐一致**。

### 个体目标与关系目标的协调

所有的家庭治疗都需要个体目标和关系目标保持一致。但有时两者会冲突，例如家庭目标希望“全家永远紧密团结”，而个体目标希望“拥有更多个人自由”。

这时，治疗师需要帮助家庭讨论如何让目标相容。例如，探讨“如何既能感受到家庭一体性，又能给每个人自由空间”。治疗目标是希望获得所有家庭成员的同意。如果目标只被部分人同意而遭到另一部分人强烈反对，治疗将难以进行。

在萨提亚模式中，我们制定的是**正向导向的治疗目标**，主要包括冰山目标和阶段目标（冰山概念后续会详细说明）。

### 促进目标建立的技术：“天气预报”

在建立治疗目标的过程中，有一个促进家庭沟通的常用活动叫“天气预报”。这个技术本身就体现了四大目标。

以下是“天气预报”通常包含的五个步骤：

1.  **表达欣赏与感激**：家庭成员分享积极体验，如开心的事，或感谢其他成员为自己做的事。
2.  **分享担忧与困惑**：用“我担心…”或“我感到困惑…”代替批评，更容易让对方感受到关心。
3.  **抱怨与提议**：提出抱怨时，必须附带一个建设性的建议。没有建议的抱怨不要提。
4.  **分享新资讯**：分享最近听到的消息、新闻或个人新信息。
5.  **表达希望与愿景**：分享对未来的希望，为交流画上积极的句号。

“天气预报”如何契合四大目标？首先，它避免直接贬低，有助于**提升自我价值**。其次，它提供建议和新信息，让人感觉**有选择**。再次，它鼓励个人表达观点并为自己的话负责，体现了**负责任**。最后，它允许表达积极和消极两部分感受，而非粉饰太平，这有助于保持**和谐一致**。

## 治疗计划的制定 📋

前面我们提到了评估的信念，治疗计划的制定需基于评估结果。评估的一个重要维度是判断家庭系统属于开放型还是封闭型。

### 开放型家庭系统的特征

开放型家庭系统的主要特征是**灵活**。这种灵活表现在它可以主动选择，甚至可以选择暂时“封闭”（如闭关修炼），但这种选择是自主、有意识、有目的的。它不拒绝任何可能性。

其主要人际动力是**允许**：允许家庭成员自由表达渴望、希望、爱意，也允许表达恐惧、愤怒、沮丧等负面感受；允许成员犯一定错误而不必惊慌，不担心被拒绝或羞辱。在这样的家庭中，一切都可以自然发生、流动，处理问题的方式也更多元。

### 封闭型家庭系统的特征

封闭型家庭系统的主要特征是**僵化**。它持有绝对化的家庭规则（如“一定要节省”、“一定要对人友好”），这些规则曾经有帮助，但因过于绝对，会让偶尔的“破例”带来巨大的内疚和焦虑。它难以根据成员需求（如孩子进入青春期）或环境变化做出调整。

其主要人际动力是**恐吓**。为了迅速恢复“稳态”，它会采用高效率但制造内在紧张的方式，如恐吓、惩罚、引发内疚感（情感勒索）和控制。

在开放与封闭的对比中，可以明显看出家庭成员**自我价值感**的高低。高自我价值感的表现是：无论个体处于高峰还是低谷，都确信自己会被家人支持、关怀和爱护。这种价值感比较恒定稳定。

**需注意**：低自尊不等于情绪低落。情绪低落是正常的情绪反应。低自尊的核心是**不允许自己表达错误**，非常害怕暴露“不好”的部分。低自尊的人组成的家庭，氛围紧张，很难安全、一致地表达内心感受。

### 评估家庭规则

除了沟通模式（后续有专门章节），评估**家庭规则**也是重要一环。家庭规则是支配家庭成员行为的准则。

对于健康的开放型家庭，其规则是**人性化**且**可调整**的。我们可以从以下方面评估：
*   **规则是否人性化**：能否根据人本主义原则调整？是否允许消极和负面存在？
*   **规则能否与时俱进**：能否伴随环境（如孩子长大）而改变？
*   **家庭是否欢迎改变**：对尝试改变是鼓励、觉得有趣，还是打压？
*   **对待差异的规则**：当成员意见不一时，如何处理？是讨论、争执，还是一方压制？
*   **信息分享的规则**：成员愿意分享信息吗？分享后是得到尊重、欣赏，还是马上被批评？这涉及家庭内的发言权。
*   **表达情感的规则**：是否可以表达愤怒？是否可以拥抱？

### 设定目标的特征与治疗过程

基于评估，我们设定的治疗目标应具备以下特征：**正向导向**、**关注内在过程**（情感、想法而非仅具体事件）、**具有体验性**（可分解、可操作）。

当评估完成、目标设立后，还需要进行**结果评估**，方式包括：来访者反馈、治疗师自我评估、临床督导评估、家人或朋友评估，以及现实测验（如纸笔测验）。

最后，我们看看萨提亚模式典型的**治疗性过程**：

1.  **治疗师准备自己**：包括内在（保持平静）和外在（环境布置）的准备，达到和谐一致。
2.  **建设性地与家庭接触**：以指向改变的方式建立咨访关系。例如，萨提亚女士常以与每位成员握手开场，这不仅是礼貌，也是一种对成员开放程度的初步评估。
3.  **引导家庭说出希望和愿望**：这是关键步骤。治疗师需要将个体化的抱怨或问题，**转化为全家共同认可的正向、体验性目标**。
    *   **示例**：继父抱怨儿子“不上学”，萨提亚将其转化为“你希望他能为未来做一些起码的准备”。进而引导出“治疗希望能帮助你了解儿子怎样才能开心快乐”这个全家人都能投入的目标。
    *   **核心**：化解个体目标为集体目标，是家庭治疗的首要任务。
4.  **在过程中关注体验性**：不是纠缠于问题内容，而是通过过程展现关系互动模式。具体方法包括：
    *   把问题转化为关系互动和应对模式来讨论。
    *   探讨问题对个人内在（感受、想法）和家庭关系的冲击（用循环提问）。
    *   重构问题，将其视为成长的机会。
    *   确定体验性的正向目标，并获得全家承诺。
    *   始终关注应对姿态、改变可能性，并探索冰山下的期待、渴望和资源。
    *   在每位成员身上寻找并认可其资源与能量。
    *   定期回顾、提供希望。
    *   邀请家庭成员互相给予反馈。
    *   布置家庭作业（如练习“天气预报”），安排下次咨询。

本节课中我们一起学习了萨提亚模式治疗的核心四大目标（提升自我价值、做选择、负责任、和谐一致），学习了如何通过评估家庭系统类型（开放/封闭）和家庭规则来制定治疗计划，并了解了将问题转化为正向、体验性共同目标的技巧，以及完整的治疗过程框架。掌握这些，有助于我们引导家庭走向更健康、灵活的互动模式。
