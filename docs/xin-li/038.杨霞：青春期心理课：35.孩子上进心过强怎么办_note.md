# 038.杨霞：青春期心理课：35.孩子上进心过强怎么办_note

在本节课中，我们将探讨当孩子的上进心过于强烈，甚至转化为焦虑和虚荣时，家长应如何理解和引导。我们将分析这种现象背后的心理原因，并提供具体、可操作的方法，帮助孩子建立更健康、长远的价值观与生活平衡。

## 理解“上进心过强”的表现与根源

上一节我们介绍了青春期孩子常见的心理特点，本节中我们来看看“上进心过强”这一具体问题。

孩子有上进心是好事，但上进心如果过强，虚荣心过强，就会出现特定症状。这类孩子总是害怕考试失败，怕被别人超过。他们总希望自己能够保持第一，即便已经保持第一，仍担心自己有一天会落后。

这样的孩子情绪是焦虑的，总是忧心忡忡。家长在平时的引导、沟通、教育过程中，要注意不要过分强调分数，不要过分强调第一，因为世上没有完美的孩子。班上第一只有一个，但每个人都想当第一。

## 家长需要调整的心态与视角

以下是家长首先需要做出的心态转变：

*   **放眼长远，不斤斤计较**：家长首先要改变自己的心态，不要对孩子在具体小事上都斤斤计较，都希望拔尖。因为人生未来的工作、发展、对社会奉献的作用及取得的成就，与今天的第一、第二、第三关系不大。
*   **重新认识“贡献”**：不能说只有第一名才有贡献。研究表明，贡献最大的往往是中等生，是那些较好的中等生，甚至是排名十二名以外的学生。有校长在毕业典礼上总结：**成绩得A的学生是国家的未来，成绩得B的学生是国家的栋梁，成绩得C的学生是国家的财富**。所有孩子未来都是有用的。
*   **培养目标应是“对社会有用的人”**：家长眼睛要放远，才能教会孩子把眼睛放远。培养孩子，其实是培养其将来成为一个对社会有用、能有所奉献的人。奉献大小取决于才华、能力、意愿、身体和热情，并非单一由成绩决定。**成绩第一名与未来是否做出最大奉献没有必然联系**。

## 帮助孩子建立正确的价值观

如果孩子把人生看作一次又一次的考试，把“得到”（如第一名、认可、赞赏）当作人生目标，他会活得像个乞丐，充满担忧和贪欲。反之，如果把“给予”和“奉献”作为人生目标，他会觉得只要尽力就可以了。

家长常希望孩子“尽力就好”，但很多要强的孩子很难放松。核心在于，孩子需要一个远大的目标和正确的价值观。他需要理解，学习的价值未来是通过**奉献**的方式体现，而非通过考试排名。充分意识到这一点，才能改变其过分追求分数的行为。

## 平衡学习与运动：关键的行动方案

有的孩子为了保持好成绩，会花大量时间看书做题。但脑力活动过度会导致胡思乱想、多愁善感、忧心忡忡。我们要注意大脑的生理活动规律：**学得越多，有时反而担心越多**。学习过度会使大脑过度敏感，从而产生焦虑、担心，甚至强迫、抑郁或躯体症状。

因此，我们需要帮助孩子平衡脑力与体力活动。以下是具体的调整方法：

*   **审视时间分配**：如果孩子总是忧心忡忡，担心被超过，家长需审视其学习时间是否过长，以及自身是否对学习强调过度，同时检查孩子的锻炼量是否太少。
*   **增加运动时间**：运动能让大脑变得适当“迟钝”，平衡过度敏感的状态。**学习过多 -> 大脑过度敏感 -> 焦虑；运动增加 -> 大脑恢复平衡 -> 缓解焦虑**。
*   **借鉴成功经验**：许多高考状元都是德智体全面发展，既爱学习，也爱运动，并有健康的体育爱好。真正的学霸不仅会学习，还会玩、会运动。
*   **落实“劳逸结合”**：对中学生而言，运动量特别重要。例如，美国一些中学规定，每天下午必须保证两小时户外运动。我们可以借鉴，让孩子在下午四五点后户外运动一至两小时。**运动能让大脑保持平衡，让自己变得不那么敏感、担心和焦虑**。

## 总结与核心要义

本节课中，我们一起学习了如何应对孩子上进心过强的问题。

首先，我们认识到过强的上进心会转化为焦虑和虚荣，其根源往往与家长过度强调分数和排名有关。接着，我们探讨了家长需要调整心态，放眼长远，以培养“对社会有用的人”为目标，而非仅仅追求第一名。然后，我们指出帮助孩子建立以“奉献”而非“索取”为核心的价值观至关重要。最后，我们给出了最具体的行动方案：**通过增加规律性的户外运动，平衡过度脑力活动，是缓解孩子焦虑、促进其身心健康的有效途径**。

记住核心公式：**健康成长 = 正确的长远价值观 + 均衡的学习与运动生活**。家长的任务是引导孩子看到比分数更广阔的天地，并为其提供保持身心平衡的支持环境。
