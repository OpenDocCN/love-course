## 第021讲：唤起行动 🎯 表白当天就成功领证，可能吗？

在本节课中，我们将要学习广告中“品牌广告”与“效果广告”的核心区别与联系，理解它们如何分别作用于消费者的认知与行为，并探讨“品效合一”这一概念的真正含义。

上一讲我们介绍了引发互动行为的四种常见动机，并列举了许多“标题党”的例子。本节中我们来看看这些例子与之前课程中提到的经典创意文案有何不同。

我们之前举的例子大多是巧妙、美好且充满智慧的经典创意文案作品。而上一讲提到的例子则更像是直接、赤裸的号召与诱惑。

出现这种差异的原因在于，上一课的主题是如何引发他人直接的互动行为。事实上，有一类广告专门用于激发这种直接的行为转化。业界通常称这类广告为**效果广告**。

**效果广告**是与**品牌广告**相对应的称呼。从广告创作者的角度来看，效果广告着力于引发直接的、当场的消费行为转化。

近年来，广告营销界常提到“品效合一”这个词，意指品牌建设与效果转化要相结合。今天，我们将尝试从几个通俗的视角讲清“品”与“效”的关系。

首先，从广告创作者的角度看。有一句很有意思的话：“广告从来不直接带来消费。” 这句话的意思是，广告通过改变认知来改变行为，这是一个较长的链条，起效时间可能很久。

但效果广告不同，它的目标就是当场促成行为改变。这个行为根据具体目标而定，可能是下单、下载、注册或留下资料等。

以下是一个简单的判断标准：
如果你在广告中听到或看到“赶快点击下方链接”这类说法，甚至只是“赶快”、“点击”、“下方链接”这些词中的任何一个，那么你看到的很可能是一条效果广告。

从学术角度看，广告营销界有一个经典的**AIDA模型**。该模型将推销产品的工作细分为四个环节：
1.  **A**ttention：引起注意。
2.  **I**nterest：诱发兴趣。
3.  **D**esire：刺激欲望。
4.  **A**ction：促成行动。

我们之前讨论的品牌广告，主要聚焦于AIDA模型的前两步或前三步（引起注意、诱发兴趣）。它就像足球场上的发球和带球。而效果广告则负责最后的一两步（刺激欲望、促成行动），完成临门一脚。

例如，以往电视上的大多数广告只负责前几步，因为观众无法直接下单购买。

对此的总结是：**品牌广告着力于对消费需求的唤起，是偏好的制造；效果广告着力于对消费行为的唤起，是偏好的变现。**

我们曾在课程中给出品牌的定义：**品牌是一种固化的偏好**。品牌广告是让你喜欢上这个品牌或产品，效果广告则是告诉你：“别光喜欢，掏钱吧。”

我们常说创意文案像求爱，目的是让别人爱上你。求爱时需要矜持、优雅。但效果广告不同，它已经到了希望当场“领证”的关头，因此风格往往更直接、更不局促。

请注意，品牌广告着力于唤起需求，效果广告着力于唤起行为，但这并非绝对。现实中，品牌广告有时也能直接唤起消费行为，效果广告有时也需要承担唤起需求的任务。

实际上，效果广告有两种类型：
以下是两类效果广告的区分：
*   **一型效果广告**：为知名品牌或熟悉的产品所做。它只需完成AIDA模型的最后一步或两步，因为前期的品牌认知工作已完成。
    *   *公式*：`一型效果广告 = 知名品牌 + 直接行为号召`
    *   例如：“Tiffany钻戒情人节特惠，预购从速。” 这条广告无需再解释Tiffany是什么。
*   **二型效果广告**：为不熟悉的品牌或产品所做。它必须在一次广告传播中完成AIDA模型的所有四个步骤，从唤起需求到唤起行为一步到位。
    *   *公式*：`二型效果广告 = 未知产品 + (引起注意 + 诱发兴趣 + 刺激欲望 + 促成行动)`
    *   例如：“宇宙牌人造晶体，10块钱8斤，快来购买。” 这条广告很难奏效，因为大家不知道这是什么。

一型效果广告无法单独起效，必须配合品牌广告。二型效果广告才是真正意义上的“品效合一”，但实现难度极大，通常只适用于客单价低、试错成本与风险低的产品。

因此，我的观点是：“品效合一”不能狭隘地理解为将“品”和“效”两个步骤合并成一个工作。一型效果广告的做法才是大多数品牌应该采取的办法。与其追求“品效合一”，不如说**品效应协同**。速配的关系，往往不如日久生情稳固。

其次，从品牌或企业的角度来看。品牌广告是**让产品更好卖**的广告，效果广告是**让产品卖得更好**的广告。品牌广告为卖出产品做准备（制造需求和偏好），效果广告则聚焦于销售过程本身（将已有的偏好转化为行为）。

从这个角度，我们可以再做一个比喻：**效果广告就像大卖场里产品货架旁的促销员**。
以下是促销员类比的关键点：
1.  它一定站在货架旁，任务就是让你直接拿起产品放入购物车。
2.  它的收入与你的购买直接挂钩，易于衡量。
3.  如果是知名品牌（一型）的促销员，主要工作是告知促销信息。如果是陌生品牌（二型）的促销员，则需要同时介绍产品功能、价值和优惠。

最后，从消费者受众的角度来看，区分这两种广告有一个简单的诀窍：
**让你觉得这个东西很值钱的，是品牌广告；让你觉得这个东西很便宜的，是效果广告。**
因为品牌传播旨在形成固化的偏好和溢价，让你觉得这是个好东西。而效果广告作为促成购买的临门一脚，则需要让你觉得特别值得买，此时强调低价、折扣、性价比或稀缺性都是合理的。

本节课中我们一起学习了广告中“品”与“效”的核心关系：品牌传播唤起需求、积累偏好；效果广告将需求和偏好变现。品牌广告未必不能产生效果，效果广告也未必不能建设品牌。二者虽未必合一，却实实在在需要协同。
