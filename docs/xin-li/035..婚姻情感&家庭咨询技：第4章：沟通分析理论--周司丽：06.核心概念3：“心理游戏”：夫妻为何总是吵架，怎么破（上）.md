## 概述：什么是心理游戏？🤔

上一节我们介绍了沟通分析理论的基本框架，本节中我们来看看“心理游戏”这一独特概念。在沟通分析理论中，心理游戏并非指通常意义上的娱乐活动。它指的是一种特殊的人际互动模式。

这种互动模式往往以所有参与者都感到不舒服、愤怒、失望或沮丧等负面情绪收尾。心理游戏是沟通分析理论中非常独特且重要的内容，理解起来可能有一定难度。

## 引入案例：两段危险的对话💥

为了帮助理解，我们先来看两段发生在伴侣之间的对话。在阅读时，请思考：你觉得他们的对话有什么问题？如果继续下去，讨论会导向何处？

以下是第一段关于晚餐选择的对话：

*   丈夫说：“今晚咱们吃什么呢？”
*   妻子说：“随便，都行。”
*   丈夫说：“那吃火锅吧。”
*   妻子说：“火锅好久没吃了，但是会上火吧。”
*   丈夫说：“那吃饺子。”
*   妻子说：“饺子倒是有肉有菜，可前天刚吃了呀。”
*   丈夫说：“吃麻辣香锅行不行？”
*   妻子说：“火锅都上火了，那吃麻辣香锅不得更上火吗？”
*   丈夫接着说：“那你只能喝粥了吧。”
*   妻子说：“那我肯定得胖了。”
*   丈夫又说：“你不是吃什么都行吗？”
*   妻子说：“你总得考虑一下我的情况吧。”

对话至此，如果继续，双方很可能发生争执。丈夫可能觉得妻子挑剔，妻子则觉得丈夫不够体贴，最终不欢而散。

以下是第二段关于回家后互动的对话：

*   妻子说：“你回来啦。”
*   丈夫：“嗯。”
*   妻子说：“咱们一会儿去哪吃饭？”
*   丈夫说：“我还不饿。”
*   妻子说：“你怎么了？”
*   丈夫：“没怎么。”
*   妻子说：“没怎么？你为什么不吃饭呀？”
*   丈夫回答说：“我就是不想吃呀。”
*   妻子又接着说：“我觉得你今天特别不对劲。”
*   丈夫回答说：“我哪不对劲了？我就是很累呀。”
*   妻子说：“你干什么就很累了。”
*   丈夫说：“我上班啊，我不能累吗？”
*   妻子又回答说：“我也上班呀，我也没像你那么累呀。”

这段对话的火药味逐渐显现。如果继续，很可能演变为争吵。妻子觉得丈夫态度冷淡，丈夫觉得妻子无理取闹，双方都会感到不开心。

## 心理游戏的定义与核心特征🎭

心理游戏是使双方都会产生不良感受的一系列**隐藏沟通**，而且具有**重复**的性质。

当人们开始玩心理游戏时，互动中一定包含隐藏沟通。同时，相似的争吵模式可能会一次又一次发生。

心理游戏有几个显著特征：

以下是心理游戏的三个核心特征：

1.  **包含隐藏动机**：因为存在隐藏信息，当人们在说一件事情时，可能想表达的是另一层意思（言外之意）。最初，双方可能意识不到这些言外之意。
2.  **言外之意逐渐凸显**：随着沟通推进，隐藏的动机或言外之意会变得明显，被双方感知到。
3.  **以负面感受收尾**：沟通最终会达到一个点，让彼此都感到困惑、误解，并可能想要相互指责。

## 理解角度一：隐藏沟通的视角🕵️‍♀️

带着心理游戏的定义和特征，我们重新审视前面的两段对话。

在第一段对话中，丈夫问“今晚我们吃什么”，妻子回答“随便都行”。这里的“随便都行”是真的随便，还是包含其他信息？

一句“随便”，至少对应三种不同的沟通模式：

*   **互补沟通（成人-成人）**：妻子说“随便”是真的随意，丈夫提出建议，妻子同意。这没有问题。
*   **隐藏沟通（包含其他目的）**：在本例中，妻子说“随便”时，心理层面的信息可能与语言层面不同。她可能处于**儿童自我状态**，心想：“看你能否猜中我喜欢什么”或“你是否足够体贴”。甚至可能隐含“丈夫总是不够善解人意”的预设。
*   **验证预设**：当丈夫提出火锅、饺子、麻辣香锅等建议均被妻子以各种理由拒绝后，似乎验证了妻子的隐藏想法：“丈夫的选择总是不够好，不够为我考虑。”最终妻子说“你总得考虑一下我的情况吧”，言外之意可能是“你总是不够为我考虑”。

在第二段对话中，妻子高兴地说“你回来啦”，表面是问候，但从语气动作可感知她处于**儿童自我状态**，像在发出“一起玩”的邀请。丈夫闷闷不乐地“嗯”了一声，从心理层面**拒绝**了这个邀请。

妻子接着问“咱们一会儿去哪吃饭”，是再次邀请。丈夫说“我还不饿”，是再次拒绝。妻子的儿童状态可能感到受伤，于是她转换了状态。

妻子问“你怎么了”，语言上是成人式的询问，但心理上可能已切换到**父母自我状态**（尤其是批判型父母），开始指责丈夫“你这样不对”。此时，丈夫可能进入**叛逆的儿童状态**开始抵触。

**表面**上，他们在讨论“怎么了”；**心理**上，他们已进入“一方指责、一方抵抗”的较量。这就是心理游戏的典型特点：互动包含两个层面，表面说一件事，心理层面进行着另一件事。

## 理解角度二：戏剧三角形🎪

理解心理游戏还可以从“戏剧三角形”的角度入手。该理论由卡普曼提出，指出任何戏剧（人际互动）都包含三个角色：

以下是戏剧三角形的三个角色：

1.  **受害者**：特征常表现为哀叹“我不行”、“我真可怜”、“我被伤害/贬低”。
2.  **拯救者**：特征表现为“我可以帮你搞定”、“交给我没问题”。
3.  **迫害者**：特征表现为向他人发号施令、做出伤害或贬低他人的言行。

在沟通中，一旦你站上这三个角色中的任何一个，就意味着进入了心理游戏，互动很可能以不愉快告终。

我们用戏剧三角形分析前面的例子：

在第一个例子（晚餐）中：
*   **初期**：妻子说“随便”，把选择责任推给丈夫，自己像**受害者**（“我不知道怎么办”）。丈夫提出建议，扮演**拯救者**。
*   **中期**：妻子每次拒绝，都从受害者转变为**迫害者**（否定丈夫的建议）。丈夫则从拯救者跌落为**受害者**（建议被否）。
*   **结局**：若丈夫最后说“你真事儿多，不吃了”，他就从拯救者变成了**迫害者**。妻子则可能感觉“连老公都不理我”，成为更惨的**受害者**。

在第二个例子（回家）中：
*   **初期**：妻子等待丈夫回来陪伴，像**受害者**（“生活无趣”）。期待丈夫作为**拯救者**出现。
*   **转折**：丈夫冷淡回应，直接成了**迫害者**。
*   **后期**：妻子感到受伤，从受害者转变为**迫害者**，开始指责丈夫。丈夫感受到指责，进一步作为**迫害者**反击。最终双方进入相互迫害的冲突。

**核心要点**：在任何沟通中，只要你站上戏剧三角形的任一角色，就注定会进入心理游戏并产生不愉快的结局。

## 理解角度三：心理游戏公式（G公式）📐

伯恩用“G公式”给出了心理游戏最精确的定义：一个沟通序列只要符合此公式，就是心理游戏。

**心理游戏 G 公式**：`饵 + 钩 = 反应 -> 转换 -> 混乱 -> 结局`

以下是公式各环节的含义：

1.  **饵**：一方发出一个**隐藏沟通**（表面说A，心理传递B）。
2.  **钩**：另一方敏锐地感知到对方发出的心理层面信息，并“咬钩”回应。
3.  **反应**：双方表面说一件事，心理上进行一系列较量，沟通持续一段时间。
4.  **转换**：在互动中，双方在戏剧三角形中的**角色发生转换**（如受害者变迫害者）。
5.  **混乱**：双方感到困惑和惊讶的时刻（“我们怎么说到这了？”）。
6.  **结局**：当事人**重复体验到**一种消极感受（如生气、受伤、挫败）。

我们用公式分析一段婆媳对话（关于给孩子剪头发）：

*   **饵**：婆婆说“你到底给孩子剪不剪头发呀？你要是没时间带孩子去，我带孩子去。”表面是提供选择，心理层面是**指责和批评**（“我说了很多次你都没做”）。
*   **钩**：儿媳对批评敏感，立刻捕捉到隐藏的指责信息，回应：“我没说我没时间呀。”
*   **反应与转换**：随后可能发生“我都说了好几次了”——“那也不能你说我就得剪”——“我带孩子去呀”——“不用你带”…… 婆婆可能从开始的**迫害者**（指责），转换到**受害者**（“我多管闲事，我好可怜”）。儿媳可能从**受害者**（被指责），转换到**迫害者**（反击）。
*   **混乱与结局**：双方最终可能都感到**生气和受伤**，并且这种模式会重复发生。

再分析电视剧《奋斗》片段（杨晓云与向南）：

*   **饵**：杨晓云说：“这坚真的爱情啊，永远是建立在金钱的基础上。”表面是讲道理，言外之意是“**你没钱**”。
*   **钩**：向南敏锐捕捉到被否定的信息，回应：“我明白了杨晓云，这不坚真的爱情，一定是没钱惹的祸。”
*   **反应与转换**：杨晓云起初是**迫害者**（贬低向南），向南是**受害者**。随后向南反击（提及杨母想让她嫁路涛），转换为**迫害者**。杨晓云被打后，又作为**受害者**反击（打耳光），再次成为**迫害者**。
*   **结局**：双方以**愤怒、受伤**收场，爆发肢体冲突。

## 心理游戏的严重程度分级⚠️

心理游戏根据其影响严重程度可分为不同等级：

以下是心理游戏的三个等级：

1.  **一度心理游戏**：在社交层面可接受，日常生活中常见。虽会引起轻微不舒服，但无伤大雅，不影响生活。例如，聚会上恭维对方穿衣品味好，对方可能转而指出你的不足，让你稍有不适，但很快过去。
2.  **二度心理游戏**：会明显改变人生轨迹。例如，导致离婚、被炒鱿鱼等重大后果。
3.  **三度心理游戏**：涉及身体健康，甚至威胁生命。是程度最严重的心理游戏。

在婚姻中，评估双方所玩心理游戏的严重程度非常重要。

## 总结📝

本节课中，我们一起学习了沟通分析理论的核心概念——“心理游戏”。

我们首先明确了心理游戏的定义：它是一种导致双方产生不良感受的、重复的隐藏沟通模式。接着，我们通过两段伴侣对话案例，引出了心理游戏的破坏性。

我们从三个角度深入理解了心理游戏：
1.  **隐藏沟通视角**：心理游戏包含表面与心理两个层面的信息，言外之意是冲突的关键。
2.  **戏剧三角形视角**：一旦在沟通中扮演**受害者、拯救者或迫害者**中的任一角色，就会进入心理游戏，并以不愉快告终。
3.  **G公式视角**：心理游戏遵循 `饵 + 钩 = 反应 -> 转换 -> 混乱 -> 结局` 的序列，最终使人重复体验消极情绪。

最后，我们了解到心理游戏有严重程度之分，从一度（轻微）到三度（危及生命）不等。
