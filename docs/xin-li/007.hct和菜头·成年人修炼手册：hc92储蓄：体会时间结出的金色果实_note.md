# 💰 成年人修炼手册·第92课：储蓄——体会时间结出的金色果实

在本节课中，我们将探讨“储蓄”这一看似简单却意义深远的行为。我们将理解储蓄的本质、它为何困难，以及它如何超越金钱本身，为我们带来心理上的安全感和对生活的掌控力。

---

储蓄，指的是城乡居民将暂时不用或节余的货币收入存入银行或其他金融机构的一种存款活动。它应该是一个从小养成的良好习惯。

没有人希望别人对自己的生活指手画脚。无论如何生活，给人的感觉是自己可以对付过去。在别人眼中看起来还有很大提升空间的生活，对于当事人而言可能已经在暗中倾尽全力。

但是，如果我重回21岁，我希望有人专门跑来告诉我储蓄的重要性。并且，那个人最好能清晰地告诉我，储蓄应该理解为一种**动词**而非名词。

储蓄在今天听起来很土，完全不如“个人理财”一类的大词。但是，一个人在刚走上社会时，甚至是走上社会十多年之后，也未必有什么可以值得去“理”的财。即便有，也未必有那个时间；有那个时间，也未必做得好，毕竟理财也是一种专业。

然而储蓄不需要多少专业知识，也不需要耗费多少时间，几乎人人都能做到。而同时，又有太多人做不到。

上一节我们提到了储蓄的普遍性与看似简单的特性，本节中我们来看看个人财富增长的真实曲线与物欲之间的冲突。

个人财富的增长存在一条极为痛苦的曲线。可能在一开始工作的头十年、头二十年，曲线几乎就是平的，略微有一点点抬头而已。但是与此同时，你的物欲是拉满的状态。在这两者之间，始终存在着很大的缺口。

所谓中年的平和，指的是个人财富终于加速增长，同时物欲也随之下降，导致缺口变小甚至弥合，于是内心终于找到了平静。但是，平静不是因为物欲下降，而是财富增长带来的自信心。

在那些痛苦挣扎的日子里，我学到了极为重要的一课：放弃幻想，面对现实。现实非常坚硬，它告诉我说，以你的收入水平之低，以你的物欲之强，你的欲望是无法通过消费来满足的。

一旦接受这个现实之后，储蓄才能成为可能。之前因为还抱有幻想，所以每个月都把工资花光。钱虽然是花出去了，但是物欲之火反而更加熊熊燃烧。

理解了欲望与消费的陷阱后，我们来看看人们通常如何应对，以及为何这些方式难以导向储蓄。

本来，这里也存在其他解法。比如说，一咬牙，攒上三个月半年的工资，狠狠心买个大件，多少体会一下物欲在某个物件上彻底满足的感觉，内心也会因此消停一会。但是，会这么做的人并不多。

更常见的是把钱花在各种廉价低质的东西上。钱是花光了，个人满足感却没有提升起来。买一双假名牌鞋其实并不便宜，但是到手之后，对于正品的欲望反而更加折磨人。于是，下个月继续来，继续花光，买回一堆能用但是又不够好的东西。

一旦进入这样的循环，人就很难去储蓄了。因为这个时候，人已经习惯了通过反复购买、自由选择的方式来暂时平复欲望，这个过程只要开始就很难停下。

所以储蓄之难在于它是难上加难。本来预算就不足，收入就不高，但是还要硬生生地从各项开支里砍下一块来，放到银行去。而且，在开始储蓄的头几年，户头上的数字增长也和日子一样缓慢。

于是很早之前就有了这样的说法：与其想着存钱，不如去想着如何赚钱；不要害怕花钱，只有花得多才能赚得多。我不知道别人的情况，反正这种说法在我这里不成立，而且我也绝不赞同。

“想办法赚钱”是一种太过乐观的说法。许多人在工作初期都是在为事业打基础，哪里有时间和资源去“赚钱”？而且，如果真有赚钱的能力，那又何必来上班呢？这些说法无非是让人心安理得地花钱罢了。那个花多少都能赚更多回来的自己，只存在于想象之中。

个人财富的增长当然不可能单靠储蓄。大多数情况下，真正有贡献的是个人收入的增长。对于大多数人而言，就是更高的职位，更高的薪酬。但是，储蓄依然非常重要。

在个人经济状况极为困难的条件下，还能每月存上一小笔，那么人就对未来还抱有希望。每个月花光，那是一种绝望的体现。不要小看储蓄这个动作，它一旦发生，就意味着你多少能够克服一点自己的物欲，夺回了一点对生活的控制权。而每多一点控制权，就少一点物欲折磨。

储蓄也会慢慢带来心理上的改善。我们都知道，现代生活要求每个人都应该有一笔紧急资金，总额大约是 **6-12个月的目前月薪**。因为你现在的月薪能够让你生活得下去，这笔储蓄的意思就是，一旦你没有了工作，还可以靠这笔钱获得大半年的缓冲时间，你可以比较从容地找寻下一份工作。

同样的道理，储蓄能给人带来安全感，更多的储蓄还能带来更多的自信心。

附带说一句，年轻的好友，无论是结婚还是搬迁到另外一座城市，除了诚挚的祝福之外，最好的礼物就是赠送一笔礼金作为安家费。因为这相当于增加了他的储蓄，为他即将开展的新生活提供更多的信心。

在建立信心之外，储蓄是非常直观的财富课程。它告诉我们，财富的累积需要极大的耐性，必须要忍耐相当长一段时间的匮乏，同时还要和自己的物欲做抗争。

当一个人看到自己的财富是如何涓滴汇聚成流的全过程，就不大容易相信一夜暴富的故事。全程体验过这种极为缓慢的增长，就会理解世间万事万物都有各自的时间。所谓成功，都是坚韧不拔在时间里结出的金色果实。

储蓄不能帮助一个人实现财富自由之梦，但是可以避免让一个人相信自己是投资专家，相信自己能发现市场上别人不曾发现的机会，凭着自信把钱全部赔光。

事实上，我自己是延迟至26岁之后才开始有所储蓄的。之前都习惯于每个月挣多少花多少，这种生活方式让我始终处于饥渴状态之中，每天折磨自己的唯一问题就是为什么钱不够花。

在我第一次离职的时候，银行户口上的存款给我提供了很大的安慰。当时我认为，再怎么不顺利，这笔钱还可以供我北漂一年多到两年，也许在这个过程中我就发现了其他机会也不一定。

接下来，我就刚好赶上了中国互联网发展的大爆发，收入水平也随之上升。但是，如果当初不是有那笔存款，我想我的北漂生活可能会过得相当窘迫，因为过于在意收入而错失不少好机会，那就会是另外一个人生故事了。

如今，全世界高速发展的好时光已经结束，各国都在慢下脚步来。我认为这个时候个人储蓄就变得更为重要，它能够帮助人们抵御各种未知的风险和变化。有储蓄的人可能会处于更为有利的位置，而零储蓄的人可能意味着更少的选择余地。

最后，我想提醒一点，储蓄并没有想象中那么难，你也不像你想象中那么穷。当初我决定开始储蓄的时候干过这么一件事：

以下是开始储蓄的一个具体行动建议：
*   我跑去银行，询问每次存钱的最低限额是多少。记得那是一个极小的数额。
*   然后，我很高兴地对自己说，这点钱根本不成问题。
*   对于当时的我而言，储蓄不是个数字，不是个数量，而是一个单纯的动作。去银行存款这件事本身对我意义重大。

---

本节课中，我们一起学习了储蓄的本质。我们认识到，储蓄不仅是存钱的动作，更是对抗即时物欲、夺回生活控制权的开始。它无法让你一夜暴富，但能为你提供至关重要的安全垫、培养耐心，并让你亲身经历财富缓慢积累的过程，从而对时间和成功有更深刻的理解。储蓄的难点在于开始，但起点可以很低，关键在于将“去存钱”这个动作，变成一种习惯和希望。
