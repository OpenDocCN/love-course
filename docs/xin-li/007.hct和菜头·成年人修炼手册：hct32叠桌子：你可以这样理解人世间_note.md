# 课程007：叠桌子模型——理解人世间的层级与选择 🪑

在本节课中，我们将学习“叠桌子”这一隐喻模型。该模型将人类社会结构比作层层叠起的桌子，用以解释资源分配、层级流动、竞争与合作等复杂现象。通过这一模型，我们可以更清晰地理解个人在社会中的位置、选择与挑战。

## 概述：什么是“叠桌子”模型？

上一节我们介绍了课程的主题。本节中，我们来看看“叠桌子”模型的核心定义。

“叠桌子”原是一种杂技表演，演员在桌面层层叠放更小的桌子，并攀至顶端表演。这一形式被引申为一种人类社会模型。

在这个模型中，世界被视作层层叠放的桌子。最底层的桌子最大，能容纳数亿人；最顶层的桌子最小，仅能围坐百十人。每一层都比下一层小一圈，能容纳的人数也更少。人们围坐在不同层级的桌子边，获取相应层级的资源（可理解为“饭菜”）。

## 个体的三种基本选择

理解了模型的基本结构后，我们来看看身处其中的人通常面临哪些选择。

以下是基于所处层级，个体可能采取的三种基本行动策略：

1.  **安守本位**：选择留在当前层级的桌子边。但需要提防上层的人掉落下来将自己挤下去，同时也要防备下层的人爬上来抢夺自己的座位。
2.  **向上攀爬**：试图爬向更高一层。这个过程可能遭到目标层级众人的抵抗而跌落，也可能成功挤出一个空位，获得更好的资源。
3.  **主动下降**：选择下降到更下一层。那里空间更宽敞，座位更容易获得，但所能获取的资源质量会下降，且日后想再爬回原层级将极为困难。

> **关键提示**：这个比喻中的“吃饭”可以广义地理解为获取**资源**、**机会**或**生存条件**。

## 模型如何解释社会现象？

现在，我们已了解了模型的结构与个体的选择。本节将探讨这一模型如何具体解释一些常见的社会与职场现象。

### 认同感与冲突的来源

同一张桌子（同一层级）的人，往往拥有相似的奋斗经历和能力水平，这带来了彼此的理解与尊重。然而，这是一种脆弱的平衡。

**公式**：`同一层级的和谐 ≈ 共同经历（理解） + 实力相当（尊重）`

一旦有人显露出要踩着同层人向上攀爬的野心，这种平衡就会被打破，冲突随即产生。例如，在职场360度测评中，同级评分往往最低，这反映了同级间潜在的竞争戒备。

### 向上攀爬的困难与动力

向上突破之所以困难，是因为每一层的座位有限。攀爬者会同时受到下一层人的拉扯和上一层人的打压，必须证明自己有应对两方压力的实力才能成功。

那么，为何人们仍前仆后继地向上爬？除了追求更优质的资源，更核心的动力在于追求更大的**选择自由**和**控制感**。

**模型视角转换**：
*   按**座位数**衡量：桌子越往上越小。
*   按**个人拥有的选择与空间**衡量：每个人享有的“桌面”越往上越大。

层级越高，个人选择越多，受现实条件的束缚越少，与同层竞争者的直接冲突也可能减少，从而表现出一种“彬彬有礼的温情”。

### 另辟蹊径与脱离游戏的风险

当然，也有人不满足于在既定结构中攀爬。

以下是几种非常规路径及其风险：

*   **另开一桌（如创业）**：试图创造一个新层级。风险在于：离开原座位后难以回头；可能找不到足够的人共建“新桌”；必须证明新桌子比旧体系更优，否则所有投入将化为乌有。
*   **起身离桌（如归隐）**：宣称退出竞争。这通常需要满足两个条件：1）其存在对同层竞争者不构成威胁；2）所在层级或上层有充足乃至溢出的资源可供其低消耗生存。一旦资源紧缩，这种状态便难以为继。

## 模型的启示：教育与个人成长

如果我们接受这个模型，便能重新理解教育和个人学习的意义。

*   **念书**：是成本最低的“爬桌”方式，能帮助个人在游戏初期就抵达一个较高的起点。
*   **终身学习**：是最温和的爬桌方式。通过持续自我发展，有时甚至能实现“跳桌”——当个人的思维和能力超越当前层级时，同层的人将无法阻拦其自然上升。

## 总结与建议

本节课中，我们一起学习了“叠桌子”模型。该模型将社会视为一个层级结构，解释了资源分配、层级流动、人际合作与竞争的内在逻辑。

许多关于职场位置、人际关系、发展瓶颈的抱怨，都可以从这个模型中找到对应的层级和“同桌”问题视角。

最后，一个重要的建议是：**无论身处哪一层级，都不要忘记转身，看看桌面之外的风景。** 如果一生攀登却从未欣赏过风景，那么获取资源本身就变成了人生的全部意义。或许，那些为了看风景而攀登的人，反而能获得更持久和内在的动力。
