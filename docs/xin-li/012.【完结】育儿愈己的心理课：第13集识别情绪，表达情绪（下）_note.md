# 012.【完结】育儿愈己的心理课：第13集识别情绪，表达情绪（下）_note

在本节课中，我们将学习如何在实际情境中识别并帮助孩子表达情绪，掌握建立情感连接的具体方法，并理解孩子情绪背后的心理逻辑。

上一节我们介绍了情绪识别与表达的重要性，本节中我们来看看如何将这些理念转化为具体的行动步骤。

关系依靠心灵去维护和感受，而非依靠方法去控制和引导。许多家长可能难以完全认同这个观点，尤其是那些思维被科学框架和条条框框限制得很紧的父母。他们的大脑可能很难接受“关系”这个概念，因为关系看不见也摸不着，需要投入全部心力去营造，这确实非常不容易。

如果父母自身从小被温柔而坚定地抚养长大，他们的父母能与其说很多心里话，体会并表达出他们的情绪，那么他们未来培养自己的孩子就会异常轻松。反之，如果父母从小被教条、僵化的方式养育，只会讲道理，甚至有时采用粗暴的方法，那么他们对待孩子的方式恐怕也会是简单、粗暴而僵化的。

这里需要插入一个题外话。许多妈妈会责怪爸爸，例如爸爸只知在外工作、不关心孩子、与孩子相处时间少等。此处并非要为各位爸爸翻案，只是希望替他们说句心里话。没有人天生就会做爸爸。而爸爸们自己的父亲，即孩子的爷爷，当年是如何培养他们的呢？或许爷爷当年就不常在家。因此，爸爸们其实并不知道好爸爸应该是什么样子。

所以，妈妈们不应强求爸爸完全按照自己的标准行事，这恐怕很难做到。更何况，爸爸们承担了大量的社会责任，需要养家糊口、提供物质资源等。我们需要看到每个人的局限性。

看到局限性，并不意味着就此放任或随波逐流。恰恰相反，在看到局限性之后，我们要在自身条件允许的范围内，尽力寻求提高。了解知识，只是第一步。

那么，当我们蹲下身子，平视孩子，并与他建立身体接触之后，就向孩子发出了一个关键信号。这个信号表明：父母不是要来改变你。一旦让孩子感觉到父母试图改变他，他的心理防御机制会立刻启动。难道我们还嫌孩子的下层大脑运作得不够激烈吗？

我们之前说过，孩子一旦感觉你要改变他，防御机制就会启动，他通常只有两个选择：攻击或逃跑。现在，我们要发出一个友善的信号，告诉他：你既不用攻击，也不用逃跑。你应该做的是，好好让妈妈陪着你，让妈妈来共同分担、承担，或者帮助你一起感受内心的那些不舒服的感觉。

然后，开始运用你丰富的词汇。平时可以多搜集一些描述心情的词汇。你描述得越准确，就越能将孩子从情绪的泥潭中拉出来。这是一条真正的规律，在我们的实验室中也有数据支持这一点：帮助孩子表达情绪。

例如，你可以说：“宝宝，妈妈看到你现在满头大汗，你心里是不是很生气啊？你是不是很难受啊？你是不是很不舒服啊？”等等，以此来帮助他表达。

如何判断你的表达是否正确呢？你需要观察孩子的反应。当你真正说中他的内心感受时，孩子的情绪通常会慢慢缓和下来。这考验的是各位爸爸妈妈“猜情绪”的功力。

这里给大家一个提示：如何能又快又好地猜中孩子的情绪呢？有一个方法可以教给大家，那就是去体会你自己的情绪。平时多对自己的情绪进行体会、描述和表达，这将有助于你更好地体会孩子的情绪。

下面，我们用一个简单的小例子来回顾和总结今天所用到的方法。

想象一下，妈妈正在厨房烧菜，起油锅时声音非常吵。宝宝在外面一个人玩积木，但他觉得无聊了，开始叫妈妈。叫了一声，妈妈没听见；叫了两声，妈妈还是没听见；叫了三声，妈妈依然没听见。结果，宝宝控制不住地嚎啕大哭起来。

这时，妈妈看到宝宝在哭，立即感到焦虑。为什么焦虑？这是一个两难问题：菜还在锅里烧着，孩子情绪已经非常激动。这时该怎么办？先处理孩子，还是先处理锅里的菜？我们可以选择先把火熄掉，然后走到孩子身边。

“哎呀宝宝你怎么了呀？”孩子可能不说话，只是哭。

那么，像这种情况，如何处理孩子的情绪，才能既治标又治本呢？根据我们说的方法，首先要低下身子，平视孩子。**眼睛一定要与他的眼睛有接触**，记住，如果没有眼神接触，蹲下来平视就失去了意义。然后，与他进行最基本的身体接触，慢慢把他搂到怀里。孩子可能会耍脾气、不愿意，因为他心里在生气。他觉得妈妈不要他了。

大家需要理解一个关键点。例如，我们把手机放到身后，问各位手机在不在？各位当然认为它在，手机又不会被吸进黑洞或自己消失。但是，如果你把手机放到一个年龄较小的孩子身后，再问他手机在不在，他可能会告诉你“手机没在”。

所以，各位要理解，为什么在你炒菜时，孩子叫你你不回应，他会那么难过。因为他看不到你，叫你你没回应，他就觉得妈妈已经“没有”了。正因如此，孩子才会那么生气和绝望，发出嚎啕大哭的声音。要充分理解这一点，孩子毕竟还小，他的思维和我们完全不同。

一般来说，孩子要到三、四岁左右，才会形成“客体恒常性”的概念，即东西虽然看不见了，但不会无缘无故消失。在这个概念形成之前，只要他找不到你，他就觉得你“死了”或者“没有了”。

好，我们回到处理这个案例的具体情节上。这时，你要抱住孩子，蹲下来，保持眼神和身体接触，然后帮助他表达，也就是猜测他的情绪。

我们都是聪明的父母，一下子就能猜到，孩子肯定是因为叫你你不回应，所以才产生情绪。我们来体会一下，这时孩子心里会有什么感觉？有点难受、有点绝望、有点抑郁，同时还有点委屈和生气。你要用这些词语，以温柔的方式表达出来。如果孩子还小，听不懂，就用更简单的词，慢慢跟他表达。

之后，当孩子的情绪被你说中，有所下降时，给他一些陪伴的时间，**最好在5分钟以上**。我知道各位父母都很忙，时间宝贵。但事实上，孩子的时间观念很弱，他的时间“不值钱”。你觉得孩子已经哭了10分钟，可能在他的感觉里，才哭了几秒钟而已。大家理解我的意思吗？孩子的时间观念和在激烈情绪下的时间感知完全被冲淡了，所以我们要多留一些耐心给孩子。

我知道各位的时间都很值钱，但至少多花一些时间来陪伴孩子、处理孩子的情绪，这应该不会让大家“亏”太多。为什么？因为你用好的方法处理一次，孩子下一次发脾气的强度和频率，通常都会比上一次有所改善，这是真实的。

当孩子的情绪慢慢缓和下来之后，我们就可以和孩子说原因，帮助他表达这件事情的前因后果。句式可以参考以下模板：

**“妈妈猜测/感觉，是不是因为宝宝一直叫妈妈，妈妈没有答应你，所以你感觉（生气/难过/委屈……）？”**

大家听清楚这个句式：你要把前因后果给孩子串联起来。因为孩子自己不能理解，为什么他会有这么大的脾气。

本节课中，我们一起学习了在实际冲突中应用情绪沟通的具体步骤：通过身体接触与眼神交流发出安全信号，用准确的词汇帮助孩子表达情绪，并理解幼儿“客体恒常性”未形成时的心理特点。关键在于，将孩子的情绪体验与前因后果联系起来，用陪伴和共情取代说教与控制，从而在每一次情绪事件中巩固亲子关系。
