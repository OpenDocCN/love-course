# 亲密关系课程 004：物化——疏离的爱 🧱

在本节课中，我们将要学习亲密关系中的一种失衡状态——物化。我们将探讨什么是物化，它如何产生，以及在亲密关系中的几种具体表现形式。通过理解物化，我们可以更好地识别和避免这种疏离的爱。

## 什么是物化？

上一讲我们介绍了亲密关系中的一种失衡，本节中我们来看看另一种形式——物化。

有时候，吵架的人会说：“你是什么东西？”或“你不就是会挣两钱吗？”。这些话之所以伤人，是因为它们通过把对方**物化**来传递一个信息：你不重要。**物化，即把人当作物品一样看待**。

这是由“怕”主导的另一种关系形式。人们通过情感疏远来获得安全感。当在关系中感到不安全时，人会不自觉物化对方，以告诉自己对方不重要。

## 物化的心理根源

提出依恋理论的心理学家鲍尔比记录了孩子与父母分离后的表现。如果妈妈回来，孩子会表现得漠不关心，不再黏着妈妈。孩子不再对任何人产生依恋，但对糖、玩具或食物看得越来越重。孩子的愿望和感受从指向人，转向了指向物。

**公式：恐惧被抛弃 > 渴望依恋 → 产生疏离态度**

当被抛弃的恐惧压过对依恋的渴望时，孩子就会表现出这种分离态度。对可能得不到的东西就疏远、漠不关心；对能够得到的东西则再度关注。

成年人在亲密关系中也有类似表现。如果一个人太担心被抛弃，他会压制自己对依恋的渴望。然而，对爱、依恋和性的需要一直存在。于是，他会采取一种折中方案：还是跟人交往，但把人当作物。这样，他可以像占有物品一样占有对方，而不用担心对方是否爱自己或会离开自己。这就是亲密关系里的物化。

## 纯粹的爱与物化的爱

请思考一个问题：如果你有爱人，你觉得他对你有用吗？你是因为这种用处才跟他在一起的吗？在亲密关系里，这样的问题常让人不舒服。因为从本性上，人们都是“爱的原教旨主义者”，渴望得到那种纯粹的爱——希望别人爱你是因为你本人，而不是因为你有用。

相反，**物化的爱是：别人爱你就是因为你有用。别人爱的是你的功能，而不是你本人。**

在一个家庭中，每个人都有一定功能（如赚钱、做家务）。但谁都希望彼此在一起首先是因为爱，而不是因为这些功能。我曾讲过，亲密关系中的爱，本质上是为了让系统稳定而设计的。如果彼此爱的是对方本人，关系常常更稳定。**物化是一种相反的模式**。

这种模式常源于对爱的不安全感。通过物化，人既拥有了一段关系，又避免了被抛弃的恐惧。但这只是单方面的如意算盘，因为物化的关系并不能真正满足人对亲密关系的需要。它把人封锁在自我之内，妄图让爱从两个人的事变成一个人的事。这对个人来说更可控，但对关系而言，少了另一方的互动，两人就很难真正走近。

## 物化的几种表现形式

在亲密关系中，物化有很多实际的表现。以下是几种常见的形式。

### 1. 经济物化 💰

现在很多年轻人反感相亲。一个朋友相亲几次后说再也不去了，因为别人一打听他的工作、收入、房产，他就感觉自己像被放到了交易市场。这就是人对物化的抗拒。

当然，也有人不这么想。曾有一个单身的“钻石王老五”找到一位爱慕虚荣的姑娘结婚。别人问他原因，他说：“我知道她是为了我的钱。但这样的婚姻才稳定。如果是为了我这个人，我是会变的。容貌会变，才华会用尽，只有钱不会变，最可靠。她喜欢钱最好，只要我有钱，我们的婚姻就是稳定的。”

这段话背后，可能藏着对爱的失望和对失去的恐惧。这样的婚姻关系或许是稳定的，但这已不是我们要讨论的亲密关系了。

### 2. 颜值与性的物化 👁️

曾经有人热衷于给女性网友的照片打分，将女性“物化”成“七分女”、“八分女”。这个活动得到很多男生追捧，变成了一场网络狂欢。

为什么？原因之一是，当爱变得复杂而不可控时，人们宁可有一个简单的标准来衡量它。于是，颜值就变成了可以打分的标准。不仅是颜值，所有外在的东西——毕业学校、工作、收入、年龄——都可以变成一个分数。然后给自己和对方打分，来确定交往是“亏”还是“赚”。于是，亲密关系就变成了一种彼此定价的交易关系。

```python
# 一个简化的、讽刺性的“物化打分”逻辑示例
def calculate_score(looks, income, education):
    # 这并非真正的爱，而是物化的计算
    total_score = looks * 0.5 + income * 0.3 + education * 0.2
    return total_score
```

这个交易系统听起来有道理，却忽略了亲密关系中最重要的情感因素和真正的人。它只是人为了让关系可控而做出的徒劳努力。

一位女性朋友曾说：“大学时很多男生追我，因为我长得漂亮，家里条件好。但我不喜欢他们，我觉得他们是因为我的美貌来爱我，那不是真的爱。我后来谈恋爱时，也不在乎对方条件，我觉得在爱里说这些很庸俗。”她真正反感的，就是那种物化的爱。

与容貌相关的性，也是物化的另一种表现。性关系原是两个人全情投入的关系。在理想状况下，**性是爱的产物，是爱的身体表达**。但有时，性也会被从爱里独立出来，这时候人（尤其是女性）就容易变成性物化的工具。

与性相关的是生殖功能。老一辈常说“多生几个孩子”，这常让新一代女性反感。一些新妈妈刚生完孩子，会怀疑自己的身份。如果爸爸置身事外，她们就会抱怨：“难道我就是你们生孩子的工具吗？”这些抱怨的背后，其实是在问：“你到底爱不爱我？”越是感觉不到爱和关心，这种被物化的疑虑就越重。

### 3. 陪伴的物化 👥

把陪伴当作物化的一种形式有些奇怪，因为陪伴常被认为有情感属性。恋爱的目的之一就是希望有个人陪。但陪伴有时也是一种物化。

**物化的本质是关注人的功能，而不是关注人本身。**

设想一个人谈过很多次恋爱，每次失恋都会很快开始新关系。这段关系并非因为他多爱对方，而是他很难忍受一个人，尤其是在失恋时期。虽然他需要的只是一种陪伴，是精神层面的需要，但那也是一种物化。

这就涉及**需要**与**爱**的区别。如果你只是需要某个人陪你而和他在一起，那就不是真的爱他，因为你关注的是对方“陪伴”的功能。从这个角度讲，你关心的也只是你自己的需要。这是陪伴的悖论。

从长久来说，只有一个人真的爱对方，跟对方建立亲密的联系，才能真的缓解孤独，陪伴才会真的有效。可是如果你只是想获得那个人的陪伴，而不关心那个人，慢慢地你会发现，就算这个人在，也无法缓解你的孤独。因为**陪伴是爱的附属品**。同时，在这种物化形式里，你走不出你自己。

## 总结与思考

本节课中，我们一起学习了亲密关系的另一种失衡——物化。

*   **物化**是一种特殊的心理防御机制。为了避免可能的伤害、防御被抛弃的恐惧，人们把对方当作物一样对待，关注对方的某种功能，希望借此占有它。
*   **物化**只是满足自己的需要，而**爱**需要人愿意奉献自己去满足伴侣的需要。
*   说到底，物化的爱都会演变成一种**有条件的爱**，而我们追求的，是那种只爱我本人的**无条件的爱**。

**公式：物化的爱 = 关注功能 + 满足自我需要；纯粹的爱 = 关注本人 + 相互奉献**
