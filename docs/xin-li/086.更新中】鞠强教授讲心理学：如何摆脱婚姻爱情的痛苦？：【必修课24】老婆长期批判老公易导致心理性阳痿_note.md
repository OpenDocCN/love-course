# 心理学必修课24 🧠：如何摆脱婚姻爱情的痛苦？——心因性阳痿的成因与应对

在本节课中，我们将要学习婚姻关系中一个常见但隐秘的问题——心理性阳痿（心因性勃起功能障碍）。我们将探讨其主要的心理成因，并理解“暗示”这一核心心理学概念如何在亲密关系中产生深远影响。

---

居强教授在多年的咨询经历中，遇到过许多谈论婚姻爱情问题的来访者。其中一个常被提及的隐私问题，便是男性的勃起功能障碍（俗称“阳痿”）。

这种现象在结婚年数较长的夫妻中更为常见。但需要明确的是，绝大多数案例属于心理性阳痿。国际学术界的共识支持这一观点，因为许多患者在夜间睡眠中，通过特定仪器检测，被发现仍存在正常的勃起反应。

一个简单的检查方法是：在生殖器上套一圈易破裂的纸环。次日清晨，若纸环被撑破，则表明存在夜间勃起，问题很可能源于心理而非生理。因此，对于许多中年男性的勃起不足，首先应考虑心理因素。

那么，导致心因性阳痿的首要原因是什么呢？

首要原因是妻子长期对丈夫的批评、不满、埋怨和指责。这表现为对丈夫要求过高，认为其没本事、赚钱少、能力差，或不够体贴等。这些评价均是从妻子个人的标准和视角出发的。

这里涉及一个重要的心理学概念：**暗示**。

**暗示**是指向他人传递一种能够影响其感觉或行为的信息，但并非直接明说。  
**公式表示：暗示 ≈ 间接的信息输入 → 引发心理与行为变化**

例如：
*   **明示**：直接对孩子说“宝宝，去睡觉”。
*   **暗示**：对孩子讲“孙悟空打累了，躺下睡着了”，从而间接引导孩子产生睡意。

在婚姻中，许多妻子虽未直接说出“你是个没用的混蛋”，但长期的埋怨、指责本身就在持续传递一个强烈的暗示：“你不行”、“你没用”。

这种长期的、负面的心理暗示，会使丈夫在潜意识中逐渐接受并内化这种否定评价，最终可能在性功能上表现出“真的不行”，从而导致心因性阳痿。

---

本节课中，我们一起学习了心因性阳痿的普遍性及其首要心理成因。关键点在于理解：**长期的负面评价和指责，会形成强大的心理暗示，足以影响伴侣的生理功能**。认识到问题是心理性的，是寻求积极改变和修复关系的第一步。
