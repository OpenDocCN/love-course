# 076.入世十三劫-红尘入世第一课：02【第二劫】看清世间真实的宏观与微观竞争 👁️

在本节课中，我们将要学习如何正确看待世界上的竞争。许多人感到痛苦，根源在于看待事物的角度、方位以及对比的对象出现了偏差。本节课将为你解析宏观与微观两个层面的竞争本质，帮助你摆正位置，减少不必要的烦恼。

## 概述：痛苦的根源与对比

很多人感到痛苦。痛苦源于看待事情的角度、方位以及对比的对象不同。

很多人喜欢看情节凄惨悲苦的电视剧。他们从别人的痛苦中寻找慰藉。幸福感来自对比，痛苦也来自对比。

因此，你需要清楚这个世界上的竞争：你和谁比？你能和谁比？以及你如何和别人比？这是你要解决的问题。当你解决这个问题后，会发现许多问题不再是问题。因为你比较的对象错了，这是自寻烦恼。

总而言之，就是自己的欲望大于自己的能力，自己的能力无法匹配自己的欲望。例如，想要追到女神但追不到，所以痛苦；想要挣到一个亿但挣不到，所以痛苦。你之所以有如此大的目标，是因为没有认清自己，不知道真实的竞争是什么。

所以，请认真听这节课。当清楚你要和谁比、你的竞争对象是谁时，你会发现内心充满喜乐。当内心喜乐和充满安定时，你散发的能量场是完全不同的。

## 宏观层面的竞争：找准你的生态位 🗺️

竞争分为两个层面：宏观和微观。

不要认为和你一块上班的同事、同学，或出现在同一场所的人，和你是一样的人。你们本质上不同，只是恰好在同一个地方上学或上班。你们背后的资源不可同日而语。

对方可能是富三代、官二代，或家里上三代持续积累。为什么对方毕业后能立刻找到好工作？对方大学时吃吃喝喝，你也跟着学，你能学得了吗？你永远只能学别人的“形”，学不到别人的“神”。对方大学逍遥快活，毕业后却能进入好单位，担任国企秘书等岗位。而你如果盲目效仿，结果会完全不同。

永远不能只看别人表面在做什么，要看别人背后的行为和举止，思考他为什么可以这样做。你们看似在做一样的动作，看似都不努力上学，但结果完全不同。对方毕业找到好工作，你毕业却找不到工作。为什么看似一样，结果却不同？因为你们背后的资源不同，你们不在一个生态位。

所以，你们不构成直接的竞争关系，你无法跟他比。很多人比错了对象。狮子和狮子竞争，羚羊和羚羊竞争。人与人之间的竞争，不是绝对值的竞争，而是社会排位的竞争。

你要根据现在所处的社会位置，去关心应该跟谁比。例如，如果你处于社会前10%的位置，却非要跟那0.001%的顶尖人群比较生活方式，就像让我去跟收藏古董名笔的富豪比较一样。我虽然不穷，但没有资格去比，因为祖上积累不同。虽然我挣钱能力不弱，但无法跟他比，我们不在一个生态位。

这就是很多人痛苦的来源：比错了对象。我应该跟谁比？我应该跟与我同样阶层、同样出身、同样年龄的人比。我比他们优秀太多，比他们更有钱，那么我应该感到幸福和快乐。

一旦比错了对象，好也是不好；一旦比对了对象，坏也是好。所以，没有绝对的事实与真相，只有认知的角度和切入的方位，这导致了巨大的差异和你内心的悲欢离合。

这就是很多人不明白的：为什么自己寒窗苦读数十载，却拼不过别人？你当然拼不过，别人的努力是三代人换来的，凭什么输给你寒窗苦读数十年？

所以，你要清楚真正的竞争对手是谁：是跟你社会起点、排位接近的那帮人。他们才是你的标杆。你要比的是：在顺势中，谁能跑得更快；在逆势中，谁不会被落下。这是我讲的宏观上的竞争。

## 微观层面的竞争：竞合大于斗争 🤝

上一节我们介绍了宏观上如何找准竞争坐标系，本节中我们来看看微观层面的竞争，即我们平时直接接触的这些人。

与你同在一個生態位的人，你大部分的同学、同事，他们不是你的敌人。你和他们不是敌我状态，而是“竞合”状态——竞争与合作的状态，而且合作大于竞争。

这个话我不能讲太深。许多富人、上层社会在做一件事：解构中下层社会，让他们无法形成合力。我只能讲到这了。对方要分化，但你要“合众”。“合众”是聚合众人之长，你才有可能逆风翻盘，混入上流社会。

你怎么能上去？一定是众人托着你上去的，一定是众人希望你能上去，作为他们的代言人。我只能讲这么一点点。他们需要有一个“代言体”，作为他们实现阶层跃升希望的标杆，给他们一丝希望。不然的话，他们的人生就丧失了希望。我上一节课也讲了，要充满希望的勇气。如果希望丧失了，就没有勇气了。我这话讲得很隐晦，也不能讲得太深。

在同一个办公室，如果两个背景相同的人，一个被提拔了，肯定是因为客观上他更努力、更能够胜任、更让领导满意、让客户更认可、让同事更服气。因为支持他的人多，他自然就赢了。而不是因为他花时间精力把另外一个人斗倒了。

所以不要单打独斗。单打独斗注定被淘汰。如果每个人都不具备“合众”的能力，力量是非常弱的。

我建议各位看一部电影：《穿普拉达的女魔头》。电影中，由梅丽尔·斯特里普饰演的主编是时尚界的权威。当她面临被辞退的危机时，逆风翻盘的机会来了。因为她早已笼络了许多关键资源，如主持人、模特、设计师等。如果老板开除她，这些资源都会被带走。老板还敢开除她吗？老板雇佣她，不仅仅因为她能力强，更因为她具备了谈判的筹码和条件。这就是合作的能力。

所以，各位在微观层面的竞争，不要树敌太多。当大家都希望你失败时，你就失败了。你一定要有一部分“拥趸”，这部分人能够全心全意配合你，是跟着你能“吃到肉”、有实现阶层跃迁机会的人。你能给到他们实际利益。

不要搞那种“杀敌一千，自损八百”的内卷和消耗战，只为了迎合某一位领导，结果把其他人都得罪了。一旦失去关键支持或客户，你就会人心尽失，一无所有。

所以要具备让很多人支持你的能力，而不是仅仅依靠某一个人喜欢你。靠一个人太难，因为这位领导也可能失势。他一旦失势，同生态位的人都想要你失败，那你就会败得很惨。

## 总结：重新定义你的竞争策略 🎯

本节课中，我们一起学习了如何看清世间真实的竞争。

人和人之间的真实竞争，不是面对面的争斗和决斗，而是体现在：
1.  **宏观上的社会排名**竞争：你需要与同生态位的人比较，在趋势中跑得更快。
2.  **微观上的合作能力**竞争：你需要凝聚志同道合的伙伴，而不是制造敌人。

因为社会资源从宏观上看是无穷尽的，我们要开放地看待竞争与合作的关系，找到志同道合的伙伴，“组团打怪”，共同提升社会排名。而不是一个人单打独斗。

所以，我在节目中也经常讲：偷偷变好，不需要去帮助那些不需要帮助的人。我们都是同样的价值观和理念，希望大家一起提升阶层排名。这才是我们要做的事情。

不必盯着远方的巨轮。先登上小船，慢慢划，总有一天能划到巨轮那儿。你指望巨轮过来接你，那是不可能的。

看待竞争有两个维度：宏观层面与微观层面。当你的竞争对手摆对了位置，很多痛苦自然就消失了。当你知道哪些人应该当朋友，哪些关系是竞合关系，摸清楚了，你就知道你的策略应该是什么。

希望各位好好努力，实现阶层跃迁，翻身把歌唱。
