# 012.【完结】育儿愈己的心理课：第01集 儿童发展心理学导论 (1) 👶🧠

在本节课中，我们将要学习儿童发展心理学的基础知识。这门学科对于理解儿童的成长以及我们自身的过去都至关重要。

---

在我学习心理学的过程中，有三门课程被认为特别重要。其中一门是儿童发展心理学，第二门是心理学史，第三门是认知心理学。

儿童发展心理学之所以重要，主要有两个原因。

第一个原因是，大多数人最终都会养育自己的孩子。在培养孩子的过程中，关于生理健康的知识，国家和医疗机构已经提供了大量信息。然而，从心理层面去理解孩子，却缺乏系统性的知识支持。因此，我特别看重儿童发展心理学。

第二个原因是，每个人都拥有自己的童年。我们的童年经历在很大程度上影响着我们未来的生活。这种影响是如何发生的？具体通过哪些层面产生影响？我们可能听过“三岁看大，七岁看老”这样的说法，但对其中的机制并不清楚。我们甚至不清楚自己是如何长大的。儿童发展心理学能帮助我们更好地梳理自己的过去，并学会如何更科学地抚养下一代。

从心理咨询的流派来看，精神分析学派的核心就在于探究童年经历。这与儿童发展心理学的研究领域是紧密相连、相辅相成的。

既然要学习儿童发展心理学，我们先从宏观框架上纵览一下这门学科。它的研究对象是儿童，目标是理解“发展”的过程与规律。

接下来，我们首先来看看婴儿的感觉发展。儿童的发展遵循特定规律。婴儿最初的发展是感知觉发展，它可以进一步分为“感觉”和“知觉”。

以下是婴儿的五种基本感觉发展，其发展顺序各有不同：

1.  **触觉**：婴儿出生后首先接触的就是触觉。医院会进行“抚触”。顺产婴儿经过产道挤压，其皮肤单位面积受到的刺激量远高于剖腹产婴儿。此外，顺产是一个与母体逐渐分离的过程，而剖腹产是瞬间分离，这会给婴儿带来不同的心理感受，例如“分离焦虑”。关于触觉及其心理影响，我们后续会有专题课程详细讲解。
2.  **味觉**：味觉是婴儿判断是否是母亲的重要依据。婴儿能够分辨乳汁的味道是否来自自己的母亲。
3.  **视觉**：婴儿初看世界时视觉比较模糊，之后才逐渐变得清晰。
4.  **听觉**：婴儿需要分辨爸爸和妈妈的声音。因为在婴儿期，生存完全依赖父母的照顾。在生与死的本能驱动下，婴儿会调动所有感官来确保父母在无微不至地照顾自己。当你的生存依赖于他人时，你也会竭尽全力去确认对方是爱护你的。
5.  **嗅觉**：婴儿需要闻到妈妈的味道。

综上所述，婴儿通过多种感觉的综合来辨认母亲：**触觉（妈妈的体温）、味觉（乳汁的味道）、视觉（妈妈的脸型）、听觉（妈妈的声音）、嗅觉（妈妈身上的味道）**。因此，婴儿感觉发展的根本目的，是为了确保自己能更好地生存，确认母亲始终在爱自己和抚养自己。没有生存需求，就没有相应的感觉发展。

上一节我们介绍了感觉，本节中我们来看看知觉的发展。知觉发展主要包括三个方面：

1.  **深度知觉**：指感知深度和距离的能力。经典的“视崖实验”就是测试婴儿是否具备深度知觉。实验让婴儿在覆盖玻璃板的“悬崖”边爬行，观察他们是否会爬过“悬崖”。
2.  **物体知觉**：指识别物体是什么以及其形状的能力。儿童对世界的认识往往从识别具体物体开始。
3.  **空间知觉**：指对前后、左右、上下等空间方位的感知能力。

除了感知觉，语言发展也是重要一环。语言发展一般分为三个层次：首先是词汇积累，然后是句子构建，最后是语法掌握和口语表达。这类似于我们学习外语的过程：先背单词，再学句子，最后练习语法和会话。关于儿童发展的更多细节，我们未来可能会开设专题课程进行深入介绍。

接下来，我们了解一下幼儿生理发展的规律。它遵循四个主要原则：

以下是四个发展原则：

*   **上下原则**：发展顺序从上到下。例如，先发展头部（大脑），然后是躯干，最后才是腿脚，所以婴儿最后才学会走路。
*   **大小原则**：发展顺序从大肌肉群到小肌肉群。先发展头部和躯干，然后是手臂和腿部，最后才是手指等精细部位。这对应着“大动作”和“精细动作”的发展。
*   **近远原则**：发展顺序从身体中心到外围。离躯干近的部位（如心脏、肺部）先发展，离得远的部位（如手指）后发展。
*   **头尾原则**：发展顺序从头到尾，这与上下原则类似。

以上是生理发展规律。既然我们这是一堂心理课，今天我们将着重介绍心理发展的规律。这部分内容非常有趣，当年学完令我深感震撼，让我明白了自己是如何长大的，也意识到父母在教育过程中可能无意间犯过一些错误。

儿童心理发展的核心规律，可以用一句话概括：它是一对矛盾。即 **“以具体形象思维为主导的认知水平”** 与 **“渴望独立参与社会实践的需要”** 之间的矛盾。简而言之，孩子很想自己做很多事情，但他们的能力和经验有限，往往做不到。

大家通常认为童年是快乐的，也希望给自己的孩子带来快乐的童年。但童年真的快乐吗？我们需要奠定一个客观的基调：童年往往并不轻松。试想，将一个能力不足的个体置于一个完全陌生的环境，他首先需要确认环境是否安全、周围的人是否友善。孩子同样面临这个问题。

我们看待问题不能只从自己的角度出发。你爱孩子，但关键在于**如何让孩子确信这一点**。童年所有的安全感、快乐和舒适，都必须建立在一个基础上：**让孩子确信你是爱他的**。只有建立了这种确信，孩子才能从焦虑和担忧中走出来，真正的快乐才能进入他的内心。如果做不到这一点，即使成年人进入一个新环境也会忐忑不安，更何况是孩子。

话不多说，我们正式进入心理学理论的介绍。
