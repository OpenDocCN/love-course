# 口才训练营第118课：🎭 违反常规练幽默玩练解析

在本节课中，我们将一起解析“违反常规练幽默”主题下的三个小故事。通过理解这些故事的幽默逻辑，你将学习如何构建和讲述出人意料的幽默情节，从而锻炼你的思维与表达能力。

## 故事一：邻居们会说什么

第一个故事是关于淘气的小明和他爸爸的对话。我们来看看这个故事的幽默点是如何通过“违反常规”的回应来构建的。

故事内容如下：淘气爱打架的小明把邻居家的小强打哭了。爸爸把他叫来质问，并问他是否知道邻居们在背后如何议论他。小明怯生生地表示知道但不敢说。在爸爸的保证下，小明说出了让爸爸意想不到的话。

以下是故事的核心对话：
```
爸爸：你知不知道邻居们在背后都怎么说你啊？
小明：我知道，但是我不敢说。
爸爸：你说，你说对了，我就不打你。
小明：邻居们都说，这熊孩子跟他爸爸小时候是一模一样的。
```

这个结尾的幽默之处在于，它没有按照常规逻辑发展（例如小明认错或找借口），而是将矛头巧妙地转向了爸爸，暗示爸爸小时候也是如此，从而产生了出人意料的效果。

有些学员提供了不同的幽默结尾，例如“有其父必有其子”，这与原意相似。还有学员构思了“为什么每次打的是一个人，却哭的是两个人呢？”，这个结尾隐含了爸爸之后也会打哭小明，同样创造了幽默。

## 故事二：砸了什么

接下来，我们解析第二个故事。这个故事通过层层递进的对话，最终揭示一个令人啼笑皆非的结果，是练习制造悬念和意外感的很好范例。

故事中，小明向妈妈坦白自己犯了错，并寻求原谅。他告诉妈妈自己砸碎了爸爸的酒瓶，妈妈因为小明态度好而表示原谅，并好奇地问他在家里砸了什么。

以下是推动故事发展的关键对话：
```
小明：我把爸爸喝酒之后的那个酒瓶子给砸碎了……
妈妈：啊酒瓶子呀，没事的……可是你在家里砸什么呢？
小明：哦，我砸了一块石头。
妈妈：石头？咱们家哪有石头啊？
小明：就是在一个铁圈上面的，那个石头亮晶晶的……
妈妈：……就是你以前还戴过呢，但是你一直都不戴……
（妈妈意识到小明砸的是她的钻戒）
```

原版故事的结尾是砸了妈妈的手表，但钻戒这个设定更能制造强烈的意外感和幽默效果，因为它价值更高，且符合小孩喜欢亮晶晶物品的逻辑。

一些学员也贡献了精彩的原创结尾。例如，有学员构思小明是“用酒瓶子砸电视里的动画角色光头强”，结果砸碎了酒瓶。还有学员设计小明是“帮爷爷砸慢了的手表”。这些结尾都遵循了“违反常规”但逻辑自洽的原则，从而产生了幽默。

需要注意的是，构思结尾时要符合基本逻辑。例如，有学员提到“砸爸爸的脑袋”，这个设定缺乏合理的动机铺垫，且不符合常理，幽默效果会大打折扣。

## 故事三：不全是抄的

最后，我们来看第三个关于考试的小故事。这个故事的幽默源于对问题字面意思的“狡猾”回应，展现了另一种违反常规思维的技巧。

故事中，小明考试考得好，但老师怀疑他抄袭。面对老师的质问，小明承认抄了一些，但没有全抄。

故事的幽默高潮出现在以下对话中：
```
老师：那你说说哪些不是你抄的？
小明：名字是我自己写的。
```

这个回答巧妙地回避了“抄袭试题内容”这个核心问题，而是抓住“哪些不是你抄的”这句话，给出了一个完全正确但毫无意义的答案（名字当然是自己写的），从而产生了幽默。

这种幽默方式的关键在于，抓住对方话语中的逻辑漏洞或进行 literal-minded（字面化）的理解，从而给出一个出乎意料却又无法反驳的答案。

## 课程总结

本节课我们一起解析了三个通过“违反常规”制造幽默的小故事。我们看到了幽默可以通过多种方式构建：**转移矛头**、**制造意外结果**以及**利用字面逻辑**。

希望你能将解析的这三个故事，连同课程提供的另外三个故事一起进行练习。讲述这些幽默小故事能一举两得：一方面锻炼你的幽默思维和创造力，另一方面提升你脱稿叙事和表达的能力。
