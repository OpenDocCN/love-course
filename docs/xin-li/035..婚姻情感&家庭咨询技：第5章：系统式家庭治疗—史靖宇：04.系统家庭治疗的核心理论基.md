# 035..婚姻情感&家庭咨询技：第5章：系统式家庭治疗—史靖宇：04.系统家庭治疗的核心理论基

在本节课中，我们将要学习如何运用系统观来理解个体在家庭中的位置，以及系统式思维与传统线性思维在处理家庭问题时的核心差异。我们将通过具体的案例，探讨如何从个体、家庭关系、代际传承及社会文化等多个系统层面理解问题，并掌握系统家庭治疗的核心工作原则。

## 从个体到系统：理解症状的多层视角 🔍

上一节我们介绍了系统家庭治疗的一些重要核心理论。本节中，我们来看看如何将个体置于系统之中进行理解。

当我们面对一个家庭时，我们既关注系统，也关注个体。个体本身也构成一个系统，包含其生理因素、心理因素等重要组成部分。

因此，我们看待不同系统层面时，都采用系统观。例如，当一位来访者或一个家庭坐在面前时，我们会思考每个人作为独立个体的独特性。

个体有其生理因素，有其成长过程中形成的重要信念，也有其固有的情感表达方式。同时，我们也会将这些个体因素置于其家庭背景中审视。

这有助于理解家庭中为何会形成特有的互动与沟通模式，这些模式与个人的个体背景存在联系。例如，在一个家庭中，孩子、母亲或父亲可能是呈现症状的人。

若先将此人置于其个体系统中观察，我们可以理解其症状的个体根源。假设我们的来访者（Identified Patient, IP）是父亲，他出现了焦虑症状。

我们可能会考虑其个体系统中的生理因素。也许这位父亲天生比较敏感，或警觉性较高。

我们还需要审视他在成长过程中形成了哪些重要信念，例如：他认为什么重要、什么有意义、什么会引起他高度重视或紧张。

此外，我们需考察他用何种方式表达情感。带着对个体的这些了解，我们再观察其当前核心家庭中发生了什么。

夫妻之间呈现何种互动与沟通模式？当丈夫出现焦虑症状时，他会先向谁表达担忧、不安和害怕？他与妻子会围绕症状形成何种互动？

当夫妻出现这样的互动后，孩子又会做出何种反应？这一系列变化之后，家庭关系可能围绕症状发生何种改变？或许只有当家庭呈现某些症状时，我们才更清晰地看到每个家庭成员扮演的角色以及家庭等级结构。

我们可能看到，这位有焦虑表现的父亲，是家庭中权力地位最高的人，在经济和家庭决策上是做主的人。但当他出现症状后，他一方面努力维持控制地位和一家之主的身份，另一方面也可能更多地与妻子交流。

家庭关系因此发生变化。原本僵化固定的亲子边界可能变得灵活，孩子更多地参与到父亲的生活中，父子间有了更多互动。

由此可见，我们既在关系的背景下、从不同系统层面理解症状，也要看症状对整个家庭系统造成了何种影响，从而寻找症状与家庭关系之间的联系。

## 拓展视野：代际与社会文化系统 🌐

同时，我们还可以将症状放在更大的系统中理解。从代际视角看，这位男士与其父母之间存在何种互动模式？

在他的父母那一代，是否存在特殊的情感或行为方式？也许在父亲的原生家庭中，也存在一个容易焦虑的人。

你可能会发现，这位父亲说他的父亲也是要求很高、特别自律、在家庭中是一家之主的形象，似乎也比较焦虑。这成了他们家庭传承的一种特点或行为方式。

甚至，当症状出现后，这位男士有机会审视自己与父母之间是否存在未解决的情结或愿望。也许这位父亲一直努力实现父亲对他的期待。

父亲可能传递的重要信息是：成功很重要，事业很重要。但这位男士可能更多地意识到家庭也很重要。正是这种代际传递的信息与他当前核心家庭关系的现状之间的冲突，可能造成了他的焦虑。

因此，焦虑既可以从当前家庭来理解，也可以从更大的代际系统视角来审视，它们之间都存在联系。

如果我们再看最外层的环，将其置于社会文化背景下，我们会思考：当今社会如何定义成功？什么样的人可被称为成功？

我们可能发现，当今社会对成功男性的要求更加多元化。既要求他在社会成就一番事业，又要求他在家庭中承担教育子女的责任，与配偶保持亲密关系，同时还要满足上一辈的期望。

因此，若将其置于更大的社会文化背景中，他的焦虑或许更好理解：有太多的价值条件和期望让他难以权衡，从而表现出焦虑。

正是症状的出现，给了他一个机会去思考如何平衡这些不同的期望，他想要过什么样的生活，以及他的家庭在症状出现后可能发生的变化。

若能将从不同系统视角理解人的症状，就可能为家庭和来访者提供更多可能性，使其能做出自己想要的选择。

## 思维模式的转变：从线性因果到循环互动 🔄

我们刚刚探讨了如何在不同层面理解症状。接着，我们来看看系统观与传统看待问题的方式有何差异。

传统模式如何看待问题呢？我们往往认为，既然这个人出了问题，就要在这个人身上找原因。

例如，太太抱怨先生在家交流很少，一回家就喜欢躲在房间打游戏或看手机。传统模式可能会考虑先生出了什么问题，也许他需要调整行为。

我们往往很在意原因出在哪里，会问“为什么”，采用的是一种线性因果的思维方式，即寻找单一原因。

通过这种思维方式，我们可能会认为：这位先生在情感表达方面有缺陷，或者性格特别内向，或者就是对妻子不够关心。

如果用这种线性思维在个体身上找原因，我们会更多地聚焦于呈现问题的个体内在。

我们会更多地从过去搜集线索，看这个人如何形成了这样的思维方式、情感表达方式和行为模式。

我们会从他的整个成长历程和过去经历中寻找一个原因。但我们会发现，即使找到原因——例如给他贴上“情感表达有欠缺”、“不够善解人意”或“性格冷漠”的标签——也往往不能解决问题。

## 系统观下的循环思维模式 🔁

如果我们换一种视角，用系统观来看问题，情况则不同。我们一再强调，任何症状的出现，都应放在家庭关系的视角中理解。

真正需要改变的不是这一个人，而是需要改变他们的关系。我们需要思考：其他家庭成员在家庭中做了什么，使得丈夫一回家就想待在自己房间里？

我们会采用循环思维的方式。也许在这个过程中，孩子和妻子无形中固化并加强了他的症状。

例如，我们试图找到相互的影响。也许太太很想与丈夫交流，但当她第一次表达时，丈夫没有给予及时回应。

可以想象，如果妻子有沟通意愿但未得到及时回应，她可能将更多情感转移到孩子身上，与孩子建立更多连接。

一旦她与孩子建立了更多连接，当丈夫回到家时，他会发现自己很难加入母子阵营。家庭氛围可能不适合他参与互动，无形中又将他推远，他可能就待在房间里，感觉未受邀请。

而这可能是丈夫内心的感受。但妻子会觉得丈夫不愿参与互动，感到更加失望，从而将注意力更多地投入在孩子身上，与孩子有更多互动。这反而让丈夫更难加入家庭互动。

因此，这有时是一个相互影响的过程。我们很难说第一个触发点是什么，但能看到相互影响的过程。所以，我们会针对互动模式开展工作。

我们更多地聚焦于现在，让她们意识到：当妻子做了什么时，丈夫更可能一个人待着；当丈夫做了什么时，会让妻子更加感到失落，从而对丈夫提出更多要求。

我们会让她们发现，问题不是出在一个人身上，而是她们在相互影响。她们其实是合作者，共同造就了当前看似有问题的模式。

## 如何打破互动循环？ 🛠️

那么，如何打破这样的循环呢？

首先，我们可以帮助家庭一起审视，她们的家庭中相互之间是如何影响的。她们的行为、情感和反应是如何联系在一起的。

以下是一个例子。

假设一个家庭求助的原因是孩子不上学。我们看到的是孩子不上学，但这只是家庭互动中的一个镜头或场面。

我们看看如果围绕孩子不上学，这个家庭是如何联系在一起的。也许在孩子不上学后，妈妈表现得非常焦虑和担心。

她会更多地表达焦虑，方式可能是特别唠叨，听起来像在抱怨。她可能会比较心烦，更多地抱怨丈夫经常晚上不回来，她一个人照顾孩子、做家务非常辛苦。

当她这么表达时，也许会让丈夫感到非常抵触，从而更加逃离，更少参与家庭活动。一旦出现这种模式，夫妻矛盾可能更突出。

当夫妻矛盾更突出时，孩子就更难以安心上学，表现出更多抗拒上学的行为。当孩子这么做时，妈妈会更加焦虑。

这就形成了一个循环。我们很难讲起点在哪里，但我们至少可以做些什么去打破这个循环，让它无法继续进行。

例如，我们可以从改善夫妻关系入手，也可以改善亲子关系（父亲与孩子、母亲与孩子的关系）。只要其中有一环发生变化，他们能够以不同的方式回应对方，这个循环圈就无法继续进行下去。

## 系统家庭治疗观总结 📝

我们来总结一下什么是系统家庭治疗观。

系统家庭治疗以系统论和控制论为理论基础，其特点是把家庭看作一个整体，关注人际系统中的互动性联系，同时也重视个体与环境的交互作用。

我们之前提到个体与环境。什么是环境？我们每个人生活在家庭中，家庭就是个体的环境。个体与环境的交互，既包括家庭环境，也包括家庭之外的大环境，这也是我们要注意的一点。

此外，系统观强调要以发展的、全面的、积极的、多样化的视角，来看待家庭中呈现的问题。

家庭治疗的诞生，不仅是带来一种新的治疗形式，更体现了一种认知和思想范式的转变，要求治疗师在思维方式上进行变革和适应。

我们可以看到，家庭治疗非常重要的一点是强调互动。我们意识到，任何问题只有放在其家庭整体背景中理解，才是一种可以带来改变的方式。

这不仅意味着把所有家庭成员请到治疗室，更是要邀请治疗室里的所有人一起探索他们的家庭中究竟发生了什么，看到他们相互之间的关系。

一旦他们之间的关系、交流模式或对问题的看法发生变化，我们就找到了解决问题的可能性。

## 系统家庭治疗的工作原则 ⚖️

最后，我们讲讲带着系统观工作时需要注意的原则。

以下是系统家庭治疗的几个核心工作原则。

1.  **在关系情境中理解问题**：作为咨询师和治疗师，在理解问题时，一定要将其放到关系情境和背景中理解。就像前面提到的，丈夫焦虑、孩子抑郁或妻子强势，都不要孤立地看。要思考：是什么样的家庭互动模式造就了这样的性格特点？要将问题放在关系情境下审视。

2.  **看到症状在系统中的合理性**：我们认为，出现的任何问题和症状，在其关系背景和家庭系统中都有其合理性。所谓的功能不良，也是所有家庭成员共同努力的结果。他们其实是希望家庭变得稳定、和睦、幸福，但他们努力和解决问题的方式，恰恰限制了发展。因此，我们要看到家庭成员的努力，同时让他们意识到，这是他们联合起来共同经历的一场“冒险”。

3.  **采用新的视角与思维方式**：作为家庭治疗师，我们要采用新的看问题角度和思维方式。不再是从过去寻找原因，而是更多地关注家庭有什么资源，看看可以做些什么让问题得以解决。我们不是聚焦于过去，而是努力理解现在，并创造更好的未来，寻找能够解决问题、让他们拥有更美好生活的方法。

4.  **具备自我反思能力，保持合作姿态**：作为一个系统家庭治疗师，要具备自我反思能力，始终把自己看作是与来访者、与家庭合作的位置。我们不是专家或权威指导者，而是要与家庭合作。我们要始终看到自己只是一个观察者，反思我们给家庭传递了什么信息、带来了什么扰动、提供了什么新观点。我们始终带着这种反思工作，不断从过程中学习，并可以将自己的表现作为对来访者、对家庭的一种示范，让他们也能从中学习并发生变化。

## 总结 📚

本节课中，我们一起学习了系统家庭治疗的核心理论基础下半部分。

我们探讨了如何从个体、家庭关系、代际及社会文化等多个系统层面理解个人症状，认识了系统观下的循环互动思维与传统线性因果思维的根本差异，并掌握了系统家庭治疗的四大工作原则：在关系中理解问题、看到症状的合理性、采用资源取向的新视角以及保持合作与反思的治疗师姿态。

理解并运用这些理念，是进行有效系统家庭治疗实践的关键。
