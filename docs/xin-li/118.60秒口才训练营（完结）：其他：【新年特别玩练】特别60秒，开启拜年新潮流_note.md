# 课程118：🎤 60秒口才训练营（完结）· 新年特别玩练：用特别60秒开启拜年新潮流

在本节课中，我们将学习如何运用60秒语音，告别传统群发祝福，打造走心、有趣、特别的拜年新方式。

---

春节临近，我们会收到大量拜年信息。多数祝福是群发的短信或微信，内容雷同，缺乏新意。即使修改了称呼，后续内容仍是套话，难以让人产生深刻印象。

另一种常见方式是发送红包。收到红包时固然开心，但这种喜悦难以持久。红包无法传递更丰富的情感与心意。

也有人认为，关系亲近应直接打电话拜年。但春节期间大家都很忙碌，不便长时间通话。即便接通，往往也只能说几句客套话，无法深入交流。

传统的拜年方式普遍缺乏新意。作为60秒口才训练营的学员，今年我们可以尝试用一段特别的60秒语音来拜年。

或许有人觉得，用60秒讲话拜年有些难度，不知从何说起。我们可以借鉴自我介绍课程中的“有趣、有用、特别”三个关键词，并稍作调整，应用于拜年场景。

将“有用”替换为“走心”。拜年语音的核心在于能否传递真挚情感。因此，我总结了拜年语音的三个关键词：**走心、有趣、特别**。

以下是围绕这三个关键词展开的具体思路。

**走心**
对于你生命中重要的人，如领导、长辈或挚友，你们之间必定有难忘的故事，或他/她曾对你产生重要影响。你可以录制一条60秒语音，讲述一个真实发生的故事或他/她对你的积极影响，结尾附上新年祝福。这种有细节、有故事、真情实感的祝福，会让对方感受到与众不同。

**有趣**
你可以准备一个笑话或段子。与其空泛地说“祝你快乐”，不如直接用一个笑话让对方当下就开心起来。如果担心自己讲得不好笑，可以精心挑选网上的经典笑话，并多加练习。重要的是，通过你的声音和语气传递诚意。录制小视频也是不错的选择。关键在于，避免使用传统的文字形式。

**特别**
你可以发挥个人才华，让祝福方式与众不同。例如，为对方唱一首歌、创作一首诗或一副对联。总之，尽可能展现你的独特之处。

此外，你还可以运用之前学过的“一二三理论”来组织拜年内容。拜年是主题，你可以从身体、事业、家庭三个维度送出祝福。或者，按照祝福对象的家庭角色（如姑姑、姑父、表妹）或辈分，分层表达祝愿。60秒的时间足以清晰表达。

---

本节课中，我们一起学习了如何构思并录制一段特别的60秒拜年语音。我们总结了三个核心原则：**走心、有趣、特别**。

*   **走心**：讲述你与对方之间真实、动人的故事。
*   **有趣**：精心准备一个笑话或段子，传递即时快乐。
*   **特别**：发挥个人才华，或用“一二三理论”分层表达祝福。

期待大家运用所学，尽情发挥创意，用特别的60秒语音，共同开启拜年新潮流。
