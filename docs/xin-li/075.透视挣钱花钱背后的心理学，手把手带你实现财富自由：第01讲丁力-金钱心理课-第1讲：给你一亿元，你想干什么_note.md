# 💰 金钱心理课 第01讲：给你一亿元，你想干什么

在本节课中，我们将要学习金钱背后的心理学，并通过一个简单的练习，开始探索你与金钱的真实关系。我们将了解限制性信念如何影响你的财务状况，并初步认识金钱的本质。

---

上一节我们介绍了课程的主题，本节中我们来看看如何通过一个具体的练习来开启自我觉察。

首先，请为我们的一个体验做一些小小的准备。

以下是准备工作所需的物品清单：
*   一张空白的白纸。
*   一支笔。
*   一个安静的环境。

当你准备好这样一个环境之后，现在可以在你的白纸上写一个“钱”字。只写这一个字。字写得越大越好。

当你写完“钱”这个字之后，请你仔细地看这个字，并且把自己的感受描述出来。

现在你需要做的事情是闭上眼睛。当你闭上眼睛的时候，去想象这个“钱”字会给你带来什么样的感觉。如果这个“钱”突然变成了一亿元，而你拥有了它，你想做哪三件事情呢？

以下是跟随练习时可能出现的画面示例：
*   第一，海边。有了钱，就可以说走就走地旅行。
*   第二，购买许多漂亮的衣服和化妆品。
*   第三，购买别墅或汽车。

很多人在描述自己拥有一亿元的时候，他们写下的仍然是现在的生活。他们忘记了，当你真正拥有一亿元时，你的选择会变得非常多，而且这些选择可能都不是你目前能想象出来的。

说到如果有一亿元最想做的三件事，我表达一下我的愿望。因为我现在是创业者，所以我在想：
*   第一个愿望是，我会投资很多心理专家，让心理学走进更多人的生活。
*   第二件事情是，我会投资于我的父母，希望他们晚年生活特别好。
*   第三个部分是，我可能会投资我的孩子。

当你写完并思考之后，仔细地看一看刚才所写到的这个“钱”字和你的三个愿望。你可能会突然发现，你不太认识“钱”这个字了。

当时在做很多线下课程的时候，很多小伙伴把字写出来之后，突然发现这个字既熟悉又陌生，越看越不像。有人觉得“钱”字像刀，很尖锐。也有人在写“钱”字的时候写得特别小，在一张大白纸上只占一个小角落。还有的人就写得特别大，一张纸都占满了。

但是你会发现，当你在白纸上写下的这个字，其实跟你的内心活动有着很大的关系。当你写得特别大的时候，你对金钱的渴望是很强的。当你写得特别小的时候，你可能会感觉钱好像跟你没什么关系。这其实就是“我需要它，但是我不爱它”。

甚至很多人在想象与金钱在一起的画面时，会觉得很多金钱像下雨一样从天而降，自己在那里很开心地跳。但是很多人也会觉得金钱快把自己砸死了，很害怕，想赶紧打把伞。甚至有人想到钱会觉得挺脏的，避之不及，觉得金钱就像传染病一样可怕。

我自己本身也是对金钱有很多负面信念的人。我也是从自己身上看到了我与金钱的关系，然后才有了这个金钱课程，才会深入地去探讨为什么我那么有创造钱的能力，却没钱。

其实很多人都是这样的。很多人都会有一种感觉：我挣钱能力不差，我一年的收入也很高，可是我好像还在负债，我好像还在被钱所控制。但我花钱的时候，心里会很紧张，买很多东西我会自责。如果不用在自己身上，买给父母或周围的人，我都觉得是舍得的。可是你把相同的价值用在自己身上，你就很恐慌，觉得钱自己用了怎么办。

所以讲到这些部分的时候，我们都会去关注一个很核心的问题：对于你而言，金钱意味着什么？如果你搞不清楚金钱对你来讲意味着什么，其实你会对金钱赋予很多错误的观点、信念，甚至很多错误的思想和体验。

在做线下课程体验的过程中，你会看到大家形形色色对于金钱的各种信念。有些人表面看起来好像觉得金钱很棒，其实内心是排斥甚至厌恶的。他觉得钱很需要，可是一点点都不享受它。所以他可能很有钱，可是他不快乐。

刚才提到的问题“金钱对你意味着什么”，我猜想有的小伙伴可能会写下：**金钱 = 爱情**。因为如果是男生就会觉得我有钱，就会有很多女孩子喜欢我。女孩子会觉得，有了钱就能打扮自己，甚至整成喜欢的样子，这样就会有爱情。也可能有人会觉得：**金钱 = 健康**。好像只有有钱了，才会有健康的身体，可以去很高档的健身场所等等。

而且很多人赚钱的目的，就是为了防止自己生病。这很神奇：牺牲了健康去赚钱，然后再用钱来治病。这就是我们说，我们为什么会对金钱产生很大的认知差异，就是因为我们没搞清楚金钱的本质是什么。

那么金钱，我们首先要想象它是一个中立的工具。它并非是万恶之源，也并非是快乐之本。它仅仅是对你而言的一个工具。但很多人对金钱而言，觉得它不是工具，他赋予了太多东西：荣誊、价值感、存在感、别人的羡慕、别人的嫉妒，甚至别人的看得起。

特别是很多人不明白金钱其实并非快乐之本的时候，那么他很容易就以为我钱多，我就很幸福、很快乐。所以为什么这个社会上有那么多人只追求金钱，并不追求快乐，他以为金钱等同于快乐。可是当他发现拥有很多金钱之后，并没有那么多快乐。所以很多成功人士，其实后来也得了抑郁症。他也有很多钱，为什么没有解决快乐的问题？

这就是我们要去看到的：原来我们在金钱的本质上面，赋予了太多负面的思想和负面的感受。而且这些感受和想法，其实都是来自于我们限制性的认知。那么限制性的认知是怎么建立的呢？它其实是建立在你的成长经历里面的。

---

本节课中我们一起学习了如何通过书写“钱”字来觉察内心对金钱的感受和潜在信念。我们认识到，许多人未能实现财务自由，可能并非能力不足，而是被潜意识中关于金钱的**限制性信念**（例如 **金钱 = 爱情** 或 **金钱 = 健康**）所束缚。金钱的本质应是一个**中立的工具**，但我们常常为其附加了过多的情感和错误期待。下一讲，我们将继续探讨这些限制性信念的来源。
