# 课程032：情感心理咨询技能大全（完结）第06章：结构式家庭治疗—陈发展：05.家庭中的三角化 👨👩👧

在本节课中，我们将要学习结构式家庭治疗中的一个核心概念——**家庭中的三角化**。我们将探讨三角化的定义、形成机制、动力模式及其在家庭系统中的表现，并通过具体案例帮助初学者理解这一重要概念。

上一节我们介绍了家庭结构、子系统和边界的基本概念。本节中，我们来看看当家庭系统功能不良时，一种常见的病理互动模式：三角化。

## 什么是三角化？

在正常的家庭三角关系中，如果成员和子系统的分化良好、界限清晰，他们可以各自执行功能。在情感上他们相互连接但不会过度卷入。这即是我们上次提到的**清晰的子系统边界**。

**公式**：清晰的边界 = 我能理解和支持你，但我知道这是你的（情绪/问题），不是我的。

在家庭中，如果一个子系统的分化不良，或者家庭结构功能失调（称为失功能家庭），面对压力时就难以解决问题。此时家庭中会产生许多焦虑情绪。家庭治疗师鲍文提出，家庭系统的核心情绪就是**焦虑**。

当夫妻处理彼此间的焦虑情绪时，常采用以下几种模式。以下是夫妻应对焦虑情绪的四种主要模式：

1.  **三角化**
2.  **冲突**
3.  **保持距离**
4.  **关系倾斜（失功能）**

其中，**三角化**是今天重点讲解的部分。它往往是后三种模式导致的结果。当夫妻之间无法融合、相互理解或支持，且通过冲突、争执或哭闹都无法解决问题时，他们常常会通过将焦虑**转移到第三方**来缓解紧张。这个过程就是三角化。

## 三角化的定义与核心

三角化是鲍文提出的概念。其基本定义是：**通过第三者的介入，转移了两个人之间的冲突**。必须符合此定义才称为三角化。

**重要**：例如，妈妈和孩子关系好，这不一定是三角化。只有当这种“好”的目的是为了转移妈妈对爸爸的不满时，才构成三角化。否则有滥用此概念的风险。

三角化只是**缓解**了冲突，并未**解决**冲突。如果两个人的关系紧张，但分化程度较好，他们可以在二人子系统内解决冲突。如果分化程度不高、界限不清，双方都希望对方让步，无法通过沟通妥协解决问题，他们就常会拉入第三个成员，企图通过三角关系来缓解焦虑。于是，**两个人的冲突变成了三个人的事情**，但核心问题并未解决，只是被转移和掩盖了。

## 三角化的实验研究：米纽庆的发现

米纽庆借鉴了鲍文的定义并进行了拓展，他通过实验研究为三角化提供了心理学依据。他研究了一个叫“科林斯”的家庭，该家庭有两个患糖尿病的女儿，大女儿特德的血糖尤其不稳定。

研究团队将家庭成员请到评估室。在访谈初期，只有父母在场讨论他们之间的冲突，孩子们在单向玻璃后观察。此时，父母的紧张度（通过自由脂肪酸水平测量）显著升高。

**关键发现**：当在访谈中段将大女儿特德引入房间，父母开始将冲突转向她并寻求她的支持时，一个有趣的现象发生了：**特德的焦虑水平急剧升高，而父母的焦虑水平迅速下降**。

这个实验表明，孩子的加入缓解了父母之间的冲突，但代价是孩子**牺牲自己**，转移了家庭的焦虑。访谈结束后，特德花了很长时间才使焦虑水平恢复到基线，而妹妹所需时间短得多。米纽庆由此得出结论：家庭冲突可以转移，且常通过孩子（尤其是父母关注或愿意为父母“献身”的那个孩子）来实现。

后续李维榕博士（米纽庆的学生）进行了大样本研究，测量皮肤电、脉搏等指标，也发现了类似模式：当夫妻讨论彼此关系冲突时，孩子的焦虑水平远高于夫妻讨论孩子缺点的时候。**孩子更在意的是父母之间的关系**。焦虑最高峰出现在夫妻讨论冲突后陷入沉默、开始疏离时，孩子特别害怕父母放弃关系。

## 三角化的形成要素与动力机制

三角化的形成需要几个要素。首先，存在一个**二元关系**（如夫妻），他们之间有无法解决的冲突或焦虑。当他们无法在二人内部解决时，就可能拉入第三者，使焦虑迅速得到缓解。

三角化关系产生后，原来的二元关系会变得稳定（例如，“为了孩子我不离婚”），但整体的系统焦虑水平并未降低，甚至可能增加，只是从夫妻二人转移到了第三方身上。

米纽庆提出了三角化的两个基本家庭动力机制：

1.  **绕开**
2.  **跨代联合**

下面我们详细看看这两种机制。

### 1. 绕开

绕开是指夫妻双方将无法解决的冲突，转移到关注孩子的消极面上。例如，夫妻一致认为“都是孩子的问题”（如打游戏、不听话）。如果一个家庭寻求帮助时，一致指责孩子，治疗师就需要小心评估其夫妻关系。

**案例**：一个家庭因孩子打游戏、成绩下降来咨询。父亲后来抱怨治疗师没给建议，并说“孩子的问题都是妈妈太宠爱导致的”。这句话实际表达的是**对妻子的不满**。孩子成了夫妻冲突的“替罪羊”。这种模式还包括父母过度关注孩子的缺点、叛逆等。

**治疗师常问的问题**：“如果你的孩子问题解决了/去上大学了，你们夫妻二人在家里会过怎样的生活？你们如何解决彼此的冲突？”

### 2. 跨代联合

跨代联合是指夫妻尝试获得孩子的支持来反对另一方。这可以表现为：
*   一方对孩子说：“都是你妈妈/爸爸的错。”
*   一方过度关注孩子，形成联盟（如“婚姻已死，但亲子关系不能死”）。

**关键**：联盟的目的是**针对第三方**（反对另一方）。如果不是为了反对另一方，仅仅是关系好，则称为“联合”。长期形成的跨代联合可能导致一方与子女关系过于紧密（界限不清、分化不良），另一方则变得退缩。这种过于紧密但看似“正面”的关系，可能构成**隐形的三角化**。

在实际家庭中，绕开和跨代联合两种机制常常并存。例如，丈夫指责孩子问题源于妻子溺爱（绕开），同时希望获得孩子支持来指责妻子（跨代联合）；而妻子则可能通过过度宠爱与孩子形成联盟。

## 三角化中的结盟形式

在三角化中，结盟有多种形式：

1.  **联合**：两人关系较近，但并非针对第三方。
2.  **结盟/联盟**：两人关系紧密，目的是针对或反对第三方。这可以是**秘密结盟**（如妈妈私下对孩子说爸爸不好），也可以是**公开结盟**（如孩子当面反对爸爸）。
3.  **多边结盟**：孩子陷入分裂，感到需要同时支持父母双方，内心冲突巨大。这在孩子不希望父母分开的家庭中很常见。

## 三角化的衍生形式

除了基本形式，三角化还有一些衍生形态：

*   **亲职化**：子女顶替了父母一方的角色，成为“小丈夫”、“小妻子”或家庭的“代言人”。这常见于夫妻关系疏离、角色缺失的家庭，或留守、贫困家庭中孩子过早承担家庭责任的情况。
    *   **代码/比喻**：如同歌曲《听妈妈的话》中“想快快长大，才能保护她”所描述的角色。
*   **天使化/核心化**：在多代同堂（如独生子女家庭）中，孩子成为所有成年人（父母、祖父母）关注的核心。当成年人的教育或关爱意见不一致时，孩子被卷入多种冲突，需要照顾多方情绪，尤其在隔代抚养家庭中更为复杂和艰难。

本节课中我们一起学习了**家庭三角化**的核心内容。我们了解到三角化是夫妻将无法解决的冲突焦虑转移给第三方（常是孩子）的过程，它只能缓解而非解决冲突。我们探讨了三角化的定义、米纽庆的实验研究证据，以及其形成的两个关键动力机制——**绕开**与**跨代联合**。此外，我们还分析了三角化中的不同结盟形式及其衍生形态，如亲职化与天使化。理解三角化有助于我们在临床实践中识别家庭系统的病理互动，从而更有效地进行干预。
