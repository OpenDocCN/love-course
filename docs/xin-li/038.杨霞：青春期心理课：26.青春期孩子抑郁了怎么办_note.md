# 038.杨霞：青春期心理课：26.青春期孩子抑郁了怎么办_note

在本节课中，我们将学习如何识别青春期孩子可能出现的抑郁情绪与抑郁症，并了解其不同类型、成因及相应的应对方法。作为家长，掌握这些知识对于支持孩子健康成长至关重要。

## 识别抑郁的信号

上一节我们介绍了课程主题，本节中我们来看看如何识别孩子抑郁的早期信号。

如果我们发现青春期的孩子出现了以下多种表现，就需要引起注意：
*   上课打不起精神。
*   不爱与人说话。
*   爱哭。
*   食欲下降，体重下降。
*   逃学，病假多。
*   情绪低落，消极悲观。
*   厌世。
*   有自残行为和自杀的念头。

这些迹象表明孩子可能已经出现了抑郁情绪，严重时则可能发展为抑郁症。

## 抑郁症的三种成因

识别信号后，理解其背后的原因同样关键。抑郁症主要源于三种不同的成因。

### 1. 化学性抑郁症
这种抑郁症与遗传因素有关。家族中可能体内含有的“快乐激素”（如**血清素、多巴胺**）水平较低。这类情况通常从小就有家族史。

**核心概念**：由于体内化学激素水平过低，导致情绪问题。**公式表示为：遗传倾向 → 神经递质水平低 → 抑郁情绪**。

对于这种情况，应采用药物治疗。药物能最快捷地补充不足的化学物质，迅速改善症状，且不影响智商与学习。家长不应因担心药物类别而因噎废食。

### 2. 压力性抑郁症
这种抑郁症由持续的高压环境引发。例如，孩子连续数月处于紧张学习状态，面临考试高压，总担心落后，同时承受着老师和家长的过高期望。

**核心概念**：长期压力超出承受范围，导致情绪崩溃。**代码逻辑可类比为：while (压力 > 承受力) { 情绪健康--; }**。

应对的关键是**减压**。家长不应将孩子所有时间排满课外班。必须保证孩子每周至少有一整天的休息时间，最好用于户外运动。如果已出现抑郁，则可能需要彻底休息一段时间，从几天到几个月不等。提前预防、劳逸结合远胜于事后被迫长期休学。

### 3. 认知性抑郁症
这种抑郁症源于消极的思维方式。孩子容易看到事情的负面，多愁善感，思维容易走极端和片面。

**核心概念**：**消极认知模式 → 负面情绪与行为**。在青春期，情绪发展与思维发展不同步，大约到25岁才能同步。这就像“两驾马车”方向不一，导致个体内心痛苦。

家长需要帮助孩子建立全面的认知方式，引导他看到事物的两面性，在危险中看到机会。家长自身的言行榜样至关重要，应避免在孩子面前持续抱怨社会、工作或他人。

## 特别关注：自残与自杀风险

过渡到更严峻的话题，我们需要特别关注抑郁可能引发的极端行为。

并非所有孩子一开始就有自杀念头，可能先从自残开始。如果发现孩子有用小刀划胳膊、用烟头烫皮肤或用笔在身上划等行为，就是明确的自残信号。

抑郁症的核心症状之一是“求死”念头，这与焦虑症的“怕死”不同。当孩子经常在言语中流露出“不想活了”、“想自杀”时，家长必须高度重视。调查表明，几乎所有实施自杀的人，在半年前就会开始流露相关言语或信号。这些绝不是玩笑或气话，而是危险的求救信号。

## 抑郁症的深层原因与终极目标

最后，我们来探讨抑郁症的一个深层原因及其根本解决之道。

一个特别重要的抑郁症成因是**人生目标的缺失**。根据抑郁理论，这是抑郁症的一个终极原因。

**核心区别**：
*   **抑郁症**：没有目标，感觉“生无可恋”，认为一切毫无意义。
*   **焦虑症**：有多个目标，处于“患得患失”的状态。

因此，解决抑郁症问题的终极目标是**帮助孩子建立起正确的人生目标和学习目标**，让他找到生活的意义和方向。
