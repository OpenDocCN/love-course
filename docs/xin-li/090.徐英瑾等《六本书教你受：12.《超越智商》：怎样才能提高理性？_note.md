# 课程090：如何提高理性？🚀

在本节课中，我们将学习如何克服非理性思维，提升决策质量。我们将探讨避免“认知吝啬鬼”行为和修复“心智程序障碍”的具体方法。

上一节我们介绍了导致理性障碍的两大根源：认知吝啬鬼和心智程序障碍。本节中，我们来看看如何解决这两大问题。

略有遗憾的是，《超越智商》这本书重在解释理性的概念以及非理性的原因，在提供解决方案方面的论述并不是特别详细。毕竟斯坦诺维奇是一位大学研究性学者，而不是时时在你身边为你排忧解难的“鸡汤大叔”。

不过话又说回来，当我们对问题本身及产生的原因拥有足够的认知时，解决方案其实已经在手边了。理论科学家们的意义在于为我们提供一个放之四海皆准的理论框架，让我们可以在这个真理框架下进行自我分析、自我挖掘，最终找到最适合自己的解决方案。

## 1. 如何避免成为“认知吝啬鬼”？

以我个人的理解，要避免成为认知吝啬鬼，照着这些吝啬表现的反面去寻找执行方案准没错。

具体来说，就是在思考与分析问题时，克服思维惰性，始终带着谨慎的态度三思而后行。多问问自己在这个问题上我是不是太自私、太自我、太片面、太偏见、太容易相信表面信息了，等等。

**核心行动公式**：**反思 = 质疑初始判断 + 多角度审视**

## 2. 如何修复“心智程序障碍”？

针对心智程序障碍方面的非理性表现，提高理性认知的解决方案，归纳起来其实只有两个。

一，安装正确的心智程序。
二，避免装载错误的心智程序。

这就像我们走路，首先你要学会正确的走路方式，然后你走在路面上时要尽量避开马路中间连续出现的泥坑，也就是跟那些拒绝质疑、拒绝证伪、拒绝可能带有欺骗性的人和事保持距离。

具体来说，我们年轻时代的求学经历和家教，其实就是你的家庭、社会和国家尝试在给你安装的认知体系与心智模型。而在离开学校、进入社会之后，我们依旧要不断补充我们自己对客观事物的正确认知，不断从智者的经验中积蓄最科学、理性的思维方式，来为我们装上可以识别所有病毒的超级防火墙。

乔布斯的一句名言我特别喜欢：`Stay hungry, stay foolish.` 这两句话的中文翻译得特别好，叫“求知若饥，虚心若愚”。

## 3. 查理·芒格的理性思维模型

最后，我想借用查理·芒格所推崇的心智模型，结束对本书的解读。查理·芒格最推崇的人生信条只有三点。

以下是这三个核心信条：

一，**简化复杂性**。我们要培养化繁为简、抽丝剥茧的分析能力。不要泛滥，不要被正常的复杂性吓住，更不要被歪曲的复杂性所吸引。

二，**消除知识盲点**。充满好奇心地吸取并建立关于世界万事万物的正确认知，尽可能地不给污化心智程序的侵袭留下可乘之机。

三，**多科学角度**。当我们真正遇到问题时，不要慌乱、麻痹，或者盲目自信。要学会多视角、多维度的科学分析方式。

总而言之，在人生漫长又短暂的旅程当中，我们会遇到各种各样的问题、挫折和诱惑。我们一方面要抱着“活到老，学到老”的态度，逐步建立与完善自己的认知体系。另一方面，在遇到问题时，要通过理性科学的思维方式避开人生道路上一个又一个的“非理性深坑”。

本节课中我们一起学习了提升理性的具体方法：通过反向操作克服“认知吝啬鬼”倾向，并通过安装正确程序与避免错误程序来修复“心智程序障碍”。最后，我们借鉴了查理·芒格的思维模型，掌握了简化问题、消除盲点和多角度分析这三项核心原则。恭喜你，又学完了一本书的核心思想。
