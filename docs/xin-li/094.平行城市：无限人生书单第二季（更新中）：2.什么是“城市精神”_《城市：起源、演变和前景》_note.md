# 📚 课程094：平行城市：无限人生书单第二季（更新中）—— 2. 什么是“城市精神”？《城市发展史：起源、演变和前景》

在本节课中，我们将要学习美国学者刘易斯·芒福德的经典著作《城市发展史：起源、演变和前景》。我们将探讨城市的本质，分析其作为“容器”与“磁铁”的双重角色，并以古希腊和古罗马城市为例，深入理解“城市精神”的构成以及城市兴衰的内在逻辑。

## 作者与思想渊源

上一节我们讨论了中国古代城市的演变，本节中我们来看看西方古代城市的盛衰。我们选择芒福德的这本书，正是因为其从历史起源的角度，系统分析了西方城市的诞生与发展。

刘易斯·芒福德于1895年10月19日出生于美国纽约。他先后就读于纽约城市学院和哥伦比亚大学，并在纽约大学学习社会学。1914年，即第一次世界大战开始时，芒福德接触到了城市规划理论，并深受英国学者帕特里克·格迪斯的影响。

格迪斯并非城市规划科班出身，因为当时该学科尚未建立。他最初攻读生物学，后转向社会问题研究。从达尔文到赫胥黎，再从格迪斯到芒福德，存在一条若隐若现的社会达尔文主义思想链条。格迪斯认为城市如同一个活的有机体，可以运用生物学和生态学的方法研究其发展动力，芒福德深受此思想影响。

1938年，芒福德出版了《城市文化》一书，这本人类学意义上的城市理论著作使其个人声誉达到顶峰，甚至登上了《时代》杂志封面。1961年，66岁的芒福德完成了集毕生功力之作《城市发展史》，并因此获得美国国家图书奖。

## 城市的本质：容器还是磁铁？

在了解了芒福德的背景后，我们进入核心议题：城市的本质是什么？芒福德在序言中提出了一个精妙的类比：本书开篇叙述了一座城市，这座城市象征性地就是一个世界；而本书结尾则描述了一个世界，这个世界从许多实际内容来看，又已变为一座城市。

这个类比引导我们思考：城市究竟是更接近一个**容器**，还是更接近一块**磁铁**？

*   **如果城市只是一个容器**：那么它就像一个碗，可以装下任何内容物（如美索不达米亚的大麦、雅典的橄榄）。这样的容器会破裂，内容物会散落，城市便会瓦解。那么，今天的伦敦、纽约与古代的雅典、耶路撒冷只是同一种容器吗？答案恐怕是否定的。
*   **城市更是一块磁铁**：除了本地居民，城市的关键在于吸引了陌生人、外来者、商人、逃难者甚至入侵者。这些人在城市发展的每个阶段都做出了特殊贡献。缺少这些人，城市或乡镇就会陷入沉闷偏狭的风气。因此，城市的核心功能在于**不断积累和传播文化与文明**。

我们熟悉的精神产品（诗歌、小说、戏剧）和日常生活中的大量发明创造，绝大多数都诞生于城市，或至少在城市中得到充分发展和传播。爱默生有句名言：“城市是依靠记忆而存在的。”短期记忆可以通过建筑、纪念碑保存，它们能穿越时间。但精神和文化更为重要，它们构成了城市的灵魂。

## 🏛️ 以古希腊为例：城市的精神高峰

理解了城市作为“磁铁”的特性后，我们以古希腊城市为例，具体看看什么是“城市精神”。

今天去雅典，能看到真正的古希腊建筑遗址并不多。著名的柏拉图学院遗址，在今天看来可能只是一片草地。然而，我们主要通过柏拉图遗留的**文本**，而非城市遗址，才能对雅典当时的精神面貌建立深刻认识。

古希腊的政治、经济、社会、文化制度都作用于城邦，使其成为人类精神文化的高峰。同时，古希腊城邦始终处于两种思想冲突之中：

以下是古希腊城邦内部的两种核心思想张力：
1.  **规定界限的冲突**：总有人试图规定法律、正义和互助合作的界限，反对家庭和家族所要求的特权。
2.  **启蒙的冲突**：总有人试图用逻辑、数学、理性和道德，将智力从野蛮无知的状态中解放出来。

在这些思想张力下，古希腊文明达到了很高水平。在芒福德看来，雅典是一座**知识之城、思想之城、智慧之城、民主之城和美學之城**，它是人类精神的高峰，这也是我们今天对其念念不忘的主要原因。

然而，这些精神文明并未能帮助古希腊城邦抵御历史的惊涛骇浪。更原始的力量——古罗马的军事暴力，轻易地抹掉了辉煌的雅典城市文明。这引出了我们的下一个思考：除了精神，城市还需要什么？

## 🏟️ 以古罗马为例：城市的物质功能与衰亡

讨论了精神文明高度发达的古希腊后，我们转向在军事、政治和经济上更为成功的古罗马，看看城市除了精神以外的实际功能，以及其衰亡的原因。

有一种观念认为，城市是由农村慢慢集聚演化而成的。但芒福德引用材料证明这种观点是错误的。许多城市是在很短时间内突然出现的。只要领袖给出明确指令，一座城市可以在几年甚至几个月内建成。**城市并不一定是社会逐渐演化的结果，它甚至常常是社会演化的原因**。

古罗马城市具有鲜明的规划特色，是为了某些明确目的（军事防御、宗教信仰、经济贸易、政治统治）而修建。我们首先探讨其军事防御功能——城墙。

以下是古罗马与古希腊在城墙建设上的主要区别：
*   **古希腊**：城邦多半先有城市，后因感受到威胁而逐渐修筑城墙。
*   **古罗马**：城市往往从建造城墙开始修建，城市与军事防御功能直接关联。古罗马士兵擅长修建防御工事，坚固的城墙是其快速崛起的重要原因。

随着罗马帝国版图扩大和人口增长，城市必须面对许多现代性问题，因此修建了大量大型公共工程。

以下是古罗马城市应对人口增长所修建的主要公共工程类型：
*   **供水与排污系统**：著名的排水沟、排水渠包含高超工程技巧，有些至今仍在使用。
*   **公共建筑**：道路、广场、神庙、公共浴场等。
*   **大型娱乐设施**：如可容纳五万以上观众的斗兽场，用于角斗和戏剧表演。

然而，隐患也正在于此。帝国人口数千万，分布在数百个城市中。原有的城市规划和制度逐渐受到挑战，**城市规划永远赶不上人口增长**，于是“城市病”暴露出来。

芒福德指出，古罗马城市主要面临以下“城市病”：
1.  **传染病**：人口聚集导致痢疾、伤寒、疟疾、鼠疫等疾病肆虐。
2.  **经济危机**：贵族奢靡生活导致奢侈品消费和贸易，引发贵金属外流、货币危机和商业萧条。
3.  **社会动荡**：缺乏有效的内部治理，导致流氓强盗横行，奴隶频繁暴动。

所有这些因素都沉重打击了罗马的城市生活。罗马帝国的衰亡史，也是每个罗马城市的衰亡史。在芒福德看来，城市衰落的关键在于**政治制度**。古罗马未能创造出一种能向外扩散权力、发展组织，并在庞大帝国内实现均衡治理，同时能有效沟通各地社会制度、建立平等互利交流的政治手段。

城市就像帝国的一个个细胞，每个细胞都在生病，整个有机体便会日趋无力，最终走向衰亡。这种现象在历史中不断重演，至今仍值得我们深思。

## 总结

本节课中，我们一起学习了刘易斯·芒福德的《城市发展史》核心思想。

首先，我们介绍了芒福德的生平及其深受格迪斯“城市有机体”论影响的思想渊源。

接下来，我们探讨了城市的双重本质：它既是一个装载内容的**容器**，更是一块吸引并辐射文明的**磁铁**。我们以**古希腊城市（如雅典）** 为例，看到了城市如何达到精神文明的高峰，成为知识、思想与美学的中心。

然后，我们以**古罗马城市**为例，分析了城市在军事防御、公共工程等物质层面的强大功能，同时也剖析了其因人口膨胀、规划滞后、政治制度缺陷而染上“城市病”（传染病、经济危机、社会动荡），并最终走向衰亡的内在逻辑。

芒福德的著作促使我们思考城市与整个世界的复杂联系。认识一个城市，往往意味着需要认识整个世界。从下期节目开始，我们将讨论城市分析的另一个重要维度：经济。
