# 课程011：从家长管教到孩子自律 立规矩的30个秘籍 🛡️
## 05 安全规矩：远离安全隐患，学会自我保护
### 28 正确求助：爸妈不在身边，如何向他人寻求帮助

在本节课中，我们将要学习一个至关重要的安全技能：当父母不在身边时，孩子应如何正确地向他人求助。这对于保障孩子的安全、培养其独立应对能力具有重要意义。

本节目由丁香妈妈和清平I-Femme联合播出。

---

让孩子在安全的情况下学会独立求助，实际上要比单纯限制其行动困难得多。因为我们既希望孩子拥有探索的自由，又必须保证其人身安全。在孩子年龄尚小时，父母若能成功教导这项技能，就等于赋予孩子一种终身受用的安全能力。

上一节我们探讨了安全的基本原则，本节中我们来看看如何具体教导孩子求助。

由于社会经验缺乏，孩子很难准确区分陌生人的善意与恶意。加之他们力量弱小，难以与成年人对抗，因此可能面临一些危险情况。那么，父母可以做些什么来加强孩子的自我保护能力呢？核心在于提前进行系统性的教导和练习。

以下是父母需要做到的三个关键点：

1.  **禁止孩子单独行动**：在公共场合，不应允许年幼的孩子脱离监护人视线单独活动。
2.  **保持在彼此视野内**：如果允许孩子在一定范围内自由活动，父母必须站在孩子身后，确保孩子始终在自己的视线范围之内。同时，也需要让孩子理解“父母在我的视野中”这一概念。
3.  **进行情景模拟练习**：通过“让我找到你”等小游戏，在家模拟在商场、户外等场景下与父母走散的情况。反复练习孩子此时应该采取的正确步骤，例如**“原地停留，寻找穿制服的工作人员（如保安、收银员）求助”**。

这些方法都是提升孩子独立应对技能的有效途径。这些原则说出来简单，但在实践中可能需要父母投入相当的时间和耐心。然而，为了孩子的安全，任何努力都是值得的。

---

本节课中，我们一起学习了如何教导孩子在父母不在身边时正确求助。我们认识到，通过**禁止单独行动、保持相互视野以及进行情景模拟练习**这三种方式，可以有效提升孩子的安全意识和自我保护能力。关于如何进一步帮助孩子加强自我保护，我们将在下节课中详细探讨。
