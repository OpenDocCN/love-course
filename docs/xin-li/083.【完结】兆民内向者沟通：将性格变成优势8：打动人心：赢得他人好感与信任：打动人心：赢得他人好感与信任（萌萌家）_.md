# 课程083：内向者沟通课 🎯 捕获人心的三步法

在本节课中，我们将学习内向者如何通过系统性的方法，赢得他人的好感与信任。我们将打破常见的社交误区，并掌握一套简单有效的三步法，帮助你将内向的性格转化为沟通优势。

## 概述：内向者的社交困境与机遇

内向者内心常存在一种矛盾状态。一方面不喜欢与人打交道，讨厌人多的社交场合；另一方面，又苦于无法与想认识的人快速建立良好关系，得到他人认可。

迫于职业发展和情感需求的压力，内向者有时会硬着头皮去做一些看似积极的展示，效果却不理想。有时想要显得合群与幽默，却总说不到点子上。

内向者很快会发现，不论情感、职场还是日常社交，大量机会都被外向者能言善道地拿走了。留给自己的机会和选择少得可怜。时间久了，内向者会认为这是性格缺陷导致的，是自己的命。

认命的结果是性格变得顽固和闭塞，出现愤世嫉俗的情绪，会生闷气，身上的气场一点点被消磨掉。

但内向者完全有翻盘获胜的机会。可以凭借出众的专业能力、超群的艺术感觉、敏锐的直觉和创造力，成为很多领域的佼佼者。历史上很多伟大的科学家、哲学家、商业领袖，都是性格内向甚至偏执的人。

本次分享的内容，旨在帮助内向者在专业能力之外，获得化解社交难题的方法。准确地说，就是捕获人心的技巧。这项技巧不仅适用于两性关系，让你更容易获得异性青睐；也适用于商业领域，帮助你更容易获得客户信赖，赢得好感。

听完这节课你会发现，走进一个人的内心并没有那么难。原来走进人内心，恰恰是内向者的优势。

## 破除三大社交误区

在深入方法之前，我们需要先敲碎一些常见的假象和误区。

以下是内向者在社交中常见的三个错误认知：

1.  **误区一：只有优秀出众者才值得被关注。** 内向者认为，在群体中只有那些层级更高、更优秀出众的人才会获得周围人关注。而自己是再普通不过的人，没有人会对自己感兴趣。
2.  **误区二：必须高调或哗众取宠才能被注意。** 内向者觉得，只有通过高调的自我推销，或做一些匪夷所思、哗众取宠的事情，才能引起别人注意。否则不会有人喜欢自己。
3.  **误区三：快速建立联系是天生能力，无法学习。** 内向者认为，和别人快速建立联系是性格使然，是与生俱来的能力，后天无法学习。即使学习也不会比性格外向的人更出色。

绝大部分内向学员都有类似想法。他们害怕跟别人比较，一比较就发现自己的不足。一旦发现自己不如别人，就会产生消极的自我暗示，进入一种死循环。

从今天起，请把这些错误想法抛到脑后。它们只会让你变得更糟。

## 核心理念：社会交换理论

破除上述误区，需要接受一个来自社会学的理论：**社会交换理论**。

社会学家相信，所有关系都建立在这个基础之上。其核心含义是：如果我们和对方相处，带给我们的**收益**大于或等于要付出的**成本**，我们就更愿意建立和维持这段关系。当我们发现跟某人交往“无利可图”时，这段关系就会走向终结。

我们有时不愿意和不太熟悉的人建立关系，就是因为还看不到预期收益，担心自己血本无归。你可以说这是一种经济思维，但这很正常。

这件事的关键并不在于付出和回报是否划算，而在于我们如何理解“回报”。事实上，社会交换理论中的回报或收益，既指有形收益，也必然包含无形收益。

每个人的价值一定包含了能为他人提供的无形收益，例如：对他人的**理解、尊重、同情、喜欢、欣赏、崇拜**，以及在此基础上建立起来的**友情和爱情**。

所以，当有人产生“这个人竟然这么理解我，还听我倾诉，他一定是个不错的人”这样的念头时，就不难理解了。哪怕你只是耐心倾听对方，都会收获好感。而这个好感，就是前面所说的“收益”。对方会认为，和你交往是值得的。

因此，建立自身价值，让自己变得重要，是一件挺容易的事。它有时仅仅取决于你多么在意对方。

上一节我们破除了误区并理解了社交的本质是价值交换，接下来我们具体看看捕获人心的系统性方法。

## 捕获人心的三步法 🎯

捕获人心的方法可归结为三个步骤：
1.  建立安全氛围
2.  建立初步好感
3.  赢得信任和认同

### 第一步：建立安全氛围 👁️

如果我们没有能够给对方营造一个安全舒适的交谈氛围，那么对方在心里是拒绝和我们沟通的。

有的人不愿意或勉强和我们说话时，他们的语气和目光会回避我们。具体表现是：看上去在和我们说话，但实际上通过不屑一顾、漠不关心的样子，来避免说出更多东西。这种现象叫“回避式沟通”。

产生回避式沟通的原因一般有两种：要么不想被打扰，懒得应付你；要么心里觉得不安全。所以，不论你现在要面对的是谁，最基本也最为紧要的任务，就是为彼此建立一个安全氛围。这是一切沟通的前提。

#### 方法一：调整你的眼神

眼神是语言之外最有力的沟通工具。目光会显露你此时是满怀期待、饶有兴趣，是否够真诚和善良。是否获得了尊重，有没有被期待和欣赏，也都能够从彼此的眼神中找到答案。

有人可能会问，在交谈时需要考虑眼睛该往哪看以及看多久吗？就像不必刻意控制呼吸节奏一样，眼神也同样不需要僵化的练习。需要我们练习的是能够静下心来，让自己沉浸在一段对话当中。毕竟一旦对方发现你的目光变得奇怪，就会明白那一刻你在想别的事了。

眼神存在性别差异。由于生理结构的天然差别，男人的目光范围和女人明显不一样。这个差别在于：女人的视野范围更大，而男人的目光是“管状”的。这可能是自然界对男女分工差别导致的。在远古时代，男人要外出打猎，管状视线最有利。所以男人在观察正前方事物和寻找远距离目标的能力强于女人。这也解释了为什么男人更擅长开车。

管状视线的弊端在于，它常常会被异性误解。例如，当眼前出现一位漂亮而性感的女士时，男人的眼神会变得直勾勾的。因为这一刻这位女士就成了视线中唯一可以关注的目标，于是所有目光都汇聚在了女士的脸蛋或其他部位上。

女人却不一样。女人在近距离视野和周围视野上的观察能力要比男人强。也就是说，当一个女人看起来正在注视着你的脸时，很可能也同时观察到了你身上佩戴的珠宝首饰。

了解到这种差异之后，男士们不妨让自己的目光变得轻柔一点，或者更多地集中在异性的眼睛上。因为这总比盯着女人的胸部或腿要好得多。这是在建立安全的对话氛围时必须要注意的问题。

眼神给人的印象往往决定了这个人的总体印象，也在很大程度上决定了别人会以什么样的方式跟你沟通。

以下是需要尽量避免的负面眼神表现：
*   **初次见面很快就把目光移开的人**：要么自视甚高，要么缺乏教养。
*   **不敢和对方对视的人**：大多是生性胆小、性格自卑或缺乏社会经验的人。
*   **喜欢上下扫视或频繁环顾四周的人**：要么凡事爱挑剔，要么缺乏耐心容易焦躁。
*   **喜欢俯视或爱从眼镜上方看人的人**：大多自认是强者，希望向对方传递权威和压力，而且不容质疑。

一个在眼神上让人讨厌的人，是绝对不会被别人从心里接受的。

#### 方法二：成为积极的倾听者

当你忽然感觉找不到话题时该怎么办？不妨坐在对方身边当一个好的倾听者。当然，倾听不是让你傻傻地坐在那一言不发，而是在对方说话的过程中，给予积极的回应。

首先，你可以**借力使力**。在听对方讲话时，最佳反应是顺着对方的话来回应，而不是重建话题。只要求你紧贴对方话语的脉络，不跳跃也不否定，在简短的回应当中表达出和对方相近或相同的立场。在这个时候你还可以提问。

**举例**：有一天，一个不是很熟的人在我身边抱怨说：“烦死了，一堆麻烦事真想辞职。”我觉得他想找人倾诉，于是我就问他：“工作上出什么事了吗？怎么会这么想？”听到我这么问，这位朋友说：“工作简直忙死了我跟你说，天天累得要死要活的。”然后我继续顺着他的话问他：“为什么这么说呢？你们公司其他部门也都这么忙吗？”这句话好像戳中了他的痛点，他马上说：“气死人的地方就在这，其他部门都很闲，就我们几个人累得跟狗一样。”就这样，我打开了他的话匣子，而且毫无疑问我让他感觉到了安全。

还有一种非常好用的倾听方式，叫**支持性回应**。在很多情况下，人们向你倾诉，其实并不是为了获得解决问题的方法，仅仅是为了得到更多情感上的鼓励和支持。所以，支持性回应必须包含体现赞同、鼓励、关心的话语，但是一定不能急于给出自己的评判，或直接给出建议。至于对方会做出怎么样的选择，实际上早已心中有数。

#### 第一步小结：营造亲和力

我们看，不论是眼神调整还是积极倾听，都是为了一件事：让自己更有亲和力。亲和力是一张快速通行证，它包含了真诚、善意和安全感，能迅速地打通你和陌生人之间的隔膜。

亲和力虽好，但千万不要做得太假。与其时时刻刻把僵硬的笑容挂在脸上，不如心存关怀体恤的态度。你一举手一投足间传递出的善意，都能够被对方接收到。

我们已经学会了如何为沟通打下安全的基础，接下来，让我们看看如何主动出击，建立初步的好感。

### 第二步：建立初步好感 💬

当安全氛围准备就绪，接下来就要试着和别人沟通了。我们该用什么样的方式和对方主动交流呢？常见的寒暄如“你好啊”、“你从哪来啊”、“最近忙什么呢”、“你看起来气色不错呀”等，很容易落于俗套。

为了不落俗套，就必须努力找到让对方立刻有兴趣的话题，或者跟这个人利益密切相关的话题。所谓“利益”，不仅仅指钱财，还包含了人们在乎的一系列事，例如：他的心情、荣誉感、被赞美被肯定的价值感、被理解的需要、舒适度、便利性、更好的机会、更优化的建议等等。

当我们把这些利益攸关的事，转换成一些轻松话题的时候，寒暄自然就变得简单了。

以下是四种导向的寒暄方式：

1.  **兴趣导向**：以对方的兴趣爱好为切入点展开话题。
    *   **举例**：你在路上遇到买早餐的同事，他说早上起晚了。联想到这是在世界杯期间，身为球迷的他估计熬夜看比赛了。于是你可以问：“昨天晚上看比赛了吧？”一听到世界杯，同事就来劲了。接着可以围绕足球展开话题。
2.  **赞美导向**：在谈话中表达出对对方真诚的赞美。
3.  **建议导向**：在适当的时机，给出贴心的建议。
4.  **理解导向**：表达出感同身受的理解。

对内向的朋友来说，不论哪种导向型的聊天都不必担心非要给一个话题下结论。你可以在任何时候结束它，不必有负担。聊天时，最好不要掺杂过多的个人好恶或价值判断，保持话题的边界，不涉及隐私，否则容易给人留下不舒服的感觉。

#### 方法：好话、好意、好借口

这是建立初步好感的组合方法。

*   **好话**：即表达尊重、欣赏、赞美含义的语言。前面讲到的兴趣、赞美、理解导向的话题，都可称为好话。
    *   **推进顺序**：如果你拿不准该从哪个话题聊起，不妨按以下顺序推进：
        1.  **谈论此时此刻你们正在经历的事**，和你们双方当下的观察感受相关。
        2.  **谈论对方有兴趣的事**，和他的见识能力相关。
        3.  **谈论对方在意的事**，和他的利益心情有关。
        4.  **向对方寻求建议或帮助**，因为“好为人师”是人的天性。
    *   不必和每个人聊天都按这四个步骤推进，要根据场景和对象灵活应对，打乱顺序完全没问题。不过有一点是肯定的：每个人都喜欢聊自己。当你向对方释放关注和欣赏时，大多数情况下不会被拒绝。
*   **好意**：即亲和力的外化动作。
    *   **举例**：随手替人开门、礼貌地请别人先进出电梯、帮助对方拿重物、随手帮人记下电话号码、专注地听别人说话并点头赞同等，都可以给人留下好印象。
    *   **特别注意**：要保持行为的边界。很多坏事都是好心办的。把自己认为好的东西强加给别人，只会招致反感。不论做什么，最好先征得对方同意。越俎代庖只会费力不讨好。
*   **好借口**：即善于借助沟通工具。
    *   **举例**：一位不抽烟的男性朋友，办事时总随身带一包烟，因为“递烟是男人之间最好的开场白”。一位做销售的女生，喜欢随身带润喉糖，见到客户就很自然地拿出来分享，陌生感马上就消除了。
    *   香烟和润喉糖此时都变成了沟通的工具。食物的传递比语言的传递更有效。与其费尽心思说什么，不如让一些简单有趣的小玩意，替你叩开对方的心扉。

通过好话、好意和好借口，我们已经能够有效地建立初步好感。接下来，我们将进入最后也是最关键的一步：如何将好感深化为信任与认同。

### 第三步：赢得信任与认同 🤝

经过前面的铺垫，你已经获得了对方足够的好感。这时想要赢得对方的信任和认同，就变得不太困难，不过仍然要用对方法。

#### 核心方法：自我袒露

最简单直接的方法就是**自我袒露**。这里涉及一个理论：**约哈里窗户理论**。

该理论将人的内心分为四个区域：开放区、隐秘区、盲目区和未知区。自我袒露，就是主动向对方透露一些你“隐秘区”的信息。当对方也向你透露一些他的“隐秘区”信息时，你们就完成了一次信息交换。这种交换有利于快速拉近距离，建立深层的好感。

和对方讲自己，传达给对方的信号是：坦诚和信任，是把对方当自己人的一种话语暗示。它背后的逻辑是：“你看我都跟你讲这么多我的个人私事了，你是不是也多少透露一点啊？”一般情况下，如果相互不讨厌，且存在进一步交往的现实必要，那么对方的话匣子自然就会打开。

**举例**：在一次访谈中，女嘉宾对婚外恋话题守口如瓶。我选择换一种方式，向她袒露了自己一段被类似感情伤害的痛心往事，并分享了我现在的看法。当我主动分享这些隐秘的感受和经历后，女嘉宾看我的眼神更加专注。随后，她也终于打开心扉，讲述了自己在婚姻中的无奈和情感补偿的故事。

沟通的本质是一种交换。当我们试图从对方口中获得更多，或试图走进对方心中时，只有一条捷径可以走：**主动袒露心声**。

当然，自我袒露能够发挥神奇作用，但也不要过多泄密。过多的泄密反而会让人感到恐惧，会再次设防，还会给人留下以自我为中心的印象，沟通效果适得其反。
