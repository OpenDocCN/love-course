# 口才课程010：诗词篇丨第四集《七步诗》📜

在本节课中，我们将学习曹植的《七步诗》。我们将了解这首诗背后的历史故事，分析诗中的核心情感，并探讨如何运用恰当的语气和情感来朗诵它。

## 诗作背景与故事

《七步诗》是一首在七步之内创作完成的诗。这种命名方式在中国古诗中极为独特，其背后隐藏着一个关于兄弟争斗的故事。

写这首诗的人名叫曹植。他有一个哥哥叫曹丕，两人是同父同母的亲兄弟。他们的父亲是历史上著名的曹操。

曹植从小就非常聪明，尤其在文学方面展现出过人的天赋，特别擅长写文章。其程度甚至让曹操有时看到曹植的文章后都不相信，怀疑是请人代笔写的。

曹植对此感到不高兴，声称自己落笔成章，无需代笔，并请父亲当面出题考核。曹操有时真的会当面给曹植一个题目，曹植也能当场完成。因此，曹操对这个小儿子刮目相看，非常喜爱，并一度想将自己的事业传给曹植继承。

但这引发了巨大的麻烦。因为在封建社会中，继承规则是“立长不立幼”。这意味着如果有王位或世袭官位需要传承，必须优先传给长子，而不是后面的儿子。这是当时社会的规矩。

所以，曹操想立曹植为接班人的想法遭到了很多人反对。这个消息传到了长子曹丕的耳朵里，曹丕从此记恨上了弟弟曹植。

后来曹操去世，最终事业还是由长子曹丕继承。但曹丕心里一直觉得才华横溢的曹植对自己是个威胁。曹丕称帝成为魏文帝后，便找借口将曹植关进监狱，要治他的罪。

这可能导致曹植丧命。有一天，曹丕到牢房中审问曹植。哥哥审问弟弟，这本身就是一种争斗。

## “七步成诗”的挑战

曹丕对曹植说：“你仗着自己有点才华就任意妄为，在父亲丧期做了不好的事情。”他罗列了许多罪名。这里我们可以记住一个词：**“欲加之罪，何患无辞”**。意思是，如果我想治你的罪，总能找到理由。

但曹丕心里也想，如果现在就杀了曹植，别人会认为自己心眼小。于是，他给弟弟曹植出了一个难题。

他说：“你一直说自己特别会写文章，父亲也这么说。这样吧，现在你走七步，在七步之内给我写出一首好诗。如果你能做到，我就不治你的罪，不让你死。”

此时曹植已别无选择。哥哥曹丕已经称帝，他说什么就是什么。曹植必须接下这个挑战，但这个题目非常难，需要在七步之内立刻写出一首好诗。

曹植确实非常有才华。他应了一声，含着泪与满腔悲愤，写出了这首《七步诗》。

## 诗歌赏析与核心情感

这首诗的大意是：锅里煮着豆子，锅底下燃烧的是豆秸（豆秆）。豆秸和豆子本来是同根生长的，如今却这样自相残杀、互相煎熬，为什么要这样做呢？

整首诗表达了悲愤之情，以及对兄弟相残的深刻思考。即兴赋诗能达到这样的深度，非常厉害。

我们来解析一下诗句：
*   **煮豆燃豆萁**：“豆萁”就是豆秸，即豆子脱粒后的茎秆。
*   **豆在釜中泣**：“釜”是古代炊具，相当于现在的锅。“泣”是哭泣。
*   **本是同根生**：豆子和豆秸本来是同根所生。
*   **相煎何太急**：互相煎熬为什么要这么急切呢？这是一个**反问句**。

最后一句反问，既表达了悲伤，也充满了愤怒。

## 朗诵的情感与技巧

既然诗歌的情感是又悲伤又愤怒，那么我们在朗诵时，应该运用什么样的情绪、语气和声音位置呢？请你体会一下。

以下是朗诵的要点：
*   情感基调：**悲愤**。
*   关键句处理：最后一句“相煎何太急”是**反问句**，语气应**不卑不亢**。
*   整体情绪：融合悲伤与愤怒的复杂情感。

我们来读一遍：
**煮豆燃豆萁，豆在釜中泣。本是同根生，相煎何太急。**

## 故事的结局

曹植在七步之内写成了这首诗，曹丕也就不能杀他。曹丕不杀他，第一层原因是曹植完成了诗作；第二层原因是他不想让天下人认为自己是个心胸狭窄、残杀亲兄弟的小人。

所以，《七步诗》从某个角度来说，是一首“救命诗”。
