# 口语表达课 010：四个声调与绕口令练习 🎤

在本节课中，我们将要学习汉语中至关重要的“语调”概念，特别是四个声调的区别与运用。掌握正确的语调，是说出标准、优美普通话的关键一步。

## 语调的重要性

上一节我们介绍了“正音”，即语音的规范。本节中我们来看看“语调”。只有语音和语调都过关，所说的才是标准的普通话。

如果语调不正确，即使每个字的发音都准确，整体听起来也不是普通话。例如，东北朋友说“服顺话是最标准的普通话”，每个字音都对，但语调出了问题。再如西北的打招呼方式“你吃了~”，句尾的“了”字音调拖长拐弯，这也是语调问题。

没有语调，语言会失去灵魂。试想用单一语调说话：“小朋友你好，欢迎你来看我，今天我们做什么游戏。”这听起来像机器人。世界上多数语言没有声调概念，例如英语的“bus”不会因语调改变意思。汉语的四个声调是其独特之处，我们必须认真研究。

## 汉语的四个声调

我们来集中学习汉语的四个声调。它们有正式名称，也有常用的小名。

以下是四个声调及其特点：

*   **阴平（第一声）**：高而平的调子。公式：`mā` (妈)
*   **阳平（第二声）**：从中音升到高音的调子。公式：`má` (麻)
*   **上声（第三声）**：先降后升，带拐弯的调子。公式：`mǎ` (马)
*   **去声（第四声）**：从高音降到低音的调子。公式：`mà` (骂)

需要记住的是，只有第三声（上声）带拐弯，其他三个声调都是单一方向的：平直、上扬或下降。

## 声调中的常见问题与概念

声调练习中有一些需要注意的现象和概念。

首先是一个常见问题：**阴平调值偏低**。例如，东北方言中，“喝碗小米粥”可能读成“喝碗小米zhōu（尾音下降）”。正确的读法应把一声拖住、读平：“喝碗小米zhōu（平调）”。再如“穿小貂”应读为“穿小diāo（平调）”，而非“diǎo（下降）”。

其次是一个重要概念：**语流音变**。汉语有独特的节奏和韵律，说话好听的人通常韵律感强，这离不开语流音变。一个简单规则是：当两个第三声的字组成一个词时，第一个字变读为近似第二声。

以下是几个例子：

*   `友好 (yǒu hǎo)` → 读作 `yóu hǎo`
*   `领导 (lǐng dǎo)` → 读作 `líng dǎo`
*   `表演 (biǎo yǎn)` → 读作 `biáo yǎn`

对比一下：“今天我们要表演节目，有领导来看，他们的态度很友好。”如果不变调，读成“今天我们要biǎo yǎn节目，有lǐng dǎo来看，他们的态度很yǒu hǎo”，就会非常生硬。运用语流音变后，句子就有了自然的韵律感。语流音变内容很丰富，我们先建立这个概念即可。

## 绕口令练习

现在，我们通过绕口令来具体练习声调。

首先，我们用《大猫毛短，小猫毛长》来重点练习阴平（一声）和阳平（二声）。

**绕口令文本：**
> 大猫毛短，小猫毛长。
> 大猫毛比小猫毛短，小猫毛比大猫毛长。

**练习要点：**
“猫”（māo）是阴平（一声），“毛”（máo）是阳平（二声）。练习时，注意把一声的调值读足、拖平，避免读成下降调。不要读成“大mǎo毛短”，应是“大māo毛短”。

接下来，我们用《松鼠爬松树》来重点练习上声（三声）和去声（四声）。

**绕口令文本：**
> 松鼠爬松树，松树住松鼠。
> 鼠爬松树，树住鼠，鼠住松树，鼠爬树。

**练习要点：**
“鼠”（shǔ）是上声（三声），发音带拐弯；“树”（shù）是去声（四声），发音直降而下。请仔细区分两者。

通过认真练习这些绕口令，过好声调这一关，我们就在说出富有韵律感、优美汉语的道路上，又迈进了一大步。

本节课中我们一起学习了：
1.  **语调的重要性**：它是标准普通话不可或缺的部分。
2.  **四个声调**：阴平（一声）、阳平（二声）、上声（三声）、去声（四声）的名称、特点及发音公式。
3.  **关键概念**：注意避免“阴平调值偏低”的问题；初步了解了“语流音变”中“两上相连，前上变阳平”的规则。
