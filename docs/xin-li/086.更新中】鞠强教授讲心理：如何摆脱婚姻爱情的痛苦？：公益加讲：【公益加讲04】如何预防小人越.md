# 课程086：如何预防小人越级告状？ 🛡️

在本节课中，我们将学习如何运用心理学知识，特别是基于中国社会文化中的“后台文化”特点，来有效降低职场中小人越级告状的概率。我们将探讨小人的行为特征，并提供具体、可操作的策略来保护自己，维护良好的工作关系。

## 理解小人行为与“后台文化”

上一节我们介绍了课程主题，本节中我们来看看核心概念。小人是指使用不能公开曝光的手段、在背后害人的人。小人喜欢偷偷地越级告状，当面却可能表现得非常尊重你。

要预防这种行为，首先需要了解中国人的一个重要心理特征：**看后台**。这与西方文化差异很大。在中国文化中，人们会评估你的后台（即你的支持者或靠山）是什么。如果他觉得你的后台很硬，就不会轻易越级告状。

中国的文化是一种后台文化。例如，《西游记》明显显示，有后台的妖怪和没有后台的妖怪待遇不同。没有后台的妖怪被孙悟空一棍打死。有后台的妖怪在孙悟空举棍时，其后台就会出现并救走他。

理解了“后台文化”这个特点后，我们就可以设计策略来降低别人越级告状、背后捅刀子的概率。小人告状的内容常常是当面一套、背后一套。如果一个人能当面真诚指出问题，同时也向领导反映，那此人品德很好，是为组织着想。小人则相反，其特点不是为组织着想。

## 核心预防策略

基于以上分析，以下是两个核心的预防策略。

### 策略一：展示坚实的“后台”

第一个建议是，经常在下属面前满怀深情地夸赞你的领导或你领导的领导，并且要讲一些细节化的事情。

例如，你可以对下属说：“我们的领导某某，是我人生最崇拜的人。”仅说“最崇拜”太笼统，还需要讲具体原因：“我崇拜他，因为他行动力非常高，创新很强。上次我在他办公室看到他处理某件事，真的很佩服。”

当你在下属面前满怀崇敬之情夸赞领导时，就暗示了你与领导关系很好。这让小人觉得无缝可钻，不敢轻易插一杠。

然而，很多未经训练的人会做相反的事：在下属面前发自己领导的牢骚，例如说“我们这头真是昏君”。当下级听到这些，当面可能不表态或敷衍认可，但内心会产生一个评估（可能是潜意识的直觉）：评估你与上级关系不好。关系不好，他的胆子就大了。他不仅可能越级告状，还可能对你的命令执行打折。命令执行打折会导致你工作做不好，进而在顶头上司面前更没面子，问题陷入恶性循环。

**核心要点公式：**
**公开赞扬领导 ≈ 暗示关系牢固 ≈ 降低被攻击概率**

所以，千万不要在直接下属面前发顶头上司的牢骚。相反，应该满怀深情地具体化赞扬，暗示自己与领导关系非常密切。

### 策略二：树立“强力反击者”形象

当一个人想要捉弄、害你或进攻你时，他一定会预估你的反应。如果他感觉你的反应会非常强烈，事实上就会降低他攻击你的可能性。

类比来说，你长得高大在马路上走，小偷偷东西的概率就明显较低。因为他要预估：万一偷东西跟你打起来，能否打赢你？

因此，在职场中做领导防止小人，要摆出一副“不好惹”的姿态。

例如，我到任何公司做总经理时（我不仅是教授，管理企业也很有经验），都会评估自己在员工和下属中的形象。如果形象“太好”，我会适当调整。我会告诉他们：“我在上一个单位，有个政府副处长来查我，借无关紧要的事乱敲好处、勒索40万。我们没给。我只花了几万块给侦探事务所，查出了他的小三和房产，把情况一捅，他现在进监狱了。”他们一听，会觉得这教授不全是书呆子，也不好惹，诡计多端。自然就没人来搞事了。

他害怕，是因为觉得你的反击力量很大。所以，一定要让下属感觉到：你这个人，好好相处是很好的；但如果把你惹翻了，后果也很可怕。如果这个印象没有形成，你就应着力去形成它。

**核心行为代码：**
```
if 下属感知到(你的反击力很强):
    其越级告状的动机 = 降低
else:
    其越级告状的动机 = 可能升高
```

通过以上两点，可以大幅度降低小人越级告状的概率。

## 课程总结

本节课中，我们一起学习了如何预防小人越级告状。我们首先定义了“小人”及其行为特征，并深入分析了中国社会文化中的“后台文化”心理。基于此，我们学习了两大核心策略：一是在下属面前展示与领导的坚实关系，以暗示强大的“后台”；二是有意树立一个“不好惹”、会强力反击的职场形象，以震慑潜在的攻击者。这两点在实际工作中非常关键，能有效帮助你维护职场权威和稳定。
