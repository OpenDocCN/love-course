# 022.【完结】：释放压力、管理情：第12节：乐观向上的秘籍：重拾正能量，抱怨

## 概述

在本节课中，我们将学习一种名为“慈心练习”的正念方法。这个练习旨在帮助我们直接体验爱与连接，从而释放压力、管理情绪，并提升整体的幸福感。我们将通过一系列引导步骤，学习如何将善意和祝福首先给予自己，然后扩展到他人乃至整个世界。

---

## 爱与连接的本性

上一节我们探讨了情绪管理的基础，本节中我们来看看如何通过培养慈心来重拾正能量。

在社会高压的今天，每个人常感觉自己像一座孤岛。我们努力追求完美，渴望通过外在的成就获得爱与认可。然而，物质利益的追求是一条难以填平的道路，未必能带来真正的爱与幸福。唯有学会爱，在爱中成长，我们才能回归本性，让内心找到真正的归属，并获得源自生命源头的赐福。

## 慈心练习的益处

临床心理学家发现，慈心练习能带来多方面的积极影响。

以下是慈心练习的主要益处列表：
*   缓解身体疼痛。
*   减轻精神紧张感和焦虑感。
*   减缓愤怒情绪。
*   提升共情能力。
*   增强自己与他人的社会连结。
*   提高身体健康指数与生活满意度。

当一个人内心充满慈爱时，他会自然地做出有利于自己、他人及众生的行为，同时也不会去做出伤害性的行为。所有与修习者接触的众生，都能感受到这份善意，从而能安稳、融洽地相处。如果全人类都拥有慈心，世界将会非常美妙。

## 慈心练习引导

绝知此事要躬行，接下来我们一起进行慈心练习。

**准备阶段**
找到一个安静且安全的地方，选择一个舒适而放松的姿势，可以坐着或躺着。开始练习时，先留意喉咙、胸腔、腹部在呼吸时的状态。吸气，呼气。想象每一次呼吸都在轻柔地按摩身体。留意此刻身体的状况，注意任何引起你注意的身体知觉。感受肌肉的紧张或放松程度，以及皮肤的温度。如果你愿意，可以闭上眼睛。

**感受被爱**
想象一个深爱着你的人或小动物，他可能出现在你的过去或当下，可能仍在世或已过世。想象他站在你的右边，从心里向你送出祝福和爱意，他希望你安全、幸福。感受来自他的温暖祝福和爱。

现在，邀请一个非常珍视你的人（可能与前者是同一人，也可能不是），想象他站在你的左边，送出对你的祝福和爱，他希望你健康。感受来自他的善意和温暖。

想象你的周围环绕着这些爱着你、爱过你的人，他们站在你身旁，向你送出祝福和爱，他们希望你健康、自在。感受你的整个身体被来自四面八方的爱意和温暖充盈。

**送出祝福（回馈与扩展）**
现在，把注意力带到站在你右边的他/她身上。就像他祝福你那样，把你对他的祝福和爱回馈给他。从心里送出祝福和爱。如果你愿意，可以重复下面的词语：
```
愿你自在。
愿你快乐。
愿你自由。
愿你平安。
```

把注意力带到站在你左边的人身上，把祝福也送给他。可以在心底默念下面的句子，就像你希望被祝福那样祝愿他：
```
愿你健康。
愿你远离不适。
愿你远离危险和伤害。
愿你远离痛苦和折磨。
```

**祝福中性之人**
现在，想象环绕在你周围的人中的一位或几位，他可能是你的亲人、爱人、朋友、同学、同事或其他人。这个人和你一样，希望自己快乐健康。接下来，在心里把你对他的祝福默默送给他，你也可以生成自己对他特殊的祝福。

想象一位你既不喜欢也不讨厌的人出现在面前，他可能是超市售货员、车站检票员、路边推销员、快递小哥等。具体是谁没关系，继续重复刚才的句子，表达你对他的祝福和爱：
```
愿你快乐。
愿你健康。
愿你安全。
```
你也可以生成自己对于他或他们的特殊祝福。

**祝福困难之人**
现在，试着把这种祝福转移到你厌恶的人身上。这个人曾与你有过节，让你产生失望、愤怒、鄙视、轻蔑、伤心等负面情绪。可以重复刚才的句子，对他送出祝福。

如果你觉得过程非常困难，也可以在每句话前加上：
```
我会尽我所能的，祝他快乐。
我会尽我所能的，祝他健康。
我会尽我所能的，祝他自在。
我会尽我所能的，祝他安全。
```
让这些祝福的句子弥漫在你的身体、头脑和内心。

**连接众生**
接下来，用你的温暖和善良把内心的爱与关怀带给一切众生。想象整个地球出现在自己面前，只是一个小小的球。想象自己用包容万物的心对地球传送关怀与祝福。这一刻，你和整个宇宙连接着，和所有人类、动物、植物、微生物都连接在一起。

感受在这个过程中，自己内心深处迸发出的温暖、善良与关爱。祝福自己，也祝福所有天空、大地、山川、海洋里的生灵：
```
愿他们健康。
愿他们快乐。
愿他们自由。
愿他们远离不适。
```
让你的内心充满这种爱与关怀的力量，把这种爱拓展到世界的每个角落。感受自己与宇宙的连接，这种连接源于你内心的善良和真诚。

## 总结

本节课中，我们一起学习了“慈心练习”。我们从感受被爱开始，逐步学习如何将祝福回馈给关爱我们的人，并扩展到中性的人、难以相处的人，最终涵盖一切众生。这个练习帮助我们体验内在的联结与爱，是扫除抱怨、重拾正能量的重要秘籍。

练习到此结束。此时此刻，你的身体和情绪有怎样的体验？在看待身边的人时，是否有了不同的视角？欢迎分享你的练习体会。

一起安定内心，向上生长。希望正念修炼课堂可以带给你幸福的力量。
