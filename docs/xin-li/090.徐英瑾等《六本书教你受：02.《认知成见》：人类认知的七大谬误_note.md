# 课程090-02：《认知成见》：人类认知的七大谬误 🧠

在本节课中，我们将学习《认知成见》一书的核心内容，重点探讨人类在思维过程中常见的七种形式谬误。我们将从逻辑和概率的角度出发，理解这些认知偏差如何影响我们的判断与决策。

## 宏观视角：什么是认知成见

上一节我们介绍了《认知成见》这本书的概况，本节中我们来看看本书第一部分的核心内容。这部分旨在从宏观上界定“认知成见”的含义。

首先需要明确，这本约十万字的书无法详尽讨论人类社会中的所有具体成见。成见种类繁多，例如针对种族、体型或外貌的偏见，有些甚至超出我们的想象。因此，本书无法逐一列举，而是从**形式的角度**进行探讨。

## 逻辑与概率：理性思维的基石

在日常生活中，我们主要依赖两种工具进行思考：**逻辑**与**概率**。这两者在现代社会中至关重要。

**逻辑**关乎推理。例如，女友约你在商场见面，商场分A座和B座，但通话中断未说明具体位置。如果你在A座未找到她，且之前信息无误，那么根据逻辑可以推断她在B座。其推理过程可以表示为：
```
前提1: 女友在商场A座或B座。
前提2: 女友不在A座。
结论: 因此，女友在B座。
```

**概率**则关乎可能性评估。例如高考填报志愿，你需要根据自己平时的排名、心仪大学在本省的招生名额以及历史模拟考成绩，来估算被录取的概率。如果概率过低，理性选择可能是调整目标。其核心是计算期望值，尽管现实中人们常受非理性因素影响（如购买彩票）。

现代社会建立在理性原则之上，无论是高校招生名额分配，还是彩票运作规则，都有其逻辑与概率基础。不理解这些规则，便难以在现代社会立足。

## 思维的七宗罪：常见的认知谬误

尽管逻辑与概率至关重要，但我们在现实中却常常违背它们。意大利心理学家皮埃特里·托马里（Pietro Tamarini）指出，人类在认知活动中常犯下七种主要谬误，他称之为“七宗罪”。

以下是其中几种典型的认知谬误：

### 1. 先入之见与锚定效应

**锚定效应**指我们的思维像抛锚的船，容易固着于最初接收的信息。例如在议价时，卖方先报出500万的高价，在你心中形成“锚点”。随后他降价至450万，你会觉得让步很大，交易容易达成。反之，如果提价则难以接受。我们对初始条件敏感，对后续变化不敏感。

### 2. 过度自信

在心理学中，**自信**指对自己判断正确性的确信程度。例如考试后，同学问你某题答案，你回答“选C”。当被问及把握多大时，你可能说“瞎猜的”（自信度低）或“算了三遍，肯定是C”（自信度高）。人们常高估自己的能力，过度自信的考生往往觉得自己考得好，但结果未必如此。

### 3. 易得性启发式（易被表征者易胜出）

**易得性启发式**指我们更容易想到那些容易在脑海中形成生动形象或印象深刻的事件。例如，肯尼迪遇刺后，凶手奥斯瓦尔德又在转移途中被杰克·卢比枪杀。许多人倾向于认为这两起事件是同一阴谋的组成部分，因为这样更容易理解（一个“阴谋论”能串联所有事件），减轻了大脑处理多个孤立信息的负担。但从概率统计看，两起独立事件恰好属于同一大阴谋的概率其实很低。

### 4. 后见之明偏误（事后诸葛亮）

**后见之明偏误**指在知晓事件结果后，人们倾向于认为“我早料到了”，并过度认为结果在事前就很明显。心理学家曾做过实验：向两组学生提供关于某南美小国选举的复杂材料（包含正负面信息），并分别告知他们“政客当选”或“落选”的虚假结果。然后让学生倒推选举成败的原因。结果，被告知“当选”的学生会重点寻找正面材料，而被告知“落选”的学生则聚焦于负面材料。他们忽略了那些被其他因素抵消或覆盖的相反信息。

这种谬误在管理评价中很危险。例如，领导看到员工小张业绩好，便总结其工作方法并要求他人学习。但这可能忽略了小张方法中潜在的、对公司长期发展不利的负面因素（如不正当手段）。仅从结果倒推原因，很可能找错原因。

### 5. 不合理的审慎

**审慎**本是好事，但**不合理的审慎**指在无意义的事情上过度谨慎。考虑以下场景：
- 你下周一需赴纽约见一位重要客户A，但他不确定是否有时间见你。
- 你在纽约唐人街还有一位朋友兼潜在合作伙伴B，他只要你到纽约就一定会见你。
- 你的时间只允许见其中一人。

问题是：你应该买机票去纽约吗？许多人因不确定能否见到A而选择不去。这其实是一个推理谬误。

从逻辑上看，这是一个**析取**问题：你到纽约后，**或者**见A**或者**见B，两者必居其一。用逻辑公式表示即 `A ∨ B`（A或B）。最好的情况是与A签合同，次好情况是与B合作。无论如何，去纽约至少能保证发生其中一件有利的事。如果不去，则什么都不会发生。这类似于高考填志愿，在不确定能上最好学校时，至少应填报一个次优选项以获得机会。许多人未能看到，在具体选择（见A或见B）的不确定性背后，存在着更大的确定性：你至少需要亲临现场以获得选择权。因过度谨慎而放弃行动，就犯了“不合理的审慎”错误。

## 本节总结

本节课中，我们一起学习了《认知成见》中关于人类思维谬误的宏观分析。我们了解到，本书关注的并非具体经验内容上的偏见，而是**思维形式上的常见推理谬误**。我们重点探讨了**先入之见（锚定效应）、过度自信、易得性启发式、后见之明偏误以及不合理的审慎**这几种类型。

这些谬误揭示了我们的思维如何系统性地偏离逻辑与概率的轨道。在下一节中，我们将把注意力转向一些更加具体的谬误形式，继续深入解读《认知成见》这本书。
