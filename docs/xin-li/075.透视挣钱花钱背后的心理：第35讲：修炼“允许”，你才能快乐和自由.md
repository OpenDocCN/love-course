# 075.透视挣钱花钱背后的心理：第35讲：修炼“允许”，你才能快乐和自由

在本节课中，我们将学习如何通过修炼“允许”的心态，从“事件层级”跃升至“游戏层级”，从而获得内心的快乐与自由，并最终吸引财富。

## 从“事件层级”到更高层级的转换

上一节我们介绍了不同的心理层级，本节中我们来看看如何从“事件层级”进行转换。转换的关键在于**不去抵触**。

例如，生活中会遇到许多“奇葩”的人或事。我们能做的是思考：为什么这些“奇葩”会出现在你的世界里？为什么这些人要带着你不喜欢的东西来“敲门”？根本原因在于，你尚未清晰区分**什么是我的事，什么是他人的事**。

这就像快递员送来一堆垃圾。收件人是你，但**你可以选择拒收**。公式可以表示为：
**我的责任 = 接收对我有益的事物；他人的责任 = 处理他们自己的“垃圾”。**

当快递员送来垃圾时，你的回应可以是：“这不是我的，请拿走，我不要。” 一旦你能分辨“什么不是我的”，你就会开始明白“什么是我的”，从而主动寻找那些能带来更多愉悦的人。

## 修炼核心：“允许”一切发生

作为“事件层级”的人，我会去寻找同样内心平和、喜悦的人共事。当我们与高频的人共事，关系就会提升到一个更棒的层级。

对于这个层级的人，核心修炼是 **“允许”**。即允许一切发生。

例如，当各种“奇葩”出现在生命里，你可以笑着面对：“我知道你是奇葩，我看到你了，这次表演很精彩，谢谢，再见。”

“允许”的实质是接纳自己的不完美。以下是一段自我提醒的代码，阐述了“允许”的力量：

```text
当我允许自己不勇敢时，我就不再恐惧。
当我允许自己不完美时，我就不再挑剔。
当我允许自己不快乐时，我就不再悲伤。
当我允许自己不上进时，我就不再焦虑。
当我允许自己不控制时，我就不再慌张。
当我允许自己不善良时，我就不再受伤。
当我允许自己不美好时，我就不再伪装。
```

我曾希望自己勇敢、完美、快乐、上进、控制、善良、美好。但这些期望其实是在掩盖内在的挑剔、恐惧、悲伤、焦虑、慌张、受伤和伪装。我抵触内心真实的恐惧部分，直到我学会“允许”。

## 自我觉察练习：探索你的“不允许”

以下是帮助你实践“允许”的五个问题。请认真思考并回答：

1.  **识别情绪**：看看哪些情绪是你不允许自己拥有的？（例如：愤怒、脆弱、懒惰）
2.  **追溯根源**：看看自己为什么不快乐？
3.  **检视信念**：看看你有哪些信念导致自己没有钱？
4.  **对比状态**：看看什么时候你比较快乐？什么时候你不快乐？
5.  **质问模式**：你为什么允许自己的不快乐存在于你的世界里？

回答完这五个问题后，你会瞬间明白。从今天起，作为“事件层级”的我，将不再困于事件本身。我将选择接纳一切发生。

## 跃升“游戏层级”：世界是游乐场

当你开始接纳一切，你会发现进入了“游戏层级”。因为你不再将事物视为阻碍，不再认为“奇葩”是奇葩时，你就遇不到“奇葩”了。

进入这个最高层级非常美妙。你会发现一切都被清理得很干净，人生就像一个“打怪升级”的过程。曾经的“奇葩”再也无法干扰你，旧的问题也不再是问题。你与频率很高、很开心的人一起，在所有事件中都能找到有趣的部分。

你看到了生命的真相：**世界是个游乐场，目的是让你快乐**。哪怕是黑屋子、过山车这些令你恐惧的设计，也只是为了让你体验刺激。你知道自己终会走出来，就像在迪士尼乐园走鬼屋，肾上腺素飙升却知道安全无虞。

到达这个层级的人，已经从更高视角俯视世界。他们活力四射，激情澎湃，一言一行都具有强大的感染力和号召力。与他们在一起，你会情不自禁地感到高兴。世界上只有婴幼儿和少数成年人能达到这种自在、放松的状态，每天的任务就是开心玩耍。

我常说：**所有的事情都只是游戏（Just a game），唯有爱值得认真对待。**

## 各层级关系与能量流动

处在这个状态的人，特别喜欢奉献。因为爱满自溢，能量多到不给出去会难受。你会特别愿意帮助别人。当你越帮助别人，你的能量就越强。

这种能量能帮助到最初级的“情绪层级”的人。只有第四层“游戏层级”的人有足够能量去帮助第一层“情绪层级”的人，其他层级的人容易被影响。

第二层“问题层级”的人可以和第三层“事件层级”的人互动，但他们会感到累，并且常在两个层级间跳跃。

当你处于第三层“事件层级”，并越来越多地实践“允许”，不让事件困扰你，最终就会到达“游戏层级”。在这个层级，你永远在玩（Just for fun），时时刻刻都开心。带着这份快乐的正能量，财富自然也会到来。

---

本节课中，我们一起学习了从“事件层级”跃升的关键——修炼“允许”。通过区分自己的事与他人的事，接纳一切发生，并完成自我觉察练习，我们可以摆脱事件困扰，以游戏心态俯视世界。最终，达到活力充沛、乐于奉献的“游戏层级”，在快乐中吸引财富与自由。
