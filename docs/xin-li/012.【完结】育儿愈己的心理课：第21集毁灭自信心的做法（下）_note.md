# 育儿愈己心理课 012：毁灭自信心的做法（下）💔

在本节课中，我们将继续探讨毁灭孩子自信心的后两种常见做法：“吹毛求疵”与“不公平的比较”。我们将深入分析这些行为背后的潜台词及其对孩子造成的深远伤害，并学习如何避免这些误区。

上一节我们介绍了“机会剥夺”和“归因不当”，本节中我们来看看另外两种同样具有破坏性的行为。

## 第三部分：吹毛求疵

“吹毛求疵”是一个贬义词，但有一个词与其意思接近，叫做“精益求精”。许多父母都喜欢对孩子说一些要精益求精的话。

在一次亲子活动中，我要求父母鼓励孩子。父母们当时会说：“我觉得孩子很有进步，他懂得分享，会感恩，也会独立。”他们夸了一通，但最后总觉得没说完，要补充一句才行。于是他们补充道：“他要是能把情绪控制得更好，那就完美了。”

这最后一句不说，就感觉话没说完。生活中，你是否也常说这种要精益求精的话呢？

以下是几个典型的例子：

*   “宝贝，你这次考了90分，真棒。下次你再努力一点，就能得95分啦。”
*   “我们家的孩子数学一直挺不错，就是语文不行。要是语文也像数学那么好，我就不用操心了。”
*   “我们家的孩子考了最好的初中，也是最好的高中，你肯定能考上最好的大学。”

每个家长都会在表扬鼓励后面，加上一个期待。但我要说的是，这根本不是精益求精，而是吹毛求疵。

因为有一个真相是：**你永远不会有且只有一个期待，你会有无穷无尽的期待**。这些话语看似是鼓励，但有没有让孩子真正感觉到是鼓励，真的要打一个问号。

经历过的人都会有这样的感触：听到前半部分挺开心，听到最后那个期待时，喜悦感不完全消失，也消了一半。在情感上，这种“鼓励”的作用，还不如说这种吹毛求疵，是毁灭自信心的重要原因。

你每次冒出的新期待，会让孩子觉得：“妈妈怎么对我又有一个新要求？”无穷无尽的要求，最终哪个孩子能真正在你架构起来的“阴架”上不断往上爬呢？他们最后一定会感到一种无力感，觉得“无论我多努力都无法达到他们的期望”。这种挫败是无法挽回的，他会逐渐失去自信心。

即使有孩子能勉强跟上父母的节奏，我认为这也很可悲。因为这样的孩子失去了自己规划期望的能力，就像一头只会围着石磨打转的驴，看到的只能是眼前那一点点的目标。

对父母来说，放下对孩子的期望非常不容易。哪怕只是改变说话的方式，把最后那个期望的“后缀”去掉，都很难。我看到很多人在了解这种模式后，立刻去表扬孩子，还是会犯同样的错误。

在目前“望子成龙、望女成凤”的大背景下，父母很难放下这份期望。但如果我们真的希望孩子内在的动力被激发，让他对自己有期望，那只有我们先放下。

## 第四部分：不公平的比较

现在要说第四个毁灭自信心的原因了，这个原因也非常常见，叫做“不公平的比较”。

比较无处不在。从孩子出生开始，可能就会有不少比较。小时候比谁先走路、谁先开口；长大了比谁眼睛大、谁个头高；上学后就更不用说了，比成绩、比证书、比谁的兴趣班多。

这里最打击孩子自信心的，是拿自己家的孩子和别人家的孩子去比。

如果你拿自己家的孩子和隔壁家什么都优秀的孩子做比较，这是非常毁灭性地打击了自己孩子的自信心。如果父母存在这方面问题，真的要好好反省，在说出那些话的当口立刻停下来。想想这就是一把把尖刀，你往自己孩子身上捅，怎么舍得？

我还要说另外一类家长，他们也是比，但会拿自己家的强项去和别人家的比。学了非常多的兴趣班，门门都要争强好胜抢第一。看起来这类孩子通常比较优秀，可是我还是要说，这种“抢第一”的行为，其实也是自信心不够强的表现，同样是一个需要关注到的、不好的毁灭自信心的原因。

因为在“比比比”的战场上，从来没有常胜将军。

如果父母在孩子心里种下的是一颗“我要和别人比较”的种子，你其实也同时传递给孩子一个感受：**我给你的爱，是当你好的时候才能得到的；当你不好的时候，爱和我都会远离你**。

我记得在群里有的家长会说：“我实在抑制不住啊，当孩子考不好的时候，我真是连看都不想看他。”你说你都是这么察觉的，孩子难道会感觉不到吗？孩子考试优异你就开心、更关心他；考得不好就连理都不想理。这种感觉非常伤害孩子。

同时，这种你对他的评价，会深刻影响他内心对自我的感受。因为你会让他心里没有一个坚定的标准，不知道到底什么样子才是足够令别人满意的。

这样的孩子成年后会发现，他们始终关注“别人对我的要求是什么”、“别人的标准是什么”。他会遇到三类问题：
1.  总有人在觉得我不好的时候，他该如何自处。
2.  总有人本来觉得我挺好，可一旦标准变了，我就不好了。比如在夫妻关系中，谈恋爱时都挺好，可一到婚后，他就会发现对方的标准又不一样了，这时会产生非常大的不安全感。
3.  一旦“被爱”的感受被剥夺，很多其他的感受都起不来，情绪和人际关系都会面临很大挑战。

所以我说，无论你是贬低自己的孩子，还是用贬低别人家的孩子来突出自己家的孩子，总之只要你把孩子推到“比比比”的怪圈里去，这种比较就非常不公平，对谁都不公平，不仅没有意义，而且伤害性极大。

每一个人都有所长，也有所短。有一个家长问我们：“我们家孩子成绩不好，我担心这样是不是会伤害他的自信心，他以后在学校会不会抬不起头？”我说这还真不一定。但是，如果作为父母，你都把他的学习当作唯一重要的事情，凡事只看他的学习成绩，那就难说了，那就一定会影响这个孩子的自信心。因为他所有的自信都建立在一根柱子上。

可是，如果我们不跟他比成绩，而是尊重每一个人的差异性，让他们去发展个性，孩子其实可以在很多不同的领域发展出自信。他也许成绩不是很好，但难保在别的地方就能找到自信。那他就不容易崩塌掉自信心。

所以，谁也不能保证在“比比比”的战场上能永远赢，所以千万不要把孩子往那个战场上去引。

## 总结与回顾

本节课中我们一起学习了毁灭孩子自信心的后两种做法。

分别说完了毁灭自信心的四种做法，我们来做总结。我认为四种毁灭自信心的做法分别是：
1.  **机会剥夺**：这是一种伺候得过分勤快，连打架机会都不给的行为。你的这些行为可能让孩子理解到“你不行”的感觉。
2.  **归因不当**：当你把所有原因向外部、向一些不可控的因素归因时，你的潜台词是在说“你没有可能能够做得更好”。
3.  **吹毛求疵**：总是觉得没有最好，你做得还不够好，潜台词就是在说“你总是这样不够好”。
4.  **不公平的比较**：这其实是一种有条件的爱，潜台词就是在说“你只有做好了，才配得上被爱”。

大家仔细思考就会发现，生活当中，你有没有说过类似的话？在你的行为中有没有这些潜台词：“你不行”、“你没可能做得更好”、“你总是不够好”、“你做好了才配得上被爱”？

总结这四种做法，你会发现其实生活当中，你可能在很多细小的部分、细小的行为动作中，都已经在毁灭孩子的自信心。

既然是这样，我们就应该从身边做起，从每一件小事情做起，关注孩子的自信。
