# 075.透视挣钱花钱背后的心理：第06讲：这么努力工作，为什么还是没钱_

在本节课中，我们将要学习一个核心问题：为什么许多人勤奋工作，却依然感觉钱不够用。我们将通过分析收入和支出的结构，并学习如何通过调整这两者来改善财务状况。

---

很多人感觉钱总是不够花，因为挣钱的速度似乎永远赶不上欲望增长的速度。例如，有学员反馈自己月薪八千，但房租、餐饮、交通等固定支出就占了大半，每月所剩无几，几乎成为“月光族”。

为什么总是觉得钱不够花呢？关键在于审视你的支出与收入是否对等。许多人并不清楚自己的钱具体花在了哪里。当支出项目繁多而收入来源单一时，财务压力自然巨大。

上一节我们介绍了审视财务平衡的重要性，本节中我们来看看如何具体分析。

要解决这个问题，需要从两个方面入手：一是增加收入来源，二是减少不必要的支出。通过“开源”和“节流”双管齐下，财务状况才能得到改善。

以下是具体操作步骤，请准备纸笔跟随练习：

1.  **列出所有收入项目**：例如工资、奖金、项目福利、红包、父母补贴等。计算每月总收入。
2.  **列出所有支出项目**：例如房租、水电、电话费、交通费、房贷等。尽可能详细地写下所有你能想到的支出。
3.  **分析并优化**：
    *   在收入部分，思考有哪些增加的可能性。
    *   在支出部分，区分哪些是“硬性消费”（必需），哪些是“弹性消费”（可削减）。
    *   新增一个“投资项”，即哪些部分可以投入资金或时间来提升未来的收入能力。

如果只关注收入和支出两项，很容易陷入“收入太少、支出太多”的死循环。引入“投资项”是打破循环的关键。这项投资可以是中长期的（如学习技能），也可以是短期的，它带来的额外收入将是改善财务状况的惊喜。

这就是“开源节流”。当你清晰呈现自己的收入结构后，就会明白在现有能力范围内，收入是固定的。但能力提升后，情况就会改变。

让我以自己为例。最初我只是一名咨询师，我的收入结构很简单：

**早期收入结构 = 工资 + 咨询提成**

其中，工资占70%，咨询提成占30%。我意识到工资是固定的，而咨询提成是浮动的。因此，要增加收入，我就必须增加咨询量。

一旦看明白收入结构，你就能客观地决定将时间花在哪里。当我知道需要提升咨询收入后，我面临一个选择：通过晋升来提高每小时的咨询费用（例如从300元/小时升至400元/小时）。但晋升的前提是能力提升。

很多人只想提高收入，却不愿投资提升能力，这是不可能的。就像不愿听课学习，却期望收入增长，这无法实现。

**提升能力本身就是一项重要的投资**。我将一部分资源投资于提升专业能力。能力提升后，我的职位从普通咨询师变为副主任咨询师，虽然提成比例未变，但单价提高了，总收入也随之增加。

此外，我还发现了新的收入可能性：培训、讲课和写稿。于是，我的收入结构从两项变成了四项：

**后期收入结构 = 工资 + 咨询提成 + 培训讲课 + 写稿**

当时的比例是：工资42%，咨询提成55%，培训讲课1%，写稿2%。这个结构清晰地告诉我努力的方向。

你可能会问，培训讲课和写稿只占1%和2%，为什么还要写上去？因为写上去，你才会知道在哪些领域还有增长空间。这是一种自我暗示，为你开辟新的可能性。

投资可以是学习，也可以是开拓新领域。当我知道自己可以讲课后，起初没人请我，我也觉得自己经验不足。于是我从免费讲座开始，大量练习，让别人看到我除了咨询还能讲课。

当我看到培训收入只占1%时，我知道可以增加它，因为培训的课时费更高。写稿虽然起初只占2%且我觉得很困难，但我鼓励自己：困难是因为不熟悉，只要大量练习就会改变。

因此，我明白了必须在收入结构中增加项目。

现在，请大家仔细看看你刚才写下的收入结构，思考还能增加哪些项目。就像我举例的培训和写稿，你的擅长和兴趣能否帮你挣钱？把它写进去，并以月收入为基准计算各项比例。

这是我的第二步：**增加收入项目**。

第三步是：**大量练习**。一切取决于勤奋。如果你懒惰，今天的课程就白听了。必须勤奋，要知道理想的生活取决于努力。丁力老师也是这样练习出来的。

---

本节课中我们一起学习了如何通过分析收入与支出结构来破解“没钱”的困境。核心方法是“开源节流”，并通过三个步骤实践：1. 列出并分析收支；2. 增加收入项目；3. 通过大量练习提升能力。记住，清晰的财务结构和持续的投资（尤其是对自身能力的投资）是走向财富自由的关键。
