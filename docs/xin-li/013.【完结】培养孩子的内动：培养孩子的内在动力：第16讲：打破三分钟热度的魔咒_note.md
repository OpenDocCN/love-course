# 课程013：打破三分钟热度的魔咒 🧨

在本节课中，我们将要学习如何帮助孩子将瞬间的意愿转化为持续的内在动力，从而打破“三分钟热度”的魔咒。我们将探讨意愿与感觉的关系，并提供具体的行动方案与坚持策略。

## 概述

许多家长认为孩子内在动力的生成如同马拉松的终点，充满仪式感。然而，真实的改变往往发生在平淡的瞬间。例如，一个孩子某天主动说“我去写作业了”，这个简单的行为可能标志着意愿的启动。但仅有意愿是不够的，它可能带来短暂的热情，即“三分钟热度”。要打破这个魔咒，关键在于稳定意愿背后的感觉，并辅以科学的行动方案与坚持。

## 意愿与感觉：动力的核心

上一节我们提到了孩子主动行为的瞬间。本节中我们来看看这背后的原理。内在动力的核心在于：**行为 <- 意愿 <- 感觉**。孩子说“我去写作业了”，说明他启动了**意愿**，开始了自主行为。

然而，如果仅仅启动了意愿，而没有启动和稳定意愿背后的**感觉**，就会导致“三分钟热度”。那种强烈但短暂的意愿，如同昙花一现，来得快去得也快。孩子会处于不稳定状态，在好的感觉和不好的感觉之间摇摆。

## 如何打破魔咒：两大方法

当孩子启动了意愿，其最大的挑战就是“三分钟热度”。要打破这个魔咒，家长需要从以下两个方面提供辅助。

### 1. 制定行动方案

孩子一旦启动了意愿，必须有一个与之相匹配的**行动方案**，否则意愿就会变成空想。规划行动方案是一种能力，并非听几句道理就能学会。

以下是帮助孩子制定方案的关键步骤：

*   **细化目标与步骤**：不能只安排“先写作业后玩游戏”的顺序。需要追问细节：作业量多少？可用时间多长？几点开始？几点结束？备用计划是什么？
*   **制定整体规划**：除了每日计划，还需要长期的行动方案。这需要根据孩子的实际情况（如知识基础）设定合理的阶段预期，避免冒进。
*   **形成可执行方案**：一个有效的方案应简洁、有重点、可执行。例如，可以围绕“弄懂当天知识”、“查漏补缺旧知识”、“用错题本巩固薄弱点”来构建。

通过这样的追问和讨论过程，家长正是在帮助孩子培养至关重要的规划能力。

### 2. 学会坚持

任何计划一旦进入执行阶段，加入了**时间**这个要素，难度都会倍增。对于孩子来说，坚持是巨大的挑战，需要家长保驾护航。

以下是辅助孩子坚持的要点：

*   **确保目标合理**：家长要抑制自己的“贪心”，不要拔高孩子自定的目标。当孩子冒进时，要理智地将其拉回现实。一个合理的目标是成功坚持的基础。
*   **提供及时反馈**：漫长的努力过程需要“进度条”。指导孩子回顾自己的连续表现（如主动写作业的天数），或指出其具体的进步（如错误率降低、速度加快）。这能让孩子清晰感知自己的前进速度，避免灰心。
*   **尊重孩子的自主性**：孩子启动意愿后，自主意识增强，对干涉非常敏感。家长在辅助时要特别克制，以观察和等待为主，让孩子自发地调整和改进。自发的改变才最可靠、最稳定。

## 总结

本节课中我们一起学习了如何打破“三分钟热度”的魔咒。关键在于理解动力来源于稳定的感觉，而不仅仅是瞬间的意愿。当孩子启动意愿后，家长需要通过**帮助制定科学的行动方案**和**辅助孩子学会坚持**来保驾护航。在此过程中，务必尊重孩子的自主性，克制控制欲。当孩子能持续行动几十天，他对事情的感觉就会稳固下来，真正的内在动力便彻底形成了。
