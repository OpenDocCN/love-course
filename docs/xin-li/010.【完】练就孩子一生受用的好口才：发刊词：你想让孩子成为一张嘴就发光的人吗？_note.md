# 010.【完】练就孩子一生受用的好口才：发刊词：你想让孩子成为一张嘴就发光的人吗？_note

## 概述

在本节课中，我们将探讨儿童口才与口语表达能力的重要性，并了解一套系统训练课程的设计理念与核心内容。我们将理解为何要从小培养孩子的表达力，以及如何通过科学、有趣的训练方法实现这一目标。

## 口才为何如此重要？

口才与口语表达能力对孩子至关重要。它是孩子自信的重要来源，能为孩子赢得更多人生机会。

良好的沟通与表达能力甚至与学校教育的成绩挂钩。优秀的表达力能为学业成绩保驾护航。

### 自信的力量

我们可以设想两个场景。第一个场景中，一个孩子能在人前落落大方、流畅地表达自己的想法，把话说得漂亮完整。大家会觉得这孩子真不错，孩子自己也会感觉良好，觉得自己很有力量。这种力量就是自信，它会支撑孩子去做更多探索，人生机会也将接踵而至。

### 表达的困境

换一个场景，另一个孩子一跟人说话就紧张。他处于“茶壶里煮饺子”的状态，心里有数却倒不出来，只能干着急。或者他跟人说话时不敢看对方，只敢盯着脚尖或地面。别人会觉得这孩子有点怯场，不太自信。孩子自己的感觉也不好。如果这种感觉持续下去，他会越来越不敢在人前说话和表达，更不用说登台演讲或参加竞选了。这样一来，他的人生机会就会越来越少。成年之后，这种感觉会愈发明显。

### 与学业成绩的关联

口才为何与学校成绩挂钩？举一个真实例子。一位朋友的孩子上二年级，有一天拿回一张有很多红叉叉、考得不好的卷子。妈妈问他：“不会的题为什么不去问老师或同学呢？”孩子憋了半天，说出两个字：“不敢。”即使老师有点严厉，不敢问老师，那同学有什么可怕的呢？主要原因在于孩子对语言这件事不够熟悉，驾驭能力不足，并且心里存在真实的恐惧，害怕被拒绝。这些都源于平常缺乏足够的语言训练。

语言训练非常重要，不仅孩子需要，大人也同样需要。例如股神巴菲特，或者电影《国王的演讲》中的英国国王，即使年岁已高，也需要进行语言训练，因为这确实至关重要。

## 讲师背景介绍

上一节我们探讨了口才的重要性，本节中我们来了解为何由我来讲授这套课程。

我大学就读于中国传媒大学播音主持学院，学习播音主持专业。简而言之，四年都在学习如何说话。毕业后，我直接进入中央电视台，成为一名一线主持人，至今已有22年。这22年我依然在琢磨如何说话，如何面对公众表达。

这四年加22年，一共26年的时间，我确实在这个领域积累了许多心得。除了学校课本教授的说话之道，再加上一线主持的实践经验，我觉得有必要分享出来，帮助孩子们在最早接触语言的这几年，能够接触到正确的方法、技巧和观念。这一点非常关键。

因为我看到很多孩子似乎接受了错误的信息。有时在表达时用力过猛，或处于紧绷状态等。这些误解如果根植在孩子脑中，就会越练越偏。对此我感到着急和心疼，因此我希望凭借我的经验，让孩子们接收到正确的信息。

## 课程核心内容结构

基于以上理念，我设计了这套课程。以下是课程包含的五个主要部分：

1.  **朗读**
2.  **正音**
3.  **气息**
4.  **诗词**
5.  **成语故事及动物故事的儿童剧演绎**

### 课程设计逻辑

如此结构课程是有深层考虑的。朗读被放在最前面，因为它非常综合。我会一点一点地告诉孩子们，学会这些要点不仅可用于朗读，也可用于日常与人交流。这样说话会特别有感染力，做到“一张嘴就发光”。

生活中，有些人一说话就让人觉得无趣、干瘪、无聊；而有的人一开口，你就特别想听，非常吸引人。差别就在于是否掌握了一个个关键点，例如情境感、画面感、基调等。我们通过朗读来激发孩子的兴趣。学习任何东西都应在兴趣引领下进行。

兴趣激发起来后，我们开始进行正音。正音是为了去除语言中的方言成分，这属于比较专业的内容。但我会采用有趣的方式、对比的方式，让孩子们明白如何区分，例如：
*   `h` 和 `f` 怎么分？
*   `n` 和 `l` 怎么分？
*   前后鼻音、平翘舌到底如何区分？

### 气息的关键作用

接下来进入气息训练。很多人不知道说话与气息之间有何关系。关系其实非常大。不会运用气息，声音就不会好听。例如，如果一直用下沉的气息说话（演示：“我一直都这样说话”），嗓子会特别累，长期可能导致声带问题。气息必须是上下贯通的（演示正确方式），这样说话声音才优美、立体、圆润。

### 文学素养与舞台表现

至于后面的诗词和成语故事，我认为这不仅仅是提高口语表达能力，也是提升文学素养，并激发孩子登台表演的欲望。我们的目标是：提高口语表达能力后，孩子愿意到人前进行公众表达。登台是非常好的一种训练方式。激发孩子想登台的欲望，这一点至关重要。

并非每个孩子未来都要成为播音员或主持人，但语言训练一定要趁早。因为语言能力将伴随我们一生。拥有优秀的口语表达能力，是为我们的人生马拉松加分的一项技能。

## 总结

在本节课中，我们一起学习了儿童口才培养的核心价值与系统方法。

最后我想强调，在孩子学习的黄金期，一定要给予他“童子功”般的训练。语言表达能力若是童子功，将终生不忘。长大后再训练行不行？当然行，但会更慢、更难。为何不在孩子小时候就为他提供这样一套课程，让他在优质的语言环境中浸泡，对语言熟悉起来、感兴趣起来、并愿意使用起来呢？这再好不过了。

希望在我的口语表达课中，我们能再次相见。
