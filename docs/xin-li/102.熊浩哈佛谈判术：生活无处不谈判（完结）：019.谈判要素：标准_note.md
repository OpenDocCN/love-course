# 102.熊浩哈佛谈判术：生活无处不谈判（完结）：019.谈判要素：标准_note

在本节课中，我们将要学习谈判的第三个核心要素：标准。我们将探讨什么是标准，为什么它在谈判中至关重要，以及如何寻找和运用客观标准来支持你的立场。

---

谈判可以改变世界，也可以改变生活。在之前的课程中，我们已经介绍了利益要素和选项要素。在谈判的七个要素中，我们已经掌握了其中两个。

本节中，我们来看看第三个要素：标准要素。什么是标准？标准是指分割利益的依据。简单来说，当我们处理冲突时，我们已经挖掘了足够多的利益，发现了共同利益、冲突利益和不同利益。然后我们用选项的方式处理不同的利益，并进行了集思广益、将创意过程和决策过程分开的对话。

但我们最终还是会回到冲突利益的分割问题上。例如，月薪到底是7000还是10000的问题，还是会回到关键的薪酬谈判。这时，我们会建议大家好好准备标准。什么是标准？再说一次，就是利益分割的依据。

我们从头开始。我们先介绍一个知识：人为什么会受到影响。如果把一个人受到说服、受到影响的原因做一个简单概括，人会受三种因素的影响。我们把它叫做三种影响力。

以下是三种影响力的具体说明：

第一种影响力叫做积极影响力。这听起来很悬，其实说白了一点都不难。它的意思是：跟我混有肉吃。即，做这个选择会产生利好。

第二种影响力叫消极影响力。它的意思是：过了这个村可就没这个店了。即，今天不上船，再也没机会上船。不跟我混，你没得好。

这两种其实都是我们在衡量过程中的利弊比较。在哲学上，这种观念属于功利主义。举一个很简单的例子，比方说要说服一个人出国。用正向影响力可以说：“出国会给你带来阅历的增长、知识的进步、语言的锤炼，让你真正变得独立。”用负向影响力则可以说：“你们班那么多同学都出国了，他们在国外受到了锻炼。你要是不出国，你就比别人差一截，可能到最后找不到工作。”

在绝大部分人的心目中，这个世界是由正向影响力（利好）和负向影响力（亏空）构成的。但事实上，还有第三种影响力。

这第三种我们把它叫规范影响力。规范影响力是什么意思？是这么做，我不跟你讨论好或不好，不跟你讨论利或弊，我跟你讨论的是：这是对的，本来就该这样做。

之前看过一个法律电影《十二公民》。决定一个人该不该枪毙，这事不该拍脑袋决定。它是一个典型的法律电影，情节描述一个法学院搞模拟法庭，最后把家长请来组成陪审团。这些家长来自不同领域、职业和社会阶层，要讨论刑事案件中的被告人是否故意杀人。

整个故事的戏剧性在于，有一个人最开始认为孩子没罪，而其他人都认为他是凶手。如何一步一步说服其他十一个人？故事中很多细节忘了，但有一句台词记忆犹新。其中一位陪审团成员一开始认为小孩有罪，听过大家发言和一番唇枪舌剑的说服后，他改变了立场。他说：“向真理低头是一件幸福的事情。”这就叫规范影响力。这样做是对的。

始终记得《哈利波特》中，邓布利多对哈利说过一段话：“黑暗在我们眼前，会有这么一个时刻，我们必须要做出选择。是选那条简单的路呢？还是选那条正确的路？”正确的路，就叫规范影响力。这本来就是你该做的，这本来就是我们应该做出的决定。并不是我想占你便宜，并不是我想在这个过程中跟你讨价还价，而是这就是市场价格，这就是客观标准，我们就应该这样做。

为了帮助大家深刻理解客观标准，我想读一段中国政法大学何兵教授曾在《南方周末》上发表的文章。文章提到，我国收容遣送制度因孙志刚案而废除。在呼吁取消该制度的声音中，有一些管理阶层的人发出质疑：废止这一制度，城市治安如何维护？他们的忧虑似乎是真实的、有根据的，因为数据显示，“三无人员”的犯罪比例可能远高于普通市民。

但这样现实的担忧，真的能成为维持这一制度的全部理由吗？200多年前，一位黑奴从非洲被带到伦敦，伺候主人近两年后潜逃。主人抓获他并带上铁铐，案件交给了英国法治史上的里程碑式人物——曼斯菲尔德法官。全国都在关注此案，因为当时英国约有15000名奴隶，每个价值约50英镑。如果奴隶都获得自由，奴隶主将损失75万英镑，这在当时是巨大数字，而法律并未禁止奴隶买卖。

曼斯菲尔德法官这样判道：“奴隶制度的状况是如此丑恶，以至除了明确的法律以外，不能容忍任何东西支持它。因此，不管这个判决造成何种不便，我们都不能说这种状况在英格兰的法律是应该被允许和肯定的。我们必须释放这个黑人，每个来到英格兰的人都有权得到我们法律的保护，不管他在之前受过何种压迫，不管他的皮肤是何种颜色。英格兰自由的空气不能让奴隶制度玷污。”15000名奴隶成了自由人。

秉承自由和法治的传统，英国当代最著名法官丹宁勋爵在其法官生涯中一再阐明这样的立场：“宪法不允许以国家利益影响我们的判决，上帝不让我们这样做。我们绝不考虑政治后果，无论他们可能有多么可怕。如果某种后果是叛乱，那我们不得不说：实现正义，即使天塌下来。”

“实现正义，即使天塌下来。”这就是非常典型的规范影响力。这件事本该这样做，而不是在利益的天平中，在利弊的取舍中，量了再量，算了再算。

人会受三种影响力的影响，分别是正向的积极影响力、负向的消极影响力，以及规范影响力（这件事的该当性、正义性、合法性本身）。这让我们想起康德哲学。除了功利主义以外，康德哲学就是在论证人有一种绝对的权力、一种绝对的意志，能够让一些价值观不因现实利害而动摇。

所以，什么是客观标准？**客观标准就是支持我们立场的、那些不由主观意志加以动摇的核心依据**。在谈判过程中，我们必须在进入谈判场之前，就为自己的定价、要求、诉求寻找到客观标准。

常见的客观标准包括法律、政策、规范、先例、惯例、习俗等。既然是客观标准，就不是由当事人一方的利益而发生转移的。有一次与上海一家著名律师事务所的合伙人聊天时，他说：“不管对方有多难缠，权势有多大，只要在谈判中告诉对方，我们这样做的依据是法律，一般对方都会乖乖就范。”这是客观标准，这不是我与你为敌，而是事实就是如此。

我们选择客观标准作为我们议价的重要准备和依据，会产生一个非常好的后果。除了让你在说服对方时更有力量、更有自信以外，它还可以吸收对方的不满。

大家想象一下，同样是关于酬劳的议价（这是很多小伙伴最关心的问题）。第一种可能：老板问：“小王，你觉得一个月应该多少钱？”你说：“一万。”老板说：“七千。”然后你们进入讨价还价的拉扯。

第二种可能：你说“一万”，并且能够回答老板在“听力三角形”中的关键问题“为什么（Why）”。你说：“老板，我做过非常仔细的调查，包括对内和对外。对内，是了解公司内与我类似资历的同事薪酬大概是多少；对外，是了解同行业、规模与影响力相当的企业，其入职员工的薪酬水平。两个部分综合考量，我认为一万是一个合理且公正的议价。”这样的话，就会非常具有说服力。你开出的价格并非信口开河，而是有所谓的客观标准。

跟大家报告一件事情：绝大多数人在谈判过程中是没有客观标准的。他们根本没想过为什么要这样定价，以至于在“听力三角形”中，如果对方持续运转提问，问到那个关键问题“为什么（Why）”时，他们将无言以对，突然非常慌张，因为从未为自己的核心诉求准备客观标准。

好的，今天我们讨论的是准备好利益分割的依据。我们说到，人被说服有三种可能：正向影响、负向影响、规范影响。而为我们的利益分割找到客观标准这种依据，其实就是为我们的利益分割找到最夯实、不受主观意志动摇的那个固有依据。

这种依据就是客观标准，它可能来自于先例、法律，也可能来自于之前的合作或相关惯例。在谈判过程中，多数人对他们的核心诉求不寻找客观标准。而如果你一旦寻找客观标准，就会让你的诉求更有效地站稳脚跟。

**客观标准是需要准备的，无法临场发挥**。所以我们通常会说，当指导一些著名企业做谈判准备时，标准的准备、素材的累积、多做一点研究，都是在为自己的诉求找到不容动摇的客观标准。

这就是今天我们想跟大家所讨论的标准的问题：为分割利益寻找到客观依据。

本节课中，我们一起学习了谈判的“标准”要素。我们了解了人受说服的三种影响力，并重点探讨了“规范影响力”和“客观标准”的核心作用。记住，在谈判前为自己的核心诉求寻找坚实、客观的依据，是让你立场稳固、说服力倍增的关键。我们下次再见。
