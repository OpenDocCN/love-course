# 118.【完】超级识人术：08、音乐——听出他内心的声音_note

在本节课中，我们将要学习如何通过一个人的音乐偏好来洞察其性格。心理学家研究发现，音乐品味与人格特质之间存在清晰的联系。我们将基于“大五人格”模型，探讨不同音乐喜好所对应的性格特征。

## 概述：音乐与性格的关联

心理学家进行了一项关于音乐与性格的研究。音乐喜好能够准确地传达一个人各方面的性格。这项研究基于大五人格特质：开放性、外向性、宜人性、责任性和情绪稳定性。其中，与音乐品味关联最紧密的是基于精力的开放性，其次是外向性和情绪稳定性。德克萨斯大学的心理学家山姆·格斯林指出，品味能揭示一个人的价值观和需求，这对生活非常重要。

当我们渴望在别人身上寻找相似品味时，需要记住一点：不同的人喜欢不同的东西，品味千差万别，我们需要学会尊重。

## 不同性格类型的音乐品味

以下是基于大五人格特质，不同性格类型的人在音乐品味上的具体表现。

### 1. 开放型：高品味的猎手

开放型的性格是高品味的猎手。他们从小时候起就开始迷恋音乐，喜欢收集唱片并自学乐器。例如，有人在十六七岁时就成为了一支乐队的主唱。

这类人具有品味猎手的所有特点：**高度开放、富有创造性和好奇心、充满想象力**。山姆·格斯林发现，开放度高的人拥有更多的书籍、CD和电影碟片。开放度越高，对品味的追求就越有冒险性。而开放度相对较低的人，对电影、音乐的选择也会相对保守。

### 2. 外向型：烹饪心动的感觉

一个生长在墨守成规小城市的姑娘，从小幻想着去大城市寻求新鲜生活。她的偶像是歌坛天后麦当娜，因为麦当娜展示了一个女人的激情和戏剧性的情感。

如今，这个姑娘喜欢在不同的艺术品中寻找感官的新鲜愉悦。这一类外向型的人对品味的追求是**生动、活跃、高调明快**。外向型的人喜欢动作片或冒险片，但也容易缠绕于“感觉”。而内向型的人对艺术和音乐通常持有批判性思维，对他们来说，形式比感觉更重要。对于音乐，他们更喜欢像巴赫《平均律》那样精心构思、有严格秩序的音乐。

### 3. 神经质型：用品味自我救赎

一个小女孩在五岁时父母便离婚了。单亲的家庭环境让她成为了一个焦虑不安、很情绪化的孩子。而她喜欢的音乐也是同样的调调，例如Portishead和Elliot Smith。

当她陶醉地听音乐时，她想象自己站在舞台上动情地演唱歌曲。她说：“听一个乐手的歌，我能感觉到她经历过些什么。” 小女孩这一行人又被称为神经质型，**情绪不太稳定，时常会焦虑、敏感、容易沮丧**。这类型的人用极富精神意味的品味装饰自己的生活。

### 4. 责任型：理性动物

Lisa是家中的长女，从小习惯挑大梁的她，做事情喜欢目标明确。她喜欢写实形的艺术品，而艺术品存在的价值就是装点屋子，让房间看起来有美感。

Lisa是典型的责任型人士，**可靠、专注、以问题为目标，喜欢规则和秩序**。这一类型的人，艺术品味不被情绪所控制，通常会喜欢传统艺术。听音乐也会带着一份冷静的态度，保持着审美的距离。

## 具体音乐类型与性格分析

对音乐的喜好是一种性格的简略表达方式。人们可以利用这一标准来迅速地判断一个陌生人的价值观、社会地位甚至种族。音乐伴随着人类的发展和成长，贯穿整个人类历史。从原始社会简单的号子，发展到如今全球多姿多彩的音乐，音乐无处不在。

以下是不同音乐类型爱好者常见的性格特征：

*   **喜欢凄美歌曲的人**：比较多愁善感，心地善良，体恤他人。歌曲如他们生命历程中的灯塔，指引他们前进的方向。
*   **喜欢交响乐的人**：信心十足，踌躇满志，凡事只想到积极的一面，所以能够迅速跟他人打成一片。但容易对别人盲目相信，导致吃亏和受损失。喜欢显露自我，处处显示自己的不平凡，希望上流社会能有自己的一席之地，有不务实的倾向。
*   **喜欢摇滚乐的人**：害怕孤独，不能忍受寂寞，喜动不喜静，爱好运动，愤世嫉俗，对社会有不满的情绪，常常把持不住自己。喜欢到处张扬，能够引人注目，但不会给人留下深刻的印象。能够将爱好作为强有力的指导，借用摇滚巨星的光环，使自己在世俗当中趋于平静，找到心灵上的慰藉。
*   **喜欢流行音乐的人**：属于平凡的随波逐流类型。在恋爱和人际交往的过程中，远离复杂的思维。深层次的自省和强力的感情是他们最不能忍受的。他们试图通过听音乐保持轻松和自在。
*   **喜欢古典音乐的人**：有较强的理性，比较自省，能够用理智来约束情感，从音乐中汲取相当多的人生感悟。但结果常常是曲高和寡，因为很少有人能与他们的思想和感情产生共鸣。
*   **喜欢爵士乐的人**：性格中感性成分占比较大，很多事情凭一时头脑发热去设计，往往脱离实际。不喜欢受到约束，我行我素，总是有一些荒唐的幻想，追求新奇，讨厌一成不变。
*   **喜欢乡村音乐的人**：成熟老练，细心而又敏感，喜欢关注社会问题。他们追求安静和自然，不喜欢大城市的纷繁与喧闹，想过一种完全由大自然控制的田园生活，并为此不遗余力。
*   **喜欢打击乐的人**：耿直爽快，充满了希望，并精心设计自己的未来。为人处事一般以和为贵，不挑剔，同时也喜欢谈笑风生，具有很强的社交能力，能够得到大多数人的喜欢。

## 总结

本节课中，我们一起学习了如何通过音乐偏好识别人心。重点在于：**音乐的喜好能够准确传达一个人的性格**。我们可以通过对一个人的音乐喜好，洞悉其在开放性、外向性、宜人性、责任性、情绪稳定性这五大方面的特质。记住这一点，你就能在社交中多一个了解他人的有效视角。
