# 情感心理咨询技能大全（完结）第12章：关系丧失的咨询—05 好奇和提问 🧐

在本节课中，我们将要学习在咨询中如何保持好奇并进行有效提问。核心在于跟随来访者，而非预设问题，通过提问帮助对方探索和表达其内心世界。

上一节我们探讨了咨询中的基本态度，本节中我们来看看如何通过具体的方法实践“好奇”。

## 如何做到真正的好奇

关键在于放弃对“好问题”的执念。问题的好坏由来访者定义，我们的目标是帮助对方讲述他最想讲的内容。咨询不同于访谈，我们需紧密跟随来访者，而非按自己的清单提问。

以下是实现真正好奇的几个核心要点：

**第一点：不假设自己理解对方的词汇**
不要认为自己已经知道对方所用词语的含义。例如，当来访者提到“悲伤”时，应通过提问进行澄清。
*   **提问示例**：
    *   你提到的“悲伤”具体指什么？
    *   如果悲伤会说话，它会说些什么？
    *   悲伤住在你身体的哪个部位？那里有什么感受？
    *   假如悲伤是一栋房子，走进去你会看到什么？
    *   假如悲伤有颜色，它是什么颜色的？为什么？
通过不断询问和澄清，我们能更深入地理解对方的真实体验。这个过程对来访者而言，往往也是第一次以全新的方式聆听自己。

**第二点：每次都像第一次见面**
无论与来访者见过多少次，或对方重复谈论某个主题多少次，都应保持初次见面的新鲜感。
*   **实践方法**：当来访者再次提到“悲伤”时，可以询问：“这次你说的悲伤，和上次说的有什么不同？”或“这次悲伤来拜访你时，你是如何回应/与它相处的？这些回应与以往有何不同？”
作为过程和关系专家，我们需要从每次讲述的细微差别中，发现过程的变化，而非自以为已经了解。这要求我们始终保持 **`not-knowing`（不知道）** 的立场。

**第三点：将故事视为文学名著**
将来访者的叙述当作引人入胜的名著来聆听，对下一个章节充满期待。这种态度能帮助我们真正在对话中保持好奇。

**第四点：慢下来，后退一步**
这是咨询中非常关键的一步。我们不应急于跳入问题并寻求解决方案，而是先退一步，观察问题本身是如何被建构的。
*   **核心思想**：如同维特根斯坦所言，哲学是为困在玻璃瓶里的苍蝇指明出路。咨询有时也是如此，我们需要帮助来访者看到“他是如何进入这个困境的”。
*   **举例**：当来访者认为“单身是个问题”时，我们不是急着帮他摆脱单身，而是问：“‘单身是个问题’这个想法是怎么来的？”
*   **禅宗公案类比**：看到一个和尚用嘴衔着树枝挂在树上，关键问题不是“他如何下来”，而是“他当初是怎么上去的？”既然他能上去，就可能找到下来的方法。后退一步，是为了看清来时的路。

同时，**慢下来**至关重要。在日常快节奏中，我们常错过细节。而在咨询对话中，正是那些被细致探讨的、曾被忽略的“风景”，可能为走出困境提供新的思路和可能性。

---

本节课中我们一起学习了在咨询中保持好奇与提问的艺术。核心在于**放弃预设、跟随来访者、澄清词汇、保持新鲜感、像读故事一样聆听，并学会慢下来和后退一步**，从而帮助对方探索其独特的体验与困境的根源。
