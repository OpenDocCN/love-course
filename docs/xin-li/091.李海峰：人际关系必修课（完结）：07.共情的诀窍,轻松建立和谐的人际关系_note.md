# 091.李海峰：人际关系必修课（完结）：07.共情的诀窍,轻松建立和谐的人际关系_note

在本节课中，我们将学习如何运用共情的技巧，轻松建立并维护和谐的亲密关系。我们将结合DISC理论，探讨两个核心原则，并学习具体的沟通方法。

---

希格蒙德·弗洛伊德曾对学生说，爱与工作是标志着人全面成熟的双重能力。前面我们介绍了如何把DISC运用在工作中，本节中我们来看看如何把DISC运用在亲密关系中。

韦伯斯特新大学词典将“亲密”定义为一种密切结合、接触、联盟或熟识的状态。这种亲密感可能出现在各种不同的关系中。据调查，选择爱情占47%，选择友谊占36%，选择家庭占14%。无论在哪个年龄阶段或哪种文化中，亲密关系都可能是个人生活满意度和情绪幸福感最重要的来源。

人生有几段这样的关系，就永远不会再谷底。除了性别以外，是什么影响了亲密关系的和谐呢？能否求同存异、达成一致意见，实际上是亲密关系持续的关键。

那要怎么做呢？本节课，我们将结合DISC理论，分享两个很重要的原则：第一，懂比爱更重要；第二，不要老是想遇到Mr. Right，你要有能力把遇到的那个人变成Right。

## 一、懂比爱更重要：了解你生命中的重要他人

本集我们谈的亲密关系，是广义的和自己生命中重要的人的关系，包括亲戚、朋友，甚至工作中的伙伴。我们没有办法照顾好世界上所有的人，但是我们生命中重要的人，我们必须要有能力照顾好。

我们一直强调，要有能力让亲戚成为有血缘关系的朋友，让朋友成为没有血缘关系的亲人。照顾好别人的前提在于了解对方，而了解一个人需要大量的时间跟精力。怎么能判断我们已经了解了对方呢？实际上就需要我们有一个能力，就是给对方传递：“你在我眼里跟世界上其他人都不一样。”

我自己有一对龙凤胎。在他们两周岁的时候，我给我儿子希希和女儿俊俊各自写了十个特质，而这十个特点是不一样的。同样，我还会把这些方法带到公司的管理中。我会要求管理者描述出每位直接下属的不同之处。如果他能做得到，并且可以传递给对方，那么他在对方心目中就是一个懂他的人。因为每个人可以得到的最好的赞赏是：“你与众不同。”

各位还记得我们在第一集的作业里面，有邀请大家给自己列一个生命中最重要的人的清单吗？看到自己的伙伴，建议你把那张清单拿出来。如果之前你没有做，那么建议你现在做。这个清单的人数不要太多，12个人以内比较合适，否则可能你根本照顾不过来。

以下是你可以做的步骤：
1.  给清单里面每个人用十个词来描述他。
2.  如果写不出十个，那么给自己一个期限，比如一个月之内写完，三天写一个应该没有问题。问题在于你平时根本没有有意识地去花这么多时间，去了解跟观察对方。

当然还有一个相对简单的方法，就是下载我们辅助资料里面的“40个特质图”，选择其中的十个进行打勾，然后把选中的十个进行排序。完成之后，实际上在你眼里，他的DISC特质就很明显了。你要有意识地用对方需要的方式照顾好他。

懂他才能更好地爱他，所以我们甚至会说，**懂比爱更重要**。你能够懂对方，沟通就能从单向的说服走向双向的交流。好的关系不是依赖，不是比对方强，而是互赖，彼此生长。

## 二、从“改变”到“影响”：让对的人出现

我们总是想遇到那个懂你爱你的人相伴一生。刚开始的时候情意绵绵，相处时间一长，各种嫌隙，以为自己遇人不淑。我们常常想去改变对方，却不知道改变太难。

我们先看一下下面这个故事。古代有个国王走在路上被石头割伤了脚，下令传国把牛全部都杀了，牛皮铺到路上石头就不割伤脚。但是有两个挑战：第一，牛皮放在路上风吹日晒会坏；第二，牛根本没那么多。所以有个聪明的臣子就给了国王一个建议：把牛皮切成一小块一小块裹在脚上。于是我们大家就有皮鞋可以穿了。

**改变世界太难，我们要做的永远是调整自己。** 你的太太、你的先生、你的朋友、你的客户、你的上级，和你往往是两个完全不同世界的人。永远不要想去改变对方，你要做的是用对方接受的方式去影响他。

成人或管理者很喜欢去改变别人，对方不改变你就不开心。实际上，你幸福快乐的钥匙就掌握在对方手里了。如果你不想改变对方，只是用对方接受的方式去影响他，那么结果往往就不一样了。

以下是处理这种关系的思路：
1.  你想想，在你的影响之下对方改变了，你自然开心。
2.  如果在你的影响之下对方没有改变，你要问自己：这段关系对我来讲还重要吗？
3.  如果重要，就继续用对方接受的方式去影响他，谁叫他对你重要呢。
4.  如果这关系对你已经不重要了，那你也不要老想去改变对方，你要做的是换一个人来影响。

所以在这种情况下，你不想去改变对方，而只是想影响对方，这个时候幸福快乐的钥匙就在你自己手上。

没结婚之前，我们脑海里面常常会设想一个完美的人，并期待对方用你喜欢的方式和和美美过一生。结婚以后才发现，他可能并不像你想象中那么完美。在这个世界上，即使最幸福的婚姻，一生中据说也会有200次以上的离婚的念头和50次想掐死对方的想法。

我自己太太的C特质里面是带点D的。她的口头禅经常是：“李海峰啊，你怎么这么蠢啊。”请教一下各位，如果你太太骂你蠢，你会不会要去跟她计较？我想你的答案跟我一样，当然不要了。你要知道家不是讲道理的地方，你知道怎么应对就行了。

对我来说，基本上三招搞定她：装傻、装可怜、装可爱。
*   **装傻**：你说我蠢，我哪里蠢？我蠢到不知道哪里蠢，哎，懒得理你了。于是我就一个人看电视很开心。
*   **装可怜**：老婆你看我这一周忙里忙外，七天飞到五个城市讲了六天课，我已经好累了，你就不要再骂我了。我太太经常就会说：“李海峰啊，你最会用这招了。”那各位就代表这招有用了。
*   **装可爱**：这个最简单的，比个剪刀手说：“对啊，蠢才会娶你啊。”

各位你要知道，一开始两个人在一起是因为彼此有差异，很可能会有一个忍受的阶段。但是随着能够理解彼此，就可以从忍受到接受。接受后，随着对对方的了解，有能力去影响对方，最后就能够从接受变到享受。**从忍受到接受到享受，靠的是理解对方和了解对方的能力。**

不要老想遇到Mr. Right，你要有能力让遇到的那个人变成Right。

## 三、运用S与I特质：处理关系中的情绪与平淡

在亲密关系中，我们要更多地调动自己的I特质和S特质，更多地关注人，而不是关注在事情上。在有情绪的时候，讲道理是没有用的，所以我们常常说：**先处理心情，再处理事情**。我们要更多地用S来照顾对方的感受，用I来调解氛围，让幸福更加长久。

婚姻出现危机的一个初期预警的信号，是尖锐的批评、过度的抱怨或人身攻击。抱怨是对事，而人身攻击则是对人了。当夫妻双方相互抱怨的时候，应该多使用S的特质，用同理心去倾听对方的想法，并理解他的情绪，才能更好地维系夫妻之间的感情。

这里我们给大家介绍一种“红绿灯法”。

**红灯**自然代表着停止，保持冷静。当我们看到对方做了一些事情，让自己不舒服，心里有情绪的时候，先停6秒，深呼吸，保持冷静。因为这个时候说出的话，很容易激怒对方不好的情绪，从而影响情感，让事情变得更糟糕。

红灯完了之后是**黄灯**。你要描述一下情境，表达感受，并寻求解决方案。
*   **情境**是当下正在发生的事情，只做描述，不要做评论，尽量保持客观。描述情境是事实，一般为动作词，不带感情色彩；评论多为形容词，一般带有强烈的个人色彩。如果我们不是在描述事实，而是在评论，通常对方会倾向于听到批评并且反驳我们，于是就变成情绪跟情绪之间的抗争，我们永远没有办法冷静下来回归到事实。
*   为什么要客观地描述情境呢？因为人在抱怨的时候，很多时候会陷入到自我保护的情绪漩涡中，屏蔽并且排斥来自别人的观点和看法。而客观描述情境，就是让彼此的注意力更多地聚焦在认同的事情上，先求同，避免触发对方情绪按钮，阻碍沟通。这就是黄灯缓冲的作用：先求同，后存异。
*   那怎么来存异呢？就是紧接着**表达感受**，说出自己的感受。可以说：“我感到……，因为我……”。比如：“我看到你打碎了我心爱的玻璃杯，我感到很难过，因为我很喜欢这个杯子，那是我妈妈送我的生日礼物。”为什么要表达感受？因为表达感受是建立在对方对自己的情绪认同，让他知道我很难过，我为什么难过。换作是你，你是不是也会难过？这样其实是引导对方有了同理心，并且反思自己的行为。
*   学过DISC以后，我们都知道凡事必有四种解决方案。所以当遇到问题的时候，我们不需要去纠结于责任或者问题发生的起因，我们聚焦在解决方案。我们永远关注需求，因为一个需求有很多种解决方案。而如果我们执着于所谓“唯一”解决方案，彼此就会有很多的矛盾。

最后是**绿灯**，选择并实施最佳的方案。当我们想清楚了解决方案，并且比对每一个方案可能会带来什么结果的时候，最后我们可以选择最合适的方案加以实施。

当亲密关系中发生争吵的时候，我们不要把关注放到事情，而更多地发挥S的特质，站在对方的角度理解对方的情绪。双方保持冷静，用红绿灯法解决问题，可以让亲密关系更加和谐。当然，罗马不是一天建成的，习惯也不是一夜之间可以养成的，还需要双方一起努力，相互提醒相互练习，坚持才能够真正的习得这个能力。

亲密关系中除了有火药味的时候，双方抱怨争吵外，也难免会遇到麻木期，也就是我们常常说的“七年之痒”。有的时候我们甚至会说，爱的反义词不是恨，而是冷漠，变得平淡没有新鲜感。这个时候适当地发挥I的特质，亲密关系就会变得更加美好。

当两个人相处久了，在一起就像左手牵右手没有激情，生活变得柴米油盐平平淡淡。这个时候我们需要更多地调动I的特质，乐观的特质。而“话多”是I的秘密武器。

这里的话多不是要让你唠唠叨叨地去说废话，而是学会用话语，让对方感受到“你懂我”、“你爱我”、“我关注到你”、“我在你眼里是跟别人不一样的”。

那怎么才可以做到话多呢？给大家推荐I特质的“三件宝”：赞扬、认同和感恩。或许你要说，道理我都懂，可是为什么就是用不好呢？因为很可能您的赞扬比较低质量，认同讲得不完美，感恩并没有由心而发，所以甚至只会说“你真美啊”、“你真棒啊”、“谢谢你啊”。

要想学会高质量的赞扬、认同跟感恩，我向各位分享一个好的“3R幸福法”。我们将一句能够带来幸福感的话，拆分为三个部分。

比如一句简简单单的“你真棒”，变成：
> “亲爱的，因为你做了这么好的菜（**理由**），让我一整天的疲惫都一扫而空（**反应**），你真棒，有你真好（**结果**）。”

其中一共有三个部分，分别是：
1.  **理由 (Reason)**：关注的缘起，不会让你想表达的感情唐突，不会给对方和自己造成不好意思。
2.  **反应 (Respond)**：关注的影响，更多地表达对我产生了什么正向的效果，建立对方对自己感受的关注和成就感。
3.  **结果 (Result)**：关注的结论，提出对方在我心目中的认知或者是地位，让对方感受到你很在乎我。

以下是具体应用的例子：

当你想表达**表扬**对方的时候，应该想到的是：赞扬对方的理由 + 给我带来的益处 + “你今天真的好美啊/好看啊”等。
*   **例子**：想告诉对方“你今天真好看”，可以说成：“亲爱的，你今天出门穿的这件蓝色裙子实在太漂亮了（理由），我今天一整天都不敢让你离开我的视线，多看你一秒我都感觉到很开心（反应），你真美（结果）。”

当你想**感恩**对方的时候，应该说：感谢对方的理由 + 给我带来的结果 + “非常感谢”。
*   **例子**：想告诉对方“谢谢你今天帮我打扫卫生”，你可以说成：“我今天都累坏了，公司开了一整天的会，一到回到家呢，看到家里这么干净（理由），一下子啊什么累什么苦都没有了（反应），我能有你真的是太幸福了，谢谢你（结果）。”

当你想表达**认同**的时候，应该说：认同对方的原因 + 对方优势带来的结果或影响 + “你真棒”。
*   **例子**：想告诉对方“你做得很棒”的时候，可以说：“你在这次马拉松得了第十名，真的是一个非常好的成绩（理由），而且每天坚持跑步10公里，也给儿子树立了榜样，我们都以你为荣（反应），你真棒（结果）。”

各位理解了吗？在维持亲密关系中，赞扬、感恩和认同是一种简单有效的方法。只要我们掌握了正确的步骤，就可以让对方感受到幸福，也会让家庭的生活充满爱。

## 总结

本节课中，我们一起学习了如何运用共情的技巧建立和谐的亲密关系。

我们探讨了两个核心原则：**懂比爱更重要**，以及**不要试图改变对方，而要用对方接受的方式去影响他**。我们学习了具体的方法：运用**S特质**和“红绿灯法”来处理关系中的情绪与冲突；运用**I特质**和“3R幸福法”（理由、反应、结果）来为平淡的关系注入活力，通过高质量的赞扬、认同和感恩让对方感受到被关注和珍视。

记住，在亲密关系中，我们要更多地关注人本身。改变对方太难，我们要做的是调整自己，用对方需要的方式去影响他，让他成为那个“对的人”。

祝大家幸福。
