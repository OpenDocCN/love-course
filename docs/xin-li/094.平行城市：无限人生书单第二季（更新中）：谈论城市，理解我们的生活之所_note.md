# 📚 课程094：平行城市——无限人生书单第二季（更新中）：谈论城市，理解我们的生活之所

在本节课中，我们将一起探讨“城市”这一主题。通过阅读相关书籍，我们将理解城市如何塑造我们的生活，以及我们如何通过不同的视角来认识城市。

城市是一个全球性的存在。全球居住在城市里的人口已超过全世界总人口的一半。发达国家的城市化率很高，且趋势是越来越高、越来越集中。中国近年来的城市化率也超过了一半。这意味着绝大多数中国人已经是城市居民，而不再是以农民为主体的国家。

相信大多数听众都生活在城市里，拥有比较丰富的城市经验。相比乡村，城市一般是我们更熟悉的环境。同时，每个人的城市经验又是相对有限的。虽然现在很多人自诩是世界公民，曾经周游世界，但一个人不太可能在全世界大多数城市里长期居住。曾经到一个城市观光旅游或待上几天，与在这个城市长期生活几个月甚至几年，认知和感受都是明显不同的。

如果你在一个城市只待了一两天，你所居住的地区、接触的人以及许多偶然因素，都会对你的判断产生巨大影响。但如果你在一个城市待得足够久，你的判断又会发生微妙的变化。

上面说的这些其实也不一定。可能很多人都听过这样的话：如果你到了一个城市（可以是开罗、伊斯坦布尔、孟买、耶路撒冷等），待上一周，回去就可以写一篇文章；待上一个月，回去就可以写一本书；可是如果你不幸待上一年，那么可能回去以后什么都写不出来。所以，一个城市对于不同人的意义是完全不同的。

无论是北上广深，或是纽约、伦敦、巴黎，每天都有很多人想尽办法要留在这里，同时又有很多人要想尽办法离开这里。有些人爱它，而另些人恨它。海明威曾经说过：“如果你有幸在年轻的时候在巴黎生活过，那么以后无论你走到哪里，巴黎都会一生追随着你，因为巴黎是一场流动的盛宴。”这句话说得太好了。不仅巴黎如此，其他很多城市也是如此。

我自己住在上海，但也经常会怀念曾经生活过的一些城市，比如墨尔本或新加坡。后悔以前对很多地方探索得远远不够，而今天再要过去就很麻烦了。不过，我们也看到过这样的情况：有一些在城市里居住很久，甚至在一个城市居住大半辈子的人，仍然对自己居住的城市没有兴趣。他们不想学习这个城市的方言，也不想了解这个城市的城市历史，内心永远留在另一个地方。

城市的意义非常复杂。我们应该怎么来讨论城市？我一直觉得城市非常难谈，因为一谈到城市就涉及很多不同的学科背景。每个人认识城市的框架也是多种多样。同时，又因为每个城市的变化速度是不一样的，有些城市你几年不去，那就完全变了样。最后，还因为每个人对于城市的经验都是私人性的，不同人说起同一个城市，体验是完全不一样的。

所以关于城市，我思考了很久很久，然后想起以前读过的一本书，那就是普鲁塔克的《希腊罗马名人传》。这本书曾经对我的思考和写作都产生过重要影响，这里也顺便推荐给大家。

普鲁塔克的写作方法是这样的：写一个希腊人，再写一个罗马人，两个人可以相互印证。希腊人和罗马人的经历当然完全不一样，没有两个人的经历是一样的。但是普鲁塔克会提炼他们的相似之处，比如继往开来的人、制定法律的人、被叛徒杀害的人等等。所以他就把两个人的经历对比着一起来写，这样我们也对比着读，就可以读出可以相互比较、相互印证的东西，非常有意思。

所以这本书就启发了我。于是我也想采用一种平行的读法。每次我们挑选一个主题，读一本外国人写的关于城市的书，再读一本中国人写的关于城市的书，形成对照。我们总共有十集节目，一共就可以讨论五个主题。所以我们这个书单的名称就叫做“平行城市”。

没有哪两个城市是完全可比的，我们也不想真正比较两个城市，只是挑选一些个人比较感兴趣的维度，切入进去和大家一起聊聊。我个人有一种偏见，那就是城市叙事在我们的知识框架当中，很多时候是不足的甚至是缺失的。

就以文学为例，城市文学在很长一段时间里，一直处于边缘位置。大家不妨找来许子东老师讲二十世纪小说的节目听一听。尤其是近半个世纪以来，中国最有名、最有影响的那些小说，大多数都是农村题材。一些朋友曾经开玩笑说，什么上海文学、北京文学，上面刊发的都是农村文学，应该叫“上海农村文学”、“北京农村文学”才对。

正因为如此，当真正出现一些书写城市的作品的时候，比如《繁花》，大家非常惊喜、口口相传。又比如最近反映上海生活的电影《爱情神话》，也引发了热烈的讨论。很多人看到自己熟悉的马路名称、熟悉的街道景色出现在文学电影作品里，都会觉得非常激动。我觉得确实应该激动，但反过来可以想想，在我们平时的文化当中，这些元素是不是出现得太少了呢？

比如香港这个城市，它的很多地名，什么旺角、湾仔、尖沙咀、深水埗等等，最早我都是通过一些港片或者TVB电视剧了解到的。虽然自己没有去过，但好像已经在电视里看了无数遍。以后有机会等到自己亲自去了以后，就觉得非常亲切。那里一定跟电视里所反映的环境有所差异、有一个落差，但是又不是完全不同。于是亲眼所见的事物，就与以前看电视的记忆、听来的八卦传说、各种来源的信息重合在一起。这时候我才会觉得，对这个地方真正有了一点点理解。

这样的工作并不容易，而且不同的城市有完全不同的特征，不可能用统一的方法去理解。

以下是几种不同类型的城市及其理解角度：

*   **自然地理型城市**：有些城市有它独特的自然地理，比如威尼斯，又或者像乌镇，跟水的关系特别密切。
*   **文化遗产型城市**：有些城市本身没有任何特别之处，但是有了不起的文化遗址，比如敦煌。敦煌这个城市本身在我看来是没什么特色的，生活质量也谈不上很高，但是它旁边就是莫高窟，那太了不起了。我知道很多人一去莫高窟以后，就爱上了莫高窟，所以一遍又一遍地去敦煌，甚至一辈子就献给敦煌。这就是这一类城市的巨大魅力。
*   **特殊产业型城市**：还有一些城市，有它的特殊产业，比如景德镇。如果你对瓷器没有特别兴趣的话，那么景德镇可能也是一个没什么意思的地方。但是如果你热爱瓷器的话，那你非去景德镇不可，因为在中国再也找不到一个拥有那么丰富瓷器传统、汇聚那么多瓷器匠人的地方。你在景德镇可以自己创作瓷器，可以找到所有与瓷器相关的工艺，那里就是瓷器的天堂。
*   **学术地标型地点**：还有一些地方、还有一些城市，有些人会关注它只是因为特殊的原因。就比如苏州吴江的开弦弓村，不知道大家听说过没有。那是太湖边的一个小村子，大概也就三千人左右，交通也不是很方便，又没有任何的旅游资源。但是很多我认识的朋友都去过那里。为什么呢？原因也很简单，1936年费孝通就在这里做了两个月的社会调查，然后写出了著名的《江村经济》。费孝通是中国社会学、人类学的泰斗，影响实在太大，所以很多人都想追随费老的脚步，去今天的开弦弓村去看看那里到底是怎么回事。

这样一来我们就会发现，每个城市的特点都是完全不同的，我们去认识这些城市的角度也不一样。比如你是一个热爱美食的人，到一个地方只关心美食，你对于佛教石窟、壁画之类没有什么兴趣，那你就不应该去敦煌，你应该去顺德。敦煌吃的东西实在没有什么特别的。像早年留守敦煌的那批人，他们的生活条件都是非常差的，冰箱都没有，肉都没有办法保存，他们能坚持下来，就是因为石窟、就是因为壁画。

那说到敦煌，不妨说几句我自己的故事。在我读大学的时候，偶然听过一次讲座。当时好像准备去听一个名人讲座，大概是金庸的讲座，听众人山人海，我到了教室以后发现根本挤不进去。这时候看到旁边的教室好像也有一场讲座，而且人不多，于是我就到隔壁去听讲座了。等坐下来，发现原来是常书鸿先生的公子常嘉煌在那里介绍敦煌。当时他放了很多影片，都是老一代的敦煌人怎么守护敦煌的，他自己也坚定表态说要献身敦煌。当时我记得看得我是热泪盈眶。

我就记得在那个视频里面，常书鸿的夫人李承仙就在视频里面说：“我也是学画画的，但是我到了敦煌以后，一下子就走不了路了，看到敦煌壁画上的那种线条、那种力量，一下子就走不了了。”我至今还能记得视频里她的动作、她的表情，还有提到敦煌时眼睛里的光彩。当时我就在想，以后我也一定要去一下敦煌。

那几年之后，我确实去了敦煌。站在莫高窟前，看着就想起陈寅恪先生那句名言：“敦煌者，吾国学术之伤心史也。”我就觉得感慨万千。我觉得自己终于站到了这里，终于可以试着勾连像张大千、谢稚柳、常书鸿、高尔泰、樊锦诗等等好几代学者他们的精神世界。他们就是在这些洞窟里面被感动，我也是一样。

直到很多年后的今天，我现在已经看了不少关于敦煌的书，在英国、美国、印度也看过很多敦煌的遗物。看到那些敦煌东西的时候，就会觉得很激动。但是直到今天，还是觉得自己没有能力讲述敦煌，因为怎么讲好像都不对。但是这座城市对我的影响是非常深的。

类似这样的对我意义重大的城市还有很多。我可能还是少见多怪，因为对我触动的那些点，对于比如像敦煌当地人而言没什么稀奇的，我的这些感受可能还是挺肤浅的。

所以，我们这档节目最终还是要回到书本、回到文本，通过阅读来探索城市。大家听了节目以后，可以与自己的经验相互印证，有机会就再多读几本书、多走一些地方，那就再好不过。

我们每一期节目会有一本核心的书。这本核心的书都是曾经对我产生过重要影响的、有关城市的书，希望大家也能喜欢它。我不想在节目里概括每本书的核心思想，也不打算告诉你这本书的主要内容。事实上我也没有这个能力，不具备什么“在20分钟帮你读完某某书”这样的一种能力。我们都知道，写一本书写得快的话要写几个月，写得慢的话要写好几年，整整几百页的内容，我们怎么可能在20分钟内就把它讲完呢？

我能够分享的，那就是每一本书触动我的地方，那些让我忍不住想要继续探索、继续阅读，最终还要和大家一起来分享的那部分内容。当然了，触动我的地方未必是作者自己最看重的地方，作者自己最为得意的地方也很可能被我轻易地放过。这都很正常，因为每个人都有自己关心的问题，也有自己思考的方式。我就是讲一下自己被这本书触动的点。

在讨论的过程当中，不可避免可能还要涉及一些其他的书。如果我觉的有必要的话，会在文稿中加以注明。是不是还要在每一期节目后面附上一个更长的参考文献呢？我想来想去还是不要了，大家去读这本核心的书就好。因为每一本书它本身就会告诉你，到哪里去寻找下一本书。

感谢你的收听，我是梁捷。让我们下期节目再见。

**本节课总结**

本节课中，我们一起学习了如何通过“平行阅读”的方法来探讨和理解城市。我们认识到城市意义的复杂性，它因个人经验、学科背景和城市自身特点而异。通过将中外关于城市的书籍进行对照阅读，我们可以从更多元、更丰富的维度切入，深化对“我们生活之所”的理解。本课程旨在引导大家从书本出发，结合自身经验，开启一场探索城市的阅读之旅。
