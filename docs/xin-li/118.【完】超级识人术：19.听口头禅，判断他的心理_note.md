# 118.【完】超级识人术：19.听口头禅，判断他的心理_note

在本节课中，我们将要学习如何通过分析一个人的口头禅来洞察其内在心理和性格特征。口头禅是人们在日常交流中无意识重复的短语，它们像心灵的密码，能够揭示说话者的情绪、心态和思维模式。

语言常被称为人类的“第二表情”。一个人语库中使用频率和重复率较高的口头禅，具有心理投射的功能，能在一定程度上揭示其内心世界。几乎每个人都有自己独特的口头禅，就像每个人都有习惯性动作一样。

这种习惯性的语言，说话者自身往往难以察觉，但旁人稍加留意便能注意到。作为一种下意识的表现，口头禅可以帮助我们认识一个人，因为它反映了人们当下的情绪和心态，同时也间接折射出其性格。

上一节我们介绍了口头禅作为心理窗口的概念，本节中我们来看看口头禅是如何形成的。口头禅的形成主要有两个原因：一是重大生活事件对人的影响；二是时间积累效应的结果。

例如，一个曾真挚投入恋爱却遭遇背叛的青年，可能在失恋后的一段时间里对爱情嗤之以鼻，随之他的口头禅也许会变成“爱情这东西，纯粹是扯淡”。当一个人多次经历相似的挫败后，这种积累效应就会在其口头禅中表现出来。

口头禅是人们进行心理宣泄的一个通道。积极的口头禅能催人奋进，而有些则带有消极意味。面对生活压力和快速变化的心态，通过口头禅来“吐苦水”，为内心提供一个舒缓宣泄的通道，对心理健康是有益的。

那么，我们该如何对待自己的口头禅呢？首先，不能就口头禅谈口头禅，而应从性格根源上寻找原因。如果能把口头禅所反映的性格问题处理好，消极的口头禅就会减少甚至消失。其次，要有意识地培养反思能力，训练自己敏锐地意识到“我怎么又产生这种心理了”，从而减少消极口头禅的发生。

从某种程度上讲，一个人习惯性反复说的日常语句都算口头禅。粗鲁无教养的人，自然少不了“国骂”之类的口头禅；而温文尔雅的人，“请”、“谢谢”则是常挂嘴边的口头禅。因此，口头禅很容易暴露一个人内心惯有的思维模式。

以下是一些常见的口头禅及其性格解析，让我们看看里面都藏着什么样的秘密：

**第一句口头禅：“问题不大”。**
这句话多出现在亲密关系或职场中。一方面，说话者表明了他的现实判断——问题的确存在；另一方面，也表现出了他一种无所畏惧的强烈自信心。他总在俯视这些问题，这是感染力很强的四个字，能给对方一种安全感。

**第二句口头禅：“都是骗人的”。**
生活中有一类人，在每次接收到新鲜事物时，会质疑其真实性。反向思考，每件事都是骗人的吗？每个人都是狡猾的吗？这句口头禅说明这类人不轻易相信别人，他们的性格多疑而偏执，还具有狡猾性格的某些特点。他认为别人在骗人，其实他自己的世界观就是：世界是一个充满谎言的地方。

**第三句口头禅：“不是……”。**
坚持自己的主见，具有喜欢贬低别人的特点。他其实是在说：“你们说的都不对，你们听我的。”这样性格的人往往善于判断和决策，也擅长领导别人。

**第四句口头禅：“可能是吧”、“或许是吧”、“大概是吧”。**
这类人自我防卫的方式和本能很强，不会将内心的想法完全暴露出来。在处事待人方面冷静，因此在工作和人际关系中容易收获好人缘。这类话也有以退为进的含义，一旦事情明朗，他们可能会说：“我早估计到这一点了。”从事政治的人多有这类口头语，用以隐藏自己的真实想法。

**第五句口头禅：“但是”、“不过”。**
喜欢说这句口头禅的人，为人有些任性，因此总是提出一个“但是”来为自己辩解。“但是”一般是为保护自己而使用的，也反映了温和的特点，说得委婉，没有断然的意味。从事公共关系的人常用这类口头禅，因为其委婉的意味，不至于令人有冷落感。

**第六句口头禅：“啊呀”、“这个这个”、“嗯……”。**
这类人词汇量较少，或是思维较慢，在说话时将其作为间歇的方法，从而形成口头语的习惯。因此，有这种口头语的人反应较迟钝。某些傲慢的公务员也有这类口头语。

**第七句口头禅：“凭什么呀”。**
事情不该是这样，可是却这样发生了。或出于心理失衡，或基于愤世嫉俗，总之就是看不惯那些与自身意愿相悖的事，并以重复出现的这句口头禅来鸣不平，缓解郁闷。这类人的性格正直，但是存在几分神经质，比较苛刻，但不一定是专门针对某个人。他们对公平和特权十分敏感。“凭什么呀”这句话其实是在诉苦，亦或是在控诉，典型的就是“愤青”情结。

**第八句口头禅：“听说”、“据说”、“听人说”。**
之所以用此类口头语，是给自己留有余地的心理形成的。这种人的见识虽广，决断力却不强。处事圆滑的人常用此类语。

**第九句口头禅：“事情不是这样的”。**
有主见、有韧性，同时也很脆弱。很多时候，这个人是刀子嘴豆腐心，未见得有多难相处。只要给予必要的尊重，他会返还给你加倍的热情，而且一条道走到黑，把你当成忠实的合作伙伴或知心朋友。从一定程度上讲，是可以依靠的。

**第十句口头禅：“我晕”。**
听到这句口头禅，你会感到一种莫名其妙的亲切感。“我晕”随处可见：别人帮他递错了东西，“我晕”；听到他不知道的八卦，“我晕”；被人恭维，“我晕”；钥匙忘带，“我晕”。任何时候，只要事情不是预计和想象的那样，就会“晕”。其实问题并没有那么严重，但习惯于在潜意识里夸大后果，并在表露于口头禅的夸张情绪中反映出来。这类人比较活泼坦诚，不隐晦个人情感，但是很容易意气用事。一句“我晕”，将所有的人和事一视同仁、等量齐观，所以他善于从广度上发现问题，但不擅长从深度上思考问题。

本节课中，我们一起学习了如何从日常的口头禅里洞察对方的性格和心理。通过分析这些无意识的语言习惯，我们可以更深入地理解他人的思维模式和情感状态，从而在社交中更加游刃有余。更多精彩有趣的干货内容，我们下期接着讲。
