# 010.【完】练就孩子一生受用：33、气息篇丨第四集：腹式呼吸可以一时用，不能一直用

在本节课中，我们将学习口语表达中另一种需要避免的呼吸方式——腹式呼吸。我们将探讨其特点、在说话时使用的弊端，并理解为何“播音腔”有时反而是呼吸方法不当的表现。

---

上一节我们介绍了**胸式呼吸**及其问题，本节中我们来看看另一种看似“专业”实则不可取的呼吸方式：**腹式呼吸**。

腹式呼吸是指在讲话时，胸部不参与呼吸过程，仅依靠腹部进行呼吸。这种呼吸方式在生活中确实存在，尤其是在不说话或睡眠时。

以下是腹式呼吸的常见表现：
*   呼吸时，腹部明显起伏，而胸腔保持静止。
*   常见于睡眠状态或身体完全放松时。

然而，在口语表达中持续使用腹式呼吸，会产生不良效果。其核心问题在于气息**上下不贯通**，导致声音虽然听起来低沉，却显得做作、不圆润。

例如，当某人说“明月肉”这三个字时，如果仅用腹部气息，声音会显得沉闷且不自然。其根本原因在于发声动力源单一，公式可以概括为：
**声音效果 = 气息来源单一 → 通道阻塞 → 音色不圆润**

因此，我们需要明确：**气为声之帅，气动则声发**。声音的弹性与持久力，与气息的运用方式直接相关。理想的气息状态应是流动且贯通的。

所以，无论是胸式呼吸还是腹式呼吸，都因为气息通道不完整而不可取。正确的方向是调动整个呼吸系统。

---

本节课中我们一起学习了：
1.  **腹式呼吸的定义**：仅靠腹部进行的呼吸方式。
2.  **其适用场景**：适用于不说话或睡眠时，而非持续口语表达。
3.  **在说话时的弊端**：导致气息不贯通、声音做作、音色不圆润。
4.  **核心原则**：应追求气息上下贯通，为学习下一课关键的**胸腹联合式呼吸**打下基础。

下节课我们将进入呼吸方法的重点，学习如何协调胸腹，实现气息的畅通与高效运用。
