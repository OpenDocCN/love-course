## 概述：成年后的情绪变化

上一节我们探讨了情绪管理的基础。本节中，我们来看看当人步入三十岁后，对待烦恼可能出现的麻木感及其应对方法。

和菜头老师指出，年岁增长会让人对生活的荒谬感减弱，大笑减少是常见现象。但这不一定是情商提高的结果，而可能源于对生活常态的更深理解。

---

## 一、关于快乐与大笑的减少 😄

用户田旭问：感觉自己进入社会后，再也没有笑到肚子疼过，这是情商增加的原因吗？如何找回这种快乐？

和菜头老师回复如下：

*   他不记得上次大笑的具体时间，但感觉多发生在30岁之前。
*   这种变化与情商高低关系不大。
*   对他个人而言，大笑往往源于看到现实中极为荒谬的事情。
*   随着年岁增长，他发现许多事并非那么荒谬，而是生活的常态，自己也可能身处其中。
*   因此，笑着笑着容易哭出来。
*   放声大笑需要相当程度的意外和荒谬感。
*   当对生活和世界熟悉到一定程度，可能会感觉太阳底下无新事，荒谬就是生活本色。
*   不能狂笑可能是一种损失，但还能微笑就好。

**核心观点**：快乐的形式随阅历改变，接受“微笑”也是一种获得。

---

## 二、关于职场讨厌的同事 👥

上一部分讨论了个人情绪。现在，我们看看如何处理外部环境带来的烦恼。用户华毅问：如何应对办公室一位本事不大、负能量满满、经常冒犯他人的同事？

和菜头老师提供了另一种思路：

*   此人可能并不针对你，你的厌烦或许另有原因。
*   在职场或小团体中，常存在一个“人人都讨厌但地位稳固”的角色。
*   原因是，大家需要这样一个人来集中矛盾，从而彼此相安无事。
*   这样的人其实危害不大，因为人人提防他，他难以为恶。
*   如果去掉这个角色，“好人们”之间就会开始互相挑毛病。
*   没有“恶人”之后，人人可能都成了“恶人”。
*   天生万物都有其作用，等这位同事消失后，你可能会怀念过去同仇敌忾的日子。

**核心观点**：某些“讨厌”的角色在群体中起到了意想不到的平衡作用。

## 三、关于自卑与不敢追求 🏃‍♂️

处理了外部矛盾，我们转向内部障碍。用户“一尊小佛”问：因极度自卑不敢追求喜欢的女孩，如何克服？

和菜头老师认为问题核心在于自我认知：

*   你需要克服的是你自己。你对自己有一套固化的看法。
*   现实中总会有充足案例不断强化“你是个自卑的人”这一看法。
*   一种可能是，你从自卑中得到了好处，例如避免了压力和挑战。
*   和菜头以自己曾从“肥胖”中获益为例（如被更多人喜欢、免除体力活）。
*   如果不威胁健康，他可能也不愿正视体重问题。
*   因此，他不会给出“一二三”条建议，让你快速回答“做不到”。
*   如果你承认需要克服自己，如实面对自己，你会找到出路。
*   如果你认为只是存在一种叫“自卑”的缺陷，可以用工具解决，那么你总会有的挑、有的选。

**核心公式**：**问题根源 = 固化的自我认知**。改变始于审视这种认知及其带来的“好处”。

---

## 四、关于在意他人看法 🏠

自我认知也影响着我们在乎什么。用户林元元问：烦恼来源于别人的看法，虽然知道不该想但总忍不住，有什么技巧？

和菜头老师建议转换视角：

*   烦恼可能源于“你想象中别人会产生的看法”，即是你自己设计制造的。
*   “不去想”不是解决办法，因为不存在一个简单的“停止按钮”。
*   可以把自我看作一间房子，别人的看法是支撑它的房梁。
*   这本身没问题，但你需要先检查你的“房子”有多大。
*   如果是一间草房或平房，就不需要强大的支撑。
*   需要用他人看法作支撑的，是像明星、社会贤达那样拥有巨大社会影响力的“大建筑”。
*   所以，请思考你的自我究竟是多大的一间房子。一间平房就没必要用摩天楼的标准，否则太累。

**核心比喻**：**自我 = 一间房子**。根据房子大小（自我分量）选择合适的支撑（在意外界看法的程度）。

## 五、关于三十岁后对烦恼的麻木 😶

现在，我们回到课程的核心问题。用户一达问：年过三十，未成家立业，再遇烦恼时变得消极、麻木或带怨气，如何看待这种转变？

和菜头老师从“领地”需求的角度分析：

*   这种转变可能是因为“太熟悉了”。
*   就像马跑圈，赛道太短易厌倦，太长易恐慌。人也像野生动物，需要合适的“领地范围”。
*   领地太小易厌倦，太大易恐慌。需要找到合适面积让自我舒展。
*   人到三十，容易因“未成家立业”给自己压力，感到焦躁恐慌，这很常见。
*   如果领地面积不需要调节，那可能是特定的年龄焦虑。
*   解决办法是继续闷头往前冲几年，焦虑会随时间削弱。
*   人总要从“愣头青”转换为“成熟社会人”，需要打过一场才能赢得一席之地。
*   转换阶段会有在真空奔跑的感觉，凭惯性冲过去就好。届时会有新任务、新责任，心态自然稳。
*   好消息是，到40岁时，焦虑感会比30岁时弱。回想起来，那会是美好的十年。

**核心概念**：**心理领地**。保持心态需要找到适合当前人生阶段的“活动范围”。

## 六、关于对抗后悔的痛苦 😔

最后，我们探讨一个更深层的情绪。用户玄德和云长问：如何对抗因后悔（尤其是关键节点的错误选择）带来的痛苦和烦恼？

和菜头老师分享了他的个人方法：

*   他也有许多后悔和追悔莫及的选择。
*   例如，现在选择用文字而非视频表达，未来回头看可能是个败招。
*   因此，**此刻非常重要**，现在的选择影响未来，未来再悔恨也无法解决问题。
*   如果把时间浪费在追悔上，就等于放弃现在，让悔恨无尽延伸。
*   每个人都是在认知不全、信息匮乏时做决定，很可能出错。
*   他的解决办法是问自己几个问题：

以下是和菜头老师用于决策自省的问题列表：
1.  我的选择是受个人好恶影响，还是基于对现实的判断？
2.  我的选择是否以躲避风险为首要出发点，从而影响了判断？
3.  依照我现在的认知和资源，我做的是否是最优选择？
4.  我是否清楚选择可能带来的最糟结局，并能承受？

*   想清楚以上问题，即使后悔，烈度也可能减弱。

**决策自检代码框架**：
```python
def make_decision():
    if 选择受个人好恶主导:
        return “需谨慎，尝试基于现实判断”
    elif 首要目标是躲避风险:
        return “风险规避可能扭曲判断”
    elif not 是当前认知下的最优选:
        return “需收集更多信息或资源”
    elif not 能承受最糟结局:
        return “需重新评估或准备预案”
    else:
        return “可做出选择，并为结果负责”
```

## 总结

本节课中，我们一起学习了和菜头老师对多种成年烦恼的解读：
1.  **接受快乐形式的改变**：从“大笑”到“微笑”是成熟的常态。
2.  **理解职场动态**：某些“讨厌”角色可能无意中维持了群体平衡。
3.  **审视自卑根源**：克服自卑始于打破固化的自我认知并审视其“好处”。
4.  **合理看待他人看法**：根据“自我”的分量，决定在意外界评价的程度。
5.  **应对三十岁麻木**：找到适合当前人生阶段的“心理领地”，接纳转换期的焦虑。
6.  **管理后悔情绪**：通过决策前自省，减少未来后悔的烈度，并聚焦于当下。

核心在于，许多烦恼源于我们看待自己、他人及世界的方式。通过调整视角和认知，我们可以更平和地面对成年生活的复杂况味。
