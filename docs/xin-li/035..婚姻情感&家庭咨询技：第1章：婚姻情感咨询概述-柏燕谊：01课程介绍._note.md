# 035..婚姻情感&家庭咨询技：第1章：婚姻情感咨询概述-柏燕谊：01课程介绍._note

在本节课中，我们将学习本系列课程的总体介绍，了解婚姻家庭咨询的基本概念及其重要性。

我是心理咨询师柏燕谊，非常荣幸受邀参与“婚姻情感和家庭咨询技能大全”这套课程的讲授。我从事心理咨询工作已有十一二年时间。

我的个案累积时长大约有一万多小时。我个人认为，对于婚姻家庭这一领域，我之所以有较深的了解和丰富的实践经验，很大程度上得益于我与媒体的合作与接触。

在与媒体的合作接触中，我们通过节目与众多家庭进行了深入交流，与他们的情感产生碰撞，与他们的思想进行沟通。我发现，对于那些可能不会主动走进咨询室的婚姻家庭中的朋友们，这些经历让我对婚姻家庭的一些本质问题有了更丰富、更多元的认识。

在这些年里，我大约干预了三千多个婚姻家庭案例。因此，我觉得这部分经历是我非常宝贵的工作经验，也希望通过这个平台，将这些经验分享给屏幕前的朋友们。

接下来，我们来谈谈关于婚姻家庭咨询，以及咨询师应如何成长与发展。要讨论婚姻家庭咨询，我们首先需要界定它究竟是什么。

我相信朋友们在一些婚姻家庭治疗的理论书籍中，已经对婚姻家庭治疗有了比较清晰的概念。我个人的理解是，婚姻家庭咨询关注的是：**每一个个体的心理状态、心理问题乃至症状，都与其所处的环境，尤其是亲密环境，紧密相关。**

这个亲密环境主要指婚姻和家庭。个体的心理状况与其所处的亲密环境密不可分。并且在这个过程中，影响不仅来自个体自己的核心家庭，还可能扩展至大家庭的影响。

例如，代际之间那种大家族式的状态，都会对个人的内在产生诸多影响。这包括他们的沟通方式、家庭内部的权力结构、地位高低、谁更有话语权等家庭模式。

这些家庭模式对于一个人的内在心理状态，有着非常直接且必然的影响。而这种影响，很可能就是形成这个人内在问题或外在症状的一个重要原因。

因此，如果我们要对一个人进行治疗，通常在咨询实践中会发现，很多来访者最初会以“我们家孩子有问题”或“我得了抑郁症”这样的个人化表述进入咨询室。

但我们会发现，如果通过对他们的婚姻家庭模式进行干预，并与整个家庭共同工作，那么对于个体症状以及家庭整体隐含问题的治疗效果会更好。
