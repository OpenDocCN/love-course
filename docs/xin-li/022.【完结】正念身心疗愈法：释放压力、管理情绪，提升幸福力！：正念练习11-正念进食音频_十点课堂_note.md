# 课程022：正念身心疗愈法 🍇 正念进食练习

在本节课中，我们将学习正念进食的核心练习方法。通过将注意力完全投入到进食的每一个细微环节，我们可以释放压力、更好地管理情绪，并提升对日常生活的感知力与幸福感。本次练习将以一粒葡萄干为例，引导你体验如何“有意识”地进食。

## 练习准备

首先，选择一种新鲜的蔬果或坚果作为练习对象。例如葡萄、草莓、杏仁或核桃仁。本次示范将使用一粒葡萄干。

请将选好的食材放在洗净的手心上。

## 观察食材

现在，让我们开始正念观察。你可能对“葡萄干”这个名字非常熟悉，但你是否真正认识此刻手中这独一无二的一粒呢？

以下是观察的几个步骤，请带着好奇心进行：

*   **视觉观察**：仔细观察这颗葡萄干的大小、色彩、形状以及表面的斑点。
*   **处理评判**：如果过程中产生“喜欢”或“讨厌”等想法，只需注意到它们的存在即可。
*   **多角度审视**：可以捏着葡萄干，顺光、逆光、由近及远地观察它的光泽与透明度。

你可能会产生疑问：“为何要在一粒葡萄干上花费这么多时间？”此时，不妨问问自己：如果把这粒葡萄干混入千百粒之中，你还能辨认出它吗？

## 调动其他感官

上一节我们介绍了如何用视觉进行观察，本节中我们来看看如何调动触觉、听觉与嗅觉。

请依次进行以下感知：

*   **触觉**：掂量它的重量，感受它在左右手中的重量差异。体会它的温度、湿度，是干燥还是湿润。
*   **听觉**：将葡萄干轻轻放在耳边捏一捏，听它可能发出的细微声响。
*   **嗅觉**：将葡萄干放在鼻前，闻一闻它的气味。留意闻到时口腔是否开始分泌唾液。

## 放入口中感受

接下来，将葡萄干放到嘴唇上，感受它的硬度或软度。然后，将它放入口腔，但先不要咀嚼。

想象它在你的口腔中进行一次“环球旅行”。现在，可以开始咀嚼了。

请注意以下细节：

*   你习惯用哪一侧的牙齿咀嚼？（左侧/右侧，前部/后部）
*   随着每一次咀嚼，口腔内唾液的分泌有何变化？
*   从咀嚼到吞咽，是一个怎样的过程？

**核心概念**在于：正念进食是将注意力像**放慢镜头**一样，带入进食的每一个环节。

## 感恩与思考

如果我们用“放慢的镜头”去看这颗葡萄干，会发现它拥有不平凡的生命旅程。

它曾是一粒种子，经过灌溉、破土、发芽，长成葡萄藤，结出果实。再经过阳光雨露的滋养成熟，最后由果农采摘、晾晒、包装、运输，经众多人的辛劳才来到你的手中。这真是一段辛苦的旅程，也是一份不浅的缘分。

想一想，我们每日所食，可能来自五湖四海，甚至跨越国界。我们的身体每天都在进行 **“摄入 -> 消化 -> 排出”** 的循环，而这些食物最终以不同形式成为了我们身体的一部分。

带着这份觉察，在今天及以后的饮食中，不妨留意：你每天在做着怎样的食物选择？在吃什么？又是以怎样的状态吃下去的？

## 课程总结

本节课中，我们一起学习了正念进食的完整练习流程。从观察、多感官感受到咀嚼吞咽，最后到感恩思考，我们体验了如何将注意力全然投入饮食过程。这个练习能帮助我们与食物建立更深刻的连接，从而在日常生活中有意识地选择与进食，提升整体的幸福感。
