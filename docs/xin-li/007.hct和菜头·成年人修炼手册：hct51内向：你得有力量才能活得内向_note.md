# 007.hct和菜头·成年人修炼手册：hct51内向：你得有力量才能活得内向_note

在本节课中，我们将探讨“内向”这一性格特质在现代社会中的处境与价值。我们将分析社会观念变迁的原因，并为内向者提供如何在当今环境中保持自我、获得尊重与成功的具体建议。

## 概述：内向的困境与误解

内向，原本是一个中性的形容词，如今却常被赋予贬义色彩，被视为不合群、缺乏竞争力。社会普遍认为需要将其转变为外向。

上一节我们介绍了内向概念面临的普遍误解，本节中我们来看看这种观念是如何形成的。

## 社会观念的变迁

社会的思潮与共识不断变化，身处其中的人容易感到无所适从。在我小时候，人们对性格内向或外向并不特别在意，甚至隐隐更认同内向，常用“内秀”一词来赞美。同时，人们认为内向者比外向者更可能安稳度过一生。

今天的情况截然不同。内向仿佛突然成了一种需要治疗或纠正的“病症”，否则便难以适应社会。

## 观念转变的根源

我认为这种观念的改变与内向或外向本身无关，纯粹是社会转型带来的新问题。

在传统农业社会，人际关系紧密，人们共同生活在彼此熟识的小地方。一个人怎么说并不重要，大家更看重他怎么做。内向者因此显得可靠，而外向者则容易被视作夸夸其谈之徒。

转型为现代社会后，随着商业与工业崛起，需要组织大量陌生人协同工作。旧有的价值判断方式失效了。老板和经理没有时间去了解一个人是否“内秀”，能说会道、敢于表态的人显得积极性更高。

同时，大规模生产使市场与销售工作变得空前重要，需要大量外向者去接触陌生人、签订单、销售产品、收回款项。

关于这一点，有一个著名的互联网笑话：

```
所有互联网公司都吹嘘自己是技术型公司、产品型公司，但只要敢亮出工资表，就会发现其实都是销售公司。帮着卖产品的人收入最高，而非技术人员或产品人员。
```

这个笑话很大程度上可能是真的。以我与互联网巨头打交道的经验为例，客服电话总是很难打通，但对方的销售却经常主动打来，不断推销各类产品。即便是乔布斯，他对公众的主要工作也是开产品发布会，那本质上就是他个人的销售大会。

所以，不是内向者突然出了问题，而是社会改变了，对外向者的需求增多了。

## 网络时代的新挑战

更糟糕的是，随着网络世界的发展，人人都被连接上网，并拥有了一个“小喇叭”，从而演变成一个新局面：**如果你在网上没有声音，那么你就不存在**。

你可以内向、可以内秀，但小喇叭在你手里。如果你不用它在社交媒体上广播，不在朋友圈自我宣传，那么你就等于不存在。

在朋友圈里，夜夜发帖叫嚷“加班真快乐”的人，真的在加班吗？真的很快乐吗？他们这是在让老板看到，并起身给自己点个赞，那才是真的快乐。老板信吗？不重要。重要的是，夜夜刷屏，刷得多了，也就给老板刷出了“勤奋”的印象。

老板向下管理，画大饼、灌鸡汤；下属向上管理，刷朋友圈、表演加班热情，反向影响老板。如此一来，大家就平衡了。

而你作为一个内向者，默默承担重任，漂亮地完成了工作，但朋友圈一片空白，邮件组里不发一言，工作群绝不冒泡。当你晚上七点半走出办公室，迎面遇见老板，他说：“怎么那么早就下班了？整天都不知道你做了点什么。”说完一脸不高兴地走进电梯，留你站在原地，一口闷气堵在胸口。

这样的事情经常发生吧？你是否也因此想过改变性格，成为一个外向的人？想要学会在完成工作的同时，也学会表演工作，将表扬与自我表扬相结合？

## 内向者的生存策略

我认为，倒不必如此悲观，更无需勉强自己。如果问题是社会变化带来的，最好从社会本身寻找解决方案。社会对人的需求改变了，但什么是不变的呢？**社会对力量的尊重从未改变**。

以下是给内向者的三个核心建议：

### 建议一：在战斗中证明力量

你不用强行改变性格或增强社交。你只需专心“打仗”，并且最好能打赢。即便没赢，作为团队中最强的伤害输出者也行。当“战争”爆发时，内向外向根本不重要。**能不能打，是任何人都无法忽视的**。证明自己能打，也就证明了自己有力量。

力量在任何时代都会受到尊重。内向而又有力量，受到的尊重甚至更多，因为别人根本猜不透你的想法。这些年来，我就是在“战争”中找到那些话少、手黑的人的。

### 建议二：与对的人为伍

社会上，有很多人认为“能说就是能打”，“没声音就是不存在”。作为一名内向者，最好远离这样的人，因为他们缺乏识人的能力。

好在，这样的人并未占据整个社会，他们只存在于最喧闹的地方。在远离人群与热闹之处，还存在着另外一批人。他们看人看事更为深入，不喜欢根据表象下结论。他们更像是从旧时光里走来的人，喜欢看手下干的活儿，喜欢看手上的茧，能够对人做出公平的判断。

努力去寻找这样的人，并想办法获得他们的认同，与他们一起工作。一开始的确很难，但只要你找到第一个，剩下的就会如雨后春笋般出现。这些年来，我就是这样一个个找到自己的合作伙伴的。

### 建议三：保持耐心，追求精准

经验和力量的累积需要时间，这比做PPT要慢很多。

打一个比方：
*   **外向者是霰弹枪**，讲究射速与伤害面积，不求精准。
*   **内向者是狙击步枪**，强调精准与忍耐，不求速度。

如果内向者没有耐心，听到身边“霰弹枪”砰砰作响，自己也跟着用“狙击步枪”打连发，那么多半没有机会命中目标。外向者勇于尝试，对每次成败并不太在意。而内向者则需要耐心积蓄力量，最好做到一击必杀。

既然内向这种性格特点在今天看来属于“前现代社会”，那你又急什么呢？慢一点，反而会准很多。内向者追求高成功率就好，这古今中外都是他们的优点。这些年来，正是内向的伙伴一次次按住我冲动的手，才让我多少体会到成功的滋味。

## 总结与共勉

本节课中，我们一起学习了内向特质在现代社会中被误解的根源，并探讨了内向者无需改变本性，而是通过**积累力量**、**选择环境**和**保持耐心**来赢得尊重与成功的路径。

最后我想说，人的性格大多处于中间状态但有所偏向。有些人偏外向，有些人偏内向。其实，另一边的行事风格自己也能够做到，只是不想那么累而已。勉强自己总不如顺从本心来得愉快。

只是在今天这个时代，想要坚持做内向者、追求宁静生活，要比先前困难许多。因为这并非个人的事，一个内向者要面对来自社会、生活及周围人的压力。当共同的观念形成，它就会试图改造个体。

所以，想要顺着自己的心意活着，意味着要付出更多努力，变得更强大，才能与外界的压力抗衡。我希望内向的人都更强大一些，拥有更多力量。那么或许有一天，这种力量能够再次扭转社会思潮，回到一个平衡点上，让内向和外向的人都能各安其位，不用总想着跑到对方那边去。

好，这一讲就到这里。关于内向，不知道你有没有什么新的感受呢？
