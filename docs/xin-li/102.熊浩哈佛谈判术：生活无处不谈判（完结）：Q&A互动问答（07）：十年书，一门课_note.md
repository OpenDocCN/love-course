# 课程102：熊浩哈佛谈判术：生活无处不谈判（完结）🎓

## Q&A互动问答（07）：十年书，一门课📚

在本节问答中，我们将探讨如何从众多沟通与谈判课程中做出选择，并理解跟随一位优秀导师的深层价值。

谈判可以改变世界，谈判也可以改变生活。大家好，我是熊浩。

各位新马来亚的用户，我收到了一个问题。这个问题是关于在众多沟通或谈判课程中，应该如何选择。这让我想起了白居易的一段诗。

这首诗叫《乱花渐欲迷人眼》。事实就是如此。这个碎片化的时代不仅仅是因为资讯比较碎片化，同时也意味着知识比较多元。

接下来，我将与大家分享一个我当年读书，甚至至今仍在做研究或选择学习资料时所遵循的基本原则。

这个原则是：**跟对人，而不是跟对标题**。

我大学读书时，遇到过不少好老师。我选课时，会以一种类似“迷弟崇拜”的方式，选择他们的课程。这位老师开什么课，我就选什么课。

这门课也许是关于公共治理的，也许是关于《红楼梦》的，我都不在意。课程内容可能部分是老师的兴趣，部分是老师的专长。我对授课者本人的判断，是我做出选择的第一要义。

跟对人就意味着跟对了方向。跟对人能帮助我们大大减少选择过程中产生的额外成本，并避免东看西看、莫衷一是的困扰。

为什么这么说？道理很简单。我们中国人有一句话叫：**听君一席话，胜读十年书**。我为什么要跟随这个人？是因为这个人真正替我读了十年书。

说起这件事，想跟大家报告一个巧合。本人进入冲突解决这个领域，研究谈判、调解和相关专业知识，到今年正好整整十年。

2007年，我到墨尔本大学求学，在艾伦·斯奈德教授的指导下，开始带有某种偶然性地推开了冲突解决这扇大门。我在中国学习时完全没有概念和认知，不知道世界上居然还有这样一个门类的知识。

所以，正是因为到了那里，出于对老师和课程内容的好奇，我进入了一个居然要钻研一辈子的领域。从2007年到2017年，整整十年。

因此我们相信，你现在为什么要选择这门冲突解决课？你为什么要进入熊浩的谈判课堂？其中有很多原因，也许有时是你看到了我在媒体上的出镜，看到了我的公共演讲，或者你是通过各种方式与我生命轨迹有所重叠的小伙伴。

但你真正值得信任的，难道不是我替你读了十年的书吗？我们这套冲突解决方法，并非从我碎片化的经验中摘选和整合而来，不是教你“遇到王老板怎么说，遇到张主任怎么讲”。

**我们的方法是站在理论的高度上，延续着冲突解决这门学科在人类历史上积累了近60年的核心脉络**。我可以负责任地告诉各位，你真的遇到了专家。

在这门课结束的时候，我们会给大家一份长长的参考书目，让你知道今天这些所谓改变你人生观念的话语，是站在何种巨人的肩膀上，是读了哪十年的书。

**跟对人，而不是追随那些由漂亮、吸引人、被营销技巧所构建和设计的标题。这是人生选择的一种高效率方式。**

我们下次答疑再见。谈判可以改变世界，谈判也可以改变生活。大家好，我是熊浩。

### 本节总结

在本节中，我们一起探讨了如何在信息过载的时代选择学习资源。核心结论是：**选择导师比选择课程标题更重要**。一位优秀的导师如同替你“读了十年书”，能为你提供经过深度积累和系统验证的知识体系，帮助你高效地进入一个领域，避免在碎片信息中迷失方向。
