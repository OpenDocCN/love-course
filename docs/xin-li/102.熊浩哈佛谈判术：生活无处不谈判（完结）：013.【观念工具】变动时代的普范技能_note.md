# 课程102：熊浩哈佛谈判术 🧠 013.【观念工具】变动时代的普范技能

在本节课中，我们将探讨谈判作为一种超越专业限制的普范技能，及其在快速变化时代中的核心价值。我们将分析为何谈判能力至关重要，并了解如何通过培养关键竞争力来应对未来的挑战。

---

世界正在加速变化。谈判不仅是解决问题的工具，更是通往未来的钥匙。它超越了具体的实用记忆，具有更深层的意义。

知识大致可分为两类。第一类是**专业技能**（Professional Skills），即特定领域专家需掌握的能力。若你与该领域无关，则无需学习此类知识。例如，外科医生的手术技巧，外行人不必掌握。

第二类是**普范技能**（Transferable Skills）。这类技能在不同领域、职业乃至人生各阶段都不可或缺。例如，沟通能力对医生至关重要。斯坦福大学教授纳维比的研究表明，影响患者是否起诉医生的关键因素，并非毕业院校或经验，而是医患沟通是否**流畅、清晰且高效**。因此，即使你是外科医生，掌握基本谈判技能也能显著改善日常工作与生活。

普范技能之所以重要，首先在于其跨领域的通用性。其次，它的必要性日益迫切。法律学者Richard Siskind在《明日律师》一书中列举了律师所需的核心技能，如文件审查、法律检索、诉讼准备、法庭辩论、人际谈判等。这些技能环节同样可类比于其他职业。

然而，一个令人担忧的结论是：如今，这些技能环节有多少仍必须由人完成，而无法被科技替代？答案很悲观——大部分都可以被替代。2015年，多伦多大学的团队开发了人工智能律师**Ross**。它能通过自然语言指令完成法律检索、文献准备等工作。年轻律师需6至8小时的任务，Ross仅需约**30秒**。据德勤估计，未来十年内，约**39%**的律师可能失业。法律这一传统中产阶级核心职业正面临巨大冲击。

各行各业都需深思：什么是你无法被科技替代的核心竞争力？概括而言，核心竞争力至少包含三个维度：

以下是核心竞争力的三个关键维度：
1.  **精神教养**：面对变动时代的内心定力，涉及哲学思考、美学沉淀及古典价值观。这有助于应对终极关怀问题，如对死亡、无意义或生命无聊的恐惧。
2.  **处理复杂性**：机器易于替代简单、线性、可重复的工作。处理复杂、非线性的情境才是人类优势所在。
3.  **人际技能**：通过协作、沟通与谈判实现共赢。

重申一次，三个关键词是：**精神教养**、**处理复杂性**、**人际技能**。

基于以上，我们提出两项具体建议：

以下是两项培养核心竞争力的具体行动建议：
1.  **在精神维度上**，寻找能让你内心安定、不慌不乱的经典文本。例如《道德经》（约5000字）或《心经》（约200字）。选择与你投缘的经典，制定阅读计划，让思想拥有“压箱底”的重量。
2.  **在处理人的复杂性与人际技能方面**，认真学习谈判课程。谈判不仅是沟通话术，更是**人际协同的方法**。通过研习谈判，你能同时锻炼人际技能与处理复杂性的能力。

三个问题，两个方法，从明天起切实贯彻。这门课程旨在优化你的身心，不仅培养你成为卓越的谈判者，更帮助你提升对生活幸福感的感知与感恩。大学教师可分为两类：只教书不育人，或既教书又育人。后者注重价值观的匡扶与哲学理念的洞察。这部分内容或许看似“无用”，但正如庄子所言：“无用乃大用”。

本节课中，我们一起学习了谈判作为普范技能在变动时代的重要性，分析了科技替代背景下人类的核心竞争力，并探讨了通过精神修养与谈判学习来培育这些竞争力的具体路径。
