# 课程119：高情商秘籍：搞定复杂的人际关系【完结】 🎯
## 08.【情感相处】怎样做，才能真的有安全感 🔐

在本节课中，我们将学习如何在亲密关系中建立真正的安全感。我们将探讨不安全感产生的心理学根源，分析四种不同人格类型在关系中的不安表现，并学习三个实用的方法来提升安全感，同时避免破坏信任的行为。

上一节我们介绍了课程的整体框架，本节中我们来看看安全感这一核心议题。

在我二十多年的咨询生涯中，女性来访者最常提到的问题是：老师，我总是忍不住想偷偷翻看伴侣的手机。如果他没有秒回信息，我就会焦虑，并连续发送数十条信息追问他的行踪。我会反复琢磨他说过的某句话，最终得出“他不爱我了”的结论。你自己或你的伴侣是否有过类似感受？这就是缺乏安全感的表现。不仅女性，男性同样会缺乏安全感。

安全感由幼年形成的内在关系模式决定。简单来说，这是你在亲密关系中的思维方式决定的。本节课将讲解四种类型的人在亲密关系中不安全感是如何产生的，以及你该如何做才能在关系中拥有安全感。

首先，我们来探讨不安全感产生的心理学原因。

一段关系中的安全感，本质是相信对方会稳定地存在，不担心他会离开。很多时候对方并未离开，为何你会感到不安？那是因为你觉得他“有可能”会离开。这种“可能离开”的预感是不安感的根源。

从心理学理论看，每个人的初始安全感源于与母亲的关系。出生六个月的婴儿认为自己和母亲是一体的。随着成长，才逐渐意识到自我与母亲的分离。但在能够完全自理前，孩子极度依赖母亲。如果在这个时期，例如母亲因工作离开，孩子未得到妥善照料，就会产生恐惧感，认为自己被抛弃了。这类孩子成年后，在亲密关系中会下意识地在脑海中复制这种可能被抛弃的场景，这就是不安全感的来源。

在此需要强调两点：首先，被抛弃的想象很多时候并未发生，也不会真的发生。其次，这种想象可能是潜意识行为，却会形成困扰情绪，影响你与伴侣的关系。

我在第一节课说过，人的内在模式由幼年经历决定。成年后的事件与教育可能让我们在一定程度上克制和掩饰这种本能，但在关键情境，尤其是亲密关系中，我们依然会不由自主地遵循本心。没有安全感的人，其核心心结是：**“我是不是在不知情时，已经被对方抛弃了？”**

了解了不安全感产生的心理学原因后，我们进一步来看四种动物类型的人他们的不安是怎么产生的。

在前面第二节课程里，我们介绍过蜗牛型的人喜欢在恋爱中黏着对方。因为蜗牛型的人在幼年时，往往有控制欲较强的父母，他们已经习惯并喜欢上这种控制与依赖并存的相处模式。因此，如果对方一天未告知行踪，或未及时回复信息，蜗牛型的人就会开始担心：“他去哪里了？他不管我了吗？我是不是已经被抛弃了？”

下面我们来看袋鼠型的人。他们最喜欢照顾另一半，一边努力付出，一边其实在依赖这个被照顾的人。因此，他们常担心自己做得不够好，对方是否不再需要自己的照顾。例如，袋鼠问伴侣“需要我先做好饭等你吗？”，若半天未收到回复，袋鼠就会想：“他是不是不需要我做饭了？我是不是被嫌弃了？”

鸵鸟型的人（常表现为“霸道总裁”）其实也很容易没有安全感。平时鸵鸟显得骄傲神气，但如果你五分钟未回复短信，或粗暴挂断其来电，他就会想：“我是不是没有魅力了？”并进一步联想到主动联系你是件丢面子的事，之后对你主动联系他，也会故意表现出爱理不理的态度。

四种类型中，斑鸠型的人相对不易在亲密关系中陷入不安情绪。斑鸠型的人喜欢独来独往，较为理性，情感上也相对淡漠。反过来说，斑鸠型的人也较难理解伴侣缺乏安全感的感受。

好了，现在我们已经完全了解了不同类型的人在关系中的不安全感是怎样的。下面我就来教你三个在相处中提高安全感的方法。

以下是三个提升安全感的核心方法：

1.  **了解另一半的类型，根据类型特点帮助提高安全感。**
2.  **充分理解对方的状态，找到和谐相处的法子。**
3.  **找到自我调解的方式。**

**一、了解另一半的类型，根据类型特点帮助提高安全感。**

如果你的伴侣是蜗牛型且安全感较低，他需要一个能照顾他的对象。你可以在他黏你、依赖你时，主动询问他需要你做什么，并告诉他“我在”。多次重复，他会逐渐感到安全，并在关系中建立起安全感。

如果你的伴侣是缺乏安全感的袋鼠型，他需要一个被照顾、被依赖的对象。那么，找点事给他做，他就会安心去做，并相信只要还在照顾你，就不会被抛弃。例如，你可以说：“**你能不能帮我做一顿糖醋排骨？**”或“**你能不能帮我游戏账号上个段位？**”你不在时，他会喜滋滋地忙你交代的事，感觉正在照顾你。当他完成并告知你时，你要由衷表达感谢。他会感到被需要，安全感自然建立。

如果你的伴侣是鸵鸟型，他喜欢被崇拜的感觉。尽可能快地回复他的信息，对他说的内容表示赞同（**点头如捣蒜**），他会变得喜欢与你交流。下次你主动联系，他也会回复得更快。他感觉你成了他的崇拜者，从而在你身边找到存在感。

如果你和伴侣都是斑鸠型，你们对待感情的方式理性而淡薄。平时各忙各的，不频繁电话短信，关系也能很融洽。这本来就是你们双方都感到舒适的相处方式。告诉你一个小秘密：许多看似一辈子相敬如宾的夫妻，很可能就是两只斑鸠型的人组成的。

**二、尝试理解对方，找到和谐相处的法子。**

天下的事架不住你情我愿。有喜欢黏人的，就有喜欢被黏的，彼此都会满意。但有时事与愿违，伴侣可能在感情上是你最喜欢的，却并非性格上最契合的类型。怎么办？这时你要学会尝试理解对方思考问题的方式，找出适合你们的、独一无二的相处方式。

很多时候我们觉得一件事无法忍受，往往是因为它与自己的想法不同，我们不理解对方的行为。例如，我有个朋友，他的老板每次讲完电话总是直接挂断，从不说“再见”或“好的，就这样吧”。我的朋友非常焦虑，总担心老板是否生气了，自己是否要被辞退。其实只是老板性子急，不喜欢拖泥带水。了解之后，不安和恐惧就消失了。

对方不停发信息，只是因为他感到不安。在可能的情况下，你可以试着照顾这种情绪。或者，把你今天听到的课分享给另一半，让他了解你的想法。不安全感的产生，往往源于双方对同一件事的理解存在分歧。例如，我喜欢问“你在哪里”并马上视频通话，而你性情淡泊或较为高冷，不喜欢这种方式。因此，重要的是理解对方行为背后的逻辑，找出和谐相处的法子。

**三、接受自我，调节自我。**

如果你或你的伴侣非常容易缺乏安全感，那么听到这里，你应该已经明白原因及长期的改善方向。但如果某天伴侣真的好久没回信息，不安情绪涌上心头，依然非常难受。现在教你一个缓解这种情绪的办法。

你可以去打一场球，或打几把游戏。也可以打电话给其他朋友，随便聊点什么。或者，事先给自己买本字帖或绘图填色本，告诉自己：**“抄完100个字（或填完一页图）之前，都不要拿起手机。”**

你可能会说：“老师，你讲的事我都没兴趣。”没关系，你可以找自己喜欢做的事，只要满足两个条件：
1.  不需要特别集中注意力（因为我们知道你现在很焦虑，无法集中）。
2.  可以让你暂时放下手机（一直盯着手机会让时间感变得异常漫长）。

学会了三个提升安全感的方法以后，我还要提醒你，当你没有安全感的时候，一定不要做下面三件事。

以下是情绪不安时应避免的三件事：

1.  **尽量不要上网购物。** 网上有个段子说：“喜欢的人未必能在一起，遇到喜欢的东西就买下来吧。”此时你很容易因心不在焉或补偿心理，买下远超实际需要的东西。
2.  **不要跟其他人讨论“对方是不是已经不爱我了”。** 对方的回答通常只有两种：一种是“他是爱你的，只是暂时有事”。如果你相信这个答案，就不会一直狂打电话了。于是对方接下来会说：“那也许他真的不爱你了。”这个答案会在你心中种下怀疑的种子，加剧当下的不安。
3.  **答应我，无论在什么情况下都不要去偷看对方的手机。** 因为结果也只有两种：一种是你发现了问题，很生气；另一种是没发现问题，你更生气，会想：“天啊，隐藏得好深！”不安情绪的产生源于信任感减弱甚至消失。在没有信任的基础上，对方无论做什么你都会觉得没有意义。请相信，**建立一段美好亲密关系的基础是信任**。所以不要轻易去做任何破坏信任感的事。
