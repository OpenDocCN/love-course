# 118.60秒口才训练营（完结）：付费答疑：【付费答疑01】练60秒要不要先写稿子？_note

在本节答疑课程中，我们将探讨一个核心问题：练习60秒口才表达时，是否需要先撰写逐字稿。我们将分析初学者常见的误区，并提供从写稿到脱稿的渐进式训练方法。

## 概述：从“写”到“说”的必经之路

许多学员在初次尝试60秒表达时，会遇到巨大挑战。有人本以为可以侃侃而谈，但实际录制时却反复几十遍也无法满意。有人则发现开口后语无伦次，难以继续。这引出了一个普遍疑问：是否应该先写稿子？写完后照着读，是否无法提升口才？

上一节我们介绍了训练营的基本情况，本节中我们来看看如何解决这个具体的起步难题。

## 核心认知：摒弃“完全脱稿”的幻想

首先需要明确一个关键概念：**没有人是完全不准备就能讲好的**。即使是专业的表达者，也需要准备。最著名的案例是罗振宇的每日60秒，他同样需要写稿和列大纲。期望自己“张口就来”，这往往是一种不切实际的欲望或幻想。对于口才基础尚浅的学员来说，充分的准备是必不可少的。

## 训练方法：循序渐进的准备策略

那么，如何高效准备，又不耗费过多时间呢？以下是针对不同阶段的步骤建议。

### 阶段一：基础薄弱期——从“写全稿”开始

如果你的表达能力目前比较弱，确实可以先把每个字写出来。但关键在于：**写稿的目的不是为了朗读**。

写稿本身是一种输出，它能倒逼你的思考，帮助你理清逻辑。写完逐字稿后，你需要从中提炼出**关键词**。

**操作公式**：
`撰写逐字稿 -> 提炼关键词 -> 看着关键词复述 -> 反复练习`

### 阶段二：练习执行——关键词复述法

看着提炼出的关键词进行讲述，一遍遍练习。复述的内容不需要和原稿一字不差，重点是把握核心思想和逻辑脉络。通过十遍、二十遍甚至三十遍的重复，你会逐渐熟练。

### 阶段三：能力提升期——过渡到“列大纲”

当你持续练习几个月，变得比较熟练后，就可以像教练一样，只**列一个简单的大纲或几个关键点**，然后直接讲述。这时，你已经不再需要撰写完整的逐字稿了。

**进步路径总结**：
`写全稿 -> 提炼关键词复述 -> 列大纲即兴讲`

## 重要原则与总结

本节课我们一起学习了练习60秒口才时关于准备稿件的核心方法。

1.  **接受过程**：所有人的进步都是循序渐进的，不要想一步登天。认为自己可以完全脱稿，可能是一种过高的自我评估。
2.  **方法正确**：写稿没有问题，但切忌写完后直接朗读。要通过“写稿 -> 提炼 -> 复述”的流程，将书面语言转化为口头表达。
3.  **持续练习**：从写全稿到列大纲，需要时间和反复的练习。每个人都是这样成长起来的，你并不孤独。

总结来说，如果你感觉起步困难，那就从写一篇200多字的稿子开始。接着提炼关键词，并依据关键词反复复述练习。这是行之有效的成长路径。
