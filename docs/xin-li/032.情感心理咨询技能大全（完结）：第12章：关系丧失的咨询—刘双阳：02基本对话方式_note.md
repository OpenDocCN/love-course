# 情感心理咨询技能大全（完结）第032课：第12章 关系丧失的咨询 — 02 基本对话方式 💬

在本节课中，我们将探讨在面对关系丧失（如失恋或离婚）的来访者时，咨询师应采用的基本对话方式。我们将重点分析咨询对话与朋友聊天的核心区别，并学习如何通过尊重、好奇和不预判的态度，为来访者创造安全的表达空间，从而在对话中孕育新的可能性。

## 咨询对话与朋友聊天的核心区别

上一节我们介绍了关系丧失咨询的概述，本节中我们来看看咨询中的基本对话方式。首先，让我们思考两个问题：当你自己经历失恋或离婚时，你希望寻找什么样的咨询师？你希望咨询师如何与你对话？另一个问题是，当你决定与朋友倾诉失恋痛苦时，这种聊天与和咨询师的聊天有何不同？

电影《婚姻故事》的开头场景提供了一个例证。影片中，咨询师要求正在办理离婚的夫妻双方，写下并当面念出对彼此欣赏和感激之处。然而，妻子当场拒绝并愤然离开。这个例子说明，咨询师的意图（引导双方看到关系中的积极面）或许是好的，但**对话的时机与表达方式至关重要**。在双方关系高度紧张时，强迫来访者表达感激可能适得其反。

那么，我们究竟希望寻找什么样的咨询师呢？大多数人可能希望找到一位能够耐心聆听、充分尊重自己，并允许自己按个人意愿讲述故事的咨询师。这个部分非常重要。

与朋友聊天和与咨询师聊天，主要有以下两点不同：

以下是朋友聊天的常见特点：

*   **快速站队与评判**：朋友往往会迅速站在倾诉者一边，对事件或另一方做出评判（例如，“离开他是对的”）。这种快速的结论可能关闭了探索其他视角的可能性。
*   **急于给出建议**：朋友很容易直接给出建议（例如，“赶快离开他吧”或“这点矛盾不算什么”）。然而，当事人脑中可能已充满矛盾的声音，他需要的或许不是一个明确的建议，而是梳理这些声音的空间。

## 合作对话咨询中的基本实践

接下来，我们看看在合作对话的咨询框架下，面对经历关系丧失的来访者，咨询师最基本的一些做法是什么。

以下是咨询师可能采取的核心行动：

*   **询问期待**：咨询师会询问：“你对我们的咨询有什么期待？”这是朋友聊天中通常不会涉及的问题。
*   **尊重讲故事的权利**：咨询师会非常耐心地聆听，允许来访者以其自身的方式（即使看似混乱或缺乏逻辑）讲述故事。这个过程本身至关重要。咨询师应避免打断，在来访者自然停顿处再作回应。
*   **给予表达空间**：咨询师会留意是否让来访者讲出了他最想讲的内容，而不是试图引导来访者走向咨询师预设的方向。这里有一个故事：合作对话创始人曾调研来访者，问何种咨询有效。一位来访者提到，曾有咨询师反复以不同方式询问同一个问题达六七次，仿佛在等待某个“正确答案”。这提醒我们，提问应是一种**邀请**，邀请对方讲述他最想说的部分，而非我们期待听到的话。
*   **保持好奇，而非急于给答案**：咨询师会保持真诚的好奇，贴近并跟随来访者的体验。对于给建议需格外谨慎，因为咨询师无法成为来访者人生的“答案”。如果来访者主动要求建议，咨询师可以说明这些建议“并非标准答案，也不一定适用”，并提供多个不同视角的可能性，其意图是展现**存在的多种路径，而非唯一答案**。
*   **尝试理解，而非自以为听懂**：咨询师会通过**澄清**来努力理解来访者。例如，当来访者说“我感到孤独痛苦”，咨询师不会假设自己理解这些词的含义，而是会询问：“你说的‘孤独’是什么意思？”或“如果孤独像一座房子，里面会有什么？”澄清过程往往能揭示我们最初的理解与来访者真实想表达的内容之间的差异。
*   **尊重期待，跟随来访者的方向**：咨询师会不断与来访者核实对话的方向是否是其希望前往的。例如，咨询进行一段时间后，可以询问：“到目前为止，我们聊的方向是你想走的吗？接下来你希望往哪里探索？”
*   **通过好奇打开新的空间**：咨询师的角色有时像是一位前往来访者内心世界“城市”的游客。通过指出来访者习以为常的事物并提出好奇的问题（“这个建筑是什么？它和那个有什么关系？”），可以帮助来访者以新的视角重新看待自己的经历，从而**看见从前未看见的，创造新的可能性**。

## 如何运用合作对话处理关系丧失议题

最后，我们再来探讨如何具体运用合作对话的方式进行失恋、离婚议题的咨询。首先，请思考：你对于失恋或离婚的人有怎样的假设？

之所以问这个问题，是因为咨询师也是人，容易对带有某些标签（如“多次失恋”、“离婚”）的人产生预设和判断（例如，“这人可能有性格缺陷”）。在咨询中，我们需要对这些假设**保持非常谨慎的态度**。

我倾向于不提前对来访者做假设。即使来访者使用某些标签（例如，医院诊断的“中度抑郁”），我们也应认识到，**标签本身说明有限**。“抑郁”对每个人的含义和体验都可能千差万别。更重要的是去探索：当这个人说“抑郁”时，他指的是什么？他与“抑郁”的关系如何？

同样，“失恋”和“离婚”本身也是标签。如果我们能**悬置自己的假设**，或许能为咨询打开新的局面。
