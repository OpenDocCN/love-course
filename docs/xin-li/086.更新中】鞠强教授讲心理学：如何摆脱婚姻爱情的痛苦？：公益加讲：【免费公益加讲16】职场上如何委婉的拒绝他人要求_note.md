# 心理学课程086：职场中如何委婉拒绝他人要求 🚫

在本节课中，我们将学习一种非语言的沟通技巧，用于在职场或家庭中委婉地拒绝他人的要求或打断冗长的发言。这种方法通过微妙的肢体语言传递信息，既能维护关系，又能有效表达立场。

---

在工作中，我们常常遇到一个难题：如何在职场上委婉地拒绝别人的要求。例如，开会时有人发言非常啰嗦。当然，你可以直接对他说：“张主任，你讲得很好，但时间不够，下次再讲。”有人认为这已经足够委婉，但实际上这种说法仍比较直接。更直接的说法是：“张主任，你废话很多，别谈了。”

那么，是否存在更委婉的方法来拒绝他人的冗长发言呢？答案是肯定的。

上一节我们介绍了直接拒绝可能带来的问题，本节中我们来看看一种更柔和的非语言技巧。

以下是具体的操作步骤：

1.  在对方发言时，保持目光接触，面带放松的微笑。
2.  持续注视对方，同时开始非常轻微地摇头。摇头的幅度要轻到只有对方能察觉，而旁人可能注意不到。
3.  保持微笑和轻微的摇头动作。通常，对方在几十秒到几分钟内就会因感受到被否定而难以继续讲下去。

这种“面带微笑地轻轻摇头”是一种高效的拒绝方式。在职场上，根据情境需要，你可以选择不同直接程度的拒绝方式：
*   **直接拒绝**：明确说“不”或“这是废话”。
*   **中等直接拒绝**：说“讲得很好，我们另找时间讨论”，这其实仍比较直接。
*   **委婉拒绝**：采用上述非语言方法，甚至可以嘴上说“我考虑考虑”，但配合摇头的动作。对方的潜意识会接收到拒绝的信号。

这种方法不仅适用于职场，也可用于家庭沟通。例如，当伴侣提出某些要求时，你嘴上可以说“好的”，但同时轻轻摇头。通常，对方会感知到你的真实态度，事情也就自然作罢。

---

本节课中，我们一起学习了一种通过“**面带微笑 + 轻轻摇头**”的非语言策略来委婉表达拒绝的技巧。这种方法利用肢体语言传递信息，比直接的语言拒绝更为柔和，有助于在职场和家庭中维持和谐的人际关系，同时明确表达自己的立场。
