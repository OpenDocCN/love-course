## 24.【自我成长】如何才能够不那么“丧”？😌  

在本节课中，我们将学习“丧”背后的心理动力，分析不同类型人群感到“丧”的原因，并提供三个实用方法来调整这种状态。  

---

### 一、理解“丧”背后的三重动力  

“丧”这个词通常带有贬义或消极色彩。本节中，我们将探讨“丧”背后的三种心理动力。  

**第一，丧可能是想休息一会儿。**  
现代社会的快节奏和高压力迫使人们积极面对各种事务，但有时我们也需要放空和休息。  
当持续积极对待某些人或事却感到动力减弱时，容易产生“丧”的情绪，甚至怀疑自己患上抑郁症。  

**第二，丧是一种自我放逐。**  
自我放逐不同于放弃，它更像一种自我流放的感觉。  
放弃意味着失去追求、兴趣、人际关系、情绪和未来，而“丧”会在此过程中不断循环，成为一种逃避。  
例如，当追求的目标无法实现或被迫放弃时，人会感到失去方向，从而陷入“丧”的状态。  

**第三，丧意味着丧失。**  
从精神分析角度看，“丧失”是一个专业术语，例如亲人离世或失恋都意味着失去亲密的人。  
“丧”常与“失”相伴，源于我们无法接受失去某些事物或人。  
长期的“丧”通常源自强烈的丧失感。  

---

### 二、四种动物类型的人为何会“丧”？  

上一节我们介绍了“丧”的心理动力，本节中我们来看看不同人格类型的人感到“丧”的具体原因。  

以下是四种动物类型的人及其“丧”的根源：  

1. **蜗牛型的人**  
   蜗牛型的人重视人际关系。  
   如果依赖的关系突然断裂，他们会产生强烈的丧失感。  
   若无法建立新的依赖关系，这种“丧”会持续加剧。  

2. **鸵鸟型的人**  
   鸵鸟型的人注重心理资本和优越感。  
   如果令其自豪的权利或职位消失，他们会感到天塌地陷。  
   例如，在竞争中失去第一的位置时，自尊受损会导致深深的“丧”。  

3. **袋鼠型的人**  
   袋鼠型的人通过照顾他人获得满足感。  
   当被照顾的对象（如孩子）不再需要他们时，会产生强烈的丧失感。  
   这种落差会让他们陷入“丧”的情绪中。  

4. **搬鸡型的人**  
   搬鸡型的人以利益为导向。  
   当缺乏可获取利益的对象或合作者时，他们会感到目标缺失，从而变得“丧”。  

---

### 三、三个方法摆脱“丧”的状态  

了解了“丧”的成因后，本节中我们将学习三个具体方法来调整这种状态。  

以下是三个实用建议：  

1. **建立新的目标**  
   当感到“丧”时，可以尝试设立新的可追求目标。  
   例如，学习新技能、培养兴趣爱好或参与有成就感的活动。  
   公式：**新目标 = 掌控感 + 获得感**  
   通过目标建立，帮助自己平稳度过“丧”的时期。  

2. **接受事实**  
   “丧”是一种必须承受的感觉，失去总会带来悲伤。  
   但无需强化这种情绪，否则容易陷入受害者角色。  
   代码示例：  
   ```python  
   if feeling == "丧":  
       accept_reality()  
       take_initiative()  
   ```  
   主动走出受害者心态，而非一味同情自己。  

3. **绕开无法解决的事情**  
   如果某些事情无法通过努力解决，可以尝试绕道而行。  
   例如，与抑郁情绪共处而非死磕，找到适合自己的应对方式。  
   关键点：关注其他事物，自然走出“丧”的状态。
