# 118.【完】超级识人术：20.你的朋友遇到情绪如何发泄自己？_note

在本节课中，我们将学习如何通过观察朋友处理不良情绪的方式，来洞察他们隐藏的性格特质。情绪发泄是每个人都会有的行为，但不同的方式揭示了不同的内心世界和性格倾向。

## 概述：情绪发泄与性格洞察

人总有受气或不满的时候，长期堆积情绪有害身心。直接反击或向他人发泄并非上策。适度的情绪发泄很重要，但何种方式最有效？这取决于个人性格，因此发泄方式也千差万别。

你是否留意过朋友处理不良情绪的方式？不同的发泄方式，其实向外界传递了信号，展现了他们平常不易显露的另一面性格。

当一个人遭遇情绪障碍，内心长期压抑，就会变得郁闷焦躁。表现为爱发脾气、常与家人争吵、责骂孩子。发泄途径可能包括扔枕头、打枕头、捶打毛绒玩偶、对沙发出气，或在电脑图片上乱涂乱画。

吵闹是语言与肢体活动的结合，是一种宣泄。争吵喊叫能在一定程度上发泄愤怒。脾气暴躁既有先天因素，也有后天影响。从生理学讲，人体含有微量化学物质**甲肾上腺素**和**血清素**。**甲肾上腺素**含量相对过低的人，脾气较急躁易怒；反之则较温顺随和。因此，人的性格脾气受先天遗传因素影响。我们常看到父亲脾气不好，儿子也往往如此。

不同的发泄方式揭露不同的性格。下面我们一起来看看，它们能告诉我们什么。

以下是十种常见的情绪发泄方式及其对应的性格分析：

**第一种发泄方式：外出旅游**
喜欢在情绪不佳时选择外出放松的朋友，面对烦恼会积极主动化解。他们性格普遍偏外向，精力充沛，做事潇洒自如，不快之事很快忘却。

**第二种发泄方式：读书**
在情绪不稳定时选择读书排解的人，性情大多平和，善于克制自己，心绪不易烦乱。焦虑或忧愁较易得到排解，内心充盈让他们充满希望和乐观，属于积极内向型。

**第三种发泄方式：听音乐**
这类人通常比较感性，性格较为柔和。他们十分在意内心情感和精神愉悦，善于驾驭和控制内心感情。

**第四种发泄方式：忍着**
这是最伤身的做法。情绪无处倾诉，烦恼无处排泄，伤痛无力治愈。科学解释，闷在心中易憋出病。但他们不懂其他发泄方式，或希望改变却无法做到，因性格所致难以快速转换。这类人多半沉默、内向、独立、稳重。

**第五种发泄方式：摔打东西**
这样的人一般性格比较暴力、豪放、倔强，情绪容易激动。男人如此给人粗暴印象，女人如此则更令人畏惧。这不仅影响不好，对家庭及子女造成的伤害更是无法估量。

**第六种发泄方式：向好友倾诉**
朋友是最忠实的听众。以竹筒倒豆的方式全部倾吐，心里很快便没有疙瘩。这是最健康有效的发泄手法。这类人性格比较和善，信任朋友，心思单纯。

**第七种发泄方式：购物**
一些人心有不快时，喜欢胡乱买东西，不管有用没用。当家堆满用金钱换来的物品时，情绪也会好转。这样的人一般比较孩子气，不太成熟，女性居多。

**第八种发泄方式：暴饮暴食**
这是一种有趣的方式，气竟能用吃来发泄。伤心难过时使劲吃东西，不管是什么，只要能吃就行。肚子实在装不下时，烦躁也随之降低。这样的人性格简单、直率、乐观。

**第九种发泄方式：一个人沉思默想**
这类型的人自控能力较强，懂得压抑怒气，使其转化为正常感情。懂得以伤害最小的方式发泄自我，是理智、善于把控自己的人。

**第十种发泄方式：睡觉**
这一类人一般比较内向、沉静、性情平和，通常不会采取过激行为。他们细心、有韧性、不大善于言谈、忍让、可依赖。

## 总结：于细微处知人助人

总之，无论使用怎样的宣泄方式，从这些点点滴滴、微小之处，我们都可以更深入了解身边的朋友。目的不仅在于知人，更重要的是在了解之后，采取相应措施去帮助有需要的朋友。关键时要及时伸出友谊之手。

同时，这还能帮助我们培养一种平衡适意的心态，使自己能轻松愉悦地面对压力，更好地享受生活、面对工作。

本节课我们一起学习了通过十种不同的情绪发泄方式来洞察朋友性格的方法。从积极的外出旅游到伤身的隐忍，每种方式都映射出独特的个性轮廓。掌握这些知识，能让我们在社交中更体贴、更有智慧。
