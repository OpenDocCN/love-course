# 075.透视挣钱花钱背后的心理：第39讲：那些能帮你挣钱的好习惯（下）_

在本节课中，我们将继续跟随心理专家丁丽老师，学习两个关键的财富习惯：如何利用外部资源保障自己，以及如何通过投资健康来守护财富。我们将深入探讨这些概念的具体实践方法。

## 第三点：学会用别人的钱来保障自己

上一节我们讨论了储蓄与规划，本节中我们来看看如何拓展资源渠道。核心在于理解“用别人的钱来保障自己”这一概念。这并非指依赖他人，而是指通过**金融工具**和**投资渠道**，让专业机构或市场的资金为你工作，从而实现风险分散和财富增值。

许多人对此的第一反应是寻找个人依靠，例如配偶。然而，这种思维具有局限性，会让你觉得生活别无选择。如果你对这种依赖感到任何不适，例如需要看脸色或无法自由支配，就需要思考其他方式。

以下是几种可行的实践方法：

*   **进行小额投资**：你可以尝试参与一些众筹项目。初始投资金额可能只需几千或几万元。公式可表示为：`初始投资 → 项目回报 → 正向投资体验`。成功体验会鼓励你进行更多投资，并建立起“投资人”的身份认知。投资人身份并非由巨额财富定义，当你投资股权或项目时，你就是投资人。
*   **利用保险工具**：保险的本质是风险转移。通过支付保费（别人的钱汇集成的资金池），在意外发生时获得保障。其核心功能可概括为：**保障自己，也保障家人**。

当你学会这些方法，就明白了什么是**风险控制**和**嫁接风险**。你会意识到，许多能用钱解决的问题，就不再是问题。当然，有人会说“没钱是最大的问题”。这就好比想中彩票却从不购买。第一步是愿意为提升财商和未来进行投入。

## 第四点：用健康保障财富

理解了外部资源利用后，我们转向内在基础。本节探讨如何通过投资健康来保障财富。这本质上是一个健康习惯的养成过程。

一个常见的思维误区是：为了省钱（例如看病）而克扣健康支出（例如饮食），这形成了一个矛盾循环：`省钱 → 损害健康 → 需要花钱看病 → 更想省钱`。这种逻辑看似合理，实则本末倒置。

我们必须打破这个循环，将健康视为最重要的投资。投资健康主要从以下三个方面入手：

*   **投资健康的身体**：关心每日饮食，选择营养、健康的食物。这些食物未必昂贵。你吃进去的每一口，都是在进行健康投资。代码逻辑可表示为：`选择健康食物 = 投资身体健康 = 减少未来医疗支出`。
*   **投资愉快的心情**：主动关注能带来好心情的事物。例如，寻找一款喜爱的饮品、看幽默段子、将喜欢的照片设为屏保。当你对生活微笑，生活也会回以微笑。心情是健康的镜子。
*   **投资及时的休息**：留意身体的疲惫信号，并在需要时允许自己休息。许多人习惯于透支，认为一切都比自己重要。尊重身体，及时休息，能有效防止心情因身体不适而变差。因为当身体难受或极度疲惫时，我们很难保持好状态和好心情。

此外，多与大自然接触也能有效改善心态。面对浩瀚的自然，你会感到自身的渺小，许多烦恼也会显得不再那么重要。这种超然的状态需要亲身去体验。

---

本节课中我们一起学习了两个重要的财富习惯。首先，我们探讨了如何通过**小额投资**和**保险**等工具，“用别人的钱”来分散风险、保障自己。接着，我们明确了健康是财富的基石，需要从**饮食**、**心情**和**休息**三方面进行持续投资。掌握这些习惯，能帮助你在追求财富的道路上走得更稳、更远。
