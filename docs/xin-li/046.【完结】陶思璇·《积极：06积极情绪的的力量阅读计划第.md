在本节课中，我们将继续精读《积极情绪的力量》第六章。本章的核心是探讨积极情绪如何帮助我们培养“百折不挠”的韧性。我们将深入理解韧性的可建构性，学习一个关键的实践细节——“给他人帮助你的机会”，并掌握一个名为“六步摆脱消极情绪法”的实用工具。

## 章节内容精要

上一节我们探讨了积极情绪的扩展与建构作用。本节中，我们来看看积极情绪如何成为我们在逆境中的支撑力量。

本章的核心观点是：**让我们百折不挠的韧性是后天习得、可以建构的资源**。能够帮助我们建构韧性的，正是积极情绪。

作者从“心”（心理感受）和“脑”（神经科学）两个方面提供了研究数据。这些数据说明，积极情绪能提高一个人的抗压能力与复原力。它可以帮助你在消极情绪的恶性循环中及早刹车，重启良好感觉，从而进入良性循环。

随后，作者通过分享个人经历，阐述了如何在日常生活中创造积极情绪，以培养韧性。书中的第116页第三个自然段是重点。阅读时，重心不应放在琐碎的生活细节上，而应体会作者通过细节传递的重要信息：**触动心灵，才会释放出更多的积极情绪。爱，比数据更重要。**

## 一个需要留意的关键细节

在这一章内容里，有一个细节需要你特别留意。

作者写道：“我第一次完全相信人们需要提供帮助。我甚至告诉别人，他们怎么做，才能带来最大的帮助。”

现在，请你回想自己的人生经历，特别是职场晋升之后：
*   你上一次请别人帮忙是什么时候？
*   你请别人帮的是什么事？
*   你在请求时是什么心情？是否感到忐忑或不安？
*   还是说，你可以安心、大方、自然地请别人帮忙，并且很享受这个过程？

在我接触过的客户中，越是优秀的人，越少开口求助。不到万不得已，绝不求助他人。即使开口，心里也在盘算如何偿还人情，极少有人能心安理得。我们习惯了自己的事自己做，请人帮忙成了一种心理负担。

但是，我们忘记了，爱的能力包括了爱与被爱。每个人都需要被看见、被听见、被接纳。**同时，每个人也都需要“被需要”。**

学会在适当的时候请他人提供适当的帮助，但又不对他人过度依赖，是心智成熟、心理健康的一种表现。

你可以在听完本节内容后，即刻转向身边的人，思考一下：你可以请他帮一个什么样的忙？这个忙既能让他感受到自己被你需要，又不会给他增添麻烦（即，让他认为是举手之劳并乐在其中）。

**温馨提示**：你可以像作者一样，从小事开始，给出具体清晰的提示。例如：“麻烦你帮我拿一杯酸奶，我要香草味的。”坚持这样做，你会有意想不到的收获。

关于“让别人帮你”的神奇力量，畅销书《别独自用餐》讲得非常透彻。如果你感兴趣，可以找来看看。

## 核心方法：六步摆脱消极情绪法 🛠️

以上是关于本章内容的要点提示。接下来，我将重点分享一个方法。因为作者对于“如何触动心灵”和“如何培养韧性”并未给出具体的实操方案。

我将与你分享一个简单有效的练习方法，帮助你在遇到挫折、被消极情绪困住时，为自己创造积极情绪。我还会在今天的练习区留下一个相关练习。

现在，请跟随我学习下面这个方法。它旨在训练你在困境中看到希望，摆脱消极情绪漩涡，更新自我设置，在黑暗中创造积极情绪。

这个练习分为六个步骤，以下是具体操作指南：

**第一步：回顾**
找一个空白笔记本或一张A4纸。在纸上写下一件对你影响较大的事情。写下事情发生的时间、地点以及你的情绪体验。
*   **示例**：你去见客户谈合作，被客户拒绝。
*   **情绪体验**：你首先感到沮丧、失望（目标未达成）。继而产生挫败感、自责（怪自己口才不好）。之后可能扩大这种情绪，认为自己倒霉，做什么都不顺。
*   **原理**：人们通常在事件当下产生某种情绪，而后因这种情绪给自己一个结论，该结论又会催生新的情绪。回顾书写能帮你更清晰地认识自己的情绪状态。

**第二步：阅读**
重复阅读你所写下的内容，不限次数，直到你感到厌倦为止。“厌倦”是一个重要的转折点。

**第三步：思考**
请你做一个非常重要的功课：认真思考，如果这件事如你所愿达成了目标，你会有哪些收获？尽可能详细地写下来。收获不仅包括物质收益，也包括情绪体验、荣誉等非物质获得。
*   **公式**：`目标达成后的收获 = 物质收益 + 情绪价值 + 其他非物质收益`
*   **目的**：这个步骤会让你如实看到目标对你的重要性，这决定了你的行动力和最终结局。

**第四步：重建信心**
如果目标确实重要，请进入此步骤。回想自己的人生或工作经历，有哪些事让你感到自豪和骄傲？让你自豪的原因是什么？对比这次挫折经历和让你自豪的经历，其中的差别在哪里？

**第五步：破局思维**
现在请想一想，除了已经试过的方法，在这件事上，你还可以做哪些事情来推动目标实现？记录下你能想到的一切可能性。
重复阅读这些新的可能性，在脑海中逐一模拟过程，感受是否还有可修改补充之处，让这些可能性更丰富、具体、可操作。
*   **代码逻辑**：
    ```python
    while 未找到满意方案：
        生成新想法()
        模拟评估(新想法)
        if 可优化：
            优化补充()
    ```
    **目的**：你的补充越精细，成功的几率就越高。

**第六步：创造新起点**
最后，也是最重要的一步：放下所有之前的经验，让自己完全“清零”。你的内在有多少新的空间，就会有多少新的机会和灵感进入。
当你完成这个练习，你会发现，办法真的比困难多。你会再次回到积极情绪状态，由内而外充满激情与活力，对你而言，一切皆有可能。
