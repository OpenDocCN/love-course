# 📚 课程091：人际关系必修课（完结）  
## 03. 🤝 赢得同事好感：如何成为一个有趣的人？

在本节课中，我们将学习如何与社交型（I型）的人有效相处，理解他们的核心特质，并掌握赢得他们好感的实用方法。

---

强烈建议大家观看视频学习。视频中插有大量真实场景案例，老师还会对案例进行深度剖析，搭配学习效果更佳。

有这样一群人，他们的热情好像一把火，能燃烧整个沙漠。他们是社交达人，从不恐惧陌生。他们热情开朗，遇到别人难过了，甚至会说：“努力不一定会有结果，但是一定会很爽。”他们快言快语，轻易承诺。在宫斗剧中，如果没有主角光环或者开挂，绝对活不过三集。但他们是你冬天里的棉袄，是夏天时的冰棍，是你的风，是你的沙，缠缠绵绵到天涯。

大家好，我是李海峰，又见面了。前两集我们谈了DISC的理念和理论，也分享了如何让D型风格用得多的人成为我们的优质人脉。布置的作业，大家要记得完成。学而不习，基本无效。

我们用个假设的情境，带领大家再熟悉一下DISC四个特质的区别。一家四兄妹，特质各不同。有天家里起火，大哥使用D风格，马上关电闸，找到灭火器。二哥使用I的风格，大喊大叫：“起火啦，来人啦！”三妹使用S的风格，起火后，一边希望消防队早点来，一边在照顾老人小孩，并且安抚大家。四弟使用的是C风格，在分析起火的原因，以及如何防范。

我们发现，四个风格运用得当，又能彼此配合的话，就能够应对各种情况。火势很小的时候，大哥使用D，问题当下就能得到解决。但如果一个人灭不了火，就要靠擅长使用I风格的二哥。大家被火分散注意力，三妹却能很好地照顾最重要的人。四弟则可以从教训中吸取经验，防范于未来。

其实我们每个人都有这四种风格，只是使用的倾向性不同。学好DISC，我们就可以让自己保持弹性，选对位置，切换模式，用好特质，发挥优势。

上集我们重点讲解了领袖型的D风格。这集我们重点和大家分享社交型的I风格。

狄更斯《双城记》中写道：“这是一个最好的时代，这是一个最坏的时代；这是希望之春，这是失望之冬；人们面前应有尽有，人们面前一无所有；人们正在踏上天堂之路，人们正在走向地狱之门。”世界有正反面，人也一样。I特质用得好，是快乐的制造者；但是如果运用过当，也是麻烦的制造者。

以上的街头采访，大家抱怨的都是以I风格使用比较多的人在沟通中遇到的问题。I特质多的人，他们富有创意，追求感觉。如果说领袖型的人更多的像霸道总裁，讲究权威，看重快和效率，那么社交型的人则更像是开心果，看重的是感觉和自身的愉悦。

描述他们用两个词：人来风、自来熟。人越多越兴奋。在他们的心目中，没有陌生人，只有朋友和还没有见过面的朋友。

社交型的伙伴，常常的表现是：他们往往有阳光般的乐观心态。他们追求快乐，也制造快乐。状态好的时候，能在大片乌云下看到天边云端有彩虹。别人说“真糟糕，只剩半瓶水”的时候，他们高兴地说：“太好了，还有半瓶呢！”

出场动静大，人未至，声先闻。《红楼梦》中，王熙凤为了取悦贾母，使用I型的风格出场。曹雪芹在书中是这么描述的：“一双丹凤三角眼，两弯柳叶吊梢眉，粉面含春威不露，丹唇未启笑先闻。”

衣着鲜艳，喜欢装饰。社交型的人像一棵圣诞树，挂了很多的手环、耳环，发型很搞怪。核心原因是希望得到他人的关注和认可。你得罪他最好的方法，就是当他不存在。I的本意词，其实不是恨，而是冷漠。

如果有一天，一个I特质比较高的伙伴跑过来很兴奋地跟你讲：“哎呀这个这个这个……”你只要说三个字就可以彻底得罪他，那就是：“我知道。”——“哇，这好玩！”“哦，我知道啦。”“你看这个真的很好玩！”“我知道我知道。”——“怎么都知道，没意思。”

讲话幽默有方，善于营造各种氛围。有位高I的伙伴写的诗是这样的：“水晶色的天空，飘着粉绿色的雨，我从台阶上摔下来，先看到了星星，然后就看到了你。”其实我们想想，什么叫水晶色的天空？什么叫粉绿色的雨？什么是台阶？什么是星星？怎么又到你？只有一种可能，就是从台阶上摔下来，眼冒星星。但I最牛的是，哪怕现场这么狼狈，它可以描述得非常唯美。

民国的才女林徽因，有首诗《你是人间的四月天》是这么表述的：“你是一树一树的花开，是燕在梁间呢喃，你是爱，是暖，是希望，你是人间的四月天！”他到底在讲什么？不过逻辑不重要，最重要的是感觉很浪漫。

而民国的才子徐志摩的诗是这么写的：“走着走着就散了，回忆也淡了，看着看着就累了，星光也暗了，听着听着就醒了，该醒的醒不来了，回头发现你不见了，突然我乱了。”各位有没有想过，为什么这么多诗人很喜欢发布I型的词？因为有I型的词，你特别能调动各种感觉和感受。

喜欢肢体的接触。对于他们来讲，千言万语不如一个拥抱。再频繁的线上沟通，不如见面时候的拥抱。如果你想让他离职，最好的方式就是把他带到单独封闭的房间，收走手机，拔掉电话线，关掉Wi-Fi。不久他就像没了氧气的花，毫无生机。

有很多的朋友。对于他们来讲，最能打动他们的句子是：“莫愁前路无知己，天下谁人不识君。”又或者是“桃花潭水深千尺，不及汪伦送我情。”

在知乎上有这样一个问题：“很重视地友情，对方却不看重，是一种怎么样的体验？”有网友给出了答案：诗圣杜甫给李白写的诗，据不完全统计，就有十首，而李白却写了一篇《赠汪伦》。各位有没有想过，如果杜甫是一个I特质的人，他能坚持着持续地给李白写这么多篇吗？我想他内心一定是崩溃的。所以如果你是一个I特质的人，也不能和李白这个臭小子一样，只顾自己爽，要给别人一些回应。否则只会出现“桃花潭水深千尺，很想把你给淹死”的局面。

社交型的人在钱上要出现条。朋友找他借钱，他会因为好面子而不拒绝。借了钱后，也不好意思让别人写欠条。在花钱上没有什么规划，好点的是月光，更差点的是半月光。忽然想起余光中的诗：“月光光，月光光，月无光，月无双，落在谁的伤口上。”

喜欢用答辞答歌来应对压力。往往一个项目还没结束，他们已经开始自我奖励了。有的时候对事情本身没有什么促进，但是自我感觉很好。

喜欢用夸张的词汇。他们会使用大量的形容词、程度副词、叠词，以及有画面感的词，还有一些流行的词汇。

我们了解了使用I风格比较多的社交型的人，我们就能尊重他的需求，做好他的拍档。他们看重别人的认可，哪怕很小的事情，他们也希望被别人知道或发现。和大家在一起，他们是群居动物。对于他们来说，哪怕站在世界的顶端，身旁没有人陪伴，又怎样？

感觉比行动更重要。有一个点子，最重要的是要拿出来分享。分享的时候如果能得到大家的羡慕，那做不做其实已经不重要了。

更看重眼前，而不是未来。他们习惯于活在当下，所以总能笑口常开。“有花堪折直须折，莫待无花空折枝。”“今朝有酒今朝醉，明日愁来明日愁。”

我们明白了社交型的风格最看重什么，那么就能够更好地跟他们沟通，让他们帮助你、欣赏你，成为你的优质人脉，就变得手到擒来。核心的关键，是让他们觉得你快乐而且有趣。

怎么做他的好朋友呢？以下是四个方法和你分享。

**以下是四个与社交型（I型）人相处的核心方法：**

1.  **给予足够的认可**：我们不要吝啬对他的赞美。有的时候不可避免要给他建议的时候，要留意赞美跟批评的比例，最好是 `3:1`。赞美的时候尽量要具体一点，不要只是说“你好漂亮”，要更细节地说“你的胸针真好看”。

2.  **给到足够多的反馈**：给到足够多的被展示、被突出的机会。有了好的反馈，他们是太阳照到哪里哪里亮；没有好的反馈，他们变月亮，初一十五不一样。

3.  **给他有趣的体验**：比如我们DISC双证班的现场，喝的是欢乐逗红罐玻璃瓶的，赠送毕业生的是带着他们名字的3D定制U盘，还会配备不同颜色的套笔，有些班级还有定制的衣服、搞笑的人物背景板。

4.  **对他们定期的跟进**：社交型的人对待设定的目标，有的时候会三天打鱼两天晒网，遇到挫折或不顺就很容易放弃。最好的方式是承诺过的事情，用书面的方式写下来，定期跟进。
