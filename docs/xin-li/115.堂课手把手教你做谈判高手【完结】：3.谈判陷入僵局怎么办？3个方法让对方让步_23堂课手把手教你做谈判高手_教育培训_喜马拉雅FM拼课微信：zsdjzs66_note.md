# 谈判高手课 115：谈判陷入僵局怎么办？3个方法让对方让步 🛠️

在本节课中，我们将学习当谈判陷入僵局时，如何运用策略让对方让步。核心在于有效展示专业知识和灵活运用谈判技巧，包括“暂时唬住对方”和“敢于耍无赖”等方法。我们将逐一拆解这些概念，并通过具体案例说明其应用。

## 专业知识的积累与呈现

上一节我们介绍了谈判的基础，本节中我们来看看如何运用专业知识。专业知识是谈判中的重要资本，但仅有积累不够，必须能有效呈现并说服对方。

专业知识包含几个层次：收集到的信息（`information`）、可用的情报（`intelligence`）、系统化整理后的知识（`knowledge`）。在谈判中，你需要将这些内容清晰表达。

以下是有效呈现专业知识的要点：

*   **匹配对方的接收习惯**：对方可能是“听觉型”或“视觉型”接收者。英文中有“sounds reasonable”（听起来有道理）和“looks reasonable”（看起来有道理）之分。你需要判断对方偏好。
*   **针对听觉型对象**：如果你的老板或客户是“听觉型”，喜欢听，你应在旁边讲解，确保语言简洁、重点突出，避免冗长重复。
*   **针对视觉型对象**：如果对方是“视觉型”，喜欢看，你的提案（`proposal`）必须书面呈现。务必仔细检查，避免错别字，因为一个错字可能让对方觉得你不认真。如果对方办公室有白板，喜欢图表，你的提案中最好包含流程图等视觉化元素。

**核心建议**：在谈判中，专业知识最好用作“盾牌”进行防御和支撑，而非主动攻击的“刀”。不要卖弄学问，因为双方知识结构不同。专业知识的作用之一是“暂时唬住对方”，为谈判争取优势。

## 方法一：暂时唬住对方

谈判在一定程度上需要虚张声势，即“暂时唬住对方”。这并非欺骗，而是一种策略性施压。

这个概念类似于两车在狭路相逢的对峙。历史上，林肯曾用身高优势在月光下投射出长长的影子，让对方误以为他难以对付而选择退让。事后对方问起，林肯的回答是：“你不让，我就让。”这说明气势和心理博弈的重要性。

**案例**：2002年，一家公司的采购经理与选定的手机托架供应商谈判。供应商以已投资20万人民币备料为由不肯降价。采购经理直接表示：“我不要了。”并解释：亚太区选定的是规格，而他谈判的是价格。如果价格超出预算，他可以选择不采购。十分钟后，供应商来电接受降价。采购经理的策略是：判断对方沉没成本（20万投资）过高，赌其不敢放弃订单，从而成功逼对方让步。

**公式**：**判断对方沉没成本 > 己方谈判破裂风险 → 施加压力逼其让步**

## 方法二：敢于耍无赖（策略性强硬）

当常规方法无效时，可以采取“耍无赖”的策略性强硬姿态，拒不退让。这适用于己方承受损失能力更强的情况。

这同样可以比喻为两车对峙：

*   **比谁更不怕“刮蹭”**：如果己方是“旧车”，对方是崭新的“奔驰”，那么可以暗示“我不怕刮蹭，但你损失更大”。这就是“穿皮鞋的怕穿草鞋的”。对于家大业大的企业，负面新闻（“见报”）就是难以承受的“刮蹭”。
*   **宣称“不会倒车”**：直接表明己方没有退路或能力有限。例如，“我是新手，不会倒车”，或“我知道欠钱，但真的没钱”。这种“示弱式要赖”往往让对方难以破解，因为对方失去了施压的着力点。

**核心要点**：使用这类方法需要认清自身位置，评估对方是否“玩不起”或“输不起”。企业尤其需要重视媒体公关，避免因小失大。

## 课程总结

本节课我们一起学习了打破谈判僵局、促使对方让步的三种核心思路。

首先，**有效积累并呈现专业知识**，将其作为谈判的“盾牌”，并学会针对不同接收习惯的对手进行有效沟通。

其次，**善用“暂时唬住对方”的策略**，通过分析对方的沉没成本和谈判底线，在关键时刻施加压力，迫使其做出让步。

最后，在特定情况下可以**采取“策略性强硬”姿态**，利用己方承受力优势或“示弱”方式，让对方感到无计可施，从而打破僵局。

记住，谈判是综合实力的博弈，灵活运用这些方法，方能掌握主动权。
