# 情感心理咨询技能大全（完结）：第09章：EFT（情绪聚焦）家庭治疗 - 第07课：情绪的建构 🧱

在本节课中，我们将要学习情绪的建构理论。我们将探讨情绪失调如何通过“情绪的恶性循环”模型表现出来，并理解个体的自我是如何由多个部分“建构”而成的。核心在于认识自我冲突与未完成情结，并学习如何在咨询中处理这些内在的冲突。

上一节我们介绍了情绪的四种失调类型，本节中我们来看看这些失调在个体身上是如何具体运作和表现的。

我们以“情绪的恶性循环”模型为例。一个**刺激**（例如，感知到的失落或失败）会诱发原发的适应性情绪反应（如难过）。这种原发情绪可能激活与过去相似场景相关的**情绪基模**（Emotional Schema），进而诱发出适应不良的情绪反应（如羞耻、无价值感）。这种适应不良的情绪又会衍生出消沉、无望的感觉，并伴随低能量的行为反应（如“我做不到”）。

如果此时个体在想法上存在冲突（例如，“我必须保持前行”的完美倾向），就会加剧不适感。这种“低能量状态”与“高要求”之间的冲突，会让人对最初的刺激产生更强烈的失落感，从而形成一个不断恶化的**负性循环**。

在这个循环中：
*   **绿色的部分**代表原发的、适应性的正常情绪反应。
*   **红色的部分**代表被激活的、适应不良的原发情绪。
*   **蓝色的部分**代表衍生出的次级情绪（如绝望）。

在咨询中，我们的工作是帮助来访者在情绪调节的适中状态下，去体会和探索。目标是引导他从衍生的情绪（蓝色），回溯到适应不良的核心情绪（红色），再进一步触及最初的情绪基模（绿色），从而看到潜在的创伤或“情绪开关”。

以下是咨询中常见的情况：

来访者常常会陷入强烈的自我批评，例如认为自己不好、不值得被爱。这种自我批评会放大羞耻感，加剧情绪困扰。我们发现，情绪问题往往与人际议题紧密交织。例如，一位经历离婚的来访者，其悲伤情绪可能加剧了“我是一个失败者”的自我批评。而这些人际议题（如被忽视）本身就是一个“情绪开关”，会触发更多的情绪反应。

个体的发展是一个**建构**的过程。我们并非像电脑一样单纯处理信息，而是将经历**意义化**。从出生到现在的所有学习、身体感知、人际关系和事件，都像积木一样，一点点构建成我们的人格和内在世界。这就是建构主义的观点：我们根据自身特征、情绪反应模式和人际互动，最终形成一个**叙事的自我**——即“我是谁”的故事。

正常的自我功能是持续变化且多面向的。它由多个不同的“自我部分”组成，就像一个内在的**合唱团**。例如，可能有一个“批判者”，一个“被批判者”，一个“思考者”，一个“感受者”。这些部分各有其角色，彼此之间也可能产生冲突。

我们来看一下自我建构的层次图：
1.  **最底层**是生理基础：神经元、神经化学物质、边缘系统等。
2.  **之上一层**是多个**情绪基模**（情绪开关），它们是高级的自我组织。
3.  **再往上**，多个情绪基模整合成一个**自我组织**。多个不同的自我组织，就构成了我们内在的多个“自己”，例如“勤快的自我”、“完美的自我”、“苛刻的自我”。

当某个情绪基模被强烈激活时，对应的“自我部分”可能会变得突出，并与其他部分产生不可调和的敌意，这就导致了**自我冲突**。例如，“勤快的自我”要求下班后继续工作，而“想休息的自我”渴望放松，两者冲突会让人感到痛苦和耗竭。

当自我不同部分之间出现敌意和压迫关系时（如“批判者”对“体验者”充满蔑视），就会导致功能失调。这可能表现为过度压抑或忽略内在的某个部分，或将冲突双方完全隔离。这会阻碍个体有效地接触自己的情绪结构。

例如，当“苛刻的批评者”不断说“你不好”，而“体验的自我”也认同“对，我就是不好”时，整个系统就会变得僵化，能量降低。那个“做不好”的脆弱自我被压抑了，可能表现为拖延行为——因为害怕做不好，所以干脆不做。

在治疗中，那个被压抑的“脆弱自我”可能隐藏起来，使得咨询师和来访者都难以接触和改变它。这个“沉默下来的我”可能通过一些不适应性的行为或习惯表现出来。

咨询师需要协助来访者识别和分开这些冲突的自我部分（例如，“批评者”和“被批评者”）。治疗的目标是让这两个部分重新接触、对话。改变的过程会使对立的双方逐渐同化或相互适应，从而增加整体的心理弹性。

自我建构失调主要有两种表现方式：

**1. 自我冲突**
这是指来访者内在的一个自我部分对另一个部分进行压倒性的压迫，导致自我功能变差。例如，“完美的自我”不断打击“不完美的自我”，导致来访者一有不完美表现就感到自己很差劲。咨询需要调解这两个自我部分之间的冲突。

**2. 未完成情结**
这是自我冲突的一种特定形式，指我们内化了重要他人（如父母）的态度。例如，我们学会了母亲的完美主义，当自己不够完美时，内化的“母亲”就会跳出来批评我们。这个“批评者”的声音，可能源于内化的父母，而非真实的自己。

这类似于精神分析中的自我、本我、超我冲突。在EFT框架下，我们关注的是内在多个“我”之间的冲突如何导致功能不完整。

以下是两种常见的、与自我建构失调相关的倾向，它们常出现在情感创伤案例中：

**依恋型抑郁倾向**
这类人可能在幼年有不良依恋经历（如被遗弃、留守）。他们内化了“我是不被爱的”脆弱自我感。在亲密关系中，他们极易因分离或失落而触发深层的被抛弃恐惧，陷入“我无法独立生存”、“我注定孤独”的灾难化感受中。这就像旧伤未愈，新的分离事件会同时触动新旧伤痛。

**完美型抑郁倾向**
这类人可能成长于高要求、低支持的环境中，未形成有力的自我感。他们擅长自我批评和攻击，核心情绪体验是羞耻和焦虑。在情感创伤中，他们更倾向于责备自己不够好、没有达到要求。

当来访者陷入“我不好、我不对”的自我批评时，通常意味着内在有一个“苛刻的自我”在不断进行指责。这个声音充满敌意或轻蔑，会加剧绝望感和无力感，甚至引发抑郁、焦虑情绪。

我们可以通过来访者的**口语表达**来识别这种内在冲突：
*   **纠结型表达**：“我想…但是我不敢…”、“我希望…但是我做不到…”。这表明两个势均力敌的自我部分在斗争。
*   **明显分裂型表达**：“我应该…但是我做不到…”、“我想…但我能力不足…”。
*   **负面自我评价**：“我是一个失败的人”、“我毫无价值”。这直接体现了“批评者”部分的声音。

本节课中我们一起学习了情绪的建构理论。我们了解到，情绪失调会形成一个恶性循环，而个体的自我是由多个部分像积木一样建构起来的。当这些部分（如“批评者”与“被批评者”）发生冲突或存在未完成情结时，就会导致自我功能失调。识别来访者口语表达中的冲突，是理解并干预其内在自我批评和情绪困扰的重要起点。
