## 方法一：前后一致法

第一种方法称为“前后一致法”。其核心原理是：**一个人撒谎时，其虚构内容的记忆并不深刻**。如果所述事件不真实，在不同时间反复询问同一问题，回答很容易出现重大不一致。

以下是应用此方法的具体步骤：

1.  **初次询问**：向伴侣提问，例如“今天你到哪里去了？”
2.  **追问细节**：针对其回答，询问与**场景图案**相关的具体细节。心理学规律表明，图案化信息更不易遗忘。例如，如果对方说“去上海交通大学听鞠强教授的课”，可以追问：
    *   鞠教授是男是女？年龄多大？
    *   听课人数是几十、几百还是上千？
    *   地点在闵行校区还是徐汇校区？
    *   鞠教授体型是胖是瘦？
3.  **记录答案**：**务必悄悄用本子记录下对方的回答**。否则时间久了，提问者自己也可能忘记。
4.  **再次询问**：间隔一段时间（例如两周后），再次询问相同的细节问题，例如“你上次听鞠教授的课，是在哪个校区？人数大概多少？”
5.  **对比分析**：对比两次的回答。如果对方确实在现场，**绝不可能**将几百人记成几十人，或将闵行校区记成徐汇校区。若出现此类**重大纰漏**，则说明当天活动可能存疑。

**核心逻辑公式**：`真实经历 -> 细节记忆稳定`；`虚构经历 -> 细节记忆易矛盾`

## 方法二：道序法

接下来我们看看第二种方法：“道序法”。此方法要求提问者具备一定的家庭权威感，否则难以实施。

其原理在于：**一个人叙述虚构的时间顺序时，很难将其倒过来流畅复述**。

应用步骤如下：

1.  **要求详细叙述**：询问伴侣当天的行程，并要求其按时间顺序详细说明。例如：“你今天到哪里去了？上午、中午、下午、傍晚分别在哪里？”
2.  **聆听并记忆**：仔细听取对方的完整叙述。
3.  **提出倒序挑战**：在对方讲完后，冷静片刻，然后要求：“**现在，请你把刚才说的顺序倒过来，从晚上开始往回说一遍。**”
4.  **观察反应**：观察对方能否流畅地倒叙。对于撒谎者而言，倒叙编造的时间线极为困难。

**核心逻辑代码**：
```python
if 叙述为真:
    倒叙复述的难度 = 低
elif 叙述为假:
    倒叙复述的难度 = 高
```
