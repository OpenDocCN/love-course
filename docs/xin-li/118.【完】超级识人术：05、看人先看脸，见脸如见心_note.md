# 118.【完】超级识人术：05、看人先看脸，见脸如见心_note

在本节课中，我们将学习如何通过观察一个人的面部特征，在对方开口之前获取关于其气质、情绪、性格和态度的信息。面部是情感与性格的直观反映，掌握面部识人技巧，有助于我们在社交中更好地理解他人。

## 观察脸型

中国传统相学认为，脸型是个人内外和谐统一的表征。通过观察脸型，可以帮助我们初步判断一个人的品行和特长。以下是三种常见的脸型及其性格特征分析。

**国字脸（方形脸）**
*   **视觉特征**：脸盘较大，轮廓感弱，呈扁平感。两腮较为膨大突出。正面看，面部两侧轮廓垂直向下，与下巴轮廓线近乎直角相交。核心特征是方额头、方下巴，整体呈四角扩张的轮廓感。
*   **性格特征**：这种人通常喜欢独来独往，具备高智慧和敏锐观察力。他们心态积极，性格坚强，不轻易气馁。但不喜欢迁就，略显固执，缺乏通融性，对决定的事情会坚持到底，因此容易与人发生冲突。他们好奇心强，喜欢富有挑战的生活。

**前额发达的三角脸**
*   **视觉特征**：下巴尖翘瘦弱，前额饱满发达。拥有此脸型者通常身材修长，但体力较差。从生理学看，额头发达意味着脑前叶发达，属于感觉神经系统发达的类型。
*   **性格特征**：如果眉毛长得黑，则对感情专一，但情感需求易受心情左右。个性较为急躁，但也较为热心，做事随心所欲。需要注意的是，与这类人合作时，他们做事可能出现**虎头蛇尾**的情况。

**圆脸型**
*   **视觉特征**：给人一种圆润丰满的感觉。
*   **性格特征**：性格磊落大方、心地温厚。若能踏实奋进，通过付出行动可以获得成功。在个性上，他们处理事务的能力可能不够严谨，但不喜欢与人争执，性情较为平和。圆脸的人通常心境开朗活泼，想象力丰富，但也正因为想象力丰富，容易导致精神不够集中。

上一节我们介绍了如何通过脸型进行初步判断，本节中我们来看看如何从更细微的眼神变化中窥探他人内心的秘密。

## 解读眼神

眼睛与人的思想情感有着密不可分的关系。观察眼神是分辨一个人品质和心术的有效方法。心思正直的人，眼神通常是明亮的。

在交谈中，可以观察对方眼神的以下变化：
*   眼神由亮变暗：这是较为正常的疲劳状态变化。
*   眼神突然变得明亮：这通常说明你的谈话内容引起了他极大的兴趣。
*   对方看着你并眨眼：这往往表示他对你印象不错，喜欢你的成分较大。
*   眼神冷峻严肃：这可能带有警告的意味，需要注意分辨。

以下是根据眼神特点进行的分类解读：

**眼神闪闪发亮的人**
*   这类人通常有野心和企图心，精力旺盛。但从另一方面看，他们也可能是个很难对付的家伙。

**目光呆滞暗淡的人**
*   从性格上讲，这类人可能缺乏斗志和情趣，是一个严肃乏味的人。

**目光忽明忽暗的人**
*   拥有这种眼神的人通常工于心计。如果你想用言语诱惑或利用他们来达到自己的目的，很可能会失望。

**主动与人接触视线的人**
*   说明其内心比较坦荡。

**眼神安详沉稳的人**
*   说明其内心有主见。

说完了眼神的解读，最后我们来探讨一下头发与个性之间的关联。

## 分析头发

一般来说，无论男女，从头发的状态也能看出一些个性端倪。

**从发型态度分析**
*   保持头发顺其自然状态的人：通常较为安分守己，性格上甚至有点内敛含蓄，比较重视自己的主观意识。
*   非常注重发型的人：通常自尊心较强，追求完美，是一个比较喜欢挑剔的人。

**从发质质地分析**
*   **头发质地强硬的人**：性格较为刚愎自用，自我意识非常强，不容易听进别人的话。他们理性大于感性，在情感表达方面可能表现得较为笨拙。
*   **发质柔软的人**：通常喜欢出风头，希望吸引外界目光，获得别人关注。他们较为自负，情感上容易动情，但对感情可能不太专一。
