# 124.【完结】：01【公开课】心

在本节课中，我们将深入探讨如何识别并处理内心的冲突，特别是通过“外化作用”来保护自己。我们将学习几种常见的心理防御机制，如投射、责任归咎和合理化，并理解它们为何无法真正解决问题。最终，我们会明白，真正的成长始于认识并接纳真实的自我。

## 处理理想化形象的问题

上一节我们介绍了理想化形象及其带来的问题。本节中，我们来看看如何应对它。

我们知道理想化形象造成了诸多问题。这时，你需要开始去了解心中哪一部分是理想化形象，哪一部分是真正的理想，然后去放下理想化的形象。开始真正按照自己的生活目标去努力。

## 自我愤怒的外化

接下来，我们看看“自我愤怒”的处理方式。

自我愤怒是对自己很愤怒。我们通过外化以后可以避免自我破坏。精神症或抑郁症患者可能因为对自己的愤怒而最终自残或自杀。但我们外化以后，对别人愤怒，可能会对别人吼叫、指责、抱怨，这保护了我们自己不伤害自己。

但大家不要以为这种保护很好，就去对别人愤怒。这种保护是无效的，常常会伤害别人，同时也破坏了自己和他人的关系，让关系越来越糟糕。总有一天你就无法外化了，或者这种外化根本不可能真正保护你。

真正的保护来自于了解自己、完善自己，并与他人建立良好的关系。这样才能避免对自己愤怒，也无需对别人愤怒。

## 内在胁迫的外化

现在，我们来探讨“内在胁迫”的外化。

内在胁迫的外化会带来一个好处：如果个体觉得这种压抑是来自于外界的，那么至少他是可以反抗的。同样，如果他相信这种胁迫是外部强加的，自己就能够避免它。外部强加的胁迫，只要不去面对或离开那个环境，总能维持一种自由的幻觉，不用受其制约。

但一个人经常走入一个误区。例如，很多人走入婚姻后觉得很不幸，认为自己的不幸都是由婚姻造成的。他会把这种不幸都强加给婚姻，认为是婚姻的模式强加给他的。但如果是婚姻强加的，为什么别人就幸福，他就不幸福呢？所以他会觉得，这是婚姻强加给我的，只要我离婚了我就自由了。是这样吗？其实不是。我们都知道，离婚并不能拯救一个人。

我们在面对内在胁迫时，会使用外化作为一种功能。但我们还是要发现，这些外化并不能真正对我们有益。

## 当现实与理想化形象相反时

我们接着来说一个问题：如果患者发现自己存在与理想化形象相反的各种特征，那又如何外化？

例如，我理想化自己应该是很温和的，结果我见到别人就发怒了。我理想化自己应该是很智慧的，可是我做决定的时候就老是做错。总之，我理想化的形象，和我现实中有一些部分是相反的，怎么办呢？特别是当这些相反的是品质时——比如我觉得自己应该勇敢、聪明、智慧、善良，但我身上正好有些特征是相反的——怎么办呢？

有一种外化方式，也是一种防御方式，叫“投射”。

**投射** 是指把自己主观上拒绝的倾向或者品质，看成是别人的。也就是把自己不想接受的品质，看成是别人的。

例如，明明自己很背叛，明明自己对别人有敌意，却总觉得别人对自己有敌意。明明自己很背叛，还说“你看这些人一点都不听话，这么背叛”。明明是自己很自以为是，还总说别人是自以为是。就是会把自己不愿意面对的这些品质，看成是别人的，怀疑别人有这种倾向。这样，他就可以说“这不是我的想法，也不是我的品质，这是别人的”。这是非常常见的一种外化。

实际上，投射这个概念需要注意：并非所有“把我的想法揣测别人”都是投射。

曾经有学员问：那按照这个意思，我每天都在投射别人了？比如我的好朋友吃饭时间快到了，他摸了一下胃，我就觉得他饿了，那我不就在投射他了吗？这和揣测或者是判断有什么区别？

有一个简单的区别：
第一，投射通常是将我们不能接受的倾向或品质投到别人身上，认为是别人的。例如，我很小气，可是我又不愿意接受我是小气的人，我觉得我很大方（理想化形象）。但事实上可能我很小气。那我和同学一块吃饭去了，结果我们俩AA了，我会说“他真小气，吃了饭还和我AA”。你看，这是把自己小气的品质投射到别人身上。这种投射通常是我们自己不愿意接受的、负面的品质或倾向。

那么，我刚才那个例子：我同学摸了一下肚子，我觉得他饿了。这个没有任何负面的品质和东西，他只是对这个行为的一个判断。比如我们笑，开心了，这是很正常的；哭，伤心了。这是我们基于无数的案例判断和人类的共同行为特征总结出来的一些经验，所以他并不是投射。

什么是投射？我再举个例子。假如今天你脸上长了个疤，长了个疙瘩，很丑，你觉得很丑。然后你出去以后给同事说话，同事看了你一眼，停顿了一下。你此刻就会想：他一定在想我很丑。这就是因为“我很丑”是我不愿意接受的，那这时候他的任何一个行为，我可能就把我的想法投射到他身上——“他在想我很丑”。这是我们说的一种外化方式，也就是把自己不喜欢的东西外化到别人身上。

## 责任归咎：另一种外化方式

但还有一种外化方式，是需要我们说一下的：把问题的责任归咎到别人身上。

这是一种单独的、不管什么动辄就把自己的责任归咎给别人的方式。我在这儿特别强调的是：我们所有心理学课程都会讲童年，但是讲童年绝不是为了让大家把责任归咎于童年。因为这种归咎和了解是不同的。

所以，有一些人他们稍微学了点心理学，觉得我现在的问题与我成长的环境有关系，与我爸妈养育我的方式有关系。然后好了，整天回去归咎于童年：“都是你们不好，都是你们怎么怎么样，都是因为你们那时候不爱我，都是因为你们不呵护我，都是因为我想要什么你们根本不关心……”总是把所有的问题都归咎于父母，谈论童年对我有多么的不好，我现在多惨。其实这恰恰是一种对自我的逃避。

我们经常说，我们了解童年不是为了去怪罪父母。我们了解童年只是为了了解我是如何走到今天的。在走的过程中，哪有个石头绊了我一下，害得我现在天天想着那块石头。那么OK，我现在放下那块石头，其实是为了让自己去掌控自己的命运。因为我们知道，童年的时候一个孩子是没有能力去掌控命运的。我们回忆那些事件，是为了让我们用现在成年人的思维、成年人的承受力、成年人的心理状态，去替我们解决曾经的那个问题。并不是说让我们现在成年了，还每天抱怨当年没有替我们解决问题的父母或者别人。

所以，如果一个人把所有的问题都归罪于童年，那么这个人一定是在外化，把责任归结给别人，以便于他逃避自我。“问题反正不是我的问题，都是你们的问题。那我现在很痛苦，你们改变吧，你们来帮我吧，你们来给我处理吧。”反正就是他把自己的生活推给别人，自己再也不做自己生活的一个推动者。这种情况下其实就是逃避自我。

我顺便提醒一下，这是我们这个课讲的时候我经常跟大家讲的。有些人说我们去了解童年怎么怎么样，实际上我们了解童年，在我看来就是了解童年恰恰只是说，知道比如有些人说我总是特别害怕什么。那如果我们不去分析，你为什么害怕？好了，可能我们找来找去发现，你害怕的根源是因为小时候曾经有过那么一件事，而这个事被你无限的放大，一直以来你害怕。那这时候我们其实现在就回忆那件事，是为了去了解它、解开它，让当年的那个内心的疙瘩给消融了。那其实很小的事情怎么会让我如此害怕呢？因为那时候我是个孩子，现在我已经不怕了。慢慢的你就会自我疗愈。所以我们不要去归罪这些东西。

## 其他几种解决内心冲突的方式

除了外化作用，在理想化形象和外化作用后边还有一个章节。这个章节还讲了其他几种我们常常会使用的方式来解决内心冲突。

以下是这些方式的简要介绍：

*   **盲区作用**：很多神经症患者，他实际行动和理想化形象其实是完全不同的，但是奇怪的是他看不到这种不同。所谓盲区就是我们像一叶障目一样，有一个东西在你身上是很明显，可是我就是看不到。为什么？不想看。其实是不想看。不想看呢，所以我们总会有条件去实现它。例如，有些人对自己的情绪体验是麻木的，所以他让自己体验麻木，他就不用去体验了。
*   **隔离作用**：一个人是怎么区隔化很多问题让自己避免冲突呢？例如，有的人对待朋友一个样，对待敌人一个样；对待家人一个样，对待外人一个样；对待工作一个样，对待生活一个样；对待平级那样，对待下属这样，对待领导又是另一样。总之，他觉得划分开这样我就不冲突了。例如，我对待领导和对待下属完全不同。他其实是在逃避自己的冲突。他觉得反正外在的东西被划格成不同的，既然你们不同，我就可以用不同的方式来对待你们。但事实上，他自己内心有冲突，比如我又想这样又想那样，这时候很痛苦。那我这样对家人，那样对同事，可不可以？好像很和谐。但事实上他自己内心还是不同意、不统一的。这就是为什么我们说人格不完善。我经常在写，我们找对象的时候，要考虑对方的人格是不是统一、是不是完善。因为有这样的人，他们对待不同的人态度完全不同，这部分是你特别要小心的。你看着他可以给你很合理的理由说：“不同的人就是不一样嘛，这说明他很现实、很聪明、很会处理事情。”但事实上我们要看什么事。体现出来的人格，不是说一个事一定有固定的做法，而是说有一些事他在变，他背后的人格不变。但是如果一个人他对着这一部分人和对着另一部分人，好像呈现出完全不同的人格特点，那大家要小心。
*   **合理化**：这是非常常见的一种防御，当然在这也是一种外化（只要是把问题归于外界的，我们都可以认为是外化作用）。**合理化** 就是为自己的行为找到一些理由。可以简单这么理解：我不管做了什么事，我总是有理由的，这叫合理化。

    例如，前一段时间网络上很流行的一件事：两个小孩钻到别人的车里，但别人的车的车锁是坏的，所以他们进去以后就出不来了。可是这个主人并不知道，然后这两个小孩就在这个车里被闷死了。这个事情很痛苦。结果小孩的父母就来指责对方：“你们为什么不锁车？”——他的孩子跑别人的车里了，然后他指责别人为什么不锁车。

    我记得还有一个事情是：有几个小孩出去，把共享单车的车锁给破坏了，然后骑着共享单车就出去了，结果出车祸了。这一下小孩的父母就开始起诉共享单车公司，原因是什么呢？说“谁让你们的锁质量不好，能被撬开呢？”这在我们外人看来觉得太不可思议了。是不是你们家出事，啥都是别人来负责呀？这恰恰是一种合理化。他哪怕找到一点点为自己（实则是推卸责任）能够负责的、可能真的是让我们看来太不合理的理由，他就会找到。例如：“对呀，你锁质量好的话，我们小孩撬不开，我们小孩不就不开车、不骑着共享单车上路了吗？不骑这个单车上路，不就不会有车祸了吗？”看，挺“合理”啊。他可以完全抛开这个社会其他的规则——他的孩子破坏了别人的东西，然后还让别人来承担责任。但是作为当事人，作为很多神经症患者，他们确实会这么去合理化的。

    例如，男人打老婆，他总有原因，哪怕很不合理的：“你为什么找男人笑？”或者再不合理他都能想出来。

    了解了合理化，我希望大家在生活中要注意自己说的合理化。因为我们其实一个人想去做一个行为的时候，他总是能找到一个合理化的理由，但实际上这个合理化可能非常不符合我们的现实、我们的生活、我们的逻辑和我们的常识。

其他几种方式书上写得很清楚，我就不陈述了。这是我们说的外化作用。

## 课程重点总结与成长方向

我觉得这本书到现在为止，重点基本上都讲完了。

重点其实就是：
1.  前边三种人际关系中的基本态度（亲近他人、对抗他人、疏远他人）。
2.  基于这三种态度的冲突。
3.  我们又会去用理想化形象来缓解冲突。
4.  然后又因为理想化形象和自我之间有很多的矛盾，为了解决真实自我和理想化的我之间的差异，我们也会去运用一些外化的作用，去保护自己不受冲突的伤害，或者不去面对这些冲突带来的焦虑。

到这儿我觉得这本书重点完成了。

大家肯定会有一个疑惑：那我读了这么多，我怎么改变自己？

实际上我一直有这么一个观点：我觉得大家学心理学，跟着这个读多了会发现，所谓自我成长，很多时候不是别人给一个答案，而是当我们知道什么是错的时候，自然就走上了正确的路。我一直这么说。因为当我们意识到我这个行为是在理想化自我了，什么叫正确呀？你开始放下这个理想化自我，就是正确。错误的反面就是走向正确的方向。只不过这是一个过程，可能在开始的时候，你会不断地、可能来来回回地认识自己。

其实没有别人比你更了解自己。每个人真的可以这么说：这个世界上没有人比你更了解你，但是这个世界上也只有你最不了解你。这个意思恰恰就是把这本书的内容给总结了。你自己内心的各种冲突只有你自己最清楚，但是为了避免这些冲突，你使用了很多的外化方式，导致你最不了解你。

那现在我们通过这本书恰恰是让我们来发现：哦，原来我可能某些东西是在强制性地要求自己呀，可能理想化了自己呀。当我们意识到这些防御并把它放下的时候，你内心的冲突会出现。但是内心的冲突是真实的你。而这些真实的你，你可以在这个中间利用反思去想：比如我在亲近他人的时候，我真的是在亲近他人吗？我是在讨好他人，让他人来控制我的生活，是因为我对自己负不起责任。可是当我们意识到这些的时候，开始去为自己的人生负责的时候，你的行为恰恰就发生变化了。

所以，我觉得这本书的重点基本上就完成了。

那么后面还有几章是讲结果：如果神经症患者这么发展下去，可能造成的结果。那部分我觉得不作为重点，大家可以作为了解就行了。因为它是一个类似于说“我们现在如果不成长，整天发展越来越严重，最后会什么样”的推演。它最后可能会恐惧、人格衰弱。它就像最后得了抑郁，只是告诉你可能会有什么结果。我觉得这个并不是我们现在去学习的重点，所以我也不再讲了，大家作为一个自我了解。

那么咱们语音的课其实就结束了。但我会给大家再加简单的一节，从这本书讲完以后，我们大概应该怎么成长。

实际上在自我成长的过程中，我已经给大家推荐过一本书了。如果你觉得你确实有很多恐惧，可能是来自以往的经验、以往的家庭，那么你可以去就着那本书做一些自我疗愈。因为自我疗愈也是一个专门很大的门类了，它并不是一天两天的事情。包括心理咨询，有的可能要做——像问题严重的，它可能做上一两年、两三年、三四年的一个精神分析，然后结合自我疗愈。所以这不是我们本书的重点。

本节课中，我们一起学习了如何处理内心的冲突，认识了理想化形象、自我愤怒、内在胁迫等概念，并深入探讨了投射、责任归咎、合理化等常见的外化防御机制。我们明白了，这些机制虽然能带来暂时的保护，但无法真正解决问题，甚至可能破坏关系。真正的成长始于放下这些防御，认识并接纳真实的自我，为自己的生命负责。希望你能将这些洞察应用于生活，踏上自我完善的旅程。
