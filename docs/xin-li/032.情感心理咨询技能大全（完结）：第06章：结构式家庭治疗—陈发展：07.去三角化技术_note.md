# 课程032：情感心理咨询技能大全（完结）第06章：结构式家庭治疗—陈发展：07.去三角化技术 🏠➡️👨👩👧

在本节课中，我们将要学习结构式家庭治疗中的一个核心干预技术——**去三角化**。我们将从如何评估家庭中的三角化现象开始，逐步讲解进行干预的基本原则和具体技术。

## 概述：什么是三角化？

三角化是家庭结构中的一种常见互动模式。当夫妻子系统（父母）之间存在未解决的冲突时，他们可能会无意识地将孩子卷入其中，形成一种稳定的三人关系模式，以此来缓解或转移夫妻间的紧张关系。我们的目标是识别并打破这种不健康的模式。

## 第一部分：如何评估三角化？🔍

我们不能直接询问家庭“你们家有没有三角化”。评估三角化需要融入对整个家庭结构的理解中。以下是评估时需要关注的几个方面。

### 评估整体家庭结构

首先，需要观察家庭围绕所呈现问题展开的叙述。其次，可以了解家庭生活中典型的一天是怎样的。这有助于我们发现，家庭所认定的“问题”有时并非核心，真正的问题可能隐藏在日常生活之中。

### 评估家庭成员对问题的理解与解决方式

家庭成员如何理解和解决他们认定的问题至关重要。例如，父亲认为孩子的问题是母亲溺爱造成的，这种理解本身就揭示了夫妻间的不一致。我们需要特别评估这种理解上的差异。

### 评估父母间的冲突

第三，要评估父母或夫妻之间是否存在长期未解决的冲突。

### 评估孩子的角色

第四，观察孩子如何理解和应对父母之间的这些问题与冲突。例如，当父母争吵时，孩子是躲起来，还是会偏向其中一方？这能帮助我们判断孩子是否被“三角化”。

### 评估“问题”的功能

最后，需要评估孩子呈现的问题（如不上学、抑郁）对父母原有冲突产生了何种影响。是让冲突暂时搁置了，还是让父母意识到了夫妻关系对孩子的影响？这揭示了三角化中“问题”可能具有的转移矛盾的功能。

### 总结与反馈

在完成上述评估后，再判断家庭结构中是否存在三角化的互动形式。如果存在，需要用家庭能理解的语言将这种模式反馈给他们。例如：“我注意到当你和太太争执时，儿子会显得特别紧张并帮妈妈说话，你平时有观察到这个现象吗？”这有助于家庭意识到他们的互动模式。

上一节我们介绍了如何系统地评估家庭中的三角化现象，本节中我们来看看，一旦识别出三角化，应该如何进行干预。

## 第二部分：去三角化的干预原则与技术 🛠️

以下是进行去三角化干预的几个基本原则和具体技术。

### 原则一：强化夫妻子系统

如果问题根源是夫妻冲突，治疗的核心是强化夫妻子系统。可以组织夫妻单独会谈，将治疗焦点定义在两人之间。目标是恢复夫妻间相互支持、共同解决问题的能力。即使夫妻感情淡漠，也应引导他们学习不将冲突转移到孩子身上。

### 原则二：强化父母子系统

对于离异或感情疏离的夫妻，我们要尊重他们的家庭形式，但需强化其作为“父母”的子系统功能。可以提醒他们：“你们可以不是夫妻，但一定是父母。”父母子系统的功能包括：**接触孩子、保护孩子、指导孩子**。任何一方都无权剥夺另一方履行父母责任的机会。

### 原则三：设置清晰边界

三角化的核心机制在于子系统边界不清。干预的关键是设立规矩、划清边界。要明确哪些话题仅限于夫妻之间讨论，孩子不应涉足。在治疗中，可以通过“家庭微雕塑”来物理化地设置边界，例如让夫妻坐在一起，让孩子坐在稍远的位置，或建议后续仅夫妻参与会谈。

### 技术：反转技术

反转技术旨在解除孩子对父母的“强制性忠诚”。当孩子习惯性地卷入父母冲突时（如自动安慰母亲），治疗师会鼓励采取与习惯相反的行为来打破模式。

**操作示例**：
*   对孩子说：“我看到妈妈难过时你会去安慰她。从今天开始，你可以试着把这个责任交还给爸爸吗？告诉他：‘爸爸，请你来安慰妈妈。’”
*   对父亲说：“你愿意从今天开始，主动把你的妻子从儿子手中‘抢’回来，承担起安慰她的责任吗？”
*   对母亲说：“下次当你伤心时，可以首先尝试向你的丈夫倾诉，而不是孩子吗？”

这些提问和引导旨在**反转**原有的自动化互动模式。

## 总结与展望 📚

本节课中我们一起学习了**去三角化技术**。我们首先了解了如何通过评估整体家庭结构、成员认知、冲突和孩子角色来识别三角化。接着，我们掌握了干预的核心原则：**强化夫妻子系统**、**强化父母子系统**和**设置清晰边界**，并学习了**反转技术**这一具体操作方法。去三角化的本质，就是将**两个人（夫妻）的问题，从第三个人（孩子）身上移除**，通过打断固有的不良互动模式来恢复家庭子系统的健康功能。
