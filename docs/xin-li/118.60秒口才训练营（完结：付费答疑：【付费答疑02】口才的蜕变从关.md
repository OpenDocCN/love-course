# 118.60秒口才训练营（完结：付费答疑：【付费答疑02】口才的蜕变从关

在本节课中，我们将探讨一个颠覆性的观点：口才练习的初期，关注自身优点远比挑剔缺点更重要。我们将理解其背后的原因，并了解口才成长的五个阶段，帮助你建立信心，迈出坚实的第一步。

上一节我们介绍了训练营的基本理念，本节中我们来看看如何正确开启你的口才练习之旅。

我发现很多学员在完成自我介绍或练习后，总喜欢说“请大家看看我哪里有不足，请多批评指正”。我郑重地告诉大家，在练习的第一个月，永远不要让别人指出你的缺点。这个观点可能比较颠覆，你或许会疑惑：不指出缺点，我能进步吗？答案是：能。

想象一下你小时候学走路。当你走得不好、摔倒时，你的父母会说“这孩子真笨”吗？不会。他们会鼓励你：“没事，站起来再走。” 当你的口才基础还很薄弱时，不要一开始就让很多人挑你的各种毛病。因为你刚开始练习，肯定存在很多问题，例如逻辑不清、语气平淡、普通话不标准等，这非常正常。如果你已经讲得像我或罗振宇一样好，那就不需要参加这个训练营了。你要明白，那些缺点或不足本来就存在，这没关系。要知道我们这是一个为期一年的训练营，第一个月不要关注自己的缺点。

因为关注缺点过多，会让你更加不自信和自卑。你应该做的是不断鼓励自己。例如：“看，我今天又完成了一个60秒练习，我非常棒。” 中国有超过13亿人口，而和我们一起练习的人还不到500人。你就是这500人中的一员，这比例是 **13亿分之500**。所以，只要你开口讲出来，你就已经很棒了，不要去在意那些缺点。这是我特别想强调的。

那么，什么时候可以开始关注自己的不足呢？当你已经能讲得比较流利，发现自己可以进一步精进，并且逐渐建立起自信之后。

在这里，我同时补充口才练习的五个阶段，帮助你定位自己当前的水平。

以下是口才能力成长的五个关键阶段：

1.  **不敢讲**：很多人不是不会讲，而是不敢开口。
2.  **要讲出来**：核心是开口，不管讲得多差，张开嘴就胜利了一半。
3.  **讲的顺**：能够比较有逻辑、有层次地表达，不要求逻辑完美。
4.  **讲出彩**：在语气语调、声音、动作、表情、手势、眼神等多个维度上提升，为表达加分。
5.  **自然讲**：无需刻意准备，拿到话题稍作思考就能流畅表达数分钟，达到收放自如的境界。

对于大部分人来说，在尚未达到“讲的顺”这个阶段时，不必苛求太多。先达到“讲的顺”，再去追求“讲出彩”。即便是我，也不见得总能达到“自然讲”的状态。所以大家不要着急，成长需要慢慢来，尤其是口才训练。只要你按照正确的方式持续练习，相信我，一年后当你再回头听今天的录音，你会发现自己蜕变之大超乎想象。

我们一起加油吧。不要想太多，不要听太多大道理，也不要看太多技巧。练起来，讲起来，你一定可以变得更好。

本节课中我们一起学习了口才练习初期应关注优点以建立自信的重要性，并了解了从“不敢讲”到“自然讲”的五个成长阶段。记住，迈出第一步并持续练习，是蜕变的开始。
