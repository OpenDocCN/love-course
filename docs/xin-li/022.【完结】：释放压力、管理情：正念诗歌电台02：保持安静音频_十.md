# 022.【完结】：释放压力、管理情：正念诗歌电台02：保持安静音频_十

在本节课中，我们将一起学习如何通过正念诗歌来释放压力、管理情绪，并提升幸福感。我们将重点赏析诗人聂鲁达的作品《请安静》，并从中提炼出实用的正念练习方法。

## 概述

思想家帕斯卡尔曾经说过，人类所有的问题都源于无法安静地坐在一个房间里。

经历了这个疲惫而大起大落的春节后，相信大家对此有了更多领悟。

生活中似乎总有许多个应接不暇，以及怎么忙也忙不完的事情。

没完没了的工作、琐事、饭局、应酬，让人喘不过气来。

就算因为疫情不上班待在家里，脑子里也有忙不完的焦虑和不安。

“请安静地暂时停下来”，是诗人聂鲁达给我们的建议。

安静坐下，才是真正的疗愈的开始。

## 诗歌赏析：聂鲁达的《请安静》

接下来，请和我一起欣赏这首来自于聂鲁达的诗《请安静》。

现在，让我们数到十二，然后全部保持静止。

在这大地上，至少这一次，让我们不讲任何语言。

让我们停下片刻，不要过多地挥舞双臂。

这将是个奇妙的时刻，无需匆忙，没有发动机轰鸣。

所有人都一起团结在，突然出现的异常气氛里。

在寒冷的海上，渔夫不会去伤害鲸鱼。

而那采盐的人，看了看自己受伤的双手。

那些策划绿色战争，毒气战火战的人，牺牲一切的胜利者，都穿上干净的衣服。

与弟兄们一起漫步于树荫下，什么都不做。

## 诗歌的深层含义与正念启示

不要困惑，我不是要求完全的停滞不前，生活需要继续进行。

但我不要它和死亡相关联。

如果我们不是单单执著于，在生命中不断进取，那么至少这一次什么都不做。

也许一种巨大的沉默，能中断我们的悲伤。

这种从未了解自己的悲伤，和对死亡逼近的恐慌。

也许大地可以教会我们，当一切看似灭亡的东西，到后来证明其实还活着。

## 实践练习：如何“保持安静”

上一节我们探讨了诗歌的深层含义，本节中我们来看看如何将“保持安静”的理念付诸实践。

以下是三个简单的步骤，帮助你开始练习：

1.  **设定意图**：告诉自己：“现在，我决定暂时停下，保持安静。”
2.  **进行计数**：缓慢地从1数到12，将注意力完全集中在数字和呼吸上。公式可表示为：**专注 = 注意力集中于（呼吸 + 计数）**。
3.  **全然静止**：在计数的过程中，保持身体静止，不讲话，不做任何事，只是观察当下的存在。

现在我要一直数到十二，你们保持安静，我要走了。

## 总结

本节课中，我们一起学习了诗人聂鲁达的《请安静》这首正念诗歌。

我们认识到，生活中无处不在的忙碌和焦虑，可以通过有意识地“保持安静”来中断。

这种练习的核心在于，暂时从“不断进取”的模式中抽离，通过沉默和静止来连接内在的自我与当下的平静。

记住这个简单的正念代码：**停下 -> 计数 -> 观察**。
