# 📚 课程078：人际交往2期 - 07. 《卡耐基口才的艺术与人际关系全集》 🗣️

在本节课中，我们将一起学习戴尔·卡耐基的经典著作《卡耐基口才的艺术与人际关系全集》。卡耐基是美国现代成人教育之父，也是著名的心理学家和人际关系学家。他从一个不善言辞的普通人，通过自身努力，成长为一位沟通大师。本书的核心内容可以概括为两大模块：**口才艺术**与**人际关系**。我们将逐一拆解其中的关键技巧，帮助你提升沟通能力，建立更好的人际关系。

---

## 第一部分：口才艺术

上一节我们介绍了课程的整体框架，本节中我们来看看如何提升口才。卡耐基将练好口才归纳为六个核心要素。

以下是练好口才的六个要素：

1.  **突破自我，克服恐惧**
    恐惧心理是普遍存在的，许多人害怕当众讲话或与特定对象（如领导、异性）交流。卡耐基指出三点：第一，你并非特例，很多人都有此问题；第二，适度的紧张能让你更专注，激发潜力；第三，即便是职业演讲家，登台前也会紧张。因此，不必过分苛责自己。

2.  **培养自信，针对性练习**
    充分的准备是自信的基础。林肯曾说，未经准备，再有才能的人也难以说出有水平的话。开口前应广泛收集素材，深入思考，理顺思路。此外，可以通过身体语言（如站直、直视观众）和心理暗示（如想象观众有求于你）来稳定心态。针对自身弱点（如逻辑混乱、声音小、小动作多）进行刻意练习和纠正，例如对着镜子或录音反复演练。

3.  **自我暗示**
    就像奥运冠军以自己为对手一样，我们需要不断进行积极的自我暗示，相信自己能够成功。这种心理建设有助于坚持和突破瓶颈。

4.  **不断学习，积累谈资**
    感觉无话可说的根本原因在于知识储备不足。你需要不断学习，给自己“积累墨水”。学习的途径包括阅读、听课、观看知识类节目（如财经、文化、艺术类内容），从生活和新闻中积累话题。

5.  **不断练习**
    口才如同游泳，不下水永远学不会。你需要抓住一切机会开口说话，将积累的话题付诸实践。例如，看到一个新闻后，主动与他人展开相关讨论。

6.  **永不放弃**
    “不断”学习与练习的关键在于坚持。你可以为自己设定小目标，例如“一个月内公开发言三次”，并为此持续努力，遇到困难时积极寻求指点。

## 第二部分：成为说话高手的六项修炼

掌握了口才的基础要素后，我们进一步学习如何成为说话高手。这需要更精进的修炼。

以下是成为说话高手的六项修炼：

1.  **尊重他人，勿当面指错**
    人人都好面子。当面指出他人的错误，尤其是上级的错误，极易引发反感，甚至招致麻烦。说话高手首先会尊重他人的意见，尝试理解对方为何如此认为，而不是急于评判和挑错。

2.  **承认错误，以退为进**
    勇于承认自己的错误，往往能赢得他人的谅解与尊重。案例中的销售员小王主动向老板承认失误并赔偿，反而获得了更大的信任。这体现了 **`诚实 + 担当 = 信任`** 的公式。

3.  **避免争论，勿正面冲突**
    赢得争论往往意味着失去关系。永远不要为了口头上的胜利而与别人发生正面冲突，尤其是面对无理取闹者。上一本书《好好说话》中提到的“拦截”而非“回击”技巧，在这里同样适用。

4.  **寻找话题，鼓励对方多说**
    沟通的关键在于倾听。人都渴望被关注，鼓励对方谈论自己及其感兴趣的事情，能让你更受欢迎。书中的案例表明，有时不说话（倾听）比拼命推销更能促成交易。

5.  **心理引导，让对方说“是”**
    在谈话开始时，设法让对方连续给出肯定的回答，能使其心理上更倾向于接受你后续的建议。技巧在于**站在对方立场提问**，引导其自己得出你想要的结论。公式可概括为：**`认同对方 -> 提出事实性问题（引导回答“是”）-> 自然导向结论`**。

6.  **重视对方，牢记其名**
    记住别人的名字，是对他人最基本的重视和尊重。卡耐基本人能记住数千人的名字。若记不住，可在初次见面时主动确认（“请问是哪几个字？”），并在交谈中多次重复，结合对方特征进行记忆。

## 第三部分：说服他人的说话技巧

在人际交往中，说服他人是一项高级技能。本节我们学习三个关键技巧。

1.  **让对方觉得想法是他的**
    最高明的说服不是灌输，而是引导。就像苏格拉底的“产婆术”，通过提问让对方自己得出结论。销售案例中，销售员请顾客自己评估旧车价值，从而让顾客主动决定购买。**核心是：提示与引导，而非直接推销。**

2.  **批评前，先自我批评**
    人在被批评时会产生抵触情绪。如果在批评他人之前，先承认自己的不足或错误，对方会更容易接受你的意见。例如：“我这个人可能比较敏感，但你刚才那样说，难免让我觉得有点不舒服。”

3.  **用建议代替命令**
    用商量的口吻提建议，远比用命令的语气更有效。例如，将“你们马上离开会议室”换成“我们的会议也很重要，能否请你们稍微提前一点结束？”，对方通常会更乐意配合。

## 第四部分：当众说话的艺术

很多人恐惧公开演讲。卡耐基提供了几个实用技巧，帮助你更好地当众表达。

1.  **讲述亲身经历**
    自己真实、难忘的经历最容易打动听众，产生代入感。例如，讲述自己如何从悲观变得乐观的亲身故事，比空谈理论更有感染力。

2.  **激发听众好奇心**
    在开头制造悬念，能立刻抓住听众的注意力。例如：“那天晚上我正在家看电视，突然接到一个电话，对方说他是警察局的……”

3.  **精确描述细节**
    笼统的叙述乏味，细节的描述生动。加入时间、场景、动作等细节，能让听众产生画面感。对比“我在家看电视”和“那是一个周六晚上，我正在看《快乐大本营》，笑得前仰后合时，电话突然响了……”。

4.  **自己充满激情**
    演讲者的情绪具有感染力。如果你自己都平淡无奇，又如何打动听众？学习那些充满激情的演讲者（如历史上的著名演说家），调动自己的情绪。

## 第五部分：与人相处的基本原则

说完了“口才”，我们进入本书的第二大模块——“人际关系”。首先，我们来看看与人相处有哪些基本原则。

以下是与人相处的五个基本原则：

1.  **管住自己的嘴**
    如果无好话，宁可沉默。在传播信息前，先问自己三个问题：**此事是否真实？此事是好事还是坏事？此事是否重要？** 问完这三个问题，很多闲话便不必再说。

2.  **微笑，用微笑感染对方**
    微笑是最有感染力的语言。爱笑的人运气不会太差，因为微笑能换来微笑。如果心情不好，可以强迫自己对镜子微笑，或看喜剧节目调节情绪。

3.  **真诚赞美对方**
    每个人都希望自己很重要。真诚的赞美，能让人感受到你的关注。例如，过年时夸赞亲友的新衣、新发型。

4.  **学会倾听，做忠实听众**
    倾听比诉说更重要。即使对方长篇大论，也要耐心听完，并用简单的回应（如“然后呢？”“原来是这样”）表示你在认真听。后续课程会有专门书籍讲解倾听技巧。

5.  **谈论对方感兴趣的话题**
    投其所好是打开话匣子的关键。可以通过尝试多个常见话题（如天气、新闻、星座）来试探对方的兴趣点。之前“应酬”主题的课程中已提供过大量话题参考。

**补充：如何优雅拒绝**
    对于不想或不能接受的请求，可以委婉拒绝。公式是：**`热情回应 + 表达同情 + 明确拒绝`**。例如：“哎呀，真不好意思，我知道你现在很为难。不过这件事确实超出了我的能力范围，很多人找过我，我也没办法，实在抱歉。能帮我肯定帮！”

## 第六部分：家庭和谐相处的艺术

最后，我们将卡耐基的智慧应用于最重要的关系——家庭关系。幸福的婚姻属于那些成熟、善于沟通、有责任感的人。

**给男士的建议：**

1.  **慷慨体贴，重在用心**：体贴不一定是金钱，而是花心思。一份手作礼物、一个记录点滴的电子相册，更能体现真心。
2.  **注意保持形象**：婚后也应注意仪表，维持吸引力。
3.  **体察并认可妻子的付出**：现代女性往往兼顾工作与家庭，更为辛苦。丈夫应主动分担家务，并真诚感谢妻子的贡献。
4.  **避免冷战**：沉默和冷战对感情伤害极大。矛盾尽量不要隔夜，主动沟通解决问题。

**给女士的建议：**

1.  **保持好性情，控制脾气**：暴躁的脾气会消耗感情。温柔远比暴躁更有力量。
2.  **善于倾听**：给伴侣解释的机会，不要用“我不听”来关闭沟通渠道。
3.  **停止唠叨与数落**：男人的自尊心害怕被不断打击和否定。唠叨常源于疲劳或对生活的不满，应通过有效沟通（如请求丈夫分担）或寻求专业帮助来改善，而非一味抱怨。

**家庭禁忌：**
    **切勿将“离婚”或“分手”作为威胁手段。** 这应是感情破裂后的最后底线，而非谈判筹码。动不动就提分手/离婚，会严重伤害感情，可能弄假成真。

## 总结与作业

本节课中，我们一起学习了《卡耐基口才的艺术与人际关系全集》的核心内容。我们从**练好口才的六要素**和**说话高手的六项修炼**入手，掌握了提升表达力的方法；接着学习了**说服他人的技巧**和**当众说话的艺术**；最后，探讨了**与人相处的基本原则**和**维护家庭和谐的智慧**。

**课后作业：**
1.  **演讲练习**：选择一个主题（例如“我的春节”），对着镜子练习演讲。运用本节课技巧：从亲身经历出发，精确描述细节，并尝试在开头制造一点悬念。
2.  **沟通实践**：在未来一周，有意识地运用“避免当面指错”、“让对方多说”、“引导对方说‘是’”中的任意一个技巧，并观察效果。

只有将知识付诸实践，才能真正转化为你的能力。坚持练习，你的沟通与人际关系必将得到改善。
