# 课程007：成年人修炼手册·春节特辑1 - 什么时候该拿别人当孩子看 👶➡️🧑

在本节课中，我们将要学习一个关于人际交往的重要视角：在何种情境下，我们应该将他人视为“孩子”来对待。我们将通过分析万维钢老师提出的问题，探讨这一行为背后的动机、影响以及更成熟的替代方案。

## 概述

本次课程内容源自万维钢老师的提问。他观察到，“他还是个孩子”是一种常见的谅解他人的视角，并由此思考自己何时会被如此对待。我们将深入剖析“将他人当孩子看”这一行为背后的心理动因、潜在后果，以及更值得追求的处世态度。

## 问题引入：万维钢老师的困惑

万维钢老师的问题是：“人到中年有两个孩子的我突然意识到，‘他还是个孩子’是个有意思的视角。这个人做错了事情，考虑到他还是个孩子，我们就会谅解他、同情他、帮助他、甚至惯着他、至少放过他、不跟孩子一般见识。可是我也是个孩子呀。我想问，你在什么情况下当别人是孩子，什么情况下不当别人是孩子呢？”

这个问题触及了人际交往中权力、期待与自我认知的核心。

## 将他人当孩子：一种隐性的权力宣告

在生活中，我极少把成年人当作孩子看待。当你把别人当作孩子时，意味着你要**谅解、同情、帮助、纵容、宽恕**他。这引发了一个根本问题：你这么做的目的，究竟是为了对方，还是为了自己？

我的看法是，这通常是为了自己。人与人之间并不平等，总有人处于强势，有人处于弱势。什么人更容易忍受和原谅他人？当然是**强者**。因此，当一个人把另一个人当作孩子来对待，他其实是用这种方式**宣布了自己的相对强势**，并由此获得了心理上的优势地位。

## 三个关键问题

基于上述分析，我们可以提出三个层层递进的问题。

### 问题一：对方是否知情？

如果对方并不能感知自己正被特殊对待，那么这种“宽容”就是无用功。他不知道自己在被容忍、被原谅。

如果对方知道这一点，那么他就仿佛得到了一根“人生保险绳”，认为自己的行为不会招致惩罚。但他可能不明白，这根保险绳**正在燃烧，是有时效的**。他误以为保护会永远存在，直到保险绳断裂的那一天，他将毫无准备地坠落。

### 问题二：后果由谁承担？

这样的事情屡见不鲜。例如，多少职场新人因为“造型可爱、年幼无知”而获得了超乎寻常的宽容。当年华逝去，他们不再能被当作孩子看待时，处境就变得空前艰难，周围人会表现出前所未有的不宽容。

那么，这种困境有多少是由他们自己造成的，又有多少是由那些最初把他们当成孩子的人造成的呢？

### 问题三：“做好人”就足够了吗？

如果承认把他人当孩子对待，根本上是**指向自我**，目的是“为了自己做个好人”，那么我必须重复一个观点：**仅仅做个好人是不够的**。

活得足够长就会明白，所谓的“好”未必等同于言辞和谐、行动温柔。相反，有时严厉的言辞、粗暴的态度和强硬的行动，更能促进一个人的成长。当然，人人都希望被温和对待，但问题在于：**你值得别人花费那么多时间精力去呵护和引导吗？** 这不是至亲，谁愿意做这样耗神费力的事？

## 从“做好人”到“行慈悲”

如果能超越“做好人”的执念，就不会纠结于是否应该像对待孩子一样宽容他人。这时，对他人的关照更像是一种**慈悲**。

而慈悲是没有偏好的。
*   **公式**：慈悲 ≠ 永远温和；慈悲 = 因时制宜的恰当行动。
*   **俯摸头顶是慈悲**，**当头棒喝同样是慈悲**。

区别在于问题的性质：
*   某些问题可以**和颜悦色地开解**，因为它并非紧急凶险。
*   有些问题则必须**予以严厉惩戒**，因为它一旦泛滥就不可收拾。

无论采取哪种方法，都是慈悲的体现。因为在行动中，一个人不考虑自己的行为在别人眼中是好是坏，**只会考虑应不应该、合不合适**。这种不偏不倚的态度在家庭中很难实现，却常见于职场。因此，有人多年后会感谢当年严格要求自己的上司。

## 自我认知与他人期待的偏差

万维钢老师的问题，可能深层在于“自己应该如何被对待”，或者说“自己如何被理解”。也许他人对他的认知（强者/成年人）和他自我的认知（“我也是个孩子”）之间存在偏差，才产生了这样的感慨。

这引出了另一个话题：**仅仅希望他人是个好人，那也是不够的**。

你可能会问：“为什么他们不包容我？不原谅我？不同情我？不帮助我？” 因为你在他人眼中，**已经是个强者了**。而所谓强者，在他人看来是“身披炮火也能依旧向前”的人。既然你看起来毫发无伤，别人就可能忍不住“掏出弹弓也射上一颗试试”。

人心是柔软而脆弱的。弹弓不能伤人，但其后附着的轻慢与恶意却很容易“伤心”。这种情况在线下或许因面对活生生的人而有所克制，但在网络上，面对抽象的ID或文字，人们很难保持同样的宽容。

## 如何面对外界的评价？

因此，不能寄望于他人都是“好人”，也应打消“所有人都喜欢自己”的妄想。在这个世界上：
*   非常喜欢你的人**很少**。
*   非常讨厌你的人**也很少**。
*   大多数人要么对你**全无感觉**，要么根本**不知道你的存在**。

所以，无论你做什么，赞美和诽谤都不是最重要的。最重要的是**争取中间的大多数**。

我的做法是：
1.  **照单全收所有赞美**，有时甚至亲自修改或自我表扬。
2.  **根本无视诋毁**。

我早已发现一个真相：绝大部分针对个人的评论都**毫无价值**。它们既无真知灼见，也非真正为你考虑。因为做出有价值的评论需要很高的理解力和独立思考能力，而大多数人做不到。人们热爱评论他人，是因为**针对他人，他人才有可能注意到自己**。

既然这更像一个“言辞游戏”，那就开开心心地玩。谁会在过家家时，真的去吃塑料碗里的沙子呢？

## 唯一的例外：赤子之心

如果说有什么例外，我会把别人当作孩子，那就只有一种情况：**对方拥有赤子之心**。

**代码描述例外条件**：
```python
if person.has_childlike_innocence() and \
   person.has_unwavering_curiosity() and \
   person.devotes_wholeheartedly_to_passions() and \
   person.stays_away_from_petty_scheming():
    treat_as_child = True  # 选择包容、支持、理解
else:
    treat_as_child = False # 其他人当不起“孩子”这个名号
```

如果一个人在成年后依然保持纯真，拥有不顾一切的好奇心，愿意为热爱之事毫无保留地投入，并远离复杂的人心算计，那么在我眼中，他依然是个孩子。无论他有多少缺点，我依然会选择包容、支持、理解。这样的人世间罕有，光是目睹其生活姿态，就能让人汲取力量。

至于其他人，其实都当不起“孩子”这个名号。归根结底，我们最多只是“像个孩子”，心里想的是别人能像对待孩子那样对待我们。

## 总结

在本节课中，我们一起学习了关于“将他人当孩子看”的深刻内涵。

我们探讨了这一行为背后隐藏的**权力宣告**属性，分析了它可能带来的**依赖与断裂**的后果。我们认识到，超越简单的“做好人”心态，转向**无偏好的慈悲**和**因时制宜的恰当行动**，是更成熟的处世之道。

同时，我们也调整了对外界评价的期待：不必奢求人人都是“好人”，也无需妄想被所有人喜欢，重要的是**争取沉默的大多数**，并学会智慧地对待赞誉与批评。

最后，我们找到了唯一的例外——对待拥有**赤子之心**的成年人，我们可以且应该给予孩子般的包容与支持。而在大多数情况下，“想要做个好人”和“想要别人做个好人”之间，往往只有燃烧的地狱。真正的成长，在于超越这两极，找到自己坚实而平和的位置。
