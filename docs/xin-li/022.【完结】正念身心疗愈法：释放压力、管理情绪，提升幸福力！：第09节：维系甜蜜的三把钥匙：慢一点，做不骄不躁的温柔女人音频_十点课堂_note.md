# 正念修炼课程 第09节：维系甜蜜关系的三把钥匙 🗝️

在本节课中，我们将学习一种经典的正念练习——正念行走。这个练习能帮助我们重新与身体建立连接，培养对自身及他人的耐心与感恩，并将这份觉察的智慧带入日常生活，从而获得更多的宁静与满足。

---

经常有学员诉说在婚姻中的委屈与无奈。他们羡慕其他夫妻的甜蜜，却只看到自己伴侣的问题，感到日子难以继续，又无法下定决心离开。

在这段亲密关系中，我们习惯将目光聚焦于对方的毛病和缺点，常常将对方对自己的照顾视为理所当然。在意见不合时，我们容易急躁、争吵。久而久之，原本炙热的爱情开始降温，避风港变成了冰窖。

古诗有云：“人生若只如初见”。试想，如果我们能暂时放下源于过往的成见，将对方当作初次相见的朋友，并忆起彼此相爱相惜的缘由，会怎样？

如果我们对待对方和自己时，能有更多的耐心和包容，会怎样？

如果我们在对方付出时，能予以及时的肯定和感激，又会怎样？

实际上，不仅是对待感情，我们对待自己也往往缺乏耐心、好奇和感恩。我们将一呼一吸视为理所当然，只有在感冒鼻塞或使用呼吸机时，才意识到顺畅呼吸的可贵。我们把身体视为理所当然，只有在疼痛不适时，才开始怀念健康的日子。

我们总觉得自己进步得不够快、不够高效、不够成功，因此时常对自己充满苛责与批判。其实，我们可以对自己更好。

## 什么是正念行走？🚶‍♀️

上一节我们探讨了在关系中保持觉察的重要性，本节中我们来看看如何通过正念行走来培养这份内在品质。

今天介绍的正念行走练习，是正念减压课程中的经典练习。它可以让我们以初学者之心，重新建立与身体的连接，体验到身体的不易与生命的宝贵，也让我们学会更加耐心和感恩地对待自己和他人。

你可以将正念行走练习中的体悟，带到日常事务和生活中，让从中获得的智慧为现实生活带来更多宁静与满足。

正念行走与正念观呼吸、身体扫描一样，是与身体连接的另一种方式。区别在于，在这个练习中，我们的注意力并非放在呼吸或身体各个部位，而是来到脚上，感知迈出的每一步，感知脚底与地面的接触。

说到正念行走，不知你是否观察过小朋友学走路的过程。小宝宝从呱呱坠地到会抬头、抬脚、爬行，大约需要半年。而要学会直立行走，又需要半年到一年。纵观人类进化史，从爬行到直立行走，我们大约用了两三百万年。

我们需要如此漫长的时间才能掌控行走，但遗憾的是，我们很容易将其视为理所当然，常常忘了行走本身是多么神奇。似乎只有在身体出现病痛、无法顺利行走时，才会念及可以行走的好。

正念行走有别于日常走路状态的一个重要区别在于：在正念行走练习中，你并没有一个要抵达的目的地。**你的每一步就是目的本身**。你不必想着接下来要去哪，甚至不需要着急迈出下一步，只是不断提醒自己：留在当下这一刻、这一步中。

## 练习前的准备与要点 📋

在开始正念行走练习前，有几个注意事项需要说明。

以下是关于场地、姿势和速度的核心要点：

*   **场地选择**：尽量选择安静的空间，室内或室外均可。最初练习时，有15到20步的直线距离即可。你也可以在瑜伽垫上完成。随着觉察能力加强，你可以慢慢将正念行走带入日常生活，例如在去地铁站或办公室的路上带着觉察行走。
*   **目光与姿势**：练习时，保持目光柔和地凝视前方即可。相信你不必看着脚也会行走，双脚很清楚它们的位置。站立时，感受身体被大地安稳承载。
*   **行走速度**：你可以用任何速度进行练习。但在最初学习时，建议尽量用缓慢的、能让你保有觉知的速度行走。减慢速度可以让你更细致地觉察步行时身体和呼吸的状态，并有机会觉察头脑里的想法。你可能会经常留意到自己想要加速脚步的冲动。

## 开始正念行走练习 🧘‍♂️

接下来，让我们一起开始练习。

首先，找到一处安静的空间。现在，让身体处在站立姿势。留意从走动或坐姿变成站姿的过程中，呼吸的变化。

让注意力来到双脚，感受此时此刻，整个身体和体重被大地安稳地承载着。留意此刻身体的重心位于中心还是偏向某一侧（左、右、前、后）。留意此刻肌肉是放松还是紧张的（脚踝、脚背、脚趾、膝盖、手臂、面部、嘴角）。如果有任何紧张，尝试将其放下。

如果分解步骤有助于保持觉察，可以将每一步行走分解为以下几个部分：

```
微微提脚 -> 缓缓抬脚 -> 向前迈脚 -> 轻轻落脚
```

在每一次提脚、抬脚、迈脚、落脚的过程中，是否能够保持着觉察？允许自己带着觉察向前走出一步，再一步。不必着急下一步该如何迈脚，不必回想上一步走得怎样，只是把注意力放在当下这一步。

如果注意力被身边的声音、味道或画面带走，可以暂停下来，安静站立一会儿，让注意力回到此刻的呼吸上。放慢脚步，等等灵魂。当心神凝聚一些后，可以再次开始动态练习。

留意要完成“迈出一步”这个简单动作，实际上需要调动多少肌肉，需要多少身体部位的全力配合才能完成。无需盯着脚尖，保持目光柔和地平视前方即可。

当你需要转身时，也可以留意身体是如何智慧地从一个方向转到另一个方向，调动了哪些身体部位和肌肉才完成转身。

当你在正念行走时，你的心在此时此地，不被过去的回忆带走，也不被对未来的计划带走。你不需要去到哪里，或成为任何人，只需要此时此地，安住于身心，只是在这一个当下，只是这一步。

可以试想，这一步会把你带入一片净土，那里没有伤害，没有污染，一切都那么圣洁、纯净。那么，你将如何踏入这片净土呢？

在任何时候，当头脑里的想法、情绪或各种故事把你带走了，你都可以再次停下来，邀请自己先全然地抵达，然后再出发。想象一下，双脚就像在亲吻着大地一样。行走本身并非一个手段，**行走本身就是归途**。

在完成每一步行走时，留意身心是否全然地抵达。留意自己在行走时的习惯，例如：

以下是你可以观察的几个方面：

*   手臂摆动的幅度与节奏。
*   迈出脚步时的大小。
*   身体的重心是靠前、靠后还是居中。
*   头部和下巴的位置是低垂还是上扬。
*   背部、头部、腿部肌肉的紧张或放松程度。

在这个过程中，请带着好奇和耐心去探索。
