## 第十八讲：语境变革 🏖️

在本节课中，我们将要学习**语境**如何影响广告创意与文案的调性，以及在新媒体环境下如何调整传播策略。

---

在上一讲中，我们探讨了调性的重要性。本节中，我们来看看调性的感知如何受到**语境**的深刻影响。

**语境**，即语言使用的环境。判断一个人的穿着是否新潮，不仅取决于其装扮和我们的审美，还与他所处的环境以及环境中其他人的穿着密切相关。

例如，在1980年代初的中国北方城市，留长发、穿喇叭裤、戴墨镜的装扮可能引人侧目。但同样的装扮若出现在同时代的纽约街头，则可能被视为复古。这表明，对风格的解读随环境而变化。

广告创意与文案的调性同理。许多高端品牌广告中呈现的奢华感，在今天的某些媒介环境中可能不再令人向往，甚至显得格格不入。这是因为人群、语境和时代都已改变。

---

上一讲结尾我们留下了一个思考题：对调性的解读还与什么有关？我的答案是：**调性还与语境有关**。

更典型的是许多中小品牌。例如，在家装市场常见的小传单上，常采用金色背景、白人模特、华服与酒杯的画面来推销卫浴产品。在二三十年前，这或许显得高端，但在今天，这种“高大上”可能已显得过时甚至“土气”。

调性并非所见即所得。广告中意图传递的调性与受众实际感受到的调性未必相等。这一点我们之前虽未直接点明，但通过前几讲的学习，应该不难理解。

---

当然，许多品牌也在努力跟随时代审美与媒介语境的变化。

例如，十几年前，手机广告为彰显高端，常描绘用户身着西装在商务舱中使用手机。如今，像iPhone、华为等品牌的广告，即使推广高端机型，其展现的用户形象也已截然不同。

我曾用一个比喻来描述这种现象：某些原本高大上的品牌广告，放在当下的媒介环境里，**就像是在海滩上穿貂的人**。貂皮昂贵，但身处人人穿着泳衣嬉戏的海滩，越昂贵的装扮反而显得越突兀。

请大家记住这个比喻。在进行任何公开表达时，都需注意周围环境——你究竟是在衣冠楚楚的精英云集的红毯上，还是在世俗男女欢闹的海滩上。

---

媒介语境为何会变化？这是一个更大的议题。但作为在媒介中发声的我们，必须了解其现状。

近年来，媒介环境正从**公众语境**向**社交语境**乃至**自我语境**转变，在一些新媒体平台上尤为明显。

我们可以用娱乐明星的呈现方式来举例：
*   **公众语境时代**：通过电视、报刊看到光鲜亮丽、姿态正式的明星。
*   **社交语境时代**：微博等平台出现，明星得以更随意地分享日常生活，与大众进行贴近、平等的交流，仿佛邀请大众进入其“客厅”聊天。
*   **自我语境时代**：以抖音为代表的平台，明星和普通人常展现出一种“旁若无人”的状态，进行更私密、个人化甚至“放飞自我”的表演。这更像是展现了在“自家卧室”里的样子。

在这种**自我语境**下，人们展现的往往是内心深处更真实的自我认知。这对于品牌而言是宝贵的：你能在此看到消费者最真实的样子，而**一个人灵魂的真相，远比其外在表象更深刻，也对我们这些试图影响他人看法和感受的人更有价值**。

---

更具体的问题来了：媒介环境变了，原本可以靠“穿貂”彰显不凡，现在到了“海滩上”，我们该怎么办？

请注意，很多品牌在此刻容易自乱阵脚。他们认为，在海滩上就要穿得最少、最无下限才能吸引目光。于是，他们不仅扔掉了“貂”，还可能陷入一种癫狂的姿态。

**必须注意：“穿貂”不好使了，不代表我们要抛弃所有的高大上。海滩上，也并非谁穿得越廉价粗糙就越受关注。没有人喜欢廉价和粗糙，海滩也有海滩的时尚与精致，只是标准不同而已。**

我最近看到一个有趣且准确的说法：现在有些常被认为“low”的东西，其实不是“low”，而是 **“low-fi”**（低保真度），它与 **“high-fi”**（高保真度）相对，强调质感的真实与不过分修饰。

以下是一个对比示例：
*   **High-fi示例**：为女孩拍摄专业写真，需精心化妆、布光、修图。
    ```python
    # 比喻性的代码描述：高保真度制作流程
    def create_high_fi_portrait():
        步骤 = ["专业化妆", "影棚布光", "专业拍摄", "精细修图"]
        return "完美、精致但距离感较强的成片"
    ```
*   **Low-fi示例**：在女孩说笑时，用拍立得随手抓拍。
    ```python
    # 比喻性的代码描述：低保真度捕捉瞬间
    def capture_low_fi_moment():
        条件 = "自然状态下的真实瞬间"
        工具 = "便捷的拍立得相机"
        return "真实、亲近、可信但可能不完美的照片"
    ```

**Low-fi** 的好处在于让人感到真实、亲近、可信。这种调性为广告传播中构建信任、打动人心奠定了良好基础。随手拍的生活照中也有美好画面，这就是“海滩有海滩的审美，low-fi有low-fi的讲究”。

例如，一些抗敏感牙膏广告会聘请真实医生，在其略显杂乱的普通办公室中，以业余的灯光和自然的语气进行讲解。这种故意保留甚至制造的“真实痕迹”与朴实气息，恰恰是广告精心设计出的、最重要的调性元素。

---

今天，我们探讨了在变化的媒介语境下如何把握调性。关于调性的话题，我们就先谈到这里。

希望你通过学习，能更好地规划每一次表达沟通时的调性，乃至你自身的调性。

**本节课总结**：我们一起学习了**语境**（尤其是从公众语境到社交语境、自我语境的演变）如何深刻影响广告调性的感知。关键在于，要识别当前所处的“环境”（是红毯还是海滩），并采用与之匹配的沟通方式（是high-fi的精致还是low-fi的真实），而非简单地抛弃高端或拥抱低俗。
