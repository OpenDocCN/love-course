# 青春期心理课 038：孩子写作业拖拉、干事磨蹭怎么办？⏳

在本节课中，我们将探讨孩子写作业拖拉、做事磨蹭的常见原因，并提供科学、实用的解决方法。我们将从学习能力、注意力、视觉发育及家庭互动等多个角度进行分析，帮助家长理解孩子行为背后的心理与生理因素，从而更有效地引导孩子。

## 了解拖拉磨蹭的原因

上一节我们介绍了课程主题，本节中我们来看看孩子写作业拖拉的具体原因。首先需要观察并理解孩子行为背后的可能因素。

以下是孩子写作业拖拉的几个主要原因：

1.  **学习吃力与能力偏科**：孩子可能因不理解学习内容而感到吃力。例如，部分女生在中学阶段对数学感到困难，部分男生则对需要背诵的英语或语文写作感到吃力。这需要通过测试来评估其学习能力甚至智商，判断是否存在偏科现象。

2.  **运动协调性与逻辑思维**：心理学家研究发现，数学能力的提高取决于运动协调性、动手能力和操作能力。公式可表示为：**数学能力 ∝ 运动协调性**。因此，提高数学成绩和理解力，需要注重全身肌肉协调性及双手配合能力的训练。

3.  **高分低能与能力训练**：如果孩子智商正常甚至偏高，但成绩不佳、写作业费劲，则可能属于“高分低能”。这好比仓库很大但没有门，说明能力通道未打开，需要进行针对性的能力训练。

4.  **注意力不集中**：有些孩子智力正常甚至优秀，但无法保持专注，导致效率低下，写写停停。

5.  **视觉记忆与加工速度慢**：部分孩子专注力尚可，但书写或阅读速度极慢。例如，写字时看一笔写一笔，而不是整体记忆后再书写。这需要训练其视觉记忆和加工速度。

6.  **视觉发育不良**：有些孩子的眼球存在轻微震动现象，导致阅读时跳行、看错偏旁部首或符号（如加号看成减号）。这通常与早期视觉训练不足，以及过度接触电视、电脑等电子产品有关。

7.  **家长过度强调引发逆反**：家长过分强调作业，容易引发青春期孩子的逆反心理。孩子可能会用拖延、不完成作业等方式，来反抗家长最在意的事情。

## 针对性的解决方法

在分析了各种原因之后，我们来看看如何针对性地解决这些问题。关键在于采用科学的方法，耐心引导。

以下是具体的解决策略：

*   **针对学习吃力与偏科**：加强薄弱科目的能力训练。例如，通过家务劳动（如包饺子、擀面条）、摆积木、玩乐高等需要双手配合的活动，来刺激大脑左右半球的信息交换与配合，从而提升逻辑思维能力。
*   **针对“高分低能”**：进行额外的能力训练，打通“智商仓库”的“通道”。例如，加强左右脑协调训练或视觉记忆训练。对于记忆困难，可采用“抄写”法，因为**写一遍 ≈ 念十遍**，能帮助信息更好地进入大脑。
*   **针对注意力与速度慢**：鼓励孩子进行体育锻炼，特别是放学后先运动。这如同“磨刀不误砍柴工”，能提高后续写作业的效率。同时，训练其整体视觉记忆，减少看一笔写一笔的习惯。
*   **针对视觉发育不良**：加强手眼协调的球类运动，如乒乓球、羽毛球、网球、投篮等。这些运动能有效训练视觉追踪和专注力，改善眼球震动问题。
*   **针对逆反心理**：家长应避免过度强调作业。有时可以尝试“反向操作”，例如放学后先请孩子帮忙做家务，孩子反而可能主动提起写作业。关键在于不对孩子最在意的事情施加过大压力。

## 总结与核心要点

本节课中，我们一起学习了孩子写作业拖拉、磨蹭的多方面原因及应对策略。

总结核心要点如下：
*   拖拉磨蹭可能是**学习能力**、**注意力**、**视觉发育**或**心理逆反**等多种因素的综合结果。
*   提升**运动协调性**和**动手能力**是增强逻辑思维与学习能力的基础。
*   对于视觉或记忆问题，针对性的**手眼协调训练**（如球类运动）和**书写记忆训练**至关重要。
*   与青春期孩子沟通时，避免过度施压，有时**“反向激励”** 效果更佳。
*   最终，这要求家长保持**耐心**，给予孩子**真正的关心**，并运用**科学的方法**来提供帮助。
