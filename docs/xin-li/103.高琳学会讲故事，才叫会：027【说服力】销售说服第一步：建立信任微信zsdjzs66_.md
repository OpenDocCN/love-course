# 课程103：高琳学会讲故事，才叫会沟通（完结）🎯 027【说服力】销售说服第一步：建立信任

在本节课中，我们将学习如何通过讲故事在销售过程中建立初步信任。我们将探讨销售的基本阶段，并重点讲解在破冰环节应讲述的故事类型及其目的。

---

销售是一个复杂的过程，无论是线下还是线上形式，大致都可划分为五个阶段。每个阶段都面临独特挑战，需要不同类型的故事来应对。

以下是销售过程的五个主要阶段：

1.  **认识潜在客户**：目标是留下良好印象，避免被对方拒之门外。
2.  **建立客户关系**：目标是增进信任，让客户更了解你及你的公司。
3.  **确定客户需求**：目标是帮助客户意识到自身需求，创造销售机会。
4.  **推销产品与服务**：目标是反复说明为何你的产品能满足客户需求。
5.  **完成交易**：目标是最终促成购买。

需要注意的是，实际销售过程更为复杂，具体在哪个环节讲述何种故事，需视情境和对象灵活调整。

上一节我们概述了销售的全过程，本节中我们来看看最初破冰阶段应如何建立信任。

心理学家指出，人们对于外来说服最初都会怀有戒心。对于陌生人，克服这种怀疑犹如克服地心引力。因此，在建立信任的初期，讲故事是打破对方戒心的有效方式。

在破冰阶段，你可以通过以下几类故事让客户了解你：

*   **关于“你是谁”的故事**：讲述个人生活或背景的趣事，能快速拉近关系，因为人们通常不会轻易相信自己不了解的人。
*   **关于“你的专业能力”的故事**：准备能体现你权威性和经验的故事，让客户感知到你的专业性。但切忌过度吹嘘。
*   **关于“你为何从事此工作”的故事**：分享你工作的内在动力和使命感。这能展现你的底层思维，并让客户感受到你是真心为了帮助他们。

讲述这些故事主要有三个目的：帮助他人更深入地了解你；展现你帮助客户的内在动力而非仅仅为了赚钱；通过有温度的故事建立共情与连接。

这种连接至关重要。它就像输液时扎入静脉的针头，只有建立了连接，后续关于产品的信息才能有效传递给客户并被接受。

---

本节课中我们一起学习了销售说服的第一步：建立信任。我们了解到销售可分为五个阶段，并在最初的破冰环节，重点应通过讲述关于你自己（你是谁、你的专业、你的初心）的故事来与客户建立连接，而不是急于推销产品。只有先成功“销售”自己，让客户愿意倾听，后续的产品介绍才有可能产生效果。
