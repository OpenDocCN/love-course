## 30. 【家庭关系】控制不住对孩子发火，怎么办？ 🔥

在本节课中，我们将要学习“控制不住对孩子发火”这一普遍困扰背后的深层心理动因，分析四种不同人格类型父母的发火原因，并掌握四个实用的解决方法。

---

### 发火的深层心理动因：恼羞成怒

上一节我们探讨了家庭关系中的常见挑战，本节中我们来看看“发火”这一具体行为。控制不住对孩子发火的背后，深层心理动因是**恼羞成怒**。

发火即愤怒。愤怒的原因来源于恼羞成怒，而恼羞成怒背后的情绪则比较复杂。

*   **所谓“恼”**：是因为面对发生的事情，心中产生了无力的感觉、失控的感觉，以及无法接受事实的哀伤的感觉。
*   **所谓“羞”**：是自己觉得这件事太糟糕了。我们有可能被别人责怪，有可能会因为这件事被别人嘲笑。再或者，我们做了一件可能损害别人利益的事，心中出现了愧疚的感觉。

举一个典型的例子。有些父母的工作需要经常出差，孩子常常一个人在家，成为所谓的“留守儿童”。父母在外工作很长时间后回到家，见到孩子非常高兴。但这个热乎劲还没过去，父母可能会问孩子：“你成绩怎么样？在学校有没有听老师的话？”

假如听到成绩不好，或者听到孩子做了坏事，父母顿时变了脸，可能对孩子拳脚相加。这对孩子来说，简直像一次坐过山车的体验。这个时候的父母就是恼羞成怒。因为他们自己心里，对于没能陪伴孩子成长是愧疚的。他们心中期待孩子一切都顺利，可以成长为他们想要的样子。

但是假如孩子没有实现这个期待，他们就会想：“要是我陪在他们身边，我亲自辅导他写作业，他成绩就不会像现在那么糟糕。”但是，他们又做不到，或者说过去没能做到，再或者未来也做不到。这时候，一种愧疚的心情涌上心头，就产生了我们刚才说的那种恼羞成怒的情绪。在愤怒的驱使下，就责怪或打骂自己的孩子。

从关系心理学的角度来说，世界上有两种感觉是最难处理的：一种是内疚感，一种就是愧疚感。父母在那一瞬间就是被这样的情绪淹没了，觉得“我没有尽责，但是我又没有能力去尽这个责”。这变成了父母的一种难处，而且这种难处还没有办法说出来。但是情绪需要宣泄，于是他们就对小孩发火。

所以，当我们控制不住对孩子发火的时候，首先要去感觉一下，我们是不是正在体会刚才说的、恼羞成怒背后的那几种感觉。这是一件非常重要的事。因为只有当你厘清了自己发火背后的真实原因，才能找到解决的办法。

但是，不知道这个原理的人，在发火之后往往想的不是找到原因，而是去把发火这件事情合理化。能找到最好的理由就是：“都是孩子不听话，孩子不好，做得很差，需要管教，所以我才这么生气，所以我才冲他发火。”这其实是认知模式出了问题。这时候，小孩变成了我们宣泄情绪的一个工具，变成了一个承受者。

在极端的情况下，有些父母还会把自己在其他地方受的气，转移到孩子身上。比方说，父母在工作中受了老板的气、受了客户的气和委屈。又或者，每次想到自己挣的钱没办法给家里人提供更好的生活环境，会产生自责。这些情绪产生之后，他们会希望世界上有人能看到他们的情绪，能够体谅他们的情绪。

这时候，假如他回到家里，看到孩子把电视机的声音开得很大，他有可能就走过去，把小孩子给揍了。这在心理学上我们称之为**情绪转移**。换句话说，就是父母把自己其他地方压抑了的情绪，转移到了小孩的身上来发泄。

所以我经常跟我的来访者说：“父母对你发火，有可能是你正在支持他们，而不是你做错了什么。”愤怒这种情绪是可以让人产生力量的，而恼羞成怒的父母，可以借由这种力量，来抚平自己心中那种无力的感觉、失控的感觉。所以有时候真的不是小孩做错了什么，而是他们替父母的情绪背了黑锅。

**结论公式**：
```
发火行为 ≈ 恼羞成怒
恼 = 无力感 + 失控感 + 哀伤
羞 = 愧疚感 + 怕被责怪/嘲笑
最终表现：情绪转移至孩子
```

所以要解决控制不住发火的问题，首先要找到自己发火的真正原因，从根本上去解决这些问题。不要当孩子需要你的时候，除了惭愧，一无所有。

### 四种人格类型的发火原因分析

学习了控制不住对孩子发火背后的深层心理动因之后，我们来分析一下，四种动物类型的人，都是因为哪些具体的原因，而忍不住对孩子发火。

以下是四种人格类型的具体分析：

1.  **蜗牛型**
    蜗牛型的人很容易就会控制不住发火。当蜗牛感觉到无力或者力量不够的时候，他们就会对孩子发火。就像刚才说的，愤怒可以产生力量感，而孩子是一个很安全的对象。他们不可能不爱自己的父母，没有长大成人之前，也不可能离开父母。蜗牛对着孩子发火，一方面会让自己获得力量感，另一方面还很安全，不会因为发火而失去对方。所以蜗牛是经常会对孩子发火的。

2.  **袋鼠型**
    袋鼠型的人对孩子发火的原因，往往是因为孩子没有去认同和肯定袋鼠爸妈。对袋鼠爸妈来说，孩子的一切都是“我”的，都是“我”给予的。当孩子不承认这一切，或者不承认袋鼠爸妈的价值的时候，袋鼠爸妈就会恼羞成怒。
    举个例子，袋鼠跟孩子说：“你看，都是因为你，我好辛苦，我养你容易吗？”这时候孩子说：“所有的父母不都是这么做的吗？爸妈不就是应该这么做吗？你居然把我生下来，就要养我，不然你干嘛要把我生下来呢？”袋鼠爸妈要是听到这个话，会一下子怒不可遏。

3.  **鸵鸟型**
    鸵鸟型的人都很自恋，需要心理资本。孩子也是他们获得心理资本的一种工具。所以鸵鸟对孩子往往都有比较高的期待。当孩子没能满足这种期待的时候，鸵鸟就会忍不住对孩子发火。
    比方说，鸵鸟会跟小孩说：“你是我儿子，你怎么才考这么几分？”有一部电影《如父如子》，其中身为精英的父亲，一直嫌弃自己的孩子弹琴、学习各种不能达成他的期待，常常因此生气。最后，这位爸爸发现儿子不是亲生的，竟然常常地松了一口气，说：“果然不是亲生的。”这就是鸵鸟型爸爸的典型表现。

4.  **斑鸠型**
    斑鸠型的人对待小孩的方式，更像西方的一些父母。斑鸠型的人的情绪隐藏得更深，或者说，他们更加自我。现实的事情对斑鸠来说，都是就事论事，不愿意投入太多的情感。所以，斑鸠会站在相对客观的角度去看待孩子，看待自己和孩子的关系。他们不会想要去为孩子承担更多的责任。
    譬如，斑鸠常常说的一句话是：“孩子啊，现在我养你是我的责任。等你成年之后，你总要找到自己的路。我有我的生活，你有你的生活。”

### 四个解决问题的实用方法

在知道了四种动物类型的人忍不住对孩子发火的具体原因之后，我要给你四个解决问题的方法。

以下是四个核心建议：

1.  **惩罚不是目的，目的是让孩子学会承担结果**
    有一个很有趣的漫画：一个老爸在打自己的儿子，一边打一边说：“叫你不听话。”这个时候，爷爷也在打自己的儿子，一边打一边说：“叫你不要打自己的孩子。”这就是我常常说的：我们对待世界的方式，就是我们曾经被对待的方式。
    那么，假如我们总用一种惩罚性的方式来对待孩子，孩子长大之后，也会继承这样一种方式。在继承的同时，一边觉得很痛苦，一边还会继续延续这样的做法。
    解决这个问题的关键在于，要明确：当你想惩罚小孩的时候，惩罚本身不是目的，让小孩学会承担后果才是这次惩罚的目的。所以，首先要改变大人自己的认知，理解自己的行为。然后才是惩罚的方式：是要用发火、暴跳如雷这种方式来惩罚，还是有其他更好的形式和方法呢？
    比方说，可以设立一个规则，事先告诉孩子，你触犯了底线会有什么样的结果。
    > 我有一个朋友，他跟小孩说：“我给你买的玩具，到了下个月这一天，它们都还是完好无损的，我就再给你买三件玩具。假如还没到这个月的结束，你就弄坏了或者弄丢了它们，我接下来的两个月都不会再给你买任何玩具。”
    这个惩罚的目的就是让小孩知道：假如我做错了事、做了坏事，这是有成本的，我是需要去承担这个结果的。

2.  **偶尔发火是可以的，但是不能变成发泄**
    第一条是关于理性地解决教育小孩的问题。第二条，就是当我们确实已经控制不住自己的情绪了，发火是可以的。假如说这是特殊情况，非常偶然才出现一次，也不需要太过自责。但是，不能把小孩当成宣泄情绪的对象，用暴力去对待孩子。

3.  **反思爸妈是不是反过来正在依赖孩子**
    在冷静下来的时候，思考一下：在我们跟孩子的关系中间，我们是不是反过来正在依赖孩子？依赖他们给予我们的安全感，依赖他们来帮助我们消化掉不良的情绪，依赖他们给予我们更多自恋的资本。我们希望孩子做得更好，是因为我们希望孩子本身变得更好，还是只是在照顾我们的情绪，满足我们自己的需要？

4.  **不要陷入受害者模式**
    我们常常会在社会新闻里看到，有些夫妻关系不和谐、没有被丈夫很好对待的女性，她们对自己孩子的脾气是特别糟糕的，她们把气都撒在孩子身上。这些妈妈把情绪投射到孩子身上，把孩子当成了丈夫的替身，潜意识里把孩子变成了一种“施害者”，而自己成为了“受害者”。于是，她们会用特别暴躁的方式来对待自己的孩子。
    这种深层关系的调整往往需要专业人士的帮助。但是在这里，我可以先教你一个简单的小办法：**发火之前，停顿三秒。**
