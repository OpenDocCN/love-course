# 078.人际交往：想跟任何人都聊得：23N0.23公：教你不伤人的说话

在本节课中，我们将要学习《非暴力沟通》这本书的核心思想。这本书将帮助我们认识到日常沟通中可能存在的“暴力”因素，并学习一种清晰、平和且高效的沟通方式。通过掌握“观察、感受、需要、请求”这四个要素，我们可以显著改善与家人、朋友及同事的关系。

---

## 🚫 什么是暴力沟通？

上一节我们介绍了课程概述，本节中我们来看看什么是“暴力沟通”。我们通常认为“暴力”指身体伤害，但作者指出，语言、表情和肢体等形式展现出的伤害同样属于暴力。这种暴力非常隐蔽，但对精神和情感的伤害不亚于身体暴力。

以下是生活中常见的几句话，它们都属于暴力沟通：
*   丈夫对妻子说：“你能不能快一点打扮呀，这么自私，大家都等着你一个人呢。”
*   妻子对丈夫说：“你成天就知道喝酒，每天都喝成这个鬼样子，喝死在外面算了。”
*   老师说：“这种题做了多少遍了，就是木头也该开窍了吧。”“你一天到晚迟到，要不要我每天叫醒你。”

这些对话听起来熟悉，却充满了指责、贴标签和威胁。暴力沟通的结果是引发对方的防御或反击，将对话引向互相攻击，最终破坏关系。

## 🕊️ 如何进行非暴力沟通？

上一节我们了解了暴力沟通的危害，本节中我们来看看解决方案——非暴力沟通。其核心是一个包含四个要素的沟通模型。

非暴力沟通包含四个要素：**观察、感受、需要、请求**。
沟通的步骤是：首先中立地观察事实，然后表达由此产生的感受，接着说明是何种需要导致了这种感受，最后提出具体的请求。

以下是使用非暴力沟通模式重构前述暴力对话的例子：

**例子1：丈夫对妻子**
*   **暴力沟通**：“你能不能快点打扮啊，这么自私，大家都等着你呢。”
*   **非暴力沟通**：“老婆，你已经打扮了四十几分钟了（**观察**），朋友们都在等着呢。我很着急（**感受**），因为我不想迟到被大家批评（**需要**）。我们六点前到那里好吗（**请求**）？”

**例子2：妻子对丈夫**
*   **暴力沟通**：“你天天就知道喝，每天都喝成这个鬼样子，喝死在外面算了。”
*   **非暴力沟通**：“老公，这周你已经喝了三次酒，有两次过了12点才回来（**观察**）。我感觉很孤独很无助（**感受**），因为我不想夜里经常一个人待着，有你回来我才有安全感（**需要**）。你能不能为了我减少喝酒的次数，并且不要那么晚回来呢（**请求**）？”

通过对比可以看出，非暴力沟通聚焦于事实和自身感受，而非指责对方，从而为解决问题创造了空间。

## ⚙️ 非暴力沟通四要素详解

上一节我们介绍了非暴力沟通的模型，本节中我们将深入探讨这四个要素的具体应用方法。

### 1. 区分观察与评论

非暴力沟通的第一步是陈述观察，而非评论。观察是描述客观事实，评论则夹杂了我们的主观评判。

印度哲学家克里希那穆提曾说：“不带评论的观察是人类智慧的最高形式。”

以下是观察与评论的对比：

| 评论（主观） | 观察（客观） |
| :--- | :--- |
| 他太不负责任了。 | 他这周有三次没按时提交报告。 |
| 你花钱大手大脚。 | 你这个月在外卖上花了2000元。 |
| 他很少陪我。 | 这周他有五个晚上在加班。 |

**小技巧**：
*   注意形容词：少用“懒”、“蠢”、“坏”等带褒贬色彩的词。
*   注意副词：慎用“总是”、“从不”、“经常”等绝对化或模糊的词语。
*   分清事实与推测：避免把“我觉得他就是没听懂”这样的推测当作事实来说。

### 2. 表达感受，而非想法

我们需要学会识别并表达自己的感受，而不是表达想法。感受是关于心情和情绪的，想法则包含了对人或事的判断、评价。

以下是感受与想法的对比：

| 感受 | 想法 |
| :--- | :--- |
| 我感到很伤心。 | 我觉得你不在乎我。 |
| 我很兴奋。 | 我觉得这个主意棒极了。 |
| 我有些焦虑。 | 我觉得这次汇报要搞砸了。 |

建立自己的“感受词汇表”（如：开心、伤心、害怕、气愤、沮丧、孤独、平静等），有助于更精准地表达内心状态。

### 3. 体会感受背后的需要

我们的感受源于自身需要是否得到满足。非暴力沟通鼓励我们直接说出需要，而不是通过指责他人来间接表达。

**公式**：**我感到……（感受），是因为我需要/看重……（需要）**。

*   **指责（无效）**：“你从来不关心这个家！”（隐含需要：我需要被关心和重视）
*   **表达需要（有效）**：“你最近很晚回家，我感到有些孤单（**感受**），因为我需要家人的陪伴和互动（**需要**）。”

### 4. 提出具体、明确的请求

在表达观察、感受和需要之后，我们需要提出具体的请求，告诉对方我们希望他做什么，而不是不做什么。请求要清晰、可行，并使用请求的语气，而非命令。

*   **模糊/命令（无效）**：“你对我好一点。”“不要乱扔外套！”
*   **具体/请求（有效）**：“我希望周末我们能有一个小时的独处时间，一起看场电影或散散步（**具体请求**）。”“请把外套挂到门口的衣架上（**具体请求**），好吗（**请求语气**）？”

## 👂 如何倾听：应对他人的暴力沟通

上一节我们学习了作为说话者如何表达，本节中我们来看看作为倾听者，当对方使用暴力沟通时，我们该如何应对。

非暴力沟通的另一面是“全身心地倾听”。当他人情绪激动或指责我们时，我们的目标不是反驳或辩解，而是去倾听他们话语背后的**观察、感受、需要和请求**。

常见的错误倾听方式（阻碍沟通）包括：建议、比较、说教、安慰、回忆、否定、同情、询问、辩解、纠正。例如：“这有什么，我经历过更糟的。”（比较）“别难过了，想开点。”（安慰）

正确的倾听方式是关注对方的情感和需要。你可以通过询问来确认你的理解：
“你看起来很生气（**感受**），是因为你希望这件事能得到公平处理（**需要**），是吗？”
“你提到方案被否定了（**观察**），现在感到有些沮丧（**感受**），是希望自己的努力被看到（**需要**）吗？”

这种倾听能让他人感到被理解和尊重，从而为真正的对话打开大门。

## 🙏 表达感激与课程总结

### 表达感激

非暴力沟通也适用于表达感激。真诚的感激能强化积极关系。表达感激的三步是：
1.  对方做了什么事（观察）。
2.  我们的哪些需要因此得到满足（需要）。
3.  我们的心情如何（感受）。

**例子**：“谢谢你昨晚帮我修改报告（**观察**）。这让我能准时提交，保住了项目进度（**需要满足**），我心里特别踏实和感激（**感受**）。”

### 课程总结

本节课中我们一起学习了《非暴力沟通》的核心方法。我们认识到，暴力沟通不仅限于打骂，日常的指责、嘲讽、贴标签都是一种伤害。通过练习“观察、感受、需要、请求”四要素，我们可以：
*   **更清晰地表达自己**，减少误解。
*   **更深入地倾听他人**，理解其真实需求。
*   **将沟通导向解决问题**，而非互相指责。

记住，沟通的目标是建立连接，而非赢得辩论。从今天起，尝试在关键对话中运用这一方法，你的人际关系可能会迎来意想不到的改善。
