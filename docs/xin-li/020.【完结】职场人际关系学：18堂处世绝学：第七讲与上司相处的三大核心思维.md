# 职场人际关系学 第七讲：与上司相处的三大核心思维 🧠

在本节课中，我们将学习与上司相处的三大核心思维。掌握这些思维，有助于你与领导建立更牢固、更信任的工作关系，为你的职业发展奠定坚实基础。

在职场中，良好的职业发展前景离不开与主管领导的稳固关系。获得领导的信任，才能争取到更多资源和机会。然而，处理好与领导的关系，需要遵循一些基本原则，特别是接下来要介绍的三大核心思维。

这三大思维是与领导处理关系的最高准则，也是经过验证的经典经验。运用好这些策略，能最大限度地加强你与领导之间的联结，确保获得信任。

## 第一核心思维：不分

上一节我们提到了建立稳固关系的重要性，本节中我们来看看第一个核心思维：“不分”。这是一种重要的思维方式，意指你与领导之间，尤其是在职责划分上，不要分得过于清楚。

很多人喜欢明确划分职责范围，对于领导安排的非职责内工作感到不快甚至产生抵触。必须记住，无论你的角色、岗位或能力如何，都不要对抗领导。

以下是关于“分内”与“分外”工作的说明：
*   **分内工作**：指工作的主要责任与义务，通常是必须完成的任务，可能有明确的指标和操作流程。
*   **分外工作**：指所有职责以外的区域，没有强制完成的任务，更多依靠个人主动性、兴趣或理想。

只完成分内工作无可厚非。但若有余力和意愿，承担一些分外工作，尤其是领导交办的分外之事，则是能力的体现。这既能提供便利，也可能带来额外价值。

关键在于，不要对领导交办的任务过于区分分内与分外。顺应领导的工作安排，并真正做好，才能赢得领导的完全信任，你的前途才会更加确定。否则，领导可能会边缘化你。请认清：上级决定你的前途，而对领导而言，你是实现目标的工具。我们的定位应是更好地辅助领导达成目标，在此过程中，获得信任至关重要。

**核心要点**：不要对领导交办的工作过分区分分内与分外。只要有余力，就尽量去做。做得越多，越容易成为“自己人”。（注意：这里主要指领导交办的任务，而非同事推诿的所有工作。）

## 第二核心思维：接受

理解了“不分”的思维后，我们来看第二个核心思维：“接受”。高效的执行力源于认同与服从，这也是与领导相处的重要思维方式。

许多人喜欢将自己的想法与领导的想法对比。一旦发现不一致，便认为领导是错的，进而产生抵触或违背领导意图的行为或情绪。这在职场是大忌。

无论你认为领导水平如何，他的组织位阶在你之上。你必须无条件地先接受这位领导及其指令。这是建立密切或正常工作关系的基础。

**公式表达**：**良好上下级关系 ≈ 接受 + 服从**

没有接受与服从，领导会感到你在工作中对抗他、带有负面情绪。这会极大损害你们的关系，让领导认为你工作态度不端正，其他方面就更无从谈起了。

## 第三核心思维：大智若愚

在学会了“接受”领导之后，我们来看最后一个核心思维：“大智若愚”。这是一种与领导相处时应保持的姿态：永远不要显得比领导更聪明。

对领导的忠诚，应体现在你是一个踏实做事、实在本分的人，而非一个爱耍小聪明的人。真正有智慧的人，往往给人感觉实在、甚至有些“愚钝”。

不要认为天下人都不如你聪明。大家的智商情商相差不大，很多时候只是别人不与你计较。宁可显得“傻”一点，也不要天天卖弄小聪明。耍小聪明是与领导相处的大忌。

**代码描述此心态**：
```plaintext
if (发现领导决策或有疏漏):
    谨慎私下沟通 or 踏实执行并准备预案
else:
    坚决执行 // 而非公开炫耀自己更“聪明”
```

例如，有员工迟到，诚实告知“睡过头了”比编造借口更好。又例如，有销售主管将公司主动上门的客户（提成较低）谎称为自己开发的客户（提成较高），以此谋利。这种耍小聪明的行为，领导迟早会察觉，并产生反感，最终断送个人前途。

---

**本节课总结**：

本节课我们一起学习了与上司相处的三大核心思维：
1.  **不分**：不对领导交办的任务过分区分分内分外，积极承担以赢得信任。
2.  **接受**：无条件先接受领导及其指令，认同与服从是高效执行和良好关系的基础。
3.  **大智若愚**：保持踏实、实在的姿态，避免炫耀聪明，更不要耍小聪明。

掌握这三大思维，能帮助你更稳健地处理与上级的关系，为职场发展铺平道路。
