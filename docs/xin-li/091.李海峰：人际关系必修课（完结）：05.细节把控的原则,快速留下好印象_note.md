# 课程091：人际关系必修课 - 05. 细节把控的原则：快速留下好印象 👥✨

在本节课中，我们将要学习如何与C型（思考型）行为风格的人有效相处。我们将深入探讨C型人的核心特质、与他们沟通时可能遇到的挑战，并掌握与之相处的具体原则和方法，从而快速留下好印象，建立优质的人际关系。

上一堂课，我们深入了解了如何与S型（稳健型）风格的人相处。本集，我们要和大家分享最后一种行为风格：C型风格。

## C型风格的核心特质 🧠

对比数量来讲，C型人更在意的是质量。D型人喜欢用眼睛去盯目标，I型人喜欢用嘴巴来说话，S型人喜欢用耳朵去倾听，**C型人喜欢用大脑来思考**。

鲁迅先生曾经说过：“当我沉默时，我感到充实；我将开口，同时感到空虚。” 这句话精准地描绘了C型人内省和深思的特质。

在面对新的选择时，I型风格往往会思考“怎么做更多”来推动事情，因为他们天生乐观。而C型风格则会思考很多“可能做不成的原因”。I型人看到问题背后的机会，**C型人看到机会后面的问题**。I发明了刀，C发明了盾；I发明了轮船，C发明了救生圈；I发明了飞机，C发明了降落伞。任何风格没有好坏，都有其价值。

## 与C型人相处的价值与挑战 ⚖️

很多时候我们会觉得，C风格用得比较多的思考型的人，考虑问题周全、成熟稳重，同时思考深入、追求完美，是很好的搭档和伙伴，他们给这个世界带来了智慧。

然而，和C型人相处问题也不少，比如敏感多疑、固执、偏执等等。我们一起来看一看吧。

需要帮助的时候，思考型的伙伴是你的智囊团，能专业严谨地提醒你忽略的地方。他们关注事情，但速度可能很慢。那如何才能好好地跟他们相处呢？我们来看看他们的具体特质。

## C型人的行为表现 📝

以下是C型人常见的具体行为表现：

*   **持之以恒，坚持到底**：思考型的人关注专注和坚持的力量。遇到喜欢的事，C风格会用“一根筋”坚持到底。他们思考深入，对认定的事情非常执着。
*   **情感细腻，润物无声**：在情感互动中，C型人的感动之处在于，很多事情你不记得了，他还记得。如果他们想关心一个人，会试图了解你、洞察你，为你做你需要的事情。他们不屑于用语言来表达内心的情感，不轻易做流于表面的赞美。
*   **追求完美，迷恋细节**：他们常常对自己跟别人要求都很高。对别人要求高不容易宽容，对自己要求高缺乏弹性。我们周围的C特质高的人，他们对标点符号错误使用、页面元素是否对齐能够轻易发现问题，并且不厌其烦地改正。
*   **习惯用数据说话**：对于他们来讲，学西餐的制作（盐**3克**）要好过学中餐的制作（盐**少许**）。这样C型人更容易掌握。
*   **需要与人保持距离**：保持距离能显示他们看待事物时的独立性与专业性。与人交往他们崇尚的是“君子之交淡如水”。他们更希望的是“我不说你也知道”的默契。
*   **思虑周详，沉稳成熟**：思考型的人说话少、想得多，不苟言笑，显得很深沉。不率先发表意见，自然就有时间去思考如何回应可以更好。C型的风格给人**谨慎感**。
*   **一诺千金，诚实守信**：由于思考型的人求质不求量，在人际关系中注重深度，因此他们的友谊有时会长达一生。他们高度重视承诺。
*   **计划周密，按章办事**：思考型的人的想法是，要么不做，要做就做最好。他们做任何事情时，首先要制定好计划，然后严格执行。如果要改变，必须给他一个他能够接受的理由。

## C型人的核心关注点 🎯

C型风格用得多的思考型的人，可能不像D那样追求效率，不像I那样追求有趣，不像S那样追求安全感。那么他们关注的是什么呢？

*   **他们关注数据**：只有通过科学、详实、严谨的数据，才能够打消他们内心的疑惑，提升他们心中的信任度。
*   **他们追求完美**：他们不满足于优秀，希望自己做到卓越。
*   **他们重视制度、程序、规范和流程**。

## 优势过当带来的困扰 😟

一种行为风格用得好，对我们的工作生活帮助很大。但有时优势发挥过大，也会带来困扰。

以下是C型特质过度发挥时可能带来的问题：

*   **很难接近，不太容易快乐**：C对别人的要求高，所以不太容易亲近；对自己要求高，所以不太容易快乐。正所谓“水至清则无鱼，人至察则无徒”。
*   **容易消极悲观**：杞人忧天，天空划过流星，I型人开心得不得了要赶紧许愿，而C型人却感叹时光稍纵即逝。
*   **对自己的不完美放不下**：追求完美总是在反省自己，无形中也给自己巨大的压力。对别人也很容易走极端。
*   **沉溺往事，敏感多疑**：C型人很可能一直活在历史和过去中，无法摆脱阴影，长期无法走出低谷。
*   **死板固执，缺乏幽默**：他们凡事都要讲得明白，一旦进入他们的思维模式，他们是相信规则、数字和流程的。他们不太懂得释放情感，没有什么幽默感。

## 与C型人相处的四个妙招 🛠️

我们明白了思考型的人看重什么，就能更好地跟他们沟通了。核心是要让他们觉得我们为人处事，和他们一样严谨、有品味。

以下是四个与C型人相处的核心方法：

*   **与思考型的人保持必要的距离**：思考型的人自我保护意识比较强，要让他们在安全距离里，从容评估你的“危险程度”和事情的风险。他们喜欢独处，如果可以给他一个单独的办公室或隔离出来的空间，他会效率更高。
*   **用数据说话**：在向C型人汇报工作时，C会习惯性质疑。避免使用“很多”、“强烈”等形容词。例如，汇报“遭到很多客户的强烈投诉”时，C型领导会问：1. 谁反应的？（评估可信度）2. 有多少人投诉？（看比例）3. “强烈”是几级？**只有准确的数据才能加分**。
*   **让C先表达**：向C型人推销或征求意见，最佳选择是引导他先说出自己的想法。C的脑中有强大的逻辑思考能力，善于“鸡蛋里挑骨头”。你说得太多，他们容易进入挑错模式。不如让他先说，根据他的观点补充完善，掌握主动权。
*   **给他一些个性化的奖励**：C型人不喜欢泛泛的表扬。你可以给他高品位的奖品（如古典音乐会门票），他会认为那是你对他品位的赞赏。另外，C看重数据，可以给他一些可量化的奖品、高科技产品，或是一些继续成长的通道（如国际课程认证）。

## 课程总结 📚

本节课中我们一起学习了如何与C型（思考型）行为风格的人相处。

1.  我们了解了和思考型的人相处的诸多痛点。
2.  我们分析了思考型的人常见的表现有哪些。
3.  我们知道了思考型的人关注的要点（数据、完美、流程）。
4.  我们学习了与思考型的人相处的四个方法（保持距离、用数据说话、让其先表达、给予个性化奖励）。

用好他们，你的世界将充满智慧。
