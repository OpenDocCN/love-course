# 课程090：《六本书教你受益终身的思维方式》04 - 理性的尺度在哪里 🧠

在本节课中，我们将要学习“理性”这一概念的不同维度，并探讨人类思维中“认知成见”的根源与意义。我们将从宏观的理性观层面，分析为何看似“非理性”的思维模式，实则是进化赋予我们的高效生存工具。

上一节我们介绍了“何曲谬误”的问题，本节中我们将站在更宏观的角度，讨论如何从理性观的层面来看待我们的种种推理谬误。

“理性”是日常生活中经常使用的词，但其含义很少被仔细审视。根据德国心理学家吉仁泽（Gerd Gigerenzer）的观点，我们平常所说的理性可以分成三种：**全能理性**、**有限理性**和**节减理性**。

以下是三种理性观的详细说明：

**1. 全能理性**
全能理性指一种无所不知、无所不能的理性观。其思想原型来源于近代法国科学家兼思想家拉普拉斯提出的“拉普拉斯精灵”。拉普拉斯认为，如果有一个全知的精灵能知道宇宙中某一刻所有粒子的初始状态和所有科学规律，它就能预测出未来任何时刻任何粒子的运动状态。这是一种在牛顿绝对时空观下，试图用理性完全控制世界的想法。

这种观点在现代的大数据思维中仍有体现。例如，有人认为21世纪最重要的资源是数据，通过搜集海量数据并运用大数据机制，就可以预测个人的消费行为、阅读偏好甚至择偶标准，从而实现“精准投放”。这种观点实质上是全能理性观的大数据版本。

**2. 有限理性**
有限理性观由美国学者赫伯特·西蒙（Herbert Simon，中文名司马贺）提出。他因这一理论获得了图灵奖和诺贝尔经济学奖。该模型的核心思想是：按照全能理性的要求去设计一个智能系统是不现实的，因为人类无法处理全部信息。

我们可以用“择偶”的例子来理解有限理性：全能理性的方式是搜集所有潜在结婚对象的信息，然后选出最优者（Mr. Right）。而有限理性的方式是设定一个“及格线”，在候选人中，**选择第一个达到及格线的人**，而不是等待可能出现的更高分者，以避免浪费时间和机会成本。

其核心思想可简化为一个决策公式：`决策 = 设定阈值 + 选择首个达标者`。虽然背后的数学模型复杂，但基本逻辑就是“满意即可”，而非“追求最优”。

**3. 节减理性**
吉仁泽认为，有限理性仍然需要复杂的资源（如设定和衡量标准）。他提出了更进一步的“节减理性”，即用最简单、最省力的方式做决策。其最重要的体现是“生态学理性”。

生态学理性与进化论背景相关，指生物体根据环境线索做出快速、有效的决策。例如，老鼠的简单推理规则是：如果一只同类因吃某种食物而死，全体老鼠就会回避这种食物。它们无需进行化学分析，仅凭“谁倒霉了就不跟从”这条简单规则就能规避风险。

为了说明节减理性的有效性，我们来看一个心理学实验。

以下是该实验的具体内容：

*   **实验设计**：让德国学生和土耳其学生判断哪支欧洲足球队更强（例如，米兰 vs. 佩鲁贾）。
*   **结果**：对欧洲足球更熟悉的德国学生得分反而低于土耳其学生。
*   **原因分析**：德国学生试图运用“全能理性”，回忆具体的联赛战绩等复杂信息，反而因信息过载导致判断不准。土耳其学生则运用“生态学理性”，采用更简单的启发式规则：**根据城市知名度（口碑）来判断**。他们认为米兰比不知名的佩鲁贾更有名，因此米兰的球队更强。这种基于“口碑”的判断，类似于老鼠间传递危险信息的机制，在信息有限时是一种高效策略。

这个案例揭示了一个有趣的心理学现象：“多气少”——知道得非常多，判断反而不精准。因为参数过多时，你难以区分哪些更重要，从而陷入信息迷阵。完全无知也无法做出判断。成绩最高的是那些知道一些常识、但信息不精准，并能运用生态学理性（如依赖口碑）的人。

除了足球实验，另一个经典案例“华生选择任务”（又称四卡问题）也能说明节减理性的特点。

以下是该任务的原始版本：

*   **任务描述**：桌上有四张卡片，每张卡片一面是文字，一面是图案。规则假设是：“凡患有受迫害妄想症的人，所画人脸的眼睛会特别大。”
*   **四张卡片**：
    1.  文字面：“本病患患有被迫害妄想症”。（需翻看图案面验证眼睛大小）
    2.  文字面：“本病患未患有被迫害妄想症”。（与规则验证无关）
    3.  图案面：画有夸张的大眼睛。（需翻看文字面验证是否患病）
    4.  图案面：画有正常尺寸的眼睛。（需翻看文字面验证是否患病——这是关键的反例证据）
*   **常见错误**：大多数被试只翻看第1张和第3张卡片，试图寻找正面证据，而忽略了翻看第4张卡片（正常眼睛）以寻找反例。从逻辑学看，验证一个规律，**寻找反例（证伪）** 至关重要。

那么，为什么人们会忽略逻辑上关键的反例证据呢？吉仁泽认为，这是因为原始问题过于抽象，与个人利益无关。如果我们将问题情境化，结果就不同了。

以下是社会契约版本的四卡问题：

*   **新情境**：山中有为迷路猎人提供应急物资的规则，猎人用完后有义务补充。
*   **验证规则**：人们不仅会检查“用了物资的人是否补充了”（正面证据），也会非常关注“有人上山却没带物资”（反面证据，即违背规则的行为）。
*   **原因解释**：当问题涉及**社会契约**和**自身利益**时，人脑会激活一种“**骗子侦测机制**”。我们变得特别敏感于谁可能违背契约、搭便车，因为这会损害集体及个人利益。而在抽象的原始问题中，这种与利益相关的思维机制未被激活，因此我们在逻辑上就显得“愚蠢”。

这证明了节减理性的核心：**我们的理性大多数时候是“节能”的，只在与切身利益直接相关时，才会开足马力进行复杂思考**。这就像猎豹，虽能高速奔跑，但只在捕猎时才全力出击。

本节课中我们一起学习了三种理性观。只有站在“节减理性”的角度，所谓推理谬误的产生才能得到很好解释。我们通过“猜球队强弱”和“四卡问题”两个案例说明：除非与个人利益（如预存的“骗子侦测机制”）关联，否则人脑很难严格按照逻辑学要求运作，因为那样不节能且不能带来切实的生物学利益。

有人会问，那职业的逻辑学家和数学家呢？生物学对此有更深解释：他们的思考往往源于小团体内的“生态位”竞争和荣誉感，这本身符合广义的进化原则。但这种专业的理性思考是特定领域的，一旦超出专业范围（如牛顿炒股亏损），人性的普遍弱点同样会暴露。

最后，关于“认知成见”，我们需要重新评估。成见是进化赋予的思维禀赋，像水的重力一样无法根本消除。我们所说的“克服成见”，并非消灭它，而是利用我们思维工具箱里不同的“成见”相互制衡。例如，用对利益的关心激发的“骗子侦测机制”，来平衡可能因第一印象好而产生的过度信任。

心理学在企业管理中可能比纯粹的逻辑学更有用，因为它更贴近人性。同时，在一个组织内，让不同利益团体都有发言权，有助于看清事情全貌。

成见本身是人类宝贵的财富，丰富的不同成见相互制衡，才能帮助我们逼近世界的真相。我们都在“盲人摸象”，关键是要让不同人摸到的信息能在一个平台上交流。今天“赢家通吃”的大数据技术，有消灭文化多样性和思维多样性的风险，这就像单一作物种植在生物学上风险巨大一样。维护人类成见与文明生态的多样性，对长远发展至关重要。

本节课中，我们一起学习了从全能理性、有限理性到节减理性的演变，理解了认知成见作为高效思维工具的进化根源，并认识到维护思维多样性的重要意义。
