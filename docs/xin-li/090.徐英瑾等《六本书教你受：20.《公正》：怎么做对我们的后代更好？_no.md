# 课程090：公正——如何为后代做出更好的选择？🤔

在本节课中，我们将学习《公正》一书的核心思想，特别是关于“共同善进路”的分配公正理论。我们将探讨如何超越功利主义和自由至上主义，从培养德性和对后代负责的角度来思考公正问题。

上一节我们介绍了自由至上主义与罗尔斯差异理论的局限，发现它们对“自由”和“应得”的解释存在争议。本节中，我们来看看本书提出的最后一种解决公正问题的思路。

## 共同善进路的核心思想

在共同善进路上，我们评判一个决定或分配是否公正时，主要考虑它**是否能培养人们的德性**，以及**对后世的影响是积极还是消极**。这与我们通常认为的“公正即给予每个人应得之物”有所不同，因为它追问：一个人“应得”的标准究竟是什么？

这个思路源自古希腊哲学家亚里士多德。他的政治哲学包含两大核心观点：

1.  **公正是目的论的**：为了决定某物的正当分配方式，我们必须探究该**被分配之物的目的或本意**。
2.  **公正是荣誉性的**：为了探究某行为的目的，我们必须讨论该行为应当**尊敬或奖励何种德性**。

## 案例分析：最好的长笛应该给谁？

为了理解亚里士多德的理论，我们来看一个经典案例：一个长笛合奏团获得了一把全世界最好的长笛。乐团成员水平不一，从80分到98分不等。这把长笛应该分配给谁？

以下是不同理论的分配逻辑：

*   **功利主义/差异理论**：应分配给水平最差的乐手。因为这样可以最大程度提升整体水平（短板变强），实现集体利益最大化。公式可简化为：`Max(团队总分) = 优先提升最低分`。
*   **亚里士多德的目的论**：应分配给水平最好的乐手。因为**最好长笛存在的目的（Telos）是奏出最美妙的音乐**，而最好的乐手最有可能实现这一目的。这是公正目的性的体现。
*   **亚里士多德的荣誉性**：将最好的工具奖励给最优秀的人，可以**激励所有人追求卓越**，塑造积极的团队氛围。这是公正荣誉性的体现。

因此，按照共同善进路，分配这把长笛不仅是为了当下演奏，更是为了**塑造一个鼓励卓越、面向未来的良好环境**。

## 现实应用：拉拉队队员的资格

亚里士多德古老的理论在现代社会是否适用？我们来看一个真实案例：一位名叫考利·斯马特的中学生拉拉队长，因患大脑瘫痪症需坐轮椅，无法完成劈叉、翻跟头等常规考核动作。学校以其未通过考试为由，要求她退出拉拉队。

如何公正地裁决？运用共同善进路分析：

1.  **思考目的**：拉拉队队员动作是**手段**，其**目的**是鼓舞士气、让观众振奋。
2.  **评估德性**：考利虽然无法完成常规动作，但她在轮椅上欢呼、微笑，同样极大地激励了观众，实现了拉拉队的核心目的，甚至更具感染力。
3.  **考量对后代的影响**：允许她留在队伍中，能彰显学校“包容、不放弃任何人”的价值观，对所有学生产生积极深远的影响。

因此，公正的决定是让她留下。这个决定关注的是**行为的目的、所表彰的德性以及对社群未来的塑造**。

## 对共同善进路的质疑与辩护

有人可能质疑：我们无法预知未来，所谓“目的”和“榜样”是否会被权威者定义，从而形成道德绑架？

*   **自由派的质疑（如罗尔斯）**：主张国家应保持中立，提供让公民自由选择价值观的框架，避免卷入具体的道德争论。
*   **共同善的辩护（如奥巴马的观点）**：在公共讨论中（如堕胎、人工智能等议题），**完全避开道德评判是不可能的**。长期回避价值讨论，反而会导致社会的虚无感和意义缺失。因此，需要一种更包容的、“对信仰友好”的公共理性。

那么，什么价值观值得鼓励？作者没有给出终极答案，但提供了一个重要的思考方向。

## 超越GDP：衡量什么才是重要的

作者引用了罗伯特·肯尼迪的演讲来阐明观点：

> 我们的国民生产总值（GDP）包括了空气污染和香烟广告，包括了清理车祸的救护费用……然而，它**没有考虑到孩子的健康、教育质量、玩耍的乐趣，也没有包含诗歌之美、婚姻的力量、公共争论的智慧**……它衡量一切，除了那些使生活富有意义的事物。

这促使我们思考：除了经济指标，**公民的幸福感和生活的意义感**是否也应成为衡量社会发展的核心？这正体现了共同善进路对“良善生活”本质的关切。

## 总结与启示

本节课中，我们一起学习了评判公正的三种进路：

1.  **功利主义进路**：公正意味着使**福利或幸福最大化**。`Max(∑Utility)`
2.  **自由至上主义进路**：公正意味着**尊重个人选择的自由**（包括市场自由）。
3.  **共同善进路**：公正意味着**培养德性**和**推行为社群未来负责的共同善**。

本书作者桑德尔教授倾向于第三种进路。理解这些标准，尤其是共同善进路，能帮助我们在面对职场规则、社会现象或司法判决时，进行更客观、更深思熟虑的分析，减少主观情绪干扰。

**最后，让我们用共同善的视角审视两个机构：**

*   **大学的目的**：不仅是传授知识或技能，更是**传递并让学生浸润于一种价值观**（如校训所载），并影响其一生及其将来所影响的他人。
*   **法院的目的**：在裁决新颖案件时，法官常会“考虑到对未来同类案件的影响”。例如，判定误诊医生承担法律责任，旨在**推动整个医疗行业更谨慎负责**，促进长远的共同善。

学习至此，你可能感到豁然开朗，也可能更加困惑，因为它挑战了我们许多习以为常的简单判断。这很正常。重要的是，**我们已经获得了一个更丰富、更关注未来与德性的视角**。剩下的，就交给时间与实践的智慧去沉淀吧。
