# 032.情感心理：第04章：沟通分析理论--：04.核心概念2：理解婚姻中的和谐

在本节课中，我们将要学习沟通分析理论（TA）中关于“沟通过程”的核心概念。我们将明确沟通的定义，并详细探讨三种不同类型的沟通模式：互补沟通、交错沟通和隐藏沟通。理解这些模式有助于我们分析婚姻及人际关系中和谐与冲突的根源。

## 沟通的定义

上一节我们介绍了“自我状态”的概念，本节中我们来看看人与人之间具体的互动过程，即“沟通过程”。

在TA理论中，我们对沟通有非常明确的定义。**沟通是指信息的一来一往**。具体来说，当我发出一个**刺激**，你做出了一个**回应**，我们就完成了一次沟通。你的回应可能成为下一轮沟通中的新刺激，如此循环往复。

因此，在TA的视角下，我们以“一个刺激对应一个回应”为单位，逐句分析沟通过程。这种分析方式非常清晰，例如在国际沟通分析师考试中，考生需要提交逐字稿，考官会严格审查每一句对话中刺激与回应的匹配情况。

## 沟通的三种类型

当我们逐句审视沟通过程时，可以发现存在三种基本类型的沟通。

以下是三种沟通类型的详细介绍：

### 1. 互补沟通（平行沟通）

在互补沟通中，当我说话时，我期望指向你的某种特定自我状态，而你恰好用我所期望的自我状态做出了回应。在沟通图示中，这两条线是平行的。

**公式表示：刺激（从自我状态A指向对方的自我状态B） -> 回应（对方从自我状态B做出）**

这种沟通是顺畅且配合良好的。以下是几种常见的互补沟通示例：

*   **父母状态（P）<-> 儿童状态（C）**：一方从儿童状态（C）发出“我累死了”的刺激，指向对方的父母状态（P）。若对方从父母状态（P）回应“来，我给你捏捏吧”，则沟通互补。
*   **成人状态（A）<-> 成人状态（A）**：一方从成人状态（A）问“今天我们几点出门？”，指向对方的成人状态（A）。若对方从成人状态（A）回答“我们九点出门”，则沟通互补。
*   **父母状态（P）<-> 父母状态（P）**：一方从父母状态（P）说“现在社会越来越乱了”，指向对方的父母状态（P）。若对方从父母状态（P）回应“是呀，跟我们那会没法比了”，则沟通互补。
*   **儿童状态（C）<-> 儿童状态（C）**：一方从快乐的儿童状态（C）说“哎，我今天发了1000块钱！”，指向对方的儿童状态（C）。若对方从儿童状态（C）回应“那咱们吃点什么去呀？”，则沟通互补。

**互补沟通的原则是：只要双方保持互补，对话就可以无限持续下去，直到任何一方主动停止。** 因此，这是一种和谐、无冲突的沟通模式。

### 2. 交错沟通

交错沟通是指，我发出的刺激指向你的一种自我状态，但你却用另一种我未预期的自我状态回应了我。这导致了沟通线路的“交错”。

**公式表示：刺激（从自我状态A指向对方的自我状态B） -> 回应（对方从自我状态C做出，且C ≠ B）**

这种沟通往往会导致谈话中断或转向。以下是交错沟通的示例：

*   **期望成人（A），得到父母（P）**：妻子问：“孩子下午几点上课？”（刺激：A->A）。丈夫回应：“孩子的事你总记不住！”（回应：来自P，指向C）。沟通被交错，原话题（获取信息）可能中断，转向争吵（“谁记性差”）。
*   **期望成人（A），得到儿童（C）**：丈夫说：“明晚我不回来吃饭了。”（刺激：A->A）。妻子回应：“哎呀我太可怜了，都没人陪我了。”（回应：来自C）。沟通被交错。
*   **期望儿童（C），得到父母（P）**：妻子说：“商场打折了，咱们快去买点东西吧！”（刺激：C->C）。丈夫回应：“你都有多少鞋了还买？”（回应：来自P）。沟通被交错。

**交错沟通的原则是：一旦交错发生，最初的对话就会被打断，随之而来的是全新的、不同主题的沟通。** 大多数情况下，交错沟通会引发误解和冲突。然而，如果有意识地使用交错沟通来打断一个无益的循环（例如，从互相指责的父母状态中跳出来，重新用成人状态提问），它也可能产生积极效果。

### 3. 隐藏沟通（暧昧沟通）

隐藏沟通的特点是，沟通同时发生在两个层面：社交层面（公开的、语言信息）和心理层面（隐藏的、非语言信息）。**沟通的最终结果由心理层面决定，而非社交层面。**

**代码式描述：**
```
社交层面信息 = “现在几点了？” (看似 A->A)
心理层面信息 = “你回来太晚了！” (实际 P->C)
最终沟通结果 = 由心理层面信息决定（对方感到被指责）
```

以下是隐藏沟通的示例：

*   **示例一**：一方冷冷地问“现在几点了？”，另一方低声答“十二点了”。社交层面是确认时间（A->A），心理层面是“你回来太晚/我知道错了”（P->C）。
*   **示例二**：一方绷着脸问“你生气了吗？”，另一方扭头说“我没生气”。社交层面是询问情绪（A->A），心理层面是“我在生气/我知道你在生气”（C/P 互动）。

**隐藏沟通的原则是：当沟通存在两个层面时，结果由心理层面（隐含信息）决定。** 研究显示，非言语信息（语调、表情、姿势）对沟通结果的影响占比高达约70%。隐藏沟通大多会造成困扰，但有时也能创造幽默或趣味，例如在某些团体游戏中的互动。

## 顺畅与不顺畅的沟通模式分析

了解以上三种类型后，我们可以更具体地分析婚姻中的沟通。

顺畅的沟通通常是互补的：
*   养育型父母（NP）<-> 顺从型儿童（AC）：彼此照顾与接受照顾。
*   成人自我（A）<-> 成人自我（A）：就事论事，解决问题。
*   自由型儿童（FC）<-> 自由型儿童（FC）：一起玩乐，分享快乐。

不顺畅的沟通常涉及交错或隐藏：
*   **求安慰（C->NP）却遭批评（得到CP）**：“我心情不好。”“别太矫情了。”
*   **求配合（FC->FC）却遭拒绝或说教**：“咱们去看电影吧！”“就知道玩，不能干点正事？”
*   **询问信息（A->A）却遭指责（得到CP）**：“我钥匙放哪了？”“你总是乱丢东西！”
*   **隐藏的指责**：表面关心“你今天怎么了？”，实则传递“你不该有这种情绪”。

## 总结

本节课中，我们一起学习了TA理论中沟通过程的核心概念。我们明确了沟通是“刺激-回应”的单元，并深入探讨了三种沟通类型：**互补沟通**（顺畅和谐）、**交错沟通**（易引发冲突中断）以及**隐藏沟通**（表里不一，结果由隐含信息决定）。

理解这些模式，能帮助咨询师在微观层面分析来访者互动的每一句话，也能帮助夫妻识别他们沟通顺畅或卡顿的具体环节在哪里，从而进行有针对性的改善，而非笼统地要求“好好沟通”。下一节，我们将继续探讨如何在冲突中应用这些知识。
