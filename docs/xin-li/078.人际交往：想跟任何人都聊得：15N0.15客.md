# 078.人际交往：想跟任何人都聊得：15N0.15客

在本节课中，我们将系统学习如何与客户进行有效沟通，掌握从售前准备到售后维护的全流程销售技巧。课程内容源自《沟通力！把话说到客户心里去》一书，我们将通过拆解核心方法，帮助你提升销售沟通能力。

***

## 📖 课程概述

本节课我们将围绕销售沟通的四个核心阶段展开：售前准备、售中沟通、售后维护以及销售禁区。我们将通过具体案例和实用技巧，帮助你理解如何将话说到客户心里，从而建立信任、促成交易并维护长期关系。

***

## 🎯 第一部分：售前准备

上一节我们介绍了课程的整体框架，本节中我们来看看销售开始前需要做哪些准备工作。充分的准备是成功销售的基础。

以下是销售人员需要做的五项核心准备：

1.  **建立自信**：克服紧张的关键在于“忘我”。将注意力完全放在客户身上，揣摩其兴趣与需求，而非担忧自身表现。公式可以表示为：**自信 = 专注客户 - 自我担忧**。
2.  **调整心态**：接受“被拒绝是常态”。通过优化第一印象和开场白降低被拒概率，并大胆挖掘新客户。好客户可遇不可求，需持续开拓。
3.  **明确销售本质**：你卖的不是产品，而是**客户需求**。必须清晰了解产品能解决客户的何种问题或满足何种需求。例如，洗发水的核心价值可能是去屑或柔顺。
4.  **培养口才与幽默感**：良好的口才和适度的幽默能缓解紧张气氛，拉近与客户的距离。例如，用“出门是兔子，办事是孙子，回来是骆驼”来形容忙碌，容易引发共鸣。
5.  **成为耐心听众**：对客户保持最基本的礼貌和尊重，无论其言论是否琐碎或带有批评。记住“嫌货才是买货人”，挑剔的客户往往有真实购买意向。

此外，对客户要像中医一样“望闻问切”：
*   **望**：观察客户的眼神、表情、动作及环境。
*   **闻**：专心倾听，适时确认与回应。
*   **问**：善于提问，挖掘深层需求（后续有专门课程）。
*   **切**：用语简洁，最终陈述要切中要害。

## 💬 第二部分：售中沟通

准备工作就绪后，我们进入最核心的销售过程。本节将探讨如何与客户展开有效对话，处理异议并推动成交。

### 如何开口搭讪

与陌生客户建立联系是第一步，切忌急功近利。

以下是三种有效的开口方法：

1.  **寻找合适话题切入**：首次见面避免直接推销。应从客户关注的话题入手建立信任。例如，销售电动拖把的业务员，可以先从客户养的狗聊起，逐步过渡到清理狗毛的需求，最后自然引出产品。
2.  **先给予甜头**：通过赠送试用装、优惠券或小礼物等方式，让客户体验价值或建立好感。例如，销售母婴产品的业务员，可以先与客户家人聊育儿话题建立关系，临走时赠送小玩具，从而赢得信任和后续销售机会。
3.  **激发客户好奇心**：利用人类的好奇心吸引客户注意。具体方法包括：
    *   **提供新奇信息**：“我们有两款新产品，可能影响您的业务，有兴趣了解吗？”
    *   **利用从众心理**：“您的许多同行都用我们的方案解决了某个重要问题。”
    *   **直接抛出利益**：“我有一种设备，能让您工厂每月省电30%。”

### 分析客户性格类型

了解客户性格有助于采取更合适的沟通策略。作者将客户分为四种基本类型：

1.  **分析型**：办公室井井有条，衣着保守。沟通应直奔主题，用事实、数据和逻辑说话，聚焦工作、公司近况及产品能解决的具体问题。
2.  **情感型**：办公室有私人物品，穿着时尚，肢体语言丰富。喜欢轻松氛围，沟通时可适当幽默，在非正式场合洽谈效果更佳。
3.  **主观型**：办公室陈列奖状文凭，穿着考究，谈话严谨。有主见，喜欢直奔主题，不易接受建议。沟通时需用事实和数据增加说服力。
4.  **随和型**：办公室有家庭照，衣着大众，温文尔雅。沟通时应放下强势姿态，保持随和，建立良好互动。

客户性格可能兼具多种特点，以上分类可作为参考。

### 处理客户反感与拒绝

客户常有防御心理，处理其反感是销售中的关键挑战。

对于**直接表示反感**的客户，可以尝试以下五种方法：

1.  **提供有价值信息**：分享市场走向、专家建议等干货，体现专业与靠谱。
2.  **巧用“第三者”**：“是您的朋友X总推荐我来的，他觉得您可能对我们的产品感兴趣。”
3.  **现身说法**：列举知名公司或同行使用产品后获得成功的案例。
4.  **让产品自己说话**：通过演示直观展示产品优势，如防火服的防火实验。
5.  **真诚沟通**：实诚比口才更重要。长期来看，真心为客户着想才能赢得持久信任。

对于**委婉拒绝**的客户，常见的托词及应对策略如下：

*   **“我很忙”**：判断真假。若真忙，礼貌约定下次时间；若假忙，可争取固定时间或强调“只需三分钟，听完您会觉得值得”。
*   **“没预算”**：通过提问探知是真实需求不足还是预算问题。
*   **“我决定不了”**：判断对方是否有实权。若无，可尝试通过他引荐决策者。
*   **“暂时不需要”**：最后争取，清晰对比“不买的坏处”与“买的好处”，直击要害。
*   **“价格太贵”**：将总价平摊到每日、每周，让客户感觉划算。例如：“地毯总价297.6元，用5年，平均每天只需0.16元。”
*   **“没兴趣”**：直接抛出核心利益点吸引注意。例如：“有一种方法可以帮您提升30%业绩，且无需额外投资。”
*   **“随便看看”**：不要放弃跟进。可以这样说：“是的，买衣服要多看看。不过我们这几款主打产品特别适合您，买不买没关系，可以先了解一下，这边请。”

### 获取客户好感

打消反感后，需要进一步赢得客户的好感与信任。

以下是五个关键点：

1.  **具备“说学逗唱”的灵活**：像相声演员一样，根据情境灵活应变。例如，服装店兄弟一个报价、一个“听错”报低价，配合促成交易。
2.  **与客户保持一致**：客户提出异议时，先认同再解释。例如，客户说衣服贵，可以先说“是有点贵”，再转向“但一分钱一分货，这衣服很配您的气质”。
3.  **行动胜过言语**：在适当时机为客户做些实事，如客户生日时送一束花，远比口头祝福更令人感动。
4.  **建立信任而非仅靠口才**：让客户感觉你实在、可靠，真正在为他考虑。
5.  **巧妙赞美客户**：
    *   **侧面赞美**比直接恭维更有效：“早就听说您白手起家，今天得向您请教。”
    *   **说给第三者听**的赞美，迟早会传到客户耳中。
    *   **从欣赏的角度表达**：“您的办公室真气派，我要是有一间这样的就满足了。”
    *   **善于发现闪光点**：任何人都有值得赞美之处。

### 激发客户兴趣与引导需求

赢得好感后，需引导客户对产品产生兴趣并最终购买。

**激发兴趣**的关键在于用产品打动客户：

1.  **按重要性介绍卖点**：先说核心优势。
2.  **专业解说**：引用权威数据或演示，打消顾虑。
3.  **用效果说话**：通过对比演示让客户直观感受产品优势，如冰箱的密封性测试。
4.  **突出核心卖点**：话贵精不贵多，根据客户关注点强调价格、质量或服务。
5.  **主动提及微小不足**：在展示优势后，主动提及一个不重要的缺点，反而能凸显真诚，增加信任。

**引导需求**是一个心理博弈过程，需步步为营：

1.  **找到痛点，使用诱导性语言**：创设情境让客户感同身受。例如，销售空调时描述夏日回家的闷热难耐，激发购买欲望。
2.  **创造需求**：当客户认为没有需求时，引导他发现潜在需求。
3.  **针对客户关心点介绍**：根据前期了解的客户关注点，针对性介绍产品功能。
4.  **利用人性心理**：使用正话反说等技巧。例如，保险柜广告：“诗牌保险柜最大的缺点就是必须用密码开锁，不然只能用焊枪切开。”
5.  **实施暗示性引导**：当客户犹豫时，给出倾向性建议。例如，客户配隐形眼镜时，暗示“佩戴隐形眼镜的人通常还会配一副框架眼镜交替使用”。

### 有效倾听

倾听与说话同等重要，有时甚至更重要。

以下是倾听的六个要点：

1.  **让客户多说**：客户有表达欲，认真倾听是对其的尊重。
2.  **不打断客户**：这是最基本的沟通礼仪。
3.  **边听边思考**：思考客户的真实需求和关注点。
4.  **听出要害**：只有听懂，才能有效引导。
5.  **全盘接收**：像海绵一样吸收客户的所有信息，无论好坏。
6.  **会听比会说更重要**：听懂需求，才能一语中的。销售不成功，往往是因为听得太少。

### 促成交易与处理临门一脚的拒绝

当出现以下**成交信号**时，意味着机会来临：

1.  **表情信号**：客户微笑、点头，对产品露出满意神色。
2.  **语言信号**：客户询问细节，如使用方法、保养、交货时间等。
3.  **行为信号**：客户反复触摸产品、邀请朋友体验、仔细翻阅说明书等。

若客户在表现出兴趣后突然拒绝，可采取以下策略：

*   **“我要考虑一下”**：制造紧迫感，暗示“今天是活动最后一天”或“不买可能会有损失”。
*   **“我想再去别家看看”**：强调自家产品的独特优势和亮点，争取客户。
*   **“我只认XX品牌”**：解释品牌溢价，强调自家产品在同等质量下性价比更高。
*   **“我要回家商量一下”**：先表示理解，再引导其自己做决定的好处（如制造惊喜），或施加轻微压力（如强调限时优惠）。

### 报价与讨价还价

价格是敏感话题，处理需要技巧。

1.  **克服报价心理障碍**：遵循“漫天要价，就地还钱”的原则，预留还价空间。最终成交价往往是中间值。
2.  **让客户感觉“赚到了”**：降价时表现得不情愿，让客户觉得获得了特别优惠。
3.  **使用价格细分法**：将总价分摊到更小单位（如每天、每月），让高价显得廉价。例如：“这套房子70年产权，算下来每天只需20元。”
4.  **提供多种方案**：给出不同价位的选择，客户往往倾向于选择中间档。
5.  **延迟报价**：对于一上来就问价的客户，先引导其了解产品价值，待购买欲望被激发后再报价。
6.  **处理支付压力**：推荐符合其预算的产品，或提供分期付款方案。

## 🔧 第三部分：售后维护

销售并非终点，良好的售后服务是维持客户关系、获得转介绍的关键。

售后服务的六大原则如下：

1.  **以诚相待**：始终对客户保持真诚，尤其在处理投诉时。
2.  **欢迎投诉**：将投诉视为发现问题、改进产品和服务的机会。
3.  **换位思考**：站在客户角度理解其诉求和感受。
4.  **避免争辩**：与客户争辩，即使赢了道理，也会失去客户。
5.  **满足需求**：及时响应并解决客户售后问题。
6.  **迅速解决**：发现问题立即处理，拖延会加剧矛盾。

## ⚠️ 第四部分：销售十大禁区

在销售沟通中，有些错误必须避免。

以下是销售中的十大禁忌：

1.  **三句话不离本行**：开场就硬销产品，最令客户反感。
2.  **丑话说在前头**：未展示优点前就提缺点，会吓跑客户。
3.  **矮子面前说短话**：切勿触及客户的短处或缺陷。
4.  **说话啰嗦**：冗长的介绍会让客户厌烦。
5.  **把话说得太绝对**：如“绝对最好”、“绝不会坏”，一旦出问题将毫无回旋余地。
6.  **刀子嘴豆腐心**：尖刻的言语就是伤人的刀子，客户没义务了解你的“豆腐心”。
7.  **追问隐私**：没有分寸地打探客户私事。
8.  **忽悠客户**：虚假宣传终将失去信任。
9.  **与客户争辩**：这不是辩论赛，赢了争论等于输了生意。
10. **贬低对手**：通过贬低竞争对手来抬高自己，反而会降低自身格调，显得人品不佳。

一位销售专家曾总结价格谈判的精髓，形成一个应对循环：**客户嫌贵谈质量 -> 客户苛求质量谈服务 -> 客户挑剔服务谈条件 -> 客户逼迫条件谈价格**。
