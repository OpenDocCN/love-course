# 课程103：高琳学会讲故事，才叫会沟通（完结） 🎯【032 感染力】不会讲故事的领导不是好领导

在本节课中，我们将探讨为什么在当今商业环境中，讲故事的能力是领导力的核心组成部分。我们将分析领导力面临的挑战，并学习领导者必须掌握的三种关键故事类型。

---

很多人都以为“昨天你对我爱搭不理，今天我让你高攀不起”这句话来自马云。其实这是网友编造的。然而，2014年马云在阿里巴巴纽约路演现场，真实演绎了这句话的精神。他说：“15年前，我来美国要200万，被30家VC拒绝了。今天我又来了，要200亿。”

你听出来了吗？这段开场白就是一个故事。它符合我们在课程中定义的故事结构：**因为**（15年前我来要钱），**但是**（你们拒绝了我），**所以**（今天我带着更大的目标回来了）。

我经常听人说，马云没什么真本事，就是靠会讲故事。每次听到这句话我都想，那你也靠讲故事试试。的确，马云的成功离不开他的故事力。无论是“十八罗汉”的创业故事，还是阿里巴巴的愿景故事，马云或许不是中国最会做生意的领导，但他的确是中国最会讲故事的领导之一。

事实上，在今天的商业环境中，不会讲故事的领导就不是一个好领导。为什么这么说呢？

上一节我们看到了故事的力量，本节中我们来看看领导力面临的现实挑战。

首先，现在的领导并不好当。
*   基层领导除了管理团队，自己也得干活。
*   中层领导看似技术活干得不多，精力更多放在管理上，但他们常常承受来自上下的压力。
*   高层领导看似风光，但需要应对内忧外患，员工、客户、合作伙伴，个个都需要操心。

美国领导力大师约翰·麦克斯维尔博士在《领导力的五个层次》中写道，领导力分为五个层次：
1.  **职位领导**：靠职权让人服从。这对老一辈可能管用，但对年轻一代效果有限。
2.  **认同领导**：依靠人际关系的影响力。
3.  **生产领导**：依靠打造高效的组织系统。
4.  **立人领导**：依靠培养和发展人才。
5.  **巅峰领导**：依靠为团队树立宏伟的目标和愿景。

无论你处于哪个层级，都需要影响你的员工。每个领导都需要知道，当你在上面讲话时，下面听的人脑子里可能一直有三个字：**凭什么？**

如果你不能回答这个问题，你的沟通就是无效的。当你发号施令时，员工心里会想“凭什么听你的？”。当你描绘公司前景时，空口白牙，人家“凭什么信你？”。

要回答这些问题，就需要搞清楚：你是谁？你有什么价值观？你的内在动力是什么？长远来看，你想成为什么样的领导？想把团队带向何方？如果这些问题都搞不清，就别惦记领导别人了，先领导好自己。

所以，通过讲故事影响下属的前提，是先搞明白自己的故事。作为高管教练，这是我工作中最重要的一部分：帮助领导者思考自己的领导力故事。

这些故事有的是个人层面的，有的是公司层面的，有的甚至是哲学层面的。无论在哪个层面，目的都是为了**影响**——影响员工、客户、消费者，以实现企业的愿景和使命。

领导者要讲的故事很多，但最重要的三个故事，其实就是回答传达使命时被问到的三个经典问题：**你是谁？来干嘛？要去哪？**

接下来，我们逐一来看这三个核心故事。

**第一， “我是谁”的故事。**
这是介绍自己和公司的故事。

**第二， “来干嘛”的故事。**
这是解释你为何在此的故事。你是来卖产品、要投资、激励员工做出改变，还是鼓舞士气？

**第三， “要去哪”的故事。**
这是给出方向、描绘愿景的故事。但光讲目标没用，还得讲明白为什么要去那里，以及怎么去。例如，在宣布公司战略调整时，需要先向员工讲明白“我们为什么要这样做”，让他们觉得有道理，然后才能讲如何带领他们实现目标。

接下来，我用阿里巴巴在美国上市时的宣传片节选，来阐述这三个故事是如何应用的。在后续课程中，我们再分别学习如何在具体工作场景中讲好这三类故事。

（以下为视频内容翻译）
“大家好，我是马云，阿里巴巴集团的创始人和董事会主席。15年前在我的公寓里，18位创始人有了一个梦。这个梦就是在某一天我们能够创立一个为成千万小企业主服务的公司。这个梦想从始至终都没有改变过：让天下没有难做的生意。”

这一段点出了马云的角色、阿里的历史，最重要的是说出了阿里的**愿景**和**使命**，以及它存在的意义——为谁服务，提供什么服务。这段开场白看似简单，但很多领导都讲不好，常常会直奔产品细节。其实，如果听众连你是谁、干什么都不知道，根本不会关心产品细节。

“如今15年过去了，我们在中国已经成了一个家喻户晓的名字。现在我们也已经准备好让全世界来认识我们。”

这一段讲的就是“**来干嘛**”。很简单，今天来就是融资。让全世界认识，是一种委婉的说法。马云深知要钱也要得含蓄。再说了，不认识你，怎么会给你钱呢？

“阿里巴巴已经走过了一段不短的旅程。要让阿里巴巴成为一家持续发展102年的企业……还有87年的时间需要我们努力。我们深知，今天很不易，明天更困难，但是未来是无限美好的。因此，我们必须更加努力，才能在未来的漫漫征程中赢得胜利。”

这一段讲的就是“**要去哪**”，也就是阿里的愿景——要成为一家持续发展102年的企业，并为此目标而努力。

在这短短几分钟里，马云把“他是谁”、“来干嘛”、“要去哪”说得清清楚楚。虽然每一部分可能不单独成为一个长篇故事，但通过后面展示的真实商家案例，观众可以进一步理解上述的愿景和使命。因此，这个宣传片看似简单，背后却充分体现了故事思维和故事力。

---

**总结一下：**
作为人类，讲故事是我们展望未来、预测、规划和解释的主要方法。而作为一位领导，展望未来、预测、规划、解释和说服他人，正是最主要的工作。所以，想要当一个好领导，势必要先成为一个讲故事的高手。

因为对外，你需要通过故事把产品和服务卖给客户；对内，你要通过故事激励员工，带领他们面对变化。无论是招聘还是裁员，每一次都是一个讲故事的过程。

在接下来的几节课，我们就来看看怎么才能把领导的故事讲好。
