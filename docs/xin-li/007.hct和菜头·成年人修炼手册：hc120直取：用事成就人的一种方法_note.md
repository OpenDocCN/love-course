在本节课中，我们将要学习一个古老而有力的概念——“直取”。这是一种通过直接行动来打破犹豫、直面目标的方法，它能帮助我们摆脱精神内耗，用行动推动自我成长。

## 概述：什么是“直取”？

“直取”是一个动词，意为直接进取，即对准目的或目标径直前进。如今这个词已不常用，它属于一种比较古雅的中文表达。

现代人往往更聪明，懂得用迂回的方式达成目的，不愿硬着头皮顶着压力直接前进。然而，我们总会遇到精神倦怠、内心无所适从的时刻。此时，“直取”可以成为一种有效的对峙方式。

它能帮助我们暂时从软弱和混乱的精神状态中解脱，获得短暂的清明，从而得以采取行动，改变自身处境。

## 为何我们需要“直取”？

上一节我们介绍了“直取”的基本概念，本节中我们来看看为何现代人尤其需要这种方法。

头脑聪明、阅历丰富的人难免思虑过多，出于审慎，他们会在行动前穷尽一切可能性。天性软弱、偏好依赖的人则容易陷入自我怀疑，对进行中的事丧失信心。真正乐天而坚定的人在生活中极少，大多数人常陷入思绪和情绪中原地打转。

此时，最简单的答案就是“直取”，但人们总是想方设法回避它。

## “直取”的初体验：从单相思说起

其实，每个人最早都有一次学习“直取”的机会，那就是陷入单相思的时候。

一般情况下，一个人陷入单相思，便茶饭不思、魂不守舍，精神异常亢奋，满脑子都是不切实际的幻想。这些幻想不会真实发生，只是在脑海中疯狂生灭。最终涌起的情绪是患得患失、进退失据，仿佛被无形绳索捆绑，陷入深深的自我折磨。

我曾被折磨过一两次，此后人生再未发生类似状况。因为我发现，即便内心极度惶恐不安，极度担忧遭拒，也应硬起头皮，径直走上前自我介绍，然后直接表白，问对方要一个答案。

在表白之前，一切都不确定，这种不确定足以长时间折磨自己。而且，无论自我折磨多久，也不可能自动产生结果。这感觉如同日本的死刑犯，宣判后不知具体何时执行，也许很快，也许要等上十几年。

**公式：不确定性 → 持续的精神内耗**

硬着头皮直接表白，无数种可能立即坍缩成两个明确结果：成，或不成。无论哪个答案，单相思至此终了。接下去可以是欢喜或痛苦，但不再是自我折磨，无需继续耗费时间却无所得。

## “直取”的实践：超越恐惧

“直取”给人的感觉很恐怖，事实也的确如此。但真正去做，会发现一切不过是内心造作出来的高峰深谷。得偿所愿未必喜出望外，婉言拒绝也未必定痛彻心扉，它给内心带来的是踏实与安定。

尘埃落定后，人总能轻松起来。若能多“直取”几次，恐惧随之淡化，最后变成一种个人行为逻辑：**喜欢就上去问一声，问一声又不会死**。

## 在事业中应用“直取”

很多时候，我们对人对事缺乏必要认知，却又被深深吸引，无法绕开。

例如，我很早喜欢写作，但至今不确定能写到何种程度。自我怀疑与杂念纷扰难免，我依然受其困扰。不过，和当初相比，这种困扰的影响小了很多。因为我直接选择写作，然后直接在网络发布文字，任人品评。

以下是实践“直取”的具体步骤：
1.  **立即开始**：不会写就去模仿名家。
2.  **直面挑战**：有论战就去查资料。
3.  **持续发布**：技巧、结构等可慢慢学，但发布第一篇文字不能等。一等可能就是几十年，个人毫无进展。

**代码逻辑：**
```python
if feeling_hesitant(about_goal):
    take_direct_action() # 采取直接行动
    publish_or_share() # 发布或分享以获取反馈
    learn_from_feedback() # 从反馈中学习
```
虽然说“直取”类似于火力侦察，未必能证明最初想法正确。但换角度想，世间值得火力侦察一番的事情本就不多，本应珍惜。而最好的珍惜方式，莫过于去“直取”。

## “直取”与自我证明

我曾对互联网产品感兴趣，觉得自己在互联网多年，应有所谓“产品感觉”，适合做产品。后来证明这多是天真幻想。但当机会来临时，我勇敢向前一步，尝试做了一回产品经理。

自己亲手打造产品，并希望获得百万千万用户。事情进展很不顺利，最糟糕时，我午夜独坐自问：“你知道你究竟在干什么吗？”

我至今记得答案：现在的局面的确困苦，我的确被卡在中间。但是，这种痛苦和当初不确信自己能否成为产品经理相比，我宁可要现在这种扎扎实实的痛苦，也不愿要当初那种虚无缥缈的痛苦。

**核心抉择：在“丢人”和“成为笑话”之间**
*   现在做不成，无非一时丢人。
*   当初一直犹豫不决，则现在就是个笑话。
*   我宁愿选择一时丢人，也不愿当一世笑话。

在失败了十几款产品后，我终于做成一款拥有几千万用户的小产品。“丢人”早已不成问题，因为绝大多数用户不知我的存在。整件事变得与他人完全无关，完全是我自己与自己的较量。

我耗费大量时间，经历诸多痛苦才意识到：自己不可能成为最好的产品经理。这一点让我难过，但我最终证明了自己有一定能力，并彻底脱离了“丢人”与“笑话”二选一的心态，在困苦中有所得。

## “直取”的益处与代价

总结下来，“直取”主要有三大好处：

1.  **中断内耗，获得答案**：它可以立即打断犹豫彷徨、原地打转的状态，让人很快知道答案，从纠结中解脱，接受可能结局，让事情终了。
2.  **以事成人，推动成长**：事情做起来，人才会跟着动起来，并在此过程中不断完善自己。人成就事是少见的，事成就人是常见的。不行动，我们永远不知道自己是否可以。
3.  **粉碎伪装，保全心性**：人之所以纠结，常因太在意自我——自我的感受、情绪、期待都会妨碍选择，让人患得患失。“直取”可以粉碎这种自我伪装，保全真实心性。因为喜欢一个人就是真的喜欢，热爱一件事就是真的热爱，投入其中可忘记自我，人在行动中能找到真实价值。一旦确认，感受、情绪、期待都变得不那么重要，一切选择后的结果其实都可以接受。

如果说“直取”有缺点，那就是它不必然指向成功、幸福或内心安乐。它可能带来更多压力、负担和痛苦，因为做出选择、采取行动就要承担后果。

这取决于每个人如何看待自己的人生：有些人一定要追问一个结果，而有的人觉得需要始终保持可能性。对于那些得不到结果就无法入睡的人而言，“直取”是一种合理的选择。

人一起心动念就开始受苦，那么不妨让这种痛苦来得更猛烈一些，于是自己也就活得更为真切一些。

## 总结

本节课中，我们一起学习了“直取”这一方法。它是一种通过直接行动来打破精神内耗、直面目标的生活策略。我们从单相思的比喻谈起，探讨了它在情感、个人事业与自我证明中的应用，并分析了其带来的益处与必须承担的代价。

关键在于理解：**行动本身具有价值，它能终结不确定的痛苦，并在过程中塑造更真实的自我。**
