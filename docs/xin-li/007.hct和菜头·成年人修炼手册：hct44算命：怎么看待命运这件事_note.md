# 课程007：成年人修炼手册·hct44 算命：如何看待命运这件事 🔮

在本节课中，我们将探讨“算命”这一现象，分析其背后的心理动机，并讨论我们应如何理性地看待“命运”这个概念。我们将通过具体的故事和观察，理解算命行为对人生的实际影响。

## 概述

算命，指解读个人命运轨迹的行为。它在提供心理抚慰的同时，也增加了人对未来的期待。许多人曾询问我关于算命的事，这促使我反思自己的言行。我个人对算命兴趣不大，但恰巧认识一些从业者。说实话，我更乐于观察他们本人，而非在意他们的断言。总体而言，算命的人比算命这件事更有趣。

## 第一节：家族往事中的“神算”

上一节我们提到了对算命者的观察，本节中我们来看看一段家族往事，它揭示了人们依赖算命的一种深层心理。

我很小的时候，听父亲说起他幼年时曾专门请人算命。回想起来，那应该是个类似寨子里巫医角色的神婆。原话是说，他将来会离祖先的宅基地很远。我奶奶听了很着急，不想儿子离身边太远，便请求神婆解释。神婆看出了她的担忧，就说可能是去省城做工一类的事情。

后来，我父亲果然从云南去了新疆，从中国西南角到了西北角。春节假期回来一趟，一半时间都浪费在路上。在我父亲看来，那个神婆算得很准。

不过，我有我的看法。我认为，父亲内心一早就有远行的念头，但放不下家乡，于是用算命来平复心中的纠结。他认同神婆的预言，是因为神婆越准，他需要为自己当日选择背负的责任就越轻。**背景离乡是命运的决定，而非自我的选择**。这样，他在自己的人生里就可以当一个安稳的旁观者，而不用去追问那个令人痛苦的问题：“如果当初我选择了……”。

对于我来说，听完这段往事，第一句话是问：“还能不能找到那个神婆？”我的想法和大多数人一样，既然存在那么一位神人，不如请她也为我算一回命。那时我还是个孩子，对人生和命运充满好奇，等不及时间一页页翻开呈现结果，而是迫不及待想翻看答案。

回答当然是没有了。这类故事永远如此：神人隐退在时间迷雾深处，想再找时早已不知所踪。我觉得未必是找不到，只是这种故事需要这样的神秘色彩，命运才有如烟似雾、难以捉摸的效果。

## 第二节：一位“科学实证派”的命师

从家族的神秘故事走出，我们再来认识一位风格截然不同的现代命师，这或许会改变你对算命行业的刻板印象。

我知道的第二位命师是位专业人士，自称受过箓，是有证有照的火居道士。我非常喜欢这位大师，他性格里有种让人难以抗拒的幽默感。

大家比较熟悉之后，我诚心请教他：“您老人家行走江湖，为什么不见身上带着法器呢？”他反问我：“你觉得我应该带什么法器？”我说：“您都没有个罗盘什么的，也没有笔墨纸砚排命盘吗？”

他老人家听完呵呵一笑，从兜里掏出一台iPad，非常熟练地打开一个APP，屏幕上立即显示出一个电子罗盘。他很得意地说：“看到了吧，谁还背什么罗盘呢？一个APP就搞定了，方向还准。”然后，他又打开另一个APP，是个电子命盘，继续说道：“算什么算呢？用这个点几下就给你排好了，为什么每次还要我亲手动啊？笨！”

我看得心悦诚服，觉得他是否高人另论，但起码是个通达之人。朋友介绍他和我认识，并坚持要了我的生辰八字送过去，请他给我算一次命。应该说还是挺准的，后来很多事情都应验了。他说我几岁转运、几岁开始赚钱，基本上都很准，前后误差不超过半年。

我内心其实有些惊奇，因为我身上有件奇特的事：我没有出生证。问我母亲，她也不记得我是几点出生，是上午还是下午。所以算命时我只能编一个时间，但他还是算得很准。我和大师说了这件事，他让人喜欢的就在于这种时刻。他浑不在意地一挥手，说没关系，他也要看算出来的结果是不是和我相符，如果对不上，他自己也会去调整。

我当时大为钦佩，觉得这位命师术士也遵循**科学的实证精神**，实在难能可贵。但那时我已接近四十岁，对算命的态度有了很大转变，对于他给我算的命数并不是那么在意，而是更喜欢拉着他聊天，请他给我说说各种神异之事。因为他有另一套世界观，对世界运转规律也能自圆其说，这让生活显得丰富多彩。

大师也给自己算过命，而且根据结果做了许多布置，包括给家人改名甚至改姓。但是，大师也没有逃过自己命中注定的劫难。最后一次听到消息，是他被人追捕不得不跑路，从此消失在茫茫人海。

## 第三节：算命的“第二步”与无尽循环

了解了两位命师的故事后，我们深入思考一下：为何我对算命的态度会发生转变？关键在于，我观察到一个几乎无法避免的麻烦。

无论是观察朋友还是观察我自己，我发现算命都避免不了“第二步”，而第二步是个麻烦。无论算出怎样的命，总有好的部分，也总有坏的部分。人性贪图圆满，对好的部分会欣然接受，觉得理所应当；但对于坏的部分却根本放不下，非要去做这第二步。

以下是第二步的具体内容：

*   **请求转运或改命**：人们会请求命师想办法转运或者改命。

而这第二步的结局只会有两个：有效，或无效。

以下是两种结局可能引发的后续发展：

*   **如果有效**：会发展到第三步，要求将效果长期化、固定化。
*   **如果无效**：情况更糟。人性会认为“算得准，但解法不灵”，因此要去别处求取解法。于是便会有第三步、第四步，没完没了。

本来没什么事，对付着也能活，但一旦起心动念，就会在这条路上越走越远。我有些朋友到处找大师算命，感觉是一定要算出满意的一命来才能停止，但这又怎么可能有停止的时候呢？

所以，为了防止自己的心不安宁、不断向外求取，我后来直接拒绝算命，把命运当作是**抽盲盒**就好了。

## 第四节：《了凡四训》的启示

面对算命可能带来的执念，我们该如何调整心态？有一本小书对我的转变帮助很大。

有一本读起来很轻松的小册子，叫做《了凡四训》。内容讲述了袁了凡这个人遇见一位命师，算定了他人生中的所有重大事项，而且每一次都应验了。于是，他知道自己能活多久，安安静静等待那天的来临，对人生再无期待。

但是，他最终还是从“定数”中逃离了出来，得到了命中本来没有的孩子，而且活过了预言中的寿数很久。如果你对算命非常有兴趣，不妨找这本书来翻一翻。虽然主题是劝人向善、多做好事以改变命运，但其中“命运可以改变”的观点非常刚健乐观，让我非常喜欢。

事在人为，总比一切注定要好。即便从梦幻、从游戏的角度来理解人生，自己手里多少能控制一点，也比被动接受来得强。

## 总结与思考

本节课中，我们一起学习了如何理性看待算命与命运。我们通过家族故事看到了算命作为心理安慰剂的作用，通过现代命师的例子看到了其行业的另一面，并深入分析了算命行为容易导致的执念循环。最后，我们从《了凡四训》中获得了“命运可以改变”的积极启示。

我知道，询问算命事项的人，大都遇见了生活中的挫折、人生中的苦楚，觉得是命运的缘故，自己才会被反复戏弄。不过，关于这件事，我是这么看的：

命运给予我们慷慨馈赠时，我们好像并没有做什么特别的事，它也就自然发生了。如果命运要降临惩罚时，我们好像也做不了什么特别的事，该来的总会来。

那么从另一方面来说，如果命运慷慨赠与时，我们可以坦然接受，觉得理所应当；那么，当命运降下惩罚时，不也是一种理所应当吗？我们又有什么特别，会认为自己为天地所钟爱，除了好命，就只有好命呢？
