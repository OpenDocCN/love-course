# 103.高琳学会讲故事，才叫会沟通（完结）：040【感染力】如何打造你的故事力？_note

在本节课中，我们将学习如何将日常素材转化为动人的故事，并探讨如何运用故事思维来重塑我们看待生活与沟通的方式。

---

生活中处处是故事，我们每个人都在不断创造自己的故事。故事力是人类与生俱来的能力，只是现代教育体系可能使其逐渐弱化。因此，我们需要有意识地重新发现并锻造自己的故事力与故事思维。

上一节我们探讨了如何搜集故事素材，本节中我们来看看如何将这些素材转化为完整的故事，并让故事思维为生活增添色彩。

去年圣诞节，我全家回西雅图休假。飞机接近阿拉斯加上空时，机长宣布引擎出现故障，需备降附近的美军空军基地。我原以为只是短暂维修，结果却在基地滞留了36小时，期间完全没有通讯信号。事后得知，故障并非“小问题”，而是一个引擎完全失效。

对多数人而言，这段惊险经历可能只是安全抵达后的一则谈资。但我从机长处理危机的方式中，反思并撰写了一篇关于领导力的文章。**故事是反思的成果**。那36小时的滞留，给了我充足的思考时间。

当然，你无需如此戏剧性的经历也能讲好故事。对一个拥有故事力的人而言，任何小事——甚至是发生在他人身上的事——都能被提炼出新观点，形成一个故事。

美国著名剧作家罗伯特·麦基曾说：“故事力是对生活的把握。” 拥有故事力的人能将普通生活，转化为更有高度、更富意味的体验。做到这一点，需要你在生活中做到：多观察、多体验、多思考、多提炼、多总结。

如果你讲不出深刻的故事，很可能因为你只看到生活表象，未深入思考其背后的本质。如果你讲不出生动的故事，很可能因为你只是“活着”，而非“生活着”——没有慢下来细心观察和体验生活中的酸甜苦辣，自然难以转化素材。

说到提炼总结，你可能会问：哪有那么多新观点、新道理可讲？的确，太阳底下无新事。世上本无全新的道理，新观点也常建立在前人基础之上。因此，我们需要通过广泛阅读来汲取这些观点与智慧——先积累，再突破。

小时候，家长常告诫要“好好学习，别看闲书”。工作后，阅读也常变得功利，只读“有用”的内容。但我越来越发现，**广泛阅读是培养口头表达能力的根本**。无论是虚构类小说，还是与工作无关的史学、美学、心理学，都会对你的思想产生潜移默化的影响。

所以，广泛阅读不仅是“读”，更是带着故事思维去读，思考哪些素材可为自己所用。这不仅能增加你的底蕴，还能让你的沟通更有力度。

那么，如何做到出口成章呢？我常听学员说：“讲故事挺难的，等我练好了再讲。” 我问：“什么才叫练好呢？”

我在英国读博写论文时，总觉得自己写的东西拿不出手，于是拖延不交，总想追求完美。我的教授看出了我的拖延源于完美主义。有一天，他语重心长地说：“**Lynn, done is better than perfection.**（完成比完美更重要。）” 此后，我也常把这句话送给学员，希望他们战胜完美主义心态。因为只有精彩的故事，没有完美的故事。精彩，是在一次次不完美的历练中积累出来的。

如果你总想着“练好再讲”，可能永远无法开始。无论是写故事还是讲故事，都需要用**输出倒逼输入**。

最后，我们来谈谈“锻造故事思维”。如果说故事力是一种沟通能力，那么**故事思维就是一种思考方式和人生态度**。当你用故事思维看待生活，就像戴上一副新眼镜，看到的世界也将不同。

例如，在故事思维里，失败不是永恒的，它只是你人生故事中的一个“冲突”。所有成功人士都曾失败过：马云大学毕业求职屡遭拒绝，连肯德基面试也未通过；新东方创始人俞敏洪曾多次被美国拒签。这样的失败故事很多，但他们并未定义自己为“失败者”，而是将自己改写成了故事里的英雄。

试想，如果你的一生完美无缺、风平浪静，每天重复，没有烦恼。这样的人生若拍成电影，除了父母，恐怕连你的孩子都不想看——因为它是一个没有冲突的故事，甚至称不上故事，索然无味。

既然没有冲突的故事不是好故事，那么用故事思维看待人生时亦然：**没有失败，就不足以谈人生**。

最后，我想问你：你会给自己讲故事吗？答案无疑是肯定的。但我想告诉你：**选择给自己讲什么样的故事，就选择了什么样的人生**。如果你日复一日告诉自己“我不行”，那你可能永远不行。如果你认为自己找不到好工作，可能就真的找不到。如果你认为自己不值得被爱，可能就永远遇不到爱你的人。

在故事中，我们强调对话的重要性，因为许多冲突都体现在对话里。在生活中，你的**内心对话**也决定了你如何面对冲突，如何挑战自己的**限制性信念**（即阻碍你前进的那些信念）。当你能换个方式跟自己讲故事，就能改变人生。

几年前我打算辞职创业时，内心非常不安。我担心没有收入，担心选错方向白费功夫。内心的故事愈发焦虑，几乎变成一部“灾难片”。后来，我换了一个故事跟自己讲：“好吧，就算闯荡一番发现自己不是创业的料，大不了回去上班，反正死不了。” 就这样，换个角度讲故事，让我放下了许多恐惧，将“灾难片”变成了“探索可能性”的励志片，从“失败了怎么办”变成了“失败也让我的故事更精彩”。

所以，选择什么样的内心故事，就是选择什么样的人生。当你拥有故事思维，你可以练习在生活中，有意识地选择对自己更有利的故事讲给自己听，而不是被大脑绑架，去讲那个吓唬自己的故事。

**如果你的生命是一本故事书，你就是这本书的作者。你要怎么写自己的故事呢？**

---

好，总结一下今天的课程。

生活中无处不是故事，每个人都是有故事的人。关键在于你是否用心体会生活、诚心反思生命、全心感受自己，并勇于讲出自己的故事。当你能做到这些，就可以说自己拥有了故事思维。

我们每个人的人生，都像一部长篇故事。无论你面临何种困境，那都只是你生命故事中的一个“冲突”。不翻到下一页，你永远不知道故事的结局。

我一直认为，讲故事是一项至关重要的沟通技能，但又远不止于此。从更深层看，它是一种思维习惯，是对自己人生这本故事书的接纳与铺写。

在课程的最后，我想用我书中副标题，也是我最喜欢的一句话来结尾：**做自己故事里的英雄，把自己活成一本畅销书。**
