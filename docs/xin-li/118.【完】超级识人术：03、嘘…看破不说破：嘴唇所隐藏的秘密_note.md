# 课程118：超级识人术 🕵️♂️ 03、嘘…看破不说破：嘴唇所隐藏的秘密 👄

在本节课中，我们将要学习如何通过观察一个人的嘴唇形状和嘴部动作，来洞察其性格特征与内心情绪。这些视觉识人的技巧源于古老的相学智慧，并得到现代心理学的部分印证，能帮助我们在社交中快速建立初步判断。

## 嘴唇形状与性格关联

从“相由心生”的观点来看，一个人长期的心态会通过五官形态表现出来。嘴巴即使不出声，也在“说话”。以下是几种常见的嘴唇形状及其可能反映的性格特点。

**红润、对称的嘴唇**通常是健康和心态平衡的标志。

**嘴唇薄**的人，性格可能更为**激进、刚强**，有时会显得**薄情**。

**嘴唇厚**的人，常被认为是**富贵长寿、富有艺术天分**之相。但嘴唇**过于厚**且突兀，则可能暗示此人**个人欲望较强**。

**嘴唇长**的人，往往**好胜心强、非常现实**，某方面的能力可能较强。

**嘴唇短小**的人，通常**富于理想但缺乏果断力**，容易犹豫不决，性格也趋于**保守**。

**嘴角下垂**的人，天性可能较为**悲观、怕麻烦**，不擅长迎接挑战，适应能力较差，容易有**抑郁、消极、固执**的倾向。

## 嘴部动作与心理状态

除了静态形状，嘴部的动态更是内心情绪的直观反映。嘴巴通过**张开闭合、向上向下、向前向后、抿紧放松**四种基本运动方式，能组合出丰富的表情。

以下是几种典型的嘴部动作及其含义：

*   **把嘴抿成“一”字形**：表明此人性格**坚强，是实干型**。交给他的任务通常能圆满完成。
*   **嘴角稍稍向上**：这种人通常**头脑机灵，性格活泼外向，心胸开阔**，容易相处。
*   **交谈时嘴角微微向后**：表明他正在**集中注意力倾听**，但意志可能不太坚定，容易受外界影响。
*   **下嘴唇往下撇**：表明他**并不相信**对方所说的话，并想寻找证据反驳。
*   **上下嘴唇一起往前撅**：表明心理可能正处在某种**防御状态**。
*   **嘴角老是向下撇**：性格可能比较**固执、呆板、内向**，很难被说服。
*   **牙齿咬住嘴唇或双唇紧闭**：说明正在**用心倾听**，或在心里**分析、对照、反省**。
*   **说话时用手掩口**（常见于女性）：性格可能较为**内向、保守**，甚至有点**自闭**，对陌生人存有**戒心**。

## 笑容中的性格密码

笑是最典型的嘴部动作，不同的笑法能揭示不同的性格。

以下是几种常见的笑容类型：

*   **狂笑（嘴角猛向上方翘）**：这类人通常**精于社交，性情温和**，让人感到亲切，有**冒险精神，作风积极**。善于处理繁杂事务。
*   **开口大笑（嘴角平开）**：性格**粗犷，不拘小节，行为大方**，但可能**缺乏耐心**，容易知难而退。
*   **微笑时嘴角稍下垂**：性格较为**内向，不善言辞**，但**注重细节**，喜欢分析。不足的是做事**持久度不够**，容易半途而废。
*   **眯眼笑（嘴角向下，几乎不张口）**：性格**倔强**，对身边的人**不够坦诚**。他们可能**多才多艺，有理想**，但**不喜欢与人合作**。

## 其他言语相关的微表情

说话时的伴随动作也能提供线索。

*   **口齿不清、说话迟钝**：可能语言能力一般，其他方面也较平庸；也可能是**不喜表现**，但拥有某方面**出众才能**的人。
*   **说话经常舔嘴唇**：很可能在压抑内心的**兴奋或紧张**，因而感到口干。
*   **说话时清嗓门、声音变调**：说明对自己说的话**没有把握**，有**杞人忧天**的倾向。
*   **男性咬住烟头并用唾液湿润**：可能是**心理不成熟**的表现。
