# 075.透视挣钱花钱背后的心理学，手把手带你实现财富自由：第07讲：提高收入的第一步_note

在本节课中，我们将学习如何通过调整收入结构、积累成功体验和主动投资自己，来迈出提高收入的第一步。我们将跟随一位咨询师的真实成长经历，理解收入增长背后的核心逻辑。

## 📈 收入结构的演变

上一节我们探讨了财富观念，本节中我们来看看个人收入结构如何随时间演变。一位咨询师分享了他的经历：在三年左右的时间里，他的咨询费从300元提升到了800元，实现了近三倍的增长。

这个增长的核心在于他认识到：**收入与努力和勤奋成正比**。他掌握了一个关键技能：**大量练习**。这个过程必然伴随着时间和精力的持续投入。

## 🔑 核心增长策略：大量练习与主动出击

当同行每周只花几小时做心理咨询时，他可能投入了一二十个小时。在讲课方面，他的策略也与众不同：只要有机会就去讲，甚至愿意免费讲授。

以下是实现收入增长的两个关键行动：

*   **大量接个案，不挑剔**：专注于如何通过每一个案例提升自己，而非选择案例。
*   **主动创造讲课机会**：通过免费或低价授课来积累经验，因为**挣钱能力与经验成正比**。

随着经验积累，他的讲课能力得到认可，课时费达到500元一小时，并开始为苹果、微软等世界一流企业进行内部培训。他意识到，这一切选择都源于当年对自己的信任和“我还可以讲课”的信念。

## 🧠 突破自我限制

在成长过程中，他从不给自己设限。他给自己的唯一准则是：

```text
你喜欢吗？你感兴趣吗？你想做吗？如果想，那就赶紧行动。
```

这种心态让他能够跨界发展，虽然学习心理学，但可以讲授信用、关爱、财富等多个主题，同时还能进行心理治疗。

## 📊 收入结构的多元化

那时，他的收入结构发生了根本性变化：
*   工资收入占比降至**30%**（尽管工资已翻倍）。
*   心理咨询作为主业，占比**50%**。
*   培训讲课占比**17%**。
*   写稿收入占比**3%**。

看到这个结构，他感受到了巨大的进步和成功体验。这引出了一个重要概念：当我们尝试新领域时，需要为自己积累**足够多的成功体验**。这种体验来自于对新事物不断尝试的**累加过程**，而非一次性的侥幸。

## 🎯 实践练习：分析你的收入结构

理论需要实践。现在，让我们来做一个具体的练习。

以下是练习步骤：

1.  准备一张纸和一支笔。
2.  写下你当前每个月的收入来源及其比例。
3.  写下你**潜在的**收入来源（即你现在还没做，但未来有可能做的事）。
4.  思考如何实现这些潜在收入：是提升技能，还是付出更多时间或金钱进行投资？

当你看到潜在收入来源的比例逐渐上升，现有收入也在增长时，你会获得强烈的成就感。持续活在成就感中，你自然会变得越来越富有。

## 💪 主动掌控 vs. 被动等待

最重要的是，这个过程让你从被动等待（如等待老板发奖金、等待项目赚钱）转变为主动掌控。你清楚地知道该如何努力、如何选择、如何分配时间和精力。

**收入的核心，在于“要不要做”，而非“难不难做”**。所有事情开始时都难，但练习到一百遍、一千遍时，就会变得轻而易举。

## 📝 总结

本节课中，我们一起学习了提高收入的第一步：

1.  **调整心态**：相信收入与努力成正比，并愿意通过**大量练习**和**主动出击**来积累经验。
2.  **突破限制**：不给自己设限，跟随兴趣和热情立即行动。
3.  **优化结构**：分析并规划你的收入来源，使其向多元化、高价值方向发展。
4.  **积累体验**：通过持续尝试和微小进步，为自己积累**成功体验**，获得成就感。
5.  **掌握主动权**：从被动等待转为主动规划和投资自己，清晰知道每一步该如何走。

从今天起，就像示例中的咨询师一样，充分相信自己也能做到。现在就开始分析你的收入结构，看看你的时间和金钱可以投向哪里，为财富增长迈出坚实的第一步。我们下次再见。
