# 032.情感心理咨询技能大全（：第11章：夫妻咨询—安洁：08做好夫妻关系调试的两个层面

在本节课中，我们将学习如何辅导夫妻发展友情和处理冲突，这是改善夫妻关系的两个核心层面。我们将探讨具体的方法和技巧，帮助夫妻建立更紧密、更健康的关系。

上一节我们介绍了夫妻咨询的整体框架，本节中我们来看看如何具体辅导夫妻进行关系调试。

## 发展友情：将伴侣变成好朋友

在辅导夫妻时，首要任务是帮助他们发展友情，将对方视为好朋友。友情是亲密关系的基石。

以下是一个关键数据：
如果一对夫妻在生活中未能发展出友情能力，他们的平均婚龄约为**16.4年**，之后便会丧失亲密感。

在亲密关系的三大要素——**亲密、激情和责任**中，当亲密感丧失时，夫妻会感觉彼此疏远，关系可能变得像“上下铺的兄弟”。

若未能发展出处理冲突的能力，婚姻平均维持时间约为**5.2年**。

### 建立“爱情银行”观念

发展友情的第一步，是帮助夫妻建立“爱情银行”的观念。每个人的心里都有一个爱情银行，需要有意识地向其中“存款”。

以下是存款与提款的定义：
*   **存款**：让对方感到开心、被欣赏和肯定，使其感受到爱。经常做这样的事就是存款。
*   **提款**：让对方感到痛苦、被批评、误解或伤害。吵架、批评对方父母等行为都属于提款。

在恋爱或新婚阶段，夫妻存款通常最多。关系疏远时，除了外部事件影响，往往是因为双方都在“提款”。

在存款与提款的比例上，有一个通用原则：如果做了一件破坏关系的事（提款），就需要做**五件**修复关系的事（存款）来弥补。

在咨询中，可以引导夫妻进行“存款练习”。例如，询问双方在生活中欣赏对方的哪些特质或行为。这属于增加关系凝聚力和爱意的步骤。

### 理解性别差异：女性如何感受爱

在夫妻关系中，女性和男性感受爱的方式常有不同。理解这些差异至关重要。

女性通常更需要倾诉感受，而男性在倾听时，需要克制立即提供解决方案的冲动。许多夫妻并不了解这一点。男性可能将妻子的倾诉视为抱怨，若不倾听，抱怨可能升级为指责。

作为咨询师，需要充当“翻译”，帮助夫妻理解：女性很多时候需要的是**倾诉和陪伴**。如果男性能够给予倾听的空间，女性的情感需求就能得到满足。

**倾听示例**：
一位女性在咨询中表达了对疾病的焦虑，例如担心流鼻血或头痛是重病征兆。她的先生回应：“没事，都是你想多了，少想一点就没问题。”

未经训练的人可能觉得这个回应合理。但在互动模式中，我们看到：妻子表达了恐惧（情感需求），丈夫用理性方案“怼”了回去，暗示是妻子“有问题”。这导致妻子闭嘴，情感互动失败，妻子可能变得更焦虑，并用更激烈的方式（如“我死了你会怎样”）表达，最终引发争吵。

实际上，妻子是在表达：“**我很害怕**。”如果先生能倾听：“我听见了，你挺害怕的，害怕身体出问题，也害怕生活受影响，我理解。”——这样，妻子的情绪就能得到安抚。

关键在于，先生是否有能力处理太太的情绪。很多时候，女性需要的不是解决方案，只是**倾听**。咨询师需要过滤掉妻子表面的焦虑，翻译出其内在的害怕，并指导丈夫如何安抚妻子，建立新的互动模式。经常这样做，女性就会感到被爱，关系会更紧密。

### 了解“爱的五种语言”

在建立友情的过程中，一个重点是帮助夫妻了解并运用对方“爱的语言”。我们很容易陷入误区：因为我爱你，所以把我认为最好的给你，但那可能并非对方所需。

有一个比喻：妻子每天给丈夫吃苹果，认为苹果最好，自己也最爱吃。但丈夫在咨询中说：“我从来不爱吃苹果，只想吃一个梨，但她从不给我。”这就是没有使用对方爱的语言。

**爱的五种语言**如下（无先后顺序）：
1.  **肯定的言词**：口头上的肯定、赞美和认同。例如：“我爱你”、“你很棒”、“这个家打理得真好”。
2.  **精心的时刻**：全心全意陪伴对方的时刻。例如：一起庆祝纪念日、专心约会、看电影。核心在于“用心”。
3.  **接受礼物**：用礼物表达爱意。礼物不一定要贵重，可以是鲜花、卡片，或是将财政大权交给对方管理等。
4.  **身体的接触**：拥抱、亲吻、牵手、性生活等。
5.  **服务的行动**：为对方做事情。例如：洗衣做饭、倒茶、准备洗澡水。

每个人主要爱的语言通常是其中一两种。做这一两种，对方就能感受到爱；做其他的，对方可能收不到爱的信号。例如，丈夫每年送妻子昂贵的包，但妻子可能只想要生日时的一朵玫瑰花。

因此，建立友情时，首先要**了解对方爱的语言**，然后“投其所好”，而非“给己所要”。这项工作需要在咨询室中进行。当夫妻很久没有“投其所好”时，关系就可能“债台高筑”，陷入恶性循环。

## 处理冲突：建立解决矛盾的能力

发展友情之后，我们需要学习如何处理冲突。处理冲突的能力直接关系到婚姻的持久性。

### 1. 先处理情绪，再处理事情

当夫妻情绪高涨、无法继续沟通时，可以建议暂停。如果双方能力尚可，则运用**倾听技术**来处理情绪。

如前文死亡焦虑的例子，先生不需要给解决方案，只需倾听：“我听见了，你很害怕。”这就是很好的倾听。情绪平复后，冲突才有可能解决。

倾听技术是伴侣需要在咨询室中演练的部分。可以设定规则：一人为发言人，一人为倾听者，每人五分钟。如果难以克制，可以借用“谈话棒”（如一支笔），拿到笔的人说话，没拿笔的人倾听。咨询师先辅导演练，再布置为家庭作业，让夫妻回家练习。

### 2. 一次只处理一个问题

夫妻关系的问题往往是长年累月多种矛盾造成的。咨询师的工作重点不是替他们解决所有具体问题（如孩子教育、金钱、婆媳关系），而是**协助他们建立自己解决冲突的能力**。

方法是：规定每次只讨论**一个**未解决的矛盾。先从矛盾张力较小的问题开始练习，例如周末安排、暑假计划，避免一开始就讨论“婆婆同住”、“谁管钱”等敏感话题。

在练习中，观察他们是否能成功讨论。这能帮助他们学习用**合作**而非**竞争**的方式沟通。咨询师可以协助他们：当讨论跑题或翻旧账时，其中一人需保持相对清醒，喊停并重申“本次只讨论这个问题”。目标是以不指责、不批评、不污蔑、不冷战的方式进行讨论。

在咨询室中，常让夫妻坐在双人沙发上，自由讨论一个平时难以讨论的问题（如孩子教育）。一旦他们陷入旧有模式（批评、指责），咨询师就要干预：“看，旧模式又出现了，停一下，重新试试好好讲话。”引导他们直面压力和冲突，一遍遍练习处理单个问题的能力，并将这种体验带回家。

### 3. 建立一致性沟通（新的互动模式）

一致性沟通是指表达自己的感受和需要，而不指责或命令对方。它需要不带情绪，用理智系统进行沟通，不太容易，但至关重要。

**一致性沟通的话术模板**：
1.  **描述具体事件**：“当……的时候”（陈述事实，不评判）。
2.  **表达自己的感受**：“我感到……”（如焦虑、着急、无力、不被理解）。
3.  **将感受形象化（可选）**：“那种感觉就好像……”
4.  **表达自己的需要**：“我需要……”（如被支持、了解、休息、空间）。

**示例**：
先生可以这样对太太说：“当我日以继夜工作仍无法赶在期限前完成，而你抱怨我没有关心家里时（事件），我感到非常焦虑、着急、无力和不被理解（感受）。那种感觉就好像骆驼已经装满了重物，别人还要往上加箱子（形象化）。我需要更多的被支持和了解，也需要一些休息和空间（需要）。”

需要注意的是，一致性沟通不是“魔法”。表达了需要，对方不一定能满足。但“表达”与“不表达”、“如何表达”结果不同。咨询师自身需具备一致性沟通的能力，并在夫妻沟通卡住时进行示范。例如，咨询师可以扮演丈夫或太太的角色，演示如何沟通，然后再让夫妻自己尝试。

避免一个误区：不要让夫妻回家后刻意地宣布“我们现在来进行一致性沟通”，这通常会把对方吓跑。一致性沟通是需要在生活中不断练习、内化的新沟通模式，而不是刻意的表演。

### 4. 对冲突进行“头脑风暴”

当双方对某个问题意见不一时，可以引导他们进行“头脑风暴”。这个方法能将人从情绪层面带到理智层面。

具体做法：让夫妻各自拿一张纸，写下针对某个冲突的所有可能解决方案（例如各写10条）。然后一起看看这些意见如何综合，达成一个共赢的方案。这同样是教导他们一种回家后可以自行使用的解决冲突的工具。

---

本节课中，我们一起学习了夫妻关系调试的两个核心层面：**发展友情**和**处理冲突**。

在发展友情方面，我们介绍了建立“爱情银行”的观念、理解性别差异（特别是女性对倾听的需求）、以及学习和运用对方“爱的五种语言”（肯定的言词、精心的时刻、接受礼物、身体的接触、服务的行动）。

在处理冲突方面，我们掌握了四个关键方法：**先处理情绪再处理事情**（运用倾听技术）、**一次只处理一个问题**（从易到难练习）、**建立一致性沟通**（表达感受和需要而非指责）、以及对分歧进行**头脑风暴**以寻求共赢方案。

这些方法和技巧都需要在咨询师的引导下，于咨询室中进行体验和练习，然后作为家庭作业让夫妻带回到日常生活中不断实践，从而逐步建立起健康、持久的新互动模式。
