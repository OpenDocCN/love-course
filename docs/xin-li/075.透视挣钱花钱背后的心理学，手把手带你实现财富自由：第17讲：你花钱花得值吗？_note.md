# 💰 透视挣钱花钱背后的心理学，手把手带你实现财富自由：第17讲：你花钱花得值吗？

在本节课中，我们将跟随心理专家丁丽老师，学习如何审视自己的消费行为。我们将探讨如何区分“消费”与“投资”，并学会让每一分钱都花得更有价值，甚至能为我们创造更多财富。

---

消费有多种形式。吃进去、喝进去的东西也算消费。但关键在于，这种消费是会损害你，还是对你有益？很多人在消费过程中，追求的是一种虚荣感或他人羡慕的眼光，而非真正对自己好。

就像毒品，它难以获得，吸食可能让人感觉“很老大”，但它对身体是巨大的消耗。同样，如果你正在使用某些消耗你生命的东西，无论它在别人眼里多么昂贵、有价值，对你而言都可能毫无意义，甚至是有损伤的。

除了投资首饰等物品，我们来谈谈“吃”。我发现周围有些朋友在吃上没有节制。这不光体现在吃胖后的难过，更在于有些食物本身对身体并无好处。

例如麻辣小龙虾，我也爱吃，但我会节制自己，一个月最多吃一次。有的朋友则在上市季节拼命吃，这其实是对身体的一种损伤，包括吃辣等。这些吃进去的东西本身是消费，但你要思考：你消费的是它带来的快乐，还是它在消耗你？

这说明了，同样是在花钱，结果却可能截然不同：是让你变得更好，还是更糟？这部分需要我们特别注意。此外，还要思考：这笔花费是帮你**保值**、**增值**，还是仅仅在**贬值**，甚至什么也没留下？

如果你能区分这几个小概念，在花钱时就会更清醒：当然要对自己好，那就买对身体有益的健康食品。在寻求保值增值时，就会想：我不仅要买得高兴，还要买到能升值的东西。

这样，就能帮助我们在这个过程中，学会**让钱帮你挣钱**。

对于理财，我再简单补充一点。我觉得像没有投资经验的“小白”用户，可以先试试银行的理财产品。它们的收益率可能不高，但能帮你**存下钱**。

理财产品最棒的地方在于，它有门槛，比如5万、10万起购。为了凑够5万，你就会开始存钱；然后又会想存够10万，因为10万档的收益通常更高。我就是靠理财产品存下钱的。

因为我想获得更高的收益，就需要更多本金。怎么办？就把钱存进去。当本金逐渐增加时，感觉会非常好，像是“小金库”增加了。

刚才我们讨论了买饰品和关于“吃”的消费。考虑到听课的女性朋友可能不少，我在此再提供一个方法。如果你很年轻，可以在这方面进行投资：**投资在你的皮肤护理上**。

是的，要保持“少女感”。我常跟朋友讲，很多粉丝见过我，知道我做情感节目16年了，能算出我的年龄，但他们会觉得我皮肤状态不错，看起来蛮年轻。

我大学毕业后第一个月的工资只做了两件事：第一，给妈妈买了一块卡西欧手表，当时600多块，对我月薪1000多来说挺多了；第二，剩下的钱买了大牌护肤品。

我从20岁参加工作到现在，从未把自己的脸当成“调色盘”，也从不看那些针对“20岁少女”、“30岁少女”的广告。我只认大品牌，抹在脸上的东西必须非常过硬。

这么多年过去，我只去过三次美容院，没有做过任何微整形。我的护肤方法很简单，不是说吃清淡（我吃得挺油腻），也不是空谈保持好心态或坚持锻炼。

我就告诉你：使用值得信赖的、信得过的**大品牌护肤品**。注重三件事：**清洁、保湿和防晒**。每天坚持做。20年后，你的皮肤会更好。任何东西都需要一个累积的过程，包括习惯的养成。

所以现在我想提醒大家：今天我们做的一个决定，是需要时间累积的。不要三天热度，听完课做一做就放弃。**钱是与时间相关的**。将钱放入时间中，它会给你带来惊人的回报。

而这个回报，取决于你是否有足够的耐心。就像汇丰银行一样，如果你懂得它，就明白金钱也是如此：它需要你小心呵护、持续投入、帮它“争值”，然后它就会给你想要的回报。

今天，我们教大家如何从方方面面学会花钱，进而学会挣钱。还记得之前在课程中，丁老师提到的钱要花在什么地方吗？

以下是四个关键方向：

1.  **花在感恩的部分**。
2.  **花在体验的部分**。
3.  **花在个人成长的部分**。
4.  **花在能让“钱生钱”的部分**。

我相信，跟课到现在的小伙伴们，已经对财富与自身的关系有了更深的了解。我们后面可以专门探讨：为何我们会感到匮乏，又如何走向丰盛？因为这是内心的、心理的改变，之后才能构建起外在的财富。

如果我们只讲方法（“数”的层面），而不触及内心（“道”的层面），你会发现有限制，仍会有很多问题和困惑。

从下一次课开始，我们将进入新主题：**你要养成哪些习惯，才能逐步富裕起来**。

**本节课总结**：我们一起学习了如何审视消费，区分了“消耗型消费”与“投资型消费”。核心在于，花钱时要思考它是让我们**变好**还是**变糟**，是能**保值增值**还是仅仅**贬值**。我们探讨了在饮食、护肤等方面的具体应用，并复习了金钱应该投入的四个方向：感恩、体验、成长和钱生钱。记住，财富积累需要耐心和时间，今天的明智决定将在未来带来丰厚回报。
