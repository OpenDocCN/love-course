# 谈判课102-004：谈判定势——是谁在做决定？🤔

在本节课中，我们将学习谈判中的一个核心基础工具：判断“谁在做决定”。理解决定权的归属，是区分谈判、说服等不同情境，并制定正确策略的第一步。

上一节我们介绍了托马斯-基尔曼冲突模型量表，它帮助我们理解人的行为模式。本节中，我们来看看如何对“事”进行更清晰的区分，关键在于分析决定权的归属。

## 决定权的三种归属

冲突解决理论将“谁做决定”分为三种基本状况。理解这些状况，能帮助我们在面对具体情境时做出准确判断。

以下是三种决定权归属的详细说明：

1.  **对方做决定**
    事件如何解决，其主要的资源和权力掌握在对方手中。市场中的大多数销售场景就是典型例子，决定权（买或不买）在消费者手中。

2.  **决定权被双方分享**
    事件的解决需要双方共同参与，彼此相互需要。这是共识性合作得以展开的前提。公式可以表达为：**合作可能性 ∝ (我方需求度 × 对方需求度)**。

3.  **决定权在他方**
    由与此事无直接利害关系的第三方来做最终判断。法庭是最典型的例子，法官作为第三方对是非曲直做出裁决。

这些观念并不抽象，还原到生活中是无比具体的。接下来，我们用一个现实场景来应用这个分析工具。

## 案例分析：如何与老板谈加薪？

假设你需要与老板谈判加薪或升职。除了准备事实资料和目标愿景，最核心且初步的环节是判断：**谁拥有决定权？**

这个判断将直接导向不同的策略准备。以下是两种可能的情况分析：

*   **情况一：你是可替代的员工**
    如果你的工作不具有不可替代性，那么决定权及相关资源几乎全部属于你的上司。此时的对话更接近“说服”情境（你求对方），而非平等谈判。错误地将其视为谈判，可能导致你在不必要时展示强硬，从而破坏对话。

*   **情况二：你是不可或缺的核心成员**
    如果你对公司有卓越贡献且不可替代，那么公司同样需要你持续努力。此时，双方彼此需要，都拥有 **`bargaining power`**（讨价还价的筹码）。这便进入了真正的“谈判”情境，双方可以进行资源交换与利益整合。

对局势判断的错误会导致严重后果。将“说服”误判为“谈判”，会无故示强；将“谈判”误判为“说服”，则会做出不必要的让步，损害自身利益。

## 谈判的真义与时代价值

权力在谁手上？谁求谁？这是我们采取任何具体策略的逻辑与行为前提。由此，我们可以更深刻地理解谈判的本质。

所谓的谈判，其本质是人与人之间因相互需要而产生的互动。**`Bargaining power`** 是核心：我有你需要的筹码，你有我需要的利益，于是我们合作、交换、融合，实现真正的利益双赢。

因此，谈判是一种**决定资源与决定权利被分享的决策形态**。这不仅是书面定义，更是谈判的真精神。

正是因为谈判具有这种彼此交换、相互配合的属性，它与21世纪社会架构的基本原则——协同与合作——密切相关。如果你相信未来世界更依赖团队协作而非个人单打独斗，那么学习谈判，就是拿到了打开协同时代大门的钥匙。

学习谈判，重要的不仅是话术，更是协作的精神与方法。

## 本节总结

本节课中，我们一起学习了谈判的“定势”——对事情性质的判断。我们深入探讨了决定权的三种归属（对方、双方、第三方），并通过加薪案例分析了如何运用此判断来选择策略（谈判或说服）。

**核心在于：判断权力在何处，是准备与进入谈判的第一步。** 只有看清局势，才能流畅运作谈判。
