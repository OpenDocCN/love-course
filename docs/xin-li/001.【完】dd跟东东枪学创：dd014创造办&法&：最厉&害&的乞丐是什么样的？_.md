# 课程001：跟东东枪学创意30讲 📚

## 第014讲：创造“办法”💡——最厉害的乞丐是什么样的？

在本节课中，我们将学习创意金字塔的最高层级——“办法”。我们将通过乞丐的故事，理解“办法”如何超越“说法”和“看法”，促成全新的认知与行为改变。

上一讲我们介绍了从“说法”到“看法”的升级，但创意的攀登并未结束。本节中，我们将探讨是否存在比“看法”更高层级的创意突破。

当我们绞尽脑汁，在“说法”和“看法”的层面上竭尽全力仍无法解决问题时，该怎么办？听完今天这一讲，你就会知道答案。我们仍从那个盲人乞丐的故事讲起。

### 从“说法”到“看法”的回顾

我们之前说过，盲乞丐面前用来乞讨的牌子，从“自由施舍”改为“春天来了，我却看不见”，这是“说法”的优化，因为它依然在用失明的痛苦博取同情。

现在假设这条街上来了另一个盲人乞丐。他也要在此乞讨，他那块牌子上该写什么？如果他是个普通的乞丐，或许会找一个更动人的“说法”，把失明的痛苦描述得更深刻。

但如果他是一个听过我们课程的乞丐，他就可以努力在“看法”的层面上寻找突破。例如，牌子上可以写：“我看不见的不只是春天，还有希望。” 这意味着，盲人的人生充满了各种缺憾和损失——看不见希望、机会、公平和信任，这比看不见春天的风景更悲惨。所以，请把钱给我，别给旁边那个。

这就是上节课所说的“看法”层级的突破。

### 引入“办法”：第三位乞丐的启示

然而，真正的问题现在才出现。这时，街上来了第三位乞丐。这位乞丐有点特别：他不是盲人。那两位盲人乞丐可以靠失明博取同情，你该怎么办？你很困难，因为你没有残疾，只有150度的近视。在博取同情上，你怎么比得过人家？

以下是现实中乞丐的做法：

多年前，我去过一次非洲。一次等红灯时，隔着车窗看见几个乞丐举着牌子乞讨。他们的牌子上写着：
> 你好，请帮助我，我很穷。但我宁愿祈祷，也不愿意去偷、去抢。你的帮助会让这个世界上少一个小偷或强盗。谢谢你。

实话实说，当时我第一次出国，也是第一次看见外国的乞丐。看到他们用这样一套话术来求人施舍，那一瞬间我非常感动。但车又开了几个路口后，我想明白了：这哪里是好人？这分明是用“去偷去抢”在威胁我们。世界上又不是只有祈祷和偷抢两种生活方式，对不对？你不劳动，在这要饭，还挺自豪。

但你会发现，这个乞丐不是希望别人因为同情而施舍，而是希望别人因为**支持他的选择**（赞赏他遵纪守法不作恶），或是**因为受到他的威胁**（为了避免世界上多一个小偷强盗）而掏钱给他。

区别在哪里？用我们之前的理论体系来梳理一下：
*   从“自幼失明”到“看不见春天”，是**说法**的变化。
*   从“看不见春天”到“看不见希望”，是**看法**的变化。因为本质上，两个盲人乞丐都是为了获取人们对失明的同情，他们促成的认知改变是一样的，只是使用的理由不同。
*   而这位非洲乞丐请注意，他要的并不是同情，而是**赞赏**或**拯救**。他已经跳出了对“失明”的同情，促成了一个**完全不同的、新的认知改变**。

### “办法”的定义与价值

现在，我们可以回答开头提出的问题了。如果我们寻求比“看法”更高层级的突破，那么我们就不能只寻找促成认知改变的**新理由**，而应该去探索：**有没有新的、可以促成行为改变的认知改变？**

这个不一样的认知改变，我称之为 **“办法”**。

是的，比“说法”更高级的是“看法”。比“看法”再高一个层级的，就是今天我们说的 **“办法”**。

上一讲我们谈到，找到一个新“看法”可以让你的创意独辟蹊径。而“办法”是更高层级的独辟蹊径。一个好的新“办法”，有时甚至可以帮助你开辟出一片新的市场。因为你期待的认知改变与竞争对手不同，你针对的也可能是另一群完全不同的人。这些人是别人没有争取到或放弃了的，而你却可以用一个新的“办法”去争取到他们。

刚才说的那位以“不偷不抢，一心乞讨”为荣的外国乞丐，被他打动而掏钱的人，很可能从来不给那些卖惨博取同情的乞丐掏钱。为什么？因为他们对老弱病残无感，缺乏同情心。但是，他们却有可能非常欣赏“自力更生”、“人穷志不短”这种正能量。对于这种人，原来的“办法”是无效的。

### 商业案例中的“办法”

许多商业品牌和广告，也因为找到了一个新的“办法”——一个新的认知改变的可能——而获得了巨大成功。

**案例一：安飞士租车**
当排名第一的租车公司在说“我们是第一，更多人选我们”，鼓励大家从众时，安飞士却在努力塑造一个新认知：**你不该选择第一名的租车公司，而应该选择第二名，因为第二名排队比较短。** 这促成了一个从“选第一”到“选第二更高效”的认知改变。

**案例二：某钢琴品牌**
一个对家长宣传“学琴的孩子不会变坏”的钢琴品牌，也找到了一个新的“办法”。许多钢琴品牌都在宣称钢琴能让孩子变成钢琴大师，通过鼓励一个远大的艺术梦想来让家长买琴。而这个品牌想要实现的认知改变却是：**它想让你知道，哪怕你并不希望孩子成为钢琴家，你也该给他买台钢琴，因为这可以让他变成一个更好的人。**
言外之意是：哪怕孩子钢琴弹得不好，你也不该埋怨，因为你的孩子不会变坏了。这可谓一举两得。

**案例三：甲壳虫汽车**
广告史上经典的甲壳虫汽车案例。当年甲壳虫在美国汽车市场上是个绝对的另类。广告大师大卫·奥格威曾说过，甲壳虫像是个“被整过形的行李箱”，并说如果有人让他给这车做广告，他非得投降不可。
但是，比尔·伯恩巴克等人却为这辆车找到了一个非常独特的“办法”。他们不像别的车一样宣扬豪华宽敞，而是试图说服当时的美国人：**小有小的好处。** 他们的广告大标题只有两个词：**Think Small**（想想小的好处）。画面上甲壳虫的形象也被印得小小的，出现在角落。
大卫·奥格威后来评价说，这个广告让当年底特律的汽车显得粗俗蠢笨，甲壳虫因此受到那些**不喜欢张扬的美国消费者的拥戴**。

请认真听一下这句话：“受到那些不喜欢张扬的美国消费者的拥戴。” `Think Small` 这个“办法”之所以成功，正是因为有这么一群“不喜欢张扬”的美国消费者。别的汽车品牌没有注意到，或一直忽视了这群消费者。而这群广告人注意到了这些从未被关注过的人和从未被说出来的需求。`Think Small` 引发了他们的认同，让他们意识到：“对呀，谁说要买大汽车？我根本不需要也不喜欢那些大笨车，小有小的好处啊！”

这个“不喜欢张扬”是什么？没错，还是**洞察**。发现新的“办法”也要靠准确的洞察。我想现在你应该能充分了解我们之前为什么再三强调洞察的重要性了。

### 构建“Idea金字塔”

办法、看法、说法。你看，我们用了好几讲的时间，总算把这三个词全都介绍出来了。

总结一下它们的关系：
*   **办法**：指我们为促成**一个行为改变**，而期望实现的**那个认知改变**。公式可理解为：**办法 -> (促成) -> 认知改变 -> (导致) -> 行为改变**。
*   **看法**：指我们为促成这个认知改变，而要传递给目标人群的**那个理由**。只要他们接受认同这个理由，认知就会改变。
*   **说法**：指我们为了传达那个看法（理由）而找到的**表达方式**。用这种方式表达，看法就能更好地被接收和认同。

你一定看明白了：**说法为看法服务**，是为了让看法更好地被接收。**看法为办法服务**，是为了让办法能真正地实现。

如果把这种关系落实到图表里，我们就得到了我之前承诺过的那个神奇工具——我称之为 **“Idea金字塔”**。

这是一个分成三层的金字塔：
1.  **最上层：办法**（策略Idea）
2.  **中层：看法**（创意Idea）
3.  **最下层：说法**（执行Idea）

为什么要画成金字塔形呢？当然有原因：
1.  **自上而下的发散性**：一个“办法”可以找到多种不同的“看法”来支持；一个“看法”也能激发出无数种不同的“说法”来表达。
2.  **自下而上的负责关系**：“说法”对“看法”负责，“看法”对“办法”负责。层层承载，层层负责。
3.  **自上而下的重要性层级**：**办法胜于看法，看法胜于说法**。多少好“说法”也比不过一个好“看法”，多少好“看法”也比不上一个突破性的、有效的好“办法”。

举个例子：我在网上看过民国时期国民党军队的征兵传单，印刷精美，字体规整，大字写着“国家兴亡，匹夫有责”。有趣的是，网上也能看到一份红军征兵的“传单”（其实算不上一张正式的传单），只是一张特别简陋的字条，上面是手写的一行小字：“老乡，参加红军可以分到土地。”
怎么样？你看明白这两种“办法”的区别了吧？
