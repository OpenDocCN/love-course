# 课程078：人际交往2期 - 饭局潜规则 🍽️

在本节课中，我们将要学习《中国饭局潜规则》一书的核心内容。饭局在中国文化中远不止是吃饭，它是一个重要的社交“局”，关乎人际关系、职业发展甚至人生机遇。我们将通过具体的故事和分析，系统地了解如何筹划与参与一场成功的饭局。

## 概述：饭局之“局”的重要性

所谓饭局，重点不在“饭”，而在“局”。吃饭只是一个形式，核心是通过这个“局”来达成特定目的。一场饭局有时可以改变一个人的一生，或直接影响职业生涯的成败。

为了让大家理解其重要性，我们先来看几个故事。

以下是三个因饭局处理不当而导致负面结果的真实案例：

1.  **赵小姐的故事**：她因不谙点菜之道，在为公司预订宴席时搭配不当，造成浪费，最终失去了秘书工作。
2.  **东北企业的故事**：一家企业在与美国公司洽谈合作时，设下过于奢华铺张的宴席，美方代表认为其作风浪费，最终拒绝了合作。
3.  **李小姐的故事**：她在与相亲对象约会时，因点菜和结账方式不当（或过于主动，或不够体谅），多次错失姻缘。

以上三个故事是我们生活中经常遇到的场景。无论是谈恋爱、谈业务，还是维系同事、家人、朋友间的感情，饭局在现代社会中都是必不可少的工具。

看似简单的饭局包含了大学问，例如：如何成功邀请客人、选择何种就餐环境、如何点菜让大家吃得开心、如何敬酒以及如何结账等。了解饭局中的学问与潜规则，是现代人的必修课。

上一节我们认识了饭局的重要性，本节中我们来看看如何设计一场完美的饭局。作者将其概括为七个关键部分。

## 第一部分：请客前的准备工作 🗓️

在发出邀请之前，必须进行周全的考虑，以确保饭局顺利进行。

以下是请客前需要考虑的几个关键事项：

*   **尊重民族与饮食禁忌**：如果客人中有少数民族，必须提前了解其饮食禁忌和风俗习惯。
    *   **例如**：回族人忌食猪肉，且一般不抽烟饮酒；蒙古族喜食奶和肉；维吾尔族饭前饭后必须洗手，且忌甩手。
*   **考虑地域口味差异**：中国东西南北饮食差异巨大。
    *   **公式**：`清淡（粤）≠ 辛辣（川、湘、赣）≠ 咸鲜（华北）`
    *   需根据客人的地域背景调整菜品口味。
*   **选择合适的餐厅**：一个基本规则是，可以请熟悉的朋友去不熟悉的餐厅尝鲜；但请不熟悉的客户或重要客人时，应优先选择你熟悉的、确保菜品和服务质量的餐厅。
*   **照顾特殊人群**：如果宴请中有老人或小孩，需考虑他们的喜好和忌口，例如老人是否有高血糖、高血压等。

除了上述注意事项，如何发出邀请并确保对方接受，是大部分人都会遇到的难题。

## 第二部分：如何成功发出邀请 📞

邀请被拒会令人尴尬。针对不同的邀请对象，应采用不同的策略。

以下是针对不同对象的邀请方法：

*   **邀请领导**：以“敬”为先。邀请前必须找到一个恰当的理由，如请教问题、庆祝项目完成等。关键在于自然、投其所好。
    *   **示例**：“经理，我正好有些行业问题想向您请教，您经验丰富。听说公司对面新开了一家不错的江苏菜馆，您看今天下班后能否边吃边聊？”
*   **邀请客户**：以“诚”为先。有四种常用方式：
    1.  **电话邀请**：直接说明原因、时间、地点及参与人员，突出宴会价值。
    2.  **发请柬**：适用于正式场合，显示尊重。通常需提前两周发出。
    3.  **登门邀请**：适用于已有一定交情的客户，可附带小礼物以表重视。
    4.  **信件邀请**：方式简单，但措辞需诚恳，让对方感到被重视。
*   **邀请同事**：相对简单，但需注意两点：量入为出，注意身份。不要因请客过于奢华而让同事感到压力或觉得你在炫耀。
    *   **常用方法**：开门见山、喧宾夺主（事先调查喜好选择餐厅）、借花献佛（借自己喜庆之事邀请）、步步为营（第二次邀请时使用）。
*   **领导邀请下属**：以“情”为先。最好事先说明原因，如慰劳辛苦，营造轻松的聚餐氛围。
    *   **示例**：“这段时间大家为了项目天天加班，辛苦了。今天我做东，下班后一起聚个餐，放松一下。”

为了更灵活地应对各种邀请场景，作者总结了八种通用的邀请技巧。

以下是八种实用的邀请技巧：

1.  **开门见山法**：直接提出邀请。`示例：“张经理，我们在某某酒楼，过来认识几个朋友吧，等您。”`
2.  **借花献佛法**：借自己的喜庆事由发出邀请。`示例：“今天中奖了，一起庆祝一下！”`
3.  **喧宾夺主法**：事先调查对方喜好，以此为由邀请。`示例：“张主任，听说您公司对面那家烤生蚝不错，离得近，晚上一起去尝尝？”`
4.  **暗度陈仓法**：以其他事情为借口，真实目的是邀请。`示例：“张主任，我这里有份重要文献想给您，晚上吃饭时顺便带给您吧？”`
5.  **声东击西法**：故意拖长交谈时间，顺势在饭点发出邀请。`示例：“您的观点太对了，受益匪浅。这都到饭点了，要不我们边吃边聊？”`
6.  **步步为营法**：第二次邀请同一人时使用。`示例：“上次那家海鲜楼不错吧？最近我又发现一家川菜馆，味道很棒，今晚再去试试？”`
7.  **诱敌深入法**：先聊些轻松话题，再切入邀请。`示例：“张主任，您是东北人吧？我特喜欢东北菜，知道有家店酱骨架做得一流，想想都流口水，晚上一起去？”`
8.  **擒贼先擒王法**：重点邀请关键人物（能做决定的人），而非泛泛邀请一大群人。

掌握了邀请方法，我们再来看看参加饭局时的外在形象管理。

## 第三部分：饭局着装与打扮 👔

得体的着装是尊重他人和场合的表现。这里强调几个容易忽略的点。

**对男士的建议**：
1.  **西装合身**：忌过大过小或花色夸张。
2.  **色彩原则**：遵守“三色原则”（全身颜色不超过三种）和“三一定律”（鞋子、腰带、公文包颜色统一）。
3.  **扣子礼仪**：单排扣可不扣；双排扣可全扣；三粒扣通常只扣上面两粒，最下面一粒不扣。
4.  **袜子搭配**：穿深色西装皮鞋时，忌配白色袜子，应配黑色或灰色袜子。

**对女士的建议**：
*   **公式**：`一个得体优雅的女性 = 三分姿色 + 一分化妆 + 两分服装 + 两分首饰 + 两分手袋`
*   **配饰**：不宜过多，全身不超过三种为佳（如手表、戒指、项链）。
*   **注意事项**：切勿在他人面前补妆或整理衣物，需处理时应去洗手间。

形象准备妥当后，接下来我们进入饭局现场，看看有哪些基本的餐桌礼仪。

## 第四部分：餐桌基本礼仪 🥢

从见面到入座，每一个细节都体现着个人的修养。

**1. 交换名片**：
*   **接收时**：应仔细观看并适当赞美，然后妥善放入上衣口袋或名片夹。切忌塞入裤兜、在手中把玩或随意放在桌上。
*   **递出时**：应起身，双手递上，让名字正向对方。
*   **禁忌**：在陌生人中四处散发名片会显得很不得体。

**2. 寒暄与介绍**：
*   **顺序**：年轻者先向年长者问候，男性先向女性问候，身份低者先向身份高者问候。
*   **称呼**：以对方的最高头衔称呼，如“王董事长”。
*   **自我介绍**：在饭桌上，简洁介绍三个要素即可：`姓名 + 供职单位 + 职务`。可根据情况补充学历、特长或有趣经历。
*   **为他人介绍**：记住“尊者居后”原则。即先把身份较低的一方介绍给身份较高的一方。
    *   **示例**：`“王总（地位高者），给您介绍一下，这位是贵妃老师（地位低者），是读书栏目的创始人。”` 介绍完后，再将王总介绍给贵妃老师。

**3. 入座礼仪**：
*   **入座方向**：应从椅子的左侧入座。
*   **座次安排**（常见圆桌）：
    *   正对门的位置为主陪（主人）。
    *   主陪右手边为主宾（最重要客人），左手边为副宾。
    *   背对门的位置为副陪（协助招待）。
    *   其余客人按尊卑顺序依次排列。
*   **其他细节**：私人物品（手机、钥匙）不要放桌上；坐姿端正；主人不动筷，客人不宜先动（汉族习俗）。

当饭局中有陌生人时，保持微笑是最好的破冰方式，能传递友善，拉近距离。

在用餐过程中，你的吃相也在无声地传递信息。

## 第五部分：用餐过程中的注意事项 🍜

有人曾说：“你怎样品味食物，别人就怎样品味你。”餐桌举止直接影响他人对你的判断。

以下是几点重要的用餐礼仪：

1.  **处理特殊食物**：吃到太烫的食物可快速喝口水；吃到变质食物，应不动声色地用纸巾掩嘴吐出并包好。
2.  **控制不雅动作**：尽量避免在餐桌上打哈欠、咳嗽、擤鼻涕。若无法控制，应用手或纸巾遮掩，或去洗手间处理。
3.  **避免当众剔牙**：牙缝不适可先漱口，若无效应去洗手间处理，切勿在餐桌上公然剔牙。
4.  **敬酒礼仪**：敬酒时，自己的酒杯杯口应略低于领导或长辈的酒杯。
5.  **倒茶与倒酒**：`茶七酒八`。茶倒七分满，以免烫手（茶满欺人）；酒可倒八分满（酒满敬人）。

用餐离不开点菜，这是展现主人细心与诚意的重要环节。

## 第六部分：点菜的艺术 📋

点菜需考虑周全：点少了显怠慢，点多了显浪费。

**点菜技巧**：
1.  **考虑禁忌与健康**：务必考虑客人的民族禁忌、宗教禁忌、健康情况（如素食、高血压、糖尿病等）。
2.  **把握数量与搭配**：`菜量 ≈ 人数 + (2-4)`。需注意荤素搭配、冷热搭配、食材多样性（鸡鸭鱼肉）及口味层次。
3.  **为外宾点菜**：可点一些有中国特色的菜肴，如宫保鸡丁、狮子头等，并注意其普遍口味偏清淡。
4.  **让他人点菜时的礼节**：如果主人客气让你点，应点价格适中、大家都能接受的菜，切忌专挑贵的点。

**酒水搭配**：
*   **与时令搭配**：`冬天 → 白酒；夏天 → 啤酒`。
*   **饮用顺序**：先低度酒，后高度酒；先普通酒，后名贵酒。
*   **特殊搭配**：吃螃蟹时，搭配性温的黄酒可以中和螃蟹的寒性。
*   **倒酒顺序**：主人第一次斟酒，应从主宾开始，按**逆时针**方向进行，最后为自己倒。

饭局接近尾声，结账环节同样考验情商。

## 第七部分：结账与西餐简介 💳

**结账的注意事项**：
1.  **异性约会**：通常由男士买单显得更有风度。
2.  **同事朋友聚会**：可采用“今天你请，明天我请”的轮流制AA方式。
3.  **商务宴请（与领导同行）**：通常领导负责招呼客人，下属负责现场结账操作（但实际支付者仍是领导）。高情商的领导事后会补偿下属。
4.  **回顾李小姐的故事**：她应在约会时适时提出AA或回请，而非总是让对方买单或总是抢着买单，以找到平衡。

**西餐礼仪简介**：
*   **座次**：女主人右手边是第一男主宾，左手边是第二男主宾；男主人右手边是第一女主宾，左手边是第二女主宾。
*   **餐具使用**：基本规则是`左手持叉，右手持刀`。
*   **建议**：若需参加重要西餐宴请，建议提前查阅详细资料进行学习。
