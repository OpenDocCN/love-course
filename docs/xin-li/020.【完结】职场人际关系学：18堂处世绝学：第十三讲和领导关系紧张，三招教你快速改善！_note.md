# 020.【完结】职场人际关系学：18堂处世绝学：第十三讲和领导关系紧张，三招教你快速改善！_note

在本节课中，我们将学习当与领导关系陷入紧张或疏远状态时，如何快速有效地改善局面。课程将介绍三种核心策略，帮助你从被动转为主动，修复并维护良好的上下级关系。

上一节我们探讨了职场中的普遍沟通原则，本节中我们来看看当与领导的关系出现具体危机时，应如何应对。

与领导关系紧张通常表现为：双方沟通与互动越来越少，交集与协作减少，甚至开始产生冲突。出现这些情况，往往意味着你已对领导产生抵触，或领导对你已产生厌恶。

当面临这种情况时，通常有三种应对办法。

以下是三种核心的改善策略：

1.  **离职走人**：这是最彻底的方式，直接离开当前环境。
2.  **主动调离领导的视线**：这是一种战略性撤退。当察觉到领导对你的好感度下降，留在其身边可能弊大于利时，主动申请调岗或外派是明智之举。这既能避免矛盾激化，也能为自己争取新的空间。正如历史案例所示，适时离开权力中心有时是更好的保全与进取之道。
3.  **主动改善关系**：这是最积极也是最具建设性的方法。关系的紧张不能只归咎于领导，因为外环境（领导）难以改变，我们能改变的是自己的行为。你需要主动采取措施来扭转局面。

上一部分我们列出了三种策略，接下来我们重点深入探讨最核心的第三种策略——如何主动改善关系。

改善关系不能依靠回避、抱怨或对抗。若不重视与领导的关系，即使工作努力也可能不被看见，甚至可能失去表现和晋升的机会。主动改善需从自身做起。

以下是主动改善与领导关系的三个具体行动步骤：

*   **尊重领导的权威**：领导的职场位阶高于你，这是组织秩序。尊重领导就是尊重组织规则和自我职业发展。任何领导都在乎权威，对于公开对抗的下属，绝不会重用。
    *   **核心概念**：`尊重 = 遵守组织位阶秩序`
*   **及时汇报工作进展**：领导都希望下属“事事有回音”。主动、及时的汇报能建立信任，让领导了解你的工作和能力。避免因恐惧沟通而躲着领导，这会让你失去被看见的机会。
    *   **核心概念**：`信任 = 持续且有效的沟通`
*   **服从指令并认同文化**：对领导的决策指指点点或总拿前公司对比，会直接导致领导的不认可。服从合理的指令，认同公司文化，是职业素养的基本体现，能从根本上减少关系紧张的诱因。
    *   **核心概念**：`服从与认同 = 减少冲突的基础`

本节课中我们一起学习了与领导关系紧张时的三大应对策略：离职、调岗与主动改善。其中，重点掌握了主动改善关系的三个关键行动：**尊重权威**、**及时汇报**和**服从认同**。记住，关系的维护主动权往往掌握在更积极、更懂得自我调整的一方手中。
