# 青春期心理课 038：情绪与人际关系 🧠🤝

在本节课中，我们将探讨青春期孩子的情绪状态与其人际交往之间的深刻联系，并学习如何帮助孩子建立健康的人际关系，从而促进其情绪健康与学习动力。

心理学家研究发现，当一个人的基本温饱需求得到满足后，他的快乐主要来源于良好的人际关系。因此，进入中学阶段的青春期孩子会特别注重人际关系，这是其情绪发展的核心需求。

在小学阶段，孩子的快乐主要来源于与家长的关系。他会通过撒娇、无理取闹或讨好等方式，从家长那里获取快乐。然而，从12岁左右的青春期开始，孩子的快乐更大程度上来源于他与同伴之间的关系。

所以在中学阶段，人际关系对孩子变得至关重要，其重要性在孩子心中甚至可能超过学习。但需要了解的是，中学生处理人际关系能力的关键养成期并不在中学，而是在3到8岁。

首先，家长需要明白，孩子高兴与否与其人际关系状况有密切联系。因此，当孩子表现出不想上学时，若根源是情绪低落，而情绪低落又源于人际关系问题，我们就需要分析原因并帮助孩子建立良好的人际关系。

一个人人际关系能力的基础，首先在于学会与父母相处。家长需要反思自己的行为模式。

以下是两种需要避免的常见家庭互动模式：

*   **过度给予型**：如果父母总是无条件满足孩子的所有要求，孩子就容易变成一个索取者，形成以自我为中心的性格。他在家里是“小皇帝”，到了学校仍会期待别人围着他转。这在中学是行不通的，因为每个孩子在家可能都是“小太阳”，人人都在“自转”，没有人会一直充当“卫星”。
*   **陪伴缺失型**：相反，如果家长因忙于工作而长期缺席，孩子由保姆或他人照看，与父母缺乏沟通，他同样会感到不快乐。因为孩子天生不具备人际沟通能力，这种能力需要在25岁前，通过父母的陪伴、教育和训练来后天养成。

如果家长发现孩子性格孤僻、冷漠、沟通不畅甚至有些自闭，就需要审视自身：是否因为工作或其他原因，忽略了与孩子的沟通和关系建设？因此，家长除了做好工作，必须抽出时间陪伴孩子，教导并示范如何进行有效沟通。

另外，有些家庭情况可能难以改变，例如单亲家庭、丧亲家庭、孤儿或父母外出打工的留守儿童，这些孩子可能怀有强烈的自卑感。

面对无法改变的环境，我们如何改善孩子的情绪？因为情绪直接影响学习，情绪不佳必然会阻碍学习。

以下是针对教师的一些可行建议，旨在通过集体活动改善学生情绪：

*   **倡导团体体育活动**：教师可以有意识地组织下课后的团体活动，如跳大绳、拔河、篮球、足球等团体比赛。这类活动能增加孩子的协调性与协作能力。
*   **激发生理积极改变**：运动能使体内自动分泌“快乐激素”。同时，以班级为单位的团体竞赛（如拔河）能增强班级凝聚力和同学间的配合。
*   **带动特殊群体**：这种方法能有效带动那些孤僻、自闭、情绪低落或不合群的孩子融入集体。教师可以尝试分组进行跑步、排球竞赛、篮球等活动，将个人运动（如羽毛球）与团体运动（如跳大绳）结合，从而改变班级风气。

活跃的孩子可以带动孤僻的孩子，当整体情绪向好，学习动力也会随之被带动。因为人只有在情绪愉快时，才更愿意投入学习。

此外，我也建议老师们有意识地增加户外运动时间。当前孩子室内学习时间过长，户外活动不足。人如果独处时间过多，容易变得敏感、不快乐，进而厌恶学习。这就是为什么有些人阅读量很大，但学习成绩却不理想——他可能因减少了人际交往时间而变得孤独、敏感甚至愤世嫉俗，这种情绪状态自然无法支持平静、愉悦地学习。

我们需要做出改变：通过增加户外运动，并进行正确引导，来帮助孩子调整状态。不能将所有时间都用于看书。我曾见过一位历史方面的天才，从小钻研历史并有作品，但高三时却患上严重抑郁症并有多次自杀倾向。这提醒我们，孩子的状态是动态变化的。

一个在小学阶段阳光开朗的孩子，进入青春期后也可能发生变化。他可能会经历一种“自我逆反”：变得内向、孤僻，从笑呵呵转为挑剔易怒。因此，我们需要从多个方面入手——包括家长自身的调整、对孩子人际关系技巧的训练、以及增加其户外运动时间等——运用这些方法来帮助孩子调整情绪，最终提升其学习能力。因为**人只有高兴了，才愿意去学习**。

---

**本节课总结**：本节课我们一起探讨了青春期孩子情绪与人际关系的紧密联系。我们了解到，良好的人际关系是青春期孩子快乐的重要源泉，而这项能力的基础在童年早期就已开始构建。作为家长和教育者，我们应避免过度溺爱或陪伴缺失，并积极通过团体活动、户外运动等方式，帮助孩子（尤其是面临困境的孩子）在集体中建立联结、改善情绪，从而为他们的学习与成长注入积极动力。
