# 育儿愈己心理课 012：第24集 什么是专注力（上）🧠

在本节课中，我们将要学习“专注力”这一核心概念。许多家长都关心孩子的专注力问题，但网络上流行的观点往往未能触及根源。本节课将重新梳理专注力的本质，澄清常见的误解，并分析影响专注力的关键因素。

## 专注力的常见误解与澄清

上一节我们概述了课程目标，本节中我们来看看关于专注力的几个流行观点，并对它们进行澄清。

以下是外界关于孩子专注力的几个常见疑问：

1.  **专注力是遗传的吗？**
    *   如果孩子有自闭症或阿斯伯格倾向，其某方面的专注力可能特别强，而其他方面则较弱，这存在一定的遗传因素。
    *   但如果认为孩子“毛手毛脚”完全是遗传自父母，这个观点并不成立。

2.  **孩子是否具备专注的能力？**
    *   请思考：你在逛淘宝或打球时是否专注？你的孩子在玩游戏时是否专注？
    *   如果答案是肯定的，那么孩子**完全具备专注的能力**。能力意味着在任何情境下都拥有完成某事的力量。公式可表示为：**专注能力 = 跨情境的专注表现**。
    *   因此，如果你观察到孩子在看电视、玩游戏时能专注，就应告诉自己：孩子拥有专注的能力。

3.  **孩子的专注力天生不好吗？**
    *   孩子的专注力天生其实是比较好的。问题往往不出在孩子本身，而在于父母的一些行为干扰了这种天生的能力。

## 父母在专注力培养中扮演的角色

既然孩子天生具备专注力，那么问题出在哪里呢？接下来我们分析父母在孩子的专注力培养中，常常无意识扮演的三种角色。

想象一下孩子正在专心游戏或学习，这与成人专注工作状态类似。以下是父母可能扮演的三种干扰角色：

1.  **打扰者**
    *   父母常因认为孩子的游戏“不是正经事”而随意打断。蒙台梭利曾指出：**孩子的游戏即是他的工作**。随意打断孩子的游戏，就如同随意打断成人的工作，会让孩子感到烦躁和不被尊重。
    *   核心干扰模式：**不提前预告，随意插入**。例如，未提前告知就要求孩子立刻停止手头的事情。

2.  **中断者**
    *   当孩子开始学习或做作业时，父母常在一旁不断纠正细节（如“字写好看一点”）。这种持续的唠叨和打断，就像施加了一个**魔咒**，破坏了孩子做事的连贯状态和内在节奏。
    *   代码式描述干扰过程：
        ```plaintext
        孩子进入专注状态 -> 父母发出纠正指令 -> 孩子注意力被迫转移 -> 内在节奏被打乱 -> 产生烦躁情绪 -> 专注状态被破坏
        ```
    *   正确做法：**语言精简，减少打扰**。多给予“要怎么做”的正面指令，而非“不要怎么做”的负面唠叨。

3.  **切换者**
    *   当孩子做错事时，父母急于当场纠正。这虽然出于好心，却强行中断了孩子正在进行的事务进程，造成其内心的巨大波动，不利于维持专注状态。
    *   处理原则：**事缓则圆**。对于非紧急错误，应有耐心等待事情做完，事后再用简洁的语言指出问题，并可让孩子按正确方法重做一遍以加深印象。这比当场打断更能保护孩子做事的“状态”和能力。

本节课中我们一起学习了专注力的本质，澄清了“能力不足”或“天生不好”的误解，并重点分析了父母作为“打扰者”、“中断者”和“切换者”这三种角色是如何在无意中破坏孩子专注力的。理解这些是提升孩子专注力的第一步。
