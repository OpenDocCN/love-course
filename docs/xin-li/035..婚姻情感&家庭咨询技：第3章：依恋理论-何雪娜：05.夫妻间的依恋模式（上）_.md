## 什么是爱？从依恋的角度看❤️

千百年来，人类通过多种方式探索爱的定义。从依恋理论的角度讲解，爱与**信任**和**安全感**有极大关联。根据鲍尔比的理论，爱是我们生命中的庇护所。当所爱之人情感封闭、得不到回应时，这会引发**依恋恐慌**。我们害怕分离，并可能因恐惧做出各种行动。

当所有依恋需求无法被回应和满足时，我们内在会拉响警报。这部分会刺激到处理情绪的**边缘系统**。因此，当我们感到危险和恐惧时，可能无法理智思考，而是直接付诸行动。这就是为什么人们在盛怒下会做出暴力行为，例如夫妻间的恶毒语言争吵、肢体冲撞或砸东西。

这些都反映了我们的**依恋警报被拉响**。在感到极度不安全的当下，我们缺少反思能力，直接用行动化的、不成熟的方式表达。本节课希望能帮助大家理解依恋内核在夫妻间如何运作，以及如何帮助夫妻意识到自己的依恋模式，学习在关系中更好地回应和表达。

综上所述，从依恋角度看，在亲密关系中获得**信任**与**安全感**是爱的基石。爱的意义和本质在于**寻求连接、寻求安全感、寻求信任**。

接下来，我们将一起分享，在婚姻模式中，所谓的“情感测试”可以从哪些方面来反馈和反应。

## 婚姻情感测试：评估夫妻依恋质量的三个维度📝

以下是评估夫妻情感连接的三个核心维度。你可以拿出纸笔，针对以下问题思考你的答案是“是”还是“否”。

### 维度一：亲近感

这个维度评估你和伴侣之间情感上的亲近程度。

1.  **我很容易能够吸引我伴侣的注意力。**（例如，伴侣是否能注意到你的新发型、新衣服或情绪变化？）
2.  **我的伴侣很容易和我建立情感的连接。**（例如，当你表达工作压力时，伴侣是给予理解支持，还是批评忽略？）
3.  **我的伴侣呈现出，我对他来说是首要的。**（当你需要时，伴侣能否将你的情感需求放在第一位？）
4.  **在这一段爱情关系当中，我并未感觉到孤独或者是排斥。**（你是否感觉被伴侣在意，而非被其家人或孩子排斥在外？）
5.  **我能和伴侣分享心灵最深处的情感，他也会倾听。**（这种倾听是“用心听到”，能产生情感共鸣，带来被接纳的感受。）

### 维度二：回应性

这个维度评估伴侣对你情感需求的回应是否及时、可靠。

1.  **如果我需要情感连接和安慰的时候，他会在我的身边。**（当你发出依恋信号，如生病、有压力时，伴侣能否在身边提供情感支持？）
2.  **当我们发出需要伴侣靠近的信号的时候，他会有回应。**（当你需要陪伴时，伴侣能否理解并回应“靠近”的请求？）
3.  **在我焦虑或者是缺乏信心的时候，我可以依赖我的伴侣。**（面对挫折时，你是否能从伴侣那里获得可信赖的支持？）
4.  **即使在我们争吵或有不同意见的时候，我也知道我对我的伴侣来说是重要的，而且我们可以找到其他的方法达成一致。**（能否在观点不同时，依然感受到自身的重要性，并寻求共识？）
5.  **如果我需要知道，我对我的伴侣来说是重要的，是具有重要性的，他就会给我信心让我更加笃定。**（伴侣能否通过言语和行动，给予你“你很重要”的信心？）

### 维度三：情感投入

这个维度评估双方是否都能积极、一心一意地投入到这段关系中。

1.  **在接近伴侣、相信伴侣时，我觉得很自在。**（与伴侣相处是否感到轻松自然，而非阻塞煎熬？）
2.  **几乎任何事情我都能信任我的伴侣。**（你是否能向伴侣敞开心扉，还是因缺乏安全感而封闭隐藏？）
3.  **即使是我们不在一起的时候，我也对彼此间的情感连接非常自信。**（物理分离时，是否依然心理上信任彼此的关系？）
4.  **我知道我的伴侣关心我的喜怒哀乐。**（双方能否觉察、命名并关心彼此的情绪体验？）
5.  **我很有安全感，确信我能与我的伴侣一同面对情感危机。**（是否相信双方有能力共同面对并修复关系中的矛盾和冲突？）

通过以上情感测试的三个板块，我们可以发现，依恋理论非常强调对情感的**回应**，以及对**安全感**和**信任感**的建立。无论夫妻争吵的具体事件是什么（如孩子教育、搬家、旅游），其深层往往是在表达：“你到底有没有听我说话？”“你到底在不在意我？”“你为什么要对我这么冷酷？”

如果在这个测试中，肯定的回答越多，通常意味着婚姻关系越稳固。如果否定的回答较多，则婚姻中可能已出现一些问题。长时间无法满足的依恋需求会导致对关系的否认、决裂甚至绝望。值得注意的是，不吵架并不一定代表关系好。有些婚姻已陷入僵局，双方冷漠、彻底失望，毫无情感交集，这种情形反而更危险。

有争执并不可怕。如果我们能对依恋理论有更多了解，就能明白彼此内在对安全和信任的需求是什么，并学习从这些角度去回应对方的依恋需求。
