# 课程012：原生家庭的四大原则（上）👨‍👩‍👧‍👦

在本节课中，我们将要学习一个非常重要的心理学概念——原生家庭。我们将探讨它的定义、重要性，并重点介绍其四大原则中的前两个：系统平衡性原则与系统隔离性原则。理解这些原则，有助于我们认识自身行为模式的根源，并为构建更健康的家庭关系打下基础。

## 什么是原生家庭？

在开始深入探讨原则之前，我们首先需要明确“原生家庭”这个概念。

所谓原生家庭，指的就是我们出生和成长的家庭，主要由我们的父母构成。例如，你与伴侣结婚并养育了孩子，那么你们就是孩子的原生家庭。同样，对你而言，你的父母就是你的原生家庭。它是每个人人生开始的地方，塑造了我们最初的行为模式和情感基础。

原生家庭本身是一个复杂的系统。接下来，我们将要探讨的四大原则，正是渗透在这个系统之中、牵一发而动全身的核心规律。理解这些原则，能帮助我们以更宏观的视角，发现以往难以察觉的家庭互动模式。

## 原则一：系统平衡性原则

上一节我们明确了原生家庭是一个系统，本节中我们来看看维持这个系统健康的首要原则——平衡性。

从个人视角看，我们可能觉得自己非常重要。但从整个家庭系统的宏观视角来看，任何个体都没有那么绝对的重要。系统追求的是整体的平衡与稳定，而非某个个体的绝对得失。这可以形象地用一个**等边三角形模型**来理解。

**公式/模型：家庭等边三角形**
想象家庭中有你、爸爸和妈妈三个人，各占据一个顶点，形成一个稳定的等边三角形。

现在，假设爸爸经常在外有婚外情，导致妈妈非常不满。此时，妈妈与爸爸的情感距离（三角形的这条边）是变长了。通常，妈妈可能会将孩子拉入自己的阵营，共同指责爸爸，形成“战略同盟”。

**结果：**
孩子与妈妈的距离变近（这条边缩短），但妈妈与爸爸、孩子与爸爸的距离都变远了。等边三角形因此失衡，变成了一个**锐角三角形**。两个人“同仇敌忾”地对待另一个人，系统就失去了平衡。

**核心要点：**
不要认为将孩子紧紧拉在身边、与自己形成同盟是好事。这对孩子并无益处，因为孩子内心需要的是父母关系的平衡，而非卷入父母的冲突。维持系统的平衡性至关重要。

## 原则二：系统隔离性原则

理解了系统需要平衡，那么如何才能达成并维持这种平衡呢？这就需要引入第二个原则——隔离性，其核心实践方式是建立清晰的边界与规则。

我们继续用上面的例子来说明。妈妈把孩子拉在身边，一起抱怨出轨的爸爸。孩子年龄尚小，没有心理学知识，只能听从妈妈的话，但内心并不会快乐。孩子会陷入两难困境：靠近妈妈对抗爸爸，会对爸爸产生负罪感；理解或靠近爸爸，又觉得事情令人难受。这让孩子内心背负了巨大的、本不应由他承担的压力。

以下是建立隔离性的关键思路：

1.  **明确责任归属**：夫妻之间的问题，理应由夫妻双方自行处理。孩子没有必要、没有义务、也没有能力介入父母的命运和创伤。
2.  **树立人际边界**：你是你，我是我。我的事情我来处理，你的事情你去处理。我们之间需要有一条清晰的边界。
3.  **认清父母角色的限制**：例如，孩子不学习，承担后果的主体是孩子自己。父母可以引导和帮助，但无法替代和主导。父母不是上帝，孩子也不是父母的救世主。

这个道理看似简单，但一旦陷入家庭系统的情感纠葛中，就容易变得混乱。人有一个本能规律：一旦建立情感连接，就渴望“合二为一”，追求亲密无间。这种“全能感”通常在孩子0-1岁时需要被逐渐打破。孩子需要经历与母亲肉体上的分离（如断脐带），再到心理上的逐步分离，最终认识到“个人头上一方天”，拥有自己独立的人生。

破坏隔离性原则，会导致家庭成员界限模糊，“你我不分”，彼此黏稠地纠缠在一起。这会让所有人都不舒服。因此，积极为孩子也是为自己“解套”，树立健康的边界和规则，是维持系统健康的关键。

本节课中我们一起学习了原生家庭的基本概念，并深入探讨了其四大原则中的前两项：**系统平衡性原则**与**系统隔离性原则**。平衡性原则告诉我们，家庭像一个等边三角形，需要维持各边关系的均衡；隔离性原则则指出，通过建立清晰的**边界与规则**来实现这种平衡，避免将孩子卷入夫妻矛盾，保障每个成员的独立性与心理健康。理解这两点，是迈向“育儿愈己”的第一步。
