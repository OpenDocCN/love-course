在本节课中，我们将要学习合作对话在团体中的创造性实践，特别是汤姆·安德森的反思团队与反思过程。这一方法在后现代心理咨询中具有鲜明的特点，它强调多人共同合作与反思，为个案工作带来了独特的视角和可能性。

## 反思团队的由来与核心理念 🧩

上一节我们介绍了合作对话的基本概念，本节中我们来看看反思团队这一具体实践是如何产生的。汤姆·安德森的反思团队与反思过程在心理咨询领域具有颠覆性意义。

在传统的家庭治疗或咨询督导中，观摩者通常通过**单面镜**观察咨询过程，并在来访者离开后讨论个案。这种方式存在一个显著问题：在当事人不在场时进行讨论，这些讨论往往是咨询师的猜测或推断，缺乏来访者的验证。

汤姆·安德森对此感到不安，并开始寻求改变。他尝试让权利关系更平等，其创新做法是：让反思团队在单面镜后聆听咨询对话，对话结束后，邀请咨询师和来访者共同来到单面镜后，与反思团队一起探讨。来访者聆听多元视角的描述后，常能产生突破惯性思维的新想法。

后来，实践进一步发展，取消了单面镜。当来访者与咨询师同处一室时，大家的话语会更倾向于为来访者着想，并未出现最初担心的冒犯情况。安德森强调，反思团队成员应以**第三人称**彼此交谈，谈论来访者的事情，而非直接对来访者讲话。这样做的重要目的是赋予来访者**不聆听**和**不回应**的权利，从而创造更大的自由与舒适度。

这一举措在当时是破天荒的，并迅速应用于包括精神科在内的多种场景。后现代流派的许多创始人正是在处理所谓“疑难杂症”的实践中发展出这些方法的。

## 什么是反思过程？ 🤔

我们了解了反思团队的由来，现在来明确“反思过程”这一概念。反思过程是汤姆·安德森思维的演进，它不局限于必须有团队。

反思过程指的是，即使只有一个人，也可以进行反思。在征得来访者同意后，一位咨询师与来访者对话，旁边坐着的另一位咨询师便可作为“反思团队”。反思的核心在于，不同视角的加入可能为对话带来新的影响。

因此，广义上我们称之为“反思过程”，它不一定需要多人团队。

## 反思团队反思什么？ 💭

明确了概念后，我们来看看在反思过程中具体可以做些什么。以下是反思过程中的一些基本准则和可行方向。

首先，在反思过程中需要注意以下几点：
*   **不给建议**：因为建议往往难以完全贴合来访者的独特情境。
*   **不做评判**：避免“对错”、“应该”等评判性语言。
*   **不直接对当事人讲话**：以维持一个让来访者可以自由选择聆听与否的安全空间。

那么，反思过程中可以做什么呢？主要是公开化你的内部与外部对话：

*   **内部对话的公开化**：分享你在聆听时的想法和感受。
*   **外部对话的观察与分享**：谈论你看到的对话过程、触动你的部分、引起的共鸣、对你的启发，或者你对当事人产生的好奇与问题。

## 生成式反思：As if 练习 🎭

除了基础的反思，还有一种特殊的生成式反思方法，称为 **As if** 练习。这体现了合作对话的核心信念：知识并非预设，而是在**关系过程**中共同建构产生的。

在具体介绍 As if 之前，需要理解两个基础要点，它们保障了反思过程的创造性：

1.  **创造聆听与说话的空间**：对话如同呼吸，说话是呼气，聆听是吸气。两者必须交替，新的意义才能在流动中生成。没有人应该只聆听或只说话。
2.  **容纳多重视角，强调声音的丰富性**：允许并欢迎所有描述，即使它们彼此不同甚至相左。这与现代主义追求“概括统一”不同，后现代尊重独特性。我们相信，当矛盾的声音并存时，可能看到更大的画面，产生新的可能性（即 **两者皆是/Both-And** 的思维）。我们的目标不是达成共识，而是让不同声音被听见，从而自然催生新想法。

以下是 **As if** 练习的具体步骤：

1.  在对话中，识别出来访者故事涉及的核心角色（如父母、配偶、孩子等）。
2.  根据角色在故事中的重要性排序（从相对边缘到核心）。
3.  邀请团体成员（每角色2-4人）扮演这些角色。**从最边缘的角色开始**，依次让扮演者以该角色的口吻发言。
4.  每个角色可以有数位扮演者，从而呈现该角色的多种可能性。
5.  扮演者基于听到的对话，以所扮角色的身份说出任何想说的话。

这个练习通常效果显著。当来访者听到同一个角色（如“妈妈”）被不同人演绎出多种版本时，常能获得意想不到的触动和新的理解。在团体咨询中，可以安排少数人与当事人对话，其他人则参与 As if 角色扮演，之后再次邀请当事人回应，往往能引发新的思考。
