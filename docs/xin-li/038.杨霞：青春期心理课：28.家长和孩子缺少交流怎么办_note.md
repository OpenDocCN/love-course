# 038.杨霞：青春期心理课：28.家长和孩子缺少交流怎么办_note

在本节课中，我们将探讨一个困扰许多青春期孩子家庭的普遍问题：孩子与家长之间缺乏有效沟通。我们将分析孩子进入青春期后心理与行为的变化，理解沟通障碍的根源，并学习具体、实用的方法来重建亲子间的交流桥梁。

## 理解青春期的变化

上一节我们介绍了沟通问题的普遍性，本节中我们来看看其背后的原因。孩子进入青春期后，其人际关系、思维方式与情绪表达方式均与童年期发生显著变化。

在12岁之前，孩子通常与家长有说不完的话，回家后乐于分享学校发生的各种事情。然而，进入中学阶段后，许多孩子开始变得沉默，不愿与家长主动交流。

一项针对中学生的调查显示，当遇到困难时，孩子们最希望获得家长的帮助。但另一项调查却表明，他们最不愿意倾诉的对象也是家长。这个矛盾的结果揭示了问题的核心：孩子需要帮助，却关闭了与家长沟通的渠道。

家长因此感到困惑与痛苦。他们能看到孩子的压抑、痛苦、学习下降或烦躁发脾气，却无法走进孩子的内心，无法提供有效的帮助。

## 家长心态的调整：从“导师”到“朋友”

理解了问题的背景后，我们来看看解决问题的第一步：家长自身心态的转变。关键在于学会“接受”。

家长需要接受孩子进入中学后可能更愿意写日记、与同学线上聊天，而不愿与父母交谈的现实。甚至，过多的追问可能导致孩子不愿一同吃饭。

研究表明，**孩子能与家长共进晚餐，对其情绪稳定至关重要**。因此，为了维持良好的晚餐氛围，家长应学会在饭桌上不追问孩子，不迫切地想知道孩子内心的想法。

许多家长担忧孩子有复杂、危险的想法。但事实上，大部分青春期孩子的想法相对单纯，他们自身也具备一定的判断力。家长越是表现得像朋友，沟通才越可能发生。

以下是家长需要做出的心态与行为转变：

*   **沟通模式转变**：从小学阶段的单向“命令-接受”模式，转变为平等的双向交流。
*   **放下权威姿态**：避免总是以导师、批评者或纠正者的身份出现。青春期的孩子可能开始“俯视”父母，认为父母的知识老化、不了解自己。
*   **放弃“撬开”心态**：彻底放下渴望强行打开孩子心门的想法，从内心接受孩子的沉默、安静，以及他与朋友交谈时的快乐。

当你越能坦然接受这些，孩子就越可能将你视为一个不让他感到紧张的“同伴”。在这种状态下，他反而可能在无人倾诉时，主动对你说：“妈妈/爸爸，我跟你说件事……”

## 建立有效沟通的具体方法

调整好心态是基础，接下来我们学习几个建立有效沟通的具体方法。

当孩子终于愿意开口时，家长的倾听方式至关重要。

**核心倾听原则：耐心听完，顺势回应。**

具体操作如下：
1.  务必让孩子把话说完，不要中途打断。
2.  避免立即表现出不耐烦或准备批评教育。
3.  顺着孩子的话语和情绪进行回应。例如，当孩子抱怨老师时，你可以说：“哦，原来是这样啊”、“这确实让人生气”。

**公式：有效回应 ≈ 理解性词语 + 接纳情绪的描述**

采用这种“顺势”方式，孩子往往会在宣泄后自己冷静下来，甚至会说“其实也没什么”。如果一开始就反驳或说教，则很容易引发争吵。

**方法二：家长主动自我暴露，降低姿态**

一个有效的破冰技巧是家长先开口说自己的“烦心事”。

例如，一位离婚后定期探望孩子的母亲，发现青春期的儿子不愿与她交流。在尝试主动分享自己工作或生活中的小烦恼后，儿子逐渐开始回应，并分享自己的心事。双方建立起一种朋友式的交流，互相理解与支持。

**方法三：鼓励体育运动，释放压力**

有时，沟通障碍源于孩子自身的心理状态。一个常见原因是学习压力大、思虑过多导致神经敏感，像“小刺猬”一样，既孤独又容易刺伤他人。

对于这种情况，一个简单有效的方法是鼓励孩子进行规律的运动。

例如，可以建议孩子：
*   `每天游泳2000米`
*   或 `每天打篮球一小时`

并坚持一周。运动后，人体内会自动分泌内啡肽等让人放松和愉悦的激素。

**内在逻辑链：规律运动 → 分泌内啡肽 → 身心放松 → 情绪愉悦 → 更愿意与人交往**

许多孩子实践后发现，运动后更想与人说话，与同学关系拉近，回家后也更能与父母有说有笑。穿插运动和劳动，能有效缓解过度学习和阅读带来的敏感与紧绷状态。

## 总结

本节课中，我们一起学习了如何应对青春期亲子沟通缺失的挑战。

我们首先理解了青春期孩子心理变化是沟通减少的自然原因。随后，我们认识到，解决问题的核心在于家长将心态从“权威导师”调整为“平等朋友”，学会接受孩子的变化。在此基础上，我们掌握了三个实用方法：**耐心倾听并顺势回应**、**通过自我暴露开启对话**、以及**鼓励运动以改善孩子情绪状态**。

记住，重建沟通是一个渐进的过程。通过成长我们自己，用更智慧、更平和的方式与孩子相处，我们不仅能改善当下的亲子关系，更可能积极影响孩子未来的人际交往模式与心理健康。
