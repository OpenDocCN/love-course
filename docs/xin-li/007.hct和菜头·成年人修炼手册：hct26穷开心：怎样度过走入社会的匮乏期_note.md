# 课程007：成年人修炼手册·hct26 穷开心：怎样度过走入社会的匮乏期 😊💰

在本节课中，我们将要学习“穷开心”这一概念，探讨如何在个人财富积累初期、面临经济与心理双重压力的“匮乏期”中，找到一种积极的精神状态与成长路径。

## 概述

从校园步入社会后，许多人会经历一段漫长的“匮乏期”。这段时期的主要特征是个人财富积累速度缓慢，但生活开销与心理预期却不断增长，从而容易产生焦虑与绝望感。本节课程将分析这一现象的成因，并分享一种名为“穷开心”的应对策略。

## 匮乏期的成因与挑战

上一节我们概述了社会新人普遍面临的困境，本节中我们来看看其具体成因。

从开始工作到30岁之前这段时间最为难熬。主要原因是，财富累积的速度远远低于心理预期，但是花钱的地方又是无穷无尽。这难免让人心生绝望，不知道这样的日子何时才是尽头。

这种现象可被视为一种“走入社会的不应期”。“不应期”是一个生物学词汇，指生物对某一刺激发生反应后，在一段时间内即使再给予刺激，也不发生反应。我认为，其主要根源在于教育的某种缺失。

回想过去，有两件事情令我百思不得其解。念书时，家长、老师乃至全社会都劝诫我们要专心读书，不要分心。一旦走出校门，他们却突然默认我们已经能够管理好个人财务，实现收支平衡。同时，又突然认为我们可以轻松地谈恋爱并找到伴侣。

要知道，为了上大学，之前需要花费12年时间打基础。为了获得大学文凭，需要花费4年时间修满上百个学分。一门课程修一学期，通常也只获得两三个学分。但是，为什么大家会认为一个人在零基础、零时间投入的情况下，可以迅速同时掌握个人财务管理和恋爱这两大世界级课题呢？

## 一个例外案例及其启示

在我认识的人里，只有一位朋友是例外。他在很小的时候就掌握了个人财务管理的技巧，实现了收支平衡，并且略有盈余。

主要原因是他的父母从事野外勘探工作，是双职工，经常离家半年甚至一年。因此，在他十岁出头时，父母每次离家前会把一年的生活费一次性留给他。然后，把家门钥匙一把挂在他的脖子上，另一把放在奶奶家。

父母走后，他的生活全靠自理。如果生活费消耗过快，他就只能祈祷父母早些回来。即使钱用尽了也无计可施，因为父母在深山中无法联络。他从小就能将个人财务管理得井井有条，主要得益于父母这种“放牧式”的管理，让他在幼年时就懂得了量入为出和合理计划。

然而，绝大多数家庭都做不到这样。我朋友的家庭也是迫于无奈，并非有意培养孩子。所以，大多数孩子毕业走上社会时，其实都是两眼一抹黑，要饱受好几年财务匮乏的折磨。

## 个人经历与时代对照

记得我刚工作时，月薪是800元人民币，实际到手大约700多元。当时还是座机时代，人们用传呼机相互联系，尤其对于我们这种需要值班或临时到岗的工种，传呼机几乎是标配。

然而，一台汉字显示的传呼机要3000多块，纯英文显示的也要1000多块钱。询问完价格后，感觉与校园相比，社会可以说是相当不友善。然而，以当时的朴素观念，父母既然已经供自己念完书，工作后就不应该再向他们要钱，反而应该每月上缴伙食住宿费。

这样一来，计算一下自己的储蓄速度和传呼机的价格，就会感到呼吸急促，血压上升。其实，后来回过头看，个人财富的累积是需要时间的。

我熬了五年之后，月薪涨到了3000块钱，加上各种补贴以及“只允许签字领钱不允许询问”的三产福利，手头宽裕了许多。但是，因为前五年被钱压得喘不过气，人的欲望被无限放大了。工资涨了四倍多，但欲望可能膨胀了十倍，最终依然不够用，银行卡上并没有多少结余。

好不容易从边陲来到首都，收入增长了三倍，但这里的房价又是家乡的十倍。人生好像就是这样，达到任何一个收入阶层，一定对应着相当于你现在收入水平十倍的潜在支出。我想，这大概就是整个社会保持前进的基本动力之一。

这样算下来，一个人从开始工作到生活基本安定下来，会有很长一段日子感到没钱，饱尝贫穷的滋味。

## 现代社会的新挑战

我二十多岁时，社会生活相对简单。说到赚外快，人们想到的是开餐厅、茶室，或者几人合资买个铺面。这样一旦亏损，大概就是亏掉一个铺面的转让费。

如今的情况大不相同。股票、基金、期货、数字货币以及形形色色的金融产品，让一个人拥有了丰富多彩的亏损方式，以及深浅不一的亏损可能。有时，我不禁想起唐代诗人杜甫的一句诗：“夜雨剪春韭，新炊间黄粱。”

我不会劝人储蓄，因为当年我自己也做不到。我也不会劝人理财，因为这远超我的知识范围。我唯一能做的就是坦承一个事实：我在刚工作时，也同样饱受贫穷的困扰。而且这件事除了用时间来解决之外，似乎并没有其他更好的方法。

这样，身处其中的人就会因为得知自己并非全世界唯一不幸的人，而多少感到一点宽慰。

## 核心建议：穷开心

有太多事情我做不到。不过，我确实有一个建议，可以让这段大多数人不得不熬过去的时光稍微好过一点。那就是三个字：**穷开心**。

回想那一段日子，除了对金钱和物欲的渴求之外，它同时也是我个人成长、获取新知最快的一段时间。有人建议，在这种阶段要控制欲望，学会延迟满足。如果欲望如此容易控制，那僧侣和修士何必十几年如一日地在静室中独坐呢？而在年轻时，谁会认同“延迟满足”也是一种满足？

没有足够的娱乐消遣，人就会有大量消耗不掉的时间和精力。以我的经验来看，把它们耗费在学习某项技能或养成某种爱好上，其效率可能是30岁之后的数倍，40岁之后的数十倍。

## 我的“穷开心”实践

二十多岁时，人大多喜欢关注新事物、新领域。对我来说，当时的新鲜事物就是网络。工作第一年，我就借贷了相当于一年的工资收入去开网吧。然后，因为手续问题，一年后关门了事。所有能拿回来的资产就是每位股东分到的两台电脑。我把其中一台卖掉，另一台搬回家，开始上网、聊天、打游戏。

并且从那时开始慢慢尝试写一些文字。两年过去之后，我在网上已经是小有名气的写手了。并且因为文章和论战的缘故，认识了一些天南海北的网友。那时候在网上写作没有一分钱收入，相反每个月要为上网支付数百元的电话费。

但对我来说，那是个业余爱好。在网络上，我所获得的认可和支持要比在单位里来得更快、更直接。

除了写作，我基本上是靠自己的力量以及网络搜索引擎，学会了如何安装、维修家里的网络。从零开始学会了如何购买服务器空间、搭建独立博客以及做搜索引擎优化。几年过去之后，我对电脑和网络的熟悉程度远超同龄人。

背后的动力很简单：如果我不自学，我就没有什么可玩的了。而因为我会调试网络、维护计算机，做一些简单的杀毒、安装系统等工作，刚好赶上了单位第一波“无纸化办公”大潮，于是变成了一个技术小能手，整天被呼来唤去地修系统、安打印机。

同样，这些工作不会给我涨一分钱工资，但我还是乐在其中，因为可以摆弄领导们的电脑（他们的电脑普遍比我的好），还可以偷偷在上面跑游戏。所有这些事情都不会带来直接收益，但是却让我觉得快乐，因此可以把我的行为称之为“穷开心”。

我觉得穷开心很好，起码让我没时间整天为了钱少这件事而痛苦，也没有太多念头缠绕在那些我想拥有而拥有不了的东西上。

## “穷开心”带来的长远价值

工作五年之后，“穷开心”悄然发生了作用。我在网上的文字被平面媒体发现，开始有了约稿。稿费虽然一开始很低，但逐渐水涨船高，甚至能打平我的工资，这是我后来离职的心理保障。

我在单位里也因为懂电脑、懂网络而获得了认可，有越来越多的任务交给我。我也慢慢升职，从边缘部门一点点调动到核心业务部门，每次调动工资都会略微上涨，几次下来涨幅也很可观。

最后，我的“穷开心”让我得以进入自己梦想的互联网行业。雇主对我的正式履历表毫无兴趣，明确表示他们招聘我是因为我的业余爱好。

我不认为这是什么成功的个人经历。事实上，如果我早一点知道预算平衡和开源节流的道理，我的人生路可能走得更为平顺，完全不需要这样曲折漫长。如果说这段经历里还有任何一点有价值的地方，我想只能是“穷开心”了。

## 总结“穷开心”的三点好处

以下是“穷开心”策略的三个核心好处：

1.  **它让你关注新事物、新领域。** 因为新生事物大家都不太关注，这意味着可以少花钱，甚至不花钱也能加入，别人甚至会邀请你进入。
2.  **它可能带来意外机遇。** 一旦新生事物发展起来，市场上会需要有经验的人，而这样的人在初期非常稀缺。这时候，即便你只是个“玩家”，因为玩得多，别人也会认为你有经验，愿意给你机会试试。
3.  **它保护你的精神健康。** 即便以上事情都没有发生，但是你愉快地度过了最难熬的几年，并没有被物欲和金钱在精神上压倒、导致人格变形，可以顺利地展开下一段人生。

现在我可以说了：当年我是用“网瘾”打败了“透支瘾”，结束了我的“卡奴”生活。那种网瘾和今天的网瘾有一点区别，那就是为了**创作**和**创造**而上网的“瘾”，多少都会留下一些成果。而那些成果在命运的拨弄下，未来会产生怎样的变化，当时忙于开心的我并无觉察。

## 课程总结

本节课中，我们一起学习了“穷开心”这一概念。我们探讨了社会新人普遍经历的“匮乏期”的成因与挑战，并通过个人案例说明了“穷开心”——即在经济拮据时期，将过剩的时间和精力投入到学习新技能、培养新爱好中——不仅能够帮助度过艰难岁月，还可能为未来的人生发展积累意想不到的资本。其核心在于**主动寻找并沉浸于低成本的创造与学习乐趣中**，以此对抗匮乏感，实现精神上的自足与成长。
