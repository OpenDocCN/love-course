# 谈判高手课程 115：谈判筹码的本质与运用 🎯

在本节课中，我们将深入探讨谈判中“筹码”的核心概念。许多人认为谈判必须有强大的筹码才能进行，但事实并非如此。我们将学习如何像诸葛亮一样，在看似没有筹码的情况下，通过认知管理和策略组合来创造谈判优势。关键在于理解：谈判是基于“认知”而非“事实”进行的。

## 谈判是基于认知的博弈 🧠

上一节我们介绍了谈判的基本框架，本节中我们来看看“筹码”的真实含义。谈判双方并非在与客观事实谈判，而是在与各自对事实的“认知”进行博弈。

**核心公式：谈判效力 = 对方相信你拥有的筹码 × 筹码的实际价值**

这意味着，对方“相信”你拥有某种筹码，与你“实际”拥有该筹码，能产生几乎相同的效果。三国时期，司马懿面对诸葛亮的“空城计”，正是与他脑海中“诸葛亮一生谨慎”的认知在谈判，而非与城门大开的客观事实谈判。如果司马懿的认知数据库里没有诸葛亮的资料，故事就会重写。

因此，“有没有筹码”本身不重要，重要的是“对方是否相信你有”。

## “赏”与“罚”的筹码构建 🔗

在谈判中，筹码常以“赏”（给予好处）和“罚”（施加代价）的形式出现。但如何定义和运用它们，需要仔细拆解。

以下是构建有效筹码时需要明确的四个关键注释点：
1.  **“有”**：指对方相信你拥有，而非你客观上必须拥有。
2.  **“东西”**：指能对对方产生价值或影响的任何事物（资源、机会、关系等）。
3.  **“赏”**：指对方想要且你能够给予的好处。
4.  **“罚”**：指对方害怕且你能够施加的代价。

**核心策略：挂钩战术**
当你缺乏直接筹码时，可以使用“挂钩”战术，将其他议题引入谈判，创造交换空间。
*   **公式**：`如果你给我A（我的需求），我就给你B（你的需求）`。
*   **关键**：通过引入B甚至C，将单一议题的谈判转化为多议题的交换，从而增加自己的议价能力。

## 筹码陈述的顺序与艺术 ⚖️

你的要求（Demand）和你给予的回报（Offer）的陈述顺序，极大影响谈判结果。顺序的选择取决于交换的性质。

以下是两种不同的情境及对应的最佳陈述策略：

**情境一：等值交换（换）**
当双方交换的物品价值大致相当时，应先提出你的要求。
*   **示例代码（低效）**：`如果我嫁给你，你能不能考虑买个钻戒给我？`
*   **示例代码（高效）**：`如果你买钻戒给我，我可以考虑嫁给你。`
*   **原理**：先锁定对方的需求（钻戒），再抛出自己的回报（婚姻），能更有效地促使对方行动。

**情境二：不等值引导（调）**
当你用一个小诱饵（Offer）去换取对方一个更大的承诺（Demand）时，应先给出诱饵。
*   **示例代码（低效）**：`如果你订我们杂志一年，就送你这块精美手表。`
*   **示例代码（高效）**：`喜欢这块精美手表吗？只要订我们杂志一年，它就是你的了。`
*   **原理**：先展示好处（手表），降低对方的防御心理，再提出要求（订阅），使整个提议更顺滑，谈判意味更淡。

**总结规律**：`Demand`（要求）和`Offer`（给予）的先后，取决于两者在天平上的重量对比。重量相当则“要求”在先；重量不等则“给予”（诱饵）在先。

## 筹码运用的关键原则 🛡️

掌握了筹码的构建和陈述后，我们还需要遵循几个关键运用原则，以确保筹码发挥最大效力。

以下是四个必须时刻自问的核心问题：

1.  **筹码是否已使用？**
    “能惩罚吗？”和“有东西赏吗？”的潜台词是“还没罚”和“还没赏”。**尚未使用的筹码才具有最大威慑力或吸引力**。一旦使用（如惩罚已执行），筹码的价值就消失了。就像父母吓唬孩子“出去就打你”，如果还没打，这句话就有效；如果已经打了，孩子就可能破罐破摔。

2.  **是否有退路？**
    **凡事先找退路，有退路的人赢**。在谈判前，必须想清楚“万一谈不成，我怎么办？”。如果没有退路（如“买不到房我就跳河”），你就会陷入绝对被动，只能任人宰割。清晰的退路方案是你敢于坚持和说“不”的底气。

3.  **是否有时间？**
    **时间本身就是强大的筹码**。在僵局中，更能耗得起的一方往往占据优势。例如两车狭路相逢，谁更有时间悠闲地抽烟、听音乐，谁就更可能迫使对方倒车。关键同样在于“对方是否相信你有时间”。在商业谈判中，即使你在乎的是时间（如交货期），也要尽量表现出你在乎的是价格，以免暴露自己没有退路的弱点。

4.  **能否借助法律？**
    **法律是拒绝不合理要求的坚固盾牌**。你可以利用法律作为“拒绝让步的正当理由”。
    *   **策略代码**：`因为法律要求/禁止，所以我不能这样做。`
    *   **原理**：当对方提出不合法或不合规的要求时，他内心通常也不预期能完全得到。利用法律作为挡箭牌，既避免了直接冲突，又可能在其“可裁量范围”内达成一个折中方案（例如，对方要100块，你因“不合规”拒绝后，给予20-30块作为补偿）。

---

**本节课总结**：
我们一起学习了谈判中筹码的本质与运用。核心在于理解**谈判是基于“认知”的博弈**。我们探讨了如何通过“挂钩”战术构建筹码，掌握了根据交换性质（“换”或“调”）决定**要求与给予的陈述顺序**。最后，我们明确了运用筹码的四大原则：**保持筹码的未使用状态、为自己准备退路、善用时间压力、以及借助法律作为防御工具**。记住，真正的谈判高手，不在于他实际拥有多少，而在于他如何让对方相信并感知到他的力量。
