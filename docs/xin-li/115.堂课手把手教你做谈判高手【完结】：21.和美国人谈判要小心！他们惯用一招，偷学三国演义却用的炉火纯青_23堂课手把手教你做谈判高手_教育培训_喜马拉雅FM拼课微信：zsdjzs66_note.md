# 谈判高手课程115：谈判中的锁定策略与节奏控制 🎯

在本节课中，我们将学习谈判中如何运用“锁定”策略来巩固立场，以及如何把握谈判的节奏。我们会探讨多种锁定自身立场的方法，并了解如何为这些锁定策略预留解套的空间，最后理解抽象议题对谈判的制约作用。

## 谈判中的锁定策略

上一节我们介绍了谈判中让步的幅度、次数和速度。本节中我们来看看如何在特定立场上“锁定”自己，即如何向对方表明自己无法再让步。

锁定立场的方法有很多种。以下是六种常见的锁定策略：

*   **民意**：以公众舆论或民情沸腾为由，声明自己无法让步。例如：“因为民意沸腾，所以我不能让步。” 这种方法常见于劳资谈判或环保谈判。
*   **白纸黑字**：援引书面规定、法令、老板指令或政府法规作为无法让步的依据。例如：“根据政府规定，不能答应。” 这有助于体现“对事不对人”的原则。公式表示为：**无法让步 = 依据(规定、法令等)**。
*   **第三者**：声称决定权不在自己手中，需要请示上级、委员会或合作伙伴。例如：“公司不是我的，需要委员会开会通过。”
*   **没有能力**：直接声明自己缺乏满足对方要求的资源或能力。例如：“要钱没有，要命一条。”
*   **伤害自己**：通过伤害自身或关联方（如绝食、自杀威胁）来施加压力，迫使对方让步。
*   **专业知识**：引用专业报告、研究或数据来支持自己无法让步的立场。例如：“根据美国XX报告，这个方案不可行。”

## 锁定策略的解套方法

我们介绍了锁定自己的方法，但精明的谈判者总会预留后路。本节中我们来看看如何为上述锁定策略设计解套方案。

每一种锁定策略都可以找到相应的解套方法。以下是针对不同锁定策略的解套思路：

*   **民意**：民意是可以改变的。通过“再次进行民意调查”，新的结果可能提供让步的借口。
*   **白纸黑字**：规定通常存在例外。可以寻找或创造“例外情况”来突破书面限制。
*   **第三者**：既然第三者是虚构或可影响的，那么自然可以声称“第三者改变了主意”。
*   **没有能力**：自身能力不足，但可以寻求“他人帮助”。例如，自己不会倒车，但可以请路人帮忙倒车。
*   **伤害自己**：通过“自己想开了”或找到新的精神寄托（如宗教信仰）来改变立场。
*   **专业知识**：知识是更新的。可以援引“新的研究发现或报告”来推翻旧的结论。

然而，有一种情况难以解套，那就是“抽象议题”。例如国家民族大义、家族清白、个人荣辱等。这类议题一旦摆上谈判桌，就很难找到台阶下，容易导致谈判僵局。

## 谈判节奏与抽象议题的陷阱

我们了解了如何锁定与解套，现在来探讨谈判的整体节奏以及一个需要特别注意的陷阱——抽象议题。

谈判需要有节奏感，通常是“让步 - 锁定 - 再让步”的循环。例如，从8000元让步到7400元（放），然后在7400元上运用策略锁定立场（锁），最后在适当时机再次让步（放）。这种 **节奏 = 放 -> 锁 -> 放** 的掌控是谈判高手的重要技能。

在运用节奏时，必须警惕“抽象议题”的陷阱。将抽象、宏大的理由（如孝心、家族荣誉）作为谈判核心，会极大地压缩自己的回旋余地，使有效的谈判策略（如杀价）难以施展。谈判时应尽可能将议题保持在“具体”层面。

## 课程总结

本节课中我们一起学习了：
1.  **六种锁定谈判立场的方法**：民意、白纸黑字、第三者、没有能力、伤害自己、专业知识。
2.  **对应的解套策略**：为每种锁定方法预留退路，保持灵活性。
3.  **谈判节奏的控制**：掌握“让步、锁定、再让步”的循环节奏。
4.  **抽象议题的风险**：避免将抽象概念作为谈判基点，以免作茧自缚。

记住，成功的谈判在于既有坚守立场的技巧，也懂得如何优雅地转身。
