# 课程091：人际关系必修课（完结）🎯  
## 04. 建立亲和力：学会支持别人的4大秘诀🤝

在本节课中，我们将学习如何与S型特质（稳健型）的人有效相处。S型人通常温和、包容，注重和谐与安全。理解他们的特征和需求，可以帮助我们建立更稳固、平和的人际关系，并让他们成为我们优质的能脉。

你是否曾觉得自己能力出众，希望在工作中大展拳脚，却陷入职场斗争而屡屡受挫？有一种人，他们平日低调如“扫地僧”，很少站在聚光灯下，却能超脱职场纷争，化解各种冲突。他们像水一样柔和，能绕过生命的阻碍。他们就是S型人。S型人有什么独特之处？我们又该如何与他们相处呢？

上一堂课，我们深入探讨了如何与社交型（I型）的人建立优质能脉，核心方法包括：给予认可、提供舞台、创造有趣体验和定期跟进。请大家认真完成作业，盘点自己的有效能脉，并下载课程学习包。学习包内含课程讲义、经济图片及各种辅助资料，干货满满。

我们用一个生活中的例子复习DISC四种风格：家中四兄妹晚上失眠，D特质的大哥会起床工作，I特质的大姐会刷手机社交，S特质的小妹会安静躺在床上数羊，C特质的小弟则会分析失眠原因并总结方法。如果说D给予勇气，I带来激情，C提供缜密，那么S则让人感到轻松、自然、无压力。

S特质的人行动较慢，关注人。他们如白开水般温和、平和、谦和，善于以柔克刚，追求“不战而屈人之兵”。S型人为世界带来和平，与他们做朋友，你能学会平衡冲突，搭建有温度的能脉。但若S特质发挥过当，可能显得懦弱、胆小。他们通常不引人注目，但当你需要时，他们会伸出坚实的双手。在冲突中，S是和谐的“润滑剂”，是守护在你身旁的天使。

他们低调，但并非不在意关注。若未处理好与他们的关系，他们不会变成恶魔，只会默默离开，去成为别人的天使。什么样的人一直在你身旁守护，却被你忽视？一个人只有心服口服，关系才能长久。现在，让我们一起学习S型人的特征，找到并保护好你的天使。

从个体角度看，S型人的口头禅主要有两句。第一句是“随便”。当你询问吃饭地点、菜品或同伴时，他们常回答“随便”。S型人担心自己的决定让别人不喜欢或为难，为避免他人不适，宁愿交出选择权，哪怕自己受委屈。但在团队中，这可能导致你得不到有价值的意见。你真诚希望人人表达想法，但S型人有意见却不说出。例如工作中讨论方案，A提A方案，S说“不错”；B提B方案，S也说“不错”。当A、B意见对立时，S可能说：“吵什么？各有千秋。”你会发现，S并未表达自己的意见。作为管理者，你不仅要盘点公司资产，更要盘点人才是否被用好。此时，你可能需要让A、B稍等，先请S表达意见，这样才能促使S贡献想法。

第二句口头禅是“对不起”。S型人说“对不起”不一定是因为做错事，而是因为他们礼貌友善，这句话能帮助他们远离他人压力和自身愧疚感。

S型人的装扮通常没有显著特点，他们力求与多数人保持一致。过于亮丽或张扬会让他们不自在，因为他们不喜欢被特别注意或出风头。S型人爱静不爱动，具有温柔祥和的吸引力与宁静婉约的气质，做事做人有种让人安静的定力。有时他们奉行“能坐不站，能躺不坐”的原则，珍惜每个放松的机会。

S型人知足常乐，随遇而安。他们知足、脾气好，对生活要求不多，乐于付出，能体验心灵开放的包容状态，感到幸福。他们包容性强，有耐心倾听他人观点，和善谦逊的个性为他们赢得许多忠诚的朋友。

从群体交往角度看，S型人强调平衡、追求和平。有不同意见时，他们点到为止，信奉“将军额上能跑马，宰相肚里能撑船”、“得饶人处且饶人”、“留得青山在，不怕没柴烧”等原则。他们与人为善，天性宽容，耐心柔和。例如，南非前总统曼德拉是首位民主选举产生的总统，也是诺贝尔和平奖得主。他曾因领导反种族隔离运动被判入狱27年。在就任总统典礼上，他邀请当年监视他的三名狱警，并说：“我年轻时脾气暴，在狱中正是他们三位帮助我学会了情绪管理。”这番话让虐待他27年的三人无地自容。曼德拉后来分享：“走出囚室、通往自由的大门时，我已清楚，若不能把悲伤与怨恨留在身后，那么我其实仍在狱中。”

S型人不喜欢拒绝别人，有时这一点会被利用，导致同情心被滥用，甚至被视为懦弱无能。有时他们答应帮忙太多，却分不清轻重缓急，最终因照顾不周而被认为工作拖拉。面对压力或不满时，S型人常默默忍受。他们并非没有情绪，只是不习惯表达，害怕引发更大矛盾。他们以“吃亏是福”麻痹自己，但若对方不成熟或不够敏感，可能纵容别人欺压自己。这说明，尽管S型人好相处，你仍需对他们保持敏感。例如，一位S特质很高的太太想和先生看电影，若先生不了解她，可能说：“去什么去？你没时间。”若先生不了解她，她可能独自哭泣；若有人了解她并陪她看电影，后果可能不堪设想。每个人都有责任照顾好生命中重要的人。

S型人自身变化和需求不多，更多扮演给予者而非索取者。他们不重视利益交换，付出即是快乐。但常见的情况是，不管别人是否需要，他们一味付出，结果并非满足他人需求，而是满足了自己想付出的需要。

我们已了解S型特质的诸多特征及其可能伴随的误解。若想与S型风格显著的人建立良好关系，需先明白他们关注什么，透过特征探寻其内在动机。S型人不似D型人追求快效率，也不像I型人关注有趣。他们首先渴望踏实平淡的生活，其次关注团队和谐，宁愿自己吃亏也要避免冲突，再者更在意安全而非挑战。他们行动有时较慢，可能错失机会。

从以上三个关注点看，S型人似乎有些逆来顺顺受，天生好脾气，却不习惯要求别人。我有一位S特质很高的朋友，一次在旅游景点想拍一张桥上无人、桥下无船的意境照，但因游客多，等了半小时仍未拍成。此时两个路过的小姑娘请他帮忙拍照，他欣然答应。对方问他在此做什么，他说明意图后，两个小姑娘立刻冲到桥头桥尾，大喊：“不要过来，有人拍照！”结果，我朋友30分钟未搞定的事，她们不到13秒就解决了。可见，S型人有时也需敢于提出要求，否则效率可能低下，甚至办不成事。

了解S型风格看重什么，我们就能让他们成为优质人脉。核心是给予S型人安全感。与S相处，无需像对D那样雷厉风行，也不必像对I那样幽默风趣；与S沟通，更需细水长流。S型人有时会得过且过，期待问题自动解决，显得被动。我曾有一位S型员工汇报工作时一直磨蹭，直到我主动询问并告知调整后的薪资，他才长舒一口气。若我当时未察觉，直接让他回去，他可能回到岗位仍心不在焉。因此，对S型人应留意其需求，为他们争取应有权益。

S型人相处时会逃避问题与冲突，容易妥协，过于在意他人想法，不敢表达意见，归根结底是缺乏安全感。我们应赞赏其团队协作精神，设法给予足够的安全感，主动邀请他们表达想法。

S型人习惯配合他人，不做主导，缺乏激情，有时容易迷茫。他们内心并非没有想法，只是需要鼓励。我们应多给予关怀和包容，让他们学会有犯错的勇气和快速修正的能力。

S型人行动速度较慢，交代工作时最好提前告知，给予接受过程，并定期提醒。若工作按时完成，多给予鼓励，帮助他们逐步培养主动提早完成工作的习惯。

拥有S型特质的能脉，你的世界将充满和平。D与D在一起争权力，I与I在一起争表现，C与C在一起争标准，唯有与S在一起，世界才是和平的。D与S合作，D做决定，S执行；I与S合作，I在前台，S在后台；C与S合作，C定标准，S认真执行。因此，只有与S在一起，我们的世界才是和平的。
