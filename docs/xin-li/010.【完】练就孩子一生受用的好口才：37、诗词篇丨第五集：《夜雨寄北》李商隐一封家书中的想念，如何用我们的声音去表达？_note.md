# 口才课程010：诗词篇丨第五集《夜雨寄北》📜

在本节课中，我们将一起学习唐代诗人李商隐的《夜雨寄北》。我们将探讨这首诗如何通过一封家书表达深切的思念之情，并学习如何运用我们的声音来传递这种情感。

## 诗歌背景与诗人介绍

上一节我们介绍了诗词表达情感的基础，本节中我们来看看李商隐的《夜雨寄北》。这首诗是李商隐在外为官时写给妻子的一封家书。诗中表达了他因秋雨阻隔、无法确定归期，从而对妻子产生的强烈思念和盼望归家的心情。

李商隐是唐朝著名的诗人。当时，他的政治靠山失势，他不得不北归回家。在此期间，他与妻子王氏经常通信以诉思念。然而，当他行至巴山一带时，遭遇连绵秋雨，道路不通，归期变得渺茫。正是在这种情境下，他写下了这首诗寄给妻子，作为一封深情的家书。

## 诗歌内容解析

以下是《夜雨寄北》的全文：
```
君问归期未有期，巴山夜雨涨秋池。
何当共剪西窗烛，却话巴山夜雨时。
```

接下来，我们逐句分析其含义。

**第一句：“君问归期未有期”**
*   **含义**：你（指妻子）来信问我何时归家，但我无法给出确切的日期。

**第二句：“巴山夜雨涨秋池”**
*   **含义**：我所在的巴山地区，秋夜雨水连绵，池塘的水都涨满了。此句借景抒情，用“夜雨涨秋池”的景象，暗示归途受阻的困境和内心逐渐累积的愁绪。

**第三句：“何当共剪西窗烛”**
*   **含义**：什么时候才能回到家，与你一起坐在西窗下，剪着烛花（促膝长谈）呢？这是诗人对未来团聚场景的美好想象。

**第四句：“却话巴山夜雨时”**
*   **含义**：到那时，我要好好跟你诉说，我今夜在巴山听着绵绵夜雨、思念你的心情。这句将当下的孤独与未来的温馨联系起来，情感深沉。

整首诗仅用二十八个字，就完整勾勒出了现实的困境、内心的思念以及对未来团聚的期盼，画面感极强。

## 情感把握与声音表达

理解了诗歌内容后，我们来看看如何用声音来表达其中的情感。这首诗的情感核心是**深切的思念**、**淡淡的忧伤**以及**温暖的期盼**。它并非激烈的呐喊，而是一种对爱人娓娓道来的、真挚而含蓄的诉说。

在朗读时，请注意以下几点：

以下是朗读这首诗的情感要点：
1.  **语速平缓**：模仿书信中娓娓道来的语气，不宜过快。
2.  **语调含蓄**：情感表达应内敛而深沉，避免过于夸张。
3.  **重音突出**：在关键词上稍加强调，如“**未有期**”、“**涨**秋池”、“**共**剪”、“**却话**”。
4.  **想象场景**：在朗读时，想象自己就是那位在秋雨之夜提笔写信的诗人，心中充满对家人的思念。

你可以联想一个类似的场景：比如父母一方出差未归，另一方或孩子心中的那份牵挂和盼望。用那种对亲人说话的自然、真挚的语气来演绎这首诗。

## 课程总结

本节课中，我们一起学习了李商隐的《夜雨寄北》。我们了解了这首诗作为一封家书的创作背景，解析了诗中通过“巴山夜雨”之景所抒发的“归期难定”之愁，以及“共剪西窗烛”所寄托的深切思念与期盼。

我们重点练习了如何把握这种含蓄而真挚的情感，并用平缓、含蓄、带有诉说感的语音将其表达出来。掌握这种情感表达技巧，能让我们的口语传达更具感染力和深度。
