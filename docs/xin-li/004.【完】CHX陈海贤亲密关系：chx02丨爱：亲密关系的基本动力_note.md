在本节课中，我们将要学习亲密关系背后的核心动力——爱。我们将从心理学的“依恋”概念出发，探讨爱的本质、特征，以及它如何像粘合剂一样，将两个人紧密地维系在一起，形成一个稳定而有效的情感系统。

我曾经参观过一个“猎人博物馆”，里面陈列了许多猎人的纪念品。有相互赠送的生日礼物，有因异地恋而产生的满满一箱火车票和飞机票，还有写给彼此的厚厚一沓情书。

每件纪念品都写满了对爱的渴望、对猎人的思念以及伴侣分开后的悲痛。为什么在分离后，人们对猎人有如此强烈的思念和渴望？究竟是什么力量把这两个人牢牢地绑在一起？

这一讲，我们就来探讨亲密关系背后的动力：爱。

亲密关系以爱为核心。没有爱，两个人就会变成搭伙过日子。爱能给人带来最大的喜悦和痛苦。可是，什么是爱呢？千百年来，无数人发出过这样的疑问，也有无数人给出了各种各样的答案。

诗人可能觉得爱是浪漫的感觉；生物学家可能认为爱是性冲动的衍生品，是吸引人繁衍后代的手段；经济学家可能视爱为一种可用于交换的情感商品。这些说法都有道理。

但在心理学中，爱有一个特别的概念，叫 **依恋**。亲密关系中的爱，来源于人类的依恋本能。

---

## 什么是依恋？从鲍尔比的理论说起

上一节我们提出了爱的核心是“依恋”，本节中我们来看看“依恋”具体指什么。很久以前，英国心理学家约翰·鲍尔比发现，孩子和父母（尤其是母亲）之间存在一种非常紧密的情感联系，他称之为“依恋”。

你是否还记得在妈妈怀里睡觉的场景？两个人呼吸连着呼吸，心跳连着心跳，仿佛变成了不可分离的一个人。在孩子很小时，只要在妈妈身边就会觉得安心；如果妈妈不在，他们就会变得非常不安，像生病一样不停哭泣，喊着“我要妈妈”。

这种依恋对婴儿的生存和发展至关重要。母亲和孩子之间稳定的依恋有四个特征。

以下是这四个核心特征：

1.  **强烈的情感**：在许多文学作品中，我们看到孩子被迫与妈妈分离时，会克服千难万险去寻找妈妈。与妈妈的分离也会带来巨大的失落感。
2.  **对彼此的有效回应**：我们常看到这样的画面——妈妈抱着孩子，用温柔的目光注视她、对她微笑，孩子会报以同样的微笑。如果孩子看妈妈时，妈妈忽略了或没有回应，孩子就会变得失落，仿佛与妈妈失去了联系。
3.  **关系的唯一性**：一旦依恋关系建立，其他照料者很难替代妈妈的角色，无论这个照料者多好。同样，妈妈也很难随便找一个孩子来替代自己原来的孩子。
4.  **纯粹的情感关系（而非利益关系）**：虽然有“养儿防老”的说法，但在依恋关系里，父母爱的是孩子本人，而不是孩子带来的用处。同样，孩子最看重的也是与父母的情感，而非父母的养育功能。

## 成年人的爱：同样源于依恋本能

你可能没想到，成年人亲密关系中的爱，也源于人类同样的依恋本能。为什么这么说呢？因为这种情感与母子之间的依恋情感，不仅有着相似的生理基础（都是**多巴胺**和**催产素**在起作用），还有着相似的体验和特征。

为了说明这种相似的体验和特征，我想请你先设想一个情景。

假如你正与一个人交往，对方看起来不错，但也有一些捉摸不定。你该如何判断对方是真的爱你，是逢场作戏，还是只想做朋友，是你自己想多了呢？我想可以通过四个特征来判断。

以下是判断真爱与否的四个特征，它们与亲子依恋的特征一一对应：

1.  **是否有强烈的情感连结？**
    *   你会想知道对方是否真的在乎你，有多在乎。
    *   他是否会记得你们在一起时说过的话，记得对你重要的日子（如你的生日）。
    *   他是否知道你的喜好。
    *   你们是否会“一日不见，如隔三秋”？
    *   如果要分手，他是痛不欲生还是满不在乎？这些都在提示你们之间情感联系的强度。

2.  **对方是否真的回应你？**
    *   你们在一起时，聊天愉快吗？能说到一起吗？
    *   这不仅意味着有相似的爱好、生活或价值观，更重要的是他能懂你，并对你所说的有兴趣。
    *   例如，你分享一件高兴的事（“我今天涨工资了”），他会替你高兴，而不是说“就这点钱有什么好高兴的”。
    *   你分享难过的事（“我昨天做了一个噩梦”），他会关心地问“什么梦？”，而不是不屑地说“噩梦嘛，又不是真的”。
    *   在这种细碎的日常回应中，你们滋润彼此，变得亲近。**有效的相互回应是真爱的标志**。

3.  **你在对方心中是否是特殊且唯一的？**
    *   他只对你一个人好，还是对别人也一样好？
    *   他说过的感动你的话，是只对你一人说，还是对很多人说？
    *   如果他对很多异性都热情，而你只是其中之一，你就会感到被欺骗，并怀疑他的爱。
    *   情侣常对彼此说：“你是我心里最特别的一个。” **“特别”和“唯一”是真爱的标志**。

4.  **对方爱的是你本人，还是你的用处？**
    *   如果他告诉你他很爱你，但也坦诚这种爱是因为你的家庭背景、高学历或对他的事业有帮助，你仍会感到不舒服，并怀疑这份爱的可靠性。

如果对方符合以上四个特征，你就有很大把握判断他是真的爱你。

你看，真爱的这四个特征，既是维系父母与孩子的情感纽带，也是维系亲密关系的情感纽带。
*   孩子会跨越千山万水寻找妈妈，恋人也会如此。
*   妈妈看到孩子的眼睛会微笑，情侣之间也会相视一笑。
*   情侣说“你是我心里最特别的一个”，孩子则会说“我只喝妈妈倒的奶”。
*   女生和男生会介意对方是否只爱自己的钱或外貌，孩子也会质疑父母是否只爱自己乖巧或成绩好。

## 爱的系统观：为了稳定而设计

其实，人终其一生都在寻求与爱人在情感上的联结，渴望从爱人身上获得亲近和回应。当两个人的情感足够强烈、有回应、彼此视对方为特殊唯一、重情感而非利益时，双方就会对彼此的爱感到满意。而这种满意又会增强彼此的连结。

如果不是这样，他们就会愤怒、伤心、感到被背叛，甚至想要离开这段关系。

为什么会有这样的设计呢？答案是：**为了让我们这个（关系）系统更稳定**。

如果从系统的角度看，有什么力量能把两个完全不同的人结合在一起，让他们甘心受限制，甚至为彼此牺牲自我呢？

以下是维持系统稳定的四个必要条件，它们恰好对应了真爱的四个特征：

1.  **力量必须足够大**：大到如果两个人不在一起就会彼此想念，如果要分开就会痛不欲生。所谓“只叫人生死相许，不爱江山爱美人”。
2.  **系统内信息交流必须畅通**：两个人对彼此的需要、想法和感受要足够敏感，要能够并愿意交流与回应彼此。
3.  **必须划定系统与外界界限**：系统里的人不能随意属于另一个同类的系统。因此，系统里的人必须认定彼此是特殊和唯一的，否则系统就太容易崩溃重组。
4.  **吸引力必须建立在稳定基础上**：为了让吸引力持久，它必须建立在对方稳定的内在特质上，而不是可变的外在条件上。也就是说，吸引对方的是你这个人本身，而不是你的外貌、财富或学历。

说到这儿，我想你已经明白了，这个吸引力就是**真爱**。而真爱的这些特质，其实就是为了维持“亲密关系”这个系统稳定有效而设计的。

> **爱就是为了在一起。**

如果这个系统不够稳定，人们就不会对系统有强烈的归属感，亲密关系也就无法变成能够超越自我、对抗时间的东西了。
