# 038.杨霞：青春期心理课：02.什么是学习专注力_note

在本节课中，我们将要学习“学习专注力”的核心概念，了解它与日常活动中“专注”的区别，并探讨其背后的心理与生理机制。我们还将介绍一些有效的训练方法，帮助提升青少年的学习专注力。

---

许多家长反映，孩子在学习时难以集中注意力，但在打游戏、阅读或做自己喜欢的事情时却非常专注。这种在特定活动中表现出的专注，是否就是我们所说的“学习专注力”呢？

我们现在所指的**学习专注力**，特指一种**全神贯注**的状态。这意味着在上课时，你的眼睛能看着老师，嘴巴能回答问题，手能记笔记，耳朵能听讲，同时大脑还能与老师同步思考。

**全神贯注 = 眼 + 耳 + 口 + 手 + 脑 协同工作**

然而，为什么有些同学在阅读、玩游戏或看电视时可以专注，上课却不能呢？关键在于，阅读主要用眼，玩游戏用眼和手，听故事用耳——这些活动大多只调用了一两个感觉通道，属于“单通道”处理。

真正的全神贯注，要求所有的感觉器官——眼睛、耳朵、思想、记忆、手和身体——都集中在一件事上，并且协调一致。大脑需要随时指挥：眼睛看老师，耳朵听讲课，手准备记笔记或发言，大脑迅速思考。同时，老师讲授的知识、方法、定理和定义，都需要被及时存储到大脑中相应的“记忆仓库”里。

那些注意力不集中的孩子，往往无法做到这一点。他们既不能在上课时保持专注，课后也难以回忆所学内容。这是因为他们的感觉器官与记忆系统之间没有建立起有效的联系，信息传递的“通道”没有打通。

这个将感觉信息传入大脑、经处理后再指挥行为的能力，在心理学上称为**感觉统合能力**。这种能力通常在25岁之前逐步形成和完善。

以下是可能影响感觉统合能力发展的早期因素：
*   母亲怀孕期间：早产、先兆流产、胎位不正、因保胎长期卧床。
*   分娩过程：剖腹产、难产。
*   婴儿期：缺乏足够的摇晃、爬行不足。
*   童年期：缺乏协调性运动，过多静态活动（如读书、看电视、玩游戏），户外活动不足。

如果孩子在15岁前未能形成良好的感觉统合能力，就会直接影响到学习专注力。因此，我们所说的学习专注力，是一个完整的过程：感觉器官接收信息，信息传递并存储于大脑，大脑再指挥神经系统和肌肉去完成任务（如听指令、写作业）。

许多专注力不足的孩子，在执行课堂任务时会出现问题。家长可能觉得孩子在家看电视、玩游戏时很专注，但老师却反馈孩子上课走神、作业完成不好，甚至在考试时也难以集中精神。这些都是学习专注力不佳的表现。

有的家长会问：通过训练弹钢琴，是否可以提升孩子的专注力呢？对于感觉通道未打通的孩子，弹钢琴这类需要复杂协调（记谱、手脚眼耳配合）的任务同样难以完成。因此，弹钢琴是专注力良好的一个“结果”，而非训练专注力的“原因”。孩子需要先解决专注力本身的问题，才能更好地进行这类复杂活动。

那么，如何训练学习专注力呢？我们需要通过专门的方法，调动孩子所有的感觉器官、大脑指挥中心以及四肢的肌肉神经系统，进行协同训练。

以下是几个有效的训练方法示例：

*   **抛接球平衡训练**：让孩子站在凳子或平衡台上，与他进行抛接球。他需要眼睛盯球，迅速接球并抛回，同时保持身体平衡、不摔下来。这训练了视觉追踪、快速反应、身体控制和注意力集中。
*   **拍球与跳绳**：每天坚持拍球和跳绳各300-500下（针对青春期孩子）。这类规律性的协调运动能有效刺激感觉统合。
*   **攀岩**：在攀岩过程中，孩子需要控制全身肌肉、眼睛和躯干，紧密配合以完成动作，是极佳的综合训练。

这些看似简单的活动，都在训练手、脚、眼、脑为同一件事全神贯注地配合。家长可以积极探索类似活动，或寻求心理医生的专业指导。

训练的关键在于持续。例如，每天先进行拍球跳绳，再去写作业或听课，能有效提升后续活动的专注力。当这种行为训练持续半年以上，大脑指挥行动就会逐渐成为一种习惯性的行为模式。

---

本节课中，我们一起学习了**学习专注力**的本质是**全神贯注**与**感觉统合**，理解了它与单通道专注的区别，并认识了其发展的关键期和影响因素。最后，我们介绍了几种通过**行为训练**来提升专注力的实用方法。在后续课程中，我们将进一步讲解更多训练技巧。
