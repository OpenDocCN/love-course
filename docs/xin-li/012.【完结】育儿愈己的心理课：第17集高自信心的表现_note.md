# 012.【完结】育儿愈己的心理课：第17集高自信心的表现_note

在本节课中，我们将要学习高自信心孩子的具体行为表现。了解这些表现，有助于家长识别并培养孩子的自信心。

一个自信心高的孩子，存在感非常强。他们认为自己的存在不需要向任何人证明。他们拥有强大的内心自我。

上一节我们介绍了高自信心的核心特征，本节中我们来看看高自信心孩子的具体行为表现。以下是高自信孩子的七点主要表现：

1.  有较高的独立性。
2.  有愿意承担责任的品质。
3.  对自己的成就感感到骄傲。
4.  能够坦然面对失败。
5.  乐于接受新的挑战。
6.  在社交中表现从容。
7.  能够清晰表达自己的需求和想法。

这些表现共同勾勒出一个内心强大、行为积极的自信孩子形象。

本节课中我们一起学习了高自信心孩子的七种典型行为表现。理解这些表现是帮助孩子建立稳固自信心的第一步。
