# 075.透视挣钱花钱背后的心理：第24讲：你活得可能是一个“假的我”（假

在本节课中，我们将要学习“假我”状态是如何形成的，以及它如何影响我们感知幸福和获取财富的能力。我们将探讨“假我”的表现、成因，并理解“真我”才是获得内心满足与外在成功的基础。

上一节我们介绍了“假我”的概念，本节中我们来看看“假我”的具体表现及其深层原因。

有些人为了维持一个被塑造得很高的外在形象，会在“本我”部分极度放纵。**公式：维持高形象 -> 内心压力 -> 本我放纵**。例如，有人会通过找小姐并上瘾来释放压力，他们把性当作释放过程。

这种行为与亲密关系不够亲密有关。这样的人似乎已经失去了建立亲密关系的能力，即“爱无能”。

以下是“爱无能”的一些表现：
*   年轻人对生活缺乏兴趣，觉得什么都“没意思”。
*   对他人缺乏兴趣，导致爱情难以发生。
*   无法发展更高层次的爱的需求。

“本我”是发展情感能力的基础土壤。如果“本我”被长期压抑、低龄化或婴儿化，就无法发展出更高层级的爱的需求。当一个人连最基本的自我照顾需求都满足不了时，讨论爱是做不到的。

许多家长给予的是“有条件的爱”，例如“你考100分我才爱你”。这种爱最终会让孩子失去爱的能力，因为他们认为爱意味着牺牲自己以满足他人。

绝大多数父母都会这样做，这使得“假我”状态的形成是一个漫长过程。通过许多人事物的不断强化，个体逐渐确信：只有在“假我”状态里才能活得好。

“假我”被误认为是一种需要提升的技能，例如察言观色、讨好他人。这类课程属于成功学分支，而非心理学。这些技巧不走心，没有用。

通往人的情感部分，永远依靠情绪、感受和对世界的感知。如果没有这些，人就和机器人无异。机器人更擅长数据分析和逻辑判断。

当前许多训练都在试图把人变成机器人，那不是真实的你。真实的你会哭、会怒、会求助、会大笑。当然，真实的表达需要符合他人和环境。

情绪表达如同穿衣服，需要找到适合的情绪，在合适的环境里展现。讲真实不是让人活得很自私。

一个成熟的、真实的人应该是什么样子呢？他既能真实地做自己，还能知道别人的感受，并且符合当下环境。**公式：成熟真实 = 真实做自己 + 体察他人感受 + 符合当下环境**。

听起来，我们有一个重新养育自己的过程。我们不能要求父母改变，因为他们可能自身就是“假我”的体现。

例如，有的父亲在家像“政委”，情感不生动；而父母在处理彼此矛盾时，可能又表现得像孩子，能力糟糕。他们在对待孩子时，却摆出另一副“假我”状态，这让孩子感到错乱。

既然我们无法改变父母给我们的前半生，那么后半生的命运是可以自己修改的。我们可以通过学习和练习，来让心态发生改变。

当我意识到自己活在“假我”里时，我就明白了：原来当我活出真我时，荣誉、价值、成就、金钱、伴侣都是随之而来的奖励，而非需要拼命追求和交换的目标。

一个真正能活出自我的人，会拥有这一切。社会上有少数活得真实、通透、洒脱的人，他们拥有很多追随者，追随者自愿给予资源和支持。

创业也是如此。当你做特别热爱的事情时，别人看到了，就愿意投资你。

当你活出真实时，物质、精神、情感等一切都会以一种让你舒服的状态呈现。你不用去抓、去追求、去努力，会很自然地拥有一切。

就像寻找“灵魂伴侣”，它不限于异性，而是一种永恒的成长关系，可以是一本书、一只宠物等任何能与你的灵魂共鸣的事物。当你找到共鸣时，那种相互辉映的感觉能瞬间疗愈你。

当你能活出真我时，会发现很多积极资源向你靠近。这就是“我自盛开，清风自来”。

当我们知道活出真我就能获得幸福时，金钱就不用特别担心了。做热爱的事情，一定能做出好成绩，金钱作为优秀成绩的奖励自然会来。

就像学校里考第一名的学生，他只是做好了“好好学习”这一件事，却得到了奖学金、保送、肯定等所有资源。

现在回到我们自身。只要你能做好一件事：**把自己真正照顾好、抚慰好**。那么，其他东西别着急，你会发现它们会慢慢走向你、靠近你。这与“追逐幸福却追不到”是完全不同的体验。

本节课中我们一起学习了“假我”的成因与影响，并理解了“活出真我”是获得幸福与财富的根基。我们认识到，幸福不是拼命追逐来的，而是在照顾好自己后，不期而遇的。
