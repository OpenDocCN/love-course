# 038.杨霞：青春期心理课：16.孩子过于敏感怎么办_note

在本节课中，我们将学习青春期孩子敏感心理的成因，理解自我意识的发展，并掌握帮助孩子建立恰当自尊心与控制敏感情绪的具体方法。

## 概述：青春期敏感心理的根源

上一节我们介绍了青春期孩子的心理特点，本节中我们来看看孩子过于敏感的具体表现与深层原因。

青春期是孩子自我意识发展的关键阶段。自我意识指个体对自己的看法，以及如何看待他人眼中的自己。自我意识包含自尊心、自律与自治。所有对自我的认识与控制都属于自我意识范畴。

自我意识在青春期占据非常重要的位置。它的发展决定了孩子对自己的看法，以及在此基础上是否具备自控能力。

## 自我意识过强带来的问题

我曾遇到一个孩子，他认为别人都在嫌弃他。起因是一次同学无意间捏鼻子说“什么味儿”。他开始怀疑同学是否嫌弃自己身上有异味。那天化学课正好提到二氧化硫的臭鸡蛋味。此后他经常上学前问妈妈：“我身上是不是有臭鸡蛋味？”他认为同学捏鼻子的动作是针对自己。这实际上是一个偶然关联。

孩子过于敏感源于青春期特有的“我向思维”。我向思维指认为所有人与事都指向自己。我们要注意，家长不应过度关注孩子。家庭是小社会，若家长过分关注，孩子会习惯这种关注，并认为外界也同样关注自己。

孩子会形成条件反射，认为同学、路人、邻居、老师都在像父母一样盯着自己。家长越关注，孩子自尊心往往越强。自尊心过强会导致过度敏感与自我中心。

自我中心的结果是自恋、自怜、自卑，有时甚至自我膨胀。心理学家研究发现，若13、14岁时自我膨胀，17、18岁时可能变为爱抱怨的人。这会影响情绪，导致不快乐。

## 培养恰当自尊心的方法

我们需要注意，孩子有一定自尊心是正常的。此阶段应学会尊重孩子的隐私、选择与人际关系。但并非100%的事情都需与孩子商量或由孩子决定。

我见过一个孩子不能提“考试”。一提考试他就紧张哆嗦，浑身发抖，出现焦虑发作。一次咨询时，我正常询问：“我听说你考试时会紧张。”刚提到“考试”二字，孩子立刻开始咄咄逼人。家长马上说：“你怎么能提考试？我的孩子不能提这个词！”后来孩子不能上学，最终去安定医院服药。家长这是过分保护了。

过分保护会导致孩子不能听就不提、不能见风就不出门、不能见光就不拉窗帘。孩子会成为温室花朵，将来可能成为废物。发现孩子对某事敏感时，我们应思考原因。

例如孩子考试紧张，可能因为期望值过高、太想得第一，或平时复习未模拟考试状态，未适应考试压力与紧张环境。我们可以帮助孩子通过锻炼身体、写作业时模拟考试状态、户外运动，让神经具备“顿感力”。

**顿感力**指迟钝的适应力，而非敏感力。若神经系统过于敏感，如同惊弓之鸟、草木皆兵，将难以生活。人需要一定顿感力，即培养坚强毅力、耐力、适应能力与抗挫折能力。

## 实践锻炼：从运动到挫折教育

我们应有意识地在平时帮助孩子。例如参加运动比赛，适应有赢有输的环境。

一位澳大利亚大学校长在迎新时说：“同学们，你们来到我的学校，我校是提供挫折与打击的场所。大学的任务是培养抗挫能力，教会你们如何沮丧，如何应对生活中的打击、情绪低落与失败。我是来给你们失败的。”

当时许多孩子与家长很惊讶，从未听说校长这样讲话。但后来发现这很重要。因为人可能学习成功、尊重、得意、鼓励与正能量，却忘了也要学习应对沮丧。若未学习如何应付沮丧，每当面临时就会手足无措。

我们在帮助孩子学会调整与控制自尊心时，实质是帮助他适应未来社会。让他适应劳动、运动、人际关系、沮丧、打击与逆境。在不断适应过程中，形成恰当的自尊心。

自尊心并非完全来自他人的尊重。它是相互的，基于我知道自己的优点与缺点。优点应发挥特长，缺点需学会克制、避免、补偿或成长改正。这是一个客观全面看待自己优缺点的过程。

## 具体练习：优缺点清单与调整方案

我们可以请青春期孩子拿出一张纸，完成以下练习：

以下是具体步骤列表：

1.  写出你的10个优点。
2.  写出你的10个缺点。
3.  面对优缺点，提出自己的理想：将来想干什么。
4.  思考如何极大发挥优势以获得成功。
5.  思考如何扬长避短。
6.  针对缺点，规划如何成长、改变、修复、改正或矫正。

让孩子自己提出方案。改正不了的缺点避免它，能改正的提高它。不要总盯着缺点（否则自卑），也不要总盯着优点（否则自大）。恰当的自尊心基于对自己客观的分析。

当我们帮助孩子形成对自己优缺点的客观分析，找到发挥优点的合适途径，并培养良好的环境适应能力时，孩子才能对自己的自尊心进行恰当的控制与调整。

## 总结

本节课中，我们一起学习了青春期孩子敏感心理的根源——自我意识的发展与“我向思维”。我们探讨了过度关注与保护可能带来的问题，如自恋、自卑与自我膨胀。关键是通过培养“顿感力”、挫折教育以及客观的自我分析，帮助孩子建立恰当的自尊心与情绪调节能力。最终目标是让孩子在认识自己优缺点的基础上，学会适应环境，健康成长。
