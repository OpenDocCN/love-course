# 118.60秒口才训练营（完结：付费课：【付费课22】思行说训练之过程

在本节课中，我们将要学习如何通过“过程回顾”训练，找到并进入“心流状态”，从而让你在练习口才或做其他事情时，既能享受过程，又能取得成果。

---

上一节我们介绍了价值回顾和成果回顾。本节中，我们来看看如何通过回顾做事的过程，找到“心流状态”。

做一件事时，从过程和成果两个维度思考，通常可以分为四种情况：
*   **过程痛苦，但有成果**：例如，一些创业者过程煎熬，但最终成功。
*   **享受过程，但无成果**：例如，一些人坚持做感兴趣的事，却难以养活自己。
*   **过程痛苦，且无成果**：例如，不喜欢自己的工作却不辞职，痛苦且难有成就。
*   **享受过程，且有成果**：这是最理想的状态。

最理想的状态，就是既享受过程，又能取得成果。这种状态在心理学上被称为 **“心流”**。

**心流** 是指人们全身心投入某件事的一种心理状态。处在这种状态的人不愿被打扰，甚至不想中断。例如，作家写作、玩家玩游戏时，常处在此状态。

那么，我们如何在工作中进入心流状态呢？这需要向游戏设计者学习，满足三个核心条件：

以下是进入心流状态的三个关键条件：
1.  **清晰的目标**：你需要明确知道要做什么。
2.  **能力与挑战相当**：公式可表示为：`个人能力 ≈ 任务挑战度`。如果任务太难（`挑战度 > 能力`），你会焦虑；如果太简单（`挑战度 < 能力`），你会感到无聊。
3.  **即时的反馈**：在行动过程中能快速看到结果或进展。

其中，第二点“能力与挑战相当”尤为重要。以口才训练为例：
*   **清晰目标**：最初目标是“45分钟内，脱稿流利生动地讲一个60秒左右的故事”。
*   **调整挑战度**：如果觉得太难，可调整为“看关键词讲述”或“将时间放宽至80秒”。如果觉得太简单，可挑战“25分钟内完成”或“严格将时间控制在60秒”。
*   **即时反馈**：练习本身（是否脱稿、用时多少）就是反馈。此外，找伙伴点评、完成目标后自我奖励、参加直播演讲获取教练反馈，都是有效的反馈方式。

理解了这三个条件，你就可以进行每日的过程回顾。

以下是每日过程回顾的具体步骤：
1.  为你当天做过的事，就“享受过程”的程度打分（1-5分，4分以上为享受）。
2.  针对打分高（≥4分）的事，分析是满足了哪几个心流条件，并思考如何优化。
3.  针对打分低（≤3分）的事，分析是哪个心流条件严重不满足，并计划如何改进。

对于这两类分析，你都可以录制成“玩练”节目。此外，你还可以对一整天进行整体分析并录制节目，结构如下：
*   今日过程回顾的整体情况。
*   你的主要感受。
*   你的思考、启发与改进措施。

因此，本周你至少可以录制三期节目：分析一件享受的事、分析一件不享受的事、以及一整天的整体分析。这样既能提升思考分析能力，又能锻炼表达。

当你真正享受做一件事时，你更有可能把它做好。请对照心流的三个条件去调整你的行动。

最后，以积极心理学家、心流概念提出者米哈伊的描述作为结尾：
> “当一个人处在心流状态的时候，你会感觉自己完完全全的在为这件事本身努力，就连自身也都因此显得很遥远。时光飞逝，你觉得自己的每一个动作、想法都如行云流水一般发生发展，你觉得自己全神贯注，所有的能力都被发挥到极致。”
