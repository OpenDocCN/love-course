## 第一部分：领导如何收拾不听话的下属

上一节我们介绍了课程概述，本节中我们来看看领导如何针对那些不服从管理、自视甚高的下属。

有一类人“恃才傲物”，倚仗才华，目空一切。他们可能确有才干，也可能是刚出校门、不懂规矩的愣头青。对于这种不知天高地厚的下属，领导常用以下方法。

以下是领导常用的三种方法。

**方法一：专攻薄弱环节**
即便才华横溢者，也有其短板。领导会刻意让其从事不擅长的工作，反复挫败其自信。
*   **公式**：`打击自信 = 反复安排不擅长任务 + 持续否定成果`
*   **举例**：若你口才好但公文差，领导便天天让你写公文并反复修改；若你笔杆子硬但口拙，便安排你负责接待，让你在饭局出洋相。目的就是磨掉你的傲气，让你学会“听话”。

**方法二：揪住错误不放**
人皆会犯错。但如果你平时不听话，领导会在你犯错后将其夸大，并时常旧事重提，以此作为牵制你的把柄。
*   **公式**：`制造把柄 = 抓住错误 + 夸大性质 + 时常敲打`
*   **举例**：领导将你的小错上纲上线，变成“风筝线”。你听话，就能继续“飞”；不听话，他就收紧线，让你跌落。

**方法三：给予“冷板凳”**
对于渴望证明才华的下属，若其不敬领导，领导会故意“雪藏”其长处，不安排任何能展现其能力的工作。
*   **公式**：`雪藏 = 明知长处 + 故意不用 + 闲置处理`
*   **举例**：让你无事可做，如同单位里的“空气”。时间一长，英雄无用武之地，自豪感便会消磨殆尽。

### 案例分析：诸葛亮如何驾驭关羽

为了帮助大家更好地理解上述方法，我们来看一个历史案例：诸葛亮如何收服高傲的关羽。

1.  **坐冷板凳**：赤壁之战，诸葛亮明知关羽善战，却不给他分配主要攻坚任务，先让其坐冷板凳。
2.  **专攻弱点**：最后安排关羽去守华容道。诸葛亮深知关羽重义气，且曹操有恩于他，此举正是利用其“重义”可能违背军令的弱点。
3.  **抓住把柄**：事前让关羽立下军令状。这便抓住了其“小辫子”，掌握了主动权。
4.  **施恩驯化**：关羽果然放走曹操。诸葛亮手握军令状可杀关羽，却选择赦免。这一连串的“挫败-抓把柄-施恩”组合拳，彻底驯服了关羽，让其甘心臣服。

这个案例也解释了为何太平年代，武将常被文官驾驭：文官工于心计，善用权谋。

## 第二部分：小领导如何通过分配任务收拾你

上一节我们探讨了针对“刺头”的通用手段，本节中我们来看看直接管理你的小领导（如科长）如何通过日常工作安排来施加影响。

不要小看科长的权力，他们虽不能直接决定你的升迁去留，却可以通过任务分配巧妙收拾你。以下是科长分配工作时的小心思。

以下是科长分配不同类型工作的策略。

*   **核心工作**：绝不会让一人独占。科长会确保核心岗位永远有替代者，甚至亲自顶上，以防有人变得“不可或缺”，从而失去控制权。
*   **临时突发工作**：将其视为“测试台”。通过下属对这类任务的态度和执行情况，来筛选“可培养”与“不可培养”的对象。
*   **不易出成绩的“垃圾”工作**（如信访、文艺）：分配给“不可培养”的下属。这类工作耗时耗力，出错易见，立功却难，且缺乏接触大领导的机会，极易消耗一个人。
*   **容易出成绩的“好”工作**：分配给亲近、听话的下属。这是一种变相的奖惩，传递“听话有好处，不听话没好果子吃”的信号。
*   **高风险、难搞定、易背锅的工作**：分配给经常反对自己的“刺头”。若其拒绝，可向上级状告其不服从管理；若其接受但出错，则可借此机会“参他一本”，将其清除出团队。

通过这样的任务分配，科长能有效凝聚团队，排除异己。这也说明，老好人往往带不好团队，懂权谋、善用手腕者才是管理人才。

## 第三部分：帝王术——驾驭能人的“三起三落”法

从科室的微观权术跳出，最后我们来看最高明的驾驭之术：古代帝王如何驯服能力超群的功臣，即“三起三落”之法。

成熟的政治家每一个动作都服务于政治。对能臣“用之又贬，贬之又提”，反复数次，绝非儿戏，而是精妙的驯化过程。

以下是“三起三落”能起效的核心原因。

1.  **制造持续感恩**：赏赐应与功劳匹配。当赏无可赏时，便找机会剥夺其所有，再找机会重新施恩。`赏赐循环 = 立功 -> 赏赐 -> 剥夺 -> 再立功 -> 再赏赐`。这能让他始终感恩，若赏赐停止，则可能生变。
2.  **破除势力圈子**：能人立功易形成个人威望与圈子。通过大起大落，让其体验“门庭若市”到“门可罗雀”的世态炎凉。几次折腾后，他便不愿再让人依附，其势力圈子自然瓦解。
3.  **明确权力来源**：通过重用与打压的反复，让其深刻认识到，所有功劳与权力完全来源于领导给予的平台、机会和支持，而非其个人能力。`权力认知 = 功劳是领导给的，而非自己固有的`。

这种做法看似无情，却是维护大局的明智之举。例如，毛泽东对邓小平，汉武帝对窦婴，都运用了此术，以防能臣功高震主。
