# 102.熊浩哈佛谈判术：生活无处不谈判（完结）：021.谈判要素：沟通品质之情绪管理（上）_note
## 021. 谈判要素：沟通品质之情绪管理（上）😤

在本节课中，我们将要学习谈判核心要素中的“沟通品质”，并重点探讨其重要组成部分——情绪管理。我们将了解情绪产生的根源，并学习管理情绪的第一个关键技能。

在之前的课程中，我们已经强调了ICON要素，即谈判的七个核心要素中的前四个：**利益**、**标准**、**选项**以及**替代方案**。这四个要素最关键的使用时机是在谈判之前。在谈判开始前，你必须清晰地规划好这四个版图。

如果简要勾勒其思维逻辑，它应是一个连贯的思维导图。

以下是其基本流程：
1.  **询问利益**。
2.  当利益的土壤足够丰沛时，进入**选项的设计**。
3.  处理选项：非冲突的利益通过交换实现最大化；对于冲突的利益，则需设定**客观标准**进行博弈。
4.  当谈判没有成功时，走向你的**替代方案**，并用它来决定你的强硬程度以及离场时机。

大家需要意识到，ICON要素并非平行并列，而是一个从入口进入，用选项进行筛查，再进入冲突利益处理（客观标准）和谈判失败预案（替代方案）的、像智慧树一样伸展连接的思维导图。

处理完谈判前的准备与筹谋，从本次课开始，我们正式进入“谈判之中”的阶段，即谈判正在进行时我们应该考量的核心要素。这就是谈判七个要素中剩下的三个：**沟通品质**、**人际关系**和**承诺**。

今天我们讨论**沟通品质**。沟通品质关注的是在对话过程中，如何实现相谈甚欢，以及如何管理好自己。这个话题涉及的内容非常细碎，因为一旦涉及沟通，就包罗万象，难以穷尽。

在我的课程中，我将集中讲解两件事。
第一件事是**情绪管理**，主要处理当你或对方即将发飙时，应如何应对与管控。
第二件事是**信息优化**，即我们如何锚定事实，避免受立场、偏好或各种无端诠释的影响。
因此，沟通品质部分将分这两个大单元来完成。

在情绪管理部分，我们今天讲两件事。
第一，我们要讲一讲人为什么会爆发情绪。
第二，我们要讲一讲怎么样来管理自己的情绪。管理情绪涉及四种特殊技能，今天可能会讲到其中一种。

首先，人为什么会爆发情绪？理由非常简单：因为你是人。
人是一种历史性的存在。解释人的行为，除了社会关系、个人成长、教育背景、社会阶层，一个容易被忽略的因素是上万年的进化历史。它深深沉淀在我们的大脑中，潜伏在我们的基因里。

1929年，美国著名心理学家沃尔特·坎农概括出人类在遇到压力、风险和慌张时的一种本能反应模式。
这种反应模式被称为 **`Fight or Flight Response`**，中文译为“战或逃反应”。

试想一个家庭情境：当你妈妈唠叨你时，你会怎么处理？很多小伙伴只会采用两种方式：要么吵回去、怼回去（战），要么静默无声、视妈妈为不存在（逃）。这种状态其实是本能地处理压力与风险情境的反应模式。

为什么会有这种反应模式？为什么一旦遇到压力，我们会快速做出某种回击性反应？因为在数万年的人类进化中，这是人类求生的基本本能。

设想我们不是生活在现代社会的今天，而是生活在数万年前的洞穴中。今天，你我作为洞穴原始人，被安排外出觅食。当我们走进森林，突然听到前方草丛传来窸窣声响，估计有野兽时，我们应该怎么做？
正确的做法是：要么赶紧拿起武器，保持紧张，准备战斗；要么撒腿就跑。只有这样，才能保证我们的生存。在相当长的历史中，人类非常弱小，没有科技赋能，没有武装包装，只能通过“战或逃”这种最本能的策略来保护自己。

直到最近三百年，随着工业革命和科技的迅猛发展，人类才变得空前强大。今天，我们不仅与自然交换，甚至敢凌驾和“欺负”自然。只有当人类足够强大时，“求真”的价值观（勇于向前、冒险、探知）才会压倒“求生”，成为主流意识形态。因此，我们今天鼓励人们去探险、创造和发现未知。

这种“求真”价值观的历史不过三百年，而“战或逃”反应模式却累积了数万年。所以，在谈判或沟通过程中，当我们嗅到风险、感到压力或不安时，最本能的反应就是应急反应——情绪爆发，准备“战斗”。

因此，人与人之间的互动便容易出现紧张。了解到情绪反应是正常的、是人类进化的本能，我们才有可能理智而平和地看待情绪，并加以处理。你在对话中对家人发怒，或在职场中对同事摆脸色，这不意味着你错了，而意味着你是“真人”。但这种本能反应需要技术加以管理。

在沟通品质部分，我们将分别介绍四种情绪管理技法。今天讲第一个：**识别**。

什么是识别？它包含两个维度的事件。
第一，识别对方是否处在恰当的情绪中。
第二，识别自己是否处在恰当的情绪中。

我们先谈对对方的识别：如何识别他情绪良好或即将发飙？有的小伙伴可能会说，有的人翻脸比翻书还快，这居然可以识别？答案是完全可以。你需要多少时间去识别？四分之一秒。

根据著名的表情研究专家、社会心理学家保罗·艾克曼的研究，人类的基础表情分为若干种。他指出，人类所有的表情可以归结为七种基本类型：**悲伤**、**愤怒**、**惊讶**、**恐惧**、**恶心**、**轻蔑**和**高兴**。无论表情如何变化，万变不离其宗。人类可以在四分之一秒内对这七种表情进行分类。

在我们的课程资料包中，提供了一个链接（可能需要手动输入）。打开这个链接，你会进入保罗·艾克曼教授设计的一个非常有趣的表情识别游戏。在这个游戏中，每隔四分之一秒，画面中的人会突然闪现一个新的表情，然后又快速回归常态。下方会出现七个选项，你需要在这四分之一秒的闪现后做出判断。绝大部分人都可以正确选择。即使选错，也能对表情的大致方向（积极/消极，开心/忧伤）做出非常准确的判断。

我想告诉大家，我们第一个关于情绪管理的工具，就是对表情的识别。你只需要四分之一秒就可以对谈判对手的表情做出精准判断，从而对其情绪做出精准预估。

可能有小伙伴会问：表情能不能伪装？对方明明不开心，却要表现出开心。报告各位，非常困难。除非接受过像美国联邦调查局那样的专业训练，或者职场环境经常要求你心口不一，否则绝大部分人无法掩饰自己的表情。表情不大受控制，它是人的本能和条件反射的一部分，所以它会“出卖”真实情绪。

**第一个识别**，就是通过表情，预判你的谈判对象是否处在恰当、合适的对话氛围和情绪中。

第二件事，是对自我进行识别：你自己是不是要发飙了？你自己是否处在良好的情绪状况中？如何做呢？

假设你在跟一个人谈判，结果你对这个人非常不愉快，觉得他糟透了、毫无教养、说话拖沓。这种评价你不会说出来，但会有一个内在的声音（`internal voice`）在你大脑中不断循环、运转。

所谓自我的识别，就是和自己情绪的声音完成一次对话。我们建议完成以下两种自我对话：
1.  **对自己的情绪进行评分**。例如：“我不高兴了，看见这个人觉得恶心。我的恶心指数是多少？1分是非常不恶心，6分是极限。”
2.  **对情绪产生的原因进行洞察**。例如：“我为什么会对他产生恶心？”

总而言之，就是对自己的情绪进行自我对话。《Getting to Yes》的作者威廉·尤里说过，如果我们能够知道自己遇到了那种真实的感受，我们就可以去克制它，防止那些于事无益的习惯性反应出现。

所以，与自己的情绪进行对话，对自己的情绪进行评估，这本身就是在对情绪进行管理了。

如果你通过快速阅读表情，发现对方马上要发飙了；或者通过与自己情绪的友善对话，发现自己马上要发飙了，你就可以让谈判过程先暂停下来。如果你发现情绪仍然可控，就可以继续推进谈判进度。

本节课我们进入了“沟通品质”的学习，并讲解了两个核心观念。
第一，我们为什么会生气？这很正常，它是人类大脑皮层的本能反应，是千年进化的必然后果。
第二，如何管理这种本能？我们将介绍四种方法。今天介绍的第一个方法叫**识别**：通过表情快速识别对方的情绪；通过与自己内在声音的对话，来辨认和管理自己的情绪。

关于其他的情绪控制方法，我们下次课再见。
