# 课程022：正念身心疗愈法 🧘♀️💖

在本节课中，我们将学习如何运用正念诗歌来面对内心的脆弱与伤口，通过接纳而非回避，促进深层的疗愈与成长。

## 课程概述

本课程由十点读书与时间之道联合出品，旨在帮助大家安定内心，实现向上生长。欢迎来到胡婷婷的正念诗歌电台。很多时候，我们缺乏勇气去正视内在的伤口，不愿面对自身脆弱的时刻。

这种持续的回避与视而不见，反而使旧伤难以痊愈。这如同一个很深的伤口，为了避免看见，我们随意在表面贴上创可贴。但只有你自己知道，在创可贴之下，深深的伤口依然存在。

如果你能学着允许自己勇敢地看见并承认伤口的存在，并在每次看见时，小心地往里面撒一些药，伤口就会开始真正的愈合。在伤口隐隐作痛时，可以运用上节课介绍的疗愈空间练习。

## 核心诗歌赏析

那么今天，婷婷想与大家一同欣赏一首短小精悍的小诗，它节选自波斯诗人鲁米的著作。这首诗虽短，却值得我们反复体会与感受。

鲁米说：
**“不要转头，就看着绷带之处，那里，也是光照进来的地方。”**

这句话强调了**直面伤痛**的重要性。其核心概念可以表述为：
**疗愈 = 直面伤口 + 接纳光照**

## 实践步骤指南

以下是基于本课内容，可以付诸实践的简单步骤：

1.  **识别回避**：留意自己在哪些时刻或面对哪些情绪时，会习惯性地选择转头或逃避。
2.  **温柔注视**：鼓起勇气，将注意力轻柔地带回你所回避的“绷带之处”——即内心的不适或伤痛。
3.  **接纳光照**：理解并相信，正是你敢于注视的地方，才有机会让疗愈的“光”（即理解、接纳与改变）照进来。
4.  **持续练习**：将这种“不转头”的觉察态度，融入日常的疗愈空间练习或正念时刻中。

## 课程总结

本节课中，我们一起学习了正念疗愈中的一个关键态度：直面而非回避。通过鲁米的诗歌，我们认识到真正的愈合始于勇敢地注视伤口，因为那里正是光照进来的起点。记住，疗愈是一个从内而外、循序渐进的过程。
