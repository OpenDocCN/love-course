# 📚 课程011：从家长管教到孩子自律 立规矩的30个秘籍 - 01 规则意识：建立规则意识的3个阶段

在本节课中，我们将要学习什么是“规则意识”，以及它是如何在孩子身上形成和发展的。我们将了解规则意识建立的三个关键阶段，并帮助你判断自己和孩子目前所处的阶段。

---

当你去电影院看电影时，会提前检查自己的手机是否调成了静音或震动。电影放映时如果有电话打进来，你会下意识地用手捂着嘴巴接电话。

你会用很小的声音甚至弯下腰躲进椅子下面。等电影放完了你起身离开的时候，会把自己的饮料瓶、零食包装袋拿走丢进垃圾桶。整个过程当中并不会有人站在你身边，监督你关掉手机铃声，提醒你小声说话或带走垃圾。

但是你却这样做了。在公共场所小声说话不打扰他人、不随意扔垃圾的行为准则已经存在你的内心，你会主动自觉地遵守这些规则。这就是你的规则意识，也就是一个人发自内心的、按照规则行事的意愿。

我们可以看到，越文明的地方，大家就会越主动地遵守规则。而越主动遵守规则，其实就是一个人从内心当中，已经不需要他人提醒就能产生的行为，也就是习惯，或者说很多人把它叫做教养。

那么，规则意识是天生的吗？它在孩子身上又是如何表现的呢？

首先，我需要坦白地告诉大家，规则意识不是天生就有的。孩子最初并不会遵守规则，他们都会按照自己的喜好来。慢慢地，孩子们就开始遵守一些规则了。

而这些规则最早的提出者就是父母，就是在孩子身边的成年人。由此可见，最早孩子们遵守的这些规则，都是因为父母特有的权威感和强制性。

可能大家听到“权威感”和“强制性”，心里有一种本能的抵触。我也一样。但是事实就是这样。父母给孩子提出要求，而年龄越小的孩子，他的反抗能力和表达能力就越弱。所以在这个时候就让我们看到，就算是父母再不想体现权威感，而这种时候权威感是加在你身上的，你没有办法回避的。

说到这个结论，我就想提醒大家了：一旦这种权威感是被加在我们身上的时候，我们就更加要注意，我们自己提出来的这个规则是不是正确的了。例如，我们是会要求孩子每天晚上睡觉之前刷牙，还是说随意的？因为“随意”和“要求”都是父母提出来的，但是他并不会影响权威感的消失或者存在。孩子对于规则实施意识不理解，就会按照自己的喜好去做。那么这个时候，父母的要求是不是准确地提出，就显得很重要了。

著名的心理学家皮亚杰在此基础上提出了一个“规则内化”的理论。是不是听起来又要讲理论了？其实大家可以想象，它经历了三个阶段。

以下是规则内化的三个阶段：

1.  **第一阶段：接受外部规范**
    孩子把外人给自己提出来的规范，先听进去了。

2.  **第二阶段：执行外部要求**
    孩子开始执行这些被提出的要求。

3.  **第三阶段：内化为自身习惯**
    在他不断执行的过程当中，他就会形成自己的规则，也就是我们说的习惯。

就拿刚才我说的在电影院的那个例子来讲。在最开始的时候，每个人都是被要求的：进电影院不能接听电话，手机要调成静音。慢慢地，你看不见那个牌子，你也会这样做了。到了最后，你已经不记得自己为什么那样做，但是你就做了。就像走路，有的时候你说，哎，你为什么上台阶的时候会抬左脚呢？就算你不提醒自己，看见台阶，你也会不由自主地抬上去。

这个过程可以总结为：**外部规范 -> 内部执行 -> 形成习惯**。我们通常就把这个过程理解成为规则意识的建立了。

那么当孩子长大以后，我们就不需要每天提醒他要刷牙，他自己就会去做。一方面，是因为孩子在每天的练习当中越来越熟练，形成了习惯。而另外一个方面，孩子在刷牙的过程当中，父母不断地去告诉他刷牙的好处，以及如果不刷牙就会带来他身体上的伤害。例如：刷牙以后，你的牙齿就可以白白的，就不会长虫子了。也可以通过一些绘本，让孩子去看到这样的一些小故事。

所以，孩子内化了父母的要求。在这种情况下，孩子就会容易从情感上接受别人的要求和观点。慢慢地，他发现自己做的这些事情、遵守的这些规则，对于自己是有帮助、有价值的，于是这个链条就完成了。

我们来总结一下这三个阶段：

1.  **第一阶段**：别人对自己的要求。
2.  **第二阶段**：在自己做的过程当中，外部不断告诉自己这样做的价值，不这样做的坏处。
3.  **第三阶段**：变成了他的一个习惯。

这就是规则意识建立的全过程。

那么，您的孩子现在处于哪个阶段呢？或者我问得更直接一点：作为父母、作为家长，你对规则意识的理解在哪个阶段？

一个规则意识强的孩子，在生活当中会表现得井井有条。他有规律的作息，按时吃饭，按时睡觉。在公众场合不会乱吼乱叫，在幼儿园和小朋友相处融洽，上课坐得住，玩游戏时知道轮流排队。

相反，一个规则意识弱的孩子，就有可能出现在另外的情况。晚上八点就困了，但是今天十点还在兴奋；饭点没胃口，其他时间又嚷嚷着饿；一出門就撒野，父母完全不敢松手，不知道下一步会干什么；在人多的時候，孩子不讲礼貌，横冲直撞，引来周围人嫌弃的目光。

所以这个时候，我们会看到，这对孩子自己有影响，对他的安全有影响，对于陪伴在他身边的成年人，也是焦头烂额。

虽然规则意识的强和弱，在孩子行为表现上有着天壤之别。但是在这里，我想告诉大家的是：孩子在六岁之前，都会经历一个非常非常重要的规则秩序敏感期。如果在这个阶段，父母可以把握住，将会对孩子的规则意识的培养，起到很好的促进作用。关于这个时期该怎么做，会在后面的课程当中有具体的讲解。
