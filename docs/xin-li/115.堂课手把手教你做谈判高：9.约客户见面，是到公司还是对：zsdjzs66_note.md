# 谈判高手课 115：谈判的实质结构——场地选择的玄机 🏢🏌️‍♂️

在本节课中，我们将探讨谈判中一个常被忽视但至关重要的因素——谈判的“实质结构”。这指的是那些不在谈判桌上直接讨论，却深刻影响谈判进程的框架性条件。我们将重点分析其中一个核心要素：**谈判场地的选择**。

## 什么是谈判的实质结构？

上一节我们介绍了谈判的基本概念，本节中我们来看看影响谈判的“框”。谈判的实质结构，就像是一个无形的框架，将谈判双方框在其中。这个框架本身与谈判桌上的具体议题无关，但它处理得好坏，却直接决定了谈判进展是顺利还是坎坷。

这个框架主要包括四个要素，而我们将首先深入探讨第一个：**谈判场地**。

## 场地选择：大有玄机的第一步

选择在哪里谈判，是实质结构的首要问题。这不仅仅是选一个地方坐下那么简单，它背后隐藏着丰富的信息和策略。

### 在己方场地谈判

在自己的“主场”谈判，优势显而易见。以下是主要好处：

*   **请示上级方便**：当需要内部协商或决策时，可以迅速联系到己方高层。
*   **搜集情报方便**：可以更容易地安排人员观察对方，或获取辅助信息。
*   **“演双簧”方便**：可以安排同事扮演不同角色（如“红脸”和“白脸”），以施加压力或推动进程。

公式可以概括为：**主场优势 = 控制力 + 信息便利性**

### 在对方场地谈判

到对方的“客场”谈判，看似被动，但也并非全无好处。有时，主动上门能发现意想不到的信息。以下是可能的好处：

*   **观察运营状况**：可以亲眼看到对方的工作状态、工厂管理、库存情况等。例如，一家生物科技公司在向美国客户展示时，因工厂内产品堆放杂乱、布满灰尘，导致客户对其“高科技”形象产生怀疑，最终影响了交易。
*   **听到“真心话”**：在非正式场合或与对方基层员工交流时，可能获得关键信息。例如，一位销售经理在拜访客户时，先从车间技师那里得知，对方已将自家机器的数据录入核心检测系统。这让他意识到客户更换供应商的成本极高，从而在后续与采购的谈判中占据了心理优势。

代码逻辑可以表示为：
```python
if 谈判场地 == “对方场地”:
    机会 = 观察对方真实运营状况 + 获取非正式情报
else:
    机会 = 控制谈判环境与流程
```

### 在第三方场地谈判

除了公司和工厂，谈判也可能发生在更休闲的场所，但这需要更高的技巧和对对方文化的理解。

**1. 球场（如高尔夫球场）**
在球场上谈判，需要特别注意对方对胜负的在意程度。如果对方非常看重比赛结果，而你大比分获胜，可能会将球场上的“旧仇”带到谈判桌上，增加难度。例如，一家台湾电脑公司代表在韩国与三星谈判期间打高尔夫惨败，感觉“生意做得好，球也打那么好”，无形中增加了心理压力。因此，许多商务人士会专门练习高尔夫、网球等运动，以适应这种社交型谈判场合。

**2. 餐厅或娱乐场所（如卡拉OK）**
这里存在巨大的文化差异，需格外留意：
*   **美国人**：可能秉持“出了会议室就不谈公事”的原则，在饭桌上更倾向于社交。
*   **德国人**：可能认为“不谈公事何必一起吃饭”，饭局依然是工作的延伸。
*   **日本人**：在正式的谈判桌上可能有所保留，反而在卡拉OK等放松的场合，才会吐露真言。他们有一句俗语：“话在肚里，歌在喉里”，意思是很多真实想法会在唱歌时流露出来。

**关键要点**：在任何非正式场合，对方的所有言行都像海绵吸水一样值得仔细吸收。许多谈判的关键信息，恰恰不在正式的会议桌上，而在这些“桌下”的互动中。

## 核心心法：区分“他”与“他的公司”

在思考谈判策略时，一个常见的误区是将谈判对手与其背后的公司完全等同。实际上，**谈判者个人的利益与公司的整体利益可能存在差异**。

例如，一家LED公司的项目经理（PM）面对客户压价。公司老板可能认为“这单利润太低，不如放弃，去开发新客户”。但项目经理的想法却是：“老板可以放弃，我不能放弃。这个项目是我的职责，如果丢了，我的职位和项目都可能不保。”

**公式**：**谈判者立场 = 公司利益 + 个人职业利益**

这种个人层面的顾虑和动机，很少会在正式谈判中明说，却可能在吃饭、唱歌、闲聊“生涯规划”时不经意流露。捕捉到这些信息，就能更精准地把握对方的真实底线和压力点。

## 谈判无所不在：故事的力量

谈判的艺术不仅在于言语交锋，还在于氛围营造和情感触动。一个精心安排的环境或一个意味深长的故事，可能比数据更有说服力。

例如，中国扬州的一家公司与德国客户谈判陷入僵局。中方安排游览千年古刹大明寺，并在寺内一面墙前，讲述了一个关于“知恩图报”的中国历史故事。讲完后，中方代表自然地补充了一句：“你看，我们中国人是懂得回报别人善意的民族。”德国客户听后颇为动容，第二天回到谈判桌，便做出了让步。

这个案例说明，谈判的“战场”可以延伸到任何能传递信息、建立连接的地方。

## 总结

本节课我们一起学习了谈判中“实质结构”的核心要素之一——场地选择。我们了解到：
1.  **主场谈判**利于掌控和信息获取。
2.  **客场谈判**提供了观察和获取非正式情报的独特机会。
3.  **第三方场地**（如球场、餐厅）需要考量文化差异和个人性格，是获取深层信息的绝佳场合，但需谨慎处理。
4.  谈判中必须区分**谈判者个人**与其所代表的**公司**，两者的利益并非总是一致。
5.  谈判**无所不在**，一个恰当的环境、一个巧妙的故事，都可能成为打破僵局的关键。

记住，优秀的谈判者不仅关注桌上谈什么，更懂得如何设计和利用整个谈判的“框”，从而引导局势向有利于自己的方向发展。
