# 012.【完结】育儿愈己的心理课：第16集 缺乏自信的常见表现 👶🧠

在本节课中，我们将要学习如何识别孩子缺乏自信心的常见外在表现。通过了解这些行为特征，父母可以更好地评估孩子的心理状态，并为后续的引导和支持打下基础。

一个自信的孩子会拥有强烈的存在感。他们认为自己的存在无需向任何人证明，内心具备强大的自我。然而，这种内在感受通常难以直接表达。我们总是需要通过观察外在的行为表现，来区分自信心充足与自信心缺乏的孩子。一方面，这可以为孩子做一个自我测评；另一方面，看清问题本身也能帮助我们更有效地解决问题。

## 缺乏自信的七种常见表现

首先，让我们来看看缺乏自信心会有什么样的常见表现。以下是总结出的七点，大家可以进行自我对照。

**1. 避免尝试新事物**

我们会看到，有些缺乏自信的孩子对于一道新菜或一个新玩具都不太愿意接受。这类孩子面对新事物时，更多地采用一种“省事”的眼光。他们担心事情会伤害到自己，因此选择回避。其行为背后的逻辑可以概括为：**“新事物 = 潜在风险 → 选择回避”**。

**2. 容易感到不被喜欢**

自信本身是一种“被爱感受”的集合。不自信的孩子就像他们的“爱的油罐”漏了，总是感觉别人不大喜欢自己、对自己不够好。他们对人际关系的敏感度会相对更高。

**3. 习惯将过错归咎于他人**

这类孩子无论做了什么，都很少肯定自己做得好的地方。对于做错的事情，他们则一律推卸责任，认为“这件事最好不要和我有关”。这正展现了他们缺乏自信、不愿面对自身不足的一面。

**4. 表现出冷漠或不在乎**

这听起来像一些高傲的“小公主”或“小王子”。然而，他们的冷漠和不在乎通常是伪装出来的，并非真实感受。可能因为他们内心自卑或不自信，所以故意表现出这种姿态，让人不易接近。

**5. 无法忍受正常水平的失败**

这类孩子在遇到挫折时，情绪会非常激动。他们认为“如果这件事是我的错，那我这个人就被否定了，我就被大家讨厌了”。因此，他们对待失败时情绪卷入很深，难以从中走出来。

**6. 容易贬低自己的天赋和能力**

这一点通常与孩子的养育者有关。孩子对自己的最初评价往往来源于他人。如果养育者经常给予负面评价，孩子就容易内化这些观点，进而贬低自己。其形成过程类似于：**外部负面评价 → 内化为自我认知 → 表现出自我贬低**。

**7. 容易受到别人的影响**

这一点与第六点息息相关。在他们内心，自我非常弱小，因此别人的意见就显得格外强大。他们不相信自己的想法，更愿意相信别人说的话。他们会从众多人的口中去拼凑“我是什么样”和“世界是什么样”，最终可能整合出一个分裂的、矛盾的自我认知，自信心水平也因此较低。

## 不自信孩子的内心世界与行为模式

如果父母发现孩子在以上七点中，有一半以上表现都比较明显，那么可以认为孩子的自信心相对缺乏。

总体来看，自信心低的孩子似乎被一种悲观主义情绪所支配。对他们而言，挑战成为焦虑和烦恼的主要来源。无论是新事物、新环境，还是略微复杂的人际关系，都会成为重大的挑战。

如果能设身处地从不自信孩子的角度去看，你会发现他们行为背后有一些“潜台词”，例如：
*   **“我不好。”**
*   **“我真笨。”**
*   **“为什么总没有人在意我？”**
*   **“其实我根本就做不到。”**

这种认为自己很糟糕的孩子，通常也很难找到解决问题的好办法。一旦他们陷入自我批判，认为自己差劲、做不好，就会变得更加消极、退缩和萎靡不振。这也解释了为何当新挑战来临时，他们的第一反应总是“我不行”。因此，他们不敢尝试新事物，不敢随意进入新环境，也不敢随意结交新朋友。一旦遇到挫败，他们很容易放弃，并急切等待他人帮助。

## 父母常见的无效鼓励

看到自信心较差的孩子，父母通常会感到焦虑和着急。然而，如果父母只会说“你怎么不去尝试一下呀”、“你试一试，你可以的，加油，妈妈相信你”这类话，其实是很无力的。对于孩子来说，如果他们内心的声音没有被真正听到和理解，他们也就不知道该如何相信父母所说的话是真的。
