# 075.透视挣钱花钱背后的心理：第02讲丁力-金钱心理课-第2：你是金钱的奴隶，还是金钱的主人

在本节课中，我们将要学习金钱背后的心理学，探讨我们与金钱的关系是“主人”还是“奴隶”，并分析这种关系如何影响我们的财富状态。

上一节我们介绍了金钱心理动力的重要性，本节中我们来看看限制性认知是如何形成的。

我们的金钱观本质被赋予了过多负面思想和感受。这些感受和想法其实都源于我们的限制性认知。限制性认知建立在个人的成长经历中。

例如，一个两岁的孩子已经懂得节约。当妈妈洗手开着水时，孩子会提醒关水以节约。这种观念来自奶奶的教导，奶奶提醒他要节约用水，打肥皂时要关水，以免浪费资源。带孩子购物时，当询问他是否想要某物，孩子会说“我可以不要”。这句话出自两岁孩子之口令人吃惊。

从心理学角度看，这可能令人担忧，孩子似乎在压抑自己的欲望。这个部分会使得他对金钱本身的价值感很低。他认为**金钱高于他，资源高于他，一切高于他**。于是他的不配得感会特别强，觉得自己不配拥有更好的东西，不配拥有自己想要的东西。他因此压抑自己的需求，将其变为满足他人、获取好评的因素。

可能有些教育工作者认为孩子懂得节俭和懂事是好事，不会过分膨胀欲望。我们不应该让人膨胀欲望。所以，钱和欲望直接相关。

但事实上，金钱和欲望一样，它是中性的。就像我们常举的例子：一把刀的价值取决于谁使用它。如果刀在医生手中，它可以救死扶伤，帮助别人。如果刀在劫匪手中，它可能伤人，伤害别人也伤害自己。金钱也是如此。

如果我们将金钱视为一种能量，那么就像我常说的“信”一样。每个人都应有权利和资源去享受，也应有资格去享受。但这个权利和资格在很多人那里需要进行比较：我具备什么才能拥有？我不具备时就不配拥有。这种对钱的“不配感”很多人都有。

说到两岁孩子的这种心理，有多少父母都是这样教育孩子的。我们必须将这部分放到现实条件中来看。

举一个简单例子：奶奶那一辈生活确实很匮乏。一家子很多小孩，一份食物要分成好几份，资源很匮乏，所以必须节约。在那个年代，这毋庸置疑是优秀的品质。

但放到今天这个年代，家里只有一个孩子。爸爸妈妈挣来的钱，很大一部分是为了帮助孩子成长和发展。如果用几十年前奶奶的思想来理解现在的生活，你会发现生活和现实都已改变。

你需要在这个世界里享受更多创造乐趣。而创造的前提是丰盛。当你很匮乏时，你无法考虑丰盛这个话题。就像马斯洛的五大需求，本能是生存。当处于生存阶段时，你无法讨论创造。

就像很多人说：我没有时间考虑赚钱，我觉得每天能把工作做好、挣点工资就不错了。他不会去想任何投资，也不会想任何自我提升。他觉得：我还花钱去上你这个金钱课？我觉得我现在一顿饭都吃不饱，还能花钱做这个？

在他看来，花钱去学习如何变得有钱，这本身就很奇怪。他会觉得：我本来就没有钱，你还让我花钱再学习赚钱，这个投入肯定不值。这其实就是一种弱者心态。

而且，当他知道了原来自己可以选择时，他也不会选择。他会认为：那个第二期课不是一千多吗？一千多块这么贵的课程才讲七天，还是线上课程，这个课程真的太贵了。于是他不会想去接触。信息来到你的世界，你看到了选择，但你会拒绝。这就是典型的**穷人思维**。

所有人面对这个世界都是公平的，只不过是每个人对于这件事的决定、选择如何使用金钱大不相同。很多人虽然现在很穷，但没关系，他拥有富人心态。

于是他就会想：我知道学习这个课程可以得到更多启发，能知道怎么拓展意识、改变思维，从而挣更多的钱。于是他就做一个决定：**我提升自我，通过学习改变现状**。这个时候，你已经为你的命运开启了一扇新的大门。

说到能量，我们会发现穷人和富人除了钱带来的自信之外，大部分是一种能量状态。富人的能量很高，是因为他们觉得自己**配得、值得拥有**。穷人的能量很低，是因为他觉得自己是金钱的奴隶，金钱正在奴役他。他会想：你有什么资格拥有金钱？

很多人都是这样：哇，我就三千块钱，你让我花一半工资去上课？那我这一半工资没了，我怎么生活呀？他想到的是眼前，没有看到未来。

而富人的做法是投资。投资是什么？我把钱花出去以后，能得到更多钱。我现在花出一千块投资在自己身上，未来我可能赚到十万、二十万甚至更多的钱。这就是主人和奴隶的区别。

当你觉得是金钱奴隶时，你永远在算计钱够不够用。金钱高于你，于是你会把很多时间浪费在挤公交、花大量时间去换取金钱。你的时间就本末倒置了：省这点钱，但是浪费了很多时间或体力。这些原本可以积攒下来挣更多的钱，但是不行，就是舍不得花。

因为在当下，金钱的本质是你的主人。当它是你主人时，它能带来的就是你永远被它奴役。你永远在为钱打工，永远觉得自己没有钱，也不可能去想怎么在有限时间里赚更多钱。

但是富人就会想：挣钱这件事，就是一个钱生钱的过程。每当我花出去一分钱时，它就能给我带来更多的钱。投资也好，提升自我也好，全都在做这件事。

可是当用穷人思维跟他这样讲时，他没有体验。他会觉得：那是因为你有钱呀，你要是没钱的时候，你敢投资自己呀？真的不好意思，人家没钱的时候真敢，真是这样子的。所以很多人是花未来的钱，但有些人是连花现在的钱都不踏实。

关于这种花钱不踏实，或者给自己买了很贵的东西后产生的懊恼、纠结、难过心态，我们后面可以慢慢分析。

刚才讲到了关于金钱给自己带来的能量，以及我们对待金钱时的一些错误看法。其实我们在一开始做的练习特别有意义。

你今天刚才在纸上写下了一个“钱”字。把这张纸放好，等到所有课程听完之后，你再拿出一张纸写一个“钱”字。两个对比一下，你会发现你的变化。而且会完全不一样，你瞬间就会发现：哇，金钱变可爱了。金钱已经不再是伤害我的了，突然发现金钱可以成为我的爱人了。

我们在后面还会给大家布置一些作业。你们在做作业的过程中就能了解到你的进步是怎么发生的，以及你怎么从一个“穷人”变成“有钱人”的路径，这会非常清晰。

本节课中我们一起学习了限制性金钱观的来源，分析了“金钱奴隶”与“金钱主人”两种心态的根本区别。我们明白了，将金钱视为一种可投资、可增长的能量，并相信自己“配得”，是转变财富状态的关键心理基础。
