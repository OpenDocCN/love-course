# 婚姻情感&家庭咨询技能大全（完结）：第1章：婚姻情感咨询概述 - 02 婚姻情感咨询师的市场定位（1）👩⚖️

在本节课中，我们将要学习婚姻家庭咨询行业的发展历程，并重点探讨婚姻情感咨询师在不同社会背景下的市场定位与从业方向。我们将对比中西方行业发展模式，并分析当前国内咨询师的主要工作领域及其特点。

## 行业发展历程对比

上一节我们介绍了婚姻情感咨询的概况，本节中我们来看看其专业领域在东西方不同的发展路径。

婚姻家庭治疗在东西方的发展过程存在较大差异。大约在上世纪二三十年代，婚姻家庭治疗开始在西方出现。

四五十年代是其发展的黄金时期。以美国为例，其运作模式呈现**协会化、专业化**的状态。美国设有婚姻家庭治疗协会。

该协会成立于上世纪四十年代左右，在五十多个州设有分会。若想成为一名婚姻家庭治疗师，必须在协会注册。各州的注册要求不同，若想在多个州从业，就必须考取多个州的从业资格并完成注册。这种协会化管理模式提供了较好的保障性和凝聚性，也利于监管。

注册后并非一劳永逸，还需完成继续教育。咨询师每年必须在协会的注册系统中，完成规定时数的督导和个人成长。这些记录会被详细备案。

协会通过一套标准学分系统进行管理。咨询师积累足够学分后，下一年方可继续执业。这种机制能推动咨询师在从业赚钱的同时，持续进行专业化与自我成长。协会在此过程中起到了监督和管理的作用。

同时，该体系依托保险制度。在美国的发展过程中，国家注入了大量资金，并使治疗与保险挂钩。大部分心理治疗费用由保险支付，除非是一些高端治疗需自费。这解决了两个核心问题：一是来访者的费用担忧；二是咨询师的客源问题。保险公司通常会向“病人”提供一份本地区可选咨询师的名单。

医院也会进行转介。医院主要通过药物治疗，但家庭治疗和个体治疗仍需通过心理咨询完成，因此医院会成为咨询师的客源之一。从美国的发展状态来看，其模式比较成熟，是协会化、专业化运营的状态。这也是我们发展的一个美好目标。

## 国内行业发展现状

了解了西方的成熟模式后，我们再来看看国内婚姻家庭咨询行业的发展现状。

我国是从2007年开始，由人力资源和社会保障部颁发**婚姻家庭咨询师**职业资格许可证。心理咨询师的出现时间或许更早或同期，印象并不深刻。

但除此之外，国内并没有硬性规定从业者必须完成多少小时的个人体验、督导或继续教育。目前这些全凭咨询师的自主学习状态。

前年，国家暂停了职业资格许可证的颁发渠道。目前有一些机构自主设置课程并颁发证书。

这种情况利弊共存。好处是大家获取证书和学习的渠道更多了。弊端在于，究竟哪家机构的证书更具含金量、更受社会认可，目前仍处于探索和检验的过程中。

这就是我们当前的发展状态。从2007年我们才开始有婚姻家庭治疗师的身份概念，至今不过十二年。我们已经发展出目前的状态，拥有了心理咨询师和婚姻家庭治疗师的庞大队伍。这是发展过程中的必然现象。

我们可能会羡慕西方国家发展得更好，但必须注意时间因素。我们要允许自己在成长中学习，在学习中成长。

## 当前主要从业方向

在了解了行业背景后，我们现在来看看，无论是心理咨询师还是婚姻家庭咨询师，当前主要的从业方向有哪些。

我本人从事心理咨询的临床工作。这里所说的“临床”是指咨询的现场接待工作，并非指医院，因为我拥有自己的咨询工作室。这是我的第一部分工作。

第二部分是媒体工作。主要是配合媒体录制情感类、婚姻家庭类节目，对当事人的心理和家庭状态进行干预。

第三部分工作是我运营的一个小型公益中心。该中心会承接一些国家采购项目，从事社会服务工作。

在我的这些工作中，我发现目前国内心理咨询师或婚姻家庭咨询师主要有以下几类就业方向。

### 1. 社工与调解员

以下是第一类从业方向：社工与调解员。我将两者分开，因为它们虽有相似性，也存在一定差异。

社工和调解员看起来都是社会的基层服务工作者。区别在于，社工有时需要直接推进一些政策性工作，而调解员更专注于情感、经济、家庭纠纷等问题的调解工作。

对于社区工作者或调解员岗位，人员需求缺口非常大。同时，对这些岗位上工作人员的内在要求——即对心理学、家庭婚姻咨询专业性的了解——也非常高。

以往的老一代社工和调解员主要凭借热情和生活经验工作。但随着时代发展，许多工作走向科学化、信息化和系统化管理。仅凭热情很难完成工作，因此对专业性的要求日益提升。

这些岗位的就业地点包括街道、政府职能部门、工会、法院或医院的调解室等。考取相关证书后，这些地方都是潜在的就业出口。

但在了解工作对象时，我们需要有一定的认识。对人而言最公平的是**时间**。每个人都会用生命时间去完成自认为最有价值的事，这种价值不等同于物质价值。

例如，青少年的价值在于迷茫中寻找自我；中年人的价值在于创造资源，建设家庭与自我。有个段子说，比尔·盖茨不会弯腰捡地上的100美元，因为他弯腰的时间能赚更多钱。但这只是段子，财富属于尊重财富的人。这个段子广为流传，背后蕴含的理念是：时间应分配给更重要的部分。

我强调这一点，是因为从我们工作对象（无论是社工还是调解员的服务对象）的时间分配上，能看出一些端倪。他们会用大量时间纠缠于家庭琐事、细小情感问题或邻里冲突，反复与工作人员纠缠。这说明他们的时间可能无法用于创造更多财富。说句大白话，可能意味着我们的工作对象物质条件相对较差，资源较为匮乏。

资源越匮乏的人，越承受不起损失，因此容易陷入计较的状态。这是社区工作者和调解员面临的工作难点之一。从时间分配上就能看出难度所在，根源在于其内在的匮乏。

同时，他们愿意花时间纠缠于某件事，说明这件事所附带的非物质价值（如尊重、爱、自信与自卑的关系等）在他们内心非常重要。因此，这类工作对象的另一个特点是自尊水平可能相对较低，表现得比较敏感。

对于社区工作者或调解员而言，如果考取了职业资格许可证，并愿意积累基础工作经验与技能，这其实是一个不错的选择。因为它能让你在人们防御最弱的状态下，看到最真实的生活。

这得益于我参与3000多场媒体婚姻家庭交流工作的经验。我发现那些案例特别真实，也特别残酷。常有人问我节目案例是否真实。至少在我目前仍频繁参与的央视、重庆卫视、江西卫视的节目中，多年来没有一个案例是虚假的。

如果说有“手脚”，那可能出乎意料：不是放大矛盾，而是弱化其中过于激烈、暴力、黑暗或残酷的部分。我们担心这些内容赤裸裸地呈现在媒体上，会让观众感到不寒而栗。因此，我们反而做了许多保护性的处理。生活永远比我们想象的更真实、更残酷。

如果在社区或调解一线工作，可能有人觉得“不高端”。我们想象中的咨询是：吹着空调，坐在45度角的椅子上，或使用催眠治疗椅，环境安静静谧。那才是咨询师应有的状态和工作环境。柴米油盐和琐碎的事情似乎“不高级”。

但我不喜欢用“高级”或“低级”这类带有价值判断的词。如果我们不去感受更多生活的真实，即便在高级的场所，也可能无法真正呈现工作的价值。只有深入生活，才能理解什么是人间、什么是心理、什么是痛苦。

在这样的工作中，你可以把自己放在积累经验、观察真实人间的位置上。去看看除了自己熟知的生活之外，还有哪些是想象不到的生活状态。这是一个非常好的途径，也是成就职业发展的很好渠道。

就在前两天，我刚结束一个社区工作者的团体工作坊（通过公益中心的政府采购项目）。我们发现，现在很多社工本身都持有心理咨询师或婚姻家庭治疗师许可证。虽然接触这些生活和工作给他们带来很大压力和困扰，但他们的感受层面也更深刻、更丰富。

例如，前一段时间在工作坊中，一位组员分享说，他发现自己不能单一、绝对地看待人。他的一位工作对象（被大家戏称为“西雅头”）是街道居民，特别蛮不讲理，总是找茬闹事，因为街道按规定拆除了他私自安装的地锁。

这位组员起初感到非常愤怒，觉得世上怎么会有这样的人。但过了一段时间，他在胡同里看到一个男人，老实巴交，见谁都点头微笑，和蔼地推着一个有残疾和智力问题的年轻人晒太阳。他询问后得知，那是“西雅头”的老伴和孩子。

他当时内心受到极大震动，突然对“西雅头”以往的蛮横、愤怒、委屈和斤斤计较有了深刻的理解。这份工作打破了他以往对于“人应该怎样”的许多内在设定和理解。

由此可见，这类工作虽然压力大、充满困惑和挑战，但其内在具有独特的魅力。

### 2. 传统心理咨询师

除了基层工作，另一部分持有证书的从业者会选择成为传统心理咨询师。

传统心理咨询师即在固定的咨询室内，按照专业设置，完成面对面、一对一或一对多的咨询。我本人一直从事这类工作，也会带领一些团体工作坊和个人体验。

但我们也不要把这种工作想得过于美好。我曾带领一个咨询师团体工作坊，其中一位咨询师分享说，他发现国内对心理咨询、心理治疗的认识仍有局限。尽管媒体宣传很多，大家会说“我心理不健康”、“我有压力”、“我焦虑”，但真要让他们为压力或焦虑花钱做咨询，除非出现了症状、婚姻巨变、家庭灾难性事件、孩子出问题、婚外情等紧急情况。

如果没到出现症状或重大事件的紧急程度，人们通常不愿意花钱咨询。这种意识仍在逐步建立中。因此，前来咨询的人往往对咨询抱有神奇和极高的期待。

根据我多年的工作经验，前来求助进行心理治疗的人，大部分在来咨询室之前，已经尝试了所有他们认为能帮助自己的方法，并且基本无效。这时，心理咨询师就成了他们最后的救命稻草。

试想，人们对最后一根救命稻草的期待有多高？他们希望咨询师真的能够拯救他们。因此，来访者对治疗师的期待和理想化程度会非常高。

我们都知道，我们只是心理咨询师或婚姻家庭咨询师，不是上帝，无法拯救一切；也不是神仙，没有魔法棒。如果你认为自己能做到那些，那可能需要更多的自我成长，那可能是自我理想化或自恋的部分。但我们确实会遇到这样的来访者，这就是传统心理治疗师所要面对的客户素描：他们对你有特别多的期待。

我记得在一次团体督导中，一位三十多岁、较有经验的女治疗师委屈地哭了。她分享说，刚从业不久时，接待了一位六十岁左右的女性来访者。这位女士的儿子有发育迟缓，但依然在母亲的主持下娶了媳妇。

这位母亲认为儿媳特别精明、心眼多。儿子婚后不再听她的话，转而听从媳妇的。她问治疗师：“你能不能帮我想个办法对付一下我这个精明、心眼多的儿媳妇？”

这位治疗师尝试了各种方法，也做了很多科普，例如解释“亲密的亲子关系力量不如亲密的两性关系”等。但老太太听不懂（她似乎是工人背景，学历不高），她说：“你说的这些我都懂，不用告诉我。你就告诉我该怎么办。”

治疗师被逼到角落，给出了一二三、四五六的建议。老太太却说：“你说这些我都试过了。我花钱找你，你说的是我知道的。你能不能说点我不知道的？”治疗师又说，儿子虽然发育迟缓，但也应有自己的生活，他与妻子感情好是合理的等等。

老太太急了，投诉要求退费。我们都知道，设置中没有“不满意就退费”的规定。被告知不能退费后，老太太说：“我看你是穷疯了，出来骗钱。这钱我今天就算白给你了，你留着去买骨灰盒吧。”说完摔门而去。这位组员在分享时哭了，她说不知道该怎么办，对方为何如此恶毒。

大家可能会遇到类似情况：不讲理的、将情绪宣泄给你的来访者。在系统的个人成长和技能培训课程中，我们会知道，这与来访者对咨询师的理想化，包括正向和负向移情有关。这些都可以在咨询过程中被咨询师用作干预的方法和手段。

但咨询师自身的内在力量、是否对自己过于理想化等，都需要通过个人体验、学习和成长来逐步练习和完成。我举这个例子的目的是：我们也不要认为在咨询室里的传统咨询师工作就特别轻松，仿佛自己是治病救人、拯救灵魂、为家庭带来福祉的。

如果你抱着“我要拯救他”的心态从事这个职业或扮演这个角色，那么你确实还需要进行大量的个人成长。因为你会发现，以那种状态工作，你必然会有些卷入，甚至会在某种程度上“使用”来访者。这当然不是我们今天探讨的主题，有兴趣以后可以在其他场合交流。

我想说的是，作为一名传统咨询师，我们面对的是急切想要立刻解决问题的来访者：“我想立刻让我孩子上学”、“我想立刻让我丈夫离开小三”、“我想立刻让我的抑郁家属好转”、“我想立刻让我的强迫症状消失”。他们总是抱有这种期待，因此也容易对咨询师产生不满，导致投诉、冲突或矛盾的出现。这也是传统咨询师需要面对的问题。
