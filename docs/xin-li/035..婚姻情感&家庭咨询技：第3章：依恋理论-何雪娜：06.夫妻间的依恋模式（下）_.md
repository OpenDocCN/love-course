# 035..婚姻情感&家庭咨询技：第3章：依恋理论-何雪娜：06.夫妻间的依恋模式（下）_

在本节课中，我们将深入探讨不同成人依恋类型在婚姻互动中的具体表现，并学习如何识别和修复因依恋需求未被满足而引发的冲突模式。我们将通过具体案例，理解争吵、恶语相向等行为背后的依恋诉求，并最终掌握在亲密关系中重建安全感的核心方法。

上一节我们介绍了依恋理论的基本概念和成人依恋类型。本节中，我们来看看这些类型如何具体影响夫妻间的日常互动。

## 不同依恋类型在婚姻中的表现

根据成人的依恋状态，他们在婚姻关系中会表现出不同的互动模式。

以下是不同依恋类型在婚姻中的典型表现：

*   **安全型成人**：这类人可以轻松地与爱人亲近，并惬意地依靠对方。他们既能从伴侣处获得安全感，也能为伴侣提供庇护。他们不担心被抛弃，因此不会持续处于紧张和焦虑的状态。他们也不会因他人过于亲近而感到不安。
*   **冷漠型成人（由回避型儿童发展而来）**：这类人不允许自己依赖伴侣，并否认依恋的价值。他们既不想过于亲近他人，也不希望他人过于亲近自己。当关系过于紧密时，他们会感到不安和危险。他们常表现出一种疏离的独立，其本质是“别靠近我”的自我保护，并经常向伴侣传达“没有你我也能过得很好”的信息。
*   **迷恋型成人（由矛盾型儿童发展而来）**：这类人通常比伴侣更迫切地想要接近对方。他们会反复询问“你去哪了”、“你爱不爱我”等问题，并常常抱怨伴侣没有达到自己的期待。他们内心充满焦虑，这种焦虑可能让伴侣感到不堪重负。
*   **混乱型/未解决型成人**：这类人由于早年的严重创伤，内在世界非常破碎，难以形成完整的自我和关系认知。他们几乎很难建立或维持稳定的亲密关系。在咨询中，他们首先需要通过与咨询师建立安全的关系来稳定内心世界。

由此可见，在婚姻模式中，不同依恋类型的人表现方式截然不同。安全型的人更容易建立和体验安全的关系。而不安全型的人由于缺乏健康关系的“模板”，常常在关系中不断受挫，感到痛苦。这正是我们需要学习依恋理论来理解婚姻的原因——从情感连接的层面，理解人们如何在关系中互动。

## 理解冲突背后的依恋信号

夫妻争吵是常见现象，但我们需要明白，许多冲突背后是对依恋的渴求。

争吵本身是正常的，但它常常是对依恋需求的表达、对分离的抗议，或是对内在依恋需求可能得不到满足的恐惧。在依恋关系中，**有回应总比没有回应好**，即使是糟糕的回应。最初的争吵可能是试图表达依恋诉求，但因无法清晰表达而演变成冲突。如果无数次尝试都失败，个人就可能陷入绝望，关系进入“僵局”。

以下是一个典型夫妻争吵场景的解析：

> 丈夫抱怨妻子总指责他工作晚、不顾家，而他工作是为了家庭。妻子则冷笑回应：“你心里真的有这个家吗？”并翻出旧账，如生孩子时丈夫没有陪伴。丈夫感到恼怒，认为妻子在翻旧账，于是形成争吵循环。

如果从依恋角度解读，双方的核心诉求其实是：**“我能依赖你吗？”、“对你而言我重要吗？”、“你需要我吗？”**。所有对话都围绕这些主题。在临床和生活中，意识到这一点是修复关系的第一步。

## 危险互动模式：恶语与指责

在亲密关系中，双方因深知彼此的弱点，恶毒的语言可能造成致命创伤。

这种攻击会让双方筋疲力尽，且伤害持久。例如，一位有童年寄养经历的丈夫，在婚姻中可能因回避情感而忽略妻子需求。如果妻子在争吵中用“有娘生没娘养”等话攻击他，会深深刺痛其旧伤。丈夫可能暴怒恐吓，这又触发了妻子对暴力的恐惧，形成恶性循环，彻底摧毁关系中的安全感。

当安全感丧失，双方容易陷入相互指责的模式。

以下是三种常见的负面互动模式，它们都与依恋损伤有关：

*   **指责模式**：双方将问题归咎于对方，例如因孩子学习成绩下降而相互指责。这种模式失去了共同思考的空间，忽略了夫妻关系本身可能才是问题的根源，并会影响孩子建立安全的依恋关系。
*   **抗议之舞模式**：一方不断索取、追问（如妻子唠叨抱怨），另一方不断退缩、沉默（如丈夫不予回应）。索取方实质是渴望情感回应和靠近，退缩方则因感到压力而回避。双方都无法解读对方行为背后的依恋信号，因此无法成为彼此的“安全港湾”。
*   **僵局模式**：这是最严重的状态，源于依恋诉求长期得不到回应，最终导致情感放弃。关系变得死气沉沉，毫无活力。这就像儿童早年发出无数次信号都得不到有效回应后的绝望状态。

需要明确的是，冲突本身并不可怕，**事后的修复至关重要**。只要能开始思考“对方需要什么？”、“如何让对方感到安全？”，行为和思维就有可能发生积极改变。

## 重建亲密关系中的安全感🔑

寻回安全的亲密关系，就如同重新建立安全的依恋。一段好的婚姻关系可以修复早年的依恋创伤，让人获得新生。

以下三点总结了如何在亲密关系中建立依恋安全感，对应情感需求的三个核心板块：

1.  **亲近：表达信任与敞开**
    亲近意味着 **“你会为我敞开胸怀吗？”** 。建立信任需要双方尝试敞开自己，即使心存疑虑。当你能向不评价、只倾听的伴侣倾诉时，这个过程本身就能帮你理清思路和情感。在足够安全时，你甚至可能分享童年经历，帮助对方更深地理解你。难点在于，分享脆弱面需要极大勇气，我们害怕被忽视、嘲笑或利用。因此，伴侣必须以接纳和共情来回应，而不是将其当作日后攻击的把柄。

2.  **回应：提供情感接纳与支持**
    回应的本质是 **“我能信赖你对我的情感”** 。这意味着首先要倾听，并接纳对方的情感流动。真正的共情始于心理上的倾听。回应既包括身体上的安慰（如拥抱），也包括情感上的抚慰和言语沟通。**在亲密关系中，情感需求应被置于首位**。例如，当妻子抱怨职场困境时，丈夫应先给予情感支持（“那一定让你很难受”），而非急于讲道理或指责。良好的回应能让双方的身心恢复平静。

3.  **承诺：体现情感投入与专一**
    承诺代表着 **一心一意和相互的情感吸引** 。它要求双方都投入情感，就像早期母婴依恋中，婴儿需要感受到母亲的专注和爱。这种 **“情感出席”** 是至亲关系的特点，意味着彼此持续、专注的情感回应。它是维系健康亲密关系的稳定剂。

通过以上讲解，我们应该了解到，寻求爱、关系中的安全感和信任基础，需要从**亲近、回应、承诺**这三个维度入手。这些都是维系良好婚姻或亲密关系的关键要素。

本节课中，我们一起学习了不同依恋类型在婚姻中的具体表现，解析了夫妻冲突背后隐藏的依恋诉求，并探讨了指责、抗议、僵局等危险互动模式。最后，我们掌握了通过**表达亲近、提供情感回应、履行情感承诺**来重建亲密关系安全感的核心路径。理解这些模式，有助于我们打破恶性循环，将挑战转化为深化情感连接的契机。
