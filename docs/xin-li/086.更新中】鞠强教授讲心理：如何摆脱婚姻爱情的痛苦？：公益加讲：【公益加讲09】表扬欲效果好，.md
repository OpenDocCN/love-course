# 心理学课程086：如何让表扬更有效？🎯

在本节课中，我们将要学习领导心理学中的一个关键技术：如何通过具体化的表扬来提升沟通效果和激励作用。我们将探讨为什么具体的表扬比笼统的表扬更有效，并通过实例来理解这一原则。

## 核心原则：表扬必须具体化

上一节我们介绍了表扬在领导技术中的重要性，本节中我们来看看提升表扬效果的一个核心原则。

大量的心理学研究表明，说话越是具体化，表扬的效果就越好；越是笼统，效果就越差。造成领导表扬效果不佳的原因有很多，而“说话不够具体化”是排在前三位的常见原因之一。

## 笼统表扬与具体表扬的对比

以下是两种表扬方式的对比，我们可以清晰地看到具体化带来的差异。

*   **笼统的表扬示例**：
    “小唐工作认真负责，勤恳敬业，能力很强，是一个非常不错的苗子。”

*   **具体化的表扬示例**：
    “小唐，我对你这一段时间的观察，发现你有五个方面非常出色：
    1.  **创新精神强**：上次你建议去公安局要一些通缉令贴在附近，果然吸引了人流，提升了店铺生意。
    2.  **工作主动性高**：你能根据组织目标和原则主动想办法弥补漏洞，这是未来领导的潜质。
    3.  **做事细致**：你的工作漏洞少，比同龄人更细致，从你整洁的桌面就能看出来。
    4.  **心胸宽广**：在争论问题时，你对不同意见情绪淡定，很难得。
    5.  **热爱学习**：你持续学习的劲头很足。
    你来到我们部门，让我觉得部门未来更有希望了。”

通过比较可以发现，越是具体的表扬，越能让被表扬者产生强烈的激励感。**具体化的沟通，其说服力和感染力远大于笼统的语言**。

## 具体化原则在日常沟通中的应用

这个原则不仅适用于工作表扬，也适用于日常生活沟通。以下是另一个生活化的例子，展示了具体化如何增强说服力。

假设你需要说服不想逛商店的伴侣，不同的说法效果不同：

*   **笼统说法**：“老婆，我肚子疼，不想逛商店。”
*   **具体说法1**：“老婆，我肚子右边疼，所以不想逛商店。”
*   **具体说法2**：“老婆，我肚子右边下部分疼，所以不能陪你去。”
*   **具体说法3**：“老婆，我肚子右边下部分疼，每隔几分钟就疼一次，所以不能陪你去。”

显然，最后一种最具体的说法，其感染力和说服力远远超过简单的“肚子疼”。大家可以体会其中的差别。

## 关键总结与实践建议

本节课中我们一起学习了表扬具体化的核心原则。记住，**具体化的表扬 = 更强的激励效果**。
