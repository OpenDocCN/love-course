## 降低任务难度

以下是两种降低任务难度、提升完成可能性的核心策略。

### 1. 化整为零（Baby Steps）

将宏大目标分解为当下即可执行的一小步。这就像面对一个巨型数列 `1+3+5+7+...`，你无需立刻算出总和，而应专注于完成眼前的第一步计算 `1+3`。

**例如**，你的目标是半年后跑完全程马拉松。正确做法不是今天就去挑战全马，而是将其分解：
```
最终目标：完成全程马拉松
↓
中期目标：完成半程马拉松
↓
近期目标：能连续跑5公里
↓
今日目标：今晚在操场慢跑1小时
```
这种“婴儿步”策略能有效降低起步的心理压力，让行动变得可触及。在谈判中，让对方支付定金就是一种“婴儿步”，它推动了整个交易进程向前迈出一小步。

### 2. 改变环境

通过调整物理环境来降低执行难度或增加坏习惯的执行成本。

*   **促进好习惯**：如果你想培养阅读习惯，可以把书放在厕所或床头等触手可及的地方，让阅读行为变得便捷、容易开始。
*   **戒除坏习惯**：如果你想减少看电视的时间，可以增加执行难度，例如每次看完都把遥控器藏到不同地方，或者像熊浩老师一样，直接选择不购买电视机。

## 设置行为契机（锚点）

行为变革的最后一个要素是“契机”，即找到一个可靠的触发点来启动新习惯。

B.J. Fogg的行为模型建议，将一个你想建立的新习惯，锚定在一个已有的日常习惯之后。公式可以表示为：
```
[已有习惯] → [新行为] → [即时奖励]
```

**例如**：
*   **锚点**：每天早晨刷牙（已有习惯）。
*   **新行为**：刷牙后立即做5个俯卧撑（想培养的习惯）。
*   **即时奖励**：完成后对自己说声“加油！”。

通过这种方式，旧习惯成为了启动新行为的可靠“扳机”。跟随本课程学习，本身就是锻炼你谈判思维、践行承诺的一个持续“契机”。

## 总结

本节课我们一起学习了履行承诺的另外两个关键杠杆：**降低难度**与**设置契机**。

*   **降低难度**可通过“化整为零”和“改变环境”实现，让目标变得具体可操作，让行动路径变得顺畅。
*   **设置契机**则是利用“已有习惯”作为锚点，来可靠地触发新的目标行为。

结合上节课的**内生动机**，**动机、难度、契机**这三者构成了行为变革的核心模型。掌握这个模型，不仅能帮助你更好地践行对自己的承诺，也能让你在谈判中更有效地引导对方做出并履行承诺。
