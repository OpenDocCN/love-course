# 011.从家长管：05安全规矩：远离安全隐患，学：25生活安全（上：用火、用气、用电
## 05安全规矩：远离安全隐患，学会自我保护
### 25 生活安全（上）：用火、用气、用电的安全守则

在本节课中，我们将要学习如何为孩子建立关于用火、用气、用电的安全守则。我们将探讨如何通过正向引导和环境防护，帮助孩子理解危险、设定界线，并最终学会自我保护。

---

有一种观点认为，不想让孩子接触的物品，只需藏起来即可，制定规则是无效的。这个观点只说对了一半。确实，将危险物品（如燃气阀门）置于孩子无法触及的高处，是一种有效的预防措施。

然而，这个观点错误地认为“制定规则没用”。我们需要辩证地看待：执行规则不一定百分百有效，但不制定规则则一定会出问题。

上一节我们提到了对“藏起来”策略的辩证看法，本节中我们来看看为何有些危险无法隐藏。例如，厨房生火做饭的过程，孩子必然会看见。许多家长因此禁止孩子进入厨房。

但人的本性是，越禁止，越好奇。关门只会激发孩子巨大的探索欲，想知道门后究竟有什么。当他第一次闯入时，由于不了解何为安全何为危险，反而更容易乱来、发生意外。

因此，我们的第一个关键建议是：**让孩子看见他应该能看见的东西**。

我们可以通过安装半透明门帘、玻璃门或保持门半开等方式，让孩子能观察到厨房内的活动。这不仅满足了孩子的好奇心，也避免了因大人“偷偷摸摸”而损害孩子的安全感。

在满足了孩子的好奇心之后，下一个关键点是帮助孩子**理解**厨房环境。

我们可以借助绘本或书籍，向孩子说明厨房里各个区域的功能、哪些地方危险、哪些地方安全。通过图文并茂的方式，让孩子建立起初步的认知。

理解了环境，就需要明确行为的边界。第三个关键点是**给孩子划定清晰的界线**，明确告知哪些物品绝对不能触碰。

为了巩固孩子的理解，我们可以通过游戏的方式进行教学。以下是两种可行的游戏方法：

*   **情景扮演法**：如果家中有厨房玩具，可以通过“过家家”游戏，让孩子扮演“考官”，指出家长在模拟厨房操作中哪些行为（触碰某些物品）是不被允许的。
*   **图画标识法**：如果家中没有玩具，可以在纸上画出厨房的简图，让孩子指认并说出哪些东西可以动，哪些绝对不能动。

通过以上“观察-理解-划界-游戏”的步骤，我们能够正向地引导孩子认识危险。

当然，仅有教育不够，我们还需从物理环境上做好防护。以下是一些具体的防护措施：

*   **使用插座保护盖**：家中位置较低、儿童易触及的电源插座，应安装保护盖，防止孩子将手指或异物插入。公式表示为：**安全隐患（低处插座） + 防护措施（保护盖） = 风险降低**。
*   **利用媒介进行解释**：对于年龄稍大的孩子，可以利用绘本、电视节目等，详细解释为什么某些东西危险、不能碰。
*   **约定安全活动范围**：与孩子共同约定家中哪些区域是安全活动区，哪些需要家长陪同才能进入。
*   **任命“家庭安全员”**：一个特别有效的小诀窍是，让幼儿园阶段以上的孩子担任“家庭安全员”，负责检查家中的安全隐患。赋予角色感能极大地激发孩子的责任心和执行意愿。

本节课中，我们一起学习了如何通过开放观察、知识讲解、设定界线和游戏互动的方式，正向引导孩子建立用火、用气、用电的安全意识，同时强调了物理防护（如使用插座保护盖）的重要性。
