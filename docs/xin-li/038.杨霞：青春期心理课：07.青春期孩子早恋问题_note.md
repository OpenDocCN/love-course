# 038.杨霞：青春期心理课：07.青春期孩子早恋问题_note

在本节课中，我们将探讨青春期孩子“早恋”现象背后的心理原因，分析其对专注力及人际关系可能产生的影响，并为家长提供理解与引导孩子的思路与方法。

## 理解“早恋”的成因

上一节我们介绍了青春期孩子的心理特点，本节中我们来看看被称为“早恋”的行为。通常，我们将青春期阶段、未满18岁的孩子建立的恋爱关系定义为“早恋”。

首先需要了解“早恋”的原因。这种行为往往与孩子的情感缺失有关。

以下是几种常见的情感缺失情况：
*   如果男孩从小缺乏母爱，或女孩从小缺乏父爱，他/她可能会寻找一种补偿，从而容易陷入“早恋”。
*   孩子内心缺乏安全感和认同感，需要一个观众、支持者、同伴或感情寄托对象。

因此，我们首先要学会理解。这好比孩子饿了会想吃饭一样，是一种天然的需要。我们不应埋怨孩子“饿”，但需要注意，青春期孩子本身情绪就不稳定。

## “早恋”对专注力的影响

孩子的自控能力在青春期同样不稳定。例如，事情顺利时他可能很高兴，不顺利时情绪就会产生波动。

“早恋”很可能会加剧这种情绪波动，从而影响其专注力。他原本应将所有心思用于青春期的成长，尤其是繁重的学习任务上。但如果需要分出一部分精力去处理与另一个人的关系，这当然会影响他的专注力。

虽然存在一种观点，认为“早恋”可能激励双方共同进步（例如“为了和他/她上同一所大学而努力学习”），但我们必须认识到，大部分“早恋”都会导致孩子情绪波动。我们在之前的课程讲过，情绪波动会影响孩子的专注力。而“早恋”正是影响孩子情绪波动的一个重要因素。

所以，“早恋”确实可能对孩子的专注力产生负面影响。

## “早恋”引发的人际关系失衡

此外，“早恋”还可能引起人际关系的失衡。青春期孩子原本与同学的关系是平均的，他可能有几个铁哥们或好姐妹。

但开始“早恋”后，他难免会更多地与那一个人相处，例如一起放学、吃饭、上自习。这可能导致其他同学（如暗恋者、原先一起打球或结伴上下学的朋友）产生嫉妒心理，使人际关系失衡。

原先的好友可能因失去与你相处的机会，而在背后议论、指责甚至说闲话。这些情况必然会导致你在情绪上产生波动。

我曾接触过一个案例：初三学生阿丽和一位男同学关系很好，确立了男女朋友关系。在其他同学的起哄和激将下，两人约定第二天在放学时的走廊上当众接吻。此事迅速被老师知晓，并上报校长。随后，双方家长被谈话，学校也施加了压力。

结局是，那位男生因承受来自多方的压力（反复谈话、家长压力、同学的事后嘲笑），学习成绩下降，最终不得不转学。他后来感慨道：“我理解了什么叫‘秀恩爱’的代价。我失去了那么多好同学，成绩也下滑了，现在不得不转到一个我不喜欢的学校。”

“早恋”行为本身，可能是孩子情窦初开阶段的一种正常情感需要。但它可能带来的后果，孩子往往难以预估和承担。他可能最初只设想“一起上下学、讨论题目”，却没想到事情会发展到难以收拾，甚至影响学业。

情感有时如同洪水，来时可能是涓涓细流，也可能势如猛兽，将人冲得措手不及。当一个人过度受他人或情感左右时，其专注力必然会受到影响。

## 给家长的引导建议

如果我们发现孩子有“早恋”迹象，首先不要“如临大敌”。我曾在中学生座谈会上提问：“你们认为人际关系重要，还是学习成绩重要？”孩子们大多回答人际关系更重要。

我接着问：“如果有一位同学生病需要巨额治疗费，你愿意倾家荡产帮助他吗？”同学们沉默了。我又问：“如果你和女朋友在铁道边散步，她的脚踝卡在铁轨里，而火车即将驶来，你会自己跑掉，还是抱着她一起面对？”同学们再次沉默，难以抉择。

我最后说：“如果你们认为人际关系最重要，那么逻辑上就应该选择倾家荡产地帮助对方，或宁死也要在一起。但事实上，很少有人能毫不犹豫地做出这些选择。”

这说明了专注力应有先后顺序，不应完全被情绪和情感支配。**首要任务是完成自我成长与自立，之后才有能力去帮助甚至爱别人**。一个尚未成年、缺乏自立能力的孩子，用什么来承担“爱”所带来的责任呢？

这不是空洞的说教，而是关于责任的问题。**爱意味着责任**。

因此，家长和孩子在面对“早恋”问题时，可以尊重情感的自然发生。但必须牢记，此阶段专注力的核心仍应放在学习上。如果“早恋”能促进学习，那固然最好。但始终不应忘记，青春期的主要任务是完成学业。唯有先完成这项任务，未来才有可能收获更成熟、更美好的爱情。

**这才是负责任的行为。**
