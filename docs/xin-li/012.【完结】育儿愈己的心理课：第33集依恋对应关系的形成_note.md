# 012.【完结】育儿愈己的心理课：第33集 👶 依恋对应关系的形成

在本节课中，我们将要学习依恋关系的核心概念，并系统地理解家长的抚养方式如何影响孩子的依恋模式，以及这种模式又如何延续到孩子未来的亲密关系中。

上一节我们介绍了依恋关系的重要性，本节中我们来看看其形成背后的具体对应关系。根据依恋理论，婴儿是从跟妈妈或其他主要养育者的互动中，获得安全型依恋关系的。因此，给孩子爱，并不一定需要亲亲抱抱或说“我爱你”。依恋关系的获得，关键在于父母**怎么做**，而不是父母**怎么想**。

父母的想法和动机是一回事，做出来的事情往往是另一回事。关键在于父母与孩子的**互动关系**。例如，孩子小时候，父母是“按需喂养”还是“按时喂养”？“按需喂养”的着眼点在孩子身上，这体现了爱。“按时喂养”可能更多是为了父母自己的便利，这需要打一个问号。所以说，不看你怎么想，主要看你怎么做。

如果孩子在0到1岁有很多心理需求，父母能满足，就是好父母。如果满足不了，即使愿望再强、再爱孩子，也没有用。这好比伴侣天天说爱你却总出差，你感受到的未必是爱。爱需要花时间和精力去陪伴，关键在于其中的互动关系。

理解了互动是关键之后，我们来辨别家长的抚养方式。以下是评估抚养方式的两个核心指标：

第一个指标是**情感性**，指父母情感投入的高低。如果父母自身“爱的油罐”充足，情感迸发通常比较热烈。如果自身“爱的油罐”不充足，没有体验过无微不至的关爱，那么想给予别人关爱也很难。因此，不要一味责怪伴侣，例如丈夫可能因其父亲当年未能扮演好角色，而不知道如何当好父亲。女人当好妈妈似乎是天生的，而男人当爸爸则需要学习。

第二个指标是**控制性**，指父母对孩子行为的控制程度。有的人爱孩子，有的人则以控制的方式爱孩子。人总希望抓住一些东西以获得掌控感，讨厌失控的感觉。因此，控制也分高和低。

将“情感性”和“控制性”两个维度相结合，可以形成四种主要的养育方式：

*   **民主型**：情感高，控制高。
*   **溺爱型**：情感高，控制低。
*   **专制型**：情感低，控制高。
*   **放任型**：情感低，控制低。

这四种方式会引发孩子不同的依恋模式。孩子会以相应的方式回应父母，形成对应关系。

以下是四种养育方式对应的孩子依恋模式：

*   **民主型**对应**安全型依恋**。孩子能从父母那里感受到**爱**（来自高情感）与**要求**（来自高控制）。爱与要求缺一不可：爱太多、要求太少，孩子没有边界；爱太少、要求太多，孩子内心没有能量。民主型家庭是我们推荐的家庭模式。
*   **专制型**对应**焦虑-抵抗型依恋**。父母很专制，孩子自己的声音发不出来，因此会对父母产生抵抗。
*   **溺爱型**同样容易导致孩子陷入**焦虑-抵抗型**或一种无能、无力的状态。通常，专制型与溺爱型会像阴阳两极一样，在家庭中互动存在。例如，祖辈溺爱，父母就可能更专制。
*   **放任型**对应**回避型依恋**。孩子既得不到爱，也得不到关注和要求，会对世界感到绝望，形成冷漠、回避的状态。很多童年受伤的人，特别是许多中国男性，容易呈现这种回避型状态。他们话少，不表达、不倾诉，最终选择“不理就行”，内心可能对世界极度悲观。

本节课中我们一起学习了抚养方式与依恋模式的对应关系。有什么样的因，就有什么样的果。抚养方式是“因”，依恋模式是“果”，两者相互对应。一个人的性格形成都有其原因，除非是石头缝里蹦出来的。理解这条主线，能帮助我们以更系统、整体的观点看待亲子关系与个人成长。
