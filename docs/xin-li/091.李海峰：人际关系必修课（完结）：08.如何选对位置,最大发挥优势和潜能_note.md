## 概述：什么是团队？

生活处处是团队。学校里有学生会、协会；公司里有部门、工会；商业中有商会、同盟。甚至国家之间也有欧盟、东盟等组织。在游戏中，我们也渴望拥有好团队。

有一个关于圈子的理论：**你的身价是你身旁最亲密的五个伙伴的平均值**。提升自己通常有两种途径：一是不断更换身边的伙伴；二是与身边人共同成长。无论哪种，首先需要在团队中找对位置，发挥优势，才能谋求发展。

## 一、 从《三国演义》看团队角色定位 🎭

上一节我们提到了团队的重要性，本节中我们来看看如何定位自己。我们可以从《三国演义》的蜀国核心团队入手，分析不同性格特质如何对应不同的团队角色。

以下是蜀国核心团队成员的性格分析：

*   **张飞**：以 **D（支配）特质** 为主。他是行动派，目标明确，行动迅速，但较少关注他人，不善于复杂布局。因此他常做“急先锋”，而非独挑大梁。他的公式可以概括为：**行动 = 专注目标 + 快速执行**。
*   **关羽**：以 **I（影响）特质** 为主。他重情重义，个人魅力极强，扮演着“战神偶像”的角色。他的存在为团队赢得了大量粉丝与舆论支持。他的核心是：**影响力 = 个人魅力 + 情感连接**。
*   **诸葛亮**：以 **C（谨慎）特质** 为主。他负责全盘规划与管理，事无巨细，追求完美。他的职能是：**管理 = 周密规划 + 精益求精**。
*   **刘备**：以 **S（稳健）特质** 为主。他是协调高手，用情感维系团队，让每个强者认识到自身价值并为共同目标奋斗。他的作用是：**协调 = 情感维系 + 价值整合**。

这个团队之所以能三足鼎立，正是因为四种特质互补，形成了合力。如果全是张飞，则有勇无谋；如果全是诸葛亮，则可能纸上谈兵。

## 二、 使用PPF模型找到你的团队角色 🔍

知道了经典团队的角色构成，我们如何分辨自己是团队中的“刘备”、“关羽”、“张飞”还是“诸葛亮”呢？

我们可以使用 **PPF模型** 从三个维度来衡量自己的团队风格：
1.  **步调 (Pace)**：我是深思熟虑（慢性子）还是充满活力（急性子）？
2.  **观点 (Perspective)**：我表达观点时，是喜欢描述细节，还是喜欢概括要点？
3.  **焦点 (Focus)**：我工作时，焦点是放在“人”上，还是放在“事”上？

例如，当听说班级里有孩子打架时，有人首先问“有没有人受伤？”（焦点在人），有人则问“为什么打架？怎么处理的？”（焦点在事）。

确认自己在PPF三个维度上的状态后，就可以对照“团队角色比对表”（可在课程资料包中下载），找到自己倾向的团队角色，例如“组织者”、“创新者”等。

## 三、 认识局限：没有人是全能英雄 🚫

找到“自己是谁”只是第一步。同样重要的是，要认识到“自己暂时无法成为谁”。这样才能放下自我，更好地与团队配合。

例如，擅长协调人际关系的人（高S），可能不擅长精细算账（低C），这时就需要寻找擅长此道的伙伴来补位。

即使是三国英雄，也有其性格局限：
*   **张飞**：不关注下属情感，最终被部下所害。
*   **关羽**：重个人情义胜过全局，导致大意失荆州。
*   **刘备**：被私人情感左右，为报仇而贸然出征，惨败。
*   **诸葛亮**：过于追求完美，未能培养出合格接班人。

**没有完美的个人，只有完美的团队。** DISC理论强调打配合、做组合，因为个人很难面面俱到，但团队可以互相提醒、互补短板。

## 四、 从《西游记》看团队中的调试与发展 🚀

找到位置后，如何发挥优势？有时优势需要根据团队目标进行调整或放大。《西游记》的取经团队在这方面提供了更复杂的范例。

以下是取经团队成员的特质与调试：

*   **沙僧**：典型的 **S（稳健）特质**，勤恳踏实，坚守岗位。
*   **猪八戒**：典型的 **I（影响）特质**，情感丰富，活跃气氛。
*   **孙悟空**：入团队前追求官位、向往自由（高D/I）。入团队后，作为技术核心（CTO），被迫承担更多责任，变得更加注重细节、谨慎探索（发展了C特质）。代码式的转变可以表示为：`角色 = “齐天大圣” -> “技术总监”；行为 += 注重细节 + 谨慎探索`。
*   **唐僧**：原是与世无争的高S和尚。成为团队领袖后，为达成“西天取经”的坚定目标，必须使用强势的 **D（支配）特质** 来把握方向。他的转变是：`目标 = “自我修行” -> “带领团队”；特质 += 强势坚定 (D)`。

为什么孙悟空和唐僧需要做出巨大调整，而猪八戒和沙僧相对轻松？关键在于**职位高低与职能含金量**。中高层管理者往往需要为团队目标做出更多的性格调试与行为改变。

## 五、 运用GPS原则，适应团队需求 📍

既然“在其位，谋其政”，那么如何主动适应团队需求进行调试呢？这里介绍 **GPS原则**。

以下是GPS原则的三个核心要点：

*   **G (Goal - 目标一致)**：个人目标必须与团队目标协调一致。团队就像一列驶向终点的火车，方向唯一。良性团队需要共同目标、运行机制和角色分配。要发挥D特质，把团队目标当作自己的目标。
*   **P (Professional - 专业精深)**：提高个人专业能力，增加专业深度。发挥C特质的严谨，追求自我突破与完美，以适应团队更高的要求。记住“一万小时定律”，但必须是正确、到位的练习。
*   **S (Support - 主动支持)**：增加自身弹性，主动补位。发挥S特质的适应性，当团队角色出现缺失时，能够转换角色，支持团队临时变动。要像“革命的一块砖”，哪里需要哪里搬。

在GPS之外，别忘了增加角色弹性，勇敢举手说“我来！”，体验新岗位，迎接新挑战。

## 总结

本节课中，我们一起学习了：
1.  **找到自己的位置**：通过PPF模型和经典案例，了解自己在团队中“可以成为谁”以及“暂时无法成为谁”。
2.  **发挥自己的优势**：通过GPS原则（目标一致、专业精深、主动支持），学习如何根据团队需求进行自我调试与发展。

没有人是全能的英雄，但只要懂得在团队中打配合、做组合，充分发挥自己的优势，就能与团队彼此成就，共同取得更大的成功。
