# 谈判高手课 115：谈判的五种结果与核心策略 🔥

## 谈判的利益分析：Must、Want、Give

上一节我们介绍了谈判的基本概念，本节中我们来看看如何分析谈判中的利益。谈判时，必须明确区分哪些是你的核心需求，哪些是可交换的，哪些可用来建立共识。

以下是三个关键的利益类别：

*   **Must（必须项）**：这是非要不可的条件。如果没有这项，谈判将无法达成协议。公式可表示为：`协议达成 = Must条件被满足`。
*   **Want（想要项）**：这是可要可不要的利益点，主要用来进行交换。
*   **Give（可给项）**：这是你可以主动让出的利益，主要用来建立互信和共识。

在商业谈判中，一个简单的排序方法是：将所有的要求都转换为金钱价值进行计算。例如，交货延迟或付款账期延长所造成的损失金额，金额越大，优先级通常越高。

同时需注意，你认为的“Must”，对方可能只视为“Want”。双方视角的差异正是谈判中可以交换和博弈的空间。

## 谈判的五种可能结果

了解利益分类后，我们来看看谈判可能导向的五种结果。谈判并非只有输赢与和解。

以下是五种谈判结果：

1.  **赢**：在需要一次决胜负或追求自身最大利益时，以达成己方主要目标为结果。
2.  **和**：当双方注重维持长远关系时，追求一种相对公平、双方都能接受的和解状态。
3.  **输**：有时为了“放长线钓大鱼”，维护更重要的长期关系，可能会选择主动或策略性地让步。例如，与上司进行“政治性”博弈时，可能需要“力战而败”。
4.  **破**：即破局、暂停谈判。这可用于挫伤对方锐气、控制谈判节奏，或为等待有利时机（如新政策出台、市场变化）。其英文对应概念可理解为 **Pause（暂停）**。破局后，谈判仍可重启。
5.  **拖**：即拖延时间。目的在于等待外部条件发生变化，从而增加己方筹码，例如等待关键数据发布或新合作伙伴入场。

许多中国谈判者常犯的错误是“不敢破”，担心破局导致关系断裂。但有效的破局需要策略，关键在于 **事先找好退路**，并为重启谈判留下可能性。

## 支撑要求的“柱子”：正当性与可行性

明确了目标和可能的结果后，你的谈判要求需要有力支撑。任何要求都不能凭空提出，必须建立在“柱子”之上。

以下是构建谈判正当性的常见“柱子”：

*   **市场数据**：如行业报告、市场占有率。
*   **成本分析**：如原材料（`塑料`、`铜箔`）的价格波动分析。
*   **权威标准**：如通貨膨脹率、官方统计数字。
*   **专业研究**：来自付费数据库或第三方机构的分析报告。

仅有正当性还不够，还必须让对方看到达成协议的希望，即 **可行性**。你需要“留一条路给对方走”，让对方看到“隧道尽头的光亮”。

一个常见误区是：为了在砍价中不落下风，故意表现出“完全不想要”的态度。但这可能导致对方直接放弃与你谈判。正确的做法是让对方觉得“有希望达成”，但“条件需要协商”。

## 谈判的关键技巧：从“是否”到“如果”

那么，如何具体地给出这条“路”呢？答案在于改变说话的方式。

谈判中，应避免使用绝对的肯定句（Yes）或否定句（No），而要多用 **条件句（If）**。

例如，顾客问“能不能便宜点？”，店主不应直接回答“能”或“不能”，而可以说：“**如果您买三个的话，我可以给您一个批发价。**” 这就是一个典型的条件句（If）。

几乎所有条款都可以进行条件化探讨：
*   “外商公司不能竞标” -> “外资持股比例低于多少才算‘外商公司’？”
*   “3月1日交货” -> “‘交货’是指完成全部交付，还是开始交付第一笔？”

这种技巧，在中国传统智慧中称为 **“百合之术”**。“百”即开门，创造可能性；“合”即关门，设定界限。谈判之道，本质上就是在开合之间找到平衡点。
