# 118.60秒口才训练营（完结）：付费课：【付费课23】思行说训练之回顾那些今天_note

在本节课中，我们将学习“回顾那些今天”这一思行说训练方法。我们将探讨如何通过回顾过去特定日子的经历，与当下进行对比，从而获得深刻的个人洞见和启发，并最终将这些思考转化为有影响力的表达。

---

上一节我们介绍了过程回顾训练，本节中我们来看看“回顾那些今天”的具体实践。

一个有思想的大脑在脱稿说对别人有帮助或者启发的话的时候，对嘴的协调控制能力，这是我们一直强调的口才定义。当脱稿表达能力已经具备时，重点应转向如何让自己“有思想”。脱离具体事实的空想难以产生真正的思想。因此，思行说训练旨在帮助大家养成从日常行为、工作或生活中进行思考的习惯。

前三周我们进行了价值回顾、成果回顾和过程回顾。本周的训练主题是“回顾那些今天”。这里的“那些今天”指的是过去的某个对应日期，例如上周的今天、上个月的今天、去年的今天或三年前的今天。回顾那一天你在做什么，并将其与今天对比，这能带来许多启发。

以下是进行“回顾那些今天”训练时可能产生的几种思考与启发：

*   **视角转变**：曾经认为至关重要的事件（如高考），多年后回顾可能只是人生中普通的一天。这提示我们，当前面临的巨大挑战在未来回顾时，其重要性也可能改变，从而帮助我们调整心态。公式表示为：`当前问题的感知重要性 = f(时间距离)`。
*   **模式识别**：回顾不同时期，可能发现相似的痛苦或烦恼以不同形式重复出现。这让我们认识到痛苦无法完全避免，从而将心态从“逃避痛苦”转向“接纳并解决问题”。
*   **经验总结**：回顾过去满怀信心启动的项目，可能会发现当初考虑不周。这提供了吸取教训的机会，帮助我们在当下将事情做得更好。
*   **价值观反思**：过去认为极具价值的事物，现在可能感觉不再那么重要。这促使我们反思：是什么发生了变化？是自己的价值观变了？什么才是对自己真正重要的？

许多道理需要时间的沉淀和个人的不断反思才能真正领悟。仅仅“听过”道理是不够的。而记录与回顾，正是推动我们思考、帮助我们更好地发现和认识自己的关键工具。否则，人生许多时光可能在模糊中流逝，仿佛未曾真正活过。

为了让生命更充实，我们需要持续记录、反思和回顾。可以从现在开始记录，这样未来就能回顾“上周的今天”、“上个月的今天”等。回顾时，问自己：当时在为何事烦恼或开心？当时认为什么有价值？今天的你再看，有何感受、思考和启发？

一旦产生思考，就要尝试将其转化。思考这些启发能否对他人有帮助、有触动？能否将其录制成一期节目分享出去？

因此，本周的练习是：当你回顾“那些今天”并产生思考后，请将其录制成节目。本周录制三期节目进行分享。形式不限。

我希望大家从今天开始积极思考，并努力爱上思考。因为热爱思考是拥有好口才的根本源泉。一个不爱思考的人，很难具备真正的好口才。让我们共同努力，成为爱思考的人，最终成为拥有好口才的人。

本节课中，我们一起学习了“回顾那些今天”这一训练方法。我们明白了通过对比过去与现在的经历，可以获得关于视角、模式、经验和价值观的宝贵启发。关键在于持续记录并将个人思考转化为能启发他人的表达，这是锤炼思想、提升口才的核心路径。
