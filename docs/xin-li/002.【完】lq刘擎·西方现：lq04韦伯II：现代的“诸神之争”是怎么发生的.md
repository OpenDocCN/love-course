# 002.【完】lq刘擎·西方现：lq04韦伯II：现代的“诸神之争”是怎么发生的

在本节课中，我们将要学习马克斯·韦伯的第二个核心思想命题——“诸神之争”。我们将探讨科学理性在成为现代社会主导思想后，为何无法解决价值观领域的冲突，以及这种“诸神之争”对个人与社会带来的深刻影响。

上一讲我们介绍了韦伯的第一个重要命题——“祛魅”，它描述了从古代到现代的重大观念转变。本节中我们来看看韦伯的第二个重要命题。

## 一、从“祛魅”到“诸神之争”

“祛魅”一词指科学理性驱散了世界的神秘气氛，人们认识到世界是物质的，可以用理性去认识。科学理性因此成为现代社会的主导思想。

但是这带来了一个新问题，即韦伯的第二个重要命题——“诸神之争”。这里的“诸神”并非指多种神灵，而是比喻人们各自信奉的**价值观**。“诸神之争”即指**价值观之间的冲突**。

## 二、事实判断与价值判断的鸿沟

那么问题来了：既然科学理性已成为主导，观念的战争是否已经结束？答案是否定的。因为这是一场关于终极价值的战争，科学理性并未获胜，也并非其战场。

用学术语言来说，科学理性有其作用范围，**它不能为人类生活的终极价值提供答案**。科学理性的力量在于**事实判断**领域，而“诸神之争”发生在这个领域之外。

以下是两种判断类型的核心区别：

*   **事实判断**：描述事物“实际上是什么”。其判断有客观标准验证。
    *   **公式/代码示例**：`if (location == “北京”) { 事实为真; }`
    *   例如：“清华大学在北京。”这是一个可以通过观察验证的事实判断。

*   **价值判断**：表达事物“应当怎么样”。其中隐含着价值高低的取向。
    *   **公式/代码示例**：`if (city == “上海”) { 价值更高; } // 这是一个主观判断`
    *   例如：“清华大学应该搬到上海去。”这表达了“在上海会更好”的价值取向。

事实判断容易达成一致，因为客观现实是验证标准。价值判断则不同，例如“上海更好”与“北京更好”的争论，双方可能各有理由且难以说服对方。

## 三、科学理性的力量与局限

在事实判断领域，科学理性是“无敌的高手”。现代科学拥有成熟的事实检验手段（仪器、实验、审核机制）。只要是事实问题，总有可能通过技术手段搞清楚，获得**高度的确定性**。

然而，在价值判断领域，情况截然不同。价值判断需要分辨好坏对错，这也需要一套标准。

在古代，人们相信世界存在一个统一的、神秘的终极秩序和答案。“祛魅”之后，我们知道自然世界是物质的，并无神秘的终极答案。那么，现代人进行价值判断的依据，就变成了**个人形成的一套价值标准**。

这套标准具有很强的主观性，受到国籍、文化、性别、职业、家境、爱好等诸多个人因素的影响。例如，一个长期养宠物的人与不养宠物的人，在动物保护问题上可能持有不同倾向。

在价值领域，**没有一把通用的尺子可以衡量一切**。事实判断“有章可循”（客观世界），而价值判断“无章可循”，我们无法断言哪一种价值观是唯一正确的。

## 四、价值多元化的困境

唯一的答案消失了，留下无数不确定的选项，我们便进入了**价值多元化的困境**。

价值多元化有其积极面：它带来了更大的选择空间和个人自由。例如，在中世纪欧洲，不信宗教可能很危险，而现代人则可以自由选择信或不信。

但是，价值多元化也有其消极面，体现在个人与社会两个层面。

以下是价值多元化在个人层面造成的问题：

*   **增加困惑与迷茫**：人总要寻求意义。韦伯有句名言：“人是悬挂在自己编织的意义之网上的动物。”在价值多元化中，我们有很多选择，却无人能断言哪个最好。
*   **“我喜欢”成为脆弱的标准**：当无法给出确定回答时，“我喜欢”常成为最终标准。但个人意愿善变，建立在此上的选择是脆弱的。
*   **引发不确定感与焦虑**：糟糕的并非选项多，而是“选了却永远不知对不对”。这种长期的内在动摇和不确定感，是现代人显著的精神特征，近乎一种时代病症。

科学理性打破了传统价值规范，却未建立新标准，造成了**价值真空**。这种空虚的不确定性，让现代人很容易被焦虑和无意义感困扰。

在社会层面，价值多元化同样带来棘手问题。公共生活中许多激烈对立的议题，本质都是价值观冲突。

以下是两个典型的社会价值观冲突案例：

*   **案例一：堕胎合法化**
    *   **一方观点**：依据宗教信条，认为堕胎等于谋杀生命。
    *   **另一方观点**：认为核心是女性是否有权自由处置自己的身体，这是个人的基本权利。
    *   **冲突本质**：两种观念都有其道德依据，难以用理性裁定孰对孰错。

*   **案例二：政治基础价值**
    *   **核心争议**：是安全与秩序更重要，还是个人自由与权利更重要？
    *   **不同导向**：前者导向强有力的国家保障秩序；后者要求严格限制政府权力。
    *   **现实表现**：此类争论（如美国政坛）可能持续数百年，不仅难有共识，反而可能导致政治分裂加剧。

## 五、韦伯的冷酷回答与伏笔

那么，能否寻求共识以弥合分歧？韦伯的回答有些冷酷：**分歧的根源如此深刻，许多冲突无法化解。终极价值彼此不相容，它们之间的斗争不会有结论**。

此处留一个伏笔：这个问题是否真如韦伯所言那般绝对？半个世纪后，德国哲学家哈贝马斯提出了不同看法，我们将在后续课程中探讨。

本节课中我们一起学习了韦伯的“诸神之争”思想。

“诸神之争”的本质是现代社会中不同价值观之间的冲突。科学理性在事实领域赋予我们把握确定性的强大武器，但在价值领域却无法给出确定答案，导致价值观冲突不断。这在个人层面引发了困惑、焦虑与无意义感，在社会层面则造成了诸多难以调和的公共议题对立。

但科学理性给现代社会带来的麻烦不止于此，我们下一讲继续探讨。
