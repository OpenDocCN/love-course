# 课程032：情感心理咨询技能大全（完结） 🧩  
## 第06章：结构式家庭治疗—陈发展  
### 13. 案例展示 🎬  

在本节课中，我们将通过一个真实的家庭治疗案例，展示如何运用结构式家庭治疗的理论与技术来理解和干预家庭系统。案例涉及一名13岁游戏障碍少年，我们将从家庭结构、互动模式及治疗过程等多个维度进行分析。请注意，为保护隐私，案例中的个人信息已进行虚化处理。  

---

### 案例背景介绍  

上一节我们介绍了结构式家庭治疗的核心概念，本节中我们来看看如何将这些概念应用于实际案例。  

这是一个13岁男孩的案例，他被诊断为游戏障碍。来访者是一名初一年级学生，主要问题是沉迷手机游戏一年，并已拒绝上学一个月。  

以下是该案例的基本情况：  

*   来访者为13岁男孩，就读于重点初中。  
*   约一年前（六年级/预备班）开始接触手机游戏，初期每天玩1-2小时。  
*   游戏时间逐渐增加至每天3-4小时，影响作业完成与睡眠，导致白天课堂打瞌睡、成绩下降。  
*   2019年春季开学后，问题加剧，甚至深夜偷偷玩游戏。  
*   父母曾采取没收手机、打骂、摔手机等方式干预，但效果有限。  
*   在就诊前一个月（10月底），来访者开始拒绝起床、上学，不与家人沟通，认为上学无意义，昼夜颠倒在家玩游戏。  

---

### 心理状态评估  

在开始任何治疗前，对个体的心理状态进行评估是至关重要的步骤。  

以下是心理评估的主要发现：  

*   **意识与定向**：清晰、准确。  
*   **外观与行为**：头发凌乱，检查时不合作。来访者声称自己“没问题”，只是为办理休学而来。  
*   **情绪**：略显低落，认为读书和人生意义不大，但无自杀意念或严重消极想法。曾有一次“跳楼”威胁，自述仅为吓唬父母。  
*   **思维与交谈**：对医生提问戒备，常用“不清楚”、“不想说”、“不知道”或“没有”来回答。无情感高涨或精神病性症状。  
*   **对游戏的态度**：认为玩游戏有意义，可以逃避写作业的痛苦和父母的唠叨。曾梦想成为电竞选手。不玩游戏时会感到不安、烦躁。  
*   **社会功能**：因长期不上学，对重返校园感到害羞，不知如何面对同学，但采取“将来再说”的回避态度。  
*   **自知力**：承认自己行为欠妥，但不认为有大问题。  

根据ICD-11的诊断标准，该来访者符合**游戏障碍**的诊断。

了解了个体情况后，我们需要将视角扩展到整个家庭系统。家谱图是理解家庭结构和关系的有效工具。

*   **夫妻关系**：用虚线连接，表示关系疏离，缺乏亲密感，沟通少，情感功能弱。  
*   **亲子关系**：母亲与儿子（来访者）之间连线复杂，代表关系既紧密又充满冲突。父亲与儿子关系相对简单。  
*   **重要事件**：母亲的弟弟在几年前去世，母亲一直未能从悲伤中走出，情绪不稳定。  
*   **代际模式**：父亲的父母（即来访者的祖父母）之间也经常冲突。  

从家谱图中，我们可以解读出以下家庭动力：  

1.  **关系三角化**：由于夫妻关系疏离，双方都将过多的情感关注投向儿子，形成了“父母-孩子”的三角关系，孩子成为家庭关系的核心。  
2.  **结构不平衡**：家庭中权力等级最高的**夫妻子系统**功能薄弱，而**亲子子系统**过度紧密，这种结构是不平衡的。  
3.  **代际影响**：父亲可能因自己原生家庭的冲突经历，为避免与妻子冲突而选择疏离；母亲则因弟弟去世的未处理哀伤，将情感更多地寄托于儿子。  
4.  **症状的功能**：孩子的游戏障碍和拒学行为，可以看作是家庭结构性压力的体现。它暂时将父母的注意力从彼此的矛盾和母亲的悲伤上转移开，使夫妻为了“共同解决问题”而开始合作，从而**扰动**了原有的僵化系统。  

**公式**：家庭压力 → 孩子承受压力 → 出现症状 → 父母联合关注症状 → 暂时缓解夫妻矛盾 → 症状被维持。这是一个循环互动模式。

### 结构式家庭治疗的理解与过程  

基于上述评估，治疗师从结构式家庭治疗的视角进行了干预。治疗共进行了5次，过程灵活整合了多种技术。  

#### 第一阶段：建立关系与探索症状意义  

治疗初期，目标是融入家庭，建立信任，并和家庭一起探索症状在系统中的功能。  

以下是本阶段的主要工作：  

*   **共情与接纳**：充分理解并共情来访者（化名“博洋”）的感受和处境。  
*   **拓展症状意义**：通过提问，引导家庭思考症状背后的关系含义。例如：“当你开始打游戏/不上学时，你的爸爸妈妈和以前有什么不一样吗？”  
*   **布置观察作业**：让家庭成员观察症状出现时，家庭互动发生了哪些变化。  
*   **发掘积极资源**：肯定父母对孩子的付出与关爱，同时将过度关注视为一种资源。  

通过第一次交流，家庭获得了一个新理解：孩子的问题不仅是“游戏障碍”或“厌学”，也是在表达对父母关系的不满，以及他渴望独立、为自己负责的成长需求。  

#### 第二阶段：扰动关系与调整结构  

在建立良好关系的基础上，治疗师开始直接扰动僵化的家庭结构，促进适应性互动。  

以下是本阶段运用的核心技术：  

*   **去三角化**：强调家庭关系界限，提升夫妻子系统的核心地位。例如，提问：“如果你们家没有孩子，你们还会在一起吗？”  
*   **家庭雕塑（微雕塑）**：在咨询室中调整座位。初始时，孩子和母亲坐一起，父亲单独坐。治疗师让父亲和母亲坐在一起，让孩子单独坐，直观地呈现并推动结构改变。  
*   **外化技术**：将“问题”与家庭分开。例如，用一个纸巾盒代表“游戏问题”，问家庭愿意把它放在哪里。孩子选择放在“自己和父母中间”，这揭示了症状在模糊亲子界限上的功能。  
*   **挑战无效假设**：提问除了打游戏，孩子还有哪些优点？挑战父母“孩子一无是处”的消极观念。  
*   **面向未来的提问**：激发改变动机。例如：“如果梦想实现了，你们家和现在最大的不同是什么？”母亲的回答是：孩子上学，她和丈夫能单独去旅游。  

#### 第三阶段：巩固变化与预防复发  

在家庭开始发生变化后，治疗的重点转向巩固新互动模式，并预防退回旧模式。  

以下是本阶段的策略：  

*   **布置正向观察作业**：要求家庭在未来一个月，专门观察并记录彼此“好的变化”。这旨在引导家庭从关注问题转向关注资源和进步。  
*   **悖论干预/预防复发提问**：提问：“如果将来让你们的儿子再复发一次，你们家可能会怎么做？”母亲笑着回答会再次焦虑。这个回答本身让她意识到，未来需要做的就是减少焦虑和过度关注。  
*   **评估与反馈**：回顾整个治疗过程，总结家庭成员的收获，并讨论未来如何应对可能出现的新问题。

### 治疗效果与总结  

在本案例中，经过五次家庭治疗，家庭系统发生了显著的结构性变化。  

以下是治疗后的主要改变：  

*   **孩子方面**：游戏行为明显减少，认为游戏“挺无聊”，是“应对内心压力的方法”。作息变得规律，并已转学开始新的学习生活。  
*   **父母方面**：夫妻关系改善，沟通增多，更能彼此理解。父亲更支持母亲处理哀伤，母亲也更能理解丈夫的原生家庭困境。  
*   **亲子关系**：父母降低了对孩子不切实际的高期望（如必须考名校），更尊重孩子的感受和选择。他们甚至带孩子实地考察了电竞俱乐部，表现出更高的接纳度。父母学会自行处理压力和冲突，不再将其过度投射到孩子身上。  
*   **家庭结构**：夫妻子系统的功能得到增强，成为家庭的核心；亲子子系统的界限变得清晰，孩子获得了更多成长空间。  

**本节课中我们一起学习了**如何将一个结构式家庭治疗的完整案例，从评估、概念化到干预逐步拆解。核心在于理解：**青少年的症状往往是家庭结构性压力的体现**。治疗不是消除症状，而是通过调整家庭内部的关系结构（如强化夫妻联盟、清晰亲子界限），使系统恢复平衡与弹性，从而让症状失去存在的“功能”而自然缓解。本案例也说明，治疗过程并非机械遵循“四步模式”，而是灵活整合各种技术，与家庭共同创造新的互动可能。  

**最后记住**：每一个被视为“问题”的行为，在家庭系统中都可能具有维持系统平衡或表达某种需求的功能。
