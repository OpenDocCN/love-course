# 谈判高手课 115：让对方感觉“赢”的谈判秘诀 🎯

在本节课中，我们将探讨谈判中的一个核心秘诀：如何通过巧妙的策略设计，在确保己方实际利益的同时，让对方在心理上感觉自己是“赢家”。掌握这一点，是达成满意且可持续协议的关键。

## 谈判桌上的两把尺：是非与利害

上一节我们介绍了谈判的基本心态。本节中我们来看看谈判时我们内心衡量事物的标准。

我们心中通常有两把衡量事物的“尺子”：一把叫“是非”，一把叫“利害”。在谈判桌上，我们主要讲“利害”，而非“是非”。有人或许认为应当明辨是非，但那更适合个人修养。在谈判中，若指出对方“错了”并要求让步，会损害对方的面子，可能导致谈判破裂。因此，我们的沟通应围绕“利害”展开，即向对方说明“让步对你最有利”。

## 感知的得失：让对方感觉“赢”

理解了谈判讲“利害”后，我们进一步探讨如何操控“得失”的感知。

所有的谈判，都必须让对方感觉他得到的比我们多。他不需要真的得到更多，只需要“感觉”如此即可。人很容易产生这种“感觉”。

**公式示例**：
假设卖方成本为6000，期望卖7000，开价8000。买方还价7000。
*   如果立刻以7000成交，卖方高兴（达到期望），但买方会觉得谈判未过中点（7500），感到不满意。
*   更优策略：卖方在8000上“闷一下”，表现出为难，最后让步到7300成交。
*   结果分析：
    *   买方感觉：对方从8000让到7300，越过了中点7500，感觉自己“赢了”。
    *   卖方实际：以7300成交，比期望的7000还多300。

核心在于：谈判桌上只能有一个“笨蛋”，那就是我们自己。我们要让对方感觉他聪明、他赢了。

## 避免“强迫感”：给予控制权

仅仅让对方感觉赢还不够，还需注意方式，避免引发抵触。

人有一个基本需求：不希望被强迫。即使结果很好，如果过程让人感到被操控或强迫，也会产生不快。因此，在谈判或销售过程中，要巧妙地为对方保留“控制感”和“选择权”，而不是替他做所有决定。

## 沟通的四个象限：从对方角度切入

那么，具体该如何沟通，才能引导对方感觉赢且不被强迫呢？我们可以使用一个有效的分析框架。

我们所有的谈判说辞，大致都可以归入以下四个象限。这个框架的核心是分析“坚持某一立场的成本与效益”。

| 象限 | 论述角度 | 论述核心 |
| :--- | :--- | :--- |
| **第一象限** | 我方成本 | 阐述“我方坚持立场，成本很低”。（例：我告你成本很低，所以我一定会坚持。） |
| **第二象限** | 我方效益 | 阐述“我方坚持立场，效益很高”。（例：我这么做好处很大，所以不可能放弃。） |
| **第三象限** | **对方成本** | **阐述“对方坚持立场，成本很高”。** |
| **第四象限** | **对方效益** | **阐述“对方坚持立场，效益不高（或有更好替代方案）”。** |

在沟通时，**第三和第四象限（从对方角度出发）的说服力往往更强**。这不一定能立刻说服对方，但可以动摇他对自己原有立场的信心，促使他思考替代方案。

**应用示例**：
假设你是一家供应商，某连锁店为促销将你的产品大幅降价，扰乱了市场价格。你的谈判说辞可以切入第四象限：
“王总，您促销的目标是提升销量和关注度，对吗？坚持用‘降价’这个方法，效益可能没有想象中高，而且会伤害整个市场。我们是否可以一起想想别的促销方案？比如联合举办市场活动、增加陈列位，同样能达到您的促销目的，还不用牺牲价格和利润。”

## 寻找关键切入点：谁将获益

最后，在策划谈判时，还有一个关键点需要考虑。

你需要思考：如果对方与你合作，**他内部有哪些人或部门将因此获益**？找到那些能从协议中直接获得好处（如业绩提升、成本降低、工作更便捷）的“盟友”或决策影响者，从他们那里切入，谈判会顺利得多。反之，如果合作会导致对方内部有人利益受损（如裁员、部门权力削弱），那么协议很可能遭到抵制。

---

本节课中我们一起学习了让谈判收益翻倍且对方满意的核心秘诀：**在心理上让对方感觉他是“赢家”**。我们了解到谈判应聚焦“利害”而非“是非”；通过操控“得失”感知，让对方感觉获利更多；同时要避免制造“强迫感”，并善用从“对方成本”和“对方效益”角度切入的沟通策略。最后，找到对方内部能从合作中获益的关键人物作为切入点，将极大推动谈判成功。记住，最好的谈判结果是：**对方开心地觉得自己赢了，而我们也实实在在地拿到了想要的东西。**
