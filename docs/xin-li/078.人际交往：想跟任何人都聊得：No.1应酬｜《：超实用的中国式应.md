# 078.人际交往：想跟任何人都聊得：No.1应酬｜《：超实用的中国式应

在本节课中，我们将学习《学会应酬，半生不愁》一书的核心内容。这本书是一本超实用的中国式应酬指南，旨在帮助我们理解并掌握在各种社交场合中得体应对的技巧。我们将从应酬的基本准备、常用技巧、说话套路、求人办事、化解困局以及交友之道六个方面进行系统学习。

## 一、 应酬前的准备及注意事项

上一节我们概述了课程内容，本节中我们来看看进行应酬前需要做哪些准备和注意哪些事项。充分的准备是成功应酬的第一步。

**第一，明确你的身份与角色。**
在应酬前，首先要思考你在对方面前应该扮演什么角色。例如，在上司面前应保持下属的姿态，在朋友面前应平等沟通，在伴侣面前则应放下工作身份。身份转换不当可能导致沟通障碍。

一个著名的例子是维多利亚女王的故事。女王深夜回家敲门，丈夫阿尔伯特问“谁？”，女王先后回答“我是女王”和“维多利亚”，门均未开。直到她回答“我是你的妻子”，门才打开。这个故事说明，未能根据交际对象调整身份和语气，会伤害对方自尊，影响关系。

**第二，注意你的着装。**
着装是你的第一张名片。人们会从你的穿着打扮揣摩你的个性、品味甚至为人。原则是“见什么人，穿什么装”。一个着装不当的例子是，一位穿着朴素的老板带秘书见客户，客户误将秘书认作老板而先与之握手，造成了尴尬。得体的着装既方便自己，也尊重他人。

**第三，理解“八小时以外”的应酬重要性。**
工作之外的应酬往往关乎人脉、资源和机会。虽然有些应酬可以拒绝以陪伴家人，但关乎职业前途的重要应酬应当积极参与。在竞争激烈的环境中，与客户或同事建立良好私交，有时能成为达成合作的关键。

**第四，在不同场合运用不同技巧。**
应酬时，如何填补“您好”和“再见”之间的空白很重要。作者针对不同场合给出了建议：

以下是针对不同场合的谈话建议：
*   **交际宴会**：可以稍高调，多谈论自己擅长的领域，给人留下深刻印象。
*   **同事聚会**：以拉近距离为目的，避免谈论公私间的私密事宜，谨防隔墙有耳。
*   **朋友聚会**：氛围应轻松，避免丧气话，多叙旧并幽默谈论现在与未来。
*   **宴请客户**：话题很重要。避免直接推销，可从与生意相关的时事热点切入，适时引入产品。
*   **相亲场合**：避免一上来就询问物质条件。应提前了解对方兴趣爱好，围绕共同话题展开交流。例如，可以学习一些关于星座、流行文化、时尚等的话题。

**第五，不要冷落“次要者”。**
在任何应酬中，都要尽可能照顾在场每个人的感受，让每个人都感觉到被重视。高情商的人会主动与可能被冷落的人交谈。

以下是照顾“次要者”的四种方法：
1.  **时常微笑示意**：用眼神、点头或微笑表示关注。
2.  **询问平常问题**：例如“菜合口味吗？”，重点是传递在意。
3.  **示意用餐喝茶**：在谈话间隙，示意对方用茶、用点心。
4.  **邀请参与话题**：例如说“小王，这事你怎么看？”，将对方拉入讨论。

总之，成功的应酬需要重点照顾主要对象，同时周全顾及在场所有人。

## 二、 应酬高手的常用技巧

了解了基本准备后，本节我们来看看应酬高手们常用哪些技巧来游刃有余地应对各种局面。

**技巧一：装糊涂。**
即使知道某事，当别人提起时，也可以表现出感兴趣的样子，鼓励对方多说。这是一种“大智若愚”的智慧。公式是：`表现兴趣 > 表现全知`。总是说“这我早就知道”的人，会让人失去交谈欲望。

**技巧二：善用自嘲。**
当因自己失误造成尴尬时，最聪明的方法是自嘲。通过调侃自己、放低姿态来圆场，既能化解尴尬，又能体现修养。例如，同学聚会上说错话后，可以补救说：“你看我这张嘴，还和当学生时一样没把门的，该掌嘴。”

**技巧三：不轻易指点别人。**
直接指出别人的错误，会打击对方的智慧、判断力和自尊心，从而引发反击。英国政治家查士德·斐尔爵士对其儿子说：“如果可能的话，你要比别人聪明，但是不要告诉人家你比他聪明。”

如果想表达不同意见，可以使用更缓和的句式：
```text
“我倒是有另一种想法，但也许不对。如果我说错了，我很愿意大家帮我纠正过来……”
```

**技巧四：思路灵活，不较真。**
应酬高手懂得世界并非非黑即白，存在灰色区域。他们不会过于较真或被条条框框束缚，懂得适时“睁一只眼闭一只眼”。斤斤计较的人很难交到朋友。

**技巧五：保持低调。**
低调是一种可进可退的处世策略。包括：
*   **姿态低调**：避免盛气凌人，保持亲和力。
*   **心态低调**：有了成绩也不看不起别人，懂得功成身退。
*   **言辞低调**：说话谦和，避免伤人。例如，一位总是断后的大将军在被称赞勇敢时说：“非好勇也，马不进也。”（不是因为我勇敢，是马跑得慢）。这种谦虚反而赢得了更多尊敬。

**技巧六：学会“留缺口”。**
不要把事情做满，留一些选择或余地给别人。在管理中这是一种智慧，在人际中亦然。例如，家中一人若大包大揽所有家务，久而久之，其他家庭成员便不再参与，并可能形成依赖与抱怨的循环。

书中还讲了一个关于“留缺口”获取借条的故事：张某借给朋友1200元但借据丢失。他给朋友发传真要求归还2100元。朋友生气地回复传真澄清只借了1200元。这份回复传真便成了新的证据。张某随后道歉说是笔误，顺利拿到了“借条”。

## 三、 应酬中说话的套路

掌握了高手的常用技巧，本节我们深入探讨应酬说话中那些行之有效的具体套路。

**套路一：红脸与黑脸。**
即根据情境需要，交替使用温和与严厉的态度。如同领导在会议上先表扬后批评，让下属摸不透心思，从而建立权威。一味和气可能被当作软柿子，适时“变脸”是必要的社交技能。

**套路二：老实人不一定总说老实话。**
特别是在面试等场合，对于“为何离职”等问题，明智的回答是聚焦于公司机制、个人发展等客观因素，而非抱怨前公司或同事。因为对方真正想考察的是你的品行和格局。

**套路三：说“打埋伏”的话。**
在提出真实请求前，先说一段铺垫的话进行试探。例如，想借朋友的车，不应直接说“把你车借我用用”。更好的方式是：“今天有件急事，租车又来不及，真郁闷。” 如果朋友有意，自然会问详情并可能主动提出借车；如果朋友无意，也会为自己找理由推脱，避免了双方直接拒绝的尴尬。

**套路四：如何得体地拒绝。**
拒绝是一门艺术，目的是既表明立场，又不得罪人。
以下是几种拒绝方法：
1.  **果断拒绝**：不拖延，不给对方虚假希望。
2.  **电话拒绝**：非当面沟通，缓冲空间大。可以说“让我想想”，再电话回复解释。
3.  **自言自语**：让对方“无意中”听到你的难处。
4.  **委婉转折**：使用“你的心情我理解，但是……”的句式。
5.  **模棱两可**：使用“再看看吧”、“等我跟领导汇报后回复你”等拖延话术。
6.  **提议他法**：“我实在不方便，要不你问问小王？改天我再回请你。”

**套路五：当着矬子不说矮话。**
切勿当面揭人短处，这只会导致关系恶化或引来对方反击。

**套路六：不在背后议论他人。**
中国俗语说：“宁在人前骂人，不在人后说人。”在一个人面前说第三者坏话，听者会想你可能也在别人面前说他，从而损害你自己的形象。
如果遇到别人在你面前说某人坏话，最佳应对是：**入耳封存，不做评价，尽快转移话题。** 如果自己的坏话传到当事人耳中，应主动道歉寻求谅解。

**套路七：回避隐私问题。**
如年龄、收入、婚姻感情、私生活等属于隐私，不要随意打听。尤其是在过年期间，追问他人收入非常令人反感。

**套路八：在失意者面前不谈得意事。**
在失业的朋友面前大谈加薪，或在失恋的人面前秀恩爱，无异于在伤口上撒盐，非常不得体。

## 四、 如何求人办事

说话套路帮助我们日常沟通，而求人办事则是应酬中的一项关键应用。本节我们来学习如何有效地求人。

**第一，克服“不好意思”的心理。**
人际关系的本质是互帮互助。脸皮太薄，万事不求人，反而会错失加深交情的机会。今天你帮我，明天我帮你，关系便在其中建立起来。

**第二，以礼开道——送礼的学问。**
送礼是求人办事常见的铺垫，需讲究方法：
1.  **送对人**：找到有决定权的人。
2.  **投其所好**：送对方喜欢或需要的东西。
3.  **权衡利益**：根据所求之事的重要性决定礼物的轻重。
4.  **因时制宜**：选择恰当的送礼时机，如对方上班前的早晨，可能比晚上家中常有客时更合适。
5.  **尊重习俗**：了解对方的文化或地方禁忌。
6.  **体现真情**：根据对象选择礼物（病人送花/营养品，老人送实用品，朋友送有趣之物，外宾送有中国特色的礼品等）。

**第三，吃透对方心理，让其难以拒绝。**
1.  **先大后小法**：先提一个大概率会被拒绝的大要求，再提真正的小要求。例如，想借1万元，先开口借10万，被拒后再提出借1万，对方出于补偿心理更容易答应。
2.  **激发同情法**：适当示弱，倾诉难处。
3.  **戴高帽法**：把对方抬高到“只有您能帮我”的位置，使其不好意思拒绝。

**第四，提出请求的技巧，避免尴尬。**
1.  **自我否定法**：“有件事想请您帮忙，但看您这么忙，估计没时间……” 给对方提供选择空间。
2.  **投石问路法**：先提出相关试探性问题。例如，想托出差的朋友捎东西，先问“这次出差要带不少东西吧？”，若对方抱怨行李多，便不宜再开口。
3.  **顺便提出法**：以随意、轻描淡写的口吻提出，减轻被拒时的心理压力。
4.  **开玩笑法**：用玩笑的口气提出，即使被拒也可归为玩笑，保全双方面子。
5.  **打电话法**：非当面提出，缓冲尴尬。

## 五、 如何打圆场，化解困局

求人办事是主动出击，而打圆场则是被动化解。本节我们学习如何在各种尴尬或冲突场合中巧妙应对。

**情况一：打招呼对方不理睬。**
首先自己保持淡定。可以尝试用更直接、提高声调的方式再打一次招呼。如果是对方因故（如闹别扭）故意不理，则不必纠结，稍后再处理。

**情况二：聊天冷场，无话可谈。**
应酬是“逢场作戏”，需要主动寻找话题。根据对方的喜好、职业、背景选择话题。一些安全且常见的话题包括：天气、热点新闻、体育赛事、影视作品、美食、旅游、养生、交通等。从中试探出对方的兴趣点，再深入交流。

**情况三：调解他人纠纷。**
1.  **两头道歉做和事佬**：在家人（如婆媳）争吵中，有时需要居中调和，向双方示好。
2.  **不偏不倚，各打五十大板**：公平地指出双方都有不对之处。例如，经理调解同事争吵时，既批评男同事不该不让着女同事，也批评女同事不该得理不饶人。
3.  **主动揽责**：将过错引到自己身上以平息争执。例如，张经理对因未通知听报告而吵架的两位同事说：“没通知你是我的错，我没让他告诉你。要怪就怪我吧。”
4.  **圆中有方，劝中带威**：在劝解时既表示理解，又暗示后果。例如，调解纠纷时可以说：“你的心情我能理解，但在这里闹出事情，对大家都不好，还得麻烦其他同事过来处理，何必呢？”

**情况四：劝架。**
*   **已动手**：立即站中间分开双方，大声喝止，并呼叫旁人帮忙拉开。注意保护自己，避免被误伤。
*   **未动手，只争吵**：保持冷静，倾听双方诉求，抓住核心矛盾点进行劝解，帮助双方平复情绪。

## 六、 交友之道

最后，我们来探讨维系长期关系的根本——交友之道。良好的朋友关系是成功应酬的基石。

**第一，充实你的“人情账户”。**
人情像存款，需平时积累，切忌临时透支。要与朋友保持定期联系（建议一周或半个月一次），而不是有事才找。逢年过节、生日时的问候和心意，都是有效的“存款”方式。真正有价值的人情需要长年累月的培养。

**第二，警惕“突然升温”的友情。**
如果久不联系的朋友突然变得异常热情，很可能是有事相求。应对方法如下：
1.  **慎收重礼**：对方若过于慷慨，要谨慎接受，避免欠下大人情。
2.  **加倍答谢**：如果对方帮忙，要加倍回报，这样即使对方日后有所求，你也能坦然应对或拒绝。
3.  **冷眼相看**：对突如其来的过热殷勤保持清醒，理性分析其动机。
