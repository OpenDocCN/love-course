# 102.熊浩哈佛：生活无处不谈判（：006.谈判开场：Purpose+

在本节课中，我们将学习谈判开场4P原则中的后两个要素：**目的（Purpose）** 与 **程序（Process）**。我们将探讨如何通过设定积极的愿景和掌控对话流程，为谈判创造有利的开端。

上一节我们介绍了谈判开场时关于“人”与“产品”的讨论。本节中，我们来看看如何通过设定“目的”和控制“程序”来进一步引导谈判。

## 一、 目的（Purpose）：设定共赢愿景 🌈

目的是指对谈判整体做出的、具有理想性和乐观色彩的预设。它与具体、现实的“产品”不同。

**目的** 与 **产品** 的差别在于：
*   **产品**：针对本次谈判，是具体的、现实主义的。
*   **目的**：针对谈判整体，是乐观的、具有理想性的。

例如：
*   “王总，我相信我们一定能够取得成功。”
*   “李老师，我相信我们今天一定能够完成让双方都满意的谈判。”

**为什么要在开场强调目的？**

社会心理学研究表明，**对场景性质的设定会有效影响人的行为策略**。

以下是一个经典实验：
1.  将大学生按性格分为“竞争型”和“合作型”两组。
2.  将两组人混合后，随机分为两个新小组。
3.  让两个小组分别完成名为 **“华尔街战斗（Wall Street Fight）”** 和 **“社区支援（Community Support）”** 的项目。**两个项目除名称外，内容完全一致**。
4.  结果发现：被分到“华尔街战斗”小组的人表现出更多竞争性；而被分到“社区支援”小组的人则表现出更多友善与合作。

这个实验说明，如果开场就将谈判设定为“双赢”、“社区建设”的场景，对方就更可能展现出友善与配合。反之，若设定为“剑拔弩张”的场域，谈判将举步维艰。

因此，建议在谈判开始时，主动、理想化地设定对话场域，引导所有参与者向合作方向调整。

## 二、 程序（Process）：掌控对话流程的隐性权力 ⚙️

程序是指谈判推进的流程与方法。控制程序是一种非常重要的隐性权力。许多人开场时说“咱们就聊一聊吧”，这等于将程序控制权拱手让人。

**程序控制如何在现实中应用？**

以下是应对常见谈判困扰——对方说话啰嗦——的两种方法对比：

*   **生硬方式（直接对抗）**：打断对方说“你不能这么讲，应该…”。这会伤害对方感情。
*   **程序控制方式（隐性干预）**：在对方说完后，通过总结和结构化其发言来施加影响。

**示例：**
对方完成一段陈述后，你可以说：
> “老李，你刚才讲的大概是三点意思。第一，在经济方面，你认为…；第二，在社会部分，你强调…；第三，在文化层面，你提出…。我概括得对吗？”

这不是指出错误，而是**通过结构化的示范，潜移默化地影响对方的说话习惯**，这是一种“润物无声”的干预。

另一个来自媒体界的例子：
*   **笨拙方法**：给说话啰嗦的嘉宾举提词板，这会令对方尴尬并产生敌意。
*   **聪明方法（程序干预）**：主持人主动总结：“王老师，您刚才表达了三点…，您是这意思吗？”（对方点头后）“那请您就这三点再清晰地讲一遍。”

这便是一种**结构性的权力介入**，通过对流程和结构的把控，实现对整体程序的有效控制，且不引发直接的人际对抗。

## 三、 综合应用示例 🧩

以下是一个将4P原则综合运用的真实案例：

某大型互联网公司（东道主）与一运动品牌谈判，但东道主未完全准备好，而对方高管已到。他们通过开场控制，争取了更多准备时间。

**开场对话解析：**
1.  **目的（Purpose）**：“王总您好，我相信我们一定能获得一个双赢、彼此满意的结果。”（渲染共赢愿景）
2.  **产品（Product）**：“今天第一次碰面，如果能把合作意向标定清楚，并诚实交换想法，我们就很满意了。未尽之事可以下次再聊。”（设定现实、具体的本次会谈目标）
3.  **人（People）**：“您公司是大企业，有些事可能需要向董事会报告。我们不妨先列个议题清单，从您能决定的部分开始谈。”（确认对方授权范围）
4.  **程序（Process）**：“列好清单后，我们不妨休息一小时，让我同事带您参观一下园区，这样您对我们的了解会更直观。”（控制流程，为己方争取准备时间）

对方很难拒绝这样的程序提议，因为它不涉及任何实体让步，只是对谈判流程的善意安排。这成功地为东道主争取了超过一小时的协调准备时间。

## 四、 核心要点与灵活运用 💡

谈判开场应涵盖四个要素：
*   **说人**：了解对方授权范围。
*   **说产品**：明确本次会谈的具体意义。
*   **说目的**：勾勒理想愿景，引导合作。
*   **说程序**：掌控谈判流程，获得隐性影响力。

**重要提示：务必灵活运用，切忌机械照搬。**

这四个工具并非每次谈判都必须全部、重复使用。例如，如果是第二次会面，已了解对方授权或已勾勒过愿景，则无需重复。应根据具体场景、与对手的关系，做弹性和有机的选择。

## 总结 📝

本节课我们一起学习了谈判开场4P原则中的 **目的（Purpose）** 与 **程序（Process）**。
*   **目的** 是通过设定乐观的共赢愿景，从心理上引导谈判走向合作。
*   **程序** 是通过掌控对话的流程与结构，获得隐性的影响力，从而更顺畅地引导谈判方向。

请记住这四个利等可取的工具：**人、产品、目的、程序**。明天就在你的谈判中用起来，并学会根据实际情况灵活处理。
