# 119.高情商秘籍：搞定复杂的人际关系【完结】：04.袋鼠型——喜欢照顾人的“老妈子”_note

在本节课中，我们将要学习四种关系模式中的“袋鼠型”。这种类型的人如同袋鼠妈妈，习惯将他人置于自己的“育儿袋”中照顾和保护。我们将详细解析其行为特征、形成原因，并提供调整建议。

## 袋鼠的生物学特征

上一节我们介绍了关系模式的整体框架，本节中我们来看看“袋鼠型”这一具体模式。首先，我们可以从真实的袋鼠身上找到其行为隐喻。

袋鼠最主要的特征是雌性袋鼠拥有一个**育儿袋**。幼崽在袋中被抚养长大。袋鼠妈妈将小袋鼠保护得很好，小袋鼠可以在育儿袋里贴着妈妈的身体，既温暖又能及时吃到奶。

当小袋鼠想探出脑袋探索世界时，袋鼠妈妈会把它的头按下去，自己带着它四处活动。

小袋鼠断奶离开育儿袋后，母袋鼠仍会活动在附近，随时给予帮助和保护。小袋鼠受到惊吓时，会立刻跳回育儿袋中。

## 人际关系中的“袋鼠型”特征

那么，袋鼠的这些特点对应到人际关系中，分别是什么样的情况呢？以下是袋鼠型人在关系中的核心表现。

首先，袋鼠型的人在人际关系中是**服务型**或**照顾型**的人。他们将照顾别人所获得的价值感作为自己的核心价值。**被他人需要，是他们的主要价值来源**。

如果要用一个形象去形容，女性袋鼠型人如同“大母神”，男性则像一个“弱弱的皇上”。他们是那个管理所有事情、让事务得以继续的人，感觉上有点像组长或村长。

他们的付出感很强。在付出的过程中，他们会觉得自己是照顾别人的“老妈子”。例如，在合作关系中，事无大小，他们全部亲力亲为。

通常与袋鼠型的人合作会有矛盾的感觉。一方面会很爽，因为很多事情他们都会包办；另一方面会很郁闷，因为时间长了，会发现袋鼠型的人在操办事情时会有所抱怨。

这其实是在合作中不信任他人的表现。因此，袋鼠型的人无法达到一种共赢的体验，会经常有不公平的感觉，并指责他人。

我们可以看到很多妈妈的袋鼠特质特别强。例如，她会做很多家务，忙里忙外，也会当着自己孩子的面做家务，并且边做边说自己多辛苦。那种很容易为别人操劳、让别人看到自己很忙碌的状态，同时嘴上说“你看我多辛苦多累，我没有功劳也有苦劳”的人，往往是袋鼠型的。

当他付出时，他是需要别人肯定的。

因为喜欢照顾别人，所以在关系中，袋鼠型的人最容易**弱化他人**。他会投射一种弱的形象给别人，潜台词是：别人都很虚弱，所以我才要把你保护起来，你才需要我的照顾。

袋鼠型是那种很有同情心的人。例如，一些会去收留流浪小动物的人，往往其内在就有袋鼠型的特质。

## 袋鼠型的控制欲与边界问题

上一部分我们介绍了袋鼠型乐于付出和弱化他人的特点，接下来我们看看由此衍生出的控制欲与边界问题。

袋鼠型的人的第二个特点是**控制欲会有点强**。试想，假如一只小袋鼠要出去看看世界，袋鼠妈妈就必须把它抓回来放进口袋里，因为她觉得外面的世界不好、不安全，所以“你必须要在我的身边”。

在这种情况下，导致袋鼠型的人在沟通模式中，经常很难听到和看到他人。他们常常会说：“我都是为你好啊。”这是他们去控制别人的一种最简单的方式。

他们希望别人能够围着自己转，非常希望自己所做的一切都被别人看见。如果说没有被看见，他会很失落、很难过，很容易把自己工具化，容易自怜。自怜就是：“为什么没有人来照顾我？为什么没有人来满足我？”

因此对袋鼠型的人，**虚空感**是最多的人生体验。如果袋鼠型的人跟一个人建立比较亲近的关系，当那个人要出去有自己的事情要做时，那一刻他会有一种虚空的体验。并且，如果在他的照顾或帮助下，身边的每一个人从弱变强了，那么在这个时候，他又会有一种虚空感。

因此往往在关系中间，如果看到自己比较亲近的人越来越强时，他不是一种满足，而是一种虚空，会有一种失落。

第三，在人际关系中，袋鼠型的人特别容易**侵入到他人的边界**，剥夺他人的很多机会。

所谓的剥夺他人的机会，就像小袋鼠想出去看看世界也会被袋鼠妈妈保护着。袋鼠型的人因为他是服务型的，而且相对来说安全感也不是很高，这就会导致弱化他人。那么在别人要去尝试一些新的东西的时候，他会先在旁边说：“不行，你最好不要。”这样就特别容易让别人失去一些探索的机会。

这也是袋鼠型的人另一个特征：**责任不清晰**。我们所说的责任，是彼此之间对于这件事情所承担的责任。袋鼠型的人容易过度承担，不光是承担对方的责任，同时他还渴望去承担他人的情绪。

所以在关系里，他会经常有一种挫败的感觉。这种挫败的感觉就是因为他没有办法承担别人的情绪时，他会感觉到很挫败。

所以依赖感很强。这种依赖感看起来好像是别人依赖他，实际上是他依赖别人。

## 袋鼠型在亲密关系中的表现

了解了袋鼠型在一般人际关系中的模式后，我们来看看它在更亲密的伴侣关系中是如何运作的。

袋鼠型的人在亲密关系中容易成为一种**父母式的伴侣**，或者让自己的伴侣**退行**。

退行的意思是：假如一个各方面都可以独立并且可以很好照顾自己的“女汉子”，遇到一个袋鼠型的男人，慢慢的就会退行到变成一个小孩子的状态，需要对方的照顾。而这是袋鼠型的人需要的。

因为在亲密关系里，他需要一种付出感，需要被他人需要。只有你退行了，你才需要他；要不然你很独立的话，就不需要他了。

在亲密关系中，袋鼠型的人很容易去找比自己弱的人作为伴侣。因此我们会发现很多袋鼠型的女性，她很欣赏或羡慕一些比较强的男性，但往往找一个丈夫，就是弱弱的那种感觉。

## 袋鼠型人格的形成原因

前面都是袋鼠型的人在人际关系中常见的模式，那么这种关系模式是如何形成的呢？

形成的原因有两种。

第一种是在原生家庭中，袋鼠型的人是被忽视的那一个。我们说往往贡献最大的，是最被忽略的。其实她是被父母忽视的那一个。最重要的不安全感在于**害怕被抛弃**。她对自己的核心评价就是：“我不够好。”不会提出自己的诉求。

另一种，袋鼠型的父母也会孕育出袋鼠型的孩子。这是孩子对父母的一种认同。就是这种父母会在关系中，一直无微不至地照顾自己的孩子，会传递给孩子一个信息：“不管你要不要，我给你的必须拿着。”当孩子认同这种养育方式，自己也会内化这种模式。

这两个原因会让孩子长大后成为袋鼠型的人，在关系中会失去自我。看上去考虑自己比较少，实际上是把别人当成实现自己需要的工具。

## 如何调整袋鼠型关系模式

当我们发现自己是袋鼠型的人，可以通过下面三个方面调整自己的关系模式。

**第一方面，要学会对自己的情绪负责，真诚直接地表达自己的诉求。**

袋鼠型的人很容易用愧疚感控制别人。那我们可以试着不带指责或者攻击，也不带让别人感觉到特别愧疚的方式来沟通。当你表达自己多么难过、多么悲伤的时候，在表达的时候聚焦于自己的感觉。

你可以告诉对方：“我很难过。”而不是告诉对方：“你让我很难过。”当然，说出自己的诉求，能够接受他人的帮助，实际上也是对他人的成全。这一点很重要。

**第二方面，要学会成全和肯定别人的价值。**

当你弱化了对方的时候，你只是希望对方离不开你而已。但这同时也让对方感觉不到自己的价值。把对方改变成离不开你的人，是因为你害怕分离或者害怕被抛弃。所以弱化别人的时候，你要意识到这一点。

当你把所有的事情都做了，其实你正在慢慢地挑战人性。就有点像是你的男朋友，如果说他不太会收拾家里，那么在这个时候，你不需要去把他的很多东西都收拾得很好。可以跟他一起收拾，并且在这个过程中肯定他的价值。

当合作完成一件事，或者我们一些诉求被满足时，我们需要向对方表达赞美和感激。这是成全对方，也是在承认对方的价值。诉求的背后其实藏着归属的愿望。在我们承认对方之后，对方会有一种归属的感觉，同时他也会觉得自己是有价值的。

**第三方面，看见他人是一个独立的个体，要保护他人，也保持我们一些小小的自私感。**

有时候自私一点恰恰是自我的一部分。他人拒绝你，或者是因为你的侵入已经伤害到了别人了。要意识到，实际上你很多的“为他好”的行为，恰恰是伤害或者侵入到他人边界的问题。

在关系中全身心付出的袋鼠型的人，会让别人变“废”，自己很累。所以袋鼠型的人恰恰是要让自己更自私一点点，更多的关注一下自己。假如你感觉到比较累的时候，不要去否认这种累，要去关照自己。比如说，拒绝一些不必要的邀约。

第二，如果你是袋鼠型的人的话，要学会对自己的情绪负责，成全及肯定别人的价值，更多关照自己一些。
