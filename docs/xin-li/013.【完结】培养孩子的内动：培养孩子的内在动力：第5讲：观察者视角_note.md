# 课程013：培养孩子的内在动力 🧠  
## 第五讲：观察者视角 👁️

在本节课中，我们将学习如何运用“观察者视角”来避免单边政策，从而更好地理解孩子的感受，保护他们的内在动力。我们将通过具体案例，掌握“切入对方感受”和“观察者视角”两个核心方法。

---

一位妈妈曾因日常小事痛哭，自责“见死不救”。事情很简单：她六岁的孩子与邻居小朋友玩耍时，跑来委屈地告状“她把我的东西弄坏了”。妈妈习惯性回答：“你们自己解决。”——这是她沿用多年的“标准答案”。

我反问这位妈妈：“孩子为此苦恼、求助了好几年，你的机械回答也持续了好几年。你可曾想过孩子的感受？”妈妈尝试代入孩子的角色：每次遇到纠纷来求助，得到的永远是“你们自己解决”。如果能自己解决，孩子何必反复求助？孩子别无他法，只能来碰运气，而结局总是一样。

那一刻妈妈才醒悟：自己对教育的刻板理解，让她听不见孩子的求救信号，让孩子多年处于无助中。她既心疼孩子，又自责不已。那么，究竟是什么阻碍了妈妈的视线？

### 单边政策的陷阱

上一节我们介绍了妈妈因固守己见而忽略孩子感受的情况。本节中，我们来看看这种“单边政策”的普遍性与危害。

妈妈的第一反应是依据育儿书原理——“孩子纠纷应自己解决”——而未评估孩子的实际困难。面对孩子反复求助，她感到烦躁，认为孩子解决问题能力差，并坚信“推出去”就能培养其能力。多年来，她沉浸于自己的主观世界，未接收孩子发出的客观信号。

这种“单边政策”在中国家长中十分常见。许多家长学习了深奥的教育理论，立志扬帆远航，最终却搁浅在沙滩上，连最基本的感受和事实都看不清。这种反差与上一代家长的教育方式密切相关。

上一代家长多以“讲道理”为主，而这正是典型的单边政策。讲道理无需看清感受或客观事实，无需接收孩子信号。家长从主观视角判断对错，便可滔滔不绝。例如，家长常嘱咐“好好学习，将来才有生存本领”，却从不探究孩子学习的真实感受或阻碍因素。一旦发现孩子不好好学习的苗头，家长便失望生气。

若家长的第一反应是论对错，还会在内心竖起防御高墙，其内心对话可概括为：

```text
“这孩子怎么这样？”
```

随之而来的强烈冲动是“教育”孩子——实则是较劲，试图将孩子掰成自己期待的模样。

例如，一位妈妈不满孩子吃饭挑食，认为“孩子事多，吃饭都不太平”。于是每天批评孩子：青菜吃得少、不尊重奶奶劳动、诉说今日生活来之不易，最后以非洲难民处境收尾。结果孩子不服，狡辩“宁愿生活在非洲”，气得妈妈吐血。可见，孩子也在较劲，双方能量浪费于冲突中。

这种做法最糟糕的后果是严重伤害孩子的“基础感觉”。日常生活鸡飞狗跳，频繁讲道理、忽视感受、胡乱介入，扭曲了孩子愿望，破坏其做事节奏，最终彻底摧毁孩子的内在动力。

### 迈向双边政策

上一节我们探讨了单边政策的危害。本节中，我们来看看如何戒除单边政策，转而建立“双边政策”。

若要培养或修复孩子的内在动力，家长首要是戒除单边政策。逻辑上，我们需重建双边政策，即：

```text
看见自己的感受 + 看见孩子的感受
了解自己的主张 + 理解孩子的决定
```

以前述吃饭挑食为例，我建议妈妈先将对错搁置，切入孩子感受：孩子放学归来，坐于餐桌前，怀揣小期待，却发现没有喜欢的菜。我们能想象那种淡淡失落。孩子自然会哀叹“不如饭店好吃”“不如同学家饭菜”。从孩子角度，有此感受与情绪并非错误。孩子仅唠叨几句，无过激行为，仍正常吃饭。

若家长不在内心念叨“这孩子怎么这样”，不点燃情绪，孩子其实表现正常。再从妈妈视角看：孩子不做饭却挑食，行为虽不算对，但不再觉得可恶，甚至有些可爱。因此，给孩子唠叨的空间，也不必为此改变全家菜谱。概括而言：

```text
你不爱吃，可以唠叨；我不生气，也不改菜谱。
```

妈妈实践后，不久家庭餐桌便恢复和谐。一次孩子看到卤鸡翅，问：“有没有加奥尔良烤鸡翅调料？”妈妈答没有。孩子说：“这样不好吃。”妈妈淡淡回应：“不喜欢就别吃。”孩子笑答：“不吃我就亏了，还是再吃一个吧。”全家大笑。

### 两个核心方法

这件小事的处理隐藏了双边政策的两个方法：**切入对方感受**与**观察者视角**。

以下是切入对方感受的具体步骤：

1.  **沿行为路线体会**：像演员一样，体会孩子做了什么、看见什么、感受到什么。
2.  **以吃饭为例**：从孩子放学开始感受——回家路上已有饥饿感；坐妈妈车里，望向窗外饭店（米线、煎饼、烤肉、火锅、肯德基），馋涎欲滴；满怀期待坐于餐桌前，却感失望。
3.  **跟随孩子**：看其所看，听其所听，想其所想，感受自然浮现。

妈妈学会此法后，感觉进入孩子感受不再困难。不久，孩子给了她实习机会：一天大雨，妈妈忘接孩子放学，直至老师来电才匆忙赶去。见面时，孩子满腹委屈，带哭腔数落妈妈“忘了接她”“一点都不爱她”。

以往妈妈会恼火：“天天接你这么辛苦，就一天忘了，还不依不饶。”但这次她迅速切换至孩子感受：从放学瞬间开始——刚放学时人声鼎沸，孩子以为妈妈很快来接；同学渐少，只剩四五人时，心里慌张；自我安慰“妈妈快来了”；直到只剩自己，心慌意乱，胡思乱想；加上大雨低温，心生悲凉。孩子本就胆小（妈妈停车取物时，她会躲车门角落“怕绑匪”），等待时必定难熬、害怕、又冷又饿。

妈妈感受到此，顿时理解孩子反应，甚至自责“过分”。随后回应温暖：立即道歉，自我批评：“哎，我心太大，差点把亲儿子丢了。”关心孩子是否冷，马上带其回家。孩子情绪迅速平复。妈妈感慨：“从前孩子挺可怜，我一直不能切入她的感受。明明值得理解的事，却常把她臭骂一顿。”

### 运用观察者视角

上一节我们学习了如何切入对方感受。本节中，我们来看看当行为路线不长或来不及慢慢感受时，如何运用“观察者视角”避免单边政策。

方法很简单：**用中立语言重新描述当下事情**。我们通过案例说明：

周末妈妈约另一对母子去游乐场并看电影。出门路上，对方来电称有急事需晚出门，建议将孩子送至她家，由她独自带两孩去玩。妈妈身体不适，正想送孩子后回家休息。但孩子一听立刻嚷着坚决不去。妈妈劝半天，买蛋挞安抚，总算送过去。

不久孩子自己跑回——原来在游乐场走散。对方家长慌张，欲通话安慰并劝孩子返商场继续看电影（票已买）。但孩子坚决不通话、不去看电影。

妈妈觉得诸事不顺：身体不适、孩子难沟通、对方愧疚，各种情绪纠缠令她心烦意乱。她想起“观察者视角”方法，不带情绪地从三人视角分别描述：

1.  **自己视角**：身体不适需休息；走丢可理解，不怪对方；商场离家近，不特别担心。
2.  **孩子视角**：不想去小朋友家、不想通话、不想看电影——她可做此选择。
3.  **对方家长视角**：很愧疚——需及时安抚沟通。

妈妈发现，当从三个方向以观察者视角看此事，仿佛看到三条清晰路线，内心不再烦躁。若重来一次，她会在路上少讲道理，只简单说：

```text
“妈妈今天不舒服，希望你能去小朋友家。你可以看书或看电视等他们。我给你买蛋挞。若你坚决不去，妈妈陪你去游乐场附近咖啡厅等——但我不舒服，不能陪你玩。”
```

孩子回家后，对通话和看电影，妈妈会简单征求意见，不勉强。然后与对方家长沟通：

```text
1. 走丢没关系，孩子大了、路熟，不必担心。
2. 孩子不去了，你们不必等。
```

妈妈表示，以往最怕情绪纷烦之事，但用此法后能快速剥离情绪、理清思路。

---

### 总结

本节课中，我们一起学习了如何运用“观察者视角”实现从单边政策到双边政策的转变。核心要点包括：

1.  **戒除单边政策**：避免仅凭主观判断教育孩子，忽略其感受与客观信号。
2.  **切入对方感受**：沿孩子行为路线体会其所做、所见、所感，以理解其情绪反应。
3.  **运用观察者视角**：用中立语言多角度描述事件，剥离情绪，清晰思路。

通过实践这些方法，家长能最大限度保护孩子的“基础感觉”与“对事感觉”，为培养或修复孩子的内在动力奠定坚实基础。
