# 谈判课102-027：人际关系之印象管理（上）👥

在本节课中，我们将要学习谈判七大核心要素中的最后一个——人际关系要素。我们将聚焦于如何通过有效的“印象管理”，在对方心中留下良好印象，从而为谈判创造有利条件。

上一节我们介绍了谈判的七大构成要素，本节中我们来看看其中的人际关系要素。人际关系要素主要从两个角度讨论：一是印象管理，二是冲突转化。今天，我们先深入探讨印象管理的第一个部分：对方为何会对我们产生好印象。

概括而言，有三个关键因素会影响他人对我们的印象。以下是这三个因素的详细解析。

## 1. 感谢与恭维 (Appreciation)

第一个因素是 **Appreciation**，即感谢或恭维。社会科学研究发现，在商业情境中，**三个好印象可以抵消一个坏印象**。例如，一个人外表得体、彬彬有礼，但若被发现有不雅举止，其之前积累的好印象可以起到缓冲作用。在家庭情境中，这种平衡更具挑战性，往往需要**五个好印象才能抵消一个坏印象**。

表达感谢是最简单、成本最低的获取好印象的方式。在日常生活中，应对他人的善意或付出表示诚挚谢意。观察成熟政治人物的公众演讲，他们通常会细致地感谢每一位重要来宾和全体受众。例如，美国前国务卿希拉里·克林顿在一次访问中，详细感谢了外交部长、大学校长、师生、媒体人员及相关组织。这种细致的感谢体现了基本教养，也是赢得好感的直接理由。

## 2. 自治与共控 (Autonomy)

第二个因素是 **Autonomy**，即自治或共同控制。在对话中，有些人像紧握方向盘的司机，害怕与他人分享话语权，导致沟通僵化。例如，领导在听取汇报后，若只是生硬地复述“你说的是第一、第二、第三点”，便没有给下属留下补充或纠正的空间。

掌握自治原则的话语策略则不同。领导可以这样说：“小王，你讲得很详细，我可能记不全。好在你还在，如果我概括有误，你可以随时纠正。你讲的大概是这三点……是这样吗？”这种“我理解的不一定对”的表述，就是 **Autonomy**。它让对方感觉到，在对话这辆“汽车”前进的过程中，自己也是参与者，也能在一定程度上控制“方向盘”。在访谈、总结或会话中，为对方留下介入和反馈的缝隙，能有效避免让对方感到彻底失控。

## 3. 寻求相似 (Affiliation)

第三个因素是 **Affiliation**，即人与人的相似性。相似性是产生好感的另一个重要原因。中文里大量以“同”字开头的词汇，如同学、同事、同志等，都体现了群体主义文化中，通过寻找共同点来建立社交连接的基本习惯。

想象在一个大型聚会中，来自各行各业的人需要寒暄相识。人们通常会首先寻找与自己近似的人，例如老乡、校友或同行。这是快速与人熟识的简单策略。其背后机理，正是通过 **Affiliation** 来引发好感。

---

本节课中我们一起学习了印象管理的三个核心策略，它们英文首字母都是A，因此也被称为 **3A法则**：
1.  **Appreciation**（感谢/恭维）
2.  **Autonomy**（自治/共控）
3.  **Affiliation**（相似）

这三个策略都能帮助你在互动中引发对方的真实好感。关于“寻求相似”在商业谈判等情境中的具体应用案例，我们将在下一次课中做详细讨论。
