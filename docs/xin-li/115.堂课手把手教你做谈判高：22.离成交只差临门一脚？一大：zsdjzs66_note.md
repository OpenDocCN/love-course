在本节课中，我们将学习谈判收尾阶段的关键智慧。许多交易在最后关头失败，往往是因为忽略了一些核心细节。我们将深入探讨如何战略性让步、运用解题模型，并掌握有效的收尾技巧，确保谈判顺利达成协议。

## 谈判的智慧：如何与何时让步

上一节我们介绍了谈判的基本技巧，本节中我们来看看谈判中的智慧——**hwain**。技巧（how）是方法，而智慧（hwain）是懂得何时及如何运用这些方法。谈判时，你需要抓住几个核心：**如何让、如何锁、让给谁、什么时候让**。

关于“如何让”，我们已经学过幅度、次数和速度。“让给谁”则意味着你的让步要给对人，并给足对方面子。

以下是关于“让给谁”的两个实例：

*   **实例一**：一位讲师在马来西亚向一家公司推介课程。总裁认可课程，报价6万马币，并让HR部门负责洽谈。讲师在与HR谈判时，主动让利1万，以5万成交。HR因此立功，总裁也满意。讲师解释，这1万是特意“让给”HR的，旨在培养一个友好的对接窗口。
*   **实例二**：一家电脑公司的销售员，每次与客户谈妥条件后，对方的副总总会出面再砍价3%。无论销售如何陈述产品优势，副总只坚持要这3%。后来，销售团队学会了在报价中预先留出这3%的余地，专门“让给”副总。因为这已成为一个固定的“出场仪式”，不给对方这个面子，后续合作会很难进行。

## 把握时机：战略性让步

那么，**什么时候让**步呢？这是谈判中最难的部分，需要智慧。

谈判中有一招叫做 **“以战略性的让步改变形势”** 。这意味着，作为一个理性的谈判者，你预计打出某张牌后，对方会有特定反应（如惊慌、愤怒），并为此预先准备好应对方案。

但如果你上桌后发现对方的反应完全出乎意料，你的第一张“骨牌”被推倒，整个预设的应对链就会崩溃，主动权易手。因此，在谈判中要审视：是否有哪张关键的“牌”压在桌上，对方赌你不敢打出去？如果你敢打出去，就可能彻底改变谈判形势。

思考时一定要多想“万一”：提出对方**肯定会拒绝**的条件，万一他没拒绝呢？提出对方**肯定不会拒绝**的条件，万一他拒绝了呢？这些“意外”都可能成为扭转局面的契机。这种策略既可用于开局，也可用于收尾。

## 收尾的艺术：凸显最终方案

关于收尾时的让步，我们来看一个例子。美国一个工会罢工要求加薪。老板先加了5元，未果；又加2元，仍未解决。老板请教谈判专家，他本计划再分次小额追加（如1元、2元）。专家建议：**“5、2”已经让出，后面要“撑一下”（坚持不让），最后再一次性让出一个较大的“3”**。

策略是：**5, 2, 0, 0, 3**。
这样，最后的“3”作为一个足够大的尾巴，能凸显这是**最终方案（final offer）**。在对方气势渐弱时，给出一个较大的让步，然后立刻“打结收住”，强调这是底线。

有人会问，这不是违反了之前“让步幅度递减（D减）”的原则吗？请注意，这是**微调**。虽然最后的“3”比之前的“2”大，但它仍比最初的“5”小，整体趋势仍是递减的（如5, 2, 0, 0, 3）。关键是避免出现像“5, 2, 0, 0, 8”这样幅度回升的情况。

## 解题模型：打破僵局的四种思维

当谈判陷入僵局，如同“分大饼”时，需要运用解题模型来创造价值。以下是四种核心方法：

1.  **增加资源法**：把“饼”做大，这样大家都能分得更多。
2.  **交集法**：寻找双方利益的共同点（交集）。
3.  **挂钩法**：将不同议题联系起来，进行交换。
4.  **切割法**：将一个整体议题切割成更小的部分，细分能创造更大的交换空间。

对于复杂的谈判，掌握这四种方法至关重要。

## 切割法的实际应用

切割法在谈判中如何运用？我们通过一个案例来说明。

一家药厂向医院推销药品，遇到一位很实在的医生。医生问：“**一定要这个价钱才能卖吗？**”
如果回答“是”，医生会说“买不起”；如果回答“不是”，那报价就显得没有诚意。

关键在于，**价格从来不是孤立存在的**。它总是与规格、数量、付款方式、交货条件等**配套组合（package）**在一起。因此，标准答案是：“**如果交货条件、付款方式、数量或规格不同，那么价格当然可以不一样。**”

谈判时，桌上的议题永远是一个**组合（combination）**。要让步，可以，但必须在**这个组合范围内**谈。这叫做设置“**Yes Button**”（同意按钮）。

**核心公式**：`价格 = f(规格, 数量, 付款方式, 交货条件...)`

不会谈判的人只盯着单一价格（100元、80元、60元）纠缠。会谈判的人懂得，目标是 **“赢者不全赢，输者不全输”**。

*   如果你是弱势卖方，无法压低价格，可以尝试在**数量或付款方式**上要求买方补偿。
*   如果你是弱势买方，无法压低价格，可以尝试在**付款周期或交货次数**上要求卖方让步，以降低你的库存成本。

这本质上是一种**代数换牌**思维：如果我在A项上让步，就要在B或C项上得到补偿。

例如，宝洁（P&G）将帮宝适（Pampers）纸尿布卖给大卖场。不同卖场的零售价不同，并非因为单包价格不同，而是因为**进场的组合（package）不同**。组合可能包括：
*   帮宝适与玉兰油（Oil of Olay）产品捆绑。
*   提前付款享受的折扣（Early Payment Discount）。
*   新店开张或周年庆的活动配合费用。
*   大宗订单带来的物流成本节省。

大卖场的采购谈的不是单包价格，而是整个**一揽子方案（package）**，不同的组合决定了最终的单品成本。这是谈判的基本思维。

以下是切割法在谈判中的常见交换句式：

*   “我们预算只有这些，但我们可以接受更长的交货期。”
*   “我们可以做到这个价格，不过需要您调整一下设计以配合我们的生产线。”（规格换价格）
*   “如果预付款比例或支票效期能改变，也许有可能。”（付款方式换价格）

## 综合运用：解决“山与海”的争端

假设公司组织旅行，一部分人想去山里，一部分人想去海边，互不相让。作为组织者，你如何运用解题模型？

*   **增加资源法**：请求老板延长假期，让大家既能去山也能去海。但可能被老板直接否决。
*   **交集法**：深入分析“山”与“海”背后的**利益**。去山是为了森林浴和吃山产，去海是为了玩水和吃海鲜。那么，找一个能同时满足“森林浴、山产、玩水、鱼虾”的地方——比如**湖边**。这就是找到了双方利益的交集。
*   **挂钩法**：与想去海边的同事协商：“**答应这次去山里，下次活动的经费我们部门多出一些。**” 将“目的地”与“经费”两个议题挂钩。
*   **切割法**：将“度假”这个整体切割成“目的地”、“住宿”、“交通”、“经费”等多个部分。然后进行交换：“**这次目的地听我们的（去山），但住宿标准按你们想要的（五星酒店）。**” 或者“**交通方式听我们的，经费分摊可以照顾你们。**” 通过切割与交换，实现共赢。

当年英国与北爱尔的谈判，也运用了类似的交集法。可见这些方法威力巨大。

## 最终推动：期限与预警

谈判到最后，有时像火车陷入沙滩，需要推力才能前进。这个推力就是**期限**和**预警**。

*   **期限**：给出一个最终时间限制。
*   **预警**：这是比威胁更高级的技巧。
    *   **威胁**是：“如果你不签，我就惩罚你。” 惩罚源自说话者自身。这会使自己陷入“到底执不执行惩罚”的困境。
    *   **预警**是：“有件事（如市场变化、政策调整）可能会发生。如果发生，或者如果我们未能达成协议，我们双方都将被迫承受损失，没有赢家。为了避免这种双输局面，我们是否应该现在达成协议？” 惩罚源自**第三方或客观形势**。

**预警**的核心是**增加对方不与你达成协议的成本**，同时自己可以解套（因为你不需保证“预警”的事一定发生）。而**威胁**则容易让自己骑虎难下。

## 核心总结：让对手“敢输”

在本节课中，我们一起学习了谈判收尾的关键智慧。

谈判的最终推动力，在于**一推一拉**：
*   **拉**：创造诱因和期待，让对方觉得“跟我谈，能得到什么”。
*   **推**：运用期限和预警，让对方明白“不跟我谈，会损失什么”。

在谈判过程中，要记得定期**小结**，明确已解决的议题，展示进展，增强双方信心。务必运用**切割法**等解题模型，追求**赢者不全赢，输者不全输**的结果。

最后，一个至关重要的概念是：**减少对方让步的成本**。当对方让步时，他担心损失什么？你需要顺着逻辑为他考虑并消除顾虑：
*   **里子**：实质利益损失。
*   **面子**：给予足够的尊重和台阶。
*   **时间**：给予考虑时间。
*   **先例**：很多人不敢让步，是怕“此例一开，后患无穷”。你要让对方相信，这次让步只是一个**特例**，不会成为未来谈判的**先例**。反之亦然，当你自己让步时，也要强调这是特例。

**谈判之道，无他，就是让对方敢上桌、敢下桌，并且“敢输”给你。** 思考并运用这些原则，你就能更好地把握谈判的临门一脚，避免功亏一篑。
