# 职场人际关系学 第十二讲：与对手握手言和的策略：神奇的富兰克林效应 🫱🏻‍🫲🏼

在本节课中，我们将要学习一种被称为“富兰克林效应”的心理策略。这一策略的核心在于，通过巧妙地请求帮助，可以有效地改善甚至扭转与对手或关系疏远者的关系，从而拉近彼此距离，化敌为友。

---

许多人都有不愿麻烦他人的习惯，认为这不太合适。性格内向者更容易形成难以突破的心理障碍，害怕求人。这会导致人际交流减少，关系逐渐淡化。

没有良好的人际关系，事业便缺乏根基。这种不愿求助的心态会导致两个主要问题。

以下是第一个问题：

*   **导致互动减少，关系生疏**：人际关系是在互动中建立的。互动越少，关系就越疏远。你越不愿意麻烦别人，对方可能越觉得你与他关系不亲密，这会放大彼此间的距离。

人与人之间的交往，关键在于彼此的距离。拉近距离的方法，必然包含“他麻烦你”和“你麻烦他”的相互往来。因此，在麻烦别人时，不必感到不好意思或亏欠。回想一下，对方过去也可能曾麻烦过你。

要改善关系，就必须突破这种心理障碍。将求助视为相互的互动与成就，而非占人便宜。突破之后，至少能让你与他人保持熟悉的联系，拉近彼此的距离。

上一节我们探讨了求助对维持普通关系的重要性，本节中我们来看看如何利用这一原理来打破与对手的僵局。

与关系疏远的普通人不同，对手往往与我们立场敌对。此时，如何打破僵局呢？答案就是“富兰克林法则”。

那么，到底什么是富兰克林法则呢？

本杰明·富兰克林曾有一位政敌，总是反对他。富兰克林说东，政敌就说西。一般人可能会选择打击报复，但富兰克林没有。他听说这位政敌有一本稀有的好书，便主动前去借阅。出乎意料的是，政敌把书借给了他。这次借书事件后，两人关系逐渐缓和，最终化敌为友。

这个故事揭示了一个深刻道理：让别人认同你的最好方法，不是无休止地说服或争论，也不是单方面地去帮助他，而是**让他来帮助你**。这会让对方感到自己被信赖、有价值，从而在心理上与你站在同一阵营。敌人也就可能转变为战友。

这里引出一个关键结论：**曾经帮过你的人，比你所帮助过的人更可靠。**

一个著名的二战故事印证了这一点。两位犹太兄弟在逃亡时产生分歧：弟弟想向父亲曾资助过的银行家求助，哥哥则主张向曾多次帮助过他们家族的木材商求助。两人分头行动。最终，哥哥在木材商冒死相助下幸存，而弟弟全家却被银行家举报，惨死于集中营。哥哥因此慨叹：曾经帮助过你的人，确实比你帮助过的人更可靠。

这就是“富兰克林效应”的深层逻辑：当你帮助一个人时，哪怕原本不喜欢他，你的大脑也会自动调整认知，告诉自己“对方是值得帮助的人”，你做了一件正确甚至高尚的事。这会产生心理上的路径依赖，使你更倾向于再次帮助他、更喜欢他。

了解了原理之后，我们在实际工作与生活中应如何运用呢？

以下是三个具体的应用方法：

1.  **不要吝于或恐惧求人**：偶尔向身边人求助（不一定是借钱），可以加深彼此的关系和感情，对方可能会因此越来越喜欢你。
2.  **建立信任惯性**：通过请求一些小的帮助（如借一小笔钱并迅速归还），让对方形成“帮助过你”的惯性。当你未来遇到更大困难求助时，他可能更愿意伸出援手。
3.  **筛选高价值朋友**：通过请朋友帮一些小忙，可以在这个过程中观察和分辨出谁是真正值得信赖、愿意提供支持的高价值朋友。

本节课中，我们一起学习了“富兰克林效应”。我们了解到，适当地请求帮助不仅能拉近普通关系，更是与对手握手言和的巧妙心理策略。关键在于突破“不愿麻烦人”的心理障碍，理解“相互麻烦”是人际互动的纽带，并掌握“让对手帮助你”以化敌为友的智慧。记住，**曾经帮过你的人，往往比你所帮助过的人更愿意再次支持你**。
