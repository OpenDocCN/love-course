# 谈判高手课 115：谈判桌上有暗号与水平交换策略 🔍🤝

在本节课中，我们将学习谈判队伍的组织结构、谈判桌上的“暗号”沟通方式，以及如何运用“水平谈判”策略来有效交换条件。掌握这些技巧，能帮助你在复杂的谈判中保持主动，并最大化己方利益。

## 谈判队伍的结构

上一节我们介绍了谈判前的准备工作，本节中我们来看看谈判队伍应如何组织。谈判队伍绝不是成员平起平坐、七嘴八舌的民主结构。

谈判桌上必须有一个“头”，形成**金字塔**式的决策结构。队伍中的其他人可以讨论，但最终由“头”来做决定。因此，公司派出的谈判代表中必须明确负责人。

## 观察者的角色与“暗号”沟通

除了负责人，队伍中还需要一个“观察者”。因为主谈者在发言时往往没有时间深入思考，观察者的角色就是冷静地观察全局。

以下是观察者的核心任务：
*   观察对方人员的身份、举止和异常反应（例如，一个穿黑衣服的人坐下引起骚动）。
*   评估谈判进程是否偏离预定计划（例如，十个问题问完却毫无进展）。
*   判断是否需要因外部因素（如总公司新命令）改变战略。
*   提醒并制止己方队员的不当言行。
*   在发现形势不利时，提议暂停谈判。

当谈判队伍人数较多时，观察者可能不希望暴露身份，也不一定坐在主谈者旁边。此时，就需要一套不引人注目的“暗号”来传递信息。

暗号不是传纸条（太明显），也不是摸头发、摸鼻子（紧张时对方可能注意不到），更不是“嗯嗯”的明示。真正的暗号需要是隐蔽的动作。

例如，观察者可以带一个本子上桌。当他发现形势不对时，就**撕掉一页纸，折起来放进口袋**。撕纸的声音和折纸的动作，就是给主谈者的暗号。主谈者察觉后，会过一阵子（而非立即）叫停谈判，以免暴露。

暂停后，观察者汇报所见，团队讨论，最后由负责人做出决定并重新投入谈判。负责人需明确表态“责任我担”，以维护团队士气。

需要强调的是，这里教授的是“谈判桌上有暗号”这一**观念**，而非鼓励大家都去撕本子。因为如果这套方法流传过广，反而会失效。如果你实在想不出暗号，用“撕本子”这招也可以，其心理作用在于：如果对方没学过，这是暗号；如果对方学过，则知道“你也是学过的”，在心理上能扯平。

## 利用对方技术人员的破绽

谈判桌上如果对方有技术人员在场，经验丰富的谈判者会**盯着技术人员找破绽**。

因为技术人员常常“知道太多，但敏感度太少”，即缺乏谈判的警觉性。他们可能在不经意间泄露关键信息。例如，采购谈判时，如果带实际使用产品的人员（user）同去，他可能会因为急切需求而说出“你必须交货，不然我就完蛋了”之类的话，这等于将己方的底牌和退路全盘托出。

## 水平谈判与条件交换

在准备好人员和策略后，我们还需要决定如何“出牌”，即谈判议题的推进方式。

谈判有**垂直谈判**和**水平谈判**两种基本方式。

*   **垂直谈判**：按顺序逐一讨论议题，谈完一个，确认一个，再进入下一个。
    *   **公式**：议题1 → 确认 → 议题2 → 确认 → 议题3 ...
*   **水平谈判**：将多个议题捆绑在一起，进行交叉讨论和条件交换。
    *   **公式**：议题1 + 议题3 + 议题4 → 打包谈判与交换

垂直谈判的问题是，当谈到后面议题想用前面议题的让步作为交换筹码时，对方可能会以“时间过去太久”、“有权不用过期作废”等理由拒绝。而且，早先让步的筹码价值可能随时间衰减，无法发挥100%的效用。

因此，有效的交换往往需要通过水平谈判来实现。这与“挂钩”策略类似，但挂钩需要额外建立联系，而水平谈判是直接将桌上已有的多个议题进行分组交换。

## 如何实施水平交换

如果你要和对方谈多个议题（例如5个），不必接受对方设定的谈判顺序。顺序本身是控制谈判节奏的工具。

你可以要求改变顺序，将有关联的议题放在一起谈。例如，将第1个和第4个议题拿到前面一起讨论。

> **举例**：在二手房买卖谈判中，付款方式和交钥匙/允许装修师傅提前进场的时间，就应该放在一起谈。如果先谈妥了对你有利的付款方式，再单独要求提前进场装修，对方很可能拒绝。而你会后悔：“早知如此，当初就不该答应那种付款方式。”这就是没有将关联议题挂钩谈判的结果。

当议题非常多时（例如20个），可以采取**分组**策略。将议题按性质分成若干小组，在小组内部或小组之间进行交换。

> **举例**：在招聘谈判中，薪资范围、福利待遇、入职日期等都是议题。即使先谈了薪资范围（如2万-3万），在后续谈到丰厚的福利待遇时，仍然可以回头重新调整薪资范围（如“福利要这么多，那薪资最高只能给2万”）。只要议题没有最终完全敲定，就可以在关联议题间进行回溯和交换。

## 谈判议题的出场顺序

在决定采用垂直还是水平谈判，以及如何交换后，还需要规划议题的出场顺序。

一个通用的原则是：**简单的议题先谈**。这能让谈判更容易上手，营造良好的开局氛围。将困难的议题放在后面。

当然，也存在特例。例如，如果你想为扮演“白脸”的队友搭建舞台，可能会先抛出难题。但这属于更高阶的技巧。在正常情况下，遵循“先易后难”的原则有助于控制谈判步骤。
