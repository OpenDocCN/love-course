# 118.【完】超级识人术：09、读物——读出对方的实际品位_note

在本节课中，我们将学习如何通过一个人阅读的书籍和报刊类型，来洞察其性格特点、情绪状态与内在品位。阅读偏好是内心世界的一面镜子，能够清晰地反映一个人的个性与追求。

## 概述：书籍是精神的伙伴

人不会总停留在寻求物质的阶段。当物质丰厚时，精神家园需要打理。当一个人尝试过一切感官刺激后，便开始寻求灵魂的皈依。欲望的枝枝蔓蔓，需要靠读书这把剪刀来修剪。

苏东坡曾写诗提及读书的好处：“读书洗剑面，未影称斧声。”读书如同饮下美酒，暖意融融。有人说，世界有十分美丽，但若没有女人，将失掉七分色彩；女人有十分美丽，但若远离书籍，将失掉七分内蕴。**“富有诗书气自华”** 是人人都明白的道理。

书是修炼魅力之路上最值得信赖的伙伴。依靠它，你将不再畏惧年龄，也不会因琐事而长久苦恼。因为你已拥有独特的心灵和丰富的情感体验，你的生活将书香四溢。

法国哲人笛卡尔说过：“读一切好书，就是和许多高尚的人说话。”读书可以使人得到优雅和风味。坚持执行读书计划，你会有所成就。读书不仅是谋生进步的手段，更是一种生存的需要和创造性的工作与生活。

养成终身读书的习惯，是成就一生的源泉和动力，也是参与未来竞争的根本。正如精雍在浙江大学提出的“四个姿势”之一：**终身读书**。生活没有书籍，就好像没有阳光；人生没有书籍，就好像鸟儿没有翅膀。在漫漫人生路上，书是一个人精神上的朋友。

## 阅读偏好与性格解析

读书与人的性格有着密不可分的内在联系。因此，我们可以通过阅读喜好来了解一个人的个性特点及情绪变化。

以下是不同阅读偏好对应的性格分析：

*   **喜欢阅读各类图书报刊的人**
    通常具有开朗的性格特征。

*   **喜欢阅读浪漫小说的人**
    感情较为丰富充实，对直觉深信不疑，感到生活充满阳光。在陷于困境或面临失败时，能够顽强抗争。

*   **喜欢看传记题材书籍的人**
    必定是一个深思熟虑的人。既有雄心壮志，又会脚踏实地做事。谦虚好问是最大特点，做出决定前会思考再三，从不轻率冒险。

*   **对小报类报纸爱不释手的人**
    性格较为乐观快活，感情外露，喜欢和人聊天并以此为乐。在谈话中常成为中心发言人。

*   **很欣赏戏剧性书籍的人**
    忧愁、痛苦和烦恼与他无缘。他是一个乐天派，笑口常开，风度潇洒，性格风趣，任何困难都不会使他为难。

*   **常读报纸和新闻性刊物的人**
    喜欢关注国内外大事，眼观世界风云变幻，使自己的言行跟上时代步伐。思维敏捷，对新事物能迅速反应。

*   **对画报或大型画册兴趣盎然的人**
    必定热情好客，爱结交朋友。是称职出色的主人，能使客人乘兴而来、满意而归。

*   **喜欢拜读《圣经》的人**
    可以断定他为人诚实，手脚勤快，尊敬智者与知识。在处理人际关系时，宽厚待人、严于律己、与人为善是重要原则。

*   **爱看侦探或破案类书籍的人**
    动脑筋和解难题是这类人的一大特点。在旁人看来束手无策的难题，他们能轻松解开，并对挑战性难题兴趣倍增。

*   **对科幻类图书如获至宝的人**
    思维发达，想象力丰富，创造性强。总想将工作完成得更出色，把自己的一生变得更美好。

*   **经常阅读妇女报刊的人**
    希望自己成为女性中的佼佼者。事业上富有进取心，工作一丝不苟、谨慎行事，不愿因过失造成遗憾。

*   **喜欢阅读财经经济类书报杂志的人**
    是一个懂得自尊自重的人。崇拜事业上有建树的人物，并以其为榜样，希望充分发挥自己的竞争力。

*   **对广为流行的时尚杂志格外青睐的人**
    很有能力，较多关注自己的身份和地位。有时甚至脱离实际地拔高自己，导致陷入尴尬。

*   **特别喜欢读诗歌的人**
    一定热爱生活，对人世间的一切钟爱有加。在大自然的熏陶下进行思索，使心灵得到净化，追求真善美。

*   **读历史书籍津津有味的人**
    尊重事实，讲究实际，重视效果。时间都花在努力工作或认真做学问上，不会参与无意义的闲聊，也不与无所事事的人来往。

*   **醉心于恐怖故事的人**
    或许对生活感到厌倦、心情压抑；或许在激烈的竞争压力面前无所适从；或许面对屡次失败不堪重负。为了摆脱内心空虚焦虑，只好用刺激寻求解脱，但效果往往并不理想。

## 总结：以书识人，择善而交

本节课中，我们一起学习了如何通过“读物”这一维度来识人。读物的喜好能让我们从性格的一角，对一个人的内心有所了解。

孔子曾说：“**无友不如己者**。”意思是，交朋友要交在某一方面优秀的人。通过阅读偏好等多方面考察对方的品格，才能对自己有更好的交代。

通过观察一个人的书架和阅读清单，我们便能对其品位、性格乃至当前心境有一个直观而深入的判断。这便是在社交中“知人知面更知心”的又一实用技巧。
