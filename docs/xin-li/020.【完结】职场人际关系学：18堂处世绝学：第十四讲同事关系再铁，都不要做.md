# 020.【完结】职场人际关系学：18堂处世绝学：第十四讲同事关系再铁，都不要做

在本节课中，我们将要学习职场中与同事相处的关键禁忌。无论你们的关系多么“铁”，有四种配合行为是绝对不能做的，否则很可能给自己带来严重的麻烦，甚至被“清算”。我们将逐一剖析这些行为背后的逻辑与风险。

## 认识职场同事关系的本质

上一节我们介绍了课程的整体框架，本节中我们来看看职场同事关系的独特性质。

职场中的同事关系是一种“三不像”的关系。它**既不是纯粹的朋友关系**，中间夹杂着竞争；也**不是纯粹的竞争关系**，在没有直接竞争时，彼此又可能像朋友一样相处。这种复杂多变的关系，决定了你与同事相处时，必须采取**完全不同于与朋友相处的方式**。

职场高手行事通常非常谨慎，原因在于害怕给未来的竞争对手留下把柄。一旦未来双方成为对手，共同竞争一个职位，这些把柄就可能成为对方对你进行致命打击的武器。因此，职场中流传着“害人之心不可有，防人之心不可无”的准则。然而，仅仅做到小心谨慎还远远不够。

## 坚决不能做的四种“配合”行为

以下是四种你必须坚决拒绝配合同事去做的行为。这不仅是对自己的保护，防止陷入恶性竞争，也是为了避免被他人利用，甚至因此背负骂名。

### 1. 共同排挤某人 🤼‍♂️

第一种不能配合的事情，是共同去排挤某位同事。

你今天和同事一起排挤别人，明天他们也可能联合起来排挤你。这无异于**搬起石头砸自己的脚**。如果有同事向你提议：“我们一起配合，排挤某某人”，或者“我们联合起来拒绝他的要求”，你必须果断拒绝。

如果你同意了这种配合，会给自己埋下巨大的隐患。提议者可能很有心机，只是在利用你达成排挤他人的目的，以便他自己上位。这对你没有任何好处，最终责任很可能被推到你头上，让你百口莫辩。因此，千万不要参与这种**损人不利己**的事情，一定要主动回避。

### 2. 联名上书 📜

第二种绝对不能配合的事情，是联名上书。

这种行为等同于向领导宣战，其结果往往只有一个：你和参与的同事一起离开。让我分享一个真实的案例：我曾担任总监时，公司有四位经理联名上书，要求开除另一位证据确凿、犯了错的物流部经理。虽然公司从制度上应该开除该经理，但他是一位股东的亲属，是股东在公司内的代言人。在股东求情后，公司决定严肃处理但不辞退。

这四位与物流部经理矛盾尖锐的经理不干了，他们联名威胁公司：“如果不开除他，我们四个就集体辞职。”总裁后来征询我的意见，我当时的回答是：“把他们四个都辞退。”我的理由是：**物流经理犯错如何处置，是公司的决定权。但任何组织都不能接受员工的威胁**。今天因为他们联名就妥协，明天就可能出现更严重的威胁行为。

这个道理类似于处理劫持人质的恐怖分子：**妥协一次，就会招致无数次更严重的威胁**。在职场中，无论遭遇何种不公，都应通过正规渠道、私下沟通的方式提出，绝不能使用“联名上书”这种威胁组织的方式。凡是参与威胁组织的人，在公司都不会有前途。

### 3. 被推举为“帮主”或“民意代表” 👑

第三种不能配合的事情，是允许同事将你推举为小团体的“帮主”或“民意代表”，代表他们去向上级争取利益。

这等于**把你架在火上烤**。《三国演义》中有一个典故：孙权杀了关羽后，心中害怕，便上书劝曹操称帝，意图让曹刘开战，自己坐收渔利。曹操看穿后大笑说：“孙权这是想把我放在火上烤啊！”

在组织里，切忌被别人当枪使。你为别人争取了利益，最后可能利了他人，却损害了自己。职场中最忌讳的，就是把自己推到风口浪尖。这不但让你毫无保留地暴露自己，还将自己置于众矢之的。职场中难免有小圈子，但为人一定要低调，不要去做这种出头椽子先烂的事情。

### 4. 配合进行违规或灰色操作 ⚠️

第四种不能配合的事情，是配合同事进行任何违反公司规定或游走在规则边缘的“灰色操作”。

这种行为往往以“行个方便”、“大家都这样”、“不会有人知道”为借口。但一旦出事，责任很可能由你来承担。同事可能会推卸责任，声称是你的主意或由你主要执行。这不仅会导致纪律处分，更会彻底摧毁你的职业信誉。

保护自己的原则很简单：**一切操作必须合规、留有记录、经得起检查**。不要为了所谓的“哥们义气”或短期方便，而配合他人进行违规操作。

## 总结与核心要点

本节课中，我们一起学习了职场中四种绝不能配合同事去做的行为：

1.  **共同排挤他人**：这是损人不利己的短视行为，会反噬自身。
2.  **联名上书**：这是威胁组织的行为，挑战管理权威，必然遭到坚决反击。
3.  **充当“民意代表”**：这是被架在火上烤，成为别人的挡箭牌和牺牲品。
4.  **配合违规操作**：这是将职业生命置于巨大风险之中，得不偿失。

记住，职场关系的核心公式是：**同事 = 合作者 + 潜在竞争者**。处理这类复杂关系时，必须恪守底线，保持清醒。你的首要任务是**保护自己的职业生涯**，在此基础上寻求良性合作。做到**谨慎参与、合规行事、保持距离、坚守原则**，方能在职场的人际漩涡中稳步前行。
