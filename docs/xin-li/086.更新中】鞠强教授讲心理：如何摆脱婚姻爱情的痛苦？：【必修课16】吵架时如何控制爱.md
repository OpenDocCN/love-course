# 心理学课程086：如何摆脱婚姻爱情的痛苦？【必修课16】吵架时如何控制爱人的脾气？（下）💑

在本节课中，我们将继续学习在亲密关系冲突中控制对方情绪的有效方法。上一节我们介绍了通过皮肤接触来安抚情绪的技巧，本节中我们来看看另一种简单实用的沟通策略。

## 核心原则：情绪优先于道理

两人一旦发生冲突，人处于情绪化状态时就不讲理。所以发生冲突后的首要任务不是讲道理，而是将对方的情绪平稳下来。情绪平稳后才可能展开有效讨论。

## 方法二：重复对方的话

除了皮肤接触，第二种简单方法是重复对方的话。以下是具体操作步骤。

首先，你需要认真地倾听对方讲话，并不断微微点头。

接着，你可以这样说：“老公/老婆，你刚刚提的意见，我归纳了一下，一共有五个要点。我不知道我听明白了没有，理解准确了没有。我跟你重复确认一下，是不是这个意思。”

然后，逐一复述对方提出的要点：
1.  首先，你认为我……没做好。
2.  第二，你认为我……没做好。
3.  第三，你认为我……没做好。
4.  第四，你认为我……没做好。
5.  第五，你认为我……没做好。

最后询问：“不知道我这样的复述准确了吗？”

## 方法为何有效：满足情绪需求

对方听到准确复述后，通常会感觉被理解和尊重。这不能保证100%有效，但60%-70%的情况下对方情绪会平复。

复述确认后，并不一定需要立刻落实所有指责。从大量统计数字显示，很多时候事情就此了结。当然，特别严重的问题仍需处理。

因为家庭本质上不是一个讲道理的课堂，而是一个**讲情绪体验的场所**。只要情绪体验好，一切就OK；情绪体验不好，再有道理也讲不清。

## 使用时的两个关键注意事项

在使用此方法时，必须注意以下两点，否则可能适得其反。

**第一，表达焦点在于“我是否听明白”。**
正确的说法是：“我担心我刚刚没有听明白，跟你确认一下我理解得是否正确。”
错误的说法是：“我不知道你刚刚讲明白了没有，我确认一下你是不是这个意思。”
核心在于强调“我”的倾听责任，而非质疑“你”的表达。

**第二，自身情绪必须保持冷静。**
如果自己比对方更暴躁，此方法无效。问题就从“如何控制对方脾气”转变为“如何控制自己脾气”。

## 关于自我情绪控制的补充

真正的自我心理状态调整需从潜意识和自我催眠入手，效果很好。但这如同游泳或开车，是训练出来的技能，绝非看书或听课就能掌握。

比较临时的应急办法是：一旦脾气急了，使劲揪自己一把，疼痛感可能让人瞬间冷静。但这并非长期有效之策。
