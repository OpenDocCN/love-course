# 成年人修炼手册·第45讲：🚀 启动时间：拖延症的一种解决方案

在本节课中，我们将学习一个实用的生活技巧——“启动时间”。我们将探讨它如何作为一种温和而有效的策略，帮助你克服行动前的停滞状态，从而更顺利地开始并完成那些你熟悉但可能心生抗拒的任务。

## 概述：什么是启动时间？

上一节我们介绍了课程的主题。本节中，我们来看看“启动时间”的核心定义。

启动时间，是一个名词，指一个人从懒散状态进入到工作状态所需的最小时间。

我不喜欢“拖延症”这个概念。因为使用这个概念后，人们很容易将自己不在状态的原因归结为外在因素。拖延症被看作一种病，得病的人仿佛是无辜的，讨论就此结束。但也不要误会，我同样不认为需要对此进行谴责或施压，将其归结为个人的惰性。每个人都有拖延的倾向，无非是严重程度不同。

所以，我们先不讨论症状或内心感受，而是单纯地就事论事。任何人都有不愿意行动的时候。面对这种情况，我们可以怎么做？根据我的有限经验，也许你需要给自己一点“启动时间”。

## 心理建设：相信“现在就能做到”

在探讨具体行动前，心理层面的准备至关重要。以下是建立信心的关键思路。

以我这个日更作家为例，每天更新两篇以上的文章，并且要保证内容质量，光是想想就让人感到痛苦。但是，类似的事情我已经做过很多次，计算起来时日颇为漫长。我有没有打开电脑却一个字都写不出来的时候？肯定有。有没有写到一半突然头脑一片空白的时候？绝对有。有没有一想到还要更新五个月这一年才结束，因此深感绝望的时候？经常有。

但是，你看不出来。你看到的是每天都有新文章出现，仿佛我手上安装了永动机，不会疲倦，也不会灵感枯竭。怎么做到的呢？在心理层面上需要做一点简单的工作：先说服自己相信。如果一件事情你每天都在做，那意味着你现在就一定能做到。

这听起来像是一句废话，但是管用。一旦你开始怀疑自己能否做到，那么很快你就会真的做不到。因为怀疑和犹豫非常消耗一个人的精神能量，消耗到一定程度，人就失去了继续前进的勇气。所以，要相信自己能够做到。这种心理暗示的力量很强大，算是最好的一种“自欺”。

你可以在别的领域也试试。具体方法就是把结果和原因颠倒过来，把你想要的结果变成原因。然后你会发现自己居然也就信了，最后也就那么做到了，就有那么奇妙。

## 行动策略：如何应用启动时间

完成了心理建设，接下来我们进入行动层面。以下是实施“启动时间”的两个具体步骤。

然后在行动层面上，给自己一点启动时间。一方面，不管自己内心如何抗拒、如何怀疑，还是要进入行动状态。对于我来说，那就是：
```
1. 打开电脑。
2. 启动Word。
3. 随手开始敲字。
```
另一方面，也不要管自己具体做了什么，先动起来。在我这里，那就是先写上一段，想到哪里写到哪里，不要修改也不要考虑好坏，大不了一会儿再删除。

然后你会发现，过了一阵子，你是真的可以展开行动了。就像身体和大脑从沉睡中苏醒过来一样，丝毫没有之前停滞不前、怀疑抗拒的感觉。从开始行动到正式进入行动状态，人需要一个类似预热的过程。对于不同的人，这段时间的长短不同，可能从5分钟到半小时不等，我们可以把它称之为**启动时间**。

## 原理：为什么启动时间有效？

理解了怎么做之后，我们来看看它背后的工作原理。这能帮助你更自觉地运用这个方法。

我很早就发现，一个人能持续做一件事情，需要身心都处于某种工作状态。但是，我们都是人，不可能长久地停留在某种状态之中，时常会因为生活或者情感上的波动掉落出来，沉浸在别的状态中。

比如说，头一晚和朋友欢聚，开怀畅饮。那么第二天早上那种兴奋感可能还没有过去，忍不住去回想。那么，人就很难在手头上的事情上兴奋起来，迟迟进入不了状态。你的大脑正在调动回忆区块，进行隐性重放，根本没有额外的能量供给创作的区块。这时候你就会焦虑，会自我怀疑，认为自己做不到。

但其实你是可以做到的，只是你的脑子转换工作模式没那么快，需要你去提醒一下。启动时间就是起这个作用。装作已经进入之前的工作模式了，装得久一点，你的大脑就会真的认为现在已经进入了工作模式，随即就会切换状态。之前你已经做过很多次的话，那么这种切换会很流畅，因为大脑熟悉这种固有的工作模式了。

## 实例：从编剧行业看启动时间

理论需要实践的验证。让我们从一个需要高度创造性的行业——编剧——来观察这个方法的普适性。

哪怕是创作这种需要高度集中精力、发挥个人创意的工作，也会服从相同的规律。在编剧界流行一个笑话，叫做“打开Word就是干”。这听起来是个非常蛮横的创作方式，但是它的确有效。和等待灵感到来相比，“打开Word就是干”要靠谱得多。

编剧，尤其是电视剧编剧，动辄要写三四十集的剧本，手头却只有一个粗略的大纲提供大概的情节走向。但是，每一集里都需要用对话、行动、情节填补血肉，让大纲一点点变得丰满起来，落实到每一场戏上。

成熟的编剧不会坐等灵感来临，而是连续数月时间每天准时起床，打开电脑开始继续往下写，写到每天自己规定的字数，然后才考虑如何修改。有时候，可能会把前几天写好的内容全部推翻重来。因为今天写到一半的时候，突然想通了剧情，或者是人物关系变化，前后终于一气贯通了。因此，也可以说是前几天在假装创作，只是为了这一刻的领悟做准备。如果不是采用这种创作方法，无论是才华还是灵感，都无法完成马拉松一样的连续创作过程。

## 核心：启动时间如何针对拖延症

现在，让我们将焦点拉回最初的问题——拖延症。启动时间是如何对它产生效果的？

启动时间针对拖延症很有效。拖延症的症结不在于拖延，而在于因为逃避而拒绝采取行动。由于拒绝行动，人更加缺乏心理动力。而启动时间则非常精准地切断了这种恶性循环，根本不让人找“我只是想做得更好”这个借口。

无论如何，你先动起来，而且对结果也并没有任何要求，只需要你先给自己一个心理暗示：因为之前做到过，所以现在也就应该能够做到。然后就可以去“假装工作”。假装到了一定时间，那自然就会开始起效了。

## 局限与前提

当然，任何方法都有其适用范围。了解启动时间的局限，能帮助你更恰当地使用它。

启动时间只针对那些你已经做过很多次而且已经取得过结果的事情有效。如果是新工作、新挑战，那么可能无论你启动多长时间也不会有多大改善，因为你大脑中根本就没有对应的流程和方法。

同时，启动时间也要求你拥有相对安定的身心状态。如果扰动很大，那么无论做什么都很难进入状态。不过，那就是另外一个话题了。

说到底，一个人每天能够熟练维持的状态只有有限的几个。而人性天然喜欢新鲜刺激，对于重复性的工作心存抵触。重点永远是你如何从中找到热情和价值，而不是寄希望于某种特定的方法。启动时间可以从技巧上给予帮助，但是对于你的“心”可能帮助不大。如果心念可以转换，人甚至可以不觉得苦的存在。

## 总结与升华

本节课中，我们一起学习了“启动时间”这个概念及其应用。

最后，如果你确信你要做的事情不单对自己有益，而且关乎到更多人的利益和幸福，可以使你和人们更加紧密地联系在一起，那么也许你就不会再受拖延症的困扰了。如果是这样的话，启动时间可以帮助你度过那些短暂的颓丧、消沉和低迷时期，确保你始终待在自己的人生轨道上。
