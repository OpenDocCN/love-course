# 075.透视挣钱花钱背后的心理：第33讲：你的欲求很美好，你值得拥有！_

在本节课中，我们将学习如何正面看待自己的情绪与欲求，理解它们背后的积极意义，并通过一个具体的练习来改善你与金钱的关系，从而为实现财富自由奠定坚实的心理基础。

上一节我们探讨了情绪的价值，本节中我们来看看如何将这些认知转化为具体的信念和行动。

## 正面看待情绪与欲求

人要正面看待自己所有的情绪。正面看待不是指强迫自己保持“正能量”，而是指当愤怒、悲伤、绝望、羞耻、烦躁、焦虑等不快乐的情绪出现时，第一，它是在提醒你某些事情；第二，它其实是来帮助你的。情绪也是能量，别把它挡在门外。

谈到此处，我们需要更新一个核心观点：**你所有的欲求都是美好的，你值得拥有最好的东西**。这是一个终极信念。情绪的产生源于需求未被满足，这个需求就是你的欲望。如果你不尊重自己的欲求，你的情绪就无法得到释放和满足。只有当你回到根源，认识到自己对任何事物的欲求都是美好的，无论是要钱、要爱还是要其他东西，并且相信自己值得拥有最好的，你才会正视自己的想法和欲望，而不会进行自我攻击和批判。

很多时候，外界并没有人批评你，都是你在批评自己。例如，有人总觉得别人瞧不起自己。但请大家看合影时都看谁？都看自己。这意味着每个人主要关注自己，而非他人。所以，“瞧不起”这件事往往源于自我否定。

## 尊重欲望，减少内耗

一旦我们尊重自己内心的欲望，你会发现，你可以选择不去满足它，但不要批判它。只要不去批判自己的欲望，内心的情绪强度就会降低很多，你不会再有那么激烈的不满或愤怒感。

当你的安全感被充分满足时，在爱情里你就不会特别想要索取或控制。这些索取和控制的行为都源于内心的匮乏感。如果你认为安全感可以内在给予，那么当需要金钱时，你会聚焦于“我怎么去挣”，而不是“为什么我老公不给我钱”。这样，金钱的安全感就得到了满足。

对于爱的安全感，如果你觉得需要被爱，你可以直接向你爱的人表达：“我想你了，我好爱你，我想听到你说你爱我，这样会让我很开心。”只要你这样表达，对方很可能会回应你。但如果你说：“你为什么不关心我？你为什么只工作不理我？”那么，即使你同样在表达对爱的需求，由于方法不对，你也得不到想要的东西。

安全感的自我满足，前提是你要先表达自己的需求。你要什么，就要说出来。如果我们在物质上能得到充分满足，我们的追求就不会仅仅停留在赚钱上。当我们不批判自己，不评判自己，不找一大堆理由证明自己不应该提出要求，或自欺欺人说这样做是为自己好时，你的内心就平静了。

## 核心信念与练习

讲了这么多，核心只有一句话：**你的欲求很美好，你值得一切最好的东西**。哪怕你把它当作心理暗示，写在你的本子上或手机里，时时去看，你都在提醒自己：你不匮乏，你值得拥有一切。

如果一个人在爱里很有安全感，他就不会去秀恩爱。如果一个人在物质上很有安全感，他就不会那么把物质当回事，就能真正放下。

最后，给大家布置一个小作业：**给自己写一封关于金钱的信**。

几年前，我曾写过一封信给我的金钱。写完后，我简直不敢相信自己竟然那么讨厌钱。我发现自己对钱的感觉很糟糕，难怪挣了那么多钱却留不住钱。

以下是写信时可以包含的几个方向：

*   **工作是获得金钱乐趣的方式之一**：思考如何让你的工作变成一件有乐趣的、获得金钱的方式。
*   **你值得富足美好的物质生活**：展开想象，具体描述你喜欢的美好生活是什么样子。例如，你想要的房子、浴缸是什么样子的。将渴望的部分具象化。很神奇，我写下的东西后来都实现了。
*   **爱是为了创造自由、和平、快乐与分享的体验**：金钱通过爱来表达时，也能带来这些体验。例如，对母亲说“我爱你”和说“我爱你”并给她一个红包，感受是不同的。金钱是一种非常好用的、通用的爱的语言。爱是一个综合的体验，不要夸大其中任何一种形式的力量。
*   **回顾与展望**：写下你自己是如何一路走过来的，通过挣钱获得过哪些成就感和满足感。最后写下你的愿望，例如：“我希望跟金钱能成为伴侣关系，它能见证我的成长，帮助我实现物质领域的自由度，让我的精神与物质相匹配。”

想象金钱是一个人，它是你的爱人、朋友、敌人还是仇人？很多人真的把金钱当仇人。写完信你可能会瞬间明白，自己之所以没有钱，正是因为内心讨厌钱。

我相信写完以后，你一定会收获非常多关于自己和金钱关系的新认识。

本节课中，我们一起学习了正面看待情绪与欲望的重要性，建立了“欲求美好，值得拥有”的核心信念，并通过书写与金钱关系的信这一具体练习，开始清理阻碍财富的心理障碍。记住，尊重自己的所有渴望，是走向内在丰盛与外在富足的第一步。
