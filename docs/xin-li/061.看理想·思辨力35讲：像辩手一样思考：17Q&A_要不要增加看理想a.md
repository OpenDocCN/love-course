# 061.看理想·思辨力35讲：像辩手一样思考：17Q&A_要不要增加看理想a

在本节课中，我们将学习如何运用辩论思维中的“需求性、根属性、解决力、损益比”框架来分析具体议题。我们将通过三个具体案例——安乐死合法化、肥胖税征收以及产品功能决策——来演示这一分析过程。

## 安乐死合法化的辩论分析 💭

上一节我们介绍了辩论的基本框架，本节中我们来看看如何将其应用于“美国应全境合法化安乐死”这一议题。

**需求性**指为何需要改变现状。以下是支持合法化的几个需求性论点：

*   **问题导向**：现状问题是晚期绝症患者承受巨大痛苦，现有医疗手段（如杜冷丁）作用有限。他们求生不能，求死无门，无法有尊严地结束生命。合法化可解决此问题。
*   **目标导向（民意）**：为满足民意。盖洛普民调显示，超半数美国民众支持安乐死，且支持率多年来呈上升趋势。
*   **利益导向（附加好处）**：在病人无法治愈且自愿接受安乐死的前提下，停止使用医疗设备维持其生命体征可节约医疗资源（如床位）。**需注意**：此论点危险，易被理解为给生命价值排序。因此，**必须强调**这是既定前提下的附加好处，而非劝服他人接受安乐死的理由。
*   **目标导向（道德权利）**：只要行为不妨碍他人，人应拥有充分处置自己生命的权利。**争议点**在于：人是否有自由放弃自由（例如，能否签署自愿卖身为奴的契约）。穆勒认为，自由原则不允许一个人有不要自由的自由。

**根属性**指现状问题的根源。痛苦的根源一定是安乐死未合法化吗？不一定。有些痛苦源于疾病尚未治愈或未使用缓解药物。因此，**正方常增加限定条件**以确保根属性，例如：超过一名医生确诊病人无痊愈可能且正承受无法忍受的痛苦，且医患均认为除安乐死外别无他法。

> **补充信息**：美国目前有10个州允许“医生协助自杀”（Physician Assisted Suicide）。除蒙大拿州外，多数州要求病人预估寿命小于6个月，并需有两次相隔15天的意愿表达。

## 肥胖税的分析框架 📊

接下来，我们运用“需求性、根属性、解决力、损益比”框架，以“台湾是否应征收肥胖税”为例进行分析。

以下是分析此政策的四个步骤：

1.  **需求性**：台湾存在饮食相关的健康问题，对医疗系统造成压力。不健康饮食导致台湾儿童肥胖比例达33%。约715万20岁以上成年人患有一项“三高”疾病。相关疾病就诊人次与健保支出巨大。
2.  **根属性**：学术研究指出，过量摄入饱和脂肪与垃圾食品是导致“三高”及心脏病风险上升的重要原因。
3.  **解决力**：研究显示，垃圾食品价格上升会降低其消耗量，并提升健康食品消耗量。例如，碳酸饮料价格上升10%，民众从中摄取的卡路里下降7.12%。**反方可能攻击**：税收难以执行（如夜市盛行）；饮食习惯固化或成瘾，价格变化可能无效。
4.  **损益比**：反方会提出弊害，例如对低收入人群不公，因其食品开销占比更高。正方则可提出弥补措施，如发放食品券。**关键点**：在定义需求时，应聚焦“健康风险”而非“肥胖”本身，以避免政策被指责为歧视肥胖人士。

## 产品决策：是否增加数据收集功能 📱

最后，我们尝试将此框架应用于一个虚构的产品决策场景：作为“看理想”APP的设计者，是否应增加收集用户私人数据（甚至打通其他平台）的功能。

**需求性**是提议的出发点，主要基于以下几点：

*   **问题导向**：现状问题是主讲人无法了解听众画像，难以进行针对性节目设计。
*   **利益导向**：获取更多数据后，APP可更精准地向用户推送感兴趣的内容，提升用户体验。
*   **目标导向**：从数据中创造价值是大势所趋，APP需进行尝试以避免在时代中掉队。

**根属性**很简单：现状问题的根源在于没有收集相关数据，即“巧妇难为无米之炊”。

**解决力**分析需分两步：

1.  **能否成功执行**：APP的底层架构是否支持？是否有相关技术人员与组织架构？
2.  **执行后是否有效**：在尊重隐私（用户知情并选择加入）的前提下，若多数用户拒绝分享数据，或已分享的数据（如职业、兴趣、跨平台习惯）与“看理想”平台内容的相关性不足，此功能能否真正预测用户喜好并提升体验？**这是存疑的**。

**损益比**评估利弊：

*   **弊（损）**：
    *   **劳民伤财**：可能需改变整体技术架构，成本高昂，且需与科技公司竞争招募程序员。
    *   **隐私疑虑**：为追求数据全面性可能引发用户不满。
    *   **用户流失风险**：核心用户可能不喜欢改变，调整推送机制可能导致其离开。
*   **利（益）**：非常不明显且不确定。

**初步结论**：弊大于利，不应增加此功能。**更佳决策**需依赖更多数据，例如参考同量级APP实施类似功能后的效果、成本、用户变动等案例，以进行更精确的损益比分析。
