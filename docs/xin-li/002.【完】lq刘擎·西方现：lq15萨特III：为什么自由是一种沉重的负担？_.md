# 002.【完】lq刘擎·西方现：lq15萨特III：为什么自由是一种沉重的负担？_

在本节课中，我们将探讨萨特哲学中关于自由的两个核心观点：自由选择为何是一种沉重的负担，以及“他人即地狱”这一著名论断的含义。

上一节我们介绍了萨特从“存在就是虚无”推出了人的绝对自由。本节中，我们来看看这种自由所带来的严酷后果。

## 自由选择的沉重负担

自由的命运意味着人总是可以有所选择，并且必须做出选择。这赋予了人掌握自己命运的自主性和尊严。然而，这种自由并非轻松的好事，它有其严酷的一面。

自由选择之所以成为负担，是因为选择必定带来后果，而人必须为这些后果负责。萨特认为，没有任何别人可以承担这份责任，你做出了选择，就要独自承担责任。

但承担责任究竟意味着什么呢？为什么只能独自承担？萨特的回答是：不能分担。因为你的每一个选择背后，都有一个评判标准，而这个标准只能是你自己给自己确立的。

以下是生活中常见的需要做出重大选择的例子：
*   毕业之后是继续深造还是直接工作？
*   选择什么职业？
*   要不要结婚？
*   要不要孩子？

我们生活在自己选择后果的塑造之中，自然希望选择有坚实可靠的依据。但萨特指出，你所有选择的依据都只是你自己的。这受到了尼采“人是自己价值的创造者”思想的影响。陀思妥耶夫斯基在《卡拉马佐夫兄弟》中的名言“如果上帝死了，那么一切都被允许了”，被萨特视为存在主义的起点。

因此，人不仅必须选择，还必须为自己确立选择的标准。于是，人没有任何托词。你不能将责任归咎于父母或宗教，因为是你自己把顺从父母或宗教教义当成了标准。

萨特认为，任何信条、主义或别人的建议都不能成为借口，这些只是用来自欺欺人、逃避责任的手段。例如，假如恋爱失败，朋友安慰说“这不是你的错”，但萨特可能会说“这就是你的错”，因为是你自己选择的人，结果自然是你的责任。

**独自承担责任**，就是自己做自己的立法者，为自己每一个选择承担绝对责任。于是，从“存在就是虚无”，萨特推出了人的**绝对自由**；从绝对自由，又推出了**绝对责任**。这是一份非常沉重的负担。

理解了这一点，就能读懂米兰·昆德拉的小说标题《不能承受的生命之轻》。这里的“轻”源自人自由而轻盈的存在方式，存在有着无限可能性。但这种自由的轻盈又是孤独而沉重的，因为你必须独自承担所有选择和自己的生命，这让人难以承受。书名深刻揭示了现代人的精神困境。

## 他人即地狱

说到孤独，你可能会想，我们还有朋友、亲人和爱人，与他们在一起是否能帮我们摆脱一些孤独呢？这就引出了萨特的另一句名言：**他人即地狱**。

这句话出自萨特的剧本《禁闭》。剧中三个被囚禁的鬼魂在等待下地狱的过程中，彼此不断欺骗和折磨，最终领悟到：不用等待地狱了，他们已经身在地狱。地狱并非刀山火海，**“永远和他人在一起，这本身就是地狱”**。

要理解这句话，需回到萨特的哲学思想。我们知道，人能自由掌握自己的生命，这被称为人的**主体性**（我是主体，意味着我有主导权）。问题在于，当两个自由的人相遇时，谁才是主体？萨特认为，人总要维护自己的主体性，因此人与人之间必定会为了争夺主体性而斗争。

每个人在和他人相处时，都试图把他人变成**客体**（对象），以此来维护自己的主体性和自由。萨特举了一个例子：当陌生人在街上打量你时，你会感到不舒服。因为他的注视下意识地将你变成了他观察的客体。在他的注视中，他是主导者，你成了被观看的“物品”。他的主体性是通过否定（物化）你的主体性来实现的。

你可以反抗，比如回看对方，用你的注视把他变成客体。在萨特看来，人和人的交往总是充满这种争夺主体性的斗争，即使在爱情中也不例外。

他揭示，我们想象中的浪漫爱情（不分彼此、合二为一）只是一种开始时的幻觉。爱情同样充满了为争夺主体性而展开的冲突和斗争。到最后，关系可能演变为：
*   **受虐**：在羞耻中享受快乐。
*   **施虐**：在內疚中感到愉悦。

常见的爱情情节是：追求者通过讨好、迎合对方，失去自己的主体性；而被追求者则通过展示魅力、接受奉献来获得自己的主体性。爱情越热烈，双方可能越接近受虐狂与施虐狂。

萨特还有一个奇妙的说法：**憎恨其实就是你承认了别人的自由**。你不会恨一个杯子或椅子，憎恨只能指向人，因为人有自由，能出于自主意识对你行动。在他人的主体性行动中，你沦为了客体，你的主体性被否定，因此你会憎恨。憎恨意味着你承认了对方的**主体性**和**自由**。

我们生活在一个有他人存在的世界，每个人都是自由的。但萨特认为，我们不可能实现理想中主体与主体相互承认、平等、尊重的“共同自由”，因为每个人都要实现自己的主体性。在这个意义上，萨特是一个悲观主义者。

## 总结：绝望中的希望与行动

本节课中，我们一起学习了萨特思想中自由带来的沉重负担以及人际关系的本质困境。

萨特的存在主义继承了尼采的思想，强调人只能自己创造自己，在自由选择中塑造自己，并承担全部责任。他和尼采一样，否认普遍客观的价值标准，但他似乎放弃了尼采的“超人”学说，强调人只能孤独地面对自己的选择。

聆听萨特的哲学，你可能会感到一种绝望：起点是虚无，终点是孤独，中间是沉重的责任。萨特也承认自己的哲学是“绝望的哲学”。

但他同时指出，存在主义也是**希望的哲学**。希望就在我们的自由之中，在人的**无限可能性**之中。我们永远有改变的潜能，不必服从任何注定的命运。

萨特向我们展现了每个人面临的精神困境，同时又告诉我们，我们永远都可以做出改变。改变靠什么？**行动**。因此，存在主义是绝望的哲学，是希望的哲学，但最重要的，它是**行动的哲学**。我们最根本的自由和可能性，都在行动中实现。

人生或许无意义，但仍然可以成为强者。萨特的思想给我们的启示或许可以总结为：**看清真相，继续战斗**。

人是徒劳的激情，注定孤独。但既然可以选择，那就去选择，然后为选择负责。我们都是推石上山的西西弗斯，但我们知道“西西弗斯是幸福的”，因为“登上顶峰的斗争本身足以充实人的心灵”。

最后，留给你一个问题，这是所有人都要面对的：**如果人生没有确定的意义，那么在你看来，我们是否还能做出有意义的选择？在你的生活中，你曾经做出的最让自己骄傲的行动是什么？**
