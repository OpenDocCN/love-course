# 课程048：社交天性精读班 - 第14天 📚

在本节课中，我们将学习《社交天性》最后一部分关于“应用”的内容。作者将核心的社交驱动力理论，应用于企业管理等实际场景，探讨如何有效激励员工。我们会接触到如SCARF模型、自我决定理论等概念，并理解非物质激励的重要性。

上一节我们介绍了人类三大社交驱动力。本节中，我们来看看如何将这些理论应用于实际管理。

作者认为，从人的社交天性（而非纯粹的物质激励）出发，可以更有效地管理企业。在现代社会，驱动员工投入工作的意愿是关键。当前的教育体系和企业培训已能解决大部分技能问题，因此员工的态度和投入意愿变得更为重要。此外，大多数岗位需要创造力，员工常需在管理者视线之外自主决策。此时，驱动力的重要性超越了能力或体力，核心问题是员工“是否愿意为你投入”。

书中首先提到了一个由神经领导力研究所提出的缩写模型。该模型包含多项要素，但作者主要讨论其中与社交相关的部分。

以下是该模型中被强调的几个社交相关要素：

*   **地位**：员工在组织中被认可和尊重的感觉。
*   **关联性**：与他人建立积极、紧密关系的机会。
*   **公平**：在资源分配、评价和机会等方面感受到的公正。

书中通过例子说明，这些非物质激励手段非常有效。例如，在企业管理、学校教育甚至亲子关系中，将努力“视觉化”（如贴小红星、进度表）能有效激励人。这种公开的、来自管理层的认可，是一种强大的社会奖赏。

作者进一步阐述，从社交天性的视角看，企业的“社会资本”或“人际心理资本”是其核心竞争力的关键。当然，这带有作者自身研究视角的倾向性。当一种理论宣称可以解释一切时，我们需要意识到它可能只是一个有效的工具，而非万能药。

书中还推荐了另一本相关著作。这本书将人分为两类。

以下是书中提到的人际行为分类：

*   **付出者**：倾向于关注他人需求，总想着如何让身边的人更好。
*   **获取者**：更关注自身利益，思考如何通过他人获取所需。

这两种人的行为模式截然不同。这本书的原名是《Give and Take》，中文版营销名为《沃顿商学院最受欢迎的成功课》。这个现象本身也印证了社交线索的力量：打动人心的画面和故事（如希望工程的照片），比事实报表或理论分析更能激发人的情感和行动决策。

这与另一本畅销书《影响力》的观点相通。《影响力》中列举的所有产生巨大影响力的要素，几乎都与人际互动、情感和价值相关，而非逻辑说教。人的决策常受情绪情感引导，而对逻辑理性的偏好远不如对人际情感的敏感度高。因此，也推荐阅读《情商》一书，它侧重于自我修炼；而《影响力》则更侧重于职场与公众场景的实践。

需要强调的是，掌握这些社交技能并非仅仅阅读即可。它是一种**习得性技能**，需要持续练习。练习的特点包括：**1) 需要经常练习，否则会生疏；2) 进展是渐进的，难以一蹴而就**。

既然本书谈到了社会奖赏的激励手段，这里分享一张在商学院讲解驱动力时常用的幻灯片，其中列举了非物质激励的关键要素。

以下是有效的非物质激励要素列表：

*   **关注**：对员工给予个体化的重视。
*   **聆听**：真诚地倾听员工的想法和意见。
*   **信任**：赋予员工责任和自主权。
*   **发挥机会**：提供让员工施展才华的平台。
*   **团结协作**：营造团队“一加一大于二”的正向氛围。
*   **成长机会**：支持员工的个人发展与学习。
*   **公平感**：确保流程和结果的公正性。

这些要素直指人心（情感），而非仅作用于大脑（理性），但它们往往比物质激励更有效。一个简单的类比是：物质激励的作用在于避免让人产生“不公平”和“匮乏”的感觉；在此基础之上，真正驱动卓越表现的往往是这些非物质要素。

举例说明，对于一名程序员，公平的薪酬（内部比较和行业比较）是基础。在此之上，能否获得关注、信任、成长机会等，才是激励其全力投入的关键。实施这些非物质激励并非为了节省成本，相反，它需要管理者投入极高的精力。如果仅靠加钱就能解决所有管理问题，那么管理将沦为成本恶性竞争，商学院也将失去价值。但事实并非如此。

本节课中我们一起学习了如何将社交天性理论应用于激励与管理。作者通过SCARF模型、人际分类等工具，强调了地位、关联、公平等社会性奖赏的重要性。我们认识到，在保障基本公平的前提下，关注、信任、成长机会等非物质要素，是驱动现代知识型员工深度投入的核心力量。掌握这些技能需要持续练习，但它们对于有效领导和管理至关重要。
