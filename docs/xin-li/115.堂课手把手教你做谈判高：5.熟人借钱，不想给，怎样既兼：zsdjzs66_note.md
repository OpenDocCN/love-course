# 115.堂课手把手教你做谈判高：5.熟人借钱，不想给，怎样既兼：zsdjzs66_note

在本节课中，我们将探讨一个常见的谈判难题：当熟人向你借钱，你既不想借出，又希望维护彼此关系和对方的面子时，应该如何应对。我们将从分析谈判中的“关系”要素入手，并提供具体的思考方法与策略。

---

上一节我们介绍了谈判筹码的重要性。本节中，我们来看看谈判准备中一个核心的要素：**关系**。

谈判前的准备涉及多个方面，其中首要的是分析双方的关系。关系的好坏直接影响谈判的策略与结果。我们不仅希望谈判能够达成目标，更希望谈判后关系的品质得以维持，双方还能保持沟通，而非变得剑拔弩张。

关系分析包括多个维度。首先，需要区分你是与公司外部的人谈判，还是与内部的人谈判。与外部人员谈判时，你可能拥有更多策略选择；而与内部同事谈判时，某些过于强硬的策略可能因需要维持长期共事关系而无法使用。

这里还存在一个关键问题：与熟人谈判和与陌生人谈判截然不同。许多人不擅长与熟人谈判。例如，一位女士卖房时，只有她的姑妈有意购买，但要求降价。不降价会得罪亲人，降价则自己购房的首付款就不够了。她因此感叹，如果不是亲戚该多好。

那么，我们是否希望谈判对手完全是陌生人呢？通常也不是。有点熟悉但又不太熟的关系，可能最为理想。处理这种关系需要特别的思考方式。

以下是关于“关系”在谈判中影响的几个核心思考面向：

根据研究，中国人在处理关系时，内心有一个天平。天平的一端是**交情**，另一端是**要求**。没有一定的交情，人们往往难以开口提出要求。这符合“交浅不言深”的原则。

但问题往往出现在后半段。当一个人基于自认为足够的交情提出要求，却被对方拒绝时，他通常不会反思自己的要求是否过分，而是会觉得对方“不够意思”、“不给面子”。他会认为：“我们这么多年的交情，我第一次开口你就拒绝，太不给我面子了。”此时，**交情、要求与面子三者紧密捆绑在一起**。

我们常说“买卖不成仁义在”，但现实中，“仁义”常常难以维系。例如，商人间喝酒时常说“生意归生意，朋友归朋友”，但若多次合作不成，酒局也会变得尴尬。因此，这种三合一的捆绑关系使得谈判，尤其是熟人间的谈判，变得复杂。

在谈判中，若想防止对方提出你认为过分的要求，一个策略是：**设法简化你们之间的交情**。从天平的“交情”一端减重，使其不足以支撑起过高的“要求”。

如何简化交情呢？虽然很难完全分割，但可以尝试将交情区分为**私人交情**和**业务交情**。我们更需要的是纯粹的业务交情。过多的私人交情，往往会带来许多额外的要求。中国古语“君子之交淡如水”充满了政治智慧。年轻时或许难以理解，但随着阅历增长，当被过密的私人关系压得喘不过气时，才能体会到这句话的深意。类似地，“难得糊涂”、“人不知而不愠”等，都是随着年龄增长才有深刻体悟的智慧。

其次，要认识到**对内谈判**和**对外谈判**在关系考量上有所不同。因此，处理方法和使用的战术也应有所区别。

第三，探讨是否应该**用让步来培养关系**。这是一个高难度的操作。如果操作不当，一味让步可能会降低自己的格调和在对方心中的价值。例如，在商业中，有时通过小幅让步（如赠送礼品）来建立长期客户关系是可行的。但大多数情况下，在重要谈判中过度让步，可能不仅无法培养关系，反而会损害自身地位。

更稳妥的方法是：**通过分享信息和研究成果来培养关系**。这意味着你要想办法让客户与你共同成长。这对于建立长期、稳固的伙伴关系至关重要。

比起请客吃饭或赠送礼物（这些容易被遗忘），帮助对方成长是更有价值的投资。例如，赠送一套高质量的课程就是一种有效的方式。上课是投资自己，也能用来建立客户关系。在实践中，有些公司会为其重要客户或合作伙伴购买培训课程，以此作为维护关系、共同提升的方式。这种关系不再是单纯的对立买卖关系，而是进化为共同成长的**伙伴关系**。

---

本节课中，我们一起学习了在面临熟人借钱等敏感请求时，如何通过分析并管理“关系”来进行婉拒。核心在于理解交情、要求与面子的捆绑，并尝试通过简化私人交情、区分内外谈判，以及用“共享成长”而非单纯“让步”的方式来维护健康的长期关系。记住，目标是既照顾对方感受，又守住自己的原则。
