# 儿童发展心理学 012：埃里克森精神分析理论（0~1岁）👶

在本节课中，我们将要学习埃里克森精神分析理论中关于婴儿期（0-1岁）的核心发展任务。我们将了解这个阶段对孩子一生信任感建立的关键影响，以及父母应如何行动。

## 理论背景

首先介绍一位重要的心理学家：埃里克森。他发展出的理论建立在心理学大师弗洛伊德的基础上。弗洛伊德是精神分析学派创始人，提出了潜意识等概念。

埃里克森曾是弗洛伊德的追随者。他在弗洛伊德理论体系的基础上，结合自身经历，将其细化和具体化，编撰出儿童心理发展理论。在弗洛伊德的精神分析看来，人长大后的状态与童年经历密切相关。

埃里克森本人因小儿麻痹症终生需坐轮椅。事实上，许多心理学大师都有坎坷的人生经历。因为对人生深刻的体验，尤其是对痛苦的理解，对于从事心理学工作至关重要。

## 发展阶段划分

埃里克森将人的心理发展划分为多个阶段。考虑到听众多为幼儿家长，我们着重介绍到12岁为止的前四个阶段。

以下是四个阶段的划分：
*   **婴儿期**：0到1岁。
*   **幼儿期**：1到3岁。
*   **学前期**：4到6岁（幼儿园阶段）。
*   **小学期**：7到12岁。

这种划分具有其内在的心理发展逻辑。接下来，我们将详细探讨第一个阶段——婴儿期。

## 婴儿期（0-1岁）：基本信任 vs. 基本不信任

上一节我们介绍了发展阶段的划分，本节中我们来看看第一个关键时期：婴儿期。

0到1岁的婴儿最大的特点是极端无助。此阶段的宝宝若无母亲照料便无法生存，完全依赖于抚养者。

这个时期孩子需要完成一项核心发展任务：**建立对世界的“基本信任”**。试想，如果一个人对外界完全无法信任、理解和依靠，他的一生将充满怀疑。而当婴儿什么都做不了时，他如何建立最初的信任呢？这两个问题逻辑上互为前提。

因此，0到1岁父母的养育方式，直接决定了孩子一生是否信任他人与世界。这一点至关重要。无论是后续的心理咨询实践，还是对不同个体的访谈研究，都证实了这一观点的正确性。

婴儿期孩子极度弱小，生存完全仰赖父母。父母需要做到以下两点，来帮助孩子形成“基本信任”，克服“基本不信任”。

### 核心养育原则一：按需喂养

父母应进行“按需喂养”，而非“按时喂养”。核心是提供孩子需要的，而非你想给的。

试想一个场景：你年老卧床，护士不顾你是否想吃，定时将食物塞入你口中。你会感到舒适吗？你会对世界抱有善意吗？同理，当孩子不想吃时，强行喂养只是为了抚养者自身的便利，却让孩子遭殃。

**公式：建立信任 = 提供对方所需**

给予对方真正想要的东西，而非你想给予的东西，是建立关系和信任的基础。在婴儿期，父母给予孩子无条件的爱再多也不为过。

### 核心养育原则二：保持环境稳定

必须维护抚养环境的稳定性。避免频繁更换主要抚养人（如妈妈、奶奶、阿姨轮流照顾）。

对于婴儿而言，频繁更换照料者意味着什么？想象一下：你每晚睡在伴侣身旁，次日醒来发现身边的人变成了陌生人。你会是什么感受？婴儿刚刚适应了母亲的面容、声音和气味，突然变成一个行为举止完全不同的陌生人，他无法理解这种变化。

当孩子刚刚努力适应了新照料者，眼前的人又变了。他应与谁建立信任？如果每天照顾他的人都不一样，信任关系根本无法维系。

因此，整个童年期的养育核心可归结为两个字：**稳定**。

稳定的抚养环境才能造就稳定的心理状态。频繁更换抚养人、搬家或转学，可能导致孩子注意力难以集中、心理韧性（心理肌肉）出问题，以及性格不稳定。

## 课程总结

本节课中我们一起学习了埃里克森发展理论的第一阶段。我们了解到，0-1岁婴儿期的核心冲突是 **“基本信任”对“基本不信任”**。孩子通过父母无条件的爱与稳定的抚养环境，来建立对世界最初的、至关重要的信任感。这个阶段的成功过渡，为孩子一生的心理健康与社会关系奠定了基石。
