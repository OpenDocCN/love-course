## 概述

我们透过金钱，能够观察到形形色色的人。金钱像一种显色剂，能将人们心底的各种欲望和心态显现出来。心理学认为，人们的不同心态，往往不在于生活中具体发生了什么事，而在于看待这些事的视角。在挣钱这件事上，同样存在视角的差异。了解你目前所处的层级，就能明白为何有些人与你活得不一样。

## 第一层级：情绪层 😫

上一节我们提到了视角的重要性，本节中我们来看看最基础的层级——情绪层。

假设将人的生命状态分为三到四个层级，最底层的人容易受情绪困扰。这个层级的人永远在为情绪做事，活在沮丧、消极、悲观和厌世的状态中。他们需要花费大量时间解决情绪问题，无法进行理性思考。简单来说，这是一种“生病”的状态，只有高兴和不高兴，且大部分时间是不高兴。

以下是情绪层级的主要特征：

*   **社会功能受损**：类似抑郁症、焦虑症、强迫症的状态，有时会丧失部分社会功能。
*   **人际关系糟糕**：难以建立良好的社交圈，常抱怨父母，朋友因其负能量而远离，工作中无人愿意合作。
*   **行为极端**：要么攻击性极强，对外发泄；要么对内攻击极强，丧失生命力，萎靡不振。
*   **缺乏思想**：行动冲动或极度压抑，情绪不经过大脑思考。

**与金钱的关系**：情绪层的人与金钱几乎没有良性交流。他们情绪化、喜怒无常的状态，如同一个让“金钱”（拟人化）不愿靠近的人。因此，他们的生活等级通常处于社会最底层，活得痛苦且可悲。

**核心建议**：如果你遇到处于情绪层的人，建议尽快远离。与他们沟通解决问题是不可能的，因为他们无法进入理性讨论的层面。这种情况需要专业心理医生甚至药物干预，普通人切勿试图“拯救”，以免被其拖入负面漩涡。

## 第二层级：问题层 🤔

离开了纯粹的情绪困扰，我们进入第二个层级——问题层。这个层级的人开始面对问题，但方式并不健康。

问题层的人，永远在向外界抛出问题。“怎么办”、“我不会”、“我做不到”是他们的口头禅。他们从不为自己负责，认为别人必须帮助自己，否则就是坏人。这个层级的人普遍悲观、消极、懒惰，不愿独立思考，与他们相处会感到压抑。

以下是问题层级的主要特征：

*   **推卸责任**：将一切问题和责任丢给他人，认为都是别人的错。例如：“如果你工资给高点，我就会努力。”
*   **缺乏感恩**：对别人的付出视为理所当然，充满冷酷、自私和怀疑。
*   **散发负能量**：习惯打击他人，用“关心”的形式投射自己的焦虑（如“你这么大了还不结婚怎么办”）。
*   **消耗他人**：他们来到这个世界的目的，仿佛就是给别人添麻烦。

**如何改变？** 改变始于关注点的转移。你关注什么，就会吸引什么。关注烦恼得到烦恼，关注悲伤得到委屈。

以下是改变的具体步骤：

1.  **管好语言**：停止抱怨、指责、说闲话和坏话。
2.  **学会祝福**：多说祝福别人的话，这同时也是在祝福自己。
3.  **停止攻击**：意识到攻击别人就是在引发别人攻击自己，是自我攻击的一种形式。

**公式**：`你的关注点 ≈ 你吸引来的事物`

一旦你管好了语言、视听和生活环境，不再聚焦于问题，而是聚焦于美好，你的生活就会开始改变。

## 第三层级：事件层 🌸

解决了问题和情绪困扰后，我们来到更有建设性的第三层级——事件层。

事件层的人已经能够释放出爱与美好的能量。他们生命中减少了内心的拉扯，能很好地为自己负责，状态稳定且愉悦。他们自成一格，擅长做自己喜欢的事，相信“一切都是最好的安排”。

以下是事件层级的主要特征：

*   **散发喜悦**：像花一样绽放，让接近他们的人感到愉悦。
*   **自我负责**：个人状态很好，内心平静，少有情绪和问题困扰。
*   **安逸与相信**：活在一种安逸和信任生命安排的状态中。

**新的挑战**：这个层级的“问题”在于，当一个人把自己调整好后，常会萌生帮助他人的愿望。于是，挑战就从内部转向了外部。

**核心困扰**：你会吸引大量需要处理的事件，总是忙个不停，感觉“没有你就不行”。每天需要处理大量人际关系和具体事务，虽然是在做有益的事，但可能会陷入忙碌的漩涡。

**代码描述其状态**：
```python
if 个人状态 == “良好”:
    产生 帮助他人的意愿
    导致 外部事件数量激增
    结果 陷入“忙碌但充实”的循环
```

## 总结

本节课中我们一起学习了生命层级的三个模型：**情绪层**、**问题层**和**事件层**。

*   **情绪层**的人被情绪主宰，与金钱和世界隔绝。
*   **问题层**的人不断抛出问题并归咎外界，需要学习管理语言和关注点。
*   **事件层**的人自我状态良好，但容易因帮助他人而陷入事件处理的忙碌中。

理解这些层级，能帮助你更清晰地认识自己和他人，明白不同行为背后的心理动机，从而在追求财富与个人成长的道路上，做出更明智的选择。
