# 011.从家长管：03生活规矩：养成良好的吃饭、：13读书：建立3个连接，让

在本节课中，我们将要学习如何帮助孩子，特别是幼儿，培养真正的阅读兴趣和习惯。核心在于理解阅读习惯的构成，并学会通过建立三个关键“连接”，让孩子自然而然地爱上读书。

一位两岁孩子的妈妈为了帮助宝宝养成阅读习惯，要求孩子每晚读三十分钟绘本。宝宝无法坚持，总想挣脱去玩玩具。每晚的阅读时间都以宝宝的烦躁哭闹收尾。家人劝妈妈说孩子还小，不要拔苗助长。

这位妈妈意识到阅读的重要性并进行了坚持，这是值得肯定的。但她犯了一个常见错误：她规定的三十分钟阅读时间，不匹配两岁孩子的年龄特征。让一个两岁的孩子安静坐着听三十分钟故事，是非常困难的。

成年人常有的误区是：认为“我讲故事，孩子就坐着听”。同时，容易把阅读习惯简单理解为“坐着听故事或看故事”。对于年纪小的孩子，关键在于让他感受到阅读的价值，并让书成为他生活中自然的一部分。

另一种说法是“孩子还小，喜欢做什么就做什么，不要刻意培养”。这也不对。习惯确实需要从小培养，但对于幼儿，如何培养阅读习惯是有学问的。

今天，我将教大家几个绝招。以下是培养幼儿阅读习惯的三个基础环境要素：

1.  **眼里有书**：孩子生活的环境里，用眼睛能看到家里有书。
2.  **手摸到书**：孩子在不需要帮助的情况下，手能够得着书。例如，给三岁孩子看的绘本不应放在他拿不到的高处。
3.  **耳听到书**：经常有人给他讲书里的故事，或他能听到与故事相关的内容。

我们重复一下，要让孩子在生活中 **眼里有书、手摸到书、耳朵听到书**。这才是真正阅读习惯的构成基础。

可以看到，孩子阅读习惯的发起人是父母。因为孩子年龄小，能力有限，需要爸爸妈妈配合。父母可以在家多放书，有意识地把书放在孩子触手可及的地方，并经常给孩子讲故事。

此外，家里稳定、安静、整齐、不受打扰的阅读环境也有利于孩子。假设孩子天天能看到妈妈安静阅读20到30分钟，他就会意识到，看书是家里每个人都会做的事，而不仅仅是父母要求他做的事。

在做到了“眼里有书、手摸到书、耳听到书”的环境建设后，我们就要做一件特别重要的事，其关键词是 **“连接”**。这个连接有几层含义。

上一节我们介绍了阅读环境的基础建设，本节中我们来看看如何通过“连接”深化阅读兴趣。以下是建立三个关键连接的步骤：

1.  **兴趣与内容的连接**：把孩子感兴趣的东西，和他可以看的绘本进行连接。例如，喜欢车的小男孩，可以选择与车相关的绘本。孩子在生活中看得到车，在阅读内容里也能用上，就会产生兴趣。
2.  **阅读与生活的连接**：让孩子把书中看到的内容，在生活实践中进行对应。例如，孩子看了关于消防车的书，可以有意识地带他去博物馆或生活中指认：“宝宝，你看，这就是我们书上看到的消防车。”这样，阅读的知识就渗透到了生活中。
3.  **情感与价值的连接**：让孩子觉得阅读是一件重要且有成就感的事。例如，当孩子通过看书认识了各种车，并在生活中指认出来时，及时给予肯定：“哇，你能看懂并能辨别这么多车，真棒！”这建立了积极的情感连接。

通过这三种连接，可以让孩子深刻感受到阅读的价值。

本节课中我们一起学习了如何为孩子培养阅读习惯。

首先，是家庭环境的建设，分为三个方面：**眼里有书、手摸到书、耳朵听到书**。

其次，作为父母，我们要主动为孩子建立三个关键连接：
*   第一个连接：**兴趣与内容连接**。
*   第二个连接：**阅读与生活连接**。
*   第三个连接：**情感与价值连接**。

为了帮助孩子养成阅读习惯，我们要营造这样的环境，让孩子慢慢拥有这些能力。但必须注意，首先要让孩子对书感兴趣，并符合孩子的兴趣点和能力发展，避免提出孩子做不到的过高要求，或将阅读时长、频率定得太死，导致阅读变成双方都苦不堪言的事。

我们都知道，运动对孩子的身心健康非常重要。很多父母会因孩子懒、不爱运动而头疼。下次课，我将告诉大家，怎样帮助孩子克服惰性，养成规律的运动习惯。
