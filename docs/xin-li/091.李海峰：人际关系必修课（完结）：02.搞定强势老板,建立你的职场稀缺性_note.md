# 091.李海峰：人际关系必修课（完结）：02.搞定强势老板,建立你的职场稀缺性_note

在本节课中，我们将学习如何与领袖型（D特质）的强势老板有效沟通，建立你在职场的独特价值。我们将深入分析领袖型领导的特征，理解他们的关注点，并掌握与之相处的具体策略。

上节课我们介绍了DISC行为风格的基本理论。本节中，我们来看看如何与其中最具挑战性的“领袖型”（D型）领导打交道。

生活工作中，你是否常感到郁闷？例如，总觉得和老板“八字不合”。每次汇报工作都可能被批评；平时做事得不到表扬，但只要犯一点小错误，就会遭到严厉指责。这些抱怨的对象，通常就是**领袖型（D风格）**特质较高的人。

职场中与领袖型领导交流会遇到怎样的情况？我们通过两位同学的遭遇来了解。

**场景一：尼昊凡的汇报**
> 老板等一下，上个月的工作报表出了一些问题，需要您确认一下。
> 好你说吧，尽快点。
> 就是上个月的Emmie，她不是已经离职了吗？公司已经给她办好了一切的离职手续。
> 对，可是她男朋友那边好像出了问题…

尼昊凡的汇报没有逻辑性，无关铺垫过多。本次汇报描述了过多的原因与过程，而D型领导更关注**结果**。在这样冗长乏味的汇报中，D型领导早已失去兴趣与耐心。

> …所以我想让安米在公司再以临时工的身份做一个月，工资是以前的八成。
> 明天不用来上班了。

领袖型领导十分关注效率与做事速度，行程安排满满当当。他们希望汇报是**开门见山、直入主题、简明扼要**的。而此时的尼昊凡，已经让D型领导怒火中烧。

**场景二：向不凯的困境**
老板不断给向不凯分配新任务：整理文件、接待客户、翻译、出差…向不凯一开始热情满满，有求必应。然而随着工作量不断攀升导致分身乏术。他没有意识到领导不断分配任务背后的用意，片面以为领导在压榨他，于是产生了辞职的念头。

我们来总结一下，常用D特质的领袖型人的特征有哪些：

以下是领袖型（D型）人的四个核心特征：
1.  **以目标为导向**
2.  **行动力强**
3.  **自尊心超强**
4.  **创新多变**

你以为获取领袖型人认可的方式是拍马屁、送礼、请客吃饭吗？其实都不是，而是**你做事的方式**。别人怎么对你，其实是你教他的。你要搞定领袖型的人，首先你要懂他。

领袖型的人常常使用D风格，那他关注的到底是什么？
*   当D给你分配更多任务时，代表他认可你，甚至想提拔你。
*   D比较擅长**关注事**，而不是关心人。对他来讲，关心你的工作就是在关心你。
*   D讨厌和“笨蛋”在一起。只要你反应慢，他就会觉得你是笨蛋。

现在你懂他了，就有办法影响他。明白领袖型的人看重什么，那么让他欣赏你、帮助你，就水到渠成。核心是让他觉得你**不止做事快，而且效率高**。

如果你天生反应不快，该怎么办？首先不要找借口。以下是四个实用的小妙招：

**第一招：想方设法节约他的时间**
这会让他觉得你“快”。你要学会讲重点，注意多用“什么最重要”、“什么最紧急”这类对时间优先级敏感的词。需要时可列清单，让他感觉一切尽在掌握。给他资料时，务必整理好并标注重点。如果事情复杂，当面汇报易被打断，可采用**书面沟通**，让对方全面了解。

给领袖型领导的书面资料需注意：
1.  重点清晰，方便快速抓取。
2.  好的视觉体验，使用简洁的文字。

**第二招：及时反馈，设置时间点**
领袖型的人目标导向、行动快，同时自尊心超强。及时反馈能显示你的尊重，也满足他对事物的掌控欲。你要定期、频繁地与他沟通汇报，切记不要只是闷头做事。

**第三招：不要给他做问答题，多给他做判断题和选择题**
例如，你可以说：“领导，我们遇到一个问题。我的想法是这样处理，您看是否合适？”这既显示尊重，又能体现你动过脑筋。如果你能想出几个解决方案并分析利弊，再让领导做选择，他会对你刮目相看。

**第四招：不断提出新的点子**
表现出你的求知欲和创新能力。领袖型的人常有危机感，渴望通过新技术获得更强竞争力。如果你可以不断提出点子，刷存在感，让领袖型的人将“新点子”与你划等号，那么恭喜你，你就占据了他的心智。

在沟通中，你可以：
1.  告知上级本项目采用了哪些**全新技术**，亮点与不同在哪里。
2.  告诉他解决了什么**问题**。
3.  告诉他给公司带来了什么**好处**。

时间过得真快，又到了要说再见的时候。我们来回顾一下本集所学：

本节课中我们一起学习了：
1.  我们深入分析了**领袖型（D型）人**的特征。
2.  知道了与他们沟通的**核心要点**。
3.  明白了他们真正关心的是**效率、结果与能力**。
4.  掌握了四个与领袖型人相处的**小秘诀**。

赢得领袖型人脉的关键是，要让他们觉得你**快而且高效**。用好他们，相信你的人生处处充满机会。不要忘了，领袖型的人脉会让你的世界充满希望。

**课后作业时间：**
1.  写下你周围D特质较高的人，想想之前沟通不畅的情景。尝试用今天所学的方法重新沟通，看看有无新变化。
2.  跟你的朋友、同事或家人分享讨论D特质的应对方法，并协助他们解决与D特质高人相处时遇到的障碍。

要记得，**教是最好的学**。同时多帮帮身边的人，因为财富不是你永远的朋友，但朋友却是你一辈子的财富。
