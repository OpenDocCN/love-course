# 003.【完】chx陈海贤·家庭关系21讲：chx20疗愈：什么样的家庭才是好家庭？_note

在本节课中，我们将探讨一个核心问题：究竟什么样的家庭才能被称为“好家庭”？我们将打破对家庭“完美和谐”的静态想象，转而理解家庭作为一个动态系统的本质——**修复能力**。

---

在前面的课程中，我们探讨了家庭可能遇到的各种问题与挑战。说了这么多问题，你或许会好奇：那究竟什么样的家庭才算是一个好的家庭呢？

说起好的家庭，你或许会立刻想到夫妻恩爱、母慈子孝的场景。餐桌上，妈妈慈爱地对女儿说“妈妈最爱你”；丈夫热心地与妻子讨论全家旅游计划。

这些温馨的画面当然会出现在家庭中。但如果你认为家庭天然就应该是这样，那多少有些将家庭理想化了。这就像把奢侈品当作了日用品，认为它应该每天都有、人人都有，否则就是一种反常。

其实并非如此。如果你把镜头拉近，就会发现即使再和谐的家庭，也存在许多冲突与矛盾。那位对女儿说“最爱你”的妈妈，转头就会瞪着眼睛训斥：“不写完作业，不许看电视。”那位与妻子讨论旅游计划的丈夫，也会因妻子看一部无聊的电视剧而对她嗤之以鼻。

如果你把时间线拉长，纵观家庭的整个生命周期，便会发现每个家庭都会遇到诸多危机，经历许多惊涛骇浪的时刻。有些危机甚至关系到家庭的生死存亡。

所以，好的家庭一定包含温馨，但温馨并非家庭的全部。那么，究竟怎样的家庭才算理想呢？

曾有一位学员向家庭治疗大师米纽庆提出这个问题。米纽庆回答说：
> 一个理想的家庭，其实就是一个**有修复能力的家庭**。并没有一个家庭是没有冲突、没有问题的。只要这个家庭具备了修复冲突、解决问题的能力，那它就是一个足够好的家庭。

这很像中国人常说的“家家有本难念的经”。好的家庭并非家里的“经”容易念，而是不管它多难念，家人也会想办法一起念下去。

**一个有修复能力的家庭，就是一个足够好的家庭。**

这句话看似简单，却意义深远。它将“好家庭”的定义，从一个**静态的、理想化的模板**，转变为一种**动态的、解决问题的能力**。我们看待家庭困难的视角，也因此发生了转变。

通常，我们认为一个家庭遇到难题，就说明这个家庭不好。但如果你认同“好家庭就是有解决问题能力的家庭”，那么对家庭里发生的许多事，你就会有不一样的看法。

我曾处理过许多涉及出轨的家庭。有时我会对他们说：伴侣出轨，当然是一件伤人的事。你可以选择原谅，也可以选择不原谅。但是，**如果你选择原谅，那么“出轨”就不是这个问题的全部**。

故事的另一半是：我们遇到了一个巨大的困难，我们如何去面对、如何去解决。要知道，**家庭面对问题的反应，正是这个故事最重要的另一半**。

受伤、指责、抱怨是一个故事的版本；重新发现彼此、原谅和放下，也是一个故事的版本。它最终成为一个什么样的故事，取决于你们现在的行动。

因此，**不把家庭遇到的问题看作一种“情感的缺陷”，而看作一个需要去解决的“难题”**，这个视角常常能给家庭带来新的生机。

让我再举一个例子。

有一位姑娘曾有一段维持多年、投入很深的恋情，但因为关系过于纠缠，两人最终分手。为了早日走出痛苦，她很快开始了一段新恋情并结婚。

该如何看待这段感情和这个新家庭呢？从“理想爱情”的角度看，这段感情似乎存在瑕疵。我们通常会认为，这位女生做得不对，不应利用新男友来疗伤，这对男方也不公平。

但从“解决问题”的角度出发，你就会有不一样的看法。你会想：任何一段感情，都有其特别的开始。一见钟情是开始，两小无猜是开始，在情感撕裂后认识新欢，也是一种开始。**比开始更重要的，是两个人如何去发展和维持彼此的感情**。

我们都见过许多拥有完美开始，但最终分手的爱情故事；也见过许多从“疗伤”开始，最终却幸福终老的爱情故事。

这是说他们这段感情没有问题吗？当然不是。这位女生可能确实放不下旧情，男方也可能感到被利用。但从解决问题的角度出发，我们就不会总是盯着这些“缺陷”，而是会去看**我们能做什么**，并从中发现新的可能性。

也许这位男生能提供别人无法给予的安全感；也许这位女生能在新感情中发现什么是更可靠的关系。如果这对伴侣有办法解决他们遇到的问题，那么这段恋情就会从一个“借人疗伤”的故事，转变为一个**开局曲折但结局圆满**的故事。而那位念念不忘的前任，也会变成两人情感故事的注脚。

那么，这个新的家庭就会是一个好的家庭。

当然，这并非否认问题解决的难度。对任何一个家庭，问题都不会凭空消失。家庭问题也往往不是一个人能解决的，需要家庭成员的共同努力。

那么，善于解决问题的家庭，有哪些共通的特点呢？在咨询室里，我观察到那些具备修复能力的家庭，通常拥有以下三个特征。

以下是好家庭的三个核心特征：

1.  **能够正视问题**
    好的家庭有一种天然的诚实。他们愿意沟通遇到的事情，而不是隐藏秘密或粉饰太平。家庭问题通常有两个层面：
    *   **事实层面**：例如“丈夫要去外地工作”。
    *   **关系层面**：例如“丈夫去外地工作，导致夫妻感觉疏远”。
    我发现，越是好的家庭，越善于正视和讨论**关系层面**的问题。而越是关系紧张的家庭，越容易说“我们没有问题”。这或许是因为这些家庭既感受到问题的困扰，又不相信自己能解决，于是用善意的谎言掩饰过去，以避免焦虑。久而久之，问题就成了“房间里的大象”。
    就像一个孩子曾对我说：“我爸妈总说我们是一个幸福的家庭，很长时间我也相信。但我去同学家，看到他们饭桌上有说有笑，每个人都可以聊烦恼。而我们家吃饭永远沉默，我就开始怀疑了。”
    更极端的情况是，如果家庭中有人受伤，家人却一直掩饰，那么受伤者会对整个家庭产生怨恨。就像《你当像鸟飞往你的山》书中，哥哥长期虐待妹妹，父母却一直掩饰说“没问题”，甚至否定妹妹的记忆。妹妹所受的伤从未被承认和看见，这就成了她心中永远的痛。**如果一个问题不被承认，甚至成为家庭秘密，那么讨论和解决便无从谈起。**

2.  **形成解决问题的良性互动模式**
    不知你是否有这样的印象：有些家庭谈话，听着让人很享受。他们不一定在谈大事，但每个人都有兴趣说自己的事，也有兴趣听别人的事，家庭氛围轻松。拥有这样的互动模式，在遇到问题时，才能有彼此的理解与支持。
    但有些家庭的谈话方式，会让人感到窒息甚至想逃离。他们的对话中包含了大量控制与否定。
    就像一位在外读书的儿子向爸爸抱怨：“你说我不跟你们交心，可我跟你们讲什么呢？你跟我说话，就只会问‘学习好不好’。你关心过我在想什么吗？关心过我生活怎么样吗？”
    这位爸爸当然不是不关心儿子，只是这家人习惯了用令人不愉快的方式表达关心。当有人表达担忧时，另一人常说：“你想这么多干嘛？”对说话者而言，这或许是一种安慰；但对倾听者而言，这却像一种拒绝，仿佛在说：“我对你说的不感兴趣，别再说了。”久而久之，家里就变得无话可说。**在一个无法良好沟通的家庭里，解决问题也会变得异常困难。**

3.  **拥有良好的家庭结构**
    什么是良好的家庭结构？核心在于：**夫妻应成为家庭中最重要的子系统**，两人相互支持，共同支撑起这个家。
    在前面的课程中，我们多次提到一种失衡的结构：母亲与孩子相互依赖，而父亲找不到自己的位置。就像一位爸爸抱怨的：“在我们家，别说孩子，连养的狗都比我重要。”
    爸爸处在这种位置，他去管教孩子，孩子自然不会听。而妈妈呢，她不是埋怨孩子为什么不听爸爸的话，而是埋怨爸爸“你为什么用这种方式教育孩子”。仿佛问题不在孩子，而在爸爸身上。当妈妈批评爸爸时，实际上就消解了爸爸的权威。
    而一个好的家庭，父母会通过对彼此的支持，来塑造彼此在家庭中的权威。**只有在这种稳固的“父母同盟”结构下，家庭问题才更容易被解决。**

---

本节课中，我们一起学习了“好家庭”的真正内涵。我们认识到，**一个好家庭并非没有问题的“完美家庭”，而是一个具备“修复能力”的家庭**。这种能力体现在三个方面：**正视问题的勇气**、**良性沟通的互动模式**以及**夫妻同盟为核心的家庭结构**。记住，家庭的生命力不在于避免风浪，而在于共同修复航船、继续前行的能力。
