# 课程032：情感心理咨询技能大全（完结） 🧩
## 第06章：结构式家庭治疗—陈发展
### 09. 常见的治疗技术 🛠️

在本节课中，我们将要学习结构式家庭治疗中一系列核心且常用的治疗技术。这些技术是治疗师与家庭工作、理解并改变其互动模式与结构的重要工具。

上一节我们介绍了如何与家庭建立关系，即顺应家庭的互动模式。本节中，我们来看看在整个治疗过程中会运用到的其他关键技术。

治疗大师米纽琴（Minuchin）将治疗视为一门手艺。他认为，治疗师需要拥有一个“百宝箱”，里面存放着各种技术，以便根据每个家庭的独特情况灵活选取。结构式家庭治疗的技术主要可分为以下几大类。

以下是这些技术类别的概述：
1.  **进入和顺应**
2.  **引起并处理互动**（活现）
3.  **勾画结构**
4.  **强调和修饰互动**
5.  **明晰界限**
6.  **去平衡**
7.  **挑战无效的假设**

接下来，我们将逐一详细探讨这些技术的具体操作与理解范式。

### 1. 进入和顺应

每个家庭前来咨询时，都带着其相对固定的家庭结构和稳定的动态平衡。有效的家庭治疗需要挑战那些可能阻碍家庭实现目标的、功能不良的结构。但家庭通常会抵抗试图改变他们的人，因此治疗师起初可能不受欢迎。

家庭结构的一个重要特征是维持稳定，抵抗超出范围的变化。家庭之所以遇到困难，往往是因为旧结构已不适应。因此，治疗师首先必须解除家庭的防御，缓解其焦虑，通过顺应家庭来建立关系。

以下是进入和顺应时需注意的几个要点：
*   与家庭寒暄打招呼，进行自我介绍。
*   询问并倾听每一位家庭成员的看法，邀请他们做自我介绍。
*   对每位成员保持较高的关注度，对其想法和心理活动充满好奇。
*   通常从父母开始谈话，或让家庭自己决定从谁、从哪里开始。
*   尊重家庭的等级和结构，避免轻易评价或贴标签（例如：“妈妈你太焦虑了”）。
*   秉持理解、尊重、真诚、接纳和倾听的态度，去理解家庭对其问题与结构的看法。

这一步的核心是尊重家庭的既有互动模式，如同米纽琴所说，与家庭“共舞”。

### 2. 引起并处理互动（活现）

与家庭建立关系后，关键是要让家庭结构得以呈现。当家庭结构呈现时，我们需要引起并处理其互动模式。

家庭在咨询室现场的互动（如争吵），远比他们事后的描述更为清晰和准确。这种现场互动往往是他们惯用的解决问题方式。治疗师需要足够敏感，捕捉这些互动。

家庭成员在描述他人时，常常成为“活现”家庭模式的跳板。治疗师可以借此让家庭成员现场表达对彼此的反应。

**例如**：
*   丈夫说：“太太对孩子管教太严厉了。”
*   治疗师问太太：“你的先生说你对孩子太严厉，你怎么理解和解释？”或“你现在如何回应你先生的这个问题？”
*   治疗师问孩子：“当妈妈替你表达时，你会怎么做？”（这体现了纠缠的结构）
*   太太哭泣时，治疗师问先生：“当你太太难过时，你有什么感受？你会怎么做？”（若先生无动于衷，则体现了疏离的结构）

这项技术的目标，是将家庭内部的互动模式在咨询室现场引发并呈现出来，即“活现”。

### 3. 勾画结构

当家庭结构在现场呈现后，治疗师需要帮助家庭理解其结构。这一步的核心是将症状拓展到关系层面，用专业的理解反馈给家庭。

结构式家庭治疗的基本观点是：家庭成员呈现的问题，并非纯粹的个人问题，而是其背后家庭结构功能问题的体现。因此，治疗需要将家庭的焦点从关注个人问题，转移到关注家庭互动模式与结构上来。

以下是勾画结构的方法：
*   **观察互动并反馈**：主要基于系统式提问（针对关系的提问）。
    *   **例如**：针对孩子玩游戏的问题，可以问：“当孩子玩游戏时，爸爸妈妈你们各自会怎么办？爸爸管教孩子时，妈妈通常会怎么做？”这可以勾画出“爸爸管教-妈妈干预”的结构。
    *   **例如**：反馈说：“妈妈对孩子的关注似乎比爸爸更多。”或“当儿子生病后，你们夫妻的争吵好像减少了。”这勾画了可能的结盟或转移冲突的结构。

如果缺乏对家庭结构的假设和理解，治疗将难以推进，家庭可能始终将问题归咎于个人。

### 4. 强调和修饰互动

一旦与问题密切相关的互动结构在现场呈现，治疗师需要对其进行强调和修饰。

家庭有很多互动模式，我们需要着重强调那些与当前问题密切相关的部分。治疗师通过调整语气、情绪、态度和姿势来获得“强度”，引起家庭的警醒和重视。

**例如**：
*   对孩子说：“你能多说说，你是怎样帮助妈妈缓解情绪的吗？”（强调孩子可能过度卷入父母子系统）。
*   对厌食症女孩的母亲说：“这就是你的错。”（这不是评价，而是强烈的系统反馈，强调母亲过度干预与孩子症状的关联）。

同时，我们通过塑造家庭成员的角色来“修饰”互动，为其赋予新的意义。

**例如**：
*   对父母说：“让孩子安静一下是你们的责任。严厉的管教有时也是爱和保护的表现。”这将“不管教”背后“怕孩子觉得不爱他”的假设，修饰为“管教也可以是爱”。
*   核心是：**避免替家庭做他们自己能做的事**，相信家庭拥有解决问题的资源。

### 5. 明晰界限

结构式家庭治疗特别强调子系统的边界清晰。这项技术通常在建立良好治疗关系后使用。

明晰界限意味着重新组合子系统之间的距离和亲密度。对于不同结构的家庭，方向不同：

*   **对于纠缠的家庭**：鼓励成员保持适当距离，阻止相互干扰。
    *   **例如**：对孩子说：“你可以先让爸爸把话说完吗？”或对父母说：“可以请孩子暂时在外面休息，以便你们完成夫妻间的谈话吗？”
*   **对于疏离的家庭**：挑战回避冲突的行为，鼓励直接沟通和情感表达。
    *   **例如**：鼓励沉默的丈夫回应妻子的抱怨：“当你太太这样说时，你有什么要回应的吗？”鼓励妻子直接表达期望而非抱怨。

### 6. 去平衡

功能不良的家庭常处于失衡状态，如夫妻一方与孩子结盟。治疗师可以有策略地支持某一方，暂时打破这种平衡，为系统创造变化的机会。

去平衡并非判断是非，而是引发改变的过程。

**例如**：
*   在母子过于紧密、父亲疏离的家庭中，对母亲说：“你是什么时候开始和丈夫‘离婚’，‘嫁’给了这个孩子的？”或对父亲说：“你愿意把你的太太从儿子手中‘抢’回来吗？”
*   这种干预会引发新的互动（如夫妻开始对话），旧的平衡被打破，新的平衡在斗争或冲突中得以建立，孩子可能从三角关系中解脱出来。

### 7. 挑战无效的假设

家庭常常固守一些无效的假设（认知观念），并由此产生重复但无法解决问题的互动模式。治疗师需要挑战这些假设，提供全新看法，激活家庭资源。

**例如**：
*   父母认为孩子“多动、神经质”（无效假设），导致管教、斥责等行为。
*   治疗师可将症状“改饰”为：“孩子非常活泼，渴望关注；他特别细心，当你们争吵时他会变得安静。”（新的理解）
*   这种新的认知可能促使父母改变互动方式。
*   也可用叙事的方式，与家庭共同创造新故事。**例如**：对与母亲关系过于紧密的青少年说：“我觉得你好像没有17岁，更像一个7岁的孩子，因为大多数17岁的青少年不需要完全听妈妈的话。”这挑战了家庭关于“紧密即关爱”的无效假设。

本节课中，我们一起学习了结构式家庭治疗的七项核心技术：**进入和顺应、引起并处理互动、勾画结构、强调和修饰互动、明晰界限、去平衡、挑战无效的假设**。这些技术并非按固定顺序使用，而是始终围绕一个核心目标：**针对家庭结构和互动模式**，而非仅仅针对个人问题。希望大家能将系统观和结构式理解深入骨髓，贯穿于整个治疗实践之中。
