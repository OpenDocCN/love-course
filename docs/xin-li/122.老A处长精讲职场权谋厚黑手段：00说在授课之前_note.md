# 课程122：职场权谋厚黑手段精讲 🎓

## 00 说在授课之前

在本节课中，我们将要学习一系列职场中的权谋策略与厚黑手段。请注意，本课程内容旨在揭示特定环境下的生存法则，学习者需结合自身情况审慎思考。

首先需要声明，由于公务员身份的原因，我的声音都经过特殊处理。

一般的授课人在讲课之前，都会夸大其词地讲述自己的履历，以便让学习课程的人信服。我不自夸，我只是一名普通的公务员。

在四十岁左右的年纪，我勉强熬到正处级别。每天在省委办公厅，都是战战兢兢地服务领导。每天要做的事情，就是揣摩领导的意图，顺从领导的爱好，照顾好领导的情绪，做领导工作上的帮手，生活上的参谋。

我是二零零零年大学毕业的，开始时作为选调生，在一个镇政府工作。随后到了县委办公室，再到市级部门，最后到了省委办公厅。期间曾被借调到国家部委一段时间，因为家庭原因，我放弃了留在部委的机会，回到了省委办。

这段履历说起来非常简单，几句话就能概括，我却熬了十八年。我是一个没有背景的人，我的父母以及岳父母，都是老实巴交的社会底层。他们没有办法在职场上指点我，甚至他们很多固化的意识，还会拖我的后腿。

所以我一路走来，基本上都是自己在奋斗，自己一步一步力争上游。我参加工作十八年来，浸淫于官场，潜心于研究，权谋厚黑。我自己也一步步因此得到了提拔重用，从一个镇政府的普通办事员，到现在的省委办公厅处长。

我有一整套的权谋经验想交给你们，但是你们一定要有洗心革面的气魄，要有重新再来的勇气。因为你们从小接受的教育是阳的教育，但这个世界是险恶的。

我的课程就是想让你变成一头狼，变成一头阴险狡诈，但是外表忠厚的狼。我会教你在职场上所有猎杀的手段。

以下是本课程将涵盖的部分核心手段：

*   巧妙不动声色地搞掉你身边的小人和绊脚石。
*   有效地捧杀小人。
*   挑拨同事关系。
*   虚让功劳。
*   攀附权贵。
*   树立威信。
*   敲打他人。
*   拉帮结派。

朋友们，在学习之前，请先把所有的道德标准通通扔掉，请扔掉那些正大光明的教育。孔孟让你诚实待人，淡泊名利等等，都是骗孩子的，都是教你成为一个奴性的人。

而我教给你的，是帝王术中的权谋厚黑，是教你成为支配自己命运的人。我就简单讲这几句，算做开课之前的前言。
