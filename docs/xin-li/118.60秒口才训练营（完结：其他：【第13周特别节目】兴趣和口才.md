# 118.60秒口才训练营（完结：其他：【第13周特别节目】兴趣和口才

在本节课中，我们将探讨兴趣与口才之间的深层联系，并学习如何将兴趣转化为热爱，从而提升你的表达自信与感染力。

## 📝 本周主题：如何陈述你的兴趣

上一节我们介绍了本周的训练主题。本节中我们来看看具体如何向他人陈述你的兴趣。

本周的练习主题是关于如何向别人陈述你的兴趣。我聆听了所有学员提交的关于兴趣的节目，发现大家的兴趣各不相同。有人喜欢跑步，有人喜欢看书，有人喜欢游泳，有人喜欢玩魔方，有人喜欢练字。世界非常有趣。

回到本周的练习主题，我们有一个常见场景：在面试过程中讲述你的兴趣。这里有一个关键的不同点。我并非让大家简单地讲述练习兴趣的过程。更重要的是，在讲述兴趣的结尾，你需要陈述这个兴趣带给了你什么。可能是某些能力的提升，可能是某些品质，或者是你的一些感悟的积淀。例如，你懂得了坚持的意义。

因为你是在面试。你不是在向面试官推销这个兴趣的好处。你推销的其实是你自己，是通过这个兴趣在推销你自己，而不是在推销这个兴趣。这是非常不一样的。我们后边在专门练习销售时，例如，我会给大家一个兴趣，让你向别人推荐为什么也要拥有这个兴趣。这两者是不一样的。大家一定要搞清楚这个逻辑。

我听了很多同学的练习，最大的问题就在于此。你到底是在讲述这个兴趣，还是在向别人推荐这个兴趣，还是你通过这个兴趣在销售你自己？这是我们本周练习的根本所在。

## 🔗 兴趣与口才的深层关系

上一节我们明确了陈述兴趣的核心是推销自己。本节中我们来深入探讨兴趣与口才之间看似不直接却至关重要的联系。

今天我想花更多时间聊一聊兴趣和口才的关系。有的同学可能会觉得兴趣跟口才有什么关系。表面上来看，它确实没有那么大关系。我也不想讲那么多技巧的东西。其实我想讲一个非常重要的话题，这个话题对我的人生影响非常大。

这是一个我在以前学职业规划时学到的概念。首先，每个人都会有很多兴趣，但很少能成为自己的工作。我们都听过一句话：如果你的工作就是你的兴趣，那是人生一件多么有价值的事情。但是如何能让兴趣跟工作产生关系？当然，兴趣跟口才又有什么关系？这是我后边要着重花一点时间来聊的。

我以前学职业生涯规划时，跟老师学到三个关键词。我们大部分人经常说的都是“兴趣”。其实兴趣背后还有两个关键词：一个叫“乐趣”，我相信你听过；另一个词叫“志趣”，就是一生以此为志。

这三个词的关系是什么？平时你可能有很多兴趣，比如你喜欢看书，喜欢画画等等。一个人很容易对一些事物产生兴趣，兴趣也很容易失去。因为兴趣有时只是“一时兴起”，你可能会对一个东西感兴趣。就像我刚才听了一个同学的练习，他讲就是因为有一次在电视上看见《最强大脑》里玩魔方，因此对魔方产生了兴趣。但是也可能过一段时间就不感兴趣了。

乐趣就不一样。你什么时候会有乐趣？比如你去KTV唱歌，你在朋友间唱得比较好，俗称“麦霸”，你就会对唱歌有乐趣，因为你获得了成就感。这里乐趣跟兴趣最大的区别是，你中间具备了一定的能力，能把兴趣做得比较好，就会获得乐趣。

我可以举一个自己的例子。比如我最开始是2014年8月开始玩电台的。我从来没有过什么主播梦，就是因为一个朋友推荐，我就感兴趣，回去研究。我最开始录的全是关于销售类的电台节目，录了好久也没有太多粉丝。后来我做“说话改变世界”这个电台时，可能选题越来越好，讲的内容也好一些，包括对于电台的推广、运营、起标题都越来越好了。具备一些能力之后，粉丝不断增长。我记得有一段时间每天大概增长一百多人。这种不断的反馈让我越来越发现录电台的乐趣所在。这就是增加了能力。

第三个叫“志趣”。志趣是什么意思？就是你在做这件事的过程中有成就感，但如果你又能发现其中的意义所在，加入你的价值观，让你找到意义感，你就会更加热爱这件事。所谓的志趣，就是能让你热爱的事情。

我给大家举个例子。比如我自己平时打台球（桌球），我觉得打得还不错。我没有花太多时间精力，但在业余选手中还算不错。我也有乐趣，但并没有花太多时间在台球上。因为我在这个事情上找不到那么强的价值观和意义感。但是录电台这件事，后来我找到了越来越多的意义在里面，所以我就把它发展成了志趣，或者说我的一个热爱。

比如当我能看到很多同学因为加入训练营，从不敢讲到能讲出来，然后慢慢讲得顺，甚至有的同学渐渐达到讲出彩，现在有一些伙伴已经加入我们的“想说团”，完成了170个练习节目，已经开始享受说话。这是我特别开心的事情。这种开心不是金钱或掌声能换来的，而是那种满足感。这就会达到一种志趣，我甚至想以此为一生的一个事业，我会热爱它。这就是从兴趣到乐趣，再到志趣，一步一步走过来的。

它是怎么走过来的？因为兴趣加入了，你记住一个公式：

**兴趣 + 能力 = 乐趣**

**乐趣 + 价值观 = 志趣（热爱）**

当你明白这个逻辑之后，你会发现应对兴趣的策略应该是什么样的。其实每个人对很多事物都可能产生兴趣。但是因为你的时间和天赋有限，不可能在所有兴趣上都投入那么多时间精力去练习。记住，从兴趣到乐趣，需要不断的刻意练习，产生能力之后，你才会产生成就感，才会变成乐趣。这就需要投入时间。但人的时间和精力有限，所以你需要选择聚焦，在一些有限的兴趣上投入时间练习，变成乐趣，获得成就和自信。

## 💎 兴趣如何影响口才

上一节我们理解了兴趣转化为热爱的路径。本节中我们具体看看这个过程如何最终影响你的口才。

那它跟口才的关系是什么？在我看来，兴趣是老天赋予你的一种天赋和资源。有的人天生对画画感兴趣，有的人天生对体育感兴趣。没有理由的兴趣就是兴趣，就是你会不由自主地想投入时间，去关注那个领域。那是老天赋予你的一些天赋和资源。这会决定你在某个领域持续投入，比如看那个领域的书、资讯，这都会给你产生大量的输入和思考。

如果你能投入一些时间和精力在那个领域练习，你就会获得成就和自信。当你获得成就和自信，这对口才有非常大的帮助。这是非常重要的。

就像我刚才听了一个同学的节目，他因为在家长会时唱了一首歌，获得了很多老师和其他同学家长的赞赏，因此产生了很大的自信。你说这种自信对他的兴趣、对他的口才有没有很大帮助？我想肯定是有的。

进一步说，一旦你能加入价值观，成为你的志趣和热爱时，你的内心会有很多话想去说、想去分享，因为那已经成为你的热爱了。当一个人有话想说的时候，再加上一些技巧和训练，口才就变好了。

所以，兴趣跟口才的关系，表面上好像没有那么大直接关系。但是因为兴趣，它可能深远地影响一个人，可能让一个人产生乐趣、志趣，产生他的热爱。而口才是一个人内在的知识、思维和很多积淀的外显能力。兴趣可能决定了一个人底层很多东西。口才只是一个人外边显现出来的东西。你说兴趣和口才是不是产生了许多深远的、非直接的影响关系？但其实这是非常重要的。

而且，兴趣也可能会影响你的工作、生活，甚至最终发展成为热爱。发展成热爱之后，就可能成为你一生的事业。

## 📌 核心要点总结

本节课中我们一起学习了兴趣与口才的深层联系，以及如何将兴趣发展为热爱。

以下是本课的核心要点总结：

*   **陈述兴趣的核心**：在面试等场合陈述兴趣时，重点在于说明兴趣带给你的能力、品质或感悟，本质是**通过兴趣推销你自己**。
*   **兴趣发展的三阶段**：
    1.  **兴趣**：一时的好奇与关注。
    2.  **乐趣**：通过**刻意练习**获得能力，并从中得到成就感。公式为：**兴趣 + 能力 = 乐趣**。
    3.  **志趣（热爱）**：在乐趣中加入个人**价值观**，找到意义感。公式为：**乐趣 + 价值观 = 志趣（热爱）**。
*   **兴趣对口才的影响**：
    *   兴趣是你的天赋和资源，引导你持续输入和思考。
    *   将兴趣发展为乐趣和志趣的过程，能带给你**成就感和自信**，这是好口才的重要基础。
    *   当你对某事充满热爱（志趣），你会产生强烈的分享欲，这是提升口才的内在动力。

兴趣是老天赐予你最宝贵的财富。花一点时间和精力，把它练成乐趣，最终找到你的热爱。你的口才也会跟着变好。我希望这一周的分享能对大家有所启发。因为我不想讲太多口才的技巧，其实底层的东西往往是最重要的，就像大树的根才是最重要的。表面的技巧在网上能找到很多。我希望大家能早日找到你的热爱，然后早日可以跟我分享你的喜悦。
