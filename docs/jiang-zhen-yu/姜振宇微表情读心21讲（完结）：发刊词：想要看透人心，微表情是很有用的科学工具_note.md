# 姜振宇微表情读心21讲（完结）：发刊词：想要看透人心，微表情是很有用的科学工具_note

在本节课中，我们将要学习微表情的基本概念及其作为科学工具的实用性，并了解本课程的研究背景与核心原理。

我是姜振宇，欢迎来到我的微表情课堂。微表情是我的专项研究。我很高兴能与专业的心理学平台“意心理”合作，为大家带来这门微表情课程。

我从2007年攻读博士学位时开始研究微表情。我曾先后与公安机关、检察机关、纪检监察机关等非常重视证据和逻辑的司法机关进行合作。我的研究对象是实际案件中的各类嫌疑人，观察他们说谎话、说实话、伪装表演，以及真情流露的时刻。因此，我的研究积累了大量的实际素材和关于微表情的理论模型，并取得了很好的研究成果。

后来，机缘巧合之下，我开始专门参与和微表情相关的电视节目。2011年，江苏卫视的节目《非常了得》让更多观众了解到，微表情在国内也有正经的学术研究，而不仅仅是电视剧里的台词总结或互联网上的各种伪科学。之后，我陆续参加了民生节目《四大名助》、脑力节目《最强大脑》以及目前正在参与的情感类节目《非诚勿扰》。在这些节目中，我通过表情来解读不同的人、不同的事和不同的心思，也因为解读过于透彻而被冠以“钢铁直男”的标签。

实际上，必须有鲜活的人和关切的事，才会让人出现真切的微表情和肢体动作。正是因为见了这么多形形色色的普通人和嫌疑人，才使我对微表情的理解更加深刻和透彻。

本次与“意心理”的合作，也希望能为大家带来专业但不枯燥、清晰而有趣的课程。我希望大家能够支持，也希望更多人能理解微表情的本质。

其核心原理其实非常简单，可以用一个基本公式来概括：

**微表情的本质 = 言语内容 ≠ 微小表情/动作**

如果一个人说的话和他的微小表情或动作不一致，那就说明可能存在问题，需要你小心不要被骗倒。

想要看透人心，微表情是很有用的科学工具。大家需要动脑筋来学习，我在这里等你。

本节课中，我们一起学习了微表情课程的研究背景、实践基础及其核心判断公式。我们了解到，微表情是一门基于大量实际观察的科学，其关键在于识别言语与非言语信号之间的不一致性。
