# 姜振宇微表情读心21讲（完结）：09.悲伤表情：怎样看出对方是真悲伤还是在演戏？ 😢

在本节课中，我们将要学习如何识别悲伤情绪的真伪。我们将深入探讨悲伤情绪的心理根源，并详细拆解其对应的面部表情特征，特别是眉毛、眼睛和嘴巴的细微变化。通过学习，你将能够分辨出一个人是真情流露，还是在用悲伤进行表演。

## 悲伤情绪的心理根源

上一节我们介绍了恐惧情绪，本节中我们来看看悲伤。悲伤类情绪与恐惧在面部结构上有相似之处，但心理动因截然不同。

在中文语境里，“悲伤”一词通常指代非常难过、伤心的状态。然而在现实生活中，许多状态在学术上都可归为悲伤类情绪。

以下是生活中常见的、属于悲伤类的情绪状态：
*   无奈
*   愧疚
*   后悔

这些情绪有一个共同的核心特征：**一个坏的、损失性的结果已经发生**。这个损失可能是精神上的、物质上的，或是社会规则上的。关键在于，当事人认为自己**无力改变或挽回**这个局面。

因此，我们可以用一句话总结悲伤的心理状态：**无力挽回的损失**。

## 悲伤表情的核心特征：眉毛与眼睛

理解了心理动因后，我们来看看它如何体现在脸上。悲伤表情的核心特征集中在眉毛和眼睛的组合上。

以下是悲伤表情在眉眼区的关键动作：
1.  **眉头向上蹙起**：这与恐惧表情的起始动作相同。
2.  **眼睛逐渐闭合**：这是与恐惧表情的关键区别。恐惧时眼睛会睁大，而悲伤时眼睛会闭合。闭合得越小，通常表示越难过、越伤心。

这个组合可以总结为一个简单的公式：**眉头上扬 + 眼睛闭合 = 悲伤**。

当他哭得越厉害，感到越无力、越沉沦时，眼睛就越难睁开，但眉头却会高高扬起。你可以做一个有趣的实验：让一位平时坚强的同事唱一些表达弱势、受苦受难角色的悲伤歌曲（如《你把我的女人带走》、《冰雨》）。只要他投入演唱，很快就能观察到“眉头上扬、眼睛闭合”的表情。接着立刻换一首激昂的歌曲（如《男儿当自强》），这个表情组合便会立刻消失。

**为什么悲伤时眼睛要闭合？** 因为当人处于“无力挽回的损失”这一瞬间，他的注意力完全集中在内心的损失上。他不需要关注外界新的视觉信息输入，因为当前这件损失（无论是亲友离去、事业受挫还是财产损失）对他来说就是最大的事。

## 如何判断大哭的真伪？

当一个人嚎啕大哭、声音很大时，我们需要结合嘴巴的动作进行整体判断。在生活中，无论是面对情绪激动的嫌疑人，还是莫名哭泣的亲友，判断其悲伤的真伪都至关重要。

以下是判断大哭真伪的两个核心面部特征：

**第一个特征是眉毛。** 如果一个人在哭的时候，**眉毛没有向上扬起**，那么他的哭很大可能是假的。许多演员在表演极度伤心时，会错误地将眉毛向下皱紧，而不是本能地上扬。

**第二个特征是下嘴唇的形状。** 如果是真的大哭，为了配合剧烈的呼吸，嘴巴会张开，并且下嘴唇会呈现一个独特的形状。

真哭时，下嘴唇会呈现一个**波浪形的“W”形**。这个形状从嘴角开始，下唇会下滑到谷底（微微露出一点后槽牙），然后向上扬起至嘴唇中部（挡住下牙），接着再次向下滑，形成波浪。这个“W”形是由嘴角的颈阔肌、连接下唇的降下唇肌和颏肌共同作用形成的，是一组难以故意做出的联动。普通人甚至需要多年练习也难以刻意模仿，但只要真哭，这个动作就会自然出现。

因此，观察大哭者下嘴唇是否呈 **`W形`** ，是判断其情绪真伪的强有力指标。

## 悲伤的衍生表情

如果不是嚎啕大哭，只是有些委屈或难过时，表情又会有所不同。

此时，嘴巴通常会呈现我们在第四课（嘴部课程）中讲过的“否定嘴”形态。这主要是由于**颏肌和降口角肌的收缩**造成的。

## 课程总结与思考

本节课中我们一起学习了悲伤情绪的核心知识。

我们首先明确了悲伤的心理状态是 **“无力挽回的损失”** 。随后，我们详细解析了悲伤的面部表情特征：
*   **眉眼特征**：**眉头上扬** 配合 **眼睛闭合**。
*   **大哭特征**：真实的大哭伴有下嘴唇的 **`波浪形（W形）`** 。
*   **衍生特征**：轻微的悲伤可能呈现“否定嘴”。
