# 姜振宇微表情读心21讲（完结）：21.呼吸反应：如何找到刻意面无表情的对手的弱点？_note

在本节课中，我们将要学习微表情分析的最后一个高级技巧：呼吸反应。当对手能够完美控制面部表情和肢体动作时，我们将探讨如何通过观察其呼吸的细微变化，来洞察其内心的真实状态。

---

## 课程背景与核心问题

上一节我们介绍了如何通过眼睛的运动来分析对手。本节中我们来看看，当对手连眼睛的运动都极力控制，甚至完全面无表情、肢体静止时，我们该如何应对。

你的对手可能“面沉似水，城府极深，老奸巨猾”。在对抗时，他没有表情，你看不到扬眉、眼睛稳定不睁不闭，嘴角也没有任何变化。这时你无法分析其表情。

当时钢铁老师提出了一个思路：人可以主观控制肌肉不动，不给对手留出观察的破绽。但是，没有任何一个活人可以不动眼睛，所以我们才需要分析眼睛的运动。然而，仅分析眼睛无法得到全面信息。如果对手只有眼珠在动，你只能根据**注视时间、转移的频率和方向以及眨眼的变化**来判断一些神经系统的状态指标，但无法分析其复杂的思路。

那么，比眼睛运动更基础的指标是什么？聪明的同学已经想到了：呼吸。

没有一个活人可以不呼吸。所以，即使他什么都不动，他的呼吸也是一种可以观察到的外显动作。如果呼吸上出现了变化，我们就可以推断出一些结论。今天我们就讲解这件事。

这已经到了非常高级的“斗法”阶段。如果你能遇到一个什么动作都没有的对手，说明你的水平也很高了。

## 呼吸的生理学原理

在探讨如何观察之前，我们首先需要理解呼吸的基本作用。

呼吸的作用是为人体吸入氧气，并将体内代谢产生的二氧化碳呼出。公式可以简化为：
**吸入 O₂ → 身体代谢 → 呼出 CO₂**
这个过程保证了体内能量的持续燃烧，维持肢体和内在机体的稳定运行。

由此我们可以得知一个常识：如果体内的代谢（各项功能的工作）变得兴奋、密集、高能耗，那么身体需要的氧气就会增多，产生的二氧化碳也会增多。因此，客观上会需要**吸入更多氧气，呼出更多废气**。

反之，如果一个人的身体接近低功耗状态，没有任何兴奋的神经、肌肉或内脏表现，甚至连血液循环都进入低沉期，那么整个身体的代谢就会下降，对氧气的客观需求就会降低，呼出的废气也会相对减少。

这套简单的生理物理学基本现象，决定了我们观察对手呼吸的方法。

## 如何观察与分析呼吸反应

以下是观察和分析对手呼吸反应的三个核心步骤。

### 1. 建立呼吸基线

首先，你需要观察并记录对手在平静状态下的“呼吸基线”。

你可以用心数一下他呼吸的频次和单次呼吸的时长。同时，观察他呼吸时胸腔的起伏程度。这些外显指标可以作为一项极其客观的物理指标记录下来。

**代码示例（伪代码，描述观察逻辑）：**
```
观察开始：
    记录初始呼吸频率 = 数60秒内的呼吸次数
    记录初始呼吸幅度 = 观察胸腔起伏的明显程度（高/中/低）
基线建立完成。
```

### 2. 识别呼吸的兴奋性变化

接下来，你可以开始与对手进行互动（提问、回答、辩论等）。无论他是否说话，重点观察他的呼吸频率和幅度是否产生变化。

如果一个人内心变得兴奋，需要更多氧气，通常有两种表现方式：

*   **深呼吸（幅度增加）**：单次吸入和呼出的气体量增大。例如，运动员上场前感到紧张时，可能会扬起头，用鼻腔或口腔深深吸一口气。
*   **急促呼吸（频率增加）**：呼吸幅度不变，但单位时间内的呼吸次数增加。例如，剧烈运动后，身体会本能地进行快速浅呼吸以满足急需。

**核心结论**：在安静的对抗中，无论观察到以上哪种现象，都可以推断：**他的身体内部比他外表呈现出来的要兴奋很多**。这种呼吸增加是生理性的，难以完全抑制。

### 3. 识别呼吸的抑制与低沉变化

与兴奋相反，我们也要能识别出呼吸减弱所代表的状态。

如果你观察到对手的单次呼吸幅度减少（胸腔起伏变弱），或者能明显感觉到他的呼气比吸气更长、更多，说明他身体机能需要的氧气在减少，代谢水平下降。

结合我们之前课程所讲，这种状态常伴随着“无力感”。由此可以判断，这个人当前的神经系统处于压抑、不兴奋、不想折腾的状态。多半情况下，他可能认输了、认怂了或放弃了。

## 高级技巧：识别对抗性的深长呼吸

在比较克制的情境中，如果一个人不想让你看出他的兴奋或紧张，他会刻意控制呼吸。

这时，你可能会观察到一种 **“深长呼吸”** 。它与普通的深呼吸不同：
*   **普通深呼吸**：幅度大，可能伴随仰头、挺胸，目的是快速摄入大量氧气。
*   **对抗性深长呼吸**：表面幅度没有明显增加，没有大动作，通常用鼻腔进行。但仔细观察会发现，他一次吸气和呼气的**时长变得很长**。外表看似平静，说话音量和频率也无明显变化。

**核心结论**：如果你观察到这种小心翼翼、刻意控制的深长呼吸，可以直接推断出两个字：**对抗**。为什么？因为他在用尽量掩饰的手法控制呼吸节奏，这恰恰暴露了他想欺骗和掩饰真实状态的对抗心态。

## 知识延伸：深呼吸为何能缓解紧张？

在课程结束前，分享一个相关的常识：为什么人在紧张时，深呼吸能帮助平静？

其原理与我们前面讲的内容相关。大脑重量约占体重的2%，但其耗氧量却达到全身的**20%-25%**。如果大脑供氧不足，就会出现专注力、认知力和理性思考能力减弱（例如上台演讲忘词），同时还容易情绪不稳定（暴躁、易怒、易崩溃）。

紧张的本质，是情绪化决策暂时压制了理性思维。当你进行深呼吸时，能瞬间提升血液中的含氧量。大脑获得更多氧气后，思考能力会相对增强，理性思维便有可能重新压制住情绪的慌乱，让你能记起背过的词，理清策略，进行正常表达。

公式简化表示：
**紧张 → 理性思维被压制**
**深呼吸 → 提升血氧含量 → 大脑功能恢复 → 理性思维重新主导**

## 课程总结与重要原则回顾

本节课中，我们一起学习了如何通过呼吸反应来判断对手的内心状态：
1.  **呼吸加剧**（频率加快或幅度增大）通常意味着**内心兴奋、紧张或处于对抗状态**。
2.  **呼吸减弱**通常意味着**情绪低沉、放弃或认输**。
3.  **刻意控制的深长呼吸**往往暴露了**隐藏的对抗心态**。

在整套肢体动作分析课程即将完结之际，需要再次向大家重申两个至关重要的原则，这与第一课的内容遥相呼应：

1.  **社交行为不定真伪**：大量行为是对方故意做给你看的。切勿贸然对这些刻意行为进行直接分析。
2.  **必须先建立基线**：无论是呼吸、动作角度、幅度还是速度，都必须先观察并记录对方在自然或平静状态下的“习惯基线”。所有分析都应基于对“违背基线的异常运动”的观察。

在工作中，你应该运用我们讲过的测试方法，通过提问、施压、安慰等多种互动方式，去激发对方产生“违背基线的异动”。然后，再结合本套课程所教的肢体动作分析和微表情知识，进行综合判断，从而更准确地理解对方的真实感受，避免误会，提高沟通效率。
