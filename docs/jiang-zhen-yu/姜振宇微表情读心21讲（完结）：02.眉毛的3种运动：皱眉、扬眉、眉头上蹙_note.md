# 姜振宇微表情读心21讲（完结）：02.眉毛的3种运动：皱眉、扬眉、眉头上蹙_note

在本节课中，我们将要学习眉毛在微表情分析中的核心作用。眉毛不仅是面部的重要器官，更是构成和定性人类情绪的关键部分。我们将详细解析眉毛的三种基本运动及其对应的心理含义。

眉毛长在脸上，除了美观，还有两个重要作用。第一个是生理作用。当额头有汗水或下雨时，眉毛能在眼睛上方形成具有导流作用的屋檐效应。这样，汗水和雨水就能被导向两侧，离开眼睛的视觉区域。这既保护眼睛免受侵害，也保证了视线的清晰。

第二个重要作用是参与表情构成。眉毛参与所有表情的构成，并起到关键的定性作用。

那么，眉毛能做几种动作呢？其实只有三种。

以下是眉毛的三种基本运动：

1.  **皱眉**：将眉毛向中间皱起。这个动作通常出现在遇到困难、感到为难、疑惑、压抑、讨厌或生气时。公式可表示为：**皱眉 ≈ 负面情绪（困难/厌恶/愤怒）**。
2.  **扬眉**：将眉毛高高扬起。这个动作主要代表两种含义：一是**惊讶**（“怎么回事？”）；二是**社交默契或挑逗**（“你懂的”）。因此，扬眉既可能是自发的情绪反应，也可能是刻意的社交行为。
3.  **眉头上蹙**：眉毛靠近内侧的两端向上扬起。中文里专门用“蹙”来形容这个动作。它构成了人类**恐惧**和**悲伤**情绪的表达。例如，唱悲伤情歌时，人的眉头常会不自觉地向上蹙起。

上一节我们介绍了眉毛的三种基本运动模式，本节中我们来看看它们在具体情绪中的表现和应用，为入门级解读打下基础。

以下是三种眉毛运动与核心情绪的对应关系：

*   **惊讶表情中的扬眉**：眉毛上扬，帮助睁大眼睛，以获取更多视觉信息。这是惊讶表情的典型特征。
*   **厌恶与愤怒表情中的皱眉**：在皱眉肌和降眉间肌作用下，双眉向中间下方皱紧。这表达了对视觉信息的重视、反感、憎恨或进攻意图。公式可表示为：**皱眉 + 特定面部组合 = 厌恶/愤怒**。即使在现代文明社会，面对棘手难题（如复杂的数学公式）时，人也会因感到麻烦和压力而皱眉。
*   **恐惧与悲伤表情中的眉头上蹙**：在额肌中束的提拉下，眉头向上扬起。这个动态**只会**出现在恐惧和悲伤情绪中。
    *   **恐惧**：眉头上蹙常配合眼睛睁大（惊慌失措的样子）。
    *   **悲伤**：眉头上蹙常配合眼睛闭小或紧闭（失声痛哭的样子）。

关于眉头上蹙，有一个重要的历史观点需要了解。心理学家保罗·埃克曼在其著名的面部运动编码系统（Facial Action Coding System, **FACS**）中曾指出，双眉头上扬是一个非随意的动作，即不能被人主观控制。因此，一旦观察到这个迹象，就可判定对方产生了恐惧或悲伤。

然而，这个结论现已被修正。因为控制眉头上扬的额肌中束是骨骼肌，而所有骨骼肌理论上都可以通过训练被意识控制。所以，经过练习，这个动作是可以被刻意表演出来的。

本节课中我们一起学习了眉毛在微表情中的核心地位。我们了解到眉毛具有生理保护和表情构成的双重功能，并重点解析了它的三种运动——**皱眉**、**扬眉**和**眉头上蹙**——分别与厌恶/愤怒、惊讶/社交、恐惧/悲伤等情绪紧密关联。掌握这些基础知识，是解读面部情绪的起点。
