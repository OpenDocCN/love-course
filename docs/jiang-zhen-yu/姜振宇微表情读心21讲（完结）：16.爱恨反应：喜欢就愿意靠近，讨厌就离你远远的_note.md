# 姜振宇微表情读心21讲（完结）：16. 🫂 爱恨反应：喜欢就愿意靠近，讨厌就离你远远的

在本节课中，我们将要学习“爱恨反应”。这是一种通过观察人与人之间身体距离的变化，来推断其内心好恶的微表情技术。我们将拆解其核心规律，并学习如何通过简单的测试来辨别真伪。

## 概述：什么是爱恨反应

爱恨反应描述的是心理距离与身体距离之间的关联。这种反应并不复杂，也没有庞大的故事背景。我们只关注微小的距离变化，这种变化完全体现了人与人之间心理距离的转变。

举个例子，你应该有家人。无论是父母、配偶还是孩子，你都有这样的体会：当你特别高兴、特别亲近时，比如孩子考上好大学或家人手术康复，你会表达热烈的拥抱或长时间的亲吻。这是喜欢一个人的表现，是愿意与一个人待在一起的表现。

但同样是这位家人，也有让你非常生气的时候。当他们真的惹急你，让你气得想吵架时，你还愿意去拥抱或拉他们的手吗？你恐怕恨不得自己走到小屋里待着，或者气势汹汹地想干一仗。这时，任何肢体上的亲密动作都不会表达了。

世界上最亲近的家人，也会因为情绪上的喜欢或不喜欢，导致肢体距离变得非常亲近或非常排斥。这就是爱恨反应。

## 爱恨反应的两大子类

爱恨反应具体可以拆分为两大类子反应：亲近反应和逃离反应。

### 1. 亲近反应：喜欢你就靠近你

亲近反应表达了他喜欢你。如果一个人愿意主动地和你靠近，有两种可能。

以下是两种可能性的详细说明：

*   **第一种可能：刻意表演的亲近。** 这是一种设好的局。对方控制自己的行为表达亲密，想告诉你“兄弟，我跟你关系铁”。例如，一把搂住你的肩膀使劲摇晃，或一把握住你的手使劲地握。这种热情的江湖表达是可以被训练出来的。即便他心情不好、非常讨厌你，在该有这些动作时，他依然能做出来。
*   **第二种可能：发自内心的喜欢。** 对方是真的很喜欢你。因为异性之间觉得亲近，或同性之间觉得仰慕、尊重，也会愿意表达出肢体距离的靠近。例如，在地铁上，两个看对眼的人会比其他陌生人更愿意凑近了站，甚至挨着站。

那么，如何破解第一种刻意表演的亲近呢？这是本节课的一个关键点。

如果你不确定一个人是真心喜欢还是假装热情，你可以用相同或相反的方式进行反测试。当我们对一个行为的意义产生质疑时，至少要进行两次以上的测试才能鉴定其真伪。

以下是两种测试方法：

*   **方法一：以热情回应热情。** 你也刻意控制自己的行为，去搂他的肩膀，把嘴凑到他耳边，特别亲密地说一些恭维话。观察他的反应。通常，如果对方不是真心，在你这一顿“热情”之下，眼神里可能会透露出警惕，肢体上可能会出现闪避、冻结或退缩。这些排斥热情的反应是验证真伪的第一道线索。
*   **方法二：以疏远回应热情。** 你扮演一个偏偏君子，有理有据地略微远离他一点。他凑近，你就把头远离；他搂你肩膀，你就把他的手掰开并挪开身体；他找你喝酒，你就表达拒绝。观察当你拒绝他的那一瞬间，他的热情是否立刻被扑灭。

这两道题测下来，基本上什么样的老狐狸都会露出破绽。因为：
*   一个真心喜欢你的人，当你反过去喜欢他时，他会更加高兴和愉悦。
*   一个真心喜欢你的人，如果你有理有据地进行了躲避和排斥，他会感到不好意思和愧疚，而不会出现进攻性的敌意。

所以，不管从哪个方向测试，你都可以瞬间测出对方的热情是真是假。

### 2. 逃离反应：讨厌你就远离你

上一节我们介绍了亲近反应，本节中我们来看看与之相反的逃离反应。

如果一个人真的不喜欢你，你凑近他，他会离你远一点。

以下是逃离反应在不同情境下的表现：

*   **站姿：** 可能会向后移动脚步，向旁侧转身，或向后仰身体。
*   **坐姿：** 由于身体重心被椅子固定，他可能只能够躯干后仰，把脸和你的距离拉开。

最经典的场景是：两个人隔着一张桌子热火朝天地聊天，突然因为价钱或方案谈不拢，其中一个人会推开桌子，把身体向后仰，同时减少语言输出量。那个向后仰的身体，就是经典的逃离反应。

生活里的例子也很多。比如在公交车上，你凑近一个姑娘站得很近，甚至手臂挨到一起。如果周围空间足够，那个姑娘的正常反应是躲开你，挪到别处去。如果她没有躲开，且周围空间很大，那可能说明她也对你有好感。

## 核心规律与重要前提

讲到这里，我们来进行总结。

我们用学术化的语言来总结一个规律：**身体距离和心理距离成正比**。

用公式表示就是：
**心理亲近度 ∝ 身体接近度**

一个人越喜欢另一个人，他就越希望靠近，越允许自己靠近，甚至非常期望地主动靠近。而一个人越不喜欢（如讨厌、害怕、难过）另一个人，他就会想办法离这个人远远的，至少是拒绝接近，包括语言上也会减少交流。

这就是爱恨反应的总规律：二者成正比。

但是，我们必须加上一个重要的前提原则：**在没有外力拘束的情况下**，身体距离才一定和心理距离成正比。

外力拘束分为两种：

1.  **物理局限性：** 例如公交车人特别多，挤得像沙丁鱼罐头。这时你和别人挨着，不代表心理亲近，只是物理空间决定了只能如此。
2.  **规则局限性：** 例如上课或开会时，座位是固定的，领导指定你和讨厌的人坐在一起。在这种情况下，即使你心存喜恶，也只能听从规则的限制，无法自行选择距离。

在这两种外力制约下，不能直接用肢体距离来推理心理距离。

## 如何应对逃离反应

如果出现了逃离反应，例如对方聊着聊着身体往后挪了，尽管脸上还带着微笑，举止也得体礼貌，你就要注意了。

因为逃离反应背后是厌恶性的情绪。而厌恶情绪一旦开始滋生，最佳的应对策略是赶紧刹车停止，千万不要再增强和积累。

厌恶情绪是人和人之间破坏性最强的情绪。如果对方一旦开始讨厌你，那么接下去你说的任何一句话，他都会站在对立和嫌弃的立场上选择性地听，甚至不听，慢慢找到越来越多讨厌你的理由，最后给你一个否定的结论。

所以，如果你发现对方肢体距离拉开，同时脸上出现微表情课程里讲过的厌恶线索（如眉毛皱起、眼睑闭合、上唇提升、鼻唇沟加深），你一定要赶紧减轻当前的刺激源，缓缓退出来，不要再说这个话题了。

如果这个话题（如合同价格）是你必须谈的核心问题，那么不妨开诚布公地把它扔到桌面上，公开进行对立性的争执。**把厌恶提升成愤怒是可以的，把厌恶停止掉，换一个话题，取消刚才的厌恶，也是可以的。** 就是别在别人讨厌的地方不断加强、反复和强调，那样最后吃亏的一定是你自己。
