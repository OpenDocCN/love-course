# 姜振宇微表情读心21讲（完结）：04. 嘴的4种运动 🗣️

在本节课中，我们将要学习人类嘴部的运动形态，并了解如何通过观察嘴部的特定动作来解读其背后的情绪含义。嘴是面部表情的重要组成部分，其运动虽然复杂，但我们可以通过梳理几种核心形态来简化分析。

## 嘴的功能概述

上一节我们介绍了眼睛和眉毛的运动，本节中我们来看看嘴。首先，我们需要理解嘴的基本功能。对于哺乳动物（如猫、狗、狼、狮子、老虎）而言，嘴主要有两个作用。

以下是嘴的两个基本生理功能：
*   **呼吸**：嘴是重要的呼吸通道。例如，狗在天气炎热时会伸出舌头并喘气。
*   **饮食**：嘴是消化系统的重要组成部分，涉及撕咬、咀嚼、吞咽等过程。牙齿、舌头和下颌骨都是为此服务的运动功能器官。

对于人类而言，嘴在上述两个基础功能之上，还衍生出另外两项独特功能。

以下是人类嘴部的两项衍生功能：
*   **语言**：嘴是人类进行语言表达的关键器官。
*   **构成表情**：嘴是形成面部表情的核心部件之一。虽然动物也有表情，但那更多是人类解读的结果，而人类的表情则具有明确的社会与情绪表达意图。

明确了呼吸、饮食、语言和表情这四大功能后，我们便能更清晰地分析人类嘴部的运动。

## 嘴部的基础运动形态

接下来，我们通过一些简单的发音练习，来直观感受嘴部肌肉的运动方式。你可以跟随以下描述，对着镜子进行观察。

以下是五种基础的嘴部运动形态：
1.  **上唇上扬**：单独将上嘴唇向上拉开，露出上牙。这个动作由**提上唇肌**控制。在动物和人类中，这通常与发狠或威胁的情绪相关。
2.  **下唇下降**：保持上唇基本不动，单独将下嘴唇向下拉，露出下牙。例如发“欸”音时的口型。这个动作由**降下唇肌**控制。
3.  **嘴角侧拉**：嘴角向水平方向两侧拉伸。例如发“呃”、“欸”音时的口型。这是**颈阔肌**等面部肌肉水平收缩的结果。
4.  **嘴唇张开**：下颌张开，上下嘴唇自然运动成圆形。例如发“啊”音时的口型，是嘴唇肌肉张成最大圆形的状态。
5.  **嘴唇聚拢**：分为两种形态。一是将嘴唇“嘟”出来，常表现为撒娇、不乐意的弱势形态；二是将嘴唇“拧”在一起发力，代表自我控制、压抑，不愿表露情绪，常见于吃力、委屈时。

## 核心分析：四种有效的表情形态

了解了基础运动后，我们现在聚焦于对解读情绪真正有效的嘴部形态。关键在于，**我们需要排除语言发音带来的千变万化的口型干扰**，因为这些是后天习得的社会化产物，而非天生的情绪表达。

当我们只回归到嘴的生理功能（呼吸、饮食）及由此衍生的纯粹表情功能时，有价值的观察形态就变得清晰起来。

以下是用于微表情分析的四种有效嘴部形态：
1.  **上唇上扬**：通常出现在**轻蔑**和**愤怒**的表情中。
2.  **嘴角侧拉**：通常是感受到**疼痛、劳累、难受或吃力**时的反应。例如被突然刺痛瞬间的嘴型变化。
3.  **否定嘴**：这是一个组合动作，表现为**颏肌**（下巴上的肌肉）向上收缩鼓起，同时**嘴角向下**。这个形态代表了内心的**不乐意、否定**，常伴随**悲伤、委屈、愧疚**等情绪。
4.  **笑容嘴**：这是人类特有的表情。表现为嘴角向耳朵方向的**斜上方**提升，同时通常伴随眼睛的闭合（关于笑的专业解读，我们后续会有专门课程）。这个形态代表了**积极、愉悦**的情绪。

掌握了这四种主流形态，你就能快速捕捉到人脸微小表情的情绪指向。当然，在失声痛哭、极度厌恶或愤怒等复杂情绪中，嘴型会有更微妙的变化，但它们都源于以上几种基础形态的组合与强化。我们将在后续的表情解读课中详细展开。
