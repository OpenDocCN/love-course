# 为什么社会的混混从不缺女朋友-P1-高情商撩妹思维-BV18s4y1D7qg_note

在本节课中，我们将要分析一个常见的社交现象：为什么一些看似不务正业的“社会混混”身边总是不缺女朋友，而许多条件不错的男性却面临交友困境。我们将拆解其行为模式背后的核心思维，并提炼出可供普通人学习借鉴的要点。

---

上一段我们提出了核心问题，本节中我们来看看“混混”们展现出的第一个关键特质：强大的内心。

**第一，内心十分强大。**

他们面对任何女性，无论对方多么优秀或漂亮，都敢于主动尝试。其心态可以概括为：**“你丑不要紧，万一她瞎呢？”** 这种心态使他们敢于行动，从而在概率上超越了99%连尝试都不敢的男性。你若不尝试，成功便无从谈起。

了解了强大的心态是基础后，接下来我们看看他们是如何将想法付诸行动的。这涉及到第二个特质：极致的勤奋。

**第二，非常刻苦。**

他们的努力程度超乎常人想象。以下是他们广泛使用的社交工具列表：

*   某性、某聊
*   某默、某叹
*   漂流瓶

可以说，一切你能想象到的社交与聊天工具，他们都能熟练运用。相比之下，如果你一年都认识不了一个新异性，就如同在商业中等待客户上门。**“客户”需要主动去寻找。**

掌握了广泛接触的方法后，如何推进关系就成了下一个关键。这引出了第三个核心思维：不害怕失去。

**第三，从不害怕失去。**

遇到心仪的对象，他们会立刻上前认识。因为对方也可能对你有好感，从而创造吃饭、看电影等约会机会。如果你连话都不敢说，关系便无法进一步发展。在约会中，他们遵循“该拉手就拉手，该抱就抱”的原则，不会唯唯诺诺。逻辑在于：**你不主动，她也不会因此喜欢你；别怕失去，因为你本就未曾拥有。**

---

本节课中我们一起学习了所谓“社会混混”在异性交往中展现出的三种可借鉴思维：**内心强大**，敢于尝试；**行动刻苦**，广泛接触；**不惧失去**，主动推进关系。这些思维的共同点在于将焦点从“害怕被拒绝”转移到“主动创造可能性”上，值得在社交中反思与应用。
