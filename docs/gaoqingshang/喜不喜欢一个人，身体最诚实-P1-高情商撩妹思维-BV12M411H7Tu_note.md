# 课程名称：高情商撩妹思维 - 第1课：判断女生好感的两个关键问题 💡

在本节课中，我们将学习如何通过两个简单的问题，初步判断一个女生是否对你有好感。这个方法旨在通过观察对方的反应模式，获得相对清晰的信号。

## 概述

许多人在与心仪对象相处时，常常困惑于对方是否对自己有好感。本节课将介绍两个操作性强的提问方法，帮助你通过对方的回复内容和态度，快速评估当前关系的发展可能性。

## 核心方法解析

上一节我们概述了本课的目标，本节中我们来详细看看这两个具体的问题及其解读方式。

### 问题一：测试对方对你的关注度

第一个方法是向对方发送一条信息，询问她对你的看法。以下是具体操作步骤：

1.  **发送信息**：通过手机给她发消息，内容为：“你觉得我傻吗？”
2.  **分析回复**：
    *   **积极信号**：如果她回复“你很傻”，并且围绕这个话题展开讨论，说了很多相关的话。这通常表明她愿意与你互动，对你有一定兴趣。
    *   **消极信号**：如果她只回复一个“？”。这通常表示她不想接话，对你当前的话题不感兴趣。
3.  **后续行动**：若收到消极信号（如问号），应适可而止，不要继续追问，以免引起对方反感。

### 问题二：测试对方对你的好奇心

在完成第一个问题的测试后，我们来看看第二个旨在引发对方好奇心的提问方法。

1.  **发送信息**：在睡前给她发消息。消息内容为：“你觉得我怎么样？算了，还是先睡觉吧，晚安。”
2.  **关键操作**：发送后立即关机睡觉。
3.  **分析次日回复**：
    *   **积极信号**：第二天醒来，如果发现她回复了一连串消息，或追问“你昨天晚上到底想跟我说什么？”。这强烈表明她对你的话题很感兴趣，并且对你有好奇心。
    *   **消极信号**：如果她只回复了一个表情包或一个问号。这通常意味着她反应平淡，兴趣不大。
4.  **后续行动**：若收到消极信号，建议调整心态，将注意力转移到其他事情上。
