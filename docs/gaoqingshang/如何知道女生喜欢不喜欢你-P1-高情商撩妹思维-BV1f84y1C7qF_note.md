## 四种关键表现

女生只在需要倾诉或感到寂寞时主动找你，之后便恢复冷漠。这表明你可能被定位为“备胎”，她不会将你视为感情生活的重心。

*   她只在特定情绪下联系你，关系呈现“招之即来，挥之即去”的模式。
*   当她遇到心仪的对象时，与你的互动会明显减少，回应也会变得冷淡。
*   为维持“备胎”关系，她会间歇性地给予你一些关注，以安抚你的情绪。

你们平时互动亲密，类似情侣，但一旦你试图表白或确认关系，她就会开始回避、含糊其辞。

*   她会通过语言或肢体动作给你“心里有你”的暗示，例如：`“天冷时，我总会想起你给我披外套的那个冬天。”`
*   对于不经意的肢体接触，她可能不会明确拒绝，只会害羞躲闪。

当你向她吐露心声或需要她陪伴时，她总是以“忙”为借口，没有精力回应你的情感需求。

*   矜持可以理解，但连续多日不回复消息，通常与“工作忙”无关。
