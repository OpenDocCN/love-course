## 第一种：喜怒无常型 😔

上一节我们介绍了本课的整体框架，本节中我们来看看第一种类型：喜怒无常的女生。

这种女生特别容易情绪化。她就像一个负能量的黑洞，充满各种抑郁、焦虑和对世界的不满。无论周围环境如何，她都很难开心起来。与这样的人相处，你不仅无法带动她的情绪，反而可能被她的负面状态影响，导致你自己的状态越来越糟糕。

**核心特征公式**：`情绪稳定性极低` + `持续输出负能量` → `关系消耗巨大`

## 第二种：自控力与耐受度低下型 😠

了解了情绪化的问题后，我们接下来看看第二种由内在控制力不足引发的问题类型。

这种女生的自控能力特别差，对不顺心事情的耐受度非常低。但凡遇到一丁点不如意，她就会摆出脸色，仿佛全世界都亏欠她。当你出于好心去安慰时，她甚至可能调转矛头向你发火。

以下是与此类人相处时可能出现的典型情况：

*   你们相处的大部分时间会在争吵中度过。
*   关系中的小摩擦极易升级为大规模冲突。
*   你的情绪能量会持续被她的愤怒和抱怨所消耗。

最后，我们来看一种看似有魅力但缺乏责任感的类型。

这种人对自己的外貌和条件非常自信，也很聪明，善于利用自身优势获取异性的好感。她可能会一边给你树立“深情人设”，一边继续“养鱼”（即同时与多人保持暧昧）。

*   没有为一段严肃关系做好心理准备。
*   缺乏**为这段关系放弃或减少某些个人自由**的信念。
*   因此，不建议与这样的人建立长期情侣关系。
