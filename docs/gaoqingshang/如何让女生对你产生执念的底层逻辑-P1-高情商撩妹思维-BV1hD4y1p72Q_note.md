# 课程一：执念生成的底层逻辑 🧠

在本节课中，我们将要学习如何让女生对你产生执念的底层逻辑。我们将从吸引的基本公式开始，逐步拆解正极情绪与负极情绪如何相互作用，最终累积形成执念。理解这些基于人性的不变规律，是掌握情感主导能力的关键。

## 吸引的基石：情绪公式

上一节我们提到了执念的生成与情绪有关，本节中我们来看看吸引力的核心公式。吸引力并非凭空产生，它遵循着特定的情绪规律。

我之前讲过一个公式：**吸引会产生正极情绪，而正极情绪可以推动负极情绪，负极情绪则可以推动执念的生成与累积**。

这个公式揭示了情感发展的动态过程。正极情绪如开心、欣赏是关系的起点，但它需要负极情绪的“推动”才能转化为更深层的、令人难以割舍的执念。

## 执念现象与人性规律

你们其实很多时候都会看到，一个价值很高的女生，她可能价值比我还高，但她却天天心心念念着我，在我面前表现得很卑微。

为什么会产生这种反应呢？实际上，就是我利用了人性里面的一些“机制”，即执念的生成原理。要理解这个概念，我们必须先回到吸引本身。

## 掌握不变的情感规律

我们要明白，整个与女生互动的体系，一定是有迹可循的。万般皆有法，它一定是有规律的。

那么我们只有掌握了这些规律。什么叫规律？规律一定是**不变的法则**。道理是不变的。

我们知道，我们去与女生相处，首先得记住这些核心公式。就像背九九乘法表一样，你得记下来。这个公式是不变的，再过十年、二十年，它都不会变，因为它是基于人性的底层规律。

很多时候，具体的套路可能会变，聊天的话术可能会变，但是这些根本的规律是不变的。所以说，我们只有记住了这些规律，你们才能真正地去掌握情感，掌握主导情感的能力。

## 本节总结

本节课中，我们一起学习了执念生成的底层逻辑。我们明确了**吸引→正极情绪→推动负极情绪→累积执念**这一核心公式，并认识到基于人性的情感规律是永恒不变的。掌握这些不变的法则，是摆脱盲目尝试、真正主导情感互动的第一步。
