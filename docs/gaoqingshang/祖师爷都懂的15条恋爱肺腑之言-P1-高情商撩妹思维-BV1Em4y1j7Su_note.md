**二、恋爱关系的建立顺序**
恋爱是“先上车后买票”的过程。先有事实关系，后有名义关系。在没有实质关系进展时表白，通常会被拒绝。唯一例外的情况是对方早已非常喜欢你，此时不表白也能在一起。流程可以简化为：
**互动 → 产生感觉与事实 → 确立关系**
而非：
**表白 → 等待同意 → 开始互动**

但女人是吸引来的，不是求来的。
