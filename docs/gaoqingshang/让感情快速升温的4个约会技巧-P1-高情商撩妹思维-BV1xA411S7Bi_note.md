上一段我们提到了早期接触的重要性，本节中我们来看看第一种具体的接触方式：**触摸**。

在对话开始的头几分钟内，你可以通过“强调观点”来自然地发起触碰。你唯一要做的，就是用一种寻常且非侵犯性的方式，轻触她的手臂或肩头。

**公式**：`强调观点 + 短暂、随意的肢体接触 = 自然打破触碰屏障`

理解了基础的触摸后，我们来看看第二种更具互动性的方法：**做个性测试**。
