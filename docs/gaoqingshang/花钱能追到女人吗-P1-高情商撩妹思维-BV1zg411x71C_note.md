## 🧠 爱情的科学本质

上一节的故事引出了一个核心矛盾，本节中我们来深入探讨爱情的本质。这是科学家研究的结果，具有权威性。

科学家指出，爱情是你身体对某个人分泌三种激素的结果：
1.  **苯基乙胺**：让人感到极度兴奋，产生“来电”的感觉。
2.  **去甲肾上腺素**：让人血压和心率升高，产生心跳加速、血液流动加快的感觉。

如果你觉得难以理解，可以将它们统称为 **荷尔蒙**。也就是说，只有当你的身体对一个人分泌了荷尔蒙时，你才会产生爱情的感觉。
