# 课程名称：高情商撩妹思维 - 第一讲：疫情隔离期的联系策略 📱

## 概述

在本节课中，我们将要学习在因疫情等不可抗力因素导致无法与女生见面时，应如何调整沟通策略。核心在于理解情感发展的周期性，并学会在正确的时机采取行动，避免无效努力。

## 核心原则：情感周期与时机

上一节我们介绍了课程主题，本节中我们来看看核心原则。情感发展具有周期性，无法强行倒退。

例如，如果你与一个女生关系升温后又冷淡，想退回之前的热络状态会非常困难。因此，在无法见面的阶段，与其进行可能消耗好感度的无效聊天，不如暂时将关系“放置”，等待更好的时机。

**核心公式**：`吸引力积累 -> 收到足够IOI（兴趣指标） -> 展示可得性`

这个流程不可颠倒。在未建立足够吸引力和未收到女生明确兴趣信号前，过早表达喜欢（即可得性）通常效果不佳。

## 无法见面时的具体策略

理解了基本原则后，我们来看看无法见面时的具体操作。

以下是几个关键策略点：

*   **停止无效聊天**：如果确定短期内无法见面，继续漫无目的地聊天意义不大。对方可能因缺乏现实接触而兴趣减退，你也可能被其他能见面的竞争者“截胡”。
*   **专注自我提升**：将无法见面的时间用于锻炼、健身或其他自我提升项目。这不仅能增加你未来的吸引力，也能让心态更平稳。
*   **等待与重启**：暂时“放置”关系不是放弃。待疫情缓解、可以见面时，或当你通过社交动态展示出更高价值后，再重新开启联系。此时，如果她对你有基本吸引力，窗口会再次打开。

## 常见问题解答（Q&A）

针对大家常见的困惑，以下是具体解答。

**Q：妹子离我很远，怎么聊？**
**A：** 严格按照上述流程聊。先通过社交互动建立吸引，收集兴趣指标（IOI），再考虑推进。

**Q：在学校该通过朋友介绍，还是自己收号？**
**A：** 自己主动认识并获取联系方式（收号）通常更可靠，因为这直接展示了你的自信和意图。

**Q：错过了联系窗口怎么办？**
**A：** 错过窗口，就耐心等待下一个窗口出现。期间继续提升自己，并通过社交平台适当展示价值。

**Q：可得性（可德性）具体是什么？**
**A：** 可得性指的是通过行为或语言，向女生表达你对她有兴趣、且是可供她选择的对象。**注意**：这是在吸引建立之后才进行的步骤。

## 关于学习周期与执行力

最后，我们来谈谈一个关键问题：掌握这些思维需要多久？

学习成长的速度完全不取决于课程本身，而完全取决于你个人的**执行力**。

以下是学员进展不同的实例，说明了执行力决定结果：

*   有的学员一周内就有显著进步。
*   有的学员一个月表现非常出色。
*   也有学员用了三个月或半年达到很好水平。

每个人努力的程度不同。如果报了课程却从不学习、不实践，自然无法成长。只要你决心学习并拥有强大的执行力，配合正确的方法，进步会非常迅速。

## 总结

本节课中我们一起学习了在疫情等无法见面的特殊时期，如何运用高情商思维处理与女生的关系。核心要点是：尊重情感周期，避免在无法见面时进行消耗性的无效聊天；将精力转向自我提升；并理解“吸引先于可得性”的核心流程。记住，最终成效取决于你的执行力。坚持学习与实践，方能快速成长。
