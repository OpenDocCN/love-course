# 聊天技巧教程：有效提升对话深度（下） - 第1课 📘

在本节课中，我们将学习如何通过增加回复的信息量，使对话更顺畅、更具吸引力。核心在于理解并应用“信息单元”的概念，让双方都能轻松参与聊天。

---

很多人在聊天时遇到对方不回复的情况。这通常是因为你回复的信息单元太少，导致对话难以继续。我们需要增加回复中的信息单元。

例如，当你说“我今年不太想回去，每年一回去七大姑八大姨就各种灵魂拷问”时，这段话包含了多个信息单元。女生因此能做出很好的回应。很多时候，不是女生不想回消息，而是她不知道如何回复你。

聊天不仅是回复对方的消息，更要让对方能够回复你的消息。因为你发的消息，有时女生也不知如何回应。因此，聊天时不仅要考虑自己，更要站在对方角度，考虑对方的感受。

以下是构建有效回复的一个核心公式：

**信息单元解析 + 感受 + 时间线 = 聊天脉络**

你可以根据聊天目的，例如需要建立吸引或提供安全感，沿着这个脉络继续对话。
