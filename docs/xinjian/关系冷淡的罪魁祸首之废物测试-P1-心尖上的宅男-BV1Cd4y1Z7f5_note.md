## 1. 劳务关系型测试 💼

上一节我们介绍了废物测试的基本概念，本节中我们来看看第一种常见类型：劳务关系型测试。

有些女生会让喜欢的男生出力帮忙。千万不要觉得这个时候，百依百顺是正确的做法。当她觉得你呼之即来，挥之即去的时候，你们就没有任何的机会了。所以这个废物测试是属于一个隐形的一类，不容易被发觉的。

以下是这种测试的典型例子与应对策略：

*   **典型例子**：女生让你帮她写个工作总结、帮她点个外卖、帮她买个东西。
*   **核心原则**：切记无谓的付出，尽量做出价值的互换，不要一味的服从对方。
*   **应对方法**：例如，她让你帮她写个工作总结，你可以说“没问题，关键写这个东西要熬夜，可能喝杯咖啡精神要好些”。这个时候一定要注意你说话的方式，所有的价值千万不要超越对方的要求，否则对方就很容易认为你是个很小气的人。比如对方让你寄个快递，你让对方请你吃饭，这就有点过了。

## 2. 女生自我锁定（打压）型测试 🪞

了解了如何应对劳务索取，我们接下来看看第二种测试：女生通过自我贬低来观察你的反应。

女生会说：“我觉得我最近腿粗了”、“我觉得我的脸好大”、“我觉得我最近好像胖了”等等。这个情况分两种，需要区别对待。

以下是针对两种不同情况的应对策略：

*   **情况一：她其实有自信，在“凡尔赛”**。如果她真的是身材很好，很明显是有自信的人，却拿着自己的优点去说自己有缺点。那么我们就可以反其道而行之，这样会起到意想不到的效果。就比如她说“我最近好像是胖了”，那其实她的身材是蛮好的，那么我们就可以适当调侃她：“你的小腿都还没有我胳膊粗呢”。这个时候很容易激起女生的反抗情绪，开始跟你打闹调侃，同时心里也会觉得这个男生跟其他的男生不一样。
*   **情况二：她真的在否定自己**。如果对方真的是在否定自己，觉得对自己的颜值身材不自信。这个时候你可以适当安慰一下，或者与第三方进行比较。比如她最近好像胖了，你可以说“你这样都说胖了，那胖子听了都想打人了”，或者说“一只手都能抱起来的就不算胖，我来试试看”。这个时候千万不要去否定对方，注意特别是千万不要去出谋划策，告诉对方该怎么减肥、怎么样去变漂亮。其实这就等于是你在认同她的观念，你也觉得她胖。

## 3. 金钱型废物测试 💰

除了情感上的测试，直接涉及利益的测试更为棘手。本节我们探讨第三种：金钱型废物测试。

测试形式包括：让你帮忙充话费、点外卖、借钱、发红包、转账等。只要是和金钱、利益挂钩的，那就是金钱型。

遇到这种情况，如果不是骗子，那就是绿茶。一般和你关系不到一定程度的女生，是不会拉下脸来跟你借钱的。所以能开口向你借钱的女生一般都是老手，并且借出去的钱，你就别想再要回来了。

如果遇到这样的女生，有一句话可以总结：**靠什么吸引女生，以后也要靠什么来维系**。所以如果女生的这种行为令你感到不快，还是尽早地撤离。要不投资得越多，你会越放不下。要用真正的个人核心魅力去吸引她。

最后一种测试往往包裹着甜蜜的谎言，我们需要格外警惕。

女生说：“等我毕业了我们就在一起”、“如果我分手了，你就是我的首选”等等。相信我，兄弟们，如果你听到这些，那你就是备胎。她不喜欢你，只是觉得你对她好。

本节课中，我们一起学习了四种常见的废物测试及其应对方法。想要通过女生的废物测试，我们只需要记住以下几点核心原则：

运用这些原则，可以帮助你更从容地应对交往中的挑战，建立真正相互吸引的关系。
