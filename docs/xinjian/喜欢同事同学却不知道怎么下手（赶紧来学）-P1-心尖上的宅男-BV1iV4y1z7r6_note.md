# **喜欢同事同学却不知道怎么下手** - P1 - 心尖上的宅男 - BV1iV4y1z7r6

## 概述
在本节课中，我们将学习如何无伤地追求同事或同学，避免尴尬和负面影响。

## 如何成为朋友
**公式**：共同话题 + 共同爱好 = 友谊

以下是成为朋友的步骤：

1. **观察了解**：观察同事的兴趣爱好。
2. **研究兴趣**：深入了解这些兴趣爱好。
3. **偶遇交流**：在偶遇时，用共同话题开始对话。
4. **分享内容**：分享好玩的段子、搞笑视频等。
5. **日常互动**：偶尔聊天，保持朋友关系。

## 如何暗示
**公式**：开玩笑性质的暗示 = 暗示

以下是暗示的步骤：

1. **开玩笑**：用玩笑的方式表达好感。
2. **观察反应**：注意对方的反应，如害羞。
3. **慢慢变化**：对方对你的感觉会慢慢变化。

## 如何试探
**公式**：观察行为变化 = 探测喜好

以下是试探的步骤：

1. **观察变化**：观察对方的行为变化，如变得认真或话多。
2. **判断喜好**：根据变化判断对方是否喜欢你。

## 总结
本节课中我们一起学习了如何无伤地追求同事或同学，包括成为朋友、暗示和试探的步骤。希望这些方法能帮助你成功追求心仪的人。
