# 【脱单攻略】从那我走，看当代女生对一夜情的看法 - P1

大家好，我是宅男，用理工男思维分析情感问题。今天我们要探讨的是，当代女生对一夜情的看法。

## 情感问题中的常见现象

我们总在晚上，或者是听朋友，说一些一夜情的故事。这样的事情呢，现实中真的很多吗？为什么我一个也遇不到呢？每次当我鼓起勇气，想去试试的时候，就会出现一个鲜活的感叹号，不禁想问一句，你能不能帮我，把我这个人的心，带到最后一步，不禁想问一句，你们这些小姐姐，都是哪里人之类的。

## 女生对一夜情的普遍态度

其实大部分的女生，都不接受一夜情，她们很难接受，只有肉体上，没有心灵上的关系，她们觉得吃亏，觉得恶心，觉得在作践自己。

## 什么样的女生会一夜情？

那什么样的女生，会一夜情呢？这个问题得分两种情况来看：

### 第一类：职业原因导致的一夜情

有些女生接受一夜情。例如，那些做外围的、做小姐的女生，会一夜情。原因很简单，因为看得比较开，每天接触的都是油腻男，她们对颜值，比一般人有更高的期待，她们需要那些小奶狗、虫虫的爱情和身体，但是很多情况下，她们本身职业的原因，并不适合谈恋爱，所以排斥爱情。但是又渴望那种，除了金钱之外的，心动的感觉。

### 第二类：羡慕他人而尝试一夜情

有些情况下，很多女生愿意尝试一夜情。例如，和那些做外围的、做小姐的女生走得比较近的女生，她们看到这些女生又有钱，身边还帅哥不断，羡慕，但是她们不知道的是，给她们钱的和这些帅哥，压根没有半毛钱关系，甚至她们还经常倒贴，但是这些身边的女生，会心里不平衡，觉得自己这么清纯，不值得，所以也想去试试。

## 女生排斥一夜情的原因

女生会排斥一夜情的原因，是不浪漫、没感情。女生是只有有感觉了，才会有别的想法，所以正常来说，只要有一段时间的接触和了解，是很难产生感觉的。

## 一夜情中的价值观

对于一夜情，我们要抱什么样的价值观呢？

1. 恋爱要多接触几个异性，第一个呢，就从恋爱到结婚，固然是好的，但是也很难有很好的结局，因为只有多接触才知道，自己真正喜欢的是什么样的，什么类型的人，又是适合自己的。
2. 一开始骗女生说我喜欢你，我想和你在一起一辈子，睡了之后呢就不理了，这种都不是渣男，是人渣。
3. 单身的时候，有浪漫的邂逅也是可以的。
4. 不要看别人关心去羡慕，别人愿意做的事情，你不一定愿意去做。别人承受的东西，你不一定承受得了。
