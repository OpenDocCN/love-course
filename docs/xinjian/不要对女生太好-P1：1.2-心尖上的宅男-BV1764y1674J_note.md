当男生喜欢一个女生的时候，很自然的反应是“我要对她好”。

女生也一直说希望男生对她好，但是她只说了一半。女生只喜欢她喜欢的男生对她好。

所以并不是说男生不应该对女生好，而是你一上来就对女生好，让别人感觉你是有所企图。这是一种交换：**你用对她的好去交换女生的爱情**。

那么当你对女生越来越好的时候，女生会压力越来越大。当她“还不了”你爱情的时候，这就成了她的负担，她就会拒绝你对她的好，也同时拒绝了你这个人。

往往在一开始就对女生好，女生并不会因此喜欢你，还会有相反的效果。

因为男女交往的时候，一开始女生首先关注的是：对这个男生有没有兴趣，然后才是这个男生对我怎么样。

一开始最重要的不是对她好，是让她觉得“哎哟，这个男生还不错哦”，是让她看到你的闪光点。而且一开始就“舔”，会让你的魅力大打折扣，相当于告诉女生：你觉得自己配不上她。

爱情分不同的阶段，不同的阶段要做不同的事情。
