# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是赋值，公式如下：
`变量名 = 值`

例如，我们创建一个名为 `message` 的变量，并让它存放文字“你好，世界！”。
```python
message = "你好，世界！"
```

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有几个内置的基本数据类型，它们决定了数据可以进行的操作。

以下是三种最常用的基本数据类型：

*   **整数**：表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`, `“学习编程”`。

## 如何查看和使用变量？ 👀

我们知道了如何创建变量，现在来看看如何查看它的内容以及在程序中使用它。

在Python中，要查看一个变量的值，最简单的方法是使用 `print()` 函数。它会将括号内的内容输出到屏幕上。

以下是使用 `print()` 函数的示例：
```python
name = “小明”
age = 18
print(name) # 这行代码会输出：小明
print(“年龄：”, age) # 这行代码会输出：年龄： 18
```

你可以直接打印变量名，也可以将变量与文字组合在一起打印。

## 一个简单的实践例子 ✨

让我们把上面学到的知识组合起来，写一个简单的程序。

假设我们要欢迎一位新用户，并告诉他今天的日期。

```python
# 创建变量
username = “张伟”
current_year = 2023
current_month = 10
current_day = 26

# 使用变量打印信息
print(“欢迎你，”, username, “！”)
print(“今天是”, current_year, “年”, current_month, “月”, current_day, “日。”)
```

运行这段代码，你将会在屏幕上看到一句完整的欢迎语和日期信息。通过改变变量 `username`、`current_day` 等的值，你可以轻松更新输出内容，而无需修改 `print` 语句本身，这就是变量的威力。
