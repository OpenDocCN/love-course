## **溺爱与孝道**

**公式**：溺爱 → 扭曲的爱情

## **爱情的方法**

孩子的教育需要方法，爱情同样需要方法。当你一次次喜欢一个人却一无所获时，你应该意识到爱一个人需要有所保留。你要尊重别人，更要尊重自己。

**公式**：尊重自己 → 爱情的方法

**公式**：卑微 → 扭曲的关系
