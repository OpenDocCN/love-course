# 追女生聊天教程 📚 P1：第八集

在本节课中，我们将学习与女生聊天时的两个核心技巧：聊天频率与聊天深度。掌握这两点，能帮助你避免常见错误，建立更自然、有效的关系。

## 聊天频率问题 📊

上一节我们介绍了课程主题，本节中我们来看看聊天频率。许多人在追求女生时，误以为高频率聊天能增进感情，但结果往往相反。

我统计过所有向我求助的人。他们描述与女生聊天的情况时，都会提到以下字眼：

*   之前我和她几乎每天聊天。
*   有时候从早上聊到晚上。
*   女生什么话都跟我说。

而她们最后得到的结果却惊人地相似：

*   女生和我聊天越来越冷淡。
*   找我的次数越来越少，甚至完全不找了。
*   然后被告知：“我觉得我们还是做朋友比较好”。

这种高频率的聊天，本应让感情日渐浓厚，为何会遭到拒绝？为何聊着聊着就没下文了？很简单，原因在于你们日渐浓厚的感情不是爱情，而是友情。

那么，聊天的节奏应该是怎样的呢？以下是正确的节奏指南：

*   **初期认识阶段**：刚认识时应多聊，以便了解彼此、形成印象。因此开始时聊天应频繁些、时间长些。
*   **认识后的日常阶段**：聊天绝不能定时定量。很多人像完成游戏任务一样，在固定时间出现且聊个没完，这会让女生感觉糟糕。她若不回复觉得不妥，若回复则像在完成任务，令人崩溃。
*   **避免“任务式”深聊**：男生常希望每天深聊，且越久越好。例如，第一天聊一小时，第二天两小时，他就觉得进步了；若第三天只聊十分钟，便觉得退步了，并怀疑“她是不是不喜欢我了”。追女生不是玩游戏，不要用这种思维。
*   **正确方式**：认识后的聊天应是不定时的。平时稍作互动，偶尔深入交流，切忌没话找话。当话题已很尴尬却强行继续时，聊天便失去了意义。聊天的目的是交流，是像动物间互相梳理毛发般的礼节。你这不是在表达礼节，你这是在薅毛。

正确的方式是：将生活中遇到的有趣、好玩或八卦的事情分享给对方，附上自己的看法，或听听对方的看法。分享完即可结束。偶尔会有深聊，但这种情况较少，例如当双方都很有聊天兴致，或对某事特别感兴趣、越聊越有劲时。

和女生的关系不是由聊天频率决定，而是由**聊天深度**决定的，是由几次重要的聊天构成的。平时互动只是为了混个脸熟、蹭个热度。高频率聊天只会消磨你们之间的神秘感和新鲜感。再有趣的事，让你天天做也会觉得无聊，这就是女生的想法。偶尔，你还可以让她意犹未尽；而若每天都是你自己意犹未尽，效果就适得其反。

## 聊天深度问题 🎯

上一节我们探讨了聊天频率，本节中我们来关注聊天深度。与女生确定关系，离不开几次有深度的聊天。

这些深度聊天通常关于以下内容：

*   **第一次深度聊天**：这决定了女生今后如何与你相处。若能聊得深入，代表她对你有兴趣，可将你视为发展对象。女生会在第一次聊天中决定：是可以接触、做普通朋友，还是完全不想接触。如果这次聊天深入，她对你的印象会不错。
*   **第二次深度聊天**：发生在她对你产生兴趣和好感时，她特别想和你聊天，分享她的过去，也就是她打开话匣子的时候。此时一定要认真倾听并给予回馈。很多宅男会把话题转向以“我”开头，讲述自己的事。要记得用“**你**”，而不是用“**我**”。当女生开始与你分享她的喜怒哀乐时，你就进入了她的生活。在她身边这样的人，算上闺蜜，不会超过五个。
*   **第三次深度聊天**：这时她会刨根问底地询问、考验甚至刁难你。例如，“你为啥和前女友分手？”“你之前谈过几个？”“我觉得你是渣男。”恭喜你，这代表女生开始喜欢你了。但很多人意识不到问题的严重性，这是她最后的考验。此时要深入交流，把自己的想法、对她的感受、她所有顾虑的事情，都一一向她交代清楚。

其实并没有那么多需要深聊的内容。平时就是日常互动，不要强行深聊。这就像你看新闻，突然记者疯狂地与你走心，你会感到不适。如果你每天看新闻，他都上来走心一遍，第二天再看，他又要走心，你可能会直接关掉电视。你会想：“我来看个新闻，你没事老跟我走心干嘛？”走心需要时机，例如大学毕业聚餐，酒过三巡，大家开始走心。为什么？因为时间、环境、情绪都到位了。如果你大学刚入学就天天在寝室走心，别人肯定会觉得你不太正常。

## 总结 ✨

本节课中，我们一起学习了与女生聊天的两个关键点：频率与深度。

关于聊天的这两点，大家记住后会对你的聊天能力有很大提升。这是一个大致的聊天节奏，很多人正是因为缺乏节奏，所以关系进展很快失败。

其实，和女生聊天还有一个点也特别重要，就是要会“撩”。这是你和女生关系能否升级的关键，也就是女生所说的“有感觉没感觉”中的那个“感觉”。

**本节课核心要点回顾**：
1.  **聊天频率**：初期可多聊，之后应不定时互动、偶尔深聊，避免高频率任务式聊天消耗新鲜感。
2.  **聊天深度**：关系由几次关键深度聊天推动，主要发生在初次印象建立、对方打开心扉以及最终考验阶段。平时避免强行深聊。

掌握好节奏，在合适的时机进行深入交流，才能有效拉近彼此距离。
