女生却可能严肃地说：“你能说人话吗？” 需要明白，幽默与“皮”之间最大的区别在于：**幽默**是你能通过一个通俗的道理，联系到女生的生活、理想和爱好；而**“皮”**则是在隔靴搔痒，没事找事。

同时必须记住：所谓幽默是为了**缓解尴尬**，而不是**制造另一种尴尬**。切勿强行幽默。否则只会给对方留下“直男硬装老司机”的差评印象。
