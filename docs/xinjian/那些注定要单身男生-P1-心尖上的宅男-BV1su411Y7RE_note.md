## 街头采访的意外发现 🎤

上一节我们提到了课程主题，本节中我们来看看支撑这个观点的实际调查。

我曾进行一项街头调查，采访不同女性，询问她们不欢迎哪种类型的男生。采访对象主要是陌生女性。

起初，我们假设答案是那些不高不帅、没钱没事业的男生，因为这是常见的抱怨内容。

我们采访了许多手挽手的情侣，发现大部分男生的条件都很普通，很少有“高帅富”。

其中一位女士的话让我印象极为深刻。

她说，如果一个人声称“我想等到某个时候再谈恋爱”，那么这个人十有八九绝对脱不了单，而且绝对是一个脱单困难户。

## 机会的本质与“准备”的陷阱 ⏳

上一节我们看到了来自女性的直接观察，本节中我们来看看这种现象背后的核心逻辑。

这个世界上的机会都是短暂且转瞬即逝的。

机会不会询问你是否准备好，也不会因为你是否优秀而等你。

只要你稍有犹豫，机会扭头就走。

那些胆怯且永远都在做准备的人，一直在闷头苦干，却错过了一个又一个机会。

他们对着机会的背影请求再给一些时间，但对不起，没有时间。

**闭门造车的准备在情感中没有意义。**

情感中出现的问题，只有在真正谈恋爱时才能发现。

也只有经历了感情的人，才会知道自己哪里做得不好。

## “完美男友”幻想与内心匮乏感 🎭

上一节我们讨论了无效准备，本节中我们来深入分析这种“永远在准备”心态的心理根源。

有些人一直努力想成为“完美的男朋友”，设定一系列必须达成的条件：
*   一定要有钱
*   一定要有学历
*   一定要有房有车
*   一定要有发展前景

与其说这是在准备，不如说是在逃避。

目标一直在变，但目的只有一个：**弥补内心的匮乏感，让自己更喜欢自己一些。**

他们的内心独白往往是：
```
我要考上好大学 -> 我要找到好工作 -> 我要月入过万 -> 我要买房 -> 我要还清贷款 -> 我要有自己的事业
```
他们永远对自己不满意，永远不认可自己，永远觉得自己没有准备好开始一段感情。

这样的人，怎么可能轻松地开始恋爱呢？
