## 反面案例分析

上一节我们介绍了吸引力的重要性，本节中我们来看看一个常见的错误示范。

以下是一段反面聊天案例的解析：

*   男生一开始的自我介绍是：“我表哥把你推给我了，说你是个乖乖女，让我好好珍惜照片我也看了很漂亮。” 这种说话方式显得目的性过强。
*   随后聊天内容转向查户口式的提问，例如询问工作、生活等基本信息。
*   整个对话虽然篇幅不短，但内容空洞，缺乏情绪互动，没有任何话题能深入展开。
*   女生从开始的热情回应，逐渐变得冷淡，最终不再理会。

这种情况非常普遍。无趣且无聊的对话无法让女生产生感觉。

很大一部分男生的聊天水平与此案例相似。问题不在于某句话说得不好，而在于整个聊天思维是错误的。让女生喜欢上你需要三点：**印象好 + 感觉好 + 相处好**。

在这个案例中，印象普通，没有特别感觉，聊天也无法正常进行，相处自然不好。这几点都做不好，女生就不可能喜欢你。

## 正面案例学习

理解了反面案例的问题后，我们来看一个成功的正面案例，学习如何在认识初期进行有情绪的聊天。

案例背景是男生和女生在一次社交聚会认识，彼此印象不错。男生主动添加微信后，开始了以下对话。

对话中聊到家庭琐事，女生问：“你跟你爸妈都是一起住吗？”
男生回应：“各自为政，爸妈都在家呢。别说我了，说说你吧。”
女生回：“那你问吧。”

此时，可以运用我们之前讲过的技巧。男生问：“那我问了…为什么你这么漂亮，反而还一直单身？” 这就是**联想称赞法**。

女生的反应是：“因为遇不到自己喜欢的人，或者说喜欢的人不喜欢我，然后我就一直单着了。你为什么也单着呢？”
女生的话匣子被打开，并反问男生，说明她愿意多了解对方。

男生回应：“我不会追女生，也有你说的原因。”
女生意识到男生在夸她，回复：“对了，不要说我漂亮，那都是照片。”
男生立刻发起攻势：“要不要现在自拍一张？想看。”

至此，对话充满了情绪互动。女生对男生的感觉变得特别好。对于“印象好、感觉好、相处好”这三点，男生已经做到了前两点，女生对他的好感度显著提升。

女生甚至后来问道：“你是不是谈了很多段恋爱？这么会说话。” 这表示女生已被吸引，认为男生优秀且受欢迎。

## 核心技巧解析

为什么不到一天的聊天就能让女生认为男生很优秀？接下来我们解析这段聊天中的核心技巧。

如果你认真看完案例，会发现男生大量使用了赞美，但方式非常巧妙：

以下是男生使用的一些赞美句式示例：
*   “等你愿意了拍一张最美的放到朋友圈。”
*   “看到照片之后有点感觉了…”
*   “中通外直，不蔓不枝，香远益清，亭亭净植。”
*   “我决定了，接下来几天无聊的时候就找你聊天，更多的发掘你的魅力。”
*   “不仅如此，还有一个华丽的躯壳。”

这些夸赞有趣且自然，丝毫没有降低自身价值。例如，男生将女生朋友圈的自拍照放大后发回，这种互动就很有意思。他说“看了照片之后有点感觉了”，能瞬间激发女生的情绪，随后补充“就有一点点”，将赞美、调侃和幽默结合，展现出极大的个人魅力。

当女生好感度上升后，这种连续而自然的、从自身感受出发的赞美（如文艺的比喻“想到了一朵花…”），会让感觉迅速升温。

后来女生问：“你为什么还想找我聊天呀？” 这表明她对男生有了超出朋友的感觉，并在意男生对自己的看法。男生的回应是：“今天本来是想闲聊一下的，然后发现你还挺有趣的…不仅如此，还有一个华丽的躯壳。” 这种不经意间的赞美让女生防不胜防。

在这段聊天中，男生十句话有九句都在夸赞女生，且每一句都从自身感受出发，显得非常自然。

## 成果与总结

后续发展是，男生和女生仅聊天不到三天，女生就喜欢上了男生。

女生说：“我睡不着了…感觉这段时间和你的聊天有点依赖你了。我认真的想了一下，我觉得和你聊天很开心，也害怕哪一天你突然就不理我了。” 这表明女生已经非常在意男生。

男生回应：“你傻呀，想那么多干嘛，怎么会不理你？”
女生直接表白：“其实我挺喜欢你的，不知道你喜欢我吗？我希望你能够很认真的告诉我。”
在导师指导下，男生回应后，两人正式确立了关系。

本节课中我们一起学习了建立吸引力的核心。女生能否喜欢上你，需要理由。这个理由就是：聊天能否做到轻松自然、有无情绪互动、能否展现你独特的人格魅力。

**核心公式：吸引力到位 + 能左右女生情绪 = 女生可能主动表白**

当你的吸引力足够，能够影响女生的情绪，那么女生向你表白就是一件正常且简单的事情。在座的各位，通过学习和练习，同样可以做到。
