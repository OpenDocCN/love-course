## 游戏概述与基本规则

国王牌是一个需要至少三人参与的纸牌游戏。游戏的核心道具是数字牌和一张代表“国王”的K牌。

**游戏准备公式**：`玩家数量 = n`，所需纸牌为数字牌 `1` 到 `n-1` 各一张，外加一张 `K` 牌。例如，四人游戏需要 `1, 2, 3, K` 四张牌。

游戏开始时，由一人坐庄洗牌并发牌。每位玩家（包括庄家）随机抽取一张牌作为自己的“底牌”，但**不能查看**。剩下的牌则作为“公共牌”。

## 游戏流程详解

上一节我们介绍了游戏的道具和准备，本节中我们来看看具体的进行步骤。

1.  **抽牌与亮王**：所有玩家抽牌后，唯一手持K牌的玩家需要亮出K牌，公开其“国王”身份。
2.  **国王制定规则**：国王可以任意指定两个数字（例如“1号和3号”），并命令他们完成一个动作（例如“拥抱3秒钟”）。国王**不知道**每个数字对应哪位玩家，也可能指定到自己。
3.  **执行指令**：所有玩家同时亮出自己的底牌。被国王点中的数字所对应的玩家，必须按照国王刚才宣布的规则完成互动。

以下是游戏的一个关键机制：
```python
# 伪代码逻辑示例
if player.card == “K”:
    player.role = “King”
    king.announce_rule(“NumberA”, “NumberB”, “Action”)
else:
    player.role = “Citizen”

# 亮牌后
for player in all_players:
    if player.card in [“NumberA”, “NumberB”]:
        player.perform(“Action”) # 必须执行动作
```

## 游戏的策略与趣味点

这个游戏的魅力在于其公平性和不可预测性。国王在制定规则时，无法预知谁会中招，这增加了戏剧性。

*   **公平性**：规则由国王制定，因此所有玩家都需要遵守。如果一位女性玩家抽到K并制定了“2号和3号亲吻”的规则，而她自己恰好是2号，她也必须执行。这避免了玩家只制定对他人不利的规则。
*   **社交压力**：当有人（尤其是制定规则的国王自己）被规则选中却想退缩时，其他已完成指令的玩家可以形成友好的群体压力，鼓励游戏继续，维护了规则的严肃性。
*   **尺度控制**：动作指令的尺度可根据场合调整。在公开场合如小酒馆，适合“喝交杯酒”、“对视10秒”等轻度互动。在私密空间如家中，则可以设计更有趣或更亲密的互动。

## 特殊情况处理

如果所有玩家亮牌后，发现没有人抽到K牌（即K牌留在未抽取的公共牌中），则视为特殊情况。

**特殊规则**：此情况下，由本轮的**庄家**接受惩罚，例如喝一杯酒。之后，由庄家重新洗牌开始下一轮。

## 适用场景与总结

本节课中我们一起学习了“国王牌”游戏的完整玩法。这是一个非常适合在聚会后期、气氛热烈时进行的破冰游戏，能有效促进参与者之间的肢体接触和情感交流。

**核心总结**：
1.  至少3人参与，用数字牌和一张K牌进行。
2.  抽到K的玩家为“国王”，有权制定两个数字牌持有者之间的互动规则。
3.  所有玩家必须遵守规则，包括国王自己，这保证了游戏的公平与刺激。
4.  游戏尺度可根据环境（公开/私密）灵活调整。
5.  若K牌未被抽中，则庄家受罚。
