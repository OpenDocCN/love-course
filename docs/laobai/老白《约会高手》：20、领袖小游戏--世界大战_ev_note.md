# 老白《约会高手》：20、领袖小游戏--世界大战_ev_note

在本节课中，我们将学习一个名为“世界大战”的社交游戏。这个游戏的核心在于分组对抗，通过简单的游戏互动来增进团队成员间的默契与联系，尤其适合在社交或约会场合中拉近彼此距离。

## 游戏准备与分组 🤝

上一节我们介绍了游戏的基本概念，本节中我们来看看如何开始。游戏首先需要进行分组。

游戏参与人数需为偶数。例如四个人就分成两人一队，六个人则分成三人一队。分组方式采用“石头剪刀布”进行公平选拔。

以下是分组的具体步骤：
1.  所有参与者通过“石头剪刀布”决出一位胜者。
2.  胜者有权从剩余参与者中选择一人成为自己的队友。
3.  被选中的人与胜者组成一队，剩余的人自动成为另一队。
4.  如果人数较多，可以重复步骤1和2，由胜者轮流挑选队友，直到队伍人数均等。

分组完成后，就形成了两支对抗的队伍。

## 游戏规则与进行 🎲

队伍分好后，就可以开始游戏了。游戏内容可以自由选择，例如玩骰子、“石头剪刀布”或“十五二十”等。

在游戏开始前，必须提前约定好惩罚规则。例如，约定败方队伍每人需要喝一杯酒。这是保证游戏公平、顺利进行的关键。

以下是游戏进行的一个示例流程（以骰子游戏“吹牛”为例）：
1.  两队轮流派出一名队员进行对决。
2.  在玩骰子过程中，同队队员可以互相交流、查看骰子，创造自然的身体接近机会。
3.  队员间可以互相支持，例如说“别怕，我帮你看看”、“大胆叫，有我在”等，营造团队感和安全感。
4.  当一方队员全部落败时，该队伍即为输家，需接受预先约定的惩罚。

**核心规则强调**：所有特殊规则必须在开局前说明。例如，如果约定“一人击败对方全队则惩罚翻倍”，就必须在游戏开始前明确告知所有参与者。中途临时增加规则是无效的。

## 游戏的精髓与作用 💡

这个游戏的精髓并不在于胜负本身，而在于它为团队成员间创造的合作与互动场景。

在游戏过程中，因为同属一队，成员会自然产生“我们是一伙的”这种一致对外的团队认同感。你们可以互相商量、掩护，甚至替对方承担部分惩罚。这种共同经历和“并肩作战”的感觉，能快速拉近两人的距离，增进亲密感和联系。

通过几轮游戏下来，队友间的默契和信任感会显著提升。这正是在社交场合中打破隔阂、深化关系的一个非常自然且有效的方法。
