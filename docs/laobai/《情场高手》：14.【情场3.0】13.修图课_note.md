# 《情场高手》课程14：情场3.0 第13课 - 修图课 📸

在本节课中，我们将学习如何通过修图软件优化个人照片，使其更具吸引力和质感。课程将介绍三款常用软件，并详细讲解从基础瑕疵修复到高级滤镜调整的完整流程。

## 软件介绍

以下是本节课将用到的三款修图软件：

1.  **Snapseed**：一款功能强大的免费滤镜与图片编辑软件，图标为树叶。
2.  **美图秀秀**：一款广为人知、易于上手的图片美化软件，擅长人像处理。
3.  **VSCO**：一款提供丰富滤镜和高级调色工具的软件，部分功能收费，但免费功能已足够强大。

其中，Snapseed和VSCO主要用于调整图片整体色调和质感，而美图秀秀则专注于人像细节的修饰。

## 修图流程详解

修图的核心流程是：先使用**美图秀秀**处理人像细节，再使用**VSCO**进行整体色调和质感调整，最后用**Snapseed**添加风格化滤镜。

### 第一步：使用美图秀秀处理人像

首先，我们打开美图秀秀，导入需要处理的照片。本节将以一张全身人像照和一张正脸人像照为例进行讲解。

对于全身人像照，我们主要关注整体轮廓和姿态的调整。

以下是美图秀秀“美容”功能下的常用工具及其作用：

*   **瘦脸瘦身**：此工具非常实用，可以调整脸部轮廓、身体线条。例如，可以将肩膀推得厚实一些，以增强安全感。操作时需注意力度，避免将直线推弯，导致不自然。
*   **祛斑祛痘**：用于消除面部的斑点、痘痘等瑕疵。选择合适大小的笔刷，点击瑕疵处即可。
*   **祛皱**：用于淡化皱纹、颈纹等。对于明显的血管或非皱纹纹理可能效果不佳。
*   **眼睛放大**：可以适度放大眼睛，使眼神更有神。使用时将“+”号对准瞳孔即可。
*   **增高塑形**：可以调整腿部长度，改善身材比例。
*   **去黑眼圈**：用于淡化眼周暗沉。

**不推荐使用的功能**：
*   **一键美颜/美妆**：效果夸张，容易失真，且美妆功能可能显得不够阳刚。
*   **智能优化**：自动化程度高，缺乏个人控制。
*   **面部重塑**：自动调整面部结构，效果不可控，失去了手动修图的意义。

处理完人像细节后，点击保存即可。

### 第二步：使用VSCO调整色调与质感

上一节我们使用美图秀秀完成了人像的基础修饰，本节中我们来看看如何使用VSCO为照片增添高级质感。

打开VSCO，导入从上一步保存的照片。点击编辑界面，我们将手动调整以下参数：

1.  **曝光**：控制图片明暗。向右滑动变亮，向左滑动变暗。公式可表示为：`图片亮度 = 原始亮度 + 曝光调整值`。
2.  **对比度**：调整色彩对比。向左滑动对比减弱，画面变灰；向右滑动对比增强，色彩更鲜明。
3.  **锐化**：增强图片细节和清晰度。建议适当调高，使图片更“结实”。调整时可放大图片观察边缘细节。
4.  **清晰度**：微调结构对比。不建议调得过高，否则会产生不自然的“浮雕感”。
5.  **饱和度**：控制色彩鲜艳程度。向右滑动更鲜艳，向左滑动更清淡。
6.  **颗粒**：添加胶片质感。适当添加可以提升图片的复古氛围。
7.  **色调分离**：
    *   **阴影色调**：为暗部添加颜色，通常加一点蓝色（`HSL阴影 -> 蓝色 +`）。
    *   **高光色调**：为亮部添加颜色，通常加一点黄色或橙色（`HSL高光 -> 黄色 +`）。

每个参数的调整都需要自己反复滑动感受，找到最适合当前照片的数值。调整完成后，保存图片。

### 第三步：使用Snapseed添加风格化滤镜

经过VSCO的精细调整后，我们最后使用Snapseed为照片添加一个画龙点睛的风格滤镜。

打开Snapseed，导入处理好的照片。点击【工具】->【复古】。在众多滤镜中，编号7、8、9、10的滤镜都比较常用，可选择一款应用。

如果觉得滤镜效果过重，可以在应用滤镜后，**用双指在屏幕上上下滑动**，调出参数调整菜单：
*   **样式强度**：降低此项可减弱滤镜效果。
*   **晕影强度**：调整此项可控制画面四角的暗角效果，增强则突出画面中心。

调整至满意后，导出保存为副本即可。

## 正脸人像修图示范

现在，让我们将学到的流程应用在一张正脸人像照上。

1.  **瑕疵修复**：在美图秀秀中使用“祛斑祛痘”功能，选取与瑕疵大小匹配的笔刷，点涂去除痘痘、疤痕、黑头等。需要耐心细致。
2.  **轮廓调整**：使用“瘦脸瘦身”功能，微调脸部轮廓（如下颌线）、鼻梁线条，甚至可微调发型轮廓，使整体更和谐。
3.  **细节完善**：使用“去黑眼圈”淡化眼周暗沉，使用“眼睛放大”让眼睛更有神。
4.  **后续处理**：完成人像修饰后，同样遵循上述流程，使用VSCO调整质感，再用Snapseed添加复古滤镜。

## 课程总结

本节课我们一起学习了系统化的手机修图流程。

1.  **核心流程**：**美图秀秀（人像修饰）-> VSCO（质感调色）-> Snapseed（风格滤镜）**。
2.  **修图原则**：修饰的目的是弥补小缺陷、提升质感，而非改头换面。操作应精细、自然，避免过度。
3.  **关键技巧**：多使用手动调整功能，而非一键特效；每个参数的调整都要通过对比感受效果；最后的滤镜强度可通过滑动屏幕灵活控制。

最重要的是多加练习，用不同的照片尝试，你一定能找到最适合自己的修图风格，让每一张分享出去的照片都更具魅力。
