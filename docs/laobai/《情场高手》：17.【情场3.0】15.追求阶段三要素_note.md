# 《情场高手》课程17：🎯【情场3.0】15. 追求阶段三要素

在本节课中，我们将要学习追求阶段的核心目标与具体操作方法。在成功建立吸引、让女生对我们产生好感之后，关系便进入追求阶段。这个阶段是女生进行“废物测试”的高发期，她们会提出一系列问题来考验男生的真心与价值。掌握本课的三要素，能帮助你顺利通过测试，稳固关系。

上一节我们介绍了如何建立吸引，本节中我们来看看进入追求阶段后需要做什么。

## 追求阶段的核心：展现靠谱（DP）

在L3吸引阶段之后，女生更在意的是你是否**靠谱**，以及你是否真的喜欢她。因此，追求阶段最主要的目的就是做到**DP**，即让她感觉你是一个可靠、真心的人。

那么，如何让她觉得你真的喜欢她呢？我们需要关注以下三个要素。

## 追求阶段三要素

以下是实现“靠谱”（DP）目标的三个关键行动要素。

### 1. 筛选（C机）

首先，我们要让她感觉到你也在考察她，这个过程就是“筛选”。我们只有在认真考虑一段关系时，才会仔细挑选伴侣。反之，如果只是玩玩，就不会那么在意。因此，当你进行筛选时，她会明白你是认真的。

具体可以这样做：
*   **询问恋爱史**：例如，问她“你谈过几个男朋友？”。如果她回答“三个”，你可以说“比我多一个”，并配上一个委屈的表情。
*   **了解重要经历**：例如，问她“有和男生同居过吗？”。如果她说“没有”，你可以回应“那还好”。如果她说“有”，你可以进一步问“多久？”，然后根据她的回答（如“两个月”），表示“哦，那还好”。这传递出的信息是：你介意某些点（如同居时间），但你在综合考量。

关键在于，你表现出在意她的某些“缺点”（如恋爱经历多、某些生活习惯），但最终选择了包容和接受。这种“在意却接受”在她看来就是一种爱的表现。

### 2. 暖男行为

进入追求阶段后，你可以适当地对她好，表达关心和照顾。这与初期无原则的“跪舔”有本质区别。此时的好，是建立在已有吸引和好感基础上的升温行为，效果会很好。

以下是一些暖男行为的例子：
*   **留意细节**：例如，注意到她吃饭时会把葱花挑出来，下次单独约会点菜时，特意嘱咐所有菜都不要放葱。
*   **体现照顾**：一起走路时，让她走在靠人行道的内侧，自己走在靠近机动车的外侧。
*   **细微关怀**：上出租车时，用手为她垫一下头顶，防止她碰头。

这些细节能让她感受到你的用心和真诚，从而确信你是真的喜欢她。

### 3. 走心互动与进退策略（DT）

在互动中，需要掌握主动的节奏。男性的主动在于“靠近”，女性的主动则表现为“在男性靠近时给予配合或不拒绝”。

这里有一个核心原则：**当女生主动时，我们要先后退**。

这类似于逗猫：
*   一个东西轻轻晃动，会吸引猫咪好奇靠近。
*   当猫咪伸出爪子去挠时，如果这个东西突然扑上来，猫咪会感到威胁而立刻跑开。
*   正确的做法是，当猫咪（女生）主动“挠”你时，你稍微后退一点，这会让她更加好奇，进而再次靠近。
*   等她再次主动时，你再给予配合或正面回应，关系推进就会非常稳固。

如果女生主动时，你立刻急切地推进关系，她会觉得你过于随便或饥渴，反而可能退缩。先后退，能增加她的好奇和投入感。

## 课程总结

本节课中我们一起学习了追求阶段的核心方法与策略。

1.  **追求阶段的核心目标**是做到**靠谱（DP）**，让她感受到你的真心与可靠。
2.  **实现DP有三要素**：**筛选（C机）**、**暖男行为**和**走心互动与进退策略（DT）**。
3.  **筛选之所以有效**，是因为只有当我们真心喜欢一个人时，才会认真地去考察和筛选对方。
4.  **对女生好是必要的**，但只有在L4追求阶段进行才有效果，在前期吸引阶段效果不佳甚至反作用。
5.  **互动节奏的关键**：男人的主动是“靠近”；女人的主动是“配合靠近”。当女生主动时，我们要懂得先后退，以增加她的好奇与投入，待其再次主动时再稳妥推进关系。

掌握这些要素，能帮助你在追求阶段更好地通过女生的测试，让关系健康、稳定地发展。
