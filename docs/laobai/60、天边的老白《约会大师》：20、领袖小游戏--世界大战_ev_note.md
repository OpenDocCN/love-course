# 约会大师课程 60：领袖小游戏——世界大战 🎮

在本节课中，我们将学习一个名为“世界大战”的社交互动游戏。这个游戏的核心是通过团队对抗的形式，创造合作与交流的机会，从而自然地拉近参与者之间的距离。

## 游戏准备与分队 🤝

上一节我们介绍了游戏的基本概念，本节中我们来看看如何开始游戏。首先，我们需要将参与者分成两队。

游戏需要至少四人参与。如果是四个人，就分成两人一队。如果是六个人，就分成三人一队。分队的规则如下：

以下是分队的具体步骤：
1.  所有参与者通过“石头剪刀布”来决定谁有优先选择队友的权利。
2.  赢家可以从剩余的人中选择一人成为自己的队友。
3.  如果人数较多，可以重复“石头剪刀布”的步骤，由赢家依次挑选队友，直到队伍人数均等。

例如，四个人（A, B, C, D）进行游戏。A和B先通过“石头剪刀布”决定谁先选人。假设A赢了，A选择C作为队友。那么队伍就分成了A、C一队，B、D一队。

## 游戏规则与玩法 🎲

分队完成后，就可以开始游戏了。游戏的核心是对抗，但具体玩什么游戏可以由参与者自行决定。

以下是几种常见的游戏选择：
*   **石头剪刀布**：简单直接，适合快速开始。
*   **骰子（吹牛/大话骰）**：需要一些策略和演技，互动性更强。
*   **十五二十**：考验反应速度。

在游戏开始前，必须提前约定好惩罚规则。例如，约定败方队伍每人喝一杯酒，或者整队共喝两瓶酒。一个重要的规则是，如果一方队伍中有人能“打通关”（连续击败对方所有成员），惩罚是否需要翻倍，也必须在开局前说清楚。

游戏流程以“石头剪刀布”为例：两队轮流派人上场对决。败者下场，换本队下一人上场，直到一队的所有成员都被击败。根据事先约定的规则，败方接受惩罚。

## 游戏中的互动技巧 💬

这个游戏的精髓不在于胜负，而在于利用团队合作创造自然的互动机会。作为队友，你们是“一致对外”的。

在游戏过程中，可以运用以下技巧来增进默契和亲密感：
*   **交流与商量**：在玩骰子等游戏时，可以自然地靠近队友，低声讨论战术，查看对方的骰子情况。这是一个拉近物理距离的绝佳借口。
*   **互相支持与调侃**：当队友表现出色时，可以给予鼓励和称赞。当队友失误或对方队伍失误时，可以进行友好的调侃，营造轻松有趣的氛围。
*   **共同承担**：当队伍失败需要接受惩罚时，可以主动提出为队友多分担一些，这能体现出关心和担当，创造“安全感”。

通过几轮游戏的合作，队友之间的联系和亲密感会自然而然地更进一步。

## 总结 📝

本节课我们一起学习了“世界大战”这个社交游戏。我们了解了如何通过“石头剪刀布”进行分队，明确了游戏规则和惩罚机制必须在开局前约定。更重要的是，我们探讨了如何利用队友的身份，在游戏过程中通过交流、支持和共同承担来创造互动机会，从而有效地拉近彼此的距离。记住，游戏只是载体，真诚的互动才是目的。
