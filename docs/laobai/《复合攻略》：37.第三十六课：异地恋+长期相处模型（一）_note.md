# 《复合攻略》：37.第三十六课：异地恋+长期相处模型（一）🪁

在本节课中，我们将要学习异地恋的本质，并开始探讨一套旨在维持和深化异地恋关系的相处模型。我们将首先分析异地恋中恐惧感的来源，然后介绍模型的第一步——保持精彩。

## 概述：异地恋的恐惧源于何处？

之前课程里，我们说到要详细讲解异地恋和长期相处。这节课开始，就给大家讲一套完美的异地恋方法模型和长期相处之道。

首先，第一个问题是异地恋很可怕。这并不是我说的，是网络上各种知乎、微博上都在讨论，觉得异地恋缺乏安全感，太可怕了。很多人谈异地恋就色变。

那到底异地恋它可怕在哪里呢？我们在害怕的时候，到底在害怕什么？是害怕距离，还是害怕被挖墙角，还是有其他害怕的东西？这里的本质是什么？我们就要去谈这个本质。

为什么要扯这个事儿，而不直接去讲具体的方法技巧？因为如果你把你到底在害怕什么、把异地恋的本质核心搞清楚了，其他问题都会迎刃而解。

在这之前，我先讲两个小的案例。

以下是两个反映异地恋常见问题的案例：

*   **案例一：** 我之前有一个学员，他交了一个女朋友是异地恋。他女朋友在美国读书一年，其实他女朋友回来两次，但是哥们儿一年当中飞了3次美国。第三次我才知道，因为当时我找她要带东西。我问她为什么要去找女朋友。她说因为吵架，女朋友一生气就要去喝酒、去玩。她不放心女朋友，觉得不安全。
*   **案例二：** 另一个哥们是广州的，他女朋友在上海，是一个平面模特。女朋友工作后经常会有一些社交局。这个同学很不爽，聊天到了后期基本上就三个问题：“在哪”、“在干嘛”、“和谁在一块”。一开始女生还会回答，但到后来女朋友实在不耐烦了。他女朋友直接跟他说：“我是你女朋友，不是你的员工，我不用给你打卡。你为什么每一次都问我在哪在干嘛和谁在一起，而根本不去关心我是否开心，我的情绪是什么样的？我们分手吧。”这就是他们最后一次谈话。

你看这两个案例放在一起，你会发现他们有共同点，就是没有安全感。换句话说，就是对女朋友不放心。这就是异地恋真正不和谐的地方，就是对女朋友不放心。

换一句话来说，就是对女朋友失去了掌控感。如果我们把谈恋爱比作放风筝，那女朋友就是你的风筝，你拿着一根线在这里放。如果她一直在你的掌控当中，你可以在下面抽烟，可以做任何事情。你不会管她往这边飘还是往那边飘，她跟谁在一起，她今天去唱歌还是去看电影，你完全无所谓，因为你有掌控感。

可是异地恋距离一拉开，很多信息不对称，你就会觉得这根线突然断了。主要是断了线的风筝，你突然间发现你失去了对她的掌控感。

所以当我们在害怕异地恋的时候，我们害怕的是什么？我们害怕的是失去这种掌控感。我们害怕的是线断了，你掌握不了她，你收不回来。你想再次收回她的时候不行了。这是你真正害怕的东西，这是你真正恐惧的地方。

所以，异地恋怎么去掌控你的女朋友？异地恋怎么去重新获得这种掌控感？才是我们真正要去解决的核心中的核心。

## 为何会感到失控？

那么，为什么我们会有这种失控的感觉呢？主要有以下几点原因。

以下是导致异地恋中失控感的七个主要因素：

1.  你害怕她会因为别的男人而对你移情别恋。
2.  你害怕她会因为距离对你淡忘。
3.  你害怕她会因为各种原因给你减分。
4.  你害怕她会因为寂寞而背叛你。
5.  你不知道她能否抵得住诱惑。
6.  你不知道她能否抵得住寂寞的侵蚀。
7.  你不知道她需不需要你了，以及你们感情基础是否充分，是否可以突破空间带来的障碍。

说完之后是不是觉得有点恐怖？但是这个确确实实是每一个异地恋人内心在想的事情。

## 解决方案：异地恋相处模型

针对我们要如何去掌握这种对女朋友的掌控感，我设计了一个异地恋的相处模型。只要按照这六步去做，我们的感情就会得到升华。

以下是异地恋相处模型的六个步骤：

1.  保持精彩
2.  设定规则
3.  鼓励社交
4.  渗透生活
5.  共同成长
6.  筹备见面

可能你乍一听还不知道什么意思，有点懵。没关系，我们后面的话会对这六点进行深入展开的讲解。

我们看到，前面三步（保持精彩、设定规则、鼓励社交）其实是“干掉你的竞争对手”，也就是那些对你女朋友虎视眈眈的其他男人。在干掉你们的竞争对手之后，再让你们的感情有进一步的提升。

## 第一步：保持精彩 🎯

上一节我们介绍了异地恋的核心恐惧和整体模型框架，本节中我们来看看模型的第一步——保持精彩。

很多人在谈了异地恋之后，就不断地去让自己关注对方，不断地去问。比如他们是高中恋爱，上大学之后，他不断去问她的生活怎么样、身边什么情况，然后自己在寝室里面打游戏，就不断把自己的生活完全寄托在女生身上。这是错误的。

那正确的应该是什么呢？应该是你的生活很精彩，必须足够的精彩，然后邀请你的异地恋女朋友跟你一起来分享。

我记得之前有一个同学在咨询我一个问题的时候，就是他跟女朋友高中毕业，女朋友去了上海，他自己留在了合肥上大学。他发现女朋友第一周在军训，第二周就去参加各种各样的社团，比如滑板社，或者去外面学了纹身。然后他自己呢，就是不断生活在一种对女朋友的担心和恐慌当中。

他担心女朋友去学滑板，会不会认识一些滑板少年；去学纹身，会不会认识些社会上的人，然后把他给PK掉了。然后他就不断地去骚扰、去追问他的女朋友“你到底还爱不爱我”，最后导致他女朋友受不了了，要跟他分手。

这就是一个错误的方式，就是他没有去保持他自己的生活精彩。这个理念就是：你必须把自己的生活弄得很精彩，然后让女生加入进来。如果是面对面，那是加入进来；如果是异地的话，就是有东西可以分享给她，她依然会觉得你很优秀。

说句难听的，你已经跟她拉开距离了，都说了“近水楼台先得月”，你已经拉了这么远，你能靠什么呢？你靠骂她、靠去指责她、靠去要求她吗？这些肯定不行。你只能靠你精彩的生活去吸引她。

吸引到什么程度呢？吸引到让她觉得她身边那群同学、同事，其他男的都不如你有吸引力，都不如你的生活精彩，都不如你这么有故事。你必须做到这样一个程度：你的精彩要让她觉得她身边的一切都是幼稚的，都是一些一般般的人。这就是你要达到的目的。

那么，通过三个途径可以去达到“保持精彩”这样的状态。

以下是实现“保持精彩”的三个具体途径：

1.  **朋友圈：** 也就是微信朋友圈。你不需要每一条都是你在做什么、任何一件事都发照片私发给你朋友。你只要把你精彩的生活，发一部分到你的朋友圈里，让她从朋友圈里感受到你的生活。这里要有一个基础：你真的不是每一天都在寝室里面打游戏，你要做一些有意思的事，比如去足球场上发泄荷尔蒙打打篮球，或者做一些有意思的事。这也是倒逼你成为一个优秀、精彩男人的一个途径。
2.  **社交生活：** 还是那句话，在城市里不要天天下班就回家，不要只知道跟几个朋友喝个啤酒吃吃火锅，不要三点一线。你要去参加一些聚会，去KTV，去玩，去自驾游，各种各样的社交生活不能断。
3.  **丰富兴趣生活：** 有条件呢可以去潜水、旅行，没条件的可以弹个吉他、玩滑板。这都是OK的，可以增加你的兴趣生活，也是保持精彩的一个部分。

好，我们回顾一下。第一个你要“保持精彩”，保持精彩主要分为三个路径：第一，朋友圈；第二，社交生活；第三，兴趣生活。
