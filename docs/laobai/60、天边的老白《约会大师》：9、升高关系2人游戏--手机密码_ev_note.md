# 课程60：天边的老白《约会大师》第9课：升高关系双人游戏——手机密码魔术 🎩

在本节课中，我们将学习一个有趣的互动小游戏。通过这个游戏，你可以在约会或社交场合中，以一种巧妙且令人惊讶的方式，“猜中”对方的手机密码，从而创造轻松愉快的互动氛围，并有机会自然地拉近彼此距离。

## 游戏准备与开场

上一节我们介绍了游戏的整体概念，本节中我们来看看具体的操作步骤。首先，你需要邀请一位朋友（例如一位女生）参与这个游戏。

以下是游戏的开场引导步骤：
1.  向对方提出挑战：“我信不信我知道你的手机密码？”
2.  将游戏包装成一个小魔术或心理游戏，以增加趣味性。
3.  请对方拿出自己的手机并配合你的指令。

## 核心操作步骤

在成功引起对方兴趣后，就可以开始游戏的核心操作部分了。

以下是引导对方进行计算的关键步骤：
1.  请对方解锁手机，并打开**计算器**应用。
2.  让对方在计算器中**随便输入一个六位数**。
3.  接着，让对方用这个六位数**减去他/她自己的手机密码**。
4.  请对方告诉你最后的**计算结果**是多少。
5.  最后，让对方直接**锁屏手机**，并将手机交给你。

## “读心”与揭秘表演

拿到手机后，便是展示“魔力”的时刻。这个过程需要一些表演技巧来增强效果。

以下是营造悬念并最终“猜中”密码的表演流程：
1.  你可以拉着对方的手，看着对方的眼睛，声称能从眼神中“读出”密码。
2.  在短暂的对视和互动后，拿过对方的手机。
3.  在锁屏界面，从屏幕底部向上轻滑（以苹果手机为例），调出控制中心或快速访问菜单，点击其中的**计算器**图标。
4.  此时计算器界面会显示刚才的运算结果。你只需连续按下两个键：**AC**（全部清除） -> **=**（等于）。
5.  神奇的事情发生了：屏幕上会重新显示出对方刚才**减去的那个数字，也就是他/她的手机密码**。

## 原理与注意事项

这个游戏看似神奇，其实原理非常简单，它利用了计算器的一个特性：**连续按“AC”和“=”键，可以重新显示上一次的运算参数**。

用公式表示即：
`AC` -> `=` → 显示上一次减法运算中的减数（即手机密码）

但成功的关键在于表演和细节处理。

以下是确保游戏顺利进行的几个重要注意事项：
*   **设备限制**：此游戏主要适用于**苹果iPhone**的计算器应用，其他手机的计算器逻辑可能不同。
*   **清除痕迹**：在通过上述方法看到密码后，务必立即再次按下**AC**键清空计算器显示，然后再锁屏。否则对方拿回手机时可能会看到密码还留在屏幕上。
*   **牢记密码**：看一眼密码后要瞬间记住，在后续的“猜数字”表演中需要用到。
*   **表演节奏**：整个过程的重点不是技术，而是互动。你可以慢慢“猜”出每个数字，通过询问“这个数字比5大吗？”等方式与对方互动，最终成功解锁手机，制造惊喜。

## 互动情景演示

为了让大家更直观地理解，我们来模拟一段完整的对话和操作情景。

假设对方的手机密码是 **740928**。
*   **你**：“打开计算器，随便输入一个六位数，比如1234567。”
*   **对方**：输入 `1234567`。
*   **你**：“现在减去你的手机密码。”
*   **对方**：输入 `- 740928`，计算器显示结果为 `493639`。
*   **你**：“结果是493639对吗？好，现在锁屏把手机给我。”
*   （你拿到手机后，从锁屏界面调出计算器，此时显示为 `493639`。）
*   （你按下 `AC`，屏幕清零，再按下 `=`，屏幕显示 `740928`。）
*   （你记住密码 `740928`，再按 `AC` 清零，然后锁屏。）
*   **你**：（开始表演）“让我感受一下…第一个数字是…7吗？”（输入7）“第二个是…4？”（输入4）… 直至全部“猜中”并解锁手机。
