# 老白《约会高手》第13课：金字塔喝酒小游戏 🃏

在本节课中，我们将学习一个名为“金字塔”的多人喝酒小游戏。这个游戏适合朋友、同事聚会时玩耍，人数从两人到多人均可，人越多通常越有趣。我们将详细介绍游戏的道具准备、规则流程以及核心玩法。

## 游戏准备与摆牌 🏗️

上一节我们介绍了游戏的适用场景，本节中我们来看看如何准备和布置游戏。

首先，你需要一副扑克牌。游戏开始时，需用部分扑克牌垒出一个金字塔形状。

**摆牌规则**：从最底层开始，每一层比下面一层多一张牌。例如，最底层放6张牌，上一层放5张，再上一层放4张，以此类推，直到塔尖放1张牌。

摆好金字塔后，将手中剩余的牌平均发给所有玩家。发牌方式可以是一张一张轮流发，确保每人手牌数大致相同。

## 游戏核心规则 ⚙️

准备好了金字塔和手牌，接下来我们进入游戏的核心规则环节。

金字塔的每一张牌都对应一个“喝酒口数”。我们需要先定义一杯酒由多少“口”组成（例如，**一杯酒 = 三口**）。然后，为金字塔的每一层分配基础口数：

*   最底层（牌最多的一层）：每张牌对应 **1口**。
*   往上一层：每张牌对应 **2口**。
*   再往上一层：每张牌对应 **3口**。
*   … 以此类推，塔尖的牌对应口数最高。

以下是各层口数对应的公式描述（假设塔有6层）：
`第N层单张牌口数 = (金字塔总层数 + 1 - N)`
例如，一个6层的塔，最底层(N=6)口数为1，最顶层(N=1)口数为6。

## 游戏进行流程 🔄

理解了基础口数，现在我们来一步步看看游戏如何进行。

游戏按顺时针或逆时针顺序轮流进行。轮到你的回合时，你需要从金字塔中翻开任意一张尚未被翻开的牌。

1.  **翻牌与揭示口数**：翻开一张牌，并公布这张牌的点数（如K、Q、8）以及它所在层数对应的口数（例如，翻开塔尖的牌，就是6口，即两杯酒）。
2.  **出牌与指定**：此时，所有玩家（包括翻牌者）可以查看自己手牌。**翻牌者必须首先行动**：如果手上有与翻开牌点数相同的牌，可以打出这张牌，并指定任意一名其他玩家喝该牌对应的口数。
3.  **反击与救援**：被指定的玩家如果手中也有相同点数的牌，可以打出进行“反击”，将喝酒责任翻倍后反弹给指定他的人（例如，从6口变为12口）。**其他未被指定的玩家**如果手上有相同点数的牌，也可以选择打出进行“救援”，代替被指定者承受喝酒责任，并可以重新指定一名玩家喝酒（口数不变）。
4.  **无人出牌的后果**：如果翻牌者自己手上没有可出的牌，或者他出牌指定后，无人（包括被指定者）打出牌进行反击或救援，那么最初被指定的玩家（或翻牌者自己）就必须喝下对应的口数。

以下是核心互动的代码逻辑描述：
```python
# 伪代码逻辑
if 翻牌者手上有匹配牌:
    翻牌者出牌并指定玩家A喝酒
    if 玩家A手上有匹配牌:
        玩家A反击，喝酒口数翻倍，责任反弹给翻牌者
    elif 其他任何玩家手上有匹配牌:
        该玩家救援，代替玩家A喝酒，并可重新指定玩家B
    else:
        玩家A喝酒
else:
    翻牌者自己喝酒
```
**关键点**：出牌时机是一种策略。有时保留牌到最后可能更有利。

## 游戏结束与结算 🏁

当金字塔所有的牌都被翻开并结算完毕后，游戏进入最终结算阶段。

此时，检查所有玩家手中剩余的手牌。**每剩余一张手牌，就需要喝一口酒**。例如，如果你还剩3张牌未打出，就需要额外喝3口酒。

这个阶段可能导致大量饮酒，是游戏被称为“喝酒很猛”的原因之一。

## 策略与社交应用 💡

这个游戏不仅是喝酒，更包含丰富的社交策略。

你可以利用游戏规则进行互动：
*   **保护朋友**：如果你在意的人被指定喝酒，你可以用手中的牌“救援”他/她。
*   **增加趣味**：可以针对游戏中的任何人，善意或玩笑地使用“指定”功能，活跃气氛。
*   **把握时机**：不要急于打出所有的牌。观察局势，在关键时刻使用“反击”或“救援”，可以控制场上的喝酒节奏。

这个游戏非常适合同事聚餐、朋友旅行、宿舍聚会等场合，能快速炒热气氛，但请注意饮酒适量。
