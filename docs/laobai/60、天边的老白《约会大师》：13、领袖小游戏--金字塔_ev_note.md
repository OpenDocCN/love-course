# 60、天边的老白《约会大师》：13、领袖小游戏--金字塔_ev_note

在本节课中，我们将学习一个名为“金字塔”的多人喝酒小游戏。这个游戏适合朋友、同事聚会时玩耍，人数从两人到多人均可，人越多通常越有趣。我们将详细介绍游戏的道具准备、规则流程以及核心玩法。

## 游戏准备与摆牌 🏗️

上一节我们介绍了游戏的适用场景，本节中我们来看看如何准备游戏。

游戏需要一副扑克牌。首先，用部分扑克牌在桌面上垒出一个金字塔形状。

**摆牌规则公式**：从底层开始，每一层的牌数比下一层多一张。例如，底层放6张，上一层放5张，再上一层放4张，以此类推，直到顶层放1张牌。

摆好金字塔后，将手中剩余的牌平均发给所有玩家。发牌可以一张一张轮流进行，确保每人手牌数量大致相同。

## 游戏规则详解 📜

准备好了金字塔，接下来我们深入讲解游戏的核心规则。

金字塔的每一层牌都对应着需要喝下的“口数”。通常，约定一整杯酒等于三口。金字塔最底层（牌最多的一排）每张牌代表**一口酒**，往上一层每张牌代表**两口酒**，依次递增。顶层唯一的一张牌代表酒量最多，例如六口酒（即两杯）。

以下是游戏的核心流程：
1.  **翻牌**：从金字塔顶层或任意一层，翻开一张牌，例如翻出了一张“6”。
2.  **指定与喝酒**：翻牌者查看自己手牌。如果手中有相同点数的牌（如另一张“6”），可以将其打出，并**指定**任意一位玩家喝下该层对应的口数（例如，第二层是两口）。
3.  **反击与翻倍**：被指定的玩家如果手中也有相同点数的牌，可以将其打出进行**反击**，将喝酒指令反弹回翻牌者或其指定的另一人，并且**喝酒口数翻倍**（例如，从两口变为四口）。
4.  **无人响应**：如果翻牌后，无人打出对应点数的牌，则由**翻牌者自己**喝下该层对应的口数。

## 游戏策略与技巧 🎯

了解了基本规则后，本节我们探讨一些游戏中的策略和有趣的应用。

出牌时机是关键策略。不要急于打出救人的牌，可以等待观察，在最后时刻出手，这样可能更有利。

这个游戏具有很强的社交互动性。例如，你可以选择保护你想照顾的人，也可以针对你看不顺眼的人。如果看到喜欢的女生被指定喝酒，而你手中有救命的牌，就可以下场“英雄救美”。

游戏持续进行，直到金字塔所有的牌都被翻开。最后，计算每位玩家手中剩余的手牌，**每剩一张牌，需喝一口酒**。

## 适用场景与总结 🎉

在最后一节，我们来回顾一下这个游戏的适用场合并总结要点。

“金字塔”是一个纯粹的喝酒游戏，节奏快，喝酒量可能很大。它非常适合同事聚餐、同学宿舍聚会、朋友旅行或自驾游时，用来活跃气氛、促进交流。

**本节课总结**：
我们一起学习了“金字塔”喝酒小游戏的完整玩法。主要内容包括：
1.  用扑克牌搭建**金字塔**并分配手牌。
2.  理解每层牌对应的**喝酒口数**。
3.  掌握**翻牌、指定、反击**的核心流程，其中反击可使口数**翻倍**。
4.  学习**保留手牌、伺机而动**的策略和游戏的**社交互动**技巧。
5.  记住游戏结束后的**结算规则**：剩余手牌需罚酒。
