## 如何做出正确邀约

通过延续话题加深了解后，接下来便可以尝试邀约。一切不以邀约为目的的聊天，都是无效的。

在生活中，你常会听到“有空一起吃饭”这类客套话，你通常不会拒绝，只会回答“好呀”。模糊邀约也是如此，它通常不会被直接拒绝。

**错误做法**：直接指定时间，如“明天下午三点见面”。这会让女生觉得不受尊重，容易拒绝。她会想：“你怎么知道我明天有空？”或“我凭什么要听你的安排？”

**正确做法**：出于礼貌和对女生的尊重，应先询问对方的空闲时间。可以问：“你一般什么时候比较有空？” 这种问法显得细心且带有试探性，尊重对方的时间安排。

2.  **“我一般周末有空”**：如果你在周一、周二聊天时得到这个回复，不要立刻敲定周末的约会。因为距离周末还有好几天，变数较大，她自己也未必能确定周末的安排。

**邀约核心原则**：注意邀约的方式方法，尽量减少女生拒绝的机会，同时不要暴露过强的需求感。这将极大提高你邀约成功的概率。
