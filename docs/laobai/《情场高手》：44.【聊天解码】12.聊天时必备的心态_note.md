# 《情场高手》：44.【聊天解码】12.聊天时必备的心态_note

在本节课中，我们将学习与女生聊天前应具备的核心心态。心态是成功沟通的基础，正确的思维模式能让你在互动中更自然、更自信。我们将探讨三种关键心态，并通过具体技巧和案例帮助你理解如何应用。

## 概述：心态建设的重要性

很多兄弟感情经历较少，面对女生时会感到不自然。自信和坦然说起来简单，做起来却困难。本节课程旨在通过心态建设，帮助你克服紧张，建立有效的沟通基础。

## 第一节：从“暗示”到“去做”心态

上一节我们提到了心态的重要性，本节中我们来看看第一种核心心态：“去做”心态。

许多传统的心态建设方法，例如不断暗示自己“我很优秀、我很自信”，效果有限。你只是在欺骗自己，世界的普遍价值观不会因此改变。例如，女生天生可能更青睐高富帅，这是你无法通过自我暗示改变的客观事实。你需要做的是先接受这个事实，然后着手改变自己。

追求女生的核心，不在于你如何看待世界，而在于女生如何看待世界。因此，你需要的是“去做”心态——通过实际行动提升自己，从而获得真实的自信。

**公式：真实改变 → 获得正面反馈 → 建立自信**

我自身的经历就是一个例子。我曾因身高而自卑，尝试过自我催眠等方法但收效甚微。后来，我开始彻底改造形象：健身、提升品味、广泛阅读、接触不同的人。这些实实在在的行动让我获得了他人正面的反馈，从而真正相信自己变得优秀了。

## 第二节：提供价值的心态

从“去做”心态中，衍生出两个重要的聊天前心态。首先，我们来看**提供价值的心态**。

开始聊天前，你必须明白：你的目标是尽量多**提供价值**，而非**索取价值**。请记住这个核心公式：

**提供价值 = 产生感觉**

只要你为女生提供了足够多的价值，她对你的感觉就会更好。提供价值主要分为两类：情绪价值和实际价值。

### 1. 情绪价值

情绪价值并非指你一个人滔滔不绝，而是指你能够**影响她的情绪**，让她在与你聊天时产生波动，例如一会儿笑、一会儿紧张、一会儿期待、一会儿失落。

以下是三种可以制造情绪波动的聊天技巧：

**技巧一：推拉**
推拉是通过先“推开”（轻微否定或调侃）再“拉近”（肯定或亲近）来调动对方情绪。
```
案例：
女生：（发来与朋友的合影）
你：一看就知道右边的是你。（拉——暗示她好看）
女生：因为我最美嘛。
你：不是啊，右边的胸大一点，你的明显没这么大。（推——调侃）
女生：打死你。
你：但我喜欢胸小点，穿衣服好看。（拉——再次肯定）
```

**技巧二：曲解**
曲解是故意以幽默、自恋的方式误解对方的话，引导她来追逐或解释，营造骄傲风趣的感觉。
```
案例：
女生：昨天居然梦到你了。
你：占我便宜没有？
女生：（发表情）
你：完了，肯定被你占了。
女生：你希望被我占还是没被占？
你：我应该是反抗了，而且很强烈。
```

**技巧三：打压**
打压通常用在女生态度突然变冷淡或姿态变高时，用于维持你自己的框架。注意，这不是人身攻击，而是态度上的轻微“回击”。
```
情况一：女生回复敷衍
你可以回复：“你平时都是这样和人聊天的吗？” 或 “好好说话。”
情况二：聊天中女生因某句话反馈变差
（案例中，女生对“月底见面”表示疑虑）
你：不靠谱你还聊？我反正没时间和我觉得不靠谱的人闲聊。（摆出强硬框架）
```

这些技巧可以组合使用。例如，将曲解与打压结合，在拉升关系时，如果女生自我感觉过于良好，可以适当打压。

### 2. 实际价值

实际价值指女生能直接通过视觉感知到的价值。你可以通过分享生活状态来提供这种价值。

以下是提供实际价值的示例：
*   **场景**：在高级餐厅吃饭。
    *   **行动**：私聊发一张美食或环境照片，描述状态。
*   **场景**：在健身房健身。
    *   **行动**：发一张健身照片或一段小视频。
*   **场景**：开车、做饭、参加有趣活动。
    *   **行动**：拍摄照片或视频分享。

**注意**：索取价值则相反，表现为不断向女生提问，如“在干嘛？”、“吃了吗？”。这些提问本身价值很低。同时，确保你分享的内容是高质量的，避免在脏乱差的环境拍照，否则会产生反效果。

## 第三节：游戏者心态

除了提供价值，另一个关键心态是**游戏者心态**。这包含两种子心态。

**第一种：一边玩一边聊的心态**
很多兄弟聊天时守着手机，秒回消息，这会暴露强烈的需求感。正确的方式是像打游戏一样：收到消息后，可以先把手上事情做完再回复。回完一条后，就把手机放在一边，过段时间再看。这能有效降低你的得失心。

**第二种：选择性心态**
你不能只和一个女生聊天。如果只有一个目标，你会投入全部精力，得失心变得极重，容易因对方一句话而紧张。女生不会喜欢紧张兮兮的男人。

选择性心态就像打游戏：只有一条命时会非常紧张；命很多时则放松自如。拥有更多聊天对象（资源），你自然会更从容。这不仅是一种心态，更是一种实际的状态——你积累了大量社交资源，从而拥有选择权。

## 总结与心态边界

本节课我们一起学习了与女生聊天的三种核心心态。

首先，心态的基础是 **“去做”心态**，通过真实行动获得自信。由此衍生出 **“提供价值的心态”**，牢记“提供价值=产生感觉”，并学会通过情绪价值（推拉、曲解、打压）和实际价值（分享高质量生活）来实践。最后，我们掌握了 **“游戏者心态”**，包括一边玩一边聊和保持选择性，以降低需求感，保持从容。

最后，必须认识到：我们不是人民币，无法保证每个人都喜欢自己。你的精力应放在那些对你**反馈还不错**的女生身上。不必纠结于一开始就对你冷淡的人，在自身没有犯错的情况下，试图让她们回心转意通常很困难。

以上，就是你在开始微信聊天前，应该具备的核心心态和状态。
