# 《情场高手》：46.【聊天解码】14.聊天中的发散性思维_note

在本节课中，我们将学习如何运用发散性思维来解决聊天时大脑空白的难题。我们将探讨男女思维的差异，并掌握三种实用的方法，让你与女生的对话变得轻松有趣。

---

许多人在与女生聊天时会感到大脑一片空白，不知道如何回应。造成这种情况通常有两个原因。一是心态问题，这类人与朋友聊天时滔滔不绝，但与女生交流时就不知所措。另一种情况是性格内向，与任何人交流都感到困难。

上一节我们提到了聊天困境的成因，本节中我们来看看解决问题的核心：理解思维差异。

男人和女人的思维模式有所不同。男人的思维通常是线性的，倾向于直接解决问题。而女人的思维是网状的，更具发散性，话题可以四处跳跃。你必须学会这种网状的发散性思维，才能与女生畅聊。

这个思维模式任何人都可以学会，不要以性格内向为借口。通过实战研究，我们总结了三种具体方法。

以下是三种帮助你开启发散性思维的方法：

1.  **关键字法**
    将女生的回复拆解成多个关键词，并针对每个关键词进行回复。
    *   **公式**：`回复 = 针对(关键词)的回应`
    *   **示例**：女生说“我和闺蜜在南京路逛街呢”。
        *   关键词“我”：可以回复“羡慕，我还在苦逼地赶方案呢。”
        *   关键词“闺蜜”：可以回复“看你那么瘦小，缺不缺个拎包的男闺蜜？”
        *   关键词“南京路”：可以回复“南京路据说坏人比较多，你可别跟着他们学坏了。”
        *   关键词“逛街”：可以回复“这么喜欢逛街，看来我得去健身了，不然以后都没力气陪你逛了。”
    这样你便一下子拥有了多种回复选择，只需筛选最合适的一条即可。

2.  **联想法**
    通过女生的话，联想到其他相似的事物或场景，并进行类比回复。
    *   **公式**：`回复 = 由(话语)联想到(事物/场景)`
    *   **示例**：中午12点，女生说“还在睡觉”。
        *   联想到“猪”：可以回复“这个点儿还在睡觉，跟我奶奶家养的猪好像。”
        *   联想到“懒”：可以回复“这么懒可不行，你得好好工作。”
        *   联想到“床”：可以回复“都这个点了，你还在床上，看来你床上功夫还蛮厉害的。”
    联想法能有效打开话题，为你提供更多回复思路。

3.  **感受评论法**
    不对女生回复的具体内容做评价，而是对她这句话带给你的整体感受进行评论。
    *   **公式**：`回复 = 对(整体话语)的感受评价`
    *   **示例**：女生生气地说“你怎么能这样啊，你太污了吧！”
        *   感受评价：可以回复“你好凶呀。”或“你生气的样子还蛮可爱的，我以前以为你一点情绪都没有呢。”
    这种方法跳出了具体内容，直接回应情绪，常常能产生意想不到的效果。

掌握了以上三种方法，面对女生的任何一句话，你都能衍生出多种回复方式，只需从中挑选最合适的一条即可。

有些兄弟不知道如何开启对话，例如互发表情后，容易陷入“很高兴认识你，我是XXX”这种无趣的自我介绍模式。

这里提供一些可直接使用的话术：
*   对方回复微笑表情，你可以说：“小姑娘看起来挺爱笑的。”
*   如果对方朋友圈自拍很多，你可以说：“喜欢自拍的女生应该是单身了。”如果她问为什么，你可以回答：“不然为什么不让男朋友出镜呢？”
*   如果非要自我介绍，可以加上一个形容词来引导话题方向，例如：“你好，我是**羞涩的**XXX。”或“Hello，我是**会拍照的**XXX。”如果聊到私密话题，甚至可以说：“你好，我是**火好的**XXX。”
