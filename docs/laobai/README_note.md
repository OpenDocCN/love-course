# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量与数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“Hello World”存储在其中：
```python
message = "Hello World"
```
此后，当我们使用 `message` 时，Python就知道我们指的是“Hello World”这个值。

## 基本数据类型 🔢

理解了如何存储数据后，我们需要知道能存储哪些类型的数据。Python有一些内置的基本数据类型，它们决定了数据可以进行的操作。

以下是几种最常用的数据类型：

*   **整数**： 表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**： 表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**： 表示文本信息，需要用单引号或双引号包围，例如 `‘你好’`, `“Python”`。
*   **布尔值**： 只有两个值，`True`（真）或 `False`（假），常用于逻辑判断。

## 如何查看和转换类型？ 🔄

在编程时，有时我们需要知道一个变量的类型，或者将一种类型转换为另一种类型。

使用 `type()` 函数可以查看变量的数据类型：
```python
number = 100
print(type(number))  # 输出：<class 'int'>
```

如果需要转换类型，可以使用特定的函数。例如，将字符串转换为整数，或将整数转换为字符串：
```python
# 字符串转整数
num_str = "123"
num_int = int(num_str)

# 整数转字符串
num_int = 456
num_str = str(num_int)
```
这种操作在接收用户输入（默认为字符串）或拼接输出信息时非常有用。

## 动手试一试：一个简单示例 ✍️

现在，让我们将以上知识结合起来，完成一个简单的程序。这个程序会存储你的名字和年龄，然后打印一句问候语。

以下是代码步骤：

1.  创建两个变量，分别存储姓名（字符串）和年龄（整数）。
2.  使用 `type()` 函数检查年龄变量的类型。
3.  将年龄转换为字符串，以便与其他文本拼接。
4.  打印一条包含这些信息的句子。

```python
# 1. 创建变量
name = "小明"
age = 20

# 2. 检查类型
print(type(age))  # 输出：<class 'int'>

# 3. 类型转换（为拼接做准备）
age_str = str(age)

# 4. 打印信息
print(name + "今年" + age_str + "岁了。")
# 输出：小明今年20岁了。
```
