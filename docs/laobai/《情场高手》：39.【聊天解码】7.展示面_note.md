# 《情场高手》课程39：聊天解码（7）—— 展示面 🖼️

在本节课中，我们将要学习“展示面”的概念及其在社交互动中的重要性。展示面是个人形象的数字化呈现，核心在于通过社交媒体（如朋友圈）有效地展示自己，吸引他人注意并开启对话。

## 什么是展示面？

上一节我们介绍了聊天的基础，本节中我们来看看如何通过展示面为聊天创造机会。展示面是追女生的基础环节，其核心是**扬长避短**。它通过你的社交媒体资料（如朋友圈）告诉别人：**你是谁、你有什么、你的生活是什么样的**。

女生不回复或态度冷淡，有时并非因为聊天技巧问题，而是对方根本没有兴趣开启对话。从客观角度讲，打造良好的展示面能让女生愿意与你聊天，并愿意进一步了解你。

## 展示面的五大组成部分

展示面主要由以下五个部分构成，每一部分都承担着不同的传达功能。

以下是展示面的核心组成部分列表：

1.  **昵称**：初步判断你的品味。
2.  **头像**：告诉别人你是谁，证明你是一个真实的人。
3.  **背景墙（朋友圈封面）**：展示你的价值观。
4.  **签名**：传达你的三观（世界观、人生观、价值观）。
5.  **常规朋友圈**：让别人更深入地了解你的生活。

## 昵称：简单即高级

昵称是他人对你的第一印象。避免使用“追风少年”、“哥是个传说”等非主流或暴露年龄的昵称。

正确的做法是追求简洁。可以是一个英文名，或英文与汉字的组合，例如：`Mr.白`。总之，无论采用何种形式，**简单就是高级**。

## 头像：第一印象的关键

头像至关重要，它决定了他人是否愿意与你接触。我们来做一次换位思考。

*   如果你看到的头像是精心拍摄、有吸引力的，你会有很大几率产生好感。
*   如果头像质量低下、毫无美感，即使对方人品性格很好，你也可能不愿接近。

原因在于第一眼没有产生火花，或你认为彼此不属于同一个世界。因此，头像必须认真对待。

在学习如何拍摄头像前，先了解哪些照片不适合用作头像。

以下是绝对不能用作头像的照片类型列表：

1.  **风景照**
2.  **动物照**
3.  **证件照**
4.  **卡通形象照**
5.  **网图或画质很差的照片**

### 如何拍摄正确的头像？

正确的头像应注重环境和氛围。例如，将人物缩小，视觉中心不在于人物本身，而在于人物所处的**有意境的网红背景**，从而突出整体文艺、好看的感觉。

你也可以借助道具营造氛围：
*   使用**宠物**或**娃娃**营造温馨感。
*   利用**手机、围巾、帽子**等道具进行局部遮挡，不全脸出镜，制造一种**神秘感**和**好奇心**。

## 朋友圈封面：你的黄金广告位

如果说头像是吸引女生点击的“引流器”，那么朋友圈封面就是你的“黄金广告位”。如果使用不当，就是在告诉别人你是一个很low、没有品位的男人。

以下几种类型一定不能用：

以下是绝对不能用作朋友圈封面的类型列表：

1.  **网图**
2.  **大头自拍**（无论帅否，都容易显得自恋）
3.  **旅游照**（特别是比剪刀手的）
4.  **普通的风景照**（除非拍得非常艺术）

### 应该选用什么样的封面？

封面必须展示你的**独特价值**。所谓价值，是别人拍不出来的、**逼格很高**的照片，能体现生活水准，并且是大多数人接触不到的。

例如：
*   潜水的照片
*   坐飞机头等舱的照片
*   冲浪的照片

如果你觉得目前生活缺乏这些元素，以下是一些备选方案：
1.  使用一张有设计感的**纯文字图片**。
2.  去网红下午茶、网红餐厅等**环境时髦**的地方拍摄环境照。
3.  在网络上寻找一张逼格很高的照片（需谨慎，避免滥用网图）。

**核心原则**：拍好的照片一定要修图。坚持“**不修图，不发朋友圈**”的原则，你的展示面就成功了一半。

总之，封面图需要告诉看你朋友圈的人：**我是一个精致、生活丰富、有高价值和品味的男人，我拍的照片比别人好看**。

## 朋友圈签名：一句话体现品位

签名可以是一句意味深长的话，也可以是一句英文。目标是让女生觉得你是一个**有品位的男人**即可。

## 日常朋友圈：你经营的产品

头像是“引流器”，封面是“广告牌”，而日常朋友圈就是你为自己经营的“产品”。头像和封面再好，如果日常朋友圈质量低下，也是徒劳。

首先，要明白哪些朋友圈不能发。

以下是绝对不能发的朋友圈内容类型列表：

1.  **转发的心灵鸡汤或无病呻吟的文章**（显得没有自己的生活，容易被定义为“屌丝”）。
2.  **纯文字**（提供的信息价值少，女生无法了解你）。
3.  **随意拍的朋友玩闹、吃宵夜照片**（如果环境或朋友形象不佳，会拉低你的价值。给朋友拍照也要认真，并选择环境不错的地方）。
4.  **工作广告、推销信息**（如需发布，请使用微信“**分组可见**”功能，将客户与社交圈分开）。
5.  **太过专业的学术性内容**（除非你是摄影师、音乐人等可直接展示职业魅力的行业。像程序员、金融从业者频繁发专业内容，外人看不懂，也不会因此觉得你厉害）。

### 什么样的朋友圈才是正确的？

那么，应该如何打造高质量的朋友圈内容呢？

以下是推荐发布的朋友圈内容类型列表：

1.  **美食**：发的是“美食”，不是兰州拉面、沙县小吃或街边烤串。目的是**打开聊天窗口**，让女生有兴趣问“这是哪里？看起来好好吃”，而不是记录一日三餐。
2.  **丰富的娱乐生活**：展示健身、骑马、旅游、游泳、射箭、桌球、家庭聚会、看画展或摄影展等活动。传达的信息是：**“我是一个愿意上进、愿意给自己投资的男人，和那些整天在家打游戏的人不一样。”**
3.  **宠物或小朋友**：女生对可爱的事物抵抗力低。展示宠物或婴儿是**打开女生聊天窗口最有效**的内容之一，同时也能体现你的爱心。
4.  **预选展示**：即在朋友圈晒你和其他**女生们**的合照（注意：是复数）。注意两点：一是避免单独与一个女生合照，以免被误会是女朋友，如需发布需在配文中澄清关系；二是合照女生的颜值不能太低，否则会拉低你的价值。这能引发女生的思考：“为什么他身边这么多女生？**因为他优秀。**”
5.  **工作与学习**：例如，在凌晨4点发布“大小黄鸭熬夜陪我赶方案”配上工作照。简单展示自己是一个**爱学习、努力工作、有上进心**的男人，在别人睡觉时你仍在奋斗，瞬间与众不同。
6.  **朋友圈配文原则**：配文**宁少勿多**。目的是留白，制造神秘感，让女生有提问的欲望。如果讲得太清楚，反而减少了互动可能性。如果不知道怎么写，可以参考相关社交技巧论坛的配文板块。
7.  **音乐分享**：分享你听过的好听的、有感触的歌。歌曲选择很关键，如果欠缺音乐品味，可以借助音乐APP的“每日推荐”功能。
