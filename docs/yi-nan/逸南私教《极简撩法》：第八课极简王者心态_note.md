# 逸南私教《极简撩法》：第八课极简王者心态_note

在本节课中，我们将要学习如何构建一种强大而平和的“王者心态”。这种心态是高情商的核心，能帮助你在社交和情感关系中保持自信、从容，并做出最有利于自己的选择。我们将通过十个具体的思维和心态要点来展开。

## 概述：为什么要修炼情商？

修炼情商能带来诸多益处。这些益处主要可分为三类。

第一，高情商能让你拥有更好的生活模式，具备管理自身情绪和处理社交关系的能力。一个情商高的人不会长期宅在家中，不会缺少朋友，不会自卑、泪丧或不善于表达。与他人相处时，他知道如何开启有趣愉快的对话，懂得“见人说人话，见鬼说鬼话”，让别人觉得与他相处很开心。他也能轻松处理好自己的社交关系。

第二，高情商可以结识更多有趣的人，结交更多朋友。高情商者有自己的生活和兴趣爱好，懂得享受生活，让自己开心，也能给身边的人带来快乐。

第三，当你的情商和思维都高于常人时，你的能力也会随之提升，从而更好地实现自我需求和创造更多价值。情商高意味着你不会与情商低的人争执，不会被他们误解，也不会盲目跟风。你清楚自己想要什么，知道如何行动以满足需求、创造价值。

以上是情商的三个核心好处。接下来，我们看看如何具体修炼这种情商。

## 一、双向选择思维 🤝

修炼情商的第一步，是建立“双向选择思维”。这个思维的核心是：无论你多优秀，都只有一部分人会喜欢你。你只需要筛选出这部分喜欢你的人，并与之互动、推进关系即可。

因为你要明白，有些人可能自我认知不清，自身价值一般，却只喜欢网红类型或外貌出众的人。如果你们根本不匹配，你在对方面前可能毫无吸引力，而对方在你眼中却光芒四射。关键在于，你通常只能吸引到与你同等优秀或不如你的人。

因此，你需要尽力提升自身价值，如外形建设等，不断增强吸引力。这样，当你出现在他人面前时，才会有更多人喜欢你。你只需从中筛选出喜欢你的人进行互动。对于那些不喜欢你、不想互动的人，可以直接忽略或拉黑。

## 二、社交思维 🗣️

上一节我们介绍了筛选的重要性，本节中我们来看看如何主动开启社交。“社交思维”很简单：在生活中或想认识心仪对象时，要大胆一些。

无论是在搭讪还是约会中，胆大都很重要。首先，用开朗和友善的方式去认识你想认识的人。其次，用有趣和“打情骂俏”的方式对待你感兴趣的异性。

如果你想认识一位异性，就开朗、友善地上前打招呼，表达想法，并引导对方加微信即可。结果无非是对方觉得你不错愿意加，或不愿意加。这本身也是一种筛选，愿意加你的人后续才可能聊得起来。

用有趣和打情骂俏的方式互动很重要。因为如果一直用朋友式的问答、信息交换来互动，你很容易沦为对方的朋友或备胎。你需要植入有趣、略带暧昧的互动方式，这样关系才容易拉近，而不会陷入友谊区。

很多学员问约会时如何升级关系。关键在于，你的重心不应放在“如何升级关系”上，而应放在“如何让约会氛围变得开心”上。当氛围愉快时，牵手或亲吻都会是自然、轻松的事。氛围才是最重要的。

## 三、游戏者心态 🎮

什么是游戏者心态？它意味着当你与异性互动时，应保持一种“一边玩一边撩”的状态。该忙时专心忙，忙完了想找她聊再去聊。特别是在前期，不要把所有的注意力都放在对方身上。

这样你就不会有太强的需求感，聊天也不会小心翼翼。很多兄弟初期需求感过强，正是因为他们太认真，缺乏游戏者心态。一旦觉得某个异性不错，就频繁发信息、过度关心、抱着手机等待回复、分析对方反应，导致自己紧张，聊天不自然，对方也不会喜欢。

因此，想聊好天，首先要调整好自己的心态和状态。当你的心态正常、状态开心，像玩游戏一样去互动，不看重结果，专心聊好当下该说的话时，你就很容易把对方约出来，或让关系变得更亲密。

## 四、情绪平稳 😌

“情绪平稳”指不受他人看法或情绪的影响。例如，当你想做某事（如搭讪）时，身边的人可能会说“你这样不对”、“别人会嘲笑你”、“你会被拒绝”等来阻碍你。你会被这些看法影响，是因为情商不够高，不够清楚自己想要什么，不够坚定。

其实想法可以很简单：如果我真的想认识她，那就轻松自然地去认识，表达我想表达的，引导收号即可。给我号，我就有机会；不给我，也没关系，至少我尝试了。我不在乎结果。给了号，我才有机会约你；不给，说明你对我没兴趣，那也不必强求。

记住，无论在生活中、工作上还是感情中，都要清晰自己想要什么，按照自己的方式去生活和行动，不要受他人看法和情绪影响。

此外，不要被得失所影响。当你不把结果看得太重时，做任何事都会非常轻松。当你不想着一定要和对方发展什么，目的性不强时，反而可以聊得很轻松、很嗨，对方也会更喜欢你。前期不要把得失看得太重，除非你们真的在一起了，那时再去珍惜也不迟。

最后，不要嫉妒不属于你的事物，不要轻易发脾气。不要因为别人比你好就去嫉妒、说坏话，这只会破坏你的社交关系。要懂得控制自我情绪，懂得带给别人开心有趣的体验，也懂得释放自己的情绪。

你可以在独处时做让自己开心的事，或去没人的地方呐喊宣泄。但不要将易怒的状态带入生活、工作和社交中。要学会用轻松有趣的状态面对他人。脾气发多了，人在情绪失控时可能做出打人、骂人等让自己后悔的事。如果确实是对方的错惹怒了你，你可以爆发；但如果是自己情绪控制能力差，就需要调整。

## 五、正确看待失败 📉

正确看待失败非常重要。很多兄弟对失败有误区，认为“失败是因为我太差劲”、“我入不了对方的眼”、“我不行”，把拒绝当成对自我的否定和无能。这是非常错误的，这只会让你越来越自卑，魅力与能力也无法提升。

你要正确看待失败。记住，每一次失败都是在突破自己，让自己变得更好。不要产生自卑和挫败心理，要把失败当成像吃饭一样正常的事。因为只有失败才能让你进步，让你不断反省和提升不足。

当你失败时，不要多想，只需思考下次如何做得更好。例如，聊天失败了，就总结原因：是我说话太浮夸，还是吸引力太差？约会失败了，就总结经验：为什么对方不开心？总结原因，持续改进即可。每个成功的人都经历过很多失败。

如果你想从小白快速成为情商高、很会撩的人，必经的阶段就是很多失败。你需要接触很多人，尝试很多突破，运用方法去练习、去做，才能得到好的结果和回馈。这个过程也是你不断增长经验和经历的过程。

就像第一次搭讪，你知道这是好事，但会紧张、害怕失败，这很正常。因为做任何有利的事时，你都会害怕失败。所以要正确看待：失败不重要，重要的是我去做我认为对的事，结果已不重要。

## 六、拒绝攀比，多经历 🚫📈

不要和别人攀比。很多学员会问：“为什么别人把妹比我厉害？为什么他约了好几个？” 其实不必攀比，因为别人可能在背后付出了很多努力，经历了很多挫折和失败。你技术差就多点耐心，多去提升和练习。

和昨天的自己比，和去年的自己比，看自己是否变得更优秀、更好。如果是，那你就在成长。不必和比你优秀的人攀比，攀比会给你带来压力，让你觉得自己不如人，从而不开心。所以，不要攀比，反省自身不足，让自己越来越优秀，这样你就能悄悄超越很多人。

多去经历，多去玩。很多人很宅，平时除了上班就是在家打游戏、点外卖，一年出去玩不了几次。这种情况下，他的社交能力和情商会很差。

我让你们多出去玩，是因为在玩的过程中，你会接触各种各样的人。当你和他们聊天、遇到喜欢的异性去互动、甚至去酒吧练习时，你就在积累经验。从能搞定三分异性，到搞定四分、五分……从搞定一个，到搞定十个，你的经验和经历就慢慢积累了。

这时，你心中的那种自信就已根深蒂固，无人可以动摇。因为你有这样的经历和经验，所以能更好地面对一切。我希望你们能真正去练习、去经历、去玩，不断累积成功的经验。成功的经验越多，自信就越强。

## 七、前期感情投入原则 ⚖️

前期对感情不要太认真，不要疯狂投入和跪舔。要学会将心比心，投资平衡。前期可以付出比对方多一点，但不能太多。

怎么理解呢？当你刚加了一个异性时，只需正常互动即可，不用想她会不会喜欢你、什么时候成为女朋友、什么时候结婚等。想得太多无益。把过程聊好，有感觉就约见面，约会时做一些亲密行为让关系升温。发生关系后，再去配合态度，引导确定关系，就水到渠成了。

前期不要对感情太认真，不要疯狂投入和跪舔。你可以主动聊天、主动暧昧、主动约她，但要看她的反应。如果她反应好，你可以更主动；如果她持续反应不好，你可以把她放一边，或去提升自己，或表达态度。

前期的付出，比如请对方看电影、喝东西，都没关系。但不要想着买礼物、送东西、讨好她来让她喜欢你。这是让你越陷越深的错误做法。前期只需在互动中花该花的钱，不该花的（如送礼物、发红包）不要花。这些事应在关系已经很亲密，如恋爱或情侣阶段再做。

如何实现投资平衡？很简单。例如，你和对方出去看电影，你请她看电影，可以让她请你喝杯甜品。或者让她帮你拿电影票，让她牵你的手。这些都是对方对你的投资。你还可以让她为你付出感情、时间、肉体上的投资。钱并不能代表什么，因为钱容易赚，但感情、用心付出和在意才是最重要的。你要学会用第三方的东西引导对方投资。

## 八、建立个人标准 📏

要用自己的标准去对人对事。例如，我的标准是：只对对我有兴趣的异性互动，不会跪舔讨好女人。我只会按照自己的标准合理地对女人好。

比如，在她表现好时，我可以奖励她：“宝贝请你喝个甜品”、“请你看电影”、“奖励你一个棒棒糖”。我也不会欺骗和伤害亲近的人。这是你自我的原则和做人做事的标准。

你要设定好自己的标准：我只和对我有兴趣的人互动。前期我可以主动聊天、提供话题，但当我感觉到你敷衍、没兴趣时，我会立刻换人，或先把你当备胎。你当一个异性的备胎，你就是备胎；你当一百个异性的备胎，那么她们就是你的备胎。

不要跪舔讨好，该怎么互动就怎么互动，该怎么调情就怎么调情，该怎么约就怎么约，用男女朋友的方式去互动。按照自己的标准对对方好，按照正确的方法互动。能够被你约出来或愿意和你互动的人，才是真正喜欢你、能和你深入发展的。不用想太多。

当对方表现好时，你也可以进行奖励，比如请她喝东西、看电影、送小礼物。因为人都喜欢对自己好的人，但这种好一定是基于你自己的标准。

## 九、学会提供价值 💎

做任何事情都要学会提供价值。你对别人好，别人才会对你好；你付出，才会有收获。

这一点很简单。你对别人好，别人会记住，也会对你好，在你需要时帮助你。你付出努力去学习、去实践，你也会有巨大的进步和成长。

## 十、空杯心态 🫗

最后一点是“空杯心态”。这意味着你要养成虚心学习、不断提升的好习惯。

为什么要这样？因为社会在发展，科技在进步。不要觉得自己懂得多了就很拽、目中无人。任何时候都会有比你更优秀、你驾驭不住的人出现。如果你想预防这种情况，就要提前做好准备：不断学习、提升自己、丰富自己。

觉得自己穿得不够潮，就每月买一套更好看的衣服；觉得自己哪方面不足，就去提升、练习。这样你就会越来越好。当你感觉自己已经满足时，其实就没有成长空间了。

我希望你们都抱着空杯心态，不断学习、提升、实践。就像撩妹一样，从撩自己觉得一般的，到撩好看的、特别喜欢的，这样你就已经很棒了。虚心学习、不断提升，很重要。

## 总结 🎯

本节课中，我们一起学习了构建“王者心态”的十个核心思维与心态要点：双向选择思维、社交思维、游戏者心态、情绪平稳、正确看待失败、拒绝攀比多经历、前期感情投入原则、建立个人标准、学会提供价值以及空杯心态。

学习了这些心态和思维后，你要记在心上，刻意去练习和养成。切记不要再用以前错误的思维和方法去与异性互动，因为那样是无效的，也会让你自己尴尬。学了就要去练习，去刻意养成这样的思维和习惯。

当然，成长是痛苦的。但当你的经历和思维沉淀下来时，你就真正蜕变了。当你的思维和经历沉淀了，当你很有经验、很有感悟、情商很高、搞定过很多异性时，你就真的沉淀并蜕变了。你也不会再为感情之事烦恼。
