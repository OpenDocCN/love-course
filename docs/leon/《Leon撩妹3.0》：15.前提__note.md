## 为什么是“前提”而非“意图表达”？

上一节我们介绍了课程主题，本节中我们来看看为何选用“前提”这个概念。主要有两个核心原因。

第一，“意图表达”一词常隐含着低姿态。它通常表现为“我可不可以加你微信”、“可不可以跟你一起吃个饭”、“可不可以做我女朋友”，这给人一种你在追逐对方的感觉。而“前提”则不同，它意味着你拥有自己的标准，并在对对方进行**符合你标准**的评估后，才决定是否升级关系。例如：
*   **前提表达**：`我觉得你还蛮不错的，我们可以加个微信聊一聊。`
*   **前提表达**：`跟你相处起来，我觉得你还挺符合我一些标准的，我觉得我们可以开始交往试试看。`

两者的最大差别在于，前提的核心是**“我有标准，并且我对你进行复格”**，而非单纯地“我想要”。

第二，人际沟通的前提具有多样性。它包括男对女的前提、社交（朋友）的前提、生意上的前提等。而“意图表达”通常只局限于男女关系这一层。在实际与女性互动中，这三种前提往往是混合（`mix`）出现的。拥有“前提”这个概念，能让你更清晰地意识到沟通是由多种前提构成的，而非单一元素。这有助于你在朋友前提过度时，及时将自己拉回男对女的前提上。
