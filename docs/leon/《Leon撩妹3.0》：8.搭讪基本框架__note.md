## 开场白：建立第一印象

上一节我们介绍了课程概述，本节中我们来看看搭讪的第一步：开场白。开场白是大多数人在搭讪过程中遇到的第一个卡点。问题可能源于你没有轻拍女生的肩膀、讲话的声音节奏不对、眼神不够坚定等。这些因素可能导致女生在第一秒就直接拒绝你。

核心问题可以总结为三个字：**坚定感不够**。坚定感包含你的眼神、声音以及适当的身体接触（如轻拍肩膀）。这个动作非常重要，因为它能让女生停下脚步。如果你只是简单地说“hello”，女生可能直接走开。所以，对于初学者，请务必记住并运用“拍肩”这个动作。

以下是关于开场白的正确做法：
*   正常情况是，在大家同向前行的过程中，你轻拍她的肩膀，让她转身。
*   然后简单地说一句“hey”即可。
*   避免绕到女生面前去开场，那样显得过于刻意。

## 意图表达：直接与间接

完成了开场，接下来我们需要表达意图。意图表达无需复杂化，对于初学者，直接说“你好，我觉得你很漂亮，想认识你”即可。直接表达意图是初阶的标准做法。

然而，随着你从初阶进阶到中高阶，意图表达会逐渐变得微弱和间接。这是因为人与人之间的关系存在博弈。如果你过早地让女生完全了解你的意图，她可能会将自己视为“奖品”，使你陷入被动追逐的姿态。

两者的区别可以通过以下例子感受：
*   **直接意图**：`“你好，我觉得你很漂亮。”`（隐含：我想得到你）
*   **间接意图**：`“我觉得你可能会是我喜欢的类型，所以过来跟你聊聊天。”`（隐含：你或许符合我的标准）

关于更微妙的意图表达技巧，我们将在中高阶课程中详细讲解。

## 问题解决：应对常见障碍

在表达意图后，你通常会遇到四类问题。解决这些问题的总体思维流程分为两块：**动力大于阻力**。在动力方面，你需要保持坚持、淡定、自信，甚至可以植入一些幽默感，让女生感受到你的价值，愿意为你多停留一会儿。

以下是四类常见问题及其具体的解决思路：

1.  **女生表示很急有事**
    *   **解决方法**：给出一个“假性时间限制”。例如：“给我10秒钟的时间。” 在展示了足够价值后，这个请求通常会被接受。女生通常不差这一分钟。

2.  **女生把你当作推销员**
    *   **解决方法**：直接澄清意图。例如：“你知道我是来干嘛的吗？我是来搭讪你的，不卖东西。” 如果这招不管用，可能需要复盘自己的外在形象（衣着、坦荡度）是否让女生产生了固有归类。

3.  **女生表示有男朋友**
    *   **解决方法**：解决她的道德压力。可以问：“你会和你男朋友结婚吗？” 如果会，则礼貌离开；或者说：“我只是真诚地想交个朋友。” 如果她有闺蜜在场，也需要安抚闺蜜，例如说：“我要加她微信，你别告诉她男朋友好不好？以后我们可以一起玩，我也给你买吃的。” 以此化解闺蜜带来的压力。

4.  **价值感不够**（女生看不上你）
    *   **解决方法**：价值感建设主要放在下一个“信息互换”环节去体现和解决。

如果你能顺利通过“问题解决”这一关，你已经超越了大多数人。

## 信息互换：展示价值与建立连接

信息互换环节至关重要，它极大地传递了“你是谁”、“你的价值是什么”以及“你的故事”。这个环节首先要做到基本对话流畅。

以下是六个适合与陌生人聊天的基本话题，你可以由浅入深地使用：
*   年龄
*   职业
*   居住地
*   家乡
*   当下正在做什么
*   是否有被搭讪的经历

作为一个新手，你需要先熟练、灵活地应用这六个话题，然后才能进行关联、创造甚至“扭曲”。

**什么是“扭曲”？** 核心概念是制造**矛盾**。矛盾是引发好奇心的原动力。例如：
*   **年龄冷读**：对方28岁，你问：“你多大？有20吗？”（制造与事实的不一致）
*   **职业扭曲**：对方是老师，你问：“你是不是警察？”（制造身份认知的矛盾）
*   **籍贯猜测**：对方是上海人，你问：“你是台湾人吗？”（制造地域认知的差异）

通过制造这种不一致性，可以激发女生的好奇心，让聊天变得有趣。这是一个进阶技巧，我们后续课程会深入讲解。

除了基本话题，还有一个重要技巧：**评估**。评估有两个作用：第一，筛选掉不符合你标准的女生；第二，在提问过程中体现你的价值感，掌握对话框架。

以下是三个可以频繁使用的评估问题示例：
1.  `“你身边的朋友一般都怎么形容你？”`
2.  `“你觉得你自己有什么样的优点或优势？”`
3.  `“你在乎另一半的颜值多一些，还是性格多一些？”`

当你提出这些有深度的问题时，对方会感觉到你是一个有标准的人，并且框架（主导权）掌握在你手中。但请注意，如果你的价值感不够，过于生硬地使用评估可能会引起反感。需要根据现场互动灵活运用。

在信息互换环节，掌握了基本话题和评估技巧后，你可以进一步融入幽默感。但前期请先保证对话流程的顺畅。

## 收尾与升级：完成搭讪

最后一步是收尾。当你判断聊天已达到一个高峰，且难以继续提升时，应该在热度下降前进行收尾。

收尾主要是为了完成 **`Number Close`** （获取联系方式）或直接邀约。如果你判断女生窗口很大（兴趣指标充足），可以直接约杯咖啡。

让我们回顾整个流程中常见的失败点：
*   **开场**：死于不够坚定，未使用拍肩等动作。
*   **意图表达**：相对简单，直接表达即可。
*   **问题解决**：很多人死在这里，无法应对销售误解、有男友、价值感不足等情况。
*   **信息互换**：对话卡住，需要熟练掌握基本话题和评估技巧。
*   **收尾升级**：聊得很好却不敢要联系方式或邀约。

掌握这五个流程，你的搭讪能力就能实现从0到60分的跨越。
