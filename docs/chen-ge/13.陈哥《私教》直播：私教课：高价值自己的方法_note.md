# 13. 陈哥《私教》直播：私教课：高价值自己的方法 🎙️

## 概述
在本节课中，我们将学习如何通过聊天提升自己的高价值，以及如何有效地传递自己的价值。

## 一、高价值的概念
**公式**：高价值 = 情绪价值 + 言语价值 + 性格价值

### 1. 情绪价值
情绪价值是指你在聊天中展现出的情绪状态，包括开心、不开心、积极、消极等。

### 2. 言语价值
言语价值是指你的言语风格、性格特点等，包括你的说话方式、幽默感、知识面等。

### 3. 性格价值
性格价值是指你的性格特点，包括你的做事风格、价值观、人生观等。

## 二、如何提升高价值
### 1. 有目的性地说话
在聊天过程中，你要明确自己的目的，并围绕这个目的进行对话。

### 2. 传递情绪价值
在聊天中，要尽量传递积极的情绪，避免传递负能量。

### 3. 保持真实
在聊天中，要尽量保持真实，不要刻意去扮演一个角色。

### 4. 传递言语价值
在聊天中，要展现自己的言语价值，包括你的说话方式、幽默感、知识面等。

### 5. 传递性格价值
在聊天中，要展现自己的性格特点，包括你的做事风格、价值观、人生观等。

## 三、案例解析
以下是一个案例，展示了如何通过聊天提升自己的高价值：

**案例**：陈哥与妹子的聊天记录

1. 陈哥：你小时候有没有拿着鞭炮去炸牛粪的经历？
2. 妹子：有啊，我小时候也经常玩这个。
3. 陈哥：那时候我们都是拿打火机一点，然后直接就炸了，炸了我一脸的牛粪。
4. 妹子：哈哈，我也是，那时候好开心啊。

**分析**：
1. 陈哥通过提问，引导妹子回忆起自己的经历，从而拉近彼此的距离。
2. 陈哥在聊天中，展现了自己的幽默感和真实性格，提升了自身的高价值。
