# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。这个标签就是**变量名**，而盒子里装的东西就是**值**。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```

例如：
```python
name = “陈哥”
age = 30
```
在这段代码中，我们创建了两个变量：`name` 存储了文本“陈哥”，`age` 存储了数字30。

## 2. 基本数据类型 🔢

变量可以存储不同类型的数据。Python有几个内置的基本数据类型，最常见的有以下三种。

以下是三种核心的数据类型及其说明：

*   **整数**：没有小数部分的数字，例如 `-5`, `0`, `100`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由文本字符组成的数据，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `“你好”`, `‘Python’`。

## 3. 查看与理解数据类型 🔍

我们如何知道一个变量里存的是什么类型的数据呢？Python提供了一个非常方便的函数 `type()`。

其使用**公式**为：
```python
type(变量名或值)
```

例如，运行以下代码：
```python
print(type(“陈哥”)) # 输出：<class ‘str’>
print(type(30)) # 输出：<class ‘int’>
print(type(3.14)) # 输出：<class ‘float’>
```
`str` 代表字符串，`int` 代表整数，`float` 代表浮点数。了解数据类型很重要，因为不同类型的数据能进行的操作是不同的。

## 4. 变量的命名规则 🏷️

不是任何单词都能作为变量名。为了代码清晰且能被Python正确识别，为变量命名需要遵循一些规则。

以下是命名变量时必须遵守的核心规则列表：

*   变量名只能包含**字母**、**数字**和**下划线**。
*   变量名不能以数字开头。
*   变量名是**区分大小写**的（例如 `age` 和 `Age` 是两个不同的变量）。
*   不能使用Python的**关键字**（如 `if`, `for`, `while`）作为变量名。

良好的命名习惯能让你的代码更易读，例如使用 `user_name` 而不是 `a`。

## 5. 一个简单的实践示例 💻

现在，让我们把以上知识结合起来，写一个简单的程序。

```python
# 定义变量
user = “陈哥”
birth_year = 1994
current_year = 2024

# 计算年龄（整数运算）
age = current_year - birth_year

# 打印结果
print(“用户：” + user)
print(“出生年份：”, birth_year)
print(“当前年龄：”, age)
print(“年龄数据类型：”, type(age))
```
这段代码演示了如何定义变量、进行数学运算，以及打印混合类型的结果。
