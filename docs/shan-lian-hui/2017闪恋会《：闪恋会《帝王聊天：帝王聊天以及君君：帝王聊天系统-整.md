# 课程2017-03：帝王聊天系统 - 整体聊天思路和流程（中）💬

在本节课中，我们将要学习帝王聊天系统的核心部分：聊天的三个层次。我们将详细拆解如何提供话题、延续话题以及如何恰当地结束话题，并学习多种实用的聊天技巧，确保你的对话能够流畅、有趣地进行下去。

## 聊天的三个层次

上一节我们介绍了聊天的基本框架，本节中我们来看看构成这个框架的三个具体层次。聊天的过程可以清晰地划分为：**提供话题**、**延续话题**和**结束话题**。

### 第一层：提供话题 🎯

提供话题是对话的起点。我们主要从“你我他模型”出发来开启话题。这个模型指的是：聊“我”（自己）、聊“你”（对方）以及聊“他”（第三方事物）。

以下是提供话题的几个方向：
*   **聊“我”**：介绍自己的名字、状态、兴趣爱好、星座等。
*   **聊“你”**：询问对方的名字、星座、兴趣爱好、童年经历等。
*   **聊“他”**：讨论美食、电影、娱乐八卦等第三方话题。

例如，开场时说“我是张铁”，就是在从“我”的角度提供话题；问对方“你叫什么名字？”，则是从“你”的角度提供话题。

### 第二层：延续话题 🔄

当对方回应你提供的话题后，对话就进入了延续话题的阶段。这是保证聊天不冷场的关键。延续话题主要有四种方式。

#### 方式一：扣关键词聊天

这是最核心的技巧。即从对方回复的语句中，抓住一个或多个关键词进行扩展和回应。

**核心公式**：`回应 = 抓住对方语句中的关键词 + 围绕该关键词展开陈述或提问`

例如，对方回复：“孙杨😏”。这里可以扣的关键词有：
1.  **“😏”表情**：可以回应“笑得这么坏呀”。
2.  **“孙杨”这个名字**：可以回应“挺好记的，和我以前一个同事同名”。

再如，对方说：“冰糖雪梨现在全糊了”。可以扣的关键词有：
1.  **“冰糖雪梨”**：可以回应“你还会做冰糖雪梨呀？”。
2.  **“糊了”**：可以回应“说不定糊了另有一番滋味呢？”。

掌握了这个方法，你就不会担心没有话题。初期练习时，可以尝试找出对方话中的所有关键词；熟练后，选择最有利于推进关系或最有趣的那个关键词来延续。

#### 方式二：延续自己的话题

如果对方的回复中没有明显的关键词可扣（例如只回复“嗯”、“对的”），或者你觉得自己的话还没说完，可以主动延续自己之前提供的话题。

例如，你问：“煮的什么好吃的？”，对方回：“煮的东西忘了”。你可以延续自己的话题：“看来是你不饿，所以忘了”。这相当于扣了自己提问中“好吃的”这个关键词的延伸。

#### 方式三：使用“你呢？”进行平衡

聊天需要注意平衡，不能只聊对方或只聊自己。在分享完自己的情况后，使用“你呢？”来将话题抛回给对方，是一种非常自然有效的延续方式。

例如：
*   对方问：“你做什么工作？”，你回答后可以问：“你呢？”
*   对方问：“你多高？”，你回答后可以问：“你呢？”

这保证了对话在双方之间自然流转。

#### 方式四：主动分享自己

当对方讲述完自己的某方面情况后，你也可以主动分享自己对应的方面，实现话题的平行延伸。

例如，对方说：“我是做护士的”。你可以分享自己：“我做传媒的，感觉你们护士挺辛苦的，经常要上夜班吧？”这样既回应了对方，又提供了关于自己的新信息点（职业、对护士的看法），让话题得以延续。

**总结一下**：延续话题的四种方式是：**1. 扣对方关键词；2. 延续自己话题；3. 使用“你呢？”；4. 主动分享自己**。在实际聊天中，这四种方式常常混合使用。当发现一个话题已经充分展开后，就可以自然地过渡到下一个话题，重新从“你我他模型”中提供一个新话题，开启新一轮的循环。

### 第三层：结束话题 🚪

适时结束话题和开启话题同样重要。好的结束能为下次聊天留下期待，避免因聊到无趣而损耗对方的兴趣。

#### 方式一：正常结束

这是最普遍的方式，用于话题自然聊完或时间已晚。
例如：“不早了，我先睡啦，晚安。”、“有点事，先忙了，有空再聊。”

#### 方式二：高价值离场

当对方反应冷淡或你想主动切断对话时，可以使用此方法。通过展示一个积极、有价值的自身状态来结束对话，留下良好印象。
例如：“我朋友约我去尝尝他的厨艺，先不聊啦。”、“健身房的小伙伴催我出发了，回头说。”

#### 方式三：自然切断

如果对话已经非常平淡，也可以选择不再回复，让对话自然冷却。这通常用于对方兴趣度明显很低的情况。

**关键原则**：最好在对方兴趣度尚可（处于高峰或平台期）时主动结束，而不是等到聊到无话可说、兴趣下降时才仓促结束。这会让对方觉得意犹未尽，期待下次交流。
