## 概述：什么是速推？

速推是指在第一次与女生约会时，快速确定关系或发生关系。这个流程讲究速度，通常在一到两小时内完成。它与常规互动方式不同，需要特定的技巧和方法。

## 第一核心：心态准备

上一节我们介绍了速推的基本概念，本节中我们来看看成功速推的第一个核心要素：心态。

你必须做好彻底失去这个女生的心理准备。当你决定今晚要“搞定”一个女生时，你的目标必须坚定，不能因害怕失去而犹豫。这意味着你需要承担所有责任，心态上不能动摇。

**核心心态公式：目标坚定 + 承担风险 = 速推心态基础**

## 第二核心：筛选目标

在做好心态准备后，下一步是筛选合适的女生。并非所有女生都适合速推，关键在于识别“窗口”较大的对象。

以下是筛选女生的关键点：

*   **判断互动氛围**：在聊天中多使用暧昧词语，观察对方是否接受或配合。

## 第三核心：升高关系与流程操作

筛选出目标后，整个速推流程的核心在于及时、不断地升高关系。这需要你主动创造、打开并升级“窗口”。

整个流程遵循“创造窗口 → 打开窗口 → 升高关系”的循环。关键在于带领，而非征求同意。

在速推过程中，你可能会遇到女生的各种拒绝。以下是应对策略：

*   **进入私密空间后**：女生可能仍会矜持。需继续耐心进挪，遵循“尝试-被拒-暂停-再尝试”的循环。

## 速推的利弊与总结

本节课中我们一起学习了速推的核心秘密，现在我们来总结其利弊。

**速推的优势**：能在极短时间内达成目的，注重当下情绪的享受。
**速推的弊端**：容易让女生觉得过程太快，可能导致事后反悔，关系无法长期发展。因此，若追求长期关系，不建议使用速推。

**核心总结**：
1.  **心态是根基**：敢于承担彻底失去她的风险。
2.  **筛选是关键**：不是所有女生都适合速推，要识别窗口。
3.  **流程是路径**：遵循“吸引-升高-转场-私密”的流程，主动带领，循序渐进。
4.  **坚持是秘诀**：面对拒绝，要脸皮厚，坚持不懈地尝试。

速推的真谛在于心态强大、目标明确、不断尝试。它就像销售，通过不断的坚持和说服，最终达成交易。
