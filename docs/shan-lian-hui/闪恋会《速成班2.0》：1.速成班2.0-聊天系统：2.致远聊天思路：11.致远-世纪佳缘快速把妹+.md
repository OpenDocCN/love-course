4.  **电话试探**：即使你再失误了，去赴约了。这时候再看这个妹子。你可以给她打电话，大概聊一聊，比如微信上聊过什么名字、聊过哪些点，看她知不知道。比如你微信上她跟你讲她是做服装的，一般很多女生是做服装的。那你说“我记得微信上你跟我讲过，你是做文员的，是吧？”你看你这样话就来个试探。她如果讲“对啊，是的”，那明显聊的时候不是一个人。懂这个意思吧？就是诈她。见面聊也行，或者打电话聊。就是你提其中一个点，故意改成错误的。她跟你讲过她是卖衣服的，你说“你是在跟我讲，你是公司做服装的，是吧？”你看她如果跟你聊的，她肯定知道做什么的不可能变。她跟你说，如果她真的跟你聊，她跟你讲“不是啊，我跟你讲我是做服装的。”那如果说她讲“是的呀，我跟你讲我是做文员的”，说明聊的肯定不是一个人。
