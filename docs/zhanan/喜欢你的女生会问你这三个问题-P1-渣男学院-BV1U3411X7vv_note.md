## 1. “你在干嘛呢？” 📱

她这样问的潜台词是：“看看我是不是你喜欢的类型。”这个问题看似好回答，其实有点难度。回答得好，你在她心目中的好感度会飙升；回答不好，可能会一落千丈。

这句话，真的就是隐晦的表白。潜台词是：“其他人怎么看我我根本不在乎，我只关心我在乎的人怎么看我。”大多数女生都一样，只有对这个人产生了感情，才会在意他的看法，想知道自己在对方心里的分量。

这句话问出口，风险其实很大，是把自己放在了一个被动的位置让你评价。因此，女生不会无缘无故问你这个问题。这说明她对你的好感度已经很高了，她想确认你的心意，进而获得确认自己心意的勇气。

**核心概念**：她的心情一定是**纠结加期待**的。如果你也喜欢她，那就勇敢、真诚地给予正面回应。例如：“我觉得你很好啊，性格可爱，和你在一起总是很开心。”
