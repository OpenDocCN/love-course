## 核心观点阐述

女人对你不主动就是还没接受你。这个信号是判断关系进展的关键指标。

上一节我们介绍了核心观点，本节中我们来看看具体的行为分析。

当女人不主动找你，而你却忍不住想要找她的时候，不妨先问问自己：她真的那么期待你去找她吗？这个自我提问能帮助你回归理性判断。

如果她喜欢你的话，她早就开口了。因为喜欢一个人是藏不住的，这是一个基本的人性逻辑。女人并没有想象中那么复杂，她们的好感通常会通过主动行为表现出来。

## 行为模式解读

对女人来说，不热情就是拒绝。不回应的就是答案。忽冷忽热就意味着你可有可无。这些行为模式可以总结为一条简单的公式：

**兴趣水平 ≈ 主动频率与热情程度**

以下是几种关键行为及其含义：

*   **不主动联系**：意味着你不在她的优先考虑范围内。
*   **回应冷淡或拖延**：意味着缺乏继续对话的动力。
*   **态度忽冷忽热**：意味着她只在自身需求出现时（如感到无聊）才会想到你。

也许她并没有那么想你，她只是在无聊的时候找了你一下。而你却想了很多。这种情况的代码逻辑可以比喻为：

```python
if woman.is_bored(): # 当她感到无聊时
    action = contact(you) # 行为是联系你
else:
    action = None # 其他时间无行动
# 这不能直接推导出 woman.is_interested(you) == True
```
