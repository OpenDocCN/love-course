上一段我们提到了课程概述，本节中我们来看看第一个关键特征：性格有趣。

## 2. 懂得“厚脸皮” 🤝

了解了性格的重要性后，我们进入第二个特征：懂得适时“厚脸皮”。

这种方式能有效降低对方的防备心理。
