# 渣男学院 P1：为什么遇到女人会怯懦？🤔

在本节课中，我们将要学习男性在面对心仪女性时产生怯懦心理的根本原因。我们将剖析这种心理的来源，并揭示其背后隐藏的真相，帮助你理解并克服这种不必要的恐惧。

## 怯懦的真正来源

上一节我们介绍了课程的主题，本节中我们来看看怯懦感的真正来源。

男人的怯懦从来都不是来源于女人，而是来自于那些他们看不见的男人，那些他们想象中的阿尔法男。这些幻想中的敌人，让我们始终处于一种无形的压力之中。

## β男的退缩心理

理解了压力的来源后，我们来看看这种压力如何具体影响行为。

大多数β男在喜欢上一个女人的时候，都会产生一种想要退缩的念头。他甚至会找各种各样的借口，来合理化自己的怯懦。

以下是β男常见的退缩借口清单：

*   我不会撩妹。
*   我开不起好车。
*   我没有1米85。
*   我不帅。
*   我没有月入10万。
*   我不自信。

这些想法的出现，其实并不来源于你对自身的厌恶，而是来自于你对喜欢的女人应该适配男人的幻想。你觉得像她这样的女人，身边一定围绕着高价值的男人。

## 高价值男性的稀缺性

那么，现实情况是否真的如你所想呢？本节我们来分析情感市场的真实状况。

但事实却往往与你想象的有所冲入。在情感市场里，高价值的男性从来都是稀缺资源。**1000个男人里，可能只有不到5%的人，拥有天生的阿尔法特质和领袖地位。**

真正让你退缩的，并不是这个女人和你自身的条件，而是你想象中的这些高价值竞争对手。

## 想象中的竞争对手

既然高价值男性如此稀缺，我们喜欢的女性身边是否真的存在这些强大的对手呢？

但是这些竞争对手真的存在吗？你喜欢的女人，其实未必真如你所想的那样，可以适配上这些高价值的阿尔法的。她身边围绕着更多的是和你一样的贝塔男、怂货，或者是伪装后的诸葛。这些人就真的比你强吗？

## 择偶标准的真相

许多女性会公开自己的择偶标准，这常常加剧了男性的焦虑。让我们来解读这一现象背后的逻辑。

事实上你可以看到很多女生，都在社交平台标榜自己的择偶标准是“185的大帅哥，月入百万的富二代”。这本质上其实是一种废物测试，用择偶标准来侧面提升自己的价值，并筛选掉一些没有勇气的贝塔男。

但你思考一下，如果她的身边真的围绕着这些高价值的阿尔法，她还需要这样标榜自己吗？

## 怯懦心理的成因

分析了外部现象后，我们最后来总结导致男性普遍怯懦的两个核心原因。

男人普遍会产生这种怯懦，背后的原因有两个。

以下是两个核心原因：

1.  **媒体误导**：首先是因为当代的主流媒体和影视作品，都在习惯性地模糊爱情的本质，导致没有自信的男人普遍觉得女人嘴里的择偶标准，才是男人获取自信的根源。
2.  **能力缺失**：其次是因为你没有技巧和强大的阿尔法特质作为支撑。

而这些都是可以通过后天学习和修炼得到的。
