# 课程名称：识别“喜欢你的信号” - P1

## 概述
在本节课中，我们将学习如何通过观察一个人的行为变化，来判断他是否对你有好感。课程内容基于常见的社交信号总结，帮助你理解这些行为背后的心理。

## 课程内容

上一节我们概述了本课程的目标，本节中我们来详细看看具体的识别信号。

以下是判断一个人是否可能喜欢你的五个关键行为信号：

1.  **他开始在乎自己的仪容仪表**。例如，一个平时不修边幅的人，在可能见到你之前会特意整理外表，公式可表示为：`见面前的准备程度` 显著高于 `日常准备程度`。
2.  **你的笑话对他特别有吸引力**。他会觉得你讲的笑话格外好笑，即使笑话本身可能很普通。
3.  **他倾向于在你面前展示优点**。他会抓住机会表现自己擅长的一面，并尽可能避免在你面前出丑。
4.  **他可能会故意保持距离**。有时，越喜欢一个人，反而会因紧张或害羞而故意表现得疏远，这是一种矛盾的心理反应。
5.  **他会对你的社交产生情绪波动**。看到你与其他异性愉快交谈时，他可能会感到失落，并需要自我安慰。

## 总结
本节课中我们一起学习了识别好感的五个常见行为信号。记住，这些信号需要结合具体情境和多个表现综合判断，单一行为并不足以定论。理解这些信号有助于你更好地解读他人的心意。
