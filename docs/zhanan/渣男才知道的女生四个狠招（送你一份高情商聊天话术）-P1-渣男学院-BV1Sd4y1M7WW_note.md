# 渣男学院 第1课：识别四种服从性测试 🧠💔

在本节课中，我们将要学习女性在亲密关系中可能无意识或有意识使用的四种“服从性测试”。理解这些测试的本质，有助于你识别关系中的潜在压力，并建立更健康、平等的互动模式。

上一节我们介绍了课程主题，本节中我们来看看第一种测试：经济测试。

## 1. 经济测试 💰

这种测试通过物质要求来检验对方的服从程度和投入意愿。

*   每年苹果发布会后，不少男性需要为女友购买新手机，而自己可能仍在使用旧型号。
*   部分女性会要求频繁更换手机，并可能以“你不买，我就找别人买”等言论施加压力。
*   其核心目的是**训练对方心甘情愿地为她花钱**。

了解了物质层面的测试后，我们接下来看看在社交层面的控制手段。

## 2. 社交剥离测试 🚫👥

这种测试旨在孤立你的社交圈，以增强你对她的依赖和服从。

以下是常见的表现形式：
*   干涉你与朋友的交往，对你的社交关系指手画脚。
*   要求你删除某些异性甚至同性好友，并以“恋爱需要坦诚”作为理由。
*   其逻辑在于：你的朋友越少，她的危机感越低，而你的反抗意识和资本也越弱，从而**服从性越强**。

当社交圈被压缩后，个人的底线可能更容易受到挑战。下面我们探讨第三种测试。

## 3. 失格测试 ⚖️

这种测试通过不断挑战和突破你的底线，来削弱你的自我人格与尊严。

其典型模式包含两个步骤：
1.  **试探底线**：先摸清你在关系中的容忍限度。
2.  **突破底线**：通过过激行为一次次超越你的限度，例如：
    *   一吵架就提分手。
    *   拉黑所有联系方式，进行冷暴力。
    *   通过寻找暧昧对象等方式刺激你。

面对这种行为，通常只有两种反应：**绝不惯着（可能导致分手）** 或 **一再忍让（导致自尊受损）**。

最后，我们将分析一种更隐蔽的、体现在日常相处中的测试。

## 4. 价位测试 🥮

这种测试通过日常细节中的不平等对待，来弱化你的社交地位，让你习惯卑微的相处模式。

具体表现例如：
*   女朋友吃剩的东西必须由男朋友吃完。
*   好的东西（如西瓜最甜的部分）必须优先给女性。
*   曾有案例中，男性因吃了月饼里的蛋黄而被女友在社交平台公开指责。
*   其长期目的是**让你习惯并接受“女尊男卑”的相处地位**，过程可能令人非常辛苦。

本节课中我们一起学习了四种常见的服从性测试：**经济测试**、**社交剥离测试**、**失格测试**和**价位测试**。识别这些模式是维护健康关系的第一步，关键在于建立相互尊重、平等沟通的互动原则。
