## 什么是晕轮效应？

藏在这个现象背后的秘密，便是心理学上的一个概念：**晕轮效应**。

其核心定义是：当你对某个人的某个特征或者特质产生**正面印象**后，这种印象会影响之后你对对方所有表达和操作的看法和定义。

我们可以用一个简单的**公式**来描述这种认知过程：

```
初始正面印象 → 认知滤镜 → 对所有后续行为的正面解读
```
