# 由浅入深的三十六个话题-P1-渣男学院-BV11T411t7R5_note

在本节课中，我们将学习一套由心理学家设计的、旨在快速建立深度人际连接的对话方法。这套方法包含三十六个循序渐进的问题，通过引导双方分享个人经历、价值观和情感，可以有效拉近彼此距离，创造“灵魂共鸣”般的体验。

上一节我们介绍了课程的整体目标，本节中我们将详细列出这三十六个核心问题。以下是完整的三十六个话题列表，建议在轻松、私密的氛围中，与对方轮流回答。

1.  如果现在有办法可以让你知道关于自己当下、未来乃至任何事情的真相，你会想知道吗？
2.  有没有什么事是你一直梦想去做但没有做的？为什么没有做？
3.  你觉得你人生中最大的成就是什么？
4.  在一段友谊之中最看重的是什么？
5.  你最宝贵的记忆是什么？
6.  你最糟糕的记忆是什么？
7.  假如你知道自己在一年内会离开这个世界，你会改变现在的生活方式吗？为什么？
8.  友情对于你来说意味着什么？
9.  爱与感性在你生活中扮演着什么样的角色？
10. 你觉得自己的童年比其他人更快乐吗？你家人之间的关系怎么样？
11. 你和你妈妈之间的关系是怎么样的？
12. 如果可以在这世界上所有人中任意选择，你想邀请谁共进晚餐？
13. 你想要出名吗？想以什么样的方式出名？
14. 打电话之前你会先排练一下要说什么吗？为什么？
15. 对你来说完美的一天是什么样的？
16. 你上次唱歌是在什么时候？
17. 如果你能活到九十岁，同时可以一直保持三十岁时的心智或身体，你会选择保持心智还是身体？
18. 你是否曾经想过自己会以怎样的方式死去？
19. 人生中的什么东西最令你感激？
20. 如果你能改变你成长过程中的一件事，会是哪一件？
21. 人生的前十六年是怎么样度过的？
22. 你觉得我们身上的哪些品质是好的？
23. 我们身上有相似的地方吗？是哪些呢？
24. 如果你明天一觉醒来就能拥有某种才能或能力，你希望那会是什么能力呢？
25. 你觉得如果我们一起从家里出门，我们会做的第一件事是什么？
26. 你此刻最想分享的是什么？
27. 如果你想和一个人进入亲密关系，有什么重要的事情是对方需要知道的？
28. 复述并告诉对方，你喜欢他身上的什么东西。
29. 和对方分享生命中一些尴尬时刻。
30. 上次在别人面前哭是什么时候？自己哭呢？
31. 给出可得性，告诉对方，你喜欢上他，是因为他身上的什么品质。
32. 你觉得什么东西是严肃到不能开玩笑的？假如有的话。
33. 如果你今晚就将死去，而且没有机会和任何人联络，你会因为之前没有对别人说什么话，而感到遗憾吗？为什么到现在都没有对他们说这些话呢？
34. 假设你想拥有的全部东西，都在你的房子里，现在房子着了火。在救出家人和宠物之后，你还有一次机会安全地冲进去，取出一件东西，你会拿什么？为什么？
35. 你经历过朋友或亲人的离世吗？当时是什么感觉？
36. 说出一件你的个人问题，问对方如果遇到此事，要如何解决。另外，也要让对方如实告诉你，在他眼中，对于这个问题的感受是怎样的。

了解了所有问题后，我们需要理解其背后的原理。这三十六个问题并非随意排列，其设计基于**心理学**原理。

这套问题改编自美国著名心理学家**Arthur Aron**的一项经典心理学实验，旨在研究如何快速建立人际亲密感。Gary在大量实践后，对原始问题和步骤进行了修改和优化，使其更适用于实际社交场景。每个问题都隐藏着一个或多个心理学效应，引导对话由浅入深。

最后，我们来探讨如何应用这些话题。在完整地交流这三十六个话题之后，双方的了解和信任通常会达到一个很高的层次。

此时，如果再配合适当的氛围营造和关系推进技巧（可理解为 **“张力拉满”** 的互动），你会发现，对方很可能已经对你产生了深厚的情感联结。

本节课中我们一起学习了建立深度连接的三十六问对话法。我们从列出全部问题开始，了解了其背后的心理学实验基础，最后探讨了实践应用的关键。掌握这套方法，能帮助你在交流中更有效地打开心扉，构建真诚而深刻的人际关系。
