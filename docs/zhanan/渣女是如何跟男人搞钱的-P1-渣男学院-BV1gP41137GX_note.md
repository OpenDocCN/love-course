# 渣男学院 · 第一课：识别“渣女”的搞钱套路 🕵️♂️

## 明导：摊牌式索取 💸

上一节我们介绍了课程概述，本节中我们来看看第一种手法——“明导”。这种方法的核心是直接摊牌，不掩饰索取意图。

“渣女”会明确告知追求者，自己“不再用耳朵相信爱情”，只认可“真实付出”的行为。其潜台词是：**追求我必须用金钱证明诚意**。

采用此方式的女性，通常没有固定工作，并会周期性“神秘失踪”。她们对外宣称去旅游或探亲，实则是寻找目标进行敛财。这种手段较为直接，段位不高，且容易留下证据导致“翻车”。

## 暗导：精细化运营备胎 📊

了解了直接的“明导”后，我们来看看更隐蔽的“暗导”模式。这种方法强调长期管理和精细化运营，不易被察觉。

“渣女”会同时发展大量“备胎”，并通过分组和备注进行系统化管理。其核心策略是**通过测试确定付出意愿，并据此制定索取策略**。

以下是其常用测试与分类方法：

*   **节日红包测试**：在节日发送小额红包（如几元钱），根据对方回馈的金额多少进行分类。回馈少的划为“低价值”目标，回馈多的则作为重点“培养”对象。
*   **频繁生日测试**：一年内庆祝多次“生日”，根据对方所送礼物的价值，进一步确认其经济付出意愿和能力。

## 组团搞：组织化协同作战 👭

如果说“暗导”是单兵作战的进阶版，那么“组团搞”则体现了组织化和团队协作，杀伤力最大。

野心最大的“渣女”会组成小团体，角色配合，共同实施计划。其核心模式是**利用“闺蜜”角色进行场景营造、施加压力和抓住把柄**。

以下是典型的团队协作步骤：

1.  **闺蜜助攻**：当你约目标A（如Tina）时，她会带上闺蜜B。B私下向你透露A对你有好感，并指明A喜欢的特定物品。
2.  **场景消费**：A会“恰好”将你带到闺蜜C的店里，那里正有她“喜欢”的物品，引导你购买。
3.  **社交施压**：A带你参加闺蜜们的聚会。其他闺蜜会以“你抢走了我们朋友”为由，要求你进行“赔偿”，即为你本不熟悉的整个聚会买单。
4.  **制造把柄**：有的闺蜜会私下对你进行略带玩笑性质的诱惑。如果你把持不住，就会留下把柄。目标A会据此与你翻脸，而你为了挽回关系，不得不再次花钱安抚。
