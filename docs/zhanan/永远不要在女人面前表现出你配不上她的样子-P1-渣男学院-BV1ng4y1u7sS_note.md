# 渣男学院 P1：永远不要在女人面前表现出你配不上她的样子 👑

在本节课中，我们将要学习一个核心的社交原则：为何以及如何在女性面前避免表现出低价值感。理解并实践这一点，对于建立吸引力和健康的互动关系至关重要。

## 核心原则：避免传递低价值感

你只要在一个女人面前表现出你配不上她、你很脆弱、生活很不如意的**感觉**，那么女人就会在心里淘汰掉你。她就不太想和你玩了。你都已经缴械投降了，还玩什么？女人择偶本身就是在“缴墙守护”。

上一节我们介绍了核心原则，本节中我们来看看其背后的心理机制。

## 心理机制：挑战欲与神秘感的丧失

你自己都不满意自己，你没有神秘感，她也就没有挑战欲了，就懒得在你心上浪费时间了。特别是在女人面前，用“你配不上她”这种语气跟她说话，把她像祖宗一样供养着，即使你再有内涵、再有钱、实力上比她强很多，也会给她一种错觉：就是你不行，你很好拿捏，你不值得喜欢。这叫做手握一副好牌，却打得稀烂。

## 进化心理学视角：择偶效仿

女人选男人，在进化心理学上存在“择偶效仿”。一个容易拿捏的男人，约等于对女人没有要求，也不敢对女人有要求。这说明他是被其他女人挑剩下的，是缺乏择偶选择权的。别人不要的，她又怎么会看上呢？

理解了进化心理学的背景后，我们来看看女性如何判断男性的价值。

## 价值判断：行为线索的重要性

女人找男人，骨子里就是在找高社会价值的男人。这个不像是外表身材那样可以直观判断，她们得依赖男人传递出来的**行为线索**。这个线索不仅包含你说的话，还包含你的语气、语调上有没有底气，肢体语言能不能传递出自信。这些线索，能映射出你是不是很值钱。

早在人类语言甚至文明诞生之前，女生就已经演化出这个能力了。一个不断获得成功的男人，跟一个总是经历失败的男人，表现出来的状况是完全不一样的。

## 姿态设定与一致性考验

一个男人出现在女人面前，她不知道这个男人经历过什么样的过往、拥有过什么样的社会资源。她只能通过男人传递出来的线索和姿态，来判断这个男人是不是值钱。

你可以随意设定自己的姿态，但是必须要经受住女人的**一致性考验**。你不断地化解考验的过程，就是女人对你投入度不断上升、最终喜欢上你的过程。女人口里所说的“感觉”，就是这样积累上来的。
