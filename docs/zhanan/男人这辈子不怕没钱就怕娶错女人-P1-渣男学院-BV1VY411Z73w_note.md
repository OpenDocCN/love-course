# 渣男学院 P1：识别“错的女人” 🚫

在本节课中，我们将学习如何识别几种可能对男性生活产生长期负面影响的特质。理解这些特质有助于做出更明智的人生选择。

上一节我们介绍了课程主题，本节中我们来看看具体的几种类型。

以下是四种需要警惕的特质：

1.  **不懂得鼓励，习惯讽刺打击**
    这种伴侣不提供情感支持，反而经常通过讽刺和打击来贬低对方。

2.  **消费无度且抱怨收入**
    其行为模式可概括为：**自身收入（低） < 个人消费（高）**，同时却抱怨伴侣收入不足。

3.  **偏袒娘家，排斥婆家**
    无论何时，都将娘家人的意见置于最高地位，视婆家人的意见为多余。

4.  **在外揭短，不顾及伴侣面子**
    习惯在公开场合揭露伴侣的短处或缺点，不尊重其尊严。

正如一位学员所述：“我们家那位中年男子…”，这种情况在现实中并不少见。

本节课中我们一起学习了四种需要警惕的伴侣特质，包括缺乏鼓励、消费失衡、家庭关系处理不当以及不尊重伴侣。识别这些特征是建立健康关系的第一步。
