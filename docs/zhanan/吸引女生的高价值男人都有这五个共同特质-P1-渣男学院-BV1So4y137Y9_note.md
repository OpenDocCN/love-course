# 课程一：从“匮乏者”到“富足者”心态的转变 🧠

在本节课中，我们将要学习情感世界中的“马太效应”，并深入剖析导致情感困境的“匮乏者心态”是如何形成的。我们将对比“穷人思维”与“富人思维”在感情中的不同表现，并明确从心态层面进行转变的核心路径。

大家应该都听过“穷人思维”和“富人思维”。拥有穷人思维的男人，在遇到事情时，会认为这是命运注定的，并用借口安慰自己，使失败合理化。在感情方面如果经历失败，就会觉得自己天生欠缺外在条件，因为这些条件的缺乏，才导致没有女人喜欢自己。

而拥有富人思维的男人，在遇到事情时，他们也会抱怨，但他们会思考需要怎么做才能改变现状，并且不会把原因归咎于外在条件以获得安慰。拥有富人思维的人，通常在遇到挑战时会更加兴奋，越困难越能激发他们的斗志。

不同的思维会造就不同的心态，不同的心态则会带来不同的结果。对于一个不掌握技巧、没怎么谈过恋爱的男人来说，更容易因穷人思维而进入“匮乏者心态”，进而转变成“供养者特质”。他们不相信凭借自己就能获得恋爱，不相信真实的自己也能创造出本能的吸引。

一个男人如果在感情里长期处于稀缺、不自信、寻求自我安慰的状态，必然会进入匮乏者心态。进入匮乏心态的男人，主要有以下三点特征。

以下是匮乏者心态的三个核心特征：

1.  **缺少思路和技巧，导致谨小慎微、唯唯诺诺。**
    很多男人在和喜欢的女生相处时，会陷入避免犯错的思维里：“这句话能说吗？会不会不好？是不是我做错了什么，她不理我了？”当你因思路缺失进入匮乏心态后，就容易受女生影响，情绪变得敏感。心态会先从内部崩溃，逐渐显露于外部，最终造成更多行动错误，并因此导致追求失败或关系结束。

2.  **因错觉导致害怕失去。**
    在和喜欢的女生相处时，害怕因自己犯错而失去对方。当男人陷入匮乏心态时，会错误地觉得择偶资源对自己是稀有的，从而错估自身价值，觉得世界上可能只有很少女人愿意接纳自己。一旦认为失去这个女生就等于失去未来和繁衍的唯一可能，那么离真正失去也就不远了。

3.  **低价值的恶性循环，导致无选择权。**
    无选择权是造成匮乏心态的主要原因。无选择权的男人会让女人觉得你是低价值的，会认为“没有女人喜欢你，一定是你的问题；别人都不选你，我也不选你”。在心理学上，这种行为叫“择偶复制效应”。

上一节我们剖析了“匮乏者心态”的形成与特征，本节中我们来看看其对立面——“富足者心态”。

“择偶复制效应”在女性选择配偶的过程中更为明显。女人在择偶时，会模仿同伴的择偶选择，她们会优先选择那些已经被其他女人选择过的男人。

而一部分掌握了技巧、与女生相处经验比较多的男人，对他们来说，制造吸引很简单。这些拥有“富足者心态”的男人，在感情世界里，必然也会进入“富足者”的思维模式。

拥有富足者心态的男人，所展示出的不怕犯错、无所谓的态度以及丰富的选择权，就足以打败身边90%的同龄男人了。

所以，我们应该认清自己，抛弃固有的穷人思维，去做那个在感情里拥有正确心态和技巧的“富足者”。

本节课中我们一起学习了情感中的“马太效应”，对比了“穷人思维”与“富人思维”如何导向“匮乏者”与“富足者”两种截然不同的心态。关键在于认清自身思维模式，通过积累经验和技巧，建立自信与选择权，从而完成从情感“弱者”到“强者”的根本转变。
