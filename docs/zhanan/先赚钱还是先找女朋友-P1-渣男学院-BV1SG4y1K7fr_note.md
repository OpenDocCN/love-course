## 核心计算模型 💰

我们首先来算一笔账。假设你目前一个月能挣八千元，这已经是不错的收入水平。全国月薪上万的人根本不到5%。

你工资八千元，除去吃喝拉撒、日常花销、房屋水电、偶尔吃烧烤和充值花费，最后每月还能存下五千元。

**计算公式：**
```
月存款 = 月收入 - 月总开销
年存款 = 月存款 × 12
十年总存款 = 年存款 × 10
```
代入数值：
```
月存款 = 8000 - 3000 = 5000 元
年存款 = 5000 × 12 = 60000 元
十年总存款 = 60000 × 10 = 600000 元
```
因此，一个月存五千，一年存六万，存十年你才存六十万。
