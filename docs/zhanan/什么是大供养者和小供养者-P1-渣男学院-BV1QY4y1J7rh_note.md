**小供养者**主要是指基因条件不佳、养育价值也不高的那部分男性。

很多人都以为供养者都在内卷，其实这个看法不全面。

她们还要假装把对方当成情人。情人的待遇当然比供养者的待遇要好得多。如果能选择情人的待遇，没有几个男人愿意选择供养者的待遇。

关键在于，情人待遇有“真”“假”之分。

所以，如果你得到的是供养者的待遇，不要怀疑，你就是个供养者。不会有女人傻到给一个情人供养者待遇。

但假如你得到的是情人待遇，那么你应该首先掂量一下：自己是真情人还是假情人？对方到底是想要你的基因，还是你的养育价值？
