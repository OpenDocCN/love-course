# 渣男学院 P1：快速调节恋爱心态的3个技巧 💘

在本节课中，我们将要学习三个核心技巧，帮助你快速调整在面对心仪女生时容易紧张、患得患失的心态。掌握这些方法，能让你在恋爱互动中更加从容自信。

## 技巧一：平常心看待目标 👥

上一节我们介绍了课程概述，本节中我们来看看第一个核心技巧：以平常心看待你喜欢的女生。不要将她神化或过度仰望。你心目中非常喜欢的女生，其实也是一个普通人。

女孩突然打扮好出现在你面前时，可能光彩照人。但若你一想到自己过得像个“龌龊屌丝”，就会感到配不上她。关键在于调整认知视角。

为什么小说中的韦小宝很会泡妞？因为他从小在妓院长大，见识了女人好的一面和不堪的一面，因此能以平常心对待。而你在成长过程中，可能缺少了可以全方位了解女人的机会。

以下是实践此技巧的具体建议：

*   **结交女性朋友**：有机会认识几个女孩做你的闺蜜，不要着急把每一个认识的女孩都追求一遍。
*   **延迟满足感**：要能够去延迟自己的满足感。**公式：放弃眼前一颗糖 = 可能得到后面十颗糖**。即，**`不急于追求当前一位女性 => 可能获得与更多女性交往的机会`**。
*   **建立长远视角**：记住，能熬得过暂时忍耐的人才能拥有未来。

## 技巧二：提升外形与优越感 👔

我们已经学会了以平常心看待目标，接下来，我们看看如何通过外部改变来快速提升内在自信。通过提升外形来提升自己的优越感，是一个见效显著的方法。

如果你出门能够盛装出行，你的状态就会立刻变得不一样。一个男人拥有一身得体的定制西装，或者一套合适的潮牌服饰，能极大增强自信。

如果你能修炼到靠自身气场就产生自信，或许可以不用穿得特别好。但那需要大量的实战经验和系统的学习训练。对于初学者而言，穿着得体是可以立竿见影出效果的方式。

很多人一辈子都不会有那个转折点，敢去做决定改变自己。从外形的提升开始，就是迈出第一步。

## 技巧三：降低心理预期 📉

上一节我们探讨了如何通过外形提升自信，本节中我们来聊聊心态调节的另一个关键：降低心理预期。很多兄弟和女生相处时，预期过高，这会导致一系列问题。

一上来聊天就认为这个女人是真爱，认为这辈子就想跟她过了，这把自己预期拉得太高。当你心里预期过高时，你就会非常害怕失败，因为你已经将她视为“未来的老婆”。

虽然现在还不是，但你已经在心理上把这个女人，当作了自己一生中最重要的女人。其实，你压根和她什么关系都不是。

一旦你把这个女人当成属于自己的，就会自动地去神化她，因为你希望自己未来的灵魂伴侣是完美的。尤其是，你会幻想和她在一起的美好场景，这将会进一步扩大自己的需求感。

当这个女人和别的男生有一些暧昧互动时，你的嫉妒心就容易爆发。你因此表现出来的一些行为，往往会给你减分。

以下是预期过高带来的具体问题：

*   **关系失衡**：如果你轻易地认同了这个女人，你们之间的关系就会不平衡。
*   **进入讨好框架**：你会让自己开始进入跪舔、讨好的模式。
*   **对方停止付出**：女人得到了你的完全认同，也就不会再对你付出。因为你都已经把她当自己老婆了，这是对她最大的“认可”，她无需付出任何努力就能获得你的认同，又怎么会愿意继续为你投入时间和感情呢？

所以，不要轻易地去认同一个女人，更不要把她看作你自己的一部分，尤其在你还不了解她的时候。因为她现在并不属于你。以后就算和你谈恋爱，也不一定会跟你结婚。就算结了婚，也不能算是完全“属于”你，因为人是自由的，她随时可以离开你。
