# 恋爱经济学：为什么很少有男生坚持追女生？ 🧠💔

在本节课中，我们将从社会变迁和风险收益的角度，分析为何当代男性在追求异性时，越来越少地采用“死缠烂打”的持久策略。我们将探讨其背后的风险系数、竞争环境与回报预期的变化。

---

## 核心风险系数的变迁 📈

上一节我们引入了主题，本节中我们来看看追求行为的核心变化：**风险系数**。

过去，追求行为的风险收益模型相对稳定。男性在追求初期投入大量情感与时间成本（可视为 **初始投资成本 C_i**），一旦成功建立关系，其“投资”便进入相对稳定的“持有期”。因为社会观念使得女性“不会轻易反悔分手”，关系的**违约风险 R_b** 较低。这使得长期来看，男性有机会在关系中调整权力平衡，甚至占据主导，实现 **净收益 N_p > 0**。

其模型可简化为：
**N_p = (长期关系收益 B_l) - (初始投资成本 C_i) - (低违约风险导致的损失 L_l)**

然而，当下的风险模型已截然不同。初始的高投入不再能保证低违约风险。即使追求成功，关系也极为脆弱，女性“说反悔就反悔，说分手就分手”。这意味着 **违约风险 R_b** 急剧升高，且男性需持续投入更高成本以维持关系（**持续维护成本 C_m**），否则将面临投资彻底沉没的风险。

此时的模型变为：
**N_p = (不确定的短期收益 B_s) - (C_i + C_m) - (高违约风险导致的损失 L_h)**
由于 `B_s` 不确定且 `L_h` 很高，**N_p** 很可能为负值。因此，从理性决策角度看，高强度的持久追求策略不再是最优解。

## 竞争环境的加剧与策略失效 🥊

理解了风险变化后，我们再来看看外部竞争环境如何使传统策略失效。

现在的女性选择面更广。这意味着男性追求者面临的**市场竞争强度 M_c** 大大增加。传统的“死缠烂打”（即高时间精力投入）作为一种差异化优势已经减弱。当竞争基数变大时，单一维度的执着并不构成核心竞争力。

以下是“死缠烂打”策略失效的两个关键点：

1.  **追求阶段的有效性降低**：在充分竞争的市场中，仅凭“坚持”这一特质很难脱颖而出，成功概率 **P_s** 显著下降。
2.  **关系维护的难度飙升**：即使进入关系，男性也需持续提供高情绪价值或物质付出以维持关系稳定，试图快速扭转关系权力结构（“把关系给掰过来”）的行为，极易被对方判定为“你变了”，从而触发关系终止。

这类似于投资中，一个曾经保本高收益的渠道，现在变成了高风险、低收益且竞争激烈的红海市场，理性投资者自然会减少投入或寻找新策略。

## 社会观念的演变与个体理性选择 🧩

上一节我们分析了外部竞争，本节我们将探讨内部动机——社会观念的演变如何影响个体的理性选择。

过去的观念类似于一种“契约”：确立关系后，双方默认进入长期合作阶段，变更契约的成本很高。这使得前期投入具有明确的长期回报预期。如同过去人们坚信“高考定终身”，值得反复投入（复读），因为考上后的收益（铁饭碗、社会地位）是明确且丰厚的。

其逻辑链为：
**高强度付出 → 达成目标（关系/高考）→ 获得长期稳定收益**

而现在的社会环境发生了变化。关系中的契约精神减弱，“违约”成本降低。同时，社会层面的“考上好大学”与“获得好人生”之间的因果关系也不再牢固，“毕业即失业”成为常见现象。当长期回报变得不确定时，前期的高强度执着就失去了依据。

以下是当前更普遍的理性策略选择：

*   **绕路走（策略转型）**：不再硬磕“追求”本身，转而提升自身综合价值（如事业、外貌、情商），吸引对方主动靠近。
*   **爬梯子（寻求杠杆）**：利用社交圈、共同兴趣等更高效的方式建立连接，而非直接进行情感上的“硬推销”。
*   **暂停投资（及时止损）**：评估后发现目标收益与成本不匹配，或风险过高，选择退出当前“市场”，等待更好时机或目标。

这就像面对一堵墙，过去撞破一面就能通关；现在发现后面墙叠着墙，理性人自然会放弃“铁头”策略，寻找更聪明、更省力的方法。

## 总结与核心要点 📝

本节课中，我们一起学习了当代男性较少“死缠烂打”追求女性的深层原因。

核心要点在于，这是一种基于环境变化的**理性策略调整**。主要驱动因素有三点：
1.  **风险收益模型改变**：前期高投入不再能锁定长期低风险回报，整体追求行为的预期收益下降，风险升高。
2.  **市场竞争加剧**：女性选择增多，传统“执着”策略的竞争力下降，且在关系中维护成本高昂、主导权难转移。
3.  **观念与回报预期变化**：社会契约精神减弱，长期关系的确定性降低，类似于“高考改变命运”信仰的松动，使得高强度执着失去意义。

最终的策略转向更理性的“投资”思维：评估风险、注重效率、及时止损，并致力于提升自身基本价值以增强吸引力。
