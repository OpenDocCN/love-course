## 营造心理高价值的三个核心策略

以下是实现这一目标的三个核心策略。

### 1. 保持心理高位 🧗‍♂️

人在面对高价值异性的时候，本能上就会产生仰慕情绪，不知不觉就会放低自己的身段去捧高对方。如果你不自觉地迎合讨好、各种嘘寒问暖，这么做的结果，就是在潜意识里告诉对方：**你配不上她**。

当你处于心理低位，无论说了什么做了什么，都是比她低了半截。这就违背了女人择偶的“慕强法则”。你把她捧得越高，就是在潜意识里把自己搞得越低，你这个人在女生眼里就会变得廉价。

暧昧的段子会变成油腻的套路，真诚的关心也成了别有用心。关系会处得非常累，经常是出力不讨好。就算是勉强走到一起，关系也很难扭转，你始终是处于弱势、被挑选的一方。

**核心公式：**  
`你的行为 = 潜意识信号` → `信号（迎合、讨好） = 价值低` → `吸引力关闭`

所以说，想要让女人喜欢你甚至倒追你，一定要把这个位置关系反过来。第一重要的是要保持住这个心理高位。你只有让她感觉你不比她低，敢跟她开玩笑调侃她，不在无脑迎合的时候，才能触发女人基因里的心动开关。

### 2. 假定她在追逐你 🏆

当你假定她喜欢你、她在追逐你，并且将这种假定转化为对应的语气跟她聊天时，就是把自己塑造成一个“奖品”。你的位置就摆正了，女人在潜意识里，才会认为你的价值比她高，吸引的发生是顺其自然的。

    *   **回应：** “怪我，让你等太久了。”
*   **场景：** 女生说在吃饭。
    *   **回应：** “吃吧，不用等我，我还得忙会。”
*   **场景：** 她如果说“谁要等你了”。
    *   **回应：** “我妈说，漂亮的女孩子最喜欢说反话。”

这个“假定追逐”一定要是潜移默化的，不要让女人感觉你在自嗨。要顺着她的话，自然地把这个框架夹带进去，不给她直接反驳的机会。这么做不仅在无形当中推进了关系，还保持住了心理高位。

反过来，当你卖力追求对方的时候，对方只会认为你是有所企图的，你的价值是比她低的。当女人感觉已经“吃定”你，招招手就能得到的时候，你就没有了神秘感，女人也就没有挑战欲了。这就像打游戏，毫不费力就通关、毫无挑战性的时候，这个游戏你就懒得玩了。

你想要吸引到她，就一定不能害怕失去她。吸引这件事情本质上，就是一种框架上的博弈。女人会对你的内在强度不断做测试。你一旦表现出害怕失去，你的整个言行举止就会变质。
