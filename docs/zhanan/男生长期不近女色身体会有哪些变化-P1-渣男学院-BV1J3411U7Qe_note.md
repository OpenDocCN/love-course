上一节我们概述了课程主题，本节中我们来具体看看长期不近女色可能带来的第一个显著变化。

**一、整体形象管理能力下降**

许多男生单身时间久了，会逐渐忽视个人形象管理。由于缺乏想要吸引的特定对象，对外在评价的在意程度也会降低。

*   不注重面部清洁，例如不刮胡子。
*   整体给人留下不修边幅、邋遢的印象。

了解了外在形象的变化后，我们进一步探讨这种状态对社交与情感能力的影响。

*   容易变得**大男子主义**，处事方式趋向简单直接。

*   更倾向于与同性朋友相处，并从中获得主要的情感满足，从而愈发习惯单身状态。

最后，我们来看看这种生活方式对最基础的生理机能可能产生的影响。

*   体能下降，可能出现“爬楼梯气喘吁吁”等状况。
*   整体精神面貌显得不够充沛。
