上一节我们介绍了课程的目标，本节中我们来看看第一个核心话术。

**一、请求语音消息**
你可以尝试发送这样一句话：“可以给我发段语音吗，没别的事，就是想听听你的声音。” 这句话直接表达了你想听到对方声音的渴望，将关注点放在她独特的个人特质上，而非具体事务，能有效拉近情感距离。

理解了直接表达想念后，我们接下来看看如何通过对比来加深印象。

**二、运用对比突出唯一性**
以下是具体表达：
*   “每次我看到别的女生，我都会想到你。”
*   “但是我每次看到你的时候，都会忘记其他女生了。”

这句话术通过“想到你”和“忘记其他人”的对比，巧妙地传达出她在你心中的独特性和不可替代性。

在表达了她的独特性之后，我们可以用一种更含蓄的方式来表达关心。

**三、含蓄表达关心**
“我想你现在一定很忙，所以只要看到前面三个字就可以了。” 这句话的核心公式是：**完整关心 = 表达想念 + 给予空间**。它先表达“我想你”，又立刻体贴地表示不必回复，这种克制反而更能触动对方。

除了直接的关心，将聊天与积极的体验关联起来也能产生良好效果。

**四、将聊天与美好体验绑定**
你可以这样说：“我发现，每次和你聊完天之后，我都睡得特别好，有你在真好。” 这句话将“与你聊天”和“睡得香”这个积极结果进行了心理关联，暗示对方是你的安心源泉。

最后，我们以一个带有调皮性质的叮嘱来结束睡前对话。

**五、以调皮叮嘱收尾**
“睡前不要想我，否则失眠就不好啦。” 这句话使用了反向暗示的手法，表面是叮嘱对方不要想，实际效果却是引导对方去想，以一种轻松幽默的方式加深自己的存在感。
