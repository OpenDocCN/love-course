# 渣男学院 P1：二十几岁不谈恋爱的后果 🚨

在本节课中，我们将要学习一个关于人生规划的重要课题：如果一个人在二十几岁时选择回避恋爱，可能会面临哪些长期的社交与情感后果。我们将通过一个具体的男性案例，分析其行为模式、心理变化以及最终可能陷入的困境。

---

一个男生在二十几岁时说：“我就不谈女朋友，单身挺好。”他找人喝酒、打游戏，他的兄弟都会陪他出去。

然而，等他过了三十岁，情况开始发生变化。他叫这个哥们喝酒，人家已经结婚了；叫那个兄弟打游戏，人家正在看孩子。

他开始显得和别的人格格不入。不需要别人说他，他自己就觉得不舒服，感到孤单了。

这个时候他再去谈恋爱，接触到的女生基本上都是年龄偏大的。能满足他期待的、单纯可爱的姑娘，早已把真心给了别人。

留给他的只有现实的要求。他因此越发觉得谈恋爱是一件非常痛苦的事情。公式可以概括为：**不谈恋爱 → 孤独；谈恋爱 → 痛苦**。

最终他后悔莫及。这印证了一个道理：出来混，早晚是要还的；倔强，是要付出代价的。

上一节我们分析了长期单身可能带来的社交孤立与择偶困境。本节中，我们来看看如何解决与异性聊天时找不到话题的根本问题。

大部分聊天进行不下去，最根本的原因就是找不到话题，只会每天问“吃了吗”、“睡了吗”、“在干嘛”、“忙不忙”。

解决这个问题，最根本的方法是用异性思维去聊天。男生是理性的，聊天旨在解决问题、直击重点；而女生是感性的，聊天重在分享自己的感觉和生活状态。这个核心差异一定要划重点。

以下是理解异性思维差异的一个具体例子：

> 女生说：“我最近在网上学很火的扭胯舞，但是怎么跳都卡不上节奏，真讨厌。”

大部分直男看到女生这样说，可能会问“扭胯舞是什么？”，或者安慰说“不会就算了，不学了，别生气”。持有这种想法的，可以说是“直男晚期选手”。

正确的回答方式是：“我最近也经常刷到扭胯舞的视频，卡节奏确实很难。你说我去跳会不会很骚气？”

这种回答形式，不仅可以让女生接你的话题继续聊，又能和女生的情绪保持在同一频率上。专业上这叫**情绪共振**。其核心还是用女生喜欢的聊天方式：分享自己的感受和生活状态。

这个方法可以适用于你跟任何人的沟通。
