## 概述

“渣男”是一个网络流行语，通常指在感情关系中行为不当、不负责任的男性。其行为模式并非无迹可寻，背后往往有一套可被归纳的思维与行动逻辑。理解这些基础概念，有助于我们更清晰地辨识相关行为模式。

## 核心行为公式

上一节我们介绍了课程概述，本节中我们来看看其核心的行为驱动公式。我们可以用一个简单的公式来描述其决策逻辑：

**核心公式：`收益 = 即时满足感 - (道德成本 + 潜在风险)`**

在这个公式中：
*   **即时满足感** 指行为带来的短期情绪或物质价值。
*   **道德成本** 指个人良知或社会规范带来的内心消耗。
*   **潜在风险** 指行为可能导致的未来损失（如关系破裂、名誉受损）。

当计算出的 **收益 > 0** 时，该类行为被触发的可能性会显著增加。这解释了为何某些行为会反复出现，因为它们被个体感知为“有利可图”。

## 常见特征列表

理解了核心驱动公式后，我们可以进一步观察其外在表现。以下是“渣男”行为中一些常见的特征列表：

1.  **缺乏共情能力**：难以真正理解或顾及伴侣的感受和需求。
2.  **承诺与行动不一致**：经常做出口头承诺，但很少或从不兑现。
3.  **界限模糊**：在已有伴侣的情况下，仍与其他异性保持暧昧或不恰当的亲密关系。
4.  **逃避责任**：在出现问题时，倾向于指责对方或寻找外部借口，而非反省自身。
5.  **信息不透明**：对自己的行程、社交关系等重要信息刻意隐瞒或撒谎。

## 基础辨识代码示例

认识特征后，我们可以尝试用更结构化的方式来思考。以下是一个极度简化的Python代码示例，用于演示如何通过条件判断来辨识一些基础行为模式（请注意，此为概念演示，现实情况远为复杂）：

```python
def behavior_analyzer(actions, words):
    """
    一个简单的行为分析函数。
    :param actions: 实际行动列表
    :param words: 口头承诺列表
    :return: 风险提示
    """
    red_flags = []

    # 检查承诺与行动是否一致
    if set(words).isdisjoint(set(actions)):
        red_flags.append("警告：承诺与行动严重不符。")

    # 检查是否常找借口
    excuse_keywords = ["都怪你", "我也不想", "是Ta先..."]
    for exc in excuse_keywords:
        if exc in ' '.join(actions):
            red_flags.append("警告：习惯性逃避责任。")
            break

    # 检查信息透明度（模拟）
    if "隐瞒行踪" in actions or "含糊其辞" in actions:
        red_flags.append("警告：信息不透明。")

    return red_flags if red_flags else ["未检测到典型风险行为。"]

# 示例调用
用户行动 = ["约会迟到", "隐瞒行踪", "忘记生日"]
用户承诺 = ["我会改", "下次一定", "我只爱你"]
result = behavior_analyzer(用户行动, 用户承诺)
print("分析结果：", result)
# 输出：分析结果： ['警告：承诺与行动严重不符。', '警告：信息不透明。']
```
