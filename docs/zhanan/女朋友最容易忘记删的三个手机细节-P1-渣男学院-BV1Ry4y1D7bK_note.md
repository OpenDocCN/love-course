# 女朋友最容易忘记删的三个手机细节-P1-渣男学院-BV1Ry4y1D7bK_note

在本节课中，我们将学习如何通过检查手机上的三个特定细节，来了解女朋友的社交互动情况。这些细节通常容易被忽略，但能揭示最近的频繁联系人。

---

上一节我们介绍了课程目标，本节中我们来看看第一个需要检查的细节。

**第一个细节是短视频软件的分享列表。**

打开手机上的短视频软件。
随意点开一个视频，然后点击“分享”按钮。
分享界面通常会有一个“分享给朋友”的选项。
**在这个列表中，排在前两位的联系人，就是她最近通过私信互动最频繁的人。**

这个列表是由系统算法自动抓取和排序的。
**它并非个人可以手动控制或编辑的。**
用户也无法单独删除这个互动记录。
因此，通过这个列表，你可以清楚地了解她与谁的亲密度更高、互动更频繁。

---

了解了第一个基于算法的细节后，我们接下来看看第二个需要关注的方面。

**第二个细节是社交软件的“消息回复”时间戳。**

以下是检查步骤：
1.  打开常用的社交软件（如微信）。
2.  进入与特定联系人的聊天窗口。
3.  注意查看每条消息的发送与回复时间。
4.  计算**两次回复之间的平均时间间隔**。
    *   公式：`平均回复间隔 = (最后回复时间 - 首次发送时间) / 回复次数`
5.  对比她与你聊天时的回复速度。

如果她对某个人的回复速度显著快于其他人，尤其是在非工作时段，这可能意味着她对该对话的优先级更高。

---

前面我们探讨了分享列表和回复速度，最后我们来审视第三个关键细节。

**第三个细节是手机相册的“最近删除”相簿。**

很多人认为删除照片就等于清除了记录。
然而，大多数手机系统都有一个“最近删除”的文件夹，被删除的照片会在这里保留30天左右。
直接检查这个相簿，可能会发现一些已被“删除”但尚未被永久清除的图片或截图。

---

本节课中我们一起学习了检查女朋友手机社交痕迹的三个关键细节：
1.  短视频软件的“分享给朋友”列表，揭示了系统判定的最近频繁联系人。
2.  社交软件中的消息回复时间间隔，能反映对话的优先级。
3.  手机相册的“最近删除”相簿，可能保留了以为已删除的信息。

通过系统地关注这些细节，你可以更清晰地了解她的社交互动模式。
