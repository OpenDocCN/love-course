# 恋爱课程 01：理解关系中的“越权”行为与两种思维模式 🧠

在本节课中，我们将要学习男女交往中一个常见的误区——“越权”行为。我们将深入探讨其本质、产生的原因，并对比两种截然不同的人际关系思维模式。理解这些概念，能帮助你在建立关系时更加尊重对方，避免不必要的冲突。

## 越权行为的本质

上一节我们介绍了课程的主题，本节中我们来看看什么是“越权”。

越权的本质，是越权方过于想达成自己的目的，而做出所谓“抢买抢卖”的行为。具体到追求异性，本质就是你作为越权方想达成自己的目的，然后想逼着别人来跟你“成交”。

这个比喻可以这样理解：你想买一个东西但钱不够，对方不卖给你，你却非要用这个价钱买，甚至用除了付钱以外的其他办法，希望对方把东西卖给你。虽然这个比喻并非完全恰当，但核心逻辑相似。

当一个人急于想得到女生时，对方会觉得你提出的要求，无法给予她真正想要的东西。因此她不想“成交”。这不是指金钱，而是指对方的核心需求未被满足，例如安全感、稳定感等。

## 意志如何被改变？

理解了越权的定义后，一个关键问题是：人的意志如何被改变？这直接关系到越权行为为何会失败。

人的意志是由心理需求被满足时决定的。当需求被满足，意志就可能改变。

以下是两个例子来说明：

*   **交易行为**：如果让你每天洗碗，你通常不愿意。但如果洗碗一个月奖励一台车，洗两个月奖励更贵的车，很多人就会愿意。这是因为物质需求被满足了。
*   **情感与责任**：如果母亲手受伤了，请求你帮忙洗碗，出于亲情和责任，你也会愿意。这是情感需求或责任感被激发了。

人的意志改变，要么出于需求被满足，要么出于责任感被激发，或是有情感的需要。如果这些条件都不具备，却硬逼对方做一件事，就会导致抵触。即使对方被迫做了，心里也会积攒负面情绪，长期来看会损害关系。

## 越权行为的具体表现与后果

那么，在追求关系中，越权行为具体有哪些表现？又会带来什么后果呢？

在人际关系中，越权就是向另一方提要求，让对方做违背本心的事情。例如威逼、利诱、跪舔、纠缠，这都是通过生硬的方式让对方做违悖本意的事情。

这种行为本质上提出了超越当前关系阶段的要求，让对方感受不到尊重。不被尊重主要有两种情况：

1.  强求对方做不愿意做的事。
2.  否定对方的意见和感受，不倾听对方说话。

很多人在追求中被拒绝时，完全听不进对方“不合适”、“不想谈恋爱”等明确意见，总试图说服对方。或者在被拒绝后，仍持续送礼物、找借口联系对方。

这些行为会让对方感到麻烦和压力，从而产生戒备心。一旦对方产生戒备，你做的任何事情都可能被解读为问题和麻烦。对方为了解决问题，可能会选择彻底回避你，即“解决掉问题的源头”。

## 两种人际关系思维模式

我们分析了越权行为的表现和恶果，其根源在于一种错误的人际关系思维。接下来，我们对比一下两种根本不同的思维模式。

在人际关系中，存在两种思维模式：

**第一种：控制与剥夺思维（排序思维）**

当人急于得到短期利益时，会产生这种思维。其核心是**零和博弈**，即你的优势建立在对方吃亏的基础上。这是一种“巧取豪夺”的思路，使用暴力和智谋去算计、控制对方。

在男女关系中，总想着“怎么把对方约出来”、“怎么引导对方去私人影院”，研究各种技巧和步骤去达成目的，就属于这种思维。它关注的是“控制的科学”，即如何算计他人。

**第二种：连接与关怀思维（理解思维）**

当人真心想要理解他人感受、关怀对方时，会产生这种思维。其核心是**同理心**和建立连接。它追求的是理解与双赢，激发的是彼此的责任感。

例如，父母不是用命令强迫你洗碗，而是坦诚自己的困难并寻求你的理解与帮助，这就是在激发连接思维。对方能感觉到真诚，而非算计。

## 课程总结

本节课中，我们一起学习了恋爱关系中的“越权”行为。

我们首先明确了**越权的本质**是强迫对方满足自己的需求，而忽视对方的意愿。接着，我们探讨了**人的意志**只有在需求被满足或情感被激发时才会真正改变。然后，我们列举了越权行为的具体**表现与后果**，即导致对方反感、戒备并最终远离。最后，我们揭示了其根源在于**控制与剥夺**的思维模式，并倡导转向**连接与关怀**的理解思维。

记住，健康的关系建立在相互尊重与理解之上，而非单方面的算计和强求。
