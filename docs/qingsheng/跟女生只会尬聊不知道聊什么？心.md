# 跟女生只会尬聊不知道聊什么？心

在本节课中，我们将要学习“尬聊”现象的本质及其成因，并掌握一套通过建立熟悉度和展示价值来自然开启对话、避免尬聊的实用方法。

## 什么是尬聊？🤔

上一节我们提到了尬聊的普遍性，本节中我们来看看尬聊的具体表现。跟女生总是在尬聊，不知道说什么。我们来解析一下尬聊。

典型的尬聊包括：
*   在干嘛
*   吃了吗
*   到哪了
*   要注意安全

这些都属于没话找话的典型尬聊。

## 尬聊的根源：想法与不熟悉 🕵️♂️

理解了尬聊的表现后，我们需要探究其背后的原因。如果我跟你一样的情况，我也会尬聊。原因很简单，你本来跟人关系就不怎么熟，你还老想着去跟人家聊天。

我肯定也会尬聊，因为我对人家一无所知，我只知道他长得好看，我只想着怎么跟他聊天。尬聊的原因是什么？其实原因很简单，是你有想法。你没想法，你至于跟人家尬聊吗？没想法，你至于在完全不合适的情况下去跟人聊天吗？

## 核心策略：停止没话找话，创造展示机会 🚫

认识到问题所在后，我们来看看解决方案。我的做法不是说，我不会跟人家尬聊。我跟你们的不同是什么？我不会去没话找话说。我有的时候我宁可不跟你聊天，我哪怕我对你感兴趣。我宁可不跟你聊天，我宁可去等一个机会。

或者说，我在不跟你交流这个时候，我去找机会。就比如我在路上跟人搭讪，人家一开始不想给我微信，我想了点办法，然后把人家微信搞到手了。搞到手了之后，当时已经加了好友，然后我跟他说话，他就是那种一问一答的，明显语气很冷淡。

## 实战步骤：从朋友圈到关联话题 🔄

上一节我们介绍了停止强聊、等待机会的核心思路，本节中我们来看看具体如何操作。在朋友圈里面，或者说我会看一下，我朋友圈里面都发了什么内容。或者我会有意识地去发一下朋友圈，给他去看一些可能他会去了解我的信息。

你比如说，我生活当中我去炒个菜什么的。首先为什么要这么做？原因很简单，第一个我跟你没什么聊的，这种情况下，你对我回应冷淡的情况下，这一般是什么问题？我首先不说人家是不是对我感兴趣这个问题。基本上只要你去主动撩这个女生，她没有任何道理完全不关注你，肯定会去看你的朋友圈，肯定会去想这到底是个什么样的人。

我跟你聊天，你如果不怎么回我的话，我过几天，我再找个事。比如说我跟你在某条路上认识的，我在这个路边，比如我看到猫在捉老鼠，我就把猫捉老鼠这个视频拍下来，然后发给他。

以下是具体操作的两个关键点：

1.  **制造关联性话题**：发信息说“上次我们见面的地方，这旁边有个大黄猫，经常在这里捉老鼠，挺好玩的”。就类似去找这种跟我们之间有关系性的事，开始跟他聊。
2.  **自然展示与延伸**：如果我在朋友圈里面发过猫的话，他肯定看到了，他也知道这个人养猫的。然后我是不是可以跟他稍微聊一下这个事？当然我一般情况下，我是不会特意去跟女生聊我养猫这个事，就是说如果说刚好这个事有关联的话，我可能会拿出来说一下，看一下人家对这个话题感不感兴趣。

## 效果升华：建立认可，告别尬聊 ✨

通过关联话题开启对话后，如何将对话深入并建立稳固的印象呢？就说其实除了养猫，我平时的生活也挺简单的，就说我就大概在网上写写文章。你说这个时候人家就开始说，你到底是干嘛的？你如果第一个点，如果切中了人家的兴趣或者跟你们之间相关的话，然后这个时候我再交流，是不是你就可以转其他话题？

因为他之前看过你的文章，人家就会说你到底干嘛的，会看你在朋友圈里面，也会分享类似这样的文章。这个时候你这又是展示优点的时候了。就说我也没什么，我们在网上做这一行，然后再给他看几篇文章，都写的挺好，有的东西确实不错。

你想再搞个几个点，虽然我说的是我自己的本身的，你自己也可以有这个东西去发展的。最基本的人家对你的这种认可度，至少来说他会不会觉得这种人当朋友挺好的？他对这种东西他挺有研究的，他挺了解的，平时我们也没有机会去认识这样的这种朋友。

你说之后你们再去聊天或者说是干嘛的，还会不会尬聊？至少你可能一两次尬聊，你到后面你基本上就不会尬聊，因为有了一个基本的，我们说叫**认可度和熟悉度**。

本节课中我们一起学习了尬聊的根源在于“有想法”但“不熟悉”，破解的关键在于停止没话找话的强聊。我们掌握了一套通过经营朋友圈展示生活、寻找关联事件自然开启话题、并在对话中自然展示自身价值以建立认可度和熟悉度的完整方法。核心公式是：**停止强聊 -> 创造展示机会 -> 寻找关联话题 -> 自然建立认可 -> 告别尬聊**。
