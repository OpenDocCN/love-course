## 1. 多传递情感，少传递目的 💖

上一节我们介绍了信号传递的重要性，本节中我们来看看第一个核心原则：情感与目的的平衡。

传递情感意味着关心对方的感受和状态，而传递目的则意味着只关注自己的需求和目标。后者容易让对方感到压力和反感。

**核心公式**：
```
正确信号 = 多考虑对方感受 + 少强调个人索取
```

以下是具体做法：

*   **关注对方细节**：留意并记住她的喜好、习惯和舒适度。例如，天冷时她是否穿得单薄，吃饭时长发是否不便。

*   **化解矛盾**：发生争执时，多从对方的角度思考问题，而非只坚持自己的立场。
*   **展现积极情绪**：见面时微笑、表达见到她的开心，这本身就是传递“我喜欢你”的积极情感信号。

**关键点**：少传递目的性，不代表不能有目的。例如，“想约你见面”是正常目的，但“想从你身上占便宜”则会让人反感。你的目的应是增进关系，而非索取。

## 2. 多表达关注，少传递情绪 🧘

理解了情感与目的的区别后，我们进入第二个原则：在互动中，如何管理自己的输出。

“表达关注”是指将注意力放在对方身上，而“传递情绪”则是指将自己的慌乱、无聊或不知所措等状态暴露给对方。

**核心代码**：
```python
# 错误示范：传递负面情绪

# 正确示范：表达积极关注
```

以下是具体做法：

*   **避免尬聊**：当对方联系你时，不要用“嗯嗯啊啊”或开启无聊话题来勉强回应，这只会传递你的无措。
*   **直接表达关注**：可以坦诚地说你正在关注她，例如“我正在看你的朋友圈”。
*   **巧妙结束对话**：当你不得不结束聊天时，可以表达不舍并转移关注点。例如：“真想继续聊，但我得去洗澡了。睡前能看看你吗？发张照片给我吧。”
*   **保持状态稳定**：不要让对方感觉到你的慌张或迷茫，这会给对方带来情绪负担。

**关键点**：让对方感受到你的在意和上心，同时保持自身情绪的稳定和从容。
