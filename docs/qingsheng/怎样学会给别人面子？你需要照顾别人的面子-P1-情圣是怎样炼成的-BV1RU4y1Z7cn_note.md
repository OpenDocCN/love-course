## 什么是“场面”？与“体面”的区别

上一节我们介绍了“体面”，即如何做好自己。本节中我们来看看“场面”是什么。

“场面”就是指你要给别人面子。你需要照顾别人的面子，其根本目的是让他人产生被重视和被尊重的感觉。

## 核心概念：每个人的“自重感”

这里涉及一个核心心理学概念。美国作家卡耐基在《人性的弱点》一书中提出：**每个人都有自重感**。

这个公式可以理解为：
**自重感 = 被重视的感觉 + 被尊重的感觉**

换到中国文化语境中，满足他人的自重感，就是“会做场面”、“会做人情”。其关键在于：在表面上要让人家过得去，要让人家觉得自己有面子、被尊重。

## 哪些行为会伤害别人的面子？（反面案例）

以下是几种常见的不尊重他人、伤害对方面子的行为，请注意避免：

1.  **地域歧视与刻板印象**：例如，当面询问“你们河南人是不是都偷井盖？”
2.  **不当称呼**：在不熟悉的情况下，直接直呼对方全名。
3.  **公开让人难堪**：例如，当众揭别人的短处或管束对方。
4.  **不顾场合的“直言不讳”**：即使你说的是事实（比如指出对方胖），在不合适的场合说出来，只会让人尴尬和不适。这种行为除了满足表达欲，并不能增进好感。

## 情商的核心：让人感到舒服

我们的课程旨在探讨情商的本质，即如何在与人交往中，让人感到舒服。这涉及到杜月笙曾总结的三个层面：体面、情面、场面。

我们已经讨论了体面（做好自己）和场面（给对方面子）。在下节课中，我们将尽量给大家讲解什么叫“情面”。情面关注的是，你如何让别人觉得与你亲近。
