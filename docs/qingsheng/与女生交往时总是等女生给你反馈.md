# 与女生交往时总是等女生给你反馈

在本节课中，我们将要学习两种核心的思维模式：主动型思维与被动型思维。我们将探讨它们如何形成，以及它们如何深刻影响我们在生活、工作和人际关系（特别是与异性交往）中的行为与结果。

## 概述：我们为何要讨论思维模式？

我们发现，大多数人都缺乏一种关键的思维方式。这种思维方式学校不会教授，但它却是解决现实问题的核心。

学校教育的模式通常是提供理想化的模型和给定条件。例如，在数学题中，小明跑30米，小红跑60米，问题会明确给出所有参数。这种教育培养的是一种“解题式”思维：只要遵循既定步骤和方法，就能得到确定答案。

步入社会后，许多工作也延续了这种模式。入职后，领导通常会安排明确的工作流程和步骤。员工只需按部就班，就能完成既定任务。长期处于这种环境中，人们容易形成一种固定的思维模式：认为凡事都有标准流程，只要按流程走，就能获得预期结果。

然而，现实与学校的最大区别在于，现实中的条件往往是不确定的，没有现成的“解题步骤”。

## 主动型思维 vs. 被动型思维

上一节我们介绍了学校教育带来的思维定式，本节中我们来看看与之相对的两种现实思维模式。

**主动型思维**是一种“获取资源型”思维。其核心在于，在条件不确定、没有固定步骤的情况下，主动采取行动进行尝试，并从结果中学习和调整。其公式可以概括为：

**行动 -> 反馈 -> 调整 -> 再行动**

**被动型思维**则是一种“等待条件型”思维。它期望外界给出明确的条件和步骤，只有在自认为“条件成熟”或“方法正确”时才会行动，否则就处于观望和焦虑中。

以下是两种思维在追求异性时的典型表现对比：

*   **主动型思维者**：愿意主动尝试认识对方，不惧怕被拒绝。他们认为“大不了被拒绝，但试一下就有机会”。
*   **被动型思维者**：总是等待“完美时机”或“正确方法”。他们会反复空想“该怎么追”，却迟迟不采取实质行动。

为什么许多文化水平不高的人却能创业成功？正是因为他们具备主动型思维，敢于在未知中尝试。这与追求异性类似，等待资源（异性）主动上门是不现实的，必须主动出击。

## 被动型思维的常见表现

理解了核心概念后，我们来具体看看被动型思维在情感领域的几种常见表现。以下是几种典型情况：

1.  **空想筹划，缺乏行动**：遇到心仪对象，花费大量时间空想如何追求，学习各种理论，但始终没有迈出实质性的一步。
2.  **迷信“秘籍”**：认为追不到女孩是因为没有掌握某个绝招或秘密技巧，将成功寄托于寻找外部“方法论”。
3.  **依赖反馈，患得患失**：总是等待对方给出明确的积极反馈后才敢推进下一步；一旦对方反应冷淡，就陷入焦虑和自我怀疑，停止行动。

这些表现的共同点在于，将行动的主动权交给了外部条件或他人，而非掌握在自己手中。

## 主动型思维的实践：以教员思想为例

那么，如何培养和实践主动型思维呢？我们可以从一些卓越实践者的思想中汲取智慧。

教员的思想是主动型思维的极高体现。其精髓在于：在充分分析客观条件的基础上，充分发挥人的主观能动性，不等待万事俱备，而是创造条件去达成目标。这要求我们：

*   **接受条件的不确定性**：承认现实不像习题，没有给定的X和Y。
*   **主动尝试与验证**：通过实践（行动）来测试想法，获取真实反馈。
*   **迭代与调整**：根据反馈迅速调整策略，而非固守一套僵化的“话术”或“步骤”。

过去一些PUA（搭讪艺术）试图将人际交往简化为固定步骤，最终导致了严重的社会问题，也无法帮助人建立真实的亲密关系。因为它违背了主动型思维中“根据真实反馈灵活调整”的核心。

## 总结

本节课中我们一起学习了主动型思维与被动型思维的根本区别。

*   **被动型思维**源于我们所接受的“解题式”教育，它让我们习惯于等待明确条件和步骤，在情感中表现为空想、迷信秘籍和患得患失。
*   **主动型思维**是应对不确定现实的必备能力。它是一种“获取资源型”思维，核心在于**主动行动、获取反馈、快速调整**。在人际关系中，它意味着主动创造机会，并能够承受尝试过程中可能出现的拒绝或挫折。

要改变在情感中的被动局面，首先需要从改变思维模式开始：从等待“解题条件”的被动者，转变为在不确定性中主动“创造答案”的实践者。
