# 课程一：情圣是怎样炼成的？掌握与女生交往的核心思维 🧠

在本节课中，我们将要学习如何与女生建立真诚的连接，避免因错误的思维和行为方式而引起对方的反感。核心在于理解“连接思维”与“排序思维”的区别，并调整自己的心态与行为。

很多时候，女孩子讨厌你，并非因为你没钱或其他外在条件，而在于她觉得你接近她的目的是想占便宜，而非真心对她好。

这就像在路边遇到野猫野狗，如果你直接冲上去试图抓住它，它一定会逃跑。因为你们不熟悉，它不清楚你的意图，当它明显感觉到你可能带来侵害时，躲避是本能反应。这是生活中非常简单的道理。

你们只是看到了表象，认为“对女孩子好，却让她跟了别人”。问题的关键不在于“对她好”这个行为本身，而在于你的“好”是虚弱的、愚蠢的、急功近利的，并且是带有明确目的的。

例如，你刚认识一个女孩子，你所认为的“好”就是送她一部苹果手机。这反而会让女孩立刻远离你。彼此还不熟悉，就送上贵重礼物，这会让对方产生疑虑：你接下来打算做什么？我是否需要为此付出未知的代价？这给人造成了巨大的心理压力。

你们很多自以为的“好”，实际上是一种很低级的方式，会让人产生恐惧和害怕。你们没有思考过自己对别人好的方式有多低级。

很多人从一开始就打着把对方“搞到手”的主意，强调自己的欲望。这种情况下，别人通常会产生排斥。这在心理上，是把自己当成了未成熟的人，只会一味地向别人索取：“给我，给我”。仿佛“我弱我有理，我没有，所以你应该给我”。

因为你自身匮乏，才会总想着向别人要。如果你自身富足，你会更倾向于思考如何给予。例如，如果你真的富有，邀请喜欢的女孩吃饭、为她买衣服，这本身应该是一件快乐的事。带朋友去高级场所玩乐，给女朋友买贵重的包包，也应感到乐趣。

你只有在匮乏时，才会觉得做这些事是不开心的。正因为你弱，你心中没有爱，本就没有钱，什么都没有，所以你才会伸手向别人要。因为你缺这些东西，所以会被人讨厌，这是一个恶性循环。

这种来自他人的反感和排斥，会伤害你的自尊心。一旦在生活中自尊心受伤，感到不被尊重，人就会启动“与人交往的排序思维”。

跟女孩子交往，要采取的是 **“连接思维”** 。在什么情况下我们会采取“排序思维”呢？是在与人争夺的时候，例如比拼武力、智谋、权谋、技巧、套路。这属于排序思维。

而 **“连接思维”** 的核心是：关心、关怀、温柔、善意、体贴、诚恳。这是能够让你与人建立连接的思维方式。前一种（排序思维）则会导致矛盾，割裂你们的关系。

例如，就连站出来帮别人一个忙，你都觉得是为了获取好感，是为了故意装作仗义。你已经自私到这个地步，怎么可能拥有朋友？其实，真正去追求女孩，就是给出爱和关怀，能够宠溺她，能够诚恳地对待她。这很简单，没有那么多乱七八糟的。

首先，你们之所以会这样，是因为缺乏自信心。说白了，在与女孩交往时，你缺乏自信，所以才会想去套路别人、算计别人。一个人没有自信，根源在于自尊水平太低，对自己没有要求。

以下是提升自我、改善交往的四个关键步骤：

**第一步：提升自尊**
你需要提升自尊水平，对自己提出要求，建立自信的根基。

**第二步：找准角色定位**
认清自己本身的角色，并说符合该角色的话。如果你是个普通人，就不要装逼。放得开一点，活泼一点，搞笑一点。即使是个普通人，但做一个无所畏惧、真诚开朗的人，女孩子也不会讨厌。

**第三步：多关注他人**
只有多关注别人，对方才能感受到你的关心，才会喜欢你。如果你只关注自己，别人就会讨厌你，觉得你在算计他，觉得你这个人怪怪的。

**第四步：学会换位思考**
在我的课程中，有一节专门教授换位思考。即如何分析别人的需求，如何与人聊天。说白了，你不会聊天，是因为你不会换位思考。一旦学会换位思考，你就知道应该聊什么、怎么说、话说到什么点上、以及何时该转换话题。

很多时候，你们需要学习的，是自己动脑筋思考的能力。

本节课中，我们一起学习了与女生交往的核心误区在于急功近利的“排序思维”，而正确的方式应是真诚的“连接思维”。关键在于提升自尊、找准定位、关注他人并学会换位思考。改变思维，才能改变关系。
