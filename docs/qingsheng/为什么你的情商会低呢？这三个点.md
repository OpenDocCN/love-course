# 为什么你的情商会低呢？这三个点

在本节课中，我们将要学习情商的三个核心障碍：脾气控制、自我中心以及价值判断。理解并克服这些障碍，是提升情商、建立更成熟人际关系的第一步。

## 1. 脾气控制：情绪稳定的基石 🧘

上一节我们介绍了情商的核心定义，本节中我们来看看第一个关键点：脾气控制。情商高的人，首要特征是能够在任何情况下保持内心的安全感。这种安全感源于对各种场合的从容应对能力。

以下是具体表现：
*   面对批评时，知道如何妥善回应。
*   面对赞美时，知道如何恰当处理。
*   初次见面时，知道如何得体交流。
*   面对他人反应时，知道如何灵活应对。

因此，高情商常给人成熟稳重的感觉。成熟与情商紧密相关，而控制情绪是成熟的第一步。**公式：情商高 ≈ 情绪稳定 → 应对得体 → 显得成熟**。

## 2. 突破自我中心：扩大你的认知圈 🌍

仅仅控制情绪还不够。接下来，我们探讨第二个障碍：自我中心。其核心问题在于个人认知的“圈子”过于狭小。

这会导致一个严重问题：你只熟悉自己“一亩三分地”里的事情。一旦遇到陌生或超出掌控的情况，就容易陷入慌乱。

例如，生活中突然出现一位极具吸引力的人，许多人会不知所措，进而行为失当。常见的错误模式是：开始不理智地一味讨好、赠送礼物，结果往往沦为“备胎”或被骗。**代码逻辑：if (遇到陌生强刺激) { 情绪上头 → 理性失效 → 行为错误 }**。

## 3. 减少价值判断：看清事实本身 🔍

在学会控制情绪并扩大视野后，我们来看第三个障碍：过多的价值判断。这会影响我们对事实的客观认知和判断。

“过多的价值判断”是指，遇到事情或人时，本能地优先判断“这对我好还是坏”、“这是好人还是坏人”，而不是先客观分析事实本身。这种预设立场会扭曲认知，阻碍你做出理性、恰当的回应。

---

本节课中我们一起学习了阻碍情商发展的三个核心陷阱：**无法控制脾气**、**自我中心导致的认知狭隘**，以及**过多的价值判断影响客观认知**。识别这些自身问题，是迈向高情商和成熟人际交往的第一步。
