# 课程一：告别“少爷脾气” 🧠

在本节课中，我们将要学习一个在人际交往，特别是恋爱关系中非常关键的概念——“少爷脾气”。我们将探讨它的表现、本质，并学习如何通过提升心性成熟度来克服它，从而建立更健康、更持久的关系。

## 什么是“少爷脾气”？🤔

上一节我们介绍了课程主题，本节中我们来看看“少爷脾气”的具体表现。它意味着一个人的心性不成熟。最重要的一个表现是，人还停留在以自我为中心的阶段。因此，很容易对自己不理解、或不符合自身利益的事情闹情绪。甚至很容易把这种情绪固化成自己的一种价值观，变成对事情的看法和理念。长此以往，这会使人，尤其是对自己未来的发展，很难真正融入社会。因为融入社会需要与他人，尤其是男性，建立各种关系。无论是生意上的合作关系、职场上的工作关系，还是家庭关系（包括婚姻、父母子女关系），其实都与我们的心性相关。

所以我们要讲的是，人的心性成熟，一定与一个人的担当有关。担当意味着很多时候，你除了要把自己的生活过好，有时还要挑起生活中其他人的责任。也就是说，你越有担当，承担的事情越多，你代表的东西就越多。例如，你是一家之主，担起一个家的责任，你就代表这个家。你是一国之主，挑起国家的重担，你就代表这个国家。因为担当意味着你一个人的意志和状态要代表一个群体。最基础的担当，是把自己的生活担起来。这代表你自己，代表你的门面、形象和对外打交道的方式。我们古语，儒家中有句话叫“**立己达人**”。意思是首先要“立”起自己。这个“立”不是指立起你的“自我”。现在很多说男生“脱不了单”，实际上与他们社会融入性很差有很大关系。就是说，他还没有走向社会。例如，在走向社会之前，你首先走向恋爱，经营一段恋爱关系。这代表你生活中有了别人。很多人，比如大学生暑假回家，根本不帮家里干活，整天关在房门里玩电脑手机。这哪有什么担当？所以，当人进入一段恋爱关系时，你要承担的是另一个人对你的期望、信心，以及对未来的期许和托付。当然，有人通过短期荷尔蒙也能进入恋爱。但很多这类人的恋爱关系不能持久，无法给对方一个结果。所以我们这一系列课程最主要的理念，是希望让大家更好地融入社会，更快地让自己变得可靠、靠谱、值得信赖。

因此，我首先想告诉大家，为什么现在很多男生脱不了单是因为“少爷脾气”。有“少爷脾气”不代表你本身是少爷。现在有些人没有少爷的命，却空有少爷脾气。他不能理解别人，不懂得关心别人，甚至受不了一点委屈，吃不得一点亏。这就是我们今天要讲的主题。

以下是“少爷脾气”的几种常见表现：

1.  **跟女生交往总是急于要反馈**：只要女孩没及时回应，就容易情绪化。
2.  **对待女生的一些小任性不能包容**：总是要求男女平等，觉得自己不能吃亏。认为多付出一点、多包容一点就成了“舔狗”。但包容他人和跪舔他人是两回事。跪舔属于丢失权利，包容等于忍耐情绪。
3.  **周围人总评价你心性不成熟**：但你又不知道如何变得成熟。

## “少爷脾气”的本质与危害 ⚠️

上一节我们列举了“少爷脾气”的表现，本节中我们来深入探讨它的本质和可能带来的危害。

首先，对于无法接受的事情，接受的前提是理解。你为什么接受不了女生的一些行为？原因很简单，你不理解女性的处境。在谈接受之前，要先尝试去理解。如果你一开始就先入为主，觉得事情太麻烦、太难、不公平，那后面就没有然后了。人对待一件事情，如果遇到不能接受或利益有损的情况，例如，我党当年与国民党打仗，教员的首要要求就是研究国民党，甚至要从国民党方面学习。所以，如果你要理解社会上的一些理念，你要知道这些理念怎么来的，为什么有人这么说，再去反驳也不迟。如果一开始就抱着反对的想法，这对他人和自己都是不负责任的。我希望有的同学，在反对之前，先试着去理解一下。这对你，对你未来的感情和家庭都有好处。

“少爷脾气”的本质是心性不成熟，以自我为中心，缺乏担当和共情能力。它最大的危害是破坏关系，错失良机。

我过去一年做的情感咨询，90%的案例都是因为“少爷脾气”把大好局面葬送了。例如，有一个典型的案例：一个男生工作能力、人品各方面都没问题，但有点少爷脾气。他和一个女生聊得很好，约会很多次，女生可能在观察他。有一次，女生迟到了，让他等了一两个小时。他憋不住，觉得对方太不尊重人，就发了脾气。结果，这件事就黄了。有人觉得等两个小时发脾气很正常。但关键是，在你想追求对方的关键阶段发脾气，即使你再有理，关系也大概率会结束。如果此时你能控制住脾气，女生反而会觉得你脾气好、包容她。但如果你发脾气，女生一定会想：现在还在约会你就这样，以后在一起了指不定会怎样。发完脾气后，男生又觉得受不了，回头低三下四地找女生，女生反而觉得他没感觉了。一来二去，关系就彻底结束了。

## 如何让心性更成熟？🌱

上一节我们看到了“少爷脾气”的危害，本节中我们来看看如何通过提升心性成熟度来克服它。

首先，我要澄清，我不是来批评大家的。“少爷脾气”虽然不好，但这是一个正常现象。这说明男生普遍晚熟。以前男生二十四五岁就显得很老成，现在很多男生三十多岁还像年轻人一样，气质上晚熟。

那么，如何变得成熟？关键在于培养担当和共情能力。

1.  **从理解开始，而非对立**：遇到不理解或觉得不公平的事情时，先别急着反对或发脾气。试着去理解对方为什么这么做，处境如何。就像研究对手一样，理解是沟通和接纳的第一步。
2.  **分清“包容”与“跪舔”**：包容是强者对情绪的忍耐和对关系的维护，其核心公式是：**包容 = 理解 + 情绪管理**。而跪舔是放弃原则和尊严的讨好，其核心是权利的单方面丧失。在恋爱中，适当的包容是必要的。
3.  **以目标为导向管理情绪**：发脾气几乎从不能帮你解决问题或达成目标，它只会增加解决问题的成本。在想要达成目标（如追求心仪对象）的关键阶段，管理好情绪至关重要。问问自己：发这顿脾气，对我的目标有帮助吗？
4.  **从小事开始练习担当**：担当可以从经营一段恋爱关系开始，也可以从关心家人、负责地完成工作做起。当你开始习惯为他人、为一段关系、为一个集体考虑时，你的心性自然会走向成熟。记住“立己达人”，先立稳自己，再惠及他人。

## 总结 📝

本节课中，我们一起学习了“少爷脾气”这个概念。我们了解了它的三种主要表现：急于求反馈、不能包容小任性、被评价为不成熟。我们探讨了它的本质是心性不成熟和以自我为中心，并通过案例看到了它如何破坏关系、错失良机。最后，我们学习了克服“少爷脾气”的方法：从理解他人开始，分清包容与跪舔，以目标为导向管理情绪，并通过练习担当来促使心性成熟。希望这节课能帮助你更好地认识自己，在人际交往中变得更加成熟、可靠。
