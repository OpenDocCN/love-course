# 为什么女生感觉你“有毒”？心理：每日解析一个你在男女交往中的底

在本节课中，我们将深入探讨一个在男女交往中常见的心理现象——“人际防御”。我们将解析其产生的原因、表现方式，以及它如何影响你传递给对方的信号，最终导致关系陷入僵局。通过学习，你将理解为何有时你的行为会让对方感到困惑和不适，并学会如何调整自己的心态和行为模式。

## 人际防御的起源：情绪不流畅与内在冲突

上一节我们概述了课程主题，本节中我们来看看“人际防御”究竟从何而来。

人际防御是情绪不流畅的表现。情绪不流畅意味着一个人内心存在冲突。这种内在冲突往往与自己对某件事的判断和看法有关。

例如，你认为自己没钱对方就不会喜欢你，你认为付出就是“舔”，或者你认为自己给不了对方幸福。你持有这些前置看法，但你的核心欲望却保持不变——即你想和对方在一起。

因为你既有这个欲望，同时又有很多其他的想法和判断。想法和判断越多，你的内在冲突就越多。冲突越多，顾虑也就越多。

顾虑越多，你就总会思考“我应该这样，我应该那样”。你想得特别多，但实际表现出来的东西却特别少。你心里“应该”的东西越多，你对对方的情感输出就越少。

## 模糊的信号：为何对方感到困惑？

上一节我们了解了内在冲突如何产生，本节中我们来看看这种冲突如何外化为令人困惑的信号。

你给对方传递的信号会变得莫名其妙。回想关于“信号”的课程，你的行为传递了特定信息。

以男生为女朋友精心挑选礼物为例，相比直接转账让她自己买，前者的行为传递了**爱的信号**。

但当你顾虑过多时，你的表现会变得奇怪。可能有女生向你反馈过，觉得你的表现有点奇怪，不知道你要干嘛，也不知道你对她是什么意思。

感觉你好像想撩她，但又觉得不是真心的，是试探性的。女生可能会反馈说“感觉你情绪不稳定”，或者说“你是个好人”。你的反馈会越来越差。

女生对你的反应是：她搞不清楚你在干嘛，她也感觉不到你是喜欢她的。你传递出来的信号会让女生不知道如何去解读。

女生对你的感觉和态度，取决于你给她传递了什么信号。当她说“你是个好人”时，通常是因为你的行为给她这样的感觉：你想追她，但又有点畏畏缩缩，不敢去表达，好像总想很多，又表现出自己很纯情的样子。

女生会觉得混乱，感觉这个男生有点黏黏糊糊，不知道他自己在干嘛，心里感觉不对劲。

所以，如果你本身内在冲突和顾虑越多，你传递出来的信号就会很模糊。

这就相当于**WiFi信号不强**，对方感受不到你对他的喜欢和爱，可能只感觉到别的东西。你的信号属于那种“WiFi信号不强”的类型。

## 关注自我 vs. 关注关系

上一节我们提到了气魄的重要性，本节中我们来看看当过度关注自我时会出现什么问题。

当你全程都在想“我应该这样，我应该那样”的时候，别人为什么会觉得奇怪？

因为当你在想“我应该这样”的时候，你解决的是自己的顾
虑，明白吗？你解决的是你自己的问题，你**不是在跟对方谈恋爱，你在解决你的问题**。

你说吸引怎么吸引呢？难道你本身没有吸引力，然后能变出吸引力来吗？很多时候，你想得越多，附加条件越多，反而不知道怎么办。

有人说得对：对方会觉得“你这个人有毒”。

## 面对拒绝：是放弃还是解决问题？

## 人际防御的四大根源

上一节我们看到了恶性循环的后果，本节中我们系统性地总结导致“人际防御”心态的四个核心原因。

为什么你会有人际防御？

**1. 自我评价太低，容易在意他人评价**
你会通过他人对你的态度和评价来评价你自己。自我评价一低，很容易通过他人对自己的评价来评价自己。
*   **公式**：`自我价值感 = f(他人评价)`。当函数`f`的权重过高时，自我价值会随他人态度剧烈波动。
*   例如，你喜欢上一个女生，女生对你态度好，你就自我感觉很好；态度不好，你就自我感觉糟糕，认为“果然我很差劲”。

**2. 缺乏承担风险的魄力**
这样的人会害怕遭遇挫折，总希望一切确保万无一再出手，导致行为和表达上束手束脚。
*   **表现**：总希望有人手把手教，用几个“招”一定能成。但追女生的趣味正在于其不确定性带来的刺激感，如同博弈。缺乏承担风险的魄力，因为害怕遭遇处置（挫折）。

**3. 自我效能感太低**
认为自己一旦发生什么问题就搞不定，所以会害怕处置（挫折）。
*   **对比**：为什么有人勇敢？因为他觉得就算遇到问题或挫折，自己也可以搞定。而自我效能感低的人，如觉得“异地问题搞不定”，就会想退缩。当你想“继续还是放弃”时，实际上就是想退缩，因为你怕搞不定。

**4. 习惯进行价值判断**
这种容易在意别人评价的人，自己也特别习惯去评价别人。习惯性地对事情进行价值判断的人，非常容易情绪化。
*   **定义**：价值判断往往带有情感色彩。例如，女生不理你，价值判断是“这女的是傻逼”或“这女的对我没兴趣”。非价值判断则偏重事实，如“她可能在忙”。
*   **后果**：价值判断越多，越容易有人际防御。因为你对每件事都做价值判断（如“她回我了，肯定喜欢我”），这会导致心理包袱和负担。

## 如何改变：从价值判断到事实分析

上一节我们剖析了人际防御的根源，在本节也是最后一节中，我们来看看切实可行的改变方法。

如果你想做一个能够搞定事情、增长能力和智慧的人，请做到以下几点：

*   **少对别人的事情指手画脚**，少随便对网上信息做价值判断。
*   **多发表基于事实的不成熟观点**，把事情理清楚再说。
*   **在生活当中多去看事实**，多去想现实因素、客观因素。

这样做的好处是：
1.  首先，情绪会少一点。
2.  其次，你解决问题的能力会变强大。

**举例对比：**
*   **价值判断路径**：女生不理你 -> “她是不是不喜欢我？” -> 下次见女生时有负担 -> 纠结“我应该说/做什么？” -> 表现不自然。
*   **事实分析路径**：女生不理你 -> “她是不是心情不好？/我找她的时间有问题？/我穿着有问题？/我说话方式有问题？” -> 调整这些现实点 -> 提升下次互动或与其他女生互动的质量。

当你总在想“她是不是不喜
欢我”，你就会有心理顾虑和负担。有负担就会想“我今天应该这样、应该那样”，结果把自己搞得人不人鬼不鬼地在人家面前。
