# 情圣是怎样炼成的 - 第1课：跪舔行为的核心因素 🔍

在本节课中，我们将深入探讨两性关系中“跪舔”行为的核心成因。我们将从心理动机出发，分析其背后的逻辑链条，并帮助你理解如何识别和避免这种模式。

---

## 核心因素：期望与可能性

上一节我们介绍了“跪舔”这一现象，本节中我们来看看其根本原因。导致跪舔的核心因素，是所有相关问题综合到一起的答案，其实就是**期望**。期望即**可能性**，是你主观上判断一件事是否有希望、是否有可能成功。

当你觉得自己可能性不大时，就会解释很多人为何会去跪舔。许多人跪舔女性，本质上是因为他们觉得自己成功的可能性不大，所以才会不停地去讨好对方。

## 跪舔的逻辑链条

跪舔并非单一行为，而是一个有步骤的逻辑过程。以下步骤构成了跪舔行为的完整链条：

1.  **自我价值感低**：首先，你觉得自己不行，自我价值感不足。
2.  **判断可能性低**：接着，你觉得自己可能搞不定对方，成功可能性不大。
3.  **产生强烈欲望**：尽管觉得可能性低，但你仍然非常想要得到对方。
4.  **采取讨好行为**：为了达成愿望，你开始采取讨好行为，例如发红包、送礼物、对对方予取予求、对方说什么你都答应。

## 焦虑与防止失去的动机

许多同学会问：“我这样做会不会是在舔？”“我那样做会不会显得太在乎他？”在如今网络思潮普遍批判“舔狗”的环境下，存在这些疑问本身就说明问题。

只要你开始问这些问题，往往就意味着你已经处于跪舔的心态中。因为当一个人自我价值感足够、对事情有掌控感时，通常不会产生这些焦虑。人们只有在感到焦虑、对局面控制感不强时，才会反复纠结于行为是否得当。

这种焦虑感会导致你总是以**防止失去**为动机去接触女孩。所有的跪舔行为，本质上都是为了防止失去。

## 跪舔的后果：压抑真情实感

当你以防止失去为动机去接触女孩时，会导致一个关键问题：你无法在女孩面前流露真情实感。

几乎所有被称为“舔狗”或“备胎”的人，都无法真实地表达自己的情感，他们的情感被严重压抑。这就是为什么“舔狗舔到最后一无所有”。

女性在考虑接受一个男性时，需要感受到对方的**真实情感**和**坚定态度**。如果你的心理机制导致对方无法感受到这些，关系就很难成功。

## 不自觉的心理状态

跪舔往往不是一种有意的投机取巧行为，而是一种**不自觉的心理状态**。许多人也很烦恼，并不想跪舔，但受困于心理机制而无法突破。

这种无法突破的心理机制，正是导致各种奇怪行为和言论的思想根源。

**本节课总结**

本节课中，我们一起学习了“跪舔”行为的核心心理因素。我们了解到，其根源在于**自我价值感低**和**对成功可能性的悲观判断**，这催生了以防止失去为动机的讨好行为，最终导致真情实感被压抑，关系难以健康发展。理解这一逻辑链条，是改变这种行为模式的第一步。
