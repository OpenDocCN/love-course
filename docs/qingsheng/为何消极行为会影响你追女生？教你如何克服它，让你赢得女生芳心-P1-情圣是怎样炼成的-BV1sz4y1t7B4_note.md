## 概述：消极行为的根源与影响

## 第一节：识别并解决消极思维与价值观

上一节我们介绍了消极行为的根源，本节中我们来看看如何具体识别和解决消极的思维与价值观问题。

消极思维和价值观，指的是社会上人云亦云的那些话，以及网络上流行的各种抱怨性舆论。这种思维和价值观对你的行为没有半点好处。社会上网络上那些舆论，能不听就少听。

第一个方法是不要跟着网上大多数人的想法一样。如果一个人按照大多数人的思维能够做成一件事情，那这个世界上就不会有这么多普通人了。

解决了思维问题，接下来我们看看如何改变行为。这样的人最难改变的是行动力太差。

**行动力公式**：`想法 -> 立即行动 -> 经验积累 -> 能力提升`

所以，很多时候你首先要管理你的行为，管理你的行动。

在培养了行动力之后，我们还需要关注内在精神世界的健康。本节将探讨如何避免精神上的“自我放纵”。
