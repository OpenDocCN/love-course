## 概述：依赖外部技巧的陷阱

你越学习那些技巧，你对自己就越没有信心。这很简单，因为你离开那些技巧就什么也不会了。你越学习那些技巧，你就越自卑。你学习那些技巧的目的，是因为这不是靠你自己的能力去解决问题。

人如果永远依赖某些东西，自己就永远不会成长，甚至可能越来越退步。

## 依赖的恶性循环：以技巧为例

以下是依赖技巧带来的问题：

1.  **能力停滞**：依赖技巧如同儿童依赖父母。儿童若一直依赖父母，成年后便缺乏社会生存能力。同样，依赖技巧会让你失去独立处理人际关系的能力。
2.  **自信降低**：你的能力没有提升，自信就无法提高。你很清楚这一点，因此在面对异性时会更加胆怯，不敢表达自己。
3.  **安全感缺失**：长期依赖会导致你在需要独立面对问题时缺乏安全感。例如，遇到漂亮的女性时，你感到的是紧张和焦虑，而非兴奋。

**核心公式**：`依赖外部技巧` → `个人能力停滞` → `自信降低` → `安全感缺失`

上一节我们介绍了依赖如何导致安全感缺失，本节中我们来看看由此产生的“困难感”。

你之所以关心如何聊天、如何约会、如何使用肢体接触，都是因为紧张和焦虑，而不是兴奋。你觉得与异性交往是困难的。

**关键点**：言行不恰当 → 女性拒绝（但原因不明）→ 强化困难感与焦虑

**核心逻辑**：`缺乏自我要求` → `无底线追求结果` → `失去他人尊重` → `无法建立真正吸引力`
