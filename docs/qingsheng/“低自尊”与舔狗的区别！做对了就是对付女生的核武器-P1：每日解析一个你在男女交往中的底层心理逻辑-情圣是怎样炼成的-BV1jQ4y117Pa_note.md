# 课程名称：情圣是怎样炼成的 - 第1课：低自尊与舔狗的区别 🧠💔

## 概述
在本节课中，我们将要学习男女交往中的一个底层心理逻辑：如何区分“低自尊”与“舔狗”行为，并理解主动出击的正确方式。掌握这个逻辑，能有效改变你在感情中的态势。

---

感情中决定谁占优势、谁处劣势的，永远是背后的逻辑：谁更需要对方多一点。如果这个核心需求关系不改变，你就永远无法改变你们之间的态势。

有人谈论各种“框架”或技巧。但如果你自身条件不佳，却生硬地套用框架，没有人会理你。

主动行为表面上看起来确实有点“掉价”，比如你主动去找别人。

但是你会发现，如果你的主动方式能得到对方认可，那么初期短暂的“掉价”是值得的。举一个最简单的例子，我们不说谈恋爱，说创业开公司的人。

你见过很多创业开公司的人，当老板开第一家店的时候，同样需要主动出击。他自己也要变成员工，出去拉客户，主动做这些事情。不可能一开始所有事情都让员工来做。

一开始，他是不是也需要放下自尊，主动出击？是的。但是等他的企业做大、公司成功之后，他还需要这样做吗？不需要了。如果你在初期放不下面子、低不下头去主动出击，你什么也得不到。

这就是说，在生活中如果我们真的遇到自己想要的东西，第一个麻烦就是：别太看重自己的自尊心。在你一无所有的时候，自尊心是最不重要的。

包括很多学习PUA（搭讪艺术）的人，如果你们能懂得在一开始放弃掉过强的自尊心，很多事情就能做到。要能够埋下头、低下姿态，才能真正得到你想要的好东西。

很多人过不了自卑这个坎，就是因为过不了自尊心这道门槛。他们因为害怕自尊心受挫，所以不敢主动接近别人。

如果你能把自尊心暂时放下，你还害怕这些吗？你不会害怕。你会厚着脸皮，该怎么做就怎么做。

男人女人都可以这样理解。因为你想得到你想要的东西，就不要那么在意你的自尊心。当你什么都不是的时候，你的自尊心不值钱。

什么是真正的自尊？是你跟别人交往时，特别强调自尊、要求别人尊重你；还是最终你搂着漂亮的伴侣，开着豪车，住着豪宅？哪个是真正的自尊？

一定是后者：你搂着漂亮的伴侣，住着豪宅，开着豪车，这样才有自尊。但是，如果你本来不是富二代，这种自尊从何而来？它一定是在前期通过“不要自尊”换来的。

所谓的“不要自尊”就是：我能主动去争取客户，客户再怎么骂我、嫌我烦，我都能耐着性子把单子谈成，把钱赚下来。这叫做一开始不要自尊。

我们追女孩也一样，或者女性追男性也一样。一开始对方再怎么冷淡、拒绝，我通过我的方式、执着和坚持打动了他，最终他选择跟我在一起。

有人说自尊是别人给的吗？我告诉你，自尊是自己挣来的。你的自尊、你的钱、你所有的一切都靠你自己争取。如果你一开始就过于在乎自尊，你什么事情都做不成。

这是强者逻辑和弱者逻辑的区别。强者永远是：有本事，没脾气。下等人才会：既没本事，又有脾气，过分在乎自己的自尊。上等人受了气，自己消化，打掉牙齿往肚子里咽，这才是强者逻辑。

强者最不在乎情绪，只在乎如何把事情搞定。弱者永远在情绪上斤斤计较。

你们很多人都是弱者。你们这帮学PUA的人最搞笑，你们是生活中最被人看不起的垃圾、臭虫，最弱、最废、最没用的人。你们好意思在这里谈自尊？好意思说自己不是弱者吗？有比你们更弱的吗？说得难听点，有吗？

你自己跟身边的人对比一下，你比谁强？谁都比你厉害。你随便跟身边的同学、朋友、兄弟姐妹比一下，你比得过谁？你还不务正业，天天在这里研究怎么泡妞。

说得毒舌一点，你活着是在浪费这个世界的阳光空气，你吃饭是在浪费社会的粮食。真的，你不打起精神好好生活，不如去死，没有任何价值。

所以学PUA的这帮人，你们要明白，你们的尊严、你们的自尊是低到尘埃里的，谁都看不起你。还不好好振作自己，去正常地工作和生活，天天沉沦在这里面研究怎么泡妞，去满足自己下半身的欲望，你不觉得丢人吗？

好，下一个话题。

由于长期在情感上得不到回应，有些人只好把注意力转移到游戏上，逃避现实中的挫败感，让自己陷入抑郁。有这种人吗？有。

你已经对现实生活完全彻底失去信心，开始逃避了。不管你如何将注意力转移到工作上，这也不是一种良性的方式。如果你产生这种逃避心理，很多时候你需要做的是面对生活中的问题，而不是逃避。

比如感情上有问题，你要去面对自己感情上的问题，去想自己怎么解决，要迈出这一步。不要放弃自己。

为什么你会在感情上得不到回应，然后转移到游戏或工作上，逃避现实挫败感？因为你在面对困难和问题时，习惯于逃避。你最终这种大的逃避行为，实际上是你在生活中一直都在逃避：遇到一点点问题或困难，你从来不去解决和克服。

所以你就会逃避现实中的挫败感。你会发现这种人是高度敏感的。比如经常逃避问题的人，心理承受能力会差到什么地步？举个例子，在撩妹时，如果女孩子稍微对他冷淡一点，他就完全放弃；如果女孩子回消息慢一点，他就会发脾气。

所以，你们在情感上长期得不到回应，实际上是因为你们习惯于逃避，不去解决问题。包括你们学PUA，是一种终极的逃避行为：你自己已经彻底放弃治疗了，去找别人教你这种投机取巧的东西。这实际上是对困难的一种逃避。

你们要更多地让自己去面对生活中的问题，而不是逃避。所以我一直反对PUA，就是因为男人追女人，或者说男女交往这件事，说得难听点，是本能。每个人作为男性或女性，怎么跟异性说话，这不是本能吗？

很多时候，你们因为害怕受伤、害怕挫折、害怕没有回应，所以逃避了。是心里受不了那个打击，怕别人拒绝。

所以我刚才说了，很多时候你们一点承受能力都没有。我们说当一个人要成大事，我们可以看到历史上所有的英雄，难道谁是一开始就特别牛逼的？

真正的英雄，为什么很多人都佩服刘备这种人？刘备失败了一辈子，但他从来没有放弃过自己。他顶多会哭一下。他去喝酒之后，发现自己腿肚子上长出赘肉，就感叹自己年纪大了，志向还未实现。

为什么说刘备是英雄？就是因为他老是失败，但从不放弃，最后终于成就了霸业。这有点像你们追女孩：如果你被拒绝1000次还能继续追，你也是条汉子了。

很多时候，你被人拒绝几次就放弃了，甚至女孩子稍微冷淡一点，你都受不了。你们这样的心理承受能力，能把什么事情做成？

其实就是很多时候，你们自己在生活中，抗挫折、抗压、抗负面情绪的能力，自我消化能力实在太差了。所以你才会容易痛苦和抑郁。

同样的事情，为什么有的人心态好，内心强大？实际上是因为你顶着多大的负面情绪和不适感，把事情做成了，你的承受能力才会变强。

所以很多时候弱者弱在什么地方？看心理承受能力：发生一点点事情，就特别容易情绪化，我们就可以看出这个人是个弱者。

举个例子，在知乎或群里，别人稍微跟你有一点意见不合，你就能跟人喷起来。你们有没有这种状态？稍微有点意见不合，你就特别容易跟人喷起来。

为什么你们会喷起来？说实话，就是你们情绪控制能力差。你们都知道，跟人喷起来一点意义都没有：你说服不了他，他也说服不了你。两个人观点不同，吵了一天，什么好处都没得到，还互相觉得对方是傻逼。

在游戏里、网站上，各种各样的情况，都说明情绪控制能力很差。但如果说有一种情绪，比如我看这些学PUA或教PUA的人觉得恶心，我问你们这种情绪是不是正当的？这种情绪是正当的。

一般来说，或者对一些明显对大家有害的，比如在公屏上刷屏、说污言秽语、传播不正确价值观的，我一般喷的是这种人。我觉得这种情绪是正当的。

但是有的情绪，我觉得是不正当的。比如跟你观点不合，你就非要去说服别人、骂别人，我觉得这是不对的。你去跟人家怼一天，一点好处都没有，耽误你的生活，也耽误别人的生活，何必呢？

所以很多时候，喷子喷我一下，或者有点情绪，我随便他说好了。反正吹牛不上税，人嘴两张皮，我随便你去说，反正我也没做错事情。

接下来我们说自卑。见到自己喜欢的人就沉闷、情绪低落，然后别人越看到这样的你，他就越不喜欢。你越是这样，人家越对你这样；你越是沉闷低落，别人就越不喜欢。有这个情况的吗？有。

很多时候都是这种情况。感情中本来就是：决定谁优势、谁劣势。

## 总结
本节课中，我们一起学习了“低自尊”与“舔狗”行为的核心区别。关键在于理解：**真正的强者逻辑是“有本事，没脾气”**，为了达成目标可以暂时放下过度的自尊；而弱者是“既没本事，又有脾气”，沉迷于情绪内耗。同时，我们剖析了逃避心理的成因，指出**长期情感受挫后转向游戏或工作是一种逃避**，真正的解决之道是面对问题、提升心理承受能力。记住，自尊不是别人给的，而是靠你自己挣来的。
