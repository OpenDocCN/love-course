# 对他好他就会对你好，他对你不好

在本节课中，我们将学习一种在人际交往，特别是两性关系中常见的心理模式——“跪舔”。我们将剖析其核心思维习惯、背后的心理机制，并探讨其对立面——“过激”反应。理解这些概念是建立健康人际关系的第一步。

## 什么是“跪舔”？学术定义与核心思维

上一段我们提到了课程主题，本节中我们来看看“跪舔”的具体定义和表现。根据心理学研究，“跪舔”行为与一种被称为“取悦症”或“看管性性格障碍”的心理模式相关。其核心是希望通过讨好他人来获得认可、避免冲突或维护关系。

以下是具有“跪舔”倾向的人常见的四种思维习惯：

1.  **交换思维**：认为“我**对别人好，别人就会对我好**”。如果对方没有给出好的反馈，则归因于“**一定是我付出不够多**”或“**一定是我做得不够好**”。
2.  **维护思维**：认为“**只有不断付出才会维护好关系**”。这种付出（如借钱、帮忙）常常源于害怕对方离开或冷落自己的恐惧。
3.  **价值证明思维**：**通过取悦别人来证明自己的价值**。所有讨好行为都是为了获得他人的正面反馈和认可，以此获得自我认同感。例如，在追求异性时，对方的一个笑容就能成为自我价值的证明。
4.  **冲突避免思维**：认为“**取悦别人可以避免冲突和受到伤害**”。即使在受到欺负、羞辱时，仍希望通过说好话、送礼等方式换取对方的温和对待。

## “跪舔”的对立面：过激反应与心理根源

上一节我们介绍了“跪舔”的几种思维模式，本节中我们来看看与之紧密相关的另一种现象——“过激”反应。

许多激烈批判“跪舔”行为的人，其本身很可能曾有过“跪舔”经历。在意识到这种行为的问题后，他们采取了另一种极端的应对方式。例如，他们会将“对方拒绝后仍坚持联系”等行为，一概定义为“不要自尊”和“跪舔”。

**“跪舔”与“过激”如同一枚硬币的两面，根源相同，表现形式相反。** 它们的共同心理根源在于 **“压力管理的失当”**。

无论是追求心仪对象、拒绝他人请求还是融入新群体，都会产生压力。这两种行为模式错误地试图解决 **“情绪”**（如恐惧、羞耻、挫败感），而非解决 **“带来情绪的压力源本身”**。

*   **跪舔**：试图通过讨好来消除对方的“负面情绪”（实则是缓解自己的焦虑），但无助于真正解决问题（如建立吸引、达成合作）。
*   **过激**：试图通过攻击（言语或行为）来捍卫“自尊”（实则是掩饰自己的无力感），同样无助于解决问题。

这就像射击时，靶子（问题）前突然出现一个吓唬你的人（负面情绪）。正确的做法是击中靶子，吓唬你的人自然会消失。而“跪舔”和“过激”都是在攻击那个“吓唬你的人”，结果就是靶子永远打不中，且会不断有新的“吓唬者”出现。

## 如何改变：从关注情绪到创造价值

前面我们分析了“跪舔”与“过激”的成因，本节中我们探讨如何改变这种模式。关键在于将注意力从“管理他人情绪”和“维护自我感受”转移到 **“解决问题”** 和 **“创造价值”** 上。

真正的能力体现在用结果说话。无论是销售产品还是建立人际关系，最终目标都是达成积极的结果（成交、建立吸引）。脾气大或一味讨好都无法实现这个目标。

一个人获得他人真正的尊重，来源于其提供的 **价值**。

**价值 = 你能解决别人无法解决的问题，或拥有别人没有的资源。**

它可以体现在多方面：
*   特殊技能（如编程、烹饪）
*   独特资源（如专业知识、人脉）
*   优秀品质（如坚韧、诚信）

在人际交往中，你需要思考的是：**我能为这段关系或对方带来什么独特的价值？** 而不是“我该如何讨好他”或“他是否伤了我的自尊”。

将任何“遭受冷遇后仍不放弃”的行为都简单定义为“跪舔”，这是一种 **“心理投射”**。

**心理投射**：当一个人对自己有某种看法（通常是负面的）时，会倾向于认为别人也这样看待自己。自卑的人常觉得别人看不起自己，就是这个原理。

因此，那些对此反应过激的人，很可能将自己过去“跪舔”时的卑微感或受挫感，投射到了他人的行为上。健康的做法是像优秀的销售一样：**目标明确，聚焦于解决问题和提供价值，过程中保持韧性，不被一时的拒绝或负面情绪带偏。**

---

**本节课总结**：我们一起学习了“跪舔”心理的四种典型思维模式，明白了它与“过激”反应是同根同源的两种错误压力应对方式，根源在于 **压力管理失当** 和过度关注 **情绪** 而非 **问题**。最后，我们指出了改变的方向：**聚焦目标，停止讨好或攻击，转而致力于提升自我价值和解决问题的能力**。这才是赢得尊重和建立健康关系的根本。
