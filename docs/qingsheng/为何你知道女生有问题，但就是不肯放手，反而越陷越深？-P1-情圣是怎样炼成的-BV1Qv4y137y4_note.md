## 为何难以放手：沉没成本陷阱
上一节我们列出了种种糟糕感受，本节我们深入探讨其中最核心的一点：为何明知是坑，却跳不出来。

这可以用经济学中的“沉没成本”效应来解释。想象一下：

1.  对方借了你5000元（你投入了成本）。
2.  你去讨债，对方说：“我再借1000路费去外地赚钱，回来连本带利还你6000。”
3.  你计算后，觉得再投1000可能拿回6000，于是又借了（追加投入）。
4.  对方再次失信，你借出的钱变成了6000（总成本增加）。
5.  为了拿回这6000，你可能被迫继续相信对方新的理由，继续投入，直到血本无归。

在感情中，你投入的是感情、信任、时间。每一次被伤害后的不甘心，以及对方“再给我一次机会”的安抚，都诱使你追加“投资”，结果越陷越深，无法止损。

**核心代码逻辑模拟**：
```python
# 模拟沉没成本陷阱
total_investment = 0  # 总投入
while still_in_relationship:  # 当仍在关系中
    if feel_hurt:  # 如果感到受伤
        # 不甘心，并相信对方会改变
        print("再给一次机会，也许能挽回所有投入...")
        additional_investment = input("请选择追加投入（感情/金钱/时间）: ")
        total_investment += additional_investment  # 总投入增加
    else:
        break  # 清醒，跳出循环
print(f"最终总损失: {total_investment}")
```
