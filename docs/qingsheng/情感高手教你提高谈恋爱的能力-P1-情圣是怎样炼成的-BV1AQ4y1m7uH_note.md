## 直面困难：能力提升的唯一途径 💪

上一节我们介绍了课程的整体目标，本节中我们来看看能力提升的本质。

只有当一个人真正去思考、解决并克服一个困难时，能力才会提高。从来不存在所谓的“内化”技巧这回事。

技巧对人最大的危害，就是形成“技巧性思维”。这种思维会内化，但内化成的不是能力，而是一种投机取巧的心态。

以下是技巧性思维带来的具体危害：
*   促使人做任何事都想走捷径，甚至玩游戏都想开外挂。
*   养成好逸恶劳、没有底线的生存方式。
*   让弱者更弱，导致思维僵化，无法真正解决问题。

那些教授恋爱技巧的导师，其实际生活能力往往非常低下，这正是因为他们深陷这种技巧性思维。

## 强者与弱者的思维模式分野 ⚖️

上一节我们剖析了技巧性思维的弊端，本节中我们来对比一下强者与弱者的根本区别。

强者和弱者的区别，往往源于不同的思维模式所铸就的行为模式。

以下是两种思维模式的具体表现：
*   **弱者思维**：核心是“我想要”。表现为萎靡不振、内心阴暗，总在寻找捷径和套路。
*   **强者思维**：核心是“我一定要”。表现为走路带风、光明磊落，内心简单诚恳。

学习恋爱技巧的人，内心往往充满戾气，容易陷入矛盾，难以解决问题，因而缺乏自信，难以获得他人的信赖与支持。他们缺乏主见，在人际交往中永远处于被动地位。

## 自尊：自信与魅力的基石 👑

我们了解了两种思维模式，那么是什么导致了这种思维差异呢？本节将揭示其根源。

思维模式不同的根本原因，在于自尊发展水平的高低。说得直接些，许多学习恋爱技巧的人缺乏羞耻心。

例如，有人以和女孩子出去吃饭让女方买单为荣并四处炫耀，这完全是不知羞耻、没有自尊心的表现。

没有自尊，在交往中就会走向两个极端：要么跪舔，要么装逼。这些方向永远无法带来真正的自信。

**真正的自信公式**：**强自尊 → 真自信**。一个人自尊心强，才会相信自己能凭本事搞定事情，而不是依赖技巧。

当你凭借自身能力成功时，你会发现根本不需要技巧，本能就足够了。因此，提升恋爱能力的第一步，是提升你的自尊与自信。

## 建立自尊的三项核心修炼 🛠️

既然自尊如此重要，我们该如何建立它呢？本节将介绍三个具体方法。

### 1. 与困难正面交锋

建立自尊的第一步，是绝不能绕过困难。学习技巧的本质就是想绕开困难。你必须拥有与问题“刚正面”的觉悟。

这样的人在生活中会受欢迎，也能获得信任。无论是老板、朋友还是女孩子，都会欣赏和需要关键时刻能够站出来的人。

### 2. 做好自身定位

当你自身定位不清时，思维就会混乱，容易滑向技巧性思维。

清晰的定位是：认识到自己当前的不足（如经济、能力），然后专注于拼事业、培养可靠品质、建立信用。这样才能站稳脚跟，进入更好的圈子，结识真正与你匹配的人。

### 3. 学会关心他人，提高情商

高情商意味着懂得照顾他人的感受。恋爱初期，你应该展示的是礼貌、上进、温暖、真诚和友善，让对方感到踏实和舒服。

```python
# 错误做法：只关注自身欲望
if 刚认识几天:
    目标 = “牵手、亲吻、开房”
    结果 = 被对方讨厌

# 正确做法：关注对方感受
if 建立关系初期:
    展示 = [“礼貌”, “上进心”, “关心”, “真诚”]
    目标 = “让对方感到舒服和踏实”
    结果 = 建立好感与信任
```
会聊天就等于情商高，而情商高的核心在于懂得关心别人。只关注自身欲望，说话永远不会让人舒服。
