# 课程名称：如何消除社会评价对与女生交往的影响？ - 第1课 🧠

## 概述
在本节课中，我们将要学习一个在人际交往，特别是与女生交往中普遍存在且影响巨大的心理现象——**社会评价**。我们将深入探讨它的定义、表现、来源，并分析为什么单纯“克服”或“忽视”它的想法往往是无效的。理解其本质，是管理它的第一步。

## 什么是社会评价？
上一节我们概述了课程主题，本节中我们来看看社会评价的具体含义。

社会评价不仅仅指外界他人对你的看法。它更关键的部分，是你头脑中一个平时不易察觉的、对自身的评价体系。

你是否曾有过这种感觉：头脑中突然有个声音对你说“她不会喜欢你的”、“你不行”、“你搞不定”、“万一被拒绝怎么办”？这个声音会自动冒出来。

平时你可能感觉不到它的存在，但一旦面临压力、挑战或社交场合时，这个内在的社会评价体系就会出来干扰你。

## 社会评价如何影响你？
理解了概念后，我们来看看它是如何在具体场景中发挥作用的。

这个评价体系往往会让你做事不顺利。你会发现，不管是追求女生还是做其他事情，当这个体系被激活时，你的智商仿佛会下降。

例如，本来你很正常，但压力一来，脑子就乱了。设想一个大美女站在你面前，你想和她聊天，但不可能直接说“我们去开房吧”。

这时你会拼命想该聊什么，同时又不敢行动，害怕被拒绝。你会不断思索“搭讪有没有更好的招数”、“今天看到美女该不该去”。

它会限制你的行为和思维。或者面临巨大压力时，比如看欧洲杯决赛点球大战，越往后踢的球员压力越大。

这压力从何而来？无非是踢个球，关系到胜负。但那么多双眼睛盯着你，这就是社会评价的压力。如果干砸了，你就要负巨大责任，成为“罪人”。

心理学家荣格提出，人有个体潜意识和群体潜意识。社会评价的压力，实际上就来源于我们的**群体潜意识**。

以下是群体潜意识的一些例子：
*   在中国的文化习俗中，生下来就知道上门提亲要给彩礼，这没人专门教。
*   跟女生结婚要备好礼物去提亲，这也没人教。
*   女婿去岳父岳母家，要帮忙干活，这同样没人教。

这是一种集体潜意识。包括以前的风气是男生要主动追女生，这也没人教。所以，踢点球的压力就来自于你大脑里的集体潜意识，即社会评价体系。

它会给你巨大压力。此外，上台演讲、当众表演时，众目睽睽之下，也都是巨大的社会评价压力。

## 社会评价的负面影响
这种社会评价压力，对人没有太多正向意义。它会影响你做事的状态。

例如，2011年乒乓球男团半决赛，刘国正顶住巨大压力，连扳七个赛点翻盘。当时很多观众紧张到不敢看电视。

我讲这个是想说明，社会评价常常会冒出来，影响你的行为，让你状态失常，干扰你正常的心智。

很多时候与女生交往也是如此：面对你不在乎的、觉得长相一般的女生，你聊天很随意。但一旦你对某个女生上心了，这个社会评价就开始监控你的言行。

你会不断想：“这话到底对不对？”“这样说行不行？”“那样接触好不好？”只要一上心，就开始不知所措。

而且，这个东西好像无法控制。它一出现，你整个人就好像“吃错了药”一样。所以很多时候，女生会觉得追自己的男生“没几个正常的”，就像“丧尸”一样感觉不对劲。

## 关于“克服方法”的误区
讲到这里，可能有人会问：“有克服的办法吗？”我其实很反感这种问法。

为什么反感？如果一件事，你都不了解它的原因和原委，我给你一个“克服办法”，你会照着做吗？有人说“只能接受吧”、“把注意力放在事上”。

我们接着来看，社会评价到底是怎么回事。首先，什么是社会评价？我刚刚讲了，社会评价指的是你脑子里“别人会怎么看我”的想法。

这在追女孩时最容易出现。社会评价干扰严重的同学会出现一个症状：特别想知道女生对自己到底是什么意思。

甚至会因此拼命去解读：“她到底喜不喜欢我？”你这个社会评价意识越严重，就越会出现这种情况。

你会发现，以前追女生时，这个症状越严重，女生就越追不上。你越想知道女生怎么看你，就越追不上；你越无所谓，越放松，越不把她当回事，反而越能追上。

于是很多人提出一个错误想法：“对女生就要不在乎。”所以很多人试图用意识控制自己，强行不在乎。

结果呢？结果是不是让自己更难受？更不知道如何与女生交往了？因为这种情绪本身就在，压制它有什么用？

压制情绪、想消灭情绪，只会让情绪越来越僵硬。结果不仅是做不到，更会让你的情绪和情感变形。

与人交往时，你会感觉更加不对劲，产生一种僵硬感，甚至故意“装逼”，故意对女生表现冷淡。明明很喜欢，却觉得“不能太热情”、“不能表现得太在乎”。

否则会怎么样？很多人会说五个字：“**暴露需求感**”。我一直觉得这五个字特别好笑。

什么叫暴露需求感？你有需求感就有啊，暴不暴露有什么关系？它是明里暗里都会影响你。就像上台演讲，不紧张的人始终松弛，发挥自然，不会太在意观众看法。

但如果你心里紧张，很在意别人怎么看，并努力想表现好，往往表现得更差。这时别人说“你不要在意观众看法”，能行吗？

这不是一个你想适应就能适应的过程。好比一个人只有50斤力气，脑子里想“我要扛200斤的东西”，有用吗？

这叫什么“认真就输了”？不，这是人心理抗压性差的表现，是你的身体没有适应这种压力。

说得简单点，就像你只有50斤力气，却要去扛200斤的米。跟女生接触也一样，你平时接触女生少，自己没有主动争取过，面对压力时，就特别容易代偿女生对你的看法。

你心理上没有适应，却强行告诉自己“我要去适应”、“我要不在乎”，这就相当于用50斤力气去扛200斤的米，会被压垮。

所以很多人会出现典型的感性认知：“认真就输了”。要我说，认真才会赢。做什么事不认真，怎么可能赢？当然要认真。

而且，“放松”、“随意”不是你想做到就能做到的。“别人会怎么看我”这个想法一旦激活，你的行为就会受到意识干扰。

比如你走进一个地方，会想“大家怎么看我？”“会不会不喜欢我？”社交恐惧症或社交焦虑的人，特别有这种感觉，好像走出去就被审判。

包括有的人当女生的“舔狗”，跟女生交往时都有种被审判的感觉：“我这事做对了吗？她会喜欢吗？做错了吗？她会讨厌吗？”

一旦感觉被女生审判，就会变成那种“又色又恐女”的状态：特别渴望女生，又特别害怕跟女生交往。其实就是社会评价在干扰你。

主要表现是感到不自在、不知所措，就像做事时旁边有人盯着，一旦出错，这人马上就会训斥你。在这种状态下，你做什么事都不可能做好。

## 减少社会评价的关键：停止评价他人
有人问：“我评价女生，女生抱怨我不尊重，怎么办？”说实话，女生没拉黑你已经算客气了。这叫基本情商。

你和一个不熟的人第一次见面，对方马上评价你“你是不是脑子有病？”，你怎么想？你跟女生聊天时评价她，你爽吗？

这属于基本的社交礼仪问题。这样的同学可能没怎么出过门，没在社会上跟人打过交道，不知道什么叫礼貌。这可能需要你在社会上“挨过一些毒打”才知道问题在哪。

这不是某个具体办法能解决的。例如，有女生在网上发不雅录音，你说她“Low”。那说明你是个“键盘侠”。

难道你不是一边说她Low，一边又想得到她吗？键盘侠都这样，一边骂别人，一边是因为自己得不到。

她Low关你什么事？社会上还有人杀人放火呢，关你什么事？你非要去凑一句嘴才舒服吗？何必呢？人家干什么，是他的事，自有别人管。你骂了他，心里就舒服了？

很简单一个道理：你们在网上当键盘侠骂了别人，心里舒服；你去跟别人谈恋爱，骂了对方，心里会舒服吗？

还有很多失足妇女，日本还有很多老师，你怎么不去骂？我不明白你们干这事到底是为了什么。

我最希望所有听课的同学，不要成为网络舆论中的一份子。不要跟那些人一起瞎起哄，一看什么东西就要把情绪吐出来。

我告诉大家为什么不要骂人、不要评价别人：因为这除了增加你的戾气之外，没有任何意义。你在网上喜欢攻击、评价、骂别人，在生活中跟身边的人一定也会出现这种情况。

因为这是你习惯下来的。当你需要跟新人相处交往时，你也很容易延续这种习惯。人情商低的一个重要表现（尤其是男性）就是“嘴特别冲”。

一上来语气特别重的人，你是不是一上来就觉得这人是个傻逼？想想看。

我为什么让大家不要随便在网上评价别人、不要随便释放戾气？因为在网络环境中，你可以这样做的原因很简单：**你不用负责任**。

你在网上积累这种习惯多了，躲在电脑屏幕后说话冲习惯了，就会把这种习惯带到现实交流中。

有没有当过键盘侠的同学有这种感觉？最典型的人比如乐嘉，线上线下都很冲，结果毁了自己的事业和前途。

所以，生活中不要随便去评价别人。别人干什么，那是他的事。别人喜欢怎样，那是他的生活。

如果我觉得你的生活不健康，我顶多不跟你玩，但不会评价你的生活。包括朋友沾染坏习惯，我顶多劝两句，如果他希望我帮，我可以负责。但我说了你不听，我也不想说太多，因为这是你的事。

我们在生活中应多夸别人，少贬低别人，少对别人挑三拣四。这样你和别人都舒服，你也愉快，心里就不会有那么多刺。

人为什么喜欢评价别人？很简单，因为过于自我，认为“我是对的”，“我认为你不对”、“我认为你这种行为不好”。

## 核心原理：评价他人与在意他人评价的关联
我们来看一个核心原理：一旦这种社会评价体系启动，当你对别人的评价越少时，你注意，你自然也不太会在意别人怎么评价你。

明白什么意思吗？你不怎么去评价、议论别人时，你也不会太在意别人怎么评价和议论你。知道为什么吗？

当你去评价别人时，要注意，这是一种**攻击性**，或者说是一种**价值判断**。价值判断和正常判断的区别在于：“我觉得你这样很傻逼”、“我觉得你不行”、“我觉得你不对”、“我觉得你脑子有病”。

这都叫价值判断。当你习惯性地运用价值判断时，你就会在意别人对你的价值判断。你就进入了这个价值判断的框架和体系。

明白什么意思吗？你经常习惯性地进入这个价值判断体系，你就很容易：当别人对你态度不好时，你想“他是不是不喜欢我”；当别人不理你时，你想“他是不是对我不感兴趣”。

价值判断往往后面带着什么？我们前两天讲的公开课《事实与观点论》还记得吗？当一个人价值判断越多时，他的观点就越多。

**观点越多的人，掌握的事实越少**。观点越多、越喜欢价值判断别人的人，戾气就越重。戾气越重的人，对别人对他的评价和看法就越敏感。

就这样，他把价值判断这个东西无限放大了。所以我们要怎么做？生活中少对人做价值判断，要多看事实，多接纳别人，多承认别人现在的样子。

例如，刚刚那个在网上发不雅录音的女生，如果让我来看，我认识她且关系还行，我会问她：“是不是最近有点无聊、空虚了？要不要我给你介绍个男朋友？”

我顶多这样。因为我发现别人的行为时，不会去批评，而是想了解原因。很多时候一个人做一件事，不是没有原因的。

我们要知道，看别人不对的事情时，要注意什么叫**同理心**（下节课会讲）。就是怎么减少与人之间的矛盾和对立情绪。

什么叫同理心？就是我们去看到别人很多不合道理、看似无理取闹、神经病的地方时，要理解：别人做这件事时，有一套自洽的逻辑。

我的一个主要观点是：我相信人性是相通的，每个人都可以做好，每个人都是有智慧的。基于这一点，我不认为任何人本质上是愚蠢或邪恶的。我认为每个人本质上都是善良的、向好的。

那么什么叫同理心？就是说你能够用别人的道理去理解别人。人很多时候的非理性行为，只是在外部看来不合理，但在自己看来非常合理。

神经经济学里有个学术用语叫“效用”。一个人去赌博，他在输钱，但他的内在体验是有效用的：他体验了赌博过程中的愉悦感、刺激感，分泌了多巴胺。虽然他输钱了，但体验很好，所以他沉迷于此。

## 总结
本节课中，我们一起深入探讨了“社会评价”这一概念。我们了解到它不仅仅来自外界，更根植于我们内心的群体潜意识，表现为“别人会怎么看我”的想法，并在压力下干扰我们的行为和情绪。我们分析了试图强行“克服”或“不在乎”往往适得其反，因为它是一种需要心理适应而非意识压制的能力。最后，我们找到了一个关键突破口：**减少对他人进行价值判断，是减轻自身被社会评价所困的有效途径**。当你停止随意评价他人，你便自然跳出了那个在意他人评价的框架。下节课，我们将进一步学习如何运用“同理心”来更好地与人交往。
