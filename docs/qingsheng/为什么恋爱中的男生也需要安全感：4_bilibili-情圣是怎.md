# 为什么恋爱中的男生也需要安全感：4_bilibili-情圣是怎

在本节课中，我们将要学习恋爱中一个常被忽视的议题：男生的安全感需求。我们将探讨为何过度依赖外部反馈会导致情感问题，并分析现代人恋爱困难的核心原因。

## 过度依赖反馈是情感问题的根源

你们经常出现的情况是，没有获得反馈，就开始自我怀疑。这是男生容易出现的问题。女生出现的问题则是，你的反馈一直是负面的，但你不知悔改。这两种人都特别容易找不到对象。

所以，现在大多数人找不到对象、恋爱关系不稳定、婚姻不长久，主要原因就是太跟着反馈走了。这种反馈越及时，你就越容易对这件事上瘾，越容易深陷其中，而对于自身的优势、自己真正想要什么，并不清楚，也没有明确的规划和想法。这造成了情感上的迷茫和失能。这就是目前大多数人的情况：太需要反馈，即所谓的“跟着感觉走”。

有反馈就是有感觉，没反馈就是没感觉。

## 导致恋爱困难的五大原因

以下是导致个人在恋爱中陷入困境的几个常见原因：

1.  **圈子太小，社交狭隘。**
2.  **不懂自身优势**。例如，我本来是一个不爱说话的人，但为了找对象，我非要强迫自己变得会说话。这就叫不懂自身优势，非要去做自己不擅长的事，导致“邯郸学步”。
3.  **对于自己想要的不懂主动去争取**。男生见到喜欢的人不敢上前；女生则是不懂得主动进入想要的群体，去展示自己的优势和美丽。
4.  **对于自己的生活、人生没有想法和追求**。这种人很容易凡事都做错。
5.  **从过去的阴影中走不出来**（尤其是女孩子）。例如曾经受过伤、被渣男伤害过，从此变得很自卑、没有自信，不懂往前看。如果人处于这种状态，就会失去创造更多可能性的机会。

## 心态课程如何帮助你

上一节我们分析了问题，本节中我们来看看解决方案。我们推出的付费课程旨在帮助大家（男女通用）重建健康的恋爱心态。

### 1. 重铸自信心与自我定位

课程将教你建立自信的逻辑和需要努力的方向，以此首先明确你本身是个什么样的人。然后，教你如何在你的角色、身份和状态下，去提升自己。

用更文艺的话说：**每个人都是一颗星星，只不过闪着不同的光**。你是蓝色的，就闪耀蓝光；你是白色的，就闪耀白光；你是彩色的，就闪耀彩光。关键首先在于，你要清楚自己的定位。如果你本身不爱说话，就没必要强迫自己变得多么健谈。你只需要把“闷骚”做到极致。不一定闷骚就不会有人喜欢。电视上的偶像往往都不苟言笑。只有做好自己定位范围内的事，自信心自然就有了。

### 2. 训练应对焦虑的能力

本部分将训练你如何应对由于不安全感和需求感产生的焦虑，让你敢于打破内心的障碍和束缚。

这种不安全感和需求感产生的原因，正是大家太想要反馈。仔细想想，人何时会产生不安全感？何时会产生需求感？**正是当做一件事不知道会不会有反馈，或者长期没有反馈（没有结果）的时候**。例如，跟一个女孩天天聊天她总不理我；追一个女孩，她始终对我不冷不热，时间长了就受不了。

如果说你做每件事都需要这种即时反馈才能支撑下去，你的自信心如何建立？我们看到，越急着要结果的人，越不可能出结果。他需要这个反馈来让自己能继续下去，否则就没有信心。**一个有自信心的人，即使没有反馈，也能持续坚持下去，因为他相信自己最终能做到**。心态不同，结果就不同。

### 3. 培养主观能动性

如果你心态上特别渴求结果和反馈，这说明你缺乏主观能动性。

什么意思？比如，我做一件事想要得到正面反馈，但现在没等到，心里就难受。或者，“我跟女孩子没话聊，怎么追到她？她都不理我”。这种状态能追到女孩吗？这叫典型的没有主观能动性。

**主观能动性，就是自己主动创造反馈的能力**。她现在不理我，但我可以想办法让她理我：做一个她的表情包逗她笑、每天发一个她可能喜欢的段子、找她喜欢的歌分享……你有了主动创造反馈的能力，才会逐渐对自己有信心。否则，你永远心态不好，只会问“怎么办”，自己不去想“怎么办”。

### 4. 培养勇敢、诚实、可靠的品质

这不仅男生需要，女生也需要。很多时候，你有什么样的品质，就能吸引什么样的人。

**你身上的优点越多，那些跟你一样有优点的人就会向你靠拢。你身上的缺点越多，那些跟你一样有缺点的人，以及那些想利用你、伤害你的人也会向你靠拢。** 所以弱者生活中往往充满痛苦。

勇敢、诚实、可靠永远是吸引人的品质。尤其是对于长期关系而言。男性勇敢能吸引女性；女性勇敢不一定直接吸引男性，但这些品质有永远比没有好。这也是许多来学习恋爱技巧的男性所缺乏的性格优点。

在交往过程中，如果你要让一段感情开花结果，你身上自然需要培养出责任感、诚实、可靠等品质。这不仅是伴侣需要的，也符合人类社会基因选择的“程序”。看整个社会的大趋势就很清楚：女性通常喜欢勇敢、上进、有事业心、能力强的男性；男性通常喜欢温柔、贤惠、漂亮的女性。个性虽有差异，但大体离不开这种选择。

今天给大家讲这个课程，是希望大家报名参加心态建设课程。原因很简单：你们会发现，自己在与异性交往中经常做出一些“不正常”的事情。其实很多时候，在追女孩或与男生交往的过程中，你们正常一点，很多情况都能搞定。最关键的问题是，你们因为心态出了问题，才在交往时表现“不正常”，导致搞不定。
