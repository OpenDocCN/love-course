# 课程名称：情圣是怎样炼成的 - 第1课：为何追求“确定性”让你难以脱单？🤔

在本节课中，我们将要学习一个核心概念：**追求“确定性”是阻碍你成功追求异性的根本原因**。我们将通过分析“搭讪”这个具体场景，对比两种不同的思维方式，并深入探讨追求确定性所带来的负面影响。

## 两种不同的思维方式

上一节我们提到了课程的核心主题，本节中我们来看看具体表现。不会追女孩的人和会追女孩的人，在思考方式上存在根本差异。

### 追求“确定性”的思维

以下是追求“确定性”思维模式的特点：
*   他们行动前总在思考“万一被拒绝怎么办？”。
*   他们热衷于寻找“必胜的话术和方法”。
*   他们的核心诉求是：**确保自己不会被拒绝**，并**获得女生好的反馈**。
*   他们因为害怕负面感受（如尴尬、挫败）而不敢行动。

### 追求“可能性”的思维

与此相反，会追女孩的人则采用“追求可能性”的思维。
*   他们明白一个基本道理：**只要尝试就有机会，不尝试则一点机会都没有**。
*   他们不会预先设计全套流程，而是先行动，再随机应变。
*   他们的关键动作是：先打招呼或开启一个简单的互动（例如：“你一个人在这吗？”、“方便我坐一下吗？”）。
*   然后根据现场情况，依靠自身的社交能力和情商自然展开话题。

## 真实场景中的“可能性”思维

理解了两种思维的区别后，我们来看看追求“可能性”的思维在现实中是如何运作的。以下是生活中常见的、即兴的搭讪方式：
*   **借口法**：临时编造一个理由，如“我手机没电了，能借你电话用一下吗？”，之后可能再幽默地承认“哎，我手机怎么又有电了，其实我就是想认识你”。
*   **游戏法**：与朋友打赌，输的人去邀请女生，例如：“你好，我们打赌输了，规则是我要请你吃个冰淇淋”。
*   **直接法**：没有其他办法时，采用最朴实的方式，如直接上前说：“美女，加个微信”。
*   **情景法**：利用周围环境自然切入，例如在餐厅坐到独自一人的女生旁边，问“你这吃的是啥？”，或问“你是一个人在旅行吗？要不和我们组个团一起玩？”。

这些方式的核心在于**行动优先**，方式灵活、随机，甚至带点幽默，而不是依赖预设的、刻意的套路。

## 为何追求“确定性”会导致失败？

我们已经看到了两种思维在行动上的差异，那么，追求“确定性”具体是如何导致你在感情中失败的？以下是五个关键原因：

1.  **情绪被他人掌控**：当你强烈希望获得女生好感时，你的情绪就会轻易被对方的态度和反馈所左右。例如，对方不回信息你就会心慌。
2.  **心态随反馈波动**：女生对你好，你就开心；对你冷淡，你就沮丧。你的快乐建立在对方的行为上。
3.  **不敢主动作为**：因为不确定自己的行为会引发何种反馈，你害怕搞砸、害怕留下坏印象，所以不敢做出更多主动行为，只能躲在所谓的“技巧”和“流程”背后。
4.  **陷入焦虑循环**：由于你不主动，又不清楚对方的想法，你会持续处于焦虑状态，不断猜测“她到底对我什么意思？”。
5.  **忽略对方感受**：你过度关注“能否在一起”这个结果，而忽略了女生真实的感受和需求。例如，你可能追了别人很久，却连她喜欢什么都不知道。

这种追求确定性的本质，是让你成为了**女生态度的奴隶**。在关系中，你处于从属地位，节奏由对方掌控。而健康的互动通常需要男生主动推进节奏。当关键节点需要你“顶上去”时，你却因为害怕失败而退缩，女生便会觉得你不可靠、无法提供安全感，最终以“对你没感觉”为由结束关系。

## 核心总结

本节课中我们一起学习了：
*   **核心问题**：难以脱单的根源在于**追求“确定性”**的思维模式。
*   **思维对比**：“追求确定性”的人害怕失败，寻找套路；**“追求可能性”**的人勇于尝试，随机应变。
*   **失败机制**：追求确定性会导致你情绪被控、不敢主动、焦虑内耗，并最终忽略对方感受，从而无法建立真实的情感连接。
*   **行动关键**：**先行动，后调整**。感情中没有百分之百成功的剧本，增加互动和暴露的可能性，远比追求一个确定的结果更重要。

记住公式：**成功的机会 ≈ 大胆的尝试 × 灵活的应变**，而 **0 × 任何套路 = 0**。
