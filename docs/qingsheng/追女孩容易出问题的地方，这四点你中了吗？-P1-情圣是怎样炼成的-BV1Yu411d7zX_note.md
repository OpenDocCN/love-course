在本节课中，我们将探讨追求异性时常见的四种错误思维方式。这些思维模式是许多人在情感道路上屡屡受挫的根本原因。理解并改变它们，是建立健康、成功关系的第一步。

---

## 1. 思维方式固化，缺乏成长与责任承担

更出问题的地方是你们的思维方式。学习时不应只关注技巧，那些没有用。真正需要改变的是思维方式。

这种思维方式的转变，实际上源于个人在生活中承担的责任变化。随着你承担的责任增多，解决问题的难度增大，需要考虑的因素也变多，这时你的思维方式才会不同。

例如，如果你只为自己生活，就只需考虑自己。一个明确的感受是，一个人生活时间长了，情商和社交技能都会倒退。

当你开始接触他人，处理复杂问题时，他人的利益和感受就会进入你的思维框架。要处理好这些事情，就必须思考这些问题。

人失败次数多了，有时会不知原因。但如果你想做好一件事，就会慢慢理解许多原来无法理解的东西。这是第一个问题。

---

## 2. 只关注自身欲望，忽视客观事实与规律

第二个问题是只关注自己的欲望，不关注客观事实和规律。你只想着“我能不能得到”。

很多人不是这样思考的。例如，当时我对还是女友的妻子说：“你要是不喜欢我千万别答应。考虑好了再说。我不希望成为一个跟你凑合的人。我知道我爱你，但希望你也真的爱我，我们才在一起，这样才有意思。”

这与很多人对女生的思维方式完全不同。因为我尊重事实和规律：**两个人只有相爱才能过得好、舒服、幸福**。我要的不是“你”，我要的是“幸福”。如果和你在一起不幸福，得到你又有什么用？

只关注欲望，就会忽视客观条件和规律。你想省事，觉得“到手就好”，别的不管。但这种省事的想法，即使最后得到了，也会害了你。

有多少人得到了，最后关系破裂，更难受？像分手挽回这种事，即使当时挽回了，如果问题没解决，还是会争吵、打闹，损失更大。能否提前规避这些问题？

---

## 3. 出发点错误：只问“能否得到”，不问“是否做好”

第三个问题是只以自己“能不能得到”为出发点考虑问题，而不关注“你有没有做好”。

例如，和女生交往时，有没有想过：**我话有没有说好？我事有没有做好？**

有句话叫“凡人为果，菩萨为因”。别人为什么拒绝你、考验你？这是“恶果”。恶果的原因是什么？是你没做好。如果你事事都做到位、做好了，别人至于拒绝或讨厌你吗？

我不谈接受或拒绝，只说喜欢和讨厌，这是最简单的。**喜欢是不是因为你把事情做好了？讨厌是不是因为你做事情没做好？** 道理很简单，一是一，二是二。你话没说好，别人肯定讨厌你；你话说好了，别人可能喜欢你。

很多人不明白这个道理，总想着“我能不能得到”，却不考虑“我能不能把事情做好、把话说好”。

---

## 4. 情绪化严重，只想结果不要过程

第四个问题是极为容易情绪化，只想获得结果，不想要过程。

很多人的思维方式不正是这样吗？他们被情绪驱动，只渴望立刻拥有那个“成功”或“得到”的瞬间，却厌恶、逃避或无法承受达成目标所必需的漫长过程、努力和挫折。这种心态往往导致冲动决策、半途而废或在过程中制造更多问题。

---

## 总结

本节课中，我们一起学习了在追求异性时容易出现的四种核心思维误区：
1.  **思维方式固化**，缺乏因责任承担而带来的成长。
2.  **只关注自身欲望**，忽视**相爱是幸福基础**这一客观规律。
3.  **出发点错误**，只问“能否得到”，不问“是否做好”（`if (things_done_well) { like++; } else { dislike++; }`）。
4.  **严重情绪化**，只要结果，不要过程。

改变这些思维方式，是从根本上提升情感关系处理能力的关键。下一节，我们将探讨如何构建更健康、更有效的互动思维模式。
