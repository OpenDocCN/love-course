# 情商课程 01：情商与面子的学问 👥

在本节课中，我们将要学习情商的核心本质——面子的学问。我们将探讨面子在社交中的重要性、它如何作为一种“社交货币”运作，以及如何通过理解面子来提升我们的情商，从而在人际交往中更加得体、从容。

## 什么是情商？🤔

情商，本质上是一门关于面子的学问。它研究的是如何恰当地给予他人面子，同时保持自身的体面。这门学问以前可能很少有人系统地讲解过。

面子可以被理解为一种**社交货币**。在社交中，面子大意味着社交货币价值高。公式可以简单表示为：
**社交地位 ≈ 面子（社交货币）的价值**

当你的面子足够大时，即使你暂时没有金钱，你依然拥有东山再起的资本。反之，一个人可能很有钱，但若没有面子，也未必能得到他人的尊重。例如，某些“土豪”虽然富有，却可能被其他阶层的人看不起。

所以，情商说白了，就是这门关于面子的学问。

## 杜月笙的“三碗面”🍜

上一节我们介绍了面子的概念，本节中我们来看看一个经典比喻。杜月笙曾说过，人一生有三碗面最难吃：体面、场面和情面。这三碗面若处理不好，就容易在社交中出问题。

杜月笙本人常被评价为“会做人”，情商很高。他的一个核心特点是思维严谨，善于总结方法。这告诉我们，提升情商并非玄学，而是可以通过观察、思考和总结具体方法来达成的。

虽然没有人是万能的，但若我们在生活中多动脑筋、积累经验、掌握方法，就能有效避免许多不必要的尴尬。这能帮助我们在与人交往时，不至于在被怼、被挤兑或嘲讽时，只能陷入情绪化的对抗——而这种时刻，恰恰最考验一个人的情商。

## 情商的核心作用 🎯

那么，情商究竟有什么用呢？它的最大用处，是**让他人高看你一眼**。

无论是在追求异性还是日常社交中，道理都是相通的。情商高的人，容易让人竖起大拇指，觉得此人“厉害”、“会处事”。情商不足，则可能让人感觉“不过如此”，原有的好印象也会打折扣。

我们可以通过对比公众人物来理解这一点。例如，孙杨和姚明在面对类似事件时，因情商表现不同，公众对他们的评价和认可度也产生了差异。情商不足，公众形象就容易受损，让人觉得“有勇无谋”或“不成熟”。

所以，情商的作用在于建立和维护积极的个人形象，获得他人的认可与尊重。

## 如何判断自己有无面子？📊

如何在实践中判断自己在他人心中是否有面子呢？方法很简单：**看你说话管不管用**。

这是一个不需要过多纠结的问题。在日常生活中，当你需要托人帮忙或提出某个建议时，观察是否有人积极响应，就能清楚地感知到自己面子的大小。

面子是一种无形资产，它直接影响了你的社交效能和个人影响力。
