# 课程名称：防渣指南 - 第1课：如何避免遇到渣男？🧠

在本节课中，我们将学习在男女交往中如何避免遇到“渣男”或“渣女”。我们将解析六个核心的心理逻辑与行为准则，帮助你建立健康的交往观念和有效的识别能力。

## 概述：无法躲避的“吸引力法则”

你要说怎么去防怎么去躲，很明确地告诉大家躲不掉。你是什么样的人，你就会遇到什么样的人。你本身是个渣，你就会遇到段位比你更高的渣。

这个逻辑可以用一个简单的**公式**表示：**你会遇到的人 ≈ 你自身的特质**。这意味着，提升自我认知和价值观是根本的防御策略。

## 第一招：警惕表面功夫

上一节我们介绍了“吸引力法则”，本节中我们来看看具体如何应用。第一个准则适用于与任何人相处，包括找工作或处理其他事务。

不要光看别人表面功夫。不要看这个公司做的有多大，写字楼多么高级，公司里面有多少人。不要去看这个东西。

以下是具体注意事项：

*   看人时，不要只看他穿什么名牌、开什么好车、有什么头衔，就认为他厉害。
*   越是看起来“牛逼”的人，越需要警惕其内在人品。

## 第二招：人品是硬性指标

仅仅识别表面信息还不够，我们需要一个更坚固的筛选标准。第二个准则，特别是对女孩子说：如果一个人的人品不过关，他对你再好、再有魅力也不行。

人品是个很硬的指标。这个硬指标过不了的话，他再有钱、再有魅力、对你再好都无意义。他现在对你有多好，未来就可能把你伤得有多深。

## 第三招：主动摸清对方底细

确立了“人品”这个核心标准后，我们如何验证它呢？第三个方法就是：把对方的底细尽量摸清楚，不要他说什么你都信。

很多人容易遇到渣，原因就是“傻”。他说什么你信什么，不去摸清对方的底细，自然容易遇到渣。

## 第四招：保持独立与主见

了解对方的同时，我们自身也需要具备防御姿态。第四个准则是：要有自己的主见和独立性，不要过于依赖对方。

如果你事事都依赖对方，无论大事小事，就很容易遇到渣。因为渣发现你太依赖他之后，他一定会利用你对他的依赖，持续让你为他付出，而不是他来为你付出。

## 第五招：警惕狂热追求者

在交往初期，有一种常见的高风险信号需要特别注意。第五个准则是：要警惕一开始就跟你表白并且狂热追求的人，这种人80%是渣。

## 第六招：建立“扣分”价值观防火墙

最后，也是最核心的一点：什么样的人不会遇到渣？答案就是懂得对“渣行为”自动扣分的人。

以下是“扣分”行为的例子：

*   他骗你，你会对他没感觉。
*   他不守信用，你会觉得这人不行。
*   他生活放纵，你会觉得这人真不咋地。

这就是价值观的问题。如果你的价值观正确，天然会对渣行为扣分，你肯定不会遇到渣男。很多女孩子不具备这道关键的防火墙，所以容易被骗。

## 课程总结与核心理念

本节课我们一起学习了识别和避免“渣”的六个具体方法。我们课程的目的，并不是把大家培养成一个渣男。

我们认为，大家可以学习渣男的某些**能力**，但不要学他们的人品。例如，目的性强、思路清晰、肯花心思、勇于挑战，这些在做事和追求目标时都是很好的特质。

但是，最本质的区别在于：**渣男的核心逻辑是自私自利，只想着自己，不想别人**。

所以，本课程教大家的是：首先，我们把有效的能力和方法教给你们；同时，我们也教你们做一个三观正、有责任心、有担当的人。这两条结合，一个男性在处理男女关系或选择对象时，就不会出格或出太大问题。

目前市面上大多数培训课程只强调“怎么去搞定”，没有强调“要怎么去对别人负责”。我们这门课程是两方面兼顾的。

**本节课中，我们一起学习了：**
1.  认识到“你会遇到什么样的人”由自身特质决定。
2.  学会不只看表面，要警惕“光环效应”。
3.  确立“人品”为不可妥协的硬性指标。
4.  主动调查和验证对方的真实情况。
5.  保持个人独立，避免过度依赖。
6.  建立正确的价值观，对不良行为自动“扣分”。
7.  区分“能力”与“人品”，学习前者，摒弃后者，最终成为一个负责任的人。
