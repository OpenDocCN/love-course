## 判断真假拒绝：关键在于“主语”🔍

上一节我们提到了面对拒绝时的困惑，本节中我们来看看如何准确判断拒绝的真假。核心在于倾听女生拒绝时使用的“主语”。

以下是判断标准：

*   **假拒绝的特征**：当女生使用“我”作为主语来表达拒绝时，例如“我觉得我们还不熟”、“我觉得我现在还不想谈恋爱”。这通常表明她的拒绝并不坚决，更多是出于对自身状态或关系进度的顾虑，而非对你个人的否定。
*   **真拒绝的特征**：当女生使用“你”作为主语来描述感受时，例如“我觉得你人很好，但我们不合适”、“我对你没那种感觉”。这明确指向了你或你们之间的关系状态，是更直接的拒绝。

**核心概念**：拒绝的“主语”是判断意图的关键信号。
*   假拒绝公式：`拒绝内容 ≈ 女生自身状态（“我”觉得/想...）`
*   真拒绝公式：`拒绝内容 ≈ 对你的评价或关系定义（“你”很好/但我们...）`

因此，被拒绝后，重点不应只放在能否再次约出，而应分析她拒绝时的具体表述。
