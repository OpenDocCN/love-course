# 课程一：为何追女孩的男生越来越少？深度解析其原因 - P1 🧐

在本节课中，我们将要学习一个社会现象背后的深层原因：为何当下主动追求女性的男性似乎在减少。我们将从社会思潮、经济发展等多个维度进行解析，力求用简单直白的语言，让初学者能够理解其中的关联。

## 概述：现象与核心思潮

首先，男生现在之所以不追女孩，一个重要原因是**极端女权思潮**的出现。

这里所说的极端女权，也被称为“田园女权”。

## 什么是“田园女权”？🤔

以下是“田园女权”的一些典型表现：
*   要求男性必须对自己好、爱自己。
*   要求男性为自己花钱、送礼物。
*   要求男性宠爱、惯着自己，且自己发脾气时对方不能发。
*   在公共场合，可能因过度敏感而指控他人（例如，误以为他人偷拍）。

其核心特征是：**不承担责任，但无限要求权利**。

这就好比请回一位伴侣，对方不断挑剔服务不到位，自身却不创造价值。这会让男性产生“我何必如此，自己生活不好吗”的想法。

## 极端女权思潮的社会根基：全面过剩 📈

上一节我们介绍了极端女权的表现，本节中我们来看看其出现的社会经济根源。极端女权思潮的出现有其社会根基，这是社会经济发展到一定阶段后产生**过剩**的必然结果。

工业发达后，首先会出现**生产过剩**。例如，社会需求一千双鞋，但工厂通过技术升级和扩大产能，能生产一万双鞋，远超出实际需求。当前中国的服装产业就是明显的例子。

生产过剩但仍有利润，资本会继续扩张，但这会加剧过剩。同时，技术升级导致**劳动力需求减少**。

于是，社会同时出现**产品过剩**和**资金过剩**。过剩的资金无法全部投入实业，便会涌入房地产市场、股市等金融市场，导致**金融过剩**。

目前，中国与美国都面临类似的过剩局面，这催生了投机行为。工业化是一个不断排斥劳动力的过程。马克思主义在解释1929-1933年大萧条时指出，工人大量失业，资本家却仍在扩大生产，导致生产出的商品无人购买，同时资本热衷投机，两者叠加引发经济大萧条，甚至间接导致了第二次世界大战。

当前中国也处于类似的全面过剩危机中：产能过剩、资金无处可去。这导致了房价调控、社会内卷等现象。

> **请注意**：以上仅为一种经济学观点（如温铁军、林毅夫等学者持有类似看法），并非绝对事实。建议大家多方查阅资料，进行独立判断。

## 从日本看经济与思潮的关联：以史为鉴 🇯🇵

既然讲到极端女权，那么经济与社会风气究竟有何关联呢？关联性非常大。请注意，我们讨论的是**群体现象**，而非个体案例。经济学是群体科学，不能简单用社会群体现象来解释个人生活。

我们以日本女权的兴衰为例。在八十到九十年代日本经济黄金期（泡沫最高时），东京房地产疯涨，社会出现一种被称为“恋爱资本主义”的普遍现象。

以下是当时的部分社会现象：
*   部分女性找男朋友会找“四个钱包”，分别负责买包、付房租、请吃饭，还有一个是“本命”（真正男友）。
*   先富起来的男性追求女性基本靠“钱砸”，送贵重礼物（如LV包），谁肯多出钱女孩就跟谁。
*   社会财富急剧增加，学生和女性都不愁找工作，眼光变得很高、很挑剔。
*   部分95后、00后敢于顶撞领导，因为大不了不工作，家里有积蓄，或者送快递、开滴滴也能养活自己。

这些现象导致日本女权一度极度看不起本国男性，与当前中国部分情况相似。女性要求变高、变挑剔，对普通男性恋爱、结婚造成压力，类似中国一些地方的“天价彩礼”现象。同时，部分女性希望“嫁个金龟婿”或本地土豪，一劳永逸。

这一切的根源在于：**经济发展，日子好过了，人们不再为基本生存发愁，于是开始“作”**。女权、男权之争皆源于此。

## 泡沫破碎后的变化：内卷与反思 🔄

好，我们说到这是日本泡沫最高的时候。但是泡沫破碎之后呢？蛋糕变小，社会开始疯狂内卷。

以下是泡沫破碎后出现的变化：
*   **特定行业**（如风俗业）越来越发达，因为部分女性发现这样来钱快。
*   女性开始不好找工作。
*   一部分在竞争中失败的普通男性（“普信男”）选择摆烂躺平，成为“御宅族”。
*   女性开始降低择偶要求。如今日本女性找老公的普遍要求可能仅为：有工作、不加班。
*   即便如此，很多日本男性也不愿结婚。

在内卷中，输家通常是部分女性和能力不强的“弱男”。而内卷的赢家则分为“强女”和“强男”。

## 极端女权的反噬与衰落 🚫

上一节我们看到了内卷下的众生相，本节中我们来看看极端女权思潮如何走向衰落。强女往往也不认同极端女权。极端女权曾导致一些荒谬标准，例如：男上司给女下属单独发邮件算性骚扰、男性多看女性两眼也算性骚扰。

这对男性而言意味着极高的风险，可能面临“社会性死亡”。即使最终法律证明清白（如朱军案），个人声誉和事业也已遭受重创。其后果是：男性在职场和社交中会对女性敬而远之。

这就导致女性想获得资源，途径变得狭窄：要么通过与强男发生关系，要么在女性内部竞争。想通过正常社会途径晋升变得困难，因为男性主导的公司管理层会倾向于避开“难搞”的女性员工。

这好比企业招聘：一个能力一般但要求特多、动不动就要仲裁的员工，老板会首先考虑不招；即使招了，也不会将其视为核心，有裁员风险时也会优先考虑。同理，如果一个女性员工创造的价值有限，要求却很多，那么好事轮不到她，有风险时她却可能首当其冲。

因此，日本的强势女性也反感极端女权，她们更希望被当作平等的“人”来看待，而非需要特殊照顾的“女性”。一来二去，极端女权在日本失去了市场。

**女权衰落的一个最典型迹象是什么？就是特定行业（如风俗业）内容的大量涌现和多样化。**

## 总结 📝

本节课中我们一起学习了“为何追女孩的男生越来越少”这一现象背后的复杂原因。我们从**极端女权（田园女权）** 的表现切入，探讨了其背后的社会经济根源——**全面过剩**（生产、资本、金融过剩）导致的社会内卷。通过回顾**日本经济泡沫期及其破裂后**的社会变迁，我们看到了经济状况如何深刻影响两性关系和思潮兴衰。最后，我们分析了极端女权思潮如何因其**不承担责任却无限索权**的核心，以及对男性造成的**高风险**，最终导致其失去市场，甚至被强势女性反感。理解这些宏观背景，有助于我们更理性地看待当下的社会情绪与群体行为。
