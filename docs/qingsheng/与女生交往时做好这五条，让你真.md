# 课程一：如何真正地“做自己”？ 🧘♂️

在本节课中，我们将学习“做自己”的核心含义，并拆解为五个具体、可执行的步骤。通过理解并实践这些原则，你将能减少内耗，提升自信，在人际交往（包括与女生交往）中更加从容、真实。

---

## 概述

“做自己”并非随心所欲或固执己见，而是一种基于自我认可、清晰规划和高效行动的内在状态。许多人感到困扰，是因为过于在意他人评价或自身缺点，导致行动犹豫、情绪不畅。我们将通过五个步骤，系统地构建这种“做自己”的能力。

---

## 一、改正缺点，提升自我认可度

上一节我们概述了课程目标，本节中我们来看看第一步：改正缺点。这听起来简单，却是“做自己”的基石。

刘德华曾提出，真正做自己的首要一点是“改正自己的缺点，做个可爱的人”。这并非为了迎合他人，而是因为它符合我们内在的意志——几乎每个人都希望变得更好。当你认真着手改正缺点时，就会产生**自我认可**。

**核心公式**：
`自我认可度 ↑ = 自我否定 ↓ = 情绪流畅度 ↑`

一旦自我认可度提高，你就不会轻易自我否定，情绪也会更顺畅。你不会再总想着“哦，我要听他的，我要听她的”，而是会想：“我自己在一天天变好，我为什么要受你影响？”

---

## 二、戒除“应该”思维

在提升了自我认可度后，下一步是审视自己的思维模式。本节我们将探讨如何戒除僵化的“应该”思维。

“应该”思维是指给自己设定大量条条框框，例如“我这时候应该这样说话”、“我不能那样做”。这种思维会让你在社交中变得要么小心翼翼，要么不自觉地阿谀奉承。

当你自我认可度不高时，就会不自觉地想去讨好、模仿他人。戒除“应该”思维，意味着相信自己的判断，不再被外界杂乱的声音干扰。你会意识到：“我觉得凭自己就可以，为什么要听你们在那里叨叨？”

---

## 三、修改评价源

建立了内在的自信和思维模式后，我们需要管理外界的影响。本节重点在于如何掌控评价的来源。

修改评价源，是指不要以他人对你的评价作为行动指南，而是学会自己评价事情。

**应用场景示例（搭讪）**：
许多人不敢搭讪，是因为在意旁人眼光或女生看法。修改评价源后，你的思考逻辑变为：
1.  我的目标是什么？（例如：礼貌地认识一位女生）
2.  我的行为是否恰当？（例如：不骚扰他人）
3.  我对自己此次行动的规划和安排是什么？

**核心要点**：只要你清楚自己为何做某事，并有自己的想法和安排，就不太会在乎外界的看法。反之，若想法模糊、缺乏安排，就极易被外在评价压垮，不敢行动。

---

## 四、提升行为惯性

当我们学会了管理内在与外在评价后，就需要将认知转化为行动。本节我们来探讨行动速度的重要性。

真正的“做自己”往往体现在快速行动上。犹豫不决、拖延纠结正是无法做自己的表现。

**行为惯性**与胆量、焦虑程度有关，甚至与**心率**存在关联。例如，在水上乐园爬高台时，上升的心率会让人更“敢”玩刺激项目。长期行动力强的人，更容易养成“快速决策-执行”的习惯，如同形成肌肉记忆。

**代码描述其过程**：
```python
if 行动速度 == 快:
    自我内耗 = 低
    胆量与激情 = 高
else: # 行动速度 == 慢/犹豫
    焦虑与纠结 = 高
    机会流失 = 高
```
这并非天生胆大，而是可以通过后天有意识的“行为激励”来训练。例如，从不敢搭讪到觉得有趣、刺激，就是一个正向训练的过程。

---

## 五、学会行动性思维

最后，我们将把前面的步骤整合成一套更高级的思维模式。本节介绍如何用行动性思维进行规划与执行。

行动性思维指在考虑问题时，严谨规划每一步，但执行时只专注于当前第一步，并根据反馈灵活调整后续计划。

**它与僵化计划的区别**：
-   **错误示范**：追求女生时，死板规划“明天说什么、后天送什么”，这反而显得不自然。
-   **正确示范（行动性思维）**：
    1.  **规划框架**：本周目标认识20位女生，并计划约会其中5位。
    2.  **执行第一步**：今天先去尝试搭讪，了解实际能接触的数量和质量。
    3.  **反馈调整**：若发现一天只能认识5位，则调整本周目标或增加每日行动量。

曾有学员通过“每天搭讪50位女生，坚持3个月”的刻意训练，从内向变得游刃有余。关键在于**有意识、受控的训练**，而非着魔似的强迫行为。判断标准在于事情是否在你的理性掌控之中。

---

## 总结

本节课我们一起学习了“真正做自己”的五个核心步骤：
1.  **改正缺点**：提升自我认可度，减少自我否定。
2.  **戒除“应该”思维**：打破内心条框，相信自我判断。
3.  **修改评价源**：以自我规划为导向，而非他人看法。
4.  **提升行为惯性**：通过快速行动降低内耗，培养胆识。
5.  **学会行动性思维**：先规划框架，再专注执行第一步，并灵活调整。

总而言之，“做自己”是一种**没有过度顾虑、行为流畅**的状态。它不会让你做出格的事，而是让你摆脱焦虑，更高效、自信地行动。这套方法论不仅是人际交往的基础，也是个人成长的核心。结合清晰的**行为规划**与**高效执行**，你便能稳步朝着目标（如一年内脱单）前进。
