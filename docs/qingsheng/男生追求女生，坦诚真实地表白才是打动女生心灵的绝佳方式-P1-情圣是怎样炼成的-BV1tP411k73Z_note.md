# 课程名称：情圣是怎样炼成的 - P1：坦诚真实表白的核心 🎯

在本节课中，我们将要学习在追求女生时，为何坦诚真实地表白是打动对方心灵的关键方式。我们将剖析一些常见的错误应对方式，并探讨其背后的心理本质。

## 错误应对方式剖析 ❌

我们经常在网上看到一些关于追求女生的观点。以下是几种典型的错误应对方式。

首先，第一种观点是“不要在乎她对你的看法”。第二种是“做自己就好”。此外，还包括刻意“克制”某些行为，并期待女生因此产生特定反应。

## 错误行为举例与本质 🔍

你思考一下，为何会有这些行为？例如，在与女生聊天时，有些人会不自觉地奉承女生。奉承并非赞美。比如对女生说“你长得真好看，肯定有很多男生追你吧”之类的话。

甚至有些直男式的“赞美”实为奉承。例如，对女生说“你就像一缕清风一样扑面而来”。女生听到这种话通常只会感到尴尬。女生喜欢真诚的赞美。

例如，你可以对女生说：“我觉得你这个人还挺有自己看法的。”如果这与她的事实情况相符。而不是没话找话，硬去赞美。比如之前有人说：“像你这样漂亮的女生，应该没什么烦恼吧。”这属于奉承话。

包括有些男生喜欢自黑。例如，对女生说：“你看你每天工作都很忙，不像我，一天到晚在办公室摸鱼，感觉生活好无聊。”很多人说过类似的话。

前两天晚上有个视频投币很多，标题是“你只要犯了这六个错误，这辈子注定单身”。我点开看了一下，发现同行说的内容大同小异。我觉得他们说的没有错，但只是罗列现象，并未指出问题的本质。

还有一种想法是：“她肯定对我没意思，要不还是算了。”这同样是在乎女生看法的表现。“我再这样继续下去，她肯定也不会理我。与其这样，我还是算了，何必给自己找罪受呢？何必去跪舔女生？”这些都是错误的应对方式。

## 在乎他人看法的心理根源 🧠

大家思考一下，在乎别人对自己看法的本质是什么？你们自己是否有这个问题？有这个问题的人可以打1，问题很严重的打9。

你们心里一定有强烈的感受。在什么样的女生面前，你会特别在乎她的看法？有人说这些人的本质是核心价值不够。核心价值又是什么？

其实原因很简单，我们多次讲过，是**懦弱与自卑**。因为懦弱而自卑，因为自卑而懦弱。所以你才会在乎别人对你的看法。如果你不懦弱自卑，你怎么会在乎？

你看过《权力的游戏》里，兰尼斯特家族的老大，即小恶魔的父亲，说过一句话：“狮子才不会在乎绵羊的看法。”这是因为内在气场和实力完全碾压对方，才会有这种心态。

实际上，在乎他人看法是心理上很弱势的一种表现。即内在心理能量不够，处于弱势。你想想，这种弱势与什么有关？你在乎的人，一定是在某些方面比你强势。否则你不可能如此在乎。

男生最容易在乎两种人的看法：一种是能量和气场强于他的同性；另一种是非常漂亮或条件优越的女性。我是这样总结的。

## 关于家庭环境影响的讨论 🏠

有人开始提及家庭环境的影响。这与家庭环境有什么关系？如果我们看一下最近早读讲的毛主席，他就是一个非常有个人主见、性格特别独立、心地真诚善良的人。他从小如此。

他虽然出身农村，但到了十八九岁或二十多岁时，被全校评为综合素质排名第一的学生。这与家庭环境有什么关系？很多人喜欢寻找家庭环境的原因。

我建议你们读一读阿德勒的心理学。虽然我没有仔细研读过他的著作《超越自卑》，但我看书较多。那本书有一个核心观点：**你的现在可以改变你的过去**。

意思是，如果你现在过得好，那么早年家庭环境不好、父母教育不当，都可以被诠释为锻炼了你独立有主见的性格。如果你现在过得不好，就容易归咎于家庭环境。这是结果论的思维。

人特别容易寻找家庭环境这类原因，比如父母打压等。但你们看看毛主席早年他父亲是如何打压他的。人本主义心理学并不强调这个。我看过人本主义创始人卡尔·罗杰斯的书，他的《个人形成论》中没讲这个。

他可能只是借用了“人本主义”的名头。当然，西方心理学流派众多，各执一词。社科人文领域没有硬性规定，目前都是假说，假说怎能当真？

但从现实逻辑来看，童年创伤可以造就好的结果，也可以造就不好的结果。我觉得阿德勒的说法更靠谱：**你现在过得好，童年的经历就是塑造你坚强内心的助力**。

所以，最基本的原则是：人最主要的是找到我们能做的事情，去改变自己的某些方面。你光去找家庭原因，怎么找？神仙也不能让时光倒流，让你重新投胎换个家庭。

再讲另一个点，谁的原生家庭没有一点问题？谁的成长经历是一帆风顺的？谁不需要面对社会和家庭的打压？无论是善意的还是恶意的打压，你们都经历过。

有的打压动机是好的，但结果很差。有没有人从小到大，父母都用完全正确的方式对待，生活在一个充满爱、特别幸福的家庭？

前两天我妻子跟我说，她做家教时遇到一些父母。父母很恩爱，教育孩子要对别人笑脸相迎，要表现得很开心。结果呢？这个孩子回家后割腕了。为什么？因为有时候他不开心，却要强压情绪，表现出开心。

这就像你们很多人说的：我本来很在乎，却强压情绪让自己不在乎。时间长了会出问题的。明白吗？

所以，这没有一个统一的论调。但一个最基本的原则是：**我们只能做自己能力范围内可以做的事情**。像家庭矛盾、原生家庭问题，你去找父母，跟父母打官司，对他们说“都怪你们小时候那样对我，我才混得不好”，这会不会造成另一个问题？即你对父母的怨恨更多，认为你的问题都是父母造成的。

## 核心要点总结 ✨

本节课中，我们一起学习了：
1.  追求女生时，坦诚真实优于奉承、自黑等错误方式。
2.  **在乎他人看法的本质常源于内心的懦弱与自卑**。
3.  改变应聚焦于当下可做的事情，而非过度归因于不可改变的家庭环境。
4.  真诚的互动和稳定的自我心态，是建立良好关系的基础。
