# 情商提升课 01：自我意识的三重结构 🧠

在本节课中，我们将要学习情商的核心基础——自我意识。许多人际交往中的困扰，尤其是被评价为“情商低”的情况，往往源于对自我意识的理解不足。我们将系统地拆解自我意识的三重结构，帮助你理解其运作机制。

## 自我意识的三重结构

上一节我们提到了情商问题常与自我意识相关。本节中，我们来看看自我意识在学术上具体包含哪三个层面。

自我意识包含三个部分：**自我认知**、**自我体验**和**自我控制**。

### 1. 自我认知：你如何看待自己

自我认知是指“我觉得我是一个什么样的人”。这是自我意识的起点。

你们很多时候，一个人情商表现不佳，首先出现的就是自我认知问题。这种自我认知并非凭空想象，而是一种结果。自我意识本身就是一个结果。

以下是常见的负面自我认知例子：
*   许多人常说“我有抑郁症”。
*   许多人常说“我有社交恐惧症”。

你们很多时候先有了这类自我认知，然后整个人就给人感觉情商特别低。

### 2. 情商高低的能量感知

那么，什么叫情商低？其核心在于能量感知。

情商高的人，你总会觉得这个人情绪状态饱满、心理状态良好。你跟他在一起能感到舒服，因为他的情绪状态稳定且积极。

但是有一些人则相反。你跟他在一起就会感觉这个人死气沉沉、缺乏能量和热情。这就是情商低给人的直观感受。

### 3. 负面循环的形成

有很多人问，情商低是怎么来的？答案就藏在一个负面循环里。

其路径如下：
1.  **起点**：觉得自己不好（负面自我认知）。
2.  **反馈**：别人对你的反馈也不好。
3.  **体验**：你感到自卑。
4.  **失控**：自我控制失效，表现为无法控制的沮丧和情绪化。
5.  **行为**：促使你做出具有负面效果的行为。

感觉自己不好、且收到外界负面反馈的人，做事时会很沮丧并带着情绪，从而不想行动。很多时候不是我们没有能力，而是找不到动力，总感到沮丧和不被认可，进而强化了“自己不好”的认知。

### 4. 在两性关系中的体现

这种模式会直接影响行为，例如很多人不再追求异性。

其过程是：
1.  觉得自己不好（自我认知）。
2.  在追求过程中体验很差，认为对方要求高、难度大。
3.  在整个过程中，情绪体验非常糟糕。
4.  时间长了，就不想去做了。
5.  最终，在对方眼里你的情商就显得越来越低。

### 5. “直男思维”的认知局限

还有一种常见的情商低表现，源于所谓的“直男思维”。

我们说一个人情商低，常形容为“少根筋”或“一根筋”。“一根筋”特指其意识固守在自己的单一视角里。

例如，很多直男会想：“这女孩子怎么现在这么多毛病。”但他们没有考虑到，现在的女性群体已经不同了。他们觉得女性“没文化、没水平、追求打扮、收入不高却要求高”，从而感到难以追求并缺乏自信。

### 6. 社会比较与自我提升

你们有没有发现一个现象？现在能看到的男性和女性，在颜值和精致感上，普遍比过去高出太多。

在大家整体水平都提升的情况下，如果你还停留在原地，别人凭什么选择跟你在一起？这是一个简单的道理。当别人的高颜值是靠金钱、时间等资源堆砌出来时，如果你既不提升自己，又总是抱怨女性难追、拜金，那问题出在哪里？

你看现在好看的人那么多，很多女性也被迫努力打扮自己。为什么？因为如果她们不好好打扮，她们也可能找不到伴侣。这反映了普遍存在的社会比较与提升压力。

---

本节课中我们一起学习了自我意识的三重结构：**自我认知**、**自我体验**和**自我控制**。我们了解到，负面的自我认知会启动一个情绪与行为的负面循环，最终在人际交往中表现为“低情商”。同时，固化的思维模式（如“直男思维”）和忽视社会比较的现实，也会加剧人际间的隔阂。理解这些是提升情商的第一步。
