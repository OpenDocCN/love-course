# 情圣是怎样炼成的 - 第一课：一眼识别情场高手与新手 🕵️♂️

在本节课中，我们将学习一个核心概念，它能帮助你快速区分哪些男性是情场高手，哪些缺乏追求女性的经验。这个核心区别在于他们思维模式的根本不同。

## 概述：两种截然不同的思维模式

追求女性时，人的行为模式可以清晰地分为两类。理解这种区别，是掌握后续所有技巧的基础。

## 核心区别：追求“确定性反馈” vs 创造“更多可能性”

不会追女孩的人，其思维核心是希望获得**确定的反馈**。而会追女孩的人，其思维核心是希望获得**更多的可能性**。

### 什么是“追求确定性反馈”？

这种思维模式表现为，总希望事情能按一个明确的计划发展。例如，追求一个女生时，会制定详细的步骤：第一步怎么做，第二步怎么做，第三步怎么做……并认为只要按计划执行，就一定能成功。

**公式化描述**：
```
新手思维 = 追求(确定的成功路径)
```

### 什么是“创造更多可能性”？

这种思维模式则完全不同。他们不预设固定路径，而是专注于在互动中不断创造新的机会和可能性。例如，看到一个不错的女生，会直接过去要联系方式，而不会事先规划好所有步骤。如果女生拒绝吃饭邀请但态度友好，他们会转而尝试要电话号码。

**公式化描述**：
```
高手思维 = 追求(互动的可能性)
```

## 思维模式在行为上的体现

上一节我们介绍了两种核心思维模式，本节中我们来看看它们在具体行为上是如何体现的。

以下是两种思维模式在常见场景下的不同表现：

*   **场景一：送女生回家**
    *   **新手思维（追求确定性）**：希望女生明确同意自己上楼，否则就放弃。
    *   **高手思维（创造可能性）**：会尝试创造机会，例如借口“手机没电了，能否上楼充电？”或“肚子痛，能否借用洗手间？”。即使被女生用充电宝等理由化解，他们也尝试了，这就创造了“发生故事”的可能性。正如一句名言所说：“真正的男人应该不断想办法，真正的女人应该不断挣扎。”这揭示了男女交往的基本逻辑。

*   **场景二：街头搭讪**
    *   **新手思维（追求确定性）**：总在担心“万一被拒绝怎么办？”，并寻求“绝对有效的话术”，希望确保自己不被拒绝或能获得正面反馈。
    *   **高手思维（创造可能性）**：明白“只要尝试就有机会，不尝试则毫无机会”。他们会临场想办法，比如直接说“美女，加个微信”，或者用一些看似笨拙但真诚的借口（如“我打赌输了，必须请你吃冰淇淋”）。他们的关键是先开启互动，再随机应变。

*   **场景三：酒吧社交**
    *   **新手思维（追求确定性）**：试图学习一套完整的聊天流程和游戏方法。
    *   **高手思维（创造可能性）**：可能会拿着酒杯与全场女生简单碰杯，观察谁的反应更好，从而锁定目标。或者，为了要联系方式，会先为女生存一瓶酒，让酒保请她留电话，以此创造机会。

## 为何“创造可能性”的模式更有效？

我们已经看到了两种模式的具体行为差异，现在来分析为什么“创造可能性”的模式往往更成功。

追求“确定性反馈”的人，其情绪始终被女生的反应所掌控。即使前期反馈良好，但只要有一次失误，他就会慌乱，不知如何是好。因为他没有掌握关系的主动权。

而专注于“创造更多可能性”的人，主动权往往掌握在自己手中。他们不依赖于单次行动的成败，而是不断开辟新的互动路径。因此，他们不会因为一次拒绝而全盘皆输。

**核心逻辑代码描述**：
```python
if 思维模式 == “追求确定性”:
    情绪状态 = 依赖于(对方的单次反馈)
    主动权 = 在对方手中
    容错率 = 低
else: # 思维模式 == “创造可能性”
    情绪状态 = 保持稳定
    主动权 = 在自己手中
    容错率 = 高
```

## 总结

本节课中我们一起学习了情场高手与新手的根本区别。

*   **不会追女孩的人**：其思维内核是**追求“确定性反馈”**。他们害怕负面结果，总希望事情按计划发展，情绪易受他人影响，最终往往事与愿违。

记住这个核心区别：**前者在等待完全的确定性，因而常考虑负面情况；后者只为得到更多可能性，因而常常乐于挑战。** 这是所有技巧之下的底层逻辑。
