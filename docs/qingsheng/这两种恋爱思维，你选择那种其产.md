# 这两种恋爱思维，你选择那种其产

在本节课中，我们将要学习人际关系中两种截然不同的思维方式。理解这两种思维，能帮助我们看清许多社交困境的根源，尤其是在恋爱关系中。

## 概述：控制与剥夺 vs. 理解与关怀

我们首先介绍人际关系的两种基本思维方式。这两种思维模式导致的行为和结果完全不同。

第一种思维叫 **控制与剥夺**，也叫排序思维。当人急于得到某物或追求短期利益时，就容易产生这种思维。

---

## 第一节：控制与剥夺思维详解 ⚔️

上一节我们提到了控制与剥夺思维，本节中我们来详细看看它的表现。

这种思维的本质是 **零和博弈**。即：**你的优势建立在对方的损失之上**。一方占便宜，另一方就吃亏。

以下是这种思维的一些典型表现：

*   **在竞争场景中**：例如打游戏或下棋时，会想尽办法使用各种手段、阴谋诡计去战胜对方，核心是运用 **暴力** 与 **智谋** 进行巧取豪夺。
*   **在不良的恋爱教学中**：例如某些PUA技巧，其核心思路是“如何把女生骗上床”，关注点在于“得到”和“占便宜”。具体行为可能包括：如何把女生约出来、如何引导至私人场所等。这些都是控制与剥夺思维的表现。
*   **在人际互动中**：即使在与父母的互动中也可能出现。例如，父母用命令或威胁的方式（如“你不洗碗我就揍你”）让你做事，你虽然做了，但心里会极其不爽。这是因为你感受到了被控制和剥夺。

西方思想中有一部分被称为 **“控制的科学”**，即热衷于将一切事物方法化、步骤化，研究如何算计他人，如何用暴力和智谋攻伐他人，这是一种“恃强凌弱”的哲学。

网络上许多极端的男权、女权争论，其底层逻辑也往往是这种“我占便宜你吃亏”的控制与剥夺思维。

## 第二节：理解与关怀思维详解 🤝

理解了具有破坏性的控制思维后，我们来看看另一种建设性的思维。

第二种思维是 **理解与关怀**。当人真心想去理解他人的感受，关怀对方时，就会产生这种思维。

以下是这种思维的运作方式：

*   **激发责任感**：真正的理解与关怀能激发人的连接思维和责任感。例如，父母如果换一种方式沟通：“妈妈手受伤了，知道你学习忙，但妈妈也需要你帮忙，希望你能学会照顾自己，扛起家庭责任。” 虽然目的也是让你洗碗，但你不是在被“算计”，而是感受到了被需要和家庭责任。
*   **连接思维的核心**：连接思维的核心是 **同理心**。它意味着能站在对方角度思考，不急于做非黑即白的判断，能看到更深层次的人性。
*   **东方哲学的基础**：这种思维深深植根于东方文化，被称为 **“理解的科学”** 或 **“心灵的生活”**。儒家思想强调“仁”（左边是“人”，右边是“二”），探讨的就是人与人之间和谐对等的关系。我们的文化天性中就有温良、理解与关怀的部分。
*   **超越利益的关怀**：这能解释为什么在国际交往中，我们常常愿意向有需要的国家提供帮助（如疫情期间的疫苗援助）。这不仅是外交手段，也源于我们文化中“生命不分国界”的关怀精神。

## 第三节：两种思维在恋爱中的具体影响 💑

理论需要联系实际。在恋爱关系中，控制与剥夺思维会引发一系列具体问题。

当男生用控制与剥夺思维与女生交往时，通常源于 **急于求成** 的心态。这会导致两种极端行为：

以下是急于求成心态下的两种典型错误：

1.  **越权/冒进**：因为不想经历中间的相处过程，想直接得到恋爱结果。就像觉得游戏通关太慢，就想用作弊器。表现为在关系未到时急于表白、推进身体接触，甚至最后演变成威胁（如“得不到就毁掉”）。
2.  **跪舔/当备胎**：因为害怕失败，所以不敢前进，只能通过不断讨好、付出以维持微弱希望。就像害怕花死，反而不敢浇水。

这两种状态就像 **浇花**：
*   **急于求成** 就像用开水浇花，只想它快点结果，却会毁掉它。
*   **害怕失败** 就像不敢浇水，最终花也会枯萎。

问题的根源在于，你只关注自己的 **欲望**（想得到）和 **恐惧**（怕得不到），而忽略了客观事实（对方的状态、关系的自然发展规律），无法做出合乎自然的行为。

## 第四节：欲望如何降低你的智慧 🧠

为什么聪明的我们一陷入感情就容易做傻事？本节探讨欲望与智慧的关系。

**欲望和情绪是智慧最大的敌人**。当人被强烈欲望（或恐惧情绪）支配时，就会“一叶障目，不见泰山”，智商急剧下降。

以下是欲望让人变“蠢”的几个例证：

*   **不相信事实**：女生已明确拒绝，但你不死心，总想寻找“神奇方法”挽回。这时，你很容易相信骗子声称的“挽回秘籍”，从而缴纳“情商税”。
*   **被情绪操控**：例如接到诈骗电话，一听家人有危险，恐惧情绪上头，瞬间失去判断力，等钱转出去才反应过来。这不是智商问题，是情绪接管了大脑。
*   **被欲望支配**：在色欲或贪念驱使下（如以为中奖、以为即将发生亲密关系），人会主动屏蔽风险信号，心里只想“万一是真的呢？”，从而落入陷阱。
*   **恋爱中的套路**：骗子会精准抓住你的核心欲望（如结婚、得到女神），让你觉得“只差一点点”，诱导你不断投入时间、金钱、情感，最终一无所获。

**核心公式**：**强烈欲望/情绪 → 屏蔽客观判断 → 做出愚蠢决策 → 付出代价**

在恋爱中，一旦你 **急于求成**，智商下降，女生会敏锐察觉并看低你。而你因为无法可施又嫌过程麻烦，更容易走向“博一把”的极端控制行为，彻底暴露缺乏耐心、缺乏尊重和理解、以及 **耐挫折能力低下** 的缺点。

## 第五节：当代困境与思维选择 🌉

最后，我们结合时代背景，思考为何控制思维在当下更具诱惑力。

为什么有些人（如案例中的“冲哥”）对恋爱（控制）如此执着，而非将精力用于事业？

这是因为在当下社会，**干事业带来的成就感正在衰减**。物质成功（如买好车、出国旅游）的边际效应降低，不再像过去那样能带来强烈的社会认同和自我满足感。

**公式**：**事业成就感 = 获得物质 / 社会普遍物质水平**

当分母（社会普遍物质水平）很大时，分子（个人获得的物质）带来的成就感就变小了。因此，一些人转而从 **控制一段关系**、**征服一个人** 中寻求即时的、强烈的成就感与掌控感，这比漫长且结果不确定的事业奋斗显得更“诱人”。

## 总结与回顾 📚

本节课中我们一起学习了：

1.  **两种核心思维**：**控制与剥夺**（零和博弈，追求占便宜）与 **理解与关怀**（同理心，追求和谐连接）。
2.  **思维的影响**：控制思维导致冒进或跪舔，让人在关系中变得愚蠢且富有破坏性；理解思维则能建立真诚连接，激发责任感。
3.  **欲望的陷阱**：强烈的欲望和情绪会严重降低我们的判断力，是许多“愚蠢”决策和被骗局套牢的根源。
4.  **时代背景**：社会变迁使得从控制他人中获取廉价成就感的诱惑增大，但这条路最终通向自我毁灭和关系破裂。

真正的智慧在于 **克制急于求成的欲望**，**回归对他人的尊重与理解**，让关系在合乎自然的节奏中发展。这需要静心、寡欲，以及将注意力从“我要得到什么”转移到“我该如何理解对方和当前情境”上。
