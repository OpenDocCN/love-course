# 课程名称：情圣是怎样炼成的 - 第一课：核心心态与行动法则 🧠💪

在本节课中，我们将要学习如何调整心态、提升行动力，并建立健康的交往原则。课程内容将围绕“生死看淡，不服就干”的核心精神展开，同时强调底线与原则的重要性，帮助你走出犹豫不决的状态，成为一个更有魅力的人。

## 破除自负：学会理解与包容 🤝

上一节我们介绍了课程的整体目标，本节中我们来看看如何破除人际交往中的第一个障碍——自负。

当一个人过于自负时，他会认为自己总是对的，并容易看他人不顺眼。我自己在长期讲课的过程中，也意识到了自身存在这个问题。

人很容易对他人包容性过低。当别人的行为不符合自己的理念时，就倾向于批评和指责。我首先意识到，讲课时间长了，发脾气和骂人会对我的现实生活产生负面影响。我认为这是不对的，因此我需要改变。

关键在于，多站在别人的角度，根据他人的实际情况，提供合理且符合其处境的分析。在与他人交往或指出问题时，应尽可能从他的角度出发，给予合理的建议。**合理性**很重要。

每个人的情况和状况都不同。你需要根据他的具体情况，**因地制宜**地提供帮助，而不是简单地看到“不对”就批评、指责。因为一个人自身能力太强时，容易认为所有人都应和他一样，否则就会感到生气或失望。

但实际上，每个人的状况都不同。很多时候，你不能如此强制性地对待他人。这样做会让身边的人感到难受。

## 核心行动法则：先走肾再走心 🚀

那么，如何破解上述困境呢？方法是：**先走肾再走心**。

以下是针对生活中比较“蔫”、做事想太多、总担心付出没结果的人的建议：

*   **核心行动指南**：要“生死看淡，不服就干”，别想太多无关紧要的事情。
*   **目标人群**：目前身上“缺火”、缺乏行动力的人。
*   **具体方法**：用积极的价值观加持自己，提升行动力。走出“蔫”的状态，首要目标不是追求结果，而是**先让自己行动起来**。
*   **行动原则**：只要所做的事情不会伤筋动骨、大伤元气或危及生命，你就可以去干。不要被恐惧束缚。

“生死看淡，不服就干”（此处为比喻）的精髓就在于，不要想太多，先迈出第一步，让自己动起来。这是我们所说的“走肾”阶段。

当然，我们始终强调要有底线和原则。例如，**抢别人女朋友这种事就别干了**。

先行动起来。当你勇于突破自我、开始行动时，别人就能看到你身上的激情，你也能释放自己，从而真正建立自信。

## 坚守底线：原则的重要性 ⚖️

上一节我们探讨了行动的重要性，本节中我们来看看为何必须坚守底线。

我曾经也喜欢抢别人的女朋友，但后来我进行了反思。我发现，我能“撬动”的人，别人也能。这并非玄学，而是有内在逻辑的。

逻辑在于：如果你总是喜欢抢别人的伴侣，说明你会对这种类型的女生产生重复性选择。当你真正寻找女朋友时，可能也会选择同一种类型。

那么请思考：这个女生能够背叛别人，难道就不会背叛你吗？所以有句话叫：**赢人妻女者，妻女必被人赢**。

原因就在于你的口味不会变，你只会被那种“骚的、浪的、婊的”所吸引。你以后找的也是这类人，你抢过来，结局也可能一样。你潜意识里搞不定这些，凭什么认为你能“套住”她，而别人不能？不要误以为是你的魅力大。

大家要注意：从别人那里抢来的人，并不说明她有魅力，而是说明她**人品有问题**。记住，没有原则和底线的人，会给你带来一堆麻烦。

生活中与这种做事没底线的女性在一起，你不知道要赔上多少。有句话叫“婊子配狗，天长地久”，说的就是这种人。

## 培养责任感：别只顾自己多管点事 🐱🌱

接下来我们看看如何培养责任感，这对建立深度关系至关重要。

第二个要点是：**别只顾自己，多管点事**。这是什么意思呢？

如果你生活顺遂，要注意别只关注自己。抱着“玩玩就好”的心态，玩多了之后就会出现问题：你会变得不相信任何女性。如果你觉得什么女性都能“撬动”，你就会永远怀疑，自己最终交往的女朋友是否也会被别人抢走。这就是问题所在。

“别只顾自己多管点事”的意思是：在你的生活中，无论是室友还是宠物，心里总要有个计划。

以下是你可以做的事情：

*   记得什么时候给植物浇水。
*   记得什么时候给猫喂猫粮、清理猫砂。

这样，你就不会事事只顾自己。未来与人相处时，也会养成这种习惯。

还有一点：别指望每个人都很独立。他独立是他的事，你照顾他是你的事。这属于你对自己的要求或养成的习惯。

如果你在生活中什么都不管，只做好自己的事，会有一个问题：你很难与别人产生情感联结。你可能能满足性需求或其他，但在追求女生时也会出现这个问题——你根本不管她的事，只想着能否得到她。这就是典型的**只顾自己，不顾别人**。

你不管别人的事，和女生在一起时就会显得“没有水，干巴巴的”。女生会觉得和你在一起没意思。

## 练习表达：主动赞美与流露好感 😊

要建立良好的互动，学会表达好感是基础。

第三点是：**见人学会主动赞美和表达自己的好感**。这属于人际交往的基本技能。

你学会赞美并表达好感。我举一个最简单的例子：比如在街上和某个女生对视，你可以对她微笑。从这种最基本的互动开始练习。

微笑之后，你可以很自然地对她说：“我觉得你长得很好看。” 这句话要说得非常自然，不是出于想撩拨对方的目的。这样你就不会表现出表达好感时不自然的状态。

女生能感觉到，和你在一起时，你的好感是自然流露出来的。我们以前的老师讲过，从小培养孩子对父母说“我爱你”，长大后他对女生的状态就会很自然。如果小时候不培养，对女生的状态也会变得特别不自然——不敢表达，不知道如何追求和诉说。

往往有这种情况的人，可以试试给你爸妈打个电话说句“我爱你”，说得出口吗？其实就是这样。如果你平时主动养成那种习惯，主动去赞美别人、夸别人、说“我喜欢你”，很多人做不到，张不开嘴，不好意思。这就是不善于表达情感。

## 做事走心：沉下心来追求做好而非结果 🧘

光有行动不够，做事的态度同样关键。

第四点是：**只要开始做事就别躁，沉下心来多想想怎么做好**。不要老想着做一件事后马上就有结果。多思考如何把它做好，这叫做 **走心**。

如果你平时有这种积累，和女生在一起时养成这种习惯，沉下心来把每件事都思考如何做到最好，女生对你的反应和认可度会完全不同。

人的自信从哪里来？就是从这种平时养成的、看不见的习惯中来的。想想为什么和女生约会后会自卑？为什么每次约会后，你都不知道自己做了什么，女生就把你拉黑了？本来聊得好好的，你也不明白原因。

这说明你事情没做好。

## 外在修炼：声音与眼神的感染力 🗣️👀

最后，我们来看看如何通过外在细节提升感染力。

第五点是：**说话的声音尽量温柔，眼神尽量真挚**。我认为这是可以练习的。

例如，和女生说话时，把声调降低一点。眼神看着对方，尽量诚恳一些。最大的忌讳就是和女生说话时眼神飘忽。

这些东西很多时候其实可以后天练习，并非完全无法改变。这就是我们说的“水火”。所谓“水火”，其实与一个人本身的**荷尔蒙能量**是否得到完全释放有关。

我举个最简单的例子：人什么时候最容易说情话？什么时候最能表达情话？大家都会有体会，那种时候说情话是最自然的。

诸如：
*   第一次接吻之后。
*   或者第一次发生关系之后。

接吻完或发生关系后，躺在床上休息，你搂着他的时候，心里的状态是平静的。这时你最容易说情话，想都不用想，情话就蹦出来了。各位同学有过这种体验吗？

如果你已经得到完全的情感和欲望释放，身上有这种基础状态，你就可以自然地去跟人表达了。
