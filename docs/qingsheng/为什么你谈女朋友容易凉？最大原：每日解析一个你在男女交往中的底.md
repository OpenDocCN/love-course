# 课程名称：情圣是怎样炼成的 - 第1课：为什么你的恋爱容易失败？核心原因解析 🧐

在本节课中，我们将要学习一个在男女交往中普遍存在却容易被忽视的底层心理逻辑。我们将深入探讨，为什么很多人在追求伴侣时容易失败，其根本原因并非技巧不足，而是言行缺乏说服力，无法让对方看到希望和未来。

## 言行缺乏说服力的根本原因

上一节我们介绍了课程主题，本节中我们来看看问题的核心。实际上，一个最根本的原因是：所有懦弱自卑的人，其言行都缺乏说服力。

例如，当你和女生在一起时，女生不会去想象“以后跟你在一起我们会很幸福”。即便你嘴上说得再好听。

包括你和她说话时，你不会让她觉得很舒服。例如，如果你每天都能对一个女生说甜言蜜语，女生爱听，她天天听着这样的话，她会觉得“哪怕你以后什么都不给我，你每天对我说几句好听的，我也愿意跟你在一起”。

她觉得你有这样的能力，你能够哄她开心，所以很多女孩子就愿意跟你在一起。这叫什么？**你能够让人感受到你的好**。

再例如，你和女生在一起，哪怕你不会说甜言蜜语，但每次约会你都能做一桌子好吃的。

然后给她带各种各样好吃的，她就能想到：“以后跟你在一起，我天天都有好吃的”。她就会觉得挺好的，她能想到以后的生活。“我要是真的跟你在一起，我能天天吃到这么好吃的，挺好的”。

女孩生病的时候，你去照顾她。

你无微不至地去关怀她，然后去给她熬红糖水，去为她做各种事。她就会觉得，其实单身还是没那么好。有时候生病了，有人照顾挺好的，身边有人真的挺好的。原来可能不想谈恋爱的人，经过这么一次之后。

发现两个人在一起真的挺好。而且你愿意照顾她，愿意付出，她就会怎么样？好好认真考虑，愿不愿意、要不要跟你在一起。因为当你用你的行为，让她感受到你的好的时候，这就有说服力了。

说服力不是什么。

不是说用什么技巧去说服别人，用什么套路去让别人答应你，不是这样的。是你所作所为的一切。

## 关于“喜欢与讨厌”的误解

上一节我们探讨了说服力的本质，本节中我们来剖析一个常见的认知误区。有人说“反正你说什么做什么都是错的”。

不是这样的。很多人说他喜欢你，你做什么说什么都是对的；那么他不喜欢你呢，你说什么做什么都是错的。我觉得这个话是有问题的。

如果一个人喜欢你的情况下，你再做一些他认可的事情、让他好的事情、让他舒服的事情，他就会怎么样？他就会更加舒服。他喜欢你的情况下，你犯的错，他会对你包容性更大，但**并不代表说他喜欢你，你做什 么说什么都是对的**。

我只告诉你，这种人一般是那种很缺乏主见、对自己很没自信心的人才会这么做。

如果说，比如我讨厌你，但是你做了正确的事情，你做了好的事情，我同样会为你点赞。或者是说有的人本来做了一些错误的事情。

但是他之后用一些正确的行为来证明了自己，来跟大家表示“我改变了，我改正了，我有进步”，大家同样会认可他。不是说什么“一个人做什么说什么都是错的”，不是这样的。

是他讨厌你的情况下，你说错的话、做错的事情，他会放大而已。或者说，是你讨厌一个人，他做了好的事情、说了对的话，你心里面虽然觉得对，但是你行为上还是不愿意给他一个肯定和认可，不愿意给他好的反馈。

除非他慢慢地让你由“黑转路（路人）”，由“路转粉”，有这么一个过程。

才会让你更加地支持他，认可他。

## 为什么你的言行缺乏说服力？

我们为什么说你的言行缺乏说服力，别人对你缺乏信心呢？首先第一条，我们从实际的角度来说。

当你要去确定让人答应你、跟你走入一段关系的时候，心智怎样可以成熟？心智成熟首先第一条在于尊重别人，在别人讲课时不要随便提问。

心智成熟第一条就是懂得去尊重别人，把自己的需求往后靠，先听别人说完，然后再问问题。

你想想你这样做真的很讨厌的，我讨厌你，所有人都讨厌你，因为你只顾你自己，你不顾别人。你不考虑你这样问问题会打扰别人，打扰我的讲课，会打扰别人听课。你只想着你爽。

### 第一条：不考虑对方能获得什么好处

以下是第一个核心错误点。

第一个，不管你是跟人谈生意也好，跟人谈恋爱也好，你们很多时候去考虑问题的时候，不考虑“人家答应你什么条件对对方有什么好处”。

例如，你去主动追一个女孩，那肯定就是说人家怎么样，答应你一件事情嘛。答应你一件事情。

我们说当你提出条件的时候，人家答应你这件事情的时候意味着什么？意味着对方要付出，对不对？意味着对方要付出。

你不考虑答应你对对方有什么好处吗？你想想你们很多时候跟人表白的时候，“哎呀，我好喜欢你，你做我女朋友吧，你就做我女朋友吧，行不行啊？”你都是这么去说的话，人家怎么会答应你啊？

人家会想：“我答应你做我女朋友，就是说我答应你……有什么好处啊？做你女朋友有什么好处啊？”对不对？她会这么俏皮地说话。

包括是说你去约人家出来跟你约会，“那我跟你约会有什么好处吗？”对不对？包括跟人家出来约会，你这是约会呢，你就是约别人出来，你不去说约会去干嘛？有没有好吃的有没有好玩的？人家都觉得又没好吃的又没好玩的，我凭什么跟你出去啊？你面子大啊？

首先第一个，别人对你没信心，你这些东西没有说服力。他不肯答应你，就是因为你去提条件的时候，你不去说对别人有什么好处啊？

你们有没有想过你们以前都犯这个错误啊？有没有想过？比如约会不安排。我们就不说表白吧，表白只是说“我好喜欢你，我每天想你想得不行，你做我女朋友吧”，对吧？你从来没有说：“哎，你做我女朋友吧，以后我们养一只猫养一只狗，然后呢，我天天给你做好吃的，你喜欢的我都会给你买，对吧？这个如果我没钱的话，我会拼了自己的命去赚钱，然后给你稳定幸福的生活。”说过这种话吗？

你想想，人家要是有好处，人家答应你。我并不是说你给钱啊，“你答应我，我给你520块钱”，那他妈的，那是谈恋爱吗？对不对？

### 第二条：行为无法体现兑现承诺的能力

上一节我们讨论了不考虑对方利益的问题，本节中我们来看第二个关键点：你的行为并不能让对方觉得你有兑现承诺的能力。

例如，你现在一穷二白，你跟人家表白的时候，你就跟他说：“明年我们要买房子，明年我们要买房子，然后要买车子，要什么都有。”鬼才信你呢？你现在乱七八糟，你一个月两千块钱，你明年房子怎么买啊？你不是扯淡了吗？他信不信你啊？

而且你平时也是一副懒懒散散的样子，也没好好工作，也没见你有什么上进心。你说钱从哪里来？你骗我呀！

所以你的行为，你的表现，包括你提出的一些东西，你让别人觉得你没有兑现承诺的能力，他就不会相信你。他就觉得你去骗那些小女生还行，“你别骗我，你当我傻啊？”他会觉得你毫无诚意。

### 第三条：感受不到你为兑现承诺付出的努力

第三个，你让人感受不到你为了去兑现你的承诺，会去付出努力。

好吧，他们可以。你可能说：“以后我们谈恋爱，我希望给你一段婚姻，但是我最近这几年可能真的买房子，以目前这个收入水平，我可能不知道能够给你多长时间能够去买，但是我愿意去付出努力。”

然后呢，这个女生如果跟你接触，这个女生看着你，一个月你真的有行动，你真的为了以后她的幸福，你真是一个月收入比一个月收入高，你想尽办法去挣钱，去提高自己的这个收入。女孩子能够看到你这种拼搏和努力的精神。

你有这个心的话，其实女孩子就算跟着你一起吃苦，她也不会说什么的。她始终会觉得怎么样？你是真的很认真地在做，你很用心地在做，她也会愿意去考虑这件事情。

很多时候，我们说人的安全感来自于什么？

来自于别人有强烈的欲望，或者说有强烈的愿望。他想要怎么样？他想要让你过得更好。或者说他强烈的愿望，想要怎么样？想要跟你在一起。他为了“想要跟你在一起”这个目标，他愿意付出很多的那种努力。

## 核心总结：没有人愿意跟随看不到希望的人

这是我们说三条，就是你这三条都不满足的话：第一个，你不考虑你能承诺对方什么；第二个，对方觉得你没有兑现承诺的能力；第三个，也感受不到你为了去兑现你的承诺，会去付出努力。

也就是说，很简单：**没有人会跟着一个在对方身上看不到希望的人**。

你们要发现，不管是公事也好，个人也好，包括谈恋爱这些事情也好，你想要别人跟着你的话，至少要让别 人在你身上看到希望嘛。他如果在你身上看到希望的话，他愿意在你身上花时间吗？

没有人，尤其是男的向女的表白，你要让这个女孩子以后下半辈子跟着你。在你身上她如果看不到希望的话，她不可能愿意跟着你。这是个很简单的道理。

谁愿意跟一个在对方身上看不到希望的人？他觉得跟你在一起没盼头，没希望了，他不愿意跟着你。

当然我说的这个实际上还是有条件有前提的。所谓的前提是什么意思呢？就是“我还没有对你动感情，我还没有对你怎么样，投入感情的情况下”，我会怎么样？我就不会愿意跟着你。

但是呢，如果我已经对你付出感情了，已经对你投入了，已经把希望寄托在你身上，但是你一而再，再而三地辜负我，有一部分人会因为不甘心而耗着，而拖着。

而等待，而虚假地去相信别人。但是这种情况下，这种女生最后一定也会离开男的。但是前提是什么呢？这个条件是什么呢？就是说她一定会把自己输得彻底，比如已经输掉了感情，输掉了自信心，输掉了工作能力。

什么都输得彻底了。

## 本节课总结

本节课中我们一起学习了为什么很多人的恋爱容易“凉”的核心原因。我们剖析了三个关键点：
1.  **不考虑对方利益**：在提出请求（如约会、交往）时，只关注自身需求，未说明对方能获得的好处。
2.  **缺乏兑现承诺的能力**：言行不一，现状与承诺的未来严重脱节，让对方无法信任。
3.  **未展现为承诺付出的努力**：没有通过实际行动让对方看到你为共同未来奋斗的决心。

归根结底，**没有人愿意跟随一个在身上看不到希望和未来的人**。无论是恋爱还是其他人际关系，想要获得对方的认可和跟随，你必须通过切实的言行，让对方感受到与你在一起的**价值**和**盼头**。这需要的是真诚、尊重、责任感和持续的努力，而非浮于表面的技巧或套路。
