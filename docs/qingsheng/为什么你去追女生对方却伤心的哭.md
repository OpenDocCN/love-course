# 为什么你去追女生对方却伤心的哭

在本节课中，我们将要学习人际交往中一个核心但常被忽视的概念——“体面”。我们将深入分析为什么缺乏体面可能导致你的追求行为适得其反，甚至让他人感到伤心。课程将用简单的语言和逻辑，帮助你理解“体面”作为社交基础的重要性。

## 体面是第一要素 🎯

上一节我们概述了课程主题，本节中我们来看看“体面”为何如此关键。体面是人际交往的第一要素。如果你没有钱、没有面子，你和谁交往都不会顺利。甚至，如果你形象太差、社会地位太低，你去和女生说话都会引发问题。

女生会因此感到伤心。例如，一个在学校里被认为很“次”、很“low”的人去追求一个女生。女生伤心的原因在于，她会想：“这么差的人都敢追我，我到底是有多差？” 这会严重打击她的自信心。所以，体面是基础。

体面对于一个男人而言，意味着你是否拥有财富、事业、门面等社会资源。没有这些，你就什么都没有。只有当你自身具备了体面，你才能拥有“场面”，才能给别人面子，才能有资格对别人说“你是我兄弟”这类话。

反之，如果你自身不体面，却对别人说这种话，对方不会感到荣幸，反而会看不上你、瞧不起你。你表达“我愿意跟你做一辈子兄弟”，对方心里想的可能是“我才不愿意”。

## 如何获得体面？🔑

理解了体面的重要性后，我们自然会问：如何变得更体面？方法其实很简单。要得到别人（包括女生）的认可，核心不是去“得到”这个人本身，而是首先要得到她的认可。

以下是获得认可与体面的关键步骤：

*   **首要目标是获得认可**：你追求一个女生，目标不应该是直接“得到她的人”或“得到她的亲吻”。这些都没有用。你必须先获得她以及周围人的认可。得不到认可，就谈不上体面。
*   **认可赋予话语权**：举个例子，如果你们所有人都认为我是个傻瓜，你们会听我讲课吗？只有你们首先认可我，我讲的话才有价值。如果你们不认可我，我讲的一切都是废话，毫无价值。
*   **展示你的“门面”**：我们看到很多开设课程、进行个人品牌建设（PUA）的人，他们做的第一件事就是包装自己。他们展示自己多么成功、多么富有、拥有多少资源、与多少优秀女性合影。这本质上是在展示他们的“门面”和“面子”。有了这个面子，他们说的话才显得正当，才有人愿意听。人们会觉得：“这个人又有钱又厉害，他说什么都是对的。” 许多人会因此放弃独立思考，盲目跟随。

所以，体面是第一要素。我们可以将这种现象在更高层次上理解为“社交货币”。

## 体面即社交货币 💰

上一节我们探讨了获得体面的方法，本节中我们来看看“体面”在社交中的本质——它是一种“社交货币”。

面子就是社交货币。你有面子或没面子，直接决定了别人对你的态度。例如，一个身家一亿的人，愿意和身家只有一千万的人平等地交朋友、谈生意吗？他可能会觉得对方只能做自己的小弟或附庸，无法进行对等交易。

再比如，我的公众号有十万粉丝，你的公众号只有一千粉丝。你来找我做文章互推，我根本不会理你。所以，面子实际上就是一种社交货币。

网络上这种现象也很普遍：拥有一万粉丝的人只和同级别的人交流；十万粉丝以上的人有自己的圈子；百万粉丝的大V则只和同级别的人对话。这清晰地表明，面子是一种硬通货，它划分了社交圈层。

社交货币的运用方式很有意思。我们需要理解，面子在本质上是一种“惩罚机制”，而不是简单的加减法。

以下是理解“面子即惩罚”的关键点：

*   **面子非加减，而是惩罚**：面子不是给多给少的问题，而是“有”或“无”、“对”或“错”的问题。做错了，就是完全的负面效果。
*   **典型案例**：例如郭德纲和某位弟子的故事。郭德纲成名后提携（给面子）这位弟子，但弟子后来翻脸并在网上攻击他。这对郭德纲而言就是极大的“丢面子”，是一种惩罚。
*   **错误示范的惩罚**：你在公众场合想表达对寿星的重视，却送了一块“钟”（谐音“送终”），这就彻底得罪了人。你的本意是给面子，结果却变成了负面事件。这就像“马屁拍到了马腿上”，只要做错，就是全盘皆负。
*   **现实中的例子**：你结婚我随礼一千元，等我结婚你只回五百元。你这五百元还不如不回。这说明了面子不是多少的问题，而是“有没有做到位”的问题。
*   **跪舔行为的本质**：很多人在女神面前卑躬屈膝、疯狂付出（跪舔），但对方却不领情。就是因为你在她面前“没面子”。你约她，她不理会；你说话，她不回应。这本身就是“没面子”的负面状态。此时，你继续拼命送礼、发红包，这些“正面”行为并不会抵消“没面子”这个负面状态，反而会因为你姿态过低而加重负面评价，让对方更看不起你。**这完全是一种惩罚性的循环。**

所以，如何获得他人的认可？面子是自己挣来的。一个人获得认可，首先在于他能做好自己，能把事情做好。如果连自己都做不好，事情也做不好，就无法获得正面的评价和体面。

## 课程总结 📝

本节课中，我们一起学习了人际交往中“体面”的核心作用。

我们首先明确了**体面是建立一切社交关系的第一要素**，缺乏体面不仅难以获得尊重，甚至可能因追求行为伤害他人的自尊。

接着，我们探讨了**获得体面的途径在于赢得他人的认可**，而非直接追求结果，并通过展示个人价值（门面）来建立初始信任。

然后，我们深入分析了体面的本质是一种**社交货币**，它决定了你的社交圈层和他人的对待方式。最关键的是，我们理解了面子机制更像是一种**惩罚机制**而非加减法——社交行为一旦失当，就会产生全面的负面后果，正如“跪舔”行为只会加剧负面评价。

记住，**真正的体面来源于做好自己，成就事情**，从而由内而外地赢得认可与尊重。
