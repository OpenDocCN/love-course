# 认识了女生却因不得其法，导致你

在本节课中，我们将学习如何运用《矛盾论》的核心思想，来分析和解决在认识女生、与女生交往过程中遇到的常见问题。这套思维方法不仅适用于情感领域，更能帮助你理清生活中各种复杂问题的脉络。

## 背景与核心思想

上一节我们介绍了课程的主题。本节中，我们来看看《矛盾论》为何能成为我们解决问题的强大工具。

我主讲感情话题，用矛盾论来讲脱单问题，堪称“高射炮打蚊子”，是大材小用。但之所以要讲，是因为这套理论的价值远超脱单本身，对生活的方方面面都有非常积极的作用。

今年是建党一百周年。我回顾了《觉醒年代》、《大决战》等作品，并研读毛泽东的《实践论》、《矛盾论》和《论持久战》等文章。我被主席清晰的分析问题思路和方法所震撼。他后期的指导思想很大程度上源于《实践论》和《矛盾论》，并由此衍生出《论持久战》，指导了后续众多工作。

我们不仅要缅怀伟人，更要学习其思想方法，这非常有用。对我个人而言，从20岁起就有三篇文章影响深远：一是鬼谷子《本经阴符七术》中的《养志法灵龟》，讲的是如何调节心灵与行为，发挥主观能动性，需减少欲望、坚定志向。这我在24岁时领悟。其二便是《矛盾论》，它在24、25岁时对我影响巨大。

《矛盾论》有一个核心思想：**抓住并解决主要矛盾，次要矛盾便会迎刃而解**。这也是本节课的核心主旨。我们要解决当下的主要矛盾，后续问题就容易处理。同时，矛盾会发展，在不同时期会面临不同问题。

因此，我经常强调两点：
1.  要客观地看问题。
2.  要具体问题具体分析。

我反感别人一上来就问“怎么办”。因为在不掌握具体情况时，无法给出答案。就像追女生，如果连对方的喜好、生活环境、状态都一无所知，问题就无法解决。幻想通过一套固定技巧解决所有问题，这正是《矛盾论》所指出的 **“教条主义”** 错误。

## 从历史看思想流毒的影响

理解了核心思想后，我们通过历史来看看错误思想的危害。

为何要讲这些？回顾1919年前后，中国社会巨变。一个国家发生巨变，首先从哪里开始？当然从内部开始。内部的哪里？思想。为什么会有五四运动？是因为之前的新文化运动。新文化运动为何发生？是因为国家破败，仁人志士寻求救国之路。这是旧矛盾催生新事物，是矛盾论的观点。

之后，从建党到第一次国共合作，再到1927年“四一二”反革命政变。这是因为党的创始人陈独秀犯了 **“右倾投降主义”** 错误。随后在三十年代，博古、王明又犯了 **“冒险主义”或“教条主义”** 错误。这些错误的指导思想都导致了革命的严重挫折。

这说明了什么？如果党内的指导思想有问题，就会导致现实实践的失败。映射到个人生活：如果你目前不成功、失败、苦闷、看不到希望，必定是你的思想中存在需要肃清的“流毒”。党内正是在肃清了这些思想流毒后，由毛主席主持工作，形势才逐渐好转，历经抗日战争、解放战争，最终取得胜利。

我们缅怀过去，继承遗志，更要学习他们的思想。今天这堂“高射炮打蚊子”的课，就是希望大家掌握这些思维方法。

## 你的脱单困境诊断

背景介绍完毕。现在，我们具体分析你在脱单中可能遇到的困境。

以下是几个常见问题，请对号入座：

*   **生活圈子狭窄**：社交圈很小，一年到头认识不了几个人，更别说认识女生了。
*   **社交时容易失语**：好不容易有接触女生的机会，却容易紧张，不知道该说什么好。
*   **交往方法错误**：认识了女生，却因方法不当导致交往不顺。根本原因是对女性不够了解。你的方法总会犯错，女生会因各种原因冷淡或抗拒。你又错误解读这些信号，导致行为上要么冒进（冒犯对方），要么退却（让对方觉得你并非真心喜欢），最终关系不了了之。
