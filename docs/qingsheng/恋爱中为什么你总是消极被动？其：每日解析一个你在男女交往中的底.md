# 恋爱中为什么你总是消极被动？其：每日解析一个你在男女交往中的底

在本节课中，我们将要学习一个在男女交往中常见的心理现象：消极被动。我们将深入剖析其背后的底层逻辑，特别是“核心欲望”如何影响你的行为模式，并探讨如何走出这种低能量状态。

## 概述：消极被动的表现与根源

很多人会发现，自己在面对感情或社交时，总是处于一种消极被动的状态。这并非简单的性格问题，而是一种行为结果，其根源往往与无法跨越的压力点有关。

某一个压力点上她过不去。就好像我们要扛起一百斤的东西，你就是扛不起来。你到那个压力点，你就是撑不过去。你撑过去了，实际上就没事了。所以我们就确实也可以看到很多人遇到一些事情之后，比如一次挫败、一次失恋，原本开朗的性格就会变得压抑，天性无法得到释放。

谁都会有这么一个过程，从而变成一个谁都不喜欢的闷葫芦。原来你小时候可以的，但是你遇到一次挫败之后开始对自己失去自信。然后你原来开朗的性格在女生面前，你不是很能聊很能谈吗？然后变得压抑了，你天性就释放不了。每一个人都会有这个经历的，需要很长的时间去克服。如果说你用错了方法的话，你需要很长时间去克服，因为你没有办法去过压力点。

## 第一节：低能状态是一种结果

上一节我们提到了压力点，本节中我们来看看“低能状态”的本质。首先需要明确，低能状态是一种结果，而非原因，并且它不受主观意志的直接控制。

所以低能为什么会成为一种状态？一个人低能是结果。**低能状态 = 行为结果**。

以下是低能状态的两个关键特征：

1.  **低能状态不可由主观意志控制**。我觉得我现在能量状态很低，我想变得高能不是不可能的。你比如说我考试前我要不紧张，你能做到不紧张吗？我见到女孩说话，我能够我很放松，你能做到放松吗？这不是由主观意志控制的。
2.  **低能状态是由于选择了消极被动的行为方式**。你的行为方式都是消极的，你从来没有主动积极的部分在里面。真的，你就是那种很消极的那种人。

## 第二节：消极被动行为的具体表现

理解了低能状态是结果后，我们来看看导致这种结果的“消极被动行为”具体有哪些表现。这些行为模式会像枷锁一样，将你困在原地。

为什么说消极？就是说你连聊个天，你连去搞个什么东西，你都要完全去学别人的，你完全一点自己的思路都没有吗？你一点自己的想法都没有吗？所有消极被低能状态，往往是由于你选择消极被动的行为。

以下是几种典型的消极被动行为：

*   **在社交中等待**：你想去撩女生，你永远在等这个女生给你发信息。
*   **在目标上言行不一**：比如说你想减肥，你却每天都在大吃大喝。
*   **在机会前退缩**：你看到漂亮的女生，你不敢走上前去。
*   **在责任前逃避**：你不去做你应该做的事情，或者说是你不把你应该要做的事情做好，然后你就去急着去玩，就去干别的事情。

也就是说你在生活当中，你不去做你应该做的事情。这会很容易导致问题。你比如说你的社交，你说你没有朋友，实际上你都是选择了消极被动的社交行为方式。你的社交问题出在哪里？你都是选择在社交上，你消极被动的行为方式。你比如说你要有朋友，你要有朋友圈，你得自己主动去选择去建立。

## 第三节：问题的核心——欲望出了错

上一节我们列举了消极行为，本节我们将深入问题的核心：所有这些行为背后，是你的“核心欲望”出了问题。这是消极被动的根本原因。

你消极被动的行为是什么？是因为你核心欲望出了问题。明白了，就是说你消极被动的行为方式，就是因为你核心欲望出了问题。

**核心欲望**：我做一件事情，我一定要达成目标。这叫核心欲望。你比如说我要找女朋友，我一定要给自己找个女朋友，这叫核心欲望。你有了核心欲望之后，你就会有行动力。

有很多时候，我们形成了消极被动的行为方式。你比如说你减肥，要跑步可能会很累，对不对？但是你跑完之后，你会不会觉得自己很爽，然后你睡觉都睡得踏实一点，心里舒服一点？因为你做了自己该做的事情，你觉得自己减肥有希望的，至少我去做了，比我没去做好，你给自己有一个交代。

你低能的状态很有可能就是你选择消极被动，你有问题不去解决，所以说你的核心欲望就会出问题。你如果面对你的所有欲望的时候，你面对你所有你想要的东西的时候，如果你都选择消极被动的行为方式，你就会变成依赖，变成等，就变成逃避问题。你的核心欲望当然会出问题，就会变成一种**需求**。

你的欲望就会变成需求。就说我想赚钱，我想泡妹子，我想怎么怎么样。你们想我想赚钱，我想泡妹子，我想要好的工作，然后我想要好的朋友，你们是不是都有这种需求？你买那种口才社交的书学习，这叫什么？这叫被动消极。为啥？因为你解决自己欲望的方式，开始高度依赖外界。

## 第四节：依赖外界与缺乏内驱力

当核心欲望变成被动需求，人就会开始高度依赖外界解决方案，并严重缺乏内在驱动力。这是陷入消极循环的关键一环。

因为你每一件事，你比如说你想找妹子，你不好好去找，不好好去收拾自己，不好好去锻炼自己，不好好去整理自己的形象，不好好去跟妹子主动去联系。然后你要减肥，你又不去跑步，你天天蹲在家里面喝肥宅快乐水。你要交朋友，你又不愿意主动对别人好。你想赚钱，你又不好好想想，去尝试哪些事情可以赚钱。

你如果四件事情全部都是，你自己又有这个欲望，然后你行为上你选择这种消极被动的。妹子喜欢你完全跟你什么？你的穿着品味，你的谈吐气质，你的学识，你的见识，你的整个人的冲劲，你的状态，完全是个综合体。聊天是个最不重要的地方，而你却觉得我就不会聊天。

所以你如果说你的核心欲望出了问题的情况下，你解决自己欲望的方式，又高度依赖外界，比如说依赖于看书，依赖于别人讲课。因为什么？因为你严重的缺乏内在的驱动力。就是你不愿意主动去对人好，你不愿意主动去交朋友。因为你在交朋友这方面，你选择了消极被动的行为方式，所以说你总等着别人来主动来跟你交朋友。

你严重缺乏内在驱动力是什么？是你总是思考你的一些行为和做法，**是否有意义**。这就是低能状态。你做一些事情，为什么思考是否有意义？就是你觉得我自己做不到，我做不好，然后我就不去做。然后就会自己不想做，不想做的话，然后好不容易做了一下，觉得自己这个东西到底有没有意义，要是没有结果的话，那我就是白费心。你根本就没有去做这些事情的欲望。

## 第五节：恶性循环与“意义陷阱”

缺乏内驱力并不断质疑行动意义，会让人陷入一种能量越来越低的恶性循环。我们必须警惕这个“意义陷阱”。

久而久之，一个人的低能状态所养成的行为，就会成为一种惯性。整个人的能量状态会越来越低，越来越没冲劲，越来越“油”。你的整个人就成为一种惯性，等于整个人的能量状态越来越差。

因为你的腺体分泌出了问题。你看思考是否有意义，这个行为蛮正确的？我告诉你这个事情是什么，是最没有用的。一个人如果说做每件事，你都去思考这件事情意义的话，那就你基本上什么事情都做不成。人以所有东西只有该不该做，没有做这件事情有没有意义。

如果都照你这么说的话，那我们抗日战争是怎么办？抵抗没有意义，反正我们打不过，那都投降了，对不对？所有事情都是什么？不是意义论是由于什么，是你到底想不想做。

## 第六节：如何走出低谷——提升行动力

分析了问题的根源和恶性循环后，现在我们来看看解决方案。改变的关键在于**提升行动力**，用积极的行为反向塑造思维和状态。

有人说叫行为改变思维。就是说你做出积极主动的行为的话，你的腺体就会分泌，你整个人状态都会改变。所以如何走出这种状态？从提升行动力开始。

以下是五个具体的行动步骤：

1.  **面对每一件事，选择积极行为**：对每件事情你要选择积极的行为。比如说你这段时间你想去认识女生，那你搭讪你就应该上。把这种行为上的选择形成习惯。做每一件事情，你习惯去选择这种积极的行为。
2.  **尽可能加快生活节奏，不要拖延**：一个人的高能，第一个是尽量选择积极的行为，第二个是你要加快你的生活节奏，不要拖延。你想想一个人生活节奏快的话，他能量是不是自然高？你每天奔走如飞，走路带风的情况下，你是不是就能量状态要高一点？
3.  **不要养成依赖和放弃思考的习惯**：不要凡事依赖，放弃思考和处理的习惯。你永远什么事情都不去处理，你养成这种习惯了，就想什么事情都不要用脑子，什么事情别人填鸭式的教你。
4.  **合理安排规划，形成清晰目标**：合理安排和规划自己的生活，形成清晰的目标。你每天你小阶段的目标是什么，大阶段的目标是什么，你一定要合理安排和规划。
5.  **不要害怕失败**：你们现在很多人为什么低能？就是因为害怕失败。我告诉你们真正牛逼的人是什么？不是百战百胜的人，而是屡败屡战的人。失败只要不是元气大伤型的，其实都没什么。往往是说你现在什么都不是的时候，屡败屡战实际上是一种英雄情结。
