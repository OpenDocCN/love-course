# 课程名称：情圣是怎样炼成的 - 第一课：超越技巧与套路 🧠

在本节课中，我们将要学习追求异性时常见的思维误区，并探讨如何通过提升自我认知与思维方式来建立更健康、更有效的人际互动。核心在于理解，仅凭技巧和套路难以获得真诚的关系，个人成长与成熟的心态才是关键。

## 自我提升的两个核心问题

你们需要解决两个根本问题。以你们目前的状态，想找到一个优秀的伴侣可能性很低。你们的收入水平和所处的社会层次是现实因素。

因此，你们现在要为寻找伴侣做两方面的准备。

以下是你们必须解决的两个核心问题：

1.  **社会层面的提升**：在这个社会上，你必须能够出人头地，必须去提升自己。这是首要任务。不要幻想仅靠学习技巧和套路就能找到一个优秀的伴侣，这不可能。
2.  **心理层面的改变**：你们必须改变心理上“巨婴”的问题。不要让女生觉得你幼稚。即便处于同一社会阶层，女生也可能因为觉得你幼稚、不像个男人而不喜欢你。她会觉得你是个需要被时刻照顾感受的“巨婴”。

上一节我们介绍了需要解决的两个核心问题，本节中我们来看看为什么“巨婴”心态会导致失败。

## 为何“美女难追”？视角的误区

你们之前觉得美女难追。今天我讲完之后，你们还觉得难追吗？原因在于你们只站在自己的角度想问题。

什么叫站在自己角度？什么叫幼稚？

**幼稚**的定义是：我想追一个女生，只思考用什么技巧和套路去搞定她。这就是典型的幼稚，是只站在自己角度的表现。

你不从女生的处境和她的想法出发，能把事情想明白吗？今天我这样一讲，逻辑就非常清晰通顺。如果我只站在自己角度，想着用什么方法套路，那永远无法真正理解对方。

## “聊天没话题”背后的幼稚思维

这就引出了为什么总有人问“聊天没话题怎么办”这类问题。以下是这类问题为何无效的分析：

*   问“聊天没话题怎么办”的人，通常期待一个答案：第一句说什么，第二句说什么。这就是典型的幼稚思维。
*   会问这种问题的人，往往追不到女生。因为你不思考女生感兴趣什么、关心什么。
*   换位思考：如果女生找你聊天，你希望她聊什么？你希望她来关心你、问候你，或者向你倾诉。
*   但当你作为主动方时，女生想的是：“这个男的对我是认真的吗？”女生有时不一定非要聊什么具体内容，她可能只是想看到你努力找话题的样子。即使她回应冷淡，但她想看到你的努力。
*   所以，聊什么内容并不最重要。如果你一开始就放弃努力，直接问“聊什么”，这与你应该展现的状态完全不符。

## 巨婴思维：缺乏主见的根源

所以说你们幼稚。这源于一种“巨婴思维”，即巨大的婴儿思维。

以下是巨婴思维的表现与后果：

*   **不愿动脑**：总想着可以问别人，自己不愿思考。
*   **与女生期望相悖**：女生希望你努力、用心、费心思。就像让你寻找稀世珍宝，你一开始就不愿努力。
*   **在女生眼中的形象**：不动脑子，在女生看来就是没主见。一个没主见的男性，不能称之为真正的男人。我们会认为这种人“不靠谱”、“不行”。
*   **社交后果**：这种没主见的人，通常不受欢迎，别人看不上他。

## 如何成为有主见的成熟之人

那么，什么叫有主见，什么叫没主见？

**没主见**意味着：遇到事情，我不知道该怎么办。那些经常问“聊什么呀”、“怎么办呀”、“她怎么想的呀”的人就是如此。

**有主见**意味着：遇到任何事情，都能自己理清一二三，想清楚应该怎么办。这才是一个成熟的人应有的状态。

不成熟或“不像男人”的表现，就是不愿动脑筋，因为动脑筋很累。总想着“我直接捡现成的多好”，这种心理导致女孩子不喜欢你。这是你们一贯的态度造成的。

## 用心投入：摆脱巨婴状态的关键

你会发现，很多事情如果你不用心去做、不去想，就会觉得一点意思都没有。但只要你认真去做、用心去想，就会发现这件事很有意思。

就像你们听这节课，虽然我讲的内容不是娱乐八卦，但你们听下来会觉得有趣，心想“居然还会这样”。这是因为我花了心思准备，而你们也积极参与了。

所以，你们要做的第二点就是：**摆脱巨婴状态，学会独立思考与解决问题的能力**。

本节课中我们一起学习了追求异性时不应只依赖技巧，而应关注社会与心理的双重提升。我们剖析了“幼稚”和“巨婴思维”的表现，明白了只从自我角度出发、缺乏主见是关系发展的主要障碍。关键在于转换视角，用心投入，并培养独立思考和解决问题的能力。
