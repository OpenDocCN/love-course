## 什么是“跪舔”？它并非具体行为

首先需要明确，“跪舔”不是指某个具体的行为。例如，接送女生上下班或赠送礼物，这些行为本身无法被定义为“跪舔”。同样，主动与女生搭讪也不等同于“跪舔”。

“跪舔”本质上是一种心理障碍。它是一种会对个人结果产生实质性影响的内心状态，而非外在的单一动作。

## “跪舔”的心理学定义：人际防御

那么，这种心理障碍具体是什么呢？在心理学的人本主义流派中，“跪舔”被定义为一种**人际防御**状态。

具体来说，它是人际防御在初期的表现形式。其核心特征是：**当你试图与人亲近时，会表现出过度的顺从，并在此过程中抹杀真实的自我**。

例如，在社交中不断迎合对方、不敢表达不同意见、总是以低姿态出现，这些都是“跪舔”的典型表现。心理学早已对此有过研究。

除了“跪舔”，人际防御还有另外两种常见的表现形式：**对抗**和**疏离**。

以下是这三种表现形式的例子：
*   **对抗**：表现为攻击性、傲慢或持有“女孩不是靠追而是靠吸引”、“自古深情留不住，唯有套路得人心”这类观点。
*   **疏离**：表现为远离人群，持有“追什么女孩，是酒不好喝还是游戏不好玩？”这类想法。

无论是“跪舔”、“对抗”还是“疏离”，其本质都是同一种心理状态——人际防御，只是在网络和现实中呈现出不同的面貌。

## 人际防御的根源：缺乏无条件的接纳

上一节我们介绍了人际防御的表现，本节中我们来看看这种状态是如何产生的。

根据人本主义心理学家的研究，无论是儿童还是成人，出现人际防御状态的核心原因在于：**个体感受不到来自家人或重要他人的无条件接纳和认可**。

许多人可能都有这样的体会：与家人或亲近的人无法进行深层次的内心交流。你总觉得对方不认可你，从而在心理上产生了隔阂。

相反，对于那些让你感觉无论自己好坏都会被接纳的人，你更容易流露真情实感。而对于那些让你感觉需要小心翼翼、注意言行以免被嫌弃的人，你就会不自觉地处于一种神经紧绷的**人际防御状态**。

这种状态可以概括为一个公式：
```
感受不到无条件接纳 -> 人际关系安全感缺失 -> 进入人际防御状态
```

在成年后与异性交往时，这种状态会表现得尤为明显。即使对方最初对你有好感，你也会害怕随着关系深入，对方会越来越不喜欢你。你认为被喜欢和接纳是“有条件的”，这种认知导致你长期处于防御状态，过度修饰形象，无法建立真正的亲密关系。

## 人际防御的核心问题与影响

人际防御最大的危害在于：**它会阻碍真情实感的表达和流露，从而使人无法建立真实有效的亲密关系**。

许多人都有这样的痛点：与女生相处时，总觉得无法表达真实自我。无论聊了多久、送了多少礼物，甚至接触了两三年，仍感觉心和心之间的距离从未拉近。你付出很多，对方却无法给予情感反馈，正是因为你们之间没有建立真实的亲密连接。对方可能也很着急，但就是“没感觉”，根源在于你处于人际防御状态。

从心理学描述来看，人际防御会导致个体在亲近他人时，要么过度顺从、抹杀自我（跪舔），要么表现出对抗或疏离。这种防御机制会使个体形成一种不切实际的、理想化的自我形象，并试图通过行为来强化它。

例如，有些人会认为自己“应该”完美、慷慨、有吸引力。为了维持这种形象，他们会在人前有意无意地表演，结果反而导致内心深处的自卑。越是追求完美形象的人，往往自卑感越深。

在现实生活中，这常表现为“价值扭曲”。例如，有些远离群体的人，并非单纯弱小，他们可能在心底认为群体中的其他人都是“傻的”，通过贬低他人来维护自己脆弱的自尊体系。这是一种僵化的、运作不良的自尊和尊严体系。

## 根本原因：低自尊与低自我效能感

人际防御（包括跪舔）最根本的心理原因是**低自尊**。

**自尊**的定义是**自我价值感**。**低自尊**即**自我价值感低**，也就是我们常说的自卑。

低自尊的状态并不总是稳定的。同一个人在不同环境下，自尊水平可能不同。例如，一个来自农村的大学生，在大学集体中可能显得木讷、拘谨；但回到家乡，在同龄人中却可能生龙活虎、侃侃而谈。在亲密关系中也是如此：在“女神”面前可能紧张得说不出话，但在自己感觉有掌控感或不在乎的女生面前，却能展现魅力。

低自尊（自我价值感低）会导致**较低的自我效能感**。这两个概念有所不同：
*   **自我价值感低**：觉得自己“不配”，是自卑。
*   **自我效能感低**：觉得自己“搞不定”，是能力上的消极预期。

简单来说：**自卑会让人觉得自己没有能力，从而产生“我搞不定”的想法**。

## 低自我效能感的行为后果

低自我效能感（觉得搞不定）会引发一系列问题：

1.  **负面预期与焦虑**：觉得事情肯定会失败，从而产生焦虑。例如，追求女生时总担心被拒绝。
2.  **行为退缩**：因为觉得没希望，所以主动行为减少。例如，纠结该不该发信息、发什么、频率如何。
3.  **易怒**：容易因挫折而感到愤怒，例如觉得“你装什么装”，甚至可能删除对方联系方式。
4.  **过度算计**：把简单事情复杂化，总想着用什么招数、说什么话对方会有什么反应。
5.  **莽撞**：因自我价值感低、害怕夜长梦多而行为过激，只顾用自己的方式“对对方好”，反而引起反感。

这些状态都会强化人际防御，使你在亲密关系中无法产生有效行为。即使对方主动，你也可能因焦虑、对抗、退缩或过度算计而错失机会。

## 自我效能模型：理解行为与结果的关键

我们可以用班杜拉的**自我效能模型**来理解这个过程。低自尊会导致低的自我效能感（效能期望低），即“我觉得我搞不定”。

```
低自尊 -> 低自我效能感（效能期望低）-> 积极行为减少 -> 结果期望低 -> 难以达成目标
```

当你觉得搞不定时（效能期望低），你的积极行为就会大量削弱。你会不断内耗：“她不理我，是不是没戏了？”“她说做朋友，是不是把我当备胎？”这些消极想法会直接导致你的追求行动力下降。

而做成任何事，都需要一定数量和质量的积极行为。跪舔本质上是一种**无效行为**，因为它并非基于真实的互动和吸引，也无法提升关系。例如，很多“舔狗”连情话都不敢说，只会死气白咧地表达“深情”，这不是有效的亲密关系推进行为。

模型中另一个关键是**结果期望**，即你认为行为能带来理想结果的可能性。如果结果期望不高，你很难坚持有效的行动。而“急”的心态，往往正是因为结果期望低，害怕失败或错过，从而想速战速决，这反而容易导致莽撞行为，让关系恶化。

需要区分的是，**主观上认为“这件事成功的可能性大”**，与盲目自信的“我觉得我最牛逼”是不同的。前者是基于一定判断的理性期望，能导向有效行动；后者只是情绪上的自大，容易导致莽撞和失败。

**本节课总结**

在本节课中，我们一起学习了：
1.  “跪舔”不是具体行为，而是一种**心理障碍**，属于**人际防御**的一种表现形式（过度顺从、抹杀自我）。
2.  人际防御还包括**对抗**和**疏离**，根源在于个体**感受不到无条件的接纳**，导致安全感缺失。
3.  人际防御会**阻碍真情实感的表达**，使人**无法建立真实的亲密关系**。
4.  其根本心理原因是**低自尊**（自我价值感低），进而导致**低自我效能感**（觉得自己搞不定）。
5.  低自我效能感会引发**焦虑、退缩、易怒、过度算计、莽撞**等行为，这些都属于无效或破坏性行为。
6.  通过**自我效能模型**，我们理解了“觉得自己搞不定”会直接减少积极行为，从而降低达成目标的可能性。

理解这些概念是改变的第一步。认识到“跪舔”是一种源于内心低价值感和低效能感的防御状态，我们才能开始从根源上寻求解决之道，学习如何建立健康的自尊和有效的人际交往模式。
