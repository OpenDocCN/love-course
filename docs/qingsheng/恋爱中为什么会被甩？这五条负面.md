# 恋爱教程 P1：警惕五条负面期望效应，避免被甩 😨

在本节课中，我们将要学习人际关系中的一个核心心理机制——期望效应，并深入探讨在恋爱关系中，五种常见的负面期望效应是如何导致关系失败、甚至被甩的。理解并警惕这些负面效应，是建立健康恋爱关系的第一步。

## 人际关系的本质是期望效应互动

上一节我们介绍了课程主题，本节中我们来看看期望效应如何构成人际关系的基石。

人际关系的本质，以及我们整个生活的本质，其实是期望效应互动的结果。人和人之间、人和事之间，都是这个期望效应互动的结果。

这个解释可能有些片面，但大家可以在生活中去观察和发现。很多你自己在处理的事情，当你作为负责人或责任人之一时，你手头上的事情其实不取决于“这件事本身应该怎么做”或“怎么做是对的”，而是取决于**你对这件事情的解释**。

我刚刚已经举过例子。比如网络上，女生给你发一个笑脸，就有各种各样的观点：有人觉得这是对你有意思，有人觉得没意思；有人劝你继续，有人劝你放弃；有人劝你不要“舔”，也有人说轻易放弃太脆弱。人们会有各种各样的解释。

很多人之所以如此，是因为他们没有成型的三观，没有自己人生的原则和准则。一个有稳定三观的人，无论女生是对他笑还是哭，无论对方反应如何，他的解释可能是：如果对方有积极反应，那说明可能有进展；如果反应不好，那也很正常，事情本就不易。**三观能帮助你解释事情**，让你不至于对所有事情都陷入**负面的期望效应**中。

所以我在开课时就建议大家学点哲学。哲学能用辩证法看问题，比如“塞翁失马，焉知非福”。凡事都要学会辩证地看：好的事情有坏的一面，坏的事情也有好的一面。我也教过大家，当别人遇到不好的事情时，你要帮他找出事情本身的积极意义，这能起到安慰人的效果。心理学上有一个专门的流派叫**意义疗法**，其代表作是《活出生命的意义》。

**你怎么看待和解读一件事情，这非常重要**。因此，网上别人讲的各种观点，即使再有道理，你也不要全信。你需要发展出一套**对你自己有帮助的解释观点**。所以说，心态是第一位的。心态不好，其他都无从谈起。

但最难改变的是人对事情的解释。比如，我在收费课上经常问：人没钱就一定要去挣钱吗？没钱怎么办？很多人会立刻反驳：没钱当然要去挣，不然等着挨饿吗？或者去借钱啊。你看，对于“没钱”这件事，很多人有不同的解释，这反映出你积累了几十年的观念。

很多人都没有好好想过：**我对这件事的解释，对我的生活有什么影响？**

有人说“钱是一切的根基”，这是一种三观。我们来分析：如果钱是一切的根基，那反过来说就是，没钱很多事情都不能做。那么，按照这个观点：
*   创业一定要有钱才能创。
*   谈女朋友一定要有钱才能谈。
*   约会要钱。
*   总之，没钱就什么都不行。

这又产生另一个问题：**多少钱才算有钱？**

我给大家举两个我自己的例子：没钱的时候怎么创业，以及没钱的时候怎么“泡妹子”。

我刚开始讲课的时候有钱吗？没钱。我学生时代，初中就开始谈恋爱，那时一天生活费一块五毛钱。上大学时，一个月生活费五百块。怎么谈恋爱？怎么追女孩子？

我从学生时代就抱定一个观念：**用钱搞来的妹子都不是真爱**。所以我在追女生过程中绝不送礼物，也绝不乱花钱。我们那时约女生都是去压马路，或者带女生去网吧，甚至有时让女生请我吃饭。没钱就搞不定妹子吗？我追我老婆时也没花什么钱，我们在一起之后我才大量给她钱。

如果你认为“钱才是一切的根基”，或者认为某件事是你的根基，你就会变得**没有想象力和创造力**。我们那时没钱，追女生靠写情书。不仅要写得好，还要用漂亮的信纸，贴上花瓣，字迹工整。没钱出去约会，就约到河边，选明月当空的夜晚。听着随身听里刘若英的《后来》。如果你什么都靠钱，就会失去这些想象力。

浪漫的景色需要去找。我们那时没钱，就去火车站旁的小山上看晚霞和星星，听着火车汽笛，吹着口琴。这其实就是没钱时的浪漫。包括跟女生约会，可以约到家里，展示跟父母学的手艺，做个剁椒鱼头。想搞点浪漫，就买块猪肉当“猪扒”，配点花菜和劣质红酒。

**钱会破坏你的想象力和创造力**。关键很多时候不是钱的问题，而是你不会去创造价值。钱的本质不过是价值。我们做的这些事情，比花多少钱都有价值。**钱是你在创造不出价值时，被迫采用的最后一种解决方案**。只要脑子活，你可以想出很多比钱更有用的方式。

这只是一种观点。但如果你抱着“钱才是一切基础”的观点去生活，你的生活会很没意思，因为你无法发挥灵魂深处的东西，你会缺乏灵活性，整个人很僵硬。

我随便说的这几个点，都不用什么钱。只要你嘴巴会说，会哄女孩子，平时有点小主意，女孩子是很好“骗”的。年轻的时候就是这样。关键是你的观点是什么。

但我想告诉大家，拥有“钱才是一切基础”这个观点，对你的人生**有帮助还是没有帮助**？我只关心这一点。观点没有绝对的对错，任何观点都有可证实和可证伪的地方。但真正好的观点，应该是**对你去做事有帮助的**。

很多人其实都知道，拥有错误的价值观，会对你生活中的很多事情造成一种“迁换”（注：应为“牵绊”或“局限”），让你不会去想更多的可能性。所以，人穷不是问题，**一直觉得自己穷才是问题**。

人最可怕的不是真的缺钱，而是一种**对于金钱的匮乏感**。有的人确实对金钱有匮乏感。我从小家里条件也不是很好，但家人没让我对金钱有匮乏感。虽然家里说穷，不要乱花钱，但在吃穿用度上没亏待过我。所以我一直没有什么金钱上的窘迫感。这跟从小经历有关。

包括现在很多人老说缺钱，虽然我不是富养的，但该吃吃该喝喝，条件跟大家差不多。从我的成长经历看，我觉得很快乐，所以到现在我也没把钱看得很重。我觉得钱嘛，有本事就能挣。留多少钱，看你怎么用。像我这种投资眼光一般的人，钱无非是买房留个安全感，没觉得有特别多的用处。

## 什么是期望效应？

上一节我们探讨了观念如何影响行为，本节中我们来正式定义“期望效应”。

期望效应，这是我从百度百科引用的定义，因为我觉得基本正确。

期望效应本是一个心理学名词，自从被国外学者引入企业管理实践后，就不断受到企业界和理论专家的关注。期望效应在教育学和心理学领域已得到广泛认同。

教育学领域有一个经典例子：把60个学生分为A组和B组，告诉老师A组是智商较高的尖子班，B组是普通班。到最后，A组所有人的平均成绩确实都比B组高。这就是期望效应——老师认为他们行，他们就真的更行。

我再举一个个体的例子。我小学时有个同学，我并没觉得他数学成绩多好。但有一次数学老师说：“张焕同学（假设名），我觉得你数学很有天赋，学数学特别有能力。”这个人后来到初三时，其他成绩都差，但数理化，尤其是数学，拿了一个奥林匹克奖。我记得我们那时人均考40多分，我也参加了，考了40多分，他考了80多分。这就是期望效应。

期望效应又称**皮格马利翁效应**。在动物实验中也反复验证过。包括我上次讲《心理学与生活》这本书时，作者强调在心理学实验中要避免期望效应，因为很多被试者会考虑实验者希望他做到什么，然后就去做什么。一个是期望效应，一个是安慰剂效应，要特别注意这两种心理效应，它们对人影响很大。

期望是对自己或他人的一种判断。比如，你觉得那个女生不喜欢你，或者你觉得你搞得定那个女生，这就是一种判断。这种判断是基于潜意识层面（或者说内心直觉）的。比如，你觉得一个女生会不会喜欢你，这东西不由你的意志改变，它是客观的。你的理性可能想去否定自己的直觉。

所以，在网上问“这个女生到底喜不喜欢我”时，你其实很乱。但实际上，你心里问这个问题时，直觉上就知道答案——她并不喜欢你。这不是说什么第一感觉或第六感，你感官上是知道她不喜欢你的，所以你才需要反复判断。

你搞不清楚，觉得女生好像对你有意思，但又感觉哪里不对劲。这种“不对劲”的感觉就说明：**她其实没那么喜欢你**。她要是真的喜欢你，不会是那个状态。

但是，很多人会觉得：她要是没那么喜欢我，那我继续就没意思了。这就是**你对这件事情的解释和态度**。你自己怎么做，是你自己的事。但你要想一想，你这个态度对于你跟她的关系到底是好还是不好。

你所有事情都应该基于一个考量：**你的这个观点和态度，对于你要做的这件事，是有正面意义还是有负面意义？** 你唯一要考虑的是这个。

“它符合事实”不重要，因为事实可以改变。比如她现在不喜欢你是个事实，但有个时间维度问题——她现在不喜欢，以后可以变成喜欢。所以我为什么常说，女生想什么、怎么看你不重要，关键是你想要什么。你怎么去解释这些事情，怎么看待这些事情，这很重要。因为做这件事的人是你。如果你自己的整个状态都不对，你怎么可能把事做成？

所以，很多人老是忽略自己的状态，只关注用什么方法和技巧。固然方法和技巧不是不重要，但你们有没有发现，网上去学的那些方法和技巧，跟你的状态不匹配，用起来就觉得怪怪的？你不理解，为什么你整个状态和说的话、用的方法完全不匹配。

所以你们老是想去学聊天方法、技巧，为什么我经常劝你们不要去？因为你去学这些东西，本身也说明你对这件事有负面期待。你越去学，越是证明你觉得凭自己的能力搞不定这件事。

我告诉大家，不管你学还是不学，至少先自己想一想，先试一试，先努把力。不要老是说自己不努力，就想要结果，那样什么方法都帮不了你。

## 负面的期望效应为何会导致失败？

上一节我们定义了期望效应，本节中我们来看看负面的期望效应具体如何导致恋爱失败。

如果你对一件事有负面期望效应，会导致以下五个问题：

以下是五个具体的负面表现：

1.  **焦虑**
    一旦你对一个女生开始上心、在乎，本来聊得好好的，之后就会变得不知道怎么跟她聊天。只要你上心了，在乎了，就会焦虑。

2.  **失去主见**
    你会变成“当局者迷”的状态。你会不断琢磨：他到底什么意思？他今天为什么这样？为什么要那样？如果你对你们俩的关系和感情很有信心，你会老是猜这些东西吗？你会总想去学技巧和套路吗？只有当你对交往抱着负面期望时，才会去学这些。学技巧和套路代表你完全否定了自己。当你完全否定自己时，就会产生问题：你不知道该怎么办。按原来的路数，你觉得搞不定；按新学的东西，你觉得别扭。两者就会“打架”，产生所谓的PUA或“泡学”技巧中毒的状况，搞得自己都不知道怎么办。这就是典型的“邯郸学步”，或者说“东施效颦”。人家西施生病难受的样子美，东施去学，却没考虑自己的情况。很多人去学技巧套路时，没有提前照照镜子看看自己：你是哪根葱，人家又是哪根葱？为什么人家行你不行？

3.  **没事搞事**
    负面期望效应会让你在过程中总觉得人家对你眼神不对、回话太慢，然后开始纠结：他是不是不喜欢我？我是不是要去试探一下？有的人甚至会去学肢体语言解读，比如：他杯子靠得近说明喜欢我，离得远说明不喜欢我。万一人家只是无意把杯子放远了呢？你心里又在乎、受影响：我是不是该放弃了？整天纠结这些问题。本来不想还好，一想就乱了。因为害怕对方对自己失去好感，所以总想搞点招数。

4.  **过度担心**
    比如女生可能有点生气，暂时不理你了。好多人就会来问：我怎么挽回？至于吗？女生稍微不高兴、不理你，你就急得要死。甚至有的人极端到：搭讪女生没给电话，他问怎么挽回。我笑死了，我说你到人海茫茫中去捞这根针吗？他老想挽回，这叫过度担心。

5.  **急于求成**
    急于求成是因为担心夜长梦多，于是不顾对方感受，追得太猛。比如，不停地送礼物，没日没夜地找对方聊天，没话找话。这都不顾对方感受，追得太猛。

这些问题的背后本质都是**负面的期望效应**。所有追女生时的错误操作，根源都在于此。它会让你没事搞出很多事，让你在该做事情的时候不知道怎么做（比如明明聊得好，一上心就不知道聊什么），也会让你去做一些不该做的事情，同时不去做该做的事情。

这就跟阳明心学说的“**人欲过甚，则天理不纯**”一样。你被人欲控制了，天理就不纯了。什么是天理？天理就是“行所当行”。我跟大家讲过很多次，一个人什么是天理？就是做你应该做的事，同时不做你不应该做的事。你在每件事上用常识去判断，基本不会出错。

比如，你害怕开启话题，不知道聊什么。在这种情况下，很多人会想：我应不应该聊呢？虽然不知道聊什么，但我应该去找他吧？可状态这么差，还要找他聊吗？要不就算了吧。那应该怎么做？

其实我告诉大家，这个问题很简单：**你的第一反应告诉你应该怎么做，你就怎么做**。这很简单，骗不了人。你要是想多了，反而没答案，但你的第一感觉永远会告诉你。

还有一种方法是**抛硬币**。硬币正反不重要，而是在你抛出去的那一刻，你就知道了。比如，你抛的时候希望它是反面，那就说明你现在不应该聊；你希望它是正面，那就应该聊。这样是不是特别符合你的情绪和状态？

所以，每件事当中，没有一个绝对的对错。对错要“反求己心”。这就是为什么我建议大家去听我的“阳明心学”课。如果你从头到尾听一遍，就会知道：原来好多事情没什么，不要听别人怎么说，因为问你的心是最明确的。

这个东西，很多人觉得是最容易最简单的，但生活中很多人跟你讲的东西，其实是在把简单事情搞复杂。真的，是把简单事情搞复杂。但很多人分不清什么是人欲，什么是天理。

你看，很多人会有过多的思维和头脑。其实，循序渐进、一步一步来可以；单刀直入、直捣黄龙也可以。怎么说呢？这个时候问你的心啊，你觉得到哪一步了？你觉得应不应该？没有固定的东西。

很多时候，人老是想要一个固定的东西，按照流程和套路走，以为就能得到想要的。但在生活中，简单具体的事情可以这么做，比如怎么煮饭、怎么炒菜。但大家有没有发现，一旦涉及到**不确定性很高、系统性很强、变化很多**的事情时，固定的套路往往失效。

本节课中我们一起学习了期望效应是人际关系的核心，并深入剖析了五种负面期望效应（焦虑、失去主见、没事搞事、过度担心、急于求成）是如何在恋爱中导致失败和被甩的。关键在于，我们需要审视自己对事情的解释和态度，培养积极的心态，并学会在行动前“反求己心”，而不是盲目依赖外在的技巧和套路。
