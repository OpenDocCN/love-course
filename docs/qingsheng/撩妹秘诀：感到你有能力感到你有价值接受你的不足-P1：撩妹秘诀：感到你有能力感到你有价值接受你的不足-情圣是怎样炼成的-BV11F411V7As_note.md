# 课程 P1：撩妹秘诀：感到你有能力、感到你有价值、接受你的不足 🧠

在本节课中，我们将要学习自信的三个核心构成要素。理解这些要素是提升个人魅力和社交能力的基础。我们将逐一拆解“自我效能感”、“自我价值感”和“自我接纳”这三个概念，并探讨它们如何共同作用，塑造一个人的自信。

## 什么是自信？🤔

上一节我们介绍了课程的主题，本节中我们来看看自信的具体定义。自信并非一种模糊的感觉，而是由三个具体方面构成的。

首先，人的自信来自于三个点。第一个叫**自我效能感**。

### 1. 自我效能感：感到自己有能力 💪

自我效能感，即感到自己有能力。例如，在最简单的打游戏或参加体育活动时，这种感受会非常明显。

**公式：自信₁ = 自我效能感（感知到的能力）**

比如我去踢足球就很有底气，但打篮球就只能在场边努力拼抢，感觉自己在场上没什么用。在这种情况下，你肯定对自己没自信，因为你不擅长，体能可能跟不上，体重可能太重。

感觉自己在某个领域有能力，你就会拥有自信。在诸如打篮球、踢足球或打游戏等领域，能力有时就等于价值。

例如，你踢球踢得很好，在场上就会成为最有话语权、最自信的人，甚至可以去批评别人而对方不敢反驳。首先，你在某个领域、某件事情上的能力和价值，与你的自信一定是挂钩的。

### 2. 自我价值感：感到自己有价值 ⭐

然而，我们也可以看到有的人很有能力，但就是对自己不自信。

例如，曾有一个女孩向我咨询。她似乎在哈佛就读，并告诉我她在美国所有藤校中的奖学金排名都是数一数二的。但她依然感到自己没自信。她参加演讲比赛拿第一名，参加辩论比赛也是第一名，考试成绩各方面都特别厉害，但就是对自己没自信。

从任何社会角度评价，这个人已经非常优秀。但问题在于，她可能认为自己的能力并没有价值。

**核心问题：能力 ≠ 感知到的价值**

别人可能觉得你又会念书又会赚钱，工作能力很强。但你可能觉得你干的这些事情并不是你真正认为有价值的。你觉得这件事其实没有多大意义，就好像别人都觉得很了不起，但你觉得也就那样。

所以，实际上是第二条出了问题：你有能力，但你的**自我价值感**并不够。你感觉不到你做这些事情有任何意义。

一个非常重要的原因是，这些事情是别人强加给你的。那个女孩就是典型例子，是被父母的价值所绑架，逼着她去努力优秀。她感觉做这些事情只是为了满足父母，对自己没有任何意义，因此不认可这方面的能力价值。

很多人会有这个问题：明明在某一方面做事都很好，大家评价很高，但你觉得这不是你想做的，你觉得这个事情没有任何意义。你挣再多的钱，成绩考得再好，也不会觉得有意义。

例如，有的人成绩特别好，但就是追不到姑娘，或搞不好人际关系。他可能很有钱、很有能力，但自己特别无趣，希望变得有趣却天生缺乏幽默感。

所以，很多时候是自己空有能力，却感觉不到自己有价值。因为你真正想要做到的事情你做不到，而那些你做得好的事情，你并不认为它们是自己必须去做到的。

### 3. 自我接纳：接受自己的不足 ✅

第三个要素是自我接纳。

有能力但感觉不到自我价值的人，很容易关注什么？很容易看到自己的缺点，并且为此烦恼。

例如，觉得自己不够幽默、长得不够帅、不受女生欢迎、人际关系搞不好。所以，如果你有能力但缺乏自我价值感，问题往往还在于第三条：**自我接纳度不够**，不能接受自己的不足。

自信、不自信和自卑是相似但不完全相同的概念。自卑通常是在人和人比较时，或形容某种自我情绪时使用。我们不过多纠结概念，理解意思即可。

自我接纳度低的一个常见表现是，因为自己的某些缺点或“难言之隐”而自卑。例如，有的人因为自己某些生理特征而自卑，于是拼命健身，希望通过肉体上的强壮去弥补心理上的自卑。

很多人自卑的一个很大原因，其实就是**抗拒自己的缺点**。比如自己穷、长得丑、长得矮，或有某些难言之隐。你排斥它，不想它存在，但又不得不带着它。用这种排斥心理去面对，你就一定会自卑。

如果一个人没钱，又一直排斥这个缺点，他就会产生一种想法：认为钱可以解决一切问题。“我就是有钱我就能如何如何，我就是没钱，你说这个有毛用。” 人在没钱的时候，如果不接受自己穷，就会变成这样。

在公共场合，比如群聊里，如果你想吸引他人，千万不要发表“一切都是因为没钱”这类言论。这只会暴露你的短处，让你一开始就处于弱势地位。

## 什么决定了你的自信？🔑

上一节我们介绍了自信的三个构成要素，本节中我们来看看是什么在根本上影响你的自信水平。

我们刚刚说到自信是由这三方面构成的：自己的能力、感知到的价值，以及自我接纳度。

自我接纳度高的人也会比较有自信，会觉得“我本来就这样，我觉得挺好，没什么不好”。有人认为第三条（自我接纳）是最重要的，我也认同。因为有能力的人总有比你能力更强的，有价值的人总有比你价值更高的。关键是你能否接受自己能力或价值可能不如别人。

以下是决定你自信的两个更深层因素：

### 个人能力 🛠️

首先，个人能力。一个人个人能力越强，他就越容易自信。

**代码逻辑：if (个人能力高) then { 自信程度增加 }**

你见过一个做什么事情都不成，但对自己很自信的人吗？如果你做啥啥不成，今天失败，明天失败，你还敢继续尝试吗？这很简单。

举个例子：你和一群人玩，你去炒菜，结果炒得像黑暗料理，又烧焦又难吃。大家捏着鼻子吃完，让你收碗，结果你把碗全摔了。至少在今天，你连这两件事都搞不定，你还能自信吗？大家可能会说“你坐下别搞了”，彼此心里肯定对你有看法。

能力强的人一定有自信。什么叫个人能力强？就是做啥你都能搞定，别人搞不定的事情你就能搞定。这是颠扑不破的真理。

### 三观（世界观、人生观、价值观） 🧭

第二，三观。一个人三观如果特别正，这个人会比较有自信。

一个人三观如果不好，他绝对不会有自信。最典型的有以下几种：

1.  **外部归因型**：认为所有问题都是外部因素导致的。“我之所以搞不定，是因为家里没钱，是因为我长得不好看。”
2.  **固定思维型**：认为人生没有成长性。“人生下来怎么样就是怎么样，没有好爹、长得不好看、没读书天赋就完了。”
3.  **唯结果论型**：只为结果负责。事情搞定了就自信，搞不定就不自信。

如果你什么事情都唯结果论，那你一定不会有自信。因为你成功100次，总会有一次失手。只要那一次失手，你的信心就会崩塌。

将军不怕屡败屡战，最怕的是百战百胜。例如，熟悉电竞的话，2018年的RNG战队就是这样，赢了一整年，却在最重要的一场比赛中失利，导致整个队伍的士气几乎被打垮。

**总结：健康的三观应包含成长性思维和过程导向，而非单纯的结果论或外部归因。**

## 总结 📝

本节课中，我们一起学习了构成自信的三个核心要素：**自我效能感**（感到有能力）、**自我价值感**（感到有价值）和**自我接纳**（接受不足）。我们探讨了能力与价值感知的分离，以及不接纳自身缺点如何导致自卑。最后，我们分析了影响自信的深层决定因素：**个人能力**的强弱和**三观**是否健康积极。

理解这些概念是迈向真正自信的第一步。记住，自信是一个建立在自我认知、自我认可和自我接纳基础上的综合状态。
