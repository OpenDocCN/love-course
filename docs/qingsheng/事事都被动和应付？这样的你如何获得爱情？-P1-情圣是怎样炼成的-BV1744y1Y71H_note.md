# 课程一：情圣是怎样炼成的 - 被动与应付的危害 🚫

在本节课中，我们将要学习“被动”和“应付”这两种心态在人际交往，特别是情感关系中的具体危害。通过分析日常生活中的常见场景，我们将理解为何这些心态会阻碍我们建立真诚、积极的关系。

被动的态度会引发几个问题。第一个问题是，如果你不主动，没有自己的想法，你就会变得被动。被动总会导致事情做不好，而且你内心总是不情愿。例如，当别人向你借钱时，为什么这件事很难处理？

因为无论你拒绝还是答应，都容易得罪人。最主要的原因是，在这件事上你很被动。这不是拒绝得好不好的问题，而是你处于被动状态。例如，一个你非常在乎、非常喜欢的铁哥们，或者一个你心仪的女生来向你借钱。

你有不借的道理吗？如果你真心把对方当作特别好的兄弟或姐妹，你有不借的道理吗？当你真心想帮助别人时，即使你自己没钱，你也会想办法凑点钱给他。所以，被动会导致什么结果？

如果你是被动的，当别人来向你借钱时，你处于被动状态。你既不想借，又碍于情面不得不借，这种情况下你总会得罪人。因此，一旦你心态上被动，就很容易把事情做糟。

上一节我们介绍了借钱场景中的被动心态，本节中我们来看看其他情境。不仅是借钱，其他被人要求去做的事情也是如此。比如在公司里，领导总是让你端茶递水。

你会发现每次端茶递水，你都老不情愿。你觉得自己必须去应付领导。今天端茶可能笨手笨脚，明天做事可能心不在焉。你会把这件事做得非常糟糕。而且，做到一定程度时，你会怀疑领导是不是在给你穿小鞋，你会坚持不下去。

因为你被动和应付，态度消极，你心里不爽，别人心里也不爽。仅仅是这样一件小事，就会导致双方都不愉快。所以，第二个问题是，事事应付的结果是什么？

事事应付的结果就是，你总会在某件事情上出错。当一个人被动时，一出错会出现什么情况？当你被动应付一件事并出错时，你会立刻推卸责任。例如，领导让你端茶递水，而你把水打翻了。

你虽然也会说对不起，但如果你内心不情愿，你就会怎么样？你会说：“哎呀，刚刚地面太滑了，所以水洒了。”总之就是各种不情愿，为自己找借口。但如果你心态主动，很乐意做这件事，即使出了错，你也会立刻想办法弥补，改变不好的结果。

你不会给自己找过多借口，而是说：“哎呀对不起，是我不小心。”别人也能看到你的积极态度。第三个问题是，如果你事事应付，结果不好，又总喜欢推卸责任，被人说多了会怎么样？

你事事被动应付，被人说多了，会怎么样？你本来就不愿意做，还老被人说，再遇到事情你是不是就总想逃避了？因为你本来就不愿意做，又怕自己做不好，还容易被人指责。

在这种情况下，你是不是就总想逃避？我刚刚说的前三条如果成立，就会形成一个恶性循环。在你的成长过程中是不是就是这样？交不到女朋友、内心郁闷，很多问题就是这么来的。

这描述的其实就是很多人日常的状态。你被人说多了，于是就想逃避。你怕自己做不好，还容易被人说，这时你就怎么样？你跟周边的人基本上就没什么联系和交往了。

因为你把自己缩起来了。哪怕大家都处在同一个空间，别人可能在聊天，你也会感觉自己融入不了环境。时间长了，你自己都会觉得自己很没人情味。

以下是情感冷淡的具体表现：
*   在情感方面会对人冷淡。
*   一旦别人叫你做件事，你会很纳闷。
*   当你去付出时，会觉得不爽、不舒服、不想做。

因为你自己没有想法，你让别人来安排你，这就是被动和应付。很多事情都是你自己不情愿的。大家有没有想过，当你没有想法时，别人就会来安排你？生活中永远是这样。

这不是内心是否强大的问题。当你自己没有欲望、没有想法时，你会发现别人就会来安排你。你们有没有发现生活就是这样的？你不管好自己的时候，就会有人来管你。

亲人的管，可能是像老妈那样唠叨你。如果以后你找了个对你好的老婆，她可能也会唠叨你。如果不是亲人来管，就会由社会来教育你。例如，如果你天天不爱惜身体、熬夜，你的身体就会教育你。

没有人念叨你，没有人说你好。如果你天天这样玩、这样搞，身体搞垮了，你就要去医院，你就要花钱。

**本节课中我们一起学习了被动与应付心态的三大危害：1. 导致事情做不好且内心抗拒；2. 出错时习惯推卸责任；3. 长期积累导致逃避社交与情感冷淡。** 其核心在于 **`没有个人想法 -> 陷入被动 -> 被他人或环境安排`**。要改变现状，必须从建立自己的主动想法开始。
