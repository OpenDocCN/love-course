# 恋爱过程中你如何做到战略上轻视

在本节课中，我们将要学习一个核心的恋爱思维模式：如何在追求过程中做到“战略上轻视，战术上重视”。我们将深入探讨这一概念的含义、常见的错误做法，以及如何将其正确应用于实践中，帮助你建立更健康、更有效的互动方式。

## 思维方式的问题根源

上一节我们提到了核心概念，本节中我们来看看导致追求失败的根本原因。许多人的挫败感并非源于事件本身，而是源于他们对事件的看法。

例如，有人因为某些事情没做好，就认为自己很失败。这就是典型的思维方式问题。我也有很多事情做不好，经常出错，但我从不认为自己失败，因为这些事不会影响我对自己的核心看法。

问题的关键在于，面对事情时，你的解释和分析方式出了问题。在追求女生这件事上，这表现为**情绪性的思维方式问题**。同时，当你面对具体情境时，你不知道如何清理自己的思维。

以下是两个主要方面：
*   你不知道如何清理思维。
*   这必然导致情绪问题，如感到自己无用或做不好。

因为你不知道如何采取有效的思维方式去推进事情发展。无论最终成败，在这个过程中你本应有所收获。

## 常见的错误模式：战略重视，战术轻视

那么我们怎么看待这个问题呢？首先，大多数追不到女生的男生，实际上都犯了一个错误：**在战略上过度重视，在战术上却过于轻视**。

很多同学会感到疑惑：“我战术上挺重视的啊？我送礼物、研究追求方法、上网学技巧，这还不叫重视吗？”

然而，答案是：**不，你是在战术上轻视，在战略上重视**。这正是思维方式出了问题，导致追求过程一步错，步步错。只要这种思维方式持续下去，女生很可能会越来越不喜欢，失败就是一个持续的过程。

这就好比在游戏中，你还没清理掉对方的外围防御（外卡），就想直接攻击水晶，结果只能是送死。更常见的比喻是：很多男生在装备和等级都不领先的情况下，就想单杀对方，却没注意防御塔，结果被塔打死。被击杀后不服气，又去硬拼，在兵线、经验全面落后时反复“硬上”，结果被连续击杀，对方优势像滚雪球一样扩大，最终推掉你的防御塔。

很多人不理解什么是战略，什么是战术。接下来我会详细解释，并说明为什么这种模式会出错。

## 为什么“战略重视”会出错？

男生在“战略上重视”，通常表现为总是去关注自己**控制不了的事情**，从而陷入持续的烦恼和焦虑。

例如：
*   这个女生对我是什么意思？
*   她会不会喜欢我？
*   我这样做是不是没戏了？
*   我那样做是不是完蛋了？

你一天到晚关注的，都是“如何让这个女生对我有好感”、“如何让她愿意做我女朋友”。你会猜测她的想法，解读她的每个表情和动作。反馈好就觉得有戏，反馈不好就觉得没戏。

**这就是你一开始就错了**。如果你从一开始就纠结这些问题，最后肯定搞不定。相反，那些不纠结这些事的，发展往往更顺利。

因为你关注的“女生喜不喜欢你”，是**你控制不了**的。这就是战略层面的问题。除了带来自我烦恼，你还会被对方的反应所控制，导致自己的情绪变得僵硬。

对方态度好一点或差一点，你都吃不准，不知道做什么才能让对方喜欢。于是你开始“装”，想建立好形象，又怕露馅，充满担忧。情绪自然就僵硬、木讷、不会说话、内向。

由于你想要一个确定的结果，又纠结于控制不了的事，你会充满 **“应该思维”**。

例如：
*   女生不理我了，**我应该**怎么办？
*   她对我说了那句话，**我应该**怎么回？

你脑子里总是这些东西，聊天和约会就会变得非常难受。因为你总是在**防御对自己不利的结果**，希望事情向好发展，这叫做“内在防御”。你会变得放不开，过度僵硬、木讷无趣，或者过度软弱、总是讨好对方。

到了最后，你会变得完全不知道该如何主动接触。主动效果不好，不主动则毫无机会。这就陷入了最初提到的咨询者的问题中。

结果就是，你们的关系无法得到正常有序的发展。
*   如果你傲气、有自尊，两人容易发生矛盾。
*   如果你想快速推进、不要原地不动，就容易变成“直男”，急于邀约导致女生反感。
*   如果你原地不动，就容易变成“舔狗”或被发好人卡。

怎么做都是错。为什么？因为**你总在关注你控制不了的事情**。

## 正确的做法：战略轻视，战术重视

那么，什么是正确的“战略上轻视，战术上重视”呢？其实很简单。

**战略上轻视**，指的是对你**无法控制的事情**保持轻视态度。

例如：
*   这个女孩是不是喜欢你？
*   你能不能得到她？
*   你和她在一起的可能性有多大？

这些你**根本不用关心**，因为你控制不了。你越关心控制不了的事，心里就越乱，一乱就不知道该怎么办，在对方面前也会表现得畏畏缩缩。

你根本不用管对方怎么看你。想说什么就说什么，想做什么就做什么，这没关系。首先做到“战略上轻视”，你才不会害怕，情绪才能自然流动，才能放得开，避免变得木讷。

**战术上重视**，指的是对你**可以控制的事情**给予充分重视。

## 什么是“可以控制的事情”？

以下是你可以控制并应予以重视的事情：

*   **女生的喜好、行踪、朋友圈**：这些是你可以去了解的。
    *   **喜好**：例如她喜欢喝奶茶，甚至具体到喜欢苹果口味的奶茶，最好还能加点苹果碎。
    *   **行踪**：在学校里，她什么时候上课、下课；上班后，她什么时候去吃饭、几点下班。了解这些可以帮助你创造交集的机会。
    *   **交际圈**：如果她有交际圈，你想约她出来时，可以邀请整个圈子一起，事先和朋友说好，到了几点大家撤退，由你单独和她吃晚饭。

战术上重视，可以让你和女生之间拥有更多发展机会。但很多人只关注“女生不理我怎么办”、“异地除了打电话还能做什么”。哪怕你只是去了解一下她喜欢吃什么，并学着做一下，都不至于在追求中如此被动。

*   **你自身的状况**：
    *   **外形**：你的穿着打扮、个人形象。
    *   **约会安排**：约她出去时，一天的活动安排（早上、中午、晚上做什么）、路线规划、预算。
    *   **优势展示**：如何恰当地在她面前展示你的优势。
    *   **惊喜设计**：初次、二次、三次见面时，可以设计什么意外和惊喜。

这些事情都是可以设计和准备的。你们有在这些事情上花心思吗？没有。你们只关注“怎么和她聊天”、“怎么回她消息”、“约不出来怎么办”。**这不叫战术，这叫战略上的纠结**。

你总在战略层面（她怎么想）上纠结，却不在战术层面（具体事情怎么做）上安排和思考，怎么可能做得好？

## 核心区分：可控 vs 不可控

“战术上重视”的对象，是你控制范围之内的事情。例如，**“你能不能把她约出来”**，这就是你可以通过努力去影响的事情。

只要是你控制范围之外的事情，就属于“战略”范畴，应保持轻视。例如，很多人莫名其妙地去表白。表白是为了让女生答应你，但**女生答不答应，你能控制吗？不能**。

喜欢在战略上重视、战术上轻视的人，最喜欢干一件事：**赌**。

“这次我博一把”、“这次我赌一下”。结果越赌越被动。这就像装备等级落后却想去反杀，结果被防御塔打死。甚至像一级就去打肉山（游戏中的强大野怪），结果可想而知。

有人会问：“不表白，怎么确定关系？” 你看，你还是在问一个“如何确定结果”的战略问题。有什么事情是**可以**确定关系的吗？

你们总希望有固定的步骤和方式，按照几步走就能得到想要的结果。但世界上除了**技能性的事情**（比如按固定步骤炒菜），很少有这种事。追求女生不是技能性的事情。

确定关系的方式有很多种，不限于表白。
*   你可以直接牵起她的手。
*   可以直接搂住她。
*   可以直接亲吻她。
*   可以直接为她戴上项链或戒指，说“从今天起你就是我的女人”。

当然，并非推荐这些方式，只是说明**达成目的即可，方法可以多样**。表白是一种责任，不是一个必要的手段。死脑筋地认为“必须表白才能确定关系”，是很难追到女生的。

所谓“目的不明确，手段却很死板”，很多男生都这样。比如追求同校女生，对方的作息、活动路线都很固定，却还在问“她朋友圈不回怎么办”。这种死脑筋怎么竞争得过那些头脑灵活、胆子大的男生？

看上去是死脑筋，实际上是内心比较怂，不敢做一些灵活、大胆、具有进攻性的举动。

## 实战举例：搭讪

我们以“搭讪”为例，具体说明什么是战略上轻视、战术上重视。

*   **战略上轻视（关注不可控的）**：
    *   女生**会不会搭理你**？—— 不可控。
    *   女生**给不给你微信**？—— 不可控。
    *   **不要关心这些结果**，这就叫战术上轻视（注：此处视频原文口误，根据上下文逻辑，应为“战略上轻视”）。很多人问“怎么搭讪”，其实是在问“怎么搭讪对方**一定**会理我”。这种追求确定结果的思维方式，很容易产生 **“应该思维”**。
    *   只要产生“应该思维”（我应该这么说，我应该那么做），你在对方面前情绪就一定不流畅。满脑子“应该”的时候，关系就无法发展。这在心理学上称为 **“人际防御”** 。应该思维越多，关系越不顺畅，越难产生真实的情感交流和亲密关系。

*   **战术上重视（做好可控的）**：
    *   你**走不走上前**？—— 可控。
    *   你当下的**形象**好不好？—— 可控。
    *   要不要准备点**小礼物**？—— 可控。
    *   说话时能否**保持礼貌、挺直腰杆**？—— 可控。
    *   能否**适度提高音量**，加点**肢体语言**，表情**丰富一点**，保持**镇定**？—— 可控。
    *   你要把这些**尽可能控制好的事情做好**，这就叫战术上重视。

具体操作：
1.  **不要怕说错话**（战略轻视）。
2.  **做好可控细节**（战术重视）：
    *   走上前之前，整理一下头发，擦一下鞋子。
    *   可以买杯奶茶作为小开场。
    *   走近时保持礼貌，先说“你好”。
    *   挺直腰杆，不要勾着背。
    *   说话音量尽量大一点，清晰一点。
    *   说完“你好”后，等待对方有反应，再说下一句。

这几个小细节的调整，会让你的搭讪成功率比之前什么都不做要高得多。女生能明显感觉到，你和那些慌慌张张、随随便便的人完全不同。

## 累积效应与内在防御

将这个原则扩展到整个相处过程：如果每件事你都做得比别人好一点，加起来会有什么效果？很多人追不到女生，不是某个方法或套路的问题，而是**在整个过程中，每一件事都做得很拉垮**。

女生可能一开始对你有好感，但：
*   第一次聊天，你说话不尊重她。
*   第二次约会，你安排得一塌糊涂。
*   第三次相处，你又干了件蠢事。

这些事累积起来，结果怎么可能好？而且你会发现，**不是你故意要干蠢事，而是你担心自己干蠢事，结果就越会干蠢事**。你为什么想干蠢事？因为你太希望女生对你有好看法了（战略过度重视），这导致了我们之前讲的“人际防御”。

人际关系发展不顺畅，一定与你内心的防御程度有关。内心的防御程度，就体现在你在这个人面前的“应该思维”有多少。

比如那个认为“确定关系应该要表白”的同学，这就是应该思维。在我这里，表白或不表白都可以，只要事情干得漂亮、出彩就好。**没有什么必须要做的事，也没有什么绝对不能做的事**。关键问题不在这里。

我们剖析了常见的错误模式（过度关注不可控的结果，却忽视可控的过程细节），明确了战略（关注不可控之事）与战术（做好可控之事）的区别，并通过搭讪等实例展示了如何应用这一原则。记住，减少对结果的“应该思维”，专注做好过程里每一件你能做好的小事，累积起来，才能让关系自然、顺畅地发展。
