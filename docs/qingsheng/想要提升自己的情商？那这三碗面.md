# 想要提升自己的情商？那这三碗面

在本节课中，我们将一起学习如何通过理解“三碗面”——体面、场面和情面，来提升自己的情商。情商的核心是关于面子和尊重的学问。我们将深入探讨如何通过控制情绪、维护尊严以及恰当地处理人际关系，来让自己在社交中更加从容和受人尊重。

## 情商是关于面子和尊重的学问

上一节我们提到了情商的核心概念，本节中我们来看看它的具体内涵。情商并非虚无缥缈，它是一门关于如何在人际交往中维护自己与他人的“面子”，以及如何表达“尊重”的实用学问。很多时候我们感到生气，根源在于觉得自己的付出或努力没有得到应有的尊重。

重新理解情商，你会发现它在生活中无处不在。面子并非凭空而来，它往往来自于他人的给予和认可。正如一句高情商的话所说：“我哪有什么面子啊，面子都是朋友给的。”这句话既体现了谦逊，也巧妙地肯定了朋友的价值和自己的地位。

关于情商，前人早有总结。例如杜月笙曾提出需要处理好的“三碗面”：体面、场面和情面。这三者如何“吃”，正是情商高低的关键。接下来，我们将逐一解析。

## 第一碗面：体面——控制情绪，维护尊严

很多人自称情商低，常常表现为容易情绪化。这种状态可能源于长期压抑，或者是在认真做事、面对高压环境时逐渐形成的。例如，创业后面对诸多挑战，脾气可能变差；又或者遇到对自己要求苛刻、动辄指责的上司，只能默默忍受。

此外，网络上的言语冲突也极易让人情绪失控。然而，放纵情绪的后果是让自己显得“不体面”。无论是在公开场合与人争吵，还是在网络上互撕，都会让旁观者觉得双方都很失态。

体面的核心在于**能控制情绪，保持风度**。即使内心愤怒，在公开场合也能保持冷静，不说伤害对方的话，这本身就是高情商的表现。控制情绪不是靠“忍”，而是拥有处理问题的能力和底气。就像一个能轻松举起200斤的人不会觉得累一样，情商高的人处理冲突游刃有余，自然不会感到心累。

**高情商的做法往往不是对抗，而是“沉默”**。当受到无端辱骂时，不立即回击，而是冷静离开，这需要强大的内心。这种沉默如果伴随着实力和从容的姿态，反而会让挑衅者心虚，让旁观者站在你这一边。它传递的信息是：我不屑于与你纠缠，但我并非软弱可欺。

以下是体现“体面”的几个关键点：

*   **避免一点就炸**：轻易被激怒是最不明智的行为，它会让你显得愚蠢且易于操控。
*   **学会战略性沉默**：在受到攻击时，不立即回应，冷静观察。如果对方变本加厉，你的忍让会积累舆论优势，此时再采取行动就合情合理。
*   **姿态决定待遇**：你的精神状态和身体语言决定了别人如何对待你。低头哈腰、唯唯诺诺容易被视为“软柿子”；而昂首挺胸、从容不迫则能自然建立起威慑力。

体面，就是学会用自己的行动捍卫尊严，而不是通过失态的爆发来宣泄情绪。

## 第二碗面：场面——懂得给别人面子

上一节我们学习了如何让自己体面，本节中我们来看看如何让别人有面子，这就是“场面”。场面话、场面事，核心在于**在公开或社交场合，照顾他人的感受，给予对方荣耀和尊重**。

一个最典型的例子是送礼。送一份得体、甚至略贵重的礼物，不仅是一份心意，更是给对方面子，让对方能在他人面前感到自豪。这无关利益交换，而是一种社交礼仪和尊重的体现。

在语言上，体现“场面”的细节也很多。例如，向别人介绍朋友时说“这位是我的好朋友”，比单纯说“这位是我朋友”多了一个“好”字，听者的感受就大不相同，感觉更受重视。

另一个例子是描述关系：“我们以前是同事，后来成了朋友”与“我们本来就是朋友，现在恰好在同一家公司共事”。后者显然让人感觉关系更亲密、分量更重，因为它强调了友谊先于工作关系。

所以，处理“场面”的精髓在于，通过言语和行动，让他人在社交环境中感受到被重视、被尊重。

## 第三碗面：情面——表达深层的重视

“情面”是最高级的一碗面，它超越了公开场合的礼貌，进入了更私人和深刻的层面。情面的核心是**表达我对你个人的、特别的重视**。

它体现在：我送你的，正是你内心很想要，但出于各种原因不好意思开口的东西。这需要细致的观察和真诚的关心。

一个更复杂的例子是如何处理朋友借钱。高情商的做法不是简单地借或不借，而是**在过程中充分表达对朋友的情谊和重视**。例如，可以坦诚自己的难处（出示账户余额），但同时积极为对方想办法（展示为他向别人借钱的努力），甚至提出动用自己心爱之物来帮助他。虽然结果可能还是无法足额借出，但这个过程让对方深切感受到你把他当回事。

即使最终只能借出一部分，也可以暗示“这钱不用急着还，就当是朋友的心意”。这样既照顾了对方的急用（场面），又体现了深厚的情谊（情面），还避免了因催债而破坏关系。

情面，就是用心去体察对方未言明的需求，并用行动表达“你对我来说很重要”。

## 总结与核心：面子是社交货币

本节课我们一起学习了提升情商的“三碗面”：**体面**（控制情绪，维护自我尊严）、**场面**（公开场合，给予对方面子）、**情面**（私人层面，表达深层重视）。

最后，我们需要理解“面子”的本质。**面子就像一种“社交货币”**。如果你自身价值低、不体面，却想巴结地位高的人，就像身无分文的人想和百万富翁做朋友一样困难。

因此，提升情商的根本，在于**不断提升自身价值，做好每件事，赢得他人的尊重（赚取“社交货币”）**。当你自己拥有足够的“体面”时，你给予别人的“场面”和“情面”才会更有分量，你的人际关系也才能更加健康、平等和持久。否则，一味讨好与忍让，只会让自己越来越失去尊严和主动权。
