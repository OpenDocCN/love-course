# 不被别人认可，你就想从别人身上

在本节课中，我们将要学习一个核心概念：为什么“舔狗”行为无法获得真正的认可与尊重。我们将深入探讨其本质，并分析获得他人认可的关键要素。

---

## 概述：问题的根源

上一节我们介绍了课程主题，本节中我们来看看“舔狗”现象的根本原因。所有类似“舔狗”的问题，其本质都是相同的。

**核心本质**：在**没有获得他人认可**的情况下，就试图从他人身上**获取利益**或自己想要的东西。

以下是具体分析：

*   你并未获得对方的认知和认可。所有“舔狗”行为模式都一样，包括父母与子女的关系。有的父母宠爱孩子，但不听从孩子的意见，这就是不认可，认为孩子的想法或能力不值得信任。
*   他们认为你的想法幼稚，并且你没有做出任何成绩来证明自己。包括“舔狗”追求女生时，从未思考如何获得女生的认可，让女生觉得这个男生不错。

---

## 认可的重要性

理解了问题的根源后，我们来看看为什么获得认可是如此关键。人们通常只想着“如何得到”，却忽略了“如何获得认可”。

**认可的意义**：获得认可意味着在别人面前能获得尊重，能够进行平等的对话。你说的话、做的事能得到对方的重视。

**核心公式**：
`个人价值 ≠ 财富/地位`，除非 `获得认可 == True`

如果一个人不认可你，你再有钱、再厉害，在对方眼里也毫无价值。反之亦然：如果你看不起一个人，即使他后来赚了很多钱或当了大官，你依然会看不起他。

---

## 如何真正获得认可

那么，我们何时会对一个人刮目相看？仅仅是变得“优秀”就够了吗？本节我们来探讨获得认可的真正途径。

仅仅变得“优秀”或“混得好”并不能自动获得认可。如果对方内心不认可你，你变得优秀也无济于事。认可源于对方内心看法的根本改变。

以下是说明本质未变的例子：

> 我有个朋友，以前是个昼夜颠倒、无所事事的混子。后来他凭借一点小聪明和天赋，成了网络大V，赚了很多钱。但和他聊天发现，他本质没变：依然懒散、混日子、思维混乱、对朋友不讲义气。这样的人，即使他赚了一百万、一千万，我们这些认识他旧模样的人，依然不会从内心认可和尊重他。

这个例子表明，**外在的成功若没有内在品质的支撑，无法赢得真正的尊重**。

---

## 活得“有个人样”

那么，如何使别人尊重和认可你呢？关键在于你自己要活得“有个人样”。

你需要符合社会主流的价值观。例如，一个生活规律、积极进取、有事业心、责任心强的男生，至少不会让人不认同或不认可。母亲或其他人也是如此。

以下是关于“坏人”和“个性”的思考：

*   **坏人与天赋**：你们有没有发现，坏人往往很有天赋？比如实施诈骗需要聪明的头脑，但这并非社会提倡的系统技能。
*   **个性的基础**：一个人声称有个性，前提是什么？基础是什么？个性不是凭空而来的。

**个性的核心代码**：
```python
def 体现个性(目标):
    if 说到做到(目标) == True and 内心有力量感 == True:
        return “真正的个性”
    else:
        return “虚张声势”
```

有个性意味着内心有力量感，气场更强，欲望更强烈。例如，敢于打赌并全力以赴去实现，即使能力暂时不足，也会想尽办法、拼尽全力去争取，行动力强，不达目的不罢休，有不服输的精神。

---

## 个性与行为的一致性

本节我们来看看，为什么模仿“不顺着女生”这样的表面行为会失败，而真正的个性却能吸引人。

一个性格懦弱的人，单纯模仿“不顺着女生”的行为（比如女生让接却不接），结果往往是女生不再理他。这是因为其行为缺乏内在一致性。

真正的个性吸引人，是因为其**言行一致**，形成了强大的个人气场。例如，某些公众人物（如比喻中的“马宝培”），即使做出拉黑女生等行为，反而有更多女生喜欢他，原因就在于他表里如一的个性、对职业的认真态度等优点。

**关键点**：如果你本身是个懦弱的人，却空谈要“有个性”，这是不现实的。就像说“我要有钱”却看不到钱一样。人的品质无法凭空而来。

**核心总结公式**：
`受欢迎程度` 或 `财富获取能力` ∝ `性格优点数量`
`性格缺点越多` → `往往越贫穷，拥有的越少`
`性格优点越多` → `能拥有的东西就越多`

---

## 总结：品质需要锤炼

本节课中我们一起学习了“舔狗”行为的本质在于寻求利益前未获认可。我们探讨了认可的重要性，以及真正获得认可需要内在品质的支撑，如活得符合主流价值观、培养言行合一的真正个性。

这些优秀的品质（如勇敢、有个性）并非像“喝蛋汤”一样说要有就能有。它们需要经过系统的训练、环境的锤炼，以及他人的教导和引导才能获得。如果你表达“个性”时只招来他人讨厌，那就说明那并非真正的个性，而是内在支撑不足的虚张声势。请认真反思自己性格上的优点与缺点，这是改变的开始。
