# 课程一：如何建立自信？从摆脱价值判断开始 🧠

在本节课中，我们将学习自信心的核心障碍——“价值判断”，并探讨如何通过改变思维与行动模式来建立稳固的自信。我们将拆解四个关键步骤，帮助你摆脱过度在意他人眼光的困扰。

---

## 概述：自信的敌人——价值判断

我们很多时候自信心经常会被价值判断所影响。你会发现，自己常常带着一种价值判断去做事，仿佛时刻处于被人审判的情况下，因而特别在意别人的眼光。

这种状态本质上是因为你没有完全承担起做事的责任。当你真正把事情担起来之后，你的焦点就会从“别人怎么看我”转向“我如何解决这个问题”。

---

## 第一节：理解情绪的真正作用

上一节我们提到了价值判断的影响，本节中我们来看看如何正确看待随之产生的情绪。

你们可能会很讨厌那种因价值判断而产生的负面情绪，比如自卑或不自信。但我要告诉大家，不要总想着如何去“解决”情绪。情绪本身没有用吗？并非如此。很多时候，情绪其实是在提醒你：你在某方面做得不太好。它并非指责你太弱小，而是告诉你存在问题，需要你去解决。

**核心观点**：情绪是指南针，不是敌人。它指导我们去把事情做好。学会这样看待情绪，就能让负面情绪产生正面作用，帮助自己摆正许多事情。

当然，如果情绪过于严重，可能需要先消化一下，因为严重的情绪会对人产生干扰和创伤。

---

## 第二节：建立自信的四个核心步骤

理解了情绪的导向作用后，我们来看看构建自信的具体方法。以下是四个需要依次实践的关键步骤。

### 1. 学会客观看待事物，停止价值判断

首先，你自己停止对外界做价值判断，你就会减少对自我的价值判断。这样，你的情绪就不会很严重。一个人难免有情绪，但情绪不能过重。情绪过重会让你陷入其中，产生无尽的自责与各种负面感觉。

**如何客观看待？** 举个例子：你约女生，女生不出来。价值判断会是：“我哪里不好？”或“这女生有问题”。而客观判断则是：“她可能不方便”。一旦进入价值判断，情绪就来了，你会开始猜测她的想法。但客观来看，这次不出来，下次再约即可。可能是没时间、不感兴趣或安排不妥。**唯独不要去想“她是否喜欢我”或“我是否不够好”**。

**公式**：
`客观事实 + 多种可能性分析` -> `避免“自我贬低”或“指责他人”的价值判断`

### 2. 提升行动力：在经验中磨砺

人为什么容易陷入价值判断和猜疑？原因很简单：经验太少。例如，搭讪被拒绝多了，你就不会把它当回事。有的人被拒绝一次，就老想着挽回，纠结自己是否不够好。这其实就是因为在事情上磨砺得不够，行动力不足，做得少了，情绪就多了。

**代码描述**：
```python
if 行动经验 < 阈值:
    容易陷入情绪化与自我怀疑
else:
    对结果脱敏，更关注问题解决
```

### 3. 建立可验证的流程，减少不确定性

减少价值判断的第三个方法是：增加确认。即采用那些能经过过程和结果反复验证的“套路”。当你按照同样流程总能得到类似结果时，你就会很清楚自己的能力边界，不再需要多余的价值判断。

例如，你很清楚和女生这样沟通会得到何种反应。你通过**过程**和**结果**的验证，确信自己知道该如何做。

### 4. 正确看待“他人的眼光”：从被审判到负责任

如果你的群体意识过强（例如处于自卑或社交恐惧阶段），你就会非常容易在乎别人的眼光。在乎别人眼光不是坏事，但我们不能像时刻被审判一样。

**正确的在乎**：就像我讲课，也在乎你们是否听得受用。如果不在乎，我就不会去想课程如何讲好。这种在乎促使我做得更好，更有责任感和主人翁精神。所以，要勇于担责。先别想能否搞定，**先把责任担起来，再去思考如何做好**。

---

## 总结与行动路径

本节课我们一起学习了自信建立的关键：摆脱有害的价值判断。

**总结行动路径**：
1.  **理清思路**：停止对人对己的价值判断，学会客观分析。
2.  **加快行动**：在大量实践中磨砺，对结果脱敏。
3.  **完善方法**：建立并信任那些经过验证的行为流程。
4.  **关注反馈**：以负责任的态度在乎他人看法，将其作为提升自己的依据。

当你反复这样做，对自己的怀疑就会减少。你思路清晰、行动迅速、方法得当，并且懂得关心他人、根据反馈调整进步。这时，你在与人交往时，就不再会纠结“别人是否喜欢我”这样的问题。

即使出现问题，你也会想：“我验证过大家通常认可我，这次可能是对方遇到了问题。” 你不会责怪或反感对方，反而会想如何去帮助和关心他。人的成长过程，正是如此。
