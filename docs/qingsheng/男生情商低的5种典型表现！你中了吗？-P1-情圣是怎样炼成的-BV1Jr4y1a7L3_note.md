# 情商提升课 01：识别并避免五种低情商表现 🚫

在本节课中，我们将学习情商的核心概念，并通过分析五种典型的低情商表现，帮助你理解如何在日常交流中避免冒犯他人，实现更顺畅的沟通。情商并非天生，而是源于**有意识的自我约束**和**换位思考**。

---

## 情商的核心：自我约束 🤔

上一节我们提到了情商的重要性，本节中我们来看看其根本来源。所谓情商，并非一种天赋，而是来自于**平时的自我约束**。其根本在于**多替对方着想**。一个平时注重自我约束的人，情商自然不会低。

---

## 五种低情商典型表现

以下是五种在沟通中，尤其是与异性交往时，容易出现的低情商表现。我们将逐一分析其问题所在。

### 1. “我要”式聊天

这种聊天方式以单方面索取信息为核心，完全忽略对方感受，令人反感。

*   **典型话术**：“你年龄多大？”、“家住哪里？”、“发张自拍来呀！”、“你什么时候有空，我想约你出来。”
*   **问题本质**：沟通内容只有“我要”，缺乏对等尊重。
*   **改进方法**：将索取信息转化为夸赞或提供价值。例如，将“你年龄多大？”改为“你看上去挺年轻的，不会超过25岁吧？”。这用**夸赞的方式完成了解信息的过程**，是一种说话的技巧。

**核心公式**：低情商索取 = 直接提问“我要XX”；高情商获取 = 提供价值（夸赞/关心）+ 自然获取信息。

### 2. “你要”式说教

这种表现以居高临下的姿态评判或否定对方的兴趣爱好和行为。

*   **典型话术**：“你们女生天天买这些化妆品干嘛？”、“这有什么好玩的？”、“这不就是XX加个喷嘴吗？”
*   **问题本质**：以教训和说教的口吻沟通，强行输出个人观点，招人记恨。
*   **改进思路**：尊重差异，即使不理解，也不轻易贬低他人的选择。可以尝试询问：“你对这个这么了解，它吸引你的地方是什么？”来开启对话。

### 3. “我是对的”式对抗

执着于证明自己正确，常常伴随着“我是为你好”之类的说辞，容易引发冲突。

*   **典型心态**：“我是对的，你应该听我的。”、“你这么搞是错的。”
*   **问题本质**：制造了对立局面，一旦产生“对错之争”，矛盾便不可避免。
*   **高明技巧**：参考姚明的应对方式——**避免直接对抗，顺着对方的逻辑接话**。例如，当对方说“你得分超过19分我就亲你屁股”，姚明回应“那我最好每场都得18分”。这消除了对错，化解了对抗。

### 4. “我厉害”式比较

通过贬低他人来抬高自己，或在争论中炫耀资历，容易激起对方的反抗心理。

*   **典型场景**：网络杠精、争论时摆资历。例如回应质疑：“我数学再差也比你好！”
*   **问题本质**：当你试图证明“我比你厉害”时，对方也会拼命证明“我比你更厉害”，导致矛盾升级。
*   **改进方法**：在非原则问题上，无需在口头上争高下。可以用幽默或转移焦点的方式应对刁难，而非陷入“比较游戏”。

### 5. “我不”式辩解

面对任何质疑或批评，第一反应是防御性否认和辩解，拒绝沟通。

*   **典型话术**：“没有啊！”、“不是这样的！”、“你不对！”
*   **问题本质**：关闭了沟通渠道，立刻激发对方情绪。例如邻居投诉空调漏水，立即回答“没有，安装得很好”只会加剧矛盾。
*   **正确应对**：首先接纳对方的反馈。应回答：“不好意思，我去看看是什么情况。”**先处理事情，而非先反驳对方**，能为解决问题创造良好氛围。

## 高情商的“怼”与化解

当然，并非所有时候都需要退让。高情商者同样懂得如何优雅地回应冒犯。

*   **原则**：不陷入情绪对抗，用智慧化解。
*   **经典案例**：曾志伟开玩笑说黄渤“一张丑脸在电视里唱歌很难过”。黄渤回应：“我们是互相鼓励，我小时候看你拿影帝，就坚定了当演员的决心——你都能行，我肯定也能行。”
*   **效果分析**：这既是回应（怼），又表达了“赞美”，同时抬高了双方，让玩笑无法继续，且不伤和气。其**代码逻辑**可理解为：`回应 = 接纳对方形式 + 转化话语内核 + 幽默抬高`

## 课程总结 📝

本节课中，我们一起学习了情商的根基在于**自我约束**与**换位思考**，并详细剖析了五种低情商表现：
1.  **“我要”式**（单方面索取）
2.  **“你要”式**（居高临下说教）
3.  **“我是对的”式**（引发对抗）
4.  **“我厉害”式**（贬人抬己）
5.  **“我不”式**（防御性辩解）

避免这些表现的关键在于：**沟通时关注对方感受，达成自己目的的同时，让对方感到舒服**。高情商的沟通不是放弃观点，而是学会**拐个弯**，用更智慧、更不易引发对抗的方式表达。记住，**情商是可以通过有意识的练习来提升的**。
