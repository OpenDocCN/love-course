# 与女孩交往一段时间后就不再理你

在本节课中，我们将要学习追女孩过程中的核心态度问题，并分析几种常见的失败情况。我们将探讨态度如何影响行为，以及为什么仅仅学习方法技巧往往不够。课程将帮助你理解追女孩的本质，避免常见的误区。

## 第一部分：明确“追女孩”的定义与态度前提

首先，我们需要明确“追女孩”的含义。它指的是当你有一个喜欢的女孩，并且想和她在一起时，主动去接触和了解她的过程。这并非指如何“撩妹”、“约炮”或进行其他不严肃的交往。

明确了定义后，我们来探讨态度问题。态度问题有一个重要的前提。追女孩的方法容易学习，但改变对待女孩的态度，哪个更简单？改变态度显然更简单。

结合多年的经验和心理学知识，通常什么样的男生比较难脱单、追不到女孩？例如“弱男”，即生活中内心不够强大、性格有些懦弱的男生。这类男生往往有一个共同点：内心不够坚定。

## 第二部分：低自尊男生的行为模式与追女孩困境

具体来说，性格相对懦弱、有些自卑的男生，在心理学上可称为“低自尊”。这类男生大多数在做事情时态度散漫、耐不得烦、沉不下心。他们处理事情时粗线条、一根筋，心态浮躁，只想要快速得到结果。

这种状态与追女孩时的表现高度关联。他们容易只按照自己习惯的方式去追求，得到女生反馈时，又最容易进行感性判断，被表面现象吓到，而不去探究事情的本质。如果他们能稍微沉下心，注意方式方法，多观察女生，其实并没有那么难。

另一个特点是，这类男生在遇到困难、挫折或女生反馈不好时，按不住性子。女生不理睬就会开始着急，时间一长就自己情绪化，明知某些行为是错的也忍不住去犯。越懦弱自卑的人，在与女生交往时越难以控制自己的情绪和行为。

## 第三部分：方法策略与态度认知的层级关系

值得注意的是，有些平时很受女生欢迎、甚至有女生倒追的男生，并不代表他们“会”追女孩。他们可能擅长吸引异性，但遇到自己真正喜欢、需要主动去追求的对象时，反而不知所措。这同样适用于一些条件不错的女生。

要想做好一件自己想做的事，最大的问题之一，是很多人错误地将“群体策略”当作“个人策略”来使用。例如，“有钱就有女生”或“女生都喜欢钱”是一种群体策略。但当你用它来解决具体的个人交往问题时，往往会失效。

因为在具体的人与人交往中，存在一个最大的不可控因素：**情绪**。个体之间的互动充满情绪变量，无法用简单的群体规律套用。例如，29岁没房没车，只能说人生有些挫折，但并非彻底失败，更不应因此放弃恋爱。被这种物质条件困住而止步不前，本身就是一种态度问题。

所以，态度问题决定了你用什么方式和策略去做一件事。很多人特别关注方法，但方法之前是态度，而态度之前是**认知**。是你的认知、价值观和观念决定了你的态度。

如果你的认知出了问题，态度就会出问题，进而导致面对女生的方式出问题。如果只想从方法上改变，就会出现“机械主义”或“教条主义”，只会生硬地套用技巧，知其然不知其所以然。这样或许能吸引到一些异性，但最大的问题是**不能建立真正的感情**。

第三种情况：追了女孩一段时间，但对她的事情（如家庭、朋友、爱好、生活习惯等）一无所知。

关于“帅”的问题需要理性看待。女生说“觉得你不帅”，潜台词往往是“跟你在一起体验特别差”。第一眼不惊艳但相处后很喜欢的情况是存在的。男生追不到女生，一大半原因不是帅，而是**性格无趣**。
