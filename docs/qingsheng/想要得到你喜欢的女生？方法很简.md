# 课程：情圣是怎样炼成的 - P1：思维转变：从“我想要”到“她需要” 🧠💡

在本节课中，我们将要学习如何通过转变核心思维方式，来提升与心仪女生交往的成功率。核心在于将关注点从自身欲望转移到对方感受上。

## 概述：思维决定行为

想要得到你喜欢的女孩，你首先要得到她的认可。所以你需要把思维方式，从考虑你的欲望转变为什么？转变为考虑对方的感受和需求，以此来决定自己的行为特质和行为方针。这个时候，你就不会乱，你跟她之间的交往也会更顺畅。

## 两种思维方式的对比

上一节我们介绍了思维转变的重要性，本节中我们来看看两种具体思维方式的区别。关键在于，你思考的起点是“我”还是“她”。

以下是两种思维方式的详细对比：

*   **思维一：只关注自身欲望（“我想要”）**
    *   **核心公式**：`我的行为 = f(我的欲望)`
    *   **表现**：当你心里“想要得到她”的念头很强烈时，你说话做事很容易招对方反感。你越急于去得到她，你干的事情就越是招人反感，效果也越是负面的。
    *   **根源**：这种思维源于试图控制你无法控制的事情（如对方的想法和反应）。当一个人总希望控制自己无法控制的事情时，一定会烦、会焦虑。
    *   **导致结果**：**被动**。你的行为方式会根据对方的反应来决定。例如：“他对我有意思，我试一下；他对我没意思，就算了。” 或是“约会完了，他不理我怎么办？我只能想办法。” 这都是一种被动的反应模式。

*   **思维二：关注对方感受与需求（“她需要”）**
    *   **核心公式**：`我的行为 = f(她的感受与需求)`
    *   **表现**：你先考虑如何把事情干漂亮、把话说好，以此获得对方的认可。你考虑对方考虑得越周到，想得越清楚，别人对你的认可度就越高。
    *   **根源**：把注意力从“我能得到什么”转移到“我能提供什么价值/创造什么好的体验”。
    *   **导致结果**：**主动与从容**。你不再被对方的单一反应牵着鼻子走，而是基于对双方的了解，主动创造舒适的互动环境。这时，你反而能更自然地得到你所想要的东西。

## 思维转变的关键

理解了两种思维的区别后，我们来看看如何实践这种转变。关键在于调整你与人交互时的关注点。

在与任何人的交往中，如果只关心自己的问题——“我能不能得到我想要的”——那么无论你怎么做，往往都不会产生正向的效应。所以，你需要经常提醒自己：不要只考虑你自己，要多考虑对方。你替别人想得越多，别人对你的认可度就越高。

## 总结

本节课中，我们一起学习了追求心仪女生时最关键的思维转变。核心在于将焦点从 **“我想要”** 的索取心态，转变为 **“她需要”** 的给予与体察心态。这种转变能让你避免因焦虑和企图控制而做出的、招人反感的被动行为，转而通过创造舒适感和获得认可，自然而然地拉近彼此距离。记住，**得到认可，是得到一切的前提**。
