# 课程名称：自我认知与社交互动 🧠 - P1

## 概述

在本节课中，我们将要学习过度以自我为中心的行为模式会带来哪些后果，并探讨这种心态如何影响个人感受、行为以及与他人的关系。我们将从个人体验出发，分析其根源，并观察生活中更开放、行动力更强的人是如何建立社交联系的。

## 过度自我中心的后果 🤔

上一节我们概述了课程主题，本节中我们来看看过度关注自我所带来的直接问题。

最不舒服的人其实是你自己。人活得越自私越自我，首先第一个不舒服的人是自己，不是别人。你自己首先活得别扭，别人也会跟着你一块儿别扭。

以下是具体表现：

*   你觉得自己表现很差。
*   女生跟你在一起，会觉得你hold不住她。
*   女生会觉得跟你在一起好像有点浪费时间。

这样子的话，大家体验都会不好，你什么事情就会做不好。

## 行为模式的根源：性能量的觉醒 ⚡

我们刚刚所讨论的别扭感受，其根源往往与成长过程中的心理变化有关。本节中我们来看看这种变化通常何时开始。

人的这种性意识，大概是从初中二年级开始。有的可能早一点，有的可能晚一点。大概就是这个时候，我们说这个荷尔蒙能量开始从那个时候发挥作用。

以下是性能量觉醒后，在行为上可能的表现：

*   在女生面前刻意的表现自己，比如体育、打架、弹琴。
*   开始注意自己的发型和衣着，频繁照镜子。
*   开始琢磨怎么写情书。

这都是很自然而然的事情。学生时代，往往是那种最具有荷尔蒙气息的那些人最开始谈恋爱。学生时代那种小混混，或者一开始那女生开始学会打扮、开始化妆、开始打扮自己，那样的人是最开始谈恋爱的，就最开始放开自己的那一部分人。

## 开放心态与行动力的优势 🌟

了解了过度自我和性能量觉醒的表现后，我们自然会问：什么样的人更容易建立良好的社交关系呢？生活当中，我们就会发现思想越开放、行动力越强的人越容易有异性。

以下是这类人的特点：

*   **主动接近**：遇到异性会主动接近，行动力较强。行为上并不保守。
*   **性格开放**：性格相对来说比较开放，然后比较好接近的那种人往往比较容易受异性喜欢。能跟女生开玩笑，说两句就敢去动手动脚（注：此处指打破僵局的肢体互动，非冒犯行为）。
*   **社交能力强**：社会活动能力比较强，谁都能搞好关系。比如学生时代的小混混、小太妹，在学校里面人缘常常不错。
*   **注重形象**：注重自己的形象，比较看重脸面，有一定的虚荣心。

## 深入探讨：虚荣心的本质 🎭

上一节我们提到“虚荣心”，这是一个值得深入探讨的概念。我告诉大家，虚荣心完全是荷尔蒙的衍生物。

很多人常说虚荣心要不得。但是大家有没有发现，人是从什么时候开始有虚荣心的？一定是从性意识开始有的时候，他才开始有虚荣心。开始有性意识的时候就开始有虚荣心。你没有性意识之前，人是没有虚荣心的。

人开始有性意识的时候，虚荣心就会变得更强。当然，个体存在差异。就我个人而言，我不是那种特别有虚荣心的人。例如，网络上有多少人关注我，我根本不关注。我觉得别人喜欢的是我在网络上塑造的虚假形象，而不是真实的我。

对于“找个美女做女友就是虚荣”这种说法，我个人并不认同。我觉得纯粹就是感官上的愉悦，并非为了面子。实际上，经历过会发现，这有时反而可能带来负面体验，让朋友间的关系变得尴尬。

## 总结

本节课中，我们一起学习了过度以自我为中心会首先让自己感到别扭和不舒服，进而影响人际关系。我们探讨了这种行为模式可能与青春期性能量觉醒有关，并观察对比了生活中那些思想更开放、行动力更强、社交能力更突出的人所展现的优势。最后，我们对伴随性能量出现的“虚荣心”进行了一番探讨。理解这些，有助于我们更好地进行自我认知与社交互动。
