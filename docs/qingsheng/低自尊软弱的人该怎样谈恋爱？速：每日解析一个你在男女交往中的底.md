# 恋爱心理课 01：低自尊人格的恋爱困境与改变之道 🧠

在本节课中，我们将深入解析低自尊人格在恋爱中的核心问题，并学习如何通过建立健康的自我认知和行为模式来获得真正的吸引力与尊重。

## 概述：欲望、捷径与低自尊的陷阱

一个人的欲望本身没有错，但关键在于如何实现它。欲望与正确的道理并不冲突。许多人之所以会舍弃道理，是因为太想达成欲望，所以总想走捷径。

所有去学PUA技巧的人和低自尊人格都犯了同样的错误：不肯一步一步地走过去，而是幻想存在某种技巧或捷径。这种心态会导致人做很多蠢事，并且还不自知。其根本原因在于没有自尊，没有自信，因此愿意盲目相信他人，将自我肯定和判断机制全部“外包”，自己不再动脑思考。

这样的人对外表现就是软弱可欺、好糊弄、在与人相处时地位低下，因为被人看不起。没有自尊，在与人相处时就会像个傻瓜。

## 低自尊的本质：依赖、幼稚与逃避

上一节我们提到了低自尊的表现，本节中我们来看看其本质。控制欲望本身不是关键，有欲望没有错，但需要通过正确的方式去达成。许多人存在的问题是：有欲望，却不想通过那种比较困难的方式去实现。越好的东西，越需要我们付出努力和勇气，但低自尊者往往不肯。

低自尊说白了，就是**依赖性过强，毫无独立性和进取性**。它是一种心理上的幼稚和行为上的低能，既没有面对问题的勇气，也没有解决问题的能力，更没有克服困难的吃苦耐劳精神。

其呈现出来的状态就是懒散、贪玩、怯懦、祈求、底线低、没有荣誉感、没有责任心、没有羞耻心等。这样的人却往往还想着天上掉馅饼，癞蛤蟆想吃天鹅肉。学PUA的人正是如此。

## 核心问题与典型表现

肯定有人会问，这应该怎么办？低自尊人格的主要问题在于：**自己想要好的，却不愿意去遵循道理，不愿意一步一步来，不愿意承受获取过程中的痛苦**。因此，他们往往采用比较容易的办法去回避问题。

以下是两种典型表现：

*   **用容易的方式换取虚假安全感**：例如，有些人在学校里想要被人保护，就去给“大哥”跑腿。结果“大哥”看他好欺负，反而经常欺负他。因为给人跑腿是相对容易的方式，他不懂得用自己的能力、头脑和双手去面对和解决问题。
*   **不懂拒绝，陷入困境**：例如，有人害怕女性跟他借钱，不懂如何拒绝。结果自己掉进坑里出不来。当对方说“你这样还追我？你根本不配爱我”时，低自尊者甚至会想：“我爱你没错，但爱你不代表我要牺牲我自己。” 这就像那句极端的话：“你连我的屎都不敢吃，还敢说爱我？” 这典型地反映了想要好的，却不愿承受过程痛苦，总想用容易方式，结果犯贱、被人糊弄、不动脑、不付出努力、没有进取心，最终被人瞧不起。

你们的痛苦都来源于此：得不到认可、得不到尊重、交不到朋友、追不到女孩。所有学PUA的人一定都有这个问题，学得越深，越感觉完全没有存在感。

## 改变低自尊的五步实践法

认识到问题是第一步，接下来我们看看如何系统地改变。以下是五个核心的实践步骤。

### 第一步：提高对自己的要求 🔼

一个人要摆脱低自尊，首先必须提高对自己的要求。很多时候你会疑惑：为什么我做了那么多，女神还是不喜欢我？因为你没有意识到，你做的事情在别人眼里可能“就是一坨狗屎”。由于平时对自己没要求惯了，做出来的东西其实很糟糕，但你自己却觉得“付出了很多努力”、“已经做得很不错了”。实际上在别人看来，这只是“一般般”甚至“不及格”。

**公式：自我要求标准 > 自我感知的努力值**

只有当你提高对自己的要求时，你才会发现，自己觉得一般的东西，在别人眼里可能已经是“哇，做得太漂亮了”。通过这种方式，喜欢你的人才会越来越多。这才是一个人有魅力、有吸引力的根本。如果什么事都对自己没要求，呈现出来的状态和处理事情的结果都是马马虎虎、敷衍了事，别人不可能喜欢你。

真正提高要求意味着：把自己最好的一面呈现在别人面前，心里就算有再大的痛苦和委屈，也要看起来帅。什么是英雄？英雄的泪是不能被别人看见的。有再大的痛苦都只在心里流，再难受也要看起来帅。这叫做对自己有要求。

### 第二步：建立自身完整的价值体系 ⚖️

对于一些自己没做好、条件不成熟的事情，**宁可拒绝**。例如，如果你的女神不是真的喜欢你，宁可不在一起。你要的是爱情，是一段美好的、愉悦的、有激情的、好玩的关系，而不仅仅是“得到这个人”。许多“舔狗”的问题就在于，他们只是为了得到“这个人”，而不是追求一段高质量的关系，因此陷入了错误的互动模式。

当对方根本不把你当回事时，你不会觉得受伤害、难受或生气。一个有完整价值体系的人会敢于表达：“我看到你跟那个男的在一起时，心里涌现出无数情绪，那一刻我知道我是爱你的，但你的行为伤害了我，所以我们不要联系了。” 你们敢这样说吗？低自尊者永远说不出这样的话，因为你们没有完整的价值体系，不知道与人相处时应以何种方式，只会跪舔，给人感觉没个性、没脾气。谁没有脾气？当别人伤害你、欺负你、利用你时，你怎么可能没脾气？很多人只是把委屈吞下去，然后继续奴颜婢膝地去谄媚别人。

### 第三步：增强责任心 💪

这与第一条有些类似。增强责任心意味着：**与人相处时，一定要尽心尽力让别人感受到你的好、你的付出和用心**。

责任心背后其实就是自尊心。举个例子，跟女孩子出去约会，责任心强的人会想：“别人花时间出来跟我约会，我一定要安排一个完美的约会，让她开心、愉快。” 为此你会做很多安排，考虑得更周全。责任心强的表现就是：“你跟我在一起，我一定要给你最好的。” 这同样适用于任何关系：你为公司尽心尽力，老板会喜欢你；你对朋友仗义、两肋插刀，朋友会认可你。

**代码示例（伪代码）：**
```
if (与人相处) {
    目标 = 让对方感受到最好的体验；
    行动 = 尽心尽力地安排与付出；
    结果 = 建立情感连接与认可；
}
```

一个人没有自尊，情商通常不会高，与人相处也难以产生真情实感。人和人之间要发生亲密感和情感，前提就是你必须有很强的责任心，才会与人发生融洽的情感，让别人认可你这个人。

### 第四步：明确自身的原则和底线 🚫

对于一些明显是欺负、利用和愚弄你的事，**一定要学会用妥善的方法拒绝**。如果学不会妥善拒绝，至少要学会发火。这意味着你自身是有原则和底线的，碰了这条红线，对不起，我宁愿跟你绝交，即使我再喜欢你。

例如，你再喜欢一个女的，她背着你跟别人劈腿，你再爱她也只能选择分手。这就是底线。如果这次原谅了，就会有下一次。再喜欢，也只能说：“你伤了我的心，我不会容许你有下一次。” 这叫做原则和底线。

### 第五步：运用灵活的手段与大脑 🧩

与人相处、做事情，方法不是只有一种。特别是对那些被所谓“话术”困住的人：跟女孩子说话不是只有一种方法，搭讪也不是只有一句话。你要学会用你的脑子。

为什么你们会去学PUA？就是因为不愿意动脑。你们越愿意动脑，就越不需要学那些套路。你可以察言观色，可以观察情况，可以观察对方手上的物品，可以用各种各样的方法去搭讪。**脑子越不用越不会，越用才会越会**。学那些套路背后，是你的能力永远无法提升，你的勇气、胆量等各方面都不行。

只有放弃那些可以依赖的“拐杖”，以自己本来的样子走出去，你的能力才会提升。这就是我们说的：**你要有灵活的手段，就必须放弃那些可以依赖、可以依靠、可以遮掩你的东西**。只有这样，你才能获得真正的安全感。

人怎样才会有安全感？不是躲在技巧、惯例、手机或电脑背后，而是真正走出去，面对面跟人说：“今天晚上跟我回家吧。” 你敢说出这句话的时候，才真正叫做有安全感。除此以外，都没有。

## 总结与核心心法

今天课程的核心是：**对于自己想要的，一定要勇于争取；对于自己不要的，一定要学会拒绝**。

为什么这么多人被看作“屌丝”？因为你们的行为模式高度一致，且这些模式是自动的，你的脑子甚至无法控制。因此，真正的改变在于心态上的改变。只有从心态上有所改变，你的整个行为模式，包括外界对你的反馈，才会有一个巨大的转变。除此以外，什么也改变不了，而且会越来越退步。

这不是危言耸听，因为很多人已经尝试过却找不到答案，才会来到这里。可能在学习过程中会被严厉对待，但说得难听点：我骂你，总比别人把你拉黑要好。

希望本节课能帮助你开启新的认知，在新的一年，让自己有一个新的开始和真正的改变。
