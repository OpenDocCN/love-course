# 二月《核心自信》（综合版）：007.7：接纳自我，去掉条件化自信_note

在本节课中，我们将要学习如何接纳真实的自我，并摆脱依赖于外界标准的“条件化自信”。我们将探讨为何无条件接纳自己是建立核心自信的基石，以及如何识别和去除那些阻碍我们真正自信的“条件”。

## 概述

每个人与生俱来的特质，如生命、身高、长相，都是无法选择的。许多人难以真正接纳这些先天性的部分。例如，人们常会质疑自己的身高或外貌。

然而，外界关于成功与审美的标准是由社会制定的。这些标准并非一成不变，也未必属于你个人。

## 重新定义成功

上一节我们提到了外界标准，本节中我们来看看“成功”这一概念。社会常将成功与财富、地位挂钩。

但快乐富足的人生、乐善好施的行为，同样是一种成功。这里并非要重新定义成功，而是强调你应该找到自己真正有热情的事物。完成它，对你而言就是成功；即便没有完成，尽力尝试了，也算成功。

你的成功不应由外界定义，你不应活在外界的标准里，而应活在自己的标准里。必须想清楚这一点，否则你的人生将被外界控制。

## 审视审美标准

审美标准同样如此。大眼睛、小眼睛、胖、瘦，这些潮流随着时代不断变迁。例如，20世纪以前全球男性平均身高可能低于1.7米，那时身高并非普遍困扰。

为什么现在男性低于1.7米就可能不自信？因为你在活在别人定义的标准里。别人的标准并非真理。

当你坚持认为自己的身高不是问题，那么它对你而言就不是问题。如果没有人提及，你可能根本意识不到这是个“问题”。但当有三个人、十个人、甚至成千上万人说你矮时，你可能就会崩溃。

我希望你能做到，即使全世界都嫌弃你的身高，你依然能够爱自己。你不需要对抗世界，但必须有面对世界的勇气。

## 自我接纳与外在改变

很多人建议我割双眼皮，但我从未有此想法。我很清楚，如果我自己都不能接纳和爱自己，我将永远无法找到真正的自信。

对于那些喜欢整容的人，这本质上是对自我的不接纳，且可能是一个无底深渊。今天割双眼皮，明天可能就想垫下巴，后天又想削骨。颜值和审美是时代的产物，你不应陷入无止境的自我摧毁中。

广告不断宣传所谓的完美形象，目的是让你自卑，从而追逐他们的产品以获取利润。他们所说的并非真理。

因此，即使你的外表不符合当下大众审美，你也应学会欣赏自己的独特，学会爱自己。

我个人不推荐整形，但推荐“微调”。例如，矫正龅牙、治疗严重痘痘、应对秃头，这些是合理的。这如同学会护肤，属于自爱的范畴。但涉及严重动刀子的项目，我一概不推荐，那才是对自我的不接纳。

## 条件化自信的陷阱

许多人在不接纳自己的基础上，衍生出嫌弃自己情绪，这成为自卑的根本原因。

许多人内心有一个强大的错误假设：**只有各方面优秀的人才有资格自信**。当看到现实中不够优秀却很自信的人时，有自卑情节的人可能会愤怒：“他那么差，为什么还自我感觉良好？”

问题的根源恰恰在于此。那个自我感觉良好的人，他的良好感觉不太受外在条件限制。这是一种弥漫性的个体状态，外在条件的好坏、成就的多寡，很难威胁到他内心的核心自我。这就是剥离了外界一切条件的自信。

你觉得这不正常、不合理，正是因为你内心持有那个强大假设：**人只有表现良好、取得成就、各方面优秀，才有资格自信**。

然而，真正决定自信的是你的核心自我。当一个人将自我感觉良好“条件化”时，他就永远无法获得真正的自信。因为**自信的本质在于自我接纳**。自卑者之所以自卑，正是因为他们内心无法完全接纳自己。

这种不能自我接纳，导致他们认为自信是需要条件的。这些条件可能表现为必须达到某些标准，或必须在某些方面超过同龄人，他们不允许自己在某些方面不行。

有自卑情节的人在取得一定成就后，短时间内会表现出自信，甚至无比自信。但这种自信无法持久。一段时间后，他会重返自卑状态，或者需要持续保持成就来维持自信。这会让他非常疲惫，无法真正相信自己。

疲惫的根源在于对“不优秀”或“没有成就”的防御，即对于那个不够优秀、缺乏成就的自我无法做到完全接纳。尽管通过努力，这个不够优秀的自我可能已不存在，但你对它的担心依然存在。

只要这份担心存在，一个人就需要付出更多能量去抗拒它，因而无法获得真正的自信。一个对自己不够优秀、缺乏成就心存防御的人，怎么可能有真正的自信？防御本身就意味着对自己充满怀疑。即便外在条件改变了，内心那种糟糕的核心自我假设并未随之改变。

## 建立无条件自信

我们前面说过，真理和技能是难以穷尽的，世界上也不存在全能。优秀也只是一种“支撑”，即条件化。依靠外界，你无法建立真正的自信。

同时，当你依赖外界的成功来获得自信时，你依然活在别人的世界和标准里。因为当你拥有绝对自信时，你就不会依赖任何支撑。只有在你不够自信时，才需要依赖外界及其制定的标准。所谓的成功，也是外界认为的，而非你自己定义的。

因此，你必须能够认识自己，明白你是谁，并做到无条件接纳自己。如何做到呢？别忘了我们自尊课程的内容，这里不再重复。你应能做到随遇而安，乐得其所。

## 总结

本节课中，我们一起学习了自我接纳与条件化自信。我们探讨了社会标准与个人标准的区别，分析了将自信“条件化”的陷阱及其带来的疲惫与不稳定性。核心在于认识到：**真正的自信源于对核心自我的无条件接纳**，而非依赖于外界的成就或认可。通过审视并放下内心的错误假设，学会欣赏和爱真实的自己，我们才能建立起稳固、持久的核心自信。
