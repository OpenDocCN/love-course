# 二月《核心自信》（综合版）：009.9：认识真正的自己 🧘

在本节课中，我们将要学习如何认识真正的自己。这是建立稳定自信的基石。我们将探讨自我价值感的来源、完美主义的陷阱，并学习如何将自我价值与外界评价分离开来。

## 自我价值感的来源

上一节我们介绍了课程主题，本节中我们来看看自信通常何时出现。大部分人的自信和自我感觉良好，往往与外在条件的满足相关。

以下是常见的例子：
*   考试获得第一名。
*   工作能力突出，获得奖金。
*   受到父母或公司领导的夸赞。

在这些时刻，人们特别容易感到自信。这表明，大部分人的自我价值感依赖于外在条件的满足。

## 外在依赖的问题

然而，问题恰恰出在这里。因为外界条件是无常且易变的，它的丧失必然会导致自信的动摇和自卑的产生。

以下是依赖外在价值感可能带来的后果：
*   因某些事情未做好而不断责备自己。
*   怀疑自身能力，产生“我怎么这么没用”、“我怎么这么蠢”等想法。

这里并非指做好事情不能开心，或搞砸事情不应反省。核心观点是：**你的自我价值感不应完全受外界成败影响**。外界总是无常的，没有人能一直成功。十件事的成败，并不能否定你生而为人的根本价值。

## 建立稳定的自我认知

因此，只有当你的自我价值感不再完全依赖于外界条件时，你的自信才会趋向稳定。这意味着，你必须将外物与自我核心分离开来。

你需要明确：
*   你就是你。
*   你不等于你的工作（`你 ≠ 你的工作`）。
*   你不等于你的业务能力（`你 ≠ 你的能力`）。
*   你不等于你的外在或身份。

除了你的“本我”或生命本身，不要让任何其他事物影响你对自我的核心认知。你所有的技能、知识储量乃至认知，都属于你生命以外的东西。你不会某项技能，就不是你了吗？这显然不成立。

因此，除了生命本身，其他一切皆是附属。外界的成功与否、拥有的多与少，都无法左右你的本质。你必须与外界的一切评价和结果脱钩。如果你还将自身价值与外界挂钩，那么你的状态会随外界变化而波动，永远无法培养出绝对的自信。

当然，这并非主张不去学习技能或追求优秀。掌握更多知识与技能能让生活更好，但你的**自我价值感、自信和状态，不应源于此**。

## 完美主义的陷阱

同时，你会发现生活中很多人是完美主义者，这同样是没有真正认识自己的表现。

什么是社交中的完美主义？它表现为：
*   要求自己必须幽默风趣。
*   要求自己所有话题都能打动别人。
*   为了被喜欢，只能说“正确”的话。
*   认为自己身上不能有异味，永远不能犯错或显得笨拙。

许多人在互动中，将过多注意力放在完美的说辞上，希望以此留下好印象。这其实是内心缺乏自信的表现，他们试图用口头语言达成一切，却忽略了自身态度和非语言交流才是更重要的。

在一个聚会中，有些人可能什么都没说，但他的自信与从容，已经向外传递了足够多的积极信息，他依然是迷人的。社交完美主义的**核心逻辑，是对真实自我的不够接纳**，因此才努力制造完美形象以获得认可。

真正自信的人，并非完美或无缺点。世界上没有完美的人。他的自我价值感，不源于某个优点或缺点，而是源于内心深处对自我的相信。

如果你想获得真正的自信，就必须打破完美主义对你的干扰。前提是先全然接纳自己。

## 关于错误与生命价值

关于犯错，很多人会因此感到羞愧。但你要明白：每个人都会犯错。那些错误并不能代表你。

你是在生命之外的事情上犯错。你的生命本身有错吗？它就在那里，不偏不倚，不增不减。因此，你的生命不会因为成就而增加，也不会因为错误而减少（`生命价值 ≠ 成就 + 错误`）。

## 总结与核心法则

我们在“核心自信”中探讨的自我、自尊、情绪以及“你是谁”，都是为了帮助你认识真正的自己。

请记住这个核心法则：**你越清楚自己是谁，你就越自信。如果你不清楚，外界就会来帮你定义。**

外界一句“你好帅”、“你好厉害”就让你开心，外界的一些质疑贬低就让你难过。这正是没有认识到自己是谁的表现。

因此，你应该**经由自己来决定你是谁，而非外界**。

本节课中，我们一起学习了：
1.  识别自我价值感对外在条件的依赖。
2.  理解这种依赖带来的不稳定性。
3.  学习将自我核心价值与外界评价分离。
4.  认清完美主义的本质与陷阱。
5.  接纳错误，认识到生命本身的价值不变。
6.  掌握“自我定义”的核心法则，建立内在稳定的自信根基。
