# 二月《核心自信》（综合版）：课程040.40：潜意识自信 🧠

在本节课中，我们将要学习如何通过深入检测并改写负面信念与限制性观念，来从根本上改变潜意识，从而建立稳固的核心自信。

上一节我们探讨了信念对自信的影响，本节中我们来看看如何具体操作，深入潜意识层面进行工作。

你必须深入检测自己所有的信念，把所有的负面信念都挖掘出来。

以下是常见的负面信念示例：
*   我很笨。
*   我学不会。
*   我完不成。
*   我不值得被人爱。
*   我应该谨慎。
*   我不能说太多话。
*   我不能太活泼。
*   我不能表达太多。

然而，一次被拒绝不代表永远会被拒绝。一次不会不代表永远不会。一次做不好不代表永远都做不好。正如我们之前的课程所讲，那些东西并非你的生命本身。

你的自我价值感必须脱离生命以外的所有一切。所以你需要把所有负面以及那些限制性观念全部深挖出来。然后按照我们之前的方法，一个个来解决掉。这是一个庞大的工作量，但绝对是值得你花费时间去做的一件事情。

当那些限制性观念被解决、负面信念被改写时，你的信念就开始发生改变，你的潜意识也就会发生改变，自然你的人生也就开始转变。

最后需要强调的是，除了你的限制性观念必须被解决以外，你的负面信念也最好全部清零。我们常说“人心不古”，这是真的吗？我们也说“人心隔肚皮”，这是真的吗？我们常说“不要太轻易相信别人”，这是真的吗？

还有太多的负面信念就不一一举例。记住，这个世界上所有的观念、所有的信念都只有一定的正确性，它不会是唯一的真理。人世间的温暖及相互帮助又是哪来的呢？

因此，从一个更高的层面来说，这个世界上没有任何一个信念是绝对的真理。你永远可以在任何一个信念的对立面找到不同的证据。因此，如果你想要变得积极、变得自信，你一定要清除所有的负面信念，建立起正面的信念。

假以时日，如果你想再成长、再进一步，那么你就需要去掉所有的好与坏评判。就像我们之前的课程里面所讲的，不再评判，不再贴标签，只是如实地去观察它们，去感受它们。

当然，如果说到这里就扯远了。我们这个课程只讲关于自信的建立，不讲那些关于灵性的成长。所以，如果你想要改变自己的默认模式，也就是改变你的潜意识，你想要增强自己的自信，那么这节课一定要好好去做。因为这是解决你自卑的根本。

如果你能做到这一点，并按照课程的要求去做，那么你将开始掌控你的生命。你的人生不会再随波逐流，更不会被外界所操控。

整个自信的课程讲到这里就差不多要结束了。最后我们做个总结。

我们的终极自信课程讲了7个板块：身份、信念、需求与反应、行为模式、意图清晰、你是谁、潜意识。

*   **身份**讲的是别人赋予你的身份，以及你自己赋予自己的身份，以及如何保持高价值身份。你的身份决定了你会如何去回应外界。
*   **信念**决定了你的自我价值感和你看到这个世界的窗口，以及决定了你的行为举止。
*   **需求与反应**讲的是这个世界的绝对价值。更少的需求决定了你更少的反应，而更少的反应决定了你的价值。
*   **行为模式**主要讲的是如何利用你的生理来控制自己的自信程度，以及影响自己的状态。行为模式还包括了你的肢体语言、你的眼神、你的口头表达，以及理所当然感和自我娱乐。
*   **意图清晰**主要讲的是你是如何自己阻碍自己，并且你是如何导致思维混乱的原因。当你的意图及意识开始清晰的时候，你的自信会开始显现。
*   **你是谁**讲的是你对自己的认识。你越清楚自己是谁，你也就越自信。
*   **潜意识**就是上面几节课所讲的内容。
