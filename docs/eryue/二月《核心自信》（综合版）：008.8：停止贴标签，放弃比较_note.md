# 二月《核心自信》（综合版）：008.8：停止贴标签，放弃比较_note

在本节课中，我们将要学习一个阻碍我们看清事物本质、影响人际互动深度的常见习惯：贴标签与比较。我们将探讨其背后的心理机制，并学习如何通过调整注意力焦点来提升社交状态与自信。

## 概述：标签如何蒙蔽双眼 🎭

喜欢贴标签，是一个阻止自己看到本质的愚蠢行为。很多人在和外界互动时极其喜欢贴标签。例如，认为“这个人好小气”、“这个人好暗黑”、“这个人好多负能量”等等。

当你贴完标签的时候，你就决定了你的关注焦点。也就是说，如果你觉得这个人很小气，那么他后面所有的大方行为你都会选择视而不见。因为你觉得他就是一个很小气的人，哪怕你看到了他某一次的大方，你也会用其他理由来掩盖，例如“他是因为觉得对不起我，所以才请我吃饭的”。

所以，停止贴标签一定要停止，这会阻碍你看到真正的真相，而且会让你和外界架起一道难以跨越的鸿沟。

## 标签是主观感受，并非客观事实 🤔

可能你会说“这个人就是好小气”，但是你要明白，这只是你主观的感受，并非是客观事实。哪怕是客观的事实，这个人就是很多人都说他小气，你也不要去贴标签。你只需要去感受你每一次的感受就行。

因为如果你喜欢给外界贴标签，每当贴完以后，你和那个人在互动的时候，你都不是在和那个人在进行互动，而是在和他这个人的标签在互动。这就会造成你和这个人互动时的分离，你们的互动也无法进入更深层。也就是说，你的吸引力不会显现出来。

所以，改掉自己喜欢贴标签的习惯。无论是贴好的标签还是贴坏的标签，你都需要改掉这个习惯。

## 心理投射：你所痛恨的，正是你拥有的 🔄

这里多说一句，当你特别讨厌别人身上的某项特质，例如你痛恨其他人自私、小气、自大、自以为是等等。然而，你所痛恨的，正是你身上也存在。

在心理学上有个名词叫**心理投射**，也有人叫映射。简单的来讲就是：**我们在他人身上只能看到我们已经学会看到的**。而且，在一般情况下，只能看到在自己身上看到的。就是我们的自私，教我们去观察其他人的自私。

我们有时对其他人的评价会比较恰当，但有时则会夸大。最糟糕的是，有时我们对其他人的评价完全是给予想象。这就是精神分析学家所说的**投射**：一己之心，度他人之腹。

“我总是对别人感到失望。”这样的话我们经常听到。其他人真的那么令人失望吗？还是你自己太容易感到失望了呢？事实上，可能两者都有。

就是可能有些人会说，有些人客观的来说就是很自私，就是很自大。当然，如果你看到某个人很自私，亦或是看到某个人不好的一面，如果你没有情绪起伏，说明你还较为客观。但是如果你对别人身上的自私，又或是某个缺点极为痛恨，那么换一个角度来讲，你不是在恨他，而是在恨你自己。

因此，在人际交往时，我们要去进行潜意识评价，无论美丑，你不要贴标签，否则你一定会陷入逻辑。你只需要用意识去感受就行。

## 比较的陷阱：陷入社会标准与二元对立 ⚖️

同样，当我们进入社交场合时，你要做的就是纯粹的去感受外界。如果你去评判他人，亦或是给他们贴标签时，你又掉入了社会那个肤浅的标准当中。例如，你觉得“这个人好帅”、“那个人好有钱”。

自然，你贴完标签之后，你就会陷入和他们的比较当中。这会让你处于和外界竞争的一个位置，同时会让你进入二元对立。你之所以觉得这个人有钱，那是因为你觉得自己没有钱。

于是，当你没有满足这样的社会标准时，你的自信、你的状态就会开始下降。当然，如果你是处于一个超然的态度，那你不受这些规则的限制，你只是实事求是的说出来“这个人很有钱，确实很有钱”，那么他对你的状态就不会产生影响。

然而，对于新人来说，最好不要这样，否则很容易让你泥足深陷。

## 注意力分离：社交不自在的根源 🌀

很多人在社交的时候，他们和整个互动是分离的。也就是说，他人在这里，但是他的注意力却放在对方的穿着、仪态、财富、颜值等等。然后，把他所看到的和自己去进行对比。

如果他自己更好一些，那么他可能会更有优越感一些。如果他输了，那么紧接着他就会开始审视自己，评判自己。于是你越来越不自在，你会发现你和别人的互动也越来越不自在。这会形成一个社交微观动态的传达，对方也会接收到这种信息。

## 解决方案：放弃比较，专注沟通本身 🎯

所以，解决方法是什么？**放弃比较**。你要把所有的注意力放在沟通本身上，而不是分离自己的注意力。“哇，他好有钱，他怎么样怎么样”，不是这样。

大部分人都喜欢暗自比较。他们想在不如自己的人身上找到自己的自信，然而却又在比自己强的人身上看到了自己的自卑。

所以你一定要记住，当你和外界互动的时候，你所有的注意力都要放在你对面的这个人身上，不要一心二用。如果你心不在焉地跟这个人互动，其实你的注意力都放在别处：你听着他说话，你就在想“哇，他穿着这么好”、“哇，他这么有气质”、“他那么有钱”。

这就是一心二用，这就会造成你整个人的分离。于是，你整个人的状态一定会下降。接着，你会被对方所打败。

为什么？因为当你的注意力分散到去评判他所有的价值的时候，你就会暗自和自己去进行对比。你之所以觉得他那些地方那么厉害，就是因为你那些地方都不如他。暗自对比最后的结果是什么？就是你被打败。当你被打败，你的自信会下降，你的状态会下降。

所以，很多人都喜欢暗自比较，想在不如自己的人面前找到自信，但结果在比自己更强的人面前，他看到了自己的自卑。

## 总结：用意识感受，而非用标签评判 🌟

本节课中，我们一起学习了贴标签与比较的危害。你一定要停止比较，停止贴标签。你要用你的意识去感受整个外界。你所有的注意力都应该放在面前的这个人身上，而不是他背后所附带的价值上。通过放弃评判和比较，你将能更真实地连接他人，展现更稳定、自信的自我状态。
