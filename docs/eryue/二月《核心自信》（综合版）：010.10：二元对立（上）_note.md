# 二月《核心自信》（综合版）：010.10：二元对立（上）_note

在本节课中，我们将要学习“二元对立”这一核心概念。我们将探讨二元对立思维是如何形成的，它如何影响我们的情绪与判断，以及它与优越感、自卑感之间的深刻联系。

## 概述：什么是二元对立？🤔

二元对立，源于我们对于事物的判断。我们将事物分为美与丑、善与恶，这些区分源自我们自身的观念和认知。如果站在更高的角度，美可能变为丑，善也可能变为恶。

上一节我们介绍了二元对立的基本定义，本节中我们来看看它是如何在我们成长过程中产生的。

## 二元对立的起源：从和谐到分别

人在婴儿时期的快乐指数最高，处于一个和谐、自然、本真的世界。当我们长大以后，开始有了分别心。我们开始区分黑与白、对与错、好与坏，并将二者完全对立起来。

进而，我们产生了**二元对立思维**。从此，世界被人为地区分开来。这就是二元对立的世界，出现了美与不美、善与不善、喜欢与不喜欢等一系列对立。

所以，我们人生的痛苦从此多了起来。但是你要明白的是，善恶美丑是相互依存的。没有了一方，另一方也就不存在。任何一个东西的存在，都是和对立面同时产生的。

## 对立与统一：相互依存的双方

善对应的一定是不善，恶对应的一定是不恶。两者不仅对立，而且互生同生。一方以另一方的存在为前提，一方消失了，另一方自然也就不存在。两者是相互对立，也是相互依存的。

而且，两者是**对立统一**的关系。如果把两者割裂开来，美和不美、善和不善、对和错、应该和不应该，都是人为认定的。

但是这个认定，为我们带来了很多痛苦。为什么这么说？当你发现了所谓的美，你会感到开心；当你发现了所谓的丑，你会感到不开心。而你的不开心和开心，是你脑子里的概念赋予你的。

以下是理解这一点的关键步骤：
1.  **事件发生**：例如，你踩到了大便。
2.  **概念赋予**：在你的概念里，大便是“坏的”、“不好的”。
3.  **情绪反应**：这个“不好”的概念，给你的身体下达产生负面情绪的指令。
4.  **结果**：你的身体照做，你感到不开心。

如果是一个一岁的宝宝踩到了大便，因为他尚未形成二元对立的思维，他的情绪不会受到同样概念的干扰。他不一定会开心，但至少不会产生负面情绪。

所以你之所以不开心，是因为你将一件事定义为“不好”。这个“不好”的概念，指令你的身体产生负面情绪。当然，这里并非说你应该像婴儿一样处于无知状态，而是说，你应该在有知的情况下不再去定义，只需接纳事实，感受事实本身。因为事实已经发生，没有必要再给自己创造一个二元对立的世界，从而让自己产生负面情绪。

因为，即便你让自己产生了负面情绪，也改变不了已经发生的现实。我们一直是以自己赋予现实的意义来感受现实。我们所感受的不是现实本身，而是现实被我们所赋予的意义，或者说，是我们对于现实的解释。

## 痛苦之源：抗拒事实

人绝大多数的痛苦，就是对已经发生的事情不接纳。他在抗拒那个情境，同时也在对抗自己。这里“接纳事实”绝非不作为（后续课程会详述）。

言归正传，人把世界分为好和坏、上帝和魔鬼、高和低、神圣和肮脏、纯净和污秽、天堂和地狱。这些都是永远不会相交的平行线。

一旦你内在将整个存在如此分解开来，你就不可能达到自由的状态，也不可能达到持久的快乐。记住，你存在的状态，永远都不要被外界的情形所奴役。当然，这很难，需要长时间的修持。

关于二元对立，我们后面还会有一节课继续讲。但是本节课，我们要回到“优越感与自卑”这个主题上来。

## 优越感与自卑：二元对立的体现

你要知道，优越感和自卑，是二元对立的两个表现。在某些时候，你会产生优越感。例如，在你擅长的领域，你擅长历史。当和他人谈论历史时，你侃侃而谈，加上他人的赞美，你可能会感到飘飘然，产生优越感。基本上来说，你一定会有。

以下是一个具体例子：
*   **情境**：作者十来岁时从乡下搬到县城。
*   **感受**：每次下乡，相比同龄孩子，会产生莫名的优越感，觉得自己见过更多世面，更时尚。
*   **核心**：这就是优越感。

想想看，在你所擅长的领域或某种情境里，你是否也会产生优越感？而麻烦，正是来自于你的优越感。优越感其实是一种很浅薄的状态。

我们前面说过，二元对立一定是相互依存。没有长，何来短？没有丑，何来美？当你感觉自己在某个地方是“高处”时，你一定会感受到自己在某个地方是“低处”。

当你在不如你的人面前拥有优越感时，那么在比你强的人面前，你一定会有自卑感。当你在某个地方感受到了优越感，那么你也一定会在其他地方感受到自卑。

所以，你一定要小心你的优越感。每次当你感到优越感的时候，你都应该警醒自己。如何警醒？你应该更有深度一些，不要这么浅薄。

## 实践建议：超越优越感与自卑

以下是具体的行动指南：
1.  **对待不如你的人**：以后每当碰到不如你的人时，你所有的表达都应该去掉优越感。比如有人问你所擅长领域的问题时，如果你带着飘飘然的感觉去回答，除了暴露自身的浅薄，更会加深你日后的自卑。
2.  **对待比你强的人**：以后每当碰到比你更强的人时，你带着欣赏的角度就可以了。欣赏是一个开放的角度，代表你内心的暗自较劲没有被激发。但是，如果你是带着羡慕或比较的角度，你则会把他拉到你的对立面，这就是二元对立的开始，会让你变得更暗黑，同时这也是你不自信的表现，因为你被影响了。而被影响，则是在**提价值**。

真正自信的人，能够接受别人强过自己，能够欣赏别人的优秀，欣赏别人的知识储量更丰富。你学会听，学会欣赏，挺好的。别人打游戏比你更厉害，学会欣赏，挺好的。别人比你会更多技能，学会欣赏，挺好的。但是，没有什么大不了的。

因为除了生命本身，其他的一切都只是附属。

## 总结📝

本节课中，我们一起学习了“二元对立”思维。
*   我们探讨了它如何通过我们的分别心而产生，并将世界人为分割。
*   我们理解了善恶美丑等概念相互依存、对立统一的本质。
*   我们认识到，我们的痛苦常源于对事实的抗拒，以及为现实赋予的负面意义。
*   最后，我们深入分析了优越感与自卑这对典型的二元对立表现，并学习了如何通过警醒优越感、培养欣赏心态来超越它们，走向更深层的自信。

记住，你的生命不需要那些虚妄的优越感来支撑，你真正的本我也不需要那些。
