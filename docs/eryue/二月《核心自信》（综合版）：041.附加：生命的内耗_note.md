# 二月《核心自信》（综合版）：041.附加：生命的内耗 🧠

在本节课中，我们将要学习一个阻碍自信发展的核心概念——生命的内耗。我们将了解内耗的含义、表现、危害，并学习如何通过“活在当下”等具体方法来停止内耗，从而为建立核心自信扫清障碍。

如果说自我接纳是自信的前提，那么生命的内耗就是自信最大的杀手。

## 什么是生命的内耗？

简单来说，生命的内耗就是**自我搏斗**与**自我消耗**。

以下是内耗的一些常见表现：
*   **后悔**：对过去已发生的事感到懊恼。
*   **纠结**：在多个选择或想法间反复挣扎，无法做出决定。
*   **陷入消极**：被负面情绪和想法持续困扰。

为什么各方面条件都相似的两个人，一个能轻松自在，另一个却郁郁寡欢？很多时候，我们感到痛苦并非源于外界的问题或事情本身，而在于我们内心的冲突。

许多人对外界的反应过于敏感，任何一点风吹草动都会激起情绪的波澜。当你处于这种状态时，即使一天什么都没做，也会感到**充满无力感**和**身心疲惫**。这就是自我消耗。

内耗的过程就像用一把勺子，一点一点将自己掏空。因此，内耗是自信最大的凶手，也是最没有必要的一种消耗。

## 我们为何要停止内耗？

我们每天都需要应对外界的诸多事务，例如生活琐事、工作挑战和人际关系，这些已经让我们应接不暇。我们为何还要额外付出一部分精力来与自己对抗呢？

这一点值得深思。多说一句，抑郁症的根源之一，正是内部不肯放过自己，从而不断进行自我搏斗。可以说，**抑郁症是深度内耗的另一种呈现方式**。

## 如何停止内耗？

那么，我们该如何停止内耗呢？以下是几点希望能帮助到大家的方法。

### 1. 停止后悔

一位在精神病学界享有盛誉的医生曾指出，他的许多病人将大量时间花在缅怀过去上。

他们反复后悔曾经做过或没做的事，例如：
*   “要是我那次准备得再好一点……”
*   “要是我那时没有犯那个错误……”
*   “如果当时我能再勇敢一点……”
*   “如果当时我没有放手……”

这种反复的懊悔，最终会让人陷入无尽的精神消耗。

人活一世，难免有遗憾。一时的反思可以推动我们进步，但长久的懊悔只会让我们举步维艰。因此，任何关于灵性成长或哲理的书籍，都绕不开一个核心词：**活在当下**。

### 2. 理解并实践“活在当下”

对于“活在当下”，很多人并未彻底了解。其实，我们每个人的世界里都存在三条时间线：**过去**、**现在**和**未来**。

然而，我们大部分人都活在过去或未来。正如前面所说，不断缅怀过去、后悔不已；或是活在未来，不断憧憬和幻想。但我们忘了，中间的“**当下**”才是唯一的真实，是我们此刻能感受到的一切。

说到这里，可能有人会反驳：“我的生活充满问题，有太多未解决之事，我无法活在当下，无法感受自在。”

其实，这依然是活在过去。试问：**此时此刻、这一分这一秒的你，有问题吗？** 不是过去的你，也不是未来的你，就是现在。答案是：**没有**。你的内在在此时此刻没有任何问题。

但大部分人用过去和未来不断占据当下，掩盖了我们唯一存在的真实。这就是问题所在。

你们现在能明白，为何偶尔检索过去时，某段时间的回忆会一片空白吗？那是因为在那段时间里，你一直不在当下。你所有的精力都用于缅怀过去或憧憬未来。也就是说，在那段时间里，真实的你是不存在的，是被掩盖的。当你心不在焉时，你所做的事情自然不会在大脑中“存档”。

既然我们明白了三条时间线，也了解了何为真正的活在当下，那么你们应该知道如何避免因后悔而产生的自我内耗了。

### 3. 正确处理过去与未来

可能有人会钻牛角尖：“那我们永远不能回忆过去或憧憬未来吗？”

当然不是。很多时候，我们依然需要依赖过去的经验来指导现在的生活。因此，在**某些必要的时候**，我们当然需要检索以往的经历。对未来也是如此，有时我们需要对工作、生活或感情制定规划，而制定计划可能就是当下需要完成的工作。

所以，**除了必要的时候，我们应该活在当下**。因为只有这样，你才能感受到自己的真实，感受到自己的生命。

此外，切记一点：当你在检索或回忆过去时，**必须尝试以旁观者的角度去看待一切**。这样，你就能与过去的自己拉开距离，而不受当时情绪的干扰。

对于这个观点，我们举个例子：假设你需要回忆以前被分手的经历。检索这段回忆时，一定要以旁观者视角。为什么？因为如果你再次将自己置身于当时的处境中，过去难过的情绪会再次笼罩现在的你。即使此刻的你本没有问题，过去的坏情绪依然会让你现在感到难过。

所以，把自己作为一个旁观者去观察过去，你就能与过去的情绪拉开距离。过往的不开心，就难以影响到现在的你。当必要的检索完成后，**立刻回到当下**。

当你能活在当下时，生命中的许多问题都能迎刃而解。如果你追求一点灵性成长，无论是哲学、道家还是佛家，都在宣扬一个最基本的观念：**活在当下**。

---

**本节课总结**

本节课中，我们一起学习了“生命的内耗”这一概念。我们了解到内耗是自我搏斗与消耗，表现为后悔、纠结和消极，它会严重消耗我们的精力，是自信的最大杀手。停止内耗的关键在于**停止无意义的后悔**并学会**活在当下**。我们认识到时间分为过去、现在和未来，而“当下”是唯一的真实。通过以旁观者视角处理必要回忆、规划未来，并在其余时间专注于当下，我们可以有效减少内耗，感受生命的真实与力量，为建立核心自信奠定坚实基础。
