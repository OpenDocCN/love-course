# 二月《核心自信》（综合版）：011.11：二元对立（下）_note

在本节课中，我们将深入探讨自信与傲慢、谦虚与卑微之间的本质区别，并学习如何区分真正的自信与虚假的自信，最终建立不受外界评判影响的自我价值感。

---

上一节我们介绍了自信与傲慢源于不同的内在状态。本节中，我们来看看这些状态如何影响我们与外界互动，以及如何识别和培养真正的自信。

优越感只不过是**以自卑的倒立姿势在行走**。但需要明确，谦虚绝非让你卑微或低三下四，而是让你处于一个开放的姿态。当你能处于开放的姿态时，你就能够看到更多的可能性。

你不需要去吸收外界的一切。这只是为你的生命打开了一个闸口，你可以选择性地吸收你想要的东西，然后摒弃掉一些不是很适合你的东西。

然而，狂妄自满的人会在别人说出观点前就给予否认。为什么？因为他害怕自己的观点以及认知被摧毁。于是他最后那一点可怜的自尊都会消失殆尽，这其实源于他的不自信。他需要摆出一副特别的姿态，抓住最后的那一根稻草。然而这种对立只会给他带来更多的不安。

自负的人往往不愿意面对现实。他会不停地维护自身的观点，哪怕内心已经知道自己错了，但就是不承认。而自信的人则会直接面对现实。如果他错了，他反而会很坦荡地去接受。他可能会说：“你的这个观点真的挺棒，今天给我上了一课。”

对于自负的人，他的面子、他的认知，那些虚假的东西是他整个内在的支柱。如果这个支柱被打碎了，那么他的世界就会崩塌。

所以，优秀的人更在意的是自我成长这个真正的“因”。当他们发现自己的观点出现偏差，或做错了某些事的时候，他们会坦然接受自己的不正确。

你永远不要为了自己的形象而辩护。因为那样做，你是活在别人的评价里。当然，这并不是说你不能坚持自己的立场。你可以不带任何情绪地说出你的想法，并且坚持你的观点。但是，一定不要为了维护那虚假的自我形象而不断地产生辩解，哪怕你发现了自己的不对。

那如何区分你是在坚持自己的立场，还是在为自己辩解呢？

以下是简单的判断方法：
*   观察自己是否产生了情绪波动。
*   观察你的内心是不是产生了无意识的、防御性的反应。

如果你的内心已经产生了反应以及情绪波动，那么说明你就是在辩解。因为每个人对于足够确定的事情是不会产生剧烈反应的。

例如，你身高1.8米，别人说你1.6米，你会有激烈的情绪反应吗？你不会。你只会平静地告诉他你有1.8米。这就是坚持自己的立场。

反之，如果你身高1.5米，有个异性说：“你这是不是有点矮呀？”然后你激动地说：“这怎么矮了？我很好啊，如何如何……”这就是为自己辩解。因为这句解释的背后，已经产生了不安的情绪反应。

你甚至可以在辩解之上再进行辩解。比如，你明明已经起了反应，但你依然对别人说：“我没有激动。”但是，哪怕你骗得了别人，你也骗不了自己。这会让你进入一个“自我维护 -> 没有真实 -> 不断降低自信”的怪圈，因为你一直不敢以真正的自己示人。

在那些声称自己拥有自信的人中，有一部分人说的是事实。另一部分人表现出来的自信其实是假自信。也就是说，他们看似积极的自我认知，要依赖外界不断的认可。

拥有这种假自信的人是不能容忍任何反馈意见来质疑自己。尽管在某些方面他们看起来很自信，但是一旦受到威胁，他们就会马上进入自我防御状态，甚至可能会指责其他人对他们构成了威胁。也就是说，他们会不断地指责外界。

所以，**真正的自信源于内心**。这种自信不需要外部的认可，也不会一遇到威胁就崩溃。一个人对于自身价值的看法始终如一，也能善听外界的不同观点。

因此，我从来不在乎表面上的输赢，也不愿意去维护自己的形象。如果你能在5分钟之内证明我的立场是错的，或是拿出正确且行之有效的观点，我会坦然接受。这没有什么大不了的，也没有什么好丢人的。你又不是全能的神仙。

所以，当你能坦然接受自己的不足以及一些缺点，你才能完全地、真正地接纳自己，你才会培养出真正的自信。我们一直在讲，这个世界没有全能的人，这个世界上也没有任何一个人是完美的。那为何你要不停地去维护那个根本不存在的、完美的自我形象呢？

很多时候我和别人聊天时，我会说：“其实我有时候挺浅薄的，有时候挺不成熟的，有时候挺暗黑的，有时候挺小气的。”这就是我，这就是自知，且敢于谈论自己不足以及不好的一面。**当你能完全放弃维护（虚假的）自己时，你才是活在自己的世界里，你才能越来越不在意外界的看法。**

你要记住，你的本我，也就是你的生命，不会因为外界的评论而减少或增多。你就在那里。别人夸你一句，你不会变得更厉害。当然，坦然地承认自己的不足，不是叫你去自我攻击，而是坦然接受，然后再去改进自己的不足。

我们所有变得优秀的驱动力，前期可能脱离不了社会的驱动，因为我们都是社会人。但最终，你一定要脱离外界，你所有改变的驱动力一定要由外部转向内部。因为那里才是终极。所以，**最终的自信一定是要靠自己来驱动。**

记住，除了你这个肉身（身体）需要被保护外，其他心理上的一切都不需要你的保护。你越在乎什么，就越受其所累。没有什么大不了的。你对这个世界应该是充满好奇的，但是最终，是无所谓的。

所以，自信的人往往是不在乎所谓的面子的。因为他活在自己的世界里，所以他不在乎外界看他那面“镜子”。他更在乎的是自己的成长。同时，他的自信也不取决于那些固定的认知，因为真理是不可穷尽的。

我们之前一直在讲，自信更多的应该取决于你对于自己的认识。**你越清楚你是谁，你就越自信。** 所以，谦虚的背后是力量，狂妄的背后是不安。

---

本节课中，我们一起学习了如何区分真自信与假自信，以及如何建立稳固的自我价值感。

最后总结一下：为什么我们说不要在乎外界对你的看法或评判？当然，这不是说当外界攻击你时，你要像一个很挫的人待在角落里一言不发。绝对不是。你可以表明自己的观点，可以去警告他，也可以没有任何不安地直接离开。核心是，你应该保持自己的中心定力。

**本节课的核心是：你应该保持自己的中心定力。** 也就是说，你的整个自我价值感，不会因为外界的评判而动摇——人家夸你，你就开心；人家说你不好，你就不开心。让你更有自我中心感，是让你自己去感受和确立你的自我价值感，而不会因为外界评判的好与坏来影响你的情绪和状态。

这绝对不是说你碰到一些困难、指责或挑战时，就要像个怂包一样躲起来瑟瑟发抖。绝对不是这个意思。当然，你也没有必要去跟他们争辩。它的核心点在于：你的内心应该是没有（因自我怀疑而产生的）波动的。
