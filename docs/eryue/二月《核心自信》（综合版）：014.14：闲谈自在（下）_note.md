# 二月《核心自信》（综合版）：014.14：闲谈自在（下）_note

在本节课中，我们将继续探讨“闲谈自在”的核心，深入理解自信的本质、逆向合理化的心理机制，并区分初级自信与高阶核心自信的区别。我们将学习如何通过接纳自我和创造价值来建立稳固的内心基础。

---

上一节我们介绍了自信在沟通中的重要性，本节中我们来看看自信更深层的心理机制和不同层次的表现。

你是否优秀，是否满足世俗的价值观，这些都是次要的。接纳以及认可自己才是最为重要的。在与异性沟通的时候，前提条件是你要把自己看作是一个好的产品。

这样你就会有一种“我在提供价值给你”的感觉会传递出来。但如果你认为自己是一滩狗屎，或者认为自己就是一个累赘，你自己都不认可你自己，你对你自己的表达语言都充满了怀疑。

那么你整个人的状态一定是处于负面的。其他人或者说异性都不用多了解你，就会判断你是没有价值的。因为你自己都在嫌弃你自己，你自己都不相信你自己。但是假如你非常喜欢自己，你认为自己是一个好的产品。

这时异性就会看到你的自信，看到你好的状态。然后会逆向合理化的认为你是有价值的。就算实际上的你没有钱，没有车，没有房，长得也不好看，但是异性会因为你的状态，因为你的自信。

从而去合理化的去寻找你身上的优点。他可能会想，虽然看上去不怎么样，但是他这么自信，这么认可自己。那么他肯定有某个方面是特别优秀的，只是我可能暂时没有找到。但是在他找到你的优点之前。

他已经相信你有优点有价值了，说明他已经对你产生了好感。甚至可能你们已经开始谈恋爱了。甚至到最后你们分手了，他还是没有找到你具体的价值是在哪里。但是你们已经经历了一场恋爱。

他可能会想出一些非常扯淡的优点来描述你，这也就是事后的合理化。有些女孩子为什么对那些一无所有的老改犯不离不弃？他们可能会说，因为他很善良。因为他对我很好。

甚至编造出一些那个男人根本不存在的优点，这就是自我合理化。也就是你的思维必须要让你的行为符合你的逻辑。因为只有这样你的情感电路才会趋向于正常。这就是逆向合理化。

还记得前面说的那句话，就是这个世界上没有绝对的真相，所有的信息都是被描述出来的。只要他主观上认为你有价值，你就是有价值。事实上，这个世界上所有的价值全都是存在于人脑中的概念。

那么，如何去相信自己有价值，如何让自己去接纳自己呢？在这里，可以简单的讲其中的一个小点，就是一个人如果能够做到创造价值，那么他往往就能够产生一些初级的自信。比如说有钱不能让你真的有自信。

但是当你有挣钱的能力，就能够给你带来一些自信。因为无论把你放在哪里，你都可以让自己生存下去，而且还生存的不错。挣钱能力，不是说你的工作能力。

而是你在什么都没有的情况下能够想到挣当的门路去挣到钱，还能让这个钱慢慢的变多，这就是创造价值。这种人往往是精通于社交的，社交能力也是能够给人带来自信。但是我说了，这样的自信是处于初级自信。

这不是最高阶的核心自信。高阶的核心自信是来自于你的身份感，也就是“我是谁”。当你能明白，当你最终能够明白你是谁，那么你就会获得了相对稳定的自信。甚至可能你什么都没有，什么都没做。

也可以有自信，这就是一种自在的感觉，也就是无条件自信。这才是真正的自信。当你跟别人发生意见上的冲突的时候，是可以不受对方影响。只要你不受对方的影响，你往往就能够获胜。这个不是看你能不能打败他。

而是看你能不能打败你自己。如果你完全没有受影响，而且你还能够谈笑风生，而对方却气得半死。那其实你就已经赢了，对吧？那两个人互撕的时候呢，并不是谁骂的更狠，谁就获胜。谁被影响。

或者谁被影响的最惨，谁就输了。当一个人跟你开撕，你冷静淡定自若，他看上去则会像个小孩。那如果你也失态了，你们就像两个小孩子在吵架，对吧？你们两个人都是憨憨，对吧？当你外甥女说你笨。

说你傻，你会计较。你会被影响吗？当然不会。记住，越牛逼的人受到外界的影响也就会越小。被影响永远是低价值，而牛逼的人则是不断的影响他人。能理解吧？核心自信最后的境界就是不受万事万物影响的自在。

这个难吗？当然，但却是值得我们每一个人去追寻的目标。内心没有太大的情绪波动，但是你的感觉就很舒服，很自在。就比如说你在自己家的后院，今天你做了一天事情了，你感觉很累，然后回家洗了个澡。

然后躺在沙发上或者打打游戏，然后喝一罐可乐，反正就是你很舒服，就很自在。然后你很享受自己。也很享受当下这个时间。那如果这个时候有人和你说话，你也可以很自然的跟他去表达你自己。也就是说。

当你能够拥有自我愉悦的能力，你就可以在任何一个场景下获得自信。这就是我们终极版的自信里面要讲的东西。很多人跟异性接触的时候是疲惫的，是很煎熬的一种状态。这就是很不自在的感觉。但是要注意一点。

就是自在，不是让你躲在舒适区，我天天宅在家里打游戏，避开社交压力。不是，恰恰相反，人不能一直呆在舒适区里面，你不能把自己照顾的特别好，特别舒服。你要做的是什么？你要去突破。

当你去当你突破之后呢，你就会在一个原本让你感到害怕的地方重新让你感到自在和舒适。你会建立起一种能力，就是任何境遇，任何事物，无论好坏，你都可以做到去适应的你，然后做到去享受去自在，能理解吗？

OK，那我们初级版的自信呢，到这里就差不多结束了。虽然说它是初级版，虽然是入门级，但是它却非常非常的重要。就像我们之前举的例子，九九乘法表，虽然是数学里面的基础，对吧？但是它不可或缺。

你没有基础，你就不可能有上层建筑，对吧？那在我们的中级版课程里面呢，我们会讲到更多的操作，就是如何去训练自己的自信，如何去训练自己的气场，如何让自己保持真正的无条件自信。

就是这个世界上扰乱你的东西太多了，你要满足世俗的价值观，你要不断的满足别人对你的期望等等等等。你会精疲力尽，真的，你无法满足外界的所有。所以真正的自信一定是脱离了万事万物。

一定是脱离了所有的一切的东西，真正的自信，一定是要让你能够做到无条件的相信自己，就你什么都没有，你都可以依然很自信。这就是我们终极版课程真正要传达的东西。
