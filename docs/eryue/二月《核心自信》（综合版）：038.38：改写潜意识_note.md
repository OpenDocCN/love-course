# 二月《核心自信》（综合版）：038.38：改写潜意识_note

在本节课中，我们将要学习自我暗示的第三项核心规律：**替换**。我们将探讨潜意识的工作原理，以及如何通过系统性的步骤，用积极正面的信念替换掉限制我们的负面信念，从而改写我们的人生默认模式。

上一节我们介绍了自我暗示的重复与内模拟。本节中，我们来看看如何运用“替换”规律来直接影响我们的潜意识。

根据心理学研究，我们的潜意识在同一时间内只能主导一种感觉。通过将一个积极正面的思想反复灌输给潜意识，原有的负面思想就会逐渐衰弱、萎缩，新的积极思想则会占据上风。

暗示分为积极暗示与消极暗示。以下是几个生活中常见的负面暗示例子：

*   孩子高考前，父母叮嘱：“千万不要紧张。”
*   孩子成绩不好时，父母斥责：“你怎么这么笨。”
*   孩子挑食时，父母训斥：“你为什么不吃白菜？”
*   孩子尿床时，父母辩护：“他憋不住尿。”

这些言行都在无意中助长了不良倾向。重复的暗示会形成相应的心理定向，常常产生与愿望相反的效果。

科学实验证明，正面暗示能使人积极向上，负面暗示则会阻碍成功。在正面鼓励中成长的孩子，更可能拥有美好未来。反之，在“废物”、“笨蛋”等负面暗示中长大的孩子，容易被灌输自卑感，更易陷入心灵阴影。

说到这里，你可以暂停一下，思考如何将上述父母的负面暗示，转化为正面暗示。例如，将“你怎么这么笨”改为“这次没考好，我们一起来看看哪里可以改进”。

他人的暗示远没有你自己的想法重要。你怎么想，完全取决于你自己的选择。要想成功，就必须选择正面暗示，保持积极心态，让潜意识习惯朝向你期望的成功或积极方向运作。

长期坚持并不断重复，让自己处于快乐的身心状态，让头脑充满正面信念。这样行动，你更容易拥有较好的人生。

进行自我暗示时，有几点必须注意。首先，需要不断重复你的目标。

例如，在重复“每一天，每一方面，我正在变得越来越好”时，**千万不要尝试集中精神思考这句话**。你必须处于完全放松的状态。

思想的集中对意识行为很有价值，但对自我暗示而言却是致命的。为了避免单调和机械，必须保持心平气和、心理顺从的状态。如果你集中精神想着“一定要加强暗示”，这会将暗示转化为意识行为。

**公式：试图集中精神强化暗示 → 暗示转化为意识 → 在潜意识与暗示之间设立阻碍**

这会在潜意识和暗示之间设起一道阻碍，阻止后者注入潜意识。因此，当潜意识被成功呈现（即被新信念占据），你的命运就被改写了。

意识层面的东西，我们容易感知，会认为是“我的选择”。潜意识层面的东西，我们无法清晰感知，常会归咎为“命运”或被动结果。

当我们扩大觉知范围，觉察到潜意识的内容时，这些内容就会浮上意识层面。一旦我们将意识到的东西说出口，就会产生一种内驱力，有意识地将事情朝所说的方向推动。

这是因为自恋是人的一种本能。当一个意愿说出口，为了证明“我是对的”，你就会产生内驱力推动事情发展，并选择性收集符合预言的证据。这就是**自我实现预言**。

以下是实践自我实现预言、改写潜意识的具体步骤：

**第一步：写下清晰具体的目标**
展开一张白纸，写下你具体想要达成的目标或愿望。它必须基于现实条件，即有实现的可能性。
> **示例**：一位医生可以写下“成为领域内的知名专家”，但不能写“成为花样滑冰世界冠军”。

**第二步：视觉化你的愿望**
在你的头脑中，像放电影一样视觉化你的愿望已经实现的情景，仿佛它已是事实。这个想象需要注入足够的能量，否则只是空想。

**第三步：设计正向肯定句**
为自己设计简短、正向的肯定句，暗示目标已实现。注意，潜意识接受**现在进行时**。
> **代码/公式示例**：肯定句 = “我**是**苗条的” / “我**拥有**自信”， 而不是 “我**将会**变苗条” / “我**希望**有自信”。

**第四步：重复肯定句与画面**
将肯定句写在纸上，每天重复诵读。同时，在脑海中反复出现相关的成功画面，并用心感受身心的感觉。**这个练习需至少连续进行21天**。

**第五步：重温过去的成功体验**
回想过去你成功完成某个挑战的时刻。越细致地回忆当时的情形和感受，就越能激活成功的积极体验，促进当前目标的实现。**成功才是成功之母**。

**第六步：谈论愿望，如同已实现**
开始与朋友交谈或在社交平台分享时，以“愿望已实现”的口吻去谈论和憧憬。像孩子一样运用想象力，以假为真地去谈论。你的现实会跟随你想象的脚步。

**第七步：重复，重复，再重复**
新的习惯和信念通过重复而形成。当你全新的信念形成时，你的潜意识就被改写了。**形成一个全新的潜意识，至少需要21天**。

我们再次强调，这绝非讲授成功学或自我催眠，而是教授如何通过改变信念来科学地改写潜意识。所有步骤都服务于**信念的达成**与**潜意识的彻底置换**。若不能坚持，或目标完全不切实际，失败便无法避免。

一直以来，我们错误地认为行为主要受意识驱使。但从脑科学看，人的大部分行为其实受潜意识支配。

这解释了为何我们一天中大部分行为是自发且无意识的。当一个行为经反复练习，就会从意识进入潜意识，形成自发反应，例如吃饭、回家、开车。

如果每件事都需深思熟虑，大脑将能量耗竭。庞大的潜意识系统如同电脑的内存和缓冲区，将反复发生的行为存入其中，形成惯性思维，从而减少大脑消耗。

因此，意识如同冰山一角，海面下庞大的冰山才是我们的潜意识，它才是我们大脑无形的主宰。

《自控力》作者麦格尼格尔指出，人的改变通常经历四个阶段：
1.  **无意识的无能**（不知道自己做不好）
2.  **有意识的无能**（知道自己做不好）
3.  **有意识的有能**（有意识地努力做好）
4.  **无意识的有能**（不假思索就能做好）

经过不断重复，“有意识的有能”会进入潜意识，形成“无意识的有能”，即成为你的本能。

我们大部分行为受潜意识影响，因此许多错误会在生命中轮回。一旦我们意识到问题所在，潜意识就变成了意识。

此时，即使暂时无法改变，我们已经有意识地向自己植入一些积极特质，比如自信。当这些成功的想象和感觉在大脑中反复出现，就会重新进入我们的潜意识。

**这就是信念能改变潜意识的原因，而信念的达成，就在于不断的重复。**

总结一下，前面所讲的所有步骤和自我暗示规律，其核心目的都是为了**改变你的信念**——去除脑中负面的、限制性的旧信念，换上一个积极的新信念。

**核心原理：你只有不断重复一件你所相信的事情，它才会变成你的本能。当它成为本能，就意味着新信念已经形成，并下沉到你的潜意识，替换掉旧的负面信念。**

你为何不自信？为何见到心仪的异性想避开？这很可能源于过去的某些沉重经历，形成了一个负面信念（如“我不配”、“我会被拒绝”）。这个信念下沉到潜意识，成了你的人生默认模式。

所以，后来每次遇到类似情境，潜意识都会自动触发“远离”的反应。要改变，就必须先深挖出这些负面信念，然后建立新信念并不断重复，让其下沉，最终替换掉旧模式。

我们绝非教你催眠自己空喊“我会成功”。本节课的核心，是帮助你通过理解原理和践行步骤，达成新信念的建立，彻底改换掉旧有的人生默认模式。
