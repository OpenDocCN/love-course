# 二月《核心自信》（综合版）：035.35：潜意识的重要性_note

在本节课中，我们将要学习“潜意识”这一核心概念。我们将探讨意识与潜意识的区别，了解潜意识如何深刻影响我们的情绪、行为和自我认知，并学习识别两种常见的潜意识运作模式。

---

希腊阿波罗神庙上刻着三句真言，其中最著名的一句是“认识你自己”。这几个字包含的智慧至今依然深具意义。

曾有人向诗人泰戈尔提出三个问题：世界上什么最容易？什么最难？什么最伟大？泰戈尔回答：指责别人最容易，认识自己最难，爱最伟大。

因此，认识自己是一个特别重要的问题。人最难的便是认识自己，但最重要的也是认识自己。今天，我们就来聊聊关于我们自己的认知。

## 什么是自我认知？🤔

自我认知是对自己的洞察和理解，也称为自我意识。它是个体对自己存在的觉察，包括对自己的行为和心理状态的认知。

人的意识又分为**意识**和**潜意识**。

*   **意识**：指人们自觉认识到并有一定目的控制的意识现象和心理过程的总和。简单来说，意识就是你的注意力，你现在想达成什么、正在干什么、准备干什么。这就是你的意识。
*   **潜意识**：指人类心理活动中未被认识到或没有达到意识状态的部分。它是人们已经发生但未进入意识状态的心理活动过程，包括原始的冲动、各种本能以及过往经验、信念的总和等。

意识与潜意识好比一座冰山。露出水面的那一小部分是意识，而隐藏在水面下的绝大部分则是潜意识。

## 潜意识的主导力量 💪

我们正生活在一个意识变革的时代。过去几年，各领域专家在研究人类成功的内在机制时，有一个核心发现：影响人们活动最主要的因素并非意识层面，而是潜意识层面。

也就是说，我们的意识并非日常生活的核心。你所有的意识都被你的潜意识所控制着。我们大部分行为和选择，都是由我们根本意识不到的潜意识所掌控。而意识大部分时间所起的作用，只是为潜意识的所作所为寻找解释。

## 潜意识如何运作？⚙️

潜意识究竟是如何发挥作用的呢？这里有两种常见情况：自动化思维和跳跃性假设。它们会让我们不自觉地产生情绪和行动。

### 自动化思维

自动化思维是指无意识的、不带意图的、自然而然的，并且不需要努力的思维。

**例如：**
你现在读一本很有难度的书。半小时过去，你仍不能理解书中的内容。这时，你的头脑里可能会冒出一个声音：“这本书对我太难了，我永远也搞不懂，我太笨了。”随之，你产生挫败感，心情变差，于是把书丢到一边。

如果问及为何出现这样的情绪和行为，你可能会自然地回答：“因为书太难，看不懂。”但事实果真如此吗？

其实不是。真正的原因是你“觉得永远也搞不懂”这个**自动化思维**。如果现在告诉你，很多人第一次读这本书时都有同感，需要反复读好几遍才能慢慢理解，你的感受很可能会发生变化，不再认为自己太笨或永远搞不懂。

我们总认为情绪由外部世界直接引起，但外在场景并不能直接引发情绪和行为。我们对场景的**解读**才会直接引起情绪和行为。而这种解读或理解，很多并未经过思考或逻辑推理，而是第一时间自然出现在大脑中，这就是自动化思维。

如果自动化思维产生的解读，对于同一场景的不同解读会带来不同的情绪和行为，那么改变自动化思维，就可以改变后续的情绪和行为。这是情绪管理的一个重要方法。

### 跳跃性假设

如果说自动化思维还能让你从行为前的想法追溯到潜意识的痕迹，那么跳跃性假设则更为隐蔽，往往让你无迹可循。

跳跃性思维是指人们面对某些场景时，不加思索地做出某些行为的现象。

**例如：**
有些人在突然面对高层领导时，会不自觉地后退、避开或保持一定距离。

从生物学角度看，跳跃性假设这种无需对每件事思考就做出决策的行为，能节省精力，让生活更轻松。我们为何会不加思索就采取某些行动呢？原因可能包括：小时候看到父母或他人这样处理，我们模仿并内化为自己的方式；或者我们曾有相似经历，当时采取了这种行为，后来便延续下来。

跳跃性假设是跳开理性思考环节直接做出的假设。心理学著作《影响力》的作者提出过一个观点：人就像录像机，一按按钮就播放。一按按钮，我们就不加思索地做出某种行为，这背后正是潜意识的力量。

## 我们认知的世界 🗺️

我们对世界的认知都是一种假设，是大脑对事实进行解读后形成的。过程如下：

1.  **事实**发生。
2.  经过大脑**解读**。
3.  形成**假设**。
4.  产生**行为**。

这就是我们大脑的运行过程。我们生活的“真实世界”是我们“以为”的真实世界。实际上，每个人以为的真实世界都有些不同。同样，我们以为的自己，也不一定是真实的自己。

## 意识与潜意识的特性 🔄

首先，我们要明白人类自身存在两种截然不同的特性。

*   **意识（显意识）**：通过大脑思考的、自主的意识。我们可以用它来分析、判断和推理。我们平常充分思考生活，使用的就是显意识。
*   **潜意识**：不通过大脑思考的无意识。它是长久以来，我们依据显意识对事物判断的结果所积累形成的本能反应。

**例如：**
在结婚典礼上致辞，明知没人会认真倾听，你仍会紧张。想要忘记某人，思绪却总被其占据，无法专心工作。很清楚努力一下目标就可能实现，但就是提不起劲。这些现象都说明，你的潜意识远超你的意识控制范围。

## 两者的关系与总结 📝

意识层面是我们能觉知到的注意力、思考等方面。当我们对某件事通过不断思考、觉知，使其形成本能后，这项能力便会“下沉”到潜意识。

**以学开车为例：**
初学开车时，你需要战战兢兢，全神贯注于方向盘、挂挡等操作（**意识主导**）。学会之后，开车技能成为你的本能，下沉到**潜意识**。此时，你的潜意识接管了开车任务，你可以边开车边与人聊天，因为踩刹车、控方向已变成自动化的本能反应。

所以，潜意识是我们的信念以及过往经验的累积。可以将其理解为我们默认的行为模式。而意识，则是我们对某些事情进行思考、推理、分辨的过程。

意识与潜意识有着密切联系。但最终，**潜意识远比意识更强大**。
