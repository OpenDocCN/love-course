# 二月《核心自信》（综合版）：029.29：行为模式5（低价值行为模式）🚫

在本节课中，我们将要学习低价值行为模式的另外几种具体表现，特别是“过度努力获得接受”和“过于努力去表达价值”。理解并避免这些行为，对于建立真正的自信和有效的社交互动至关重要。

上一节我们介绍了低价值行为模式中的紧张和需求感。本节中，我们来看看第三种常见的低价值行为模式：**过度努力获得接受**。这种行为的核心在于，你试图通过刻意的言行来获取他人的认可，反而暴露了自己的不安全感。

以下是“过度努力获得接受”的几种具体表现：

**1. 对自己的笑话不够自信**
有些人讲完一个笑话后，自己会立刻哈哈大笑。他们想用自己的笑声来告诉外界，这个笑话很有趣。如果你经常这样做，请停止。你笑得太早，会缓解紧张氛围，可能导致最终没人发笑。如果等待10到20秒，通常可能会得到反应。如果你的笑话真的好笑，即使没人笑，只要你没有主动承认这一点，通常也没人会注意到你试图逗笑他们。讲笑话最好的方式是：讲完后，如果外界听懂了并哈哈大笑，你再跟着笑。在外界没有笑时你就笑，不是一个好习惯。因为你在努力尝试获得其他人的认可和接受。

**2. 努力获得口头确认**
很多人对自己的表达充满怀疑。他们在说完某件事后，可能会追问：“难道你不觉得这很好玩吗？”或“这非常有趣吧？”，而不是讲完故事后，让其他人自己做出反应。很多男生和女生互动时也是如此。例如，他可能会说“我需要一个女性的意见”，说完后没有继续跟进话题，而是补充解释“因为你们是女性，所以我认为你们是很好的询问对象”。这就是在努力获得口头确认。相反，正确的做法是：一旦你说完“我需要一个女生的意见”，就应该快速推进并保持对话内容的紧凑，直到有一些内容能够吸引到对方，再及时抛出你的问题，而不是进行过多解释。

**3. 重复使用无效的同类型幽默**
很多人尝试用讲笑话的方式让对方发笑。但外界很快会意识到，你不是为了让大家都开心，而是试图得到一个外界的反应，表现得就像在唱独角戏。一旦外界感觉到你正试图得到他们的反应，你的幽默就不起作用。就像很多社交技巧，你在用时没效果，别人用却非常有效。一个关键原因在于，你在使用技巧后，眼神充满了期待，仿佛在告诉对方“快笑”。对方能够看出你的刻意为之，结果自然无效。

接下来我们讲讲第四种低价值行为模式：**过于努力去表达价值**。当你试图刻意证明自己时，往往会适得其反。

以下是“过于努力去表达价值”的几种具体表现：

**1. 试图在逻辑上说服别人喜欢你**
当你尝试在逻辑上说服别人喜欢你时，你的行为会减少他们给你的情绪回应。在社交互动中，这可以说是最糟糕的事情。如果你能使用强大的社交能力，使其他人对你感兴趣，你就会创造一个好的氛围。然后，其他人会开始问你问题，以试图与你建立和谐关系。但是，如果你是试图用那些能力去打动或吸引其他人，那么你的能力也将失去功效。也就是说，你所有能力的展示都应该是自发的，而非想着以此来获得好的反馈。当你是自发的、没有其他想法时，你只是在做真实的自己，那么你就会收到更多更好的反馈。如果你是为了得到他们的反馈，他们能够看得出来，你依然在寻求他们的认同。

**2. 用太多句子解释不必要的问题**
如果你说得太多，可能会传达一种凌乱的思想。例如，当有人问你“你今天为什么穿成这样？”，如果你回答：“因为…因为天气…等等等等”，这就是过度解释。其实，你只需要看着他的眼睛说“有什么问题吗？”，或者可以直接忽略他的问题。我们前面提过，对于一些莫名其妙的问题，你可以选择不回答。

**3. 过于努力重新开始被切断的对话**
比如，你和对方说你的创业经历，忽然被外界打断了。后来，你就想方设法重新开讲你创业的话题。这其实是你希望能够给别人留下深刻印象的行为。如果你的话题真的很有趣、很吸引人，其他人后面通常会说：“哎，你刚才说到哪了？”，以此重启对话。如果你的话题被切断，要么等待其他人重启之前的对话，要么可以跳跃到另外一个主题，然后再自然地跳回之前的话题。但总体来说，最好是丢掉重回之前那个话题的想法，然后开启其他话题。因为你的需求感会被外界看出来，仿佛你没有其他话题可选择。

**4. 过度补偿不安全感**
在社交压力下，人们往往会立即想到自己的不安全感。社交压力会把他们的不安全感推向表面。例如，一个较矮的男孩子碰到一个高个子的女孩子，他立即说：“哎，你的鞋让你今天晚上看起来特别高，你的鞋应该有10厘米吧。”这其实就是在补偿自己的不安全感。同样，如果你有一份自认为低身份的工作，你可能会说：“现在的人们是如此现实，他们迷恋金钱。其实重要的是对自己的生活感到满意。”然而，其他人并没有问你的生活哲学，他们只是好奇你的职业。而你，在不断地补偿自己的不安全感。记住，你应该由内而外地去接纳自己的一切。

**5. 过度补偿失败**
当一个人过度在意外界如何看待他时，他们会常常想要用口头语言弥补差距。他们会说：“啊，今天我状态很不好，我累了”，或者“我不是得不到他，只是我不想，我只要想就可以有”。你这样做的唯一理由，就是你很在意别人如何想你。你是活在其他人的世界里。

**6. 用冗长的借口应对批评**
很多人难以接受批评，会拼命为自己解释，这将暴露他的不安全感。如果有人批评你，一个简单的暂停、点点头，或者说“你说的有道理，以前我怎么没想到”，都可以。你不应该垒起傲慢的墙。这个世界上没有人不犯错。在某件事情上做得不好，我能接受那个结果，以及为自己所犯的错买单。很多人之所以不敢面对自己的错误，就是他把那件事以及那个错完全地和自己划上等号。如果他承认了，那么他的自我价值感会迅速下降。但是，你应该把自己的生命和那件事情区分开来。你还是你，哪怕你接受了那个错，你生而为人的价值依然没有减少。就是不要总是为自己辩解，错了就是错了，应该自信地面对一切。当然，如果你觉得不是你的错，可以把事实陈述出来。但如果已经是明文规定，或者你都知道的东西，那么你如果做得还是不对，这就是你的错。你应该能够面对自己，能够接受结果。

以上讲的这些，都只是社交中需要注意事项的一部分。更多的需要你们在社交中自己去发现，然后去精进。我记得有一位名人曾说过：
> 小心你的思维，你的思维将会决定你的语言；小心你的语言，你的语言将会决定你的行为；小心你的行为，你的行为将会决定你的习惯；小心你的习惯，你的习惯将会决定你的性格；小心你的性格，你的性格将会决定你的命运。

其他的我们没有必要深挖，但是你一定要明白：**思维和行为是共生关系，改变一端，另外一端就会发生改变**。如果你的思维短时间内难以完全改变，那么就从你的行为方式开始入手。同时，注意自己的那些低价值的行为模式，比如不断的解释、用借口给自己辩解、不断补偿不安全感等等。注意那些自己的低价值行为模式。因为当你有这些低价值行为时，它会反过来影响你自己的思维。也就是说，当你有这些低价值行为的时候，你的思维会开始变弱。你的思维开始变弱，它会进一步降低你的行为方式。你所有的行为方式会让你变成一个从属或跟随者的位置。因为你总是急于获得其他人的认可，总是害怕其他人怎么想你，总是担心自己没有留下好印象。然后，外界就会发现你的需求，发现你的刻意，发现你需要他们的认同。你在整个社交动能中会不断降低，对方会不断提升。所以你一定要小心你的低价值行为模式，它会不断影响你的思维模式，你的思维模式再反过来降低你的行为方式。到最后你会发现，自己不知不觉变成了一个“傻缺”或“舔狗”。事后你可能会回想：“我当时怎么会这么做？”因为你的思维模式和行为模式在不知不觉地朝更低的方向走了。

而不像有些人，他们发现自己出现低价值行为模式，会立刻制止。立刻制止之后，他的思维就会接收到这些信息，他的思维开始会变得强势。他的思维变得强势之后，转而影响他的行为模式，他的行为模式将会变得更有力、更自信。当他的行为模式变得更有力、更自信的时候，他就会接收到好的反馈。因为你更自信，外界一定会对你有更好的反馈。这个好的反馈会进而过来影响并强化他的思维模式。所有模式再进而强化他的行为模式，这就是**正向循环**。

所以，一定要小心你的低价值行为模式，一定要注意你的高价值行为模式。不断堆砌你的高价值行为模式，减少你的低价值模式。你的思维就会发生改变。思维发生改变，你就会越来越好，你的状态也会越来越好。所以，有时候你表现不好，是因为你正在掉入负面循环。有时候你状态非常好，是因为你正在经历正面循环。这两个方向是大家特别要注意的地方，而且要勤加练习。你不可能听完这节课就立马会，它是存在于你在社交当中不断去强化的结果。它不可能让你一个晚上就改变自己几十年以来的行为模式和思维方式。它是在一天一天当中不断的累积、不断的坚持，直到把你变成真正的高价值的人为止。

本节课中，我们一起学习了“过度努力获得接受”和“过于努力去表达价值”这两大类低价值行为模式的具体表现与危害。关键在于理解思维与行为的共生关系，并有意识地从行为入手，减少低价值模式，增加高价值模式，从而启动个人成长的正向循环。如果你没理解，这节课可以多看两遍。如果理解了，我们下一节课见。
