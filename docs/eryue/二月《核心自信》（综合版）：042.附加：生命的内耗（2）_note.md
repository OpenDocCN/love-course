# 二月《核心自信》（综合版）：042.附加：生命的内耗（2）_note

在本节课中，我们将要学习如何进一步摆脱生命的内耗。上一节我们探讨了内耗的基本概念，本节中我们来看看另外两个关键点：停止活在他人眼里与停止苛求完美。

## 停止活在他人眼里 👀

叔本华曾说，人性有一个很特别的弱点，就是过度在意别人如何看待自己。很多人之所以不够快乐，就是因为太在乎周围人的反应。

无意间的一个眼神，可能会让人思虑很多，心情失落许久。朋友不经意的一句话，却会让人纠结半天。太过在意别人的看法和评价，后果往往是在敏感和讨好中委屈了自己。

以下是为何我们如此在意外界看法的两个主要原因。

### 1. 社会性动物的基因设定

我们都是群居性动物。只要是社会性动物，就无法避免需要和外界保持联盟关系。

例如狼群、蚁群、狮群、猴群都是社会性动物，它们都需要和部落保持紧密联系，才能更好地生存。而独居性动物，如老虎、豹子，则没有这样的基因设定。因为长期独自生活，它们不是很在乎联盟的重要性。

人类也是社会性动物。我们的祖先在部落时代，如果被部落驱逐或不受欢迎，生存和繁衍都会面临极大考验。虽然现代社会，个体脱离群体依然可以生活，但不可否认，拥有广泛人脉资源、获得周围人的支持，会让我们活得更好。因此，对于联盟的需要，从部落时代一直被遗传至今。

更重要的是，在满足生存和繁衍后，我们也在意归属感这种情绪体验。当我们被大众接纳时，归属感会增强；如果一直被周围人排斥，归属感就会降低。这是我们在意他人看法的第一个原因。

一定的在意是正常的。但内耗强的人则是**过度在意**。

### 2. 因自我认知不清导致内心缺乏安全感

这里的核心原因在于第二点：因为不知道自己是谁，从而导致内心缺乏安全感。

**不知道自己是谁**，意味着：
*   不知道自己真正想要什么。
*   不知道什么应该作为自己的人生准则。
*   不清楚自己的三观（世界观、人生观、价值观）。
*   对于自己所相信的不够确定。

因此，他们只能不断以他人的标准来指导自己的生活。当你对自己的认识不够时，你就会以外界的标准为标准。自然而然，你会过于在意别人的看法。

**自己是否喜欢不重要，他们喜欢才最重要。自己是否难过开心不重要，他们开心才最重要。**

如果你是这样，结果可想而知。因此，你要自己决定自己是谁。替你决定前进方向的，应该是你自己；为你决定人生取舍的，也应该是你自己。

当你越明白自己是谁时，外界对你的影响也会越来越小。因为你清楚地知道自己**是什么**以及**不是什么**。所以，你应该经由自己来决定自己，而不是外界。

对于这一点，你需要好好思考“你是谁”。你越清楚自己是谁，你就越自信；你越自信，也就越了解自己。当你能思考明白这个问题时，外界对你的影响将会越来越小，甚至消失。

## 停止苛求完美 ⚖️

摆脱内耗的第三点是停止苛求完美。在我们身边，完美主义者往往有两类人：
*   一种是对自己高标准、严要求，什么事情都要做到极致。
*   另一种是对别人要求很多，严于律人，不允许周围的人犯一点错。

不管你是哪一种，你的内耗一定会让精神状态处于疲软状态。因为这个世界上不可能什么事情都如你所愿。常言道“不如意事十之八九”，这是人生的常态。

释迦牟尼曾说“世事无常”。《道德经》有云：“大成若缺，其用不弊。”这句话的意思是，圆满的事物好像都有所欠缺，但是它的作用不会衰竭。

当然，这并非是说从此以后对所有事情都无所谓、随波逐流。而是说，你应该能够认识到“金无足赤，人无完人”。

有完美的人吗？**没有**。

与其在苛求完美的路上让自己精疲力尽，不如放松下来，给生活留一点缺口。对自己适当提出高标准，是上进的表现，这很好。但如果你要求完美，说到底，是你对于自己的不够接纳。

这个世界上至今为止，有出现过完美的人吗？**没有**。至今为止，我们能画出一个完美的圆吗？**没有**。无论多么精密的圆规，都无法画出完美的圆。所以，有任何一个人的人生是没有犯过错的吗？**也没有**。

因此，苛求完美是一场没有尽头的自我搏斗。我们应该认识到作为人类的局限性，不要陷入完美的陷阱里。从另一个角度来看，带有缺憾的人生才是完美的，不是吗？

你之所以追求完美，是因为你将所有的一切都定义为不完美。但“不完美”往往是经由我们个人的主观意识所定义出来的。

垃圾场真的不完美吗？其实垃圾场是完美的，它只是静静地待在那里，经由时间不断降解，最后尘归尘，土归土。下雨天完美吗？大冬天的寒冷完美吗？夏天40度的炎热完美吗？其实，它们都是完美的。只是经由我们的定义，它们才“不完美”。

所以，从更宏观的层面来说，**所有的一切都是完美的**。我们之所以苛求完美，是因为我们对于自己的定义是“不完美”。

对于这几句话，大家能够理解就更好，不是很理解也没有关系。总之记住，想要脱离内耗，就要去掉你的苛求完美。

---

本节课中，我们一起学习了摆脱内耗的另外两个关键：**停止活在他人眼里**与**停止苛求完美**。核心在于建立清晰的自我认知，由内而外地定义自己的价值与标准，并接纳世界与自身的不完美。当你越了解并坚定自己是谁时，外界的纷扰和内心的苛责对你的消耗就会越少。
