# 猫叔自信恋爱课堂 - P4：不敢在异性面前大声说话的不利影响 😰

在本节课中，我们将深入探讨不敢在异性面前大声说话这一行为模式会带来的具体不利影响。我们将分析这些影响如何侵蚀你的生活信心、阻碍个人成长，并揭示其背后的核心心理机制——**懦弱**与**自尊体系**的缺失。通过学习，你将理解为何建立正确的自尊和培养勇气是改变现状的关键。

## 丧失生活信心与得过且过

上一节我们介绍了行为模式的表现，本节中我们来看看它造成的首要恶果。你会丧失对于生活的信心，开始得过且过。这是浪费时间与生命最主要的问题。

## 陷入不满与抱怨的循环

以下是第一个不利影响的具体表现。

*   你只能满腹抱怨地过着自己不满意的生活。
*   你羡慕那些拥有伴侣或财富的人，自己又想要，但对现状不满意却不进取。
*   在不满意与不进取的纠缠中，你会憎恨自己。
*   你会感觉生活没有希望，人生一眼就能望到头。

## 沉溺于负面环境与情绪

这种状态持续下去，会导致你特别容易每天沉浸在负面环境和情绪中。

例如，如果你到了30岁仍一事无成，在大城市生活，将面临以下困境：
*   每天早上被迫早起挤地铁。
*   因一事无成和挤地铁的痛苦，陷入自我否定，觉得自己没用、没有希望。
*   容易因拥挤等小事与人发生口角甚至冲突，终日闷闷不乐，却没有能力改变现实。

据观察，很多处于此状态的人，未来很可能变成这样：三十岁时如此，四十岁时则可能成为被呼来喝去的“老师傅”，整天无精打采，愁眉苦脸。

## 核心症结：缺乏勇气与行动力

关键点在于缺乏**勇气**。具体来说，是缺乏**培养主动忍受困难感的行动力**。你需要勇敢一点，更能扛事。

困难感常在需要面对改变时出现。无论是主动改变还是被迫改变。

例如：
*   **主动改变**：决定减肥时，会觉得不吃饭、运动很困难。
*   **被迫改变**：从小城市到大城市打拼的头一两年，会感到特别困难，包括想家、不适应、交不到朋友等。

只要想改变，就必须忍受这种困难感或不适感。它会导致两种反应：
1.  **情绪化**：如感到迷茫、想家、烦躁，这容易让人放弃。
2.  **调用意志**：决心克服困难，坚持到底，这会让人**寻找解决之道**。

那些看起来无精打采的人，往往是遇到问题易于放弃、不能勇敢面对并积极解决的人，因此无法收获信心。

## 懦弱的本质：习惯失败

有人问懦弱具体是什么。最准确的定义是：**因为你习惯了你的无能和失败**。

当失败来临时（如被拒绝、被责骂），你内心已不会感到痛楚，这说明你已非常懦弱。你永远不相信自己能够成功一回、赢一回、搞定一回。

你对自己没有信心、做事焦虑、无法开朗、愁眉苦脸的根本原因，就在于你习惯了在各种事情上的失败，很难成功，很难对自己有信心。你从未建立起自己的**自信心体系**或**激励体系**。这是很多人人生悲剧的根源。

## 建立正确的自尊体系

首先，为何你会习惯失败与无能？有人归因于“天生脸皮薄”，但这只是胆小或懦弱者为自己找的借口。例如，不敢追女孩，怕被拒绝，这本身就是一种失败。如果你连失败都不愿承认，说明你对自己的要求已低到谷底。

一个有上进心的人会对失败有强烈感觉。如果你对一事无成、一无所有的状态毫无感觉，说明你已“没脸没皮”。

一个人变得懦弱，首先是因为他没有良好的维护自尊的方式。错误的方式（如逃避）只会导致更没自尊。

例如，比较两种情境：
*   **代码**：`去搭讪并被拒绝` vs. `完全不敢去搭讪`
*   哪个更伤自尊？虽然都可能没结果，但**敢去做**本身，在自我评价体系里就比**不敢做**更强。这是客观事实。

因此，你要懂得用正确的方式去**建立和维护自尊**，从而获取尊严，变得自信。拥有正确的自尊体系，你才会自信；错误的自尊体系会导致自负或自卑。

自信通过自尊体现。**自尊很大程度上源于自我要求**：
*   公式：**高自我要求 → 高自尊 → 自信**
*   公式：**低自我要求 → 低自尊 → 自卑**

生活中，那些对自己要求严格、力求做好每件事的人，往往更自信，生活更好，也更受欢迎，因为他们的“逼格”（自我要求）比较高。

## 解决问题的两步走

要解决当前问题，首先要解决勇气问题。

以下是两个核心步骤：
1.  **懂得什么是自尊与自信**：建立正确的自尊体系，通过高自我要求和解决问题来获得自信。
2.  **懂得什么是勇气与责任**：培养主动忍受困难、面对改变的行动力，勇于承担责任。

本节课中我们一起学习了不敢在异性面前大声说话所带来的系列不利影响：它会让你丧失信心、抱怨生活、沉溺负面环境。其根源在于**懦弱**——即习惯了失败，以及**错误的自尊体系**。改变之道在于建立正确的自尊（通过高自我要求）和培养真正的勇气（主动忍受困难感）。这是你走出困境、收获自信人生的唯一途径。
