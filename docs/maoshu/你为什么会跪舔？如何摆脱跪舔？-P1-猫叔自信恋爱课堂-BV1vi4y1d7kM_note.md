# 自信恋爱课 01：识别并摆脱“跪舔”心态 🧠

在本节课中，我们将学习一个在人际交往中，尤其是与异性交往时常见的问题——“跪舔”心态。我们将深入探讨其表现、背后的心理根源，并理解与之相关的其他行为模式。掌握这些知识，是建立健康、平等交往关系的第一步。

## 识别“跪舔”的常见表现

上一段我们引入了“跪舔”这个概念，本节中我们来看看它在实际交往中有哪些具体表现。以下是几种典型的“跪舔”行为模式：

*   **过度主动与迎合**：对方明显不理会你，你却不停地找对方聊天。对方只回复一句话，你却回复四五句，并想尽办法延续对话。
*   **单方面付出**：对方过生日并不想邀请你，你却主动前去送礼。在对方不想搭理你的情况下，你仍主动贴近。
*   **自我价值感降低**：喜欢一个人时，总容易把自己变得很卑微，俗称“热脸贴冷屁股”。
*   **面对“女神”时发挥失常**：在接触心仪对象（或称“女神”）时，总是容易紧张，表现得不正常。
*   **陷入失败怪圈**：陷入一种“只要遇到自己喜欢的女生就肯定搞不定”的循环。
*   **感到自卑与懦弱**：总是感觉自己特别自卑，并且伴随懦弱，不知该如何行动。

如果你有以上多数感受，那么你可能正处于“跪舔”心态的影响下。

## 探究核心原因：自尊发展水平

那么，是什么导致了这些行为呢？一个普遍的说法是“自卑”。但自卑就一定会“跪舔”吗？并非如此。

**核心原因在于“自尊发展水平太低”**。这里的“自尊”指的是一种稳定、健康的自我价值感。当一个人的自尊发展水平低时，他无法平稳地看待自己与他人，通常会走向两个极端：“过卑”或“过亢”。

*   **“过卑”的表现就是“跪舔”**：通过贬低自己、过度付出来换取关注和联系。
*   **“过亢”的表现则是“攻击/贬低”**：通过打击、贬低别人来抬高自己，获取心理平衡。例如，在女生的社交动态下刻意挑刺、嘲讽，说“你这照片P过头了”或“喜欢这个太肤浅”。这种行为背后的心理是：**他既感到自卑，又想维持自尊心，于是通过打击对方来拉平想象中的关系差距**。

这两种看似相反的行为，其根源都是**自尊发展水平太低**。它们都是内心空虚、无法正确评估自我价值的表现。除了“跪舔”和“攻击”，**算计、沉闷无趣、社交退缩**等也是低自尊的常见表现。

## 低自尊的其他表现：以“伪自信”为例

除了明显的“跪舔”，低自尊还可能以一种伪装的形式出现，即“伪自信”或“自以为是”。这在某些网络言论中尤为常见。

以下是这类言行的特征：

*   **习惯性质问与反问**：不直接表达观点，而是以“难道不是吗？”“你觉得这样对吗？”等句式开头，营造居高临下的压迫感。
*   **嘲讽与贬低**：言语中充满莫名的优越感，对他人进行冷嘲热讽。
*   **姿态过高**：将自己摆在“导师”或“真理掌握者”的位置，缺乏基本的平等尊重。

这种沟通方式非常令人反感，其本质如同刺猬或河豚：**内心越虚弱，越要膨胀起满身的刺来伪装强大**。这同样是自尊发展不足，需要通过虚张声势来填补内心空虚的表现。健康的自信不需要通过贬低他人来建立。

## 总结与核心要点

本节课中，我们一起学习了“跪舔”心态的识别与根源。

1.  **“跪舔”的表现**是过度主动、卑微付出、在喜欢的人面前失常。
2.  **根本原因**是**自尊发展水平太低**，导致无法建立平等的交往心态。
3.  **低自尊的两极表现**：一是向内攻击的“过卑”（跪舔），二是向外攻击的“过亢”（贬低他人）。
4.  **“伪自信”** 也是低自尊的伪装形式，其特征是自以为是、习惯性质问和嘲讽。

理解这些，是改变的第一步。关键在于**提升自尊发展水平，学会像“正常人”一样不卑不亢地与人交往**。不必因自卑而过度讨好，也无需因脆弱而虚张声势。在接下来的课程中，我们将探讨如何具体提升自尊，建立真正的自信。
