# 猫叔的编程课 第1课：变量与赋值 🐱

在本节课中，我们将要学习编程中最基础也最重要的两个概念：**变量**和**赋值**。理解它们是如何存储和操作数据的，是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个**带标签的盒子**。这个盒子可以用来存放数据，比如数字、文字或更复杂的信息。盒子上的标签就是**变量名**，我们通过这个名字来找到并使用盒子里的数据。

在编程中，变量允许我们临时存储信息，以便在程序后续的步骤中调用或修改它。

## 如何给变量赋值？ ➡️

仅仅有一个空盒子（变量）还不够，我们需要把数据放进去。这个过程就叫做**赋值**。赋值操作使用等号 `=`，但它不是数学中的“等于”，而是表示“将右边的值存入左边的变量”。

其核心公式是：
```
变量名 = 值
```

例如，在Python中：
```python
age = 10
name = "小猫"
```
这段代码创建了两个“盒子”：一个名叫 `age` 的盒子里存放了数字 `10`；另一个名叫 `name` 的盒子里存放了文字 `“小猫”`。

## 变量的命名规则 📝

为了确保程序清晰且无错误，给变量起名需要遵循一些基本规则。以下是主要的命名规范：

*   **可以包含**：字母（A-Z, a-z）、数字（0-9）和下划线（_）。
*   **不能以数字开头**：`1variable` 是无效的，但 `variable1` 是有效的。
*   **区分大小写**：`Age`、`age` 和 `AGE` 是三个不同的变量。
*   **避免使用保留字**：不要使用编程语言已经占用的关键词，如 `if`、`for`、`print` 等作为变量名。
*   **建议使用描述性名称**：使用 `user_age` 比 `ua` 更能清晰地表达变量的用途。

## 使用变量的值 🔄

一旦变量被赋值，我们就可以在代码中使用它。使用时，计算机会自动找到对应“盒子”里的值来参与运算。

请看下面的例子：
```python
x = 5
y = 3
sum = x + y
print(sum)
```
在这段代码中，我们首先让变量 `x` 的值为5，`y` 的值为3。然后，我们计算 `x + y`，计算机会取出 `x` 盒子里的 `5` 和 `y` 盒子里的 `3` 进行相加，得到结果 `8`，最后将这个结果存入名为 `sum` 的新变量中，并通过 `print` 语句显示出来。

## 变量的值可以改变 🔁

变量的“变”体现在其存储的值是可以被更新或改变的。我们可以给一个已经存在的变量重新赋予一个新的值。

例如：
```python
counter = 1
print(counter) # 输出：1
counter = 2
print(counter) # 输出：2
```
变量 `counter` 最初的值是 `1`，后来被重新赋值为 `2`。最后一次打印时，显示的就是最新的值 `2`。
