在本节课中，我们将探讨如何在人际交往中建立和维护个人尊严。尊严并非来自外界的施舍，而是源于我们自身的行为与态度。我们将分析哪些行为会损害尊严，并学习如何通过具体的行动来提升自我价值感，从而赢得他人的尊重与认可。

随着年龄增长，周围人对你的要求会变高。一个人维护自尊的方式有很多种。

有的人通过发脾气或抱怨来维护自尊，有的人则通过把事情做好来维护自尊。你认为哪种方式是强者的方式？

哪种方式是弱者的方式？拼命维护自尊、拼命说自己厉害、同时推卸责任的人，他真的有自尊吗？他所谓的维护自尊只会让人更加看不起他，效果适得其反。如果一个人勇于承担，做错事勇于承认。

然后努力把事情做好，别人就会尊重他。所以，这些不懂付出的人，核心原因是什么？是自尊意识差，即不懂得认真考虑一件事该如何做好。所有的付出，你为别人做事情。

为什么反而招致反感？就是因为你自尊意识差。只有自尊意识差的人才会去“跪舔”，才会做出让人反感的事，才不懂得尊重别人，不懂得把事情做好。他不知道如何考虑。

因为他从未想过要把一件事做好。例如，第一个问题：他的一切行为只从自身欲望出发。从欲望出发也可以。

这没有错。如果你真想和一个女孩在一起，真想获得她的认可，你应该怎么做？

你要把所有与她相关的事情做到最好，让她对你最满意，她才会愿意和你在一起。试想，你带她出去，天天吃的是山珍海味，玩的是各种她从未体验过的有趣项目。

她从未体验过那种开心，她会不愿意和你在一起吗？她得到的都是很好的体验，自然会愿意。如果你带她出去，什么都不安排，什么都不做好。

还总想着占她便宜，她和你在一起的体验极差，她还会再和你在一起吗？这就叫自尊意识差。

你越懂得自尊，就越会顾及别人的面子。有的人维护自尊的方式是，即使再穷，身上也永远干干净净，头发梳得整齐，衣着整洁，鞋子一尘不染。这是我们说一个人展现尊严的方式。我再穷。

这才是表现出男子汉气概的状态。不像有些人，整天想着怎么占便宜，怎么发生关系，用什么技巧算计别人。你们这些人为什么不被女孩子喜欢？

你就越容易有“吸渣体质”，也就是说，来找你的都是“垃圾”，你遇到的也都是“垃圾”。为什么你们会遇到“垃圾”？就是因为你们不懂得维护自尊的正确方式，不懂得获取属于自己的尊严，你们是没有尊严的人。我问你们。

没有尊严就等于没有一切。你不会有朋友，不会有事业，也不会有爱人，什么都没有。学再多技巧都没用。那么，怎么办？如何让自己有尊严？

以下是提升尊严的五个具体方法。

**第一个方法：提高对自己的要求。** 做人做事以“令人满意”为出发点去思考，这就对了。我做人做事，如何让别人满意？首先不考虑自己，而是考虑如何让别人满意。别人对你的为人处事感到满意，就会信任你、认可你、尊重你、喜欢你。你对自己要求越高，别人就越喜欢你。这是一个很简单的道理。

我们说如何让自己有尊严？就是说，你做人做事的出发点，首先要想“我怎么让人满意”，而不是“我怎么能占人便宜”。一天到晚总想着占便宜的人，你觉得别人感受不到吗？你身边的人感受不到吗？

为什么别人不愿意和你来往？就因为你一天到晚老想着占便宜。这种人会有朋友吗？别人会相信他、信任他吗？就算暂时把你当朋友，时间长了也不会继续了吧？说实话。

你们这些人是活得没尊严的。像那些沉迷于技巧的人，我劝你们好好想想如何让自己活得有尊严。你们这样下去，说实话，人生会败得非常彻底，变得连蝼蚁都不如。

**第二个方法：目标越坚定，就越要注意言行举止。** 什么意思？例如，你决定追求一个女生，下定决心一定要追到她。你做每一件事都会非常注意：这件事有没有什么不好之处或风险？我怎么做可能最好？

就是说，当你下定决心一定要得到某个结果时，你就会尽力做好每件事，就一定会对自己提高要求。

**第三个方法：管理好自己的欲望。** 例如，你可以有欲望，但不要有“需求感”。你的欲望一定要够强烈。如果对所有事情都抱有“我一定要做到”的决心，那么他做所有事都会非常严谨、认真、用心。如果你的欲望不强烈，很多事情你不会用心去做，也不会认真思考，不会把时间精力花在上面。

所以你就会懒散粗糙。你们很多学“泡学”的人，为什么你们是一群“垃圾”，遇到的也是“垃圾”，生活在“垃圾堆”里？原因就在于你们总是用“结果化思维”。什么是结果化思维？

别人帮你得到这个结果，你都去依赖技巧，依赖所谓的“导师”，依赖这些东西。但问题是，那些东西不稳定，不能保证你一定得到，不能让你完全获取那个结果。

而且，就算依赖这些东西，它也不一定永远有效。永远有效的是什么？是你自己的头脑和意愿，你自己的主观能动性。这才是真正有用的。这也是一个人建立自信的唯一途径。你不愿意吃苦。

不愿意脚踏实地，不愿意用心，不愿意下功夫，一个人很难建立起属于自己的自信，也不会有什么能力。
