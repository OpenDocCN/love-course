# 自信恋爱课堂 P3：不良行为背后的三大原因？ 🔍

在本节课中，我们将深入探讨人际交往中常见不良行为背后的根本原因。理解这些原因，是改变行为模式、提升交往能力的第一步。

上一节我们介绍了交往中的错误行为，本节中我们来看看这些行为背后究竟隐藏着哪些核心问题。

## 原因一：长期被动，缺乏主动性 🚶‍♂️

第一个原因是常年不与人交往，尤其不与人主动交往。例如，每次和女生外出都让对方带领和安排，这属于常年不与人主动交往。不主动安排事务，意味着缺乏规划能力。

长期不主动安排活动、不管理事务，会导致你不知道该说什么、该做什么，也不知道该带女生去哪里或进行什么活动。首先，这会让你变得缺乏主见。

以下是长期被动交往的具体表现：
*   从不主动发起活动。
*   从不主动邀请他人。
*   总是跟随他人的安排行动。

## 原因二：缺乏自信，自我怀疑 😔

第二个原因是缺乏自信。例如，在与女生交往时不敢点菜，一味尊重对方意见，本质是害怕把事情搞砸，根源在于缺乏自信。

缺乏自信会导致自我怀疑和自我矮化。他会担心自己点的菜女生不喜欢，不知道点什么，又怕花了钱却得不到好的反馈。这种自我怀疑和矮化，会在女生面前表现为犹豫不决和纠结。

**核心表现**：连点菜都拿不定主意。女性通常很反感男性表现出这种状态。

第一种是长期不主动交往，导致从开口说的第一句话到最后一句，都让人感到不适，即“不会说话”。

## 原因三：目的性弱，缺乏规划与耐心 🎯

第三种原因叫目的性弱。请注意，目的性弱与目的性强并非同一概念，明天我们会详细讲解目的性强的问题。

目的性弱，指的是缺乏解决问题的能力和耐心。很多人误以为急躁就是目的性强，实则不然。

例如，什么是目的性弱？就是做事情不清楚自己到底要达成什么目标。比如和女生交往，一上来就想开房或确定关系，你们认为这是目的性强，对吗？这实际上恰恰是没有目的，你只想得到那个最终结果。

因此，你会倾向于直来直去地做这些事情。比如公开表白，你思考过表白的目的是什么吗？你或许会说目的是希望对方成为你的女朋友。

那么，如果你目的性真的很强，你会在没有把握的情况下就去表白吗？如果你目的性非常强，认定“表白一定要让她成为我女朋友”，你就不会去做没把握的事。你的行为实质是急躁。

所以我说这是目的性弱，因为心里不清楚做这件事究竟出于什么目的、要得到什么，而是将自己置于巨大的风险和不确定性中，最终很可能得到恶性结果。

没有把握的事，为何要去赌？要承受巨大风险的事，为何要去赌？我并非说这件事绝对不能做。有的表白是可以的，但前提是，你经过评估认为表白后大概率能得到自己想要的结果。

然后你再仔细考虑表白的方式：是当众表白，还是通过其他更合适的方式？你需要找到一个能打动对方、让对方感到舒服的方式。例如，当众表白也并非绝对不可行。

当众表白时，你可以这样说：“我今天喝了点酒，鼓起了勇气。不管结果如何，我憋在心里很久的话一定要说出来。首先，我喜欢你，喜欢了很长时间。”

“从什么时候开始呢？从我第一眼看见你的时候就开始了。但我一直不敢表白，不敢告诉你。今天说出来，并不是非要你做我女朋友，我只是想给自己一个交代。因为我觉得再不说，你可能就跟别人走了，或者我就没机会了。”

“因为我要离开这个城市了，可能很久都不会回来。所以今天我借着酒劲，当着大家的面告诉你。无论如何，我希望你永远幸福。”你们觉得这样的表白，真的会让女生讨厌或反感吗？

你还可以告诉女生：“我喜欢你不是一两年，是很多年了。你或许是我这辈子都无法忘记的人。”如果你曾为她做过什么，也可以告诉她。

“你还记得每次下雨时，教室最后一排总有一把伞吗？还记得你上学迟到时，是谁帮你点到吗？知道你抽屉里有时会多出一两块糖是谁放的吗？今天破案了，都是我做的。”

你可以详细告诉她为什么做那些事。让所有人都能听见：“所有这些点点滴滴，我永远不会忘记。我不需要你记住我，但我要告诉你，我这辈子都不会忘记你，走到哪里都会想着你。”

这样去表达，第一，情感得到了抒发；第二，有了合理的缘由；第三，你并非在要求她做你女朋友，只是告诉她，曾经有一个人一直默默爱着她，怕她不知道。

如果这些付出和文字都是你本人所为，那么请问，这样的表白有错吗？这样的表白是冲锋号吗？是胜利的号角吗？现实生活中，这样的表白难道不能打动人心吗？你们觉得这样的当众表白有错吗？

所以，**表白本身不是问题，关键在于你能把这件事做得多好、构思得多巧妙。** 为什么你们想不到？因为你们不用心，没有认真思考如何把事情做好。

你们只是直来直去、粗糙地处理事情，怎么可能做好呢？把事情做得太粗糙了。网络上流行的某些所谓“价值观”，正是因为有些人把事情做得太粗糙，不用心思考。

你认真想想，难道这些话很难组织吗？你就是没有投入足够的精力。所有不用心追求女生的行为，都叫**缺乏诚意**。

诚意通常体现在：我对这件事用了心，仔细思考过，并确认能够把它做好。什么叫用心？用心就体现在你为女生点菜的过程中。

你提前了解了这家餐厅、女生的口味、餐厅的大致价格、环境和氛围。你和她出去吃饭，把这些都考虑到了，你觉得这事还会出错吗？包括女生喜欢吃什么菜，餐厅有没有招牌菜。

再比如，约会过程中：吃完饭去哪里？下午什么时候、去哪里兜风？哪里的风景好？什么时间送她回去？回去时是否应该拍照留念？如果她自己打车回去，是否应该记下车牌号并留好紧急联系方式？所有细节都要考虑周全。

你考虑得越周全，女生对你的满意度就越高。所以这归根结底是你用不用心的问题。你们很多人只想：“哦，我就跟女生约个会，花点钱，然后女生就得做我女朋友。”想得太简单了。

女生判断是否接受一个男生做男朋友，完全基于这一点，而非其他。说实话，我以前讲过，很多人追不到女生，不是方式方法的问题，是因为你们做事太粗糙了。做事做不好，想问题太简单、太粗糙，做事情太粗暴。就这么简单。

概括来说：如果你追的女生本来对你有意思，你做事粗糙点或许没关系。但关键在于，很多时候你主动追求的女生并不喜欢你，这时你的言行举止就显得尤为重要。
