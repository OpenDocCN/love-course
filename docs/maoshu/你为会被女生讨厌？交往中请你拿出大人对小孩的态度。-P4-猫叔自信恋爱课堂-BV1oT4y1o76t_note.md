## 核心理念：什么是“大人对小孩”的态度？

上一节我们介绍了课程主题，本节中我们来详细解读“大人对小孩”这一核心理念的具体含义。

男人对女人应该拿出大人对小孩的态度。这首先不意味着不尊重。恰恰相反，如同大人对待小孩，需要的是更多的耐心、理解和包容。你是否认为大人不需要尊重小孩？越是成熟的大人，越需要去尊重小孩。

在我们的社会文化中，许多家长难以做到的一件事，就是向孩子说“对不起”。你是否也觉得，父母曾欠你一句道歉？这种反思同样适用于两性关系：尊重是基础。

**核心理念公式：**
**成熟的态度 = 尊重 + 耐心 + 理解 + 包容**

## 态度详解：四大核心要点

理解了基本理念后，我们来看看如何具体实践。以下是“大人对小孩”态度需要把握的四个关键方面。

### 1. 给予耐心与包容，而非“哄”与“惯”

我从未说过你要像哄小孩一样总是去哄她，也没有说你要惯着她。对待小孩，你需要的是更多的耐心理解和包容，这与无原则的“哄”和“惯”有本质区别。

### 2. 懂得维护威严，而非使用暴力

你需要懂得维护自己的尊严和威严，但这不代表你要对她很凶。威严在于讲道理，告诉她“我为你好”以及“为什么不能这样”。这与蛮横的凶恶完全不同。

如果她不听你的道理，说明你驾驭不了这段关系，威严不够，她根本不把你当回事。但**绝对不意味着你可以使用暴力**。打女人是懦弱无能的表现，与“大人”的身份背道而驰。大人能随便打小孩吗？这种思想的人无可救药。

**代码式原则：**
```python
if 对方犯错:
    做法 = 讲道理(阐述原因，表明关心)  # 正确
    做法 != 发脾气或使用暴力          # 错误
```

### 3. 懂得关心与迁就，但大事需做主

大人对小孩自然要多关心和迁就，但这不代表每件事都要依着对方。在诸如“中午吃什么”、“去哪里玩”这种小事上，你可以迁就。但在大事上，你必须能够做主。如果大事你也无法拿主意，那你就失去了“大人”的立场。

### 4. 懂得夸奖与鼓励，但不容许越界

懂得夸奖、赞美和鼓励对方，不代表她可以因此骑在你头上，或获得对你指手画脚的资格。鼓励是为了促进关系向好发展，而非放弃自己的框架和底线。

## 心态升华：成为值得依靠的伴侣

通过以上四个要点的学习，我们可以总结出“大人对待小孩”态度的深层目的：在大多数事情上，你需要在对方面前表现得更为强大、可靠。

你要让对方觉得你是一个值得依靠和信赖的人，你会包容她，对她好。你要展现出一种“一切问题到你这里都能被解决”的力量感。这就是男人对待女人应有的状态和心态。

唯有如此，你才不会成为女性厌恶的那种男人：自私、小气、缺乏承担、油嘴滑舌，或是在女性面前不自觉示弱、人格不够完整的男性。
