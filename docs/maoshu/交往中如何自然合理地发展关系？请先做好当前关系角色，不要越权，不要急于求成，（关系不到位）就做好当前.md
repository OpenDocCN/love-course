# 交往中如何自然合理地发展关系？请先做好当前关系角色，不要越权，不要急于求成，（关系不到位）就做好当前

在本节课中，我们将深入探讨在男女交往中，为何总有人试图“越权”行事，以及“急于求成”背后的心理动因。我们将分析这些行为如何影响关系发展，并提供清晰的思路，帮助你保持清醒与智慧。

## 理解越权行为的根源

上一节我们强调了做好当前关系角色的重要性。本节中，我们来看看为何人们会总想“越权”。

在男女交往中，尤其是男生追求女生时，常出现试图超越当前关系阶段的行为。作为讲师，我希望男生能多理解女生，而非对抗或怨恨。同样，也希望女生多理解男生。互相理解能让关系更和谐，但这或许只是理想状态。社会意识与思潮的发展，根本上由经济基础决定。如果底层的生产关系未理顺，上层的观念讨论往往收效甚微。

因此，我们审视为何许多男生总想越权。首要原因一定是**急于求成**。与女生交往时，总想快速达成个人目的，这就会导致越权。

## 两种常见的错误心态：越权与跪舔

除了越权，另一个紧密相关的点是“跪舔”或成为“备胎”。你是否发现，追求女生时，要么表现得非常急躁，要么则畏缩不前？要么急于求成，要么过度讨好。这两种状态有时会针对同一个女生，或在同一阶段面对不同女生时交替出现。

这都与**急于求成**的心态有关。**越权**的本质是：你不想经历中间的过程，想直接得到最终结果。你觉得自己可以赌一把，认为女生似乎不反感你，好像有机会。于是你不想麻烦，觉得慢慢约会太繁琐，干脆直接表白。这就是急于求成。

第二种是**不敢前进**。你觉得希望不大，但又想要。想赌又不敢，怕输。这时就会转向“跪舔”和过度讨好。

## 一个生动的比喻：浇花的智慧

这就像**浇花**。如果你想快点看到结果，会恨不得用开水去浇，等不及水凉就拼命浇灌。如果你太害怕没有结果，则会小心翼翼，结果浇水过少，怕浇多了把花浇死。这与浇花的道理相同。

这两种错误都源于：你没有关注客观事实，只聚焦于自身的恐惧。你没有观察这束花本身处于什么状态，此刻应该怎么做。你没有找到合乎自然规律的方式。

## 欲望如何蒙蔽智慧

人很多时候，无论是经营事业还是感情，老祖宗的智慧告诫我们：**心要静下来，内心要寡欲**。原因在于，欲望太多、想法太杂，会干扰你对事物的判断，导致你做出错误行为，思考错误问题，担心不必要的事情，并忽略本应考虑周全的问题。这最终会让人失去智慧。

你是否发现，当一个人显得愚蠢、智商降低时，通常发生在你着急或缺乏斗志的时候。在这两种状态下，人会表现得非常低能，缺乏智商和头脑。

## 情绪如何让人缴纳“情商税”

我想强调的是，每个人本质上都是聪明人。但受情绪控制时，你的智商和判断力会大幅下降。最影响人智慧和能力的，实际上是你对自身**欲望**和**情绪**的处理能力。有时这真可谓“一叶障目，不见泰山”。

一个简单的例子：你心里只想如何与某个女生在一起，即使女生已明确表示“我不喜欢你”，很多人仍不死心，不断追问是否有办法挽回。当你开始问这种问题时，你就变“蠢”了。为何？不是你本身变蠢，而是你变得容易相信别人告诉你的各种“挽回方法”，并为此付费。这就像病急乱投医，买各种课程，花大量金钱，最终一无所获，回头对事情更加迷茫。

请注意，很多人缴纳的不是“智商税”，而是**情商税**。例如，有些诈骗电话利用人的担忧恐惧情绪。当情绪上涌，脑子就不清醒了，钱转出去后才反应过来被骗。这不只是因为欲望，也因为情感。

## 实例分析：欲望如何令人盲目

我曾分享过一个案例：一个男生与女生约会，虽怀疑对方是酒托，但女生先将他带回住处，提议“先下去喝杯酒，待会儿再上来”。此时男生精虫上脑，女生又脱了衣服，然后说“时间还早，才10点，我们先下去走20分钟喝点酒或带点东西上来”。这时，即使男生知道可能是骗局，也不愿相信，心想“她都把我带上来了，衣服也脱了，就坐一会儿”。结果女生点完东西就溜了。这是智商税吗？不，当人的内心被调动时，往往不愿相信事实。

我自己大学时也差点被骗。那时社会经验少，网上通知我中了一部诺基亚手机，需支付几百元才能领取奖品。虽然看过反诈骗宣传，但我纠结许久，心里想的是“**万一是真的呢**”。这种“万一”心态，正是很多人被骗时警察都劝不住的原因。后来我意识到，所有联系方式都是对方提供的，我如何验证真伪？上网一查，果然是骗局。

## 核心机制：急于求成如何降低智商

人在犯蠢时通常如此。无论是急于求成，还是利用对方核心需求进行欺骗（例如，利用女生想结婚、嫁有钱人的心理，或让男生觉得“再努力一点就能追到”，引导其不断投入），本质都是**被自身欲望迷了心智**。

**公式：急于求成 → 智商下降 → 判断力减弱 → 做出错误决策**

一旦你急于求成，智商就会开始下降。对男生而言，智商一下降，女生就会看扁你。而你没有更好办法，又图省事、嫌麻烦，就会想方设法“搏一把”。例如，当觉得所有方法用尽（讲情话、威逼、利诱、打钱都没用），耐心耗尽时，就可能采取极端手段，如威胁曝光。此时，人已陷入**只关注自身欲望，缺乏对他人尊重和理解，且缺乏耐挫折能力**的状态。

## 健康的心态与当代挑战

正常健康的心态应是：遇到不顺时，告诉自己“别着急，慢慢来。对方不愿意，不要逼迫。可以之后再想办法，多找机会接触了解。如果实在不行，也别强求”。但很多人会想：“我自己想要的东西，你不能满足我，我就受不了。”

有人问：“为何会对女人感兴趣，而不是对干事业感兴趣？”这引出一个深层问题：**对现代人而言，干事业还有多少吸引力？** 如今，拥有奔驰宝马或欧洲旅游，已不像10年前那样能带来巨大成就感。社会比较中，你可能依然觉得自己不够出色。即使获得成就，也可能感觉不到特别的爽快，因为大家似乎都更有钱了。

对于某些人，尤其是生活安逸、无需特别奋斗就能过得不错的人，努力拼搏的意义可能变得模糊。如果家庭有房有车，父母有积蓄和退休金，在小城市生活压力不大，有些人便选择安逸度日，觉得无需追求更多。这种状态下，人可能转向其他领域寻求刺激或满足，例如沉迷感情游戏。

## 社会责任与个人觉醒

从个人欲望和社会责任角度看，有些人确实缺乏奋斗的动力。但**实现个人对社会价值，也是成就感的重要来源**。例如，我坚持讲课，除了兴趣，也有一份社会责任感支撑。我相信这类课程能启发许多年轻人，让他们以新视角审视情感、人生和价值观。

尤其在我们所处的中美对抗大环境下，未来十年世界可能有变局。青年一代需要做好思想准备。历史表明，当世界老大与老二发生交替时，总伴随动荡与变革。我们不应惰性生活，而应保持清醒，做好准备。

## 总结

本节课我们一起学习了：
1.  **越权行为**的根源是**急于求成**，企图跳过过程直达结果。
2.  另一种错误心态是**过度讨好（跪舔）**，源于害怕失败而不敢前进。
3.  欲望与情绪会严重干扰判断，让人缴纳“情商税”而非“智商税”。
4.  **核心公式**：**急于求成 → 智商下降 → 判断力减弱 → 错误决策**。
5.  健康的心态是**尊重客观规律，保持耐心，接受过程**。
6.  在当代，找到超越个人欲望的**社会价值与责任感**，能帮助获得更扎实的成就感。

记住，良好的关系发展基于**做好当前角色，尊重对方感受，保持清醒头脑**。避免被一时欲望蒙蔽，你才能在关系中找到真正自然和谐的路径。
