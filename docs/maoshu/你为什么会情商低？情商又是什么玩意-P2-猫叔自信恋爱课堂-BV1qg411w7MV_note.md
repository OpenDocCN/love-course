# 情商基础课 P2：情商是什么？ 🤔

在本节课中，我们将深入探讨“情商”的本质。我们将了解到，情商并非虚无缥缈的概念，而是一门关于“面子”与“尊重”的实用学问。理解这一点，是提升人际交往能力的关键。

## 情商的核心定义

上一节我们提到了情商的重要性，本节中我们来看看情商究竟是什么。

情商是一门关于**面子**和**尊重**的学问。我们生活中大多数生气的时刻，根源往往在于感到自己的付出、努力或自身没有被尊重。重新理解这一点至关重要：**情商 = 处理面子与尊重问题的能力**。

## 为何我们会“情商低”？

理解了情商的定义后，我们来看看为何有时会表现得“情商低”。

有时我们表现出低情商，可能源于长期的情绪压抑或特定的成长经历。例如，一个原本随和爱开玩笑的人，在开始认真面对事业和复杂人际问题后，脾气可能变差。这通常发生在几种情境下：

以下是几种容易引发情绪、导致低情商表现的情况：
*   **遇到能力不足的合作伙伴**：当你觉得对方“很蠢”且影响事务推进时。
*   **面对苛刻的指责**：被他人严厉批评，且自己确实理亏，无法反驳，只能憋着难受。
*   **在公开场合被攻击**：例如在网上被人“怼”，会感到特别气愤。

当你开始情绪化，放纵怒火时，就会出现一个问题：**你会表现得非常不体面**。

## 体面：情商的外在表现

不体面的行为会严重损害个人形象，那么什么是体面，又如何做到体面呢？

公开争吵、互相揭短，无论双方谁更有理，最终都会让所有人形象受损，显得很不体面。反之，能在公开场合保持风度、控制情绪，则体现了一种基本的情商。

控制情绪并非一味忍气吞声。真正的涵养不是窝囊地憋着，而是有策略地选择应对方式。很多时候，**最有情商的做法是沉默**。

例如，当众被人辱骂，选择看一眼后一言不发地离开。这种沉默如果来自一个平时并非“软柿子”的人，会带来强大的心理威慑，让旁观者自然站在你这一边。昆山“龙哥”被反杀事件中，公众舆论一边倒支持防卫者，正是“欺人太甚”触底反弹的体现。

以下是构建体面形象的要点：
*   **学会战略性沉默与忍让**：这不是认怂，而是不落入对方激怒你的陷阱。
*   **保持姿态，而非卑微**：面对欺负，态度要不卑不亢。面不改色地承受远比低头求饶更能震慑对方。
*   **情绪稳定是强大的信号**：稳定的状态传递出“不好惹”和“有底气”的信息，让人不敢轻易把你当“软柿子”。

体面，就是学会用行动捍卫自己的尊严。随意发泄、公开撕扯的人，毫无体面可言。

## 场面与情面：给予他人尊重

让自己体面是基础，接下来看看如何通过给予他人面子来提升情商。

**场面**，是懂得在公开场合给别人面子。一个简单的例子是送礼。送出能让对方在朋友面前“有面子”的礼物，就是经营场面。

**情面**，则更进一层，它关乎“重视”。情面是照顾对方的感受，送上对方想要却不好意思开口的东西。更高明的情面，是表达出极大的重视，同时巧妙地维护自己的边界。

例如，如何拒绝朋友借钱又不伤感情？可以这样做：
1.  展示自己确实困难的证据（如账户截图）。
2.  表达为帮他而四处奔走的意愿（甚至可展示为帮他向别人借钱的聊天记录）。
3.  提出一个自己牺牲较大的替代方案（如“我这块表你拿去当了吧”）。

通常对方会因你的“重情重义”而主动放弃请求，关系反而可能更好。这其中的核心公式是：**表达极度重视 + 展现客观困难 = 维护关系 + 守住边界**。

## 情商的实践：语言中的面子

最后，我们看看如何在日常语言中运用情商。

细微的语言调整，就能给对方完全不同的面子感受。例如：
*   “这位是我朋友。” vs **“这位就是我朋友。”** ——一个“就”字，强调了唯一性和重要性。
*   “我们以前是同事，后来成了朋友。” vs **“我们本来就是朋友，现在在一家公司做同事。”** ——后者将“朋友”的身份置于“同事”之前，分量更重。

这些例子都指向一点：情商就体现在如何通过言语和行动，**给人面子、让自己体面、表达对别人的重视**。

面子是一种**社交货币**。当你感到自己对别人好却得不到回应时，可以理解为你的“社交货币”（面子/价值）尚未得到对方认可。因此，提升情商的根本，在于**不断提升自身价值，把事情做好，赢得他人的尊重**，而不是一味讨好。

---
本节课中我们一起学习了：**情商是关于面子与尊重的学问**。其核心实践围绕三点展开：第一，**保持自身体面**，不轻易情绪化；第二，**懂得经营场面**，在公开场合给他人面子；第三，**照顾他人情面**，通过表达重视来深化关系。掌握这些原则，你就能在人际交往中更加从容有力。
