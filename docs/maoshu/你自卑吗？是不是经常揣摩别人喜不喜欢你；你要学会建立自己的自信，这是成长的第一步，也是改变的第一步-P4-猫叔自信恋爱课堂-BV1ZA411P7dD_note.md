# 猫叔自信恋爱课堂 P4：停止揣测，建立自信 🧠

在本节课中，我们将要学习一个阻碍人际关系发展的常见心态——过度揣测他人想法。我们将探讨其背后的原因、带来的负面影响，并学习如何通过关注他人需求和提升自我价值感来建立健康、自信的交往模式。

## 过度揣测的负面影响

上一节我们介绍了课程主题，本节中我们来看看为什么总想“看透”别人并非好事。

揣摩透他人的全部心思并非益事。即便你能完全看透一个人，在言行中也难免不留情面，让他人感到难堪、没有面子。

当你彻底摸透一个人的心思时，此人更可能厌恶你，而非喜欢你。无人乐意被完全看透。

例如，曹操处死杨修，正是因为杨修看透了他。被人看透意味着失去主导权，言行都将受限。因此，彻底看透他人并非好事。

与人相处，有时需要“装糊涂”。古语“难得糊涂”便是此理。最好不要试图完全看透他人，否则容易将事情定得太死，对双方关系并无益处。

彻底看透一个人，必定招致对方讨厌。许多人习惯性地想要看透他人，但这并无意义。所谓“水至清则无鱼，人至察则无徒”。若将他人的小心思、琐碎计较都看得一清二楚，便难以再做朋友。

以下是具体例子：
*   **情景假设**：一位好友向你借了500元，他有钱却不想归还。
*   **核心问题**：你愿意知晓此事，还是宁愿不知？
*   **关系影响**：若你知道他有钱不还，下次你再向他借钱时，内心便会产生复杂想法。这会对原本良好的朋友关系造成损害。

糊涂一些，关系尚可维持；看得太清、眼里容不得沙子，朋友便难再做。有人认为看透可以，但不表现出来。然而，当你完全看透一个人时，很难不通过言行流露出来。

试想，如果你被人完全看透，你还愿意与此人交往吗？第一反应通常是躲避，因为自己的所有心思都被洞悉，会感到非常不适与难受。

**结论是：无需看透他人，只需理解并关注对方的需求。** 这样关系会更融洽。此外，看透他人常使人难以给对方留面子。

## 关注需求：好感的核心来源

既然不应揣测，那该如何与人交往呢？关键在于关注对方的需求。

在摸清他人需求的前提下，能够顾及对方面子和尊严并提供所需，会让人对你产生好感和感激。

这个过程需要正确的思考方式。与人交往最大的忌讳是只从自身角度考虑需求，完全无视他人的困难与麻烦。

许多人际关系问题，都源于在交往中过度自我，只关注自身需求，不关心他人感受，将个人欲望置于他人之上。这是人际障碍的根本原因。

问题并非缺乏智商或不动脑筋，而是**脑筋用错了地方**。例如，总是揣摩他人心思、试图看透对方，便是心思用错了地方，自然无法获得好感。

那么，对他人产生好感的原因是什么？答案很简单：**因为感受到对方的关心**。对方照顾你的感受、体贴你，你心里才会感到温暖，产生好感。

所谓高情商，便是能真诚地关心他人，而非怀有特定目的。因此，与人交往，要了解对方需求，前提是**真正地关心对方**。

请自问：面对喜欢的女生，你真的关心对方吗？多数人并未真正关心。若真心关心，你自然会知晓对方的需求。

真正关心他人时，你首先会考虑能为对方做些什么。例如，与人结识后，若你有积极习惯，首先会想：**“我如何照顾对方？如何让对方感到更舒服？”** 这便是关注他人需求。

## 实践方法：从关注需求开始行动

理论如此，实践中如何操作呢？尤其是在社交场合。

在陌生聚会上认识女生的最简单、最有效的方法是什么？答案依然是：**关注并回应她的需求**。

以下是具体做法示例：
*   **场景一**：你看到有女生在一旁独自玩手机，显得无聊。
    *   **行动**：端杯水走过去，友好询问：“你还好吗？要不要一起来玩游戏？”
    *   **原理**：对方能感受到你的关心，从而产生好感。
*   **场景二**：天气冷，你看到女生穿着单薄。
    *   **行动**：拿件衣服过去，询问：“穿这么少不冷吗？”并主动为其披上。
    *   **原理**：关注对方当下的不适（冷），并提供直接帮助。
*   **场景三**：在酒吧，你看到有女生醉酒，正摇摇晃晃去洗手间。
    *   **行动**：将其扶到自己的卡座休息，提供热水，待其稍缓后再送回她的朋友处。
    *   **原理**：关注到对方的安全需求（醉酒需照顾）。此举不仅帮助了她，也可能让你结识她整桌的朋友。

以上所有案例的共同点是：**关注到了他人的现实需求，并提供了恰当的帮助**，从而让人对你产生好感。

## 根源剖析：自信与自尊水平

为何人们总忍不住去猜测“他/她喜不喜欢我”？这背后有更深层的原因。

许多人之所以不关注他人需求，反而沉迷于猜测对方想法，是因为存在一个根本问题：**你对自己没有信心**。

所有猜测别人是否喜欢自己的人，根源都是自信不足。为何会缺乏自信？因为**自尊水平不够**。

自尊水平即自我尊重、自我评价的程度。你觉得自己不够好，不认为自己属于人群中比较优秀的那一部分，因此自信不足。

例如，主讲人自称自尊水平较高，源于小时候学习成绩好，自觉是学校里的优秀分子，比一般人要好，因此相对自信。

反之，许多人因缺乏自信，才总是去揣摩他人心态。当你真正自信时，看到一位漂亮又单身的女生，想法会是：**“我要好好照顾她，我相信能给她幸福和快乐的生活。”** 即使起初被拒绝，自信的人也会积极想办法。

这种自信心态同样适用于其他领域，例如销售课程时，也相信自己的课程能真正帮助到他人。

## 总结与行动指引

本节课中我们一起学习了过度揣测他人想法的危害及其根源。

**核心结论**：
1.  **停止揣测**：试图完全“看透”他人会破坏关系，招致厌恶。
2.  **关注需求**：人际好感源于真诚的关心。将注意力从“他怎么看我”转向“他需要什么”。
3.  **提升自信**：总猜测他人喜不喜欢你，是**自尊水平不足**、对自己不自信的表现。建立自信是改善一切人际心态的第一步。

**行动公式**：
```
将思考从：
    “他/她喜欢我吗？” -> “我能为他/她做些什么？”
将心态从：
    自我怀疑 -> 自我认可（提升自尊水平）
```

改变始于意识的转变。从今天起，尝试在社交中练习关注对方的需求，并着手通过提升自我价值感来建立稳固的自信。
