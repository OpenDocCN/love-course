## 错误方法的陷阱：“套路”为何无效？

上一节我们列出了常见问题，本节中我们来看看许多人试图用来解决这些问题的一个流行但错误的方法：学习所谓的“聊天话术”或“套路”。

许多人寻找“话术大全”，希望背下标准答案。但同一个话术，由不同的人来说，效果会一样吗？你们总是忽略“你是谁”这个根本问题。

所有的聊天，尤其是男女初期的接触，对方想知道的是“你是谁”。如果你不关注这个问题，话术大全毫无用处。

**例如：**
*   `你：` “你有没有男朋友？” （你敢这么说吗？效果会好吗？）
*   `我：` “我看你的眼神和走路的姿势，感觉你被我深深吸引了。” （我能说，但你说出来，女孩可能只会对你翻白眼。）

因为“我”和“你”是不同的个体。甚至一个比我更帅的人说同样的话，效果又不一样。你们做事完全不考虑自身条件和具体情境。

你们原本智商没有问题，但被这种“套路思维”洗脑后，变得不知道该如何正常说话。因为它反人性。

**核心思考题：** 你想改变一个事物的状态，思考的出发点应该是这个事物本身。
*   你想把饭煮熟，得研究米和火候。
*   你想让一个女生从不喜欢你到喜欢你，思考的出发点应该是**这个女生本身**，而不是你自己。

因此，套路在任何地方都不适用，因为它不符合世界的基本规律。你们相信它，是因为错误地把女生当成了一台设定好程序的机器，以为按对开关（执行套路）就能得到结果。

**人是复杂且个性化的。** 每个女生在不同的年龄、时间、情绪状态下，需求都不一样。虽然存在共性（如喜欢被关心、被理解），但试图用一套机械化、公式化的东西去解决所有问题，是绝对不可能的。
