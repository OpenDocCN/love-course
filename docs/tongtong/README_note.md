# 课程名称：彤彤的Python入门教程 #01 🐍

在本节课中，我们将要学习Python编程语言的基础知识，包括变量、数据类型以及如何执行简单的打印操作。这些是构建更复杂程序的基石。

## 什么是Python？🤔

Python是一种高级编程语言，以其清晰的语法和代码可读性而闻名。它广泛应用于Web开发、数据分析、人工智能和科学计算等领域。

上一节我们介绍了Python的概况，本节中我们来看看如何开始使用它。

## 安装与第一个程序

要运行Python代码，你需要在计算机上安装Python解释器。安装完成后，你可以打开一个文本编辑器，输入你的第一行代码。

以下是保存和运行Python程序的步骤：
*   创建一个新文件，将其命名为 `hello.py`。
*   在文件中输入代码：`print("Hello, World!")`。
*   打开命令行或终端，导航到文件所在目录。
*   输入命令 `python hello.py` 并按回车执行。

执行后，你将在屏幕上看到输出：**Hello, World!**。这标志着你成功运行了第一个Python程序。

我们已经成功运行了第一个程序，接下来，让我们深入了解程序中的基本元素。

## 变量与数据类型 📦

变量是用于存储信息的容器。在Python中，你无需预先声明变量的类型，可以直接赋值。

以下是几种基本的数据类型及其示例：
*   **整数**：表示没有小数部分的数字。例如：`age = 25`
*   **浮点数**：表示带有小数点的数字。例如：`price = 19.99`
*   **字符串**：表示文本信息，需要用单引号或双引号包围。例如：`name = "彤彤"`
*   **布尔值**：表示真或假，只有 `True` 和 `False` 两个值。例如：`is_student = True`

理解数据类型后，我们可以学习如何使用它们进行输出。

## 打印输出与字符串格式化 🖨️

`print()` 函数用于将信息输出到控制台。你可以直接打印变量，也可以组合文本和变量。

以下是两种常见的字符串格式化方法：
1.  **使用 f-string**（Python 3.6+ 推荐）：
    ```python
    name = "彤彤"
    age = 10
    print(f"我的名字是{name}，今年{age}岁了。")
    ```
2.  **使用 `format()` 方法**：
    ```python
    name = "彤彤"
    age = 10
    print("我的名字是{}，今年{}岁了。".format(name, age))
    ```

两种方法都会输出：**我的名字是彤彤，今年10岁了。**。f-string 通常更简洁易读。
