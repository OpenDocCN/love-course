上一节我们概述了过度付出的整体危害，本节中我们来看看具体的五种情况。

以下是过度付出会引发的五种典型后果：

1.  **对方自信心过度膨胀，将你归为备胎或工具人。** 无条件的付出会抬高对方在关系中的心理地位，使其认为自身价值远高于你，从而不将你视为平等的潜在伴侣。
2.  **你的所有付出被视为理所应当，对方不会心存感激。** 当善意成为常态，它便失去了价值。对方会习惯性接受，而不会对此产生积极的情绪反馈或感谢。
3.  **对方对你的要求会越来越多，一次未满足便可能引发争吵。** 期望值会随着你的付出而水涨船高。一旦你未能达到新建立的、更高的标准，矛盾便会爆发。
4.  **对方会越来越看轻你，不会珍惜你。** 持续的单方面付出会削弱你的吸引力和尊重感。关系的天平严重倾斜，导致你处于低位。
5.  **双方关系越来越不平等，一次不满便可能让对方大发脾气。** 在这种动态中，你已被默认为必须满足对方需求的角色。你的感受容易被忽视，而对方的情绪则成为关系的主导。

因为你在对方心中，可能已经变成了**情绪垃圾桶**或**提款机**般的角色，而非一个值得尊重和爱慕的伴侣。所以，在感情中保持适度与平衡至关重要。
