# 课程名称：高情商回复技巧 - 第1课：掌握主动权 🎯

在本节课中，我们将学习当女生主动询问“在干嘛”时，如何通过高情商的回复来把握对话主动权，建立良好的互动关系。

## 概述

99%的男生都遇到过女生主动询问“在干嘛”或“我好无聊啊”的情况。多数男生的回复平淡无奇，例如“我在打游戏”或“我在睡觉，你想做什么”。作为女生，我了解她们提出此类问题的心态，多半源于无人陪伴，感到空虚和寂寞。然而，她能主动找你，也可能同时联系其他人。因此，你的回复既不能过于猛烈，也不应太过平淡。此刻，你正在与她周围的其他异性竞争。

## 核心回复公式

上一节我们介绍了问题的普遍性和背景，本节中我们来看看一个经过验证的有效回复策略。其核心是一个简单的**公式**：

**调起好奇心 + 有趣幽默 + 解决问题**

## 公式应用详解

以下是该公式在实战中的分步应用与解析。

### 第一步：调起好奇心

当时有位男生遇到同样问题，我分享了这个公式并协助他回复。我们是这样做的：“你怎么知道我专治无聊？” 这种出乎意料的回复，在1000个人里也难有一人这么说，能有效调动对方的好奇心。结果如我们所料，女生回复道：“哈哈哈怎么治疗？”

### 第二步：展现有趣幽默

在成功激起对方兴趣后，接下来需要延续对话的趣味性。我帮那位男生这样回复：“治疗要收费的，看你想干嘛，我可以给你打个折。” 这种幽默风趣的回应，实质上仍在探究对方无聊的原因和意愿。关键在于，此回复暗含了**邀约**的意图。如果进展顺利，可以直接导向共同约会。

### 第三步：识别信号并解决问题

女生随后回复：“我宿舍一个人都没有，都跑出去玩了。” 这句话的言外之意非常明显：她感到无聊，想出去玩。此时，选择权完全转移到了男生手中。如果你喜欢她，提出邀约几乎不会失败，因为这正好解决了她的难题。如果你无意，只需随意聊聊，不接这个话茬即可。

## 案例结果与总结

我记得，后来这位男生和那位女生一起去吃了火锅，席间相谈甚欢。女生还回请了他看电影。总而言之，遇到此类问题时，关键在于把握住对方主动提供的机会。运用“**调起好奇心+有趣幽默+解决问题**”的公式，可以让你在互动中占据主动。最终的决定权，始终掌握在你自己手中。

本节课中，我们一起学习了如何通过一个三步公式，高情商地回复女生的“在干嘛”，从而在对话中掌握主动权，并创造进一步发展的可能。
