## 概述

## 核心概念：服從性測試
上一节我们介绍了课程的主题，本节中我们来看看核心概念“服從性測試”。

**服從性測試** 是指通过向对方提出一系列容易达成的小要求，并逐步增加要求的深度，来测试和培养对方的服从意愿与习惯。

## 方法与实践
理解了核心概念后，我们来看看如何具体操作。

以下是实施“服從性測試”的步骤列表：

1.  **提出简单要求**：从最微小、最容易实现的要求开始。例如，一起吃饭时，请对方帮你带一杯奶茶。
2.  **逐步加深要求**：在对方完成简单要求后，可以提出稍进一步的要求。例如，让对方帮你倒一杯水。
3.  **创造自然情境**：如果不知道如何开口，可以创造一个自然的情境。例如，说“好渴啊”，来暗示需求。
4.  **观察对方反应**：理想的结果是对方主动且乐意地完成你的要求。

总而言之，这个过程就是一步步进行试探，从而拉近彼此距离。

## 总结
本节课中我们一起学习了“服從性測試”这一概念。我们了解到，通过从简单要求开始并逐步加深，可以有效地增进互动与好感。关键在于循序渐进和创造自然的情境。
