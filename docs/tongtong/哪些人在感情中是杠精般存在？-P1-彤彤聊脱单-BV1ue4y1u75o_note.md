# 课程名称：情感关系中的“习得性无助”与脱单障碍 🧠 - P1

在本节课中，我们将要学习一种在情感关系中常见的心理模式——“习得性无助”。我们将探讨这种心态如何阻碍个人建立健康的亲密关系，并分析其典型表现。

你知道哪种男生最不容易脱单吗？不是没钱，不是不帅，而是给自己找理由的男生最不容易脱单。

我遇到太多男生都会这样。我如果说每天在家里不可能脱单，要多出去扩大朋友圈，他会说我不会打扮，我没钱，不然早就脱单了。

上一节我们介绍了“习得性无助”的基本概念，本节中我们来看看它在具体情境下的表现。

以下是当他人提出建议时，持有这种心态的典型回应方式：

*   当建议“扩大社交圈”时，回应是：“我不会打扮，我没钱，不然早就脱单了。”
*   当建议“从身边朋友开始认识”时，回应是：“没有用，他都不喜欢。”
*   当建议“提升幽默感和情商”时，回应是：“爱情要真心，刻意的改变一点都不真实。”

这种人在心理学上叫做**习得性无助**。

其核心概念可以表述为：个体在面对反复的、不可控的负面事件后，形成的一种**对现实感到无望和无可奈何的心理状态与行为**。

在情感关系的语境下，这种心态的公式化表现为：
**面对脱单问题 → 归因于不可改变的自身缺陷（如没钱、不帅）或外部条件（如没有合适的人）→ 放弃主动尝试和改变**

我跟你说，这种男生就算中了奖，他的气质也不会有一个顶天立地男子汉的样子，照样无法脱单。

这样的男生就算月老来了都帮不了他。

你身边有这样的兄弟吗？爱得出来把他们点醒。

本节课中我们一起学习了“习得性无助”在情感关系中的表现。它并非指客观条件上的不足，而是一种在面对情感挑战时，习惯性地进行消极归因并放弃努力的心理状态。识别这种模式是做出改变的第一步。
