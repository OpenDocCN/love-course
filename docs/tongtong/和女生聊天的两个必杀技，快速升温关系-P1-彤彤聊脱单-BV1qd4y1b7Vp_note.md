**注意事项**：在夸赞第三方时，应避免使用过于直白和外貌化的词汇，如“长得好看”、“身材好”。这容易引发对方的醋意或产生负面印象。更安全的方式是夸赞一些模糊的特质，例如：“声音很特别”、“气质很成熟”。
