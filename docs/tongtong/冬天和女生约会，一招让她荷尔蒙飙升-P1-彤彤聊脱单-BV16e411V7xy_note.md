上一节我们详细拆解了步骤，本节中我们来分析其原理并总结要点。这个互动设计之所以有效，是因为它遵循了情感升温的节奏：制造悬念（耳边低语）→ 给予刺激（短暂握手）→ 主动撤离（放开手）→ 留下温暖（送上热水）。这种“给予-撤回-再给予（关怀）”的模式，比一直握着不放更能激发对方的心跳和荷尔蒙。
