以下是判断对方是否投入感情不足的三个核心观察点：

1.  **沟通投入度低**：在聊天互动中，当你连续发送多条信息（例如三、四、五句）后，对方通常只回复简短的一句。这种回应方式会让你感到对话难以顺畅进行，缺乏互动与延伸。
2.  **主动性缺失**：在双方的关系中，如果你不主动联系对方，那么对方也**肯定不会**主动来联系你。关系的维持完全依赖于你的单向推动。
3.  **态度不稳定**：对方对你的态度呈现**忽冷忽热**的特点。需要注意的是，其偶尔表现出的“热络”或积极回应，其前提往往是你近期曾为对方付出过（时间、精力、物质等）。

如果一段关系中同时出现上述几种情况，通常意味着对方的情感投入非常有限。

面对这种情况，不建议继续单向地付出或抱有幻想。更理性的做法是尝试主动冷静一段时间，暂时减少或停止主动联系与付出，观察对方的反应。

经过一段时间的冷静期后，如果对方的态度和行为出现积极、持续的转变，那时再去考虑未来如何发展这段关系也为时不晚。
