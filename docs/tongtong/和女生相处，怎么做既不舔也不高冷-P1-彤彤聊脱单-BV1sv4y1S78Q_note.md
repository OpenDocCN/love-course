上一节我们概述了本课目标，本节中我们来看看第一个核心原则。

## 1. 价值差异下的主动策略

如果你喜欢的女生在某个方面价值高于你，例如外貌更出众或学历更高，那么你需要展现出比她更高的主动性。这里的“主动”并非指送礼物或奉承讨好，而是主动建立联系。

以下是主动建立联系的具体方式：
*   分享有趣的视频。
*   讨论流行的网络梗。
*   推荐好听的音乐。

在认识初期，这种轻松、有趣的互动至关重要，它能自然拉近双方距离。

理解了价值差异时的应对方法后，接下来我们看看关系发展初期另一个关键阶段。

## 2. 明确原则，平等相处

感情发展初期常会经历“权力争夺期”。目标不是压制对方，而是确保双方地位平等。女生可能会通过一些事情测试你的服从性。

此时，你需要清晰自己的原则和底线。当对方提出不合理要求时，要勇于拒绝。拒绝时无需强硬对抗，而应做到条理清晰、有理有据。这会让女生觉得你是一个有魅力、有态度的男性。

掌握了维护平等关系的方法后，我们将探讨内在心态的调整。

## 3. 培养“无所谓”心态

对待感情保持一种“无所谓”的心态，这有助于你控制需求感，避免成为情绪的奴隶。你应该将主要精力投入到个人生活中。

以下是提升个人生活的建议：
*   持续自我提升。
*   培养个人爱好。
*   经营自己的社交圈。

当你不再将全部精力聚焦于感情时，自然就不会患得患失，也无需通过讨好的方式来维系关系。

调整好自身心态是基础，接下来我们学习如何引导对方为关系投入。

## 4. 学会提出合理要求

不要做一个只会单方面付出的追求者。你应该敢于向对方提出一些小要求。要求不宜过高，最好是在对方能力范围内、无需花费太多心思就能做到的事。

以下是可提出的小要求示例：
*   帮忙买瓶水。
*   提醒你会议时间。
*   代付小额停车费。

这些看似微小的付出，实际上能引导对方逐渐为你投入时间和精力。长此以往，你们之间的关系地位会逐渐趋于平衡。
