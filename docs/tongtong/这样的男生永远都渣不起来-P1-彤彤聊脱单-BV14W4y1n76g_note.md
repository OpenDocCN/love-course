# 这样的男生永远都渣不起来-P1-彤彤聊脱单-BV14W4y1n76g_note

在本节课中，我们将要学习如何识别一类在感情中真诚、投入但可能容易受伤的男性特质。我们将分析他们的行为模式、思维特点，并探讨如何保护自己。

你知道吗，有一种男生他永远都渣不起来。我身边就有这样的男生。

他不是那种经常谈恋爱的人，也不会轻易地喜欢上一个女生。但是一旦喜欢上，就会毫无保留地付出，全身心地去陪伴自己喜欢的人。

上一节我们介绍了这类男生的基本情感投入模式，本节中我们来看看他们的情绪与思维特点。

但是这种男生遇到一点小事，就喜欢胡思乱想，情绪都喜欢写在脸上，身边的朋友都看得出来。而且他们心思细腻敏感，懂得很多的道理。

以下是他们在面对感情矛盾时的典型反应模式：

*   但女生你只要不放弃，他们就会选择无条件的原谅。
*   即使自己内心有很多话想说，也会藏在心里，不敢表达自己的真实想法。
*   女生只要给一点点甜头，他们就会开心到不行，很容易被满足。

这种男生呢，他们就天生没有玩弄感情的天赋。他从小受到的教育和心理的防线，不允许他做诸如“见到谁都说暧昧的话，看到谁都可以聊起来”的事情。

即使自己真的受了伤，还是会替对方着想。

我想对这样的男生说，你们本身就很优秀。核心的行动建议可以总结为：**`做好自己 + 持续提升 = 遇见真心`**。只要学聪明一点点，不要再遇到烂人被伤害了。你只管做好你自己，让自己越来越优秀。

我相信，总会有一天，你会遇到真心待你的人。
