# 恋爱心理学小课堂 01：揭秘女生不为人知的心理 🧠

在本节课中，我们将一起探讨几个关于女生心理的常见现象。这些观点有助于理解异性在交往中的潜在想法和行为模式，内容基于社会观察与心理学研究。请注意，个体之间存在差异，以下内容为普遍性探讨。

上一节我们介绍了课程概述，本节中我们来看看第一个核心观点。

每个女生都喜欢被男生撩。有些女生外表显得高冷，但内心同样渴望男生的接近。她们甚至享受被撩的感觉。女生通常不会承认这一点，因为承认后可能被贴上“轻浮”的标签。

## 2. 依赖与被主导的心理 🧩

大部分女生存在“慕强”心理，喜欢被男生安排事务。不夸张地说，还有小部分女生有受虐倾向，沉迷于被男生掌控和对方强势的作风中。

上一部分探讨了心理依赖，接下来我们看看沟通中的关键信号。

理解了沟通信号后，我们进一步探讨一个常见的认知误区。
