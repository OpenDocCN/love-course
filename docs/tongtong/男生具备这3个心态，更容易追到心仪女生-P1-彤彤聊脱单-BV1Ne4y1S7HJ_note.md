## 心态一：共生而非寄生 🌱

上一节我们介绍了心态的重要性，本节中我们来看看第一个核心心态：共生。共生是指两个独立的个体，自身都拥有足够的幸福感和满足感，他们在一起是为了相互给予、共同成长，从而成就更好的彼此。简单来说，这是一种让双方都感到舒适、轻松的相处模式。

与之相反的是寄生心态。寄生是指一个人试图从另一个人身上获取自身所需的“养分”，甚至是为了满足自己的需求感而向对方不断索取。

## 心态二：付出而非讨好 🎁

理解了共生心态后，我们接下来需要区分一对容易混淆的概念：付出与讨好。这是决定你行为性质的关键。

首先要明白，**付出**和**讨好**有本质区别：
*   **付出**的公式是：`行为（如送礼物） + 无附加条件 = 付出`。这意味着你送一个礼物仅仅是为了让对方开心，没有其他隐藏目的。
*   **讨好**的公式是：`行为（如送礼物） + 期望获得回报（如对方的爱意） = 讨好`。这意味着你希望通过送礼物来换取对方的感情。

两者的出发点不同，导致了结果的截然相反。

付出的人是以“取悦自己”为出发点的。他对一个人好，只是为了享受“喜欢对方、对对方好”这个过程带来的满足感，而不是希望以此作为交换来获得什么。他们的逻辑是：**“我对你好，只是因为我喜欢这么做。至于你是否喜欢我，取决于我的个人魅力，而不是这点小礼物。”**

而那些误把讨好当成付出的男生，实际上是将自己的“好”物化成了等价的交换商品。他们的逻辑是：**“我送你礼物，是希望你多陪陪我；我给你点外卖，是希望你聊天回复得快一点。”** 一旦对方的行为没有达到自己的预期，他们就会感到委屈，认为“我付出了一切，是对方辜负了我”。实际上，他们并没有意识到，这只是自己的“讨好”，而讨好本身就是低价值和不自信的体现。女生的慕强心理，注定了她们不会被这样的人吸引。
