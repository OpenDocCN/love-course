# 课程18：认识她（教你快速认识你喜欢的女孩） 🎯
## 05. 搭讪技巧：搭讪的心态 💪

在本节课中，我们将要学习搭讪时应该具备的正确心态。很多人因为心态问题而不敢行动，我们将逐一剖析并解决这些心理障碍。

上一节我们介绍了搭讪的基本概念，本节中我们来看看搭讪时具体应保持何种心态。

很多人搭讪时不敢上前，不知道说什么，感到胆怯。

这一切的原因都是心态有问题。我们来探讨搭讪时应保持的心态。

现在请大家思考一个问题：想象未来某天你已结婚并有了孩子。

假设你的孩子是个男孩。有一天他回家对你说：“老爸，我在隔壁班看到一个女孩，我好想认识她，我喜欢她。”

作为父亲，你会如何回答？你会不会骂他，说“不行，不能这样做，你怎么能去认识陌生女孩”？

我相信你们不会这样对话。因为你知道，孩子想认识隔壁班女孩并非坏事，这是人之常情。

对于我们男生来说，搭讪只是做了我们自己认为该做的事情。

**核心心态公式：搭讪 = 做我认为自己该做的事**

女孩很好看，你想认识她。你只是做了你认为应该去做的事情而已。

很多人不理解搭讪，认为它不是好行为或不正经。实际上，搭讪只是认识女生的一个渠道。

搭讪与通过别人介绍或其他方式认识女孩相比，没有本质区别。只不过搭讪要求你更加主动和勇敢。

我们在搭讪女生时，并不一定要和搭讪的女孩谈恋爱。

如果你每天谈一个女朋友，那肯定是渣男行为。但你每天认识一个新朋友、一个新异性，这没有问题。

可能有人认为搭讪认识的女孩不靠谱。这是对搭讪的误解。

因为和女生通过搭讪认识后，你们同样需要经过互相了解、沟通、建立吸引、释放好感、释放暧昧、邀约等步骤。

最后才可能进入交往阶段。并非搭讪认识后就不需要遵循恋爱流程，随便搭个讪就成为男女朋友。

所以希望大家不要对搭讪有错误想法，也不要觉得搭讪认识女孩就一定要和她处对象、谈恋爱。

搭讪只是让你认识女孩并进行初步了解，距离交往阶段还很远。

可能有人会问：就算搭讪正常，但我们真的有必要去搭讪一个陌生人吗？

如果你有这种想法，请问自己：当你走在路上，看到一个身材很好、长得很好看的正妹从面前经过时，所有男人心里可能都会想“这妹子真正，如果是我女朋友就好了”。

这里的区别在于，百分之九十的男人只敢在心里想一想，但只有很少一部分人能把命运掌握在自己手里。

掌握命运的方法就是搭讪认识她。

想一想，在平常生活中，很多人接触女生的资源有限，或者遇到的女孩中没有合适的恋爱对象。

所以我们应该去搭讪。生命有限，你在千万人之中看到一个让自己怦然心动、心跳加速的女孩，有什么理由不去叫住她、告诉她你想认识她呢？

如果你去试一试、去搭讪，至少给了自己一个认识她的机会。哪怕最后没有成为朋友，你也没有遗憾，因为你做了这件事。

但如果你犹豫不去，就一点机会都没有了。你犹豫不决无非是害怕女孩拒绝你，害怕女孩对你有不好的反应，害怕女孩表现出强烈的排斥。

其实我想告诉大家，女生对于男生搭讪她，并没有大家想象中那么反感。

甚至很多女孩非常希望有男生来搭讪她，因为这也是女孩魅力的一种表现。

例如，我之前搭讪的一些女孩会在朋友圈发：“我今天出门被搭讪了。”所以女生对男生搭讪她并没有那么反感。

但是有一个前提：你这个男人不能看上去很low。如果你很low，女孩会直接走开。你必须是一个有魅力、优秀的男生。

作为一个优秀的、有魅力的男人，女生都很想认识你。女生被男生搭讪也是女生魅力的一种体现。

大家可以在生活中观察，有些女孩会主动释放想被搭讪的窗口。

例如在一些社交场合，你会发现有些女孩频繁用眼神与你接触、进行眼神交流。这就是女生释放出的搭讪邀请和信号。

这时你就应该大胆上去搭讪她。比如我以前在夜店玩时，在我桌子喝酒，会发现周围很多来玩的女孩与我进行眼神交流，我们会隔空碰杯，然后我很自然地上前搭讪，女孩也很自然地交流。

如果在社交场合，女孩给了你搭讪邀请或窗口，但你无动于衷，这就好像别人给你发了邀请函，希望你去她家做客，你却把邀请函扔了，非常可惜，而且对对方也不尊重。

所以我们在搭讪时只需要保持平常、平和的心态，保持平常心就可以了。

如果你认为搭讪是在耍流氓，那么你上去搭讪也会传递出很猥琐的感觉。所以搭讪是一件很平常的事情。

看到目标女孩后，你只要放轻松、自然而然地走过去，跟对方打个招呼，然后静静看着对方的反应就可以了。

像我带线下学员练习搭讪时，一般会选择万达这类商场。学员练习搭讪时，会整个人站在商场广场中间东张西望，到处找女孩，一会转身，一会往这边走，一会往那边走，整个人很焦虑不安，脖子伸得老长，眼睛到处看。

如果你在搭讪时也是这种状态，这些举动会给女孩很强烈的不安全感，女孩会认为你很猥琐，会提高警惕，导致搭讪失败。

搭讪本身是一件非常需要勇气的事情，它需要你打破自己心里的枷锁，从内心肯定自己、相信自己。

你要知道你是优秀的、有魅力的，然后很自然地把你自己展示在别人面前就可以了。

你要相信，一个真正敢于搭讪的男生都是勇敢的，因为这些男生勇于改变自己情感生活的状态，他们知道自己想要什么，敢于追求自己想要的生活，再根据现实反馈进行调整、总结、提升自己，最后达到满意状态。

所以我希望大家每个人可以给自己打气、加油，鼓足勇气勇敢迈出搭讪这一步。我也希望给你们一些鼓励。

搭讪其实很简单，如果你跟着我学，可能要不了一周时间，你就能很熟练地掌握搭讪这件事了。所以千万不要把搭讪当做非常困难的事情。

可能有些兄弟自尊心比较强，没办法面对失败。我以前有个兄弟问我：“道天老师，你能不能教给我一种百分之百搭讪成功的方法？”

我现在想和大家聊聊这件事。我可以非常肯定地告诉大家，这个世界上没有任何一种方法可以让你百分之百搭讪成功，世界上也没有任何人能百分之百搭讪成功。

因为搭讪本身就是概率事件。你可以想办法提高搭讪成功率，但你不可避免一定会遇到搭讪失败的情况。

因为搭讪并不是你一个人的事情，它是需要你和女生共同完成的事情。例如你去搭讪，女孩刚好有急事马上要走，你就经历了失败；你去搭讪，女孩今天心情不好，比如来大姨妈或被扣工资，你可能就会失败；或者你根本不是女孩喜欢的类型。这些原因都可能导致搭讪失败。

有些兄弟可能会说：“我去大街上搭讪女孩被女生拒绝了，多么尴尬呀，我的脸往哪里放呀？”但是你有没有想过，阻碍你找不到女朋友的真正原因，很多时候是不是因为你太要面子了？你是不是太把自己的面子当回事，而没有采取任何行动？

最重要的是一旦你的思维形成了这种想法，形成了自己吓自己的想法，那你就只愿意待在舒适区里面，只愿意待在一个玻璃房里面，你就会形成一种玻璃心，经历不了失败，经历不起任何生活的风吹雨打。

如果你未来是这样子，这不光对于你的感情，对于你的人生、工作等很多方面都会产生不好的影响。所以我希望大家，当别人还在犹豫不敢上去搭讪、不敢付出行动的时候，你要勇敢上去搭讪，然后慢慢让你搭讪的技巧变得熟练起来，讲话变得自然起来，态度变得越来越自信起来。
