# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等）。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

**核心公式：**
`变量名 = 值`

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：
```python
message = "你好，世界！"
```

## 基本数据类型 🔢

变量可以存放不同类型的数据。Python有几种内置的基本数据类型，最常见的有以下三种：

以下是三种最常用的基本数据类型：

1.  **整数（int）**
    整数是不带小数点的数字，可以是正数、负数或零。
    ```python
    age = 25
    temperature = -10
    ```

2.  **浮点数（float）**
    浮点数是带有小数点的数字，用于表示更精确的数值。
    ```python
    price = 19.99
    pi = 3.14159
    ```

3.  **字符串（str）**
    字符串是由一系列字符组成的文本。在Python中，字符串必须用单引号（`'`）或双引号（`"`）括起来。
    ```python
    name = "小明"
    greeting = 'Hello'
    ```

## 如何使用变量？ 🛠️

创建变量后，我们就可以在代码中使用它。你可以直接使用变量名来代表它里面存放的值。

例如，我们可以打印出变量的值，或者用变量进行计算：
```python
# 定义变量
score = 95
name = "小红"

# 使用变量
print(name + "的分数是：")  # 打印文本
print(score)               # 打印变量的值

# 变量参与计算
bonus = 5
total_score = score + bonus  # 将score的值加上5，结果存入total_score
print("加上奖励分后的总分是：", total_score)
```

## 变量命名的规则 📝

为了代码清晰易懂，给变量起名需要遵循一些简单的规则：

以下是变量命名的主要规则和约定：

*   **只能包含**字母、数字和下划线（`_`）。
*   **不能以数字开头**。例如，`1st_place` 是无效的，但 `first_place` 是有效的。
*   **不能使用Python的保留字**（关键字），如 `if`、`for`、`while` 等作为变量名。
*   **区分大小写**。`Age`、`age` 和 `AGE` 是三个不同的变量。
*   **建议使用描述性的名字**。用 `user_name` 比用 `a` 或 `x` 要好得多。

## 查看数据类型 🔍

有时你可能不确定一个变量是什么类型。Python提供了一个内置函数 `type()` 来查看。

```python
height = 180
weight = 70.5
city = "北京"

print(type(height))  # 输出：<class 'int'>
print(type(weight))  # 输出：<class 'float'>
print(type(city))    # 输出：<class 'str'>
```
