# 对你越来越冷淡，回复消息很慢，：1664897922100.m

在本节课中，我们将学习如何避免在与人（尤其是心仪对象）聊天时陷入被动，导致对方回复变慢、态度变冷淡。我们将通过分析一个具体案例，指出两个常见的错误，并学习如何通过建立平等地位和展现独特标准来改善沟通。

许多人在与喜欢的人接触初期，聊天回复及时，对方甚至可能主动联系。但后来，对方回复开始变得简短，甚至只有两三个字。如果此时更加主动地追求，结果往往不理想。想要改变这种情况，就需要规避接下来案例中的错误。

以下是案例背景：男生邀请女生吃饭，女生回复说“我明天要出差，今天要早点睡”。我们将分析男生在此场景下容易踏入的两个“坑”。

## 第一坑：学会拒绝，保持平等地位 🤝

上一节我们提到了沟通变冷淡的普遍现象，本节中我们来看看第一个关键错误：不懂拒绝，导致地位不平等。

当对方以“要出差早睡”为由婉拒邀约时，很多人的本能回复是追问细节，试图说服对方。例如：“明天什么时候出差？时间不紧张就可以出来坐坐。” 这种回复的核心意图是告诉对方：我今天就想见你，希望你能抽时间。

这类似于销售员不断约见客户，不断增加自己的筹码。但这样做时，双方地位已不平等，选择权完全在对方手中。与喜欢的人沟通也是同理。

正确的做法是直接终止这次邀约，例如回复：“那行吧，改天再约。” 这传递出的信息是：我并非非你不可，我也可以选择拒绝你。这样双方就站在了平等的角度。

随后对方回复：“好的没问题，只是这次时间太赶了。” 这表明双方处于一个彼此尊重的社交地位中，对方也在礼貌地解释原因。当你明白并做到这一点时，你就已经区别于大多数男生了。

## 第二坑：超越俗套的夸赞方式 ✨

在建立了平等沟通的前提后，我们可以开启新话题。接下来我们看看第二个常见错误：落入俗套的夸赞方式。

案例中，女生出差时发了一张照片并说：“同事把我拍得好难看，朋友圈都不能发了。” 此时，典型的错误回复是：“没有啦，我觉得你很好看，这照片挺好看的。”

这样做的结果是暴露了“需求感”，因为几乎所有追求者都会这么说，显得毫无新意。

那么，如何夸赞对方又不暴露需求感呢？关键在于夸赞后建立自己的标准。

男生可以回复：“照片没有本人好看。” 女生回复：“啊，很多朋友都这么说。” 这说明她身边已有许多夸赞。

接着女生说：“但我一直觉得本人也就那样吧，哈哈。” 这里女生表现出了一丝矜持。

此时，在夸赞之后，就要建立自己的标准。例如可以回复：“主要是我不喜欢很幼稚的。” 言外之意是：我夸你并非仅仅因为你好看，而是因为你符合我的标准。

这引发了对方的好奇：“幼稚是什么意思？” 这时便可以阐述自己的标准，展现出与其他追求者的不同，从而脱颖而出。

以上两个点的核心关键在于：**双方要在平等的地位上建立沟通**。无论是在情感、工作还是生活中，都是如此。只有这样，男生才不会落入俗套。因此，喜欢的人也不会对你越来越冷淡。由于你与身边的追求者不同，对方反而可能主动联系你。

在本节课中，我们一起学习了如何通过“学会拒绝”来建立平等沟通地位，以及如何通过“有标准的夸赞”来展现独特性，从而避免聊天变得冷淡。掌握这些原则，能帮助你在人际交往中占据更主动、更尊重的位置。
