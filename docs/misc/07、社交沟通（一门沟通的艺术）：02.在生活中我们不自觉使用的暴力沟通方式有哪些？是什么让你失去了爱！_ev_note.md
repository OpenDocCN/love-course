# 课程07：社交沟通（一门沟通的艺术） 🗣️  
## 02. 生活中不自觉使用的暴力沟通方式有哪些？是什么让你失去了爱！💔

在本节课中，我们将要学习生活中常见的暴力沟通方式。这些方式往往在不经意间破坏人际关系，让我们感受不到爱。通过识别这些方式，我们可以避免使用它们，从而改善沟通质量。

上一节我们介绍了非暴力沟通的四个核心要素：观察、感受、需要和请求。本节中我们来看看哪些沟通方式属于“暴力沟通”，以及它们如何影响我们的人际关系。

---

### 暴力沟通的四种常见方式

以下是四种常见的暴力沟通方式，它们会导致人际关系疏远和伤害。

#### 1. 道德评判 🧐

道德评判是指用自己的道德标准去评价他人。当一个人的行为不符合我们的价值观时，我们容易将其视为不道德、错误或邪恶。

**例子**：  
- “这个人太难相处了。”  
- “他总是喜欢发表建议，真讨厌。”  
- “老师布置这么多作业，太过分了！”

**影响**：  
道德评判会让对方感到被指责，从而引发敌意。即使对方勉强接受批评，也往往不是心甘情愿的，长期下来会导致关系疏远。

**核心概念**：  
暴力沟通公式：**道德评判 = 用自己的标准评价他人 → 引发敌意**

---

#### 2. 进行比较 📊

比较是评价的一种形式，它会让我们忽视自己和他人的真实感受。比较往往导致自卑、嫉妒或不满。

**例子**：  
- “看看别人家的孩子，多优秀！”  
- “别人的男朋友总是那么体贴。”  
- “你为什么不能像她一样努力？”

**影响**：  
比较会让人感到不被尊重和认可，尤其是当自己处于劣势时。长期比较会破坏亲密关系，让人逐渐疏远。

**核心概念**：  
暴力沟通公式：**比较 = 评价他人优劣 → 伤害自尊与关系**

---

#### 3. 回避责任 🏃‍♂️

回避责任是指将自己的行为归咎于外部因素，而不是主动承担。这种沟通方式让我们失去对自己情感和行为的掌控。

**例子**：  
- “我不得不加班，是老板要求的。”  
- “别人都闯红灯，所以我也跟着走了。”  
- “我骗客户是因为公司让我这么做的。”

**影响**：  
回避责任会让我们成为“傀儡”，失去自主性。同时，它也会破坏信任，让人感到不负责任。

**核心概念**：  
暴力沟通公式：**回避责任 = 将行为归咎于外部 → 失去自主性与信任**

**正确表达方式**：  
用“我选择…因为…”代替“我不得不…”。例如：  
- “我选择加班，因为我想完成项目。”  
- “我选择闯红灯，因为我想快点过马路。”

---

#### 4. 强人所难 🚫

强人所难是指通过威胁或惩罚来强迫他人满足自己的需求。这种方式往往隐藏在“爱”或“关心”的外表下。

**例子**：  
- “如果你不听话，妈妈就不爱你了。”  
- “不给我买这个包，我们就分手。”  
- “考不好就别想玩游戏！”

**影响**：  
强人所难会导致对方出于恐惧或内疚而妥协，但长期下来会积累怨恨，破坏关系。即使对方服从，也不是出于真心。

**核心概念**：  
暴力沟通公式：**强人所难 = 威胁或惩罚 → 引发恐惧与怨恨**

---

### 如何应对暴力沟通？

当你发现对方使用暴力沟通时，可以参考以下方法：  
1. **识别暴力沟通方式**：明确对方正在使用哪种暴力沟通（道德评判、比较、回避责任或强人所难）。  
2. **回归非暴力沟通四要素**：用观察、感受、需要和请求的方式回应。例如：  
   - 观察：“我注意到你最近经常比较我和别人。”  
   - 感受：“这让我感到有些难过。”  
   - 需要：“我希望你能看到我的努力。”  
   - 请求：“下次可以多鼓励我吗？”  
3. **分享知识**：将本节课内容分享给身边的人，帮助更多人意识到暴力沟通的危害。

---

### 总结

本节课我们一起学习了四种暴力沟通方式：道德评判、进行比较、回避责任和强人所难。这些方式会破坏人际关系，让我们失去爱和信任。通过识别并避免这些沟通方式，我们可以更好地与他人连接，建立健康的关系。
