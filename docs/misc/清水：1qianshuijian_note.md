## 核心原则：静止与定位 🎯

上一段我们介绍了课程主题，本节中我们来看看最核心的原则。清水健强调，在开始时不要急于动作。

进入后的前5到10秒，**不要移动阴茎**，保持静止。这个阶段最重要的任务是找到对方感觉最舒服的那个点，然后停下来。这样做是为了让阴茎的形状与阴道黏膜充分**密着**（紧密贴合）。

以下是实践这一原则的具体好处：
*   **提升快感**：充分的密着能增加双方的舒适度和快感。
*   **避免疼痛**：对于女性而言，如果动作不当或身体疲惫，可能会感到背部等部位疼痛。先定位再静止的方法可以起到一定的预防作用。
*   **掌握核心技巧**：这是在体位变换时也应短暂应用的技巧，通过几秒钟的静止来形成肌肉记忆。

## 高效动作：减少无用功 💪

上一节我们讨论了静止的重要性，本节中我们来看看如何高效地动作。很多人反映在性爱中腰部容易累，或者感觉使不上劲。

造成这种情况的最大原因是，**动腰时带动了整个身体**，做了大量无用功。我们来分析一下常见的低效动作模式。

以下是三种典型的无用动作：
1.  **腰部单独剧烈运动**。
2.  **手臂为了支撑身体而过度用力**。
3.  **肩膀随着动作前后晃动**。

这些动作会让人很快感到疲劳。高效的技巧恰恰相反：**首先要固定住肩膀**。

例如，采用跪姿时，膝盖和上半身保持稳定。将手按住伴侣的肩膀或床面，固定住自己的肩部位置。这样，肩膀就不会前后移动。动作的力量主要来源于骨盆的倾斜角度和核心发力，而非大幅度的身体晃动。

**核心公式：最小限度的动作 + 固定的肩部 = 最大的快感输出**

即使将手放在伴侣身上，目的也是稳定和轻微施压，而不是推拉。轻轻按住伴侣的肩膀，利用身体自重和角度进行运动，是非常高级且省力的技巧。

## 进阶应用：利用重力 🚀

上一节我们学会了如何固定身体，本节中我们来看看如何借助重力。清水健强烈推荐以重力为主导的体位。

在这种体位中，男性几乎不用主动发力。例如，某些后入或侧卧位，让地心引力帮助阴茎沿着阴道内的G点或子宫颈（Portio）轨迹自然运动。

**动作描述**：男性保持肩部稳定不动，仅通过微调角度，让重力完成主要的“进入”动作。女性也无需大幅移动。

这种方法的优点是双方都不会累，却能通过精准的角度和持续的压力带来强烈的快感。接下来，我们看一个结合了静止原则和重力技巧的变式。

首先，以正常体位进入，保持5到10秒不动。然后，转换为一个能利用重力的体位（例如女性在上或侧卧位），并确保双方的接触点（如肩膀、腿部）稳定贴合。当腿部相互勾住时，这种稳定感会更强，非常适合刺激G点。

## 总结与常见误区 🧠

本节课中我们一起学习了清水健性爱技巧的三个核心部分：**初始的静止与定位**、**减少无用功的高效身体力学**，以及**利用重力的进阶方法**。同时，我们强调了通过身体接触（如手放腹部）来增强情感连接的重要性。

最后需要澄清一个常见误区：成人影片（AV）中的表演通常为了视觉效果而充满激烈的大幅度动作。但在现实中，女性未必能从中获得持续的舒适感。频繁的大动作可能会破坏刚刚积累的快感。

真正的舒适感来自于找到那个敏感点，然后保持**密着**和**持续的压力**，而不是剧烈的抽插。小幅度的、精准的按压往往比大动作更有效。旁观者看来可能感觉“很舒服”，但当事人的实际体验可能“完全不同”。
