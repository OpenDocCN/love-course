# 婚姻重建教程 Part10：如何与另一半“有效合作” 👫

在本节课中，我们将学习婚姻关系中的一个核心课题：如何与伴侣进行有效合作。合作是人际关系的最高境界，但在婚姻中，许多夫妻却陷入了不健康的权力斗争，导致关系紧张和消耗。我们将深入探讨权力斗争的根源、识别方法，并掌握七个转向合作共赢的关键要诀。

## 识别隐藏的权力斗争 ⚔️

上一节我们提到了合作的重要性，本节中我们来看看阻碍合作的关键原因——不健康的权力斗争。当两个人从“我”变成“我们”时，本应像团队一样协作。然而，许多夫妻却在互相竞争、攀比和较劲，争夺关系中的上风地位。

这种权力斗争让人把精力和时间浪费在与伴侣的竞争上，造成了怨恨和痛苦。很多人并未意识到自己正陷于其中，只会感觉总是在较劲或想说不。

以下是辨别隐藏权力斗争的六个特征，请对照查看：

1.  你希望对方必须按照你的方式行事，妥协让你感到非常不舒服。
2.  如果你妥协了，内心无法释怀，总想在未来赢回来或惩罚对方。
3.  当伴侣拒绝你时，你总怀疑其动机是出于恶意，是为了打压你。
4.  你经常试图证明自己比对方更优秀、更正确、动机更好。
5.  你努力让自己成为关系中爱得更少的一方，害怕爱成为被威胁的把柄。
6.  你很难向伴侣道歉，担心退让一步就会一直退让，或给对方羞辱你的理由。

如果你发现有多条符合，那么你可能正经历与伴侣的权力斗争。

## 理解权力斗争的实质与影响 ⚖️

不健康的权力斗争割裂关系，消耗彼此。夫妻争夺的权力，本质是**影响他人而不被他人影响的能力**。斗争的目的是希望自己在关系中更多地影响和掌控伴侣。

斗争可能通过争吵、吼叫来彰显权力，也可能以受害者姿态强迫对方服从。研究表明，仅有不到一半的夫妻认为彼此势均力敌，能较好合作。

在健康的关系中，权力不平衡是流动的。双方会轮流在不同领域拥有更大话语权，例如：
*   `老公在理财方面有更大权力`
*   `老婆在生活安排上有更大话语权`

或者，即使存在不平衡，双方都认为总体是适当且可接受的。权力低的一方无需低声下气或违背原则。

良性的权力斗争能促使权力低的一方提升自我影响力，且斗争不会成为焦点，双方更重视合作共赢。然而，如果斗争走向偏执和极端，双方投入大量精力在争输赢、比强弱上，就会将对方置于对立面，忘记共同目标，最终割裂关系。

## 有效合作的七个要诀 🗝️

既然合作如此重要，那么接下来就来看看具体应该怎么做。以下是帮助夫妻走出斗争、转向合作的七个关键要诀。

### 要诀一：停下权力斗争

如果你发现自己处于恶性权力斗争中，无论你是哪一方，都需要意识到这与建设夫妻关系的目标是背道而驰的。

著名的婚姻咨询师提出了停下斗争的五步法：

1.  **抽身察觉**：先冷静下来，观察是谁开始了斗争、为什么斗争、情绪状态如何。
2.  **以身作则**：控制住自己，率先放弃斗争。
3.  **谈谈斗争**：与伴侣聊聊正在发生的斗争，避免使用指责性语句。
4.  **暴露自己**：尝试说出斗争底下的真实想法和情绪。
5.  **保持耐心**：权力斗争不会立刻消失，重建连结需要时间和多次尝试。

### 要诀二：转移权力斗争

如果无法直接停下斗争，可以尝试将注意力从关系内部转向外界。关注外界问题能降低内部斗争。

处于斗争中的夫妻，可以思考哪些是你们“共同的敌人”。将对获胜的追求从伴侣身上移开，转移到工作、社交、学业等其他领域。有时，我们在其他领域受挫，才会将权力需求转移到亲密关系中。

你可以问自己：`我对自己的现状满意吗？这段关系让我幸福吗？`如果发现是通过斗争来满足需求，请尝试去其他领域满足。

研究发现，女性更看重在夫妻关系中的权力位置，权力高则亲密感高。男性则不同，即使关系中权力偏低，也能从其他领域的权力感中获得抚慰。因此，转移斗争对男性可能更奏效，但女性同样可以借此方法，将注意力转移到健身、学习等个人成长领域。

### 要诀三：减少强迫

当你向伴侣提出要求时，常常可能带着强迫意味。如果遭到拒绝，继而采用生气、威胁、冷战等高手段，希望对方屈从，这会让被强迫的一方非常不舒服。

有效合作的第三个要诀就是减少强迫。核心是：**我不要求你必须遵从，你也不一定要服从。我们共同讨论，一起决定。**

### 要诀四：把“我”变成“我们”

这是一个通过语言暗示改变思维模式的小技巧。反复使用“我们”这个词，能引导我们思考对方的立场和感受，关注共同目标，而不仅仅是单方面需求。

日常生活中，我们常把自己的目标当成关系的共同目标，但伴侣未必认同。例如，“我不想让老公玩游戏”是你的诉求，但老公仍要玩，说明他不认可。

“我们”意味着需要从伴侣角度思考：`他需要什么？他为什么这么做？` **“我”加上“他”才等于“我们”**。

例如，你不想让老公玩游戏，是担心健康和孩子受影响。这与老公想要健康、希望孩子受积极影响的愿望并不对立。你们有了共同目标，就可以合作讨论解决方案，比如寻找更健康的娱乐方式，或商定合理的游戏频率。

### 要诀五：明确分工与责任

有了共同目标，但有时仍无法合作，问题可能出在具体的责任分配上。对方不清楚自己该做什么、何时做、做到何种程度。

因此，合作中一定要明确分工。例如在家庭中：
*   `谁负责理财，谁负责日常消费？`
*   `今晚谁做饭，谁洗碗？`

光靠默契达到完美配合是少数。责任分配在商业合作中是基础，在夫妻关系中同样重要。

### 要诀六：懂得妥协

当两人意愿不一致时，要懂得做出必要妥协，放下“你必须听我的”执念。通常，双方各让一步，商量出一个折中方案。

我们可以用**双赢视角**看待妥协。过度权力斗争导致双输，即使你表面一直赢，也会增长对方不满，损害关系质量。相反，在不违背原则的前提下，你的付出让伴侣高兴，他的好心情和互惠回报也会让你快乐。

### 要诀七：别做“差评师”

我们尤其在意亲密之人对自己的评价，而差评让人难以释怀。在负面评价下，关系难以理想成长。

停止给伴侣贴坏标签、打差评。积极的赞美、鼓励和奖励，更有利于开展合作。当伴侣做得好时，请给予真诚表扬，不要视为理所当然。如果做得不好，也不要劈头盖脸批评。

心理学上的**罗森塔尔效应**指出，热切的期望和赞美能够产生奇迹。积极的鼓励能让对方变得更好。

## 共同梦想：婚姻的抗衰剂 💡

婚姻想要幸福长久，一个重要前提是两人拥有共同的梦想和目标。婚姻可以没有持续燃烧的爱情，但不能没有梦想。爱情的感觉可能在婚后几年内消退，但共同的梦想可以支持婚姻延长寿命。

脑科学研究表明，幸福指数高的人，平时总是在做某件事情。幸福的婚姻也是如此。人们在制定和实现目标的过程中，会分泌多巴胺感到快乐，这与目标是否宏大无关。越是幸福的婚姻，越能不断发现新目标，总是走在向目标前进的路上。

如果夫妻想要幸福，就必须趁着爱情荷尔蒙消退前，尽快找到你们的**第一个共同目标**。实现后，再尽快找到下一个目标。如此反复，一起合作努力，就能构建幸福的婚姻。

如果夫妻不能共同制定和实现目标，这种目标的分裂会使关系变得索然无味，甚至容易导致问题。在现实中，有共同梦想的婚姻是少数，很多人没有目标地过日子，因此与幸福体验绝缘。

请记住：**幸福的获得靠的是你们的共同目标和梦想，以及合作实现梦想的意志和努力。**

## 课程总结与作业 📚

本节课中，我们一起学习了夫妻不能合作共赢的根源——不健康的权力斗争，并掌握了与另一半有效合作的七个要诀：
1.  停下权力斗争
2.  转移权力斗争
3.  减少强迫
4.  把“我”变成“我们”
5.  明确分工与责任
6.  懂得妥协
7.  别做“差评师”

夫妻间的合作意愿非常重要。只要有合作的意愿，再大的困难也能越过。这些要诀有助于我们在婚姻中增强合作的倾向和能力。
