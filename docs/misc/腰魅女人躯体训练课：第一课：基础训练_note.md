# 腰魅女人躯体训练课：第一课：基础训练 🧘‍♀️

在本节课中，我们将要学习腰魅躯体训练的第一个基础动作。这个动作的核心是利用呼吸来带动胸腔的运动，为后续更复杂的腰部训练打下基础。掌握呼吸与身体的连接，是唤醒身体感知和柔韧性的第一步。

## 核心动作：呼吸带动胸腔

接下来我们开始第一个腰魅动作的训练。这个基础练习很简单，你只需要运用呼吸来带动胸腔。

有些人可能会疑惑，这个动作和训练腰部有什么联系。不必着急。

因为在第二个和第三个动作中，你会发现这个部分会与其他动作连贯起来。在老师的视频和许多舞蹈中，以及瑜伽体式或亲密接触时，常常通过顺畅的呼吸来调动全身的律动。

以下是具体的练习方法，请跟随老师一起练习。

*   例如呼气时，吸气时。我们先吸气。
*   在慢慢吸气的同时，将胸腔向斜上方抬起。注意观察吸气的过程。

关键在于，我们在吸气进行前部分的训练时，请记住向上提起的方向不是向前，而是向斜上方，跟随呼吸向上移动。

让我们再看一次。吸气。

吸气，吸气，吸气……（反复练习），慢慢来。

在这个过程当中，速度不要太快。慢慢去体会你的气息，它会从你的第一脉轮（海底轮），从下方私密部位开始，缓缓通过第二脉轮、第三脉轮、第四脉轮，通过我们的心脏，一点一点向上抬升。请用身体去想象。

这时也会运用到我们的性大脑和性幻想。想象有一股气体在身体中央，温柔地向上提，温柔地通过胸腔释放出来。

如果你对自己足够温柔和体贴，你会在这个过程中，慢慢学会与自己的身体建立连接。你会逐渐觉察到，自己的身体其实很柔软、很舒服、很自在。

我们再看一下，再多做两次。带上胸腔的部分，跟随老师一起吸气、吐气。

## 辅助技巧：呼吸节奏控制

这里老师再教给你一个小方法。如果你不知道吸气或吐气应该持续多长，这当然需要根据每个人呼吸的时长、自身运动节奏和身体节奏来定。

同时，你可以尝试在吸气时默数：**1, 2, 3, 4, 5**。吐气时也以同样的节奏吐气并发出声音：**“哈…” (对应1, 2, 3, 4, 5)**。

我们试一下，跟随老师来：吸气，吐气，吸气，吐气，吐气，再来一次，吐气，吸气，吐气。

跟随老师练习，看起来挺简单，是不是？你自己也可以做到，只要放慢速度，相信并开始温柔地对待自己就可以。

## 关键要点：吐气发声

第二个关键点，老师想要说明的是：当我们习惯用呼吸带动胸腔吸气后，在吐气时，请尽量发出声音。

也许最开始你会不习惯，觉得好像很奇怪。没关系，可以小声一点 **“哈…”**，或用气息 **“哈…”**，都可以。

当你比较喜欢并适应这种方式，感觉比较自在了之后，就可以在吸气后，自然地 **“哈…”** 出声。

因为你在做这个过程时，你的喉轮会被打开。身体里积压的负能量和杂质，都会在你吐气时，**“哗”**的一下被释放出来。

这个动作每天在家里练习10到15分钟。如果你感觉非常舒服和自在，可以配上音乐，延长训练时间。

## 动作进阶：连接与延展

当我们的胸腔呼吸练习得比较顺畅之后，就可以进入第一个姿势中稍微进阶一点的小训练。

当你自己感觉比较舒服，并且感觉与老师的示范差不多时，你就可以带上自己的上半身躯体，任意发挥。

并不是说我现在教给你的这一套方式就一定完全适合你。你可以在这个大框架下，去感受自己的身体，去感受自己喜欢什么。

我讲的重点就在这里：**你喜欢什么**。

例如，我经常在呼吸时会发现，我在向上提时，我的腰会跟着动，我的侧腰有时也会向斜上方跟着移动。

如果你的腰在带动时，确实会带动到这个部分，就任由你的躯体去延展就好了。

甚至有时候，在你加入应用时会发现，左右的波浪动作（wave）都可以加上。前提是你前面的基础姿势做得很到位。

这是第二个我们可以进行训练的地方，即带动你的腰身部分，配合胸腔呼吸进行练习。
