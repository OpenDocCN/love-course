# 课程28：重建甜蜜婚姻的沟通技巧 🗣️💖

在本节课中，我们将学习夫妻关系中健康抱怨的方法，并掌握应对高难度沟通场景的核心技巧。这些技能有助于避免指责升级，促进建设性对话，从而维护和深化亲密关系。

## 健康抱怨的艺术

上一节我们介绍了应对冷暴力的沟通技巧。本节中，我们来看看如何在亲密关系中进行健康的抱怨。

心理学家约翰·科特曼的研究指出，**指责是导致伴侣分手的一个主要原因，抱怨则不是**。指责是消极且需要警惕的，但抱怨在特定条件下可能是积极且具有建设性的。

抱怨时，我们通常针对具体问题进行评论。例如：
*   “我们已经很久没有一起出游了。”
*   “你最近太忙了。”
*   “你在爸妈面前不够有主见。”

这些都是就事论事、对事不对人的抱怨，内容很可能反映了事实。两个人越亲密，越可能指出对方存在的问题。抱怨本身也包含着请求和需要，目的是希望对方注意到问题并改进。例如，当伴侣抱怨“你疏忽我了，你总是没时间”，其潜在含义是：“我希望你能靠近我一点，我希望你多抽出时间来陪我。”

以下是进行健康、有建设性抱怨需要注意的两点：

**第一，不要成为习惯性抱怨者。**
健康的抱怨针对具体问题，一旦问题解决或改善，抱怨就会停止。但习惯性抱怨者会在问题解决后仍不开心，并立即寻找新问题继续抱怨。这种抱怨不再有益，会对夫妻关系造成伤害。此外，总是就同一问题反复抱怨也需要注意。如果真正看重这段关系和婚姻，就会愿意照顾对方感受并做出调整。

**第二，对事不对人，警惕抱怨升级为指责。**
抱怨时，需时刻提醒自己对事不对人。将抱怨升级为指责，意味着对伴侣本人下负面判断，并利用该结论进行攻击。这是对人不对事的，包含轻视的情绪。指责方常自视为受害者，认为自己做得更好、付出更多，而对方有很多缺点。在一段有建设性的夫妻关系里，面对压力和危机时应互相关心、支持、保护，而非一味指责。指责会让双方进入敌对状态，与亲密关系的本质背道而驰。

美国心理学家阿特金森认为，在亲密关系中指责伴侣是一种隐性攻击，比公开辱骂、暴力更隐蔽，但危害同样巨大。阿特金森总结了多种与指责相关的隐性攻击行为，如不合理的挑剔、防御、草率得出负面结论、忽视、贬低、不愿让步、表现傲慢、逃避、为自己免责等。

当我们觉得自己是被伤害的一方时，值得思考：“我是否真的完全没有过错？”

## 如何应对抱怨或指责

抱怨或指责很可能导致争吵。与其让争吵消磨感情，不如学会用恰当方式面对这些负面评价。

**第一，放弃“是对方错了”的执念。**
对方抱怨时，不要着急否认。先将“我没有”这句话吞回肚子里。如果你的第一感觉是愤怒、否定、不承认，那么对方的抱怨很可能说中了你。你需要针对抱怨内容检查自己。在亲密关系中，不要因被对方看见缺点而感到羞耻。伴侣就像镜子，会如实反映真实的你。有些地方被看到可能让你不舒服，但确实能让你成为更好的人。

**第二，学会使用替代方案及转换看法。**
当伴侣犯错时，不要觉得既然是他犯了错，就应该被责备。尝试用建设性、健康的抱怨替代指责，再用冷静的分析和表达代替抱怨。需特别警惕翻旧账的行为。解决当下问题，然后让它过去，不要在下次争吵中再次提起。

如果你是承受指责的一方，试着转换看法：将对方的指责看作是他的抱怨。告诉自己，对方这样只是因为他爱你，希望能从你身上获得更多。然后，在情绪平静后与对方沟通，告诉对方：“我不喜欢你这样看待我。”

夫妻关系中没有绝对的赢家或输家。应将彼此看作队友、盟友，一起应对问题和未知。如果总是将对方看作竞争对手，就会陷入一场场你死我活的战斗，最终很可能两败俱伤。

## 高难度沟通的类型与误区

在亲密关系中，沟通并非易事，尤其涉及不满、被忽视的需求、关系困惑等敏感话题时更为艰难。一方面害怕陷入争吵，另一方面担心自己想得太多或需求得不到回应。不知如何开口，或开口后造成误解和争吵。这种状况在夫妻关系中几乎不可避免，每个人都会遇到并因此困扰。因此，无法轻松开展、表达时有障碍有困难的，都是高难度沟通场景。它们的确很难，但往往非常重要。一旦处理不好，很可能伤害彼此，甚至断送关系。因此，了解和学习高难度沟通技巧十分必要。

以下是夫妻关系中高难度沟通的五种类型：

**第一种类型：想解决某一个具体问题。**
当你在关系中发现不好的征兆、微妙变化或不好感受时，为了不让关系出问题，你想主动发起谈话和讨论，但对方却好像没意识到或看起来无所谓。此时的沟通有一定难度，尤其是那些不明显、尚未浮出水面、仅基于自己主观感受和预判的问题。例如：“我感觉你最近总是故意躲着我，是发生什么事情了吗？”“你好像不想让我接触你的朋友，这是为什么呢？”

**第二种类型：向另一半提出你的需求。**
虽然伴侣是最亲密、最熟悉的人，但直接了当地向他提出需求，对很多人来说并不容易开口，甚至比向父母、同学、同事、陌生人提需求都难。因为我们更害怕被重要、在乎的人拒绝。我们宁可等待对方主动发现需求，也不愿主动开口，因为害怕失望。例如：“我希望这个周末你能陪陪我。”“我希望你能帮我分担一下今天的洗碗工作。”“我希望你能跟我说说你最近的烦心事或开心事。”

**第三种类型：拒绝另一半看似合理的要求。**
拒绝伴侣的要求是高难度的事，尤其是当他的要求看起来非常合理，且对这件事充满渴望和期待时，但这件事你确实很为难，或大大超出了你的能力范围与心理舒适区。例如：拒绝高额彩礼、拒绝婚前同居、拒绝在时机不成熟时双方家长见面等。

**第四种类型：深度的自我曝光。**
两个人自我曝光的程度与关系发展息息相关。深度的自我曝光常包含一些负面、脆弱、创伤性、痛苦的内容和记忆。例如：过去复杂的情感经历、自己都不愿承认的内心阴暗面等。在夫妻关系中，我们想让对方了解全部真实的自己，但这个过程极其需要勇气。我们既害怕对方无法接受，又担心主动揭开伤疤会受到二次伤害。

**第五种类型：告诉对方坏消息或坏决定。**
坏消息如：“我被开除了”“我投资失败了”等。坏决定如：决定放弃稳定工作去创业等。将这些信息传达给自己最在意的人很艰难，因为我们最不愿伤害对方，也害怕对方得知后的反应。

在关系初期，一般不会出现高难度沟通，因为热恋会让人带上玫瑰色滤镜，也只展现自己光鲜、优秀、强大的一面。高难度沟通常出现在关系中后期，这时关系已进入较稳定状态。一开始，我们会刻意回避这些有难度的沟通，因为怕引发尴尬、觉得不好意思、生怕伤害对方或破坏关系等。但其实，这些沟通迟早都会发生，是一段长期、深度关系中不可避免的。完全回避高难度对话，只可能让夫妻关系最终走向毁灭。因为越是小心翼翼，不去面对深度的有难度的沟通挑战，就越容易在后面经受不起一点点风雨考验，被轻易击垮。

很多高难度沟通都伴随着不和与争吵，因为它常涉及核心重要话题，容易引起争吵。焦点常常是谁对谁错或争论实情，容易陷入四种沟通误区：

**第一种误区：假设对错。** “我是对的，你是错的，我说的才是真相。”当讲出自己观点并为之辩护时，我们经常不会去证实它是否真的站得住脚。我们的观点和情绪都建立在“我才是对的”这一假设上。我们会不自觉地从一开始就对对方的观点和感受保持偏见和怀疑，无法理解对方在同一问题上的不同感知、想法和价值观。

**第二种误区：假设动机。** “你就是想控制我”“你这样都是为了你自己”。在沟通中，我们常常想当然地认为自己一眼就看穿了对方的动机和意图，认为自己能洞察对方某句话、某个行为背后的真相。因为是最亲近的伴侣，所以我们笃定足够了解对方。然而，人的动机和意图是复杂、多重的，有时可能完全出于善意，甚至有时对方可能没有任何意图，不过是我们把自己的猜想强加给了对方。

**第三种误区：假设过失。** 当两个人在生活中出现过失，在沟通中，往往会将重点集中在“这是谁的责任、谁该来负责”这一问题上。追责的沟通过程会制造大量负面情绪，使人无法集中精力找到问题产生的根源及解决办法。

**第四种误区：情绪沟通。** 沟通包含对话，还包含更重要的部分——对话时的情绪。高难度沟通之所以难，因为它伴随着一些激烈的负面情绪，如痛哭、冷嘲热讽、怒不可遏、肢体伤害等。情绪会影响人的判断力，发泄情绪会同时伤害自己和对方。如果不处理好自己的情绪，或对对方的情绪没有感受力、接纳力，就没有任何沟通开展的可能。而且，情绪往往是矛盾的症结所在。

## 科学开展高难度沟通的方法

在夫妻关系中，如何科学开展高难度沟通？我们来聊一聊具体方法。

**第一，不要怕暴露脆弱面。** 简单来讲，就是说出藏在心里的话，展示出隐藏的那一部分真实的自我。在争吵中、严肃的对话里，不怕暴露自己的脆弱，也不怕暴露出两个人关系的脆弱之处。这样做一定会让人感到不安全感，因为人们害怕自己的真心得不到回应，担心对方会利用自己的脆弱来伤害自己。但是，心理学研究早就告诉我们，相互的、逐渐的暴露，是提高彼此吸引力、信任以及增进关系的重要一环。高难度沟通也要求双方愿意面对关系中脆弱敏感的部分。对存在的问题，不是视而不见、避而不谈、假装一切安好，而是敢于暴露关系的脆弱之处。这是两个人进行真诚、深刻交流的必要条件。它有助于解决矛盾和发展感情，也能让你用更客观的眼光看待关系，而不是一味过分理想化。例如，你渴望婚姻，而对方还没有结婚计划。除了这个分歧，你们平时非常快乐融洽。于是，你不愿提起结婚这件事，怕破坏当下美好，但也无法压抑自己想结婚的渴望。这个时候，主动讨论这个矛盾，就是暴露两个人关系脆弱的做法。在最坏情况下，它会导致关系破裂，但一直回避、欺骗自己，最后会让矛盾在更晚、更不可控的节点爆发出来。

**第二，沟通前提前想好要说的话和沟通目的。** 不让对话被情绪干扰，这非常重要，因为在进行高难度对话时，带着情绪沟通是最常见的误区。因此，我们需要提前准备好自己要说的话，大致罗列出你想和对方讨论的几个主要点。另外，还要想清楚你要达到的结果和目的，带着目标和需求去开展沟通。这样，在沟通过程中，一旦对话偏离目的，你就能迅速察觉，并把沟通带回正轨，避免陷入无谓的纠缠和争执。

**第三，沟通中的四要素。**
1.  **抚慰情绪。** 对话中的情绪比对话的内容甚至更重要。在沟通中，你要尤其注意抚慰对方和自己的情绪，防止谈话被情绪带跑。要知道这是你和爱人之间包含情感的对话，而不是一次商业谈判。因此，你要有足够的共情力，学会用语言和肢体动作去安慰对方，同时也要抚慰自己，缓解沟通中的负面情绪。
2.  **开放宽容。** 在沟通时，两个人之中至少要有一个人能够保持宽容开放的交流态度。不要带着过多的负面假设。先入为主和以自我为中心的误区尤其需要规避，它是真诚、有效沟通的最大障碍。
3.  **对事不对人。** 这是一个很简单的道理，但在实际沟通中却常常被人忘记。不要让一场本该就事论事的沟通变成互相伤害的人身攻击。你需要时刻记住，在沟通中针对具体的事情去说，应该更多地使用“我感觉…”“我觉得…”“在我看来…”这样的句式，而不是给对方下定义，比如说“你就是…”“你一定…”等。
4.  **先说积极面。** 先从一些积极正面的方面开头，而不是一上来就滔滔不绝地说自己的负面感受、对对方的意见、关系里的问题等。用一种更温和、也更容易被接受的方式开头。比如，你感觉最近对方太忙、被忽视了，可以先表达自己对他工作上的关心，为他取得的成绩而开心，再来说回你的感受。

**第四，沟通后一起复盘。** 在沟通结束一段时间后，情绪平静，你们需要从更理智客观的角度来看待沟通的内容和结果。总结这次沟通给你们关系带来的影响。选一个比较恰当的时机，你可以主动跟对方聊起上次沟通时你的一些感受和想法，并一起讨论它给你们关系带来的变化。当你们都认为这样的沟通即使困难，但的确能让你们的关系变得更好时，那么下一次面对这样的高难度沟通，就没有那么困难了。

## 课程总结与课后作业

本节课中我们一起学习了如何在婚姻中进行健康的抱怨，以及如何应对高难度沟通。

我们来回顾一下今天的内容。婚姻中的抱怨是不可避免的，抱怨在一定的条件下是积极而具有建设性的，但我们要谨防抱怨演变成指责。不健康的抱怨常常会破坏夫妻关系，留下隐患，我们学习了如何健康抱怨的技巧。接着，我们还学习了如何搞定高难度沟通的技巧，用来应对婚姻中的挑战。这些沟通亲密术，你都可以运用到婚姻生活中。

最后，给大家布置一项课后作业。毕竟所有的改变都要学以致用、有所行动，才能取得最后的效果。

**课后作业：**
选择一个高难度沟通场景，向伴侣提出自己的需求，或者拒绝伴侣看似合理的要求。运用课程中的方法，和伴侣进行一场深入的沟通演练，看看结果如何。
