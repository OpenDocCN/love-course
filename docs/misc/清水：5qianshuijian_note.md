在本节课中，我们将要学习一种经典且基础的性爱体位——正常位，也被称为传教士体位。我们将详细解析其定义、特点、适用人群以及具体的操作方法和注意事项。

## 概述 🧐

正常位是男性在上、女性在下的面对面性交姿势。它是许多伴侣开始性交时最常采用的基础体位。

上一节我们介绍了课程概述，本节中我们来看看正常位的具体定义和核心特点。
