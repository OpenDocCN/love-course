## 吸引力是人性本能

被吸引是人性的本能。例如，在街上看到身材出众的人，会忍不住多看两眼。这就是吸引力，它不受理性完全控制。

吸引力本质上是两种价值的结合。

## 吸引力的两大核心价值

吸引力由两种核心价值构成，它们源自人类的生存本能，并延续至今。

以下是构成吸引力的两种核心价值：

1.  **生存价值（S价值）**
    *   定义：保障个体生存与繁衍的能力。
    *   原始社会体现：狩猎能力、获取食物、拥有安全住所。
    *   当代社会体现：财富、地位、提供照顾与保护的能力、稳定的性格、抚养后代的意愿。

2.  **繁衍价值（R价值）**
    *   定义：传递优良基因的潜力和特征。
    *   主要体现：外貌、健康度、生育能力的信号。
    *   常见例子：女性青睐的“八块腹肌”代表健康；男性青睐的丰满体型代表优秀的生育与哺乳能力。公式可简化为：`吸引力 = 生存价值(S) + 繁衍价值(R)`。

我们的祖先正是凭借成功应对“生存”与“繁衍”这两件大事，才得以延续基因。未能完成者则被自然淘汰。当代人类的行为，依然围绕着这两大核心。
