## 1. 保持独立，避免过度围绕女生转 🔄

上一节我们介绍了吸引力的核心并非外在条件，本节中我们来看看具体的行为模式。首先，不要整天围着女生转。有些男生追求女生时，恨不得24小时都粘着对方，嘘寒问暖，刻意迎合。但这些行为会让女生潜意识里形成“你配不上她”的想法。

为什么很多男生被当成备胎？正是因为心态已被女生拿捏。她知道她在你心中地位很高，即便一整天不搭理你，第二天你仍会主动联系她。反观你，想快点确定关系，女生却对你爱答不理。在这个等待过程中，你会变得越来越焦虑和不自信，姿态越来越低，最终可能选择破釜沉舟地表白，结果往往是被无情拒绝。

**核心公式**：`吸引力 = 自我价值 - 需求感`

记住，你自己才是最重要的。不要总是围着女生转，这种行为只会让女生觉得你无所事事，更让她觉得你非常需要她。现在你明白女生为什么不理你了吗？

所以，我们要给女生一种若即若离的感觉。这样你不仅不会被拿捏，甚至她会主动来找你。

## 2. 提供情绪价值，而不仅仅是正向情绪 💖

上一节我们讨论了保持独立的重要性，本节中我们来看看如何通过情绪互动来增强吸引力。女生喜欢你，一定是因为你给她带来了某种“感觉”。经常有男生被女生拒绝的理由是“我对你没感觉”。那么，女生所说的“感觉”到底是什么？

正向情绪是让女生开心，负面情绪则是让她难过。适当的夸赞确实可以带来情绪价值，但如果你一直在夸，就会沦为“舔狗”。你不是不会给女生提供情绪，说白了，你就是不敢，因为你把女生看得太重要了。

包括有男生经常问我：“女朋友动不动就发脾气、不理我，甚至提分手。”其实归根结底，就是因为上述原因。所以，你自己的情绪一定不要被女生带走。

上一节我们探讨了情绪价值的平衡，本节中我们来看看长期关系中的关键：聚焦思维。这里的“危机感”不仅指你，也要让女生时刻保持危机感。很多男生在追到喜欢的女生后，就开始暴露本性，不再注重个人形象，把自己最真实、最差劲的一面毫无保留地展示出来。
