# 课程07：社交沟通（一门沟通的艺术） - 05.学会正确倾听他人，让你的社交关系变得更近！👂

在本节课中，我们将要学习社交沟通中一项至关重要的技能：**正确倾听**。倾听并非被动地听别人说话，而是一门主动理解他人感受与需求的沟通艺术。掌握它，能让你与他人的关系变得更加亲近和稳固。

## 概述

上一节我们介绍了非暴力沟通的基础模型。本节中，我们来看看如何将这个模型的核心要素——**观察、感受、需要和请求**——应用于倾听之中。学会正确倾听，意味着你能在他人倾诉时，真正理解其内心世界，而非急于给出建议或安慰。

## 倾听为何如此重要？

在社交过程中，倾听极其重要，但极少有人能做到正确倾听。例如，当朋友或伴侣向你倾诉伤心事时，错误的倾听方式（如急于安慰或分析问题）往往无法缓解对方的痛苦，甚至会让对方觉得“你不懂我”。

正确的倾听则能让他人感到被理解和尊重，从而拉近彼此的关系。无论是朋友、家人还是伴侣，当他们感到被你真正倾听时，信任感便会增强。

## 常见的错误倾听方式

以下是人们在倾听时常犯的错误，这些行为实际上阻碍了我们体会他人的真实感受：

*   **急于给出建议**：“我觉得你应该...”
*   **进行比较**：“你这不算什么，听听我的经历...”
*   **进行说教**：“如果你当初...，现在就不会这样了。”
*   **盲目安慰**：“别难过了，这不是你的错。”
*   **回忆自身经历**：“我当年也遇到过类似的事...”
*   **否定对方感受**：“高兴一点嘛，别这么难过。”
*   **表示同情**：“哎呀，你真可怜。”
*   **不断询问细节**：“这事是什么时候开始的？”
*   **为自己辩解**：“我本来想早点联系你，但我太忙了。”
*   **纠正对方**：“事情的经过不是你说的那样。”

请反思一下，当别人向你倾诉时，你的第一反应是否属于以上行为？这些行为的共同点是：**你不再是在倾听，而是在急于表达自己的观点、判断或感受**。这完全背离了倾听者的角色。

## 如何做到正确倾听？三步核心法

要成为一名真正的倾听者，你需要暂时放下自己的想法和判断，全身心地关注对方。以下是三个核心步骤：

### 第一步：体会他人的感受和需要

无论对方以何种方式表达，我们都要努力去体会其话语背后的**观察、感受、需要和请求**。

**举例**：家人责备你：“你怎么能把车借给一个刚认识的邻居？你真笨！”
*   **家人的观察**：你把车借给了不熟悉的人。
*   **家人的感受**：担心、焦虑、可能觉得不受尊重。
*   **家人的需要**：安全感、被尊重、家庭财产得到妥善处理。
*   **家人的请求**（可能）：希望下次再做类似决定前能和他商量。

通过这样“翻译”，我们就能理解对方生气背后的深层原因，而非停留在表面的指责上。有时，对方可能直接给出结论（如“小王没有团队精神”），这时我们可以通过提问来了解其观察：“是因为小王没有参与哪项工作，让你这样觉得吗？”以此引导对方说出感受和需要。

### 第二步：给予对方反馈

倾听不是沉默。在理解了对方的感受和需要后，我们应该主动给予反馈，表达我们的理解。这能让倾诉者感到自己被听懂了。

反馈可以是简单的复述或确认：
*   “听起来，你是因为……而感到……，是吗？”
*   “所以你需要的是……，我理解得对吗？”

如果我们的理解有偏差，对方可以纠正我们；如果理解正确，对方会感到极大的被理解和支持。很多时候，倾诉者并不需要建议，只需要有人聆听并理解他们。

### 第三步：保持关注

在对方倾诉时，要保持全程关注，为对方创造充分表达的空间。不要过早地提及对方的请求或给出解决方案，这可能会被视作打断或给出建议。

更深层的情感往往在倾诉后期才会流露出来。只有当我们持续保持关注，让对方确信我们在认真聆听时，他们才愿意表达更真实的感受。

## 特殊情况：当你无法倾听时

我们自己也会有痛苦、疲惫、无心他顾的时候。**当你自己处于情绪低谷时，你无法有效地倾听别人**。

这时，正确的做法是坦诚地表达自己的感受和需要，而不是强迫自己去倾听或指责对方不关心自己。

**错误表达**：“你们吵什么吵！我累了一天了！”
**正确表达**：“我现在感觉很累，需要安静休息一下，暂时没办法处理你们的事情。”（**公式：感受 + 需要**）

这样表达后，对方反而可能来关心你。在亲密关系中也是如此，多表达自己的感受和需要，而非指责对方。

## 倾听的力量与总结

本节课我们一起学习了正确倾听他人的方法。**倾听是一种强大的治愈力量**，它能释放倾诉者的消极情绪，让其感到被尊重和理解。

总结正确倾听的三个要点：
1.  **体会感受与需要**：放下自己的判断，专注于理解对方的观察、感受、需要和请求。
2.  **给予积极反馈**：通过复述或提问，让对方知道你理解了他。
3.  **保持全程关注**：不打断，不急于给方案，让对方把话说完。

越是能用心倾听别人，你也越容易获得别人的倾听。掌握这项技能，将极大地改善你的人际关系、亲密关系和家庭氛围。

语言可以是桥梁，也可以是高墙。而正确的倾听，是建造桥梁最稳固的基石。
