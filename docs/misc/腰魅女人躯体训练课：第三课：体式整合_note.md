## 姿势一：青蛙式

上一节我们介绍了基础的身体准备，本节中我们来看看第一个整合体式：青蛙式。这个体式模仿青蛙的姿势，旨在打开髋部并增强腰胯区域的灵活性。

我们从猫牛式的基础姿势开始。双手与肩同宽支撑地面。

以下是动作步骤：
1.  将双脚转向内侧，使双脚掌相对并尽量靠近身体。
2.  尝试让臀部向地面下沉，进入青蛙式。如果无法坐下，保持在你能舒适达到的位置即可，不要勉强。
3.  坐下后，尽量将手臂向前伸展。
4.  低头，吸气时将腰胯向前、向上推送身体。
5.  呼气时，缓慢地将身体收回，感受臀部和胯部向后推起、向上抬离的感觉。

**动作原理**：吸气时，腰胯向前向上运动；呼气时，身体缓慢回正。这过程与第一课的核心原理一致。

这个基础动作看似简单，但非常考验髋部的灵活性。因此，在熟练掌握了前两个姿势之后再来练习，你会感觉更加轻松。

在动作中，只需运用呼吸去感受。动作要缓慢，吸气时感受腰胯被带动向前向上，呼气时慢慢收回，去体会身体向后移动的感觉，这有助于将臀部和胯部的力量集中向后推。

## 姿势二：波浪旋转式

如果你的第一个姿势已经练习得不错，接下来的这个动作会更具挑战性。它将在青蛙式的基础上，加入胯部的旋转，形成连贯的能量流动。

同样从猫牛式开始，先变换到青蛙式的脚部姿势。

以下是动作的分解讲解：
1.  完成一次吸气向前、呼气收回后，在身体再次向前时，不要直接收回。
2.  将胯部向左方送出。
3.  接着将胯部向后移动。
4.  然后将胯部向右方送出。
5.  最后将胯部向前移动，回到起始位置。

我们将这些细节连贯起来：吸气准备，呼气时，胯部依次完成 **左 -> 后 -> 右 -> 前** 的移动，最后收回。

细心观察会发现，老师在吸气时，身体会自然做出一个波浪形（wave）。这也在锻炼你手臂的支撑力量。如果平时有运动习惯，你会更容易撑起身体。

**关键要点**：整个过程不要憋气，配合呼吸进行。如果熟练，可以将动作连贯起来。如果无法完美顾及每个方向，不必强求，重点是感受身体的灵动性，让身体去它想去的地方。这个框架旨在帮助你缓慢、安全地练习。

头部保持放松，可以自然低下或随身体旋转，避免僵硬。

当你找到自己的节奏，完全放松头部和身体，跟随能量的流动去旋转时，可能会进入一种惯性状态，感受到能量在体内循环。练习时间可以从10-20分钟开始，一旦掌握并进入状态，可以享受40-60分钟练习带来的能量充盈感。
