# 婚外情治理（开战篇）第二课：剖析第三者心理 🧠

在本节课中，我们将深入探讨第三者的心理世界。了解“小三”的心理动机、成长背景及其在婚外情关系中的核心需求，是有效应对婚外情的第一步。我们将从原生家庭、个人成长、心理状态等多个维度进行解析，帮助你掌握其心理软肋。

## 课程概述

很多人会困惑：第三者究竟是怎么想的？她们为何要介入他人的婚姻？自2011年开始婚姻辅导工作以来，我发现来访者最常提出的问题就是：“什么样的人会成为小三？”、“朱老师，请帮我分析一下我们家的小三是什么样的人。”

在辅导过程中，对第三者进行分析是必要的。这些分析有的来自只言片语的线索，有的则来自我与第三者本人的直接访谈。我接触过不少主动寻求帮助的第三者，她们最初希望我能辅导她们“逆袭”成功。然而，这类第三者往往缺乏足够的心理能量去达成目标，因为她们与已婚男人的关系本身也充满了矛盾与冲突。因此，我的辅导常常转向帮助她们离开这段关系，离开那个已婚的男朋友。正是通过这些面对面的深入接触和验证，我积累了大量真实、鲜活的案例。

在开始之前，请允许我做一个简短的自我介绍。我是朱身勇，复旦大学MBA，婚姻管理导师，国家一级婚姻家庭咨询师，国家二级心理咨询师，也是婚姻管理学派的创始人。

## 第三者的心理土壤：原生家庭与成长环境

上一节我们探讨了丈夫出轨的心理动力，其核心在于婚姻内部存在“匮乏”与“不满足”。简单来说，就是在婚姻中，夫妻双方的心理需求（如渴望、被疗愈的需求）未能得到满足，或者在互动中积累了大量的不满与不适。

那么，第三者是在什么样的环境中成长起来的呢？掌握这个特点至关重要，因为它能帮助你理解第三者的心理能量水平，以及她究竟从你丈夫那里获得了什么。很多人容易忽视这一点。

### 原生家庭的深远影响

原生家庭在很大程度上决定了一个人的潜意识，即那些我们自身不易察觉的部分。它塑造了我们的底层人格和心理动力。在心理咨询中，我们强调从“认知”层面进行调整，因为认知决定了行为。调整好认知，后续的行为改变和习惯养成就会容易得多。

那么，什么样的家庭容易“培养”出第三者呢？根据大量案例，第三者的原生家庭通常具有以下特征：

**1. 父母婚姻不幸福**
*   父母可能长期分居、离婚，或其中一方有出轨行为。
*   父母经常吵架，给子女营造了糟糕的婚姻和夫妻关系范本。
*   父母角色塑造失败。例如，第三者的父亲常常是不负责任、不爱妻子、也不爱孩子的形象。这种“爱”可能只是孩子回忆中的错觉，深入内心则会发现并非如此。

**2. 父母教养方式的问题**
父母吵架会给孩子带来恐惧、焦虑和不安，导致其安全感缺失。孩子可能会自我否定，认为是自己不够好才导致父母不相爱。他们常常需要去讨好父母双方，承担本不该由他们承担的责任。
*   **父母冷战**：家庭氛围冰冷，夫妻互不关心。这种环境下的孩子会极度渴望温暖、阳光的关怀，容易被“温暖大叔”类型的人吸引。
*   **父母一方出轨**：这会让孩子认为婚姻不重要，婚姻只是为了满足个人快乐，无需顾及他人。因此，许多第三者也有类似特征，她们往往只关注自身感受。
*   **父母分居**：这会让孩子感到婚姻是残缺的。跟随的一方（通常是母亲）可能会向孩子抱怨另一方，这容易使孩子对完整幸福的家庭产生扭曲的渴望，甚至产生“拆散”或“加入”的念头。

### 教养方式的具体伤害

父母的教养方式直接影响孩子的自我价值感。
*   **否定与指责**：孩子做得好也得不到认可，反而被贬低（如“女孩子读书没用”）。这会导致孩子自我价值感低下。
*   **忽略**：孩子在家中完全被忽视，感觉自己是透明的。成年后，一旦有人给予些许关注（如打招呼、带早餐），就容易产生情感依赖。
*   **家暴**：无论是身体还是语言暴力，都会让孩子迫切想要逃离原生家庭。她们可能草率地进入一段关系，但若再次遭遇伤害，则可能转向寻找那些“对伴侣好”的男人，并试图占有这种“好”。

**总结来说，第三者的心理能量普遍较低。** 她们极度需要来自你丈夫的关心、关注和认可。这是因为在成长过程中，她们长期遭受忽略、否定、指责，甚至被抛弃。

## 第三者的个人成长与生存状态

除了原生家庭，个人成长经历也会塑造第三者的特质。

### 个人成长创伤
*   **被排挤**：可能因为发育早、个子高或成绩好而被同龄人排挤。
*   **性侵害**：曾遭受性骚扰或性侵的经历，可能使其对两性关系产生扭曲认知。
*   **恋爱挫折**：在恋爱中屡屡受挫，无法建立健康的亲密关系。其中有一类特殊的第三者：前女友或前妻，因为存在未解决的情感纠葛。

### 易于成为第三者的生存状态
当一个人处于以下心理能量较低的阶段时，更容易卷入婚外情：
1.  **情感困扰期**：自身的婚姻或感情出现危机（如夫妻吵架、无性婚姻、丈夫冷漠、婆媳矛盾、过度劳累）。甚至有些异地婚姻的妻子也可能成为别人的第三者。
2.  **情感空窗期**：单身或刚刚失恋，需要情感慰藉、认可和支持。
3.  **事业瓶颈期**：事业遇到平静或发展受阻，尤其是职场新人，渴望有人支持与指引。
4.  **身体健康出问题期**：身体患病时，人会变得脆弱，更需寻求帮助。
5.  **心理低潮期**：处于焦虑、抑郁、情绪低迷状态时。

### 需要特别说明的一类第三者
有一类女性，她们原生家庭良好，个人成长优秀，看似“什么都很好”。她们也可能成为第三者，但通常是在“被小三”（不知对方已婚）的情况下。一旦发现真相，她们有足够的心理能量迅速离开。因此，对于这类第三者，原配无需过分担心。

## 第三者从婚外情中获得了什么？

理解了第三者的心理基础，我们再来看看她在这段关系中具体获得了哪些满足。她之所以离不开，正是因为获得了这些：

1.  **认可与尊重**：你丈夫给予她极高的评价（如温柔、懂事），这种具象化的认可和尊重，可能正是她在原生家庭中极度匮乏的。
2.  **关心与关注**：她享受“女王”般的待遇，一颦一笑都能牵动你丈夫的神经。
3.  **“爱”的感觉**：无论这份“爱”真实与否，你丈夫会向她表达爱意，让她感受到被爱。
4.  **激情与恋爱感**：婚外情提供了新鲜、刺激的高峰体验。
5.  **“家”的感觉**：你丈夫可能承诺“一辈子做情人”，或在外面营造一个像“家”一样的空间，甚至生育孩子，这满足了她对完整家庭关系的渴望。
6.  **实际利益**：金钱、人脉、事业上的帮助等。
7.  **安全型依恋与承诺**：尽管可能伴有焦虑，但她觉得这段关系有盼头，获得了（哪怕是口头的）长期承诺。

## 婚外情中双方的心理动态

### 第三者的普遍心态
*   **理直气壮者极少**：只有约5%-10%的第三者毫无愧疚。
*   **道德审判普遍存在**：大多数第三者内心知道自己在破坏他人家庭，会有道德压力。
*   **婚姻诉求的起源**：起初，很多第三者并没有上位的打算。但约70%甚至更多的出轨男人会主动或被动地给予她们婚姻承诺，这催生了第三者的期待。
*   **相互守贞**：在激情期，双方可能形成一种“相互守贞”的状态，排斥其他性关系。
*   **激情期漫长**：婚外情的激情期可以非常长（甚至长达十年），因为“偷”的状态带来了持续的新鲜感和刺激感。
*   **幻觉与退缩**：双方都容易产生“对方是真爱”的幻觉。但当第三者提出婚姻要求时，男人往往会退缩。若第三者因此后撤，男人为了维持高峰体验，可能会向原配提出离婚，以此作为挽回第三者的手段。**请注意，此时提离婚往往是一种施压手段，而非真正决心离婚。**
*   **沉默成本与纠缠**：随着时间推移，第三者投入的沉默成本越大，越难离开。双方可能陷入纠缠、撕扯的状态。
*   **脆弱与贪婪**：第三者的心理基础通常是脆弱的，她的心理创伤恰好被你丈夫部分“疗愈”，导致难以分离。同时，她也存在贪婪的一面，渴望持续获取这些满足。

### 出轨丈夫的心理状态
*   **心理压力**：害怕被发现、对家庭有愧疚感、经济和社会压力、焦虑东窗事发。
*   **理性与感性的矛盾**：理性上，他可能觉得婚姻不幸，小三完美，因而想离婚；但潜意识里，他可能并未完全否定原配。如果潜意识里彻底否定，离婚态度会非常坚决。
*   **责任感的扭曲**：他可能觉得要对第三者“负责”，从而向原配提出离婚或“一夫二妻”的想法。
*   **“三人行”心理**：如果原配态度软弱，他便会倾向于维持三人行的局面。
*   **“说鬼话”心理**：他对两边说不同的话，这些话部分真实（反映他的感受），部分是为了满足双方当下的需求（如对原配说“有亲情”，对小三说“你是真爱”），让双方都愿意留在关系中。
*   **外强中干**：表面上可能强势，内心实则脆弱。若真的强大，早在出轨前就会处理婚姻问题。

## 关于辅导与应对的核心观点

1.  **第三者也是受害者**：在婚外情系统中，第三者同样受到伤害。理解这一点，原配才能以更有效、更不被情绪绑架的方式去处理问题。
2.  **分离需要过程**：婚外情中的情感依恋和高峰体验很难瞬间切断。丈夫反复联系第三者，可能因为依恋未断、体验诱惑、原配原谅后压力减小，或心存侥幸。
3.  **单纯原谅无效**：原谅不能作为交易（“你回来我就原谅”）。真正的修复需要共同面对婚姻问题，解决矛盾，重建信任，创造新的婚姻共同体。这需要专业的辅导和长期的努力。
4.  **“宿命论”是借口**：男人用“命中注定”来解释出轨，是为了推卸责任，回避处理婚姻本身的问题。
5.  **及时干预**：发现婚外情应尽早寻求专业辅导，而不是等待。辅导的重点不仅是分离第三者，更是梳理和修复婚姻内部的问题（沟通模式、冲突解决、需求满足等）。

## 课程总结

本节课我们一起深入学习了第三者的心理世界。我们了解到，第三者的形成往往根植于不幸的原生家庭和有问题的教养方式，导致其心理能量低、自我价值感弱。她们在个人成长的低谷期（情感、事业、健康出问题时）更容易卷入婚外情。在这段关系中，她们寻求的是认可、尊重、关心、爱、激情、家的感觉以及实际利益。

同时，我们也分析了婚外情中丈夫和第三者复杂的心理动态，包括漫长的激情期、相互守贞、以离婚为手段施压、以及双方外强中干的脆弱本质。理解这些心理，是原配保持清醒、采取正确策略的基础。

记住，有效的婚外情治理不是简单的驱逐或原谅，而是一个系统工程，需要专业指导，从修复婚姻本身入手。
