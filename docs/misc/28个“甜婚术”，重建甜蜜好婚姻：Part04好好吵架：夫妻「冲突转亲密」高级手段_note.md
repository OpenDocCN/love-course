# 课程28：重建甜蜜婚姻 Part04 🗣️ 好好吵架：夫妻「冲突转亲密」高级手段

在本节课中，我们将学习如何将夫妻间的冲突，特别是争吵，转化为增进了解、拉近关系的契机。我们将识别破坏关系的争吵类型，并掌握一系列建设性的应对技巧。

---

在亲密关系中，有大量的冲突正在发生，而且无法避免。心理学家戴维·巴斯的一项研究告诉我们，恋爱期的伴侣发生冲突的频率达到了平均每周两到三次。而另一项为期三年的研究中发现，夫妻每个月都会经历一到两次不愉快的争吵。

那么，我们该如何看待和应对冲突呢？有没有可能将冲突转化为推进关系、重建关系的契机呢？这节课，我们就来讲一讲夫妻吵架这个典型的冲突。

有一种观点认为，如果你想要甜蜜的爱情或婚姻，就要做好准备迎接适当的争吵。因为，有一些争吵是值得的，是夫妻之间甜蜜互动、体验相爱、增强信任的秘密武器。只是，遗憾的是，很多人，包括专业人士，都无法掌握好吵架的技巧。

这些技巧并不是教你如何尽快稀释凝人、如何取胜，而是让你和老公能清楚地意识到为何争吵、该如何争吵，以及争吵的目的是什么。在唇腔舌尖交锋之际，能够不错失深入了解自己、了解彼此、拉近关系的良机。

## 一、识别破坏关系的争吵类型

夫妻有效争吵的前提是，需要认清哪些吵架类型会破坏夫妻关系。在具体的场景中，我们能够以此迅速地辨别，更好地应对。

那么，什么吵架类型会破坏关系呢？主要有以下六种。

以下是六种破坏性争吵类型及其应对技巧：

1.  **谁对谁错型**
    这种争吵的焦点始终是到底谁对谁错这种责任归属的问题。不管引起冲突事件的本身，是大是小，两个人争吵的目的就是找替罪羊，而不是搞清楚为什么会出问题。找替罪羊是一种报复行为。关系一旦陷入这种推卸责任的怪圈，两个人的不满感就会携来，既不能真正解决问题，也不会带来任何改变。
    *   **应对技巧**：切记在争吵中推卸责任，而应该仔细地思考以下这些问题：你到底为什么在烦恼？到底是什么地方出了问题？怎么改变现在的情况？等等。

2.  **纠结琐事型**
    这一类争吵涵盖了一系列家庭纠纷，比如，谁做饭，谁洗碗，谁接孩子，谁洗衣服，马桶圈是不是该放下来等等。这种争吵一般体现了家庭中责任分配的不公，或者有一方忽视了另一方为家庭所做的贡献。这背后其实是关于家庭权力的斗争。
    *   **应对技巧**：如果你只是反复纠结，吵个没完，而不是真正地去解决问题，那么这种争吵永远都不会结束。你可以集中列出在家庭责任分配方面的不满，邀请老公一起探讨解决问题。坦白说出你的感受、需求和渴望，以免小事积累爆发出更大的矛盾，成为侵蚀你们关系的隐患。

3.  **有话不说型**
    这种类型的常见表现就是生闷气，沉默不语，独自走开，或者即使说也不会说真心话。这样做，你们的关系永远无法生化，两个人无法真正走近。在亲密关系中，“我想一个人静静”这种冲突是很致命的，因为不用出声就可以让对方感受到你的敌意。
    *   **应对技巧**：找到交流的重点，直接面对它，开诚布公地把你的压抑说清楚。

4.  **“如果你真的爱我”型**
    这种类型通常打出“如果你真的爱我”这张牌。例如：“如果你真的爱我，你就会戒烟。”“如果你真的爱我，你就会少玩游戏。”“如果你真的爱我，你就会知道送我什么礼物。”发出这种对话的人觉得老公应该会读心术，无条件的服从自己，否则对方的爱就是虚伪的。
    *   **应对技巧**：找到自己或对方为什么会经常打出这张牌的真实原因。可能你们对爱有着不同的观点，直接面对这些差异，有助于你们克服关系中的核心障碍。

5.  **“我受不了你”型**
    例如：“我受不了你这样穿衣服。”“我受不了你叭唧嘴。”“我受不了你这样走路。”连老公的简单的生活习惯也会让你抓狂，甚至演变成“我看见你就烦”。为什么他们会让你感到如此痛苦，无法忍受？这些明明在恋爱初期并不成为你的困扰。在亲密关系中，确实有些事情正在困扰着你。你对老公感到不满，但原因并不是那些让你感到受不了的事情，而是一些更深层次的问题。
    *   **应对技巧**：找到让你真正感到不满的问题核心，就核心问题和对方沟通。比这样对任何事情都看不惯抱怨的方式好得多。

6.  **“你跟你爸/你妈一个样”型**
    这种类型的争吵会戳到老公的痛处，因为长期以来，他一直害怕成为父母那样的人。而你却始终抓住这一点不放。他无意识地进行反驳，争吵陷入“你是不是真的很像你父母”这样的讨论中，毫无疑义。实际上，也许你只是担心他的某种像父母的行为可能会破坏你们的关系。而他之所以在某些方面表现得像父母，可能是他想告诉你他的生命中缺失了什么东西。
    *   **应对技巧**：可以把这种争吵当作一种非常好的滤镜，通过它来看清过往，看看他们是如何在影响你们的关系，以及你能如何改变这种情况。

一般而言，在形成一种应对冲突的习惯后，这种习惯会持续下去。但是，研究也证明，即使是习惯破坏型消极型应对方式的人，也依然可以通过学习和练习去改变自己的应对方式。

上一节我们介绍了需要避开的六种破坏性争吵类型，本节中我们来看看如何有建设性的、有技巧的应对吵架的冲突，将它转化为夫妻之间重建亲密的契机。

## 二、积极争吵：七个技巧将冲突转为亲密

以下是七个将冲突转化为亲密的实用技巧：

**技巧一：解读渴望，读懂核心诉求**

我们吵架的根源和核心的诉求是什么？是我们心中的渴望。我们渴望的并不是垃圾有没有被扔，衣服有没有人洗，这些琐碎的事情本身，而是这些事情背后所代表的东西，代表我们对被重视、被尊重、被理解、被倾听、被珍惜、被爱的渴望。这些渴望在我们的意识深处，影响着我们的一言一行、一举一动，让我们去爱，也引发我们争吵。

因此，每一次争吵，如果能洞察到它的核心——那一份没有得到满足的渴望。那么，一旦我们采取相应的措施，让渴望得到表达、承认、引导和满足，就会转变为夫妻之间的亲密感、信任感、力量感，为关系带来快乐和心甘情愿的奉献，也让两个人成长为更好的自己。

*   **核心操作**：`读懂冲突中隐藏的渴望（自己的 + 伴侣的） -> 让渴望浮出水面 -> 协商解决`
*   **常见渴望**：获得安全感、表达真实的自己、互相重视、互相了解并产生共鸣、感到自己有影响力、实现人生价值、发展人际关系、建立信任、去爱和被爱、被欣赏等等。
*   **效果**：对吵架行为进行剖析，发现彼此深层的渴望，并充分表达、给予满足，有助于你们改变自己的行为，并落实出具体的结果。这是一个学习、思考和成长的过程。在这个过程中，你们收获的不只是快乐和幸福。

**技巧二：积极互动、重建甜蜜的法宝**

一旦你读懂了自己的渴望，你就要为实现渴望而采取必要的行动，即使这会带来风险。在与对方交战的过程中，你要全心投入真实地展现自己，表达出你真实的感受和情感，提出你的渴望，以及展现肯定、喜爱、幽默等积极面。当你学会了这些，冲突就不是什么大不了的事了。相反，这样的冲突会变得易于接受和解决，还会给亲密关系带来回报。

从某种意义上来说，争吵是一种夫妻之间的高效率互动。美国心理学教授被誉为“婚姻教皇”的约翰·戈特曼做过一项研究，研究指出，幸福的夫妻在出现争吵时，积极互动和消极互动的比例是 **5:1**，而在离婚的夫妻中，这个比例是 **0.8:1**。还有一些感情非常稳固的夫妻，他们的这个比例达到了 **20:1**。

*   **核心公式**：`积极互动 : 消极互动 >= 5 : 1`（幸福底线）；`>= 20 : 1`（甜蜜目标）
*   **含义**：如果你想和老公成为一对幸福的夫妻，那么当你做出一个消极互动时，就需要做5个积极互动来抵消和弥补。这就意味着，在争吵中，你要特别注意不要胡乱地发出刻薄的有害的攻击，否则，你将要为此付出5倍的积极补偿，才能平衡好你们的关系。而如果你想要和老公甜甜蜜蜜，那么这个倍数还要从5倍提升到20倍。如果消极互动太多，而积极互动不够，两个人就会走上离婚的法庭。
*   **积极行动**：分享渴望，表达情感和感受，欢迎对方的靠近，做出积极的回应（比如展现幽默，表达喜悦，兴趣，认可等等）。让天平多向积极的一面倾斜，就能持续带来更好的亲密关系。

**技巧三：三个“不要”，尽可能地弱化消极面**

像前面所说的，每一次消极互动都需要至少5倍的积极互动才能抵消和弥补。那么，尽可能地弱化消极面也是重要的策略之一。“婚姻教皇”约翰·戈特曼就建议，在应对冲突时要做到三个不要。

以下是三个需要避免的行为：

1.  **不要退避**：当你和老公陷入争端时，不要试图逃避，也不要过于防备。戒心和逃避无法解决问题，反而会让两个人更加不安。如果你此时不方便沟通，或是需要时间思考，你可以直接告诉对方自己的难处，请求他重新安排时间来讨论冲突。
2.  **不要负面回应**：即使老公用恶劣的态度对你，你也要尽量地控制自己，不要用讥讽、挑衅、表达轻蔑等负面方式应对冲突。一方面，负面回应容易激化矛盾。另一方面，老公会更容易记住自己做过的负面的事。一旦造成伤害，两个人都要花更多的时间和精力去弥补。
3.  **不要来回地传递负面情绪**：假如你不小心进行了负面的回应，请及时地停止，否则有可能演变成一场负面情绪的“乒乓球赛”。当你意识到你们正在你来我往的侮辱和伤害对方，互相辱骂和攻击时，立刻喊停。可以举起双手要求休息：“我们停15分钟再来谈。”等两个人都平静下来之后，再进行对话，并且为之前的过激的言辞道歉。谁都有失控的时候，但事后愿意道歉体现了一种愿意承认和承担责任的成熟。

**技巧四：预设对方是出于好意**

这一点并不容易做到。你需要学习如何才能预设对方出于好意的方法，并让这种预设成为一种习惯。研究表明，当我们总想着对方不好的一面时，就会忽视对方好的一面，甚至认为他从来没有做过一件好事。用正面的方式看待伴侣，最有效的方法是将伴侣视为朋友而非敌人，意识到他是给予你安慰、支持和安全感的主要来源。这样能够抑制你的警戒心，停止对他的不合理的审视，为理解和共情提供充分的空间。

预设对方出于好意，还能给关系带来正向循环。当你这样做时，双方对关系的满意度都会提升。而相关的研究表明，冲突发生时，对关系更满意的人，又会更倾向于对冲突进行积极归因。他们会对伴侣的行为做出更积极的解释，不认为对方是出于恶意的目的。

**技巧五：明确争吵目的，在冲突中实现双赢**

争吵要有明确的目标，最糟糕的争吵漫无目的，充斥着习惯性的重复、唠叨、辩解、逃避和控制行为。争吵的目的是你要争取某些东西，而不是一味的坚持自己，反对伴侣。很多时候，我们为一些蠢事而争吵，只是出于自卫。当我们带着明确的目标争吵时，我们会碰撞出从未有过的解决方案，在冲突中实现双赢的结果。

*   **核心问题**：`停下来问问自己：“我究竟为什么而争吵？争吵的目的是什么？”`
*   **行动**：找到那个关键目标，并围绕它展开讨论。

**技巧六：50%规则法**

双方各自最多对争吵承担50%的责任，所谓一个巴掌拍不响。不管是谁挑起的争吵，你们都是这个整体的一部分，都负有责任。当你们在互相推卸责任的时候，记得提醒自己，你们各自需要承担的责任最高不超过50%。

*   **核心规则**：`我的责任 <= 50%`；`伴侣的责任 <= 50%`
*   **应用**：把这个50%的规则运用到日常的吵架中，把至少50%的问题责任归结到自己的身上。你就会很自然地对老公说出：“我确实也有做得不对的地方”这样的话。而你的老公也会回应你：“我也有错。”你们各自承担50%的责任，解决了冲突，然后重归于好。

**技巧七：100%规则法**

双方对自己的幸福承担100%的责任。尽管我们应该努力让伴侣感到幸福，但幸福本质是每个人自己的事。你需要意识到，让你幸福不是老公的责任，而是你自己的责任。如果你觉得不满，你想改变现状，那么你应该自己做点什么来实现它。

*   **核心规则**：`我的幸福，我的责任 = 100%`
*   **自问**：你渴望什么？真正想要的是什么？你希望关系变成什么样？你想得到什么结果？过去你有哪些做得不够的地方？从现在开始，你又能做什么？想想这些问题，而不是一味的抱怨指责对方，这注定无法让你幸福。

无论你处在何种争吵中，利用这七个关键性技巧，认清它的本质，学会有意的争吵方式，从而达成理解，拉近和巩固你们的关系，获得想要的亲密感和满足感，重建甜蜜。这些技巧可以单独使用，也可以成套地依次使用，都能发挥作用。

## 三、实用工具：争吵结果自测表 📋

一场有建设性的、积极的争吵，可以增进亲密性。夫妻双方在这个过程中都是赢家，会取得更多积极的结果。心理学家巴赫设计了一份“争吵结果检视表”。你可以对照检视表的11项进行自我检查，回答“是”或“否”。“是”越多代表着更多的积极的争吵的结果。反之亦然。

以下是检视表的11个项目：

1.  **伤害**：你是否较少地感到受伤和被冒犯？
2.  **信息**：你是否获得了更多关于伴侣和这段关系的信息？
3.  **解决**：是否解决了问题？
4.  **控制**：你是否获得了更多影响伴侣的能力，且是被伴侣所接受的？
5.  **恐惧**：你是否降低了对冲突的恐惧？
6.  **信任**：你是否增进了对伴侣的信任，认为伴侣是好意的？
7.  **报复**：这种冲突是否会激起你报复的冲动？（反向指标，回答“否”为佳）
8.  **和解**：是否达成了和解？两个人是否消除了对彼此的伤害？
9.  **中心性**：是否感觉你已经成为了伴侣关注的和在意的中心？
10. **自我评价**：是否让你变得更加自信？
11. **协调/温情**：与伴侣的亲密度、对伴侣的吸引力是否都增加了？

这是一个很好用的自查工具。每一次争吵结束后都可以使用起来。

## 课程总结

本节课中我们一起学习了如何将夫妻冲突转化为亲密关系的契机。

首先，我们认识了**6个会破坏关系的吵架类型**（谁对谁错型、纠结琐事型、有话不说型、“如果你真的爱我”型、“我受不了你”型、“你跟你爸/你妈一个样”型），这些是需要我们警惕和避免的“坑”。

接着，我们深入学习了**7个积极争吵的技巧**：
1.  解读渴望，读懂核心诉求。
2.  保持高比例（5:1或20:1）的积极互动。
3.  遵守三个“不要”（不退避、不负面回应、不传递负面情绪）。
4.  预设对方出于好意。
5.  明确争吵目的，追求双赢。
6.  运用50%规则，各自承担责任。
7.  运用100%规则，为自己的幸福负责。

最后，我们介绍了一个实用的**课后工具——“争吵结果自测表”**，帮助大家在每次争吵后进行复盘和改进。
