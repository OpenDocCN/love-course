# 06、聊天技巧大全：第二阶段正式对话2调动情绪上_note

在本节课中，我们将要学习如何在与他人（特别是异性）聊天时，持续调动对方的情绪。这是建立吸引力和保持对话趣味性的核心技巧。

上一节我们介绍了如何开启对话，本节中我们来看看如何让对话过程始终保持吸引力。

## 持续调动情绪是关键

最关键的是要持续调动对方的情绪。我们不能只在打招呼时发一个搞笑表情，让对方笑一下就结束，后续的聊天变得无聊。那样做依然可能导致对方失去兴趣。

我们必须像制作一部精彩的电视剧一样，不能让对话只有开头好看，后面却乏味。我们需要让对话一直保持吸引力。持续调动情绪需要掌握特定的技巧。

## 第一招：欲擒故纵

首先需要明白什么是调动情绪。调动情绪就是要让对方产生情绪波动，让对方的情绪像坐过山车一样，一会高昂，一会低落，在开心、难过、紧张、浪漫等不同状态间切换。

如果情绪一直开心，那就是一条直线向上，没有波动。如果一直难过，就是一条直线向下，同样没有波动。必须有起伏，这才是调动情绪。因此，我们要让对方的情绪不停地产生反转和反差。

欲擒故纵就是一个调动情绪的、非常好用的技巧。它有两个核心公式：

**公式一：先说对他好，然后收回你的好。**
**公式二：先说不喜欢，再说喜欢。**

以下是具体运用方法的例子：

### 运用实例一：关于“饿”的对话

例如，女生说“好饿呀”。很多不懂技巧的男生会直接说“那我给你送饭吃”或“我给你做饭”。这会让女生觉得你很无聊，没有意思。

因为你的反应轻易就能被猜透。能被轻易猜透的就是无聊的，就像一部被剧透的电影，让人失去观看的兴趣。女生也就不会再被你吸引。必须让对方猜不透，像一部引人入胜的电影，让她好奇后面会发生什么，才会盯着看。

对于男生也是如此。让对方好奇你接下来会说什么、会做什么，她才会对你产生强烈的好奇心，从而被你吸引。如果她都能猜到你的反应，那就没意思了。所以一定要让她猜不透。

因此，一上来就说“我要给你做饭”是一个错误的方法。正确的做法是运用欲擒故纵。

女生说“我好饿”，我们不要直接说“我给你做饭吃”。可以先发一张自己做好的菜的照片给她，然后问“想不想吃？”。这会让她以为你想给她做饭。

女生很可能会说“想啊”或“看起来还不错呀”。这时，你跟她说：“不给你吃，我自己吃。” 气她一下，让她永远猜不到你要说什么。

女生可能会说“那我不吃拉倒”。然后，你再反转，说：“哎呀逗你玩的，有机会给你做个烛光晚餐。”

这让女生永远猜不到你下一步要说什么，并且让她的情绪在开心和生气之间来回切换。

### 运用实例二：关于“冷”的对话

再举一个例子。女生说：“哎呀下雨了，我好冷啊。” 不懂这一招的人可能会说：“来我来抱抱你。”

今天教大家一招。你可以先跟她说：“来我来抱抱你。” 女生说“好啊”的时候，你马上运用欲擒故纵说：“但是只给你抱三秒钟哦，不许占我便宜。”

这会让女生又爱又恨，前一秒爱你爱得不得了，后一秒又想“粉拳”打死你。在这种打打闹闹、爱和恨之间，她就对你产生了欲罢不能的感觉。

### 运用实例三：关于其他话题的对话

再比如，讨论到看电影的话题时，你不要直接跟女生说“我想牵着你的手看电影”，这太无聊了。你可以欲擒故纵地说：“有机会我们一起去看，但是你不能牵我的手。”

讨论到逛街的话题时，大部分男生会说：“只要陪你逛街，逛到天荒地老我都愿意。” 这无聊透顶，没有情绪波动。我们可以这样说：“我最不喜欢陪女生逛街了，累死了。”

她肯定会觉得你可能不愿意陪她逛街。但这时你突然反转，运用欲擒故纵说：“不过陪你还可以。”

总之，秘诀就是：永远在“喜欢她”和“不喜欢”之间徘徊，一会让她洋洋得意，一会又让她想“粉拳”打你，对你又爱又恨。让她总觉得马上就能抓住你，但发现又总是差了那么一点。
