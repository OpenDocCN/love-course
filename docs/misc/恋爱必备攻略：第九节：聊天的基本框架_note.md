# 💬 恋爱必备攻略：第九节：聊天的基本框架

在本节课中，我们将要学习与异性沟通时一个清晰、有效的基本框架。掌握这个框架，能帮助你避免聊天中的常见错误，并系统地推进关系。

沟通聊天这件事情是有规律可循的，不要随着心情瞎聊。当你和女生聊天的时候，你永远要问自己：我和她的聊天进展到哪个阶段了？然后思考在这个阶段下我该做什么。

聊天的规律是由浅入深的。话题顺序分为三层：第一层是外围话题，第二层是彼此个人的话题，第三层是“我们”共同的话题。最大的禁忌就是出现“交浅言深”的情况。比如有些人一上来就跟女生说心里话：“我一个人在这个城市太孤单了，渴望有人能跟我说说话聊聊天。”女生就会感到困惑：“你谁呀？为什么跟我说这些？”这就是交浅言深的结果。所以说聊天一定要循序渐进地进行。

上一节我们介绍了聊天的三层规律，本节中我们来看看每一层的具体内容和目标。

## 第一层：外围话题

外围话题是指那些与你们双方都没有太深个人关联的生活化内容。

以下是常见的外围话题示例：
*   吃喝玩乐
*   旅游见闻
*   宠物趣事
*   娱乐八卦

例如，“我今天尝试了一家新餐厅”或“最近某部电影很火”都是可以开启的话题。在这个部分你要做的是：通过外围话题的沟通，建立起彼此的熟悉感与舒适感，让女生知道你不是一个讨厌的人。

## 第二层：彼此个人的话题

当我们通过外围话题建立了初步舒适感后，便可以自然过渡到更个人的层面。

在这个阶段，我们可以聊一些彼此有关的事情。以下是适合此阶段的话题：
*   彼此的工作与职业
*   兴趣爱好
*   双方的生活习惯
*   个人经历与故事

这个阶段你的主要目的是：通过沟通增进双方对彼此的了解。你需要让对方知道你是一个什么样的人，同时也要了解到对方是一个什么样的人。因此，你要有意识地介绍自己，侧面展现自己优秀的一面，比如你的生活品质、品行、三观等。这个阶段处理好的话，你对女生的吸引力会增加很多。随着女生对你的了解越多，你在她心目中的魅力也就越大。

## 第三层：“我们”的话题

在彼此有了足够的了解和吸引力基础之后，关系就可以向更亲密的方向发展。

在这个阶段，你要聊你们彼此共同的话题，主要聊的是跟感情与未来相关的话题。以下是此阶段的核心话题：
*   情感观与恋爱观
*   人生观与价值观
*   对亲密关系的看法

而在这个阶段主要目的是：让你们彼此的情感观和人生观进行交流和融合，从而将关系从普通朋友上升到恋人关系。

本节课中我们一起学习了与女生聊天的三层基本框架：**外围话题 -> 个人话题 -> “我们”的话题**。你要遵循着这个规律去聊天，逐步拉近你们的关系，这才是一个正确的做法。

当然，仅有这个聊天规律，是不足以支撑你们整个聊天的。我们下节课继续探讨更多沟通技巧。
