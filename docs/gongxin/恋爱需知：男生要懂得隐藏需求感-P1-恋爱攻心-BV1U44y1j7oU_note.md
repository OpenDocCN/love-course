## 为什么不能过早暴露需求感？

上一段我们提到了隐藏需求感的重要性，本节中我们来看看过早暴露需求感会带来哪些具体问题。

追求一个女生，就恨不得让全世界都知道，真这么做的，没有几个好下场。很多人偶像剧看多了，总是一上来就大张旗鼓地追求女生。你以为能让女生感受到你的真心，结果你是给别人带来了压力。女生会提高防备性，你就越来越难追到她。

## 正确的追求心态：像猎手一样

理解了过早暴露意图的弊端后，我们来看看应该以何种心态来面对追求过程。

追求女生呢，就像狩猎。好的猎手，一定是耐心冷静，不露声色，然后抓住机会，一次成功。

以下是优秀“猎手”需要具备的几个关键特质：

*   **耐心**：不急于求成，懂得观察和等待。
*   **冷静**：情绪稳定，不会因对方的反应而患得患失。
*   **不露声色**：不轻易暴露自己的最终目标和强烈情绪。
