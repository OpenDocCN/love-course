# 恋爱攻心课程 05：真诚与套路的平衡之道 💬

在本节课中，我们将探讨人际交往中“真诚”与“套路”的关系，并学习如何基于对人性的理解，而非机械模仿，来提升与异性聊天的自然度和效果。

---

自古深情留不住，唯有套路得人心。现在男女交往随处可见都是套路。但毫无疑问，真诚才是人与人之间交往的基本前提。没有真诚加持的套路，只不过是一场论输赢的搅动。无论结局如何，和幸福的方向一定是背道而驰的。

那么，我们还要不要学习套路呢？当然要。首先来理解一下套路是什么。其实和我们做其他事情一样，需要理论和规律的指导。那么所谓的套路，也是基于研究者通过实验观察，总结出来的人际交往的运行规律。

有些人又在这个运行规律的基础上，总结出来一些可操作性的具体步骤。套路可以在初期阶段，成为我们了解自己、熟悉人际互动的工具。但是我们学习的重点，仍然应该专注于去了解它的底层逻辑，而不是跳过对底层逻辑的认识。

偷懒地去模仿那些所谓的操作路数，掌握事物的运行规律，可以让我们充分发挥自己的灵活性，幻化出千变万化的操作。而亦步亦趋地去模仿操作路数，只会导致思维僵化，沦为套路的奴隶。套路可以是锦上添花。

但一定不能成为两个人交往过程中的主旋律。在真诚的底子下，用套路去粉饰，会得到意想不到的效果。反之如果让套路占据了主轴，那一定是本末倒置，得不偿失的。
