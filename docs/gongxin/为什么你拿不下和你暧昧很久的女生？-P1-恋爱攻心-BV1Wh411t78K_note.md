## 问题诊断：为何无法“拿下”？

有这么一种女生，你们聊得很好，也做了很多情侣才会做的事，像是牵手逛街看电影，亲亲抱抱举高高。虽然你们很暧昧，但你却始终拿不下她。

出现这一问题，原因无外乎两点：
1.  她在吊着你，期待你为她付出更多。
2.  她不是一个随便的女孩，还没有完全认可你。

第一种毕竟只是少数，我就不多说了。如果遇到了就果断离开，以退为进说不定机会更大。话虽如此，我还是劝你别在这种女孩身上，浪费太多时间。

第二种才是我们今天要聊的重点。女生明知道你馋她身子的情况下，还愿意和你暧昧，却不愿意把自己交给你，说明她没有完全认可你，她的心还不完全属于你。这是因为你没有给到她，足够的安全感，或者你还没有达到她心理预期值。

这类女生通常不是为了物质，更多的是追求心理上的满足。

## 核心解决思路

上一节我们分析了问题的根源，本节中我们来看看如何系统地找到解决方案。所以你就要思考一下这些问题：
*   你们认识、交往了多久？
*   你是否有明确提出要发生关系？
*   她拒绝和你发生关系的理由是什么？

这样一说，是不是感觉思路清晰了不少，懂得该如何去解决了。
