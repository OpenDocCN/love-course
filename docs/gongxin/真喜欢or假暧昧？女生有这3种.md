# 恋爱攻心课 P1：识别“假暧昧”的三种表现 🚩

在本节课中，我们将学习如何识别女生并非真心喜欢，而只是在“逗你玩”的三种典型表现。理解这些信号，可以帮助你避免陷入无谓的情感消耗，更清晰地把握关系走向。

## 表现一：只在需要时才出现

上一节我们介绍了课程目标，本节中我们来看看第一种表现。这种女生只在她自身有需求或情绪低落时才会主动联系你，而在你需要支持时却往往缺席。

以下是一个具体案例：
*   一位男生在工作不顺、心情低落时，在朋友圈发布情绪化的内容。那位他喜欢的、通常熬夜的女生在此期间毫无反应。
*   当男生解决问题、外出旅游并发布快乐的朋友圈后，该女生立即点赞并主动聊天，甚至不经意地问起“前几天还在emo，怎么突然去三亚玩了”。
*   男生由此醒悟，此前女生只在遇到不开心或需要帮忙时才找他，让他误以为对方很在乎自己。而当他需要安慰时，对方的缺席揭示了真相。他随后选择了疏远。

**核心概念**：这种关系是**单向的情绪价值索取**。公式可表示为：**她的联系频率 ≈ 她的需求强度**，而与你的状态或需求无关。

## 表现二：只保持暧昧，拒绝推进关系

明白了单向索取的表现后，我们进一步探讨第二种情况。暧昧本身可以是一种愉快的互动，但若一方只想停留在暧昧阶段而拒绝明确关系，则是一种不负责任的行为。

以下是关键特征：
*   你希望关系能进一步发展，但对方总会通过言语或行动，有意无意地与你保持距离。
*   每当你感到失望、准备放弃时，对方又会主动撩拨，重新点燃你的希望。
*   这种循环让你持续投入情感，却无法获得关系的确定性。这与只想暧昧不想负责的“渣男”行为本质相同。

**核心概念**：这是一种**关系进程的操控**。代码可以简单描述为：
```python
if 用户.感到失望并准备撤退():
    对方.主动撩拨()
    用户.希望重燃()
else:
    对方.保持暧昧与距离()
```

## 表现三：只享受付出，不愿同等回馈

在分析了情感上的模糊态度后，我们最后来看看物质和行动上的不平衡。健康的交往讲究礼尚往来，单方面的付出往往预示着问题。

以下几点需要特别注意：
*   **正常情况**：追求阶段过于吝啬（如计较一杯奶茶）的男生，通常不会被真心对待。
*   **问题表现**：但反过来，也存在女生只享受你的付出（吃喝玩乐、贵重礼物），却在你需要帮助或希望关系更进一步（如见家长）时，以各种理由推脱。
*   **关键原则**：这表明对方没有长期发展的打算，只是在短期利用你的好感获取利益。

**核心概念**：这体现了**投资与回报的严重失衡**。一个简单的原则是：**追求期间，小钱不必吝啬，大钱切勿轻许**。确保付出是相互的、渐进的。
