## 三种成功率高的试探方法

以下是三种成功率较高且不易被察觉的方法，能帮助你悄无声息地测出她的心意。

### 第一招：故意不回复消息

这是女生常用的方法，我们不过是“以其人之道，还治其人之身”。具体操作是：你故意长时间不回复女生的消息。

*   **她若不在意**：她会像没事发生一样，既不追问也不生气，继续聊之前的话题。

### 第二招：观察单独约会表现

如果女生愿意单独和你约会，至少说明她不反感你，并有兴趣继续接触。但这不能直接断定她喜欢你，还需结合约会时的具体表现来判断。

*   **精心打扮**：她每次约会是否都愿意为你洗头、化妆，精心打扮。
*   **手机使用频率**：她在约会中看手机的频率高不高。
*   **情绪状态**：她的心情是始终愉快，还是经常板着脸。
*   **互动程度**：她的饭量大不大，话多不多。

如果她每次约会都为你精心打扮、心情愉悦、很少玩手机且饭量不大，那就足以说明她喜欢你。

这是最后的大招。身体反应往往比语言更诚实。但请注意，这不是让你直接去拥抱或触摸，那样做很可能适得其反。

这里所说的肢体接触，是指“无意间的有意触碰”。即你蓄谋已久，让自己的手“不小心”碰到了她的手。

*   **积极反应**：如果她不反抗、脸红，甚至有所回应，那么她的心意不言而喻。
