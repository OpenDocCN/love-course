# 恋爱攻心 - 课程01：如何建立高位心态 🧠

在本节课中，我们将要学习如何在恋爱关系中建立并保持一种“高位心态”。这种心态的核心在于，无论关系状态如何，你都能保持自信与稳定，从而掌握关系的主动权。

## 概述：什么是高位心态？

高位心态是一种内在的自信状态。它意味着你深刻认识到自身的价值，不因对方的态度而患得患失。拥有这种心态的人，在关系中表现得从容不迫。

上一节我们概述了高位心态的基本概念，本节中我们来看看它的核心原则。

## 核心原则：稳固自我价值感

无论你在关系中处于何种状态——是被动、患得患失，还是对方态度摇摆不定——你都必须稳住自己。你永远要向对方传递一种“我不在乎”的姿态，因为你的价值非常高。

这背后的逻辑可以用一个简单的**公式**来描述：
**自我价值 > 对他人认可的渴求**
当你的自我价值感足够强大时，你就不会过度寻求对方的肯定。

理解了核心原则后，我们来看看如何具体看待自己在关系中的位置。

## 心态定位：做“奢侈品”，而非“促销品”

你应该将自己视为一件高档的奢侈品。奢侈品从不缺买家，也无需自我推销。因此，你根本不需要去要求或乞求对方选择你。

以下是基于此定位的几个关键认知：

*   你无需主动推销自己，欣赏你的人自会前来。
*   你目前只是尚未开始广泛展示自己。
*   一旦你开始着手展示自我价值，对方的选择就显得无足轻重。

从“奢侈品”的比喻中，我们自然能推导出处理关系的具体态度。

## 终极态度：无惧失去的底气

永远要持有这样一种态度：这段关系“不是你也行”。对方选择你，只能证明他很有眼光，说明他本身也不错。这并非傲慢，而是源于对自身价值的绝对确认。

---

**本节课总结**

本节课中，我们一起学习了恋爱中高位心态的构建。关键点在于：**稳固自我价值感**，将自己定位为**稀缺的“奢侈品”**，并最终养成**无惧失去的底气**。记住，强大的心态始于你对自己价值的坚信不疑。
