# 分享一个有趣的故事-P1-恋爱攻心-BV11r4y1S7nW_note

在本节课中，我们将通过一个生动的故事，来探讨人际交往中的策略与心理。这个故事将揭示一些常见的社交技巧及其可能遇到的意外情况。

## 故事概述

我有一个非常漂亮的闺蜜。前段时间，她通过工作关系认识了一位成熟且经验丰富的大叔。

## 情节发展

在一次商务饭局结束后，大叔顺路开车送我闺蜜一段路回家。

### 惯用策略的实施

以下是闺蜜常用的社交策略：

*   她故意将口红落在了大叔的车上。
*   回家后，她给大叔打电话，说：“我口红好像掉在你车上了。”
*   她过去常用这个方法结识富二代。通常对方会亲自送回或快递给她，之后她便以感谢为由提出请对方吃饭。

### 意外的转折

然而，这次的情况有所不同。

当她打电话给大叔说明情况后，大叔直接回复：“那你上我家来取吧。”

## 闺蜜的反应与后续

这个回复让姑娘感到兴奋。她找到我商量，问道：“这样会不会发展的太快了呀？”

我给出了肯定的回答：“会。”

尽管如此，她还是精心打扮，花枝招展地前往大叔家。

### 出乎意料的结局

她敲开门后，发现开门的是一位阿姨，大叔本人并不在家。

阿姨对她说：“你是来拿东西的吧。” 随后，阿姨指了指屋内一个箱子，让她自己去里面找。

闺蜜回来后告诉我，那个箱子就像一个“公共场所的失物招领处”。里面堆满了各种女性物品：

*   丝袜
*   口红
*   粉饼
*   首饰

这些都是不同女生在大叔车上“落下”的东西。真是应了那句话：旱的旱死，涝的涝死。这个故事令人忍俊不禁。

---

本节课中，我们一起学习了一个关于社交策略的故事。它提醒我们，自以为巧妙的计谋，可能在经验丰富的人眼中已是司空见惯的套路。在人际交往中，真诚或许比技巧更为重要。
