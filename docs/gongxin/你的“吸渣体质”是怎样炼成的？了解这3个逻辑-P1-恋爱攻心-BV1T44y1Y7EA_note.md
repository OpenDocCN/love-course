## 原生家庭的影响 👨‍👩‍👧‍👦

上一节我们介绍了课程概述，本节中我们来看看第一个逻辑：原生家庭的影响。

人之初，性本善。但长大后，人的行为有明显善恶之分。这很大程度上受到原生家庭的影响。许多单身人士被家里催婚，却对婚姻提不起兴趣。一提到结婚，脑海中就浮现出父母争吵、摔打东西的场景。在父母看来，夫妻吵架很正常，不算矫情。但他们不知道，子女对婚姻的恐惧是真实的，并且是被他们潜移默化影响的。

正是因为从小在这种环境中长大，很多人会以为亲密关系就是如此：只有最亲的人，才不用顾忌，可以说最直白但也最伤人的话。这会导致两种选择：

以下是基于原生家庭影响可能产生的两种心态：
1.  **排斥亲密关系**：如果最亲密的人伤我最深，那不如单身。
2.  **适应不对等关系**：认为父母也是如此相处，自己已经习惯。即使对方对你不好，你的忍耐力也可以包容这一切。

**核心逻辑**：这等于亲手将伤害自己的权利交给了对方。那些能伤害你的女生，并非能力多大，而是你赋予了她这种权利。只要她打压你，你就远离，不给她机会，她便无能为力。

对于这种情况，要想改变，最根本的是从自身做出改变，尝试新的关系模式。要知道，良性的亲密关系不会让人痛苦，它本应是互相尊重、共同成长的。你要做的不是忍耐扭曲的关系，而是开辟新的渠道。如果对方打压你、贬低你，让你变得自卑胆怯，你应该告诉自己：**`这不是爱，而是伤害`**。一定要尽早远离。

## 讨好型人格 😔

上一节我们探讨了原生家庭的深远影响，本节中我们来分析第二个逻辑：讨好型人格。

这一点很常见。很多男生用实际行动将讨好型人格诠释得淋漓尽致。还没在一起，就对女生言听计从。表达不同意见时，一看对方脸色不对就赶紧改口，生怕惹人生气。

所以，不必跟自己过不去。你应该多想想自己的优点。

## 自我价值与关系定位 ⚖️

上一节我们分析了讨好型人格的弊端，本节中我们来看看如何建立健康的自我价值与关系定位。

改变“吸渣体质”的关键在于重塑自我认知和关系模式。你需要建立平等的视角，认识到健康的感情是**`价值对等`**与**`相互尊重`**的。

以下是构建健康关系的基础步骤：
1.  **确立自我边界**：明确什么是可以接受的，什么是不可触碰的底线。用行动守卫你的边界。
2.  **提升自我价值感**：专注于发展自己的事业、爱好和能力。真正的自信源于内在的成长，而非他人的认可。
3.  **学习平等沟通**：在关系中敢于表达真实的想法和感受。好的感情是可以通过沟通磨合的，而非单方面的妥协。
4.  **及时识别与止损**：当一段关系让你持续感到自卑、痛苦时，要有勇气离开。远离消耗你的人，是对自己最大的负责。

记住，你吸引什么样的人，并不完全取决于运气，更取决于你如何看待自己，以及你允许别人如何对待你。

本节课中，我们一起学习了“吸渣体质”形成的三个核心逻辑。

改变是一个过程，从意识到问题，到逐步调整心态和行为。当你开始尊重和珍视自己时，你自然会吸引到更健康、更尊重你的关系。
