## 第一雷区：听不懂“话里有话” 🎭

## 第二雷区：缺乏边界感 🚧

理解了沟通中的潜台词后，人际交往中的“度”同样关键。本节我们来探讨缺乏边界感的表现。

没有边界感的人，不仅在行为上喜欢试探底线，在言语上也常常毫无分寸，贸然闯入他人的私人领域。

以下是基于真实经历的一个对话示例：

*   对方问：“你平时工作忙不忙啊？”
*   回答：“还挺忙的，经常拍视频到很晚。”
*   对方接着问：“这么忙啊，我看你气质也挺好的，一个月能挣个三五万不？”

这种初次见面便直接询问具体收入的行为，属于典型的 **“越界”** 。它让对方陷入回答困境，无论肯定或否定都感到不适。

除了言语边界，在互动方式上也有常见误区。本节我们关注急于提供解决方案的问题。
