上一节我们介绍了问题的普遍性，本节中我们来看看第一种具体情况。

如果真是这种情况，不建议你继续给她发送消息。

## 情况二：女生聊一半就消失了 🚶‍♀️💨

聊一半就消失的情况稍微有点复杂，我们需要进行分析。

以下是可能的原因分析：

1.  **与第一种情况一样，她对你不感兴趣**  
    这没什么好说的了，就是对你不感兴趣，导致她对你说的话题也不感兴趣。

2.  **她不知道该怎么回复你，说明你尬聊了**  
    一般刚认识时，出于礼貌女生都会回复。只不过你说的话让她实在不知道怎么回，索性就不回了。  
    举个例子：女生说“降温了，有点冷”，你回她“来我怀里，我怀里热”。这让女生怎么回？是骂你不要脸呢，还是回你一个啥？所以这种情况下，你可以试着转移话题看看。

3.  **她刚好在忙，忘记回复你了**  
    如果你们聊得好好的，她突然就不回了，就说明去忙了。这种情况很常见。  
    举个例子：她刚好公交到站了，下车、步行、买早餐、吃早餐、到公司处理工作上的事。这种情况，你就再等等，看她是不是会想起来回复你。如果过了一会回复你了，那刚好。如果过了一天半天都没回，你就可以试着再给她发一条信息看看。

4.  **你说错话，惹她生气了**  
    有些男生喜欢说一些自以为很幽默的话，以为这样就能逗女生开心。但你有没有想过你们是什么关系？你了解她吗？什么话该说，什么话不该说？有时候，在关系还不是那么熟稔的情况下，贸然和女生说一些话，很容易引起对方的反感和厌恶。所以赶紧去翻翻聊天记录，看看你是不是说错话了。如果是的话，那就赶紧道歉去吧。
