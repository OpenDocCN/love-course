# 课程：如何找回自信 - P1 🧠

在本节课中，我们将要学习自信的本质来源，以及它是如何在成长过程中被外界因素所影响的。我们将探讨原生家庭、教育环境和个人经历如何塑造我们的自信水平，并理解每个人内在都拥有自信的潜能。

## 自信的起源与影响 ☀️

上一节我们介绍了课程概述，本节中我们来看看自信的起源。每个人在出生时都具备自信的能量。人类的自信在出生时就已存在。

成长过程中的多种因素会影响一个人的自信程度。原生家庭的规矩、不同的教育方式以及个人性格的差异都会产生影响。

从幼儿园到高中的整个教育阶段，个人所处的环境会持续影响自信。遇到的老师、同学以及周围的成长环境都会产生影响。

## 自由成长与自信的关系 🌱

如果一个人的成长环境限制较少，其自信往往更强。父母的教育方式如果条条框框较少，给予孩子大部分自主能力和自由，会更有助于培养自信。

例如，只设定“不要伤害别人”这样的基本准则，而非每天强制要求“必须读多少书”或“必须做多少作业”。

在这种相对自由、管教不那么严厉的环境中成长的孩子，通常比在诸多限制下长大的孩子更自信。因为他们的思维更自由，并且认为自己拥有自由的权利。

## 每个人内在的潜能与光芒 ✨

我相信每个人都拥有内在的潜能。每个人都是一个小宇宙，都是一个小太阳，都可以发光，都有能量和热量，都有闪光点和光芒。

然而，内心的压抑和外界的影响会遮蔽这些光芒。从出生那一刻起，人就开始面对外界客观的各种各样的事情。

在成长过程中，这些外界经历会逐渐内化为个人的观念。这些观念就像天空中的云朵。

有的云是白云，有的云是乌云，有的云颜色浅，有的云颜色深。这些“观念云朵”决定了你自信的光芒能否有力地穿透出来。

这些观念可能会伴随一生，因此需要主动去处理和解决它们。如果想要找回自信，想要让真实的自己光芒四射，就需要面对这个挑战。

---

本节课中我们一起学习了自信的天然存在性，以及家庭、教育和社会环境如何像“云层”一样影响我们自信光芒的释放。理解这些是找回自信的第一步。
