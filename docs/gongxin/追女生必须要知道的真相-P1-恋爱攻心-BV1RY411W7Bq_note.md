# 💖 恋爱攻心课程 P1：追女生必须知道的真相

在本节课中，我们将要学习一个关于追求女生的核心真相。理解这一点，能帮助你避免陷入单方面付出的困境，从而建立更健康、对等的关系。

## 🧠 核心真相：女生的精明与筛选机制

追女生必须要知道的真相，其实女生都是很精明的。你和她聊天时，各种示好和付出，女生也能感知到。

但是，女生仍然会装作无所谓的样子。因为她明白，她对你不用过多的投入，你就会为她付出。

---

## 🔍 关系动态分析：备胎与筛选

大部分女生是不会拒绝一个对自己好的备胎的。这个时候，她就在进行筛选，看你能不能达到她的择偶标准。

然后，你就会下意识地向女生证明自己。这个行为模式可以用一个简单的**公式**来描述：

**你的行为** = 感受到被筛选 -> 试图证明自我价值 -> 付出更多

你也就不知不觉为女生付出了更多。

---

## ⚠️ 关系停滞的后果

如果，你和喜欢的女生是这样的关系。

那么，你不会和她有升级关系的可能。关系会停滞在“追求者”与“被追求者”的单一动态中，无法发展为平等的恋人关系。

---

## 📝 本节总结

本节课中，我们一起学习了追求初期的一个关键真相：女生能感知到你的付出，但可能会因为你的过度主动而减少自身投入，并将你置于一个被筛选的位置。认识到这种动态，是避免陷入单方面付出、从而争取建立平等关系的第一步。
