## 无法自拔的爱：现象与疑问

你有没有想过，为什么会爱上一个人到无法自拔的地步？那些让你朝思暮想的，可能只是一些微小的感受。比如当你看到他的照片，就会感到温暖。偶尔聊上几句就能心安。如果能见一面，那更是心底的欢喜。

世界上唯一不能解释的，就是为什么会爱上一个人。但是在心理学中，所有的问题都能找到根源。

## 核心概念：依恋行为系统

上一节我们提出了疑问，本节中我们来看看心理学给出的答案。为什么爱一个人无法自拔，实际上是指**依恋行为系统**。之所以称之为系统，是因为它由安全感缺失，到产生情绪，到投射对象。这是一系列的过程。

以下是构成这个系统的三个关键阶段：

1.  **安全感缺失**：可能是因为你的原生家庭而产生的自卑感。比如在同学或同事面前不自信，而逐渐形成了弱势社交的状态。
2.  **产生情绪**：因为某件事，引发了你体内脾致醇（一种与压力相关的激素）的堆积，从而导致了你持续的焦虑情绪。
3.  **投射对象**：比如你的同学或者同事，看见了你不开心，愿意陪你聊天。于是他让你找到了安全感，逐渐引发了你对他的依恋的可能性。

所以很多人的初恋，都是离自己很近的人，比如同桌或者邻居。因为关系紧密，别人对你释放了善意和友好，让你对他产生了喜欢。

你可能会说，我当时喜欢那个人，就是因为她长得好看而已。实际上，由欣赏美好而产生的喜欢，也是一样的，都属于依恋系统。

## 系统原理：对“真实自我”的渴望

那么，依恋系统的原理是什么呢？因为每个人都渴望看到一个真实的自我。镜子可以看清楚外表，但价值观、想法、品格，这些东西很难被真实地反馈，所以我们经常迷失自我。

于是你会疯狂去探索那些能给自己带来价值认同的东西。一旦遇到，你就会感到满足。比如：
*   长得很像你母亲的女生。
*   温柔漂亮的女生。
*   能和你积极互动的女生。

这些都能给你提供足够的安全感。在心理学上，真爱不过是一种对童年缺失的弥补，为了满足自己的需求而寻求的慰藉而已。

## 解决方案：向内求索与自我满足

了解了原理后，我们来看看如何应对。内心强大的人相对而言，爱一个人到无法自拔的概率就会低很多。

所以，一旦你的生命当中出现了那个让你沦陷和无法自拔的人，**多向内求**。尽可能多地去做一些能够自我满足的事情，放大自己的价值，自己弥补自己的安全感。你就不会一直沉浸在感情的漩涡之中痛苦挣扎。
