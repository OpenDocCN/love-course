# 恋爱攻心 - 第一课：扩大选择范围 🎯

在本节课中，我们将要学习一个核心的恋爱策略：避免将全部精力与情感投入在单一目标上。我们将探讨为何需要扩大社交圈，以及如何以健康、高效的方式筛选潜在伴侣。

## 核心原则：避免单点投入

不要把目标只放在一个人身上。将目标锁定在一个人身上的前提，必须是对方本身也对你有兴趣、喜欢你。如果这个女生纯粹需要靠你去追求，那么千万不要把所有注意力、心意和付出都放在这个人身上。

上一节我们介绍了避免单点投入的核心原则，本节中我们来看看如何实践这一原则。

## 实践方法：广泛接触与筛选

你只是在与她们交朋友、认识一下，这没有关系。这种行为不应被理解为“海王”或“养鱼”，因为你后续需要逐步筛选。

以下是进行筛选的两个关键方向：

*   **筛选掉不喜欢你的人**：如果你感觉到这个人对你不感兴趣，就不要让她浪费你的时间。果断地将她排除。
*   **筛选掉你不喜欢的人**：对于你不感兴趣、不喜欢的女生，也应直接排除，不要浪费对方的时间。

男生可以果断一点，处理方式不要拖泥带水。明明感觉到对方对你没兴趣，却还要继续浪费自己的时间，这无异于自讨苦吃。

---

本节课中我们一起学习了在恋爱初期扩大选择范围的重要性。核心在于**广泛接触、逐步筛选**，既要及时排除对你无意的对象，也要主动放弃你不感兴趣的人，从而高效地找到相互吸引的伴侣，避免无谓的情感消耗。
