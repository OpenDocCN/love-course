# 恋爱攻心课程 P1：追女生一定要学会“压价” 💰

在本节课中，我们将要学习一个在追求阶段至关重要的概念——“压价”。理解并掌握这一策略，能帮助你在与心仪女生互动时保持良好心态，避免陷入被动，从而更有效地推进关系。

## 什么是“压价”？🤔

上一节我们提到了“压价”这个概念，本节中我们来详细看看它的具体含义。

“压价”的意思很简单，就是**不要在女生面前表现自己对她有多么喜欢，或者多么的离不开她**。

## 为什么需要“压价”？⚠️

理解了“压价”的定义后，我们来看看忽视这一原则可能带来的后果。

因为一旦开始在她面前暴露自己强烈的需求感，那么即使你喜欢的女生很普通，后面呢也会因为追她而付出更大的代价。

以下是暴露强烈需求感后可能发生的连锁反应：

*   因为她知道你离不开她后，就会延长你追求她的时间。
*   她可能会引导你为她付出更多。
*   到最后呢你有可能什么都得不到。

最终的结果可能就像下面这个经典的“公式”：

**你的巨大付出 + 她的完全掌控 = 你收到一张“好人卡”**

她就只给你发了一张好人卡。
