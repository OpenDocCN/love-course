# 恋爱攻心课 P1：如何应对“双标”行为？ 🤔💔

在本节课中，我们将通过一个具体的情感咨询案例，学习如何识别并应对关系中不一致的“双标”行为。我们将重点分析如何建立个人边界、展现态度，从而维护自尊并可能扭转被动局面。

## 案例背景与问题

上一段我们了解了课程主题，本节中我们来看看具体的案例情况。

一位男士在追求一位女士时遇到了困惑。在被拒绝后，他有两个月未主动联系对方，之后女方主动联系了他，让他以为关系出现了转机。于是，双方恢复了较为密切的联系，几乎每天或每两天都会联系。

在七夕节（“七七那天”），这位男士为女士购买了礼物并转账了**520元**（谐音“我爱你”）。然而，在收到礼物和转账后，女士却指责他“越界”了，认为他不应该在当天送这样的礼物，并让他暂时不要联系自己。

## 核心矛盾分析

上一节我们介绍了案例背景，本节中我们来看看其中的核心矛盾点。

这位女士的言行存在明显的不一致：她**接收了具有明确示爱含义的礼物和转账**，却**反过来指责对方行为越界**。这种“双标”行为是问题的关键。

以下是双方行为对比：
*   **男士行为**：在特定节日赠送礼物并转账**520**。
*   **女士行为**：接收了礼物和钱，然后声明对方“越界”并要求暂停联系。

## 应对策略与心态建设

分析了矛盾所在后，我们来看看应该如何应对这种情况。核心在于建立自尊和展现一致性态度。

首先，建立恋爱关系不是独角戏，双方的心意同等重要。如果对方对你有基本的接受度，就不会出现如此矛盾的言行。

对于这种言行不一的情况，直接、有态度的回应往往比委曲求全更有效。当对方说出“你越界了”这类话时，你可以立即设定边界。

以下是具体的回应思路：
> 既然你觉得我越界了，那请你把礼物和**520元**转账退给我。

这种回应方式能达到几个目的：
1.  **展现个性与态度**：你不再是单方面讨好的一方，而是有了自己的原则。
2.  **将矛盾交还给对方**：让对方为其“双标”行为承担后果。
3.  **可能引发对方关注**：这种出乎意料的、有尊严的反应，有时反而能打破对方的心理预期，甚至引起她的好感和征服欲，让她“想要扳回一局”。

## 课程总结

本节课中，我们一起学习了如何应对情感关系中的“双标”行为。

我们通过一个案例，认识到面对言行不一致的指责时（例如收了**520**红包却说你越界），关键不在于自我怀疑，而在于坚定维护自己的边界。直接要求对方退还礼物和钱财，是一种展现态度、维护自尊的有效方式。记住，健康的关系建立在相互尊重的基础上，而不是单方面的妥协和付出。
