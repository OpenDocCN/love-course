很多人认为，不自信源于没有钱。钱确实能给人带来自信，尤其对男性而言。但透过现象看本质，不自信真的仅仅是因为没有钱吗？

让我们来假设一个场景。一个在农村长大的小伙子，从未离开过农村。他能爬树、能摸鱼，在村里几乎无所不能。他是绝大多数小伙伴崇拜的偶像。那么，这个小伙子在村里自信吗？答案是肯定的。

但是，当某一天他进入城市，看见到处灯火辉煌，面对太多从未见过的人事物时，他露出了胆怯。为什么他突然就不自信了呢？

接下来，我们再看另一个假设。一个富二代，见惯了灯红酒绿、豪车豪宅名表，身边总是香槟美女环绕。

有一天，他组织一群朋友去海岛冒险。结果因为意外，他们被困在了海岛上，同时与外界失去了联系。他们所带的食物也在两天后消耗殆尽。想要生存下去，就必须到山上摘野果，去海里捕鱼。

这时，他那几个比他穷的哥们，展现出了更强的野外生存能力。同行的几个女生，也将之前对他示好的目光，转移到了他那几个穷哥们身上。请问，这时候的富二代还自信吗？

那么，自信到底是什么？最本质的东西是什么？

大多数时候，钱能带来自信，是因为我们所处的社交环境需要钱这个“社交货币”。但是，一旦这个社交货币不再是钱，那么钱就失去了它带来自信的功效。

关于自信这个命题其实很深奥。它涉及到从你出生就自带的、最纯粹的“本我”，以及后天由原生家庭和社会规则所束缚你的“小我”。在本我和小我之间如何做到平衡，就是核心自信的关键。
