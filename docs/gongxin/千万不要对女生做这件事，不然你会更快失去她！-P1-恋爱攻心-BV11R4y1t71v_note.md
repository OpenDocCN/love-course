## 1. 🧐 初识阶段：吸引力源于想象

上一节我们提到了过早表白的普遍疑问，本节中我们来看看初识阶段的核心问题。

首先你得知道，在初识阶段你们对彼此不是很了解。

你想追一个女生，可能只是恰好她符合你的审美。说白了，就是你图人家长得好看、气质好。

很多时候，你喜欢的她不是真实的她，只是你想象中的她而已。

你甚至都不确定了解她之后还会不会喜欢，会喜欢她多久。

你自己都不确定，就贸然地去表白，人家女孩子怎么敢相信你呢？

> 以前的我不会聊天，不会哄女生，29岁了还是单身。后来学习了聊天技巧，不到半个月就追到了心仪的女神。对于单身的兄弟，提升自己很关键。很多技巧和套路可以直接学习应用，核心是“学以致用”。

## 2. 🤔 了解阶段：从朋友区到吸引区

了解了初识阶段的陷阱后，本节我们深入探讨所谓的“了解阶段”。

那你觉得你们认识了一段时间，互相都也挺了解了，是不是就可以表白了？

既然你这样想，那就教你一个表白100%成功的方法。

**操作代码**：
```
1. 拿起手机，给喜欢的人发送表白内容。
2. 发送后，立即将消息删除。
```
只要你删得够快，根本没有人能拒绝你的表白。天下武功唯快不破。

跟你说了别着急。虽然你们之间已经从陌生人跨入了朋友区，但你想要吸引他/她还得需要过程。

我知道你们在喜欢一个人的时候，特别喜欢给自己加戏。

以下是几种常见的“加戏”心理：

*   **他看你一眼**：你觉得他注意到你了。
*   **他对你笑一下**：你把孩子的名字都想好了。

你能保证人家确实就是喜欢你吗？

你就想象一下，要是一个你无感的朋友，突然跟你表白，你是什么感受？
