## 核心策略：如何避免冷战？ ✋

认识了冷战的危害，本节中我们来看看如何才能有效避免冷战的发生。一个核心策略是：**在和平的期间先立好规矩**，也就是我们俗话说的“约法三章”。

在恩恩爱爱、甜甜蜜蜜的氛围下，什么话都好说，立规矩这么客观的事情更是不在话下。约定的内容可以包括：

*   **约定一：** 吵架的时候不允许有冷战，当下的事情当下解决，当下的情绪当下化解。
*   **约定二：** 吵架的时候不能人身攻击，不能责骂家人。

**公式：有效沟通 = 和平期立规 + 日常强化**

在和平的时候立下规矩，然后在平时的时间多加强化，吵架的时候就能避免很多不必要的伤害。即使有时候情绪上头，也会下意识地去想到约定的内容，下意识去控制自己的情绪，这在一定程度上能降低吵架对感情的损伤。
