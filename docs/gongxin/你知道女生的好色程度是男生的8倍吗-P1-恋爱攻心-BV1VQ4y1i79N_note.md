**社会期待** + **内在感受** = **外在表现**

*   **表达方式的差异**：社会传统常鼓励男性主动表达，而期望女性含蓄矜持。

所以，当我们谈论吸引力时，不能仅凭外在行为做判断。

本节课中我们一起学习了关于两性吸引力表达差异的观点。我们认识到，女性的情感与欲望可能比表面看起来更强烈，只是受社会文化影响，其表达更为隐蔽。理解这一点，能帮助我们打破刻板印象，以更平等和深入的视角看待两性关系与吸引力。
