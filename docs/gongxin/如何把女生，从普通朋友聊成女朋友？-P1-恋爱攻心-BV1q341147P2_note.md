大部分男生不知道的一个关键点是：想要顺利进入下一个关系阶段，就必须在**当前阶段，提前做一些属于下个阶段的事情**。

普通男生往往安分守己，只敢说符合当前“朋友”阶段的话，因此关系一直停留在朋友阶段。而撩妹高手情商较高，懂得在适当时机进行合适的操作，尝试“跳跃”到下个阶段。

“跃层”的精髓就在于：**提前干点下个阶段的事**。这里涉及一个重要的技巧，叫做 **“合理化”**。
