# 恋爱攻心 第1课：识别应拉黑删除的女生类型 🚫

在本节课中，我们将学习如何识别一种在约会关系中只懂索取、不懂付出的女生类型，并明确果断处理此类关系的必要性。

上一节我们介绍了课程主题，本节中我们来看看具体的识别标准与行动建议。

## 核心识别标准

当遇到符合以下描述的女生时，应考虑将其拉黑并删除联系方式。

以下是具体的判断依据列表：

*   她与你约会见面四到五次，但从未在任何一次约会中花费过一分钱。
*   她表现得一毛不拔，甚至连一瓶水都未曾主动购买过。

## 行为分析与应对策略

符合上述标准的女生，其行为模式揭示了更深层的问题。她**只会索取而不懂付出**，并且在关系中**丝毫不关心你的感受**。更关键的是，她对此表现得**脸皮很厚**，毫无愧意。

因此，你的应对策略应当果断。可以直接向她指出问题：“**你会谈恋爱吗？你是铁公鸡吧。**” 无需感到不舍，因为你需要理性计算自己为约会投入的成本。

请思考你为约她出来所付出的代价：

*   请她吃饭
*   请她看电影
*   其他各项开销

**即便是陌生人之间，也讲究礼尚往来。**

## 总结

本节课中我们一起学习了如何识别一种在约会中极度自私、只知索取的女生类型。关键标准在于观察对方是否在多次约会中**完全不愿付出**。面对此类情况，应果断结束关系，及时止损，因为健康的关系建立在相互关心与付出之上。
