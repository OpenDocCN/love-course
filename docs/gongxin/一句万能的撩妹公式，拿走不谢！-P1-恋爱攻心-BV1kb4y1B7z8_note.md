# 恋爱攻心课 P1：一句万能撩妹公式 🎯

在本节课中，我们将学习一个应对女生提问的通用对话技巧。这个技巧的核心在于通过一个简单的反问句式，将任何问题都巧妙地转化为表达好感的契机。

## 核心公式与思路 💡

这个技巧的核心是一个固定的对话结构。无论女生提出什么问题，你都可以用以下方式回应：

**公式：反问“你问的是以前还是现在？” + 回答“以前我喜欢…，现在只喜欢（和你）…”**

这个结构的精髓在于，它首先通过反问将问题的时间维度进行拆分，为自己构思回答争取了时间。随后，无论对方如何回应，你都可以用一个对比句式，将话题自然地引导到“现在”与“她”相关的方向上，从而含蓄地表达对她的特别关注。

## 公式应用详解 🛠️

上一节我们介绍了公式的基本思路，本节中我们来看看它的具体应用方法。关键在于将女生的原始问题，套入“喜欢…”的框架中。

以下是两个具体的应用示例：

*   **示例一：关于“喜欢什么类型”**
    *   女生提问：“你喜欢什么样的女生？”
    *   你反问：“你问的是以前还是现在？”
    *   （无论女生回答“以前”还是“现在”）
    *   你回答：“以前我喜欢可爱的女生，现在只喜欢你这样可爱的女生。”

*   **示例二：关于“兴趣爱好”**
    *   女生提问：“你喜欢看什么电影？”
    *   你反问：“你问的是以前还是现在？”
    *   （无论女生回答什么）
    *   你回答：“以前我喜欢看搞笑的电影，现在只喜欢和你一起看电影。”

## 使用要点与总结 📝

通过以上例子可以看出，这个公式具有很高的灵活性。它的应用要点可以总结为以下两步：

1.  **固定反问**：面对任何问题，首先统一反问：“你问的是以前还是现在？”
2.  **对比回答**：接着，用“以前我喜欢…，现在喜欢和你一起…”的句式来组织答案。前半句承接问题本身，后半句则巧妙地将焦点转移到对方身上。

本节课中我们一起学习了一个简单实用的对话技巧。其核心在于通过“时间对比”的句式，将普通的问答转化为传递好感的时机。记住这个**“反问时间+对比回答”**的公式，并在实践中灵活套用，即可达到出其不意的效果。
