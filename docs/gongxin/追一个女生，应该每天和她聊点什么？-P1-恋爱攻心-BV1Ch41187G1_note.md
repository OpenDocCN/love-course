# 追一个女生，应该每天和她聊点什么？-P1-恋爱攻心-BV1Ch41187G1_note

在本节课中，我们将学习与女生聊天时的一个核心技巧：如何通过对话判断她的情绪状态。掌握这一点，能帮助你避免尬聊，让交流更顺畅愉快，而不是总被“呵呵”回应。

和女生聊天时，聊什么话题并非重点。关键在于保证聊天不尴尬，并能愉快顺畅地进行下去。这就涉及一个重要知识点：如何通过聊天了解女生的想法。你需要明白，为什么有些人似乎总能轻松赢得女生好感。

你总被女生冷淡回应，不是因为不够帅、不够有钱或不会说话，而是你不懂女生的心思。其实，你无需刻意揣摩她每句话的潜台词，也不用小心翼翼怕说错话。只要你能读懂她的情绪，一切就会顺利很多。

接下来，我们开始讲解具体方法。

## 一、识别女生不想聊天的信号 🥶

当女生不想和你聊天时，她的表现通常是冷淡、冷漠或高冷的。所有言语都会显得冷冰冰，常表现为不回复、只回复单字，或随手回一个表情包。例如：

*   **嗯**
*   **哦**
*   **好**

这时，你面前只有两条路：

1.  **主动告辞**。
2.  **转移话题**。

## 二、识别女生想聊天的信号 😄

女生通常不喜欢和自己不感兴趣的男生聊天。所以，当她叽叽喳喳和你聊个不停，主动分享生活，你说一句她回三四句时，说明她心情很好，很愿意和你聊天。

这时，你可以比较放松地聊天，只要不说些过分的话，一般都不会有问题。

## 三、识别女生生气的信号 😠

如果你见过她开心时的样子，就更容易判断她的情绪。如果她表现出不开心，那通常是不想理你。不想理你的原因主要有两个：

1.  对你没兴趣。
2.  生你的气了。

生气时，女生常常喜欢正话反说。把这些话串联起来，大概是这样的：

> **不要。我没事。你忙吧。我先睡了。不要给我打电话。**

## 四、识别女生撒娇卖萌的信号 😊

当对话中出现特定的语气词时，往往有撒娇卖萌的嫌疑。例如：

*   **人家...**
*   **讨厌~**
*   **嘛**
*   **哪**
*   **呀**

你可以尝试组合感受一下：**“人家不要嘛~”、“你在干嘛呀？”**
