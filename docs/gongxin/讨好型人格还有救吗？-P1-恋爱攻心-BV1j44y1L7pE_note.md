# 恋爱攻心 - 课程01：告别讨好型人格 🚫

在本节课中，我们将要学习如何识别并摆脱“讨好型人格”的陷阱。许多人在关系中感到卑微和痛苦，其根源往往在于这种不健康的心理模式。我们将剖析其本质，并提供切实可行的步骤，帮助你建立自信，学会真实地表达自我，从而赢得他人真正的尊重。

---

我相信所有男生都知道不能当舔狗，但为什么还有那么多舔狗呢。说实话，我有时候很心疼那些在感情当中很卑微的男生，但又不得不承认，他们都是在自我折磨。

在舔狗身上永远存在着一个陷阱。他们始终都坚信自己应该取悦别人，只要自己取悦了别人，那么别人也会对自己好。如果别人对他们不好，他们也很难将原因指向外部因素，然后开始找自己的原因。

所有舔狗的内心都是希望通过委曲求全，换来对方满足自己的需求。但结果往往不能达到他们的预期。而等到这个时候，舔狗又会感觉到强大的负面情绪，然后把**这些负面情绪全部都消化到了自己的身上**，开始觉得都是我不好，都是我自己的错误。

你看，他们永远都在自我惩罚，永远都是在和自己较劲，也永远都在不停地折磨自己。

上一节我们剖析了“舔狗”心态的循环陷阱，本节中我们来看看这种行为在具体关系中的表现。

当他们接触到心仪的女生的时候，就会立马无限放低自己的心理地位，想方设法的讨好对方。他会为了对方改变自己的作息习惯、穿衣风格、工作计划。甚至会因为对方随口说的一句话，或者一条根本没放在心上的朋友圈，而研究半天。

他会想：**“对方说这句话到底在表达什么，是不是我惹对方生气了？他发这条朋友圈，是不是在暗示什么？”**

在这种情况下，他完全失去了自己的生活。他把自己全部的寄托都放在了一个根本没把他放在心上的普通朋友关系的人身上。这导致他会因为对方哪怕有一点不符合他预期的行为，他都会感受到无比的委屈和难过。

我希望你一定要搞清楚，对一个人好和讨好是完全不同的。后者是一种病态的心理模式，这种人在长期关系当中，只会给对方带来负担和困扰。

了解了讨好型人格的表现与危害后，接下来的核心是找到改变的方法。以下是三个循序渐进的步骤，帮助你重建自我。

如果你是这种讨好型人格，那么我的建议其实很简单。

**第一，找到你的兴趣爱好或者长处，锻炼出来一个可以带给你成就感的技能。**

讨好型人格本质上就是自我认同度低。与其不停地和讨好型人格讲“你要自信，你要相信自己值得被爱”，还不如直接先掌握一个看得见摸得着的、能让自己有成就感的技能。当这种技能和爱好掌握了之后，周围收到的正向反馈，会不停地催化你的自信，让你更有底气去面对人际关系。

**第二，在上一项的基础上培养自己强大的内心。**

刚开始你就需要先试着一两次拒绝。我们现在只是做一个实验，看看如果你真的采取了对抗的态度，对方的反应会不会像你预设的那么可怕。实际上，一两次实验之后，讨好型人格的人就会发现，原来就算自己说“不”，别人也不会把自己怎么样。在这个基础上，你的自信会进一步上升。

**第三，开始认真地学会拒绝别人，表明自己的态度。**

有了前两步铺垫，如果你是讨好型人格的人，那么你现在或许已经可以开始大大方方地拒绝别人了。当然，拒绝别人不是全部目的，你的目的是可以大方地说出自己的态度和想法，真实地做自己。所以要敢于在生活当中积极表态，敢于展示自己的态度和看法。你越是这样做，就越会得到对方的尊重和友善。

---

本节课中我们一起学习了“讨好型人格”的运作机制与改变路径。我们认识到，无底线的讨好源于**低自我认同**，它会导致自我折磨和关系失衡。改变的关键在于：**通过掌握技能建立自信 -> 通过小范围尝试学会拒绝 -> 最终敢于真实表达自我**。请记住，健康的爱源于相互吸引与尊重，而非单方面的讨好与妥协。
