当你尝试约喜欢的女生出来，她回应“我不想动，改天吧”时，这种回复通常源于以下三种原因。

以下是三种可能的原因：

1.  **委婉地拒绝你。**
2.  **出于矜持，不能痛快地答应。**
3.  **她当时真的不方便出来。**

后两种原因（矜持或确实不便）相对容易理解，我们在此不做赘述。

上一节我们列出了三种可能的原因，本节中我们来看看最需要你警惕和正确处理的第一种情况。

我们主要讲解第一种原因：**委婉的拒绝**。如何判断是否属于这种情况呢？关键在于观察你们平时的互动模式。

如果**在平时的聊天中，她对你的回应就表现得既不主动也不积极**，那么她说“改天吧”很可能就是在委婉地拒绝你。**聊天**是关系的基础，如果连线上交流都缺乏热情，就更难指望她会愿意线下赴约。

面对这种明确的拒绝信号，你有两个选择：要么调整心态，逐渐放下对她的期待；要么寻找合适的机会，彻底改变你在她心中的印象，**重新建立**吸引力和联系。
