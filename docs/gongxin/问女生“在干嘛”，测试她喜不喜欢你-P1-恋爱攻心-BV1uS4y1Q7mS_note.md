# 恋爱攻心课程 P1：通过“在干嘛”测试女生好感度 💬

在本节课中，我们将学习如何通过发送简单的信息“在干嘛”，来初步判断一个女生是否对你有好感。这个方法的核心在于分析对方回复的内容层次，从而解读其潜在的态度。

---

上一节我们介绍了本次测试的基本思路，本节中我们来看看具体的回复类型及其含义。女生的回复内容可以大致分为三个层次，每个层次都反映了不同的兴趣程度。

以下是三种典型的回复模式及其解读：

1.  **仅回复状态**
    *   如果女生只回复了当前在做的事情，例如“在上班”或“在休息”。
    *   这种回复仅表达了客观状态，没有延伸话题的意愿。
    *   **解读**：表明女生目前对你不太感兴趣，对话可能难以继续。

2.  **回复状态 + 情绪**
    *   如果女生在告知状态的同时，还分享了她的感受或情绪，例如“刚下班，好累啊”或“在看电影，挺无聊的”。
    *   这种回复在信息之外增加了情感层面的分享。
    *   **解读**：表明女生对你有一定的好感，愿意向你表露更多个人感受，为对话创造了深入的空间。

3.  **回复状态 + 情绪 + 反问**
    *   如果女生不仅分享了状态和情绪，还反过来询问你的情况，例如“我刚健身完，你呢？”或“我在吃饭，你吃了吗？”。
    *   这种回复包含了信息分享、情感连接和主动关注。
    *   **解读**：这是一个强烈的兴趣信号。因为她喜欢你，所以才会在分享自己的同时，渴望了解你的现状。

---

本节课中我们一起学习了如何通过“在干嘛”这条信息及其回复，来初步判断女生对你的好感度。关键在于分析回复内容是否包含**状态**、**情绪**以及**对你的反问**。记住，回复中包含的要素越多，通常意味着她对你的兴趣越高。这个方法可以作为初期判断的参考，但真正的感情建立还需要更多真诚的交流和相互了解。
