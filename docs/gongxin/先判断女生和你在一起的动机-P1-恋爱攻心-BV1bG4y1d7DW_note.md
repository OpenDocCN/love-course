# 恋爱攻心 - 课程P1：先判断女生和你在一起的动机 💡

在本节课中，我们将要学习如何判断一个女生与你建立关系的核心动机。理解动机是内在的“喜欢”还是外在的“条件”，是分析感情质量与未来发展的关键第一步。

## 核心概念：内在动机与外在动机

上一节我们介绍了判断动机的重要性，本节中我们来看看两种核心动机的具体表现。其根本区别在于，关系是源于对“人”本身的吸引，还是对“人所能提供之物”的吸引。

*   **内在动机**：女生是发自内心地喜欢和在乎男生这个人本身。
*   **外在动机**：女生是因为男生提供的钱、礼物、付出或生活条件等外在因素而选择在一起。

## 内在动机的表现

如果一个女生是发自内心地喜欢一个男生，她的情感状态和行为模式会有以下特征：

以下是内在动机驱动的典型表现：

*   **不自觉的想念**：她会不由自主地想念对方。
*   **渴望陪伴**：在看不到对方的时候，会想要对方的陪伴。
*   **在乎对方感受**：她会非常在乎这个男生是否想她、是否重视她、是否在意她。

## 外在动机的表现

相反，如果一个女生是基于外在动机选择在一起，她的情感投入方式将截然不同。

以下是外在动机驱动的典型表现：

*   **缺乏情感投入**：她可能并不真正喜欢这个人，甚至没有想要恋爱的“心动”感觉。
*   **动机错位**：令她心动的不是这个人，而是对方提供的礼物、转账等物质条件。
*   **关系疏离**：这是一种疏离的状态。她的选择和判断被外在动机影响，使得“在一起”更像一项不得不完成的任务，而非沉浸其中的享受。
*   **缺乏主动需求**：她不会在独处时强烈渴望你的陪伴，也不会产生“无论如何都想见到你”的冲动。

## 总结

本节课中我们一起学习了如何区分女生与你在一起的内在动机与外在动机。关键在于观察她的情感是否指向你“本人”：

*   **内在动机公式**：`喜欢 = f(你本身)`
*   **外在动机公式**：`在一起 = f(你的钱 + 你的付出 + 你提供的物质条件)`

理解这一点，有助于你更清醒地认知当前关系的本质，并为未来的决策提供重要依据。
