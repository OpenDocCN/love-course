## 核心概念分析

上一节我们介绍了课程主题，本节中我们来看看“老好人”行为模式的具体表现与后果。

如果你从一开始就对一个人特别好，对方最初可能会感激并试图回馈你。

**公式：** 初始行为（无限付出） → 初期反应（感激与回馈）

但时间久了以后，对方会逐渐将你的付出与好意视为理所应当。

**公式：** 时间推移 + 持续单方面付出 → 认知转变（好意变为“理所应当”）

当你的付出稍有减少或不如从前时，对方会立刻认为你变坏了，是个骗子，并将所有过错归咎于你。

**代码描述：**
```python
if 你的付出 < 对方的期望值:
    对方的反应 = “你是坏人/骗子”
    归因 = “都是你的错”
```

## 反向行为模式的对比

理解了单方面付出的弊端后，我们来看看另一种截然不同的行为模式会带来什么效果。

如果你最初对一个人态度冷淡、比较苛刻，甚至有些“坏”，而后才逐渐对他好起来。

此时，对方会感到非常感激。她会觉得之前受过的苦都不重要，重要的是你现在的转变，从而感到特别感动和满意。

**公式：** 初始行为（冷淡/苛刻） → 后续行为（转变为好） → 最终反应（高度感激与满意）

## “恃宠而骄”的心理机制

从上面的对比中，我们看到人性中存在一种普遍心理。本节将深入探讨“恃宠而骄”现象。

人很容易恃宠而骄。对方会将你最初的包容视为理所当然，一旦形成这种认知，后续便难以扭转局面。

以下是“恃宠而骄”后的典型表现：

*   你不理他，他就会生气。
*   你不能有脾气，你有脾气他也会生气。

## 关系维护的困境与策略

认识到问题后，我们自然会想，能否通过冷淡来处理这种关系？本节将探讨这种做法的可行性。

你可以选择暂时冷淡对方。因为长期单方面付出，你总有承受不起、感到疲惫的一天。

若你想维持这段关系，就需要半哄半迁就。当你感到累时，可以不说话、不找他。

但当你想要拉近关系时，你仍需要去哄他、迁就他。他会质问“你怎么不理我了？你对我不好了”。

此时你可能会解释：“我对你好的时候，主动靠近你的时候，你却是拒绝或想整我的态度。”

对方听后可能会想：“好吧，你说的也有道理。” 然后你再次进入迁就和哄他的循环。

这种模式持续久了，你同样会感到精疲力尽。
