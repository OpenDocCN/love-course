# 恋爱攻心课 P1：识别女生默认接受追求的迹象 👀

在本节课中，我们将学习如何通过观察女生的行为，来判断她是否默认接受你的追求。理解这些非语言信号，可以帮助你更好地把握关系升级的时机。

---

上一段我们概述了本节课的目标，本节中我们来具体看看女生默认接受追求的三个关键迹象。以下是具体的判断标准：

**第一，女生不拒绝单独邀约。**
女生通常只与朋友或有好感的人单独吃饭、看电影。如果她没有拒绝或排斥你的此类邀请，表明她愿意与你继续相处。

**第二，女生不拒绝频繁聊天。**
当女生对某个男生不感兴趣时，回复往往会很敷衍或间隔很久。公式可以表示为：`女生兴趣度 ∝ 回复积极性与速度`。如果她对你的频繁联系始终保持积极态度，这便是你可以尝试发出邀约的信号。

**第三，女生不拒绝你的帮助。**
这意味着她对你有一定的信任基础。如果她此后还经常主动寻求你的帮助，其潜在含义是希望与你有更多交集。这时，你可以考虑将你们的关系推向更亲密的阶段。

---

本节课中我们一起学习了识别女生默认接受追求的三大迹象：**不拒绝单独邀约**、**不拒绝频繁聊天**以及**不拒绝你的帮助**。准确识别这些行为信号，是把握恋爱关系发展节奏的重要一步。
