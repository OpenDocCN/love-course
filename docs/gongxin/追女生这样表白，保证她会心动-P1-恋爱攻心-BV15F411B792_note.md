## 核心表白话术结构

上一节我们提到了直接表白的弊端，本节中我们来看看具体的高情商话术应该如何展开。其核心结构包含三个步骤。

2.  **引发好奇**：在女生追问后，接着说：“我发现我对你的思想已经不纯洁了。”

## 话术原理分析

了解了具体步骤后，我们来看看这些话术背后的原理。它主要利用了两种心理效应。

以下是其发挥作用的两个关键点：
