这些女生的朋友圈内容，通常集中于展示身材与容貌。

她们也会展示名牌商品与豪华汽车。其日常行为模式可以概括为：化妆和拍照花费三小时，修图耗费整个下午，并为配文绞尽脑汁。她们每周日都会定点更新内容，其规律性甚至超过上班打卡。

那么，支撑这种行为的动力是什么？用通俗的话来说，这种行为可被称为 **“打理鱼塘，定时投喂”**。

当你去评论或私聊她们时，这个过程就进入了下一个阶段。
