# 恋爱攻心术 - 课程P1：口红策略的陷阱 🚫💄

在本节课中，我们将通过一个真实的故事，分析一种常见的恋爱“小心机”及其可能遭遇的反套路。我们将学习到，单靠技巧而缺乏真诚的策略，在经验丰富的人面前可能毫无作用。

---

我有一个闺蜜，长得特别漂亮。前段时间，她通过工作关系认识了一位成熟多金的大叔。在一次商务饭局结束后，大叔顺路开车送她回家。我闺蜜便使用了惯用的伎俩：故意把口红落在了大叔的车上。

回家之后，她给大叔打了个电话。

她说：“我的口红好像掉在你车上了。” 我闺蜜之前就用这种伎俩搞定了不少富二代。通常，对方要么亲自送过来，要么快递过来。无论哪种方式，最后我闺蜜都会提出请对方吃饭作为感谢。

然而这一次，情况不同了。

我闺蜜打电话给那位大叔说口红落车上了。大叔直接回复：“那你上我家来取吧。” 姑娘听了有点兴奋，找到我商量，问：“这会不会发展得太快了？” 我说：“会。” 但她还是把自己打扮得漂漂亮亮、花枝招展地去了大叔家。

结果，她一敲门，是一位阿姨开的门。大叔并不在家。阿姨说：“你是来拿东西的吧？” 然后指了指屋里面的一个个箱子，说：“你去那里找找吧。”

我闺蜜回来跟我讲，那些箱子就像一个公共场所的“失物招领处”。

以下是箱子里常见的物品类型：
*   各种女生的丝袜
*   口红
*   粉饼
*   首饰

这些东西都是不同的女生在大叔车上“落下”的。真是有人拼命制造机会，有人早已见怪不怪。

---

本节课中，我们一起学习了一个关于恋爱策略的故事。它告诉我们，**真诚远比技巧更重要**。试图用固定的套路去吸引他人，尤其在面对阅历丰富的人时，很可能弄巧成拙，让自己陷入尴尬的境地。在人际交往中，建立真实的情感连接才是核心。
