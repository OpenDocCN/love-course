# 恋爱攻心课程 P1：高情商开场白指南 🎯

在本节课中，我们将学习如何根据不同关系阶段，选择合适的高情商开场白。开场白本身并无绝对优劣，关键在于是否契合你与对方当前的关系状态。我们将通过具体案例，分析初识、熟悉、暧昧三个阶段的沟通特点与技巧，帮助你避免尴尬，有效开启对话。

## 开场白的核心原则：关系决定一切

在我看来，开场白没有绝对的优劣之分，只有是否适合。在使用某个开场白前，必须明确你与对方的关系阶段，并判断在此阶段发送此类信息是否合适。

例如，你给刚认识的女生发送“在干嘛”，对方可能会感到莫名其妙，认为你没有资格询问她的行程，从而不予回复。但如果你给正处于暧昧阶段的对象发送“在干嘛”，对方则可能心花怒放，认为你在想念她，从而迅速回复你。

在男女关系中，初识、熟悉、暧昧三个阶段对开场白的运用要求截然不同，且各有特点。

下面，我将为大家举例并详细说明。

## 阶段一：初识阶段开场白

初次聊天时，一个独特的开场白的确可以给女生留下深刻印象，但对于新手而言，其中的分寸很难把握。只要女生稍不按常理出牌，对话就容易陷入尴尬。

因此，我推荐的开场白相对中庸，但又不失小幽默。只要女生看到信息，通常都会回复，这样我们的初步目的就达到了。

以下是适用于初次聊天女生的开场白示例：

> **你好，有时间吗？我有个天，想和你聊一下。**

## 阶段二：熟悉阶段开场白

对于刚认识不久、关系尚不熟悉的女生，推荐使用以下开场白。它的优势在于可进可退，并暗藏小心机。

这个开场白既能表达对女生的关心，又能不经意地展示自己的能力。女生通常对会做饭的男生抱有天然好感。同时，它还是一个连环套路。

以下是具体开场白及后续思路：

> **我在做饭，你吃了吗？**

无论女生回答“吃了”、“没吃”或是夸赞你，你都可以顺势追问：“你喜欢吃什么？改天来尝尝我的手艺。”就这样，你便不动声色地完成了邀约。

## 阶段三：暧昧阶段开场白

对于正与你处于暧昧阶段的女生，只要你没有惹她生气或让她吃醋，你发送任何信息她基本都会回复。当然，若想让感情升温、关系更进一步，就需要刺激她的神经，调动她的情绪。

这种套路式开场白正好满足要求。它本身带有强烈的暧昧色彩，可以瞬间将女生的情绪调动到极致。

以下是适用于暧昧阶段女生的开场白示例：

1.  **我昨晚做了一个很可怕的梦。**
2.  **梦见你，居然是我女朋友。**
