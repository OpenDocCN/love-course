# 恋爱攻心 - 课程P1：识别“渣女”的核心特质 🕵️♀️

在本节课中，我们将要学习如何识别一种在亲密关系中常见的负面模式，其核心特点是极度的自我中心。我们将剖析这种行为模式的表现形式与内在逻辑，帮助你建立清晰的认知。

## 概述：自私是核心特点

“渣女”最大的特点就是自私。她们最厉害的地方，在于能将这种自私合理化。她们往往满口仁义道德，实则行为与之相反，是典型的伪君子。她们擅长打着“爱”的旗号，提出各种要求，例如“爱我就应该怎样怎样”。在她们眼中的爱情，本质上是**极致利己**的。

上一节我们概述了自私的核心地位，本节中我们来看看这种自私在具体关系中的两种主要表现形式。

## 自私在关系中的具体表现

以下是自私心态在恋爱关系中常见的两种要求，它们都服务于索取而非共同成长的目的。

1.  **要求无底线的包容**
    她们会要求对方无限度地容忍自己的所有缺点、错误或任性行为，将这视为“真爱”的证明。

2.  **要求无条件的付出**
    她们会要求对方不计代价、不问回报地满足自己的物质或情感需求，并将此与爱情绑定。

## 自私爱情的最终本质

无论表面形式如何变化，这种基于极致利己的爱情，其本质都是一场零和游戏。它并非为了构建互利共赢的亲密关系，而更像是一种短期交易。因此，这种关系模式注定难以长久和稳定。

**公式：自私的爱情 ≈ 短期交易**

本节课中我们一起学习了如何通过“极致利己”的核心特点来识别不良的恋爱模式。我们分析了其合理化的言辞、在关系中具体表现为无底线索取，并揭示了其短期交易的本质。理解这些特征，有助于我们在亲密关系中设立健康的边界。
