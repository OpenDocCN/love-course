## 核心误区：混淆心动与感动

追女生的一个致命误区，就是混淆了心动和感动的区别。

在传统思维下，男人总认为对女人好，坚持不懈，最后让她感动，就是唯一的追求方式。

然而事实是，现代的女生，往往喜欢让她心动的人，而不是感动的人。

## 心动与感动的不同来源

上一节我们介绍了心动与感动的核心区别，本节中我们来看看它们各自来源于何处。

你的吸引力来源决定了结果是心动还是感动。

*   **心动的来源**：当你的心思放在**提升自己**的身上时，你的**阳光帅气**、**自然幽默**、**踏实上进**，甚至是**冷酷叛逆**，都可以成为让她心动的点。公式可以概括为：**心动 ≈ 你自身的吸引力**。
*   **感动的来源**：而当你的心思放在了**讨好别人**的身上，一味委屈自己，满足对方，她除了感叹你是一个好人，并**不会产生真正的吸引力**。代码可以描述为：

        吸引力 = 0
