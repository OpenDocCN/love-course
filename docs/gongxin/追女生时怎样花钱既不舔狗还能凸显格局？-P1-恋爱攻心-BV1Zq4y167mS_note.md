## 一、💡 小钱大方

我曾看到过很多网友说，跟男生第一次约会，他居然要求AA，有的连瓶水都不舍得给女生买。更有甚者，死气白赖要请女生去吃饭，结果吃完之后，因为女生没有答应他的追求，背后中伤人家是“捞女”。最后的结果可想而知。

说这些案例是啥意思呢？想跟在座的小老弟说一句掏心窝子的话：千万别为了一点小钱去算计。一顿饭真的没有多少钱。哪怕见过一次面之后觉得不合适，就当交个朋友，以后有来有往，你也不会白白付出。

咱常说“绅士风度”，其实真的不是多大阵势上体现出来的，往往是细节见真章。像那种为了一顿饭毁坏女生名声的，殊不知，当你散布谣言的时候，毁的其实是你自己的形象。

**核心原则**：一个大方有度的男生，一定要懂得在小钱上大方一点。吃顿饭、送个小见面礼、约一次会，相信大家也不是承担不起。就算是恋爱不成，至少你在女生的心里留下了一个好印象。要知道，金杯银杯，都不如一个好口碑。

当你自己真正成为一个心胸宽广、大方有节的男生，你遇到的女生，也会是那种会主动要求分担、愿意主动买单，甚至为你买礼物的人。男生和女生的平等，不是说一定要花一样多的钱，而是说，谁都可以做那个主动付出的人，关键是看你值不值得。

## 二、⚖️ 有来有往

这世上真的有那种舍得为一个人一直投入的“傻瓜”吗？有。可是我真的见过，为了支持女生读书，挣钱给她付学费、生活费，然后女生读出来了，却提出分手。几年的付出全部打了水漂，关键是男生不仅没有索赔，还选择了成全。人家的话是：“你跟她要她也没有啊。”可能这就是传说中的痴情吧。

但是，严肃地说，这种行为并不值得膜拜。运气好的话，你能遇到一个对你也一心一意的女生，然后俩人相亲相爱。运气不好呢？你可能就被人玩弄于股掌之中，成为被索取、利用的那个人。运气好不好这种事儿，可没有那么多时间来验证。

所以，咱们小老弟得明白一个原则：**恋爱是两个人的事情，付出也要有来有往**。你可以给她买礼物，请她吃饭看电影，但是如果女生是有意愿跟你长期交往的话，她也是应该有付出的，这些事情女生也是可以做的。那如果她不想跟你长期交往，一点付出的意思都没有的话，那咱就没有必要彼此浪费时间了。
