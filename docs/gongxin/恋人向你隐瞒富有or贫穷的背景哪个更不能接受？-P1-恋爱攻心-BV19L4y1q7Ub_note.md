# 恋爱攻心课 P1：真实是恋爱的基石 💖

在本节课中，我们将探讨一个恋爱中的关键问题：应以何种面貌开始一段感情。通过分析“隐瞒背景”这一现象，我们将深入理解“真实”在建立健康恋爱关系中的核心重要性。

## 问题探讨：隐瞒富有与贫穷，哪个更不能接受？

曾经在《奇葩说》的舞台上有一个辩题：恋人向你隐瞒富有或贫穷的背景，哪个更不能接受？

针对隐瞒富有或者贫穷的问题，重点不在富有或贫穷，而在“隐瞒”。隐瞒的背后是另外的心思或目的。

如果一个人跟你接触不能用一种信任和坦诚的姿态，以后的生活又怎么可以保证呢？所以，这两种情况都不能容忍。出现这两种情况，无论出于什么原因，背后也有不信任、不自信的本质问题。

## 恋爱的正确开端：真实

宇哥今天提起这个问题，是因为想讲一个问题：我们应该以什么面貌去开始恋爱？思来想去，再根据这两天网上大家对于人性的讨论，只有两个字：**真实**。

宇哥教兄弟们在接触女生的时候，要适度包装自己，但是同时也不要成为一个虚伪的人。这看似很矛盾，今天就来讲清楚“真实”的含义。

### 一、包装的是外在，而非品行

兄弟们应该清楚，宇哥总是呼吁兄弟们提升自身价值，包括外在价值和内在价值。

以下是外在价值提升的核心方向：
*   **外表**：保持整洁、得体的形象。
*   **衣品**：学习穿搭，展现个人风格。

这不仅是包装自己，更是对跟我们约会的人的一种尊重。

对于内在的价值，我们要通过日积月累的提升去体现，不要急功近利，不然很容易让我们被冠上“虚伪”的称号。

例如，有的人家境一般，但是为了营造自己富二代的人设，各种装阔气，最后深入接触之后，发现这个谎圆不回来了，整个人的形象也就坍塌了。

### 二、不要硬凹人设

上一节我们讨论了外在包装的界限，本节中我们来看看如何设定个人形象。宇哥记得给兄弟们讲过立人设的问题。其实立人设不是不可以，但是我们要立与我们本人相近，或者我们通过改变可以做到的人设。

**核心公式：** 可持续人设 ≈ 真实自我 + 可实现的积极改变

太完美的人设其实就是不真实的表现，因为人本来就是不完美的。而且越完美的人设，完成起来就越困难，但是崩塌了却是直接粉碎。

所以宇哥建议兄弟们，不仅要以真实的面貌去开始感情，也要在这个过程中接受自己的不完美。当女生向你寻求问题的答案的时候，一定要实事求是、如实奉告，无论是你这个人，还是关于你这个人的事情。

```plaintext
坦然自信的真实感 -> 带给女生（安全感 + 踏实感 + 信任感）
```
甚至会让女生因为你的真实对你刮目相看。

### 三、做一个正直认真的人

什么人才会惧怕真实？大概是心里并不坦荡的人，才会怕别人掀出真相。宇哥希望每个兄弟，都是有勇气展示真实自己的人。

有些兄弟会问：如果女生问到了以前的事情怎么办？那时候做了一些不成熟的事情，说出来也会影响现在的形象。

对于这个问题，兄弟们倒是可以判断，对你提出问题的女生有没有分清楚过去和现在的能力。如果有但说无妨，然后讲完这话之后，你一定得从行动上让女生信任你，让她对你形成一种坚定的正面印象。

但是这也离不开兄弟们要做一个正直认真的人。无论是对待事情还是对待感情，态度一定要认真。女生最看重的也就是你的态度了。

## 课程总结

本节课中，我们一起学习了在恋爱中保持“真实”的重要性。我们探讨了隐瞒背景的本质问题，区分了适度的外在包装与虚伪的内在伪装，并强调了建立可持续的个人形象和保持正直认真态度的关键作用。记住，一段健康感情的开始，源于彼此的坦诚与信任，而这一切的基础，正是你敢于呈现的、真实的自己。
