### 步骤一：曲解与转移

**正确做法**：将“丑”这个主观评价，曲解成是客观物体“镜子”的问题。
*   **公式**：`女生提出自我否定` -> `将其归因于外部客观因素`
*   **话术**：“**一定是镜子的问题，我不信。**”

这句话既表达了不认同她的自我贬低（“我不信”），又用幽默（“镜子的问题”）避免了严肃争论，成功转移了“丑不丑”的矛盾点。

**正确做法**：提出一个带有亲密想象空间的提议。
*   **公式**：`否定客观事实` -> `提出主观的、私人的视角`
*   **话术**：“**不信的话，你来看看我眼中的你。**”

“我眼中的你”是一个充满暧昧感的表述，它暗示了“在我眼里你是美的”，并且将两人拉入一个排他的、想象性的私密空间，极大地提升了情感张力。
