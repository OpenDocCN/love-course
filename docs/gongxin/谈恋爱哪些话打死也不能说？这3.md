# 谈恋爱哪些话打死也不能说？这3

在本节课中，我们将要学习在恋爱关系中需要极力避免的三类话语。这些话语看似平常，甚至有时被误认为是亲密的表现，但实际上它们会严重损害双方的感情，让你的异性缘越来越差。理解并避免这些沟通雷区，是建立健康恋爱关系的第一步。

## 第一类：贬低对方的话 👎

上一节我们介绍了本节课的主题，本节中我们来看看第一类需要避免的话：贬低对方的话。有些人认为关系亲密后，说话就可以随意，甚至用贬低、嘲讽的方式来表达。例如：“瞅瞅你的小肚子，你都快赶上人家怀孕五个月的了，你赶紧减点肥吧。” 或者“你看你这牙黄的，咋的，你买不起牙膏是吧？” 这类话语听起来非常伤人。

在恋爱中这样说话，很可能瞬间被对方拉入黑名单。说这些话的人，很多时候并非心存恶意。他们可能从小缺乏被夸奖的经历，很少听到善意的话语，因此不习惯用积极的方式表达，反而习惯了这种“互损”的交流模式。然而，这种模式确实不招人待见。

**核心要点**：在与异性沟通时，必须学会克制。应尽量减少对对方的负面评价，转而努力寻找并夸奖对方的优点。

**沟通公式**：
`有效沟通 ≈ 积极关注 + 善意表达 - 贬低嘲讽`

## 第二类：冷暴力式的话 ❄️

了解了贬低话语的危害后，我们接下来看看另一种极具破坏性的话：冷暴力式的话。这类话语通常表现为回避问题、拒绝沟通。例如：“你要这样想，我也没办法。”、“我在忙呢，你别瞎想行不行。” 或者干脆只回复几个句号“。。。”。

许多人在遇到问题时，自己懒得解决，也不愿面对伴侣的负面情绪，于是选择搁置。但问题在于，如果负面情绪不被处理，矛盾只会升级。对方可能因此失眠、焦虑，最终关系恶化，甚至被直接“拉黑”，失去沟通的机会。

**核心要点**：与其回避，不如直面问题。宁愿大吵一架，把问题吵明白，让双方心里没有怀疑和埋怨，也比冷处理让问题发酵要好。

**行为代码**：
```python
if problem_arises:
    # 错误做法：冷处理
    # response = “...”
    # 正确做法：积极沟通
    response = initiate_constructive_talk(problem)
```

## 第三类：翻旧账的话 📜

上一节我们探讨了冷暴力的危害，本节我们来分析最后一类需要避免的话：翻旧账。这指的是在发生新矛盾时，不断提起过去已经发生且未妥善解决的旧问题。

例如，一对情侣吵架，男生抱怨女生不关心自己，女生则反击说：“你关心过我吗？我之前生病的时候，你也没怎么样啊……” 这种翻旧账的行为，往往比吵架本身更伤人。它会让对方感到过去的错误被无限放大，问题永远无法真正翻篇，从而对关系感到绝望。

人与人相处不可能永远没有矛盾。遇到问题，应该就事论事，当下解决。如果总是把情绪和旧账捂着，累积到一定程度再爆发，局面将更加难以收拾。

**核心要点**：养成“当下问题，当下解决”的习惯。避免让过去的阴影持续笼罩现在的关系。

**沟通原则**：
`解决当前矛盾 ≠ 清算历史总账`

---

本节课中我们一起学习了在恋爱沟通中必须避免的三类话语：**贬低对方的话**、**冷暴力式的话**和**翻旧账的话**。记住，良性的沟通需要善意、直面问题的勇气和就事论事的态度。克制这些破坏性的话语，多给予对方肯定和积极的回应，才能让关系健康、长久地发展。
