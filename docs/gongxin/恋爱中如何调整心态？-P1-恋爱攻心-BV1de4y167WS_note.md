# 恋爱心态调整指南 - 课程P1 🧠

在本节课中，我们将学习如何在恋爱关系中调整心态，核心在于管理期望、聚焦积极体验，以及在必要时果断行动，以保护自身情绪健康。

上一段我们提到了调整心态的核心理念，本节中我们来看看具体如何操作。

## 核心原则：接纳而非改变 ❌

永远不要试图改变一个人。在相处过程中，我们可能会遇到各种体验。

以下是处理不同体验感的具体方法：

*   **聚焦积极体验**：只去体验和回味那些好的体验感。
*   **搁置消极体验**：将不好的体验感暂时搁置一旁，不去触碰。
*   **主动规避**：尽量绕开可能引发不好体验感的话题或情境，不去提、不去问、不去想。

这个原则可以总结为一个简单的公式：**心态 = 关注（积极体验） + 搁置（消极体验）**。

## 管理期望，避免内耗 📉

基于上述原则，我们需要管理自己的期望。这意味着：我只要我可以要的，不去强求我得不到的。

强求得不到的东西，容易产生心理落差，公式表示为：**心理落差 = 期望值 - 现实值**。过大的落差会直接影响你的情绪。

## 果断行动：设置情绪边界 🛡️

如果一段关系或某个人的行为持续影响到你的情绪，这是一个关键信号。

> 当这个人让你开始消耗自己时，你需要采取行动。

最直接有效的行动就是：赶紧与这个人分开，进行断舍离。这是保护自己情绪健康的必要措施。

---

本节课中我们一起学习了恋爱心态调整的三个关键步骤：**接纳对方、聚焦积极体验、管理期望**，并明确了当关系开始消耗自我时，应果断**断舍离**。记住，健康的恋爱关系是相互滋养，而非单方面消耗。
