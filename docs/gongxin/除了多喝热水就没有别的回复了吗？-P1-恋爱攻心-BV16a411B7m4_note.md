## 为什么“多喝热水”是糟糕的回复？

上一节我们介绍了课程主题，本节中我们来看看为什么“多喝热水”这个回复不受欢迎。

“多喝热水”这个回复是全国直男的统一回复。
这个回复显得非常敷衍。
女生自己当然知道可以喝热水，她向你倾诉的目的并非寻求解决方案。

理解了问题所在后，我们来看看女生此刻的真实需求。

女生在表达不适时，其状态像一只受伤的小兔子。

那么，具体应该如何回复呢？以下是两个步骤的回复策略。

**第一步：表达关心与主动照顾的意愿**
你可以对她说：“你会不会照顾自己？不会的话让我来。”

**第二步：进行轻松、有趣的互动**
她通常会反问：“你要怎么照顾我？”
此时，你可以用开玩笑的方式回复，例如：
*   “一个‘摸摸头’照顾三天。”

这种回复方式将关心的意图，通过幽默和情感互动表达出来，远比干巴巴的建议更有效。
