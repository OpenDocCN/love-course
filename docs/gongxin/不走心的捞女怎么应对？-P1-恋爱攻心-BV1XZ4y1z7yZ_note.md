# 恋爱攻心 - 第1课：如何应对不走心的捞女？ 💔

在本节课中，我们将学习如何识别并应对那些在关系中只索取、不走心的对象。核心策略是调整你的应对方式，从期望情感回报转变为进行清晰的价值交换。

---

上一段我们提到了应对这类人的核心思路。接下来，我们具体看看操作步骤。

以下是应对此类情况的具体行动指南：

*   **直接提出条件**：当对方提出需要你买单等需求时，你可以直接提出你的条件。例如，公式为：**对方的需求 → 你的明确条件（如“今晚去某某地方”）**。
*   **停止无效沟通**：无需进行冗长的理论或情感交流。如果对话无法达成共识，就应果断结束。代码逻辑可表示为：
    ```python
    if 无法达成共识:
        结束互动
    else:
        继续愉快交流
    ```
*   **采取对等策略**：对方是什么样的人，就采取什么样的对策。既然对方不谈感情、不重视你的付出，那么你就应该切换到“交换”模式，与他进行交易。
*   **明确表达交易意图**：可以坦率地告诉对方你的想法。例如：“**你需要什么我可以提供，而我想要的是你，是否可行？**”

---

本节课中，我们一起学习了应对“捞女”或类似不走心对象的策略。关键在于认清对方本质，放弃不切实际的情感期待，转而采用清晰、直接的价值交换方式进行互动。这能有效避免内耗，保护你的时间和精力。
