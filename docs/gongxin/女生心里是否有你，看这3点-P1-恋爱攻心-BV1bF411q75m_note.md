上一节我们介绍了本课程的目标，本节中我们来看看第一个核心判断点。

## 1. 主动联系的意愿

心里有你的女性，只要她有时间，通常会主动联系你。心里没有你的女性，即使你一夜不睡，也等不到她的一个消息。

这个行为的核心逻辑可以概括为一个简单的条件判断：

```python
if she_has_time and she_cares:
    she_will_initiate_contact()
else:
    # 等待可能无果
    your_waiting_may_be_in_vain()
```

主动联系是投入时间和关注度的直接体现，是判断兴趣的基础指标。

了解了联系频率后，我们接下来看看她对你的付出作何反应。

## 2. 对你付出的反馈

以下是区分对方反馈积极与否的关键表现：

*   **心里有你的表现**：你只要对她好一点，她都能开心够半天。你的付出会被她珍视并带来显著的愉悦情绪。
*   **心里没有你的表现**：你为她付出再多，她都不会满足，还会觉得这是你理所应当的。她的反应表现为**索取 > 感激**。

这种反馈差异，本质上体现了情感账户的平衡公式：**你的付出 × 她的珍视系数 = 关系价值**。当珍视系数接近于零时，关系价值便无法增长。

观察了付出与反馈的互动后，最后一个要点关乎冲突处理模式。

## 3. 发生冲突时的处理方式

在发生分歧或争执时，对方的行为是重要的试金石。

*   **心里有你的女性**：会偶尔忍受你的坏脾气，因为她知道你只是一时的冲动，一会儿就好。她愿意为维护关系提供一定的容忍度。
*   **心里没有你的女性**：根本不会顾及你的面子和处境，只会一味地跟你吵闹。她的目标是“赢得争吵”而非“解决问题”。

这种行为区别的关键在于，前者在冲突中仍保有**关系维护意识**，而后者则完全将其抛诸脑后。
