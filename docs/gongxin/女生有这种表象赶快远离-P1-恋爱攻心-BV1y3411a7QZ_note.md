## 五种需警惕的表现

以下是五种关键的行为表现，如果出现，建议你重新评估这段关系。

1.  **消息回复延迟且不对等**
    你发消息给她，她总是过很久才回复。你清楚她不忙，但她就是拖延回复。她给你发消息时，你总是秒回；而你给她发消息，她却轮回。

2.  **缺乏主动的互动发起**
    每次你和她聊天感觉都不错，但每次对话都是由你主动发起。她从来不会主动找你聊天。即使你尝试不找她，测试她的主动性，结果她也真的不会找你。

3.  **情感回馈如同“棉花”**
    她和你聊天时，永远一是一、二是二。她绝对不会对你撒娇卖萌，也不会对你生气发火。她对你仿佛没有任何情绪，但又非冷漠如墙，而是像一团**棉花**，让你有力使不出。无论你如何努力，也无法引起她的情绪波动，这会让你产生强烈的挫败感。

4.  **敷衍的对话方式**
    每次聊天，她都使用“嗯”、“哦”、“对呀”、“行呀”等词语回应，且回复通常不超过三个字。这迫使你需要不断寻找新话题，疲于奔命。

5.  **社交媒体的隔离**
    她在朋友圈屏蔽了你，留给你的只是一条横线。这就像你们之间永远隔着一条无法跨越的鸿沟。
