# 恋爱攻心 - 课程P1：理解供养者与情人 🧠💔

在本节课中，我们将要学习恋爱关系中的两个核心概念：“供养者”与“情人”。理解这两个角色的本质差异，是避免在感情中陷入被动地位的关键。

## 核心概念解析

许多人认为，女生在学生时期不在意男生的经济条件，而进入社会后则变得特别看重。这是一种错觉。女性对待金钱的态度始终一致，区别在于她们对“供养者”有着严格的经济资源要求，而对“情人”则没有。

以下是这两个核心概念的定义：

*   **供养者**：指主要提供**生存资源**（如金钱、物质保障）的角色。其核心公式可概括为：**吸引力 ≈ 财富资源**。
*   **情人**：指主要提供**优质基因**（如外貌、身材、魅力、情绪价值）的角色。其核心公式可概括为：**吸引力 ≈ 基因价值**。

简单来说，供养者“有钱”，而情人“有颜”（这里的“颜”泛指吸引力）。

## 社会现象与错觉

上一节我们介绍了供养者与情人的定义，本节中我们来看看它们如何体现在社会现象中。

校园里，大部分男生尚未积累财富，因此女生恋爱的标准基本偏向于颜值高、身材好的“情人”型男生。进入社会后，部分男生开始凸显财富优势。当前主流的社会现象是，有钱人似乎从不缺少女性陪伴。

这同样是一种错觉。许多失败者将自己单身的原因归结为“没有钱”，却不知道许多有钱人也仅仅扮演着“供养者”的角色。

## 供养者的困境

供养者虽然表面风光，身边美女环绕，但往往有苦难言。他们心知肚明，身边的女性只是贪图其提供的资源，并非真心喜欢他这个人本身。

供养者非常清楚，一旦自己落魄，身边的女性会立刻离开。当一个男人的基因吸引力（作为情人的潜质）不足时，他只能依靠外在的财富资源来博取女性青睐。反之，当一个男人的基因吸引力足够强时，女性甚至不会要求他的财富，还可能主动付出。

你们可以思考一下：为什么富婆偏爱找“小白脸”，而不是“油腻大叔”？这正是“供养者”角色的悲哀之处。

## 课程总结

本节课中，我们一起学习了恋爱关系中的“供养者”与“情人”概念。关键点在于：**女性对供养者有经济要求，对情人则没有**。许多看似成功的“供养者”关系，其内核是资源交换而非情感吸引，因此基础脆弱。理解这一点，是建立健康、平等恋爱关系的第一步。
