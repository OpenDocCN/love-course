## 一、互动游戏：烧大骨 🔥

首先，我们来看一个需要简单道具的互动游戏。它能制造紧张有趣的氛围，并自然地引入惩罚机制，增加互动趣味性。

**所需道具**：纸巾、杯子、打火机、骰子。

以下是游戏步骤：

1.  用水打湿杯子杯口。
2.  用一张纸巾盖住杯口。
3.  湿杯口会在纸巾上印出一个圆形水痕，撕掉圆形以外的多余纸巾。
4.  将一颗骰子放在盖好纸巾的杯口上。
5.  用打火机点燃纸巾，随后吹灭。
6.  **游戏规则**：轮流操作，在燃烧纸巾时，需确保骰子不掉入杯中。轮到谁时骰子掉入，谁就接受惩罚。
