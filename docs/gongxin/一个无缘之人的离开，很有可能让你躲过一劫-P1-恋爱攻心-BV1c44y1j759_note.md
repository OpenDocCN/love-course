在本节课中，我们将要学习一个重要的情感课题：当一段关系持续消耗你的能量时，如何识别并果断地做出“断舍离”的决定。这并非关于对错，而是关于保护自己的情绪健康与生活状态。

## 识别消耗型关系 🚨

上一节我们介绍了课程的主题，本节中我们来看看如何识别一段关系是否正在消耗你。

如果一位女性总是影响你的情绪，让你茶饭不思，并且已经严重影响到你的日常生活和工作状态，那么你应该果断离开。

## 理解“当断则断”的智慧 ⚖️

理解了问题的表现后，我们需要深入其核心原则。

“当断不断，必受其乱”。对于无能为力改变的事情，应当决断；对于生命中无缘继续同行的人，应当舍弃；每当产生偏执的执念时，应当离开。

很多时候，人若想要获得真正的成长与蜕变，就必须实践“断舍离”。

## 重新定义“离开”的原因 🔄

明确了原则，我们接下来重新审视“离开”这一行为的本质。

离开他，并不是因为他不好，而是因为你和他在一起时，你的心态会变得不好。

不必去惋惜错过他可能会留下遗憾，因为这很有可能是让你躲过了一劫。
