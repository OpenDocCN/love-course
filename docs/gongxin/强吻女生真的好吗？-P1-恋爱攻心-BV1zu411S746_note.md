## 核心原则：行动优于言语

上一节我们介绍了课程主题，本节中我们来看看表达亲密意图的核心原则。关键在于用行动代替直接的言语询问，通过观察对方的身体反应来判断其真实意愿。

以下是两个需要避免的常见错误及其正确做法：

1.  **关于亲吻**
    *   **错误做法**：直接询问“我能亲你吗？”或强行亲吻。
    *   **正确做法**：自然地靠近对方，然后停下来，将主动权交给对方。女生的身体反应非常诚实。
        *   **公式**：**你的行动（靠近 + 停顿） → 观察她的反应 → 得到明确信号**
        *   如果她对你有好感，可能会主动完成亲吻。
        *   如果她下意识地躲避或后撤，则表明她目前不感兴趣。

2.  **关于确立关系**
    *   **错误做法**：直接询问“你能做我女朋友吗？”或进行正式表白。
    *   **正确做法**：通过自身魅力吸引对方，然后适当减少主动联系的频率，测试对方的兴趣度。
        *   **代码逻辑**：
            ```python
            if 她喜欢你:
                她会主动找你
            else:
                她对你兴趣不足
            ```
        *   如果她对你有意，她会主动增加与你的联系。
        *   如果她反应冷淡、爱搭不理，则意味着吸引力不足。

## 遇到困难怎么办？

我们已经学习了通过观察与测试来识别女生兴趣的方法。如果你发现对方反应不佳，意味着当前策略需要调整。

如果你遇到女生对你明显不感兴趣或态度冷淡的情况，可以进一步学习专门的应对策略。
