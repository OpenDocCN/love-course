# 恋爱攻心 - 课程P1：自信是基石 💪

在本节课中，我们将要学习自信在情感关系中的核心作用，并探讨其来源与本质。理解自信并非后天赋予，而是如何被外界因素所影响，是建立健康心态的第一步。

## 自信是人的本能

如果一个人的自信被完全牢牢地控制住，被牢牢地封闭起来，那这个人是没有办法活在这个世界上的。他不仅会自闭，还会非常抑郁和自卑。

其实每个人生下来都是自信的。关于“自信来自于原生家庭”这个说法，需要更准确地理解。

## 自信的来源与束缚

并不是家庭给了这个小孩自信。正确的说法是，这个家庭没有遏制住这个小孩的自信。也就是说，人本身就是自信的，他本身就有自信的本能和天性。

但是，因为原生家庭的一些束缚，让一个人的自信——即一个人的本我——越来越受局限，越来越受一些条条框框的安排和设置。这让你的观点发生了转变，变得没有那么开拓和自由。

## 失去自信的后果

于是，你就变得小心翼翼，不再勇敢。失去自信的过程，同时也是失去勇敢的过程。勇敢的做自己，勇敢地去说，勇敢地去做。

因此，总结来说，不是一个人的原生家庭给了他自信。我们应该说，如果一个人是自信的，那意味着他的原生家庭没有对他的自信造成太多的束缚和影响。

---

本节课中，我们一起学习了自信作为人类天性的本质。我们明白了自信的缺失往往源于后天环境的束缚，而非先天不足。认识到这一点，是摆脱束缚、重拾勇敢做自己的第一步。在后续课程中，我们将探讨如何在实际互动中培养和展现这种自信。
