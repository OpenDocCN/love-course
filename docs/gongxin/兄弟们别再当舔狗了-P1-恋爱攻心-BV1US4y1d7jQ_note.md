# 恋爱攻心课程 P1：告别“舔狗”行为 🚫

在本节课中，我们将要学习为什么在感情中应避免成为所谓的“舔狗”，并探讨如何建立健康、平等的恋爱关系。

---

首先，我们要弄清楚的是，为什么我们喜欢当舔狗。看到这句话的时候，很多兄弟心里都默默地冒出了一句：如果不是真的喜欢，谁愿意当舔狗。这句话听起来似乎非常深情。喜欢的方式有千万种，为什么偏偏要选择去当舔狗？

爱一个人就要让自己显得如此廉价、如此卑微、如此不堪吗？喜欢是人正常的情愫，没有高低贵贱之分。不能因为你更喜欢了，你在感情中的地位就低了。这不是一段正常的感情应该有的态度。

除了舔狗本身就不是一件正确的事情之外，还有什么理由能让我们拒绝当舔狗呢？在这里我们就有必要来看一个问题：女生是怎样看待舔狗的？

泽哥之前在咖啡厅听过两个女生很有意思的对话。A说：“我想跟我男朋友分手了。”B问：“为什么？难道他对你不好了吗？”A回答：“不，就是因为太好了，什么事都顺着我，我越来越觉得没意思。”

看吧，兄弟们。为什么说不让大家在感情中去当一个舔狗？除了对感情的推进无益之外，女生也是非常不待见舔狗的。相比于舔狗，她们更喜欢有个性、能带领她们的男人。所以希望有舔狗倾向的兄弟们注意你们的行为。如果你一味地舔狗，女生只会越来越厌烦你、害怕你，甚至不想见到你。

有兄弟问我说：“泽哥，我也知道追女生不能一味地舔狗，但是就是忍不住去舔，怎么办？”有些兄弟受成长环境的影响，从小在否定的声音中长大。那种自卑心理引导出的行为和讨好型性格，往往是不意识的。

上一节我们介绍了“舔狗”行为不受欢迎的原因，本节中我们来看看如何避免成为“舔狗”。

以下是三个核心方法：

**一、下意识地拒绝**
如果偶尔女生向你提出求助，在不是必要的情况下，你要学会下意识地拒绝她，不要每次都一口答应。记住，这不是抬杠，这是在塑造你的性格和人设。不仅是对于你喜欢的女生，对你身边的朋友家人也可以这样。通过这种方式，来锻炼你的独立人格。

**二、聊天适当不要秒回，字数尽量别多于对方**
女生偶尔主动地跟你沟通一次，你就亢奋不已、主动表达，滔滔不绝地对女生信息轰炸。太多主动的后果往往都直接导致女生对你失去兴趣。两性吸引是一个玄妙的过程。相互吸引、相互引诱的恋爱，才会让你们的感情妙趣横生，更加持久。

**三、不要去猜她想什么**
因为很多时候，女生自己也不知道自己在想什么。女生的情感世界通常比男人更丰富、更细腻。她们也许会因为双十一没抢到想要的化妆品而难过，也有可能看一部电视剧有一个甜蜜的情节而心花怒放。她们的情绪来得快也去得快，虽然偶尔捉摸不透，但是很大可能不会是因为你。

所以男人千万不要被女生琐碎的情感牵着鼻子走。经常去猜测女生的心思，你会变得患得患失、紧张兮兮。这种不自然的精神状态会影响到你，低说话以及行为也会给女生留下不好的印象。

只有把自己放在一个平等的角度上，才能在此基础上建立女生对你的喜爱，展现出你的魅力。一个天生就看低自己的男人，是无法吸引到女生的。

---

本节课中我们一起学习了“舔狗”行为的弊端，并掌握了三个避免该行为的具体方法：**学会拒绝**、**控制聊天节奏**以及**停止过度揣测对方心思**。核心在于建立平等、自信的互动关系，这才是健康感情的基础。
