# 恋爱攻心 P1：何时表白，十拿九稳？ 💘

在本节课中，我们将要学习如何判断向心仪女生表白的恰当时机。我们将通过一个生动的比喻，并引入一个关键的心理学概念——“暧昧窗口期”，来帮助你识别那些表明时机已到的具体行为信号。

追女生就像打游戏。女生能够跟你聊两个星期，这代表你有参加比赛的资格，不代表你有直接能力推水晶。

我们要做的，就是先慢慢地推倒防御塔。这些防御塔在心理学上叫做“暧昧窗口期”。

上一节我们介绍了“暧昧窗口期”这个核心概念，它就像游戏中的防御塔，需要逐一攻克。本节中我们来看看，如何判断这些“防御塔”是否已被成功推倒。以下是判断“暧昧窗口期”是否成熟的七个关键行为指标。只要发生其中四条以上，你的告白便是十拿九稳。

*   行为一：她开始主动找你聊天。
*   行为二：她会和你分享生活中的细节和心情。
*   行为三：她愿意接受你单独的约会邀请。
*   行为四：约会时，她不排斥与你有轻微的肢体接触。
*   行为五：她会在你面前表现出脆弱或需要帮助的一面。
*   行为六：她开始关心你的日常生活和情感状态。
*   行为七：你们之间有专属的昵称或内部玩笑。

本节课中我们一起学习了表白的时机判断。核心在于理解并识别“暧昧窗口期”的成熟标志，即女生表现出多条上述的亲近行为。记住，耐心推进关系，观察这些信号，当时机符合多条指标时，你的表白成功率将大大提升。
