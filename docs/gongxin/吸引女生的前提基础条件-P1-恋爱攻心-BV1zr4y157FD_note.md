## 吸引的前提：关注

你们觉得一个女生被你吸引的前提是什么？答案是**关注**。只有一个女生把注意力放在你身上时，她才有可能被你吸引。她的注意力放在你身上的时间越多，她被你吸引的可能性就越大。

那么，如何才能有效地引起女生的注意呢？

## 核心策略：先主动，再停止

答案是**先主动再停止主动**，也就是我们常说的“欲擒故纵”。

一说到主动，很多人就会认为是“暴露需求感”。（关于需求感的问题，我们会在后续课程中专门讲解。）在这里，我需要明确告诉你：**主动和暴露需求感是两码事**。

如果你不主动，那你怎么和女生产生交集？你和女生就像是两条平行线，你不理她，她不理你，最后你们就是两个陌生人。

所以，真正懂得主动的男生是不会暴露需求感的。主动的核心是**明目张胆地释放好感**。只有那些“猥猥缩缩”、想爱又不敢爱的人才会暴露需求感。

真正厉害的人就是敢爱敢恨。我喜欢你，我就直接告诉你，仅仅只是告诉你而已。至于你喜不喜欢我，不重要。更不会去追问女生“愿不愿意做我女朋友”。只有“舔狗”才会在表达了喜欢之后，向女生索取一个答案。

## 表达喜欢的原理与作用

那为什么要表达喜欢呢？因为人都会对喜欢自己的人产生好感。

如果你对一个女生表达了喜欢之后，又没有后续实质性的行动，女生就会琢磨：“你是不是在调戏她？”当她开始琢磨你的想法时，她的注意力自然就会放在你的身上。

**注意力在你身上 = 具备吸引她的基础条件**
