# 恋爱攻心课程 P1：识别女生好感的六大暗示 🔍

在本节课中，我们将学习如何识别女性表达好感和亲密意愿的六种常见行为暗示。这些信号有助于你更好地理解对方的意图，把握关系发展的时机。

上一节我们介绍了课程的主题，本节中我们将逐一详细解读这六个核心暗示。

以下是识别女生好感的六大具体行为表现：

1.  **深夜让你送她回家**：在深夜时分，她主动提出或接受让你单独送她回家，这通常意味着她信任你，并可能希望有更多独处时间。
2.  **不介意共饮**：她不介意与你共用一杯水或饮料，这超越了普通的社交距离，表明她对与你间接亲密接触没有排斥感。
3.  **约会后拖延时间**：在约会结束时，她表现出迟迟不肯离去或找理由延长相处时间的行为，公式可表达为 **`约会结束意愿 = 低`**。
4.  **故意在你面前喝醉**：她主动邀你喝酒，并在你面前表现出“喝醉”的姿态，这可能是在创造需要你照顾或关系更近一步的情境。
5.  **近距离交谈**：与你说话时，她的身体下意识地靠近，远小于正常的社交距离（通常小于0.5米），这属于亲密空间的主动侵入。
6.  **眼神充满热情**：她看你的眼神直接、专注且带有情感，即所谓“火辣”或“含情脉脉”，这与普通友好眼神的区别在于 **`强度与持续时间 > 常规值`**。

本节课中我们一起学习了女性表达亲密意愿的六种关键行为信号。理解这些暗示有助于你在互动中做出更恰当的回应。记住，结合具体语境综合判断至关重要，单个信号需放在整体互动模式中评估。
