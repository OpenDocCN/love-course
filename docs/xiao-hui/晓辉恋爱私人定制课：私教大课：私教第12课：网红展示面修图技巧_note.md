在本节课中，我们将要学习网红展示面的高阶修图技巧。修图是展示面建设中的硬核技能，能够将你最好的一面呈现在社交软件上，有效提升吸引力。我们将从修图的重要性讲起，逐步拆解人像、全身照、静物及人景综合等不同照片类型的修图方法，并手把手演示核心修图流程与软件使用。

## 修图的重要性

很多人思维尚未转变，认为修图不真实，担心修图后会引起他人反感。这种想法没有与时俱进。现在很多女生自己也会修图，爱美之心人皆有之。修图的关键在于合理适度，而非过分失真。

修图之所以重要，是因为展示面的核心是展示我们最吸引人的一面。这就像餐馆菜单上的菜品图片都经过精心处理，看起来非常美味，虽然与实际端出的菜品有差距，但顾客通常能够理解并接受。社交软件上的个人形象也是如此，合理的修图处理是普遍且被接受的。

以下是两个反面教材，说明了修图过度的问题：
*   一位女士修图后与真人差距过大，变成了“照骗”。
*   另一位女士修图后形象变化巨大。

当你掌握了修图技巧后，你也能更好地识别他人的修图程度，从而判断其真实性。

## 高阶修图的核心板块

上一节我们探讨了修图的重要性，本节中我们来看看高阶修图具体涵盖哪些类型的照片处理。我们的修图教学主要分为以下几个板块：

### 1. 人像修图
人像修图主要处理以人物面部为主的照片，旨在优化神态、肤质和五官等细节。

以下是修图案例：
*   **学员案例一**：通过修图，调整了眼睛神态、添加滤镜、处理了头发、眉毛和脸型，并添加了轻微虚化。修图后的人物更帅气且适合用作头像，符合展示面“拍得帅、衣着有逼格、背景有质感”的要义。
*   **学员案例二**：修图时处理了腹部轮廓（同时确保手中杯子不变形），并调整了整体颜色色调。

人像修图可分为半身像和大头照两种。

### 2. 全身照修图
全身照修图侧重于人物整体与环境的协调，以及氛围的营造。

以下是修图要点：
*   **案例展示**：以2014年的展示面为例，通过添加暗角来提升照片质感，使其更适合在社交APP上长期使用。
*   **滤镜应用**：不同风格的背景和穿衣搭配需使用不同的滤镜，并非千篇一律。例如，可以通过调整色调营造温暖或冷峻的感觉。

### 3. 静物修图
静物修图主要针对食物、物品等，核心在于颜色调整。

以下是修图方法：
*   主要通过调整颜色和添加边框来提升质感。例如，为食物照片调色并添加白边是一种曾流行的玩法，但并非所有照片都需要加边框。

### 4. 人景综合修图
这是最难的一种修图类型，需要兼顾人物、景物、光线和颜色的整体协调。

以下是修图难点与案例：
*   **难点**：需要考虑全局，在修饰人物时不能影响其他物体（如导致变形）。很多人缺乏全局思维，只修脸部而忽略其他部分。
*   **案例一（社交认证照）**：调整了曝光，确保人物好看、衣物和食物都有质感。前期拍摄好是后期修好的基础。
*   **案例二（导师照片）**：通过添加虚化效果，使照片更有意境。对比原图，质感提升明显。
*   **案例三（风景人像）**：调整了整体颜色，对景物局部做了清晰化处理，并优化了人物面部轮廓和侧脸，使整体画风协调。

## 核心修图软件介绍

了解了不同类型的修图后，我们需要借助专业的工具。以下是课程推荐的修图软件及其核心用途：

*   **美图秀秀**：主要用于**去瑕疵**（如去痘）。
*   **Fotor**：主要用于**添加边框**，但并非所有照片都需要。
*   **MIX**：主要用于**添加滤镜**，尤其适合风景类滤镜，功能丰富。
*   **Snapseed**：**调色功能非常强大**，是网红和许多女生常用的软件。
*   **天天P图**：核心功能是**给人像化妆**，效果自然，还能进行抠图虚化。
*   **VSCO**：提供类似**Instagram风格**的滤镜，很受欢迎。
*   **Facetune**：专业用于**修人像**，能精细处理身形和塑形，非常可靠。
*   **美咖相机**：针对**面部进行精细处理**，例如自然提亮眼神。
*   **无他相机**：主要用于**自拍**。

## 修图流程详解

下面我们将进入实战环节，详细讲解两种修图流程。

### 基础修图流程（四步法）
适用于静物或人物距离较远的全身照。
1.  **美图秀秀去瑕疵**。
2.  **Snapseed调颜色**。
3.  **VSCO加滤镜并二次调色**（在滤镜基础上微调，增加质感）。
4.  **添加边框**。

### 高阶精修流程
适用于需要精细处理的人像或人景综合照。
1.  **美图秀秀去瑕疵**。
2.  **Facetune调整人脸**（修饰小瑕疵，如眼角、眉毛，而非改变脸型）。
3.  **Snapseed调颜色**。
4.  **VSCO加滤镜并二次调色**。
5.  **（可选）美咖相机进行美颜**（此步骤也可前置）。
6.  **（可选）天天P图进行化妆或抠图虚化**（制造有质感的光圈效果，而非简单的整体虚化）。

## 实战演示：人景综合照精修

现在，我们以一张学员的半身外景照为例，进行完整的高阶精修演示。

**原图问题分析**：腹部稍大，衣服有褶皱，需优化质感。

**第一步：美图秀秀去瑕疵**
*   打开照片，使用“去痘”功能处理面部及皮肤微小瑕疵。
*   **关键设置**：保存时务必在设置中将画质调整为“全高清”，以保证图片清晰度。

**第二步：Facetune塑形与修补**
*   使用“重塑”功能中的“调整大小”工具，谨慎地将腹部收小。
*   使用“修补”工具，复制周围平滑区域，覆盖衣服上的明显褶皱。
*   使用“平滑”工具柔化处理过的褶皱区域，使其更自然。
*   使用“色调”工具中的取色器，选取衣服正常颜色，涂抹在过亮或阴影不协调的区域，使光影过渡自然。
*   对比修图前后，腹部轮廓和衣服质感已得到明显改善。

**第三步：面部精细调整（Facetune）**
*   使用“细节”工具轻微提亮鼻梁等面部轮廓，增加立体感。
*   检查并微调眼睛大小，使其更协调自然。

**第四步：天天P图化妆（可选）**
*   因是侧脸，部分化妆功能受限。尝试使用“美妆”功能添加眉形，并手动调整位置和颜色浓度。
*   保存时同样需设置图片质量为“高清”。

**第五步：Snapseed调色**
*   **突出细节**：结构调整+10，锐化+20，增强轮廓但不过分。
*   **调整图片**：
    *   亮度：+45（原图稍暗）。
    *   对比度：+10（使人物轮廓更分明）。
    *   饱和度：+5（轻微增加色彩）。
    *   氛围：+20（平衡人物与背景）。
    *   高光：-30（防止过曝，聚焦面部）。
    *   阴影：+10（保留合理阴影）。
    *   暖色调/冷色调：本例不调整。
*   **（可选）戏剧效果**：如果使用，需将滤镜强度调低，并恢复饱和度，以避免效果过度。

**第六步：VSCO添加滤镜与风格化调色**
*   选择一个合适的滤镜（本例中强度降至3）。
*   进行二次微调：
    *   曝光：+0.5
    *   对比度：-0.5
    *   饱和度：+0.5
    *   高光：+1.0
    *   阴影：+1.5（增加一点灰度感，符合流行审美）。
    *   色调：+1.5（微调绿色）。
    *   暗角：+1.9（轻微增加氛围）。
    *   褪色：+1.0（增加一点高级灰质感）。
*   导出前确保图片质量为100%。

**最终对比**：精修后的照片在肤色、光影、整体色调和质感上均有显著提升，人物更突出，画面更高级。

## 实战演示：自拍人像精修

最后，我们简要演示自拍照的精修流程。

**第一步：美图秀秀去瑕疵**。
处理痘痘、皮肤小粉刺等。

**第二步：Facetune面部精修**。
*   微调眼角、开外眼角，使眼睛更有神（遵循“三庭五眼”比例）。
*   轻微缩小鼻翼。
*   使用“平滑”和“更平滑”功能处理皮肤瑕疵，但保留自然光影。

**第三步：天天P图化妆**。
*   添加“鼻梁高光”（30%强度）增加立体感。
*   加深“双眼皮”线条，并使用去痘工具清理修饰痕迹，使其自然。
*   可酌情微加“眼线”。
*   **注意**：男生化妆应极其自然，避免使用眼影、浓重卧蚕等。

**第四步：Snapseed调色**。
*   结构：+8
*   锐化：+10
*   亮度：+20
*   高光：-25

**第五步：VSCO风格化**。
*   选择一款有质感的滤镜。
*   微调：对比度+0.5，阴影色调可配合衣物颜色（如红色头巾配红色调阴影）。
*   最终使照片色调滋润、质感突出。

**修图前后对比**：精修后的自拍照在肤色、五官立体度、整体色调和光感上都有质的飞跃。
