# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去：
```python
message = "你好，世界！"
```
此后，当我们使用 `message` 这个名字时，Python就知道我们指的是那个存放着“你好，世界！”的“盒子”。

## 2. 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的种类。Python中有几种内置的基本数据类型，它们决定了数据可以做什么操作。

以下是几种最常用的数据类型：

*   **整数（int）**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数（float）**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**：表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `‘Python’`, `“学习编程”`。
*   **布尔值（bool）**：只有两个值：`True`（真）和 `False`（假），常用于逻辑判断。

## 3. 如何查看和转换类型？ 🔄

在编程时，有时我们需要知道一个变量到底是什么类型，或者将一种类型转换为另一种类型。

我们可以使用 `type()` 函数来查看变量的类型：
```python
num = 100
print(type(num))  # 输出：<class 'int'>
```

当我们需要进行类型转换时，可以使用以下函数：

*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

例如，将一个整数和字符串拼接起来，就需要先将整数转换为字符串：
```python
age = 25
greeting = “我今年” + str(age) + “岁。”
print(greeting)  # 输出：我今年25岁。
```

## 4. 动手试一试 ✍️

理论需要结合实践。让我们通过一个简单的例子来运用刚才学到的知识。

以下是一个小程序，它定义了不同类型的变量，并进行了打印和类型转换：
```python
# 定义变量
name = “晓辉”      # 字符串
score = 95.5      # 浮点数
is_passed = True  # 布尔值

# 打印变量及其类型
print(“姓名：”, name)
print(“类型：”, type(name))

print(“分数：”, score)
print(“类型：”, type(score))

# 类型转换：将浮点数分数转换为整数（舍去小数）
integer_score = int(score)
print(“整数分数：”, integer_score)
```

## 总结 📝

本节课中我们一起学习了Python的变量与数据类型。我们知道了变量是存储数据的容器，并通过赋值符号 `=` 来使用它。我们还认识了四种基本数据类型：**整数（int）**、**浮点数（float）**、**字符串（str）** 和 **布尔值（bool）**。最后，我们学会了用 `type()` 查看类型，以及用 `int()`, `float()`, `str()` 进行类型转换。掌握这些是构建更复杂程序的基石。
