# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（例如一个数字或一段文字）。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是赋值，公式如下：
`变量名 = 值`

例如，我们创建一个名为 `age` 的变量，并把数字 `25` 放进去：
```python
age = 25
```

## 基本数据类型 🔢

理解了如何“装东西”的盒子后，我们来看看盒子里可以放哪些不同类型的“物品”。Python中有几种基本的数据类型，用于表示不同种类的数据。

以下是三种最常用的基本数据类型：

*   **整数（int）**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数（float）**：表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串（str）**：表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `‘你好’`, `“Python”`。

## 如何查看和使用变量？ 👀

知道了数据类型，我们如何确认盒子里装的是什么，以及如何使用它呢？Python提供了方便的函数和方法来查看和操作变量。

以下是两个关键操作：

1.  **使用 `print()` 函数输出变量**：这是最常用的查看变量内容的方法。
    ```python
    name = “卡卡”
    print(name)  # 这将在屏幕上输出：卡卡
    ```

2.  **使用 `type()` 函数检查类型**：当你无法确定一个变量的数据类型时，可以用这个函数来查看。
    ```python
    height = 1.75
    print(type(height))  # 这将在屏幕上输出：<class ‘float’>
    ```

## 一个简单的综合例子 💻

现在，让我们把以上知识点串联起来，看一个完整的例子。

```python
# 定义几个变量
my_name = “Alice”     # 字符串变量
my_age = 30           # 整数变量
my_height = 1.65      # 浮点数变量

# 打印这些变量
print(“姓名：”, my_name)
print(“年龄：”, my_age)
print(“身高：”, my_height)

# 检查变量类型
print(type(my_name))   # 输出 <class ‘str’>
print(type(my_age))    # 输出 <class ‘int’>
print(type(my_height)) # 输出 <class ‘float’>
```

## 总结 📝

本节课中我们一起学习了Python编程的基石。我们首先将**变量**理解为一个存储数据的命名容器，并通过 `变量名 = 值` 的语法为其赋值。接着，我们认识了三种**基本数据类型**：用于整数的`int`、用于小数的`float`和用于文本的`str`。最后，我们掌握了使用 `print()` 函数输出变量值，以及使用 `type()` 函数检查变量数据类型的方法。掌握这些概念是您后续学习所有Python知识的关键第一步。
