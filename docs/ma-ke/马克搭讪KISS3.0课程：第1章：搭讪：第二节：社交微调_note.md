# 马克搭讪KISS3.0课程：第1章：搭讪：第二节：社交微调_note

在本节课中，我们将要学习如何通过观察对方的非语言信号，并即时调整自己的肢体语言和语调，来建立初步的社交连接。这个过程被称为“社交微调”。

上一节我们介绍了初步接触与眼神连接的重要性。本节中我们来看看如何根据对方的即时反应进行动态调整，以深化连接。

## 核心概念：动态反馈与和谐一致

社交互动不是单向的。对方的表情和状态会受到环境、思绪或情绪的影响。你需要观察这些信号，并调整自己的**肢体语言**，使双方状态达到和谐一致。

**核心公式：你的行为 + 对他人信号的观察 → 即时微调 → 和谐一致的连接状态**

## 如何进行社交微调

以下是实现社交微调的两个关键方面。

### 1. 观察与匹配非语言信号

你需要根据对方的面部表情来微调自己的肢体语言。目标是让对方感觉你们处于同一频道，仿佛已经连接在一起。

当对方做出某个表情时，你可以用相似的肢体语言进行回应。这能让对方觉得你是能理解它、与它在一起的人。

### 2. 调整你的语音语调

你的语音语调同样重要。语调不能是竞争性或炫耀性的，例如“我赢了游戏，所以我和你接吻”。

正确的语调应该是平和、友好且充满连接的，目的是邀请而非宣告。

## 微调的目的：建立连接与保持注意

接吻游戏本质上是一个抓住对方注意力并建立连接的游戏。前30秒的关键任务就是通过眼神连接、面部表情匹配和语调调整来建立这种连接。

建立与女性的连接主要通过两个渠道：**眼神交流**和**恰当的语音语调**。
