# 马克搭讪KISS3.0课程：第1章：搭讪：第三节 搭讪要避免做这些事 🚫

在本节课中，我们将要学习一个在KISS接吻游戏中常见的错误做法。理解并避免这个错误，是成功应用KISS系统的关键一步。

## 核心错误：将游戏与结果直接挂钩

上一节我们介绍了搭讪的基本心态，本节中我们来看看一个具体的、需要避免的行为模式。

大部分人在和妹子玩KISS接吻游戏时，会做错一件事。他们一上去就跟妹子说：“我要跟你玩十道剪刀布，然后我赢了的话我就跟你接吻。”

## 传统思维的误区

这种行为源于一种传统的、甚至是被社会洗脑的思维模式。这种模式的逻辑是线性的，可以概括为以下公式：

**赢得竞争 = 获得奖励**

例如：
*   我赢了这副牌，我就可以赢到你的钱。
*   我打赢这场篮球比赛，我就可以获得奖杯。

## KISS系统的不同逻辑

然而，在我们的接吻系统里，情况并非完全如此。**“我赢了就能怎样”** 这种必然的因果关系并不成立。

实际上，80%你会跟妹子接吻的原因，并非源于你赢得了某个游戏。接吻的发生主要依赖于其他一些更重要的因素，而不是你以为的“我跟你玩个十道剪刀布，我赢了（比如我出石头，你出剪刀），然后我们就接吻”。

那肯定不是这样的。

## 本节总结

本节课中我们一起学习了在KISS游戏中需要避免的一个核心错误：试图通过赢得一个简单的游戏（如剪刀石头布）来直接换取接吻。我们认识到，传统竞争思维中的“赢取奖励”模式，并不适用于建立亲密感和吸引。真正的关键在于游戏之外的其他互动因素。
