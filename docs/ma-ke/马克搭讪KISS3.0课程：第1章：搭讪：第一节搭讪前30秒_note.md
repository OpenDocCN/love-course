# 马克搭讪KISS3.0课程：第1章：搭讪：第一节 搭讪前30秒 👁️🗨️

在本节课中，我们将要学习KISS 3.0教程中关于搭讪的核心部分：如何在与女性互动的前30秒内，通过潜沟通建立连接，为后续的互动奠定基础。

上一节我们介绍了课程的整体框架，本节中我们来看看搭讪初始阶段的关键行动。

## 核心：潜沟通与眼神连接

最重要的是你整个人在她面前表现出的潜沟通。第一件事就是眼神沟通。在你遇到她的前三十秒，你的眼神沟通必须能够跟她建立现实感的连接。你把她拉到你的现实里面。谁的现实强，谁就可以把她拉到对方的世界里面。

## 语言的力量：像病毒一样占领思维

你的语言像病毒一样可以占领她的大脑。你仅仅是靠近她，哪怕你只是说一些简单的话。

以下是语言占领思维的原理：
*   你的话对她来说是新鲜的，她以前没有听说过。
*   它发生在不同的情境和地方。
*   你还用不同的语调去跟她说。
*   因此一切对她来说都是原创的。人都喜欢新鲜原创的东西。

因为这些原因，你的语言绝对可以占领她的大脑，把她拉到你的世界里面。她已跟你建立连接，忘记了她的朋友、今天买的东西以及一切，脑子里面只有你说的内容。

## 建立连接的步骤

在用眼神跟她建立连接之后，你就可以基本上跟她建立好连接了。这是前面三十秒做的事情：通过眼神建立连接。

接下来第二个重要的事就是，你要根据她的表情和行为进行调整。

---

本节课中我们一起学习了搭讪前30秒的核心策略。我们明白了建立强大“现实感”的重要性，并通过**眼神沟通**和**原创性语言**作为主要工具来实现。关键在于用你的存在和话语将她带入你的世界，并在过程中敏锐地观察她的反馈。
