# 马克搭讪KISS3.0课程 第3课：童年的问题——男孩与男人 👨‍👦

在本节课中，我们将要学习一个在约会和接吻流程中至关重要的心态问题：如何避免陷入“男孩”的索取模式，并成长为给予价值的“男人”。

上一节我们介绍了接吻流程的基本框架，本节中我们来看看一个常见的错误心态及其根源。

## 一个尴尬的故事

在一次约会中，女生因穿高跟鞋走路不稳，想去洗手间。男生趁此机会将女生的下巴拉过来亲了一下。女生回来后却说：“我们俩之间还不熟，你不要碰我。”结果，女生反而离他更远。

这看似是一个主动亲吻却遭遇拒绝的尴尬案例。

## 问题的根源：男孩与母亲的模式

经过思考，我发现问题的根源在于许多人陷入了**儿童与大人**的交往模式。他们将约会对象潜意识地当成了“妈妈”。

以下是这种模式的核心逻辑：

*   在婴儿期，母亲是无私的，允许婴儿索取（如喝奶）。
*   母亲不会因为婴儿的索取而离开或抛弃他。
*   因此，许多人在约会中不自觉地进入“婴儿”模式，向女人索取关注、亲密和认同。

**公式：错误心态 = 将约会对象视为“母亲” → 进入索取模式**

## 索取心态在约会中的后果

在约会中，如果你持有索取者心态，女人会敏锐地察觉到。同样，在接吻流程中，如果你是“占便宜”或“找奶喝”的心态，女人会自动产生抗拒。

她们内心的声音是：“我不想当你妈妈。我想要一个能依靠的真正男人，而不是一个男孩。”

## 解决方案：成为给予价值的阿尔法男人

那么，在接吻流程中，我们应该成为什么样的人？答案是：**阿尔法（Alpha）男人**。

阿尔法男人的特点是：
*   他们根据当下情况灵活调整策略。
*   他们懂得在正确时机做正确的事以达到目标。
*   他们的行为有策略、有流程。
*   他们是一个**给予价值**的老练者。

**核心代码：正确行为 = 给予价值(策略, 流程, 时机)**

这意味着：“我遵守社交规则，我不是来占你便宜的，我只是想让我们的互动变得更好玩、更有吸引力。”

## 丰盛心态 vs. 稀缺心态

当你以正确的心态与女人接吻时，她们会感到**获得了一个强大男人的认同**。

要记住：女人宁愿选择一个很多人分享的阿尔法男人，也不会独占一个饥渴的贝塔（Beta）男人。

因此，我们要展示的是**丰盛心态**：

*   “我生活丰富，有很多选择。”
*   “此刻我遇到你，很开心，我喜欢你，所以我愿意和你接吻。”
*   “我不是一个稀缺、饥渴、只想占便宜的男人。”

这种心态会贯穿在所有互动中。例如牵手时，只是轻轻触碰以制造好玩的感觉，而不是用力地把对方的手拉过来。

## 正确的互动模式

以下是良好互动模式的体现：

1.  **放松**：保持轻松自在的状态。
2.  **渐进**：通过眼神交汇和自然的身体靠近来升级关系。
3.  **默契**：在双方都感受到吸引的瞬间，自然而然地接吻。

**流程描述：** 相遇 → 眼神交汇 → 感受到吸引 → 彼此自然靠近 → 接吻。

## 总结

本节课中我们一起学习了约会心态的核心区别。

*   我们剖析了“男孩”模式的根源——将女性视为母亲进行索取。
*   我们明确了索取心态会导致女性的反感和远离。
*   我们定义了“男人”模式——作为阿尔法男性，以**丰盛心态**给予价值。
*   我们强调了所有互动都应遵循策略和流程，传递出“我欣赏你，但并非非你不可”的自信姿态。

记住，关键不在于“索取”一个吻，而在于“创造”一个让双方都愿意接吻的瞬间。
