# 马克搭讪KISS3.0课程：第7章收尾：约会方法 🎬

在本节课中，我们将要学习一个高效的约会流程。这个流程旨在通过营造氛围和积累性张力，自然地引导约会走向亲密关系，并最终成功邀约对方回家。

许多人的约会方式效果不佳。他们通常和女生喝咖啡、压马路或吃饭，聊天内容无关紧要，最终独自回家。这种约会缺乏明确的目标和流程。

上一节我们指出了常见约会的误区，本节中我们来看看一个成功的约会流程应该是怎样的。

我曾经历过各种失败的约会，例如看完电影后压马路，或在酒店附近突兀地提出休息。这些错误很常见。

现在，我将分享我视为珍宝的成功约会流程。我希望这个分享能帮助更多人。

我的核心流程很简单：**只看电影，看完直接回家**。关键在于在电影院内的互动。

---

## 电影院内的互动流程 🎥

我们已经和女生舒服地坐在电影院里。此时，无需刻意寻找重要话题，只需像恋人一样轻松地坐着。

除非电影播放到有情绪感染力的片段，否则我不与女生进行过多互动。

当电影出现能触动人的情节时，我会抓住这个时机。我知道女生也会被触动。

这时，我与她对视一眼。这个简单的对视，就能在两人之间建立起一种男女间的氛围，在她心中种下暧昧的种子。

之后，我继续看电影。表面上一切如常，但两人之间已存在性张力。她也能感受到这种紧张感。

此时需要保持沉默。这种性张力会随着时间持续累积。

不要过早地释放这种张力。当电影再次播放到能引起共鸣的情节时，才是行动的时机。

我只需轻微地碰一下她的肩膀或衣服，然后让她看向我，同时我稍微靠近她。

随着这一碰，累积的性张力会得到极大的释放。对她而言，这一碰能带来强烈的、类似亲密接触的释放感。

这是一个建立男女关系框架的过程。通过先眼神对视积累张力，再通过触碰释放，你已经被她归类为“可以发生关系的男人”。

## 巩固关系与最终邀约 🤝

在电影的后半段，可以进一步巩固这种感觉。

以下是几个简单的身体接触示例：
*   轻轻抚摸她的后背，感受一下温度，持续两三秒即可。

这样做，你们之间就已经建立了足够的男女之间的感觉。

随着电影接近尾声，你们会产生一种“无论外界如何变化，我们始终在一起”的联结感。这时，带领她就变得非常简单。

你可以在电影散场时，开启一个略带私密性的话题测试。例如：“你知道吗，日本的和服是不穿内衣的。”

如果她愿意接续这个话题深入聊下去，那通常是一个强烈的积极信号。

这时，你可以直接提出转场邀约。以下是几个常用的借口：
*   “我有件衣服忘在家里了，陪我回去拿一下吧。”
*   “我手机没电了，跟我回家拿个充电宝吧。”

通常，她会迫不及待地答应。有时她可能会口头拒绝，比如说“我在这等你”或“我先回家”。

这只是口头上的矜持。你需要再坚持一下：“走吧，就拿个东西，很快的，然后我们再去别的地方吃点东西。”

她知道你是个有经验的玩家，不会把责任推给她。当网约车还有三分钟到达时，直接牵着她的手上车即可。就这样，成功带她回家。

这是一个可以一次搞定的完整流程。
