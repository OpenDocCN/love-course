
## 二、真诚倾听
倾听是建立信任的关键。以下是一些倾听技巧：

1. **全神贯注**：在聊天时，不要分心，全神贯注地听她说话。
2. **点头示意**：通过点头表示你在认真听她说话。
3. **回应她的观点**：适当回应她的观点，让她感受到你在关注她。

**代码**：
```python
# 示例代码：倾听技巧
def listen_sincerely():
    print("请开始说话，我会认真倾听。")
    while True:
        sentence = input()
        if sentence == "结束":
            break
        print("你刚才说的是：", sentence)

# 调用函数
listen_sincerely()
```
