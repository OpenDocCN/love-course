# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 变量：数据的容器 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们使用等号 `=` 来给变量赋值。

**核心公式：**
`变量名 = 值`

例如，执行 `name = “笨小子”` 这行代码，就意味着创建了一个名为 `name` 的“盒子”，并在里面存放了“笨小子”这个数据。之后，当我们使用 `name` 时，Python就知道我们指的是“笨小子”。

## 基本数据类型 🔢

理解了变量是容器后，我们需要知道容器里能放哪些“东西”。这些“东西”的类型就是数据类型。Python有几种内置的基本数据类型。

以下是三种最常用的基本数据类型：

1.  **整数**
    表示没有小数部分的数字，例如：`-5`, `0`, `100`。

2.  **浮点数**
    表示带有小数点的数字，例如：`3.14`, `2.0`, `-0.5`。

3.  **字符串**
    表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如：`‘Hello’`, `“笨小子”`。

## 如何查看与使用 🔍

我们如何知道一个变量里存放的是什么类型的数据呢？又该如何使用它们？

**核心代码：使用 `type()` 函数**
```python
age = 18
print(type(age))  # 输出：<class ‘int’>
```
这段代码将变量 `age` 的数据类型打印出来，结果是 `int`，代表整数。

变量之间可以进行运算。数字类型可以进行数学计算，字符串可以用加号 `+` 进行连接。

**核心代码：数据类型运算**
```python
# 数字运算
a = 10
b = 3
print(a + b)  # 输出：13
print(a * b)  # 输出：30

# 字符串连接
greeting = “你好，”
name = “笨小子”
message = greeting + name
print(message)  # 输出：你好，笨小子
```

## 总结 📝

本节课中我们一起学习了Python的变量与基本数据类型。
我们了解到变量是存储数据的容器，并通过赋值符号 `=` 来存放数据。
我们认识了三种基本数据类型：用于整数的**整数**，用于小数的**浮点数**，以及用于文本的**字符串**。
最后，我们学会了使用 `type()` 函数查看数据类型，并实践了数字的运算与字符串的连接操作。
