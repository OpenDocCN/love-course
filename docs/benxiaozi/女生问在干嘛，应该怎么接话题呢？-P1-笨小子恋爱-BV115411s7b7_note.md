以下是几种回应女生询问“在干嘛”的方法：

**公式**： “我（正在）+ 日常活动”

### 2. 展示关心
**公式**： “你（看起来）+ 情绪状态”

**示例**： “你看起来有点累，需要休息一下吗？”

**公式**： “我（在）+ 幽默活动”

**示例**： “我正在和猫咪玩捉迷藏呢，它总是找不到我！”

### 4. 引导话题
**公式**： “你（最近）+ 话题”

**示例**： “你最近有没有什么新鲜事？”
