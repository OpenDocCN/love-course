# 米娅情感《男性高情商养成记》📚：014. 第13节 - 识别并摆脱“圣爹心态”💡

在本节课中，我们将要学习一种在亲密关系中常见的心理模式——“圣爹心态”。我们将剖析其本质、成因，并与女性的“圣母心”进行对比，最终帮助你识别这种心态并建立更健康的情感关系。

## 认识“圣爹心态”与“圣母心”🔍

上一节我们介绍了情商在关系中的重要性，本节中我们来看看一种可能消耗你的特定心态。

与“圣爹心态”相对应的是“圣母心”。后者指的是一种善良神圣、渴望拯救万物苍生，用母性光环普照人间的奉献心态。这种心态表现为看似善良、想要将对方从水火之中挽救出来的付出行为。

不仅女人会有这种心态，男人也会有，而且很常见。虽然男人和女人表达这种心态的方式非常雷同，但是产生这种心理状态的机制却有所不同。下面我们就分别来分析一下。

## 女性“圣母心”的成因🤰

由于女性的生理构造，使她们承载着孕育下一代的神圣使命。

不管一个女人是否生育过，在女人的基因里都刻有母性。区别在于母性是否被激发，以及被谁激发。很多女人生了孩子之后，母性被大大激发，原来天不怕地不怕的自己突然也有了软肋。

为了孩子可以做出很多努力和牺牲，甚至把孩子放在人生的第一位。同时，也有一些女人的母性不仅被后代激发，也会被男人激发，这也就是人们常说的“圣母心”。这类女人把正常情况下会在后代上投入的母性情怀，投射到了一个自己爱的男人身上。

以下是“圣母心”在情感中的具体表现：
*   母亲在养育后代时，会竭尽所能地为其提供充足的营养和食物。
*   母亲会拼尽全力地保护后代的安全。
*   母亲为了培养自己的后代，会投入大量的时间和精力。
*   母亲希望自己的后代健康成长，并且未来长大成人之后能够有所建树。

而“圣母心”的女人会把这种母爱运用到自己的情感世界。比较明显的表现之一，就是会对一个男人有着无底线的包容，甚至说是纵容。不管对方此刻是多么脾气暴躁、多么不上进，甚至是有不良嗜好，以及对自己不忠诚。

“圣母心”的女人总是认为自己可以用感情来感化对方，认为对方是可以被培养和塑造的，认为只要自己不放弃，对方早晚有一天会做出改变。然而，往往事与愿违。

这个时候，“圣母心”的女人不仅不会意识到自己已经出现了选择判断性的失误，而且还会陷入一种错误的归因模式。她们会认为这个男人之所以变成今天这样，都是自己没有改造好他，没有管教好他，会思考“是不是当初自己如果没有怎么怎么做，也许他现在就不是这样了”。

于是，她们就在自己给自己制造的陷阱中越陷越深，深深痛苦，无法自拔。

## 男性“圣爹心态”的根源🛡️

大家听到这些，可能觉得这样的女人很愚蠢很无知，但殊不知男人也会这样。也许现在在听课的你就正在经历或者曾经经历过。

不管是原始的部落、古代的封建社会，还是当下的现代文明，毋庸置疑的是，男人都是力量和生产力的象征。那么男人的“男”字怎么写？**上面一个“田”，下面一个“力”**，在田里出力的人始称为男人。

在几万年前的原始部落，男女分工就很明确。男人负责狩猎，为家人带回食物；女人负责守巢和哺育后代。同时，男人还要负责部落的安全，要保护自己的妻儿不受猛兽的攻击和入侵者的侵害。

随着农业革命的发展，人类逐渐由原始部落步入到古代文明，开始有了国家、阶级、统治和战争。但是刻在男人和女人基因里的东西还是被悄无声息地传承了下来。同时也是由于生理构造的局限，女人的力量无法驾驭兵器和战车，所以上阵杀敌、保卫疆土仍然是男人的工作，包括开垦田地、挖沟造渠，主力仍然是男人。

那么到了现代社会，虽然人类已经进入了高度的文明，女人也承担了相当一部分的社会生产力，但是流淌在男人和女人基因里的东西始终是无法被彻底消灭的。即：**男人一直承担着保护者的角色，骨子里都会有保护弱小的本能；而女人则一直需要被保护**。自古英雄难过美人关。

男人这种自带的英雄主义情节和对弱者的同情心，就是“圣爹心态”的根源。很多“圣爹心”的男人选择跟一个女人在一起，不是因为两个人价值匹配、三观契合、门当户对，甚至不是因为自己很爱这个女人，而是因为觉得这个女人楚楚可怜，需要人疼、需要人照顾、需要人保护。

因此觉得“我应该跟他在一起，来拯救她。如果我不来拯救她，可能她就会过得很惨”。时间长了，慢慢越陷越深，甚至误以为这是爱情。

## “圣爹心态”的本质与危害⚖️

那么真相是什么？真相是在你认为自己深陷爱河的时候，却不知道事实往往并非如此。

因为这种所谓的奉献、所谓的付出，其实都并不是出于真正的爱情，而是为了满足自己的英雄主义情节，满足自己无处安放的保护欲。甚至，在对这个女人付出的同时，可以让自己获得强烈的满足感、存在感，甚至让原本缺乏自信的自己，居然可以在某种程度上找到自信，觉得自己终于被人需要了。

这种情感跟爱情好像有点像，但是完全是两回事儿。这种情感看起来好像是男人一直在付出，但其实男人的付出本质上不是利他行为，而是利己行为。**你是为了满足你自己，才对对方做了这么多。而对面的女人只是你自己情绪的载体。**

说到这里，你可能会恍然大悟。但很多人往往并不具备这样的判断力，认为自己付出了很多，对方应该被拯救了，对方应该变更好了，对方应该朝着自己原来预想的方向去发展了。

但大多数情况是，对方并没有改变。当事情没有按照自己预期的方向发展的时候，男人就开始陷入了深深的疑问和自我怀疑。

以下是“圣爹心态”者常见的痛苦疑问：
*   为什么我对他那么好，他总是无动于衷呢？
*   当初跟他在一起的时候，他过得特别不好，我都没有嫌弃他，反而把最好的都给他。我为他付出了这么多，为什么他还要弃我而去呢？
*   可能是我做的不够好吧。他现在这个样子，我还有什么办法能够挽救他吗？我实在不想看他这样沉沦下去。
*   我觉得他现在这个样子非常糟糕，我要如何去改变他？我希望他能变得更好。
*   为什么我对他这么好，他却以为理所当然？为什么我做的这么多，还是无法改变他？我本以为我的爱能感化他，没想到他却让我如此痛苦。

这些痛苦都是来源于你的“圣爹心”。其实你大可不必。

## 如何摆脱“圣爹心态”🕊️

首先，要知道这个世界上是没有救世主的，你也不是。没有人可以真正地去拯救另外一个人。**能够拯救他的只有他自己。如果一个人自己不想改变的话，谁去拯救都没有用。** 要明白，这是一个最基本的自然规律。

其次，“圣爹心”真的不是爱。美好健康的感情，一定是能够滋养你的，一定是两个人共同成长、相互助益的，是能给你提供能量的，同时你会感到安全、放松和愉悦。

但如果一段所谓爱情，让你身心俱疲、痛苦万分，甚至影响了你的正常生活，成为了消耗你的工具，那么这就不是好的感情，甚至并不是什么感情，而是你给自己制造的思维陷阱。

所以，不要因为怜悯和同情去跟一个一直消耗你的人在一起。他并没有你想象的那么弱小，你也没有自己想象的那么强大。**爱是相互给予，并不是单线投入。** 你要看清自己的内心：你们到底是因为相爱而在一起，还是因为你的“圣爹心”和执念。
