## 核心思维：吸引力源于需求满足

上一节我们介绍了课程主题，本节中我们来看看吸引力的底层逻辑。

一个人能够被另一个人吸引，核心在于**对方满足了我的需求点**。这是基本逻辑。

需求点因人而异，可能是情绪价值、物质或颜值。许多男性不明白这个道理，按照自己的想象行事，付出大量努力却只感动了自己。

对方不为所动的原因，是你没有搞懂对方需要什么。对方不在乎你给了什么，只在乎他需要什么。这一点男女相通。

举个例子，你为对方做了100件事，但这100件事里没有一件是他需要的。另一个人只为他做了一件事，而这件事恰好是他的刚需。最终与这位女生牵手的会是另一个人。

所以不要盲目付出。你最好先研究对方需要什么，以及你能否提供。

如果对方需要的你实在无法提供，就别费力气了。成功的可能性很低，即使勉强成功，后续结果也不会好。

这里用到一个核心观念：**用户思维**。你要把对方当成你要推销的客户。

首先你要研究你的客户，找到他的痛点，然后有针对性地营销。产品就是你自己。

## 避免操之过急：学会延迟满足

理解了吸引力的基础后，我们来看看追求过程中的一个常见错误：操之过急。

许多男性在追求心动的女性时非常急迫，急于确定关系。例如，一位30多岁的男粉丝，相亲认识一个女孩，第一次见面就特别喜欢。

他在第一次见面吃饭结束、下电梯时突然向女孩表白，搞得女方措手不及，结果被拒绝。在后续约会的半个月里，他又频繁在不合时宜的场合表白。

他总觉得女孩对他也有意思，但屡遭拒绝。后来女孩告诉他，自己比较在意仪式感，不希望他总是随地表白，感觉不被重视。

这是一个典型案例。有句话叫心急吃不了热豆腐。

两个人刚认识，如果男人非常着急确定关系，会让女人产生防备心，怀疑对方是否有目的或问题。除非女方对男方也是一见钟情，否则男人应学会延迟满足。

此时要做的不是频繁表达心意，而是建立吸引，恰当展示优点，让对方想要持续了解你，对你感兴趣。

当对方对你的感觉达到一定程度，或许还会期待你的表白。这时自然事半功倍。

这是一个过程，就像煲汤一样。前面要小火慢熬，等到食材都熟了，最后出锅前再放盐调味。放盐就相当于表白的动作。

## 创造吸引力：从展示到刺激

明白了时机的重要性，我们进一步探讨如何有效创造吸引力。

一个简单又深刻的道理是：如果你想勾起别人吃牛排的欲望，把牛排摆在他面前固然可以，但最令人无法抗拒的是煎牛排的吱吱声。

这会让他的脑海中浮现牛排在黑色铁板上香味四溢的画面，这是最直接有效的刺激。

## 给予确定性：行动与名分并重

在创造吸引的过程中，明确关系同样关键。

有人说男人追女人用行动就行，不需要表白。这是不了解女性思维。女人需要确定性。

## 运用策略：不完美原则与拒绝的艺术

确定了关系的基础后，我们来看看如何通过策略提升自身价值。

许多男人在追女孩时总是给得特别满，恨不得24小时随时待命，随叫随到，像舔狗一样。

这能体现你的诚意，但对女人产生不了吸引力。很多时候也要适当学会拒绝。

这里用到一个**不完美原则**。大家都知道，在最好的餐厅订位很不容易。即使知道所有桌子都被订光了，还是有很多人想去排队。

当广告鼓吹商品限量供应、每位顾客只能买两件、只供应到星期天时，就是在使用不完美原则。

举个例子，一家非常火的美发沙龙，每天都能听到接待人员与顾客的有趣对话。无论老少，待遇都一样。

对话如下：
```
接待人员说：“早上好，欢迎光临。”
客人说：“我想请你帮忙预定星期二21点的服务，可以吗？”
接待人员查了一下预约单（实际上这个时间是空着的），却回答道：“对不起，这个时间已经被预定了。但我可以帮你安排到10点半，您看这样可以吗？”
顾客马上会说：“太谢谢你了，那我就预定10点半吧。这样真好，我一定会准时到的。”
```

实际上，无论你要预定的时间是否已被预定，你都无法要到想要的时间，除非提前好几周去预定。

店主精通不完美原则。他们知道要让人们觉得服务很抢手，这样人们对他们的价值观念就会提升，以后就会想再来。

男人在追求女人时也是这样。随叫随到不会提升你的价值，有时候也需要策略性地拒绝。

## 激发占有欲：引入竞争机制

运用策略提升价值后，另一个有效方法是引入竞争机制。

例如，你家里有一个摆件，摆在柜子里很久，久到你都记不得它的存在。突然有一天，家里来了客人，看到这个摆件大加赞赏，非常喜欢，于是问你能不能送给他。

这时很多人可能就不会大方了。原来这个东西这么好？当初自己没留意吗？这回不但不能送出去，还得把它摆在一个更显眼的位置。

就像在拍卖场，一件商品竞拍的人越多，价格就拍得越贵，这件商品就显得越有价值。

## 利用环境：吊桥效应的心理学应用

除了主动引入竞争，环境也能在无形中影响情感。我们经常在电影或电视剧里看到英雄救美的情节。美女获救后，两人眼神相对，擦出爱情火花。

为什么会这样？在心理学上是有研究的，这叫**吊桥效应**。

这是在卡比兰诺峡谷吊桥上做的一个有趣实验，证明了恐惧对于求爱的作用。实验如下：

一位男士站在吊桥中央，向路过桥的女孩搭话，问对方能不能参加一项调研活动，并留下自己的电话以便后续联络。

在另一座固定的砖桥上做了同样的实验，并且在桥上执行试验的是同一批女生。然后比较留下电话的数量。

结果那些通过吊桥的女孩留下的电话更多。原因是在吊桥上的女孩处于刺激性生理紧张状态。

在紧张恐惧的状态时，刚好出现一位男性，心里的紧张恐惧得到缓解，对对方产生了一种情绪上的依赖，容易被误以为是一见钟情。

而在砖桥上的女孩并没有处于紧张恐惧状态，也没产生爱慕情感，所以留下的电话也少。

根据这个实验，社会心理学家得出的结论是：性冲动实际上是恐惧引起的，而不是异性相吸。我们误以为异性相吸，是因为求爱时的化学物质刺激引起的恐惧和性冲动没有区别，所以会混淆。

危险的信息刺激人体产生的肾上腺素，和热恋时的情况相同。在这两种情况下，肾上腺素都会让我们心跳加快、血压升高、血糖浓度增大。

这就可以理解为什么两个人的单独旅行会增加亲密感。

学习了各种技巧后，我们必须回归根本。技巧只是在给你的价值锦上添花。
