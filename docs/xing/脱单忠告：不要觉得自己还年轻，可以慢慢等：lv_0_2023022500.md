## 年龄与脱单难度的关系
随着年龄的增长，脱单变得越来越困难。以下是几个原因：

### 1. 社交圈缩小
随着年龄的增长，人们的社交圈往往会缩小。年轻时，我们可能有很多朋友和同事，但随着年龄的增长，我们可能会因为工作、家庭等原因而减少社交活动。

**公式**：社交圈大小 = 年轻时社交活动频率 × 随着年龄增长的活动频率

### 2. 期望值提高
随着年龄的增长，人们对伴侣的期望值也会提高。年轻时，我们可能更注重感情，而随着年龄的增长，我们可能会更加关注对方的物质条件、家庭背景等因素。

**代码**：期望值 = 感情需求 × 物质需求 × 家庭需求

### 3. 时间压力
随着年龄的增长，人们面临的时间压力也会增加。年轻时，我们有更多的时间去寻找合适的伴侣，而随着年龄的增长，我们可能会感到时间紧迫。

**公式**：时间压力 = 剩余时间 × 脱单需求
