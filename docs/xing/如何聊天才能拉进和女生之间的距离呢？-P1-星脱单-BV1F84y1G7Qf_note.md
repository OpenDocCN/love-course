## 聊天技巧
1. **避免成为解说家**：不要让对方觉得你的生活很精彩，你这个人很牛。
2. **时间阶段**：刚认识时，通过交换信息互相了解；熟悉后，找共同点，延伸话题。
3. **小段子**：在暧昧期间，适当使用小段子，增加趣味性。
4. **深入话题**：在关系密切时，找独特、私密的话题，偶尔使用情话。
