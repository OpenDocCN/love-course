# **恋爱干货教程 - 如何识别只想玩玩的女孩** 🌟

## 概述
在本节课中，我们将学习如何识别那些只想与你玩玩，并不想与你建立未来关系的女孩。

## 特征识别
以下是识别只想玩玩的女孩的四个特征：

**公式**：女孩感受无关性 = 女孩感受不关心你的喜怒哀乐

这种女孩只关心自己的快乐，如果你有情绪，她可能会指责你。

### 2. 从不道歉
**公式**：女孩道歉率 = 0

无论争吵原因如何，即使错误在她，她也不会道歉，而是选择忽视你，迫使你去妥协和道歉。

### 3. 只会索取，不会付出
**公式**：女孩付出率 = 0

她会觉得你的付出是理所当然的，并通过明说或暗示的方式引导你付出更多的时间和金钱。

### 4. 不关心你的长期发展
**公式**：女孩关心程度 = 不关心你的未来

她对你的未来没有兴趣，只关心现在，不在乎明天。

## 小心陷阱
与这种女孩在一起，你需要小心，她们不值得你浪费太多时间和精力。小心最后人财两空，你可能会很惨。

## 结语
本节课中我们一起学习了如何识别只想玩玩的女孩。记住，爱自己，不要浪费时间和精力在那些不值得的人身上。
