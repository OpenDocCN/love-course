## 概述
在本节课中，我们将学习如何判断你的女朋友是否出轨。以下是一些关键迹象和行为的分析。

## 第一招：手机静音
**公式**：手机静音 → 害怕被查看 → 避免接听电话

- 突然害怕你看她的手机。
- 接电话时总是回避你。
- 连进厕所都要带着手机。

## 第二招：沉默寡言
**公式**：沉默寡言 → 对手机比对你更热情

- 对你逐渐失去表达欲和分享欲。
- 对着手机多过对着你。

## 第三招：情绪波动
**公式**：情绪波动 → 无缘无故的开心或暴躁

- 经常无缘无故的开心或者暴躁。
- 心情像过山车一样捉摸不透。

## 第四招：匆忙挂电话
**公式**：匆忙挂电话 → 经常出门 → 偶尔失踪

- 联系她总是匆忙的挂电话。
- 老是出门。
- 偶尔会失踪。

## 第五招：失去耐心
**公式**：失去耐心 → 容易厌烦 → 小事争吵

- 对你很容易就会厌烦。
- 少有一点小事都能跟你吵架。

## 第六招：抗拒亲密
**公式**：抗拒亲密 → 胡言乱语

- 突然开始抗拒跟你有亲密的接触。
- 或者对你胡言乱语。

## 第七招：风格突变
**公式**：风格突变 → 穿搭改变 → 暧昧行为

- 突然换了一些你平常没见过的穿搭。
- 可能还会喷其他的香水。
- 突然变得暧昧。

## 第八招：突然示好
**公式**：突然示好 → 过度殷勤 → 强调爱意

- 会突然跟你示好。
- 对你无故献上极大的殷勤。
- 强调她只爱你一个人。

## 总结
本节课中我们一起学习了如何通过观察女朋友的行为和态度来判断她是否出轨。这些迹象和行为可以帮助你更好地了解她的心理状态，从而做出正确的判断。
