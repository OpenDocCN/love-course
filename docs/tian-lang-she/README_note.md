# 天狼社课程：Python基础入门（第1课）🚀

在本节课中，我们将要学习Python编程语言的基础概念，包括变量、数据类型以及如何执行简单的运算。这些是构建更复杂程序的基石。

## 1. 什么是Python？🐍

Python是一种高级、解释型的编程语言。它以简洁明了的语法著称，非常适合初学者入门。Python的设计哲学强调代码的可读性。

上一节我们介绍了Python的概貌，本节中我们来看看如何开始使用它。

## 2. 安装与运行Python

要使用Python，首先需要在你的计算机上安装它。你可以从Python官方网站下载适合你操作系统的安装包。安装完成后，可以通过命令行输入 `python` 来启动交互式解释器。

以下是验证安装是否成功的步骤：
*   打开终端或命令提示符。
*   输入命令 `python --version`。
*   如果屏幕上显示类似 `Python 3.x.x` 的版本信息，说明安装成功。

## 3. 第一个程序：打印“Hello, World!”👋

在编程世界中，第一个程序通常是输出“Hello, World!”。这能确认你的环境配置正确。

在Python中，这非常简单，只需一行代码：
```python
print("Hello, World!")
```
将这段代码保存为 `.py` 文件并运行，或者在交互式解释器中直接输入，你将在屏幕上看到这行问候语。

## 4. 变量与基本数据类型🧱

变量是存储数据的容器。在Python中，你无需预先声明变量的类型，直接赋值即可。

以下是几种基本的数据类型：
*   **整数**：如 `age = 25`
*   **浮点数**：如 `price = 19.99`
*   **字符串**：如 `name = “Alice”`
*   **布尔值**：如 `is_student = True`

了解了如何存储数据后，我们来看看如何操作它们。

## 5. 基本运算🔢

Python支持常见的数学运算。你可以对数字进行加、减、乘、除等操作。

运算可以通过直接的算术符号完成：
*   **加法**：`result = 5 + 3` （`result` 的值为 `8`）
*   **减法**：`result = 10 - 4` （`result` 的值为 `6`）
*   **乘法**：`result = 7 * 2` （`result` 的值为 `14`）
*   **除法**：`result = 15 / 3` （`result` 的值为 `5.0`，注意结果是浮点数）

字符串也可以进行“相加”，这被称为**拼接**：
```python
greeting = “Hello, ” + “World!”
print(greeting) # 输出：Hello, World!
```
