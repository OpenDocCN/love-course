# 天狼社觉醒搭讪课《新手游戏》：第八课 搭讪的三个阶段 🎯

在本节课中，我们将要学习搭讪过程中的三个核心阶段。理解这些阶段有助于你分析自身状态，并采取最适合的策略。

## 概述

搭讪过程可以划分为三个主要阶段：初段期、出长期和短期。每个阶段都对应着不同的心态、挑战和应对方法。明确自己所处的阶段，是制定有效行动方案的第一步。

## 搭讪的三个阶段

上一节我们概述了三个阶段的重要性，本节中我们来详细看看每个阶段的具体含义。

以下是三个阶段的定义与特点：

1.  **初段期**
    *   这是搭讪的起始阶段。核心在于你自身如何克服最初的障碍并采取行动。这个阶段归结于你，你自己怎么拯救你自己，看你是否适合进入出长期。

2.  **出长期**
    *   如果你判断自己适合并能够突破初段期，那么你就进入了出长期。这个阶段是技能与信心积累的关键时期。

3.  **短期**
    *   如果评估后认为适合短期策略，那就采用相应的方法。这意味着根据实际情况选择最直接有效的互动方式。

## 阶段总结与应用

好了，这就是我们应当了解的搭讪三个阶段。你应该经历或者已经身处其中某个情况。大家现在应该都了解了。

那么，根据你自身的状态——适合出长期，你就努力成长；适合短期，那就采用短期策略。如果无法进入出长期，那就没办法，接受现状并寻找其他突破口。
