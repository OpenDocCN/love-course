# 社交补习班《阿涵学长自信社交课》第1课：不再畏惧负面社交反馈，在社交里勇敢自信 🚀

在本节课中，我们将要学习如何打破对负面社交体验的畏惧，塑造强大的内心。这是实现自信社交的第一步。

## 概述

很多人因为害怕尴尬、没人回应、被人议论等负面体验，而在社交中畏首畏尾，不敢行动。本节课程将逐一剖析这些常见的恐惧，帮助你改变认知，从而在社交中勇敢自信地展现自己。

## 核心心态转变：从畏惧到接受

上一节我们概述了课程目标，本节中我们来看看如何具体看待和处理那些让我们恐惧的负面社交体验。

我曾认为内心强大是虚的，直到自己改变后才明白，内心强大是行动的基础。如果我们过于在意别人的眼光和评价，就会被束缚，不敢尝试社交、约会、演讲或创业，最终一事无成。因此，打开心态至关重要。

我们最畏惧的，往往是社交中可能出现的负面体验。以下是六种最常见的负面体验，我们将逐一分析它们是否真的值得畏惧。

在开始分析前，我想说：一个人如果还没下定决心改变，往往是因为过去“痛”得还不够。我曾有七年时间被别人的眼光左右，什么都做不了，那种毫无作为的感觉让我受够了。人生只有一次，我们不应该因为害怕负面反馈，而错过想做的事和想体验的人生。想清楚，是“尴尬没人回应”更可怕，还是“一辈子不敢做想做的事”更可怕？答案显然是后者。

## 六种负面社交体验剖析

以下是六种让我们在社交中退缩的常见负面体验，我们将逐一拆解。

### 1. 害怕尴尬

很多社交困扰者都特别害怕尴尬，比如约会冷场、聚会没话聊。我曾也很畏惧，但后来对尴尬的看法发生了三个改变。

**第一，尴尬是正常的，不必自责。**
*   任何人，包括社交达人，都会遇到尴尬场景。尴尬是社交中的必然存在，无法完全避免。
*   有些场景或关系本身就容易尴尬，这与个人能力无关。
*   不要把尴尬等同于不善交际或失败。它不丢脸，很正常。
*   尴尬过后，不必深度怀疑自己。有些尴尬是应该发生的，换成别人也一样。

**第二，尴尬不全是你的责任。**
*   社交是双方或多方的互动。冷场或尴尬，可能是对方不会接话、不善言辞。
*   永远不要把社交中所有不愉快的责任都揽到自己头上。这只会增添不必要的心理压力。

**第三，尴尬是一种主观感受。**
*   **尴尬程度取决于你的认知：** 你觉得事情严重，它就尴尬；你看得平淡，它就不那么尴尬。你感受到的尴尬，可能远超过客观事实。
*   **你觉得尴尬，不代表别人也觉得：** 你的心理感受不等于对方的。你可能觉得冷场很糟，但对方可能觉得没什么，甚至没注意到。
*   当感到尴尬时，可以告诉自己：“可能是我把后果看得太重了。对方可能根本没受影响。”

综上所述，我们不应再对尴尬“谈虎色变”。它很正常，不必把所有责任归咎于自己，也不必觉得丢脸。

**过渡：** 理解了尴尬的常态性和主观性，我们就能更平和地看待社交中的冷场。例如，一位学员在约会中遇到冷场，她没有自责，而是想“不说话就不说话呗”，甚至哼起了歌，反而很快找到了新话题。人不可能一直说话，偶尔的沉默也是休息。别把小事看得太重。

**做事时的尴尬（如演讲、组织活动）**
我们不仅怕相处尴尬，也怕做事时遭遇尴尬，比如演讲没人鼓掌、组织活动没人响应。

我曾认为尴尬场面等于失败，所以很怕。后来我发现，**尴尬是通往成功的必经之路**。

*   **周杰伦的例子：** 他早期上节目被冷落，非常尴尬。但如果他因此逃避，就不会有后来的天王巨星。这说明，从新手到高手，尴尬是躲不开的环节。
*   **我自己的例子：** 创业第一期只有七八个学员，群里互动常无人回应，非常尴尬。但我没觉得丢脸，认为这是成长必经阶段。坚持下来，才有了后来的发展。

**核心观点：** 不怕尴尬，才是一个人能有所作为的应有素质。在做任何事（演讲、分享等）前，如果感到害怕，就告诉自己：“这是我成为高手的必经之路，我要面对它。” 这样心态会完全不同。

### 2. 害怕没人回应

我曾特别害怕主动搭讪得不到回应，觉得非常丢脸。后来，我的初恋女友改变了我的看法。

她非常自信，在路上热情地和同学打招呼，即使对方没理她，她也完全不受影响。我意识到，**真正自信的人，不在乎别人是否回应**，因为他们自我认同，不需要外界确认。

**怕别人不回应，反而显得很“Low”**
*   你越怕别人不回应，越容易被影响，这恰恰显得你缺乏底气、不自信。
*   反过来，你越有实力，越自信；越自信，就越不在意别人是否回应。
*   你越不在意，反而给人感觉你越有底气、越有价值、越有魅力。

**心态转变：**
后来我主动社交时，会这样想：我主动说话是出于我的自信和善于交际。如果你不回应，那是你的问题（可能是你放不开、不善交际），而不是我的问题。我有底气，所以我不怕。

**核心概念：永远不要轻易被别人对你的态度所影响。**
*   别人对你冷淡，你就消极、怀疑自己，这是一种低价值的表现。
*   真正有实力的人，内心稳定，不会轻易被外界评价动摇。
*   如果你想显得有价值、优秀，就必须做到不那么轻易受别人影响。

### 3. 害怕有人不喜欢

我曾非常在意别人看法，怕别人说我不好，甚至想通过讨好来抹去差评。后来我发现，**有人不喜欢是件好事**。

**第一，你不可能被所有人喜欢。**
*   能被所有人“喜欢”的只有“老好人”，他们靠讨好迎合换取廉价认可，但没人真正欣赏、在乎他们。
*   我们要成为有魅力、有个性、有价值的人，而不是老好人。

**第二，有人不喜欢，证明你有个性。**
*   有魅力的人必然有个性，有个性就必然有人喜欢、有人不喜欢。
*   有人不喜欢你，恰恰说明你坚持了自我，展现了个性，这是魅力的前提。

**好与坏总是共存**
*   任何行为都有两面性。例如，上台演讲可能被说“爱出风头”（看似负面），但同时也会被赞“勇敢”（正面）。
*   如果你因为怕“爱出风头”的评论而不敢上台，那你也同时失去了“勇敢”的认可。
*   **总结：当你能接受有人不喜欢你时，喜欢你的人才会出现。当你要求所有人都喜欢你时，可能没人特别不喜欢你，但真正喜欢你的人也没有。**

我的经历：大一我低调乖巧，没人说我不好，但也没人称赞我，存在感很低。大二我突破自我，开始演讲、约会，虽然有人给我负面评价，但同时也获得了更多人的欣赏、关注和友谊。得到的正面反馈远多于负面。

### 4. 害怕被人笑话 & 5. 害怕被人议论

这两点我放在一起讲，因为它们密切相关。我曾因怕被笑话和议论而非常拘束。

**关于“被笑话”**
*   **旧认知：** 被拒绝、讲不好会被笑话，很丢脸。
*   **新认知：** 他们笑我要不到微信/讲得不好，我笑他们不敢要/不敢讲。**我觉得我是在被一群“怂”的人笑话，这反而是好事。**
*   **逻辑：** 我敢做他们不敢做的事（搭讪、演讲），哪怕结果不完美，我的“敢”已经比他们的“不敢”强太多了。能被平庸者嘲笑，说明我正在做不平凡的事。

**二八定律视角**
*   世界上20%的人优秀，80%的人平庸。你想成为20%，就必须做80%的人不敢做、不理解的事。
*   因此，被那80%的人笑话，是你走向卓越的证明。我反而享受这种“被平庸者嘲笑”的感觉。

**关于“被议论”**
*   **旧认知：** 被议论是坏事，要避免。
*   **新认知：** **富有争议、被人议论，是一个人优秀的标志。**
*   **逻辑：** 越是厉害、有所作为的人，越会受人关注，也越会引发不同观点的议论。一个默默无闻、毫无作为的人，根本没人议论他。
*   被议论，证明你有影响力、存在感和价值。所以，不要怕被议论，议论越多，可能说明你越有作为。

### 6. 害怕留给别人不符合自己期待的印象

我们总希望别人眼中的自己，符合自我期待（如自信、优秀）。但常担心因某次表现不好，而留下坏印象（如自卑、笨拙），因此不敢社交。

**被误解是常态**
*   每个人看到的你，都是片面的、局部的。没人能24小时全面了解你。
*   因此，**你一定会被别人误会**。就像亲戚只看到我一次家庭聚会没演讲，就认为我不会说话，但他们不了解我日常的工作。
*   我们也在误解别人。这是人际交往的常态。

**承认无能为力，是成熟的标志**
*   我们永远无法完全控制别人对我们的主观评价。
*   我们可以努力展现好的一面，但不必苛求每个人都了解真实的你。
*   把精力从“纠结别人的看法”转移到“打造客观成绩”上。接受有些事你无能为力，心态才会更豁达、成熟。

**过渡：** 从极度在意他人看法，到变得不那么在意，是一个心智成熟的过程。当你理解了世界的现实规则，就会越来越豁达。

## 总结

本节课我们一起学习了如何应对六种常见的负面社交体验：
1.  **尴尬**：是社交常态，不必自责，也非你一人之责。
2.  **没人回应**：越不在意，越显自信与底气。
3.  **有人不喜欢**：这证明你有个性，是好事的开始。
4.  **被笑话**：可能是你正做著平庸者不敢做的事。
5.  **被议论**：往往是有所作为和影响力的副产品。
6.  **被误解**：是人际常态，接受自己无法控制所有评价。

把这些认知理解透彻并践行，你的心态会放松很多，内心会强大起来。你将不再被他人眼光左右，敢做更多事，敢展现个性。越不在意看法，反而成绩会越好。这是我的经验之谈，你也可以试试看。
