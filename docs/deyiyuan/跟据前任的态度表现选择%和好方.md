# 跟据前任的态度表现选择%和好方

在本节课中，我们将学习如何通过观察前任的态度表现，来判断关系所处的阶段，并据此选择合适的复合策略。课程将分为四个递进的阶段进行讲解，每个阶段都有明确的表现特征与应对方法。

## 概述：理解关系阶段的信号 🎯

判断关系升级时机的核心，在于识别前任释放出的好感信号。这些信号如同路标，指引我们采取正确的行动。尚未恢复联系的朋友，可以通过以下程度的好感信号来判断时机。

## 第一阶段：有回应（普通朋友关系） 🤝

上一节我们概述了识别信号的重要性，本节中我们来看看第一个阶段——有回应。此阶段勉强算是普通朋友关系。

此时前任的表现是：会回复你的消息，但内容很平淡，态度也很礼貌。他至少还愿意搭理，这说明你们之间的冲突浓度已经降得很低。前任算是把你放在了朋友的位置。

以下是此阶段的对策：
*   不要恭维他。
*   不要舔他。
*   不要动不动就送礼物送温暖。
*   更不要贸然表白。

正确的做法是先做朋友，等待机会。这个阶段很容易犯的错误是沉不住气。你们可能会想：“我现在不做点什么，他怎么改变对我的态度呢？”真相是，贸然进行错误操作，可能会让你永远被放在普通朋友的位置，甚至可能连普通朋友都做不了。

最好的做法是：按前任能够接受的频率，保持联系或邀约见面。耐心等待前任以及自身发生变化。因为即使你什么都不做，也可能会有其他因素让他的心情和态度发生变化。你唯一能做的，就是保持联系，细心观察，并抓住出现的机会。

## 第二阶段：类似好朋友 👫

在建立了稳定的普通朋友联系后，本节我们来看看第二个阶段——类似好朋友。此时应该向更像情侣的关系去升级。

此时他的表现是：很愿意跟你倾诉，会对你说很多话。内容很具体，感受也很丰富，话题也比较敏感。这代表他觉得你能给他安全感和信任感，认为你是理解他的人。

你应该怎么办呢？**仍然不要急于升级关系**。否则前任可能会说：“本来还想跟你做朋友的，看你非要这样，那就算了吧。”因为你目前处在一个尴尬的局面：前任会有一种“失去你好像也没什么大不了”的感觉。

所以，我们一定要先牢固好朋友的关系。等前任对你的关系有了相当的依赖之后，再去循序渐进地表白和突破。

## 第三阶段：友人以上，恋人未满 💑

牢固了好朋友的基础后，关系可能进入一个暧昧期，也就是友人以上、恋人未满的阶段。

此时的表现是：经常找你聊天，回消息很及时，对你的生活充满好奇心，会关注你、关心你，甚至可以接受肢体接触。但这也不代表你提复合，对方马上就会接受。他可能还需要一个考虑的过程。

你的态度应该显得轻松、自信、诚恳、豁达。传递的感觉是：“我知道你对我还有好感，我依然喜欢你，所以希望你能再给我个机会。但如果你后悔了，我也能够接受。”

## 第四阶段：临门一脚 ⚽

当暧昧期积累足够的好感与依赖后，关系便来到了最后阶段——离复合只差临门一脚。

此时前任的表现是：你很容易就能把他约出来，他也愿意表明跟你在一起，听从你的意见，接受你的要求，并且不排斥任何肢体接触。这个时候，他基本上已经冰释前嫌，重新开始喜欢你，你的吸引也已经做得足够。

接下来就可以更大胆一些。但有时他也会刻意表现出轻微的抗拒，其实他只是想看到你对他表现出强烈的需求。不用害怕这种试探。

## 总结：循序渐进的复合过程 📈

本节课中，我们一起学习了复合的四个渐进阶段：
1.  **有回应阶段**：保持礼貌联系，耐心等待。
2.  **类似好朋友阶段**：巩固信任，避免冒进。
3.  **友人以上阶段**：自信表达，尊重对方节奏。
4.  **临门一脚阶段**：果断行动，应对轻微考验。

以上四个阶段，就是复合循序渐进的全过程。请对照这些表现，看看你们的关系目前属于哪一种，并采取相应的策略。
