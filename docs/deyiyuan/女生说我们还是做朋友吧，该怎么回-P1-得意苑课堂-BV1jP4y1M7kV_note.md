# 恋爱技巧课 01：当她说“我们还是做朋友吧”的应对策略 💔

在本节课中，我们将通过一个真实案例，学习当喜欢的女生提出“我们还是做朋友吧”时，如何扭转局面，重新掌握关系主动权。核心在于理解并运用情绪推拉与框架控制，而非盲目进攻。

## 案例背景与常见错误

上一节我们介绍了课程主题，本节中我们来看看一个典型的失败案例。一位男生向喜欢的女生表白后，收到了“我们还是做朋友吧”的回复。

男生随即情绪失控，追问：“你心里是不是有喜欢的人了所以拒绝我？”女生回答：“没有啊，我只是不想谈。”这种有希望又追不到的感觉让男生彻底崩溃。

## 第一步：立即停止追问

当女生说出“做朋友”时，她的情绪处于抵触状态。盲目的进攻和死缠烂打只会加剧她的抵触心理。因此，首要策略是**立即停止一切追问和施压行为**。

我让案例中的男生停止了追问。半天后，时机成熟，我指导他只回复了一个字：“好”。

这个简单的回复标志着男生态度发生了根本转变，从之前的死缠烂打变为现在的冷淡。女生立刻察觉到了变化，问道：“生气了？”

## 第二步：实施“冷处理”与展示价值

既然男生的态度开始转变，女生会感到难以适应。特别是当她已习惯从男生这里获得关注时，更不会轻易接受他的离开。

以下是“冷处理”期间的核心操作：

*   **暂时不理睬**：男生没有回复女生的“生气了”的询问。
*   **制造心理落差**：连续几天冷落女生，让她内心产生疑惑：“他怎么不找我聊天了？”“怎么跟之前不一样了？”“是不是认识了新的女生？”
*   **展示社交价值**：我让男生在朋友圈发布与其他女生的合照，制造他已开始新生活的印象。

## 第三步：引导情绪与反转框架

女生看到朋友圈后果然评论道：“哎呦小日子过的挺滋润嘛，我说怎么就不理我了呢。”

此时，我们已经拿回了关系的主导权。关键操作是**不急于回复**，将她晾在一边，让她继续琢磨，放大情绪。直到第二天晚上，才冷淡地回复：“哎，怎么了。”

这样做的目的是持续激发女生的情绪。虽然她之前不接受恋爱，但一想到男生突然退出，她的情绪会被无限放大。

## 第四步：运用“推拉”技巧锁定胜局

过了一会，女生主动问道：“照片上那个女生是你女朋友吗？”我让男生先别回。果然，不一会女生又发来消息：“可以追我到一半就不追了呢？”

此时，女生已经开始花心思与男生聊天，目的是想让他继续追求自己。因为她感觉到自己的魅力受到了挑战，产生了不甘心的情绪。

于是我指导男生回复：“怎么了，想我了？”女生回：“没有啊，既然你不想追了，那咱们也没有什么关系，你就互删吧。”

**核心要点**：千万不要被女生这种“威胁”吓到。我让男生回复：“随便。”

女生看到后立刻不淡定了：“哎，你之前对我的喜欢都是假的吗？我对你很失望。”男生继续冷淡回应：“这还有关系吗？”

我们继续以退为进，引导女生主动挽回。果然，过了一会儿女生回道：“我不想失去你，我前面不懂得珍惜，是因为我在考虑啊我们之间到底合不合适。”

现在轮到女生主动挽回了。此时我们使用“推拉”技巧：先推开——“那你去找你合适的。”；再拉回——“哎，我已经尽我最大努力去追求你了，你还在犹豫。”

**推拉公式**：**推开（制造失去感） + 拉回（展示付出与价值） = 激发对方珍惜**

这让女生感觉到我们是可能失去的，同时又表明我们已经足够努力，是她自己在犹豫。接着女生回答：“哎，如果我现在想改变我那天的答案，你还愿意给我们一次机会吗？”

案例到此结束，男生如愿与女生确定了情侣关系。

## 课程总结

本节课中，我们一起学习了当女生提出“做朋友”时的完整应对策略：

1.  **停止施压**：立即停止追问和死缠烂打。
2.  **态度反转**：用简洁冷淡的回应（如“好”）改变自身框架。
3.  **冷处理与价值展示**：暂时冷落对方，并通过社交展示（如朋友圈）提升自身吸引力。
4.  **引导情绪**：不急于回应，让对方产生情绪波动和失去感。
5.  **推拉定局**：运用“推开-拉回”技巧，最终引导对方主动挽回。

追女生的关键在于选对方法，理解情绪与框架的互动，而非一味付出。
