# 课程：你选哪个？ - P1 - 得意苑课堂 📊

在本节课中，我们将要学习一个被称为“巴赫测试题”的趣味心理测试。这个测试通过一个简单的家务选择问题，来分析个人的性格倾向与潜在的伴侣忠诚度。我们将逐一解析每个选项背后的含义。

## 概述

这个测试的核心是提出一个问题：如果一辈子只能做一件家务，你会选择哪一种？选项包括做饭、洗碗、拖地和倒垃圾。根据选择，可以推测一个人的责任心和出轨可能性。

## 测试内容与解析

以下是测试的具体问题和各个选项的解析。

**问题：** 如果一辈子只能做一件家务，你会选择哪一种？
*   **A 做饭**
*   **B 洗碗**
*   **C 拖地**
*   **D 倒垃圾**

上一节我们介绍了测试的基本问题，本节中我们来看看每个选项对应的具体分析。

以下是各个选项的性格与忠诚度分析：
*   选择 **A 做饭** 的人很有责任心，也很忠诚，是一个非常好的伴侣。其**出轨率**为 **5%**。
*   选择 **B 洗碗** 的人一般是有贼心没贼胆。其**出轨率**为 **30%**。
*   选择 **C 拖地** 的人有出轨的想法，但是对自己又有道德的要求。其**出轨率**为 **50%**。
*   选择 **D 倒垃圾** 的人需要特别注意，这种人大多数结婚之后还是会在外面乱来。其**出轨率**为 **70%**。

## 总结

本节课中我们一起学习了“巴赫测试题”。这个测试通过一个简单的家务选择，以趣味性的方式探讨了不同选择可能反映出的性格特质与关系忠诚度。它被称为“震惊世界的巴赫测试题”。你可以思考一下，这个测试的结果你觉得准吗？
