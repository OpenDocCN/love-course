在本节课中，我们将要学习一段感情走向终结时通常会经历的四个心理阶段。我们将剖析从最初的沉默到最终分离的完整过程，帮助你识别关系中的潜在信号，并理解每个阶段的核心特征与动态变化。

一段感情走到终结绝不是因为一次争吵，往往早有预兆。真正的分手是从两个人开始沉默的时候就已经开始了。

## 第一阶段：沉默期 🤐

上一节我们提到了感情终结的预兆，本节中我们来看看第一个关键阶段——沉默期。这个阶段的核心特征是沟通的减少与内心的质疑。

在这个时期，双方会开始质疑对方是否还爱自己，并思考彼此是否真的合适。以下是沉默期的几个典型表现：

*   你们会开始质疑对方到底还爱不爱你。
*   你们会思考彼此之间是否合适。
*   你们开始不停地挑对方的毛病。
*   你们慢慢地对彼此感到失望。

可能你也试图改变过，但是最后发现徒劳无功。提前准备放弃的那个人，就已经开始计划分手了。

## 第二阶段：爆发期 💥

在经历了漫长的沉默与积累后，关系会进入一个冲突公开化的阶段，即爆发期。这通常由一次矛盾后的激烈争吵所触发。

你们开始争论谁付出的多和少，以及谁对谁错。主动提分手的人想尽快地抽离，开始新的生活，但是被分手的人开始拼命地纠缠和挽留。

**关系公式：对立立场 → 关系僵化**
双方完全站到了对立面，因此两个人的关系越来越僵，直到双方彻底拉黑，彼此断绝了所有的联系方式而告终。

## 第三阶段：冷静期 🧊

当激烈的冲突尘埃落定，双方会进入一个表面平静但内心波澜的时期，即冷静期。这是失去对方后的一段各自疗愈与适应的时光。

彻底失去对方之后，你们各自平静地过了一段时间。发现回归到生活之后，还是会变得有些不适。回忆里面全都是你们之间的点点滴滴，在回忆中很痛苦，在痛苦中反思和挣扎。

以下是双方在此阶段的不同心理状态：

*   **主动提分手的人**：可能会有一点后悔以及遗憾。
*   **被分手的人**：已经接受了现实，打算放弃并且尝试新的生活。

但是你们总会想起对方，忘掉一个爱的人确实是蛮难的。

## 第四阶段：反复期 🔄

在尝试向前走的过程中，情绪和决定往往不是线性的，而是会进入一个摇摆不定的反复期。这是内心挣扎最激烈的阶段。

这个时候你们开始反反复复。在反复期，主动提分手的人会后悔当初的冲动，但是又拉不下脸来主动找你。人都是这样的，做错事情很容易，但是承认错误很难。

**行为模式代码描述：**
```python
if 感到后悔:
    尝试间接接触对方()
else:
    保持沉默()
```

所以他会间接性地开始接触你。而被分手的一方，开始想着接受新的生活，一边又沉浸在失去挚爱的痛苦之中，走不出来，想要复合。但是又想起刚分手的时候，去纠缠对方，对方那种冷漠的态度。就是这样子的，慢慢地错过了彼此。
