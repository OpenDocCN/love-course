# 课程名称：冷暴力心理分析 🧊 第001课

在本节课中，我们将要学习“冷暴力”的定义、习惯使用冷暴力者的核心心态特点，以及面对这种情况时应采取的根本态度。内容基于对特定行为模式的心理分析。

## 概述：什么是冷暴力？

冷暴力是一种不通过身体暴力，而是通过精神冷漠和情感疏离来伤害他人的行为。它的核心表现是**不明确、不回应、不拒绝、也不接受**，并强行阻断一切亲密互动。

上一节我们介绍了冷暴力的基本定义，本节中我们来看看习惯使用冷暴力的人通常具备哪些心理特点。

## 习惯冷暴力者的三大心态特点

以下是习惯使用冷暴力的人最常见的三种心理状态。

1.  **极度自私与自我**
    这种人的思维逻辑是：`我现在不想说，我就是不说`。他们只在意自己的感受，而将你的感受视为次要。他们的行为出发点永远是自身的舒适度。

2.  **缺乏责任感**
    他们对关系中的责任感没有概念。他们不明白一段关系的喜怒哀乐需要双方共同经营和解决。由于缺乏责任感，他们不会考虑你是否难受，甚至将让你难受视为目的达成。

3.  **共情能力缺失**
    在日常生活中，我们看到陌生人哭泣也会于心不忍，但习惯冷暴力者对他们最亲密的人却无法做到这一点。最令人难受的是，他们甚至不愿说一句“我现在心情不好，想一个人待一会”，因为他们的目的并非缓和关系，而是让你心寒。

了解了这些核心心态后，我们还需要关注他们在亲密关系中的一种特殊行为模式。

## 亲密关系中的行为逻辑：追求自由而非约束

在情感世界里，这类人追求绝对自由，反感被约束。他们的典型逻辑是：“你怎么又提让我烦的事了？”或“你怎么不能给我一点空间？”。

他们无法给予伴侣最基本的尊重，但对伴侣以外的所有人却可能很有礼貌。他们明知你在等待回应，却故意将冷暴力的时间拉长，以此作为一种控制和惩罚的手段。

## 总结与核心应对策略

本节课中我们一起学习了冷暴力的定义，剖析了习惯使用冷暴力者的自私、无责任感和缺乏共情三大心态特点，以及他们在关系中追求单方面自由的行为逻辑。

面对冷暴力，最重要的认知是：**自救远比想方设法去修补一段关系更为重要**。理解对方的心理并非为了容忍或改变对方，而是为了认清现实，保护自己的情感健康，并做出对自己最有利的决定。
