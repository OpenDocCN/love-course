# 女生说在干嘛，应该怎么回？-P1-得意苑课堂-BV1mY411J71m_note

在本节课中，我们将学习当女生主动发来“在干嘛”时，如何通过不同段位的回复方式，把握聊天节奏，提升对话质量，并避免常见的沟通误区。

---

很多男生在不知道聊什么时，会主动给女生发“在干嘛”、“吃饭了吗”这类信息，试图让对方开启话题。女生往往觉得这类开场白平淡无趣，提不起兴趣。但你也没有更好的办法，只能让聊天草草结束。

那么，当女生主动给你发“在干嘛”时，你不可能也表现得提不起兴趣，让对话冷场。究竟应该如何回复呢？

上一节我们提到了常见的沟通困境，本节中我们来看看不同段位的回复方式，你可以对照检查自己属于哪一种。

以下是三种典型的回复方式及其效果分析：

1.  **钢铁直男式回复**
    *   **场景**：女生问“在干嘛呢？”，男生不假思索地回答：“我在吃饭呢。”
    *   **分析**：这是典型的直男回复，话题终结者。女生通常只会回一个“哦”。此时男生意识到不对，试图反问“你在干嘛呢？”，女生很可能回答“没干啥”。因为男生最初过于平淡的回复，已经给女生的热情泼了冷水，导致对话难以继续。

2.  **过度油腻式回复**
    *   **场景**：男生明白女生主动发信息可能意味着“我想你了”，于是想让气氛暧昧起来，回复道：“哎，我在想你啊。”
    *   **分析**：这种方式虽然看出了女生的心思，但回复过于直接和油腻，让女生感觉不到真诚。她可能会觉得来找你聊天很“下头”，是自己一时冲动。

3.  **高情商引导式回复**
    *   **核心思路**：当女生问“在干嘛”时，说明她可能在想你。此时，我们要做的是**掌握对话主导权**，引导女生暴露需求感，而不是自己过早暴露。
    *   **案例演示**：
        *   女生：“在干嘛？”
        *   男生：“有事说事，没事就直接说想我呢。” **（掌握主导权，发起进攻）**
        *   女生：“哈哈，就是问问你嘛。”
        *   男生：“那看完就是没有事情，那就是等于说想我了。” **（引导对方承认需求）**
        *   女生：“为什么不说你想我了？”
        *   男生：“我正准备拿你手机给你发消息，就看到你消息了。虽然很磨叽，但是你居然捷足先登了。” **（制造有趣画面，不点破但营造暧昧）**
        *   女生：“哼，男人的嘴骗人的鬼。”
        *   男生：“男人的嘴只能骗到好色鬼，你好色一点我肯定可以骗到你。” **（幽默化解，继续推拉）**
        *   女生：“哈哈，我才不好色。”
        *   男生：“你好不好色只有未来男朋友才知道吧。” **（植入“未来男朋友”概念进行试探）**
        *   女生：“哎，都不知道那个男的在哪呢。”
        *   男生：“要不要我给你介绍介绍？” **（以退为进，测试对方态度）**
        *   女生：“介绍谁？”
        *   男生：“你还真打算让我介绍呀小海胆？你和我聊天的时候还想着别的男人，是不是不太礼貌了？” **（释放醋意，升级暧昧情绪）**
        *   女生：“哈哈。”
        *   男生：“那你打算什么时候承认这次的聊天动机，是因为你想我了？” **（情绪升温后，主动收网，引导对方确认）**

通过这个案例可以看出，高情商的回复不是直接回答在做什么，而是通过**推拉、调侃、植入心锚**等方式，让对话变得有趣，并逐步建立情感连接。
