# 复联的时候，前任的冷淡、爱答不

在本节课中，我们将要学习如何应对前任在复联时表现出的冷淡与回避。核心在于理解并突破对方的心理防御机制，从而为关系的进一步发展创造可能。

## 理解心理防御的本质

互联的时候他总是对我爱搭不理，甚至干脆拉黑删除要怎么去沟通，这个问题最近问的频率是比较高的。那你就要破掉他的防御。有的时候人的防御，他不是从认知的角度去破的，因为两性之间这种关系当中，可能会有一些突破口。

上一节我们介绍了心理防御的存在，本节中我们来看看如何找到并利用这些突破口。

## 寻找关系的突破口

任何关系的建立与修复，都需要从理解对方开始。以下是寻找突破口的两个核心步骤：

1.  **投其所好**：首先你需要先了解，他对什么东西感兴趣。任何的关系，我们想搞定对方的时候，都是先从投其所好开始的，让他觉得我跟你是一路人。这种集体意识，我们还是要去遵循的。
2.  **切入频道**：然后再去根据他的个性，不断的去切入到他的频道里面。这个就是**实人实性**，即针对具体的人，采用符合其特性的方法。

## 调整沟通策略

如果你只按照自己舒适的沟通方式，但是却不是对方想要的东西，你的关系永远都会停留在原点。人心都是这样的，我想要靠近强大的、有力的、有趣的、美好的事物。

因此，关键在于展示对方所看重的一面。以下是你可以尝试突破的几个方向：

*   **展示力量**：体现你的可靠性与能力。
*   **展示趣味**：分享积极、有趣的生活与观点。
*   **展示美好**：呈现你积极向上、有价值的一面。

你看你从哪个方面，哪个角度上去突破，这个都是关系当中的技能。你从哪个技能上去突破，塞到这种认知和心理，可能会好一点。先从技能角度去试试看。

## 总结

本节课中我们一起学习了如何应对复联时前任的防御心理。核心路径是：**理解防御 → 投其所好 → 切入对方频道 → 展示对方所看重的价值（力量/趣味/美好）**。记住，突破防御的关键在于将关注点从“自我表达”转向“对方接收”，通过实人实性的策略，逐步重建吸引与连接。
