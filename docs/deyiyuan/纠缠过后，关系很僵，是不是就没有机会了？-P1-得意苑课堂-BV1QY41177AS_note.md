# 纠缠过后，关系很僵，是不是就没有机会了？ - 得意苑课堂 P1

## 📚 课程概述
在本节课中，我们将要学习分手后，如果已经进行了“死缠烂打”式的纠缠，导致关系陷入僵局，应如何正确地进行复合。课程将分析纠缠行为的负面影响，并提供一套清晰、可操作的挽回步骤。

---

## 🤔 为何纠缠会导致关系恶化
上一节我们概述了课程目标，本节中我们来看看为什么纠缠行为会让情况变得更糟。

都知道分手后死缠烂打是不对的。虽然道理都懂，但是还有很多小伙伴忍不住去纠缠对方，导致情况变得越来越糟糕。

为了避免大家继续犯错，需要强调对象为什么不喜欢在分手后被纠缠。我举个例子来说。

不知道你们走在路上有没有遇到过那种推销商品的推销员。你已经很明确地告诉他不用，然后他还一直跟着你缠着你。没有遇到过的大家也可以想象一下。

你们会因为他的诚意和迫切态度去买他的商品吗？不会。你心里只会更加厌烦这种行为，只想说快点走，离远点。

同理，分手后纠缠的行为也是这种下场。本来你们还没有到删除、拉黑的地步，甚至还能正常聊几句。但是你忍不住经常去找他，动不动就提复合。

你越提复合，对方就越有心理防御机制。最后他可能就不知道怎么面对这份感情，甚至不想理你了。

---

## ✅ 纠缠后如何复合：具体步骤
那么，关系僵化后还有挽救的余地吗？答案是肯定的。具体该怎么做呢？以下是挽回的核心步骤。

### 第一步：冷处理（断联）
首先你要做的是冷处理。你可以理解成是断联。总之就是不联系、不打扰。

纠缠过后你们的矛盾已经激化。只有冷处理过后，双方恢复理智，慢慢重新回归舒适感，才能进行下一步。

那么这段时间你该干嘛呢？你需要调整好心态，重新认识自己，重新塑造自己，为接下来的复联做铺垫。

### 第二步：复联破冰
其次就是我们常说的复联破冰。你可以先从一些日常小事作为破冰的开始。

以下是破冰时可以尝试的方法：
*   朋友圈里的节日问候。
*   朋友圈下边简单的交流。
*   偶尔打声招呼等。

目标是让双方陌生的关系重新暖起来。这里需要注意的是，一定不能说让对方感到压力的话，否则就会让对方再一次逃避。

### 第三步：深化联系与展示改变
接下来就是最重要的一步，就是需要你们深化联系。尤其是你的朋友圈的内容，一定是积极的正能量的内容，能带动起轻松无压力的那种聊天氛围。

因为只有当对方感觉到舒坦了，觉得和你相处还是挺轻松自在的，想着是不是要回到你身边，这个时候才适合去提复合，挽回的机率也才会越大。

我举个简单的例子。如果说外面下雨了，你点的外卖迟到了一个小时还没到，你会怎么描述呢？

**消极描述（错误示范）：**
`“好惨啊，好烦啊，外卖到了早饿死了，都怪这破天气。”`
对方本来对你就有一定的负面情绪，你再传染一些消极的想法过去，对方心里会更加不愉快。

**积极描述（正确示范）：**
`“下雨了，点的外卖还没来，幸好昨天买了点小零食可以填填肚子。”` (可以再发一个可爱的表情)
对方听了之后，渐渐地就会感觉到你的改变。

## 💎 核心要点与心态准备
对于真性分手来说，想要复合，光靠聊天肯定是不行的。但是，不可否认的是，聊天是挽回全程当中一个非常重要的方面。

而且最重要的是，聊天中你是主场。你可以在这个沟通当中，引导对方的情绪，帮助他放松戒备，接纳你的感情。

**核心公式：舒适感 + 积极展示 = 重建吸引**

所以在整个沟通过程当中，你必须有强大的心态，要有被拒绝的准备，还要有沟通引导的智慧。

## 📖 课程总结
本节课中，我们一起学习了纠缠后如何挽回关系。我们首先分析了纠缠行为如同“强推销”一样令人厌烦，会激发对方的心理防御。接着，我们明确了挽回的三步走策略：**冷处理（断联）→ 轻松破冰 → 深化联系并积极展示改变**。最后，我们强调了积极心态和沟通智慧的重要性。记住，挽回的关键在于重建舒适感和正面吸引力，而非施加压力。
