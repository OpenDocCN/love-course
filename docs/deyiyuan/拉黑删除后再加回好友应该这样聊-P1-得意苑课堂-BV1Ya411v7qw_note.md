# 拉黑删除后再加回好友应该这样聊 - P1 - 得意苑课堂 📚

在本节课中，我们将要学习分手后被对方删除拉黑，重新加回好友后应该如何进行有效沟通。核心目标是避免再次被拉黑，并逐步修复关系，为成功挽回奠定基础。

分手后被删除拉黑，加为好友之后，不要着急。控制不住地乱发消息会导致再次被拉黑。加为好友后，做好以下三点更容易挽回成功。首先要保持有效的展示渠道，即让对方能看见你。

你需要让对方慢慢接受你的转变。可以放心地去做以下三件事情。

以下是具体需要执行的三项行动：

1.  **正常发布积极的朋友圈状态**。逐渐展示你的情绪改变和价值改变。例如，可以发布自己的学习心得，或展示工作中的进展。
2.  **与共同朋友正常互动**。这样对方会在别人的朋友圈里刷到你。频繁的间接接触能增加好感，对方也能从朋友那里了解到你的变化。
3.  **关注对方的动态，偶尔点赞但不评论**。请放心，人都有好奇心。即使分手过程再惨烈，对方也一定会关注你的朋友圈。

如果说你正在面临情感问题，可以私信“复合”，获取进一步帮助，让你爱的人重新爱上你。

本节课中，我们一起学习了重新加回好友后的核心策略：保持冷静、通过社交平台进行积极的自我展示、利用共同朋友增加间接曝光，以及适度关注对方动态。关键在于循序渐进地展示改变，重新建立吸引，避免因操之过急而再次被拒。
