# 恋爱关系中的态度效应：温度理论 🧊 - 课程 P1

在本节课中，我们将要学习一个关于恋爱关系中互动模式的比喻——“温度理论”。这个理论通过“温度”这一概念，形象地描述了个人态度如何影响伴侣的回应和行为模式。

## 概述：一个观察案例

上一段我们提到了课程主题，本节中我们来看看一个具体的实例。我见过最淡定的女人是我闺蜜。她谈恋爱时，从不频繁主动给男朋友打电话或发微信。她的理由是：如果对方不忙，就会联系她；如果对方在忙，联系则会耽误其工作；如果对方不忙又不联系她，那她更没必要主动联系对方。

这个案例揭示了关系中一种基于对方态度而调整自身行为的模式。

## 核心理论：温度与状态的比喻

从上述案例出发，我们可以提炼出一个核心比喻。女人如同**H₂O**，其状态取决于你所施加的“温度”。这个关系可以用一个简单的条件逻辑来描述：

```python
if 态度温度 == 0:
    她的状态 = "冰"
elif 态度温度 == 100:
    她的状态 = "沸腾"
else:
    她的状态 = "不冷不热"
```

具体来说：
*   如果你用**0度**的冷漠对待她，她会立刻凝结成冰。
*   如果你用**100度**的热情去爱她，她会立刻沸腾。
*   如果你用**50度**的平常心对她，她也会回以不冷不热的态度。

## 理论的普遍性

你或许对此将信将疑。然而，这反映了许多人际关系互动中的一个常见事实。行为常常是对外界刺激的镜像反应，尤其在亲密关系中，双方的态度会相互影响并形成平衡。
