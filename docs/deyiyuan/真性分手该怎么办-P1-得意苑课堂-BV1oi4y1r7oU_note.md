在本节课中，我们将学习如何识别一段关系是否属于“真性分手”。了解这一点是采取正确后续行动的第一步。

上一段我们提到了课程主题，本节中我们来具体看看真性分手的核心特征。真性分手并非一时冲动的决定，它通常伴随着一系列明确且彻底的行为模式。

以下是判断是否为真性分手的几个关键行为特征：

*   **彻底切断联系**：对方主动断绝你们之间所有的联系方式。
*   **全面拉黑回避**：具体表现为微信被拉黑，电话无人接听，短信不予回复。
*   **拒绝现实接触**：更甚者，对方会在现实中避而不见。
*   **进入零反馈状态**：综合表现为一种完全不予理会的状态。

**核心判断公式**：**真性分手 ≈ 主动断绝 + 全面回避 + 零度反馈**

如果你遇到的情况符合以上所有描述，那么这很可能是一次真性分手。

本节课中，我们一起学习了真性分手的定义及其典型行为特征。识别这些信号是理解当前处境、避免错误应对的基础。在接下来的课程中，我们将探讨面对这种情况时，哪些事情是绝对不应该做的。
