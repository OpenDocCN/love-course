## 行为一：信息回复的严重失衡 💬

上一节我们介绍了课程概述，本节中我们来看看第一个关键行为：沟通中的回复模式。

当一方持续投入而另一方明显敷衍时，关系热度是不对等的。具体表现如下：

*   **你秒回，她轮回**：你可能在午饭时间发去大段信息，分享餐厅特色与感受，而她直到晚上才回复一个简单的字，就此结束话题。
*   **延迟回复与借口**：你晚上发送的信息，她通常到第二天才回复，常用理由是“昨晚睡着了”。
*   **聊天模式单一**：你们之间的对话基本局限于“你问她答”的问答形式，缺乏自然的延伸与互动。

## 行为二：单向的情感倾诉 🗣️

了解了沟通频率的失衡后，我们进一步探讨沟通内容的性质。健康的交流是双向的，而以下行为则表明她只将你视为单向输出渠道。

以下是典型的单向倾诉行为：

*   **她滔滔不绝，你认真倾听**：她经常向你大段讲述自己的事情，而你每次都认真聆听。
*   **你被敷衍或忽视**：每当你开始讲述自己的事情时，她便会敷衍应对、转移话题，甚至直接消失。
*   **关系本质**：这种模式意味着她可能将你当作**情绪垃圾桶**。公式可以概括为：**她可以向你倾倒情绪，但你无法获得同等的倾听**。

## 行为三：刻意回避与“装不懂” 🎭

最后，我们来看看最需要警惕的信号：对情感暗示的刻意回避。兴趣会产生强烈的关注力，反之则会表现为“钝感”。

一个对男生感兴趣的女生，会高度关注对方，分析他的每一句话，敏锐察觉其情绪变化。核心逻辑是：**兴趣 → 高关注度与高敏感度**。

相反，如果你对她发出的各种暗示，她都表现出听不懂或没接收到，那么这通常不是因为她真的迟钝。代码可以描述这种逻辑：
```python
if she_pretends_not_to_understand(your_hints):
    reason = "not_interested"  # 原因是不感兴趣
else:
    reason = "genuinely_oblivious"  # 原因才是真的迟钝
```
结论是，她并非不懂，而是**不想懂**，根源在于缺乏兴趣。
