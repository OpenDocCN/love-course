# 挽回教程 第1课：读懂分手后的真实想法 💔

在本节课中，我们将学习分手后对方的真实心理状态，以及为何纠缠和错误沟通是无效的。理解这些核心概念，是打破对方心理防御、开启有效挽回的第一步。

## 分手后的沟通误区

上一节我们提到了理解对方想法的重要性，本节中我们来看看分手初期最常见的沟通错误。

如果说你们是真心分手，那么在分手之后，真的不要去找对方沟通了，没有用的。对方处于一个极度的抵触、抗拒、讨厌的状态，这个时候你说什么，他都是听不进去的。

而且很多人在当下去沟通的时候，并没有沟通到一个点上。大部分的人都会采用错误的表达方式。

以下是几种典型的无效沟通话语：

*   我真的知道错了，我改。
*   你不希望我这样行啊，我去解决，我变好了，你给我一个机会，你相信我好不好。
*   我那么爱你，你这么喜欢你，你怎么就说变就变了。
*   以前你不是这个样子的呀，你看在我们这么久的感情的份上，再给我一次机会吧。

## 对方的核心心理逻辑

了解了错误示范后，我们需要剖析对方听到这些话时的真实想法。

对方的核心心理逻辑可以用一个简单的**公式**表示：
`过去的感情` + `当下的错误` ≠ `必须继续的理由`

怎么理解呢？难道就因为你很喜欢我，你很爱我，你很在乎我，我就一定得喜欢你，一定得爱你，一定得对你好吗？凭什么。现在人家已经跟你分手了，这个时候现在讲什么“以前很爱”、“以前喜欢”，你觉得人家会搭理你吗？他会在乎你的想法和感受吗？

## 有效沟通的核心原则

那么，什么才是正确的沟通方向呢？上一节我们看到了对方的心理，本节我们来学习沟通的核心原则。

我已经说过很多次了，分手之后你跟对方沟通，是表达对方的感受、对方的想法，让你跟他共情，而不是去表达你的感受，让对方按照你的想法、你的标准来跟你相处。你这么做，怪不得人家不想跟你聊，怪不得人家对你爱搭不理。

很多人会误解这个原则。以下是他们对“表达对方感受”的错误理解：

*   我知道你是怎么想的，我真的知道自己做错了。
*   所以说我要去改变，你相信我好不好呀。

## 信任崩塌与行动证明

仅仅口头表达“理解”和“改变”是远远不够的，因为信任可能已经崩塌。

如果说这一件事情，你第一次犯，人家一定会原谅你。但是很抱歉，如果说这个问题你出现过很多次，你每次都说改，但是你真的改了吗？这个时候去沟通、去跟他说是没有用的，你只能够去用行动去证明你改了。

这个也是为什么很多人说，我已经把姿态放到这么低了，对方还是没有反应。因为你前面已经刺激过他了，你的这些行为在他的心目当中，已经不能够产生非常大的波澜。而且这种行为，本质上你只是去感动对方，道德绑架没有意义，他不会相信你了。

## 打破对方的心理预期

当信任破裂时，强行沟通往往适得其反。这时，我们需要采取更高级的策略。

所以说这个时候，你跟对方拉开距离，先让对方冷静、缓一缓，是一个非常好的选择。因为这个时候他选择和你分手，他一定觉得“我已经不需要你了”，他就用我们之间的这段关系的破裂，来解决他这个问题。

他预判了你的行为，他的心理建设是：**他想着你一定来挽留纠缠**。这些行为通通都是在他的心理建设当中的。

所以说在这个过程当中，我们不要当下去解决，不要去满足他的心理的一个预期和建设，不要让他觉得跟你分手，他是对的。要打破他的框架，要不然他一直都是跟着他自己的想法去走，没有办法去跟你产生同频，也就不会去跟你沟通，就算你跟他说出话来，他也不会相信。

## 课程总结与最终建议

本节课中，我们一起学习了分手后对方的心理状态和沟通的核心要点。

所以分手之后，不要太让他钻到自己的框架里去、钻牛角尖，否则你很难去跟他沟通，很难去复合。

最后再强调一遍，分手之后没有必要太过纠缠他的想法：“他还喜不喜欢你”、“他还在不在乎你”，纠结这种事情没有意义。还不如好好地想着，如何去跟对方沟通、去解决问题，这个才是真正意义上的挽回。

**核心行动指南**：停止纠缠 → 接受分手现实 → 给予双方空间冷静 → 未来通过**行动**而非言语证明改变 → 在合适的时机进行共情式沟通。
