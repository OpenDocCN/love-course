# 情感修复课 01：如何应对“固着心态”的前任？💔

在本节课中，我们将要学习一种在情感挽回中常见且棘手的情况：面对一位既不拉黑也不删除你，但对你实施“冷暴力”的前任。我们将深入分析这类人的心理状态——“固着心态”，并探讨在这种困境下寻找突破口的唯一有效途径。

## 问题定义：冷暴力与不回应状态

挽回过程中，你可能会遇到一种情况：无论对前任说什么，对方都毫无回应。他既不删除你，也不拉黑你，完全是一种冷暴力的状态。遇到这种状态，应该如何寻找挽回的突破口？

## 核心心态分析：理解“固着心态”

上一节我们定义了问题，本节中我们来看看导致这种行为的核心心理。

要解决问题，首先需要从他的思维角度去理解：他为什么会对你有这样的态度？能做这种事情的人，内心通常非常坚定。这个人很有原则和框架，在心理学上，我们可以将其理解为一种 **“固着心态”**。

**固着心态** 指的是：一个人对某件事的认知一旦形成，就极难改变。例如，我认为一个东西是方的，它就绝对不会是圆的。无论你怎么向我解释它是圆的，也改变不了我的看法。

以下是拥有“固着心态”者的几个关键特征：

*   **内心极度固执**：非常有主见，很少能轻易接受他人的影响。
*   **内心坚定且有韧性**：对自己的判断深信不疑。
*   **认知固化**：如果结合你过去的行为，让他认定“你就是这样的，你改变不了”、“我们不合适”、“没有未来”，那么“我们不可能再成为恋人”这个认知就会深植其心。
*   **不断强化偏见**：他会不断用这个固化认知来深化对你的理解。

因此，之前你任何试图说服他、改变他想法的言行，都会以失败告终，他根本不为所动。

## 唯一突破口：引导自我认知改变

既然外部说服无效，那么我们该怎么寻找突破口呢？唯一的办法是：**让他自己改变对你的看法**，并且这种改变必须是潜移默化、发自他内心的。

具体如何操作？关键在于：**在不暴露需求感的前提下，通过合理化的相处过程，将你改变后的状态持续展示给他**。

这种状态的展示，绝非一两天之功。展示周期完全由对方心态的固执程度（问题难易程度）决定。根据经验，这个过程可能持续一两个月，甚至长达半年，但最终确实有人成功挽回。

这个方法如果运用得当，非常有效，但因为它耗时很长，很少有人能坚持下来。所以，如果你确实遇到了一位有“固着心态”的前任，并且真心想要挽回，那么在情感修复时，你大概率需要使用这个方法，而且这可能是**唯一**的途径。

**核心行动公式**：
```
挽回成功 = 不暴露需求感 + 合理化相处 + 持续展示真实改变 + 坚持（时间）
```

## 课程总结

本节课中，我们一起学习了如何应对拥有“固着心态”的前任。我们了解到，面对他们的冷暴力，任何直接的劝说和解释都是无效的。唯一的突破口在于放弃说服，转而通过长期的、无压力的自我状态展示，潜移默化地引导对方自己改变对你的固有认知。这个过程考验的是你的耐心、真诚和真正的改变。
