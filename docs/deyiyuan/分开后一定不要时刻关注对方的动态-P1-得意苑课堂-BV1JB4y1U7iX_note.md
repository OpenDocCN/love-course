# 情感修复课程 01：理解分手后的关键阶段 🧩

在本节课中，我们将要学习分手后情感变化的几个关键阶段。理解这些阶段有助于你采取正确的行动，避免因错误操作而错失复合的机会。

## 排斥期：情绪的对立与拉扯

上一节我们介绍了课程概述，本节中我们来看看分手后的第一个阶段——排斥期。根据分手情况和感情程度的不同，主动提出分手的一方会决绝地想要逃离这段关系。而被分手的一方，往往会陷入疯狂求复合的状态。当两人经历这种情绪上的激烈拉扯后，关系便会进入下一个关键时期。

## 冷静期：复盘与反思的关键阶段

在经历了情绪化的排斥期后，双方会进入分手过程中最为关键的时期——冷静期。进入冷静期后，双方都会对分手的原因以及分手后的各种行为进行复盘。

以下是冷静期中常见的思考内容：
*   思考自己在感情中有哪些不足之处。
*   反思自己哪些地方做错了。
*   考虑哪些方面需要改进。

如果在排斥期一味求复合并遭到反复拒绝，那么在冷静期，你很可能也会产生怀疑。你会思考对方是否真的不再爱你，是否应该放弃，觉得“不行就算了”。很多试图挽回的人正是在这个阶段选择了放弃，导致原本可能复合的感情成为遗憾。

与此同时，你的前任在经历了情绪爆发期后，同样会开始回想你们之间的感情。他可能会思考自己是否也有过错，是否对你要求过多，是否没有给予足够的耐心和理解。此时，他也会进入一个非常纠结的状态。

## 窗口期：关系走向的决定时刻

当双方都经历了冷静期的反思后，关系便可能进入第三个时期——窗口期。此时，前任会开始考虑是否要复合，并可能会测试你的态度。

以下是窗口期可能出现的一些表象行为：
*   重新添加你的联系方式。
*   悄悄观察你的社交动态。
*   突然打电话给你以试探你的态度。

很多人会将这些行为误解为对方也想复合，于是又开始各种急切的操作，这反而可能让前任更加坚定分手的决心。需要明确的是，这仅仅是分手后的一个“窗口期”。在这个时期，前任从完全拒绝你的状态，转变为愿意重新考虑这段关系的状态。**这段感情最终能否复合，很大程度上是由这个时期双方的表现所决定的。**

---

本节课中我们一起学习了分手后情感发展的三个关键阶段：**排斥期**、**冷静期**和**窗口期**。理解每个阶段的特点和对方的心理状态，是采取正确行动、避免挽回误区的基础。记住，在窗口期保持冷静和恰当的表现至关重要。
