## “拉黑删除”的本质与真正结束的标志

拉黑删除真的不是想要跟你结束的意思。相比你这个人而言，她更爱的是你们的回忆。

以下是区分“假性分手”与“真性分手”的关键标志：

*   **假性分手（当前讨论的行为）**：行为反复（拉黑/删除/加回），情绪驱动，目的是**引起重视和获取安全感**。

所以，如果你是因为这样的情况分手的，那么你一定要去复合，听得懂吗。
