## 潜在信号解析

上一节我们概述了课程目标，本节中我们来具体看看这些潜在的复合信号有哪些表现及其背后的心理动机。

    *   **潜在心理**：这并非纯粹的远离，而是希望看到你为此表现出不离不弃的态度。这是一种对你在意程度的测试。

2.  **躲避与拒绝**
    *   **表现**：在核心问题解决前，她可能躲避见面、逃避沟通或拒绝你的联系。

        ```python
        if 根本问题未解决:
            复合状态 = 暂缓  # 避免重蹈覆辙
        else:
            复合状态 = 可考虑
        ```

3.  **言不由衷的绝情话**
    *   **表现**：她可能会说“我不爱你了”、“我们没可能了”或指责你的种种缺点。

4.  **情绪仍受你影响**
    *   **表现**：拉黑等激烈举动本身是关键信号。
