# 课程名称：识别不健康恋爱关系的迹象 - P1 🚩

在本节课中，我们将学习如何识别恋爱关系中一些可能不健康或存在问题的行为迹象。了解这些迹象有助于你更好地评估自己的关系状态。

## 不公开你的存在

上一节我们介绍了课程概述，本节中我们来看看第一个具体迹象：伴侣不公开你的存在。

如果你的伴侣不在朋友圈公开你们的关系，可能有一些合理原因。例如，你们在一起时间较短，他认为关系尚不稳定。或者，他本身不喜欢在社交媒体上分享私人生活。也可能是工作原因导致不便公开。这些情况都可以理解。

然而，如果你的伴侣**刻意隐藏**你的存在，这就绝对有问题。以下是几种具体表现：

*   打电话时，他总是偷偷摸摸去外面接听，故意避开你。
*   在情人节等特殊日子，他会在朋友圈发布暗示自己单身的内容。例如，明明你们共进了晚餐，他却发“享受一个人的夜晚”这类文字。
*   每当手机响起，他都表现得特别紧张。
*   他会频繁地清理手机聊天记录。

## 评估与行动建议

上一节我们列举了“刻意隐藏”的具体行为，本节中我们来谈谈如何评估并采取行动。

如果你的伴侣出现了上述列举中的一两种行为，你就需要认真考虑是否要更换伴侣。

如果你的伴侣**同时具备**了上述所有或大部分行为，那么建议你尽快结束这段关系。

---

本节课中我们一起学习了如何识别伴侣“刻意隐藏关系”的不健康行为，并了解了根据这些迹象的严重程度来评估关系、考虑采取行动的基本思路。保持清醒的认知是维护健康关系的第一步。
