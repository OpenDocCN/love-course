## 核心原则：理解与平衡

上一节我们介绍了课程主题，本节中我们来看看回应的核心原则。关键在于理解对方需求背后的情感，并在满足需求与维护自我边界之间找到平衡。**公式**可以概括为：**得体回应 = 理解意图 + 表达关心 + 明确立场**。

## 具体回应步骤

以下是应对女生让你买礼物请求的三个关键步骤。

1.  **确认需求与表达关心**

    *   同时，表达你对她喜悦的在乎，可以说：“看到你喜欢的东西，我也很开心。”

2.  **根据关系阶段灵活应对**
    *   如果处于恋爱初期或追求阶段，可以侧重心意而非价格。提议：“我觉得亲手做的礼物更有意义，不如我们一起DIY一个？”

3.  **引导转向情感互动**
    *   将话题从物质礼物转向情感投入和共同体验。可以提议：“礼物是一方面，但我更想为你花时间，这周末我们一起去那个你提过的地方逛逛怎么样？”
    *   强调你重视的是彼此的陪伴和共同创造的回忆。
