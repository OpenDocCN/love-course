# 恋爱经济学入门：为什么“好白菜”总被“猪”拱？ 🐷🥬

在本节课中，我们将通过一个简单的“等级模型”，分析恋爱市场中一个常见的现象：为什么条件出众的女生（“好白菜”）有时会和看似不匹配的男生（“猪”）在一起。我们将拆解其中的决策逻辑和心理因素，帮助你理解背后的规律。

## 第一部分：恋爱市场的“等级匹配”模型 📊

上一节我们提出了核心问题，本节中我们来看看一个用于分析的基础模型。

假设我们将人群依据综合条件（如颜值、身材、经济等）分为四个等级：**A**（最高）、**B**、**C**、**D**（最低）。这个分类对男性和女性都适用。

通常，在恋爱初期，男性相对主动。因此，我们可以按照男性的竞争力从高到低（从男A到男D）来模拟他们的选择顺序。

以下是他们的选择逻辑：

1.  **男A的选择**：理论上，男A的最佳选择是女A。但在实际追求中，他会考虑成功率、时间与物质成本。在“退而求其次”的心理下，**男A往往会选择追求女B**，并且成功率很高。
2.  **男B的选择**：当男A们选择了女B，男B的最佳选择（女B）大多已被占据。因此，**男B通常会转向追求女C**。
3.  **男C的选择**：同理，**男C最终会选择女D**。

这个链条导致了一个直接结果：**最高等级的女A被“剩”了下来**。同时，最低等级的男D在常规模型中失去了匹配对象。

## 第二部分：打破常规的“逆袭者” 🏆

上一节我们介绍了常规的匹配模型，本节中我们来看看模型之外的“意外”情况。

于是，一种令人惊讶的现象诞生了：**女神A（女A）最终可能被屌丝男D（男D）追到**。但并非所有男D都能成功，他们分化为两类：

*   **成功者**：拱走“白菜”的“猪”。
*   **大多数**：目睹“白菜”被拱，心中愤愤不平的旁观者。

这就引出了我们要解答的核心问题：**为什么成功的那个“男D”不是你？**

以下是成功“逆袭者”通常具备的三个关键特质：

### 1. 心态：不自我设限

**核心公式：成功概率 ≈ 尝试次数 × 基础概率**

面对心仪对象（女A），普通男生首先想的是：“我配不配？”而逆袭者想的是：“我该怎么追？”前者因害怕失败（浪费精力、得不到回报）而不敢行动，概率直接降为0。后者无所畏惧，敢于尝试，从而获得了非零的成功概率。

### 2. 特质：幽默、皮厚、有才华

*   **幽默**：决定了相处的舒适度。如果你总是把天“聊死”，关系就很难推进。
*   **皮厚（抗打击能力强）**：追求本身可能伴随拒绝。心理素质差，容易中途放弃。
    > 就像历史中的朱元璋，出身低微却成就大业，没有强大的心理素质是无法完成“革命”的。
*   **有才华**：这是超越外表吸引力的关键。才华能有效淡化外貌的不足。
    ```python
    # 简化的吸引力计算公式
    综合吸引力 = 颜值 * 权重1 + 才华 * 权重2 + 财富 * 权重3
    # 当才华值(权重2)足够高时，可以弥补其他项的不足
    ```

### 3. 行动力：立刻行动，边做边学

成功者信奉“先完成，再完美”。他们在当下就展开追求，并在过程中不断调整和提升自己。即使暂时“装”，也会努力装得自然，并最终将这种提升内化。

而许多人则停留在抱怨层面：“女生都只看钱和脸”。但请反思：你自己是否愿意接受一个各方面不足且不愿进步的对象？如果不愿意，又凭什么要求对方接受这样的你呢？

## 总结与行动指南 🚀

本节课中我们一起学习了“恋爱等级模型”和逆袭者的成功秘诀。核心要点如下：

1.  **常规匹配存在链条**：A男选B女，B男选C女，C男选D女，可能导致A女和D男在模型内落单。
