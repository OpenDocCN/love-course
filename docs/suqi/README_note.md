# 课程一：Python基础入门 🐍

在本课程中，我们将学习Python编程语言的基础知识，包括变量、数据类型和基本操作。这些是构建更复杂程序的基石。

## 概述

我们将从最基础的概念开始，逐步了解如何在Python中存储和处理信息。

## 1. 变量与赋值

上一节我们介绍了本课程的目标，本节中我们来看看如何存储数据。

在Python中，变量用于存储信息。你可以把变量想象成一个贴有标签的盒子，标签是变量名，盒子里装的是数据。使用等号 `=` 可以给变量赋值。

```python
name = "苏七"
age = 25
```

以上代码创建了两个变量：`name` 存储了字符串“苏七”，`age` 存储了数字25。

## 2. 基本数据类型

理解了如何存储数据后，我们需要认识数据的几种基本类型。

Python有几种内置的核心数据类型，用于表示不同种类的信息。

以下是三种最常用的基本数据类型：
*   **整数**：表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：表示文本信息，由单引号 `‘ ’` 或双引号 `“ ”` 包围，例如 `“你好”`, `‘Python’`。

## 3. 基本运算

知道了数据的类型，我们就可以对它们进行运算了。

Python支持对数字进行数学运算，对字符串进行连接等操作。

以下是常见的运算符示例：
*   **加法**：`+`， 用于数字相加或字符串拼接。
    *   公式：`result = number1 + number2`
    *   代码：`sum = 5 + 3  # 结果是8`
*   **减法**：`-`， 用于数字相减。
    *   公式：`result = number1 - number2`
*   **乘法**：`*`， 用于数字相乘或重复字符串。
    *   代码：`message = “hi” * 3  # 结果是“hihihi”`
*   **除法**：`/`， 执行除法并返回浮点数结果。
    *   公式：`result = number1 / number2`
    *   代码：`quotient = 10 / 2  # 结果是5.0`

## 总结

本节课中我们一起学习了Python的基础知识。我们了解了如何使用变量来存储数据，认识了整数、浮点数和字符串这三种基本数据类型，并尝试了加法、减法、乘法和除法等基本运算。掌握这些概念是编写Python程序的第一步。
