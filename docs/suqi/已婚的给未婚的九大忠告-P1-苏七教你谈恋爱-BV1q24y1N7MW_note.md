观察她对生活的态度是乐观还是悲观。一个人的幸福与否，很大程度上取决于他对自己生活和工作的满意度。

如果她对工作和生活都不满意，就不要与她交往。当然，不是每个人都能过得很好，但一个重要的判断标准是：**她是否乐观**。一个悲观的人会把你也变得悲观，而且很难有好的生活和工作状态。悲观的人有悲观的人生，这是自然的，所以要找一个乐观的女人。

### 八、婚姻观念：寻找“门当户对”的伴侣

她必须是一个喜欢婚姻、愿意跟你谈未来生孩子和生活的人。这里要注意：**你们俩要门当户对，是同一个阶层的人，有共同的生活愿望和基础**。

要找一个能够从整体考虑你们关系的人。她的思维模式应该是“**我们**一起会怎样”，从两个人的利益共同体出发，而不是永远只考虑“我自己”——自己的感受、自己的好处、自己的利益。

同样，一个聪明的女人也是奔着结婚、生小孩、经营人生和家庭去的，而不是搞些没用的东西。
