## 利用心理战术追求女生
### 理解心理战术
想成功追到一个女生，让她对你投怀送抱，你得学会利用她，即让她为你付出。

### 案例分析
- **同事案例**：学员利用迟到找同事帮忙，月底请同事吃火锅作为答谢，最终赢得芳心。
- **养宠案例**：学员利用女生爱养小动物的心理，请求女生帮忙照顾宠物，之后请女生吃饭并带她去猫咖，最终赢得女生全心的照顾。

### 心理战术的误区
- **过度付出**：毫无底线的对女生好，效率极低。
- **小套路**：玩点小套路，让女生先得到一些好处。

### 核心概念
**公式**：追求女生 = 自我提升 + 心理战术

## 女生心理分析
### 女生心理
- **忽冷忽热**：当你越是对女生示好，女生越是对你忽冷忽热，甚至不在意你的感受。
- **自我提升**：那些把时间精力，都花在自己身上的男生，更能吸引女生的注意力。

### 核心概念
**公式**：女生心理 = 自我提升 + 示好
