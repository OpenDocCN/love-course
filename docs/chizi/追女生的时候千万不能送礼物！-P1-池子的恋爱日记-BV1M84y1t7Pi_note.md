# 恋爱教程 01：追求阶段为何应避免送礼？🎁

在本节课中，我们将要学习一个在追求女生时常见的误区——过早赠送礼物。我们将分析其背后的心理机制，并解释为何这一行为可能导致关系降温，而非升温。

最近有一位兄弟向我咨询。他说在追求女生时基本没犯错，相处也挺舒服。但就因为送了一次礼物，女生便开始慢慢疏远他。现在主动找女生，对方要么敷衍回应，要么直接不回消息。

其实，超过90%的男生在追求阶段都送过礼物。但其中绝大多数，在送礼后不到两周就会被女生冷落。请回想一下，你是否也有过类似经历。

即使你们聊得很好，甚至关系即将升级，一旦你送出礼物，关系就可能迅速下滑。初期表现为女生不再主动、聊天频率降低，逐渐发展成敷衍、冷漠，甚至不再回复。

你可能想破脑袋也意识不到，这正是送礼导致的。在你尚未吸引到女生时，送礼是一种激进的示爱行为。它不仅会暴露你的需求感，还会让女生产生排斥心理。

如果女生只把你当朋友，她接受礼物时会感到压力。接受礼物仿佛意味着接受你这个人。万一以后不再联系，这份礼物会让她为难。

即便女生接受了礼物，也不一定接受你这个人。如果你遇到的是交往广泛的女生，你送出的礼物很可能如同石沉大海，毫无回响。

只有在成功吸引女生，或正式交往之后，送礼才能发挥积极作用。它能维持感情，并让感情持续升温。在此之前，送礼等同于变相表白。如果女生不喜欢你，她在拒绝礼物的同时，很可能也会拒绝你这个人。

许多兄弟在追求女生时，总想着如何讨好对方，如何让女生觉得自己很好。于是他们一上来就各种关心、送礼。殊不知，这在女生看来是一种极其掉价的行为。

为什么？因为对她好的男生可能很多。如果谁对她好，她就和谁在一起，那她早就不是单身了。因此，关键在于学会吸引她，而非单纯讨好。
