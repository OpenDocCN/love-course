# 恋爱基础课 01：如何隐藏需求感？ 🤫

在本节课中，我们将要学习追求心仪女生时至关重要的第一步：**隐藏需求感**。这是建立吸引、避免压力、让关系自然发展的基石。

很多男生在追求女生时，控制不住自己对女生好，经常把“我喜欢你”写在脸上，几乎等同于直接表白。但你的高需求感不仅不会让女生喜欢你。

还会给女生带来很大的压力，导致你很难追到喜欢的女生。女生喜欢你的本质是**吸引**。一个女生喜欢你，一定是被你身上的优点所吸引了。在女生没有被你吸引之前，你的表白再真挚，女生也不会从不喜欢你变得喜欢上你。

所以，喜欢一个女生不要让她知道，要偷偷地喜欢并隐藏自己的需求感。

上一节我们明确了隐藏需求感的重要性，本节中我们来看看具体该怎么做。核心在于：**先建立吸引力，再采取行动**。

直到你建立起自己的吸引力，做好准备以后，你就可以准备追求女生了。这时，可以把你学过的聊天技巧、互动方法通通用一遍，主要的作用就是**展示自己**，吸引女生。

爱情其实就像一个天秤，公式可以表示为：
**关系稳定度 ∝ (你的付出 × 她的回应)**
只有双方保持平衡，关系才可以平稳地发展下去。

以下是维持感情平衡需要注意的几个要点：

感情需要一方付出，另一方给予回应。无论哪一方，过度付出都会打破彼此的平衡。

女生不仅不会接受你的好，还会把这一切当成理所当然。这样你就很难追到女生了。
