# 恋爱心态课 01：掌握核心心态，自然吸引女生 💖

在本节课中，我们将要学习追求女生时最关键的一环：心态建设。一个正确的心态是成功吸引女生的基础，它能让你在互动中更自然、更自信，从而避免许多常见的错误。

## 摒弃刻意，回归自然

上一节我们介绍了心态的重要性，本节中我们来看看如何摒弃刻意行为，回归自然状态。

男生追女生一定要保持好心态。要让自己看起来更自然，不要太刻意。不要装得连自己都不认自己。控制好自己的感情，放轻松。不要过于紧张，也不要太拘谨。放下你之前在她面前塑造的那么多的条条框框。不要当什么高冷男神。

## 避免“角色扮演”，做真实的自己

理解了自然的重要性后，接下来要避免陷入“角色扮演”的误区。

也不要做什么贴心暖男。这些在女生不喜欢你的时候，都会让女生感觉到很刻意。她会觉得你对她的好都很刻意。

以下是几个刻意行为的例子：
*   在天冷的时候，脱下自己的外套给她。
*   她下班的时候，等她一起送她回家。

你又不是她的男朋友。你这么刻意，女生不好意思拒绝就算接受了，也只是会觉得亏欠了你，欠了你的人情。

那么，如何表达关心才不显得刻意呢？以下是自然关心的正确示范：

## 平衡投入，提升自我价值

过度关注对方会让自己心态失衡，因此需要平衡精力分配。

不要把精力都放在女生身上。投入越多，你越难保持平常心。把多出来的精力用来提升自己。也可以多接触几个女生。

这个时候你就会发现，即使女生不回复你信息，你也不会患得患失。这种心态反而更能吸引女生。而且即使你追不到，你也不至于在深夜一人流泪，独自买醉。

## 理解吸引的本质：价值先行

核心吸引力公式可以概括为：**吸引力 = 自我价值 / 需求感**。过高的需求感会显著降低你的吸引力。

喜欢你的女生，你做什么都会配合你。不喜欢你的女生，你给她摘下星星来，都是错的。

所以提升自己，保持好吸引力。当你追女生的时候，稍微表露一些需求感。女生如果喜欢你，自然会向你走来。

## 警惕“讨好”陷阱，学习“吸引”方法

许多人在追求初期容易陷入“讨好”模式，这恰恰是吸引力的大敌。

很多男生在追女生的时候，总是习惯性地先想着怎么讨好女生，怎么让女生觉得自己很好。于是一上来就各种关心送礼物。
