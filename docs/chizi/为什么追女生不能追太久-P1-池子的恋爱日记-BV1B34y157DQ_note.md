## 🔍 追求时间与成功率的关系

上一节我们介绍了课程的主题，本节中我们来看看追求时间与成功率之间的具体关系。

追女生的时间周期越短，成功率越高。这是总结众多经验后得出的结论。

也就是说，在一周内追到女生的**最大化成功概率**，比一个月内追到的概率要大。而一个月内的**最大化成功率**，又比一年的成功率要大。如果你追一个女生超过一年仍未成功，那么你转身离开的机会不到两成，甚至可能不具备备胎的资格。

## ⏳ 吸引力窗口期

了解了时间与成功率的关系后，我们来深入分析其背后的原因：吸引力的窗口期。

我所谈过的女朋友数量已不少，但基本都在7天以内确立关系。因为一旦超过一周仍未追到，我的基础吸引力就会开始下降。此后无论做什么，效果都微乎其微。

这个问题源于**吸引力**的本质。一个人对陌生的异性可以快速产生强烈的吸引力，但这种吸引力来得猛烈，去得也突然。它是一种短暂的情趣，如果未能及时升级关系，就会迅速消退。

**吸引力变化模型**可以简化为：
```
初始吸引力 → 及时互动与升级 → 关系确立
初始吸引力 → 未能及时升级 → 吸引力消退 → 关系冷却
```

## 📉 从热络到冷淡的转变

你可能有类似经历：遇到一个女生，她起初态度热情，回复信息迅速，甚至主动了解你。你们曾有一段时间无话不谈。

但突然某天之后，她不再主动联系你。当你再找她时，她的态度变得十分冷淡。两人从无话不谈变为无话可谈。此时你发现自己已喜欢上她，但你们之间仿佛隔着一层无形的阻碍。

为什么会这样？追女生如同逆水行舟，不进则退。在热度很高时，没有及时推进关系，后面必然会进入冷淡期。一旦进入冷淡期，吸引力就很难再起作用。

## ⚡ 追求策略：趁热打铁

基于以上分析，我们明白“顺其自然”需要懂得分寸，但追求女生一定要趁热打铁。

不要认为认识时间短，女生就一定会拒绝。如果女生说“我们才刚认识，还没准备好”，那你就不应继续拖延。很多男生在相处中习惯性讨好女生，关心送礼无微不至，但在表达心意时却总遭拒绝。

这类男生或许心地善良，但方法往往无法吸引女生。追求需要策略和时机。

以下是追求中应避免的常见误区：
*   误以为时间长能培养感情。
*   过度讨好而非展示吸引力。
*   在对方态度冷却后仍盲目坚持。
*   不敢在吸引力高峰时升级关系。
