# 女生会跟长得丑但是对她很好的男生谈恋爱吗？-P1-池子的恋爱日记-BV1He4y1b7gt_note

在本节课中，我们将要学习如何正确看待外貌在恋爱关系中的作用，并理解除了单纯“对女生好”之外，哪些核心特质才能真正建立吸引力。

---

关于“丑”的定义远比“帅”要苛刻。“丑”能作为“帅”的反义词。如果说“帅”是一个让人一眼就喜欢上的特质，那么“丑”就是一眼就能让人讨厌的特质。但很多人其实都只是在人群中不起眼的程度，远远没有到达“丑”的这种境界。

真正“丑”的人经历过被人因为外貌议论，甚至以“丑”为理由被骂，他们更能真实地了解这个看脸的世界。你无法否定人的内心深处都是欣赏美的。

所以，“帅哥”更受女生欢迎这个是毋庸置疑的。而“丑男”呢，确实也会降低给对方的第一印象，从而降低你追女生的概率。

这么说你可能体会不出来，咱们可以举个例子。

以下是基于例子的推演：
*   如果一个长得很像凤姐的人来追你，对你很好，你会同意吗？肯定不会，对吧。
*   那如果一个长得很像凤姐的人，他对你非常非常的好，每天中午给你带饭，晚上给你洗衣服，你会同意吗？可能会有一点点的动摇。
*   那来一句狠的，一个长得像凤姐的人，他是勇敢、坚强、善良、乐观、聪明、上进、博学，身价几亿，你会同意吗？这个时候，会有小部分人动摇了吧。

尽管做不到让所有人都喜欢你，但是只要你足够优秀，就能让小部分人喜欢上你。

上一节我们讨论了极端例子，本节中我们来看看现实中的选择逻辑。虽然说不会跟“长得丑但是对我好”的人在一起，但会跟以下类型的男生在一起：

以下是能构成吸引力的特质列表：
*   长得丑，但很有上进心的男生。
*   长得丑，但是幽默风趣、很能哄我开心的人。
*   长得丑，但是持家、做菜样样精通的男生。
*   同样也会跟长得丑，但是一直努力活着、未来有着无限可能的男生在一起。

但不会跟一个长得丑、只会对我好的男生在一起。

其实女生讨厌你，多半还是因为你的**吸引力**不够，不能给她一种心动的感觉。吸引力可以从形象、情商、聊天这些方面来提升。

如果你实在不知道如何改变，那可以寻求指导。核心公式是：**吸引力 = 外在建设 + 内在价值 + 社交技能**。

---

本节课中我们一起学习了：外貌只是第一印象，并非决定性因素；单纯“对你好”不足以构成吸引力；真正的吸引力来源于多元化的内在价值与特质，如上进心、幽默感、生活能力与未来潜力。提升自身综合价值，才是建立健康恋爱关系的关键。
