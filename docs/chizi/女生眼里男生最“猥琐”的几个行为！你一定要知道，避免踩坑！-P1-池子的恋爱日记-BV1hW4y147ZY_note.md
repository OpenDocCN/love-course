# 恋爱避坑指南 01：女生眼中最“减分”的三种行为 🚫

在本节课中，我们将要学习在女生眼中最不讨喜、甚至会被定义为“猥琐”的三种男生行为。这些内容基于真实的女生群体调研，旨在帮助你识别并避免这些社交雷区，从而建立更健康、更受欢迎的异性关系。

上一节我们介绍了课程的目标，本节中我们来看看具体是哪三种行为需要特别注意。

## 1. 关系未到时的越界肢体接触 🤚

在双方关系尚未建立足够好感与信任时，贸然进行肢体接触是首要禁忌。你可能认为轻微的肢体接触（如摸头、碰肩）或故意用身体阻挡去路能拉近关系、制造暧昧氛围。

然而，这种行为忽略了一个大前提：**女生是否已对你建立好感**。如果答案是否定的，你自认为的幽默或有趣，在女生看来很可能等同于“占便宜”和“油腻”。

**核心错误示范**：若女生已明确表示反感，你却回应“看一下怎么了，我又不会把你怎么样”。这种言论会极大加剧负面印象。

## 2. 目光不礼貌地游移 👀

许多男生在遇到外貌出众的女生时，会不自觉地控制眼神。上下打量、目光在敏感部位游移，甚至与同伴低声讨论，这些行为都非常失礼。

**关键认知**：女生对停留在自己身上的目光异常敏感。即使没有实际的身体接触，这种被审视的感觉可能比触碰更令人不适。如果被发现后眼神闪躲，反而会坐实你的心虚与不尊重。

## 3. 表白失败后的不当言行 😤

追求被拒后感到沮丧是人之常情。但有一部分男生会将拒绝视为“天大的委屈”，进而向周围人不断诉苦、抱怨，甚至诋毁女生。

**错误行为公式**：`被拒绝` -> `四处诉苦/诋毁` -> `影响对方名誉` -> `给对方留下“得不到就想毁掉”的极端印象`。

这种行为不仅对女生的名誉造成伤害，也会彻底暴露你的低情商与偏执，让旁观者对你的评价急剧降低。
