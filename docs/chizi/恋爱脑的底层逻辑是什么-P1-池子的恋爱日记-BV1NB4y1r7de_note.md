# 恋爱脑的底层逻辑是什么 - 课程P1 🧠💔

在本节课中，我们将要学习“恋爱脑”这一思维模式的本质与底层逻辑。我们将剖析其核心特征、形成原因以及在亲密关系中的具体表现与影响。

---

恋爱脑是一种以爱情至上的思维模式。恋爱脑者会把全部精力、全部时间、全部心思都放在爱情和恋人身上。他们用一种近乎偏执的付出，去换取恋人的回应，恨不得把自己的心掏空献给对方。

上一节我们描述了恋爱脑的表象，本节中我们来看看其内在动因。为什么恋爱脑可以做到这种程度呢？这就要从恋爱脑的底层逻辑开始说起。

**恋爱脑本质上是没有学会爱自己的。** 他们内心渴望获得很多很多的爱，于是他们选择先付出很多很多的爱，试图以此来换取他人的爱作为回报。

---

理解了其内在动因后，我们需要进一步分析这种付出模式的特点。以下是恋爱脑付出的几个核心特征：

恋爱脑的付出是有条件的，是需要对方积极回馈的。他们的付出原则可以概括为：**“只要我付出了，你就必须感激我，必须对我更好、更爱我。”**

不管对方是否需要这份感情，恋爱脑都会一股脑地对你好。在他们的认知世界中，“对你好”这种价值，能很好地体现自己的存在感，并让对方感受到你的不可取代性。

但往往，这份“对你好”的感情，才是最容易被取代的。因为这是一种任何人都能做到的行为，而且别人说不定能做得比你更好。

---

人们常说“越努力越幸运”，“越付出就会有越多收获”。这个逻辑在学习和职场中是合适的，但是在感情中恰好相反。

在感情中，往往越努力就是越得不到，越付出就会越卑微。所以，恋爱脑思维真的特别容易引发感情的“内卷”和“内耗”，让双方陷入不停付出的竞赛，同时也消耗彼此大量的精力。

一旦其中一方无力回报这份过于沉重的爱，这段感情就很容易不欢而散。

---

因此，兄弟们千万不要成为恋爱脑。一旦发现自己付出太多精力在感情上，就要有意识地抽出部分精力去提升自己，去维持自身的吸引力。这样才能让感情更加长远和稳定。
