# 恋爱基础课 01：女生最反感的三种追求方式 🚫

在本节课中，我们将要学习三种在追求女生时最容易引发反感的行为。了解并避免这些行为，是建立良好关系的第一步。

上一节我们介绍了课程目标，本节中我们来看看具体是哪三种行为容易让女生反感。

## 1. 不合时宜的“开车”（低俗玩笑） 🚗

女生与闺蜜私下聊天时，可能会开一些尺度较大的玩笑。但与异性聊天时，她们在这方面会极为收敛。

切记不要无缘无故地开低俗玩笑。不要以为气氛好就可以这样做。在你没有完全取得女生信任、让她了解你的为人之前，开这种玩笑一定会引起女生的反感。

这种行为会让女生觉得你是一个色狼，并且是一个渣男。她会认为你的私生活很混乱，为人轻浮，会对每一个女生都说这样的话。即便你不是这样的人，你也无法解释清楚。

## 2. 关系未定时的“越界”行为 🚧

以男朋友的身份去约束、管制或要求女生，这类越界行为最容易引起女生的反感。

以下是几种典型的越界行为：

*   女生出去玩，你追问她几点回来、几个人去。
*   叮嘱她“别玩太晚，早点回来”。
*   要求她“回来给我发个信息”。

首先，你不是女生的男朋友，她去哪里玩你无权过问。其次，还没在一起你就这样管她，那在一起之后岂不是要完全绑在一起？做事要注意分寸，越界行为不可取。

## 3. 连续不断的“查户口”式提问 📋

你想了解女生，这可以理解。但如果你不停地问问题，从女生的名字问到脚底有多少颗痣，那就有点过分了。

在你们关系不熟时，不停地提问是一种单向索取价值的行为。人的本能是讨厌被索取的。

所以，你和女生聊天的核心应该以**建立吸引力**为主，然后再进行**互相了解**。记住，一定要是互相了解。

公式可以概括为：**有效交流 = 建立吸引力 + 互相了解**

即使你再怎么了解她，只要她不了解你，她就不可能被你吸引，你们也不可能走到一起。

---

其实追女生并没有想象中那么难。你只需要做好一件事：**吸引她**。如果她还没有被你吸引，你就着急让她成为你的女朋友，那么大概率会失败。

很多男生在面对喜欢的女生时，都会陷入这样的困境：无微不至的关心、讨好、送礼物，反而导致女生提不起兴趣，最后被拒绝。

本节课中我们一起学习了三种需要避免的追求方式：不合时宜的低俗玩笑、关系未定时的越界行为，以及连续不断的查户口式提问。记住，建立吸引是关系发展的核心。
