- **现象**：男生24小时都想和女生聊天。

### 3. 保持平衡

- **原因**：过度依赖会破坏平衡，使关系变得不稳定。

- **原因**：朋友关系没有压力，更容易展示自己的价值。

### 2. 逐步增进感情
- **方法**：在日常聊天中调侃，增进感情。

### 3. 展示价值
- **方法**：通过互动展示自己的魅力和优点。
