## 实验：三扇门与人类的选择心理

上一节我们提到了“吊着”现象，本节中我们来看看支撑这一观点的心理学实验。行为经济学家丹·艾瑞里曾进行过一个著名的“三扇门”实验。

实验场景如下：实验室中有三扇门。每位参与者拥有100次点击机会。每扇门后藏有不同金额的奖励，但只有点击打开后才知道具体数额。

实验规则是：当你持续选择某一扇门时，另外两扇未被选择的门会逐渐缩小并最终消失。然而，门消失后，你只需消耗一次点击机会，就能让它重新显现。

**实验核心机制**：
```plaintext
while (有机会) {
  选择一扇门 -> 获得奖励；
  其他门 -> 缩小；
  if (门消失) {
    消耗一次机会 -> 门重新出现；
  }
}
```

实验结果非常有趣：尽管参与者**明知**门不会真正永久消失，他们依然会**控制不住地**消耗机会去让消失的门重新出现，以确保所有选项都“在线”。

实验结论指出，人类本质上是**追求多选项的动物**。即使理性上知道最优解，感性上仍希望为自己保留退路，并且希望这些退路时刻处于可视范围内。

## 从实验到现实：感情中的“吊着”行为

理解了人类对选项的执着，我们就能更好地分析感情中的行为。实验结论恰好解释了为什么女生会喜欢“吊着”男生。

以下是感情中“吊着”行为的具体逻辑：

*   **代价**：为了维持这条后路，女生需要定期花费一些时间和精力去“撩”一下备胎。
*   **收益**：只需付出较小代价，就能让备胎持续保持关注和忠诚，这对女生而言是一笔划算的“投资”。
