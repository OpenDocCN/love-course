## 第一个方法：撤销信息法

上一节我们介绍了课程目标，本节中我们来看看第一个具体方法：撤销信息法。这个方法的核心是利用人们普遍存在的好奇心。

具体操作是，你先随意给女生发送一条语音或图片消息，然后迅速将其撤回。由于女生的好奇心通常比较重，你快速撤回消息的行为，一定会引起她的好奇，猜测你究竟发了什么内容。

如果这位女生比较在乎你，或者对你有一定的好感，她很可能会主动来询问你撤回了什么。当女生主动开启对话时，便意味着她愿意与你交流，此时聊天的主动权就转移到了你的手中。你可以自由地选择想聊的话题。

以下是操作步骤：
1.  选择一条无关紧要的语音或图片消息发送。
2.  在发送后迅速使用应用的“撤回”功能。
3.  等待女生主动询问，或在其产生好奇后自然开启话题。

## 第二个方法：一语道破法

了解了如何激发好奇心后，我们再来看看第二个能直接引发女生情绪波动的方法：一语道破法。这个方法的要点是，准确地说出女生当下可能的状态或行为。

如果你能精准地猜中女生正在想什么或做什么，她会不由自主地产生情绪波动，并对你产生强烈的分享欲。实现这个方法并不困难，关键在于时机的选择。

例如，在上班时间，你可以给她发送：“好好上班，别玩手机了。” 如果在晚上，则可以发送：“快睡觉吧，别玩手机了。” 这样的开场白直接点破了她的潜在行为。

相信即使性格较为高冷的女生，看到这样贴近她实时状态的开场白，也会忍不住立刻回复你的信息。
