## 核心概念
- **舒适感聊天**：通过合适的聊天方式和话题，使双方在交流中感到轻松愉快。
- **话题无限**：通过发散性思维和自然切换话题，使聊天内容丰富多样。

## 聊天技巧

### 1. 深浅适度
- **适用场景**：适用于认识一个月内。
- **目的**：解除女生的防备心理。
- **方法**：用姓名加状态方式开头，如：“嗨，我是XXX，熬夜冠军。”

### 2. 避免直接评价
- **情境**：对方分享状态后。
- **错误做法**：直接评价对方的状态。
- **正确做法**：不要急于评价，保持倾听。

### 3. 主动分享状态
- **情境**：对方分享状态后。
- **方法**：主动分享自己的状态，如：“我一直在忙都还没吃呢，你在吃什么呢？”

### 4. 发散性思维
- **情境**：聊到具体话题，如“吃”。
- **方法**：将话题发散到双方喜好，如口味。

### 5. 自然切换话题
- **情境**：话题聊得久后。
- **方法**：自然地切换话题，如：“我上次在XXX酒吧吃了碗牛肉面，那是我吃过的最好吃的。”
