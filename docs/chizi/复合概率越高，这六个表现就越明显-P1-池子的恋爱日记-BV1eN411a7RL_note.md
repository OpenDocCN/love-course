# **P1 - 池子的恋爱日记 - 复合概率提升指南**

## 概述
在本节课中，我们将探讨如何通过观察前任的行为来判断复合的可能性。

## 复合概率提升的六个表现

以下是六个前任行为，表明复合机会较大的情况：

### 1. 分手未公开
**公式**：\( P_{\text{未公开}} = \text{无公开分手消息} \)

- 对方没有向亲友公开分手消息。

### 2. 社交媒体伤感文字
**公式**：\( P_{\text{伤感文字}} = \text{伤感文字数量} \)

- 分手后在社交媒体上发布伤感文字，数量越多，表明对方越难以忘记你。

### 3. 保留联系方式
**公式**：\( P_{\text{保留联系}} = \text{保留的联系方式数量} \)

- 对方保留了某种联系方式，如QQ、电话号码等。

### 4. 愿意做朋友
**公式**：\( P_{\text{做朋友}} = \text{愿意帮助解决问题} \)

- 分手后表示愿意做朋友，并愿意帮助你解决问题。

### 5. 分享生活点滴
**公式**：\( P_{\text{分享生活}} = \text{分享生活点滴的频率} \)

- 分享无关紧要的事情，如有趣的事或视频。

### 6. 增加接触频率
**公式**：\( P_{\text{增加接触}} = \text{接触频率} \)

- 增加与你的接触频率，如眼神交流或打听你的情况。

## 总结
本节课中我们一起学习了如何通过观察前任的行为来判断复合的可能性。如果你发现前任展现出上述六个表现，那么你们复合的机会就非常大了。如果你需要帮助挽回感情，可以寻求专业建议或分享你的故事。
