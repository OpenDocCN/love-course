# 恋爱脑的内心独白教程 - P1

## 概述
在本节课中，我们将探讨恋爱脑的内心世界，分析恋爱脑男生在追求女生时的心理和行为模式。

**以下是恋爱脑男生的心态：**

- 对女生有很高的期望和要求。

- 女生对他好就开心，女生对他冷淡就猜忌和患得患失。
- 为了得到女生的更多回报，不计成本地付出。

## 恋爱脑的逻辑
**恋爱脑的逻辑可以用以下公式描述：**

```
付出爱 = 获得爱
```

## 恋爱脑的行为
**以下是恋爱脑的行为：**

- 围绕着对方，没有心思学习和工作。

- 失眠、患得患失、自责、怀疑，没有安全感。
- 得不到女生，自己活得也很累。

- 避免成为舔狗。
- 在后台留言“学习”，获取追女生攻略。

## 总结
本节课中，我们一起学习了恋爱脑的内心世界和行为模式。希望这些知识能帮助你更好地理解自己和他人的情感需求。
