## 课程概述

在本节课中，我们将学习一种简单直接的搭讪方法——“直球法”。这种方法强调真诚与自信，避免复杂的套路和尴尬的对话，旨在帮助你高效且自然地获取心仪对象的联系方式。

---

## 核心概念：摒弃套路，直接表达

当前社交环境中充满了各种套路和试探。然而，一颗真诚的“直球”——即直接、坦率的表达——往往比任何精心设计的“土味情话”都更有效。你不必使用诸如“我发现你长得很像一个人……我的心上人”这类可能引发尴尬的台词。

**核心公式**：**搭讪效果 ≈ 真诚度 × 自信度 - 套路复杂度**

---

## 具体步骤与实践

上一节我们介绍了“直球法”的核心思想，本节中我们来看看具体如何操作。整个过程自然流畅，关键在于直接和尊重。

### 第一步：自信接近并真诚赞美

首先，自信地走向对方。保持一个安全的社交距离，避免让对方产生警惕。然后，直接给予真诚的赞美。

**示例代码/对话**：
```
走近，微笑。
“你好，我觉得你很好看/气质很好。”
```

### 第二步：短暂停顿后提出请求

在表达赞美后，间隔几秒钟，让对方消化你的善意。接着，清晰直接地提出你的请求。

以下是提出请求的关键点：
*   **直接说明目的**：清晰表达你想认识对方的意愿。
*   **使用询问句式**：给予对方选择权，体现尊重。
*   **保持简单**：无需额外解释或找借口。

**示例代码/对话**：
```
（等待2-3秒）
“我想加一下你的微信，可以吗？”
```

## 方法原理与结果分析

这种方法之所以高效，是因为它极度真诚。在多数只有一面之缘的场景下，直接是最好的策略。

**结果通常有两种**：

1.  **成功获得联系方式（概率很高）**：如果对方对你有基本好感，通常会愿意接住你这颗“直球”。这意味着你们有进一步发展的可能。
2.  **被拒绝**：如果对方拒绝，这本身也是一种清晰的信号，表明其当下兴趣不高。即使通过复杂套路勉强获得联系，后续发展机会也有限。
