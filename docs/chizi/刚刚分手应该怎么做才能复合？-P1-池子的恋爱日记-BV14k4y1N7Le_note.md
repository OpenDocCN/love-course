# 刚刚分手应该怎么做才能复合？-P1-池子的恋爱日记-BV14k4y1N7Le_note

在本节课中，我们将要学习分手初期，针对两种不同情况（完全断联与尚有联系但回应冷漠）应采取的核心行动策略。理解并正确运用这些方法，是迈向成功挽回的第一步。

## 情况一：完全失去联系 🔌

上一节我们介绍了课程概述，本节中我们来看看第一种常见情况：你与前任已没有任何联系方式，且对方态度决绝。

如果此时此刻你联系不上对方，那么你需要找到一个能重新建立联系的渠道。但关键在于，不可以第一时间就去联系对方。

以下是具体操作步骤：
*   **寻找联系渠道**：例如，对方虽将你拉黑删除，但你仍知晓其社交账号、手机号等信息。
*   **延迟行动**：完全有理由过一段时间后，再尝试用小号去添加联系。没有必要急于一时，从而将自己所有的后路都堵死。

## 情况二：尚有联系但回应冷漠 ❄️

在了解了完全断联的处理方式后，我们接下来分析另一种更复杂的情形：你们之间尚有联系窗口，但对方的回应特别冷漠。

相比起第一种情况，这种局面往往更加难以相处。但其好处在于，你们之间保留了一个至关重要的联系窗口，这对于后续复合行动特别重要。

然而，大多数人在这个阶段容易犯错。以下是需要避免的关键错误：
*   **避免自暴自弃**：不要因为对方态度稍微冷漠，就开始恶语相向。
*   **认清真实目的**：明明内心非常想复合，却用错误方式逼迫对方将你拉黑删除，这会将自己置于非常被动的位置。

如果你实在不知道该如何具体着手挽回，可以寻求进一步的指导。

**本节课总结**：本节课中我们一起学习了分手初期两种核心情况的应对策略。对于完全断联，重点是**寻找渠道**与**延迟联系**；对于联系尚存但回应冷漠，重点是**保持窗口**与**避免情绪化犯错**，稳住现有联系是后续一切行动的基础。
