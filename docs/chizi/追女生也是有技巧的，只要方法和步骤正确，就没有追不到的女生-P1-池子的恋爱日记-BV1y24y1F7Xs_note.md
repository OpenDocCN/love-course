# 追女生技巧教程 - P1

## 概述
在本节课中，我们将学习如何正确地追求女生，通过四个阶段逐步建立关系。

## 追求女生四个阶段

### 1. 陌生阶段
**目的**：相互认识。
**行动**：
- 在平等的情况下交换信息。
- 寻找共同点。
- 不要刻意讨好，保持自然。

### 2. 朋友阶段
**目的**：增加互动。
**行动**：
- 多聊天，调动情绪。
- 如果女生内向，多邀约线下见面。

### 3. 好朋友阶段
**目的**：交心。
**行动**：
- 聊心事，交换秘密。
- 约会时去有意义的地方。
- 做有纪念意义的事，制造共同回忆。

### 4. 暧昧阶段
**目的**：试探。
**行动**：
- 慢慢靠近，观察女生的反应。
- 通过肢体接触判断女生的态度。
- 如果女生不排斥，可以表白。

## 总结
本节课中我们一起学习了如何通过四个阶段正确地追求女生。记住，不要急于求成，保持自然，真诚地与女生互动，才能最终赢得她的心。
