# 恋爱技巧课 01：适度冷落，让关系更平衡 💔

在本节课中，我们将要学习一种在恋爱关系中保持平衡的重要技巧：适度冷落。核心在于理解付出与索取之间的动态平衡，避免因过度讨好而丧失吸引力。

## 过度付出的弊端

上一节我们介绍了课程主题，本节中我们来看看为什么不能一味付出。一味的对女生包容、对女生好，女生会越来越令你不在意。只有适当冷落女生，才能让女生更加靠近你。

和女生相处要尽量保持双方付出和索取之间的平衡。一定不要付出太多，付出太多显得太卑微。

## 保持平衡的原则

理解了单方面付出的问题后，我们来看看如何建立健康的平衡。也不能索取太多，那显得太小气太无知。你可以迁就她，但是不能纵容她。

以下是“适度冷落”技巧可以应用的几个关键场景。
