## 一、 网络社交软件

网络社交软件是一个入门简单但精通很难的软件类别。简单的入门只需要几张得体的照片，就可以认识到平时生活圈三倍以上的女生。市面上常见的软件包括探探、积目、陌陌、Soul，微博勉强也算一个。

以下是这类软件的几个共同点：
1.  **入门简单，精通难**：成本低，认识异性的数量大。
2.  **用户低龄化**：年纪较大的朋友可能不太容易玩得通。
3.  **特别现实**：非常看重照片水平或照片中透露出的价值。主流标准是“要么好看，要么有钱”。但如果本身有特点，也可以脱颖而出。

然而，若论及异性质量、见面速度或在快餐式爱情中寻找长期关系，这就变得困难了。我们可以将使用过程分为四个步骤。

### 1. 展示照片

在任何软件上，我们的头像和照片都是最重要的第一印象。干净、整洁、有背景衬托，不仅是一张大头照，能达到这个感觉就是合格的照片。如果能带入一些个人特点与才能展示则是最好的。

以下是几个展示个人特点的例子：
*   **喜欢运动**：可以放一些运动中的照片，或展示运动带来的好身材。
*   **热爱做饭**：可以是一些买菜或与厨房相关的照片，不一定非得展示在油烟中做饭的场景。
*   **喜欢旅行**：照片背景风景够好，我们展示的状态够阳光即可，或是展示旅行纪念品。

照片与个人资料结合，可以强化我们的个人标签。这一点对双方都一样。我们也可以通过对方的展示面了解其状态，并以此作为开启对话最简单直接的内容参考。

例如，就对方的旅行照片，我们可以尝试发出这样的打招呼信息：
```
你的照片背景城市看起来很美，是哪里呢？
```
这种人畜无害的疑问，女生通常没有理由不回答。总之，别人看到的样子，必须是我们想让别人看到的样子。

### 2. 资料与标签

在社交软件上，每位你发出聊天邀请的异性，除了看照片，都会看个人资料与标签。因此，我们希望吸引什么样的异性，决定了我们要展现自己的哪些方面。

例如，如果我们喜欢文艺、高知的女生，就要在个人资料中介绍自己喜欢电影、书籍、知识领域，还可以写明喜欢的生活方式和感情立场。这些都会成为与感兴趣异性之间可聊的话题。

以探探平台为例，除了个人签名和信息，我们还可以设置标签与兴趣。除了预设标签，还可以自己编辑想表达的、与众不同的特点。

例如：
*   `真诚交流，希望遇到单身又可爱的女生`：既表达了真诚态度，也袒露了实际需求。
*   `爱运动、爱美食，希望遇到一位后半辈子一起吃一起玩的女生`：显得态度端正、形象立体，让女生觉得是一位有生活、充满热情的靠谱男生。

### 3. 加微信策略

通过社交软件认识异性的最终目的，还是要深度发展关系。所以无论怎么聊，最终都要把欣赏的、聊得来的异性筛选到微信中，进入下一步的自我展示阶段。

这一步会出现快与慢两种方式：
*   **快一点**：互留微信，快速拉近距离。**好处**是结识异性数量多，可以向更多人展示自己，增加机会概率。**弊端**是牵扯精力。
*   **慢一点**：多了解一些，再进入自我展示体系。这是保守派打法，其**好处**不言而喻：异性质量可能更高，共同话题更多。例如，跟一个女生在软件上聊了一个月，兴趣爱好、工作三观都聊得差不多了，再加微信就可以尝试邀约了。

### 4. 社交态度

在社交软件中，我们存在一种大概率行为：**疯狂右滑**。好处是我们可以拥有永远划不完的头像和很多“配对成功”的机会。但弊端是，这种不筛选、不认真的态度，同样会得到等价的回报。

这里建议大家还是要抱有“认识更多异性、丰富生活社交和寻找一段长期关系”的立场来认真对待社交软件。疯狂右滑、来者不拒的态度，并不会让我们的社交能力与结识异性的质量有所提升。所以请认真对待每一次选择与机会，针对自己喜欢的类型和标签下功夫，才是对自己和他人都负责的行为。

## 二、 相亲平台软件

下面进入我们这节课的第二大点：相亲平台软件。这里我们要先说一下前提：我们的朋友中，有一部分群体觉得自己到了适婚年龄，对爱情抱有期望值较低，只想找一个能一起面对生活琐碎、一起过日子的人。对于这部分朋友，相亲软件完全是可以考虑的途径。

不过要注意以下几点：
1.  **需要经济投入**：通常需要花钱开通会员等。
2.  **目的性强**：男女双方目的性都比较强。
3.  **技巧话术不同**：很多在社交软件上的技巧和话术不再适合这些场景，因为大家的衡量标准更偏向物质。

相亲属性的平台和软件，如上一课提到的世纪佳缘、珍爱网、百合网等，其实就是线上的婚姻介绍所，和线下相亲本质上没有特别大的区别。它们通过严格的个人信息审查、价值大数据的多维匹配和付费VIP身份叠加，帮助用户找到更合适的另一半。

其本质是把男性的**生存价值**（财产）作为第一指标，去匹配女性的**繁衍价值**（外貌等）。公式可以简单理解为：
```
匹配度 ≈ 男性生存价值 × 女性繁衍价值
```
虽然这很现实，但选择权还是在大家手里。有人可能会说，那就不线上相亲了，自己又不是大富翁。但我们对于线上相亲平台这个事情是保持中立态度的。我们可以把它当做一个有趣的经历，一次可以和我们喜欢的女生调侃的谈资。只要没有大投入、不花大钱，参与一下又何乐而不为呢？

## 三、 识骗技巧 🛡️

这个网络时代骗子很多。谁知道和你线上聊天的是一个可爱的女孩，还是胡子都没刮完的抠脚大汉呢？所以这里和大家说说识别骗子的基本常识。

骗子无非就想骗钱。所以，凡是涉及到见面两次以内就让我们花钱的，都需要谨慎。无论是让你购买特产、玩线上游戏/赌场，还是以自己生日或节日为由让你发红包等。

以下是几点常见的明显特征：
1.  **照片模糊**：如果照片像素低、模糊，大概率是网上找的图，因为上传会多次压缩。但清晰也可能是盗图，这只是其中一个指标。
2.  **犯男生聊天雷区**：对面是不是抠脚大汉，可以看他是不是老问问题，查户口（问工作、爱好等）。一旦我们聊点别的，他就岔开话题，接不上玩笑，那对面可能也是一个“钢铁直男”。
3.  **没聊多久就要钱**：的确有部分女生是这样的。这种把男人当作提款机的，无论是谁、是不是骗子，一律无视。情节严重者，建议直接拉黑。**请注意：一个能发展长期关系的女孩，是不会在见面不到两次的时候主动索要礼物和钱的。要么是极度没有安全感，要么与骗子相关。**
4.  **特别主动**：这里指的是总会找各种理由邀约我们，并且连去哪里、干什么都指定好了。这大概率是“酒托饭托”，或是强行推销产品的。可以尝试问一下，如果对方要你参与什么下载和关联，就拉黑吧。
