# 最绅士认识女生：第八课：搭讪 - 打破常规的相遇法则 🚀

在本节课中，我们将要学习如何通过“搭讪”这一非常规方式，打破“男生避之不及，女生趋之若鹜”的固有印象，有效扩大社交圈并增加遇到理想爱情的机会。我们将探讨搭讪的核心优势、具体途径以及正确的实践步骤。

## 概述

国人普遍存在一个自相矛盾的逻辑：坚定地认为结婚应该找一个自己爱的人，并且大多数人觉得相爱是两个人能够结婚的首要前提。

然而，现实中基于相爱而结合的情况却很少。大多数人通过亲友介绍或线上线下的相亲活动，找到一个能一起过日子的人。

那么，令人怦然心动的爱情去哪里寻找？我们不否认通过亲友介绍或婚姻中介有小概率遇到爱情，但这种概率很渺茫。原因很简单：**基数不够大，范围不够广，选择不够多**。

此外，还有一个重要因素：**不够戏剧化**。很大程度上，两个人相识的戏剧化程度，铺垫了之后故事的独特性。戏剧化的相识往往伴随着感性，能增强人们的代入感。

而前面提到的相亲、线上交友等方式，双方都带着理性姿态，过程在意料之内。因此，若想遇到爱情，非常规模式比常规模式概率更大。

所以请记住：无论何时何地，只要我们有颗搭讪的心，都有可能突然遇上自己理想中的异性。事实上，人们常常会在毫无准备的情况下，突然撞上自己梦寐以求的人。

## 一、搭讪的优势与途径 🛤️

搭讪在接触人群数量、戏剧化程度、真实性、时间成本以及人群优质程度上，都有其独特优势。

以下是搭讪的几个核心优势：

### 1. 场景无限，机会众多
首先，我们无需前往特定场合。**所有公共场合都是物色和观察的机会**。我们甚至可以自定义场景和搭讪对象的类型。

*   **想找白领**：在午休和下班时间去CBD即可。
*   **想找有共同兴趣的**：出现在自己喜欢的活动或店铺中即可。
*   **想找不同社会阶层的**：现代社会的信息能告诉我们哪个地点具有哪种属性。

这是其他线上交友方式中的VIP会员都未必能享受到的优待。日常生活中的搭讪几乎不占用额外时间，可能买个早饭都能遇到吸引我们的异性。这种**任意的场合、任意的时间和可操作性**，使得能接触到的人群数量远超线下聚会。

### 2. 真实度得以保证
因为初次相识就能见到对方毫无修饰的真实形象，**真实度得以保证**。而且随着搭讪数量的提升，观察力和社交直觉也会相应提高。通过第一眼判断一个人大致情况的能力会越来越强。

### 3. 成功概率更高
通过搭讪，你能够直面心仪优质对象的机会，比其他任何形式都要高。我们喜欢的他/她，未必参与特定活动、使用某个社交软件或有很广的社交圈，但他/她**一定要上街**。这是两条原本平行线唯一有机会相交的可能性。

这些优势也同时展现了搭讪的**戏剧化程度**。在所有带有刻意为之的相识方式中，搭讪应该是戏剧化程度最高的一种。其他如马路上撞在一起、音乐节邂逅、朋友婚礼相遇，都是纯粹偶然的不可控事件。**搭讪是处在必然和偶然之间的绝佳存在**。

可能有人会说找不到途径。其实，大家只是不敢开口而已。当实在找不到共同话题时，**诚恳且直白**不失为一种最朴实而有力的形式。

例如：
> “你好，很抱歉，我知道这很唐突，但我很想认识你。因为很少有机会见到像你这么棒的姑娘。如果可以的话，我想多了解你。”

事实上，如此坦诚会让好感加分。

还有朋友说自己太宅，很少出门。**那就养条狗**。根据某项研究表示，养狗的人比不养狗的人脱单几率要高70%。虽然听起来离谱，但仔细想想确有道理。当代青年单身的主要原因之一就是认识异性的机会太少。养狗后必须出门遛它，自然就增加了社交圈，恋爱几率便会提高。

若说遛狗都无法接触异性，那可能是方法不对。请早起一点，看到其他遛狗的人就主动打招呼。毛顺的就问用的什么牌子的洗护液，能不能加个微信把链接发来；穿着衣服的狗就问哪里买的衣服，能不能加个微信把店铺链接发来。总能找到共同话题。

有朋友一开始可能觉得这样社交目的性太强。但想要恋爱、摆脱单身，这就是我们想达到的目的。既然目的本身没有问题，为什么不能坦然面对？**想脱单不是一件丢人的事**。抱着新的社交目的去审视现有社交圈，可能会收获意外信息。

## 二、搭讪的正确步骤 🧭

聊完了上面的问题，可能还有一些朋友感到困惑。下面我们来说说搭讪的正确步骤。

以下是实施搭讪时可以遵循的几个阶段：

### 1. 以“请求式”搭讪开场
在书店、画展、电影院等场合，都会有值得搭讪的女生。如果我们开口，这种巧合的交谈或许值得继续。

*   “你也很喜欢这个作者？喜欢她的人好像不多，找到同类很开心。”
*   “你对这幅画怎么看？我们能不能交流一下？我不太了解这幅画的表达。”
*   “你好，我能否请问你对这部电影的评价？我在研究一个关于电影的课题。”

如果男生外形不猥琐，这种看似诚恳、中立、有交流意愿的“请求式”搭讪，女生基本不会拒绝。然后无论对方回答什么，都可以顺着话题延伸，从而获取联系方式。

### 2. 快速搭建聊天体系
如果成功认识了喜欢的人，那就要**快速建立聊天体系**。如果等上十天半个月再联系，对方可能已经忘了你。

其次，要**学会聊天，会找话题**，不要让气氛变冷。这里有一个小技巧：在聊天时**善于自黑**，能把气氛调动起来。如果不能靠外貌吸引她，那就让她觉得你很有趣。

另外，在认识初期要**控制聊天频率**。在没有感觉到她对你也产生好感时，不要24小时缠着对方。最好每隔两到三天互动一次，即一周最多找她聊两到三次。这样既不会暴露需求感，也不会让她忘记你的存在。

当这样聊了一段时间后，女生会养成习惯。如果有一天你没找她，她甚至会在心里想：“哎，这两天他怎么没来找我聊天？”这样的效果更好。

### 3. 适时结束并约定下次
聊天总会有尾声。**在她觉得无聊前结束对话**，她会更加期待下次与你聊天。适当地吊起对方胃口。

为了方便长期联系、沟通感情，可以给她一些提示，告诉对方自己什么时候有空，暗示她也可以主动联系。这样有来有往，大家很容易就变成熟人了。

### 4. 不要害怕被拒绝
对于搭讪失败，其实没什么。因为大家本来就是陌生人，失败了也只是变成不喜欢的陌生人，对我们没有实质性影响。

我们应该调整心态。比如今天遇到了喜欢的人，你搭讪了四个女生，其中两个同意，两个拒绝。不要因为被拒绝而难过或自我怀疑，而是应该想：**有两位你喜欢的异性同时被你吸引了，你怎么这么厉害**。
