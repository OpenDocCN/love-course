# 课程名称：Python基础入门 - 第01课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要一个地方来存放它，这就是**变量**。同时，数据本身有不同的种类，比如数字和文字，这些种类被称为**数据类型**。掌握如何创建变量并了解不同的数据类型，是开启编程之旅的钥匙。

## 1. 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）用来存放数据（值），而标签（变量名）则帮助我们记住盒子里装的是什么。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。

```python
# 创建一个名为 message 的变量，并将字符串 "Hello, World!" 存放进去
message = "Hello, World!"
```

上一节我们介绍了变量的比喻，本节中我们来看看创建变量时需要遵循的具体规则。

以下是命名变量时需要遵守的几个关键规则：
*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名不能是Python的保留关键字（如 `if`, `for`, `while` 等）。
*   变量名是区分大小写的（`myVar` 和 `myvar` 是两个不同的变量）。

## 2. 基本数据类型 🔢

变量可以存放各种类型的数据。Python有一些内置的基本数据类型，最常用的有三种。

### 2.1 字符串（String）

字符串是文本数据，在Python中用引号（单引号 `'` 或双引号 `"`）括起来。

```python
name = "嘉琪"
greeting = '你好，世界！'
```

### 2.2 整数（Integer）和浮点数（Float）

整数是不带小数点的数字，浮点数是带小数点的数字。

```python
# 整数
age = 25
count = -10

# 浮点数
price = 19.99
pi = 3.14159
```

### 2.3 布尔值（Boolean）

布尔值只有两种可能：`True`（真）或 `False`（假）。它们通常用于逻辑判断。

```python
is_student = True
is_raining = False
```

## 3. 查看与转换数据类型 🔧

有时我们需要知道一个变量的类型，或者将一种类型转换为另一种类型。

使用 `type()` 函数可以查看变量的数据类型。

```python
height = 180
print(type(height))  # 输出：<class 'int'>
```

以下是几种常见的数据类型转换方法：
*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

```python
# 将字符串“123”转换为整数
num_str = "123"
num_int = int(num_str)

# 将整数转换为字符串
my_age = 20
age_str = str(my_age)
```

## 总结

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。
我们了解了变量如同一个存放数据的命名盒子，并学习了如何用 `=` 为其赋值。
我们认识了三种基本数据类型：用于文本的**字符串**（`str`）、用于数字的**整数**（`int`）和**浮点数**（`float`），以及用于逻辑判断的**布尔值**（`bool`）。
最后，我们还掌握了使用 `type()` 查看类型，以及使用 `int()`, `float()`, `str()` 进行类型转换的方法。
