# 恋爱心理学课程 01：理性看待“卑微式”挽回 🧠

在本节课中，我们将通过一个社会事件，分析在亲密关系中以极端卑微姿态试图挽回对方的行为背后的心理动因与社会评价。我们将探讨为何这种行为无效，并学习建立健康平等关系的基础认知。

## 事件回顾：雨中长跪的挽回者

近日，一则社会新闻引发了广泛讨论。在四川达州，一名男生为了挽回感情，手持一束玫瑰花在街头长跪不起。

据目击者称，这名男生跪了将近一天一夜，当时天还下着雨。期间有多人前来劝说男生离开，但均未奏效。

他从28号下午一点多，一直跪到了29号上午10点多才离开。警方也曾到场劝其起身，但他并未听从。

他甚至反向告知对方，自己这样下跪并不违法。古语有云：男儿膝下有黄金，只可跪天、跪地、跪父母。

## 行为分析：可怜与可悲的背后

上一节我们回顾了事件经过，本节中我们来看看这种行为背后的心理与社会含义。说实话，这种做法既可怜又可悲。

我们无从得知男生身上具体发生了什么，但这类行为并非偶然现象，似乎已成为一种社会常态。

故事中的男主角，可能只是一个涉世未深的青年，并不真正理解什么是爱情。直到故事中的女主角出现，给了他从未有过的感觉。

他的逻辑可以简化为一个**公式**：
> **“只要能与她在一起”** → **“愿意改变自我，做任何事”**

在这种心态驱使下，他愿意去改变自己，愿意去做那些他从未做过、甚至原本不愿意做的事情。

## 认知偏差：对方眼中的“唯一性”

在旁观者眼里，那位女主角可能只是一位普通的女孩。但在他的认知中，对方是特别且唯一的存在。

然而，他以如此卑微的姿态去恳求，换来的却是对方冷漠无情的回应。很多人嘲讽这位男生，认为他一文不值。

是的，即便他长跪不起，女生依旧选择不予理睬，并用非常恶劣的态度对待他。可是，如此的冷漠和无情，都没能阻止他“热脸贴冷屁股”的行为。

## 核心概念界定：从“恋爱脑”到“执念”

按照常见标签，我本可以给他的行为贴上“恋爱脑”的标签。但现在，我们更倾向于用另一个词来形容这种状态：**关系执念**。

以下是“关系执念”的几个关键特征：
*   **认知窄化**：将全部自我价值与一段关系绑定。
*   **行为极端**：认为通过自我牺牲或极端行为可以改变对方决定。
*   **忽视反馈**：完全无视对方明确的拒绝信号。
*   **自我感动**：将自身的痛苦行为误认为是深爱的证明。

## 总结与启示

本节课中，我们一起学习了如何理性看待亲密关系中的“卑微式”挽回行为。我们分析了其背后的心理动因，即由认知偏差导致的**关系执念**，并指出了这种行为无法换来尊重与真爱，只会损害自我尊严。

健康的爱情应建立在**平等、尊重与双向奔赴**的基础上，其核心公式是：
> **健康关系 ≈ 相互尊重 + 有效沟通 + 共同成长**

任何需要以彻底放弃自尊为代价才能维持的关系，本身就已经失去了存在的基础。希望本课能帮助你树立更理性、健康的亲密关系观念。
