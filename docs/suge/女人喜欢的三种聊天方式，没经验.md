## 概述：聊天问题的核心

99%的人无法让女生爱上你的原因，都出在聊天问题上面。很多人遇到的问题是，女生连天都不想跟你聊。

那么到底该怎么办呢？没关系，苏哥今天就结合一个案例来告诉大家，只要我们跟女生聊这三个话题，女生绝对会抢着做你的女朋友。

## 案例背景与分析

一位28岁、单身两年的男生在游乐场认识了一位女生，并被她的气质吸引。当他尝试找女生聊天时，对方却显得不耐烦并找借口离开。我们首先来看看他的聊天方式存在什么问题。

**兄弟的初始聊天记录：**
```
兄弟： “那一天看到你跟两个小朋友在玩水，是你的侄子吗？”
女生： “对啊是我姐的。”
兄弟： “哈哈差点以为你结婚了，那你为什么没有结婚呢？”
女生： （回了一个尴尬的表情）“你问这个干嘛？”
兄弟： “有点好奇。”
```

### 问题诊断：错误的话题起点

我们跟女生刚刚认识的时候，最主要的聊天方式，就是要消除陌生感。像这个兄弟一样，一上来就聊对方的家庭、聊情感经历，这些私人话题不但不会消除陌生感，反而还会加剧对方对你的防备意识。

所以女生回复：“有什么好好奇的？”很明显，女生不想在这个话题上面纠缠下去。这个兄弟也察觉到女生的不耐烦，然后就慌了，赶紧说：“没有啊，就是想问你有没有男朋友。”

这句话更是错上加错。“关心”这两个字，直接把你的目的暴露无遗。于是女生说：“抱歉我现在不想谈恋爱。”这时，聊天氛围瞬间变得剑拔弩张。

## 核心方法一：从浅层次话题入手，消除陌生感

既然兄弟这种聊法是错的，那么正确的打开方式到底是什么呢？不会聊天的兄弟，还不赶紧拿起你的小本本做好笔记。

前面说过了，当前阶段我们最主要的任务，就是消灭陌生感。所以最好是从浅层次的话题聊起，这样子才能给到这个女生感觉是轻松的，是没有压迫感的。

于是，我让兄弟接着前面的聊天回复。以下是调整后的对话策略：

**步骤1：幽默化解，弱化需求感**
```
兄弟回复：“那太好了，这么说呀，单身狗家庭又多了一个新成员。”
```
我们先通过一个简单的幽默调侃，化解尴尬，同时弱化一下需求感。

**步骤2：转移焦点，消除戒备**
然后我们突然又像刚刚醒悟过来一样，故作惊讶地说：
```
“等一下，你说‘抱歉’是什么意思？你是不是误会了什么了？”
```
为什么我要多说这一句话？就是为了让这个女生以为自己想多了。这样子一波处理之后，我们刚才暴露的需求感，基本上就能够消弭无形了，女生对于我们的戒备也会随之减弱。

事实上也确实如此，很快女生回了：“没有呀，我还以为……”女生的情绪明显是有些缓和了。可见，我们只需要按照正确的聊天节奏跟她聊天，完全是可以聊出好感的。

## 核心方法二：调动情绪，强势推进关系

那么现在我们要怎么回呢？按照很多兄弟的习惯，肯定要跟女生拉家常。这么聊也是可以的，但是这些话题不能给到女生任何的情绪。这个时候，反而是我们强势推进的最佳时机，女生一定会想不到我们会杀一个回马枪。

**步骤3：开玩笑式反问，升温情绪**
于是，兄弟用开玩笑的语气说：
```
“以为什么呀？是不是以为我要给你介绍男朋友？”
```
女生的情绪瞬间升温，回到：“哈哈没有啦，那你干嘛要问这些问题？”可见，这个女生已经相信我们确实是没有目的性的。

**步骤4：保持态度，持续调动情绪**
那么后面这句话我们又该怎么回呢？按照很多兄弟的习惯，肯定又要想方设法给女生解释了。解释什么呢兄弟们？这简直就是一个送分题，只需要保持前面的态度继续前进，就能够很好的调动女生的情绪。
```

```
女生有点好奇，她说：“啊你认真的呀？”
我们回：“你想得美啊，我自己都单身还给你介绍。”
再一次调动女生的情绪，同时呢，悄悄的开起了一波暧昧。

## 核心方法三：开启暧昧话题，建立情感共鸣

那聊到这里呢，我们就不要再去聊什么浅层次的话题了。因为呢，这些话题升温效果是有限的。我们要开启更加深入的话题了。那什么是更加深入的话题呢？经常关注我的兄弟们肯定想到了，那就是暧昧话题。赶紧记下来。

**步骤5：强势开启暧昧，但不暴露需求**
于是呢，我们回到：
```
“啊，你说这话我可就要想歪了。”
```
女生问了：“想歪？往哪里歪啊？”
我们就说：
```
“你说我是自私鬼，不就等同于承认我把你占有了吗？这话呀可千万不要乱说，我会找不到女朋友的。”
```
在这里呢，我们强势的开启暧昧，但是呢又没有暴露任何的需求感。那这种感觉呢，女生是根本招架不住的。

那女生回到：“好了我也不是那个意思，你也别只顾开我玩笑。说真的你为什么不找女朋友呢？”看到没有兄弟们，女生的情绪呢起来之后，他的话题也跟着打开了。而且呢，我们幽默有趣的一面，也成功打动了女生。不知不觉当中啊，女生对我们产生了一定的兴趣跟好感。

**步骤6：推进到情感共鸣，展示自我价值**

```

```
