## 第一宗罪：盲目的表白 💥

上一节我们概述了课程内容，本节中我们来看看第一宗罪：盲目的表白。

在不确定女生是否真正喜欢你的时候，就冲动地去表白，这无疑是将自己的底牌完全暴露给对方。例如，女生同意跟你吃饭看电影，你误以为她对你有意思，便盲目表白。在一起是两个人的事情，单方面的冲动决定往往会导致失败。

以下是盲目表白的两种典型错误话术：
*   **做我女朋友可不可以。**
*   **我喜欢你，我能追你吗。**

这种表白可能引发两种不利反应。有些女生可能会开玩笑地说：“V我500看看诚意”，这将你置于两难境地。运气不好时，遇到经验更丰富的女生，她可能会回复：“我也觉得你挺好的，但是发展太快了，我不想这么着急就确定一段关系”，从而婉拒你。

## 第二宗罪：没有分寸的殷勤 🛠️

明白了盲目表白的危害后，我们接下来分析第二宗罪：没有分寸的殷勤。

```

实际结果：被当作便利工具，而非吸引对象。
```

我曾经也这样做过，认为秒回是尊重，接送是体贴。但现实往往是，秒回换来的是对方轮回式的回复，接送则换来的是被当成免费司机。

## 第三宗罪：肤浅的关心 💬

过度殷勤往往伴随着低价值的付出，这自然引出了第三宗罪：肤浅的关心。

当关心被视作肤浅，关系进展不顺时，有些人会陷入第四宗罪：卑微的心态。

