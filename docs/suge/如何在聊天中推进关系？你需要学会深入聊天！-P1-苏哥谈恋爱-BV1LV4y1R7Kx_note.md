## 案例分析：低质量聊天的困境

一位93年的IT行业兄弟求助，他喜欢上同公司不同部门的一位女生，已聊天近一个月，但女生对他仍像陌生人一样，从不主动说话，每次回应也很敷衍。

以下是他们的聊天记录：
*   兄弟说：“看你朋友圈前两天去海南玩了吗？”
*   女生回：“对。”
*   兄弟说：“有两年没有去了好怀念那边的海水。”
*   女生回：“那你下次也去。”
*   兄弟回答：“主要是没有时间。”
*   女生就回：“好吧。”

这就是典型的“直线聊天”。我们换个角度，假如你是女生，看到这些消息会怎么回复？兄弟想了一下说，确实有点不知道该怎么接。这就对了，话题很浅且没有互动，最关键的是触及不到女生的核心情绪点，让对方没有任何回复的欲望，聊天自然无法继续。

上一节我们看到了低质量聊天的例子，本节中我们来看看如何改进。想跟女生聊出好感，以下两个点必不可少：

**1. 参与感**
必须让女生一起参与到话题中，并与她互动。

**2. 互动性**
尤其是情绪上的互动，而非单纯的信息交换。

兄弟回答：“哈哈那我们还挺像的。” 这里赋予双方一个共同点，将两人拉近到同一个话题中，这就是**参与感**的建立。

女生回：“哪里像？懒吗？”

兄弟本能地想回“对，我们都很懒”，但被制止了。这种回复没有情绪，容易把天聊死。此时需要一点迂回策略来拉伸对方的情绪。

于是兄弟这么说：“我是说，我们都是外卖党。没想到你居然告诉我这么大一个秘密。”

画面感瞬间就有了。**什么叫做互动？这就是互动。** 女生对兄弟的好感迅速升温，基本脱离了陌生感。

此时兄弟问是否可以直接表白？当然不行。这时女生仅仅只是有好感，还未建立亲密关系，盲目表白会让前功尽弃。所以还不行，必须再加一把火，为邀约赋予一层特殊的“意义”。

3.  在适当时机通过**赋予共同点**和**特殊意义**来拉近关系。
