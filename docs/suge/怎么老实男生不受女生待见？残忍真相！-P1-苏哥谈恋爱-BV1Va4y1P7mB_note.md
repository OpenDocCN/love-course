有一个销售公司给员工做过一个活动。他们分别准备真假两批银首饰。销售A组去卖成本9块9的假银饰，而销售B组摆地摊卖成本三四百的真银饰，两组的定价都是99元。

活动开始前，几乎所有人都认为B组会获胜。毕竟原本值几百块的真银饰，现在只卖99元，应该随便卖。然而结果恰恰相反，A组把所有假银饰都卖光了，而B组仅仅只卖了两套。

你们是否也对这样的结果感到不可思议？其实道理很简单。把几百块的银饰降价到99元卖，顾客的第一反应是：你是骗人的，或者东西不干净。没人会做亏本买卖。但是小组A，他们给每一个首饰都换上精美礼盒，同时强调这是活动限量售卖。顾客一看，这么精美的首饰和包装只卖99元，所有首饰直接被抢购一空。
