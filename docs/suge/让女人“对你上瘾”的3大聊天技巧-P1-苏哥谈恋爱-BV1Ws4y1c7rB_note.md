## 第一招：刺激想象力 🎨

**核心操作**：避免使用直白、客套的语言，转而使用带有画面感和暧昧意味的表述。

*   **普通方式**：在开车送女生回家后说“没事，正好顺路”。女生听到后仅会感到感激，不会产生心动感觉。

因此，与女生聊天时，切记生活可以直白，但涉及情感互动时不宜过于直白。
