## 核心理念：消除“追求感”

追女生最好的方式，可以总结为两句话。第一句话是：先别让自己觉得你在追求她。第二句话是：别让女生觉得你在追求她。

上一节我们介绍了核心理念，本节中我们来详细分析这两句话的含义。

### 第一点：别让自己觉得在“追求”

说实话，我追女生的时候，从来没有认为我在追她们。在我看来，我只是遇到了一个很不错的女孩子，我去跟她接触聊天而已。我没有在追求她，毕竟我们才刚刚认识，我对她也不是很了解。

她对我也不很清楚。我们不能说见一面就喜欢上对方，就去追求。而且我只是现在觉得她还不错，但我可能相处了，就会觉得她不是我的菜了。然后不理她不联系了，这都很正常。

如果你开始把你接触她的过程，看作你在了解和接触她，而不是你在追求她，那么很大程度上你已经成功了。因为你要是抱着一种“我在追求她”的想法的时候，你很容易犯前面提到的两种错误。

以下是抱着“追求”心态时容易出现的两个问题：

1.  **心态上自然地就会放弃自己**：因为你觉得自己在追求她，所以她说什么话你也接，她表现不好的时候你也不敢说，她让你去做这个事情，你就二话不说，而且还动不动就给人家表忠心，当舔狗。
2.  **得失心和控制欲自然就强了起来**：因为你觉得你在追求她，所以对方一丁点的举动，你会过分地去解读，然后心态开始患得患失。

不管女孩子一开始是怎么对你的，有没有好感，这种长期以往相处下去，只有一个结果：她会变得恃宠而骄。

如果刚开始你追求这个女生的时候，你的得失心就特别的重，那么就会像这样子的：“你今天有没有给我回消息？”、“前几天给你买的礼物你收到了吗？”、“我们去吃饭你怎么只拍美食不拍我，你是什么意思？”

其实这些话都是非常吓人的。最后你坐不住了，冲动问这个女生究竟什么意思，还能不能处。女生这个时候开始会有压力。刚开始跟你相处的还可以，但是后面总感觉你在催促她，觉得你在给她压力。她也不确定跟你在一起是因为感动还是喜欢，要不先处个朋友试一下。然后呢，你就被发了好人卡。

当你接近这个女生的时候，你一定要告诉自己：**恋爱它本身就是一个互相选择的过程**。公式可以表达为：`关系发展 = 相互了解 + 双向选择`。

和你聊天看看我们合不合适，我也没有说一定要和你发生什么。如果你的表现让我不开心，我会直接告诉你。如果说你表现的我不爽呢，我也会疏远你。只要认识到这一点，把你们相处的过程看作是你在了解她，你的心态自然就会平稳下来。

### 第二点：别让女生觉得你在“追求”

怎么叫做不要让这个女生觉得你在追求她呢？这主要是针对男生追求女生的时候，经常会出现以下两个错误。

以下是让女生感到“被追求”压力的两种典型行为：

1.  **不停地表白和一个劲地想要结果**：总是去问她一个心理答案，急于确定关系。
2.  **不停地付出和讨好对方**：想通过这种方式来展示自己的诚意，证明自己的喜欢。

要知道，大部分男生跟女生表白的时候，她们根本没有做好一个心理准备。在这样一个情况下，她们给你的一个结果，本能地就会选择拒绝你。

**你不是要的一个结果，你的任务是做好一个过程**。明白没有？这就是追女生的本质。

我们要一直在创造一个让她认识你和了解你的机会。我们聊天、约会都是让女生进一步地了解你：你长什么样子的，你的穿着打扮是什么样子的，你的气质内涵等等。

你在追求女生的时候，一半以上的精力要花在这个方面：**展示自己和提升自己的价值**上面。代码逻辑可以描述为：
```python
# 错误做法
energy = 100
energy_on_pleasing = 70  # 70%精力用于讨好
energy_on_self = 30      # 30%精力用于展示自己

# 正确做法
energy = 100
energy_on_pleasing = 30  # 30%精力用于互动
energy_on_self = 70      # 70%精力用于提升和展示自己价值
```

你必须让她看到你有价值，而不是和她强调“我是真的喜欢你，你能不能给我个答案”。你就伴随着自己，不要老想着找个机会表白，然后怎么样一决高下。

其实情场高手从来不表白。约会的时候就大胆搭个手，去游乐园就带着女生去鬼屋。只要相处的过程足够的开心，结果自然就会到位。

## 总结

本节课中我们一起学习了追求异性时应有的正确心态。核心在于**消除双方的“追求感”**，将关系定位为**平等的相互了解和选择过程**。关键在于做好过程、展示价值，而非急于索取结果或通过讨好来证明自己。记住，**自然舒适的相处，本身就是最好的推进**。
