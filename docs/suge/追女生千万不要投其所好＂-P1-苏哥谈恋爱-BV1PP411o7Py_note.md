# 💔 恋爱教程 第1课：停止讨好，建立吸引

在本节课中，我们将通过一个真实案例，学习在追求异性时为何要避免一味讨好，以及如何通过建立自信和自我框架来赢得真正的吸引力。

## 概述

很多男性在追求女性时，倾向于迎合对方的所有喜好和要求。对方让做什么就做什么，对方想要什么礼物就送什么礼物。然而，即使花费大量金钱和精力，往往也无法成功追到心仪的女生。真正擅长恋爱的男性，从不迎合女生。

上一节我们介绍了“讨好”策略的无效性，本节中我们来看看一个具体的案例，展示当男性适当保持“自私”（即维护自我框架）时，追求过程会变得多么简单。

## 案例背景与分析

一位32岁的男性学员求助，他谈过四次恋爱，但每次都不超过三个月，且均被女方提出分手。最近，他经朋友介绍认识了一位以结婚为目的交往的女生。他追了她一段时间，送了无数礼物，满足了对方许多过分要求，但地位似乎连“备胎”都不如。一次约会中，女生喝了他买的咖啡，却只顾和闺蜜视频聊天，不与他交流。

他感到困惑：**“她为什么如此肆无忌惮？难道真是我对她太好了吗？”**

学员的态度非常卑微。当我问他“你敢把她拉黑吗”时，他毫不犹豫地拒绝了。这并不意外，因为许多男性在深陷感情时，恨不得把全世界最好的都给对方，根本不敢做出任何可能“得罪”对方的行为。

然而，这恰恰成了你的软肋，也是女生为所欲为的资本。**男人只有以强大的自信和强势的态度去引领女生，才能真正让女生对你欲罢不能。**

> **“你知道男生为什么喜欢一个女生吗？”**

女生回复：“你说这个是在怪我吗？”

我们指导学员回复：
> **“你愿意这么想就这么想吧。”**

女生回：“笑死，不就是跟我闺蜜聊天吗，又不是男人。你是我什么人？”

面对女生的指责，我们指导学员这样回复：

我们让学员虚晃一招：

女生回复：“对啊，就是这样子啊，我想你太多了~”并附上一个可爱表情包。

女生看似在撒娇，这是她的高明之处，她以为你认输了，只要抛个媚眼，你就会忘记之前的不快。但我们只是虚晃一枪。我们让学员回复：“我去洗澡了。”之后无论女生再说什么，都暂时不再回复。

## 巩固框架与关系推进

当然，仅仅冷淡对待是追不到女生的。第二天晚上，我们进行简单试探：

如果女生答应，就可以通过线下互动快速推进关系；如果拒绝，就先冷处理。女生回复：“太阳从西边出来啦？”这完全不按套路出牌，目的是抢夺关系中的主动权。

我们告诉学员，只要记住之前的原则——**保持自信和自我框架**——就能立于不败之地。学员领悟后回复：

> 女生问：“什么土方子？”
> **“多喝岩浆。对了，加點蜂蜜效果更好。”**

一句无厘头的话，反过来把女生打懵了。女生回：“你……”学员继续按照指导说：

过了一会儿，女生继续发消息：“怎么了宝宝？你说话？”此时学员内心非常挣扎，很想回复。我告诉他一定要稳住，女生已经快要认输了，只有守住黎明前最后的黑暗，才能拨云见日。

## 总结与核心原则

看到学员欣喜若狂，我告诉他，这只是一个开始。接下来只要不犯错，她早晚会成为你的女朋友。果不其然，仅仅十几天后，两人就见了双方父母。

本节课中我们一起学习了追求异性的核心原则：

1.  **停止讨好**：迎合与讨好只会降低你的吸引力，让你沦为“舔狗”或“备胎”。

3.  **引领关系**：以强势、自信的态度去带领和引导关系的发展，而不是被对方牵着鼻子走。
4.  **保持耐心**：即使天赐良缘也会历经波折，改变长期形成的互动模式绝非一朝一夕。与其患得患失，不如勇敢前行。

男人只有拥有强大的自信和自我意识，才不会被女生当作舔狗和备胎。只有摆脱卑微的姿态，女生才会沉迷于你的人格魅力。
