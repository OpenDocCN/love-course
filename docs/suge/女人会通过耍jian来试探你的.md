# 女人会通过耍jian来试探你的

在本节课中，我们将要学习一个在追求阶段常见但容易被忽视的现象——“废物测试”。我们将通过一个具体案例，分析其背后的逻辑，并掌握如何通过调整思维和沟通方式，有效应对此类测试，从而建立平等、相互尊重的关系基础，顺利推进感情发展。

## 理解关系停滞的逻辑误区

很多男生都遇到过这样的问题：和女生相处得不错，也聊得来，但就是无法更进一步。

如果你也有这样的困扰，那么需要首先明白一个道理：女生为什么会愿意保持良好关系，却不愿继续发展。

事实是，大多数男生在追求女生时，存在一种逻辑误区，认为“追求就要多付出”。但在这种单方面付出的模式下，女生只会觉得你很卑微，没有价值。甚至，很多原本对你有兴趣的女生，会被你惯出坏毛病，培养出把你当作备胎的想法。

这话听起来很刺耳，但这就是现实。一旦你明白了这个逻辑，再去升温关系就会简单很多。

## 案例分析：单方面付出的困境

以下是学员遇到的一个典型案例，它清晰地展示了上述误区。

学员和女生认识大半年，期间为了追求她费尽心思。
*   女生生理期时，他可以凌晨两点起床熬红糖水，再开几十公里的车送过去。
*   女生生病住院，他愿意推掉工作去医院照顾。

女生看到男生的做法很感动，之后相处愉快，甚至会主动找男生聊天、偶尔请他吃饭。但这之后，无论男生再怎么付出，女生都不愿继续发展关系。

一次聊天中，女生主动说：“在吗？有个事挺不好意思说的。”男生立刻回应：“你跟我还有什么不好意思的，你说吧，什么事情。”女生接着说：“我周六准备搬家，想问问能不能请你帮个忙。”

## 关键转折：识别测试与改变策略

我相信大多数男生此时会想都不想就同意，甚至觉得帮女生做事能提升好感。但我告诉你，这绝对不可能。

换位思考一下，如果你身边有一个对你一呼百应、马首是瞻的女性朋友，你的第一反应是喜欢她，还是觉得这个人很没底线？答案肯定是后者。

这个问题是大多数兄弟最容易踩坑的地方：总觉得和女生关系不错，想追到她就要对她更好，让她看到心意。但你这么做，恰恰拉低了自身价值。

**核心公式：健康关系的基础 = 相互感知价值 + 相互付出**

记住，两个人走到一起的基础一定是相互感受对方的价值，互相付出才行。

## 应对策略：展示价值与设立框架

于是，我们指导学员这样回复：“周六吗？周六有个同学请我吃饭，不知道有没有时间。不过既然你都开口了，我看能不能推迟一下。”

这句话首先告诉女生，我们的世界并非只围着她转，我们也有自己的事。这既保持了态度，又强调了自我价值。其次，后半句让女生感受到我们对她的重视。至于最后帮不帮，要看她的反馈。

女生回复：“哦，之前怎么没听你说过，不会是个女同学吧？”

女生主动问是否是女同学，说明她已经开始关注男生的个人生活了。原因很简单：之前男生对她百依百顺，这次却因别人的聚会犹豫，这让她产生了一丝危机感。这正是我们想要的效果。

我们接着回复：“猜对了，看来你还是蛮懂我的，一猜就中。”

女生回：“呵，重色轻友的好家伙，你不想帮，那我找别人去了。”

很多兄弟看到这句话就慌了。但请冷静想想：女生真想找别人，还会故意跟你说一声吗？而且前面加了“重色轻友”，其实是在刺激我们。如果这时我们破功了，就又会被她拿捏，追求难度会重新升级。

退一步讲，就算她真的去找别人了，我们就要去讨好她吗？那样不还是被当作备胎吗？

**核心原则：对女生好可以，但前提是等价交换。我们可以付出，但她也得做出让步。这样我们的付出才有回报，我们的重要性和个人价值在她眼里才会得到重视。**

于是我让学员说：“谁重色轻友？再说我也没有说不想帮你。”

女生秒回：“只不过什么？只不过你那个女同学太漂亮了是吧？”

看，女生的醋意出来了。这说明我们的重要性和价值在她心里迅速提升了。

接着我们可以说：“只不过搬家是个大工程，我可不想当免费的劳动力，你说呢？”

这里进一步强调我们的价值，让女生明白：想让我们对她好，她也得对我们好才行。这种相互尊重的相处模式，也会让女生觉得更自然，更容易产生好感。

女生回答：“好吧，大少爷，到时候结束了，我请你吃顿大餐可以吧？”

女生看似不情不愿，实际上是在傲娇。没关系，我们再加把劲。

“你要是觉得不好的话，千万别勉强自己。我只是觉得不管什么时候，两个人相处都应该相互理解，只有这样关系才会长久，对吧？”

这里我们以退为进，在拿回主动权的同时，保持了自我态度。后面一句话向女生展示了我们的感情观，让她知道我们是一个有责任心、懂得体谅的人。

女生很快回答：“你误会了，我不是不乐意的意思。总之，你先选地方吧，要是有时间的话，到时候再一起看个电影。”

## 思维转变带来的结果

这就是思维的重要性。之前女生还在吊着我们，而现在跟我们聊天都开始紧张，甚至主动约我们看电影。这代表追求她已经有了起码90%的胜算。

剩下的10%，要看后续表现。于是，后面我又教了学员一些约会技巧（主要以思维和心态为主，因为技巧方法再牛，终需好思维来支撑）。结果学员很争气，两人在这次约会后飞速发展，不到三天就正式确定了关系。

## 课程总结

本节课中，我们一起学习了“废物测试”的识别与应对。
