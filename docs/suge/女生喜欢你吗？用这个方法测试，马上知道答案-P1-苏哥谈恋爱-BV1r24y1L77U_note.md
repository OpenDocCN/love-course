## 1. 分开后的主动联系 📱

见面分开后，对方通常有两种选择：不主动联系，或主动联系你。

**核心逻辑**：对你有好感的女生，通常会主动发消息。消息回复越快，往往代表兴趣越大。如果对方没有主动发消息，你可以先发一条试探。若她秒回并能与你热聊至互道晚安，则表明好感度较高。

> **公式**：主动联系频率 ∝ 好感程度

## 2. 对你的事情记忆深刻 🧠

上一节我们提到了联系频率，本节中我们来看看对方是否关注你的细节。

一个对你有好感的女生，会记住与你相关的各种细节。例如，你白天随口说的一句话，她可能还记得；你很久以前发的朋友圈照片，她也能拿出来与你讨论。这说明她在用心关注你。

以下是这类表现的具体例子：
*   你提过的小事，她事后仍记得。
*   她提及你过去发布的社交动态。
*   她询问你曾提到的爱好或计划。

只要你不犯严重错误，与这样的女生发展关系会顺利很多。

## 3. 期待下一次见面 📅

除了线上互动，线下见面的意愿也是重要指标。

**核心逻辑**：愿意答应或主动提议第二次约会，是存在好感的明确信号。
