## 理性聊天的陷阱

上一节我们提到了聊天中的常见困境，本节中我们来看看导致这种困境的核心原因：**理性聊天**。

首先需要明确的是，与女生聊天时，切忌采用纯粹的理性方式。为什么？

因为理性聊天通常带有强烈的目的性，例如解决问题或询问信息。这种聊天方式不仅枯燥乏味、缺乏舒适感，而且无法提供情绪价值。女生的感受会是越聊越无趣，甚至产生反感，从而导致对话无法继续。

可以说，**至少90%的尬聊都源于此**。但绝大多数男生并未意识到这一点。

### 什么是理性聊天？

以下是理性聊天的一个典型例子，请注意其对话模式：

```plaintext
男生：我下班了，你在干嘛呢？
（女生很久未回复）
男生：人呢？
女生：我在看电视。
男生：看的是什么电视啊？
女生：你问这个干嘛？
男生：没有啊，就是问一问。
```

这种**一问一答、目的性强、缺乏情绪互动**的对话，就是理性聊天。它无法让女生产生继续聊下去的欲望。

这句话不再是提问，避免了侵略性，同时成功激发了女生的好奇心（“他怎么了？”）。只要双方没有深仇大恨，这种说法的回复率极高。

**2. 幽默回应，拉升氛围**
女生果然好奇地回复：“怎么了？你不用上班了吗？”
男生回答：“我正在上班。”
女生更疑惑了：“不会吧？什么工作？”

此时，很多男生会直接回答职业来展示价值，但这会立刻满足女生的好奇心，导致话题结束，又回到一问一答的模式。

正确的做法是采用**调侃幽默**的方式：

这种开玩笑的回复瞬间将聊天氛围变得轻松活泼。女生会觉得有趣，情绪也随之上升。她回复：“哈哈哈哈这是什么鬼。”

男生可以调侃道：“哈什么哈呀，隔着屏幕都能听到你的笑声了。”

这里直接开启了暧昧，并巧妙引导女生往恋爱方向联想。女生回复：“我之前怎么没发现你还挺会聊天的呀。” 可见其情绪已经非常高涨。

这样可以提升女生后续主动聊天的可能性。女生当场给出了承诺：“那以后尽量多聊两句哦。”
