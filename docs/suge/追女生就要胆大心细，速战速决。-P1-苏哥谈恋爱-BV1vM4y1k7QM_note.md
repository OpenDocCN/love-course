## 核心心态：保持强者姿态

上一节我们介绍了课程主题，本节中我们来看看追求女生时必备的核心心态。对于女生，必须保持强者心态。无论对方多么优秀，你都要将她视为一个普通的女生。你越是把女生当女神，她越可能把你当作**舔狗**。心态是后续所有技巧的基础。

## 案例背景与分析

以下是案例背景介绍。

一位男生在朋友生日会上遇到一位理想型女生，通过朋友获得微信后开始聊天。但女生反应冷淡，回复礼貌且不主动，男生感到兴趣度不高，不知如何推进。

## 策略实施：反其道而行之

上一节我们了解了男生的困境，本节中我们来看看如何用“出其不意”的策略破局。当女生主动发来自拍照并说“在减肥”时，其潜在目的是寻求夸奖。常规的夸赞会使你落入她的预期框架。

正确的做法是进行**反向操作**。我指导男生回复：
```
都这么瘦了还减？到时候成了瘦皮猴就没人喜欢了。
```
此举制造了情绪落差，打破了女生的预期。

## 建立独特性与展示高价值

以下是建立自身独特性的对话要点。

女生追问：“男生不都喜欢苗条的女生吗？” 我们回复：
```
那我可能是那小部分吧。
```
暗示我们与她身边的普通男生不同。当女生问“你喜欢什么样的？”时，避免直接说“我就喜欢你这样的”这种跪舔话术。我们回复：
```
其实我觉得喜欢一个人是没有标准的，最重要的是遇到了聊得来、互相理解的。
```
此回复展示了成熟的**恋爱观**，传递了高价值。

## 引导情绪与暴露需求感

上一节我们展示了高价值，本节中我们来看看如何引导女生的情绪。女生说“感觉你和其他男的确实有点不一样”，并问“你就没有遇到过这样的女生吗？”。

此时**绝不能暴露需求感**。我们回复：
```

```
我们句句暗示女生，但句句不提她，促使她主动确认。果然，女生问：“你说的该不会是我吧？” 至此，她已进入我们的对话框架。

## 推拉技巧与最终主导

以下是运用推拉技巧调动情绪的步骤。

我们需要进一步推拉，调动她的情绪。我们回复：
