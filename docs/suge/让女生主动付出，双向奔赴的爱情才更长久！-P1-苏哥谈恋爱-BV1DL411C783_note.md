**核心公式：好感 = 价值交换**

如果你想让一个女生对你产生好感，就必须具备“价值交换”的思维。

然而，你都不知道是什么忙，一口答应显得自己没有原则，无形中降低了自己的价值感。

高情商的男生会这么说。在这里，我让兄弟这样回复：“帮忙可以啊，不过我可是要收报酬的哦。”
