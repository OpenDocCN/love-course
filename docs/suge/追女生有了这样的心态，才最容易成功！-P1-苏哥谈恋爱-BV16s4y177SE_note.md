# 恋爱心态课 01：掌握“随便”心态，轻松赢得芳心 💖

在本节课中，我们将学习一种在追求心仪对象时至关重要的核心心态。许多人在追求过程中受挫，往往并非因为条件不足，而是心态出现了偏差。我们将深入剖析这种心态，并提供具体方法，帮助你摆脱患得患失，大幅提升成功的可能性。

## 概述：心态决定成败

很多人追求不到喜欢的女生，根本原因在于心态不对。错误的心态会导致原本可能成功的关系，因为自身的焦虑、犹豫而失败。这是最不应该出现，也是最可惜的情况，甚至可能成为长久的遗憾。

本节视频将揭示追求女生时应保持的正确心态。理解并掌握这种心态，不仅能让你避免患得患失，还能显著提高成功的概率。

## 一个典型案例：心态失衡的全过程

以下是许多兄弟都会遇到的典型困境，它清晰地展示了错误心态如何一步步破坏关系。

一位兄弟分享了他的经历：
*   他在逛街时遇到一位长相身材都不错的女生，犹豫很久才鼓起勇气搭讪。
*   一开始女生有些防备，表示这样不太好。他心想算了，但女生最终还是加了他的微信。
*   回去后他发了消息，对方半小时没回复，他便开始焦虑，怀疑对方看不上自己或自己聊天太无趣。
*   后来女生回复说刚才在洗澡，并解释了原因，他感到非常开心。
*   大约聊了一周后，他尝试邀约对方，但被以有事为由拒绝。他因此认定女生根本不想见面，只是无聊时拿他消遣。
*   他越想越心烦，便不再主动联系，而对方也从未主动找他。但他心里始终放不下，不想错过，于是陷入“是否该继续联系”以及“该聊什么”的纠结中。

在我看来，很多兄弟都和这位兄弟一样，最终输在了心态上。

## 核心心态：随便与无所谓

接下来我们要探讨的核心心态，可能初听有些抽象，但请点赞收藏，慢慢体会。总有一天你会发现，掌握这种心态不仅对谈恋爱，对你的学习、工作和生活都会有巨大帮助。

这个心态就是：**随便，无所谓**。

### 心态的应用场景解析

上一节我们看到了心态失衡的案例，本节中我们来看看“随便”心态如何具体应用在不同场景中。

**1. 初始接触阶段**
比如案例中的兄弟，在犹豫是否要搭讪时，就应该想：这没什么大不了的，我们只是随便搭讪一下，交个朋友，无所谓成不成功。能发展最好，不加又能怎样？我们并不会损失什么。你搭讪她、喜欢她，无非是她的长相身材符合你的审美标准。但满大街都是漂亮女孩，随便上街都可能遇到喜欢的，甚至更好的，再去搭讪就好了。

**2. 互动反馈阶段**
后来这位兄弟因为女生没及时回消息就开始胡思乱想，女生一回就欣喜若狂，女生不回就患得患失。这种心态本身就有问题。即使女生想认真发展，你也会因为这种心态被牵着鼻子走，最终可能沦为舔狗、备胎或工具人。

我们换个思维：**随便你回不回。你回了，我就聊；你不回，我就忙自己的事，或者找别人聊天。无所谓。**
一旦你用这种方式和女生接触，你会发现，女生反而可能比你预计的更主动、更配合。

### 心态转变的实战效果

如果你不信，我们来看这段聊天记录。我让那位兄弟给女生发消息：“出来看电影，今天还挺暖和的。”
女生回复：“我还不确定下午有没有空，再说吧。”
很多人看到这里就觉得失败了，放弃了。

但我们的回复非常“无所谓”：“行啊，再说呗。你要有事我就约其他朋友。”
这时，女生的态度变了：“哈哈，其实应该可以吧。你想看哪一部呢？”

看到了吗兄弟们？这真的不是玄学，**心态决定成败**。

那天下午，兄弟和女生看完电影后给我发消息：“宇哥，我们看完了，现在准备去吃饭。我好想牵她的手，但是她的手一直揣在兜里，我不敢。万一她觉得太快，我就凉了。”
我告诉他：“你直接牵。女生既然愿意出来，就不要怂。直接把手放到她口袋里，这种情况一般不会拒绝。就算拒绝了又怎样？你就是丢个脸而已，又不会丢条命。追女生就是要胆大心细脸皮厚。”

没过几分钟，兄弟的消息来了：“宇哥，牵手了！她愣了一下，但还是把手给我了。”那天他们牵了很久的手，没过几天就在一起了。

## “随便”心态的深层逻辑

所以说兄弟们，“随便”是一种心态，也是一种人生态度。大家都听过“认真你就输了”这句话，却不知其所以然。其实把这句话放在你追女生的过程中，认真思考一下就懂了。

很多兄弟和女生聊天时太过“认真”，绞尽脑汁、思来想去，这样做出的决定往往会导致失败。当然，这也不代表你们和女生聊天可以张口就来、胡说八道。

“随便”是一种态度，而且是一种非常吸引女生的态度。它并不代表我们不认真聊天。**认真本身没有任何问题**，就像你认真学习和工作一样，这很好。但是，和女生聊天不是做作业或完成工作，需要按部就班地走程序。如果那样，还追求什么呢？这已经违背了兄弟们的初心。

## 重新定义“追求”：享受自然过程

其实，压根就没有“追女生”这个概念。从相识、相知到相爱，都是一个自然而然的过程。我们的初衷是享受这个过程，和喜欢的人相处，生活因此变得快乐。道理就是这么简单。

只有当你用“随便、无所谓”的态度去和女生相处，才能让自己处于最舒服、最自然的状态下与对方接触。这就是我们所说的“主场优势”。就像你和朋友聊天时无所不谈、非常自然快乐、毫不紧张；或者在你最擅长的领域做事，你会焦虑吗？当然不会。你不会感到纠结，也不会流露出过多的需求感，做事干净利索，像吃饭喝水一样简单。

## 将心态融入生活：由内而外的改变

兄弟们，只有当我们在生活中真正具备了这种“随便”的态度，才能在追求女生的过程中自然而然地体现出来。自己硬装“随便”是没用的，会破绽百出。

以下是培养这种生活态度的具体建议：
*   **果断行动**：遇到很想买的球鞋，直接拿下；周末想去山上旅游，说走就走。
*   **干净利落**：做事不拖泥带水，不留遗憾，不要想那么多。

当你能做到这些，再回到追女生这件事上：
*   想要微信，直接要。
*   想要聊天，直接聊。
*   想要见面，直接约。
*   聊得好了，就直接确定关系。

可能有的兄弟会说：“你这不就是莽夫吗？那不是很容易失败吗？”
其实不是的。这个态度背后的逻辑是：**我们对自己所做的事情已有清晰理解，内心非常明确这么做稳赚不赔，就应该这么做。** 慢慢地，这会练就成为我们的本能反应。到了这个阶段，追求任何女生你都不会再胆怯了。
