# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

变量可以理解为一个存储信息的“盒子”。在Python中，你可以给这个盒子起一个名字，然后把数据放进去。之后，你就可以通过这个名字来使用盒子里的数据。

创建一个变量非常简单，使用等号 `=` 即可。等号左边是变量名，右边是你想存储的值。

**公式：** `变量名 = 值`

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存储在其中。

```python
message = "你好，世界！"
print(message)
```

运行这段代码，控制台会输出“你好，世界！”。这里，`print()` 是一个函数，它的作用是打印出变量 `message` 中存储的内容。

上一节我们介绍了变量的基本概念，本节中我们来看看Python中常见的数据类型。

## 基本数据类型 🔢

数据类型决定了变量可以存储什么样的数据，以及可以对数据进行哪些操作。Python有几种内置的基本数据类型。

以下是几种最常用的数据类型：

*   **整数 (int):** 存储没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数 (float):** 存储带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str):** 存储文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`, `“学习编程”`。
*   **布尔值 (bool):** 只有两个值：`True`（真）和 `False`（假），常用于逻辑判断。

你可以使用 `type()` 函数来查看任何变量或值的数据类型。

```python
age = 25          # 整数
price = 19.99     # 浮点数
name = "苏哥"     # 字符串
is_student = True # 布尔值

print(type(age))      # 输出：<class 'int'>
print(type(price))    # 输出：<class 'float'>
print(type(name))     # 输出：<class 'str'>
print(type(is_student)) # 输出：<class 'bool'>
```

了解了不同类型后，我们来看看如何让它们之间进行互动，这就是类型转换。

## 类型转换 🔄

有时，我们需要将一种数据类型转换为另一种。例如，将数字转换成文本以便拼接，或者将文本形式的数字转换成真正的数字进行计算。

以下是常见的类型转换函数：

*   `int()`: 将数据转换为整数。
*   `float()`: 将数据转换为浮点数。
*   `str()`: 将数据转换为字符串。

```python
# 将字符串数字转换为整数进行计算
num_str = "100"
num_int = int(num_str)
result = num_int + 50  # 现在可以进行数学运算了
print(result)  # 输出：150

# 将数字转换为字符串进行拼接
score = 95
report = "我的分数是：" + str(score)
print(report)  # 输出：我的分数是：95
```

请注意，并非所有转换都能成功。例如，你不能将 `“你好”` 这样的纯文本字符串转换为整数。
