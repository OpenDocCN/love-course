## 识别“纠缠”行为

上一节我们提到了要避免死缠烂打，但许多人不自知地陷入了“纠缠”。那么，什么是“纠缠”呢？以下是一位学员的案例，可以帮你找到答案。

这位学员说：“宇哥你好，我喜欢一个女生半年了，最近也在尝试追她，但一直没效果，你能帮我看看是怎么回事吗？” 随后他发来了聊天记录。分析后发现存在以下几个典型问题，你可以自查是否也有类似情况：

以下是“纠缠”行为的四种表现：
1.  **聊天从不主动结束**：只要女生回复，这位学员必然会回复，从不主动终止对话。
2.  **话题不断切换**：当女生很久没回复时，他会另起一个新话题继续聊。
3.  **过度关心**：女生出去玩时，他会千叮咛万嘱咐。
4.  **单向开启话题**：总是他在找话题，女生从不主动开启对话。

以上四个问题不分先后。如果你没有这些问题，说明你的大方向基本正确。这四个问题统称为 **“纠缠”** ，其本质是 **你对女生的依赖程度 > 她对你的依赖程度**。

可能有兄弟会想：“我喜欢她，追她，纠缠她，不一直是这样做的吗？” 但你一直这么做，就代表是对的吗？为什么你追女生很辛苦？因为你从一开始就没有建立正确的认知。

## 打破错误认知，建立新框架

现在你明白了这是“纠缠”，但可能还不想改变，原因有二：一是没有新方向，二是这已成为你的习惯。案例中的学员也是如此。在语音沟通了很久之后，才打破了他错误的认知。但只要度过这最艰难的一关，后面的事情就简单了。

后来，我让他给女生发这样一条消息：

好好想想，女生接着回复：“是的，你怎么才发现？” 有兄弟会觉得这是女生在示好，其实不是。这叫做 **以退为进**。此时一定要避免暴露你的需求感，否则只会让女生对你瞬间失去兴趣。同时，油腻的行为也是一种纠缠。

所以这里可以回复：
