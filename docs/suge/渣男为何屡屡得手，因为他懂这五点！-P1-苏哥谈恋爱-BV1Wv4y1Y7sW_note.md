## 第一点：控制心理预期 📊

上一节我们介绍了课程概述，本节中我们来看看如何控制对方对你的心理预期。心理预期具有不可逆性。一旦对方确立了对你的心理预期，你就需要做得更好，才能获得良好反馈。

**公式：实际表现 > 心理预期 = 正向反馈**

以下是具体说明：

*   **举例说明**：约会前，如果你不断强调自己非常帅气有型，但见面后女生觉得你不过如此，她会感到失望甚至被欺骗。反之，如果你提前降低预期，见面后实际形象超出她的预期，她则可能更积极地夸赞你。
*   **对“好”的预期同理**：如果你一直对她非常好，某天稍有松懈，她就会觉得你变了。

**如何解决心理预期问题？关键在于“正向反馈”和“推拉”技巧。**

*   **正向反馈**：在她对你好时，你才相应地对她更好。当她得到反馈并开始加大付出时，你也可以逐步提升。
*   **引导习惯**：通过这个操作，你可以慢慢引导对方形成主动回馈的习惯。虽然过程中你不断提升在她心中的预期，但对方也养成了反馈的习惯。
*   **保持情绪波动**：这样，她在相处中情绪会保持波动状态，也更积极主动。例如，连续每晚说晚安后突然停止，她会开始思考并惦记你。

**核心**：一旦学会控制她对你的心理预期，你将能掌控爱情的主动性。

最后，在塑造好人设的基础上，我们需要明白吸引力不是一成不变的。吸引力是波动的，因人而异。每个女生对吸引自己的男生都有不同的定义。

以下是核心方法与举例：

*   **错误示范**：就像你拿着吸引A女生的优点去吸引B女生，对方自然不会喜欢上你。

*   **正确做法**：追求一个女生时，你必须先明白什么样的男人能吸引她。你得先让对方察觉到你身上有和她相同的特质或满足她核心诉求的优点，她才会开始欣赏你、接受你、肯定你。
