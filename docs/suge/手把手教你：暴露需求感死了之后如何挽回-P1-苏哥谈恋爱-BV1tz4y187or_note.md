# 手把手教你谈恋爱 第1课：需求感暴露后的挽回策略 💔

在本节课中，我们将学习一个在追求过程中常见的问题：过早或过度地向女生表达好感，导致对方因感到压力而疏远，即“暴露需求感”。我们将通过一个具体案例，拆解问题所在，并学习一个名为“以攻为守”的高级沟通策略来扭转局面。

## 问题概述：需求感暴露的后果

很多男生会遇到这种情况：经常向女生示好或者说了一些过于直白的话，然后发现女生突然就不想聊天了。这就是常说的“暴露需求感”，因为女生觉察到了你的目的。但双方关系又未发展到可以建立亲密关系的程度，女生唯一的选择就是逃避。

## 案例引入：一个典型的错误示范

下面我们来看一个求助案例。这位兄弟说：“苏哥，我是你的铁粉，江湖救急。简单说一下情况：我跟他认识一个多月，因为工作上经常有联系，很快我就喜欢上她了。可是我没什么追女生的经验，就经常找她聊天、送礼物、约吃饭。但今天聊天后，她突然不理我了。我反应过来，肯定是我刚才的话暴露需求把她吓跑了。”

上一节我们介绍了问题背景，本节中我们来看看这位兄弟与女生的具体聊天记录，分析错误所在。

以下是导致关系急转直下的关键对话片段：
*   兄弟先约女生吃火锅，女生以“明天要面试”为由拒绝。
*   兄弟说：“怕什么，人生在世就要活得任性。”

*   兄弟说：“上班是男人的本分，任性是女人的权利，好好享受你的任性吧。”

至此，聊天还算正常。但接下来兄弟的两句话，让情况瞬间逆转：
*   兄弟说：“所以你也可以更加任性一点，比如说做我的女神。”
*   女生回：“看缘分吧。”
*   兄弟又说：“不做女神那就做孩儿他妈。”
*   此后，女生再也没有回消息。

兄弟的判断没错，正是最后这两句话严重暴露了需求。其实按照他的说法，之前就已多少有所暴露。但如果没有这“压死骆驼的最后两根稻草”，两人顺其自然发展，问题本不大。

## 核心策略：以攻为守

那么，事情已经变成这样，该如何挽回女生的好感呢？下面大家赶紧拿出小本本。今天，我就教你们一个在其他地方根本学不到的招式：**以攻为守**。

掌握这招，不仅不用怕暴露需求，运用得当的话，甚至可以化危机为转机，一举追到女生。其核心逻辑是：在对方以为你不敢承认时，你反而一本正经地说出“真话”，对方会本能地朝着相反方向去想，从而产生疑惑和动摇。

### 第一步：正面承认，制造反转

我指导兄弟给女生发了这样一句话：
```
嗯，是不是以为我在给你表白？
```
这句话发出去，只要女生看到，不出10分钟必定会回复。果不其然，女生很快回了：“我哪知道。”

兄弟接着回：
```
哈哈哈哈，我的确是想跟你表白。
```
我知道肯定有兄弟看不懂：这不是头铁吗？为什么非要继续撞墙？原因很简单，在这种情况里，不管是解释还是逃避，都没有任何意义，反而会让女生看透你的底细。在沟通学上，有一门技巧叫做“正话正说”。

当对方以为你不敢承认的时候，你却一本正经、煞有介事地说出真话，对方反而会惊疑不定，本能地朝着相反的方向去想。没错，当你说你就是要跟她表白时，女生却会觉得你是在开玩笑。

你们看，这里女生回了：“切。”网聊要适可而止。很好，女生这一系列反应，完全是跟着我们的思路在走。

那么危机解除了吗？没这么简单。一旦女生冷静下来品出味道，还是有很大风险。接下来，我们就要迂回包抄，随后引蛇出洞，以实击虚，最终拿下“城池”。

我和兄弟等了10分钟左右，才说：
```

```
大家想一想，女生看到这句话，又会是什么想法？可能有兄弟会觉得：这样会不会显得我们立场不坚定？不，非但不会，反而会让这个女生进一步地迷糊，甚至是动摇，弄不清你到底是认真的还是开玩笑的。

那么带着这种疑惑，女生就会忍不住地把你们的关系，往恋爱的方向去进行联想。果然，女生回了：“呃，不会吧。”这里，女生的思维是有点混乱的。

那肯定有兄弟要说：我知道了，下面我们要趁热打铁加强攻势。但是兄弟们，你们忘记我刚才所说的吗？要干嘛呢？要引蛇出洞啊。现在女生其实依旧占据着优势位置，盲目进攻要吃大亏。

```
女生回了三个问号，然后沉默了半分钟才说：“那就好。”莫名其妙的三个字让这个兄弟再次紧张了起来，以为是自己回复太冷淡，女生不高兴了。

于是，我让兄弟直接拆穿女生的小心思：

```
女生再也忍不住了，给兄弟发了长长的一段话。话很长，但是核心就三个字：“我接受。”

到这里，案例差不多也就结束了。后面兄弟在我的指导下，继续提升自己。一个月过去，跟女生的关系还在上升。
