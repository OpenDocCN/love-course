## 案例背景与分析

那么问题来了：在这种情况下，男生还有机会吗？有的兄弟可能说肯定没机会了，有的说还有20%，有的说50%。案例中的兄弟也很想知道他追到女生的可能性有多大。

我的答案是：**90%**。

我之所以这样判断，是因为仔细想想就能明白：如果一个女生对你真的毫无兴趣，她不可能花时间写这么长一篇小作文。她甚至可能懒得拒绝你，直接不理你。案例中的女生写了300多字，这很可能花费了她半小时甚至一小时。这种情况很容易理解：女生对男生的好感是存在的，只是这份好感尚未达到“喜欢”的程度。此时，男生可能给了她较大压力，因此女生在理性思考后选择了拒绝。但即使她拒绝了，那份好感依然真实存在。

此时，一定要避开理性思维，我们要通过**感性思维**去和女生沟通。

所以可以问女生这样一个问题：“那我还可以喜欢你吗？”

然后，我让兄弟这么回复：“你说可以我就继续喜欢你，你说不可以我就不喜欢你了。我听你的。”

这里也要告诉所有人：千万不要在女生拒绝你的时候跟她说“不管怎么样我都会一直喜欢你的，我会一直等你”。你说这种话，只会让她离你越来越远。想象一下，当一个女生对你说同样的话，你会怎么回？所以，刚才的问题对女生来说很难回答，因为她对男生也有好感，因此会非常纠结。

过了几分钟，女生回了这么一句话：“我是不会拒绝任何人喜欢我的，懂吧？”很明显，女生已经有了一点小情绪。

那么这个时候，你可以尝试把女生“激怒”一下。我们回：“原来你是这样的人啊。”

大家感受到了吗？到这里，基本上就可以确定女生对男生的感情。只要后面他不犯错，这位兄弟就可以跟这个女生在一起。

聊天快结束了，但此时女生还是有点理性的。所以我们要用一句话，让女生的理性思维彻底转为感性思维。

**核心技巧公式**：`一句充满想象空间的感性话语 = 打破理性僵局 + 植入暧昧可能`

例如：“那我今晚可能会梦见你。”

大家想一想，女生看到这句话之后，会是什么心情？我告诉你，她会很不“平静”。因为此前女生已经把自己的内心话全都表达出来了，而且她也承认对兄弟有好感。所以这个时候，我们可以去适当调侃一下。

## 技术总结与核心心法

回过来，我们仔细看一下这个案例：从一开始女生理性状态下的拒绝，到最后轻松状态下的聊天，并且还带了一点暧昧的气氛，这前前后后，仅仅用了不到几个小时。

3.  **如何调动情绪**：比如这个案例中，“我可能会梦见你”这样一句调侃式的话语，是非常讲究的。当然，你直接冲过去跟她说“我可能会梦见你”，效果也不一定会好。**关键**在于前期通过“忽略-提问-引导-表达感受”的步骤，铺垫好了情绪基础，让这句感性的话成为打破僵局的临门一脚，而不是突兀的骚扰。
