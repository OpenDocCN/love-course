# 恋爱技巧课 P1：营造画面感 🎨

## 什么是“营造画面感”？ 🤔

上一节我们提到了“营造画面感”这个概念，本节中我们来详细看看它的定义。

简单来说，营造画面感就是在与女性聊天过程中，学会描述具体的场景和画面。因为画面给人的感觉更加真实，也更有代入感，这能快速拉近你们之间的关系。

理解了画面感的核心后，我们来看看如何具体操作。以下是植入情侣共同生活画面感的一个例子。
