# 七分学堂：Python基础入门 第1课

在本节课中，我们将要学习Python编程语言的基础概念，包括变量、数据类型以及如何进行简单的数学运算。这些是构建任何Python程序的基石。

## 🧱 变量与赋值

上一节我们介绍了课程目标，本节中我们来看看什么是变量。变量是用于存储信息的容器。你可以把它想象成一个贴有标签的盒子，标签是变量名，盒子里装的是数据。

在Python中，使用等号 `=` 为变量赋值。等号左边是变量名，右边是要存储的值。

**代码示例：**
```python
name = “七分学堂”
age = 3
```

## 🔢 基本数据类型

理解了如何存储数据后，我们来看看Python中几种最基本的数据类型。数据类型决定了数据可以进行的操作。

以下是三种最常用的基本数据类型：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：包含小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `“Hello”`, `‘Python’`。

## ➕ 基本数学运算

掌握了数据的类型，我们就可以对它们进行计算了。Python支持所有常见的数学运算。

以下是基本的数学运算符：

*   **加法**：`+`
*   **减法**：`-`
*   **乘法**：`*`
*   **除法**：`/` （结果总是浮点数）
*   **取整除法**：`//` （丢弃小数部分，只保留整数）
*   **取余**：`%` （返回除法运算的余数）
*   **乘方**：`**`

**代码示例：**
```python
result = 10 + 5 * 2 # 先乘除后加减，result的值为20
quotient = 17 // 3 # quotient的值为5
remainder = 17 % 3 # remainder的值为2
```

## 🧪 动手试一试

现在，让我们将以上概念结合起来，完成一个简单的实践。我们将创建一个程序，计算并显示两个数字的和与乘积。

**任务步骤：**
1.  创建两个变量，分别存储两个数字。
2.  创建第三个变量，存储这两个数字的和。
3.  创建第四个变量，存储这两个数字的乘积。
4.  使用 `print()` 函数将结果输出到屏幕。

**参考代码：**
```python
# 定义两个数字
num1 = 12
num2 = 8

# 计算和与乘积
sum_result = num1 + num2
product_result = num1 * num2

# 打印结果
print(“两数之和为：”, sum_result)
print(“两数之积为：”, product_result)
```

运行这段代码，你将在屏幕上看到计算出的和与积。
