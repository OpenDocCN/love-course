# 七分学堂《恋爱核武器》：《恋爱核武器》模块二、个人ip建设篇：网红修图指南_note

在本节课中，我们将学习如何通过后期修图，将普通的照片优化为高质量的展示面素材。修图是个人形象建设的关键一步，能让你的社交形象更具吸引力和质感。

## 为什么要修图？

上一节我们介绍了拍照的构图技巧，本节中我们来看看为什么拍完照还需要修图。

拍照分为前期拍摄和后期处理。过去使用胶卷相机，拍完照需要冲洗才能得到最终照片。同样，在现代数码摄影中，原相机拍出的照片是“原始素材”，它可能过于真实地暴露了我们面部的瑕疵、环境光线的不足或构图的微小缺陷。

摄影师拍完照后，通常会检查照片的工整度、构图、对焦、锐度、明暗、色彩饱和度、高光和暗部细节。即使我们前期姿势摆得再好，如今高分辨率的手机镜头也会放大我们平时不注意的面部细节。如果将这些未加修饰的原图直接用作社交展示面，效果可能不理想。

因此，一张完美的照片需要经过前期拍摄（人物与构图）和后期修图（处理与滤镜）两个步骤。

## 修图的核心方向

修图主要围绕以下几个核心方向进行，目的是让照片更接近我们眼中真实、自然且美好的样子。

以下是修图需要处理的几个关键点：
1.  **还原真实与自然**：修正原相机过度写实带来的缺点，让照片更接近镜子中或我们眼中看到的自己。
2.  **优化视觉美感**：通过加工，让照片变得更精美、更有质感。
3.  **修正环境缺陷**：改善因天气、光线导致的照片昏暗、色彩平淡等问题。

## 修图前后对比案例

在具体学习操作前，我们先通过几组对比图，直观感受修图带来的巨大变化。

**案例一：室内人像**
*   **原图问题**：光线昏暗，人物面部不立体，服装有褶皱，背景有杂物（如灭火器），人物显得年龄偏大。
*   **修图后**：亮度提升，人物面部经过修饰，背景杂物被移除，人物显得更精神、年轻，整体气质提升。

**案例二：室外环境人像**
*   **原图问题**：阴天拍摄，天空灰暗，建筑和天空部分噪点多，人物在环境中不突出。
*   **修图后**：通过软件替换或优化天空，整体画面变亮，人物与环境融合更好，照片更具氛围感。

## 常用修图软件及核心功能

接下来，我们将结合几款常用软件，讲解具体的修图步骤。我们主要使用 **美图秀秀**（简单易上手）和 **FaceTune**（功能更专业）。

### 美图秀秀基础处理

美图秀秀非常适合新手进行基础的人像处理。

以下是使用美图秀秀处理人像的核心步骤：
1.  **面部处理 (`人像美容`)**：
    *   **磨皮/祛痘**：使用`祛痘`或`祛皱`功能，处理脸上的痘痘、痘印或皱纹。可以手动点涂，也可适度使用自动功能。
    *   **面部重塑**：使用`面部重塑`功能，微调脸型。例如，如果觉得脸宽，可以适当调整`脸宽`参数，让脸型更流畅。
    *   **修容笔**：用于弥补细节。例如眉毛较淡，可以用`修容笔`取色后轻轻涂抹，让眉毛更立体。
2.  **整体色调调整 (`图片美化` -> `编辑`)**：
    *   苹果手机用户可直接使用相册自带的`编辑`功能。
    *   **自动调整**：点击`自动`，软件会智能优化照片明暗和色彩。
    *   **手动调整**：
        *   `曝光`/`亮度`：提升整体画面亮度。
        *   `高光`：调整画面最亮部分，让人物更突出。
        *   `色温`：降低色温（向蓝色调整）会让照片显得更“酷”；提高色温（向黄色调整）会让照片显得更“暖”。这是营造照片风格的关键。

### FaceTune进阶技巧

FaceTune在细节处理和局部调整上更加强大。

以下是FaceTune的一些进阶功能：
*   **色调工具 (`色调`)**：用于统一或修改局部颜色。
    *   操作：点击`色调` -> 选择`取色器` -> 点击照片中你想取色的区域 -> 再点击`色调`，然后用手指涂抹需要改变颜色的区域。例如，可以用于补全发际线或改变局部环境色。
*   **修补工具 (`修补`)**：类似于仿制图章，可以复制画面某一部分覆盖到另一部分，用于消除杂物。
*   **细节处理 (`修图`)**：包含`平滑`（磨皮）、`细节`（锐化）等功能，处理更精细。

### 辅助软件推荐

除了上述软件，还有一些专门用途的软件非常好用：
*   **Quickshot**：专门用于一键替换或美化天空，效果丰富。
*   **VSCO**：拥有大量高质量滤镜和精细的调色工具，是提升照片质感的利器。
*   **美颜相机**：如果完全不会修图，可直接使用美颜相机拍摄，但其效果有时会显得不自然。

## 完整修图流程演示

现在，我们以一个较复杂的案例，串联起修图的完整流程。

1.  **基础人像处理 (使用美图秀秀)**：
    *   导入原图，进入`人像美容`。
    *   使用`祛痘`、`祛皱`处理面部瑕疵。
    *   使用`面部重塑`微调脸型、眼睛大小等。
    *   使用`瘦脸瘦身`功能调整身材比例（如让头身比更佳，显得更高）。
2.  **背景杂物清理 (使用美图秀秀)**：
    *   进入`图片美化`，使用`消除笔`工具，涂抹需要移除的背景杂物（如乱入的灭火器、无关人物）。
3.  **整体调色与滤镜 (使用VSCO或手机自带编辑)**：
    *   根据照片风格（酷感或温暖），调整`色温`、`色调`、`饱和度`、`对比度`。
    *   选择合适的滤镜，并调整滤镜强度，让效果更自然。
4.  **细节精修 (使用FaceTune)**：
    *   处理头发毛躁、补全发际线（使用`色调`取色填补）。
    *   使用`修补`或`克隆`工具处理更复杂的背景瑕疵。
    *   用`细节`工具锐化眼睛等部位，让神情更突出。

## 学习建议与作业

修图是一项熟能生巧的技能。不要害怕动手尝试。

以下是给初学者的学习路径建议：
1.  **模仿开始**：在小红书等平台搜索“拍照姿势”、“网红打卡地”，模仿别人的拍照角度和修图风格。
2.  **善用工具**：熟练掌握一到两个修图软件的核心功能即可满足大部分需求。
3.  **注重真实**：男生修图切忌过度，以自然、提升质感为目标，避免失真。
4.  **持续练习**：拍照后坚持自己修图，积累经验。

**本节课作业**：拍摄一张自己的照片，分别保存原图和修图后的成品，对比两者的区别，并将成果分享出来。

本节课中我们一起学习了修图的重要性、核心方向、常用软件及完整流程。记住，修图不是为了造假，而是为了还原美好、优化细节、提升照片的整体质感。掌握这项技能，不仅能打造出色的个人展示面，也能在社交中为你加分。现在，就拿起手机和修图软件开始练习吧。
