# 朱利安《高频沟通》课程1：介绍 - 1.3 情绪技能VS分析技能 🧠💬

在本节课中，我们将要学习社交技能中两个核心组成部分：情绪技能与分析技能。理解它们的区别、各自的作用以及如何平衡运用，是提升沟通能力的基础。

## 什么是情绪技能？🤔

情绪技能是内在的、关乎自我状态管理的能力。它本质上关乎**做你自己**或**保持自然**。

上一节我们介绍了沟通的整体框架，本节中我们来看看构成社交能力的第一个支柱——情绪技能。它并非通过理性思考获得，而是通过感受和体验来培养。

以下是情绪技能包含的一些关键方面：

*   **状态管理**：指你当下的感受和情绪状态。你无法通过纯粹的逻辑思考来“解决”糟糕的情绪状态。例如，当你走在街上，想认识某人却感到紧张、拘谨、无法流畅思考时，这不是运用分析技能的时候，而是需要调动情绪技能。
*   **氛围营造**：学习如何与人建立连接，并享受与人交谈的过程本身。
*   **自信与沉稳**：你有多自信，有多沉稳。你是否容易过度反应？保持镇定、不易被激怒就是一种情绪技能。
*   **自我娱乐**：能够在互动中自得其乐。
*   **协作心态**：拥有协作而非竞争的心态。例如，如果你假设环境中的每个人都是你的朋友、站在你这边，你就更容易放松并让真实的自我闪耀。这属于情绪技能，而非分析技能。
*   **为赢而玩**：保持不拘束、开放的状态。
*   **准确的社会意识**：这部分也说明了情绪技能必须优先发展。如果你不培养情绪技能，你就无法看清现实，也无法识别正确的社交模式。不仅如此，如果你总是陷在自己的头脑里，分析一切、怀疑一切，你甚至没有足够的心理带宽去感知你正在获得的经验和参考信息。

**情绪技能构成了社交能力的约80%。**

## 什么是分析技能？🔍

分析技能是更偏重认知、可通过学习与打磨提升的能力。它建立在情绪技能提供的稳定基础上，对沟通进行优化和校准。

在理解了内在的情绪管理后，我们来看看如何通过外在的、可分析的方法来优化我们的表达。分析技能是对情绪技能的自然补充和增强。

以下是分析技能包含的一些关键方面：

*   **模式识别**：正如之前提到的，识别社交互动中的规律。
*   **框架意识**：意识到不同社交动态和框架的存在。
*   **打磨语言技巧**：这非常关键。例如，即使在进行一段演讲时，我可能状态很好，自然地表达了一个概念。但事后回看或思考时，我可能会分析：“如果我用另一种方式说那个概念，先说这件事，或者在那里加个笑话并稍作调整扩展，效果会不会更好？”这种优化就是分析技能。你可以打磨**真实性**和**自然感**。在“做自己”的基础上，你可以通过打磨来强化那个真实的你。
*   **提升社交直觉**：培养对“在特定情境下什么合适、什么不合适、什么容易引起共鸣、什么不能”的感知力。这部分非常重要。
*   **社交校准**：学习如何切入那种真实性并将其表达出来，同时学习如何让它易于被理解，从而与你交谈的对象产生连接。如果你的表达无法与对方连接，那么即使你是真实的，这也算不上一次互动，不是真正的社交。这部分能力很大程度上来源于分析技能。

**分析技能构成了社交能力的约20%。**

## 总结 📝

本节课中我们一起学习了社交技能的两大核心：情绪技能与分析技能。

*   **情绪技能**是内在基础（约占80%），关乎状态、感受、连接和做真实的自己，无法纯粹通过思考获得。
*   **分析技能**是外在优化（约占20%），关乎模式识别、语言打磨和社交校准，可以在真实自我的基础上进行学习和提升。

关键在于，你必须先建立真实的情感基础，然后才能用分析技能去有效地打磨和强化它。两者相辅相成，共同构成高效的沟通能力。
