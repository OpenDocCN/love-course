# 朱利安《高频沟通》课程七：自主练习 7.4 🧠 拿掉社交过滤（口无遮拦）

## 概述

在本节练习中，我们将学习如何识别并移除内心的社交过滤机制。这种过滤机制常常阻碍我们自然流畅地表达，导致“无话可说”的困境。通过一个简单的30秒练习，我们将训练自己捕捉并直接说出脑海中浮现的任何想法，从而解除表达上的自我审查。

## 理解内心的过滤机制

上一节我们探讨了沟通中的基本障碍，本节中我们来看看一个更具体的问题：我们内心的“社交过滤器”。

如果你曾经历过无话可说的时刻，那是因为你内心存在某些过滤机制。事实上，你永远不可能真的无话可说，这是不可能的。原因在于，你们都不是达赖喇嘛，没有人能时刻保持内心绝对的宁静。如果你仔细倾听，此刻你脑海中的那个“小声音”正在喋喋不休。它可能在思考：“你什么意思，我不是达赖喇嘛？” 这本身就是可说的内容。

如果你只是简单地叙述脑海中正在发生的事情，你就能一直说、说、说、说下去，甚至可能在晚上试图入睡时也停不下来。问题在于，这些想法在内心涌动，而你却设置了重重过滤器：“这足够好吗？”“这符合标准吗？”

这个“标准”的高低取决于你交谈的对象。如果你将某人置于高不可攀的位置，你会想：“为了给那个人留下好印象，标准必须很高。”“这不够好，不能说。”于是你把它压下去。接着想：“这个呢？这个呢？”对于每一个浮现的信息，你要么判断它足够好并说出来，要么觉得不够好而压抑下去。这就是大多数人的做法。

这也解释了为何当你与自认为“同级别”或“不如自己”的人交谈时，那个“标准”会消失或降低。你的任务就是去熟悉这个机制，并习惯性地降低或完全消除这个标准。

## 核心练习：30秒“口无遮拦”

以下是本次30秒练习的具体步骤。你将与伙伴面对面进行，轮流发言，每人30秒。

你的任务是：**清空思绪，等待有东西“蹦”出来**。等待一个想法、一个图像、一段记忆、一个人，任何东西。等待任何信息进入你的脑海。然后，不要问自己“这够酷吗？”“这能说吗？”“说出来别人会怎么看我？”“会破坏我的形象吗？”“我还酷吗？”，**直接说出来**。

一旦你说出来，再次清空思绪，等待下一个东西“蹦”出来。

关于这个练习，有几点需要注意：
*   **不要描述环境**：你不能说“房间、研讨会、灯光”这类眼前的东西。
*   **不要延续前一个词**：如果你清空思绪后出现“水、船、泰坦尼克号”，不行，你不能这样联想下去。等待一个全新的东西出现。

如果你能做到这一点并坚持练习，**仅仅缩短从想法产生到说出口的时间**，你就永远不会无话可说。你之所以会词穷，是因为脑海中有太多东西在翻涌，而你却在不断评判：“这个不够好。”“这个可以吗？”“他们不会喜欢这个。”这导致你陷入越来越深的循环，阻力越来越大，情况越来越糟。

**直接说出来**。然后，猜猜会怎样？如果你说了什么，比如“Jesus”，你可能会想：“等等，什么？”那就顺势而为。是的，“Jesus”，你知道的，就像他发明了钢笔，对吧？无论它是什么。

## 总结

本节课中，我们一起学习了沟通中一个关键的内部障碍——社交过滤。我们了解到，“无话可说”的根源并非缺乏想法，而是对想法的过度审查和评判。通过“30秒口无遮拦”练习，我们实践了如何**清空思绪、捕捉第一念头并直接表达**，从而训练自己绕过这些过滤器，实现更自发、更流畅的沟通。记住，核心在于**缩短从“想到”到“说出”的时间，并停止自我评判**。坚持练习，你将能显著改善沟通中的流畅度与自信。
