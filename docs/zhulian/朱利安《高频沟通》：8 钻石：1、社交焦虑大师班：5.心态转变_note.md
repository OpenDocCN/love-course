# 朱利安《高频沟通》课程8：钻石心态 - 第1节：社交焦虑大师班（5.心态转变）💎

在本节课中，我们将学习如何通过一系列具体的心态转变，从根本上改变你与社交焦虑的关系。我们将探讨如何停止抵抗、识别隐藏的获益、打破负面反馈循环，并最终将行动的动力来源从恐惧转变为爱。

## 概述：心态转变的核心

理解问题的根源是内在的，但这并不意味着你不能同时采取行动或在心理层面进行工作。关键在于一切都要源于正确的地方，并且你理解所有部分是如何组合在一起的。

当涉及到那些心理层面的心态转变时，第一步至关重要。

## 第一步：改变你与社交焦虑的关系 🧘‍♂️

这一步非常关键。可以说，让你困在社交焦虑状态中，或者当你被触发时真正起作用的，是**抵抗**。如果你抵抗某件事，你所抵抗的就会持续存在。

假设有一种情绪，你体验到的是与现实相符的、适量的社交焦虑。即使你将自己置于社交场合，可能仍会感到些许不适，但那是正常的、适量的感觉。

然而，接下来会发生的是，一层又一层的抵抗叠加其上，从而放大了焦虑感。这就像你感觉糟糕，然后你因为感觉糟糕而感觉更糟，如此循环，不断放大。

正是这种循环让我们陷入失控的漩涡。这时你需要学会放手，但首先要在心理上改变你与它的关系。问问自己：**如果这就是目标，我能爱它吗？我能无条件地爱它并接受它吗？**

仅仅通过改变你与它的关系，它对你的控制力和强度就会下降。想象一下，如果你看过的所有视频、所有教练课程都告诉你，目标就是体验社交焦虑，那会发生什么？你就不会陷入“哦不”的恐慌漩涡，而是会说：“是的，就是这样。”

所以，改变你对它的标签。它不再是负面的东西。**社交焦虑是好的。爱它。社交焦虑是你的朋友。每一种情绪都站在你这边，每一种情绪都是你的朋友。**

这能真正切断那种失控的恶性循环动态——为感到害怕而害怕，为感到糟糕而糟糕。通过提醒自己“如果这很棒呢？”来切断它。

## 第二步：识别隐藏的获益 🎯

认识到总是有一个隐藏的获益，解释你为什么会有这样的感受。抵抗确实是让它持续存在的原因之一，但还有**执着和成瘾**。

你可能非常执着于体验它，因为或许有一个核心信念在作祟：你认为“这让我保持活力，这让我安全”。你可能会想：“我应该一直偏执、过度思考并体验社交焦虑，因为这能让我安全。我不想让自己陷入危险，我不想让人走进我的内心，我不想信任错的人。体验恐惧能让我安全。”这可能是你的策略，可能是你认为的获益。如果你认为它能让你安全，你当然会紧紧抓住它。

这引出了下一点，一个重要的心态转变。

## 第三步：提醒自己，你可以出于爱而采取保护措施 ❤️

我们通常认为，保护自己的唯一方式就是体验恐惧。例如，开始关注健康的唯一原因，是害怕不关注健康的后果；赚钱的唯一原因，是害怕贫穷。我们认为，防止被利用、防止信任错人的唯一方法就是恐惧，因此我们必须一直感到害怕、质疑一切、保持防御状态。

然而，关键在于：**你可以出于爱而采取同样的行动步骤**。这不仅仅是恐惧的问题，而是源于何处的问题。改变源头。

例如：
*   你可以关注健康，不仅仅是因为你害怕不这样做的后果，而是因为你爱自己、尊重自己的身体。
*   你可以出于自爱而获得成功、赚钱。
*   你可以出于自爱和自尊而设定某些界限，不一定非要由恐惧驱动。

所以，这里往往存在一个常见的获益。但你可以问自己：**体验这种感觉是如何让我感到安全的？它可能如何使我受益？**

一旦你识别出它，就可以运用一些心态转变来帮助自己度过当下，然后才能真正地放手，触及问题的根源。

上一节我们探讨了如何识别并接纳焦虑背后的获益，接下来我们来看看如何打破由焦虑引发的负面感知循环。

## 第四步：打破负面反馈循环 🔄

内在状态总是会影响外在感知。如果你经常体验很多恐惧和社交焦虑，其运作方式是：它会劫持你的注意力和**网状激活系统（RAS）**。

然后，你会透过这个滤镜来看待现实，无意识地（甚至是有意识地）寻找与之相符的证据，同时屏蔽或忽视任何与之不符的事物。

假设你外出并想去酒吧认识一些人。如果你带着恐惧进入，你对环境的看法将与没有恐惧时截然不同。你可能会走进场所，第一个假设就是：每个人都在盯着我，每个人都反对我，每个人都在评判我。你会想：“我一个人来的，大家可能都在想他一个人在这儿干什么？这里的人可能都互相认识，他们看起来都很刻薄、很吓人。”你把所有这些都投射到他们身上。

那么，这样看待现实会对你的感受产生什么影响？它会放大恐惧。所以我体验恐惧 -> 过滤我的感知 -> 看到可怕的事物并将现实扭曲成可怕的样子 -> 这让我感到更害怕 -> 从而更严重地劫持我的注意力 -> 寻找更可怕的事物 -> 让我感到更害怕……如此循环放大。

这就是那个会让你陷入困境的负面反馈循环。你可以切断它。从根本上切断它需要内在的工作，但在当下，你可以**转移你的注意力和RAS**。

将其转移到相反的证据上。如果你体验到很多恐惧，那就去寻找证明“这里不应该有恐惧体验”的证据。寻找任何能让你质疑、能对抗恐惧状态的证据。

例如，你可以在环境中寻找微笑。就只是寻找人们脸上的笑容。就这么简单的事情，突然就会改变你的感知，打破“周围都是可怕的人”的想法。

在更宏观的层面，你可以进行“寻找善行”的练习。我让客户每天写下他们看到的**三件善行**，世界上善良的迹象。这提供了证据，可以打破“每个人都很可怕、刻薄、想害我、只为自己着想”等想法。不，如果你开始寻找善行，就会发现相反的证据。

这确实可以打破那个反馈循环：**寻找相反的证据**。

从那里出发，你可以采取下一个心态。

## 第五步：采取社交科学家的心态 🧪

这也有助于将你的自我价值从正在发生的事情中剥离出来。人们外出时常常会想：我的自我价值取决于我表现得多好，以及别人如何回应我。所以，如果我与人交谈，对方回应良好，我的信心和自我价值就上升；如果交谈不顺利，就下降。

结果往往是，他们的社交体验非常不稳定，时好时坏，这真的取决于运气。他们希望今天会不一样，但即使外出，这也不过是一场“加一减一”的游戏：我和一个人交谈，反应好，加一；和另一个人交谈，反应好，加二；和某人交谈不太好，减一，但还是正一。然后你甚至可能开始过滤，寻找一些容易获得好反应或更安全的互动来获得更多“加一”。你的外出时间就像在玩这个游戏，试图保持绿色（正值），而不是真正地去玩、去认识人、活在当下，不让一切都关乎你和你的自我价值。

为了帮助解决这个问题，你需要**跳出自我**。放手去做，但要跳出自我，在心理上告诉自己：“这不再全是关于我和我的自我价值。让我们采取这种社交科学家的心态。”这就是你的身份：你外出时，就是一个社交科学家。

当你与人交谈时，更多的是关于：“嘿，这次互动怎么样？这个人是谁？我此刻感觉如何？他们感觉如何？什么能产生共鸣？”而不是“这让我变得更好了吗？我是一个更好的人了吗？我的自我价值上升了吗？”不，你只是外出，观察正在发生什么，分析你内心的数据，分析行动、回应、人群、模式、动态等方面的数据。你只是在收集所有这些信息。

仅仅是这种小小的社交科学家心态转变，就可能产生巨大的影响。就像：“哦，你只是一个科学家。”这在我早期确实帮了我大忙。那时我非常依赖这种问责心态，这最终促使我采取了行动。但除此之外，很多时候我都在想：“我是一个社交科学家。”你不想一直停留在这个工具层面，但它的确有帮助。我会想：“好的，第一个朋友，我先打招呼，情况如何？好的，当这样时，我这样说了，然后发生了这个，很好。下一个人，哦，那不一样，这里这里这里。”你的注意力集中在模式上，而不是你自己身上。

接下来，我们看看如何确保每次行动都有收获。

## 第六步：为获取参考经验而行动 📝

无论发生什么，采取行动都有价值可以提取。**你永远不会因为采取行动而失去任何东西**——除非你的自我价值与之挂钩，然后你会想：“哦，那我就会得到一个减一。”把自己从中抽离出来，你总能学到东西。你不会输。

所以，你采取行动，结果没有按计划进行，你可以从中吸取教训。这是一个很好的参考经验。你采取行动，结果按计划进行，很好，这是另一个很好的参考经验。**任何时候只要涉及行动，就有参考经验，就有可以从该参考经验中提取的价值。你不会输。**

也要提醒自己这一点。比如：“我有点害怕和那个人打招呼。”嘿，别把这变成关于你的事。你不会输。

这很有趣，就像读书一样。人们不读书的一个很大原因是，他们知道自己**无法读完那本书**。看看过去，很少有人能读完一整本书。所以他们会想：“我尝试过，我涉猎过，我读了所有这些书，但从未读完。那么，如果我不会读完，我为什么要开始读一本新书呢？”这就是他们的心态。但论点是：虽然那可能不是最优的，但猜猜看，假设你再也读不完一本书了。你认为什么对你更有益：再也不读书，还是读上百本书的第一章？读第一章就有价值可以提取。你不必为了获得价值而读完一本书。

这里也一样：为什么我要打招呼，如果它可能不顺利？如果我不保证它会按计划进行呢？如果它不按计划进行，那就是浪费吗？不，有价值可以提取。道理相同。

最后，我们来看一个能提升行动能量的心态。

## 第七步：以“愤怒”为目标 🚀

这归根结底是理解转变的不同层次。我教授过一个“转变量表”，它从低到高包括：**冷漠、悲伤、恐惧、愤怒、勇气、欲望、目标、爱**。

当你开始自我成长时，往往会经历这些阶段。每向上移动一个层次，你就能获得更多的能量和力量。以采取行动为例：出于恐惧的行动，像是“哦，防御措施”，然后你采取行动。是的，你会采取一些行动。但如果你实际上提升到“愤怒”的层次——这里指的是那种“加把劲，下定决心，让我们干吧”的愤怒——你也会采取行动，但会有更多的力量和能量。然后你可以一直向上移动到“爱”。

所以，一个让人们走出恐惧的方法是：**瞄准下一个更高的层次**。什么是比恐惧更高的层次？是愤怒。这不是那种“去他的别人”的愤怒，而是更像“来吧，让我们做这个，让我们坚持下去”的愤怒，一种非常自我激励、自我鼓励的愤怒。

大多数自助社区就停留在这个层次：那些拼命三郎，自我攻击、自我憎恨的层次。你不想停留在这里。但为了采取行动，为了下定决心说“采取第一个行动”（这在“行动-触发-释放-重复”的公式中），提升到愤怒的层次真的很有帮助，而且是更多自我鼓励的愤怒。

我常举的例子是：你在玩电子游戏，不断失败同一关。一个人可能会开始螺旋式下降，陷入悲伤（受害者心态），甚至看起来像愤怒，他们会说：“哦，这太糟了，太难了，都是因为手柄，去他的。”结果呢？他们扔下手柄，气冲冲地走开，甚至可能自我攻击：“你太烂了，一切都太烂了。”这不是目标，这是你想要避免的。

另一个人可能有完全相同的经历，一次又一次地失败同一关，但每次失败实际上都激励了他们。他们失败后会说：“再来一次，上！再来一次，来吧，再来一次！”这激励他们采取行动。在这里也要采取这种方法：“哦，这很可怕。哦，这太糟了。哦，这太难了。来吧，上！恐惧？让我们上！”瞄准那种愤怒。

这些是更侧重于外部情境的心态转变。

## 总结 🎯

本节课中，我们一起学习了七种关键的心态转变来应对社交焦虑：
1.  **改变关系**：停止抵抗，尝试接纳甚至“爱”上焦虑。
2.  **识别获益**：发现焦虑背后你认为的“安全”等隐藏好处。
3.  **转换动力**：认识到保护措施可以出于自爱，而非恐惧。
4.  **打破循环**：主动寻找相反证据，扭转负面感知滤镜。
5.  **科学家视角**：以观察和收集数据的心态参与社交，剥离自我价值。
6.  **为经验行动**：坚信任何行动都能提取有价值的“参考经验”，永不失败。
7.  **提升能量**：利用自我激励的“愤怒”来获得更多行动力量。

通过持续练习这些心态，你可以逐步削弱社交焦虑的控制，建立更健康、更有力量的内心状态。记住，转变是一个过程，每一次尝试都是向前的一步。
