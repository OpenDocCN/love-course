# 朱利安《高频沟通》课程8：💎 钻石法则1：社交焦虑大师班（7. 三种疲劳）

在本节课中，我们将学习一个关键的模型——“三种疲劳”。这个模型解释了社交焦虑中常见的三种精神消耗状态：决策疲劳、人设疲劳和自我评判疲劳。理解并克服它们，能有效减少内耗，让你在社交中更轻松自如。

## 概述：什么是“三种疲劳”？

“三种疲劳”是一个用于分析社交焦虑核心阻碍的模型。它指出，过度的思考（决策疲劳）、维持虚假形象（人设疲劳）以及严厉的自我批评（自我评判疲劳）会大量消耗你的心理能量，让你感到疲惫、停滞不前。接下来，我们将逐一拆解这三种疲劳及其应对策略。

## 1. 决策疲劳

决策疲劳发生在你开始过度思考或陷入偏执时。理论太多，导致你不断纠结：“我该说这个吗？我该做那个吗？我该打招呼吗？还是不该打？”这是社交焦虑的一大支柱，它消耗你，让你困在原地。

针对决策疲劳，有以下几点建议。

以下是应对决策疲劳的具体方法：

*   **采取行动**：不要只看教学内容。对于每观看一小时的内容，至少要采取一小时的行动。甚至比例可以更高，例如每观看10分钟内容，就采取一小时或更长时间的行动。关键在于，必须始终平衡理论学习和实践行动。否则，你会陷入过度思考和理论过剩的决策疲劳，从而引发大量焦虑，真正让你停滞不前。

*   **原因在于**：通过结合理论学习与实践，你才能开始内化知识。这使你能在此基础上，逐步添加更多、更复杂的理论。否则，你就会卡住。

*   **一个类比**：这就像玩一款电子游戏（例如《地平线：零之曙光》）。游戏设计是渐进式的：第一关教你一些基本操作（例如按这些键组合可以做出某个动作）。当你掌握了第一关，第二关会在基础上增加更复杂的任务。这是一个渐进的学习过程。到了最后一关，你可能在做各种复杂的操作，但很多基础动作已经变成“自动驾驶”般的本能。如果你只是阅读游戏手册，学习所有复杂的按键组合，然后直接去玩最后一关，你一定会不知所措。你需要先内化基础，让它们变成自动反应，从而腾出更多的心理带宽去处理更复杂的内容。社交技能的学习同理。

*   **一个关键原则**：在实践中，应注重**宏观概念**而非**微观概念**。
    *   **微观概念**示例：保持放松的肢体语言、注意不同的面部表情、留意自己的音调等。这些是需要意识到的细节。
    *   **宏观概念**示例：**“进入身体感受，走出头脑思考”**。做到这一点，就能自动处理好许多微观层面的表达。再比如，**“学会进入一种自然流畅的状态”**，这能让你不必过分担心要说什么或怕没话说。因此，专注于一个宏观概念，它能解决众多微观问题。这样做，你可以用更少的专注达成更多的效果，这在实践中非常有益。

上一节我们探讨了因过度思考而产生的决策疲劳，接下来我们看看第二种疲劳，它源于我们对外在形象的执着。

## 2. 人设疲劳

人设疲劳源于你对自我形象的焦虑。你对自己是谁、希望如何被看待有一个特定的形象，并且害怕别人会看穿你的伪装。同时，你非常需要获得好的反应和认可来维持这个形象。

对此，请记住：**重在表达，而非印象**。这不是为了管理某种印象，而是真实地表达你自己。

一个有效的外部工具是：**做一些打破常规的事**。

例如，假设你外出时，内心非常希望自己被看作很酷、高价值，像詹姆斯·邦德一样。这种想法会束缚你，你会想：“我所有的行为都必须符合这个人设。詹姆斯·邦德会怎么做？我不能做任何詹姆斯·邦德不会做的事。”这就像一个无形的监狱。

这时，去做一些打破这个模式的事情。比如，一个经典的练习是：**立刻做几个俯卧撑**。即使在一个高档场所，你也可以说“我要做俯卧撑了”，然后做上五个。可能会有人看你，但最重要的是，这对你意味着什么？它**毁掉了你被看作“詹姆斯·邦德”的可能性**。你不再能维持那个“酷”的形象了。这听起来可能令人沮丧，但实则**极度解放**。突然之间，那个“我必须很酷”的模具被打碎了。你可以完全退出那个“装酷游戏”，做回你自己。

我们了解了维持虚假形象带来的消耗，最后一种疲劳则来自我们内心最严厉的批评者。

## 3. 自我评判疲劳

自我评判疲劳包含大量的自我攻击和担忧。

一个很好的外部工具是：**收集并庆祝你的胜利**。

你也可以在采取行动后，审计内心的声音。例如，你和某人打了个招呼，进行了一分钟的互动，然后离开。对大多数人来说，一旦离开，内心立刻开始自我攻击：“你应该做得更好。刚才太糟糕了。那个人会怎么想？你为什么没那样做？”这是一个充满自我攻击和担忧、令人不悦的声音。

与之相反，你应该收集那次胜利：“嘿，你和某人交谈了一分钟，是的！”给自己一些鼓励和关爱，拿下那次胜利。

这样做的好处是：
1.  你将能乘上积极动能的浪潮，不断收集一个接一个的胜利。
2.  仅仅是你承诺要改善沟通技巧、人际技能，并走出去实践，与更多人交谈（这比很多人一生中主动交谈的次数都多），这一点就值得大力赞扬。大多数人因为太害怕而不会这么做。仅凭你在做这件事的事实，你就应该给自己很多肯定。
3.  为了让成功经验内化并重塑你的认知参照，你必须让这些成功“落地”。如果总是自我攻击、自我攻击，效果会很差。

## 总结

本节课中，我们一起学习了“三种疲劳”模型：**决策疲劳**、**人设疲劳**和**自我评判疲劳**。我们了解到，过度思考会消耗精力，应对方法是**平衡学习与行动，并聚焦宏观概念**；维护虚假人设会带来压力，解决之道是**打破常规，真实表达**；而严厉的自我批评则会阻碍进步，关键在于**收集并庆祝每一个小胜利**。掌握这些策略，能帮助你减少社交中的内耗，更轻松、自信地与人连接。
