# 朱利安《高频沟通》：8 钻石：3、魅力大师课：11.8 从脱口秀中学习_note

在本节课中，我们将探讨如何从现场脱口秀表演中学习沟通与魅力的技巧。我们将分析喜剧演员如何运用身体语言、节奏和内容结构来吸引观众，并提炼出适用于日常沟通的核心原则。

现场脱口秀蕴含着巨大的学习价值。对我个人而言，这是在新冠疫情封锁之前非常重要的事。

洛杉矶的脱口秀氛围非常出色。我每周至少去看一次脱口秀表演。这是我非常热衷且喜爱的事情，是我最喜欢的活动和爱好之一。洛杉矶拥有最棒的演员阵容。你在现场可以看到，一位喜剧演员站在观众面前。

很多观众甚至不认识你。你如何能抓住他们的注意力？这当然离不开幽默。你也能观察到他们构建笑话的方式。

尤其是亲身观看时，这种感觉更为明显。亲身观看与通过电影或网飞观看的不同之处在于，你和现场氛围之间没有隔阂。你能直接感受到它。你能看到他们引爆全场的时刻，也能看到他们冷场的时刻。例如，我看过丹·库克的演出。

很多年前我年轻时很喜欢他。后来我多次看他的现场演出，但他的表现并不好。我感觉他的水平下降了。但有一次，他的表演对大多数人来说很失败，但对我来说却是看过最棒的脱口秀之一。他完全脱离了以往的套路，尝试极端的新素材。那种素材令人震惊，整个房间的观众都愣住了。所以，再次强调，亲身观看能让你看到幕后的东西。如果你能去现场，我强烈建议你去。如果不能，可以上YouTube或网飞看一些节目。但就提升魅力和学习如何按下“有趣”按钮而言，今年是关键。

我记下了丹·库克，他是我早期的影响者之一。我从他身上获得的重要价值是：用全身说话。就像我之前说的，他非常擅长将整个身体投入到表演中。在讲故事时，他会跪下来。他有一些标志性的、有点奇怪但毫不夸张的丹·库克式动作、姿态和姿势，这些与他本人高度一致且符合他的个人风格。你可以看到，有人把自己变成了一个漫画式的形象。我从他身上学到的主要就是如何用身体说话。

罗素·布兰德是另一个例子。从丹·库克那里学到用身体说话，从罗素·布兰德那里，类似于我们讨论乔丹·彼得森时说的，他有一种非常聪明地阐述不同观点的方式。同时，他将幽默与教育融合在一起。他的喜剧秀不仅仅是小品，里面实际上有生活教训，有时甚至是更深层次的精神教诲，尤其是在当下，与幽默混合在一起。这是我从他那里学到的重要一点。

吉姆·杰弗里斯。这前三名并非绝对是我的最爱，但他们是最早对我产生最大影响的人。吉姆·杰弗里斯在脏话运用方面影响巨大。这听起来好笑，但如何把握说脏话的时机，以及用脏话为你所说的内容增添画面感、冲击力和趣味，这是一门学问。

此外，还有一点是倾向于略带震惊的幽默边缘。所以，这三位在我心中占有特殊地位。

以下是我列出的其他我非常喜爱的演员，每一位都是满分十分。

*   戴夫·查普尔

*   安德鲁·桑蒂诺（非常棒，在他因播客爆红之前我就看过他的一些演出，他在一些电视节目中的表现也很出色）
*   米奇·格拉泽
*   托尼·洛克（克里斯·洛克的兄弟，我看过他的一场慈善演出，非常精彩）

关于托尼·洛克的那场慈善秀：那是在洛杉矶圣诞节期间的一场慈善演出，你需要买票并带一份礼物，他们会把礼物送给孩子们，比如孤儿院。托尼上台后，整场表演都是即兴的，他一直在嘲讽活动组织者，说对方只是为了品牌和生意，并非真心做慈善，非常有趣，是我看过最棒的演出之一。

关于哈兰·威廉姆斯：他出现在同一场演出中。那是一场很奇怪的秀，他上台后彻底冷场了，可以说是最糟糕的表演之一。他拿起所有玩具说“这些孩子都会得到玩具，除了一个”，然后砸碎了一个玩具。他可能想当作一个玩笑，但玩具真的碎了，他说“有一个孩子不会有玩具了”。没人笑，只有我轻笑了一声。但随后他的整个表演，也是我喜爱的部分，完全是即兴的，有着非常独特的幽默感和词汇。他即兴编造了一个疯狂的故事，甚至给自己起了个假名，扮演一个角色，整个故事建立在之前喜剧演员表演过的笑话基础上。他听了之前所有人的表演，然后即兴发挥出来，真的很厉害。

本节课中，我们一起学习了如何通过观察优秀脱口秀演员的表演来提升个人魅力与沟通能力。我们认识到，**用全身说话**（`embodying your message`）、**精准的节奏把控**、**将幽默与有价值的内容结合**，以及**现场即兴互动**的能力，都是构成魅力的关键要素。无论是亲身前往现场感受氛围，还是通过视频学习结构，核心在于观察、分析并内化这些沟通大师们吸引和打动观众的技巧。
