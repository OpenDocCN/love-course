# 朱利安《高频沟通》课程笔记：4.12 如何滔滔不绝 🗣️

在本节课中，我们将学习如何避免在社交中陷入无话可说的尴尬境地。核心在于理解谈话的层次，并调整对自身言语价值的认知。

## 概述：谈话的四个层次

首先，请记住谈话有四个层次。理解这一点意味着，你所说的话其实是**最不重要**的一层。

如果你过于看重自己说的话，你就已经搞错了重点。人们无话可说的首要原因，就是他们**过分看重自己要说的话**。

试想，如果你说的每句话都至关重要，那么你当然会感到压力巨大，最终无话可说。给自己设定“必须说出完美的话”这种高标准，只会让你不堪重负。

## 核心原则：降低言语的标准

上一节我们介绍了谈话的层次，本节中我们来看看如何降低对言语内容的标准。

避免无话可说的唯一方法，就是认识到：**我说什么真的没那么重要**。我可以犯错，这没关系。

这同样适用于任何表达场合。如果我试图追求完美，反而可能会结巴或卡壳，因为我给自己施加了太多压力。例如，如果我像写书一样要求自己说话，压力会大到让我从“第一章”开始就无话可说。

所以，请认识到：**说什么并不重要**。在四个层次中，如果有一个可以搞砸，请永远选择搞砸“说什么”这一层。在实际交流中，偶尔的停顿、口误或说些“嗯”、“那个”都无关紧要，对方很可能根本不会注意到。

## 具体方法与实践

理解了核心原则后，以下是具体可操作的方法。

### 1. 降低言语的价值门槛

你需要降低对自己必须说出“金句”的期待。那种“必须说出惊艳的话”的想法，第一不自然，第二反而显得你**很刻意、很努力**。

设想一个场景：一个精灵出现，许诺你能在整场互动中永远说出最完美的话。你会接受吗？如果你的答案是“会”，那你反而害了自己。因为对方会觉得你的回答太完美、太不自然、太奇怪，甚至觉得你**太刻意、用力过猛**。

**公式：高价值 = 低门槛**

通常，我们设定了过高的标准，不断自问：“这够好吗？”。这恰恰是所谓“低价值”人士的做法——设定高门槛并试图取悦他人。**真正高价值、高地位的人，反而可以轻松地说些“傻话”**。因此，如果你想显得高价值，就降低标准，告诉自己“无论我说什么，都足够好”。

这传递出的信息是：即使偶尔笨拙，也显得自然、真实、不刻意。对方会想：“一个正在努力表现的人，会说这种看似‘低价值’的话吗？不会。哇，这个人真酷，他居然能轻松地说出这种话。”

### 2. 享受无话可说的时刻

如果无话可说的情况发生了，请**享受它**。这是一个可以让你自己都感到惊喜的时刻。

不要被它困住。你可以准备一些“万能话题”来过渡。例如，在一个浪漫或认识新朋友的场景中，我用了很多年的一个万能问题是：**“你为什么这么爱我？”** 我可以在一次互动中反复使用它，每当不知道说什么时，就问这个。

或者，你也可以让自己被惊喜。当我无话可说时，我会想：“哇，我接下来会说什么呢？让我们看看。” 这变成了一种纯粹的、当下的兴奋感。

### 3. 应用“接纳-夸张-大笑”公式

这是一个可以应对多种情况的强大公式：**接纳（Embrace） -> 夸张（Exaggerate） -> 大笑（Laugh）**。它能帮你化解某件事带给你的压力和力量感，创造心理空间。

以下是应用步骤：
*   **第一步：接纳它。** 如果无话可说了，那就想：“太棒了，我居然无话可说了，耶！”
*   **第二步：夸张它。** 将情况夸张到一个你觉得 genuinely（ genuinely 这个词保留，强调“真正地”）好笑的程度。
*   **第三步：大笑。** 当你为此大笑时，它就失去了对你的所有控制力。

例如，如果我无话可说了，我可能会说：“哎呀，我没话说了。真希望我上朱利安的社交技巧课时多做点笔记。” 通过夸张到你觉得荒谬可笑的地步，无话可说这件事就失去了力量，甚至变成了你的谈资。

### 4. 转述你的内心想法

要认识到，除非你像冥想大师一样内心毫无杂念，否则你**不可能**真正无话可说。你的大脑时刻都在运转。

如果你不说话，你的大脑也在说话。你只需要**把你脑子里正在想的东西说出来**。这样你就永远有话可说。即使你脑子里想的是：“天啊，我没话说了，天啊，我没话说了。” 你也可以把这句话告诉对方。

### 5. 记住谈话是双向的

最后，请记住一个关键点：**谈话不是一个人的独角戏**。你不是要一个人承担整场对话的压力。如果让任何人独自支撑整个互动，他们都会多次无话可说，这很正常。

所以，有时你停止说话是正常的。这时，你可以**安静下来，微笑着看着对方**。你内心的框架和意图应该是：“现在轮到你了。” 然后对方就会加入进来。当对方无话可说时，你也可以接上。这是一种**双方互相帮助**的完整互动，而不是“我必须通过永不停歇的说话来打动对方”。

## 总结

本节课中我们一起学习了如何避免社交中无话可说的核心技巧。我们首先理解了谈话的层次，认识到言语内容是最不重要的。接着，我们学习了降低言语价值门槛、享受无话可说的时刻、应用“接纳-夸张-大笑”公式、转述内心想法以及牢记谈话是双向互动这五个具体方法。关键在于放下完美主义的压力，以轻松、真实的态度进行交流。
