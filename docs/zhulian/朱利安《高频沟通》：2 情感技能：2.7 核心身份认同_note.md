# 朱利安《高频沟通》：2 情感技能：2.7 核心身份认同_note

在本节课中，我们将要学习“核心身份认同”这一概念。它与核心信念类似，是你认为自己与世界的关系中所处的身份。核心信念关乎“世界是什么”，而核心身份认同则关乎“在世界中，我是谁”。这也可以理解为，你对自己抱有哪些核心信念。

这包括你相信自己值得什么，以及相信自己不值得什么。正是这种认同，维持着我们所谓的“成功上限”。

## 认识你的核心身份

上一节我们介绍了核心身份认同的定义，本节中我们来看看如何觉察它。一个极佳的方法是进行以下练习，我们强烈建议你花时间进行内省和反思。

这个练习是：问自己，如果让你从电影、电视剧、书籍或任何作品中挑选一个虚构角色，哪一个角色最能描述你？

请注意，不是选你“想成为”的角色，而是选你真正产生共鸣、觉得“那完全就是我”的角色。例如，在很长一段时间里，我深深共鸣的角色是“自我毁灭的艺术家”。这个原型主导着我的行为，这就是我看待自己的方式。这导致了许多自我破坏，因为，一个“自我毁灭”的人怎么可能获得完全的成功呢？

## 常见的身份认同原型

以下是人们常持有的一些核心身份认同原型：

*   **受害者**：总是感到被伤害或委屈。
*   **配角**：总是忙于帮助他人，而非主导自己的人生。
*   **弱者/逆袭者**：就像电影《洛奇》那样，为自己创造重重挑战，不让成功轻易到来，总是需要付出巨大努力。
*   **怀才不遇者**：就像《心灵捕手》中的角色，拥有巨大潜力却不去实现。
*   **等待被选中的蜘蛛侠**：等待一个“被蜘蛛咬”般的魔法时刻，才终于愿意采取行动去改变和转型。
*   **背景板**：像电影里无人注意的背景人物。
*   **注定孤独的英雄**：虽然采取了所有行动，但命中注定要孤独。
*   **殉道者**：总是为他人或某种信念牺牲自己。

我们多少都带有这些原型的影子。

## 进行自我剖析

现在，请将你的意识带回来：如果我必须挑选一个角色，哪一个最能概括我？它不必涵盖你的全部，但哪一个最突出？

接着，请分析这个角色：
*   这个角色有哪些特质？
*   这些特质带来了哪些**好处**？
*   这些特质带来了哪些**弊端**？
*   这个角色有哪些行为**模式**？

通过这个分析，你可能会恍然大悟：“哇，也许我为了在社交生活中保持与那个角色的一致和协调，正在破坏很多方面。” 我们会无意识地让自己的行为与核心信念及核心身份认同保持一致。

## 课程总结

本节课中，我们一起学习了“核心身份认同”。我们了解到它是你对自己在世界中角色的根本定义，影响着你的自我价值感和行为上限。通过识别与你共鸣的虚构角色原型，并分析其特质与模式，你可以更清晰地看见那些可能正在无意识中引导你、甚至限制你的自我认同，这是迈向有意识改变的第一步。
