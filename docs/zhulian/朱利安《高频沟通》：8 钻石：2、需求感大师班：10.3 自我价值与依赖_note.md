# 朱利安《高频沟通》：8 钻石：2、需求感大师班：10.3 自我价值与依赖_note

在本节课中，我们将要学习需求感的内在运作机制，理解我们如何通过“自我施加的匮乏感”来制造需求感，并学会如何通过内在工作回归完整的自我状态。

## 需求感的运作机制图解

上一节我们探讨了需求感的表现，本节中我们来看看其核心的心理模式。请记住下面这个图解，它清晰地展示了需求感是如何产生的。

想象一个代表完整自我的图形。需求感的作用方式，并非保持这种完整。我们会在内心制造一种分裂，最终将自我的一部分剥离出来，放置在一堵墙的后面，然后去追逐它。

## 理解“自我施加的匮乏”

这个过程就是：你取走了自我的一部分。你不再是“我是完整的、我是圆满的、我足够好”的状态，而是认为“我要把自我的这部分拿出来”。

然后，你将它放置在一堵墙、一个目标、一件事物的后面，接着你去追逐那个事物，以拿回自我的那部分。

以下是几个具体的例子：

*   **以手机回复为例**：你认为“我不够好”。这里的“墙”就是对方的回复。一旦我收到回复，我就拿回了那部分自我。
*   **以金钱目标为例**：我将自我的一部分放在“赚到X金额”这堵墙后面。一旦我赚到X金额，我就拿回那部分自我，变得完整。
*   **以他人认可为例**：我将自我的一部分放在“获得认可”后面。如果我和这个人交谈并且他们喜欢我，我就拿回了那部分自我。
*   **以恋爱关系为例**：我将自我的这部分放在“一段关系”后面。一旦我脱离单身，我就拿回这部分自我。

这就是我们的行为模式。因此，这本质上是**自我施加的匮乏**：你主动将自我的一部分快速藏到一堵墙后面。这堵墙可以是一个人、一件物品、一个目标，可以是任何事物，然后你去追逐它。这就像在说“让我变得完整，我需要它，我需要克服这个来拿回我的那部分”。这是有害的、有毒的、低价值的行为。

## 解决方案：通过内在工作保持完整

相反，通过进行内在工作，提升情感技能，你停止将自我的一部分放置在某些墙后。你在行动之前就是完整的，在行动之后也是完整的。你仍然可以去追求某个特定的目标、事物或人。

但这不再能“增强”你，你也不会在行动之前感到“被削弱”。核心公式如下：

```
完整自我 (行动前) -> 采取行动 -> 完整自我 (行动后)
```

## 实践中的常见陷阱

我相信你可能经历过这种情况。例如，你学习社交技巧，你打算出去和一群人交谈，练习你在这里学到的东西。这很好。

但你可能会陷入这样一个陷阱：你不是带着“我是完整的、我足够好、我要出去练习”的心态，而是这样想：“我得到了所有这些建议，让我们付诸实践。我要拿走自我的一部分，把它放在一堵叫做‘外出练习时表现良好’的墙后面。我要出去练习，如果我练习得好，反应良好，表现出色，得到了好的回应，那么我就能拿回那部分自我，我会感觉良好；如果我没有，那么我就刚刚围绕‘外出练习’这件事，发展出一种充满需求感的关系。”

这就是它的运作方式：**任何时候，只要你将自我的一部分放在某堵墙后面，需求感就被激活了。**

## 关键原则与总结

因此，关键在于：你在行动前就是完整的，行动后也是完整的。你当然可以去练习，但那不会“增强”你。你拥有多少、做了多少都无关紧要，这永远不会影响你的“存在”本身。

你现在就是完整的。你当然可以采取行动，如果你受到鼓舞去取得更多成就，那就去取得。但不要认为这将会“增强”你这个人，不要认为外在的任何事物能填补内在的空虚。如果内在产生了匮乏感，那是一个需要由内而外解决的课题。

但请记住这个图解：这就是需求感。每当你体验到它时，归根结底就是——你正在将自我的一部分放在一堵墙后面然后追逐它。这是自我施加的，这很疯狂。一旦你意识到这一点，你会想：“等等，什么？”我们总是认为需求感是外在的，是别人造成的。但事实是，这是你自己造成的。你对自己做了这件事。你创造了那种匮乏感，然后你去追逐它。

这就像一个疯狂的例子：告诉自己你现在不能拍手，然后 embark on 所有努力去学会拍手。不，你应该意识到，你本来就能拍手。意识到你是完整的，然后放下执念。

本节课中，我们一起学习了需求感源于“自我施加的匮乏”——即将部分自我价值寄托于外物并加以追逐。通过持续的内在建设，我们可以学会在行动前后都保持完整的自我认知，从而从根本上消除需求感，以更自信、从容的姿态面对目标和人际关系。
