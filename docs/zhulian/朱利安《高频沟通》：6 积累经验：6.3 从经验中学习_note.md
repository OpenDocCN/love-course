# 朱利安《高频沟通》课程六：积累经验 | 6.3 从经验中学习 📚

在本节课中，我们将学习如何系统地分析你的社交互动经验，并利用这些分析结果来指导你未来的行动和学习内容。核心在于建立一个“行动-分析-优化”的循环，让真实的生活经验成为你进步的指南针。

## 分析你的经历

上一节我们讨论了积累经验的重要性，本节中我们来看看如何从这些经验中提取价值。当你回顾自己的社交互动时，必须对自己诚实。你需要明确：这段经历与哪部分学习内容相关联？

例如，你可能学习了大量的沟通理论。当你外出社交并遇到特定情况时，你应该思考：“这个情况有趣，它关联到我学过的哪部分内容？”然后，回去复习那部分内容。人们常问应该观看或学习什么内容。答案是：一旦你建立了知识体系，就应该让生活、经验和具体事例来告诉你需要复习什么。发生了某事，它必然关联到知识体系的某个部分，那就去重新学习那个部分。

如果你不熟悉学习内容，你就无法正确解读经历。但反过来，这些经历会让你对学习内容有更深的理解。两者之间存在美妙的协同效应。因此，你应该始终处于“行动 -> 分析 -> 学习内容 -> 再行动”的循环中。

## 选择新的尝试

有时你可能会感到有能力并想尝试新事物，推动自己的舒适区，做一些不习惯或不同的事情。一个常见的挑战是：如何选择要尝试的新事物？如何选择那些不符合我常规行为模式的事情？

方法是：**基于你最突出的“卡点”来选择**。

每次你写下最重要的三条经验教训时，看看哪一个最频繁地出现。然后，基于这个“卡点”来决定行动步骤：需要调整什么？我一直在那样做，让我停止；我总是忘记做这个，或者害怕做那个，让我尝试去做。

因此，你最常出现的“卡点”或教训，决定了你应该学习的内容和采取的行动。

以下是具体操作步骤：

1.  **观察互动**：进行社交互动。
2.  **进行分析**：拆解互动，找出最重要的三个教训。
3.  **确定重点**：基于教训，思考如果尝试不同的做法，什么会最有帮助。
4.  **计算回报**：问自己，如果只能改进其中一点，哪一点能带来最大的回报？

在微观层面，你可以这样计划：今天外出社交一小时，然后分析，得出三条教训。明天，就专注于解决最突出的那一条。第二天继续实践、分析，如果同一个问题依然突出，就继续攻克它；如果出现了更重要的新问题，就转向它。你可能需要为一件事花费一周时间。

从微观到中观，你可以问：这一周里，出现最多的教训或需要改进的点是什么？接下来一周就专注于此。再到宏观层面，以月为单位审视，找出最重要的宏观问题。就这样，持续让数据和经验来决定你在内容和行动上的关注重点。

## 保持分析心态

这确实需要运用科学家般的逻辑思维。你拥有原始材料，即那些“顺其自然”但并非本能的经历。你的任务是审视这些材料，找出模式，进行剖析、分析、调整和打磨。这是一种非常理性、非情绪化的方法。要对自己诚实，给予自己严厉的反馈。

但是，一旦你提取了价值，就要**收集胜利，重播胜利**。

这意味着，当你外出社交时，总会有一些互动不按计划进行，这很正常。就像学走路不可能不摔倒一样。从不顺利的经历中也能提取价值。但大多数人的默认模式是，不断在脑海中回放那些不顺利、失败的经历，反复咀嚼为什么自己会那样感觉，重播那个失败的自己。

你想从失败中获取的只是数据，这是失败唯一的价值。你不应该反复重播失败，而应该重播胜利。这样，你才能开始与那个成功的、真实的自己融为一体。

以我为例，这么多年里，对于任何所谓的“失败”，我都会 brutally honest（残酷地诚实）地提取数据，但然后我会庆祝行动本身，庆祝获得的经验，重播胜利的时刻，真正开始培养一种向上的螺旋，激活你的“胜利者效应”。

## 克服自我阻碍

这里还需要意识到：你对某事投入越多，你的自我就越容易卷入其中。

如果你以某种方式社交了一段时间，你会非常依恋那种方式。这是第二难的部分。

第一难是，人们会采取行动但不分析，所以总在重复同样的轨迹。
第二难是，人们分析了行动，但依然继续做同样的事，因为他们的自我卷入其中了。

你必须把自我放在一边。如果你得到的数据开始表明：“你这样做、这样说并不是最优的”，那就放弃它，改变它。不要固执，不要顽固，不要一次又一次地在同样的事情上加倍下注，指望这次会不同。现在你有了数据，就去改变。

我看到过有些人固执到多年重复做同一件事。你必须意识到，在人生进步中，尤其是在社交方面，**让你走到今天的，未必能带你走向未来**。

这意味着，有些方法在一段时间内，你意识到它们、使用它们、表达它们，并且效果很好，极大地帮助了你的社交。但之后你开始得到数据，发现那恰恰是搞砸事情的原因。那时你就必须放弃它，即使它可能帮助了你很多年。

让你走到今天的，不是带你走向未来的。决定未来走向的，是数据。**相信数据，而非你的自我。**

---

**本节课总结**：我们一起学习了如何从经验中系统学习。关键在于建立“行动-分析-优化”的循环，用真实的社交反馈指导复习和尝试。要诚实分析，找出“卡点”并针对性改进，同时保持理性，庆祝胜利而非沉溺失败。最重要的是，要信任客观数据，勇于放弃不再有效的旧模式，让持续的经验和数据引领你不断进步。
