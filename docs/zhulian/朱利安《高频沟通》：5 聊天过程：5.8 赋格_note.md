# 朱利安《高频沟通》课程5：聊天过程 · 5.8 赋格 🎵

在本节课中，我们将要学习社交互动中的一个核心平衡艺术：在展示价值的同时进行“赋格”。这指的是在对话中，既要让对方看到你的独特魅力，也要让他们明白你为何选择与他们交流。

上一节我们介绍了在对话中建立吸引和投资的重要性，本节中我们来看看如何通过“赋格”来维持社交天平的稳定，确保互动顺畅进行。

## 价值与赋格的平衡

对话中必须存在一种平衡。其运作机制如下：你倾向于主动开启对话，展示你个性的一面。对方会因此感知到你的**价值**。

当对方感知到价值时，也会产生第二个情感钩子：**情感投资**。因此，整个流程是：展示价值 → 引发投资 → 进行赋格。

这就好比从一开始就设定框架，让对方明白你为何与他们交谈。你必须让他们知道，像你这样有价值的人，是出于**价值层面的原因**与他们交谈，而非仅仅因为情境（例如“我想认识你”或“因为这件事”）。

如果对方心想：“这个人如此出色，为什么会和我这样平凡的人说话？”你就需要通过赋格让他们明白原因。

## 为何需要赋格？

赋格意味着让对方知道原因。无论情境如何，当你与一个自觉平凡的人交谈时，他们内心会问自己：“这个充满价值的人在我身上看到了什么？为什么在所有人中选择花时间与我交谈？”

尤其是当对方自尊水平较低时，这会是他们脑海中的第一个念头：“为什么是我？这个人社交能力出众、见识广博、风趣迷人，而我有点害羞、并不出众，他为什么会对我感兴趣？”因此，你必须让他们知道原因。

否则，他们会感觉不对劲，怀疑你是否在捉弄他们、试图诈骗，或者伪装成另一个人。他们会觉得“这太奇怪了，这说不通”。

**没有赋格的价值展示是说不通的。**

## 如何进行赋格？

以下是几种进行赋格的核心方法，它们应该自然地发生，但你需要有意识地去关注这些动态。

**表达好奇心**
这是最常用的方法。起初你可能在展示自己、开玩笑，然后你发现“这个人其实挺酷的”，于是你开始询问关于他们的问题。例如：“顺便问一下，你是做什么工作的？”或者“等等，告诉我更多关于这个的事，我真的很想知道。”这表现出你的兴趣，但并非需求感，而是一种“等等，我想深入了解你”的好奇。你可以借此寻找共同点，例如：“这太巧了，我也是！”。**基于事实，切勿撒谎**，即使是简单的共同点（如“你也会说英语？我也是！”）也有效。

**展示脆弱性**
分享一些个人的、私密的事情，向一个你还不太熟悉或不十分信任的人，展露你通常不会轻易示人的一面。这向对方传递的信息是：“这个人愿意分享如此真实和私人的事情（并且在此情境下是社交可接受的），他们一定不会对任何人都这样。他们一定真的很喜欢我、信任我。”

**情感化的直接表达**
你可以直接说：“你知道吗，我真的很喜欢你。我不知道为什么，我无法用语言形容，但就是有这种感觉。”或者“我们之间有种特别的连接感。”你不需要给出逻辑解释，只需陈述你被吸引的事实。

所有这些方法都能达到赋格的目的。但你必须持续进行，随着你的价值展示或对方投资的增加，不断进行赋格。否则，价值与赋格之间的差距会过大，破坏平衡。这并不意味着对方不觉得你出色——你可能展示了过多价值，对方依然认为“这个人太棒了”，但他们可能会因为觉得自己配不上而不愿与你长时间相处，或者认为其中有问题。

## 平衡的长期演变

关于价值与赋格的平衡，一个常见问题是：在友谊或关系建立之初需要这种平衡，那么在关系稳固后，这种平衡还需要持续吗？强度会减弱吗？

这是一个很好的问题。在关系初期，这种平衡无疑会被**放大**。因为你们彼此还不熟悉，你正在快速提升价值并展示广泛的个性层面，所以你需要迅速用赋格来平衡。由于初期价值展示的强度很大，赋格也会相应被放大。

从长期宏观角度看，这种动态依然存在，但它会更多地**自然发生**，你不再需要有意识地进行逻辑思考。只是在开始时，因为你试图快速提升价值并充分展示自己，你需要有意识地加强赋格来与之平衡。

本节课中我们一起学习了社交互动中的“赋格”艺术。我们明白了，单纯展示高价值可能让对方感到不安或疏远，因此必须通过表达好奇心、分享脆弱性或直接情感认可等方式，让对方明白你被吸引的原因，从而维持价值与认可之间的健康平衡。这种平衡在关系初期尤为重要，随着关系深入，它会逐渐融入自然的互动节奏中。掌握这一点，能让你的沟通更加顺畅、真诚且富有吸引力。
