# 朱利安《高频沟通》课程5：聊天过程 - 5.5 共振 🎯

在本节课中，我们将要学习对话中的“共振”阶段。这是成功开启对话并建立连接后，维持高质量互动的核心环节。我们将深入探讨如何识别共振时机、理解其本质，并掌握实现自然共振的方法。

## 从“吸引”到“共振”

上一节我们介绍了对话中的“吸引点”。一旦你通过了第二个吸引点，对话的能量分配就会从“90-10”（你主导90%）转变为“50-50”的平衡状态。

**公式：** `对话能量比：吸引点前 = 90:10` -> `吸引点后 = 50:50`

此时，你可以放慢节奏。这意味着你已经成功“吸引”了对方。很好，接下来让我们进入“共振”阶段。

这是任何对话都会进入的阶段。无论是在友谊、浪漫关系还是商务场合中，如果你想认识新朋友，或与已有一定默契的人开始互动，流程都是一样的：永远是先“开启”，再“吸引”，只有通过了吸引点，你才能转向“共振”。

在共振阶段，你可以开始更逻辑性地谈论你想达成的目标等等，但这一切都发生在“共振的海洋”中。例如，即使我现在进行一场商务会议，我也会遵循相同的步骤：像对任何人一样开启对话，穿插不同的玩笑和话题，直到“吸引”成功，然后转向共振，最后在共振的氛围中讨论更逻辑性的商业议题。

## 识别“吸引”信号

这里再次强调一个重点：要意识到你已经“吸引”了对方。许多人认为社交被拒是因为没有吸引成功，但我发现，大多数社交被拒其实发生在已经吸引成功、但当事人没有意识到的时候。

**代码描述此误区：**
```python
if 已吸引成功 == True and 意识到吸引 == False:
    行为 = 继续用力展示个性
    结果 = 显得怪异、刻意、 desperate -> 被拒绝
elif 已吸引成功 == True and 意识到吸引 == True:
    行为 = 放松，进入共振
    结果 = 互动良好
```

你可能走上前，处于兴奋状态，其实已经吸引成功了，但你还在试图展示个性，这反而显得怪异、刻意和 desperate，从而导致被拒。如果你能意识到“哦，太棒了，已经吸引成功了”，那你就可以放松了。

这种意识到吸引的过程通常发生得非常快。大多数情况下，它发生在头15秒内，你就能通过两个吸引点。虽然也可能耗时更长，但那是例外，不是常态。前提是你的“氛围状态”调整得当。所以，**意识到你已经吸引成功**，这点至关重要。

## 共振的本质：垂直体验

从此，对话进入美好的50-50平衡状态。根据我的经验，这是互动中最有趣的部分。在这里，你可以放松下来，享受与对方相处的时光。

关于共振，很多人的一个主要障碍（尤其是像我这样逻辑性强的人）是必须明白：共振不是基于议程的。当你与某人共振、只是去了解和体验对方时，这不是一件逻辑性的事情。人们常想：“好吧，要共振，下一步是什么？我要达成什么？”而实际上，共振本身就是目的，就是为了共振而共振。目标就是共振，就是为了体验彼此而体验彼此。

这就是为什么你有时会看到人们谈论一些你可能认为没什么建设性的话题，但他们却乐在其中。因为重点甚至不在于话题本身，而在于连接和体验另一个人类个体。

这引入了互动中**水平发展与垂直发展**的概念。
*   **水平发展**是更逻辑性的：现在我们做这个，现在我们讨论那个等等。就像商务会议中，坐下、开启、吸引，然后在会议中会有这些水平方向要讨论的事情：我们要讨论这个，要解决那个问题，要做这个。
*   而在更情感化的层面上，存在着事物的**垂直面**，那就是共振。

**公式：** `高质量互动 = 水平发展（逻辑议程） + 垂直发展（情感共振）`

因此，共振本质上是这种垂直体验，就像是为了做而做。当你基于这个基础（共振）时，你就可以水平向前推进。

## 共振的价值与挑战

能够共振也是一个人高价值以及良好社交智商的标志。尽管这听起来很简单——“嘿，放松下来去了解对方就行”，但实际上比看起来难得多。

带着特定目标进入互动，然后努力达成那个目标，接着是下一个目标，这实际上容易得多。因为我们觉得：“好吧，我明白了，我知道我在哪，我知道问题是什么。”但对于共振，**没有目标**。关闭那种目标导向的思维，真正从结果中解脱出来，真正享受事物更情感化的一面，要困难得多。在共振中，即使谈论一个你毫无兴趣的话题，也能产生共鸣。

这显示出你是一个内心真正安定的人。内心不安的人会思虑过多，无法共振。如果你感到拘束，你将难以共振；如果你觉得对方高不可攀，你也难以共振。所以，如果你能真正地、以真实的方式唤起并进入共振状态，这价值极高。它表明：“哇，这个人真的很安定。他们真的很活在当下，没有微观管理，没有隐藏议程，没有伪装。这太棒了。”而且他们从社交层面理解，在对话的这个阶段，这样做是合理的。

能够共振是极高价值的体现，同时也意味着你不会对某些事情过于耿耿于怀或执着。如果你能自如地进入并保持共振，那就是黄金状态。

## 如何实践共振：循环与爆发

事物的垂直面，我称之为 **“一切的小爆发”**。

你应该想象一个无尽的循环，在这个循环中，你时而风趣，时而严肃；时而大声说话，时而轻声细语。你就像是在重复播放各种小话题、小氛围或小情绪的短暂爆发。

这与“下一步是什么”的思维不同。一旦你吸引成功，很棒，那就**多做吸引成功时在做的事**，而不是总想着“好吧，现在下一步是什么？”。更多的是**保持相同**——不是指完全相同的某个具体行为，而是在**氛围和互动的整体“氛围胶囊”** 上保持相同。只是让这种状态持续下去，并让各种小爆发不断重复。

**代码描述共振循环：**
```python
while 处于共振阶段:
    情绪或行为 = 随机选择([风趣， 严肃， 大声， 轻声， 话题A， 话题B...])
    执行(情绪或行为， 持续时间=“短暂爆发”)
    保持(整体氛围胶囊)
```

这就是关于共振和维持对话、闲聊、了解他人、体验他人、为交谈而交谈的基本原理。不要过度思考，不要总想着“在价值方面我处于什么位置”，而是“我此刻就在这里，与某人连接着”。

当这一点真正做好时，你会进入一种特定的状态，时间飞逝。真的毫不费力，就像进入一种半恍惚的、梦境般的状态，你只是在流动。如果感觉很沉重，你仍然思虑过多，你将无法做到。这是无法伪装的，这也正是其如此有价值的原因。我们称之为 **“诚实信号”** ，即那些很难伪装的东西。真实的共振，极难伪装。
