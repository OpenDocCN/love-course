# 朱利安《高频沟通》：2 情感技能：2.3 锻炼真我_note

在本节课中，我们将学习如何通过“放手”来直接锻炼你的“存在”状态，并探讨如何重新连接真实的自我，而非停留在表面的伪装上。我们将理解为何大多数社交建议只是在教你“伪装”，并学习将关注点从“现在该怎么办”转向“为什么会这样”的思维转变。

---

上一节我们介绍了情感技能的重要性，本节中我们来看看如何通过“放手”来直接作用于你的“存在”状态。

**放手**是你直接作用于“存在”状态的方式。你需要从“存在”开始，记住这一点，并让它逐渐渗透到你的“行动”和“习惯”中去。如何作用于“存在”状态？答案是：放手、放手、再放手。

一个可以附加的外部方法是，让自己置身于那些已经拥有你所期望的“存在”状态的人群之中。我对此非常谦虚，因为我观察多年发现，如果人们花大量时间与我相处，即使他们原本的“存在”状态并不自信，仅仅通过接近，我的“存在”状态也会影响到他们的“存在”状态。这虽然不触及根本原因，但你可以将其与“放手”结合起来使用。

这里再次强调，你需要认识到，其中80%的内容仅仅是**不给自己设障**。有趣的是，如果我们审视自己的生活，会发现我们能够轻松地与那些我们认为与自己水平相当或不如自己的人社交。当你和某人交谈时，你并不太在意，一切自然流动。但面对那些你真正想交谈、想建立联系的人时，你却会僵住。你会想，为什么我不能在面对我想交谈的人时，也保持那种状态？

**放手 = 如何在社交技能上表现得自然**。任何告诉你锻炼情感技能不是通过“放手”，或者主要靠分析技巧的人，他们本质上是在教你如何行骗。他们是在教你如何成为一个“社交骗子”。可悲的是，市面上大多数建议都是如此。例如，“这里有三个让你看起来更自信的技巧”，只是教你做某件事，完全不涉及自我修炼。大多数建议只是在教你如何伪装，如何打造一个完美的外表，如何表现得比你实际更好，如何获得你本不该得到的结果，因为你并未付出相应的努力。

这一切都归结于那句老派的自我提升格言：**要得到你想要的，你必须配得上你想要的**。

---

现在，我们来探讨具体的工作方法，因为“放手”听起来可能很模糊。首先要理解的问题是“做你自己”这句建议的局限性。

你常听到这句话，对吧？最好的建议是什么？做你自己。这实际上是最好的建议。但问题在于：**大多数人并不知道自己是谁**。他们不知道如何做自己，他们没有接触到那个真实的自我。正如我之前谈到的，假设你出生时，感觉是开放、舒展的。随后发生了一些事，你可能会想：“哦，也许我该收敛一点，我应该更害羞一点。” 因为某件事发生了，你会想：“好吧，我保持安静。” 你打造了一个外表，压抑了自己，并且长期以来一直如此生活，以至于你让自己相信，那就是你。嘿，我想这就是“假我”。如果你伪装得足够久，你会自我洗脑，认为那个“假我”就是真实的自己。

所以，当人们听到“做你自己”的建议时，他们会想：“哦，做那个假的我，好的。” 不，“做你自己”意味着做**真实的自己**。重新连接那个真实的自我，并成为他。

这也是为什么这个建议会被误解并用作借口。例如，“就做我自己好了，那我就不采取任何行动了。” 尤其像我以前那样，非常逻辑、非常分析性。如果做性格测试，我的迈尔斯-布里格斯类型是INTJ。我天生更内向，不是那种能轻易光芒四射的人。你可以深入争论外向与内向，但其中大部分实际上源于我的后天 conditioning。如果有人告诉我“嘿，做你自己”，那我就会是内向、逻辑且安静的。但事实是：不，不，不。万一那不是我真正的样子呢？

这就是你必须问自己的问题：**今天的你，是你真正的样子吗？这是你与生俱来的样子吗？这是你来到这个世界时的样子吗？** 如果你感到一些摩擦，或者内心有一部分觉得“作为这个版本的自己，感觉有点不对劲”，那么就要意识到：这实际上是我在伪装。你认为的“做自己”并不是真正的“做自己”。这一点至关重要。

那么，当谈到“假装直到你成功”这句话时，它实际上是：**表现得真实，直到你记起——你现在正在伪装**。顺便说一句，如果你能领悟这一点，它不仅会改善你在提升社交技能和与人连接方面的做法，也会在自我提升的各个方面带来巨大帮助。

整个自我提升领域的一个核心问题是，人们认为此时此刻、在开始自我提升之前的自己，就是“真实的自己”。所以，要认识到这一点。这就是为什么“放手”意味着放下一切维持着那个“假我”存活的东西。

---

我们的方法是从“情境自信”转向“核心自信”。同样，审视你的生活，你真正依赖什么来获得情境自信？这里有一个问题可以问问自己：

假设一个神奇的精灵出现在你身边，说：“嘿，我可以满足你一个愿望。我可以给你金钱、外貌、生活方式、朋友、名声，任何你想要的。😊 你会许什么愿，来最大程度地影响你日常的自信感？” 它可能无法解决所有问题，但假设你能得到一个愿望，并且挑战是尽可能大地提升你的自信百分比，那会是什么？对一些人来说是金钱，对另一些人来说是外貌，也许是朋友、名声、更好的工作、激情等等。审视这一点，审视真正让你陷入头脑、让你僵住的是什么。

正如我们讨论的，这是一个“认识”而非“成为”的过程。一个关键的思维转变是：**将你问自己的“现在该怎么办”改为“为什么”**。

例如，这是情感技能，不是分析技能。但在情感技能方面，假设你很害羞、很拘谨、陷入头脑。大多数人会想：“哦，我很害羞，我很拘谨，我不知道该说什么。现在该怎么办？” 这很糟糕。“现在该怎么办”是一种怎样的思路？它是这种思路：我在这里（状态不好），现在该怎么办？现在该怎么办？现在该怎么办？把它改成“为什么”。哦，我很拘谨。**为什么**我会拘谨？这将你的焦点从“现在该怎么办”（这种为了补偿而进行的构建）转移到“是什么在告诉我我状态不好？是什么让我此刻感到拘谨？” 这会将你引向内省。

情感技能的很大一部分就是内省，了解你自己。问自己：我是如何变成今天这个虚假的外表的？是什么让它保持存活？

回到老派的自我提升比喻。很多人都用这个，不确定托尼·罗宾斯是不是第一个提出的。想象一张桌子。有桌面，也有桌腿。如果你想“打破”这张桌子，人们通常会尝试敲打桌面。你可以用尽全力敲打，但可能没什么效果。最简单的方法是什么？**拆掉桌腿**。这就是我们在这里要做的事。“假我”是桌面。那么，是哪些参照物、哪些经历、哪些“桌腿”在支撑它存活？这就是你问“为什么”的原因。然后你会想：“哦，是那个。” 于是，它便逐渐消逝。

---

本节课中，我们一起学习了通过“放手”来直接锻炼“存在”状态的核心方法。我们认识到“做你自己”的真正含义是连接并展现真实的自我，而非后天形成的伪装。我们剖析了为何许多社交技巧只是“伪装”，并学习了将思维从寻求解决方案的“现在该怎么办”转向探究根源的“为什么”。关键在于内省，识别并拆除支撑“假我”存活的那些“桌腿”，从而让真实的自我自然浮现。这是一个认识自我，而非强行改变的过程。
