# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

变量可以理解为一个贴有标签的盒子，用于在计算机内存中存储信息。你给这个盒子起一个名字（变量名），然后把数据（值）放进去。之后，你就可以通过这个名字来使用盒子里的数据。

创建一个变量非常简单，使用等号 `=` 进行赋值即可。等号左边是变量名，右边是你想存储的值。

```python
my_name = "小鹿"
my_age = 10
```

上一节我们介绍了变量的比喻，本节中我们来看看如何具体地命名和使用变量。

### 变量的命名规则

以下是命名变量时需要遵守的几个核心规则：

*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如 `age` 和 `Age` 是两个不同的变量）。
*   避免使用Python的保留字（如 `if`、`for`、`while` 等）作为变量名。

## 理解数据类型 🔤

数据并非只有一种形式。Python将数据分为不同的类型，每种类型决定了数据可以做什么操作。最常见的基本数据类型有三种。

### 1. 字符串（String）

字符串用于表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。

```python
greeting = “你好，世界！”
```

### 2. 整数（Integer）和浮点数（Float）

数字主要分为两种：整数（没有小数部分）和浮点数（包含小数部分）。

```python
age = 25        # 这是一个整数（int）
price = 19.99   # 这是一个浮点数（float）
```

### 3. 布尔值（Boolean）

布尔值只有两种可能：`True`（真）或 `False`（假）。它常用于逻辑判断。

```python
is_student = True
has_graduated = False
```

## 如何查看数据类型？ 🔍

在编程时，你有时需要确认一个变量的类型。可以使用 `type()` 函数来实现。

```python
print(type(“小鹿”))  # 输出：<class ‘str’>
print(type(100))     # 输出：<class ‘int’>
print(type(3.14))    # 输出：<class ‘float’>
print(type(True))    # 输出：<class ‘bool’>
```

## 一个简单的实践示例 💻

现在，让我们把以上知识结合起来，写一段简单的代码。

```python
# 定义几个变量
pet_name = “旺财”
pet_age = 3
is_dog = True

# 打印这些变量及其类型
print(“宠物的名字是：”, pet_name)
print(“它的类型是：”, type(pet_name))

print(“宠物的年龄是：”, pet_age)
print(“它的类型是：”, type(pet_age))

print(“它是一只狗吗？”, is_dog)
print(“它的类型是：”, type(is_dog))
```

运行这段代码，你将在屏幕上看到变量存储的值以及它们各自的数据类型。
