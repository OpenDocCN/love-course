## 4. 催命型

如果说“没有底线型”是主动冒犯，那么“催命型”则是通过被动施压让人反感。这种类型表现为连续、密集且低价值的追问，像在查户口或索取关注。

例如，连续发送：
*   “在吗？”
*   “在干吗？”
*   “吃了吗？”
*   “去哪了？”
*   “怎么不回我了？”

这种连环追问本质上是**单向索取对方的价值**（时间、注意力、生活动态），而没有提供任何有趣的交流内容。在女生对你好感度不足时，这会给她带来极大的压迫感，结果往往是“聊着聊着就没了”。

**核心概念**：价值展示 vs. 刻意炫耀。真正的吸引力来源于内在特质和自然流露的价值。
