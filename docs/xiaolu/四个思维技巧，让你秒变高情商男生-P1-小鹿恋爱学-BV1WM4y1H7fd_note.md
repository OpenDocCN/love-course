# 高情商思维入门课（一）：四个核心思维 🧠

在本节课中，我们将要学习四个核心的思维技巧。这些技巧旨在帮助你提升在人际交往，特别是与异性相处时的情商表现。掌握这些思维，能让你更清晰地认识自己，更有效地与人沟通，从而在社交中脱颖而出。

---

## 第一思维：关注自己 👤

上一节我们介绍了课程概述，本节中我们来看看第一个核心思维：关注自己。

很多男生在追求女生时，总喜欢猜测女生的想法。一旦开始猜测，你就已经处于被动。你越猜测，情绪就越容易波动，越容易“上头”。

我们与女生相处的初衷是为了获得开心和快乐。但如果我们将自己所有的快乐都建立在对方身上，仿佛只有她才能带来快乐，那就陷入了困境。

**核心公式**：**你的吸引力 ≈ 你自身的快乐与自信状态**

一个女生最初被你吸引，正是因为你与她建立关系前所展现的快乐、自信和潜在价值。如果你在相处后失去了这些特质，变得唯唯诺诺，只会围着她转，那么你在她眼中的吸引力就会逐渐消失。

**关键在于**：保持自身的独立性和积极状态，而不是将情绪主导权完全交给对方。

---

## 第二思维：筛选思维 ⚖️

理解了关注自身的重要性后，我们进入第二个思维：筛选思维。这关乎你对待关系的根本态度。

追求女生并非依靠单纯的真心和付出就一定能获得回报。这就像求职：

**代码示例（类比）**：
```
if (不符合岗位要求 && 无工作经验) {
    print("虽然我真心想要这份工作，请给我机会。");
} else {
    print("我被录取了。");
}
// 结果很大概率是：请求被拒绝。
```

什么是弱者思维？弱者思维是在感情中无法控制情绪，遇到喜欢的女生立刻“上头”。这本质上是“见色起意”，却自我感动为“深情”。得不到便焦虑，进而进行一系列廉价的讨好，试图证明自己能让对方幸福，却从不思考对方是否需要你定义的“幸福”。

以下是弱者思维的典型内心独白列表：

*   她为什么不回消息？
*   昨天聊得好好的，今天怎么冷淡了？
*   她被别人追走了怎么办？我得抓紧。
*   我好烦躁，胸口发堵。
*   她不见我，我想去她家楼下找她。
*   我看不到她，我就难过。

什么是强者思维？强者思维只关注自己的核心需求，把时间和精力投资在提升自我上。它意味着坚守原则和底线。

**核心公式**：**你的原则 = 不可妥协的边界**

一旦他人触碰你的底线，无论对方是谁，都应予以回应，而非妥协。你需要明确自己的标准，告诉对方你能接受什么，不能接受什么。对方能满足，就继续相处；不能满足，就果断换人。

永远不要试图说教或改变一个人。就像我真诚建议你“少玩手机、别熬夜、坚持健身”，我的真诚或许能短暂影响你的想法，却很难改变你对这些习惯的情绪依赖。你在感情中的“真诚”劝说，大概率也只能起到同样有限的效果。

**记住**：成年人的择偶，是**只筛选，不改变**。

## 第三思维：共情思维 🤝

学会了以筛选思维建立框架，接下来我们需要填充沟通的技巧，这就是共情思维。

学会共情，关键在于不要轻易对女生提建议。无论对方是否询问，都不要急于告诉对方“你应该怎么做”。

举个例子：女生因为工作不开心。

直男的做法是立刻问：“是什么问题？我来帮你解决。” 他们的思维聚焦于“解决问题”。

但女生向你倾诉，真的是希望你解决问题吗？如果想解决问题，她可以找同事或上司。她找你，是希望在你这里获得情感上的安慰和理解，倾诉她的委屈。

请思考：当你向非本行业的朋友抱怨工作的烦心事时，你是希望他提供解决方案，还是希望发泄情绪，甚至希望他能认同你的感受，一起吐槽？如果对方上来就质问：“这情况难道不是你自己有问题吗？” 你下次还会找他倾诉吗？

同理，女生需要的是**共情**与**聆听**。只有当你与她站在同一情感战线，理解她的感受，后续的发展才有可能。

**核心要点**：先处理情绪，再处理事情。倾听和理解远比武断的建议更重要。

## 第四思维：以退为进 🎣

掌握了共情这一沟通利器后，我们来看最后一个策略性思维：以退为进。这源自《孙子兵法》，意指当持续进攻效果不佳时，适当的撤退反而能赢得最终胜利。此方法在感情互动中同样适用。

我发现很多男生在和女生聊天时完全无法控制节奏。对方一回复，就恨不得聊通宵。还有的男生把“早安”“晚安”当成每日打卡任务，不管对方反应如何，都机械执行。这类行为在女生看来千篇一律，非常无趣，无法带来任何新鲜感。

因此，我们要学会“刹车”，懂得**以退为进**，从而一击制胜。

例如，在女生聊得兴致正高时，你突然停止回复，去忙自己的工作。这时，女生的心里就会产生疑问：“他去干嘛了？怎么突然消失了？” 她的脑海里会更多地浮现你的身影。

一旦女生产生了这种好奇和期待的情绪，她就会更期待与你下一次的对话，甚至可能主动来询问你为何突然离开。

**核心策略**：在适当的时候主动结束对话，制造轻微的“悬念”和“需求感”。

适时、适量地运用以退为进，不仅能让你牢牢掌握聊天的主动权，也能让下一次的聊天开场变得更加自然和容易。

## 课程总结 📝

本节课中我们一起学习了提升情商的四个核心思维：

1.  **关注自己**：保持自身快乐与自信的状态，这是吸引力的根源。
2.  **筛选思维**：建立原则与底线，只做选择，而非试图改变他人。
3.  **共情思维**：沟通时优先倾听和理解情绪，而非急于提供解决方案。
4.  **以退为进**：掌握互动节奏，适时撤退以创造更大的前进空间。

最后需要铭记的是：我们想方设法提高情商、努力成长，应该是为了**成为更好的自己**，而不是因为被某人伤害，或为了讨好某个人才被迫改变。只有秉持这个初衷，你在人际交往中才会真正无往不利。

这个世界上没有那么多天赋异禀，优秀的人总是在努力地翻山越岭。**提升自己，最为重要**。
