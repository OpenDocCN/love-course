## 关系不平等的根源：无意识的“买卖关系”

上一节我们点明了关系不平等的现象，本节中我们来看看其根本原因。你和女性关系不平等，核心在于你们无意中促成了**买卖关系**。这种关系是你自己一手造成的。

## “供养者”模式的陷阱与后果

认识到根源后，我们来看看这种错误模式的具体表现与危害。许多男性从一开始方向就错了，试图通过花钱购买感情，通过付出来换取对方。这形成了“你出钱，他出力”的可怕关系。更可悲的是，在当前社会，这种关系对部分男性而言竟被视为“不错”。

理解了陷阱的模式，我们进一步分析其动态发展过程。这就是为什么关系初期请客吃饭时对方态度尚可，但发展到买包、转账后，对方反而态度冷淡。因为**人类的欲望是无穷的**。你的持续、无界限的付出，实际上在不断**养大对方的胃口**。

