上一节我们概述了课程目标，本节中我们来看看第一个核心定律：**吸引而非追求**。

许多男性在相处时，错误地将大部分精力用于直接“追求”对方，例如一味讨好与付出。正确的方法是，将超过70%的精力用于**建设自身生活和提供价值**。

*   积极的生活态度与丰富的兴趣爱好。
*   清晰的三观与独特的生活方式。
