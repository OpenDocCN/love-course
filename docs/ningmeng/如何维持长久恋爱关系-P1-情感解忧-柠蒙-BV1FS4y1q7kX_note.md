# 恋爱关系维护教程 第1课：理解激情消退与长期关系本质 💑

在本节课中，我们将要学习长期恋爱关系中的一个核心规律：激情的自然消退。我们将探讨如何正确看待这一过程，并学习维持健康关系的两个基本方法。

## 激情消退是常态 🔄

我们要知道激情是会消退的，平淡才是常态。

对于这个问题比较困扰的人，可能是没有谈过太长久的恋爱，或者说虽然谈得比较长，但是也谈得比较痛苦。

其实在恋爱中，热恋期一般也就是前面的三五个月，一旦过了这段时间，激情就开始消退，慢慢地就趋于平淡。

**长期关系公式：热恋期 → 激情消退期 → 平稳期**

这是长期关系的必经之路。如果你认为恋爱每天都是充满激情的，那么你可能只适合谈一些几个月的恋爱。

很多情侣之所以分手，就是因为他们其中至少有一方，接受不了关系变平淡，从而去纠缠对方，想让对方像之前一样热情，做出一些伤害感情的行为。

## 应对之道：接受现实与沟通 🤝

上一节我们介绍了激情消退是长期关系的自然规律，本节中我们来看看如何正确应对这一变化。

应对的最好办法就是接受现实。我们不是十几岁的小孩子了，你不要一直活在泡沫剧里，活在小说里面。我们要明白热情的消退，它是必然的结果。

### 不同步时的沟通策略

以下是针对双方脱离热恋期时间不同步的沟通建议：

*   **如果你是先脱离热恋期的一方**：你一定要和对方沟通，你一定要表达你并不是不爱他，而是你要把更多的精力放在生活上面。因为两个人在一起，是为了生活变得更好，而不是沉溺于当前。
*   **如果你是后脱离热恋期的一方**：那么你就要给予对方最大的理解。他不是不爱你，他只不过是比你更早地脱离了热恋期而已。

## 在平淡期中审视关系 🔍

当然你们也可以在激情消退的时候，看清楚这段关系。你们去看一看这段关系到底是不是你想要的，看一看对方的缺点你能不能接受，你是否因为对方的存在而变得更好。

**关系健康度自检代码：**
```python
if (关系是你想要的 == True) and (能接受对方缺点 == True) and (因对方变得更好 == True):
    print("对方是合适的人")
else:
    print("需要重新审视关系")
```

假如你们的答案相互都是“是”，那么说明对方是个合适的人。人对的，长期关系里面，是不需要太多的相处技巧的。

## 维持长期关系的两个核心要点 ⚙️

当确认彼此是对的人之后，维持关系就变得简单而纯粹。以下是两个核心的行动要点。

### 1. 保持开放沟通

有什么想法千万不要憋在心里。对方又不会读心术，你在想什么他怎么知道呢。如果你总是憋在心里憋久了，那就是“退一步越想越气，忍一时越想越亏”。有些情绪你是没法独自消化的。

如果你们不保持沟通，很容易就会因为一些鸡毛蒜皮的事情而吵起来。因此当你心里有不满的时候，一定要进行沟通。

**沟通原则：**
*   这个沟通不是让你歇斯底里的抱怨。
*   沟通之前尽量保持冷静，这点非常重要，否则只会火上浇油。
*   尽量冷静下来，先去了解对方的想法，再来表达自己的想法。

你会发现很多时候，对方很可能会和你有相同的想法。每次矛盾的解决都是关系的升华，你会发现你们的关系反而会越来越稳固。

### 2. 主动创造激情

激情其实还是可以有的。当然我刚才说了，热恋期过后激情就开始消退，趋于平淡。你们现在可能也没有之前那样的冲动去玩浪漫了，但是激情可以迟到却不能缺席。

你们现在的关系肯定比之前更实在，更接地气，但是该有的激情还是要有的。在长期关系当中，偶尔来点小浪漫，可以让你们重新体验到热恋的感觉，让你们的感情更加有质感。

以下是创造浪漫的灵感：

> 就比如谁说送礼物就一定要是节日，或者认识的周年纪念日呢。我上个月就在电影院送了我女朋友一只萝卜丁（口红）。她当时有点惊讶地问我：“今天是什么节日吗？”我说：“去年的今天我在这里吃了你的口红，今天我要还给你。”这就是回忆杀。然后她就又主动地让我把她的口红给吃了。这感觉、这激情、这浪漫不就来了嘛。
