在本节课中，我们将通过一个真实的故事，探讨在感情中失去自我、过度付出的心态是如何形成的，并学习建立健康情感关系的基础——**爱自己**。

## 个人经历：过度付出的代价

我曾深刻理解那种喜欢一个人就想拼命对她好、愿意做任何事的心情。我19岁时喜欢上隔壁班的班花，为此连续两个多月每天只吃一餐，饿到低血糖，只为攒钱买礼物。

终于，在那个每月生活费只有500元的年代，我攒够了1000元买了一条项链。现在回想，我自己都有些感动。那是我最开心的时候，也是最难过的时候，因为她从未答应我，却又一直让我觉得还有希望。

## “舔狗”的典型处境

有些人就是这样，明明不喜欢你，却又不肯放过你。以下是这种处境的一些表现：

*   礼物照收，但收完便离开。
*   最初会说谢谢，后来连谢谢都懒得说。

有一次，我打了两个月暑假工，买了两张五月天演唱会门票，想约她一起去。我忘了她当时具体说了什么，总之我被她说服，眼睁睁看着她拿走了两张票，和另一个男生去了演唱会。

回到寝室，一位室友正在播放陈奕迅的《红玫瑰》。歌词“得不到的永远在骚动，被偏爱的都有恃无恐”在那刻显得格外扎心。我用了整整一个学期才走出阴影。我至今记得那位室友对我说的话：**“如果你只爱别人不爱自己，你不做舔狗谁做舔狗？”**

## 从迷失到觉醒的弯路

后来，我活成了自己曾经最讨厌的样子。有几年我跑偏了，成了一个“渣男”。直到2019年，我才真正走回正道。这些年，从“舔狗”到“渣男”再到学会真正去爱，让我深刻体会到爱一个人最正确的方式。

上一节我们看到了过度付出的后果，本节中我们来看看什么是健康关系的基础。

## 爱的正确方式：坚持自我

爱一个人最正确的方式，不是做一条只会围着对方转的哈巴狗，而是**坚持好自己**。我们可能已经习惯了因为喜欢而无底线地弯腰、委屈自己。但时间久了，对方会习惯你的低姿态，习惯俯视你。是你自己让对方觉得你不重要。

人应该把自己放在首位。以下是坚持自我的重要性：

*   只有坚持好自己，你才能给对方一个优质的爱人。
*   只有坚持好自己，你才能给你的后代一个合格的父母。
*   只有坚持好自己，你才能给你的父母一个合格的儿女。

核心逻辑可以归结为一个简单公式：**好好爱自己 → 才能好好爱别人 → 别人才会来爱你**。

## 自我建设是吸引力的根本

我常在私信中看到类似情况：为了喜欢的女生放弃985高校去普通院校；为了喜欢的女生放弃大好工作和前途，跑去异地像没头苍蝇一样。最后受挫了，又来问我“为什么”、“怎么办”。

我很想骂他们，但想到自己当年也好不到哪去，只能表示同情。你们要明白，**只有当你自己闪闪发光时，才是你最有底气的时候**。

如果因为喜欢一个人，你就放弃**自我建设**与**自我发展**，那么你会变得黯淡无光。结果就是：

*   不喜欢你的人，会更加不喜欢你。
*   喜欢你的人，也会慢慢远离你。

感情虽说是双向奔赴，但请想一想，如果你放弃了自我建设，别人凭什么要向你奔赴？所以归根结底，依靠的不是对方，而是你自己。

## 总结与行动指南

本节课中，我们一起学习了“舔狗”心态的成因与危害。通过我的个人经历，我们看到了过度付出如何导致自我价值感丧失，并最终可能失去对方的尊重与感情。

关键在于觉醒与改变。吃过一次亏，就该醒一醒。从现在开始，请将生活的重心放回**自我成长**与**自我关爱**上。记住，健康的爱始于健康的你。当你成为一个完整、发光的个体时，你才能吸引并维系一段平等、健康的情感关系。
