# 情感解忧-柠蒙 课程 P1：追女生多久可以牵手？⏳

在本节课中，我们将要学习一个常见的情感困惑：追求女生需要多长时间才能有亲密进展（例如牵手）？我们将分析这个问题的本质，并提供一些实用的思路来提高效率。

## 问题的本质：时间并非关键

上一节我们介绍了课程主题，本节中我们来看看这个问题的核心。追求女生多久能牵手或在一起，其关键因素在于人本身，而不在于追求的时间长短或次数多少。

这让我想起了一个远古时代的问题：女生要多久才会喜欢上追她的男生。你们追一个妹子，追得连正常人的思考都没有了吗？你咋不去问你的老板，我入职多久可以当领导啊。这些问题的关键看人，不看时间和次数。

甚至有时候都不看你追她有多努力。就比如你们可能会遇到这种情况：你追了一个女生辛辛苦苦追了一年都没追到，结果她和一个认识才三天的男的在一起了。是不是很崩溃啊。喜欢是情绪反应而不是逻辑思考。

不可以用时间来衡量。就比如她可能只需要一天时间就喜欢上我，但也可能一年半载都对你提不起兴趣。如果单从细腻的角度来说确实很崩溃。但我还是那句话，这是看人不看时间次数。不同的人去追她难度系数是不一样的。

## 初期好感度决定进程

理解了问题的本质后，我们来看看具体情境。如果女生在初期对你的好感度就比较高，那么你就会发现，女生所谓的考验所谓的测试基本没有。她对你的反馈贼好。她对你反馈好，你可能第一次约会就牵手，第二次你们就在一起了。整个过程相当的顺利。

反之，那些追了三五七年都不想的哥们，就是最典型的反面教材。

## 如何提高效率？

那么说了这么多，怎么提高效率缩短这个过程呢？我说点最初级又实用的东西，主要有两点。

以下是两个核心的改进方向：

1.  **提升自己的外在价值，给女生留个好印象。**
    我之前就说过。你与其每天给她早晚安，每天送早餐送午餐，甚至攒钱给她买个包，还真不如给自己做个好看的发型，买一身帅气的衣服。然后找一个会拍照的朋友，给你拍几张照片发在朋友圈。她对你的印象好，对你的反馈就会好。这比什么都好。我就很烦，有些哥们他可以省吃俭用几个月，给女生买个一两万的包，眼睛都不眨一下。你让他花150去做个发型，花五六百买身衣服就要了他老命了。对于这样的人我只想说，活该。

2.  **刺激女生的情绪。**
    女生大多都是比较感性的。相对于那些平铺直叙的相处方式，她们会更加在意那些可以让她们产生恋爱的感觉的特质或者行为。你可能会说了，我本身就是根木头，我不会啊。没关系，技术不行，环境来凑。吊桥效应听说过吧？照样让女生对你产生心动的感觉。虽然说这种感觉是比较虚幻的，但也比你只知道傻傻的等要好得多。

以下是利用环境刺激情绪的一个具体方法：

*   **选择刺激性约会场景：** 比如约会的时候，你可以选择一些像鬼屋、密室逃脱之类的比较刺激的地方。但是注意，你自己最好先去玩一玩，适应一下。别到时候你们都是第一次去，你的表现比女生还差，那就尴尬了。这些场景不但利于肢体接触，还能刺激女生对你的依赖心理。而危机时刻你又能带领她保护她。这种时候女生很可能就会认为，这种心跳的感觉是你带给她的。

当然我说实话，这种方法真的很初级。女生经历的越多，效果就越弱。你们在菜鸟的时候还可以顶一顶，到了后面就一定要摆脱对环境的依赖。

## 设定止损期限

最后，给自己设置一个停止期限。比如可以是三个月。在这三个月的时间内，你已经很努力了。如果超过了这个时间，还是没有进展，她还是没有态度上的表示，那就算了吧。

你记住，除非对方有什么心理上的疾病，比如说抑郁症，否则无论男女，当自己喜欢的人出现之后。他可能会矜持一下，或者想把战线拉长一下，以此展现自己的真诚或者价值。这些都是正常的。但拉扯个一两回也就算了，不会无限制地把你拖着。因为自己心里也是，恨不得马上和对方在一起，生怕对方被别人给截胡了。这种心理你有她也有。
