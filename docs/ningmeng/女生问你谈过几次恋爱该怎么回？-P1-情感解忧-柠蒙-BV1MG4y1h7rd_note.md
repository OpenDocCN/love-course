首先恭喜你。女生提出这个问题，说明你已经进入她的考虑范围。回答得当，你们的关系将实现质的飞跃，发展恋爱关系指日可待。

**核心概念**：只有感性层面的互动，才能深入女生内心，唤起恋爱感觉。

因此，我们不应直接给出数字。以下是推荐的回应步骤与思路。

上一节我们指出了理性回答的弊端，本节中我们来看看具体的感性回应方法。

如果能彻底忘记最好。

这样的回复通常能激发对方的好奇心。女生很可能追问：“为什么？”

随后，女生可能回复：“想得美。”事实上，聊到这里，女生通常已经心动了。
