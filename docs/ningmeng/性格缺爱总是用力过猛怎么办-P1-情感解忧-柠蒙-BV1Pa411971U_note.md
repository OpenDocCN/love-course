# 性格缺爱总是用力过猛怎么办-P1-情感解忧-柠蒙-BV1Pa411971U_note

在本节课中，我们将要学习性格缺爱在感情中的表现、其深层原因，以及如何建立更健康的情感模式。缺爱常与原生家庭经历有关。有些人一生被童年治愈，有些人一生都在治愈童年。缺爱且不自信的人在感情中容易“用力过猛”。

上一节我们提到了缺爱的普遍背景，本节中我们来看看它在感情中的具体表现。

以下是缺爱者在感情中常见的四种表现：

1.  **控制不了情绪，非常敏感，经常情绪失控，感性压制理性。**
2.  **经常有不切实际的想法，生活比较空虚。** 想成为社交达人，但不敢行动。
3.  **非常希望受到别人认同。** 一旦被否定，情绪波动就会非常大。
4.  **一旦喜欢上一个人就会非常痴迷，非常想要依赖对方。**

我们要记住一个核心公式：**缺乏爱 ≠ 缺乏自我**。可能很多人建议“让自己忙起来”，但这并非根本解决之道。忙碌过后，夜深人静时，空虚感依然会袭来。

了解了具体表现后，我们进一步探讨其背后的行为逻辑。

缺爱不自信可能源于从小获得的爱较少。因此，一旦喜欢一个人，就容易无底线地付出和讨好，因为害怕失去而患得患失。但必须明白一个情感规律：**在感情中，你越是害怕失去，就越会失去**。我曾有同样经历。

在最初经历恋爱时，我会付出全部，并期望对方同样回报。感情中极度情绪化，通过“作”和患得患失来让对方证明她有多爱我。这听起来是否很像一些女孩的行为？我以前也觉得自己完全没错。

我就希望她对我就像我对她一样。但你必须明白，**对方是一个独立的人，你应该去尊重她**。付出更应是一种自觉，而不应成为逼迫对方的工具。

我们这类人在追求或恋爱时，常常自己先热情得像火山爆发，对方可能才刚刚预热，就被我们的“爆炸”吓跑了。换个角度思考：如果刚认识两天，就有人对你爱得死去活来，你刚产生的一点好感也可能立刻被磨灭，甚至转为厌恶。即便因为感动在一起，关系也难以长久。

**因为这段关系从一开始就不健康。** 感情中的跪舔行为和情绪失控，确实非常令人厌恶。

我曾与女友在一起时，也非常在意她是否秒回信息。我认为“我说我爱她，所以她也必须秒回信息爱我”，否则就会非常不高兴，并做了许多傻事。我以为这是爱的表现，她应该接受。

然而，**爱不是情绪垃圾桶，不是枷锁，不是束缚**。

**爱是自由，是尊重，是理解，是彼此的支柱和依靠，需要共同经营与进步。**

所以，请清醒一点。不要自以为是地认为那就是爱。请用以下问题审视自己的感情：

*   你觉得失去自我、爱一个人的爱健康吗？
*   你觉得需要对方竭力去证明他有多爱你的爱健康吗？

**自信一点，自爱一点。爱人先爱己。** 我们都应该用正确的方式去爱别人，而不是用爱的名义去绑架、去互相消耗。

其实，无论男女，都不存在所谓的“纯粹的讨好型人格”，那只是“舔狗”行为。

答应我：

**我们要舔就舔自己。**
