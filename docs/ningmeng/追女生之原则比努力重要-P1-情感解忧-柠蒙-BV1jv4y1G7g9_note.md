# 追女生之原则比努力重要 - P1 🎯

在本节课中，我们将要学习一个在追求女生过程中至关重要的核心观念：选择比盲目努力更重要。我们将探讨感觉的主观性，并学习如何通过筛选策略，高效地找到真正适合你的那个人。

## 感觉的主观性：为何努力并非万能

上一节我们提到了课程的主题，本节中我们来看看追求过程中最常遇到的困惑。你暗恋一个女生，费了很大劲去认识她并和她交朋友，她却说对你没感觉。你讲了一个段子，朋友们都笑了，唯独你最在意的那个女生没有笑，这让你很郁闷。

有些女生虽然不是很漂亮，但就是给你一种如沐春风的感觉。有些女生虽然很漂亮，但就是对你冷若冰霜，你和她接触时一直如履薄冰。在网络上也是一样，有的女生在你打招呼后，你们就能很愉快地聊起来。而有的女生，不管你怎么聊，她就是对你提不起兴趣。有的女生，你能和她有说有笑。而有的女生，她总是用“嗯”、“啊”、“哦”来敷衍你。

## 问题的根源：并非总是自身原因

于是，你开始寻找自身原因：是口才不好说错话了，还是学历不高见识不够，是个子太矮，还是收入太低？其实很多时候都不是这些原因。因为感情是一种很主观的感觉，就像交朋友一样。大多数人，我们只是礼貌性的点头之交。甚至还有一小部分人，你一看到她，就觉得她很讨厌。只有极少数人才会和我们一见如故，或者成为好朋友。

不管你有多么厉害的恋爱技巧，不管你是个多么厉害的高手，喜欢你的都只是一部分人。就算再大牌的明星，也会有一大堆黑粉。所以在追女生的过程中，有个很重要的技巧，那就是**学会选择，懂得放弃**。

## 核心原则：选择与筛选策略

上一节我们理解了感觉的不可控性，本节中我们来看看如何应用“选择”这一核心原则。我有一个哥们，一个多才多艺、又白又美的妹子主动来撩他，甚至主动撞到他怀里，他都没有感觉。最后，他找了一个各方面明显逊色的姑娘。他说他和这个姑娘在一起特别有感觉。你说这上哪里说理去？

恋爱的审美观毫无道理和标准可言，就是一种很主观的感受。情人眼里出西施。女人对男人的第一印象也是感觉，这个感觉可能是相貌、才华、阅历，也可能是风度。所以，追女生最明智的选择就是：你最好先明确你喜欢的是哪个群体的女生，然后多去认识这个群体内的女生，最后选择对你有好感的，排除掉那些对你不感冒的。

**核心策略公式：目标群体 → 广泛接触 → 筛选反馈 → 重点培养**

## 实践方法：从想象到行动

只选对的，不选累的，除非你想要挑战极限、磨练自己。如果你现在的好友列表里都没有几个女生，那说明你多半还停留在想象阶段，想的太多，做的太少。当你拥有足够多的资源，你随便聊不了几次，就能很轻易地将那些对你不感冒的女生剔除掉，同时重点培养和你互动效果最好的那一个。

以下是实施筛选策略的具体步骤：
1.  **扩大社交圈**：主动去认识更多你感兴趣群体内的女生。
2.  **初步互动**：进行简单、轻松的交流，观察对方的反应和态度。
3.  **评估反馈**：根据对方的回应积极性、话题参与度判断其兴趣水平。
4.  **果断筛选**：将反应冷淡、兴趣缺缺的对象及时移出重点关注列表。
5.  **集中精力**：将时间和情感投入在互动良好、对你有好感的对象上。

## 破除心理障碍：正当筛选与道德感

我发现很多兄弟的思维还停留在几十年前，他们觉得做这个事情很有罪恶感。我就想问一句：我又不是让你去欺骗人家感情，又不是让你去养鱼养备胎，也不是让你去欺骗人家，更不是让你每个都撩。你哪来这么多的罪恶感呢？

你以为女生就和你一个人接触吗？人家好友列表里，哪些人是可以不用理会，哪些人可以做朋友，哪些又是可以考虑继续发展的，人家分得可比你清楚。每一个对你反应不好的女生，都有极大可能浪费掉你大把的时间。你从这些女生身上节省出来的时间，可能都足够你从谈恋爱到结婚，到孩子学会打酱油了。

> 时间是最值钱的。
