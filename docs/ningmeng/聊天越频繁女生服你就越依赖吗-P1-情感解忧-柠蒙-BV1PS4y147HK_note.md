几乎所有没有经验的新手都会产生这样的想法：将聊天的频繁程度与对方对自己的依赖程度划上等号，认为这两者是成正比的。正是因为有了这种想法，所以他们从来来不及去思考，聊天真正的目的是什么。

他们也不知道该如何去把握聊天的频率，只知道每天进行没有方向的闲聊。昨天有位学员找到我，说前几个月在亲戚的婚礼上认识了一个女生，主动添加了联系方式，但之后聊天时，女生对他非常冷淡。

除了焦虑型追问，还有一种情况非常普遍，即当女生突然不回复时，他也不会去追加信息，而是打开之前的聊天记录，去复盘自己的聊天有没有问题。

其实这种方案是可行的，但关键在于心态和目的。有些人复盘聊天记录，不是为了总结经验教训，而是想找出自己有没有说过得罪女生的话。这两者做的事情看似一样，但性质上大不相同。

以下是几种常见的错误应对方式：

*   **强硬质问型**：直接问，超级强硬，不达目的不罢休。
*   **沉默内耗型**：倒是不问，但是躲在角落里，默默地自我怀疑。

说到底，你要学着去判断自己的行为是否有利于你们关系的发展。这样，你就不会那么纠结了。
