# 怎么控制自己的需求感-P1-情感解忧-柠蒙-BV1eW4y167V5_note

在本节课中，我们将探讨一个在人际交往，尤其是追求异性过程中常见的问题——**需求感过强**。我们将分析其表现、根源，并学习如何通过调整生活方式来有效控制它，从而建立更健康、平等的关系。

---

我大学时，班上有位同学人称“追女十八崩”。从这个外号就能看出他是个有故事的人。他连续追求了十几位女生，结果全部失败。那段时间，我甚至不敢看他的眼睛——他的眼睛泛着绿光，嘴角还流着口水。

仿佛在说“是男的也行”。后来，我们几个朋友坐下来讨论，研究他为何会连续失败十几次。最终，我们得出了一个结论：**需求感爆棚**。事实上，很多追求失败的情况，根源就在于需求感过强。需求感就像寄生虫一样寄生在我们体内。

帮助我们搞砸一次又一次。需求感可怕吗？其实一点都不可怕，因为是人就会有需求感。可怕的是你不能控制你的需求感，甚至你根本没有意识到自己需求感爆棚了。如果你有以下几种情形，那么你就应该明白，今天你“需求感”了。

以下是需求感过强的几种常见表现：

*   **过度渴望融洽**：非常渴望和对方建立融洽感，于是表现得过于“绅士”，结果反而束手束脚。
*   **身体语言暴露**：在现实中与异性交谈时，身体会不由自主地向前倾。
*   **情绪易受波动**：过于单纯，当对方调侃你时，你会当真甚至产生负面情绪。
*   **过度思考回复**：聊天时总要深思熟虑，反复琢磨对方每句话的意思，想着“我怎么回复才能吸引她”。
*   **迫切想要了解**：总是想要了解对方更多的个人信息，恨不得去偷看她的户口本。

以上你中了几条？中的越多，说明你的需求感越强。

---

当然，有人可能会说：“如果我能避免以上几条，我是不是就能控制我的需求感了呢？”我的答案是：**当然不可能**。需求感这种东西是发自你的内心，有就是有，无就是无。当你自己不能控制自己的时候，你连放个屁里面都是满满的需求感。

既然需求感发自内心，那么要控制它，就必须从根源出发。首先，我们要明白：你为什么需求感会爆棚？

因为**匮乏**，因为缺爱。当一个人获取爱、关注和情感满足的方式极其有限时，生活中一旦出现一个看起来还不错的人，就会立刻把获取爱的“魔爪”伸向对方。这一点不分男女，只不过男生的比例可能更大一些。

而对方对你这种需求感的厌恶，就好像一个多年没有联系的老同学，突然打电话对你嘘寒问暖，目的却只是为了向你借钱一样，会让你感到非常不爽。

---

所以，如果你真的想要解决需求感问题，你就需要**拓展更多能够获取爱和情感满足的方式**。如果你的生活除了工作，就是吃饭、睡觉、打游戏，是不可能找到更多获取爱的方式的。

你需要在这“三点一线”之外，去体验不一样的生活，从新的项目和体验中获取价值感与快乐。

以下是你可以尝试拓展生活的方式：

*   **培养兴趣爱好**：培养一两个能让你沉浸其中的兴趣爱好。
*   **参加社交活动**：主动参加一些线下或线上的社交活动，认识新朋友。
*   **丰富周末生活**：在周末约上一两个朋友出来玩，丢掉电脑游戏。哪怕只是去咖啡厅点一杯咖啡坐一下午，或者去图书馆待一下午也可以。
*   **尝试新体验**：偶尔去体验一下不同的娱乐活动，例如听现场音乐或参加聚会。

当你获取爱、快乐和自我价值的方式足够多时，你就不需要全部从某一个人那里获取。你的情感支撑点分散了，需求感自然就能得到控制。

反而到了这个时候，因为你的生活经历更丰富，你能给对方提供更多有趣的话题，彼此的交流气氛反而会更热烈。什么时候才是我们追求异性最好的状态？就是当你发现这个世界上，还有很多东西比“追女生”更有趣的时候；当你意识到这个世界上，还有很多精彩你未曾体验的时候。

她没回我消息？**爱回不回。**

本节课中，我们一起学习了需求感过强的表现与根源。关键在于认识到，需求感过强源于个人生活的单一与情感来源的匮乏。控制需求感并非强行压抑，而是通过**丰富自己的生活、拓展情感支撑点**来实现。当你自身的世界足够精彩和丰盈时，你便能以更从容、平等的姿态去面对任何关系。
