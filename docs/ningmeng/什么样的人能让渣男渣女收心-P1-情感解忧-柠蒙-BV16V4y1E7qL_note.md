***

上个月，我和一位小老板吃饭。他年轻时是个名副其实的浪子，仗着家里有点生意，当年声色犬马。即便结婚了也不老实，天天想着怎么去泡对面小区的大学生。但这两年，他突然就安分下来了。

他对老婆也好了，不管多忙也要回家吃晚饭，陪孩子写作业，之后再陪老婆聊聊天、看看电视。我问他为什么突然转变这么大，他笑了一下，跟我说：“玩腻了，想开了。”

我还认识一个“渣女”。她很漂亮，也很有能力。当年读书时，最高纪录同时交往三个男朋友，其中两个还互相认识。但这个家伙后来也安分了，和男朋友老老实实谈了两年，在2018年结婚了。后来我问她怎么还有这么专一的一面，她跟我说：“该结婚的年龄不能像之前那么玩了，玩到最后会把自己玩成剩女的，该稳定下来了。”

还有一个例子，是我刚毕业工作时认识的一位同事。这个人特别传统，对女朋友特别忠诚，也十分体贴。他不是没有面对过诱惑。相反，他和女朋友是异地恋，他就算搞点什么事情，女朋友也很难察觉。但他依然坚持着专一和忠诚。用他的话来讲：“我觉得对感情不忠的人都是卑鄙的，我从小到大，最瞧不起的就是这种人。”

三个例子讲完了。你们看出点什么没有？

不管是从一开始的专一忠诚，还是后来的浪子回头，很多人都会很浪漫地认为，他们做出这样的选择，是因为一个特别的人，是因为爱情。

但是我发现，实际上很多人做出转变，更多是来自于这三个例子所代表的事情：

以下是促使行为转变的几个关键因素：

1.  **经历足够多之后，发现当年的贪念和欲望，也不过如此，还不如过好当下。**
2.  **对于损失（如家庭、稳定关系）的恐惧。**
3.  **对于不稳定状态的在意，以及年龄和其他客观因素（如社会压力）的压力。**
4.  **严格的家教，以及较高的道德品质的约束。**

你觉得一个人变好真的是因为你吗？我做了大量的案例调查和分析之后，来给你讲一下最真实的情况：

**一个人如果想变好，他不想继续再坏下去，他自然能表现得很好。一个人如果想学坏，对自己又没有什么要求，你再好都没有用。**

如果真的存在“我很好，就能够让渣男渣女收心”的道理，那么请问，为什么会有那么多条件优秀的人，会被渣男渣女伤害呢？

有的人有一种“浪子情结”，可能是小说或者泡沫剧看多了，总觉得自己就是那个能够让渣男渣女回头的人。有些人特别痴迷于这种场景，觉得能够让一个人为自己而改变，还是由坏变好的那种，是一件特别有成就感的事情。

但是恕我直言，**99%的人都不具备这种能力。** 你得要有多厉害，才能去改变一个人的本性？

好，你非说自己是那1%。那么一个更有趣的问题来了：**你为什么从一开始，就去找一个好的对象？你非要把自己的能力和感情，用在“废品改造”上面呢？** 尤其是这个改造的结果还不确定。

很多人就是觉得自己一定能够改变这个渣男渣女，一定能够让他们收心，但结果却是一次又一次地被伤害。

我知道可能有人要反驳了，会说：“我以前的对象也特别渣，现在对我也很好，我们现在也好好的。”

你们只是犯了“幸存者偏差”的错误。你只是在那个时间点上，正好遇到了一个自己想要收心的“渣渣”，然后你误以为，他收心的这个行为是因为你。当然，你现在去问他，他肯定也会说，他是为了你而改变。

呵呵，动动嘴皮子，说几句善意的谎言，就能让自己的对象开心，何乐而不为？

所以，这个问题根本就不成立。真的没有几个人可以强大到，去改变一个“渣渣”的本性。他成长了二三十年的价值观，你和他交往才不到两三年，你觉得自己有那个本事，去撼动他的内心世界吗？

**他改了，仅仅是因为他想改。他以前选择做了一个“坏人”，现在他想当一个“好人”。**

而其他人，千万不要想着去当“渣渣”的最后一任。因为你们没有那个运气。
