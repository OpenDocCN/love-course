# 学会做一个聋子 - P1 - 情感解忧 🐸

在本节课中，我们将通过一个经典寓言故事，探讨如何在个人成长和情感提升的道路上，有效应对外界的质疑与噪音，专注于自己的目标。

## 青蛙爬塔的寓言 🏁

一群青蛙组织了一场攀爬比赛，终点是一座看起来非常高的铁塔塔顶。铁塔周围有很多青蛙观众在围观。说实话，其实没有哪只青蛙相信参赛者能够到达终点。

因此，观众们都在议论：“这太难了，不可能的，塔太高了。”听到这里，大部分参加比赛的青蛙开始泄气了。只有一小部分情绪高涨的青蛙，还在努力往上爬。

然而，下面的观众接着在喊：“这太难了，不可能的，塔太高了。”越来越多的青蛙累坏了，慢慢地他们都退出了比赛。最后，所有的青蛙全部都放弃了。

只有最后一只青蛙，他越爬越高，没有一点想要放弃的意思。最终，他成为唯一一只到达塔顶的胜利者。

## 成功的关键：屏蔽噪音 🧏

其他青蛙都想知道他是怎么成功的，于是大家都跑过去问他：“你哪来这么多的力气爬完全程的？”最后大家发现，他是一个聋子。

聪明的人应该已经知道我想表达什么意思了。这个故事的核心在于：**专注目标，屏蔽无关的负面干扰**。用公式可以表示为：

**成功概率 ∝ 专注度 / 外界噪音**

## 现实中的“噪音”与影响 💬

上一节我们通过寓言理解了原理，本节中我们来看看现实中常见的“噪音”形式。作为情感号，我发现很多人特别容易受到闲言碎语的影响。

你们在提升自己的过程中，可能会受到很多质疑甚至嘲讽。以下是几种典型情况：

*   **关于外在提升**：当我建议建立自信从打理外型开始时，有人反馈，他弄的发型和穿搭，被人说“骚包”，他就不敢继续了。还有人护肤、贴面膜，被人说“娘们儿唧唧的”，他也不敢继续了。
*   **关于展示自我**：他在朋友圈放了几张帅照，被人说“装逼”，他又不敢继续了。
*   **关于锻炼能力**：有人勇敢地去搭讪，想锻炼自己的心态和交际能力，结果被人说“渣男”、“海王”、“不老實”，他也不敢搭訕了。
*   **关于追求所爱**：有的人遇到自己喜欢的姑娘，勇敢地去接触，结果身边的人给他泼冷水，说他“自不量力，没钱没颜追什么女生”。

## 学会“做聋子”的心态建设 🛡️

面对上述情况，我想说：适当的时候做个“聋子”吧。你搭理他们干什么？

人性的弱点在于“不患寡而患不均”，就是见不得别人好。大家本来都是安安心心地躺平说闲话，凭什么你现在想翻身？那不能，我必须把你给按死。

所以，你在乎那些失败者的看法干什么？他们能给你发个老婆呢，还是能让你工资翻倍呢？如果都不能，那你就做自己的事情去。你过得好不好只有你自己知道。

## 远离绊脚石，坚定自我 🚶‍♂️

我很喜欢看古龙的小说，有句话我至今记忆犹新：“每个人站的地方本来都是平等的，只看你愿不愿意往上爬。你若站在旁边乘凉，看别人爬得满头大汗，等别人爬上去之后，你再喊这个世界不公平，那这才是真的不公平。”

我跟你们讲，对你们冷嘲热讽、泼冷水、见不得你上进的，就是这种人。要远离这些人生绊脚石。

## 一个鼓舞人心的例子 ✨

上星期我见了一个朋友，曾经是个情场高手，当然现在“上岸”了，老婆孩子热炕头。这人五短身材，也没什么钱，但是当年专挑高个儿的妹子交往。

我在成长的过程中，也曾经无数次想要放弃。但是每次看到他我就想：“连他都行，我也行。”我们喝酒的时候我提了一嘴，是你让我感觉到我们男人又行了。

他当时叼着烟，笑眯眯地对我说：“本来就行。”

本节课中，我们一起学习了“做聋子”的智慧。我们从青蛙爬塔的寓言中理解了屏蔽噪音的重要性，分析了现实生活中各种质疑的声音，并建立了“专注目标、远离绊脚石”的心态。记住，你的成长之路由你定义，他人的闲言碎语不应成为你前进的阻力。你，本来就行。
