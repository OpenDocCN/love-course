# 聊天聊出来的喜欢是真的喜欢吗-P1-情感解忧-柠蒙-BV1y94y197Hw_note

在本节课中，我们将探讨一个常见的情感困惑：通过聊天产生的喜欢是否真实。我们将分析这种情感的本质、其背后的心理机制，以及如何正确处理网络聊天中产生的好感。

---

聊天聊出来的喜欢，并非真正喜欢一个人。这种喜欢是在和自己的幻想谈恋爱。

各位是否有过网恋经历？是否会有这种感觉：总是不由自主地胡思乱想。对方稍微冷淡一点，马上变得敏感多疑、无所适从。这是因为大脑接收的所有信息，都会被不由自主地主观合理化。这意味着，让我们心动的并非对方这个真实的人。

而是我们想象力二次加工之后的产物。仅通过聊天认识的人，顶多只有30%是真实的。其他部分，如人品、相貌、性格，都是我们主观塑造出的理想模样。

举个例子。网恋对象发来一张精致早餐照片，你可能更愿意相信他是一个热爱生活的人。但实际上，他可能三天没洗头，脏衣服堆积了一星期。因此，非常不建议各位长时间发展一段网恋。

即使聊得再投机、感觉再对，很大程度上也只是在和自己的幻想谈恋爱。对方只是一个容器，被你想象成理想伴侣。也就是说，你喜欢的并非这个人，换谁都可以。

对方只是承担了你对恋爱的憧憬。你只是享受了一种虚拟满足的快感。网恋见光死几率特别大，是因为前期把对方想象得太美好了，以至于真正见面时会产生巨大心理落差。

曾经对恋爱的美好憧憬在瞬间被击碎。所以说，聊天聊出来的喜欢，只是爱上了自己的幻想。对方其实也是如此。但你们根本不知道真实的对方是怎样的：真实姓名、工作、情感经历、性格如何，你们是否是一路人。

一场网恋让谈恋爱中的关键因素，都被一时兴起的幻想忽略掉了。以至于真正在现实中见面后，才发现彼此一点都不合适。

记住一句话：聊天只是互相认识和了解的工具，绝不能成为产生爱情的唯一凭证。

网恋最正确的做法是：两人通过聊天互相有了粗略了解，至少证明彼此挺聊得来。然后聊一两个星期，最多28天，就要尽快敲定线下见面时间，试探有无长期发展的可能性。

如果见面后彼此感觉不错，就可以继续推进关系。反之，也可以及时止损，不浪费彼此时间和感情。

人的情绪新陈代谢周期一般是28天。超过这个时间限度，又缺乏真实性交往去巩固关系，就很难再保持高度亢奋的暧昧状态。

之后自然是越聊越无话可说，越是在网上先“恋”就越觉得索然无味，聊着聊着也就散了。

聊天聊出来的喜欢是短促、脆弱的，一般不会长久。甚至有些人断联超过三天，原本对那个人的冲动，三天就可以荡然无存。

所以无论男女，都不要把缘分压在虚无缥缈的网恋上。风险性太高，回报率太低。

聊天产生的喜欢一般不会长久。但在网络中**适度**接触，在**现实**中相互吸引、相互了解和包容，这样成就的感情往往才能细水长流。

---

本节课中，我们一起学习了“聊天聊出来的喜欢”的本质。我们明白了这种情感大多源于自身幻想，而非对真实个体的了解。关键点在于：聊天是工具，不是爱情凭证；网恋需尽快回归现实检验；真实相处与了解才是长久感情的基础。希望大家能理性看待网络聊天中的好感，走向健康、真实的情感关系。
