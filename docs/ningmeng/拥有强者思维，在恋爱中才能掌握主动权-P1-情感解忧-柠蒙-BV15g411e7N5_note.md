# 恋爱强者思维入门教程 🧠 - 第1课

在本节课中，我们将学习如何在恋爱关系中建立“强者思维”，从而掌握主动权，避免陷入被动和妥协的困境。我们将深入探讨强者思维与弱者思维的核心区别，并通过具体的方法帮助你培养这种思维模式。

---

## 强者思维与弱者思维的核心区别

上一节我们介绍了课程目标，本节中我们来看看两种思维模式的根本差异。

弱者思维的核心是**恐惧**。拥有这种思维的人，其关注点永远在他人身上：别人会不会喜欢我、会不会讨厌我、会不会不理我。这最终导致他们将对方置于主导地位。

而强者思维的核心是**反思和行动**。他们的关注点首先在于自身，不会轻易为了对方降低自己的底线和原则。虽然听起来有些冷酷，但往往这样的人更具魅力和吸引力。

---

## 如何培养强者思维：两大核心要点

理解了基本概念后，我们来看看如何具体培养强者思维。以下是两个关键的建议。

### 第一点：掌控感

掌控感是指无论外界如何变化，自身都不受干扰的能力。拥有强者思维的关键在于实现对自我的掌控。

在社交中，从你与任何人接触的第一秒开始，你们便进入了权力争夺期。有强者思维的人不会受外界变化干扰。此时，对自身的掌控至关重要。如果你无法控制自己，在相处中一再妥协，甚至不断降低底线，那么你就只能被对方掌控。

**例如**：你们认识不到一周，对方一直暗示想要礼物，第一次吃饭还要求带上闺蜜。如果你不乐意，对方就说你小气、不是真心。意志不坚定者很容易妥协，从而陷入恶性循环。

因此，拥有掌控感的关键在于**敢于拒绝**。面对上述情况，你可以明确表态：“我只给女朋友送礼物”，或直接拒绝带闺蜜吃饭的要求。当然，这并非让你事事拒绝，但持有这种态度，你才能被平等对待。这种掌控感不仅有益于情感，对工作和学习同样有帮助。

### 第二点：主导力

强者在一段关系中通常是主导者。但这里的主导并非试图控制所有琐事，而是**主导关系向更好的方向发展**。而弱者则喜欢控制一些与大方向无关紧要的细节。

我发现现在很多男生被一些观念误导。**例如**：男生第一次和女生见面吃饭，女生想吃烤肉，男生偏要说“我昨天才吃过，不想吃”，即使他很久没吃了。问他原因，他说是为了“显示自己的态度”，不去顺从女生。这个理由令人无语。所谓“有态度”，应体现在原则和底线上，而非一顿饭的选择。只有弱者才会在意这些小事，因为他们容易做到；而大事需要更多支撑，他们往往做不到。

如果你还不明白，可以想想工作中的情况：有的领导为了体现权威，每天紧盯考勤，甚至离谱地规定上厕所时间。而真正有能力的领导，思考的是如何合理安排工作、提高公司整体效率。

那么，如何拥有这种主导力呢？记住一句话：**抓大放小**。

**这是什么意思呢？** 如果你每天纠结于女生是否秒回消息、是否说早安晚安、是否问你在干嘛，那你就是拘泥于小事。你需要关注的是那些大的、原则性的要求，例如对方是否人品好、善良、生活独立。这些才是让关系变好、支撑你们走到最后的关键。当你关注这些大事时，再回头看那些小事，就会发现它们无足轻重。哪个女生没有一些小个性呢？作为男生，你只需主导关系的大方向即可。过分关注小事，反而会降低自身价值，这正是弱者思维的开端。

## 总结

本节课中，我们一起学习了恋爱中的“强者思维”。我们首先区分了以恐惧为核心的弱者思维和以反思行动为核心的强者思维。接着，我们探讨了培养强者思维的两个具体方法：建立**掌控感**（敢于拒绝，坚持自我）和培养**主导力**（抓大放小，关注关系发展的大方向）。掌握这些思维，能帮助你在感情中游刃有余，掌握主动权。
