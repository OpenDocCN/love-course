# 课程名称：柠檬 🍋

在本节课中，我们将要学习关于柠檬的基本知识，包括其定义、主要成分、常见用途以及如何挑选和储存。

柠檬是一种常见的水果，属于芸香科柑橘属。它的果实呈椭圆形，果皮为亮黄色，富含芳香油。柠檬的果肉多汁，味道极酸。

## 主要成分与特性 🧪

上一节我们介绍了柠檬的基本定义，本节中我们来看看它的核心成分与特性。

柠檬的主要酸味来源是**柠檬酸**，其化学式为 **C₆H₈O₇**。此外，柠檬富含维生素C，其分子式为 **C₆H₈O₆**。

以下是柠檬的一些关键特性：
*   富含维生素C和多种抗氧化物质。
*   果皮含有柠檬烯等芳香油。
*   果汁的pH值较低，呈酸性。

## 常见用途 🛠️

了解了柠檬的成分后，我们来看看它在日常生活中如何被应用。

柠檬的用途非常广泛。在烹饪中，柠檬汁可用于调味、去腥和防止果蔬氧化。在清洁领域，因其酸性，可用于去除水垢和作为天然清洁剂。柠檬也常被用于制作饮品和甜品。

以下是柠檬的几个主要用途场景：
*   **烹饪调味**：为菜肴增添酸味和清新香气。
*   **家庭清洁**：稀释后的柠檬汁可清洁厨房和卫生间表面。
*   **饮品制作**：柠檬水是经典的解渴饮品。

## 如何挑选与储存 📦

最后，我们来学习如何挑选新鲜的柠檬以及如何正确储存它们。

挑选柠檬时，应选择重量沉实、果皮紧绷有光泽、颜色鲜亮的果实。避免选择表皮柔软或有霉点的柠檬。

储存柠檬时，可以参考以下方法：
*   短期储存（一周内）可置于室温阴凉处。
*   长期储存应放入冰箱冷藏室。
*   切开的柠檬需用保鲜膜包裹切口并冷藏。
