# 狼群 - 網聊神技課：EP7【S1 基礎】優質自介：人格設定 🐺

在本節課中，我們將學習如何撰寫一個優質的個人自介。自介是交友軟體中除了照片外，吸引女生注意並促使她們右滑的關鍵。我們將探討自介的重要性、常見錯誤寫法，並提供正確的撰寫範例與技巧。

上一節我們討論了照片的重要性，本節我們來看看如何透過文字自介來強化你的個人形象。

## 自介的重要性 📝

自介的核心作用在於傳達你的個性與態度，並篩選出合適的聊天對象。以下是自介的幾個關鍵點：

*   **展現個性與態度**：自介是呈現你個人特質與生活態度的窗口。
*   **篩選合適對象**：女生會透過自介判斷是否與你合拍，幫助你避開不適合的對象。
*   **需要持續測試優化**：自介不是一成不變的，需要根據配對率反覆測試和調整。公式可以表示為：`最佳自介 = 持續測試(不同版本的自介內容)`。

## 不恰當的自介案例與分析 ❌

以下是幾種常見的錯誤自介寫法，請務必避免。

**1. 內容空白或自我貶低**
許多男生因不知如何下筆而留白，或使用自嘲、自我貶低的語句。在建立第一印象的階段，這會直接讓女生失去興趣。

**2. 羅列身高體重**
除非你的身高體重是顯著優勢（例如185公分/72公斤），否則不要列出。這可能傳遞錯誤訊號（如只想約炮），或讓非優勢條件過早暴露。記住交友軟體的原則是「揚長避短」。

**3. 放置個人社群帳號（如IG）**
在自介中放置IG帳號，通常傳達的是「缺粉絲」或「缺乏安全感」。作為男生，這不是一個值得推薦的做法。

**4. 傳遞無聊或過度需求的形象**
例如：
*   **無聊陳述**：「台中人，在台北工作，找穩定關係。」這創造了無趣的形象。
*   **過早暴露需求感**：「想找穩定的女友，每天互道早安。」這會在未見面前就給女生巨大壓力，違反了交友軟體從「認識」到「深入」的逐步過程。
*   **無效提問**：「你喜歡貓還是狗？」這類問題無法有效展現你的個性或開啟有趣對話。

## 恰當的自介寫法與核心要素 ✅

一個好的自介應能鮮明展現個性、創造互動感，並具有篩選力。我們來看兩個範例。

### 範例一：簡短有力型（大衛哥風格）

```
喜歡衝浪、滑雪、德州撲克、籃球。
到處吃美食，告訴我一個你最近發現最特別的餐廳？
From 西雅圖，如果你也是，給我個 shoutout。
```

**這個自介的優點在於：**

*   **鮮明個性**：透過興趣（衝浪、滑雪）塑造出陽光、外向的形象。
*   **強互動感**：「告訴我一個你最近發現最特別的餐廳？」直接提供了一個高品質的開場話題，引導女生主動互動。
*   **價值展現與篩選**：「From 西雅圖」暗示了留學背景，提升了社交價值。同時也能吸引有相同背景的人。

**技巧延伸**：即使沒有留學背景，也可以用類似結構。例如，如果你是台南人，可以寫：「Grew up in Tainan. If you ever in Tainan, let me show you around.」穿插簡單英文能提升質感。

### 範例二：詳細豐富型（工作狂風格）

```
一名男性創業家/工作狂。享受學習與冒險。
喜歡的美食從路邊攤到無菜單料理。為了上台參加健美比賽而訓練。閱讀、德撲、KOL觀察家。
喜歡的活動：電影、衝浪、打球，偶爾看展找靈感。
有一隻狗叫發抖。
如果你只是漂亮但笑點太高，我們可能合不來。
告訴我一個你看過最好笑的諧音梗？
```

**這個自介的優點在於：**

*   **立體人設**：「創業家/工作狂」傳遞上進、自律的形象。「為了健美比賽訓練」展現了決心和毅力。
*   **具體化興趣**：將「喜歡吃東西」具體為「從路邊攤到無菜單料理」，畫面感更強，延伸性更高。
*   **強篩選性**：「如果你只是漂亮但笑點太高…」明確篩選出符合自己喜好的對象，這是一種自信的表現。
*   **互動引導**：以「告訴我一個你看過最好笑的諧音梗？」結尾，同樣創造了互動入口。

## 總結與行動指南 🎯

本節課我們一起學習了優質自介的撰寫心法。關鍵在於透過文字呈現你的**個性**、**態度**，並設計**互動**與**篩選**機制。

**請記住以下行動步驟：**

1.  **避免雷區**：刪除空白、自我貶低、身高體重、IG帳號及過早暴露需求感的內容。
2.  **確立核心形象**：思考你想傳遞的主要人格特質（如陽光、細心、有事業心）。
3.  **選擇格式並撰寫**：參考提供的長、短兩種範例格式，撰寫你的自介初稿。原則是：`優質自介 = 鮮明個性 + 具體興趣/生活 + 互動提問 + 篩選聲明`。
4.  **持續測試與優化**：將不同版本的自介投入實際使用，觀察配對率變化，並根據你的生活成長定期調整內容。

自介是你的人格設定檔，請用心經營，並在實踐中不斷迭代。下期見！
