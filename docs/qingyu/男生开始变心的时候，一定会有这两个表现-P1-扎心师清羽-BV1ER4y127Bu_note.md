# 男生开始变心的时候，一定会有这两个表现-P1-扎心师清羽-BV1ER4y127Bu_note

在本节课中，我们将学习如何识别亲密关系中一方情感发生变化的两个典型行为信号。理解这些信号有助于我们更客观地评估关系状态。

## 概述

当一段亲密关系中的男性情感开始发生变化时，其行为模式通常会出现可观测的转变。本节课程将具体分析两种标志性表现，它们往往是感情进入危机阶段的预警。

## 表现一：从接纳到厌烦

上一段我们概述了课程目标，本节中我们来看看第一个关键信号：态度从接纳转变为厌烦。

他会开始嫌你烦。过去，你找他聊天会让他感到开心。但现在，你每次找他，他都觉得没有必要。他甚至会认为你耽误了他的时间。

他不仅对你失去耐心，还会将你的关心和陪伴视为累赘。他不再主动与你分享他的所见所闻。即便是与你多说几句话，他也会讥讽你，说你烦。

因此，当男生变心时，他不仅会忽视你对他的好，反而会无限放大你的缺点。如果你表现出不开心或受伤的样子，他也只会敷衍地说一句“别多想”。他表现得好像很直男，但事实上，他心里什么都懂，什么都感觉到了。只是你对他来说已经不再重要，或者说他已经打算离开你，所以才懒得在你身上花费心思。

## 表现二：持续的疏离与回避

了解了态度转变的信号后，我们接下来分析第二个表现：行为上持续的疏离与回避。

他总是对你说他在忙，或者手头有事情要处理，说“待会再聊”，然后人就消失了。在接下来的很长一段时间里，你们的聊天记录显示都是你在主动找他。

他也几乎不再邀请你参与他的生活，例如陪他打篮球或吃饭。这是一个重要的分界点。

当感情中的一方觉得你可有可无，对你不再有足够的需求感时，就说明他的心已经不在你这里了。他可能宁愿刷手机，也不愿把时间花在你身上。

在这种状态下，他有很大概率会去寻找下一段能让他动心的感情，也就是通俗意义上的移情别恋。

## 总结

本节课我们一起学习了识别亲密关系可能发生危机的两个关键行为信号：**态度上从接纳转为厌烦**，以及**行为上出现持续的疏离与回避**。理解这些信号有助于我们更清醒地认识关系现状，从而做出理性的判断和决策。
