---

我发现最近很多人在面对复联的时候，都有一个错误的心态，就是在断联结束之后，都不愿意主动的去联系前任。

你们要搞清楚一点，虽然说在前期我们已经做了一系列的铺垫、提升或者说弱联系，在一定程度上可以调动对方的情绪，甚至让对方在分手这个事情上产生质疑，但是你不主动去复联，是很难让对方重新回到你的身边的。

很多人会说，自己不愿意去联系的理由，是觉得自己主动了，会显得很卑微很掉价。我想让他来主动联系我，因为我不想让他觉得我还在追他在乎他。

其实这样的想法完全是以自我为中心。往往分手后那个敢于去主动联系前任的人，那才是真的“我”。

## 为什么你害怕主动？ 🧐

上一节我们指出了不愿主动的普遍心态，本节中我们来看看这种恐惧的具体表现和根源。

我以前有个学员就来问我，他说：“老师，分手后我也断联了一段时间，我把重心都放在了自己生活上，我学会了很多。然后这段时间，他也给我的朋友圈经常点赞评论，我觉得是时候了，我以为我是有勇气去找他的，但是我始终迈不出那一步。”

他的困惑非常典型，主要集中在以下几点：

以下是该学员（以及许多人）内心的具体担忧：

1.  **我不知道我该说什么**：害怕开场尴尬，找不到合适的话题。
2.  **要是去找他会不会显得我很低位**：担心主动行为会降低自己的价值和尊严。
3.  **就算是复合了，我是不是再也找不回主动圈了**：恐惧即使复合，权力格局也会永久倾斜，自己将永远被动。

## 重新定义“主动”与“自尊” 💪

了解了常见的恐惧后，我们需要从根本上重新审视“主动联系”与个人价值之间的关系。

首先你们要明白，如果说他不主动找你，你是完全可以重新开始的，没问题的。如果说你还是以复合为前提的，那你的断联时间够了，是需要主动去联系对方的。

确实，主动联系前任会让你觉得你失去了自尊，但是这种不祈求对方回应的勇气，真的是代表自己输了吗？这反而是一种“向下兼容”对方的体现。

因为一个人的自尊，**公式**不是：`自尊值 = f(谁先联系)`
这个公式很幼稚。自尊并非由“谁先联系谁”、“谁先承认放不下”这类行为决定或贬损的。

相反，真正心理成熟的人，他越知道自己想要什么，知道自己在做什么，所以说他不会在意对方的评价和眼光。

## 掌握真正的主动权 🎯

那么，什么才是真正的主动权呢？让我们来澄清这个核心概念。

而且你要清楚一句话：**主动权一直都在你的手上**。

也许你会觉得联系前任，在表明自己的内心需求，在暴露自己的需求感。但是我想告诉你们的是，主动权一直在你手上，不管你是否联系他。

就算你是那个先主动的人，就算他对你的情绪、对你的行为有所怀疑，你也可以用你的行动和你的状态来证明。因为你说话的态度、你聊天的情绪、你处理事情的方法，这些都能让他感受到你改变的自信。

**代码**描述这种状态可以是：
```python
if 我主动联系前任：
    我的状态 = 自信、平和、基于自我提升
    传递的信息 = “我很好，且我愿为我们提供一个可能性”
    主动权归属 = 我（由我的状态和框架决定）
```

## 如何迈出第一步：搭建桥梁 🌉

理解了心态和主动权之后，我们来看看如何将理论付诸实践。

所以说在一起准备就绪的时候，我们就该主动的迈出那一步，建立一个连接的桥梁。

因为绝大多数情况，彼此都是不够勇敢的，都不敢迈出第一步，都不给对方台阶下。一方面就是俗话说的“好马不吃回头草”，就算他已经后悔了，他也不一定敢去联系你。另一方面就是他看见你已经做出了改变，可能以为你已经走出了这段感情，开始了新的生活。

以下是“搭建桥梁”这个行动的关键解读：

*   **桥梁是机会**：它不代表屈服，而是为自己和对方创造一个重新评估关系的安全空间。
*   **你是桥梁的提供者**：这本身就是一种力量和自信的体现，意味着你拥有给予机会的能力。
*   **目的明确**：正如课程所说，“你是以复合为目的的，那你就要先做好这个桥梁，先做那个给出桥梁的那个人。” 行动服务于你的目标。
