# 复合的核心：给失恋的双方！ - 课程 P1 🧠💔

在本节课中，我们将要学习分手后如何理性看待复合，无论你是想挽回的一方，还是被挽回的一方。我们将探讨常见的错误心态、正确的思考方向以及有效的行动步骤，帮助你走出情绪漩涡，做出明智的决定。

如果你现在正在犹豫是否要去挽回，或者挽回正在进行中，无论你是哪一方，我希望你可以认真听完我接下来要说的话。

你可能因为分手的事情感到焦虑、担心、悔恨、痛苦，甚至是绝望。你已经尝试了所有能用的方式。

对方的态度依然很坚决。你现在感到非常无助。这些感受其实都源于你还爱着这个人，你很不甘心。你认为只要说出自己的诉求、表明改正的决心就可以了。

但是请问，除了用嘴说，你还做了什么？软磨硬泡、死缠烂打、威胁恐吓。

下跪道歉，一边卑微地求复合，一边还想着保留所谓的尊严。你做出了不给对方留一丝念想、不给自己留一丝余地的姿态。这恰恰也是你不懂爱的表现，是你不会爱。挽回一个人难吗？不难。难的是挽回自己。

很多人只是在自我的视线里，进行自我感动式的付出，以为拼尽全力就能够让对方感同身受。

可是日积月累的矛盾，是能够在双方情绪都不稳定的时候化解开的吗？所以你的方向就错了。你再怎么努力都无济于事。

我们可不可以，在挽回之前，先动用理智的大脑思考一下？你想和他重新在一起的原因，你想清楚了吗？你有审视过这个原因吗？

感情是双向选择。你无法强迫一个人重新回到你身边。两个人走到分手这一步，不可调和的问题是什么？

如何解决？为什么会造成这样的局面？真的和好了，能否像从前那样毫无芥蒂地继续相处？

你冷静下来，站在他的角度上去想一想。你试着去理解对方，去考虑一下彼此心里那根刺是否能够拔掉。

你想好了这些问题，再去真诚地和对方沟通。让对方感受到你这次所做的、所说的，里面包含了他需要的那一部分。

这样坦诚的沟通，他才可能重新接受你。就算没有马上答应，他内心的负面情绪也会化解一部分。他也可能会跟你说说他的看法。

他是怎么想的。他也很大可能会留有余地。这样你们之间才能够增进相互了解。这样才有利于你去挽回感情。

所以我建议你现在给自己，还有对方，一些时间冷静一下。在这个期间，想想能做什么。

能够让他感受到你的诚意和爱意。同时，你好好经营一下这段时间如同行尸走肉般的自己。

如果是提分手的一方，你也看到了我的视频。我想对你说一段话。

也许你目前对这个人很抵触。他曾经一次次触碰你的底线。

让你觉得很失望。你觉得这个时候他能回头来找你，虽然自己似乎被重视了，但已经晚了，对吗？因为你认为爱需要及时给予，过了那个时候再来弥补，其实已经不需要了。迟到的深情比草更贱，对吧？

但我想说的是，虽然会有失望，但其实你也是幸运的。至少这个人他真的很认真地爱过你。这样拼尽全力去挽回你的人，可能你这辈子也只能遇到这一次了。

你害怕他失去理智的纠缠。这只是因为他害怕失去你而用力过猛。这其实是他在无计可施时的办法。

他现在狼狈不堪，在你看来已经失去了当初在你心里闪闪发光的样子。但他可以放下尊严、丢失面子，只是因为他知道自己错了，他很想跟你继续走下去而已。

所以你可以先安抚好对方的情绪。但是建议你不要伤害他。

尽量不要用冷漠的语言和无缝衔接新恋情，去伤害对方的心。在他看来，其实只是想再努力一下。只要他经历了，不管结局如何，他至少没有遗憾了。

所以当下的厌烦可能只是这一段时间。也许通过这次分手他也会成长、会改变。

也许你还会爱上他。但就怕那个时候，你们已经没有任何回转的余地了。

他这么用力地挽回你，并不是因为你真的无可替代。而是在他那里，他觉得你很好，是他给你的爱加上了很多光环。

所以我希望，如果你们不是因为不爱了，那就不要让彼此等太久。在他变好的过程中，如果还可以，我希望你可以参与进去，而不是彼此都痛苦地等待和相互折磨。

---

本节课中，我们一起学习了分手后关于复合的核心思考。对于挽回方，关键在于**停止情绪化纠缠，转向理性沟通与自我成长**，公式可以概括为：**有效挽回 = 真诚沟通 + 理解对方 + 自我提升**。对于被挽回方，则建议**给予理解和空间，避免用伤害的方式拒绝，审视感情的真实价值**。记住，复合的目的不是简单地回到过去，而是共同解决导致分手的问题，建立更健康的关系。给自己和对方一些时间和耐心，让改变和成长发生。
