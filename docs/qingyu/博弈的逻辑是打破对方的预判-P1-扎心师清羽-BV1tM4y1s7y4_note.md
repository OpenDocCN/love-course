# 博弈的逻辑是打破对方的预判 - 课程P1 🧠💔

在本节课中，我们将要学习如何运用博弈思维，来应对一段失望的感情关系。核心在于理解并打破对方的负面预判，通过调整自身策略来扭转局面。

## 概述：博弈在感情挽回中的应用

让一个想分手的人重新对你产生兴趣，需要运用博弈。一段令人失望的感情背后，往往是负面的人性占据了上风。此时，对方对你冷淡和恼怒是他的逻辑，他甚至希望你立刻消失。

即使他曾有过不舍，这种情绪也终究是昙花一现。因此，如果你现在选择正面硬刚，不会有好结果。他会认为你在对抗他，从而再次加深你们之间负面的印象和阴影。

## 第一步：注意力抽离 🎯

博弈可以帮助你绕过这层阴影。你不和他争论，也不和他对抗。你也不认可他的游戏规则。既然他的认知停留在此，你就选择绕道而行，从侧面去影响他的想法。

所以，博弈的第一步是进行注意力的抽离。你需要将更多精力和时间落实到行动上。无论是想和他争论、道歉还是讨好，甚至使用各种技巧，只要你的注意力仍然集中在他身上，你的所有行为都可能失控。

你要做的是逐步抽离对他的关注，转而专注于你自己。如果你的眼睛时刻追着他、盯着他转，那你如何看清脚下真正的路呢？

许多人心态不稳、方向不明，就是因为花了太多精力和心思，将注意力放在他的反馈这件事上。我们做事不能三心二意。在“专注他导致持续内耗”和“暂时绕过他，专注行为本身”之间，傻子都知道该怎么选。

## 第二步：停止自证 🚫

第二点就是不要去证明。无论是什么问题，你都没有必要去证明你的观点或你的正确性。一个问题能否解决并非至关重要。当你和他已经站在对立面，处于对抗的前提下，你说任何话、做任何事都是没有意义的。

无论你如何企图通过证明自己的正确来打动他，这些都是无效行为，因为他不会认同你。一个站在审判者高度、给你贴上你不能接受的标签、诱导你自证清白，却又对你的证明视而不见的人，本质上是为了引诱你落入自证的陷阱，从而实现他对你的控制或羞辱。

这就像电影里那个为了证明自己只吃了一碗粉而抛开肚子的人。你的自证行为与此类似，将你推入了深渊中的牢笼。

另外，一个本身就与你处在对立面的人，即使你解决了一个问题，他还会提出另一个问题来为难你。因此，想让他改变想法，只能让他自己感受到并主观上认可这些信息是对的，他才愿意改变。

## 第三步：顺势而为 🌊

第三点，博弈讲究顺势而为。人与人之间的相处就像流水一样，不要陷入某个方向过分执着。方法有很多种，能解决问题的答案也不止一个。

很多时候，问题不需要刻意去解决。如果遇到阻碍，想想水流。它一定会冲击石块、冲击阻碍，闹个你死我活吗？不是的。水流会绕过去，用层层布局的水流去冲刷、去覆盖。

所以，想要做到复合，你不一定要身先士卒。适当地停下来，静观其变，同样非常重要和有用。而且一旦事成，就像潜龙飞天一样，对准他的弱点、对准他的需求处，一击即中。

## 总结

本节课中，我们一起学习了博弈思维在感情挽回中的三个核心步骤：**注意力抽离**、**停止自证**以及**顺势而为**。关键在于打破对方的负面预判，通过专注于自身和策略性行动来影响局面，而非直接对抗。记住，博弈是绕过障碍，而非撞碎障碍。
