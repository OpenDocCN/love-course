在本节课中，我们将要学习如何理解和挽回回避型依恋的伴侣。回避型依恋者在亲密关系中常常表现出疏离和逃避，这源于他们独特的心理防御机制。理解其行为背后的逻辑，并采取正确的方法，是修复关系的关键。

## 理解回避型依恋人格

上一节我们介绍了课程目标，本节中我们来看看回避型依恋人格的核心特征。

回避型依恋者并非不需要爱。恰恰相反，因为太在乎，所以害怕再次受到伤害。因为害怕伤害，他们便假装不在乎，采用逃避和自我压抑的方式来保护自己。这是一种**自我保护机制**。

> 在童年时期，若个体未能得到父母充分的爱，他大概率会发展为回避型人格。

在他们的认知里，与其等待别人施舍爱，不如自己爱自己来得痛快。因此，面对分离时，他们的反应往往是平淡和冷漠的。

## 挽回的核心原则：切忌断联

理解了回避型依恋者的心理后，我们来看看挽回时最关键的禁忌。

在挽回对方时，绝不能采取断联的方式。如果你选择断联，就会触发他的自我保护开关。一旦开启自我保护，无论你说什么，在他看来都是托词。

> 他会想：“你说得那么好听，还不是说离开就离开了。”

如果断联时间过久，等到他因你而产生的恐惧感消失，他便不会再对你有任何期待。此时断联，只会导致他真正的觉醒和离开。这与普通恋爱中对方忽冷忽热的情况截然不同。

## 挽回回避型伴侣的三个步骤

既然断联不可取，那么具体该如何操作呢？以下是挽回回避型依恋伴侣的三个核心步骤，基于成功的实践经验总结而来。

### 第一步：坚定选择，帮助解压

回避型依恋者内心害怕被否定，渴望无条件的爱。面对分手，他们内心脆弱，一方面认为自己无能，无法解决问题；另一方面害怕你的负面回应。

此时，你需要引导他，弱化你们之间的矛盾，让他感受到这个矛盾在你心中并非不可逾越。你要帮他解压，将他从惯性的逃避思维中拉出来。

你可以这样说：
```
“即便分手了，我觉得我们还是不能去怪谁，因为感情里没有对错，那是两个人心动的结果。我相信你是个好孩子，也相信自己的选择。你不要逃避我就好，我也没有想干什么，只是单纯想和你聊一聊。毕竟我们相处那么久了，依赖感还是存在的。你不用害怕，我也不会纠缠你，只是觉得这个事情，没有必要那么在意。”
```

### 第二步：控制节奏，给予空间

回避型依恋者特别注重个人边界感，他们需要足够的个人空间来感到舒适和自在。只有这样，他们才有精力去维系或修复关系。

你需要学会控制联系的节奏，给予他所需的空间和安全感。同时，要不卑不亢地向他道歉，承担自己该承担的责任。这种态度会让他感到非常安心。

### 第三步：表达主见，带领对方

回避型的人喜欢被带领，也希望有人能说服他、帮他承担责任、引导他往前走。如果你的态度摇摆不定，情绪不稳定，他也会跟着犹豫不决。

你需要表达你的主见和决心，带领他建立对关系的信心。

你可以告诉他：
```
“此时此刻我已经成熟了，我对你的爱是出于欣赏，不是捆绑，也不是压迫，更不是道德绑架。我就是想和你一起，我就是想带领你往前走。”
```

当你做到这一点时，你们的关系修复过程就会变得特别顺利和舒适。
