# 恋爱避坑指南：识别低成本追求者 🚩 - 课程01

在本节课中，我们将学习如何识别那些试图以最低成本与你建立亲密关系的男性。了解这些关键信号，可以帮助你避免陷入不平等的、缺乏真诚投入的关系中。

---

上一段我们概述了本课程的目标，接下来我们将逐一解析低成本追求者的六个典型特征。以下是具体的识别要点：

**第一点：急于推进关系**
这类男性会表现出过快的亲密需求，并有意识地试探你的个人底线。他们的目的是快速跨越正常关系的发展阶段。

**第二点：只关注自身吸引力**
他并不真正关心你的个性、喜好或感受。他的关注点完全在于**你是否喜欢他**，以及**你喜欢他的程度**，以便据此判断可以进行的肢体接触范围。

**第三点：引导性自我对标**
他会有意无意地询问你喜欢的男性类型，随后将自己代入那些描述中，暗示他就是你所期待的人。同时，他不会给你充足的时间去真正了解他的为人。

**第四点：打卡式关心**
他只会进行间断性的、低投入的联系，例如偶尔报备行程。但当你热情回应时，他却可能消失很长时间。这种模式是缺乏持续投入意愿的体现。

**第五点：偏好深夜邀约**
一个常见的模式是，他经常在较晚的时间约你。这通常意味着他不想在常规约会中投入时间，其意图可能直接指向将相处时间过渡到夜间。

**第六点：缺乏自省，言行不一**
这类人通常不会认为自己有任何问题。他们典型的表现是“又当又立”：既不愿在你身上投入过多时间，也不愿花费过多金钱，却同时声称自己对你是“真心的”。

---

本节课中，我们一起学习了识别低成本追求者的六个核心特征。总结来说，这类行为的共同点在于**追求快速回报**、**缺乏真诚了解**、**投入程度极低**且**伴随自我合理化的说辞**。保持清醒，给予关系正常发展的节奏和时间，是避开此类陷阱的关键。
