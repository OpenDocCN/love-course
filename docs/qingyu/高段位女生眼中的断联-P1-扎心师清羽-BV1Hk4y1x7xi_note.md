# 高段位女生眼中的断联 - 课程P1 🧠💔

在本节课中，我们将学习“高段位”女性视角下的“断联”策略。我们将剖析断联如何影响双方的心理、价值感知以及关系走向，并为你提供清晰、理性的行动指南。

---

真正高段位的女性，甚至能将你断联后多少天会回来找她，都估算得很清楚。她对你的畏惧感来源于：你敢主动与她断联，你敢反过来去“掠”她。而你的价值感提升则源于：你能做到不怕失去。

上一节我们提到了价值感的提升，本节中我们来看看断联时间长短带来的心理变化。

你敢彻底不理她。你与对方断联越久，你就会越理智，越觉得自我价值高。反过来，你越害怕失去她，纠缠消耗越久，就越会觉得自己没用，慢慢变成自我安慰。值得注意的是，不少人在断联过程中，才能真正确定对你的感觉。

以下是断联行为可能引发的对方心理反应：

*   尤其是你之前天天围着她转，离不开她。你敢断了，她才可能高看你。
*   如果你对她好，她就轻视你；你不理她，她反而来劲了。这只能说明这个人骨子里的“剑”（贱）是没得治了，那就应该让她彻底滚出你的世界。

谈恋**爱要用脑子，而不是凭感觉用情绪**。不知道**该不该说的话不说**，不知道**该不该做的事不做**，不知道**该不该信的人不要相信**。

即便她让你心动，她不选择你又有什么用呢？没用心就是没用心，无论怎么美化都仅此而已。只沉迷于她跟你在一起时那一点点甜，而对于让你知难而退的狠话，你却一句都听不进去。

从你一股脑把所有感情希望都寄托在她身上开始，就注定了悲惨的结局。一个人不喜欢你，有时候跟你好不好没有任何关系，只是因为你不是她的“菜”，或者她自视清高。

价值都是相对的。你在她这里价值不够，并不代表你真的价值不够。她眼高手低，又想要对方条件好，又想要情商高、懂事、理解她，满足她的大男子主义。她没意识到她想要的完美伴侣根本不存在，总觉得自己一定能找到更好的。

以下是当对方将你视为非最优选择时的表现：

*   你并非她权衡利弊之后的最优选择，她自然会对你有诸多挑剔，包容性极低。
*   这是她自己尚未完成的人生课题。你需要让她多去接受社会的“毒打”，看清现实。

姑娘，你要做的不是自我怀疑，或者在她身上激发出自己无限的征服欲，而是要把她忘掉，去真正值得你的人那里享受人生。

记住一句话：**有情饮水饱，无情催人老。若要想分手，稀世都想走。**
