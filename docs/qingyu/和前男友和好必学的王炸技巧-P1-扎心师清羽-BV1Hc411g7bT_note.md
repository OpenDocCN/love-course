# 挽回前任教程 P1：钓鱼式复合技巧 🎣

在本节课中，我们将学习一种被称为“钓鱼式复合”的挽回技巧。这种方法适用于分手后前任态度冷淡、拒绝沟通甚至拉黑删除的情况，尤其当你已经尝试过纠缠但无效时。我们将分步拆解核心策略，帮助你停止错误行为，重新掌握关系主动权，并最终引导对方回应。

## 停止错误行为与调整心态

在深入具体技巧前，必须先纠正两个关键错误。上一节我们概述了课程目标，本节中我们来看看挽回初期必须完成的准备工作。

**核心前提是：立即停止无效攻势，并调整脆弱心态。**

以下是需要立刻执行的两点：

1.  **立刻停止所有猛烈的求和行为。** 在未掌握正确方法前，你的行动无法触及对方真正的软肋和内心。
2.  **收起玻璃心，停止自我感动。** 卑微和纠缠只会将对方推得更远，这完全不划算。

完成以上两点后，你的策略核心将转变为：**表面上冷淡处理，暗地里进行刺激，并顺势观察。**

## 理解策略核心：重建平衡与主动权

想知道为什么要采用“表面晾着，暗地刺激”的策略吗？因为分手后，尤其是经过你的纠缠，双方关系已处于极度不平衡的状态。挽回的核心在于夺回关系主动权。

如果关系地位不对等，双方就无法进行有效沟通。

**那么，如何拿回主动权？答案是：打破“狭窄框架效应”。**

用通俗的话说，就是避免与对方硬碰硬，合理运用策略的柔韧性。这本质上是一种 **“以柔克刚”** 的攻势，配合循序渐进的方法来重新夺权。

## 分步操作指南

理解了策略背后的逻辑，现在我们来看看具体的操作步骤。以下是钓鱼式复合的三个阶段。

### 第一阶段：表面晾着（约1周）

既然分手是他提出的，那么我们就顺从他的意思，给他空间。这就是“表面晾着”阶段，持续时间大约一周。

### 第二阶段：暗地刺激（首次接触）

一周后，主动给他发一条消息。消息内容可以是请他帮一个小忙，或咨询他擅长领域的问题。

**关键操作是**：无论他是否回复，在两小时后，追加发送第二条消息，告知“不用了，我已经解决了/处理好了”。这就是“暗地里的刺激”。

### 第三阶段：深度刺激与收网（二次接触）

再等待一周。

1.  将他留在你这里的物品打包寄回。
2.  附上一张简短的便签。便签内容需包含以下要素：
    *   肯定他曾经对你的爱。
    *   肯定你对他付出的爱。
    *   表达目前的遗憾与无力回天的感觉。
    *   附带一两句简单的关心话语。
    *   **严格控制字数在五六十字左右。**

完成这一步后，等待即可。三天内，对方很可能会传来消息。

### 最终阶段：回应与巩固

当对方发来消息时，不必急于回复。

1.  可以先随意发布一条与感情无关的朋友圈。
2.  等到晚上有空时，再去回复他的消息。

**回复时必须注意两点：**
1.  不要谈论感情话题。
2.  不要恋战，做到“一触即退”。

此后，你便可以静观其变，坐收渔翁之利。
