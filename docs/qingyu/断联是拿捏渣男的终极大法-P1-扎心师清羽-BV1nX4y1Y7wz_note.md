我经常看视频的人都知道，断联不是让你们演给他看，而是做到真正的不怜惜。它也不是数着日子等着他来找你。像那种“我自己都憋得不行，对方有一点小动作就睡不着，想要去情绪反扑”的，说实话，你们断多久——断一年、断五十年——都没有用。
