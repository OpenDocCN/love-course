# 断联前如何让对方愧疚 - 课程P1 🎯

在本节课中，我们将学习如何在一段关系面临危机时，通过恰当的策略引发对方的愧疚感，从而为关系的修复创造机会。核心在于理解愧疚感的心理机制，并掌握一种有效的沟通方法。

---

## 理解愧疚感的作用机制 🔄

上一节我们介绍了课程目标，本节中我们来看看愧疚感在关系中的关键作用。

只要你有稍微的变化，对方就能够感知得到。常常吵架的情侣回想一下，是不是内心其实挺喜欢吵架的感觉。每次吵完架之后，对方来哄，和好之后感觉关系更亲密了，甚至会感觉比之前更好。

因为男人是一种会把愧疚感转化为行为的生物。

## 如何利用愧疚感修复关系 ❓

那么如何利用愧疚感，让对方想跟你和好呢？很多来找我咨询的学员，什么纠缠、打电话，什么都用过了，对方不仅不搭理，甚至还适得其反。

那么在这种情况下，如何给对方制造愧疚感呢？不要怕，学会我下面这段话。看完之后，他肯定会对这种关系有所思考。

以下是用于引发思考的关键沟通模板：

> “其实你没有必要那么冷淡的，我从来没有想过要纠缠。你那么明显的敷衍，我怎么可能不知道呢？我只是一直骗自己。我很想放弃你，却又舍不得。想继续又觉得自己很委屈。我用最好的沉默和坚持，只为了看清你的答案，让我以后想起来也不会再有遗憾。很抱歉，我的出现给你带来了困扰。接下来的日子，你不找我，我也不会主动再去找你了，你的联系我也不会删，毕竟曾经深爱过，只是不再抱有任何的期望了。你是我第一个改变自己去倒贴的人，我从不后悔爱上你，但是你却让我输得下巴来抬。现在呢，我放过你，也放过我自己。爱你是真的，舍不得也是真的，无能为力也是真的。可能很久以后你才会懂得，像我这么骄傲的人，这需要多大的勇气才会来求你。我现在只是不想让我们的关系那么僵。”

发送这段话后，你就不要再去联系对方了，他自然会来找你。

## 抓住机会窗口并做出改变 🪟

当对方对你产生愧疚感之后，就会提供一个窗口。简单来说，就是很多时候难以拒绝你，也会给他了解和看到你改变的机会。

在得到这个窗口之后，你得理清楚你们分开的表面原因和根本原因，明白了对方想要的是什么，针对性的做出改变。这是所有和好的前提。

以下是需要针对性改变的几个方面：

*   **沟通问题**：如果沟通方式有问题，先学会怎么样沟通。
*   **性格习惯**：如果性格习惯有问题，去学会怎么样收敛。
*   **观念差异**：如果恋爱观有问题，去学学怎么纠正。

但是在后续的接触过程中，不要跟对方说了你做了多少努力，改变了多少。你千万不要表现出你就是为了他才这么努力的，这会给对方很大的压力。

## 无声地展示你的变化 📱

刚分开的时候，双方都是不太愿意见面的，所以可以通过日常的聊天，和对方可以看见的社交平台体现。

比如说你之前总是情绪化，每天都情绪起伏比较大。那么可以跟对方某次聊天中穿插一句：“有点想调整一下自己的生活方式。”然后再去通过朋友圈之类的展现出来。

和好最大的困难点是什么？就在于就算改变了，对方也看不见。但是利用对方的愧疚感，可以直接把这个难点攻破。只要你有稍微的变化，对方就能够感知得到。

## 维持机会窗口的要点 🔑

只要你不一直做让对方反感的事情，比如说一直发消息打电话，让他对你的愧疚感消失殆尽，那么这个窗口就能长时间的保持。而这个就会是你们和好的契机。

---

本节课中我们一起学习了如何通过引发愧疚感为关系修复创造窗口。核心步骤是：发送深思熟虑的告别信息触发对方思考 → 利用产生的愧疚感获得沟通机会 → 冷静分析问题根源并针对性改变 → 通过日常互动无声展示变化 → 避免纠缠以维持机会窗口。记住，改变是为了自己，而展示改变需要策略。
