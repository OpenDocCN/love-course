## 复联前的核心心态调整

上一节我们介绍了复联的常见困惑，本节中我们来看看成功复联前必须调整的两个关键心态。

很多挽回者在经历断联后准备复联时，常常困惑第一句话该说什么。简单的问候显得轻率，直接聊天又很突兀，而长篇大论的道歉或争执又可能引起对方反感。

首先需要纠正一个误区：复联话题本身没有对错，只有是否合适。很多人功利心太强，一提到挽回就立刻想要复合，一提到复联就要求必须成功。但事情并没有那么简单。

即使是最终成功挽回复合的人，一开始的复联过程也往往是磕磕绊绊的。因此，在挽回复联前，你需要为自己树立两个核心心态：

1.  **不求对方一定回复**。
2.  **对方回复后，不追求立竿见影的效果**。

记住公式：**你的预期越低，成功的可能性就越大**。因为你的低预期意味着你带给对方的压力更小。你让对方感到轻松，他才不会排斥与你接触。

## 复联信息必须避免的四大雷区

在明确了正确心态后，我们来看看具体操作中必须避开的陷阱。以下是发送复联内容时需要注意的四个关键雷区。

**第一个雷区：挽回中不要提及感情。**

这就像国家谈判中的主权问题，是原则和底线。分手后，你们的立场是根本冲突的：你想挽回，他想分手。你越是想通过谈判把感情问题谈清楚，对方就越会让你彻底死心。

**第二个雷区：挽回中不要去质问或追问。**

许多挽回者会情绪失控地追问前任，例如“你怎么不回我消息？”“你是不是出轨了？”“你不是说做朋友吗，为什么不理我？”。这其实是一种权力乱入的现象。你们已不再是情侣，你却依然想停留在情侣的角色上滥用职权，对方一定会更快地把你推开。

**第三个雷区：挽回中不要长篇大论。**

情侣在热恋时最喜欢腻歪和狂发消息。而分手后，他们最怕的同样是腻歪和信息轰炸。满屏的文字会带来铺天盖地的压迫感，这种方式必须避免。

**第四个雷区：挽回中不要去激怒对方。**

人性都有弱点，不要再去二次伤害你的前任。即使不为他考虑，也要为你自己考虑。毕竟你还爱他，伤害他对你没有任何好处。

## 复联的正确路径与总结

如果你能成功避免以上四点，就已经迈出了成功的第一步。至少，你控制住了事态，没有让其进一步恶化。

永远不要试图通过**说服**或**讲道理**来挽回对方。正确的方式是通过**潜移默化**的影响来达到挽回的目的。
