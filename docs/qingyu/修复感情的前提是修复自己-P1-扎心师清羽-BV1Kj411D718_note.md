## 恶性循环的根源

你的抱怨和控诉来自于他的不满足。他的不满足来自于你的负面情绪。你的负面情绪来自于你的依赖过重。而你的依赖来自于你的内核脆弱。

**公式：内核脆弱 → 依赖过重 → 负面情绪 → 对方不满足 → 抱怨控诉 → 关系崩坏**

这是一个恶性的循环。在这种不对等的关系里面，你们两个人谁都不想放过谁，彼此都很累。
