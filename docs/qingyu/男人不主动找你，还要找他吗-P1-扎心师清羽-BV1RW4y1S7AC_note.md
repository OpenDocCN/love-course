# 恋爱沟通课 01：当他不主动找你时，如何应对？💔

在本节课中，我们将学习当男性伴侣不主动联系时，女性可以采取的一种非纠缠式沟通策略。核心是通过间接表达感受来观察对方的反应，从而判断其真实态度。

上一节我们介绍了课程主题，本节中我们来看看具体的操作步骤。

以下是建议在社交媒体（如朋友圈）发布的四句话术，用于表达感受并观察对方反应。

1.  **听不快不慢的歌，想不远不近的人。满心期待的事情，好像从来没有让我快乐。**
2.  **原来有些人，你自以为很重要。**
3.  **我已经不期待你联系我了。我本就不是什么主动的人，却唯独对你上了心。**
4.  **我确实很想你，但是我也不想再打扰你了。不是你不重要，而是我不知道，我在你心里是否重要。**

这些文案的核心逻辑是：**表达失落与疏离感，而非直接质问或索取关注**。其目的是创造一个让对方可以主动回应的“安全”窗口。

如果对方对你有好感，看到这类表达后，他可能会通过评论或私信等方式刷存在感，从而为你们再次开启自然对话创造契机。

本节课中我们一起学习了通过间接、非纠缠的方式表达情感，以测试对方关注度的沟通方法。关键在于保持自尊，避免直接追问，而是通过展示自身状态来引导对方做出反应。
