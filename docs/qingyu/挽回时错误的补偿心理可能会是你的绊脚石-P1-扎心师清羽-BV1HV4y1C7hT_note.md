# 挽回心理学课程 01：警惕“过度补偿”心理 🚫

在本节课中，我们将要学习在感情挽回过程中一种常见但有害的心理状态——“过度补偿”。理解并避免这种心理，是建立健康挽回心态的第一步。

很多人认为失恋意味着失去一切，而挽回则需要付出一切。他们认为分开全是自己的过错，是自己没有履行承诺，并认为只要对方能够回来，自己可以同意任何要求。但你们必须知道，挽回需要两个人的共同意愿。

**选择“乞求”其实已经错了一大半。** 分手后最常见的情绪认知，是认为分手都是自己的错，感觉对不起对方。他们觉得自己只要多做一点，对方就能回来。但你无条件的付出和弥补，根本无济于事，不仅不能让对方回心转意。

反而会让你自己陷入“过度补偿”的漩涡。接下来，我们来详细讲解什么是过度补偿。

## 什么是“过度补偿”？🤔

过度补偿本质上是一种逃避行为。从心理学来说，补偿是一种心理防御机制。当我们无法处理生活中某一方面的失败时，就会把注意力转移到自己擅长的地方。

例如：
*   当我们意识到自己才华并不出众时，会想用“勤奋”和“坚持”来补偿。
*   当我们无法解决生活中的压力时，会用“购物消费”来转移焦虑。
*   当我们处理不了感情失败时，就会想用“弥补对方”来试图挽回情感。

在实际情况下，我们根本不会意识到自己正在“补偿”，甚至可能都没有意识到自己对感情生活产生了误判。只是一门心思地想要补偿、想弥补、想挽回。但这种补偿心理，除了能够满足你自己的内心需求，并不能解决双方情感的具体问题。分手的核心矛盾依然存在，你只是想通过“补偿行为”来逃避面对它。

## “过度补偿”的心理根源 🧠

这种心理和我们的成长经历很相似。童年时期，如果你不写作业、弄坏衣服或犯任何小错误，都可能受到老师或父母的批评。这让我们认为“做错事是可耻的”，只有生活完美才会让我们心安理得。

所以，当我们面对爱情失败时，会因为这个结果而感到羞耻。我们会像童年时向父母老师道歉一样，去向伴侣“赎罪”和“补偿”，却不敢触碰导致爱情失败的真正原因。

那么，过度补偿具体是如何起反作用的呢？

## 为什么“过度补偿”有害？⚠️

首先，过度补偿是一种“边界违规”。边界是区分自我和他人的屏障，是我们保护内在脆弱性、担忧和不安全感的方式。人的成长过程就是在寻找并建立这种边界。青春期的情绪叛逆，就是向父母索要这种边界。

而过度补偿正是一种边界违规。补偿者无法接受被“抛弃”，他们会跨越伴侣设定的安全边界，强行想把对方拉回到自己身边。过度补偿者总是将感情失败归结于自己，这是一种“归因倾向”。简单来说，就是面对负面结果时，会过度简化问题原因。

例如：
*   “生活中不这样做的人就是不道德的。”
*   “我被分手了，一定是我做错了什么。”

在思考失败时，从自己身上找原因是最简单的路径。当恋情出现危机时，伴侣中越依附这段关系的一方，就会越拼命争取。这源于孩童时期形成的强烈“依恋风格”。

然而，跨越彼此边界进行挽回，通常会产生相反的效果。最终，前任会更快地建立新的、更坚固的边界，就像叛逆期的少年一样。补偿者越是强求，伴侣就越会反抗。

## “过度补偿”与“好人综合症” 😇

过度补偿也是“好人综合症”的一种表现。“好人卡”并不只是一个网络玩笑。有这种倾向的人会将自己视为“好人”，并将自己的“善行”视为求爱时唯一可做的事。

因此他们会自我认为：**`我对对方好 + 付出 = 对方应该被感动并回心转意`**。

如果情感出现了裂痕，他们就会认为是自己做错了什么。这导致他们认为分手后，弥补对方是理所应当的事。他们相信感情中的所有问题都可以通过“示弱”和“付出”来解决。所以当对方不遵守这个他们内心的“规则”时，他们常常会陷入手足无措的境地。

## 正确的认知：爱情的本质 💖

在意识到以上这些之后，你必须树立一个核心认知：**容忍和妥协换不回真爱**。

那么爱情的本质是什么？是吸引。而吸引的本质，就是你自己。健康的感情建立在相互吸引的基础上，而非单方面的补偿和赎罪。

本节课中我们一起学习了“过度补偿”心理的表现、根源及其危害。关键在于认识到，挽回不是通过自我贬低和过度付出来实现的，而是需要重建个人魅力与清晰的边界。下一节，我们将探讨如何在分手后停止补偿行为，并开始真正的自我建设。
