# 情感修复教程 第1课：分开半年后复合的概率与策略 📊

在本节课中，我们将从心理学角度分析分手半年后双方的心理状态变化，并探讨此时尝试复合的概率与核心策略。课程将拆解“情绪周期”、“好奇点创造”和“可得性”等关键概念，并提供具体的行动步骤。

## 情绪周期与复合概率分析

从心理学角度看，人的情绪周期一般在**28天**左右。在这个周期内，对方对你的负面情绪会逐渐减弱。

即使是他主动提出分手，只要你们曾经真切地爱过，那么在这个时间段内，他一定会想起你，心里也会产生后悔、失落或者不甘。当然，他的这些行为并不代表他舍不得你，或想跟你重新在一起。

这只是人的欲望起源中必然经历的一个过程。人难免会触景生情，对过去的美好产生莫名的憧憬和留恋，这是非常正常的事。

但如果分开已经过了半年，情况就完全不同了。对于这段感情的失败所造成的创伤，他已经自动愈合了。至少，他已经开始慢慢放下过去，迎接新的生活。

如果你要问，半年后重新在一起的概率大不大，那么可以明确告诉你：**一定没有刚分手时的概率大**。

## 长期分开后的优势与突破口

不过，长期分开也有另一个好处。半年之后，他对你的细节已经忘得差不多了。你在他印象中大部分都是你的优点，因为人都不愿意去存储那些不愉快的记忆。

如果你们之前分开没有本质上的矛盾，当你再次出现时，他会对你产生很大的兴趣。你想重新开始，突破口就在这里。

## 重建联系的三步策略

以下是重建联系并激发对方兴趣的三个核心步骤。每一步都需要精心策划和执行。

### 第一步：创造好奇点

首先你要创造好奇点。例如，你们开始交往时，他印象中的你是一个文静的小姑娘。时隔半年，你已经变得非常活泼开朗，说话风趣幽默。

那么他对你的好奇点就会很多。他会想你的变化很大，会好奇这半年你经历了什么。好奇感由此产生。

### 第二步：策划首次互动

恢复联系后的第一次沟通和第一次见面非常重要，你需要去策划一下。第一步是引导对方与你沟通。

产生兴趣之后，你可以进行第二步：选择见面“抓巧”，或直接通过聊天进行“合理化”。这取决于你的具体情况。这一步主要起铺垫作用。

### 第三步：模糊关系与制造可得性

再到第三步：模糊关系，制造机会，给他一定的**可得性**。

**可得性**不是主动送上去，而是让对方主动来靠近你。这需要很多的沟通技巧。关于“可得性”的详细解释，可以回顾我往期的视频内容。相对于前两步，这一步会难一点。
