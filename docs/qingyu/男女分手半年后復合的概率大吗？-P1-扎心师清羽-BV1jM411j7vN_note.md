## 情绪周期与复合时机 ⏳

上一节我们介绍了复合的基本可能性，本节中我们来看看影响复合的关键时间因素——情绪周期。

人的情绪周期一般为 **28天左右**。这是分手后最佳的复合时期。即使是前任主动提出分手，但你们的过往相处是真真切切的。他会在某个夜晚想起你们的过往，心里难免会产生后悔、失落和不甘。

这解释了为什么有的情侣分手一年或两年后还能复合。经常有朋友咨询说，分手很长时间了，为什么前任还翻看自己的朋友圈，有时也发布一些关于自己的、暗示性的动态。这是否说明他忘不掉自己？

这其实并不能说明什么。因为这个时候，他多半进入了 **28天以后** 的阶段，开始怀念你们之前的美好回忆。但这同时也是他逐渐放下你、放下这段感情的一个过程。

在此期间，他会受到许多不同因素的影响。

以下是可能影响他想法的关键因素：

*   **现实的压力**
*   **新欢的出现**
*   **主观意识的变动**

这些因素都有可能打消他想要重新跟你在一起的想法。

## 复合的真正决定因素 🔑

大家可能认为分手半年后很难复合。但你要相信，时间只是一小部分因素。真正决定复合概率的，一定是分手后彼此的改变。

在我看来，复合其实就是两个人的二次磨合。复合与时间长短没有直接关系。有的人不去解决彼此的矛盾，分手一两天也无法重归于好。有的人在分开后的两年内，重新审视彼此，做出改变，还能够重新开始。

因此，分手后只有两个人都能静下心来，正视这段感情存在的问题，都抱有复合的念头，并且为了修复这段感情而做出改变，才会有机会重新在一起。

怕就怕两个人都没有想好，只是因为一时的寂寞而选择复合。刚复合时可能还会有些惊喜，但彼此身上的性格缺陷没有任何改变，又会回到折磨彼此的路上。这样的复合不过是重蹈覆辙。

## 复合路上的心态与行动 🛤️

同时，在复合的路上，你们切记要摆正心态，切勿过焦过躁。不要因为一时的迫切而吓跑对方。

无论分开多久，也要告诉自己：**只要他还单身，我就有机会让他重新爱上我。**

不要害怕没有方向，也不要无所作为，更不要纠缠、卑微乞求地留在对方的生活圈内。

你要做的是想明白以下几个核心问题：

以下是需要你深入思考的关键问题列表：

1.  你们到底是因为什么分手的？
2.  彼此在这段感情中的优缺点是什么？
3.  如何去改变？
4.  双方的核心需求是什么？
5.  有没有主客观原因影响着你们？
6.  如何改变这个状态？

想明白这些，才是真正影响你挽回概率的必要所在。如果你想要修复这段感情，那么这些问题是你无法避免的。
