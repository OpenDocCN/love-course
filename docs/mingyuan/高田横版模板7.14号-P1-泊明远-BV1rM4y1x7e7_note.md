# 课程：理解“回避态” 🧠 - P1

在本节课中，我们将要学习一个在分手后常见但令人困惑的心理状态——“回避态”。我们将探讨其定义、形成原因以及背后的心理动机，帮助你理解前任为何在分手后表现得冷漠和疏离。

## 什么是“回避态”？🤔

上一节我们介绍了课程主题，本节中我们来看看“回避态”的具体定义。

如果在分手以后，你的前任对你非常冷漠，不回消息、不接电话、完全不理你，这种情况在心理学上被称为“回避态”。

对方会在临近分手或分手后，以完全回避的状态对待你。在分手前产生矛盾时，他可能会疯狂宣泄情绪，但冷静一段时间后再提出分手。或者，在发生矛盾后，他一直处于冷落你的状态。

分手之后，他既不愿意聊，也不愿意见面，完全处于一种逃避状态。他这样做是为了保证自己不受伤害。

## “回避态”的形成原因 🔍

理解了定义后，我们深入探讨导致这种状态的两个核心原因。

### 1. 环境因素导致

很有可能他目前面临巨大压力，无论是工作还是学业。恰巧此时你们的感情出现了裂痕，于是他选择了自我逃避。

在他们面前是一个两难选择。当面对事业压力时，他会把原因归结于自己，所以先屏蔽外界声音，让自己缓一缓。

如果压力来自家庭内部，他就会认为原因在你身上，从而选择以分手来规避你，同时关闭心门，不再让你的举动动摇他的感情。

### 2. 不知如何面对

这个原因需要从两个方面看。

第一，可能在之前的感情里，你留下了太多负面影响，他已疲惫不堪，不想再与你有任何瓜葛。他觉得再沟通感情问题也会重蹈覆辙，为避免再次受伤，不如什么都不说、什么都不做，直接回避。

第二，你们两人现在没有一个合理的身份交流。你是他的前任，而之前矛盾形成的刻板印象仍在，这让他不知如何面对你。因此，若想交流，必须合理化你们之间的关系。

## 导致分手的深层动机 ⚙️

很多时候，他选择分手也是出于以下两个原因。

以下是具体的动机分析：

1.  **你的行为不稳定**：他们讨厌思维和情绪都不稳定的人。如果你一会儿这样一会儿那样，他根本猜不透你的想法。如果在相处过程中，你斤斤计较，把一点小矛盾都上升到很高的高度，这也会让他们很疲惫。
2.  **姿态失衡与过度付出**：因为担心失去他而放弃自己的姿态，为他付出很多，这也会让他觉得不自然，造成心理压力，从而导致他提出分手。
3.  **你施加的压力**：压力是导致分手的核心原因。例如：
    *   `你太黏人`，要求他无时无刻的陪伴。
    *   `你太强势`，总想主导关系、控制他、打压他。
    *   `你太活跃`，在他疲惫时总想拉着他到处玩。在他看来，这都算是在逼他。

同时，他们这种人又很能包容，经常处于默默忍让的状态。所以你会发觉，每次有矛盾时，好像都是你在抗拒到底，你也觉得这种相处方式没问题。而他突然有一天毫无征兆地提出分手，让你搞不清原因，这就是“回避态”。

---

本节课中，我们一起学习了“回避态”的概念、其形成的两大原因（环境压力与不知如何面对），以及导致关系破裂的深层行为动机（不稳定、姿态失衡、施加压力）。理解这些是应对此类情况的第一步。
