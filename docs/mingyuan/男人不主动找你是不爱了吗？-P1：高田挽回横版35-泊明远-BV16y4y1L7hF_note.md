# 💔 情感课程 P1：如何判断对方是否在乎你？

在本节课中，我们将学习如何通过对方的行为模式，理性判断其情感投入程度。我们将分析“不主动”背后的潜在含义，并帮助你建立清晰的认知框架。

你心里其实早就有答案了。男人一旦动心，就会恨不得把世界上最好的东西给你。

世界上没有不懂得聊天的男人，只有不想聊天的男人。没有不会哄人的男人，只有不愿意哄你的男人。也没有不会挑礼物的男人，只有不愿意给你送礼物的男人。

那些你期待的事情，不是他做不到，而是他不想做。不主动找你，仅仅是因为他不想而已。

因此，如果你感觉不到对方的爱，无需质疑。他就是没有那么喜欢你了。

爱情最可悲的不是分开，因为分开至少会有一个结果。最怕的是爱了这么久，你付出了一切，到最后连一个像样的交代都没有。

本节课中，我们一起学习了如何通过行为而非言语来判断对方的情感投入。核心在于认识到：**“不主动” = “不想”**。当你的感受持续得不到回应时，这本身就是一个明确的答案。
