在本节课中，我们将要探讨一个普遍的情感困惑：如何看待一段看似“没有结果”的感情。我们将分析“结果”在感情中的真正含义，并思考过程本身的价值。

很多人都有一个疑问：一段没有结果的感情还要继续吗？那么，什么才算是有结果的感情呢？人活这一世，最终的结果是死亡。如果只追求最终结果，那么我们活着又是为了什么呢？因此，我们追求的往往是一个过程，而非那个终极的“结果”。感情的结果就是结婚吗？

结婚后，你仍然需要面对各种家庭问题。如果处理不好，也可能导致感情破裂。那么，这能算是真正的“结果”吗？反过来，我们直面现实地说：如果你断定这段感情没有结果，开始了你或许会有遗憾，但如果不开始，你更不会后悔。

那么，这段感情不会有结果，但没有结果就完全没有价值吗？你要知道，你在一段感情中成长的情商和恋商，都在教你自己如何更好地爱别人和爱自己。终其一生，我们想要的，不过是一个可以抵抗岁月漫长、共同面对人生战场的伴侣。

你更需要知道，**先学会爱自己，才能学会爱别人，也才会有人来爱你**。你在这个世界上，并不是为了结婚和繁衍后代，而是为了看这个世界：花是怎样开的，水是怎样流的，太阳何时升起，夕阳何时落下。
