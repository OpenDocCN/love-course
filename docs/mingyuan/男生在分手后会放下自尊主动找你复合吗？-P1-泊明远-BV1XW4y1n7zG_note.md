# 男生在分手后会放下自尊主动找你复合吗？-P1-泊明远-BV1XW4y1n7zG_note

在本节课中，我们将从心理学角度探讨一个常见问题：分手后，男生是否会放下自尊主动寻求复合。我们将深入分析其背后的心理动机，并为你提供清晰的判断思路。

## 核心观点：答案是肯定的 ✅

分手后男人会放下自己的尊严去找你吗？会的。我站在男生的角度，可以非常肯定地告诉大家，不管什么类型的男生，也不管他有没有新欢，分手后都有可能放下自己的尊严去找你。

## 行为动机：并非出于卑微或感动

但他这么做并不是卑微地讨好你，也不是因为感动或者良心发现，看到你之前为他付出。而是他在权衡利弊之后，为了确保自己能够不后悔。

## 关键理论：未完成情节 📚

心理学上有一个非常著名的词，叫**未完成情节**。它指的是我们天生就有一种把处理事情有始有终的驱动力。在事情尚未处理完的时候，潜意识里会不断地提醒我们去完成它。

上一节我们介绍了“未完成情节”这一核心概念，本节中我们来看看它在感情中的具体表现。

## 在爱情中的应用

同理，在爱情里面，我们之所以会对一段失败的感情耿耿于怀，甚至做出一些难以理解的事情，并不是因为我们有多让对方喜欢。而是本质上我们没有办法接受一段不完整的感情，所以我们才总是停留在过往的需求里，沉溺在过去的行為模式里。

## 一个生动的比喻

打一个比方，你看到一张纸上有一个缺口，你会下意识地把它补上，让它成为一个完整的图形。这其实就是人类的一种本能，一种倾向于事物完整性的本能。

## 执念的根源在于自身

而我们每个人都迫切地希望，在生活当中每一件事情，我们都有始有终。很多时候，我们持续地对一个人或对一件事念念不忘，并不是在于对方，而在于我们自身。害怕是因为我们自己不够勇敢，而失去了本该很美好的东西。

## 压抑与转移无效

很多人在分手之后面对种种的不甘和悔恨，都会选择压抑或者转移。但事实证明那是没有用的。

以下是关于未完成愿望的阐述：
*   对于那些没有结果的事情，或者是没有被满足的愿望，都会被我们带到日后的生活里。
*   这也是一种本能，我们是没有办法去对抗的。
*   而我们唯一能做的就是去完成它。

## 回忆与联系的真正目的

所以为什么有那么多人在分手之后，都去回忆过去呢？去联系过去？难道真的是非这个人不可吗？肯定不是。他们想要的、执著的，都不过是一个结果。那无论这个结果是好坏都没关系，只要给他一个过程，然后慢慢去适应，这个事就过去了。

## 给你的行动建议 🎯

所以说在面对前任主动找你的时候：
第一，你要擦亮眼睛。
第二，更要抓住时机。

那以此为突破口，说不定你们的结局，就变成了白头到老。
