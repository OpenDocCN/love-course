# 分手后挽回教程 第1课：当他删除你后，如何发送三条消息 📩

在本节课中，我们将学习一种特定的沟通策略。当分手后对方删除了你的联系方式（如微信），你可以通过发送三条结构化的短信，来引导对方主动联系你，并表达你的感受与释然。

上一节我们介绍了课程的整体目标，本节中我们来看看具体的消息内容与设计逻辑。

## 第一条消息：表达受伤与信任的落差

这条消息通过比喻，传达出你因信任被辜负而感到的受伤情绪。

```
鱼那么信任水，水却把鱼煮了。
树叶那么信任风，风却把它吹落了。
我那么信任你，而你却把我给伤了。
```

## 第二条消息：转换视角，归于自身

在表达受伤后，这条消息进行视角转换，将责任归因于自身或外部因素，而非直接指责对方，这有助于降低对方的防御心理。

```
后来我才知道。
把鱼煮的不是水，而是火。
吹落树叶的不是风，而是秋天。
伤我的不是你，而是我的自作多情。
```

## 第三条消息：表明决定与释然

最后一条消息旨在表明你已做出决定，准备放下。这种带有一定决绝意味的释然，有时反而能引发对方的关注或反思。

```
我终于决定不等了，把爱而不得的你归还于人海。
就当风没吹过，你没有来过，我没有爱过。
```
