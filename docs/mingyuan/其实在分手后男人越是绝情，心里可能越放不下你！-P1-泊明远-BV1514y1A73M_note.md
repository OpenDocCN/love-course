分手后最令人难过的，往往是男性突然的冷漠与绝情。明明曾经相处甜蜜，分手时他却干脆利落，不给任何机会，拒绝沟通，甚至删除拉黑联系方式。

如果他不那么爱你，本可以平和分手，甚至提议继续做朋友。因此，越绝情的男人，其实可能越爱你。那么，他为何选择绝情呢？

上一节我们提到了绝情行为可能与深藏的情感有关，本节中我们来看看具体有哪些心理动机。

3.  **避免纠缠带来的压力**
