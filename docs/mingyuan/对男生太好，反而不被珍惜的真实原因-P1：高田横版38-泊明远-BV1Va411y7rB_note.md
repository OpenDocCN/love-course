# 对男生太好，反而不被珍惜的真实原因-P1：高田横版38-泊明远-BV1Va411y7rB_note

在本节课中，我们将要探讨一个常见的情感困惑：为什么在亲密关系中付出越多，有时反而越不被对方珍惜。我们将从心理学的角度，分析其背后的根本原因，并帮助你理解健康关系的基础。

很多女性心中都有一个疑问：为什么对爱人付出越多，反而得不到珍惜，甚至会被冷暴力分手。这真的是因为“得到了就不重要”吗？现在，老师将多年的咨询经验告诉你。

我们先说结论：你对她很好却不被珍惜。

根本原因并不是你对她太好，而是你丢掉了自我。想想看，在对方已经触及你底线的时候，你是不是没有直接告诉对方，而是选择自己默默流泪？在她对你冷暴力的时候，你是不是放弃了反抗？

反而去低姿态地认错道歉，可怜巴巴地求她不要离开。这些都是丢掉自我的表现。

知道为什么你会这么做吗？你可能会说是因为你没得选，你太害怕失去她了。但其实，是你给自己的“标价”太低。

你认为爱情里只要对对方好，对方才会爱自己。你甚至不顾自己的感受，做出一定程度的牺牲。说到底，是你内心深处觉得自己不值得，所以你才会过度付出，降低姿态来向对方“讨”爱情。

而最后的结果就是：**你对自己的感受不尊重，导致了她对你的不尊重**。所以你的退让、妥协，都变成了理所当然。

爱的基础是尊重。而对方尊重你的基础，是你能够百分之百地尊重你自己。如果你都不认可自己的价值，那你必然在这段关系中处在低位。

我是泊明。如果你现在也正在经历这样的情感困惑，那可以联系老师来帮你。
