# 情感修复教程 第1课：分手后的心态博弈 🧠

在本节课中，我们将要学习分手后一个关键的心理策略：如何通过“不主动联系”的行为，影响前任对你的判断，为可能的修复创造有利条件。

## 核心心态：城门与防备 🏰

上一节我们介绍了课程主题，本节中我们来看看一个核心比喻。如果一扇城门打不开，无论撞击多么猛烈，人都无法进入。进城的前提是城门打开。

在情感语境中，这个比喻可以翻译为：**对方因你的纠缠（骚扰）产生了防备心理**。此时他的心门对你关闭，并且保持高度警惕，防止你突然回头。在这种情况下，任何直接的“进攻”（联系、挽回举动）通常都是无效的。

## 关键策略：静待时机，制造错觉 ⏳

理解了对方处于防备状态后，我们来看看应该采取什么策略。他也一定会有松懈（打瞌睡）的时候，趁虚而入才是真正的机会。

因此，当前阶段不要主动去找他，不要联系他。这个行为的目的是：**让他觉得你已经放弃了**。这种“你已经彻底放下”的错觉，会降低他的戒备心，这对于我们而言是最有利的局面。

以下是执行此策略的两个要点：

*   **彻底断联**：停止一切形式的主动联系，包括社交媒体互动。
*   **保持一致性**：你的沉默和行为需要保持一致，以强化“已放下”的印象。

## 总结与支持 🤝

本节课中我们一起学习了分手后初期的心态博弈。核心在于理解对方的防备心理，并通过“不主动联系”的策略，制造你已经放下的假象，从而为关系可能的转变创造空间。
