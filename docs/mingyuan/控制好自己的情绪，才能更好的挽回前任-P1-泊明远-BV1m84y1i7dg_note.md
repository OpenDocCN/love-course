## 🎯 情绪稳定的核心价值

你只要能够做到情绪稳定，那么你躺着就能赢。

那些能够成功挽回的人，他们都有一个共同的特点，就是他们的情绪极其的稳定。

而我们大多数普通人，都是脾气不太好的，那么在一段关系里面，容易被对方牵着鼻子走，搞到最后非常容易患得患失。

上一节我们概述了课程目标，本节中我们来看看情绪稳定的具体表现。

演员杨幂曾经说过一句话：“我已经戒掉了情绪，不管天大的事情，都会跳脱中间这个情绪化的过程。我时刻提醒我自己，给我一天或者说两天的时间，让它过去。”

所以说能够控制情绪的人，他们强就强在了，无论对什么事情上面，都不会手足无措、患得患失。

## 🔄 如何实践情绪管理

放不下的事情，数个一二三就让它放下，也就过去了。

那些没价值的事情，该丢就丢，一定不要让它发酵。

一定不要让它们影响到我们的决策和判断，也不要让任何人去左右我们的情绪。

如果说前任可以随时操控你的情绪，让你哭、让你笑、让你生气、让你崩溃，那真的是一件很恐怖的事情。

所以我们一定要学会去掌控自己的情绪。

掌握了情绪管理的基本理念后，以下是具体的行动原则：

1.  **掌握主动权**：掌握主动权的人，才能掌握最后的结局。例如，你刷抖音不停地去刷，或者看电视不停地换台，这个遥控器一定是掌握在自己手里的。
2.  **将情绪武器化**：你的情绪就是你的武器。你可以发火，但是你每次的发火都要有目的才行。你是为了得到一些东西，或者说得到你应有的利益，才会去做的。这样才会有人为你的情绪去买单。
3.  **以静制动**：而当你不动了，才会有其他人因为你的淡定而抓狂。他才会好奇：“为什么你突然就没反应了呢？那是不是发生了什么我们不知道的事情？”

这样的你，才能够达到复合的目的。
