# 分手后挽回教程 01：如何为前任搭建“降低排斥”的台阶 🪜

在本节课中，我们将要学习分手后挽回的一个核心策略：如何通过“给台阶”的方式，降低前任对你的负面情绪和排斥心理，为后续的沟通和关系修复创造可能。

所有的分手都有其原因。分手后若想挽回对方，通常源于两种心理：一是你发现自己确实深爱并需要对方；二是你无法接受对方离开的事实。

然而，对方表现出绝情态度，至少说明他当下对你的印象完全是负面的。正因如此，此时你向对方展示再多的好也无济于事。因为此时此刻，他会想出各种理由说服自己你不好，从而告诉自己，与你分开是正确的选择。

## 为何需要“台阶”？🤔

上一节我们提到了分手后的负面印象，本节中我们来看看如何破解这种僵局。很多时候，当你的努力看似无效时，你需要为彼此搭建一个台阶。

如果你想让他质疑自己分手的决定，你表现得越放松、越缓和，对方才有可能慢慢降低对你的抵触和戒备心理。

以下是搭建“台阶”的一个具体做法示例：

> 既然我们都努力过了，走不到最后也没有关系。大家好聚好散就行了，以后还是朋友。

这种做法，比你任何的道歉、哭闹都更加有效。关键在于，你需要主动解开施加在对方身上的压力。

## “台阶”带来的积极变化 🔄

只要他不抵触、不排斥你，那么当他再次想到你时，大概率就不会只想到你的不好。并且，你这种释怀的态度，反而会让对方忍不住去琢磨你是否真的放下了，从而对你进行情绪投资。

大家应该都知道，情绪会引导行为。如果前任开始忍不住琢磨你，大多数会出现两种情况。

以下是两种可能的情况：

1.  在你主动联系的情况下，两个人之间的联系会越来越好。
2.  在你没有主动找对方的时候，他甚至会主动来找你。这时他可能是来刷存在感的，也可能对你有其他情感需求。

但这其实都是好的现象。所以说，你当下的心态，以及你能否为彼此搭建一个台阶，就变得至关重要。

---

本节课中，我们一起学习了分手后挽回的第一步：通过“给台阶”来降低排斥。核心在于**以放松、缓和的态度接受现状，解除对方的压力**，从而扭转负面印象，引导对方对你进行情绪投资，为后续的互动打开窗口。
