# 分手后如何调动前任情绪 - 课程01：制造不可得性 🎭

在本节课中，我们将学习分手后如何通过制造“不可得性”来调动前任的情绪。这种方法的核心在于让对方产生失去感与不甘心，从而为关系的转变创造机会。我们将详细探讨其原理、适用场景以及具体的操作步骤。

## 核心概念：制造不可得性与失去感

上一段我们提到了调动前任情绪的目标，本节中我们来看看其核心机制。与其说给前任制造颠覆感，不如说关键在于制造**不可得性**。

**不可得性** 在复合过程中的效果和应用是：让对方感觉你已不再轻易可得，从而激发其失去感与征服欲。

有看过之前视频的应该知道，在挽回过程中，我们需要给前任制造**失去感**。制造失去感就是调动前任不甘心的情绪。这种方式更适用于无吸引力的分手，或者遇到了情感经验丰富的“海王/海后”。

## 适用前提与核心策略

首先明确一个大前提：如果是无吸引力的分手，或者遇到了渣男/渣女，那么想要复合，**逆袭是你唯一的选择**。

开始行动后，你需要利用好自己的社交状态，并与对方建立起**弱联系**。这种弱联系不需要你主动发起，而是通过社交状态的展示来完成。

## 操作步骤详解

以下是利用社交状态制造不可得性的具体步骤列表：

1.  **展示全新的一面**：通过社交状态，展示他完全不了解你的另一面。
2.  **控制出现节奏**：消失一段时间后，再开始更新你的新状态。
3.  **颠覆对方预期**：让他觉得分手后，你并非如他想象中那般颓废，反而是如释重负。
4.  **引发对方怀疑**：就是要让他去怀疑，提出分手是不是一个错误的决定。
5.  **打破固有形象**：为什么明明是他甩了你，你却不像他想象中一样卑微求复合？
6.  **展示积极生活**：反而很开心的去接触其他异性，扩大自己的社交圈。

## 策略效果与主动权转换

当以上步骤执行后，他对你的征服欲就会被调动，就有很大的可能会来撩拨你、试探你，观察你到底还在不在意他。

那么到了那个时候，我们就可以顺势把**主动权**，掌握在自己的手里。
