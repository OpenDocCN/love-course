# 情感分析课 P1：断联博弈论 🔍

在本节课中，我们将要学习一个常见的情感问题：当两个人都不主动联系时，最终谁会先忍不住打破沉默。我们将从行为模式和心理动机的角度，分析其中的规律与核心原理。

## 核心结论：男生先忍不住

上一节我们介绍了本课探讨的核心问题，本节中我们来看看具体的结论。

如果双方断联，最终先忍不住联系对方的人，通常是男生。这个结论没有例外情况。其背后的逻辑在于，只要男生心里仍然在乎对方，他就无法长期克制主动联系的冲动。男生的思维模式通常较为直接，不会过度迂回。

**核心逻辑公式**：
```
IF (男生心里有你) THEN (他会主动联系)
```

因此，如果断联后男生始终没有主动联系，那就传递出一个清晰的信号：你对他而言已经不再重要。认识到这一点至关重要，你需要避免继续沉浸在不切实际的期待中。

## 例外情况：你的行为如何影响结果

上一节我们得出了普遍结论，本节中我们来看看唯一能改变这个结果的特殊情况。

存在一种例外：即使男生非常爱你，他也能忍住不找你。这种情况的发生，完全取决于你在“断联期”内的行为。

以下是导致这种例外发生的关键行为列表，这些行为会向对方传递错误信号：

*   你在断联期间主动找他聊天。
*   你持续关注他的社交动态。
*   对方给你一个微小的互动（例如点赞），你便立刻热情回应。

这种**卑微的姿态**会被对方敏锐地察觉。你的行为非但不能促使他回头，反而会带来两个负面效果：
1.  你为他提供了情感能量，缓解了他的失落感。
2.  你彻底打消了他因“失去感”而产生回头欲望的念头。

## 课程总结

本节课中我们一起学习了“断联博弈”中的基本规律。我们明确了在通常情况下，在乎你的男生会先忍不住联系。同时，我们也分析了唯一能打破此规律的关键：即你在断联期间表现出卑微和主动的需求感，这会消除对方的危机感，使其不再有主动的动力。理解这些行为背后的心理动机，有助于你更清醒地看待一段关系。
