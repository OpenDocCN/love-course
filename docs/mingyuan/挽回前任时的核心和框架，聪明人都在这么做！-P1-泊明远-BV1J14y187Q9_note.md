## 概述：复合的通用框架

很多人都想知道分手后复合的框架是什么。无论何种形式的分手，都存在一个通用的核心框架。掌握这个框架，你们复合的几率就会无限增加。长远来看，按照以下四个步骤进行是关键。

## 第一步：心态建设 🧘‍♂️

为什么要把心态放在最前面？因为心态虽然看似不起眼，但成功与否很大程度上取决于你的心态是否稳定。使用再多的技巧，如果心态不稳，效果也会大打折扣。心态建设的核心在于以下几点。

以下是心态建设的四个核心要点：

1.  **控制情感需求**：在任何时候都不要暴露你的需求感。不要让对方觉得你很着急想要挽回他，那样只会把他吓跑。
2.  **理性复盘感情**：你需要认真复盘你们的感情，反思自己。在分手初期，你在对方眼里留下的印象是什么？因此不要主动联系对方。如果去打扰对方，会加重他对你的厌烦和不适感。这段时间我们需要等待，给他时间冷静。我们要做的，是从相识、相知、热恋，到平淡、争吵，回想你当时说了什么、做了什么，是否会伤到对方的心。
3.  **充实与提升自我**：感情中的核心竞争力是你的自身价值。搞钱是第一位的，感情永远不要放在搞钱的前面。一段好的感情必须有物质基础作为保障。你想要吸引他，让他后悔离开你，就必须提升自我。人都是趋利避害的，当你变得更好时，曾经拥有你的人也会产生强烈的不甘情绪，这就是人的损失厌恶心理。
4.  **保持高姿态重新吸引**：当你完成上述三个步骤后，会重新与他取得联系。我们要做的，是在聊天中展示你的改变和价值，赢得他的好感，争取打开复合的窗口。但在初步复联时期，要知道不要聊个没完，不要聊感情，经营好自己的人设。

分手结束后，你通常有两种心态：要么自责，要么指责对方。这在心理学上称为**归因模式**：**内归因**与**外归因**。

*   **内归因**：将感情失败归结于自身。这是一种积极的归因模式，有助于我们的反思和成长。
*   **外归因**：将感情失败归结于他人或外部因素。

## 总结与核心原则

最后强调一点，无论你们因何种原因分手，结果都是分开了。你去找前任复合，要明确一点：**绝对不是哭求对方别离开你**，而是让对方在分手后发现，你才是对他最好的选择。这样的复合才更具优势。
