# 分手后断联，发社交圈对复合有帮助吗？ - 课程P1：心态建设篇 🧠

在本节课中，我们将要学习分手后断联期间，发布社交动态的核心心态与原则。理解这一点，是决定你后续所有行动是否有效的关键。

我们都知道大多数的分手都是需要断联的。

那么断联后到底要不要发社交圈呢？

我告诉你，你的一切带有目的性的、和小心思的朋友圈都不要发。

**核心公式：**
`发布动机 = 纯粹自我表达` ✅
`发布动机 = 刻意展示给前任` ❌

如果你的动机只是为了让前任看到你过得很好，那么在这场分手后的心理博弈里，你就已经输了。因为你的心态就是错的，这证明你还没有到能够坦然发朋友圈、展示自己的时候。

试想一下，如果他对于你精心准备的动态无动于衷，你是不是又要陷入不断的猜测和伤感？万一他只是随手点了个赞，你是不是又会变得心绪不宁，开始幻想复合的可能？

这个时候，不管他做什么，都能轻易影响到你的情绪。这根本就是一个赔本的买卖，你为什么要这样做呢？

---

那么，什么时候你才能发动态呢？

那就是当你发自内心地觉得：我不管你看不看，我就是想发。我高兴，我乐意，这跟你没有半毛钱的关系。

**心态代码示例：**
```python
if 发布动机 == “自我表达” and 情绪状态 == “稳定独立”:
    print(“可以发布”)
else:
    print(“请暂停发布，继续自我调整”)
```

当你真正拥有了这个心态，那么恭喜你，你就可以随意发了。而这个时候，你通过社交圈散发出来的气场和感觉是截然不同的，甚至会颠覆前任对你固有的看法。

只有到了这个阶段，他才会开始感到慌张，害怕彻底地失去你。
