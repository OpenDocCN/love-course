# 情感修复课 01：前任不主动找你，就等于放下了吗？ 💔

在本节课中，我们将探讨一个常见的分手后困惑：前任不主动联系，是否意味着对方已经彻底放下。我们将通过换位思考和分析情感中的“输赢”观念，帮助你理清思路，为自己的感情决策负责。

---

昨天晚上，一位女生私信我一个问题。她说，如果分手后你的前任没有主动找你，是不是就等于放下你了。而当你产生想要去找对方的冲动时，为了维护自己的底线和自尊，你也要强忍着这股冲动，坚决不去找对方。

因为谁先低头，谁就一定会输。她问我是不是这样的。

其实这个问题很好理解。我们先假设“不主动找你等于放下你”这个条件是成立的。这时，我们站在对方的角度上去想一下，进行换位思考。当你忍着没有去找对方的时候，她是不是也会觉得你也放下她了呢？

第二个问题是，当你想要去找她的时候，要忍住这个冲动。但在我看来，你们两个人，一个以为对方会挽留，一个觉得对方根本就不会离开。你不觉得这样的情况很苟且吗？

上一节我们探讨了“不主动联系”可能造成的双向误解，本节中我们来看看关于“主动”与“自尊”的纠结。

你要不要去找一个人，要不要去挽回一个人，都是你自己的事情，也是你自己要做出的决定。你要为你的人生负责，为你的幸福负责。但感情这件事没有明确的输赢，你也没有必要非做任何一件事都跟着对方的节奏走。

你看到的只是对方没有找你，但你不知道她到底有多么想你。当你特别想她的时候，你有做到言行一致吗？所以，不要给自己的懦弱找理由、找借口。如果你想要放弃，那就大大方方地放弃。如果你还想挽回，那就勇敢地去做。

因为，**真诚才是挽回前任最大的套路**。

---

本节课中，我们一起学习了如何理性看待前任的“不主动”。关键在于，不要将感情简单归结为“谁主动谁就输”的博弈，而应听从自己内心的真实想法，并为自己的选择负责。感情中没有绝对的公式，但真诚永远是通往理解的桥梁。
