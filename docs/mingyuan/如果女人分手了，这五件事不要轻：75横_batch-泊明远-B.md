**健康关系 = 相互尊重 + 平等沟通**

## 2. 停止情感暴露与信息追踪 🚫

第二，不要发伤感朋友圈，也别去打听他的任何动态。你以为这样就能唤起他对你的关注吗？这个比中彩票的几率都还要低，这基本上就是属于自我欺骗。

第四，千万不要轻易的接受他求复合，也别心软。记住，你不是收破烂的。他想求复合可以，那就让他拿出点诚意来，不能光说不练。

## 5. 避免不必要的见面 🚷

第五点，也是最重要的一点，分手以后千万不要轻易的和他见面。他是别有用心的，但你更不能揣着明白去装糊涂。
