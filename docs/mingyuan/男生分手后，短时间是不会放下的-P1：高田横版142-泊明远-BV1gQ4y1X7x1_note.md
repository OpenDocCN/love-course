## 核心心理机制

上一段我们提到了分手后的普遍现象，本节中我们来看看其背后的核心心理机制。

男生在分手后，短时间内不会彻底放下。原因是人都有感情。虽然双方当前关系不佳，他感到疲惫和厌烦，但这些负面情绪不会否定曾经的过往。负面情绪只会将那些美好甜蜜的回忆暂时冻结。

**公式：当前感受 = 近期负面情绪 + 被冻结的过往美好**

当真正分开后，如果没有继续创造更多的负面情绪给他，那么曾经的美好甜蜜回忆会重新浮现。当他评估认为“在一起的美好”大于“在一起的痛苦”时，他就会重新考虑回头。

**决策条件：IF (美好记忆 > 痛苦记忆) THEN 可能复合 ELSE 享受单身**

反之，如果痛苦记忆持续占上风，他会选择继续享受单身生活。

## 男性情感处理特点

了解了基本机制后，我们进一步探讨男性处理情感的特点。

大部分男生在感情上是专情的。他们的情感处理方式往往是内敛的。

以下是男性常见的两种情感处理方式：
*   将情绪藏在心里，不对外表达。
*   不轻易拿出内心深处的感受进行分享。

正因为心里积累了如此多的不开心，他们不可能轻易地放下这段感情。这种内压式的处理，延长了情感恢复的周期。
