# 分手后想复合，应该从哪里开始 - 课程01：心态建设 🧘‍♂️

在本节课中，我们将要学习分手后想要复合时，最核心且首要的一步：心态调整。很多人误以为第一步是反思或改变，但我们将揭示一个更根本、能让你轻装上阵的起点。

## 第一步：说服自己，而非改变自己

复合的第一步是什么？有些人认为是心态建设、反思错误、改善对亲密关系的认知。但这些都不是第一步。第一步并非要求你去反思、努力和改变，而是要去说服自己。你需要说服自己的内容是：**复合不了也没有关系**。

让我们仔细体会一下这两种心态的区别。

*   **“我必须挽回，不能失败”**：在这种心态下，你感受到的是一种深入骨髓的压力。你的一举一动、一言一行都透露出“沉重”二字。任何微小的努力和改变，都会与“复合”这个终极目标深度绑定，压力巨大。什么都还没做，心就已经累得不行，哪里还有力气去真正行动呢？
*   **“挽回不了也没有关系”**：切换到这种心态，你是否感觉松了一口气？心中的石头稍微放下了一点。在这种状态下再去行动，是否会轻松很多？此时，你的行为会变得纯粹：提升就是提升，沟通就是沟通，无需背负太多额外负担。

因此，第一步要做的，不是让自己更积极、更努力、更迫切，而是**先停止内耗**。

## 建立“最坏也就这样”的心态

上一节我们介绍了心态转换的核心，本节中我们来看看如何具体建立这种轻松的心态。以下是两个关键点：

**1. 不主动破坏就是进步**

通常情况下，只要你**不作、不闹、不捅娄子**，你们的关系最多就会维持现状，不会继续恶化。换句话说，如果复合的定义是“存在一丝可能性”，那么，只有你的“作死”行为才会导致彻底失败。不作所谓的“挽回”操作，反而不会失败。所以不要害怕，在大多数情况下，“不作为”相对于“做错事”是更佳的选择。你一定要建立起“**最坏也就是现在这样**”的心态底线。

**2. 放下对目标的执念**

我们从小接受的教育是“要定一个目标”，并要不断提醒自己。但在挽回这件事上，反复提醒自己“我要复合”真的有用吗？这只会让你更加焦虑。挽回这么大的事情，非得时时刻刻想着才行吗？不想着就会忘记吗？

其实，你只需要在当下的基础上，**一点一点地往前走**就好了。每天睡觉前，可以想想今天有没有获得什么。但请注意，这个“获得”不是指在挽回关系中的进展，而是**对你自身的获得**。

## 关注过程，而非结果

从被全网拉黑，到重新成为好友，再到能心平气和地说上几句话，再到对方可能主动联系你……如果你一直只盯着最终目标，就会充满挫败感。但如果你回头看，就会发现**自己已经走了很多步**。

所以，索性**走到哪算哪**，享受这个往前走的过程。人就是在这种一步一步前行的路径上，变得强大起来的。一旦你自身变得强大，那个你无需刻意紧盯的目标，反而会迟早实现，可能比你想象的还要快。

而这些——**接纳自己可能会失败，但依然对自己保有期许，不计较一时成败，只是慢慢地往前走**——才是挽回真正意义上的第一步。只有这样，挽回才能从一个望不可及的沉重梦想，变成一个你自然而然实现的结果。

---

**本节课总结**：本节课我们一起学习了复合旅程的起点。我们认识到，最关键的第一步并非急于改变或行动，而是进行深刻的心态调整：说服自己“复合不了也没关系”，以此卸下沉重压力；建立“不破坏即进步”的底线思维；并学会关注自我成长的每一步，而非执着于最终结果。这种轻松、专注于过程的心态，是后续所有行动能够有效开展的基础。
