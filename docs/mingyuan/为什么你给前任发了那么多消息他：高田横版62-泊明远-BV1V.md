# 为什么你给前任发了那么多消息他：高田横版62-泊明远-BV1V

在本节课中，我们将要学习为什么在分手后，频繁地给前任发送消息往往得不到回应，以及如何停止这种无效的“死缠烂打”，为双方创造一个冷静反思的空间，从而为可能的挽回打下基础。

## 🤔 为什么消息石沉大海？

你给前任发送了大量消息，对方却选择视而不见。你付出了满腔的热情与期待，换来的却是对方的冷漠与回避。

上一节我们介绍了沟通无效的现象，本节中我们来看看其背后的核心原因。不要简单地责怪对方心狠，这往往是因为你所采用的沟通方法存在根本性的错误。

关键在于，你们之间先前积累的误会是否已经解除，存在的矛盾是否得到了真正的解决。如果这些根本问题依然存在，那么无论对方是谁，都很难立刻冷静下来，心平气和地与你重新相处。

因此，若想让他回到你的身边，必须首先改变策略。

## 🛑 关键一步：停止死缠烂打

理解了问题所在后，我们来看看具体的改进方法。首要且最关键的一步，就是立即停止一切形式的“死缠烂打”。

以下是实施这一步骤的具体方案：
*   **设定冷静期**：首先，为自己设定一个为期一周的“锻炼”期。在这段时间里，主动停止所有单向的信息轰炸。
*   **创造反思空间**：这个冷静期的目的，是给予对方，同时也给予你自己，一个充分反思和情绪平复的时间与空间。
*   **等待情绪平复**：只有当双方的情绪都从剧烈波动中逐渐平静下来后，才能理性地去思考和规划下一步的挽回行动。
