# 情感分析课程 01：识别前任等待复合的信号 🕵️♂️

在本节课中，我们将学习如何通过观察特定行为模式，来判断你的前任是否可能在等待你主动提出复合。我们将逐一分析五种关键表现，帮助你更清晰地理解对方的潜在意图。

上一节我们介绍了课程的主题，本节中我们来看看具体有哪些行为信号。

以下是前任可能等待你复合的五种常见表现：

1.  **联系方式未完全切断**：对方没有拉黑或删除你全部的联系方式，例如微信、电话等。
2.  **绝情但仍有回应**：分手时态度坚决，但当你主动发送消息时，对方仍然会予以回复。
3.  **保持有限沟通**：对方偶尔会接听你的电话，或者回复你的短信。
4.  **见面邀约态度犹豫**：当你提出见面请求时，对方可能会找理由拒绝，但在拒绝过程中表现出明显的犹豫。
5.  **未开始新恋情**：分手后，对方并没有立即开始一段新的感情关系。
