## 分手原因剖析

上一节我们概述了课程主题，本节中我们来看看这类分手的具体成因。失望累积性分手的正确挽回思路，首先不要去道歉，也不要去纠缠。因为你们的问题，已经不是道歉就可以解决掉的。你们的主要问题已经变成了信任崩塌。通常是因为生活当中的一些小事、相处方式等等，他接受不了。

他也跟你说过很多次，结果要么是你不重视，根本不改，要么就是改个两三天，然后又变回去。情绪积压的多了，你们就没有办法像正常人一样去沟通，不是冷战就是冷暴力。原本他想着每对情侣没有矛盾，我也还爱着你。

要不这件事情我就退步一下。但是再有一次重大矛盾发生的时候，就会彻底爆发，那你们就会分手。而分手的原因就是你改不了，他也忍不了。这不是不爱了，而是不敢爱。那信任也就是这样崩塌。其实他回过头来也会想不明白。

为什么会爆发，为什么会控制不住自己。原因就是他没有注意到，你已经积累到满的情绪。

## 核心挽回策略：重建信任

了解了分手的深层原因后，本节中我们来看看挽回的着重点。这类挽回的着重点，应该是放在怎么重新建立信任。这个信任是指让对方相信，跟你复合以后不会再经历之前的矛盾。

而且时间相较于一般的挽回来说，会要做好心理准备，比较长。但是如果一旦和好，你们的感情就会迅速升温，不会回到过去，但是会超越过去。

以下是具体的行动步骤指南：

1.  **停止道歉与纠缠**：所以不要去道歉，你只需要让他冷静一段时间，拉开你们的距离。
2.  **等待抵触情绪消退**：等到他没有那么抵触你的时候。
3.  **重新建立联系**：再通过一些社交平台，或者说共同好友，重新地跟他建立联系。
4.  **展示切实改变**：让他看到你的改变，让他去相信，跟你的复合不会重蹈覆辙。

## 总结与忠告

本节课中我们一起学习了累积失望型分手的本质与挽回路径。爱情从来都不是非黑即白。如果你很幸运地收获了一份很好的感情，那请你清醒一点，不要等到这份感情被磨没了，才幡然醒悟。

成功的挽回关键在于**用持续、稳定的改变重建信任**，而非言语上的承诺。这是一个需要耐心的过程，但结果是值得的。
