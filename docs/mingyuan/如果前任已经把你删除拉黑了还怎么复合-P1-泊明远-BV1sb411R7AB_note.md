# 分手挽回教程 第1课：被删除拉黑后如何复合？💔

在本节课中，我们将要学习当你的前任已经将你删除或拉黑后，如何正确理解这一行为背后的含义，并采取有效的步骤来恢复联系，为复合创造机会。很多人误以为被拉黑就意味着彻底结束，但事实并非如此。

## 理解删除拉黑的本质 🔍

上一节我们提到了被拉黑不等于彻底结束，本节中我们来看看这一行为背后的真实心理。

很多人都认为，被拉黑删除就是不爱，对方很绝望，才会对自己这么狠。甚至会觉得都删除拉黑了，怎么可能有机会去复合。但这只是你自己想的，并不是真的。

为什么会删除拉黑？大概率就是因为情绪太大。不知道该如何发泄，所以跟你说了也解决不了。于是情绪上到了极点，然后又下不去，最终把删除拉黑当成了情绪的出口。

**核心公式：删除拉黑 ≠ 不爱 = 情绪出口**

情绪来源于什么呢？来源于在乎。想一想，你会随便在大马路上跟一个陌生人生气吗？你会给通讯录里没有联系过的人，甚至没怎么说过话的人，把它拉黑吗？不会。

所以拉黑删除的源头在哪？在在乎，是在意。如果你懂得这一层，你才算是真的有复合成功的机会，心态也会逐步变稳。

## 采取正确的初步行动 ⏳

理解了行为背后的情绪本质后，接下来你要做的事情就很简单。

你呀，先等两天。我知道肯定有人会说，你不是说不要等吗？你不是说不要断联要趁热打铁吗？而我想说的是：**非必要不断联，不要盲目断联，也不要只断联不做其他**。

断联对挽回起不到决定性的作用，而不用断联它就不能做，适合的时候可以做。这个时候为什么要断联呢？其实很简单。

删除拉黑有一个好处是你们很多人都会忽略的，那就是这个行为本身，它既是宣泄了情绪，也能够进一步地避免你们发生激烈冲突。

所以此时再等两天的目的是：让之前的情绪矛盾再给释放一下，然后矛盾浓度也再稀释一下。最起码不要在情绪点上迎难而上。

## 设计恢复联系的“锚点” 📞

在情绪得到初步冷却后，下一步就是如何恢复联系方式。

千万不要换一个联系方式就去发小作文。不是不能用，而是你们一般都写不好。所以结果大概率都是进一步刺激了对方的情绪，然后再加回来就很难了。

其实说穿了，就是要在分手后的删除拉黑，找一个合理化的理由。换句话说是，你要给对方一个合理的台阶。

以下是几种可能的“合理台阶”或联系锚点：

*   你真的需要问一个具体、必要的问题。
*   你需要发送一个对方可能需要的文件。
*   其他符合你们具体情况、不显突兀的理由。

当然，这个要结合你们的具体情况，去寻找和设计这个复联的锚点。

## 总结与寻求帮助 🤝

本节课中我们一起学习了被删除拉黑后的复合策略。

我们首先剖析了删除拉黑是**情绪出口**而非**爱的终结**的本质。接着，我们强调了在情绪高点后**适当等待**以稀释矛盾的重要性。最后，我们探讨了恢复联系的关键——寻找一个**合理化、低需求的“锚点”**来给对方台阶下。
