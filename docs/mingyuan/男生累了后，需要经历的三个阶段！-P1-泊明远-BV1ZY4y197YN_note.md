# 亲密关系解析 01：男生感到疲惫的三个阶段 🧠

在本节课中，我们将要学习男生在一段亲密关系中感到疲惫并可能想要退出的心理发展过程。我们将这个过程拆解为三个清晰的阶段，帮助你理解其背后的逻辑与原因。

---

上一段我们提到了男生在关系中没有明显矛盾却想分手的情况，核心原因往往是“他累了”。那么，男生为什么会感到累呢？本节中我们来看看导致疲惫感产生的三个阶段。

## 第一阶段：从热恋到冷静 🔥➡️🧊

两个人从热恋期过渡到冷静期，男生对女生的好奇与探索基本告一段落。此时，**责任**与**共同目标**开始取代最初的新鲜感。

然而，许多男生不愿意去做那些需要长期言辞承诺和规划的事情。如果在这个阶段，男生没有用责任来约束自己，也没有与伴侣共同制定人生目标，他就很容易陷入迷茫。

**公式表示：**
`热恋期（探索与好奇） → 冷静期（责任与目标缺失） → 迷茫`

迷茫之后，随之而来的便是最初的疲惫感。

## 第二阶段：从冷静到平淡 🧊➡️🌫️

前面我们介绍了第一阶段会以迷茫和疲惫收尾。疲惫感的来源在于，男生开始难以承受两人日常的平淡。

此时，他的重心可能开始转移到自己的事业上，并以事业为挡箭牌，回避与你的沟通。或者，他觉得和你在一起该做的事都已做完，关系变得越来越无聊。

以下是其他可能吸引他情绪的事物：
*   与朋友社交
*   喝酒
*   打游戏

当他的情绪被其他事物吸引，对关系的投入自然会减少。

## 第三阶段：放弃经营与最终疏远 🚫

经历了前两个阶段，男生在短期内会变得懒得去经营感情、懒得沟通、也懒得维持仪式感。他们开始期待一种“顺风顺水”、无需费力经营的感情。

换句话说，进入这个阶段后，对男生而言：
*   **当下**，亲密关系带来的情绪满足感越来越低。
*   **长远**，他们也看不到情绪满足的愿景，不愿去计划、实现或追求。

如果不去经营感情，伴侣往往会通过“作”和“闹”来寻求关注。闹一两次，男生或许还能勉强满足，但次数多了，他就越来越难感受到关系中的积极因素。

**代码逻辑描述此心态变化：**
```python
if 不经营感情:
    伴侣行为 = “作”或“闹”
    if 闹的次数 > 容忍阈值:
        感受到的积极因素 -= 1
        心态疲惫值 += 1
```

久而久之，心态会越来越疲惫。当疲惫累积到顶点，问题看似无法解决时，男生就可能选择“解决掉人”——即结束关系。

---

本节课中，我们一起学习了男生在亲密关系中感到疲惫并可能选择退出的三个渐进阶段：**从热恋到冷静的迷茫期**、**从冷静到平淡的回避期**，以及**最终放弃经营、导致疏远的疲惫期**。理解这些阶段有助于看清关系变化的脉络。
