# 情感修复课程 01：主动联系前男友后他的心理分析 🧠

在本节课中，我们将要学习从男性视角分析，当你主动联系前男友时，他可能会产生的想法和感受。我们将拆解分手后的心理状态、有效沟通与无效纠缠的区别，并提供行动建议。

## 分手是深思熟虑的决定

上一节我们介绍了课程概述，本节中我们来看看男性提出分手时的心理基础。

对于男生来说，分手通常是经过深思熟虑的。当你们的关系出现问题后，他也会提出一些合理的诉求。

如果情况一直没有得到改善，并出现了不可调和的矛盾，既然解决不了问题，那他只能选择结束关系。因此在这种情况下，男生和女生的心理是相似的：**失望积累到一定程度就会导致分手**。

所以，在分手初期，前任对你有抵触情绪是非常正常的现象。

## 有效沟通与无效纠缠的区别

理解了分手初期的心理后，我们来看看联系他时，不同沟通方式带来的不同效果。

如果你可以好好地与前男友沟通你们之间的感情问题，并且他还愿意回复，这至少说明沟通是有意义的。无论是想复合，还是仅仅做朋友，这类话题都值得探讨。

以下是有效沟通的要素：
*   共同复盘恋爱过程和分手原因。
*   将此过程视为自我成长的机会。

但如果你只是纠缠于“复合”这个结果，或者一直控诉他过去的问题，对于男生来说，他不会愿意浪费时间和精力在这种他认为毫无意义的事情上。

**核心逻辑**：`如果你认为分手全是他的错，那么彻底分开即可，不要试图用你的标准去改造别人。`

## 建立信任需要实际行动

明确了沟通方式的重要性后，我们最后来探讨如何重建信任。

说一句实在话，分手后大多数前任都是可以联系到的。前任之所以反感，往往是因为你写了太多“小作文”、打了太多“感情牌”，却只得到了负面反馈。

如果问题在于你，而你不去反思自己的问题，反而纠结于“他爱不爱我”，那对方肯定无法对你建立信任。

**行动公式**：**承诺改变 → 立即行动**

你说你会改，就不要停留在口头承诺，需要快速有效地行动起来。
