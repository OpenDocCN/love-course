在本节课中，我们将学习如何通过观察特定行为模式，识别一段关系中对方可能“心里没有你”的三种关键表现。我们将逐一解析这些行为，并理解其背后的含义。

上一段我们介绍了本课程的目标，本节中我们来看看第一种具体表现。

## 表现一：缺乏主动联系 📵

第一种表现是：你不去找他，他也不会来找你。这意味着关系的维持完全依赖于单方面的主动。如果双方都有意，联系应当是双向和自然的。

## 表现二：交流中的冷淡回应 🥶

接下来我们看看交流中的表现。第二种表现是：聊天时，你一句他一句，他永远显得特别高冷。这种互动模式可以用一个简单的公式来描述：

**交流热度 = (回应积极性 × 话题延伸度) / 回应延迟**

高冷的回应通常表现为**回应积极性低**且**话题延伸度趋近于零**。

以下是这种表现的具体特征：
*   回应简短，仅限于回答，不展开新话题。
*   不表达好奇或关心，不询问你的状态。
*   回复间隔时间长，显得敷衍。

## 表现三：模糊不清的态度 🎭

最后，我们来探讨态度问题。第三种表现是：他的态度永远都是不主动也不拒绝。这是一种典型的模糊策略，既不愿承担责任，也不想明确结束。

这种态度可以类比为一段等待输入却永不执行的代码：
```python
while True:
    # 等待“明确信号”输入
    signal = get_signal()
    if signal == “你主动推进”:
        pass  # 不执行任何操作，保持现状
    elif signal == “你要求澄清”:
        print(“你误会了。”)  # 拒绝承认既定模式
```
循环持续进行，关系状态无法更新。
