# 分手心态调整指南 🧠 - 课程 P1

在本节课中，我们将学习分手后如何调整心态的核心步骤。重点是采取实际行动，帮助自己走出情感低谷，重建生活重心。

## 断绝一切联系 🔌

上一节我们介绍了课程概述，本节中我们来看看调整心态的第一步。分手后，必须断绝与前任的一切联系。不要以做朋友或放不下为借口。你不缺这一个朋友。这个过程虽然困难，但必须执行，因为分手后对方的任何行为都会被你的情绪放大，严重影响你的心情。

以下是执行此步骤的关键点：

*   删除所有联系方式。
*   避免访问对方的社交媒体。
*   不要寻找借口与对方产生交集。

## 确立自我核心价值 💎

在完成了断绝联系后，我们需要将注意力转向自身。无论何时，自己才是最重要的。世界很大，不必执着于一人。在爱情中经历失败并不丢人。

核心心态可以概括为以下公式：

**自我价值 > 一段已结束的关系**

## 积极向前看 🚀

摆正心态后，接下来要着眼于未来。感情失利是常见经历，最终能走出困境的人往往都拥有积极向前看的态度。

以下是帮助你向前看的方法：

*   接受分手事实，承认关系已经结束。
*   将精力投入到个人成长、兴趣爱好或事业发展中。
*   相信未来会有新的、更合适的人际关系出现。

## 总结

本节课中我们一起学习了分手后调整心态的三个关键步骤：**断绝一切联系**、**确立自我核心价值**以及**积极向前看**。记住，调整心态是一个过程，允许自己难过，但更要采取行动帮助自己走出来。你的感受很重要，但你的行动决定你恢复的速度。
