# 情感分析课程 P1：判断前任是否仍有情意 💔

在本节课中，我们将学习如何通过观察几个关键行为，来判断分手后的前任是否对你仍有感情。这些迹象可以帮助你理解对方的潜在心理状态。

上一节我们介绍了课程主题，本节中我们来看看具体的判断依据。以下是三个核心的观察点：

1.  **联系方式与主动帮助**
    如果前任保留了你的联系方式（如微信、电话），并且偶尔会主动关心你或帮助你解决问题，这通常表明他/她仍在意你。爱意难以完全隐藏，即使曾有过争吵或伤害，不舍的情绪依然可能存在。

2.  **默默关注你的动态**
    这类前任可能性格较为骄傲，不轻易主动联系或认错。但他/她可能会通过关注你的社交媒体（如微博、抖音、网易云音乐），甚至使用小号来查看你的近况。这种关注往往是沉默的，是一种藏在心里、背后的观察。要知道，藏在心里的怀念，有时最为深刻。

3.  **你能影响其情绪**
    如果你能轻易地牵动前任的情绪，那是对你仍有感觉的强烈信号。例如，你在朋友圈发布了一张收到鲜花的照片，随后通过共同朋友得知前任为此感到不快。这表明你的行为依然能激发他/她的情绪波动，对方尚未真正放下。

以上三点是判断前任是否有复合意愿的几个常见表现。然而，需要注意的是，在考虑修复关系的过程中，不应仅停留在被动猜测。在判断出对方可能仍有感情后，更应学会主动引导，创造积极的互动和情绪波动，这能让后续发展变得更加顺理成章。

本节课中我们一起学习了通过观察**联系方式保留**、**动态关注**以及**情绪影响**这三个方面，来判断前任的内心状态。记住，观察是第一步，理解之后的有效行动才是关键。

---
**总结**：本节课的核心在于识别情感残留的迹象。关键在于观察对方是否**保留联系**、是否**默默关注**你，以及你的行为是否仍能**影响其情绪**。这些信号是判断复合可能性的基础。
