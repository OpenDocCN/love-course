# 那些成功复合的人，分手初期都是怎么做的-P1：高田38横-泊明远-BV1Jr4y1H7y9_note

在本节课中，我们将要学习那些最终成功复合的人，在分手初期是如何应对的。理解并实践这些核心策略，是修复关系的第一步。

经历过分手的人都明白，分手的初期是很痛苦的。

这种痛苦会促使你第一时间想要去挽回，而不是去好好想一想究竟哪里出了错，以及我们为什么要挽回。

因为大多数人对于损失和获得的敏感程度是不对称的。面对损失的痛苦感，要远远大于获得的愉悦感。在心理学上，这被称为 **损失厌恶**。

但是大部分人意识不到这一点。所以在痛苦感的驱使下，第一时间会想要去挽回。

他们不会思考为什么你们会分手，只会在自己的情绪支配下，做出一些不可避免的行为。

然而，你做出这种枝端的行为，只会增加对方的反感，将对方越推越远。

当然，也存在有些人在分手初期，会有一种无事负重的感觉，自身的状态还不错。

但好景不常的是，分手的痛苦感会慢慢地占据他的大脑。

因此，我不建议在分手初期尝试联系对方。因为这个时候的你是被情绪操控的，很难理性地去做出一些分析判断。

---

上一节我们介绍了被情绪驱使的冲动行为带来的危害，本节中我们来看看正确的初期应对核心是什么。其关键在于管理好“损失厌恶”心理，为理性思考创造空间。

以下是分手初期应遵循的几个核心步骤：

1.  **接纳情绪，暂停联系**  
    承认并接纳自己的痛苦情绪，但明确意识到此刻联系对方是情绪驱动的。强制设定一个“不联系”的冷静期。

2.  **进行自我审视**  
    利用冷静期，将注意力从“挽回”转向“思考”。问自己：关系破裂的核心原因是什么？双方各自的责任是什么？

3.  **重建个人状态**  
    通过规律作息、运动、社交或培养新兴趣等方式，稳定自身情绪，避免被痛苦感完全吞噬。良好的个人状态是后续一切行动的基础。

---

本节课中我们一起学习了分手初期的关键心理机制——**损失厌恶**，以及由此产生的冲动挽回行为的危害。成功的修复始于初期的冷静、暂停与自我审视，而非被痛苦驱使的纠缠。管理好情绪，为理性分析和后续行动做好准备，是走向成功复合的第一步。
