# 分手后发什么样的消息，对方才是

在本节课中，我们将要学习分手后如何与前任进行有效沟通。核心目标是避免引起对方反感，并重建良性的对话渠道。我们将重点探讨消息内容的设计原则、情绪管理以及沟通策略。

## 收起恋爱脑，避免情绪绑架 🧠

都说分手之后根本没有办法和前任说上话，不管发什么对方都不回复，或者反应冷淡、完全不想沟通。这时该怎么办呢？第一点是收起你的恋爱脑，不要用情绪绑架对方。这一点对大部分人来说非常困难。

可以说，能够收起自己的恋爱脑，在挽回的道路上就已经成功了一大半。因为恋爱是人的本能，在情感遭受冲击之后，自然会**产生崩溃、痛苦、焦虑等情绪**。在这种情绪的驱使下，你会做出很多不理智的行为。

## 识别并避免不理智行为 🚫

以下是情绪驱使下常见的不理智行为示例：
*   信息轰炸、围追堵截
*   疯狂道歉、拼命求和
*   卑微纠缠

其实这些情绪反应都非常正常。但任何一件事情的成功，都不是靠情绪胡抓乱挠来的。每一个成功的人都比你想象当中的更加沉着、冷静、睿智。因此，我经常说的一句话是：**情绪是一生之敌**。

## 精心设计第一条消息 ✍️

我们在去联系前任的时候，第一条消息一定要仔细斟酌，想好怎么去发。需要分辨哪些内容是对方反感的，哪些信息是对方看到就不想理你、不知道该怎么回答，或是可能误会你而将你拉黑或删除的。这并不难找。

我们完全可以换位思考。如果我们主动提出分手，那当下你最讨厌接收的信息是什么？例如：“在干嘛？”“吃了吗？”“睡了吗？”，这些显得很无聊。或者说：“你是不是在外面有人了？”“你是不是早就想好了？”，这会让他很反感。又或者：“我对你这么好，你为什么还要离开我呢？”“我到底是哪里不好，你告诉我呀！”，这样只会消耗对方的耐心。

更有甚者会说：“再给我一次机会吧，我以后再也不这样了。”试问，这样真的有诚意吗？我看不出诚意在哪里。这些消息一旦发出，只会消耗对方的耐心。

## 学习发送让对方感到舒适的消息 😌

那么，什么样的消息会让对方感到舒服，甚至有回复的欲望呢？我简单举两个例子。你可以跟他说：“你现在不想回复我没有关系，等你情绪好点再说。”这样不要给对方压力，对不对？你这样说，对方就会毫无压力。

总之，你要记住，你去传递的是**正面的情绪**，不要让对方反感。如果实在不会说，哪怕不说都比你乱说要强。

## 警惕暴露需求感和引发负面回忆 ⚠️

最可怕的是，有的人还会说：“我还爱你”“我还忘不了你”。这样不就是让对方想起和你在一起的经历吗？越想可能越讨厌你，而且暴露了你的需求感。不说呢？不让对方觉得你没有人情味、很现实，还想把他带回到过去，他就会瞬间变得不想回复你。

记住，**包含浓烈情感的话语是情侣间独有的**。你们已经分手了，这个时候请务必理性一点。

## 保持理智，规划长期沟通 📈

不管对方给你回复什么，你都要保持理智。只有在理智的状态下，你才能轻松回复，不被对方带节奏。你复联时的每一句话，看起来很随意，实则都是有含义在里面的，不要不经过脑子就脱口而出。

就算对方不回复你，这也是很正常的，你也不要因此失落和难过。总之你要记住，前期复联的目的就是**重建联系**。只要可以开始正常的沟通，就可以慢慢地去规划后面的事情。记住，不要着急，不要想着一步登天，要慢慢来。

本节课中，我们一起学习了分手后沟通的核心原则：**管理自身情绪、避免情感绑架、精心设计消息内容、传递正面情绪、保持理智态度**。关键在于通过不引起反感的沟通，逐步重建联系，为后续发展打下基础。我是泊明，我们下节课再见。
