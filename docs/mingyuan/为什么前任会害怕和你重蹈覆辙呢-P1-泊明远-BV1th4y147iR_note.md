## 1. 沟通是导致分手的主因 💬

上一节我们介绍了课程主题，本节中我们来看看导致关系破裂的首要因素。所有导致分手的最主要原因是沟通问题。回想你们曾经相爱的时刻，过去的价值匹配如今已不再匹配。请认真思考，是否在每次解决问题时，当对方说了某些话或做了某些事，而你不认可或不认同时，你本意是希望对方改正，但反馈给对方的话，在他/她眼中却成了情绪化的责怪。

以下是几种典型的负面沟通方式：
*   使用负面语言和过于强硬的语气。
*   态度让人难以接受，例如说出挖苦、讽刺的话。
*   进行侮辱或看不起对方的言行。

这些沟通方式会引起争执和不适。人们最终爆发分手，是因为在相处过程中堆积了太多不满、怨言和负面情绪。这些最终成为了“不合适”的借口。你必须明白，无论是在分手前还是分手后，当前任的回应和举动让你不满意、不舒适时，你都不应该表现出任何不好的情绪。只有这样，对方才能逐渐感觉到，与你的沟通和相处中，你调整了、变化了、改善了，不再有之前那种束缚和隔阂。

## 2. 如何调整你的心态 🔄

理解了沟通问题后，我们来看看阻碍你有效行动的深层原因——心态。世界上有两种人：固化型心态和进步型心态。

以下是两种心态的具体表现：
*   **固化型心态**：自以为是地坚持自己的想法，总认为挽回不了是对方不给机会、是对方的原因，自身问题不大，为自己找借口。也可能因为不确定结果而不敢行动，害怕失败。或者认为“我遇到不好的人，可以遇到下一个更好的”。
*   **进步型心态**：认为只有通过学习和成长，学会方法和底层逻辑，并加以运用，就一定能办到。

你是否一开始也认为“没办法了，他没回应，要不就算了吧，再等等”？如果你有过这样的想法，你就是第一种心态。即便别人告诉你怎么和好，在这个过程中，但凡出现一些你认为难以完成的行为，或者对方没有很快给出良好回应时，你就不自觉地退缩、打退堂鼓。即使你抱着试试看的心态去做，一旦效果没有立竿见影，你就会说“没用”，接着开始放弃。

然后，你可能把希望寄托在算命、塔罗牌或星座上，被动地等待对方回来找你。最常见的一种表现就是找理由，例如：“我也想和好，可是我接受不了他冷漠”、“我面子上挂不住”、“我不想去主动找他，这明明是他的问题，他为什么不主动？”从而把自己陷入恶性循环。**你拥有什么样的心态，就会得到什么样的结果。**

## 3. 调整情感上的认知误区 🧠

上一节我们探讨了心态的重要性，本节中我们来看看在情感认知上常见的错误。不要认为“付出了就有回报”。

例如，不要认为：
*   对方感到“失去感”就会后悔。
*   你发朋友圈对方就会回来看你或主动联系你。

事实并非如此。挽回成功往往需要多种刺激，并且需要坚持进行多点、持续的刺激才能成功。这不是做一两次就能见效的。**公式：成功挽回 = 理性的针对性改变 + 感性的持续触动 + 时间的积累以重建信任。**

你需要在理性上做出针对性改变，并在感性上持续触动对方，坚持到一定程度，让对方重新产生信任，这才可能成功。除了控制情绪和保持良好心态，没有一夜暴富或一两天就成功之说。你们的问题不是一两天造成的，是长期累积的，所以复合本身也需要时间。
