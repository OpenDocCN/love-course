# 挽回教程 01：理解前任的抵触心理与自我防御机制 🐢

在本节课中，我们将要学习分手后前任表现出强烈抵触心理背后的原因。我们将深入探讨“自我防御机制”这一核心概念，并理解为何这种情况下，挽回的主动权反而可能在你手中。

分手后，如果他对你表现出抵触，说完狠话并扬言要断干净，随后删除拉黑所有联系方式，同时在朋友圈展示低落情绪，这种行为实质上是启动了**自我防御机制**。这种情况下，和好的可能性反而很高，感情的主动权其实掌握在你的手中。

之所以这样判断，是因为他对你的感情并未减少，但分手带来的痛苦触发了其本能的自我防御。经常有刚分手的人前来咨询，后悔想要挽回，却发现已将所有能联系到前任的方式都切断了。他们通常会解释，分手初期过于难过和迷茫，不愿承认那个与对方大吵大闹的自己，因此决心要坚定、要断联，冲动之下删除了所有联系。结果发现自己根本无法入睡，想念前任，既无法拉下脸面去找对方，又害怕再次受伤。

实际上，双方都承受着同样的痛苦。这类人有一个共同特点：回忆其有情绪时，是否只顾宣泄情绪、逃避问题，或者关系分分合合、态度忽冷忽热，有时表达想念，有时又否认喜欢。还有一类人，分手后痛哭许久，隔天却告诉你会去相亲、会有新人陪伴。所有这些举动，都表明他依赖感情、更容易受伤，此时**自我防御机制**便被触发。他会无法克制地做出拒绝外界交流的举动，就像乌龟缩回自己的壳里。你越是想要敲打外壳，他缩得越紧。

上一节我们介绍了“自我防御机制”的表现与成因，本节中我们来看看破解这一难题的三个核心策略。

以下是破解前任自我防御的三个关键步骤：

1.  **减少压力源**
    核心是尽量避免拉着对方谈论感情，因为这是主要的压力来源。你不提，前任自己也会反复思考分手带来的痛苦和焦虑；你一旦提起，便会给他带来双倍甚至三倍的焦虑。

2.  **提出妥协条件**
    双方各退一步，提出一个折中的方案。你可以对他说：“你可以断联，我接受。但有些我们一起做的事情需要收尾，简单地断绝对双方都是折磨。” 男性可以借助共同爱好，提议相关的事情还想继续聊聊；女性则可以以“自己还没走出来”为借口，用柔软的语气诉说，前男友通常容易心软。

3.  **给予短期奖励**
    奖励是人的行为驱动力。可以在前任态度稍有好转后，表达你的理解和感谢。其实只需要提及“这段时间我们的关系很稳定”即可。目的是向他表明，你并非一个恃宠而骄、一味纠缠的人，你很清楚他本可以离开却选择了留下。

4.  **成为情绪榜样**
    容易陷入自我防御的人内心通常敏感。你对情感的理解层次需要高于他。当他内心脆弱时，你要有耐心；当他退缩时，你需要前进。这本质上遵循一个**执求原则**：`让对方觉得，和你在一起永远是安全的`。从而绕过他的自我防御机制。

本节课中，我们一起学习了分手后前任抵触心理背后的“自我防御机制”，并掌握了通过**减少压力、提出妥协、给予奖励和成为榜样**来破解僵局的策略。记住，核心是创造安全感，让对方觉得与你相处是舒适而非威胁的。
