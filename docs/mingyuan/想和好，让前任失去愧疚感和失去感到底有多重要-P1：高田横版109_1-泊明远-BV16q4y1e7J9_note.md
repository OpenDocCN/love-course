# 💔 情感修复教程 01：重建关系中的愧疚感与失去感

在本节课中，我们将学习如何通过特定的沟通策略，在情感修复过程中，巧妙地激发对方的愧疚感与失去感。这两种感觉是促使对方主动回头复合的关键心理动力。我们将解析一段示例文本，并理解其背后的原理与应用方法。

上一节我们介绍了课程的核心目标，本节中我们来看看具体如何操作。

## 🔑 核心心理机制：愧疚感与失去感

男生如果同时产生**愧疚感**与**失去感**，他自己就会产生回头复合的意愿。关键在于，这两种感觉很难同时传递给对方。

以下是激发这两种感觉的核心逻辑：
*   **愧疚感**：源于对方意识到自己在这段关系中负有责任，或对你的付出有所亏欠。
*   **失去感**：源于对方意识到你已经彻底放下，并且他可能永远失去你了。

## 📝 关键行动：发送预设信息

老师提供了一段预设文本。只要你将这段话照抄并发送给他，就能有效地在他心中植入这两种感觉。

以下是这段文本的内容：
> 这段感情的破裂我负主要责任。以前是我太傻太笨，我只知道对一个人好，但是我不知道这个样子也会给他压力。谢谢你教会了我。我以后不会再为任何人再丧失自己，当然也包括你在内。

## ⚠️ 后续操作与效果

发送信息后，你必须停止主动联系他。不再理睬他这个行为，会让上述信息的效果发挥到最大。

基于此策略，预计在**半个月内**，他必然会主动来找你。
