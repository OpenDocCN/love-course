# 情感课程 P1：什么样的男人不值得原谅 🚫

在本节课中，我们将要学习如何识别那些不值得被原谅的男性行为模式。理解这一点，有助于保护自己在情感关系中免受反复伤害。

## 核心原则：不要轻易原谅反复伤害你的人

上一节我们介绍了课程的主题，本节中我们来看看具体的核心原则。

记住，不要轻易地去原谅一个反反复复伤害你的男人。

**核心公式**：无底线的原谅 + 无原则的包容 ≠ 浪子回头

你没有底线的原谅，没有原则的包容，很难换来他的浪子回头。

## 对方的真实反应：得寸进尺

理解了核心原则后，我们需要认清对方可能产生的真实反应。

他的只是他的得寸进尺。

他会有他的演技和实际行动告诉你，你真的很傻。

## 给学习者的建议：建立独立性与寻求帮助

认识到问题模式后，采取正确的行动至关重要。以下是给你的两点核心建议。

希望你做一个任何时候都不依靠他人来获取任何满足的人。

我是博明，如果你正在经历这样的情感问题，私信我，我来帮你。
