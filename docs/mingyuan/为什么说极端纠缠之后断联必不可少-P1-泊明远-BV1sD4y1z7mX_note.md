# 情感修复课程 01：为什么极端纠缠后必须断联？🚫

在本节课中，我们将要学习为什么在感情破裂、尤其是经历过极端纠缠行为后，“断联”是必不可少的第一步。我们将分析纠缠行为的负面影响，解释断联的核心作用，并探讨如何正确执行断联策略以实现自我提升和关系修复。

很多人分手后，会进行信息轰炸、电话轰炸，或动用好友、亲戚、家长劝说，甚至采取上门围堵、单位蹲守等极端纠缠行为。我相信，当你看到这条内容时，你或多或少都做过以上某件事。首先，我非常理解你疯狂行为背后的不安。

但现在还不是你释放情绪的时候。你当前所做的一切纠缠，无疑是在放大你的缺点。你当下最应该做的事，是断开一切联系。无论你之前做了多少，从现在开始，你必须断联。你不用担心断联会让对方忘记你。

只是因为此时此刻，对方能够想起你的，只有这些负面情绪。你以为做的一切是在感动对方，但在对方的视角里，你就像一块甩不开的牛皮糖。你只有停止这一切纠缠，才能让对方感到“失去”，而这种负面印象才不会继续堆叠。

当然，有些情况下，断联了两三个月、四个月，对方还是没有回头。这个时候你就会产生疑问：问题出在哪里？我告诉你，很有可能是因为，你之前在疯狂纠缠的时候，触碰了对方的底线，而且是不可逆的点。

这个时候你的断联，其实没有多大效果。那有什么办法呢？你记住，所谓“不可逆的点”，就是你触碰了对方的原则性底线。而底线这个东西，会根据对方的价值判断而随时波动。

我打个比方：假设你的底线是非常讨厌有人在公共场合抽烟。如果你的相亲对象在约会时，当着你的面抽烟，你是不是会非常厌恶，想转头就走？但如果这个人是彭于晏或吴彦祖，你会不会为他降低底线呢？你甚至还会觉得他很男人。

你发现了没有？对方是否认为某个点“不可逆”，取决于你本身的价值。如果你的价值高，他能够为你降低底线；如果你的价值本来就很低，他的底线也会相应抬高。

因此，在断联期间，你最重要做的事情，就是进行爆发式的自我提升。**自我提升 = 价值重塑**。这件事你再难也得去做。一天到晚去幻想、去担心他，不如切切实实地改变自己，让自己发光发热。

我是泊明。如果屏幕前的你，现在正处在这样的感情困境中，不知道该如何去做，可以寻求帮助，与我聊一聊。
