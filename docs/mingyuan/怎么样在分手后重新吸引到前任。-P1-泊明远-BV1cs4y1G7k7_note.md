# 分手后重新吸引前任的底层逻辑 🧠 - 课程 P1

在本节课中，我们将学习分手后如何重新吸引前任的两个核心心法。这些心法揭示了长期关系背后的底层逻辑，虽然听起来有些残酷，但却是实现复合的关键。

## 第一心法：价值平衡是关系的基石

上一段关系结束，本质上是因为双方的价值天平发生了倾斜。本节中我们来看看如何理解并重建这种平衡。

短期的感情靠的是荷尔蒙的吸引。而感情的长期保持，背后依靠的肯定不是“爱”，也不是你们有多少感情基础，而是**价值平衡**。

“爱”的背后，实质是**能否满足对方的需求**。能满足需求的特质，就是你的价值。你们最初能在一起，一定是价值相对平衡的。然而，人在不同时期、不同社交环境下，需求会发生变化。任何关系的底层运行原则就是**平衡**。

分手意味着对方认为你身上有某一方面存在重大的价值缺陷，并给对方造成了很大困扰。因此，你要把不平衡变为平衡。具体做法是，找出对方在过往关系中对你的不满点。

针对这一块进行具体的价值提升。平衡价值没有你想的那么复杂。只要你思维变了，你的语言、行为、外表，甚至是性格脾气，一切都会随之变化。但改变自己固有的思维想法，这才是最难的。

## 第二心法：用情绪波动升华关系

上一节我们介绍了价值平衡是关系的基础。本节中我们来看看，在价值平衡之上，如何让感情得到升华。

爱情的升华，背后依靠的**一定是情绪波动，而不是情绪稳定**。如果你让对方一直很快乐、很开心，时间一长，对方不但会腻，而且欲望会膨胀，你也会越来越累。

如果你一直拼命付出，只为给对方制造舒适感和甜蜜感，这样只会给对方建立一个基础的、毫无波澜的感受。对方的情绪感受没有受到强烈的冲击，就不可能产生情绪的吸引。

因此，你要让对方在舒适稳定一段时间后，进行一次**情绪冲击**。这个情绪冲击，既可以是高潮，也可以是低潮。也就是说，你可以让他很快乐，也可以让他很失落。

简单来讲，就是关系不能太平淡。需要隔一段时间，用一次激情唤醒一下沉睡的情绪。哪怕是隔一段时间吵吵架，都会比一直保持稳定的情绪好得多。

所以，关系需要有平淡、有激情，也要有低落。不能一直平淡，也不能一直激情，更不能一直低落。

---

本节课中我们一起学习了重新吸引前任的两个核心心法：**价值平衡**与**情绪波动**。掌握了“价值”与“情绪”这两个关键，别说复合，几乎没有你搞不定的感情。
