## 男性放弃感情的典型模式：主动冷淡 🥶

上一节我们介绍了本课程的主题，本节中我们来看看男性放弃感情时的具体表现。

男性决定放弃一段感情时，大部分会选择冷淡的方式。具体表现为**一点点冷落和忽视**，甚至**主动制造一些障碍**，目的是让对方感到厌烦并疏远他。这个行为本质上是在**制造契机**，其逻辑可以概括为以下公式：

**男性主动冷淡 = 制造对方离开的契机（让对方主动离开 或 促使对方选择离开）**

很多女性并不明白这种行为背后的意图。
