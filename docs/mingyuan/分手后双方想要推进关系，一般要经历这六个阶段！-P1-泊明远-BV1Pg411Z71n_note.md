# 情感修复教程 01：分手后关系推进的六个阶段 🧩

在本节课中，我们将学习分手后，主动提分手一方可能经历的心理阶段。理解这些阶段有助于我们采取正确的行动，避免踩雷，从而更有效地推进关系，最终走向复合。

## 阶段一：敌对阶段 ⚔️

上一节我们介绍了课程概述，本节中我们来看看第一个阶段。刚分手时，对方处于敌对阶段。此时他对你抱有强烈的负面情绪。这种负面情绪不仅体现在情绪上，也体现在态度上。**你在他的眼中是一个负面的存在**。你们两人处于敌对关系。

## 阶段二：陌生人阶段 🚶

当敌对情绪逐渐消退，关系会进入陌生人阶段。此时，他对你的负面情绪可能已经降低。**他将你视作一个陌生人**。这是关系从冰点开始解冻的起点。

## 阶段三：普通朋友阶段 🤝

从陌生人阶段，关系可以推进到普通朋友阶段。这是我们建议进行复联（恢复联系）的起点。以下是复联时的核心行动准则：
*   以朋友的身份为基础进行相处、交流和接触。
*   核心目标是打消他对你的戒备心。

## 阶段四：朋友到暧昧的推进 💬

从普通朋友阶段开始，我们需要通过聊天和互动来逐步推进关系。这个过程是递进的：**普通朋友 -> 朋友 -> 亲密好友 -> 暧昧关系**。在这个过程中，沟通技巧和边界把握至关重要。

## 阶段五：必须避免的“雷区” 💥

在关系推进到暧昧或亲密好友阶段之前，有一个绝对不能触碰的禁忌。这个禁忌就是：**不要和他谈论感情，不要聊过去**。因为过早开启这些话题，会将他拖拽回对你的负面固定印象中，让他想起不愉快的经历。这会导致本来向好的关系发展瞬间“打回原形”，且修复过程会比第一次更加艰难。

## 阶段六：复合窗口期与识别信号 💑

最后一个阶段是复合窗口期，也是大家最关心的部分。当关系出现以下多个信号时，意味着窗口期已经到来：
*   **信号1**：曾经的暧昧感重新出现，或者昵称重新被使用。
*   **信号2**：他允许你进入他的私密空间（如卧室）或查看他的手机。
*   **信号3**：他不排斥与你的肢体接触（如牵手、拥抱）。
*   **信号4**：他对你的热情是唯一性的，不同于对其他人。
*   **信号5**：他开始能够与你探讨感情中出现过的问题。
*   **信号6**：他不排斥与你单独约会（如吃饭、看电影）。
*   **信号7**：对于你们的聊天，他始终给予积极有效的反馈。

当满足以上多数条件时，**他已经基本准备好重新接纳你**。此时，即使你不主动提复合，他也可能会主动提出。

---

本节课中，我们一起学习了分手后关系推进的六个阶段：从敌对、陌生人、普通朋友，逐步推进到朋友、暧昧，最终抵达复合窗口期。关键在于理解每个阶段的特点，并在“普通朋友”到“暧昧”的推进过程中，严格避免过早谈论感情和过去。最后，识别出复合的多个关键信号，就能抓住最佳时机。
