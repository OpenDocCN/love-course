# 当你们已经没办法好好沟通时，先

在本节课中，我们将要学习当对方因情绪抵触而无法正常沟通时，应如何操作。核心在于理解平复对方情绪比直接挽回更重要，并掌握具体的行为步骤。

就在对方连好好跟你说话都做不到的时候，平复他的情绪，永远比挽回本身更重要。想象一下，当父母管教你时，你在气头上，哪怕知道自己不对，也会下意识地顶撞。一个带有抵触情绪的沟通是完全没有意义的。无论你谈论的内容多么正确，最后极有可能不欢而散。

因此，在对方明确不想复合的前提下，我们应该怎么去操作？主要分为以下三点。

## 第一点：行为上照顾对方情绪 👐

上一节我们提到了抵触情绪的危害，本节中我们来看看如何在行为上应对。你能操控好自己的情绪，并操控好别人的情绪，这个就叫做情商。这一点应该很好理解。但是做咨询这么久，我见过太多人在挽回过程中，只管自己想不想做，而不管对方想不想接受。那么，他凭什么要相信你能有所改变呢？想要跟你继续在一起，不可能。

以下是具体的行为准则：
*   **对方不愿说话时**：你就少聊两句，及时撤退。
*   **对方不愿见面时**：你就不要再纠缠对方，及时止损。

## 第二点：重建信任 🔄

在学会了停止激化矛盾后，下一步是重建破碎的信任。不管是通过朋友圈、共同好友还是侧面传递，目的都是让对方重新信任你，能给他带来良好的恋爱体验，然后才能选择相信你。

说白了，就是你要好好地深挖一下你们分手的原因。有很多人是嫌弃或者讨厌对方粘人、不给空间、失控、情绪化，或者说没钱、没有上进心、人丑还不知道收拾自己等等。所以你就要去做针对性的展示。

**核心逻辑**：`对方看到你的改变 -> 他的态度和情绪随之变化 -> 出现信任 -> 改善对你的负面印象`

只有对方看到你的改变，他的态度和情绪才能随之变化，才能出现信任。有了信任，才能改善对你的负面印象。

## 第三点：试探性操作与反馈处理 🎯

最后，在尝试操作时，一定要懂得去试探对方，这一点非常重要。但是，试探对方的意义在于获得对方的反馈，无论这个反馈是好是坏。

很多人一听到对方没有给自己好脸色，或者说反馈很差，就开始暴跳如雷，情绪激动，撂挑子不干。一定要记住，获得负反馈时，一定要控制好自己的情绪，做好及时止损。控制好自己的情绪是重中之重。如果你连自己都控制不好，你怎么能期望对方回来呢？

正反馈的时候，那就更好了，要趁热打铁，及时地去递进关系。聊天沟通时要多次试探。只有在每次的试探中，才能知道对方对你的排斥心态是否有下降。

以下是关系递进的步骤：
*   **判断信号**：如果对方从断联到愿意复联，说明他的负面情绪在减少，接纳程度在提升。
*   **逐步递进**：你可以一点一点地加大联系频率，增加聊天次数。
*   **核心原则**：关系由弱变强、逐步升温是一个循序渐进的过程，千万不要着急。越着急，你的心态就会越崩。

---

本节课中我们一起学习了在挽回初期沟通受阻时的核心策略。我们明确了**平复情绪优先于解决问题**的原则，并分三步展开：**1. 行为上照顾对方情绪，及时止损；2. 通过针对性展示重建信任；3. 通过试探获得反馈，并依据反馈（正/负）冷静处理或逐步递进关系**。记住，这是一个需要耐心和情绪控制的循序渐进过程。
