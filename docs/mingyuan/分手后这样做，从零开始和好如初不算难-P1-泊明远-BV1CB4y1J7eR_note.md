# 分手后复合指南：从零开始重建关系 - 课程01 🧊

在本节课中，我们将要学习分手后如何正确迈出第一步，理解分手初期的心理状态，并避免常见的错误行为，从而为后续的复合打下稳定基础。

## 理解分手初期的“冰冻期” ❄️

上一节我们介绍了课程的整体目标，本节中我们来看看分手后第一个关键阶段。

分手后想要复合，需要关系上的层层递进，这样复合才能变得更加稳定。大部分情侣分手后会快速进入一个“冰冻时期”。提出分手的一方，不想再继续联系。

最起码在短时间内不想再继续联系。因此，分手后你们的关系基本上处在一个陌生人的状态，甚至比陌生人更糟糕。因为他对你已经不需要具备最起码的礼貌，跟你聊天也会产生特别强的抵触情绪，不太想跟你接触。

## 为何纠缠与哀求会适得其反 ⚠️

理解了冰冻期的存在后，我们来看看为何急于求成往往会导致反效果。

他害怕被纠缠。你跟他联系就是在违背他的意志，因为他的意愿是分手后要跟你撇清关系。大部分人在分手后联系对方，都是抱着明确的目的去互动、去求复合，甚至有的不愿复合就变成了吵闹和威胁。

以下是这些行为传递出的错误信号：

*   你所有的行为都在告诉对方：“我不同意分手，我想和你继续在一起。”
*   他清清楚楚知道你的目的，所以他才排斥、抵触甚至攻击你。

这就像你做了一份工作，已经非常累又赚不到钱，还让你心里很不舒服。你提出离职后，公司对你苦苦哀求，但有用吗？你会回去重新上班吗？肯定不会。你只会更加坚定离职的想法，而且离职后心里还会暗爽：“你现在终于知道后果了吧，那当初早干嘛去了？让你后悔去吧。”

面对你的纠缠和苦苦哀求时，你的前任也是这么想的：“你现在知道求我了，早知今日何必当初呢？”

## 正确的第一步：从冰点走向朋友关系 🤝

明白了错误行为的后果，本节我们来看看在冰冻期应该采取的正确策略。

当对方提出分手，关系处在冰点状态时，不要去求复合。因为这样只会让他更加坚定分手的决心，同时也会让你的价值感变得越来越低。

我们要做的，是从冰点的状态，逐步达到一个朋友的关系，一点一点地去渗透。做到这一点并不难，只要你不再继续“作死”。

以下是达成此目标的两个核心要点：

*   **不暴露你想复合的需求感。**
*   **不去破坏你们之间的舒适区。**

毕竟他对你也不是极度抵触，人性都是肉长的，甚至还会慢慢跟你说话。如果你是真心想挽回，就可以按照这个步骤去做。
