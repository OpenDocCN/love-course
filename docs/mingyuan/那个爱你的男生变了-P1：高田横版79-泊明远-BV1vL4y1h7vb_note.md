# 情感关系解析课程 P1：识别关系中的变化迹象 🧐

在本节课中，我们将要学习如何识别亲密关系初期与稳定期后，伴侣行为模式可能发生的关键变化。我们将通过具体的行为对比，帮助你客观地分析一段关系的发展状态。

你是否注意到一个现象？在追求阶段，男生似乎拥有无限的空闲时间。关系稳定之后，他却变得异常忙碌。渐渐地，他不再是无时无刻不给你发消息、连晚安都要重复好几遍的那个人。

后续你甚至可能察觉到更多变化。

以下是关系可能发生转变的一些具体表现：

*   **互动频率降低**：他不再主动发送大量消息，沟通的意愿和热情明显减退。
*   **注意力转移**：你发现他与其他异性存在界限模糊的暧昧互动。
*   **情感温度变化**：他失去了早期的温柔与耐心，态度转向冷淡和疏离。
*   **责任归属错位**：发生矛盾时，他从不主动承认自身错误，总将责任归咎于你。

即便面对上述情况，我明白，仍有一部分人难以割舍已有的感情投入。
