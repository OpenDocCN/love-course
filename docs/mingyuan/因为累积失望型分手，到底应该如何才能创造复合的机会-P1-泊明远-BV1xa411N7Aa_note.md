## 什么是累积失望型分手？🤔

上一节我们提到了课程主题，本节中我们来看看这种分手的具体定义。

累积失望型分手所产生的矛盾和情绪，大多数来源于一些细微的摩擦，是一点一滴积累出来的。

例如，你们会因为某些事情、相处方式或你的行为，让对方产生许多不满。对方可能已经明示、暗示或提示过你很多次。而你的做法要么是不在意，要么觉得道个歉就好了，或者你根本认为这是一件小事。

但是你没有意识到，如果他不在意，就不会一而再再而三地提醒。

## 分手的演变过程：从警告到决裂 🔄

理解了定义后，我们来看看这类分手是如何一步步演变的。

之后你们会因为这些小事上升到分手，因为他想通过提分手的方式，让你做出让步和改变。

如果你真的意识到了问题并且改变了，那么之后就会没事。如果继续一如既往，再一次分手就会演变成真性分手。

**核心公式**：`小矛盾忽视 + 承诺未兑现 + 重复失望 = 信任崩塌 → 真性分手`

这时再去道歉是没有用的。狼来的故事大家都听过，一开始他相信你会改，但是你一次又一次地糟糕表现，让他攒够了失望，就不会再次相信你。

## 复合的核心与初期禁忌 🚫

上一节我们介绍了分手的演变过程，本节中我们来看看复合的关键所在。

这种类型分手的核心原因是**不信任**，并不是不爱了。因此，复合的核心就是要重新地建立信任。

但是在分手初期一定要注意，纠缠是没有用的，只会让情况变得更糟。前任也只会越来越排斥你。

以下是分手初期必须避免的行为清单：
*   频繁打电话、发信息道歉或求复合。
*   去对方住处或公司围堵。
*   通过共同朋友施加压力。
*   在社交媒体上发布悲伤或指责性的内容。

所以你只有先去照顾好前任的情绪，等关系稳定了，再去解决你们之间的问题。
