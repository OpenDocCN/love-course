在本节课中，我们将要学习如何挽回一位持有“固有心态”的前任。这种心态表现为态度极其坚定、不删除也不拉黑你，但采用冷暴力方式不予回应。我们将深入分析这种心态的成因，并找到有效的应对策略。

## 理解“固有心态” 🧠

上一节我们介绍了课程主题，本节中我们来看看什么是“固有心态”。

无论你对前任说什么，他都不予理睬，不删除你也不拉黑你，完全处于一种冷暴力的状态。遇到这种状态应该怎么去挽回，你需要认真理解。

首先，你需要从他的思维角度思考一个问题：他为什么对你有这样的态度？需要明确一点，能做这种事情的人，通常内心很坚定。这个人很有原则，很有自己的框架。

我们可以将其理解为心理学上的一种状态，叫做**固有心态**。其核心公式可以概括为：

**个人认知 == 绝对真理**

这是什么意思呢？例如，我对一件事情的认知是“它是方的”，它就绝对不会变成“圆的”。无论你怎么说、怎么解释“它就是圆的”，也改变不了我的看法。

他内心极度固执，认定的事情非常有自己想法，很少能够被别人轻易影响。这种人不仅坚定，而且很有韧性。如果加上你之前的一些做法，让他认为“你就是这样的，你改变不了，那你就不适合我，没有我想要的未来”，那么他的结论就是：

**“我们不可能再成为男女朋友关系。”**

这种认知扎根在这样的人心里。他也会不断地用这种认知，去否定你对他的了解。所以，之前无论你说什么，试图改变他的想法，任何试图改变这个人行为的一切努力，都会以失败告终。他根本就不为所动。

## 寻找挽回的突破口 🔑

理解了“固有心态”的顽固性后，本节中我们来看看如何找到挽回的突破口。

在这种情况下我们该如何去做，或者说怎么去找到一个突破口？唯一的办法是：**让他自己改变对你的看法**。而且这种改变必须是潜移默化的。

那么，如何去改变呢？关键在于：**在不暴露需求感且合理化的相处过程中，把你新的状态展示给他看**。

这种状态展示，不是一天两天就可以完成的。其周期根据情况的难易程度所决定。我之前见过分手半年才挽回的，也见过分手一两个月就挽回的，最后对方的确回来了。这个办法如果运用得恰到好处，真的很好用，但是使用的时间比较长，非常考验你的耐心。

所以，如果你确实遇到了一个有这样“固有心态”的人，并且也确确实实想挽回对方，那么在你情感修复的过程中，这大概率是唯一有效的途径。你明白了吗？

## 核心行动指南 📋

以下是基于以上分析，总结出的具体行动步骤：

1.  **彻底停止争辩与说服**：认识到任何直接的言语说服对“固有心态”者都是无效的，立即停止试图通过讲道理改变他想法的行为。
2.  **进行彻底的自我改变**：将焦点从“改变他”转移到“改变自己”。你需要有真实的、积极的、与过去不同的状态提升。
3.  **创造“合理化”的展示机会**：通过共同朋友、社交媒体或必要的低频次、低需求感的联系，自然地让他看到你的变化。例如，在朋友圈分享你积极生活的动态，而不是直接@他或诉苦。
4.  **保持极致的耐心**：做好长期准备。改变一个固执者的认知如同滴水穿石，需要时间和持续一致的正面展示。
5.  **专注于自身成长**：将挽回期视为自我提升期。无论最终结果如何，一个更好的你都是最大的收获。

## 总结与鼓励 💪

本节课中，我们一起学习了如何应对持有“固有心态”的前任。

我们了解到，面对这种不删除、不拉黑但冷暴力的状况，核心问题在于对方内心固化的负面认知。直接对抗或说服是无效的。唯一的突破口是通过**潜移默化的自我改变和状态展示**，让对方自己动摇并改变对你的看法。这是一个考验耐心和策略的过程，但也是走向真正关系修复或自我成长的必经之路。
