# 如果分手后想要重归于好，必须要跳过这几个坎-P1-泊明远-BV1Cg411y773_note

在本节课中，我们将学习分手后试图挽回前任时，几乎所有人都会遇到的两个主要“大坑”。理解并避开这些陷阱，是成功修复关系的第一步。

分手后，每个人都急切地想挽回前任。在这条路上，你们必然会遇到两个大坑。大部分人不知道自己失去的是什么。如果不弄清楚这一点，就会像无头苍蝇一样。你知道自己需要改变，但不知道具体该改什么。

于是，很多人会采用健身、看书、做饭这些老套路，然后带着所谓的“真心”去找前任求复合。毋庸置疑，这通常会遭到对方的拒绝和绝情。这就是第一个坑：在挽回时，你的贪婪和急迫会不断把对方推远，而你自己往往意识不到这一点。

如果不改变这些行为，就永远无法走上正轨。你需要思考的是，你到底哪里不讨人喜欢。或者可以确切地说，你一开始也是讨人喜欢的，也是能发光发热的。和你在一起的伴侣，最初看到你是高兴的。但稳定关系的副作用，就是疲惫和懈怠。

当我们双方度过了热恋期，又没有那么多的好奇心和探索欲之后，彼此的缺陷就会暴露出来。以下是几种常见的缺陷表现：

*   **缺乏安全感**：有的人极度缺乏安全感，但通过过度索取或控制对方来满足自己。
*   **原生家庭影响**：有的人受原生家庭影响，会不自觉地带着父母的弱势行为方式去处理亲密关系。
*   **失去自我**：有的人慢慢失去了自我，随着时间推移，颜值下降、身材走样、不思进取、得过且过。

哪怕一开始，你确实能让对方感到高兴，现在你也做不到了。再加上分手后，你的失控、急功近利和纠缠，会把局面变得越发糟糕。

回想一下，你是怎么变成今天这个样子的？变成这个不讨人喜欢的样子。甚至分手后，去麻烦朋友帮忙，弄得朋友都翻脸了。在这一点上，没有人能够告诉你答案。

而这，就是第一个坑。一旦迷失，找不到自己的问题所在，就永远不会有跳出来的想法。
