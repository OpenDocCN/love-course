# 情感课程 P1：分手后男生的心理真相 🔍

在本节课中，我们将要学习男生在分手后才会逐渐明白的一些心理真相。课程将重点分析男生的“膨胀心理”及其变化过程，并探讨女生在应对时应避免的常见错误。

## 男生的膨胀心理 💭

上一节我们概述了课程主题，本节中我们来看看男生的膨胀心理具体是如何形成的。

当一个男生被一个女生深深爱着的时候，他总是容易高估自己。他会觉得自己的魅力特别大。你越是爱她，她的内心就越是充满了自信。

## 膨胀心理的后果与现实打击 🎯

了解了膨胀心理的成因后，我们来看看这种心理会导致什么后果，以及现实如何打破这种幻想。

这种膨胀会让他看不清事实。他会认为你离开了她，你就再也找不到更好的。她离开了你，那她随时可以开始更好的恋情。

可是当她真正的失去，并开始寻找新欢的时候，往往就会被现实所打脸。当时还觉得不错的女生，发现并没有那么好。

而且随着接触的慢慢加深，她会发现她的魅力，并没有她想象中那么大。

## 男生的挽回心思与女生的错误应对 ⚠️

当现实击碎幻想后，男生的心理会发生关键转变，而女生的反应往往决定了关系的走向。

这个时候她就开始动了挽回的心思。

但是很多女生并不了解男生的这一点。反而看她一回头，就又卑微祈求去求复合。然后让她的自信再次爆表。

这样怎么能够复合呢。

## 总结与建议 📝

本节课中我们一起学习了男生分手后的心理变化轨迹：从因被深爱而产生的**膨胀自信**，到失去后寻找新欢时遭遇**现实打击**，最终可能产生挽回想法。关键在于，女生若以卑微姿态回应其回头，只会再次助长对方的**膨胀心理**，无助于健康复合。
