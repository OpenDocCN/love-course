# 焦虑型依恋：如何打破“高开低走”的困局？ - 课程 P1 🧩

在本节课中，我们将要学习焦虑型依恋在亲密关系中为何容易陷入“高开低走”的模式，并探讨如何通过“角色转变”来打破僵局，重新掌握关系的主导权。

## 从“索取者”到“创造者”：理解角色转变 🔄

明明一开始彼此都十分投入，最后却演变成焦虑型不断地索取，对方不断后退的局面。

如何才能突破这种形式，拿回关系当中的主导权呢？

我发现，在这场较量中获胜的焦虑型，无一例外，他们都会经历一个叫做角色转变的过程。

上一节我们介绍了关系陷入僵局的普遍现象，本节中我们来看看破局的关键——角色转变。

### 两种角色状态：索取者 vs. 创造者

**索取者的状态是情感匮乏向外索取**，需要伴侣的时刻回应来证明爱的存在。

例如，当对方暂时没有回复消息时，焦虑型的情绪就会产生一系列判定：

*   他肯定没有看到。
*   他装作没有看到。
*   他是不是有了别的打算？
*   他这是在逼我分手吧。

并且会找各种各样的证据来佐证自己的猜想。

而这个时候，你的伴侣是完全没有办法去处理这种炸裂式的情绪轰炸，他更不可能从中看出你的内心脆弱和渴望。

他们会对这种极端情绪感到困惑，觉得你敏感多疑反应过大，最后不得不起用防御机制来进行应对。

**创造者的状态是内心充盈向外给予**，他们看待问题会更加的客观、理性和真实。

他们也明白人是具有独立性的，当自己不再患得患失，不再去强求对方必须认可和服从的时候，才是一段关系轻松舒适、良性发展的基础。

## 如何实现角色转变？ 🛠️

那我们应该如何从索取者到创造者的角色进行转变呢？

我建议大家从意识和行为的角度去切入，尽量让自我意识和行为达成统一。

意识和行为不同意（即知行不合一）的后果是：在意识层面，你明白发泄情绪是不对的，但行为上依然无法掌控。

改变其实分为两个步骤。

### 第一步：认知觉醒

首先是认知层面上的觉醒，学会理性和客观地看待问题。

以下是需要重新审视的两个核心认知：

1.  **关于自己的不安**：你的不安和焦虑，更多的是不是来源于小时候没有获得父母足够的关爱和认可呢？这份爱的缺口其实并不应该由伴侣来进行填补。
2.  **关于对方的行为**：不对他的行为进行过度的解读，也不把自己的付出当做筹码。例如，你给对方准备一个礼物，出发点应该是“送这个礼物让他开心”，而不是“必须看到他有一定的回馈才觉得满足”。

### 第二步：行为实践

当你明白了这层逻辑，我们再落实到第二层：**从现在开始，假装自己是一个情绪稳定的人**。

当你想要发泄情绪的时候，你只需要告诉自己：“我是一个成熟的情绪稳定的人，而发泄情绪并不能够达成我最终的诉求。”

你将试着用更加成熟冷静的方式来解决问题。

以下是可供尝试的替代行为：

*   去等待对方的回应。
*   好好地表达自己内心深处最真实的感受。

而不是用嘶吼或者发泄的方式去进行攻击。

其实这也是反复训练自己的一个过程，在一次次的纠正过程当中，你的情绪处理开关会变得更加的可控，最终意识和行为达成统一。而对方也会因为你的改变而改变。
