# 前任动不动就要和你分开，到底在：高田横版82-泊明远-BV15

在本节课中，我们将要学习一种常见的情感关系现象：频繁提出分手背后的真实心理动机。我们将剖析这种行为背后的核心逻辑，帮助你理解对方并非真正想要结束关系，而是出于其他情感需求。

---

那些动不动就提分手的人，到底是怎么想的？他是真的想要和你分手吗？

当然不是。情绪上头时，什么话都能说出来。如果他动不动就和你吵架，并且一吵架就说分手、想要放弃这段感情，那只能说明他内心对于这段感情是缺乏安全感的。

他提分手并不是真的不爱了，更多是在和你赌气。他就是想用这样的方式，来引起你的重视，让你更在乎他。

真正想要离开的人，从来都是静悄悄的。那么，你有遇到过吗？

---

上一节我们介绍了频繁提分手行为的表面现象，本节中我们来看看其背后的核心心理机制与应对思路。

以下是关于这种行为模式的几个关键点：

*   **核心动机是寻求关注**：提出分手常是一种**引起重视**的策略，其公式可表示为 **“提出分手 = 寻求关注 + 测试感情牢固度”**。
*   **本质是安全感缺失**：频繁争吵并伴随分手威胁，是内心**安全感不足**的外在表现。
*   **与真正分手的区别**：真正的离开往往是沉默且决绝的，而情绪化的分手宣言更像是一种**情感上的呼叫**。

如果你现在也正在面临这样的情感困境，那么可以寻求专业的帮助。
