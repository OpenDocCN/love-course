# 分手后不犯错，就几乎等同于和好！ - 课程01：重建关系的核心认知 🧠

在本节课中，我们将要学习分手后如何避免常见的错误行为，并理解重建关系的核心逻辑。关键在于停止自我感动式的纠缠，认清现实关系，并采取正确的行动方向。

上一节我们介绍了课程的整体目标，本节中我们来看看为什么错误行为会阻碍和好。

分手后，你的卑微和纠缠才是他厌恶你的根源。这只是自我感动的行为。这些行为给对方带来的都是为难和不理解。因为你只关注了自己的情绪和情感需求。

在一段好的关系里面，一定是彼此都需要对方存在，并且在对方那里得到了自己想要的。仔细思考一下，你可曾给过他想要的。别跟我说什么爱他，只要回来你就会改。这话狗都不信。所以说不要再犯错，那你的和好就已经成功了。

不论你现在是在纠结和不和好，还是说想和好不知道该怎么做，都认真看。在理解这个内容之前，我们要先搞清楚他为什么会回来。

本质上说，你具备他离不开你的优点，或者你不可以被其他女人所替代。再或者，你是他身边最优秀的那个人。但是不论哪种原因，你都要记住了，卑微、纠缠、歇斯底里和威胁，都不可能成为他回来的理由。

我的评论区里面也经常出现，分手后我威胁他辱骂他，但是我后悔了。现在我感觉我们两个之间不可能了。别担心，这并不是什么大事。前提是你能够听进去我所说的。

以下是分手后最常见的三个错误心态，认清它们是改变的第一步。

1.  **怀疑自己**：也可以说从内心深处对自己不认可，企图通过一些对方担心的事情和人，让对方回头。其实你们分手的事实已经成立了，即便是他现在很在意的东西和人，你的身份也影响不了他的决定。
2.  **摆不清自己的位置**：你们分开其实已经是事实了。这个时候很多女生还把自己放在女朋友的位置上。你去要求他不能做什么，去监督他和别的异性交往等等。你要认清自己的身份，你现在只是前女友。他有权利去追求别的女生。而他也有权利去厌恶你。同样，你也有权利去挽回他，就让他再爱上你不是吗。所以说，如果你再想和他走到一起去，一定先认清自己和他的关系到底是什么。只有这样才能有针对性的做事。
3.  **纠结他还爱我吗**：在你的眼中，爱是感情的所有，从来不去考虑本质，一直去纠结。总在想他爱我的话和我才有意义，那不爱的话就没有意义了。你知不知道，其实还有另外一种可能，就是你完全可以做到，再让他爱上你。

所有分开的女生都应该给自己一个决心。结局只有两个：**在一起** 或者是 **结束**。纠结犹豫只会毁了你这段感情，让自己陷入到乱糟糟的情境当中。

最后关于和好，不犯错其实已经可以让大多数的感情走下去了。太多太多的感情结束，原因都是因为分手之后的那些神操作。所以你要做的就是千万别犯傻。该怎么做，能怎么做，三思而后行。如果想不通的话，可以到评论区说出你的问题。

本节课中我们一起学习了分手后阻碍和好的核心错误心态：自我怀疑、定位不清和纠结于“爱不爱”。记住，停止错误行为是成功挽回的第一步。关键在于认清分手事实，摆正自身位置，并专注于如何重新创造吸引力，而非沉溺于过去的情绪中。
