# 自律，才是正确的挽回方式 - 课程 P1 🧘

在本节课中，我们将要学习分手后如何通过“自律”来正确地进行关系修复。我们将探讨为何克制比纠缠更重要，并分析导致感情破裂的核心情绪问题。

相信你们都听过这么一句话，叫做喜欢是放肆，爱是克制。克制才能让他回头找你，克制才能够让他更爱你。所以分开之后，不要因为你自己对他的那些需求感，你很想他，想找他说话聊天，所以忍不住地去找他。

这样做其实真的没有用。这么多的前车之鉴告诉我们，你控制不住自己的情绪和需求感，去和对方建立互通联系的时候，只会让他越来越讨厌你。而且你这么做，真的不能代表你很喜欢他，你只能证明自己是一个自私，而且歇斯底里的人。

所以你们要明白，真的是有些情绪你们得克制，不应该出现在感情当中，特别是在分手挽回的这个阶段。

那么问题来了，为什么曾经一个很喜欢你，很爱你，对你百般的包容和忍耐的人，他会突然间变得就不爱你了。

上一节我们提到了克制的重要性，本节中我们来看看导致感情变化的具体原因。以下是几个核心原因：

*   **你的坏情绪没有克制，所以他很累。**
*   **你的好奇心没有克制，所以他觉得很绝望。**
*   **你的控制欲没有克制，所以他觉得很恐怖。**

他最终想逃离你。那些多疑、敏感、不自信、患得患失的情绪，等到他彻底扛不住的时候，就成为了压垮骆驼的最后一根稻草，结果导致你们就彻底分开。

那么你在这段感情当中，无论是你的情感、生活，乃至于各个方面，都是依附于对方存在的。对于他来讲，他在和你谈恋爱，他就是在负重前行。一个爱你的人，喜欢你的人，他可以对你负重前行，但是他不能一直对你负重前行。因为他在爱你的同时，他谈恋爱的本质，也是需要被爱的。

你们之间是**相辅相成**的状态，才能够长久地发展一段亲密关系。这可以用一个简单的公式来描述健康的关系：**付出 ≈ 获得**。

我十分理解你们想要全身心地投入这段感情，想要去融入对方的生活。那我也理解你，想要去了解他的一举一动，事无巨细地关心他的情绪，时时刻刻地知道他在做什么。我也尊重你对于这段感情的认真和执着。

但是你要明白，这些东西并不是你肆无忌惮地去和他相处的理由。当这些东西成为理由的时候，它就不再是爱和喜欢了，而是一种抵触和厌恶。因为你是强行地把自身的认知和思维加到对方的身上，而对方他并不想要。

今天老师并不想教你们什么所谓的话术、技巧。因为我觉得感情最终本身还是要回归到你的身上。如果你的认知一直停留在过去，你们觉得一个人他跟你谈恋爱了，他就要永远地喜欢你为你付出，那么对你这个好一成不变，那么很遗憾地告诉你，你的爱情是不会走到最后的。

你的**托付心理**这么重，怎么可能收获更好的爱情呢？爱是永恒，但你们之间的爱情并不是。因为如果你们相爱的话，其实根本不会分手，根本不会出问题。能分开就已经说明了你们的感情没有达到爱的这个阶段。他只是喜欢你，而事实就是没有办法把这种喜欢延续下去，把它慢慢地发酵成爱。

我是泊明远。如果屏幕前的你现在也正是面临这样的情况，那么后续课程老师会来帮你。

本节课中我们一起学习了：分手后纠缠源于需求感失控，这只会让对方厌烦。感情破裂往往源于坏情绪、过度好奇和控制欲的**不克制**，这些会给对方带来压力。健康的感情需要**相辅相成**，而非单方面依附。挽回的核心是改变自身认知，放下沉重的托付心理，理解喜欢不等于永恒的爱，并通过自律为关系创造新的可能。
