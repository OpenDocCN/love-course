# 复合心理学教程 01：无需主动求和，让前任主动回头 🤔

在本节课中，我们将学习如何通过理解对方心理、降低自身需求感以及进行服从性测试，来引导前任产生复合的念头，而非一味主动求和。

上一节我们介绍了课程的整体目标，本节中我们来看看具体如何操作。真正的去改变一个人很难很难。与其浪费时间让他理解你，不如带入到他的思维里，分析他的想法。俗话说不破不立。

如果不能互相成就，你就没有办法获得新的情感体验。你可以好好想一想，假如你是他，你愿意接受一个什么样的伴侣呢？你能接受什么样的挽回方式呢？如果你实在想不明白，我们一起来梳理一下思路。

在想清楚之后，你只需要考虑一个问题：为什么分手后，前任总是躲避你，甚至拉黑你？是不是你的一举一动，始终透露着一种高负荷需求，无形之中，给前任增加了不少的压力。

所以想要挽回顺利的话，要学会先进行**需求处理**。现在开始，把你们的聊天频率给降下来。聊天内容要从讨好卑微，偏向于理解共情。因为谈恋爱的本质是找同类。在沟通的过程当中，比起“我爱你”，“我懂你”才是他目前所需要的。

而当他习惯了你的体贴，可以突然抽离。这就会激发他的好奇心，甚至能激发他的戒断反应。**得不到的永远在骚动**。

要做到这些，离成功复合也就只有一步之遥，就是**服从性测试**。吸引对方会被投入，他开始不习惯你的疏离是一件好事，就说明他对你不再冷漠。

以下是**服从性测试**的三个等级，需要层层递进：

*   **第一级服从**：找一些他擅长的事情，请他帮忙。
*   **第二级服从**：找一些无关痛痒，甚至能够凸显暧昧的话题，最后引导他回忆以前的事情。
*   **第三级服从**：要进行进一步的互动，例如一起互怼，甚至于邀约。

如果对方接受这些测试，那就继续升级，去玩暧昧。如果对方拒绝了你，你就可以再等一段时间，然后再从一级开始重新的测试，直到全部通过为止。这个时候，哪怕你不主动找复合，对方也不会轻易的让你离开。

本节课中我们一起学习了复合的核心思路：通过降低需求感、提供理解与共情，并运用分级的服从性测试来引导对方重新投入。关键在于分析对方心理，而非强行改变对方。
