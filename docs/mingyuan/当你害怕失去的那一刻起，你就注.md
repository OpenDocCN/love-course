# 当你害怕失去的那一刻起，你就注

在本节课中，我们将要学习一个在修复关系（尤其是复合）中至关重要的核心技能：**试探**。很多人因为害怕失去而变得被动，最终导致关系失衡。我们将通过理解试探的原理和方法，帮助你重新建立框架，夺回关系中的话语权和主动权。

当你开始害怕失去的那一刻，你就注定会输得一塌糊涂，并且没有任何办法。你之所以越来越被动，一直被对方牵着走，是因为对方很笃定：惹你生气了，哄你两句就好；即便他冷落你，过两天你也会主动上门。

即便他犯了很大的错误，你都会替他不断找理由，说服自己直到妥协。所以，不必再问“他为什么这样对我”。他其实已经在生活细节中，通过一点一滴的试探，完全掌握了你。你也因此给了他伤害你的资本。

他为什么还会去付出、去用“舔狗”般的行为迎合你？在关系中，除了学会捍卫自己的框架，还有一个很重要的技能就是**试探**。就像别人不断地在试探你一样，你也要在他的底线边缘进行试探。这样才能为我们争取更多的利益空间和话语权。

## 如何进行有效试探

以下是进行有效试探的具体步骤和逻辑。

1.  **从小处开始，逐步推进**
    在你一点点的试探中，如果对方可以接受，那么这个程度就是安全的。然后我们再慢慢地继续往下试探。

2.  **观察反馈，设定边界**
    直到对方有了爆发性的反馈，或者你觉得已经试探到底线了，我们就立刻以玩笑或轻松的形式收回。

3.  **明确认知，心中有数**
    经过这个过程，在你的认知里，你和他之间应该如何相处，他能接受到哪个点，你的心里就有数了。

## 理解“关系圆圈”理论

上一节我们介绍了试探的步骤，本节中我们来看看如何形象地理解这个过程。我们可以把双方的可接受范围想象成一个圆圈。

我们博弈的目的，就是把这个圈一点一点地往外扩大。当我们了解到双方真正可以接受的范围有多大时，那么这个圆圈内的空间，就是我们和他交往时可以肆意活动的安全范围。

但是，当你比他更害怕失去时，你是不敢给自己争取更多利益的。这个道理不仅适用于两性关系。

## 试探的普适性

其实，除了两性关系，在其他社会关系当中也是如此。别人都是在不断的试探中，然后对你形成一个定位，再按照这个定位去对待你。
