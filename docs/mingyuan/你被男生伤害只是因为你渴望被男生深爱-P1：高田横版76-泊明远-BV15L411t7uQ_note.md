# 课程：情感认知与自我疗愈 - P1：理解情感依赖的本质 🧠

在本节课中，我们将要学习如何理解在一段不健康的情感关系中感到受伤的核心原因。我们将剖析“渴望被深爱”这一心理状态如何导致痛苦，并初步探讨转变的方向。

---

上一段我们提到了在感情中感到受伤的普遍现象，本节中我们来看看其背后的具体心理机制。

你被男生伤害，只是因为你仍然渴望被这个男人深爱。简单来说，就是你期望他变回那个曾经深爱你、对你负责、对你温柔、能够为你撑起一片天的男生。

以下是这种心理状态通常伴随的行为表现：

*   你一直在委曲求全。
*   他却迟迟没有改变。
*   他可以在外面和别的女生暧昧。
*   他也可以对你爱搭不理，挑三拣四。
*   你却只能默默地承受这种委屈。

感情好像变成了你一个人的独角戏。归根结底，这只是因为你还渴望被这个男人深爱。他曾经对你的温柔与包容，让你依然对他抱有期待。

---

如果你发现自己正陷入这种状态，并且希望改变现状，那么认识到问题是第一步。本节课中我们一起学习了情感伤害与内心渴望被爱之间的直接联系。理解“**伤害源于未满足的期待**”这一核心公式，是迈向自我疗愈和关系重建的关键起点。在后续的课程中，我们将探讨如何具体地放下执念并找回主动权。
