# 分手后就是忍不住想去联系前任应：高田横版92-泊明远-BV1P

在本节课中，我们将学习分手后如何有效管理情绪，停止忍不住联系前任的冲动。我们将通过分析心理机制并提供具体步骤，帮助你走出困境，专注于自我成长。

## 理解对方行为的本质

上一节我们介绍了课程目标，本节中我们来看看联系冲动的根源。你需要明白一个核心事实：他不联系你，意味着他心里没有你。

如果他心里有你，他一定会主动联系你。你在这边伤心难过，他可能在别处轻松愉快。你感到痛苦，主要是因为分手后只记住了美好的回忆。

## 扭转回忆焦点

以下是扭转回忆焦点的具体方法：

*   现在请努力回忆他曾经如何伤害你。
*   如果记忆中全是美好，你们当初就不会分手。

## 审视放不下的对象

接下来，我们审视你真正放不下的是什么。你放不下的可能不是他这个人。

你真正放不下的，可能是有人陪伴的感觉。解决方法就是去寻找新的陪伴。

你放不下的，也可能是自己曾经的付出，心有不甘。你需要承认自己当初看错了人，不必不甘心，下次好好珍惜即可。

## 转向自我提升

以下是行动的核心方向：

*   把精力集中在如何让现在的生活变得更好。
*   专注于提升自我。
*   当你变得足够优秀时，自然就想不起他是谁了。
