上一节我们介绍了本课程的主题，本节中我们来看看第一种表现。

## 表现一：矛盾的心理拉锯战

第一种表现是当事人内心陷入激烈的矛盾。这种矛盾可以用一个简单的**公式**来描述：

**“思念驱动” vs “尊严驱动”**

以下是这种矛盾心理的具体体现：
*   一方力量（思念）驱使个体去联系、挽回对方。
*   另一方力量（尊严或理性）则告诫个体应该放弃，保持体面。
*   这两种力量持续对抗，导致行为上的犹豫和反复。

理解了内心的矛盾后，我们接下来看看第二种更为波动的情绪表现。

## 表现二：反复无常的情绪周期

第二种表现是情绪在短时间内剧烈波动，无法稳定。这种状态类似于一段**代码**中的无限循环，在两种极端情绪间切换：

```python
while True: # 情绪循环
    mood = "想通了" # 瞬间释然，感觉能向前看
    # ... 短暂间隔后 ...
    mood = "想不通" # 再次陷入痛苦，觉得未来毫无意义
```

以下是这个情绪周期的两个关键节点：
*   **瞬间想通**：在某个时刻，个体感觉自己已经释怀，能够走出阴影。
*   **下一秒想不通**：但很快又陷入沮丧，认为失去对方后未来变得没有意义。
