## 重塑吸引力的核心：塑造核心竞争力

上一段我们提到了“重新吸引”的概念，本节中我们来看看其核心——塑造核心竞争力。这意味着你需要拥有属于自己、他人无法替代的优势。在当今快节奏、充满诱惑的现实环境中，分手后对方无疑会面临更多选择。我们的目标是让对方感到无法离开我们，一旦离开便会承受巨大损失。

试图改掉所有缺点来迎合对方是反人性的做法。这种方法或许暂时有效但难以持久，并且和好后可能衍生出新问题，导致你需要不断改变。因此，关键在于塑造你的核心竞争力。当对方理性衡量得失，意识到离开你将付出惨重代价、弊大于利时，他便自然难以离开你。

## 塑造核心竞争力的三个步骤

以下是塑造个人核心竞争力、实现重新吸引的三个具体方法。

**第一，强化既有优势。**
回想你身上已有的优点，特别是对方曾经常常夸赞你的地方。如果实在想不起，可以思考在朋友眼中你最突出的优点是什么。然后，深耕并强化这个优势，使其足够突出和耀眼。

**第二，创造新优势。**
如果你在对方心中的形象已经彻底崩塌，甚至令其反感，那么继续强化旧优势可能适得其反。此时，你需要分析对方当前最看重或最需要的是什么，并针对这一点重新塑造一个核心优势。这个新优势需要塑造到能让对方感到惊喜和刮目相看的程度，即**超出对方预期**，而非仅仅满足其需求。

**第三，打造差异化优势。**
如果以上两招均不适用，你还可以选择打造差异化优势。具体做法是：做到与大多数同性不一样。简而言之，就是 **“人无我有，人有我优”** 。找到一个大家都难以做到，唯独你能做到并且能做得更好的方面。当你成功塑造出这样的优势，并且它足以掩盖其他缺点时，你就形成了强大的核心竞争力。
