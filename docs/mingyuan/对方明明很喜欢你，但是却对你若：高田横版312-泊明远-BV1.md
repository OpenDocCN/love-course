# 对方明明很喜欢你，但是却对你若：高田横版312-泊明远-BV1

在本节课中，我们将要学习一种在亲密关系中常见的心理模式——回避型依恋。我们将探讨其核心特征、背后的心理动因，以及它对关系的影响。理解这些内容，有助于我们更好地解读伴侣的行为，或进行自我觉察。

明明很喜欢对方，却偏要刻意疏远，这是一种典型的回避行为。这种心理往往源于个体从小缺乏安全感，从而形成了回避型依恋模式。

在心理学上，现实中的回避型依恋具有七个显著特征。

以下是这七个方面的具体表现，你可以对照观察，你的恋人是否具备以下几点。

1.  **刻意疏远亲密关系**：即使内心非常喜欢对方，也会主动保持距离。
2.  **逃避沟通或缺乏沟通能力**：倾向于将正常的沟通演变为争吵。
3.  **表达能力弱**：习惯通过说反话或做相反的事来表达真实情感。
4.  **缺乏安全感**：认为感情脆弱不堪，不相信任何关系能够长久稳定。
5.  **鄙视亲密**：将恋爱视为一种侮辱，害怕被感情束缚。
6.  **在感情中缺乏自信**：表面看似处于高位，实则对自己要求苛刻，不允许自己犯错或出丑，认为只有“强者”才值得被爱。
7.  **认为克制和痛苦是幸福的支撑**：并非不想被爱，而是认为必须与逃避相结合，这份爱才更有安全感。

通过以上几点，我们不难发现，回避型行为者对亲密关系的整体态度是矛盾的。他们渴望接近你，却又不敢接近；他们需要被爱，也需要去爱，甚至比常人更渴望温暖。

然而，正常人面对亲密关系的态度是：**喜欢 → 所以我接近**。

而回避型依恋者的态度则是：**喜欢 → 但喜欢可耻/没用/会受伤 → 所以我逃避**。

本节课中，我们一起学习了回避型依恋模式的七大特征及其背后的矛盾心理。理解“渴望爱”与“恐惧爱”并存的复杂心态，是处理此类关系的第一步。如果你或你的伴侣符合上述情况，认识到这是源于安全感缺失的心理模式，而非简单的“不爱”或“故意刁难”，将是改善关系的重要开端。
