# 分手后还能做朋友吗？- 课程 P1：高田横版 20

在本节课中，我们将探讨一个常见的情感问题：与男生分手后是否应该做朋友。我们将分析其背后的心理动机，并帮助你理清思路，做出适合自己的决定。

## 核心观点分析

上一段我们提出了问题，本节中我们来看看关于分手后做朋友的核心观点。通常，分手后双方仍保持朋友关系，主要源于以下两种情况。

以下是两种常见情况的详细说明：

1.  **双方从未真心相爱**：关系本身缺乏深厚的情感基础，分手后转为朋友关系相对容易。
2.  **一方仍抱有希望**：其中一方希望通过“朋友”的身份维持联系，以期挽回关系或缓解分手痛苦。

## 关系状态的本质选择

在明确了上述动机后，我们需要理解亲密关系结束后的本质。在一段已经结束的恋爱关系中，实际上只存在两种根本选择。

以下是两种根本性的路径：

*   **彻底分离**：选择断绝联系或不互相打扰，让彼此开始新的生活。
*   **重新复合**：双方决定重新在一起，给彼此的关系一个明确的未来。

## “做朋友”的潜在动机

那么，主动提出分手后做朋友，其背后的主要动机是什么呢？核心目的往往是**安抚情绪**。

这个行为不仅是在安抚对方的情绪，更是在安抚提出者自己的内心。其潜台词是：“我并非绝情之人，我为大家保留了余地。”但这最终变成了对方是否接受这个“台阶”的问题。

而对于被分手的一方来说，同意做朋友常常是**以朋友之名，行挽回之实**。他们希望通过这个身份保持联系，寻找机会重建亲密关系。

## 总结与行动建议

本节课我们一起学习了分手后做朋友的复杂心理。综上所述，在上述两种情况下，几乎无法建立起**单纯的朋友关系**，因为总有一方“图谋不轨”。

然而，如果我们换个角度思考：无论对方目的如何，对方选择做朋友，毕竟保留了一个**沟通渠道**。

**如果你心有不甘，不愿就此放手，那么可以抓住这个“做朋友”的机会，尝试去挽回对方。** 关键在于，你需要清醒地认识到当前关系的本质，并明确自己的目标。
