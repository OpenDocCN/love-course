# 情感修复课程 01：分手后未互删，如何判断复合可能性？ 💔

在本节课中，我们将要学习如何分析分手后双方未删除联系方式的潜在含义，并判断关系修复的可能性。核心在于通过对方的互动态度，识别其真实心理状态。

很多分手后的女性会产生一个想法：虽然关系结束了，但双方并未互相删除或拉黑联系方式，这是否意味着复合的机会特别大？客观地说，这种情况的结果因人而异。你们分手后没有互删，只能说明一个问题。

要么是双方未曾真正爱过对方，要么是彼此心里仍存不舍与幻想，渴望未来某天能够重修旧好。不删除联系方式，就是为彼此保留的最后一条退路。如果是你自身舍不得，就需要判断对方属于哪种情况。

那么，如何判断呢？方法很简单：观察他回复你消息的态度。以下是几种关键的态度类型及其含义：

*   **态度热情且随意**：如果他回复时毫无顾忌，对话依然热情，可以无所不谈，甚至言语间带有轻浮成分，那就说明他已经不爱你了，至少现阶段已无爱意。
*   **态度冷淡或沉默**：如果他回复得很冷淡，甚至不回复你的消息，这恰恰可能表明他内心舍不得。因为每一次与你的对话，对他而言都可能是一种内心折磨。他想和你交流，但现实会不断提醒他你们已分手的事实，因此他会**压制自己，克制自己的热情**。

上一节我们介绍了如何通过回复态度判断对方心理，本节中我们来看看这对复合意味着什么。对于后一种情况——即对方因不舍而表现得冷淡或回避——只要你采用正确的方法与他相处和沟通，他回心转意真的只是时间问题。

本节课中我们一起学习了如何解读分手后未互删行为背后的心理，以及通过沟通态度判断复合可能性。记住核心原则：**热情可能代表释然，而克制可能意味着在乎**。我是泊明，如果你正面临类似的情感困境，可以通过私信寻求进一步的修复指导。
