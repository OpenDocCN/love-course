## 核心概念：情感空窗期

上一段我们提到了断联，本节中我们来看看其背后的关键心理阶段——空窗期。

一个真正爱过你的人，分开后，他的精神世界会进入一个疲劳期。这个阶段就是我们所说的**空窗期**。因为爱累了，他不可能很快再去爱上另外一个人。

**公式：真挚情感结束 -> 精神疲劳期（空窗期）-> 无法快速开始新恋情**

## 断联的作用：积累思念值

理解了空窗期的存在后，我们再来探讨断联在这个阶段扮演的具体角色。

在空窗期里，他会去复盘、去纠结、去想你。而断联，就是为了增加他对你的思念值。等到这个值达到了一定的程度，他才会主动来找你。

**核心逻辑：断联 -> 增加思念值 -> 思念值达到阈值 -> 主动联系**

## 行动指南

以下是基于上述原理，你可以采取的核心心态调整：

*   **停止担忧**：无需过度焦虑对方在断联期间会立刻有新欢。
*   **理解过程**：认识到他的情感恢复需要时间，空窗期是正常心理过程。
*   **保持断联**：坚持断联策略，为对方留出思念和反思的空间。
