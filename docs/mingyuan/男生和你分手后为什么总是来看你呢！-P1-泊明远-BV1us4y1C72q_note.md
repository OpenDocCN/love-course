## 核心概念：多看效应

心理学家乔曾进行过一项实验。他向参与者展示一系列异性的照片，展示频率分别为**20次**、**10次**和**1次**。随后，他请参与者评价对照片中人物的喜爱程度。

实验结果表明，**出现频率最高的照片，获得的喜爱程度也最高**。这意味着，单纯的**重复曝光**就能有效提升好感度。这个现象在心理学上被称为“多看效应”或“曝光效应”。

公式可以简单表示为：
**曝光频率 ↑ → 熟悉度 ↑ → 好感度 ↑**

上一节我们介绍了“多看效应”的实验基础，本节中我们来看看它如何应用于实际的情感修复。

## 原理的应用：创造存在感

基于上述原理，我们在指导修复关系时，会建议你经常在社交媒体发布动态，例如分享生活或发布自拍。同时，也可以策略性地创造一些“偶遇”的机会。

其核心目的是：**持续在对方的世界里刷存在感**。因为人类天性倾向于对更熟悉的事物产生好感。即使你们之间曾有过激烈矛盾，但当对方情绪平复后，他依然会倾向于与最熟悉的人交流。

以下是应用此原理的几个关键点：
*   **保持可见**：通过间接方式（如朋友圈）让对方持续看到你的状态。
*   **展示积极面**：内容应侧重于展示你平静、积极的生活，而非抱怨或悲伤。
*   **自然不刻意**：所有的“曝光”都应显得自然，避免目的性过强。

这就是我们常说的“日久生情”背后的科学依据。无论是在恋爱中，还是在修复关系时，**存在感到位是后续一切进展的基础**。
