## 🔍 核心心理分析

上一节我们介绍了本课程的主题，本节中我们来看看具体的原因分析。

对方不删除你，也不理你，是因为在他心里，你的位置依然存在。他内心对你仍有感情，或者说，他还没有完全将你从他的情感世界中移除。

他之所以选择不回复你的消息，是出于一种顾虑。他担心一旦回复了你，可能会引发你的持续联系或情感上的纠缠。这种互动可能会对他当前的生活状态或心理平静造成**负面影响**。

我们可以用一个简单的公式来描述这种心理权衡：

**回复风险 = 潜在的情感纠缠 + 对现有生活的干扰**

当对方评估这个“回复风险”过高时，便会选择沉默。
