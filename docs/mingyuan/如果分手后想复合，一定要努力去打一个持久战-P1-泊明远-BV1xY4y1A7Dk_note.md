# 复合教程 01：持久战心态建设 💔

在本节课中，我们将要学习在分手后寻求复合时，为何必须建立“打持久战”的心态，以及这种心态的重要性。

上一节我们介绍了课程主题，本节中我们来看看为何复合需要持久战心态。在复合的时候，我们要做好打持久战的准备。如果你们是真心分手，那么在挽回前任时，就必须进行重要的心态建设。不要想着两三天就能够成功挽回对方。如果挽回是那么容易的事情，世界上就不会有那么多为情所困的人了。

既然你们已经闹到了分手的地步，那肯定不是因为一点点小矛盾或小摩擦。这意味着你们的感情已经出现了比较大的问题。而这个问题，并不是那么容易就能解决的。

当对方决定要跟你分手、结束你们的关系时，无论你们是否保持联系，关系是否已经断裂，或者已经陷入反复分合的循环，只要你想要去复合，就一定要做好打持久战的准备。

你以为真的下定决心要跟你分手的人，特别是男人，他是非常理性的动物。他不会因为你的三言两语就跟你和好如初。他可能会权衡利弊，考虑复合之后将面临怎样的痛苦，并且已经做好了分手后的准备。

这个时候，你千万不能急于求成，逼迫对方跟你复合。你需要换一种思路，尝试去复盘，找到你们分手的根本原因。然后想办法解决你们之间的矛盾，消除彼此的误会和偏见。而这个过程，一定是非常漫长且难熬的。

以下是建立持久战心态的几个关键点：
*   接受过程的漫长性，做好心理准备。
*   避免短期内未见效就失去耐心。
*   切忌因焦虑而去闹矛盾或发脾气。

如果你确定你们是真心分手，并且你也想要去挽回对方，就一定要做好打持久战的准备。不要因为三五天没有效果，就立刻失去耐心，去闹矛盾发脾气。这样做对挽回不仅毫无作用，而且会加深对方对你的负面情绪，让你的挽回之路更加艰难。

本节课中我们一起学习了在复合初期建立“持久战”心态的核心原因。关键在于理解问题的严重性、过程的漫长性，并避免因急躁而采取错误行动，导致情况恶化。
