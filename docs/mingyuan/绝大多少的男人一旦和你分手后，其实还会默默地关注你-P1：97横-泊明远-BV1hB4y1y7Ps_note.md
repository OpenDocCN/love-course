# 分手后男性心理分析教程 🧠 - 课程01

## 男性分手后的关注行为 👀

上一节我们介绍了课程主题，本节中我们来看看男性分手后的具体行为。

绝大多数男性在分手后，其实还会默默地关注前任。他们关注前任是因为并没有完全放下这段感情。然而，他们通常不会主动联系前任，因为害怕双方的关系重蹈覆辙。

## 复杂的心理动机 🔍

了解了基本行为后，我们进一步分析其背后的心理动机。

男性在分手后还存在一种比较隐晦的心理：他们不希望、也不允许前任过得比自己更好。当他们偷偷关注前任时，如果看到对方特别伤心失落，他们会感到放心。这证明了前任舍不得他，还爱着他。

以下是这种心理的具体表现：
*   发现前任分手后过得更好、更开心时，他会开始难受。
*   心里会产生一系列疑问。
*   如果分手是由对方提出的（“被踹了”），这种心理会更强烈。
*   他会担心：前任过得这么好，打扮得这么漂亮，如果被别的男人追求怎么办？
