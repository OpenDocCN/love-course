# 分手后断联，千万不要抱有太多的：高田114横_batch-泊明

在本节课中，我们将要学习分手后“断联”这一行为背后的核心逻辑与正确心态。理解并执行这些原则，是走出情感困境、实现自我成长的第一步。

上一节我们介绍了课程的主题，本节中我们来看看断联时为何不应有过多顾虑。

当你在执行断联时，无需有过多顾虑。不必每天去窥视对方的动态，也不必担心对方是否找到了新欢。你的这些小心思和小动作，对方是完全能感知到的。这违背了我们断联的初衷。

那么，断联的初衷究竟是什么？

以下是断联的两个核心目的：

1.  **给对方制造真正的失去感**：这需要通过持续的自我调整来实现，彻底摆脱过去那种患得患失的状态。
2.  **找回自我的价值**：专注于自身重建，提升个人价值。

当你们未来重新建立联系时，你才能给对方一种焕然一新的感觉，重新带来新鲜感。

这种感觉源于你对过去的彻底告别，相当于你们重新开始了一段恋情，而不是重复过去的错误。
