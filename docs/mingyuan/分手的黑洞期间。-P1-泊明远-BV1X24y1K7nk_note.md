# 分手复合教程 01：平稳度过“至暗时刻” 🕳️

在本节课中，我们将要学习分手后一个至关重要的阶段——“黑洞期”或“至暗时刻”。这是大部分试图挽回关系的人都会经历的艰难时期，我们将探讨其核心原因，并学习如何以正确的心态和方法平稳度过。

## 理解“黑洞期” 🕳️

分手后、复合前存在一个“黑洞期”，它淘汰了约80%的尝试者。大部分人在分手后试图挽回，但在大约两个月时，心态极易崩溃。这就是无法跨越的“分手黑洞期”，我称之为“至暗时刻”。

## 必经的挫败与核心心态 🧠

只有经历了这些挫败感，才有可能取得最终的结果。这是所有希望挽回旧情的人必须面对的历程，除非当事人是游刃有余的情场高手。

上一节我们介绍了“黑洞期”的概念，本节中我们来看看成功度过此阶段所需的核心心态。

如果你能理解本视频的内容并付诸实践，你平稳度过此期的可能性将大大增加，最终复合的成功率也会显著提升。

许多人抱有“一夜暴富”的心理，试图通过一两个行动在几天内复合成功。这种情况我见过也经历过，但其概率**不足5%**。剩下的**95%** 则不会如此顺利。

那么，剩下的95%就没有机会了吗？当然有。但这要求你耐得住寂寞，克制住内心的复杂情绪。首先，你必须建立以下心态：

**公式：决心挽回 = 做好持久战准备**

当我们下定决心要挽回前任时，就必须做好打持久战的准备。这场“战争”可能持续三个月、四个月，甚至更久。

如果你声称深爱对方，认为对方特别且对你意义重大，却连两三个月都无法坚持，那这能算真正的爱吗？如果你整天只纠结于对方是否还爱你，而自己却无法坚持付出，你又如何配得上拥有这个人？

## 从抗拒到信任：时间的作用 ⏳

你需要让前任从厌恶你、抵触你，转变为认可你、期待你、信任你。这必然需要时间。

我打个比方：如果你用了半年或更长时间让对方失望透顶，或犯下大错，那么你就应该怀有真诚而坚定的信心去弥补，而不是投机取巧、反复无常、假装演戏或继续纠缠。

为什么前任会如此抗拒你？因为她认为你是一个对她没有价值、甚至有害的人。分手后你的纠缠和烦扰，只会匹配到她对你的狠心、冷漠、绝情、敷衍和放狠话。

## 导致失败的普遍原因 ❌

许多人正是在这个阶段失败了。很多人努力了一个月，心态就崩溃了。

面对她的冷漠、敷衍、绝情，一次或许能承受，但两次、三次之后，很多人就感到痛苦并放弃了。这正是那**95%** 的人复合失败的核心原因。

所以，想要复合成功，一是需要好的心态，二是需要正确的方法。心态不好，你做什么都是错的，更谈不上方法正确，做什么都不会有结果。

## 有效行动与无效行动的对比 ⚖️

因此，不必再问我“为什么做了这么多还没用”。因为你做的那些很可能本就是无效的。

请看那些成功复合的案例：他们分手后没有纠缠，没有立即采取大量行动，而是利用这段时间复盘、反思自我。等到时机成熟，他们没费多大功夫就和好了。他们的做法与你形成了鲜明对比，并且没有让局面恶化。

## 如果局面已恶化，该如何应对？ 🔄

但是，如果你已经恶化了局面，那么接下来你需要做的就是：更加耐得住性子与寂寞。因为难度变大了，她被你推得更远了，你的挽回之路也就更长了。
