# 分手后想复合，如何挽回一个对你彻底失望的人-P1-泊明远-BV1HT41137LM_note

在本节课中，我们将学习如何挽回一个对你彻底失望的人。核心在于转变思维模式，从“弱者思维”转向“强者思维”，并通过观察与理解双方的“边界性”来重建舒适的关系。

---

最近有一个问题值得探讨。一位粉丝留言询问如何挽回一个彻底死心的人。实际上，世界上没有无法挽回的爱情，也没有绝对做不到的事情，关键在于你的思维方式。

很多时候，你的“弱者思维”会让你觉得事情难以办成，但事实并非如此。我首先要问你：你知道你的伴侣想要什么吗？你是如何了解对方需求的？分析一个人的需求，往往需要你先端正自己在情感修复关系中的内在思维模式。

许多人的思维中存在一个误区：认为“我”是主体，对方是客体，是附庸于我的存在。这种思维是错误的。在现代两性关系中，两个人都是主体，彼此平等且独立。

因此，不存在谁为主、谁为辅，或谁为谁付出的问题。正确的方法是：学会观察对方的内在需求。

如何观察呢？需要分析对方的人生与家庭状态。一个人在人生中缺失什么，就会特别想弥补什么。例如，如果一个人小时候缺爱、缺乏陪伴，那么他渴望的就是陪伴。

但仅仅做到陪伴可能还不够。例如，如果他小时候与父母关系疏离，并且已经习惯了这种疏离感，那么他在恋爱中的需求就会是矛盾的：他既需要你在身边，又需要在不需要你时保持距离。换句话说，他内心渴望亲密，却又害怕过于亲密，处于一种纠结状态。

找到这种纠结的平衡点，是恋爱成功的关键。如何寻找呢？其实很简单：观察他何时对你有需求。这需要你耐心地观察和了解彼此的需求点。

有一个简单易记的逻辑：当一个人没有特别的兴趣爱好时，他的对外社交往往处于一种逆反状态，这时他会特别需要有人陪伴。当他需要陪伴时，你可以尝试给他打个电话，看他是否愿意在电话里聊下去，或者是否愿意与你见面。

如果他愿意，就表明他此刻需要你陪在身边。这时，你要不断地提醒自己，可以试探性地问他：“我明天回家吧？”或提出类似的、暗示你想离开的状态，观察他是否会挽留你。

如果他不挽留，就证明他此刻不想让你留在身边，你可以离开。这被称为“人与人之间的边界性”。当你了解这种边界性，知道对方的边界在哪里，你就能让这个人感到与你相处非常舒适和愉快。

这就是我们所说的爱。这样的恋爱，很难分手。即便分手，他之后找到的所有伴侣，可能都不如你，因为没有人能再给他同样的感觉。这就是恋爱的规则。

恋爱的规则是：通过观察对方的边界性，来了解彼此的真实需求，并学会适应这种需求。但这只是一半，另一半是：你也要有自己的需求和边界性。

我的边界性是什么？我想要什么？如何表达出来？这就引出了另一个话题：底线问题。许多女孩和男人在恋爱中无底线地妥协，不敢坚持自己真实的想法，对方一表现出强硬态度就立刻放弃。

我告诉你，这一定是错的。感情中我们是互相独立的，但要记住，我们还有动物本能，那就是征服与被征服。动物的本能就是征服与被征服。只要你把握住这一点，并坚持自己的观点和不妥协，无论对方如何争吵或闹情绪，都请坚持住。

这时，你收获的不会是对方的离去，而是你对对方的征服。用这样的思维去谈恋爱，世界上就没有你谈不好的人。这叫做“强者思维”。

我发现很多人在谈恋爱时，都采用一种“弱者思维”，把所有寄托和希望都放在别人身上，却从未向内审视过爱情的本质。首先，你要向内看，观察自己。其次，你要学会认真观察对方的真实需求是什么。

这里说的真实需求，不是指“我想吃什么菜”、“我喜欢什么东西”这类物质需求，而是他内在的精神需求。因为所有物质层面的东西，谁都可以给予，但精神需求是独特且不可替代的。

其实，谈恋爱的本质，是想获得一个心灵上的港湾。这个港湾能让他感到舒服，才是你们恋爱成败的关键。
