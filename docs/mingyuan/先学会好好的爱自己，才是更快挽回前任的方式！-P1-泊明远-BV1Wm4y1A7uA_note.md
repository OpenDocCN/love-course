# 先学会好好的爱自己，才是更快挽回前任的方式！-P1-泊明远-BV1Wm4y1A7uA_note

在本节课中，我们将要学习如何在经历情感挫折后，通过专注于自我关爱来修复内心创伤，并为未来的行动积蓄力量。核心在于理解，真正的修复始于自身。

如果没有人能够理解你努力了很久却未得到理想结果，我们不妨先停下来，学会更好地爱自己。我分享过许多关于心理成长和疗愈的内容，但我也理解你可能难以听进去。这没关系，我们都曾如此。

上一节我们提到了停下来审视自我的必要性，本节中我们来看看情感挫折带来的具体影响。

我在被分手时也精神恍惚，只想抓住救命稻草，渴望立刻得到结果。但这并非因为我们自身或其他原因，而是我们的**信念**、**希望**和**价值感**因被抛弃而遭受了巨大创伤。

我们不过是在强撑一口气，凭着不甘心去努力恢复关系。这种关系可能在我们内心早已遍体鳞伤。不幸的是，情感失败似乎总伴随着事业挫折与生活不如意，从而引发失望、愤怒、恐惧与迷茫。

这些负面情绪会时刻侵扰我们的理智与动力。这或许就是我们人生的至暗时刻。我知道你很倔强，也很痛苦，甚至不确定自己是否还爱着前任，只是希望一切能回到从前。但我仍要说：

如果没有人懂得爱你，我们就要先学会自己爱自己。这不是让你放弃挽回，也不是让你压抑难过，而是能否先为我们自己腾出一段时间，来做一回真正的自己。这段时间无需很久，也不会影响你的事业或挽回计划。

所以，无需害怕和内疚。利用这段时间让自己躺着，不去想挽回，不去想感情，只是躺着，让内心疲惫的自己感受到爱和力量。

接下来，我将分享一个简单易行、能帮助你恢复元气的小技巧。

以下是具体操作步骤：

*   **准备**：静静地躺在床上，闭上眼睛。
*   **构建世界**：想象你拥有一个完全属于自己的世界，在这里你可以主宰一切。
*   **回忆美好**：按照时间顺序，从幼年到现在，一件件回忆那些让你感到快乐和幸福的事情。
*   **收藏美好**：将这些美好的记忆全部装进这个想象的世界中。
*   **沉浸享受**：在其中静静地享受这些美好。
