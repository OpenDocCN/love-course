# 分手复合指南 第1课：识别“完全回避”状态 📵

在本节课中，我们将要学习一种特定的分手后状态——“完全回避”。当对方表现出这种状态时，恰恰是关系最容易修复的时机。我们将详细解析这种状态的表现，并理解其背后的含义。

---

分手后，如果对方主动切断了你们之间的所有联系，这种状态被称为“完全回避”。

具体表现包括：微信被拉黑、电话不接、短信不回，甚至避免与你见面，呈现出一种完全不理会的姿态。

如果你也遇到了这种情况，无需焦急，也不必放弃。

**拉黑与删除** `!=` **感情彻底破裂**。对你视而不见，也 `!=` 无法重新在一起。关键在于，你需要采用正确的方法去挽回。
