## 概述：两种关键心理状态

男性分手后的心理反应有其典型模式。掌握这些模式，能帮助你避免踩雷，并采取正确的应对方式。

### 第一种心理：渴望被挽回

大多数男人在提出分手时，内心其实渴望女方去挽回他。

然而，一旦你低头去挽回，就可能陷入被动。他会因此更有安全感，并可能将受伤的情绪放大，认为是他伤害了你。男性的直线思维更关注事物的对错，这最终可能导致他产生一种有恃无恐的心态。

其表现公式为：
**提出分手 + 被主动挽回 → 安全感增加 + 情绪发泄 → 冷漠态度**

从而，他可能对你爱答不理。

### 第二种心理：因不被挽回而否定关系

上一节我们介绍了男性渴望被挽回的心理，本节中我们来看看相反的情况。如果分手后你不去挽回他，他可能会更生气，情绪更激烈。

因为他会认为，他在你心里一点都不重要，得不到珍惜和重视。他会从内心否定自己的价值，甚至快速地从这段感情中抽离，认为你不值得他留恋。

这个时候，如果你采取彻底断联的方法，当你未来想重新复联时，可能会发现机会已经彻底消失。当然，也有部分男生会因为忍不住痛苦而回头找你。

但他的内心可能已经埋下了一颗随时会离开的种子。因为你不去挽回他，本身就被他视为一种不重视，这构成了二次伤害。所以他再次离开你，可能只是时间问题。
