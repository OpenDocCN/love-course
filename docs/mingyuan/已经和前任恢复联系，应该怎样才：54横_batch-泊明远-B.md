## 🧠 心态调整：摆正自身位置

上一节我们介绍了课程概述，本节中我们来看看沟通前至关重要的心态准备。

不要把自己放在伴侣的位置上去和前任聊天。许多人一上来就发送“早安晚安”或“在干嘛”这类消息。试想，如果你是不想复合的一方，收到这类信息会感到不适。

对方可能的回复是：“我们真的结束了”、“不要再这样发消息”、“这给我带来困扰”或“我们不可能了”。能收到这类回复还算好，更常见的情况是对方已读不回。

**核心心态公式**：`沟通姿态 = 朋友身份 ≠ 伴侣身份`

---

## 🗣️ 首次接触的两种正确方式

调整好心态后，我们来看看如何迈出第一步。以下是开展第一次聊天的两种推荐方式。

**第一种方式：通过社交动态互动**

如果你看到对方发布了一条积极向上的动态，可以点个赞或留下一句真诚的赞美评论。然后观察对方是否回复。

如果对方回复速度快或态度好，你可以耐心等待下一次互动机会。需要注意的是，你的评论不应带有深情意味或夹杂个人情感，以免引起前任的排斥心理。

**第二种方式：创造线下见面机会**

如果没有联系方式，最佳方式是寻找机会线下见面，例如朋友聚会、同学聚会等场合。

对于分手细节不严重的情况，对方可能看在朋友面子上与你聊几句。对于情节严重的情况，对方可能感到被骚扰。

此时建议不要主动搭讪，而是采取以退为进的策略。你需要传递出一种“你不爱我，但外面还有人欣赏我”的感觉，以此来试探对方态度。但要注意尺度，过度表现会让他觉得你并非真心，是一个见异思迁的人。

**核心操作代码**：
```python
if 有社交媒体联系:
    行动 = 点赞或真诚评论()
else:
    行动 = 寻找线下自然见面机会()
```

## 💡 话题选择与沟通节奏

掌握了破冰方法后，我们来探讨聊什么以及何时聊。

在复联阶段，不要没事就找对方闲聊。初期沟通必须基于具体的事情，而且最好是“非他不可”的事情。

随着两人状态缓和，再逐步调整聊天频率和内容。

**关于沟通时机**

许多人容易犯的错误是24小时不间断发消息，不管对方是否回复，想起一条就发一条，没有回应就开始写小作文。这些都是反向操作，不建议采用。

在这个阶段，你必须揣摩对方心理，对症下药。复联阶段切记不要给对方施加压力，也最好不要谈论感情。应该找一些轻松愉快的话题，例如聊聊最近发生的事情。

**核心原则**：`沟通目标 = 不求加分，但求不扣分`

这些都是在为后面的复合打好基础。
