上一段我们提到了分手后拉黑行为的存在，本节中我们来具体分析其背后的原因。

反复拉黑删除你的人，其行为具有明确的心理动因。分手后他采取的拉黑和删除行动，必然伴随着强烈的情绪。如果他反复执行拉黑与删除你的操作，这证明他的情绪因你而产生剧烈波动。这清晰地表明他内心深处依然深爱着你，而你曾做出某些让他感到伤心的事情。

因此，他利用拉黑和删除作为手段，来宣泄内心的不满。然而，他又无法真正放下这段关系，处于矛盾状态，从而一次又一次地将你重新加回。这种行为意味着他希望再次给你一个机会。这充分说明他爱你的心意是真诚的，其内心的矛盾与情绪的反复也是真实存在的。

以下是关于此行为核心要点的总结：

*   行为驱动：**情绪宣泄**。拉黑删除是表达不满的直接方式。
*   矛盾本质：**放不下与不舍**。反复加回体现了无法割舍的情感。
*   情感内核：**深爱与受伤**。行为波动根源在于依然深爱但曾被伤害。
