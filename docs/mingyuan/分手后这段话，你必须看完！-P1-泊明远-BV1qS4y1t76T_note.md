## 恋爱关系的普遍阶段与问题 🔄

上一节我们提到了分手后的即时反应，本节中我们来看看恋爱关系通常经历的阶段。

其实任何恋爱关系都相似，都会经历热恋期、矛盾期和平淡期。初始恋爱时，人们常认为“只要对方对我好，就能在一起”。相处久了，可能发现对方是直男，不懂浪漫。

**公式：恋爱阶段 ≈ 热恋期 + 矛盾期 + 平淡期**

## 不断更换伴侣的循环 🎡

以下是人们在多次恋爱中常见的一种循环模式。

当伴侣无法提供所需的情绪价值，你提出分手，进入下一段感情。在新感情中，你找到了颜值高、懂浪漫、能满足需求的人，但发现对方缺乏上进心，整天玩游戏。

不知道好好工作，你觉得在一起看不到未来，于是再次分手。第三次，你遇到懂浪漫也有上进心的人，但你觉得相处很麻烦。

感觉在一起很受委屈，最终又选择分手。兜兜转转一圈后，感觉最初的那个人最好，又想去复合。

## 问题的核心：爱的能力 🧠

很多人遇到问题时，会下意识认为“如果当初和另一个人在一起，肯定会幸福”。但真相是，你并不具备爱的能力。你和谁在一起都可能不幸福。

我们常把分手理由推到对方身上，认为换人就能幸福。但真的没有天生合适的两个人，都需要互相磨合才能融洽。

**核心概念：爱的能力 = 经营感情的能力 + 正确沟通的能力 + 让彼此更相爱的能力**

## 给被挽回者的建议 🤝

对于被挽回的一方，建议回头看看眼前这个爱你的人。只要他能意识到自身问题并勇于改正，不妨再给一次机会。

完全重新开始，可能浪费更多时间精力去遇到新人，也不一定更好。你们原有深厚的感情基础，不妨回头看看。如果连看一眼的机会都没有就放弃，非常可惜。
