# 情感识别课程 01：识别伴侣的彻底放弃迹象 🚨

在本节课中，我们将学习如何识别亲密关系中，一方已决定彻底放弃但未明确表达时，所表现出的特定行为模式。这些迹象往往隐蔽，容易被误解为关系中的普通冷淡期，但实质是情感抽离的开始。

上一节我们介绍了情感关系中的常见问题，本节中我们来看看一种特定的“放弃模式”的具体表现。

## 核心行为模式解析

当一方决定放弃感情后，其行为会进入一种 **“消极顺其自然”** 状态。其核心公式可概括为：

**表面维持 - 情感投入 = 关系空壳**

这意味着，对方仅维持最基本的互动形式，但抽离了所有情感价值。

以下是构成这种行为模式的具体迹象列表：

*   **回应但无热情**：你联系他，他会回复；你发起对话，他也会接话。但回应中缺乏热情、关心或延伸话题的意愿。
*   **回避情感交流**：他不再表达爱意，**甜言蜜语消失**。同时，他也**不会主动说出“不爱了”或“分手”** 这类明确结束关系的话语。
*   **耐心丧失与逃避**：当你表达抱怨或情感需求时，他表现出**没有耐心**。更极端的情况下，会**直接玩消失**，切断联系。
*   **关系态度消极**：他对你们的关系持**得过且过**的态度。其行为逻辑是：**你想继续，他就被动陪伴；你想分手，他也不会挽留**。

## 总结与过渡

本节课我们一起学习了亲密关系中“彻底放弃”的隐蔽迹象，其特点是**维持形式、抽离实质**。识别这些模式是应对的第一步。
