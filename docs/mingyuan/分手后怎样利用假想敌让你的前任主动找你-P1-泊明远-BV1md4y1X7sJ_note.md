以下是运用“假想敌”技巧时必须注意的核心原则和常见误区。

**核心原则：保持模糊性**
制造假想敌的意义在于它是模糊的，目的是让他猜不透但又很着急。记住，不是让你直接秀恩爱，或者公开宣布和别人的新恋情。如果你这样做，就是搬起石头砸自己的脚。你的前任会认为你已经开始新恋情、彻底放下了。或者，你刚制造完假想敌又回头找前任，只会让别人更讨厌你。

**核心公式：**
```
有效假想敌 = 存在暗示 + 状态模糊
```
因此，假想敌的核心就是让别人摸不着头脑，好像你有，又好像没有。这样的效果才是最佳的。
