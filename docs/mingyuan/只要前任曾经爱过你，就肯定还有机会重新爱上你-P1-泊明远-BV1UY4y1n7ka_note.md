# 只要前任曾经爱过你，就肯定还有机会重新爱上你-P1-泊明远-BV1UY4y1n7ka_note

在本节课中，我们将要学习如何正确理解分手时对方的绝情话语，并认识到只要曾经存在过真爱，挽回就拥有坚实的基础。核心在于消除负面印象，重新展示吸引力。

最近有很多女生询问，前任是否真的会重新爱上自己。请放心，大多数男人在情绪激动时，也会说出许多情绪化的话语。例如：“我后悔和你在一起”、“我从来没有爱过你”、“我们本来就不合适”。如果你相信了这些话，那么你就真的中招了。

可能在分手时，很多男人表现得很坚决。在有情绪的时候，他们甚至会说“从来都没有爱过你，以前对你的爱都是假的”。你需要明白，人在有情绪的时候，不会有好的言语。

当对方说这些话时，也只是图一时的爽快，并未意识到会伤害到你。不论男女，是人都会有情绪。你可以回想一下，自己是否也是如此：情绪上来时，各种折腾，或者说一些伤害对方的话，但那其实都不是你的本心。

无非是想要对方更多的在乎和关心而已。当然，对方现在看起来很绝情，也不是没有原因的。可能是生活中的矛盾，也可能是你的坏脾气。这些都会让对方很累，并掩盖住了她爱你的心。其实你们并不是不爱，只是方法用错了。

你身上也有曾经她喜欢的地方。只要解决了你们曾经的矛盾，并重新展示你曾经的魅力，她还会再次爱上你。你要记住，挽回最难的，不是对方已经不爱你了，而是人和人在相处过程中形成的那些**固定印象**。

在挽回的过程中，如果无法消除这些负面印象，即使对方还爱你，两个人都很难长远地走下去，对方也很难下定决心和你复合。在挽回中，任何时候都要讲究技巧，消除对方心目中的负面印象也不例外。

所以，不管她现在说什么，你只要相信你们曾在一起，那么你在她的心里就一定是有位置的。谁的时间都是有限的，在好的年华里她愿意陪你，就说明你对她本身很重要。**只要她曾经爱过你，就一定有机会再次爱上你**。

也许很多时候，你自己都不知道身上有那么多的优点在吸引着她。即使现在你们分开了，但吸引她的地方也会一直存在。就好像你喜欢吃草莓一样，即便很久不吃，重新再看到草莓时，你也想着能尝一口。

她也是这样。可能现在你们之间会有一点点不愉快，但是只要你消除了前任对你的负面印象，对方就很有可能重新慢慢地接受你。也会因为你的改变，重新唤起对方曾经对你的感情，最终重新爱上你，复合成功。

本节课中我们一起学习了：分手时的绝情话多是情绪产物而非本心；挽回的关键在于消除负面印象和解决固有矛盾；只要曾经存在真实的吸引力，通过正确的方法展示改变，完全有可能让对方再次爱上你并成功复合。
