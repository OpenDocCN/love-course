# 分手后想让前任尽快和你和好的黄金七步法 - 课程P1 🫂

在本节课中，我们将要学习一套系统性的方法，旨在帮助你在分手后，理性地处理关系，并引导前任主动回头。这套方法包含七个必须严格按顺序执行的步骤。

如果说你想要前任回头，你只需要做好接下来所说的七步。如果你爱的人绝情到要跟你分手，并且看似没有回头的余地，那么你千万不要频繁地去给他发消息。你只需要做好接下来所说的这七步。

那么他就会主动回头。七个步骤缺一不可，必须严格地按照顺序来执行。建议大家点赞收藏，反复观看，并且拿出一张纸来，按照我所说的顺序写下来，总结自己遇到的问题和方法。遇到问题时，不要慌，不要去难过。

有情绪是没有用的，只有理性的分析，才能够找到解决方案。今天我只讲核心，没有其他内容。

## 第一步：认可分手需求，以朋友相待 🤝

上一节我们介绍了本课程的核心目标，本节中我们来看看具体的第一步。你需要认可对方分手的需求，并以朋友的身份存在。这意味着接受分手的事实，不争辩、不纠缠，为后续互动创造一个平和的基础。

## 第二步：保持适度联系 📞

在认可了对方的需求后，接下来我们需要管理联系频率。保持联系，但不能频繁地去给对方发消息。过度的联系会带来压力，适度的距离反而能保留沟通的窗口。

## 第三步：复盘关系问题 📝

在建立了基本的联系规则后，我们需要向内审视。找到你们相处时的各个问题。以下是复盘的具体方法：
*   复盘之前相处的细节。
*   把这些问题写下来。
*   逐个地去分析。
*   找到自己的解决方式。

## 第四步：展示自我改变 ✨

分析了问题并找到解决方案后，下一步就是将计划付诸行动。改变自己，并且让他看到你改变的一面。真正的改变不是口头承诺，而是通过行动让对方感知到你的不同。

## 第五步：强化核心吸引力 💫

完成了基础的改变后，我们可以进一步放大你的优势。加强自己对对方的吸引力。你之前吸引他的地方是什么，那么你就把这个地方无限去扩大。这能重新唤起他对你的美好印象。

## 第六步：引导对方表达 🗣️

当你的改变和吸引力开始产生效果时，互动方式也需要调整。引导对方表达自己。即使表面上同意了分手，你也可以找他去聊天，让他多去表达。而你呢，一定要少说话，多倾听。

## 第七步：重建情感连接 🌉

最后，当线上互动已经建立良好氛围时，可以尝试推进关系。建立连接。如果你们已经聊得很不错了，也可以去约见面。以下是见面时的注意事项：
*   见面时不要去提过去。
*   要像是朋友一样去相处。
*   找到机会，好好地解释自己过去的种种不好。

这七步一定要按照严格要求去执行，一步一步地来。中间切记不能着急，不然的话，对方一旦产生排斥心理，那么你就前功尽弃。

只要把握好每一个细节，你就会发现，整个过程会越来越轻松。
