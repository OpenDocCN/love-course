# 当前任对你有失去感时，才会想跟

在本节课中，我们将要学习一个在感情挽回中至关重要的概念——“失去感”。我们将探讨它的本质、重要性以及如何正确运用它，而不是错误地制造纠缠。

## 什么是“失去感”？🤔

上一节我们提到了“失去感”这个概念，本节中我们来看看它的具体定义和表现。

很多人都在问，怎么样才能让前任对你有失去感。你首先要问问自己，分手后你有没有持续地做出让前任认为你还想跟他复合的举动。如果有的话，失去感就永远不可能出现，因为你的前任知道你就是想要跟他复合，那么他就可以肆无忌惮地搞你心态。

失去感的主要表现在于博弈。很多时候，就是看谁比谁更不害怕失去对方，或者说敢于承担失去的风险。

## “失去感”背后的心理学原理 🧠

理解了它的表现后，我们需要深入其本质，这源于一种普遍的心理现象。

它的本质其实就是**损失厌恶心理**。它反映了人们的风险变化并不是一致的。许多人宁愿选择有百分百的机会去拿三千块钱，都不会选择80%的机会去获得四千块钱，那么这个就是害怕损失的心理表现。

我再举一个例子。你在和对方在一起的时候，你觉得他已经是你的人的时候，你自然不会去珍惜这些，那一些细节你会觉得无所谓，或者说根本就想不到。但是当对方要和你分开的时候，或者说分开之后，你会想得很多，那之前的对方是有多好，自己却有很多地方没有做到位。

所以人们只有失去的时候才会懂得珍惜，在拥有的时候，你自然不会察觉的太多。

## 如何正确营造“失去感”？🛠️

了解了原理，那么对于想挽回的一方，具体该如何操作呢？这其实对于挽回者来说，就是这样。但是对于被挽回者来说，也是同样的。当你们的问题得到了一定的调整，或者说你的价值也得到提升之后，完成这些之后，你就可以适当地营造一些失去感，让对方想到你的好，并慢慢地对你产生情绪和心理波动。那你这个收获的一些事情，才会双倍地加分。

所以现在，你们明白这个事情的重要性了吧。

## 关于风险的常见顾虑与应对 💪

然而，迈出这一步需要勇气，很多人会心存恐惧。但是有的人他就是害怕，万一真的失去了怎么办。那么我在这里说上一句，世界上永远都是收益和风险作为正比。

什么都不做，是最安稳的吗？但凡是你想做点什么，你想得到点什么，那么你总该第一步，先去做些什么。
