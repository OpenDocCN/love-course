## 概述：保留联系方式的积极信号

当你和前任分手后，如果对方没有删除拉黑你的联系方式，并且两人之间依旧可以进行正常联系，那么在挽回的起点上，你已经占据了一定的优势。

对方选择保留联系方式，说明他在产生负面情绪并决定分手的同时，内心其实还是放不下你。

当然，即便如此，你也不能掉以轻心。前任心里放不下你却依旧选择分手，这证明你们两人之间出现了难以解决的矛盾和问题。

## 沟通情况分析与应对策略

想要找到矛盾点，沟通是关键。分手后的沟通情况可以分为以下几种，以下是具体的分析和应对方法。

### 情况一：敷衍式回应

如果聊天时你侃侃而谈，他却只是很敷衍地回复几句，那么这种情况需要注意。

**核心应对公式：**
```
if (回复内容 == “嗯”、“啊” || 多次不回复) {
    及时结束话题；
    等待约一周后再次试探；
}
```

如果他连续多次不回复，或者回复总是“嗯”、“啊”这类词语，那么你需要适可而止。在他情绪不高时，及时结束话题。

像这样的情况，说明他还处于刚分手后对你比较排斥、心怀戒备的阶段。这时应该简单说两句就结束聊天，等待大约一周后，再试探前任的情况。

### 情况二：积极搭话式回应

如果他也在和你积极搭话，那么就证明他内心在某个时刻想要跟你复合，正处于考察你表现的阶段。

此刻需要你抓住机会，找到正确的突破点，去突破他的心理防线，从而实现成功挽回。

**核心行动准则：** 必须找到正确的突破点。如果这一步没有走好，所有的努力都可能付诸东流。
