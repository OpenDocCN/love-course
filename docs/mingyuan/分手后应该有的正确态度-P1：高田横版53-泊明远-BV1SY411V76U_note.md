# 分手后应该有的正确态度 - P1：高田横版 53 🧠💔

在本节课中，我们将要学习如何客观地评估前任在分手后的行为，并据此判断其人品。我们将分析几种典型的分手后态度，帮助你认清不同行为的本质，从而更好地保护自己并向前看。

很多人分手后对前任的认知，都是从热恋期的表现去评判的。但是这个不准。我建议你以后只看他分手后的人品。

上一节我们提到了评估标准应从分手后开始，本节中我们来看看什么样的行为可以被认为是得体的。

能够明明白白地跟你提出分手，不隐瞒不逃避，分完就彻底消失的，一定是中国好前任。

接下来，我们将对比分析其他几种常见的分手后行为模式。

以下是几种需要警惕的分手后态度：

*   **敢爱敢恨型**：那种分手后撕破脸皮放狠话，见一面恨不得就把对方一脚踢飞的，都是敢爱敢恨的热血青年。
*   **消耗纠缠型**：但是有一种前任特别恶心人，明明不爱你还吊着你，跟你玩冷暴力，装受害者去博取诸位人士的同情。
*   **恶劣伤害型**：最后不奉衔接新欢还烂你的人，那么这就是明明白白的人渣。遇上这样的真的是你倒了大霉。
