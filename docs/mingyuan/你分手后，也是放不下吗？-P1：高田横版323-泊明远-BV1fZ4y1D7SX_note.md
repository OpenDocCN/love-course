# 分手后情绪管理教程 - 第1课：如何停止内耗 🧠

在本节课中，我们将学习如何识别和处理分手后“放不下”的典型心理状态，并引导你做出清晰、果断的决策，以停止无意义的内耗，为生活翻开新的一页。

## 识别“放不下”的核心症状 🤔

上一节我们概述了课程目标，本节中我们来看看“放不下”的具体表现。如果你在分手后长期感到痛苦，首先需要判断自己是否陷入以下循环状态。

以下是“放不下”的典型心理特征列表：
*   内心非常纠结，一方面想挽留对方。
*   另一方面又想就此放弃，总是举棋不定、犹豫不决。
*   即使偶尔下定决心、想通了，过几天又会陷入想不通的状态，心里堵得慌。
*   不断权衡对方是否值得，又害怕自己未来会后悔。
*   或者是因为不甘心自己已经付出的“沉没成本”而反复挣扎。

**核心概念**：沉没成本是指已经发生且不可收回的支出，如时间、金钱、情感。理性决策应忽略它，公式表示为：**理性决策 ≠ f(沉没成本)**。

## 停止内耗：为自己设定决策期限 ⏳

如果你持续处于上述状态，你就是在浪费自己的生命和时间。朋友，你需要果断一点。

以下是停止内耗的关键步骤：
*   **给自己设定一个明确的考虑期限**，到期必须做出选择。
*   **前提是**，你必须能承受这个选择可能带来的最坏结果。
*   人生若在不断的选择犹豫和反复中度过，就是在浪费时间和青春。

## 做出选择并坚定执行 💪

选择一旦做出，就要对自己“狠”一点，坚决执行。

以下是两种路径的明确执行方案：
*   **路径一：毅然决然地离开**
    *   彻底断绝所有联系方式。
    *   断绝一切可能的交集。
    *   清除所有能看见、能听见的与对方有关的蛛丝马迹。
    *   忍不住时，主动设置一些方法（如转移注意力）来帮助自己忘记。
*   **路径二：坚定不移地挽回复合**
    *   一旦确立这个目标，就要拥有坚定的心态。
    *   拥有这种坚定心态，事情就成功了一半。
