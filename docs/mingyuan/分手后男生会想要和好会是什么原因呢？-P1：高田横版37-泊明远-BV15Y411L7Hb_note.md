## 分手后男生想复合的核心原因

上一节我们介绍了课程的主题，本节中我们来看看男生想复合的根本驱动力。

所有的男生想要复合的原因，都是因为他们找不到更好的了，而不是因为有多么爱你。

你可以思考一下这个逻辑：如果一个男生可以轻易找到一个很爱他、很漂亮的女生，那么他就不会选择回头。

## 一种特殊情况的补充分析

在理解了普遍原因之后，我们还需要关注一种特殊的情况。

这种情况比较特殊：男生明明找不到更好的，但是遇到一些女生去苦苦纠缠，从而给了他充足的自信。

其心理公式可以概括为：
**“虽然自身条件平凡 + 持续获得外部肯定 → 产生能找到更好伴侣的错觉”**

结果就是，尽管他很平凡，可是他一直相信自己可以遇到更好的。

## 课程总结与建议

本节课中我们一起学习了男生在分手后想复合的两个核心心理动机：一是**未能找到更优替代**，二是在**持续获得纠缠肯定**后产生的**错误自信**。
