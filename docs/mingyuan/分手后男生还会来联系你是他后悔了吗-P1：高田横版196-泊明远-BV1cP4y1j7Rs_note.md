上一段我们提到了男生联系可能并非出于后悔，本节中我们来看看他联系你的真实心理状态。

分手后男生还经常联系你，是他后悔了吗？别高兴得太早。舍不得不代表他就想回头。人都是有感情的，就算是他想要分开，失去一段关系，他也会感觉到不舒服。他还会跟你联系，但是不提感情，这个时候你就要当心了。

他其实是在一步一步地适应分手后的生活，一步步地从之前的感情当中抽离出来。只要你去回应，就满足了他的需求。当他完全适应了之后，就不会再回头了。所以你要在他还有需求的时候，完全抽离。

1.  **识别需求**：判断他的联系是出于情感依赖（**需求**）还是真心复合意愿。
2.  **停止回应**：在他还有需求时，选择**完全抽离**，不给予情感反馈。
3.  **打破适应周期**：你的抽离会打破他“联系-获得满足”的**适应过程**，使其重新审视关系。
