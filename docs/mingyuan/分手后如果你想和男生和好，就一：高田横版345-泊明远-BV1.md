# 分手后和好教程：如何不让男生发现你的意图 - P1 🧠

在本节课中，我们将要学习分手后若想与男生和好，为何以及如何避免让他察觉你的意图。核心在于理解男性心理的某种共性，并采取相应的策略。

## 核心原则：隐藏意图 🤫

上一节我们介绍了课程主题，本节中我们来看看核心原则。如果你想和男生和好，那就首先要让他觉得，你不想和好。也许你现在还不理解，但是只要你理解了，就能明白和好的真谛。

## 心理基础：择偶标准与男性心理 🧩

首先你们曾经在一起过，说明你在他内心，其实你符合他的择偶标准的。可是男人很奇怪。

以下是关于男性心理的一个关键矛盾点：
*   就算你符合他的择偶标准，你如果过于粘着他，他反而不想跟你继续下去了。
*   但是如果你敢于放下他，他反而会有很强的失去感，开始缠着你。

这就是所有男生的共性。这个共性可以用一个简单的心理公式来描述：**需求感与吸引力的反比关系**。即，当你表现出过高的需求感（粘着他），你的吸引力在他眼中会下降；反之，当你表现出低需求感（放下他），会激发他的“失去感”，从而提升你的吸引力。
