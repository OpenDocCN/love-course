## ⏰ 第一点：何时发送消息

上一节我们介绍了课程概述，本节中我们来看看发送消息的时机。

如果你刚分手，最好不要马上发消息。因为你会发现很多事情都不能讲。最少要间隔一两天，给对方一点时间去消化，也让自己平复心情。

如果你已经度过了断联的阶段，那么只要你的情绪稳定下来，随时都可以联系。这时一定要做好一个心理准备：对方的态度一定没有之前热情，包括你们的聊天氛围也一定会比较陌生。

重点不是要一直和对方聊，我们只是试探他还愿不愿意回复你。

---

## 🎬 第二点：如何开场

了解了发送时机后，我们来看看最关键的第一步：如何开场。分手后的第一次聊天，往往决定了这段关系的走向。

如果你带着情绪向对方卖惨，或者嘲讽质问对方，那肯定只会让他更加反感，不想搭理你。例如：“你不是说永远爱我吗？要跟我结婚吗？你知道我有多难受？” 你越是这样说，对方就越觉得你无法沟通。

以下是三种正确的开场方式：

1.  **询问近况式**：例如问他“你最近怎么样？”
2.  **自我状态描述式**：说你自己这几天的感觉还蛮轻松的，可能之前的关系确确实实有一点太累。或者提及以前一些甜蜜的事情，尽量以一种轻松的语气去说。
3.  **求助式**：用一些问题去向对方求助，挑一些他擅长的地方。正常来说，男人是不会拒绝的，除非你已经纠缠、骚扰过，让他很不耐烦。

---

## 📝 第三点：聊天中的核心原则

成功的开场只是第一步，本节中我们来看看聊天过程中必须遵守的核心原则。

首先，一定要搞清楚自己的位置。你们已经分手了，不管你是什么样的想法，事实就是这样。你死缠烂打，或者说“想你”、“爱你”，这些真的不是你应该做的事情。你最多就只是一个前任，或者普通朋友。

因此，分手后聊天，**一个字也不能聊情感话题**。可以去聊工作、生活、朋友，但是不聊感情。

这里记住一个核心原则：**“三三聊天法”**。

```python
# “三三聊天法”伪代码逻辑
if 距离上次联系 >= 3天:
    发起聊天
    if 对方反应积极:
        可适当多聊几句
    elif 对方态度一般:
        只聊三句左右就结束
else:
    保持等待
```

具体解释如下：
*   **频率**：每隔三天去联系一下。
*   **内容**：每次聊天一定是说事情，不要闲扯。说的事情，一定要是对方感兴趣的，这样才不会过于冷场。
*   **节奏**：如果对方的反应比较积极，就可以多聊几句。如果对方的态度一般，那你就聊三句左右结束。说太多容易变得无聊，让对方反感；而完全不说，又会让本来简单的关系彻底断掉。

## ✨ 第四点：重建吸引力

前面的所有原则都有一个共同目的：不让对方反感，不要让对方跟你的接触感到压力。这个前提做好之后，你才能够去展示自己，重新建设对他的吸引力，让他重新在心里考虑要不要和你更近一步。

想一想，最开始的时候，他为什么要和你在一起呢？等到你们可以正常聊天了，对方对你的回应也比较及时、比较热情的时候，你就可以说自己的事情，展示自己身上能够让他感兴趣的点。

我们聊天的目的是为了让他产生一个认知：并不是你这个人不好，只是你们之前的关系有些矛盾，处理的方式不对。

当线上聊天没有问题了，我们才能够计划线下的见面，让他直观地感受到你的成长和变化。到那时，让他主动考虑复合也不是什么难事。

## 📚 课程总结

本节课中，我们一起学习了分手后复联的四个关键步骤：
1.  **选择合适时机**：刚分手需冷静，情绪稳定后可试探联系。
2.  **采用正确开场**：避免情绪化，用轻松的问好、状态分享或求助打开话题。
3.  **遵守聊天原则**：摆正位置，禁谈感情，运用 **“三三聊天法”** 控制频率与内容。
4.  **逐步重建吸引力**：在舒适感建立后，通过展示积极变化，重新吸引对方。

记住，复联是一个循序渐进的过程，核心在于创造轻松、无压力的沟通氛围，让对方重新发现你的价值。
