# 情感修复课程 P1：如何引导前任主动联系你？ 🤔

在本节课中，我们将学习如何运用心理学原理，通过调整自身行为和心态，引导前任主动与你联系，为可能的复合创造机会。核心在于理解并利用人的“逆反心理”，避免纠缠，专注于自我提升。

## 理解逆反心理与核心策略

上一节我们提到了课程的核心目标。本节中，我们来看看其背后的心理机制和基本策略。

人普遍存在逆反心理。当前任决定分手时，你越是想让他做什么，只要他感觉到被逼迫，他就一定会反着来。因此，你需要反其道而行之，打破分手后纠缠不休的常态。

核心策略可以概括为：**停止施压 + 自我提升 + 制造神秘感**。

## 具体行动步骤

理解了核心策略后，以下是具体的行动步骤，请务必按顺序执行。

**第一步：停止一切纠缠行为**

如果之前你一直在纠缠对方，现在必须立刻停止。不要做任何回应，保持冷静与沉默。此举是为了切断对方对你的负面预期和压力感。

**第二步：专注于自我提升与展示**

无论是男性还是女性，都对美好的事物抱有向往。当你把重心完全放在自己身上时，你自身必然会发生积极改变。

以下是你可以着手进行的方面：
*   **发展兴趣**：去做任何你想做、能让自己变得更好的事情。
*   **展示改变**：通过社交状态（如朋友圈）适度展示你的新生活和新面貌。

这些行动的目的是制造强烈的反差，彻底扭转你之前一味纠缠的负面形象。

## 应对联系与保持吸引力

当你完成自我改变并开始展示后，可能会迎来前任的主动联系。本节中我们来看看如何正确应对，以维持吸引力。

前任主动联系，意图无非两种：**试探你** 或 **寻求复合**。许多人曾在此处犯错。

例如，前任只是简单联系，你却因过于激动而回复一大串信息，这可能导致对方再次冷淡或消失。如此一来，你在分手后所做的长期铺垫可能全部白费。

你的真实想法和感受，不需要让他一清二楚。你越是让人琢磨不透，越有吸引力。当他发现你过得一天比一天好，却“看得到，吃不到”时，才能最大程度激起他的好奇心和征服欲，促使他主动靠近你。
