# 分手后想要对方和你和好，千万不：40横_batch-泊明远-B

在本节课中，我们将要学习分手后，如果你希望与对方和好，必须避免的一些关键错误行为。这些行为往往会将对方推得更远，阻碍关系的修复。

## 避免乞求与强行解释

上一节我们介绍了课程目标，本节中我们来看看第一个需要避免的行为：乞求对方与强行解释。

千万不要去祈求对方。不要想着当下就把事情跟对方解释清楚。如果你们之前能够解释清楚、能够有效沟通，那么你们的关系很可能不会走到分手这一步。分手本身说明你们之间的沟通已经存在问题。

因此，才会出现你所说的“对方听不进去”的情况。此时，你需要先梳理好自己的情绪，不要一根筋地、一股脑地非要逼对方听你说话。我知道你可能已经意识到自己的问题所在，但关键在于你能否解决它。

**核心问题在于**：你能否控制住自己的情绪？你能否保证未来不再出现同样的问题？如果你这些都还没做到，那么你对对方说任何话都是无效的。

## 认识“狼来了”效应与压力累积

理解了强行沟通的弊端后，我们进一步探讨言行不一的后果和矛盾的本质。

这些问题说多了，就像“狼来了”的故事一样。到了后期，即使你说的是真话，对方也可能不再相信。而且，你们的矛盾和压力并非一日形成，也不要指望一天就能解决。这需要一个循序渐进的过程。

**感情中的基本公式是**：`不劳而获` 和 `一劳永逸` 是不可能的。

## 切忌骚扰与施加压力

认识到问题需要时间解决后，我们来看看另一个常见的错误做法：通过骚扰来施加压力。

很多人在这个阶段为了复合，会不断地去骚扰对方，不断地向对方施加压力。例如，一直追问对方是否还喜欢你、是否还爱你、是否还能复合等等，逼问对方做出选择。结果往往是越问对方越绝情。

你不要去指责对方绝情或狠心，指责对方已经不爱你了。这些反应很大程度上是被你持续的逼迫所激发出来的。

---

**本节课总结**：本节课我们一起学习了分手后挽回时需要避免的三个主要行为：1）避免乞求与强行解释；2）认识到问题无法瞬间解决，避免“狼来了”效应；3）切忌通过骚扰和逼问来施加压力。修复感情需要先管理好自身情绪，并采取循序渐进的方法。
