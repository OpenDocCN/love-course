# 情感修复课 01：理解“反复拉黑删除”背后的心理 🧠

在本节课中，我们将要学习如何解读分手后，对方反复将你拉黑又加回这一行为背后的深层心理与情感逻辑。理解这一点，是修复关系的第一步。

## 核心解读：拉黑删除是情绪的信号 📶

上一节我们介绍了课程主题，本节中我们来看看行为背后的核心原因。

你一定要记住，分手后他的拉黑和删除，一定都是带有情绪的。这个行为可以看作一个**情绪公式**：**拉黑/删除 = 强烈负面情绪 + 关注**。他如果反复拉黑删除你，那说明他的情绪，在为你波动很明显。

## 矛盾行为揭示真实情感 💔

理解了拉黑是情绪信号后，我们进一步分析其矛盾性。

他是深爱你的，你做了一些伤他心的事情，所以他用拉黑删除，去发泄自己的不满。但是他又放不下你，很矛盾的一次又一次，把你加回来，想再次给你一个机会。这就说明他爱你的心，一定是真的。

以下是这种矛盾心理循环的典型表现：

*   **行为A：拉黑/删除**
    *   **心理状态**：感到受伤、愤怒、失望，需要切断联系来保护自己或表达抗议。
*   **行为B：重新加回**
    *   **心理状态**：不舍、怀念、抱有希望，无法彻底割舍情感联结。

## 总结与行动指引 🛤️

本节课中我们一起学习了“反复拉黑删除”行为的心理密码。我们了解到，这并非单纯的绝情，而是对方内心深爱、受伤且极度矛盾的体现。情绪的巨大波动恰恰证明了你在其心中重要的位置。
