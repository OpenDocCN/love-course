## 概述

任何一段感情的结束，都不是无缘无故的。无论是谁在决定放弃的时候，都一定是在寒风当中站了很久，一定是积攒了无数的失望和委屈，才让自己彻底放弃了。这可以用一个简单的公式来描述关系的消耗过程：

**失望累积 = 初始容忍度 - (挑战底线行为 × 次数)**

## 男性的决策过程

上一节我们概述了关系结束的本质，本节中我们来看看男性的具体决策过程。

在感情当中，大多数的男人都不会轻易地说分手，因为男人都比较理性。他会一次次地给你机会，可是你却错误地认为他永远都不会离开你，然后继续肆无忌惮地挑战他的底线。

要知道，冰冻三尺非一日之寒，当失望和委屈达到了顶点，再爱也会停下来。

## 导致离开的关键行为

理解了决策过程后，我们来看看哪些具体行为会加速这个“失望累积”的过程。

以下是消耗感情好感度的主要行为列表：

*   忽略他的感受
*   消耗他对你的好感
*   挑战他的底线和尊严

当你连尊严都不给男人留的时候，他再爱你也会选择离开。这可以用一个条件判断来描述其决策逻辑：

```python
if 尊严被严重挑战 == True:
    离开 = True  # 即使仍有爱意
else:
    离开 = False
```
