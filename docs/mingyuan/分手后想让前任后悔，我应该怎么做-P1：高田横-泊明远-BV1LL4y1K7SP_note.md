# 分手后想让前任后悔，我应该怎么做 - P1：心态调整篇 💔

在本节课中，我们将要学习分手后如何调整心态，这是让前任后悔并可能挽回感情的第一步。核心在于停止纠缠，转而进行深刻的自我反思与价值重建。

上一节我们介绍了课程主题，本节中我们来看看分手后常见的错误心态及其危害。

很多人分手后会纠结一个问题：为何自己付出那么多爱，最终却被分手。心中不甘，于是去乞求对方回来。这通常是错误的第一步。

如果说你真的想要去挽回一段感情，那么你要做的并不是当下马上急着去联系对方。而是先把你的心态调整好。

以下是调整心态前需要进行的步骤：
*   反思并复盘，思考两人关系结束的真正原因。
*   回想对方是如何变得爱搭不理、不理不睬甚至施加冷暴力的。

你何必用你的热烈去贴人家的冷屁股呢。很多人在此时害怕失去对方，但对方并不害怕失去你。

这就像去银行借钱。你知道什么样的人在银行能借到钱吗？就是有钱、有资产的人。

**公式：可借贷额度 ∝ 自身资产价值**

如果你是个穷人，没有工作也没有资产，银行不会把钱借给你。当然不会。

你只有真真正正地拥有了那种不害怕对方失去的勇气，这时你才能真正的去挽回对方。

本节课中我们一起学习了分手后心态调整的核心：停止纠缠，通过反思厘清问题，并像积累资产一样重建自我价值与不惧失去的底气。只有当你不再害怕失去时，你才掌握了关系的主动权。
