# 分手后想让前任主动找你复合，你

在本节课中，我们将探讨一个在情感关系中常见的现象：为何“无视”有时比“重视”更能产生吸引力。我们将分析其背后的心理原理，并学习如何通过调整自身姿态来重建吸引力。

## 无视背后的吸引力原理

大家是否发现一个规律：当你不把对方当回事时，对方似乎挺喜欢你；一旦你开始认真，对方反而开始冷落你。

为什么会这样？背后的原理是什么？因为“无视”会让人觉得你背后有力量。当一个人表现出不被他人轻易影响的状态时，容易让对方产生一种“心生敬畏”的情绪。

相反，过分地重视对方，可能会让对方觉得“你也不过如此”，从而降低了你在他心中的价值和吸引力。

## 现实生活中的“无视”案例

我们可以从生活中观察到“无视”产生权威与魅力的例子。

例如，在酒桌上显得很厉害的人，他往往会无视你。对方越无视你，你就越觉得对方厉害、有霸气。你们公司的老板肯定也经常无视你，而他越无视你，你可能会越觉得“这个老板很有魅力”。

有人可能会反驳：像马云那样的大老板，很多厉害的人看起来很亲民。这是因为马云和你没有直接的利益关系。如果他是你的顶头上司，你看他会不会无视你。**无视是一种姿态，是一种真正高价值的行为**。

## 姿态差异：好男人与“渣男”的对比

为什么很多男生条件不错，却不受女生待见？那是因为他太过于重视对方了，姿态放得太低。

而一些所谓的“渣男”，兜里可能掏不出100块钱，但女人缘却不断。这让很多老实人感到气愤。为什么会这样？

因为“渣男”可能缺钱、缺德，但他们一定不缺的是“老子不缺女人”的心态。他们持有一种“爱谈不谈，打心底无视女人”的态度，这种态度反而吸引了女人。

所以，单从架构力、吸引力的角度来说，“渣男”其实做得挺不错。他们很懂得利用人性，去调动对方的情绪。单从这个角度来讲，就值得我们去学习和思考其背后的行为逻辑。
