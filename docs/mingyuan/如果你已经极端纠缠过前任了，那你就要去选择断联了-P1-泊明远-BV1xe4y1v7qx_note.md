# 情感修复课程 P1：断联与自我提升 🧩

在本节课中，我们将要学习分手后一个至关重要的策略：断联。许多人在经历分手后会陷入情绪漩涡，做出各种纠缠行为，但这往往适得其反。我们将深入分析纠缠的负面影响，阐明断联的必要性，并探讨当断联效果不佳时，应如何通过提升自我价值来破局。

## 理解纠缠的负面影响

上一节我们介绍了课程主题，本节中我们来看看分手后常见的行为模式及其后果。

许多人在分手后，会进行信息轰炸、电话轰炸，动用亲朋好友劝说，甚至上门围追堵截，进行各种极端的跪求和纠缠。我相信当你能够看到这条视频的时候，你或多或少都做过以上的某件事情。

我非常能够理解你们疯狂纠缠背后的那种不安的情绪。但是现在还不是你适庞的时候。现在你做的一切纠缠，无疑是扩大了你的缺点。

## 断联：当前阶段的核心策略

理解了纠缠的危害后，我们来看看当前最应该采取的行动。

你当下最应该做的事情，是断开一切联系。不管你之前做了多少事情，那么从现在开始，你给我断联。

不用担心断联会断掉，对方会忘记你。因为此时此刻，对方心目当中能够想起你的，只有这些负面情绪。你以为你所做的一切都是在感动对方，但是，在他的视角里面，你就像是一块甩不掉的牛皮糖。

你只有停止这一切的纠缠，才能让对方感受到真正的失去。而那种负面壁垒，才不会继续地磨成灰底。

## 当断联失效时的深层原因分析

上一节我们介绍了断联的基本策略，本节中我们来看看断联可能失效的情况及其原因。

当然，有一些情况，断联了三个月、四个月，他还是没有回头。那这个时候你又要问我了，问题出现在哪里。

老实告诉你，很有可能是，在你之前疯狂纠缠的时候，你触碰到了对方的底线，而且是不可逆的理。那这个时候你怎么断联，其实都没有多大效果。

## 破解之道：提升自我价值

分析了问题根源后，我们来看看根本的解决方法。以下是核心的行动指南。

你记住，不可逆的理念就是，你触碰到了对方的底线。而底线这个东西，会依据对方的价值，而随时活动。

我打个比方，你某个底线是非常讨厌这个人在公共场合抽烟。那如果你的相亲对象，在你约会的时候，当着你的面抽了烟，你会觉得无比地恶心，特别讨厌他想转身就走，或者说，你已经否定了这个人。

但如果对方是吴彦祖，或者是彭于晏呢？你会不会为他降低底线？甚至你还会觉得他很男人呢。

那发现了没有，对方不可逆的点是，取决于你本身的价值。你的价值高，他能够为你降低底线。那如果你的价值很低，他的底线也会降低。

那么在断联的时候，你最重要的事情就是，爆发式地疯狂提升自我价值。这件事情再难，你也得去做。一天到晚的幻想去担心它，倒不如切切实实地改变自己。
