# 情感分析课 01：分手如何检验真爱？💔

在本节课中，我们将探讨一个情感话题：分手后的心理与行为反应如何成为检验真爱的重要指标。通过分析几种典型的表现，我们可以更清晰地识别一段感情在自己生命中的分量。

上一节我们介绍了课程的主题，本节中我们来看看分手后可能出现的几种具体心理状态和行为模式。

以下是分手后常见的四种心理与行为表现：

1.  **睡眠规律紊乱**
    你每天晚上都很难入睡，到了早晨又很不愿意醒来。因为只要一醒来，你就必须面对已经分手的事实。

2.  **情绪难以自控**
    你总是会假装，让自己看上去很洒脱。但一个人的时候，或者说和自己好朋友在一起的时候，却完全掩饰不住。很多时候，眼泪和那种心痛的感觉，就止不住地向外流淌。

3.  **持续关注对方**
    你会经常翻看对方的动态。不管是不是被拉黑删除，你都会习惯性地去点开他的头像。然后慢慢地你就会发现，他不但动态没有更新，也渐渐地从“三天可见”，变成了一条直线。

4.  **内心空虚与回忆侵袭**
    你的心里总是空落落的，时常会有一种被掏空的感觉。你也经常会回忆起一起曾经吃过的餐厅，一起压过的马路，一起逛过的公园。但回过神来却发现，早已经是物是人非。

如果以上描述中，你满足至少三条，那么这强烈暗示对方曾是你生命中不可或缺的人，这段感情对你而言意义非凡。
