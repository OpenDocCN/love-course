# 情感修复课程 01：分手后是否需要删除前任联系方式？📱💔

在本节课中，我们将探讨分手后一个常见的困扰：是否有必要删除前任的联系方式。我们将分析保留联系方式可能带来的心理影响，并提供清晰的行动思路。

## 断联后保留联系方式的潜在影响

上一节我们提到了分手后的联系方式问题，本节中我们来看看保留联系方式可能带来的具体影响。

说实话，有些人分手后留下联系方式，不如将其删除。只要联系方式还在，你就能看到前任的朋友圈。于是，你总会忍不住想去窥探对方，渴望掌握前任的所有动态，以此满足自己的控制欲和窥探欲。

一旦对方有新的动态，无论这些内容是否与你有关，都会加剧你的焦虑。例如，昨天他发了一条动态，似乎是和某人出去玩。聚会照片里出现了一个女生，难道他们正在接触？他昨天在抖音上同时关注了一个新妹子。

他们是什么关系？在一起了吗？你工作做不好，吃不下睡不着，每天活得像个侦探。这真是越看越焦虑，越焦虑越想看，形成一个恶性循环。

## 为何删除与拉黑可能仍不足够？

当然，最关键的是，即使你拉黑删除所有联系方式，甚至开小号，你仍然可能想方设法去查看。

换位思考一下，假如你发现你的前任每天来看你几十遍，你会不会感到害怕？我之所以这么说，是因为很多人被前任发现这种行为后，情况往往变得更糟。

## 错误行为对复合可能性的影响

以下是窥探行为可能导致的后果：

*   你的复合方式，除了纠缠，就没有其他办法了。
*   你自身状态都无法保持稳定，复合就更加遥遥无期。
