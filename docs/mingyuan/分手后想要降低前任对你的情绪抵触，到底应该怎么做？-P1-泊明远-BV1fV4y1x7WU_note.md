# 分手后想要降低前任对你的情绪抵触，到底应该怎么做？-P1-泊明远-BV1fV4y1x7WU_note

在本节课中，我们将学习分手后如何降低前任对你的情绪抵触，为可能的挽回创造一个舒适的沟通空间。情绪抵触是挽回过程中的主要障碍，克服它需要避免一些常见的错误行为。

不知道大家有没有发现，当你想要去主动联系前任的时候，他对你的态度会非常排斥和抵触。我们想要挽回前任，就势必要和对方建立起一个有舒适感的空间，才能创造出新的吸引力去吸引对方。

如果对方特别排斥我们，这对于复合来讲就会是难上加难。所以，我将告诉你们四个关键点。如果你还想去挽回前任，一定不要去做以下事情。

## 第一点：放平心态，避免受害者思维

上一节我们提到了建立舒适感的重要性，本节中我们来看看如何调整自己的心态。不要一分手就把自己放在了受害者的角度上。

因为80%以上的分手，多多少少都会有各自的缺点和问题。不要把所有的失望、失败的原因，归咎到对方身上。当然，除了一些情况是特例，比如：
*   **劈腿**
*   **家暴**
*   **冷暴力**
*   其他一些让你无法忍受的恶劣习性

这些情况肯定是对方的问题。之前有一个挽回者找到我时，他一边向我发泄对前任的不满和怨气，数落前任对自己的亏欠，又强调了自己付出了多少，把对方塑造成一个十恶不赦的坏人。

我告诉他说，如果对方真是这样，你就没有必要去复合了。过了一会，他回复说其实还是放不下她，舍不得这段感情。既然你都想要挽回她还放不下她，干嘛自己给自己折磨呢？

我希望你们可以明白一点：**即使是你们在这段感情里失败了，责任也一定是双方的。** 你想挽回，你的心态就必须要好。摆平不了心态，你的挽回就注定会失败。

## 第二点：沟通时避免提及感情与负能量

调整好自身心态后，我们来看看沟通中需要注意什么。如果分手后，你和前任还可以保持联系，可以进行正常聊天，那么此时就不要总是去和她提感情，提起你们的过去。

同时，也不要表达一些非常负能量的东西，例如：
*   “我失去了你，我就睡不着觉。”
*   “我失去了你，我就生不如死。”
*   “你之前对我那么好，怎么现在就变了呢？”

其实这些话都很没有意思。过去的是非对错我们先不论，最起码你现在是不能道德绑架人家的。一次两次她可能会觉得内疚和有歉意，但次数多了，只会觉得你烦人又恶心。这种情况下，就非常容易把你拉黑删除。

## 第三点：停止追问无意义的问题

在沟通中，除了避免负能量，还要注意话题的选择。不要去纠结一些没有必要的问题，例如：
*   “你还爱不爱我？”
*   “你为什么要跟我分手？”

你一次次的提这个话题，只会让她陷入到过去的困扰和痛苦当中，根本不会考虑你现在是怎么样的状态。

分手的原因我们要去追究，但一定不是让你挂在口头一遍一遍地说出来。挽回本身就是一个修复的过程。随着时间的流逝，你们的分手原因会逐渐淡化。这期间要做的，不是一遍遍重复你们的分手原因，而是你知道了原因之后，愿不愿意去改的问题。

那么最后人家原不原谅你，答不答应跟你复合，和你道歉的次数和态度更没有关系。想要她真正的接纳你，一定是你现在的状态能让她幸福。

## 第四点：精简沟通，不说废话

最后，我们来谈谈沟通的形式和效率。千万不要去发一个60秒的语音方阵。别说是你的前任，就算是一个耐心很好的人，都不一定听得完。她只会觉得你的需求感非常强。

以上这些内容，我只告诉你关键。想要和前任顺利复合，察觉对方的真实状态和情绪是很重要的。你做的再多，你做的再优秀，你变得更好，如果她内心都是抵触的，你做多少都会没有用，因为她不认可你。

本节课中我们一起学习了降低前任情绪抵触的四个关键点：**摆正心态、沟通避雷、停止追问、精简表达**。记住，挽回的基础是重建舒适感和吸引力，而这一切都始于降低对方的抵触情绪。我是泊明远，如果屏幕前的你现在也正在面临这样的情况，可以进一步寻求帮助。
