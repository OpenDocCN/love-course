# 把前任的逃避心理给阻断，这样对

在本节课中，我们将学习如何通过阻断前任的逃避心理，引导对方心甘情愿地与你复合。我们将从认知平衡、需求处理和关系测试三个核心步骤展开，帮助你走出对立状态，重建健康的互动关系。

## 平衡双方认知，重新定位关系 🤝

上一节我们介绍了课程目标，本节中我们来看看第一步：平衡认知。因为你始终和前任处在对立面上，所以很难让他真正接纳你。

不走出错误的关系定位，你永远无法获得新的情感体验。你可以试着把自己放在前任的角色里，来审视这段感情。只有这样，你才能真正看清自己的行为，在对方那里是否能够被接受。

## 进行需求处理，提高自身框架 🏗️

理解了认知平衡的重要性后，我们进入第二步：处理需求感。为什么你的做法会让前任逃避？是不是你的一举一动都透露着高需求感？

想要复合顺利，你需要先进行需求处理，把你的个人框架重新建立起来。核心在于降低表面需求，提升自身价值。

## 服从性测试：吸引对方回归投入 ✅

完成了自身框架的重建，我们来看看如何通过互动测试推进关系。第三步叫做服从性测试，目的是吸引对方逐步回归投入，最终达到复合。

当前任因为你的一反常态感到不习惯，说明他对你已产生情绪波动，并注意到了你的改变。此时可以开始分级测试。

以下是三级服从性测试的具体操作：

*   **一级服从**：提出一个非常微小、几乎无需成本的要求。例如：“我可以用一下你的某视频会员吗？”
*   **二级服从**：让对方为你做一件力所能及的小事。这需要他付出少许时间和行动。
*   **三级服从**：提出需要对方付出更多精力的请求。例如：“可以帮我取个快递吗？”或“周末有空一起看场电影吗？”

对方若接受测试，便可继续升级。他若拒绝，你就暂停一段时间。在此期间，持续进行自我提升，然后再进行测试，直到通过为止。

当对方通过较高级别的测试时，意味着投入已经形成。此时，即使你不主动提复合，对方也不会轻易让你离开。

## 课程总结 📚

本节课中，我们一起学习了通过三个步骤阻断前任的逃避心理：
1.  **平衡认知**：换位思考，摆脱对立关系定位。
2.  **处理需求**：降低需求感，重建高价值个人框架。
3.  **分级测试**：通过**服从性测试**引导对方逐步增加投入，最终实现复合。

记住，复合是一个重建吸引和舒适感的过程，而非单方面的说服或施压。
