# 分手后复合指南：八个必经阶段自测 📊 - 课程 P1

在本节课中，我们将学习分手后希望与前任复合的完整过程。我们将详细拆解从分手到复合通常会经历的八个阶段，帮助你进行自我评估，了解自己当前所处的位置以及距离成功复合还有多远。

## 概述：复合是一个阶段性过程

从分手到复合并非一蹴而就，而是一个需要经历特定阶段的过程。理解这些阶段，可以帮助你避免犯错，并采取正确的行动。

## 八个复合阶段详解

以下是分手后复合通常会经历的八个阶段。你可以对照自己的情况，判断当前所处的阶段。

### 第一阶段：分手与冷淡期

对方突然提出分手，表示不想再继续这段关系。对方态度突然变得冷淡，甚至不再回复你的消息。

### 第二阶段：纠缠与恶化期

你开始通过纠缠、道歉、认错、承诺等一系列操作试图挽回。但这些行为往往让对方感到压力，想要远离你，甚至可能导致你被拉黑或删除联系方式。

### 第三阶段：冷静与止损期

你开始冷静下来，意识到需要做一些正确的事情。你停止纠缠，开始对关系进行冷处理。

上一节我们介绍了情绪化的纠缠只会让关系恶化，本节中我们来看看如何进入建设性的阶段。

### 第四阶段：尴尬复联期

在经历了一段时间的冷冻期后，你尝试与对方恢复联系，但很可能会再次碰壁。因为对方已经平静的生活再次被你打破，对你的负面印象仍有残留。

**核心要点**：复联初期的态度必须是平稳、自然的，不能带有强烈的目的性。

### 第五阶段：舒适感重建与友谊区

你开始研究并选择正确的话题进行交流，降低联系的频率，专注于增加相处的舒适感。你的自我提升略有成效，并被对方看在眼里。

对方开始犹豫是否要再给你一次机会，但又担心会重蹈覆辙，担心你的改变只是暂时的伪装。于是，你们的关系进入了“友谊区”。

### 第六阶段：初次见面与印象刷新

你尝试邀约对方见面以寻求突破。初次见面时，你保持了情绪稳定和良好的状态，这为你增加了印象分。

**重要原则**：在此阶段，依旧不能主动提出复合。对方对你的改变感到惊喜，但依然夹杂着很大的犹豫。

### 第七阶段：顾虑打消与进入暧昧期

经过一段时间的邀约和联系，你通过线上和线下一致的良好表现，打消了对方的大部分顾虑。你开始尝试进行适度的暧昧，对方也愿意配合，并开始怀念你们曾经的美好。

于是，你们的关系正式进入了暧昧期。

### 第八阶段：契机利用与关系确认

你利用了一个合适的契机（例如，通过异性追求者刺激对方的占有欲，或采用后撤策略制造失去感），在对方情绪的引导下，使其产生了重新与你在一起的强烈愿望。

至此，成功复合。

## 总结：定位你的复合旅程

本节课中，我们一起学习了分手后复合的八个典型阶段：**分手冷淡期 → 纠缠恶化期 → 冷静止损期 → 尴尬复联期 → 舒适感重建期 → 初次见面期 → 暧昧期 → 关系确认期**。

理解这些阶段能帮助你理性看待复合过程，避免焦虑，并针对当前阶段采取最有效的策略。你现在正处于哪个阶段呢？
