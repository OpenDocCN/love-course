# 如果你也正在经历真性分手，应该怎么做-P1-泊明远-BV1o3411M7uV_note

在本节课中，我们将要学习当伴侣以“性格不合、相处很累”为由提出真性分手，且拒绝沟通时，你应该采取的核心行动原则。课程将围绕建立自我独立性、有效处理双方情绪以及进行实质性成长展开。

上一段我们提到了面对坚决分手时需要具备的基本心态，接下来我们将深入探讨三个具体的行动建议。

以下是三个核心的行动建议，旨在帮助你建立健康的心理基础并创造关系的新可能。

**第一，维持边界感，正视焦虑。**

如果对方表现出明显的抵触情绪，例如拒绝讨论感情话题、沟通、见面，或回避你的深刻道歉与反思，你应当立即停止这些行为。

有时，正确的解决方式恰恰是暂时不解决，或不逼迫对方立刻解决问题。

你可能会担心“他被忘了怎么办”、“他完全放下了怎么办”或“他遇到新的人怎么办”。这种时刻，你是在试图转移自身的焦虑，你在赌对方是否会像过去一样再次妥协。

但方向错误只会让两人越走越远。盲目刷存在感，除了短暂缓解你表面的焦虑外，并无实际作用。

真正解决焦虑的方法建立在**自我成长**的基础上。

**第二，学会接纳对方情绪，将落脚点放在解决问题上。**

例如，你发了一条内容安全的消息，对方却隔了很久才回复一两个字。这时，你应尝试接受一个前提：对方的一切回应都具有其合理性。

对方没有责任和义务必须配合你的所有要求。坦然接受他的不回应或负面情绪，是一门需要学习和强化的功课。

感情是流动的状态，并非 `1 + 1 = 2` 的数学公式。你不给对方压力，并温和地表达，只要他接收到，属于你们之间的改变就在悄然发生。

你在此需要做的功课很多，例如：完善自我人格、探索你在感情中不诚实的具体表现、掌握经营感情的能力。**有效的表达**与**共情对方感受**的能力，才是更新对方对你负面印象的核心关键。

**第三，维持自信且松弛的个人状态，多做少说。**

不要将所有问题罗列出来并告知对方“你正在改变”。改变对任何人而言，都是一件需要持续投入精力并经受时间考验的事情。

如果你信誓旦旦地向对方表示你能改变，这并非负责任的表现，反而容易给人留下“说空话”的印象。

从意识到问题，到知道问题所在，再到寻找解决方法，最终应用到实践中使关系发生真实改变，中间还有很长的路要走。

在咨询中，我常提醒来访者警惕陷入“受害者心态”。从认知觉醒的一刻起，你的目的就不仅仅是挽回这个人，而是让自己**具备经营长期感情的能力**。

让自己在面对这段关系，乃至未来任何一段关系时，都能拥有更多主动权。这才是我们应该追求的目标。

本节课中，我们一起学习了应对真性分手的三个核心原则：**保持边界以管理焦虑**、**接纳情绪并聚焦问题解决**、以及通过**切实行动而非空谈**来维持自信状态。关键在于将焦点从“挽回对方”转移到**实现自我成长与人格完善**上，从而为任何关系奠定健康的基础。
