# 想要和前任重归于好，这两句话千：高田横版301-泊明远-BV1

在本节课中，我们将学习分手后想要重归于好必须牢记的两个核心原则。这两个原则将帮助你理清思路，采取正确行动，避免在挽回过程中陷入被动或做出无效努力。

## 第一原则：认清现实，摆正位置

上一节我们介绍了课程概述，本节中我们来看看第一个关键原则：“屁股决定脑袋”。这句话的意思是，你必须认清分手已成事实的现状。你和他当下的关系是什么，就只做符合这个关系身份的事情，不要做出越界的行为。

**核心行动指南：**
*   在一起时都未曾给予的过度关心，分开后反而殷勤示好，这种行为不合常理。
*   越界行为不仅让你讨不到任何好处，反而会让对方觉得“这招对你好使”，从而降低你的价值和吸引力。

## 第二原则：用行动代替空谈

理解了身份边界的重要性后，我们进入第二个原则：“事实比承诺重要”。无论你说得多么天花乱坠，在对方看来都可能苍白无力。因为在决定分手前，对方通常已经给过你多次未被察觉的改正机会。

**核心行动指南：**
*   不要总去质问或评论对方“为什么分手”。
*   你应该做的是努力弥补过去关系中自己不足的地方。
*   通过实际行动来打破对方对你原有的刻板印象。
*   同时，做一些积极的事情，来抵消你们之间累积的负面情绪。

遵循以上两点，你才能避免糊里糊涂地犯错，从而导向一个可能的好结果。

**本节课总结：**
本节课我们一起学习了挽回前任时必须牢记的两个核心原则。第一是“认清现实，摆正位置”，要求我们接受分手事实，不做越界之事。第二是“用行动代替空谈”，强调实际行动远比口头承诺更能打破僵局、重建信任。我是泊明远，如果你正面临类似的情感困境，可以寻求进一步的帮助。
