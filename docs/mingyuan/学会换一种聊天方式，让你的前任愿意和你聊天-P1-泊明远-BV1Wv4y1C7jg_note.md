## 概述

在本节课中，我们将要学习如何通过改变沟通方式，让你的前任愿意与你重新开始聊天。我们将探讨沟通的本质、聊天的核心目的，并重点学习如何避免无效的“封闭式聊天”，转而使用更有效的“开放式聊天”来引导对话。

---

## 沟通的本质与目的

沟通的本质是释放、接受和反馈情绪与信息的过程。

聊天的目的大致可以分为两块：**信息交换**和**情绪交换**。

你可以想一想，自己和前任聊天的时候，是不是从来都没有想过他找你聊天的目的是什么。那是不是从来都没有意识到，这是一味地陷入到自己的情绪当中。

---

## 常见的错误聊天方式

上一节我们介绍了沟通的目的，本节中我们来看看分手后常见的错误聊天方式。

很多人从来不会去思考对方的情绪，只顾着宣泄自己的情绪。对于分手之后的联系，大多数都是在前任不想回复你的时候去聊，前任不想聊的话题，也就是扯感情上的事情，最后闹得两个人都是不愉快收场。

那么你换位思考一下，如果是你，你会回这种信息吗？这种聊天你的前任还能回复你，不是出于礼貌，就是他的性格太温和。

另外，就是有一部分人的聊天，毫无目的地尬聊，聊了半天一句有用的信息都没有。那除了早安晚安在做什么，其余的话题就不会说了。

而我们再来换位思考一下，你对一个人没有一点好感，甚至反感的情况下，你会回复这么无聊的消息吗？

---

## 如何让前任愿意回复：开放式聊天

了解了问题所在后，我们来看看解决方案。怎么让前任愿意回你消息呢？做一个最为基础的改变，那就是聊天的时候，我们要拒绝封闭式的聊天，多用开放式的聊天。

也就是我们在聊天的过程当中，在信息交换的时候，多给自己留一些可以延续的话题。

很多人听到这里可能会有点懵，什么叫封闭式的聊天，什么叫开放式的聊天？其实也很简单。

以下是两种聊天方式的定义：

*   **封闭式聊天**：一问一答。他的回答选项只能是“是或者不是”、“有或者没有”。
    *   **公式**：`你的问题 -> 对方回答（是/否） -> 对话结束`
*   **开放式聊天**：提出能引发对方描述、分享或表达观点的问题。
    *   **公式**：`你的问题 -> 对方描述/分享/表达 -> 你根据回答延伸新话题 -> 对话继续`

---

## 开放式聊天的应用与示例

封闭式聊天存在明显的问题。举个例子，你想找对方聊天，然后你问他“吃了没”、“睡了没”，他就只能回答你“吃了”或者是“没吃”。那不回你的就很尴尬，他回你也不知道接下来要说什么。

或者你按照之前在一起时候的相处模式，没事分享点开心的事情、好玩的事情，这种大多数发出去他都不回。所以查户口或者单纯分享性质的话题，在分开后其实是不适用的。这就是封闭式聊天所存在的问题。

即便你的前任他开始愿意回答你、愿意回复你，也会因为你的聊天过于无聊，而不知道跟你说些什么。

那么我们就要学会用开放式的聊天去引导对方的回复，并且在这个对话过程当中，你起码要预留一个以上可以延续话题的点。

简单来说，当你开启话题的时候，你不怕没有话说。

以下是开放式聊天的应用思路：

比如聊前任，他很懂车，他也知道你有车或者也开过。那么关于“车子的保养费用去哪里做比较靠谱”、“能不能让他推荐一下”或者“最近想换轮胎有什么建议”，这都是可以延伸的话题。

**代码示例（对话思路）：**
```
糟糕的封闭式提问：你的车保养了吗？ -> 对方：保养了/没保养。
良好的开放式提问：我车快到保养期了，听说XX品牌的车保养特别讲究，你之前是怎么选的保养店？ -> 对方：哦，我一般去...因为... -> 你：原来如此，那除了这家，还有别的推荐吗？或者需要注意什么坑？
```

这不是比你之前那种封闭式的、一问一答式的聊天要强多了呢？

## 总结

本节课中我们一起学习了如何通过改变聊天方式与前任重建沟通。我们首先明确了沟通的本质是情绪与信息的交换。然后分析了分手后常见的错误聊天方式，如只顾宣泄情绪或进行无聊的“封闭式”尬聊。最后，我们重点学习了**开放式聊天**的核心技巧：通过提出能引发对方描述和分享的问题，并在对话中预留可延续的话题点，从而引导对话自然进行，让对方更愿意参与交流。

换一种聊天方式，也许更容易让前任愿意跟你聊天。
