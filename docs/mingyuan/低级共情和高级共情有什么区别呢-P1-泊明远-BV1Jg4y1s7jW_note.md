## 低级共情与高级共情的核心区别

很多人认为共情就是换位思考。

仅仅进行换位思考是不够的。你必须走入对方的内心，体验对方的感受，才能发自内心地表示理解。否则，你只是口头上表示理解，内心并不认可，最终连自己的真实心声和观点都无法表达。这就是盲目的共情。

所以，心口不一就是低级共情的一种表现。这不是真诚，而且会让对方觉得你虚伪。

## 低级共情的典型表现：以道歉为例

以下是情侣间发生矛盾时，低级共情的一个常见场景：

*   **场景**：情侣闹矛盾。
*   **低级共情做法**：为了哄好对方而去道歉，说“都是我不好，让你不开心了，别生气”。
*   **问题分析**：这未必是你真实的想法（“真的是我不好吗？”）。你本身也不开心，即便忍耐着自己的性子去哄对方，对方也不一定会积极回应，甚至可能让你“别说了”。这就是低级的共情。

## 如何实现高级共情：话术与结构

上一节我们看到了低级共情的无力，本节中我们来看看如何通过改变话术来实现高级共情。

如果此时我们改变一下表达方式，高级的共情话术可以如下构建：

```
“抱歉，我刚刚不该把话说得那么难听，这真的不是我的本意。是我没有控制好情绪，当时没能理智地耐心听你的真实想法，确实惹你生气了。我能感受到你听到那些话时的失望和难过。我很希望能积极解决这个问题，不想以后因为同样的问题闹得不愉快。所以，我想先听听你内心的真实感受和想法，然后你再来听听我的，可以吗？”
```

这段表达遵循了一个清晰的逻辑结构，以下是其核心步骤：

1.  **先说明自己的问题**：主动承认自身在情绪或行为上的不当之处。
2.  **表达对对方的理解**：陈述你能感受到对方的情绪（如失望、难过），表明你走入了对方的内心。
3.  **提出对美好未来的期待**：表达共同解决问题的意愿和面向未来的积极目标。
4.  **阐明你的沟通目的**：清晰地提出接下来希望如何沟通（如交换彼此的真实想法）。

## 高级共情的核心口诀

总结以上高级共情的步骤，可以提炼出一个简单易记的口诀：

**责人先责己，再夸人再说事。**

这个口诀概括了高级共情的行动顺序：先进行自我反思和归责，接着肯定和理解对方，最后才共同探讨事情本身。
