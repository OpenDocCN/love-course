## 概述：为何你处于低位？

分手后长期处于低位，你该如何翻身？如何才能让前任对你产生“失去感”呢？首先，你需要问自己一个问题：分手后，你是否持续做出了让前任认为你还想复合的举动？

如果你的答案是“有”，那么“失去感”就永远不可能出现。因为你的前任明确知道，你就是想挽回他。这样一来，对方就可以肆无忌惮地影响你的情绪和心态。

## 低位状态的本质

如果你正在被前任拖着、吊着，你需要明白一个事实：分手后你的关心和付出，他并不需要。你在对方心目中，可能已经成为一个毫无价值、可有可无的人。

既然你已经看到了这里，说明你想要主动出击，想要制造“失去感”，也想让对方尝尝失去你的滋味。那么，我们接下来就深入探讨其核心。

## “失去感”的核心：相互博弈

“失去感”的核心在于双方的相互博弈。很多时候，这场博弈比拼的是谁比谁更不害怕失去对方。或者说，是谁更有勇气去面对和承担失去的风险。

许多成功逆袭的案例都印证了这一点：在断联一段时间后，前任看到你最近过得很好、很滋润，反而会突然来撩拨你，刷存在感。他可能会先后给你打无数个电话，发无数条短信。

**关键行动公式：**
```
当对方主动联系时，你的回应策略 = 不接电话 + 不回信息
```

如果你一个电话都没接，一条信息都没回，他就会发现联系不上你、找不到你了。直到这个时候，他才可能真正意识到：自己快要失去你了。

## 掌握选择权

当选择权掌握在自己手里时，要不要复合、想不想继续，就完全取决于你自己了。主动权回到了你的手中。
