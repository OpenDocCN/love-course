# 分手后前任很绝情，还有没有挽回的机会呢？-P1-泊明远-BV1T8411V7qX_note

在本节课中，我们将要学习如何分析并应对分手后前任表现出的绝情态度。核心在于理解并满足对方的深层期待，从而找到真正的挽回方向。

当你的表现超过了对方的预期，那么再绝情的人也会主动回头。分手后前任很绝情，是否还能挽回？这类问题可能是咨询中出现频率最高的一种。大部分面临这种情况的人，都会被前任的绝情态度所干扰。

因此，他们无暇去思考关系的本质是什么。看了很多技巧和干货，除了让自己更加迷茫和焦虑之外，似乎也没有什么作用。但是这个问题在我看来，其实只需要一句话就能够突破当前的困境，那就是：

**当你能满足另一半的期待，离开的人自然就会回头。**

那么，前任的期待到底是什么呢？根据以往的案例经验，我总结了三个主要的方向。你可以按照这三个方向，来对照一下自己的感情发展，找到真正的挽回方向。

---

上一节我们提出了“满足期待”的核心观点，本节中我们来看看具体是哪三个方向。以下是三个关键的期待方向：

**第一个方向：价值增长期待**

这是大部分感情破裂的最直接原因。这里的价值包含了**物质、新形象、未来**等等。假如在一起的时候，你的用心是90分，到现在变成激情消退、目标达成，那么你的用心变成了50分。

而对方最直观的感受就是：“你不爱我了”。我们都希望感情会长久，但是大部分人在恋爱里，能给对方的奖励都是越来越少。比如在一起后，对另一半越来越敷衍，身材越来越差，物质奖励越来越少，个人成长的速度也越来越慢。

再加上缺少对于未来的规划和目的，那么分手就是一个必然的选择。如果是这个问题，那么你就要好好想想，对方对你的期待是什么，你该怎么去做出提升。

**第二个方向：幸福增长期待**

失望、疲惫、无感情而分手，都有这些原因。很多人对于爱情的认知和需求，只是停留在最基础的层面。因此一段关系短则几天，长则一两年，就会让人失去继续下去的动力。

这个动力是什么呢？比如在一起之前，生理冲动会让人花100分的心思去追求、讨好。但是生理需求满足了，继续的动力又是什么？于是有人用物质来建立联盟（例如，“和我在一起有钱随便花”），有人用承诺给予希望（例如，答应结婚生子），也有高手给出终身的目标。那么，在到达终点之前，只要你懂得经营，幸福感就会不断地增长。

但是如果你的目的提前达成，收获感就会不断地降低。这就像一家企业不懂得经营，没有做大做强的希望，还在不断地亏损维持，那为什么还要继续呢？所以，如果是这两个问题，那么你只需要思考两件事：
1.  你们的希望在哪里？
2.  怎么样让他相信，你们能实现这个希望？

**第三个方向：竞争者期待**

简单来说，就是出现了一个人，拉高了对方的期待值，激发了他对你的不满意。当然，这里的竞争者不是特指新出现的人，而是另外一个圈子的价值干扰。

例如：
*   女生突然发现自己的闺蜜男友是高富帅，她可能会觉得自己的男友是矮瘦穷，妒忌让人失去理智。
*   或者，她最近进入到一个新的圈子，这里面异性的价值普遍高于自己的伴侣，那么你猜她会不会出现不平衡心理呢？

如果你们的关系，分手的成本过高，那么竞争者期待就会变成她对你的要求和争吵。但是如果你们的感情平淡，分手了也没有什么损失，那么分手几乎是必然的。如果你遭遇的是这类问题，建议果断放手，因为竞争期待没有终点。

---

最后，大家都不再轻易地被情敌左右，选择考虑别人的期待，但也要考虑自己的期待。谈恋爱是为了幸福，而不是为了成就谁，或者去拯救谁。**能够双赢的关系，才值得你去努力。**

本节课中，我们一起学习了如何通过分析“价值增长”、“幸福增长”和“竞争者”这三个核心期待方向，来理解前任绝情背后的原因，并找到可能的挽回路径。记住，满足对方合理的深层期待，是让关系回暖的关键。
