# 在绝情的男生也能复合-P1：高田横版104-泊明远-BV1Uh411p78W_note

在本节课中，我们将要学习如何理解情感关系的基础，并探讨即使面对绝情态度，关系依然存在复合可能性的核心原因。我们将分析爱情的开始与经营不善的区别，并明确方法的重要性。

---

每一段感情的开始都基于双方存在过的爱意。只要两个人曾经在一起，就证明彼此之间有过爱的连接。哪怕只在一起一天，这种开始也意味着对方曾对你怀有感情。

现在对方表现出绝情态度，这通常不是爱意完全消失的标志。绝情态度的产生，往往源于感情经营过程中出现的问题。双方可能没有采用有效的方法来维护和发展这段关系。

这不是任何一方个人的错误。问题的核心在于方法，而不在于人本身。用对方法是改善关系的关键。

---

本节课中我们一起学习了情感关系复合可能性的基础。我们明确了：感情的开始证明爱意存在；当前的绝情多源于经营方法问题；解决问题关键在于找到并应用正确的方法。理解这些是迈向有效复合的第一步。
