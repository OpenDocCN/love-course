# 情感修复课程 01：明确联系前任的目的 🎯

在本节课中，我们将要学习一个关键的情感修复原则：在采取任何行动之前，必须首先明确你联系前任的根本目的。清晰的目标是决定后续所有行为和策略的基石。

## 明确核心目的

上一节我们介绍了课程主题，本节中我们来看看如何具体分析你的目的。联系前任时，你的内心动机决定了行为的性质和潜在后果。

你需要想明白一个问题：你联系前任，目的到底是什么？你是想要和好，还是仅仅为了聊天以解决自己的情绪苦闷？亦或是希望对方帮助你走出痛苦的过程？

## 根据目的采取行动

理解了目的的重要性后，我们来看看不同目的对应的行动策略。你的目的不同，采取的方式自然就是不同的。

以下是基于不同目的的行动建议：

*   **目的：缓解自身痛苦**
    *   如果你的目的只是为了让自己不难受，那么你可以选择联系。最坏的结果无非是被删除或者拉黑。

*   **目的：寻求复合**
    *   如果你的目的是想要去和好，那么就不要去轻易联系。至于是否要在关系紧张的时候见面，这需要我们**从长计议**。

## 约束行为与理解差异

如果你能够真的明白并理解上述意思，其实你就该知道自己怎么做。如果你还想要和好，你首先要约束的就是自己的行为。因为我们一旦做错，后续情况只会越来越糟糕。

其次，无论男生还是女生，都应该多去了解彼此的思维逻辑变化。

以下是男女思维差异的一个常见例子：

*   **女性常见思维**：女生生气时，可能总喜欢用“分手”来吓唬对方，以获取到自己想要的关注。
*   **男性常见反应**：刚开始，男生可能会很紧张。但是次数多了，男生也就疲惫甚至厌烦了。对于男生来讲，每次提分手，都像是在他心里**扣分**。直到最后一次，他可能会直接答应，甚至主动提出分手。

## 关系破裂的核心与修复

当你错误的相处和经营方式导致关系破裂时，对方就不会再去在意你提分手的目的是什么。因为他只会关注自己的情绪，并想要逃离。

一段感情的长期稳定，需要的是彼此的包容、调整和互相的让步。
