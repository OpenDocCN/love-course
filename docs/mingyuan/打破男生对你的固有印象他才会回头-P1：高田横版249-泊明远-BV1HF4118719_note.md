# 💔 情感修复课程 P1：如何打破固有印象促复合

在本节课中，我们将要学习一个核心的情感修复策略：通过打破对方对你的固有行为预期，来重新激发他的好奇心和吸引力，从而为复合创造可能性。

## 核心原理：打破预期以重塑吸引力

上一节我们介绍了课程的主题，本节中我们来看看其背后的原理。长期恋爱关系容易形成固定的互动模式，对方会对你的反应产生**准确的预期**。这种可预测性会削弱新鲜感与吸引力。

**公式：可预测性 → 吸引力下降**

因此，改变你的一贯反应模式，做出**出乎对方意料的行为**，是打破僵局的关键。你的所有非常规举动，都能干扰他的固有认知，从而增加你在他眼中的神秘感和吸引力。

## 具体操作策略：反向行动指南

理解了原理后，以下是两个核心的反向操作步骤，用于在实践中打破他对你的固有印象。

1.  **当对方提出分手时**
    传统预期：你会哭泣、纠缠或卑微祈求。
    打破印象行动：保持冷静，坦然同意他的决定。这种镇定和尊重会让他感到意外，并开始重新评估你的价值。

2.  **当对方主动联系你时**
    传统预期：你会热情、急切地回应。
    打破印象行动：给予冷淡、简短的回复，甚至暂时不回复。这会让他无法获得预期的情感满足，从而激发他的探究欲和失落感。

## 策略目标与总结

本节课中我们一起学习了情感修复的“破局”思维。通过上述反向操作，目的是让他无法再用旧的模式来定义你，从而**有可能促使他主动反思并产生挽回的念头**。记住，改变是吸引的开始。
