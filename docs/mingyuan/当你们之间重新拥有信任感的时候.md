# 当你们之间重新拥有信任感的时候

在本节课中，我们将要学习复合后如何有效消除两人之间的隔阂。核心在于理解印象如何影响感觉与预期，并通过具体行为改变这些印象，从而重建信任与亲密感。

## 理解隔阂的根源：旧印象的固化

上一节我们概述了课程目标，本节中我们来看看隔阂产生的根本原因。

两人复合后，关系得以恢复，但过去冲突中留下的负面印象可能依然存在。例如，如果你当初是因发脾气而提出分手，那么你当时情绪失控的样子，会像一张照片般印在对方的脑海中。

这种印象会导致对方认为你是一个**情绪不稳定**的人。这里存在一个关键概念：**他对你的印象，等于他对你的感觉**。印象具有惯性，它不会单纯因为关系状态（如复合）的改变而自动更新。

**公式：印象 = 感觉**

因此，即使你们已经复合，若他对你的旧印象仍是“爱发脾气的人”，那么他对你的感觉就依然是**害怕**与**反感**。这正是隔阂持续存在的心理基础。

## 消除隔阂的核心：用新印象替换旧印象

既然旧印象是隔阂的根源，那么消除隔阂的关键就在于**铲除旧印象，树立新印象**。

那么具体该如何操作呢？我们引入第二个关键概念。

## 关键概念：印象如何塑造预期

他对你的印象，直接决定了他对你的**预期**。

**公式：印象 → 预期**

如何理解？如果他对你的印象是“情绪不稳定”，那么当你再次遇到烦心事时，他就会**预测**你可能会发脾气、大喊大叫。这种预期会让他处于防备状态，阻碍亲密感的自然流动。

因此，树立新印象的策略，就在于主动管理并打破这种负面预期。

## 行动指南：如何通过“超预期行为”重塑印象

树立新印象的核心方法是：**多做一些超预期的行为**。因为超预期的行为会反过来管理并改变他对你的感觉，感觉变了，印象也就随之更新。

以下是实施“超预期行为”的具体思路：

当他认为你会发脾气时，你表现出冷静与理智。
当他认为你会指责他做得不好时，你反而给予理解与支持。

这些行为都违背了他基于旧印象产生的预测，因此是“超预期”的。当你持续、足够多地做出这类行为时，他会产生一种新的感觉：“你和之前不一样了”。这种感觉，我们称之为**刮目相看**。

**核心逻辑链：超预期行为 → 改变感觉 → 更新印象 → 刮目相看**

当他对你刮目相看时，旧的隔阂便不再重要。新的正面印象和感觉会自然吸引他，**让他重新爱上你**。

## 课程总结

本节课中我们一起学习了复合后消除隔阂的完整逻辑。

1.  **隔阂源于旧印象**：过去的负面行为形成的印象，会持续产生负面的感觉。
2.  **印象决定预期**：对方会根据对你的印象，来预测你未来的行为。
3.  **破解之道是超预期行为**：通过做出冷静、理解等超出对方负面预期的行为，来改变其感觉与印象。
4.  **最终目标是“刮目相看”**：当新的正面印象建立，隔阂消失，关系便能真正重启。
