# 把握分手后的平衡感，就很难有复合不了的感情-P1-泊明远-BV1nd4y1U7vf_note

在本节课中，我们将要学习分手后如何把握关系的“平衡感”。这是挽回后期成功与否的关键，考验着你对这段关系的认知深度。许多人在关系出现转机时，因为无法掌握平衡而前功尽弃。我们将通过三个核心要点，帮助你建立平等、健康的关系互动模式。

上一段我们提到了平衡感的重要性，接下来我们具体看看什么是关系中的平衡感。它指的是双方处于平等、互相尊重的状态，没有一方需要刻意讨好或依附另一方。尤其在挽回后期，当联系恢复甚至出现暧昧时，保持平衡能避免你陷入被动。

如果你不能掌控这种平衡，复合就容易变成单方面的讨好，导致你被对方牵制，甚至感到被“吊着”。因此，无论你是想从朋友做起慢慢挽回，还是决定放弃，双方都应该是平等的。没有人必须需要谁，也没有人必须讨好谁。良好的平衡感是一段健康关系的核心。

为了帮助你掌控关系平衡，以下是三个必须做到的要点：

首先，要做到进退自如。这指的是在和前任互动的过程中，你既能投入，也能抽离。你可以全身心投入，让对方感受到你的积极和热情；而当对方试探或冒犯你时，你也能适时后退，亮明自己的底线。这样做能防止你沦为低自尊的讨好者。在一段关系中，能够进退自如的人，擅长用更宏观的“上帝视角”看待问题，因此会更加理性。他们既能享受关系中的甜蜜，也能及时从痛苦中撤离，避免受到不良互动的影响。

上一节我们介绍了“进退自如”的心态，本节中我们来看看第二个要点：坚持互惠互利。很多人认为“我爱她，就要让她感受到我的爱，她就会回头”。但现实会让你明白，感情中也存在“利益”交换。我们与一个人恋爱或结婚，除了被对方吸引，也会贪恋对方的好、对方的付出，以及这段关系带来的快乐与幸福。对方也同样如此。你们之所以分开，往往是因为他觉得和你在一起不舒适、不快乐。因此，你必须坚持互惠互利的心态，主动提供舒适感，让对方觉得和你在一起有“好处”。同时，这段关系也应该能让你感到舒适、获得好处。只有这样，关系的天平才能维持平衡。

接下来是第三个要点：降低你的占有欲。强烈的占有欲会让你过于看重“复合”这个结果，从而不由自主地向对方施加压力。结果就是，你们之间脆弱的平衡关系会被打破。随之而来的，往往是对方为了逃避压力而与你划清界限。我希望大家明白，任何一段健康的关系都需要保持一种“连接感”，而不是“占有感”。这种连接感能让对方在关系中感到舒适，同时也能让他更尊重你，尊重你的意愿。

本节课中，我们一起学习了掌握关系平衡感的三个核心方法：进退自如、互惠互利和降低占有欲。记住，平衡的关系是平等、互相尊重的，它能让双方都感到舒适，并为成功的挽回或各自的新生奠定健康的基础。
