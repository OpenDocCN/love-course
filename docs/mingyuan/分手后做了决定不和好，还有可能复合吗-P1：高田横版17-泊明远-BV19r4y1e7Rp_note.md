# 情感修复课程 P1：决定不回头后的复合可能性 🔄

在本节课中，我们将探讨一个常见的情感困惑：当一方明确决定分手且不愿回头时，两人是否还有复合的可能。我们将分析关系中的权力动态，并指出改变现状的关键所在。

---

上一段我们提出了核心问题，本节中我们来具体看看为何一方会显得“说一不二”。

你认为他不会改变，是因为他在你面前从来是说一不二。你给了他放纵的条件，并且不停地退让。

以下是这种互动模式带来的后果：

*   这会使他勉强卑微到尘埃里。
*   他肯定会表现得说一不二。

---
