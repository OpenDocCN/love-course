在本节课中，我们将学习分手后如何通过重建自我价值来有效应对情感危机。核心在于将关注点从“挽回对方”转移到“提升自己”，这是修复关系或开启新生活的根本。

## 🧠 为何我们会自我否定？

上一节我们提到了关注自我的重要性，本节中我们来看看自我否定感的来源。分手后感到自己很差并产生恐惧，通常是因为被抛弃、被背叛的经历。对方的离开，是用实际选择否定了我们的价值。

那么，关键问题在于：我们对自己的看法究竟是如何形成的？

如果世界上只有你自己，你对自己将一无所知。如果世界上只有两个人，其中一人说你好看，这便构成了你认知的一部分。在爱情中，你唯一的爱人若说你不值得被爱，这种来自挚爱之人的否定，极易摧毁一个人的自尊心。

## 🔄 破解否定：多元的评价体系

然而，这种负面影响并非无解。能对我们做出评价的重要人物，不只有爱人，还包括亲人、朋友和其他人。爱情本身也不只有一段，我们还有前任和潜在的下一任。

因此，如果分手后你正在自我否定，认为自己配不上对方或遇不到更好的人，请现在就拿出一张纸，重新梳理你的价值。

以下是重建自我价值清单的三个步骤：

1.  **回顾个人成功史**：找到过往让自己感到满意的成就或时刻，详细写下来。
2.  **收集亲友的肯定**：记录下亲人、朋友眼中你的优点，以及他们给予你的支持。
3.  **梳理恋人的积极评价**：从第一段恋情到现在，找出历任恋人对你的积极评价并记录。

记录优点时，要尽可能详细，哪怕再细微的亮点也要写下，并找到对应的事实依据。完成这个过程后，你往往会发现，自己其实是一个非常优秀的人。

## 💖 爱情只是生活的一部分

最后，我们需要建立更宏观的视角。爱情如同事业一样，只是我们生活的一部分，它不能决定我们人生的成败。只要你愿意努力和用心，你依然拥有很多收获幸福的机会。

无论是为了挽回一段感情，还是为了追求未来的幸福，我们都必须首先学会爱自己。

本节课中我们一起学习了分手后自我否定的根源、如何通过梳理多元评价来重建自我价值，并认识到爱情在人生中的合理位置。记住，真正的改变始于自身。
