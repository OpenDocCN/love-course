# 挽回前任首先要明白如何付出，才会达到预期的效果！-P1-泊明远-BV1DV4y157YN_note

在本节课中，我们将要学习挽回前任过程中的核心前提：如何以正确的方式付出，才能达到预期的效果。许多人在分手后因焦虑而采取错误行动，结果往往适得其反。我们将剖析这些行为的本质，并建立健康、自信的心态基础。

很多人在分手后非常焦虑。刚分手时恨不得把自己的全部给对方，觉得只要能够做到这些，对方就能回到身边。

可接着你就发现，对方要么对你置之不理，要么冷淡地回应。这种方式最大的问题在于，你对自己本身不自信。

你需要不断地去满足对方提出的附加条件，然后才觉得对方能够回来。如果对方真的是因为这些附加条件回到你身边，你会开心吗？你会满足吗？

这很简单，就像你去买东西。你喜欢这个东西，但是你发现有那么一点瑕疵。这时销售为了把东西卖给你，就给你打折，给你送赠品。

但你的心里还是会想，是不是还会有更大的折扣。很多时候，你的前任就是这样想的。

你对自己不自信，觉得自己很差劲。觉得自己没有办法再去吸引到对方，没有办法再让对方再次喜欢自己。

所以就想着用无底线的方式去倒贴，去换取别人对你的回应。

那么我今天就告诉你，要从一开始就得摆正你自己的心态。你们已经分手了，最坏的情况就是分手了，你还害怕什么呢？再者还能遇到什么更坏的程度呢？

对方的条件或许非常好，你在他眼里其实也并不是差。如果你真的没有那么差劲，他从一开始就不会选择你，就不会和你在一起。

你连这点认知、这点自信都没有，怎么可能挽回成功呢？

而且大多数情况下，他离开你不是因为你有多差，而是因为讨厌和你的相处模式。并且只是对方觉得你没有办法改变而已。

你不要有那种习惯性的思维方式和戏剧性的心理控制。你要相信自己是一个有优势的人，也是有发光发热的点。

你们俩之前在一起相爱，就一定是互相吸引的。所以说，拿出你自己的自信。

所有的感情无非就是：**你的需求来换我的付出，我的付出换你的需求**。

那感情流动的本质，不就是对等的交换吗？

那你记住，不要再做那些无底线的事。专注做好那些大概率会发生的事情。

以下是分手后初期应该专注做好的高概率正确事项：

*   **做好一个合格的前任**：刚分手时，保持冷静和尊重，不纠缠。
*   **稳定自身情绪**：管理好自己的情感，避免被情绪支配行为。
*   **提升工作表现**：让你的工作开始发光，创造价值。
*   **获取外界认可**：让你的价值得到更多外界的正面反馈。
*   **减少情感依赖**：不再依赖他的反馈来确认自我价值。
*   **明确现状与未来**：能够清醒地看到自己当下在哪里，未来要去哪里。

当你看清自己所处的阶段，把有限的资源放在最重要的事情上，你所做的一切才有可能会成功。

本节课中，我们一起学习了挽回前任的基石——建立自信与正确的心态。我们明白了无底线付出的危害，理解了感情的本质是**对等的价值交换**。关键在于停止错误的行为，将精力专注于自我提升和那些能产生积极结果的高概率事件上。我是泊明远，如果你现在时常还理不清思路，不知道自己该如何去做，可以寻求进一步的指导。
