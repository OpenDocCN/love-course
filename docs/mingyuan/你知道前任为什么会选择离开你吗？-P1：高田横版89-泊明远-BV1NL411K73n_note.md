# 情感修复课程 P1：理解分离的核心动机 🧠

在本节课中，我们将要学习一个关于情感关系分离的核心心理模型。这个模型将帮助你理解，当一段关系结束时，对方行为背后的根本驱动力是什么，从而让你能更理性地看待分离，而非仅仅纠结于情感层面。

## 分离的根本原因：趋利避害

上一节我们提到了需要理性看待分离，本节中我们来看看驱动人类行为，包括结束关系的一个基本原则。

最真实的回答是，对方选择离开，是为了逃避痛苦并追求快乐。具体来说，在他的认知框架内，他认为与你继续在一起是痛苦的。同时，他认为离开你，是获取快乐最便捷的方式。

因此，你不应总是用“爱”或“不爱”来框架化这个问题。

## 人类行为的底层逻辑

理解了分离的动机源于对痛苦的逃避，接下来我们深入探讨一下这种行为背后的普遍人性逻辑。

人被逼到一定份上，其行为都是趋利避害的。这可以看作一个基础的行为公式：

**行为选择 = 追求 perceived benefit（感知利益） - 避免 perceived cost（感知代价）**

在对方的情境中，与你在一起的“感知代价”（痛苦）超过了“感知利益”（快乐），因此他做出了离开的选择。

以下是关于此逻辑的几个关键点：

*   **认知决定感受**：对方的痛苦源于他**认为**和你在一起是痛苦的。这种认知可能基于现实矛盾、累积的失望或单纯的感受变化。
*   **路径选择**：他将“离开”评估为当前情况下，解决痛苦、获得快乐**最便捷的路径**。这并不意味着这是唯一或最好的路径，只是在他当下的判断中如此。
*   **超越“爱”的讨论**：将问题简化为“不爱了”，会阻碍你理解关系中实际存在的、导致他感到痛苦的具体问题（如沟通模式、需求未被满足、压力等）。

## 课程总结与引导

本节课中，我们一起学习了对方选择离开的核心心理动机——即**趋利避害，逃避感知到的痛苦，追求感知到的快乐**。你需要理解，这是一个基于个人认知和当下情境的决策，而非对你不价值的终极判决。
