# 分手后想让前任看到你的改变，我：高田56横_batch-泊明远

在本节课中，我们将要学习情感修复中的一个核心问题：如何让前任看到并相信你的改变。许多人在分手后急于展示自己的不同，但方法不当反而显得刻意。我们将探讨什么是本质的改变，以及如何自然地让改变被察觉。

---

我一直强调，挽回的核心在于改变。无论是在评论区还是在咨询过程中，我经常被问到这样的问题：“我们是异地恋，已经被删除拉黑了，我们连见面都见不到，我怎么能让他看到我的改变呢？如果看不到我的改变，我又怎么去挽回他呢？”针对这个问题，我今天做一个统一的回复。

首先，改变是一个过程。它是沉淀自己、不受外界任何因素影响的一种本质上的转变，这才叫真正的改变。我举一个非常简单的例子：当你做一件事情时，很多人都在否定你。那么，是用你做出的**结果**去证明自己更有说服力，还是告诉别人你在做这件事时付出了多少努力、经历了多少艰辛更有说服力呢？

虽然对于被挽回的人来说，他不应该只看到一个改变后的你，但我也想告诉大家：如果你说想挽回，那么一定是你改变后的样子站在他面前会更有说服力。当你愿意改变之后，你还愿不愿意选择他，那是你自己的事情。哪怕你变得更好之后去选择别人、选择更好的人，都不会有人怪你，因为你变得更有价值了。

为了挽回而做几天、顶多几个月的改变，和你脑子里一直在想“我到底怎么样才能让这个人看到我的改变”，这些都不是本质的改变。这叫“秀”，这是因为失去感感到非常痛苦，为了挽回而刻意营造的人设。

那为什么说是“做秀”呢？因为你是为了复合来设定人物特点，但本质的你并不是这样的。为什么很多人和好之后还是会分手？因为在以后的交往过程中，因为一点点小事，你的人设就崩塌了。你说这样的改变有什么意义呢？

本质的改变，是你自己真的很想改变。不管这个人回不回来，我都知道这个问题是我身上一个非常大、非常大的问题，我必须要改掉。我是为了让自己更好，让我自己更加分。这时候所呈现出来的所有状态，都是非常真实的。这不是立人设、做秀。

我可以告诉你：就算你跟那个人删除了所有的联系方式，也见不到面了，那么爱你的人一定会在一个你不知道的角落里，默默地关注你的一举一动。只要你改变了，他都是可以察觉到的。真的不要刻意去改变，因为在你见到这个人的时候，你的举手投足、一举一动都会被对方识破。而你真正改变之后，自然流露出来的状态，别人也是可以察觉到的。真的不需要你去刻意地突出你哪方面的改变——太刻意的东西反而很假。

那顺其自然，放平心态。我也真心希望你的改变是因为自己愿意改变，而不是为了某个人的无奈之举。

---

本节课中，我们一起学习了情感修复中关于“改变”的核心要点。真正的改变是源于内心的、为了自我提升的本质转变，而不是为了挽回而刻意表演。当你真正发生改变时，即使没有直接联系，对方也能察觉到。关键在于**沉定自己，顺其自然**。

记住公式：
```
真正的改变 = 自我驱动的内在提升 ≠ 为复合而设的短期人设
```
