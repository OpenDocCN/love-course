# 挽回教程 01：如何理解前任的“绝情”与“冷漠” 🔍

在本节课中，我们将要学习如何正确解读前任在分手后表现出的“绝情”态度。许多挽回者会因此感到绝望，但我们将分析其背后的心理机制，并理解为何“绝情”有时比“冷漠”更容易应对。

## 核心观点：绝情比冷漠更容易复合

在绝大多数情况下，绝情都要比冷漠客气更容易复合。很多人在挽回中，面对前任的绝情态度，什么都没做就已经开始丧失信心。但你们真的不必因此而绝望，这不过是**没有找到对的方法而已**。

## 为何人们认为绝情难以挽回？

为什么大部分的人都认为绝情的前任比较难以挽回呢？其实无非就是大家都按照固有的思维去考虑你们之间的问题。在你们的想法当中，对于感情的固定印象是**减分制**的。

在这种固有思维当中，你会觉得他的绝情是偏向于理智，是深思熟虑之后做出的结果。

## 情感抉择的本质是感性的

但事实真的是这样吗？其实大多数感情方面的重要抉择，几乎都是偏于感性的。也许理智上觉得分得可惜，但感性上就是想分。

所以，在你们分手以后，他绝情的表现，比方说拉黑、删除、逃避和你见面、对朋友家人不愿意提起你和他之间的过往等等，这都是常规操作而已。这也恰恰表现出了，在他心里，他依旧在意这段感情，**害怕被你影响**。

## 你需要的是耐心与理解

所以这个时候，你更需要有一点耐心对待这个事情。很多人选择分手的原因，其实并不是没有爱，而是爱得太疲惫，所以才会用退出来寻求解脱。

你要理清这个东西。你想复合，最重要的就是**重新给予对方对这段感情的信心**。千万不要因为前任表面的绝情就退缩，就失去了一个重新在一起的机会。
