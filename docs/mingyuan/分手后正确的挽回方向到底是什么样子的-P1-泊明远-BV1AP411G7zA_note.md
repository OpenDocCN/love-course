# 分手后正确的挽回方向到底是什么样子的-P1-泊明远-BV1AP411G7zA_note

在本节课中，我们将学习分手后如何调整心态并采取正确的行动方向。课程内容基于个人经历与感悟，旨在帮助处于痛苦中的朋友找到自我提升和积极生活的路径。

---

你现在可能正在经历分手，感到迷茫和痛苦。作为一个经历过同样痛苦并最终获得幸福的人，我想与你分享一些话。

屏幕前这个充满活力、积极向上的我，或许就是你们未来的样子。所以不必羡慕，请认真听我说。

我知道你此刻正处在痛苦和崩溃的状态中，可能因失恋而颓废、抑郁，甚至产生消极的想法。你当然可以允许自己有几天的伤心难过时间，但千万不可以颓废太久。

上一节我们谈到了允许自己短暂悲伤，本节中我们来看看具体应该怎么做。以下是行动步骤：

1.  立刻去洗一个热水澡。
2.  挑选出你最好看的衣服。
3.  认真护肤，好好打扮自己。
4.  出门转一转。
5.  多拍几张好看的照片。

完成上述行动后，拿出一张纸，进行以下规划：

1.  写下你想成为的样子。
2.  写下你想要达成的目标。
3.  写出达成这些目标的具体计划。

同时，你需要做到以下几点：

1.  拒绝向所有亲朋好友反复控诉。
2.  给自己一定的空间去享受孤独。
3.  不要反复思考对方为何无缝衔接、为何不要你、为何分手。

这是他的事情，他的人生由他自己决定。而你的人生，是由你自己走出来的。

如果想超越对方，就必须不成为对方。给自己半年或一年的时间，将全部精力投放在自己身上。因为将来你会觉得曾经的自己多么可笑，也会感慨该难过的人一定不是你。

然后，专注于好好工作，每天都开开心心的。你必须明白一个道理：**没有人会喜欢整天愁眉苦脸、以泪洗面的人**。

所以，让自己尽可能地变好、变优秀，这才是最不吃亏的，也是最终最正确的挽回方向。

你明白了吗？
