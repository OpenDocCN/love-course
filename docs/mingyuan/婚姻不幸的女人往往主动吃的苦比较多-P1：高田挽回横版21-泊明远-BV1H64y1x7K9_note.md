# 婚姻经营课 P1：识别自我消耗模式 🧐

在本节课中，我们将要学习识别一种在婚姻中可能导致不幸的自我行为模式。许多女性在关系中感到不幸福，并非完全由外部因素造成，其自身主动选择承担的“苦”往往多于常人。通过分析以下三种常见特质，我们可以更清晰地审视自己的行为模式。

上一节我们介绍了课程主题，本节中我们来看看具体的三种自我消耗特质。

以下是三种常见的、可能导致婚姻中过度付出而自身感受不佳的行为模式：

1.  **过度利他**：你非常善良，凡事优先考虑他人感受与需求，却常常忽略或压抑自己的真实想法与需要。
2.  **缺乏边界**：你心肠很软，即使自己感到为难或不情愿，也倾向于接受对方的要求，不愿拒绝以免“为难”别人。
3.  **包揽一切**：你非常勤快，家庭内外的大事小事都任劳任怨、亲力亲为，承担了绝大部分责任与劳动。

这些特质本身是美德，但若失去平衡，其核心运行机制可以概括为一个**公式**：

`单向过度付出 + 自我需求压抑 = 关系失衡与内心耗竭`

当这些行为长期持续且得不到对等的关怀与回报时，付出者容易产生委屈、疲惫和被忽视的感觉。你可能会困惑：为何勤恳、善良、体贴反而换来了不幸的感受？

本节课中我们一起学习了在婚姻关系中三种可能导致个人幸福感降低的主动付出模式。认识到这些模式是调整的第一步。如果你发现自己正面临类似的困扰，可以寻求进一步的探讨与分析。
