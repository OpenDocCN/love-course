# 分手后保持联系能复合吗？💔 - 课程01

在本节课中，我们将探讨分手后与前任保持联系这一行为对复合可能性的影响。我们将分析其背后的原理，并提供两个关键的操作要点，帮助你理解在何种情况下保持联系有利于复合，以及如何具体推进。

---

上一段我们提到了保持联系对复合可能有利，但这并非无条件成立。接下来，我们来看看必须注意的两个核心要点。

**第一要点：价值提升与情绪供给**

如果对方在聊天中回避情感话题，这表明你目前的价值不足以让他考虑回头。对他而言，维持现状的压力仍然大于重新在一起的快乐。

因此，你的核心任务是：**提升自身价值**并**持续展示**给对方。同时，你需要**持续性地提供正向的情绪价值**。

**第二要点：制造可控的失落感**

在完成价值提升和展示后，你需要策略性地制造对方的失落感。例如，当对方主动分享生活并询问你的状态时，你可以利用此机会。

以下是具体操作方法，旨在营造一种“你正在逐渐离开”的**状态**，而非真正这样做：

*   **调整回复频率**：将之前的1:1回复比例，改为1:2。即对方发送两条消息后，你再回复一条。
*   **展示精彩生活**：在社交平台（如朋友圈）发布你丰富、积极的生活动态。

这些操作的目的是让他感觉到你似乎正在离开，但只要他稍作努力，就有可能重新抓住你。

---

本节课我们一起学习了分手后保持联系的策略。我们了解到，保持联系本身是复合的有利条件，但关键在于后续的两个步骤：首先是通过提升和展示价值来增加吸引力；其次是巧妙制造失落感，以激发对方的主动意愿。请注意，以上方法仅为参考框架，具体操作需根据你们的实际情况灵活调整。
