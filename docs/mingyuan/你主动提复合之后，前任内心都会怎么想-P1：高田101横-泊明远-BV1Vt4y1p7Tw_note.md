# 你主动提复合之后，前任内心都会怎么想-P1：高田101横-泊明远-BV1Vt4y1p7Tw_note

在本节课中，我们将要学习当你主动提出复合后，前任内心的真实想法。理解这些心理活动，是避免错误行为、制定有效修复策略的第一步。

大部分人被分手后，在分手后都会选择去不断的纠缠，骚扰前任。他们会各种表忠心，例如说“我爱你”、“我会改的”、“我离不开你”以及“再给我一次机会吧”。但是你会发现，其实不管你说什么，你的前任就是不想搭理你。

对方甚至连个好脸色都没有给你。作为一个手上有着上千份情感案例的博主，今天我就跟你们说一些实话：为什么对方不愿意搭理你，他的真实心理是什么样的。如果你也遇到类似的情感问题，可以直接私信找我聊一聊，有问必答。

上一节我们提到了前任的抗拒态度，本节中我们来看看，当你做出某些具体行为时，对方内心究竟在想什么。

以下是几种常见挽回行为及其引发的前任内心反应：

*   **行为：** 你跟对方反复的认错，保证不会再犯，你一定会改。
    *   **对方内心想法：** “要改早就改了，何必等到现在呢。”

*   **行为：** 你绞尽脑汁地写了一篇深情小作文，企图打动他。
    *   **对方内心想法：** “又来这一招烦不烦啊，早知今日何必当初呢。”

*   **行为：** 当你鼓起勇气想要找他的时候。
    *   **对方内心想法：** “都分手了，见面还有什么意思呢，已经没有关系了，别骚扰我行不行。”

他始终不愿意跟你复合的原因，总结起来就是：**他觉得你这个人不可能改变，就算复合了也会重蹈覆辙，彼此浪费时间**。这就是我为什么总是跟你们强调，如果你想要成功复合，就一定要提升自己经营感情的能力。

你们都相处这么多年了，你是什么样的他都再清楚不过。你只有让他有眼前一亮的感觉，完全颠覆了他对你的印象，那你才有可能成功。

如果现在你比较被动，他对你也比较绝情，电话不接信息不回，那你就别再胡思乱想、纠缠他了。给彼此一个冷静的空间，也给自己一个自我提升和改变的空间。这个阶段就是我们常说的 **“冷静考察期”**。

如果在冷静考察期内，你做得很好，各方面都想通了，并且你自己也有改变和提升，那和好的机会就一定会变得非常非常大。

本节课中我们一起学习了前任在收到复合请求后的典型心理活动，其核心在于对方对“改变”的怀疑。因此，成功的修复不在于言语的保证，而在于通过实际行动和真正的自我提升，来打破对方的固有认知，重建信任与吸引力。
