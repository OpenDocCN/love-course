# 分手后断联的四种状态解析 🧠 - 课程01

在本节课中，我们将要学习分手后，前任在断联期间通常会经历的四个心理阶段。理解这些阶段，有助于你把握时机，更理性地处理情感关系。

## 概述：情绪波动与习惯养成

分手后，双方普遍会产生情绪波动。你的情绪波动可能表现为渴望和好、希望建立联系。而对方则会进入一个心理变化周期。我们常听说“21天养成一个习惯”，同样，大约也需要21天时间，前任对你留下的负面印象和经历才会逐渐淡化。

上一节我们提到了情绪波动的基础，本节中我们来看看对方心理变化的具体阶段。

## 断联的四个心理阶段

对方在分手后会经历接受期、厌恶期、怀念期和决定期这四个阶段。

### 第一阶段：接受期

此阶段的核心是对方在心理上接受“我们已经分手”这个事实。这是他进行自我说服、初步接纳分手形式的时期。

### 第二阶段：厌恶期

在此阶段，对方可能会对你产生厌恶感。分手的原因往往源于感到讨厌、不喜欢或相处不舒服。只有积累了足够的负面感受，他才会最终决定分手。

### 第三阶段：怀念期

当21天效应开始发挥作用，对方从厌恶期中走出来后，便会进入怀念期。此时，他脑海中对过往美好相处的回忆会逐渐变得清晰。

### 第四阶段：决定期

这是决定是否要与你重新建立联系的阶段。对我们而言，这也是**最佳复联时机**，是开始展现自身吸引力的关键阶段。

## 总结与行动指南

本节课中我们一起学习了分手后前任心理变化的四个阶段：接受期、厌恶期、怀念期和决定期。理解这些阶段能帮助你避免在错误的时间采取行动。
