## 核心心态分析

上一段我们提到了本节课的主题，本节中我们来看看具体的心态构成。男生一边拒绝你的和好请求，一边又在和你联系，他到底在想什么呢？

其实这种情况是特别特别常见的。他不和你和好，但是又愿意和你聊天，有时候还会挺主动的。原因是他内心依然爱你，但是在一起他又觉得累。

有时候又特别烦，他还没有说服自己，去做最终的选择和决定。虽然两个人名义上是分手，但其实还是情侣关系。

他心里想的最多的就是：**“要是你的脾气能改一改就好了”**。这样他就能更好地说服自己回头。

## 关键行为与潜在风险

理解了核心心态后，我们来看看如果你的应对方式不同，会导致何种走向。

以下是两种可能的行为路径及其结果：

*   **路径一：积极改变**
    *   如果你展现出积极的改变，让他看到关系改善的希望，他便有可能说服自己，重新考虑复合。

*   **路径二：重复旧模式**
    *   如果你依然和之前一样，无休止地阻挠，让他看不到一丁点的希望，他就会说服自己，慢慢疏远，然后放下。

## 总结与帮助

本节课中我们一起学习了“拒绝复合但保持联系”这种矛盾行为背后的心理。其核心公式可以概括为：**“残余爱意 + 对过往矛盾的疲惫 = 观望态度”**。对方正处于一个需要看到“改变”与“希望”来说服自己的决策期。
