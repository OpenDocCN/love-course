# 💔 分手后沟通指南 - 课程 P1：高田横版 127

在本节课中，我们将要学习分手后是否应该与前任保持联系。我们将分析发信息的潜在后果，并提供清晰的决策思路。

很多人都会问，分手后到底该不该和对方发信息。你发也可以，不发也可以。但是有一件事你要明白，就是对面那个人已经不是你回忆里的那个人。你们已经分手了。

大家都是成年人，也都不傻。你在想什么，对方大概也能猜得到。对方想什么，你也能猜得到。

## 🤔 发信息前的关键考量

上一节我们提到了双方都能猜到彼此的心思，本节中我们来看看发信息的具体影响。你想发信息的时候就可以发，但是你要知道这个信息发了之后会有什么样的后果。

以下是两种主要后果：
*   要么就是他难受。
*   要么就是你难受。

如果大家都不会难受，你爱发什么发什么。但是如果说至少有一方难受，那就要权衡利弊去想一想。

## ⚖️ 如何做出决定

思考清楚后，你需要做出选择。以下是两个可行的方向：
*   要么就要求自己善良一点。
*   要么就祈祷对方善良一点。

偶尔一次失态没有关系。它可以为我们平凡而普通的生活增加一点花絮和插曲。不可能谁都活得像劳动模范一样，而大家也都可以放松一点。
