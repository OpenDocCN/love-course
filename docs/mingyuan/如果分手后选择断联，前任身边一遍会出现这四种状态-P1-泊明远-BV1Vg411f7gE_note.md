# 挽回心理学课程 01：分手断联后前任的四种心理状态 🧠

在本节课中，我们将要学习分手后选择断联，你的前任内心通常会经历的四个心理阶段。理解这些状态，能帮助你更好地把握挽回的时机和策略。

## 课程概述

分手后，双方都会经历情绪波动。你的情绪波动可能倾向于和好并希望建立联系，而对方则会进入一个相对规律的心理变化周期。这个周期大致会持续21天左右，与“21天养成一个习惯”的规律类似，前任对你的负面情绪会在此期间逐步消退。

## 四种心理状态详解

上一节我们概述了分手后的情绪周期，本节中我们来详细看看前任会经历的四个具体阶段。

以下是分手断联后，前任身边通常会出现的四种心理状态：

1.  **接受期**
    此阶段的核心是接受分手这个事实。对方会首先进行自我调整，并开始对你们的关系产生负面认知。

2.  **厌恶期**
    基于接受期产生的负面认知，对方会开始对你本人产生厌恶感。只有感到讨厌、不舒服或不喜欢，其分手的决定才会被强化。因此，厌恶期是一个必然过程。

3.  **怀念期**
    这就是前面提到的“21天效应”开始发挥作用的阶段。对方会逐渐从厌恶期中走出来，进入怀念期。此时，其脑海中关于你们在一起的美好回忆会变得越来越清晰。

4.  **决定期**
    当怀念产生后，便进入了决定期。对方会开始考虑是否要与你重新建立联系。**这个阶段被认为是挽回中最佳的复联时机**，也是你重新展示自身吸引力的第一个关键开端。

## 课程总结

本节课中，我们一起学习了分手断联后前任心理变化的四个阶段：接受期、厌恶期、怀念期和决定期。理解这个周期，特别是抓住“决定期”这个**最佳复联时期**，对于有效展开挽回行动至关重要。
