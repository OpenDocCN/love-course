# 太爱他忍不住去联系他该怎么办-P1：高田横版182-泊明远-BV13u411o73z_note

在本节课中，我们将要学习一个在情感关系中常见的难题：当你非常思念对方、忍不住想要主动联系时，应该如何应对。我们将通过一个具体的心理场景分析，理解“需求感”与“能量供给”的相互作用，并找到有效管理自身冲动的策略。

很多来访者曾向我倾诉：“老师，我就是忍不住想去联系他，该怎么办？”现在，请你闭上眼睛，和我一同想象一个场景。

假设我要求你十天之内不要主动联系他。你坚持到了第八天或第九天，感觉快要忍不住了，决定必须给他发一条消息。然而，就在你打开聊天窗口准备发送时，你发现他刚刚给你发来了一条“在吗？”。

这时，你大概率就不会再想发送你原本准备好的消息了。因为你的“需求感”瞬间得到了满足或转移，于是你又能轻松地再多坚持十天。这条他主动发来的消息，为你提供了继续等待的“心理能量”。

反之，这个逻辑同样成立。如果你选择主动去联系他，那么除了为他提供“心理能量”、满足他的被需求感之外，这个行为对你自身没有任何益处。你的主动联系可能恰好让他获得了所需的确认感，从而延缓了他主动联系你的动力。

或许，你只需要再坚持一天，他就会主动给你发消息。关键在于，谁先按捺不住这种“需求感”，谁就可能在心理博弈中暂时处于供给能量的位置。

**核心概念公式**：
```
你的主动联系 = 为他提供心理能量
他的主动联系 = 为你提供心理能量
博弈关键在于：管理并降低自身即刻的“需求感”
```
