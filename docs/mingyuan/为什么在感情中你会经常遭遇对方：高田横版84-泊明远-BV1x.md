# 为什么在感情中你会经常遭遇对方：高田横版84-泊明远-BV1x

在本节课中，我们将要学习情感关系中冷暴力的形成机制与核心原因。我们将分析为何一方会反复使用删除、冷战等策略，以及这背后反映出的关系权力动态。

## 关系初期的试探与底线突破

两个人在一起，如果出现矛盾，对方可能会倾向于使用删除联系方式甚至冷战的方式来对待你。你知道为什么会这样吗？因为这一招对你屡试不爽。

关系刚开始时，双方通常互相尊重。他不敢轻易这样对你，是因为他对你还不够了解。

所以不敢轻易地去“拿捏”你。但随着接触越来越多，他开始摸清楚你的软肋和性格。于是，他会一点一点地去突破你的底线。

## 策略的有效性与权力转移

每一次的试探，如果你都表现出妥协和服从，那么下一次他就会加大力度。

直到你彻底失去话语权和掌控权。当对方掌握了话语权和掌控权时，你就只有听话，才能够留在这段关系当中。

而他之所以能有这些行为，很大程度上并非因为他有多大的能力和才华，而是因为他很清楚：**你离不开他，特别需要他**。你为了不失去他，就会甘愿放低自己的标准，降低自己的底线。

## 行为模式的固化与关系结局

在这个时候，其实你早就已经失去了自我。人在很多时候是自私的。当他发现一个策略对你非常有效，那么他就会反复使用这种策略，直到养成习惯。

所以，当你发现关系中有一方牢牢掌握着话语权，并且对你忽冷忽热时，最终的结果通常有两种：
*   你撑不住，分手了。
*   你一个人在这里苦苦维持。
