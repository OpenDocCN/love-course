被分手后如何掌握主动权？如何让对方主动联系你？很多人认为这不可能，因为分手是对方提出的，对方怎会主动？但凡这样想的人，在这段关系中基本处于被动地位。

面对分手这件事也是如此，许多人觉得自己是被抛弃的一方，感到痛苦难受。但这并不代表你输了，你完全可以将对方从心理高位上拉下来。具体该怎么做？其实很简单。

核心是让对方比你更放不下，并从你这里感受到挫败感。首先，我们需要给对方制造心理落差。提出分手的人通常自认为处于上风，他预期分手后你会情绪低落、哭泣纠缠，工作生活一团糟。

他甚至会不时偷看你的动态，寻找你消极痛苦的证据。这种心理源于他的优越感。如果你真的按他的预期行事，就正中他的下怀，无形中让他觉得自己非常优秀。

能够左右另一个人的情绪和思想，会让人感到骄傲。他会因此更加自信，认定你离不开他。所以请记住，即使你内心确实痛苦，表面上也必须表现得云淡风轻。

每天正常饮食、社交、聚会、健身，在社交平台展示积极向上的生活状态。你需要让他知道，分手这件事对你几乎没有影响。此时，琢磨和揣测的人就会变成他。

他会开始怀疑你是否真的放下了，是否已开始接触新的人。关键在于牵动对方的情绪，让他来猜测你的状态。其次，我们需要重新找回自身价值。所有分手都有原因，至少说明在这段感情中，你存在让对方不满意的地方，或你已不具备他需要的价值。

此时，我们不仅要重新找回自我价值，更要让他觉得失去你是他最大的损失。这也是你需要自我反思之处：生活习惯是否不佳？社交圈是否狭窄？工作与人生是否缺乏规划？

如果你自己能意识到这些问题，你是否需要去提升？你所有的不甘与痛苦，往往源于觉得对方过得比你好。此时若一味抱怨、纠缠、渴望对方心软，只会让你显得更廉价和软弱。

你必须变得更好、更优秀。要让他清醒地意识到，错过你非常可惜，让他后悔、焦急却得不到你。只有这样，主动权才能回到你手中。请记住，掌握一个人不是靠言语打压，而是通过实际行动让他觉得，当初是他自己看走了眼、做错了决定。
