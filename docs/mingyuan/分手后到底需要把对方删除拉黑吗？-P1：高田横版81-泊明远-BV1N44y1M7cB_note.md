## 观点一：若想复合，不应删除

在各种讨论中常有一种说法：如果仍有复合的想法，便不应删除对方。这样做可以避免日后因后悔而显得矛盾。

## 观点二：删除源于害怕失控

你们选择删除，往往是因为害怕自己控制不住情绪，再次卑微地去求和。这背后是自尊心在起作用。

因此，你需要学习一项关键技能：**狠下心来控制自己**，以保护自尊心。这正是练习自控力的好机会。

**核心公式：自控力 = 驾驭自身情绪与行为的能力**

无论你是借酒消愁还是内心痛苦，练习自控力都至关重要。一个人如果连自己的冲动都驾驭不了，又如何掌控人生呢？

所以，一切要从**控制内心**开始。

## 观点三：删除并非有效隐藏

上一节我们提到了控制内心的重要性，本节中我们来看看删除行为的实际效果。你删除对方，以为对方不知道你的意图吗？结果往往是你偷偷去翻看对方的微博、各种社交平台，躲躲藏藏，这并无意义。

不如将其微信留在好友列表，但选择**屏蔽他的朋友圈**。这样，短期内你不会看到他发布的任何内容。

**操作代码示例（心理行为描述）：**
```
if 想查看对方近况：
    点开其朋友圈 → 了解动态
else：
    保持屏蔽状态 → 避免日常干扰
```

你无需在意别人怎么想。当你想了解他近况时，可以主动点开查看；平时则保持屏蔽。直到你彻底走出这段感情。

届时，再重新开放其朋友圈即可。

## 观点四：成熟的人凡事留有余地

以下是成熟处事的原则：做人留一线。此刻你们不合适，不代表未来永远不会合适。

当我们都成长为更成熟的人，回首往事时，或许仍会觉得当初那个人最好。若那时他未娶、你未嫁，仍有可能在一起。

因此，做事不要太绝。成熟的人总是懂得留有余地。
