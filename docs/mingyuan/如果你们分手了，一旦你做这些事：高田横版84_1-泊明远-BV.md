# 如果你们分手了，一旦你做这些事：高田横版84_1-泊明远-BV

在本节课中，我们将要学习分手后应避免的几种行为。这些行为不仅无助于关系的修复，反而可能降低复合的可能性，并让你在未来感到后悔。我们将逐一分析这些行为的本质及其负面影响。

## 🚫 禁忌一：过度讨好

上一节我们介绍了课程概述，本节中我们来看看第一种禁忌行为。

分手后对他越好，他就会越觉得你们两人不合适。其本质在于，他的内心会认为是你配不上他，所以你才需要不停地讨好。

**核心逻辑**：`过度付出 -> 被解读为价值不对等 -> 强化对方“你配不上”的认知`

## 🤝 禁忌二：大包大揽与空头承诺

了解了过度讨好的问题后，我们接下来分析另一种常见错误。

首先，你做出的承诺基本上无法完成。此类案例极为常见。其次，他也只会认为你这是分手后的一时冲动，根本不会相信你。

**行为模式**：
以下是“大包大揽”的典型表现：
1.  承认所有错误，包括非自身责任的问题。
2.  做出过度且不切实际的承诺（例如，“我一定会改掉所有缺点”）。

## 😔 禁忌三：过度自我否定

除了对外在关系采取错误行动，对内的心态调整也至关重要，过度反思便是陷阱之一。

分手后你会反思自己。然而，越反思，你就可能越看自己不顺眼。越看自己不顺眼，你就会越没信心，从而导致复合的几率越来越小。

**心态变化链**：`反思 -> 过度自我否定 -> 信心丧失 -> 复合概率降低`

但实际上，情况可能并没有那么糟糕。问题可能仅仅在于相处和处事的方式欠妥当，而你的初心是好的。因此，分手后，不应长他人志气，灭自己威风。保持信心总是有益的。

---

**总结**：本节课中我们一起学习了分手后应避免的三个主要行为：**过度讨好**、**大包大揽与空头承诺**以及**过度自我否定**。这些行为会损害你的个人价值感，削弱对方复合的意愿，并打击你的自信心。避免这些陷阱是走向理性处理分手的第一步。
