# 揭秘假名媛的生活-P1：1646987552682.m

在本节课中，我们将要学习一种特定的社会现象：那些看似无需工作、生活光鲜亮丽的“假名媛”群体的日常生活模式，并深入揭秘她们可能的收入来源。我们将通过分析其行为模式、社交策略和盈利手段，来理解这一现象背后的逻辑。

---

你是否好奇，那些每天睡到下午一两点才起床，随后开始健身、逛街、享用下午茶，晚上则打卡夜店的女性，她们的收入究竟从何而来？这是一个粉丝向我提出的问题。

这位粉丝描述，他认识了一位比较特别的女生。我询问特别之处在哪里。他表示自己从未见过这样的女性。我让他将这位女生的朋友圈内容发给我查看。我看到照片中的女生身高约175厘米，拥有大长腿，外貌出众。我于是问他，所谓的“特别”是否指特别好看和特别高。

他表示，自己好奇的是这位女生每天下午一点多才起床，接着便开始健身、逛街、与姐妹下午茶，晚上则去夜店打卡，并且每天都发布朋友圈。他们是在一次聚会上相识并添加了微信的，之后聊天不多。但他通过天天观看她的朋友圈，发现这位女生似乎不需要工作。他确定她并非富二代，因此对她的收入来源感到困惑。

实际上，我的微信好友列表里就有许多这样的女性。她们的共同点是朋友圈里充满了各种展现身材的精致照片，例如穿着包臀裙、露出大长腿，能够驾驭各种服装风格，并在各种酒店、三亚等地打卡。

今天，我将为你揭秘这一现象。我与许多这类女性聊过，询问她们的职业。大部分人的回答可归纳为三类：**某公司股东**、**某品牌设计师**，或是**做自媒体的**。她们看起来年纪不大，却声称自己在澳大利亚或美国留学，无论原本专业是什么，对外统一宣称是金融或MBA（工商管理硕士）。

与这类女性聊天，关键并不在于你聊什么话题，而主要取决于你朋友圈所展示的内容，是否与她朋友圈的内容相匹配。她们的目标客户是事业有成的中年男性或暴发户。如果你的朋友圈没有展示相应的财富或生活层次，那么基本无需与她们深入交流。真正的富二代或创一代了解她们的套路，且她们一个比一个精明，通常不会中招。

因此，这些女生在接近潜在客户时，一定会精心伪造出气质修养和学识极高的形象。一般来说，她们赚钱的途径主要有三种。

以下是三种常见的盈利模式：

**第一种，最低端的套路：以庆生为由进行邀约。**
她们会以过生日为理由，邀请你到她的卡座稍坐。最夸张的情况是三天过一次“生日”。这时，转账和名牌包等礼物就可能到手。等到酒过三巡，她们可能会搂着你说想要开酒。只要你的卡一刷，第二天她的销售提成就能到手。收到的名牌包和奢侈品，可以转手在闲鱼或二手奢侈品店卖掉，再用这笔钱购买仿品，外人难以分辨。你有时看到她们频繁去各地旅游打卡，并非为了放松，而是为了更换城市，尤其是在北上广等经济发达地区，重新寻找潜在客户，重复上述操作。

**第二种，利用直播平台赚取零花钱。**
她们的互联网思维通常很好，会通过直播来赚取一些零花钱，以满足日常开销。能遇到真正“大佬”的机会很少，只能偶尔碰运气。你给她刷礼物，她提供的是陪伴与暧昧的聊天氛围，与她在聊天室里高冷、等待“良人”的形象形成反差。

**第三种，进行资源整合。**
例如，A大佬有客户资源，B大佬有技术资源，她们就在中间做资源整合，或者成立新公司，从中赚取差价。说实话，这种人我还挺佩服的，她们很有商业头脑。在圈子里，这样的人凤毛麟角，但通常最后能真正混成事业有成的女老板。

前几年医美行业刚兴起时，很多早期整容的女性就利用了这种方法。一是认识大佬，二是了解医美行业的暴利。接着，她们混熟一位整容医生，与大佬合计，由大佬出钱开一家整容医院，她们自己占20%左右的股份，再给拉来的整容医生一些技术股，让医生再找来几位同行，她们就可以躺着赚钱了。

那么，有的女生没有找到大佬给她投资开医院怎么办？没关系。她们会拉上一群圈子里的闺蜜，大家每人出十万、五万，凑上两三百万就把医院开起来了。这是真实发生的事情。一位从事医美的女生告诉我，她们一家起步资金两百万的医院，背后有四十多个股东。在早几年，这行特别赚钱。真的就是二三十个女性，凑钱开一家医院，挖来两三个整容医生，找十几个护士。然后，这几十个股东既是投资人，又是业务员。她们去各种酒吧、夜店结识年轻女孩，和她们交朋友，以自身的整容经历作为案例，说服她们进行整容。

如果这些年轻女孩没有钱怎么办？她们会推荐各种贷款渠道。如果贷款都还不上怎么办？她们会说：你都整容变好看了，不去谈恋爱让男朋友帮你还吗？如果你不知道如何找到有钱的男人，她们会表示：我们来教你啊。于是，就衍生出了“上海名媛”几十人拼单酒店、跑车、甚至丝袜的现象。

顺便提一句，那种“男版名媛”是怎么来的呢？是跟女人学的。因为女人拼酒店、跑车是为了找到有钱的男人供养她们；而男人拼酒店、跑车，则是为了接近这些寻找有钱男人的女性。用一句话概括，这是一场“记忆游戏”。这种男人，在不了解内情的人口中可能被称为“情人”（QR），但在圈内女性口中，则被称为“杀猪盘”。

作为一名“优秀”的“混圈女”，最重要的一课就是学会识别谁是真正的富二代。我之前与这类女生聊得较多，并将相关识别技巧整理在了聊天记录解析中。

上面所说的这些，目的还是让大家远离这种女生和这种生活方式。没有相应的资本和手腕，就不要轻易涉足这个圈子。脚踏实地工作，过好每一天才是正道。生活中，这样的女生其实也是少数。
