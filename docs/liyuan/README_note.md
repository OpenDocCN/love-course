# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述

程序的核心是处理数据。为了处理数据，我们需要将数据存储在计算机的内存中，并用一个名字来标记它，这个名字就是**变量**。同时，数据有不同的种类，例如数字和文本，这些种类被称为**数据类型**。掌握如何创建变量并了解不同的数据类型，是开启编程之旅的关键。

## 1. 什么是变量？ 📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（内存空间）里可以存放东西（数据），而标签（变量名）则帮助我们找到并使用这个盒子里的东西。

在Python中，创建一个变量并为其赋值非常简单，只需要使用等号 `=`。

```python
my_name = "李远"
```

这行代码创建了一个名为 `my_name` 的变量，并将文本“李远”存储在其中。现在，只要我们使用 `my_name`，Python就知道我们指的是“李远”这个值。

上一节我们介绍了变量的比喻，本节中我们来看看使用变量时需要遵守的几个核心规则。

以下是定义变量时需要遵循的主要规则：
*   **变量名只能包含**字母、数字和下划线（`_`）。
*   **变量名不能以数字开头**。
*   **变量名是区分大小写的**。例如，`age`、`Age` 和 `AGE` 是三个不同的变量。
*   **应避免使用Python的保留字**（如 `if`、`for`、`while`）作为变量名。

## 2. 基本数据类型 🔢

变量可以存放各种类型的数据。Python有一些内置的基本数据类型，最常用的有三种。

### 2.1 字符串（String）

字符串是用于表示文本的数据类型。在Python中，你需要用引号（单引号 `'` 或双引号 `"`）将文本括起来。

```python
greeting = "你好，世界！"
author = '李远'
```

### 2.2 整数（Integer）和浮点数（Float）

数字主要分为两类：整数和浮点数。
*   **整数**是不带小数点的数，可以是正数、负数或零。
    ```python
    age = 25
    temperature = -10
    ```
*   **浮点数**是带有小数点的数。
    ```python
    price = 19.99
    pi = 3.14159
    ```

### 2.3 布尔值（Boolean）

布尔值只有两个：`True`（真）和 `False`（假）。它们通常用于逻辑判断。

```python
is_student = True
is_raining = False
```

## 3. 检查数据类型 🔍

有时你可能不确定一个变量是什么类型。可以使用 `type()` 函数来查看。

```python
print(type(my_name))   # 输出：<class 'str'>
print(type(age))       # 输出：<class 'int'>
print(type(price))     # 输出：<class 'float'>
print(type(is_student))# 输出：<class 'bool'>
```

`<class ‘str’>` 表示“字符串类”，即该变量是一个字符串。

## 4. 一个简单的实践示例 💻

让我们把以上知识结合起来，写一个简单的程序。

```python
# 定义几个变量
name = "李远"
birth_year = 1990
current_year = 2023
height = 1.75
likes_python = True

# 计算年龄
age = current_year - birth_year

# 打印信息
print("姓名：", name)
print("年龄：", age)
print("身高：", height, "米")
print("喜欢Python吗？", likes_python)
```

运行这段代码，它会根据出生年份计算出年龄，并将所有信息打印到屏幕上。

## 总结

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。
我们了解到变量是数据的命名容器，并通过 `=` 来赋值。
我们认识了四种最基本的数据类型：用于文本的**字符串（str）**，用于整数的**整数（int）**，用于小数的**浮点数（float）**，以及用于逻辑判断的**布尔值（bool）**。
最后，我们还掌握了使用 `type()` 函数来检查变量类型的方法。
理解这些概念是后续学习更复杂的操作（如运算、条件判断和循环）的坚实基础。
