在本节课中，我们将要学习一个关于人类行为根本动机的观点。我们将探讨生存与繁殖如何构成人生的核心目标，并理解学习恋爱技巧在这一框架下的意义。

## 生存是第一目标 🛡️

上一段提到了人生的终极目标，本节中我们来看看第一个目标：生存。

这是一本叫人把灭的书籍，但里面讲了一个让我们永生的方法。这里写到人生的终极目标，首先我们的第一目标是生存。不能生存就没有生命。

这里他讲的比较书面，我按照我的理解给大家去讲。首先我们都想要活下去，我们都有很强的求生本能。所以我们的第一目标就是生存。

**核心公式**：`人生首要目标 = 生存`

## 繁殖是第二目标 🔄

然而，仅仅生存是不够的。我们也明白，我们不可能永远的存在，不管以什么样的方式。总有一天我们会在这个世界上消失。

所以，第二目标就是繁殖。通过繁殖我们有了下一代。有一天你没了，你的孩子会把你的基因一代一代的传下去。就是你不存在了，但是你的基因还在这个世界上。

所以繁殖也算是变相的一种永生。

**核心概念**：`繁殖 ≈ 基因的永续 = 变相的永生`

## 恋爱是实现目标的手段 💑

基于以上两个目标，作者得出了一个关于人生行动的结论。

所以作者认为我们的人生需要做两个事情：努力的活下去并且去爱。而且为了求爱成功，或者说找到一个更好的对象，我们应该学习怎么谈恋爱。

以下是实现“爱”与“繁殖”这一目标的具体行动思路：

*   **行动一**：努力活下去，保障生存。
*   **行动二**：学会去爱，建立亲密关系。
*   **行动三**：学习恋爱技巧，以提高求爱成功的概率，从而更好地实现繁殖这一永续目标。

## 总结 📝

本节课中我们一起学习了关于人生目标的递进式观点。我们认识到，**生存**是基础，而**繁殖**是基因层面实现永续生存的关键。在这一逻辑下，**学习恋爱**成为了一项重要的生存技能，它帮助我们更好地完成“去爱”与“繁殖”的人生任务，从而实现某种意义上的延续。
