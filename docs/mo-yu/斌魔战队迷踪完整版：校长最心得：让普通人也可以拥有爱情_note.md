## 恋爱与婚姻是两件事 🔄

外在条件决定两个人是否在一起，说白了就是看脸。内在条件则决定两个人能在一起多久。如果内在不合适，例如家庭背景或价值观不匹配，关系往后发展就会感到吃力。
