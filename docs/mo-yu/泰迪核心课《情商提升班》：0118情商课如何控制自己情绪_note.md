# 泰迪核心课《情商提升班》：0118情商课 如何控制自己情绪 🧠

在本节课中，我们将要学习情商的核心概念，并探讨如何在日常交流中更好地管理情绪、理解他人，从而建立更和谐的人际关系。课程将围绕两个关键点展开：如何让他人更易接受你的观点，以及如何敏锐地察觉并照顾他人的情绪。

## 情商的核心：两点关键认知

高情商主要体现在两个方面。

第一点是让你的观点更容易被别人接受。表达同一个意思或概念，不同的人会有不同的效果。

有些人说话的方式让人不爱听，甚至让人想立刻反驳。相反，有些人说话，即使观点与你冲突，你也会觉得舒服，愿意倾听。

第二点是能够理解他人的意思，懂得并照顾他人的情绪。这并非因为有求于人，而是在任何社交场合中，都能迅速判断出他人的敏感点，知道什么话会让人不舒服，什么话会让人舒服。

这一点其实很难做到。因为很多时候，你觉得无所谓的一句话，在别人听来可能非常刺耳，这就是“说者无意，听者有心”。

## 避免无意的伤害：从案例中学习

以下是几个常见的、因无心之言而伤害他人的例子。

**案例一：关于体重的玩笑**
我在大学时，曾以为男生不在乎体重，经常拿自己140斤的体重开玩笑。但我一位同样140多斤的同学听后脸色就不好看了。我本意是自嘲，却无意中戳中了他的敏感点。

**案例二：评价公众人物**
如果你说“李一峰的演技真差”，而对方恰好是他的粉丝，那么无论你的观点多有道理，对方都会感到不舒服。你只是在发表个人观点，却可能无意中伤害了一个群体。

这些情况在人际交往中很难完全避免，尤其是与不熟悉的人交往时。作为年轻人，我们既不能过于老成持重、沉默寡言，也不能口无遮拦。关键在于找到平衡点。

## 核心沟通原则：多分享，少评价

为了最大限度地避免因言语让人不开心，我总结了一个核心原则：**尽量分享，少评价，少下结论**。

在朋友间的聊天中（非工作场合），应多分享你的所见所闻，而不是急于用“我觉得”、“我认为”来给事情或人下定论。

最低情商的行为，就是将某一件或某一类事情进行笼统的概括和总结，这极易误伤他人。

**反面案例一：地域歧视**
例如，看完电影《十二公民》后说：“你看，在北京犯法的好多都是河南人。”这就是典型的地域歧视和以偏概全，极易引起在场河南人或其亲友的反感。

**反面案例二：群体标签**
再比如，“我觉得女生25、6岁还没谈恋爱不太正常。”这句话可能有一定道理，但打击面太大，很容易得罪人。你可以谈论某个具体朋友的例子，但千万不要对整个群体下结论。

**反面案例三：职业或爱好偏见**
“我觉得PUA都是臭流氓”或“做IT的都挺呆的”。这种把你见过的个别案例上升为整个群体的标签，同样会伤害很多人。

## 如何实践“分享”而非“评价”

那么，什么是“尽量分享”呢？就是多陈述事实，少发表个人论断。

因为人与人交流，往往不是纯粹的逻辑辩论，而是情感和观点的碰撞。你可以引用事例或他人的观点。

例如，你可以说：“我之前看到一条微博说，女孩子25岁之后找对象好像会难一些。”这只是你看到的一个现象，不代表你的观点。这样，在对方可能反感之前，你还有讨论和解释的空间，而不是直接否决了某种可能性。

记住：你分享的是观察，不是你的定论。

## 克服刻板印象与确认偏差

为什么我建议大家30岁之前“别太有底线”？这里的“底线”不是指道德底线，而是指过于强硬、狭隘的原则性。

例如，“玩QQ空间的人都特别low，我绝不和他们交朋友”或“追韩剧的妹子都很肤浅”。这不是有原则，而是情商低，源于你对世界认知的局限和刻板印象。

**什么是刻板印象？**
刻板印象是一种有偏差的认知。例如，如果小米手机的良品率是99.9%，但你的朋友恰好买到了那0.1%的问题机，你就可能对整个品牌产生“质量不好”的刻板印象。人们很容易因为个别负面事件，对整个事物产生难以改变的坏印象。

网络舆论也常利用这一点。要成为一个情商高、明辨是非的人，我们必须警惕这种思维，学会独立思考。

## 培养双面思考与换位思考能力

我作为辩手的经历教会我一个重要道理：**任何事情都有两面性**。

例如，讨论“医患矛盾错在医生还是患者”时，目的不是争出对错，而是练习站在双方的角度思考问题。只有理解双方的立场，才能看清问题的全貌。

人性有一个弱点，叫做“确认偏差”或“逆向合理化”：人们倾向于相信自己愿意相信的，并为此寻找支持证据。

例如，一个女生明知男友是渣男，但如果你指出这一点，她可能会生气。因为她需要维护自己的选择，会选择性回忆男友微小的优点，来证明自己的选择没错。

## 在沟通中应用：不争对错，管理情绪

基于以上分析，当你面对他人的质疑或相反观点时，请遵循以下步骤：

1.  **不要急于反驳**。先站在对方的角度思考，他为什么这么说？他的出发点是什么？
2.  **理解情绪而非争论事实**。人际交往中，尤其是亲密关系里，很少有绝对的大是大非，多的是一些情绪感受：“我觉得你不在乎我”、“我觉得你态度不好”。
3.  **顺着对方的思路沟通**。你不需要证明对方错、你对。只需要表示理解他的出发点，并针对他关心的点提出解决方案，事情往往就能平息。

例如，上司批评你工作不努力。你不要直接反驳，而是想：“他可能是因为我某件事没做好才这么认为。”然后你可以回应：“领导，您说的那件事我确实没安排好，我本周打算这样改进，您看行吗？”

**一个情绪管理小技巧：五秒法则**
当你特别想发火或反驳时，先在心里默数5秒（1…2…3…4…5）。这既能让你情绪稍微平复，也能让对方觉得你的回应是经过思考的，更容易被接受。吵架时，双方往往只在意态度，根本听不进对方的话。

## 课程总结与课后练习

本节课我们一起学习了如何通过管理情绪和改善沟通来提升情商。

**核心要点总结：**
*   高情商体现在：**让他人易于接受你的观点** 和 **敏锐察觉并照顾他人情绪**。
*   核心沟通原则：**多分享事实，少做评价和下定论**，避免以偏概全伤害他人。
*   关键思维方法：**警惕刻板印象和确认偏差**，培养**双面思考**和**换位思考**的能力。
*   实践行动指南：在冲突中**不争对错，先理解对方情绪和出发点**，并使用**五秒法则**管理即时情绪。
