# 泰迪核心课《情商提升班》第一课：学员自我剖析与情商认知 🎤

在本节课中，我们将通过聆听多位学员的真实经历与自我剖析，来初步理解情商在职场、社交及个人成长中的核心作用。这些分享揭示了常见的社交误区、情绪管理问题以及自我认知的盲点。

## 学员分享：职场新人的困境与觉醒

我叫杨又鸣，24岁，是一名业主代表。在目前这份工作之前，我跳过两次槽。刚毕业参加工作时，父亲告诉我，在单位要舍得吃苦、舍得吃亏，并说老员工欺负新员工是常事。起初，我确实感到被排挤。

我一直认为，这只是老人家说的老员工欺负新员工的惯例。我每天过得很压抑，得过且过，从未想过自己是否有做得不好的地方。后来我发现，和我同期入职的新人却很受欢迎。起初我不明白原因。

大家都是新来的，为什么大家喜欢他，却排挤我？我意识到，我自身携带的负面能量也在影响他们。于是，我开始学习让自己快乐起来，微笑着和每个人打招呼。慢慢地，我融入了这个大圈子。

但随后问题又出现了。我邀请大家的方式很“逗逼”，喜欢开别人玩笑，却从未考虑过别人的感受。因为大家都在笑，我就以为大家喜欢我。结果发现，这无意中得罪了很多人。那时我才明白，无论是不是开玩笑，都不能去否定别人。

幽默不能用别人的缺点来取乐。慢慢地，我开始在说每一句话前，考虑别人的感受。我会想，如果我站在他的角度听这些话，会不会生气。特别是在现在的工作中，我更能深刻感受到，如果我想让别人服从我的安排，肯定他比批评他更加有效。

当然，我还有很多不足，所以我今天才参加这个情商补习班。其实我觉得，发现问题比解决问题更重要。如果都不知道自己哪里做错了，还谈什么改变自己。其实错了就错了，我觉得知道改正就行，不丢人。那种知道自己错了还厚着脸皮说“我就是这种人，这就是个性”的人，我觉得挺可悲的。我就是这种人，只要面子，听不进不同意见，一意孤行，最后碰得头破血流。

我来这里就是来改变的。我不挑战任何人，我只挑战我自己，我要让自己更优秀。这就是我今天的自我介绍。

**核心公式：有效沟通 = 考虑对方感受 + 积极肯定**

## 学员分享：国企生存法则与价值互换

大家好，我叫张思，是一名国企职工。我从事钢铁行业，这个行业最近两年不太景气。我16岁出来工作，到现在已经将近6年了。

我经历最多的是国企和私企的不同。私企是，只要你干好工作，能给老板带来多大利益，就能拿多少钱。国企的区别在于，有时候不在于你多能干，而在于你多“会来事”，能和领导处好关系。我的师父，也就是我现在的头儿，说过一句话：想要在国企混好，要**么你有人，要么你有文凭**。

我刚工作时，第一年就是个“傻逼”，什么都不懂。第二年，我调到了二期工程。我跟的一个班长，是我刚来时因为租房有过争执的人。他后来一直管着我、压着我。虽然现在关系还可以，但更多是利益上的东西。

我跟这些人相处的关系……我跟这个班长，当时我们斗得很厉害。因为刚参加工作想法不一样，而且我的工作岗位又不是那种特别死板的。当时很多人想抱团。我感觉最深的就是意见不一样，而且人家官还比你大。当时比较傻，跟人家对着干，觉得自己对。

后来这两年，磨得也差不多了，感觉比之前进步了不少，会察言观色了，不像以前那么“逗逼”和“傻逼”了。我接触补习班后，最大的感受就是真的像泰迪说的那样，并不是说你要把到更多的妹，而是要去改变自己，把自己变得更优秀。就像刚才泰迪说的**价值互换**，你有什么样的价值，就能换来什么样价值的朋友。

**核心公式：职场生存 = 硬实力（文凭/技能） + 软实力（人际关系）**

## 学员分享：恃才傲物的反思与目标设定

大家好，我叫刘哲，来自辽宁，是一名公务员。我属蛇，89年出生。我来到这里，最主要是想改变我自己的缺点，用四个字形容就是“**恃才傲物**”。

我刚进入咱们恋爱补习班的时候，被各种人骂，群里人全在骂我，他们从来没见过心态像我这样“黑暗”的人。但是，我在这边交流了大概两个星期，就感觉自己的心态不至于说完全变化，因为人的蜕变是需要时间的。但我可以明显感受到，我不会再带给周围人那么大的压力。很多人在与我相处时，也不会像以前那样，位置高的人对我说话藏三分，位置低的人对我说话避重就轻。我感觉做人更快乐了。

第二，我在参加恋爱补习班这段时间，感觉泰迪、宋导、志宏老师他们三个人带的团队，是一个非常负责的团队。所以我相信，我来参加这个情商班，能够学到更有用的东西。

第三，身边很多人都说我是个“官迷”，我也承认。我想尽快往上走，我一直坚信一点：**一步领先，步步领先**。他们劝我说“眼看他起高楼，眼看他楼塌了”。对于这种说法，我开始很在意。但跟泰迪交流一段时间后，我就觉得，哪怕我将来楼塌了，那我也曾经站在楼上俯视过你。而你只是站在平地上，看着我盖楼，看着我楼塌，你没有这个经历。而且退一步说，如果我的楼不塌呢？我的楼就立在这里呢？那我就一直俯视着你。

所以我来到这里，一方面是想跟着几位导师，尽快把我这个“恃才傲物”的缺点改掉——恃才可以，不能傲物。另一点是，我也希望跟大伙多交流。我的心态现在只是说跟正常人一样，我希望将来能够像我名字后面的“太阳”一样，去带给其他人温暖。

**核心概念：自我成长的目标应是“恃才而不傲物”，并追求为他人提供价值。**

## 学员分享：自我管理带来的事业突破

大家好，我是来自贵州贵阳的杨大卫。我今年二十七了。我接触恋爱补习班是从六月底开始，这个过程从泡妞总结，到情商班个人的提升和自我管理。

以前我在公司做事是得过且过、安逸享乐那种感觉。自从情商提升上来，学习了看书，不管是看书还是根据补习班里大家的交流，我自己吸收之后，就感觉个人不能安于现状。最近我在公司内部竞争上，竞聘上了一个区域经理，等于说是把自己那么多年的潜能爆发出来了。

大家从我的说话也能听出来，我对演讲也很有问题，等于说没有头绪。我也是从自我锻炼开始。在进入补习班之前，大概十一月之前，我做事很随意，随遇而安。但是进入之后，接触情商这个概念，觉得自我管理和自我提升必须有一个更大的改变。

你自己在这个社会上扮演那么多角色，背负着很多责任。不管是你现在意识到还是以后能意识到，你肯定有一个成熟的过程。然后自我慢慢地从内化出来之后，感觉第一个就是在家里你必须承担起责任，不管是排行第几，都要有对家里的责任心，比如父母的赡养、兄弟姐妹的照顾。

我不知道大家有没有那种想法，就是你在家里，不管处于什么角色，当你自己的情商、自我认识上来之后，你不仅想让自己成功，你还会想让周围所有人都跟着受益，有种“一人得道，鸡犬升天”的感觉。

第二个也说一下“泡妞”这件事。最近我感觉进来之后，我还蛮“失败”的，因为我把重心全部转移到事业上来了。导-师们说的一句话很好：**一个人，其实泡妞是一件很小的事情。当一个人的价值体现上来之后，你得到的不仅仅是泡妞或金钱上的事情。**

情商提高之后我觉得自己提高很大，所以很想继续跟着导师学习，和大家交流。

**核心公式：个人成就 = 自我管理 × （硬价值 + 软价值）**

## 学员分享：留学生的自信建设与需求感管理

大家好，我叫金城，来自山东，是一名留学生，今年22岁。刚刚我还在想要是演讲会不会紧张，不过发现本来觉得不会紧张，还是多多少少有一点。

其实一开始来演讲的时候，就知道一件事，演讲第一句话叫做“hook”，要抓住别人的吸引力。那么好，我今天就讲一件事来抓住大家的注意力。今天我很认真地听了前面16个人的演讲，这16个人当中，有4个人是没讲自己工作的。我也把听的每一个人的职业、优点和缺点记录下来了。就像泰迪说的，第一次见到一个人之后，要努力记住他的名字，如果记不住，可以拿笔记下来，这是很加分的。

我觉得我自己的优点和特长，就是我非常喜欢听别人的声音，优点和特长就是我非常自信。在没来到这个团队之前，我除了对女孩方面不够自信之外，对任何事情都有高度的激情、热情和自信。每一天我觉得我的生活可能趋于高价值一点，因为我家境还可以。但往往这样家境生长出来的人，比如像我，比较缺少关怀和爱护。我小时候就感觉到得到别人给我的爱比较少，所以加强自己的努力、建设自己，让心灵和身体双方面变得强大，是我从小一直的信条。

但是直到遇到泰迪，我才发现其实万事万物都一样。当我们发现“需求感”这个东西，当你无意间抱有需求感的时候，其实就是你预选太少。假如你有一个赚十万块的工作，有五家公司同时应聘你，那么你对这五家公司就可以产生一种筛选。就像女孩子也是一样，当你只有一两个女孩子的时候，你会需求感暴露得特别多。

从这个反面角度折射出来说明，我们应该做的是努力提高自己，不管是在各方面软价值硬价值。我这次通过泰迪和我是得道的学员，通过得道的讲话开始学习，我感觉到作为一个男人，应该学习一些女人的特征，比如我们应该打扮自己，穿衣更“骚”更帅，更引领潮流，让我们至少看上去更年轻、更积极阳光。

因为人是“视觉动物”。你可能听我跟你说的几句简单的话，就会改变你对我的想法；你可能看我今天的着装和打扮，就知道我今天来到这里是非常尊重你，或者说我有提前准备好。包括从另一个方面折射出来看的话，我从穿衣打扮第一步开始改变的时候，就会慢慢影响我旁边的人。

**核心公式：吸引力 = 高价值展示（外在+内在） / 需求感**

## 学员分享：外圆内方的处世哲学

大家好，我叫贾志祥，在群里的昵称叫志祥。今天非常开心，能在情商提高班和各位兄弟们、还有泰迪见面交流。

我个人的情况是这样的：目前我在深圳工作，来深圳已经有12个年头了。虽然现在也没有什么很大的成就，但在这么多年的摸爬滚打当中，体会了很多人生和社会各个方面的法则。就像刚才泰迪讲的一些，我觉得非常实用、有道理，就是人之间实际上是一种价值的互换。

我特别想讲一下，早几年前有一本非常流行的书叫《方与圆》。其中概括起来就是说，做人要**外圆内方**。你要有自己的框架、态度、底线，但是外面要圆滑，就像清朝的铜钱一样。这样的人在社会中、在与人交往的过程中，才能够游刃有余，处理好各种关系。

为什么这样讲？我个人以前从学校出来的时候，当年高考家里的期望非常高，一定要考上清华北大才可以。当然我自己也不是那块料，没考上，只考了二本三本之类的学校。父母就不同意，因为还有弟弟要上学，那我只能出去打工挣钱了，所以就南下深圳开始一个人打拼。

摸爬滚打那么多年，再加上自己的性格也属于偏内向，社交直觉方面，可以说是个朦朦胧胧的孩子。刚出来懂什么？什么都不懂。在工作中经历了很多，也看到过一些非常残酷的办公室政治。如果你站错队，你就一定被干掉。我真的是忍辱负重，坚持到现在这家外资企业。

所以我就觉得，我想跟大家分享的就是，如果你不能更好地处理你的……（信号中断）

**核心概念：成熟的处世智慧是坚守核心原则（内方）与灵活运用社交技巧（外圆）的结合。**
