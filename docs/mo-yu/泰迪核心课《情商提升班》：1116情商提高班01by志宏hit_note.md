# 泰迪核心课《情商提升班》：1116情商提高班01 🎤

在本节课中，我们将要学习两个关键的沟通技巧：如何优雅地拒绝他人，以及如何通过“少即是多”的原则进行高效沟通。掌握这些技巧，能帮助我们在人际交往中既保持自我边界，又能清晰有力地传达信息。

## 一、如何优雅地拒绝他人 🤝

上一节我们提到了说话表达方式的重要性，本节中我们来看看第一个具体技巧：拒绝。直接说“不”对很多人而言非常困难，尤其是面对熟人时。勉强答应超出能力或意愿的请求，往往消耗大量精力，且可能因做不好而招致埋怨。因此，学会拒绝至关重要。

以下是“先怪罪自己”的拒绝方法：

*   **核心逻辑**：将无法帮忙的原因归咎于自身的不足或客观限制，而非直接否定对方。这为对方提供了台阶，避免了关系僵化。
*   **应用场景一：拒绝帮忙修电脑**
    *   **低效拒绝**：“不好意思，我没时间”或“我不想修”。
    *   **高效拒绝**：“哎呀，我最近太拖沓了，老板催的报告还没写完。现在帮你修，我怕心不在焉反而弄坏你的电脑，我真是太磨叽了，真不好意思。”
*   **应用场景二：拒绝借钱**
    *   **低效拒绝**：“我没钱”或“不想借”。
    *   **高效拒绝**：“我手头是有点钱，但最近我自己也有急用。如果前脚借给你，后脚我又得找别人借，更麻烦。要不你再问问别人？”或者“哎，我最近花钱大手大脚，是个月光族，没攒下什么钱，真帮不上你。”

这种方法的核心是 **`不帮忙的原因 = 自身问题（忙、能力不足、习惯不好）`**，而非 **`不帮忙的原因 = 直接拒绝对方`**。

## 二、沟通原则：少即是多 🎯

在学会了如何说“不”之后，我们再来看看如何更好地“说”。许多人认为口才好就是能说、话多。但实际上，信息过载会让人抓不住重点，导致沟通失效。

“少即是多”原则指出，**`沟通效果 ≠ 话语数量`**，过多的、无重点的言语就像冗长的“小学生行为规范”，令人难以记住核心内容。

以下是应用“少即是多”原则的关键方法：

*   **识别最高等级决定**：在沟通前，明确你**唯一**必须让对方接收到的核心信息或必须达成的核心目标。这就像军队任务中的“最高等级决定”——当情况有变只能完成一件事时，那件事是什么？
*   **聚焦核心，舍弃冗余**：围绕这个“最高等级决定”组织语言，敢于舍弃非关键的细节和补充说明。与其面面俱到却让人遗忘，不如一针见血。
*   **具体应用对比**：
    *   **低效沟通**：主管事无巨细地唠叨所有注意事项，员工左耳进右耳出。
    *   **高效沟通**：主管用一封简洁的邮件，清晰列出最关键的行动项和截止时间。

这个原则的公式可以概括为：**`有效沟通 = 提炼核心信息 + 简洁清晰传达`**。

## 总结 📝

本节课中我们一起学习了两个提升情商与沟通力的实用技巧。
1.  **优雅拒绝**：通过“先怪罪自己”的方法，将拒绝的原因引向自身限制，既能维护边界，又能照顾对方感受，避免关系尴尬。
2.  **高效沟通**：秉持“少即是多”的原则，在沟通前明确核心目标（最高等级决定），并围绕其进行简洁有力的表达，确保关键信息被对方准确接收。

掌握这两点，能帮助我们在人际互动中更加从容、清晰和有效。
