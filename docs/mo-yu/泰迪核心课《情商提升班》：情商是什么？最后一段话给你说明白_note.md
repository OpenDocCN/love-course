# 泰迪核心课《情商提升班》第1课：情商是什么？🤔

在本节课中，我们将一起学习情商的本质。课程将澄清关于情商的常见误解，并提供一个清晰、实用的定义，帮助你理解情商为何重要以及如何提升它。

## 课程概述

很多人对情商有模糊的印象，例如认为情商高就是“会来事”、“讨人喜欢”或“圆滑”。这些理解并不完整。一个人受欢迎，并不一定意味着他情商高。事实上，大部分人的情商都有提升空间。情商与智商不同，智商更多由先天决定，而情商主要关乎我们日常的为人处世，是可以通过学习和实践改变的。

一个核心观点是：**智商决定了一个人的下限，而情商决定了一个人的上限**。拥有一定智商，你可以从事技术工作养活自己；但你能活得多好、走得多远，则很大程度上取决于你与人相处的能力。因为在这个社会中，我们总是需要与他人交流、协作来完成事情。

所以，首先要建立信心：**情商是可以提高的**。这是我们开设情商提升班的前提。

## 情商的核心三要素

根据我的总结和经历，情商可以简化为三个核心要点：

1.  **感知他人情绪的能力**
2.  **控制自己情绪的能力**
3.  **保持目的与行为高度统一的能力**

请务必记住这三条。接下来，我们将逐一深入探讨。

### 1. 感知他人情绪的能力 👀

这种能力要求你在与人交流时，能敏锐地察觉对方的情绪状态。人与人交流时，70%的情绪信息来自表情、肢体语言等非语言信号。

以下是提升这种能力的一些具体方法：

*   **观察肢体语言**：例如，与人面对面交谈时，观察对方的双脚是否朝向你。如果对方身体对着你但脚尖指向别处，这可能意味着他对当前话题不感兴趣。
*   **辨别语言反馈**：注意对方回应的内容。是真诚的感兴趣，还是敷衍？例如，英文中的“sounds interesting”（听起来很有意思）往往意味着“没意思”。如果对方总是用总结性的话语（如“挺好的”、“加油”）或发表情包回应，可能是兴趣索然的信号。
*   **避免自说自话**：最令人反感的一种行为是，完全沉浸在自己的表达中，无视对方释放出的“不感兴趣”或“想结束话题”的信号。例如，对方已经说“我要吃饭了”，这可能是一种委婉的拒绝；而“我要去吃饭了”则可能只是告知行程，稍后可以继续聊。你需要感受其中的区别。

感知他人情绪在人际关系中至关重要。例如，在恋爱中，很多分手并非“莫名其妙”，而是其中一方长期忽视另一方的情绪暗示所导致。在工作中，无法读懂同事或客户情绪的人，容易遭遇真正的拒绝。

**过渡**：理解了如何感知他人，我们还需要学会在群体中运用这种能力。在多人社交场合，如酒局或新圈子，你需要注意整体气氛。

*   **融入新群体**：如果你想快速融入，可以观察并“讨好”那个主导话题的人（雄性领袖）。多给他递话、接茬，让他带你进入对话圈。
*   **化解尴尬**：当现场出现冷场或尴尬时，主动去做那个转移话题、化解尴尬的人。长此以往，你会成为圈子中不可或缺的一员。
*   **照顾他人**：如果聚会中有特别内向的人，主动与他开启一些话题，帮他刷存在感，他会非常感激你。

**总结**：提升感知他人情绪的能力，需要从认真倾听、仔细观察做起。感受到气氛变化后，要主动采取行动（如转移话题）。坚持练习，你会发现自己更受欢迎。

### 2. 控制自己情绪的能力 🧘

上一节我们探讨了如何洞察他人，本节我们来看看如何管理自己。控制情绪并非指没有脾气或一味隐忍。情商高的人也会生气，但他们会管理愤怒。

首先，请记住一个原则：**在冲动、情绪化的时刻，绝对不要做任何决定**。先把事情放一放，等情绪平复后再权衡利弊。历史上很多智者都懂得这一点，比如唐太宗面对魏征的直谏，虽然生气，但能控制情绪，权衡利弊。

生活中，我们难免遇到令人厌烦的“小人”。与他们纠缠，只会浪费自己的时间和精力。狮子不会因为野狗吠叫而去咬它。不与傻瓜论短长，不是因为大度，而是因为你的时间和人生值得用在更有价值的事情上。

控制情绪最难的一点，在于我们最容易伤害最亲近的人。对亲近的人发脾气，说完狠话后又后悔，这是情商低的表现。情商高意味着，如果你经过思考后决定采取行动（哪怕是发火），你不会为此后悔，因为你的行为和目的是一致的。

**过渡**：控制情绪不仅是为了避免冲突，更是为了有效沟通。许多沟通障碍源于不良的说话习惯。

一个常见的低情商习惯是**习惯性否定他人**。无论别人说什么，第一反应就是“不对”、“不是”、“不可能”。这会让对方感到不被尊重，将对话双方置于对立面，使合作或沟通难以进行。

我不是说你不能表达不同意见，而是要注意方式。你可以先肯定对方合理的部分，再用“同时，从另一个角度看…”这样的方式引出你的观点。这不是虚伪，而是为了让沟通更顺畅，最终达成你的目的。在商业谈判或重要交流中，这种技巧尤为重要。

不要用“乔布斯情商也低”这样的特例作为自己不提升情商的借口。成功因素有很多，情商是其中重要的催化剂，能帮助你将现有资源和价值最大化。

### 3. 保持目的与行为的高度统一 🎯

通过前两节的学习，我们了解了情商中对内和对外的要求。现在，我们来探讨情商的终极表达：**保持目的与行为的高度统一**。无论是情商还是智商，我们的最终诉求都是价值最大化，把事情办成。

举个例子：女朋友深夜想打电话聊天，但你明天要早起加班。低情商的做法是抱怨：“我这么累，你能不能懂事点？”然后挂电话。这会导致争吵升级。高情商的做法是表达态度：“虽然我很累明天要早起，但好久没陪宝宝了，再聊五分钟吧。”这样，懂事的女生反而会催你去休息。女生的“作”很多时候是在试探态度，而非索取具体事物。

另一个典型例子是追求女生。很多人误以为“对她好”就能换来爱情，于是送礼物、摆蜡烛表白，结果让女生尴尬，反而把对方推远。你的目的是和她在一起，但行为（公开施压）却与目的背道而驰。她想要苹果，你却花光积蓄买了一车香蕉。

**需要强调的是，情商不是圆滑虚伪**。同样一个行为“你踢我干嘛”，在朋友聚会时说，是低情商（让提醒你的人尴尬）；但在必须摊牌的商业谈判中说，则可能是高情商（因为你的目的就是打破僵局）。关键在于，你的行为是否服务于你的最终目的。

## 课程总结与情商的实用定义

本节课我们一起学习了情商的三个核心要素：感知他人情绪、控制自己情绪、保持目的与行为统一。情商不是喜怒不形于色，也不是圆滑虚伪。

我为大家总结一个最实用的情商定义，请你记熟：

**情商，是通过对他人情绪的感知，对自己情绪的控制，给出对方想要的回应，并运用情绪，通过恰到好处的行为，最终达成自己目的的能力。**

高情商的最终好处，是让你自己一直开心，也让身边的人开心，从而创造一个更顺畅、更成功的人生环境。
