# 泰迪核心课《情商提升班》：1116情商提高班02_note

在本节课中，我们将要学习两个关键的情商沟通要点：一是当他人没有回应时，如何进行自我反思；二是在表达时，如何避免使用过多借口影响沟通效率。我们将通过故事和场景分析，帮助你理解并应用这些原则。

## 第四点：无回应时的自省 🤔

上一节我们探讨了倾听的重要性，本节中我们来看看当沟通似乎受阻时，我们应有的心态。别人没有理会你，并不代表对方没有听到。对方可能明白你的意思，但选择“装不懂”，或是对话题不感兴趣，或是不想回应。

为了说明这一点，请看以下故事：

一位爷爷发现，他的老伴（奶奶）越来越少跟自己聊天了。爷爷心里嘀咕，是不是奶奶年纪大了，耳朵聋了，听不清楚说话。他非常担心，于是决定测试一下奶奶的听力。

有一次回家时，爷爷想测试奶奶。他用钥匙轻轻打开门，然后用力关上。他看到奶奶还在厨房煮饭，没有任何反应，也没说话。爷爷心中一惊，认为奶奶听力真的不好，自己这么用力关门都听不到。

爷爷就大声喊：“我回来了！”奶奶没有回应。爷爷赶紧走到客厅，更大声地喊：“我回来了！”奶奶还是没有回应。爷爷特别着急，赶紧走到厨房，对着奶奶的耳朵大喊：“我回来了！你是不是聋了呀？”

奶奶转过头来说：“聋的是你。从你开门进来，我已经跟你说了三句话了，你一句都没听到。”

这个故事看似是个笑话，但其实揭示了生活中很多交流的真相。我们说话时，常常会嫌弃对方表现差的部分，但这很可能是我们自己的问题。我们必须时刻自省。

我特别担心大家在生活中变成一个特别爱抱怨他人的人，比如觉得“小张傻逼，小李迟钝，老王有点闷”，觉得所有同事都有点笨。但其实可能是你自己有点问题。我们自己“听不见”，却一心以为是对“听不见”。

许多个性成熟、见过世面的人，当着我们的面，不太想直接点破我们的错误。所以他们会选择不回应这个话题。但我们却以为对方糊涂迟钝，不了解自己的意思。

当我们大呼小叫、反复表达，而对方不回应时，我们应该冷静下来，思考一下：以对方的历练和智商，是真的听不明白，还是其实是我们自己没有搞清楚情况？

很多时候，我们大吼大叫对方没有“听见”，但对方没有转过头来说“聋的是你”，是因为他们给我们留了面子。明白吗？

这种情况可能不是每个人都会遇到，但生活中如果你曾经历过类似场景，一定要自省。记得提醒自己：究竟是对方的问题，还是我的问题？

## 第五点：摒弃无效借口 🚫

最后，我们来谈谈“借口”的问题。

我前一阵子参加了一个相亲活动。我不是去相亲的，而是作为恋爱导师帮大家搞搞气氛、搞搞活动。其中有一个环节是大家唱歌，玩游戏输的人要唱歌。

最后一个男生输了。他看着是一个挺有精气神的男人，特别有自信。但他一说话，就会让大家有点反感。轮到他唱歌了，他上去就说：“哎呀，其实我前天感冒了，今天昨天都不太能发出声音。其实我不想唱歌的。但是我唱歌其实也一般，不过今天嗓子而且不在家，不过今天选到我了，就让我唱吧。”

请想象一下，在那种场景下，他说了这样的话。如果你是一个异性，你对他什么感觉？你可能会想：“你啰嗦什么呢？要做就赶紧做，不想唱就下去啊，说什么呢？”

再设想另一个场景：比如你在开会做报告。如果你上来先不说报告，而是先说：“哎呀，我准备这份报告非常难。我昨天感冒了，我老爸呢又身体不舒服，我先陪我爸去看医生。我自己呢搞得也完全没有精气神，所以我资料准备得非常不充分。”

说实话，大家不会因为你这句话就对你产生同情或谅解。大家可能往往第一时间表达出的情绪是：你有事赶紧说，我们还有别的事呢，开完这个会还有别的会呢。

我们现在身处的时代，每个人每天要接触大量的信息，知道太多好的、坏的、差的和有趣的事情。我们的注意力已经被训练得高度稀缺。在沟通开始时使用冗长的借口，往往无法博得同情，反而会消耗他人的耐心，给人留下推卸责任或能力不足的印象。

**核心公式可以概括为：**
**无效沟通 = 冗长借口 + 核心信息延迟**

因此，在需要表达或行动时，应该直接切入主题。如果结果不尽如人意，再以简洁的方式说明客观原因，这比事先找借口要有效得多。

---

本节课中我们一起学习了两个重要的情商沟通技巧。首先，当他人不回应时，应首先自我反思，而不是一味指责对方。其次，在表达和行动时，应避免使用冗长的事前借口，直接、高效地传递核心信息才是关键。掌握这两点，能让你的沟通更加顺畅和受人尊重。
