# 泰迪核心课《情商提升班》：0918情商课-认知情绪，控制情绪 🧠

在本节课中，我们将要学习情商的本质，并掌握通过认知、控制和观察情绪来提升情商的核心方法。课程将分为理论讲解与实用技巧分享两部分。

## 课程概述

许多人对于“情商”存在误解，认为它等同于会做人、有眼力见或精通人际技巧。本节课将澄清这些概念，并指出情商的真正核心在于对情绪的掌控。我们将从根本原理出发，帮助你建立提升情商的内在能力，而非仅仅学习外在的“术”。

## 第一部分：情商的本质与核心能力

上一节我们概述了课程目标，本节中我们来看看情商究竟是什么。

外界对“情商”一词常有误解，例如将其等同于“为人处事让人舒服”或“人际交往的润滑剂”。这些虽然是高情商可能带来的表现，但并非其本质。

情商（Emotional Intelligence）拆解来看，“情”指情绪（Emotion），“商”指衡量系数（Quotient）。其核心是个人在情感与情绪方面的能力高低。

我认为，系统性的高情商包含以下四种核心能力：

以下是四种核心能力的列表：
1.  **识别自身和他人情绪的能力**：即有眼力见，能听懂言外之意。
2.  **分辨不同感受的能力**：能够区分相似情绪间的细微差别。
3.  **描述感受的能力**：能够准确用语言表达自己的情绪状态。
4.  **控制情绪，或使用情感线索引导思考和行为的能力**：这是情商的最高体现。

只学习外在技巧（“术”）无法真正内化这种能力。就像学习恋爱技巧，如果只学话术而不懂对方情绪，就无法真正提升。情商提升的关键，在于学会识别并控制情绪，那些社交技巧只是在此基础上的自然表现。

## 第二部分：提升情商的三步核心训练法

理解了情商的构成后，本节中我们来看看如何通过三个步骤来切实提升它。

高情商者的最终目标是**控制情绪，而非被情绪控制**。无论是生气时口不择言，还是紧张时手足无措，都是被情绪奴役的表现。

### 第一步：认知情绪

控制情绪的第一步是认识它。你需要了解自己在各种情境下的情绪反应。

*   **公式**：**情绪觉察 = 识别（情境 + 内在感受）**
*   **方法**：你可以像记录日志一样，记下在特定场景（如每周一早晨、见到领导、遇见喜欢的人）时，你产生的主要情绪（如焦虑、紧张、害羞）。人的基本情绪有喜、怒、哀、惧，但实际情绪是这些基本情绪的复杂混合。不必追求像心理学家一样精确，但需识别出生活中常出现的几种关键情绪。

### 第二步：留心身体反应

情绪会引起生理变化，而控制身体反应能反过来影响情绪。

语言可能会欺骗，但身体往往很诚实。例如，紧张时可能会双手抱胸、肩膀缩紧或呼吸短促。

*   **方法**：
    1.  在第一步中，当你识别出“我在与领导谈话时感到紧张”。
    2.  接着观察自己紧张时的身体反应，比如“我会不自觉地双手抱胸”。
    3.  在下一次类似场景中，**有意识地控制自己不做那个动作**（如深呼吸，将双手自然垂下）。这通过行为改变来施加心理暗示，帮助你更好地掌控情绪。

### 第三步：观察他人情绪

情商最终要应用于社交。在能感知和控制自我情绪的基础上，开始观察他人。

通过观察他人的表情（如皱眉、嘴角下垂）、肢体语言（如身体放松、笑容灿烂）和语调，来判断对方的情绪状态。例如，当你粗鲁地指派任务时，下属拱着腰、言辞冷淡，这表明他带有抵触情绪。

**情商的真正含义可以归结为：认知情绪、控制情绪、观察情绪。** 能做到这三点，你就能了解自己、洞察他人。至于具体如何行动，你可以根据实际情况灵活选择。

## 第三部分：五个立即可用的社交技巧 🛠️

掌握了核心心法（“道”）后，本节中我们补充一些能立即使用的社交技巧（“术”），让你的学习更加充实。

以下是五个实用的社交技巧列表：

1.  **习惯并善于夸赞别人**
    *   这是提升人际亲密度的最有效技巧之一。人们即使知道赞美可能有目的，也会下意识地感到愉悦。
    *   **技巧升级**：避免笼统的“你真漂亮”，尝试**通过细节进行赞美**。例如：“你今天这个丸子头特别配这条连衣裙，显得很清纯。”

2.  **要反对，先赞同**
    *   人际交流70%是情绪交流。如果直接反对，会立刻引发对方的对立情绪。
    *   **方法**：在表达不同意见前，先肯定对方观点的合理部分。例如：“这个提案在活跃度上想法很棒！（先赞同）不过在某个细节上，我有点不同的想法……”

3.  **学会拒绝**
    *   不要因为面子或讨好他人，而接受超出自己责任范围或意愿的要求。人性容易将帮助视为理所当然，一次不帮就可能抹杀之前所有的好。
    *   **核心**：维护自己的权益和边界，你的个人框架才会更稳固。

4.  **不要过分迷信拓展人脉**
    *   人脉不在于你认识多少人，而在于有多少人愿意为你提供帮助。
    *   **公式**：**有效人脉 ≈ 情分深厚的人 + 价值互换的伙伴**
    *   通讯录里的大多数人只是“联系人”，盲目扩展无效社交反而浪费时间。

5.  **切莫交浅言深**
    *   尤其在职场等存在利益关系的环境中，在对对方的立场和背景了解不深时，不要急于吐露真情或过多个人信息。
    *   **原因**：人在陌生环境中安全感匮乏，容易通过加快交流进度（交浅言深）来获取安全感，但这可能带来风险。
