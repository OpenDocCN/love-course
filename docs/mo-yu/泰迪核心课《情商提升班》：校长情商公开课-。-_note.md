# 泰迪核心课《情商提升班》：校长情商公开课-。-_note

在本节课中，我们将要学习情商的核心定义、它在生活与工作中的关键作用，以及如何通过一个简单的自我评估来了解自身的情商水平。课程将重点解析情商的两个核心支柱，并列举常见的情商不足表现，帮助你建立清晰的提升方向。

## 课程概述

情商涉及的范围非常广泛。它可以用来在亲密关系中建立更好的连接，也可以在职场中帮助我们避免犯错、处理复杂的人际关系甚至竞争。在开始深入讲解之前，需要明确一个核心观点：智商决定人生的下限，情商决定人生的上限。这意味着，掌握一门技术或知识能保证基本生存，但若想获得更大成就、创造更多价值，就必须学会与人打交道，懂得“借力”。正如古语所言：“君子性非异也，善假于物也。”真正厉害的人并非事事亲力亲为，而是懂得在关键时刻寻求帮助、整合资源。

## 情商的两大核心支柱

上一节我们概述了情商的重要性，本节中我们来看看构成情商的兩個最关键的方面。

### 1. 感知并回应他人情绪 🤝

第一个核心点是**能否准确感知和把握他人的情绪**。你需要判断对方是否生气、是否有情绪波动、或正在为何事纠结。这种能力至关重要。例如，在恋爱关系中，如果女生不开心了，但你无法感知她不开心的原因，可能会说出让关系恶化的话。很多矛盾都源于双方对彼此情绪的误读。能够感知他人情绪，并根据对方的真实意图（即使对方没有明说）给予恰当的回应，是一种高级能力。从小的方面说，这关乎儿女情长；往大了说，这涉及政治与外交的艺术——许多看似“官方”或模糊的表态，背后都是对复杂情绪和局势的精准把握。

这种能力在最日常的“谈判”——砍价中就有体现。你需要通过观察对方的眼神和表情，感知他对你出价的接受程度。如果感知到对方情绪上可以接受，但态度上有些僵持，就需要运用“推拉”技巧，在强硬出价后适当缓和关系，给予对方想要的台阶下，从而达成交易。

### 2. 控制与运用自我情绪 🧘

第二个关键点是**控制自己情绪的能力**。有一个著名的建议是：生气时写的信不要寄出去。人在情绪激动时，常会自以为理智，实则容易做出伤人伤己的决定。学习情商的目的，是让生活中所有事情的价值最大化，避免因情绪失控造成不必要的损失。许多人际关系矛盾都可以通过冷静后的沟通解决。

但控制情绪并非没有情绪。**真正的控制在于善用情绪**。例如，在演讲或教学时需要调动情绪以感染他人；在谈判时可以“假装”生气来设置底线，让对方摸不清你的真实想法。知道自己何时该用什么情绪来达成目的，才是真正的情绪掌控。这与“无法统一做事目的和结果”正好相反——后者是本想达成A目标，却因情绪失控做出了导致B结果的行为。

## 常见情商不足的表现

了解核心概念后，我们可以通过一些具体表现来审视自己。以下是七种常见的情商不足表现，你可以据此进行自我评估。

以下是情商自我评估的七个维度，每个维度请根据自身情况在1-5分之间打分（1分为最符合，5分为最不符合）：

1.  **说话做事不考虑他人感受**：是否经常自我中心，言语无意中伤害他人？
2.  **读不懂他人的婉拒**：是否常把“我考虑一下”、“下次再说”当成真的机会？
3.  **总是制造社交尴尬**：是否常在群体中无意提及他人的短板或敏感话题？
4.  **无法控制自己的情绪**：是否经常因冲动说错话、做错事，事后又后悔？
5.  **过度敏感，怕被伤害**：是否总觉得别人在孤立或嘲笑自己，像刺猬一样自我保护？
6.  **过度依赖他人**：是否在感情或生活中过度依赖某个人，失去自我和独立世界？
7.  **无法统一做事的目的与结果**：是否经常好心办坏事，结果与初衷背道而驰？

> **评估建议**：如果你的总分低于10分，建议先多经历、多观察，打好基础。如果高于28分，说明你已有很好的情商基础，参与课程可能以分享和精进为主。分数在10-28分之间，且常感到在人际、职场中被误解或吃亏，那么本课程将非常适合你。

## 重要原则与总结

本节课中我们一起学习了情商的核心是“感知他人”与“控制自己”。最后需要强调几个原则：

*   **学习情商是为了严于律己，而非评判他人**。社会上掌握资源的人未必情商都高，但提升情商能让你更好地与各类人协作，放大自身价值。
*   **永远不要用自己的标准去衡量和评价他人的得失**。每个人的选择都有其背景和原因，尊重是高效沟通的前提。
*   **自信源于重复的成功体验**。真正的核心自信，是在自己擅长且反复验证过的领域，面对任何质疑都不会心虚的状态。

情商并非天生固定，它可以通过学习和有意识的练习来提升。本课程旨在提供一个高质量的同辈交流环境，让大家能就工作、生活中的真实人际问题进行讨论，借鉴不同领域学员的经验，少走弯路。
