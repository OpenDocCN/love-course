# 泰迪核心课《情商提升班》第一课：做个职场心机boy 🧠💼

在本节课中，我们将要学习如何在职场环境中运用情商智慧，识别并处理复杂的人际关系，从而更有效地实现个人目标。课程将涵盖自我情绪管理、识别他人情绪、职场定位以及一系列实用的沟通与行为技巧。

## 课程概述

情商提升的核心目的是为了获得更多利益，即实现“利益的道化”。这主要依赖于两件事：一是感知并控制自己的情绪；二是感知并回应他人的情绪。掌握这两点是在职场中生存和发展的关键。

## 第一节：感知与控制自我情绪

上一节我们概述了情商的两个核心，本节中我们来看看如何具体实践第一点：感知并控制自己的情绪。

感知自己的情绪并加以控制，意味着在情绪产生时能够意识到它，并选择不做出被情绪驱动的、可能损害利益的反应。

**例如**，作为团队领导者，当员工犯错时，第一反应可能是愤怒。如果此时用“你傻逼吗”这类人身攻击的语言，员工会立即进入防守状态，产生隔阂，不利于后续工作的开展。正确的做法是，首先**感知到自己正处于愤怒状态**，意识到“我正在发火”。许多人危险之处在于，他们愤怒时并不自知，反而认为自己的反应理所当然。我们的目标是进行有价值的对话，以更好地完成任务、获取利益。

## 第二节：感知与回应他人情绪

理解了自我情绪管理后，我们进一步探讨如何感知并回应他人的情绪，这是团队协作和领导力的基石。

人与人、小团队之间会产生“化学反应”。一个团队的强大与否，不在于有多少个“大神”级人物，而在于成员之间能否融合并创造最大价值。就像五个全明星球员未必能打赢一个只有一个核心球员但配合默契的队伍。

在职场团队中，那个被称为“大哥”的人，往往不是技能最强或口才最好的，但一定是情商最高的。他能够敏锐地感知到团队中每个人的情绪变化。例如，在会议中察觉到某人情绪低落时，他能及时转移话题；在微信群或现场，他能注意到谁被冷落、谁受了委屈、谁过于张扬，并做出恰当的回应。这就是**感受别人的情绪并予以相应回应**。

## 第三节：认清职场关系的本质

在学习了情绪管理的基本功后，我们需要建立一个至关重要的认知基础：职场关系的本质。

现实中的职场关系，可以比作**嫖客与妓女**的关系。老板是嫖客，员工是妓女。我们通过工作提供价值（服务），老板支付报酬。这种关系本质上是价值交换，而非情感依附。

你在公司的话语权和地位，不取决于你是谁，而取决于**你能提供多少价值**。你能让老板“爽”、解决的问题越多，你的权利就越大。那些抢着做事并能做好的人更容易晋升，因为组织会发现“离了他不行”。

因此，切勿沉迷于公司的“企业文化”或“归属感”等虚幻概念。要清醒地认识到，你来职场是为了赚钱养家。当无法提供价值时，老板会毫不犹豫地终止关系。多思考如何提供价值、解决问题，而非制造问题。

## 第四节：实用沟通与行为准则

建立了正确的认知后，我们来学习一些能立即应用的细节技巧。以下是几个关键场景下的处理原则：

**1. 关于“在吗？”的沟通**
当收到几百年不联系的人发来“在吗？有空吗？”，通常意味着对方有求于你。最佳策略是**假装没看见**。如果事后被问起，可以借口“手机出了问题”。反之，当我们有求于人时，也应避免问“在吗？”，而应直接将具体事情用一句话说明，给对方选择回应或忽略的空间。**对方不回复，本身就是一种拒绝**，不应再追问。

**2. 约人/邀约的框架**
约人时（无论是同事还是异性），不要问“你有空吗？”，这给了对方一个轻易拒绝的机会。人的思维有惰性，应直接给出框架。
*   **公式**：`[具体时间] + [我想/我们] + [具体事项]`
*   **示例**：“周六下午你没事吧？我带你去尝尝新开的那家餐厅。” 这比“周六有空吗？”成功率更高。

**3. 销售与引导技巧**
优秀的销售不会把主动权完全交给客户。例如，卖手机时，问“先生有什么需要吗？”通常得到“我随便看看”的回复。而问“先生您是**自己用还是送人**？”，则默认对方有购买意向，并进行引导。与人交往同理，有时需要温和地主导节奏，而不是总让对方做决定。

## 第五节：职场生存的重要原则

掌握了具体技巧，我们还需要遵循一些更高阶的生存原则，以避免因小失大。

**1. 警惕小恩小惠**
牢记“**出来混总是要还的**”和“**无功不受禄**”。接受同事的小恩小惠，未来可能需要成倍偿还。保持清醒，控制贪小便宜的情绪。同事关系应保持适当距离，能AA制尽量AA。

**2. 保持自知之明，懂得取舍**
要有“审美”，知道什么是好的。但更重要的是有“自知之明”，清楚自己能配得上什么。盲目追求好高骛远的目标（如绝对配不上的异性或职位），只会导致挫败和资源浪费。**懂得舍弃**不切实际的幻想，根据自身条件做出明智选择。

**3. 表现策略：藏锋与亮剑**
*   **藏锋**：初入职场或新环境，即使有能力做到100分，也先做到80分。这为你留下进步空间，避免一开始就把领导的期望值拉得太高，也防止招致同事嫉妒。枪打出头鸟，能力越强，可能被分配的任务越艰巨。
*   **亮剑**：**学会邀功**。事情做成后，要低调且详尽地向关键人物汇报：“老板，有件事我觉得应该跟您汇报一下，当时情况是……，我处理的方式是……，您看这样行吗？” 既展示了成果，又体现了谦逊和尊重。同时，**学会不背锅**，避免成为闷头做事却无人知晓、出事却首当其冲的人。

**4. 明确主次：价值优先**
在公司，你的**首要价值是完成任务、解决问题、产出业绩**。社交能力和情商是重要的助力，但绝不能本末倒置。确保本职工作至少达到及格线，再辅以良好的关系处理能力，这样的人往往发展得最好。

**5. 同事关系：保持距离**
同事就是同事，务必**保持安全距离**。走得太近，容易陷入人情绑架（如频繁被要求帮忙、顶班）。切勿对同事掏心掏肺、倾诉秘密或抱怨公司。**祸从口出**，你说的任何话都可能成为未来的把柄。我们提升心机，首要目的是**防止被别人坑**。

**6. 技能暴露：伺机而动**
除了本职工作外的技能（如PS、做PPT、剪视频），**不要过早主动声张**。一旦人人皆知，你会陷入免费帮忙的漩涡，且一次不帮就可能前功尽弃。应在别人急需、主动求助时，再“为难地”出手，这样你的帮助才显得有价值。

## 第六节：培养理性思维，避免被误导

最后，我们要锻炼一种能看透表象的理性思维，防止被各种信息洗脑。

人们容易被生动具体的故事说服，而忽略背后的逻辑和“幸存者偏差”。
*   **案例一**：“红颜薄命”似乎有道理，但我们忽略了那些不红颜也薄命的人，只注意到了漂亮的案例。
*   **案例二**：二战时，军方发现返航飞机机翼弹孔多，决定加固机翼。但统计学家指出，真正需要加固的是机身，因为**被击中机身的飞机大多没能返航**（幸存者偏差）。

在听取任何一面之词（尤其是涉及利益时）时，要保持独立判断。例如，对于高收益投资承诺，要牢记“**风险与收益成正比**”。当对方只强调收益时，务必警惕。一个简单的原则：对声称年化收益远超10%的项目保持极度怀疑。

混血儿往往漂亮，不是因为混血本身神奇，而是因为**能实现跨国婚恋的个体，本身在外形、社会地位等方面往往已是佼佼者**（另一种幸存者偏差）。看清事物背后的逻辑，谁也忽悠不了你。

## 课程总结

本节课中我们一起学习了情商在职场中的应用框架：
1.  **核心基础**：感知并控制自我情绪；感知并回应他人情绪。
2.  **关键认知**：认清职场关系是价值交换的本质，远离虚幻的情感依附。
3.  **沟通技巧**：避免无效开场（如“在吗？”），学会用框架邀约，掌握引导式沟通。
4.  **生存原则**：警惕小恩小惠；保持自知之明与懂得取舍；懂得藏锋与适时亮剑；明确工作价值优先；与同事保持安全距离；有技能要伺机而动。
5.  **思维提升**：培养理性思维，警惕幸存者偏差，对信息保持独立判断。

将这些原则内化于心，在面临职场抉择与人际互动时反复思考、实践，你将能更从容地应对复杂环境，保护自身利益，并更有效地达成目标。情商提升是一个持续的过程，如同婴儿成长，积累到一定阶段，必将爆发出巨大力量。
