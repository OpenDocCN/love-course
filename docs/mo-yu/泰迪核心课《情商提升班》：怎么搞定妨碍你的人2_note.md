# 泰迪核心课《情商提升班》：怎么搞定妨碍你的人2_note

在本节课中，我们将学习两种核心的激励模式，探讨如何通过了解他人兴趣来有效施加影响，并介绍“透明帘子”这一重要概念，以实现更顺畅的人际互动与目标达成。

## 两种激励模式：奖励与惩罚

上一节我们介绍了情商的基础概念，本节中我们来看看两种不同的激励模式。

奖励的方式与期望得到的行为直接挂钩，因而形成一个正向的激励。这种激励方式强化了期望的行为。

**公式：** `奖励 -> 强化期望行为`

处罚的方式是将不当的行为与所接受的处罚形成一个负向的激励模式。这种模式仅仅在不当的行为与接受处罚之间形成了平衡，而正确的行为并没有得到有效的强化。

**公式：** `惩罚 -> 抑制不当行为`（但未强化正确行为）

惩罚只能让人停止做某些事情，但是惩罚之后的效果，仍然不会激励被惩罚对象主动向良好的方向改进。这就是奖励比惩罚更有效的原理。

## 激励的核心：建立正确关联

在实战中，重要的不是采取何种激励方式，而是如何把奖励和要导向的结果联系在一起，形成一个合理的激励模式，从而有效地实现奖赏的目的。

以下是生活中常见的例子：

*   **信用卡额度：** 按时还清欠款，信用卡额度就会上涨。这激励用户完成“按时还款”这一正确行为。
*   **渔夫与蛇的寓言：** 渔夫因怜悯放走蛇口中的青蛙，并给蛇酒喝作为安慰。蛇误以为“带来青蛙”就能获得“酒”作为奖励，于是叼来更多青蛙。渔夫本想阻止蛇抓青蛙，却错误地奖励了该行为。

这个寓言引申出一个管理哲理：你奖励的行为，可能并非你真正期望的。

以下是管理中常见的错误激励案例：

*   期望激励贡献，却奖励了苦劳。
*   期望提高效率，却奖励了加班时长。
*   期望团队合作，却只奖励了个人业绩。
*   期望鼓励创新，却惩罚了未能实现的创意，奖励了墨守成规。

因此，在设计奖励或惩罚时，必须思考：站在对方的角度，你的措施是否会引导他朝你期望的方向发展？**关键是要让对象明确知道，他是因为何种行为才受到奖励或惩罚。**

## 如何了解他人兴趣：投其所好

了解对方兴趣是有效实施奖励或惩罚的前提。中国人讲究“投其所好”。如果对方需要的是认可，你给予金钱反而可能产生反效果。

那么，如何了解他人的兴趣呢？以下是一些方法：

*   **观察社交媒体：** 查看对方朋友圈、微博等经常发布的内容。
*   **直接询问相关人：** 通过共同的朋友或家人进行了解。
*   **区分工作与生活账号：** 尝试接触对方的私人社交账号以获得更真实的信息。

如果不了解对方兴趣，盲目付出，在对方看来可能显得别有用心，反而会拉远距离。关心对方的兴趣，体现了你的了解和尊重，这本身就是一笔情感储蓄。

## 利益驱动与“透明的帘子” 🎭

人与人交往存在利益驱动。利益就像人的骨架，必不可少。但骨架外需要有血肉和皮肤，再外面还需要得体的衣服。这就是“透明的帘子”——**虽然行动由利益驱动，但需要进行包装，不要显得赤裸裸。**

**代码示例：**
```plaintext
核心：达成目标（如借物品、获得帮助）
步骤：
1. 建立情感连接（谈论共同经历、表达关心）
2. 提出核心请求
3. 附加温情回馈（如赠送小礼物、表达感谢）
```
例如，想找老师借音响，可以先聊聊近况、回忆师生情谊，临走时送上水果，然后在门口“顺便”提出借用请求。这虽然本质是交换，但因为有“帘子”（情感铺垫和心意）的包装，让对方更容易接受。

不懂包装，即使想给予好处，也可能无法让人接受。人与人交往，可以通过捆绑利益来促使对方行动。在对方表现好时奖励，表现不好时惩罚。通过了解其兴趣来决定奖惩内容，并用“透明的帘子”进行包装，能使你的意见更容易被采纳，从而达成目标。

## 课程总结

本节课我们一起学习了：
1.  **奖励与惩罚**两种激励模式的原理与效果差异，强调奖励应直接关联并强化期望行为。
2.  激励的**核心在于建立正确关联**，确保你的奖惩措施能引导对方朝你期望的方向发展。
3.  通过**观察、询问**等方式**了解他人兴趣**，是实现“投其所好”、有效施加影响的基础。
4.  引入了 **“透明的帘子”** 概念，指出在利益驱动的人际互动中，**进行适当包装**（如结合情感、尊重等形式）能让你的意图更容易被接受。

这些方法不仅能帮你应对妨碍你的人，更能让你在管理、社交、亲密关系等各个方面，更有效地引导他人，达成共赢的目标。
