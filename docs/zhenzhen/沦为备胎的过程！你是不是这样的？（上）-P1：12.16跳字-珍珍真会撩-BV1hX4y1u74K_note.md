# 恋爱避坑指南 01：识别“备胎”陷阱 🚫

在本节课中，我们将要学习如何识别在情感关系中沦为“备胎”的早期迹象。我们将拆解“渣女”的核心逻辑，并分析你不知不觉陷入“备胎”角色的第一步。理解这个过程，是避免情感伤害、建立健康关系的第一步。

## 什么是“渣女”？🤔

在深入步骤之前，我们需要明确一个核心概念。所谓“渣女”，其本质是**只想收获不想付出**。她们的根本目的是让你追求她，但始终不与你确定正式关系，从而可以持续获得你的好处与付出。所有行为都围绕这个目的展开，万变不离其宗。

## 沦为备胎的五大步骤

基于上述核心逻辑，你沦为“备胎”的过程通常可以分为五个步骤。以下是第一步的详细解析。

### 第一步：散发个人魅力，引你上钩 🎣

对方会通过各种方式展示吸引力，营造出一种“容易接近”或“有机可乘”的假象，让你在不知不觉中对她们产生兴趣。具体手法包括但不限于以下几种：

*   **展示外在魅力**：经常在社交动态发布精心修饰的个人照片，或穿着性感，通过视觉冲击吸引你的注意。
*   **释放暧昧信号**：言语或动作带有轻佻、暧昧的意味，让你产生“她对我有意思”或“她很缺伴侣”的错觉。
*   **伪装单纯无害**：扮演“傻白甜”的角色，让你觉得她天真单纯、很好欺骗，降低你的防备心。
*   **倾诉情感空虚**：向你透露自己感情不顺、内心空虚，营造脆弱感，让你觉得“有机会趁虚而入”。

正是在这种潜移默化的相处过程中，你逐渐对她产生了兴趣，为后续步骤埋下了伏笔。
