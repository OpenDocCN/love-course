# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。这个“标签”就是变量名，而“盒子”里装的东西就是数据值。

在Python中，为一个变量赋值非常简单，使用等号（`=`）即可。等号左边是变量名，右边是你想存储的值。

以下是给变量赋值的基本语法：
```python
variable_name = value
```

## 基本数据类型 🔤

理解了如何存放数据（变量）后，我们需要知道能存放哪些种类的数据。Python有一些内置的基本数据类型，最常用的有三种。

以下是三种核心数据类型及其介绍：

1.  **整数**
    *   **含义**： 整数是没有小数部分的数字，可以是正数、负数或零。
    *   **代码示例**： `age = 20` 或 `temperature = -5`

2.  **浮点数**
    *   **含义**： 浮点数是带有小数点的数字，用于表示实数。
    *   **代码示例**： `price = 19.99` 或 `pi = 3.14159`

3.  **字符串**
    *   **含义**： 字符串是由零个或多个字符组成的文本数据，需要用单引号（`‘’`）或双引号（`“”`）括起来。
    *   **代码示例**： `name = “珍珍”` 或 `message = ‘Hello, World!’`

## 使用变量与类型查看 🛠️

创建了变量之后，我们可以在代码中通过变量名来使用它存储的值。此外，我们还可以使用一个内置函数来查看某个变量中数据的类型。

在Python中，使用 `print()` 函数可以将变量的内容输出到屏幕。使用 `type()` 函数可以查看变量的数据类型。

以下是操作示例：
```python
# 赋值
my_name = “珍珍”
my_age = 25
my_height = 1.65

# 使用变量：打印输出
print(my_name)
print(“年龄：”, my_age)

# 查看类型
print(type(my_name))   # 输出：<class ‘str’>
print(type(my_age))    # 输出：<class ‘int’>
print(type(my_height)) # 输出：<class ‘float’>
```

## 总结 📝

本节课中我们一起学习了Python编程的基石。我们首先将**变量**理解为一个存储数据的命名容器，并通过 `=` 进行赋值。接着，我们探讨了三种**基本数据类型**：用于整数的 `int`、用于小数的 `float` 以及用于文本的 `str`。最后，我们掌握了如何使用 `print()` 函数输出变量值，以及如何使用 `type()` 函数检查变量的数据类型。掌握这些概念是您后续学习更复杂编程知识的关键一步。
