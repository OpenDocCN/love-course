# 恋爱聊天技巧课程（1）🎯：避免“审问式”聊天

在本节课中，我们将要学习如何避免一种常见的错误聊天方式——“审问式”提问。这种方式会让对方感到压力，从而破坏轻松的交流氛围。我们将通过一个具体例子来分析问题，并理解其背后的原理。

大家是否注意到了课程开头例子中的对话模式？从我与大量学员的交流经验来分析，许多男性朋友在聊天时采用的方式，确实容易让气氛变得尴尬。他们似乎不太清楚应该如何与心仪的女生展开自然舒适的对话。本节课，我将为大家剖析一个核心的误区并提供正确的思路。

## 核心误区：连续提问制造压迫感

上一节我们提到了聊天中常见的尴尬情况，本节中我们来看看导致这种情况的核心原因——连续提问。

观察开篇的对话示例，其中一方在持续不断地提出问题。这种沟通模式类似于一种审问，会给聊天对象带来无形的压迫感。没有人会喜欢这种被逼问的交流方式。

试想一下，如果你被人接连追问以下问题，你会作何感受？

以下是这种“夺命连环问”的典型例子：
*   你有房吗？
*   你有车吗？
*   你每月工资多少？

这种接连不断的隐私追问，只会让对方觉得个人边界被侵犯。在这种压抑的氛围下，女生自然不愿意继续愉快地聊下去。
