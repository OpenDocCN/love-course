# 恋爱技巧课程 01：避开扣分行为，让女生主动靠近你 🚫➡️❤️

在本节课中，我们将学习与女生相处时需要避免的几种“扣分行为”。这些行为会降低你在女生心中的价值感，导致关系停滞甚至倒退。理解并避开这些陷阱，是建立健康吸引力的第一步。

有些男生性格内敛，不常与女生交流，恋爱经验也较少。当终于遇到一个聊得来的女生时，男生会以为遇到了对的人，并希望关系能更进一步。

然而，女生却可能突然变得冷淡，微信回复很慢，也约不出来。男生往往想不明白原因，又不敢直接询问。实际上，这很可能是因为你在相处过程中出现了“掉价”行为，女生在心里不断给你扣分，从而从对你有好感变得无感。

她没有删除或拉黑你已经算不错了，若你还幻想与她有更亲密的举动，那更是异想天开。

## 第一点：缺乏主见

做任何事都询问女生“行不行”、“好不好”、“能不能”。在女生看来，这并非尊重，反而会让她觉得你靠不住，没有自己的主意。

例如，女生精心打扮后问你：“我们待会去哪呀？”如果你回答：“随便，你想去哪就去哪。”她会觉得不如待在家里点外卖。女生希望有人能带领她、保护她，让她可以放心依靠。如果自己的伴侣像领袖或王者一样有决断力，那会更好。

因此，你需要表现得坚定一些，不要再轻易说“随便”。如果实在不知道如何接话，可以说“都听你的”。因为在女生看来，前者是敷衍，后者是宠爱。

为什么女生容易对带领她的男生产生好感？为什么女生对偶像剧里的“霸道总裁”人设情有独钟？现在你应该明白了。该坚定的时候如果软弱，女生怎么会感到满足？

当然，有的男生过分坚持己见也是不行的。例如，连挑个电影都过于较真，会让女生觉得你只是想看电影，而不是想和她约会。

## 第二点：舔狗行为

女生告诉你：“双11剁手了，这个月只能吃土了。”于是，月薪三千的你给她转了两千，还担心她不够花。女生失恋了心里难受，半夜给你打电话，明明第二天要早起上班，你还是出去陪她喝酒买醉。她想要的你都给，不惜花费金钱、时间和精力。

你以为付出总会有回报，总有一天能抱得美人归。其实，那些买彩票的人也是这么想的，觉得总有一天能中头奖。

一味对女生好，不断投资她，确实会提升她的价值。但随之，她的眼光也会越来越高，于是你就显得不匹配了。

**核心公式：** 你的价值 < 你帮助提升后的她的价值 → 你失去吸引力

把钱投资在自己身上。当你变得优秀时，倒追你的女生没有十个也有一百个。

## 第三点：斤斤计较

这类男生永远成不了“舔狗”，因为他们希望用最少的投入，获得最大的回报。

例如，你请女生吃饭后，会各种暗示女生回请你。给女生送礼物后，还会假装不经意地透露价格，暗示对方回赠等价的礼物。

这类男生心眼小，斤斤计较。女生只会觉得你不像男人，和你在一起只会心累。别人谈恋爱都甜甜蜜蜜，她又何必自讨苦吃？

对于这类男生，建议努力去找富婆吧。如果你的经济条件确实有限，只能请女生吃路边摊，礼物只能选平价商品，那你一定要让女生看到你的上进心和发展前景。
