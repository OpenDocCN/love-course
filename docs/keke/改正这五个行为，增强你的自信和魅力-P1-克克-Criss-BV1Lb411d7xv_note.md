# 课程一：改正五个行为，增强你的自信和魅力 🚀

在本节课中，我们将学习五个会削弱你自信和魅力的常见行为，并探讨如何改正它们。这些行为在与他人交往，尤其是与异性相处时尤为常见。通过识别并调整这些行为，你可以显著提升个人吸引力和自信心。

## 行为一：不敢表达真实想法

上一节我们介绍了课程概述，本节中我们来看看第一个行为：不敢表达真实想法。

首先思考一个问题：当女生问你“我们今天去哪玩”时，你会怎么回答？给你两秒钟时间思考一下。你是不是会说“我都可以”、“听你的”、“随便”？我知道你是想尊重别人的选择，表达自己的善意和友好。但如果你每次都这样说，结果只会恰恰相反。这种行为无疑是在迎合别人。

尤其是在和喜欢的人相处时，会给对方两种感觉：
*   第一种，没有主见和领导力。
*   第二种，害怕表达，没有自信。

兄弟们要明白，你的真实想法说出来，能否被接受其实并不重要。重要的是，当你表达出来的时候，会让对方有意识地采取你的意见，那么你就拥有了话语权。即使最后不被采纳，时间一长，你的话也是值得被尊重、被听到的。那么长久以往，对方就会本能地关注你的态度。

所以有时候，兄弟们身处在不被关注的环境中，一部分原因就是不敢表达自己的真实想法。“我吃不了辣”、“我不喜欢”、“我想干什么”等等，这些想法都可以大胆地提出来。因为社交本来就是双方站在平等的位置上相处。当你敢于说出来的时候，就会极大地提升你的自信心和人格魅力。

## 行为二：渴求被他人肯定

了解了不敢表达的危害后，我们来看第二个常见行为：渴求被他人肯定。

我们经常意识不到这件事，但在生活中非常常见。比如你完美地完成了一份工作，一定希望被上司夸奖；或者拍了一张你觉得特别好看的自拍发在朋友圈，希望被疯狂点赞。但恰恰是这种想法，让你的视野变得特别狭窄。因为带有期许地完成一件事情之后，就很容易把自己局限于目标的回应当中。所以一旦结果不能让你满意，就会产生焦虑、害怕等负面情绪，然后陷入到闭环当中。这也就是你自信心降低的原因。

那到底该怎么做呢？其实很简单，就是一个心态的问题。以下是调整心态的方法：
*   不管做什么事情，在快要完成的时候，提前暗示自己：结果怎么样都无所谓。
*   我并不是很在乎别人的看法。
*   我也不在乎是不是非他不可。

当你能够建立起这样的想法的时候，你就有更强的信心来做当下的事。千万不要小瞧自我暗示的力量，因为思想决定行为。虽然它不能成为女生喜欢你的决定性因素，但是可以帮你极大地散发出自己的吸引力和魅力。

## 行为三：长期停留在不适的环境中

解决了心态问题，我们来看第三个也是最关键的行为：长期停留在不适的环境中。

它不仅会摧毁你的自信心，还会不断地摧毁你的自尊心。为什么我会这么说？因为我总结了几千份案例，发现大部分男生在面对女生的时候自卑、怂，甚至连和女生对视的勇气都没有，就是因为他们长期在压抑自己，情绪无法得到释放，每天都封闭在负能量的环境当中。不仅仅是工作中，更多的是在情侣相处的环境里。

之前有个朋友，本来是一个非常积极乐观的人。自从和一个女生在一起之后，变得丧、焦虑，每天都在挣扎。因为女生没事就抱怨哪哪不如意，然后男生就会想，是不是自己哪里做的不好。但其实这些负能量的散发源头，都是女生。

所以如果你身处的环境能够改变你的情绪和心态，我的建议是不如放弃。不然长期以往，它会激化你的信心，摧残你的自尊心。虽然我是一个情感博主，致力于让每一段感情都能向上发展，但还是要说一句实话：不论男女，当你在一段感情里发现无法让自己变得越来越好的时候，那结果还不如一个人生活。

## 行为四：提前做大量的准备

明白了环境的重要性后，我们来看第四个行为：提前做大量的准备。

比如和女生第一次约会，提前做攻略：聊什么话题、吃饭有什么忌口、要看什么类型的电影，都提前准备好，掌握着每一个细节，就是为了给对方留下最好的印象，投其所好。这么做是没什么问题。但糟糕的是，因为你太刻意地去做准备，会导致自己丢失了最关键的状态：随性。

你忘记了，约会是两个人互相了解的过程，本身就是在一种随意的状态下进行。越是剧本型的约会，状态会越让女生感到拘谨，更加无法真实地去展现自己、表露自己。而且一旦在过程中，出现了和计划相反的情况发生，就会显得自己手足无措，前言不搭后语。这就成了恶性飞轮，让自己在社交中越来越不自信。

所以做准备是可以的，但不要这么“满”。留自己一个空白期，就像考试前一天，老师总是说不要看书一样，给大脑放空一下。这么做也是为了能让你进入到一种非常轻松的状态。毕竟一份完美的答卷，是 **70%的努力 + 30%的随机发挥**。

## 行为五：贬低自己，提前降低别人的预期

最后，我们来看第五个行为：贬低自己，提前降低别人的预期。

我发现很多男生在不自信的时候，总是会自我贬低。比如和朋友去唱歌，为了不想自己太丢人，就会说“我歌唱的不是很好听，不要有太多期待”。我知道你这么说的原因，是想通过降低别人对你的预期，让他们不要那么失望。但就因为这个预期，会导致你一旦暴露一点瑕疵，就会被无限放大。别人就会认为：“哦，原来你真不会唱歌”。慢慢地，你就会变得更加不自信。

所以不要再企图通过贬低自己来降低别人的预期。你该怎么做就怎么做，做得不好，慢慢改变就行了。

当然，如果降低预期能发挥你在优势的地方，最后的结果，反倒会让你不鸣则已，一鸣惊人。比如你做饭很好吃，某一天一群朋友来家做客，提前先告诉他们：“等一下随便做几个菜，不好吃，别埋汰我”。他们此时就不会抱太大的期望，但最后尝到的却是一桌美味佳肴。那你在他们心目中的印象，是不是就提高了这个档次呢？这就是所谓的“扮猪吃老虎”。

---

本节课中我们一起学习了五个会削弱自信和魅力的行为：不敢表达真实想法、渴求被他人肯定、长期停留在不适环境、过度准备以及自我贬低。每一点后面的解决思路，才是本节课的精髓。记住，自信和魅力源于真实的自我表达、稳定的内在心态以及健康的人际环境。
