# 两性关系法则解析：布里夫法则 🧠 - 课程01

在本节课中，我们将要学习一个名为“布里夫法则”的两性关系理论。这个法则可以帮助我们理解许多常见的情感与社会现象，例如高离婚率背后的原因、女性在关系中的某些行为模式等。通过学习，你将能从一个新的视角审视两性互动。

## 课程概述 📋

根据民政局的统计数据，自2003年起，我国的离婚率持续攀升。到2020年，平均离婚率已达到45%。其中，由女性主动提出离婚的比例高达73%。这组数据令人惊讶，也促使我们思考背后的原因。

无论屏幕前的你是已婚还是未婚，了解这些现象背后的逻辑都很有必要。为什么近一半的婚姻会走向失败？为什么通常是女性提出离婚？本期内容将介绍一个非常实用的两性法则——布里夫法则。

这条法则几乎可以完全解读两性行为。例如，它能解释为什么女生有时会“作”且不讲道理，为什么“舔狗”往往没有好结果，以及婆媳矛盾为何产生等。这些困扰很多人的问题，都可以在这条法则中找到解释。

## 布里夫法则的起源 🦁

罗伯特·布里夫出生于1876年。他年轻时是一名外科医生，后来投身于社会人力学研究，早年还写过小说。他最著名的贡献就是这条“布里夫法则”。

该法则的核心内容是：**在动物的社群里，雌性具有决定性，而非雄性。当与雄性结合无利可图时，这样的结合就不会发生。**

以草原上的狮群为例。众所周知，狮群是以雄狮为中心的社会。狮群中最强壮的雄狮，即狮王，拥有交配权和优先享受食物的权利。雄狮在享受这些权利时，负担的责任是保卫地盘。它们大多时候无所事事，只是在等待挑战者的出现。

这场保卫地盘与地位的搏斗通常需要赌上性命。如果雄狮输了，它就失去了最强的雄性地位。布里夫法则的残酷之处在于：没有母狮会去接近或跟随这位失去地盘多年的落败者，因为它已经无法提供任何利益。

被驱逐家园的雄狮必须找到下一个狮群抢夺地盘。通常，在负伤的情况下，它无法存活到下一次战斗。

## 自然界的繁衍策略与人类社会的映射 🔄

在自然界中，雄性与雌性有着不同的繁衍策略。雄性因为交配的代价低，倾向于与更多的雌性交配。而雌性必须负担起孕育和养育的责任，因此选择极为谨慎，她们会选择能带来更多利益的交配对象。

所以，雄性之间必须互相竞争，以获取交配权。这对理解人类两性行为有很大帮助。

在人类社会中，男性通过金钱、社会地位、才艺、外貌等进行竞争，也是为了博得女性的青睐。现实中如此，网络上更是如此。在社交软件上，一个条件中等的女性，比起条件中等的男性，得到的“喜欢”可能是其数倍。她们可以坐下来，慢慢挑选最吸引她们的对象。

布里夫法则解释了女性“慕强”的本性。她们通常是“向上兼容”。换句话说，对女性而言，男性的价值保障非常重要，因为这关系到后代能否更好地生存。

在部落时代，如果男人无法给予女性更多的生存好处，那么女性就不会与他结合，否则生存下来的概率就会大大降低。虽然现代社会已改善很多，但这是刻在基因里的东西。

## 布里夫法则的三个推论 📝

由布里夫法则，可以延伸出三个重要推论。

以下是第一个推论：
1.  **男性在过去对女性所提供的好处，不一定能够维持两者在未来的关系。** 也就是说，不管你在过去为对方提供了多少价值，如果你本身创造价值的能力一直不高，那她可能还是会不喜欢你。
    例如“舔狗”。为什么“舔狗”提供了那么多价值，女生还是不喜欢呢？因为“讨好”这个行为本身，就是一种价值很低的行为。在部落时代，只有弱者会去讨好强者。无法独立生存的人，通常会通过讨好他人来获取食物和生存资源。因此，女性的基因里会潜在地排斥这种通过讨好来获取资源的弱者。

以下是第二个推论：
2.  **男性为了维持未来与女性的关系，所提供的任何当前好处，大多数都是短暂的。** 只有提供了一种让女性基因觉得可以持续发展、有价值的东西，关系才更稳固。我们通常称之为“潜力股”。
    即使你现在穷困潦倒，但如果你霸气外露，时刻坚信自己“一定能打下一片大大的江山”，这种信心会让一个男人特别有魅力。

以下是第三个推论：
3.  **男性向女性承诺未来的好处，对于当前及未来两者结合所起到的作用十分有限。** 俗称“画饼”。“饼”是会露馅的。换句话说，女性更注重短期收益。早期大多数年轻女性无法理解长期收益是什么。
    所以有些男生会好奇：为什么我这么努力工作上进，就是没有女生看得上我？其实很简单，因为你不能在当下立即给予女生看得见、摸得着的好处。这与“自信”不同。自信或“装逼”，是一种雄性自我实现的行为。“我装逼是我的自我需求，不管有没有你，我都要做。”这更像是一种老虎咆哮、宣誓自己地盘的行为。

## 运用法则分析常见行为 🧐

基于布里夫法则，我们来简单分析一下女性的两个常见行为，以加深理解。

上一节我们介绍了法则的三个推论，本节中我们来看看具体的行为分析。

首先，很多男生搞不明白：为什么女生喜欢“作”，而且不讲道理？其实，带入布里夫法则就很容易理解。女性更期望从男性这里获取更多的关心、关注，以及期望男性对自己好。有的“好”代表资源，有的“好”代表让她有掌控的感觉。

由此可以得出结论：**女性“作”的行为，本质上是对男性的“驾驭”，其目的是在一段感情当中更有安全感。** 一切尽在掌控之中，自然安全感十足。所以一些女生经常哭闹、经常“作”，但是男生只要一对她好、一哄，她立马就好了。这个“好”，有的是资源，有的是听话的态度。

其次，为什么会有婆媳之间的矛盾？我们继续带入布里夫法则，就很容易得出结论。**婆媳矛盾的本质，是两个女人在互相竞争一个家庭的主导权。** 妻子责怪丈夫“你不应该太听妈妈的话，不然你就是个妈宝男”。男人过度孝顺或过度顺从母亲的行为，违背了伴侣的意愿。所以女性在漫长的进化过程中，形成了排斥这种行为的本能，由此产生了婆媳矛盾。

## 法则的普遍性与总结 🎯

根据布里夫法则，任何与你有任何形式关系的女性（除了你的母亲），都或多或少地想看看你能为她带来什么样的好处，或者看到未来你能给她带来什么样的好处。无论是金钱、社会地位，又或者是幸福的爱情等精神需求。

这不只是在女性身上出现，在特定的环境下，男性也是如此。例如在工作晚宴上，你会特别花心思去认识对你升职有帮助的主管。
