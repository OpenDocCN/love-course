## 五个导致长期单身的行为通病
什么样的男生一看就是单身？有一种男生常年单身，他们身上有几个出奇一致的特点。比如讨好女生，喜欢围着女生转。其实远远不止这些。以下是这类男生的五个通病。如果你中了两条以上，你已经是长期的单身狗了。如果你中了五条以上，可能连一次恋爱都没谈过。

### 1. 目的性太强（需求感爆棚）
很多兄弟很疑惑，什么是需求感。很简单，你渴了想喝水，饿了想吃饭。喜欢一个女生，你就不断地去靠近她。你一说话，你的行为里就充满了各种对女生的示好。归根结底，你越想得到她，你就越不由自主地低姿态地去讨好她。但是你这么做，只会让女生越来越不喜欢你。在追求女生的时候，做这种掉价的行为，只会让女生觉得你这个人非常没有魅力。

**核心概念**：需求感过强表现为 **`行为 = 急切示好 + 低姿态讨好`**。

### 2. 经常为自己解释
领导不会跟下级解释事情，将军也不会跟士兵解释事情。只有下级会跟上级解释，只有低价值的人会跟高价值的人去解释。同理，跟女生聊天时，女生没有听懂，很多兄弟就喜欢像写作文一样去跟女生解释清楚。那么这个时候，女生可能看都没看完，甚至都不想看。很多兄弟喜欢跟女生讲道理，说服女生是一个非常错误而且不自信的表现。当你想要通过讲道理去说服女生的时候，女生一定会拒绝。

上一节我们介绍了需求感过强的问题，本节中我们来看看另一个常见错误：过度解释。

### 3. 献殷勤，讨好女生
很多兄弟在跟女生聊天的过程中会有这种表现。比如说，女生给你发个消息，他们会立刻秒回。其实这就是一个很掉价的行为，会让女生觉得你闲得没事做。而且跟女生聊天的时候，如果你一直示好——女生冷了你让她多穿衣服，女生饿了你就请客吃饭，女生累了你就去接她下班——这些表现就像古代皇上跟太监的关系一样，从来只有单方面的讨好。兄弟们一定要记住，有地位低的人才会去巴结讨好地位高的人。在聊天当中如果你这么做，会让你的地位和价值感都很低，女生根本就看不起你，所以绝对不会对你有好感。

理解了过度解释的问题后，我们接下来探讨一种更直接降低自身价值的行为：单方面讨好。

### 4. 自以为是，急于求成
这是很多兄弟聊天的大忌。在聊天的时候暴露很强的目的性，特别想让这个女生马上喜欢自己，赶紧开启甜甜的恋爱。但是兄弟们，你们能不能站在女生的角度去思考一下。人是有思维的，有感情的。你这么做，女生肯定会想：你到底是奔着感情去的，还是只是因为你馋她身子？了解情况的女生她知道你的情商不低，不了解的早就被你吓跑了。

除了行为上的讨好，心态上的急躁也是一个大问题。本节中我们来看看急于求成的表现。

### 5. 不断自我怀疑，否定自己
很多兄弟有了喜欢的女生之后，就把跟女生聊天当成头等大事。每天盯着手机消息，哪怕女生只回复他一个表情，他还会高兴得不得了。但是由于自己的聊天技术比较欠缺，聊着聊着女生也就觉得无聊，就不怎么回了，有的甚至直接消失。这个时候，这部分兄弟的内心会非常忐忑不安，自己跟自己作斗争，想了很久，一定跑去跟女生道歉认错，说自己哪里做得不够好，我会改之类的。“我到底哪里惹你不开心了？”“你是不是生气了？”你可能觉得你在修补这种关系，可是很遗憾地告诉你，女生看到你这些话，只会更加想逃离你，躲着你。兄弟们，你们一定要知道一个点：内心和外在都有强大气场的男生才会受欢迎。女生根本就不会喜欢你那种唯唯诺诺、毫无主见的样子。

最后，我们来看看一个内在的、但同样具有破坏性的问题：自我怀疑与否定。
