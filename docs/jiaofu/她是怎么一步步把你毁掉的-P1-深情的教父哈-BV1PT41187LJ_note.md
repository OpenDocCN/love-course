## 概述：一个被毁掉的人生案例

前两天见了一位很久没联系的朋友。了解他的近况后，我感到心痛。他毕业于985高校，今年29岁。毕业时去了一家很好的国企，月收入两万多，工作非常体面。但现在却在送外卖，每天奔波，只能勉强温饱。

而这一切的原因，都是因为一个女生。他为了追求那个女生，可以说付出了一切，丢掉了自己的铁饭碗，还背上了几十万的网贷。但最后，那个女生却狠心地离开了他，和一个有钱的老男人结婚了。他直到现在也没办法走出阴影。

可以说，他的人生彻底被这个女生给毁掉了。那么，我今天就来给大家讲一讲，一个男人是怎么被女生给毁掉的。

## 易受伤害的群体特征

他们对待每个人都很真诚，从来没有花花肠子。对于爱情，他们总是抱着很美好的想法，不追求找一个多么漂亮的女朋友，只想两个人能够简简单单，毫无保留地对待彼此。心里有什么话从来藏不住，遇到喜欢的人也不会玩什么套路。

他们只想真诚地对对方好，然后表白，跟那个女生在一起生活，一起上班，过好自己的日子。可越是这种简单的男生，越是容易被有心机的女人给毁掉。

## 操控步骤详解

以下是女生如何一步一步毁掉一个男生的具体过程。

这些女生在刚认识你的时候，会给你一种错觉，让你觉得她们好像真的很喜欢你，你和她好像是有戏的。

*   比如刚开始的时候，会主动找你聊天，甚至每天跟你聊到很晚。
*   会跟你说晚安、早安。
*   会跟你分享一些她的日常：今天干了什么，去了哪里，心情很好。
*   又或者跟你分享一些自己的负面情绪，让你觉得她很信任你。
*   最后再说一句：“我真的只愿意把这些心里话分享给你，你真的很贴心。”
*   有的时候，女生甚至还会跟你说一两句“我想你了”，或者对你表达关心。

这个时候，你脑海中已经快“高潮”了，觉得你们之间马上就可以原地结婚。你甚至觉得她已经是你女朋友了。

可是，当你这么想的时候，你就开始对她越来越上心：每天都会找她聊天，脑海中全部都是她。会因为她的一个早安开心半天，也会因为她几个小时不回你的消息而坐立不安。

当你达到这个状态的时候，她们就开始进行第二步。

### 第二步：引导投资

她会开始让你为她一点点付出。

*   最开始的时候，是让你帮忙拿一些快递。
*   或者是上班的时候帮她打卡。
*   又或者说去她家里帮她喂猫。
*   这样一些力所能及的小事。

这个时候，她会给你展现出知恩图报的一面：一方面给你语言上的感谢，另一方面会给你一些小恩小惠（比如一杯奶茶）。让你觉得这个女生还挺好的，挺懂事的。

然后你就更加喜欢她，你觉得自己真是捡了个宝，遇到这么好的姑娘。一定要把自己的所有都给她，不能辜负这个女生。

接下来，就是她们收网的时刻。

*   她们会让自己的生活费“永远都不够”。
*   她还会在你的生活中，扮演一个弱女子的形象：哪怕她能够单手踢桶，她还会故意假装拧不开瓶盖。
*   哪怕她学富五车，她还会故意被你蹩足的笑话逗笑。
*   哪怕她是深谋老算，她还会装作职场上的傻白甜，让你不断去给她帮忙。

然后在这个过程中，不断地对你表示出崇拜和谢意，诱导你进一步地付出，却从来不给你一点实质性的回馈。

当你开始有点怀疑，是不是把你当工具人的时候，她往往会对你说：“你对我太好了，可是我真的没什么能为你做的。我是不是接受你太多的好了？”——这种表里表气的话。

这时候，虽然你明明想要她做你女朋友，但是因为你脸皮薄、因为你胆子小、因为怕被拒绝后朋友都没得做，在女生的引导下，你会说出那句让你万劫不复的“魔咒”：

**“没事，这都是我应该做的。我对你的好是不图回报的。”**

这句话一说出来，基本上你就可以被“判死刑”了。因为接下来，你会对她投入越来越多。

*   你开始过节给她送更加贵重的礼物，什么奢侈品包包。一个奢侈品要你几个月的工资，但是你为了能让她开心，你给她买。
*   她房租不够，你借网贷也给她付。
*   更有甚者，女生会说什么自己家里人生病了，要几十万的医药费，实在不知道怎么办。这个时候，你又砸锅卖铁给她凑钱。

我之前那个朋友，就是因为女生说她爸病了需要手术，借了50万的外债给这个女生。

这个时候，女生会怎么对你？她会配合你的演出：和你一起出去玩，和你约会、和你吃饭。她会让你周围所有的人都觉得你们是一对，板上钉钉的一对。但就是不和你发生实质性的关系：手不给你拉，嘴不给你亲，床不给你上。

时间久了，当你忍受不住这种没有名分、看得见却摸不着的感觉的时候，你想要跟她表白，确定你们之间的关系。这个时候，就到了女生的第三步。

### 第三步：使用“万能语录”

当你想要确定关系时，她们会使用一套“万能语录”来拖延和安抚你。

她们会对你说：

*   “你人真的很好，你对我也很好，我知道。可是我现在还不想谈恋爱。”
*   “我要把心思放在工作上。”
*   “我想多陪陪我爸妈，给我点时间好不好？”
*   “我被前任伤害太深了，我没有安全感，我害怕再次被伤害，所以别着急好吗？”

这时候，再委屈巴巴看着你的眼睛，你还能说什么呢？你只能够再来上一句：

**“我尊重你的想法，没关系，我愿意等你。之前的渣男不懂得心疼你，我会心疼你，我们就慢慢来。”**

我之前的那个兄弟，就是“慢慢来”了整整三年。

然后，女生就开始循环上面的步骤：继续给你希望，让你感觉到她是喜欢你的，让你觉得你们是有未来的；再引导你投资；当你想表白确定关系的时候，再拖延你。直到榨干你的最后一点价值，然后女生就拍拍屁股走人，告诉你“我对你没感觉，抱歉”。

这个时候，你付出了所有的一切，最后却是人财两空。这种感觉，体会过的人都懂，像心里面被插了一把刀子一样。然后，你还安慰自己：不怪她，是自己没有能力给她提供更好的生活。

当一个男生被女生这样对待以后，整个人真的被毁掉了。他失去了稳定的工作、背上了沉重的债务、更重要的是，他的自信心和对感情的信任被彻底击碎，陷入长期的自我怀疑和痛苦之中。

本节课中，我们一起学习了情感操控者常用的“给予希望、引导投资、使用万能语录”三步法。识别这些模式是自我保护的第一步。请记住，健康的关系建立在平等、尊重和相互付出的基础上，任何让你长期感到卑微、焦虑且得不到明确回应的关系，都值得警惕。
