# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？📦

变量可以理解为一个贴有标签的盒子，用于在计算机内存中存储信息。上一节我们介绍了编程的基本概念，本节中我们来看看如何创建和使用变量。

在Python中，创建一个变量并为其赋值非常简单，其基本语法是：
`变量名 = 值`

例如：
```python
name = "校长"
age = 30
```
这段代码创建了两个变量：`name`存储了字符串“校长”，`age`存储了数字30。等号`=`在这里是“赋值”的意思，而不是数学中的“等于”。

以下是使用变量时的几个关键点：
*   **变量名**：由字母、数字和下划线组成，不能以数字开头，且区分大小写。
*   **赋值操作**：使用等号`=`将右边的值赋予左边的变量名。
*   **动态类型**：Python的变量类型由赋予它的值决定，无需提前声明。

## 认识基本数据类型 🔤

变量可以存放不同类型的数据。就像盒子可以装书本、水果或工具一样，不同的数据类型用于表示不同种类的信息。

Python中有几种内置的核心数据类型，最常见的有以下三种：

1.  **整数**：表示没有小数部分的数字，例如 `-10`, `0`, `100`。
2.  **浮点数**：表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
3.  **字符串**：表示文本信息，需要用单引号`‘ ’`或双引号`“ ”`括起来，例如 `‘你好’`, `“Python”`。

我们可以使用 `type()` 函数来查看任何变量或值的数据类型。其公式为：
`type(要检查的对象)`

让我们通过一个例子来巩固理解：
```python
height = 175          # 这是一个整数
weight = 70.5         # 这是一个浮点数
message = “欢迎学习”  # 这是一个字符串

print(type(height))   # 输出：<class ‘int’>
print(type(weight))   # 输出：<class ‘float’>
print(type(message))  # 输出：<class ‘str’>
```

## 变量的基本操作 ➕➖

创建了变量之后，我们就可以对它们进行操作了，比如进行数学计算或拼接文本。

对于数字类型（整数和浮点数），可以进行加减乘除等算术运算。
```python
a = 10
b = 3
print(a + b)  # 输出：13
print(a * b)  # 输出：30
print(a / b)  # 输出：3.333...
```

对于字符串，可以使用加号`+`进行拼接。
```python
greeting = “你好，”
name = “张三”
full_message = greeting + name
print(full_message)  # 输出：你好，张三
```

需要注意的是，不同类型的数据不能直接混合运算。例如，不能直接将字符串和数字相加，需要先进行类型转换。
