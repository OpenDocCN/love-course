上一段我们提到了本节课的分析对象。接下来，我们首先从决定面部基础的“头骨与轮廓”开始分析。

他的颧骨略微突出，属于骨多肉少的类型。这种结构让人看起来更清瘦，并且在衰老后更能凸显保持年轻的优势。

分析完整体轮廓后，我们将目光聚焦到“眼睛与神采”，这是面部光彩的核心起点。

李泰容拥有扇形的双眼皮和信眼。他的眼珠尤其大、黑、亮，卧蚕也非常饱满，这使得眼睛显得汹涌有神。

整张脸的光彩都从这里开始。这种眼型在气质上显得聪明、机灵，给人一种精致感。

眼睛的神采需要眉毛的衬托。下面我们来看看“眉毛与男性气质”。

在李泰容的脸上，眉毛是其大眼睛的最佳辅助。他的眉毛浓密、粗直。

**公式：** 阳刚的眉毛美学 = 浓 + 粗 + 直。

从美学角度看，这样的眉毛极具男性的阳刚之美。

眉眼的框架搭建好后，“鼻子”作为面部的中心，对和谐度至关重要。

**公式：** 和谐鼻眼比例 → 鼻宽 ≈ 眼间距。
