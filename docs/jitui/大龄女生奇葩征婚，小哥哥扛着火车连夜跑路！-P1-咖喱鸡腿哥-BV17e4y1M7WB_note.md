# 课程01：如何理解“需求”与“现实”的差距 🧐

在本节课中，我们将通过一个生动的社会案例，学习如何分析个人诉求与现实条件之间的巨大差异。我们将把一段描述性的文字，转化为结构清晰、易于理解的教程，并在这个过程中练习信息提炼与逻辑梳理。

## 案例背景概述

有一位来自外地的29岁女性，现居成都。她的基本情况如下：身高158厘米，体重50公斤，拥有二本学历。她曾在深圳外企工作数年，后通过跳槽来到成都。她自我评估薪资待遇良好，并认为自己的颜值在成都颇具竞争力，身材也令自己满意。

她尤其自信于个人风韵，认为其魅力令人过目不忘。因此，她自视甚高，对普通男性并不感兴趣。

## 具体的择偶要求

以下是这位女性在相亲时提出的详细择偶条件。

*   **基本条件**：男方身高必须超过180厘米，外貌出众。
*   **学历与收入**：必须拥有硕士及以上学历，年薪需达到200万元以上。
*   **资产要求**：必须在成都市青羊区拥有房产，座驾价值不低于40万元。
*   **个人特质**：要求男方擅长烹饪，情商高，热爱生活，且格局宏大。

## 核心的婚姻前置条件

上一节我们列出了她的基本择偶标准，本节中我们来看看她提出的更为具体的婚姻前置经济要求。

1.  **婚前购房**：男方需在其老家省会城市，为她的父母和姐姐全款购买三套公寓（因当地限购政策）；同时需为她在当地全款购买一套住宅。
2.  **彩礼金额**：彩礼最低要求为200万元。
3.  **婚后经济安排**：婚后男方所有收入必须交由她管理。
4.  **家庭赡养**：每月需支付其父母5万元生活费；另需支付其姐姐、姐夫5万元生活费。

## 附加条件与相亲现场反应

除了经济要求，她还提供了其姐姐的家庭背景作为附加说明。她的姐姐身有残疾，姐夫条件一般，两人无固定职业、无房无车，收入不稳定，日常开销依赖信用卡。

她强调，符合其条件的追求者众多，需排队等候通知。本次被通知见面的男性，在听完所有要求后，反应剧烈。该男性感到肌肉痉挛、呼吸不畅，极度震惊之下迅速离开茶馆，沿高速公路奔跑了二十多里地方才平复心情。

## 课程总结

本节课中，我们一起学习了一个关于婚恋市场极端要求的案例。我们将其杂乱的口语描述，系统性地整理为**背景介绍**、**个人要求**（`基本条件`与`经济条件`）以及**事件结果**几个部分。这个案例清晰地展示了，当个人诉求（`Demand`）严重脱离普遍现实（`Reality`）时，可能导致的荒谬结果。理解这种差距，是进行任何有效沟通或谈判的重要前提。
