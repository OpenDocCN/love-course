# 颜值分析课程 01：解析“冰霜美男”张颜齐的清冷感来源 🧊

在本节课中，我们将要学习如何从专业角度分析一个人的面部特征，并以艺人张颜齐为例，具体拆解其被称为“冰霜美男”的视觉成因。我们将从脸型、三庭五眼、五官细节等多个维度进行剖析。

## 脸型与轮廓：男性化气质的基底

上一节我们介绍了课程概述，本节中我们来看看张颜齐的脸型基础。他的脸型轮廓分明，下颌角非常明显。这种组合塑造出强烈的男性化气质。

## 三庭五眼与面部立体感

分析完基础脸型，我们进入更精细的比例分析——三庭五眼。张颜齐的中庭略长。从五眼比例来看，由于太阳穴有轻微凹陷，因此并非完全标准。

他面部的立体感很强。在很瘦的身体状态下，他面部的多个骨点都清晰在线。

以下是支撑其面部立体感的关键骨点：
*   眉骨
*   颧骨
*   鼻尖
*   下颌角

这些骨骼结构共同支撑起了他面部的立体感。即使未来面部软组织松弛，仍有骨骼支撑，因此整体结构不易垮塌。

## 眼睛：魅惑与冷酷的矛盾结合体

面部立体感由骨骼奠定，而眼睛则是整张脸的视觉中心。张颜齐拥有一双桃花眼，内眼角（瑞儿蕾兽）和外眼角都较尖，搭配小平行双眼皮和小卧蚕，为原本魅惑的眼睛增添了稚嫩感。

笑起来时外眼角上挑，眼神显得真诚迷人。但他的眼睛存在轻微的下三白，且眼型弧度下垂。这带来一种冷酷颓废的感觉，个性十足。

## 鼻子：高挺与弧度的精妙平衡

眼睛决定了神态，鼻子则主导了面中部的结构。他的山根起点很高，与高眉弓和深眼窝衔接流畅。

山根的宽度适中，大约是鼻翼宽度的三分之一。鼻梁高挺，伴有轻微的驼峰突起。这个恰到好处的弧度，让鼻子不会过于硬朗，反而增加了辨识度，且不破坏美感。

## 唇形与整体气质整合

最后，我们来整合下半张脸的特征。他的嘴唇偏向M唇，嘴角轻微上扬，人中较短，下唇比上唇略厚，形状立体。

不笑时，嘴角略微下垂。这种唇形与他的其他五官组合起来，共同营造出难以接近的冷酷感。他本身的气质也属于酷帅类型，因此被粉丝称为“冰霜美男”。

---

本节课中我们一起学习了如何逐步分析面部颜值。我们以张颜齐为例，从**轮廓分明**的脸型和**立体骨骼**出发，探讨了其**三庭五眼**比例，重点剖析了**下垂眼型**与**下三白**带来的冷感，以及**高挺带驼峰**的鼻子和**M唇**如何共同塑造出独特的“冰霜美男”气质。这种清冷感是多个面部特征与其个人气质共同作用的结果。
