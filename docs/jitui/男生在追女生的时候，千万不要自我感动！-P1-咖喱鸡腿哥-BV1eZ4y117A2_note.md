在本节课中，我们将要学习一个在追求过程中至关重要的概念：避免自我感动。许多男生在追求女生时，容易陷入自我感动的误区，这不仅无法推进关系，反而会让对方感到压力与不适。理解对等关系的逻辑，是健康交往的第一步。

## 自我感动的逻辑误区

上一节我们提到了自我感动的概念，本节中我们来看看其背后的核心逻辑。许多男生不明白一个道理：**你越是在乎一个人，你就越得不到一个人**。追求行为本身没有问题，但关键在于双方的关系必须对等。

现在，很多男生一旦喜欢上一个人，就容易将对方视为整个世界的中心。这种状态容易导致行为走火入魔、思维偏激，并做出一厢情愿的事情。

## 自我感动的典型表现

以下是自我感动式追求中常见的几种行为模式，这些行为往往会让女生感到“下头”。

*   整夜发送大段文字给女生。
*   不顾对方需求，肆意表达关心。例如，姑娘想吃荔枝，你却送了一箱苹果。
*   在付出后，追问对方为何不感动、为何不喜欢。
*   赠送自己亲手制作但对方并不需要的礼物（如手工包、乐高），尽管对方已明确表达过其他喜好（如化妆品）。
*   过度解读女生的行为，例如对她发的朋友圈反复琢磨半宿。
*   想方设法逼问女生“爱不爱我”这类问题。

这些行为都属于**自我感动**。你做的这些让对方有压力的事情，她怎么可能因此爱你？你总是做这些不被珍视的举动，她出门可能都想躲着你。

## 如何建立对等关系

理解了错误行为后，我们来看看正确的做法。健康关系的基础是**对等**。这意味着你的行为节奏应与对方回应相匹配。

核心原则可以概括为以下代码逻辑：
```python
if her_reply_speed == SLOW:
    my_reply_speed = SLOW  # 不秒回
if she_is_busy == True:
    do_not_disturb()       # 不打扰
```
爱得太满，物极必反。在相处中，你应该保持自信和从容的气场。还是那句话，能愉快相处就谈，不能就果断离开。

## 总结与提醒

本节课中我们一起学习了“自我感动”的危害与对等关系的重要性。有些时候，男生自以为是的付出，不过是多余的自我感动。你不仅没有感动到她，反而给她造成了极重的心理负担。

**自我感动式的付出，既危险又愚蠢。** 千万不要让一段感情，成为你一个人的独角戏。
