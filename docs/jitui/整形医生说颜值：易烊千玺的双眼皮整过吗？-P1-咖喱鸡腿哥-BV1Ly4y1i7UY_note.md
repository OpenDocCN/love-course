在本节课中，我们将要学习双眼皮形成的科学原理，并通过分析明星案例，帮助大家理解遗传与后天发育对眼部形态的影响。

---

有粉丝来信询问，易烊千玺如何从单眼皮变成了双眼皮。今天我们就重点讲解这个问题。

易烊千玺小时候是单眼皮，某些角度看起来是内双，属于扇形的、延伸到内眼角的内双。后来，随着年龄增长，他的外貌发生了变化。

突然之间，人们觉得他变得帅气，认可了他的颜值。此后，他越来越好看，上演了从单眼皮到双眼皮的“逆袭”，五官也变得更加立体。他在15岁时，面部就已褪去了少年的稚气。

因此，许多质疑者提出疑问：易烊千玺的双眼皮是否源自整形手术？那么，这到底是什么原因造成的呢？

其实，最主要的原因是遗传。双眼皮是**显性基因**，单眼皮是**隐性基因**。其遗传规律可以用以下公式表示：
```
假设控制双眼皮的显性基因为A，隐性基因为a。
父母均为双眼皮（基因型可能为AA或Aa）时，
后代仍有1/4的概率为单眼皮（基因型aa）。
```

易烊千玺的颜值变化，主要源于他小时候良好的面部基础和父母基因的作用。因为基因的表达并非一蹴而就，许多人在成长过程中，显性基因的性状会逐渐表现出来。

其中就包括了双眼皮的形成。我凭借八年外科手术的经验明确回复大家：易烊千玺的双眼皮绝非通过整形手术获得，它是后天自然形成的，绝对没有动过外科手术。
