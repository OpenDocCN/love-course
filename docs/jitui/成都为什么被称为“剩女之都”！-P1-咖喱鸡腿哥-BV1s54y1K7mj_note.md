## 现象概述：成都的“剩女”现象

成都目前被称为“剩女之都”，主要原因是该市的单身女性数量已突破100万。单身男女的比例达到了惊人的 **1:4**，并且这一比例还有进一步扩大的趋势。

## 核心因素分析

上一节我们概述了成都婚恋市场的整体现象，本节中我们来看看导致这一现象的具体核心因素。以下是几个关键原因：

1.  **本地优质男性资源稀缺**：成都本地的优质男生数量本来就较少。

2.  **外部竞争加剧**：每年都有大批外貌出众的外地女生涌入成都，争夺原本属于本地女生的婚恋市场资源。

3.  **择偶地域偏好明显**：成都本地女生中，绝大多数只愿意寻找本地男性。统计显示，**最多只有20%** 的本地女生愿意考虑外地来的男生。这导致婚恋市场结构严重失衡。

4.  **择偶标准筛选严格**：本地条件相对较差的男性（例如，无房无车、收入较低、外貌不佳）很难被女性看上，从而被淘汰出局。这使得市场上可供选择的男性数量进一步减少。

## 市场供需失衡的结果

分析了结构性原因后，我们来看看这些因素如何共同作用，导致市场供需的最终状态。

市面上女生的数量和男生的数量出现了严重的倒挂。此外，一部分自身条件优秀的“大龄剩女”对找对象持比较随缘的态度。

这部分女性的特征是：
*   收入足以养活自己。
*   工作非常稳定。
*   住房与出行没有后顾之忧。
*   生活品质相对较高。
*   婚姻观是“宁缺毋滥”。
*   虽然接触过不少优质男生，但最终因各种原因未能成功。
