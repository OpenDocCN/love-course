上一节我们介绍了本课程的目标，本节中我们来看看具体的原则。如果你的个人条件、家庭背景和收入都处于普通水平，并且希望寻找一位能踏实过日子的女朋友，你需要牢记以下两点核心原则：

1.  **尽量减少不必要的经济开销。**
2.  **在原则上保持平等，不刻意讨好或“伺候”。**
