## 案例背景：一次被忽视的周末约会 💔

上一节我们介绍了本节课的主题，本节中我们来看看一个发生在成都的具体案例。一位男生为女友精心策划并付出金钱的周末活动，却接连遭到女友的否定和嫌弃。

## 行为分析：情感互动中的价值错位 ⚖️

在分析了背景后，我们进一步审视女方的后续行为。这位女生在整个周末选择躺在沙发上，沉浸在大型社交群聊中。

以下是她在群聊中的具体行为列表：
*   她抢到了一个金额为**1.26元**的拼手气红包。
*   随后，她向群里发送了大量表达感谢的客套话，例如“感谢中国首富”。

## 核心问题：付出与回应的严重失衡 📉

她的行为与男友的付出形成了鲜明对比。男友准备了**数千元的预算**，希望共度一个高质量的周末。而女方却为了**1.26元**的红包，在网络上与他人热聊半夜。

这种互动模式揭示了情感关系中的一个核心问题：**价值感知的错位**。一方认为珍贵的心意和物质付出，在另一方眼中可能微不足道，甚至不如极小的即时娱乐反馈。

我们可以用一个简单的公式来理解这种失衡：
```
一方的情感/物质投入 >> 另一方的情绪价值反馈
```
当这个不等式持续成立时，关系就会变得不健康。

## 对比反思：建立平等尊重的互动意识 🤝

通过对比，我们可以更清楚地看到健康互动应有的样子。在某些需要付费购买陪伴服务的场合（例如，花费800-1000元在KTV包房请人聊天），互动是明码标价、权责清晰的。

反观案例中的关系，男方付出了高额成本，却只换来了女方极低的关注度和敷衍的回应。这种模式长期持续，会严重消耗主动付出方的热情和自尊。正如高尔基曾间接警示的：有些恋爱关系，连狗都不愿意身处其中。

## 课程总结

本节课我们一起学习了如何通过具体案例识别情感关系中的“无效付出”。关键点在于，健康的感情需要双方对彼此付出的**价值有基本认同**，并且**回应与付出应大致对等**。单方面持续的高投入，若只能换来极低的关注和敷衍，这种关系便是失衡和消耗性的。建立关系时，应追求相互尊重、价值匹配的良性互动。
