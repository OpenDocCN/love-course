## 什么是“刀妹”？

“刀妹”特指那些**外貌出众、身材姣好，但自身家庭条件一般**的女性。她们从周边或其他城市来到成都，核心目的明确：**通过建立恋爱或朋友关系来获取经济利益**。我们可以将其行为模式概括为一个目标函数：

**核心目标：Maximize(经济收益 | 建立亲密关系)**

## 成都本地刀妹与外地刀妹的区别

上一节我们介绍了“刀妹”的基本定义，本节中我们来看看基于地域背景的不同，“刀妹”的策略有何显著差异。主要区别体现在行事风格与节奏上。

以下是两种类型“刀妹”的详细对比：

*   **成都本地刀妹：追求“短平快”**
    *   **特点**：眼光精准，能快速识别潜在目标。
    *   **策略**：迅速接近并确立关系，随即开始索要财物。
    *   **行为分级**：
        *   针对“新手”：索要包包、首饰等。
        *   针对“熟练目标”：直接要求房子、车子。
        *   激进策略：要求对方结束现有婚姻关系。
    *   **核心公式**：**效率 = 收益 / 时间**，致力于在最短时间内实现收益最大化。

*   **外地来蓉刀妹：擅长“放长线”**
    *   **特点**：外表装扮更为精致，日常生活显得优渥。
    *   **策略**：不急于求成，先进行情感投资。
    *   **行为模式**：
        *   初期不仅不让男方花钱，甚至主动赠送贵重礼物。

        *   当男方自愿进行“回馈”时，金额往往巨大（例如30万起步）。
    *   **核心策略**：**投资未来更高回报**，通过前期投入建立深度信任，最终获取远超投入的收益。
