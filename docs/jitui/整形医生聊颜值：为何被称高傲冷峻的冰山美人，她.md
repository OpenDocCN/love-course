# 整形医生聊颜值：为何被称高傲冷峻的冰山美人，她

在本节课中，我们将要学习如何从专业角度分析明星的颜值特点。我们将以韩国女星郑秀晶为例，拆解其被称为“高傲冷峻的冰山美人”的容貌构成要素，并理解其独特美感背后的原因。

## 概述：亚洲审美下的独特存在

郑秀晶的长相是普遍亚洲人比较喜欢的类型。她没有夸张的绝对大眼，以及受人追捧的锥子脸，反而五官很精致。

韩国的审美更加偏向于甜美清纯、白净清秀、眼睛笑起来弯弯、没有什么攻击性的长相。但是像郑秀晶这种个人特点强烈，尤其是面部攻击感很强的艺人，可以说做到了独树一帜。

喜欢她的人就会特别喜欢，不喜欢的人就永远会对她无感。她的颜值不可能做到符合所有人的审美，但她有她自己独特的气质。

## 五官细节分析

上一节我们概述了郑秀晶颜值的整体定位，本节中我们来看看她五官的具体特点。

### 灵动的眼睛

郑秀晶的眼睛我非常喜欢，有点三白眼的感觉。郑秀晶的迷之眼皮经常被黑粉们拿来说事。

事实上，郑秀晶本来就应该是双眼皮。只是我注意观察，她早期出道的时候双眼皮不是特别明显，所以说她给人高冷、清冷的感觉。

但大家仔细观察，郑秀晶的眼睛真的很灵动、很清澈，笑起来会变成一道弯弯的月牙。卧蚕也比较适中，像一个小孩子。

### 名品鼻子

在郑秀金的五官里，我个人最喜欢的就是她的鼻子。她的鼻子绝对是传说中的名品鼻子。

正面看是纯正的高鼻梁，侧面看鼻头非常尖，而且形成多头鼻。比起整体小巧的鼻子而言，郑秀金的这种自然、具有硬气、立体的鼻子显得非常吸睛。

我特别说一下，这两年我就发现很多的女生都是拿着郑秀金的鼻子照片来找我，希望做一个类似于郑秀金的鼻子。所以由此可见，很多女生的个人审美还是非常有品味的。

### M唇与整体气质

郑秀金是典型的M唇。她的嘴型薄而长，嘴角较尖，上唇的唇角弧度上扬，这才是天然的微笑唇。

不过也有人说，郑秀金的五官会给人一种深冷、冷淡的感觉。其实很大的原因就是她的五官比较英气。

如果有些许难理解的话，类似的颜值大家还可以参考我们台湾的女神林青霞。

## 脸型与发型修饰

分析完五官，我们再来看看脸型与整体轮廓如何影响气质。

### 脸型特点

我再说说她的脸型。郑秀金的脸型较长，下巴比较尖。侧脸看，面部轮廓确实不是特别流行这种线条。她的颧骨比较高，这是一个弱项。

### 发型的修饰作用

所以中分就比较适合她了。我个人认为这并不是很大的缺陷，可以用发型来做修饰。

有刘海的郑秀金多了甜美清纯的感觉，弱化了五官的英气。但是这种空气刘海也增加了她的甜美的气质。

我个人感觉，她的高冷、高贵冷艳势更加能够吸引到异性的目光。

## 总结：独特气质的构成

本节课中我们一起学习了如何分析郑秀晶的颜值特点。

郑秀晶出道以来一直都是一股清流一般的存在，眼神充满了不屑，一脸的生人勿近的气质。但是这样美艳的颜值却丝毫不妨碍粉丝对她的喜欢和追捧。

消瘦的脸颊、细长的双眼、线条清晰的鼻梁，眼神时常带有迷离的感觉。即使她有时候感觉像是摆了一张臭脸，但依然这样好看。

**我给郑秀晶的颜值打98分。**
