# 恋爱心态课 01：高段位男生的核心心态 🧠

在本节课中，我们将要学习高段位男生在情感互动中的核心心态。我们将剖析普通男生与高段位男生在思维模式上的关键差异，并帮助你理解如何建立更强大、更健康的情感观。

## 核心心态：内心强大与兼容

上一节我们介绍了课程主题，本节中我们来看看高段位男生的核心特质。

高段位男生在情感互动中最核心的特质是能够做到内心强大、宠辱不惊和上下兼容。他们不会像有的男生一样，持有“你不主动联系我，那我也不主动联系你”的对等心态。他们认为，男生主动这件事并不丢人，也不掉价。

在感情里，真正让你处于被动地位的，是你不断的内耗，以及等待对方联系你的那种弱者心态。

## 普通男生的情感误区

了解了高段位男生的心态基础后，我们来看看与之相对的普通男生常陷入的困境。

普通男生在等待女生联系你的过程中，会投入很多无效情绪。因此，这些男生普遍会有一个很大的情感误区，其逻辑可以概括为：

**公式：** 我的付出 > 对方的回报 → 产生自卑感与内耗

他们会认为“我为她付出了这么多，但是她什么都没有回报我”，进而产生一种非常强烈的自卑感，从而消耗自己的能量。

## 高段位男生的定义与表现

那么，高段位男生究竟是如何定义和表现的呢？这与数量无关。

高段位男生，绝不是指他列表里有多少女生，也不是指他交往过多少个女友。而是指即使他遇到了心动的女生，他也不会为这件事而产生内耗。

他们知道，感情这个事情是用来取悦自己的，而不是来找虐的。所以他们的心情和情绪都会非常稳定，会围绕着“怎么开心怎么来”的原则。

以下是他们的一个典型行为模式：

*   **主动联系：** 当高段位、具备强者思维的男生想要去主动交流情感时，他就会主动去联系。
*   **情绪收放：** 他这一秒可能还在与这个女生亲密互动，下一秒就可能马上投入到自己的工作中。他能够释放自己的情绪，也能够收得住、控制得住自己的情绪。

## 关键差异：强者思维 vs. 弱者心态

对比之下，弱者心态的普通男生，就会为了“要不要主动”这件事情反复纠结。这样一开始，你就完全处于被动地位了。

所以，普通男生一定要学习强者思维男生的心态。其核心代码可以描述为：

**代码：** `心态 = 成为“想爱就爱”的高位越级者`

## 总结与收获

本节课中，我们一起学习了高段位男生的核心心态。

我们剖析了普通男生因等待和计较回报而产生的内耗，也学习了高段位男生如何以取悦自我为中心，从容地掌控情绪的收放。关键在于培养“强者思维”，即让自己成为一个遵循内心、处于高位的决策者。你学会了这一点，再好看再优秀的女生，你也能够悠然自得地相处。
