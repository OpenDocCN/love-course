# 情感课程 01：AA制婚姻的风险与本质 💔

在本节课中，我们将通过一个真实的咨询案例，探讨AA制婚姻模式可能带来的问题与风险。我们将分析其背后的心理动因，并思考婚姻的本质。

---

昨天，一位女性向我咨询。她结婚近一年，丈夫在券商工作。婚后两人的钱从未放在一起，基本实行AA制。婚前双方各自购买了车和房，因此婚后也各花各的钱。

如果有共同开销，例如这次他付，下次就我付。家里添置小家电、沙发，甚至探望父母购买礼物，都采用AA制。日常开销如每周逛街购物，也遵循“上一次他结账，这一次轮到我”的规则。

听完她的叙述，我感到有些无奈。这种将日子过得如此“讲究”的夫妻，或许老了不会得老年痴呆，因为婚姻已变成一堂数学课，时刻锻炼着记忆力。

别的城市情况我不清楚，但在成都，近年来AA制婚姻确实不少。随着部分男性或女性经济地位的提升，AA制成了许多家庭的选择。他们号称“工资自持，债务自负，家庭开销按比例分摊”，追求彼此独立、和谐共处。

然而，必须认识到，在这种婚姻状态下，多数婚姻存在风险，离婚率也可能逐年上升。人会感觉婚姻不安全，因为只有钱才是最可靠的。这无异于将婚姻过成了一道数学题。

我认为，如果两个人已经结婚，AA制的生活方式非常不可取。当下出现大量AA制婚姻，根源在于对生活不确定性的担忧。人们会想：“我与你能否白头偕老？如果不能，现在在一起，AA制便于未来算清账目。”这也是一种自我保护的体现。

要我说，这位女性也有些天真。谁规定婚后必须与她过AA制生活？如果婚前就明确是AA制，我建议趁现在还没有孩子，尽早考虑退出。女性不必在这方面过分要强。我们不以真爱为幌子敛财，但应有的态度必须明确。

如果婚姻都要AA制，那结婚的意义何在？不如各自过日子。况且，婚姻中如何做到绝对AA？未来如果怀孕生子，难道让丈夫代怀五个月吗？进入产房后，他也能分担分娩的剧痛吗？剖腹产挨的一刀，他能帮忙承受半刀吗？孩子出生后，是父母各上一个月班，还是轮流照顾？哪家公司会如此“仗义”？

女性生育或多或少会影响事业，这是众所周知的事实。男性不应斤斤计较，像女性一样算计。自己的妻子不愿供养，难道指望别人来养吗？如果男性确实无力承担养育妻子或孩子的责任，我建议暂时不要结婚，以免让一个好姑娘受这份委屈。

家是什么？家是一个整体。**AA制夫妻往往意味着“大难临头各自飞”**。一个平时在经济上都算得如此清楚的人，在大是大非或大病大灾面前，女性有多少信心对方会尽心尽力救治、舍得为你花钱？难道要在ICU里，拿着计算器和你对账吗？平时生活尚且如此计较，面对巨额开销，他是否会掏钱？答案显而易见。

女性通常为家庭做出了更多隐性付出。如果严格实行AA制，女性的生育风险、生育成本、家务时间、牺牲事业发展机会等，都难以在AA制中得到有效体现。这样的AA制婚姻，真的公平吗？

婚姻的本质，是身体、感情和经济上的 **“你中有我，我中有你”** 。是共同创造财富，共同享受生活，共同抵御风险。

因此，AA制婚姻是否会幸福，我无法断定。但我确信，分得太清楚的婚姻，注定难以幸福。

---

本节课中，我们一起探讨了AA制婚姻的潜在风险。我们分析了其背后对不确定性的恐惧，并指出婚姻的本质在于融合与共担，而非精确分割。希望这些思考能帮助你更深入地理解亲密关系的构建。
