上一段我们提到了现象，本节中我们来具体看看沟通中常见的几个矛盾点。

1.  **邀约与回避**

3.  **态度与标签**
    *   **男性视角**：尝试以不同态度进行互动。

4.  **细节与整体评价**

    *   **核心矛盾**：局部的、偶然性的行为被上升为整体人格的否定，即 `“靓仔” -> “屌毛”`。
