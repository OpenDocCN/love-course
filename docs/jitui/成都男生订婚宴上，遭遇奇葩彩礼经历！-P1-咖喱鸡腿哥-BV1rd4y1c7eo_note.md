# 婚俗谈判避坑指南 💍 第一课：识别不合理婚俗要求

在本节课中，我们将通过一个真实案例，学习如何识别婚俗谈判中可能出现的、超出常规且不合理的财务要求。案例涉及一对准备结婚的情侣，在订婚宴上，女方家庭临时提出了额外的、基于女方年龄和体重的“身价钱”和“岁数钱”。我们将以此为例，拆解其中不合理的逻辑，帮助你建立基本的判断标准。

## 案例背景

成都一位男生与来自西北某地的女友恋爱两年，感情稳定，准备谈婚论嫁。男生对女方家乡的一些婚俗略有了解，并做了心理准备。订婚宴前，双方私下已就主要事项达成初步共识。订婚宴本意是让两家见面，敲定细节。

## 已商定的八项要求

以下是双方之前已基本谈妥的八项要求清单：

1.  **婚房**：一套120平米以上的全款精装修婚房，婚后需加上女方名字。（该房由男方购买）
2.  **彩礼**：彩礼金额为188888元。服装与婚纱照费用另计。
3.  **首饰**：需准备“五金”，包括戒指、耳环、项链、手链、手镯，总计价值约13万元。
4.  **居住安排**：婚后不与任何一方老人同住，女方强调“一天也不行”。
5.  **礼金与嫁妆**：婚礼收的礼金留下15万元，剩余38800元中，拿出2万元给女方的弟弟妹妹，最后的18800元用于购买被褥、化妆品等作为嫁妆。（这意味着女方家庭未从自身资产中支出嫁妆）
6.  **酒席**：按每桌2000元标准，需15桌以上，总计3万元，由男方承担。
7.  **红包**：包括“开箱”、“速洗”、“换衣服”、“送嫁妆”、“挤门”等各种环节的红包。大红包先准备2000元，婚礼当天再准备些小额红包应付。
8.  **其他礼金**：改口费10001元，下轿费10000元。男方家庭接受了这些要求。

男方父母有退休金，为让小两口过好日子，同意婚后在房产证上加名。以上各项总计花费约35万元。男方已全款购房，压力巨大，且已预付婚车、婚庆等前期费用。

## 订婚宴上的突发情况

上一节我们介绍了双方事先商定的条款，本节中我们来看看订婚宴上发生的意外转折。

正当男方家长准备按约定支付彩礼和各项费用时，女方母亲突然提出两项额外要求：“身价钱”和“岁数钱”。其计算方式基于女方的体重和年龄。

其**计算公式**为：
`总费用 = (体重 × 200) + (年龄 × 300)`

具体到本案：
*   女方当前体重为120斤。
*   女方年龄为29岁。
*   代入公式计算：`(120 × 200) + (29 × 300) = 24000 + 8700 = 32700元`

男方家庭听闻此要求后，脸色变得非常难看，认为这无异于“卖女儿”。他们没有当场答应或拒绝，表示需要回家商量。事后，男生从女友处得知，这个主意是女友外婆出的。男生最终咨询他人意见，并决定重新考虑这桩婚事。

## 核心问题分析与警示

基于以上案例，我们可以总结出几个需要警惕的信号：

以下是识别不合理婚俗要求的关键点：

1.  **临时加码**：在事先已达成详细协议后，于正式场合突然提出新的、重大的财务要求。
2.  **将人物化**：要求中涉及按**体重、年龄等生理指标进行明码标价**，这违背了婚姻基于感情与责任结合的初衷。
3.  **计算方式荒诞**：使用`(体重×单价)+(年龄×单价)`这类将人商品化的计算公式，缺乏基本的尊重与合理性。
4.  **单方面付出失衡**：在整个婚俗安排中，女方家庭未体现出对新建小家庭的物质支持（如嫁妆），要求高度集中于男方家庭的经济付出。
5.  **家庭决策影响**：不合理要求往往源于对方家庭中个别成员（如案例中的外婆）的提议，且其家庭成员可能普遍认同这种观念。

## 总结

本节课中，我们一起学习了一个因婚俗谈判引发矛盾的案例。核心在于识别那些**临时增加**、**将人物化**（例如用`体重×200 + 年龄×300`公式计价）、且导致**付出严重失衡**的要求。面对此类情况，需保持冷静，重新评估双方家庭价值观是否匹配。婚姻是平等的结合，任何使一方感到被勒索或商品化的要求，都值得慎重考虑。
