## 目标人群选择 🎯

上一节我们介绍了课程主题，本节中我们来看看“捞女”如何选择目标。

“捞女”专门挑选自卑且原则性不强的男生作为目标。不要认为自身条件普通就能避免成为目标。当前竞争激烈，“捞女”群体内部也存在内卷现象。

## 核心策略：如何“画饼” 🥞

上一节我们了解了目标人群的特征，本节中我们来看看“画饼”策略的核心。

“捞女”最擅长描绘诱人的未来图景（即“画饼”），并能让男生自愿接受。她们描绘的“饼”内容充实，能迅速让男生情绪高涨，并在事后仍回味无穷。

以下是具体的话术技巧与应用场景：

### 引导付出的话术 💎

当“捞女”希望男生付出物质时，她们不会直接索取。

低级话术示例：“反正以后结了婚就不分你我了。”（这是普通女生可能使用的直接方式）

高级话术示例：“这个系列的项链，拍婚纱照时戴应该很显贵气、很好看，颜色也很衬我们两个人。”

**公式**：`具体物品 + 关联未来共同场景（如婚纱照） + 赞美与双方的联系`

一句话同时规划了未来、提供了具体内容，且过渡自然。这给予了男生无限的想象空间，并引导他为想象中的未来场景付出，项链便可能顺利到手。

### 获取包容与让步的话术 👜

如果你希望男生多包容你、让着你，“捞女”会使用以下策略。

话术示例：“亲爱的，我这个人平时比较顽皮一点。你老是这么让着我、哄着我，有时候我觉得自己挺幸运的……”

**核心逻辑**：先承认自身“小缺点”，再表达对对方包容的感激。这种“褒义化”处理，让男生感受到“需要持续包容你”的情绪反馈。

接着可以顺势提出：“那个包包背在我身上，是不是我就能淑女一些，不那么顽皮了？”这样，包包的需求便被自然地提出并可能实现。

### 争吵后的安抚与引导话术 🚗

在双方吵架之后，“捞女”擅长将矛盾转化为未来承诺与物质引导。

话术示例：“亲爱的，其实每次吵完架我都特别难过，我也不想和你吵。你也知道我骨子里有时候很倔，冷静下来我就特别后悔。以后我们有了孩子，这样对孩子也不好。我也不是故意的，就是想让你多注意我一点，多陪我一点。我们以后就不要带着情绪过夜了。”

男生听后会觉得女方善于自我反省，容易自我感动。此时，许多男生会回应：“我以后在忙的时候，你要照顾好自己。下周我们去看看车，看到车你就等于看到我了。”

**代码逻辑描述**：
```python
if 发生争吵():
    表达难过与后悔()  # 降低对方防御
    关联未来家庭责任(孩子)  # 升华矛盾，指向共同未来
    表达真实需求(渴望关注与陪伴)  # 将争吵原因归结为“需要爱”
    提出共同约定(不带情绪过夜)  # 展示解决问题的诚意
    return 可能触发男生的补偿心理与物质承诺(例如看车)
```
这样，购车的计划便被引入。

## 策略的迭代与危害 ⚠️

上一节我们分析了具体话术，本节我们来总结其演变与最终目的。

当前的“捞女”策略已经迭代升级。过去可能是一味索取而不付出，现在则是通过给予对方关于美好未来、幸福婚姻、可爱孩子等的期许，来获取当下利益。

原则性不强的男生听完这些描绘后，容易情绪激动，误以为对方是能陪伴自己一生的人。但往往最终结局是人财两空。
