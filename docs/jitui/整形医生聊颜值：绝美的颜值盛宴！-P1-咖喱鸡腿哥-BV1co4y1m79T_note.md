# 颜值分析教程 01：五官结构与面部美学 🧑‍⚕️

在本节课中，我们将以艺人檀健次的面部特征为例，系统性地学习如何分析一个人的五官与脸型。我们将拆解面部各个组成部分，理解其如何共同构成整体的视觉印象。

## 整体脸型与轮廓分析

上一节我们介绍了课程目标，本节中我们来看看脸型的整体判断。檀健次的脸型属于长方脸，其轮廓具有以下特点：

*   额头平直，眉骨立体，带来英气感。
*   颧骨至下巴的线条流畅，下颌方正，骨骼感明显。
*   面部软组织适中，整体呈现柔中带刚、大气舒展的观感。
*   皮肤细腻，奠定了清爽干净的整体基调。

## 眉眼细节分析

了解了整体轮廓后，我们进一步聚焦于最具表现力的眉眼区域。以下是其眉眼的主要特征：

*   **眉毛**：粗直眉形。
*   **眼睛**：大外双，眼型偏圆，属于桃花眼。眼尾较长且上翘，眼神聚光。
*   **神态**：眼角偏钝，增添可爱感。笑起来卧蚕明显，显得眼神温暖有神。

其眼睛属于中等大小，但因明显的双眼皮而显得较大。公式可简化为：**视觉眼大小 = 实际眼大小 + 双眼皮加成**。

## 鼻部与唇部特征

眉眼决定了神情，而鼻唇则深刻影响着面部的立体度与气质。鼻部是决定五官立体度的核心。

檀健次的鼻梁高挺，但山根纤细。鼻翼不宽，鼻头精致，这些共同增加了鼻子的立体感，也带来了些许高冷的气场。其唇部特征如下：

*   嘴唇偏薄，唇线清晰（标准M唇）。
*   双唇抿合时显得精致，有时会流露出些许骄傲感。
*   笑起来时弧度甜美，极具亲和力，这种唇形让许多女性羡慕。

## 侧颜与整体氛围

最后，我们来整合侧脸与动态表情，完成整体氛围的评估。侧脸轮廓清晰，眉骨与山根立体，形成了优秀的侧颜线条。

结合其温柔的眼神、带酒窝的笑容，共同塑造出十足的青春少年感。综上所述，其五官搭配协调，颜值优越。在动态表演中，他展现出的天赋与灵气，为其整体形象增添了巨大魅力。

## 总结

本节课中，我们一起学习了从**整体轮廓**、**眉眼细节**、**鼻唇结构**到**侧颜氛围**的完整面部分析方法。以檀健次为例，我们看到如何将分散的五官特征综合起来，评估其构成的整体美感与气质。记住，面部分析是一个整体性工程，任何单一特征都需放在全局中考量。
