# 情感课程 01：识别需远离的恋爱对象 🚩

在本节课中，我们将学习如何识别在恋爱初期就应果断远离的女性类型。了解这些关键信号，可以帮助你避免在感情中浪费宝贵的时间、精力和情感。

上一节我们介绍了课程目标，本节中我们来看看具体的七种需警惕的行为模式。

以下是七种关键的危险信号：

1.  **初期借钱**：双方认识不久，对方便开始提出借钱。
2.  **持续哭穷**：经常在你面前抱怨经济窘迫，传播负面情绪。
3.  **工具化倾向**：将你视为工具人，随意差遣你做各种事情。
4.  **关系边界模糊**：身边存在大量所谓的“哥哥”、“弟弟”或“男闺蜜”，关系纠缠不清。
5.  **单向索取**：在关系中只知索取，从不付出。有事时才出现，无事时则消失，且丝毫不能吃亏。
6.  **态度若即若离**：对你的态度冷淡且不稳定，让你明确感到自己对她而言可有可无。
7.  **来者不拒**：对你的礼物照单全收，对你的邀约也从不拒绝，但关系并无实质进展。

无论对方外表多么出众，一旦发现上述问题，都应果断撤离。这类行为模式往往反映出其人品或原生家庭及成长环境存在深层问题。

本节课中我们一起学习了七种在恋爱初期就应警惕并远离的行为特征。识别这些信号是保护自己免受情感伤害的重要第一步。
