# 课程一：面部美学分析基础 🧑‍⚕️

在本节课中，我们将要学习如何从专业的面部美学角度分析一个人的颜值。我们将以艺人罗渽民为例，拆解其面部结构、五官比例与气质特点，帮助你理解构成“神仙颜值”的关键要素。

## 概述：什么是“三庭五眼”？

上一节我们提到了面部美学分析，本节中我们来看看其基础理论——“三庭五眼”。这是衡量面部比例是否协调的经典标准。

罗渽民的三庭分布均匀，五官比例恰当。由于年龄小，他脸蛋的胶原蛋白十分充足。面部的线条干净、流畅、利索，没有杂乱的骨骼瑕疵。

## 五官与轮廓分析

了解了基础比例后，我们进一步分析具体的五官和轮廓特点。以下是罗渽民面部特征的详细拆解：

*   **头肩比与颈部**：细长的脖子配上宽平肩。宽平肩的优势在于可以优化头肩比，让头脸显得更加小巧，更符合当下的审美。
*   **眼睛**：他长着一双杏眼，内眼角圆润，拥有扇形的双眼皮，这放大了他的眼睛。尖尖的外眼角配上眼尾微微下垂，形态类似一些无辜的小狗。眉眼间距非常开阔，给人心胸开阔、非常开朗的观感。
*   **卧蚕与瞳仁**：他拥有漂亮的卧蚕，从眼头到眼尾都有，瞳仁下方最为饱满，这使得他的眼形更加圆润。眼睛黑白分明，深色的瞳仁会显得更加温柔。这真是一双迷人又多情的眼睛。
*   **眉骨与鼻子**：罗渽民的眉骨很高，但他很少显露出来，所以大家一般不会注意到。鼻子上的肉感多于骨感，鼻梁有一定的高度。有一个漂亮的弧线连接眉心和鼻梁。
*   **下巴与唇部**：他的帅气还得益于有一个小巧且微微上翘的下巴。笑起来的时候，下巴会拉长一点点。唇边的两道笑弧（也叫“猫咪纹”），笑起来会特别吸引人。短而深的人中显得非常幼态，连接上唇的唇珠。
*   **笑容**：再加上一笑就露出来的洁白牙齿，真是一个好看的帅弟弟。

## 气质与可塑性

分析完静态的五官，我们来看看动态的气质与整体可塑性。这对于整体魅力的评估至关重要。

罗渽民出道时间早，所以大家很幸运，能够从他身上看到少年感的演化过程：从幼齿到清冷，气质变得成熟，扮相变得精致，可驾驭的风格也越来越多。他本身干净帅气的底色没有丢失，所以他的可塑性很高，各类发色都能驾驭。

## 总结与评分

本节课中，我们一起学习了如何从“三庭五眼”的比例、具体五官特征（眼、鼻、唇、下巴）、头肩比以及气质演变等多个维度，系统地分析面部颜值。

我给他的颜值打分是 **98分**。
