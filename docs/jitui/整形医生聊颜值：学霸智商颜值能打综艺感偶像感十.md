# 颜值分析教程 01：如何系统分析面部特征 🧑‍🏫

在本节课中，我们将以训练生唐九州为例，学习如何系统、客观地分析一个人的面部颜值。我们将拆解面部各个部位的特征，并理解这些特征如何共同构成整体的气质与风格。

## 概述：认识分析对象

首先，我们简要了解分析对象。唐九州是《青春有你3》的人气训练生，以开朗、温暖、聪明且帅气的学霸形象为人所知。他此前参与过芒果TV的自制剧，在青3舞台上的出现令人意外。他的颜值具有多面性，在舞台上冷静锐利，在综艺中则显得精致可爱。

## 五官细节分析

上一节我们概述了分析对象的基本情况，本节中我们来看看他面部各部位的具体特征。我们将从眼睛、鼻子、嘴唇等核心部位逐一拆解。

### 眼型与眉眼

他的双眼皮隐约可见，眼尾开裂且内眼角形态分明。这种眼型含蓄且具有少年感，符合古典中国审美，并且非常传情。眼睛能传达丰富情绪，这与他的成熟度和性格有关。

以下是眉眼部分的其他特征：
*   **卧蚕**：非常温柔。
*   **眉毛**：眉峰朝上，画得非常高，非常美。
*   **眉骨**：高眉骨，立体度好。

### 鼻形分析

他的鼻形走向刚性有力，鼻骨分明，鼻梁如同雕塑般存在。但正面观察，鼻尖有些过长。侧面看，鼻梁高度尚可，但山根与眉骨高度的衔接不够特别流畅。这两处不够精致的地方，反而增加了他的温柔气息。

### 唇形与下颌

他的唇形非常标准，是典型的M唇，上唇薄于下唇，下唇形状流畅均匀，唇珠也非常明显。需要指出的是，男生的唇肉过厚会显得油腻，而唐九州的嘴唇厚薄适中。

以下是下面部轮廓特征：
*   **下颌折角**：分明，轮廓清晰。
*   **下颌线**：非常流畅。
*   **酒窝**：一对招风酒窝非常吸睛。

## 整体气质与风格总结

在分析了各个面部特征后，我们现在来综合评估其整体气质。他不笑的时候，给人安静、认真甚至冷冷的感觉；一笑起来则眉眼弯弯，温柔的张力感染人心。他身上结合了少年感与男人味，冲突感与和谐感体现得淋漓尽致。

他的气质独特且具有多面性。这种高级的多面气质，能驾驭多样化的硬照拍摄。因此，他这款颜值艺人在未来时尚圈可能会非常受欢迎。虽然已经出道，但他依然年轻，拥有无限潜力。在青3节目中，可以看到他的坚毅、纯真、温柔和细腻。

## 课程总结

本节课中，我们一起学习了如何系统地进行颜值分析。我们以唐九州为例，逐步拆解了其眼型、鼻形、唇形及下面部轮廓的特征，并探讨了这些特征如何共同塑造其多面化的整体气质与风格。最终，我们给出了一个综合性的颜值评分：**93分**。
