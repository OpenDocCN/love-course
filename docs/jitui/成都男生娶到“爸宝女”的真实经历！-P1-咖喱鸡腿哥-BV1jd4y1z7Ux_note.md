## 案例背景介绍

一个成都的男生之前娶到一个“爸宝女”，婚后才发现娶到“爸宝女”真的是痛不欲生。这位兄弟朋友们都叫他K仔，今年29岁。他和“爸宝女”的婚姻就维持了一年多的时间。对于这段失败的婚姻，他觉得对他来说就是一场噩梦。

## 关系建立与初期迹象

上一节我们介绍了案例的基本情况，本节中我们来看看这段关系是如何开始的。

K仔和他前妻是自由恋爱认识的。交往的过程中他就发现，这个女生平时爱耍小脾气，但是他也没有过分地放在心上。他觉得恋爱中的女人都应该是这个样子，结婚以后踏踏实实过日子就行了。

婚礼上K仔的岳父拉着他的手对他说：“我现在把女儿交给你了。我是男人你也是男人，我宠她20多年，剩下的日子就交给你。”按理说，对妻子好是每一个已婚男人的责任，K仔愿意承担这样的责任，他答应得很痛快。没想到这句话就成了他婚姻噩梦的开始。

## 婚姻中的矛盾模式

了解了关系的开端后，我们进入婚姻内部，观察其中反复出现的矛盾模式。

结婚以后，这个女生还是和婚前一样爱耍小脾气。两个人之间只要有矛盾，不管对错，都必须是男生低头认错，不然两个人一定会冷战。

以下是这个女生最常说的几句话：
*   “我长这么大了，爸妈都没有对我说过什么。”
*   “我凭什么被你欺负。”

所以，在这个女生的眼里，**承认自己的错误 == 被欺负**。但是K仔希望让这个女生明白，她不仅是父母的女儿，还是一个妻子，未来还将是一个母亲。而且，幸福的婚姻需要的是感情上的**双向奔赴**。一味的妥协只会让之后的生活造成很大的困难。

## 外部干预与问题升级

当内部沟通失效时，矛盾往往会寻求外部出口。我们来看看这个案例中发生了什么。

所以之后几次，这个女生的无理取闹，哪怕两个人冷战，K仔也不认错。本来那个男生想的是让她冷静下来，想一想看清现实。但是没想到这个女生把矛盾，就直接延伸到了娘家。一通诉苦之后，女生的父亲把电话打来质问了，说结婚的时候你答应的好好的，你得宠着我女儿。

因为岳父发话了，K仔只能忍着主动求和。所以这样的事情，一年之内发生了好几次。

以下是女生形成的错误问题解决逻辑：
1.  与丈夫发生矛盾。
2.  立即打电话向父亲诉苦。
3.  父亲介入，向丈夫施压。
4.  丈夫被迫妥协。

女生把“有矛盾就找爸爸”这样的观点，当成了解决家庭矛盾的**资深法宝**。所以这个妹子就觉得，自己能够拿捏住男生。但实际上，这样一次又一次、不分青红皂白的退让和妥协，已经让这个男生对婚姻的希望破灭了。

## 关系破裂的导火索

不断的妥协积累着失望，最终，一个具体事件成为了压垮骆驼的最后一根稻草。

后来两个人关系完全破裂的导火索，是因为两个人攒了十几万块钱，打算买台车。结果这个女生在没有经过男生同意的情况下，擅自用了**几万块钱**，买了一个包和一些珠宝首饰。所以两个人为此吵了起来。

这哥们倒不是心疼钱，只是觉得这个女生买东西之前没跟他商量，非常气愤，而且现在暂时车也买不了了。但是这个女生就觉得，这个男生就是心疼钱。她认为：“如果说自己的父亲，是绝对不会因为这样的事情而埋怨自己的。”于是她又给她爸打电话。

这个女生的父亲在电话里面就对这个女婿说：“我这个女儿是宠大的，他发脾气你就得顺着。不就是一个包和一些珠宝首饰吗？你要是买不起，我把钱还给你。”

## 反思与关系终结

面对最终的局面，当事人进行了彻底的反思，并做出了决定。

所以听到这样的话，这哥们是直接崩溃了。结婚一年多时间，所有的矛盾都是必须要让他认错。这个女生完全就是父母宠大的这么一个主，是对方父母心中的宝，日常生活中各种惯着。在是非对错上，自己是一直妥协。那么这样的婚姻，又有什么意义呢？所以也就一年多的时间，这两个人就离婚了。

## 案例总结与启示

本节课中我们一起分析了K仔的真实经历。所以这就是一个成都的男生娶了“爸宝女”之后的真实经历。我们不去站立场评判谁对谁错，只是想通过这个事儿，提醒各位：如果你在关系中遇到了类似“爸宝女”的伴侣，你需要保持警惕。

健康的婚姻关系建立在**相互尊重、有效沟通和共同承担**的基础上，而非单方面的、无底线的妥协。**无底线的妥协，再大度的男生也受不了。** 认识并设立关系的边界，是维护长期亲密关系健康的重要一课。
