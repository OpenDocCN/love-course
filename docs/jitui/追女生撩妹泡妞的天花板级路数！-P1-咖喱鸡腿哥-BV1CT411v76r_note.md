# 追女生教程 第1课：掌握心理博弈的核心 🎯

在本节课中，我们将要学习追求异性时一种高级的心理策略。其核心并非传统的“真诚”路线，而是通过制造不确定性来吸引对方的注意力，从而掌握关系的主动权。

## 核心原则：避免过度真诚

追女生的至高境界，是不要让对方觉得你过于真诚和老实。你没有听错。为什么这么说？因为这些女生有备胎和鱼塘，她们难道不知道谁暗恋她、谁喜欢她、谁对她好、谁在追求她吗？她心里能感受不出来吗？

相反的，谁在敷衍她，谁在和她逢场作戏，谁在给她画大饼，她能不清楚吗？现在的女生聪明得很，心里跟明镜似的。所以你那些小心思，她很快就能判断出来。

## 心理博弈的本质

人千万不要做自作聪明的事情，让人家看你的笑话。但是，你又要追妹子，又要撩她，怎么办？你的分寸感和边界感在哪里？所以高手们玩这一套就很有办法。这就跟打扑克、打麻将、打游戏一样。

要让对方觉得真真假假、假假真真，永远让对方猜不透你。他才会觉得逢场作戏是降与良才，互相过招、猜招才有意思。他会感受到你是个有趣的人才，愿意陪你唱这出戏。相比之下，其他那些一味讨好的男生就显得很幼稚。

## 关键操作：制造不确定性

记住，重点是要让对方猜不透你，让他一直怀疑、幻想加沉醉。他心里会一直琢磨：“这人是不是喜欢我呀？”“如果喜欢我，为什么不跟我表白呀？”“怎么前几天那么热情，这几天又突然这么冷淡啊？”

他老是琢磨你的心思，他的**沉没成本**就变多了，人就上头了。主动权就开始慢慢向你这边倾斜，你的机会自然就多了。

## 爱情的比喻：酒与水的艺术

爱情是酒，实在是水。他想喝酒，你不能洒石头，也不能直接给他水喝。重点是要把这杯酒酿好，让他觉得好喝、还上头、还想续杯、想一直喝。这才是女生爱上你的秘方。
