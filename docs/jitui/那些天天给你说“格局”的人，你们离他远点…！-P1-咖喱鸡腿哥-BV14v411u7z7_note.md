# 格局课 01：警惕“格局”话术 🚨

在本节课中，我们将要学习如何识别一种常见的社交话术——“格局论”。许多人喜欢把“格局”挂在嘴边，但他们的真实意图可能并非如此。我们将剖析这种言论的本质，并提供清晰的应对思路。

## 现象：泛滥的“格局”讨论

如今，许多人动辄谈论“格局”。听者常常感到困惑。晚上出去喝酒吃烧烤，一桌男人互相谈论“格局”，但实际并无实质内容。一晚上能听到无数遍“格局”，令人厌烦。说这些话的人，通常自身并无真正的格局。

## 本质：单向的道德绑架

所谓的“格局”往往是一种不对等的要求。你可以随意对待我，但我不能要求你。你可以戏弄我，但我不能与你计较。一旦我与你计较，就显得我没有格局。你可以让我吃亏，可以冤枉我，但我却不能以同样的方式对待你。否则，就是我没有格局。这本质上是一种单方面的道德约束。

以下是这种逻辑的核心表现：

*   **标准不对等**：对方可以做的行为，你却不能做。
*   **责任转嫁**：你的正当反应会被定义为“没格局”。
*   **利益牺牲**：要求你单方面承受损失。

## 应对：远离空谈“格局”者

那些天天在你面前说“格局”的人，你应该离他们远点。哪天他遭遇不幸，甚至可能怪罪是你造成的。喜欢用“格局”说事的人，通常自身并无格局。他们只是利用“格局”这个虚无缥缈的概念，来压制你、给你洗脑，将自己置于道德高地，从而掩盖问题的真正矛盾。

你所说的“格局”，往往只是别人没让你占到便宜时的抱怨。自身基础不牢，却好高骛远地谈论格局，是毫无意义的。

## 特征：识别这类人的常见表现

通过观察，这类人通常具备以下特征：

*   **财务状况**：存款可能不足几万元，却觉得百万都是小钱。
*   **收入认知**：自己月薪几千，却总想寻找月入过万的对象。
*   **消费观念**：自己连十万元以下的车都买不起，却看不上宝马、奔驰、奥迪（BBA）等品牌。
*   **自我感觉**：自身并无实质成就，却充满莫名的优越感。

他们整天把“格局”挂在嘴边，对他人指点江山。这样的人，或许一直活在自己的幻想里。

---

本节课中我们一起学习了“格局”话术的虚伪性。我们剖析了其作为单向道德绑架的本质，并总结了空谈格局者的常见行为特征。核心在于认识到，真正的格局体现在行动与实力上，而非用于要求他人的口头禅。面对不断用“格局”施压的人，保持距离是明智的选择。
