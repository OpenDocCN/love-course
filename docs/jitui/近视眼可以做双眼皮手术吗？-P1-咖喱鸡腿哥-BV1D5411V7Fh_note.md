# 近视眼可以做双眼皮手术吗？-P1-咖喱鸡腿哥-BV1D5411V7Fh_note

在本节课中，我们将要学习近视眼手术与双眼皮手术之间的关系，明确这两种手术是否可以同时或先后进行。

上一节我们介绍了课程主题，本节中我们来看看具体的解释。

近视眼手术是作用于眼球内部以矫正屈光状态的手术。双眼皮手术则是在眼睑外部进行的整形手术。这两种手术的操作部位完全不同。

以下是两种手术互不影响的核心原因：
*   近视眼手术（如LASIK）的**操作区域**在**角膜**或**眼内**。
*   双眼皮手术的**操作区域**在**眼睑皮肤**和**睑板**。

因此，从医学原理上讲，这两种手术互不干涉。

你可以选择同时接受这两种手术，也可以根据个人安排，先进行其中一项，稍后再进行另一项。手术的先后顺序不会对彼此的效果或安全性造成影响。

本节课中我们一起学习了近视眼手术与双眼皮手术的兼容性。关键结论是：由于两者作用部位不同（眼球内部 vs. 眼睑外部），因此可以安全地同时或分次进行，互不影响。
