# 课程名称：韩流美学分析 - 第一讲：裴珠泫的“神颜”解析 👑

在本节课中，我们将要学习如何从面部结构和美学风格的角度，分析被誉为“南韩女爱豆第一神颜”的裴珠泫（Irene）的颜值。我们将拆解她的面部轮廓、五官特点以及独特的个人气质，以理解其“神颜”称号的由来。

## 概述：裴珠泫的颜值地位

裴珠泫入选了2020年亚太区最美的100张面孔。这个消息对其粉丝而言是意料之中的事。她的“出圈神图”数不胜数，基本达到了日常露脸即上热搜的地步。

从出道至今，她的造型基本没有失误，能够轻松驾驭各种风格，并成功收获了“南韩神颜”的称号。

## 第一节：面部轮廓分析

上一节我们提到了裴珠泫的广泛认可度，本节中我们来看看她面部轮廓的基础。她的面部轮廓几乎是完美的鹅蛋型。

从太阳穴到颧骨，再到下巴的线条都非常流畅。**发际线到眉毛的距离**与**鼻子到下巴的距离**几乎相等，这构成了完美的面部比例。

她的脸蛋属于整体均衡的鹅蛋脸，没有任何一部分存在明显的瑕疵。

## 第二节：五官细节解析

在了解了完美的轮廓基础后，我们进一步分析她的五官细节。她的眼睛内眦略微明显，加上上行的双眼皮，这是符合东方韩系审美的特征。

内眼角略微下勾，但整体眼型走势是向上的。比起华丽的眼睛，她的鼻子形状更显自然。

从正面看，鼻头很圆，带有可爱的感觉。从侧面看，鼻梁线条**高耸且鲜明**，鼻尖略长，但并未夸张凸出，因此显得既可爱又成熟。

她的嘴巴大小非常标准。鼻翼宽度与嘴角宽度的比例接近 **`2:3`**。唇形属于薄唇，且上唇略薄于下唇。

## 第三节：独特的个人气质

然而，我认为她能够成为“神颜”的原因，还在于她有一股“世代女团”中独一无二的清冷淑女神韵。

韩国世代女团的成员年龄普遍偏小，风格以娇俏可爱、元气满满为主。唯有年龄偏大的裴珠泫独占了这份“清冷”气质。

因此，她日常“出圈”的神图，几乎都是走这种不食人间烟火的风格。

## 总结与评分

本节课中我们一起学习了裴珠泫的面部轮廓、五官比例以及独特的清冷气质，这些因素共同构筑了她的“神颜”地位。

综合以上分析，我给裴珠泫的颜值打 **98分**。
