# 整形医生聊颜值 第1课：张极的颜值分析 🧑‍⚕️✨

在本节课中，我们将从专业整形医生的视角，系统分析TF家族练习生张极的颜值特点。课程将遵循“三庭五眼”、“四高三低”等面部美学标准，逐一拆解其五官与轮廓的细节。

---

## 正脸分析：三庭五眼 📐

上一节我们介绍了课程概述，本节中我们来看看正脸的分析标准——“三庭五眼”。这是衡量面部比例均衡性的经典方法。

以下是张极正脸符合“三庭五眼”标准的具体表现：
*   他的面部横向与纵向比例协调，符合“三庭”（发际线到眉间、眉间到鼻底、鼻底到下巴）大致等分的原则。
*   面部宽度在眼睛水平线上大约为五只眼睛的长度，符合“五眼”的比例，这奠定了其端正的基础。

## 侧脸分析：四高三低 📏

分析完正脸比例，我们转向侧脸轮廓。侧脸美学通常关注“四高三低”，即四个高点与三个低点构成的起伏线条。

以下是张极侧脸在“四高三低”上的体现：
*   **四高**：他的额头、鼻尖、唇珠与下巴尖较为饱满突出，形成了立体的侧脸线条。
*   **三低**：眉额交界处、人中沟以及下唇下方的凹陷清晰，这使得面部轮廓富有层次感，而非扁平。

## 五官精致度分析 👁️👃👄

了解了整体轮廓，我们进一步聚焦于五官的细节。精致度是评价颜值高低的关键维度。

**眼睛**：张极的眼睛略细长，拥有标准的扇形双眼皮。内眼角开阔，眼角与眼尾均呈尖细状，眼尾微垂。双眼皮褶皱丰润，眼型如同弯月，充满少年感与水润感。其眉骨不低，因此眼窝显得深邃，即使静态时眼神也富有情感传递力。

**鼻子**：他的鼻梁挺拔，但相较于攻击性强的男性鼻型，其鼻子偏柔和。眉骨到山根的角度过渡自然，属于优异的直挺鼻。鼻梁呈直角下延，中段无突兀起伏，鼻翼宽度适中。无论是正面的海鸥线，还是侧面的鼻梁高度、鼻尖表现点与鼻基底，都非常优秀。

**嘴巴**：他的嘴型小巧，嘴唇偏薄，显得十分秀气。唇部线条精致，唇珠特别明显。

## 面部轮廓与整体骨相 💎

在细致剖析了五官之后，我们来观察支撑这些五官的“地基”——面部轮廓与整体骨相。

张极的面部轮廓利落硬朗，下颌线清晰，拥有优越的“太平洋宽肩”和身形。其骨相在细节精致的同时，整体呈现出有力量感的偏男性化特征。高耸上扬的眉骨、存在感强烈的颧弓与颧骨，都为他未来转向更阳刚的路线提供了支撑。

## 总结与评价 📝

本节课中我们一起学习了如何用专业美学标准分析张极的颜值。总结如下：
1.  **比例**：正侧脸分别符合“三庭五眼”与“四高三低”的基准。
2.  **五官**：眼、鼻、嘴的细节处理非常精致，兼具柔美与立体感。
3.  **骨相**：整体轮廓硬朗，骨相基础扎实，具备向成熟阳刚风格发展的潜力。

综合以上分析，张极的颜值评分为**99分**。
