# 护肤基础课 01：面膜的正确使用频率 🧖‍♀️

在本节课中，我们将要学习面膜的核心功效以及科学的使用频率。了解这些知识能帮助你更有效地护肤，避免因不当使用而损伤皮肤。

## 面膜的核心功效 💧

上一节我们提到了课程主题，本节中我们来看看面膜的主要作用。大家需要明确一点，面膜最基本且主要的功能是**为皮肤补充水分**。

其作用原理可以简化为一个基础公式：**面膜 = 密集补水**。它通过暂时性的密封环境，帮助皮肤吸收水分和精华成分。

## 为何不建议每天敷面膜 ⚠️

了解了面膜的补水本质后，我们来看看过度使用的潜在风险。如果每天都使用面膜进行密集补水，可能会对皮肤屏障造成负担。

长期每天敷面膜可能导致两个主要问题：
*   **皮肤屏障受损**：过度水合可能削弱皮肤自身的保护能力。
*   **诱发敏感肌**：屏障功能下降后，皮肤更容易受到刺激，变得敏感。

## 科学的面膜使用指南 📅

认识了潜在风险，接下来我们学习如何安全有效地使用面膜。遵循以下建议，可以让你在获得补水好处的同时，最大限度地保护皮肤健康。

以下是具体的操作建议：

1.  **频率**：每周使用**2至3次**面膜最为适宜。
2.  **时长**：每次敷面膜的时间应**控制在10分钟以内**，不宜过长。
3.  **后续护理**：敷完面膜后，建议涂抹乳液或面霜来锁住水分。

本节课中我们一起学习了面膜的基础知识。我们明确了面膜的核心功效是补水，理解了每天敷面膜可能带来的风险，并掌握了每周2-3次、每次不超过10分钟的科学使用方法。正确护肤，才能让肌肤保持健康状态。
