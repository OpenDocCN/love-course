# 颜值分析教程 01：如何解析周柯宇的“桃花眼”与面部结构 🧐

## 概述：周柯宇的面部基调

周柯宇的五官非常秀气，属于能让人一眼就注意到的类型。他的五官形状都一致的清秀漂亮。

## 第一节：迷人的“多情桃花眼” 👁️

上一节我们概述了周柯宇清秀的面部基调，本节中我们来看看他最具特色的眼睛。

尤其是他的眼睛，眼尾微微下垂，看起来有一种柔美感。眼神传递到位，眼睛的走势非常流畅。

以下是构成其“桃花眼”的几个关键要素：
*   **双眼皮**：宽窄适中。
*   **臥蚕**：饱满。
*   **眼角**：带有笑意，弧度微微上跳。
*   **瞳仁**：乌黑湿润。

综合以上特征，这是一双多情的桃花眼，笑起来非常迷人。

这属于非常精致的鼻型，而且鼻子整体很小巧。他的脸原本就非常小，再搭配上这样的鼻子正好合适。
