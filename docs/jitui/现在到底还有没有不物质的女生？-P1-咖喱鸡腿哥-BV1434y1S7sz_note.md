# 恋爱攻略课 01：如何识别与接近“文艺女生”🎨

在本节课中，我们将要学习如何识别一类被称为“文艺女生”的群体，并了解与她们交往的核心策略。许多男生抱怨在恋爱中感到疲惫和物质压力，而这类女生可能提供一种不同的交往模式。

## 什么是“文艺女生”？🤔

上一节我们提到了恋爱中的普遍困扰，本节中我们来看看一种特殊的解决方案。文艺女生，也被称为文艺青年，广泛分布在城市中。她们通常具有多愁善感、优柔寡断、缺乏行动力以及眼高手低等特征。
