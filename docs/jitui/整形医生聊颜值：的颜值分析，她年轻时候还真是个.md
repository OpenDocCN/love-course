# 颜值分析教程 01：贾玲的颜值变迁分析 👩‍⚕️

在本节课中，我们将学习如何系统性地分析一个人的颜值变化。我们将以喜剧明星贾玲为例，通过对比她年轻时期与当前的外貌特征，来理解影响颜值的核心要素，如脸型、五官、身材等。分析将遵循客观、结构化的方法，避免主观臆断。

---

上一节我们概述了本节课的目标，本节中我们来看看贾玲年轻时期（变胖前）的面部基础条件。

全脸是没有任何的明显硬伤的。长得最好的是脸型和额头。她的脸型是属于较大一类的瓜子脸。颧骨、颧弓还有下颌骨发育得比较适当，线条是非常的流畅。还有一个很明显的特点就是她的尖下巴。直到现在，胖起来的贾玲她的尖下巴还都存在。她的额头的长度饱满度，当年也是非常可以的。发际线在我看来也不错。

五官也是非常的端正。脸部线条流畅。身材十分的纤细，身材比例也很棒。面部轮廓没有明显的硬伤。嘴角两个浅浅的梨窝，笑起来非常甜美。我以前看了一个采访是关于贾玲，有人说过，她在年轻的时候是一个美人，这么一看还真是这个样子。

不过现在的贾玲和以前的她，做个对比变化确实太大了。所以还是印证了那句俗话：女生一胖就是毁所有。所以春天来了，我奉劝大家把身上多余的脂肪，都该吸的吸掉。

在了解了她的整体优势后，接下来我们再看看年轻没长胖时候的贾玲面部存在的一些具体瑕疵。

*   **眉眼**：介于浓淡之间。眉毛浓密，眼睛还算有神，但是眼睛不算大。而且形状不够优美，内双眼皮不够秀气。有那么一些大小眼以及眼距过近的问题。眼睛的漂亮与否是别人对你的第一感觉，尤其是女生一定要想办法把自己眼睛变得漂亮。

*   **嘴形**：嘴形的形状比较漂亮，但是上嘴唇太厚，和她的五官面部都不太搭配。
