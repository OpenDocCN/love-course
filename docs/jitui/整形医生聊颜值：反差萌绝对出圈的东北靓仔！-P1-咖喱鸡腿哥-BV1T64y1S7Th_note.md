# 颜值分析教程 01：张嘉元的骨相与五官解析 🧑‍⚕️

在本节课中，我们将学习如何从专业角度分析面部结构。我们将以艺人张嘉元为例，拆解其骨相与五官的构成特点，帮助你理解“精致”与“立体”在面部美学中的具体体现。

---

## 一、 整体骨相分析 💀

上一节我们概述了课程目标，本节中我们来看看张嘉元的面部基础——骨相。骨相是面部结构的根基，决定了脸的轮廓和立体度。

张嘉元的五官线条比较精致，形状极佳。  
他是窄五官中的精品。  
其骨骼饱满立体，整体起伏度不错。  
额骨饱满宽阔，颧骨线加分。  
T区（眉骨与鼻骨区域）与高额头承接自然，漂亮立体无需多言。  
正面脸型过渡自然流畅，一气呵成，无凸起无凹陷，自上而下收窄。  
饱满的面中部与高鼻基底意味着脸上挂得住肉，非常抗老。  
下颌骨呈方形且有折角，折角收得高，缓解了下巴内缩的钝感。  
他的骨相在娱乐圈中属于中上等，是五星好评的骨相。

---

## 二、 五官细节拆解 👁️👃👄

了解了优秀的骨相基础后，我们进一步分析具体的五官。五官的形态和搭配决定了面部的风格与气质。

他的长相更加精致，拥有小头小脸和优越的头身比。  
以下是其五官的具体特征：

*   **眉毛**：眉毛偏平直，走势不大。
*   **眼睛**：眼睛是魅惑的柳叶眼，下眼睑曲度大，自带眼睑下至效果，同时眼尾微微上扬。眉眼平和，眼神清澈，有轻微的下三白。眼波柔软，既勾人又无辜，属于软萌无害型。
*   **颌面**：颌面规整，是标准的直面形，显得清正贵气。
*   **鼻子**：鼻骨稍粗，增添了不同的英气。他的鼻子山根高，与眉骨之间没有明显的凹陷。鼻梁挺直，鼻骨和鼻软骨之间，有自然的微微转折。鼻头是方中带圆，与鼻翼处有自然的衔接线，不锋利也不钝。即使是大笑，鼻翼也不会随着表情发生很大变化。这是一个难得的既英俊又有福气的鼻子。
*   **嘴巴**：嘴巴精致度也很高，唇峰明显。自然状态下唇形是平直的，代表的是理性和克制。

---

## 三、 风格总结与评分 📊

综合以上对骨相和五官的分析，我们现在可以总结张嘉元整体的面部风格和给人的视觉印象。

他本身有贵公子气息，骨骼精致，线条走向干净。  
一双柳叶眼拉近了距离感，但低眉垂眼的时候，又带来了一丝脆弱感，让人很想靠近。  
基于以上分析，我给张嘉元的颜值打 **95分**。
