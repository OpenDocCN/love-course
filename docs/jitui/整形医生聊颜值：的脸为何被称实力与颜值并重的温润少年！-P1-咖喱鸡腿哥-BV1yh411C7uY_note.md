# 颜值分析教程 01：张真源的温润少年感解析 🧑‍⚕️✨

在本节课中，我们将学习如何从专业的面部美学角度，分析时代少年团成员张真源的颜值特点。我们将拆解其面部轮廓、五官细节与整体气质，理解其为何被称作“实力与颜值并重的温润少年”。

## 面部轮廓与第一印象 🌟

上一节我们概述了本课内容，本节中我们来看看张真源的面部轮廓。他的面部轮廓不带任何攻击性，给人以秀气的感觉。这样的轮廓让人赏心悦目。

## 眉眼分析：会说话的眼睛 👀

了解了柔和的面部轮廓后，我们进一步分析其五官。眼睛是心灵的窗户，张真源的眉眼极具表现力。

以下是其眉眼的具体特征：
*   **眼型**：眼睛是舒展的大平行双眼皮，眼角微微上挑。瞳仁位置适中，位于眼睛中部，无上三白或下三白现象。
*   **神态**：眼睛笑起来像月牙，眼尾细长上扬，能用眼睛传达感情。
*   **眉毛**：眉形秀气，眉毛长而翘，墨长饱满。眉骨弧度佳，衬托出额头饱满与眼神深邃。
*   **辨识度**：眉毛在组合中很有辨识度，无论皱眉或挑眉都很有表现力。

## 鼻部美学：面部的立体担当 👃

分析完传神的眉眼，我们转向面部的核心——鼻子。鼻子的形态直接影响面部的立体感和气质。

以下是其鼻部的美学特点：
*   **鼻梁与山根**：鼻梁挺直，山根挺立，增加了面部立体感，让气质清爽。
*   **鼻头与鼻翼**：鼻头小巧，鼻翼宽度适中。鼻翼宽度与眼距大致相当，比例协调。公式表示为：**协调比例 ≈ 鼻翼宽度 ≈ 眼距**。
*   **整体评价**：鼻子在面部整体中显得柔和、和谐、舒展。

## 唇形、下颌与整体气质 💋

在剖析了核心五官后，我们最后来观察唇形、下颌线等塑造整体气质的关键部分。

以下是其下庭与皮肤的特点：
*   **下颌线**：下颌线非常流畅。
*   **唇形**：拥有像猫咪一样的薄嘴唇，使下庭显得无辜可爱。自然状态下嘴角略微上翘。
*   **皮肤**：皮肤白皙，肤质健康。
*   **整体气质**：结合其勇敢、善良、温柔、努力的性格与优质声线，将温润如玉的少年纯真感展现到极致，塑造了完美的男友形象。

## 课程总结 📝

本节课中，我们一起学习了张真源的面部美学分析。我们从**柔和无攻击性的轮廓**入手，细致拆解了其**传神而精致的眉眼**、**挺拔且比例协调的鼻子**，以及**流畅的下颌线与可爱的唇形**。这些要素共同融合，搭配白皙健康的肤色与个人特质，最终成就了其独特的“温润少年”气质。
