# 颜值分析教程 第1课：如何客观分析面部结构与气质 🧐

在本节课中，我们将学习如何系统性地分析一个人的面部特征与整体气质。我们将以艺人马嘉祺为例，拆解其五官、脸型、比例与气质特点，帮助你理解“颜值”背后的客观构成要素。

## 概述：面部分析的基本框架

面部分析通常从整体轮廓、五官细节、比例关系以及气质呈现等多个维度展开。一个耐看的面容往往是这些要素和谐组合的结果。

## 五官与脸型分析

上一节我们介绍了分析的基本框架，本节中我们来看看具体的五官与脸型特征。

马嘉祺的五官并非绝对的完美和精致，但单看五官却有着独特的美感。他拥有标准的单眼皮且眼尾比较尖，鼻梁不算高挺，却是标准的小翘鼻。他拥有一张让女生羡慕的瓜子脸，配合精美的五官，在其小薄唇的衬托下，整个面部流露出一种清秀之美。

## 气质与耐看度分析

分析完静态的五官，我们进一步探讨动态的气质与整体印象。

马嘉祺是典型的耐看型男生，长相属于淡颜系，第一眼就让人感到惊艳的概率比较小。或许他的颜值并非第一眼就能惊艳或吸引到你，但是他的气质非常到位。有些照片中，他看起来是非常干净、帅气的小少年；有时则显得非常有公子气。

理解了气质，我们需要关注支撑面容的底层结构——骨骼与轮廓。

马嘉祺是少年爱豆中少见的、缺乏胶原蛋白的一张脸，这与他极低的体脂率有关。原本应该是流畅的舒缓型长相，却因为脸部过于消瘦，使得脸颊和眼窝凹陷，同时显得颧骨突出。在正侧脸的区分上，都自带这种阴影感，与他的尖锐五官相呼应，组成了直线感很强的脸型。**公式：直线感增强 → 冷酷感与攻击性增强**。
