# 颜值分析教程 01：刘学义的五官与骨相解析 🧑‍⚕️

在本节课中，我们将以演员刘学义为例，系统性地分析其面部特征。我们将从整体骨相框架入手，逐步拆解其五官细节，理解“立体感”与“东方神韵”如何结合，塑造出独特的俊朗气质。

## 整体骨相与面部框架 🏛️

上一节我们概述了课程目标，本节中我们来看看刘学义的面部骨骼基础。他的脸型瘦削，下颌角明显，下颌线长且舒展。颧骨颧弓虽然折角明显，但脸型依然窄小。

**核心公式：窄骨骼框架 + 清晰折角 = 上镜小脸基础**

以下是其骨相的几个关键点：
*   骨骼框架窄，构成上镜的小脸基础。
*   眉骨、颧骨、鼻尖和下颌角等该有的骨骼点均在线。
*   这些骨骼点共同支撑起了他面部的立体感。

## 侧面轮廓与面型分析 📐

了解了正面框架后，我们从侧面观察他的轮廓。刘学义属于轻微的凸面型。在平视状态下，鼻下点位于垂直平面的外侧。这种面型非常受欢迎。

**核心概念：凸面型 + 清晰下颌线 + 高点呼应 = 完美侧颜**

其侧面轮廓的构成要素如下：
*   清晰的下颌线。
*   眉骨、下巴尖与鼻尖的高度相互辉映。
*   这些要素共同构成了完美的侧面弧度，拥有恰到好处的棱角感。

## 眼部神态与细节 👁️

面部的立体感离不开深邃的眼部。刘学义的眼睛非常有神，温和而不凌厉。他拥有不宽不窄的平上眼皮，浓密的睫毛加强了眼部的存在感。

**核心公式：高眉弓 + 深眼窝 = 深邃眼神感**

以下是其眼部的具体特征：
*   高眉弓和深眼窝使眼睛“窝”在里面，深邃感恰到好处。
*   上眼睑呈半圆形，没有锐利的折角线。
*   笑起来弧度温柔，搭配浅浅的卧蚕和卷翘的睫毛。
*   整体眼神含而不露，温柔似水。

## 鼻部立体度与协调性 👃

眼睛的神韵需要挺拔的鼻梁来承接。刘学义的鼻梁以及山根都非常高。鼻尖很尖，鼻梁非常挺直。他的鼻翼也算小巧，不显粗犷。

**核心概念：高起点山根 + 适中宽度 + 微驼峰 = 自然立体鼻**

其鼻部的加分细节包括：
*   山根起点偏高，与高眉弓和深眼窝衔接流畅。
*   山根宽度适中，约为鼻翼的1/3。
*   轻微的驼峰鼻，拥有恰到好处的弧度，不会过于硬朗。
*   这增添了个人面部的辨识度。

## 总结与评分 📝

本节课中，我们一起学习了如何系统分析一个人的颜值。通过对刘学义的分析，我们看到，优秀的骨相（立体且轻盈的框架）是高级颜值的基础。精致的五官（深邃的眼、挺拔的鼻）在优秀骨相上得以完美展现，共同融合出现代立体感与东方温润神韵。

基于以上分析，刘学义的颜值评分为：**95分**。
