# 课程名称：相亲条件分析与自我认知 🎯

在本节课中，我们将通过分析一位80年出生的女士的相亲要求，来学习如何清晰、结构化地梳理个人条件与择偶标准。我们将重点关注信息的分类整理、核心要求的量化表达，以及如何将个人特质与期望进行结合描述。

---

## 个人信息概览 👤

上一节我们介绍了课程目标，本节中我们来看看这位女士的基本个人信息。

*   出生年份：1980年（冬季出生）
*   身高：168厘米
*   体重：105斤
*   星座：双鱼座
*   学历：中央民族大学本科、本硕连读（硕士学位）
*   籍贯：吉林省吉林市龙潭区
*   现居地：四川省成都市
*   行业：某互联网公司
*   婚姻状况：离异
*   子女情况：有一个6岁的孩子
*   自我外貌评价：自认非常美丽，被朋友称为“东北关之琳”
*   MBTI人格类型：**ESFJ**

了解了基本信息后，我们进一步探索她的成长经历与性格特点。

她描述自己从小是父母和老师眼中的乖乖女，学习成绩好，人缘佳。大学专业自主选择，硕士毕业后曾赴日本留学一年。职业生涯始于上海，后转至深圳，拥有多年跨境电商工作经验。

以下是她的性格与生活爱好简述：

*   **性格两面**：静时很宅，动时很疯。
*   **爱好清单**：
    *   运动：爱好打高尔夫，最好成绩为85杆。
    *   影视：喜欢动漫和美剧。
    *   美食：热衷于品尝所有美食。
    *   挑战：敢于尝试生吃蚂蚱。
    *   饮酒：能喝白酒，酒量从一斤起步，且只喝酱香型白酒。
    *   旅行：喜欢到处旅行和参与各种活动。
*   **宠物**：饲养了一只黑色泰迪犬。
*   **家庭背景**：成长于温暖有爱的独生家庭，但不娇惯。父亲是医生，母亲是护士，父母是办公室恋情，目前均在老家，暂不来成都。

## 情感经历与择偶要求 💑

上一节我们介绍了她的个人生活，本节中我们来看看她的情感观念与具体的择偶标准。

她有过12段感情经历和一次婚姻，自认每段都是真心交往，并从中更了解自己和理想的感情。

以下是她的核心择偶要求（底线）：

*   **年龄范围**：1983年至1993年出生。
*   **婚姻状况**：未婚。
*   **生活习惯**：无不良嗜好，非“妈宝男”。
*   **学历**：研究生及以上。
*   **资产要求**：
    *   房产：面积200平方米以上，无贷款。
    *   车辆：价值100万元以上。
    *   存款：**存款 >= 800万元**。
*   **其他**：不接受异地恋。拥有商铺是加分项，她可提供东北优质人参和鹿茸油资源，提议共同在成都经营。

以下是她的择偶加分项：

*   **外形**：身高188厘米以上，形象干净阳光。
*   **性格**：聪明幽默。
*   **生活方式**：能带她进行环球旅行。
*   **理想型参考**：美剧《老友记》中的角色**钱德勒·宾（Chandler Bing）**。

## 课程总结 📝

本节课中，我们一起学习了如何系统性地解析一份个人相亲资料。我们将其分为了**个人信息**、**经历性格**和**择偶要求**三大模块。关键点在于将主观描述（如“非常美”）与客观条件（如学历、资产）相结合，并使用**代码**或**公式**（如 `存款 >= 800万元`）来清晰量化核心要求，使信息一目了然。通过这个案例，我们可以掌握如何有条理地展示自己或理解他人的综合情况。
