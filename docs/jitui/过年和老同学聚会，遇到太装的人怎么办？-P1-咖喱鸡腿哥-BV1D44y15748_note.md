# 课程名称：应对聚会中“装”的人 🍻 - 课程编号：001

## 概述

在本节课中，我们将要学习如何在过年老同学聚会中，识别并妥善应对那些行为夸张、爱炫耀或令人不适的人。通过具体的分类和策略，帮助你从容面对这类社交场合，保持自己的舒适与风度。

---

## 识别聚会中的几种典型行为

过年回家难免被老同学、老朋友邀约聚餐喝酒。这种场合通常难以推脱。酒过三巡后，难免会遇到一些特别爱装的人，或者酒品不佳的人。碍于同学或朋友的身份，我们往往不便直接离席，但坐在那里又会感到难受，让整顿饭都吃得不舒服。

以下是几种在聚会中常见的令人不适的行为类型：

*   **炫耀型**：有一种人，在饭局上滔滔不绝，说个不停。看似非常厉害，其实没什么真本事。他们虚荣心特别强，喜欢表现自己。一落座就开始炫耀车、包、手机、烟、酒，逮着一切机会开始炫耀。哪怕他们家的狗，都要比别人家的狗会咬人一些。对于他们说的话，你千万不要特别当真，最多信三分就够了。

*   **挑剔型**：还有一种人，一落座就开始挑剔。挑酒、挑菜、挑餐厅、挑服务、挑环境。从上桌到散场，会一直叽叽歪歪说个不停，这也不行那也不行，嫌弃这讨厌那，非常让人讨厌。

*   **轻浮型**：第三种是，见到有女生，尤其是漂亮女生在场时就异常兴奋，不知道姓什么了。谁都没有他的俏皮话多。甭管这个女孩是谁领来的，他总想找个机会往人家身上蹭，管人家要联系方式，弄得他跟这女生是失散多年的兄妹一样，吃相特别难看。

*   **劝酒型**：第四种是在酒桌上不停劝酒、灌酒的人。他倒酒劝酒的时候比谁都积极，不把别人喝出洋相不算完。这都什么年代了，如果把人家灌醉了，万一出事，一桌人都得担责任。跟别人喝酒是尽兴，但跟这种人喝酒就是要命。不灌倒几个，他浑身就不舒服。这种人通常不能控制自己的情绪，你最好敬而远之。

*   **势利型**：还有一种更令人反感的人，就是对服务员呼来喝去。这种人你最好也要远离。因为人生而平等，劳动者也没有任何贵贱之分。这种人觉得平时没有太多机会显摆自己，到了酒桌上，就盛气凌人，对服务员极度苛刻，觉得花点钱就高人一等了，不知道哪里来的优越感。这种人通常心胸狭隘，对人特别不友善，大家要尽量远离。

*   **摆谱型**：最后一种是被集体讨厌的类型。他们可能在外面混出点小名堂，于是在同学朋友之间打官腔、摆架子、装神弄鬼。这种人可能在外面混了个一官半职，职位不高架子不小，本事不大脾气不小。无论你跟他说什么，他都给你打马虎眼；无论你跟他交流什么，他都尽说一些官话、套话、空话、假话。

---

## 核心应对策略：捧杀法

上一节我们介绍了聚会中常见的几种令人不适的行为。本节中，我们来看看如何应对其中最典型的“摆谱型”人物。

当你遇到那种爱打官腔、装神弄鬼的人时，可以尝试“捧杀”策略。具体操作如下：

**核心公式：顺势抬高 + 提出具体请求 = 令其骑虎难下**

你让他装，然后在桌子上将他一军，给他提一个具体的要求。例如，你可以说：
> “**老同学/谁谁谁，你看你现在都这么成功了。我老家还有个亲戚正在找工作，人也不错，你看能不能给安排一下？**”

这个请求会产生两种结果：
1.  他确实有本事，当场给你安排了。（可能性极低）
2.  他无法兑现承诺，只能推脱或回避。

无论哪种结果，你都成功地“将”住了他。他以后在你面前就不敢再装了，因为他知道你会当真，并可能提出他无法解决的“难题”。

## 总结与核心原则

本节课中，我们一起学习了如何识别聚会中的六类令人不适的行为，并掌握了应对“摆谱型”人物的“捧杀”策略。

最后，我们需要牢记一个核心社交原则：**酒品看人品**。人品即人心。虽然酒品和人品一样，有时可以暂时伪装，但酒品至少从一个角度为人品提供了参考。

在酒局上，一定要做到有修养、从容淡定、彬彬有礼。哪怕喝多，也要张弛有度，平等对待每一个人。遇到不太能喝酒的人，尽量不要强人所难。要做到有酒胆、有酒风、更有酒品和酒德。

**总结**：对于过年聚会中那些酒品不好、沾点酒就忘形的人，大家尽量少跟他们喝酒。与他们交往不仅可能丢人掉价，而且也不值得深交。保持自己的风度和原则，才是高质量的社交之道。
