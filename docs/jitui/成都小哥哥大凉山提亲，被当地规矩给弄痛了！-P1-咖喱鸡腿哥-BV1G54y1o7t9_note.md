## 大凉山彩礼制度详解

上一节我们介绍了案例背景，本节中我们来看看大凉山地区独特的彩礼规则，这些规则让这位成都男性感到震惊。

以下是当地彩礼制度的核心特点：

1.  **彩礼与学历挂钩**：当地有按学历划分彩礼等级的习惯。学历越高，彩礼金额越高。
    *   **公式**：`彩礼金额 ∝ 学历等级`
    *   大学生、研究生的彩礼起步价为30万元人民币。
    *   博士、事业单位编制人员的彩礼金额更高，达到50万至80万元人民币也并不罕见。

2.  **彩礼的接收方**：这笔高额彩礼并非给予新娘本人，而是分配给新娘的家人。
    *   **代码描述**：`recipients = [“父母”， “叔叔婶婶”， “舅舅舅妈”， …]` （即新娘家族成员集体分配）。

3.  **支付形式与时间**：彩礼必须在结婚当天，以现金形式一次性付清。
