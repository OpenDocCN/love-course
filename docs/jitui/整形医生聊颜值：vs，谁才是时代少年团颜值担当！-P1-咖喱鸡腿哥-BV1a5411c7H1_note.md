# 颜值分析教程 01：如何客观分析明星颜值 🎭

在本节课中，我们将学习如何像专业整形医生一样，客观、结构化地分析明星的颜值。我们将以时代少年团的两位成员——丁程鑫与宋亚轩为例，拆解颜值分析的维度，帮助你理解“美商”的核心概念。

---

## 概述：颜值分析的基本框架

颜值分析并非单纯比较谁更好看，而是一套结合了**骨相**、**皮相**、**气质**与**辨识度**的综合评价体系。审美固然存在个人差异，但专业分析会基于相对公认的标准进行。

上一节我们概述了分析框架，本节中我们来看看具体如何应用这些维度。

---

## 第一部分：丁程鑫的颜值分析

丁程鑫的长相具有高辨识度，气质干净清爽，有邻家弟弟的感觉。

以下是丁程鑫颜值特点的具体分析：

*   **轮廓与骨相**：他的面部轮廓越来越明显，骨相优越。**公式：立体感 = 眉骨高度 + 鼻梁高度 + 下颌角清晰度**。他的侧脸纵深优越，眉骨、鼻梁和下颌骨发育突出，构成了深邃立体的轮廓。
*   **皮相与留白**：皮相偏严细，脸上留白足够。皮相与骨相之间的平衡感优越。
*   **气质与风格**：气质兼具艳丽与硬朗，同时保有少年感。其风格可概括为 **`风格 = 清新少年感 + 侵略性美貌`**。没有女相感，是典型的校园清新风格。

他的颜值符合大众审美标准，是一种带有侵略性的明艳之美。

## 第二部分：宋亚轩的颜值分析

宋亚轩的长相自带一种忧郁和脆弱感，拥有“小漂亮”的别称，气质非常独特。

以下是宋亚轩颜值特点的具体分析：

*   **五官与骨相**：五官立体，眉眼深邃。其眉骨与鼻梁的夹角（T字区立体度）在亚洲人中少见，更接近欧美骨相。**代码类比：骨相结构 = {‘眉骨’: ‘高’， ‘鼻梁’: ‘立体’， ‘下颌线’: ‘清晰’}**。
*   **气质与可塑性**：气质温润干净，像水一样纯净。他的气质可塑性极强，可以在 **`[‘病娇’， ‘清冷’， ‘古风’， ‘可爱’]`** 等多种模式间切换。
*   **笑容与辨识度**：笑容具有多变性，时而天真，时而魅惑。这种多变性与他独特的骨相结合，形成了极高的辨识度。

他融合了“又A又软”的矛盾特质，骨相优越，气质纯净。

## 第三部分：总结与核心观点

本节课中我们一起学习了颜值分析的多个维度。

审美存在个性化差异，但专业分析基于骨相、皮相、气质等客观要素。无论是丁程鑫的**明艳立体**，还是宋亚轩的**忧郁温润**，都是其独特辨识度的来源。

**核心公式：整体颜值 = 骨相基础 + 皮相表现 + 气质加成 + 辨识度**

因此，在欣赏颜值时，我们应避免非此即彼的“互撕”，而是学会欣赏不同风格的美。提高个人“美商”的关键在于理解这些构成要素，并建立自己的分析框架。
