# 金融防骗课 01：识破“赠与”背后的法律陷阱 🏢💰

在本节课中，我们将通过一个真实案例，剖析金融圈中一种利用法律和商业规则精心设计的陷阱。你将了解其运作原理、关键步骤以及如何识别和防范此类风险。

## 概述：一个“赠与”公寓的陷阱

金融圈的部分人士，确实具备头脑。他们不仅能识别出某些动机不纯的接近者，还能利用规则让对方服服帖帖。近期，成都就发生了这样一个案例。

一位从事投资行业的男士，因其工作性质，身边常围绕着许多抱有特定目的的漂亮女性。他虽心知肚明，但碍于工作关系不便直接拒绝。

## 第一步：抛出诱饵，建立合作

其中一位艺术院校的女生，持续向他示好，意图获取利益。这位男士便顺水推舟，提出赠送她一套公寓，并利用自身资源与她合伙做生意。

女生听后非常高兴。男士随即提出具体方案：公寓将登记在女生名下，但购买时需以她名下**新注册的公司**作为主体。理由是“既然要一起做生意，这样操作合情合理”。

**关键操作**：`以公司名义购买资产`

## 第二步：利用“合理”规则，埋下伏笔

陷阱就设在这里。这位精英男士利用了女生的贪心，提议以公司名义购买公寓，并告知这样可以**合理避税**。面对这种“利好”，女生通常会同意。

当然，公司的**法定代表人**也自然由这位女生担任。

## 第三步：引入“风险共担”，稀释股权

在购买前，男士会告知女生：合伙生意不应由一人独资，以分散风险。因此，他会邀请一位朋友入股，持股约**10%**，女生则持股**90%**。这个理由听起来同样没有破绽。

假设公司注册资金为800万，这位朋友便出资80万（10%），并用这笔钱购买了公寓。

**股权结构公式**：`女生持股 90% + 朋友持股 10% = 公司100%股权`

## 第四步：虚构业务，诱使签字

随后，男士会安排一项虚假业务，告诉女生这笔生意能赚几百万，并让她以公司法定代表人的身份，签署相关合同并走完业务流程。

如果双方关系正常发展，或许不会有后续问题。

## 第五步：触发陷阱，收回一切

但如果女生在几个月或半年后“见异思迁”或让男士“上火”，陷阱便会启动。

男士会安排另一家关联公司（B公司），起诉女生担任法代的这家公司（A公司），要求A公司履行那份虚假合同，并追究女生的出资责任，同时申请查封房产。

## 陷阱的核心与受害者的困境

此时女生将发现：
*   公司这笔业务被做成“亏损”。
*   所有合同、流水、明细都真实有效，法律手续齐全。

她会瞬间不知所措，因为无论怎样起诉都很难胜诉。

问题关键在于：这家新公司的合同章、所签合同、走账记录、业务明细等所有核心文件，都掌握在男士手中，他绝不会交给女生。整个证据链早已被精心设计好。

## 总结与警示

本节课我们一起剖析了一个利用商业规则和法律程序设计的精密陷阱。其核心逻辑在于：

**利用对方的贪慕虚荣，通过合法的公司架构和合同流程，将赠与的资产（公寓）置于一个可由设计者完全控制的风险之下。一旦关系破裂，设计者便能通过法律手段收回资产，并使对方陷入财务与法律困境。**

这个案例警示我们，面对看似诱人的“馈赠”或“合作机会”，尤其是涉及公司注册、股权、合同签署等法律行为时，必须保持警惕，弄清所有条款的长期含义，切勿因小利而陷入被动。在商业世界里，合规意识和法律常识是最好的护身符。
