# 课程：面部美学分析入门 - 01 🧑‍⚕️

在本节课中，我们将通过分析一个具体案例，学习如何从专业角度观察和描述面部特征。我们将以艺人张艺兴的面部结构为例，拆解其五官与骨相特点，理解这些特征如何共同构成独特的个人风格。

## 面部骨相基础分析 🦴

上一节我们介绍了课程目标，本节中我们来看看面部骨相的整体框架。骨相是面部轮廓和立体感的根基。

张艺兴的面部骨相具有较高的可塑性。他的额头偏矮，眉骨高耸，眼睛略显狭窄。这种结构在生动之余，增添了一抹清冷感。他的眉骨高，下巴尖，下颌骨线条清晰。下颌角靠近咬合线，但略低于咬合线，这使得他的正面与侧面轮廓能干净利落地呈现四个转折角。

他的颧骨有棱角，高光点和起伏线条非常出色。颧弓宽度适度，太阳穴没有明显凹陷。

以下是其骨相特征的总结：
*   **额头**：偏窄。
*   **眉弓**：高。
*   **鼻骨**：高。
*   **下颌角**：正面观呈方形，位置清晰。
*   **颧骨**：线条干净、明晰。

## 五官细节与风格塑造 👁️

了解了基础骨相后，我们进一步分析五官细节。五官的形态和搭配是塑造个人气质的关键。

眉毛是其相对弱项。素颜时，因眉毛形态和略有下垂的眼角，加上可能休息不足，会显得不够有神。

张艺兴的眼睛很干净。内眼角尖锐，眼角与眼睑之间存在明显的下折角，这加强了他眼睛的力量感。当他瞪圆双眼并低头时，会传递出强烈的力量感和气势。

他的鼻子属于微驼峰鼻，鼻部的海鸥线（指鼻头与鼻翼连接的线条）比较陡峭。鼻梁高，鼻头稍显圆润有肉。

他的嘴唇形态性感且柔和，嘴角上翘，唇形饱满，属于知名的“M形唇”。

## 整体协调与风格总结 🎭

最后，我们将各部分特征综合起来，看它们如何协同作用，形成整体的面部风格。

张艺兴的面部融合了英气、柔和与萌感。这使得他能驾驭多种妆容与风格，无论是硬朗还是柔美都能达到极致，且不会显得违和。其五官可塑性高，能适应不同造型，并总能传递出高级感。

这种独特且协调的面部美学，是他在全亚洲拥有大量粉丝的原因之一。很多观众在见到他第一眼时，便会被其容貌特质所吸引。
