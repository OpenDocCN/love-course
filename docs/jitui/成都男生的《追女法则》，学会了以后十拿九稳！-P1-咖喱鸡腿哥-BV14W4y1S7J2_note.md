在本节课中，我们将要学习如何在恋爱关系中建立健康的互动模式，掌握主动权，避免因过度付出和失去自我而陷入被动。核心在于调整心态与行为，像猫咪一样保持独立与吸引力，而非像狗狗一样一味讨好。

## 过度付出的后果

上一节我们提到了掌握主动权的重要性，本节中我们来看看过度付出会导致哪些具体问题。

你的女朋友之所以可以肆无忌惮地伤害你，能够做到不关心、不在乎、不怕失去你，原因在于你自己惯的。是你的爱和付出让她得意忘形。是你的没原则和没底线让她觉得你是个窝囊废。

她已经如此伤害你，你却觉得自己无所谓。你给予的越多，她越觉得你配不上她。但你也别可怜自己。你不是爱她吗？你不是舍不得她吗？你不是满眼都是她吗？你不是离开她就活不下去吗？这就是你义无反顾的下场。

你也别抱怨，别不甘心。就算你天天难受也要憋着。这都是你自己找的。总结起来就一个字：**该**。因为在你的眼里只有她，而她的眼里却有无数个你。事情的本质就这么简单。

## 学习猫咪的相处哲学

理解了问题的根源后，我们需要寻找解决方案。所以男生在恋爱里要像猫咪一样学习，而不要学狗狗。

猫咪这种动物不会24小时黏着你。它只会在自己渴了、饿了或无聊的时候过来靠近你。男生在恋爱中也应该如此。

以下是需要掌握的几个关键行为模式：

*   **把握聊天节奏**：不要没完没了地给对方发信息。
*   **保持独立性**：能做到无聊时黏着你，又能在忙时不搭理你。
*   **敢于表达情绪**：不要担心发脾气就会失去对方。

## 建立框架与利己思维

学会了像猫咪一样保持独立后，我们还需要在互动中建立自己的框架，并培养核心的利己思维。

你要敢于在感情里发脾气。不要担心发脾气就会失去对方。当她积极回应你时，你可以表现得开心一些。但当她对你的态度比较冷淡或冷漠时，你也可以给她摆点脸色看看。

让她在不知不觉中去适应你的脾气。当她开始适应你的脾气时，你就离成功不远了。

最后，你一定要学会利己思维。很多男生在恋爱中都有一个误区：她不喜欢我，是不是我做错了什么？其实根本不是。她就是那样的人。

所以你要建立一个信念：

**我喜欢你，不是因为你有什么了不起，只是因为我乐意、我高兴。**

**我愿意对你好，也不是觉得你有多值得，只是因为我想这么做。**

**我要是不乐意、不舒服了，我马上可以换人，绝不是非你不可。**

所以，一旦你建立了这样的心态，就没有什么女生是你搞不定的。
