## 心态的起点：青春红利期 🎯

一个成都小伙相亲时，被一位大龄单身女性指责，因而想了解这类女性的真实心态。需要明确，这里讨论的是“部分”现象。

在18岁到28岁的十年青春期内，部分女性会收到大量异性不计成本的示好与付出。

然而，部分女性对此不做任何明确回应，甚至肆无忌惮地接受这些好意。

## 心态的发展：自我认知的膨胀 🔄

上一节我们介绍了青春期的优势处境，本节中我们来看看其心态如何变化。

长期被追捧使部分女性产生错觉，认为自己永远不缺追求者。

于是，她们开始同时与若干异性保持亲密往来，并进行比较与情感操控。

在此期间，她们也可能与其中几人正式交往，但恋情均以分手告终。

分手后，她们通常将责任完全归咎于对方。

以下是其常见的归因逻辑：

*   她们会引用各种情感鸡汤来支持自己的观点。
*   她们会在闺蜜圈中倾诉自己在这段感情中的“付出”与“伤害”。

## 行为模式：索取与合理化 📝

在交往过程中，部分女性的实际付出可能极少。

以下是其典型表现：
*   从未请对方吃过一顿饭。
*   未送过对方一件像样的礼物。

然而，对方所有的关怀、体贴、请客、旅游等付出，都会被她们合理化。

她们认为：“这些都是追求者应该做的。做不到这些，凭什么追我？”

与此同时，她们可能保持抽烟、喝酒、蹦迪、与多人暧昧等行为，但仍对外宣称自己是“好女孩”。

若有人反驳，她们会指责对方“格局有问题”。

## 择偶标准的形成与矛盾 ⚖️

当部分女性玩够了，想找个人照顾自己“疲惫的心灵”时，便会制定极高的择偶标准。

标准通常包括：有房有车、事业有成、体贴入微、年入百万并愿意为她付出等。

她们的信条是：如果条件不能满足，宁可“高质量单身”，也不“低质量恋爱”。

她们始终觉得是对方配不上自己，对方必须时刻围着自己转。

即便年龄增长，部分女性的择偶标准依然居高不下。

后来，她们发现年龄越来越大，越来越没人迁就自己了。

于是，她们开始降低择偶标准，且一降再降。

此时，她们可能转而宣扬：
