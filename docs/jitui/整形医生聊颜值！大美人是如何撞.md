# 整形医生聊颜值！大美人是如何撞

在本节课中，我们将要学习如何从专业角度分析面部特征。我们将以艺人孔雪儿为例，并与杨幂进行对比，具体拆解她们在脸型、五官等方面的异同，帮助你理解构成“颜值”的核心要素。

## 脸型轮廓分析

孔雪儿是一名选秀舞台出道的女生，被粉丝称为美得不可方物，但也被诟病缺少一些气场。对比她的旧照，孔雪儿的脸型明显瘦了很多。

有些人会认为她是否进行了削骨手术。其实不然，从正面看，她的下颌角依然比较小巧。不过，她也多次被认为和杨幂的下半张脸高度重合。

上一节我们介绍了整体印象，本节中我们来看看她们脸型的具体差异。

以下是两人下颌轮廓的关键点对比：

*   **下颌角**：两人的下颌线条都特别流畅，没有过于明显的下颌角。但杨幂的下颌角拐点位置偏低，而孔雪儿的下颌角几乎没有看到明显拐点。
*   **下颌收势**：孔雪儿的下颌向上收窄比较明显，下颌没有杨幂那么翘。
*   **正面观感**：孔雪儿的颧骨比较宽，两颊又窄，下巴尖，所以正面看起来特别像**锥子脸**。也因为颧骨宽，从某些角度看会显得有“心机感”。不过她喜欢用长发来制造氛围感，弱化了面部线条的攻击性。

## 五官细节剖析

分析完脸型轮廓后，我们进入五官的细节部分。鼻子和唇形是面部特征中非常关键的部分。

从鼻子上看，伴随着鼻小柱下垂和鼻翼缘退缩的情况，她们两个都存在鼻孔轻微外露的现象。在鼻子都很挺拔的基础上，两者又有区别：

*   **杨幂**：鼻尖表现点更突出，鼻尖微翘。
*   **孔雪儿**：鼻尖更为圆润。

接下来，我们看看唇形。以下是两人唇部特征的相似之处：

*   **唇型**：两者的唇型蛮相似，唇峰都不太明显。
*   **线条**：唇部线条都比较平缓，嘴角呈向上扬的趋势。
*   **微笑习惯**：尤其是微笑时，嘴角都呈圆形，并习惯性地用一侧的嘴角上扬来带动微笑。

## 软组织与比例调整

了解了静态的五官结构，我们再来看看面部软组织和整体比例的影响。这关系到面部的年轻感和协调性。

孔雪儿早期的软组织非常饱满，有充盈的苹果肌和婴儿肥，显得青春可爱。她现在减肥减得非常猛，面部轮廓线更加清晰了。

但是，她的脖子相对较粗是她外形上的一个明显短板。从专业角度看，建议孔雪儿适当地增加下巴的长度和翘度。她的下巴属于非常尖的类型，下颌角拐点不明显，所以正面看下巴收得非常紧，多少有些比例失衡。

## 总结与评分

本节课中，我们一起学习了如何系统地分析面部颜值。我们以孔雪儿为例，对比杨幂，从脸型、五官到软组织进行了逐层拆解。

孔雪儿的脸足够特别，五官中，锐角带着细腻，青春中带着攻击性，很有记忆点。综合以上分析，我给孔雪儿的颜值打 **93分**。
