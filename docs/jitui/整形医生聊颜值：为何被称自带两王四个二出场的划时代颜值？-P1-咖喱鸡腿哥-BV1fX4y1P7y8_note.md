# 颜值分析教程 01：解析郑在玹的“划时代”颜值 🎭

在本节课中，我们将以NCT成员郑在玹为例，系统性地分析其面部结构与气质特点。我们将拆解其被称为“自带两王四个二出场”的颜值构成，学习如何从专业角度评估一个人的外貌。

## 五官的周正与和谐 🌟

上一节我们介绍了课程概述，本节中我们来看看郑在玹五官的整体特点。

郑在玹的五官非常周正。第一眼看去，会让人觉得他是一个干净的男生。他的形象给人以阳光、清澈、舒服、温暖的感觉。无论何时何地，无论状态如何，他的表情都非常自然。

他的全身没有油腻感。身材和颜值的搭配非常和谐。加上他本身独特的气质加持，难怪一出道就让粉丝们为他疯狂着迷。他的脸不单调，非常帅气，能够支撑其面部从不同角度观看。因此这张脸可塑性非常高。

以下是其五官和谐带来的具体感受：
*   **干净自然**：无油腻感，状态始终自然。
*   **阳光温暖**：气质清澈，给人舒服的感觉。
*   **可塑性强**：面部结构支持多角度观看与多种造型。

## 少年感与造型消化力 🎨

了解了整体和谐感后，我们进一步分析其带来的具体气质与表现力。

郑在玹长着一张少年感十足的初恋脸。笑起来时脸上肉嘟嘟的感觉，给人很亲切。他的脸蛋和身材让他能够很好地消化各种造型。加上他很深的酒窝，能为他的颜值添砖加瓦。

郑在玹的外貌在他不笑的时候，给人一种距离感，甚至可以说是一种神秘感。这也是让他魅力加分的因素之一。

以下是其少年感与多面性的体现：
*   **初恋脸**：少年感十足，笑容亲切。
*   **造型广度**：脸型与身材能驾驭多种风格。
*   **气质反差**：不笑时产生距离感与神秘感，增加魅力层次。

## 眼部细节分析 👁️

郑在玹最开始吸引到我的地方是他的眼睛。现在我们来仔细分析其眼部的构成。

他的眼睛眼形细长，有两道剑眉和完美的扇形双眼皮。眼头比较尖锐，很有东方感的中式美。卧蚕非常饱满，上眼睑中部圆润。因此当他微微睁大眼睛的时候，有一种无辜感。这种无辜感加成后，笑起来眼睛眯成一条缝，有着憨憨的温柔感。不笑的时候，也给人偏冷的感觉。这让他的粉丝对他充满了欲擒故纵般的向往。

以下是其眼部特征分解：
*   **眼形**：细长，眼头尖锐，具东方美。
*   **双眼皮**：完美的扇形。
*   **卧蚕**：非常饱满。
*   **神态效果**：微睁时显无辜，笑时显温柔，不笑时显冷感。

## 鼻梁的突出影响 👃

眼睛是心灵的窗户，而鼻梁则是面部的支柱。本节我们探讨鼻梁对整体气质的关键作用。

他的鼻梁在整个人的五官当中比较突出。鼻梁对于整个人的颜值影响非常大。郑在玹的鼻梁属于高鼻梁。鼻翼两侧没有多余的肉感，所以整个人的气质就会显得非常精致干练，很有精神。鼻梁的线条非常流畅，鼻头圆润，但是不会显得特别憨气。而且因为他的鼻梁长得比较突出，所以他侧脸是非常完美的感觉，找不到一丝丝的缺点。

鼻梁的构成可以用以下核心要素概括：
```
高挺度：高
鼻翼：紧致，无多余肉感
线条：流畅
鼻头：圆润但不显憨
侧脸效果：完美
```

## 面部轮廓与唇部特点 💋

分析完鼻梁，我们来看面部下半部分的轮廓与唇部特征。

郑在玹属于典型的少年长相。虽然他被他的粉丝和一些黑粉诟病有水肿易胖的体质，但他的面部轮廓依然是唇红齿白的类型。唇峰比较丰满，唇珠很明显。因为有唇珠，整个人会带有一种小孩子的天真和单纯。

如果非要在他的脸上找一些不足，大概就是他的下巴稍微有那么一点前凸。但是他鼻梁挺直，身材线条流畅，整体轮廓好看。那么点问题可以忽略不计。

以下是其唇部与轮廓特点：
*   **唇部**：唇红齿白，唇峰丰满，唇珠明显，增添天真感。
*   **轮廓**：整体为少年长相，线条流畅。
*   **微小不足**：下巴略有前凸，但被整体优势掩盖。
