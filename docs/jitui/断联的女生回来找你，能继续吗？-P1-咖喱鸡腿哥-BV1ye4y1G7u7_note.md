## 1. 作为“比较后的选择”回归 🔄

这种情况通常发生在对方经历了其他不顺利的感情后，通过对比觉得你更好，因而回头试图挽回。此时，她的行为模式会显得格外主动和温柔。

**核心逻辑公式**：`她的回归动机 = 经历其他感情挫折 + 与你比较后认为你更优`

在理解了“比较后选择”的回归动机后，我们接下来审视一种更不健康的关系模式。

## 2. 将你视为“情感备胎” 🛞

这种情况下，对方并不打算与你建立严肃的恋爱关系，但也不愿完全失去你对她的关注和付出。这是一种自私的情感策略。

以下是“备胎”策略的典型特征：

*   **低成本维系关系**：在没有找到更合适对象前，与你保持联系对她而言省时省力。
*   **随时可替换**：一旦遇到她认为更好的目标，她会毫不犹豫地离开你。
*   **关系不对等**：她享受你的付出，但不愿给予对等的承诺或情感回报。

---

从“备胎”到更功利的层面，第三种动机的目的性更为直接和物质化。

这是目的性最强、也最应警惕的一种情况。对方回归纯粹是为了从你身上获取实际利益或好处，而非情感。

**核心行为模式代码**可描述为：
```python

    主动联系你
else:
    无视你
```

她的联系与否完全取决于你对她是否有利用价值。在不需要你时，她甚至会避免任何纠缠。
