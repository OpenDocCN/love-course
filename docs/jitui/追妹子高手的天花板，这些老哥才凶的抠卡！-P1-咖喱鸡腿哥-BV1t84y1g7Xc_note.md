## 反向操作的核心：不纠结于“她是否喜欢我” 🔄

上一节我们介绍了高手们的整体心态，本节中我们来看看他们的第一个具体操作：不纠结于对方的好感。

高手追妹子基本上都在反向操作。他们几乎不在乎女生对他们的即时好感程度。妹子长得招人稀罕可以勇敢追，但不要花太多时间和精力去琢磨对方到底爱不爱你。爱情这玩意虚无缥缈。

他嘴上说爱你一万年又有什么用。反过来想，如果你喜欢的女生她打死不喜欢你，那你花了很久的时间、很久的精力，费了很久的心思也捞不着。你亏不亏。况且你的过度在乎就是在给对方压力。

一想到你，人家回去连上厕所都不顺畅了。所以，格局放大一点。

以下是常见的、需要避免的纠结行为：
*   总是猜对方对你什么感觉。
*   猜对方有没有跟别的男生接触。
*   猜为什么对方不回你消息。

这些行为又费时又费力又不讨好又得不到答案。

## 关注重点转移：从“感觉”到“能力” ⚙️

既然纠结对方的感觉无用，那么我们应该关注什么？本节中我们来看看高手们真正的关注点。

其实女生对你有没有好感根本不重要。重要的是现阶段你有没有这个能力，让她以后愿意主动见你，愿意主动地喜欢上你。

你也不要在乎一时间能不能做成情侣。这个世界有的时候，你越想干成一件事，就越干不成。人生不如意十之八九，阴差阳错的事多了去了。

**核心公式：成功 ≠ 当前好感度， 成功 = 未来吸引力 × 耐心**

人家当下不喜欢你，不代表未来不喜欢你。你不能现在强迫人家喜欢你。所以，暂时失败了没搞成，要保持一颗平常心。
