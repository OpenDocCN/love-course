# 网红辛巴事件解析：直播带货的法律风险与后果 📺 - 课程01

在本节课中，我们将要学习近期备受关注的“辛巴假燕窝事件”。我们将从事件回顾、相关法律条文、责任界定以及事件影响等多个方面，对这一网络直播带货的典型案例进行解析，帮助初学者理解其中的核心法律概念与商业逻辑。

## 事件回顾与最新进展

上一节我们介绍了课程的主题，本节中我们来看看事件的具体经过。

受燕窝事件影响的辛巴，阔别一个月以后，再次出现在他徒弟的直播间里。值得注意的是，再次出现的辛巴依然在为此道歉，双手合十态度诚恳，弯腰鞠躬达到30秒左右。

但是事件的另一端，我们看见辛巴所售卖的假燕窝被立案的消息逐渐爆出，称其可能会面临15年的有期徒刑。

不过今天广州市白云区市场监督管理局回应称，辛巴售卖的假燕窝只是在调查，还没有被正式的立案，有了最新的消息会向社会公布。

## 涉及的法律规定：刑法第140条

了解了事件概况后，我们来看看此事件可能涉及的核心法律条文。

不管辛巴是被调查了还是被立案了，我们国家的刑法140条是明确规定了。该条文针对生产、销售伪劣商品罪，其核心判定标准可以概括为以下公式：

**刑事责任触发条件 = (以次充好 || 以假充真) && 销售额 >= 200万元**

根据条文，以次充好，以假充真，涉及销售额达到200万以上的，就要判处15年的有期徒刑，甚至是无期徒刑。

## 责任界定的关键因素

法律条文是基础，但具体到个案，责任如何界定呢？本节我们来分析决定辛巴是否需负刑事责任的关键点。

今天也有律师解读，要判断辛巴是否有刑事责任，还要具体看他是不是这个品牌的销售商，要看他直播带货时签订的合同和其他的材料。

以下是界定其法律身份与责任的关键因素列表：
*   **合同性质**：直播带货时签订的是**销售合同**还是**推广服务合同**。
*   **法律身份**：在交易链条中被界定为**销售商**还是**广告代言人/推广者**。
*   **主观认知**：是否**明知**商品为假货，还是被商家所蒙蔽。

律师讲到，如果界定辛巴签的是销售合同，是销售商，他则有可能面临刑事责任。但是如果他也是被商家所蒙蔽的话，那么最后的处罚可能就是民事和行政处罚责任，不会承担刑事责任。

## 事件的深远影响

无论法律结果如何，此事件已经对当事人产生了不可逆的影响。上一节我们讨论了法律责任，本节我们来看看事件带来的商业与信誉后果。

不管最后辛巴是被罚款还是被判刑，总之他本人的信用已经在假燕窝这个事件当中，完全破产。今后想要复出再次带货直播的可能性几乎为零。

这个结论基于一个简单的商业逻辑：**信任是直播电商的核心资产**。一旦失去消费者信任，其商业价值便归零。
