## 什么是浓颜系？ 🤔

上一节我们介绍了课程目标，本节中我们来看看核心概念“浓颜系”。浓颜系通常指五官量感大、面部立体度高、视觉冲击力强的长相风格。其特点可以用一个简单的**公式**概括：

**浓颜系 ≈ 大五官 + 高立体度 + 强色彩对比**
