# 小仙女IU，为何被称国民妹妹，

在本节课中，我们将以韩国艺人IU（李知恩）的面部特征为例，学习如何进行基础的面部美学分析。我们将逐一拆解她的五官特点，并理解这些特点如何组合，共同塑造出和谐且具有感染力的整体形象。

上一节我们介绍了课程目标，本节中我们来看看对IU面部整体的初步印象。

我个人认为，IU李知恩的长相很普通，既不丑也不特别漂亮。我前段时间刚看完她的新剧《德鲁纳酒店》。她在这部剧中的造型确实很漂亮，并且成功驾驭了通常难以掌握的复古风格。

## 眼睛的分析 👀

上一部分我们提到了整体印象，现在我们来具体分析她的眼睛。IU是典型的鲶鱼系长相，双眼间距不宽。她的眼睛本身不小，但很擅长通过化妆来修饰。仔细观察可以发现，IU的眼睛在妆容上努力调整了比例，从而瞬间弱化了宽眼距可能带来的不和谐感。

我之前只看过她和边伯贤演的《步步惊心：丽》。这次看《德鲁纳酒店》，我觉得IU的眼睛很有故事。在剧中，能够从她的眼睛里感受到她所要传达的感情，她的感染力非常强。

## 鼻子的分析 👃

分析完眼睛，我们接着来看她的鼻子。IU的鼻子确实不高，但鼻型很好看，非常小巧，而且鼻翼不外扩。因此，鼻子在她脸上的存在感最弱。这就是我们常说的“鼻子不高挺但也好看”的原因。

很多人鼻子不好看，正是因为鼻子在面部的存在感太强了。大多是因为鼻子肉感重，在脸上会显得非常突兀，让人第一眼就只注意到鼻子。IU的鼻子就没有这方面的问题。

## 嘴巴的分析 👄

接下来，我们分析她面部存在感最强的部位——嘴巴。IU的嘴巴不厚也不薄，不大不小，厚度和大小刚刚好。整个嘴巴的线条非常优美，为她的颜值加分很多。

其实眼睛和鼻子好看的人很多，但是嘴巴好看的人并不那么多。所以一张好看的嘴巴会非常吸引人。

## 整体协调性与总结 📝

我们已经逐一分析了五官，现在让我们从整体来看。总结来说，IU李知恩的漂亮在于她的“恰到好处”。她的每一个五官都不是特别精致，但组合在一起就刚刚好。

也有很多粉丝讨论IU是否整容。在我看来，只有整容失败才叫“整容”。通过微调使自己变美完全没有问题，爱美之心人皆有之。

因此，我给IU的颜值打90分。
