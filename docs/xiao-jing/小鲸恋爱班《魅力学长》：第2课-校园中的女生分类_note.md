# 小鲸恋爱班《魅力学长》：第2课-校园中的女生分类 🏫

在本节课中，我们将要学习如何在校园环境中有效地吸引女生。我们将分析吸引力的核心构成，并重点探讨在校园这一特定场景下，如何通过提升个人形象与人格魅力来获得优势。

恋爱是一场相互筛选的过程。在校园恋爱中，为了让自己得到更多女生的青睐，必须拥有足够多能吸引女生的筹码。这些筹码往往只存在于整个校园里20%的男生手里。简单来说，只要你成为了这20%的男生，就基本不愁没有女朋友。

这里所说的筹码，可以理解成你的高价值。根据经验，男人吸引女人的筹码基本分为三类。

以下是三类核心筹码：

1.  **第一类：形象**
    这包括身高、身材、外貌和穿着。身高与外貌很大程度上是天生的，我们基本无法改变。但穿着打扮可以在一定程度上改进或掩饰先天不足。

2.  **第二类：财富地位**
    这包括财富、权势和社会地位等。这些可以在后天获取，但需要长时间的努力与机遇。

3.  **第三类：人格魅力**
    这包括自信、幽默、口才、肢体语言和社交能力等。这类特质可以在不太长的时间内有效地打造出来。

以上三点都是吸引女人的主要特质，在任何环境中都适用。那么，校园环境与社会环境的主要差别是什么呢？

上一节我们介绍了三类通用筹码，本节中我们来看看校园环境的特殊性。我前面提到的第二点——财富地位——在校园中就显得不是尤为重要了。原因很简单：上学时，大部分学生从家里获得的生活费差距不大。

因此，在校园恋爱中，我们只需要做好两点就足以吸引很多女生：一个是看起来还不错的外型（后续课程会单独讲解），另一个就是你的人格魅力。

那么，人格魅力如何通过学习和实践来提升呢？这主要分为两个部分。

## 第一部分：从内而外的自我提升 🧗

这个部分要求你有意识地持续培养相关特质，实现脱胎换骨的改变。

以下是你可以着手去做的几件事：

*   **在男性群体中取得领袖地位**

*   **不带目的地多结交一些异性朋友**
    你身边不可能个个都是让你心动的女生。对于那些中等或感觉一般的女生，你要做的就是保持融洽的关系。因为不带目的，你会更轻松自然地相处，从而积累与女生交往的宝贵经验，克服害羞、不知道说什么的障碍。这能极大提升你的情商和对女生的了解。此外，当你身边的女性朋友多了以后，她们也可能为你介绍单身闺蜜。

*   **多和比你出色的人在一起**
    人是学习性很强的物种，环境塑造人。多关注那些在社交、恋爱方面强于你的人，留意他们的言行，辩证地吸收对你有用的东西，然后付诸实践。时间会检验一切。

*   **走出你的寝室**
    大部分学生的业余时间花在了打游戏和无意义的事情上。女生不会喜欢这样生活的男人。幸运的是，很多女生在校园中也同样迷茫，她们希望有一个男人能带领她们丰富生活。所以，首先你自己要知道如何活得精彩。

## 第二部分：通过技巧与惯例进行展示 🎭

这个部分是通过一些具体的方法和话术，让你能够立刻展现出那些有魅力的特质。

以下是几个实用的技巧示例：

*   **预选**
    多和一些长得好看的女生搞好关系（不撩，只做好朋友），或在大型活动时与她们合影并发到朋友圈。当其他女生看到你身边有很多优质女生时，会下意识地认为你是有价值的男生。

*   **约会时保持轻松状态**
    和女生约会时，逼自己表现得云淡风轻，不紧张，很随意。不要总想着找话题，顺其自然就好。先做到让聊天顺畅进行，再结合技巧增加趣味。

    *   女生说“在写作业”，你可以回：“为什么你没课的时候可以写作业，而我只能在寝室休息？太不公平了。”

    例如，当一个女生总调侃你，你可以说：“我发现你有一个特点，还蛮特别的。”引起她的好奇后，她肯定会问“什么特点？”，这时你可以接着说：“特别的作死。”用这种方式调戏女生，能展现你的幽默。

更多话术惯例会在后续课程中总结。但我想强调的是，很多兄弟在运用技巧时，会遇到女生不配合或效果不佳的情况。这是因为你没有通过“第一部分”的自我提升，将这些特质内化到自己身上。

惯例技巧不能一味地复制粘贴。我们首先要尝试理解技巧背后的原理，然后内化吸收，灵活变通。例如“制造悬念”这个技巧：

同样一个技巧，可以演变出各种表达形式。没有哪句话术是100%一成不变的。你必须明白话术背后的意思，然后内化到自己的思维中。
