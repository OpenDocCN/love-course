# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

变量可以理解为一个存储信息的“盒子”。你给这个盒子贴上一个名字（变量名），然后把数据（值）放进去。之后，你就可以通过这个名字来使用盒子里的数据。

在Python中，给变量赋值非常简单，使用等号 `=` 即可。等号左边是变量名，右边是你想存储的值。

**核心公式：**
`变量名 = 值`

例如，下面的代码创建了一个名为 `message` 的变量，并把文本“你好，世界！”存储在里面。
```python
message = "你好，世界！"
```

上一节我们介绍了变量的基本概念，本节中我们来看看如何给变量命名。

### 变量命名规则

给变量起名字需要遵循一些简单的规则，以下是主要的命名规则：

*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名不能是Python的保留关键字（如 `if`、`for`、`while` 等）。
*   变量名是区分大小写的（例如，`age` 和 `Age` 是两个不同的变量）。

良好的命名习惯能让代码更易读，建议使用描述性的名字，如 `user_age` 而非 `ua`。

理解了如何创建和命名变量后，我们还需要知道变量里能放哪些不同类型的数据。

## 基本数据类型 🔢

数据类型定义了变量中所存储数据的种类。Python有一些内置的基本数据类型，最常用的有三种。

以下是三种最基础的数据类型介绍：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数、负数或零。
    *   **代码示例：** `age = 25`， `count = -10`

2.  **浮点数**
    *   表示带有小数点的数字。
    *   **代码示例：** `price = 19.99`， `pi = 3.14159`

3.  **字符串**
    *   表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   **代码示例：** `name = “小鲸”`， `greeting = ‘Hello’`

不同类型的数据可以进行不同的操作。例如，数字可以加减乘除，字符串可以拼接。

### 查看数据类型

有时你可能不确定一个变量的类型，可以使用 `type()` 函数来查看。

**核心公式：**
`type(变量名)`

例如：
```python
height = 180
print(type(height))  # 输出：<class ‘int‘>
```

## 一个简单的综合示例 💻

让我们把上面学到的知识组合起来，写一个简单的程序。

```python
# 定义几个变量
user_name = “张三”
user_age = 30
user_height = 175.5

# 打印这些变量及其类型
print(“姓名：”, user_name, “， 类型：”, type(user_name))
print(“年龄：”, user_age, “， 类型：”, type(user_age))
print(“身高：”, user_height, “， 类型：”, type(user_height))
```

运行这段代码，你会在屏幕上看到每个变量的值和它对应的数据类型。

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。我们知道了变量是存储数据的容器，并通过 `变量名 = 值` 的公式进行赋值。同时，我们认识了三种基本数据类型：用于整数的**整数**、用于小数的**浮点数**和用于文本的**字符串**，并学会了用 `type()` 函数来检查类型。掌握这些是开启后续所有编程学习的大门。
