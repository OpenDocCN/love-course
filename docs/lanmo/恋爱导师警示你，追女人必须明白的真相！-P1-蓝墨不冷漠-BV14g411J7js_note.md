## 一、选择倾向：渣男与老实人 🃏

跟喜欢的人聊天时，核心不是“询问”，而是“让他做选择”。例如，不要问“我们周末做什么？”，而是给出选项：“周末是想去看电影，还是去新开的咖啡馆？”
