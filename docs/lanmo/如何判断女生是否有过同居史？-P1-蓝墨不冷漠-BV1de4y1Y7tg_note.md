# 恋爱小课堂 01：如何判断女生是否有过同居史？ 🏠

在本节课中，我们将学习一种通过对话技巧，间接判断女生是否有过同居史的方法。核心在于通过询问关于前任的日常生活细节，而非直接提问，从而避免尴尬并获取信息。

---

直接询问女生“你是否与前男友同居过”非常不礼貌，极易引发对方反感。

高情商的男生会采用更委婉的方式。以下是具体的操作步骤。

**以下是核心的提问流程：**

1.  **提出初步问题**  
    你可以这样问她：“你和你上一个男朋友交往了几年？他早上会给你做早饭吗？”  
    此时，无论她回答“会”还是“不会”，你都能推断出他们已经同居了。因为只有共同生活，才会涉及“早上做早饭”这样的日常细节。

2.  **提出跟进问题**  
    如果她对第一个问题的回答是“我们还没住在一起”，你可以接着问下一个问题：“那么，在你过去的男朋友里，有没有谁早上起来给你做过早饭呢？”  
    通过她对过往恋爱细节的回忆与描述，你同样可以判断她是否有过同居经历。

---

通过以上两个递进式的问题，你可以根据对方的回答内容，自然地推断出她是否有过同居史。关键在于关注“共同生活”所产生的特定生活细节。

本节课我们一起学习了一种通过询问日常生活细节来间接判断女生情感经历的方法。记住，沟通的核心是尊重与技巧，直接冒犯的提问不可取。
