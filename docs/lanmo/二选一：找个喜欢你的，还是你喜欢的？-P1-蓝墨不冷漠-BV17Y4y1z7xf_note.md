# 恋爱课 P1：如何选择伴侣？💘

在本节课中，我们将探讨一个经典的情感问题：在建立恋爱关系时，是应该选择喜欢你的人，还是你喜欢的人？我们将分析单方面付出的风险，并介绍建立健康、平等恋爱关系的基本原则。

## 核心原则：双向选择是前提

选择伴侣时，一个根本前提是对方也必须喜欢你。这种喜欢可以是最初就存在的，也可以是通过你的吸引而后产生的。不要接受任何单方面的爱恋。你需要的是热烈、坦荡且坚定的感情，只有这样，你才能避免在感情中受伤。

## 为何要避免单方面付出？

上一节我们提到了双向选择的重要性，本节中我们来看看单方面付出（俗称“舔”）为何不可取。这种方式能让你更好地保护自己。

通过讨好换来的关系是不对等的，也难以长久。当你感到疲惫无法继续讨好时，对方可能会认为你变了，从而离开你。结果是你既劳累又伤心。此外，女性天性慕强，通常不会喜欢只会讨好的追求者。

追求异性时需要保持自己的态度、框架和原则，而不是一味讨好。

## 建立平等甚至略高的姿态

既然不对等的关系有害，那么该如何建立互动呢？关键在于保持平等甚至略高的姿态。

你与女性是平等的，甚至姿态可以比对方略高一点，这样才能形成吸引。你可能会因此失去一两个对象，但长远来看，你会吸引更多女性的青睐。然而，目前90%的男性都以一种低姿态的追求者身份去接触女性，这其实需要改变。

你需要理解什么是 **“奖品属性”**。

**核心公式：你的姿态 = 你是奖品，而非追求者**

你的表现应该让女性觉得，得到你就像得到奖励。而不是让你追到她这件事，变成她对你的奖励。你要有“我是万人迷、是帅哥、是男神”的心态，让女性觉得“能和你在一起是我的荣幸”，认为你是上天派来的天使或月老赐予的礼物。这种心态会让对方更加投入。

## 策略对比与情感投入

从概率上看，同时卑微地讨好十个人，与同时自信地对待十个人相比，反而是后者能获得更好的结果。核心就是要让对方喜欢你。

即使你并不特别喜欢对方，也远好过你喜欢一个只想和你玩玩的人。为了维持一段不平等的感情，对方付出三分，你却付出八分，最终感动的只有自己。

在感情中，我现在采取一种“自私”的策略。所谓双向奔赴才有意义，那么即使我们相距一百米，我也最多只奔向你四十米，让对方奔向你六十米。

这样做有两个原因：
1.  **投资越多，爱得越深**：对方在这段感情中投入更多，就会更爱你。
2.  **自我保护**：因为我曾经害怕过，所以现在需要保护自己。就像一场梦，愿你送我归家的路不必太远。

---

**本节课总结**：我们一起学习了选择伴侣的核心应是**双向喜欢**，剖析了单方面付出的危害，强调了建立**平等或略高姿态**的重要性，并引入了**“奖品属性”** 这一关键心态。最后，我们探讨了在双向奔赴中合理分配情感投入的策略，以保护自己并维系健康的恋爱关系。
