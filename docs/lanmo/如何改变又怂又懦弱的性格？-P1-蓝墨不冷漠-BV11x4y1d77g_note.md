# 如何改变又怂又懦弱的性格？-P1-蓝墨不冷漠-BV11x4y1d77g_note

在本节课中，我们将要学习如何识别并改变“又怂又懦弱”的性格特质。我们将深入探讨这种心态的成因，并提供四个具体、可操作的步骤来帮助你建立自信、敢于表达和拒绝，最终成为一个有态度的人。

许多人在生活、情感或工作中，总是表现出又怂又懦弱的性格。如果你认真实践接下来要说的四点，能做到其中至少两点，就能逐步改变讨好型自卑人格，变得敢说敢做。

首先，要理解造成这种心态的原因。很多情况都与个人从小的家庭环境有关。例如，小时候家教严格，父母习惯打压式教育，这容易造成性格自卑和心软。由于父母很少给予鼓励，导致个人不自信，总是在意他人想法，形成讨好型人格。

想要改变，就必须认真理解接下来的内容。内容可能直接而真实，但必定有效。

上一节我们探讨了性格形成的潜在原因，本节中我们来看看改变的第一步：调整你对他人关注的认知。

以下是核心观点一：

*   **核心观点**：其实根本没有人在注意你，也没有人在乎你。这说法直接，但符合事实。
*   **具体阐述**：在日常社交中，只要没有影响到他人，根本不会有人持续关心你的具体言行。例如，聚会中表现好，他人最多夸赞一句；即使当场出丑，目击者也可能在三天内就忘记此事。
*   **公式/代码描述**：`社交关注度 ≈ 你对他人造成的影响程度`
*   **实例分析**：所谓“社恐”，常源于过度在意他人看法。以搭讪为例，很多人担心当众被拒很丢脸。但现实中，大家各行其路，根本无人留意。这正是脸皮厚的人往往更容易成功的原因。
*   **行动指南**：记住，只要你不觉得尴尬，尴尬的就是别人。

理解了过度关注自我的误区后，我们需要主动积累社会经验来强化内心。

以下是核心观点二：

*   **核心观点**：多去跟“恶心”的人接触和碰撞。人的成长常源于经历不好的人和事，而非仅靠遇见好人。
*   **具体阐述**：讨好型人格尤其需要通过与那些自私、只会索取和利用他人的“烂人”打交道，来给自己上一堂现实的社会课程。这能警醒你：无底线的“好”可能一文不值。
*   **公式/代码描述**：`心理韧性增长 = 经历负面事件并反思`
*   **行动指南**：只有承受过代价，屠龙少年才能真正变得强大。

在与外界碰撞的过程中，我们需要管理好内在的自我评价系统。

以下是核心观点三：

*   **核心观点**：自尊心别太强。过强的自尊心常是过度自卑的表现，源于害怕被忽视和看不起。
*   **具体阐述**：例如，有人与他人互动未得到即时回应，就立刻认为对方讨厌自己。这种极端想法需要纠正。
*   **实例分析**：真正的自信是“作为你自己，但一样喜欢自己”。观察身边自信的人，他们可能某些方面不如你，内心或生活也有问题，但他们能认清并接纳自己。
*   **公式/代码描述**：`健康自信 = 自我认知清晰 + 自我接纳`
*   **行动指南**：我们不是人民币，无法让人人都喜欢。试图让所有人满意只会徒增烦恼。男性应该学会“自恋”，这是自爱和自信的表现。

最后，也是至关重要的一点，是从根本上接纳改变的起点。

以下是核心观点四：

*   **核心观点**：怂和懦弱并非你的错。性格非天生，而是成长环境、家庭教育、社会经历等多因素塑造的结果。
*   **具体阐述**：理解这一点很重要：你并非自愿如此，而是在无形中被各种压力塑造。例如，工作中领导PUA你，主要源于他的职位和资历带来的底气；感情中女生对你冷漠，可能因为她身边有众多追求者，给了她高高在上的资本。
*   **公式/代码描述**：`性格现状 = Σ(环境因素 + 经历因素)`
*   **行动指南**：你的目标不应是改变他人或贬低自己，而是努力提升自我价值，从外到内优化自己，让心态强大起来。怂和懦弱不可怕，可怕的是安于现状，甚至将其美化为“老实善良”，从而任人宰割。
