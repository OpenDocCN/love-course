# 课程名称：蓝墨云班课基础操作指南 🎓

在本教程中，我们将学习蓝墨云班课这一教学平台的核心功能与基本操作流程。我们将从注册登录开始，逐步了解如何创建课程、管理学生、发布资源与活动，并查看学习报告。

---

## 第一课：平台注册与登录 🔑

上一节我们介绍了本课程的整体目标，本节中我们来看看如何进入蓝墨云班课平台。

首先，您需要在手机应用商店或电脑浏览器中访问蓝墨云班课。下载并安装应用程序，或直接打开官方网站。

以下是完成初始设置的步骤：

1.  打开蓝墨云班课应用或网站。
2.  点击“注册”按钮，使用手机号或邮箱创建新账户。
3.  根据提示填写个人信息并设置密码。
4.  完成注册后，使用账号和密码登录平台。

登录成功后，您将进入个人主页，这是所有教学活动的起点。

---

## 第二课：创建与管理班课 📚

成功登录后，下一步就是创建您的第一个教学班课。

班课是组织教学活动的核心单元，相当于一个虚拟班级。创建过程非常简单。

以下是创建新班课的具体操作：

1.  在主页点击“+”号或“创建班课”按钮。
2.  输入班课名称（例如：**2023级春季高等数学**）、课程描述和封面图片。
3.  选择适合的课程分类与设置。
4.  点击“完成”即可生成一个专属班课号和二维码。

班课创建后，您可以通过班课号 **`INVITATION_CODE`** 或二维码邀请学生加入。在班课内，您可以利用顶部菜单进行各项管理。

---

## 第三课：管理学生与成员 👥

班课创建好后，我们需要将学生组织进来。

有效的成员管理是开展在线教学的基础。您可以通过多种方式让学生加入班课。

以下是管理学生成员的几种方法：

*   **邀请加入**：在班课详情页分享班课号或二维码，学生通过搜索或扫码加入。
*   **手动添加**：如果您有学生名单，可以手动输入学生信息并添加。
*   **成员分组**：进入“成员”页面，可以创建学习小组，方便进行小组任务与讨论。
*   **权限设置**：您可以指定助教或学生担任特定管理角色。

所有成员加入后，他们的名单会清晰显示在“成员”列表中，便于您后续进行考勤与学情跟踪。

---

## 第四课：发布教学资源与活动 📤

成员到齐后，就可以开始丰富的教学活动了。

蓝墨云班课的核心功能是支持教师发布多样化的教学资源与互动活动，以激发学生的学习参与。

以下是您可以发布的主要内容类型：

1.  **教学资源**：包括文档、PPT、视频、音频和链接。点击“资源”上传文件或输入网络链接即可。
2.  **课堂活动**：
    *   **签到**：发起一键签到或手势签到，公式可表示为 **`签到成功 = 学生定位 + 在规定时间内响应`**。
    *   **投票/问卷**：快速收集学生意见。
    *   **讨论/答疑**：发起话题，鼓励学生回帖交流。
    *   **作业/测试**：发布任务，设置截止时间与评分方式。
3.  **通知**：随时向全班发布重要通知，确保信息传达无误。

发布活动时，只需点击班课底部的“+”号，选择相应功能并按指引操作即可。

---

## 第五课：查看学情分析与报告 📊

教学活动结束后，评估学习效果至关重要。

蓝墨云班课提供了强大的数据分析功能，帮助您量化学生的学习进程与参与度。

以下是您可以查看的核心学情数据：

*   **经验值排名**：系统根据学生参与资源学习、活动完成度等行为自动计算经验值，生成排行榜。其计算逻辑类似于：
    ```python
    总经验值 = 资源学习值 + 活动参与值 + 签到值 + 其他奖励值
    ```
*   **活动报告**：每一项发布的投票、作业、测试等都有独立报告，显示提交人数、平均分、具体答案等。
*   **资源学习详情**：可以查看每位学生是否已查看您发布的资源，以及学习的时长。
*   **综合报告**：整合一段时间内所有学生的学习行为数据，形成可视化图表，便于整体评估。

您可以在班课的“活动报告”或“学情分析”板块中找到这些数据，它们是进行教学反思与个性化指导的重要依据。

## 总结

本节课中，我们一起学习了蓝墨云班课从注册登录到进行完整教学管理的全过程。关键步骤包括：**注册登录平台**、**创建班课**、**管理学生成员**、**发布资源与活动**，以及最后**查看学情报告**。掌握这些基础操作，您就能有效地利用蓝墨云班课组织在线教学，并跟踪学生的学习状态。
