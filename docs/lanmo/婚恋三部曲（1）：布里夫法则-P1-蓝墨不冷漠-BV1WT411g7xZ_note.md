# 婚恋三部曲（1）：布里夫法则 🦁

在本节课中，我们将要学习社会人类学家罗伯特·布里夫提出的“布里夫法则”。这条法则揭示了动物社群乃至人类社会中，两性关系背后的一种根本性选择逻辑。我们将通过狮群的例子来理解其核心，并探讨它如何影响现代人类的婚恋行为。

## 布里夫法则的起源与核心

罗伯特·布里夫出生于1876年，年轻时是一位外科医生，后来踏入社会人类学研究领域。他最著名的贡献就是这条“布里夫法则”。

**布里夫法则的核心是：在动物的社群里，雌性具有决定权而非雄性。当与雄性的结合无利可图时，这样的结合就不会发生。**

## 自然界中的例证：狮群社会

上一节我们介绍了法则的核心，本节中我们来看看它在自然界中的具体表现。草原上的狮群是一个众所周知的、以雄狮为中心的社会。

狮群中最强壮的雄性，即狮王，拥有交配权和优先享用食物的权利。雄狮在享用这些特权时，担负的责任是保卫地盘。大多时候无所事事的雄狮，在挑战者出现时，常常会赌上性命。如果他们输掉了这场保卫地盘与地位的争斗，就等于失去了最强雄性的地位。

而布里夫法则的残酷就在于此：
*   没有母狮会提供食物给败下阵来的雄狮。
*   没有母狮会去接近或者跟随失去地盘多年的落败者，因为他已经无法提供任何利益。

被驱逐出家园的雄狮必须找到下一个狮群抢夺地盘，但通常在负伤之下，无法重回巅峰。

## 两性的不同繁衍策略

自然界中，雄性与雌性间有不同的繁衍策略。雄性因为交配的代价低，倾向于与众多的雌性交配。而雌性必须担负起孕育和养育的责任，因此谨小慎微，会选择可以带来更多利益的交配对象。所以，**雄性间必须相互竞争以取得交配权**。

这对了解人类两性行为有很大帮助。在人类社会中，男性以金钱、社会地位、才艺、外貌等互相竞争，以博得女性的青睐。在现实中如此，在网络上更是如此。

以下是社交软件上的一个常见现象：
*   一位条件中等的女性，比起条件中等的男性，得到的“喜欢”可能是其数倍。
*   她们可以坐下来慢慢挑选最吸引她们的对象。

## 布里夫法则对人类婚恋的启示

布里夫法则解释了女性“慕强择偶”的本性，她们通常是“向上兼容”。换句话说，对女性来说，价值保障非常重要，因为她们要保证后代能更好地生存。

在部落时代，如果男人无法赋予女性更多的生存好处，那女性就不会与他结合，否则生存下去的几率就大大降低了。虽然现代社会好了很多，但这种基于本能的择偶倾向依然存在。

由布里夫法则可以衍生出三个重要推论。

**第一，男性在过去对女性所提供的好处，不一定能维持两者未来的关系。**

也就是说，不管你在过去为对方提供了多少价值，如果你自己本身创造价值的能力一直不高，那对方可能还是不喜欢你。例如“舔狗”，为什么提供了那么多价值，女性还是不喜欢呢？那是因为“讨好”这个行为本身，就是一个很低价值的行为。在部落时代，只有弱者讨好强者，只有无法生存的人，会通过讨好他人来获得食物生存。女性的基因会在潜意识里，让她们排斥这种通过讨好来获取资源的弱者。

**第二，男性为了维持未来与女性的关系，所提供的任何当前的好处，大多数都是短暂的。**

只有提供一种让女性的基因觉得可以持续发展价值的东西，我们通常称之为“潜力股”。即使你现在穷困潦倒，但是你霸气外露，时刻坚信“老子一定能打下一片大大的江山”的信心，那这个东西会让一个男人特别有魅力。而且只要你行为上也照着你的想法去做，那就是一种领袖的气质，不仅会吸引女性，更会吸引更多的男性愿意追随你。

**第三，男性向女性承诺未来好处，对当前及未来两者的结合所起到的作用十分有限。**

这俗称“画饼”，但饼会露馅儿。换句话说，女性更注重短期收益。择偶期的大多数年轻女性，无法理解长期收益是什么。所以有些男人会好奇，为什么我这么努力、工作上进，就是没有女生看得上我？其实很简单，因为你不能在当下立即，给女生看得见、摸得着的好处。

## 社会价值观与生物本能的冲突

这与“自信”是不一样的。自信或“装杯”，是一种雄性自我实现的行为。“我装杯是自我需求，不管有你没你我都要做”，这更像是一种老虎咆哮森林、宣誓地盘的行为。

我们相信忠诚、荣誉、饮水思源与责任感等美德。但这些价值，是社会价值观投射到男性身上的价值。实际上，拥有并重视这些价值的女性占极少数。当我们也将这些概念投射到女性身上，当我们因相信而付出、期待承诺或回报时，就落入了布里夫法则的陷阱。

这些价值不是任何人天生就有的，是后天家庭、社会教导男性要重视这些价值，比如男生要绅士、要会照顾女生、保护女生。不论是精心安排还是机缘巧合之下，促成了现在对女性有利的社会环境，女性也许也在成长过程中，耳濡目染地接受了这些价值。但普遍来说，她们最大的利害关系，不是这些社会价值，而是她以及她血亲的利益。

生物的本能让我们去确保自身基因的传承，包含她的子女，以及其他共享她部分基因的亲人。而“理”，从来就不在她本能的考量范围里。
