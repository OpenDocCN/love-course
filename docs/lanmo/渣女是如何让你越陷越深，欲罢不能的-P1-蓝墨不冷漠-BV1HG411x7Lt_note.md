## 五种核心心理策略

上一节我们概述了情感操控的存在，本节中我们来详细看看五种具体的策略。

### 1. 捧杀：激发表现欲

每个人都天生喜欢被崇拜的感觉，擅长情感操控者深谙此点。当她们想快速从目标身上获取好处时，最高效的方法就是满足其被崇拜的需求。

以下是具体做法：
*   将对方抬高到非常重要的地位。
*   让对方沉浸在崇拜的目光和交替的甜言蜜语中。
*   这会使对方激发强烈的表现欲，更努力地展示自己，甚至心甘情愿地为对方付出一切。

### 2. 制造焦虑：利用竞争心理

上一节我们看到了如何通过赞美让人主动付出，本节中我们来看看如何通过制造压力来操控行为。擅长情感操控者最擅长利用人的竞争心理来制造危机感。

**核心逻辑**：`制造稀缺性 + 暗示竞争者存在 → 引发焦虑与争夺行为`

以下是具体做法：
*   抓住对方的痛点（如害怕失去）来制造焦虑。
*   她们通常会同时与多人保持联系，一方面将感情视为交易，另一方面也需要足够多的“选项”来营造自己很抢手的行情。
*   为每个目标塑造潜在的竞争对手。你越焦虑她会被抢走，就越会拼命对她好以吸引其注意。

### 3. 让利：制造“赚了”的错觉

**核心操作**：`自身高价值展示 + 对目标进行打压 → 产生自卑与亏欠感`

以下是具体做法：

*   与此同时，她会时不时地打压你，例如指出你不如她的同事帅、不如老板有钱、不如男闺蜜贴心。
*   这种对比会让你产生“她和你在一起是你高攀了”的感觉，从而产生自卑和亏欠情绪。此后，无论她提出什么要求，你都更容易义无反顾地答应。

*   这种情绪上的起伏让你时而难受、时而快乐，但总能让你对她魂牵梦萦。
*   久而久之，你会习惯生活中有她的情绪波动。一旦她突然消失，你甚至会感到生活平淡、无趣。

### 5. 制造“舒适”的孤立：圈养与隔绝

最后一种策略是构建一个封闭的环境。有些操控者自己行为开放，却要求对方百分百专一。
