# 约会迟到怎么办—情形3：故意的迟到 🕐 - P1

在本节课中，我们将要学习约会中一种特殊的情况：**故意的迟到**。我们将探讨其背后的策略逻辑、如何区分邀约类型，以及如何利用这种“测试”来判断对方的真实意愿。

---

可能有人会不明白，明明已经约好，为什么还要故意迟到。如果说前两种迟到情形是无意识的，那么这种故意的迟到，更像是一种约会策略。

首先，我们需要明确邀约的两种基本类型：**模糊邀约**和**直接邀约**。

以下是这两种邀约的定义与区别：

*   **模糊邀约**：向对方发出邀约意向，但未确定具体细节。例如：“下次有空一起吃饭。” 这种邀约的作用是给对方植入一个未来可能约会的心理预期。
*   **直接邀约**：向对方发出包含具体时间、地点的明确邀约。例如：“今晚六点在海底捞一起吃个饭，然后我们去酒吧坐会儿。” 这就是一个确定的、直接的邀约。

上一节我们介绍了邀约的类型，本节中我们来看看如何将“故意迟到”作为一种策略应用于直接邀约中。

例如，我与一位女生约好晚上七点一起吃饭。我可能会在六点，即女生准备出门前，给她发送一条消息：“我这边可能会迟到十五分钟，我们七点十五见吧。”

这个行为的核心意图在于：**判断女生是否真的期待这次约会**。

以下是这个策略背后的逻辑分析：

*   如果女生本身并不想见面，或临时反悔，但又因已答应而不愿承担“不守信用”的标签，她可能会不情不愿地前来。这种约会体验很差，且没有意义。
*   告知“可能迟到十五分钟”对约会本身影响不大。如果女生欣然接受，说明她是真心想见面而赴约。
*   如果女生顺势说：“那算了吧，不如下次。” 这便判断出她本意就不想出来。你恰好给了她一个拒绝的台阶。

因此，这可以说是一个判断测试，是帮助你筛选对方真实意愿的小技巧。

总而言之，无论是谁迟到，当特殊情况发生时，不必着急。仔细分析其中的原理，利用一些特定情况，也能达到特定的效果，这并不完全是件坏事。

---

本节课中我们一起学习了“故意迟到”作为一种约会策略的应用。我们明确了模糊邀约与直接邀约的区别，并分析了如何通过一个简单的“迟到测试”来有效判断对方的约会诚意，从而避免无意义的社交投入。
