## 第一节：好感情的标准是什么？

上一节我们提出了核心问题，本节中我们来看看一段好的感情究竟应该是什么样的。

但一旦遇到分歧，便争吵不休，最终吵到分手。因此，衡量一段感情，不是看好的时候有多好，而是看最差的时候有多差。只有解决了最差的情况，感情才能走得更远。
