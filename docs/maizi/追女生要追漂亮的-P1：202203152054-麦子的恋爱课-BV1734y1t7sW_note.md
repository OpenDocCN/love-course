# 追女生要追漂亮的 - P1：202203152054 - 麦子的恋爱课 🎯

---

追女生一定要追美女。不敢追美女，或者觉得漂亮不重要的，都是错误的观点。

总有一些人在评论区留言说，娶老婆最重要的是心肠好，颜值不重要。这种言论非常片面。心肠好和漂亮并不冲突。

在对方同样心肠好的前提下，你愿意找个漂亮的，还是愿意找个丑的？如果说追不到，这可以理解。但如果说自己不喜欢，对美女没有欲望，这显然不正常。

一个男人的初心，就是喜欢美女，喜欢漂亮的、身材好的、腿长的。男人可以没有房，没有车，但不能说对漂亮的女人没有欲望。

男人要活得纯粹一点。哪怕失去了所有东西，如房子、车子、游戏、钓鱼，最后剩下的那一点，也应该是喜欢美女。这是所有男人的共性。

---

上一节我们讨论了追求美女是男人的天性，本节中我们来看看为什么说漂亮的女生其实更好追。

越漂亮的女生，其实越好追。你不要觉得女生这么漂亮，肯定有很多人追。你这么想，别人也这么想。结果就是，漂亮的女生，反倒没人敢追。

只要你稍微用点心，就能追到。相反，那些长相一般的女生，才是最难追的。因为你觉得她好追，别人也觉得她好追，导致你的竞争对手太多。

以下是这类女生可能带来的挑战：

*   这些女生会更敏感多疑。
*   她总觉得，你追她不是因为喜欢她，而是你觉得她好追。
*   所以她会各种为难你。

最爱钓着男人的，往往不是美女，而是那些长相一般的女生。
