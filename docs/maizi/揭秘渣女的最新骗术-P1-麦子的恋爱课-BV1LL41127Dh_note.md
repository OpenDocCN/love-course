## 骗局核心模式 🔍

上一段我们提到了本节课的主题，本节中我们来看看这种骗局的具体运作模式。

这种骗术的核心在于**不对等的价值交换**。其基本流程可以概括为以下公式：

**骗局流程：女生赠送（低价/假货） -> 索取回礼（高价/真品） -> 循环往复**

具体来说，女生会主动先送给男生礼物，然后要求男生回赠价值更高的礼物。如此循环，女生付出的总成本极低，而男生则花费巨大。

## 价值不对等的本质 💰

理解了基本模式后，我们进一步分析其关键：礼物价值的严重不对等。

*   **女生赠送的礼物**：通常是手链、玉佩等**容易造假、价格不透明**的商品。你很难判断其真伪与实际价值。
*   **男生回赠的礼物**：通常是护肤品、手机等**明码标价、价值透明**的商品。

**举例说明**：
1.  女生送出一条标称500元的手链（实际成本可能极低）。
2.  女生要求男生回赠一套1000多元的护肤品。
3.  女生再送出一块标称2000多元的玉佩（实际成本可能仅数百元）。
4.  女生再次要求男生回赠一部3000多元的手机。

在这个例子中，女生可能仅花费了少量成本（例如500元），却成功换取了男生价值5000元的真实商品。
