在本节课中，我们将要学习一些在恋爱关系中，女生可能表现出的、具有普遍性的行为习惯。了解这些行为有助于增进彼此的理解，让关系更加和谐。

上一节我们介绍了课程主题，本节中我们来看看这些具体的行为表现。

以下是常见的六种行为模式：

1.  **咬人**：女生会时不时地想咬伴侣一下，且力度可能没有轻重。这种行为源于强烈的喜爱，其心理公式可以概括为：**喜欢程度 ∝ 想“吃掉”对方的冲动**。
2.  **掐手臂**：男朋友的手臂是她们最喜欢掐的部位，从中能获得某种愉悦感。
3.  **随时随地亲吻**：只要旁边没有旁人看见，她们就想亲吻伴侣，地点不受限制。其行为逻辑类似于一段条件判断代码：
    ```python
    if 周围无人 == True:
        执行亲吻(伴侣)
    ```
4.  **把男朋友当“垃圾桶”**：自己不想吃的食物、不想用的物品，都会交给男朋友处理。
5.  **把腿放在对方腿上**：只要环境允许，她们喜欢将腿搁在伴侣的腿上，将其当作舒适的垫腿石。
6.  **喜欢被搂着睡觉**：因为被伴侣搂着睡能带来极大的安全感。
