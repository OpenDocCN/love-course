# 麦子的恋爱课 P1：暧昧聊天技巧 🎯

在本节课中，我们将学习如何在与女生进入暧昧阶段时，通过高情商的对话技巧，自然地推进关系并营造暧昧氛围。核心在于避免过于正经的对话，学会用轻松、有趣的方式“调戏”对方。

上一节我们介绍了课程主题，本节中我们来看看具体的操作话术。

以下是三个高情商的撩妹话术示例，每个话术都设计了一个互动环节，引导女生进入你设定的轻松、暧昧的聊天框架中。

1.  **关于“特长”的提问**
    *   你可以对她说：“跟你聊天这么久了，你还不知道我的特长吧。”
    *   当女生好奇地问“什么特长”时，你回答：“**想你的时间特长**。”

2.  **关于“冬天被子”的提问**
    *   接着问她：“你知道我冬天喜欢盖什么被子吗？”
    *   她通常会回答“我怎么知道”，此时你回复：“是**你这个暖暖的宝贝**。”

3.  **关于“对联与横批”的邀请**
    *   临近睡觉时间，你可以对她说：“长夜漫漫无心睡眠，有感而发出个对联，你来帮我想个横批吧。”
    *   然后给出对联：
        *   **上联**：天已黑夜已静，全身上下洗干净。
        *   **下联**：没人搂没人绑，一个人睡觉真的很烦躁。
    *   此时，她会自然地顺着你的思路去思考横批。无论她给出什么答案，你都可以说：“我的横批就是**陪我聊聊**。”

通过以上对话，聊天的气氛会变得轻松而暧昧。这为接下来的“深夜畅聊”创造了自然的契机。

本节课中我们一起学习了三个在暧昧阶段推进关系的具体话术。关键在于通过提问和互动，营造轻松有趣的氛围，从而引导对话向更亲密、更深入的方向发展。
