## 容易丢失主动权的五种心理

在追求或恋爱关系中，一些特定的心理状态会显著削弱你的主动地位。以下是五种典型的、容易导致丢失主动权的心态。

### 1. 依赖心理
持有这种心态的人认为，只要持续对对方好，对方迟早会被感动并接受自己。这种想法将关系的进展完全寄托于单方面的付出上。

### 2. 畏惧心理
这类人害怕打破现状，认为只要待在对方身边，就总会有机会。他们往往不敢推进关系，只是被动等待。

### 3. 赌博心理
这种心态表现为不计后果地行动，例如在不合适的时候鲁莽表白。其核心逻辑是“不管成败，先去表白试试”。

### 4. 退缩心理
一旦察觉到对方的反应不是非常积极或热烈，持有退缩心理的人就会立刻萌生退意，准备放弃。

### 5. 蛮干心理
与退缩心理相反，蛮干心理是指完全忽略对方的反馈和感受，只固执地按照自己的想法和节奏行事。

以上任何一种心态，只要让女生感觉到不舒服或压力，你就会陷入更加被动的境地。

## 丢失主动权的核心原因

上一节我们介绍了五种导致被动的心态，本节中我们来看看其背后的根本原因。

丢失主动权的核心原因是**需求感**。强烈的需求感会催生自卑心理，而自卑又会进一步导致自我矮化。

**自我矮化**的公式可以描述为：
```
自我评价 < 对方评价
```
在这种心态下，你会觉得自己样样配不上对方，方方面面都不如对方。一旦出现这种心态，追求成功的可能性就微乎其微。

具体来说，当你对女生产生了需求——比如你动心了，并且想要得到这个女生——这就是需求。需求就是“想要得到却暂时无法被满足”的状态。就像饿了想得到一碗饭，渴了想得到一碗水，但暂时得不到一样。

**关键结论是**：在感情互动中，谁先动心，谁往往就会更被动。因为需求感会直接转化为在关系中的低位。

## 丢失主动权的后果

当一个人因上述心态而完全丢失主动权后，最终的结果往往是“赔光一切”，然后只能默默退出。因此，丢失主动权通常意味着关系走向失败的开始。
