# 恋爱课 P1：解读女生“想要了”的五句暗示 🎯

在本节课中，我们将学习如何解读女生在对话中可能发出的、表达好感和兴趣的五句关键暗示。理解这些潜台词，有助于你把握时机，推进关系。

无论男女，喜欢上一个人时，都会时刻想知道对方在做什么，因为会时刻想到对方、想要见到对方。因此，当女生经常问“你在干嘛呢”时，不要觉得她烦。这其实是一句情话，翻译过来就是：**“我想你了。”**

**4. “谁谁谁又恋爱了。”**

**5. “你喜欢什么样的女生？”**

这一点可谓最为明显。女生通常比较矜持，喜欢一个人时会反复试探。她表面上问你“你喜欢什么样的女生”，实际上想问的是：**“你喜欢我这样的女生吗？”** 因此，回答这个问题时千万不要敷衍。一定要描述女生身上的具体特征，你夸赞的细节越多，两人关系水到渠成的可能性就越大。
