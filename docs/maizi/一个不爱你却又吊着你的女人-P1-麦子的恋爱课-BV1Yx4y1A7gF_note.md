# 一个不爱你却又吊着你的女人-P1-麦子的恋爱课-BV1Yx4y1A7gF_note

在本节课中，我们将学习如何识别一种不健康的恋爱关系模式：对方并不爱你，却通过一系列行为让你持续抱有希望，从而“吊着你”。了解这些具体表现，有助于你更清晰地判断关系现状，保护自己的情感。

---

上一段我们概述了本节课的主题，接下来，我们将逐一拆解这种关系中常见的四种具体表现。以下是核心的四种行为模式：

1.  **需要帮忙时才联系你**
    她只在心情不好需要倾诉，或有事需要帮忙时才会主动联系你。在平时，她则懒得搭理你。

2.  **单向接受物质付出**
    你为她花的钱，例如购买的礼物、发送的红包、请客吃饭等，她会照单全收。但她从来不会为你花钱。公式可以表示为：**付出 → 她 | 付出 ← 你 = 0**。

3.  **对你的示爱装傻回避**
    你喜欢她这件事，她不可能感觉不到。她选择装傻，这样便可以只享受你对她的好，而无需为此付出任何承诺或回报。

4.  **在你欲放弃时给予希望**
    当你感到疲惫、准备放弃时，她又会主动来关心你，给你一点点希望，让你舍不得放下，从而继续被她吊着。这形成了一个循环：**放弃意图 → 她给予微小关怀 → 重燃希望 → 继续付出**。

---

本节课中，我们一起学习了“不爱你却吊着你”的四种关键表现：**工具化联系**、**单向物质索取**、**情感装傻**以及**间歇性给予希望**。识别这些模式是建立健康关系认知的第一步。希望这些分析能帮助你更客观地看待关系，做出对自己更负责任的决定。
