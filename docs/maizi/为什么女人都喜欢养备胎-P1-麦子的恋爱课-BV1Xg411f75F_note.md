# 恋爱课 P1：为什么女人都喜欢养备胎？🤔

在本节课中，我们将通过一个简单的故事，探讨“养备胎”这一行为背后的深层逻辑与历史渊源。我们将从进化心理学的角度，理解这种行为模式的形成原因。

---

在远古时期，一个部落首领的妻子需要依靠男人获取食物，因为她自身不具备狩猎能力。如果她不为自己准备后路，生活会面临巨大风险。

例如，她怀了部落首领的孩子。某天，首领在外出狩猎时不幸身亡。结果会怎样？她失去了食物来源。其他男人知道她怀的不是自己的孩子，很可能不会供养她。这个女人和她未出生的孩子就可能面临饿死的困境。

那么，如果她“养备胎”情况会如何不同呢？

以下是故事发展的另一种可能：

*   在部落首领外出时，她与部落里的一个年轻人建立了秘密关系。
*   她捧着这个年轻人的脸说：“我真正爱的是你。我不爱那个老头。我现在怀孕了，怀的是你的孩子。但这是我们的秘密，你不能告诉他。”
*   实际上，孩子生父是谁，女人自己可能也不确定。在缺乏DNA检测的古代，模糊血缘关系对女人有利。
*   她认为孩子是她的，年轻人认为孩子是他的，这样最好。两个男人都可能因此供养并善待她。
*   后来，部落首领真的意外身亡了。这个年轻人会继续把食物分给女人，因为他相信孩子是自己的。
*   并且，这个年轻人很可能成为新的部落首领。于是，女人得以继续维持优越的生活。

这个故事的核心逻辑可以概括为一个简单的策略公式：**最大化生存保障 = 分散风险 + 模糊血缘关系**。

历史上，将这一策略运用得极为成功的例子是武则天。她最初是唐太宗李世民的妃嫔，后又与唐高宗李治结合。其结果是：**无论是父亲（李世民）还是儿子（李治）当皇帝，她都能成为皇后乃至皇帝。这确保了她在权力更迭中始终处于不败之地。**

所以，不要总是用“养备胎”这样难听的词汇去指责。从女性的生存策略来看，这可以被理解为“多个选择多条路”。其行为逻辑类似于播种：**种下一颗种子（建立多个潜在关系），就可能收获许多果实（获得多重生存保障）**。

既然这种策略能显著提高生存概率，那么“我为什么不这么做呢？”这便成了一种深植于历史与本能的选择。

---

本节课中，我们一起通过一个远古故事，理解了“养备胎”行为背后可能存在的进化心理学原因。我们看到了在资源依赖环境下，分散风险、确保生存是一种深刻的策略选择。历史案例也表明，这种策略在复杂的社会关系中同样适用。理解这一底层逻辑，有助于我们更理性地看待两性关系中的某些现象。
