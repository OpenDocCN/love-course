# 恋爱基础课 01：掌握关系主动权 🎯

在本节课中，我们将要学习如何建立健康、平衡的两性关系，避免因过度付出而失去吸引力。核心在于理解人性心理，通过提升自我价值和引导对方投入来获得主动权。

## 过度讨好的弊端

上一段我们提到了课程主题，本节中我们来看看为什么对女生太好反而效果不佳。

在女生眼里，讨好行为等同于跪舔。一个男人无论多优秀，只要他在关系中为女生付出太多，愿意做任何事情，他就不会得到善待。

**核心观察**：你对她越好，她可能觉得你越廉价；你对她高冷，她反而可能对你越来越上头。

## 激发征服欲是关键

理解了过度付出的问题后，我们来看看女生在关系中的真实需求。

80%的女生在关系中，并不需要你对她有多好。她们渴望的是你能激起她们的征服欲。因此，不断讨好她，她可能越来越不搭理你；适当“吊着她”，她反而可能对你欲罢不能。

## 放大你的价值

那么，如何正确激发对方的兴趣呢？关键在于放大你的价值。

你的价值高低，并不完全取决于你本身的价值，而是取决于她为你投入了多少。人性如此：投入越多，越重视。想让对方持续喜欢你，就需要引导她不断为你投入时间、精力和情感。

**价值公式**：`感知价值 = f(对方投入度)`

## 学会“来回拉扯”的技巧

放大价值需要方法，一个有效的方法是进行“来回拉扯”。

例如，有时对她表示关心，让她“上头”，觉得你喜欢她；接着适时撤离，比如不回消息，让她等待，觉得你好像又没那么喜欢她。这会让她陷入猜测，思考你的真实意图。

以下是操作的一个简单循环：

1.  **表达兴趣**：给予适度关注和关心。
2.  **适时抽离**：减少联系，制造不确定性。
3.  **引发猜测**：让对方思考你的态度和动机。
4.  **重复循环**：根据对方反应，重复以上步骤。

## 改掉暴露需求感的毛病

掌握了拉扯技巧后，要巩固主动权，还需从自身习惯入手。

想要获得主动权，首先要改掉动不动就想表达需求的坏毛病。不要频繁主动表达“我有多爱她”、“我有多离不开她”。暴露过多需求感，只会降低你的价值。

**人性原理**：自己通过努力得到的东西，永远比别人给的更加珍贵。

## 爱人三分，吃饭七分

最后，我们来总结建立健康关系的核心心态。

请记住：爱人三分就好，吃饭七分就饱。健康的感情中不需要跪舔。女朋友从来都不是讨好来的，而是吸引来的。一段感情的开始关键也不是付出，而是吸引。

当然，提升吸引力和经营关系的能力需要学习与实践。这个世界上天赋异禀的人不多，优秀的人总是在努力翻山越岭。在努力提升自己、努力赚钱的同时，别忘了学会如何去爱，学会如何让你爱的人也爱上你。
