# 追女生的最佳心态——猎人心态 🎯 - P1：横 - 麦子的恋爱课

在本节课中，我们将要学习一种被称为“猎人心态”的追女生核心心态。我们将剖析为什么你越喜欢的女生越难追到，并详细拆解猎人心态的四个核心要点，帮助你建立正确的心态框架，从而在感情中掌握主动权。

---

你有否发现，那些你越喜欢、越想追的女生，反而追不到。而那些抱着随便撩一撩想法去接触的女生，反而很容易追到。

之所以会这样，是因为你的心态不对。你和喜欢的女生交往时，你害怕搞砸了，害怕失去她。所以你和她聊天的时候，你会想，我这句话要怎么回，才能回得很完美。因此你的聊天就会显得很刻意。约会的时候，你也会紧张拘束，导致尬聊冷场。

你完全不是真实的自己，你更像戴着一个社交面具。你总会想，我这样说会不会让她不高兴，我这样做会不会她不理我。你不敢表达自己的喜怒哀乐，就像一个机器人一样。你越是这样，女生越没感觉。

上一节我们分析了错误心态的表现，本节中我们来看看问题的根源和正确心态是什么。

之所以会这样是因为你的心态出了问题。那追女生的正确心态是什么呢？就是**猎人心态**。

猎人心态的核心一共有四点。

以下是猎人心态的第一个核心要点。

**一要沉住气，不要轻易出手。**
一个好的猎人，他会先伪装隐蔽，等猎物靠近了再出手。而不是一上来就贸然开枪。很多男生在追女生的时候，喜欢一上来就表达“我有多喜欢你，我有多爱你”，然后展开疯狂的追求。这种急不可耐、想要得到她的行为就会显得你很轻浮、很饥渴，同时也暴露了很强的需求感。和猎人狩猎一样，你一旦暴露了，就很难追到她了。真正的高手会先隐藏自己的需求感，先对女生表达无兴趣，等女生对自己有好感之后再出手。

理解了沉住气的重要性后，我们来看看如何在互动中保持真实。

**第二点，敢于表达自己的真实想法。**
在真正出手之前，猎人会先伪装自己，不会轻易地惊动猎物。但千万不要让你的猎物以为他才是主人。很多男生在追女生的时候，总是习惯性地去讨好。女生提的一些要求，他都会尽力地去满足，不敢拒绝，哪怕内心很不愿意，也不敢说出来。如果你不敢得罪女生，只会迎合讨好她，那在女生看来，你就是一个没有魅力的菜鸟猎人。你越是讨好她，她越看不起你。当她仗着你对她的好肆无忌惮地索取时，你就应该把枪亮出来，给她点颜色看看。因为女生都是慕强的。从远古开始，女生就是依附于男人而活。女生都希望吸引到优秀的猎人来捕猎她。对女生来说，一个**有血有肉、敢做敢说的猎人**比任劳任怨的追求者更能给她恋爱的感觉。

除了表达真实想法，一个优秀的猎人还必须有自己的行为准则。

**第三，制定自己的规则标准。**
一个优秀的猎人一定有自己的原则底线。例如：
*   **天黑之前必须要下山回家**，不管今天有没有收获。
*   **女生不回信息只能追加一条**，如果还不回，那今天就不要聊了。
*   **超过三百斤的熊很危险**，见到了就一定要躲开，不要想着狩猎她。
*   **整天为你要钱的女生都是渣女**，碰到了直接拉黑，不要想着去追她。

男生谈恋爱一定要有原则底线。例如：
*   连续邀约三次都不出来的，放弃。
*   连续三次约会都不让牵手的，放弃。
*   一个月还没有升级关系的，放弃。
*   直接和你要红包要礼物的，放弃。

轻易妥协就是不尊重自己，最后的结果就是人财两空。

最后，我们来探讨猎人心态中最具决定性的特质：领导力。

**第四，要有领导力。**
猎人会自己决定要不要开枪，而不是去问猎物：“我可以打你吗？”。很多男生在追女生的时候会不停地问女生：“我可以牵你的手吗？”、“我可以抱你吗？”。点菜的时候问他：“你想吃什么？”、“我们吃完去哪玩？”。女生不喜欢做决定，她喜欢听男人的意见，这就是女人的天性。所以在感情里，女生往往是被动的一方，男人要主导这段关系的发展。例如：
*   男人要主动开启聊天，主动找话题。
*   聊得差不多了，要主动结束聊天，和女人说“今天有点晚了，早点睡吧”。
*   聊得很好的时候，男人要主动邀约，说“有空一起吃个饭吧”。
*   约差不多的时候，男人要主动去牵手。
*   包括后期的表白确定关系，包括见家长、什么时候结婚，包括两个人吵架了，男人也要主动地去和好，不要打冷战。

你越是强势地主导这段关系，就越容易给女生恋爱的感觉。
