## 核心潜规则：越在乎，成功率越低

追女生有一套潜规则。你对女生越在乎，你追到的成功率越低。你表现得越不在意，女生反而越喜欢你。可以说，**无所谓的心态才是撩妹的关键**。

## 具体操作指南

上一节我们介绍了“无所谓”心态的核心作用，本节中我们来看看具体该怎么做。以下是三个关键要点。

正确的回复应该是把女生的情绪带动起来，例如开玩笑地说：“瞎说，你哪有肚子呀？”

追女生是“有心栽花花不开，无心插柳柳成荫”。你越在乎结果，发挥就会越差，全程会表现得很不自然。和妹子聊天时会紧张，会不自觉地去讨好女人。在女生眼里，这是没有魅力的。

以上这些行为都是过度关注女生、围着女生转。说白了，这就是“舔狗”行为。

过度关注一个女生，只会让你的心态炸裂，恋爱脑上头，最后成为舔狗。
