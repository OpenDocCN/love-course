# 追女生要学会吹牛逼 - 课程P1 - 麦子的恋爱课 🎯

在本节课中，我们将要学习一个关键的社交技巧：如何通过“包装”自己的语言和信息，在追求女生的过程中建立更好的第一印象，避免因过于直白而减分。

---

想成功追求女生，必须学会适当地“吹牛”。许多人至今单身，原因在于过于实在，尤其是在和女生聊天时，只会实话实说。

例如，昨天有一位粉丝，在聊天中主动告诉女生：“我被公司开除了。” 你很诚实，说的也是事实。

但这对于你追求她有何帮助？女生会如何想？你被公司开除，说明你工作能力差，老板不想用你。这首先传递了负面价值。女生会顾虑，如果以后结婚，你轻易被开除，她和孩子的生活如何保障。

这种聊天方式纯粹是给自己挖坑。“被开除”这种话有必要说吗？如果非要说，也必须换一种方式来包装自己。

你需要换个说法。你可以说：“我主动离职了。离职的原因是公司给的薪水太低。以我目前的能力，换一份工作能挣更多钱。” 这在女生看来是加分的。

或者说：“之前的公司规模小，发展空间有限，有能力的人也不多，我学不到东西。我需要找一家大公司，多学本事历练自己，未来才能挣大钱。”

这也是加分的。女生会认为你是个潜力股，有野心和魄力，从而对你产生好感。

**核心概念：同一件事，不同的表述会导致完全不同的效果。**
*   **减分表述**：`我被公司开除了。`
*   **加分表述**：`我主动离职，为了挑战高薪/寻求更大发展平台。`

男人一定要记住，学会用“吹牛”来包装自己。例如，如果你的职业听起来普通，可以尝试用更专业的说法来重塑印象。

以下是几个职业表述的转换示例：
*   **搓澡的**：人体表皮污垢清理工程师。
*   **收废品的**：无用外包装管理协会会长。
*   **看大门的**：进出口管理高级工程师。
*   **保安**：区域安全系统检察员。
*   **爆米花的**：爆谷类食品加工爆破工程师。

任何工作，换一种说法，都能立刻显得高大上。
