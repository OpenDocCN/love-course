# 恋爱心理学基础课 - 第一讲：四个关键定律 🧠

在本节课中，我们将要学习四个关键的心理学定律。理解这些定律，有助于你在感情关系中建立更健康、更平衡的互动模式，避免常见误区。

## 1. 贝勒效应：为何付出难以持续获得感动 💔

上一节我们介绍了本课程的目标，本节中我们来看看第一个定律——贝勒效应。该效应指出：当一个人经历强烈的刺激后，后续施加的刺激会显得微不足道。

**公式**：`后续刺激感知度 = 初始刺激强度 / 已接受刺激的累积量`

这意味着，你对一位女性的好，通常只会让她感动一次。例如，你花费8000元为她购买一部苹果手机，她初次收到如此贵重的礼物，可能会感动数日。一个月后，你再次赠送一台iPad，她的感觉可能已趋于平淡。再过一个月，你花费3万元购买一个包，她可能觉得在意料之中。持续数月如此付出后，她会将这一切视为理所当然，不再感动或感恩，甚至认为这是你应做的。

## 2. 墨菲定律：越害怕失去，就越容易失去 😰

理解了单方面过度付出的问题后，我们来看看第二个定律——墨菲定律。其核心是：你越担心某事发生，它就越可能发生。

**代码描述**：
```python
if fear_of_losing == True:
    probability_of_losing.increase()
```

如果你从一开始就认为“这位女生对我至关重要，我必须追到她，不能有任何失误”，那么在后续交往中，你会变得紧张、拘束、小心翼翼。这种状态会让对方感到不自然，从而削弱吸引力。同时，你可能会在聊天中本能地讨好、迎合对方，将她捧上“神坛”，自己则处于低位。这会导致对方盲目自信，认为可以找到更好的选择，因而更看不上你。

## 3. 沉没成本效应：为何难以离开一段不好的关系 ⛓️

上一节我们探讨了恐惧心态的影响，本节我们来分析第三个阻碍——沉没成本效应。它指的是：人们因已投入且无法收回的成本（时间、金钱、精力）而难以放弃当前行动。

**公式**：`决策粘性 ∝ 已投入的沉没成本`

例如，你追求某人两年，花费十万元。即便明知对方不喜欢你、对你不好，你也难以放手，因为你会觉得放弃意味着这两年时间和十万元彻底浪费。然而，沉没成本已然无法收回。若不懂及时止损，继续盲目投入，最终可能浪费五十万元和五年时间。反之，若想让对方更珍惜你，应引导她为你付出时间、金钱和情感。投入越多，她才可能越珍惜。

以下是引导对方投入的几个方向：
*   **时间投入**：邀请她共同参与需要花费时间的活动或计划。
*   **金钱投入**：在合理范围内，让对方也承担部分约会或共同开销。
*   **情感投入**：鼓励对方分享想法、为你做一些小事，增加情感联结。

## 4. 蔡格尼克记忆效应：如何提升自身吸引力 ✨

我们已讨论了过度付出、恐惧心态和沉没成本。最后，我们学习第四个定律——蔡格尼克记忆效应，以主动提升吸引力。该效应表明：人们对未完成或未得到的事物记忆更深刻，感觉更珍贵；反之，对轻易得到或永远不会失去的东西则不易珍惜。

**核心原则**：`感知价值 ∝ 1 / 获得的难易度`

90%的男性容易陷入两个极端：一是过早地全身心付出、倒贴，让对方毫无挑战性地获得全部好处；二是告诉对方“我会永远等你”，这通常导致沦为永久备胎。感情关系本质上是男女心理的博弈。只有理解并合理运用人性，才能在关系中占据高位。占据高位并非为了操控，而是因为高位者通常更有能力用心维护一段关系的健康与平衡。若想建立长久稳定的关系，**必须先确立自身框架与价值，再对对方好**。

**本节课总结**：本节课我们一起学习了四个关键的心理学定律：**贝勒效应**（刺激递减）、**墨菲定律**（怕什么来什么）、**沉没成本效应**（难以割舍已投入的）以及**蔡格尼克记忆效应**（未完成的更珍贵）。理解这些定律，旨在帮助你避免单方面过度付出、克服恐惧心态、学会及时止损，并通过提升自身吸引力和引导双向投入，在感情中建立更平等、健康的高位姿态。记住，健康的关系基于相互尊重与价值吸引。
