假设有一个女生，她同时在和10个男生聊天。在这10个男生中，有1个是她特别喜欢的，另外9个是她完全不喜欢、也看不上的。

这个例子清晰地表明，**相同的语言和行为，会因发出者是否具备吸引力而产生完全相反的效果**。
