**原则一：投资于共同体验**

理解了投资共同体验的重要性后，我们来看看需要避免的消费类型。

**原则三：优先投资于自身**

有钱尽量花在自己身上。例如，和女生逛街时，你可以为自己购买价格较高的衣服或裤子，也可以顺便为你的父母买些礼物。

相反，整天只为她购买物品的人，可能会被她视为“池塘里最大的那条鱼”。当你过度投资于她，使她感觉变得“优秀”时，她反而可能觉得你配不上她。
