# 追女生的三不原则 🚫 - 课程P1 - 麦子的恋爱课

在本节课中，我们将要学习一个在追求女生过程中至关重要的“三不原则”。掌握这个原则，可以帮助你有效筛选对象，避免在不合适的人身上浪费时间和情感，从而更有可能找到真正相互喜欢、双向奔赴的伴侣。

## 原则概述

追女生有一个“三不原则”。遵循这个原则，你绝对不会遇到渣女，并且谈的恋爱都是与真正喜欢你、愿意双向奔赴的女生。

记住，如果你遇到符合以下情况的女生，无论如何追求都只是在浪费时间。

## 三不原则详解

以下是“三不原则”的具体内容，每一条都对应着一个明确的筛选标准。

1.  **连续约会两次都约不出来的不要。**
    这意味着，如果你主动邀约两次，对方都找理由拒绝或无法成行，那么就应该及时止损。这通常表明对方兴趣不足。

2.  **三次约会还一点都不主动不能推进关系的不要。**
    在三次约会之后，如果对方依然完全被动，没有任何主动联系或推进关系的迹象（例如愿意进行更深入的交流或适当的肢体接触），说明关系难以健康发展。

3.  **四次约会女生一分钱都不花的不要。**
    经过四次约会，如果女生从未在任何消费上有所表示（例如主动买杯饮料、付一次小账），这可能意味着她只是在单方面享受你的付出，缺乏共同经营关系的意识。

## 总结

本节课中，我们一起学习了“追女生的三不原则”。这个原则的核心在于**设立明确的边界和观察期**：**约两次不出则止**、**三次无进展则弃**、**四次无付出则断**。它不是一个苛刻的公式，而是一个帮助你保持清醒、尊重自我价值、引导你寻找平等互惠关系的实用工具。请记住，健康的感情建立在相互吸引和共同投入的基础上。
