## 28天法则：关系推进的关键期 ⏳

上一段我们提到了课程的核心主题，本节中我们来看看判断是否应该放弃的第一个心理学依据：**28天法则**。

从心理学角度来说，放弃追求的一个普遍时间点是**28天**。这意味着，当你和一个女生聊天很火热，但在28天的期限内，双方没有发生任何推进关系的行为，那么这段感情几乎不会有结果。

## 什么是“推进关系的行为”？ 🔍

理解了时间框架后，我们自然要问：如何判断关系是否在推进？本节将定义关键的行为指标。

推进关系的行为是指那些能明确拉近双方距离的互动。例如：
*   线下邀约女生不拒绝。
*   出现暧昧的昵称。
*   聊天互动频繁。
*   不排斥与你进行肢体接触。

即使是慢热的女生，在28天内可能不会发生亲密行为，但你应能感觉到她的主动，例如她会主动关心你、主动找你聊天。

## 需要放弃的明确信号 🚩

如果关系没有如上述般推进，反而出现以下消极信号，则代表女生对你完全没有兴趣，你应该考虑放弃。以下是三种核心的负面反应：

**1. 回复敷衍，经常消失**
她的回复总是字数很少，如“嗯”、“哦”、“好的”，这本质上是礼貌性的敷衍。如果聊天经常突然中断、她人不见了，且这种情况是常态而非偶然，你就应该果断放弃。

**2. 始终无法邀约见面**
当线上聊天很火热，你想约她吃饭以进一步了解时，如果她拒绝你一次，可能出于害羞。但如果她一而再、再而三地拒绝，始终约不出来，你就不要再浪费时间了。

**3. 主动进行物质索取**
当你们的关系还没有实质进展时，她就开始有意无意地暗示你给她买东西、带她去高档餐厅或向你借钱。这种情况必须放弃。

## 为何必须放弃“索取型”关系？ 💸

上一节我们列出了包括物质索取在内的消极信号，本节中我们深入分析第三种情况的底层逻辑。

原因很简单：对方已经把你标记为**供养者**。她与你接触的目的就是为了获取金钱或物质。公式可以概括为：
**未确定关系 + 主动索取物质 = 将你视为工具人**

一个女人如果还没和你确定关系就开始索要物质，说明她根本不喜欢你。她对你的定位是一个**移动的ATM**，其原则是在你这里“能捞多少捞多少”。不要幻想通过满足其物质需求来感动她、让她喜欢上你，她利用的正是这种心态。对此，正确的做法是**及时止损**，拉黑删除，远离渣女。
