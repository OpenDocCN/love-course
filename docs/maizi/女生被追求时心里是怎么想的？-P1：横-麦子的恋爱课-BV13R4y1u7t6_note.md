## 核心心理活动分析

上一节我们概述了女生心理的复杂性，本节中我们来看看具体的心理活动阶段。

**`考察期类比：买衣服需要考虑尺码、面料、价位等，选择人生伴侣自然需要更审慎的考察。`**
