# 追女生越在乎，越得不到-P1-麦子的恋爱课-BV1bd4y1U7MU_note

在本节课中，我们将要学习一个在恋爱关系中常见的现象：过度在乎反而可能导致关系失衡，甚至无法获得对方的青睐。我们将剖析其背后的心理机制，并提供建立健康对等关系的核心心法。

## 过度在乎的陷阱

上一节我们介绍了本课的主题，本节中我们来看看具体表现。你越在乎一个女生，你就越得不到她。这并非鼓励去做不负责任的人，但你需要明白一个核心道理：**在乎的程度需要对等**。

我见过许多人因为这个原因而单身。一旦你喜欢上一个人，你的世界里就仿佛只剩下她。她发布一条朋友圈动态，你都会反复琢磨半天其含义。

## “自我感动”与关系失衡

理解了现象，我们来探讨其背后的行为模式。许多人不仅自己深陷其中，还会想方设法地让对方知道你有多在乎她。这种行为本质上是**自我感动**。

关键在于，健康的在乎是相互的、对等的。如果对方回复消息很慢，你也绝不应该总是秒回。如果对方说正在忙碌，你也绝不应该去频繁打扰。一旦你的付出和关注明显越过了对方给出的界限，她就会产生一种已经“吃定”你的感觉。

以下是这种心态可能引发的后果：
*   她会开始在各种事情上拿捏你。
*   她可能会吊着你，不给予明确的关系进展。
*   她可能逐渐变得不尊重你的时间和感受。

## 人性与恋爱心法

上一节我们分析了失衡关系的结果，本节中我们来看看其人性根源并给出解决方案。不要试图去挑战人性。每个人在轻易得到某样东西（包括过度的好感和关注）之后，往往不会首先想到珍惜，而是会下意识地去看看**是否存在更好的选择**。

最后，分享一个建立优势关系的核心心法，其核心可概括为：**`先得到，再深爱`**。在恋爱的初始和吸引阶段，理性判断和彼此价值的匹配远比单方面的深情投入更重要。

如果顺序颠倒，变成了**`先深爱，再争取`**，那么你很可能就已经失去了关系的主导权，变得难以“得到”你期望的关系。

---

本节课中我们一起学习了：过度且不对等的在乎会导致你在关系中失去吸引力与主动权；健康的关系建立在相互尊重与价值对等的基础上；在恋爱初期，应遵循“先建立吸引，再投入深情”的心法，避免陷入自我感动的陷阱。
