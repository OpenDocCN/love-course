## 心理动因：负罪感的转移与消解

上一节我们提到了“备胎”被贬低称呼的现象，本节中我们来看看驱动这种行为的核心心理动因——**负罪感的消解**。

当一个人在做自认为不道德的事情时，内心会产生负罪感。为了能够继续这种行为并维持心理平衡，个体需要一种机制来减轻这种不适感。

**核心公式：`实施不当行为 -> 产生负罪感 -> 丑化对方 -> 降低负罪感 -> 行为持续`**

一个经典的类比是电信诈骗犯。假设他骗走了老人的养老钱，如果他持续思考受害者的悲惨境遇，强烈的负罪感会让他无法继续行骗。因此，他的心理防御机制会启动：

以下是诈骗犯可能用于丑化受害者、为自己开脱的典型思路：
*   受害者是因为“贪小便宜”才上当的。
*   受害者“智商不足”，即使不被我骗，也会被别人骗。
*   我的行为是在给受害者“一个教训”。

这套逻辑的核心在于，**通过贬低和丑化对方，将过错归咎于受害者自身，从而将自己的行为合理化，减轻内心的道德谴责。**

## 恋爱关系中的投射：从“付出”到“丑化”

理解了普遍的心理防御机制后，我们将其投射到具体的恋爱关系中。当一方持续付出而另一方只接受不回报时，类似的心理过程便会发生。

此时，她非但不会感激你的付出，反而可能产生相反的认知。因为她越是意识到你真诚的付出与爱意，她的**愧疚感**就越强烈，这会使她直面自己“利用他人”的不道德行为，这是她内心无法接受的。

因此，为了消除这种愧疚感，维持心理平衡，她唯一的方式就是启动心理防御——**持续地丑化你**。这与诈骗犯丑化受害者的逻辑如出一辙。
