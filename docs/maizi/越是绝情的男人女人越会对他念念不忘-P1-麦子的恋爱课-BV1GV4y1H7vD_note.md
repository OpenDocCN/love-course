在本节课中，我们将要学习一种看似矛盾的情感策略：为何适度的“绝情”反而能加深你在对方心中的印象，以及如何通过三个核心原则来实践它，从而建立健康、平等的关系。

上一节我们介绍了“绝情”策略的核心理念，本节中我们来看看具体需要掌握的三个关键行为准则。

## 一、觉察不爱，果断离开 🚪

第一个原则是：一旦觉察到不被爱，便应干脆利落地离开，不拖泥带水。男人的“狠”要狠到骨子里。这背后是人性的规律：**你越是表现出一副苦苦挽留、卑微的样子，对方越是会轻视你**。

果断离开的公式可以概括为：
**觉察到不被爱 -> 停止投入 -> 立即撤离**

## 二、设立边界，勇敢拒绝 🛑

第二个原则是：在该拒绝的时候坚决拒绝，不要委曲求全。当你掏心掏肺付出的同时，也必须审视对方是一个什么样的人。

以下是需要警惕的情况：
*   有些人，你千方百计地为他付出，他却可能绞尽脑汁想着如何利用你。
*   你对他越好，他越会得寸进尺，仗着你对他的感情而为所欲为。

因此，核心原则是：**善良和付出应该留给值得的人**。

## 三、先爱自己，再爱他人 ❤️

第三个原则是：懂得先爱自己，再去爱别人。观察现实不难发现，那些对自己有要求、有原则的人，往往更受人喜爱。

你对待自己的方式，就是在教别人如何对待你。所以：
**你越是爱自己，别人才会越爱你、越尊重你，不敢轻视你。**

本节课中我们一起学习了“绝情”策略的三个核心原则：果断离开不爱的关系、勇敢拒绝不合理的要求、以及将爱自己置于首位。总之，无论如何，请务必记住：**永远不要放下自我，去爱一个让自己卑微的人。**
