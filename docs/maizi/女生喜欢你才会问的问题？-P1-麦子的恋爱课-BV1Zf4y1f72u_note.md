**正确回应思路：**

**推荐回应示例：**

这样回应，女生会觉得你真正懂她，彼此默契，从而大大增加好感。

**错误回应示例：**

**正确回应思路：**

**推荐回应示例：**
“感情这种事，我觉得是遇到相互喜欢的人，才会有谈恋爱的想法。”

记住，谈论感情时一定要表达出自己的标准。只有低价值的男人才会没有标准，来者不拒。
