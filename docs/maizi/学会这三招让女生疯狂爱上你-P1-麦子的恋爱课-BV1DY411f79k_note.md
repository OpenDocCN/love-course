## 第一招：索取大法

上一节我们介绍了课程概述，本节中我们来看看第一招“索取大法”。其核心心理学原理是：**人们对自己帮助过的人会更容易产生好感**。因此，在互动中，你需要巧妙地创造机会让女生为你提供帮助。

*   **创造求助场景**：你也可以说感觉自己皮肤最近有些干燥，请她推荐一套护肤品。
*   **给予正向反馈**：在她提供帮助后，一定要真诚地夸奖她，例如：“你眼光真好，选得特别合适。”
*   **制造答谢机会**：随后，可以借此机会感谢她，例如邀请她吃饭。这一系列互动能自然地拉近双方距离，为关系升级创造条件。

## 第二招：接触大法

掌握了通过求助建立好感的技巧后，我们进入第二招“接触大法”。恰当的身体接触能有效激发女生的情绪波动，制造心跳加速的感觉。

## 第三招：推拉大法

在学习了如何通过身体接触升温关系后，我们来看看第三招“推拉大法”。这是在聊天中制造情绪张力、让女生产生感觉的高效方式。其核心是**先“推”（制造一点否定或距离感），再“拉”（表达肯定或亲近感）**，以此带动对方的情绪起伏。

以下是一个经典对话示例：

*   **正确推拉**：你可以回答：“**不想**。想了也见不到，想了心里还难受，有啥用？”
    *   **“不想”** 是“推”，制造一点小失落或挑战。
    *   **“想了也见不到，想了心里还难受”** 是“拉”，间接而强烈地表达了想念，并赋予了情感深度。
    *   通过这一推一拉，女生的情绪会随之波动，更容易对你产生深刻印象和情感依赖。
