在本节课中，我们将学习如何识别女性在亲密关系中的“假性爱”行为。所谓“假性爱”，指的是一种以自我为中心、缺乏真正关爱与尊重的相处模式。了解这些表现，有助于我们更清晰地认识关系状态，保护自己的情感健康。

上一节我们介绍了“假性爱”的基本概念，本节中我们来看看具体的八大表现。

以下是识别“假性爱”的八个关键行为特征：

2.  **不解释，你爱怎么想怎么想。** 她拒绝沟通，对你的感受和疑问采取冷漠态度，不愿为消除误解做出努力。
3.  **不认错，觉得犯错的都是你逼的。** 即使明显有错，她也不会承认，反而会将错误归因于你的行为或态度迫使她如此。
4.  **吵架后冷暴力，每次吵架都是你低头。** 发生冲突后，她习惯使用冷暴力，关系的修复永远需要你主动妥协和道歉。
5.  **爱翻旧账，你对她的好一点没记住。** 她倾向于牢记你的过失并反复提及，却选择性遗忘你所有的付出与好处。

7.  **从来不关心你，只爱她自己。** 她的关注点始终围绕自身需求与感受，缺乏对你生活、情感和健康的实质性关心。
8.  **从心里看不起你。** 她内心认为与你结合是“下嫁”，感到委屈，认为你理应对她进行单向度的、全然的付出。

本节课中我们一起学习了“假性爱”的八大典型表现。识别这些行为是建立健康、平等恋爱关系的第一步。希望这些知识能帮助你更好地审视关系，做出明智的选择。
