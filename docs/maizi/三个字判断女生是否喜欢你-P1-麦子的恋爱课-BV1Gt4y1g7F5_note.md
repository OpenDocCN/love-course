# 麦子的恋爱课 P1：三个字判断女生是否喜欢你 💬

在本节课中，我们将学习一个简单的方法，通过发送三个字来初步判断一位女生对你的态度。这个方法基于对方回复内容所反映出的心理状态。

## 概述

核心方法是：通过微信向心仪的女生发送“你胖了”这三个字，然后根据她的回复内容和语气，分析她对你的好感程度。不同的回应方式，揭示了她在意你评价的不同层次。

## 判断方法与分析

以下是四种典型的回复情况及其对应的心理分析。

**第一种情况：负面回应**

如果她回复“滚”或“关你屁事”，这表明她不仅不喜欢你，甚至可能对你有些反感。此时，你应该停止进一步的追求。

**第二种情况：中性确认**

如果她回复“哦，是吗？哪里胖了？”，这表示她在意你对她的看法，对你有一定的好感，但这种好感目前并不强烈。

**第三种情况：解释原因**

如果她向你详细解释，例如“最近总熬夜，还天天吃夜宵，所以才胖了”，这说明她非常重视你的意见，不希望在你心中留下负面印象。她很可能对你有好感。

**第四种情况：积极改变**

如果她回复“哦，那好，明天我就开始减肥，锻炼身体”，这表明她愿意为你做出积极的改变，向你展示她的可塑性。这种情况，**十有八九她就是很喜欢你**。
