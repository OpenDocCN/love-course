而是说能给你加分的话，让女生对你有好感的话。那么，怎么能正确聊疫情这个话题呢？有三个方向。

上一节我们指出了聊天中抱怨的弊端，本节中我们来看看三个正确的聊天方向。

1.  **展示你的高价值**。
2.  **展示你的正能量**。
3.  **多聊让女生开心的话题**。
