## 禁忌一：一味讨好

很多男生认为女生都喜欢被宠爱，于是采取不断送礼物的策略。随着赠送的礼物越来越多，你内心的期待也会随之升高，总认为送到某个程度后，对方就会答应与你在一起。

**公式：** 一味讨好 ≈ 不断付出 + 过高期待 → 关系失衡

时间久了，如果女生对你表现出几次冷淡，你的心态就容易崩溃。此时，女生已经在关系中占据了上风，而你的位置则变得被动，也就是常说的“舔狗”。

上一节我们介绍了单方面过度付出的问题，接下来我们看看第二个需要避免的行为。

## 禁忌二：缺乏主见

你切忌一味迎合女生，对她所有的观点都表示赞同。事实上，适度地展现自己的主见和想法，反而会让女生觉得你成熟可靠，是一个能提供安全感的伴侣。

以下是展现主见与盲目迎合的对比：

*   **缺乏主见：** “你说的都对。”“我都听你的。”
*   **有主见：** “我觉得这个方案不错，不过我这里有个想法，我们可以讨论一下。”
