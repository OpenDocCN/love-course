在本节课中，我们将探讨一个普遍的社会现象：为什么在现代婚姻中，经济负担（如买房、买车、彩礼、办酒席）通常更多地落在男性身上。我们将分析其背后的社会心理与结构性原因。

---

上一节我们提到了现象，本节中我们来看看一位粉丝的具体困扰。一位粉丝抱怨说，当男人太累了。结婚需要准备房、车、彩礼和酒席，整套流程可能要花费两百多万元。以他目前的工资计算，可能需要三十年才能攒够。他因此提问：为什么结婚总是男人出钱？

---

## 根源分析：不团结与“内卷”🔄

我认为，根源在于男性群体内部不够团结，存在“窝里斗”的现象。假设有一位男性能够团结全国所有男性，并与女性群体协商，提出诸如“买房买车双方各出一半”、“取消彩礼”等条件，并坚决表示若不同意就不结婚。只要坚持下去，这种谈判很可能成功。

这类似于历史上的工人大罢工运动。例如，争取八小时工作制。最初，工人每天需工作十几个小时，疲惫不堪。于是他们集体罢工，要求实行八小时工作制，否则宁愿饿死也不复工。

坚持斗争后，老板最终不得不同意八小时工作制。然而，这项制度传到我们这里，却演变成了“996”、“007”的工作模式，且加班常常没有报酬。原因何在？因为我们不够团结。“你不干，有的是人排队等着干”。

因此，老板便有恃无恐。同理，你认为男性群体能团结起来吗？我认为肯定做不到。

---

## 破坏团结的几种行为🧩

以下是几种导致无法团结、加剧“内卷”的常见行为：

1.  **自愿承担者**：总有一部分男性愿意主动承担买房、买车、支付彩礼的费用，并且他们的父母也支持，甚至愿意借钱帮助他们。
2.  **成功者的嘲讽**：一旦有男性通过提供物质条件找到了对象，他可能会反过来嘲笑那些单身的男性。
3.  **观望者的效仿**：另一部分持观望态度的男性，看到买房买车似乎“有效”，便认为这个方法靠谱，于是也开始效仿，加入购房购车的行列。
4.  **恐惧落后者**：还有一部分人，害怕自己“输在起跑线上”，因此也选择跟随大流。

## 女性的选择与“死循环”⚙️

作为一个理性的女性，只要不傻，通常会在有房有车的男性和没有的之间，选择前者。女性的这种选择倾向，会反过来促使更多男性认为必须买房买车才能获得婚恋优势，从而进一步加剧了竞争。

**核心逻辑公式**：
```
男性内部不团结 → 部分人提供物质条件 → 女性倾向于选择有物质基础的男性 → 更多男性被迫加入物质竞赛 → 内卷加剧 → 无法形成统一谈判力量
```

这便形成了一个难以打破的“死循环”。几乎是无解的。
