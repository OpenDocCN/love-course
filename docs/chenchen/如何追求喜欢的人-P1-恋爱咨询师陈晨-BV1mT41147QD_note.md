# 如何追求喜欢的人 - P1 - 恋爱咨询师陈晨 - BV1mT41147QD

## 概述
在本节课中，我们将学习如何克服追求喜欢的人时的心理障碍，提升沟通情商，建立双向奔赴的感情。

## 核心概念
**心态改变**：改变心态是追求喜欢的人的关键。

### 不受欢迎的男生类型
- 油腻
- 猥琐
- 直男
- **怂**

### 案例分析
一位粉丝向恋爱咨询师陈晨求助，描述了自己在追求喜欢的人时遇到的困境。

### 案例细节
- 男生通过朋友介绍认识女生，初次见面时因自身条件不足而感到心虚。
- 男生因故未能赴约，之后多次道歉，但女生态度冷淡。
- 男生在聊天中表现出逃避和自卑，导致女生对其印象不佳。

### 解决方案
1. **逃避解决不了问题**：面对感情问题，只有尝试才能知道结果。
2. **假象的怂**：男生内心深处并不想随便凑合，但害怕暴露自己的缺点。
3. **自信面对问题**：不要规避自己的短板，用幽默和自信的态度展现自己。

### 案例改进
- 男生在聊天中主动提及身高问题，并用幽默的方式化解尴尬。
- 男生用赞同观点的方式拉近与女生的距离。
- 男生用假设关系的方式拉近彼此的距离。

### 总结
本节课中我们一起学习了如何克服追求喜欢的人时的心理障碍，提升沟通情商。记住，怂永远解决不了问题，我们要敢于直面自己的缺点，敢于去尝试。希望屏幕前的你，永远不要说“我还没有做好准备”，因为真正的准备是勇敢地去追求。
