## 概述
在本节课中，我们将探讨为什么人们在恋爱中会担心对方离开自己，以及这种担忧背后的心理原因。

## 核心概念
**公式**： 担心对方离开 = 自我价值感低 + 对方期望高

### 为什么总会担心对方离开
1. **害怕真实的自己被拒绝**：你害怕的是那个真正的你自己，暴露在他面前时，他一定会离开你，因为你知道自己配不上他，因为你知道你自己不够好，因为你知道你们两个人在一起的时候。

2. **对对方的期望过高**：你有很多事情对他是有期望的，而且这些事情都是很重要、很致命的事情。你也非常清楚地知道，这些东西瞒得了一时，瞒不了一世，终归有一天你是要告诉他的。

3. **害怕真相揭露后的后果**：你非常清楚有一些事情，你一旦告诉他，他一定会离开你的。
