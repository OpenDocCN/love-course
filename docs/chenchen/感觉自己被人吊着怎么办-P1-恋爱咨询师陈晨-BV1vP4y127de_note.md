# 恋爱关系课 01：如何摆脱患得患失的心态？💔

在本节课中，我们将要学习一个核心概念：为什么在感情中你会感到被“吊着”而患得患失，以及如何通过调整心态和行动来重新获得关系的主动权。

## 问题根源：唯一的“选择”等于没有选择

上一节我们介绍了课程主题，本节中我们来看看问题的核心原因。

你之所以在面对她时，总是纠结、紧张、放不下，是因为在你的世界里，她是唯一的。你认为是你选择了她，其实并不是。**当你只有一个选择时，你就是没有选择**。

这种“唯一性”会让你陷入被动。

## 患得患失的负面影响

基于上述心态，会产生一系列连锁反应。

你会发现你自己变得患得患失。你永远无法在她面前，呈现出一个最好的自己。你总是会选择妥协。在不断的妥协中，你自己也容易失落。

因此，**只有处在一个良好的自我状态下，你才可以在自己的两性关系当中，更加如鱼得水**。

## 解决方案：创造“选择权”

那么，如何建立并维持这种良好的自我状态呢？关键在于创造选择权。

你要让自己多一些选择，哪怕只是你自己觉得多了那么一些选择。这样你就不会过于地在乎一个人了。

以下是具体可操作的方法列表：

*   **拓展社交圈**：多跟一些异性去聊天。例如，我大号以前推荐过一个软件叫“南南”。
*   **核心练习**：不断地去跟异性聊天，锻炼自己的心态。这其实是一件非常重要的事情。
*   **自我展示与试错**：你应该在多个异性面前，有选择地展示自己，去试错，去包装自己。

通过以上练习，你逐渐就会得到异性的青睐，从而在心态上建立起真正的自信和选择权。

---

本节课中我们一起学习了：患得患失的心态源于将对方视为“唯一选择”。要打破这种困境，核心是主动为自己创造更多的社交可能性和选择权，通过实践来提升自信与魅力，最终在关系中找回平衡与主动权。
