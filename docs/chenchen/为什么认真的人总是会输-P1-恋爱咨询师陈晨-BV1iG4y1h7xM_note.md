# 为什么认真的人总是会输 - P1 - 恋爱咨询师陈晨 - BV1iG4y1h7xM

## 概述
在本节课中，我们将探讨为什么在恋爱中过于认真的人往往会遭遇失败。我们将分析认真投入的态度及其可能带来的后果。

## 核心概念
**公式**： 真心 = 认真投入 + 真诚相待

## 认真投入的后果
以下是认真投入可能带来的后果：

### 1. 过度付出
人是这样的一种动物，一旦开始认真的去喜欢上一个人，他会快速的掏心掏肺，把自己的一切都套入了。

### 2. 失去神秘感
你一口气一股脑把所有的都给了，你就像赤裸着站在他面前一样，他对你失去兴趣了，你这个人没有神秘感了。

### 3. 丧失吸引力
那他怎么可能会喜欢你。

## 保留自己
### 1. 学会保留
我觉得到底要不要认真，认真没有错，但你要学会保留自己。

### 2. 爱与相爱的区别
我没有问我自己，你认真的啥，对方认真的吗，爱从来都没有用，相爱才有用。

## 总结
本节课中我们一起学习了为什么认真投入的人往往会输。在恋爱中，学会保留自己，平衡付出与接受，是维持关系的关键。
