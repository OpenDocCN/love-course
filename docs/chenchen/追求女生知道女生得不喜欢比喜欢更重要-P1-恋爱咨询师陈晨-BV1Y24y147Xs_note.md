## 从海底捞服务说起

上一节我们提到了核心观点，本节中我们来看看第一个例子。我们都去过海底捞，它的服务到底好不好？它曾经可能很好，但现在却未必。它曾经好的原因，是因为其他饭店没有提供它那样的服务。

然而，当所有饭店都开始模仿这种过度热情的服务时，问题就出现了。有的客人进入饭店后，其实并不希望被人频繁打扰。对于这部分客人来说，过度的服务反而会让他们感到反感。

因此，在琢磨对方喜欢什么的竞争环境中，如果你能敏锐地感知到对方“不喜欢”什么，并避免去做那些事，你就已经取得了优势。**好的服务是有边界的，而不是一趟一趟地送东西。**

## 社会关系中的智慧

理解了服务业的例子后，我们再来看看人际关系中的应用。你肯定见过很多人为了建立社会关系而去送礼。通常的做法是去问：“你们老大喜欢什么？他喜欢什么我就送什么。”

但真实的情况往往是：在双方还不熟悉的情况下，你送了礼，对方可能因为顾虑而不敢收。那么，聪明人会怎么做呢？

以下是聪明人的做法步骤：
1.  先去打听对方“不喜欢”什么。
2.  在与对方接触时，首先确保自己不会做出让对方反感的行为。
3.  目标是先成为一个让对方不讨厌的人。
4.  随着关系熟络、对方开始关心你之后，再以对方喜欢的方式送点东西。

这种策略放在谈恋爱里是一模一样的。追求初期，与其费尽心思猜测和迎合女生的喜好，不如先观察和避免她所反感的行为。这能让你更快地建立起舒适感和信任，为后续发展打下坚实基础。
