## 核心第一步：掌握“广撒网” 🌐

上一段我们提到了课程主题，本节中我们来详细看看“广撒网”这个核心技能。它本质上是一个需要练习的技术活。

**核心公式**：**广撒网 = 对每一位接触到的异性留下良好印象**

以下是“广撒网”的具体操作原则：

*   无论对方高矮胖瘦、外貌如何，你都需要在她们面前留下一个特别好的印象。
*   目标是让对方觉得你是一个有价值的人。

*   因此，应以一种“普遍建立联系”的心态，与你所能认识的所有异性进行交往和交流。
