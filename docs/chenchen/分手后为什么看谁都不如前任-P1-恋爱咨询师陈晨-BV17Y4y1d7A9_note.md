# 恋爱心理学课程 01：分手后为何看谁都不如前任？🤔

在本节课中，我们将要学习一个常见的分手后心理现象：为何会觉得新认识的任何人都比不上前任。我们将从心理依赖和自我认知的角度，剖析这种现象背后的原因。

## 核心概念解析

上一节我们介绍了本节课的主题，本节中我们来看看其背后的核心心理机制。

当一段重要的恋爱关系结束时，你可能会感觉失去的不仅仅是一个人，而是一部分自我。这是因为在亲密关系中，双方往往会形成深刻的情感与心理联结。

**核心心理公式可以概括为：**
**自我完整感 = 原有自我 + 亲密关系提供的价值认同与情感填充**

当关系终结时，公式后半部分骤然缺失，导致个体感到自我不完整，处于一种“空壳”状态。

## 主要原因分析

以下是导致“看谁都不如前任”这一感受的几个关键原因。

1.  **情感世界的掏空**：她曾经是你情感世界的核心，她的离开仿佛带走了你世界里的重要内容，使你感到内心空荡。
2.  **认知资源的占用**：你的思维被与她的美好回忆所占据，这消耗了大部分的情感与认知资源。
3.  **比较基准的扭曲**：你将记忆中理想化的前任，作为评价所有新人的唯一且不合理的标准。
4.  **自我能力的暂时丧失**：处于痛苦和失落中的你，暂时失去了客观发现和欣赏他人优点的心理能力。

## 如何理解当前状态

我们已经分析了原因，现在来进一步理解你当前的心理状态。

你此刻的感受并非真实的、完整的你的感受。因为你的整个世界曾被她填满，你的视野和判断力也因此变得狭窄。你的脑海里充满了她的优点，这种思维上的垄断，自然让你难以看到其他人身上的闪光点。

这就像你的心理“频道”被一个节目独占，无法接收其他信号。

## 课程总结

本节课中我们一起学习了分手后觉得新人不如前任的心理成因。关键在于认识到，这种感受源于突然的情感剥离导致的自我不完整感，以及记忆占主导的扭曲认知。明白这一点是走出阴影、重建自我并客观看待他人的第一步。
