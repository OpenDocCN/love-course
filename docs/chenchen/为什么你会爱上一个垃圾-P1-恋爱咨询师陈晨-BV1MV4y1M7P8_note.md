# 恋爱心理学课程 01：为什么你会爱上一个“垃圾”？ 🧠💔

在本节课中，我们将要学习一个常见的恋爱心理现象：为什么有时我们会为不值得的人付出，并在事后感到困惑。核心在于理解潜意识如何将“付出”与“爱”划上等号。

## 付出与爱的潜意识等式

上一节我们介绍了本课的主题，本节中我们来看看其核心心理机制。

你可能会遇到一种情况：你全心全意对一个人好，对方却离开了你。事后你可能会困惑，自己当初为何会看上那样一个人。

原因在于你的潜意识里存在一个等式：**付出 = 爱**。

## 关系中的付出策略

理解了核心等式后，我们来看看这个等式在现实关系中的体现。

在约会时，有些人非常擅长让对方花钱，无论是吃饭、看电影，还是索要各种礼物。

以下是这类行为背后的两个关键点：
*   他们并非爱占小便宜或经济窘迫。
*   他们的根本目的是想方设法让你持续付出。

## 付出如何转化为爱

那么，这种引导付出的策略为何有效呢？本节将揭示其心理原理。

对方深谙一个道理：**付出等于爱**。

这个等式在心理上有两种作用路径：
1.  一个人可以因为爱而去付出。
2.  当一个人付出足够多时，其潜意识会将这种**习惯性付出**本身认定为是爱。

---

本节课中我们一起学习了“付出=爱”这一潜意识等式。我们探讨了它如何让你为不值得的人持续投入，以及他人如何可能利用这一心理。理解这一点，是审视自身恋爱模式的第一步。
