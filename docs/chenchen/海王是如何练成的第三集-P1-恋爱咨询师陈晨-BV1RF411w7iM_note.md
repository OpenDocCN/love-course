在本节课中，我们将要学习海王在情感互动中一种独特的策略：如何通过低成本但高情感价值的行为来建立深刻连接，这与大众认知中的挥霍行为截然不同。

上一节我们探讨了海王行为的基本模式，本节中我们来看看他们如何具体实施一种被称为“雪中送炭”的策略。

大众通常认为，海王是通过带女孩去夜店并挥霍开酒来吸引对方。在海王眼中，这是一种特别愚蠢的行为。

海王的做法截然不同。当他锁定一个目标女孩后，如果女孩去夜店玩，他不会陪同参与。但是，他会在女孩散场走出夜店时，突然出现并接她。

同时，海王会递上一个保温杯，并附上一句话：“我不能陪你玩，但是我愿意照顾你。”

无论这个行为中的道具（如车）是租借还是购买的，关键在于其产生的心理影响。女孩在夜店内面对他人的挥霍时，清楚知道对方的意图。

然而，夜店外出现的这个男生，传达出的信号是希望与她认真发展关系。

什么是浪漫？什么是感动？在这一瞬间，这些感受会全部涌现。因为每一位海王都深刻理解一个核心原则：

**锦上添花永远比不上雪中送炭。**
