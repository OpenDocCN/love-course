# 海王是如何练成的 第一集 - P1 🎭

在本节课中，我们将要学习一种常见的社交现象：为什么有些人外貌并不出众，却总能吸引众多异性，并维持良好的关系。我们将通过拆解其具体步骤，来理解背后的原理与策略。

## 第一步：广泛建立联系 🌐

上一节我们介绍了课程主题，本节中我们来看看海王策略的第一步。他们会主动与所有自己感兴趣的人建立初步联系。

以下是他们常用的具体方法：
*   以请教或求助为借口开启对话，例如询问化妆品品牌或送礼建议。
*   在获得帮助后，迅速给予回馈（如发红包），以此塑造一个“知恩图报”的良好形象。

通过这种方式，他们成功地在对方心中树立了正向的初始人设。

## 第二步：成为无话不谈的朋友 💬

在建立了初步联系后，下一步的目标是深化关系，成为对方生活中重要的情感支持者。

以下是此阶段的关键行为：
*   在对方生活不如意时，及时出现并提供安慰，展现善解人意的一面。
*   无论是分担痛苦还是分享快乐，都表现得像一个完美的倾听者与陪伴者。

这使得对方在情感上对其产生好感和依赖，为后续发展打下基础。

## 第三步：制造“占有感”与依赖 ⏳

成为朋友后，核心策略是让对方在心理和时间上对其产生占有感。这里引入一个核心概念：**四小时定律**。

**公式：** 每日专注时间 ≈ 心理占有程度

该定律基于一个简单原理：人一天有24小时，其中8小时用于工作学习，8小时用于睡觉。当一个人每天投入在另一个人身上的时间达到**4小时**左右时，意味着其大部分空闲思绪都被占据，从而产生强烈的心理依赖和占有感。

## 第四步：制造落差并引导对方推进 🎣

当对方已经产生依赖后，策略进入关键阶段：主动后撤，制造心理落差，引导对方主动推进关系。

以下是此阶段的具体操作：
*   **暂时搁置**：突然变得忙碌，减少联系或不再主动搭理。
*   **制造失落感**：当对方主动联系时，以“我很忙”等理由回应。这种忽冷忽热的态度会激发对方的失落感与不安。
*   **引导升级关系**：为了弥补这种失落感，对方会更迫切地想升级彼此关系。此时，海王会持续制造这种推拉感。

这个阶段的核心是让对方开始思考：“他的心里到底有没有我？”一旦产生这个疑问，对方在心理上就已经陷入了被动。

## 总结与核心原理 🔑

本节课中我们一起学习了“海王”建立关系的四个递进步骤：**广泛撒网建立联系 → 深化情感成为知己 → 利用时间投入制造占有感 → 后撤制造落差引导对方主动**。

最终，一个人会对条件看似并不出众的对象如此上头和动心的根本原因，可以用一个原理概括：**温水煮青蛙**。整个过程循序渐进，通过情感渗透和时间积累，让对方在不知不觉中深陷其中，难以抽离。
