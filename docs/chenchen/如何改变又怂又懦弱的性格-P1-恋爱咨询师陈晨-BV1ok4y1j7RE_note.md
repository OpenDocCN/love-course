## 概述
在本节课中，我们将学习如何改变又怂又懦弱的性格，摆脱讨好型自卑人格，成为一个有态度的人。

## 核心概念
- **讨好型自卑人格**：过分在意他人想法，缺乏自信，容易讨好他人。
- **自尊**：对自己的价值和尊严的认识。

## 教程内容

### 1. 认识自己
**以下是一些关于认识自己的要点**：

- **公式**：自我认知 = 内在感受 + 外在表现
- **代码**：`self_awareness = inner_feelings + outer_behavior`

- **不会有人在乎你**：在日常社交中，真正在乎你想法的人很少。
- **例子**：在聚会中，即使你说错了话，过不了三天别人也会忘记。

### 2. 遇到恶心的人
**以下是一些关于遇到恶心的人的要点**：

- **公式**：成长 = 经历 + 反思
- **代码**：growth = experience + reflection

- **多碰到恶心的人**：通过遇到不懂感恩、过度索取的人，认识到自己的讨好行为一文不值。
- **例子**：遇到不断压榨你的人，让你明白自己的价值。

### 3. 不要太看重自尊
**以下是一些关于自尊的要点**：

- **公式**：自尊 = 自我价值 + 自我尊重
- **代码**：self-esteem = self_value + self_respect

- **不要把自尊太当回事**：过分看重自尊会导致自卑和过度小心翼翼。
- **例子**：一个男生因为女生没有回应他的搭话，就认为女生不喜欢他。

### 4. 确定自己的价值
**以下是一些关于确定自己价值的要点**：

- **公式**：价值 = 能力 + 付出
- **代码**：value = ability + contribution

- **不要害怕改变**：改变性格和习惯需要时间和努力，但这是改变的开始。
- **例子**：通过提升自己，找到适合自己的路，逐步改变现状。

## 总结
本节课中，我们一起学习了如何改变又怂又懦弱的性格，摆脱讨好型自卑人格，成为一个有态度的人。希望这些内容能帮助你走出困境，找到属于自己的价值。
