# 课程名称：如何改变又怂又懦弱的性格 - 第1课 🧠

在本节课中，我们将要学习如何识别并改变“又怂又懦弱”的讨好型自卑人格。我们将通过四个核心要点，帮助你理解这种心态的根源，并提供具体的改变思路，让你能够更自信地表达自我、拒绝他人，不再轻易受外界影响。

## 心态成因分析

上一节我们介绍了课程目标，本节中我们来看看这种心态是如何形成的。

造成这种心态的原因，大多数与从小的家庭环境有关。例如从小家教严格、经常受到打压、遇到问题容易心软。这些经历会让人慢慢地养成对事情缺乏信心、过分在意他人想法的讨好型自卑人格。

## 改变的四个核心要点

了解了成因后，我们来看看具体的改变方法。以下是四个需要领悟并实践的核心要点。

### 第一点：明白“不会有人在乎你”

首先，你需要明白一个看似直白但至关重要的道理：在日常社交中，真的不会有人时刻在乎你想什么、说什么或做什么。公式可以概括为：**他人关注度 ≈ 你对他人利益的直接影响度**。

只要你没有切身地影响到其他人的利益，你的表现好，别人至多评价你情商高；表现差，也只会被当作一个很快被遗忘的笑话。唯一持续在乎你是否出丑、给人留下什么印象的，只有你自己。

所以，即使你说错话，别人过不了三天就会忘掉。即使你当场感到非常尴尬，别人也只会根据你日后展现的价值来重新评价你。

例如，你在聚会中看到一个很好看的女生，一开始不敢要联系方式。但喝多后在朋友怂恿下尝试，结果成功了。你以为这是酒精的力量，实际上是你学会了放开自我。真正纠缠你的，永远是你自己。有句话说得好：**只要你不尴尬，尴尬的就是别人**。

### 第二点：主动经历以警醒自我

上一节我们讨论了调整自我认知，本节中我们来看看如何通过外部经历来强化改变。

为了让记忆更深刻，需要清楚地认识到：讨好型人格的人，应该多接触一些“恶心的人”。这能给你上一堂生动的教育课。

原因在于，只有当你多次遇到那些不懂感恩、不断压榨和过度索取的人之后，才会明白你的习惯性讨好真的一文不值，且容易被利用。这就像一个“屠龙少年最终认清恶龙套路”的故事。只有付出过沉重代价，才会真正产生改变的想法。当你彻底看清这些人的套路后，便不会再感到害怕。

需要明白，讨好型人格的问题不在于你身处哪个圈子，也不是靠互相理解就能解决。人与人之间的平等，需要相互付出来平衡。用公式表示即：**健康关系 = 相互付出 ≈ 价值平衡**。如果付出的价值无法对等，无论你怎么讨好或更换对象，都是徒劳。

### 第三点：别把自尊看得太重

接下来，我们探讨另一个关键障碍：过度膨胀的自尊心。

部分自卑的人，可能因长期受忽视，反而把自尊看得太重，太把自己当回事。这导致做事小心翼翼，不敢说话也不敢行动。

例如，我曾有个学员，只要他的行为没得到对方即时回应，他就直接判定对方不喜欢他。他在聚会上和一个女生搭话，当时女生正与别人交谈而忽视了他，他便认为对方对他毫无好感。这种想法非常极端。

理解这种心态，但必须认清：**自卑实质上是另一种形式的自负**。试图彻底改变性格和习惯意义不大，因为这只是在否定过去的自己。真正的改变始于认清并接纳自己。

举个例子，你可以观察那些你认为很自信的人，他们可能在某些方面比你更邋遢或想法更阴暗。只是他们认清了一个事实：**有人喜欢你就必定有人讨厌你**。想让所有人喜欢你的结果，只会让自己难受。因此，建立自信更应做到自爱，多照顾自己的感受，而非揣测他人。

### 第四点：认清责任归属并提升自我

最后，我们要建立一个根本性的认知：怂和懦弱并非你的天生错误。

怂和懦弱从来不是天生的，而是由成长环境、家庭及社会带来的多方面压力所造成的。想通这一点，你就能明白这不是你个人的原罪。

例如，工作中领导打压你，往往不是你的原因，而是他的资历和职场环境赋予他的底气。情感中，女生对你冷漠或打压，也常是因为她身边有众多追求者，这构成了她的底气。

所以，改变的关键在于尽可能去提升自己，找到最适合自己的道路，一步一步向上攀登。这样才能扭转你所处的局面。短暂的怂和懦弱并不可怕，可怕的是你长期待在这样的环境中任人宰割，还自以为是在忍辱负重。

## 总结与延伸

本节课中我们一起学习了改变讨好型自卑人格的四个核心要点：调整“他人关注度”的认知、通过经历警醒自我、放下过度自尊、以及认清责任归属并专注自我提升。

你可能会觉得这些内容与情感问题高度相关，但我认为，这恰恰是许多情感问题的根源——**心态**。无论你是在追求女生时总是卑微讨好，还是聊天总是很平淡，这些问题基本上都与自身心态息息相关。当然，这些原则同样适用于你的工作与日常生活。
