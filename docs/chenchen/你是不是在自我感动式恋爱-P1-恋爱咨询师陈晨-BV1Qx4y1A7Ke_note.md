# 你是不是在自我感动式恋爱-P1-恋爱咨询师陈晨-BV1Qx4y1A7Ke_note

在本节课中，我们将要学习一种常见的恋爱误区——“自我感动式恋爱”。我们将剖析其核心特征、心理动因，并帮助你识别自己或他人是否陷入了这种模式。

---

很多人会提出这样的疑问：“我对她那么好，她为什么不动心呢？” 这其实就是典型的“自我感动式恋爱”的表现。陷入这种模式的人，从未真正考虑对方的需求、喜好或性格特质。他们的关注点完全集中在 **“我自己”** 的付出和感受上。

他们认为，只要自己付出了，对方就 **“一定”** 要有回馈。这种逻辑 **`付出 = 回报`** 是不切实际的。他们会想尽办法讨好对方，并且自己的每一个讨好举动都能深深地感动自己。然而，问题在于，这些付出往往是对方 **不需要** 的东西。因此，他们通常无法获得期望中的恋爱结果。

上一节我们介绍了“自我感动式恋爱”的基本表现，本节中我们来看看它带来的另一个后果：埋怨对方。当付出没有得到预期回报时，他们会开始埋怨，心想：“我为你付出那么多，你却不在意我、不接受我。” 他们甚至会认为对方没有良心。这种怨恨情绪，恰恰源于其 **“自我感动”** 的核心本质。

以下是“自我感动式恋爱”的几个关键特征：

*   **单向付出**：行动出发点是自己认为“好”的东西，而非对方的实际需求。
*   **预设回报**：内心存在一个公式：**`我的付出 = 你应给的回报`**。当等式不成立时，会产生强烈挫败感。
*   **自我陶醉**：容易被自己的“深情”和“牺牲”所感动，沉浸在悲情叙事中。
*   **指责对方**：当结果不如意时，倾向于指责对方冷漠、不识好歹，而非反思付出是否有效。

---

本节课中我们一起学习了“自我感动式恋爱”的完整逻辑链条：从 **忽视对方真实需求** 的单一视角出发，进行 **无效的自我感动式付出**，最终因预期落空而 **产生埋怨**。理解这一模式，是建立健康恋爱关系的第一步。
