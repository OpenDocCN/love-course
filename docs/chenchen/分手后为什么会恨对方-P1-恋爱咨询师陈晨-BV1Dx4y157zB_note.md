## 为什么分手后会恨对方

**核心概念：**
- 爱的付出与回报
- 心理预期与现实的落差

### 原因分析

1. **爱的付出与回报**
   - 当你深爱一个人时，你会无条件地付出和投入。
   - **公式**：爱 = 付出 + 投入

2. **心理预期与现实的落差**
   - 当对方不再爱你，你的付出和投入变得毫无意义。
   - **代码**：当预期（爱）≠现实（不爱），心理落差产生。

### 案例分析

- 当你为对方付出很多，而对方却选择离开，你会感到痛苦和愤怒。
- 这种痛苦和愤怒源于你对爱的期望与现实之间的差距。

## 如何避免恨意

1. **调整心态**
   - 不要对爱情抱有过高的期望。
   - **公式**：心态 = 期望 / 现实

2. **学会放手**
   - 当对方离开时，学会接受并放手。
   - **代码**：放手 = 接受现实 + 学会放下
