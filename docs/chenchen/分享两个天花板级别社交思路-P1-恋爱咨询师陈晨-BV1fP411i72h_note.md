**以下是案例的详细分析：**

以下是两种高情商的聊天方式：

**公式**：情景创造 = 假设一个有趣的情景 + 调侃对方

**步骤**：
1. 男生先回复女生的问题。

2. 观察对方的回复，了解他的内心想法。
3. 根据对方的回复，继续深入话题。
