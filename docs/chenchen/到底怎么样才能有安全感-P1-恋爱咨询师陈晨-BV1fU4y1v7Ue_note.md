# **恋爱咨询师陈晨 - 如何获得安全感 - P1**

## 概述
在本节课中，我们将探讨如何获得安全感，特别是恋爱中的安全感。我们将分析安全感的本质，以及如何通过自我认知和关系经营来建立和维护安全感。

## 安全感的本质
**公式**：安全感 = 自我认知 + 关系经营

### 安全感是一种陷阱
很多女生追求的安全感可能是一种陷阱。回忆一下，在热恋期，你是否曾感受到满满的安全感？然而，在分手阶段，这种安全感可能会逐渐消失，甚至完全感受不到。

### 安全感的消失
分手后，你可能会质疑对方是否真的爱过你。这是因为你没有让这份安全感延续下来。人心都是善变的。

### 自我认知的重要性
安全感只有你自己才能给自己。同时，在给对方安全感的同时，也要让他有危机意识。这两者是相辅相成的。

### 明确底线
明确自己的底线，没有碰触底线时，给足他安全感。一旦他尝试接近你的底线，就要给他危机感，让他知道你敢碰触我的底线，我转身就走。

## 情绪与荷尔蒙
喜欢和爱是一种情绪，是荷尔蒙催发出来的冲动。这种情绪来得快，去得也快。只有懂得经营关系的人，才会将这种情绪持续地维持下来。

## 总结
本节课中，我们一起学习了如何获得安全感。通过自我认知和关系经营，我们可以建立和维护安全感。记住，安全感只有你自己才能给自己。同时，在给对方安全感的同时，也要让他有危机意识。
