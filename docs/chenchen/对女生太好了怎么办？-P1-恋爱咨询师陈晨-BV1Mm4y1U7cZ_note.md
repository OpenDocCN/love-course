- 但如果你一直这样关注别人，你就会忽略自己的感受。
- 长时间下来，你会发现你的好并没有换来同等的回报。

### 2. 朋友与你的看法
- 你的朋友可能会觉得你已经做得足够好了。
- 但她可能并不这么认为，甚至会觉得你有点烦人。

- 对一个人好，并不能保证她也对你好。
- 过度的付出只会让她认为是你欠她的，而不是她欠你的。
