# 恋爱咨询师陈晨 - 搭讪开场白技巧教程 - P1

## 概述
在本节课中，我们将学习如何运用不同的开场白技巧来提高搭讪的成功率。

## 功能性开场白
以下是功能性开场白，适用于社交场合，通过提问来开启对话。

**公式**：
- 提问：你知道附近的商场在哪吗？
- 目标：获取信息，降低社交压力。

## 观点性开场白
观点性开场白通常基于某件事情寻求对方的意见。

**公式**：
- 提问：你觉得这件衣服A好看还是B好看？
- 目标：获取对方观点，建立互动。

## 筛选性开场白
筛选性开场白针对特定目标群体，快速筛选合适对象。

**公式**：
- 描述：我感觉你是一个特别温柔的女生。
- 目标：明确目标，筛选合适对象。

## 意图性开场白
意图性开场白直接表达搭讪意图。

**公式**：
- 描述：我觉得你挺好看的，想认识一下。
- 目标：直接表达意图，避免浪费时间。

## 总结
本节课中，我们学习了不同类型的开场白技巧，包括功能性、观点性、筛选性和意图性开场白。通过掌握这些技巧，你可以提高搭讪成功率，更好地与他人交流。
