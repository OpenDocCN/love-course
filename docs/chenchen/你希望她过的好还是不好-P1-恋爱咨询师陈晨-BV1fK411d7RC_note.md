# 恋爱咨询师陈晨 - 你希望他过得好还是不好 - P1

## 概述
在本节课中，我们将探讨恋爱中的情感态度，特别是关于希望对方过得好或不好的心理。

## 核心概念
**公式**：真爱 = 善意对待

## 你希望他过得好还是不好
你希望他过得好还是不好，这取决于你是否真正爱他。如果你说爱他，但希望他过得不好，这可能说明你并不真正爱他。真正的爱意味着即使对方离开，无论是否伤害过你，你都会希望他过得好。

## 爱的定义
不要轻易说“我爱他”，而是要反思你是否真的爱对方。回顾你的前任，你会发现并不是对每个人都抱有恶意，总有一个人在你的内心，你愿意善意对待。这个人，才是你的真爱。

## 爱的深度
爱一个人很难，不要轻易对人说“爱”。事实上，离开你的人过得好与不好，与你无关。你的情绪不应受他的影响。

## 总结
本节课中我们一起学习了关于爱的深度和真正的爱意味着什么。希望你能从中获得启示，更好地理解自己和他人。
