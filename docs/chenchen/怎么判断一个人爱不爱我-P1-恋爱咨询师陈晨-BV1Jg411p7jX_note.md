## 核心概念
**公式**：爱 = 感觉 + 行为

### 判断爱的迹象
以下是判断一个人是否爱你的迹象：

1. **清晰的感觉**
   - 当你确定他爱你时，你比任何人都清楚。
   - 爱是可以感觉到的，通过他的眼神、话语和动作。

2. **不需要询问**
   - 当你不断思考并询问这个人是否爱你时，他也会这样问自己吗？
   - 你不需要问这种幼稚的问题，当你遇到真爱时，你会知道的。
