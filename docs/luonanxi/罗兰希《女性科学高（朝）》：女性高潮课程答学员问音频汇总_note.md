## 避孕方式详解

上一节我们介绍了课程概述，本节中我们来看看具体的避孕方法。以下是常见的女性避孕方式及其说明。

1.  **紧急避孕药**
    *   需在性生活后72至120小时内服用，且越早效果越好。
    *   超过72小时服用，避孕失败率显著增加，且对身体伤害较大。

2.  **短效口服避孕药**
    *   需按完整周期服用，通常以28天为一个周期。
    *   常见品牌如“妈富隆”或“优思明”，每盒21片。
    *   服用方法：每日一片，连续服用21天后，停药7天。
    *   注意事项：需阅读说明书，心脏病等疾病患者可能不适用。

3.  **避孕针**
    *   首次注射需在月经开始后的5天内进行，可立即生效。
    *   每年需注射约4次，即每3个月一次。
    *   注意事项：骨质疏松、神经性厌食、长期饮酒或吸烟者需谨慎使用。

4.  **避孕贴**
    *   功效持续一个月，需连续使用三片，每片使用7天。
    *   使用7天后需停用一周，以使月经正常来潮。
    *   通常贴于腹部或臀部等部位。
    *   注意事项：哺乳期绝对不可贴于胸部。

5.  **皮下埋植**
    *   将一根细小软管植入上臂、臀部或大腿皮下。
    *   并非口服药物，通过缓释作用达到避孕效果。
    *   提示：国内部分公立医院可能未普及此项目，建议咨询当地公立医院。

6.  **其他避孕方式**
    *   还包括阴道避孕环、子宫隔膜、宫颈帽、避孕海绵、避孕泡沫、避孕药膏/凝胶、阴道栓剂及宫内节育器等。

**选择建议**：女性避孕方式多样，应选择双方都喜欢且适合的方式。对于长期、稳定的伴侣，在双方完成健康检查后，可考虑非安全套的避孕措施。若为新结识的伴侣，仍推荐使用安全套，因其兼具避孕和预防性传播疾病的双重作用。

如果采取了避孕措施后，仍无法在性爱中投入，可能需要配合放松练习或寻求性心理健康辅导。

## 性焦虑与疼痛应对

上一部分我们探讨了避孕方法，接下来我们关注性生活中的心理与身体感受问题。以下是针对特定焦虑和疼痛的解答。

**问题一：担心不洁与疾病感染**
*   初期性生活后出现妇科炎症（如宫颈炎）是常见现象，如同温室花朵初次接触外界环境，身体需要适应过程。

**问题二：插入前焦虑与疼痛，插入后无感**
*   核心问题在于“操作性焦虑”：因过往不良经历（如疼痛）或对疾病的担忧，导致在性行为前产生预期性焦虑，无法放松，从而难以体验愉悦。
*   解决方案：

    2.  与伴侣进行“非插入式”的性游戏：专注于全身爱抚与交流，帮助身心放松。
    3.  在充分放松后，再尝试逐步、轻柔地插入，以消除对疼痛的恐惧。
    4.  此过程需要双方充分的情感沟通。女性需主动引导伴侣帮助自己放松。
