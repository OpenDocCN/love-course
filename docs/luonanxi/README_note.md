# 罗南希的Python入门教程 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，这个盒子用来存放数据。在Python中，我们通过一个名字（标签）来访问盒子里的内容。

创建一个变量非常简单，只需要使用等号 `=` 进行赋值即可。

以下是创建变量的基本语法：
```python
变量名 = 值
```

例如：
```python
name = “罗南希”
age = 25
```

这里，`name` 和 `age` 就是变量名，它们分别存储了字符串“罗南希”和数字25。

## 核心数据类型 🔢

我们已经知道如何把数据放进“变量”这个盒子里，接下来我们需要了解盒子里可以放哪些种类的东西。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型：

1.  **整数**
    *   描述：不带小数点的数字，可以是正数或负数。
    *   代码示例：`score = 100`， `temperature = -5`

2.  **浮点数**
    *   描述：带小数点的数字。
    *   代码示例：`price = 19.99`， `pi = 3.14159`

3.  **字符串**
    *   描述：由文本字符组成的数据，需要用单引号 `‘ ’` 或双引号 `“ ”` 包围起来。
    *   代码示例：`greeting = “你好，世界！”`， `author = ‘罗南希’`

4.  **布尔值**
    *   描述：只有两种值，`True`（真）或 `False`（假），常用于逻辑判断。
    *   代码示例：`is_learning = True`， `is_completed = False`

## 查看与理解类型 🧐

有时我们不确定一个变量里存放的是什么类型的数据，这时可以使用 `type()` 函数来查看。

它的使用方法是：
```python
type(变量名)
```

例如，执行 `print(type(age))` 会显示 `<class ‘int’>`，告诉我们 `age` 是整数类型。

理解数据类型很重要，因为不同类型的数据能进行的操作不同。例如，数字可以进行加减乘除，而字符串通常可以进行拼接和查找。

## 一个简单的综合示例 💻

让我们把上面学到的知识组合起来，写一个简单的程序。

```python
# 定义几个变量
student_name = “张三”
math_score = 95
science_score = 88.5
is_passed = True

# 打印这些变量
print(“学生姓名：”, student_name)
print(“数学分数：”, math_score)
print(“科学分数：”, science_score)
print(“是否通过：”, is_passed)

# 查看变量的类型
print(“name的类型是：”, type(student_name))
print(“math_score的类型是：”, type(math_score))
```

运行这段代码，你会在屏幕上看到变量的值和它们的类型信息。

## 总结 📝

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。

我们了解到，**变量**是存储数据的容器，通过赋值语句 `=` 来创建。而**数据类型**定义了数据的种类，我们主要接触了四种基本类型：用于整数的`int`，用于小数的`float`，用于文本的`str`，以及用于逻辑判断的`bool`。最后，我们还学会了使用 `type()` 函数来检查任何变量的数据类型。

记住，就像学习任何新语言要先认识字母和单词一样，熟练掌握变量和数据类型是你能用Python流利“说话”的关键第一步。
