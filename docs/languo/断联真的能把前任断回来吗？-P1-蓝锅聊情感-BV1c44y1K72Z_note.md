## 概述

断联是一种有策略的情感冷却期，其目的并非单纯地不联系对方，而是通过一系列心理博弈，为关系修复创造空间和可能性。真正的断联包含三个循序渐进的阶段：较量、刺激与逆转。

## 断联的三个核心阶段

上一节我们概述了断联的整体框架，本节中我们来详细看看这三个具体阶段是如何运作的。

### 第一阶段：较量

第一阶段的核心是心理博弈，比拼的是双方谁更能“放下”对方。此阶段的目标是打破原有的依赖模式，重建个人边界。

以下是本阶段的关键行动要点：
*   **核心行动**：彻底不主动联系对方。
*   **心理目标**：专注于自我回归与情绪稳定，而非关注对方动态。
*   **成功标志**：你不再因思念或焦虑而试图联系对方，心态趋于平静。

### 第二阶段：刺激

当你成功度过了第一阶段，对方可能会开始试探。第二阶段“刺激”便随之而来，其本质是对方对你是否真正放下的测试。

以下是本阶段常见的试探方式及应对策略：
*   **试探行为**：对方可能主动联系你，或在社交平台（如朋友圈）发布暗示开始新生活的动态（注意：这通常是一种测试，并非代表真有新欢）。
*   **正确应对**：你必须保持比第一阶段更加冷静和从容的态度。公式可以概括为：**你的冷静 = 对方的心理落差**。你的无动于衷会刺激对方产生失落感。

### 第三阶段：逆转

这是关系动力发生根本性转变的阶段。当对方因你的冷静而产生强烈的心理落差后，其行为可能升级。

以下是本阶段的特点与最终目标：
*   **可能情况**：对方受到的刺激可能使其做出更激烈的反应，例如更频繁地联系，或表现出情绪波动。
*   **核心要求**：你必须扛住所有形式的“打击”，始终保持稳定。这就像 **`if (对方试探) { 保持冷静; }`** 一样，需要严格执行。
*   **最终结果**：当你稳如泰山，而对方情绪波动达到顶点时，关系的主导权便发生逆转。下一步，往往是对方开始认真考虑挽留你。
