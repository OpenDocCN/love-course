# 情感沟通课 01：如何巧妙回复“在干嘛” 💬

在本节课中，我们将学习当女生主动询问“在干嘛”时，如何超越直男思维进行回复。我们将分析常见错误，并掌握一种能调动情绪、促进关系发展的核心回复框架。

---

## 常见错误回复分析

许多男生在面对女生主动询问时，通常会陷入两种典型的回复模式。

以下是两种常见的错误回复方式及其问题：

1.  **油腻型回复**：例如回复“在想你啊”。这种回复过于直接和轻浮，容易让对方感到不适。
2.  **总结型回复**：例如直接回答“在散步”、“在吃饭”。这种回复像任务汇报，语气生硬，无法延续对话。

上一节我们分析了两种典型的错误回复，本节中我们来看看这些回复为何会导致聊天失败。

典型直男式的回复很难让对话进行下去。女生往往只会回复“那你继续吧”来结束聊天，这还算好的情况。更常见的是，对方会用表情或问号来回复你。

你以为女生问“在干嘛”是真的好奇你在做什么具体事情吗？女生通常不会那么无聊。她肯主动给你发消息，无论内容是什么，都暗示着她对你有较高的好感度。“在干嘛”只是一个借口，意味着她想找你聊天。

## 核心回复框架：状态+行为+感受

为什么你总是把握不住对方给的机会？如果聊得好，一两句话就能升级关系。你离拥有女朋友可能就差这一步。

有些兄弟学到“要具体回复”后，就生硬地套用。例如女生问“在干嘛”，你回复“在散步，你呢？”；女生说“我刚下班”，你回“噢今天下班挺早啊”。你自己觉得这样聊天能聊出感情吗？

你不能用和兄弟聊天的方式跟女孩子聊天。这种一问一答式的对话会让双方都陷入尴尬境地，缺乏情绪互动，最终结果只能是朋友。很多兄弟感到困惑：为什么平常和女生聊天感觉挺好，但准备表白时，女生却说“我们还是做朋友吧”？

原因就在于你根本没有调动起对方的情绪，一直在进行没有营养的聊天，慢慢消耗了最初的好感。时间长了，女生会感到疲惫。如果这时出现一个会聊天的男生，你就直接出局了。

现在，让我们来看看正确的回复方法。别着急，我告诉大家，正确的回复应该是：**表达状态 + 表达行为 + 表达感受**。

其中最重要的就是表达你此时此刻的感受。以下是一个应用示例：

*   **女生输入**：`在干嘛呢？`
*   **男生回复**：`看到你在问我干嘛，我心情很好（表达状态）。正准备拿起手机找你聊天，就发现了你的消息，太默契了（表达行为与感受）。`

这样的回复之后，女生就会跟你越来越有的聊，对你越来越关注，从而喜欢上跟你聊天的这种状态，慢慢把好感升级为喜欢。这时你再表白，就是水到渠成的事情。

## 思维重于话术

我所讲的东西并不是固定的话术套路，而是一种思维方式。有些话术不要盲目去套用，具体问题需要具体分析。

掌握这种思维方式后，不管和谁聊天，都能让对方对你有好感，尤其是对喜欢的女生非常有效。赶紧去试试吧。

---

本节课中我们一起学习了如何高情商地回复“在干嘛”。关键在于理解对方问话背后的意图（开启聊天），并运用“**表达状态+行为+感受**”的框架进行回复，重点是传递积极情绪，而非简单汇报事实。记住，思维比固定话术更重要，灵活运用才能有效提升沟通质量，促进关系发展。
