## 第一招：推拉法则 🔄

上一节我们介绍了课程概述，本节中我们来看看第一个核心技巧——推拉法则。其核心在于通过“否定”（推）和“肯定”（拉）的交替使用，来调动对方的情绪，从而掌握互动的主动权。

简单来说，推拉法则的公式是：**先给予轻微否定或挑战，再给予肯定或奖励**。这种循环能有效激发兴趣。

以下是应用推拉法则的一个关键要点：
*   **关键在于情绪波动**：不要总是在琐碎的聊天中使用推拉，而应将其用于关键的情感交流时刻，以产生更大的情绪影响力。

让我们通过一个例子来具体说明。例如，当女生问你“你是不是喜欢我”时，错误的回应会立刻满足她的好奇心，让她失去探索欲。

正确的推拉应用如下：
1.  **推**：我可不会喜欢经常问男生这个问题的女生。（制造轻微紧张或否定）
2.  **拉**：但你给我的感觉很特别，我还是第一次对女生这么感兴趣。（随后给予肯定和独特关注）

这种方法完全掌握了女生的情绪节奏。
