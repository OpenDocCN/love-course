上一节我们介绍了课程的目标，本节中我们来看看具体的话术模式及其背后的逻辑。

以下是第一种常见的话术模式：

1.  **与其他追求者比较**：对方声称其他男性曾为其花费大量金钱（例如“花了多少个W”）却未成功。其目的是**自抬身价**，暗示你需要付出更多才能竞争。
    *   **核心逻辑**：制造稀缺性和竞争焦虑，让你感觉必须通过超额付出来证明自己。

接下来我们分析第二种利用前任进行对比的手段。

2.  **与前任的付出对比**：对方提及前任曾慷慨赠送贵重礼物（如车、表或巨额花费）。这利用了男性常有的**面子心理**和不愿“输给前任”的比较心态，引导你加大投资。
    *   **核心逻辑**：激发你的竞争意识和比较心理，使付出偏离真实情感需求。

第三种话术则通过营造外部参照来施加压力。

3.  **羡慕他人的伴侣**：对方在你面前刻意表现对其闺蜜获得丰厚馈赠的羡慕。这旨在让你产生**愧疚感**（觉得“给她的爱不够”）和**保护欲**（“不能让我心爱的女人委屈”），从而促使你进行贵重付出。

最后，我们探讨一种直接强调物质行动的话语。

4.  **强调“实际行动”而非承诺**：对方常挂嘴边的话是“不要画大饼，要看实际行动”或“我已过了耳听爱情的年纪”。真正意图在于将你的注意力导向**物质付出**。
    *   **核心区分**：**注重感情的女性，看重男人的人品与品格；看重利益的女性，才重点在乎你的付出。**
