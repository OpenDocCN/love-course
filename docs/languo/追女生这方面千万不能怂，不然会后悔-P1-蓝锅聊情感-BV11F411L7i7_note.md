# 追女生教程 第1课：告别“怂”与“猥琐” 😎

在本节课中，我们将要学习如何避免在追求女生时表现出“怂”或“猥琐”的行为。理解这些行为的本质并加以改正，是建立自信、提升吸引力的第一步。

## 什么是“猥琐”？🤔

上一节我们提到了要避免“猥琐”，那么究竟什么是“猥琐”呢？简单来说，“猥琐”可以用一个字概括：**怂**。在百科上的解释是举止扭捏、拘束、很不自然。

例如，你很想牵女生的手，但又不敢，害怕被拒绝或被骂耍流氓。于是你一会儿看她的眼睛，一会儿看她的手。当你鼓足勇气伸手时，伸到一半又开始退缩。或者，你不敢直接牵手，只敢找机会牵一下女生的袖口，或趁过马路时拖一下她的胳膊。

## 为何这些行为不可取？🚫

在我们女生看来，上述行为非常猥琐和令人反感。大家都听说过“公车色狼”吧？在公交车或地铁上，有些猥琐的男生会偷偷蹭着女生。你想一下，刚才描述的那些行为和“公车色狼”有什么区别？

想牵手，就应该大大方方地牵。你在怕什么呢？关键在于自信。

## 读懂女生心思的重要性 💡

很多小伙伴不知道，读懂女生的心思对于追求女生极其重要。我见过太多男生，因为读不懂女生的心理，也不知道该如何追求，于是一味地对女生好。直到女生表明不感兴趣，他们才意识到讨好式的追求没有用，甚至起反作用。

经过我的指导，他们大多数不是脱单了，就是结婚了。我还总结了一些聊天记录和案例分析，可以帮助快速提升个人形象和吸引力。想看的兄弟，可以给我的后台发一个“1”，或在评论区留言“1”。

## 具体问题与表现 📋

以下是几种常见的、体现“怂”或“猥琐”的具体行为模式：

*   **眼神闪躲**：有的男生，女生稍微穿得性感一点，他就不敢正眼看，但内心又想看。于是内心挣扎，在女生低头或转头时赶紧偷看两眼。等女生回过头，他又不敢看了。这种行为也很猥琐。
*   **聊天表情暴露不自信**：在微信聊天中，总发那些“捂脸笑”或类似的无底气表情。女生能通过这个表情看出你的不自信、说话没底气、害怕说错话惹她生气。于是你发这个表情以示自己没有攻击性——这**就是举止扭捏不自然**。想想你和朋友、哥们聊天时会总发这些表情吗？
*   **邀约时绕弯子**：约女生吃饭时，想想你是怎么约朋友的？你会直接说：“晚上有空吗？一起喝点。”简单直接，有什么说什么。但你是怎么约女生的？你可能会问：“你喜欢吃什么？”“你什么时候休息？”“我前两天吃了XXX，觉得挺好吃的。”这样绕来绕去不敢直接邀约，就是拘束不自然，显得你很猥琐。

## 核心：自信与直接 🚀

“猥琐”的根源在于**不自信**。男生谈恋爱一定要自信，要拿出点“霸道总裁”的感觉（并非真的霸道，而是指果决的气场）。你越简单粗暴，越像个爷们，女生反而越有感觉。

用公式来表达就是：
**有效吸引力 ≈ 自信程度 × 行为直接度**
