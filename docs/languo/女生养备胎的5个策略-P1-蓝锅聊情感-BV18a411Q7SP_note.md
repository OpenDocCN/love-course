在本节课中，我们将学习识别一种被称为“养备胎”的情感互动模式。我们将具体分析五种常见策略，帮助你理解其核心特征，并思考如何应对。

上一节我们介绍了课程的主题，本节中我们来看看具体的五种策略表现。

以下是五种常见的“备胎”策略表现：

1.  **不表態**
    只与你保持暧昧关系，不明确表达自己的真实情感或意图。

2.  **不確定關係**
    不与你确定恋爱关系，同时不拒绝你对她好，并可能欣然接受这些付出。

3.  **不主動**
    在互动中从不主动，总是由你发起联系，她不会主动找你。

4.  **不負責**
    认为你的所有付出都是心甘情愿的，因此她无需为此承担任何责任。其核心逻辑可以概括为：`你的付出 = 你自愿的选择`，因此 `她的责任 = 0`。

5.  **吊著你**
    当你准备放弃时，给你一些希望；当你重新靠近时，又给你泼冷水。这个过程会循环往复，其模式类似于：`给予希望 -> 让你失望 -> (循环)`。

本节课中我们一起学习了“养备胎”的五种具体行为模式，包括不表态、不确立关系、不主动、不负责和循环吊着对方。理解这些模式是建立健康情感关系的第一步。
