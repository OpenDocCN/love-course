# 情感课程 01：识别并远离消耗型关系 🚫

在本节课中，我们将学习如何识别一种不健康的情感关系模式——“消耗型关系”。我们将分析其特征，并明确为何应远离此类关系。

## 什么是消耗型关系？🤔

上一节我们介绍了课程主题，本节中我们来看看消耗型关系的具体定义。它指的是一段由一方主导，另一方不断妥协、丧失自我尊严的情感互动模式。

**核心公式**：**消耗型关系 ≈ 单方面付出 + 尊严感持续降低**

## 消耗型关系的主要特征 📋

识别消耗型关系，可以从以下几个关键特征入手。以下是其主要表现：

*   **关系主导权丧失**：你无法主导关系走向，相处中越来越感到没有尊严。
*   **单向妥协与忍让**：总是你在考虑对方的感受，顺从对方，并由你做出妥协和改变。
*   **维护责任错位**：大多数时候，由你在维护感情和付出。关系出现问题，也需要你主动去调解。
*   **对方权益优先**：对方优先考虑自身权益。如果自身权益未满足，则不会考虑你的感受。
*   **拒绝自我改变**：对方不会主动改变自身存在的问题。
*   **挑剔与不满**：对方只会挑剔你的毛病。即使你已经做得很好，对方仍处处表示不满。

## 健康与消耗型关系的对比 ⚖️

理解了消耗型关系的特征后，我们将其与健康关系进行对比，能更清晰地看到其危害。

一段健康的感情可以成就一个人，让双方共同成长。相反，一段糟糕的感情背后，往往存在一个持续消耗对方能量和自我的伴侣。

## 核心结论与行动建议 🎯

本节课中我们一起学习了消耗型关系的定义与特征。总结来说，如果你在一段关系中感到主导权丧失、尊严感下降，且总是单向付出、妥协，而对方不愿改变并持续挑剔，那么这很可能就是一段消耗型关系。

面对这种情况，最明智的选择是远离，而不是继续坚持。保护自己的情感能量和心理健康，是迈向健康关系的第一步。
