## 核心原则：关注行为而非言语

上一节我们介绍了本课程的目标，本节中我们来看看判断女生是否喜欢你的核心原则：**她的行为比她的言语更能反映真实想法**。如果她的行为表现出以下模式，你可能需要重新评估这段关系。

以下是四种关键的行为信号：

1.  **单向索取型陪伴**
    她只在需要你的时候要求你出现，但当你需要她时，她总是不在。她常用“我现在没空”、“我很忙”等理由敷衍你。这种不平衡的付出模式，说明她可能将你视为一个方便的“备胎”。

2.  **情绪垃圾桶型倾诉**
    她总是向你倾诉负面情绪，但倾诉完毕后对你的态度就变得冷淡。这表明她的情感重心并不在你身上，你可能只是她处理情绪的临时工具，她很可能有更喜欢的其他男生。

3.  **间歇性失联型互动**
    她会偶尔主动发消息，但也会突然消失好几天不回信息。女生可能因矜持或忙碌暂时不回复，但连续多天失联，意味着你并未被她放在心上。公式可以表示为：**`她在意你` ≠ `长时间无理由失联`**。

4.  **拒绝升级的暧昧**
    平时你们可以打情骂俏、肢体接触，感觉关系很近。但只要你正式表白，她会立刻划清界限，说出“我只当你是朋友”或“你很好，我配不上你”等话。这本质上是好感不足，你未能真正吸引她，所以她采取若即若离的态度。

## 总结与行动建议

本节课中，我们一起学习了识别女生是否真心喜欢你的四个关键行为信号：单向索取、情绪宣泄、间歇失联和拒绝关系升级。

如果你喜欢的女生表现出以上多种行为，你就需要清醒认识到，她可能并未对你投入真心。及时识别这些信号，有助于你将宝贵的感情、时间和精力投入到真正相互尊重、双向奔赴的关系中。
