# 课程 01：识别三种高风险伴侣类型 🚫

在本节课中，我们将学习识别三种在亲密关系中可能带来高风险的个人特质。了解这些特质有助于避免情感与财务上的重大损失，从而保护个人生活的稳定与事业的顺利发展。

## 第一种：挥霍无度的女性 💸

上一节我们介绍了课程概述，本节中我们来看看第一种需要警惕的类型：消费习惯远超自身收入能力的女性。

这种类型的核心问题在于其**消费支出 >> 收入**。她们可能月薪三千却消费过万，月薪过万却负债累累。其消费资金往往来源于父母、伴侣或借贷，而非自身劳动所得。即便做出节约承诺，行为上也难以持续。

以下是此类女性的常见行为模式：

*   消费水平与自身收入严重不匹配。
*   资金用途多集中于非必要享乐，如频繁聚餐、高消费娱乐。
*   习惯于向他人（尤其是伴侣）索取以维持高消费生活。

## 第二种：极端自我中心的女性 🦊

了解了财务风险，我们再来探讨性格特质带来的风险。第二种类型是极度以自我为中心、缺乏共情能力的女性。

其核心逻辑是：**所有问题 == 对方的错**。她们通常无法体谅伴侣的处境，将自己的感受和需求置于绝对优先地位。

以下是此类女性的典型表现：

*   缺乏换位思考能力，不关心伴侣的实际情况与困难。
*   习惯性指责，将关系中的所有不愉快归咎于对方。
*   记忆具有选择性，容易忘记对方的付出，包括重要日期（如伴侣生日）。

## 第三种：索取型“普信女” 🎭

最后，我们分析一种在关系中只知索取、不懂回馈的类型，俗称“普信女”。

其核心行为模式是**单向索取 + 低度尊重**。她们可以坦然接受对方的好，却不愿给予对等的反馈，甚至可能轻视对方的付出。

以下是此类女性的主要特征：

*   在节日、纪念日等场合，只期待收礼，没有回赠意识。
*   进行不合理比较，常用“别人都有”作为索取理由。
*   当需求未被满足时，可能会采取无底线的吵闹或冷战施压。
*   对伴侣缺乏基本的尊重与感恩。

---

本节课中我们一起学习了三种在亲密关系中需谨慎对待的女性特质：挥霍无度型、极端自我中心型和单向索取型。识别这些特质有助于建立更健康、平等、互惠的情感关系，避免个人生活与事业受到不必要的负面影响。
