# 为什么那些社会混混的女朋友接着

## 第一点：内心必须强大 💪

上一段我们提出了核心问题，本节中我们来看看第一个关键特质：强大的内心。

无论女生多么优秀或漂亮，他们都敢于尝试。而你即使面对条件不如自己的异性，也不敢主动上前搭讪。

请记住，无论你自认为外貌、经济条件或性格如何，关键是要敢于行动。核心在于脸皮要厚，要勇敢上前接触。万一对方恰好喜欢你这一类型呢？

又或者，万一对方“看走眼”了呢？如果你连尝试都不敢，却幻想拥有美好的感情，这无异于白日做梦。

**核心公式**：`行动 = 机会`。不行动，机会永远为零。

## 第二点：付出超乎想象的努力 🔍

内心强大是基础，但仅有勇气还不够。本节中我们来看看他们付出的实际努力。

他们的努力程度可能超乎你的想象。各类社交软件、聊天工具，甚至是漂流瓶，凡是你能想到的社交途径，他们都会充分利用。

你如果一年都认识不到一位新异性，如何与他们竞争？这个过程类似于拓展客户，必须主动出击才可能有结果。

以下是他们可能广泛使用的途径：

*   各类主流社交应用
*   兴趣社群与线下活动
*   通过朋友介绍扩展圈子
*   不放过任何可能的线下搭讪机会

## 第三点：不怕失去，勇敢试错 🚀

除了努力拓展渠道，在互动阶段的心态也至关重要。本节中我们来看看他们的第三个特质：不怕失去。

在可以承受失败的年纪，就应该勇敢试错。遇到心仪的对象，立刻主动认识。也许对方对你的印象也不错。

如果印象不错，就可以尝试邀约吃饭或看电影。如果你遇到喜欢的人连话都不敢说，又如何能推进关系呢？

在约会过程中，该牵手时就牵手，该拥抱时就拥抱，不要犹豫不决。如果你不主动进行肢体接触，就无法判断对方是否对你有好感。

你害怕失去，但问题是你目前也并未拥有。以上几点只要你能做到，对你有好感的女生通常都会给出积极回应。
