# 聊天技巧课程 01：间歇性联系法 💬

在本节课中，我们将要学习一种名为“间歇性联系”的聊天方法。这种方法的核心在于通过有策略地控制回复消息的时机，来维持对方的好奇心与期待感，从而在对话中建立吸引力和神秘感。

## 核心概念：何为间歇性联系？

上一节我们介绍了课程目标，本节中我们来看看核心概念“间歇性联系”的具体含义。它并非指彻底不与对方联系，而是指**不要总是秒回消息**，并且**回复的时间不固定**，但确保当天的消息在当天内回复完毕。

## 方法实施步骤

理解了概念后，以下是实施间歇性联系法的具体操作步骤。

1.  **避免秒回**：收到消息后，不要立刻回复。这会让对方觉得你可能有自己的事情在忙，生活充实。
2.  **不定时回复**：选择不同的、不规律的时间段进行回复。这种不确定性会引发对方的好奇。
3.  **当日事当日毕**：确保在当天结束前回复对方的留言。这是维持基本礼貌和联系感的关键。

## 至关重要的注意事项

在运用以上方法时，有一个必须警惕的禁忌，否则将前功尽弃。

*   **切忌在“失联期”发布朋友圈**：如果你在长时间未回复对方消息的期间，却在社交媒体上发布了新动态，对方会认为你故意忽视她，从而感到不被尊重，导致印象分大幅下降。

## 总结

本节课中我们一起学习了“间歇性联系”聊天法。其精髓在于通过**控制回复节奏（不秒回、不定时）**来创造神秘感和期待感，同时遵守**当日回复**的底线以维持基本联系。请务必记住，避免在未回复时更新社交动态，这是该方法生效的前提。
