# 追女生教程 01：学会扬长避短，有效展示自我 💬

在本节课中，我们将要学习一个核心的聊天原则：如何在与女生交流时，通过扬长避短来有效展示自己的优点，从而建立吸引力，避免因过早暴露缺点而失败。

---

最近我发现，很多男生追不到女生，问题不在于情商，而在于智商。具体来说，许多男生在和女生聊天时，选择的话题并不恰当。例如，有的男生会说：“这两年因为疫情，我都没挣到什么钱。” 甚至还有人会说：“我赔了多少钱。”

也有的男生在国庆假期表示：“我也想出去玩，但是很穷，兜里没几块钱。” 这种一上来就“自爆”弱点的聊天方式，很难追到女生。还有一些男生虽然不聊钱，但他们会说：“我这个人比较直，不太会说话。”

或者说：“我情商比较低，不太懂女生。” 这本质上和聊钱是一样的。即使女生最初对你有好感，也可能被这样的聊天内容劝退。这就好比一个女生一上来就告诉你：“我生活比较乱，和几十个男生交往过，一身妇科病。”

“我还流产过很多次，可能以后都生不了孩子。而且我脾气很差，也不会孝顺。我们结婚后，得让你妈来伺候我，伺候不好我就骂她。我花钱也大手大脚，你一个月挣两三万根本不够我花。”

如果是这样的女生，你还会继续和她聊下去吗？显然不会。因此，请记住一个核心观点：**人无完人，每个人都会有缺点。但女人会为了得到你的优点，而接受你的缺点。** 所以，在刚开始聊天时，你必须学会展示自己的优点。

**关键在于：扬长避短。**

例如，如果你的个子很高，身高一米八七，这就是你的优点。你可以主动问女生：“你有多高呢？” 女生一般会回答，比如一米六五或一米六八。这时你可以说：“我们是最萌身高差，我一米八七。” 这样，优势就自然展现出来了。

再比如，如果事业是你的优点，你自己是公司老板。你可以先和女生聊：“你之前是做什么工作的？” 聊着聊着，女生很可能会反问你的工作。或者，如果她不问，你可以主动说：“我是做XX的，自己开了一家公司。” 从而展示你的优势。

反之，如果你的工作暂时不具备吸引力，例如你是一名月薪三千的保安。那么，在初期聊天中，**可以先不聊工作**，转而聊一些其他能展示你亮点和优势的话题，以此来吸引女生。这不是一场“说实话”的比赛，最老实、最诚实的人未必能获得青睐。

和女生聊天需要多一个心眼。原则是：**不是你有什么就说什么，而是说对你有帮助的话。** 要学会在聊天中展示你的优势和亮点，而不是自己给自己挖坑。

其实，追女生并没有想象中那么难。你只需要做好一件事：**吸引她**。如果她还没有被你吸引，你就急于让她成为你的女朋友，那么失败的概率会很大。目前，很多男生在面对喜欢的女生时，会陷入一个误区：无微不至地关心、讨好、送礼物。

这种行为往往导致女生提不起兴趣，最终拒绝你。如果你很想从这种情况中脱身，可以关注后续的攻略。

---

本节课中，我们一起学习了与女生聊天时的核心策略：**扬长避短**。关键在于初期要有效展示自己的优点，建立吸引力，而不是过早暴露缺点或进行无意义的“自爆”。记住，聊天是展示价值的舞台，而不是坦白缺点的审讯室。
