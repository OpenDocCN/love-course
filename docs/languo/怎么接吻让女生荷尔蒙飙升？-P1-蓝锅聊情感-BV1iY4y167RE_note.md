## 第一课：基础亲吻方法

上一节概述了课程目标，本节中我们来看看第一个基础亲吻方法。此方法强调轻柔与试探，避免过于直接。

具体步骤如下：

1.  缓慢靠近对方。

3.  将对方拉入怀中。
4.  轻轻触碰对方嘴唇即可。
