## 概述 📋

判断女生是否喜欢一个人，可以通过观察其下意识的行为细节。这些细节往往比言语更真实。本节教程将详细拆解四个关键观察点，帮助你进行判断。

## 核心判断依据

上一段我们介绍了观察行为细节的重要性，本节中我们来看看具体有哪四个关键点。

    **你的道歉 + 她的愧疚感 = 共同解决问题的意愿**
    反之，如果她总是无理取闹，无论你做什么都觉得是错的，并且不断消耗你的尊严，那可能说明她并非真心。

4.  **表现出黏人与占有欲**

本节课中，我们一起学习了判断女生好感的四个行为特征：主动分享行踪、手机不设防、在乎你的感受以及在争吵后的态度、表现出黏人倾向。
