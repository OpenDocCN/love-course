# 情感避坑指南 第1课：三种需要谨慎交往的女性类型 🚫💔

在本节课中，我们将学习识别三种在亲密关系中可能带来较大风险或消耗的女性类型。了解这些特征有助于建立更健康、平衡的两性关系。

上一节我们介绍了课程主题，本节中我们来看看第一种需要警惕的类型。

## 第一种：消费无度的女性 💸

这种类型的女性主要特征是花钱没有节制，消费水平远超自身经济能力。

以下是其主要表现和行为模式：

*   **收支严重失衡**：例如，月薪三千元却能花费一万元，月薪一万元甚至能透支消费上百万元。
*   **消费去向**：资金多用于与朋友吃喝玩乐、追求物质享受和社交媒体展示。
*   **经济来源可疑**：其超额消费的资金，很可能来源于父母或伴侣的资助。
*   **承诺不可靠**：即使口头承诺改正，但行为上很难真正改变，最终的经济压力往往会转嫁到伴侣身上。

公式表示为：**她的消费额 >> 她的收入额**。

## 第二种：极度自我中心的女性 🦊

这种类型的女性通常以自我为绝对核心，缺乏换位思考的能力，习惯将责任归咎于对方。

上一节我们了解了经济层面的风险，本节中我们来看看性格与相处模式上的问题。

以下是其典型特征和相处模式：

*   **是非标准单一**：在所有矛盾中，她总是对的，而你是错的。这是一种情感上的操控（PUA）。
*   **缺乏共情与体贴**：例如，当你因急事无法及时陪伴生病的她时，她不会理解你的难处，只会责怪你。
*   **漠视伴侣付出**：她容易忘记你的好处和对她的照顾，甚至可能记不住你的重要日期（如生日）。
*   **核心逻辑**：她的需求与感受是第一位，你的处境和理由不被纳入主要考虑范围。

```python

    责任方 = "你"
else:
    责任方 = 视情况而定 # 但功劳很可能归于她自己
```

## 第三种：自私双标的“普性女” ⚖️

这种类型女性常表现为在关系中一味索取，却不愿平等付出，并且持有双重标准。

上一节我们探讨了自我中心的问题，本节中我们来看看关系中的不平等与双标行为。

以下是其常见表现：

*   **享受好处却不回报**：可以坦然接受你的好，同时可能与其他人保持暧昧，甚至反过来轻视你。
*   **节日索取单向化**：在情人节、七夕等节日，只期待收到你的礼物，却不会为你准备。
*   **进行无底线比较**：当需求未被满足时，会无理取闹，常用话术是“别的女生都有，我也要有”。
*   **双重标准**：只会要求对方付出，却不会用同等标准要求自己。其核心心态是：只要与我有关，你就必须满足我；你如何与我无关。

其行为模式可概括为：**索取 >> 回报**，且 **对伴侣的要求 >> 对自己的要求**。
