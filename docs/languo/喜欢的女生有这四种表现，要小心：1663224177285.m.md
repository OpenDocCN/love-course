# 情感识别课程 01：识别四种需警惕的女生表现 🚨

在本节课中，我们将学习如何识别女生在交往中可能发出的四种“不感兴趣”或“利用你”的信号。理解这些表现有助于你避免在单方面投入的感情中浪费时间和精力。

上一节我们介绍了课程目标，本节中我们来看看具体需要警惕的四种行为表现。

以下是四种需要你格外留心的表现：

1.  **信息回复不积极且行为矛盾**
    她从未秒回过你的信息，并且会在未回复你时，自己发布朋友圈。这表示她并未将你置于优先位置。其核心逻辑可总结为：**她的行为优先级：自己的事 > 回复你**。

2.  **只在有需求时出现**
    她平时对你很冷淡，但总在特定时间点（如情人节、需要帮忙时）主动联系。这种行为模式可以描述为：**联系频率 ∝ 她对你的需求程度**。她并非为增进感情而来，而是为获取帮助。

3.  **长期聊天但拒绝见面**
    你们可能已聊了一个多月甚至更久，但每次你提出约会见面，她都会避而不谈或转移话题。这制造了一种亲密的错觉，但实质关系并无进展。关键点在于：**健康的好感度会推动线下见面**。如果始终无法见面，需警惕对方身份真实性。

4.  **排斥任何肢体接触**
    约会时，她非常介意甚至嫌弃与你有轻微的肢体接触，表现出明显的后退动作。这在亲密关系发展中是一个重要信号：**身体距离 ≈ 心理距离**。持续的排斥通常意味着情感上的疏远。

本节课中我们一起学习了四种在感情中需要警惕的女生表现，包括信息回复矛盾、功利性联系、拒绝推进关系以及排斥肢体接触。识别这些信号能帮助你更理性地判断一段关系是否值得继续投入，保护自己的情感不被浪费。
