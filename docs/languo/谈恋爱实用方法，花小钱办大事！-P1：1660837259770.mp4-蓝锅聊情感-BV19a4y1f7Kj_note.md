理解了金钱的工具属性后，我们来看看第一个具体策略：少量多次。

## 核心策略二：见缝插针

学会了“少量多次”的日常关怀，我们再来看看如何在关键时刻“见缝插针”。

所以，一味的给女生花钱，是99%的男生犯的最大错误。如果你已经这么做了，并且也想学习花小钱办大事的方法，那么可以实践以上策略。

本节课中，我们一起学习了“花小钱办大事”的恋爱核心思想。我们明确了金钱是**建立联系的工具**，而非态度本身。并掌握了两大实践策略：
