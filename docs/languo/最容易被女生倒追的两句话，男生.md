# 情感课程 01：掌握两句话，让女生主动追求你 💬

在本节课中，我们将通过一个真实案例，学习如何通过简单的两句话，巧妙转换你在关系中的位置，从而吸引女生对你产生兴趣，甚至主动追求你。核心在于理解并运用“边界感”与“角色转换”的概念。

---

许多男生认为追求女生需要投入大量时间、精力和金钱，结果却可能不尽人意。既然主动追求如此困难，何不转换思路，让女生来倒追你呢？本节课将展示如何通过几句关键对话，吸引女生暴露需求感，主动向你靠近。

一位学员曾求助，他周末发给女生的消息迟迟未收到回复，非常焦虑。通过分析他们的聊天记录，我发现两个关键问题：一是聊天缺乏目的性，关系无法推进；二是男生缺乏边界感。例如，女生说和闺蜜吃烧烤点多了，男生直接回复“那我过来一起吃”，在关系未到那一步时，这种话会让女生感到突兀，甚至激发她的防备心理。

**核心问题**在于：聊天没有遵循当前关系阶段应有的“边界感”。

## 第一步：重启对话，建立舒适感

上一节我们分析了问题所在，本节中我们来看看如何重启对话。我让学员发送了一条消息：“能不能不要老是发表情包，我都不知道怎么回你了。”

**为什么这样回复？**
1.  它延续了上一次的聊天，自然开启话题。
2.  女生对他有一定好感，才会分享状态。只要把握住合适的边界感，女生不会抗拒聊天。

女生果然回复了。随后他们简单聊了几句，学员提议“出来溜达溜达”，但被女生直接拒绝。许多人在此可能放弃，但对我们而言，机会才刚刚开始。

## 第二步：关键反转，植入新概念

以下是实现角色转换的关键对话：

我让学员回复：“你看我是不是给你省钱了。”
女生回答：“省不省钱啊那都是我的。”
学员接着回复：“看来让你请我看电影的事情泡汤了。”

这时，女生自然会疑惑：“为什么是我请你看电影？”（通常默认是男生请女生）。

**关键点解析**：
女生常拒绝男生，主要并非完全不给机会，而是为了保持矜持，不希望自己被看作“轻易可得”。我们故意反其道而行之，将女生置于“追求者”的位置，向她植入“男生才是被追的角色”这个概念。

于是，我让学员解释：“你请我看电影很容易，但是我请你看电影非常难。”

**这句话的作用**：
*   **公式**：`提出非常规要求` -> `解释原因（设立门槛）` -> `提供可能性（打开窗口）`
*   它合理解释了“为何是女生请客”，同时暗示“我愿意见你，但需要你主动一步”。

女生的意识开始松动，她回复：“你是怕我不去是吗？”（这是一种商量语气）。

## 第三步：引导确认，锁定需求感

既然窗口已经打开，我们便大方承认，不再绕弯子。学员回复：“因为我想见你，但是呢不知道你想不想见我。”

接下来，女生说出了关键一句：“你喜欢我吗？”

**注意**：此时绝不能简单回答“是”或“不是”。回答“是”会将你的需求感升至最高，重新激发女生的防备心理。

**正确做法**：继续将“进攻角色”抛回给女生。
学员回复：“嗯感觉你挺喜欢我的。”

女生随之暴露了需求感，她回答：“你怎么感觉的呀？”这等于间接承认了对男生的好感。

通过这简短的几句交流，女生便承认了对这位学员的喜欢。此后，两人关系迅速推进，第三次约会便确立了关系。
