## 概述

本节将介绍识别特定行为模式的五个核心要点。这些要点基于行为逻辑而非表面现象，目的是提供可操作的判断依据。

## 五大核心特征解析

上一节我们概述了本课程的目标，本节中我们将详细解析每一个特征。

### 1. 好感确认法则 ❓

如果你无法确定对方是否对你有好感，那么答案通常是否定的。对你有好感的人会通过行为让你感知到她的心意。

以下是基于此法则的具体判断逻辑：
*   对你有好感的人会主动释放信号。
*   如果你感知不到任何信号，不应为其寻找“性格腼腆”等理由。
*   核心结论：**感觉不到 = 不喜欢**。

### 2. 物质索取测试 💰

在双方关系尚未建立时，就提出吃饭、礼物或借钱等要求的，需高度警惕。真心喜欢你的人会体谅你的经济状况。

以下是此类行为的关键警示点：
*   以“测试诚意”为名的早期索取。
*   借钱行为尤其可疑，需思考其还款意愿。
*   核心建议：遇到此类情况，应果断远离。

### 3. 情绪胁迫行为 🚫

经常将“分手”挂在嘴边，以此作为情绪发泄或威胁手段，是极不成熟且有害的行为模式。

以下是该行为的特征与影响：
*   利用分手作为解决争执的惯用手段。
*   这种行为会反复发生且可能升级。
*   核心忠告：不要幻想改变对方，应优先保护自己。

### 4. 情感打压（PUA）识别 👁️

通过贬低你的外表、能力等方式进行打压，即使打着“为你好”的旗号，也是不尊重和伤害你的表现。

以下是情感打压的典型表现：
*   言语上的贬低与嫌弃。
*   忽视你的感受，缺乏基本尊重。
*   核心行动：立即远离持续打压你自信的人。

### 5. 卖惨博同情叙事 🎭

通过讲述自己悲惨的情感经历来塑造专一、痴情的受害者形象，从而博取同情和索取资源。

以下是此类叙事的常见套路与目的：
*   将自己描绘为情感受害者。
*   将所有过错归于前任。
*   核心逻辑：其叙事旨在道德绑架，让你为她的“过去”负责。

本节课我们一起学习了识别特定行为模式的五个核心特征：**好感确认法则**、**物质索取测试**、**情绪胁迫行为**、**情感打压识别**以及**卖惨博同情叙事**。记住这些特征的目的是为了做出更清醒的判断和决策，保护自己的情感与利益。如果遇到同时符合多项特征的情况，最明智的做法是及时抽身。
