## 开场：打破客套，建立领导印象

**核心技巧**：对于刚认识的陌生人，用轻松的话语打破客套非常有效。一开始展现出的领导力，能给女生留下很好的印象。

女生回复：“那你想怎么聊呢？”这表明女生开始积极互动。不拒绝就是主动，这是一个好兆头。

**过渡**：上一节我们介绍了如何用轻松的开场打破僵局。本节中，我们来看看如何通过包装自己来强化第一印象。
