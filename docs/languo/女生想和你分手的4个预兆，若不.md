# 情感课程 01：识别分手预兆的四个关键信号 🚨

在本节课中，我们将学习识别亲密关系中可能出现的分手预兆。通过了解这些关键信号，可以帮助你更早地察觉关系的变化，并思考如何应对。

## 信号一：脾气变差与指责增多

上一节我们概述了本课内容，本节中我们来看看第一个具体信号。当伴侣的脾气变得比以往更差，并且对你所犯的小错不断指责时，这可能是一个关系疏远的迹象。

## 信号二：分享欲显著下降

在了解了情绪变化的信号后，我们接下来观察沟通模式的变化。伴侣逐渐失去分享欲望是一个重要信号。其表现模式可以概括为以下对比：

以下是分享欲下降前后的典型表现：
*   **之前**：即使是小事也乐于与你分享。
*   **现在**：可能连续多天都不回复你的消息。

## 信号三：回避见面与相处

沟通减少之后，对方在行为上也可能开始疏远。伴侣开始频繁拒绝与你见面，是关系降温的明显行为指标。

以下是热恋期与疏远期的邀约对比：
*   **热恋期**：希望时刻与你相处。
*   **疏远期**：当你主动邀约时，她常以“工作忙”或“有安排”等理由拒绝。

## 信号四：拒绝共同规划未来

当日常相处出现问题时，长远规划的意愿更能反映关系的核心状态。伴侣不再愿意与你讨论或规划共同的未来，是关系可能走向终结的深层信号。

其反应模式可以描述为：
```python
if 谈论未来计划:
    她的反应 = “十分冷淡的回复”
else:
    沟通照常
```
这意味着，一旦话题涉及未来规划，对方会表现出明显的冷漠和回避。
