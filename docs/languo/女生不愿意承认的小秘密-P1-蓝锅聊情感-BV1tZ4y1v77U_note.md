## 1. 外貌是重要的初始吸引力 👀

```
初始接触意愿 = f(外貌吸引力)
```

当他人通过初始接触，进一步发现你具备有趣、温柔、有学识等内在品质时，你的整体吸引力会得到增强。

## 2. “忙碌”并非不回信息的真正理由 ⏰

理解了外貌的初始作用后，我们再来审视沟通中的另一个常见现象。本节将分析信息回复背后的潜在含义。

女生即使忙碌，也极少会一整天都没有时间回复一条信息。部分男性倾向于进行自我欺骗，并主动为对方寻找借口。

*   对方可能确实在忙碌，但忙碌的对象是回复其他人。
*   “没有时间”往往等同于“没有给你分配时间优先级”。

## 3. 热情骤降通常意味着注意力转移 🔄
