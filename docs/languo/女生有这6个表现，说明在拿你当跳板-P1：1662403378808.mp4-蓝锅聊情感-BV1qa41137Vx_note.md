# 情感识别课 01：识别“跳板”关系的六个迹象 🚩

在本节课中，我们将学习如何识别一段关系中，对方可能并未投入真心，而只是将你视为暂时过渡或向上攀爬的“跳板”的六个关键表现。这些迹象有助于你更清醒地看待关系现状，避免不必要的内耗。

## 概述

一段健康的关系建立在相互尊重和双向奔赴的基础上。然而，有时一方可能并未打算长久发展。以下六个表现，是判断对方是否真心的重要参考。每一个表现都源于实际案例的总结。

## 六个关键表现

上一节我们概述了本课的核心目标，本节中我们来详细看看这六个具体表现。

以下是识别“跳板”关系的六个关键迹象列表：

1.  **争吵后总是需要你主动低头**
    否则关系就无法和好。她在这段感情中计较的是输赢，而非解决问题。真心喜欢你的伴侣会愿意轮流主动沟通，追求关系的持久，而不是执着于“谁先认输”。

2.  **只强调自己的付出，忽视你的努力**
    关系的付出应讲究**等价交换**。但她动辄只把自己对你的好挂在嘴边，并以此作为筹码。同时，她会贬低你的付出，让你觉得做什么都是错，永远得不到正向回应。根本原因在于她不够喜欢你。

3.  **从不或懒得向你解释**
    例如，她背着你与其他异性交往，当你询问时，她只会说“那是朋友，你爱信不信”，甚至反过来指责你不够信任她。这种态度会让你陷入自我怀疑。心里有你的女生，不会如此处理问题。

4.  **从不讲道理，永远是你错**
    只有重视一个人，才会愿意耐心讲道理。同理，如果女生心里有你，她会倾听也会与你沟通。如果在她那里你永远都是错的，你就需要意识到问题的严重性。

5.  **公开表示亲人朋友永远比你重要**
    这句话像是一份“免责声明”。潜台词是：“以后我的坏脾气你要忍受，当我的亲友需要我时，你必须退让。我已提前告知，若想继续在一起，就要接受这种不对等的待遇。”

6.  **总把“分手”挂在嘴边，动辄删除拉黑**
    女生撒娇或小打小闹，可能是为了寻求关注。但频繁以分手作为威胁，动辄切断联系，则是不在乎的表现。真心喜欢你的人，第一反应是解决问题；而不喜欢你的人，会觉得“分手正好，反正我不缺你一个”。

## 总结

本节课中，我们一起学习了识别一段关系中对方是否将你视为“跳板”的六个迹象：**总是你低头、忽视你的付出、拒绝解释、不讲道理、将你排在亲友之后、以及常提分手**。如果你在关系中察觉到多个此类表现，就需要冷静评估这段关系的健康程度。认清现状是做出明智决定的第一步。
