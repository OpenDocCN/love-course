# 情感课程01：识别真心与假意 🔍

在本节课中，我们将学习如何通过四个关键特征，判断一位女性对你是真心投入，还是仅仅在“玩”感情。这些特征基于具体的行为模式，能帮助你更清晰地审视一段关系。

## 特征一：凡事只考虑自己

上一节我们介绍了本课程的目标，本节中我们来看看第一个特征。如果对方在相处中表现出高度的自私，凡事只优先考虑自己的利益和感受，而很少顾及你的需求和情绪，这是一个需要警惕的信号。

**核心表现**：她的决策公式倾向于 **Max(自我利益)**，而非 **考虑双方共赢**。

## 特征二：以考验为名进行测试

接下来，我们探讨第二个特征。如果对方频繁地为你设置各种“考验”，例如提出不合理的要求或制造困境来测试你的反应和忠诚度，这可能意味着她并非以真诚建立关系为目的。

**核心表现**：关系互动模式类似于一场持续的 `if-else` 测试：
```python
if 你通过考验：
    她暂时满意
else：
    她表示失望或施加压力
```

## 特征三：接受付出但不公开关系

在了解了前两个基于互动的特征后，第三个特征关注关系状态。她乐于接受你的关心、爱意和物质付出，但在社交层面始终拒绝公开你们的关系，避免让你进入她的朋友圈或家庭圈。

**核心表现**：关系状态处于 **“地下化”**，公式可表示为：**接受你的付出 ≠ 承认双方关系**。

以下是此特征下的常见行为：
*   避免在社交媒体上发布与你相关的状态。
*   拒绝带你参加朋友聚会或家庭活动。
*   当被问及时，模糊处理或否认你们的恋人关系。

## 特征四：态度忽冷忽热，难以捉摸

最后，我们分析第四个特征，即情绪与投入度的不稳定性。她对你的态度会无缘无故地突然冷却或升温，行为模式难以预测，让你感到困惑和不安。

**核心表现**：她的情感投入度曲线是 **不规则波动** 的，而非稳定上升，这使你无法建立稳定的安全感。

## 核心总结与行动建议

本节课中，我们一起学习了识别“玩感情”的四个核心特征：**自私自利**、**不断考验**、**关系不公开**以及**态度忽冷忽热**。此外，视频还补充了“不主动、不拒绝、不负责”的“吊着”行为。

每个人心中都有一把衡量关系的尺子。如果你所处的感情关系呈现出上述特征的恶性循环，最明智的选择是及时止损，尽快结束这段消耗你的关系。
