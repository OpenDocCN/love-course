## 圈子狭窄是首要障碍

上一节我们概述了课程目标，本节中我们来看看导致单身的第一个主要原因：社交圈子狭窄。

导致你现在还没有脱单的原因，首当其冲的就是你的圈子小。朝九晚五两点一线的生活，再加上网上让你先搞钱、就有对象的毒鸡汤，自然而然你长期就是单身。所以当务之急就是脱离社畜的生活模式。

以下是这种生活模式的具体表现：

*   每天睁开眼洗漱开始上班。
*   晚上下班之后回家点个外卖打游戏。
*   到了周末开始自我洗脑，熬夜周休休息一天一定要睡到自然醒。

整个人的社交范围非常拘泥于家和公司之间。偶尔参加一下朋友的聚会，遇到的都是那些无法下手的朋友。公司不让发展办公室恋情，参加朋友聚会认识的新异性资源，都是那种又高冷又腼腆的女生。偶尔打个游戏网聊，认识的大多也是搞搞暧昧，聊着聊着就没结果了。这样的话你还怎么能够让异性注意到你呢？你要怎么能结识真正的缘分？

因此，丰富的业余生活你要走出去，结识异性。当你的业余生活能够涌入不断的异性的时候，自然会被他吸引，异性的基数也会变得越来越大。你脱单的概率不就越高了吗？

## 自我管理：不怕你“烂”，就怕你“摆烂”

了解了拓宽圈子的重要性后，我们来看看第二个关键点：个人形象与自我管理。

其次你还没脱单，是不怕你烂，就怕你摆烂，一定要支棱起来。说一个很残忍但是又很现实的事实，就是你自己的外在管理能力欠佳。现在很多人抱怨，说什么婚恋市场都如此的看脸。其实从某种意义上来说，大多数人的恋爱冲动，往往都是源自于他们见色的起意，才能谈到日久生情。你不能说你烂你还摆烂吧。

但这里的“色”不单单是你先天的长相，更多是来自后天的塑造和包装。也就是说，一个男生你要长期吸引女生的注意，那么你可以从颜值上面去下手。你可以不是很帅，但是你一定要干净整洁。

**公式：吸引力 = 先天条件 + 后天塑造**

比如你的衣品，那对于女生来说，你是不是就有价值了呢？

## 核心关键：理解并满足对方需求

最后也是最重要的，你不懂女生的心理。恋爱是一个制造需求、给予需求的过程。需求的本质实质上没有大家想象的那么复杂。你如何去满足一个人的需求，很大程度上要通过认知的路径方法。那个太深了，不给大家扯大道理，我以后会专门出视频去讲，也会放在课程里面做一个比较核心的章节。

我简单地举个例子。在聚会上你总会遇到一些和你同样落单的人，但实际上他的心态和你一样。他内心的需求，可能是希望被关注、被陪伴。所以你这个时候，如果能观察一个社交圈里面能量场比较低的人，你能够去照顾这个人，是不是他的回应就会很好呢？是不是他的需求就被满足了呢？

**代码示例（互动逻辑）：**
```
if 感知到对方有“被陪伴”需求：
    主动发出邀请（例如：“一块打个车回家吧”）
else：
    继续观察或寻找其他互动契机
```

你能够主动发出邀请，比如散会了，来一天这么晚了，一块打个车回家。那这个时候会感觉一种莫名的陪伴和暖意涌上心头。对于女生来说，此时此刻，你就是我的白马王子。当然你明白了原理，就可以在各种各样的场景里面游刃有余。其实你可以反过来想一想，对方是不是有求于你，你才是有价值的。
