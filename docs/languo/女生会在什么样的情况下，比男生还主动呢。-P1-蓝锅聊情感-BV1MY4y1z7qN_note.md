## 理解“延迟满足”的核心

上一节我们介绍了课程的主题，本节中我们来看看其核心原理。女生在什么情况下会比男生更主动，关键在于“延迟满足”。这是一种情商非常高的心理技能。

其核心公式可以表述为：
**主动性的激发 = 自我控制 × 对方的情感投入**

## 如何实践自我控制

理解了原理后，我们面临一个实际问题：如果你非常想念对方，控制不住想联系他，该怎么办？

3.  **设定等待期**：给自己设定一个硬性规定，比如“至少24小时后再考虑是否联系”。
4.  **记录感受**：将你想说的话或感受写下来，但不发送。
