# 情感课程 01：四种需摒弃的、易遭女生嫌弃的行为 🚫

在本节课中，我们将要学习四种在交往中容易让女生感到不适甚至嫌弃的男性行为。理解并避免这些行为，有助于建立更健康、更受欢迎的社交形象。

---

上一段我们概述了本课主题，本节中我们来看看具体是哪四种行为需要注意。

以下是四种需要摒弃的具体行为：

1.  **主动向女生发送健身照片**

3.  **错误理解“坏”与“帅”的界限，并开不合时宜的玩笑**
    “男人不坏女人不爱”中的“坏”，指的是风趣、有魅力的冒险精神，而非低俗的“猥琐”。例如，自以为穿着拉风、动作豪迈就是帅，甚至开不合时宜的“车”（指色情玩笑），并误将女生的礼貌回应当作真心夸奖，这种行为会遭人厌烦而不自知。

4.  **用眼神上下打量对方**
    遇到心仪的女生时，用眼神从头到脚地打量对方，被发现后迅速移开视线，这种行为会被视为**非常猥琐且掉价**。正确的做法是，如果想认识对方，就大方上前交流，即使被拒绝也无妨。
