# 情感课程 01：如何判断女性是真心相爱还是逢场作戏？ 💔

在本节课中，我们将学习如何识别女性在感情中的真实意图。我们将分析五个关键特征，帮助你判断对方是真心投入，还是仅仅在“玩”感情。

---

上一段我们介绍了本课程的目标，本节中我们将具体探讨女性“玩”感情的五个明显特征。

以下是五个需要警惕的行为特征：

1.  **自私**  
    对方凡事只考虑自己，经常忽视你的感受。

2.  **考验与测试**  
    对方会用各种理由和借口来考验你、测试你，让你感到困惑和被操纵。

3.  **不公开关系**  
    对方接受你的关心、爱与付出，但拒绝公开你们的恋爱关系。

4.  **态度忽冷忽热**  
    对方对你的态度琢磨不透，时而热情，时而冷淡。

5.  **循环“吊着”你**  
    尤其是在你准备放弃时，对方会主动联系你，给你希望，使关系陷入 **“冷淡 -> 给予希望 -> 再次冷淡”** 的恶性循环。

---
