在本节课中，我们将探讨一个在亲密关系中普遍存在的现象：过度付出与低自尊的形成。我们将分析其背后的心理机制，并提供两个核心的底层逻辑，帮助你建立更健康、更平等的情感关系。

许多男性在进入恋爱关系后，会不自觉地从一个“追求者”转变为“主动付出者”。这种角色的转化往往在当事人毫无察觉的情况下发生。

这种无形且过度的付出，会直接导致你在关系中变得越来越低自尊。

---

### 低自尊的表现与后果

上一节我们提到了角色转化与过度付出，本节中我们来看看低自尊的具体表现及其带来的后果。

以下是低自尊在关系中的常见表现：

*   **对方生气时，你会第一个道歉。**
*   **对方对你冷暴力时，你会控制不住自己主动联系。**

这样的相处模式持续越久，你越容易被对方轻视。网络上常见的“男人花钱却得不到真心”的例子，其根源往往就在于这种不平等的付出模式。

---

### 建立高自尊的两个底层逻辑

认识到问题后，关键在于如何改变。前面我们介绍了低自尊的形成与表现，本节中我们来看看如何通过建立底层逻辑来规避它。

男生必须学会用**逻辑**去爱女人，而不是用**感觉**去爱女人。以下是建立高自尊感的两个核心方法：

**第一，学会给自己立规矩。**

核心是保持情绪稳定，避免大起大落。从今天起，做任何事情，不要因为得到而狂喜，也不要因为失去而长期消沉。

一个懂得定期给自己放假、旅行释放情绪的男人，是情绪稳定的。这样的你不会因为朋友的一个感动瞬间就掏心掏肺，也不会因为女人的某个瞬间就爱到不行。公式可以概括为：**情绪稳定性 = 避免极端情绪反应 + 定期自我疏导**。

**第二，懂得筛选圈子，包括女人。**

一个真正值得被爱的男人，具备“大客户思维”。他非常清楚，只有一方单方面付出的感情，一定是低质量的。

真正优秀的女人，她的所有表现都是你能切实感受到的，她在滋养你，让你成长。相反，低质量的女人会给你虚幻的陪伴，让你陷入“你给我物质就是给我爱”的恋爱幻想中。这种关系对你的认知成长毫无意义。她的“爱”往往与你的赚钱能力挂钩，这可以用一个条件语句来描述：
```python
if 你的赚钱能力提高:
    她就会跟着你
else:
    她就可能离开
```
拓展高质量的社交圈，比一句虚幻的“我永远爱你”更为坚实。
