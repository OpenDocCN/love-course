## 核心表现：消极与敷衍的互动模式

上一节我们介绍了课程主题，本节中我们来看看最核心的行为表现。当对方不再喜欢你却仍维持联系时，其最明显的特征是互动中的极端被动与敷衍。

*   **绝对不主动联系**：如果你不找她，她绝对不会主动找你。如果你不发消息，她也绝对不会给你发消息。
*   **回应敷衍或延迟**：即使你主动发了消息，她的回复也会非常敷衍，或者隔很久才回复。
*   **常用借口推脱**：甚至可能完全不回复，然后在第二天用“睡着了没看到”或“手机没电了”这类借口来解释。

## 行为背后的逻辑：逃避责任与情感操控

了解了表面的互动模式后，我们进一步分析这些行为背后的心理动机。其根本目的是避免自己成为关系结束的“责任人”。

2.  **拒绝主动终结关系**：她不会主动提分手，因为不想扮演“坏人”角色，那会显得她太“渣”。
3.  **诱导对方提出分手**：她的策略是“吊着你”，刺激你因无法忍受而主动提出分手，从而让她扮演受害者的角色。
4.  **完成责任转嫁**：最终，她会将关系破裂的全部责任推到你身上。这一切都在她的计划之中。
