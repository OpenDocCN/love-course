# 情感沟通课程 01：当女生主动问“在干嘛”时的高情商回复指南 🎯

在本节课中，我们将学习如何高情商地回复女生主动发来的“在干嘛”信息。我们将分析不同段位的回复方式，并掌握让对话氛围升温的核心技巧。

---

许多男生在不知如何开启话题时，常会发送“在干嘛”、“吃了吗”这类信息，试图让女生打开话匣。然而，女生往往觉得此类聊天平淡无趣，完全提不起兴趣。男生对此束手无策，对话只能草草结束。

但反过来，当女生主动给你发“在干嘛”时，你总不能也提不起兴趣、让聊天冷场吧？究竟该如何回复呢？本节我总结了不同段位的回复方式。

让我们来看看你属于哪个段位，是钢铁直男还是高情商男生？一起来做个自我检测吧。

以下是第一种典型回复：

*   **男生回答得相当诚实**：女生问“在干嘛呢”，想都不想直接回复“我在吃饭呢”。这是典型的直男回复，让女生完全无法接话。
*   **对话陷入僵局**：女生可能只回一个“哦”。这时男生才察觉不对劲，开始反问女生“那你在干嘛呢”，试图让对方提供话题。
*   **女生感到无语**：女生此时可能更无语，直接回复“没干嘛”。正是因为男生之前表现得过于风轻云淡，直接给女生的热情泼了一盆冷水。这冷水一淋，太阳都能被你变成水星，聊天自然进行不下去。

处于这个段位的男生，需要多加练习。接下来我们看看第二种情况。

上一节我们看到了过于平淡的回复会冷场，本节中我们来看看另一种看似聪明、实则可能弄巧成拙的回复。

以下是第二种需要警惕的回复方式：

*   **男生意识到女生的潜台词**：一个女生在无聊时，没去看电视、逛淘宝或打游戏，而是主动找你聊天，这其实是在**变相地说“我想你了”**。
*   **试图营造暧昧但方法错误**：男生虽然明白女生的意思，也知道要让聊天氛围暧昧起来，但他这样回复：“**我在想你啊**”。
*   **回复过于油腻**：这种聊天方式很可怕。虽然看出了女生的心思，但回复太直接、太油腻，让女生感觉不到真诚。女生只会觉得和你聊天“很下头”，后悔自己主动找你。

这也不行，那也不行，到底该怎么回呢？不要慌，一起来看一个成功案例。

之前我们分析了两种错误的回复方式，本节我们将通过一个具体案例，学习如何掌握主动权，营造暧昧氛围。

**案例背景**：一位男生在玩密室逃脱时认识了一位女生，女生主动加了他微信。之前聊得不错，但关系卡在“朋友以上，恋人未满”的阶段。这时，女生发来一句：“在干嘛”。

**核心思路**：如前所述，女生说“在干嘛”往往等于“我想你了”。所以我们要做的是**掌握主动权，直接发起进攻**。

以下是分步对话解析：

1.  **主动出击，引导对方**：
    *   我让男生回复：“有事说事，没事就直接说想我了嘛。”
    *   女生回复：“就问问你了。”
    *   我们继续回：“看来是没有事情，那就等于是在说想我了。”
    *   **技巧说明**：平时多是男生向女生暴露需求感，而这里是在**让女生主动暴露出她更多的需求感**。

2.  **制造情绪，避免直白**：
    *   女生回复：“为什么你不说想我了？”
    *   很多男生会直接说“我正在想你啊”，但这无法调动女生情绪。
    *   **高情商回复**：“正准备拿起手机找你聊天，就看到你的消息了。虽然很默契，但是你居然捷足先登。”
    *   **技巧说明**：这里我们**点破不说破**，让聊天变得有意思起来。

3.  **巧妙应对调侃，推进关系**：
    *   女生说：“男人的嘴骗人的鬼。”如果你直接开始解释，就上当了。
    *   **高情商回复**：“男人的嘴只能骗好色鬼。你好色一点，我肯定能骗到你。”
    *   女生回：“我才不好色。”
    *   我们继续回：“你好不好色，只有未来的男朋友才知道吧。”

4.  **以退为进，化解试探**：
    *   女生回：“都不知道那个男的在哪呢。”这句话很关键，看似自嘲，实则在**试探我们的态度**。如果表现得太着急，会让她失去好感。
    *   **高情商回复（以退为进）**：“要不要我给你介绍介绍？”
    *   女生问：“介绍谁？”
    *   **机会来了，我们回复**：“你还真打算让我介绍？小坏蛋，你和我聊天的时候，还在想别的男人是不是不太礼貌啊？”
    *   **技巧说明**：这样去聊，暧昧的气氛和情绪自然就产生了。**不用说自己是她未来的男朋友，但又暗示了这个可能性**。

5.  **化被动为主动，锁定动机**：
    *   女生回复：“哈哈。”看到女生情绪不错，我们应该**化被动为主动**。
    *   **最终一击**：“那你打算什么时候承认，这次聊天的动机，就是因为想我了？”
    *   **效果分析**：这几句话能让女生的情绪一下子被提起来。

本节课中我们一起学习了回复“在干嘛”的进阶技巧。让女生喜欢你，无非抓住几个核心点：**给她良好的感觉**，以及**通过聊天让好感升级**。
