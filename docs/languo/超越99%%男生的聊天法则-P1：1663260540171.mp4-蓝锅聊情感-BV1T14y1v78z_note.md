# 超越99%男生的聊天法则 - 课程01：三大核心法则 🚀

在本节课中，我们将要学习三个核心的聊天法则。这些法则旨在帮助你更有效地与女生沟通，建立更好的互动关系。我们将依次介绍聊天代入法、共情法和逆向思维法，并通过具体例子说明如何应用。

---

## 聊天代入法 🎯

上一节我们介绍了课程概述，本节中我们来看看第一个法则：聊天代入法。面对刚认识的女生，你可能不了解她的喜好。聊天代入法在这种情况下非常适用。

例如，你想约女生吃饭，但不知道她想吃什么。大部分男生会直接问“你想吃什么”或“你猜我想带你吃什么”。女生通常回答“随便都行”。如果你相信这个回答，你就是那99%的男生。

正确做法是让男生说：“你觉得我一会儿会带你去吃什么？”这会把女生带入一个具体场景。大部分女生的回答会是她真正想吃的东西。这是有心理学依据的。

假如女生回答“火锅”，你可以说：“可以啊，咱俩心有灵犀。”女生会感到高兴，你也不用为选择餐厅而发愁。

## 共情法 🤝

在掌握了聊天代入法后，我们进入第二个法则：共情法。假如你们已经初步相处，彼此有一些好感。女生可能会无意识地跟你吐槽，例如今天说跟同事闹矛盾，明天说姐妹对象又分手了。

以下是大部分男生的错误回应方式：
*   实在不开心你换一个。
*   分就分吧，多大点事啊。

这就是典型的直男思维。这些回应除了给女生安慰，就是转移注意力。女生跟你聊这些，只是想吐槽一下。如果不会说话，那就别说话。

实在不知道怎么办时，可以采用“同吃”方法。假如女生吐槽公司老板，你可以说：“我突然想起来，我有一次工作没完成好，被领导劈头盖脸一顿痛批。我真是太懂你了。我知道一家味道很好的螺蛳粉，开了几十年了，中午我带你去嗦一下。”

短短的几句话，不仅跟女生站在了同一个阵营，还能达成情绪上的共鸣。女生会觉得你真的懂她，你没有敷衍她。

## 逆向思维法 🔄

上一节我们探讨了如何建立情感共鸣，本节中我们来看看第三个也是最高阶的法则：逆向思维法。这一点需要认真听，因为它决定了你最终会成为“舔狗”还是“暖男”。

很多男生对女生随叫随到。女生一撇嘴、一撒娇、一掉眼泪，男生就觉得没有办不到的事情。这种做法恰恰相反，你越随叫随到，女生可能越不喜欢。

想让女生记住你，就得学会逆向思维。你越特殊，她越记得你。

例如，女生想让你帮忙取一个快递。一般男生会二话不说：“好，我来，你快递号发我。”拿回来后女生说“谢谢哥哥”。但如果你不拿，女生可能还会记恨你。

正确做法是让男生说：“可以啊，给你优惠一杯奶茶，先记账上，两次就两个。”这是在调侃女生，要求她为我们支付所谓的“价值”。一般男生不会这样做。

这种等价交换的思维，如果你能记在脑子里，下一个女朋友就是你的。这样的聊天既能创造话题，又能一举两得。
