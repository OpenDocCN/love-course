# 情感技巧课程 P1：引导对方主动称呼的互动游戏 🎮

在本节课中，我们将学习一个简单有趣的互动游戏。这个游戏能通过轻松的方式，在约会中创造亲密氛围，并自然引导对方说出特定称呼。

## 游戏准备与介绍

首先，你需要邀请对方参与一个名为“快速问答”的小游戏。这个游戏的公开目的是测试智商和反应速度。你需要向对方说明规则：问题提出后，必须在一秒钟内给出答案。

## 游戏问题与步骤

以下是游戏的具体流程，包含三个按顺序提出的问题。

**第一个问题**
你问：“桃子、李子、猴子，哪一个不是水果？”
对方通常会立刻回答“猴子”。这个问题简单，用于建立游戏节奏和对方的信心。

**第二个问题**
紧接着你问：“火车、风车、汽车，哪一个不是交通工具？”
对方大概率会秒回“风车”。第二个问题延续了相同的思维模式，进一步巩固对方的答题惯性。

**第三个关键问题**
然后你立刻提出第三个问题：“老爸、老妈、老公，哪一个跟你没有血缘关系？”
在之前的快速反应惯性下，对方通常会不假思索地回答“老公”。

## 游戏的收尾与效果

当对方说出“老公”这个答案后，不等其反应，你立刻回应：“我老婆真聪明。”
这句话完成了整个互动，将游戏答案巧妙地转化为彼此间的亲密称呼。这种方法利用了心理学中的反应惯性原理。

## 课程总结

本节课我们一起学习了一个通过快速问答游戏引导称呼的社交技巧。其核心在于通过前两个问题建立固定的回答模式，在第三个问题中利用这种惯性获得期望的答案，并通过即时回应将答案转化为亲密互动。整个过程需要连贯、快速地进行。
