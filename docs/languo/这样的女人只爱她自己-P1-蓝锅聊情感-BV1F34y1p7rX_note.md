# 情感分析课 P1：识别自我中心型伴侣 🧐

在本节课中，我们将学习如何识别一种在感情中以自我为中心、行为模式具有特定倾向的伴侣类型。理解这些特征有助于建立更健康的情感认知。

## 核心特征分析

上一段我们介绍了本节课的主题，本节中我们来具体看看这类伴侣的核心行为模式。其根本动机可以用一个公式来概括：**她的决策始终遵循 `自身利益最大化` 的原则**。

以下是此类伴侣的几个典型特征：

*   **易受外界影响**：她容易被其他异性吸引或“撩”动，主要根源并非外界诱惑有多强大，而在于她自身内在的不安定性。
*   **情感权衡功利化**：她在感情中会持续进行得失权衡。她今天的承诺可能基于你当前提供的价值，明天的离开也可能源于他人提供了更高的价值。
*   **承诺的短暂性**：她的情感承诺是短暂且易变的。她可以今天说爱你，明天却为了另一个男人毫不犹豫地离开。
*   **难以满足与多头维系**：她难以在单一关系中获得长期满足，可能同时维系多段关系，寻求“家庭稳定”与“外界刺激”并存。
*   **爱的对象是自己**：其所有行为的底层逻辑是自爱。她爱的并非某个特定的伴侣，代码可以表示为：
    ```python
    if decision.benefits_self == True:
        take_action(decision)
    # 其行动始终以是否有利于自己为判断条件
    ```

## 行为模式总结

本节课我们一起学习了“自我中心型”伴侣的关键特征。总结来说，这类伴侣的情感行为主要由内在的不安定性和对自我利益的极致权衡所驱动，其忠诚度与选择高度依赖于外部所能提供的价值变化，且核心情感指向是自身而非伴侣。

理解这些模式，旨在帮助建立清晰的情感认知边界，而非用于简单评判。在复杂的情感世界中，保持清醒的洞察力是自我保护的重要一环。
