## 🔍 第一部分：通过见面行为判断

上一节我们介绍了课程的核心目标，本节中我们来看看如何通过实际见面来判断女生的态度。

你可以尝试把女生约出来。见面时的行为和反应是判断其真实意图的重要依据。

以下是三种不同态度对应的行为模式：

1.  **喜欢**：如果女生喜欢你，她会愿意出来见面。在相处中，她可能不抗拒一些轻微的亲密接触（如不小心碰到手），或与你对视。在拥挤的环境中（如地铁），如果她的反应是害羞而非抗拒，那么你们关系升级只是时间问题。公式可表达为：**愿意见面 + 不抗拒亲密 = 喜欢**。

2.  **拒绝**：如果她在行为上表现出明显的排斥和嫌弃，与你保持距离，见面后态度明显变冷、懒得理你，这表示她对你完全没有兴趣，甚至连考察你的想法都没有。公式可表达为：**排斥行为 + 见面后冷淡 = 拒绝**。

3.  **吊着你**：如果她在行为上拒绝你、排斥你，但在语言上却不断给你希望，这就是在吊着你。这类女生本身可能情感经验丰富，善于操控对方期望。

## 💬 第二部分：通过聊天模式判断

如果无法约出女生，我们也可以通过日常聊天来观察端倪。聊天中的语言模式和提供的希望值是关键线索。

最擅长吊着男生的女生，从来不会在语言上直接拒绝，反而会不断给予男生希望，鼓励他们“加油”、“再努力一点”。

以下是“吊着你”的典型聊天特征：

*   她们不会真正亲近你，但会为你付出的努力给予小额“奖赏”（如一次约会、一次牵手），让你激动并继续投入。
*   她们有时会主动进行一些轻微的接触或勾搭，让你心潮澎湃，但永远不会让关系有实质性进展。
*   她们常说的话术是：“你要是爱我，就要听我的。” 这是高手经营希望的手段，让你不断看到近在眼前却无法触及的目标。

很多男生的死穴在于太相信女人说的话，而不去思考她的行为。一个美女的几句话就能让男生心花怒放。但正确的判断方法是：**别听她说什么，要看她怎么做**。即使她说“爱死你了”，却总以客观理由回避你，那就是不喜欢。

> 女生也可以通过同样的方式来判断男生。
