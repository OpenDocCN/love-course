## 概述：警惕关系中的特定话语

## 核心概念与引导付出

上一节我们概述了课程目标，本节中我们来看看第一个核心概念：**引导付出**。

某些话语旨在引导对方持续进行物质或情感上的付出，其逻辑可以概括为：**对方付出越多 → 对方的情感投入（沉没成本）越高 → 对方在关系中越被动**。

以下是与此相关的一句典型话语及其分析：

*   **话语**：“我早已过了用耳朵听爱情的年纪。”
*   **潜在含义**：这句话暗示，仅凭言语表达的爱意不够“实在”，需要对方通过实际行动（如送礼、请客、转账）来证明其喜欢程度。其目的是**引导你持续付出**。
*   **正常情况对比**：一个对你没有建立好感的正常人，通常会谨慎接受你的付出，以免造成误会或心理负担。
*   **关键判断**：如果你喜欢一个女生，而她对你并无明确兴趣却仍引导你付出，这很可能是一种**拿捏**你的策略。
