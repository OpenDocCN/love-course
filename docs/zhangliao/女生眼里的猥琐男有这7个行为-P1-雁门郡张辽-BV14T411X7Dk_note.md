## 一、说话油腻与不合时宜的调情 💬  

上一节我们介绍了课程概述，本节中我们来看看第一种反感行为。  

说话油腻表现为不分场合地使用土味情话或骚话。这些话本身可能无问题，但必须在关系适当时机使用，否则会显得轻浮或像调戏。  

**核心概念示例**：  
```
不当时机 + 油腻话语 = 反感情绪
```

## 二、过度吹牛与自我夸大 🌪  

接下来，我们探讨第二种行为。  

喜欢吹牛的男性常提及“我的朋友很厉害”“我过去多成功”或“小时候多厉害”，这容易给人留下不踏实、虚荣的印象。  

以下是吹牛的常见表现：

- 夸大个人经历  
- 重复强调过去荣誉

## 三、聊天开场白单一乏味 💤  

在了解了吹牛问题后，我们转向沟通方式。  

第三种行为是聊天时永远使用同一套开场白，例如“美女在干嘛”“出来喝东西吗”“约吗”。这反映出缺乏互动技巧，不懂如何逐步拉近关系。  

**建议**：可参考多样化的聊天记录解析，学习如何从冷启动逐渐活跃对话。
