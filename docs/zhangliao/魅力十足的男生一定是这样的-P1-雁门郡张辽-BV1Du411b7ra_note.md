## 核心吸引力特质分析

上一节我们提到了吸引力并非源于外在条件，本节中我们来看看具体哪些内在特质更具吸引力。

以下是女生通常喜欢的男生特质列表：

*   自信、自在
*   强势、有侵略性
*   胆大、有控制力
*   无法被顺服、不在乎
*   有神秘感
*   有点大男子主义

这些特质组合起来，给人一种**自由自在、游刃有余**的感觉，就像第一次去某个地方或做某件事，却如同回到家里一样熟悉。这些点常被俗称为“坏男人”、“二流子”或“渣男”身上的必备特质。

## 缺乏吸引力的特质对比

了解了具备吸引力的特质后，我们再来看看与之相对、通常缺乏吸引力的特质。

以下是女生通常不喜欢的男生特质列表：

*   老实、呆板、无趣
*   弱势、跪舔、胆小
*   教条、内向、平淡
*   没主见、谨小慎微
*   条条框框多、自欺欺人
*   宅、被价值观念束缚
*   喜欢占据道德制高点
*   没见过世面、情感经历少
*   没有恋爱选择权
*   表达能力差
*   心理素质差、易紧张
*   以他人为重心
*   爱照顾他人而忽略自己

这些点常被大家俗称为“老实人”的特质。这可能令人难以相信，但这就是事实。**吸引力 = 内在特质 + 行为模式**，它与金钱或帅气没有直接关系。
