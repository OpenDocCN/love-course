## 五大核心心态解析

以下是追求女生时需要培养和坚持的五种核心心态。

### 1. 不要过度的期望

在还没怎么样的时候，不要让对方知道你超级喜欢她。这个技巧指的是，聊天见面呢，都不能表现出“我太喜欢你了”，你会把女生给吓跑。自然的相处，反而呢，对方呢会相处得很舒服，觉得你不错。

**核心公式：** 好感展示度 < 当前关系亲密度

### 2. 发自内心的欣赏与尊重

发自内心的欣赏对方，尊重对方。尊重对方之前呢，你要先尊重自己。不要因为喜欢一个人而变得卑微，做出很掉价的事情。因为如果你自己都不尊重自己，那么别人会更加不尊重你。

### 3. 保持边界感与空间

给自己和对方空间。空间为何如此重要？就是追一个人的边界感，一定要把握好。很多人喜欢就去追，就去送礼物，其实这个是超出了你们目前关系的。女生会反感，而不是喜欢，会觉得有压力。一定要做合情合理的事情。

### 4. 保持坦率与真诚

这点很重要。如果你觉得对方这样你不喜欢，你就直接说。很多人为了迎合对方，明明不喜欢呢，也说喜欢；明明不开心呢，说自己很开心。那这样的做法是非常错误的。你觉得是照顾别人，其实在女生的眼里，觉得你没有男子汉的气概，没有主见，很跪舔，那她自然不会喜欢你。

### 5. 接纳而非改造

如果感觉不合适，不要试图去改变。尊重对方的生活习惯、个人习惯，在可能的情况下，找到一个平衡点。不要试图去改变对方，你一定会失败。成年人的世界，只有选择，没有教育。
