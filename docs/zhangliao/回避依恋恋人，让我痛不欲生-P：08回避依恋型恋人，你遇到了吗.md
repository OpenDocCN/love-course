回避依恋型恋人在亲密关系中，常表现出矛盾的行为。他们渴望爱，却又害怕过度亲密。以下是一些典型特征。

以下是回避依恋型恋人的八个主要特点：

1.  **表达爱意模糊不清**
    他们很少直接、明确地表达喜欢。其信号通常是含混的，需要对方去猜测。只有在被逼问时，才可能给出较为明确的答案。

2.  **极度重视个人独立**
    他们享受独处。公式可表示为：**渴望亲密时长 < 需要独处时长**。即使想念伴侣，共处一段时间后也容易感到厌倦，迫切希望回归个人空间。

3.  **习惯性打压伴侣**
    这源于内心的不自信。他们越喜欢对方，越可能通过贬低对方来掩饰自己对长期关系的怀疑。若伴侣侵犯其私人空间，他们会指责对方“太粘人”或“太敏感”。

4.  **主动保持身心距离**
    当关系可能更进一步时，他们会强调“当前距离就很好”。无论在物理还是心理层面，他们都要求巨大的个人空间。代码逻辑类似于：
    ```python
    if 伴侣试图拉近距离:
        回避依恋者的反应 = “强调距离感”
    ```

5.  **严格强调关系界限**
    他们的处事原则是界限分明。“你的是你的，我的是我的”。他们对过度热情感到不适，任何超越其心理界限的要求都会引发反感。

6.  **怀有不切实际的亲密幻想**
    他们内心渴望一个“完美伴侣”：能随时提供陪伴，却永不索取，并给予他们极大的个人自由。这几乎是一个无法实现的矛盾诉求。

7.  **在感情中武断，不愿妥协**
    他们善于压抑情感，且对自己的决定非常固执。面对矛盾，常见的反应模式是：**分歧 → 逃避 → 爆发**。其中，“逃避”是最显著的表现。

8.  **不愿清晰表达真实诉求**
    他们习惯隐藏真实想法和需求，害怕暴露后会伤害彼此。因此，沟通时总是含糊其辞，不愿明确说出自己到底想要什么。
