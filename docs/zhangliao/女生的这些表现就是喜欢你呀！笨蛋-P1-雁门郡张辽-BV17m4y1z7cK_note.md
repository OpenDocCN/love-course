## 核心问题：当女生说“你不懂我”时，她在表达什么？ 🤔

上一节我们概述了课程目标，本节中我们来看看一个常见的具体问题。

一位学员反馈，他与一位女生相处融洽，关系有些暧昧，能聊天也能约会。但当他尝试推进关系时，女生情绪出现波动，并两次埋怨“你不懂我”。这让他感到困惑。

**核心概念解析**：当女生说出“你不懂我”时，其潜台词公式可概括为：
`女生埋怨 = 她在乎你 + 她希望你懂她 + 她对现状感到着急`

这意味着，女生的埋怨并非负面指责，而是一种积极的信号。她因为在乎你，并内心已默许你的男友身份，才会为你未能领会她的心意而着急。

## 问题根源：为什么关系会卡住？ 🛑

许多男生在接收到女生的好感信号后，会因为担心“是否冒昧”、“是否显得轻浮”而犹豫不决，不敢主动迈出关键一步。
