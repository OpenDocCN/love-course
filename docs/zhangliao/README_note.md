# 课程名称：张辽 📚

在本节课中，我们将要学习三国时期曹魏名将张辽的生平事迹、主要战绩与历史评价。我们将通过梳理其关键事件，来理解他为何被誉为“五子良将”之首。

## 早年经历与辗转投主 🧭

上一节我们介绍了课程概述，本节中我们来看看张辽的早期生涯。张辽字文远，雁门马邑人。他起初是并州刺史丁原的部下。丁原命张辽带兵前往京城，受大将军何进管辖。何进派张辽前往河北招募士兵。张辽招募了一千多人。何进失败后，张辽率兵归属董卓。董卓败亡后，张辽又带兵归属吕布，升任骑都尉。

## 归顺曹操与崭露头角 ⚔️

了解了张辽早期的辗转后，我们来看看他人生最重要的转折点。吕布被李傕击败，张辽跟随吕布向东逃奔徐州。吕布自称徐州牧，张辽兼任鲁国相。曹操在下邳击败吕布，张辽率领部下投降曹操。曹操任命张辽为中郎将，赐爵关内侯。张辽屡立战功，升任裨将军。

以下是张辽归曹后的主要升迁路径：
*   投降曹操，被任命为**中郎将**，赐爵关内侯。
*   因战功升任**裨将军**。
*   此后在征战中继续晋升，最终官至前将军。

## 威震逍遥津与巅峰战绩 🏆

张辽最为后世所称道的，是他在合肥之战中的表现。接下来，我们将详细解析这场经典战役。曹操征讨孙权回师后，命张辽与乐进、李典等率领七千余人驻守合肥。曹操征讨张鲁时，给护军薛悌留下一封信函。信函边缘写道：“敌军到来时再打开。”孙权率领十万大军包围合肥。众人一同打开信函。信函中写道：“如果孙权到来，张辽、李典二位将军出城迎战，乐进将军守城，护军不得参战。”

张辽解读指令后，决定主动出击以挫败敌军锐气。他连夜招募敢死队员，得到八百人。张辽杀牛设宴犒劳将士，准备明日大战。次日清晨，张辽披甲持戟，率先冲锋陷阵。他斩杀数十名敌兵和两名敌将。张辽大呼自己的姓名，冲入敌军营垒，直达孙权帅旗之下。孙权大惊，手下众将不知所措。孙权逃到一座高土丘上，用长戟自卫。张辽喝叱孙权下来接战，孙权不敢动弹。孙权军队发现张辽兵力很少，便聚众将张辽层层包围。张辽左右突围，向前猛冲，终于打开包围圈。张辽带领部下数十人冲出重围。其余部众呼喊：“将军要抛弃我们吗？”张辽于是再次冲入重围，救出其余部众。孙权人马望风披靡，无人敢抵挡张辽。

这场战斗从清晨持续到中午，吴军士气完全丧失。张辽回城后加强守备，军心得以安定。孙权围攻合肥十余日，无法攻破，只好撤军。张辽率领各军追击，几乎再次擒获孙权。曹操高度赞赏张辽的功绩，任命他为征东将军。

## 历史评价与后世地位 📜

通过逍遥津之战，我们看到了张辽的勇猛与统帅才能。最后，我们来了解他的历史地位与评价。陈寿在《三国志》中将他列为“五子良将”之首。他的军事才能与忠勇精神备受推崇。张辽病逝后，谥号为“刚侯”。其子张虎继承爵位。

---

本节课中我们一起学习了曹魏名将张辽的生平轨迹。从他的早年辗转，到归顺曹操后屡立战功，再到合肥逍遥津以少胜多、威震江东的巅峰战绩，我们看到了他作为杰出将领的勇猛、果决与统帅力。张辽的故事是三国历史中一段以勇气和战术创造奇迹的典范。
