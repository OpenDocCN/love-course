# 聊天技巧入门：如何避免尬聊并提升吸引力 🗣️💡

## 课程概述

在本节课中，我们将要学习导致聊天失败的核心原因，并掌握如何通过调整自身形象和沟通方式，来提升聊天的吸引力和愉悦感，从而建立更顺畅的人际关系。

## 为什么别人不想和你聊天？ 🤔

上一节我们概述了课程目标，本节中我们来看看聊天失败的首要原因：吸引力不足。

决定别人想不想跟你聊天的关键，是你的整体形象不够有吸引力。这类似于男生都爱跟美女聊天，女生也喜欢跟有魅力的帅哥聊天。但请注意，形象这一点不一定单指外貌。

你的形象取决于你的整体表现。例如，一个第一眼看起来是帅哥的人，如果他脏话连篇，举止粗鲁，女生也会生厌。反之，一个其貌不扬，但是风趣幽默还很智慧的人，女生和他相处一段时间后，会因为他的人格魅力而给他整体形象加分。

因此，不要因为自己的长相不够好而产生自卑心理。只要你肯把自己收拾一下，养成一个好的性格，你的形象这一关就过关了。当女生第一印象觉得一个人不怎么样时，就无法幻想和这个人在一起生活。没有幻想就没有聊天的动力，再加上如果这个人还一直缠着自己，就只想远离他。

## 如何让聊天愉快地进行下去？ 🎯

说完了别人愿意跟你聊天的前提，那么再来说说聊下去我们该怎么做。一定要明白一个道理：不管是谁，跟别人聊天最主要的目的，就是让别人跟你聊天的感觉很愉快，而不是跟你聊完了心情不爽。

女生一开始愿意跟你接触，结果过了几天就不理你了，越来越冷淡了，这是为什么呢？因为跟你聊天让她心情很不爽。

为什么女生跟你聊完了感觉心情很不爽呢？因为你让她感觉话不投机。一切的话不投机、聊得让人不爽，和你说话的内容没有直接关系，主要是因为你总是带着自己的主观情绪和想法去跟对方聊天。你的个人观念和情绪越强烈，你越无法给别人提供情绪价值。

## 核心技巧：避免观念冲突，提供情绪价值 💎

好了，重点来了。男生的逻辑思维比较强，在聊到观念类的话题时，就特别容易发表自己的看法，然后就把天聊死了。我们聊天的目的是为了给对方提供情绪价值，是为了了解对方，我们一定要改掉这个坏习惯。

聊天高手在聊天时的心态是这样的：**“你的观念或许不对，但是我不会反驳，甚至可以顺着你说。反对的意见我保留，我的目的不是为了跟你争辩，而是了解。”** 我们要有自己的立场，但是表达方式一定要比较委婉。你又不是在辩论，一切以双方聊得开心愉快为主。

你有自己的立场和主见这很好，但是有些事情会引起别人的不适，那么我们可以选择不表达出来。你跟别人的观念不一样，这就是聊天的大忌，千万不要去抬杠。

## 实用口诀：一顺二问三夸 ✨

如果还没有明白的话，就要记住一个聊天口诀：**一顺、二问、三夸，委婉表达立场**。

以下是这个口诀的具体解释：

*   **顺**：就是顺着对方去说，保留自己的意见，多说对方爱听的话。
*   **问**：就是通过适当的提问，引导对方说出自己的想法。
*   **夸**：就是当对方说到自己引以为傲的事情时，你要及时的夸赞。
*   **委婉表达立场**：这一点就很好理解了，即在必要时，以不引起冲突的方式表达自己的不同看法。

## 课程总结 🏁

本节课中，我们一起学习了导致聊天不畅的两个核心问题：**吸引力不足**和**聊天体验不愉快**。我们明白了形象是综合表现，可以通过提升外在和内在来改善。更重要的是，我们掌握了聊天的核心目的——**提供情绪价值**，并学会了通过 **“一顺二问三夸，委婉表达立场”** 的口诀来避免观念冲突，让对话变得轻松愉快。记住，聊天的艺术在于倾听、理解和共情，而非辩论与说教。
