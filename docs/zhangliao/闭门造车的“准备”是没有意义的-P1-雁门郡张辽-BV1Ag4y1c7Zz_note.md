# 闭门造车的“准备”是没有意义的 - 课程01 🚫

在本节课中，我们将要学习一个关于情感与个人成长的核心观点：脱离实践、一味等待“完美时机”的准备工作往往是无效的。我们将通过一个街头采访的案例，分析为何“永远在准备”的心态会成为建立亲密关系的障碍，并探讨更积极有效的行动方式。

## 街头采访的发现 🎤

上一节我们提到了课程的主题，本节中我们来看看一个具体的调查案例。我曾进行过一次街头采访，主要询问陌生女性一个问题：什么样的男生不受欢迎？

起初，我们和大多数人想法一致，认为原因是“不高不帅、没钱没事业”。毕竟这是常见的抱怨。

然而，在采访了许多情侣后，我们发现大部分男生的条件都很普通，真正的高富帅很少。其中一位女士的话让我印象深刻。

## 一句深刻的话 💬

以下是这位女士提出的核心观点：

*   她说，如果一个人总说“我想等到某个时候再谈恋爱”，那么这个人大概率会一直单身。
*   她认为，这样的人是“脱单困难户”。

## “准备”的陷阱与机会的本质 ⏳

开始之前，告诉大家一个小秘密：长按点赞可以一键三连。没试过的小伙伴可以试一下。

这世界上大多数机会都是短暂且稍纵即逝的。机会不会问你有没有准备好，也不会因为你优秀而特意等你。只要你稍有犹豫，它就会消失。

那些胆怯、永远都在做准备的人，常常在埋头苦干中错过一个又一个机会。他们只能对着机会远去的背影请求更多时间，但答案往往是：没有时间。

**因此，闭门造车的准备是没有意义的。**

## 情感需要在实践中学习 ❤️

在情感关系中，出现的问题只有在真正谈恋爱时才能被发现。也只有亲身经历过感情，才会知道自己哪里做得不好。

有些人一直努力想成为“完美的男朋友”，设定一系列目标：要有钱、有学历、有房有车、有幽默感。

与其说这是在“准备”，不如说是在“逃避”。他们的目标不断变化，但根本目的只有一个：弥补内心的匮乏感，让自己更喜欢自己一些。

以下是这类人常有的心态链条：

*   我要考上一所好大学。
*   我要找到一份好工作。
*   我要月入过万。
*   我要买房。
*   我要还清贷款。
*   我要有自己的事业。

他们永远对自己不满意，永远不认可自己，永远觉得自己“没有准备好”开始一段感情。这样的人，怎么可能轻松地开始恋爱呢？

## 总结与行动呼吁 🏁

本节课中我们一起学习了“过度准备”的弊端。没有人会一直等你，每个人在你身边停留的时间都是有限的。如果你抓不住那个人，就要做好永远失去他的准备。

在爱情中，输家永远是胆怯的那一方。没有人会逼你，只是看你自己想选择什么样的生活。

人生茫茫，别留遗憾。别等伤过了才追悔，别等错过了才珍惜。趁年华尚好，趁阳光不燥。毕竟，年轻时的爱和冲动，是你一生都值得追忆的美好。
