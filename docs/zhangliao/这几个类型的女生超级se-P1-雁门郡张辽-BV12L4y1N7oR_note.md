## 一、偏好网络言情小说的女性 📚

上一节我们介绍了课程概述，本节中我们来看看第一种行为模式。

以下是此类小说对读者的具体影响：

*   **氛围渲染**：作者擅长营造浪漫与暧昧的气氛，使读者沉浸其中。
*   **期待感转移**：读者最初被爱情情节吸引，但会逐渐转变为对作品中亲密、激情过程的期待。
