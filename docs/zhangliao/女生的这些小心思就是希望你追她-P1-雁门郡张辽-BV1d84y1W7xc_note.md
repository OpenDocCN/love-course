在收集女生信息的过程中，如果她积极回复，说明有戏；如果她排斥或不回应，则可以换下一个目标。

*   “今天天气真冷，降温了好冷啊。”

她选择跟你分享，而不是跟家人或闺蜜说，说明她已经习惯与你分享生活，你已正式进入她的亲密圈。这种状态持续一周左右，关系就可以向暧昧阶段推进。
