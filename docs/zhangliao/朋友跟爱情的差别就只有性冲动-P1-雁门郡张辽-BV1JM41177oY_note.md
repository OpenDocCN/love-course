## 朋友与爱情的相同点

朋友与爱情关系有许多共同之处。两者都需要双方在**人品**、**三观**上保持一致，能够愉快地聊天和相处。

上一节我们介绍了朋友与爱情的共同基础，本节中我们来看看它们的关键差异。
