# 魅《撩术》：《思维扩展篇》两性关系中不变的规律-下_note

在本节课中，我们将学习在恋爱关系中需要避免的禁忌行为，以及为了建立健康关系而必须培养的核心能力与价值体系。理解并避开这些陷阱，是提升个人吸引力和关系质量的关键。

## 禁忌一：跪舔行为

上一节我们介绍了恋爱中的一些规律，本节中我们来看看需要避免的禁忌。第一条是“跪舔”。

跪舔本身是一种**低价值**的行为。其核心在于，当你认为自己价值低于他人时，才会采取这种讨好方式。这种行为在职场上也常见，通常是价值较低者为了寻求认可而讨好领导。

然而，依靠他人永远无法获得真正的认可。问题的核心在于忽略了自我提升。正确的思维是专注于**让自己变得更有价值**，通过提升和完善自己来获得尊重。

**核心要点：**
*   跪舔是低价值行为。
*   获得认可的根本在于提升自我价值。
*   人与人本质平等，可以对他人好，但不应盲目讨好。

## 禁忌二：暴露需求感

第二条禁忌是过早或过度地“暴露需求感”。

当你遇到一个女生，表现出非常强烈的喜欢和害怕失去的感觉时，这就是需求感过强。在关系初期，双方地位越平等，你的需求感就应越少。

你需要根据对方的反馈来调整自己的行为。如果对方反馈不佳，你仍暴露强烈需求，会给对方造成压力。反之，如果对方对你有兴趣，你可以适当回应。需求感的暴露应该是**平等且平衡**的。

**核心要点：**
*   过早暴露强烈需求感会给关系带来压力。
*   需求感的暴露应基于对方的反馈，保持平衡。
*   不要在没有得到积极回应时盲目付出和认真。

## 禁忌三：缺乏自信

自信源于对自我的认可与满意。一个自信的人欣赏镜子中的自己，不会感到自卑。

缺乏自信的人往往对自己不满意，却不愿改变。能够长期忍受自己的缺陷（如肥胖、邋遢），本质上是一种不自爱的表现。这种循环会导致越来越自卑，进而不敢在人群中表达自己，显得封闭且缺乏魅力。

对女生而言，一个不自爱、不自信、没有吸引力的人是难以产生感觉的。提升自信的过程就是**完善自我**的过程。

**行动指南：**
以下是提升自信的一些方法：
*   如果肥胖，就去健身减肥。
*   如果皮肤不好，就去护肤、改善形象。
*   多看书，提升内在。
*   好好打扮，关爱并尊重自己。

只有先认可和爱自己，才可能获得他人的爱。自信是魅力的基础。

## 禁忌四：缺乏主见

缺乏主见是指没有独立思考和判断的能力，容易盲目听从他人意见。

在社会中，许多受骗上当的人都是缺乏主见、没有独立思辨能力的。他们凭感觉或他人言论来判断事物好坏，而非依据客观事实。

在两性关系中同样如此。没有自己的思想和观点，意味着你是一个没有灵魂的人。当女生问及你的爱情观或三观时，你若无法回答，就无法进行深层次交流。约会时事事询问女生意见，会让对方觉得你无法带领关系，缺乏安全感。

**核心要点：**
*   要做有独立思想的男人。
*   不盲目听信，要用经验和思考判断是非。
*   在关系中能够带领对方，提供安全感。

## 禁忌五：缺乏理性

缺乏理性表现为喜欢凭感觉行事，例如“我感觉她喜欢我”、“我感觉她是个好女孩”。

我们需要追问：这些感觉的来源是什么？是否有客观依据支撑？单凭感觉判断是不成熟、情绪化的行为，会显得幼稚且价值低。女生会认为这样的男人不具备可靠的生存能力。

在关系中，单凭感觉无法带给女生安全感。许多问题其实是自己想象出来的，例如还没搭讪就担心被打、对方有男朋友怎么办。没有经历就去批判，是不理性的。

**正确做法：**
`理性判断 = 依据经验 + 观察客观反馈（如对方的行为）`
我们需要依靠经验、社交直觉和事实依据去探索真相，而非依赖可能扭曲事实的想象。

## 禁忌六：胆小懦弱

胆小懦弱本质上是自卑的一种表现，害怕被拒绝，不敢面对失败。

这种人习惯于待在舒适区内，通过逃避来避免遭受拒绝和伤害。他们通常没有能力承担后果和责任。在女生看来，这样的男人无法提供安全感，甚至需要被保护。

**核心要点：**
*   胆小懦弱无法给予安全感。
*   必须勇于走出舒适区，通过实践和挑战来实现自我成长。

## 禁忌七：想法天真

想法天真是指思维过于单纯简单，总是从自己的主观立场出发去思考问题，不会换位思考。

人是复杂的，每个人都有独立的思想和需求。如果只活在自己的世界里，用自己的想法去揣测甚至捆绑对方，就会导致关系中出现许多摩擦和问题。

真实的两性关系需要**理解对方的想法**，让双方的观点进行碰撞与磨合，检验三观和性格是否匹配。

**核心要点：**
*   不要总是活在自己的世界。
*   学会站在对方立场思考问题，才能更好地理解与接纳对方。

## 禁忌八：偏执固执

偏执是指活在自己的世界里，不愿意接受他人的建议，固执地认为自己是正确的。

这种人格缺陷在于，其坚持的“对”往往没有丰富的经验或事实作为支撑。在做事上可以执着，但在对人的情感上不能固执。因为每个人都有自由意志，你无法强迫别人改变或爱你。

固执地追求一个不喜欢你的人，最终很容易让自己受伤。感情不是靠一味固执追求就能得到的。

## 禁忌九：自甘堕落

自甘堕落是指明知自身有缺陷，却放弃自我改进，安于糟糕的现状。

这是一种“放弃治疗”的状态。我们必须避免成为这样的人。如果发现自己有这种倾向，需要及时理解和改正，否则在长期情感关系中会引发严重问题。

## 两性关系中必备的四种能力 🛠️

了解了需要避免的禁忌后，接下来我们看看为了经营好关系，需要培养哪些核心能力。

### 1. 查研观测能力
查研观测是了解对方的重要方式。通过分析对方的言语、潜台词、回复内容和状态，来判断其真实感受和意图。

这项能力能帮助你判断对方是否对你有兴趣。如果反馈不佳，你就应该停止或减少付出，这是一种理性行为。掌握这项能力，能让你更好地了解对方，从而掌控关系。

### 2. 换位思考能力
换位思考是为了更好地读懂对方行为动机的绝佳方式。当你不理解对方为何这样做时，通过换位思考可以尝试理解其想法。

两性矛盾的根本原因常常是**缺乏理解**。拥有换位思考的能力，可以体谅、理解并接纳对方，从而轻易化解许多矛盾。在社交中，这项能力能让你拥有强大的预判力和掌控力。

`沟通顺畅 = 细致观察 + 充分理解`

### 3. 调情能力
许多男生在与女生相处时过于正经，无法刺激对方的情绪。如果女生觉得和你在一起情绪很平淡，就会觉得“没感觉”。

调情的能力直接作用于刺激女生的情绪，让她产生迷恋和你在一起的感觉。这是让女生快速对你产生好感的关键能力。缺乏情绪张力的互动，很容易让关系陷入“友谊区”。

**核心要点：**
*   需要刺激女生产生高低起伏的情绪。
*   避免让互动一直处于平淡状态。

### 4. 带领女生的能力
在关系中，女生通常需要被男生带领。你需要去主导和掌控感情的发展方向。

拥有这项能力，当真爱（真命天女）出现时，你才有能力抓住并经营好这段感情，避免因能力不足而错过或失去机会。提升能力，是为了当幸福来敲门时，你能有能力开门迎接。

## 吸引力的核心：三大价值体系 💎

那么，女生究竟为什么会被我们吸引？其核心在于以下三大价值体系。

### 1. 生存价值
生存价值指你能为生活提供的保障和安全。对女生而言，这意味着选择你，她及其后代能拥有更稳定、安全的生活环境。

在**长期关系**中，生存价值（如经济能力、责任感）至关重要。它是安全感的基础。短期关系则对此要求较低。

`生存价值 ≈ 提供安全感与生活保障的能力`

### 2. 人格魅力
人格魅力是你内在特质的外在体现，包括性格、谈吐、气质、自信、为人处事等。

例如，在台上侃侃而谈的自信，诚实可靠的品质，都构成强大的人格魅力。这是吸引他人的内在闪光点。

### 3. 繁衍价值
繁衍价值主要指外在的吸引力，如英俊的相貌、好的身材。

从进化角度看，更好的外在条件意味着后代可能拥有更优的基因和社交优势。这是最直观的吸引力来源。

**价值吸引定律：**
你主要凭借哪种价值吸引对方，就会吸引来更看重该价值的群体。
*   展示生存价值 → 吸引看重物质保障的女生。
*   展示人格魅力 → 吸引看重内在思想的女生。
*   展示繁衍价值 → 吸引看重外表的女生。

因此，我们需要**同步提升**这三项价值。综合价值越高，吸引女生的层次和范围就越广。

## 关键补充：刺激情绪的能力 🎭

除了提升价值，还需要掌握一项关键能力：**刺激女生情绪的能力**。

让女生觉得和你在一起开心、浪漫、有各种不同的情绪体验，能极大增强她的幸福感。情绪体验越丰富，她对你的感觉就越好。

`情感浓度 ∝ 情绪体验的多样性`
学习恋爱学的最终目标，不仅是自我提升，更是掌握制造情绪和感觉的能力，从而实现在任何关系中都能吸引对方、带领关系，最终达到**情感自由**的境界。
