## 一、 思维转变：从“术”到“道”

上一节我们介绍了课程的主题，本节中我们来看看核心的思维转变。

从前我们可能会听很多撩妹高手讲，刚加上女生要去看对方的朋友圈再开启话题，或者要去拉伸关系，或者发个表情。这些方法都停留在“术”的层面，即停留在表面。

所有表面的东西，都有深层次的逻辑关系和理论支撑。很多兄弟发现，同样一句话或方法，别人用能成功，自己用却会失败。这是因为很多学员没有理解这些方式方法背后支撑的本质。

当我们不理解一件东西的本质，只理解它表面的东西时，很多时候是用不好这个方法的。我们今天要讲的，就是刚加上女生时，应该如何互动、如何聊天的底层本质。

## 二、 核心认知：吸引的本质与价值评估

上一节我们强调了理解本质的重要性，本节中我们来深入探讨吸引的底层逻辑。

首先要知道，如果两个人分值差异太大，例如对方是八分女，你是四分男，你很多时候是吸引不到她的。

吸引的本质一定是**高价值吸引低价值**，或者**平行吸引**。即在一个价值层面上才能产生吸引。就像你很难吸引范冰冰，因为你们不在一个价值层面。

我们首先要提高对自己的认知：你的条件、你的社交圈子。也要了解对方的条件、对方的社交圈子。例如，你是一个月薪2000的工厂打工者，想追富家千金基本不可能，因为你们的社交圈子和条件差异太大，你身上可能完全没有对方需要的价值点。

了解这个认知很重要，因为你会发现用很多方法、话术都容易聊死。这可能是因为用错了策略。策略可以提高聊天互动的精准度。用对策略，聊10个可能成功七八个；如果只停留在话术层面，聊100个可能只走运成功三四个。因此，我们必须着重理解策略的重要性。

对自我和他人的认知，需要不带主观批判。很多会员会用主观感觉去评判别人。例如，感觉一个人不错，就会把他的各方面分值拉得很高。你喜欢一个女孩子却不知道喜欢她什么，只觉得喜欢。这样在一起后很容易腻，分手时也完全不知道为什么。这样的感情经历没有营养，无法带来成长。

每一段感情都应该让我们学到东西。因此，在认识女生或与女生互动前，一定要理性地去观察、评判双方，分析两个人之间的差距。

观察对方时，一定要通过行为去分析数据。因为很多女生会伪装，可能分值不高但会把自己伪装得很高，例如通过修图软件。我们要通过行为去观察分析。

吸引的底层逻辑和原则是：吸引一定是高价值吸引低价值，或者是平行吸引。学习恋爱学或与人交往，你最多只能吸引比你分值高两个层级的人。例如，如果你是五分男，最多只能吸引到七分女。八分到九分的女生，你根本接触不到她们的圈子，更别谈吸引。

还有一个重要概念是**价值**与**价值感**的区别。你本身的价值和你带给别人的价值感，是两个不同的概念。可能你本身是八分男生，但你带给别人的价值感只有五分，那你在别人眼里就是五分的男生。这就是很多学员本身价值高，却撩不到女生的原因。

这中间存在一个变量，叫做**链接价值**。链接价值乘以你本身的价值等于价值感。这是一套公式：

**价值感 = 链接价值 × 本身价值**

链接价值简单理解，就是我们学习恋爱学的目的，即如何展示我们的价值。例如，我的链接价值是5，本身价值是6，那么我的价值感得分是30。如果你的链接价值是1，本身价值是9，那么你的价值感只有9分。如果我们是情敌，你很可能竞争不过我。这是一套需要理解的公式。

## 三、 价值评判的六个要素与算法

上一节我们引入了价值和价值感的概念，本节中我们来看看具体如何量化评估。

价值评判的六个要素如下：
1.  身材
2.  长相
3.  品味时尚度
4.  男性社交认证（你的圈子，你是否是雄性领袖）
5.  女性社交认证（身边女性朋友对你的认可度）
6.  生存价值（车子、房子、存款等）

男女双方对价值评判的侧重点不同。

**男生看女生**的要素重要性排序如下：
1.  身材
2.  长相
3.  时尚度/品味
4.  男性社交认证（周围男生对她的评价）
5.  女性社交认证（周围女生对她的评价）
6.  生存价值

要评判一个女生的分数，可以按以下方法计算：
为每个要素打分（满分10分），然后将六个分数相加，再除以6，得到她的平均价值分数。例如：
长相8分，时尚度7分，男性社交认证6分，女性社交认证7分，生存价值5分，身材9分。
总分 = 8 + 7 + 6 + 7 + 5 + 9 = 42
平均分 = 42 / 6 = 7分

**女生看男生**的要素重要性排序如下：
1.  生存价值
2.  预选价值（女性社交认证）
3.  雄性领袖价值（男性社交认证）
4.  时尚度/品味
5.  长相
6.  身材

男生评判自己的分数，算法同上。需要为自己每个要素打分。例如，生存价值：月入3000可能算1分，月入100万可能算10分。女性朋友数量：有5个可能算1分，有50个可能算10分。通常，男生很少能达到四五分，能达到六七分已经是非常优质的男生。

我们必须对自己和他人有一个理性的认知。

## 四、 策略应用：根据价值差选择聊天方式

上一节我们学会了如何评估分数，本节中我们来看看如何根据分数差来制定具体的互动策略。

分析完自己与对方的分数后，采取相应策略。

**情况一：你的价值比女生高（例如，你是7分，她是5分）**

策略是：**先拉后推**。
1.  **先拉（表达兴趣）**：多聊她，表达对她的兴趣。例如，“感觉你挺不错的”、“感觉你挺优秀的”。多建立可得性。在男高女低的状态下，可以表达自己的可得性。这就是为什么有些导师一上来就能和女生聊得很深入，甚至聊私密话题，因为他的分数足以碾压对方。
2.  **输出三观（建立深层次链接）**：这时女生可能会有自卑心理，觉得你是不是玩玩而已。因此，你不仅要表达可得性，还要表达自己的情感经历、感情观、价值观，与对方建立深层次链接。这是在私教课中重点讲解的内容。
3.  **后推（抛出废测，进行筛选）**：聊完深入话题后，需要抛出废测推开她。例如，“你是不是对所有男孩子都这样？”、“你会做饭吗？”、“你平时对待男朋友的态度是什么样的？”、“你会乱发脾气吗？”。这很像女生平时对男生的测试：先主动撩你，表达兴趣，然后当你投入后，她反而后撤。
4.  **压迫表白**：筛选过后，可以引导对方自我暗示。例如问“你喜欢我什么？”，对方讲出喜欢你成熟稳重等特质后，基本就可以确定关系了。

这套流程适用于你的价值比女生高的情况。

**情况二：你的价值与女生相当或略低（例如，都是5分）**

策略是：**先推后拉**。
1.  **先推（侧面展示价值，多聊自己）**：不要暴露需求感。多聊自己，侧面展示价值。例如，对方说“你也喜欢健身啊？”，你可以回应“我平时也健身”，这句话就带出了自己的价值。这个阶段属于“推”，即不暴露需求感，只展示价值。
2.  **收集兴趣指标**：在互动中观察女生对你的兴趣信号。
3.  **后拉（暴露可得性）**：当兴趣指标足够后，可以进行“拉”。例如说“感觉咱们还蛮搭的”、“感觉你挺适合做媳妇的”。这时可以暴露一些可得性。
4.  **建立深层次链接**：接着，像情况一那样，进行感情观、三观等深层次链接。
5.  **筛选与表白**：最后进行筛选和表白。

这两个流程的关键在于，必须先计算分数。刚认识一个女生，应先通过朋友圈等信息，根据她的身材、长相、时尚度、社交认证、生存价值来评判她是几分。我们大部分兄弟评判女生都靠感觉，达不到理性状态。

感觉经常会骗人，这是因为**光环效应**：你觉得一个人某个点好时，这个优点会像一个光环，盖住他所有的缺点。你看到的都是优点，但实际上缺点被掩盖了。因此，必须理性分析，才能更真实地看待彼此关系，决定如何发展。

## 五、 课程总结与重点回顾

本节课中我们一起学习了与刚认识女生互动的核心思维与策略。

首先，我们讲了应有的思维：
1.  **对自我的认知**：清楚自己的条件、圈子在哪里。如果你的圈子在工厂，你的目标就应该是周围如隔壁厂、奶茶店的女生。就像卖产品要找精准客户。
2.  **对对方的认知**：理性评估对方的价值。在四分时偏要追求八分女生，是在浪费时间。
3.  **策略差异性的重要性**：用错策略必死。很多兄弟话术没错却失败，往往是因为策略错了。学习这些的目的是提高互动的精准度。
4.  **客观的价值评判**：不能带主观性，要用第三视角，通过行为和观察来分析数据。

其次，我们学习了吸引的底层逻辑：你本身的价值和你带给别人的价值感中间存在一个变量——**链接价值**。并非你价值高就不需要展示。公式是：**价值感 = 链接价值 × 本身价值**。一定是价值高且会展示，给他人的价值感才足。

接着，我们掌握了价值评判的六个要素及男女不同的重要性排序，并学会了打分计算方法。大家需要记下来，养成习惯，在认识女生时为她打分。

最后，我们学习了根据价值差选择策略：
*   **你比女生分值高**：吸引力已足，策略是**先拉后推**。表达可得性，给予对方信心，再走心，最后压迫表白。
*   **你比女生分值低或相当**：策略是**先推后拉**。多展示自己价值（推），收集兴趣指标后再暴露可得性（拉）。如果分值低时一开始就表达兴趣（如“美女你好漂亮”），会因为缺乏吸引力而失败。
