## 爱情光谱理论

爱情光谱理论是一种用于分析人际关系和情感互动的理论。它将人际关系分为七个层次，从陌生关系到亲密关系。

### 七个层次

1. **陌生关系**：彼此不认识，没有交集。
2. **熟悉关系**：认识但无深交。
3. **朋友关系**：有共同兴趣，相互了解。
4. **恋人关系**：情感亲密，有恋爱关系。
5. **伴侣关系**：生活伴侣，共同生活。
6. **亲密关系**：情感深度交流，心灵相通。
7. **灵魂伴侣**：精神契合，共同成长。

### 公式

```
爱情光谱 = 陌生关系 + 熟悉关系 + 朋友关系 + 恋人关系 + 伴侣关系 + 亲密关系 + 灵魂伴侣
```

4. **沟通方式**：女性更倾向于细腻、委婉的沟通方式。
