# 雨哥同学会：雨哥干货课1-20集：19《为什么你一直学不会聊天》_note

在本节课中，我们将要探讨一个核心问题：为什么很多人在学习聊天技巧后，反而感觉更不会聊天了。我们将深入分析心态变化、学习误区，并提供清晰的行动指南，帮助你找回方向，实现有效沟通。

## 心态转变：从单纯到迷茫

上一节我们介绍了聊天的基础，本节中我们来看看心态如何影响你的学习进程。

很多兄弟在接触我的课程或案例后，反馈说感觉自己“越学越不会聊了”。这并非你的聊天能力退步，而是你的心态发生了巨大变化。

在遇到我之前，你就像山村里的单纯孩子，聊天全凭感觉，没有好坏标准。但进入我的圈子后，你有了衡量聊天好坏的标准。很多兄弟忘记了学习的初心。起初可能是为了追到喜欢的女生，但后来，通过接触不同的爱情观，可能会觉得某些女生不值得去爱，胃口变大了，但能力未能同步提升，从而产生焦虑。

这就像我带你从宁静山村进入繁华都市，灯红酒绿令人沉迷，但你一摸口袋，钱却不够。这种落差导致焦虑，让你误以为是聊天能力不足。实际上，这是心态发生了变化，你已不再是那个单纯的孩子。

## 初心与选择：爱情与前途

以下是关于如何在感情中做出选择的核心观点。

对于30岁以下的兄弟，建议选择你喜欢的；对于30岁以上的兄弟，建议选择喜欢你的。前者需要冒险精神和主动争取，后者则只需接受。年轻时有时间、精力和金钱去追求喜欢的，应将精力放在对的事情上。

关于爱情与前途哪个更重要，多数人的答案是前途。但一段正确的爱情不会影响你的前途，只会照亮你的前路。看待问题不应极端，需多角度思考。真正的爱情对前途有益。人生是一场修行，爱情是其中必不可少的部分。

永远有更好的选择，但若总想着下一座山更高，便难有成就。眼下的才是最好的。如果遇到合适的女生——非渣女、非物质女，不嫌弃你当前的经济状况，颜值过得去——便应安心相处。这对你的未来有积极影响。当然，若自身条件优越，可选择权更大，但对大多数普通兄弟而言，不应被某些观念带偏。

很多兄弟学习的初心就是追到喜欢的女生。实际上，你们若真想脱单，比普通人要容易得多。在遇到喜欢的女生之前，可以广泛接触；但遇到之后，应以她为中心进行交流。

## 学习误区：避免“中毒”与盲目参照

上一部分我们讨论了选择，现在我们来剖析学习过程中常见的几个心态误区。

以下是三个需要避免的核心误区：

1.  **不要以我为参照。** 这是我的工作，你们的情况与我不同。玩可以，但遇到真正喜欢的女生时，懂得“收心”很重要。
2.  **不要“中毒”。** 不要沉迷于所谓的“PUA”（搭讪艺术）或各种专业术语（如IOI、高价值展示）。高情商与PUA是两回事。学习某些速推技巧就像吸毒，只有零次和无数次，难以收手，对你有害无益。
3.  **你不需要做到完美。** 不必让每个女生都喜欢你。举例说明：一个人相亲，为迎合不同对象去学做饭、买车买房，最后因长相被拒。他不可能为此去整容。这说明，不是每个女生都适合你。**永远有人喜欢不完美的你。** 你无需为女生改变所有缺点，而应尽可能放大自身优势。

## 提升之道：为自己而改变

认识到误区后，我们来看看如何正向提升自己。

学习聊天或情商，不应是为了某个特定的人。为了取悦他人而学，会带来巨大压力且难以持久。真正有效的方式是为自己积累和扎根，根扎得越深，未来绽放得越美丽。

女生是很好的学习对象。优秀的女生从不为了男人改变自己，她们永远以最佳状态面对世界，因而拥有众多追求者。你的目标不应定得过于远大。要为自己变得优秀，并设定每日小目标。

以下是变得优秀的两大切入点：

1.  **从外在形象开始改变。** 这是打造自信的第一步。型男与邋遢男，气质截然不同。穿搭不需昂贵，但需合宜。找到适合自己的风格，能极大提升自信和气场。当你对镜中的自己感到满意时，自然会成为一个有态度的人。
2.  **调整内在心态。** 再次强调，前途通常比爱情重要。当你没有遇到爱情时，选择前途；当必须二选一时，真正的爱情不会让你为难，它会对你的前途有利。同时，要学会主动争取，也学会主动放弃。不要害怕错过，过去的就是回忆。

## 答疑与核心原则

最后，我们针对一些具体问题，提炼出普适的应对原则。

**如何防止被“PUA”或道德绑架？**
保持一个核心心态：**我不吃亏**。坚持等价交换原则。例如，女生要求发红包，你可以答应，但同时要提出你的要求（当然需合理合法）。只要秉持“不吃亏”原则，就很难被操控。

**什么是聊天中的“框架”？如何建立？**
“框架”本质上是心理博弈，指一段对话中谁在主导话题走向。查看你的聊天记录，看每段对话是围绕对方、你自己还是某个话题展开，而话题又是谁引出的。建立框架需要有完整的聊天思路，知道每句话的目的，是为了铺垫还是引导新话题。目前你们框架弱，主要是因为思路不完整，聊到哪算哪。按照教授的方法：开场、调动情绪、拉伸关系、展示价值，逐步练习即可。

**总是被打击，如何建立自信？**
每天早晨对着镜子自我肯定：“我很优秀，我很帅，我很棒。”这种持续的积极暗示潜移默化中会提升自信。同时要明白，有人讨厌你，恰恰说明你已具备一定影响力。

本节课中我们一起学习了心态对聊天学习的深远影响。我们剖析了从单纯到焦虑的心态转变，明确了爱情与前途的关系，指出了“以我为参照”、“中毒”、追求完美三大误区，并给出了从外在形象和内在心态入手提升自我的具体路径。记住，聊天是技能，心态是根基。不要迷失，坚持为自己而改变，你会在正确的道路上越走越稳。

**总结今日关于心态的四个要点：**
1.  不要以我为参照。
2.  不要沉迷“PUA”等技巧而“中毒”。
3.  你不需要完美，合适就好。
4.  努力为自己变优秀，从外在形象着手。
