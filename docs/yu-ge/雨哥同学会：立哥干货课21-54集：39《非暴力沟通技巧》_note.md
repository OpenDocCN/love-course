## 什么是暴力沟通？

有会员反馈说，课程中实例较少，因为我们主要讲授干货和知识点。

课程内容更偏向技巧性，需要你们去内化与吸收。在学习过程中，如果有任何不理解的地方，可以提问，与我沟通，以完成内化吸收。

因此，课程案例会相对少一些。那么，我们回到今天的主题：什么是暴力沟通？

暴力沟通主要包括以下几种形式。

### 1. 道德评判

道德评判是指，当一个人的行为不符合我们的价值观时，我们就认为他是不道德的。

例如，你的女朋友可能只是因为心情不好，不想说话，并非针对你。但你可能因此站到道德制高点，认为“她又开始装模作样了”、“你就是在破坏感情”、“你不爱我”。

再比如，有些人在地铁上看到小孩吵闹，会直接认为“没素质”。但你可能不了解，孩子或许遇到了难过的事情。人性常常如此，人们喜欢站在道德制高点评价他人，且常常“双标”——自己可以犯错，却不允许别人犯错。

另一个例子是网络暴力，比如在明星捐款时，指责“你赚那么多，才捐这么点”。这就是典型的道德评判，评判者自身可能并未付出，却用高标准要求别人。

**核心公式：暴力沟通 ≈ 用自己的价值观评判他人 ≠ 描述客观行为**

### 2. 进行比较

我相信很多兄弟和立哥一样，小时候常听到“别人家的孩子”。女朋友也可能说“我闺蜜的男朋友如何如何”。

这种比较像刀子一样戳在心里。所以，千万不要拿别人作比较。

今天我想从原生家庭讲起，分享一些干货，帮助大家提高情商。会员课是一种筛选，设立门槛是为了找到真正愿意提升自我的人。

我们上一辈的父母，他们所处的环境与传统和现在不同，许多人并未准备好如何教育孩子。他们可能沿用了“棍棒底下出孝子”的暴力方式。

立哥讲授非暴力沟通，乃至后期的长期关系课程，都是希望你们成为更好的人。这不仅关乎脱单，更关乎让我们未来的下一代更加幸福。

上一辈人不太懂得鼓励教育，我们从小受到的打击多于鼓励。父母虽好，但有时不会换位思考，还会拿我们与他人比较。

前阵子有个新闻，一个初三男孩被母亲在走廊扇了耳光后，跳楼身亡。我作为教育行业出身的人，看到后非常心痛。这种暴力沟通的方式，代价是惨重的。

在恋爱中，也不要拿女友与他人比较。比如“你看我兄弟的女朋友多听话”。我们本意是希望对方更好，但暴力比较的方式只会适得其反。

### 3. 回避责任

回避责任是指，当我们做不到某事或失败时，将责任推卸给他人。

例如，事情搞砸后说“要不是你，我会这样吗？”；或家长对孩子说“要不是因为你，我早离婚了”、“我这么辛苦都是为了你”。

这种话淡化了个人责任。但我们必须意识到，所有事情都是你自己的选择。除了你自己，没人能逼你做任何事。即使被胁迫，在“服从”与“承担后果”之间，你依然做出了选择。

**核心代码（思维框架）：**
```python
if situation_happens:
    # 你永远拥有选择如何应对的权利
    my_response = my_choice(situation)
    # 而非：my_response = forced_by(someone_else)
```

就像有人选择嫁给富豪，就要承担相应的家务；你选择不提升自己，就要面对可能单身的结果。一切都是你选择的代价与取舍。

### 4. 进行威胁

你们是否听过这样的话：“再不听话，妈妈就不要你了”、“再不回家就离婚”、“不买就分手”。

听到这种话，心里会非常难受。这就是典型的暴力沟通——通过威胁来迫使对方服从。

以上四种：道德评判、进行比较、回避责任、进行威胁，都是暴力沟通，大家要避免使用。

## 如何进行非暴力沟通？

上一节我们识别了四种暴力沟通模式，本节我们来看看如何用非暴力沟通来替代它们。以下是三个核心技巧。

### 技巧一：区分观察（事实）与评论（观点）

这是我们与人相处时最容易犯的错误：分不清事实与观点。

**定义：**
*   **事实（观察）：** 客观发生的事件，像摄像机记录下的画面，包含具体的时间、地点、人物、行为。
*   **观点（评论）：** 个人对事件的解读、评价或猜测，常包含模糊的形容词。

**举例分析：**
*   对方迟到30分钟。
    *   暴力沟通（观点）：“你怎么迟到**这么久**？”（“久”是主观感受）
    *   非暴力沟通（事实）：“我们约好10点，现在是10点30分，你迟到了30分钟。”
*   对方连续一周晚归。
    *   暴力沟通（观点）：“你怎么**天天**这么晚回来？”（“天天”可能被反驳）
    *   非暴力沟通（事实）：“这周你有五天都在晚上11点后回家。”

**练习方法：** 像写合同或播报天气预报一样说话，只说具体的、可验证的信息。例如，不说“风很大”，而说“风力7级”。

### 技巧二：表达感受，而非评判或猜测

表达感受是建立情感连接的关键，但很多人混淆了“感受”与“想法/猜测”。

**感受词汇表：** 开心、兴奋、感动、害怕、焦虑、伤心、孤单、生气、紧张、失望、内疚。
**评判/猜测词汇表：** 我觉得你不爱我了 / 我觉得你忽略了我 / 你这个人真差劲。

**对比示例：**
*   **暴力沟通（评判/猜测）：** “你天天不回家，是不是外面有人了？你不爱我了！”
*   **非暴力沟通（事实+感受）：** “亲爱的，你这周有五天晚上11点后才回家（**事实**）。我感到很孤单，也有点担心我们的关系（**感受**）。”

如果父母能说“这次成绩比上次低了10分，妈妈有点担心”，而不是“你就不是学习的料”，孩子会更愿意自省和努力。

### 技巧三：明确并表达自己的需求

所有负面情绪的背后，都隐藏着一个未被满足的需求。暴力沟通发泄情绪，而非暴力沟通则探寻并表达需求。

**核心逻辑：** 情绪（生气/伤心） → 源于 → 需求未被满足。
**关键转折：** 将“指责对方”转化为“明确自己的需求”。

**对比示例：**
*   **暴力沟通（发泄情绪）：** “你回来这么晚，死外面算了！”（背后需求：希望对方早点回家陪伴）
*   **非暴力沟通（表达需求）：** “你这周都很晚回来（**事实**）。我感到有些失落（**感受**），因为我希望我们能有更多相处的时间（**需求**）。我们下周可以安排两次一起吃饭吗？（**请求**）”

**直接提出请求：** “亲爱的，我希望我们每周能至少约会一次，来增进感情。” 清晰直接地提出需求，对方更容易理解和配合。

## 总结与升华

本节课中，我们一起学习了非暴力沟通的核心技巧。

首先，我们识别了四种**暴力沟通**模式：道德评判、进行比较、回避责任和进行威胁。这些模式会伤害关系，阻碍沟通。

接着，我们掌握了三个**非暴力沟通**的关键步骤：
1.  **区分观察与评论**：只说具体事实，不用模糊形容词评判。
2.  **表达感受**：使用情绪词汇坦诚表达感受，而非指责或猜测。
3.  **明确需求**：觉察情绪背后的真实需求，并清晰、平和地提出请求。

非暴力沟通的本质，就是 **“好好说话”** 。它不仅仅适用于恋爱，更能改善你与家人、朋友、同事乃至未来子女的相处方式。暴力沟通可能影响他人一生，而我们学习这些，正是为了打破伤害的循环，建立更健康、更温暖的人际关系。

我们都可能受过伤害，但可以通过学习与自省，成为更好的自己，并让下一代拥有更幸福的成长环境。这就是立哥分享这些内容的初衷。
