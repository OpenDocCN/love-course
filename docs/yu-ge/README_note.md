# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

其核心操作是赋值，公式如下：
`变量名 = 值`

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“Hello World”存储在其中。
```python
message = "Hello World"
```

## 基本数据类型 🔢

理解了如何存放数据后，我们来看看Python中常见的数据有哪些类型。Python会自动识别你放入“盒子”里的数据是什么类型。

以下是几种最基础的数据类型：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由引号包围的文本，例如 `"你好"`, `'Python'`。
*   **布尔值**：只有两种值，`True`（真）或 `False`（假），用于逻辑判断。

## 如何查看和使用类型？ 🔍

有时我们需要明确知道变量里数据的类型，这时可以使用 `type()` 函数。

例如：
```python
age = 25
print(type(age))  # 输出：<class 'int'>
```
这段代码会告诉我们变量 `age` 中存储的是一个整数。

## 一个简单的实践示例 ✨

让我们把以上概念组合起来，写几行简单的代码。

以下是代码示例和分步解释：

1.  创建一个字符串变量来存储名字。
    ```python
    name = "雨哥"
    ```
2.  创建一个整数变量来存储年龄。
    ```python
    age = 30
    ```
3.  使用 `print` 函数将它们打印出来，并组合成一句话。
    ```python
    print(name + "今年" + str(age) + "岁。")
    ```
    注意：`str(age)` 是将整数 `age` 转换为字符串，因为只有相同类型才能用 `+` 连接。

运行这段代码，你将在屏幕上看到：“雨哥今年30岁。”
