# 雨哥同学会：立哥干货课21-54集：25《吵架之后如何才能增进感情》_note

在本节课中，我们将要学习如何在情侣吵架之后，不仅修复关系，还能让感情更进一步。吵架是感情的试金石，关键在于我们如何处理冲突。我们将探讨一系列原则、技巧和具体步骤，帮助你掌握“有效吵架”的艺术，将矛盾转化为增进感情的契机。

## 核心原则：吵架中的绝对禁忌🚫

上一节我们介绍了课程目标，本节中我们来看看吵架时必须坚守的底线。如果你在争吵中做了以下三件事，将严重伤害感情，甚至导致关系破裂。

以下是吵架时千万不能做的三件事：

1.  **人身攻击**：这是最严重的禁忌。人身攻击指辱骂对方本人或其家人，例如“你是个臭婊子”、“我操你奶奶”。这种语言将对方物化，彻底摧毁了爱的根基。吵架应围绕“爱”进行（例如“你一点都不爱我”），其背后仍有爱的存在。一旦进行人身攻击，爱便瞬间消散。
2.  **动手**：无论多么愤怒，绝不能动手打人。家暴是无法挽回的。即使控制不住情绪，可以摔不值钱的物品（如抱枕、苹果），但绝不能伤害对方。君子应“反求诸己”，保持风度，不因对方的行为而降低自己的品格。
3.  **牵扯家人**：吵架应对事不对人，更不应攻击对方家人。例如“你妈离婚嫁了三个男人才教出你这样的女儿”。许多人能接受你批评他，但绝不能接受你侮辱他的家人。

## 核心技巧：控制情绪与沟通公式🧠

明白了哪些事不能做之后，我们来看看应该怎么做。这一切的基础在于**控制情绪**。情绪并非不可控，它源于大脑的判断。在生气前，大脑会理性评估“现在是否可以发飙”。例如，在公共场合或重要人物面前，你通常能忍住怒火。这说明，**你可以有意识地打断情绪的爆发**。

以下是控制情绪和有效沟通的核心方法：

*   **技巧一：主动暂停**。当你感觉即将失控时，可以主动离开现场，去厨房深呼吸，告诉自己“现在不想说了，回头再谈”。愤怒时尽量少说话，因为说出去的话无法收回。
*   **技巧二：温柔开场，妥协收场**。这是增进感情的关键沟通公式。
    *   **温柔开场**：不以指责开始，而以询问和理解开始。
        *   错误示范（指责开场）：“你他妈说我不爱你？那你爱过我吗？”
        *   正确示范（温柔开场）：“你觉得我不爱你，我很难过。能告诉我为什么你会这样认为吗？是我哪里做得不够好？”
    *   **中间过程：提出共同愿景**。在沟通中，将双方拉回同一战线，例如：“我知道你很生气，但我们吵架是为了让感情更好，而不是更糟，对吧？”
    *   **妥协收场**：提出解决方案，各退一步。例如：“我了解了你的想法。我以后会注意……那你是否也可以……？”

## 实战应用：吵架后的修复步骤🛠️

掌握了核心技巧，我们将其应用到具体场景中。吵架后往往会进入冷战期，这是修复关系的关键窗口。

以下是度过冷战期并增进感情的具体步骤：

1.  **判断是否主动**：首先判断吵架原因是否涉及**原则性问题**。
    *   **非原则问题**（如约会迟到、忘记承诺）：你应该主动破冰。
    *   **原则问题**（如对方索要贵重财物、辱骂你家人、与异性暧昧且拒不认错）：不应主动，必须等对方认错，否则考虑结束关系。
2.  **主动沟通，运用公式**：如果决定主动，就使用“温柔开场，妥协收场”的公式进行沟通。
3.  **设立未来愿景与规矩**：和好后，共同设立未来规则，防止同类争吵。例如：“以后吵架，谁也不准离家出走”、“冷战不许超过一天，谁超时谁就送对方一个小礼物”。
4.  **进行深度道歉与和解**：这是感情升温的关键。找一个安静的时刻（如睡前），进行一次结构化的深度交流。框架是：**互相道歉 + 理解对方感受 + 展望未来**。
    *   **道歉**：“亲爱的，我要为那天凶了你道歉。”
    *   **理解感受**：“我仔细想了，我那样说肯定让你特别伤心和没有安全感，我知道你心里很委屈。”
    *   **展望未来**：“我以后会努力更理解你，不再说那种伤人的话了。你愿意也和我聊聊你的想法吗？”
    *   引导对方也道歉，如果对方拒绝沟通或拒不认错，则需要重新评估这段关系。

## 思维升华：高情商沟通的普适价值🌟

本节课的技巧不仅适用于亲密关系，更是一种高情商的沟通思维，适用于所有人际交往。

其核心是：**摒弃暴力沟通，追求非暴力沟通**。很多人习惯于情绪对抗，只求别人理解自己，却从不尝试理解别人。真正的成熟在于，**你希望别人怎么对待你，你就怎么去对待别人**。

例如，在职场中，批评下属时：
*   低情商：“你怎么天天迟到？干活不过脑子吗？”
*   高情商：“你一直很负责，交给你的任务总能完成得很好。同时，我希望你能更主动一些，提出自己的想法。当你不仅能执行，还能创造时，你的发展空间会更大。”

当你学会用理解代替指责，用建设性方案代替情绪发泄，你不仅能经营好感情，更能赢得所有人的尊重，让事业和生活都更加顺利。
