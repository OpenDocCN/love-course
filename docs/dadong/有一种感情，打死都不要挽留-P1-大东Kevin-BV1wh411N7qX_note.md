## 🔍 识别消耗型关系

上一节我们介绍了课程主题，本节中我们来看看这种关系的具体表现。

有一种感情你真的不要去挽留。即便你现在非常喜欢、非常舍不得，也请先冷静下来回忆。

以下是你们在一起时可能出现的特征：
*   你感到特别内耗，情绪持续被消耗。
*   你的经济状况可能受到影响，资源被消耗。
*   你的感情不断付出，但得不到对等的回馈。

在这种关系里，对方从来不会感恩你的付出。

## 🧊 冷漠与指责的循环

了解了关系的消耗性特征后，我们进一步探讨其互动模式。

更别提对方会懂你的生活、理解你。因为他从来就不会照顾你的情绪。

以下是矛盾发生时的典型模式：
*   你们之间只要有矛盾争吵，他就会用冷暴力来威胁你。
*   他从来不觉得是自己的错。
*   如果你等待他来跟你低头认错，这**绝对不可能**。

## ⚖️ 失衡的权力与责任

上一节我们看到了矛盾如何被处理，本节中我们来分析这种模式形成的根源以及结果。

因为你以前一次次的妥协和退让，让他知道自己吃定你了。其核心逻辑可以概括为一个简单的公式：**你的退让 → 他的确信 → 你的更投入**。

哪怕最后你们分手了，他也会把所有责任都推给你。

所以，这种只有你一个人在爱、一个人在维持的感情，请务必放手。这等同于在代码中执行一个退出循环的命令，以释放被占用的资源：
```python
if relationship == “one_sided_maintenance”:
    let_go() # 执行“放手”函数
    set_free() # 为自己解锁
```
