# 情感关系基础课 01：记住这五点，避免陷入卑微 🚫🐕

在本节课中，我们将学习建立健康情感关系的五个核心要点。这些原则旨在帮助你保持自我，避免在关系中过度付出而失去平衡，从而走向更平等、相互尊重的互动模式。

## 第一点：尽量保持冷静，避免过度上头

上一节我们介绍了课程概述，本节中我们来看看第一个要点：控制情感投入的程度。

我们不是机器人，无法完全避免对某人产生强烈好感。但你需要思考一个问题：一方越上头，另一方也会同样上头吗？实际上，另一方只会因此变得越来越清醒。所以，你必须控制好自己的需求感。

当你给女生发送一大串对话或小作文时，你注定是更上头的一方，而对方只会更加冷静。原因在于，你的行为在外人看来已经不正常了，而你自己由于上头却并未察觉。

## 第二点：情绪稳定是赢得在乎的关键

理解了控制需求感的重要性后，我们接下来探讨情绪稳定的价值。

为什么大部分人在恋爱中都有“患得患失”的心态？说到底，你们谈的就不是一场平等的恋爱。对方不像你那么喜欢，所以一开始的时候，不要迫不及待地展示自己的全部真心。那样做一点用都没有，那只是在自我感动。

## 第三点：不要过度在意对方而失去自我

我们知道了稳定情绪的重要性，现在来明确“在意”的合理边界。

怎么做才算是“过度在意”呢？其实很简单：你因为在意对方，而放弃了一些自己的原则，并不断推后自己的底线。一旦你进入这种循环，对方对你肯定会越来越冷淡。

你只需要记住一点：好的感情一定是一种“有来有回”、棋逢对手的状态。

## 第四点：爱自己是获得被爱的前提

在设定了个人边界之后，我们需要建立更核心的认知：自我关爱。

无论是谈恋爱还是追求女生，最好的状态就是你**没有委屈自己**，但是你们的感情却一直在向上发展。所以，你要先学会爱自己，不然没有人会重视你。

## 第五点：筛选而非改造，是关系的核心逻辑

最后，也是最重要的一点，关乎关系的根本模式。

不要去想着改变对方，也不要为对方做多少改变。总有人和你是匹配的，也总有人和你不匹配。我经常看到一些男生说：“我做的哪里不够好，你告诉我我一定改。”但事实上，任何问题都有两面性，你的改变只是妥协，并不会让关系变好。

所以你要记住：**好的感情是筛选**。

当你在一段关系中总是你在付出，对方却在原地观望时，你想通过讲道理让他改变，那是不可能的。道理很简单：和他相处几十年的父母都无法改变他，你又怎么可能改变他呢？

为什么有的人在感情中越来越卑微？就是在对方第一次打压你的时候，你接受了。然后对方就会一步步试探你的底线，这就是人性。

以下是需要你牢记的行动准则：
*   感情中只做筛选。
*   不要试图通过说服和讲道理让一个人爱上你。
*   当你把自己所有好的一面都展示以后，对方对你依旧没那么喜欢，那只能说明你们目前不匹配。
*   遇到不匹配的情况，就赶快离开，不要纠缠。
*   千万不要当一个摇尾乞求的哈巴狗。

将重心多放在自己身上。当你自己变得足够好的时候，之前错过你的人，他一定会后悔。
