# 什么是无所谓的态度？保持这几种

在本节课中，我们将探讨恋爱中常被提及的“无所谓”态度。许多人只知其然，不知其所以然。我们将深入分析这种心态的本质，并学习两个核心原则来帮助你建立并保持这种健康的心态。

## 理解“无所谓”心态的本质

你肯定听过这样一句话：恋爱中要掌握主动权，遇到喜欢的人要有“无所谓”的态度。这句话其实很鸡汤，因为后半句——该如何去保持一个好的心态，怎么去“无所谓”——没人告诉你。

上一节我们提到了“无所谓”态度的重要性，本节中我们来看看它的本质。首先，我们要重视一点：没有一个人可以做到遇到喜欢的人，能够完全“无所谓”的。从古至今，我们看过那么多杀伐决断的民族领袖，那么多卓尔不群的商业精英，都会因为一个喜欢的人，夜不能寐。

总而言之，这种患得患失的心态就是人性，不可能被彻底改变，但我们可以学习控制它。

## 心态的基石：安全感

一提到心态，就会关联到另一个词语：安全感。你要记住一句话：**安全感从来都不是别人给的，是自己给自己的**。如果你企图通过对方来满足自己的安全感，最终就会变得患得患失。

那么，具体要怎么做呢？本期内容不说什么聊天话术技巧，记住以下两个核心原则就足够了。

## 核心原则一：自我价值认同

原则一的核心是：**只有自己看得起自己，对方才会看得起你**。

以下是需要自我审视的问题：
*   你是不是那种女生给你一点甜头，就感动得不行，行为上加倍奉还的人？
*   这种心态下，你的潜沟通就是在表达：“你现在就是我能够遇到的天花板了。”
*   这会在对方脑海里形成一个认知：当有一天他不想珍惜你了，你肯定会更加珍惜他。

我们从幼儿园开始就被教育要学会感恩，这是对的。但很少有人告诉我们：没有无缘无故的好。别人对我们好，是因为我们身上有些东西提供了价值，不管是物质价值还是情绪价值。

所以，正确的心态是：你要知道是因为你配得上对方的好。做到**宠辱不惊**即可。公式可以表达为：
**对方的好 ≈ 你提供的价值（物质/情绪）**

## 核心原则二：坚守自我角色

原则二的核心是：**一生最重要的角色是做自己**。

千万不要给自己打上一个“必须做什么”的标签，否则你就会不由自主地想要做一些事情来迎合他人。

以下是几种常见的束缚性标签：
*   谁告诉你家世书香门第，毕业就必须当老师？
*   谁告诉你男人就必须养着女人？
*   谁又说女人就必须在家里带孩子？

如果你被这些外界的规则束缚，你的行为就会以“不产生矛盾”为出发点，那最后难受的只有你自己。所以要敢于说“不”。

放在情感中，你要让对方明白一件事：我和你过去不认识，只是现在在一起了。如果我们不能够成为更好的自己，那这段关系可能就没办法继续了。你要知道，在一段关系中不被重视，往往是因为你把自己当成了对方的附属品。

放在学习工作中，你要找到自己最喜欢的角色定位，这样你才会释放出自己最大的能量和热情。

## 课程总结

本节课中，我们一起学习了“无所谓”态度的内涵与构建方法。

我们首先理解了这种心态是人性的一部分，关键在于控制而非消除。随后，我们探讨了安全感的真正来源在于自身。最后，我们掌握了两个核心原则：**自我价值认同**与**坚守自我角色**。通过践行这些原则，你可以建立起宠辱不惊的底气，在关系中保持平等与自我，从而真正掌握心态的主动权。
