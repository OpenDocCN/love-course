# 大东的Python入门课 #1：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等）。我们通过给盒子起一个名字（变量名）来使用它。

在Python中，我们使用等号 `=` 来给变量赋值。这意味着把等号右边的数据，放进左边名字所代表的“盒子”里。

**公式/代码：**
```python
variable_name = value
```

例如：
```python
age = 25
name = "大东"
```
这里，我们创建了两个“盒子”：一个叫 `age`，里面放了数字 `25`；另一个叫 `name`，里面放了文字 `"大东"`。

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道“盒子”里装的是什么类型的东西。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其说明：

*   **整数**：没有小数部分的数字，例如 `-10`, `0`, `100`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由一系列字符组成的文本，需要用单引号 `'` 或双引号 `"` 括起来，例如 `"你好"`, `'Python'`。
*   **布尔值**：只有两种取值，`True`（真）或 `False`（假），用于逻辑判断。

了解数据类型后，我们如何查看一个变量里装的数据是什么类型呢？Python提供了一个内置函数 `type()` 来帮助我们。

**公式/代码：**
```python
type(variable_name)
```

让我们用代码来实践一下：
```python
height = 1.75
is_student = True
city = "北京"

print(type(age))       # 输出：<class 'int'>
print(type(height))    # 输出：<class 'float'>
print(type(city))      # 输出：<class 'str'>
print(type(is_student))# 输出：<class 'bool'>
```

## 变量的命名规则与操作 ✏️

知道了如何创建变量，我们还需要遵守一些命名规则，并学习如何操作它们。

以下是变量命名的主要规则和常见操作：

*   **规则1**：变量名只能包含字母、数字和下划线 `_`。
*   **规则2**：变量名不能以数字开头。
*   **规则3**：变量名是区分大小写的（`age` 和 `Age` 是两个不同的变量）。
*   **操作**：变量的值是可以改变的，这就是它“变”的含义。

请看下面的例子：
```python
score = 90  # 创建一个变量score，赋值为90
score = 95  # 将变量score的值改为95
print(score) # 输出：95

# 尝试一些不合法的命名
# 2nd_place = “John” # 错误！不能以数字开头
# my-score = 10      # 错误！不能使用减号
```

## 总结 🎯

本节课中我们一起学习了Python编程的基石：**变量**与**数据类型**。

我们首先将变量比喻成存放数据的“带标签的盒子”，并学会了使用 `=` 进行赋值。接着，我们认识了四种基本数据类型：**整数**、**浮点数**、**字符串**和**布尔值**，并使用 `type()` 函数来查看类型。最后，我们掌握了变量的命名规则，并理解了变量的值是可以更新的。

记住，变量是程序中存储和操作数据的基本单元，而数据类型决定了你可以对这些数据做什么样的操作。熟练掌握这些概念，将为后续学习更复杂的编程知识打下坚实的基础。
