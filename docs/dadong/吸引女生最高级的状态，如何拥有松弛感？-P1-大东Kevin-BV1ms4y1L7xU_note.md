## 获得松弛感的三个核心方法

以下是获得并保持松弛感的三个关键点。

这点我放在第一个说，是因为我遇到很多的男生，他们可能看过一些内容，在前期是有松弛感的，但总会在某些时刻掉链子。例如，因为对方有没有回复自己的消息而心情跌宕起伏，或者很在意对方的每一个问题。

说一个非常常见的场景：两个人第一次见面都很好，回到家聊天，女生话里话外都在传递给男生一个信息——“你是不是很喜欢我？”大部分的男生会做出非常严肃的回答。再比如，有的男生表情包很丰富，女生就会说：“你是不是经常给其他女生发这些表情包？”男生就会去做解释。当女生说“我们这样进展是不是太快了”，男生就会变得很慌张。

上一节我们介绍了松弛感的重要性，本节中我们来看看如何长期保持这种状态。第一点就是**不要太在意结果**。

### 2. 注意力放在对方身上

第二点，**注意力放在对方身上**。

我有许多的学员，第一次都可以跟女生见面，其实就说明女生是对你有好感的，不然是不可能和你见面的。但之后你们聊天变冷淡，她开始拒绝你的第二次、第三次邀约，原因就在于你们过于专注自己。你会去复盘自己说的这句话好不好，这句话是不是可以把女生给逗笑。这种思维只是在满足你自己。

兄弟们要相信，双方能够见第一次面，但之后见不了面了，其实就是你搞砸了。但女生不会告诉你，她只会说：“我对你没有感觉。”她不可能告诉你详细的原因。而且，越优质的女生，这种筛选的过程会越快，因为她们身边不缺乏各种各类的人去追求。

总而言之，社交经验越丰富的男生，他们对细节的判断越准确、越快速，也是我常说的社交直觉非常强。你要去有意识地培养这种经验，那就需要把注意力多放在对方的身上，而不是刻意调整自己的一言一行。

看到这里肯定会有很多兄弟说：“我每天忙工作没有时间接触异性”、“我比较内向，不喜欢天天和外人社交”。那也没有关系，你只需要记住一句话：**获得松弛感的前提，就是把需求感降到最低**。

如果你继续死缠烂打，女生就会把你不断地减分，并且对你的负面印象会非常深刻。即使你过了段时间变优秀了，你再去找这个女生，她也会觉得你还是以前的样子。
