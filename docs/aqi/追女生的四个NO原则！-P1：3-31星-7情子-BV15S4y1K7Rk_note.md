## 第一点：不解释

在追求或恋爱阶段，女生可能会提出各种测试性问题，例如：“你是不是经常撩妹子？”或“你是不是有很多前女友？”。

**核心做法是：不要解释。**

错误的做法是急忙辩解，例如说“我很老实，不是你想的那样”。这种解释会让你陷入被动。

正确的做法是，以高姿态回应。例如，你可以说：“我很少和女生聊天，一般都是女生主动找我。” 这种方式能提升你在关系中的位置，避免成为被动的弱势方。

**公式：面对测试 → 不解释 → 高姿态回应**

例如，聊天时对方突然不回复了，你就应该停止发送消息。继续追问只会急速消耗她对你的好感。

## 第三点：不对抗

许多男生喜欢以“为你好”的名义说教，例如“别熬夜”或“少喝酒”。在追求阶段，这属于越界行为。

**核心做法是：支持而非说教。**

你目前的身份是追求者，而非男朋友。过多干涉对方生活，会被认为是多管闲事，导致对方厌烦。

正确的做法是支持对方的决定，不与她对抗。这样她会觉得与你相处很舒服，才更可能愿意和你在一起。

**原则：追求者身份 → 提供支持 → 避免说教**

---

## 第四点：不道歉

道歉是情侣之间调节关系的行为，在追求阶段应尽量避免。

**核心做法是：小事化无，直接跳过。**

如果你犯了小错误，例如说错话或调情尺度稍大，意识到后直接跳过即可，继续正常聊天或约会。对方通常不会将这些小事放在心上。

不道歉可能只扣5分印象分，但一旦你为此反复道歉，则可能被扣50分。当你的好感度被扣完，成功的机会就渺茫了。

**公式：小错误 → 不道歉 → 自然跳过 → 继续互动**
