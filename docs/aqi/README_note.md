# Python基础教程 1.1

📚 在本节课中我们将要学习Python的基础语法和基本操作。

## 1. Python简介

Python是一种解释型、面向对象、动态数据类型的高级编程语言。它具有语法简洁、易于学习、可读性强等特点。

## 2. Python安装

以下是Python的安装步骤：

1. 访问Python官网：https://www.python.org/downloads/
2. 下载适合自己操作系统的Python版本。
3. 双击安装程序，按照提示完成安装。

## 3. Python环境配置

以下是Python环境配置步骤：

1. 打开命令提示符或终端。
2. 输入`python`命令，如果出现Python解释器提示符`>>>`，则表示Python环境配置成功。

## 4. Python基本语法

以下是Python的基本语法：

```python
# 定义变量
a = 10
b = 'Hello, World!'

# 输出变量
print(a)
print(b)
```

## 5. Python数据类型

Python中的数据类型包括：

- **数字类型**：int（整数）、float（浮点数）、complex（复数）
- **字符串类型**：str（字符串）
- **布尔类型**：bool（布尔值）

## 6. Python运算符

Python中的运算符包括：

- **算术运算符**：+（加）、-（减）、*（乘）、/（除）、%（取模）、**（幂）
- **比较运算符**：==（等于）、!=（不等于）、<（小于）、>（大于）、<=（小于等于）、>=（大于等于）
- **逻辑运算符**：and（与）、or（或）、not（非）

## 7. Python控制流

Python中的控制流包括：

- **条件语句**：if、elif、else
- **循环语句**：for、while

## 8. Python函数

以下是Python函数的基本语法：

```python
def 函数名(参数):
    # 函数体
    pass
```

## 9. Python模块

以下是Python模块的基本语法：

```python
import 模块名
```
