# 课程一：阿七 🐱

在本节课中，我们将要学习如何创建并理解一个名为“阿七”的简单角色。我们将通过一系列步骤，从定义基本属性到赋予其行为，来构建这个角色。

## 角色定义

上一节我们介绍了课程目标，本节中我们来看看如何定义一个角色。一个角色通常由一些基本属性构成，例如名字、年龄和品种。

以下是定义“阿七”基本属性的代码示例：

```python
class Cat:
    def __init__(self, name, age, breed):
        self.name = name    # 名字
        self.age = age      # 年龄
        self.breed = breed  # 品种

# 创建阿七这个角色实例
a_qi = Cat(name="阿七", age=3, breed="中华田园猫")
```

在这段代码中，我们创建了一个`Cat`类，并通过`__init__`方法初始化了它的三个属性。随后，我们实例化了一个具体的对象`a_qi`。

## 角色行为

定义了静态属性后，一个角色还需要有动态的行为才能生动起来。行为可以通过类中的方法来实现。

以下是赋予“阿七”一些基本行为的方法：

```python
class Cat:
    # ... 之前的初始化代码 ...

    def meow(self):
        return f"{self.name} 说：喵～"

    def eat(self, food):
        return f"{self.name} 正在吃 {food}。"

    def sleep(self, hours):
        return f"{self.name} 睡了 {hours} 个小时。"
```

现在，阿七可以喵喵叫、吃东西和睡觉了。例如，调用`a_qi.eat(“小鱼干”)`会返回“阿七 正在吃 小鱼干。”

## 核心概念：类与对象

为了更深入地理解，我们需要掌握两个核心概念：**类**与**对象**。

*   **类**：就像一张蓝图或一个模具。它定义了角色应该有哪些属性（数据）和行为（方法）。我们的`Cat`就是一个类。
*   **对象**：是类的一个具体实例，是根据蓝图制造出来的实际产品。我们创建的`a_qi`就是一个对象。

它们的关系可以用一个简单的公式表示：
**对象 = 类()**

## 总结

本节课中我们一起学习了如何构建“阿七”这个角色。我们从定义一个包含属性的`Cat`类开始，然后为其添加了`meow`、`eat`、`sleep`等方法来实现行为。最后，我们理解了**类**作为蓝图和**对象**作为具体实例的核心概念。通过`a_qi = Cat(...)`这行代码，我们将抽象的蓝图变成了一个可以交互的具体角色。
