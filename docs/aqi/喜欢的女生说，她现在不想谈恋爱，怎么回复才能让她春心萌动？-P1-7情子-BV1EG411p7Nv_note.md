# 课程名称：如何应对“不想谈恋爱”的回应 🎯
## 课程编号：情感沟通 101

在本节课中，我们将要学习当喜欢的女生表示“现在不想谈恋爱”时，如何通过有效的沟通策略进行回应，以维护好感并创造进一步发展的机会。

---

当你鼓起勇气向喜欢的女生表白之后，女生却说“我现在不想谈恋爱”，你会怎么办。遇到这种情况，大部分男生都会感到不知所措。等你回过神来时，你可能会真诚地告诉她：“没关系，我可以等你。”但需要明白，如果你真的选择等待，最终的结果可能只会等到她结婚的通知你。

上一节我们提到了无效的等待策略，本节中我们来看看另一种常见的错误回应。

另一部分男生会像我的一个学员一样，思维方式非常直接。女生说不想谈恋爱，他就以为女生真的不想，于是去追问女生：“为什么呀？是我不够好吗？”“是我没有照顾你的感受吗？”“我到底哪里做错了，我可以改呀。”兄弟们，你们觉得这样追问，最后女生会怎么做？她可能会回复你一句：“对不起，你是个好人。”或者更直接地将你拉黑。

在追女生的时候，因为自己说错话，导致女生对你爱搭不理；或者因为过度讨好，让女生对你产生反感。如果你遇到这类问题，可以借鉴接下来的方法。

很多兄弟此时会问：这样不行，那也不行，那我应该怎么回复才是正确的呢？接下来讲的内容全是干货。

首先我们要明白一点：女生说她不想谈恋爱，**并没有说“我不喜欢你”**。如果她不喜欢你，直接拒绝就可以了，但她没有这样做，说明她对你还存有好感。那为什么她会说不想谈恋爱呢？因为在她心里，对你还存在顾虑。只要我们打消掉女生的顾虑，就可以让她接受我们的告白。

所以，这里我们要采取“以退为进”的策略，打女生一个措手不及。

以下是具体的回复示例：

> 好吧，虽然挺可惜的，不过还是要谢谢你。我已经很久没有那种心动的感觉了，这种感觉很好，但我会尽快忘记。你也不用太在意，我们以后还是继续当朋友吧。

当女生跟你说完“不想谈恋爱”这句话的时候，她已经预想到你可能会对她死缠烂打，哀求她再给你一次机会。而我们这样回复，让女生的猜测落空。在这一推一拉之间，哪怕女生原本没有那么喜欢你，听到你这样说，她也会感到失落。

女生可能会回答：“怎么气氛一下子就变沉重了？其实我对你还是有好感的，只是怕你得到太快，不会好好珍惜。”

看到了吗兄弟们？女生的回复已经说出了拒绝我们的真实原因：她在担心我们不会珍惜她。如果我们刚刚表现得很舍不得，甚至是死缠烂打，都会让我们的追求变得更加困难。

既然现在我们得到了真实的原因，那么只需要继续进攻，打消女生的顾虑就可以了。

以下是进一步的回复思路：

> 怎么会呢？你都是我女朋友了，我肯定会加倍对你好。

如果我们只是这样说，女生可能会觉得这是找借口敷衍她，担心追到手之后根本不会重视。所以，我们还要再加一句：

> 如果你不放心的话，我们可以先试着像情侣一样相处。感觉好的话我们就在一起，如果感觉实在不合适，我们就继续做朋友。

这个时候女生会很犹豫，因为她要界定两个人之间的关系。如果我们强行升级关系，就会激起女生的戒备心。这里我们用一个温和的方式，来向她提供她所需要的安全感：“如果觉得不合适，那就做回朋友。”这等于给了女生一个台阶，告诉她我们不会因为她拒绝了我们，就不再对她好。给足了女生安全感，打消了她的顾虑，女生自然也就更乐意接受我们。

女生可能会回答：“那我们先试试。”到了这里，就相当于默许了关系可以进一步发展，之后就要看男生的表现了。

兄弟们要记住，就算女生明确拒绝了你，也还是有机会追到她的。最关键的是掌握聊天的思维，而不是只关注女生抛给你的问题。如果你胡乱回答，让女生从委婉拒绝你，变成了从心底里拒绝你，那么你们最后可能连朋友都没得做。
