上一节我们分析了过度付出的心理陷阱，本节中我们来看看正确的应对方法。

这里有很多人就会说了，难道我对她好都是没用的吗？我们不应该要对别人好吗？

当然不是。在该对女生好的时候，我们也还是要对她好的。但是**怎么对她好**很重要。

这是一种无形的吸引力，能让女生对你持续“上头”。
