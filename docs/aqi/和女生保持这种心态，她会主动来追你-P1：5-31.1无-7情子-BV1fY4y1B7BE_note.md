## 建立心态的两个核心要点

理解了心态的重要性后，接下来我们具体学习如何建立它。以下是建立追女生心态的两个核心要点。

### 第一点：自重自爱

首先要做到自重自爱，自己看得起自己，让对方感觉与你在一起是值得的。这需要我们丰富自己的阅历，做到宠辱不惊。

**核心公式：** 自我价值 = 丰富的阅历 + 稳定的内核

请审视自己：是不是那种对方给你一点甜头，你就感动到不行的人？对方一个简单的举动，就让你倾尽所有？这种心态会向对方传递一个信号：他目前给你的反馈，如果让你感到满足，就会轻易在他脑海中形成一种对你的固定认知。当有一天他对你失去感觉时，这种认知就会成为他内心贬低你的理由。

我们从小被教育要学会感恩，这没错。但世界上没有无缘无故的好。别人对你好，是因为你身上有他所需的物质或情绪价值。因此，面对他人的好，你需要抉择，要让自己配得上这份好，让对方觉得你值得拥有更好的。这才是健康的恋爱相处状态。

### 第二点：勇敢做自己

你这一辈子，最该成为的就是你自己，而不是成为某个你崇拜的明星，或是“某某的男朋友”。否则，你就会不由自主地做一些事来迎合这个标签。

**核心代码：**
```python
if request_conflicts_with_self():
    response = "勇敢说不"
else:
    response = "欣然接受"
```

没有人规定男人必须养着女人，也没有人规定女人必须在家带孩子。但社会上有太多人选择束缚自己，对对方过于宽容，仿佛一切以对方不生气为原则。

所以，要勇敢说“不”。你要让对方明白，你与他过去的生活毫无关联，只是现在在一起，生活中有了交集。如果因为和你在一起，导致我变得不像我自己，那么我们的关系就无法继续。

你要让对方意识到，你是一个独立、随时可以离开，但同时也可以为对方做出牺牲的人。这样，对方才会懂得珍惜你。

## 总结：最好的心态是什么

本节课中，我们一起学习了建立正确心态的两个核心：自重自爱与勇敢做自己。
