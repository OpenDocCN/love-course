## 第三特质：建立双向付出的平等关系 ⚖️

上一节我们探讨了情绪表达，本节我们聚焦于关系的基础模式：**双向付出**。许多男生在追求中会单方面持续付出，这看似是喜欢的表现，实则破坏了关系的平衡。

一段健康的恋爱关系必须建立在双方共同投入的基础上。单方面付出（俗称“舔狗”行为）是低价值的表现。

**不平等付出的发展路径示例**：
*   **起点**：女生请你帮忙搬家，你毫不犹豫答应。
*   **发展**：她提出的要求逐渐升级，如凌晨接她、随时充当司机等。
*   **结果**：你变成了一个“合格的工具人”，女生不会因此珍惜你。

改变现状的方法很简单：**学会反向提出要求，引导对方也满足你的需求**。例如，请她帮你挑选送人的礼物，或为你选几件衣服。在这种小事上，对方通常不会拒绝。

**核心公式**：**单向付出 -> 关系失衡 -> 价值感低**  **VS**  **双向互动 -> 联系紧密 -> 关系健康**

通过这种有来有往的互动，你们的联系会更加紧密，关系也会走向平等。
