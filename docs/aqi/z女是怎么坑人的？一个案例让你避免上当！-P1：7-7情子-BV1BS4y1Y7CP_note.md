当目标开始主动后，对方会通过积极的情绪反馈来强化你的投入行为，让关系迅速升温。

**核心公式**：`高价值赠与 / 低价值回馈 ≈ 维持关系可持续性的策略`

**撤退话术逻辑**：`否定对方人格 + 抬高自身需求 + 将责任归咎于对方 = 完美退场`

本节课我们一起学习了“索取型”关系模式的完整生命周期。其核心套路可以总结为：**营造人设、创造机会、情感升温、物质索取、小额回馈、关系模糊、榨干撤退**。

以下是几个关键的防范要点，帮助你识别类似陷阱：

1.  **警惕快速升温的暧昧**：过快的、超出常理的情感热度可能是引导付出的手段。
2.  **审视付出的对等性**：留意关系中物质与情感付出的平衡性，长期单方面高额付出需警惕。
3.  **确认关系的明确性**：如果对方长期以各种理由（如情伤、考验、时间未到）拖延确立明确关系，则需重新评估其动机。
