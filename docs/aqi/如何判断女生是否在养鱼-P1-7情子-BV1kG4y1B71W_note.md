## 识别“养鱼”行为的三个关键信号

上一节我们介绍了课程的目标，本节中我们来看看具体的判断方法。如果你在追求过程中感到关系停滞不前，对方态度反复，可以留意以下三类典型行为。

3.  **只在有需求时联系你**
