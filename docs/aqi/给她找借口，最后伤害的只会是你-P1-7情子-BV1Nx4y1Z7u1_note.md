在本节课中，我们将要学习如何识别并应对一种常见的情感操控模式：对方通过讲述过去的悲惨经历，来为当下对你的不当行为开脱。理解这一点，有助于你建立健康平等的两性关系。

## 核心情境分析

如果你喜欢的女生对你说，她以前被渣男伤害过，她的原生家庭也对她不好，在读书的时候遭遇到校园暴力。这导致她对每个靠近她的人不信任，并在你面前各种卖惨。

你千万不要去当那个冤大头。我知道你在想什么。

## 常见错误心态与后果

上一节我们介绍了核心情境，本节中我们来看看一旦你产生同情心态，会导致怎样的错误逻辑和行为。

以下是基于同情心可能产生的错误想法：

*   **错误想法**：她都这么惨，我应该对她好一点，我多为她付出一点，让她感受到还是有人关心她的，不就把她给拿下了吗？

当你产生这样的想法之后，这个女生接下来无论做什么行为，你都会给她找台阶下。

例如，她对你很冷淡。你会给自己找心理暗示说：“没事，她以前受过伤，不会轻易相信别人。”

那你就彻底完蛋了。哪怕到最后你们分手了，你都不会觉得是女生的问题。凭什么她用一段以前的经历，就能让你迁就她呢？

## 正确的审视角度

了解了错误心态的后果后，我们需要建立正确的评估框架来审视关系。

你应该审视你们这段关系。评估标准可以用以下**公式**表示：

**健康关系 ≈ 她关心你 + 她理解你 + 你的付出得到相应回馈**

具体而言，你需要判断：
1.  她有没有关心你？
2.  她有没有理解你？
3.  你给了她的付出，能不能得到相应的回馈？

你不要因为她以前的过往，就想去包容她、迁就她。她不会觉得你很贴心，只会认为是自己的魅力，吸引到你过来“舔”她。不要被她表面的东西迷惑了。

## 核心原则与行动建议

基于以上分析，我们可以总结出处理此类情况的核心原则。

现在是现在，以前是以前。你不要像一个圣母一样，什么都去原谅她，给她找借口。

在处理两性关系里面，我们可以用一条简单的**代码逻辑**来理解：
```python
if 她对你冷淡:
    原因 = “不喜欢你”  # 而非“她受过伤”
elif 你的付出得不到回馈:
    状态 = “她在吊着你”  # 而非“她需要时间”
```
给她找再多的借口，那有什么用呢？辛苦的不还是你自己吗？

记住：可怜人必有可恨之处。清醒点吧，兄弟。
