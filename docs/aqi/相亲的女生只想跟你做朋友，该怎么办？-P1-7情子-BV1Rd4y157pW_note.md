# **相亲女生只想做朋友？如何逆转局面？** 🤔

## 概述
在本节课中，我们将探讨如何在与相亲认识的女生交往中，从朋友关系转变为更亲密的关系。

## 案例分析
相亲认识的男生和女生认识一个月后，女生提出做朋友。以下是他们的聊天记录：

**男生**： 终于忙完了来找小新你聊聊天了。
**女生**： 哈哈。
**男生**： 上班了吗？
**女生**： 居家办公。
**男生**： 那你应该不忙了吧？
**女生**： 还行。
**男生**： 感觉现在好无聊啊。
**女生**： 不理我。
**男生**： 你不理我，我好伤心啊。

## 原因分析
女生对男生的聊天方式感到厌烦，因为男生一直在表达需求感，没有掌握聊天的思维框架。

## 解决方案
1. **掌握聊天思维框架**：了解如何延续话题，避免尴尬的提问。
2. **活跃气氛**：通过幽默和玩笑缓解尴尬，让话题自然延伸。
3. **迂回策略**：通过玩笑和暗示，逐步推进关系。

## 实战案例
**男生**： 我们做朋友了，你找我借钱咋办？
**女生**： 不会的。
**男生**： 你就不怕我跟你借钱吗？
**女生**： 那我也不借，哈哈。
**男生**： 做妹妹就比较好。哥哥还能照顾你，带你潇洒。
**女生**： 好主意，行。
**男生**： 那明天哥哥带你去玩一下，你想坐又不敢坐的过山车，害怕的时候呢，就别太用力抓我的手腕。
**女生**： 不会，我顶多拽烂你的衣服，哈哈。

## 总结
本节课中，我们学习了如何在与相亲认识的女生交往中，从朋友关系转变为更亲密的关系。通过掌握聊天思维框架、活跃气氛和迂回策略，我们可以更好地与女生沟通，建立更亲密的关系。
