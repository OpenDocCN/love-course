上一节我们介绍了情敌出现的普遍性，本节中我们来看看如何迈出第一步。

## 第二步：以退为进，制造反差

以下是关键回复策略，核心在于反常规操作。

我让兄弟回复一句：

你以为这是在放弃？其实我们是在**以退为进**。用一种反常规的回答制造反差，勾起女生的好奇心。

女生果然追问：

这就是追女生常用的 **“曲解法”** ，故意曲解女生的意思，来拉近两人关系。

这时候就是你趁势说的好时机。
