## 2. 第一个套路：间歇性强化 🎰

间歇性强化的概念源于心理学家斯金纳的著名实验。该实验将三只猫置于不同的笼子中，每个笼子都有一个按钮。

以下是实验的三个设置：
*   第一只猫：每次按下按钮，都会获得食物。公式可表示为：`按下按钮 → 获得食物（概率100%）`。
*   第二只猫：按下按钮后，有时获得食物，有时没有。公式可表示为：`按下按钮 → 获得食物（概率X%， 0<X<100）`。
*   第三只猫：无论怎样按按钮，永远不会获得食物。公式可表示为：`按下按钮 → 获得食物（概率0%）`。

实验结果出乎许多人意料：按按钮次数最多的，是第二只猫。这就是“间歇性强化”的效果——**不确定的奖励会最大程度地维持和强化某种行为**。

如果一直无条件满足女生的所有需求，就像实验中的第一只猫，她会逐渐对你的好感到“免疫”，认为这是她应得的。她知道只要提出要求，你就会满足，从而对你的兴趣下降。

但如果你运用间歇性强化，先热情后冷淡，再恢复热情，在这种情绪波动中，女生反而更容易对你产生期待和上瘾的感觉。

那么，如何实践呢？关键在于选择性满足对方的需求。

以下是具体操作步骤：
*   **制造落差**：例如，今天她想喝奶茶，你可以选择不买，让她产生轻微的情绪低落。
*   **给予惊喜**：隔天，再找一个随意的理由，突然给她买一杯奶茶。
*   **效果**：这种不可预测性会让女生猜不透你的心思，从而更期待你下一次的好意。
