## 行为一：建立信任感——暂存手机 📱

上一节我们介绍了课程概述，本节中我们来看看第一个核心行为：在约会中建立初步的信任感。

**核心公式**：`主动展示信任 = 传递安全感 + 塑造正面形象`

以下是此行为带来的具体好处：
*   **快速破冰**：请对方保管物品是一种变相的“套近乎”，但姿态并不卑微。
*   **掌握主动**：这个行为由你发起，主动权在你手中，避免了因过于讨好而丧失吸引力。

## 行为二：展现重视度——巧妙通话 📞

我们已经了解了如何通过建立信任来提升好感，接下来，我们看看如何在约会中巧妙地展现你对她的重视。

**核心逻辑**：`制造场景 + 侧面表达 = 引发对方积极联想`
