# 聊天思维的四种层次，你在第几层呢？ - P1 - 7情子 - BV1wU4y1r74K

## 概述
在本节课中，我们将学习如何根据聊天思维的不同层次来提升与女生的交流效果。

## 第一层次：基础交流
以下是基础交流的例子：
- **例子**：你在干嘛呢？大家平常工作好不好？买什么东西？是的，买衣服呀，夏天穿的衣服，吃饭了吗？刚吃完吃了什么？好吃吗？
- **核心概念**：基础交流主要涉及日常生活的询问，缺乏深度和互动。

## 第二层次：基本沟通
以下是基本沟通的例子：
- **例子**：今天天气不错，吃早餐了吗？没呢，你呢？早餐都不吃减肥吗？不吃早餐很久了，我跟你不一样，我每天早餐都吃好多，一天不吃就不习惯，那你多吃点，嗯。
- **核心概念**：基本沟通具备基本的沟通能力，但缺乏引导话题的能力，容易陷入无趣的对话。

## 第三层次：主动表达
以下是主动表达的例子：
- **例子**：你现在的状态是什么？如果是你的话，你会怎么回？绝大多数男生的聊天思维都是停留在第二层次，就是告诉对方自己正在做的事情，然后去反问对方，比如，我刚打完游戏，你呢？
- **核心概念**：主动表达能够表达自己的状态、行为和感受，但容易导致对话陷入僵局。

## 第四层次：情感共鸣
以下是情感共鸣的例子：
- **例子**：女生主动给你发个消息，我今天心情一天都不好烦死了，认真的思考三秒钟，你会怎么回？你的本能反应是不是去问她，到底发生了什么事情，为什么心情不好？
- **核心概念**：情感共鸣能够将自身与话题联系起来，调动女生的情绪，产生暧昧情愫。

## 总结
本节课中我们一起学习了聊天思维的四种层次，从基础交流到情感共鸣，每个层次都有其特点和适用场景。通过提升自己的聊天思维，我们可以更好地与女生交流，增进彼此的感情。
