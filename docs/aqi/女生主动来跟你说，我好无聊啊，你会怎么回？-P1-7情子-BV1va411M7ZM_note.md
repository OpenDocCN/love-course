# 聊天技巧课程 01：如何回应“我好无聊啊” 💬

在本节课中，我们将学习当女生主动说“我好无聊啊”时，如何高情商地回应，从而抓住聊天机会，调动对方情绪，避免把天“聊死”。

---

对于女生主动找自己聊天，这是一个很好的机会。一个会聊天的男生，会立马抓住这个机会。不会聊天的男生，你给他多少机会，他都抓不住。

我有一个朋友因为不懂聊天。当女生跟他说“我好无聊啊”，我朋友回复：“你为什么会无聊啊？你没有自己的事情做吗？”——这简直是直男思维到顶了。我真不知道朋友的脑子里面，到底是装了什么东西，才会这样回复。

如果你这样回答：“我也不知道。”——好家伙，话题直接聊没了。于是我朋友又硬生生转移了一个话题：“你现在在干嘛？”女生回复：“没干嘛，我继续打游戏去了。”

女生本来一个人觉得挺无聊，想找一个人聊聊天。结果发现跟朋友聊天更加无聊，还不如继续去打游戏。好感莫名其妙掉了一大截。

如果有兄弟还是不知道应该怎么去跟女生聊天，或者说你在追女生的时候，你遇到了很多的难题，找不到办法解决。那你可以先告诉我你遇到的问题，我来帮你解决你的难题，让你可以感受一下女生对你的热情主动。

所以有很多男生都说：“女孩子主动找我聊天，我还聊不下去吗？”你看这个案例里的女生够主动了吧？但此男生有能力两句话把天聊死。

你要知道，女生无聊主动找你，那就说明她是愿意跟你聊天的。但是再聊下去，取决于她觉得跟你聊天有没有意思。

有男生立马就悟到了：“诶，我知道该怎么做了。”然后兴高采烈地回了一句：“无聊的时候可以找我聊聊天啊。”女生回答：“那聊什么好呢？”——诶，男生又卡壳了。怎么办？我也不知道聊什么好。

过了5分钟，不知道该如何回复，然后硬着头皮回：“我也不知道聊什么。”女生回个：“哦。”看到没有？就是两句话，男生就能把天聊死。

所以说，以后不要跟我说什么“女生主动就很好聊”。我告诉你，不会聊天的男生，遇到再主动的女生，你都聊不下去。

而情商高的男生呢，他们会抓住这一点点的机会，然后把这个机会无限的放大，让女生爱上和自己的聊天。

你要知道，一个人到什么时候会很无聊？就是内心很空洞、很空虚，没有情绪的时候，其他时间很平淡的时候，就会觉得很无聊。所以，此刻我们要给女生**情绪**。

同样是女生说“我好无聊”，高情商的男生会回复：“你无聊的时候才想起我？”诶，这简单的一句话，瞬间就能调动女生的情绪。情绪上来了，那聊天就变得有意思了。

女生回答：“鬼才想你呢！”——看到没有？这就有种小情侣之间打情骂俏的感觉了。

男生回答：“怪不得我觉得心里凉飕飕的，原来是你在想我。”这样聊天呢，女生就会觉得：“诶，这个男生好有意思。”聊天一下子就起来了兴致。

女生回答：“哼，你自恋得有点过头了，我才没有想你。”过了几分钟女生又说：“你现在在干嘛呢？”——女生口口声声说不想男生，现在却问男生在干嘛，这就是典型的口是心非。

那现在应该怎么回答？情商低的男生又开始说：“我在上班”、“我在工作”、“我在打游戏”。妹子的情绪刚刚上来，你这样回复，那聊天不立马又变得很平淡了吗？

而高情商的男生呢，就继续调动女生的情绪。男生回答：“我在想，你刚刚到底有没有想我。如果想我的话，我就告诉你一个秘密。”——这是**情绪调动**加**好奇心吸引**。

女生此刻的情绪本来就已经在高点了，又来了一个“秘密”，她就很好奇这个秘密到底是什么。果然女生问了：“秘密？什么秘密？”

男生说：“你都不想我，我干嘛要告诉你？”这个时候女生终于放下了矜持，跟男生说：“好好好，那我想你总行了吧？快告诉我秘密。”

看到没有？你们感觉聊了很久，但其实男生就是跟女生说了四句话，就让女生主动承认想自己了。

所以，面对女生聊天的时候，学会引导、调动女生的情绪，你就能在聊天之中，让女生对你产生喜欢的感觉。
