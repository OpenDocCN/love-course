## 1. 积极回应你的主动

```
女生回应积极性 ∝ 她对你的好感度
```

## 2. 对你表示高度认可

在理解了回应的含义后，我们进一步探讨更深层次的表现：认可。

此时的交往状态，更像是双方在情绪上互相牵引。

因为她已经把你的事情当作她自己的事情，默认你是“自己人”。
