# 02托德白天游戏【萌萌团队-皮：3.如何开始一场对话：4如何开始一场对话家庭练习_n

在本节课中，我们将学习如何以更自然、更有力的方式开启对话。我们将介绍几种具体的练习方法，帮助你克服紧张感，建立流畅的交谈能力，并专注于过程而非结果。

---

上一节我们介绍了克服开场紧张的基本概念。本节中，我们来看看几种更具技术性的开场练习，确保你的开场方式有力、自然，并能让你和对方都感到舒适。

这些练习旨在减少对话中的“奇怪感”。第一种开场练习的目标是开始谈话，但不“打击”任何人。在这种情况下，你可以和任何人聊天，包括男生。

但我建议你至少和一个女生一起练习。你可以在街上找到很多人。当有人在街上时，你可以跟着他们走一段路。你不“打击”他们，不阻止他们，也不索要电话号码。你只是跟着走，并和他们聊一段时间。

就这样，保持友善和开心的态度。不要问任何问题，只是边走边聊。最后一段时间，如果他们表示拒绝或觉得奇怪，你仍然可以继续聊。如果他们明确拒绝，那也没关系，回到街上即可。

然后当下一个人走过，你便可以跟着他们走，并尝试完成一段时间的交谈。当你做得好时，可以将此练习升级到下一层，尝试更困难的开场。例如，下一次你可以跟着他们走更长时间，或者将手放在头上增加难度。

这些是更进一步的技巧。但要再次记住，你只是开始一段正常的交谈。你不会“打击”他们。你会感受到交谈氛围是正常和自然的。你能够变得友善。

你能够在交谈中，不再向某人提出任何问题，不再“打击”某人。这是非常重要的一段时间。因为你不提问、不“打击”，这实际上能让你开启许多通常因紧张和恐惧而难以开启的对话组合。

当你能够做到这一点，整个过程的后续部分会更容易。所以，你只需走一走，一直聊天，持续一段时间或一段距离。这就是全部。你不“打击”某人，不问问题，你只是在聊天。如果他们拒绝你，那就没问题，回到起点再试一次。

不要丧失自信，不要担心失败、不成功，或者对方走进了商店。有些事情你无法控制，有些事情并不重要。最重要的是练习。

接下来的练习，其实是一种变化。你可能听过“Tiler的六步法”，用以维持大量动力。整体过程能让你保持正确的心态，同时进行行动。

以下是该方法的第四到第六步练习：
*   第一步：开启所有对话。
*   第二步：减少互动时间。
*   第三步：不要被评价左右。
*   第四步：每次互动都找到有趣的点。

我会更深入地解释这些步骤。

首先，**开启所有**。简直是开启所有对话，不只是开启那些不让你紧张的，不只是开启旁边没有男生的，不只是开启那些看起来很热情的人。简直是开启所有。

开启所有不代表必须有深入的交流，不代表必须有45分钟的长谈。如果你在互动中不想和一个人交谈，你不需要离开，你只需要开启所有人。开启可能只是说一句“你怎么了？”、“你看起来很有趣”或“我只是去见我的朋友，没什么大不了”。你不是为了开启而开启，而是为了有互动。但你不能有“开启的理由”，你必须成为这场交谈的发起者。

下一个，**减少互动时间**。很自然，主要是一开始交谈就直接想到下一步。在一个完美的世界里，互动时间会是“负的”——你跟一个人交谈，然后立刻跟另一个人交谈，同时你还在跟第一个人交谈。但实际上，当你跟一个人交谈，你立刻去找下一个交谈对象，这就足够了。开启所有和减少互动时间很重要，是为了推动行动。

下两个步骤最重要的是让交谈成功，并增强你的能力。它们是：**不要评价**。

不要评价女生是热情还是难相处，谁在乎呢？不要评价这场交谈是好是坏，谁在乎呢？过去发生了一些事，你可以从中学习，然后继续前进。不要评价自己，因为你表现好坏，或因为一个女生喜欢你，你就觉得是世界之王；如果另一个女生不喜欢你，你也不会突然受不了。不要评价，它只是一个事实，一个过去的事实。你会从这里重新开始。

下一个是，**每次交谈都要找到有趣的事**。很多男生去交谈，他们会专注于负面的事，找到每个负面的原因、每个错误。因此，他们会把日子弄得很糟糕，让一切变成下坡路，每个方向都变得更难。

另一方面，如果你在每次交谈中都能找到有趣的事，每个方向都会变得更容易。有趣的事可以很简单：可以是你对那一套说辞很聪明；或者是她拒绝你的速度比你想象中更快，你觉得这很可笑；或者是她穿着最疯狂的衣服；或者是你走过来时忘了说英文，开始嘟囔，你觉得这很可笑。不需要你很窝心或很顺滑。找到有趣的事，就是把你的焦点变成一个积极的看法，而不是消极的看法。

所以规则是：开启所有对话，在交谈中减少时间，不要评价，每次交谈都要找到有趣的事物。做这个练习，然后，这就是要有过程的方向。你不在乎结果，这不是关于女生是否喜欢你。这就是关于过程本身。

如果你完全遵循这四个步骤，即使你得到坏结果，在这场比赛中也不重要，因为成功在于过程。如果你偶尔偏离了过程，却得到了好结果，这很棒，但要把好结果看作是偏离过程的结果。这就是你想要遵循这四个步骤以达到最佳成功的原因。

下一个练习，也是要做大动作。这就是我们称为 **“30秒游戏”** 的练习。它很适合与之前的“减少交谈时间”练习结合。

以下是具体做法：
你会开始倒计时，从30、29、28、27... 当你数到0时，你必须和别人交谈。你将有30秒的交谈时间。这个交谈可以和任何人，最好是女生，因为我们在进行游戏，不只是交谈。但只要能够和别人快速交谈就行。不要让自己站在一边，不要让自己被打扰。要有很多重复，然后，保持自己的高标准，告诉自己：“我正在做的，我的时间是值得的，让我们做到最多。”

下一个练习，这是一个比较进阶的练习。我们要做的是，开启一些**非常不合理**的东西。

以下是一些很好的例子：
如果你想要一些非常不合理、很差的选择，就开启这个话题，然后从这里把设定转过来。你必须真的教会自己，开启这个东西不重要。你可以说任何话，只要你愿意，不论你是否继续说下去，只要你保持一个很好的气氛。

如果你想要其他不合理的东西，就找个有趣的谜语或说法，然后用这个作为开场。或者找个有趣的词，从你喜欢的电影里，然后用这个词作为开场。不管是什么，只要说一些有趣的、好玩的东西作为开场。

这就是我们需要的有趣元素。我们练习的是，下一个指标是从你的嘴巴说出来，从结果中体现，然后真正地自我开启，真正地展现自己的有趣，而不是依赖对方来提供趣味。

所以，做一些不合理的事，不要担心是否会让你气氛爆发。只要你很有趣，这就是我们所要的胜利。像很多这类授权一样，这是一个过程的授权，而不是结果的授权。如果做得正确，结果自然会来。

最后一个练习，我会给你们。这是一个比较进阶的练习。你要在一天内出门，正常进行游戏，但只有一个限制：**你不可以问问题**。

以下是具体规则：
你可以说话，做声明，发出声音，做动物的声音，跳舞，或是保持静默，不管是什么，但你不可以问问题。

主要是，很多男生在和一个人交流时，他们会通过提问来索取价值。他们会问一个女生关于她自己的事，问她做什么，问与对话相关的事。这就是提问。提问的意思是：“请你告诉我有趣的事情。”

当你做一个声明，你会说：“这是有趣的”、“这是我的意见”、“这是我的观点”。所以你会把你的个人特质和交流方式，变得更主动，更具领导力。这就是我们所知的男人的素质。

这就是主要的练习：一天内出门，你不可以问问题。如果你问了问题，那也没关系，不要打击自己，只要认识到你做了，然后继续交流，不再反问问题。

---

本节课中我们一起学习了多种开启对话的进阶练习方法。从简单的跟随聊天，到遵循“开启所有、减少时间、不要评价、寻找趣味”的四步过程法，再到“30秒游戏”、“开启不合理话题”和“禁止提问日”等具体训练。核心在于将焦点从对话结果转移到过程本身，通过大量重复和心态调整，培养自然、有力且有趣的交谈能力。记住，成功在于持续练习和享受过程。
