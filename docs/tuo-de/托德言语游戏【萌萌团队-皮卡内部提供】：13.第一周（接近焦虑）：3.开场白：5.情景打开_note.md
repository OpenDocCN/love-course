# 托德言语游戏【萌萌团队-皮卡内部提供】：课程13.第一周（接近焦虑）：3.开场白：5.情景打开 🎬

在本节课中，我们将要学习一种名为“情景打开”的开场白技巧。这种技巧的核心在于，根据你与对方所处的具体环境和情况来发起对话。我们将探讨它的适用场景、优缺点以及如何有效使用。

---

上一节我们介绍了不同类型的开场白，本节中我们来看看一种与环境紧密相关的特殊类型——情景打开。

情景打开，是指根据你与对方所处的具体环境和即时情况来发起对话的开场白。我个人通常不太偏爱这种打开方式，但它确实有其实用价值，因此值得介绍。这种打开方式的主要问题是力量较弱，但同时它也非常安全。

这意味着它与情绪相关，并且与高度的情绪相关。但在某些特定情境下，这反而是一件好事。在情景中直接使用强力的“开关”（指强力开场白），可能会显得很奇怪。情景打开能给你机会，去尝试一些降低风险的事件。

例如，你身处一个人多的场合，可能是在一条大街上，或是其他公共场所。你使用的打开方式基本上是公开的，很多人都会看到。这本不应是你的问题，你应该能够处理这种压力。但对女生来说，这会使她承受很多压力。

在这种情况下，你可能需要做一个情绪上的、低调的打开。可能是一些简单的谈话。如果女生对谈话有所回应，她就会和你交流。这次交流会给你更多的许可，让你能继续进行正常的谈话。如果你对某些事情进行非常直接的交流，这可能会很奇怪。

如果她喜欢你，因为她感觉到了周围的目光，她知晓当前的情况，她可能会产生负面的反应。甚至可能，现场还有其他人在场，或是发生其他干扰事件。这或许不可能，但可能会发生。这类安全的打开方式，在这种公开场合下是可以接受的。

---

另一个情景打开可能适用的情况是：你已经有了一个很好的交流基础，不需要再去强行“破冰”。

可能在一些情况下，你已经介绍了某人，并且已经和他交流了一段时间。你只需要继续和他交流，不需要再去使用一个强力的“开关”来开启对话，因为你将会有很多时间。

去跟他交流，做一些很有趣或者很出人意料的事情，会显得很不合理。另一个例子是，如果有人在你身边，比如在某个时刻，一个女生坐在你身旁，例如在飞机上。你不需要使用“你很可爱”这类直接的打开方式，因为她就在你身旁。在飞机上，如果这样说会显得很奇怪。

如果已经有了很好的交流基础，你将会有很多时间去跟他交流，因为你就在他身旁。有时候，这可以是一个很好的打开契机。例如这件事，发生在我头上，这就是一个情景打开。在你头上，也会有情景打开的机会。

但如果你坐在星巴克里或者在机场，你只需要说：“今天时间过得好慢，好无聊，有点懒。” 但这很正确，很正常，没有人会对你这句话产生过度反应。如果有人很友善，并且愿意交流，你可能会有一点点机会，跟他开始这次对话。

这次打开的结果是，因为你没有在交流中投入过多，很多时候，甚至是大部分时候，你会在其他“开关”之后使用它。这就像是对话的“开关”，或是对话的开始。如果你说“时间过得好慢”，女生可能会回应：“我懂你意思。” 或者：“对啊，今天是星期天。” 但她不会做出非常热情或能够主动带动交流的回应。

所以你必须要从这里开始去发展对话。要提高情绪，要从她回应的说法中去发展，继续推进。我们会在下星期的课程中讨论，如何继续交流以避免冷场。这就是你必须做的。或者，你可能只有很少的交流时间，然后你可能需要再次开启一个不同的“开关”。

所以，这不是最好的打开方式，但它在困难的情况下，或者在你拥有很多时间、不需要把“开关”的能量一次性耗尽的情况下，是很好的选择。你只需要把感觉传递出去，然后观察反应。当它成功时，对方的投入度会很低，不会像强力“开关”那样，反而会显得很自然。

---

因此，在那些需要介绍的场合，在那些你会和她相处很长时间的情况下，在你会有再次机会跟她聊天的场合，无论你做什么，这些都可以是使用情景打开的好情况。

另外一种情况是，你可能会做一些情绪上的铺垫。例如，如果有个女生在健身房，你见过她，你会认识她。你可能会先传递出你的情绪。如果她回应并和你交流，那很好。如果你跟她很快熟络起来，你找她，你会看到她，但你会说什么？然后你间接地，听到一些影响其他东西的事情，就以此为契机。

所以可能，在这种情况下，把一个小想法说得比较轻松。再说，有很多用途。在他们，你去你的房间，他们在做一些假日活动，你留言说出了这些事件的宣传，这样的事情可以有意思。或者你坐飞机，女生坐在你旁边，你说：“谢天谢地，他们总是把我安排在爱哭的女生旁边。希望这次不会……” 其实这可以做得更好，因为我正在做一个推拉，我正在说希望会有更好的伙伴。但如果你把这部分放下来，然后说：“谢天谢地，我通常都坐在爱哭的女生旁边。” 这会是一个情绪上更开放的表达方式。

所以，任何在这种情景下的留言，都可以让事情开始。这种打开和谈话的方式，不需要局限于男女之间的特定框架。

形容得好。

本节课中我们一起学习了“情景打开”这种开场白技巧。我们了解到它是一种安全但力量较弱的打开方式，适用于公开场合、已有一定交流基础或需要长期相处的场景。它的核心在于利用即时环境发起自然、低压力的对话，为后续交流铺平道路。关键在于从对方的回应中自然发展对话，并适时引导情绪。
