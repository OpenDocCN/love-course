在本节课中，我们将要学习成功者如何看待世界。核心观点是：他们视世界为一系列概率事件，关注长期的平均结果（预期价值），而非单次的成败。我们将通过概念讲解和现实例子来阐明这一思维方式。

## 世界是随机的

上一节我们介绍了赢家之道的基础，本节中我们来看看成功者眼中的世界。我认为，那些最著名的世界级人物，他们看待世界的基本视角是：世界本质上是随机的。

我的意思是，世界上的一切事情，都有一定的可能性发生或不发生。在众多可能发生的事情中，变化非常迅速，存在无数种可能性。

你永远无法拥有完全的控制，只能拥有极端的**统计控制**。

## 理解预期价值

为了理解统计控制，让我讲解一个我从拳击或格斗中学到的概念：**预期价值（Expected Value）**。

每一场比赛，每一次行动，都有一个预期价值。你可以预期得到正确或错误的回报。

但这并不代表你在**那一次**具体的比赛或行动中，就一定会得到正确或错误的回报。然而，如果你重复这个行动一千次或一万次，你最终得到的结果，很可能就会接近那个**预期价值**。

## 现实世界的例子

以下是几个现实世界的例子，用以说明如何基于概率而非单次结果来做决策。

**例子一：深夜回家**
假设两个人离开一家酒吧。一个人是独自回家的。另一个人也是独自回家的。现在，独自回家的人可能更容易出事，更容易遭遇不测。可能就在那一天，某个独自回家的人真的出事了。

但这并不代表“独自回家”这个决定本身是最坏的选择或带来了最坏回报。这只是代表存在风险，而有人不幸成为了小概率事件的受害者。你的判断和决定应该基于**可能性**（概率）。即使独自回家的人出事了，做出“独自回家”决定的人，在当时的信息下，其决定可能是合理的。这合理吗？这只是在看待可能性，而不是看待单次结果。

**例子二：硬币游戏**
例如，如果我给你这个提议：我会抛一枚公平的硬币。如果是正面，你付我两块钱。如果是反面，我付你一块钱。从长期来看，每次我玩这个游戏，我都会平均赚取五毛钱。

计算过程：预期价值 = (0.5 * -$2) + (0.5 * $1) = -$0.5（对你而言）。反之，对我而言是+$0.5。

可能我抛了一次硬币，结果是正面，然后我输了（付给你一块钱）。但我输了这一次，并不代表我做了坏的决定。这合理吗？因为我基于概率做出了对我最有利的选择。我们看待的应该是**可能性**（长期期望），而非单次结果。

## 如何应用：从具体到平均

那么，这到底该如何应用到生活和游戏中呢？首先，你需要一个**足够大的样本**。

如果你只做一件事一次，这没有提供足够的数据来判断你所做的事情是否正确，以及如何改进。你需要许多次重复。

例如，你和一个女生聊天，她因为有男朋友而拒绝了你。这**并不代表**你做错了什么。如果你和一百个女生聊天，每一个女生都说“我有男朋友了”，那么很可能你现在做的方式有问题。所以，你需要一个样本。

你需要有一个概念：你所做的事情的**平均结果**是怎样的，而不是纠结于某一个具体案例的结果。你应该基于**平均结果的改善**来看待事情。

这是思维转变的第一步：从关注具体事件，转向关注统计平均。

## 评估你的处境与调整策略

我建议你，要根据你身处的具体情况来评估你的**风险**与**收益**。

你会参与许多活动，例如商业谈判、体育运动或游戏。在这些活动中，有时你会面临高风险、高收益的抉择。

例如，你进行一项运动，比如踢足球。到了比赛最后时刻，你的队伍在终场前需要得分。此时，你可能会采取高风险策略（如长传冲吊）。如果策略成功，你就赢了；否则，你就输了。但你并没有很多尝试机会，因为如果不采取高风险策略，比赛时间耗尽你同样会输。所以你会选择高风险。

但你永远不会看到一支足球队在**每一场比赛的每一分钟**都采取高风险策略。因为那是一个很差的策略（长期预期价值为负）。

同样的事情也发生在人际交往中。如果你表现得很差，女生想要拒绝你，你可能会尝试非常高风险的举动（比如强行肢体接触），或者说一些非常普通、低风险的话。但如果事情进展顺利，女生喜欢你，你们度过了一个美好的夜晚，之后你送她回家或她送你回家，在那一刻就不应该有很高的风险行为。你要玩得安全，表现得友善，不要破坏已经赢得的有利局面。

## 核心：最大化预期价值

所以，你必须审视你当前处境的**预期价值**。你当前处境的预期价值，包含了你的行为所带来的影响。

在你做出的每一个决定中，你都必须追求**最高的预期价值**。这不仅与游戏有关，与运动有关，与商业有关，与你想要做的任何事情都有关。

**本节课总结**

本节课中，我们一起学习了成功者如何看待世界。核心要点是：
1.  接受世界的**随机性**，我们只能进行**统计控制**。
2.  理解并运用**预期价值**的概念来指导决策，关注长期平均回报而非单次结果。
3.  通过积累足够**样本**来评估和改善自身行为的平均效果。
4.  根据具体**处境**动态调整风险与收益的权衡。
5.  在所有决策中，始终以**最大化预期价值**为最终目标。
