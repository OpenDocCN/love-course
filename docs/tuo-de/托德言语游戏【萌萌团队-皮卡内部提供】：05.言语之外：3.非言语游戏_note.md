# 托德言语游戏【萌萌团队-皮卡内部提供】：05.言语之外：3.非言语游戏_note

在本节课中，我们将要学习“非言语游戏”的概念。我们将探讨如何通过身体动作和特定行为来传递信息，这些信息有时比言语本身更具力量。理解并掌握这些技巧，能让你在沟通中更加游刃有余。

## 核心概念：非言语游戏

上一节我们介绍了言语和声音在游戏中的运用，本节中我们来看看“非言语游戏”。这并非指身体语言，如站姿或笑容，虽然那些也很重要。这里所说的“非言语游戏”特指通过身体动作和行为来传达特定信息的游戏。

**核心公式**：`非言语游戏 = 特定身体动作 → 传达特定信息`

## 非言语游戏的含义与示例

非言语游戏是指你的行为本身会传递出一个特定的信息。这些行为可能伴随着一句话，但其核心信息是由动作本身表达的。

以下是几个具体的例子：

*   **示例一：接受提议**
    假设有人向你提出一个商业建议，你一边说“我会接受”，一边将手放在手机上。这个放手机的动作就是一个非言语游戏，它强化并具体化了“接受”这个信息。

*   **示例二：亲吻或击掌**
    一个亲吻或一个击掌（High Five）也是典型的非言语游戏。它们有自己约定俗成的含义，无需言语解释。

*   **示例三：游戏词句的应用**
    有些游戏词句，如“不要说什么你能笑”，其本质是鼓励用最少的言语和力量来达成效果。非言语动作正是实践这一原则的绝佳方式。

## 非言语游戏的力量与运用

非言语动作之所以强大，是因为它通常比直接的语言陈述更含蓄、更有力。用面部表情或一个简单的动作来代替“我不同意”这句话，往往能传递出更强烈的情绪和态度，同时显得更有分量。

这种力量体现在不同的程度上，需要我们仔细观察和体会。
