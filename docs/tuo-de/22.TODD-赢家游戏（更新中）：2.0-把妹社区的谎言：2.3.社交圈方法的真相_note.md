## 2.0-把妹社区的谎言
### 2.3. 社交圈方法的真相 🔍

在本节中，我们将深入探讨社交圈方法的真相，分析其潜在的负面影响与局限性，并帮助你理解为何它不应被视为核心的社交技术。

社交圈方法存在诸多负面性。第一，女性对社交证据的反应非常强烈。第二，在社交圈中，女性可以获得更多接触机会。第三，在社交圈内，女性之间会建立更多信任。因此，在社交圈环境中，女性会展现出更多魅力。

我建议自然地建立社交圈。你可以享受与同性的交往，这是我过去所批评的田园式社交，因为人们常常盲目融入，而其他人则处于戒备状态。然而，你的社交圈确实重要。首先，社交圈本身可能陷入停滞，这非常可惜。如果你在社交圈内没有运用“游戏”技巧，你不会被淹没。家庭中有很多人未被淹没，运动队中有很多人未被淹没，学校中有很多人未被淹没，社交圈内也有很多人未被淹没。

你仍然需要“游戏”技巧。许多人认为学习社交圈就是全部，他们不必学习“游戏”，不必学习塑造人格，不必学习如何提升自己，也不必学习其他技能。他们认为只需学习社交圈就能让他们感觉良好，就像在学校里做个好孩子一样。

这感觉虽好，但会让你蒙受损失。你会想把这项技术变成一个不存在的捷径。创造社交圈的最佳方法其实是透过“游戏”，是通过“游戏”来遇到人，然后把大家聚集到一起。这样，你将拥有整个世界。社交圈还存在更多问题。

如果你在社交圈内工作并在社交圈内约会，那么约会圈外的人会变得困难。在社交圈内约会那些人本身就很困难，因此你的选择会受到很大限制。如果你有更多想法，这将成为一个问题。第二，你正在建立一个名声。假设你经历了一次失败，你可能会让整个社交圈对自己感到失望。

社交圈有很多缺点。你可能会想，我可以利用社交圈找到新工作，但你没有学习“冷启动”的方法。然后你在餐厅遇到一个很有魅力的女孩，你现在该怎么办？你会想把她带入你的社交圈吗？这仍然需要“冷启动”的能力。因此，观点是：社交圈是一项很好的辅助技术，但它不是最根本的技术。

如果你没有最根本的吸引人的能力，如果你没有最根本的“收尾”能力，如果你没有最根本的“冷启动”能力，那么我为什么要这样做？有人说，做社交圈能让事情更容易，更生动，更有生命力。就好像，你建立了社交圈，女孩们就会主动来找你。

但这本身就是一个问题。是的，你建立社交圈。首先，这并不容易，需要花费大量时间去维护和运作。所有人都在建立社交圈，不只是你。你认为自己很特别，就能得到通告吗？很多人都想这样。所以这不只是关于生命力的事情，这需要花费大量时间、大量精力。

那些擅长社交圈的男人，他们花了一辈子时间在建立社交圈。他们非常擅长规划他们社交圈的下一步行动。所以这些男人会说，追求女孩很可怕。他们在追求女孩时，实际上是在进行更精细、更复杂的社交圈建设，而不是直接去追求女孩。他们也在建立社交圈上花费了大量时间，尤其是那些专注于建设社交圈的男人。

接下来的问题是，当你建立了社交圈，当你加入后，你认为你能坐下来，它就自动维持吗？你认为你不必和那些人保持联系吗？你认为你不应该欠那些人的人情吗？你绝对会这样想。所以你创造了很多不需要花太多时间维持的工作，但那仍然需要大量时间。

那是你花费了大量时间、大量心理努力在建立和维持社交圈上，而这确实是一项工作。所以，如果你享受自己所做的事，如果你真正享受社交圈带来的活动，而不只是为了达到接触女人的目标，那么你应该去做。你应该最大化你建立的社交联系，在这场合作中绝对可以。

但是，如果建立社交圈意味着你要花时间在你不喜欢、不尊重、不爱的人身上——你知道整个概念——你最多花时间在那五个人身上。你最多花时间在那五个人身上。你想那五个人，比如一些社交圈建设者、一个商人，还有一些酒友，那是你想要的生活吗？

如果你想要的生活，你享受它，那就花一辈子去建立那个社交圈。如果你想要的生活，你不想花那么多时间，那就把时间花在你的生意上，去赚钱，去享受好生活，去见其他人，享受好假期。这就是更好的“社交圈”，因为你会遇到的不仅是热情的女孩，还有上层的女孩：有目标、聪明、不酗酒、注重健康等等。

所以，我所说的社交圈是一件很大的事吗？绝对是。有朋友和信任的人一起做事是一件很大的事吗？绝对是。你会花一辈子的生活去享受好生活吗？社交圈绝对不是。你花费的时间和精力会比你想象的更多。

你花费时间学习和实践的，应该是“冷启动”的方向。此外，它并没有同样的发展性。它让人们为了好运而开心地做事，但实际上并不让你成为一个人，并不让你的生活成为一个人的生活，而且还需要维护。是的，需要维护，而且你还没有建立好根本的技术。

最后，我们来看看社交圈里最好的朋友，他们会不会约会很有魅力的女生？是的，他们约会。一年多约会一个有魅力的女生，或者一年多约会一个有魅力的女生。他们不会约会很多有魅力的女生。但有魅力的女生会约会很多有魅力的女生。

对我来说，我会一年多约会一个有魅力的女生吗？是的，如果没有，我会约会很多有魅力的女生。在八九分级里，可能不是顶级的。是的，我会约会很多有魅力的女生，同时进行。是的，我会有能力在我的生活中轻松地创造其他部分。我会运用我的技术。我会有，但他们不会。我会有，和我自己，做我喜欢的事，而不需要担心我正在建立的连接，或者我是否很厉害，或者是很好的朋友。

是的，你们能建立社交圈，能有那种豪华吗？不，这就是为什么我不会在生活的每个部分都做社交圈。把自己的社交圈，他要把所有技术和资源都投入进去吗？我认为这是挑战，卡梅伦。我认为你不应该这样做。

**总结**

本节课我们一起学习了社交圈方法的真相。我们分析了社交圈依赖社交证据、提供更多接触机会与信任感等表面优点，但重点揭示了其核心问题：它不能替代根本的吸引力、“冷启动”和“收尾”等核心社交技术。建立和维护社交圈耗时耗力，且可能限制你的约会选择与发展。真正的成长来自于掌握核心社交技能，并以此为基础，自然地去遇见和连接那些与你志趣相投、能丰富你生活的人，而不是将社交圈建设本身视为终极目标或捷径。
