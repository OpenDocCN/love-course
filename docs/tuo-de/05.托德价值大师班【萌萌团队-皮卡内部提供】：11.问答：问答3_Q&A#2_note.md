所以，人们有自己的价值观，但他们也追随强大的影响。他们追随强大的领导力。他们追随高价值的人，就像你通常也会一样。如果你身边有一个高价值的人，你会自然地想做类似的事情。当他们谈论自己时，你会自然地想说：“哦，我也是。这是我们之间的相似之处。”当你感觉到别人有高价值时，这是一种自然的冲动。

现在，如果你学习了社交技巧，你会知道这是一种低价值行为或“找平”之类的，你可能会克制自己这样做。你可能会阻止自己，但那种冲动仍然存在。记住，吸引力不是一种选择，对价值做出反应也不是一种选择。然而，你如何回应它，或你选择如何处理它，是可以选择的。这基本上就是社会压力。

社会压力是你如何看待世界，与另一个强大的影响所提出的不同世界观之间的冲突。有时你只是说“不”，并保持你的观点。有时你说：“你知道吗？那有道理。”你接受它并完全跟随。很多时候，通常是某种妥协。通常，有些事情你非常坚信，不愿意妥协；有些事情你不太在意，愿意妥协。

认为人们只是对刺激做出自动反应的想法是不准确的。我们比那更复杂。我们绝对会感受到吸引力，会被吸引我们的事物吸引。我们绝对会注意到对我们有价值的事物。但我们在如何解读它方面确实有很多自由意志。
