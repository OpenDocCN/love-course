# 托德言语游戏【萌萌团队-皮卡内部提供】：15.第三周（吸引）：1.吸引技巧：02.推拉_note

在本节课中，我们将要学习“推拉”这一核心吸引技巧。推拉是一种通过交替给予正面（拉）和负面（推）反馈来制造情绪波动和吸引力的沟通方法。掌握它能让你的互动充满张力与趣味。

## 概述：什么是推拉？🤔

推拉是吸引技巧中最主要的构思之一。其核心在于“抖摆”的概念，即在互动中交替展现正确与错误、喜欢与讨厌的两面性。许多其他技巧都包含推拉的成分。

## 推拉的心理基础 💡

上一节我们概述了推拉，本节中我们来看看它的心理基础。这种技巧模拟了一种经典的情感波动。

例如，在1950年代的老电影中，经常有这样的场景：女生一边撕扯花瓣，一边念叨“他爱我，他不爱我”。这种不确定性和情绪起伏，正是女生所喜欢或能被营造的感觉。

因为这种混合了负面情绪的感觉非常引人入胜，它能激发对方的兴趣和感动。这会使她想“赢”得你，想让你感到惊讶。这与直接完成对话不同，你并非简单地陈述，而是制造一种“唉呀，真可惜”的张力。

## 推拉的核心机制 ⚙️

推拉的核心机制可以概括为一个**公式**：
**吸引力 = 正面反馈（拉） + 负面反馈（推）**

其目的是通过设定预期（拉），然后做出反向冲击（推），让对方感到胆怯或意外，从而制造出有趣的情绪波动。如果你只是平淡地与人交流，什么都不会发生，也无法传递有效的吸引信号。

## 推拉话术示例 🗣️

我会解释这个现象，并给你们一些例子。以下是经过测试的推拉话术示例，它们在两年间不断演变，你可以相信其有效性。

**首先，介绍几个经典的推拉句式：**

1.  **“我爱你，但我恨你。”**
    或者“我有点爱你，又有点恨你”可能是更好的变体。整个概念在于矛盾的情感陈述，例如“我爱你但我不喜欢你”，或“我爱你但其实我不爱你”。这类话术是最基本的推拉，效果会很深刻。

2.  **“我爱你，直到你说了那句话。”**
    或者“你很酷，直到你做了那件事。”这是另一种推拉话术形式，其结构“你…但…”非常清晰。

3.  **“你刚刚失掉了一分。”**
    你可能会问，为什么这是一个推拉？这不只是一个负面声明吗？“你刚刚失掉了一分”在初次互动中是一个负面指示，它降低了你的价值。这一点是比较负面的，甚至是完全负面的。

**接下来，分析一个轻微但有效的推拉实例：**

“你刚刚失掉了一分”之所以是推拉，是因为“失分”并不意味着失去所有，它仍然保留了一些东西。这种轻微的贬低包含了两种元素：既有的价值和刚刚失去的部分。

在现实世界中，你的很多推拉话术都会比较轻微。它不一定是强烈的“我喜欢你，但我恨你”。很多人可能更倾向于使用负面表述，但纯粹的负面就只是负面。

纯粹的负面表述是：“我喜欢你。”而推拉是：“我喜欢你…（但现在）”、“我以前喜欢你…（但现在）”、“当我安排你的时候，我还喜欢你…（但现在）”。这会让人感到惊讶，觉得事情本不该如此，从而产生一种缺失感。

这种“缺失感”与你的负面推拉相结合，会让对方感觉这些互动有意义。最后，虽然可能有些复杂，但通过结构化的练习和观察，你会掌握它。

## 如何练习与运用 🥊

学习如何让推拉变得更好，可以观察单口喜剧演员，尤其是他们的表演方式。

他们会引导你走向一个方向，然后让你惊喜。我遇到过一个人，他的方式起初有点可怕，但我认为他是一个优秀的演员，而且必定精通此道——那就是安东尼·杰塞尔尼克。他在许多笑话中运用了出色的推拉风格：先引导你走向一个方向，然后出其不意地反转。这正是最有趣的部分。

## 总结 📚

本节课中，我们一起学习了“推拉”技巧。我们了解了它的定义是交替使用正面与负面反馈来制造吸引力。探讨了其模仿情感不确定性的心理基础，并通过**公式 `吸引力 = 拉 + 推`** 理解了核心机制。我们分析了几种不同强度的话术示例，从强烈的矛盾表达到轻微的贬低。最后，我们知道了可以通过观察单口喜剧等表演来更好地理解和运用推拉，让互动充满惊喜和张力。
