# 托德言语游戏教程【萌萌团队-皮卡内部提供】：18.第六周（叙事）：1-叙事：18-叙事-最后的思考 🧊

## 概述

在本节课中，我们将要学习叙事在长期关系中的核心作用及其发展规律。我们将探讨故事如何从微小开端逐渐成长为关系的主导部分，并理解主动构建故事的重要性。

## 教程正文

你与某人建立了长期关系。这段关系本身可能被遗忘，但其中的故事会留存下来。故事是不断增添内容和构建而成的。

那种故事的基础，最初就像一颗**小冰塊**。然后它逐渐生长，变成了**冰柱**。最终，它可能扩展成整个**北方**或**南方**的版图。无论如何，它从一个或大或小的冰块开始。

它变得越来越大，实际上是在不断成长。关系越深入，这个进程就越明显。正因为这个原因，关系中其他游戏性的元素在逐渐减少。

而故事及其影响力则在增加。这就像在关系中形成了一颗小冰块。一旦许多游戏性的元素消失，你并不会完全停止在关系上“游戏”。

但这正是长期关系的定义。以下是关于叙事在关系中演变的几个关键点：

*   **女生的视角**：女生在关系中的想法，以及女生对自己关系的看法，会越来越被故事所占据。
*   **故事的比重**：关系持续得越久，故事就会变成关系中更大部分。
*   **早期构建**：因此，你需要在关系最初阶段就开始有意识地构建它。

这样，它才会成长为你期望的样子，而不是你未曾预料的样子。所以，要将思考转向故事，并且要明白，在关系深入发展后，故事会成为关系的主要部分。

在最初阶段，你会有你的目标、你的评估以及其他各种考量。但当你决定喜欢一个女生，当她决定喜欢你时，你们便开始共同构建“我们的故事”。

这就变成了至关重要的事。甚至从第一刻起，故事就开始发展。因此，必须注意，不要让它意外地发生。因为在一年或更久的相处之后，故事注定会成为关系的主要部分。

## 总结

本节课中，我们一起学习了叙事在长期关系中的动态发展。我们了解到，故事会像冰块一样从微小基础不断成长，最终成为关系的核心。关键在于从关系初期就有意识地共同构建故事，引导其向期望的方向发展，而非任其意外形成。
