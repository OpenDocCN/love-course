# 课程02：托德白天游戏【萌萌团队-皮卡内部提供】🚀
## 5. 如何获得技能：4. 如何获得技能内场练习

在本节课中，我们将学习一系列具体的练习方法，帮助你通过实际行动来掌握社交技能，特别是在白天与人互动的场景中。这些练习旨在让你逐步适应社交环境，建立自信，并学会如何自然地开启和推进互动。

---

### 概述：从打招呼开始

上一节我们探讨了技能获取的理论基础，本节中我们来看看如何通过实践来内化这些技能。首先，我们将从一个简单但有效的练习开始：主动打招呼。

这个练习的目的是帮助你习惯“走出自我”，以最自然的状态开始社交。具体做法是，在一天之内，主动向遇到的人打招呼。无论是男性还是女性，都像对待平常人一样开始一段简单的对话。这个过程起初可能会感觉奇怪，但坚持去做会让你发现其中的乐趣。

这个练习不仅能让你感受到社交的趣味，还能帮助你理解良好社交的缘由，并让你逐步进入一个更开放、更主动的社交角色。

---

### 练习一：五级运动体验清单

接下来，我们介绍一个结构化的练习方法，帮助你循序渐进地突破舒适区。

以下是“五级运动体验清单”的具体内容。这个清单包含从第一级（最舒适）到第五级（最害怕或最不舒服）的一系列互动动作示例，例如进行眼神接触、开启对话、轻轻触碰对方手臂（如“海鸟体 edge”）、进行类似交谊舞的互动（“搅拳”）、共同观看短片、进行肢体碰撞游戏、玩“learner 口袋”游戏、尝试快速轻吻脸颊（“Peeker”）、或是进行“igglypins”等互动游戏。

使用这个清单的方法是：当你进行互动时，从清单上的第一级（你最舒服的一件事）开始行动。成功完成第一级后，再尝试第二级，依此类推。

**公式：** `当前行动 = 清单[已完成上一级索引 + 1]`

这是一个可以让你逐步前进的检查列表。它采用了一种渐进增加难度或令你紧张的方法，因此你不需要直接跳进深水区。你可以慢慢地感受整个设定，适应后再向前推进。

通常情况下，你会开始进行一些实际的互动。通过这个清单，你绝对可以完成一个循序渐进的检查过程。

### 练习中的调整与应对

在进行这些动作时，有时你可能会被拒绝，或者自己感到些许不适。

如果发生这种情况，你完全可以暂停下来。放松一下，聊点别的，再次放松。重复这个过程，之后再尝试进行实际动作。

如果对方的反应缺乏责任性（例如她不愿意拥抱），你可以选择跳过那一步，直接尝试下一步。或者，可以转换动作，例如改为类似跳舞的转身动作。跳过那一步，再继续前进，因为她可能感到有点不舒服。

但如果她只是感到些许不适，你可以选择向前一步，放松一下，再放松一下。重复这个“放松-前进”的节奏，这就是“跳过两步，向前走”的动作模式。

### 练习二：融入高能量环境

下一步练习，虽然并非立即要做的动作，但将是你在“日游戏”中的一个重要选择。

我建议你尝试寻找节日庆典或派对场所，找一个让人们很活跃的地方。因为这将会提升你自身的能量水平。

在“日游戏”中学习很容易停留在低能量状态，因为很多时候你可以轻易离开。例如，在咖啡店与女生交谈，环境能量很低，你不需要花费太多精力去吸引她的注意力。或者你看到一个女生站着，能量也很低。

但如果你学到的仅限于此，特别是在“日游戏”中，那么更高能量的“节日游戏”对你来说将会更加困难。你也不会获得太多乐趣，无法以你想要的方式提升能力和达成目标。

因此，我希望你定期（不需要每次，或许每月一次）去一个“节日游戏”的场地。“节日游戏”场地是一个很好的选择。你也可以去“夜游戏”的场地，只是为了提升你在“日游戏”中的舒适度。但关键是去一个能量相对较高的环境。

我认为节日、派对这类活动，能让互动更有活力和动态。另一个选择是，去街上人多、热闹、有人跳舞、有人玩游戏的地方。

每次在这样的环境中练习游戏，就像健身时增加重量。如果你能习惯“提升重量”（即适应高能量环境），那么日常的社交活动相比之下就会变得更容易。

### 练习三：非语言互动游戏

接下来的练习是一个挑战：进行“不讲话”的游戏。

“日游戏”非常依赖语言，因此，要进行完整的互动而不说话是困难的。你可以尝试，但可能不会获得太多成功。

我建议你这样做：在互动的第一分钟里尝试“不讲话”游戏，或者更好的方式是，坚持“不讲话”直到对方先开口说话。

具体做法是：不要说话，通过玩耍、发出一些声音、做出表情等方式进行互动。然后，只有当对方跟上你的节奏时，你才跟上她。

这样做首先是因为，很多时候如果你不讲话，你只能通过跟随她并做一些微小的动作来互动。因此，你必须解决这个困难。但同时，你也在教会自己：说话并不太重要。

**核心概念：** 重要的不是说话，而是**感觉**、**能量**、**肢体语言**和**触碰**。

这是一个很好的练习，用于解决对“说话”的过度依赖。这很有趣，因为很多人会纠结于“我该说什么”，他们认为这是一个巨大的任务，需要把所有话都说出来。但有时，少说话反而效果更好。

无论如何，这个练习能真正提升语言的价值，让你把说话视为一种荣幸而非负担。你的语言游戏也会得到提升，因为你学会了何时“拿走”语言。

所以，练习这个。我不建议在整个互动中完全不说话，因为在白天游戏的环境中这很难做到。但可以尝试坚持到对方先说话，或者坚持前30秒不说话。这样做能让你自己变得更有活力、更有表现力，推动你的舒适区边界，从而让你更有“体形”（即存在感和影响力）。

### 练习四：肢体接触开启者

最后一个练习，我将教你如何运用“体形”（即肢体接触）来开启互动。你必须学会在互动开场时就使用肢体接触或其他方式。

我会给你四个可以使用的“肢体接触开启者”：

以下是四个具体的“肢体接触开启者”动作：

1.  **魔法按钮**：我称之为“魔法按钮”。这是指轻按对方的肩膀，以吸引她的注意力。这个动作会打破那种无形的肢体接触障碍。你已经在一开始就触碰了她。这个动作不严厉，也不可怕，但它能有效破除隔阂。你不要只是说“嗨，不好意思，可以跟你聊聊吗？”，而是说“嗨！”的同时，轻轻按一下她的肩膀。这会有很大的区别，能让你更有信心，吸引她的注意力，为后续更亲密的互动铺平道路。
2.  **轻拉转身**：当你和一个女生在一起时，你也可以轻轻地拉她转身。这是一个在日常互动中风险稍高的动作，但也不错，可以让你更放松一点。我会说这是我建议你做的最高风险动作之一。
3.  **“神之手”**：接下来是“手上的神”。你会在对话交流中，把手伸出来，让她有机会握住你的手。如果她握住了，你可以把她拉近，拥抱她，搂住她，拉她更近一点并和她说话。如果她没有握住你的手，没问题，你只需自然地把手放下，继续说话，就像什么都没发生一样。但这是一种提供快速升级关系机会的方式。这是“神之手”，你在对话中给她握住你手的机会，让事情变得更好。如果她回应好，就推进；如果不好，也没关系。
4.  **手挽手动作**：最后，我教你一个最高风险的经典游戏动作。你可能见过，这是一种过去十年来非常有效、非常有趣的原因：**手挽手动作**。你让女生握住你的手，然后你们手挽手。你把她拉近一点，像跳舞一样。这立刻就设定了男女之间的氛围，而不是朋友之间的氛围。这是一个有趣的互动，而不是尴尬可怜的。通过这个方式，你能很快设定一个亲密的氛围。

你这次练习的基本目标，就是在每个组合互动中，使用这类肢体接触方式来开启氛围。你不能进行那种“拿着话筒”、站在离女生四米远、不能碰她的对话。你必须开启氛围，即使只是一点点。

### 总结

本节课中，我们一起学习了四种核心的内场练习方法：从主动打招呼建立习惯，到使用“五级运动体验清单”循序渐进；从主动进入高能量环境提升状态，到练习非语言互动以重视感觉而非言语；最后学习了四种具体的肢体接触开启技巧来快速建立亲密氛围。这些练习旨在通过实际行动，帮助你逐步克服社交障碍，更自然、更自信地进行日间互动。记住，关键在于持续练习和逐步推进。
