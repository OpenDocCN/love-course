# 课程05：价值的内在博弈 🧠 - 2. “状态”与“心流”相关概念

在本节课中，我们将要学习“状态”与“心流”这两个核心概念。我们将探讨它们的含义、作用，以及如何在实际生活中理解和运用它们，同时避免对其产生过度依赖。

## 概述

“状态”和“心流”是描述我们心理和表现模式的重要概念。理解它们有助于我们更好地工作与学习，但关键在于如何智慧地利用它们，而非被它们所控制。

## 什么是“状态”？

上一节我们介绍了价值的内在博弈，本节中我们来看看“状态”这个概念。

“状态”指的是你当下的心情或情绪。当你心情好时，你通常会表现得更好，这在心理学上是得到验证的。例如，很多人心情好时工作效率更高；或者当人们听到某些音乐时，状态会得到提升，从而表现更佳。

人们通常能清楚地意识到自己的“状态”。但问题在于，如果你一直依赖“状态”——比如总想着“我必须保持好心情”或“我必须听特定的音乐才能工作”——那么你就会产生依赖。

以下是关于“状态”的几个要点：
*   **状态可以被利用**：你可以聪明地适应它，以最佳方式去行动。
*   **状态不能成为依赖**：你不能让整个工作过程都依赖于维持某种特定状态。
*   **状态不是借口**：你不能对老板说“我今天很生气，所以不能工作”，这是无效的。状态是用于理解、制定策略和规划的工具，而非逃避责任的严重借口。

因此，我们不希望你用“状态”作为借口。我更喜欢使用“心流”这个概念。

## 什么是“心流”？

理解了“状态”的局限性后，我们再来看看更积极的“心流”概念。

“心流”是指你对你所做的事情感到深度投入、充满活力。当你处于心流中时，整个世界仿佛都围绕着你，你能真正地专注并做到最好。

我所知道关于“心流”的最好例子就是电子游戏。电子游戏的设计就是为了让你进入心流状态。它通过提供恰到好处的挑战、成功与失败的对比，让你持续游玩。你可以开始玩一款游戏，然后通宵达旦，忘记时间，完全沉浸在那个世界中，这就是进入了“心流”。

但是，如果游戏挑战对你而言太难（不断失败）或太简单（不断轻松获胜），你都会感到不适，而无法进入心流状态。

## 如何对待“状态”与“心流”？

那么，我们应该如何看待并运用这两个概念呢？

在“状态”方面，我们应关注如何进入“心流”，如何创造适度的挑战，如何与自身合作，以及如何营造能带来成果的情境。但你永远不能依赖它。

在“心流”方面，它同样不是一个借口。你应该做的是那些能引导你进入心流的行为。最有趣的部分在于：你不能强迫自己进入某种“状态”，也不能强迫自己进入“心流”。

**`正确的做法是：做正确的事，然后让心流自然发生。`**

心流是一种无意识的功能，就像打嗝一样。如果我命令你“现在打嗝”，你可能做不到。同样，你很难直接命令自己的心脏高速跳动（虽然你可以通过跑步间接做到）。一旦你开始思考它，反而很难实现。

所以，你很难直接控制“心流”和“状态”，但你可以间接影响它们：
*   **做正确的行为**
*   **做正确的事情**
*   **专注于正确的行为和事情**

但你不能刻意去追求。如果你一直在想“我现在处于心流吗？我的状态好吗？”，你只会把自己困在头脑里，反而无法专注。

因此，正确的做法不是去监控和检查自己是否处于“心流”或“好状态”，而是去**做那些能直接产生正确反应的行为**。

这些都是很好的概念，你应该将自己的练习围绕它们来组织。但你绝对不能让自己受制于它们。你必须能够在或不在“心流”与“好状态”中都能独立工作。

**`你不应该把它们用作借口。`**

## 总结

本节课中，我们一起学习了“状态”与“心流”这两个核心概念。
*   **“状态”** 是你的即时情绪，可以被利用来提升表现，但不能成为依赖或逃避责任的借口。
*   **“心流”** 是深度投入和最佳表现的心理状态，它需要通过做正确的事来自然引发，而非刻意追求。
*   关键在于**专注于产生结果的行为本身**，让积极的状态和心流作为副产品自然出现，而不是反过来被它们所控制。永远不要用它们作为无法行动或表现不佳的借口。
