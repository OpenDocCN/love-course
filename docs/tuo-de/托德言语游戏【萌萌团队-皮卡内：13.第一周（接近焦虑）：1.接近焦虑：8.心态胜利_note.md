# 托德言语游戏【萌萌团队-皮卡内部提供】：课程13 - 第一周（接近焦虑）：1.接近焦虑：8.心态胜利 🏆

在本节课中，我们将要学习一种核心心态：**“每个方式都会有赢”**。这种思想旨在帮助你克服接近焦虑，将每一次互动都视为一次成长和胜利的机会。

---

上一节我们探讨了接近焦虑的表现，本节中我们来看看如何通过转变心态来取得胜利。

核心思想是：无论互动的直接结果如何，你都能从中获得积极收益。这可以用一个简单的公式来描述：

**结果 = 获得成功 或 获得经验**

这意味着，每一次尝试的“胜利”并不完全取决于对方的即时反应。

以下是这种心态带来的几种具体“赢”的方式：

*   **直接成功**：你可能会成功找到新的女朋友。
*   **获得学习**：即使结果糟糕，你也能从中学习，这本身就很棒。
*   **收获体验**：事情可能不顺利，但过程有趣，你能得到好故事和好经验。
*   **增强动力**：这种思想是一个很好的激励因素，让你愿意持续行动和学习。

这种思想不仅适用于某一次具体的“接近”方式，也适用于整个社交互动游戏。对于初学者，尤其是那些有思想负担的学生，我会告诉他们：每一次尝试后都要回来告诉我那次尝试好在哪里。

有时女生的反应是笑，有时她们会说什么，有时她们的反应可能很激烈，让人震撼。她可能会感到不舒服，但关键在于，**每个方式都会有赢**。

如果你秉持这种心态，每一次尝试都会带来胜利。你会学得更快、更坚持、更有效。这会让你玩得开心，更想去实践和学习。

“每个方式都会有赢”的思想，能帮助你以同样的行动方式，获得持续前进的动力。我个人很喜欢这种思想。有时，一次我完全没想到的、看似不正常的互动方式，恰恰因为这种心态而变得有价值。

---

本节课中我们一起学习了克服接近焦虑的终极心态：**“每个方式都会有赢”**。记住，无论结果如何，你都在变得更好、更强大。将每次互动都视为一次胜利，你会更自信、更坚持地在社交道路上成长。
