# 托德言语游戏【萌萌团队-皮卡内部提供】：17.第五周（评估）：1-评估：1-评估介绍 🎯

在本节课中，我们将要学习“评估”阶段的核心概念。这是建立实际联系与承诺的第一步，它发生在初步的“娱乐”互动之后。我们将探讨如何判断双方的兴趣倾向，以及为什么这个步骤对于发展一段特别的关系至关重要。

在过去的两个星期中，我们谈论了娱乐。虽然娱乐对游戏来说绝对重要，但它只是一个方法。你不只把女生带到房间里，或是把女生带到关系里。你需要的是，实际的联系、承诺，你和她是特别的一种，或者是在某种方式中合作。

娱乐是一个很好的方法开始。因为如果你正在想，立刻成立一个联系，女生可能还不太在意你与你联系。但如果你先建立娱乐，她可能会想与你联系，你会有很多机会与你联系。

上一节我们介绍了娱乐作为关系的起点，本节中我们来看看如何从娱乐过渡到更深层的连接。今天我们要谈论的，是第一部分的联系，就是“评估”。

评估是你决定她喜欢哪一种。基本上是你两人决定，你喜欢哪一种。但我们做得很好，所以你决定她喜欢哪一种。我们要谈论如何让这件事发生，为什么重要，一些特定的技巧。然后我们会展示，一些现实的互动，当中有些关键性。

以下是评估阶段的核心目标与作用：

*   **目标**：判断双方是否对建立一种“特别”的联系有共同兴趣。
*   **作用**：将关系从泛泛的娱乐，导向有潜力的、独特的双向选择。

或是有些怀疑，谁会先成立，或是先向另一个人表达自己。我们现在进入吧。

本节课中我们一起学习了“评估”阶段的基础。我们明确了评估是在成功娱乐之后，用于判断双方是否愿意建立更深、更特别联系的关键步骤。它为后续的实际联系和承诺奠定了基础。在接下来的课程中，我们将深入探讨进行有效评估的具体技巧和现实案例。
