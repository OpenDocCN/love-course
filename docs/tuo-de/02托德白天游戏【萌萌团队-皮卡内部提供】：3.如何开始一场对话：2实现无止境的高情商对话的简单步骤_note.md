# 课程02：如何开启高情商对话 🗣️

在本节课中，我们将学习如何开始一场对话，并实现流畅、高情商的交流。核心在于理解开场白并非关键，掌握对话开启后的互动原则才是核心。我们将通过一个简单的三步框架来实现这一目标。

## 概述：开场白真的重要吗？

你可能期待本节会提供大量开场白、第一句话或针对女性的示例。我们确实有开场白列表，但需要强调的是：**开场白本身并不重要，开场后的互动更重要**。

你可以用“我爱你”或“我恨你”开场，也可以用“我需要你的意见”或“别进去”开场。开场方式有成千上万种。然而，如果你在开场后就停下来，话题便会终结。开场白的目的是**开启对话**，仅此而已。目的不是用一句话就让人喜欢上你，而是创造一个可以继续交流的起点。

理解这个思想和信念系统会带来好处，并引导对话走向深入。那么，在现实中如何实践呢？

## 核心三步法：闭嘴、玩耍、继续聊

想要实现高情商对话，你需要一个可操作的动作流程。这个流程就是：**闭嘴、玩耍、继续聊**。无论发生什么，都遵循这个流程。如果你能做到这些，你就能很好地完成开场部分，并引导对话走向有意义的交流。

让我们深入探讨每一个步骤。

### 第一步：闭嘴（Speak Up）

当你开始谈话时，你必须**主动发声**。你需要把自己置于一个必须交流的情境中。

如果你过于安静和犹豫，首先，你可能无法引起对方的注意。其次，即使你开口了，对方也可能会想：“你为什么这么安静？你觉得你做的事很可怕吗？你不想被人看到吗？”

关键在于：**如果你认为自己做的事很可怕，对方也会觉得可怕。** 你不需要担心自己的行为是否“可怕”或是否会“很快完蛋”。非常重要的一点是，要相信你所说的话值得被倾听，并自信地表达出来。

这能带来社会认同和接纳等积极反馈，同时也能确保你获得必要的注意力。

### 第二步：玩耍（Have Fun）

在对话现场，专注于**享受互动过程本身**。这能帮助你更好地参与对话讨论。

如果你专注于说自己想说的话，而不是绞尽脑汁地想进入她的世界或按动她的“按钮”，你会感觉更好。你会更有价值感，更自信，更能活在当下。

**做你自己的裁判。** 不要问“我能不能让她喜欢我？”，而要问“我能不能进行一次良好的互动？”。专注于创造良好的互动过程，这是另一个重要概念：**你应永远专注于过程，而非结果**。

你无法控制结果，也无法控制他人。你只能控制自己和过程。如果你做了正确的事，你只需知道自己做得很好，而不必过分在意对方的反应。

所以，专注于“让她喜欢我”这个结果，不如专注于“玩耍”和享受过程。让“她喜欢你”成为“玩耍”的自然结果。专注于过程，结果自会到来。

### 第三步：继续聊（Keep Talking）

无论发生什么，**继续聊下去**。这代表你承担起了延续对话的责任。

当话题自然发生时，你是那个让它保持活力的人。这也意味着，你会把你想表达的内容持续输出。

遵循“闭嘴、玩耍、继续聊”的步骤，你就能很好地开启聊天。对话会持续一分钟、两分钟、五分钟，最终达到真正的交流。然后你便可以坐下来，享受一次美好的聊天。

这些就是你应该在现场思考并执行的动作。你会主动发声，专注于玩耍和当下发生的事，而不是担心她是否喜欢你。然后，无论如何你都会继续聊下去，不让对话冷场。在交流初期，在获得足够多的积极信号之前，你必须坚持继续聊。

## 关于身体语言的补充说明

在本节结束前，我想快速谈谈身体语言。很多人过于关注用身体语言来开启交流。

我们会谈到身体语言，特别是在涉及移动、姿态和角度时。我会分享不同的方式来处理这部分。身体语言和角度确实重要，但更重要的是它们的来源。

**如果你拥有正确的态度，身体语言会自然反映你的态度。** 如果你试图用错误的态度去“表演”身体语言，你会做出非常怪异的事情。

我们曾花费数年时间教学员各种具体的身体语言技巧。这本身非常有意义，你几乎能观察到，当学员开始做“正确”的身体语言时，他们会得到更好的反应。

然而，发生的情况是：学员读到这些技巧后会变得焦虑。然后他们接近女性时，会刻意做那25个“正确”的身体语言动作，结果反而显得很不自然。他们知道“正确”的身体语言，但因为从小没有这种习惯，在焦虑和自我监控下，他们会突然变得僵硬、声音变小，或者过度表演。

后来，我们改变了方法。我们放弃了直接教授身体语言，而是让人们专注于“发声”、“树立正确思想”和“建立理想心态”。突然之间，我们没有告诉他们任何身体语言技巧，他们停止了自我监控，专注于态度和声音。结果是，他们成功了。

我想强调的是：身体语言重要，但**身体语言源于态度**。实际上，我看到当人们做好“发声”、树立正确态度和理想心态时，即使他们的身体语言不完美，他们也能成功。因为如果他们的思想和心态正确，身体语言自然会跟上，或者变得不那么关键。身体语言、角度等，都会随之改善。

## 总结

本节课我们一起学习了如何开启并维持高情商对话。

我们首先明确了**开场白本身并非关键**，真正的重点在于开场后的互动。接着，我们掌握了一个简单的核心三步法：**闭嘴（自信发声）、玩耍（享受过程）、继续聊（承担责任）**。这个流程帮助你专注于可控的过程，而非不可控的结果。

最后，我们探讨了身体语言与态度的关系，指出**正确的态度会自然引致恰当的身体语言**，切勿本末倒置。

记住这些原则并加以练习，你将能够更轻松、更自信地开启任何对话，并引导其走向有意义的深度交流。
