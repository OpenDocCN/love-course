## 0.2 托德的欢迎与课程概述

在本节课中，我们将学习“赢家游戏”的核心概念与学习方法。课程旨在帮助你理解如何将有效的思维和行动模式应用到个人经历中，从而获得更好的结果。

欢迎来到Winter Game。我期待你的加入，并相信你会喜欢这个节目的内容。在开始之前，我想向你介绍Winter Game的起源、内容以及如何利用这个节目来获得最佳效果。

首先，这个节目的内容源于我本人在游戏、商业、运动和学习等领域取得成功的经历。这些经历使我能够与你分享一些见解。

我不希望你仅仅将我视为教练。Winter Game的主要目标是，你所做的一切都应是有效的，并且基于你自身的经历。因此，我希望你通过学习，感觉自己正在听取一位成功人士的分享。

课程中可能会涉及一些概念，但我不希望你只是机械地接受它们。我希望你将我教授的一切概念，与你自己的生活经历相结合，并运用这些概念去审视它们对你生活产生的影响。因为这些概念都源于我亲身验证过的真相。

我确信这些概念会对你有所帮助，并且能与现实世界中的事务相印证。但我希望你相信这些概念，是因为它们本身在现实世界中为你带来的效果，而不是因为它们出自于我。这一点在课程学习中尤为重要。

我所说的一些内容，并非不真实，而是处于教师角色的人所讲授的。因此，我希望你相信这些内容。我是你的教师，也是你的教练，但请不要将我视为无所不知的权威。我知晓许多成功的事情，但我并非通晓一切。

最终，任何决定要融入你生活、特别是你思维一部分的内容，都必须源自于你自身，源自于你自己的生活经历。这是我想强调的第一点。

## 课程的个人性与结构

上一节我们介绍了课程的基本理念，本节中我们来看看这门课程的独特之处与整体框架。

这是我个人色彩最浓厚的一门课程。在过去的工作中，我很少讲授这类内容，原因有几个。

第一，你会看到，过去很多内容我本人并不相信，并且我认为它们像是一场噩梦。因此，当我不同意某些想法时，很难去教授与之相关的内容。

第二，坦白说，我未曾教授过这类内容。这类内容非常个人化，涉及你如何向他人展示自己看待世界的方式、你的思想以及你的个人挣扎。这需要极大的开放性。教授一个技巧是陈述一个事实：你使用它，它就会生效。而教授这类内容，则需要分享个人的故事，需要开放心态。这不是我过去常做的事。

这门课程是我最自豪的成果。我希望我的经历和生活历史能在这个课程中帮助你。

## 课程内容与使用方法

现在，我们来谈谈课程的内容构成以及如何使用它。

课程基本上分为四个部分：
*   **第一课：关于自助领域的谎言与真相。**
*   **第二课：我的个人观点。** 我称之为“内容上的不适”或“不适合输家的内容”。这是在游戏、批评、足球、高层商业和我的学业中，我所观察到的导致输家分心的不同因素。这不仅仅是技术或才华之类的东西，而是让一个人在条件相同时获胜，另一个人失败的原因，甚至是让一个弱者能够击败本不该击败的对手的原因。我们稍后会详细解释这一点。
*   **第三课：我所讲的“进攻性游戏”。** 我会解释每个让人们能够采取进攻、做他们应该做的事以取得进展的难题。
*   **第四课：引导你从无所作为，到进行最进阶、最困难的“进攻”。** 甚至是那些在游戏中经常被回避的部分。

有些东西我认为应该去做，我确信在这个游戏中，我所提供的是最有教育意义、最诚实的内容。这就是我的四部分内容。你可以在任何情况下解读它。

以下是关于如何学习的建议：

我建议先观看《内容上的真相》，因为它会为你提供一个背景和应用的框架。

此外，我建议，如果你长期身处“Pickup Community”或参与过一些自助课程，可以将《真相》影片多看几次，以理解其中的新意。它们包含的观点很新颖，不仅仅是看待事物的方式（看待事物的方式并不重要），它们很新颖、很简洁。或者说内容是好是坏？不，我不说内容是好，但它们有其缺点。所以多看几次，理解其中的新意，特别是如果你长期在“Pickup Community”中或进行一些游戏，你可能需要去解读它们几次。

对于其他影片，我建议你反复观看。每次观看“冠军游戏”影片，都会发现很多深度和因素。你会发现，随着你学习的进步，在不同的时间点，你会找到更新、更深入、更好的教学。在赢的任务中，它就像是剥洋葱。你可能会看到里面的层次，就像外面的层次一样，只是更深入、更有趣。所以我建议你去看它们，再次解读它们。

我不会告诉你必须看多少遍才能理解我使用的措辞、项目以及整个解释。

## 行动的重要性与课程深度

最大的要点，我想告诉你的是，这里有一个行动课程。设置它的原因在于，游戏正在进行中。这是因为，在“冠军游戏”的教学中，我经常强调要有行动和结果。没有行动，就不会带你到达任何地方。因此我强烈建议大家在课程中参与行动部分。

但即使你没有参与行动课程，也必须去尝试。我再次提醒你，在《实话》的《实战》模块中，你无法判断实战是好是坏，除非你去进行一些比较。这对于所有课程内容都一样：你无法完全评估它们，除非你去尝试、去应用、去获得真实的经验。

如果你像我一样，只是坐下来观看这个课程，可能会觉得有趣，也会带走一些东西，学到一些东西，但不会获得全部。只有当你去应用课程内容，然后再回来重温时，才能获得课程的全部价值。这个课程有很多深度，我鼓励你去获取最绝对的价值。

所以说，再次感谢你来到这里。享受所有材料。这真是我心中的东西。我看到了更多人在这里，在“Pickup Community”中，在其他人之中。如果你允许，这应该会完全改变你的原则。
