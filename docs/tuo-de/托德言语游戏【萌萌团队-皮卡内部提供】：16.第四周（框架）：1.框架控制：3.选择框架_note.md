# 托德言语游戏【萌萌团队-皮卡内部提供】：16.第四周（框架）：1.框架控制：3.选择框架_note

在本节课中，我们将要学习如何通过选择不同的“框架”来控制和定义社交互动的意义。框架是一个强大的工具，它能影响你对自己、他人以及整个情境的看法。我们将探讨不同框架带来的好处与坏处，并学习如何有意识地选择对你有利的框架。

## 框架的力量与影响

框架是一个强大的工具，它可以控制一个状况的意义，无论是对自己还是对他人。这意味着你需要了解，选择不同的框架会带来什么好处和坏处。

这两个不同的版本（即不同的框架），各有其好处和坏处。有些好处和坏处是源于自身的，而有些坏处则源于环境、情况或互动中的女性。

## 内部与外部框架的好处与坏处

以下是关于内部与外部框架好处的分析：

*   **内部框架的好处**：内部的画面（即你内心的想法和视角）会影响你的观点。如果你将社交互动视为一个有趣的游戏，这个框架会让你感到放松，并给你带来一些进步的紧张感（即积极的动力）。
*   **外部框架的好处**：强大的框架能力可以让你发展自己。现在你将拥有一个让你更有可能将互动视为游戏的视角。这个视角与女生和你的互动本身没有直接关系。也有可能，你学习游戏（即社交技巧）就像你输了游戏一样，你学习游戏，你会变得不像自己。这可能是你学习过程中的一个阶段。

这可能会影响你的学习过程。另外，你会把游戏（社交互动）当成一个很好的机会，让你发展自己，让你成为一个很好的“玩家”。你会知道这个技巧，这是一个很好的游戏。这是一些很好的例子，让你会觉得这个游戏是有用的。

## 框架如何重塑现实

结合你自己的观点，你会明白，同一个生活情境可以被改变成具有负面、伤害性的框架，也可以被改变成具有正面、帮助性的框架。

你可能会审视你对女生的互动。你会觉得，你对女生的价值和生活的好坏，这会是一个好用的衡量标准。让你更有信心。在互动中，会不会觉得你对女生的价值，会觉得你对她们不值得？如果会这样想，这可能会导致你对她们的负面结果，或者导致女生对你的负面结果。

你会用这些基本的理念，来表达到这个现象。这就像是：谁在互动中给予价值？是你，是她，还是她们？这就是谁有价值。是你，是她，还是她们？谁有价值的你，她？谁在追求？谁想要互动的价值？这就是互动的转折点，是一种胜负的转折，还是一种输的转折。

这就是“你赢了她就要输”，那可能会是一个比较少有亲密关系的圈子。另外，在交际上，有很多圈子是关于你对性情的观点。如果你很害羞、不安心，或不适合性情，那圈子你内在的。如果是在交际上，可能会让女生感受到很多同样的东西，或记录很多同样的东西。

这会使她更少感到性感，更少愿意去玩乐，更少愿意跟你交往，或是一夜之间的交往，或是早晚跟你交往，或是在交往中认识你等等。所以你可以看到，你所想的一切，是被对方的交往所展现的，以及你所看待的交往，是会影响到实际交往的。

## 框架在言行中的体现

所以每一件事，每一件想法，都会有一个具体的影响。这会在不同的方式中发生。你会怀疑或是你感到自信，然后就直接跟着说话。

这会影响到你用的言语选择。例如：
*   **框架A**：你问女生的电话号码。（`你：可以给我你的电话号码吗？`）
*   **框架B**：你问她可以请（索取）她的电话号码。（`你：我能有幸得到你的电话号码吗？`）

框架A会形容她为你做了她的好事。而框架B会形容她为一个不太重要的事。而且框架B会让她给你一个号码，这不是个单向的“好事”，是一种共同的事，因为你俩喜欢对方。

所以这些画面（框架）是被强迫、引用、实施的。不管你用什么词语，它们都在每一件事里，在你的身体语言里，在你的眼神里。而且它们都在你用的词语里。

因此，这会影响到你用的词语。因为每一句你用的词语，不只是代表一个A-frame（单一框架），也有很多A-frame（多种框架）。而改变一句，改变一句，改变一整个A-frame。通常都可以。

本节课中，我们一起学习了“选择框架”的核心概念。我们了解到，框架是定义互动意义的强大工具，它分为内部和外部，并各自带来不同的好处与坏处。通过有意识地选择积极、平等的框架（例如，将互动视为共同参与的“游戏”而非单向“索取”），我们可以重塑对情境的看法，影响自己的言行，并最终改善社交互动的过程和结果。记住，你的每一个想法、每一句话都承载着一个具体的框架。
