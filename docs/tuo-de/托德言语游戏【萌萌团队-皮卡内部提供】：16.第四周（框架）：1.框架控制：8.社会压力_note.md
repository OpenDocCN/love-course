# 托德言语游戏【萌萌团队-皮卡内部提供】：16.第四周（框架）：1.框架控制：8.社会压力 👥

在本节课中，我们将要学习框架控制的最后一个核心概念：**社会压力**。我们将探讨其运作原理、适用场景以及如何正确使用它来促进积极的人际互动，而非用于操控他人。

上一节我们介绍了在决策前或决策中创造情境的方法。本节中我们来看看如何利用时间因素，在决策后施加影响。

## 概述：决策后的影响力

社会压力本质上是一种在他人做出决定后施加影响的方法。它与之前讨论的框架控制类似，但关键区别在于作用的时间点。之前我们讨论的是在决定之前创造情境。而社会压力则作用于决定之后。

例如，你向某人提问后，他们开始陈述观点。此时，你可以通过持续地、明确地表达你的认可（或不认可）来施加压力。这种压力旨在让对方为了维持你的认可，而倾向于做出符合你期望的后续选择。

## 核心机制与公式

社会压力有效的核心机制可以用一个简单的**公式**来描述：

**对方的行动倾向 ≈ 获得认可的欲望 - 失去认可的压力**

这个工具在轻松地表达批准或不批准时非常有用。它让对方去做他们想做的事，但同时会顾虑到可能失去你的认可。这种方法通常只在特定条件下有效。

## 生效的前提条件

以下是社会压力能够生效的关键前提：

*   **已建立好感**：类似于游戏中的大部分互动以及之前创造的框架情境，当你已经喜欢某人或某人已经对你有好感时，这种方法更有效。
*   **对方在乎你的认可**：对方越是在乎你的看法和认可，社会压力的效果就越明显。

## 正确使用指南

鉴于这是一种强大而微妙的影响力工具，我建议你将其用于积极正面的目的。

以下是正确使用社会压力的几点建议：

*   **用于好事，而非坏事**：利用它来鼓励良好的行为和决定。
*   **创造双赢，而非输赢**：目标是获得共同胜利的机会，而不是制造你赢他输的局面。因为这是健康且正确的方式。
*   **避免用于控制**：如果你不断制造输赢对立的情境，长期来看会令人难以忍受。你无法也不应该试图控制一个人的全部。这样做会导致严重的人际关系问题，引发对方对你控制手段的抗拒。

所以，不要试图用社会压力去控制他人。这不会真正有效，在道德上也不正确。我之所以提及它，是因为这些是柔和而有效的方法，应当用于实现你内心关于建立良好关系和积极氛围的初衷。

## 实践应用举例

社会压力可以帮助他人做出更好的决定。关键在于引导的方向。

例如，某人拥有梦想或目标，并且正在为之努力。此时，运用社会压力去**鼓励他们坚持并做好**，远比将社会压力用于其他目的（比如让他们顺从你的个人喜好）要好得多。前者是建设性的支持，后者则可能沦为操控。

**本节课总结**

本节课中，我们一起学习了框架控制的最后一个工具——社会压力。我们明确了它是在决策后施加影响的方法，其有效性建立在已有的好感与对方对你认可的在乎之上。最重要的是，我们强调了必须将这种柔和的工具用于积极目的，如鼓励他人实现目标、营造良好关系氛围，而绝不能用于操控或制造对立。正确使用影响力，是建立健康持久关系的关键。
