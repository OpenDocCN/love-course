# 托德言语游戏【萌萌团队-皮卡内部提供】课程10：初阶者任务4 - 你好 👋

在本节课中，我们将要学习一个非常基础但至关重要的社交任务：向他人说“你好”。这个任务的核心是练习在日常生活场景中，主动、自然地向他人发起简单的问候。

## 任务目标与概述 🎯

上一节我们介绍了基础的社交意识，本节中我们来看看如何将其付诸实践。你的任务是：在一天中，抓住那些简单的机会，向遇到的人说“嗨”。这并非要求你进行深入的对话或打扰他人，仅仅是完成一个简单的问候行为。

## 核心行为准则 📝

以下是执行此任务时需要遵循的核心行为准则：

*   **场景选择**：在点餐、买咖啡或任何日常互动中，向为你服务或在你面前的人说“嗨”。
*   **对象无差别**：问候的对象可以是任何人，无论男性或女性。
*   **从熟悉环境开始**：如果你在办公室工作且不常社交，可以从认识但不太熟的同事开始。明天进入办公室时，当他们看向你，请说“嗨”。
*   **使用名字（进阶）**：如果你知道对方的名字，说“嗨，[名字]”会更好。这是一个更个人化的动作，但我们当前的重点是完成基础的社交交流。

## 任务执行要点 💡

现在，我们来明确一下执行此任务的具体要点。你的当前工作仅仅是说“嗨”，而不是开启一段对话。

*   **行为本身即重点**：重点是你的**行为**——让你在这些简单、明显的情况下说出问候。
*   **无需深入**：这**不代表**你需要“跟着走”或让对方停下来进行长谈。你做的是一项基础工作：与人们建立初步关联。
*   **避免过度商业化**：你不会进行那种刻板、商业化的交流，也不会无休止地“跟着走”。你只是直接地说：“嗨，你好。”

## 可能发生的情况与应对 🔄

有时候，你说“嗨”之后，对方可能会回应“嗨，你今天怎么样？”或者开始与你对话。

*   **享受对话**：如果这种情况发生，这绝对是很棒的。请享受对话，顺其自然。这是我们未来会深入做的练习。
*   **保持核心目标**：但此刻的重点**不是**对话。重点是完成问候这个行为本身。你只是在做简单的、低门槛的社交启动。

## 总结 📚

本节课中，我们一起学习了“初阶者任务4：你好”。我们明确了任务是在日常场景中主动向他人说“嗨”，强调了行为本身的重要性而非对话深度，并区分了基础问候与可能引发的进一步互动。记住，核心公式是：**识别机会 -> 说出“你好” -> 任务完成**。现在，请在你的一天中去实践它吧。
