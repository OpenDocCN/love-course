# 托德言语游戏【萌萌团队-皮卡内部提供】：09.接近焦虑：1.接近焦虑导言 🧠

在本节课中，我们将要学习“接近焦虑”这一概念。接近焦虑是许多人在尝试与陌生人，特别是心仪对象交谈时，会体验到的一种紧张和恐惧感。我们将探讨它的本质、来源以及为何它既是真实的感受，又是不必要的心理障碍。

---

邀请来学习Pickup（搭讪艺术）的学员，在过去的15年里，对我而言一直是个很有趣的问题。一方面，这种焦虑感非常真实；另一方面，它又完全是件蠢事。当我还是Pickup学员时，我自己就深受其扰。

我一直很不喜欢主动去接近他人。虽然我一直想和很多女生聊天，但直到18岁，我从未真正行动过。我总是给自己找借口，比如“她可能很忙”，或者“我不知道该说什么”。即使对方常常表现出可接近的信号，我也会找到一个理由不去行动。这是我成长过程中一个很大的困扰，我总是在内心抱怨自己。

因此，从这个角度看，接近焦虑的感受是**非常真实**的。然而，从另一个角度看，接近焦虑又是**非常不真实**，或者说非常愚蠢的。我敢这么说，是因为我观察到，每当一个学员在某个场合，凭借足够的渴望、勇气或社会压力，最终克服了焦虑并采取行动时，他们总能成功接近。**接近的困难从来不是不可避免的，它显然是你内心的一件事，而非现实世界中的客观障碍。**

但它确实是一条非常真实的心理轨迹。老实说，它源于一个非常真实的进化背景。接近焦虑的根源，在于我们从小在部落社会中的成长经历。在那种环境下，做出不当的社交举动，尤其是向高价值的异性成员表达兴趣，可能会带来非常残酷的社会后果，比如被排斥或惩罚。

你的大脑会想象：如果你被整个群体拒绝，那将是灾难性的。然而，在21世纪这个相对安全的世界里，这份恐惧在很大程度上已经**不合理**了。它是一种过时的本能反应。

所以，我想让你们明白：**接近焦虑是不合理的**。感到焦虑并不意味着你是个坏人、不是真男人、或者没有生命力。这只是一个需要被理解和改变的心理模式。我明白这种感受非常强烈且真实，但从逻辑上看，它现在已不合时宜。

这就像任何恐惧一样，恐惧本身源于一个自然的地方（自我保护），但可能在当下情境中已经失去了意义。我记得小时候第一次要跳过高台跳水。站在跳台顶端往下看时，我感到极度害怕。整个游泳池的人似乎都在看着我。但当我最终说服自己跳下去之后，我又想再跳一次。

**接近恐惧就应该是这样：它是一种恐惧，但恐惧中蕴含着激情，也能转化为一种好的动力。**

接下来，从本导言到整个关于接近焦虑的课程，我会提供五种不同的视角来理解它，其中一些涉及心理层面，一些涉及实际操作。我们将探讨如何拆解这个问题，以及可以用哪些方法来应对接近焦虑。

然后，我们会有两套不同的实践课程：
*   **第一套实践课程**针对那些几乎没有接近经验、需要从基础开始的学员。它将引导你从较容易的体验，逐步过渡到较难的体验，直到你完成一次完整的接近。
*   **第二套实践课程**则针对那些已经有过接近经验，但在特定情况下仍会感到焦虑的学员。例如，普通的一对一接近可能没问题，但接近一个团体中的女生、或者有男性同伴在旁的女生、又或者在众目睽睽之下行动，这些情况可能依然令人生畏。

总而言之，我们将探讨五个哲学观点、五个物理/实践方法，并提供两套共八个实践任务。我不会讲得过于深奥，主要是想让大家明确三点：
第一，感到接近焦虑是完全正常的。
第二，它主要存在于你的脑海中。
第三，它是完全可以被克服的。

请跟随我一起学习这门课程，我将为你们指明道路。
