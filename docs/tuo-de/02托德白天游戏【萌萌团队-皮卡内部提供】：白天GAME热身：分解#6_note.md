# 课程02：托德白天游戏解析 🎯

在本节课中，我们将分解一段名为“托德白天游戏”的实战互动记录。我们将学习如何在实际对话中建立联系、进行升级，并理解自然互动背后的核心原则。课程内容基于一段真实的白天搭讪场景，我们将逐句分析其策略与思维过程。

## 概述：场景与开场

我们再次回到第一天的场景。这是白天游戏的一部分。对话始于简单的问候：“你好，你好吗？谢谢，我是托德。你很可爱，我来跟你聊天。我没有化妆，很累，我只是想放松一下。”

上一节我们介绍了基本的开场，本节中我们来看看如何从简单的问候过渡到更有趣的互动。

## 核心互动：赞美与升级

我表达了对她口音的喜爱：“我喜欢你的口音，真的吗？你听过最差的口音吗？”

她问：“最差是最差的还是最酷的？”
我回答：“最差是最酷的，因为它是从……我听过别人说话，我总是觉得……噢，糟糕。”

以下是对话中一个关键的转折点：
*   我分享了一个个人故事：“首先，我的家人是德国人，我是美国人，但我的家人是德国人。”
*   我给出了一个观点：“我喜欢德国人的口音，其实是最酷的。好，所以它是酷的。”
*   我进行了试探性的升级：“有点酷的？是的，我觉得有点酷的。你有很多酷的吗？”

我们在这里发生了什么？我进来的时候，很平静、很自然，好像我认识她，就像她没什么大不了。然后她实际上立刻像是在帮助我，她就像是在开自己玩笑。我实际上就是在评论她的口音。当她说话的时候，我其实很喜欢她的口音。

## 策略解析：安全地表达兴趣

我最初觉得她的口音是……首先，我认为她很酷，但我还觉得她的口音是非常酷的。所以我只是在那一刻，就说出了一句赞美。

你想想在那一刻，如果女生做了一些事，真正地说，那是非常酷的。为什么不说呢？如果她有让你觉得她很酷的特质，你就说出来。现在，你觉得你很酷，我会做任何事情。我会把我的心给你。我不用再去评判你。

明白这个：你应该有一个“你很酷”的心态，这只会帮你打开大门。你不需要向她证明自己。所以这很重要，这是一个好处。

然后还有一点，推拉声明是很重要的。这是我展示的：我讲她的口音，但我真的在说她。我讲她，因为她的口音是她的一部分。但这很棒，因为这比较直接。如果我只是说“你看起很酷”，这就比说“我喜欢德国的口音”更个人化。但后者还是关于她的，所以这是一个更安全的表达方式。

再说，任何一个方式的升级，这是我喜欢做的。我喜欢走，我喜欢够清楚地表达自己在升级，但我喜欢保持安全，然后得到反应。如果她喜欢，或者不喜欢。如果她不喜欢，我可以再升级多一点。但降级很难，所以你要不断地、一直在升级。你不想坐着不升级，但你想通过小小的升级，让你能够一直加到正常的程度。

## 对话推进：建立合理性与联系

当她回应正常后，我会更个人化地说：“你有点‘脏’吗？”（这里指口音有趣、不标准）。她会说“我的口音很脏吗？”，我会说“你很脏吗？”。现在是个人的，现在是非常有意义的。但我只做了这个，当我觉得她给了我许可，如果这很合理。

“对，我们可能会在一起，这很奇怪。” 所以你再给我记下名字，因为当你讲话的时候，我只是在听你的口音。我只是在听你的口音。

好，你在公园里做什么？我在散步，在跟朋友见面。所以现在我们都处于一个合理的状态。我坐下来，她在问我问题，她在放松的地方。这很合理，很热情，很好的情况。

我现在很开心，我看到你，我被你启发来打招呼。现在我把这个切开，知道这是一个更长时间的交流，总共有20分钟，可能20分钟到半小时。但我只在5分钟内展示给你，所以我只是在展示给你这类游戏和升级的部分。

知道这里有切开的时间，基本上只是一堆她跟我的、比较自然的细节。这很普通，这没什么特别，但知道这很重要，因为这让互动很紧密。当结合其他游戏技巧，比如有些人认识你，这很重要。

## 未来规划与收号

你在（这个）国家做什么？我在学习。这很棒，你是学生或是MBA？不是MBA，我是学生。好的，我同意，这也很棒。她正在谈话中。

我同意，我正在想，在未来做一些很相似的事。你是学生学员还是学生？我只剩下三个星期。我们很快会成为朋友。这么直接的方式，很不常见。如果我们有三年，我可以帮你暂停，躲在你棵树上。但只要我是一个男人，就说“我们成为朋友”。

是的，你两星期后，有多少人认识你？我很多时间去旅行，其实… 我们有办公室，这是我们第二大型的公司。你所在哪里的公司？我们是真正的社交动力，真正的社交动力。我们教大家学习社交，公开谈判。明白，朋友。

你待在纽约？我待在芝加哥。但我下星期会回到纽约。我们可能要去吃一口饭，或是喝一杯。下星期喝一杯。事实是，在纽约后，我们有我们的决赛，我的表演，还有两份书籍，在7月20日。我… 7月7日，我再次有空。好的，我们7月7日再谈。7月38日，好像是个计划。

## 文化差异与注意事项

这是一个有趣的小提醒：如果你和外国女生交流，不要用这样的词。不要告诉外国女生，你会得到一大堆的“烤饼”。这不是很多人会这样说，但这只是愚蠢的词。这不太重要，因为她是受过教育的，她英语说得很好，她会理解。但特别是，像一个非英语母语的女生，如果你说一些俚语，你可能会把她搞得很误会。所以要小心这种东西。这只是建议，把复杂的语言放下，至少用简单的词，而不是用流行文化的俚语。

“如果我们不见面，那我们会在夜晚，在车上过去的车。” 听过这句话吗？没有。我确实有手机。

## 结束互动与总结

每次都要… 是的。大家好，我是笑谭，我是罗… 你现在有两台手机吗？是的。你现在在警察局吗？我妹妹到了。好的，她要去哪一分？不，我想现在就要去。你号码呢？

这件事是一个很细致的细节：她要去等她姐姐在机场。她在等她姐姐在机场，等到那时候，她刚接到她姐姐的电话，现在要去等她姐姐。所以这件事很难做，比起要号码多花两秒钟。好的，我现在在打给你。

所以当她走过，就叫我托德，很德文。我听到有很多德文。什么？我不知道那些德文，但没有R。奇怪，我猜是美国人，但没有R，没有人知道。是的，所以那是I。好的，很高兴认识你，你也好。谢谢，很舒服吧？

她在打给我姐姐。去找你姐姐，我去找我的朋友。你们两个，祝你们好。你们两个，我们很快见。

## 核心要点与最终建议

我马上把手伸到麦克风上，这样它不会再掉下来。这只是一个很普通的方法，不算太复杂。但我想要示范的是：她是一个很吸引人的人，而且她知道你不需要做太多。这就是白天游戏的美丽，因为没有高级的球场和很多的（夜店）气氛。

如果你是普通人，你可以透过好的形态和好的气氛，得到很好的结果。特别是在这种情况下：如果我看到一个女生坐下来，躺下来，在一个地方上，在一天中，在放松，她可能会很难（被搭讪）。但这其实是最容易的情况，这（反而）是你想要的。

这是一个不相关的故事，但我遇到的最难（搭讪）的女生，是在一个演艺学院的演技学院。我早上到学校，她早上到学校，她是第一位，在我等候的地方，我第二位。她和我坐在门外，我们都要等候，没选择说话，而且我们有共同的兴趣。我走过后，我发现这个情况，我看起很不理性。我跟她说谢谢，因为我只知道好事会发生。

这种（放松的）情况，很容易，但也很幸运。很少会遇到一位女生自己在做什么。但当你这样做，你（的表现）应该会更好。

**本节课中我们一起学习了：**
1.  **自然开场**：从简单问候过渡到个性化互动。
2.  **安全升级**：通过赞美对方特质（如口音）来表达兴趣，同时保持可退让的空间。核心是 **“小小的、持续的升级”**。
3.  **建立合理性**：通过询问背景、分享计划，让互动和收号显得自然合理。
4.  **文化敏感性**：与外国女生交流时，避免使用难以理解的俚语，用词应简单清晰。
5.  **白天游戏优势**：白天场景中，对方状态更放松，互动压力更小，**好的形态和自然的气氛**往往比复杂技巧更有效。
6.  **抓住机会**：最理想的搭讪场景往往是对方独自一人、处于放松状态时。
