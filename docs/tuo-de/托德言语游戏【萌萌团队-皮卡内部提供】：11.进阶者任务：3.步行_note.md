# 托德言语游戏【萌萌团队-皮卡内部提供】：11.进阶者任务：3.步行_note

在本节课中，我们将要学习如何与正在行走中的女性进行互动。这是一个进阶技巧，关键在于理解她们的状态并采取合适的策略。

上一节我们介绍了静态环境下的互动，本节中我们来看看当目标处于动态时，情况有何不同。

## 动态互动的挑战

当女性在移动时，例如正在走路，直接上前搭讪可能会遇到阻力。她们可能会说自己很着急，或者表示不想被打扰。最受欢迎的女性通常在移动中，因为她们有明确的目的地，也有想见的人。

她们也清楚，如果停下来，可能会面临被搭讪的情况。我认同这一点。即使她们原本可能愿意停下来喝杯咖啡，但因为正处于移动状态，她们会倾向于拒绝。她们会给自己设定界限，而很多男性没有意识到这一点。

如果女性在行走，她们常用的回应是“我在赶时间”或“不想被打扰”。

## 两种核心策略

因此，与移动中的女性互动，需要学会两种方法。

以下是两种核心策略：

1.  **跟随她们行走**：即一边与她同向行走一边交谈。
2.  **邀请她们停下来**：即尝试让她暂停脚步进行交流。

这两种方法我都喜欢。但如果女性在移动中，她们最初可能会以“很着急”为由拒绝。关键在于你的应对方式：如果她们说在赶时间，你可以尝试邀请她们停下来；如果她们在走，你可以选择跟随她们一起走。

但最重要的是理解一个核心概念：**移动中的女性依然是普通人**。

## 理解核心：共情与换位思考

因为女性在移动，她们同样会感到被打扰的压力。这里有一个简单的换位思考公式：

```
如果 你很着急
并且 你最喜欢的人（如超级模特或偶像）过来找你聊天
那么 你可能会说：“不，我现在很忙。”
```

假设你很着急，你会说“我急着要走”。但如果你愿意，也可以说“请跟我一起走”。女性也是一样的逻辑。

如果你是一个高价值的男性（“高级模特”或“好男生”），她们其实想和你聊天。你只需要**给她们一个机会**。

## 你的任务与练习

你的核心任务是：**练习与移动中的女性互动**。

与她们一同移动，并解决“她在赶时间”这个常见问题。还有很多技巧可以让这个动作变得更容易，但我希望你们先去实践。

技巧只会在你拥有“球”（指勇气和机会）并开始练习时才会出现。只要你开始行动，就能练习。
