# 托德言语游戏【萌萌团队-皮卡内部提供】：11.进阶者任务：5.公众视线_note

在本节课中，我们将要学习如何在有他人在场的公共场合，克服心理障碍，自然地与女性展开互动。这是一个关键的进阶挑战，能帮助你在真实社交场景中建立自信。

## 挑战概述

第四个挑战是一个巨大的挑战。没有人愿意在众目睽睽之下，用冰冷的方式去接近他人。很多时候，如果有一个女生单独坐着或站着，你可能会去和她聊天。但假设有几个人坐在两三米远的地方，你会感觉非常奇怪，不想去做这件事。然而，这可能是你的唯一机会。

## 核心心态与策略

上一节我们介绍了挑战的基本情况，本节中我们来看看应对的核心心态。你必须完成这个挑战。例如，你坐在地铁站，周围也有其他人坐着，你可能永远不会再见到那个女生。这就是你必须做的事情。

我建议你以最开心、最好、最社交化的方式去做。如果你担心她会觉得奇怪，就不要一开始就表现得太过分，尤其是在有其他人在场的情况下。

你也可以把周围的人看作是她的“团队”，但浪费这个机会是很愚蠢的。我向你保证，不会有人因此责怪你。因为你在做一件对她有益的事——前提是你是一个友善的人。如果遇到不友善的人给你制造困难，那可能是因为他们自己心情不好，他们可能想自己接近她。

## 适用场景列表

以下是适合进行此项练习的一些典型公共场所：

*   地铁站
*   火车站
*   公共汽车站
*   健身房（女生可能在跑步机上，其他人在使用别的器械，这也是一个很好的机会）
*   其他公共区域
*   办公室等场所

这些地方通常有很多人，女生也常常会和别人在一起。如果你想要练习在这种情况下的互动，健身房是另一个绝佳的机会。

## 最终行动指南

不要担心别人的想法。当女生处于公司或团体环境中时，你必须去接近她。因此，要做好准备，以最好的状态去行动。
