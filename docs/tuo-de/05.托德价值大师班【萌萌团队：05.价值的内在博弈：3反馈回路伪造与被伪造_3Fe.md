# 05.托德价值大师班【萌萌团队：05.价值的内在博弈：3反馈回路伪造与被伪造_3Fe

在本节课中，我们将探讨价值创造过程中的“内在游戏”与“外在游戏”，并深入理解“反馈回路”如何在这两者之间运作。我们将学习如何通过行动启动一个积极的循环，从而加速个人成长与价值实现。

---

上一节我们介绍了内在游戏与外在游戏的基本概念。本节中，我们来看看它们之间是如何通过“反馈回路”相互作用的。

这个内在游戏和外在游戏的概念，引出一个问题：应该学习技术，还是应该学习内在游戏的内在思考？我明天会详细讲解内在游戏，现在也会简要提及。但我个人主要是一个专注于外在游戏的实践者，原因就在于下面这张“传送图”。

**传送图公式：**
`外在行为 -> 现实结果 -> 内在状态强化 -> 更佳外在行为`

以下是该反馈回路的具体解析：

你可以观察内在游戏，在这个中间游戏上，你可以观察外在游戏的行为。你正在真实世界中行动，你所做的事情会最直接地决定结果。你的行为会导致相应的结果。

有趣的是，结果会反过来改变你的内在游戏。如果你取得了好的结果，这会增强你的内在游戏。这会让你的内在游戏状态，使得做出良好行为变得更容易，或者让你的行为本身变得更好。这就是我所说的“进一步的进步”。

这就是我所说的“假装做到，直到真正做到”。去外部世界练习，获取结果，然后以正确的方式做事，取得正确的成绩。你正在创造好成绩，你也在妥善处理失败。因为通过这些做到的事，你会启动这个“传送图”。由于这是一个自助循环，你的起点在哪里并不重要。

你可以从一个很低的行为层面开始，也可以从一个很低的内在游戏层面起步。如果你启动了这个传送图，你就会做出你的好事，你会得到预期的、并且是快速的传送。

---

本节课中，我们一起学习了价值创造的“内在博弈”。核心在于理解**外在行为、现实结果与内在状态**三者构成的**反馈回路**。关键在于立即行动，通过外在实践获取反馈，从而正向强化内在信念与能力，形成一个自我加速的成长循环。记住，起点不重要，启动这个循环本身才是关键。
