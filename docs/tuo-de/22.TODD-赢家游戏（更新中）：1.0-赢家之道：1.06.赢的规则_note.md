# 课程22：TODD-赢家游戏（更新中）🎮 1.0-赢家之道：1.06. 赢的规则⚖️

在本节课中，我们将要学习“赢的规则”这一核心概念。无论是在运动、游戏，还是在商业、社交乃至生活中，了解并掌握规则是取得胜利、避免失败的根本前提。我们将通过多个领域的实例，阐明知晓规则的重要性。

## 理解规则的重要性

在任何一场竞争或游戏中，你都必须知晓规则。这一点在体育运动和游戏中最为显而易见。

以下是几个来自体育界的纯粹例子：

*   **篮球实例**：在国际赛事中，球员克里斯·韦伯在球队已没有暂停的情况下叫了暂停，这导致了技术犯规，从而使他的队伍输掉了比赛。
*   **足球实例**：如果防守球员移动稍慢，进攻方会将队伍阵型前压，快速开球发动进攻。如果球场上一方球员过多，裁判会首先自动判罚。这是一种战术，你必须比对手更清楚规则。

## 规则不清导致的意外失败

另一个令人费解的例子发生在足球赛事中。由于赛事规则，获胜队伍的球员自己打进了一球（例如乌龙球），然后将球开出。因为规则使然，这一行为反而可能让对手打得更好，结果对手抓住机会进球并取得了胜利。只要有球员遵循了（对自己不利的）规则，就可能发生这种疯狂的情况。

这些奇怪的场景和意外的机会，只会在你不清楚规则时出现。如果你不清楚规则，你将被这些情况淘汰出局，而无法使其对你有利。

## 规则超越运动与游戏

这种规则意识甚至超越了运动和游戏的范畴。

例如在商业或金融领域，如果你不清楚规则并违反了它，你不能以“我不知道”作为借口。你必须清楚自己在做什么，必须了解正在发生什么。**你必须知道规则，否则你会陷入大麻烦。** 因此，你必须了解你所参与的“游戏”有何种规则。

## 社交与两性互动中的规则

这一点在社交游戏和与女性的互动中也至关重要。

在社交和与女性的互动中，你需要了解最显而易见的规则。在任何情况下，一个最根本的规则是：女性不希望被当作小丑一样对待或显得尴尬。

例如，你不会直接问“你想跟我回家吗？”。你会说：“我知道一个安静的地方，我们可以去喝一杯。”你会采取更有效、更得体的行动。

又例如，当你带女性回家时，你不会让她感到目的过于明显，因为那会让她感觉不好。你会让整个过程显得更自然、更轻松，这样她才能自然地跟随你。这涉及到对女性心理的理解和互动方式的调整。

## 高阶社交场合的规则

还有其他理念适用于更复杂的社交环境。

例如，如果你想进入高级夜店或宴会，你需要知道该做什么、该穿什么、该和谁交谈，甚至是如何去到那里并与女性交流。当你进入这样的场合，就如同进入一个不同的“地盘”，你必须知道如何行事，才不会让自己被请出去，同时还能顺利地与女性交流。

这是一个很好的例子：如果你正在与女性交流，比如与酒店员工、服务员或女招待互动，我会告诉你两件我生活中的相关经历。

上一节我们介绍了规则在社交中的普遍性，本节中我们来看看具体实例。

以下是两个个人经历：

1.  当我进入高级夜店时，我常去与在那里工作的女性交流。我会让她们真的愿意跟随我，我们能有很好的互动，她们似乎真的很喜欢我。然而，当这种情况变得过于明显时，店方管理者会叫她回工作岗位，之后她可能就不再回来。她会选择跟着我离开，这有时会让我感到奇怪甚至不安，不知该如何处理。经过多次经历后我发现，**规则是：她不应该在工作时间随意离开岗位**。因此，我必须找到一个合理的应对方式。我会对她说：“你知道吗，我想和你聊聊，但我不希望你因此和老板产生问题。我们不如换个地方，这样我们可以更自在地交流。”
2.  另一个例子是我第一次与从高级夜店认识的女性进一步发展。基本上，你不能直接在那种场合进行深入交流，因为那样显得不妥。所以我们在社交媒体上交流，然后约定见面。当她放下行李后，就跟着我离开了。但我必须确保整个交流过程遵循一定的“规则”，使她不会因此惹上麻烦。

## 核心总结

在本节课中，我们一起学习了“赢的规则”这一贯穿所有竞争领域的核心原则。

**核心公式是：成功 = 知晓规则 + 运用规则。**

不管你所参与的是体育游戏、社交互动、两性关系，还是人生这场大游戏，**你必须知道规则**。清晰了解规则能让你避免无谓的失败，识别潜在的机会，并将局势导向有利于自己的方向。这是成为赢家的基础之道。
