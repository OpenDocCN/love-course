# 托德言语游戏【萌萌团队-皮卡内部提供】：04.游戏时代：1.没有游戏的时代_note

在本节课中，我们将探讨“没有游戏的时代”这一概念。我们将了解在“游戏”作为一种明确技巧体系出现之前，人们（尤其是男性）在社交与两性关系中的普遍状态与困境。

## 概述

本节将描述四种类似但有所不同的“游戏”形态在历史中的演变。首先从最短、最初始的形态开始，这或许也是世界上最根本的想法之一。

## 没有游戏的时代

上一节我们概述了课程内容，本节中我们来看看“没有游戏的时代”具体指什么。

这个想法在互联网普及之前就已存在，并且在网上被广泛分享和讨论。这是许多人深信不疑的观念：如果你想吸引女性，或者想成为足球队长，你需要做好工作、开上好车。这就是社会向我们推广的所谓“市场价值”。

这代表了我们向往的生活方式以及吸引女性的方式。然而，许多人在这方面遭遇失败。失败的原因可能有很多：也许你实际上取得了成功，做了很多事，但你可能会发现自己缺乏社交技巧，或者根本没有去接触女性。这并不意味着你就能吸引到女性。

又或者，你可能属于那种并不轻松自如的人。也许你不是天生最有魅力的男性，或者你现在没有最理想的工作，而你又不愿仅仅为了吸引女性而改变自己。这不是另一种方法，而是属于那个“没有游戏的时代”。

## 个人经历：足球与社交

以我个人的足球经历为例。我十八岁开始踢球，本应是一个成功的年轻人，但我当时并非如此。我是一名足球运动员，足球是一项重要的运动。我的学历不错，外貌也尚可——不算出众，但也不差。然而，我并未成功吸引到女性。

为什么呢？因为我没有学会社交，没有学会应对批评，没有掌握这些技巧。这是许多人的通病，也是“没有游戏”的原因所在。当时，“游戏”并不存在，影响结果的是其他因素。

## “没有游戏”的负面影响

这种状况实际上严重阻碍了学习进程。它导致许多男性放弃，不再去追求和探索两性关系的奥秘。这让男性固步自封，停留在自己的舒适区。

因此你会看到，有些男性只是自以为能吸引女性，或者你会看到一些肤浅的“把妹学问”。例如，我成长过程中看到的《Maxim》这类杂志。这就是我成长的环境，也是我最初未能成功的原因。

## 游戏作为技巧

我不想在此赘述失败经历，因为既然你在这里，在我的频道和这个节目中，你就会明白“游戏”是一套技巧，是一件可以学习的事情。

但我想指出，世界上大多数人并不这么认为。世界上大多数人认为“游戏”是真实存在的、与生俱来的东西。这是你拥有的巨大优势：你是世界上少数正在积极追求、并在这个领域取得进步的人之一。

## 本节总结

本节课中，我们一起学习了“没有游戏的时代”这一概念。我们看到了在明确的社交技巧体系出现之前，人们往往依赖外部条件（如工作、车辆）并忽视内在社交技能的发展，这导致了许多人在两性关系中的困惑与失败。认识到“游戏”是可以学习和提升的技巧，本身就是一个重要的起点。
