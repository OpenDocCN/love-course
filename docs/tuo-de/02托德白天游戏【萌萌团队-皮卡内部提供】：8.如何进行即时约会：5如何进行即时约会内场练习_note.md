# 课程02：托德白天游戏 🎯 - 8. 如何进行即时约会：5. 即时约会内场练习

在本节课中，我们将学习如何进行即时约会的内场练习。这些练习旨在帮助你通过具体的行动和思维训练，在现实互动中创造更多约会机会。我们将重点介绍三个核心练习方法。

上一节我们介绍了即时约会的基本概念，本节中我们来看看具体的内场练习方法。

## 核心练习一：主动测试与价值承担 🧪

这里有些在场的运动可以帮助你获得更多的机会。第一个练习是主动测试女生。很多时候我做热身时，我会开启一段对话，直接询问她：“为什么你和我有关系？” 我听到很多疯狂的想法，因为你承担了价值。

因为你做了不循规蹈矩的事。真正的答案是因为我问了她，她才和我产生联系。因为我问了，如果我没有问她，她就不会做。现在所有的互动都变成了一个对的框架，但是你只需要问她。这就是我们在这两个运动中要做的：我们只需要教你如何问她。

以下是第一个核心练习的步骤：

*   **定时测试**：每次你跟一个女生聊天，达到10分钟或更久，你一定要试试看她。你甚至可以设想一个“时光机”场景，但核心是每次聊天约10分钟时，你必须尝试测试她的兴趣度。
*   **开启“时光机”**：你必须从第一个“时光机”场景开始思考。你必须想：“我怎么让她进入这个场景？” 你甚至可以在聊天开始10分钟时就引入，或者由她开启话题。如果她反应较快，比如开始问你问题，这通常是她对你感兴趣的信号。
*   **快速迭代实验**：第三个方法是更快速地设想“时光机”，这是一个技术性实验，会帮助你推进关系。关键在于，你需要从第一个“时光机”场景开始构思。

## 核心练习二：洞察与时机把握 🔍

从第一个“时光机”场景开始，或者可能不是第一个，但关键是那个能“让她喜欢你的时光机”。所以你需要知道：她在做什么？她的计划是什么？她的整体安排？她接下来要去哪里？这些是你需要知道的小部分信息。

但你必须知道这些，比其他事情更重要。她现在有多少空闲时间？以及她的立即计划是什么？如果你知道这些小部分的信息，你会知道什么时机的“时光机”会有效，你也会知道哪些“时光机”场景会有什么用。

有些时候，你不应该强行推进“时光机”，因为你太着急。在这些情况下，你只需要拿到电话号码，但了解上述信息是重要的，你应该在每一组互动中都设法了解。

## 核心练习三：在计划中实践 🏃

所以在这些计划中练习。将洞察对方状态和选择恰当时机，融入到你的每一次互动尝试中。

本节课中我们一起学习了即时约会的三个内场核心练习：**主动测试以承担价值**、**通过“时光机”框架引导互动**，以及**洞察对方状态以把握时机**。记住，练习的关键在于**主动询问**、**框架构建**和**信息收集**，并在实际互动中灵活运用。
