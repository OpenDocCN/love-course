# 托德-《吸引力解锁》课程22：剥夺认同 🧩

在本节课中，我们将要学习一个关于沟通与吸引力的核心概念——“剥夺认同”。我们将探讨如何在对话中通过特定的回应方式，来创造吸引力和张力。

上一节我们介绍了对话中的基本互动模式，本节中我们来看看一种更进阶的技巧。

## 核心概念：什么是“剥夺认同”？

“剥夺认同”是指在对话中，不轻易给予对方所期待的肯定或认同性回应。其核心在于**不直接、不完整地满足对方寻求认可或延续话题的意图**。这并非冷漠，而是一种创造张力和吸引力的策略。

## 关键方法与原理

以下是实现“剥夺认同”的几个具体方法和其背后的原理。

### 1. 后撤与接受
首先，任何回应（包括批评）都可以带有不同的“重量”和价值。关键在于，你的回应不应是简单的附和，而应具有独立的价值判断，甚至带有一种“奖励”或“惩罚”的属性。

### 2. 静默与留白
许多人在与女性对话时会不停说话。但即使女性想对对话内容提出批评或看法，她也可能无法插入。如果对话者一直说话，不给对方空间，那么对方就很难参与进来。因此，**主动留下静默和空间**，让对方来填补，甚至让沉默持续到略显“奇怪”的程度，这种张力会非常强烈。

### 3. 通过批评来回应
我们都知道，直接批评对方通常是不好的。但这里的“批评”并非针对个人或主题，而是一种**特别的、带有压力的回应方式**。例如，对对方的某个观点或行为进行轻微、特别的点评，这能制造强烈的互动感。这属于外在吸引力游戏的一部分。

### 4. 吸引力的其他形式
其他形式的吸引力可能表现为不同的风格。例如，你可以使用像 **`Mini Coldreads`**（快速冷读）这样的技巧，通过评论对方说话的方式或特点来制造吸引力。但本质上，**任何具有独立价值的、不迎合的回应，都可以看作是一种“剥夺认同”**，它是构建吸引力的基础。

## 总结

本节课中我们一起学习了“剥夺认同”这一沟通技巧。我们了解到，它并非不友好，而是通过**后撤与赋予回应以价值**、**善用静默创造空间**、以及**使用特别的“批评”式回应**来构建对话中的张力与吸引力。掌握这一技巧，能让你的互动更具深度和吸引力。
