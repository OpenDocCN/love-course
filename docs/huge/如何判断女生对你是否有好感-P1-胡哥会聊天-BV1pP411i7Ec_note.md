## 一、眼神接触 👀

上一节我们介绍了课程的整体框架，本节中我们来看看第一个关键信号：眼神接触。眼睛是心灵的窗户，能传递丰富的情感信息。

以下是判断眼神接触是否包含好感的几个要点：

*   **对视频率**：如果女生经常与你进行眼神交流，并且对视时间超过普通社交的1-2秒，这可能是一个积极信号。
*   **瞳孔变化**：当人对感兴趣的事物或人时，瞳孔会不自觉地放大。公式可以表示为：`兴趣程度 ∝ 瞳孔放大程度`。
*   **眼神躲避后的回看**：她可能会在与你对视后害羞地移开目光，但不久后又再次看向你，这种“回看”行为值得注意。

通过观察这些细微的眼神变化，你可以初步感知对方是否对你保持关注。

## 二、身体语言 💃

在了解了眼神的信号后，我们进一步观察她的身体语言。身体会不自觉地暴露真实感受。

以下是体现好感的身体语言特征：

*   **身体朝向**：她的身体（尤其是脚尖和躯干）是否自然地朝向你的方向。`身体朝向 = 兴趣指向`。
*   **距离拉近**：在交谈或相处时，她是否不排斥与你缩短物理距离，或主动进入你的个人空间。
*   **模仿动作**：她会下意识地模仿你的小动作，比如同时端起杯子喝水或调整坐姿，这是一种建立默契的表现。
*   **整理仪容**：在你面前，她可能会频繁地整理头发、衣服或妆容，希望在你眼中保持良好形象。

身体语言比言语更诚实，是判断好感度的重要依据。

## 三、对话互动 💬

最后，我们来分析对话中的互动模式。语言内容和交流方式能直接反映她的投入程度。

以下是对话中蕴含好感的几个迹象：

*   **主动开启与延续话题**：她是否会主动找你聊天，并在对话中通过提问（“然后呢？”、“你呢？”）来延续话题，避免冷场。
*   **分享个人细节**：她愿意与你分享个人生活、感受、家庭或未来的计划，这表示信任和亲近。代码逻辑可类比为：`if (分享深度 > 日常寒暄) { 好感度++ }`。
*   **积极回应与支持**：对你的经历和观点，她给予积极的回应、鼓励或支持，而不是敷衍或否定。
*   **使用昵称或调侃**：她可能会给你起一个专属的昵称，或用轻松的方式调侃你，这是关系亲近的表现。

积极的对话互动是双方情感连接不断加深的证明。
