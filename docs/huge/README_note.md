# 课程名称：胡哥的编程入门课 - 第1课：变量与赋值 🧑‍💻

在本节课中，我们将要学习编程中最基础也最重要的两个概念：**变量**和**赋值**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子用来存放数据，比如数字、文字等。标签就是变量的名字，通过这个名字，我们可以在程序里找到并使用盒子里的数据。

在大多数编程语言中，创建一个变量（即准备一个盒子并贴上标签）的过程叫做“声明”。

## 如何给变量赋值？➡️

仅仅有一个空盒子还不够，我们需要把数据放进去。将数据存入变量的操作就叫做“赋值”。这就像把具体的物品放进盒子里。

赋值操作通常使用等号 `=` 来完成。但要注意，在编程中，`=` 通常不表示“等于”，而是表示“将右边的值赋予左边的变量”。

以下是赋值语句的基本格式：

```
变量名 = 值
```

例如，如果我们有一个叫做 `age` 的变量，我们可以这样存放数字 `25`：

```python
age = 25
```

这行代码的意思是：将数值 `25` 放入名为 `age` 的变量盒子中。

## 变量的核心特性 🔑

理解了基本操作后，我们来看看变量的几个关键特性。

以下是关于变量的三个重要特点：

1.  **变量可以反复赋值**：盒子里的东西可以更换。例如，先执行 `age = 25`，再执行 `age = 30`，那么变量 `age` 里最终存放的值就是 `30`。
2.  **变量名具有唯一性**：在同一作用域内，一个变量名通常只指向一个“盒子”。后一次的赋值会覆盖前一次的值。
3.  **变量需要先赋值再使用**：在尝试获取一个变量盒子里的内容之前，你必须确保已经往里面放了东西。否则程序可能会报错。

## 一个简单的例子 ✨

让我们通过一个简单的例子把上面的概念串联起来。假设我们要计算两个数字的和。

首先，我们声明两个变量 `a` 和 `b`，并分别给它们赋值：

```python
a = 5
b = 3
```

现在，我们声明第三个变量 `sum`，并将 `a` 和 `b` 这两个“盒子”里的内容相加，结果存放到 `sum` 这个新盒子里：

```python
sum = a + b
```

此时，变量 `sum` 中存放的值就是 `8`。我们可以通过打印它来验证：

```python
print(sum) # 输出：8
```

这个例子展示了如何通过变量来存储中间结果，并组织程序逻辑。
