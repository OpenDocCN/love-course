# 课程名称：凡哥的编程入门课 - 第1课：变量与赋值 🧠

在本节课中，我们将要学习编程中最基础也是最重要的两个概念：**变量**和**赋值**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了本课的主题，本节中我们来看看什么是变量。你可以把变量想象成一个**带标签的盒子**。这个盒子用来存放数据，而标签就是变量的名字。通过名字，我们可以在程序的其他地方找到并使用这个盒子里的数据。

在大多数编程语言中，创建一个变量（即准备一个空盒子）被称为“声明”变量。

## 如何给变量赋值？➡️

仅仅有一个空盒子还不够，我们需要往里面放东西。这个过程就是**赋值**。赋值操作使用等号 `=`，但它不是数学中的“等于”，而是表示“把右边的值，放进左边的变量里”。

以下是赋值语句的基本结构：
```python
变量名 = 值
```
例如：
```python
age = 18
name = "凡哥"
```
这里，数字 `18` 被放进了名为 `age` 的盒子里，字符串 `"凡哥"` 被放进了名为 `name` 的盒子里。

## 变量的命名规则 📝

为了确保程序能正确识别你的“盒子标签”，给变量起名需要遵循一些规则。

以下是主要的命名规则：
*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如 `Age` 和 `age` 是两个不同的变量）。
*   避免使用编程语言中的保留关键字（如 `if`, `for`, `print`）作为变量名。

## 变量的使用 🔄

一旦变量被赋值，我们就可以在代码中通过它的名字来使用里面存储的值。

例如，我们可以进行运算：
```python
x = 5
y = 3
sum = x + y  # sum 的值现在是 8
```
我们也可以打印变量的值：
```python
print(name)  # 这会在屏幕上输出：凡哥
```

## 总结 🎯

本节课中我们一起学习了编程的基石：**变量**与**赋值**。我们了解到变量就像一个贴有名字的存储盒，而赋值操作 `=` 则是将数据放入盒子的动作。掌握如何正确地命名和使用变量，是开启你编程之旅的关键第一步。在接下来的课程中，我们将学习如何用这些“盒子”进行更复杂的操作。
