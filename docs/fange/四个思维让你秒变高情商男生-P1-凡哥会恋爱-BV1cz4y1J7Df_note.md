## 第一点：知己知彼，引导而非追问 🤝

上一节我们概述了课程目标，本节中我们来看看第一个核心思维：了解对方并巧妙引导。追女生如同博弈，关键在于了解对方，而非盲目收集信息。你需要引导女生主动参与互动，并及时抓住她释放的信号。

*   **避免直接提问**：不要直接问“你想吃什么？”，这通常只会得到“随便”或“都可以”的回复。
*   **利用已知信息**：回想她在聊天或朋友圈中提到的喜好。例如，她提过想吃的餐厅，你直接定下即可。
*   **用游戏化方式引导**：如果不了解她的喜好，可以问：“你猜我今天会带你去吃什么？”。她的答案往往就是她的真实想法。
*   **给予有限选择**：在细节上让她做选择题，例如“要不要辣？加不加冰？”。公式是：**提供选项A/选项B，而非开放提问**。

记住，女生更喜欢做选择题，而不是填空题。

## 第二点：共情优先，而非解决问题 🫂

## 第三点：拒绝无条件要求，保持平衡与神秘感 ⚖️🔮

上一节我们学会了提供情绪支持，本节我们来探讨关系中的平衡原则。你需要拒绝对方的任何无条件要求，并保持一定的神秘感，以维持吸引力和关系的对等性。

以下是具体实践方法：

1.  **识别并拒绝无条件要求**：当对方只索取不付出时，即为无条件要求。例如，她让你陪逛街却拒绝你的邀请；或总是让你买单、点外卖却无回馈。
2.  **不怕拒绝**：不要担心拒绝会导致关系结束。无底线付出只会让你的好被视为理所当然。公式是：**她的索取 ≈ 你的付出**，两者应基本对等。
3.  **学会反击要求**：当对方提出单方面需求时，你可以温和地提出交换条件，例如“陪你去逛街可以，不过晚上你得陪我看看我喜欢的电影。”
4.  **保持神秘感**：不要事无巨细地汇报日常。让自己像“薛定谔的猫”，不“打开盒子”，对方就无法完全确定你的状态。这能维持好奇心和危机感。
