## 三大核心聊天技巧

上一节我们概述了课程目标，本节中我们来看看第一个核心技巧：如何通过有效的赞赏开启良好对话。

### 1. 技巧一：细节赞赏法

赞赏是人际关系的润滑剂。所有人都希望得到他人的认可和赞同，女生也不例外。当你给予女生适当的夸赞时，对方对你的好感很容易增强。但夸赞需要讲究方法，生硬的夸赞只会起到反效果。

以下是高情商聊天的核心公式：
**高情商聊天 = 夸具体细节 + 真诚表达**

关键点在于：夸奖具体的某一细节，而非模糊的整体。

以下是不同夸赞方式的对比示例：

*   **A男生（模糊夸赞）**：你今天真漂亮。
    *   **女生回应**：谢谢。
*   **B男生（具体但笼统）**：你今天穿的这件衣服显得气质真好。
    *   **女生回应**：真的吗？谢谢夸奖。
*   **C男生（细节描述+真诚表达）**：今天你穿的这件外套，也太衬你的气质了吧。我之前见过别人穿类似的款式，可穿不出你这个感觉。要是给你拍个照片发个朋友圈，估计大家都得找你要链接。
    *   **女生回应**：哈哈，哪有这么夸张。这件衣服是……（开始聊起来）

分析：C男生的夸赞抓住了具体细节（外套、气质对比），并通过举例进一步肯定，让女生感觉被认真关注。夸赞时抓住细节去描述，越详细越好，容易打开女生的话匣子。但注意不要违心夸赞，否则会让对方觉得虚伪敷衍，结果适得其反。

### 2. 技巧二：合理共情法

很多男生在和女生沟通时，习惯站在客观角度分析事实，试图帮女生解决问题。这种理性思维对沟通意义不大，若对方情绪波动大，反而会让女生觉得你不理解她。

最好的沟通方法是运用共情。共情即能感同身受对方的心情状态，在心理学上被称为同理心。

具体做法是：站在对方角度感受其情绪，然后替她说话，或分享自己类似的遭遇。切忌理性地帮她分析事实。

高情商聊天的公式是：
**高情商聊天 = 认同情绪 + 分享共同遭遇**

**示例对比：**

*   **女生说**：哎好烦啊，老板又无缘无故批评我。
*   **错误回应（理性分析）**：老板说两句也没什么吧。而且我们更应该从自身找原因，搞清楚了状况，下次老板就不会说你了。
    *   **分析**：从理性出发这话没错，但刚被批评的人需要的是情绪认同，而非说教。
*   **高情商回应（合理共情）**：
    1.  男生说：哎可不是嘛，当老板的都一个样。上周我也是，在开会的时候被老板给臭骂了一顿，当时想死的心都有了。
    2.  女生说：就是，真的是气死我了。
    3.  男生说：当时我也快气死了。不过我就用了一个小时，就让我的心情180度大转变。
    4.  女生很好奇：怎么做到的呀？
    5.  男生说：这样，你家楼下那边有一家火锅非常不错，我已经定好位置了。咱们边吃，我边跟你讲，现在就走。

**核心思维**：当女生表达不开心时，不应过多关注她所说的内容，而应着重关注她此刻的心情。不要辨析她的行为或情绪对错，而是帮她将难过和不开心表达出来。做到这一点，女生会觉得你很懂她，是可以交心的人。这样的相处会让双方都舒服，她也会更依赖信任你。

**注意**：不要过度共情。如果你也深深陷入对方的感受无法自拔，会形成两人一起不断吐槽，最后一起陷入负面情绪的困局。恋爱的本质是轻松和开心。想让女生喜欢你，就要做到能带领女生走出情绪困境，而非一起沉沦。

### 3. 技巧三：搭桥聊天法

很多男生觉得跟女生聊天没话题，或不会延伸话题，总是为聊不下去而苦恼。如果你也如此，搭桥聊天法是最佳选择。

应用此法，首先需提取女生话里的一个关键词，这个词就是“桥”。有了桥梁后，再结合自己的感受，就能无缝衔接对方话题，避免总是重启话题的尴尬。

公式是：
**高情商聊天 = 关键词 + 你的感受**

**示例：**

*   **男生**：在干嘛呢？
*   **女生**：我在和闺蜜逛街呢。
*   **一般直男回应**：好的你先逛吧，玩得开心点。（浪费了聊天机会）

**运用搭桥聊天法：**
首先找关键词：“我”、“闺蜜”、“逛街”。

*   **搭“逛街”桥**：逛街没我在，是不是感觉缺了点什么？
*   **搭“闺蜜”桥**：两个闺蜜一起逛，我猜你们肯定偷偷议论过我。
*   **搭“我”桥**：朋友都夸我眼光好，你如果买小裙子的话，我愿意做你的首席鉴定师。

用上搭桥聊天法后，话题就变得容易延伸。

**两点注意事项：**
1.  一定是表达你的**感受**，而非**想法**。感受是主观的。如果表达想法，例如“下次逛街可以叫我一起”或“找时间我请你和闺蜜吃饭”，有很大可能变成舔狗。
2.  如果女生只回复“逛街”两个字，需识别出女生目前不想或不便聊天，此时不应继续打扰。

## 四大聊天禁区

掌握了三大技巧，我们再来看看聊天中必须规避的四大禁区，这些错误会严重破坏你的吸引力。

以下是聊天中需要避免的四种常见错误行为：

1.  **聊天频率过高**
    *   错误认为靠毅力就能追到女生，于是每天找女生聊天。结果不是查户口就是嘘寒问暖，没话题硬找话题。开场白永远是“在吗？”“吃饭了吗？”，结束语是“嗯嗯”、“好的”、“去吧”。这种聊天让女生毫无情绪波动，味同嚼蜡。过分热情不会让女生觉得你关心她，反而会留下不好印象。对你有好感的女生会担心你无所事事、不求上进；对你没好感的女生会觉得你油嘴滑舌、不怀好意。

2.  **直奔主题，过于急躁**
    *   追女生最忌讳过于急躁。有的男生聊天时迫不及待暴露自己对女生的意图，生怕对方不知道自己喜欢她。一旦被拒绝或感觉不感兴趣，就可能恼羞成怒，甚至删除拉黑或破口大骂。要知道，条件不错的女生身边不缺追求者。即使她最初对你有好感，你心急火燎的邀约和表白，也只会让她把你归为“舔狗”一类。想脱颖而出，就要表现得与众不同。聊天前期不要过早暴露意图，采用温水煮青蛙的方式，让女生慢慢被你吸引。在不确定女生对你有好感前，切勿盲目邀约或表白。

3.  **频繁使用“撤回”功能**
    *   过度依赖“撤回”功能。说错话或后悔了，就立刻撤回，以为女生看不见。这其实是掩耳盗铃。且不说发出瞬间女生可能已看到，频繁撤回的行为本身就容易给女生留下“不会说话”、“不自信”的印象。在两性交往中，这容易形成女高男低的不平衡关系。有的女生会因此觉得你缺少男子气概。跟女生聊天，一定要大大方方。你的慎重小心，在女生看来可能是不自信。宁可让女生觉得你不会聊天，也别让她觉得你没自信。聊天技巧可以学，但没有自信，基本就会被淘汰。

4.  **以“男朋友”身份自居（过早失去边界感）**
    *   这是最应该避免的禁区，也是导致前功尽弃的最可惜原因。能到这一步，通常已和女生处在暧昧阶段。女生对你有好感后，可能会主动找你聊天、探索你，甚至给你很多暧昧机会。很多男生此时会落入“陷阱”，按捺不住开始以女生男朋友的身份自居，失去边界感，想要干预女生的生活，把女生当作私有品。这会让已进入爱情幻想的女生猛然惊醒，迅速从暧昧状态中脱离，由感性转为理性。这时再想补救就非常困难。需注意，这个“陷阱”并非女生故意制造，而是此阶段天然存在。因此，即使女生给了好的回应，也切记不可操之过急，而应保持并加深暧昧程度，让女生慢慢沦陷，甚至可能越过“表白”这一高风险步骤。
