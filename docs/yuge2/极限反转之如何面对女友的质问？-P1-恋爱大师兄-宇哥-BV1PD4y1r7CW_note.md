# 极限反转之如何面对女友的质问？ - P1 🎭

在本节课中，我们将学习一种被称为“极限反转”的沟通技巧，用于应对恋爱关系中的质问场景。我们将解析其核心原理，并通过具体步骤演示如何应用。

---

上一节我们介绍了课程主题，本节中我们来看看“极限反转”的核心思想。其本质是一种**非对抗性沟通策略**，核心公式可概括为：

**接纳情绪 + 转移焦点 + 重塑框架**

这意味着，当面对质问时，首要目标不是辩解事实，而是通过认可对方感受来化解对立情绪，进而将对话引导至更建设性的方向。

---

理解了核心理念后，以下是实施“极限反转”的三个关键步骤。

第一步：情绪共鸣。立即停止解释或反驳，转而完全接纳对方的情绪。你可以使用类似“我知道你现在很生气/难过”的语句，其代码式表达为：
```python
response = “我完全理解你现在感到[对方情绪]。”
```

第二步：模糊焦点。在情绪缓和后，不直接回答具体质问，而是将话题升华到关系层面。例如，将“你为什么又迟到？”转化为“你总是把工作看得比我重要吗？”的讨论。

第三步：框架重塑。主动提出一个积极的、面向未来的解决方案，替代当前的指责性对话。核心是提出一个**共同行动**，例如“那我们以后每周都定一个专属的约会之夜怎么样？”

---

本节课中我们一起学习了“极限反转”技巧。我们首先明确了其**接纳→转移→重塑**的核心公式，然后逐步拆解了情绪共鸣、模糊焦点与框架重塑这三个关键操作步骤。掌握这一方法，有助于将潜在的冲突转化为深化关系的契机。
