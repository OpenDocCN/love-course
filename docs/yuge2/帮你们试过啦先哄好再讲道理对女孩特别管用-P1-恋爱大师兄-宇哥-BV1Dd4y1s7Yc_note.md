上一节我们介绍了本节课的主题，本节中我们来深入解析其背后的逻辑。当人们处于强烈的情绪中时，逻辑思考能力会暂时减弱。此时讲道理往往是无效的，甚至可能激化矛盾。

**有效沟通 = 情绪安抚(共情 + 肯定) + 理性沟通(事实 + 解决方案)**

## 具体操作步骤 📝

理解了核心策略后，我们来看看如何将其付诸实践。以下是实施“先哄好再讲道理”策略的三个关键步骤。

1.  **识别并接纳情绪**
