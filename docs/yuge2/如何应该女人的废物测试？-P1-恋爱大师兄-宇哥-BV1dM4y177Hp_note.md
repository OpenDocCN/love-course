# 恋爱教程 01：如何应对女性的“废物测试”？ 🧪

在本节课中，我们将要学习一个在男女交往初期至关重要的概念——“废物测试”。我们将了解它的定义、常见类型，并掌握一套简单有效的应对策略。掌握这些技巧，能帮助你在互动中保持吸引力，顺利通过对方的“考核”。

## 什么是“废物测试”？ 🤔

“废物测试”的定义很简单。女性会通过一些语言和行为，来判断你是否是个“废物”，以及你是否值得她投入感情。

一个女生在和男人交往之前，会提出很多问题，来观察你是如何应对的。应对得好，就会激起她的情绪变化，她会把你当作潜在恋人。应对得不好，她就会把你移出潜在恋人区，当成朋友，甚至直接让你出局。

## 七种“废物测试”的应对方法 🛡️

上一节我们介绍了“废物测试”的基本概念，本节中我们来看看具体的应对方法。我为你总结了七种最常见的测试类型及其破解之道。

2.  **需求询问型**

7.  **兴趣测试**
