# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据。标签就是变量的名字，我们可以通过这个名字找到并使用盒子里的数据。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号（`=`）。

```python
my_name = "宇哥"
my_age = 30
```

以上代码创建了两个变量：一个名为`my_name`的“盒子”里存放了文字“宇哥”；另一个名为`my_age`的“盒子”里存放了数字30。

## 基本数据类型 🔢

我们已经知道如何把数据放进“变量”这个盒子里。接下来，我们需要认识一下盒子里可以放哪些不同种类的数据，这就是数据类型。

Python有一些内置的基本数据类型，最常用的有三种：

以下是三种核心数据类型的介绍：

1.  **整数（int）**
    *   表示没有小数部分的数字，可以是正数、负数或零。
    *   例如：`10`, `-5`, `0`

2.  **浮点数（float）**
    *   表示带有小数点的数字，用于处理更精确的数值。
    *   例如：`3.14`, `2.0`, `-9.81`

3.  **字符串（str）**
    *   表示文本信息，需要用单引号（`'`）或双引号（`"`）括起来。
    *   例如：`"Hello World"`, `'Python'`

## 如何查看和使用类型？ 🔍

了解了有哪些数据类型后，你可能会问：我怎么知道一个变量里存的是什么类型的数据呢？

Python提供了一个非常方便的函数`type()`来查看变量的数据类型。

以下是使用`type()`函数的示例：

```python
height = 1.75
print(type(height))  # 输出：<class 'float'>

message = "学习Python"
print(type(message)) # 输出：<class 'str'>
```

同时，不同类型的数据可以进行转换。例如，将整数转换为字符串，或者将包含数字的字符串转换为整数。

```python
# 整数转字符串
num = 100
str_num = str(num)  # str_num现在是字符串"100"

# 字符串转整数（前提是字符串内容像数字）
str_value = "50"
int_value = int(str_value) # int_value现在是整数50
```
