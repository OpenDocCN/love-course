# 睡前小故事 - P1 - 恋爱大师兄-宇哥 🐱

在本节课中，我们将学习如何通过一个简单的睡前故事，理解基础编程概念中的“循环”与“条件判断”。我们将使用一个关于小猫的故事作为比喻，让初学者能够轻松掌握这些核心思想。

---

## 故事背景与核心概念 🎵

故事始于一段持续的音乐声。在编程中，重复执行某项任务，就像音乐循环播放一样，我们称之为“循环”。

上一节我们介绍了故事的开始，本节中我们来看看如何用代码描述这种重复。

循环可以用以下**代码**来描述：
```python
while 音乐在播放:
    播放音乐
```
这段代码表示：只要“音乐在播放”这个条件为真，就会不断地执行“播放音乐”这个动作。

---

## 故事中的关键事件 🐾

音乐声结束后，故事中出现了一声“喵~”。这可以看作是一个“事件”被触发。在程序中，我们经常需要等待某个条件达成，然后执行相应的操作。

以下是实现等待与触发的逻辑，它结合了循环和条件判断：
```python
while True:
    if 音乐停止:
        发出“喵~”的声音
        break
```
这段代码的含义是：程序会一直运行（循环），并不断检查“音乐停止”这个条件。一旦条件满足，就执行“发出声音”的动作，然后使用 `break` 跳出循环。

---

## 总结与回顾 📚

本节课中我们一起学习了两个基础编程概念。

首先，我们通过持续的音乐引入了**循环**的概念，它用于重复执行任务。
其次，我们通过“喵~”声的触发，学习了**条件判断**，它让程序能够根据不同的情况做出反应。

将循环与条件判断结合使用，是构建所有程序逻辑的基石。希望这个睡前小故事能帮助你更形象地理解这些概念。
