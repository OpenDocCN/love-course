# 追女生是男人就千万别怂 - 课程P1 🎯

在本节课中，我们将要学习一个核心观点：在追求感情时，犹豫和过度准备是脱单的最大障碍。我们将分析“怂”的心态如何让你错失良机，并探讨开始一段关系的真正关键。

---

有哪种男生是特别不受女人喜欢的吗？我以前也认为是那种不高不帅、没钱没事业的男生，毕竟大家都是这样说的。

而事实是，你会发现一大帮普通的男生，每一天也都在脱单。

其实，我们大多数人都是普通人，这并不影响我们爱别人，也不影响我们被别人爱。

但只要你认为“我要等到什么什么时候再谈恋爱”，那么你放心，你短时间内肯定脱不了单，而且一定会脱单困难。

做情感咨询快十年了，我发现单身的男生，还真的不是那些长得丑的或者没有钱的，而是那些没有机会出现在女生生活里面的男生。

## 1. 怂男永远在准备 🚫

上一节我们提到了“没有机会”的男生，本节中我们来看看他们的典型特征：永远在做准备。

怂男永远都在做准备。这些所谓的准备，只是他们去逃避执行的借口而已。这类人从来都是最难脱单的。

### 好感窗口稍纵即逝

首先，好感窗口都是稍纵即逝的。在这个世界上，不管什么机会都是短暂而美好的。

就像一个形色匆匆的美女站在你面前跟你说“走不走”，她不会问你有没有准备好，也不会因为你是否优秀而多等你一分钟。只要你稍微犹豫一下，她扭头就走，而旁边那个普通的、甚至有点猥琐的胖子，却得到了机会。

每一个女生的青春都是短暂的。她们不得不考虑自己的未来，也害怕碰不到真命天子。她们也会想“为什么没有男生来追我呢”、“为什么他还不来找我”。

她们也会读一些鸡汤，例如：
*   “那些从来不主动的男生，就是不够喜欢你。”
*   “别浪费太多时间在一个人身上，耽误的永远是你自己。”

而那些怂男永远在低头做所谓的准备，错过了一个个路过的姑娘，或者可怜兮兮地望着姑娘的背影讨价还价，问她：“哎，能不能再给我一些时间，我马上就好了。”

**对不起，没有时间。**

### 闭门造车毫无意义

其次，任何闭门造车的准备都是没有任何意义的。

你在感情当中会在什么时候出现什么问题，最好的解决方案是什么，这些都只有真正去谈恋爱才能够体会。也只有经历感情的人，才知道自己哪里做得不够好。

他们说自己“还没有准备好”，说自己“还需要再努力”。但是，他们从来没有问过别人：“你希望找一个什么样的男朋友？”、“你想和我一起过什么样的生活？”

他们一直努力地想成为他们自己心目中那个“完美男朋友”：**一定要有钱，一定要长得帅，一定要有学历，一定要有房，一定要有车，一定要有八块腹肌**……反正，就是他现在自己不具备什么，那个东西就是一定要有的。

其实，说钱，比你穷的一大堆；说学历，比你低的也很多；说长相，还是有比你差的。

与其说你在准备，还不如说你在逃避。因为不管任何事情，在机会面前，人人平等，也人人都不是一百分。王校长还不够帅呢，万希哥和王校长比起来，还不够有钱。但只要你不行动，你就一定会错过机会。

## 2. 我们永远在弥补匮乏感 🔄

上一节我们分析了“准备”的陷阱，本节中我们来探讨这种心态背后的深层原因：内心的匮乏感。

其实从我们进入社会以来，目标一直都在变，但是目的只有一个：就是在弥补自己内心的匮乏感，让自己更喜欢自己一些。

这个逻辑链条通常是这样的：
*   “我要考上好的大学就好了”——因为你读的大学不够好。
*   “我要有钱就好了”——因为你的薪资还不够高。
*   “我要买房就好了”——因为你还没有房。
*   “我要还清贷款就好了”——因为你有了房之后，又顺带有了贷款压力。

你永远都有遗憾。我们一生都在弥补遗憾的路上。没有尽头的我们，永远对自己不满意，永远对自己不认可，永远也觉得自己还没有做好准备开始一段感情。

你说这样的人，怎么可能会轻松地开始一段感情呢？

说句实在点的话，有多少要花百万彩礼才能娶回家的小姑娘，当初不是和一个穷小子，在小旅馆、在终点房长大的吗？

**没有人会一直等你。** 她们心中的标准，也没有那么确定。女生心里的标准，是用来要求不喜欢的男生的。遇到喜欢的这个男生，**他本身就是她的标准了**。

## 3. 爱情不等人，行动是关键 ⏳

了解了心态问题后，本节我们聚焦于行动的必要性。

女生喜欢一个男生的时候，其实并不是想象中那么深情和坚定的。她会在这段时间之内，特别特别喜欢你。但是如果女生在你身上得不到回馈，那么她就会心灰意冷，就去找别人了。

之前有个女粉丝问过我，她说喜欢的男生跟她说：“自己现在没有能力给她更好的生活，将来能力到了之后再来找她。”那自己要等这个男生吗？

我的回答是：**赶快离开他**。

因为我们找对象，找的是一个可以共度难关的伴侣，而不是一个一直在做准备的怂男。

每一个人进入社会之后，压力都会越来越大，麻烦也会越来越多。因为未来，你是要成为别人的领导、成为别人的丈夫、成为别人的爸爸、还要成为一个好儿子的。这个是成长和成熟的过程，是每一个人都需要经历的事情。

**每一个人在你身边停留的时间都是有限的。** 如果你抓不住那个人，那么就要好好地做好永远失去那个人的准备。

没有经历过，你永远不知道爱情是什么样子的。要想看清楚，最起码你要多经历，这个才是恋爱的道理。

**爱情之中的输家，永远都是怂的。所以你只要怂，你就必定输。**

---

**总结**

本节课中我们一起学习了追求感情时最关键的心态：**切忌“怂”和“等”**。

1.  **机会窗口短暂**：好感不等人，过度准备只会让你错失良机。
2.  **准备脱离实际**：闭门造车的完美主义是逃避行动的借口，真正的成长在关系中发生。
3.  **根源是匮乏感**：总觉得自己“不够好”而不敢开始，是一个需要克服的内心循环。
4.  **行动是唯一解**：女生不会无限期等待，关系的标准遇人则变。共度难关比等待“准备好”更重要。

人生漫漫，别留遗憾。别等伤过之后才去后悔，别等错过之后才知道珍惜。**从现在开始，勇敢一点。**
