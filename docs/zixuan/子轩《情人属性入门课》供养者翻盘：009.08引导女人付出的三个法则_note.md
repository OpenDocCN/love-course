*   **金钱成本**：她为你支出的物质财富。
*   **情绪成本**：她为你产生的情绪波动和情感输出。

## 引导投入的底层方法论 ⚙️

**第一点：大投入带动小投入**
用你较大的付出去引导对方较小的付出。初始比例可以参考 **你的付出 : 她的付出 ≈ 3 : 1**。例如，你花费约一万元，可以尝试引导她为你付出三千元左右。随着关系深入，她的投入比例会逐渐提高，理想状态是达到 **5 : 5** 的平衡。吸引力极强的个体，甚至可能获得 **1 : 10** 的高回报。
