# 子轩《情人属性入门课》005.04：吸引力 🧲

在本节课中，我们将要学习情感关系中的核心基石——吸引力。我们将探讨为何缺乏吸引力会导致你沦为“供养者”，并了解构建吸引力的几个关键方向。掌握吸引力，是你从被动地位翻盘、获得情感主动权的第一步。

很多人在感情中没有位置和话语权，只能充当供养者。本质原因是缺乏吸引力。你没有足够的筹码，因此没有底气去争取自己想要的权利。

如果你对对方的基础吸引力不够，在追求时会非常被动。在一起之后，你会更加被动，很容易陷入供养者模式。

当你不具备吸引力时，对方会为你预设许多标准。例如，要求你“慢热看诚意”，或者暗示“你再努力一点就能追上我”。对方也可能提及前任或追求者赠送的贵重礼物。

如果你拒绝或反驳这些标准，对方可能直接终止关系，将你删除或拉黑。原因在于你对他没有吸引力，他离开你没有任何心理负担。

恋爱阶段和婚姻阶段也是如此。如果对方认为你“不为我花钱”或“对我不够好”，他就敢以分手相要挟。例如，房子不加名就要分手。他之所以敢这样做，是因为你除了提供金钱和情绪价值外，对他没有任何吸引力。

如果缺乏吸引力，一切技巧和套路都将是空谈。因此，必须记住：**吸引力是一切感情的原点**。不具备吸引力，你就无法获得真正的爱情，只能沦为供养者。

---

上一节我们明确了吸引力的重要性，本节中我们来看看第一个具体方向：形象吸引力。

形象决定了女人对你的策略，是把你当作供养者还是情人。思考一下，为何有些女人对你姿态很高，不怕你离开？因为她知道你的形象不佳，你只能通过付出巨大代价（时间、物质、精力）去追求她。

一旦她发现你的形象有质的提升，她必然会担心失去你。因此，真正高段位的女人会倾向于把男朋友往“丑”了打扮。她们深知，男人形象提升再结合其价值，很少有女人能抵挡。

我曾有过切身教训。我恋爱九年的前女友总说“形象不重要，男人随便穿就行”，她给我挑最难看的衣服和最老气的搭配，还不让我护肤，认为男人有事业就有一切。

后来，我在她和闺蜜的聊天记录中看到了她的真实想法：她正是因为觉得我很有吸引力，才故意把我打扮得不起眼，以减少其他女人对我的兴趣。

所以，无论你正处于恋爱还是婚姻中，都必须提升自己的形象。如果形象不够，你在感情中的吸引力和博弈权都会大大降低。

但请注意，形象并非单指颜值。女人看男人看的是整体感觉，包括**视觉、听觉和嗅觉**。即使你身高或颜值不突出，也完全可以通过打造全方位的形象来提升吸引力。系统课程中有详细教学，此处不再赘述。

理解了外在形象的基础作用后，我们进入一个更内在、更具主导性的维度：性张力吸引。

对于绝大多数供养者而言，你在感情中处于附属位，服务于女人。在这种情况下，你是没有性张力的。

什么是性张力？**性张力就是你作为男人的领导权、进攻性和话语权**。它体现在身体上能征服对方，在内核和心理上敢于拒绝、敢于进攻、不按常理出牌。通过行为动作、肢体语言、语音语速语调，让对方彻底为你臣服。

在建立了个人气场（性张力）之后，我们需要夯实吸引力的现实基础：高价值吸引。

首先，你必须明白自己在这段感情中的价值是什么。**你的价值就是你的依仗，是关系中最重要的筹码**。你必须牢牢掌握这个价值，而不是交给对方。

例如，如果你主要提供物质价值（即物质供养者），那么什么是最宝贵的？**金钱和现金转账**。这两项应尽量避免直接给予。你应该将钱花在双方的共同消费上，如旅游、吃饭、娱乐等。

赠送礼物需谨慎选择奢侈品。因为二手行业成熟，对方收到奢侈品包后很容易变现。绝大多数供养者在感情中被女人牵着鼻子走：对方要包就买，对方想看电影就全程陪同服务。

有些兄弟甚至受传统思维影响，动不动就送包、送车、买房，将主动权拱手让人。女人被伺候习惯了，自然会恃宠而骄，搞不清自己的位置。

仔细想想，你花了钱，就应该享受对应的服务和情绪价值。但事实往往相反。当然，有一部分高段位女人情商高，善于提供情绪价值。

但请记住：**只要一个女人只提供情绪价值，却从不回礼，或者回礼价值比例低于7:3（你7她3），那么她很可能在套路你。**
