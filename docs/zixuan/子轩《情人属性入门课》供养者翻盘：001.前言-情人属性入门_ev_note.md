---

上一节我们介绍了供养者策略，本节中我们来看看情人策略。情人策略的好处在于，当你拥有情人属性时，女性喜欢你是因为感觉、心动和无需条件的爱情。你可以获得双向奔赴的关系，甚至对方会主动追求你。

拥有情人属性的男人绝不缺乏女性关注。这里的不缺女性，指的是你只需做好自己，无需刻意搭讪或进行超出正常范围的社交，也会有女性主动被你吸引、向你靠近。

这种体验是绝大多数供养者终其一生都无法体会的。即便有些条件优越的供养者被人追求，对方也只是图钱，不会对你产生真正的冲动和欲望。

那么，如何成为具有情人属性的男人呢？以下是四个核心要点。

以下是成为情人属性男人的四个关键步骤：

1.  **引爆吸引力**：将你的吸引力打造到极致。这是至关重要的一点，也是区分供养者与情人属性的关键因素。
2.  **情绪体验**：请注意，是**情绪体验**，而非情绪价值，两者有本质区别。在后续课程中，将教你如何提升你给对方带来的情绪体验。
3.  **内核自信**：这是情人策略的核心。绝大多数男性受到的教育是要以女性为先。这种潜意识导致你在与女性发生冲突时，会下意识选择退让、容忍和包容。这种状态在女性眼中毫无内核、毫无自信。这也是供养者在感情中没有话语权的核心原因。
4.  **博弈**：这是一切长期关系（无论是恋爱还是婚姻）的核心。当你发现女性逐渐对你冷淡、抱怨甚至想离开时，就说明你的博弈能力出了问题。

此外，在博弈中还有一个非常重要的点，叫做**引导投入**。这一点至关重要。一个女性对你的爱源于她的投入，她能爱你多久也取决于她的投入。这也是供养者得不到爱情的根本原因，因为绝大多数供养者在感情中都是主动、过度投入的一方。

感情中有一个定律：**谁投入越多，谁就越离不开**。所以，请思考你当前的关系出现问题，是否源于你的过度投入。
