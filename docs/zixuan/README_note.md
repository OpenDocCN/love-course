# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量与数据类型。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，创建一个变量并给它赋值非常简单。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如，`name = “子轩”` 这行代码创建了一个名为 `name` 的变量，并将字符串“子轩”存储在其中。之后在程序中，你就可以使用 `name` 来代表“子轩”这个值。

## 2. 基本数据类型 🔢

理解了如何存放数据后，我们需要知道能存放哪些种类的数据。Python中有几种内置的基本数据类型，它们决定了数据可以进行的操作。

以下是三种最常用的基本数据类型：

*   **整数**： 表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**： 表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**： 表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘你好’`, `“Python”`。

## 3. 查看与理解数据类型 🔍

当你拿到一个变量时，如何知道它里面装的是哪种类型的数据呢？Python提供了一个内置函数 `type()` 来帮助我们。

以下是使用 `type()` 函数的示例：
```python
age = 25
print(type(age))  # 输出：<class ‘int’>

price = 9.99
print(type(price)) # 输出：<class ‘float’>

my_name = “子轩”
print(type(my_name)) # 输出：<class ‘str’>
```
这段代码分别创建了三个不同类型的变量，并使用 `print(type(...))` 语句打印出了它们各自的数据类型（`int` 代表整数，`float` 代表浮点数，`str` 代表字符串）。

## 4. 一个简单的综合示例 ✨

现在，让我们把以上知识点结合起来，写一个简单的程序。

以下是定义变量、使用变量并检查类型的完整流程：
```python
# 定义变量
user_name = “张子轩”
user_age = 20
user_height = 1.75

# 使用变量打印信息
print(“姓名：”, user_name)
print(“年龄：”, user_age)
print(“身高：”, user_height)

# 检查每个变量的类型
print(“user_name的类型是：”, type(user_name))
print(“user_age的类型是：”, type(user_age))
print(“user_height的类型是：”, type(user_height))
```
运行这段代码，你会在屏幕上看到输出的个人信息以及每个变量对应的数据类型。

本节课中我们一起学习了Python的变量与数据类型。我们知道了变量是存储数据的容器，并通过 `=` 来赋值。我们还认识了三种基本数据类型：**整数**、**浮点数**和**字符串**，并学会了使用 **`type()`** 函数来查看变量的类型。掌握这些是后续学习更复杂操作（如数学运算、字符串处理）的坚实基础。
