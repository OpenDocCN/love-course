# 大鹏高级聊天会员课程_27_第三单元：女生说“最近长胖了”怎么应对？ 💬

在本节课中，我们将学习当女生在聊天中提到“最近长胖了”时，应如何理解和回应。我们将分析其背后的心理，并提供具体的沟通策略。

## 课程概述

在现代社会，尤其生活在一二线城市的女生，往往存在比较严重的容貌和身材焦虑。当前社会是一种颜值经济。已有不少研究表明，在同等能力和领域条件下，颜值高的人收入比颜值一般的人高出30%左右。

在婚恋市场，每年都有源源不断的更年轻、更好看的新人涌入。这解释了为什么大部分女生都存在身材和颜值的焦虑。

## 理解女生的真实意图

当女生在聊天过程中对你说“我最近长胖了”，首先需要明白她为何分享此事。女生愿意主动分享自己的生活，说明在她眼里你是一个合格、可以倾诉的对象。

女生说自己长胖，并非要你帮她解决实质问题。她不需要你买减肥药或督促她减肥。真实情况是，很多女生一方面说自己胖、要减肥，另一方面晚上又跟朋友去吃吃喝喝、喝酒宵夜。

在她想要减肥这件事上，你不必太当真。太认真，你就输了。

## 具体回应方法

以下是三种应对女生说自己长胖的沟通方法。

1.  **表达认可与偏好**
    你可以告诉女生：“我觉得你现在这样就刚刚好。其实我不喜欢太瘦的。” 这种方式直接给予对方认可，缓解其焦虑。

2.  **用幽默化解**
    你可以说：“那你先吃饱呗，吃饱了才有力气去减肥。” 这是一种轻松、幽默的回应，能有效调节聊天氛围。

3.  **调皮地表示“支持”**
    如果女生坚持说要减肥，你可以说：“那我支持你。为了考验你的毅力，我在吃宵夜的时候多拍点视频、多拍点照片给你。” 这种回应方式能让女生清楚你是在开玩笑，并以一种调皮的方式有效调动双方的聊天情绪。
