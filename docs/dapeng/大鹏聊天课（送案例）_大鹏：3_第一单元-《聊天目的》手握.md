# 大鹏聊天课（送案例）_大鹏：3_第一单元-《聊天目的》手握

在本节课中，我们将要学习聊天的核心目的。明确聊天的目的，就像手握指南针，能确保我们在交流中不迷失方向，从而制定有效的策略。

## 聊天的两个阶段

做任何事情前，都需要明确目的，聊天也不例外。在我的聊天体系里，聊天被分为两个主要阶段：**吸引前**和**完成吸引后**。同一件事，在不同阶段，我们的目的截然不同。

上一节我们介绍了聊天的两个阶段，本节中我们来看看每个阶段的具体目的。

### 阶段一：吸引前

在成功吸引女生之前，我们的聊天有两个核心目的。

以下是吸引前的两个核心目的：

1.  **了解女生，展示自己**
    *   这是开启聊天时的首要目的。你需要通过交流获取女生的信息，同时向她传递你是一个什么样的人。例如，你可以通过分享生活片段或观点，来展示自己是一个**阳光、有才华、活泼或幽默**的人，从而初步吸引对方。

2.  **保持联系，完成邀约**
    *   从认识女生到成功邀约，中间需要一个过程。这个目的就是将过程分解为可执行的步骤：先开场了解，再通过几次互动保持联系，最后在合适时机完成邀约。公式可以概括为：**开场 → 信息互换（了解）→ 保持联系 → 判断时机 → 完成邀约**。

### 阶段二：完成吸引后

当我们已经成功引起女生兴趣，或已经确立恋爱关系后，聊天的目的就发生了变化。

以下是完成吸引后的核心目的：

*   **维护关系**
    *   此时，聊天的核心转变为**维护关系**。这包括日常的问候、分享生活中的趣事和小情绪，目的是加深情感连接，让关系持续稳定发展。

## 课程总结

本节课中，我们一起学习了聊天的核心目的。我们将其分为两个阶段：在**吸引前**，目的是 **`了解女生，展示自己`** 和 **`保持联系，完成邀约`**；在**完成吸引后**，目的则简化为 **`维护关系`**。明确这些目的，能帮助你有条不紊地推进每一次交流。
