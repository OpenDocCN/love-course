# 大鹏聊天课（案例篇）_39：第五单元 - 当女生说自己是“照骗”时，如何回复？ 📸

在本节课中，我们将要学习一个在线上聊天中常见的场景：当女生自称是“照骗”时，应该如何得体地回应。我们将分析女生这样说的原因，避免低情商的回复陷阱，并掌握两种能营造良好聊天氛围的回复策略。

## 课程概述

我们现在生活在一个普遍使用美颜和修图技术的时代。生活在大城市的女生，几乎每个人都会化妆。化完妆后，她们还会使用美颜相机拍照，拍完照后，再用修图软件精心修饰。相信不少同学都在抖音上看到过化妆前后判若两人的视频。

因此，我们在与女生见面聊天的过程中，遇到女生实际情况与照片有差距的概率其实很大。这时，我们需要分析女生主动说自己是“照骗”的原因。

## 女生自称“照骗”的原因分析

无论女生本人与照片的差距具体有多大，在聊天过程中，我们绝不能顺着她的话去确认。例如，直接说“你真的是照骗”或“你本人比照片丑好多”，这种回复会显得情商很低，甚至可能导致失去见面的机会。

那么，女生为什么会主动说自己是“照骗”呢？以下是两个主要原因：

1.  **不够自信**：女生知道自己修图修得比较过分，因而对真实样貌有些不够自信。
2.  **调侃与缓和氛围**：女生可能只是通过调侃自己，来缓和与你之间的聊天氛围，让对话更轻松。

## 低情商回复的反面案例

我曾经有一位学员，在处理这种情况时就显得情商很低。当女生在聊天中说自己是“照骗”时，他回复了一句：“你是不是化妆技术不太好啊？”

结果，女生反过来攻击他：“你是不是刻意把自己包装成富二代？” 两人随即进入了互怼模式，聊天难以继续，最终双方都拉黑了彼此。

## 正确的回复思路与策略

在聊天过程中，最重要的目标是**把天聊开心**，核心是维护**良好的聊天氛围**，而不是纠结于某个具体结果。如果你在见面时觉得对方与照片差距过大，完全可以掉头就走，这没有问题。但如果在聊天阶段就把氛围搞砸了，你甚至可能失去见面的机会。而我们聊天的目的，众所周知，正是为了和女生见面。

所以，当女生说自己是“照骗”时，正确的回复方向是化解尴尬、提升氛围。以下是两种有效的回复方式：

*   **赞美化解法**：回复“本人比照片好看，也算照骗吗？”
*   **自我调侃法**：回复“没事的，我平时自己也修图。”

当你使用以上任何一种方式回复女生时，通常都能让女生的聊天情绪变得非常开心。在我过往的聊天案例中，多次出现女生自称“照骗”的情况，使用这类回复后，女生的情绪反馈都相当不错。

## 补充：从朋友圈判断颜值的经验分享

另外，在这里分享一些个人经验。如果一个女生的朋友圈里没有发自己的照片，或者发的照片经常看不清脸、照片数量很少，那么现实中她可能不是特别好看。

相反，但凡对自己容貌有点自信的女生，通常会在朋友圈不断展示自己。看女生朋友圈时，不要只看最好看的那张，试着找到一张你觉得最普通的，问自己是否能接受。如果你连她最普通的照片都无法接受，那么见面时她的样子大概率就和那张差不多，你也就没有必要去见面了。

很多同学会陷入自我想象，喜欢反复翻看女生最好看的几张照片，然后进行幻想，这是相当没有必要的。通常来讲，女生出来见面，其真实样貌比照片差个两三分，属于正常情况。因为女生经历了化妆、美颜相机拍照、精心修图这一整套流程。如果经过这样处理后的照片看起来还很一般，那么见面时她是“照骗”的概率就非常大了。
