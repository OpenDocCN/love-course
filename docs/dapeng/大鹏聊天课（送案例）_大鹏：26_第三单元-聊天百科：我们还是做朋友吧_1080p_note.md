# 大鹏聊天课（案例）第26课：女生说“做朋友”怎么办？🤔

在本节课中，我们将要学习当女生说出“我们还是做朋友吧”时，如何理解其背后的含义，以及如何正确应对这种情况。课程将分析表白行为的潜在问题，并提供具体的应对策略。

## 课程概述

女生之所以对你说“我们还是做朋友吧”，一定是因为你在过往的聊天互动过程中主动示好或表白了。

通俗来讲，就是你向女生表白，或表达了希望两人在一起的意思。而女生觉得你们之间的感情不足以支撑确定关系，从而对你说“我们还是做朋友吧”。

## 核心问题分析：为何要避免表白

有句话是这样讲的：解决问题的最好方式就是防止问题的出现。你给了女生拒绝你的机会，所以女生才会拒绝。

我们不建议在聊天中和女生表白，是因为很多人误以为自己和女生的感觉只差一点点。

他们以为靠着真诚的表白和表达，就能让女生答应最终确定关系。这种思维是大错特错的。

所以当你向女生表白后，再来问“我被女生拒绝了怎么办”，或者“女生已经说要跟我做朋友，我怎么办”，我告诉你，最上策就是不要表白。

不要给女生一个拒绝你的机会。正确的方法是使用我教你的聊天步骤，通过正确的吸引方式，一步一步与女生互动，吸引到她。

## 应对策略：如果已经被拒绝

如果你确实已经表白并遭到了女生的拒绝，女生说“那我们还是做朋友吧”，正确的应对方法是什么？

答案是：先跟她做朋友，然后等待一个契机。

女生总会有一个人搬家、一个人无所依靠的时刻。当她需要帮助时，你刚好出现在她眼前，我们抓住这个机会就可以继续与她互动，甚至确定关系。

如果女生某天已经有了对象，我们也要学会及时止损。

千万不要永远地备胎到底。
