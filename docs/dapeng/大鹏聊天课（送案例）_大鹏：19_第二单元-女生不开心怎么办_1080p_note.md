# 大鹏聊天课（案例）_19：第二单元 - 女生不开心怎么办 😌

在本节课中，我们将要学习当聊天对象（女生）心情不好时，应如何正确应对。核心在于理解自身角色，把握关心的尺度，避免陷入被动关系。

## 课程概述

课程开始前，请先思考一个问题：当你自己不开心时，通常会如何处理？

大部分人的答案可能是：睡觉、找朋友喝酒倾诉、或者做一场运动。这是我们自己面对不开心情绪的常见方式。

## 女生不开心的原因

上一节我们介绍了自身处理情绪的方式，本节中我们来看看女生不开心的可能原因。原因多种多样，例如：
*   工作不顺利
*   身体不舒服
*   同事为难
*   父母催婚

情况复杂，我们只需知道“她今天不开心”这个结果即可。

## 核心问题与错误做法

那么，当我们聊天时遇到女生心情不好，应该怎么办？

以下是几种常见的错误做法：
*   给她发红包
*   刻意哄她开心
*   请她吃饭

**核心原则**：这些都不需要做。你要清楚，女生不开心是她自己的事情。

## 正确做法：适度关心

你要做的就是适度关心，让女生感受到你在乎她即可。千万不要发红包或刻意哄她，这完全没有必要。

有些同学会问：女生都不开心了，难道我什么都不做？

**核心概念**：你要明白，女生不开心跟你关系不大。你只是和她聊天，无需为她生活中的所有遭遇负责。

如果你们已是男女朋友，哄她开心是完全可以的。如果你们只是聊过天或见过一两面，则没有必要做额外的事情。

## 具体操作与关系边界

女生不开心时，我们简单关心一下即可。她不开心，我们就少聊点天，等她心情好了再聊。

**原因**：想把一个不开心的人哄开心很难。哄她之后，反而可能让你和女生的关系陷入被动，这完全没有必要。

我曾有一位学员，对方稍有风吹草动他就焦急不已，甚至饭吃不香、觉睡不好。我问他：你是否曾这样关心慰问过你的父母？他羞愧地低下了头。

有些同学太把对方当回事，反而导致对方爱搭不理，甚至完全拿捏你。作为学员，要对自我有要求：**不要跪着去讨好女生**，一定要记住。

## 课程总结

本节课中我们一起学习了如何应对女生不开心的情况。

首先，我们分析了女生不开心的多种原因，指出我们无需为她生活中的所有事情负责。

我们要做的不是帮她解决问题，而是清楚一个聊天原则：**女生开心就多聊，女生反应一般就少聊**。
