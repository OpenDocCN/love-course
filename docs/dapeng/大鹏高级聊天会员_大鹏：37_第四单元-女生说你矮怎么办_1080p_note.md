## 身高在互动中的影响 📊

上一节我们介绍了课程主题，本节中我们来看看身高对于我们和女生互动中的具体影响。随着国家经济发展和物质条件进步，现在的95后、00后平均身高比90后、80后高出3到8厘米不等。

*   必须穿着内增高鞋。

## 如何有效提升视觉身高 👟

了解了身高的影响后，我们来看看如何实际提升视觉身高。我的真实身高在168厘米左右，但我通常向女生报的身高是175厘米。

这通过以下公式实现：
`报出身高 = 真实身高 + 鞋跟高度 + 内增高垫高度 + 发型高度`

在我的案例中：
*   真实身高：168厘米
*   鞋跟高度：不低于3厘米
*   内增高垫：2到3厘米
*   发型高度：3到4厘米
*   总计增加：约7厘米

因此，向女生报出175厘米的身高是可行的。许多男明星和女明星都会使用增高方法。如果你目标定在1.8米，就应该购买增高垫并换掉平底鞋。

同时要注意，女生在约会时通常也会增加身高。一个女生穿3到5厘米的鞋子很常见，再通过帽子或穿搭，增高7到8厘米也非常正常。这是当前的普遍情况。

上一节我们讨论了如何提升视觉身高，本节中我们来分析究竟是哪些女生更在意男生的身高。实话告诉你，往往是那些自身身高不高的人，才会更加在乎对方的身高。

大家经常在社交软件上看到一种女生，她们会写明“1.8米以下不要滑我”。这种情况印证了一个观点：**越缺少什么，就越在意什么**。

## 应对身高要求的方法 🤝

分析了在意身高的女生类型后，我们来看看具体的应对方法。这时有些同学会问：即使使用了增高垫和发型，身高仍达不到1.8米，应该怎么办？

如果你遇到一个特别在乎身高、坚持要找1.8米以上男生的女生，我们应该做的是：**与女生互相筛选**。

这意味着她看不上你，同时你也看不上她，彼此就不需要再聊天了。我们应该去找到那些不是特别在意我们身高的女生。
