# 课程1：《30天计划》重制：中级篇04 - 放弃开场白，学习自然开场 🎯

在本节课中，我们将学习如何摒弃固定的开场白，转而使用更自然、更灵活的方式开启对话。这是从搭讪初学者迈向中高级选手的关键一步。

在初级训练中，我们介绍了三种固定的开场方式。然而，真正的搭讪高手并不依赖任何固定的开场白。这意味着你可以利用任何观察到的事物作为对话的起点，这是最自然的搭讪状态。

许多人会问：如果没有了开场白，我该如何开启对话呢？以下是三个核心的开场原则。

**以下是三个核心的开场原则：**

1.  **描述客观事物**：例如，你可以就对方的穿衣打扮、身高外貌等客观存在的事物开启对话。
2.  **描述心情**：你可以根据自己当下的心情，或者推测对方的心情来开启对话。
3.  **描述当下情景**：这是最常用的技巧。根据对方正在做的事情或所处的场景开启对话。

描述当下情景的应用非常广泛。例如，如果对方正在喝咖啡，你可以问：“你这杯茶好喝吗？”（即使那是咖啡）。如果对方在等公交车，你可以问：“你在这里等车等了很久吗？那班车还没来吗？”这些都可以成为自然的开场。

上一节我们介绍了三个原则，接下来我们看看如何在实践中运用这些原则。在演示中，我使用了一个拍手的动作作为开场。这并非刻意设计，而是当时的灵机一动。我没有过多考虑，直接使用了这个方式，效果非常好。

**以下是演示中运用的关键点：**

*   开场基于一个突发、即时的观察（有蚊子）。
*   对话围绕当下情景和互动自然展开。
*   没有使用任何预先准备的台词。

因此，在《30天计划》的终极阶段，你需要学会摒弃之前所有的固定开场白，学会用任何观察到的事物作为开场。具体的训练方法和计划在我们的课程社区中提供。

当你能够真正做到抛弃开场白，以不变应万变时，你才能真正成为一个搭讪的中高级选手。

本节课中，我们一起学习了放弃固定开场白的重要性，掌握了三个自然开场的原则（描述客观事物、心情、情景），并通过实例观察了如何在实际互动中灵活运用。记住，自然与即兴是高级搭讪的核心。
