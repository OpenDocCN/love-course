# 柯李思魅力原理：2020魅力原理『已更新完』：魅力原理第七集_note

在本节课中，我们将要学习“属性”这一核心概念，并了解如何通过着装来展示和转换个人属性，从而在不同社交场合中塑造理想的第一印象。

上一节我们介绍了“价值”与“能量”，本节中我们来看看“属性”如何影响人际互动。

## 1. 不同女性的不同属性 👩

以下是三个不同属性女性的例子：

*   **青春活力型**：这类女性给人感觉青春、漂亮。与她们恋爱会让人感到刺激、精彩，并能满足虚荣心。
*   **温柔顾家型**：这类女性带有浓厚的“妈妈”感觉，让人联想到会照顾家庭、做饭、照顾孩子的形象。
*   **妖艳短期型**：这类女性通常展现出开放、具有短期吸引力的属性。

## 2. 第一印象的重要性 👀

我们并不真正了解视频中的三位女性。她们的内心可能与其外在表现截然不同。然而，人们总是会凭借第一眼看到的印象和属性来做出初步判断，这就是第一印象的力量。

## 3. 男性的属性 👨

女性身上有属性，男性同样如此。女性看到不同类型的男性，也会在脑海中进行分类。

以下是男性常见的几种属性：

*   **短期关系属性**：如果男性身材健壮、帅气、爱玩又会玩，其短期关系的属性会非常高。`属性强度 = f(外形， 社交能力)`
*   **老公属性**：具备此属性的男性通常更加稳重、有担当、经济条件优越、情绪稳定、懂得呵护照顾女性。
*   **男朋友属性**：此项属性与老公属性相似，但各项指标（如年龄、成熟度、经济实力）相对低一些，与女方的差距不会拉得太开。

## 4. 属性的展示 🎭

着装打扮是转换个人属性最快速的方式。但着装必须与行为举止相匹配。

> **核心原则**：避免“高能量着装，低价值行为”。如果穿着吸引眼球（高能量），但行为却内向、不会社交（低价值），会带来更差的印象。

## 5. 着装的TOP原则 ⏰👥📍

选择着装时，需考虑TOP三要素：

*   **T (Time)**：时间
*   **O (Object)**：对象
*   **P (Place)**：场合

公式可以概括为：`合适着装 = 匹配(时间， 对象， 场合)`

## 6. AB款西服的应用 🤵

在不同场合，需要灵活调整着装的正式度。

*   **正式场合（如商务会谈）**：穿着标准西装、皮鞋，能体现专业、职业与敬业。
*   **半正式或休闲场合（如约会、见家长）**：可选择“AB款”搭配（如上衣和下装颜色/款式不同），或休闲西服（内搭T恤、不系领带、搭配小白鞋），以降低正式感，提升亲和力。

## 7. 不同场合的着装选择 🏞️🎉

着装必须与场合属性对应。

以下是几个例子：

*   **户外运动（徒步、野营）**：应以运动风、冲锋衣等为主。
*   **主题派对（如嘻哈音乐会）**：可穿着宽松衣物、佩戴球帽，以融入环境。
*   **见家长**：应避免过于夸张或带有强烈反叛属性的着装（如全身皮衣、骷髅头饰）。

## 8. 属性的转化与适应 🔄

我们并非专业演员，无法瞬间完美切换角色。但可以从着装开始，有意识地进行调整。

**实践方法**：为不同造型风格的服装，搭配对应的场景和对象。通过长期练习，该属性会逐渐内化，使你具备在不同属性间调适的能力。

正如视频中歌手“阿宝”通过更换西装、演唱英文歌，瞬间从“土气”转向“洋气”，虽然转换初期可能生涩，但充分展示了着装对属性与印象的强大转换力。
