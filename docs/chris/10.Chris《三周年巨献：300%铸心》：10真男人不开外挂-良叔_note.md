# 10.Chris《三周年巨献：300%铸心》：10真男人不开外挂-良叔

Hello大家好，我是梁叔，欢迎来到三周年的回馈客。今天分享2022年峰会中的一个新概念——外挂。

**什么是外挂**？

外挂是自信背后的东西。例如，你开劳斯莱斯、法拉利，佩戴百达翡丽、劳力士手表，穿着名牌服装，这些都属于外挂。

**外挂的危害**

1. **无法深度吸引**：开外挂可以吸引异性，但无法进入深度吸引。
2. **不自在**：开外挂的人会感到不自在，感觉与自身不兼容。
3. **虚假自信**：购买高仿奢侈品，认为这样可以获得自信，实则并非如此。

**为什么开外挂会不舒服**？

1. **提醒真相**：开外挂会让人意识到自己的真实面目。
2. **债务感**：为了维持外挂，需要不断展示好的一面，消耗心力。
3. **不真实**：无法做最真实的自己，心力被遮蔽。

**如何获得核心自信**？

1. **真实展示**：不经意间展示自己的优点。
2. **不做作**：不刻意追求外挂，做真实的自己。

**总结**

本节课中，我们学习了外挂的概念、危害以及如何获得核心自信。希望你们能从中受益，成为真正的真男人。
