# 搭讪大师《聊天之道》第64课：👑 阅男无数柜姐（十七）征服：女人最后的抵抗

在本节课中，我们将学习如何应对关系升级前的最后一道障碍——女性的“最终抵抗”。我们将通过一个完整的聊天案例，解析如何通过“推拉”、“给台阶”和“施加压力”等技巧，帮助对方合理化自己的行为，从而自然地突破最后的心理防线，完成关系的收尾。

上一节我们介绍了持续施加框架和推拉技巧，本节中我们来看看如何应对对方最后的犹豫与拒绝，并最终完成关系的升级。

---

**梁叔**：上个回合我不断揪住女生的框架，防止她口头逃避。我持续地将她推开。直到某天下午日落时分，她删除了我的微信。

**古医生**：你的方法确实很磨人。可以判断，如果进入长期关系，她会“折磨”你。

**梁叔**：这时她问“干嘛？”，我给她发了一本书的照片，书名叫《春风十里，不如你》。我说酒店刚好有这本书。她回复说这是她最喜欢的一句话。我说“有点巧”。她让我用心体会，说这句话很有意境。

我接着问：“你早班啊？”她回答：“对呀，咋了，5点起床的搬砖人。”并附上一张照片，说公司报销要自己做，早班没客人就在做报销。

我评论道：“这谁写的字啊，那么丑。”她回复：“我自己。小学生写字就这样子的，你不懂欣赏。”

于是我发了一张日落（或日出）的照片。她问：“你还在[某地]？”我回答：“前天回来了一趟，想见你一面的。”**这里开始“拉”**，因为前面推得太多，这个回合需要拉回来。她之前删了我，我正好借此找台阶。

**古医生**：是的，你没有回复她，而是自己找了一个台阶——“不联系了”。

**梁叔**：我说：“我今天回[某地]。”她问：“回[某地]？几点啊？”我说：“12点。”她确认：“飞[某地]吗？”我说：“对。”她回复：“好的。”她假装平静，实则内心兴奋。

我点破她：“假了，你给台阶了，马上就给台阶了。”她问：“那你有时间吗？我给你剪指甲。”

我回复：“明天。”她说明天休假，很想在家躺一天，最近上班太累。我说：“那先休息好。”**一个成熟的人不会因自身需求强迫他人迎合**。我表示理解。

**古医生**：当你允许别人休息时，窗口就出现了。她会觉得这个反应很好。

**梁叔**：她接着问：“你明天几点空？”我回答：“取决于你要睡到几点。我一般下午有空。”她提议：“要一起午饭吗？你就剪个指甲，还是时间比较多。”

我回复：“你早上多睡会，吃了再过来。时间到时候可以安排。”她发了一个“吹泡泡”的表情（我以前发过的），问：“下午干嘛呀？”我反问：“你过来不？”随后发了一个酒店定位。

她说：“还没有。”我建议：“那你吃了饭再过来。”她回复：“我都没有收拾。”我说：“猜到了。”她说已经吃完饭了，问我：“你下午要工作？”我答：“现在在工作。”她问：“到几点？”我催促：“你快收拾啊。”她说：“我要洗个头发。”

**古医生**：女人什么时候会洗头？当她想要精心打扮，在你面前呈现最美状态时。反正我跟你出差一般不洗头。她第一次见你时没洗头戴帽子，到后面就逐步开始打扮收拾了。

**梁叔**：她说要洗头，我回复：“那你要磨蹭到几点？”**敢于命令对方，源于自信和明确的吸引力**。她定下时间：“三点半到。”我说：“见我可以随意点，不要刻意。”她问：“下去哪里？”我说：“去酒店。”

**重点来了**。她问：“去酒店干嘛？”我直接表达：“剪指甲。”她拒绝：“不要。你来我家这边，我不要去酒店。”

我回应：“你不想单独聊聊？我觉得你对我有一些误解。”**开始给她找台阶**。她问：“为什么要在酒店聊？我怕你控制不住自己。”

我解释道：“我脖子拉伤了，做不了什么。你就担心你自己？我脖子都受伤了，能对你干嘛吗？你对我做什么还差不多。”**这是一个台阶**，她无法验证我脖子是否真受伤了。

她说：“酒店非常不安全。”这个“不安全”指可能在酒店被拍摄，也意味着她认为去酒店就可能发生关系。

我回复：“快过来，废话真多。”她坚持：“我不要。”我保证：“不做其他的，就是单纯的聊天。”她说：“我还未成年，不相信你。”我将其上升高度：“那信任出问题了。不相信，那见面意义就不大了。”**再次推开，并上升到关系维度**。

**古医生**：这里涉及一些调教内容，本次聊天中暂不展开。

**梁叔**：她说：“那就不见了吧，哪有约酒店聊天的，吓死人。”我说：“OK。”**又一次推开**。紧接着又拉回来：“我真是脖子拉伤了，想你按按。”**一边推开，一边刷心疼感，促使女生快速合理化**。

她说：“我又不会按脖子。你不应该找家推拿店治疗一下脖子吗？”我推开她：“你今天好好休息。”她回复：“好的。”

我没有回复。她主动发消息：“今天天气很好，我们不找个地方坐坐？”**她其实很想见你，只是需要合理化一个台阶**。

**古医生**：是的，她需要过去自己心里那一关。

**梁叔**：我坚持自己的想法：“我不会碰你，信不信由你。”她发了一个表情，**我没有回复**。为什么不回？这是重点。**不回复本身就是施加压力，逼她做出选择**。这运用了“沉默的力量”。

**古医生**：这会给对方一种感觉：如果这次不见，可能就没有下次了。同时也明确告知：去酒店，但我不会碰你，信不信由你。

**梁叔**：有人可能会问，到了酒店你真的不会碰她吗？为什么不能是她碰我呢？这其实就是给对方一个台阶。女人在矜持时，关系升级必然会遇到阻碍。我们要做的就是帮助她合理化，给她台阶下。

收尾是一个顺其自然的过程。防备心在工程完成后，自然会解除。最后一页聊天记录显示，女生到家后发了“后悔”二字，证明已经全部收尾。

这个回合是整个案例中聊天记录占比最大、讲解最细的部分。每一节都有侧重点，展示了如何处理破除防御的过程：给台阶、推开、允许窗口出现、运用沉默和压力，促使女生自己做出选择，打破防护罩。

**以下是本节课的核心要点总结**：
1.  **推拉结合**：在长期推开后，适时给予拉力，平衡关系张力。
2.  **提供合理化台阶**：当对方产生抵抗时，为其行为提供合理的解释（如“脖子受伤”），降低她的心理负担。
3.  **施加选择压力**：通过沉默、不回复等方式，制造紧迫感，促使对方尽快做出决定。
4.  **坚持核心框架**：即使对方犹豫，也要清晰表达自己的意图和底线（如“我不会碰你”），同时保持姿态。
