一个女性聊天的终极目的，是把男人约出来。从女性的视角来看，她与一个男人聊天，第一步永远是**筛选**。

因此，她的第一步永远是去筛选一个男性。

**关于“神秘感”与心态的过渡**

为什么需要保持神秘感？我们可以换个角度思考。

如果不放在两性角度，一个刚认识不久的人，在第一天就滔滔不绝地讲自己的过去，急于告诉你他是什么样的人，你会怎么看他？

首先，这个人缺乏边界感，急于套近乎。你会觉得他内心缺乏安全感，急切需要他人的了解和认同。

其次，你会考虑这个人是否对你有所企图。所谓“无事献殷勤，非奸即盗”。

所以，人际关系中需要一种距离感，这种神秘感就体现在这里。永远是走一步看一步。

女性也会通过你的这种行为，感受到你是一个有价值的人，一个能与她保持适当距离的人。

因为一个Beta男或屌丝，总是急切地想要获得他人（包括女性）的认可。

我们可以通过商品的角度来理解：如果一个推销员急切地想卖出商品，说明这个商品可能没有价值、无人问津。如果商品本身有价值，他会展示，但不会那么急切地想卖出。因为他清楚，你不买，总有下一个人会买。

这是一个心态层面的东西。女性通常具备这种心态，因为在这个社会里，女性的选择权往往大于男性，她们身边总不乏追求者。但很多男性不具备这种心态，因为他们内心匮乏，急切地想“把自己卖出去”，具体行为上就体现在“急于展示一切”这个点。
