# CHRIS搭讪达人计划：解惑篇08废物测试（彩蛋）_note

在本节课中，我们将学习几种简单实用的技巧，用于在搭讪现场化解女生的“废物测试”。这些技巧以娱乐感为核心，能让你更轻松地应对挑战。

上一节我们介绍了化解废物测试的三种传统方式。本节中，我们来看看两种更具娱乐性的现场技巧。

## 技巧一：模仿对方讲话

第一种技巧是模仿女生讲话的语气和内容。这能制造一种轻松、戏谑的氛围，让对方感到无奈，从而巧妙化解测试。

以下是具体操作方式：
*   **举例**：当女生说“你就是个小朋友，把你带回家，我妈肯定不喜欢你这样的。”
*   **模仿回应**：你可以用类似的、略带夸张的语气回应：“哎呀，你妈妈不喜欢我这样的？那我可得好好表现一下了。”

这种方式的核心是**加入娱乐感**，像开玩笑一样互动。

## 技巧二：假装信号不佳

第二种技巧是假装没听清或环境干扰，以幽默的方式回避问题的直接压力。

以下是具体操作方式：
*   **举例**：当女生提出一个棘手的测试时。
*   **幽默回应**：你可以说：“啊？你刚才说什么？好像信号有点不太好。”或者“风太大了，我听不清呀。”

这个技巧同样建立在**娱乐感**的基础上，能有效转移话题焦点。
