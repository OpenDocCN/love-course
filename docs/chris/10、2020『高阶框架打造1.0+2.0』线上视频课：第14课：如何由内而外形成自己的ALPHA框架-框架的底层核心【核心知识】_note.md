# 10、2020『高阶框架打造1.0+2.0』线上视频课：第14课：如何由内而外形成自己的ALPHA框架-框架的底层核心【核心知识】

## 概述
在本节课中，我们将学习如何由内而外形成自己的ALPHA框架，探讨框架的底层核心，包括信念管理和情绪管理。

## 信念管理
### 核心概念
信念系统越强，框架就越强；信念系统越弱，框架就越弱。

### 教程内容
1. **信念系统管理**：许多人的信念系统存在问题。通过视频素材（兰岛洞穴）展示信念系统的重要性。
2. **生活系统**：将生活视为一个系统，解决工作中的问题、家庭中的问题、社交中的问题，提升经验和能力。
3. **框架感营造**：通过解决生活中的问题，提升自身吸引力。

## 情绪管理
### 核心概念
情绪越容易被他人影响，框架就越弱。

### 教程内容
1. **情绪管理**：情绪管理对于建立强大框架至关重要。
2. **情绪影响**：不要试图管理他人的情绪，因为自己也无法控制。
3. **情绪管理素材**：通过视频素材展示情绪管理的重要性。
4. **情绪一致性**：情绪管理不良会导致言行不一致，削弱框架。

## 总结
本节课中我们一起学习了如何由内而外形成自己的ALPHA框架，探讨了信念管理和情绪管理的重要性。希望同学们能够将所学知识运用到实际生活中，成为一个能控制好自己情绪且具有信念管理的阿法男人。
