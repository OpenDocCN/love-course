# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存放数据。每个变量都有一个名字（即标签），我们可以通过这个名字来使用或修改盒子里的数据。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

```python
name = "Chris"
age = 25
```

以上代码创建了两个变量：`name` 和 `age`。`name` 这个“盒子”里存放了字符串 `"Chris"`，而 `age` 这个“盒子”里存放了数字 `25`。

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们还需要知道“盒子”里装的是哪种类型的数据。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其说明：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数、负数或零。
    *   代码示例：`count = 10`，`temperature = -5`

2.  **浮点数**
    *   表示带有小数点的数字，用于处理更精确的数值。
    *   代码示例：`price = 19.99`，`pi = 3.14159`

3.  **字符串**
    *   表示文本信息，需要用单引号 `'` 或双引号 `"` 括起来。
    *   代码示例：`greeting = "Hello, World!"`，`author = 'Chris'`

4.  **布尔值**
    *   只有两种取值：`True`（真）和 `False`（假），常用于逻辑判断。
    *   代码示例：`is_student = True`，`has_completed = False`

## 如何查看和转换类型？ 🔄

我们有时需要知道一个变量的类型，或者在不同类型之间进行转换。Python提供了内置函数来完成这些操作。

*   **使用 `type()` 函数查看类型**
    你可以使用 `type(变量名)` 来查看任何变量的数据类型。
    ```python
    print(type(age))    # 输出：<class 'int'>
    print(type(name))   # 输出：<class 'str'>
    ```

*   **使用类型转换函数**
    以下是常用的类型转换函数：
    *   `int()`：将数据转换为整数。
    *   `float()`：将数据转换为浮点数。
    *   `str()`：将数据转换为字符串。
    *   `bool()`：将数据转换为布尔值。

    ```python
    number_str = "123"
    number_int = int(number_str)  # 将字符串"123"转换为整数123

    score = 95
    score_str = str(score)        # 将整数95转换为字符串"95"
    ```

## 让我们动手试试 ✍️

理解了概念后，最好的学习方式是实践。请看下面的代码示例，它综合运用了本节课的知识点。

```python
# 1. 定义变量
user_name = "Alice"
user_age = 30
user_height = 1.65
is_new_user = True

# 2. 打印变量及其类型
print("姓名:", user_name, "类型:", type(user_name))
print("年龄:", user_age, "类型:", type(user_age))

# 3. 进行类型转换
age_str = str(user_age)  # 将年龄从整数转换为字符串
message = user_name + " is " + age_str + " years old."
print(message)  # 输出：Alice is 30 years old.
```
