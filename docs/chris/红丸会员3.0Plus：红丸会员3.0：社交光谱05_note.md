# 红丸会员3.0Plus 课程05：面子、里子与底子 🎭

在本节课中，我们将通过一个生动的餐厅故事，深入探讨东方文化中“面子”、“里子”与“底子”这三个核心概念，并学习如何在社交互动中运用这些智慧。

---

上一节我们介绍了“面子”与“里子”的基本概念。本节中，我们通过一个日常生活中的具体事例，来进一步理解这些概念的实际应用。

这个故事发生在一家生意兴隆的餐厅。一位服务员遇到了一位傲慢的顾客前来购买汉堡。由于餐厅生意很好，汉堡通常需要排队很久才能买到。这位顾客排了一个小时的队，但轮到他的时候，服务员告知：“不好意思，我们今天的汉堡卖完了，你下次再来吧。”

如果你是这位顾客，你会感到生气。这位顾客确实生气了，并且脾气不好，开始大吵大闹。他说自己白白等了一个小时，结果什么都没买到。他的吵闹引来了众人的围观。

此时，如果你是这位服务员，你会怎么做？

通常有两种做法。

以下是第一种常见的应对方式：
*   坚持让客人回去，让他明天再来买。

这种做法不能说是错误的。因为作为服务员，你是按照公司规定行事。卖完了就是卖完了。你坚守了自己按规章办事的原则。

然而，在东方文化背景下，第一种做法非常不明智。虽然在人前你做得没有错，但在人后你却输了。你没有顾忌到客人的面子。同时，周围围观的人也会对这家店的生意产生负面评价和影响。

顾客此刻内心真正想要的，并非一定要吃到那个汉堡，而是为他白白排队一小时讨一个说法。

因此，一个情商高、聪明的服务员会采取第二种做法。

以下是高情商的应对方式：
*   将这位客人单独拉到一边。
*   告诉他：“不好意思先生，今天汉堡确实卖完了。但请你改天再来，我出于个人会给你优惠。下次你来买汉堡时，我会赠送你饮料和小吃。”
*   同时请求对方：“请不要打扰其他客人，也不要将我单独给你优惠的事情告诉其他客人，否则我会很难做。”

这样一个简单的承诺，既给足了对方面子，也完美化解了矛盾。虽然你在人前看似输了，但在人后其实是你赢了。你化解了这次危机，未来甚至可能因此被提拔为店长。

所以说，Beta智慧属于“人前法”，核心是给足对方面子。而“里子”和“底子”则属于“人后法”，更像是一种Alpha智慧。

因此，有一句话叫做：**成年人只看利弊，不看对错**。在人前争一个是非对错没有意义，只是一种非常幼稚的表现。以这件事为例，如果你在人前争输赢、争对错——坚持“汉堡就是没了，你无理取闹是你的问题”——即便你争赢了，对你个人利益有任何增长吗？其实并不会。

这就是我们要学习的Beta智慧。

---

最后，我们来谈谈“底子”这个概念，它同样至关重要。

“底子”即**价值**。那个小小的服务员之所以不敢擅自做决定，就是因为他没有足够的底气。他在店里可能不具备足够的话语权，个人价值不够高。而这家店生意很好，有时允许你赶客人走，都是因为这家店有底气、有价值。店长拥有更大的话语权，这也是底子层面的东西。

所以，你需要了解到，底子固然重要。而**社交智慧本身也是一种底子**，它是一种软实力，是一种无法被简单量化的智慧，但确实是个人实力的一部分。

如果你是一个小服务员，虽然不能擅自做决定，但你可以这样说：“我可以帮你向店长争取优惠。”这也是一种高情商的做法。

---

在中国式社交中，存在几种不同的局面。

以下是几种常见的社交局面：
*   **人前赢，人后也赢**：这种局面要求你的底子必须足够硬。但这样做很容易遭人嫉妒，被批评为“冠冕堂皇”。
*   **人前输，人后赢**：这是大部分中国企业家拥有的智慧。代表人物如李嘉诚、马云（辞任阿里巴巴CEO）。他在人前像一位“乡村教师”，似乎没有面子，但实际上是阿里巴巴的控股人。
*   **人前赢，人后输**：这是最不明智的一种。这种人往往是政治斗争中的牺牲品，被高官权贵当作棋子利用。例如电影《让子弹飞》中的老六，为了争辩自己“没有多吃一碗粉”这个对错，最终剖腹自证，壮烈牺牲。他的牺牲毫无意义，只是政治斗争的牺牲品。

上述三种情况中，最好的其实是第二种。

李嘉诚有一条生意智慧：和别人做生意时，永远让对方觉得是自己占了便宜，也就是“让利”。这其实就属于一种Beta智慧，是“人前输，人后赢”。因为如果是第一种“人前人后都赢”，完全不给对方留余地，你就在无形中埋下了怨恨的种子，最终会在其他地方受损，得不偿失。

刚开始学习红丸理论的兄弟，可能会领悟到自己过往“太好”，属于一种讨好人的人格。因此对长辈“吃亏是福”的教诲非常反感，认为正是这样才会永远被欺负、占便宜，抬不起头。

但事实上，“吃亏是福”是一句非常具有哲理的话。真正的吃亏指的是“人前输，人后也输”。但如果你能做到“人前输，人后赢”，这恰恰是一种极高的智慧。

为什么古代领主和君王讲究“君权神授”？因为一国之君再大，也要在教皇或宗教领袖面前低头。原因很简单，这就是一种Beta智慧。连最高权力之巅的人也运用这种智慧。

教皇没有实权，国王拥有实权。所以，**教皇看似比国王大（人前赢），但实则人后输；国王在人前低头（人前输），但实则人后赢**。

相反，如果国王不信奉“君权神授”，灭了教皇，虽然他做到了“人前人后都赢”，但他的皇权也可能被实力更强的人取代。因为篡位者登基会变得“合理”：只要我实力更强，灭了最有权力的人，我就可以篡位。“君权神授”的规则就被破坏了。

所以曹操是个非常聪明、具有Beta智慧的人，他“挟天子以令诸侯”。相反，袁术则没有Beta智慧，是典型的反面教材。他妄自称帝，名不正言不顺，在两年半内就被消灭了，成为史上在位时间最短的皇帝之一。

这足以说明，在金字塔顶端的Alpha男性，同时也具备Beta智慧。过去我们更多讲述的是Alpha智慧。在红丸3.0中，我们将重点探讨“人前”的Beta智慧。
