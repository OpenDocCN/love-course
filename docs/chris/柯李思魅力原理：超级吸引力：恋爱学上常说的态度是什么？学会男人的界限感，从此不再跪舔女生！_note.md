## 第一部分：清晰的界限感就是你的态度 🚧

上一节我们概述了课程内容，本节中我们来看看什么是“态度”。在恋爱学中，我们所说的“态度”，其核心就是拥有清晰的界限感、属于自己的标准和原则。

界限感可以理解为：你知道自己能接受什么，不能接受什么。具体可以分为以下四个方面：

以下是关于界限感的四个核心方面：

1.  **我自己做什么行为，我自己能够接受。**
    *   例如：作为一个男生，我能接受自己化妆、打扮得很潮、修眉、吃西餐或请女生吃小龙虾。这些是我的个人标准。

2.  **我自己做什么样的行为，我自己不能接受。**
    *   例如：我绝对不能接受穿着脏衣服出门，也不能接受女生让我给她发红包。这些是我的底线。

3.  **别人对我做什么样的行为，我可以接受。**
    *   例如：长辈给我夹菜，或者朋友开一些不过火的玩笑，我可以接受。

4.  **别人对我做什么样的行为，我不能够接受。**
    *   例如：别人开很过火的玩笑、女生让我发红包或买我经济能力无法承受的东西、女生要看我的手机，这些行为我不能接受。

大部分男人的问题是：别人对他做什么行为，他都能接受；他自己做什么行为，也都能接受。他几乎没有“不能接受”的事情。例如，别人开过分的玩笑他能忍，女生要红包他给，女朋友查手机他同意。这都代表你是一个没有原则、没有界限感的男人。
