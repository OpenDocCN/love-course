“同理心陈述”的核心是：通过完整的语言，把对方对你的戒备心理说出来，并表明你并无威胁，从而化解她的顾虑。

许多学员询问搭讪是否有明确的流程。答案是肯定的，一个清晰的流程可以大大提高成功率。

**1. 打开**
“打开”是指你成功吸引组合的注意力，并让他们对你产生基本信任，越过了“上钩点”。标志是对方愿意正面回答你的问题，并提供一些基本信息。

**2. 前提评估**
在女生愿意聊天后，你们会进行基本信息交换（如：为什么在这逛街？听口音是哪里人？）。在此过程中，你需要加入“前提”，即将对话引导到男女交流的层面。

**3. 收尾**
流程的最后阶段，根据互动情况自然收尾，可以是转场、收号码、即时约会或转场约会。
