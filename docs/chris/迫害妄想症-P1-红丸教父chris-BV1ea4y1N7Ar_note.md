# 迫害妄想症 - 课程P1：剖析女权主义的归因错误 🔍

在本节课中，我们将探讨一种被称为“迫害妄想症”的社会现象，特别是在某些女权主义论述中的体现。我们将分析其核心论点，并学习如何从辩证关系的角度审视权利与义务的对等性。

## 核心论点：被夸大的“压迫”

上一节我们介绍了课程主题，本节中我们来看看其核心论点。某些女权主义观点认为，女性在历史上始终处于被男性压迫的境地。

但根据辩证关系来看，这种压迫其实并不存在。女权主义者常将历史处境归因于男性压迫。

以下是常见的归因举例：
*   **裹小脚**：女权主义者称这是男性对女性的压迫。
*   **禁止女性读书**：同样被归因为男性的压迫行为。
*   **中世纪贵妇束腰**：被视为男性审美对女性身体的摧残。

## 流行文化与自愿选择

上一节我们讨论了归因于压迫的案例，本节中我们来看看另一种视角。许多历史现象其实是当时流行文化与时尚的产物，女性是自愿参与的。

裹小脚或贵妇束腰，是当时的一种流行时尚文化。女性是自愿这么去做的。

我推测，如果女权主义继续发展，未来可能会对现在的时尚做出类似解读。

以下是基于此逻辑的推测举例：
*   **身体穿孔与纹身**：未来或被称为“父权压迫”。
*   **染发与美甲**：也可能被定义为“父权审美”的压迫。

这非常可笑。女权主义最大的问题在于**归因错误**。它将所有问题总结为男性父权的压迫，但事实并非如此。

## 辩证关系：权利与义务的对等

上一节我们指出了归因错误的问题，本节中我们来看看为什么必须从辩证关系思考。我强调从辩证关系看问题，原因非常简单。

女性没有权利的同时，也享有**福利**。当女性不需要承担某些权利对应的责任时，她也免除了相应的**义务**。

**权利与义务是对等的**。

为什么女权往往只强调权利，不强调责任与义务？原因就在于此。
