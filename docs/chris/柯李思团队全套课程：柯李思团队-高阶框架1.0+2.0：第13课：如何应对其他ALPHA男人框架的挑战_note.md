# 柯李思团队全套课程：高阶框架1.0+2.0 🛡️ 第13课：如何应对其他ALPHA男人框架的挑战

在本节课中，我们将学习如何识别并应对社交互动中来自他人的框架挑战。我们将通过分析一段视频素材，理解不同角色的框架表现，并重点学习一种名为“非常规框架”的应对技巧。

前面的课程讲解了框架的结构以及推拉的结构。本节中，我们来看看如何应对框架的挑战。

下面我们来看一段视频素材。

视频中，两个男生对一位女生说：“是吗？很可恶，很可恶。女生一向都知道你不和她们说话。最糟糕的是你试着假扮，因为你真是个好男人，而且你确实有一个能为了Legra做的东西，而不是其他男人。”

随后，他们问女生：“你叫什么名字？”女生回答：“Egan。”男生又问：“你认为在西边的Jets新的足球场所的设计是什么？”女生刚准备回答，男生打断说：“让我猜猜吧。可恶。”

分析完毕，我们回来。

大家看完了。以下是不同角色的框架分析。如果觉得形容框架不太好形容，可以换个词，思考他们各自的表现是什么。

在视频中，这两个男生的表现很明显是想捉弄这个女生，让她出丑。他们问问题但不等女生回答，就说“恶心”。这摆明了就是想捉弄女生，**他们的框架就是“我要捉弄你，让你难堪”**。

女生的框架也很明显。两个男生问她，她很好地去应和。另外一个男生问她问题，女生也很好地应和了。没有等女生说完，男生又说了一句“恶心”。女生此时所表现出来的框架，就是社交场合中迎合别人。但面对这种捉弄和打压，她没有采取什么措施，**她的框架是相对弱势的**。

那么胖子的框架是什么？一个是回忆黑人史密斯给她说的话，“她是不一样的”。她在社交场合中想展现自己不一样的地方。面对别人这种打压和欺负，自己又是女生身边的男生，那自然充当了一个护花使者的角色。

于是她把这个话以同样的侮辱和打压的方式，回敬给了这两个男的，然后走开了。做得很好，没有去停留。男生说“你仔细想想，我们待会再回来”，这个就应对得很好。**她的框架维持和维护得比较好，同时也没有引发任何冲突**。这里是非常值得大家去学习的。

这也是社交当中在女生心中非常加分的一种社交行为信号，处于社交定位点比较高的一种位置。

下面来讲解一个核心技巧点，叫做“非常规框架”。

有一种情况是，你需要先进入到对方的框架里面。你进入到对方的框架里面之后，用对方的框架去反转对方的框架。**核心逻辑是：让他用自己的东西证明他自己的东西是不合理的**。

前面视频当中，胖子的做法就是一个非常规框架，让对方框架自相矛盾。这就是非常规框架推拉。

具体指的是，我们需要进入到女生的框架里面去，然后用女生的框架证明她自己的框架是不合理的。有一种做法就是用她的矛去攻她的盾，最后让她感觉自己是自相矛盾，达到这样的效果。

本节课很多方面都涉及到了框架识别，包括人的做法和内心想法背后的框架。框架代表了什么，以及不同的场合，人和框架都是会变的。

本节课中，我们一起学习了如何分析社交互动中的框架挑战，认识了强势框架、弱势框架和维护框架的行为。更重要的是，我们掌握了一种高级应对技巧——**非常规框架**，即通过进入并利用对方自身的框架逻辑，使其自相矛盾，从而优雅地化解挑战。
