## 生活动力与自我管理 🔄

大家有什么问题可以问一下，今天这个课程是一个答疑的课程。我们课程的重点课题已经全部讲完了。接下来之后会有两个课程，我会讲一些新的内容，包括一些床上的技巧，以及男生的一些自我保健的方法，一些密传的方法。之后几节课我会准备一下到时候给大家讲。今天我们就主要以答疑为主。

那么我先来讲一讲，第一个就是，你是不是好好地找到了自己的这个最重点的一件事情要去做。就比如说，有些人喜欢运动，运动可以是他意识当中最重要的一件事情。那有些人觉得事业是他最重要的事情。你就是要找一件事情作为你生命的方向。我们总活着总需要一点方向。

还有一点就是，如果说你一直感觉心情不好，没有动力去做事情，还有一点就是你或许可能需要调整一下你的作息时间，以及关注一下你的一些健康问题。因为有时候我们会被有些东西忽略掉，就是我们有时候脾气不太好，很烦躁，听到有些人听到一些声音都觉得很烦，人旁边说话都觉得很烦。这其中有一部分原因可能是因为我们身体的一些状况所引起的，可能并不是因为真的觉得烦。

这个就讲得有点复杂了，当中就有点涉及到一些医学上的东西，这个我就不便于多讲，因为我也不是特别专业。但是这个会跟你身体有一定的关联。就比如说有人经常会抑郁，那么他也会有身体各方面机能出现一些状况，会导致抑郁。焦虑也是因为身体的某些状况。

那这个哥们，我就这么跟你说吧，真的非常非常有感觉的女生，不是说我找一个就是一个。肯定我们需要在茫茫人海中，去找到那个让自己最有感觉的女生。所以真的要让你找到那个很有感觉的女生，不是一件那么容易的事情。除非你是包办婚姻，自己家里就帮你准备好一个女生让你结婚，否则你真的要找一个很有感觉的，除非你运气好一下就能碰到，否则的话还是比较难。

然后呢还有一点就是，你不一定一定要完全追求那种劲爆的。为什么完全要劲爆？生活最美的地方不是在于它的轰轰烈烈，而在于它平淡安静的时候。你能够静下心来，能够观察身边事物的一切，很安静时候那种美，这个有时候也是很令人幸福的一件事情。并不代表劲爆才一定是能够让你愉悦，它只是一种刺激，它是一种欲望。我们人活着有时候就是需要去控制、去收敛自己的欲望。如果你的欲望无限地去膨胀的话，它必然会毁了你。所以说我们时刻地要去收敛自己。
