# 柯李思团队全套课程：CHRIS搭讪达人计划：达人篇01肯定感练习_note

在本节课中，我们将学习如何从搭讪新手成长为达人。课程将通过一个真实的学员案例，展示新手在搭讪中遇到的典型问题，并手把手指导如何突破。核心在于掌握并练习“肯定感”，这是成功开启对话的第一步。

从新手变为达人，欢迎来到搭讪达人课程。本产品记录了由我亲自指导学员小王，帮助他从新手逐步突破，最终成为高手的完整过程。在本视频中，你会发现你所遇到的问题，很可能也是小王曾经遇到的问题。

这些问题，我会在视频中进行手把手指导，告诉新手应该如何去做、怎么做、以及如何突破。首先，我想问你，你之前有搭讪的经验吗？小王之前也有搭讪过，但次数比较少。

并且搭讪时一个很突出的问题，就是会感到紧张。我们先来看一下小王自己去搭讪时，是什么情况。他的搭讪过程如下：“嘿，哈喽，我觉得眼镜挺有个性的。嘿，嘿，你结婚了吗？没有。这个是妈妈？对，对。哈喽，你穿的是……我加微信？你为什么一上来就会把我当成加微信呢？啊，那你靠边吧。嘿，你结婚了吗？结婚了呀。看你穿的挺性感的，然后过来给你打个招呼，因为我以为你是没结婚的。我结婚了我孩子都上小学了。哦，好吧，拜拜。”

我刚才看小王去搭讪，全程也在听。我发现有一个最大的问题，就是女性目标不愿意停下来。在大部分情况下，我们一开始搭讪时，第一个要点就是要吸引女生的注意力。

我看到有一些女生给到了你注意力，比如她看了你一眼。但当你继续想跟她说话时，她又回到了自己原本要做的事情的状态，然后径直往前走。这是我认为在一开始的时候，大部分新手都会面临的问题。

他们往往会卡在这个阶段很久，甚至感到非常受挫。因为他经常去搭讪，感觉女生都很高冷、不理他，感觉自己是在热脸贴冷屁股，很打击自信心。

其实，这就是我们在搭讪时需要做到的第一个关键点：**肯定感**（或称确信感）。很多时候，举个例子，当你去搭讪一个目标、一个女生，上前要跟她讲话时，你自己内心都不是非常确信自己到底要做什么。你有这个感觉吗？

这可能源于不同的信念。有些人可能觉得做搭讪这件事很奇怪、不好、甚至很猥琐。他会感觉自己上去是不是打扰了对方。如果他抱着这样的想法和信念，他的肯定感显然就不足。

第二种情况是他不知道上去该做什么，有时就是不知道该说什么，脑袋短路了。你可能也会有这种情况，比如今天刚出门搭讪时，前几次尝试脑子根本没打开，一片空白。

你不知道上去要说什么，所以即使你把对方叫住了，自己也不知道该说什么。这个时候，你的肯定感也是不足的。

我们先从这一个点开始讲，如何训练这种肯定感。要让一个人100%确信自己上来到底要干什么。我们可以参考一些生活场景，比如在马路上看到的交警，或者地铁站门口查身份证的警察。

那些警察的肯定感就很强。他可能上来的瞬间，你就知道他想要对你做什么，知道他到底要上来干嘛。所以，我们前期练习肯定感，也可以做类似的练习。

我们现在来到了一个地铁口。我想到了一个很有意思的练习肯定感的方法。接下来，你站在这个地铁门口。因为这是一个地铁口，会有人进来。你要做的是，练习一件你确信要做到的事情。

记住，你现在必须非常确信一定要做到这件事：**禁止他们往这个口里走，让他们往那个口进去**。如果他们想往这个口走，你就说“这里不能走”，并指引他们去那个口。你先练习一下，我看看你是怎么做的。

如果他们问为什么不能走，你可以说“这里在修”。具体怎么说看你自己，但关键是你必须**很确信**。一般来说，如果你很确信，他们应该不太会问。就像你在路上遇到一个要查你身份证的人，你通常也不会问“为什么要查”。

（学员练习过程）：“这里不能走，这里不能走。诶，那边，往那边。那边那边，那边。那边，这边堵住了，那边进地铁口。那边，你上地铁是吧？那边，那边进地铁口，这边堵住了。那边进地铁口，那边那个口，这边堵住了。这边在修吗？那边进地铁口，地铁口在那边，那边进，这边堵住了。那下去，堵住了。”

好，你刚才做了这个练习。我看做得还可以，大概可以打60分。我注意到有一个女的，你说“这里不能走”对吧？然后她看了一眼，又绕到你后面去了。那个时候，你的肯定感就不足了。第一下比较确定，但之后就下降了。

肯定感取决于你怎么去想、怎么去思考，取决于你的信念如何看待这个事情的重要性。例如，为什么有些人去搭讪时对事情不确信？因为他可能觉得“我去搭讪不会成功”、“我要不到号码”，或者觉得做这件事很猥琐、是在索取别人。

原理是一样的。你需要把你现在做这个行为的内在确信度再提升。我举个例子：如果这个口下面有个杀人犯，下去就会死，你会不会阻止他们下去？你肯定会拦住他，不能让他下去。你的表现绝不会是现在这个样子。

你对一个事情的确信感，取决于你多么相信这个事情的严重性。好，那我们再练习一次。

（学员第二次练习过程）：“嘿嘿，这边堵住了，那边修，那里再修，下面再修，下面再修，这边。地铁口吗？再修。OK，下面再修，下面再修，地铁口再修。”

是这样的：我刚才看你第二次的表现，还不如第一次。我看到你去阻止他们进去时，给人的感觉并不是真的不想让他们进去。你懂我意思吗？你心里知道再做这个事情是假的，连你自己都不确信这是真的。所以你的肯定感明显不足，他们进不进去你都无所谓。

你不是很坚定、确定地认为他们确实不能往这里走。你没有把这种感觉表现出来。这种潜在的沟通没有传递出去。这其实放在搭讪里也是一样的：你是否真的确定要上去跟那个女孩说些什么？

就这个练习行为来说，如果你换成搭讪，上去要跟一个女生说话。如果你非常确定——比如你非常确定进来的人不能进来——你可能还没说“这边在修”，对方就会停下来。他会感觉你好像有很严肃的事情要警告他或告诉他。

你会把这种感觉传递给对方。在搭讪时也是一样。如果你非常确定你要上去认识她、跟她说话，可能你还没讲出目的，你上去的瞬间她就会停下来，感觉你要说些什么。“哎，怎么了？” 你有没有遇到过这种搭讪情况？对方说“哎，怎么了？什么事情？” 但这前提是，你要对这个事情非常确定。

练习这个的目的，就是要练习这种“非常确定”的感觉。不是说你是在拍恶搞视频或者搞笑，不是这个意思。你要去感受，当别人进来时，你真的从内心确定这个事情。如果你真的确定，你是可以影响到他的，他会感觉“啊？怎么了？下面发生什么了吗？”

但从我现在看到的表现，以及路人给你的反馈来看，他们并不相信你说的话。他们完全不信，感觉你好像有点毛病，然后就走了。这说明你自己对这个事情的肯定感相当不足。好吧，所以你再试一次。

（学员第三次练习过程）：“哎哎哎，下面再修，下面再修，下面再修，下面再修。哎哎哎，这边下面堵住了。嘿嘿嘿，下面堵住了，那边去地铁口是那边。”

---

本节课中，我们一起学习了搭讪中至关重要的 **“肯定感”**。我们分析了新手因内心不确定（如觉得打扰对方或不知该说什么）而导致目标不愿停下的问题。通过 **“地铁口拦截练习”**，我们模拟了如何像警察一样，通过坚定的信念和外在表现，将“我确信此事重要”的感觉传递给对方。关键在于，你的确信程度决定了对方是否会认真对待你。在后续练习中，请不断内化并提升这种对自己行动目标的坚定信念。
