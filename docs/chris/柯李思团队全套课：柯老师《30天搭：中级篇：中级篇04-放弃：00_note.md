# 课程名称：柯李思团队《30天搭讪计划》大师班·中级篇04 - 放弃开场白，学习自然开场 🚀

在本节课中，我们将要学习如何摒弃固定的开场白套路，转而掌握最自然、最灵活的搭讪开场方式。真正的搭讪高手没有预设的台词，他们能利用周围的一切作为开场。我们将通过三个核心原则和实战演示，帮助你达到这一境界。

---

在初级训练中，我们介绍了三种具体的开场方式。然而，依赖固定开场白会限制你的应变能力。本节中我们来看看如何突破这一限制，实现真正的“无招胜有招”。

真正的搭讪高手没有任何固定开场白。这意味着你可以利用任何事物作为开场，这是最自然的搭讪状态。

很多人会问：如果没有开场白，那该说什么？

这里告诉大家三个开场原则。

以下是三个核心的开场原则：

1.  **描述客观事物**。例如，你可以根据女生的穿衣打扮、身高外貌作为开场。
2.  **描述心情**。你可以根据自己当时的心情，或推测对方的心情来开场。
3.  **描述当下的情景**。这是最常用的技巧，根据对方正在做的事情或周围环境来开场。

描述客观事物，比如你看到一个女生，她的穿衣打扮，还有她的身高外貌，你可以以此作为开场。

描述心情，你可以根据你自己的心情，或者是这个女生当时的心情，来作为开场。

描述当下的场景，也是最常用的一个技巧。比如说，你看到这个女生当时在干嘛，你就可以以此开场。比如女生在喝咖啡。

比如她在等公交车，你就可以上去问：“哎，你这个茶好喝吗？”或者“你在这里等车，等了很久吗？哪班车还没有来？”都可以作为开场。
