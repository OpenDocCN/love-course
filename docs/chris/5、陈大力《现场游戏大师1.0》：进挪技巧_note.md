## 进挪的四大类型

进挪并非盲目进行，我们可以将其分为四种类型，以便于理解和操作。

以下是四种主要的进挪类型：

1.  **礼节型**
    *   **定义**：社交礼仪性质的触碰，不带性意味。

2.  **保护型**
    *   **定义**：在特定情境下，出于关心和保护目的的触碰。

3.  **亲密型**
    *   **定义**：表达好感和亲密意图的触碰，涉及更多私密部位。
    *   **示例**：第一次约会可触碰的非私密区域包括：脸部（如轻捏脸颊）、手臂（大臂、手肘、手腕）、背部（轻推引导）、同侧肩膀。**注意**：胸、臀、大腿、腰腹部等属于私密区域，在关系初期应避免触碰。

4.  **无意识型**
    *   **定义**：看起来自然、随意、非刻意设计的触碰。
    *   **示例**：聊天大笑时轻碰对方手臂、并肩行走时偶尔的肢体接触、引导方向时轻触背部。这类进挪最自然，也最不易被拒绝。
