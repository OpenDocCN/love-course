# 07.阿尔法★求爱应急指南：第七课《被动聊天法》女生微信不回你消息怎么办？

在本节课中，我们将学习如何使用被动聊天法来应对女生微信不回消息的情况。

## 1. 主动聊天与被动聊天

**主动聊天**是以自己为中心进行聊天，例如询问对方在干嘛、吃饭了吗等。这种方式属于硬破冰，容易让对方感到压力，失去聊天的兴趣。

**被动聊天法**则是让对方感觉到聊天是由对方发起的，从而激发她的兴趣。

## 2. 被动聊天法的四种开场

以下是四种能够激发女生主动聊天的开场方式：

### 2.1 朋友圈聊天法

当女生发朋友圈时，这是她最想聊天的时候。根据朋友圈的配文进行评论，例如：“所以你是打算继续宅在家里还是找点乐子呀？”

### 2.2 刨根法

通过提问让对方解释自己的行为或言论，例如：“我记得你上次说女生不喜欢找比自己高的女生，我想知道是你这样想的，还是所有女生都这样想？”

### 2.3 斗逼法

通过有趣的对话让对方笑出来，例如：“呼叫二狗子，呼叫二狗子！”

### 2.4 求助法

在某个领域寻求对方的帮助，例如：“我请教你一下，你觉得这两套衣服哪套更适合在聚会上穿？因为我看你平时真的很会穿搭。”

## 3. 对方不回消息怎么办

如果对方不回消息，不要连续发信息，以免让对方感到压力。可以尝试轮换使用不同的聊天方法，保持新鲜感。

## 4. 放鸽子问题

对方放鸽子是正常现象，不要过于在意。将对方列入轮流名单，继续与其他人聊天。
