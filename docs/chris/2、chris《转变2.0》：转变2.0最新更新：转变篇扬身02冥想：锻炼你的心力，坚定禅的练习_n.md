## 记忆的两种形式：图像与情绪

上一节我们介绍了主动转化，本节中我们来看看被动转化。在讲解被动转化之前，我们需要先理解记忆是如何形成的。

我们之所以会形成心理“刻印”，通常源于童年时期的原生家庭经历或其他事件。这些不幸的事件被记录下来，导致我们日后戴着“有色眼镜”看待世界，认知产生偏差，并对某些人、事、物产生负面情绪和偏见。

记忆主要分为两种：

1.  **图像记忆**：指对具体场景和画面的记忆。
    *   例如，童年时与朋友玩耍不慎落水，朋友却视而不见。回忆起溺水时的具体画面，就是图像记忆。

2.  **情绪记忆**：指事件所伴随的情绪感受被身体记录下来的记忆。潜意识具有自我保护功能，可能会模糊痛苦的图像记忆，但相关的情绪却被身体牢牢记住。
    *   例如，小学时因表白被全班嘲笑，那种羞耻和痛苦的情绪被记录下来。导致日后与异性交流时，会不自觉地触发同样的负面情绪，阻碍正常社交。

我们要清理的潜意识创伤，主要就是这些**情绪记忆**。

## 情绪记忆储存在何处？

那么，这些情绪记忆究竟储存在哪里？很多人觉得情绪很抽象，似乎存在于胸口或大脑。

实际上，**情绪记忆被你的身体记录下来**。当你产生情绪时，身体会伴随一系列生理变化，这些变化被全身的肌肉和细胞所记忆。

例如：
*   生气时，你会呼吸急促、肾上腺素升高。
*   焦虑时，你可能会心慌、胸闷。
*   面对心仪的异性时感到害怕，这种情绪也是通过身体的紧张、心跳加速等反应表现出来的。

因此，要清除潜意识的情绪记忆，**关键要从身体入手**。

## 核心练习：坚定禅 🪷

以下是清理潜意识刻印的关键练习——“坚定禅”。建议在完成第一课练习并持续一个月后，再进行本练习。

这个练习要求你以盘腿姿势（单盘或双盘均可）进行冥想，全身保持正直，头部抬起，然后开始观察呼吸。

在练习过程中，你会不可避免地出现几种不舒适的身体感受：

*   身上发痒。
*   夏天冥想时流汗。
*   盘腿导致的腿麻、腰酸等难以忍受的感觉。

通常，我们的本能反应是去缓解或逃离这些感受，例如抓痒、擦汗或调整姿势。这恰恰是潜意识保护机制在起作用。

“坚定禅”练习的核心在于：**在至少一小时的冥想过程中，保持姿势一动不动，无论升起何种不适感。**

## 如何应对升起的感觉？

当不舒适的感觉出现时，这就是练习的重点：**如实地观察它**。

请注意，观察意味着不加以批判。例如，当身上发痒时：
*   **错误做法**：心想“好痒，太讨厌了，真想抓一下”。这种抗拒会加剧痒的感觉。
*   **正确做法**：只是客观地觉知，“我注意到身上有一个痒的感觉”，然后像科学家通过显微镜观察微生物一样，平静地观察这个感觉本身。

当你能够如实观察时，会出现一个神奇的现象：**生起即灭去**。你会发现，痒的感觉随着你的观察，逐渐减弱直至消失。对于腿痛、腰酸等其他感觉，也采用同样的观察方法。

这个观察并体验感受“生灭”的过程，就是内在清理的开始。如果你始终在抵抗和厌恶这些感受，就无法掌握这个重要工具，也就无法在后续课程中清理潜意识。

因此，你必须先培养出对一切身体感受保持观察，并体证其生灭过程的能力。这是至关重要的第二步。
