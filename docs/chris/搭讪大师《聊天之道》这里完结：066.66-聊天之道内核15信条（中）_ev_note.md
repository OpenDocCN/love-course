## 第六条：避免“极度禁止”与“极度允许”🚦

上一节我们介绍了前五个信条，本节中我们来看看第六条：避免“极度禁止”与“极度允许”。这两种极端心态都会阻碍你在聊天中的自然表现。

**极度禁止**是指在聊天中过度限制自己的行为和表达。具体表现如下：

以下是“极度禁止”心态的常见表现：
*   发送消息前反复纠结，担心说错话或显得笨拙。
*   在关系中过度压抑自己的需求，例如从不迟到、不向对方提出任何要求。
*   在社交或实战中不敢主动与女生交流，即使对方释放出兴趣信号也不敢升级关系。
*   禁止自己表现出真实的状态和感受。

**极度禁止**的公式可以概括为：**内心有想法 → 行为上禁止 → 变成无追求、不敢表达的人**。

**极度允许**则走向另一个极端，即不顾及关系阶段和对方感受，放任自己的所有冲动。具体表现如下：

以下是“极度允许”心态的常见表现：
*   不考虑是否已建立吸引，过早使用亲密话术、开车或进行框架评估。
*   当自身需求未被满足时，容易产生攻击性，例如因对方回复不佳而辱骂对方。
*   完全拒绝任何束缚和否定，期待所有行为都被允许。

**极度允许**的根源往往在于自我尚未完全建立，将某个具体行为的否定等同于对自我的全盘否定。

**解决方法**是追求 **“60分满足”** 。在聊天中，不必追求完美的90分对话，因为那不可控且会带来压力。达到60分的舒适状态即可，这允许你保持自然和放松。就像制作视频，初期可能只有五六十分，但通过持续建设，可以逐步向七十分、八十分迈进。关键是在“极度禁止”和“极度允许”之间找到平衡。

## 第七条：时间是祝福还是诅咒？⏳

在探讨了心态平衡后，我们来看看如何看待成长过程中的时间与练习。时间、经验和练习，对你的两性生涯而言，究竟是滋养还是伤害？

这取决于你的视角。这里分享两个学员的故事来说明两种不同的心态。

**第一种是“计较经脉式”成长**。这类学员非常在意即时的投入产出比。例如，第一天搭讪10人被拒9人，成功率10%，他便感到挫败想放弃。第二天搭讪5人成功1人，成功率20%，他又立刻感到自信爆棚。他的状态完全被短期结果所左右。

**第二种是“终身建设者”心态**。另一位学员同样经历高拒绝率，搭讪100人可能被拒99次。但他秉持“永远打不死的小强”信念，相信所有经历都是成长的组成部分。他的目标是成为**参天大树**，而非轻易被风吹倒的树苗。因此，他不惧怕过程的艰难，专注于积累经验。

两者的核心区别在于：前者害怕过程和结果不好；后者不怕，因为他清楚最终方向是成为大树。真正的自信和对自己命运的掌控，源于这种持续建设、不惧挫折的心态。请记住，你的目标是**参天大树**。成长的关键在于过程而非瞬间的结果。过程扎实了，好的结果自然会来。

## 第八条：不要一味追求流畅性⚡

理解了长期建设的心态，我们再来看看聊天中一个具体的误区：对流畅性的过度执着。在聊天中，切忌一味追求流畅性。

因为过度追求流畅，反而会导致流畅性断裂。这就像拉弓射箭，弓弦拉得过紧，箭未射出弓可能先断。在聊天中，如果你认为“有机会就必须聊得火热”、“有好结果对方就一定离不开我”，一旦体验出现断裂（比如对方反应变淡），你可能会因感觉受挫而停止互动，从而使聊天真正断掉。

我们内心常有两个声音：体验告诉你“做不了”，认知告诉你“可以做”。人们天生喜欢流畅感，因为它带来愉悦；而排斥断裂感，因为它带来类似“死亡”的不适。但请记住，**聊天中可以存在断裂感**。这没有关系，及时调整，坚持行动，断裂感就能被修复。有窗口时就畅快聊，感觉断裂了就暂停一下，之后再重新发起互动。如此，你才能在关系中走向强大。

## 第九条：无敌心态训练公式🛡️

最后，我们来学习一个强大的心态公式，帮助你应对聊天中的各种挑战。这就是**无敌心态的训练公式**。

无敌心态由三个核心要素构成：
**无敌心态 = 冷血 + 目标 + 淡定**

*   **冷血**：并非指无情，而是**极度清晰自己想要什么**，并不为此犹豫或拖泥带水。在关系中，这意味着能果断做出符合目标的决定，例如当意识到关系无法继续时，能冷静结束。
*   **目标**：始终清楚自己的方向和目的。
*   **淡定**：无论遇到多大的情绪冲突（如对方不回复、给出负面评价），都能保持冷静，分析问题而非陷入情绪攻击自我或他人。

当三者结合时，你便能在两性关系中进入一种“游戏”的无敌状态：不受复杂情绪牵绊，不被环境左右，坚定地朝着目标前进。

例如，一位目标明确的朋友通过组局认识了一位感兴趣的女生，但短暂交流后发现并非理想型，他便立刻停止投入精力，即使场面略显尴尬也不愿浪费彼此时间。这就是**冷血（清晰不要什么）+ 目标（寻找理想型）+ 淡定（不顾及尴尬场面）** 的综合体现。

## 总结

本节课中我们一起学习了聊天之道的四个核心信条：

2.  将时间、经验和练习视为成长的祝福，以“终身建设者”的心态瞄准成为参天大树的目标。
3.  不必一味追求聊天的流畅性，接纳并修复断裂感是成长的一部分。
4.  通过 **冷血、目标、淡定** 三要素构建无敌心态，从而在沟通中保持清晰与强大。
