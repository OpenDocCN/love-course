# 红丸会员3.0Plus 第22集：红丸人生法则02 - 框架就是一切 🧱

在本节课中，我们将要学习两性互动与关系中的核心概念：**框架**。我们将探讨框架如何决定关系的强弱动态，以及如何建立并维持强势框架，从而在人际交往中占据主导地位。

***

在任何交流过程中，都存在表层信息和里层信息。里层信息中有一个至关重要的结构，你可以将其理解为建造房屋时使用的钢筋。这个“钢筋”就叫做**框架**。

那么，什么是框架呢？框架就是你采用的一种**互动模式**。这种模式将决定他人对待你的态度。框架可分为表层框架和里层框架。

***

为了帮助理解，我们来看一个简单的例子。

假设你今天去商店购物。从你走进商店询问价格的那一刻起，你的底层框架就是**买方框架**，而店员或老板的框架则是**卖方框架**。

在讨价还价的过程中，你可能会对老板说：“就当交个朋友，便宜点卖给我，下次我介绍朋友也来买。”这便是一种**表层的框架**尝试。

如果对方接受了你的这个框架，他就会在你的思维主导下行动。他可能会说：“好吧，就当认识一下，这次就便宜卖给你了，我真不赚钱。下次记得带朋友来。”

你会发现，尽管你是买家，但当你拥有更强势的框架时，你就能主导对方的行为。

***

上一节我们通过购物例子了解了框架的基本概念，本节中我们来看看框架在两性关系中的应用。

因此，阿尔法男性必然是框架高于女性的男性。他在一段关系中处于**领导者**和**主导者**的位置。而贝塔男则在关系中比女性更弱，常常不断为女性妥协。
