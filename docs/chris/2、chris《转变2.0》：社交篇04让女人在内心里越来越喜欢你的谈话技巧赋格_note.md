# 课程2：Chris《转变2.0》社交篇04 - 让女人在内心里越来越喜欢你的谈话技巧：赋格 🗣️💬

在本节课中，我们将要学习一种名为“赋格”的聊天辅助技巧。它的核心作用是在吸引产生后，为女人的情绪提供一个合理的逻辑出口，从而让她在内心层面更加认可和喜欢你。

## 概述

上一节我们介绍了聊天的本质是营造一种感觉，内容本身并不重要。本节中我们来看看聊天中一个具体的技术性辅助点——赋格。需要明确的是，赋格只是一个**辅助技巧**，用于推动聊天进程。没有它，也不影响你与异性交往。

## 什么是赋格？

赋格的核心目的是：当吸引产生时，女人可能因为突如其来的强烈好感而感到不知所措，甚至表现出矜持或拒绝。此时，赋格就是为她这种情绪提供一个**合理的解释**和出口。

从字面意思理解，赋格就是**赋予女人资格**。它告诉女人：“你喜欢我、接受我的理由是什么。”

在第二课关于“情人与供养者”的模型中，我们提到了“体验自我”和“叙事自我”。最初那些引发吸引的行为，刺激的是女人的**体验自我**，让她产生情绪上的好感。而赋格，则是去说服女人的**叙事自我**，让她在逻辑上明白自己为什么喜欢你、你为什么接受她。

**公式：稳固吸引力 = 吸引（刺激体验自我） + 赋格（说服叙事自我）**

当两者同时进行时，你的吸引力会变得非常稳固。

## 如何进行赋格？

赋格的操作非常简单，可以从任何话题切入。

以下是常见的赋格话题方向：
*   **兴趣爱好**
*   **情感经历**
*   **美食**
*   **旅行**
*   **工作**

我个人最喜欢使用**旅行**和**美食**进行赋格。

### 赋格实战示例：美食话题

以下是一个以美食为例的赋格对话流程：

1.  开启话题：询问对方“你喜欢吃什么？比如日本菜、韩国菜还是东南亚菜？”
2.  深入互动：在聊天的过程中，可以问“你这么喜欢吃，那你会不会做这些菜？”
3.  初次赋格：根据她的回答，进一步赋格。例如，如果她说会做，你可以问：“那你有没有给你男朋友做过吃的东西？”
这个过程就是一个简单的赋格。当发现共同点时，例如你们都喜欢吃日本菜，可以借此进行积极的肢体互动，如击掌、碰拳或拥抱，以强化“我们喜欢相同事物”的连接感。这就是**赋予女人资格**的过程。

### 赋格的深入：自我赋格

当女人开始对你有吸引力时，她会渴望通过你的“筛选”。因此，她常常会进行**自我赋格**。

例如，当你问“你有没有给你男朋友做过吃的？”，她可能会回答：“当然！我做的可好吃了，我经常做饼干给他吃，大家都说很好吃。”从她的话中，你能听出她在试图向你证明，她**拥有获得你认可的资格**。这就是女人的自我赋格。

### 赋格实战示例：旅行话题

旅行同样是绝佳的赋格话题。

以下是展开旅行赋格的步骤：
1.  由浅入深：可以先问“你喜欢旅游吗？去过哪些地方？比如云南、丽江、大理？”
2.  引导展望：在聊天中，可以提及“那你有没有和男朋友一起去旅游过？”
3.  描绘愿景：你可以分享自己的愿景：“我的梦想，就是和一个女孩去很多地方，留下特别的回忆。”
此时，观察她的反应。如果她说“我也觉得这样很浪漫”，这很可能就是她在进行**自我赋格**。

## 赋格的奖励机制

在初步赋格阶段，如果对方给出了正面、积极的回应，你可以采用“奖励”机制。

以下是简单的奖励方式：
*   **与她击掌**
*   **与她碰拳**
*   **给她一个拥抱**

这些互动能即时强化积极反馈。
