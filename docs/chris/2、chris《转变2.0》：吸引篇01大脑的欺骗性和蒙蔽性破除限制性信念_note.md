## 什么是限制性信念？

我们人的大脑其实具有欺骗性和蒙蔽性。简单来讲，很多人会自己骗自己。如果你一旦建立起了一个错误的信念，通常我们称这种信念为**限制性信念**。

一旦你有了这样的一个限制性信念以后，不管想法和借口多么荒谬可笑，你都会自己去支持自己大脑里面错误的限制性信念。这俗称自己骗自己。

例如，我们通常会拍摄一些展示社交能力的视频。最常见的，我们会到街上拍摄一些互动视频。看到我们视频的人，第一反应可能是：“你这请的一定是托。”如果我们向他证明了请的不是托，拍摄是真实的，他又会说：“那是因为你比较帅，所以你可以成功。”如果我们又给他看了我们不修边幅、很邋遢时出去搭讪也成功的视频，他还会说：“那是因为你在的城市比较开放，换作在我们乡下肯定不行。”

这就是一个大脑自己欺骗自己的过程。当一个无法解释的事实摆在他面前时，他更愿意相信自己大脑里编辑出来的、毫无逻辑的荒谬借口，而不愿意相信眼前的事实。

## 常见的限制性信念有哪些？

以下是老实人或情感新手最常见的一些天真限制性信念：

1.  **相貌决定论**：因为我的相貌不怎么样，我不是帅哥，所以女人不喜欢我。他认为长得帅才是吸引女人的关键。
2.  **财富决定论**：我不是百万富翁，漂亮女人理所当然应该跟有钱人、明星在一起。
3.  **道德评判**：如果一个女人那么快和一个男人上床，那是因为她比较随便。我要找的肯定不是这样的女人。
4.  **被动等待缘分**：我更愿意相信缘分。我现在为什么是单身？因为缘分还没有来临。很多人把这作为自己不主动出去搭讪、不主动找女人、不主动谈恋爱的借口。他选择被动等待别人来找他，很多人单身一年、两年、三年就是因为这样。
5.  **拒绝自我提升**：我不需要有魅力。作为一个男人，他说我不需要变得有魅力，不需要提升自己，不需要去健身，不需要看时尚杂志打扮自己，不需要培养魅力或幽默感。他的借口是：“我希望以后找一个老老实实结婚的女人。”这是他放弃自我成长的借口。
6.  **先事业后感情**：他觉得年轻的时候应该努力工作、赚钱，不应该把时间放在谈恋爱、找女人身上。因为他觉得事业成功以后，有了钱，自然就会有女人。这也是很多人不主动谈恋爱、不主动出去搭讪的借口。很多男人，比如一些高收入的IT从业者，可能年薪很高，但依然单身。他不停努力工作，觉得有了事业以后自然有女人，但事实上并没有。

此外，还有许许多多各式各样的限制性信念，有一些可能只存在于你自己身上。
