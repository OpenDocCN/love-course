# chris《红丸会员4.0自由蓝图》：1.什麼是紅丸男的精神自由_note

在本节课中，我们将要学习“自由”这一概念背后的真相，剖析现代社会对自由的普遍误解，并揭示红丸男实现真正精神自由所必须理解的底层逻辑——**秩序与权力**。

---

过去，你已经听过太多关于自由的口号，例如情感自由、财富自由、人生自由。亚里士多德也曾说过，人应该是自由的人。那么，究竟何谓自由？

大部分人认为，自由就是“我可以做任何我想做的事情，成为任何我想成为的人”。然而，这并非真正的自由。现代社会的左派普世价值观，通过平等主义的包装，却在大力鼓吹这种无序的自由。

以下是这种思潮的一个典型例子：
*   LGBT群体最爱鼓吹：只要你认为自己是女人，那么即便你的生理性别是男性，你也是女人。因为人生来自由，拥有选择自己性别的权利，所以应该尊重每个人自由的人权。

这种思想看似有道理，实则是一种典型的平等主义思想。而这种“平等”背后，意味着没有秩序。它是具有破坏性与毁灭性的。

因为如果每个人都可以自由地、不受约束地表达自我的欲望，那么社会迟早会变成人间炼狱。例如，“我看你不爽，我就可以一刀捅死你”，最终会演变成一个人吃人的社会。

现在西方“白左”的底层群众，就处于这种淤泥和粪堆的状态里。他们自由地表达自我与欲望，看不惯所有的反对意见，甚至疯狂地诋毁、攻击与自己意见相悖者。这是一种**无序的、走向虚无的自由**。

---

上一节我们剖析了“无序自由”的危害，本节中我们来看看人与社会的本质关系。亚里士多德同时还说过：**人是天生的政治动物**。

也就是说，人从出生的那一刻起，就是社会性动物，无法脱离群体与社会。人与组织的关系，就构成了人与社会的关系。

这就像一滴水：
*   它在陆地上很快就会干枯。
*   但它在大海里却能够生生不息，甚至可以孕育无数生命。

---

讲到这里，我们就要聊到关于“价值”的话题。我一直强调，价值并不来源于你的财富、外在、银行卡存折里的数字、你的车、你的表、你的衣服等等。

但这并不意味着“你认为自己是谁，你就是有价值的”。因为价值还必须取决于周围的环境是否认可你，认可你是一个什么样的人。

所以，**价值必须在集体中才能产生**。而任何一个集体，就会有权力结构的存在。如果脱离集体，所谓的“价值”是毫无意义的。例如，钱如果没有人认可它的货币属性，那么它就是一张废纸。

而权力所带来的，就是一种**秩序**。因此，真正的自由是**秩序下的自由**，而不是无序下的自由。

所以，一名红丸男要彻底觉醒，在了解秩序的同时，也必须培养自己的**权力意识**，才能获得真正的自由。

在接下来的课程内容里，我将会为你讲述，为什么现代社会大部分男人的权力意识被谎言所掩盖的具体原因和真相。

3.  **红丸男的核心**：要实现真正的精神自由，必须理解秩序并培养**权力意识**。
