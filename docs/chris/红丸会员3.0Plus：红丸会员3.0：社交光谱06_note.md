## 案例分析：道歉的力量

以下是两个形成鲜明对比的公关案例，说明了道歉的智慧与不道歉的后果。

**案例一：海底捞的典范公关**

当时海底捞发生了一个食品卫生与安全问题。有人发现海底捞后厨有老鼠，卫生用具存在一些不卫生的操作，例如用火锅漏勺掏下水道等。

这个事情一经爆发，在网上掀起了轩然大波，让海底捞“以客为上帝”的形象瞬间崩塌。

但是，海底捞当时的做法是：他们的公关在第一时间，在舆情尚未完全爆发时，不到三小时内就立刻发表了道歉声明。

人非圣贤，孰能无过。每个人都会犯错，关键在于你是否有认错的态度。海底捞在声明中给出了这样的观点，并保证接下来会在大家的监督下把食品安全问题做得更好。

同时，他们也用这种态度包容了分店的店长和员工，给他们改正的空间，在监督的同时也原谅他们的错误，希望他们之后会改进。

海底捞当时的公关案例是一个非常经典的典范。尤其是当一个公众平台或公众人物做错事时，如果他及时道歉，公众往往就无法再继续严厉地指责他了。

**案例二：吴亦凡事件的反例**

相反的一个例子，就是最近的吴亦凡事件。他做了错事，但是宁死不承认，不为自己在品行方面的恶劣行为道歉。

最后，他被愤怒的舆论直接打击到谷底，最终银铛入狱，全平台账号被封禁。

所以说，道歉是一种Beta智慧。道歉这个行为表面上看似有些“掉价”，但实际上背后蕴含着一种力量。

## 有效道歉的四个关键点

这里有兄弟会问：柯老师，你刚才不是讲吗，道歉有个前提。有些Beta男的道歉不是Beta智慧，就是Beta行为，因为他的出发点是错的。

你的道歉是否有力量，取决于以下四个关键点。

**第一点：道歉是对事，不是对人**

你的道歉行为，并不是对自我的一种否定。你要清楚这一点，因为你针对的是事情。你可能事情确实做错了，但这并不代表你否定你自己这个人。

人非圣贤，孰能无过。你做错事情是没有关系的，只要你之后可以改正，可以让自己变得更好，那你这个出发点就是正确的。它并不是对自我的一种否定。

**第二点：道歉是恢复自尊与释放情绪**

你要了解，道歉其实可以恢复你的自尊和尊严，也是对你过错的一种忏悔和情绪的释放。

一个真正高自尊的人，是不会害怕承认自己错误的。同时，这能恢复他的自尊和尊严，释放负面情绪，而非对自我的一种否定。

这里有一个经典例子。我们之前在外网视频中，拿台湾一位两性教练（被老婆“归零”后）的事情做案例分析。他在我们的案例视频下，承认了自己的错误。

这就是一个非常高自尊的表现。他可以承认自己做的事情是错的。

不像国内某些所谓的两性导师、“大神”，他们明明做了一些不好的事情（例如教学员诈骗，自己去骗人），但不愿意承认。这看起来好像很高傲，但其实内心是低自尊的，他不愿意承认自己以前做错了。

**第三点：道歉是原谅自己，而非乞求他人原谅**

我们刚才讲到了“人非圣贤，孰能无过”。这句话其实也是讲给你自己听的。

想一想，你自己一直以来在成长的路上，做了那么多事情，其实已经很累了。做错了，没关系，你可以自己原谅你自己，让你自己再次变得更好。

这也是一种更大的智慧力量，是让你变得更好的一种力量。所以你要清楚，你是原谅你自己，而不是通过道歉去乞求他人的原谅。这个出发点很重要。

相反，如果你是后一种出发点（乞求原谅），那么你其实是活在别人的评价里的。

**第四点：道歉能建立人际关系中的安全感**

你道歉其实可以在人际关系中获得一种安全感。你可以让对话进入一种更坦诚的局面。

因为如果你不道歉，如果你的某些行为是错的却不道歉，那么人际关系中的对话就会进入一种危险状态——双方会进入一种设防、开启“防护罩”的模式，僵持不下。

但是，当有一方道歉的时候，你们就会进入一种安全对话里面，彼此更容易坦诚相待。

---

这就是道歉背后的底层逻辑，以及它可以给你带来的一些好处。我们要了解到，在适当的时候，为你自己的一些行为去道歉，并不代表这是Beta行为，而是一种Beta智慧。

当然，你必须区分你的前提是什么。
