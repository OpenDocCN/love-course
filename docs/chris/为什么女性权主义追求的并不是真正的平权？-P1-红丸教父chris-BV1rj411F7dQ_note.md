# 女权主义与平权辨析：课程 001 🧠

在本节课中，我们将探讨一个关于女权主义与平权概念的常见观点，并分析其内在逻辑。我们将解析“女权主义追求平权”这一表述的具体内涵，并探讨其在实际社会语境中可能指向的对象。

---

上一节我们引出了关于女权主义追求平权的讨论，本节中我们来看看其中存在的一个核心逻辑争议。

原观点认为：女权主义追求与男性平权，因此女性不会在经济上依赖男性，会保持独立。

**真相**在于，女权主义所追求的“平等”，其参照对象并非全体男性，而是一个特定的群体。

以下是关键辨析：

女权主义追求的并非与**底层男性**平等，而是与**上层男性、顶层男性**追求平等。

这个观点没有阐明“平等”中最关键的主体和课题角色。它只描述了一个宽泛的平等概念，但事实并非如此。

我们可以通过女权主义的职业愿景来理解这一点：

*   女权主义希望成为**女老板、女企业家、女科学家**。
*   女权主义**不想成为消防员、警察，或是在工地上搬砖的基层劳动者**。

因此，女权主义所说的平等，是想与社会精英、顶层人士平等，而非与底层劳动者平等。

---

上一节我们明确了女权主义“平等”诉求的指向性，本节中我们来探讨这种诉求带来的社会与婚恋影响。

我们的社会需要大量男性从事基层工作，以维持社会的基本运转与稳定。这些基层工作者是维护社会稳定的重要力量。

那么，一个追求与精英平等、不愿从事基层工作的女权主义者，会如何选择她的生活伴侣呢？

以下是基于此逻辑的推论：

她不太可能与一个普通的底层男性结婚并共同分担经济压力。原因主要有两点：

**第一，女性的择偶天性：慕强择偶。**
即使女性通过发展事业变得经济独立，这种行为本身也可被视为一种“慕强”的体现——追求成为强者。当女性不断向上发展，追求与精英平等时，她自然难以看得起不如她的底层男性。

**第二，男女择偶逻辑的差异。**
这是一个关键区别，可以用一个简单的对比来描述：

*   一个男性的事业上升，他的择偶选择权通常会**扩大**，因为他可以“向下兼容”。
*   一个女性的事业上升，她的择偶选择权反而可能**缩小**，因为她通常拒绝“向下兼容”。

因此，当现代女性普遍走上追求事业精英化的道路时，她们对伴侣的要求会水涨船高，其实际可选择的婚恋对象范围可能会变窄，这与男性的情况相反。

---

本节课中我们一起学习了如何辨析“女权主义追求平权”这一概念。我们认识到，其所指的“平等”具有明确的向上指向性，即与上层男性而非全体男性平等。这种诉求根植于慕强择偶的天性，并导致了男女在事业成功后面临截然不同的婚恋选择逻辑。理解这一点，有助于我们更深入地讨论性别平等议题所涉及的多重维度。
