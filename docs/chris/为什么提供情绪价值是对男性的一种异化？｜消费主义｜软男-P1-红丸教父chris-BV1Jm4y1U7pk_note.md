# 课程 P1：为什么提供情绪价值是对男性的一种异化？ 🧠💔

在本节课中，我们将探讨“情绪价值”这一流行概念，分析它如何被消费主义和特定性别观念所塑造，并解释为何在某些观点看来，要求男性提供情绪价值是一种“异化”。我们将从生物学差异、社会角色演变以及权力博弈等角度进行解析。

这通常被定义为给对方带来积极的情感体验，例如情感支持和关注。然而，由于情绪价值是一个抽象的概念，任何不同程度的情感都可以被包括在其中。

上一节我们介绍了情绪价值的普遍定义，本节中我们来看看其背后的生物学基础。在传统的性别认同中，男性应该是强壮、无所畏惧和不动声色的代表。相较于男性，女性的情感表达更为丰富。

这是因为女性体内分泌大量的催产素，而男性则不具备这种生理机制。而这种对于情绪价值的强调，会使一个男性在表达情感方面感到生理上的不适。本质原因是特定观念对传统阳刚男性的一种异化。

以下是这种义务转嫁的一些例子：

*   照顾伴侣
*   投入家庭

这种异化使得男性变得越来越女性化。因此，某些理论绝不提倡男性给女性提供所谓的情绪价值。

最关键的是为女性提供情绪价值，这种思想在某些观点看来是极具危险性的。因为情绪实际上是女性擅长的战场。

这就意味着让男人去进入女人制定的游戏规则里。许多男性往往会把情感视作为是一种纯粹的情绪，然而在女人的眼中并非如此。

她们会想要借此翻身做主，在关系当中去剥削剥夺男性的权利。所以你经常会看到女人借着情感的名义，在关系中对男性进行情绪勒索。

以下是几种常见的负面情绪表现：

*   失望
*   伤心

一个男人应该保持理智。在一个理想男性的表达里面不应该有过度的情感化，以及女性化的行为特征。这会让一个男人变得软弱甚至依赖他人，进而在这场情绪游戏里往往会陷入被剥削、被虐待的处境。

理智是一个男人最坚固的铠甲。男人应该通过理智来抵御情感勒索。就像是一个八岁的小女孩，她又哭又闹吵着跟她爸爸索要她不应该得到的玩具，而一个理智的父亲会出于公正明确地拒绝她的要求。

因为被异化的男性，他会变得多疑、敏感、情绪用事。这个会让一个男人失去理智的判断力和决策能力。

过去的男人往往是更坚韧、立志和独立的。他们通常会被视为是强壮、勇敢、果断和有责任感的代表，同时也被期待承担家庭和社会责任。
