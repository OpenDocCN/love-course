# 柯李思魅力原理：2020魅力原理『已更新完』：魅力原理第四集_note

在本节课中，我们将要学习能量由什么决定、潜意识与情绪感觉的关系、后天培养的重要性以及教育与学习环境的影响。我们将逐一解析这些核心概念，帮助你理解并提升个人魅力。

## 第一部分：能量由什么决定 ⚡

上一节我们介绍了魅力的基础概念，本节中我们来看看能量由什么决定。你的能量主要由以下三样东西决定：生命能量、身体状态和情绪。前两者是我们不可控的初始条件。

以下是决定能量的三个要素：

1.  **生命能量**：这是与生俱来的能量，由生辰八字决定。生辰八字代表出生时的年月日时，共八个字。它反映了母亲怀孕时的营养、心情、父母关系及居住环境，这些因素会潜移默化地影响胎儿的原始能量。
2.  **身体状态**：指出生时的健康状况。每个婴儿出生时存在差异，例如早产、营养不良或非常健康。这些细微差别会随着成长产生蝴蝶效应，影响长远发展。
3.  **情绪**：这是我们可以部分调控的因素。情绪状态直接影响能量水平，进而影响个人魅力。

## 第二部分：潜意识与情绪感觉的关系 🧠

理解了能量的决定因素后，我们来看看潜意识与情绪感觉的关系。潜意识形成于三岁以前，受早期家庭环境影响巨大。它如同冰山，海平面以上可见部分仅占10%，而海平面以下不可见的90%才是决定我们行为与情绪的关键。

以下是潜意识的核心特点：

1.  **潜意识是欲望系统**：可以理解为荷尔蒙系统或欲望系统，对人的影响持续一生。
2.  **意识难以抗衡潜意识**：意识是六岁后习得的知识，而潜意识是三岁前形成的本能。例如，意识层面想戒烟，但潜意识层面可能仍渴望吸烟。
3.  **潜意识决定情绪**：情绪不受理智控制，而是源于潜意识。例如，莫名喜欢或讨厌某人，往往源于潜意识的判断。

## 第三部分：后天培养 🌱

前面我们探讨了先天因素，本节中我们来看看后天培养。这是我们可以着力改变的部分。通过持续学习和实践，我们能够逐步提升能量与魅力。

后天培养的关键在于“学”与“习”的结合，遵循“三分学，七分习”的原则。改变是日积月累的过程，如同下棋，每次只能移动一步，但坚持每日进步，身边的人终将察觉你的变化。

**公式表示**：`改变 = 学习 × 实践 × 时间`

## 第四部分：教育与学习环境 🏫

最后，我们来探讨教育与学习环境。了解上述原理后，当你成为父母时，就会意识到胎教和六岁前的教育至关重要。影响他人的关键在于做好自己。

以下是教育环境的两个要点：

1.  **父母是孩子的榜样**：孩子是父母的复刻。家庭关系是否和睦、父母行为是否端正，比老师的教育更重要。例如，父母在家抽烟，孩子很可能模仿。
2.  **截停负能量**：如果成长过程中有负能量影响，如家庭不睦导致性格内向或社交焦虑，应努力让这股负能量在自己这一代截停，避免传递给下一代。

此外，后天学习环境也很重要。无论是与朋友交流、在线学习、线下课程、健身还是实践约会，都属于学习和练习的范畴。

本节课中我们一起学习了能量的决定因素、潜意识与情绪的关系、后天培养的方法以及教育与学习环境的重要性。记住，先天条件无法改变，但后天努力是提升魅力的关键。坚持学习与实践，你一定能逐步蜕变。

**总结公式**：`个人魅力 = 先天能量 + 潜意识管理 + 后天培养 × 学习环境`
