# 柯李思团队全套课程 第09课：用软硬框架“推拉”制造情绪波动 🎭

在本节课中，我们将学习一个核心互动技巧——“推拉”。我们将探讨如何通过语言和行为的“推”与“拉”来调整互动框架的软硬，从而有效制造浪漫感与情绪波动，并推进关系。

---

上一节我们介绍了“推拉”的核心价值，本节中我们来看看“推拉”的具体定义与情感效果。

“推拉”是通过交替传递拒绝（推）与接纳（拉）的信号，来营造情绪张力和浪漫感的互动技术。其本质是**在保持自我框架的前提下，满足对方需求**。

一个典型过程是：先多次拒绝（推），最后再同意（拉）。这能让对方在短暂失落（推）后，因最终满足（拉）而产生更强的愉悦感。女性常将这种体验描述为“浪漫”。

---

理解了基本概念后，我们进入实践环节。以下是“推拉”的四种具体形式：

1.  **语言上的推，语言上的拉**：完全通过对话内容传递拒绝与接纳。
2.  **行为上的推，行为上的拉**：完全通过行动传递拒绝与接纳。
3.  **语言上的推，行为上的拉**：用语言拒绝，但用行动接纳。
4.  **语言上的拉，行为上的推**：用语言接纳，但用行动拒绝。

你可能会觉得有些抽象，让我们结合实例来分析。

---

回顾课程开头的例子：女生想养狗，男生三次说“喜欢猫”或“养猫吧”（语言上的推），最后同意“那还是养狗吧”（语言上的拉）。

在这个互动中：
*   **整体结构**：一个大的“拉”（最终同意养狗）中，掺杂了三个小的“推”。
*   **效果**：自我框架（喜欢猫的立场）未被破坏，同时满足了对方需求。情绪在一推一拉中产生波动，浪漫感由此营造。

---

为了加深理解，我们再分析一个更复杂的例子。上一节我们看了单一场景，本节中我们来看看混合框架下的“推拉”应用。

视频对话摘要：
*   男：“明天我想见你。”（**语言上的拉**，切入男女关系框架）
*   女：“你都这样单刀直入泡妞吗？”（女性测试窗口出现）
*   男：“我是说，我明天约了你的治疗看病。”（**语言上的推**，切换回医患关系框架）
*   *（女生情绪明显由高转低）*
*   男：“现在去看吧。”（**行为上的拉**，提出即时见面，强势切换回男女关系框架）

**分析**：
1.  互动中存在两种身份框架：**医患关系**（较远）与**男女关系**（较近）。
2.  男生先用语言拉近关系（拉），观察到女生态度后，又用语言推远（推）。
3.  女生的情绪低落反应，揭示了其内心更倾向于男女关系框架，这是一个强烈的兴趣窗口。
4.  男生随即用行动（提出立刻见面）进行更直接的拉近，完成关系推进。

**核心秘密**：你可以利用“推拉”来**测试和切换互动框架**。通过观察对方在你“推”（切回普通关系）和“拉”（切入亲密关系）时的情绪反应，可以精准判断对方的兴趣度与接受度。
