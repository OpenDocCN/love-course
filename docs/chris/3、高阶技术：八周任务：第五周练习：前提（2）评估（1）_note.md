## 概述：本周练习的核心目标

## 第一部分：如何定义并列出特质 ✍️

首先，我们需要明确“特质”与“行为”的区别。特质是指一个人内在的品质或性格特点，而行为是这些特质的外在表现。

以下是关于特质的具体说明：

*   **特质是内在品质**：例如“会照顾人”、“成熟”、“积极上进”、“随性爱冒险”、“追求新事物”或“乖巧”。这些描述的是人的性格内核。
*   **行为不是特质**：例如“喜欢做饭”、“喜欢运动”。这些是具体的、可观察的活动，是特质可能带来的行为表现，但其本身并非特质。

**为什么必须区分并专注于特质？**
因为如果你只关注行为，你与对方的互动模式将是有限且容易耗尽的。基于特质的评估则不同，它是一个可以持续使用的框架。

## 第二部分：理解并运用“评估” 🔄

评估是基于特质进行互动的核心方法。它不是一个一次性的话术惯例，而是一个贯穿始终的互动框架。

评估的关键在于其**可持续性**。它可以从相识初期一直应用到长期关系中。通过持续评估对方的特质，你可以不断深化互动，创造丰富的交流话题。

**任务二：构思五项“否定前提”的表达**
在继续练习“前提”技巧的基础上，本周引入“否定前提”。否定前提是一种以看似负面或调侃的方式表达兴趣或建立联系的方法。

以下是关于否定前提的说明和示例：

*   **概念**：否定前提不是真正的否定，而是一种夸张、玩笑式的表达，用于创造张力或幽默感。
*   **与夸大前提的关系**：它和我们上周练习的“夸大前提”（如“我爱你”）是同一光谱的两端，都是强烈的情绪投射。
*   **使用要点**：使用时需要夸张的语气和恰当的语境，避免让对方感到真正的冒犯。
*   **示例**：
    1.  “我恨你。”（在轻松语境下，回应对方的某个可爱举动）
    2.  “你这样让我头疼。”（调侃对方的有趣或调皮行为）
    3.  “我预感我们以后肯定会经常吵架。”（暗示彼此有个性，互动会很有趣）
    4.  “你再说下去，我可能要跟你打架了。”（玩笑式地表达对争论的兴趣）
    5.  “你太麻烦了，但我好像不讨厌。”（委婉表达好感）
