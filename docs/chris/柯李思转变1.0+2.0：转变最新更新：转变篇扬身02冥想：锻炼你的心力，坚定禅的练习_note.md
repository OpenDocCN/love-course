## 转变篇：扬身02冥想——锻炼心力，坚定禅的练习

在本节课中，我们将学习转变课程中关于冥想的第二部分，重点介绍“被动转化”的概念及其核心练习方法。

上一节我们介绍了主动转化，即通过与人事物互动来消除偏见。本节中我们来看看另一种清理潜意识创伤的有效手段——被动转化。

### 什么是被动转化？

被动转化是清理潜意识创伤、特别是童年创伤与刻印的唯一有效手段。在讲解具体方法前，我们需要先理解记忆的运作方式。

### 记忆的两种形式

我们形成刻印，通常源于童年或原生家庭的经历。这些不幸事件被记录下来，导致我们以偏差的认知看待世界，并对特定人事物产生负面情绪与偏见。记忆主要分为两种：

**1. 图像记忆**
图像记忆指对事件场景画面的记忆。例如，童年溺水时同伴视而不见的画面，被潜意识记录下来。

**2. 情绪记忆**
情绪记忆指事件所引发的情绪感受被身体记录。潜意识具有保护功能，可能会遗忘痛苦的图像记忆，但相关的情绪却被身体细胞和肌肉记忆下来。当遇到相似情景时，便会触发负面情绪。

以下是情绪记忆的一个例子：
> 一位学员因小学表白被嘲笑，感到羞耻。这种情绪被身体记录，导致他直至大学都无法与女性正常交流，因为交谈会触发痛苦的负面情绪。

因此，潜意识清理的核心对象是这些**情绪记忆**。

### 情绪记忆存在于身体

情绪并非抽象概念，它具体存在于我们的身体之中。当情绪产生时，身体会出现相应的生理反应，例如：
*   生气时呼吸急促、肾上腺素升高。
*   焦虑时心慌、胸闷。
*   面对女性时感到害怕，也是通过身体的不适反应表现出来。

所以，清除潜意识情绪记忆的关键在于**从身体入手**。

### 核心练习：坚定的禅

以下是进行被动转化前必须掌握的基础练习。建议在完成第一课并持续练习一个月后，再进行本练习。

本练习要求你以盘腿姿势（单盘或双盘）坐直，保持头颈正直，然后开始观察呼吸。在此过程中，身体会产生各种不适感，如瘙痒、流汗、腿麻酸痛等。人的本能是去缓解或逃离这些感觉。

但本练习的核心在于“坚定”：在至少一小时的冥想过程中，**保持身体完全不动**。

#### 如何应对不适感？

练习的重点是学习如实观察身体的感觉，步骤如下：
1.  **观察而不批判**：当瘙痒感出现时，仅仅去观察“痒”这个感觉本身，而不是产生“好痒，真讨厌，好想挠”的批判性念头。批判会强化不适感。
2.  **保持客观觉知**：像科学家用显微镜观察微生物一样，客观地觉知身体的感受。**公式表示为：`观察 = 客观觉知(身体感受)`**
3.  **体会生灭现象**：当你能够如实观察时，会发现感受会出现“生起”和“灭去”的现象。例如，痒感会在被持续观察后逐渐减弱直至消失。其他如痛、酸等感觉亦然。

这个过程被称为“身体生灭”，是内在清理的基础。如果你始终在抵抗或厌恶感受，则尚未掌握这个关键工具。

### 课程总结

本节课我们一起学习了被动转化的原理与核心预备练习。我们了解到：
1.  潜意识创伤主要以情绪记忆的形式储存在身体中。
2.  清理这些记忆需从身体感受入手。
3.  “坚定的禅”练习通过培养对身体感受的客观观察力，让我们体会到感受的生灭过程，为后续直接清理潜意识刻印打下基础。

当你能够每天坚持一小时“坚定的禅”练习并保持不动时，即可进入下一阶段的练习，直接清理内在的潜意识刻印与童年创伤。
