## 理性分析：搭讪的风险与收益

上一节我们提到了心态的重要性，本节中我们来看看如何理性看待搭讪。拒绝并不可怕。在我们的线下教学中，第一天常会安排一个叫“脱敏”的环节。

“脱敏”的意思是让大脑脱离社交敏感状态。很多人对拒绝反应过度敏感，会过度抗拒。甚至因为害怕，在接近目标前会非常焦虑。脱敏就是让你的头脑去适应搭讪被拒绝的状况，脱离搭讪时出现的敏感状态。

那么，脱敏具体要怎么做呢？首先，我们通过观看几段搭讪被拒绝的影片来建立认知。

以下是几段搭讪被拒绝的影片对话示例：

*   “衣服上是什么菜？辣椒炒肉吧，看着挺好吃的，你会不会做？” —— “不会不会，不是厨师。”
*   “帅哥你没事吗？我跟我朋友要去吃东西。” —— “我们也有事。”
*   “哈喽，等一下。” —— “咋子？” —— “你们两个是不是卖眼镜的？” —— “我抓这个，在找路啊。”
*   “你这个眉毛是柳叶眉吗？” —— （对方无表情离开）
*   “你结婚了吗？” —— “结婚了。”
*   “你知道我是要干嘛的吗？” —— “不用谢谢。”

我们可以看到，在无数搭讪被拒绝的反应中，最坏的情况也只是女生喝斥一下，例如“你要干嘛？”或“你有事吗？”。并没有出现大家想象中非常恶劣的情况，例如搭讪被打，这基本不可能发生。

但是，仍然会有兄弟说，即便这样也很害怕。在大街上搭讪被喝斥，会觉得自信心受打击，觉得自己在做不好的事情。

### 换位思考：比较不同场景的风险

现在我们换位思考一下。假如你今天不是在大街上搭讪，而是在你认为更安全的社交圈里，比如学校、办公室职场或朋友圈里，去接触一个女生。那么，相比在大街上，哪个风险更高呢？

事实上，在已知社交圈里的风险更高。原因如下：

*   **公式：社交圈风险 = 后续尴尬遭遇概率 × 社交评价受损程度**
*   在职场或学校搞砸了一个社交圈的女生，你会抬头不见低头见。第二天、第三天，每天都会遇到她，场面非常尴尬。
*   接触失败后，她可能会在圈子里说你一些不好的东西，这会降低你的社交评价，让你的价值在社交圈里变得更低，影响你的名声。

因此，在已知社交圈里接触女生，风险更高。但是，在大街上搭讪一个女生，即便搞砸了，她讨厌你，但你有可能再也见不到这个人了。

我们先不说最好的情况，仅从最坏情况分析：

*   **代码逻辑（伪代码）：**
    ```python
    if 场景 == “大街搭讪”:
        if 搞砸了:
            后续影响 = “大概率永不相见，对原有生活无实质影响”
    elif 场景 == “已知社交圈”:
        if 搞砸了:
            后续影响 = “每日相见，尴尬；社交评价可能受损”
    ```

就这一点来讲，搭讪其实是一个**低风险、高收益**的事情。因为即便搞砸了，最坏的情况发生，她对你的原有生活并不会造成实质性影响。

而如果你成功了：

1.  小成功：与女生有良好交流。这可以提高你的社交能力和自信心，让下一次搭讪更好。
2.  大成功：有很好的收尾，成功邀约。这就多了一个与女生发展关系的机会。

而且，大街上的美女是取之不尽的。搭讪相比社交软件也有许多优势。所以说，搭讪是一个低风险高收益的事情。

很多人害怕抗拒，其实都是被自己非理性的恐惧打败了，并没有理性分析。它其实比在已知社交圈里认识女生、或通过朋友介绍要靠谱得多。前提是，你要掌握搭讪，让它变成可以给你带来实质性结果的事情。

## 三大核心心态原则 🧠

在让搭讪变成有收益的事情之前，我们必须先不害怕、不抗拒搭讪。理性分析之后，如果还是不敢，怎么办呢？这里给大家讲几个关于搭讪时心态的要点。

以下是三个核心的心态原则：

### 原则一：减少思维决策的内耗

不要把能量消耗在思维的决策上。今天你看到一个女生，想搭讪但又害怕，处在这种纠结状态里，一直犹豫，这会非常消耗你的思维能量，让你的状态越来越差。

我的建议是：当你看到一个正妹，要把思维决策的时间减少。也就是说，你看到一个正妹出现后，要不就上去搭讪，如果决定不搭，就不要再去想他了。不要后悔“我刚才没搭”，一直想一直想，这非常不好。

大部分新手通常卡在这一点：今天出来要搭讪，结果犹豫犹豫一直没出手，在大街上逛了两三个小时都没行动。原因就是把能量消耗在了思维决策上。要知道，人的思维是相当消耗能量的。我们不要把过多能量浪费在思维决策上。

**行动指南：** 当你看到一个女生，决策反应一定要快。做“是”或“否”的决定，不要停留在中间态无限消耗能量。

### 原则二：每一次都是全新的开始

每一次搭讪，其实都是一个新的开始。不要把它跟之前的搭讪联系在一起。很多人会把每一次搭讪跟之前的状况联系，比如今天出门搭了十个，前面九个都失败，就会认为第十个也会失败，形成惯性思维。

但你要知道，每一搭都是新的开始。从女生的角度来讲，你是她今天遇到的第一个陌生人，对她来说，不存在前面的“九搭”。很多男生会把前面的失败，带入到新的搭讪情景中。

这跟一些经济学游戏类似。很多人赌博，会凭以前的运气判断后面的概率。但事实上，就像抛硬币游戏，每次概率都是50%，是不确定的，与过往概率没有联系。

**公式：单次搭讪成功率 ≠ f(历史失败次数)**

搭讪也是一样。虽然随着技巧熟练、反应能力变强、越来越自然平和，你的成功率会逐渐提高。但在心态上，一定要避免把之前的失败带入到新的搭讪中。

### 原则三：专注当下，而非结果

搭讪不要把目标看得太大，先从小目标开始。很多人搭讪压力大，是因为一开始就把既定目标设置得太高，认为上去搭讪就一定要成功，让她成为女朋友或有好发展，才算成功。

但事实并非如此。虽然你需要后面的结果，但在迈出第一步时，应该把当下的那一步做好，而不是思考后面的结果。

一个禅师的故事令我印象深刻，可以引荐给大家：

> 有一个人每天上山砍柴，背下山烧火。他觉得这个过程很累，每天出门上山都肩负重担。
> 另一个人心态完全不同。他觉得上山就是上山，把注意力放在上山路途的风景上，一路美景尽收眼底，过程轻松欢快。到了山顶砍柴，也觉得轻松快乐。背柴下山时，看着下山风景，也感觉不到重担。他上山砍柴和下山烧柴的过程非常轻松快乐。

同样的行为，截然不同的心态，带来的过程情绪体验也截然不同。如果你搭讪时也想成是以结果为导向的事情，压力会非常大。我们不是不需要结果，而是说在做每一步时，要把当下所做的事情做好。

**行动指南：** 就拿搭讪举例，当你看到女生的第一刻，想把注意力放在当下：这个女生为什么出现在这里？她为什么穿这个衣服？她是在等人吗？她是在等待我的出现吗？当你把注意力放在更当下的时候，你的交流会更自然，更注重过程，也会拥有我们提到的“流动感”。
