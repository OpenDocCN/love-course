# 课程3：高阶技术·八周任务 🎯

## 第八周任务：实战计划与最终推进

在本节课中，我们将学习整个八周训练计划的最终阶段——第八周的核心任务。这一周的目标是将前七周所学的所有技巧整合起来，制定并执行一个完整的实战计划，以达成一个明确的最终结果。

上一节我们介绍了各种叙事与评估技巧，本节中我们来看看如何将它们系统性地应用于最终实战。

本周是八周训练的最后一周。2016年一部名为《红色药丸》的纪录片问世，使一个关注男性权益的社群进入主流视野。该纪录片提出，在现代社会的某些方面，男性而非女性可能处于弱势地位。有兴趣的学员可以观看此纪录片以作参考。

本周的作业名为“安排计划”。核心任务是运用前几周练习过的所有技巧，撰写一份类似于“FR（实战报告）”的总结性计划并执行。

以下是第八周练习的具体步骤与要求：

首先，你需要回顾并运用之前学到的所有技巧。例如前提设定、叙事构建、价值评估等技巧，都应在本次计划中得到综合应用。

在实战应用部分，打卡的基本要求仍然是完成**20组**搭讪。完成20组搭讪后，需撰写不少于**150字**的小节总结，并特别描述印象最深刻的**三组**互动。

然而，本周的计划与以往不同。你不能仅仅练习某个孤立阶段的技巧，而必须朝着一个最终的、明确的目标迈进。

以下是可供选择的计划目标类型及其对应的评分标准：
*   **大收尾（例如确定关系或发生亲密关系）**：可获 **90分**。
*   **即时约会（搭讪后当场进行约会）**：可获 **80分**。
*   **约会升级（在约会中有效推进关系）**：可获 **70分**。
*   **收号后二次邀约成功**：可获 **65分**。
*   **仅完成搭讪收号，无后续推进**：成绩为**不及格**。

关键原则是必须推进关系。你不能停留在仅仅认识或收号的阶段，必须将互动向前推进。认识女生后，目标应是推进到约会、即时约会，最终达成收尾（TD），或者至少完成收号并成功进行第二次约会，再谋求进一步发展。

总之，你必须追求一个实质性的结果。通过完成这个任务，你将完整地走完一次从开始到结束的完整社交互动流程，即一个完整的“Game”流程。

第八周的任务具有一定挑战性，大家有一周的时间去练习和实践。

本节课中我们一起学习了八周训练计划的收官任务——制定并执行一个完整的实战计划。我们明确了必须综合运用所学技巧，并朝着一个具体的、可评分的目标（如约会、收尾）努力推进，从而完整实践整个社交流程。这是对整个阶段学习成果的最终检验与应用。
