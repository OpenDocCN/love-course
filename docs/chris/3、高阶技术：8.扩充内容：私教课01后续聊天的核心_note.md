# 3、高阶技术：8.扩充内容：私教课01后续聊天的核心_note

在本节课中，我们将学习如何确保所有学员都能顺利访问课程PPT，并了解后续课程的核心内容。课程的重点是解决不同设备访问课件的问题，并引导学员进入正式学习。

## 课程概述

本节课旨在解决课程开始前的技术准备问题，确保每位学员都能通过电脑或手机访问到本次“高阶技术私教课”的PPT课件。这是顺利进行后续核心内容学习的基础。

## 确认课件访问情况

首先，我们需要确认所有学员能否看到共享的PPT课件。

电脑端的学员应该可以直接在屏幕上看到PPT。如果无法看到，可能是连接问题。

手机端的学员可能无法直接在屏幕上看到PPT。针对这种情况，我们提供了解决方案。

## 手机端访问PPT教程

对于使用手机参加课程的学员，请按照以下步骤操作，以便下载并打开课程PPT。

以下是具体的操作步骤：

1.  获取分享的PPT链接。
2.  将PPT链接复制到手机浏览器的地址栏中。
3.  在浏览器中打开该链接。
4.  您可以选择在线查看，或下载PPT文件到手机。
5.  使用手机上的办公应用程序（如WPS Office、Microsoft PowerPoint）打开下载的PPT文件。

通过以上步骤，手机端的学员就可以查看到本次课程的全部大纲和内容。

上一节我们介绍了手机端访问课件的具体方法。确保课件访问是学习的第一步，当所有学员都准备好后，我们就可以顺利进入后续课程的核心聊天技巧部分。

本节课中我们一起学习了如何为“高阶技术私教课”做好课前准备，重点是确保不同设备（电脑和手机）的学员都能成功访问课程PPT。这是保障所有学员同步进入学习状态的关键步骤。在接下来的课程中，我们将深入探讨私教课后续聊天的核心技巧。
