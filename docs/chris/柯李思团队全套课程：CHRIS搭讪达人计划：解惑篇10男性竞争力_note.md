# 柯李思团队全套课程：CHRIS搭讪达人计划：解惑篇10 🥊 男性竞争力

在本节课中，我们将学习一个特殊的搭讪案例，探讨如何在已有其他男性搭讪过目标女性的情况下，通过策略和沟通技巧展现更强的男性竞争力，从而成功建立联系。

从新手到达人，欢迎来到搭讪达人。今天我们分享的案例有些特殊。这个案例发生在我们拍摄《搭讪达人》课程时，出现了一个非常有意思的状况：我们看到有其他人也在搭讪同一个女性。

后来我们做了一件事。这件事是：当他搭讪完以后，我上去再次搭讪。很多人会担心这样效果不好，其实并非如此。在30天计划里也有类似的现场视频。

那个现场视频是用来破除你限制性信念的。在《搭讪达人》课程中，我们可以更多地讲解策略。在这种情况下，你可以通过特定做法，体现出你比那个男性更有竞争力、更具吸引力。

你可以用一种能将他比下去的方式。我们先来看一下这个视频。这个视频没有录到声音，因为当时麦克风不在我身上，是我们的学员拿着麦克风在拍摄他们的画面。

因为那个人搭讪完后，那位女性可能要走，所以我立刻再次搭讪，没来得及拿麦克风。但我们可以一起看画面，我来讲解当时发生了什么。

画面中就是这位女生，我走了上去。当时我采用的“打开方式”可能让人意想不到。这个打开方式非常简单，是一种自我调侃的方式。

在我以前的免费视频里讲过，你可以用一种“自我矮化”的方式上去搭讪。我说：“刚才那个男的你喜欢吗？” 那位女性说：“喜欢，怎么了？” 因为她刚才被另一个男性搭讪了。

我问：“你为什么不会喜欢他？我看那个男的还挺帅的，不像我，对吧？” 我接着说：“你看，我没有他帅。” 我用了这样的方式。然后我问她在这边做什么。

她告诉我她是来买东西。她们这几个人应该是一起的。其中一位我之前觉得很好看。她当时用手机给我看照片，问我这种烟多少钱。我说我也记不清了，大概七八十一包。

她说她买错了。她说她是来买一种烟，那个东西是送人的。刚才那个小女孩想加她们两个的微信。她买的是熊猫香烟。我跟她聊了一些基本信息。

她看起来经济条件不错，在报警那边都是高专职……（此处可能指职业或背景）。然后这里我就准备收尾了。这里聊了什么记不太清了，大概是交换基本信息，然后做一些简单的评估。

这里有一个细节想提一下。我之前在免费视频也讲到过。正常的搭讪，肯定一看就能看出来。当时我做了一个小细节：关于她问我的烟的话题。

在结尾的时候我说……而且她提到了一个事情，我前面忘记讲了：她说她有男朋友。我就说：“没关系，就聊一聊。” 我说她有男朋友也没关系。

你要知道，大部分女生的男朋友都可能不是最终伴侣。她可能确实有男朋友。但搭讪时已经进入了对话状态。

所以后面有这个细节：我在走的时候，把烟作为话题。我说：“你这个烟应该不是送领导的吧？” 她说：“应该是送给你爸爸的？” 我说也不是。因为熊猫烟其实还挺贵的。

她说也不是。我说：“那肯定不是送给你男朋友的。” 然后她笑了，说：“对。” 这个细节我在之前的视频里解释过。它的第一层含义是：你可以读懂事物背后更深层的意义。

这其实是一种社交直觉的表现，可以体现你的社交经验很丰富，你是个**阿尔法（Alpha）男性**。还有另一层意思：你可以解读出，她男朋友可能是个**贝塔（Beta）男**。

所以后面这个女生在后续跟我聊天时，反馈相当好。我也做了“播种”（即铺垫后续联系）。但我一直没时间去约她。

所以这个案例想告诉大家：不是别人搭讪过的人，你就不可以搭讪。你还是得去搭讪。因为女生喜不喜欢你，关键不在于她被多少人搭讪过，而在于搭讪她的人是谁。

她有男朋友其实也是一样的道理。你要去体现出来，你可以通过你的一些行为、表达和一些潜在沟通的东西，展现出你的**阿尔法特质**。

把你的竞争力去碾压掉其他男性。这完全可以通过技术层面的东西做到。所以不要给自己找借口，比如“她有男朋友就怎么样”。这是一个最典型的状况：这个女性既有男朋友，又被人搭讪过，但她还是接受了我的搭讪，后期还跟我聊了很多。
