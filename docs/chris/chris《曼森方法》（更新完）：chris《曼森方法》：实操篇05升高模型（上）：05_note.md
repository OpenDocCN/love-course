## 升高模型的三个组成部分

升高模型分为三个部分：肢体、情绪和话题。

1.  **肢体**：指身体接触和亲近行为。
2.  **情绪**：指在互动中营造和激发的情感氛围。
3.  **话题**：指交谈的内容和方向。

现在，请按照你心目中的优先级，对以上三个部分进行排序。
