# 7、chris『红丸会员』：转变01与内在的高我合一_note

在本节课中，我们将学习如何运用“转变力”的方法来清理潜意识。首先，我们需要理解意识究竟是什么。

## 意识的比喻：马车、马夫与乘客

为了形象地理解意识，可以将其比喻为一架马车。这架马车的马，代表你先天出生时的自我，称为**本我**。而驾驶马车的马夫，则代表你后天形成的自我，称为**小我**。

马夫认为他可以决定马车的去向，马也认为是它在拉着马车奔跑。然而，真正决定去向的既不是马夫，也不是马，而是马车上的**乘客**。

这个乘客就是**高我**，也可称为**大我**。这个“我”是一个更宏观、更整体的存在。在开始冥想时，你首先要获得的能力，就是连接这个高我或大我。

## 一个真实的故事：连接高我与疗愈

接下来，我想分享一个过去参加现场转变课程的学员的真实经历。

这位学员来到我们的课程，我教给了他一些冥想方法。在过程中，他讲述了自己的童年经历与创伤。小时候，因为父亲患病，全家收入都用于为父亲治病，导致家庭非常贫困。他在学校因经济条件差而受到同学歧视。

这位学员非常努力。由于先天条件不好，他后天格外刻苦，并因此获得了一个特等生名额。有一天，他开心地告诉母亲这个消息。然而，学校老师与母亲谈话后，母亲走出来对他说，家里实在供不起他深造，建议放弃这个名额。

这件事让他深受打击，他开始怨恨父亲，认为是父亲造成了当时的处境。这段童年经历导致他后来非常痛恨金钱，因为父亲的病是家庭贫困的根源。

分享完故事并参加完课程后，他开始按照我的方法每天坚持冥想。有一次，他激动地找到我，说在冥想过程中，他突然能够感受到父亲当年陷入困境时的情绪和感受。

许多童年的记忆涌现出来，但这次他不是从自己的角度，而是仿佛从**父亲的角度**去亲身经历那些事。当他感知到这一切时，他突然明白，造成他当前处境和信念的，既不是父亲，也不是他自己。

想通之后，他感到释怀，认为自己不应该如此怨恨父亲。我建议他，既然想明白了，可以给父亲打个电话，聊聊这些感受和想法。

于是当晚，他立刻打电话给父亲，进行了几十年来唯一一次深度沟通。这次沟通化解了彼此长期的心结，让他看开了许多事，也解除了许多限制性信念。

## 什么是主动转化？

上述过程就叫做**主动转化**。那么，如何进行主动转化呢？

首先，你需要能够感知他人的感受。我们可以把小我想象成一个**点**，平时我们的意识都聚焦在这个点上。但实际上，在这个点之上，存在一个更高的意识维度，即**大我**。

大我涵盖了许许多多的小我“点”。通过冥想，你可以从这个“点”慢慢上升、扩散，从而连接到**高我**。

当你处于高我的视角时，你就可以连接到这个点、那个点，连接到许许多多的点。这种状态在佛教中也被称为**三摩地**或**合一**状态。在此状态下，你可以感受到他人的感受。

通过感受他人的感受，你可以采取行动，进行主动转化，从而解除童年创伤、消除“刻印”（即固有的负面印记），并纠正一直以来的偏差认知。

## 具体方法：冥想练习 🧠

那么，具体如何做到这一切呢？这就是我们今天的第一个练习：**冥想**。

冥想有几个要点。除了我们之前在免费视频中讲过的——观察你的呼吸——你还需要注意：吸气时觉察吸气，呼气时觉察呼气。但请记住**不要数数**，因为当你数数时，念头会集中在数字上，而不是在观察呼吸本身的感觉上。

你必须把注意力放在呼吸从鼻子进出时的感觉上：**吸气，呼气**，观察这个感受。

在观察过程中，你的头脑会出现许多念头和想法，这些叫做**妄念**。大部分人在初学冥想时，都会陷入妄念，被想法带走。这非常正常，不必感到奇怪。

当你发现自己被念头带走时，应该怎么办？答案是：**观察那个念头**，仅仅是观察。很多人会陷入念头的内容，例如：“今天中午吃什么？吃冒菜吧？算了，还是吃火锅，好久没吃了。” 这就陷入了念头。

但你只需像一个旁观者那样观察它。可以想象自己在看一场电影，电影里有剧情和角色，但你不会认为电影里的人就是你。你只是在观看。

当你以这种观察者的角度去观察念头时，你会发现念头升起，然后灭去。当念头消失后，你重新将注意力拉回到呼吸上，继续观察：**呼气，吸气，呼气，吸气**。

建议每天至少练习**30分钟**。对于新手，最好早晚各做30分钟。

## 冥想的作用：打开意识，连接智慧

这个练习有什么用呢？告诉大家一件很有意思的事：我们的大脑是全身除了心脏以外，唯一不会休息的器官。睡觉时，你的肺、胃可能在休眠，但大脑不会。这就是为什么人在睡眠时会做梦。

我过去在寺庙灵修时，一位师兄戴了一块智能手表。出来后，他将数据上传到朋友圈，发现一个有趣的现象：数据显示他一天有长达20小时的“睡眠”。因为在寺庙里，每次冥想长达10多个小时。

为什么会这样？他说，因为在冥想进入深度状态时，手表记录的健康状态被视为**浅度睡眠**状态。所以，当你真正冥想进入**入定**状态时，你的意识就打开了。清醒时，你的意识可能只停留在小我层面；但冥想进入状态后，意识就上升到了**本我**和**高我**。

潜意识中的许多活动会浮现在你面前。因此，一开始你要找到那种感觉，进入那种状态。进入状态后，你能体会到连接大我、高我的感受。

通过这种感受，你可以获得并学习高我给予的**智慧**。然后，将这些智慧运用到生活中，进行主动转化的过程，从而消除你的“刻印”和童年创伤。

## 总结

本节课中，我们一起学习了：
1.  通过马车比喻理解了**本我**、**小我**和**高我**。
2.  通过真实故事，了解了连接高我如何帮助**疗愈创伤**与**转化信念**。
3.  学习了**主动转化**的概念，即通过连接高我视角感知他人，从而解除限制。
4.  掌握了具体的**冥想练习方法**：观察呼吸、不数数、以旁观者角度观察妄念。
5.  理解了冥想的作用在于**打开意识**，连接更高智慧，并将其用于生活转化。

坚持练习，你将逐渐学会与内在的高我合一，开启自我转变的旅程。
