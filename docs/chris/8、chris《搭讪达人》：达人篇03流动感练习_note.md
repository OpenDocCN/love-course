# 8、chris《搭讪达人》：达人篇03流动感练习_note

在本节课中，我们将学习搭讪中的一个核心概念——**流动感**。你将了解如何摆脱机械化的对话模式，让交流变得自然流畅，从而提升搭讪时的自信与自在感。

---

## 从固定思维到自然流动

上一节我们探讨了搭讪中的基础心态。本节中，我们来看看如何克服对话中的“机械化”问题。

你可能会在交流时使用一些固定的语句。这可能是因为最初不知道说什么，于是背诵了一些话术，后来形成了习惯。即使对方反馈良好，你有时也会因为不知道接下来该说什么而匆忙结束对话。

这种状态源于你在对话中过度思考“我应该说什么”，而不是自然地表达。与朋友聊天时，你不会这样思考。那种自然的状态就是**流动感**。

**流动感**是指在搭讪时，你能达到一种非常自然自在的感觉。大部分人在搭讪时会不断思考自己该做什么、该说什么。而流动感要求你像与朋友交谈一样，想到什么就自然地表达出来。

---

## 理解思维的两种模式

我们的思维在工作时遵循一定的原理。固定思维更偏向使用**左脑**，负责逻辑与序列；发散思维更偏向使用**右脑**，负责直觉与创意。

搭讪中的“机械化”源于过度依赖左脑的固定思维。练习流动感，就是训练自己从左脑的固定思维，切换到右脑的发散思维。

---

## 流动感三步练习法

以下是培养流动感的三个关键步骤。你需要依次练习，让对话自然展开。

**第一步：不害怕尴尬**
即使出现冷场，也不要认为这很尴尬。接受对话中的自然停顿。

**第二步：用语言引导思维**
当突然没话讲时，让你的行为先动起来。可以使用一些汉语中的虚词或助词来填充思考间隙，例如：
```
“嗯...是的...”
“我觉得...”
“我想...然后...”
“所以...”
```
说这些词的同时，你正在强迫思维从左脑切换到右脑，让大脑运转起来。

**第三步：相信直觉，直接表达**
当你大脑产生想法时，不要怀疑“该不该说”。相信你的第一直觉，直接说出来。不要担心内容是否奇怪或丢脸。在练习阶段，这是必要的。

例如，如果你突然觉得对方的裤子很奇怪，可以直接说：“哎，我觉得你的裤子好奇怪。” 无需在意对方的反应。反复练习后，你的交流会越来越自然，逐渐拥有流动感。

---

## 处理移动中的目标

在实战中，你可能会遇到对方不停下脚步的情况。除了流动感，你还需要学会“锁定”对方。

最好的锁定方式是**眼神**。你需要始终保持与对方的眼神接触，进行强有力的眼神沟通。

如果对方缓慢移动，你可以一边说话，一边缓慢移动到她的侧前方，然后逐渐停下脚步。如果对方走得急，你可以使用肢体动作和语言让她停下，例如：
```
“哎，停一下停一下。”
```
关键点在于，你的**肯定感**必须强于对方要离开的意图。你的脚要坚定地停下来，让她为你驻足。当她停下后，你再调整到与她平行或侧前方的位置，方便交流。避免一直跟在后方，这会增加对方交流的负担。

## 本节课总结

本节课我们一起学习了搭讪中的**流动感**。我们分析了固定思维与发散思维的区别，并掌握了三步练习法：**不惧尴尬、用语言引导思维、相信直觉直接表达**。同时，我们也学习了如何通过眼神和肢体动作，有效“锁定”移动中的目标，增强对话的肯定感。

通过反复练习这些技巧，你可以摆脱机械化对话，让搭讪交流变得自然、流畅、自信。
