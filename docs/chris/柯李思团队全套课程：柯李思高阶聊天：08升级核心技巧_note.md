# 柯李思团队全套课程：柯李思高阶聊天：08升级核心技巧 🚀

在本节课中，我们将要学习两个最实用的升级技巧，帮助你有效地推进与女生的关系。升级，字面意思就是关系的推进。虽然肢体接触是最直接的升级方式，但在网络聊天中，我们更多地依赖于语言上的推进。

我们需要明确推进关系的目的和方向。推进关系主要有两个方向：一是让聊天氛围更具性张力，使关系从陌生、熟悉向更亲密的方向发展；二是朝着现实见面的目标推进，将线上交流发展为线下互动。

在开始具体技巧前，需要明确一个前提：你需要在良好的聊天氛围基础上进行，或者至少对方的反应是积极的。在这个基础上，你可以不断尝试推进。

以下是本节课将要介绍的两个核心升级技巧。

## 技巧一：角色扮演 🎭

上一节我们定义了升级，本节中我们来看看第一个技巧——角色扮演。角色扮演是指你与女生共同进入一个假定的角色关系中进行互动。

你可以扮演任何角色组合，例如：
*   哥哥与妹妹
*   爸爸与女儿
*   主人与仆人
*   老师与学生

在设定的角色框架内进行调情和关系升级。这是一个非常有效的技巧，原因在于：当使用角色扮演时，女生明白你是在开玩笑。因为你们共同假定了一个虚拟角色，所以在此背景下进行升级，你的风险很低。即使升级未被接受，对方也会理解这只是一个角色扮演游戏，避免了尴尬。

## 技巧二：假定邀约 📅

在理解了角色扮演后，我们来看第二个技巧：假定邀约。这个技巧是指，假定你们已经要见面或已经在一起做某件事，并描述具体的细节。

例如，你可以说：
*   “我们吃饭的话肯定不能坐在一起吃，因为你喜欢吃辣，我喜欢吃甜的，我们要分桌。”
*   “我跟你在一起喝酒的话，你肯定喝不过我，我要把你给干倒。”

这些表述的共同点是，你已经假定了见面的事实和具体场景。这样做的优势在于：如果你直接询问“要不要出来”或“我们去干嘛”，女生可以直接拒绝。但当你提前说出见面后的具体活动框架时，她无法简单地拒绝整个构想，因为她无法否定一个尚未发生的具体细节。

## 技巧组合应用 💡

以上两个技巧可以组合使用，效果更佳。我们通过一个具体例子来说明如何将角色扮演与假定邀约结合。

假设聊天涉及到年龄话题。女生问：“你多大？” 你回答：“我今年28快30了。” 女生说：“你好大啊，你好老啊。” 这时，你可以引导至角色扮演：
> 你说：“对啊，我确实很大，我都可以做你爸爸了，所以你是不是应该叫我一声爸爸？你是我12岁的乖女儿。”

这样，你们就进入了“爸爸和女儿”的角色扮演。紧接着，可以融入假定邀约：
> 你说：“乖女儿，爸爸周末可以带你出去玩，带你去吃儿童套餐吧？”

这个组合让你在一次互动中，能更自然、顺利地朝着现实见面和关系亲密化的目标推进。本质上，这是将基础的聊天材料，通过一些技巧性的“游戏”（game）进行包装，使关系推进过程变得有趣而顺畅，而非平淡无奇。
