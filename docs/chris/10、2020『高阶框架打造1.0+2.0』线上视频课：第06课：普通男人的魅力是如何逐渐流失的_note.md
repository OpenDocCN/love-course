# 10、2020『高阶框架打造1.0+2.0』线上视频课：第06课：普通男人的魅力是如何逐渐流失的

## 概述
在本节课中，我们将探讨日常生活当中男性魅力逐渐流失的原因，并学习如何通过价值展示和框架建设来提升吸引力。

## 价值与价值感
**公式**：价值 = 绝对价值 + 相对价值

以下是关于价值和价值感的解释：

- **绝对价值**：指个人实际拥有的价值，如收入、财产、才华等。
- **相对价值**：指对方感受到的价值，即对方对个人价值的认知。

**例子**：
- 网络流行语“原以为你是个青铜，没想到你是个王者”体现了价值与价值感的差异。
- 奢侈品包包在不同环境下的价值感不同。

## 框架的力量
**公式**：框架 = 外在条件 + 内在框架

框架是指个人在社交互动中的心理状态和行为模式。

**例子**：
- 视频素材中的王子与王妃的故事展示了框架的力量。尽管王子的绝对价值高于王妃，但最终是王妃的框架更强，导致王子妥协。

## 推拉技巧
推拉技巧是指在关系中通过一推一拉的方式，保持自己的框架，同时给予对方可得性。

**例子**：
- 视频素材中的王子在王妃的要求下，先蒙事，然后掀开盖头，揽入怀中，深情地看着她蒙事，既维持了自己的框架，又满足了王妃的需求。

## 总结
本节课中我们一起学习了价值和价值感、框架的力量以及推拉技巧。通过这些知识，我们可以更好地理解自己在社交互动中的魅力流失问题，并学会如何提升自己的吸引力。
