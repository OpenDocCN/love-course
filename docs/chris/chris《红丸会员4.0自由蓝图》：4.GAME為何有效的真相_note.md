在本节课中，我们将要探讨GAME在两性互动中为何有效的深层逻辑，并解析权利、责任与自由之间的核心关系。

听完前三节课关于客观权利的阐述后，一个常见的问题是：既然客观权利是事实，那么提升自己是否才是学习GAME的唯一出路？首先，需要明确回答有此疑问的兄弟。

提升自己是一条正道，这绝对正确。但在我们当今的社会中，GAME之所以显得非常重要，是因为这个时代的大部分人并非活在绝对的现实之中。

大部分人活在一种主观的想象里。他们认为自己主观想象的权利大于客观事实的权利，从而扭曲了眼前的现实。这个真相体现在：无数高估自身价值的女人，正是被无数低估自身价值的龟男、舔狗、软男所塑造出来的。

想象一个客观权利巨大（例如富有）的男人，在他喜欢的女人面前，却将自己想象得一文不值。他在主观权利上认为自己配不上这个女人。

此时，这位有钱人的客观权利便完全无法生效。这就是GAME在当今时代运用起来会非常有效的原因。你必须理解这背后的逻辑。

如果不能理解如此简单的真相，那么学习GAME就是在白费功夫。因为，如果现代社会像古代一样，人被明确划分为三六九等，每个人的主观权利都与客观权利相匹配，那么你在两性关系中学GAME的可操作余地也会非常局限。

因此，学习GAME的同时，必须了解红丸男的权利意识。每个人都向往权利，却未必知道权利意味着什么。**权利并非肆意妄为，权利必须依靠他人的支持与配合才能产生**。如果一个王国里只有国王自己，那么他的权利毫无意义。

所以，权利与责任是一体的。拥有的权利越大，承担的责任也就越大。阿尔法男在享受权利的同时，其实承担了更多责任所带来的压力。

即便在古代权利巅峰的皇帝，也不得不每天重复处理无聊的工作，例如批阅奏折。这是他支配权利后必须尽到的责任。他必须这么做，否则他的权利就会被颠覆。

例如，如果皇帝开始不愿处理这些重复无聊的工作，转而荒淫无度。那么随着他放弃责任，权利就会逐渐转移。宰相可能手握实权，臣子们表面配合，背后却不再支持皇帝。最终，皇帝的权利被架空。

而皇权被颠覆的后果，就是皇帝会被直接干掉。因此，大部分人只看到权利带来的好处，却没有看到权利背后的责任和风险。

古代能做皇帝的人并非由自己说了算，他无法自由选择命运。因此，在我看来，皇帝并非我们所说的自由男人。

反观权利结构之下，作为被权利支配的角色，这未必是坏事。因为他们所需承担的责任往往更少。就像历史上，女人通常处于被支配的角色，所以女人天生缺乏一种责任意识。

但这并非坏事。因为没有责任，意味着可能拥有更大的自由。这也是许多成年人向往儿时状态的原因，觉得孩童时期是自己人生中最开心愉悦的时候。正是因为你在幼时受父母庇护。

在那样的状态下，你无需承担任何责任，可以想做什么就做什么，完全放飞自我。因此你每天都很开心快乐，因为你的状态建立在绝对安全的、无责任的自由之上。

女权主义将女人这种“去权利化、去责任化”的状态形容为男人对女人的压迫。女权呼吁女人获得权利本身没有错，但问题在于，她们所呼吁的是一种没有责任的权利，是一种无限膨胀的权利。

这就是女权运动错误的原因。女人想要权利却不想尽责任，这必然导致错误的结果。而这些历史上错误的结果，会由一些愚蠢的支持者来买单，例如那些支持女权的“白骑士”、龟男、软男和舔狗。

从这个角度讲，女人会想找一个能够庇护她的阿尔法男人。为什么女人总喜欢说“我希望找一个能像宠爱小女孩一样宠爱我的男人”？因为女人想在这段关系中扮演那个放弃责任、自由快乐的角色，就像小孩一样。

而男人无法完全“去责任化”。这也是我们红丸理论所说的：**男人的自由，来源于尊重秩序下的另一种相对的自由**。身为一个红丸男人，你必须理解这一点，理解这个游戏的规则就是这样运行的。
