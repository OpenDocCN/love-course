# 【女性主义】为什么田园女权是女

在本节课中，我们将要学习“田园女权”这一概念在中国社会背景下的起源、其核心特征，以及它与现代女权主义的根本区别。我们将从社会文化、经济结构和个人心理层面进行分析。

## 社会文化背景：城市化进程与思想差异

上一节我们介绍了课程概述，本节中我们来看看“田园女权”产生的社会土壤。

本质上田园女权跟现代女权没有任何区别。我们先来讲为什么在中国这个环境里会出现“田园女权”这个词语。你会发现这个词在西方是没有的。这其实就是中国特有的一种社会文化、经济进步所导致的。

我们先从大环境来看为什么会有田园女权。我们要清楚一个事实：中国在2020年刚达到60%的城市化比例。这意味着中国40%的人口生活在农村。因为中国是一个农业大国，有很多农村、二三线城市、五六线城市。

在那些地方的人，他们其实跟现代化城市里的人在思想上的差别很大，成长环境也不同。所以其实我经常说这样一句话：在中国，人与人的差距很大。我们说得夸张一点，你可能在不同的城市环境里，有些地方是现代社会，有些地方是第三世界。

如果你在像上海这样的现代化大都市里面，这样的城市跟国际是非常接轨的。但如果你在安徽、河北的一个农村里，二三线城市的一个农村里面，你在那样的环境里，思想跟一个在现代化城市里的人会有很大的差距。因此，中国特有的这种农业大国背景和高农村比例，就导致了“田园女权”这个词的出现。

## “田园”二字的含义与核心特征

理解了宏观背景后，我们来看看“田园女权”这个标签本身意味着什么。

田园女权的重点是“田园”这个词。什么是“田园”？就是农村里面支持女权的人，所以叫“田园女权”。这样的人，他的思维模式跟现代女权会有什么区别？区别在于：田园女权本质上并不是一种左翼思想，它骨子里还是传统保守的。

虽然我是上海城市里面长大的，但是我也接触过很多这样的客户或者群体。你要知道在农村里面，农村人的思想是什么样的：**重男轻女**。原因很简单，因为农业社会男性是最强的生产力，女儿就是赔钱货。所以农村的思想是非常保守的。

因此，田园女权骨子里它是一种右翼思想，是传统保守的。父母都给他灌输：你作为一个女人，就是个赔钱货，重男轻女。

我给大家讲一个很极端的故事。以前我爸有一个东北的朋友，这个东北人就是在东北的一个小城市里长大的。他有一个女儿，但是他爸就特别的重男轻女，看不起女儿，骨子里那种观念认为女儿就是赔钱货。所以他就一直让他的老婆生，结果生了几胎都是女儿。生出来的女儿赔钱怎么办？就送给他的亲戚来抚养，他就一定想生一个儿子。

甚至到最后极端到什么程度？他的老婆已经上了年纪不能生了，他还要到外面想办法“种精求子”。那些地方人的观念是非常传统保守的：女儿是赔钱货，儿子就是赚的，一定要生儿子而不是生女儿。

## 内在冲突与行为模式

在传统观念下成长的人，接触现代思想后会产生怎样的化学反应？本节我们来探讨这种内在割裂。

田园女权是在这样的一个环境下面长大的。但是因为现在互联网信息时代，社会的这些思潮、文化舆论又很发达了，他们又接触了网上这些所谓新时代左翼女性的思想以后，就会导致他会有一种内部的割裂。

他骨子里面是一个传统保守的女人，但是在网上一看那种激进言论，他说“原来我是在被压迫，原来我的父母一直重男轻女是父权对我的压迫”。然后他就要做一件什么事情：**去责任化**。他要反过来翻身做主。

但是他们所谓翻身做主的方式是跟现代女性不一样的。因为他骨子里还是传统保守的，他所谓翻身做主的方式，不是像现代女性、现代女权那样去打拼自己的事业、模仿男人。

以下是他们的核心行为模式：

*   **目标对象**：找一个beta男，找一个软男跟他结婚。
*   **核心手段**：勒索他一笔，剥削他一笔。
*   **关系模式**：让这个男人结婚以后变成个“龟男”，在家里归着老婆，所有的事情（如洗衣服、做饭、洗碗、带小孩）都由龟男包办。
*   **理论依据**：因为田园女权说“我的生育是最有价值的”。所以他们所谓宣扬女权思想，是为了去抬高自己在婚恋市场当中的溢价权。

这是田园女权行事的一种方式。所以为什么很多人会看到现在有什么天价彩礼，就是乱七八糟的这种结婚，女的要求男的买车买房。

## 问题本质：权利与义务的割裂

最后，我们来剖析这种思维模式的根本问题所在。

从我的角度来讲，如果你不能给女人一个保障，这是你的问题。因为你作为一个男人，这是你最基本在这个世界上要去为自己打造生存空间的责任。这是你的问题。

但是田园女权的问题在哪里？他的问题在于：他想要所有的东西，但是他却不尽自己的义务。因为他觉得尽自己的义务是在被父权压迫。他们想做什么？他们想什么都不付出，然后做一个甩手掌柜。

所以这是最底层的一种、非常愚蠢的一种想法。为什么？就凭他有一个子宫，有生育能力吗？这就是田园女权的想法。

**本节课总结**

本节课中我们一起学习了“田园女权”这一概念。我们了解到它是中国快速城市化进程中，传统农村保守思想与互联网时代激进女权话语碰撞后的特殊产物。其核心特征在于**思维内核的保守性**（右翼、重男轻女环境塑造）与**行为诉求的激进性**（要求权利、拒绝义务）之间的深刻矛盾，最终表现为在婚恋市场中通过强调生育价值来索取单向利益，而非追求平等的个人发展与责任共担。
