# 课程一：现场游戏大师1.0 - 快速约出女生的方法（上）🚀

在本节课中，我们将要学习一种被称为“速约”的现代两性交往思维。我们将探讨其本质、意义、所需的前期准备以及核心规律，帮助你理解并适应新时代的社交模式。

## 概述：什么是“速约”？🤔

很多人将“速约”理解为“约炮”。本节课确实会涉及这部分内容，但主题是“约炮”。这并非什么值得惊讶的事情。

在很多人的观念里，“约炮”可能是一个比较夸张另类的现象。事实并非如此。很多人抵触这种文化，最主要的原因是中国早期那种传统式的思维。

这种思维认定，人与人关系发展过快是不靠谱的。关系的确定，通常被认为需要先认识，再接触，然后一起共事合作，最后发展成长期关系。婚姻、工作、朋友关系都是如此。

很多人会认为，关系确立过快可能不利于关系发展。事实真是这样吗？未必。

当然，关系发展较快可能会带来一些不利因素，但这不起决定性作用。两个人关系能否相处得好，关键在于双方，甚至更多在于个人方面。

因此，速度快并不一定代表关系不稳定。中国传统思维认为，两性关系的确认应该先熟悉，谈恋爱，先交往，再牵手，上床的事情慢点再说，甚至有些人认为要婚后才行。

这种思想不能说是错的，但它已不利于在现代社会生存。一旦社会踏入21世纪，科技进步和文明发展直接导致了人们生活习惯的改变。

例如，六年前我们睡觉前可能不会玩手机，而是看书听音乐。但现代人会在床上玩手机。科技进步导致人类生活习惯发生变化，我们也要适应时代的变迁。

今天讲的课程，我认为是现代人对于恋爱观的一种革命。这不仅是讲约炮，更多的是告诉大家一种两性交往中的思维和逻辑，而不仅是一个技巧。目的是告诉大家应该顺应时代。

这个时代发明了很多高科技，互联网在中国非常发达，甚至在某些方面已领先于国外市场。医学也在进步，例如有新闻称美国公司研制出解决艾滋病的疫苗。时代已有许多突破性的科技发展，我们也应该顺应社会。

现代人应该顺应社会生存，不要被老思想禁锢。后面我会逐步讲解今天课程的内容。

## “速约”的起源：一个美丽的艳遇传说💫

下面来讲一下“速约”的起源。我写下一句话：一个美丽的艳遇传说。

无论是男人还是女人，科学家做过调查，在最初还是陌生人的阶段，男女双方对彼此的期待率最高，相应的感觉也最强烈。吸引是一种情绪。

在交往初期的界面，男女双方的吸引力各自都是最强烈的。这是不容置疑的。那时我们大脑中产生大量多巴胺。也就是说，人面对陌生人会更有感觉。

因此，你经常可以在马路上发现，有些身边有男朋友的女人，有时还会瞟你。甚至有男朋友的女人可能更过分地出轨，发生一些伦理上不道德的事情。这说明人类本质上都喜欢寻求刺激。

人类的好奇心推动人类进步。同样，这份好奇心放在两性情感关系中也至关重要。所以，“速约”本质上就是一个美丽的艳遇。

大家不要把它想象成见不得人或见不了光的行为。但你自己心里要清楚，不能随便乱说。也只能跟几个好哥们聊天时说，昨天有个艳遇，“速约”了一个妹子。

私密课与公开课不同，私密课是精准定位，公开课是大概脉络。这位同学对课程分析得很到位。

在我看来，“速约”就是一种故事，是你人生中的一种故事。一场婚姻可以被认为是人生中一个至关重要的生活化节点。每个人都可能结婚，这很日常。婚姻本质上不是什么很伟大的事。

不要看心灵鸡汤把婚姻讲得特别伟大、高大上。这些都是人类对这种关系的鸡汤性描述，也是为了便于社会维系稳定的一种方式，将婚姻描绘得比较完美。否则，如果主张不婚主义，整个社会肯定会乱套。

婚姻比《非诚勿扰》靠谱多了。《非诚勿扰》是相亲节目，但结婚并不神秘。今天我的一些言辞可能会与你以往的恋爱观产生冲突甚至矛盾。

但我觉得这是剖析事物的本质和本源，不得不说。我个人认为婚姻就是一种日常，不是一件很伟大的事情。所谓伟大，一定存在于特定性或唯一性、不可比性。每个人都会结婚，伟大在哪里呢？

我们说某人是伟人，他一定有特定性，是最独特的。婚姻不是什么独特的东西，在我看来就是一种日常，是人类维系社会稳定的一种关系存在方式。

说白了，就是维系社会稳定的一种方式。最早婚姻可追溯到与宗教有关。原始部落，包括教堂里的牧师，国外要去教堂，中国是拜天地。在上帝或天地面前见证爱情，为的是告诉你，婚姻是有人看着的，不要随便忽悠。

本质上它源自宗教。当时人类科技文明可能不那么进步，需要靠这种维新主义思维去维持人类社会关系。否则，没有婚姻制度，随便找女人生一堆孩子，谁来照顾？最后负担还是落在政府或当权者手上。

在一个部落，孩子需要整个部落照顾，又不能扔了或杀了，这也是部落香火的延续。怎么办呢？总得有人照顾。所以出现了婚姻制度，告诉你别乱来，否则上帝会惩罚你。你就得结婚，养出来的孩子自己照顾，因为他是家庭的一份子。

圣经里也说，男人别出去乱搞，如果和女人发生关系，就得养她，把她作为家庭的一份子。很简单，生出来的孩子需要你来照顾，没人愿意替你承担这份责任。这是维系社会平衡的一种方式。

所以说，“速约”本质上讲，就是一个美丽的传说。但我觉得这也不是传说，它是可以实现的。

## “速约”的意义🎯

下面来讲一下“速约”的意义。概括来说，“速约”在你生命中的意义到底在哪里？

前面讲了一个大概的、比较美好的意义，我觉得是一种故事性。它在你生命中是一种故事，当你回首往事时，无论是向兄弟描述还是自己回味，生命都会因此更加丰富多彩。它带有故事性，是一个精彩的故事。

“速约”的意义在于建立良好的资源。如果你养成一个习惯，什么习惯呢？就是要多去收集资源的习惯。

只有当资源累积到一定量时，“速约”的可能性才会成比例地提升。很多人会有疑问：为什么资源量大了，“速约”的可能性就大呢？

道理很简单：如果你要学游泳，在陆地上学会的可能性有多大？还是直接跳到河里、江里、海里？虽然可能冒着淹死的风险，但你绝对会在里面学会游泳，而且速度会很快。这就是为什么资源量达到一定量、建立资源池后，“速约”的可能性会变得很大。

“速约”的意义也在于，如果你有“希望在人生中有美妙故事”的想法，就要养成收集资源的习惯。养成习惯后，你必然会积累这样的资源池，建立资源池。

我以前也有这个习惯：每当我空闲时，包括现在工作再忙，也会抽出一点时间聊妹子，收集资源。我可能搭讪的机会不多，只能在拍视频时稍微搭讪一下。

我也可能在闲暇时，抽出碎片时间聊妹子，谈谈也好，默默也好，一直聊。很多人会说，大力哥，为什么你聊天那么好？因为你聪明，是导师，帅，有好的展示面，思路清晰。

我告诉大家，很多人都会找这样的理由：因为你什么什么，所以我没有。在失败的人面前，理由有千万个。但成功的理由只有一个：**坚持做自己的事情**。

你不需要那么多理由。我也不是天生就会，是文化将我领到这个领域。我出生时也不是会妖媚的人。我跟大家说过以前的经历，我也被女人甩过，不理睬过，案例太多了。

包括我自己在修炼初期，也会碰到各种阻碍。现在这个阶段，我也会碰到一些难对付的难题。当你碰到这些问题时，第一时间不要想“完蛋了，我跟别人不一样”。因为他帅，他高，我还没那么高呢，我净身高其实没意义，干脆跳楼算了。

我以前一个领导每次见到我就调侃，他问我身高，我说一米七，他说你这是三等残废。这句话在我心里印象非常深刻。当时我就觉得很不服气，这人太会说话了。要不是看在领导份上，我肯定要上去干你。

后来，经过我自己在实战中摸索，我发现，其实你身上有很多缺点，可能并不足以影响你。缺点一定存在，但同样你也有一些核心竞争力和优势。只要你能发挥优势，规避劣势，你自己作为武器，也是可以使用的。

“速约”的意义还在于，它能让你更懂得两性吸引的规律。所谓规律，就是让大家清楚，“速约”的可能性并非天方夜谭，不只限于传说。我们可以把传说变成现实，它有现实的可能性存在。

只要你们相信，它的可能性一定是存在的。这就是我们“速约”的意义。

## “速约”的前期准备🛠️

好，下面来讲“速约”的前期准备。我们该怎样去准备“速约”？如果我要快速约一个妹子，需要做些什么事情？

之前几节课该讲的都讲了，之后还会有一节课专门讲聊天。你知道大概会怎么聊了，接下来就该去练了。这件事我没办法完全将意识传达给你。我唯一能做的就是从聊天记录分析中帮助你，告诉你聊天记录的因果关系是什么。

或者分析某个学员的聊天记录，分析其中的因果关系。剩下练习的事情，就得靠你自己了。这没办法，无论你多聪明、有钱、有条件，实践永远属于你自己。

当我们前期准备了一些聊天技巧，关于“速约”方面应该准备什么呢？

**首先，你需要调节你的玩家属性。**

所谓玩家属性是什么呢？第一，是心态。玩家和普通男人之间的区别在哪里？在于心态。普通人对待感情非常重视，很在乎。玩家对待感情怎么说呢？不能说他不在乎，但会把这件事看得比较淡薄，或者说觉得轻如鸿毛。

他不会像普通男人那样，觉得认识一个喜欢的女生是重于泰山的。大家明白这道理吗？轻如鸿毛，重于泰山。普通男人看到一个喜欢的女人，会觉得这是我的泰山。但玩家会觉得，这就是一片羽毛而已。今天得到了是幸运，失去了也不过是一片羽毛。羽毛有多重呢？拿在手里跟空气一样。

但很多人确实做不到。这种调整需要经历实战过程。如果有兄弟从一开始来这里，逐步开始实战并有一些成绩出来，你们想一想，现在的心态和刚开始时的心态会一样吗？绝对会有变化。

你的心态会逐渐向“轻如鸿毛”的方向发展。但一开始大家不可能说得那么轻巧，可能心里还是放着一块石头。我相信很多人是这样。但随着时间的推移，你会发现，你对感情这件事并没有之前那么在意了。

感觉实战聊天有些难，确实。实战聊天需要一些开悟。什么是开悟？可能某一天你突然发现，哎，我聊天还不错哦。这就是突然的一件事。前期你会走过一段非常难的瓶颈期，一个比较难的学徒期。

那时你会去学习某个话术，再自然地把它内化成自己的东西。然后，你会进入一个比较自然的过渡过程，觉得可以跟妹子聊一聊了，甚至可能约出来。但到后面你会发现，还会进入一个瓶颈期。有些妹子你还是聊不过，聊天还是存在各种问题。

这时你就要去突破瓶颈期。突破之后，你会发现，聊天根本不用去想，该怎么聊就怎么聊。今天手机聊两句话，过一会儿继续聊。现在多数情况是抱着手机咔咔咔聊很久。

拿我自己来讲讲吧。我觉得我的学习能力还算可以，但不能算最优秀。我也参加过各种类型的培训，除了恋爱培训，还有其他方面的培训。在这些培训里，我算是一个中等偏上的学习能力的人。

我的聊天，从刚开始学到会聊天、能约出妹子，大概花了多久呢？当我约出第一个妹子时，花了差不多三四个月时间。但有一个前提：那时市面上聊天类型的教程非常稀缺。

没人会教你怎么去聊天、怎么去“速约”。那时市面上看到的东西都是很多人写的“一夜情”贴，比如今天约了个妹子，带她去开房，花了十块钱。这种东西完全没有学习价值，都是装逼文。

所以当时我们的学习教材有限。对于当时的我来说，唯一能提升自己的方式就是找一个哥们。大家学东西，如果能找一个哥们，其实能帮助你在修炼路上有推动作用，取得比较好的成效。

老师，每次和妹子出来吃完饭之后，妹子对我的感觉都降低很多。是吧？那说明你能约出来，但存在一个问题：你线上线下不一致。线上可能让妹子觉得你是有价值的人，但一到线下，价值可能就不匹配了。

还有一种情况是，你价值依然是这个价值，但问题在于你绝对不自信，没有自我肯定。自己看不到自己存在的价值。一到现场就怂，自我否定。说错一句话就低头，进挪一下都不敢。甚至进挪时，妹子一拒绝你就觉得自己毛手毛脚。

约会有难度，大力来帮你。这已经变成了一句标语性的话。那么我当时是怎么约妹子的呢？跟大家讲一讲。我当时有一个哥们，我建议大家找一个哥们一起练。平时可以偶尔见面，聊一聊感情上的事情。

像我当时刚开始练的时候找了一个哥们，名字就不说了。这哥们是我以前的高中同学。我之前学过跳舞，这哥们也喜欢跳舞，我去跳了一段时间。后来工作忙就不跳了。但这哥们跟我有相投的地方，他也喜欢练习新鲜好玩的东西。

比如他喜欢摄影，所以我的头像就是他帮我拍的。我有很多展示面都是他拍的，他摄影技巧确实不错，拿过大奖。至于找到这么一个哥们，真凭运气了。因为之前高中时我跟他关系不怎么样，老打架，两个人不对付。不是仇人，但互相看不爽。

没想到大学一毕业，咱俩关系特别好。他给我拍了很多展示面等等。后来这哥们又跟我一起练恋爱学，当时是PoA社区。现在PoA社区也被别人唾弃了。我觉得是有些人把PoA的招牌砸了，没办法。但当时文化确实给我带来了很多受益的东西。

我跟这哥们一起去学习理论技巧。跟很多学员一样，初期我们都把重心放在微信聊天上。那时微信刚开始有，大概是2012年末、2013年初。那时没有展示面的概念，我跟很多人一样，不懂展示面。

当时我跟这哥们就是一起聊天。怎么一起聊呢？比如我今天认识一个新妹子，我记得上一课讲过一点，我会去找他，因为他有时加班，我就在他办公室聊天。聊了两句话之后，如果不会回了，我就跟那哥们说，来来来，一起看看这句话该怎么回。

然后大家一起头脑风暴，想出后面那句话。当时满脑子就想这些话术，怎么把聊天聊得跟飞一样精彩。其实现在想想完全没必要，聊天自然聊下去就行了。很多时候展示面做足了，聊天就会轻松一点；展示面做不足，聊天就很累。

很不自信，有些价值在约会不会展示，展示出来让妹子感觉自己在装逼。对，这说明你在展示这些价值时可能有一些不太自然的地方。怎么说呢，装逼要装，我觉得还是要看人。有些妹子不喜欢人装逼，但有些妹子必须要你装逼，这要根据人来。

比如一些物质拜金、价值层面比较高的妹子，你装个逼是应该的。但如果你对自己价值差不多的妹子去装，她可能会有厌恶感。凭什么你喜欢把这些价值放在身上去招摇炫耀？人家也会觉得有些反感。

网络科是恋爱学吗？我现在讲的这些东西是我自己实战，以及学习各种理论知识（心理学、恋爱学、性学等）总结出来的。到底是什么学问？是PUA还是恋爱学？我觉得这不重要。就像打拳练拳一样。

李小龙说过，没有最强的功夫，只有最强的人。他发明截拳道，标语是：**以无法为有法，以无限为有限**。意思是，把世界上所有没有规律的东西变成有规律，将所有无限的东西在有限的范围内使用。

这句话很有哲学意义，同样可以用到我们这个学问中。实际上很多知识你都可以用得到，就看你怎么把这些知识用到自己身上来。并不是乱来，是在有一定的章法规律下进行，它一定能帮助你使用出这些技巧。人生如戏，全靠演技。

好，那么我接下来继续往下面讲，有些问题后面再去解答。然后，下面就是你要自己调节这么一个玩家属性。所谓玩家属性，你要把自己的心态调整好。不能觉得今天好不容易约出来了，没有推倒，没有发生关系，就感觉很亏。

你有这样的心态，第一步就输了。你老在乎这么一个目标，老在想该不该拥有这个目标，那你当然在第一步就输了。俗话说，谁先在乎谁先输。同样，在我们这里也是一样的，你在乎你就输。
