## 概述：为何需要色相冥想？ 🤔

当你遇到一个你认为高出自己许多的女性时，你可能会感到自己配不上她。今天的色相冥想练习，就是为了让你在任何女性面前，都可以保持一颗最自然、最平等的心。

上一节我们介绍了练习的目的，本节中我们来看看如何开始。首先，将注意力放到呼吸上，慢慢放松身体。

找一个最舒适的姿势，安静地坐下，并时刻保持觉知。现在，开始深深地吸气。色相冥想练习的目的，就是为了消除你的分辨心，消除你的批判心态。

在吸气的过程中，感受身体越来越放松。你感觉没有任何烦恼，没有任何困扰。

以下是身体逐步放松的引导：

*   从脚部开始，放松所有肌肉。
*   放松小腿的肌肉。
*   放松大腿的肌肉。
*   放松腰部的肌肉。
*   放松手和手臂的肌肉。
*   放松肩膀的肌肉。
*   最后，放松额头的肌肉。

此刻，你全身上下的肌肉都处于非常放松的状态。

## 核心练习：消除分别心 👁️

现在，将你的注意力放在你对面的人或物体上。放下你关于这个物体的所有联想和信息。

**核心概念**：`她在你面前仅仅是一个人，仅仅是一个女人。没有所谓的高低之分，没有所谓的美丑之分。`

开始一段色相冥想的练习。在听到提示音之前，一直保持你的觉知。

如果此刻你对你的身体有所谓的美丑之分，你会感到不舒服。持续练习，直到这种分别心减弱。
