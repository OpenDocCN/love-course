# 课程5：陈大力《现场游戏大师1.0》：声音的奥秘 🎤

在本节课中，我们将要学习声音的奥秘，了解声音如何影响他人对我们的印象，并掌握一些基础的练习方法来改善自己的声音。

声音是人最先能够记住的元素。这在心理学和生物学上是一个科学认知。声音是记忆的第一位。例如，当我们听到一段熟悉的旋律，就能立刻联想到相关的场景或作品。声音之后是颜色，然后是味道，再是触觉等其他感觉。因此，好的声音在吸引他人，尤其是异性时，具有很大的优势。

上一节我们介绍了声音的重要性，本节中我们来看看声音是否会影响他人对我们的判断。

声音的强弱并不直接决定一个人是否在刻意奉承或迎合。声音能够影响人的情绪，但与你是否在讨好他人没有直接关系。例如，在搭讪时，要求学员提高音量，实际上是在帮助他建立更强的心理框架和态度。行为会影响心理，心理也会影响行为。一个内心坚定的人，声音自然会响亮有力。因此，声音温柔与否，与是否恶意奉承是两回事。关键在于眼神和态度是否依然有力量。然而，声音过轻会让他人听不清，从而无法有效关注到你。

接下来，我们来深入了解声音的结构。

声音由多个部分组成。平时说话时可能没有察觉，但其中存在一些结构。

以下是声音的几个核心组成部分：

*   **音高**：指声音的高低。例如，在激烈争辩时，声音会自然变高；而在说悄悄话或表达深沉情感时，声音则会变低。
*   **音势**：指音量的强弱，其中包含一种力量感和穿透力。例如，有力地说“我宣布”与平淡地说“我认为”，音高可能相似，但音势不同。
*   **音长**：指发音时间的长短。在聊天中，可以通过拉长或缩短某个音来突出重点。例如，在讲故事时，拉长“哎——”这个音，可以引起听众的注意。
*   **音质**：指声音的品质或质量。不良的说话习惯，如结巴、声音沙哑、大舌头等，都会影响音质。这些习惯可以通过练习来修正。

了解了声音的结构后，我们来看看如何在实际表达中运用它们。

在表达时，需要注意抑扬顿挫。这意味着说话要有节奏、有停顿、有轻重变化。就像写文章有黑体字突出重点一样，说话时也可以通过加重音来强调关键信息。例如，在讲述一个关于小狗的故事时，可以加重“小狗”这个词。这样做能让表达更有画面感，更容易让他人理解和投入。切忌语速过快，像连珠炮一样把话说完。

除了语言本身，我们的身体状态也会影响发声。

发声时的身体状态很重要。首先要放松，肩膀下沉，双腿自然打开，这是一种自信的姿态。说话时要配合手势，这不仅能帮助打节奏，也是情绪表达的一部分。要避免抖腿、摸头、抠鼻子等小动作。站姿要挺胸收腹。关于肢体语言的更多细节，可以回顾相关课程。

现在，我们来探讨发声时的意识状态。

发声意识是指发声时的心理状态和感觉。

以下是几种重要的发声意识：

*   **穿透意识**：感觉自己的声音能够穿透面前的物体。可以拿一个杯子作为参照物进行练习，想象声音像子弹一样穿透它。例如，练习说“今天的天气特别好”，并感受声音的穿透力。这有助于养成有力的发声习惯。
*   **抛物线意识**：感觉自己的声音像抛出的石头一样，呈抛物线轨迹传递到远处。这在大型演讲或会议中尤其有用，能帮助声音传递得更远、更清晰。这种意识能帮助你创造发声的力量感。
*   **节奏意识**：说话要有节奏感和抑扬顿挫。避免毫无停顿地快速说话。可以通过朗读文章来练习，在适当的地方停顿，并带入相应的情绪。例如，对比两种朗读方式：“警长从柜台上拿起一本拍纸布和一只铅笔然后说道你如何和他们通过报信……”和“警长从柜台上拿起一本拍纸布和一只铅笔，然后说道：‘你如何和他们通过通报信？’”。后者显然更有节奏和感染力。

要让表达清晰有效，吐字清晰是关键。

吐字清晰至关重要，它确保别人能听清并理解你的话。

以下是关于吐字清晰的要点：

*   **咬准字头字尾**：说话时，刻意咬准每个字的开头和结尾。可以拿报纸或小说进行朗读练习。
*   **说好普通话**：尽量说标准的普通话。如果身边环境影响较大，可以多和普通话标准的朋友交流。
*   **辅助练习方法**：有一个特别的练习方法是，用手托住下巴，尽量保持下巴不动，靠上颚活动来说话。例如练习说：“两岸的屋舍越来越密，河道越来越窄。”这个练习能帮助锻炼口腔肌肉，使发音更标准。
*   **运用方言或外语**：在聊天中，适当加入一点对方家乡的方言或外语，可以活跃气氛。例如，遇到四川妹子，可以学一句“川妹子”，遇到东北人，可以学一句“老铁”。

最后，我们来学习一种能让声音听起来更性感的技巧，以及如何保护嗓子。

气泡音是一种让声音听起来更性感的发声技巧。它产生于嗓子部位，听起来带有一种轻微的颗粒感或震动感。例如，用气泡音说：“今天天气不错。”或者在晚上用微信发送语音时使用，能增添魅力。练习时，可以尝试从嗓子发出持续的“啊——”音，并感受其震动。

保护嗓子非常重要。平时不要过度用嗓，避免大声喊叫。注意休息，尽量少抽烟、少喝酒。少吃冷饮和辛辣刺激的食物。多喝热水，这对嗓子有保护作用。约会前，可以使用漱口水保持口气清新。

本节课中我们一起学习了声音的奥秘。我们了解了声音在人际吸引中的重要性，剖析了声音的音高、音势、音长、音质等结构。我们还学习了如何通过抑扬顿挫、肢体语言和不同的发声意识（穿透、抛物线、节奏）来改善表达。同时，掌握了吐字清晰的练习方法、气泡音技巧以及保护嗓子的注意事项。通过持续的练习，你可以有效地改善自己的声音，使其更具吸引力和感染力。
