# 红丸觉醒课程 - 第1课：感谢女性主义？女性主义与红丸理念辨析 🧠

在本节课中，我们将深入探讨女性主义的核心观念，并将其与红丸理念进行对比分析。我们将理解两者根本的不同，并解释为何从红丸视角看，现代女性主义的某些发展反而为部分男性创造了新的机遇。

## 核心理念的差异

上一节我们明确了课程主题，本节中我们来看看女性主义与红丸理念的根本不同。女性主义的出发点，或者你在网络上看到的那些所谓女权言论，其核心观念无一例外是把男人踩在脚下。

或者说是去跟男人寻求一种权利上的平等。因为女性主义的内核是处处要跟一个男人去竞争，说白了就是要跟这个男人竞争出一个输赢。其隐含的公式是：**竞争 = 女人要赢 + 男人要输**，因为竞争一定有输赢。

## 性别对立的根源

所以这也是很多人认为，现在互联网关于两性的论调会出现性别对立的原因。但是，从红丸理念的个人出发点来看，并没有想要去实践女性主义那种所谓的理想抱负，即挑战父权。

## 红丸对女性主义的立场

我不反对女性主义，你可以这样认为。我不反对的建立基础是，我并不想把女性主义打倒。我告诉大家一个最简单的道理：因为在今天的社会里，你之所以能跟许多女人很快地去发生短期关系，今天的女人普遍没有那么传统保守，这是否要归功于女性主义？

因此，我没有必要去打倒女性主义。如果你是一个有选择权的红丸男，这对你是一件有利的事情。这是否是这个道理？

## 辨析不同的“女性主义”

只是说，田园女权是很保守的。我们上一期讲过，田园女权不属于现代女性主义，它其实是女性修正主义，或称女权修正主义。但是，大部分现代女性在一线城市，你之所以在“转盘子”（指同时与多位女性交往）的过程中，能跟这些女人很快发生短期关系，这个功劳要归功于女性主义，要归功于第四次女性主义运动。

## 红丸的包容性与揭露本质

所以，我没有理由去反对女性主义。我所说的反对是指“打倒女性主义”，我并没有理由去把女性主义打倒。因此，红丸理念其实是非常具有包容性的。我们只是探讨并揭露女性主义的本质，没有说要打倒它。

因为其实很多男人并不明白，不理解。如果你是个蓝丸男，还没有接触红丸理念的话，你并不理解女性主义的内核是什么。所以从这一点上来讲，在今天的这个社会，你能够跟许多的女人很快地发生短期关系，这是否得归功于女性主义？

**本节课总结**：本节课我们一起学习了女性主义与红丸核心理念的根本差异。我们分析了女性主义“竞争”内核带来的性别对立，明确了红丸理念并不旨在打倒女性主义，而是揭露其本质。同时，我们认识到现代女性主义的某些发展（如性观念开放）客观上为实践红丸策略的男性创造了条件，并辨析了“田园女权”与此的不同。关键在于理解本质，而非简单对抗。
