## 如何维持“以自我为核心”的状态

我们继续。看一下问题。

苏苏问：“如何调整以自我为核心？感觉很多时候能捕捉到那个状态但不能维持。”

你不能维持的原因，是你的生活跟你以自我出发点的状态有时会产生矛盾或割裂。你可能无法在所有时候都做你自己，以你自己为核心。

但你要知道，你在不同的情况下，你还是在做你自己。你并不是说，今天可能……我举个例子：你今天去参加一个公司领导的聚会。作为公司的一员，在场其他人的职位都比你高。你感觉在那个场合下，没有办法做你自己，没有办法以自我出发点为核心。

但其实不是这样。你要明白的是，你在任何时候其实都是在做你自己，只是你把你自己限定在某一个范围里。你在某些情景下做你自己，它的范围可能是不同的。在那种情景下你也做你自己，但你会限定一个范围。你要找到这个范围跟你自己当中的一种平衡。

你不可能跟你的领导在饭局上，像跟你平时兄弟一样开玩笑，讲一些不分场合的话。那肯定不是这么一回事。

在红丸4.0里，我们会分享一个最重要的东西，叫做 **客观权利**。你要明白，在特定的环境或组织里，你得去说一些什么或做些什么，它会符合一定的条件。如果你不符合这样的条件，那你就把这个范围限定在一定的程度里。你不要去超出那样的范围。

你可以去超出那个范围，可以那么做，没有问题。你可以百分百做你自己。但你要知道，如果你超出了那个范围，它所带来的结果是什么？你能不能去承担这个结果？你能不能去承担这个责任？如果你不能，那你就不要这么做。

这个责任可能就是你会丢掉你的职业，丢掉你的工作。

“柯老师怎么提升个人能量？”

其实你这个问题跟上一个兄弟的问题很像，就是怎么提升个人能量。这两个问题是比较类似的问题。我给大家回答一下这个问题。

很多人会去找状态：有时候我没有状态，有时候我有状态。我有状态的时候，整个人能量会很好、很高；我没有状态的时候，我就特别丧。

**你不要把你自己能量高的好状态，跟能量不高的不好状态，自己去进行比较**。你能量不好的时候，你也得接受你自己的状态。而你每次能量好的时候，你在状态里能量很好的时候，那样的状态越来越多，其实它会慢慢提高你的一个**默认状态**。所以你的这个默认状态很重要。

你平时日常生活当中没有状态的时候，你的这个默认状态就会慢慢一点一点变高。这像什么呢？你在状态里的时候，你能量很好，状态很好，它就像给你套了一个**buff**一样。但是，它会增加你默认状态的下限值，会慢慢慢慢地提高。是这样子的。

而不是说你的两种状态：一个没有状态的时候就是差的，然后有状态的时候就是好的。不是这样的一个逻辑。所以当你知道这一点以后，你要接受你的默认状态。你的默认状态你不能说是不好。很多人以为我的默认状态是不好的，你要去接受它，你的默认状态就是这样子。这就是你的一个默认状态。

但是你的默认状态，它也会慢慢慢慢地整体来说会不断地提高。如果你不接受你的默认状态，你觉得你的默认状态是很差的，那你就会陷入到一种好状态跟不好状态的循环里面。
