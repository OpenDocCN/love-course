# 10、2020『高阶框架打造1.0+2.0』线上视频课：第04课：beta逆袭的核心运行原理以及如何规避成为beta男人

## 概述
在本节课中，我们将探讨吵架中常见的硬框架雷区，以及如何避免成为beta男人。

## 视频场景分析
在一段男女吵架的视频中，双方都表现出硬框架，维护自己的感受和体验。男生虽然能量高，但能量是负的；女生能量低，但能量是正的。男生在场景中犯了以下关键错误：

### 错误一：情绪失控
情绪失控会导致情伤指数下降，做出后悔莫及的事情或造成难以挽回的后果。情绪失控时，框架感会丢失。

### 错误二：暴露性需求感
在吵架时进行亲密接触请求是非常唐突和危险的，会降低自身吸引力。

## 避免成为beta男人的原则
### 原则一：不要动手打人
打人不会带来任何好处，反而会使问题恶化。

### 原则二：不要威胁对方
威胁只会带来嫉恨，对双方造成折磨。

### 原则三：不要在吵架时提出亲热请求
这是低价值、低位的行为，会降低自身吸引力。

### 原则四：不要扩大矛盾
不要对父母、女朋友或重要的人出口不逊或指责谩骂，以免加剧关系恶化。
