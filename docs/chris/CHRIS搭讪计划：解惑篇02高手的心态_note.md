从新手到达人，关键在于心态的转变。大部分新手在搭讪时最害怕的情况就是被拒绝。

## 被拒绝并不可怕：认识“脱敏”

上一节我们提到了对拒绝的恐惧，本节中我们来看看如何应对它。其实拒绝并不可怕。在我们的线下教学中，第一天经常会教学员一个环节，叫做“脱敏”。

**脱敏**的意思是让大脑脱离社交的敏感状态。很多人会对拒绝反应过度敏感，会过度抗拒。甚至因为害怕，在接近目标之前会非常焦虑。脱敏就是让你的头脑去适应搭讪被拒绝的状况，脱离在搭讪时会出现的那种敏感状态。

## 如何实践“脱敏”？ 👀

以下是实践“脱敏”的具体方法：首先观看几段搭讪被拒绝的影片。

（影片内容描述：多次尝试与不同女性搭讪，均遭到简短拒绝或无视，最坏的情况是对方喝斥“你要干嘛？”或“你有事情吗？”，但并未出现肢体冲突等极端情况。）

我们可以看到，在刚才无数的搭讪被拒绝的反应中，最坏的情况也是女生就喝斥了一下。并没有像大家想象当中出现非常恶劣的情况，比如搭讪被打，这些基本是不可能发生的。

## 理性分析：搭讪是低风险高收益的事

但是还是会有兄弟说，即便这样也很害怕。在大街上搭讪被喝斥，会觉得自信心受到打击，会觉得自己在做一件不好的事情。

那么我们换位思考一下。假如今天你不在大街上去搭讪，你换作在你的已知社交圈里，你认为那个更安全的环境里，比如在你的学校里，在你办公室的职场上，在你身边的朋友圈里面，你要去跟一个女生搭讪，它的风险相比在大街上来说，是更高还是更低呢？

**事实是更高的**。因为假如今天你在职场上、在学校里搞砸了一个社交圈的女生，你会抬头不见低头见。这个场面会非常尴尬。她可能会在社交圈里说一些对你不利的话，降低你的社交评价，影响你的名声。

**公式：已知社交圈搭讪风险 > 街头搭讪风险**

但是在大街上搭讪一个女生呢？即便你搞砸了，她讨厌你，但是你有可能再也见不到这个人了。我们从不谈最好的情况，先从最坏的情况分析。如果你在大街上搭讪失败，最坏的情况发生了，它对你原本的生活并不会造成实质性的影响。

而如果你成功了，我们就不说大成功，可能你今天跟这个女生有一个良好的交流，这可以提高你的社交能力和自信心。如果有一个大成功，你就多了一个可以跟女生发展关系的机会。而且大街上的美女是取之不尽的。

**总结：搭讪是一个低风险高收益的事情。** 很多人害怕抗拒，其实都是自己被自己打败了，并没有理性地去分析。它比在已知社交圈认识女生要靠谱得多。前提是，你要掌握搭讪技巧，让它变成可以给你带来结果带来收益的事情。

## 高手心态三大原则 🧘

在让搭讪变成有收益的事情之前，我们先不能害怕，不能抗拒搭讪。理性分析之后，可能还是有人不敢。这里给大家讲几个关于搭讪心态的要点。

### 原则一：减少思维决策的内耗

第一个要点是，不要把能量消耗在思维的决策上。假如今天你看到一个女生，你想搭讪但又害怕上去，你处在这种纠结的状态里，一直在犹豫。这非常消耗你的思维能量，会让你的状态变得越来越差。

**建议是：** 当你看到一个正妹，要把你的思维决策时间变少。也就是说，你看到目标出现后，**决策要快**。要不就上去搭讪，如果决定不搭，就不要再去想她，不要后悔。

```python
# 决策逻辑示例
if see_potential_target():
    make_decision_quickly() # 快速决策：是 或 否
    if decision == "是":
        approach()
    else:
        move_on() # 不再纠结，继续前进
```

大部分新手卡住的原因，就是把能量消耗在了思维决策上。人的思维其实相当消耗能量，所以不要浪费在无限的纠结中。

### 原则二：每一次搭讪都是新的开始

第二个心态原则是，每一次搭讪其实都是一个新的开始，不要把它跟之前的搭讪联系在一起。

很多人会把每一次搭讪跟之前的状况联系在一起。比如今天出门搭了十个女生，前面九个都失败了，就会认为第十个也会失败。这会形成一种惯性思维。

但你要知道，**每一搭都是新的开始**。从女生的角度来讲，你是她今天遇到的第一个陌生人，对她来说不存在前面的九次失败。很多男生会把前面的失败带入新的搭讪情景中。

这就像一个经济学游戏。抛硬币每次正面朝上的概率都是50%，与之前的结果无关。搭讪也一样，虽然随着技巧熟练成功率会提高，但在心态上，一定要避免把之前的失败带入新的搭讪中去。

### 原则三：关注过程，而非结果

第三个心态原则是，搭讪不要把目标看得太大，先从小目标开始。

很多人搭讪压力很大，因为一开始就把既定目标设置得太高，认为上去搭讪就一定要成功，要让她成为女朋友。但事实上并非如此。虽然你需要后面的结果，但在迈出第一步时，你应该把当下的那一步做好，而不是思考后面的结果。

这里有一个禅师的比喻：两个人每天上山砍柴。第一个人觉得整个过程是负担，想着要砍柴、搬柴、烧火，很累。第二个人则把注意力放在当下：上山时欣赏风景，砍柴时专注砍柴，下山时欣赏另一番美景。同样的行为，因心态不同，体验截然不同。

**应用到搭讪：** 当你看到女生的第一刻，应该把注意力放在当下。例如：“这个女生为什么会出现在这里？她为什么穿着这件衣服？她是在等人吗？” 当你的注意力放在更当下的细节时，你的交流就会更自然，更注重过程，从而拥有“流动感”。

## 课程总结 📝

本节课中我们一起学习了搭讪高手必备的三大核心心态原则。

1.  **减少内耗**：快速决策，避免在“去或不去”中无限消耗能量。
2.  **重启心态**：每一次搭讪都是独立的，不将过去的失败带入新的互动。
3.  **关注当下**：注重搭讪过程本身的体验与交流，而非执着于最终结果。

记住，搭讪是一项可以通过练习掌握的低风险社交技能。克服内心的恐惧，调整好心态，是迈向成功的第一步。
