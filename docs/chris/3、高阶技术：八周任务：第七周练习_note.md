# 3、高阶技术：八周任务：第七周练习_note

在本节课中，我们将学习如何将设计好的叙事内容应用到实际对话中，并了解复盘总结的重要性。

## 概述

课程已连续进行两周。此前因故暂停了半个月。观察到大家的打卡提交较少。如果大家积极完成了任务，那么第六周和第五周的内容都是“前提评估”。若之前已准备好叙事材料，本周我们将进入新阶段。

上一节我们介绍了前提评估，本节中我们来看看如何开始实践叙事。

## 第七周核心任务：叙事应用

第七周将开始进行叙事练习。前提是你们需要先完成叙事内容的撰写。我知道杨同学已将材料发给我。在你们撰写完成后，我会进行修改。之后，你们便可以将修改后的叙事练习投入实际使用。

**本周练习的核心是**：将你设计好的叙事内容，我会通过私人微信抽时间进行批阅，指出优点与需要修改之处。我会给出反馈，例如“这里好，这里不好，回去改一下”。实际上，你们写的一些内容初期可能用不到，无需一开始就讲述过多。

## 叙事设计要点

在本周的免费视频中已说明，无需一开始就讲述过多。但有些内容可以预先设计，特别是那些常被问到的问题。

以下是常见的可设计问题列表：
*   你是哪里人？
*   你做什么工作的？
*   你为什么来这里？
*   你和你朋友是怎么认识的？

如果你只是普通地回答，例如“我是一个软件工程师”，这通常不是一个好的回答。明白吗？

## 本周具体练习要求

本周的任务是：将你为这些常见问题设计的叙事，融入到实际对话中，分散在交流中使用。同样需要完成**20组**对话练习。

练习后需要撰写小结。以下是撰写小结的具体要求：
*   记录下对你来说印象最深刻的三个对话组合。
*   将其详细写下来。

## 关于任务进度的提醒

然而，有些人可能还没有完成第六周的任务。许多人的打卡进度可能还停留在前两周。要知道人都有惰性，独自一人时容易懒散，不愿打卡。这无法强制，全靠自觉。

学习任何东西都依靠自觉，就像学英语需要背单词一样。只听老师讲课是学不会的。你需要打卡，需要完成任务。不让你们写小结，写小结的目的是为了**复盘**。不复盘，你的社交直觉就不会增长。

参加过我们线下活动的学员应该很清楚，我们每次都要进行复盘，并把小结写得非常细致。

## 总结

本节课中我们一起学习了第七周的核心任务：将设计好的叙事应用到20组实际对话中，并通过对印象最深刻的三个组合进行复盘小结，来提升社交直觉。这一切都依赖于持续的行动和自觉的练习。
