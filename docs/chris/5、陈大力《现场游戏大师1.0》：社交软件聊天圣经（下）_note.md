## 概述

视频播放时，有时会因为网络或播放器原因导致画面跳转或内容缺失。为了确保能看到完整的教学内容，需要执行一个简单的操作来调整视频显示。

上一节我们介绍了社交聊天的核心心态，本节中我们来看看如何确保学习过程本身不受技术问题干扰。

## 解决视频跳转问题

如果视频播放出现跳转，导致部分内容无法观看，可以采取以下步骤进行调整。

以下是具体的操作指南：

*   **操作**：点击视频播放窗口的**右上角**。
*   **目的**：此操作可以调整视频显示设置，帮助显示出因跳转而未播放的内容。
*   **核心动作**：这个步骤可以总结为一个简单的指令：`点击(视频右上角)`。
*   **重复执行**：如果问题依然存在，可以多次重复此操作，直到视频画面正常显示所有内容。

通过执行以上操作，可以有效解决因技术问题导致的内容缺失，保证学习过程的连贯性。
