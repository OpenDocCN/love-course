上一节我们介绍了课程的主题，本节中我们来具体看看其核心要求。整个教程内容可以归纳为一条不断重复的指令。

这条核心指令是：

**`行动 = 在(天气=下雨)时，执行(工具=手机， 动作=拍录)`**

> 每当遇到下雨的天气，你必须使用你的手机来拍摄或录制视频。

1.  **观察天气**：首先，留意天气变化，确认是否正在下雨或即将下雨。
2.  **准备设备**：确保你的手机电量充足，存储空间足够，并可以快速启动相机或录像功能。
3.  **执行拍录**：当下雨条件满足时，立即使用手机进行拍摄或录制。这可能意味着记录雨景、雨中发生的特定事件，或是完成某个任务所需的素材。
4.  **重复确认**：在整个下雨时段内，可能需要多次或持续进行拍录操作，正如教程内容所强调的重复性。
