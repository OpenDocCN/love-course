# 红丸会员4.0自由蓝图：第三课 - 🧲 吸引力与权利意识的关系

在本节课中，我们将要学习吸引力与权利意识之间的核心关系。我们将探讨主观权利与客观权利的区别，理解它们如何共同塑造我们的现实，并分析这种认知在两性互动中的具体体现。

## 主观权利与客观权利

上一节我们介绍了课程的整体框架，本节中我们来看看权利意识的具体构成。权利分为主观权利和客观权利。

*   **主观权利**：来源于想象，没有条件。
*   **客观权利**：来源于事实，有条件。

我们所经历的一切，是借由主观想象来产生客观事实，再由客观事实来保证主观想象。如此循环往复，组成了每个人眼前的现实。

## 权利失衡与现实的扭曲

当主观想象的权力大于客观事实的权力时，现实就会扭曲，以至于偏离真相。

例如，一个女人主观想象自己是一个十分妹，但客观事实是她是一个又胖又丑、满脸麻子的两分妹。这时，她就会扭曲自己的现实。她认为自己有十分妹的权利，就会去拒绝比自己优秀的男人，甚至认为所有男人都配不上自己。但凡是一个红外男，都会知道她的这个现实是被扭曲的。

男人其实也一样。如果他主观想象自己是一个十分男，但客观现实他却是一个情商很低、没有谈吐的男人，那么在周围旁人的眼中，他就是一个怪人。

## 平等主义的影响

导致人们偏离真相、现实被扭曲很大程度的原因，正是平等主义的普世价值观。因为平等主义告诉人们，你只需要做你自己你就是最完美的，导致他们即便忽略了自身的缺陷，也认为自己是最完美的，进而成为一种精神色盲的状态。

现今我们的社会中，现实被扭曲的女人占大多数，而男人是少数。或者说，男人的现实是往另一个方向扭曲，让大部分的男人都会去低估自身的价值。这个原因在于，两性市场的天秤早已失衡。大部分的女人都认为自己是更有价值的，她们都会去高估自己的价值。

例如，许多的两分妹会认为自己是十分妹，所以她看很多两到六分的男人，在她们的眼中就都是普性男。

## 群体想象与客观事实

当许许多多的女人来源于想象的主观权利越来越多的时候，某种程度上，它就会影响来源于事实的客观权利。因为所有的男人与女人都陷入到这种偏离真相、偏离事实的想象当中不能自拔。而这种想象一旦成为一种群体想象的时候，那么它就会影响客观事实。

比如，女权就是女性群体幻想出来的主观权利，但它最终不符合客观权利的规则，不符合客观权利的条件，那么终有一天它将会崩溃，就像韩国和日本一样。

## Game中的权利意识

在Game当中，主观权利也包括：**“我觉得自己内心是配得上女人的”**、**“我觉得自己是值得拥有这一切的”**。

而客观权利指的并不是大部分人所认为的硬价值，例如名牌车表。这些只是消费主义制造出来的幻想。客观权利是：**你是否拥有对客观事物的支配权**。而这种权利的形式通常是以组织为单位的。

以下是举例说明：
*   学校里的老师相较于学生来说，他就有更大的支配权。学生只能支配他的书本，而老师可以支配学生。这就是老师的客观权利。
*   学生作为被支配者，会把这种组织的权利结构带入到两性的权利结构当中，把高高在上的老师想象为高高在上的男人。所以女人，受她们的慕强心理驱使，会很容易喜欢上老师这样的角色。这就是为什么校园里面师生恋非常普遍的原因。

但是，如果这位受女学生敬仰的老师，在亲密关系当中，他作为男朋友的角色，他把自己套进了“蓝丸男、龟男、舔狗、软男”的这个框架里面去，进而放弃了他自己的主观权利，那么这位老师对女学生的吸引力也会瞬间失去。

## 权利循环与现实的改变

所以说，主观权利与客观权利，是一种循环往复的关系。它将组成我们眼前所看到的现实，也会改变我们眼前所看到的现实。

我们再举个例子。如果你通过学Game改变了主观权利，某种程度上也会影响你的客观事实。例如，你撩到了一个远超出自己SMV的女人，并且进入到了一段关系当中。

我们做这样一个假设：你是一个上班族，这个女人她是一个女网红。女网红身边接触的男人数量非常多，虽然说大部分都是龟男舔狗，但一定比你更优秀的男人数量是更多的。那么当你在跟这个女人接触的现实当中，她身边出现这样的男人的时候，很大概率下你会有被劈腿的风险。

那么你身为一个男人就会想：既然如此，我能不能让这个女人放弃她作为网红的这个工作呢？但是，你得思考一个问题：你通过学Game撩到了一个远超出自己SMV的女人，但是进入到了这个关系当中，你这样去要求她，但她凭什么要为了你放弃？

## 客观权利的条件

所以说，你虽然有提出这样权利的资格，但是你要符合客观权利，它是有一定条件的。如果你满足了某些条件的话，这个女人也许她会为了你去放弃这份工作。

以下是条件举例，在这些条件当中物质底线就是条件之一：
*   你能否提供足够的经济保障，让她无需工作也能维持甚至提升生活水平。
*   你能否在其他方面（如情绪价值、社会地位）提供远超她当前工作所能带来的价值。

当你无法满足客观权利的条件时，你就无法去改变这个现实，甚至在未来你还会承担相应的后果，比如如果你要坚持这段关系的话。

## 面对现实与真相

所以说到了我们的这个现代社会，即便大部分人认为男女是平等，两性之间是不存在权利结构，觉得只要有这个爱情就可以了，但是客观的权利依然存在是一个事实。而我觉得，这是每个红外男所要面对的一个现实，所要面对的一个真相。
