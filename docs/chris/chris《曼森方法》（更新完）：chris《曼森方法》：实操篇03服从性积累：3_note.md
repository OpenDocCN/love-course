## 什么是服从性积累？🎯

上一节我们介绍了曼森模型的基础框架，本节中我们来看看“服从性积累”的具体含义。

服从性积累是一个渐进的过程。它指的是通过一系列逐步升级的要求或测试，让对方（通常是女性）逐渐增加对你的顺从程度。

如果女人不愿意出来跟你约会，那么她不可能愿意跟你接吻。如果女人不愿意跟你接吻，那么她又怎么会愿意跟你上床。所以在约会过程中，测试女人的服从性，就是测试她对你的兴趣以及接纳程度最有效的方式。这个过程叫做服从性测试。

但是今天我们要讲的东西，它要比服从性测试更进一步。这个概念就叫服从性积累。如果说服从性测试更趋近于是一种试探或者带领的话，那么服从性积累就更像是在训练一只小狗的时候，你去主动创造一些训练的行为。比如握手，比如让她捡球，比如让她蹲下。

所以其实从这个关系的角度来讲，女人从刚开始认识你的时候，到最后和你上床，跟你确立关系，它其实就是一个服从性积累的过程。而这样的一个方式，放在长期关系里面，同样也是适用的。
