## 高阶框架的定义
**公式**：高阶框架 = 硬框架 + 软框架

### 硬框架与软框架
以下是软硬框架的组成和作用：

- **硬框架**：相当于基柱的脊椎，是支撑框架的基础，代表个人原则和立场。
- **软框架**：相当于基柱的椎间膀，是框架的灵活性，代表对他人利益的考虑。

### 硬框架与软框架的关系
- 硬框架不等于好框架，软框架也不等于坏框架。
- 好的框架需要软硬结合，根据实际情况灵活调整。

## 硬框架与软框架的呈现
### 硬框架的呈现
- 当与他人发生矛盾，只考虑自身利益时，呈现硬框架。

### 软框架的呈现
- 当考虑他人利益时，呈现软框架。

## 吸引力和可得性
### 吸引力和可得性的关系
- 吸引力和可得性是影响关系的两个关键因素。
- 吸引力和可得性需要保持平衡。

### 吸引力和可得性的调整
- 通过推拉来调整吸引力和可得性，使关系保持在健康状态。
