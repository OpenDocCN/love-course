在本节课中，我们将要学习一个核心概念——“谦逊的力量”。这是Beta智慧中至关重要的一个部分。我们将探讨谦逊的本质、它如何影响个人成长，以及如何在保持主见的同时实践谦逊。

***

真相一旦入眼，便无法视而不见。欢迎来到红丸会员3.0的第8节课。本节课将讲解一个非常核心的概念：谦逊的力量。这也是Beta智慧中一个重要的概念。

许多历史上的伟人都非常注重谦逊这一点。例如，莎士比亚曾说过：“一个骄傲的人，结果总是在骄傲里毁灭了自己。”著名的古希腊哲学家苏格拉底也曾说：“我最大的智慧，就是知道我一无所知。”除了知道自己一无所知，只有神才是全知全能的。

《易经》中也讲到：“天道亏盈而益谦，地道变盈而流谦，鬼神害盈而福谦，人道恶盈而好谦。”这里的“盈”，指的是一种满的状态。

“天道亏盈而益谦”是指上天的物理法则：事物达到盈满之后必然开始减少。例如，太阳升到最亮之后便会落下，光芒逐渐减弱；月亮满月之后，便会逐渐变成月牙。相反，如果事物不处于盈满状态，太阳在最微弱的光芒里会逐渐变强，月亮变成月牙后会逐渐变成满月。

“地道变盈而流谦”是指大地的法则：当一种事物达到盈满时就要发生变化。例如，水永远往低处流，当它到达高峰后便会向低处流去。高山会逐渐变为海底，丘陵会变为洼地。

“鬼神害盈而福谦”是指鬼神的法则：损害骄盈的人，使其退去锋芒，而赐福给谦逊的人。

“人道恶盈而好谦”是指人间的法则：人们都讨厌骄傲自满的人，喜欢谦逊之人。人的本性都不喜欢一个人非常骄傲自大、自满。

因此，谦逊本质上是一种对自己永不自满、永远进取的精神。谦逊如同大海，已然宽广博大，却位于百川之下而容纳百川。

很多兄弟在学习红丸理论后，了解到阿尔法男性的一个观念是要“主导”。在变得具有主导性之后，他们便想让自己变得更有主见。因为贝塔男性过去的人生往往是被支配的——被女人支配，被周围的人支配——从而变得没有主见。

但是，这种想法一旦走向极端，主见就会变成固执己见，听不进别人的意见。可能有些意见对自己是有帮助的，但有时为了维护自己的尊严，也会固执己见。这其实就失去了谦逊，变得骄傲、自大，甚至高傲自满。这是许多男人本性中的高傲和自满。

人往往就毁于高傲和自满之中。因此，在不失去自己主见的同时，又可以运用这种Beta智慧，通过谦逊让自己不断提升，不让自己进入盈满的状态而失去福德。

古人常说，《易经》已经把天地鬼神的世界运行法则说得很清楚了。当一个人觉得自己已经满了，开始高傲、骄傲的时候，他就会逐渐失去自己的福德，也就是他过往积累的成功会慢慢失去，最终毁于一旦。

在现实中，我们也可以看到很多这样的例子。就拿我们这个行业来讲，我们永远对自己处于一种谦逊的状态，永远不觉得自己是最厉害的，从来不会标榜自己就是业内第一、中国第一、世界第一这样级别的导师。因为我们永远觉得自己是需要学习的。当你自己已经开始盈满，你就会失去自己的福德，最终逐渐毁于一旦。

因此，无论你处于任何人生阶段、任何领域或行业，始终都要保持这样的心态。这其实就是一种自省的能力，也是李小龙所讲的“空杯心态”：永远要把自己过往的成见抛开，认为自己是一个永远不满、需要学习的人。

通过《易经》的古话我们了解到，当你始终觉得自己是不满的时候，天地鬼神以及人道都会认为你是谦逊的，才会赐福给你，让你从不满走向满意。而当你到达满意状态，就会逐渐开始跌落，开始毁于一旦。

那么，如何正确地达到这种谦逊呢？

以下是实践谦逊的几个关键点：

**第一点，拥有自省的能力。** 你要客观地看待自己。不是说你过去在某些方面取得了一些成功，你就看不清自己了，认为自己所说的一切都对，盲目迷信自己所有的决定都是正确的。这时你就失去了自省的能力。虽然男人有主见，可能在过往的选择中不断取得成功，但一旦陷入骄傲自满的状态，他做出的决策往往就是错误的。

在很多金融行业里，我们都可以看到这样的例子。有些币圈大佬或投资高手赚了很多钱以后，最终毁于一旦，在高额杠杆中毁灭。就是因为他太骄傲自满，一直觉得自己过去的选择正确，固执己见，最后毁在了自己的骄傲中。这与上述古人告诉我们的道理是一样的。

所以你要自省。不管你取得多大的成功，首先都要觉得自己有一种谦逊，认为自己的成功靠运气，并非因为自己最厉害才成功。你永远觉得自己是需要学习的、对自己不满的，这样才不会将以前成功所积累的福德亏损掉。因此，你需要拥有自省的能力。

**第二点，谦逊能让你获得他人的意见。** 我们讲“人道”，所有人都喜欢谦逊之人，讨厌骄傲之人。将这个例子放在职场上，你可以深刻体会到这一点。一个人在职场求职过程中，你会发现周围的同事、领导不愿意无偿帮助你。这时，你的品德就显得很重要。

如果你在职场为人很骄傲，你会发现所有人都不愿意指点你、告诉你，甚至不愿意与你合作。但如果你显得很谦逊，不管对上级还是下级都很谦虚，那么其他人其实是愿意教你一些东西的。因此，你要变得谦逊而不是自满。

以我们自己为例，如果了解搭讪大师比较久的兄弟一定知道，我们从最开始成立到现在，产品历经了很多阶段的变化。从一开始推出“转变30天计划”，到后面逐渐推出“曼僧”、“红丸”，再到现在的“红丸3.0”。很多兄弟都知道，我们是业内最专注于做产品、产品口碑最好的团队。

但为什么我们可以把产品不断精进、做到最好？因为我们在前进的过程中，一直在不断听取用户的意见和反馈。虽然我们在做产品时有自己的坚持，但并非认为产品已经最好，完全不听取用户意见。那种想法就是已经盈满了，失去了谦逊，这时是做不出好产品的。因为产品本质上是以人为本，我们要解决的是用户的需求，而不是去展现自我的一种厉害。

所以，我觉得这也是谦逊与盈满、骄傲的一个底层逻辑区别。前者是自我意识的无限延伸，而后者其实是集体意识的无限延伸。你要知道，你做这个产品是为了让你自我的意识无限延伸，还是为了让更多人获益、让集体意识无限延伸？这基于你做一件事时的出发点：是为了让自己更获益，还是让更多人更获益？显然，我们一直以来都贯彻后者的想法。这才是让我们团队可以坚持到现在、走得更远的原因。

包括我自己最早在国内开始讲红丸以后，也有许多人开始效仿，效仿我们的产品、内容。当然，红丸也并非我一人完全原创，它的理论基础来自于西方著名两性专家Rolo Tomasi所写的《理想男性》一书。但在这个过程中，我们不断根据中国用户，听取他们的意见和想法，不断原创出符合中国人的理论基础，才能让我们把这个事情做得更久。

而很多人只是一味觉得“红丸这个东西厉害，我听了以后就天下无敌了”。但你们可以看到，对我们的这些效仿者来讲，他们都走不长远，原因就在于此。因此，谦逊并不等同于没有主见，而是你有自己的坚持，但也会听取他人的意见。我觉得这是非常重要的。

**第三点，一个男人内在的强大，是对外在的接纳和包容。** 我刚才也讲到了，真正的强大并不是你自我意识的无限延伸，而是一种集体意识的无限延伸。这就是为什么红丸一直强调男性是一个命运共同体。我觉得真正的强大，是你可以包容不同的观点，接受并客观看待不同的观点，并且求同存异。这也是Beta智慧，是一种大智慧。

**第四点，延伸到合作框架。** 一个强大的框架并非对抗，并非零和游戏（不是你死我活），而是可以在包容其他框架的同时，最终和其他框架达成合作。而合作会变成共赢。因此，这也是最终很核心的一点。我觉得大家可以慢慢体会这一点，不管是在人际交往、创业还是与女人互动的过程中，你会发现这一点都是最终的大智慧。
