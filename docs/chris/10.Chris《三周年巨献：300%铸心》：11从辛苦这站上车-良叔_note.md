# Chris《三周年巨献：300%铸心》：11从辛苦这站上车-良叔

## 概述
在本节课中，我们将学习如何克服敏感，提升个人心力，从而成为一个强大的男人。

## 承受
**定义**：承受是指学会接受，改变你能改变的，接纳你不能改变的。

**公式**：承受 = 接受 + 改变 + 接纳

**解释**：敏感的男人往往心理承受能力较低，对外界刺激反应过度。通过后天适当的引导，可以提升承受能力。

## 转变
**定义**：从内心的小黑屋走出来，多接触真实世界。

**公式**：转变 = 内心转变 + 外部接触

**解释**：敏感的男人往往喜欢将自己封闭在内心世界，无法感染他人。通过多接触外界，可以拓宽视野，提升个人能力。

## 聚焦当下
**定义**：不要总惦记过去，也不要总思想未来，而是要沉浸于当下。

**公式**：聚焦当下 = 忽略过去 + 忽略未来 + 沉浸当下

**解释**：敏感的男人往往容易陷入对过去的回忆或对未来的担忧中。聚焦当下，才能更好地把握现在。

## 舒适与辛苦的转换
**定义**：舒适与辛苦是相互转换的。

**公式**：舒适 → 辛苦 → 强大

**解释**：人生就像一辆永不回头的列车，起始站是舒适，终点站是辛苦。通过经历辛苦，可以变得强大。

## 心力
**定义**：心力是一种综合的心智能力，能让你识别所处的环境，判断应该采用什么样的策略去做这个事情，以及怎么去主导这个行为。

**公式**：心力 = 心识 + 心智能力

**解释**：心力可以通过经历挫折和困难来提升，从而让你变得更强大。

## 总结
本节课中，我们学习了如何克服敏感，提升个人心力，从而成为一个强大的男人。希望你在2022年能够如愿以偿，实现自己的目标。
