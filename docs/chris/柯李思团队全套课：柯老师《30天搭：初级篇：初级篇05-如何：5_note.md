# 柯李思团队全套课：柯老师《30天搭：初级篇：初级篇05-如何：5_note

在本节课中，我们将要学习搭讪过程中一个不可避免的环节——如何应对“打枪”（即被拒绝）。我们将重点探讨面对拒绝时应有的心态，并介绍一种实用的心理应对技巧。

## 概述：打枪是常态

再厉害的搭讪高手都会被打枪。打枪本身并不重要，重要的是你在面对打枪时的心态。

上一节我们介绍了搭讪的基本开场，本节中我们来看看当开场没有得到积极回应时，我们应该怎么办。

## 心态调整：从失败中学习

通过观察搭讪实例可以看到，即使经验丰富者也会频繁遭遇拒绝。关键在于，不能因为被打枪而感到丢脸或失去面子。相反，应将每次打枪视为一次练习和学习的机会。

当你敢于直面失败，并觉得它并没有什么大不了的时候，你就会慢慢走向成功。但如果你一开始就被失败打倒，那么你可能会离成功越来越远。

## 核心技巧：积极幻框法

对于新手而言，打枪可能会严重挫伤自信心。此时，可以运用我们在基础材料中教授的一种应对方法：**积极幻框法**。

积极幻框法是一个成功之母。它的核心公式并非指失败多次后就会成功，而是：

**敢于直面失败 + 认为失败并无特别 = 逐步走向成功**

这个框架帮助你重新定义“打枪”的经历，将其从“负面打击”转化为“中性甚至积极的学习过程”。

以下是应用积极幻框法的几个要点：

*   **接受常态**：认识到拒绝是搭讪过程中的普遍现象。
*   **剥离个人化**：对方的拒绝通常与你个人价值无关，可能源于她当时的心情、环境或已有伴侣。
*   **关注过程**：将注意力放在“我敢于行动”本身上，而非单一的结果。
*   **快速复盘**：简单回顾互动，思考是否有可以微调的地方，然后迅速放下，继续下一次尝试。

## 总结

本节课中我们一起学习了如何应对搭讪时的打枪。我们明确了打枪是常态，无需因此感到挫败。核心在于心态的调整，并通过**积极幻框法**将失败经历重新定义为成长路上的普通一步。记住，成功的搭讪者并非从不被打枪，而是从不被枪打倒。
