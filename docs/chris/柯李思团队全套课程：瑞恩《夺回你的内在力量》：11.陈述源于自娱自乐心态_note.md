# 柯李思团队全套课程：瑞恩《夺回你的内在力量》：11. 陈述源于自娱自乐心态 🎭

在本节课中，我们将要学习如何通过培养“自娱自乐”的心态来提升与异性沟通的吸引力。核心在于学会以自我为中心的幽默感进行交流，而非刻意取悦对方。

## 概述

自娱自乐是人际交往，特别是在与异性互动中，一个非常核心且必要的元素。它本质上是一个人的幽默感体现。许多初学者并不理解什么是自娱自乐，也不知道如何有趣地与女生沟通，从而导致互动显得生硬、无趣。

## 自娱自乐的核心定义

上一节我们概述了自娱自乐的重要性，本节中我们来看看它的具体含义。

自娱自乐真正的意思是：在与任何人沟通时，你所说的话题、内容及表达方式，全部源于你内心真正觉得好笑、有趣的感受。简单来说，**你开的玩笑和说的话，都应围绕着自己觉得好玩的事物，而不是为了让对方开心或觉得好笑**。

因此，你的每一个行为、每一句话、输出的每一个内容，都必须以**自己为核心**，而不是以女生为核心。

## 常见错误心态与转变

以下是许多初学者常犯的错误心态。

大多数初学者在与女生互动时感到无话可说，根本原因在于他们的注意力、心态和思维完全围绕着对方。他们在揣测女生的笑点和兴趣，从而过滤掉了自己原创的、觉得好玩的内容，导致最终无内容可输出。

然而，女生真正想要感受到的，恰恰是你最原始、原创的幽默感。自娱自乐的模式就是：**所有聊天话题、内容和讲话方式，都源于我认为最好笑、能把我自己逗乐的东西**。这个模式让你完全围绕自己的幽默感进行，同时让女生感受到它。

需要明确一点：并非你输出的所有内容，女生都会觉得好笑。

## 从错误中学习：个人经验分享

我曾经也陷入过试图聊“女生感兴趣话题”的思维误区。这种心态导致我无话可聊，甚至让自己变得尴尬，因为我不知道她想听什么。

当你试图猜测对方想听什么时，你整个人的说话方式、行为，乃至浅层沟通（如眼神、语气、流畅度）都会让女生感到尴尬。你自己可能并非一个尴尬的人，但这种心态导致了尴尬的结果。

后来我发现，当我所有的话题、内容甚至表达方式，都源于让我自己觉得好笑的东西时，虽然女生不会觉得所有内容都好笑，但其中一大部分确实能引发共鸣。这恰恰给了你最好的反馈。

## 建立反馈点（Reference Point）

当一个女生对你的自娱自乐内容给出真实反应时，你就能获得一个“反馈点”。

通过这个反馈点，你能了解到女生大概对什么样的内容有情绪反应，对什么样的内容毫无兴趣。**只有通过实战，释放自己的自娱自乐，你才能知道自己的哪部分原创内容能获得女生的良好反应**。

## 实战案例解析

以下是一个经典的自娱自乐沟通案例。

```
男：我感觉你可能比我要大一点。
女：你也大一点。
男：我大概19。
女：你也可能像19岁。
男：我像多大？
女：前面也得25。
男：我就是19。
女：多大？
男：23，好吧，比我大4岁。
女：不可能比你大，真的。
男：我19岁，怎么可以比你大19岁。
女：说话的语气也不像19岁。

```

在这个实战中，我从开场到收号，完全以自娱自乐的方式沟通。我（一个近30岁的人）对女生说“我19岁”，并完全进入角色，把她当作一个小女孩来逗乐。这种方式让我自己觉得非常有趣且投入。

对于女生而言，她处于半信半疑的状态。她看到我很认真（忍住不笑），且对整个对话非常投入。因为我的状态和情绪都很好，这种自娱自乐的模式成功破冰，并将女生带入良好的互动状态中。

即使她心里知道我不是19岁，但她也会因为感受到我的乐趣而愿意继续这段沟通。

## 自娱自乐与正经对话的平衡

在沟通中，你不能100%的时间都在自娱自乐。

如果你全程都在搞笑和幽默，女生会觉得你不正经，不会把你当回事，事后可能就把你忘了。因此，沟通需要在正经对话和自娱自乐之间来回切换。

同样，全程过于正经也会让女生觉得你严肃、无聊、无趣。**关键在于两者间的灵活切换**：该幽默时幽默（源于自己认为搞笑的内容），该严肃时严肃（如讨论私人信息、建立约会、收联系方式等）。

这样，女生才会觉得你既有趣，又是一个有正常逻辑思维、可以交往的正常人。

## 自娱乐乐的进阶应用：化解敏感话题

自娱自乐在社交中作用巨大，不仅能破冰，还能化解尴尬场面，甚至是尴尬的性话题。

当你想与女生将聊天内容从普通话题升级到更亲密甚至带点“猥琐”的话题时，最轻松简单的代入方式就是加入幽默感。**以自娱自乐的方式聊性话题，能使话题变得不那么敏感和尴尬，更容易被接受**。

因此，自娱自乐是整个与女生交流沟通过程中不可缺少的元素。缺乏它，互动将寸步难行。

## 核心原则：为自己而说

我在与女生沟通时，甚至会冒险说出一些比较极端或冒险的话，可能女生并不觉得好笑，甚至有点讨厌。但我始终坚持一个原则：**我说的每一句话都是为自己而说，输出的每一个内容都源于自己的幽默**。

她可以觉得好笑，也可以觉得不好笑。如果她觉得不好笑，我立即调整，输出更多我觉得好笑的内容。

你的整体思维必须从“**我如何让这段对话对她有趣**”转变为“**我如何让这段沟通首先让我自己感兴趣**”。如果你自己对对话都不感兴趣，她更不会感兴趣，因为这会影响你的情绪和状态。

## 关于状态与情绪的深层影响

我们之前的课程提到过状态和情绪。影响状态情绪的很大一部分因素，就是有没有自娱自乐。

只有当你自己在互动中找到乐趣，你才能持续输出，女生也才能感受到你的乐趣，并被你的情绪所感染。

## 纠正错误认知：幽默不是背套路

很多新手在学习自娱自乐时有一个错误认知：认为自娱自乐和幽默就是去背笑话、记段子、学套路。

这完全错误。原因在于：如果你遇到一个脱口秀演员，他跟你开了一个玩笑把你逗乐，你只会觉得这个玩笑好笑，但不会觉得这个人有幽默感，你会认为这只是个“好套路”。

同理，如果你第一次接触女生时用一个套路把她逗乐，她对你并不会有强烈的吸引力。但如果你看一个脱口秀演员表演两小时，他持续输出原创的搞笑内容，结束后你会觉得这个人本身非常有幽默感和魅力。

与女生相处一模一样。让她感受到你有幽默感、有原创力、能自娱自乐的，是在一段时间内（比如搭讪聊天的一晚，或一次约会），你在她预料之外的时刻，**持续输出自己原创的幽默感**。她才会感觉到你这个人是有魅力的。

**幽默感 = 持续输出原创内容**，而不是使用几个套路来展现根本不存在的自娱自乐。

## 心态的彻底转变

因此，在学习与实战中，你必须彻底转变心态和逻辑思维。

你做的一切、说的一切，核心目的不是为了让这个女生开心。**她不是中心，你自己才是这段沟通的所有中心和核心点**。

你说的每一句话、做的每一件事，必须源于让自己觉得有趣、搞笑、好玩。因为只有这样，当你在互动中遇到困扰、障碍或女生的阻碍时，你**自己原创的自娱自乐能力**才能帮助你通关、推进，并最终达到你想要的结果。
