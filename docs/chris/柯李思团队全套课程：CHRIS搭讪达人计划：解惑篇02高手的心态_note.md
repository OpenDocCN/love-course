## 克服对“拒绝”的恐惧

从新手到达人，心态是第一步。大部分新手在搭讪时最害怕的情况就是被拒绝。

其实拒绝并不可怕。在我们的线下教学中，第一天经常会教学员一个环节，叫做“脱敏”。脱敏的意思是让大脑脱离对社交拒绝的过度敏感状态。很多人会对“被拒绝”反应过度，甚至因为害怕而在接近目标前感到非常焦虑。脱敏就是让你的头脑去适应搭讪可能被拒绝的状况，脱离这种敏感状态。

那么，脱敏具体要怎么做呢？

首先，我们通过观看几段搭讪被拒绝的影片来建立认知。影片展示了各种被拒绝的反应，例如女生简单回应“有事”、“不用谢谢”或直接走开。我们可以看到，在最坏的情况下，女生也只是喝斥一句“你要干嘛”，并没有出现想象中非常恶劣的情况，例如被打。这基本是不可能发生的。

但是，仍然会有学员表示，即便知道最坏情况不过如此，他们依然害怕。他们担心在大街上搭讪被喝斥会打击自信心，会觉得自己在做一件不好的事情。

## 理性分析：搭讪是低风险高收益的行为

为了解决这种恐惧，我们需要换位思考。假如你今天不是在大街上搭讪，而是在你认为更安全的社交圈里，比如学校、办公室或朋友圈，去认识一个女生。相比在大街上搭讪，哪个风险更高？

事实上，在已知社交圈里认识女生的风险更高。因为如果你在职场或学校里搞砸了，你会和对方抬头不见低头见，场面会非常尴尬。对方甚至可能传播对你不利的言论，降低你的社交评价和价值，影响你的名声。

**因此，在已知社交圈里认识女生的风险更高。**

但是，在大街上搭讪一个女生，即便你搞砸了，对方很讨厌你，你也有可能再也见不到这个人了。我们先从最坏的情况分析：即便她非常讨厌你，甚至给了你一巴掌，你第二天还见得到这个人吗？你见不到。

**搭讪公式：风险 = 后续影响的概率 × 影响程度**

在街头搭讪场景中，“后续影响的概率”极低，因此总体风险很低。

相比之下，如果你成功了，收益却很高。我们不说最大的成功，即使只是进行一次良好的交流，也能提高你的社交能力和自信心，让下一次搭讪变得更好。如果你有一个好的收尾并成功邀约，你就多了一个发展关系的机会。而且，大街上的美女是取之不尽的。

因此，搭讪其实是一个**低风险、高收益**的事情。很多人害怕和抗拒，其实是被自己非理性的恐惧打败了，并没有进行理性分析。相比在已知社交圈认识女生或通过朋友介绍，掌握街头搭讪技能要靠谱得多。

## 搭讪时必备的三个核心心态原则

在让搭讪变成有收益的事情之前，我们必须先不害怕、不抗拒搭讪。理性分析后，如果还是不敢，该怎么办？

以下是关于搭讪心态的三个要点。

### 原则一：减少思维决策的内耗

第一个要点是，不要把能量消耗在思维的决策上。今天你看到一个女生，想搭讪但又害怕上去，如果一直处在这种纠结和犹豫的状态里，会非常消耗你的思维能量，让你的状态越来越差。

**建议：** 当你看到一个正妹出现后，要减少思维决策的时间。你要不就上去搭讪，如果决定不搭，就不要再去想它。不要后悔“我刚才没搭”，并一直纠结于此。大部分新手外出搭讪时，常常卡在犹豫不决的状态，逛了两三个小时都没有出手，就是因为把能量消耗在了思维决策上。

**代码式原则：**
```
if (看到心仪目标):
    决策 = 快速判断(‘上前’ 或 ‘放弃’)
    if 决策 == ‘上前’:
        执行搭讪()
    else:
        彻底放下，不再回想()
避免状态 = ‘持续纠结’
```

所以，当你看到一个女生，你的决策反应一定要快。只要做“是”或“否”的决定，不要在这个中间态中无限消耗你的能量。

### 原则二：每一次搭讪都是新的开始

第二个心态原则是，每一次搭讪都是一个新的开始，不要把它跟之前的搭讪联系在一起。

很多人会把每一次搭讪和之前的状况联系起来。例如，今天出门搭了十个女生，前面九个都失败了，他就会认为第十个也会失败，因为“刚才的女生拒绝了我”。这会形成一种惯性的消极思维。

但你要知道，每一搭都是新的开始。从女生的角度来说，你是她今天遇到的第一个陌生人，对她而言，不存在所谓的“前面的九次失败”。很多男生会把之前的失败带入新的搭讪情景中。

这就像一个经济学游戏。很多人赌博时会凭以前的运气判断后面的概率。但事实上，就像抛硬币游戏，每次正面向上的概率都是50%，与之前的结果没有联系。

**公式：P(成功|第N次) ≠ f(前N-1次结果)**

搭讪也是一样。虽然随着你的技巧越来越熟练，反应能力越来越强，你会越来越自然平和，成功率会逐渐提高。但在心态上，你一定要避免把之前的失败带入到新的搭讪当中。

### 原则三：关注过程，而非结果

第三个心态原则是，搭讪时不要把目标看得太大，要先从小目标开始。

很多人搭讪压力大，是因为一开始就把既定目标设置得太高。他们认为上去搭讪就一定要成功，要让对方成为女朋友或有一个好的发展，这才算成功。但事实并非如此。

虽然你需要最终的结果，但在迈出第一步时，你应该把当下那一步做好，而不是一直思考后面的结果。

一个禅师的故事令我印象深刻，这里可以引荐一下：有两个人每天都要上山砍柴。第一个人觉得整个过程很累，想着“要爬很高的山、砍柴、搬下来、烧火”，每天出门都肩负重担。第二个人心态则完全不同，他觉得上山就是上山，会把注意力放在上山路途的风景上。他一路欣赏美景，过程轻松欢快。到了山顶砍柴，他也觉得轻松快乐。背着柴下山时，他依然欣赏风景，感觉不到重担。

同样的行为，截然不同的心态，带来的情绪体验也完全不同。

**核心思想：注意力焦点模型**
```
低压力心态：注意力集中在 -> [当下环境] -> [自身行为] -> [互动过程]
高压力心态：注意力集中在 -> [最终结果] -> [可能失败] -> [自我评价]
```

如果你搭讪时也想成是一个以结果为导向的事情，压力就会非常大。我们不是不需要结果，而是说在做每一步时，要把当下所做的事情做好。

以搭讪举例，当你看到女生的第一刻，你想去搭讪她，就应该把注意力放在当下：这个女生为什么出现在这里？她为什么穿着这身衣服？她是在等人吗？当你把注意力放在更当下的细节时，你的交流会更自然，更注重过程，也会拥有我们在课程中讲到的“流动感”。

所以，第三个心态原则也非常重要：把注意力放在当下，而不是结果上。

## 课程总结

本节课中，我们一起学习了搭讪新手必须掌握的三个核心心态原则。

首先，我们通过理性分析，明确了街头搭讪是**低风险、高收益**的行为，帮助你克服对拒绝的非理性恐惧。

接着，我们深入探讨了三个关键心态：
1.  **减少决策内耗**：快速决定，避免在“去与不去”之间持续消耗能量。
2.  **每一次都是新的开始**：不将过往失败的经历带入新的互动中。
3.  **关注过程而非结果**：将注意力放在当下的互动细节上，而非遥远的最终目标。

记住，强大的心态是成功搭讪的基石。掌握这些原则，能让你在行动时更自信、更从容。我们下一节课再见。
