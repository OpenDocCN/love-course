# 柯李思团队全套课程：CHRIS搭讪达人计划：解惑篇08：废物测试（彩蛋）🎁

在本节课中，我们将学习几种在搭讪现场应对“废物测试”的实用小技巧。这些技巧以娱乐感为基础，旨在帮助你轻松化解互动中的挑战。

上一节我们介绍了应对废物测试的三种传统方式。本节中，我们来看看两种更具娱乐性的现场应对技巧。

## 技巧一：模仿对方讲话 🗣️

这种技巧的核心是模仿女生讲话的语气和内容，制造一种轻松、戏谑的氛围，让对方感到无奈又好笑。

以下是具体操作方式：
*   **举例**：当女生说“你就是个小朋友对吧”或“我妈肯定不喜欢你这样的”。
*   **回应方式**：你可以模仿她的口气说“你就是个小朋友对吧”或“我妈肯定不喜欢你这样的”。
*   **效果**：这种方式加入了娱乐感，能有效破解测试，但仅适用于现场互动。

## 技巧二：假装听不见 🙉

这种技巧通过幽默地转移话题或制造情境，来忽略测试本身，从而化解压力。

以下是具体操作方式：
*   **举例**：当女生提出测试时。
*   **回应方式**：你可以说“啊？你在说什么？”或“哎呀，好像信号有点不太好，风太大了我听不见”。
*   **效果**：这也是一种以娱乐感为基础的、好用且有效的现场应对方式。

这两种方式都需要你以开玩笑的心态去使用，才能达到最佳效果。如果你在搭讪时觉得思考如何应对测试太困难，可以尝试使用这两种娱乐感强的技巧。

当然，如果你平时性格比较一本正经，不太擅长营造娱乐感，那么使用之前课程中提到的**检答、忽略或抛回（橡皮泥）** 这三种传统方式也是完全可行的。最重要的原则是保持**一致性**，选择最适合你个人风格的方式。

本节课中我们一起学习了两种以娱乐感为核心的现场废物测试应对技巧。希望这些“彩蛋”能帮助你在搭讪互动中更加自如、玩得开心。
