# 32.2022-紅丸4.0《自由藍圖》：4.GAME為何有效的真相_note

在本节课中，我们将探讨GAME在两性互动中有效的根本原因，并深入理解权利与责任的关系，以及它们如何塑造现代社会的交往动态。

听完前三节课后，可能会有人提出疑问：既然客观权利是事实，那么提升自己是否是学习GAME的唯一出路？首先，需要明确回答有此疑问的兄弟。

提升自己是一条正道，这绝对正确。但在我们今天所处的社会里，GAME之所以非常重要，是因为这个时代的大部分人并非活在绝对的现实之中。

大部分人活在一种主观的想象里。他们认为自己主观想象的权利大于客观事实的权利，从而扭曲了眼前的现实。这个真相就是：无数高估自身价值的女人，正是被无数低估自身价值的龟男、舔狗、软男塑造出来的。

想象一个客观权利巨大（例如，富有）的男人。但在他喜欢的女人面前，他却把自己想象得一文不值。他在主观权利上认为自己配不上这个女人。

此时，这个有钱人的客观权利就完全无法生效。这就是GAME在这个时代运用起来会非常有效的原因。你必须明白这背后的逻辑。

如果不能理解如此简单的真相，那么学习GAME就是在白费功夫。如果现代社会像古代一样，人被明确划分为三六九等，每个人的主观权利与客观权利都相匹配，那么你在两性关系中学GAME的可操作余地也会非常有限。

因此，在学习GAME的同时，你必须了解红丸男的权利意识。每个人都向往权利，但却不知道权利意味着什么。权利并非肆意妄为，**权利必须由他人的支持与配合才能产生**。如果一个王国里只有国王自己，那么他的权利毫无意义。

所以，权利与责任是一体的。拥有的权利越大，承担的责任也就越大。阿尔法男在享受权利的同时，其实承担了更多责任带来的压力。即便在古代权利巅峰的皇帝，也不得不每天重复处理无聊的工作，例如批阅奏折。

这是他支配权利后必须尽到的责任。他必须这么做，否则他的权利就会被颠覆。例如，如果皇帝开始不愿做这些重复无聊的工作，开始荒淫无度。

那么随着责任的放弃，权利就会逐渐转移。宰相可能手握实权，所有臣子表面配合，但背后不再支持皇帝。皇帝的权利就会被架空，而皇权被颠覆的后果就是皇帝被直接干掉。

因此，大部分人只看到权利带来的好处，却没有看到权利背后的责任和风险。古代能做皇帝的人并非由他自己说了算，他无法自由选择自己的命运。所以，皇帝在我看来，并非我们所说的自由的男人。

反观权利结构之下，作为被权利支配的角色，这未必是坏事。因为他们所要承担的责任往往更少。就像历史上，女人往往是被支配的角色，所以女人天生缺乏一种责任意识。

但这并非坏事。因为没有责任，意味着你可以拥有更大的自由。这也是为什么许多成年人向往儿时的状态，觉得孩童时期是人生中最开心愉悦的时候。正是因为你在小时候受父母庇护。

在那样的状态下，你无需承担任何责任，想做什么就做什么，完全放飞自我。所以你每天都很开心快乐，因为你的状态建立在绝对安全的、无责任的自由之下。

女权主义将女人这种去权利化、去责任化的状态形容为男人对女人的压迫。女权呼吁女人获得权利本身并没有错，但问题在于，她们所呼吁的是一种没有责任的权利，是一种无限膨胀的权利。

这就是女权运动错误的原因。女人想要权利却不想尽责任，这必然导致错误的结果。历史上这些错误的结果，会由一些愚蠢的支持者来买单，例如那些支持女权的傻逼白骑士、龟男、软男和舔狗。

从这个角度讲，女人会想找一个能庇护她的阿尔法男人。为什么女人总喜欢说“我希望找一个能把我像小女孩一样宠爱的男人”？因为女人想在这段关系中扮演那个放弃责任、自由快乐的角色，就像小孩一样。

而男人无法完全去责任化。这也是我们红丸所说的：**男人的自由来源于尊重秩序下的另一种相对的自由**。身为一个红丸男人，你必须理解这一点，理解这个游戏的规则就是这样运行的。
