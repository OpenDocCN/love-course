## 什么是贝勃定律？🧠

贝勃定律指的是：人经历强烈的刺激后，再施予的刺激，对他来说就会变得微不足道。

**公式描述**：`刺激反应 = 初始刺激强度 / 后续刺激强度`。当初始刺激持续且强烈时，后续同等强度的刺激所引发的感知反应会显著降低。

---

上一节我们介绍了贝勃定律的核心概念，本节中我们来看看它在现实生活中的一个具体例子。

例如，你家楼下新开了一家烤肉店，味道很好，于是你每天都去吃。不出一个礼拜，再让你去吃，你绝对不想吃了。这是因为持续的、相同的味觉刺激，让你的满足感阈值提高了，同样的美食变得不再有吸引力。

---

理解了生活中的例子后，我们来看看贝勃定律在恋爱关系中的具体表现。

在恋爱里，为什么你一直对她那么好，她却越来越疏远你？为什么你的付出没有回报？这正是贝勃定律在起作用。因为你持续地为她付出，会让她对你的付出产生“情绪免疫”和“习惯化”。

以下是贝勃定律在恋爱中的具体影响：

*   **付出变得毫无波澜**：她饿了，你立刻点外卖送去；一过节，你的礼物早就准备好了。在恋爱初期，这确实有效。但随着时间的推移，这会让她产生一种错觉，认为你做的这一切都是理所应当的。
*   **付出价值贬损**：一旦她习惯了这种想法，你之后的一切付出都会变得毫无意义，其效果甚至可能比不上一个普通朋友突然送来的一束鲜花。
*   **自我定位偏差**：你也可能逐渐沦为“讨好型人格”，这并不是感情变淡了，而是你的持续付出，无意中助长了对方的贪婪。

这时如果再出现一个懂得把握节奏的竞争者，你们的关系很可能就结束了。
