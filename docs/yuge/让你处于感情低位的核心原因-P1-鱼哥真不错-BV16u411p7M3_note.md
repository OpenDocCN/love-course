## 思维对比：强者 vs. 弱者

就像我们接下来要分析的“主动联系对方”这件事，不同思维模式的人会有截然不同的表现。

**强者思维**的男生想要交流情感时，会选择主动联系。他们可以在这一秒与女生亲密互动，下一秒就立刻全身心投入自己的工作。他们能做到这一点，是因为掌握了对自己情绪的**完全主动权**。他们能够控制自己何时开始（联系），以及何时停止（投入工作）。

**弱者心态**的男生在面对“要不要主动”这个问题时，则会犹豫不决，陷入内耗。他们的思维链条通常是：
1.  如果主动，会不会显得掉价？
2.  还是等他来找我吧。
3.  他怎么还不来找我？（开始焦虑）
4.  他是不是在忙？是不是在和其他男生聊天？（陷入猜测与内耗）

## 培养强者心态的核心：主场意识

那么，如何养成强者心态呢？有一个最核心的概念你必须理解：**你一定要具备主场意识**。

举个例子：假设你主动邀请别人吃饭。那么，被邀请的一方通常只有两个选择：接受，或不接受。然而，作为发出邀请的**你**，却拥有众多选择：
*   你可以选择今天请，也可以选择一个月后再请。
*   你可以选择只请他一人，也可以邀请多人共同参与。
*   你甚至可以改变主意，选择不请了。

这个**主动权**始终掌握在你的手里。相反，被邀请的人反而是在跟随你的节奏。因此，具备强者思维的男生，永远保持着一种“想爱就爱”的从容心态。
