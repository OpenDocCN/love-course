## 高情商的核心：共情与给予价值

一个有情商的男人往往受人欢迎。一个低情商的人则容易在感情和人际关系中屡屡受挫。所谓高情商，就是能站在对方的角度思考对方想要什么、需要什么。这也就是共情能力。然后，以对方舒适的方式给予价值，并顺带完成自己的目的。

**核心公式**可以总结为：
**高情商互动 = 共情 + 给予价值 + 达成自身目的**

## 公式解析：为何“给予价值”如此重要？

人会本能地被给予价值的人吸引。对于索取价值的人则是本能地排斥。

**举例说明**：
乞丐向你乞讨要钱，你给了他一顿饭钱，就是在给予价值。乞丐向你乞讨就是在索取价值。乞丐因为你的给予吃了顿饱饭，等于收获了价值，自然很开心。你被乞讨走了一顿饭钱，等于失去了价值，你自然就会反感。

因此，通过**共情加给予价值**的方式，来完成自己的目的，是高情商聊天的底层逻辑。
