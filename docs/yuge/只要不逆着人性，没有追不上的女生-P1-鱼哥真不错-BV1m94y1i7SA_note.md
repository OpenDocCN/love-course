# 恋爱基础课 01：理解人性与吸引 🎯

在本节课中，我们将要学习如何通过理解人性，而非对抗人性，来建立对异性的吸引力。我们将探讨情绪的核心作用、吸引力的非逻辑性，以及如何自然地推进关系。掌握这些原则，能帮助你避免常见错误，更有效地与心仪的女生建立连接。

## 1. 关注情绪，而非标准 🎭

上一节我们介绍了课程的核心主题，本节中我们来看看吸引力的第一个关键：情绪。

其实女人自己也不知道会被什么样的男人吸引。她们嘴上说喜欢真诚的、对自己好的男人，那只是逻辑上认为的、脑子里想象的。即便真有这样的男人出现，也只是满足了他的逻辑标准，没有情绪上的心动，一样会因为“没感觉”而被拒绝。

相信你肯定听过“女人是情绪动物”这句话。她们选男人主要看感觉。所谓的感觉，是被触发了吸引力开关之后的一种本能感受。就像你在电梯里看到好看的小姐姐会立即紧张心跳，女人会跟随自己的心动感受而做出选择。

她们会选择那些情绪上让她动心的，而不是仅满足了她逻辑标准的男人。有些男人浑身缺点，只要这个女人情绪上喜欢，也会强行找到值得喜欢的理由。你条件不差，对女人百依百顺，女人情绪上不来电，也总能找到不喜欢的借口。

这个就叫做 **逆向合理化**：**人会跟随自己的情绪做出选择，然后进行逻辑上的合理化**。

你千万别去告诉她，你们有多般配，你会对她有多好，这都是在试图改变她的逻辑。追女生这个事情，其实一直都是围绕“怎么让她对你有情绪回应”、“怎么有感觉”下功夫。她情绪上喜欢你了，甚至会改变自己的标准来接纳你。你不要逆着她。

## 2. 喜欢本身，不讲条件 ❤️

理解了情绪的主导地位后，我们进一步探讨：真正的喜欢是不附加条件的。

你一定要明白一个道理：如果一个女人真心喜欢你，想跟你有以后的话，那么她会主动把你们之间的障碍消除掉。不管是父母不同意、年龄差过大，还是异地恋问题。如果她跟你提出来一堆的条件和要求，你不解决她就不接受你，那么她并没有那么喜欢你。你对她来说没有那么值得喜欢。

喜欢这件事本身是不讲条件的。就像你遇到心仪的女生，会忍不住的紧张心跳，这个过程是无条件的。并不是说对方先满足了你的要求，你才有了这个心动感受。

她给你摆一堆条件要求，无非是出于补偿心理：你没那么让她动心，这个额外要求就会变高，以换取对方的心理平衡。即便你满足了她的要求，她也只是暂时妥协。

记住：**人会对现实妥协，但是本能是不变的**。刻在女人基因里的本能，就是跟有魅力的男人谈恋爱，跟没有魅力的男人谈条件。靠妥协勉强来的感情，风险系数也是极高的。

## 3. 心照不宣地推进关系 🤝

当我们建立了基本的吸引和连接后，如何自然地让关系更进一步呢？答案是：心照不宣。

人都不想被定义为随意的人。推进关系的时候，要学会给对方留台阶，这个叫心照不宣。

以下是几个具体操作的例子：
*   你约女生吃饭要说：“这家店东西好吃。”
*   约女生看电影要说：“新上映的片子好看。”
*   这都比你直接跟女人说“我对你有想法，咱俩见个面呗”要好上一万倍。

你但凡问一个女人：“一个男人约你吃东西、带你看电影，这个男人想干嘛？”只要是个智力正常的女生，立即就能给出答案。所以说，女人其实知道你在想什么，但是她需要一个台阶，让一切看起来是顺其自然，而不是被提前预谋的。

以下是创造“台阶”的更多例子：
*   牵手的时候可以说：“担心你把人家车给撞坏了，赔不起。”（营造轻松、保护性的氛围）
*   拥抱的时候可以说：“这样我就能温暖一整天了。”（赋予行为浪漫或友情的意义）

你别管这些理由有多尴尬，这都比你直接说穿意图、不给台阶要好。人性就是这样，越是喜欢你的女生，就越是想表现得自己不随意。你给了这个台阶，女人才能更好地合理化这些行为。只有这样女生才会这样想：“他是担心我的安全才牵着我”、“我是为了鼓励他才给他这个抱抱的”。

## 4. 吸引机制，超越逻辑 🌌

最后，我们必须认识到，吸引力的底层机制是超越人类理性逻辑的。

吸引这个东西是不讲逻辑的。就像你不能用人类的语言跟动物交流。比方说女人偏爱身高180的男人，你问她们“身高180有什么实际用途？”即便你辩论赢了，也改变不了她们的择偶偏好。这就是人性，你不能逆着来。

吸引这套机制早在人类文明诞生之前就已经存在了。你不能用高度文明、绝对理性的视角来审视她。

以下是两个违反吸引力本能的例子：

所以说，你不要有那种逆人性的想法。**只要不逆着人性，没有追不上的女生**。
