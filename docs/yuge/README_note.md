# 鱼哥的Python入门课 #1：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

---

## 什么是变量？📦

你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等），而标签（变量名）则帮助我们记住盒子里装的是什么。

在Python中，我们使用等号（`=`）来给变量赋值。其基本语法是：`变量名 = 值`。

**公式/代码**：
```python
my_name = "鱼哥"
age = 30
```

上一节我们介绍了变量的比喻，本节中我们来看看如何具体地命名和使用变量。

### 变量的命名规则

以下是给变量起名字时需要遵守的一些基本规则：

*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如，`age` 和 `Age` 是两个不同的变量）。
*   应避免使用Python的保留关键字（如 `if`、`for`、`while` 等）作为变量名。

## 基本数据类型 🔢

变量这个“盒子”里可以放不同类型的东西。Python有一些内置的基本数据类型，用来定义数据的种类。

### 1. 整数 (int)

整数是不带小数点的数字，可以是正数、负数或零。

**公式/代码**：
```python
count = 10
temperature = -5
```

### 2. 浮点数 (float)

浮点数是带有小数点的数字，用于表示实数。

**公式/代码**：
```python
price = 19.99
pi = 3.14159
```

### 3. 字符串 (str)

字符串是由零个或多个字符组成的文本数据，需要用单引号（`'`）或双引号（`"`）括起来。

**公式/代码**：
```python
greeting = "Hello, World!"
name = '鱼哥'
```

### 4. 布尔值 (bool)

布尔值只有两种可能：`True`（真）或 `False`（假）。它们通常用于逻辑判断。

**公式/代码**：
```python
is_learning = True
game_over = False
```

---

## 查看与转换数据类型 🔄

有时我们需要知道一个变量是什么类型，或者将一种类型转换为另一种类型。

### 使用 `type()` 函数查看类型

`type()` 函数可以返回任何变量的数据类型。

**公式/代码**：
```python
print(type(10))      # 输出：<class 'int'>
print(type(3.14))    # 输出：<class 'float'>
print(type("文本"))  # 输出：<class 'str'>
```

### 数据类型转换

你可以使用像 `int()`, `float()`, `str()` 这样的函数来进行类型转换。

以下是几个类型转换的例子：

*   **将字符串转换为整数**：`int("123")` 会得到整数 `123`。
*   **将整数转换为字符串**：`str(456)` 会得到字符串 `"456"`。
*   **将整数转换为浮点数**：`float(7)` 会得到浮点数 `7.0`。

**公式/代码**：
```python
num_str = "100"
num_int = int(num_str)  # 现在 num_int 是整数 100

score = 95
score_str = "我的分数是：" + str(score)  # 必须将整数转换为字符串才能拼接
```

## 总结 📝

本节课中我们一起学习了Python编程的基石：
1.  **变量**：像一个带标签的盒子，用于存储数据。通过 `变量名 = 值` 来赋值。
2.  **基本数据类型**：包括用于整数的 **`int`**、用于小数的 **`float`**、用于文本的 **`str`** 和用于真/假判断的 **`bool`**。
3.  **类型操作**：使用 `type()` 查看数据类型，使用 `int()`、`str()`、`float()` 等函数进行类型转换。

掌握这些概念后，你已经可以让计算机记住一些信息并进行简单的处理了。下一节课，我们将学习如何使用这些数据进行基本的运算和操作。
