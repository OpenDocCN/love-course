## 流程拆解：一场典型的“舔狗式”相亲 🔄

上一节我们概述了“舔狗式”相亲的概念，本节中我们来看看它的具体发生过程。整个过程通常遵循一个固定且令人疲惫的脚本。

以下是其标准流程的步骤分解：

1.  **开端：外部压力介入**
    在双方亲友的推动和期待下，男女双方交换联系方式。

2.  **互动：单向的沟通尝试**
    男方被迫主动发起对话，努力寻找话题。女方则反应冷淡，回复敷衍。

3.  **困境：尴尬的持续**
    由于缺乏经验，男方陷入“尬聊”局面，对话难以推进。

4.  **伤害：隐私与尊严的侵犯**
    部分女方会将令人尴尬的聊天记录分享给他人观看，加剧男方的心理压力。

## 后果与指责：不公的归因 ⚖️

当男方因持续受挫而选择终止联系后，指责的矛头往往会完全指向他。这个过程揭示了问题中的归因偏差。

以下是后续发展的常见情况：

*   父母或媒人会质问男方进展如何。
*   男方解释无法继续聊天后，会遭到长辈的批评。批评内容通常为：“**男生需要脸皮厚、嘴巴甜、更主动。**”
*   此时，几乎无人去询问女方的态度和表现。
*   女方的解释往往是：“**是这个男生不够主动。**”

在这场闹剧中，承受了所有主动压力并最终受到指责的男方，可能恰恰是唯一没有行为过错的一方。其核心矛盾在于过时的观念与单方面责任的强加。

```
【传统“男追女”观念】 + 【女方无反馈/不情愿】 = 【男方的单方面挫败与指责】
```
