## 第一部分：5W1H聊天心法

聊天遇到困难，往往是因为缺乏系统性的沟通思维。上一节我们概述了课程目标，本节中我们来看看最基础的聊天心法——5W1H。这是一个适用于所有人际交往的通用沟通模型。

聊天的基本要素是发散性思维。在使用此心法前，需确保自己处于发散性思维而非逻辑性思维中，以发挥最大效果。

**5W1H** 具体指代以下六个要素：

*   **What (是什么)**：当看到对方展示的生活片段（如运动照片），可以自然地评论并表达好奇。例如：“你这项运动看起来很有趣。” 这通常作为交流的引子，用于开启话题或转入私聊。
*   **Why (为什么)**：探究对方行为或喜好的原因。例如：“你为什么喜欢上这项运动的？” 可以带着预设进行略带调侃的提问，如“你小时候肯定是运动高手吧”。当对方回应后，便可运用后续将讲到的“拆字法”从回答中提取新话题。
*   **When (何时)**：在聊天中加入时间线索。例如：“你通常什么时候去运动？”、“是从什么时候开始喜欢的？” 此要素常与 `Why` 结合使用，能在话题偏离时帮助拉回节奏，同时，对方愿意分享时间线索也是建立初步信任的标志。
*   **Where (何地)**：围绕位置线索展开。例如：“哪里是比较适合这项运动的场地？”、“我们小时候在哪个城市上学？”。围绕地点展开分享，并从中获取新信息以延续话题。
*   **Who (谁)**：当双方建立起一定信任后，话题可围绕“人”展开。例如讨论彼此是什么样的人、或各自交际圈中的人。在氛围合适时，可引导话题进入更深层次的交流。
*   **How (如何/感受)**：这是指情绪与感受。需要将 `How` 不断穿插在前五个 `W` 中，核心是**调动对方情绪**。例如，当对方谈及运动显得开心时，就多聊运动；当提及往事略显失落时，则表达理解与共鸣。每次获得新关键词后，对话便可从第一个 `W` 重新开始循环。

**核心循环公式**：`获取新信息 -> 运用5W1H展开 -> 调动情绪(How) -> 拆解出新关键词 -> 重新循环`

掌握此心法后，聊天话题便能像套娃一样层层展开，无穷无尽。接下来，我们将探讨支撑此心法的两种关键思维与技巧。

## 第二部分：发散性思维

聊天需要你处于发散性思维中。那么，什么是聊天中的发散性思维，又该如何培养呢？

通俗地讲，聊天中的发散性思维就是进行漫无边际的联想，从对方话语中提取看似无关联的点，再找出有价值的内容进行表达。

以下是培养发散性思维的三个联想方向：

*   **向上联想**：向更宏大、抽象的方向联想。例如，女生说爱看文学书 -> 联想到作家狄更斯 -> 联想到当时的英国社会 -> 话题可自然过渡到“时代”这类宏大主题。
*   **向下联想**：向更具体、调侃的方向联想。例如，从“爱看文学书” -> 联想到“你平时是不是个很严肃的人？” -> 可对此进行友好调侃，制造聊天张力。
*   **平行联想**：向相关联的平行事件联想。例如，从“爱看书” -> 联想到“你有那么多时间看书，工作/学业应该比较清闲吧？” -> 自然引入关于工作或生活节奏的话题。

**联想方向代码表示**：
```python
联想方向 = ["向上", "向下", "平行"]
for 方向 in 联想方向:
    根据对方话语进行{方向}联想
    提取有价值的话题点
```

每个人联想的内容不同，但方向万变不离其宗。接下来，我们学习一个能将联想落地的具体技巧——拆字法。

## 第三部分：拆字法实战应用

拆字法，顾名思义，是将一句话拆分成不同的关键词，然后围绕每个关键词展开话题。这个方法简单实用，内化后可作为其他高阶技巧（如推拉、前提假设）的施展基础。

以下是拆字法的具体使用步骤：

1.  **听取对方语句**。
2.  **拆解关键词**。
3.  **选择关键词展开**。

**实战举例**：
假设女生说：“我今天和朋友去了太古汇看包，脚都要走断了。”

我们可以从这句话中拆出四个关键词：`朋友`、`太古汇`、`看包`、`脚断了`。

以下是针对每个关键词的展开方式：

*   **针对“朋友”**：例如，“是上次合照里那个很好看的女生吗？又给你当购物参谋啦？”
*   **针对“太古汇”**：例如，“我上次去那边，发现一家烤肉店的牛肉品质相当不错。”
*   **针对“看包”**：例如，“又去看包啦？看来以后家里的财政大权不能交给你了。”（略带调侃）
*   **针对“脚断了”**：例如，“辛苦了，这么累。看来下次得找个能坐着喝下午茶的地方逛。”

从这四个方向中选择一个最贴合当下语境的话题进行回应即可。平时可以与朋友对话时多加练习，习惯成自然。
