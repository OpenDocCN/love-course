## 聊天的本质

上一节我们概述了课程目标，本节中我们来看看聊天的本质。

聊天的本质是**情绪上的交流**与**关系度的推进**，而非简单的一问一答或信息互换。

许多人误以为只要双方聊得足够多、足够了解，对方就会因此喜欢自己。这个想法过于天真。例如，即使与女生聊了1000个话题，如果交流仅停留在话题表面，双方没有情绪互动，关系就会原地踏步，并且话题终将枯竭。有些人总说没话题可聊，正是陷入了这个误区。

因此，聊天的本质在于情绪交流、关系推进以及内容本身，而非简单的信息互换。

## 聊天流程

理解了聊天的本质后，接下来我们进入具体的聊天流程解析。以下是构成有效聊天的四个核心步骤。

所有社交互动都存在“预期”这一概念。你去面试，互动的预期是获得工作机会；你去买衣服，互动的预期是买到合适的衣服。那么，与女生互动的预期就是发展恋爱关系，让她对你动心并成为男女朋友。

所以，聊天必须学会制造聊天预期。只有互动方向正确，才能到达终点。如果方向错误，所有努力都是在浪费时间和精力。

明确了聊天预期后，我们来看看如何具体开启和结束一次对话。

成功开启对话后，如何让话题源源不断地进行下去呢？

最后，我们重点探讨聊天中最关键的一环：情绪的调动。
