# 绅士派私教课 180108：聊天禁忌与思路1 💬

在本节课中，我们将要学习聊天中的核心禁忌与基本思路。聊天是许多人在社交中常遇到的难题，很多人困惑于该聊什么以及如何根据对方的反应进行回复。实际上，许多人忽略了一个关键点。

很多人认为聊天必须是完全系统化的，每一句话都需要经过深思熟虑和反复锤炼。但从我们现在的视角来看，如今已是2018年，聊天在整个社交互动中的重要性，已不像过去时代那样占据绝对主导地位。

过去时代是怎样的状况呢？这很简单。例如，我从2009年就开始通过搭讪认识姑娘。那时没有微信，没有探探等社交软件，也没有展示面的概念。那时可能只有QQ，你或许能在QQ空间里放几张照片，那大概算是当时的展示面，但那时大家并不特别注重这个。

可能空间里放的都是一些非主流或杂乱的内容。所以，当你搭讪或认识一个姑娘后，你拿到她的联系方式时，会发现你们之间唯一的连接点只是一个电话号码。

你和她能如何沟通呢？只能发短信。说实话，从短信时代开始，这对许多人来说就是一个大问题。因为那时你与她的连接只能通过聊天来建立。你需要通过聊天告诉她你是一个什么样的人。

你需要通过聊天告诉她：我今年多大，身高多少，职业与身份是什么，我对你是什么感觉，以及我们未来可能的发展方向。你需要通过聊天，一步一步地、以合理的方式透露并展示所有这些信息。

但在当今时代，情况已大不相同。原因很简单：在聊天开始之前，你们已经有了一个叫做“展示面”的东西。

因此，我们经常告诉内部课程的学员一个实情：很多兄弟常来对我们说“这些姑娘都不理我，一定是我聊天有问题，帮我解决一下”。这时我会先告诉他们：你需要先把基本的展示面做好。当姑娘从一开始你打招呼就不理你时，问题通常不出在聊天上，而出在展示面上。

上一节课我们刚刚讲完展示面的构建。在讲完资源收集之后，我们现在才开始讲解聊天。聊天是建立在前三个阶段已经完成的基础之上的。

以下是构建聊天基础的三个阶段：

1.  **自身形象改造**：首先，你需要完成自身形象的改造。
2.  **展示面呈现**：将改造好的形象，转化为你的展示面，并在微信朋友圈等平台清晰体现出来。
3.  **建立初步认知**：这样，别人就会对你有一个基本的认识。

这时，你再去和她聊天，就是一种正常的互动。这不像是一个一无所有或杂乱无章的状态。这很简单，就像一个人收拾好自己之后的状态——例如，她今天已经洗完澡，修饰了自己，做好了发型。

本节课中，我们一起学习了聊天在当今时代的定位，明确了其重要性已相对降低，并强调了聊天必须建立在良好的自身形象与展示面基础之上。这是开启有效互动的第一步。
