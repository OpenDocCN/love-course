TD属于进挪的一部分。现在，我们系统学习进挪的原则与顺序。

总体思路是：**从最不敏感的身体部位，逐步推进到最敏感的部位**。

4.  **搜身**（抚摸胸部或臀部）：可有可无，视环境和对方开放程度而定。

2.  **不跳步**：稳妥操作，最好不要跳过某个阶段。
3.  **及时释放**：进挪后一定要及时释放（松手）。例如牵手十秒左右就可以自然松开，搂腰过马路或人多时扶一下即可松开。一直不松手会显得需求感过强。
