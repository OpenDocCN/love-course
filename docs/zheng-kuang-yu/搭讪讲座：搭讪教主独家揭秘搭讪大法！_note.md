# 搭讪教程 02：提升熟悉度与渐进式搭讪法 🧠

在本节课中，我们将学习如何通过增加熟悉感和运用渐进式策略，来有效降低对方的防备心理，从而提升搭讪的成功率。核心在于创造一个自然、无压力的互动环境。

上一节我们介绍了搭讪的初步心态，本节中我们来看看如何在实际场景中巧妙运用“熟悉感”这一关键因素。

## 增加熟悉感以降低防备

发展关系成功的一个重要因素是增加对方对你的熟悉度。假设你想认识一位在同一办公大楼但不同公司工作的女生，如何搭讪最好？

如果你在早上8点45分进公司时在楼梯间看到她，我告诉你一个方法。

从明天开始这一个星期，你8点20就到公司楼下。看报纸也好，吃早饭也好，最重要的是当她进来时，立刻跟着她一起走进电梯。为什么？为了让她多看到你。

人有一种奇怪的心理。我今天在路上突然跟你讲话，你可能觉得我是坏人，不知道怎么办，会害怕。但如果我是住在你同一个小区，你从小看到大的邻居，只是没讲过话，你觉得我跟你讲一句话你会恐惧吗？不会。

仅仅因为你多看到我几次之后，你的恐惧感就会降低。所以你要让自己被对方多多看到。

## 电梯内的互动技巧

好，现在跟大家讲一个诀窍。你一走进电梯的时候，要去抢那个帮大家按电梯楼层按钮的位置。你知道为什么吗？

这可以营造一个你乐于为大家服务的、大方得体的形象。最重要的是，你可以借此强迫自己与她开始展开对话。

“请问去哪一楼？” 你就是要帮她按。她可能之前都不需要别人帮忙按，但重点是你要主动问她。

大家进电梯时，其实有很多这种可以强迫或故意挑起话题的机会。说真的，走进电梯，按自己的楼层就好了，对不对？不，你偏要当那个服务大家的人。

这个时候请你记得，除了帮她按楼层，别做得太明显。“请问去哪一楼？帮你按。” 第一天可能是这样的，先按楼层。

## 实施渐进式“打游击战”搭讪法

第二天呢，你又“碰巧”遇到了，跟她进电梯按楼层的时候，可以说：“你去六楼？六楼是哪家公司啊？我在八楼，我们八楼是某某公司。”

她可能回答我们公司是做半导体的。讲两句，各位就不要再讲了，讲两句就好。你知道为什么吗？这个时候又涉及一个技巧了，这叫做“打游击战”搭讪法。

什么意思？各位想一想，女生们也想一想有没有道理。今天一个不认识的男生，只是在同一栋大楼里跟你讲话，聊了两句之后，就问：“那你叫什么名字？我叫某某。我们保持联络，你给我姓名电话吧。” 女生心里的警钟立刻就响起了，对不对？

但是如果这个男生只是像我这样子，第一天问“去哪一楼？”，第二天聊“你们公司做这个？我们公司做那个的？”，第三天的时候，跟你聊了几天都没问你的名字，我叫你？我问你的名字？

各位，你每一次都只聊一聊，但就是不跟她要联络方式，她没办法拒绝你啊。

## 为何此法难以拒绝

女生被男生搭讪时，所谓的“拒绝”通常发生在对方索要姓名、电话、Email的时候。你只是跟她聊公司状况、聊天气，然后说“拜拜”，她是没有办法拒绝你的。

她不可能在跟你聊了几句，你说“拜拜”之后，突然把你叫回来说：“你给我回来！你明明应该跟我问姓名电话，好让我拒绝你的！你怎么可以不问？” 不可能，各位，这不可能。

所以，各位，你就用这种方式打游击战。今天聊一聊，明天聊一聊，后天聊一聊。然后第三次、第四次的时候才跟她说：“聊了很多次，还不知道您的大名。这是我公司的名片，您有名片吗？我们可以交换一下名片，以后公司也许会有合作。”

这样，女生就觉得自然多了。为什么？因为她已经看到你好几次了，每一次都跟你讲几句话，但是你都没有跟她要姓名电话Email，她不可能拒绝你。并且会因此对你产生一点熟悉感，对不对？

好，各位。用这种方式搭讪，搭讪要成功，还有第三个要点（我们将在后续课程中探讨）。
