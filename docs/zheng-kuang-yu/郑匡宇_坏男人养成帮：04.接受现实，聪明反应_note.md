# 郑匡宇_坏男人养成帮：04.接受现实，聪明反应_note

在本节课中，我们将要学习如何面对与异性互动中的挫折，并采取聪明的应对策略。核心在于调整心态，避免自我贬低，并在付出时保持清醒，以展现自身价值。

## 信念七：拒绝归因与心态调整

上一节我们探讨了互动的基本心态，本节中我们来看看当遭遇拒绝或冷淡时该如何思考。好人常将拒绝归咎于自身，而坏男人则能客观看待，避免不必要的负面情绪。

被女人拒绝时，好人相信一定是自己哪里做错了。坏男人会认为那一定都是对方的问题。

虽然我很想强调“都是对方的问题”，但根据实际案例，有时确实是男生的问题。当你对一个女生，在两人还不是男女朋友时，就付出过多关怀，这确实是你的问题。

如果你按照我在书和演讲中的建议操作，例如每周打电话两到三次，短信每三天或一周左右发一次，女生通常能感受到你的关心。对你有兴趣的女生自然会回应。

大部分人最大的问题在于：当打电话对方未接，或传短信对方未回时，大部分好人会开始负面思考。例如，认为对方正在约会、不想理自己，并因此胡思乱想。

然而，坏男人遇到此类问题时，绝不会用这些负面想法打击自己。对方未接电话很正常，可能他正在忙。即使打了两通电话对方都没接，也没关系。

以下是坏男人的应对步骤：

*   简单地留一条短信，说明打电话的原因，例如邀请对方参加活动。
*   发送后，等待对方回应。
*   即使对方没有回拨电话，那又如何？隔两三天再打即可。

传短信对方未回，也许是对方没收到、手机关机，或根本不在国内。想那么多干什么呢？

所以，一个简单的道理是：坏男人绝不会胡思乱想，用负面东西打击自己的自信。因为我自己就是这样的人。

在对女生时，存在所谓的“公主与仆人理论”。你要幻想自己是王子，公主才有可能喜欢上你。如果你总是做仆人的行为，如跟屁虫、尾随其后，或猛打电话，对方怎么可能把你想象成王子呢？

因此，我要告诉各位：看重自己，提高自身价值，就从你对待女生的态度开始。电话未接没关系，隔两三天再打。打电话时，也不要追问为何不接电话。

更不要在对方两三次未接电话时，就开始想象挫败场景打击自己。你必须明白，你追求的是一个彼此欣赏的对象。当对方完全不跟你出来、不接电话时，这就表示得很清楚了：他不是那个正确的人。

所以，千万不要贬低自己，不要把自己想得太差。对方不理会你，那是他的问题，不是你的问题。

除了这个信念，还有更多成为魅力坏男人的信念等着我们来开发。就让我们拭目以待吧。

## 信念八：付出与止损点

上一节我们学习了如何调整被拒绝时的心态，本节中我们来看看在付出时应如何保持理智。好人倾向于不断付出，以为会有回报；而坏男人懂得设定止损点，及时抽身。

好人倾向于一直付出，因为他们以为付出就会有回报。坏男人懂得设止损点，一看苗头不对，马上壮士断腕。

好人跟坏男人最大的区别就在这个地方。其实我们一开始认识女生时很正常，无论是好人还是坏男人，都会对女生表示某种善意或好感。但此时女生的反应，会决定好人跟坏男人后续行为的不同。

好男人与坏男人在一开始都会做某种程度的付出。但我之前提到过，坏男人的付出是经过精密计算的，并且知道要有回报。不同的是，好男人在付出时是一股脑地付出，完全不计较，并且习惯于一点一点地持续付出。

他们的想法是，只要持之以恒地在女生身边默默付出，最后女生一定会被她们的善意和善良打动。但我要说一个残忍的事实：并不会。

因为你在付出过程中，并没有展现你的优势，没有激起女生对你的好感，也没有让女生觉得与你心灵相通。那女生为什么会喜欢上你呢？而且，做同样事情的人非常多，不差你一个。

坏男人与好男人付出的不同之处，除了懂得精密计算外，还在于懂得在过程中展现自己的优势，让女生被他的优势所吸引。

举个例子，教主我自己在追求前女友时发生过一个故事。当时我们在聊MSN，她还不是我的女朋友。她突然问我：“你可不可以帮我看一下我学弟的英语话剧？我英文不太好。”

我当时第一个反应是（毕竟是坏男人的领袖）：这个话剧会不会很长，耽误很多时间？当她将话剧文件发过来，我接收后发现只有三页左右。我浏览了一下，发现原作者写得不错，我评估大概花20分钟就能修改完成。

于是我想：可以做。为什么呢？因为这项工作对我前女友（英文不好）来说非常困难，但对我只需20几分钟。这能让女生发现我的英文优势，何乐而不为呢？

只要花一点小小的付出和时间，就能获得很大收益。而且，我在接到工作时，马上告诉前女友：“没问题，我帮你改。但别忘了跟你学弟学妹讲，这是搭讪教主帮他们修改的，演出时要致谢一下。”这又宣传了我自己。

同时，前女友问我：“我不知道该怎么感谢你。”我回答：“很简单，一起吃个饭吧，什么时间？”于是，我直接约好了下一次的饭局。

所以我要跟大家说，如果你想吸引女生，并不是说在帮忙或付出时什么都不做。而是在付出过程中，一定要想办法将其与你的优势、以及能打动对方、建立信心的事情关联起来。这样就能借由你的优秀来吸引女生。

因此，再给大家一个建议：如果你是公司的小主管，请千万要“假公济私”。让你欣赏的异性加入你负责的项目中，让女生看到你神采飞扬、运筹帷幄的样子。这样，女生就容易被你自然而然地吸引。

除了这个信念，还有更多成为魅力坏男人的信念等着我们来开发。就让我们拭目以待吧。

**本节课总结**

本节课中，我们一起学习了两个核心信念。首先，面对拒绝或冷淡时，要**接受现实**，避免负面归因和自我打击，保持自信和风度。其次，在付出时，要**聪明反应**，懂得计算投入产出，并巧妙地将付出与展现自身价值相结合，同时设定清晰的止损点。记住，提升魅力的关键在于自信、自爱和策略性的互动。
