# 搭讪圣经课程3：自信与沟通基础训练 🧠💬

在本节课中，我们将学习如何通过系统性的练习，从零开始建立与异性（或陌生人）交谈的自信和基本沟通能力。课程将分为两个核心方法进行讲解。

对于缺乏自信、不敢与心仪女性交谈的男性，如何训练自信和改善沟通态度？搭讪教主在此提供两种方法。

## 方法一：镜子对话练习 🪞

上一节我们提出了自信训练的需求，本节中我们来看看第一种具体练习方式。

请你从今天开始对着镜子练习。你可以站在镜子前面。

幻想一个场景。假设在公交车站或咖啡厅，你看到了一个想要认识的女性。这时，你突然睁开眼睛，模拟已经开始与她对话。

请注意，睁开眼睛后，看着“她”并说：
```
你好，刚才我一直在咖啡厅就注意到你了，但是一直没有机会跟你说话。我很想认识你。我叫[你的名字]，你呢？我在[你的工作地点]工作，你呢？
```

请注意，你要主导话题。千万不要说完“想认识她”之后就没有话讲。当你主导话题之后，对方就有可能顺着你的话说下去。

这时你要注意观察。如果你看着镜子里的自己，出现手乱动、乱抓、乱摸、乱搔痒，或者你会笑场，或者你的声音含糊不清、结结巴巴，那么请记住，在现实场景中，这种情况会更明显。

所以你要做的是，看着镜子里的自己，慢慢修正行为举止。我知道你想认识异性，你没有恶意。但当你表现得太紧张、太害怕时，对方就会觉得你是一个奇怪的人，从而想要拒绝你。

## 方法二：渐进式陌生人接触训练 🚶♂️👀

除了自我对话练习，还有一个可以训练和陌生人接触的方法。以下是分三个月的渐进式训练计划。

### 第一阶段：眼神与微笑接触（第一个月）

第一个月，请在公交车、火车上，或走路过程中，当你看到任何一位你想要认识、感兴趣的异性时，都对其保持点头微笑。

这个练习的目的是训练你敢与完全不认识的陌生人进行眼神接触。这两个月都只需点头微笑，无需说话。

### 第二阶段：增加语言问候（第二个月）

从第二个月开始，请注意，除了点头微笑，请你多加一句“你好”。

这个阶段的目的是，除了训练你敢与陌生人点头微笑，还训练你敢说出一声问候。

这两个月的训练，能让你在期间与上百甚至上千个人进行过“点头微笑说你好”的练习。关键在于，对方无法拒绝这种问候。不相信会有任何人以非常负面或生气的态度回应，例如：“先生，你刚刚干嘛对我微笑？”或“先生，你刚刚干嘛跟我说你好？我不好。”不可能有这样的人。

通过这种对方无法拒绝的单方面训练，你开始培养出敢与陌生人说话或进行第一步接触的勇气。

### 第三阶段：加入真诚赞美（第三个月）

第三个月，请你除了点头微笑说“你好”之外，多加一句称赞的话。

称赞别人真的没有那么困难。你可以观察对方的衣着、服装、手上拿的书或正在看的报纸，并借此称赞对方。例如，你可以说“你的衣服很漂亮”或“你的鞋子很漂亮”。

同样，当你说完这些话后，对方有可能完全不理你，但也非常有可能回你一声“谢谢”后，主动问你：“你也住在这附近吗？”或“其实我常常在公交车上遇到你。”对方很有可能就会直接与你对话了。

这时，你就要把握机会，继续把话题说下去，继续主导话题。

本节课中我们一起学习了两种建立搭讪自信的基础训练方法：**镜子对话练习**和**为期三个月的渐进式陌生人接触训练**。这些练习旨在通过安全、可重复的方式，逐步克服紧张感，培养自然、主导的沟通能力。对于完全没有搭讪经验、也不敢与女性说话的人来说，这是一个非常好的系统性练习方案。请大家试试看。
