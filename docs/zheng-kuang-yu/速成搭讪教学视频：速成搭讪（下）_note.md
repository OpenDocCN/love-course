# 速成搭讪教学 🎯：速成搭讪（下）

在本节课中，我们将要学习几种实用的搭讪开场方法，并了解如何自然地获取联系方式，以及如何为后续互动铺路。我们将探讨“废话搭讪法”、“硬找话题法”、“见缝插针法”和“诚实搭讪法”等核心技巧。

---

第一句话就能保证搭讪成功吗？我最初出书《全民搭讪运动》时，总结了16年的搭讪经验。如今四年过去，经验已累积至20年。

我依然坚持那个观点：**根本不存在搭讪时的必杀绝句**。但如果你能像我之前课程所讲，通过良好的观察力，通过增加对方对你的熟悉度，为自己创造一个有力的“舞台”，你确实可以准备一些出色的开场白，让女生眼前一亮并对你印象深刻。然而在大多数时候，这种增加熟悉度和创造有利舞台的机会并不会自动出现。

那么搭讪时到底该说什么呢？现在就来为大家分析。

## 1. 废话搭讪法 💬

首先介绍的是废话搭讪法。这种方法的核心是，你明明说的是一句“废话”，但它却是一个非常好的开场白，一个与对方开启对话的契机。我们很多人在参加团体活动时都想认识新朋友，但常常是你在等别人开口，别人也在等你开口，结果大家陷入沉默。因此，讲出第一句话就变得至关重要。

以下是废话搭讪法的具体操作示例：

*   **场景一：在活动或演讲中。** 我会设法坐在感兴趣的异性附近，或在她擦肩而过时“硬问”一个问题。例如，我明明知道洗手间在哪里，却偏要问她：“小姐，请问一下，洗手间在什么地方？” 等她回答后，我会说：“好，谢谢。我的包包先放在你旁边，我待会坐这个位置，你可以帮我看一下包包吗？” 然后我就去“洗手间”（这当然是借口）。回来后，就能很自然地开始聊天。这就是用一个你明明知道的信息，一句“废话”，强迫与对方开启对话并产生关联。
*   **场景二：在社交活动中。** 你可以问：“哈喽，请问一下，你今天怎么知道这个活动的？” 对方会解释他的信息来源。我可能会说：“哦，我是看报纸知道的。” 这些信息本身并不重要，**关键是用一句话让对方觉得“有人跟我说话了”，对话就能就此展开**。

## 2. 硬找话题搭讪法 🛩️

上一节我们介绍了如何用“废话”开启对话，本节中我们来看看如何主动“硬找”话题。这种方法与废话搭讪法类似，核心是无论如何都要“套”出一句话来。

以下是一个硬找话题搭讪法的实例：

*   **场景：在飞机上。** 有一次我从韩国飞回台湾，旁边坐着一位可爱的泰国女生（后来聊天得知她是华人）。在飞机上，很多人看起来都很冷漠，会立刻戴上耳机或看书。这时怎么搭话呢？我灵机一动，开始看飞机上的杂志，并打开了座位阅读灯。那个灯其实是固定好的，但我站起来，假装研究它，然后故意问那位女生：“请问一下，你知道这个灯是可以调的吗？”（我明知不能调）。我用英文问她，她当然会说：“我不觉得那个灯能调。” 但这不重要，**因为我的目的就是要跟她讲上话**。对话开启后，我接着问：“你们怎么会来韩国玩呢？我发现你们这个团体好像很多人……” 就这样自然地聊了起来，最后也成功要到了她的联系方式。

## 3. 见缝插针搭讪法 🧩

除了主动创造机会，我们也要善于利用现有的社交场景。这就是见缝插针搭讪法。

无论是在学校还是职场，假设你很想认识一个人（比如同事的客户，或学校里心仪的女生），但一直没有机会。突然有一天，你看到一位朋友（哪怕不熟）正在和她说话。

以下是具体操作步骤：

1.  **立即介入**：马上走上前，站在他们旁边。
2.  **礼貌开场**：先对那位女生说“不好意思，打扰一下”，然后转向你的朋友（假设他叫阿伦）开始说话。
3.  **制造三人对话**：例如问：“阿伦，我问你一下，上次社团提到的那个活动具体怎么安排？我们下次见面是什么时候？” 阿伦回答后，你可以转向旁边的女生说：“这位也是我们社团的同学吗？” 不管她回答是或不是，你都要想办法将场景变成三个人都能参与的聊天。
4.  **后续跟进**：简单问候，得知对方姓名。下次再遇到她时，就可以自然地打招呼。如果她露出疑惑的表情，你可以提醒她：“我们上次是通过阿伦，在某某地方认识的。” 这样就能再次开启对话。

## 4. 诚实搭讪法 💖

前面介绍的方法都需要一定的场景或技巧。但有时你真的没有那么多时间、技巧或机会去创造一个“自然而然”的搭讪情境，这时该怎么办？

最后介绍的方法是诚实搭讪法。即诚恳、诚实地告诉对方你心里的想法。

例如，在公交或地铁上，一位女生一上车就吸引了你的目光。你左思右想，发现没什么东西可以开启对话，而你可能再过三站就要下车了。这时，你可以直接走上前说：“哈喽，小姐，你一走上车我就看到你了，我觉得你气质很好，我很想认识你。”

**关键点来了**：当你说完“我很想认识你”之后，大部分人下一句会问“可以吗？”，然后摆出笑脸等待回应。这时女生可能会说“不方便”、“不好”或“我有男朋友”。

所以，使用诚实搭讪法时，**说完“我想认识你”后不要等对方回答，要立刻自己接下去说**。例如，你可以接着说：“我看你气质很好，非常好奇你是不是从事服务业？不知道气质怎么会这么好。” 不管对方回答是或不是，你都要继续主导话题，比如：“我有个朋友在酒店工作，你的气质跟他很像，你该不会也在酒店工作吧？”

**记住，要一直主导话题，不停地说下去**。这看似是临场反应，但更重要的是你**日常生活的积累**：你是否吸收了大量不同信息？是否看了很多书？是否参加了各种活动并与不同的人善意攀谈？只有具备这些“基本功”，才能在说完“我想认识你”之后，有源源不断的话题可以延伸，主导谈话氛围。

## 5. 如何获取联系方式与后续邀约 📞

与对方聊得不错之后，一定要获取联系方式。要成为搭讪高手，“心机”一定要重。这里的“心机”是指在聊天时，一旦发现对方态度亲切、聊得不错，你脑中就要不断重复提醒自己三样东西：**姓名、电话、Email（或QQ/MSN）**。

如果不反复提醒自己，很可能出现这种情况：你们聊得很开心，你打算三分钟后要联系方式，结果对方突然说“我的站到了，拜拜”，然后离开，留下怅然若失的你。为了避免这种“白搭”的情况，你必须提前准备。

**技巧一：交换信息，掌握时间。**
在聊得不错时，可以“心机很重”地突然冒出一句：“Iris（假设对方名字），今天跟你聊得非常开心。我待会在台北车站这一站下，你在哪一站下？” 对方会说出一个站名。**这个技巧在于：你抛出自己的信息，换取了对方的信息**。这样你就知道自己还有几站的时间可以用来铺垫并索要联系方式。如果对方知道你即将下车，那么你说“我要下车了，我们保持联络吧，这是我的号码，你的号码也给我吧”时，对方也更容易接受。

**技巧二：为要联系方式提供合理理由。**
当对方问“为什么要给你电话？”时，你的回答很重要。建议这样说：“我们就是交个朋友。我平时很喜欢参加某某活动（例如演讲、读书会、展览），也很喜欢交朋友。希望下次有这类活动时，我可以告诉你，我们可以一起去。” 用这种方式邀约，对方会更乐意给出联系方式。

**为什么这种方式有效？**
因为用公开活动邀约，会让对方感觉场景公平、公正、公开，很有安全感。你还可以补充：“其实你可以带朋友一起来，我也会带我的朋友。” 对方会觉得这是一个朋友聚会，戒备心会降低。

**但真正的技巧在于：**
你想一想，你带这个女生一起去听演讲、参加读书会或看演唱会（最好是免费的），难道你们不会在活动前一起吃个饭吗？难道活动后不会一起喝杯咖啡聊聊天再走吗？当然会。所以实际上，女生就有了这些时间与你单独（或小范围）相处。在这个过程中，通过共同活动获得新知，彼此聊天更有话题，交流更多，联系更紧密，对方喜欢上你的几率就会变高。
