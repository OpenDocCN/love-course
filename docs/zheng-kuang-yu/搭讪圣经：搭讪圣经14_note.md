# 搭讪圣经：搭讪圣经14_note

在本节课中，我们将要学习“迷男方法”中的第二个核心技巧——否定(Neg)。这是一种通过轻微打压或质疑来吸引对方注意、建立平等对话框架的方法，尤其适用于应对高价值目标。

上一节我们介绍了迷男方法的第一种技巧。本节中我们来看看第二种被称为“否定(Neg)”的策略。其核心含义是：在社交场合（例如夜店）遇到非常漂亮的女性时，不要立即上前赞美其外貌。

大多数男性看到漂亮女性的第一反应是接近并称赞其美貌，试图以此开启对话。但迷男方法强烈反对这种做法。

以下是迷男建议的做法步骤：

首先，当你看到目标时，不要急于上前说话。你应该先坐在旁边，表现得对她不太在意。但前提是，你已经通过之前的技巧（如奇装异服）吸引了她的注意。你的服装吸引了女生，但你的态度又显得不怎么关注她，这种反差能进一步引起她的好奇。

接着，在一个看似不经意的时刻，比如当这位像超模一样漂亮的女性经过你身边时，你才与她说话。但你的眼神不能是祈求关注的样子，而应该是不经意地看到她。

然后，你的第一句话是进行一种“假性确认”：
```
“请问，你是模特吗？”
```
此时，女生的可能反应是承认。这背后隐含的心理是：她可能会认为这又是一个只看重她美貌或名气才来搭讪的男性，从而将你归类为“苍蝇男”。

然而，你的下一个反应至关重要。你不能接着问“真的吗？你是哪家公司的？”或“我在哪里见过你？”。相反，在她说“是”之后，你的下一句话是进行“否定”：
```
“我讨厌模特(I hate models)。”
```
你需要带着些许严肃的表情看着她，说出这句话。这时，女生通常会感到惊讶并询问原因：“你为什么讨厌模特？”

此时，你再给出解释，完成整个“否定”框架：
```
“因为我前女友就是模特。通过她我认识了很多模特，但我发现她们大多很无趣。除了时尚，她们似乎没有其他生活专长或兴趣。我希望你不是这样的人。”
```
说这段话时，你需要专注且略带严肃地打量她。这个流程会激发女生对你的兴趣。因为她发现，别人都把她捧得很高，而你的态度却与众不同。这能卸下她将你视为“苍蝇男”的心防，也打破了她原本高傲的自尊。

你通过显示自己的不同，让她更想与你交谈，并向你证明她与其他肤浅、无趣的模特不一样。

本节课中我们一起学习了“否定(Neg)”技巧的完整应用流程：**通过“假性确认 -> 否定 -> 合理化解释”的组合，打破常规的赞美模式，建立平等、有趣的互动框架，从而有效吸引高价值目标的注意力。**
