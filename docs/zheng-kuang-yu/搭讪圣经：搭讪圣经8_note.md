# 搭讪圣经课程8：五个女人的故事（一）—— 小唇的启示 🎯

在本节课中，我们将学习搭讪教主郑匡宇分享的第一个关键故事。这个故事揭示了在情感互动中一个至关重要的心态误区，并通过一个真实案例，帮助我们理解感情的本质与健康的互动方式。

## 故事背景

上一节我们介绍了本系列课程的由来。本节中，我们来看看第一个女人——“小唇”所教会我的事情。

这个故事发生在我高二升高三的暑假前夕。我当时就读于成功高中。

## 相识过程

以下是相识的具体经过。

*   我与朋友小胖一同参加一场女校的音乐会。
*   中场休息时，我看到了小唇和她的朋友。
*   我的朋友小胖恰好认识小唇身旁的女生。
*   我运用了“见缝插针法”，主动上前交谈，营造出大家都是朋友的氛围。
*   我们因此相识，并聊得非常愉快。

## 关键的追逐与重逢

中场休息结束后，小唇表示需要回家。我回到座位后感到十分遗憾，因为没有留下她的联系方式。

大约五六分钟后，我决定采取行动。我告诉朋友小胖我对音乐会兴趣不大，但对小唇兴趣很大。他支持我，于是我们立刻收拾书包，朝她可能等车的方向飞奔而去。

幸运的是，我在公交站找到了尚未离开的她。那一刻的感觉非常美妙，仿佛小说中的情节。

## 关系的发展与突变

之后，我送她回家，并成功要到了电话。我们相约再次见面，感情似乎很不错。

然而，大约一两个月后，情况突变。她开始不接我的电话，我完全无法联系到她。

## 残酷的真相与领悟

经过多次尝试，她终于接听了我的电话。这次通话让我感到非常痛苦。

我向她倾诉思念，甚至提到曾因想她而差点发生意外。我质问她为何忍心这样对我。

她的回答是：**“我又没有叫你这样喜欢我。是你自己要这样喜欢我的，关我什么事？”**

这句话对我而言如同晴天霹雳。挂掉电话后，我仔细思考，发现她的话非常有道理。

我与她认识的时间并不长，约会的次数也不多，并未深入了解她的人格与个性。我对她的强烈感觉，很大程度上源于初识时的愉快体验，以及我受到电影、电视、小说影响而产生的“一见钟情”的自我暗示。

**是我自己决定加深对她的感觉，是我自己决定陷入这种单方面的深情。** 她说的确实没错。

## 核心教训

我把这个故事分享给大家，是希望大家明白一个道理：很多时候，是你自己让自己陷入情感困境的。

对方可能并未产生同等的感情。健康的感情永远是两情相悦的，需要双方互相喜欢、欣赏，并慢慢累积感情与信任，而非单方面的猜测与幻想。
