# 《搭讪圣经》电台访谈音频：09. 亲身搭讪经历说明 🎧

在本节课中，我们将通过一个具体的搭讪实例，来探讨当无法依赖特定场景或熟悉度时，如何依靠**绝对自信**来成功开启对话。我们将分析对话的每个步骤，并总结出可实践的核心原则。

---

上一节我们讨论了利用地点和创造有利舞台的重要性，但那些机会并非总能遇到。本节中，我们来看看当机会突然出现在路边或地铁站时，应该如何应对。

**核心概念是：绝对自信。** 其公式可以概括为：
**成功搭讪 = 绝对自信 + 主动开口**

以下是基于自信进行搭讪的关键心态和步骤。

### 心态建设：自信是基石

许多人会担心开口说什么、问题是否合适、是否冒犯对方。这些顾虑源于想得太多。自信的心态是：**“我想认识她，和她说话是一件有趣的事，我享受这个过程。”** 拥有这种信念，你就不会因对方习惯性的初始拒绝而受挫。

### 实战案例解析：地铁站搭讪

以下是一个在地铁站发生的完整搭讪过程，展示了自信如何引导对话。

1.  **观察与开场**
    我在地铁长椅上注意到一个可爱的女生走过来坐下。我没有犹豫，直接对同伴（韦纯）大声说：“你看旁边这个女生，长得很像周慧敏耶。” 这句话故意让女生听到，自然地引起了她的注意。

2.  **直接表达意图**
    当她看过来时，我立刻对她说：“对，你果然长得很像周慧敏。像你这么优质的女生一定要认识的。你现在是学生还是在工作？” 这里直接表达了欣赏和想认识的意图。

3.  **应对习惯性拒绝**
    女生听到后笑着摇手拒绝。这是非常常见的“习惯性拒绝”，就像人们通常会先拒绝街上发的传单一样。自信的人不会因此退缩，而是会“拒绝别人对你的拒绝”。我看出她的拒绝并非针对我本人，于是继续说：“其实你也不用先拒绝，跟我们聊聊之后你就知道。所以你现在是学生还是在工作？”

4.  **抓住话题，延伸对话**
    她回答是“大学生，今天去面试”。这提供了一个绝佳的话题切入点。因为我熟悉面试技巧，便就此展开聊了下去：为什么面试穿正装、面试中遇到的问题、她所学的专业（外语系）与应聘职位（银行）的关联等等。**对话的关键在于像翻阅一本好书，从一个线索（A）自然延伸到另一个线索（B）。**

5.  **再次索取联系方式并幽默化解**
    聊了一会儿后，我直接说：“我待会再过三站要下车了，你把你的姓名电话email给我吧，这样我们还可以保持联络。” 她表示电话不方便，但email可以。我没有纠缠，而是继续聊面试话题。在下车前，我用了一个幽默对比：
    > “你看，我和韦纯一表人才，跟你讲话这么礼貌，要个联系方式你才给个email。刚才面试你的那个猪头主管，态度可能不好，搞不好还不录取你，结果你却把姓名电话住址全给他了。这公平吗？”
    这番话让她觉得好笑又害羞，最终她笑着把姓名、电话和email都给了我。

6.  **总结对话核心**
    整个过程中，我没有过多担心，想开玩笑就开，想说什么就说什么。信念很简单：认识她让我开心，我也要把这种开心带给她。**搭讪的本质就是自信的展现。**

### 创造“有利舞台”的小技巧

即使没有现成的共同场景，也可以通过一些小举动吸引注意，为自己创造开场机会。

以下是一个例子：
*   **准备特别道具**：我常在公共交通上看韩文书（在台湾相对少见）。
*   **选择目标附近**：进入车厢后，有意识地站在或坐在感兴趣的女生旁边。
*   **等待并抓住信号**：当对方好奇地看向我的书时，我立刻主动问：“你也懂韩文吗？”
*   **自然开启对话**：对方回答“不懂”后，我可以接着说：“我看你往这边看，以为你懂呢。我学韩文是因为在韩国工作……” 对话就此展开。

同理，特别的穿着、正在进行的独特活动，都可以成为“被注意”的契机。一旦对方注意到你，你就可以主动开口，例如：“我们认识吗？为什么一直往这边看？” 从而顺势开启对话。

### 关于“缘分”的思考

有人会觉得搭讪不是缘分。但**缘分是需要被主动看见和抓住的**。想想看，在特定时间、地点遇到特定的人，这本身就是一种缘分。你的主动搭话，就是开启这段缘分的钥匙。当你习惯释放善意、主动交流，你会发现生活中处处是缘分。
