# 搭讪圣经课程11：搭讪实例演练 🎬

在本节课中，我们将通过一个具体的业务拜访场景，学习如何循序渐进地与心仪对象建立联系并成功获得联系方式。整个过程强调自然、渐进和态度的把握，而非单次冒进的表白。

---

## 场景设定

上一节我们探讨了搭讪的基本原则，本节中我们来看看一个需要运用策略的具体场景。

假设你是一名业务员，需要去台北101大楼拜访客户。在你客户所在的楼层，有一个前台接待小姐（我们称她为小S），你希望认识她。

这个场景的特殊性在于，这位女生是客户公司的前台，你会因为业务关系多次到访。因此，我们的策略不是一次性出击，而是利用多次见面的机会，逐步建立熟悉感和信任。

---

## 核心原则与策略

面对这类会在固定地点长期工作的女生，最关键的原则是：**不要第一次见面就直接表达爱慕**。

**核心公式**：`搭讪成功率 ≈ 熟悉度 × 自然度 × 态度`

在景气不好的环境下，这类工作通常比较稳定。如果你第一次见面就说“小姐你很漂亮，可以认识你吗？”，很可能会失败。你需要做的是增加对方对你的熟悉度。

既然你知道她会长期在这里工作，而你需要多次拜访客户，你就可以将一次完整的“认识过程”分解到多次拜访中完成。

以下是具体的行动步骤：

1.  **第一次接触**：仅完成必要的业务对话（例如请她帮忙通报）。目的是留下一个正常、专业的初次印象。
2.  **后续接触**：在每次业务对话之外，尝试多聊一两句。话题可以关于工作、是否忙碌、是否是学生兼职等。
3.  **关系升级**：在第三次或第四次接触时，可以尝试询问对方的姓名、电话或电子邮箱。

**关键代码逻辑**：
```python
if 见面次数 == 1:
    对话内容 = 纯粹业务沟通
elif 见面次数 <= 3:
    对话内容 = 业务沟通 + 简短寒暄(工作、天气等)
else:
    对话内容 = 业务沟通 + 个人信息交换(姓名、联系方式)
```

记住，如果你不主动索取联系方式，对方几乎不可能主动给你。

## 实例演练与分析

现在，让我们跟随超级业务员阿德，看看他如何分三步与前台美女小S成功搭讪。

### 第一周：留下印象

阿德第一次见到小S，内心激动，但遵循了“第一次不多说”的原则。

**对话示例**：
> 阿德：你好，我要找简单移动的Lawrence先生，请问他在吗？
> 小S：请稍坐，我帮你通报。
> 阿德：好的，谢谢。

**分析**：阿德做得对。第一次见面只完成必要流程，为后续互动打下基础。他决定下次要多说几句话。

### 第二周：开启寒暄

一周后，阿德借拜访客户之机再次前来，主要目的是与小S进行更多交流。

以下是阿德第二次的对话改进示例：

> 阿德：嗨，又见面了，我还是来找Lawrence，麻烦你通报一下。
> 小S：好，请稍等。
> 阿德：（利用等待时间）对了，你们这边有几位员工？我好像每次来都只看到你。
> 小S：我们秘书有三位。
> 阿德：真的啊？那你都是刚好值班吗？因为我觉得你上次服务态度和谈吐都很好。Lawrence还没出来，可以跟你聊一下吗？
> 小S：可以啊。
> 阿德：这是你的全职工作吗？做多久了？你是属于101公司，还是简单移动这边的？

**分析**：阿德利用等待时间开启对话，表现很好。但有一个可以改进的地方：他连续提问，像在审讯。更好的方式是**先分享自己的信息，再换取对方的信息**。

**改进建议**：在问问题前，可以先说：“我因为和简单移动有业务往来，所以常过来。你是这里的秘书吗？还是属于哪家公司？”这样对方会更愿意分享。

### 第三周：获取联系

第三周，阿德准备完成最后一步：获取联系方式。

**对话示例**：
> 阿德：第三次见面了，一直没请教怎么称呼你？
> 小S：我是Sylvia。
> 阿德：Sylvia你好，我是Daniel。我常来这边找Lawrence，挺巧每次都是遇到你。可以跟你交换个名片吗？
> （交换名片后，阿德查看名片）
> 阿德：这上面是公司邮箱吧？你个人常用MSN或Email吗？因为我下周可能一段时间不过来，没见到你觉得怪怪的，可以认识一下，要个MSN吗？
> （获得MSN后）
> 阿德：对了，顺便留一下你的手机号吧？我下周有个朋友在国父纪念馆隔壁有演出，如果你下班有空，我们可以先吃个饭再去看表演。

**分析**：这次接触是成功的关键。我们从中可以总结出几个要点和注意事项：

1.  **态度要自然，避免过度礼貌**：不要说“可以请教一张名片吗？”，直接说“可以跟你交换一下名片吗？”更显大方。过度小心翼翼反而会让对方觉得奇怪或缺乏男子气概。
2.  **利用现有信息延伸话题**：拿到名片后，可以就上面的信息（如职位、公司）展开聊天，例如：“你这么年轻就是经理了？”
3.  **巧妙索取私人联系方式**：如果名片上没有个人联系方式，可以像阿德一样，以“方便以后联系”或“分享活动”为由，自然地问询MSN、个人邮箱或电话。
    *   **技巧示例**：笑着问“你们公司上班可以用MSN吗？”根据回答顺势往下聊，并最终交换联系方式。
4.  **提出具体邀约作为理由**：以朋友聚会、演出等活动为由邀约，顺理成章地交换电话，为后续真正约会铺路。
5.  **应对戒心**：如果对方只给MSN，可以开玩笑说：“那你要是很少上线，我岂不是永远联系不到你了？还是把电话也给我吧。”用轻松的方式化解尴尬。

## 课程总结

本节课中，我们一起学习了在一个重复出现的固定场景（如业务拜访）中如何策略性搭讪。

**核心要点总结**：
*   **策略**：采用“分次渐进法”，将一次完整的搭讪分解为多次简单的互动，逐步提升熟悉度。
*   **态度**：保持**自然、大方、镇定**。基本的礼貌要有，但避免过度客气和小心翼翼，这反而会降低吸引力。
*   **技巧**：先分享自己，再询问对方；利用现有物品（如名片）展开话题；以具体活动为由，自然地索要私人联系方式。
*   **关键**：你的最终目的不仅是拿到联系方式，更是通过整个过程**吸引对方**。从容不迫的态度本身就是一种吸引力。

记住，搭讪是一门需要练习的艺术。从每一次互动中反思和改进，你会越来越从容。
