# 《搭讪圣经》电台访谈音频：11.搭訕實例演練_note

在本节课中，我们将通过一个具体的业务拜访场景，学习如何循序渐进地与一位前台服务人员（你想认识的女性）建立联系并成功获得联系方式。整个过程强调自然、渐进和态度的把握，避免过于直接或礼貌。

---

## 场景设定

假设你是一名业务员，需要去台北101大楼拜访客户。在你客户公司所在的楼层，有一位在前台服务的女性吸引了你的注意。你想认识她，但直接表达意图可能会适得其反。

## 核心原则：渐进与熟悉度

像这样有固定工作的女性，通常不会轻易离职。因此，最佳策略不是第一次见面就直接表达好感，而是通过多次接触，逐步增加她对你的熟悉度。基本方法是：每次拜访客户时，在必要的业务对话之外，尝试多与她交谈一两句。

**公式**：`搭讪成功率 ∝ 熟悉度 × 自然度`

---

## 第一次接触：留下基础印象

第一次见面时，目标仅限于完成必要的业务沟通，为后续接触埋下伏笔。

以下是第一次对话的示例：

```
阿德：哈囉，小姐你好。我要找一下简单移动的Lawrence先生，不知道他现在在不在。
小S：你稍等，你旁边坐一下，我帮你通报一声。
阿德：OK，好，谢谢。
```

这次接触后，阿德意识到需要更多接触，决定下次拜访时尝试多说几句话。

---

## 第二次接触：开启简短对话

一周后，阿德再次拜访。这次他利用等待客户的时间，尝试与前台小姐小S进行更长的交流。

以下是第二次对话的示例：

```
阿德：欸，哈囉小姐，又是我。今天又见到你了，我还是来找Lawrence，可是还是麻烦你帮我通报一下好了。
小S：好，你稍等一下喔。
阿德：欸，为什么你们这边有几个员工啊？我好像每次来都只看到你。
小S：我们这边秘书有三位。
阿德：真的啊？那你都是刚好值班还是怎么样？因为我觉得，其实上次来你给我的印象很深刻，我觉得你的服务态度跟整个谈吐很好。那Lawrence还没出来，可不可以跟你聊一下呢？
小S：喔，可以啊。
阿德：这是你的全职工作还是？
小S：是啊。
阿德：真的？那你就做几年了？
小S：快两年了。
阿德：你是属于101的这个公司的部分呢，还是简单移动他们这边的？
小S：其实我们是跟101租办公室这样子，然后我们再分租给简单移动。
（此时客户Lawrence出现，对话中断）
```

**分析**：阿德利用时间空档开启对话是很好的做法。但可以改进的一点是：他连续提出了多个问题。更好的方式是先分享自己的信息，再询问对方，这样对话会更平衡自然。

例如，可以先说：“我本身在XX公司，因为跟简单移动有业务往来，所以常过来。你是这边的秘书吗？”

## 第三次接触：交换联系方式

又过了一周，阿德进行了第三次拜访。这次的目标是自然地交换联系方式。

以下是第三次对话的示例：

```
阿德：哈囉小姐，第三次又见到你了。我今天还是来找Lawrence的，可是还是麻烦你先帮我通报一下好了。
小S：好，你稍等喔。
阿德：不好意思今天第三次见到你，可是我一直没有请教怎么称呼你？
小S：我是Sylvia。
阿德：Sylvia你好，我是Daniel。因为跟Lawrence有一些业务往来，所以我经常会到这来。可是也很有意思的，每次刚好都是遇到你。可以跟你交换个名片吗？
小S：好啊，你稍等一下喔。
（交换名片）
阿德：我看了一下，似乎上面并没有你个人用的邮箱？因为这是公司的邮箱吧？
小S：对，这是公司的email。
阿德：我想你个人应该用你自己的email。你常不上MSN之类的？
小S：常啊。
阿德：好吧，因为我想我下个礼拜可能会有一段时间没有过来，所以没有见到你突然觉得怪怪的。可不可以跟你认识一下，然后跟你要个MSN？
小S：喔好，你等一下喔。
（小S写下个人联系方式）
阿德：那可不可以也顺便留一下你的行动电话？因为我想下个礼拜我刚好有个朋友有个演出，在国父纪念馆隔壁。我想如果你下班有空，我们可以先吃个晚餐，然后顺便去看个表演。
小S：好啊。
```

## 关键要点总结

本节课我们一起学习了在一个重复出现的场景中，如何通过三次渐进式接触成功完成搭讪。以下是核心要点：

1.  **态度至关重要**：保持泰然自若、大方镇定的态度，比过分礼貌或小心翼翼更能吸引对方。你的目的是吸引，而不仅仅是获取联系方式。
2.  **信息交换应对等**：在询问对方信息前，先主动分享自己的相关信息，能使对话更自然，减少被质问的感觉。
3.  **利用话题与情境**：
    *   利用名片上的信息（如职位）展开话题。
    *   通过提出具体的后续活动邀约（如看表演），自然地索要电话号码等更直接的联系方式。
4.  **索要方式要灵活自然**：避免使用“可以请教一下…”这类过于客套的说法。直接、自然地提出“我们可以交换一下名片吗？”或“那留个电话方便之后约时间？”会更有效。
    *   **代码示例（心态）**：`if (对方只给MSN): 幽默地说：“那你要是网络上的死人，我怎么办？电话给我吧。”`

记住，搭讪的每一句话和每一个动作都应服务于最终目标：在获得联系方式的同时，建立起吸引力和舒适感。
