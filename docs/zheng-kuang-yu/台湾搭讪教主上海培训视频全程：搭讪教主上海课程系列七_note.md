# 搭讪课程07：电梯场景搭讪全攻略 🛗

在本节课中，我们将学习如何在电梯这个特殊且常见的场景中进行有效搭讪。电梯空间封闭、时间短暂，掌握正确的策略至关重要。

## 概述：电梯搭讪的核心场景

许多上班族每天早晨会在同一栋办公大楼里，遇见其他公司的异性。想要认识对方，电梯便成了一个关键场所。

## 常规流程：建立“好人”形象

上一节我们概述了场景，本节中我们来看看具体操作。核心策略是：**早到公司，关注大门，目标出现后立即跟随进入同一部电梯**。

进入电梯后，有一个关键诀窍：
1.  电梯门打开时，应抢先进入。
2.  主动站到控制面板前，询问并帮助所有人（包括目标）按下楼层按钮。
3.  最后按下自己的楼层。

这个动作的目的是**营造一个乐于助人的正面形象**。公式可以概括为：**抢先进入 + 帮助按楼层 = 建立初始好感**。

## 多次相遇的渐进策略

如果对方与你在同一栋大楼工作，意味着你们有多次相遇的机会。这时可以采用渐进策略。

以下是操作步骤：
*   **第一周**：重复几次“帮助按楼层”的动作，混个脸熟。
*   **下一周**：在同样帮她按完楼层后，可以开启对话。例如：“最近每天早上都看到你，你在哪家公司？” 进而介绍自己的公司。
*   **后续**：因为会常见面，可以在一次聊天中问名字、聊工作，下一次就可以尝试索取私人联系方式。

## 一次性机会的紧急应对

但有时，你帮助对方按了楼层并开始聊天后，可能发现对方只是临时来访（例如“来看朋友”）。这意味着这是第一次也是最后一次见面机会。

此时必须当机立断，把握唯一机会。具体方法是：
1.  当她到达楼层（例如7楼）准备说“再见”并走出电梯时。
2.  你应跟随出去，并直接表明：“我知道你只是来看朋友，之后可能不会来了，但我真的很想认识你，所以先跟你出来聊一下。”
3.  之后便可以用主导话题的方式，直接交换基本信息（如姓名、工作），完成搭讪。

## 实战案例分析与应变技巧

在掌握了基本策略后，我们通过一个真实案例来看看如何应对复杂情况。我曾在一栋楼的12楼（一家桌游社）遇到一位女生。

当时场景很棘手：我出电梯，她进电梯。我立刻寻找关联点，问道：“你好，你是来这边玩桌游的吗？”（因为该楼层主要是桌游社）。她回答“是”，对话得以开启。

但当时我们卡在电梯门处，门来回关合撞了她几下。我立刻做出决定：如果犹豫，她就会进入电梯离开。于是我马上说：“不好意思看你被门撞了三次，我们出来聊一下吧。” 顺势将她带离电梯口。

聊天几分钟后，障碍出现：她的朋友找来叫她。我的思路非常明确：**已经开始就不能白费**。我立刻对她说：“不然让你朋友先走吧，我们再聊几分钟就好。” 我说“三分钟”是为了让她感到时间有限、不会耽误太久，结果我们愉快地聊了十分钟，并顺利交换了联系方式。

在这个案例中，聊天内容围绕：
*   我来这里的目的（办活动）。
*   她来这里的原因（公司 nearby，例如阿里巴巴）。
*   一些轻松的话题（如公司是否要搬家）。

## 总结

本节课我们一起学习了电梯搭讪的完整方法。关键在于**主动创造帮助他人的机会以建立良好第一印象**。根据是**多次相遇**还是**一次性机会**，灵活采用渐进策略或果断出击。即使场景尴尬或出现突发障碍（如朋友介入），只要**目标明确、当机立断**，并善用**时间限定话术**（如“再聊三分钟”），就能大大提高成功率。搭讪并没有想象中困难，关键在于掌握方法并勇敢实践。
