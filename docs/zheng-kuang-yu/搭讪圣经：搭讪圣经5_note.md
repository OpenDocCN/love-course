上一节我们探讨了“花心”的本质，本节中我们来看看当你的搭讪行为被他人指责时，具体该如何处理。

## 回应女性的质疑 💬

当你在搭讪时，心仪的女性可能会反过来质问你。以下是应对这种情况的核心策略。

**核心论点**：优**质的人本来就应该互相认识**。任何关系都应从友谊开始。

以下是具体的回应步骤：

1.  **坦然承认并设定关系基调**：当对方问“那你一定会跟其他女生搭讪咯？”，你可以理直气壮地回答：“对呀，优质的人本来就应该互相认识。我现在认识你，是把你当成一个朋友。”

2.  **强调渐进过程**：说完后，可以拍拍她的肩膀（如果氛围允许），告诉她：“别急，朋友要慢慢互相认识。我现在只是开一个头，我们还有很多时间互相了解。”

3.  **应对“花心”指控**：如果对方指责你搭讪就是“花心”，请运用上一课的论点进行反问。你可以问她：“那我想请问，你只会让一个男生追你吗？如果你不是只让一个男生追你，那你又凭什么不让我去认识其他女生呢？”

4.  **提出直接方案**：你还可以接着说：“如果你只让我一个人追你的话，好，那我们就两个人直接开始交往。我也只对你一个人好，也只跟你一个人讲话，这样可以吧？”

通过以上对话，你既保持了框架，又将关系的选择权巧妙地交还给了对方。

处理了女性的质疑后，另一个常见困扰来自朋友圈。下面我们来看看如何应对男性朋友的不支持甚至嘲笑。

## 处理朋友的负面评价 👥

很多男性朋友自己渴望认识女性，却会嘲笑或阻止你去搭讪。面对这种情况，你需要果断处理。

**核心原则**：你应该结交**志同道合**、愿意为爱情努力寻找出路的朋友，而非打击你勇气的人。

以下是具体的判断与行动指南：

1.  **识别不良朋友**：请立刻与那些嘲笑你、骂你、阻止你成功的“朋友”绝交。这样的人不仅不是好人，更不可能是好朋友。

2.  **进行终极反问**：你可以请他扪心自问：“当你看到一个自己想认识的女生时，会不想去跟她说话吗？”答案必然是肯定的。

3.  **认清本质**：他是一个没有勇气的人，并且喜欢破坏你的努力、嘲笑你、阻止你成功。这样的人不是真正的朋友。

4.  **主动筛选圈子**：你应该去结交那些和你一样没有女朋友，并认为应该努力为自己爱情寻找出路的人。与积极向上的人为伍。
