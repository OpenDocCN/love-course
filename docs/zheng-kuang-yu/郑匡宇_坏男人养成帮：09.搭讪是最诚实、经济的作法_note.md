# 郑匡宇_坏男人养成帮：09.搭讪是最诚实、经济的作法_note

在本节课中，我们将探讨网络交友的局限性，并学习为什么直接搭讪是更诚实、更经济的认识心仪对象的方式。我们将分析网络交友的常见问题，并与现实生活中的互动进行对比。

## 概述

网络交友常被描绘为结识优质异性的便捷途径，但实际体验往往令人失望。本节将揭示网络交友的陷阱，并论证直接搭讪的优越性。

## 网络交友的幻想与现实

上一节我们介绍了认识异性的不同途径，本节中我们来看看网络交友这一具体方式。许多人受电影或电视剧影响，幻想通过网络结识优质女性并发展浪漫关系。

**信念17**：好人总是幻想电影和电视剧的美好情节会出现在自己身上，因而大量依赖网络交友，企图制造网络情缘。

然而，现实情况往往截然不同。尝试过网络交友的人通常对以下情形不陌生。

以下是网络交友常见的几个问题：

1.  **流程繁琐且需付费**：在交友网站看到心仪的异性后，首先需要注册会员。注册后，往往还需付费才能主动向对方发送信息。
2.  **回应率极低**：即使顺利发送了信息，也常常石沉大海，得不到任何回复。
3.  **择友条件苛刻**：心仪的女性常设定过高要求，例如身高175厘米以上、大学学历、月收入六万以上等，令多数普通男性望而却步。
4.  **交友目的不明确**：有些女性会在个人描述中直接声明“只交朋友，不交男朋友”或“已有男友”，仅想聊天，拒绝提供联系方式。

看到这样的描述，难免令人心灰意冷。网络世界对很多人而言既好奇又令人害怕。许多女性在网络交友时并非真心寻找伴侣，而是抱有强烈的防卫心态。

## 搭讪：更诚实与经济的做法

既然不应完全依赖网络交友，那么什么才是更好的方式呢？如果你接受搭讪是一种更有效的交友方式，并付诸实践，就会发现现实与网络的巨大差异。

坏男人知道，网络应该只是用来了解对方的工具。**直接用搭讪的方式，来和心仪的女生认识，才是最诚实、最经济的做法。**

在现实生活中，许多女性并不像网络上那样设定诸多苛刻条件。当你通过搭讪直接认识对方时，她能够通过你的**外表、谈吐、风度与气质**来综合判断是否愿意与你交朋友。这就是“见面三分情”。

相比之下，网络上对方无法真实地看到你，只能通过设定各种条件来筛选，试图排除她认为不合适的人。

## 行动建议：提升自我并主动出击

与其幻想虚无的网络情缘，不如采取实际行动。

以下是两个核心的行动方向：

1.  **提升自我**：无论是**外型衣着**，还是**讲话的幽默感与气质**，都需要持续改进。
2.  **主动搭讪**：看到心仪的异性，直接上前交谈，留下好印象，然后自然地索要**MSN或电话**等联系方式，再逐步加深了解和信任。

我认为这才是最实际、最有效、最经济的交友方式。不要在网络上继续浪费时间。两个未曾谋面的人先投入大量感情，见面后却可能因容貌不符预期而互相嫌弃，这对双方而言都是一种可怕且不公平的经历。

## 关于“公平”的信念

在男女交往中，许多男性会对“公平”问题感到困惑与不满，例如总是自己主动联系、承担约会费用等。

**信念18**：好人在对待女生时，倾向于在原则和价值观之间摇摆，不知所措。坏男人知道，坚定自己的立场，但也了解男女之间的差异，才能得到女孩子的喜爱。

除非你是天生的帅哥或条件极其优越，否则在交往初期，男性多付出一些是常态。然而，这种付出不是盲目的。

**每一个付出，都要经过精密计算，都要让自己与对方在生活上、情感上产生关联。** 这样的付出才有效益，才可能让对方真正喜欢上你。

在男女交往中，有一个重要观念：**不要跟女生讲“公平”**。当女生问你“为什么对我这么好”时，不要回答“所以你知道该怎么做了吧”。这看似公平，却无法赢得芳心。

正确的回应是给予温暖、欣赏的反馈。例如，你可以握着她的手说：“因为你很好，我不对你好要对谁好呢？” 或 “我一直记得父母的话，人往高处走。我相信和优秀的人在一起，我才能全心付出。”

最好的交往状态是，双方都了解并接受彼此在生理和心理上的差异，同时欣赏这些差异所带来的吸引力。

## 总结

本节课中我们一起学习了网络交友的局限性，并认识到直接搭讪是更诚实、更经济的认识异性的方式。我们分析了网络交友的常见陷阱，对比了现实互动的优势，并给出了提升自我与主动出击的行动建议。此外，我们还探讨了男女交往中关于“公平”的信念，指出在交往初期理解差异、智慧付出比追求表面公平更为重要。

除了这些信念，还有更多成为魅力坏男人的信念等待我们去发掘。
