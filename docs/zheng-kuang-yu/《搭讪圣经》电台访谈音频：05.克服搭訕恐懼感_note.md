# 《搭讪圣经》电台访谈音频：05.克服搭訕恐懼感_note

在本节课中，我们将学习如何系统性地克服与陌生人，特别是心仪异性交谈时的恐惧感。我们将通过一个循序渐进的三个月训练计划，以及多种辅助练习方法，帮助你建立自信，让搭讪变得自然顺畅。

---

上一节我们介绍了搭讪前的基本礼仪。掌握礼仪能让过程更顺利，但许多人仍感到搭讪是巨大考验，存在莫名恐惧。本节中我们来看看如何克服这些恐惧感，让搭讪更顺利成功。

要克服所谓搭讪的恐惧，在心理建设之后，还可以在其他部分加强自信，克服与异性说话时的恐惧。具体做法是以三个月为单位，慢慢克服与自己想认识的异性说话的恐惧。

以下是针对完全没有搭讪经验、完全不敢与异性说话的人设计的三个月训练计划：

**第一个月：建立眼神接触的勇气**
这个月里，请对每一个你有点兴趣、想认识的女生，开始做四目相接、点头微笑的动作。例如在公车上、早餐店或捷运站，只要看到任何有兴趣的异性，请想办法与对方眼神相对。对到之后，点头微笑，然后继续做自己的事。这个月只需完成这个动作，目的是克服与陌生人，特别是有兴趣的异性进行眼神接触的恐惧。

**第二个月：从眼神到简短问候**
这个月，请在做同样眼神接触和点头微笑后，增加说一声“你好”。说完后，同样不再继续说话。这样做的好处是，你进一步克服了不敢与对方面对面说话的恐惧。有时，对方可能会疑惑地回应。如果对方主动说话，你可以根据自身情况选择是否接话；如果不知如何回应，用微笑带过即可。这个月训练的目的是敢于说出第一句问候。

**第三个月：从问候到开启对话**
这个月，在四目相接、点头微笑并说完“你好”之后，请再加一句话。这句话可以是称赞，或是借助对方身上的物品、正在做的事情来开启对话。例如，对方在看报纸，你可以问“今天股票是涨还是跌？”；或节日临近，对方戴了相关饰品，你可以称赞“你这个帽子很漂亮，很有节日气氛”。这样做的目的是通过释放善意，自然地开启一段对话。

---

以上是在外可以进行的训练。在家里，你还可以通过以下方法加强练习：

**对着镜子练习**
许多成功的演讲者或主播都常对着镜子练习。你可以假设一个搭讪场景，闭上眼睛想象，然后睁开眼睛对着镜中的“对方”说话。注意观察自己是否会结巴、手不知该放哪里。通过不断练习来修正这些举止。

**与朋友互相练习**
你可以与好朋友约定，互相扮演搭讪者与被搭讪者的角色进行练习。要求是认真投入，因为现实中对方不会一开始就对你笑嘻嘻。与真人练习比对着镜子练习临场感更强，是很好的训练方式。

**寻找榜样（Role Model）**
很多人与异性说话时不知所措，是因为没有模范典型。你可以观察电视剧或电影中的男主角是如何与女性自然交谈的。例如《欲望城市》中的男性角色或汤姆·克鲁斯的电影。记住他们的形象和态度，幻想自己就是主角，思考在类似场景下你会如何说话和行动。

---

最后，需要强调的是，并不存在一句“必杀绝句”就能保证搭讪成功。搭讪的核心并非技巧堆砌，而是提升自信、口才以及面对挫折的能力。当你个人提升到一定程度后，搭讪会成为一个非常自然的认识人的方式。

本节课中我们一起学习了克服搭讪恐惧感的系统性方法，包括为期三个月的渐进训练计划，以及在家中可以进行的镜子练习、朋友互练和寻找榜样等方法。关键在于持续练习与自我提升，让搭讪成为你自信、自然社交的一部分。
