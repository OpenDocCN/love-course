# 恋爱心理学 第1课：什么是废物测试？ 🧪

在本节课中，我们将要学习恋爱社交中的一个重要概念——“废物测试”。理解并妥善应对它，对于建立健康的人际关系至关重要。

所谓废物测试，就是对方会故意采用质疑、打压、否定、说教或沉默等方式让你感到难堪。其目的是测试你在这种高压情况下的反应。

因此，如果在面对对方的废物测试时，你展现出暴怒或者失态，那就正中对方下怀，对方很可能会立刻疏远你。

因为对方会由此判断，你是一个无法妥善处理逆境和管理自身情绪的人。如果你连自己的情绪都照顾不好，未来就更不可能照顾好对方的情绪和感受。

---

上一节我们介绍了废物测试的基本定义和目的，本节中我们来看看它的核心逻辑与应对本质。

废物测试的核心在于**压力筛选**。其逻辑可以概括为一个简单的公式：

**你的反应 = 对方判断你情绪稳定性和成熟度的依据**

面对测试时，你的反应比测试内容本身更重要。沉着、幽默或从容不迫的回应，能通过测试；而被激怒、辩解或乞求认同，则会导致测试失败。

以下是应对废物测试时需要牢记的几个关键点：

*   **保持框架**：不要被对方的话语带偏节奏，坚持自己的态度和原则。
*   **化解而非对抗**：可以使用幽默、曲解或反问等方式，轻松化解对方的刁难，而不是直接冲突。
*   **展现高价值**：通过稳定的情绪和自信的姿态，证明你是一个内在强大的人。

---

本节课中我们一起学习了“废物测试”的概念。我们明白了它是一种通过施加压力来观察对方反应的行为，其根本目的是筛选出情绪稳定、内心强大的伴侣。记住，应对的关键不在于言语上的胜负，而在于你是否能始终保持从容与自信。掌握这一点，你就能在人际交往中更加游刃有余。
