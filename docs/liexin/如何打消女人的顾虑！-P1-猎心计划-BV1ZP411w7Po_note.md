# 课程名称：复格技术详解 - P1 🎯

在本节课中，我们将要学习一种名为“复格”的沟通技术。复格的核心在于，通过为你的言行提供一个合理的理由或解释，来引导对方的感受和决策，使其更易接受你的意图。我们将探讨复格在三种常见场景下的具体应用。

## 概述：什么是复格？ 🤔

复格是一种沟通策略，其核心公式可以概括为：**行为 + 理由 = 合理化**。它通过为你即将采取或已经采取的行动附加一个原因，来打消对方的疑虑、降低你的需求感，或使你的请求显得更自然。

## 应用一：打消对方顾虑 🛡️

在人际交往中，当你提出一个请求或邀约时，对方可能会产生顾虑。复格的作用就是预先提供理由，消除这些不确定性。

以下是使用复格打消顾虑的具体方法：

*   **邀约场景**：你想约对方见面，对方可能担心目的地或你的为人。此时，复格可以提供明确的邀约理由。
    *   **示例**：“明天有一家新开的海鲜店，听说特别好吃，要不要一起去尝尝？”

*   **表达好感场景**：当你向对方表达喜欢时，对方可能会疑惑“为什么喜欢我”。复格可以提供喜欢对方的具体原因，合理化你的情感。
    *   **示例**：“这段时间相处下来，我发现你不仅会认真打理生活，还特别善解人意，这一点非常吸引我。”
    *   **技术结合**：此处的复格常与“天平”技术（后续课程会讲解）结合使用，通过指出对方具体的内在品质，而不仅仅是外在表现，使表达更具穿透力。

## 应用二：被动社交开场 🎭

上一节我们介绍了如何用复格打消顾虑，本节中我们来看看如何用复格开启一段社交。主动开启对话有时会显得需求感过强，而“被动社交”复格能将开启对话的责任归因于外部因素或对方的行为。

以下是使用复格进行被动社交开场的技巧：

*   **核心思路**：将“我想找你”转化为“因为某个情况，我不得不/很自然地找你”。
*   **对比示例**：
    *   **无复格（主动）**：“在干嘛呢？”或“最近忙什么呢？”
    *   **有复格（被动）**：“最近看你好久没发朋友圈了，你没事吧？”
*   **效果分析**：后者的理由是基于“对方没发朋友圈”这一行为，使你看起来是出于关心而被动开启对话，这能有效降低你的需求感，并掌握关系推进的主动权。

## 应用三：规划行为原因 🔄

复格不仅能用于对话开场，更能深度规划你一切行为的原因。无论是你的付出还是索取，都可以通过复格，将原因归结到对方、自己、双方关系或第三方身上，从而使行为更容易被接受。

以下是四种不同的归因方向及示例：

*   **归因于对方**：将你的请求与对方的责任或承诺挂钩。
    *   **示例**：“你说你爱我，可连我最喜欢的手表都不愿意送我，这让我怎么感受到你的爱呢？”

*   **归因于自己**：将你的请求与自身的经历、性格或情感需求联系起来。
    *   **示例**：“我从小很少收到礼物，所以特别希望喜欢的人能送我礼物。礼物是什么不重要，重要的是这个行为能让我确信自己值得被爱。”

*   **归因于双方（我们）**：将你的请求与你们共同的关系或未来绑定。
    *   **示例**：“我现在戴的这块表是前任送的。我不希望留下关于过去的物品，只想要充满我们共同回忆的东西。”

*   **归因于第三方（其他人）**：利用外部环境或他人的行为作为理由。
    *   **示例**：“明天我生日，朋友们都准备了礼物。我在想，你会不会送我那块我最喜欢的手表呢？”

在实际应用中，你需要根据对象和情境灵活选择归因策略，但其底层逻辑不变：**通过复格，为你的一切行为提供一个合理化的解释**。

## 总结 📝

本节课中，我们一起学习了复格技术的核心与应用。我们了解到，复格是通过 **“提供理由”** 来合理化行为、引导感知的沟通工具。具体包括：1）**打消对方顾虑**，为邀约或表白提供具体理由；2）**被动社交开场**，将开启对话归因于外部因素；3）**规划行为原因**，将付出或索取的原因巧妙地归因于对方、自己、双方或第三方。掌握复格，能让你的沟通显得更自然、更有说服力。
