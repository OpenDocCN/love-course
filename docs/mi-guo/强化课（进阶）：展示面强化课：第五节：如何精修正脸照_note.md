# 📸 强化课（进阶）：展示面强化课：第五节：如何精修正脸照

在本节课中，我们将学习如何精修一张正脸人像照片。我们将使用美图秀秀，通过一系列步骤，将一张普通的自拍照，修整为一张精致、有质感的肖像照。课程将涵盖从基础调整到细节打磨的全过程。

---

上一节我们学习了优秀照片案例的赏析，本节中我们来看看如何亲手将一张普通的正脸照变得出众。

首先，我们需要明确本节课的核心：**精修脸部**。这意味着照片中的人物面部是绝对主体，修图的重点在于让五官更立体、皮肤更干净、整体气质更突出。

## 🛠️ 准备工作与软件选择

以下是开始修图前需要了解的几个要点。

**核心工具**：本节课全程使用 **美图秀秀** 应用。经过对比多种软件，美图秀秀在手动调整和细节处理上最为方便有效。

**拍摄建议**：
*   使用苹果手机的后置摄像头拍摄，质感更好。
*   如需自拍，请使用美图秀秀内的相机功能，而非手机原装前置摄像头，效果更佳。
*   选择简洁的背景（如一面白墙、一扇门），穿着普通的衣服即可，重点在于面部。

## 📝 核心修图流程详解

我们将修图流程分为几个大的阶段，每个阶段都有其明确的目标。

### 第一阶段：基础优化与面部结构调整

在导入照片后，我们首先进行整体性的基础处理。

1.  **去除软件标识与水印**：检查并去除照片角落的美图秀秀等logo，让照片看起来更自然。
2.  **智能优化 - 去雾**：点击“智能优化”，选择“去雾”功能。这个操作可以消除因光线或镜头造成的模糊感，让照片更通透。注意根据照片实际情况调整强度，避免过度。

完成基础优化后，我们进入重塑五官的关键步骤。

3.  **面部重塑**：点击“去美容”，找到“面部重塑”功能。
    *   **3D素颜**：首先使用此功能微调面部在画面中的上下、左右位置，使脸部居中。
    *   **调整脸型**：依次调整下巴（收窄）、脸宽（内收）、额头（上提）。额头稍上提可以视觉上增加发量。
    *   **调整眉毛**：降低眉高，使眉毛靠近眼睛，增加五官立体感。根据个人情况调整眉间距。眉倾斜角度建议呈“内八”状，显得更有神。
    *   **调整眼睛**：增大眼睛尺寸和眼高。调整眼距至协调，并微调眼睛倾斜度。
    *   **调整鼻子**：通常缩小鼻翼、提升鼻梁高度、修饰鼻尖（使其更精致）。
    *   **调整嘴唇**：根据大小调整唇宽，提高唇高可以使五官显得更紧凑。

**操作核心**：在此过程中，要养成**不断对比原图**的习惯，确保调整自然，避免失真。

### 第二阶段：手动精修与细节打磨

面部结构调整好后，我们需要进行更精细的手动修饰。

上一阶段我们搭建了好的面部框架，本节中我们使用“瘦脸瘦身”工具进行精雕细琢。

4.  **瘦脸瘦身（手动推脸）**：
    *   **轮廓**：推入颧骨，收尖下巴，修饰下颌角（使其清晰），让脸型从圆润变为立体。
    *   **细节五官**：
        *   **眉毛**：修饰眉形，使其整齐有型。
        *   **眼睛**：拉长外眼角，增大黑眼珠，微调内眼角。这相当于完成了“开外眼角、开内眼角、眼下至”的视觉手术。
        *   **鼻子**：修饰鼻孔形状为椭圆形；推高鼻翼，缩小鼻翼宽度；从两侧向内推鼻梁，使其更窄更挺；提亮山根（鼻梁上部）阴影，增加立体感。
        *   **嘴巴**：简单修饰唇形，使其规整。

5.  **皮肤处理**：
    *   使用“去黑眼圈”功能，涂抹面部痘印、暗沉、法令纹等瑕疵。此功能能淡化瑕疵而不像磨皮那样失真。
    *   同样用“去黑眼圈”工具，在鼻梁中央画一道，可以充当“高光”，让鼻梁更挺。
    *   使用“消除笔”点除明显的痣或杂物。

### 第三阶段：整体润色与氛围营造

细节修饰完毕后，我们为照片注入灵魂——色彩与氛围。

6.  **身体修饰**：不要忽略脖子和肩膀。收细脖子，拉宽肩膀，可以立刻提升整体气质和安全感。

7.  **添加美妆（谨慎使用）**：
    *   点击“美妆”功能，**切勿使用一键美妆**。
    *   **口红**：选择最淡雅的颜色（如豆沙色系），强度调到10%-20%，仅用于提升气色。
    *   **眉毛**：选择一款有型但不过分的眉形，轻微加深。
    *   **五官立体**：使用此功能，轻微增强面部阴影和高光，让轮廓更分明。

8.  **叠加滤镜与调色**：这是提升照片“高级感”的关键。
    *   **滤镜叠加**：按顺序添加以下滤镜，每个强度不宜过高（通常10%-30%）。
        1.  自然 - **MOE**
        2.  自然 - **MO2**
        3.  质感 - **M18**（关键，显著提升质感）
        4.  质感 - **M1**
        5.  质感 - **MM3**（增加暗调高级感，强度5%-10%）
    *   **增强调整**：点击“增强”，进行手动微调。
        *   **亮度**：+20
        *   **对比度**：+20
        *   **高光**：-10
        *   **暗部**：-5
        *   **饱和度**：-10（营造高冷感）
        *   **色温**：-20（偏冷色调）
        *   **色调**：-5（轻微增加紫粉色）
        *   **结构**：+15（**非常重要**，替代锐化，让照片清晰有质感）

9.  **最终检查与收尾**：
    *   再次检查发型，用“瘦脸瘦身”整理过长的或杂乱的头发。
    *   使用“亮眼”功能，涂抹眼睛、眉毛、嘴唇，让局部更有神采。
    *   最后，保存前再次与**原图对比**，确保变化显著但自然。

## ✅ 总结

本节课中，我们一起学习了精修正脸人像照片的完整流程。我们从**选择工具与拍摄**开始，经历了**面部结构重塑**、**五官细节手动精修**、**皮肤瑕疵处理**，最后通过**添加微妆效、叠加滤镜和精细调色**，完成了整体氛围的营造。

**核心公式**可以概括为：**基础优化 + 结构重塑 + 细节精修 + 肤色净化 + 氛围调色 = 一张精致的正脸照**。

记住，修图的目的是提升美感，而非换头。关键在于**反复对比**，**微量调整**，让一切变化看起来合理自然。多练习几次，你就能掌握这套“换脸术”，让自己在照片中的形象焕然一新。下节课，我们将探讨如何精修侧脸照片。
