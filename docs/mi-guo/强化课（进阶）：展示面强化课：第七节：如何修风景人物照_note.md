# 强化课（进阶）：展示面强化课：第七节：如何修风景人物照_note

在本节课中，我们将要学习如何修整一张远景风景人物照。我们将从身材比例调整开始，逐步深入到面部细节修饰、背景杂物清理，最后通过滤镜和色彩增强，将一张普通的雪景照修出高级感。整个过程将使用“美图秀秀”App完成，核心在于理解并运用“瘦脸瘦身”、“消除笔”和“滤镜增强”这三大功能。

## 概述

上一节我们学习了如何修整近照的脸部和侧脸。本节中，我们来看看如何处理一张包含人物和广阔风景的远景照片。这类照片的修图重点在于整体氛围的营造和细节的和谐统一。

## 第一步：基础优化与身材调整

首先，在美图秀秀中打开需要修整的风景人物照。

以下是基础处理步骤：
1.  **智能优化**：点击“智能优化”，对照片进行初步的自动校正。
2.  **去雾**：点击“去雾”功能。对于风景照，尤其是远景，去雾能显著提升清晰度。建议至少使用两次。
3.  **进入“人像美容”**：点击右下角的“去美容”（或“人像美容”）功能。

进入“人像美容”后，我们首先调整身材比例。

以下是身材塑形的核心步骤：
1.  **拉长腿部**：在“增高塑形”或“美体”类目下找到“美腿”功能。将画笔对准从臀部到脚踝的区域，适度拉长。注意不要一次拉得过多。
2.  **拉长全身**：使用“增高”或“全身塑形”功能，将整个人物的身材向上拉高，使比例更协调。
3.  **调整小腿**：再次使用“美腿”功能，重点微调小腿线条，使其与拉长后的大腿比例和谐。
4.  **调整背部与肩膀**：使用“瘦脸瘦身”功能。
    *   **推直背部**：将人物略显驼背的线条向内推，塑造挺拔感。
    *   **提升肩膀**：将肩膀向上方微微推动，提升气质。
    *   **平衡比例**：因拉长腿部可能导致上身显短，可适当将腰线部位向上微调。

## 第二步：面部细节精修

由于是远景，面部可能无法被“面部重塑”功能自动识别，我们需要手动使用“瘦脸瘦身”功能进行精细调整。

以下是面部修整的关键点：
1.  **调整头发**：用“瘦脸瘦身”将前额和后脑的头发推蓬松，增加发型的厚度和立体感。
2.  **塑造下颌线（并脚）**：这是亚洲男性修图的重点。将下颌角区域向下、向外微微推动，塑造更清晰、立体的下颌轮廓。
3.  **调整额头与鼻子**：
    *   **额头**：若额头显窄，可向两侧推宽。
    *   **鼻子**：塑造鼻梁弧度。将鼻尖向前推翘，鼻梁中部可稍向内收，形成自然曲线。若鼻子显大，用大范围的“瘦脸瘦身”将其整体向内收缩。
4.  **提升面部肌肉**：将因角度问题下垂的面颊肌肉向斜后方推，让面部更紧致。
5.  **调整眉眼**：
    *   **眉毛**：将眉毛向下推动，缩短眉与眼的距离，使五官更紧凑。
    *   **眼睛**：将图片放大，用小范围的“瘦脸瘦身”工具，将眼睛轮廓向外微微扩大，实现“放大”效果。同时可以调整眼尾、眼角的角度。
6.  **平滑肌肤**：使用“去黑眼圈”功能，手动涂抹在面部皱纹、法令纹或不平整处，能有效平滑肌肤且效果自然。此功能也可用于提亮鼻梁，制造高光。
7.  **使用“眼睛放大”功能**：如果手动调整不够，可直接使用“眼睛放大”工具进行点击放大，然后再用“瘦脸瘦身”做微调。
8.  **修饰脖子**：用“瘦脸瘦身”将脖子线条向内侧推，使其更修长、有气质，同时可将衣领或帽子调整得更规整。

## 第三步：净化背景与营造意境

人物修整完毕后，我们需要处理背景，移除杂物，突出风景。

以下是净化背景的步骤：
1.  **使用“消除笔”**：在工具栏中找到“消除笔”。这是清理背景杂物的神器。
2.  **耐心涂抹**：将画笔大小调至合适（通常可先调大），耐心地涂抹想要消除的物体，如多余的人物、栏杆、杂物等。
3.  **利用智能填充**：“消除笔”会自动计算并填充被抹除区域的背景，使其与周围环境融合。如果一次效果不佳，可撤销后重复涂抹或缩小画笔进行精细处理。
4.  **原则**：目标是让背景只剩下纯净的雪山、天空和雪地，营造空旷、高级的意境。

## 第四步：滤镜调色与最终定稿

最后一步是为照片注入灵魂——色彩与光影。

以下是滤镜与增强的流程：
1.  **应用滤镜**：点击“滤镜”。讲师个人常用的是 **MOE** 和 **MO2** 系列滤镜。可依次尝试，强度通常设置在 **30%-40%**。
2.  **尝试“质感”滤镜**：可尝试“质感”类滤镜，但注意可能会让画面变暗，需谨慎调整强度。
3.  **进行“增强”调整**：这是关键步骤，点击“增强”。
    *   **亮度**：大幅提高亮度，营造雪地阳光感。
    *   **对比度**：提高对比度，让画面更通透。
    *   **高光/阴影**：降低高光，提高阴影，平衡光比，突出细节。
    *   **色彩**：
        *   **饱和度**：适当提高饱和度，让人的肤色和环境的色彩更鲜活。
        *   **色温**：向蓝色方向微调，强化雪地的清冷感。
        *   **色调**：可向红色微调，为画面增添一丝暖意。
    *   **结构**：增加结构，让画面细节和纹理更清晰。
4.  **二次调整与微调**：完成初步调色后，再次观察，对过亮、过暗或色彩不协调的部分进行微调，直到满意。
5.  **最终裁剪与变形（可选）**：使用“瘦脸瘦身”或“编辑”中的“旋转裁剪”功能，可以拉伸山坡使其看起来更陡峭，或裁剪掉多余天空，重新构图。

## 总结

本节课中，我们一起学习了如何将一张普通的远景风景人物照修出高级感。整个流程可以总结为：**基础优化 → 身材重塑 → 面部精修 → 背景净化 → 滤镜调色**。核心在于灵活运用“瘦脸瘦身”进行形体塑造，用“消除笔”创造纯净背景，最后通过“滤镜”和“增强”功能赋予照片独特的氛围和质感。修图需要耐心和细心，多练习就能掌握其中窍门。
