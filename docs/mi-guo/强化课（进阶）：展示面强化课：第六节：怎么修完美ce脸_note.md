在本节课中，我们将学习如何修饰侧脸照片，使其更具立体感和美感。我们将使用美图秀秀，通过一系列步骤，从调整身材比例到精细处理面部细节，最终打造出一张出色的侧脸照。

## 🛠️ 修图步骤详解

以下是使用美图秀秀修饰侧脸照的具体操作流程。

### 1. 基础优化与身材调整

首先，打开美图秀秀，选择“图片美化”并导入需要处理的照片。

进入编辑界面后，第一步是进行基础优化：
*   点击“智能优化”。
*   使用“去雾”功能，适当提高照片清晰度。

由于这张照片包含了身体部分，我们不能直接进行面部重塑，需要先调整整体身材架构。

以下是身材调整的具体操作：
*   在“去美容”中找到“增高塑形”功能。
*   选择“美腿”选项，将人物整体拉高，使姿态更舒展。
*   使用“瘦脸瘦身”工具手动调整身体轮廓。将脖子向内推，肩膀向上调整以显宽，但需保持圆润感。
*   将胳膊肘部位向外拉，下身也相应外扩，使身材比例协调，避免上宽下窄。
*   最后微调腿部线条，使其看起来更自然。

调整后，对比原图，人物身材显得更加大气。

### 2. 面部结构重塑

身体架构调整完毕后，我们开始修饰面部。侧脸修饰的重点在于轮廓和立体感。

首先进行“面部重塑”：
*   使用“3D塑颜”功能，根据侧脸角度微调面部朝向（例如稍向下、稍转正）。
*   调整脸型，如果颧骨较宽，可适当向内推。
*   重点处理头发。侧脸照中，软塌的头发会显得头型扁平。需要用“瘦脸瘦身”工具将头发向外推，营造蓬松感。注意前额和后脑勺要蓬松，但头顶不能过圆。
*   拉长鬓角，增加立体感。

### 3. 五官细节精修

面部大体结构确定后，开始精细修饰五官细节。

以下是五官修饰的关键点：
*   **眉毛**：如果眉毛较短，需用力向眼尾方向拉长。
*   **眼睛**：侧脸照中眼睛不明显，可简单调整眼皮，无需过度修饰。
*   **鼻子**：侧脸最凸显鼻子。将鼻头向外推，鼻梁向内推，形成翘挺的立体感。同时将鼻翼向内收，使鼻子更精致。
*   **轮廓**：侧脸那边的轮廓线需要精心修饰。用“瘦脸瘦身”工具将脸部轮廓向内微调，营造凹陷感，使脸型更真实立体。下巴调尖后，旁边区域需稍向内收，形成自然过渡。
*   **眼角**：虽然看不到眼球，但内眼角和外眼角的形状可以微调，并推拉阴影部分以增强立体感。

### 4. 添加妆容与局部美化

细节修完后，为照片添加自然妆容。

进入“美妆”功能：
*   选择一款自然的“唇妆”。
*   根据脸型选择一款合适的“眉毛”，弥补原生眉毛的不足，使五官更立体。

添加妆容后，如果发现头发因推蓬而显得过厚，可用“瘦脸瘦身”工具将头发整体往回微调，保持圆润但不过于厚重。

### 5. 肤色提亮与瑕疵去除

侧脸容易因光线产生阴影，导致脸部显黑。

使用“去黑眼圈”功能：
*   重点涂抹鼻梁，勾勒出高光，使鼻梁更显高。
*   仔细涂抹脸部阴影区域，提亮肤色。注意要少量多次，避免涂抹过度导致不自然。
*   处理人中、嘴唇周围等颜色过深的部位。

如果提亮后觉得嘴唇偏厚，可用“瘦脸瘦身”工具将嘴唇向内侧推薄。

### 6. 背景与道具优化

人物修饰完毕后，需要优化背景和道具，使画面更整洁。

*   使用“瘦脸瘦身”工具调整手中玩具熊的形态，使其更可爱。
*   使用“消除笔”工具，将背景中杂乱的物品（如拖鞋）和地板纹理消除，营造干净的纯色背景效果。

### 7. 滤镜调色与最终定稿

最后，通过叠加滤镜和手动调色，为照片注入高级感和写真质感。

以下是滤镜叠加与参数调整方案：
1.  **滤镜叠加**：依次选择并叠加以下滤镜，每个强度不宜过高（约20%-30%），且务必关闭滤镜自带的美颜功能。
    *   `MOE`
    *   `MOR`
    *   `MO8`
    *   `M1`
    *   `MM3`（最后加10%左右增加高级感）
2.  **手动调色**：进入“增强”功能，手动调整参数。
    *   **亮度**：适当调高，提亮面部。
    *   **对比度**：调高以增强画面层次。
    *   **高光**：调低。
    *   **暗部**：如果脸部偏黑，可调高以提亮暗部。
    *   **饱和度**：微调。
    *   **色温/色调**：根据喜好微调。
    *   **结构**（清晰度）：适当调高，使细节更扎实。
3.  **最终检查与微调**：完成调色后，返回“去美容”使用“亮眼”功能，适当加深瞳孔、眉毛等黑色部分，平衡肤色。可重复检查调整各项参数，直至满意。

## 📝 本节总结

本节课中，我们一起学习了侧脸照片的完整修饰流程。

核心要点总结如下：
*   **发型是基础**：侧脸照需将软塌的头发推蓬松，塑造饱满的头型，拉长鬓角。
*   **轮廓是关键**：重点修饰侧脸轮廓线，营造立体凹陷感。鼻子需“鼻尖外翘，鼻梁内收”。
*   **身材要舒展**：利用“增高塑形”和“瘦脸瘦身”调整拘谨的姿势，肩部打开，胳膊肘外扩，形成稳定对称的身形。
*   **肤色需均匀**：用“去黑眼圈”工具仔细提亮面部阴影。
*   **背景宜干净**：使用“消除笔”清理杂乱背景。
*   **质感靠叠加**：通过多个低强度滤镜叠加和精细的手动调色，提升照片整体质感。

通过以上步骤，可以将一张普通的侧脸照，修饰成精致、立体、富有美感的作品。下节课，我们将探讨远景风景照的修饰技巧。
