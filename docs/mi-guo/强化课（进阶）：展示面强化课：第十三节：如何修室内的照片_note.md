# 📸 强化课（进阶）：展示面强化课 第十三节：如何修室内的照片

在本节课中，我们将学习如何修饰在室内拍摄的照片。室内与室外环境差异显著，光线和背景都不同，因此修图策略也需要相应调整。我们将通过一个具体案例，详细介绍从基础调整到细节精修的完整流程，帮助你掌握将普通室内照修出“网图”质感的核心技巧。

## 🧹 第一步：基础清理与身形调整

上一节我们介绍了室外修图的要点，本节中我们来看看室内修图的第一步——基础清理。室内照片常因灯光复杂、背景杂物多而显得杂乱，因此首要任务是净化画面并优化人物身形。

**核心操作**：使用修图软件的“去物”功能清除背景杂物。

以下是身形调整的具体步骤：
1.  使用“增高”功能，在头部、下巴、手臂和下身设置节点，整体拉长人物比例。
2.  使用“瘦脸瘦身”工具，手动将领子调高，以提升气质。
3.  将肩膀适当修宽。
4.  使用最大范围的“瘦脸瘦身”工具，整体向内推压，使身形显得更瘦。

完成这些步骤后，人物的整体轮廓和气质会得到初步改善。

## ✨ 第二步：面部精细重塑

基础身形调整后，我们进入更精细的面部重塑阶段。这一步的目标是优化五官和脸型，使其更立体、协调。

**核心概念**：利用光影对比塑造立体感，例如通过加深鼻梁两侧阴影来让鼻子显得更挺。

以下是面部重塑的详细流程：
1.  进入“面部重塑”功能，调整脸型、额头、眉毛间距等基础参数。
2.  重点调整眉毛形状，将其修成有倾斜度的“剑眉”，避免平眉带来的无辜感。
3.  调整眼型，使眼尾略微上翘。
4.  通过推压鼻梁两侧制造阴影，让鼻子显得更挺拔，同时收窄鼻翼。
5.  适当调薄嘴唇。
6.  使用“去斑去痘”和“去黑眼圈”功能，让皮肤显得更光滑干净。
7.  调整发型轮廓，将过于圆润的头型修出一些棱角，避免显得“憨厚”。
8.  检查并修正耳朵等细节，确保整体协调。

## 🎨 第三步：滤镜调色与背景优化

面部修饰完成后，我们需要为照片赋予统一的色调和质感。室内光线往往不如室外明亮通透，选择合适的滤镜至关重要。

**核心公式**：`最终色调 = 基础滤镜 (MO1) + 风格滤镜 (MO2/M18) + 手动参数微调`

以下是调色与背景优化的步骤：
1.  选择 **MO1** 滤镜，为室内环境奠定一个干净、不显脏的基调。
2.  叠加 **MO2** 或 **M18** 滤镜（强度约40%），为照片增添质感，但需避免过度导致照片过黑。
3.  进入“增强”模块进行手动调整：
    *   适当提高亮度。
    *   提高对比度，降低高光。
    *   调整暗部：调高可使脸部更白，调低则更显轮廓。
    *   略微降低饱和度。
    *   关键一步：**降低结构（或清晰度）**，这能柔化背景墙面上的纹理瑕疵，让画面更干净。
4.  使用“裁剪”功能，采用九宫格构图法，将人物主体放在交叉点上，增强照片的空间感和专业性。

## 📝 总结与下节预告

本节课中我们一起学习了室内照片的完整修饰流程。我们从**基础清理与身形拉长**开始，通过去物和比例调整奠定好基础；然后进行了**面部精细重塑**，优化五官、皮肤和发型；最后通过**滤镜叠加和参数微调**统一了照片色调与质感，并利用裁剪优化了构图。

经过这十三节课的学习，我们已经涵盖了室外、室内、全身、半身、正脸、侧脸以及身材照等多种场景的修图方法。修好照片只是第一步，如何将它们自然地展示出去同样重要。下节课，我们将探讨**如何建设个人朋友圈**，让你精心修饰的照片既能展现最佳状态，又不会显得虚假，从而真正吸引他人的注意。
