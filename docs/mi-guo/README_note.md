# 课程名称：蜜果 🍯

在本节课中，我们将要学习如何制作一种名为“蜜果”的甜点。这是一种将水果与蜂蜜等配料结合，经过简单处理制成的美味食品。教程将分步讲解从准备材料到完成制作的完整流程。

## 准备材料 🛒

上一节我们介绍了蜜果的基本概念，本节中我们来看看制作它需要哪些材料。以下是所需的核心材料清单。

*   新鲜水果（例如草莓、蓝莓、桃子），约500克。
*   蜂蜜，100毫升。
*   柠檬，1个。
*   密封玻璃罐，1个。

## 处理水果 🔪

准备好材料后，下一步是对水果进行预处理。这个步骤决定了蜜果最终的口感和保存期限。

以下是处理水果的具体步骤。

1.  将水果彻底清洗干净。
2.  用厨房纸巾吸干水果表面的水分。
3.  根据水果种类，去皮、去核或切成小块。

## 制作蜜液与混合 🥣

水果处理完毕，接下来我们调制关键的蜜液并将其与水果混合。这是赋予蜜果独特风味的核心环节。

蜜液的配方很简单，其基础公式为：**蜜液 = 蜂蜜 + 柠檬汁**。

1.  将柠檬榨汁，滤去果渣。
2.  在一个大碗中，将蜂蜜与新鲜柠檬汁混合均匀。
3.  将处理好的水果块放入蜜液中，轻轻搅拌，确保每一块水果都均匀裹上蜜液。

## 装罐与保存 🫙

混合完成后，我们需要将半成品妥善封装以便保存和发酵。正确的装罐方法是成功的关键。

以下是装罐与保存的注意事项。

*   将混合好的水果与蜜液一同倒入消毒过的干燥玻璃罐中。
*   确保蜜液完全浸没所有水果块。
*   拧紧瓶盖，在瓶身贴上标签，注明制作日期。
*   将罐子置于阴凉避光处保存，等待3至7天即可食用。

## 总结 📝

本节课中我们一起学习了“蜜果”的制作方法。我们从准备材料开始，逐步完成了水果处理、调制蜜液、混合装罐以及最终保存的全过程。记住核心公式 **蜜液 = 蜂蜜 + 柠檬汁**，并确保装罐时水果被蜜液完全浸没，你就能轻松制作出美味可口的蜜果了。
