## 第一部分：护肤流程

上一节我们介绍了课程的整体框架，本节中我们来看看护肤的具体流程。男士护肤可以简化为三个核心环节。

护肤在我这里只讲三个方面：
1.  **洁面**
2.  **保养**
3.  **面膜**

### 1. 洁面

洁面是护肤的第一步，根据是否化妆，流程有所不同。

*   **化妆后的洁面**：需要使用**卸妆水**。因为男性皮肤分泌的油脂较多，普通洗面奶难以彻底清洁。卸妆产品种类很多，我主要推荐两种：
    *   **卸妆水**：日常推荐使用。
    *   **卸妆湿巾**：价格较贵，适合出门旅行时携带，也建议在和女生外出旅行时备上几张。

*   **不化妆的洁面**：直接使用**洁面乳**（即洗面奶）即可。选择洁面乳时，最重要的是看**价格**和是否适合自己肤质。好的产品自然有它的道理，但你需要在自己的预算范围内，通过试用找到最适合自己肤质、用起来舒服、味道好闻的产品。通常洁面产品会以“三件套”（洁面乳、水、乳/精华）的形式出售。

### 2. 保养

男士保养非常简单，与女性复杂的流程不同。

我们只需要两样东西：
*   **水**：作用是补水。使用时轻轻拍打在脸上直至吸收。
*   **乳**（或精华）：作用是锁住水分。在水吸收后，将乳液均匀涂抹在脸上即可。

很多皮肤问题，如出油、粗糙，其实都是因为缺水导致的。因此，做好基础的补水保湿至关重要。

### 3. 面膜

面膜对于改善皮肤状态很有帮助。男生敷面膜的频率可以是一周两次，每次15分钟。敷完后建议清洗脸部，以免精华液残留导致粘腻。面膜的精华液还可以用来敷手或胳膊。选择面膜时，只需要购买**补水**功能的即可，无需追求美白等复杂功效。美白不能单靠面膜实现。

**成本计算**：假设一片面膜10元，一周敷两次，一周花费20元，一个月（按四周算）保养花费约80元，这是大多数人都能负担的。

### 特殊情况：痘痘肌肤

对于脸上有正在生长的红肿痘痘，或痘痘消退后留下的痘印，需要特殊处理。

以下是几款推荐的产品，但效果因人而异，建议试用：
1.  **八副瓶**：药店有售，价格不贵，可用于淡化痘痕和痘印。
2.  **科研室焕白精华**：价格约330元。
3.  **GOLAV**：进口产品，作用是阻断黑色素形成、镇定皮肤。

## 第二部分：化妆流程与演示

护肤是基础，接下来我们将进入本节课的核心部分——化妆。我将一边演示，一边讲解男士化妆的完整流程和所需装备。

### 化妆所需装备清单

以下是演示中将用到的所有化妆品，按使用顺序排列：
1.  **洁面产品**（已介绍）
2.  **妆前精华**（也称**隔离**）
3.  **粉底液** + **化妆蛋**
4.  **散粉**
5.  **眉笔**
6.  **隐形眼镜**（美瞳，非必需）

*注：本节课暂不涉及高光、阴影、眼影、唇膏等进阶内容，只教学最基础的入门流程。*

### 分步化妆演示

现在，我将关闭直播美颜，在无美颜状态下为大家逐步演示化妆过程。

**第一步：洁面与修眉**
*   用洁面湿巾或洗面奶彻底清洁面部，去除油脂和汗水。
*   **修眉**：用眉刀修理眉毛杂毛。遵循“修三边”原则（上边缘、眉峰到眉尾的斜边、下边缘），修出基本眉形。如果自己不会修，可以去美容店或纹绣店请专业人士修理，甚至可以考虑纹眉。

**第二步：基础护肤与妆前**
*   洁面后，拍上**爽肤水**和**乳液**进行基础保湿。
*   涂抹**妆前精华**（隔离）。它能使妆容更服帖自然，也是与女生交流的好话题。涂抹时别忘了照顾到脖子，避免脸和脖子色差过大。

**第三步：上粉底**
*   将**化妆蛋**用水浸湿并拧干。
*   将**粉底液**点涂在脸上，用湿化妆蛋轻轻拍打均匀。有痘痘的地方可以重点遮盖。
*   **色号选择**：粉底液通常分偏白、偏黄、偏黑等色号。偏黄色号遮瑕效果较好。建议去实体店试用，或根据自己肤色（偏白、偏黄、偏黑）来选择。
*   **关键**：一定要将粉底液过渡到脖子，确保颜色衔接自然。

**第四步：定妆**
*   使用**散粉**，用粉扑或刷子轻拍全脸。这一步是为了吸收多余油分，让妆面变得干爽、持久，不粘腻。

**第五步：画眉**
*   粉底可能会遮盖部分眉毛，需要补画。
*   **眉笔选择**：建议购买双头眉笔（一头是笔，一头是眉刷）。
*   **画眉方法**：找到眉头、眉峰、眉尾三个点，先轻轻勾勒出眉毛轮廓，再填充内部。遵循“眉头浅、眉尾深”的原则。
*   **新手技巧**：强烈建议购买**眉卡**，将其贴在眉毛位置，直接填充颜色即可，非常方便。
*   **修改技巧**：如果眉毛画重了或画错了，可以用眉刷扫淡，或用蘸有粉底液的化妆蛋边缘进行修饰。

**第六步（可选）：佩戴隐形眼镜**
*   佩戴合适的隐形眼镜或美瞳，可以让眼睛看起来更有神。

**最终效果对比**：通过以上步骤，即使没有使用美颜，妆容也能显著改善肤色、遮盖瑕疵、提升眉形，让人看起来更加精神、帅气。

### 化妆流程总结与成本核算

我们来总结一下男士基础化妆的六个步骤及所需时间：
1.  **洁面+基础护肤**：约1分钟
2.  **妆前隔离**：约2分钟
3.  **上粉底**：约2-3分钟
4.  **定妆（散粉）**：约1分钟
5.  **画眉**：约10分钟（练习后可缩短）
6.  **佩戴美瞳**（可选）：约2分钟
**总计**：熟练后可在20分钟内完成。
