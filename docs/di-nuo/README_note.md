# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子。这个盒子（变量）可以用来存放数据（比如数字、文字等）。标签就是变量的名字，我们通过这个名字来找到并使用盒子里的数据。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。

```python
my_name = “迪诺”
age = 10
```

这里，我们创建了两个“盒子”：
*   名为 `my_name` 的盒子里存放了文字“迪诺”。
*   名为 `age` 的盒子里存放了数字 `10`。

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道盒子里装的是哪种类型的东西。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其示例：

1.  **整数**
    *   描述：没有小数部分的数字，可以是正数、负数或零。
    *   代码示例：`score = 100`， `temperature = -5`

2.  **浮点数**
    *   描述：带有小数点的数字，用于表示更精确的数值。
    *   代码示例：`pi = 3.14159`， `price = 19.99`

3.  **字符串**
    *   描述：由一系列字符组成的文本。在Python中，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   代码示例：`greeting = “Hello World”`， `name = ‘Alice’`

4.  **布尔值**
    *   描述：这种类型只有两个值：`True`（真）和 `False`（假）。通常用于逻辑判断。
    *   代码示例：`is_student = True`， `game_over = False`

## 如何查看和检查类型？ 🔍

有时我们不确定一个变量里装的是什么类型的数据，Python提供了 `type()` 函数来帮助我们查看。

```python
my_var = 42
print(type(my_var)) # 输出：<class ‘int’>
```

`<class ‘int’>` 表示 `my_var` 是一个整数类型。同样，你可以检查其他任何变量。

## 一个简单的实践例子 💻

现在，让我们把以上概念组合起来，写几行简单的代码。

```python
# 定义几个变量
product_name = “巧克力”
quantity = 3
unit_price = 12.5
is_available = True

# 打印这些变量及其类型
print(“产品：”, product_name, “， 类型：”, type(product_name))
print(“数量：”, quantity, “， 类型：”, type(quantity))
print(“单价：”, unit_price, “， 类型：”, type(unit_price))
print(“有货：”, is_available, “， 类型：”, type(is_available))

# 计算总价
total_price = quantity * unit_price
print(“总价是：”, total_price)
```

运行这段代码，你会看到每个变量的值以及它们所属的数据类型。最后，我们还利用变量进行了一次简单的乘法计算，得到了总价。

## 总结 📝

本节课中我们一起学习了Python编程的基石：
*   **变量**是一个用于存储数据的命名容器，通过 `=` 赋值。
*   **数据类型**定义了变量中所存储数据的种类，我们主要认识了**整数**、**浮点数**、**字符串**和**布尔值**这四种基本类型。
*   使用 `type()` 函数可以检查任何变量或值的数据类型。

理解变量和数据类型是后续学习更复杂操作（如运算、判断、循环）的基础。请务必动手尝试创建自己的变量，并观察它们的行为。
