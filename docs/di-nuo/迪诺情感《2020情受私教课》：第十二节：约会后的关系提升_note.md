# 迪诺情感《2020情受私教课》：第十二节：约会后的关系提升 💘

在本节课中，我们将要学习约会结束后如何有效地推进关系。约会不是终点，而是关系发展的关键节点。我们将探讨约会后如何正确联系、避免常见错误，并最终学习如何通过表白来确定关系。

## 约会后的核心原则

上一节我们介绍了约会本身，本节中我们来看看约会结束后应该怎么做。一段男女关系的进程，如同逆水行舟，不进则退。一次优质的约会后，至少让女孩对你感觉不错，此时你必须思考如何让关系更进一步。

以下是约会结束后需要遵循的三个核心原则：

1.  **结束之后，别着急联系**
    轻易得到的东西往往不被珍惜。约会刚结束回到家时，即使对方主动告知已到家，也不要立刻展开密集聊天。给对方一些回味和思考的空间，显得你并不急于求成，反而可能引发对方的好奇与自我怀疑。

2.  **联系时切忌跪舔**
    遇到很对胃口的女生时，很多男性会冲动地无限度示好，比如马上送贵重礼物。这种行为会让关系开始变质。对方可能会慢慢习惯你的好，或者逐渐感到厌烦。一定要控制自己，保持平等姿态。

3.  **控制后续的追求节奏**
    可以选择隔一段时间再进行下一次邀约，避免让对方产生压力。不见面时，可以通过互动保持联系，让她习惯你的存在。下次邀约的时间可根据上次约会的舒适度决定，但建议至少间隔三四天。

按照以上流程操作，你与女生的关系将稳步推进，为最终确定关系打下良好基础。

## 如何确定关系：表白的艺术

当我们成功推进关系后，便会来到最后一步：确定关系。对于许多男性而言，确定关系常与“表白”划等号。那么，到底应不应该表白呢？

针对新手，通常不建议贸然表白，因为新手难以精准把控情感的“闸门”。力度过大可能吓跑对方，力度过小又无法传达心意。但如果你按照前述流程操作，则无需过分担心“表白死”。

一个有效的表白通常包含形式和内容两方面。形式指在微信、面对面、送花或写情书等场景中进行。而表白的内容则有一个通用公式可以参考。

### 表白内容核心公式

一个结构清晰、不易暴露过度需求感的表白内容，可以遵循以下公式：

**表达过往感受 + 保证现在 + 主观判断/期望**

以下是每个部分的详细解释：

*   **表达过往感受**：让女生知道你因为什么原因喜欢她。例如：“从我刚认识你的时候，就觉得你是一个很特别的女孩，那种单纯让我产生了想要保护你的冲动。”
*   **保证现在**：避免开空头支票，聚焦于当下你能做到的承诺。例如：“我不会说遥远的未来，但我能保证，只要我们在一起的每一天，我都会真诚地对待你，不让你受委屈。”
*   **主观判断/期望**：
    *   **主观判断**：将你对她的喜欢，描述为对你们共同经历的留恋，这样更客观，更容易引发共情。例如：“与其说我只是喜欢你，不如说我是特别喜欢和你在一起时的那种轻松快乐的感觉。”
    *   **期望**：在感觉对方兴趣很高、成功率大时使用，直接表达对未来的期待。例如：“所以，我希望我们能给彼此一个机会，开始一段甜甜的恋爱，不要浪费这份相遇的美好。”

你可以根据对方的反应，灵活选择使用“主观判断”还是“期望”。如果对方态度犹豫，就用“主观判断”；如果对方态度积极害羞，就可以转向“期望”。

### 表白后的结果与应对

表白后无非两种结果：成功或失败。

*   **成功**：关系确立，进入恋人相处模式，可分为长期认真或短期关系，由双方共同把握。
*   **失败**：失败后双方关系可能呈现三种状态：变得更好、维持原状或变得更差。
    *   **变得更好**：通常意味着你们成了“哥们”，能玩到一起但不想谈恋爱，多了一个好朋友。
    *   **维持原状**：对方可能说“再等等看”，关系没有突破但也没有倒退。
    *   **变得更差**：对方明确拒绝，态度变得疏远甚至不耐烦。

面对失败，关键在于识别对方是“真拒绝”还是“假矜持”。“真拒绝”会带有明显的距离感；“假矜持”则可能表现为欲拒还迎、害羞且立场不坚定。

## 应对女生常见拒绝话术

以下是女生在表白时可能提出的一些问题及应对思路，供你参考：

1.  **“太快了，我们才认识没多久。”**
    *   **分析**：没有明确拒绝就是有机会。
    *   **应对**：“感情深浅不在于时间长短。我觉得我们相处很愉快，不想因为所谓的‘时间不够’而错过对的人。”

2.  **“我们还不够了解。”**
    *   **应对**：“了解是一辈子的事。我相信在一起的过程，就是最好的互相了解的过程。”

3.  **“我这个人比较慢热。”**
    *   **分析**：“慢热”可能只是不够喜欢的托词。
    *   **应对**：“没关系，我们可以慢慢来。但至少请让我有一个可以光明正大对你好的身份。”

4.  **“我还没从前任的伤害里走出来。”**
    *   **应对**：“每个人都带着过去，但我们不能一直活在过去。我希望我能给你带来新的、快乐的回忆。”

5.  **“我想奔着结婚去，但我们现在经济能力不行。”**
    *   **应对**：“我理解你的考虑。我也是认真对待感情的。我们可以一起设定目标，共同努力去创造我们想要的未来。如果要求完全超出我的能力范围，那我会尊重你的选择。”

6.  **“做朋友不是挺好的吗？”**
    *   **应对**：“我不缺朋友，我缺的是一个像你这样的恋人。我希望生活里有你，不仅仅是以朋友的身份。”

7.  **“我怕谈恋爱会被束缚。”**
    *   **应对**：“好的爱情是共同成长，而不是束缚。我会尊重你的空间，我们是在彼此的世界里添加快乐，而不是剥夺自由。”

8.  **“家里给我安排了相亲，但我又觉得你挺好。”**
    *   **应对**：“如果你对我有好感，我希望我能成为你的第一个‘相亲对象’。我们可以正式地、以结婚为前提互相了解一下。”

9.  **“你太渣了，我怕你渣我。”**
    *   **应对**：“‘渣’是涉及欺骗的行为。我对待感情是认真的，和我在一起后，我会公开我们的关系，给你安全感。我的朋友多少与人品无关，我从未在感情中欺骗过任何人。”
