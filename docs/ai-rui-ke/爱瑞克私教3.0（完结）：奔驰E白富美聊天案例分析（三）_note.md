# 爱瑞克私教3.0（完结）：奔驰E白富美聊天案例分析（三）

## 概述
在本节课中，我们将深入分析奔驰E级白富美的聊天案例，学习如何通过聊天技巧推进关系，并了解如何识别和应对兴趣指标。

## 一、案例背景
爱瑞克与一位白富美通过App认识，并逐渐发展成朋友关系。以下是他们的聊天记录。

## 二、聊天分析
### 1. 初识阶段
- **基本交换**：爱瑞克通过询问对方在做什么，了解对方的基本情况。
- **推进关系**：爱瑞克邀请对方一起吃宵夜，进行基础了解。

### 2. 深入了解阶段
- **兴趣指标**：对方分享美食图片，表示认可。
- **推进关系**：爱瑞克夸奖对方，并表达想要一起尝试的意愿。

### 3. 情感交流阶段
- **换位思考**：对方刚下飞机，爱瑞克选择不打扰。
- **情感表达**：爱瑞克关心对方，并表达体贴之情。

### 4. 邀约阶段
- **朋友圈互动**：通过朋友圈展开话题，延伸到邀约。
- **兴趣指标**：对方分享生日照片，表示期待。
- **情感表达**：爱瑞克夸奖对方，并表达祝福。

## 三、核心概念
### 1. 兴趣指标
**公式**：兴趣指标 = 互动频率 + 互动内容
**代码**：兴趣指标 = frequency * content

### 2. 反撇
**公式**：反撇 = 前期展示 - 后期展示
**代码**：反撇 = (前期展示 - 后期展示) / 前期展示
