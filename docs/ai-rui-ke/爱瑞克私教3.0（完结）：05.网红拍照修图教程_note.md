# 爱瑞克私教3.0（完结）：05.网红拍照修图教程

## 概述
在本节课中，我们将学习如何通过拍照和修图提升个人形象，特别是如何拍出网红风格的照片。

## 拍照技巧
以下是拍照的三个关键点：

1. **形象塑造**：确保个人形象帅气，包括发型、服装和配饰。
2. **场景选择**：选择有逼格的场景，如高端商业综合体、网红墙等。
3. **动作与表情**：保持自然，避免僵硬，尝试连贯的动作和表情。

## 九宫格构图
使用九宫格构图可以帮助拍摄出更具艺术感的照片。将照片分成九宫格，确保人脸位于网格交叉点附近。

## 修图步骤
以下是修图的三步法：

1. **FaceTune2粗调**：调整人脸的五官和脸型。
2. **美图秀秀微调**：进行更精细的调整，如瘦脸、瘦身、修容等。
3. **Snapseed加滤镜**：添加滤镜，调整光线和色彩。

## 修图软件介绍

- **美图秀秀**：用于微调，如瘦脸、瘦身、修容等。
