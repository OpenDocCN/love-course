# 爱瑞克私教3.0（完结）：12.如何打造优秀立体展示面

在本节课中，我们将学习如何打造一个优秀且立体的社交软件展示面。

## 1. 展示面的重要性

**公式**： 优秀展示面 = 高价值 + 立体感

一个优秀的展示面需要具备以下特点：

- **高价值**： 展示你的生存价值，如职业、成就等。
- **立体感**： 从多个角度展示你的生活，如吃喝玩乐、工作、旅行等。

## 2. 失败的展示面

以下是一些常见的展示面错误：

- **不自信**： 没有头像或头像模糊不清。
- **猥琐**： 拍照姿势不自然，给人感觉不自信。
- **炫富**： 过度展示物质财富，引起反感。
- **转发无关内容**： 占用广告位，影响展示效果。
- **消极悲观**： 发表负面言论，给人留下不良印象。

## 3. 展示面设置要点

- **帅**： 照片要清晰，五官要突出。
- **价值**： 展示你的生存价值，如职业、成就等。
- **丰富多彩**： 展示你的生活，如吃喝玩乐、工作、旅行等。

## 4. 展示面拍摄技巧

- **背景**： 选择干净、高端的背景，体现你的生活品质。
- **内容**： 展示你的生活，如美食、旅行、运动等。
- **配文**： 轻松愉快，避免装逼炫耀。

## 5. 多元化展示面

- **吃喝玩乐**： 展示你的生活品质。
- **勤奋努力**： 展示你的工作态度。
- **偶尔煽情感性**： 展示你的情感世界。
- **接地气**： 展示你的真实一面。
