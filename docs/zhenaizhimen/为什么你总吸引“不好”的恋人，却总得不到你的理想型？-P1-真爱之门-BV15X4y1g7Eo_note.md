# 真爱之门课程 P1：为什么你总吸引“不好”的恋人，却总得不到你的理想型？ 🧲💔

在本节课中，我们将要学习一个核心概念——“爱的词典”。这个概念将帮助你理解，为何你总是陷入相似的不健康关系模式，以及为何你理想中的伴侣似乎遥不可及。我们将通过分析你的内在信念系统，揭示你吸引特定类型伴侣的根本原因。

## 什么是“爱的词典”？ 📖

上一节我们介绍了课程的主题，本节中我们来看看核心概念“爱的词典”。

“爱的词典”是指你内心深处一系列关于“什么是爱”的标准和信念。它定义了在何种情况下，你会感觉自己被爱、被重视。这套词典并非天生，而是在你成长过程中，由家庭、环境和社会经历塑造而成。

以下是“爱的词典”的一个简单定义：
**爱的词典 = 你感觉被爱所需满足的一系列内在标准和条件**

现在，请暂停阅读，思考并写下你的答案：当别人（特别是异性）对你做什么事情时，你会感觉自己是被爱的？

为了帮助你理解，以下是一些常见的“被爱的词典”条目示例：
*   当对方秒回我的消息时，我感觉被爱。
*   当对方聊天时显得比我更热情、更主动时，我感觉被爱。
*   当对方为我花钱或送我礼物时，我感觉被爱。
*   当对方愿意与我进行肢体接触时，我感觉被爱。
*   当对方主动约我出去时，我感觉被爱。

你的清单是独一无二的，它反映了你的个人历史。例如，一个在严苛家庭长大的人，其“被爱的词典”可能条目稀少；而一个在充满鼓励的环境中成长的人，其词典则可能丰富得多。

## “被爱的词典”也是“不被爱的词典” ⚖️

理解了“爱的词典”是个性化的标准后，本节中我们来看看它的第一个重要特性。

你感受到的“被爱”的感觉，并非无条件的。它依赖于外部世界是否满足了你的特定标准。这个世界是“二元性”的——事物常被我们分为好与坏、对与错。

因此，建立在外部条件上的“被爱感”极不稳定。你会发现，你的“被爱的词典”同时就是你的“不被爱的词典”。只需在每个条目前加上“不”或“没有”，它立刻转变。

以下是转换示例：
*   **被爱**：对方秒回我 -> **不被爱**：对方没有秒回我。
*   **被爱**：对方聊天很热情 -> **不被爱**：对方反应冷淡。
*   **被爱**：对方送我礼物 -> **不被爱**：对方没有送我礼物。

这导致你的情感体验像坐过山车，随着外界是否满足你的条件而在“天堂”与“地狱”间波动。

## “被爱的词典”还是“你爱他人的词典” 🔄

看到了“被爱”与“不被爱”本是一体两面后，本节中我们来看看这个词典如何影响你爱别人的方式。

你用来感受爱的标准，同样会成为你向他人表达爱的方式。这是一种潜意识的投射：你认为能让自己感受到爱的方式，也是能让别人感受到爱的方式。

例如，如果你从小被教育“把好吃的让给你是爱你”，那么当你遇到类似情境（如盘子里只剩一块好吃的鸡块），你很可能会下意识地把它让给约会对象，并认为这是在“爱”对方。这就是“我是为你好”背后的逻辑——用自己认可的方式去爱别人。

另一个关键例子是关于安全感的：
如果你认为“对方秒回=他爱我/我安全”，那么当对方给你发消息时，你也会感到压力，认为“我必须秒回=这样他才能感受到我的爱/他才会有安全感”。即使你手头正忙，也可能无法专注，必须抱着手机回复。这导致你无法享受自己的生活，爱变成了负担和相互索取。

## 为什么你总吸引“不好”的恋人？ 🎯

我们已经了解了“爱的词典”的三个一体层面，现在回到最初的问题：为什么你总吸引不到理想型？

原因在于共振法则。你只会与那些和你拥有相似“爱的词典”的人产生深度联结和吸引。因为只有他们行为，才能触发你“被爱”的感觉。

**理想型（如独立、自信、有空间感）的行为模式：**
*   可能不会秒回消息，因为她专注于自己的事情。
*   享受聊天，但不需要时刻黏在一起，尊重彼此空间。
*   她的爱体现在信任、尊重和真实的陪伴上。

**你的“词典”可能产生的解读（如果词典充满不安全感）：**
*   她不秒回 = 她不重视我/我做错了什么。
*   她不时刻黏着我 = 她不够喜欢我。
*   我感受不到她的爱，只感到受伤和焦虑。

因此，一个真正独立自信的理想型，其行为无法满足你当前“词典”里对“被爱”的渴望（如时刻关注、高频联系）。你感觉不到连接，关系自然无法推进。

**相反，你会吸引什么样的人？**
你会吸引那些同样渴望通过“紧密联系”、“时刻确认”来感受爱的人。他们的行为（黏人、索取关注、频繁互动）恰好满足了你的“被爱标准”。你会觉得：“他这么需要我，他一定是真的喜欢我。” 于是，你们因为相似的“爱的词典”（本质是相似的不安全感和索取模式）而共振在一起。

这形成了一个循环：你用自己“不被爱的词典”去解读理想型的行为，感到受伤；同时又被那些能触发你“被爱感”（但可能不健康）的人吸引。

## 如何改变并吸引理想型？ ✨

认识到问题源于内在的“词典”后，本节我们来看看解决之道。

改变外在关系的关键，在于修改你内在的“爱的词典”。你需要疗愈那些“当……发生时，我就感觉不被爱/受伤”的信念和创伤。

**核心转变路径：**
**旧的词典（基于条件） -> 疗愈与觉察 -> 新的词典（基于本自具足）**

目标是达到一种状态：无论他人如何行为，你都能保持内在的平静与快乐。他人的行为只是他人的选择，不再是你情绪的开关。当你内心充满安全感和自爱时，你自然会散发一种能量，吸引那些同样完整、自信的个体。

以下是你可以开始的步骤：
1.  **觉察**：列出你的“被爱/不被爱词典”，看清你的模式。
2.  **疗愈**：通过冥想、内在工作（如课程中提供的工具）或寻求专业帮助，去疗愈词典背后对应的童年创伤或限制性信念。
3.  **重建**：有意识地培养新的信念。例如，将“他不秒回我=他不爱我”转变为“他有自己的空间，我尊重他，我也享受我自己的时间”。
4.  **行动**：在新的信念下行动。当安全感来袭时，练习不立刻从外界索取确认，而是回归内心，给自己安抚。

## 总结与行动建议 🚀

本节课中我们一起学习了“爱的词典”这一核心概念。我们明白了：
1.  你拥有一套独特的、关于何为“被爱”的内在标准（**爱的词典**）。
2.  这套词典同时定义了你在何时感觉“不被爱”，它让你情绪受外界摆布。
3.  这套词典还主导了你“爱他人”的方式，导致你可能用自己需要的方式去爱别人。
4.  你总吸引“不好”的恋人，是因为你们拥有相似（不健康）的“词典”；你无法吸引理想型，是因为你们的“词典”不匹配，你无法正确解读和接收他们的爱。
5.  改变的唯一途径是**由内而外**地疗愈和修改你自己的“爱的词典”，让自己先成为内心完整、有安全感的人。

如果你想系统性地学习如何疗愈创伤、修改潜意识信念，可以关注相关课程（如提到的“创造法则”）。真正的转变始于深刻的自我觉察和持续的内在练习。

请记住，你拥有改变内在剧本的力量。当你更新了“爱的词典”，你便更新了你整个世界，包括其中的人与关系。
