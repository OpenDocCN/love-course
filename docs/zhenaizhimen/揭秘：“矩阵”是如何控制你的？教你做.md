# 课程P1：揭秘“矩阵”如何影响你，并掌握流畅沟通的技巧 🧠💬

在本节课中，我们将探讨一个被称为“矩阵”的社会心理系统如何影响我们的自我认知和沟通方式。我们将分析为何在聊天或演讲时会感到紧张、无话可说，并学习如何摆脱预设脚本的束缚，实现自然、流畅的自我表达。课程的核心是理解内在信念如何塑造行为，并提供实用方法帮助你在任何对话场景中都能自信应对。

## 从预设脚本到自由表达：一个自我发现的起点

上一节我们概述了课程目标，本节中我们来看看沟通中一个常见问题的起点：对预设脚本的依赖。

我以往录制视频时，习惯于提前写稿子或列大纲。这样在录制时就不会忘词。你可能有类似体验，例如领导要求你在会议上发言，你会先列好提纲，以防自己在会议上说不出内容。或者当你试图与一个陌生人聊天时，你可能会担心自己没词可说。

这两天我和hero合体录视频后，我开始尝试不提前准备稿子，不提前准备任何内容，在大脑一片空白的情况下直接开始讲。这样做是因为当你预设了要讲的内容，你的脑子会一直想第一点是什么，第二点是什么，千万不能忘。如果中途卡壳，没有顺畅流动的感觉，你会害怕这种空缺状态。我试着扔掉这些东西，不让这些预设内容占据我的大脑。就像现在站在这里，头脑中一片空白，这样可以随意讲任何当下想到的事情。

## “矩阵”的构成：社会信念如何塑造你的自我

上一节我们探讨了对预设脚本的依赖，本节中我们来深入看看这种依赖背后的深层系统——“矩阵”。

我们想要自己在讲话时表现得很完美，这是因为我们对自己有一个自我的形象，你可以把它理解为小我（ego）。我们在社会上生存时，特别在意别人对我们的看法。因为我们的自我价值感来自于外在，来自于其他人的评价。我们从小到大，父母可能会说你太矮、太丑、学习太差，你得好好学习、功成名就、考上好大学、找一份靠谱的工作，才能让父母脸上有光，才能让自己成为国家的栋梁。在类似的信念下，包括老师、同学、身边所有人，甚至电视媒体广告，都在传达你有问题：皮肤不够好、不够酷、不够帅、是个屌丝，你需要买房、买车、买化妆品、买衣服，才能感觉到自己的价值，才能有脸在这个世界上生存下去。我们无条件、无意识地接受了所有这些信念。

这个社会就像一个巨大的矩阵。电影《黑客帝国》中，每个人都在一个独立密封的营养仓里，靠营养液生存，靠电路给大脑产生神经刺激，生活在一个幻想中。我们的这个世界，其实也是一个巨大的矩阵。这个矩阵在哪呢？就是蒙蔽我们双眼的整个世界，就是我们头脑中的所有信念。所有你相信的东西，并不一定就是真的。比如“我现在不够好”这个信念。你可以感受一下自己内在的状态和感受。如果从0分到10分给自己打分，你会打几分？有几个人能给自己打10分呢？如果没有你积累的东西——钱包里的钱、房子、车子、工作、身份、所有来自外界的认同——如果把这些全部丢掉，你没有所有身份和物资，那么你是谁？你是一个怎样的人？你还会感觉到自己是有价值的吗？你还会感觉到自己是被爱的吗？

我们在矩阵中生存时，其实是通过外界来获得价值感、存在感、认同感，也就是获得爱。这就是矩阵插在我们身上的管子。

## 管子的形态：对外的依赖与成瘾

上一节我们定义了“矩阵”和它提供的“管子”，本节中我们来看看这些“管子”具体以哪些形式出现。

这个管子可能有各种形式。例如，你喜欢看的抖音、快手等娱乐，它们会刺激你的神经，让你感到短暂的愉悦，从而逃避你的生活。想想你的生活都在做什么？你在家里看电视、上网，你宁愿从电视上的科教频道看《人与自然》，都不愿意自己亲自去自然界走一走、看一看。你在体验别人的生活。你看电视剧、看所有的娱乐节目、看明星真人秀、看别人的生活，对别人的生活上瘾。为什么你要看别人的生活？因为你不喜欢自己的生活，你觉得自己没有办法实现自己想要的东西，没有办法过上自己想过的生活。所以，你把注意力关注在了外界，关注在别人的生活上面。

其实，你对自己的生活是可以完全掌控的。你可以随时随地的去做自己想做的事情，成为你想成为的人。到底是什么限制了你？是你的头脑，是你的信念。你从小到大接受的这些信念，可能到现在已经有点意识到了这个社会、这个矩阵（matrix）是如何操控你的。

## 矩阵的操控逻辑：从信念到行为

上一节我们看到了对外依赖的各种表现，本节中我们来系统化地理解矩阵操控你的完整逻辑。

以下是矩阵操控你的核心步骤：

1.  **植入不完美的信念**：首先通过信念让你相信，自己是不完美的，自己是有缺陷的。信念公式可以概括为：**“我缺X”**，其中X可以是名牌包、名牌衣服、别墅、宝马车、一份有面子的工作、一个优秀的伴侣等。

2.  **降低自我基准**：当你相信了所有这些关于“我很差”、“我很糟糕”、“我需要通过外在东西来弥补”的信念后，你的基准状态、自我价值感、自我评价就会降低。例如，你的内心代码可能从 `self_worth = 10`（满分）变成了 `self_worth = 3`。

3.  **产生不配得感**：由于自我评价降低，你会产生不配得感。核心信念是：**“我是三分的，我不是十分，所以我配不上任何我想要的东西。我没有办法实现我的理想，没有办法过上我想过的生活。”**

4.  **转向外部寻求补偿**：因此，你才会被各种娱乐项目所吸引。你可能看网络小说，让小说里的主人公去过你想过的生活；你可能看电视剧，羡慕里面的浪漫爱情故事，但你内心深处不相信自己会遇到，觉得自己不配得，认为这些都是童话。

5.  **形成依赖与成瘾循环**：当你相信了所有这些，你把你的生活放到一边，然后开始向外寻求。同时你对所有这些外部的刺激感到上瘾。就像你今天攒钱买了一件特别想要的名牌衣服，穿上它感到自己状态好了一点，感到身上笼罩了一个光环。但过了两天，这个光环退去，你穿上这件衣服感觉和以前没有任何区别，内心仍然不快乐、不自信。然后你又开始攒钱买另一件。买了好多件以后，你发现原来这个东西是不能持久的，没法让自己拥有彻底的快乐和圆满。

伴侣关系也是类似的逻辑。很多人觉得，一个很好的伴侣能让我完整。但这是一种信念。即使你找到理想伴侣，他对你很好，但在一起过一段时间后，你内心当中匮乏、不舒服的感觉又冒出来，你会觉得这个人已经没有办法让我产生新鲜感，没有办法让我快乐。这就是你的信念：你觉得别人要让你快乐，你需要别人来让你快乐。在伴侣关系中，很多人陷入冲突争吵，是因为你在要求你的伴侣用特定方式爱你，要求伴侣对你专一。即使他在忙自己的事情，当你觉得缺爱、不开心时，你会觉得“你应该陪我，你应该做我想让你做的事”。这是爱吗？这是索取，这是控制。

## 破解矩阵：实现流畅沟通的两个核心心法

上一节我们完整剖析了矩阵的操控链条，本节中我们回到最初的沟通问题，并给出破解之道。

关于最开始说的，当你试图和一个人聊天，比如看到一个喜欢的人，想过去聊两句，但你会很害怕。为什么害怕？因为你担心自己站到对方面前后，不知道该说什么，头脑没有词。就像我开头说的，我自己录视频时也曾经有这个阶段，我必须要写一个很完美的稿子，甚至背下来，列一个大纲，这样就可以很完美地讲出，从而从外界获得认同和赞赏。但这永远不会结束，这是一个死循环，因为我总会担心如果忘了怎么办，如果没做得很完美怎么办。如果你自己的存在感、价值感建立在外界，它总是会波动，总会有人不喜欢你，总会有人不管你多完美，他都会说不喜欢。这是他们的自由，你为什么要在意别人的看法？

如果你担心自己没词，担心自己不知道该说什么，其实这很多时候是因为你的说话门槛太高了。想想你和你最熟悉的朋友聊天时畅所欲言，可以什么话都说。这个感觉我称之为“流动感”，你可以理解为是一种氛围。为什么你可以和朋友进入流动感的状态？很重要的一点是，你和他聊天时，不会担心自己卡壳忘词。因为当你没话说时，你就会玩手机、发呆，可能会等对方和你聊两句。你在朋友面前，是感觉到自己被接纳、被爱的。你相信你的朋友不会介意你卡壳或忘词，他不会因为这件事而评判你。你们的关系非常自由，想聊什么就聊什么，想聊就聊，不想聊就不聊。在这样的关系之下，你才能进入流动感。

以下是实现流畅沟通的两个核心心法：

1.  **创造安全的心理氛围**：模拟与好友聊天的信任与接纳感。告诉自己，对方可能也和你一样，对话的本质是连接，而非表演。
2.  **全然活在当下**：不要考虑下一步会发生什么，不要考虑下一句我该怎么说。没有最完美的讲话方式，没有最完美的下一刻。不要去提前安排，因为安排是一种限制。你可能安排得很完美，但实际情况可能比你想象的更好。难道你要因为你预设的100分，而抛弃掉即将到来的200分吗？
