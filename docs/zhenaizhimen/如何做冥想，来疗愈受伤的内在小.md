# 如何做冥想，来疗愈受伤的内在小

在本节课中，我们将学习一种不同于主流正念冥想的疗愈式冥想方法。我们将通过一个真实的疗愈个案，了解如何运用“自我传讯”工具，连接并疗愈内在受伤的部分（即“内在小孩”），从而从根源上转变焦虑、不安全感等情绪困扰。

## 概述：两种主要的冥想路径

冥想主要分为两种类型。

第一种是基于注意力的冥想，常被称为**正念冥想**。例如念经、打坐或跟随主流冥想音频进行练习。其核心公式可概括为：

**注意力 → 当下时刻（或特定对象，如经文）**

这类冥想能帮助练习者暂时获得内心的平静，减少纷杂的念头。然而，它通常无法从根本上转变一个人内在的“音”（即核心的情绪模式和信念）。这就像进行一次短暂的“身外之旅”，结束后仍需回到原有的情绪现实中。

上一节我们介绍了基于注意力的冥想，本节中我们来看看我们将要学习的第二种冥想路径。

我们将要学习的是一种基于**觉察和感受**的冥想。这种方法不是将注意力从自身移开，而是完全转向内在，直面我们通常逃避的感受和创伤。其核心公式可概括为：

**注意力 → 内在感受与创伤 → 觉察与疗愈**

这种方法旨在从根源上解决问题，效果持久稳定，而非暂时逃避。

## 疗愈工具：“自我传讯”冥想

真爱之门体系中最主流的疗愈工具是“自我传讯”冥想。它通过引导我们与内在的感受对话，挖掘并疗愈深层的创伤记忆。

以下是使用“自我传讯”进行疗愈的基本步骤框架：

1.  **连接身体感觉**：放松身体，将注意力转向内在，感受身体哪个部位有不适或情绪。
2.  **与感觉对话**：将那股感觉视为一部分自己，在心里对它说：“你好，我自己，你想告诉我什么？”
3.  **接收信息**：等待内在浮现出画面、记忆或直接的答案。
4.  **与内在小孩互动**：若浮现出童年受伤的自己（内在小孩），给予他无条件的接纳、爱与允许。
5.  **疗愈与整合**：通过陪伴、安抚，让受伤的部分释放情绪、感到安全，并最终长大、与主意识整合。

## 个案演示：疗愈不安全感与焦虑

本节我们将通过一个真实个案，看“自我传讯”工具如何应用。案主的困扰是莫名的焦虑和不安全感。

**第一步：连接与初步识别**

我首先引导案主放松，将注意力放在身体感觉上。他觉察到胸口有“很重的感觉”，混合着害怕的情绪。每一种内在痛苦，在身体上都有具体的位置。

**第二步：与内在小孩建立连接**

我引导案主对胸口的感觉使用自我传讯工具。他模糊地看到了一个童年场景：自己蜷缩在墙角哭泣。这是典型的“受伤的内在小孩”形象。

在连接过程中，常会遇到一些障碍，以下是几个常见障碍及应对思路：

*   **障碍一：记忆模糊，没有画面**
    *   **应对**：持续使用自我传讯工具温和引导，信任潜意识会逐渐呈现信息。
*   **障碍二：内在小孩恐惧、封闭，不回应**
    *   **应对**：通过工具持续传递安全感、接纳与耐心，例如说：“我在这里陪着你，你是安全的。”
*   **障碍三：情绪强烈，难以深入**
    *   **应对**：允许情绪释放（如哭泣），这是疗愈的关键步骤。

**第三步：深入创伤场景与情绪释放**

通过持续引导，案主看到了更清晰的创伤记忆：因顶嘴被阿姨用藤条责打，并且被禁止哭泣。当案主在冥想中拥抱那个受伤的自己，并允许她哭泣时，被压抑多年的情绪得到了释放。

**第四步：疗愈、成长与整合**

在感受到无条件的爱后，案主的内在小孩逐渐平静，并在一个想象的“安全屋”中修复、睡觉。随后，她**长大了**，从受伤的小孩状态成长为更轻松自在的年龄。这标志着这部分自我已从创伤中解脱。

我随后引导案主将这部分已疗愈的自我与主意识**整合**。当案主再次回顾创伤场景时，那个阿姨仿佛成了一个“过去的假人”，事件不再带有情绪的冲击力。这验证了创伤已被疗愈，变成了单纯的记忆。

**第五步：更多内在部分的浮现**

当第一部分内在小孩被疗愈后，案主立刻感觉到**另一部分不同的自己**（另一种情绪）浮现出来，“像排队一样”。这说明一个人内在通常有多处“灵魂碎片”需要被看见和疗愈。

## 核心原理与生活应用

上一节我们看完了个案过程，本节中我们来总结其中的核心原理及其对实际生活的意义。

**1. 内在小孩决定外在体验**
我们的情绪状态和行为模式，常由内在未疗愈的“小孩”主导。例如，在社交或约会时，可能一部分自己想行动，但另一部分自己却感到恐惧和退缩，导致“踩油门又踩刹车”的纠结状态。外在技巧无法战胜内在的恐惧。

**2. 疗愈带来真正的一致性**
当内在受伤的部分被疗愈，变得开心、有力量时，你的内在状态会趋于**一致**。这种一致性会外化为真正的自信和动力，无论是人际关系还是个人生活，你都将体验到更顺畅、更快乐的经历。

**3. 个人状态是一个频率区间**
一个人的整体状态，是其所有内在部分状态的“平均值”或“区间”。频率最低（最受伤）和最高（最快乐）的内在小孩，决定了这个区间的范围。疗愈就是提升整体区间的过程，让状态的波动基线不断提高。

**4. 痛苦是疗愈的入口**
生活中的痛苦、焦虑等负面体验，其实是内在小孩发出的“求救信号”。它们通过创造重复的受伤剧情，只为引起你的注意。当你开始愿意觉察和疗愈它们，人生的旧有模式才会真正改变。

## 总结与行动建议

本节课中我们一起学习了疗愈式冥想的核心——“自我传讯”工具，并通过个案看到了如何连接、疗愈受伤的内在小孩。

如果你想开始自我疗愈：
1.  认识到外在困扰源于内在未解决的创伤。
2.  尝试在安静时，连接身体的感觉，并温和地向内在提问。
3.  若遇到困难或希望进行系统学习，可以寻求专业指导。

真爱之门提供了多种针对不同困扰的疗愈工具。持续地疗愈内在，是将人生主动权掌握在自己手中的开始。当越来越多的内在小孩被解救、成长，你将从一个受困的“内在婴儿”，成长为完整而有力量的“内在成人”，真正开始享受生命。
