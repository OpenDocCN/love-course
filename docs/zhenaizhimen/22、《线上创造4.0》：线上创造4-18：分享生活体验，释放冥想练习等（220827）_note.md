## 课程概述 📖

本节课记录了一次线上创造课程的互动过程。学员们分享了各自的生活体验、内在感受，并在引导下进行了释放冥想练习。核心在于学习**允许**和**觉察**——允许任何感觉、想法、评判自然呈现，并觉察它们，而不试图改变或解决。我们将看到，无论是焦虑、评判，还是身体感受，都可以通过简单的“允许发生”来转化。

## 一、评判与存在感：从思思的演讲谈起 🎤

上一节我们概述了课程主题，本节中我们来看看学员Y分享的关于“评判”的体验。他提到观看了思思关于“金钱”与“评判”的演讲，并由此反思了自己的行为模式。

*   **评判的本质**：Y意识到，自己过去评判他人，有时是为了“找存在感”。评判行为本身，可能源于一种内在的不安全感或需要被认可的感觉。
*   **允许评判发生**：经历这次分享后，Y的态度转变为“无所谓了”，即**允许**自己有时会为了找存在感而去评判，而不对此进行二次评判或批判。
*   **对灵性概念的审视**：Y指出，思思演讲中“爱自己”等灵性概念，也可能成为一种新的评判体系，与世俗评判在本质上相似。他强调，许多教导停留在逻辑和描述层面，而非真实的体验层面。

**核心公式**：
```
评判行为 → 可能源于“找存在感”的需求 → 解决方法：允许该行为/感觉发生，不加批判
```

## 二、真正的表扬与能量感受 ✨

在讨论评判后，Y进一步分享了对于“表扬”的细微感受，这帮助我们区分了不同能量带来的体验。

以下是Y对两种表扬的感受对比：

1.  **令人不适的表扬**：例如“你真棒”、“真聪明”。Y感受到这隐含了一种评判和条件——“如果没做好，就不棒了”。这种表扬带来的能量是紧绷和令人不舒服的。
2.  **令人舒适的表扬**：当自己做了“傻事”，对方只是微笑或传递出“你这样做是可以的”的接纳感。这种表扬不涉及好坏评判，而是传递出一种全然的允许和接纳的能量。

**核心概念**：真正的接纳不是基于行为的优劣进行评价，而是传递出一种**允许其存在**的能量。

## 三、生活状态的转变：从“想死”到“观察想法” 🌱

接下来，学员金先生分享了自己学习创造法前后剧烈的状态变化，这是一个关于内在转变的生动例子。

*   **转变前的状态**：去年此时，金先生描述自己“每天就想死”，感到非常暴躁，甚至有极端的冲动。这代表了被痛苦想法完全接管的状态。
*   **转变后的觉察**：学习后，他明白了“死”只是一个想出来的画面。他的核心转变在于，从**认同想法**变为**观察想法**。
*   **思维的重新定义**：金先生总结道：“眼睛的功能是接受光线，思维的功能是思考。” 思维的功能就是处理信息、贴标签。当不再把思维的内容当成必须解决的“问题”时，巨大的平静便产生了。

**核心代码**：
```python
# 转变前：认同思维内容
if thought == “我想死”:
    emotion = “绝望”
    action = “被情绪驱使的极端行为”

# 转变后：观察思维过程
def observe_thought(thought):
    # 认识到：这只是一个“想法”，不是“我”
    awareness = “我注意到我有一个想法：” + thought
    return awareness
# 结果：情绪自然流动，无需“解决”想法
```

## 四、佩佩的困惑：外在转变与内在不开心 😔

学员佩佩的分享提出了一个常见困惑：即使外在行为和性格发生了积极转变，内在的“不开心”感可能依然存在。

*   **外在转变**：佩佩从感觉实验室里大家都说她坏话，变得感觉“很受欢迎”，性格也开朗了很多。
*   **内在不变**：然而，她强调“但是我还是不开心”。这表明，改变外在行为模式或人际关系认知，并不自动等同于解决核心的内在感受。
*   **工具与本质**：佩佩提到，她曾用iPad详细记录个案中的画面、感觉、工具，但后来发现这些记录的都是“升起的东西”（即表面的念头和现象）。真正的疗愈在于引导对方进行“觉察”和“交托”，而非记录这些不断变化的表象。

**核心比喻**：
> 那些升起的想法和画面，就像冬天暖气井盖冒出的热气。我们不需要去收集和记录每一缕热气（升起的现象），只需要知道底下有暖气（内在感觉/模式）即可。

## 五、晨晨的噩梦：梦、现实与潜意识的程序 🎮

学员晨晨分享了自己做噩梦的体验，引导者借此解释了梦、现实与潜意识的关系。

*   **梦与现实的同一性**：引导者指出，梦和现实都是潜意识的投射，没有本质区别。两者都像是由潜意识“驱动程序”运作产生的体验。
*   **梦的浮现意义**：当一个人更愿意触碰内在时，潜意识可能会将过去无法接受的、恐惧的内容（如噩梦）慢慢呈现出来。这可以视为连接更紧密的信号，而非坏事。
*   **处理恐惧画面**：对于脑海中反复出现的画面，关键不是分析内容，而是**觉察它带来的身体感觉**（如心轮发堵、腹部紧绷），然后允许并交托给这种感觉。

**核心模型**：
```
潜意识程序 → 驱动产生 → { 梦境体验， 现实体验 }
处理方式：不纠结内容（代码），而是觉察运行程序时产生的“感觉”（系统负载）。
```

在互动中，晨晨观察到引导者似乎能自动“屏蔽”无厘头问题，保持头脑清空。引导者分享了自己的状态：

*   **允许流过**：引导者也会焦虑，但允许焦虑感流过，“就三分钟”，不沉浸其中。对待无厘头信息，也是“说了就说了，然后就过去了”。

**核心心法**：**不对内在体验（如焦虑）进行二次评判和纠缠，允许它们如云飘过。**

## 七、Jason的释放冥想实录与感悟 🎵

学员Jason进行了一次近40分钟的释放冥想，过程中伴随着强烈的情绪和身体释放（如干呕、大叫）。他的分享和感悟颇具启发性。

*   **论文焦虑的消失**：Jason发现，之前写论文时伴随的死亡焦虑，现在自然消失了。他不再一边写一边批判自己，而是能更平和地面对截止日期。这体现了对过程**交托**后，创造自然发生。
*   **释放中的念头**：冥想中，他脑海中闪过开学后如何在宿舍练习、对同学不礼貌言行的不爽、对宿舍环境的压抑感、离家的不安全感、对感冒的担忧、对未来规划的迷茫等各类念头和感觉。引导者的回应始终是：“**就让这个（感觉/想法）发生**。”
*   **引导方式的进化**：Jason注意到，现在的冥想引导词很少，效果却更好，因为工作完全交给了学员自己的“感知力”。

**核心练习**：
```
1. 觉察任何升起的感受（堵、恶心、压抑、担忧...）。
2. 在心中承认：“我允许这个感受发生。”
3. 不做任何事去改变、压制或分析它，只是允许它以本来的样子存在和流动。
```

*   **引导核心**：面对如此强烈的羞耻和痛苦，引导的核心依然是**允许**：允许羞耻存在，允许厌恶存在，允许矛盾存在，允许“无法快乐相处”的感觉存在。不对这些感受进行任何否定或修正。

**核心领悟**：最深的疗愈，始于对最不堪感受的**全然允许**，而非改变或消除它。

## 九、木子与礼盒的释放及礼盒的领悟 💡

最后，木子和礼盒进行了释放练习。礼盒在结束后分享了一个关于人际关系的深刻领悟。

*   **礼盒的领悟**：他指出，改善一段关系（与父母、伴侣、朋友），并不取决于你外在变得多好（更成功、更有钱、更帅）。即使外在改变，旧有的关系模式依然会重复。
*   **业力即内在模式**：这里所说的“业力”或“纠缠”，指的是**潜意识中固化的关系模式和创伤记忆**。必须回头面对和释放这些过去事件积压的**感受**，关系才能真正转变。
*   **甚至“想改善”也是模式**：引导者补充，甚至那个“我现在要做点什么来改善关系”的想法和行为，也常常是源自那个旧的、充满匮乏和挣扎的模式。真正的转变始于处理过去的感受，而非基于旧模式发起新行动。

**核心总结**：
```
外在改变 ≠ 关系改善
关系改善 = 面对并释放旧事件中积压的感受（内在模式/“业力”）
```

## 课程总结 🎯

本节课中，我们一起学习了“线上创造”的核心实践：

1.  **允许一切发生**：无论是评判、焦虑、羞耻、恐惧，还是身体不适，第一反应是**允许**它存在，不加抵抗或二次评判。
2.  **觉察重于分析**：关注感觉在身体的体现，而非思维对故事内容的分析。想法和画面只是“升起的东西”，感觉才是关键。
3.  **内在体验即现实**：梦、想法、情绪和外在事件，都是潜意识程序的投射，处理它们的方式是统一的——觉察与允许。
4.  **转变在于关系模式**：深刻的人际关系转变，源于清理过去积压的感受，而非追求外在条件的改变。

最终，我们学到的最简单也最强大的工具就是：在每一个当下，**承认并允许你的真实体验如其所是地发生**。这就是释放，这就是创造。
