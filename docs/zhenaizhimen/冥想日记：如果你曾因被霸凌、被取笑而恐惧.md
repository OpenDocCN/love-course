# 冥想日记：如果你曾因被霸凌、被取笑而恐惧

在本节课中，我们将学习如何通过“潜意识释放冥想”来处理因童年霸凌、嘲笑或排斥经历所留下的深层恐惧与创伤。这些未被处理的记忆常常在成年后影响我们的人际关系、自我价值感以及对世界的信任。

---

## 概述：创伤记忆如何影响我们的生活

上一节我们介绍了课程的主题。本节中，我们来看看创伤记忆的本质及其长期影响。

创伤记忆与普通记忆的关键区别在于情绪负载。当你回想普通事件时，你不会产生强烈的负面情绪。然而，创伤记忆被唤醒时，你会重新体验到事件发生时的恐惧、羞耻或痛苦。这些情绪能量被“冻结”在潜意识中，持续影响你的信念与行为模式。

例如，童年被霸凌的经历可能在你潜意识中植入 **“我不被喜欢”** 或 **“我必须讨好他人”** 的信念。这些信念会成为你成年后人际关系中重复出现的模式。

## 第一段分享：唤醒与释放被霸凌的创伤

以下是第一位学员在11月13日的冥想日记分享。

> 我感觉进入那种状态有点恐惧，不愿想起昨天想起的事情。但还是想要释放自己达到解放。我三次慢速呼吸后，意识移到了尾椎骨下，开始回想亲生父母抛弃十天，不知不觉感到手在抓着大腿，显露出其他负面的片段：被同学欺负鼻子出血，拿着长长的大竹竿大树干怼着大个儿。老师说我的种种恶心辱骂词语，全班男同学对我做的事情，喉咙在气胀胸部在堵塞。父亲对我说的“不欺负别人为啥欺负你”。还好脑中画面突然，现在的妈当年挡住了我亲生父亲的说辞，我的气慢慢的开始缓和。不再抗拒发生的事情，做最后的慢速呼吸逐渐平稳。

这位学员在冥想中回溯了多重创伤：被父母抛弃、被同学霸凌、被老师辱骂、被父亲语言暴力。这些事件共同构成了他对世界和他人的深层恐惧，即所谓的 **“AT立场”**——一种无形的心理屏障。

通过面对并释放这些记忆，他缓和了身体的紧张反应（如喉咙发胀、胸口堵塞），情绪也逐渐平稳。这展示了释放过程如何帮助缓解由创伤记忆引发的即时生理与心理反应。

## 第二段分享：连接内在小孩与不配得感

上一节我们看到创伤如何被具体唤醒。本节中，我们来看看创伤如何与“内在小孩”及“不配得感”相连。

以下是第二位学员在13日的分享。

> 上班时头脑又冒出上学时被霸凌时的经历。这种循环不知道重复多少次了。一开始有抵触，不过觉察到后慢慢想着慢慢接纳这感受。到后来情绪越来越糟糕。下班时回到宿舍过了一会冥想释放。冥想回溯感觉自己心中有很深的不配得感，觉得好多东西自己都不配得。边感受边释放，觉得自己心中有个受伤的小孩在哭泣，好想抱紧他，但隐隐约约只能听见一丝声音。

这位学员的经历揭示了创伤记忆的重复性。潜意识中的“内在小孩”——即当年受伤的那部分自我——会不断通过创造类似的生活场景（如人际关系中的被排斥感）来发出信号，渴望被看见和疗愈。

**“不配得感”** 的公式可以表示为：
```
不配得感 ≈ 早期创伤事件 + 由此形成的限制性信念（如“我不值得拥有”）
```

例如，童年时物质需求被拒绝（“父母不给我买想要的东西”）可能形成 **“我不配拥有”** 的信念。被霸凌则可能形成 **“我必须讨好他人才能被接纳”** 的信念，这是讨好型人格的根源。

疗愈的关键在于：不是处理当前触发你情绪的人或事（如冷落你的伴侣），而是回溯并释放最初造成该创伤的根源事件。

## 第三段分享：追溯家族业力与脉轮堵塞

上一节我们探讨了个人历史中的创伤根源。本节中，我们将看到创伤有时能追溯到更深的家族层面。

以下是第三位学员在16日的分享。

> 一开始喉咙底部到胸口处有些拉扯的感觉，然后腹部以下的会阴处也有一股感觉堵在那里。接着就开始冥想，渐渐的感觉到腹部一股灼热的感觉浮上来，胸口也有些闷的感觉游来游去。逐渐感受，然后问自己上一次体验到是什么时候，问自己第一次是什么时候。一幅遗忘已久的画面重播：当时学校里喝饮料的杯归还时可以换一角钱，但一群同班级的朋友们不屑去换，我常常就跟他们收杯子去换。换来的是他们常常的讥笑和羞辱的语言。当时的我都为了能融入群体，每次都只是忍着陪笑。当时我零用钱一天才不到一块钱，只是赚钱的想法以此换取些钱。回溯到这里感觉很重，也明显感觉到，这可能只是我自己记忆中的第一次。还有个第一次源头并不是我自己的记忆。在不断问自己第一次，一阵子后那感觉被拉到了，似乎是我母亲的记忆，并且也是在她小时候。感受一阵子后释放了这些感觉，身体也轻松了。

这位学员的体验非常深刻：
1.  **身体感知能量堵塞**：他明确感知到喉轮（表达）、心轮（爱）和脐轮（关系）的能量堵塞，这与被嘲笑（无法表达）、渴望融入（关系）和金钱羞耻感直接相关。
2.  **回溯根源事件**：他忆起通过收集瓶子换钱却被同学讥笑的场景，这同时关联了 **“人际关系创伤”** 和 **“对金钱的羞耻感”**。
3.  **连接家族创伤**：通过持续追问“第一次”，他在潜意识中链接到了母亲在童年时期的类似创伤记忆。这说明了某些限制性信念和情绪模式可能通过家族业力传承。

这段分享揭示了“潜意识释放冥想”的强大之处：它不仅能处理个人记忆中的创伤，还能触及并释放那些从家族中继承而来、本不属于你的信念与情绪负荷。
