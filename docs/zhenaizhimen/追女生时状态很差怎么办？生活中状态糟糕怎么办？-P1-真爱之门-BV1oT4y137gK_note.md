# 课程名称：状态管理基础 - 第1课：认识状态循环 🔄

在本节课中，我们将要学习一个普遍影响我们生活与社交表现的核心问题：状态波动。我们将探讨状态好坏时的不同体验，分析状态低迷时的具体表现，并引出对背后循环机制的思考。

## 状态的两极体验

当你处于良好状态时，你去搭讪、与女生聊天、和女生约会，你会感觉无往不利。

相反，当你状态极差时，你会发现做什么都不顺利。搭讪会失败。聊天会陷入僵局。和女生约会会出现冷场。

我相信每位观看这个视频的人都曾经历过，并且是无数次地经历过状态不好的时刻。

## 状态低迷时的连锁反应

状态糟糕时，生活也会变得一团糟。你感觉在生活中什么都不想干。

不仅如此，当你去做什么事情时，会感到处处不顺。例如，等外卖时，外卖员送错了地方。工作时，发现毫无创意，什么都不想干。看书时，看了没两眼就发现根本看不下去，只想睡觉。

## 状态与负面体验的关联

当你状态不好时，你会发现所有那些你恐惧、厌烦、不喜欢的东西全都出现了。

这引出了一个核心问题：到底是什么原因，导致我们会经历这样的循环？

---

本节课中，我们一起学习了状态对我们行为与结果的巨大影响，识别了状态低迷时的具体表现，并提出了对背后成因循环的疑问。理解这一现象是管理自我状态、提升生活与社交质量的第一步。
