# 课程01：印度疯狂摩天轮体验教程 🎡

在本节课中，我们将跟随一段视频记录，学习如何在印度体验一座被称为“疯狂摩天轮”的游乐设施。整个过程包括发现游乐场、购票、乘坐体验以及感受分享。我们将把这段充满现场感的叙述整理成一份步骤清晰的教程。

---

## 发现与前往游乐场 🚶‍♂️

上一节我们介绍了课程目标，本节中我们来看看如何发现并前往这个特别的游乐场。

视频记录者从地铁出来后，直接前往一个叫“红宝”的地方，意外发现外面有一个游乐场。游乐场里最引人注目的就是一个高速旋转的摩天轮。

## 购票与准备过程 🎫

在决定体验后，下一步就是购票和做乘坐前的准备。

以下是购票时发生的对话与关键信息整理：
*   价格询问：最初询问的价格是“四万块钱”，但经过沟通，实际价格为“五十块钱”一个人。这里的“万”可能是指当地货币“卢比”的单位口语表达。
*   支付方式：使用现金支付，票款为“一万块钱”（应指100卢比左右）。
*   乘坐规则：一个座舱（“一盒”）可以坐两个人。

拿到票后，他们便前往摩天轮准备乘坐。

## 乘坐体验与感受 🌀

购票完成后，最核心的体验环节开始了。本节我们将详细拆解乘坐过程中的感受。

摩天轮并非缓慢观光型，而是高速旋转。启动后，转速非常快，大约**6秒钟就能完成一圈**。这种高速旋转带来了强烈的失重感和离心力。

以下是乘坐过程中的主要感受描述：
*   初始感觉：感觉“不危险”，但设计“让我们感觉危险”。从地面看，设施“很稳定”。
*   高速体验：当摩天轮真正转起来后，体验变得极为刺激。用“好可怕”、“很猛”、“太棒了”、“爽”等词形容。
*   安全措施：与国内游乐场不同，这个设施没有存放个人物品的柜子，乘坐者需自己抓牢物品。
*   结束状态：结束后感到非常激动和兴奋，有人用“我死了”来形容这种极致的体验感。

## 体验总结与建议 📝

经历了高速旋转的刺激后，我们最后来总结一下这次体验的核心要点。

本节课中我们一起学习了在印度体验疯狂摩天轮的全过程。关键点在于：这是一个高速、刺激的游乐项目（旋转周期约 **T = 6秒/圈**），而非缓慢的观景摩天轮。体验前需确认好实际价格，并做好随身物品在高速中固定的准备。对于追求刺激的游客，这是一个独特而令人难忘的体验。

**核心体验公式可以概括为：刺激度 ∝ (1 / 旋转周期)**。周期越短，旋转越快，体验越刺激。

---
**课程总结**：本节课通过分解一段实地视频，详细说明了在印度体验高速摩天轮的步骤、注意事项和核心感受。整个流程从发现、购票到乘坐，突出了其**高速**（6秒一圈）与**刺激**的核心特点，为初学者提供了直观的参考。
