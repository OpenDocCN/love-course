# 疗愈日记 P1：如何拔掉阻碍你创造金钱的潜意识限制？ 💰

在本节课中，我们将通过一个真实的疗愈个案，学习如何识别并拔除那些阻碍你创造金钱的潜意识限制。你将了解到，外在的金钱困境往往是内在信念和创伤的反映，并学习到具体的疗愈步骤。

## 概述：金钱困境的内在根源

很多朋友感到困惑：明明很想赚钱，也付出了很多努力，但收入依然很低。从真爱之门的视角来看，每个人的外在体验都是由内在状态创造的。你的外在体验是**果**，你的内在状态和信念系统是**因**。

**公式：外在现实 = f(内在状态)**

如果你发现自己无法创造期望数量的金钱，这说明在你的内在状态或信念系统层面，存在某些潜藏的信念模式或创伤模式，阻碍了你创造想要的体验。

## 第一节：连接潜意识的金钱卡点

上一节我们介绍了金钱困境与内在状态的关系。本节中，我们来看看如何通过连接潜意识，找到具体的金钱卡点。

在个案开始时，疗愈师引导案主进入放松状态，连接潜意识，去感受与金钱卡点相关的身体感觉和内在画面。

以下是案主连接到的关键画面和感受：
*   **身体感觉**：案主首先感受到腹腔有一股害怕的感觉。
*   **内在画面**：随后，脑海中浮现出一个垃圾堆的画面，天色已黑，街上无人，能看到别人家中家人团聚。
*   **核心感受**：在这个画面中，案主感觉自己很渺小，并被父母抛弃了，但最初并未在画面中看到自己的形象。

## 第二节：定位创伤场景中的“内在小孩”

仅仅连接到创伤画面还不够，我们需要找到困在场景中的那个“内在小孩”（即过去受伤的自我），才能进行疗愈。

我引导案主在画面中寻找自己。最终，他看到了一个很小的自己，感到害怕、无助，不确定回家的路，感觉家很遥远。

以下是关于这个“内在小孩”的更多信息：
*   **创伤事件**：父亲为了惩罚他不听话，将他丢在路边。
*   **即时影响**：他独自等待，看着一辆辆摩托车经过，期待父母来找他，等了很久父亲才出现，并且一直在骂他。
*   **植入的信念**：这个事件植入了一个核心信念——**“我不能表现出父母不喜欢的部分，否则就会被抛弃。如果被抛弃，我就会死。”**

## 第三节：在安全场景中尝试疗愈

找到“内在小孩”后，我们需要将他带离创伤场景，到一个安全的地方进行疗愈。

我引导案主带着内在小孩离开街道，前往他内心觉得安全的地方——一个想象的小岛。起初，他看到了自己和家人开心玩耍的画面，但随后发现那只是他渴望而不可得的幻想，现实中他依然独自一人，感到孤独和委屈。

## 第四节：高灵介入与信念清理

在安全场景中，我们引入了疗愈的关键力量。我引导案主呼唤一只猫咪来到内在小孩身边，因为猫咪常是“爱的老师”。

以下是猫咪出现后的关键互动与疗愈过程：
*   **猫咪的身份**：猫咪自称是案主的朋友，由地球母亲派来陪伴他。它传递了一个重要信息：**“你本来就是被宇宙无条件爱着的，那些伤痛体验是你自己选择去学习的。”**
*   **识别限制性信念**：案主表达出“我没有能力照顾自己”的感觉。猫咪引导他发现，这个信念源自父亲（“爸爸教我”）。**这个因创伤植入的信念，正是阻碍他创造金钱的核心障碍之一。**
*   **可视化清理**：案主在心轮位置看到了这个信念的实体——像黑色石头一样的堵塞物。他请求猫咪帮助清理。在内在画面中，猫咪将这些堵塞物不断刨出。
*   **疗愈效果验证**：清理后，案主描述内在小孩的感觉从害怕无助变为“很放松、很平静”，验证了清理的有效性。

## 第五节：下载宇宙金钱智慧

清理障碍后，通道变得畅通，此时可以连接更高维度的智慧，获得关于创造金钱的指导。

案主开始与这只猫咪（实为一位高维指导灵）深入沟通，下载信息。

以下是猫咪传递的关键信息：
*   **高灵身份**：猫咪是案主的“心轮使者”，在宇宙中通过管道与地球上的一部分人连接，传递信息。
*   **与金钱的关系**：猫咪负责帮助人们**创造金钱**，它与“金钱与智慧天使”是搭档。
*   **关于案主的问题**：
    *   **问能否赚到三千万**：猫咪展示“很多金子”的画面，直觉远超三千万。
    *   **问何时开始赚很多钱**：猫咪展示一个**管道由细逐渐变粗**的画面，寓意随着内在疗愈的深入，金钱通道会拓宽。
    *   **问创造金钱的建议**：答案是**“继续疗愈”**。
*   **与真爱之门的关系**：猫咪与“真爱之门”意识之间有一条非常粗大的连接管道。

## 第六节：疗愈验证与个案收尾

获得智慧下载后，我们需要回到最初的创伤场景，验证疗愈是否真正完成了转变。

我引导案主回到“被父亲丢在路边”的场景。此时，他的感受发生了根本变化：他对父亲说出了“我们回家吧”，并能够感恩父亲帮助他体验这些课题。父亲也回应“我们回家吧”。这份恨意与创伤得到了化解。

最后，我引导案主从潜意识状态回归现实。个案结束后，案主透露了一个惊人的同步性：疗愈师在个案前写在记录里的“金钱与智慧天使”，正是案主在连接中看到的、与猫咪搭档的那位高灵。这印证了连接的真实性与高维协作。
