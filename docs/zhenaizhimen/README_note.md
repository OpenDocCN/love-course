# 真爱之门 🚪💖

在本教程中，我们将学习如何构建一个名为“真爱之门”的简单程序。这个程序的核心是判断两个人是否匹配，我们将通过一个基础的“兼容性”计算来实现。整个过程将涉及变量、条件判断和基本输入输出的使用。

## 1. 定义核心变量

上一节我们介绍了本教程的目标，本节中我们来看看程序需要哪些基础信息。任何判断都需要输入数据，这里我们需要两个人的名字和一个代表他们之间某种联系的“兼容性分数”。

以下是程序开始时需要定义的两个关键变量：
*   `name1`： 第一个人的名字，例如 `"小明"`。
*   `name2`： 第二个人的名字，例如 `"小红"`。
*   `compatibility_score`： 一个介于0到100之间的数字，代表初步的匹配度，例如 `75`。

在代码中，我们可以这样初始化它们：
```python
name1 = "小明"
name2 = "小红"
compatibility_score = 75
```

## 2. 设置匹配阈值

定义了基础数据后，我们需要一个标准来判断分数是否足够高。这个标准就是一个“阈值”，只有当兼容性分数达到或超过这个阈值时，我们才认为匹配成功。

以下是关于阈值的说明：
*   阈值是一个预设的临界值。
*   我们可以将其设置为 `60`。
*   如果 `compatibility_score >= 60`，则视为通过初步匹配。

我们可以用以下公式表示这个逻辑：
**匹配条件： compatibility_score ≥ 阈值**

## 3. 进行条件判断并输出结果

现在，我们有了数据和标准，接下来就是根据比较结果做出判断并告知用户。这是程序做出决策并产生输出的关键步骤。

以下是实现判断与输出的逻辑步骤：
1.  使用 `if` 语句检查分数是否达到阈值。
2.  如果达到，输出一条成功匹配的信息。
3.  如果未达到，输出一条未能匹配的信息。

其核心代码结构如下：
```python
threshold = 60
if compatibility_score >= threshold:
    print(f"{name1} 和 {name2} 的真爱之门打开了！")
else:
    print(f"{name1} 和 {name2} 的真爱之门暂时关闭了。")
```

## 4. 尝试运行程序

理论需要实践来验证。让我们将前面所有的代码片段组合起来，形成一个完整的、可以运行的程序。

以下是完整的代码示例，你可以复制并运行它：
```python
# 定义变量
name1 = "小明"
name2 = "小红"
compatibility_score = 75

# 设置阈值并判断
threshold = 60
if compatibility_score >= threshold:
    print(f"{name1} 和 {name2} 的真爱之门打开了！")
else:
    print(f"{name1} 和 {name2} 的真爱之门暂时关闭了。")
```
运行这段代码，你将在屏幕上看到结果：“小明 和 小红 的真爱之门打开了！”。

---

本节课中我们一起学习了如何构建“真爱之门”程序。我们掌握了定义变量（如姓名和分数）、设置判断阈值、以及使用 `if-else` 语句进行条件判断并输出结果的基本流程。你可以通过修改 `compatibility_score` 的值或 `name1` 和 `name2` 的内容来探索不同的结果。
