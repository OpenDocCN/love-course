# 如何避免遇到操蛋、差劲的人事物

在本节课中，我们将要学习如何分辨并转换你的内在状态，从而彻底改变你对外部世界的体验。我们将探讨“匮乏”与“丰富”两种内在状态如何像滤镜一样，塑造你眼中的人事物，并学习如何通过“觉察”与“释放”这两个核心步骤，摘下“匮乏的眼镜”，持久地创造你想要的生活。

## 从个人经历谈起：两种截然不同的世界体验

上一节我们介绍了课程的核心目标，本节中我们来看看两种不同的内在状态如何具体影响我们的生活。

我从去年开始接触内在工作，每天进行释放练习。在这个过程中，我发现了两种有趣且截然不同的体验状态。

以下是第一种体验状态：

*   你会感觉特别开心，看每个人都像是朋友。
*   即使是路人、情敌或你不喜欢的人，你也会觉得他们挺可爱。

以下是第二种相反的体验状态：

*   你会突然感觉看什么都不爽。
*   即使是别人一句普通的话，也会让你非常生气。
*   你看朋友、路人甚至陌生人，都会觉得他们很傻逼。

这两种状态就像波动一样，在我的生活中交替出现。在某天下午醒来时，我可能会莫名感到很不爽。这时，我其实是戴上了一个**匮乏的眼镜**。

## 匮乏的眼镜：你的内在如何为世界染色

上一节我们描述了两种不同的体验，本节中我们来深入探讨其背后的原理：内在状态如何像眼镜一样，为外部世界染色。

在这个眼镜里，所有客观上正常的事物，看起来都很垃圾。原因并非那些人事物客观上很糟糕，而是你的**主观内在状态**在起作用。

我在“创造法则”中讲过：你所戴的那个意识不到的隐形眼镜，会把你的外部世界染色。

**公式：你的外部体验 = 你的内在状态**

一个内在愤怒的人，他的愤怒会把外部世界染色。他会在生活的所有证据中寻找能让他发怒的事情。你身边或许有这样的人，甚至你自己就是这样的人。他们总是没来由地发火，根本原因在于他们内在的预设是：“所有人都是傻逼，都是他们的错。”

因此，带着愤怒的眼镜，在任何状态下看到的东西都会觉得不爽。

## 一个生动的例子：拒绝小费的出租车司机

为了更具体地说明内在预设如何影响行为，我们来看一个例子。

在“创造法则”里，我举过泰勒坐出租车的例子。他遇到一个脾气非常差、感觉全世界都很操蛋的出租车司机。泰勒想做一个试验，到站后给了司机20美元小费并表示感谢。

司机的反应是：他根本无法接受，甚至把钱扔了回去。因为接受善意小费这件事，与他“世界充满恶意”的内在信念完全相反，他做不到。

同样，一个充满恐惧的人，恐惧也会为他的世界染色。比如我的母亲，出门时每一分钟都在担心赶不上车、丢东西，她的恐惧滤镜让她无法平静地体验旅程。

最可怕的是，你意识不到自己戴着一副**隐形的VR眼镜**。你意识不到世界被过滤了，你看到的并非真实。那些让你暴怒、嫉妒的事情，客观上可能只是一件无关紧要的小事。

## 匮乏是常态，丰富是偶然

回想你的生活，是否经常有某段时间，感觉看什么都不爽，一切好像都脱离了控制？

我相信这是很多人的常态。夸张地说，一天中可能有80%的时间，我们都处于这种**匮乏的状态**中，感觉什么都不对劲，每个人都带有恶意。

但你也会偶尔体验到另一种状态：感觉一切都没问题，即使是你讨厌的人，今天也觉得有点可爱。你不再觉得他的话是在控制或攻击你。

过去我不明白为什么生活会在两个世界中切换。现在我明白了：当你带着匮乏的眼睛，匮乏的思维和感受就会为你的外部世界染色。

你的所有外部体验，完全取决于你的内在体验。这不是比喻，你的外部世界是由内在创造出来的。

**代码示例（比喻）：**
```
if (inner_state == "焦虑") {
    external_experience = "一切都糟透了";
} else if (inner_state == "平静") {
    external_experience = "一切都很可爱";
}
```

你已经体验过这两种状态。直观的感受是：我们不想要匮乏，而想拥有获得新衣服、被赞美时的快乐状态。我们一直追寻它，因为在这种状态下，你会感觉整个世界都爱你。

## 旧活法走不通：向外寻求的快乐是短暂的

如果你看过我之前的视频，你会明白通过任何外在途径获得的快乐都是非常短暂的。

无论是买了新衣服、取得新成就，这种快乐只能持续很短时间。之后，你会再次跌回那种熟悉的、操蛋的匮乏状态中。

大部分人不知道别的活法，于是陷入循环：状态没了 -> 不快乐 -> 去做更多事获取短暂快乐 -> 再次跌落。这没有用，因为你的内在基底状态是匮乏的，任何外在刺激带来的好状态终会消散。

## 新活法：主动摘下匮乏的眼镜

有一种新的活法：主动把你内在那个匮乏的隐形眼镜摘掉，换上一副丰富的眼镜。

这听起来科幻，但其实你经常在两种眼镜中切换。你大部分时间戴着匮乏的眼镜，但偶尔会因为外部刺激或不小心进入当下，体验到丰富的状态。

现在我要告诉你：你可以自己控制这种切换。你可以让自己持久地摘掉匮乏的眼镜，戴上稳定的丰富眼镜。

如何做到？这需要做内在工作，分为两个核心步骤。

## 第一步：觉察——看见你的滤镜

内在工作的第一步是**觉察**。如果你察觉不到，你会一直用“我很好”来欺骗自己。

当朋友问“你怎么样？”，我们总是机械地回答“我很好”。但这不是真实的答案。如果你真正静下来，感受你的胸口和身体，让外部世界安静，你会发现内心其实有一些焦虑和躁动不安，并不是想象中的“很好”。

首先，你要察觉到，自己其实处在一种持久的、匮乏的焦虑中。之所以感觉不到，是因为我们一直在**向外寻求**并习惯性逃避这种焦虑。

因为感受焦虑是痛苦的，所以我们会用各种上瘾行为来逃避：不断看手机、吸烟、喝酒、滔滔不绝地讲话……人们无法忍受自己的内在状态，已经对自己撒了很久的谎：“我没问题”。

所以，第一步是觉察你的内在真实感受。

其次，是觉察我在视频开头讲的那种状态：当你突然感觉看所有人都是傻逼时。这个时候，希望这个视频能像一个响指提醒你：嘿，这不是外界的问题，是因为你戴上了匮乏的眼镜。

以下是帮助你开始觉察的要点：

*   在下一次感到看什么都不爽时，尝试察觉。
*   你可能无法立刻做到，但可以事后后知后觉。
*   关键是，最终你能意识到：“我正处在那个操蛋的、匮乏的状态里。”

这说明你觉察到了。你看到了自己被一个滤镜控制住了。那个在匮乏状态中的你，并不是真实的你，而是被有色眼镜蒙蔽的你。如果能觉察到，下一步就好办了。

## 第二步：释放——清理内在的堵塞

第一步是觉察，那么第二步是什么？第二步就是**释放**。

这种匮乏的内在状态，并非我们真实的样子。你真实的样子是内心本自具足的丰富状态，但外面包裹了厚厚一层信念和制约，比如“你不配”、“你不够好”。

**公式：真实的你 = 丰富的内核 + 负面的信念包裹层**

你真正想回到内在的丰富状态，就必须把这些“垃圾”释放掉，不能再无视或欺骗自己。

第一步是看到这层包裹，第二步就是释放它。这里教你一个简单的方法，也是我常教的**圣多纳释放法**。

当你处在看什么都不爽的状态中并察觉后，过去你可能会发火、吵架或用上瘾行为逃避。但现在，如果你想彻底改变，就不能再逃避了。

你唯一的办法就是去**感受它**。

以下是进行简单释放的步骤：

1.  **感受身体**：不带评判地去感受你内在的那团火。感受腹部的紧绷、胸口的堵塞，感受身体里那团被堵住的痛苦能量。
2.  **接纳与臣服**：完全接纳并感受这种痛苦、紧绷的感觉。将身体完全交托给它，臣服于这个感觉。
3.  **持续感受**：这个过程可能持续十分钟到半小时。坐着或躺着，静静感受，承认这些负面能量一直存在于你身体里。
4.  **等待流走**：随着时间的推移，你会发现这些感觉慢慢减轻，仿佛从身体里流走。最终，你会体验到一种如释重负的感觉。

这些能量之所以困在你体内，是因为过去当它们（可能来自童年评判）出现时，你因恐惧而压抑了它们，从未真正感受过。它们其实很想离开，你只需要给它们一个通道——那就是感受。

## 系统的释放方法：连根拔起负面模式

为了更彻底、更系统地释放，我在“创造法则”课程中教授一套专业的方法。它分为三个步骤，教你如何放下那些控制你的负面感受，进入持久稳定的快乐状态。

无论你是否是学员，通过臣服于感受或系统方法释放后，你会发现：你对待人事物的看法变了。

这是我的亲身体会：在做释放前，我觉得围观的路人很傻逼。但我意识到，问题不在他们，而是我的内在处于**操蛋的投射**中。我觉察到了。

然后，我使用课程中的系统释放方法：

1.  **慢速呼吸法**：让自己进入当下，扎根，让头脑的噪音安静。
2.  **寻找根源**：进入潜意识，寻找当下操蛋感受的根源。比如，害怕失去伴侣的根源，可能不是对方，而是婴儿时期对失去爱的恐惧。
3.  **释放根源**：通过冥想与潜意识互动，臣服并接纳这些感觉，让根源事件浮现，并将其连根拔起。

**比喻：负面感受像树上的叶子，系统释放是找到树根并拔掉它，让叶子永久消失。**

释放完成后，我回到现实，发现世界完全不同了。之前觉得操蛋的人和事，现在感觉毫无问题。即使是情敌或害怕的人，也觉得可爱。因为当你处在**丰富的状态**中——“我值得被爱，我很可爱”——这个状态就会为你的整个外部世界染色。
