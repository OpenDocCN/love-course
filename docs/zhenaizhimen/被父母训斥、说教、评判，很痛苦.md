# 被父母训斥、说教、评判，很痛苦

## 概述
在本教程中，我们将通过一个真实案例，学习如何识别、回溯并疗愈由原生家庭创伤（如被父母训斥、说教、评判）所引发的深层痛苦。你将看到疗愈师如何引导案主连接潜意识，运用特定工具释放负面情绪，并最终唤醒内在力量与爱的过程。

---

## 1. 认识原生家庭创伤的根源与影响

原生家庭课题是每个人都需要面对的根源性课题。许多人在成长过程中都有被父母打骂、打压或评判的经历。虽然这些不愉快的记忆可能随时间淡忘，但它们并未真正消失。

这些创伤体验被深埋在潜意识中，以不易察觉的方式持续影响着我们的日常情绪。当外界的人、事、物触动了你的“情绪按钮”，这些潜藏的创伤便会浮现，让你再次体验到痛苦。

例如，有时他人一句看似无关紧要的话，却让你感到莫名气愤或情绪激动。这正是因为对方的言行唤醒了你内心深处的原生家庭创伤。

上一节我们认识了创伤的潜伏与触发机制，接下来我们将探讨如何从根本上疗愈这些创伤。

---

## 2. 疗愈的核心：用合一爱对待内在创伤

从真爱之门的视角来看，彻底疗愈潜意识创伤的唯一途径，是**用合一爱的方式去对待创伤和受伤的自我体验**。这意味着愿意去爱自己，接纳自己的所有感受。

爱本就存在于我们内心。我们之所以感受不到它，是因为原生家庭的创伤让我们的“内在小孩”不再相信自己值得被爱。

在我们的成长过程中，父母是权威。我们会无意识地迎合、顺从父母，以获取他们的爱与认可，这对于尚未具备独立生存能力的孩子来说，是一种生存本能。

**公式：童年生存策略 → 压抑真实自我 → 形成创伤**

当我们被父母以非爱的方式（如打压、评判）对待时，幼小的我们会感受到父母的负面能量。如果不顺从，我们就会恐惧失去父母的爱，这近乎一种死亡的体验。因此，我们选择了压抑自己的真实感受、自由意志和内在力量。

然而，如今的我们已是拥有自由意志的独立个体。我们完全可以选择挣脱原生家庭创伤的枷锁，不再受其影响，活出自己的人生。

---

## 3. 疗愈流程详解：从回溯到清理

以下是通过个案展示的具体疗愈步骤。每个步骤前，疗愈师都会引导案主进入特定状态。

### 3.1 连接与回溯创伤场景
疗愈师首先引导案主闭上眼睛，回溯最近一次与父亲发生争执并感到痛苦的具体场景。

**引导代码示例：**
```
1. 请回忆与父亲起争执的场景。
2. 观察场景中自己的表情（如哭泣、愤怒）。
3. 感受身体出现的反应（如胸口发闷）。
```

案主成功连接到了当时的自己，感受到愤怒、无奈和胸口的堵塞感。

### 3.2 运用超意识工具补充爱的能量
疗愈师引导案主为那个“受伤的内在小孩”（即当时的自己）补充“合一爱”的频率能量。

**核心概念：内在小孩**
> 内在小孩是我们内在感受的拟人化象征。当内在小孩感受到爱与呵护，我们自身的感受也会随之转变。

在接受一段时间的“合一爱”能量后，案主反馈胸口发闷的感觉消失了，愤怒和无奈的情绪也减少了约70%。

### 3.3 深入回溯与清理能量植入
疗愈师引导案主沿着时间线，回溯更早的、更根源的创伤事件（如小学时因偷玩被父亲抽打）。

**疗愈动作：**
1.  **识别低频能量**：引导案主“看到”父亲当时传递过来的暴躁、不耐烦的“黑红色能量”如何进入自己的身体（如喉咙、胸腔）。
2.  **清理能量**：引导合一爱的能量到来，清理和净化这些植入体内的非爱能量。
3.  **剪断能量管道**：引导案主看到自己与父亲之间存在的负面能量连接管道，并运用“课题分离”工具，将不属于自己的情绪能量归还，同时剪断管道。

完成这些步骤后，案主再回看被抽打的场景时，内心的恐惧和悲伤已大幅消散，内在小孩变得勇敢、有力量。

---

## 4. 验证疗愈效果与巩固转变

在疗愈了数个根源性创伤场景后，疗愈师引导案主重新回顾最初引发痛苦的几个场景，验证内在体验是否发生转变。

**验证结果：**
*   案主内心焦躁、紧张的感觉已经消散。
*   面对曾经的创伤场景，内心感到平静，充满了安全感和力量。
*   案主反馈，感觉自己“内在充满了力量很放松”，以后和父亲说话会更有底气，恐惧减少。

这证实了当内在的“因”（创伤感受）被疗愈后，所创造的“果”（对事件的体验）也随之彻底改变。

---

## 5. 疗愈后的反馈与总结

疗愈结束后，案主回到现实，给出了直接的反馈：
*   感受到了前所未有的、强烈的内在平和与放松。
*   意识到自己过去的内在力量一直被父亲压制。
*   相信未来生活中的恐惧感将大幅减少甚至消失。

**总结**
本节课中，我们一起学习了原生家庭创伤的疗愈全过程：
1.  **识别模式**：认识到当前的情绪困扰源于早期的创伤体验。
2.  **回溯连接**：勇敢地连接并看到那个受伤的“内在小孩”。
3.  **能量疗愈**：运用“合一爱”的能量去呵护、滋养受伤的部分，并清理外来的负面能量。
4.  **切断枷锁**：通过“课题分离”，剪断与父母之间的负面能量连接，收回属于自己的力量。
5.  **验证巩固**：通过回顾验证疗愈效果，让新的、有力量的体验取代旧的创伤体验。

疗愈的本质是**爱自己**。当你愿意看到并疗愈内在的创伤，你就能逐渐摆脱历史模式的束缚，以更轻松、自信、勇敢的状态迎接生活。每个人都可以通过自我觉察与爱的练习，完成这场内在的转变。
