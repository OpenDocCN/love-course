# 课程01：为什么你总觉得自己很差劲、很失败？ 🧠

在本节课中，我们将探讨一个普遍的心理困境：为什么你总觉得自己很差劲、很失败，并感觉没有人喜欢你？我们将通过一个现场互动练习，深入剖析“自我认同感”这一核心概念，并理解它如何像滤镜一样影响我们感知世界和与人互动的方式。课程将揭示内在的“创造循环”，并为你提供跳出这个循环的初步视角。

***

## 现场互动：体验能量的“卡住”与“流动”

课程从一个现场练习开始。一位名叫小豪的学员被邀请上台，他分享了自己内心的矛盾：在别人眼中他阳光向上，但自己却时常感到自卑和犹豫，尤其是在与异性相处时。

为了让他和在场所有人体验不同的心理状态，我们进行了两组对比练习。

以下是第一个练习：感受表达时的“卡住”。

我们尝试大声喊出来。第一次喊叫时，虽然声音很大，但能明显感觉到他在“卡住”自己，就像一边踩油门一边踩刹车。内在有一部分能量在阻止他全然表达。

接着，我们进行了第二个练习：在“抗拒”与“接纳”两种能量场中表达自己。

*   **在抗拒能量中表达**：小豪需要在30秒内做一个即兴演讲，而台下观众会用最恶毒的语言攻击他。他描述当时的感受是：头脑一片空白，感觉被排挤，非常慌乱，并试图用更大的声音去盖过攻击，但表达依然很“卡”。
*   **在接纳能量中表达**：同样的30秒演讲，台下观众转变为全力赞美和拥护他。他描述的感受是：思路清晰，压迫感减轻，虽然仍有紧张，但整体状态舒畅很多。

通过前后对比，可以清晰地感受到，在不同的外界能量（抗拒 vs. 接纳）下，我们内在的表达状态和能量流动方式截然不同。

为了让所有人都有切身体会，现场学员两两组队，轮流体验“攻击/表达”和“赞美/表达”两种模式。

分享环节中，学员们反馈了多种感受：
*   被攻击时，内心会感到被吓到或习惯性防御。
*   被赞美时，反而会产生“羞愧感”和“不配得感”，怀疑对方的赞美是否真诚。
*   即使明白这是练习，情绪依然会被对方的强烈情绪所影响。

这些真实的反馈引出了我们本节课的核心议题：**自我认同感**。

***

## 核心模型：三种自我认同感滤镜

自我认同感，简单说就是你内心深处如何看待和评价自己。我们可以将其大致分为高、中、低三种水平，它们就像三副不同的眼镜（滤镜），决定了我们看到什么样的世界。

以下是三种自我认同感的定义：

1.  **高自我认同感 (High Self-Esteem)**：核心信念是“我值得被爱，我本来就很棒”。当受到赞美时，会觉得像听到“今天天气真好”一样自然，并坦然接受。
2.  **中等自我认同感 (Medium Self-Esteem)**：核心信念是“我大体还不错，但仍有不足”。当受到赞美时，能看到赞美，但内心会有一部分声音感到害羞或抗拒，可能会回应“哪里哪里”。
3.  **低自我认同感 (Low Self-Esteem)**：核心信念是“我不行，我不配”。当受到赞美时，**滤镜会让他根本看不到赞美**，反而会怀疑：“他说的是真的吗？他是不是在取笑我？这不对劲。”

低自我认同感者的滤镜最为关键，它会导致一个严重的后果：**选择性收集信息**。他们只会注意到那些符合“我不行”信念的证据（比如失败、拒绝），而自动过滤掉那些“我很棒”的证据（比如成功、赞美）。

***

## 创造循环：滤镜如何塑造你的现实

上一节我们介绍了三种自我认同感滤镜，本节我们来看看低自我认同感滤镜如何形成一个自我实现的、令人痛苦的循环。这个循环就是“创造法则”在心理层面的体现。

我们可以用以下流程来描述这个循环：

```
收集信息 (通过低自我认同滤镜) -> 唤醒不安全感 -> 采取焦虑/防御性反应 -> 得到负面结果 -> 印证初始滤镜
```

让我们用一个学员提供的真实搭讪案例来分解这个循环：

1.  **收集信息**：他夸一个女生可爱，对方转头不理他。通过他的“低自我认同”滤镜，他收集到的信息是：“这个女生好装”（负面解读）。
2.  **唤醒不安全感**：这个解读唤醒了他“我价值低”、“我不配”的不安全感，内心感到受挫。
3.  **采取反应**：为了防御这种不安全感并试图拉平价值，他采取了“打压”对方的策略（例如，说她头发油）。这是一种索取认可、对抗的能量。
4.  **得到结果**：女生直接走掉了。
5.  **印证滤镜**：这个结果完美印证了他最初的滤镜：“看吧，我就是不配，我得到这种反应是正常的。”如果女生没走但反应冷淡，他也会用“及时止损、高价值离场”等策略逃离，最终依然印证“我不配得到好的反应”。

**这个循环的可怕之处在于**：即便偶尔出现了“好”的证据（比如另一个女生对他反应很好），低自我认同滤镜也会将其扭曲（“我反而被吓到了，这不对劲”），或者迅速忽略，继续专注于下一个“不好”的证据。如此循环往复，不断创造出“我总是很失败、没人喜欢我”的现实体验。

***

## 课程总结与行动建议

本节课中，我们一起学习了“自我认同感”如何作为滤镜影响我们的感知。通过现场练习，我们体验了在抗拒与接纳能量下表达自己的不同状态。我们重点剖析了**低自我认同感**的滤镜，它会导致我们只收集负面信息，从而启动一个“收集信息 -> 唤醒不安全感 -> 防御反应 -> 负面结果 -> 印证滤镜”的**创造循环**，让你不断体验并强化“自己很差劲”的现实。

要打破这个循环，起点是**觉察**。意识到自己正戴着怎样的滤镜看世界，是改变的第一步。当你再次感到“我不配”或怀疑他人的善意时，可以停下来问自己：我是不是在用低自我认同的滤镜过滤信息？有没有被我忽略掉的正面证据？
