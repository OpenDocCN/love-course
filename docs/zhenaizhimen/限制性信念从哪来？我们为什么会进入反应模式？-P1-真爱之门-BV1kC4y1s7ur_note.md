在本节课中，我们将学习“条件作用”（Conditioning）这一核心概念。我们将探讨限制性信念如何形成，以及人类为何会进入自动化“反应模式”。理解这些机制是迈向更清醒、更临在生活的第一步。

## 什么是条件作用？ 🔗

上一节我们介绍了课程主题，本节中我们来看看“条件作用”的具体定义。

作为儿童，我们被训练成要以一定的方式在地球上生存。我们被灌输了父母的信念、思想和观念。这些本来不是我们自己的，但我们被期待着要把这些当成是自己的。如果我们不接受他们的这些条件限制，我们可能会被评判、排斥或者否定。这只是条件作用的一个方面。

## 条件反应的经典实验 🐕

理解了基本定义后，我们通过一个著名实验来具体看看条件反应是如何运作的。

另外，我们也习惯性地以一定的方式对环境做出反应。一个经典的例子就是巴普洛夫的狗。伊万·巴普洛夫是一位俄国心理学家，他演示了一条狗的行为可以受制于条件。每次铃声一响，狗就会以一定的方式来回应。

以下是该实验揭示的两个关键反应类型：

*   **非条件反应**：狗在接近食物时会流口水，这是很自然的、本能的反应。公式可表示为：`食物刺激 -> 本能流口水反应`。
*   **条件反应**：巴普洛夫发现他可以把铃声和喂食联系起来。狗甚至可以在没有食物、只听到铃声的情况下流口水。公式可表示为：`铃声（中性刺激）+ 食物 -> 流口水`，经过训练后变为 `铃声 -> 流口水`。

## 条件作用如何影响人类生活 👨‍👦

实验揭示了动物学习的机制，那么这种机制在人类生活中是如何体现的呢？

条件反应的过程影响了人类生活的许多方面。比如，假设你父亲是个很爱生气的人，并且当你还是个小孩时，他会经常把火发到你身上。对你父亲的怒气，你的非条件性反应就是感觉害怕和退缩，这是自然和本能的反应。

但是，如果你父亲每次向你发火时总是挑起眉毛，那么挑眉毛这个动作和发泄怒火也许就会让你形成联想。而这种联想就可能会发展成为条件反应。

## 从童年到成年的条件反射 🔄

我们看到童年经历如何建立联想，现在来看看这种模式如何跨越时间影响成年后的我们。

许多年后，你可能不明白为什么你会害怕你的老板。因为老板有一个爱挑眉毛的习惯，尽管他没有生气，作为条件反应，你仍然会感到害怕。这种条件反应通常是发生在无觉知层面的。

## 头脑状态与条件反应 🤯

了解了具体例子，我们上升到更普遍的层面：我们的思维状态本身是否就是一种条件反应？

在某种程度上，我们可以说，当我们在头脑中时，我们所经历的一切都是一种细微的制约性回应，或者更准确地说，是条件反应。因为当我们在头脑里时，我们所经历的一切都是过去的经验、信念、见解和意识的联想所染色、影响或者扭曲。我们并没有真实地如实体验当下时刻。我们所体验的当下时刻被我们的投射扭曲了。

## 总结与出路 ✨

本节课中我们一起学习了“条件作用”的机制。我们了解到，许多限制性信念和自动化反应模式源于童年形成的条件反射。这些模式在无觉知中运行，扭曲我们对当下的真实体验。

我们越临在，那么我们受这些条件反射、反应和行为的限制就会越少。觉察是打破自动化链条、重获内在自由的第一步。
