# 课程 P1：为何计划总失败？揭秘“自我提升”的常见陷阱 🕳️

在本节课中，我们将探讨一个普遍现象：为何精心制定的自我提升计划常常难以坚持。我们将深入分析其背后的心理动机，并指出基于“评判”和“比较”的自我提升模式中存在的两个主要陷阱。最后，我们将学习一种更健康、更持久的自我提升新视角。

我是诗诗。今天，我想与你聊聊自我提升这个话题。

说到自我提升，我们可能都有类似的体验。例如，想去健身、学英语、画画，或者学习其他各种技能。你可能已经学习了很多东西，成为了一个所谓的“斜杠青年”。

例如，你会弹琴、作诗、写文章、摄影、运动、做饭或调酒。就像我之前也学过调酒、做滴胶和画画等我喜欢的事情。你可能也从这些活动中获得了一些成果，成为了一个颇有才华的人。

那么，这种自我提升到底存在什么陷阱呢？

## 陷阱一：基于评判与功利的动机

对我自己而言，在自我提升的过程中，最初的动机往往是：我觉得自己好像比别人差了一些，在某个方面有所欠缺。

当我和同学、同事或一些朋友站在一起比较时，我会觉得自己好像挺差劲的，和他们一起玩时有点抬不起头。在你的成长和交友过程中，你可能也会下意识地与别人比较。

甚至有些人不与你比，他们也会主动来和你比较。这是因为当他们在某些方面有长处或优点时，例如家境更好、学习成绩更优、身材更好或外语更流利，他们需要通过与他人比较来获得存在感和价值感。

因此，他们会过来与你比较。类似于：“我英语还是挺不错的”、“你看我画的这幅画怎么样？其实我也没认真学，就稍微看了点教程”，或者“这次考试我也没怎么复习，但没想到考得还不错”。这种不经意的“秀”可能会让你感到不舒服。

你会觉得对方好像在炫耀，好像在拿自己的优点和你的缺点作比较。你当然会感觉输了，感觉自己被贬低，好像一文不值。你的觉察是准确的，对方正是在通过这种方式从你身上获得存在感和价值感。

这可能会激发你后来想要进行所谓的“自我提升”。我们通常会先通过一个评判系统来评估自己当下的状态。

例如，我可能给自己打三分、二分、一分，或者认为自己是个“垃圾”，什么都不会，根本没有价值。大众的评判标准往往是：会这个、会那个、取得了某项成就、拥有某些东西。如果我没有这些，我就是个“垃圾”，我就必须学点东西。

要么自暴自弃，要么积极上进。我们把这种自我提升称作“积极上进”，认为要做一个积极进取、自强不息的人。我不会这个，就要去学；身体不好，就要去健身；英语不好，就要去学。我们认为这是正确且理所应当的。

但是很多人会发现根本坚持不下去。为什么？可能学英语两天后，就觉得太难了，不想坚持。会想：我怎么还没达到四级、六级或同声传译的水平？

这就是基于评判和功利主义去学习和自我提升会遇到的卡关。你会卡在结果上，过分关注：我学了三天画画，到底能画出什么？我健身一星期了，怎么还没有腹肌？

你会开始过多关注结果，发现自己离目标依然很远。你可能才走了三步，却还有一万步要走。于是，你不想继续了，内心产生了抗拒。你会想：要不我再等等，反正努力也不是一天两天的事，先休息两天。

然后你就把这件事搁置一边，回到了之前的生活：刷抖音、看剧、玩游戏、和朋友闲聊。没过两天，当出现一个很优秀的朋友，或者一个你喜欢的、觉得太优秀以至于自己配不上的异性时，你又会想：我要是当初坚持下来就好了。

然后你又想回到那个项目中，比如要去健身、减肥、开始跑步。但跑了不到五天，你会发现今天不想起床，心想：休息一天吧，反正不差这两天。

因此，当我们在自我提升时，这是第一个陷阱。如果你是出于“我一定要得到某种结果，要通过自我提升获得满足感和价值感”这种功利目的，你就会卡在“我现在到底有没有达到目标”这个问题上。如果没有达到，你就会产生挫败感，失去行动力。

## 陷阱二：陷入无休止的比较

第二个陷阱是，在进行自我提升时，我们会禁不住与他人比较。

例如，我健身三个月，肌肉练得差不多了。这时我会产生一种评判，开始想：你看那个人好像没健身，他的身材、体型、姿态好像不太对劲。和他一比，我好有存在感。然后我去和健身两年的人比，他的肌肉太完美了，我有点不敢比，我必须更加努力。

这就是自我提升的第二个陷阱：比较。你会和比你差的人比较，以获得存在感和价值感，急于展示成果，渴望获得他人的认同和肯定。同时，你也会和那些看似更优秀的人比较，即向上比和向下比。

当你向上比时，又会觉得自己怎么还是这么差，怎么还不够努力，于是又想逼自己继续努力。我知道很多人会在自律或自我提升中获得一些甜头，但它似乎无法带来持久的状态。你总会有种隐隐的焦虑感。

当你学了这个学了那个，你会发现自己还需要学很多东西。当你遇到一个你觉得很优秀、自己配不上的人时，你仍会觉得自己没有存在感和价值感，必须学更多东西，必须用更多“武装”加在自己身上，让自己有更多闪光点，成为一个全才。

然后，你又开始了新一轮的自我提升，就像一个永无止境的循环。你会在这个循环中把自己搞得很累。

## 全新的自我提升视角：从接纳与热爱出发

说了这么多，你可能会问：难道我们要停止自我提升吗？难道自我提升不好吗？我并不是说它不好。你依然可以做那些事情，但我希望你用一种全新的出发点来做。

这个出发点不是通过否定自己，不是出于内在的匮乏感——那种“我觉得我缺什么，必须通过努力、每天坚持和煎熬来逼自己去做，以获得一点点价值感”的心态。

我希望你通过一个全新的出发点：**完全接纳自己**。我很接纳自己当下的状态。虽然用那套评判、比较、分离的系统去比，我可能是个“垃圾”，可能什么都没有，什么都不会。为什么不去接纳这样的自己呢？你不需要那套评判系统，只需要完全接纳自己当下的状态。

完全接纳我是谁，完全地爱自己、欣赏自己、赞美自己、支持自己、鼓励自己。用这种状态去做你想做的事情。

你喜欢画画，那就去做。通过画画这个过程，你已经享受到了快乐，感受到了那种愉悦，你很喜欢这种感觉。你会跟随自己的内心和感觉一直做下去。你的技巧和技能会自然得到提升，这种提升是**副产品**。而你做这件事所获得的快乐，才是最大的回报。

当你这样做时，无论进步了多少，哪怕今天只进步了一点点，你也会觉得“哇，我好棒”。或者你根本不关注自己的进步，只是在享受这个过程，享受做喜欢事情的每一天。那种感觉很爽、很放松、很投入。你全身心沉浸其中，很专注，处于一种类似“流动感”或称之为“**心流**”的状态。

在这种状态下，你会很轻松地获得高效率的提升。而且你不会用成果来判断自己的自我提升到底做得怎么样、够不够好。你不会给自己新的评判，不会感到约束或逼迫。你甚至不需要“坚持”和“努力”，你只是在享受。

你可以在取得一点成绩后展示出来，但你根本不需要别人的认可。例如，你画了一幅学画三天的作品，有人可能会说：“你画的是什么东西？好垃圾。”你根本不需要别人的认可，你会觉得：他有他的评价，但我很享受、很喜欢，没关系，我继续画，继续做自己喜欢的事情。

当别人这样评价你时，你也会明白他为什么评价你：是因为他想通过评判你来获得存在感和价值感。你看穿了他的模式，就不会受他的影响。这就是我所提倡的自我提升方式。

## 总结

本节课中，我们一起学习了自我提升计划常失败的深层原因。我们剖析了基于“功利结果导向”和“无休止比较”的两个主要陷阱。更重要的是，我们探讨了一种更健康、更可持续的自我提升路径：**从完全接纳和热爱自己出发，享受过程本身，让技能的提升成为自然发生的副产品**。记住，真正的提升源于内心的愉悦与专注，而非外界的评判与比较。
