在本节课中，我们将要学习一系列基础概念，这些概念是后续所有学习的基石。我们将逐一解析每个核心定义，并通过公式或代码进行清晰描述，确保初学者能够轻松理解。

---

## 什么是变量？📦

上一节我们介绍了课程的整体目标，本节中我们来看看第一个核心概念——变量。变量是存储信息的容器，你可以把它想象成一个贴有标签的盒子，标签是变量名，盒子里装的东西就是变量的值。

在编程中，变量用于保存数据。例如，在Python中，你可以这样定义一个变量：
```python
my_name = "梵公子"
```
这里，`my_name` 是变量名，`"梵公子"` 是存储在该变量中的值。

---

## 理解数据类型 🧱

理解了变量作为容器后，我们来看看容器里可以存放什么类型的东西，这就是数据类型。数据类型定义了变量所代表的数据的种类。

以下是几种基本的数据类型：

*   **整数**：表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数**：表示带有小数点的数字，如 `3.14`, `-0.001`。
*   **字符串**：表示文本信息，需要用引号包围，如 `"你好，世界！"`。
*   **布尔值**：表示逻辑判断的真或假，只有两个值：`True` 和 `False`。

---

## 掌握基本运算 ➕➖✖️➗

知道了数据的类型，我们就可以对它们进行各种操作，最基本的就是运算。运算不仅限于数学计算，还包括对文本的处理。

以下是常见的基本运算类型：

*   **算术运算**：包括加(`+`)、减(`-`)、乘(`*`)、除(`/`)等。例如：`result = 5 + 3 * 2`。
*   **比较运算**：用于比较两个值，返回布尔值。例如：`is_equal = (10 == 10)` 结果为 `True`。
*   **逻辑运算**：用于组合布尔值，如与(`and`)、或(`or`)、非(`not`)。例如：`flag = True and False` 结果为 `False`。

---

## 认识控制流 🔀

当我们能够进行运算和比较后，就可以让程序根据不同的条件做出决策，或者重复执行某些任务，这就是控制流。控制流决定了代码的执行顺序。

以下是两种基本的控制流结构：

*   **条件判断**：使用 `if`、`elif`、`else` 关键字，让程序根据条件选择执行不同的代码块。
    ```python
    if score >= 60:
        print("及格")
    else:
        print("不及格")
    ```
*   **循环**：使用 `for` 或 `while` 循环，重复执行一段代码，直到满足特定条件。
    ```python
    for i in range(5):
        print(i) # 这会打印数字0到4
    ```

---

## 总结 📝

本节课中我们一起学习了编程的四个基础概念：**变量**是存储数据的容器；**数据类型**定义了数据的种类；**基本运算**允许我们对数据进行操作和比较；**控制流**则赋予了程序决策和重复执行的能力。掌握这些概念是编写任何程序的第一步。
