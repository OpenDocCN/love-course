# 课程05：曲线思维发散训练 🌀

在本节课中，我们将要学习一种被称为“幽默式推拉”的核心聊天技巧。我们将深入探讨其背后的原理，并通过具体例子和训练方法，帮助你掌握如何在对话中制造情绪波动，让交流变得生动有趣。

## 推拉的核心作用

上一节我们介绍了课程主题，本节中我们来看看“推拉”的基本定义和作用。

推拉，通俗解释，就是把女生推开再拉回来，以制造情绪波动。在聊天过程中，推拉的核心作用只有一个：**制造幽默、风趣、好玩、有趣的感觉**，让女生觉得你搞笑。

这种技巧在几年前非常流行，因为它能让聊天全程充满笑声。虽然随着个人风格成熟，可能减少使用，但对于吸引年轻女生或快速建立兴趣，推拉依然具有显著作用。

## 幽默背后的原理：反差感

理解了推拉的作用后，我们不禁要问：让人发笑的原理是什么？本节将揭示幽默背后的核心机制。

能够让女生开心快乐的原理，关键在于一个概念：**反差感**。所有搞笑内容的底层逻辑都是制造意想不到的反差。

例如，在周星驰的电影《功夫》中，主角叫阵“瘦子”，结果出来的是一身肌肉的壮汉；叫“小矮子”，站起来的却是两米高的大汉。这种预期与现实的巨大落差，就是反差感，它能制造出幽默效果。

## 聊天中的推拉实例

理论需要结合实际。以下是几个在聊天中运用“反差感”进行推拉的具体话术实例：

*   **实例一：关于爱好**
    *   女生问：“你平时爱干嘛？”
    *   你回答：“我平时爱看书游泳打飞机。”
    *   女生表示惊讶后，你再说：“哎，其实跟你开玩笑的，我这样的人怎么可能看书呢。”
    *   **核心**：先建立一种形象（推），再用反转否定它（拉），制造反差。

*   **实例二：关于关系**
    *   你说：“我们不要再联系了。”（推）
    *   女生问：“为什么？”
    *   你回答：“因为我喜欢上别人了。”
    *   女生追问后，你说：“我喜欢那个人将来会叫你妈妈，叫我爸爸。”（拉）
    *   **核心**：用看似严重的话推开，再用浪漫或搞笑的话拉回。

*   **实例三：关于环游世界**
    *   你说：“不和你聊天了，我要去环游世界了。”（推）
    *   女生问：“你要怎么环游世界？”
    *   你回答：“等我见到你之后，围着你绕一圈，就环游世界了，因为你就是我的全世界。”（拉）
    *   **核心**：将宏大的概念最终落到对方身上，形成甜蜜的反差。

*   **实例四：关于聊天流量**
    *   你说：“我不想和你聊天了，我再和你聊天，我手机就快……”（推，留下悬念）
    *   女生好奇追问：“快要什么？”
    *   你回答：“我觉得我要再和你聊天，我可能手机流量就不够用了。”（拉，给出一个朴实无华的理由）
    *   **核心**：引导对方产生浪漫预期，然后用一个日常理由打破它。

总之，推拉的基本模式是：**一句话把女生推开，紧接着另一句话把她拉回来**。关键在于通过制造预期与结果的反差来产生幽默效果。

## 如何培养幽默感与推拉能力

掌握了原理和例子，你可能想知道如何系统性地培养这种能力。以下是几种有效的训练途径：

*   **途径一：浏览幽默社区**
    *   建议下载“皮皮虾”这类软件，经常翻阅里面的有趣视频和，特别是“神评论”和“神回复”，学习网友如何机智、幽默地接话和制造反差。

*   **途径二：观看语言类节目**
    *   推荐每周观看《天天向上》这类节目。观察主持人之间如何对话、抛梗、接梗，以及如何自然地在互动中制造幽默。这是学习现场语言节奏的很好方式。

*   **途径三：聆听传统相声**
    *   多听郭德纲等相声演员的作品。相声是语言艺术的精华，能帮助你深入感受幽默的节奏、铺垫和包袱（笑点）的设计。

当然，作为课程学员，你也将获得大量经典的推拉话术库，可以直接参考和使用。

## 总结

本节课中我们一起学习了“幽默式推拉”这一核心聊天技巧。

我们首先明确了推拉在对话中的根本作用是**制造幽默感**。然后，我们揭示了所有幽默背后的核心原理是**制造反差感**，即打破对方的预期。接着，我们通过多个具体的话术实例，展示了如何在实际聊天中运用“一推一拉”来制造这种反差。最后，我们提供了几种实用的训练方法，帮助你从日常生活中培养这种幽默思维。

记住，推拉的基本思维就是**反差萌**，核心操作是**一推一拉**。只要你能在对话中巧妙地制造反差，就能产生幽默效果，从而解决聊天枯燥平淡的问题。希望你能按照提供的方法进行训练和尝试，让你的交流变得更加生动有趣。
