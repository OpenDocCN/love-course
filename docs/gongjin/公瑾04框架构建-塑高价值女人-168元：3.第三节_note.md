## 核心概念解析：PU与MV

上一节我们介绍了框架的基本认知，本节中我们来看看构建框架前必须理解的两个关键指标。

**PU** 是“亲子不确定性”的缩写。它指男性对于后代的不确定感。女性可以百分百确定孩子是自己的，但男性无法百分百确定。这种不确定感是男性心理的基石。

**公式**：`高PU = 给男性带来高不确定感和低安全感`

PU值高的女性会给男性留下“随便”、“易出轨”的印象。例如：外形妩媚艳丽、性格外向、异性交往复杂、夜生活丰富、感情经历多，或在特定场所（如酒吧、KTV）工作的女性。

PU值低的女性则普遍被认为“靠谱”，没有复杂的异性关系。男性对这类女性抱有好感，因为结合意味着更高的后代确定性和安全感。

男性对高PU女性的想法通常是发展短期关系；对低PU女性则倾向于考虑长期婚姻。

---

**MV** 是“婚姻市场价值”或“伴侣价值”的简称。它指个人在婚恋市场中的综合价值。

**公式**：`MV = 身高 + 体重 + 颜值 + 身材 + 智商 + 情商 + 教育程度 + 家庭背景 + ...`

当你的MV足够高时，你的择偶标准可以更高，框架可以更强。如果MV不够高，则需要尽量降低PU来综合属性，提高竞争力。

## 区分正确框架与错误框架

理解了PU和MV后，我们进入正题：如何区分框架的对与错。每个女性都应该有自己的标准和框架，但很多人因坚守错误的“小框架”而错失良缘。

### 什么是应该坚持的“大框架”

在恋爱关系中，有些原则是必须坚持的大框架。

例如：**男人必须主动追求女人**。但这并非指事事都要他主动。我们倡导的是“被动型主动”，即在关键节点上引导对方主动。

以下是必须坚持的大框架领域：
*   关系推进的主动权（如确定关系、上床、见父母）
*   婚姻重大事项（如结婚、房子加名）
*   家庭权力分配（如财产大权）
*   核心家庭关系（如婆媳关系）
*   男方在大事上是否尊重女方意见

后续课程会详细讲解这些部分。

### 什么是需要警惕的“错误小框架”

许多女性在大事上不计较，却过分纠结于细节，这是本末倒置。

以下是常见的错误小框架清单：
*   **认为女生绝不能主动联系男人**：联不联系不重要，重要的是能否把握关系节奏。内心强大的人可以通过小主动引发对方更大的主动。

*   **过分纠结于表面形式**：如计较对方是否每天说早安晚安、是否称呼宝贝等。这些并非关系的核心。

坚守这些小框架，往往源于内心脆弱、恐惧不被重视。真正的高价值女性，心态是松弛的，因为她们有选择。

## 如何正确建立框架

明确了框架的区分，我们来看看建立框架的正确方法。这里存在一个常见误区：很多人一上来就急于建立框架。

### 建立框架的基础：吸引力

无论与新认识的男生接触，还是处于挽回阶段，首先要建立的都不是框架，而是**吸引力**。

吸引力是决定一切框架建立的基础，是所有技巧能够生效的前提。没有吸引力，任何高超的技巧都无效。

吸引男人靠的是自身价值和表现方式，而非苦口婆心的教育或处心积虑的付出。永远不要把“为男人付出”放在第一位。

### 框架的建立与破除

框架不仅包括建立，也包括**破除**。在有能力破除他人框架前，你需要先破除束缚自己的框架。

**需要破除的“大框架”举例**：
*   女人就必须相夫教子、做贤妻良母（这应是选择，而非必然目标）
*   女人必须为孩子和家庭牺牲事业
*   女人不能同时认识多个男性，有备胎就不是好女人

**需要破除的“小框架”举例**：
*   男人必须主动发消息、说早晚安
*   约会必须由男人提出并买单（在所有细节上）

破除这些束缚，建立以自我感受为中心、尊重自我的新框架，才能在亲密关系中建立真正有价值的框架。
