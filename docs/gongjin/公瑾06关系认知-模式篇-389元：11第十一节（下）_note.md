# 公瑾06关系认知-模式篇-389元：11第十一节（下）💡

在本节课中，我们将要学习亲密关系中三种常见的负面互动模式：**利益争夺战**、**自以为是的公平**和**复合博弈**。理解这些模式，能帮助我们识别关系中的陷阱，并学会如何转向更健康、更合作的相处方式。

## 一、利益争夺战：从小事到敌对战场 ⚔️

上一节我们介绍了关系中的一些基础模式，本节中我们来看看第一种具体表现：利益争夺战。

当伴侣双方因小事争执，并逐渐将彼此置于对立面时，就进入了“利益争夺战”。此时双方不再关注问题本身，而是陷入“谁对谁错”、“谁赢谁输”的博弈中。这种模式下没有赢家。

相比较来说，我们大家都想要第一种相处模式。所以这个时候呢我们要从这种平等轻松的姿态走过来，不要一开口呢，就想着指责和控诉。一旦你们搞成这个样子呢，最后两个人的情绪一定会逐步的发酵到失控。

我们大家真的大可不必这样，大多数的事情呢，你越平静的去解决他，就越能轻松的被解决掉。所以一旦最近和男朋友之间呢，有一个事情死死的卡在一个地方的时候，你不妨的停下来去观察一下，是不是你们两个人已经走上了这个敌对的战场。

好的，以上呢就是第一种利益争夺战，有小的事情引发两个人进入到敌对的战场，开始了相互的博弈。

## 二、自以为是的公平：绑架式的付出 ⚖️

了解了敌对博弈的危害后，我们来看第二个常见问题：自以为是的公平。

这句话翻译一下，就是“我都做到1了，那你就应该做2”。这句话在感情里面是非常让人讨厌的。就是当一个人面对规则的时候，他不仅仅强迫自己服从，还要求别人也要来服从，尤其是当我因为一个规则不开心的时候，那我就把别人也拉下水来和我一起服从这个规则。就是有一点“我不好过，你也别想好过”的感觉。

那我们还是通过这个案例来看一下，具体是怎么回事呢。

就比如说有个姑娘，她叫小芳。她有一个男朋友还没有结婚。然后呢，有一次她在洗衣服的时候呢，不小心把男朋友的白衬衫染上色了。这个衬衫呢还是新买的，刚穿了一两次的样子。这个小芳呢她真的有点不小心，因为她也没有发现这个衣服里面呢有一件衣服居然会掉色的。

虽然染的不严重，但是小芳呢还是觉得要赔男朋友一件，因为小芳呢这个人她有一个特点，她不喜欢欠人情。然后她也希望自己也是一个好人的形象，好女朋友的人设。

虽然男朋友知道这件事情以后呢，她也没有生气或者责怪小芳什么的，但是小芳还是主动的掏了钱给男朋友又买了一件同款的衬衫，这件衬衫吧还挺贵的，但是小芳还是买了，到货了之后呢，男朋友也挺高兴的，就欣然接受了。

然后呢，在男朋友的心里边，这件事情就过去了，本来就算小芳没有重新买一件，男朋友也不会觉得有什么，但是既然小芳已经买了男朋友当然也很高兴啊，她很自然的就接受了。

但是呢在这个时候，小芳其实是有些不高兴的。因为她觉得男朋友至少应该客气一下，或者给自己发一个红包，或者买点什么其他的礼物来给自己啊。她心里会有这个期待，毕竟自己不是故意的，而且还花了那么多的钱，结果男朋友也没有提出回报的意思。所以她心里边不太舒服，但是她也没有说出来，只是在心里面默默的惦记着这个事情，总感觉好像亏了点什么。

结果呢有一天这个男朋友来到小芳家里的时候呢，她做了一个什么事情呢？就比如说她用小芳的洗面奶洗了一把脸。然后这个时候呢，小芳也在卫生间里，她在敷面膜，然后她看到了男朋友在挤这个洗面奶的时候呢，一不小心挤出来很多很多。然后这个小芳也就看到了，他有脸开玩笑的说，哎呀，你看你挤了这么多，这瓶本来就快用完了。

可能小芳在说的是他是开玩笑的。但是呢当然由于之前的一件衬衫的事情，他还一直记在心里，她这次开玩笑的时候，也会有一些期待在里边，她会期待着说男朋友能够说一句，哎呀，没关系啊，那你用完了，我就再给你买一瓶。那不管这个男朋友有没有买，他都会觉得好受一点，他觉得你看吧，上次是我给你买的东西，这次就应该你给我买了。

但是呢这个事情要是真的就这么简单就好了。现实是什么呢？现实是这个男朋友，他听到这句话之后感觉很不舒服，你看他就一瓶洗面奶吗？你还这么斤斤计较的，大不了我不用了呗。然后他随手就洗了把脸还有点生气的就回到客厅去了。

这个时候小芳呢肯定是脸受伤的，他相当于是受到了双重的打击。第一，他没有得到期待中呢，男朋友要给他买这个东西的信息，第二，他开玩笑的一句话，反而让男朋友还生气了。

从这个案例当中呢，我们可以发现他们俩在交往的过程当中呢，这个小芳啊一直在追求虚无的公平。简单来说呢，就是如果我因为一个规则不开心，那我就把所有人拉下水来和我一起服从这个规则。

小芳的心态呢也是：上次我付出了，那么这次就轮到你了。如果你不这么做，那么我也不这么做了。如果我这么做了，你就必须要付出同等的代价，否则我就不爽，我就要告诉你。大家来看一下绑架的味道，这不就来了吗？这个小芳是你们无数人的影子啊。

大家回想一下，在亲密关系里边是不是也会有这样的通病，就是你呢一直在屈从规则，无法自我负责啊。

就在这样的案例当中呢，小芳的付出并不是真心的。她是出于某种规则，也就是小芳呢也是这种规则的受害者。他被迫在这个关系里面要成为一个好人，他不敢反抗这个规则，为了让自己心里边平衡，她也期望别人也都服从这个规则，否则她也就不舒服。所以呢她在给男朋友买了衬衫之后呢，他就非常期待着男朋友也应该为我做些什么。他她紧盯着这些不放的时候，他就会非常累，非常的紧张。那么可想而知啊，他们的关系也不会好。

我们可以再举个例子啊，就相当于现在我们刷抖音的时候呢，不是经常有些视频为了带货也好，跟风也好，就是有很多的视频会说啊，给男朋友织个围巾什么的，有些女人呢可能就会上头，下单买了这种手织的围巾，然后呢就高高兴兴的给男朋友去织围巾，然后等把围巾送给男朋友的时候，他心里面是很高兴的，很期待的啊。

那比如说呢有些视频又在宣传什么秋天的第一杯奶茶，什么十根烤肠之类的。他看到这些东西的时候呢，会非常期待男朋友也能给自己买这些，哪怕就十几块钱的东西，如果男朋友送了，她也会非常的高兴的。那如果没有送呢，她就会觉得有点不爽，觉得之前的围巾白织了。

你看啊这类人最大的特点就是因为我已经做过了，所以你也要做，否则呢我心里面就会不平衡。

那在这个时候呢，不管是这个小芳呢还是上边这个织围巾的姑娘，他们最大的问题就是她屈从规则，没有办法去自我负责。

### 如何做到清晰边界，求同存异？

所以对他们来说呢，他们真正要做的是什么呢？就是清晰边界，求同存异。

那我们展开来说呢，关于如何求同存异在这里，一共可以分为四个关键点。

以下是四个关键步骤：

1.  **心口合一，遵从本心**
    你从一开始就要心口合一，就是说你怎么想的，你就怎么去做，不想买这件衣服，你就不买，不想织这个围巾就别织。你买衣服和织围巾这件事情只能是因为你想去做，他才有意义。如果你就是因为想要回报，为什么不利用这些时间和金钱，直接去买想要的东西呢？去做真正能够让你高兴的事情呢？我们做事情最主要的是要遵守自己的本心啊。

    > 核心公式：**我的行动 = 我的真实意愿**，而非 **我的行动 = 对规则的服从 + 对回报的期待**

2.  **拒绝外部规则绑架**
    不要逼迫着自己去向规则看齐，像什么“女人就应该贤惠的过日子”、“就应该去做贤妻良母”。这些外部的对你的期待或者是外部的规则，你不要去理会，不要逼着自己去服从，否则呢就又回到我们前几节课讲到的无效付出了。这些逻辑其实是一样的。听过我认知课的同学都知道，我们整体的认知体系的核心宗旨是我们的关系是要讲究平衡和平等的。

    > 关键认知：**关系健康 = 平等 + 平衡**。我想做就做，不想做就不做，不为人设所累。

3.  **直接表达感受与期待**
    你可以直接去表达自己的感受。当男朋友直接接受了你买的衣服，你心里面不舒服，你可以直接讲的：“哎，你这个人哈你就这么收下了，你也不说句谢谢啊，你不请我吃个饭什么呢？你好歹跟我客气一下嘛，请我吃点好吃的，表示表示啊”。不用在心里面自己去暗暗的生气，然后去找机会搞他一下，你不如直接去表达。

4.  **允许对方与“我”不同**
    最后一点呢也是比较重要的一点，那就是允许对方和我不同。这里是很多人在亲密关系里面一个非常常见的雷区呀。很多人他都没有办法理解。为什么我的伴侣和我不一样呢？在处理一件事情的时候，为什么我想这样他想那样，那不行，我必须要求他和我一样，我才是对的。

    就像我们前面几节课还举过一个例子，就是老婆觉得必须要每天洗澡，老公觉得那没那个必要。二三十年的习惯啊，就相当于有些已经刻在了骨子里边，怎么可能说改就改。对吧？如果你一直强调一直盯着去叫他改变，那你不仅仅是在为难他，更是在为难你自己。我们说在夫妻当中呢，最重要的就是平等和尊重。

    > 核心原则：**他允许我成为我，我也允许他成为他，我们才能变成我们。**

当你真正了解并允许对方的不同后，你就能调整自己的心理预期，减少落差和不满。你的付出将更纯粹，关系的流动也会更健康。

这里边就是我们的第二个部分，叫做自以为是的公平。

## 三、复合博弈：通往两败俱伤之路 🥊

我们接着来看本节课最符合主题的一个部分：当我们处于天平的两端，就会陷入的境地——复合博弈。

第三部分呢，其实就是本节课最符合我们主题的一个部分，就是当我们处于天平的两端，我们就会陷入的境地。就是我们的第三个部分叫做复合博弈。什么意思呢？就是大家都在求赢，拼命的想要占据高位。在这个过程当中不断的去搞事情，使手段。但是结果呢两败俱伤。复合博弈的最简单的解释就是两败俱伤啊。

给大家举个例子来说，比如说你和老公今天下班一起回家，你想吃火锅，老公想吃烧烤，谁都不让着谁，你就是想吃老公呢也不想退步。你们两个呢都觉得谁先低头，谁就输了。其实我们外人来看这有什么事吧，今天吃火锅，明天吃烧烤呗，就是很简单的一件小事，根本就不至于吵架，对吧？

但是呢这两个人就上头了，他们相当于是带着这种对抗的思维再相处。就是我不愿意放过任何小事。我偏要在这种细枝末节上来去扣这个细节。于是呢你们在大街上就开始赌气，谁都不想低头。最后的结果是什么呢？就是你们俩什么都没吃上，直接呢两个人都很生气的回家了。哎，两个人还饿着肚子，这个结局就是复合博弈，谁没赢，大家都是输家，就是两败俱伤的局面。

如果你们夫妻两个人呢在相处当中呢，长期的陷入到这种竞争和对抗，你们俩的心理地位就一定是忽高忽低，特别没有安全感。

### 如何打破复合博弈？

好，我们具体来看一下这个东西应该怎么解决。

首先呢你们两个人当下的状态一定是防御的。两个人都非常畏惧服软这件事情。所以呢两个人心里面的想法就是我不想主动低头。但是呢我害怕失去。这种状态下，所谓的“主动”常常是带着条件的、防御性的，反而会加剧对方的逃离感，形成恶性循环。

那么我们应该怎么办呢？我们要用柔软的姿态，大家都知道这句话啊，叫做“百炼钢化成绕指柔”，这就是我要你们修炼的。以暴制暴，绝对不是好的方式。

我们在亲密关系当中，最需要的两个字就是“看见”。

以下是打破博弈的关键：

1.  **看见自己的需求**
    和老公吵架之后，你一定会有想要和好的这个期待，这就是你当下最大的情感的需求，这是我们大部分人都会有的。但是很多人不敢直面这个需求。他会觉得如果我主动了，可能会被拒绝。于是他整个人都是纠结的，一边想主动，一边又不敢主动。

2.  **看见对方的需求**
    你是这样，你老公也是这样。所以呢你要明白一点，很多人说啊老师啊，我老公就是喜欢冷战，不是的，没有人喜欢冷战，没有任何人喜欢冷战的，他是在逃避问题，他不会解决矛盾，所以她用冷战的方式来逃避来保护自己，而且她也会害怕被拒绝啊。所以大家要记得要看见老公的这个需求。

**核心解决之道：对自我需求高度支持和负责**

这里面的重点就是我们每个人都要做到对自我的需求高度的支持和负责。就是当我觉察到自己的需求之后，我会付出行动来满足我的需求，并且我愿意承担后果，不管结果是好还是坏，我都愿意自己承担。我不会强求让别人来为我的行为负责。

> **关键认知：这是我的选择，他有拒绝的自由，不需要背负着我的期待来满足我的需求。**

这个认知是能够改变你们之间的复合博弈的这个局面的。当你整个人放松下来，不强迫对方一定要给你好的回应，不要求他为你的情绪负责，对方也会开始放松并回应。这时，接受他的回应，而不是赌气端着自己，你们才能渐渐卸下防御，走向良性循环。

## 总结 📝

本节课中我们一起学习了亲密关系中三种破坏性的模式：

1.  **利益争夺战**：因小事陷入敌对博弈，忘记合作初衷。
2.  **自以为是的公平**：用“付出”绑架对方，追求虚无的公平感，实则无法自我负责。
3.  **复合博弈**：在对抗中争输赢，最终导致两败俱伤。

破解这些模式的核心在于：**回归本心、清晰边界、直接表达、允许不同**，并最终做到**对自己的需求高度支持和负责**。记住，健康的亲密关系是合作，而非博弈；是“我们”一起面对问题，而非“你”和“我”的对抗。

希望大家认真对照课程中的案例，观察自己生活中是否有类似的影子，并按照所学知识去实践和改变。
