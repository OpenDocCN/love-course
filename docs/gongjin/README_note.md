# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存放数据。每个变量都有一个名字（即标签），我们可以通过这个名字来使用或修改盒子里的数据。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如，`name = "公瑾"` 这句代码创建了一个名为 `name` 的变量，并将文本“公瑾”存放进去。

## 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道“盒子”里装的是什么类型的东西。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型：

1.  **整数**
    整数是不带小数点的数字，例如 `-10`, `0`, `100`。
    ```python
    age = 30
    ```

2.  **浮点数**
    浮点数是带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
    ```python
    price = 9.99
    ```

3.  **字符串**
    字符串是由引号包围的文本，可以使用单引号 `' '` 或双引号 `" "`。
    ```python
    name = "公瑾"
    greeting = '你好'
    ```

4.  **布尔值**
    布尔值只有两个：`True`（真）和 `False`（假），常用于逻辑判断。
    ```python
    is_student = True
    ```

## 如何查看和使用变量？ 👀

知道了如何创建变量和数据类型后，我们来看看如何查看和使用它们。

以下是两个基本操作：

*   **打印变量**：使用 `print()` 函数可以将变量的内容显示在屏幕上。
    ```python
    name = "公瑾"
    print(name)  # 输出：公瑾
    ```

*   **修改变量**：通过赋予变量一个新的值来更新它。
    ```python
    score = 90
    score = 95  # 现在score的值变成了95
    print(score)  # 输出：95
    ```

## 一个简单的综合例子 💻

让我们把上面学到的知识组合起来，写一个简短的程序。

```python
# 定义几个变量
user_name = "公瑾"
user_age = 28
user_height = 1.75
is_developer = True

# 打印这些变量的信息
print("姓名：", user_name)
print("年龄：", user_age)
print("身高：", user_height)
print("是否是开发者：", is_developer)
```

运行这段代码，你将在屏幕上看到关于“公瑾”的一组信息。
