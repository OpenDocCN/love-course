# 公瑾01重建-危险关系重建-499元：03第三节_note

在本节课中，我们将要学习心态建设。心态是认知与情绪的组合，它决定了我们能否冷静地分析局面并做出最有利的选择。当心态失衡时，我们无法看清现实，容易做出混乱和失控的行为。因此，建立正确的心态是处理后续问题的基石。

## 拒绝讨好：停止自我攻击

上一节我们讨论了认知与情绪的关系，本节中我们来看看如何拒绝讨好。许多人在遭遇伴侣出轨后，会不自觉地攻击和鄙视自己。这种心态需要调整，因为出轨是对方的行为，你不应为此承担负面情绪和羞耻感。

以下是常见的错误认知及其调整方法：

*   **错误认知**：认为“老公出轨，是因为我不够好”。
*   **调整方法**：出轨是对方选择用最糟糕的方式处理婚姻问题，这并非你的错。你需要停止自我攻击，转而分析关系中的深层原因。

*   **错误认知**：担心“别人会在背后对我指指点点”。
*   **调整方法**：他人的看法无法定义你的人生。这种羞耻感往往源于你内心对自己的不接纳，是一种自我投射。

*   **错误认知**：认为“女人的人生价值由男人的专一与否决定”。
*   **调整方法**：你的人生成功与否，不应由他人定义。这就像你的饱腹感不应由旁人来决定一样荒谬。你需要建立独立的自我价值体系。

为了更清晰地理解，我们可以用以下公式来描述心态失衡的恶性循环：
**错误认知 → 负面情绪（羞耻、恐惧）→ 讨好行为 → 关系进一步恶化**

## 警惕“甜蜜陷阱”：摆脱托付心态

在拒绝了讨好心态后，我们需要警惕另一种常见陷阱。许多女性在成长过程中，被社会灌输了一种“托付心态”，认为找到一个好男人就能获得幸福，从而放弃了自我成长和独立。

以下是“甜蜜陷阱”的典型表现：

*   **表现一**：将个人快乐完全寄托于伴侣。认为“男朋友/老公就应该让我快乐”。
*   **本质**：这是一种情感绑架。爱是心甘情愿的付出，而不是“应该”的义务。你的快乐应由自己负责。

*   **表现二**：对婚姻抱有刻板幻想。认为“结婚=幸福”，而不思考婚姻中的责任、风险与个人成长。
*   **本质**：这是一种逃避现实的依赖心理。婚姻不是人生的保险箱。

*   **表现三**：寻找“潜力股”，指望对方带自己“飞黄腾达”。
*   **本质**：这依然是托付心态，将个人命运完全交予他人，极易在对方变化或失败时陷入被动。

陷入“甜蜜陷阱”的后果往往是**人财两空**。当你逐步放弃自己的生存能力、精神独立和价值提升，完全依附于对方时，你的价值就在无形中递减。一旦关系生变，你将毫无招架之力。

## 建立负责任的心态：你永远有选择

最后，也是最重要的一点，是建立为自己人生负责的心态。无论遭遇什么，只要你不放弃自己，就永远有重新开始的可能。

我们通过一个案例来理解。学员小A的老公在事业受挫后出轨，并声称小三能在事业上帮助他。小A最初感到无比被动和挫败。

然而，通过调整心态，小A将此事看作：两人在人生路上携手爬山，老公只是“崴了脚”，却因此像个巨婴般发脾气、逃避，并寄望于外人（小三）能背他走。

小A没有选择去“背”起老公，而是做出了以下负责任的行为：

1.  **专注自我**：她将精力投入到自己的工作、孩子和社交生活中，稳固自己的“山头”。
2.  **设定边界**：当老公情绪化地试探或指责时，她不予回应；只有当老公能好好沟通时，她才进行交流。
3.  **掌握主导权**：她明确，自己的世界是开放的，老公若想同行，必须符合她的要求，拿出合作的态度。

最终，老公在被小三欺骗后，逐渐认清现实，并开始以更尊重、更成熟的方式与小A相处。小A通过**负责任的心态**，不仅度过了危机，还重塑了关系的权力平衡，增强了对未来生活的信心。

这个案例告诉我们：**当下永远是起点**。出轨只是一个事件，并非人生的终结。只要你拥有为自己负责的心态，就能重新拿回人生的主导权。

## 课程总结

本节课我们一起学习了心态建设的三个核心部分：

1.  **拒绝讨好**：要彻底剔除“男人出轨是我的错”的错误认知，停止一切自我攻击和讨好行为，这只会让对方更轻视你。
2.  **警惕甜蜜陷阱**：要清醒认识到，将人生幸福完全托付给他人是危险的。你需要保持独立，持续强化自身价值。
3.  **建立负责任的心态**：这是最重要的心态转变。无论发生什么，你都要为自己的人生负责。记住，**当下永远是起点**，你永远有能力重新开始，做出对自己最有利的选择。

希望你能从这节课中汲取力量，尽快从负面情绪和错误心态中解脱出来，以更强大、更清醒的姿态面对未来的挑战。我们下节课再见。
