### 核心三原则：平衡、平等、一致性

我要强调一个非常重要的点，三个词：**平衡、平等、一致性**。这是我们所有内容的基础，后面很多方面都会用到。我的内容基本围绕这三个点展开。这里大家可以做个笔记。

以下是这三个核心原则的解释：

1.  **平衡**
    *   简单来说，就是我们投入和回报的大致平衡。我的付出和他的付出大致平衡。

2.  **平等**
    *   平等的意思是，我对自己是什么样的要求，我也用同样的标准去要求对方。或者说，如果我能包容自己的情绪，那么我也要同样去包容对方的情绪。双方的标准和被接纳的程度是平等的。
    *   不会出现“我要求自己怎么样，但他可以不做到”，或者“我要求他怎么样，我自己却可以不做”的情况。你们之间是相互平衡、平等的。简单点说，彼此都不可以双标。

3.  **一致性**
    *   一致性跟沟通相关，指的是在相处过程中，你们一直用一种更加坦诚、真诚的方式在表达。
    *   一致性可以简单理解为真诚，就是你想什么就说什么，用直观的方式表达出来。当然，直观不是有话直说，而是你可以把你的想法表达出来，不一定用语言，可以用各种方式，但它不包含攻击别人。
    *   简单点说，就是**你可以讲述你的愤怒，但不可以愤怒地讲述**。

所讲的这些目的很简单，就是我们要站在一种平等、自由的基础上来经营你们的关系。

## 经营关系的核心：认清人性与修炼内功

## 对感情的准确定位与过渡

下面我们再讲一个重点：**对感情的定位**。对感情的定位包括什么呢？就是我刚刚说的，感情它是一个正常过渡的。你们一定要记得，就是什么状态下做什么角色、做什么事情。如果你的角色乱套了，你会很难受。

比如，你们还没有确定关系，你就要求对方表现出有多爱你；比如你们刚刚认识一个月、在一起三个月，你就要求对方为你把更多的信心和决心拿出来，这肯定不现实。再比如，你们两个人现在正在闹离婚，或者对方跟你提了分手、离婚，你就要求对方对你有很高的情绪投资，这显然也是不现实的。

简单点来说，就是在**什么情景下，什么定位**，否则难受的是你自己，并且关系会越来越僵。这是很多姑娘都会踩的坑。

举个例子，比如你们现在正在吵架、冷战、尴尬等等。首先第一件事，允许你和他当下都是生气的，他躲着你是正常的，你要允许这件事情发生。有的姑娘是这样的：“你和我生气，你不跟我沟通，哼，我今天一定要跟你沟通。”因为在你的世界里，你觉得他不跟你沟通就是错误的。但实际上，当下你第一步应该是先明白你们是什么状态，什么状态干什么事，这个才是应该去接纳的，而不是“你做什么一定是正确的”。如果你们两个人之间完全以你的意志为主导，你这样去想、去做，对方是非常不舒服的。

### 感情应该如何过渡

我说一下感情应该怎么过渡。举个例子，还是举例子：如果对方要分手、要离婚，这个过渡应该是怎样的？这种情况下，你们两个人肯定是带着情绪的，要么你很难过、生气，要么他很失望、愤怒等等。有这样的负面情绪在，这个时候我们应该做的事情是让两个人先冷静下来。

不能马上说“你现在一定要像以前一样爱我，不能跟我提离婚”，这不现实。这中间我们的过程是：对方先冷静，你也冷静 → 然后你们两个能够平静下来 → 然后你们能够说上几句话 → 然后你们能够说上比较多的话 → 然后你们慢慢恢复到正常状态，两个人不会有那种很不舒服、很尴尬的感觉 → 然后慢慢地像朋友一样能够去交换更多信息 → 再慢慢地能够偶尔打打闹闹一下，开个玩笑 → 然后再慢慢过渡到两个人之间有点暧昧 → 然后两个人再慢慢靠近 → 然后就是慢慢达到有点恋爱的状态 → 然后我们再去回归以前那种比较好的状态 → 然后我们才可以去修复以前的心结。

咱们正常的情感过渡应该是这样的。你们可以把这个过渡想象成阶梯，你得一层一层地上。第一层你就没有踩稳，你就要爬到第四层，你一定会摔跤的。如果你爬到第三层退回来了，爬不上去了，那么你要想的是我赶紧回到第二层，第二层踩稳了之后，才能走第三层。

有很多姑娘会说，这是不是需要很多时间啊？这正是我要说的。这个过程可能你们听起来会觉得很可怕：“天啊，这么多环节，我们得走到何年何月啊？”不是的。所有的环节，有的环节可能一天就完成，有可能一天完成几个环节，而有的环节你可能需要很久才能过渡。但是如果你们着急，可能就会逼着他做一些不符合当下他情绪的事情，那么你刚刚上去的台阶可能就退回去了。

因为很多姑娘习惯性地觉得自己做些主动的行为是很委屈的，尤其是对方在没有给回应的时候。这个事情真的很重要啊，这是很多姑娘的误区：我要这个男人马上怎么样怎么样。这个事情如果你着急去追着要，要他一定马上给你什么东西的话，他也会很难受，心里不舒服。所以，认清当下的局面很重要。什么局面干什么事，你舒服，他也舒服。

还有，你不要觉得吵架一定是痛苦的。吵架也是可以舒服的。因为在吵架的时候，如果我能够把糟糕的情绪发泄出来，并且允许对方也把糟糕的情绪发泄出来，实际上这个吵架并不会成为一种很糟糕的心结和一道伤疤，它会变成一个很小的插曲就过去了。但是如果你们非得要求对方一定要怎样怎样，它就真的会变成一个非常糟糕的情况。

他主动与否，跟你的状态、你们的相处模式有很大的关系。其实不是对方不主动，而是你没有创造让对方主动的环境，并且你还不断地去要求对方如何如何，这样也是非常危险的。

### 情绪价值的本质

今天我们再谈一个**情绪价值**。有一个概念：一个人不会因为你很优秀就爱上你。比如，我们不会莫名其妙地爱上马云，然后觉得马云好棒，没有他会很痛苦，这绝不会。我们只会因为这个人给我们带来了好处才会爱上他。

同样的，我们换位思考：你现在拼命把你的各方面价值提升，你的爱人不会因此对你回心转意。因为什么呢？只会因为你真正去了解他，给他带来了他想要的好处，他才会重新爱上你。

所以，强调情绪价值，这个东西是什么呢？最高级的情绪价值，就是你每天的状态都是放松的、愉悦的，你的爱人自然也会觉得很舒服。

### 女性独立的真正含义

然后，女性是否独立，要看什么呢？要看和她在一起的男人是否自由。男人要的独立女人，不是女人自己能够挣钱就叫独立，那完全是错误的。一个女人能否独立，是要看这个男人跟她在一起的时候，是否放松，是否自由。

我说的这些，还是希望能够触发你们的一些思考。最核心、最高级的情绪价值就是你自己每天都很开心，你每天开开心心的，这对于男人来说是最高级的情绪价值。其次是什么呢？其次就是男人跟你在一起，他是自由的，他是放松的。
