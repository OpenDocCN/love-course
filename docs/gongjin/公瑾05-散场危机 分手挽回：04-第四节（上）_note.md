# 💔 公瑾05-散场危机 分手挽回：04-第四节（上）

在本节课中，我们将要学习分手挽回三部曲中的第一步：找到关系的症结所在。弄明白分手原因，找出核心问题，对于挽回至关重要。

## 🔍 为何要寻找分手症结？

分手原因是导致关系破裂的最直接因素。它往往也是挽回不成功的最主要因素。因为对方对之前的相处模式感到恐惧，害怕一旦复合，又会陷入到之前那种不愉快的模式中。

对于分手后并非老死不相往来、仍保持联系的情侣，你会发现对方可能愿意联系甚至主动聊天，但就是不愿复合。这是因为对方舍不得你这个人，但恐惧你处理问题的某些方式。

有些人在挽回过程中，明明初期还能交流，但随着接触增多，对方反而越来越想逃离。这一定是因为你在接触过程中，采取的解决问题方式和相处模式，仍是以前那种让对方抗拒和不满的模式。这种情况下，你必然会把对方越推越远。

对方本就因对你不满而分手。若你用相同方法挽回，对方必然会非常排斥。因此，在挽回阶段，不要急着去谈心、发小作文或约吃饭。一定要先找到分手原因，并有意识地避免以该方式与对方接触。此举至少能做到不减分。

很多姑娘咨询时，所谓的“挽回”似乎只是想将对方拉回身边继续相处。这并非挽回关系，只是在为你的依赖寻找方向。

真正的挽回，应是一边与对方接触，一边改变自己、优化自己，解决导致分手的矛盾点。这种改变需发自内心，而非为糊弄对方、骗其回来继续相处。

由此，大家应清楚一点：导致分手的从来不是矛盾或分歧本身，而是你们面对这些问题时的解决方式。

那么，挽回的方向是什么？即找到分手原因后，用更好的相处方式、更高级的解决问题方式去更新和提升自己。成功挽回者，皆因切实改变了之前糟糕的相处模式，双方包容度更高，在挽回对方的同时也提升了自己。

当然，很多姑娘会说分手是因异地、父母反对或三观不合。其实并非如此。世上没有所谓的性格完全合适，无人能与你三观完全一致。不要陷入怪圈，动辄以三观说事。你甚至可能不清楚三观具体指什么。

大家需记住：应更新你们解决矛盾的方式，而非紧盯矛盾本身。亲密关系中摩擦与矛盾必然存在，否则何来“磨合”一说？世上也不存在完全合适的人，相处中总会有矛盾与分歧，因为你们角度、成长环境、对世界的认知皆不同。

即便是朝夕相处的父母，你们也会有吵架、赌气、互相不理解的时候。例如你想周末睡懒觉或晚睡，妈妈可能唠叨你早点休息；你晚上点外卖，妈妈可能说晚上吃东西不好、外卖不健康等等。这都很正常。

因此，当与对方有分歧时，你们应做的是以足够包容度去面对和解决问题，而非一边嚷着三观不合，一边不情愿地相处。

若你遇到一个人，觉得与他三观一致、情商很高，你说的话他都能接上且感兴趣，他说的话都是你爱听的，相处很舒服。那么无一例外，他是在向下兼容你，其情商和城府一定在你之上。

所以，不要过分追求三观一致。婚姻中幸福的夫妻大多求同存异，说白了就是尊重和允许。换而言之，若你看不惯伴侣吸烟，更应反思自己为何最初不找一个不吸烟的，而非强迫其戒烟。为自己的选择负责，就是最大程度地爱自己。

大家记住这句话：**为自己的选择负责，就是最大程度地爱自己**。

## 📝 对照反思：常见的分手相处模式

在今天的课程中，我们将讲解几组常见的、易导致分手的相处模式。希望大家对照这些现象，反思你们之前的相处中是否出现这些问题。

这些问题可能不是单一出现，也许几个同时存在。在此过程中，切勿想着完全照搬我的想法、觉得偷到答案就行。而应思考为何这样做，背后的逻辑是什么。只要培养出这种思维模式和举一反三的习惯，相信大家在本课程中定有巨大收获。

以下是几种常见模式：

### 1. 强势控制 vs 回避防御

例如有这样一对情侣：女生比较强势，不管什么事都喜欢自己做决定，必须按她的想法来，否则就发飙。连男朋友出门穿什么衣服她也要管，甚至男朋友的朋友她也要过滤一遍。遇到分歧时，她处理问题的方式也很强势，不断输出不满，拼命想掌控局面。

她强势的背后，大概率是因为非常容易不安、缺乏安全感。若一个人不够强大、内心胆小恐惧，表现出的就是两个极端：要么咄咄逼人，用满身刺保护自己；要么小心翼翼，敏感脆弱、如履薄冰。

这类女生表面强势，内心却非常缺乏安全感。因此吵架时，她会第一时间把错误推到对方头上，举起双手表示自己没错，全是对方的错。她表现出的是指责别人。指责的目的是控制对方。她内心太弱、太害怕，所以迫不及待想把一切攥在手里。

我们常说“什么锅配什么盖”。这话虽不好听，但对于情感的确如此。很多人来直播间连线，说着伴侣诸多坏毛病，这里不好、那里不行。但大家有没有想过，为何对方那么不好还娶了你？那么多坏毛病还和你生活了十几年？

所以，大家真的要从自己身上找原因，不要只盯着对方缺点不放。对方如何是对方的事，你要反思的是如何能找到更好的，或试着去接受。不要整天抱怨，像个怨妇一样充满负能量，既改变不了现状，还把自己弄得很糟糕。

接着说“什么锅配什么盖”。例如这个强势的女生，往往会找到一个回避型的男朋友，即那种看着很弱但擅长隐性攻击的人。当然角色并非固定，也许你们刚好反过来：老公是大男子主义，而你喜欢逃避。此处我们仍按女生强势的路线举例。

当女生遇事咄咄逼人、强势自作主张时，男人肯定会下意识逃避，因为他本身就是有点回避、害怕问题、不会解决矛盾的性格。尤其当女人步步紧逼时，男人会更想逃避，进入完全的防御状态，整个人很抗拒、很防御。例如把自己关在房间或一个人出去走走，不想面对你。

而女生一旦接收到男朋友逃避的反馈，就会更加愤怒，因为她的气撒不出来。于是男生感到更加恐惧，更想回避。此时女生因总是得不到男生的回应和配合，且能感受到男生的隐性攻击（可能她不清楚何为隐性攻击），也会觉得很受伤，因为她得不到回应。

这强势的女生是善战性格，急于把锅甩给别人。但她想发泄时，男人根本不搭理她。这种感觉就像一拳打在棉花上，很别扭。于是她心中怒火更加膨胀，男人越恐惧后退，她就越是咄咄逼人。这样就进入了恶性循环。这两人关系能好吗？

反过来女生还会说对方冷暴力。这种情况在直播间也常见。这是一种非常典型的亲密关系特征：一方咄咄逼人，善于指责和控制；另一方总是回避，善于使用隐性攻击进行对抗和保护自己，然后各种不配合。

如果你和前任因这种局面分开——每次遇到问题，都是你指责和控制他，或由你引发吵架局面，然后他很防御、很抗拒，最后大家筋疲力尽走向分手——那么不管谁强势、谁回避，总之若你们相处模式是一个强势一个回避，且通过复盘相处过程发现此现象，你就要清楚：当你想挽回这个男生时，在分手后与他接触的过程中，绝不可沿用这种方式交流。

因为他对你的印象仍停留在你很强势、很控制的状态。只要你再次以此方式与他交流，他对你的恐惧和抵抗就会逐渐加深，此时难度会增加，挽回之路会难上加难。

我给大家举个例子。有句话大家肯定听说过：**一朝被蛇咬，十年怕井绳**。即我们之前在某个地方或被某个东西伤害过后，一定会有心理阴影。例如你曾被蛇咬，那么以后的日子里，哪怕没看到蛇，只看到长相和蛇差不多的打水用的绳子，你都会感到害怕。这种生理上的条件反射是人类控制不了的。

我们在感情中也会有这种感觉。若想挽回对方，就一定要改变之前惯用的糟糕相处模式，以新的模式重新相处。即这个女生在挽回男生时，不可强势地指责对方、怪罪对方狠心分手、把所有事情都压在对方头上，想以此吃定他、让他愧疚难受。若仍使用此方式，那么百分之百不会修复成功。人家怕的就是你这点，你还顶风上，怎么可能挽回成功呢？

所以，若你在关系中是特别强势的女生，在挽回阶段一定要改变这个特点。当然，你要真实地、有意识地改变自己，不是装一两天把前任骗回来再原形毕露。

我常给大家举一个例子：有一个员工来到一家公司上班，正常上班时间是8点，他每天7点半就来，开始打扫卫生、忙上忙下。很多人第一反应肯定是：哎呀，这新来的真能装，我看你能装几天。结果这个人第一个星期这样，第二个星期还这样，一个月时这样，两个月时还这样。直到这时，还会有人觉得他在装吗？一定不会了。

所以大家在尝试做出改变时，不要心里想着稀里糊涂弄过去就行，先把他骗过来再说。那就算复合成功也没用。而且这种伪装坚持不了多久，一旦再被识破，你们的关系会变得更加难以修复。

因此，若你们属于第一种类型，且你是比较强势的一方，那你在挽回时一定要更多地去包容，给前任更多时间和空间，允许他没有回应、不配合你。同时，你也要更加平静、平等地与男生互动，通过时间和行动证明：我不会像以前一样对你居高临下、劈头盖脸地指责你，把所有事情都推到你头上。只有做到这一点，对方才有可能看到你的改变，然后渐渐敢和你接触。

### 2. 过度依赖 vs 压力过大

接下来我们看第二类：女生的依赖性特别强烈。相对应的，男方一定会有非常大的压力，不管是心理上还是生活上。这种模式也比较常见。

大家认真听一下。有些模式你可能觉得不是在说你，其实就是在说你，只是你自己没发现，或你在逃避、否认自己糟糕的模式而已。大家不要逃避，而要真正认清自己，找到你们的情感问题所在，然后对症下药。

依赖性特别高的女生，通常有何表现？即一种“巨婴思维”，认为“我和你在一起，你就应该为我做点什么”。例如每次吵架，男人都必须让着她、必须先低头；每个节日，他必须转账送礼物；家里有什么事，都必须由这个男人出马，他一定要比她多做些什么。

这种情况下，你会发现他们之间所有事情都交到了男生手上。看上去男生非常累，女生好像比较幸福，因为她有什么需求，就让对方帮助实现。实际上，这个女人是可怜的，她所有的情绪都由这个男人做主。对方心情好、愿意理她，她可能就高兴一点；对方不想理她、不愿帮她做点什么，她就一定会很难受。你说这样的人能得到真正的快乐吗？肯定不能。

对于这个男人来说，他的压力非常大，需要承接女朋友的全部情绪，有什么事都要由他负责。他当然很累。久而久之，这个男的也会被洗脑，他可能会配合女人来绑架自己，认为“我是一个男人，我就应该这样那样对待你；吵了架我就应该大度一点先来找你服软；你生气我就要哄你，谁让我是男人呢”。所以这个男人一定会越来越累。

这就相当于你们遇到的所有问题，都是这个男人扛着。他就像在养女儿一样，付出所有心血去满足女朋友，什么事都由他自己负责。而且女生的欲望和期待是无休止的，像一个无底洞。所以这个男生的厌倦程度一定是与日俱增的。

这种相处模式一定会导致一个结果：这个女人的价值感会越来越低。因为她看上去很享受被人捧着、照顾着，但实际上她付出得越少，她的价值感流失得就越快。
