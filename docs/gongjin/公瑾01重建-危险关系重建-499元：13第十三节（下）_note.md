# 公瑾01重建-危险关系重建-499元：13第十三节（下）_note

在本节课中，我们将要学习如何打破夫妻关系在出轨危机后陷入的恶性循环。我们将深入剖析双方的心理状态，并掌握一套从“看见”到“重建”的实操方法，帮助关系从敌对走向缓和。

## 现状分析：恐慌与敌对

上一节我们介绍了危机后的普遍困境，本节中我们来看看这种困境的具体表现。

出轨事件发生后，信任被严重破坏。无论你是出轨方还是被伤害方，双方都会陷入巨大的恐惧。这种恐惧导致你们无法轻松相处，也无法接受现状。

你们无法拒绝事实本身。因此，双方的状态会越来越糟。你们可能会发现，无论是买菜、吃饭还是睡觉，任何小事都可能引发争吵。原因在于双方都很紧张，就像两只受伤后彼此防备的小老虎，不停地“呲牙”示威。

在这种持续的内耗中，双方会慢慢耗尽精力，或者其中一方选择离开。这是一个很糟糕的现状。你们很清楚这个现状，因为你们此刻正在受伤。

为了保护自己，你不得不树立防御，甚至想去攻击对方。因为你担心自己会被再次打倒。双方都保持防御和攻击的姿态，就像一手持矛、一手持盾，都不想显得脆弱，都害怕被对方“干掉”。

以下是这种状态下常见的几种处事方式：

*   **逃避与甩锅**：遇到问题选择回避，或互相推卸责任。
*   **指责与咄咄逼人**：直接攻击对方，言语激烈。
*   **抱怨与施压**：通过展示自己的“悲惨”（例如，在朋友圈暗示因对方出轨导致生活混乱）来抢占道德制高点，用道德大棒敲打对方。

这些行为看似是受害者的脆弱表现，实质上是攻击的一种方式。而对方为了防止被打倒，则会用“逃避、甩锅和指责”来回应。于是，你们很容易陷入“一方抱怨施压，另一方逃避指责”的恶性循环，反之亦然。

因为糟糕的相处模式已经形成。任何日常小事，一旦进入这个模式，都会演变成逃避或指责抱怨。这时，你们会感到非常失望和无奈。双方可能都不想离婚，希望家庭能维持下去，但关系却搞成了僵局。

于是只想逃避，却又逃不掉，不想离婚又不知如何是好，就会变得暴躁不安。双方的敏感、防御和攻击会因此更加严重，紧张加剧，彻底陷入恶性循环。你们的“盾”越来越厚，“矛”越来越尖，攻击与防御越来越熟练，痛苦也随之加深。

这时，问题已经与出轨本身关系不大了，核心是你们之间固化的糟糕相处模式。

## 第一步：看见👀

讲完这个令人窒息的现状，我们来说破解的第一步：看见。被看见的力量是极其强大的。

“看见”是什么意思呢？举个例子：假设你10岁时在学校被几个同学欺负了，有人故意绊你。你感到特别受伤和委屈。回到家后，你默默走进房间，不想吃饭，只想睡觉。你试图用坚硬的外壳把柔软的自己包裹起来，假装什么都没发生。

这时，如果你的妈妈走过来，只是简单地说：“你看起来很不开心哦。”仅仅是这样一句看见你状态的话，没有安慰，没有承诺。可能就这么一句话，你就会控制不住地掉下眼泪，把所有委屈都哭出来。

为什么？因为你被看见了。你瞬间觉得安全了，原本强撑的坚硬外壳瓦解了。这就是“看见”的力量。

当你们和老公陷入了恐慌和敌对的状态时，“看见”对方就具有非常重要的意义。当然，如果你们的婚姻问题是经年累月形成的，“看见”的效果不会像小孩的例子那么立竿见影。请记得不要立刻追求效果，要真心去做。

以下是学会“看见彼此”的要点：

*   **理解行为背后的动机**：我知道对方这些行为的背后是怎么回事。例如，老公拒绝你的提议（如周末一起出去玩），可能并非本意。他只是在进行防御，防止自己过度暴露和接触，也担心失去主导权。他的拒绝背后是恐惧。
*   **保持自身平静**：当你发现对方在拒绝或攻击时，不要立刻紧张并陷入自己的防御状态（如指责“你太糟糕了”）。停下来去感知：他不是不喜欢孩子或不想要这个家，而是他不知道如何相处，所以干脆拒绝。搞清楚他背后的情绪，你也会放松下来。
*   **看见是相互的**：你也可以引导对方看见你。例如，当老公提议时，如果你也想拒绝，可以这样说：“我感觉我们俩最近相处得不是很好，我觉得有点尴尬，要不这次就不去了吧。”你是在表达事实和感受，而不是攻击。人类无法拒绝事实本身，这种一致性表达会让对方更容易理解你。
*   **分辨攻击背后的目的**：当对方用攻击的方式回应你（如“吃什么吃，就知道吃”），要去分辨他背后的目的。他可能只是为了自保，防止与你相处时发生不愉快，并用攻击来阻止你持续试图连接。
*   **引导对方看见自己**：当你发现自己下意识攻击对方时，可以马上自我觉察，并告诉对方：“哎呀，我也不是说你不好，我就是突然想到上次不愉快的经历，情绪上来了。”这样就是在引导对方看见你的情绪来源。

当双方都能被看见之后，你们的状态就会好很多。

## 第二步：允许✅

看见的下一步，必然是允许。看见和允许基本上是同时发生的。

“允许”是什么意思呢？在你们换位思考后，你会发现：如果我是他，在这个局面下，我可能也会防御，也会攻击。你会得到一个清晰的概念：**其实你没有我想的那么可怕，你也会害怕受伤**。

一旦看见了彼此的真实状态，你整个人会平静很多。原来我不是唯一的受害者，我也会伤害人，我也不是好惹的。很多人在出轨后会故意把自己搞得很“惨”很“弱”，但实际上，你们也在用“弱”来绑架和攻击对方，你们并没有想象中那么惨。

允许，就是承认我们双方当下的感受，接受当下的事实。大家记得，解决问题的第一步是了解事实，接着就是接受事实，然后把藏在背后的情绪和诉求，敢于暴露出来。

例如，老公回到家，看到孩子玩手机，攻击性地指责：“小孩怎么还在玩手机？”如果你没有“看见”，会迅速回怼，引发争吵。但如果你“看见”了，你会发现他背后的真实情绪是**焦虑**，他非常在意孩子的成绩。

那么，你允许老公关心孩子的成绩吗？你是允许的。所以，你可以停下来说：“老公，你是不是很担心孩子的成绩？但是这么急这么凶的语气让我也很紧张。”——言下之意是，你对他真实的诉求（关心孩子）是允许的，只是不接纳他表达的方式。这样，双方都会放松很多。

允许说白了，就是愿意平等客观地看待这些情绪，因为了解，所以不再恐惧。当你的注意力转移到对方真实的诉求上（如都希望孩子好），你们就有了共同目标，就不用再害怕彼此了。

## 第三步：松软🕊️

让一切平静下来。感情状态才是一切的核心。

很多人在学习过程中会有一种糟糕的心态：我已经看见他了，也允许了，照做了，为什么他还是这样？你要记得，你们糟糕的相处模式是经年累月形成的，不可能因为一两次改变就立刻逆转。

永远都要关注事实，这一切只是一个过程，你没有掌控对方的开关。你需要知道并允许对方有一个“情感过渡期”。任何人的情绪转变都需要一个过程。他从危险不安的状态到放松，需要慢慢积累，尤其是多年情感造成的问题，更需要大量的事情来营造新环境。

例如，你们最近发生了100件事，99件都在吵架攻击。老公不会因为你做一件好事就彻底放松，因为这只占1%。但如果你持续做很多让双方感到放松平和的事，好事堆砌得越来越多，他才会慢慢“松软”下来。

所以，请明白这是一个过程，允许对方和自己都有一个情感过渡期。你也不可能听完课就立刻变得特别柔软，从僵硬到柔软，你自己也需要过程。

松软是对双方的。在尝试柔软的过程中，如果对方重伤你，你也会竖起防御；同样，如果对方尝试柔软，你不可趁机重伤他（如摆出高傲姿态），否则他会发现“软一点没有好处”。如果老公依然坚硬，你要有耐心；如果他尝试柔软，你要回馈他的柔软，帮助感情走向更好的状态。

不要过分关注形式（如他是否回信息、是否回家睡觉），这其实是在回避问题。要关注状态：你们相处是否放松？是否更坦然？你自己是否更柔软？这些才是重点。很多人的“修复”只搞形式，逼对方做表面文章，只会让两人更累。

松软，说白了就是让两个人软下来，就是停战。你不再防御和攻击，明白了对方行为背后的诉求。当你逐步软下来，不再紧张地盯着和控制对方，你们可以平静友善地相处。那些对形式的控制会慢慢消散，你老公也自然会跟着松软下来。

## 第四步：重建🏗️

当两个人都走向松软之后，我们就可以开始重建了。这时，你们才从敌对走向拥抱。前面所有步骤都是从负分走向零分，而当松软达到一定程度，就可以从零分走向加分了。

这时，我们可以重新搭建对话平台。在这个过程中，我们可以开口讲述自己的感受，也愿意承认对方的感受。我们不再通过控制来寻求安全感，关系中没有对错高低，我们可以平等沟通。通过这样的互动，不断拓宽彼此的情感容量。

让我们结合一个常见的冲突场景，应用以上所有步骤：

**场景**：老公回到家，看到孩子玩手机，劈头盖脸指责：“他怎么还在打游戏？”你可能条件反射地防御：“他为什么不能打游戏？”

**传统恶性循环**：双方迅速进入情绪战场——他觉得你放任孩子，你没责任感；你觉得他态度恶劣，还出轨没资格说你。争吵从“孩子玩手机”转移到“谁更负责”的道德高地，再升级到人格攻击（“你简直有病”“你这样子我才会出轨”），最终无人真正关心孩子，双方情绪彻底失控。

**应用“看见-允许-松软-重建”**：
1.  **看见**：停下来思考。他为什么攻击？**他的核心诉求是焦虑孩子的成绩**，只是表达方式糟糕，且可能借机发泄对关系的不满。
2.  **允许**：允许他关心孩子成绩的诉求是合理的。不与他争夺道德高地，不陷入“你出轨你更坏”的战场。
3.  **松软**：让自己先软下来，不防御不回怼。理解他需要情感过渡，不会立刻改变语气。
4.  **重建**：将对话拉回事实和共同目标。
    *   你可以平静地问：“老公，看到孩子玩手机，你是不是很紧张他的作业？”（指出核心诉求）
    *   接着表达感受：“你这个又急又气的语气，搞得我很紧张哦。”（一致性表达）
    *   然后转向解决问题，可以叫来孩子：“宝宝，爸爸是担心你的作业。我们来给爸爸看看，今天的作业是不是已经写完了？”（共同关注孩子）
    *   最后建议未来做法：“老公，下次你可以先问问孩子‘作业写完了吗’，这样大家都会舒服点。”（提供建设性方案）

通过这样的方式，你们避免了情绪混战，解决了实际问题，并为未来的沟通建立了新模式。

## 总结📝

本节课中，我们一起学习了打破出轨后关系恶性循环的完整路径。

我们分析了危机后双方陷入的**恐慌**与敌对状态。接着，我们掌握了四个核心步骤：
1.  **看见**：不急于下判断，停下来感知对方行为背后的情绪和诉求。
2.  **允许**：承认并接受双方当下的感受和事实，因了解而不恐惧。
3.  **松软**：关注感情状态而非形式，允许情感有一个过渡期，让双方逐步从防御攻击中柔软下来。
4.  **重建**：在关系松软的基础上，重新搭建平等、坦诚的对话平台，从共同目标出发解决问题。

恐慌是帮大家分析现状，而**看见、允许、松软、重建**才是需要实操的重点。一旦把前面三个基础打好，重建就是水到渠成的事。重建的成功经验越多，你们关系中的“白子”（积极因素）就越多。感情修复后，外部问题（如小三）自然会脱落。
