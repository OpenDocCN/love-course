# 公瑾05-散场危机 分手挽回：08-第八节_note

在本节课中，我们将要学习挽回阶段的正确方向。上节课我们介绍了挽回的几个雷区，很多人可能因此感到紧张。本节中，我们将明确挽回的核心目的，学习在挽回阶段需要做好的三重管理，并掌握与前任接触时的具体行动准则。

## 明确挽回的目的 🎯

首先，我们需要思考一个问题：你想挽回的目的是什么？是为了习惯有人陪伴，是为了享受恋爱的甜蜜，还是为了挽回那个独一无二的人？你需要清楚一点：挽回的目的不是为了挽回对方，而是为了**挽回这段关系**，或者说**重新建设这段关系**。这个过程的核心是**拥有解决问题的能力和拥有爱的能力**。挽回不是要说服对方回到你身边，而是要你走到对方身边，去营造一个安全舒适的氛围，让他相信你们还能有未来。

以下是三个关键的挽回目的：

**第一点，挽回是为了让自己好受一点。**
挽回这件事很平常，就像饿了吃饭、渴了喝水一样，是一个满足自己的过程。任何人失恋都会难受，你完全可以为了让自己不那么难过而去挽回。但要注意方法，不是你一难受，就一定要对方负责。你可以支持自己挽回的想法，但一定要记得**为自己负责**。

**第二点，挽回是一个自我修复的过程。**
失恋这个词是负面的，但失恋这件事不一定负面。很多人分手后痛苦，但在接受事实并开始新生活后，生活反而更精彩。挽回是为了你自己，是你放不下他。在这个阶段，你做的所有事都是为了满足自己。只要你在挽回的路上向着目标努力，就会比躺在床上缅怀过去更踏实。你会不断踩雷、获得新认知、改变自己。这些努力都会成为你未来感情道路的基石，因为**你一直在成长**。

**第三点，挽回是为了解决相处的问题。**
当你们走到分手这一步，问题通常已积攒很久。如果不去解决核心问题，矛盾只会越来越深。正向的挽回，可以理解为一个**解决问题的过程**。只有解决掉相处问题，调和糟糕的相处模式，两个人才能走得长久。相处的核心问题如果不解决，复合一万遍也没用。前面的课程带大家分析了分手原因，挽回拼的就是你能不能找准这个原因，或者说找准相处的雷区，然后进行修正。

例如，如果你总是看不惯对方的某个缺点（如过度节俭），并因此唠叨、打击对方，最终导致分手。这时你需要考虑：**你到底能不能接受这个缺点？** 如果不能，也不要想着去改变对方，因为你**绝对改变不了**他。如果接受不了，就不要费心挽回。如果能接受，在以后的交往中就不要一直盯着这个缺点，尽力做到求同存异，避免重蹈覆辙。

我们所追求的是**正向的、有效的挽回**，一定是以**解决问题为根本**。在此基础上重新接触对方，让他看到你的改变，相信你能改掉之前的模式。只有这样，挽回成功的几率才会更大。

## 挽回阶段的三重管理 📊

明确了挽回目的后，我们来看本节课的第二部分：在挽回阶段需要做好的三重管理，分别是印象管理、期待管理和目标管理。

### 1. 印象管理

印象管理是一个心理学概念，指人们试图管理和控制他人对自己所形成印象的过程。在挽回阶段，你可以有意无意地通过修饰自己，来完成对方对你印象的转变。这个过程可分为三个时间点：过去、现在和未来。

例如，之前对方对你的印象可能是负面的，因为你可能踩过“死缠烂打”的坑。在这种攻势下，对方会对你的信息产生厌恶和恐惧。要改变这种状况，你需要做好印象管理。

针对“对方恐惧和你对话”的例子，你需要做出相应改变。比如，在交流时找到**合理性**。以前你们总说废话，现在要把话题找在点子上。就像推销员的例子：当前任对你的印象还停留在“你只会来烦我”时，你需要改变他这个刻板印象。当你真的想找他时，可以找一个合理的话题交谈，让他知道你不是来骚扰或刷存在感的。

**公式：合理话题 = 具体事件 + 中性询问**

例如，你可以和前任说：“你现在这个房子租金是多少？我有个朋友在那边工作想租房子，但好像都挺贵的，所以来打听一下。” 这样的交流方式，比问“吃了吗？睡了吗？”更容易让对方有回复的欲望。

在和前任谈话时，**事情本身的比重要大于你的情绪**。同时，你要时刻观察对方情绪的波动。即使你做到了不索取、有事说事，对方可能也不会马上配合，因为他对你还不信任，回复可能很冷漠。比如他只回复“我的房租2000”，就没有下文了。这时你需要**打破他的预期**。一旦他冷漠回复，你就可以简短结束话题，说一句“好的，谢谢”，然后结束对话。千万不要恋战。

随着你们之间“有事说事”的频率逐渐超过“没事找事”的对话，对方一定会放松很多，对你的印象也不再是厌恶和恐惧。你们能更平和、平静地交流和接触。这个过程就是从**仇人走向陌生人，再到朋友**的过程。

### 2. 期待管理

期待管理，简单来说就是**提高对方对你的期待值**，让他把目光重新挪到你身上。具体怎么做呢？

**首先，制造期待感。**
这一点可以通过提高对方的好奇心来实现。例如，你可以跟他说：“我记得你之前给我分享过一个炒股的APP叫什么？我最近想了解一下这方面的知识。” 他回复后，可能会问：“你怎么突然对这个感兴趣了？” 你可以说：“人都是会变的嘛，最近突然感兴趣了，还得靠你这个老司机带带我。” 然后你可以主动接触这个话题，再找个借口去忙别的事。

如果对方只是回复APP名字，不再理你，也不用紧张。你可以在他回复后说：“谢谢你，等我赚了钱请你吃饭。” 然后去做别的事。这句话相当于留了一个话柄。当下他不想聊，不代表以后没话聊。

建议大家以**试探和接触**为目的，并不一定要得到什么回应，而是教大家如何根据对方的回应做出判断，判断下一步该怎么走，以及针对前任不同的反应，应该在哪些方面做出更多努力。

**其次，制造失落感。**
这一点可以通过**断联**来实现，目的是打破对方的预期，尤其是当结果是负面的时候。例如，你以为考试能考100分，结果只考了80分，失落感就会很强烈。这相当于你的前任以为你会一直在身后等他，一回头就能看到你。但当你制造了失落感，他会期待落空，产生强烈的**损失厌恶感**。这种情绪会让他自然地认为你很重要，从而萌生出不舍感。

**最后，引导对方增加投入。**
有了前两点的铺垫，第三点就很好理解了。当你掌握了期待感和失控感的分寸，在操作过程中，就是引导对方为你增加投入的过程。这个部分总结来说就四个字：**少说多想**。不要暴露你的需求感，要给对方留出足够的想象空间。同时，你也需要利用更多时间去思考，面对前任当下的状态，你要做出什么样的回应，而不是一门心思扑上去，总想着把对方抓回来。

挽回一定要讲究技巧。死缠烂打根本没有用。要是你哭几场、作几天对方就回来了，那你们百分之百还会再次分手，因为问题的源头没有解决。两个人都在逃避问题，等问题持续发酵，下一次爆发时，挽回成功的几率会更加渺茫。

### 3. 目标管理

这一点很好理解。我们可以用一句话概括：**把你自己的目标变成你们两个人的目标**。挽回是你的目标，复合是你们两个人的目标。人类都愿意主动去做自己期待且对自己有好处的事。所以，你需要让对方感受到，复合这件事对他也有好处，他才会愿意配合你。

简单举一个过程：
1.  你需要平息之前的相处雷区，让前任对你重新获得信任。
2.  你需要利用断联来打造人设，让前任对你刮目相看。
3.  你需要刻意展示你的改变，让前任对你产生兴趣。
4.  你需要提供更多的时间和空间，让前任对你有一个想象的过程。
5.  你需要找到一个合适的契机，让前任发现复合的好处。

以上五点就是你在目标管理过程中，需要按照顺序去做的几个流程。大家可以着重记笔记，对每个点都多多思考，并有选择地去实现。

## 与前任接触时的四个方向 🤝

明确了目的和管理方法后，我们来看本节课的最后一个部分：你在和前任接触时，具体需要怎么做。

**第一个方向：聚焦事情本身，禁止向对方索取情绪价值。**
“有事说事”这四个字非常重要。很多人觉得这样很冷漠，不像是亲密关系里该有的样子。但其实，大部分夫妻的相处模式都有问题，要么一开口就吵架，要么双向冷战。在这种情况下，“有事说事”就是你们当下最好的沟通方式。不带着攻击和指责，是修复关系的第一步。

在分手挽回阶段，做到“有事说事”并不容易。不带着情绪去交流，在关系不好时是非常困难的。带着情绪的交流，本质上就是在**索取情绪价值**。

例如，分开后你有一个产品调研很难弄，而前任擅长这方面。你去找他帮忙时，**不可以**是那种装可怜的弱者姿态：“哎呀，这个东西烦死了，我弄了几天一点头绪都没有，你来帮帮我吧。” 这就是明显的索取，对方大概率会拒绝或不回复。这其实也把天聊死了，没有给对方选择的权利。

**正确的表达方式**应该是：“我最近在搞一个调研，在网上查了很多资料，但还没有什么思路，弄到一半卡住了。我能不能请你帮我看看下一步该怎么弄？” 这样的表达，对方更容易接受，也更愿意帮忙。得到帮助后，你可以说：“太感谢你了，什么时候有空请你吃个饭吧。” 如果他拒绝，你就说：“好的，那以后有机会再约。” 或者“你有什么需要我帮忙的可以跟我说，我在XX方面还挺厉害的。” 然后结束对话。

尽管这次接触可能没有加分项，但也不存在减分情况，这很难得。因为挽回不是一个一蹴而就的过程，不可能一上来就聊得火热、感情迅速升温。这种情况一定比你那种想索取和依附对方的情况更让人接受。

**第二个方向：见好就收，禁止长篇大论。**
在挽回过程中，对方不一定全程抗拒。因为有感情基础，有时你们可以心平气和聊天，但不一定能谈感情。当对方愿意和你聊下去时，很多人容易犯一个通病：**容易上头**。聊着聊着就聊嗨了，看到对方一直回复，就开始事无巨细地分享生活，或讲过去的陈芝麻烂谷子，甚至开始索取情绪价值。更夸张的是，给前任发好几条50秒的长语音，或拉着对方没完没了地视频。

换位思考一下，如果你是被挽回的人，会有兴趣听完每条语音吗？明知是废话，还会浪费时间听吗？听了之后会有回复的欲望吗？所以，希望大家在交流时**松弛一点**，不要急于分享一切，或急于拿到某个结果。对方现在对你这些事情不感兴趣。一定要克制。

另外，当对方明显对你的话题感到紧张时，一定要转移话题，不要长篇大论。这时宁愿结束对话，也不要聊没有营养的东西。记住：**在挽回的关键时期，不要说你想说的，而是要说你应该说的和对方愿意听的东西。**

**第三个方向：觉察对方状态，禁止强迫对方意愿。**
这一点很好理解，就是字面意思。在接触过程中，你需要**察言观色**，识别对方的情绪，不可以完全以自己的感受为主。你想聊就一直聊，要求对方一直配合你，这不合理，也不利于挽回。

举一个反面案例：前几天直播有个被挽回的人来求助。他的前任在疯狂踩雷：一会儿发消息求见面，一会儿说他没良心，一会儿买东西到楼下，死缠烂打，必须要求男生满足她的想法。被拉黑后，她就来家里找，搞得男生都想搬家了。男生明确说过不要这样纠缠，但对方过几天又会来找。

这个案例中，男生的前任在挽回过程中丝毫不讲逻辑，完全不在意章法，就是跟着自己的情绪走，同时一点也不关注对方的感受和情绪。如果你在挽回时也这样，希望你立刻停下来，去**看见和尊重对方**。但凡前任表现出一点不耐烦，就赶紧撤退，降低聊天成本，降低找他的频率，拉长交流的空间。在每次接触时，一定要关注对方的情绪和态度，尊重他的选择。一定要学会**尊重和允许**。

**第四个方向：找准自身定位，以朋友的标准相处。**
这一点我们也反复提到。很多人根本没有这个意识：虽然已经分手，但还会期待对方来哄自己、陪自己、安慰自己。这个思想很可怕，属于**超现实的期待**。在这种念头下，你对前任的需求感会非常高。但期待越高，失望越大。在高期待的背后，一定是更多的痛苦和焦虑。

所以，希望各位同学能够放下期待，不要在分手后还以恋人的标准去要求对方。说白了你已经没有那个资格了。还有些人会觉得：“对方是不是在等着我去找他？是不是在等着我给他台阶？每次我找他，他应该也会很高兴吧。” 很多人会陷入这种臆想中，这其实是在逃避，无法接受现实。但这绝对不可以成为你持续打扰对方的理由。分手后的聊天，也许不像你以为的那样美好，可能是**你之蜜糖，彼之砒霜**。

## 总结 📝

本节课中，我们一起学习了挽回阶段的三个核心部分：
1.  **明确挽回的目的**：是为了自我感受、自我修复，以及解决根本的相处问题。
2.  **做好三重管理**：通过**印象管理**改变对方对你的负面认知；通过**期待管理**重新吸引对方注意并引导投入；通过**目标管理**将个人目标转化为共同目标。
3.  **掌握接触时的四个方向**：聚焦事情本身、见好就收、觉察对方状态、找准朋友定位。

每个部分都有几个细分的小知识点，希望大家做好笔记，把课程多听几遍，多去思考哪个方向适合你们当下就去做。如果踩了雷也不要紧，可以先停下来，让自己有时间去反思，不要一门心思往上冲。挽回注定是一个漫长的过程，希望大家能有足够的耐心去关注感情的修复质量，而不是急着去得到一个结果。
