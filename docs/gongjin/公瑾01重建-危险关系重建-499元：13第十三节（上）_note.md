# 公瑾01重建-危险关系重建-499元：第十三节（上）💔 理解敌对与恐慌

在本节课中，我们将要学习当婚姻因一方出轨而陷入敌对与抗拒状态时，双方的心理动态与核心问题。我们将剖析这种紧张关系背后的原因，并理解“恐慌”是如何破坏关系功能的。

这应该是每个人都会经历的。在情感里面，一方出轨之后，双方一定会经历一个状态，就是两个人之间会变得非常敏感和小心。当然，有的家人可能会说，我们俩没吵架，老公出轨，我就一直忍着。一样的，不管你忍不忍，你们双方一定会感觉到不对劲。

因为出轨这件事情已经爆出来了，它就是一个重磅炸弹。所以这个时候你们两个人会持续地感到不舒服。你老公可能会抗拒你们两个人之间的关系。但是你老公也不是专门单向地敌对你，实际上他也会有反复。

比如，有的时候他会觉得对不起你；有的时候他有一点就着，好像有很多东西让他不满。他阴晴不定，一会儿把你搞得很紧张、很焦虑。你老公出轨，你就已经很紧张、很焦虑了，再加上他还阴晴不定。

这个时候你们双方就一定会变得很难相处。你想修复婚姻，但老公又是这个死样子。然后你就不知道该怎么办。你说你忍吧，忍也是没有用的，忍了之后你会更痛苦、更难受，因为你忍，老公也还是那个样子。

所以很多人这个时候就崩溃了，就不知道该怎么办了。这里我们来说一下怎么办。今天会分为5个大点来讲：第一个点是搞清楚怎么回事儿，第二个点是看见，第三个点是允许，第四个点是松软，第五个点是重建。

## 1. 搞清楚怎么回事：关系为何如此敌对？🤔

我们一个个来说。我在第一课里面讲到了一个东西，就是我们解决任何问题的思维应该是怎么样的。这个思维的习惯一定是要先了解问题本身，对吧？了解问题本身之后，知道发生了什么，我们再去解决。这个思维是一直贯彻在我所有的课程里边，这里也是一样的。

关系很敌对，老公很抗拒，我们之间到底怎么了？这里用了一个词来解释，叫 **恐慌**。

恐慌之下呢，还有3点需要理解。

### 1.1 失去连接与功能破坏

首先，当我们失去连接，婚姻的功能性可能会被破坏。其实你们夫妻双方有一方出轨之后，意味着什么呢？我在第一课里边就已经讲过，男人出轨的几大要素。首先第一大点就是夫妻双方的感情出问题，双方在感情里都很不舒服。

这句翻译一下，其实就是 **功能性被破坏**。

**功能性**是什么意思呢？很简单，我们在关系里边能够舒服、轻松、踏实、平静地去享受这种关系，你们当下肯定是不可能的。你们双方在这种关系里边，两个人能够被支撑、被看见、被了解、被接纳、被允许，当下肯定也没有，对吧？

然后这种情况下呢，你们夫妻双方也会丧失对话和沟通。因为这个时候大家都会很防御和攻击。大家也都知道这个现状并不好。你们彼此之间根本就没有办法正常对话，大多数对话不是死死地防着，就是小心翼翼地在试探对方，要么就是很想把对方干死，攻击对方。

所以这个时候呢，你们双方一定会失去连接。当一段关系失去连接，所有的感情基础基本通通被破坏掉了。所以功能性呢，也是一定会被破坏掉的。两个人被迫接受一个现状，就是夫妻双方感情现状是很差的。

### 1.2 双方都受到伤害

然后再加上出轨这个事情的爆发，就意味着你们双方肯定都受到了伤害。当然有的人会说，老师，我老公他出轨，他没受到伤害。其实他也受到伤害了。不是说出轨的男人，他就一点伤都没有，这个不现实。出轨的男人，他在各个方面他也是能够感受到实实在在的伤害和痛苦的。

什么意思呢？这里有一句话：**人类无法自欺欺人，无法忽视、拒绝事实本身**。

你老公出轨了，就意味着几个点。第一个点，他对不起整个世界，对吧？他一定会有这种想法。他老婆孩子，他在身边的所有人，都会在道德上去谴责他。然后老婆孩子任何一个点都可能会激发他的防御心理，因为他是一个过错方。

第二个，他内心深处可能会攻击自己。如果他还有小三，小三在情感层面、道德层面也可能是她的债主。如果她不离婚，去跟小三结婚，她也对不起小三。所以无论从哪个角度来讲，你老公在道德层面他是处处被限制，处处感到不舒服。

同时，我们无法拒绝事实本身。什么意思呢？你也很清楚，他和老婆的感情是出了一堆的问题的。而且每一个出轨的人在内心深处都会非常怀疑自己、攻击自己，他很清楚自己处理情感、解决问题的方式非常的垃圾。这个时候呢，他整个人是无法去拒绝当下的事实的，但所有的事实都会告诉她一个结果：她的一切都很糟糕。

所以这个时候大家都是感到不舒服。而且加上有的女性，她们老公出轨之后呢，她们可能也不太会处理，那么会进一步地直接给老公用攻击的方式。比如说有的家人会吵会闹，会逼迫他等等。这个时候呢，老公也可能会受到伤害。但是我不是说你们有错，我能理解你们这么干。

### 1.3 本能反应：恐慌与不信任

你们大家不要紧张，当你们看到这么一个现实的状态的时候，你们还有谁可能会安全很淡定的？这是不可能的。所以大家这个时候呢，一定是会感到很受伤。

当我们人类受伤之后呢，原始的本能就会被激发。什么叫原始的本能？养过宠物的可能都知道，他们会互食。遇到了危险，会恐慌，会变得敏感、紧张。我们人类也是一样的。

于是你们夫妻双方就再也无法去轻松地信任了。所以很多人跟我说，老师，我要怎么去信任我老公？请大家放过自己。你们老公出轨了，他不会再信任你的。因为我们每个人都知道，老公出轨对老婆伤害很大。就好比你把别人砍了一刀，对方现在血淋淋的，你凭什么现在相信对方，对吧？同样的，你被对方砍了一刀，你现在身上血淋淋的，你凭什么再信任对方？

本节课中我们一起学习了，当出轨事件爆发后，关系陷入敌对状态的深层原因。核心在于双方都陷入了 **恐慌**，这导致了情感连接的断裂、关系功能的破坏以及本能性的不信任。理解这些是修复关系的第一步。
