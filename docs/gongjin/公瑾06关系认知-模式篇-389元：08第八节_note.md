# 公瑾06关系认知-模式篇-389元：08第八节_note

在本节课中，我们将围绕“吵架”这一主题展开学习。我们会探讨吵架是如何发生的、为什么有些争吵会越吵越糟，以及如何通过有效的沟通方式，将吵架转化为解决问题的契机，从而改善亲密关系。

## 吵架是如何发生的？🤔

上一节我们介绍了相处模式中的一些基本概念，本节中我们来看看吵架的具体发生机制。

当我们在关系中感到失望、需求未被看见、期待未被满足，或与他人的看法不一致时，内心会产生愤怒、指责和攻击的情绪。这些情绪通过语言方式输出，就形成了吵架。这是一种自然的反应，如同婴儿饿了会哭，是出于为自己考虑的本能表达。

然而，这种方式往往以牺牲关系为代价。因为大多数人在吵架时，常伴随语言暴力。喜欢使用语言暴力的人通常控制欲较强，他们通过打压、指责、不停拉扯等方式，用语言困住对方，以掌控局面。

以下是这类人在语言暴力时的典型目的：
*   目的并非解决问题。
*   急迫地需要对方看到自己受伤的情绪。
*   试图通过打压，让对方认可自己，或让对方感到自身价值感低，从而获得优越感。

## 为什么我们会越吵越糟？🌀

了解了吵架的起因，我们进一步分析为何争吵容易升级并陷入恶性循环。

有些人吵架时喜欢说狠话、常提分手、指责伴侣不够爱自己，或因为小事大发雷霆。脾气越大，往往内心越虚、越焦虑，需要通过虚张声势来获取力量，缓解心理焦虑。这类人有一个明显特点：容易情绪化。在与伴侣产生矛盾时，个人情绪总是大于事件本身。

这可能是因为过往积攒的情绪一次性爆发。于是我们开始翻旧账，使矛盾升级。

**举个例子：**
女生的情绪是逐渐累积的。A事件：老公袜子乱放，你看见但没说。B事件：吃饭时老公一直刷手机，你想分享工作见闻却没说出口。C事件：下车时老公自顾自往前走，没帮你拎东西，你仍不开心但未表达。D事件：孩子找爸爸时，爸爸只顾玩手机敷衍应对。此时，你终于爆发，借孩子之事指责老公。但实际上，你想表达的是“你眼里还有我吗？”，却因不好意思直接表达需求，或觉得为前几件小事吵架显得矫情，而选择了借题发挥。

老公感到莫名其妙，觉得至于发这么大脾气吗？于是争吵开始。如果你开始翻旧账，目的就变成了“打败他”。在这个过程中，你越说越生气，没完没了。最后，老公被迫“服软”，你的情绪暂时宣泄完毕。但问题并未解决，老公也不知道你的真实需求，只认为你强势、不温柔。他作为被动吵架的一方，情绪未得到合理宣泄，每次事件都为他积累一次负面情绪，为未来关系埋下隐患。

**核心问题在于：**
你们吵架的目的并非解决问题，只是在宣泄情绪，让自己好受一点。你的需求不是“为了孩子”，而是期待老公看到你的情绪、感受你的愤怒，过来哄你。这时，你看起来咄咄逼人，实则非常“托付”——你将自身情绪交给了对方。他哄你，你就好受；他不哄或与你吵，你就焦虑，负面情绪加深，吵得更厉害。

**关于“哄”的误区：**
有些姑娘总期待男人来哄自己，并将此与爱不爱挂钩。从男性角度看，哄人通常有两个目的：
1.  **安抚情绪，搞定你**：目的是让你消停，别闹了。
2.  **逃避问题**：说几句好话就能让事情过去，无需真正面对和解决问题。

哄是一种“递减效应”，慢慢你会觉得他在敷衍。除非你也恐惧面对问题，选择共同逃避。但问题始终存在。因此，不要总索取“哄”这种情绪价值，这会使你显得依赖且脆弱，不利于个人成长与关系健康。

如果总是通过吵架来宣泄情绪和索取情绪价值，双方只会共同逃避问题，关系越来越对抗。一旦陷入对错之争，事情本身已不再重要，谁输谁赢才是关键。赢的一方看似赢了面子，却可能输掉对方的爱。亲密关系中的对错往往取决于视角，关系本身比输赢更重要。

## 如何有效吵架，让关系更近？✨

既然吵架有时不可避免，我们就要学习如何“好好吵架”，让其为我们所用，疏散情绪，拉近关系。

好的吵架可以是解决问题的一种方式，通过它说出想法和感受，不是为了强加思想或必须按对方感受行事，而是为了“求同存异”。夫妻间“床头吵架床尾和”，有些夫妻越吵关系越好，正是找到了舒服的相处模式。

**有效吵架的核心原则：**
*   **允许平等表达**：双方都可以说感受，也可以随时闭嘴。
*   **目的不是说服**：重点是表达自己，不憋在心里。若能相互认同，则更好。
*   **关注彼此感受**：尊重对方的表达。
*   **就事论事**：看见事情本身，不要无休止翻旧账。翻旧账只会让矛盾升级，无助于解决任何一件事。
*   **以解决问题为目的**：不是“我解决你”或“你解决我”，而是“我们一起解决问题”。

正向的吵架允许彼此有情绪，但更要**掌控局面**，不让情绪淹没自己。要准确识别情绪，吵的是“事”，而不是“情绪”。输赢根本不重要，重点是**事情有没有被解决**，以后还会不会因此再吵。

**关键区分：把“人”和“事”分开**
吵架应围绕“事情本身”（利益之争），而不是“事情背后的人”。如果上升到人身攻击，吵的就是输赢和“生死”（地位之争）。没有人愿意被定义为“犯错的人”或“糟糕的人”。

**举例：**
老公与女同事边界感不清，常帮忙。老公觉得没什么，但你无法接受。
*   **错误吵法**：人身攻击。“你就是个渣男，跟女同事搞暧昧！”（攻击人，否认人品）这会导致老公拒绝认错，甚至可能因你的强化而格外关注该同事，关系恶化。
*   **正确吵法**：就事论事。“老公，我不喜欢你给别的女人帮忙，我吃醋。我知道你对她没想法，也相信我们的感情。但我就是不愿意，不喜欢这样。我们看看怎么办好？”（强调人是好的，但事情不接受）这样更可能吵出结果，解决问题。

**警惕语言暴力：**
语言暴力也是暴力，摧残人的精神。如果你对伴侣语言暴力（大嗓门、不停输出），对方可能说不过你，渐渐演变为冷战。反之，如果伴侣对你语言暴力，你也需要站起来反抗，表达感受和底线。

**给伴侣正向反馈：**
当对方稍有配合时，一定要鼓励，而不是持续提要求。不要指望对方是你肚里的蛔虫。当他做了一件事，表达感激和开心，而不是抱怨另一件事还没做或做得不够好。给予正向情绪价值，以“赚到了”的心态相处。几个回合后，你们会变得更默契、更宽容。

## 吵架时的实用建议 📝

以下是吵架时可以用来稳住自己的几种方式，请务必牢记：

1.  **把握情绪**：不要被情绪牵着走，这是最基本的一点。
2.  **及时暂停**：当觉察自己或对方情绪失控、脱离理智时，马上停下来，留出空间和时间平复情绪，防止争吵无休止。
3.  **放弃控制**：不要总想着控制对方。
4.  **反思自我**：当对方陷入沉默，先想想自己是否过于咄咄逼人，而不是直接控诉对方冷暴力。
5.  **慎说狠话**：情绪上头时别说狠话，也别太在意对方的狠话（不能不信，也不能全信）。
6.  **避免逼问**：不要在吵架白热化时逼问“你是不是不爱我了”，此时答案无意义。
7.  **拒绝受害者心态**：不要把自己渲染得很可怜，这得不到同情，反而可能让对方看低你或感到压力。
8.  **停止被害妄想**：不要总觉得对方要置你于死地，从而疯狂攻击。
9.  **坚守原则**：吵架是就事论事为了解决问题，不是人身攻击或宣泄情绪。
10. **警惕粗暴上瘾**：不要对粗暴的解决方式（如摔东西、语言暴力）上瘾，这会依赖并推开伴侣。
11. **重申底线**：关于家暴，再次强调，粗暴方式只有0次和无数次，务必零容忍。

## 越吵越糟关系的特征 ⚠️

我们可以将本点与“如何有效吵架”对比记忆。越吵越糟的关系通常有以下特征（即吵架的“雷区”）：

1.  **把吵架当宣泄工具**：只为发泄情绪，不解决问题。
2.  **渴望收服控制对方**：总想改变或控制伴侣。
3.  **沟通通道双标**：不允许对方说话（视为攻击），也不允许对方闭嘴（视为冷暴力）。
4.  **争夺控制权**：双方都想占据上风，吵赢对方。
5.  **不直接表达感受**：态度强硬，不坦诚。
6.  **忽视对方感受**：看不见或不在乎伴侣的情绪。
7.  **情绪管理失控**：任由负面情绪肆意增长，造成持续伤害。

请大家记住这七个点，并在亲密关系中尽量避免。这些是吵架时常见的错误，需要我们有意识地去规避和调整。
