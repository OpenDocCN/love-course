# 公瑾06关系认知-模式篇-389元：02第二节_note

## 概述

在本节课中，我们将要学习如何区分亲密关系中的“表达”与“控制”。很多人分不清什么是合理的表达，什么是表达背后的控制，这也是导致沟通逐渐减少的主要原因。我们将通过具体的例子和清晰的步骤，帮助你理解并实践健康的表达方式。

---

## 区分表达与控制

上一节我们介绍了关系互动的形式，本节中我们来看看表达与控制的核心区别。

很多人会抱怨伴侣不喜欢沟通、逃避问题或冷暴力。这时需要反问自己：伴侣为什么要喜欢和你讲话？和你沟通他能得到什么好处？

例如，如果你老公和你聊10分钟，你就给他1000块，你看他愿不愿意聊。或者你问他：“老公，今天晚上我是给你捏脚还是揉背？我是给你做排骨还是做海鲜？我这个月的工资是全上交还是留100块？”你看看他合不和你聊。

你想要的沟通，可能只是追求形式上的互动。你以为沟通能解决问题、加深感情，但忘了一个重要前提：必须是两个人互相配合，而不是你拿刀架在他脖子上逼他聊。他都感觉自己的命快没了，还跟你聊什么？

抱怨伴侣冷暴力的人，80%自己都是施暴者。你需要审视自己沟通的目的：你无非是想要对方听你的、满足你或证明他有错。总之，这个沟通一定是为了满足你。这时，你等于拿着一把刀让对方妥协，换做是你，你愿意吗？

还有一些姑娘在关系里不敢发脾气、不敢表达需求、不敢说出真实想法。这说明你在亲密关系里很不自由、不够轻松。但你们的早期关系是这样的吗？大多数都不是。如果早期就是这样，也许你们走不了这么远，毕竟早期的感情也不深，还能够脱离开。

就像上节课讲的，两个人从陌生到熟悉，相识相知到相爱，关系从好到坏，从坏到好，都有一个过程，不是一下子完成的。沟通的通道从关闭到开放，再从开放一步一步走向关闭。在这个过程中，表达欲持续下降，直到渐渐闭嘴，最终剪断和伴侣之间的交流链接。

表达是我们与生俱来的本能。不管是男人还是女人，我们都有倾诉欲、分享欲，都渴望展示自己，希望别人关注到自己。同时，每个人也渴望了解别人、倾听别人。所以大多数人刚开始进入亲密关系时，会积极表达、建立交流链接。

但很多人不会表达，受原生家庭、成长环境、求学经历等影响，形成了不会表达的特点。当他不会正向表达，就经常引发负向糟糕事情的发生。随着糟糕事情越来越多，这个人就越来越不会表达、不擅长表达，走向不愿意表达，直到最后彻底选择闭嘴。

这里说的“闭嘴”，指的不仅仅是嘴巴上的关闭，包括肢体语言、行为、神态等。可能你们关系不好之后，他不仅不和你说话，家里的事情也放手不管了。这时，你可能会感受到关系让你很不舒服。

## 常见的情感体验

在这个过程当中，我们会有哪些常见的情感体验呢？

以下是你在亲密关系中可能有的几种体验，也可以带入到你老公的角色：

1.  老公总是违背你的想法，所以你常常对他很不满。
2.  你觉得你是对的，老公就应该听你的。
3.  你会为你的控制行为去找借口，进行逻辑上的自洽。
4.  综合以上几点，你特别希望伴侣能按照你的想法做出改变。

这几点大家可以自己对照一下。

就像直播间很多姐妹问：“老师，我老公特别爱喝酒，劝他也不听，我该怎么办？”这就是一个典型案例。老公爱喝酒，你劝了一遍又一遍，管了一遍又一遍，但一点用都没有。你希望他改变，但你从未关注过他为什么喜欢喝酒，也没有好好思考为什么自己要管他喝酒这个事情。

很多姐妹说：“我是为他好，为他的健康着想。”当你为你的控制找到“为他好”这个冠冕堂皇的借口时，觉不觉得这一幕特别熟悉，有点似曾相识？

我经常举这个例子：当你想管住你老公、想改变他、让他听你话的时候，有没有觉得你活成了你父母的样子？当你十几岁时，你父母是不是经常跟你讲：“这个事情不能做，那个事情不能做，你要这样做，你要那样做，不能跟这个玩，不能跟那个玩。”他们给你规定了很多条条框框，他们打着的旗号也是“为你好”。那时你的感受是什么？是不是也很烦？难道你的父母真的是为你好吗？你要想一想，他们背后是为了满足自己的安全感。他们怕你走向不好的状态，脱离他们的掌控，或者你的人生走向不好时，就等于他们的人生很糟糕，等于他们是失败的父母。

同样，今天你也是一样的。那时你不想父母“为你好”，你只是想要自由。其实你老公现在也是这样的。他喝酒也许是因为有压力，不知道怎么发泄。你说他不知道喝酒不好吗？他当然知道，他又不傻，都是成年人了。但可能你老公现在就像叛逆时的你，你就像那时的父母。你越是劝他戒酒，或者越是管他，他越是反抗，越是不顺从。

所以你在亲密关系里，很容易把表达变成了控制。你原本的意愿可能真是为了老公好，怕他喝多了对身体不好。但这句话可能还是在骗你：你是怕他身体不好、身体糟糕了，未来你要付出很多代价，要给他治病、花钱、陪伴他。这对你来讲，是缺失安全感的事情。所以你根本谈不上是为他好，你只是为了满足你自己而已。

但时间久了之后，他一再拒绝、一再紧闭，渐渐的你的表达会变得有些强势，逐步变成了控制。

表达和控制是完全不同的两种方式。我们正常的表达自己时，就是可以合理说出我的需求。我可以没有理由，只是我想要这样去做。就像：“老公，我今天想去看电影。”老公说：“今天可能不行，我今天可能要加班。”那你不会觉得受到攻击，你会觉得：“好吧，那我就自己去看嘛，或者等到明天后天你什么时候空了，我们再去看。”

而控制是：我是对的，所以你必须要听我的。就像老公喝酒这件事，你会站在一个高度上义正言辞地告诉你老公：“老公，你这样是不对的，你应该戒酒，我是为你好，我是对的，你要听我的。”你可以感受到，合理表达和绝对控制是完全不同的两种行为。

所以这节课的重点，就是我们如何去好好表达，不要用控制去代替你的表达。

## 如何实践健康表达

我们这节课同样给大家整理了四个要点：

1.  改变认知
2.  理解控制
3.  认识防御
4.  合理表达

我们分别来看一下。

### 1. 改变认知

首先第一个点是改变认知。认知是你对人性的理解，对亲密关系的供需生态、基本功能的一个理解。这个理解不是你过往以为的。

我经常说：抛开人性谈感情都是幼稚。如果你只是靠喝了几碗毒鸡汤，听别人讲了一些东西，就想经营好亲密关系，无异于痴心妄想、痴人说梦、天方夜谭。总之，你可以想尽一切类似的词语来形容这个事情。

那人性是什么呢？**人性是趋利避害，人性是不顺从，人性是一切的行为都是基于对自己有利。** 无论是谁做了什么样的事情，哪怕是杀人放火，哪怕是牺牲自己的生命，背后深层次都一定是为了自己。所以都不要把自己说得有多高尚，什么“为了谁谁谁”。哪怕就算是父母为了孩子，都是在实现自我价值。别说什么为了伴侣，千万别打着爱的幌子去绑架别人，这是一件非常恶心的事情。当然，我们也不能允许别人用这样的方式来绑架我们、控制我们。

就像我们上面说的例子，当你在亲密关系中有表达走向控制的时候，你就需要去改变了，否则你们的关系就会越来越糟糕。你的控制欲渐渐加强时，你老公的自我防御也会逐渐增加。那么你们之间的沟通通道，会距离关闭越来越近。

当然，很多姑娘会说：“老师，我是想改变的，但是我不知道怎么去改。”在这里给大家几点建议。你可以记下来，但也不要只是记下来、写完了放在哪里就不管了，然后跑过来说：“老师，我听不懂。”或者“老师，我听了，我听了好几遍了，还做笔记呢，但是我还是不知道怎么去做呀。”大家不要这样。你既然买了课、花了钱、要解决问题，那就要对自己负责。你听课要多听几遍，写写笔记，要去琢磨，按照课程里讲的去做，找到适合自己的方式，然后做出改变。当然，你要先改变认知，再去改变行为的方式。永远记得一句话：**认知决定了你的思维模式，你的思维模式就决定了你的行为。**

好，那么我们接下来简单讲讲做出改变的几个小步骤。

首先，你要“看见”。看见是什么呢？看见对方的需求，这个需求说的是深层次的需求。老公爱喝酒、爱吸烟，你看到他是一个有不良嗜好的男人，但你没有走进他更深层次的层面。“看见”这两个字其实是很有力量的。心理学里面讲，最好的爱就是“看见”。这个看见，是看见行为背后的期待和渴望。

好的，那么我们接下来说第二个，你要“允许”。允许他的存在，允许他成为一个人。这里是什么意思呢？为大家解释一下，“允许他是一个人”指的是你要尊重他，尊重他是一个独立的、自由的个体。你老公是一个成年人，他有他自己的选择，他有他自己的权利和自由。没有人喜欢被束缚的，人人都渴望自由。那有人说了：“哎呀，那渴望自由结什么婚呢？”这个自由跟结婚是没有冲突的。你允许他自由，他也允许你自由，你们之间才会建立好的关系。有个好的关系呢，才会有好的婚姻质量，有好的婚姻质量呢，才会有长久的稳定、一个幸福的家庭。如果人连最基本的自由都没有了，还谈什么幸福？幸福个屁呀，对吧？所以啊，当他发现他做什么事情都是被允许的时候，他就会松弛下来，会有这个自由的感觉。然后有了这个自由的感觉，你们之间才会有更好的交流。

最后一点就是，你要去“解放你自己”。别整天去盯着这个男人，看他喝酒啊、吸烟啊、打架啊、乱扔臭袜子等等等等。这些你不要盯着这些东西去看。当你过分去关注他这些行为的时候，你已经不是你自己了，而且同时你也不允许他是他。好好去品一下这句话：真正健康的亲密关系、好的亲密关系，就是你是你，他是他。正因为你是你，正因为他是他，所以才有了你们。如果你不允许他是他，他不允许你是你，那么你们是怎么来的呢？你们找的是什么？找个什么玩意儿啊，对不对？你找的时候你怎么想的呢？对吧？所以啊，这些东西你要记住了，去好好品一品。

当你整个人贴在你老公身上，当你整个人把焦点都关注在他身上，密切关注他的行为的时候，你不再是全部的你，就好像你有一部分是由你老公组成的。你十分钟看不到他，或者你几分钟不去唠叨他，你就觉得浑身不自在。而你老公的感受是什么呢？就一个字：烦。三个字：烦死了。当你想要掌控你老公的时候，你是很累的，你要管东管西，什么都想说了算。而你老公被你管着的时候，他整个人都是恐惧和厌烦的，非常想要逃离。这样下去，你想象你们之间的亲密关系会走到一个什么样的地步。

当然，也有很多学员会说：“老师，可是我也表达了呀，但是他还是不改呀，那你说我表达的意义在哪里呢？”这其实是一个误区。有很多人同样的疑问：“哎，老师，我又好好沟通啊，但是老公可能前脚答应了我，过会儿他就忘了，然后该咋咋地，他也没改呀。”

那为什么说这是个误区呢？表达其实是件“向内求”的行为。就是说：我要表达，是因为我想要说出口。对于你给我的反馈，我不做更多的要求。你顺应了我表达的需求，那我一定是很高兴的。但如果你拒绝了我，或者你不认同我的表达，我也是可以接受的。否则，你的这个表达就是控制。

我经常跟大家讲：这个世界没有人是按照你的思维来活的，没有人活着是为了你活的。对吧？你想，可能吗？

举个例子来说：今天你和老公出去买菜，回到家之后，老公把菜放到了门口，然后就洗手了。当你换了鞋很自然地想要把这些菜拎到厨房的时候，你突然发现好像每次买菜回来都是这样，每一次都是老公放在门口就不管了，然后你拎着菜去了厨房。可是这次你们买的菜有点多，如果你自己一趟一趟送的话，可能要走两三趟。

我们正常表达可能就是直接和老公去说：“老公啊，你别洗手了，今天买的菜有点多、有点重，你和我一起把菜先放到厨房去，我们再去洗手呗。”那你老公可能会说：“好啊。”然后走过来提着菜去了厨房。那还有一种可能，他可能已经在洗手了，或者他还是拒绝了你。也许他会说：“老婆，你就辛苦一下拎过去就行了。”然后转身该干嘛干嘛去了。那你可能就会瞪她一眼，骂她一句“真懒”，然后自己把这些菜提过去了。

在这个过程当中，我们两个人都是轻松的、自由的、没有顾虑地去表达了自己真实的想法。你也可以没有顾虑地答应或者拒绝。我不害怕你的拒绝，你也不用因为怕拒绝了我而感到有什么样的负担。这是一个很正常的相处。

还有一些姑娘呢，可能她想到在楼下的时候，就是老公拿着大部分的菜，那现在自己就直接多走了几次，把菜运过去算了。她相当于抹杀了自己表达的机会。当她一步一步拎着菜走向厨房的时候，她心里可能是不太开心的，因为她的需求是没有表达出来的。

或者还有一些姑娘可能一上来就会命令、要求老公把菜放过去。如果老公做了，她可能就会觉得理所应当的。老公要是拒绝了，她可能就会把问题升级到了一个新的高度，会指责、会控制。这些肯定都是要改变的。

如果你在提出了需求以后，老公没有满足或者直接拒绝了你，你会因此产生伤心、指责、抱怨等等负面情绪，那说明你在亲密关系里边，你“向外求”的心态是很弱的。就是你的认知里边，你觉得表达一定要得到回应，而且你会期待这个回应是正向的。如果不是，那你就会很痛苦。因为你没有关注你的需求，而是把需求当成了对抗的工具。你没有思考过你为什么想表达自己，你只是想通过表达来控制对方，让他去做你期待的事情。

**表达是为了说出我的感受，而不是必须要求你去怎么做。** 这句话大家可以记一下，这其实就是“向内求”的核心。我的表达是为了我自己，为了说出来，我会很舒服，而不是要通过说出来去为难你和控制你、去操控你。我有表达需求的自由，你也有拒绝我的自由。我们允许彼此、接纳彼此，所以在关系里边才会觉得轻松，促成关系里边的良性循环。这就是当你把需求向内求的时候，你能够得到的一个良好的反馈。

反过来说，当你向外求的时候呢，看上去你好像是在控制对方，实际上呢，你是把你自己的情绪的开关都交到了对方的手里。你等于把你自己的情绪的控制权交到了别人的手里，对吧？这个其实很好理解的。就是当你把自我需求当成工具来控制对方的时候，他拒绝你，你会非常非常的难受，就相当于把把柄落在了你老公的手里。你老公只要拒绝你，你就会情绪收场；他要是迎合了你，你就开心。时间久了之后呢，你就会像一个神经病一样，人家好你就高兴，人家不好，你就难受。而且这种情绪的影响不是短期的，你会长长久久地被你老公掌控着你的喜怒哀乐。你不觉得很恐怖吗？

其实你真的可以直接去表达的。你就把你自己的真实的想法说出来，允许对方认可或者拒绝。你在表达需求的时候呢，是在支持你自己。说出来是为了自己舒服。这个之前就讲过了，这里就不再多说了。同样道理，在和孩子相处的时候也是一样，更多的去“看见”和“允许”。

### 2. 理解控制

接下来我们来看第二个部分：控制。什么意思呢？就是说在你的认知里是分不清楚表达和控制的。就是当你的内心产生了一个想法、你有需求的时候呢，你会把它提出来，但是你说出来的目的不是为了让自己好受，而是用你的这个需求来去掌控对方。

常见的掌控方式有两种：一个是情感PUA，另一个就是判断对和错。我们分开来讲一下。

首先来说一下情感PUA。你作为一个母亲，可能你每天早上要起来给孩子做早饭，准备好他今天要穿的衣服，你要做很多的事情，而且不是一天两天这种短期的，是要长期坚持去做的。尤其是做早饭这件事情，你可能要定一个很早的闹钟，然后换着样式地去给孩子做，而且还要有营养、要健康，对不对？

那么很多妈妈在做了这个事情之后呢，她就会期待着孩子能够开心的去吃早饭，能够珍惜自己的劳动成果。“不用你有多感谢我，但是你要乖乖的吃啊，不要浪费，弄得到处都是的，糟蹋粮食这是不行的。”但是呢，小孩子他是不懂你这些想法的，对吧？他也许没有关注到你的这些情绪啊。可能你叫他起床之后，他就慢吞吞地走出了房间，或者他都懒床了很久才起来。因为他可能还没有睡醒嘛，他可能就会比较磨蹭。然后你催了他几次之后呢，他终于洗漱好了，坐到了那个桌子面前开始吃饭。但他又没有像你想象的那样啊，没有你期待那样的去好好吃饭，他可能弄得到处都是，桌子上弄得很脏很乱。你看见了之后，你可能就会很生气、很受伤。你会觉得：“我辛辛苦苦做的早饭，你就这样不珍惜。我起了那么早给你准备了这么久，每天我这么辛苦啊。”

那你可能就会跟你的孩子说：“宝贝啊，你看你这个早饭，是妈妈早上起来给你做的，妈妈很辛苦啊。妈妈都没有睡醒，妈妈很累的，妈妈很困的，但是为了你不能饿肚子，妈妈还是爬起来给你准备了这些。妈妈真的很辛苦啊，你不可以这样浪费粮食。妈妈会伤心呢，你乖乖的吧，他们都吃掉，或者你吃不下也别浪费啊，别弄得到处都是，妈妈还要擦桌子等等等等，真的很累哦。”

那这个就是典型的情感PUA。你用你的辛苦、你的付出来绑架孩子，让孩子觉得妈妈这么不容易，我必须乖乖的吃完。这会让孩子有很大的压力。她的确能够感受到妈妈的辛苦，但是她接受到的更多的事：“好好吃饭是妈妈给我布置的任务，我必须要听话，否则就是对不起妈妈，是在伤害妈妈。”在这个念头下呢，孩子真的会很有压力的。为什么很多小孩子小的时候不爱吃饭？就是家长在反复地告诉他：“你不要挑食，不喜欢吃的东西要吃，为了你的身体健康，为了你能长高，你不能挑食，对不对？”其实这是一个很奇怪的事情呢？你看我们小的时候可能父母就是这样教育我们，我们当时是很抗拒的，对吧？不喜欢吃就不喜欢吃呗，还硬要吃，逼着自己去吃，很烦，对不对？但是当我们长大之后呢，有些人又在逼自己的孩子，都是在以爱之名，打着为你好的旗号去逼你做你不喜欢的事情。
