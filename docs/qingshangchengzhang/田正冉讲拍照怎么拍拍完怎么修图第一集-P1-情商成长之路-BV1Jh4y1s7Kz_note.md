# 📸 田正冉讲拍照 #第一集：明确主题与基础修图

在本节课中，我们将学习如何拍摄一张“有逼格”的照片，并掌握基础的后期修图技巧。课程将分为两个主要部分：前期拍摄的核心原则与两款实用修图软件的操作演示。

一张优秀的照片，其成功始于拍摄阶段。无论后期如何修饰，如果前期拍摄不佳，最终效果也会大打折扣。因此，课程的前几节将重点讲解拍照、构图、环境选择和姿势摆拍等方法。

## 🎯 第一部分：前期拍摄的核心原则

前期拍摄是照片质量的基石。以下是四个需要遵循的核心原则。

### 1. 明确主题
一张好的照片必须有一个明确且鲜明的主题。无论是表现人物、事物、故事还是某个情节，主题都必须清晰，毫不含糊，让人一眼就能理解你想表达的核心。

### 2. 处理主体与加分项的关系
想让别人对你的照片留下深刻印象，你需要让照片中的主体与背景等加分项产生互动。这种互动能营造出“无形装逼最为致命”的效果。

### 3. 注重构图与画面
如果只是随意记录，可以忽略构图。但若想用手机拍出“大片”效果，就必须注意对焦、光源角度和构图等方面。这些细节将在后续课程中详细展开。

### 4. 理解美化（修图）的作用
美化即修图，它将贯穿整个课程。修图并非不重要，但在“装逼照片”的创作中，它并非最重要的一环。前期拍摄才是核心，后期修图仅起到润色作用。

上一节我们概述了前期拍摄的四大原则，本节中我们将深入探讨第一个原则——如何明确主题。

## 🍽️ 如何明确主题：以朋友圈为例

主题明确意味着你想通过照片传达给观看者（例如朋友圈中的异性）的核心信息。你需要清晰地将这个信息传递出去。

以下是几个明确主题的具体方向：

*   **美食主题**：分享美食是一个经典主题。但需注意，不要在朋友圈连发九张全是食物的照片。偶尔发一张有特色的美食是可以的。关键在于“定位”，如果你走高端路线，定位不应是人均消费几十元的地方；如果走特色好玩路线，则要找有特色且性价比高的地方。
*   **人物形象主题**：朋友圈中，你的人物形象至关重要。首先，外形风格不能出错，这里指的是穿衣打扮的风格。建立独特的个人风格（例如小清新混合嘻哈风）能让你在他人眼中拥有更高的辨识度。
*   **面部展示**：朋友圈中必须有你的脸。可以通过自拍或他拍实现，通常他拍比频繁自拍显得更自然。除了特殊有趣的场合，建议多采用他拍。
*   **肢体语言与景别**：朋友圈应包含不同景别的照片，以全面展示你的肢体语言和形象。景别指对人物取景范围的大小。

接下来，我们详细解释一下不同景别的运用。

### 不同景别的运用指南
景别决定了人物在画面中的呈现范围，不同的景别有不同作用。

*   **特写**：聚焦面部。能细微刻画人物表情和形象。如果你的颜值高，可以适当多放特写；如果长相普通，则不宜过多。
*   **近景**：取景范围大约到胸口或腹部。能很好地展示除脸部外的肢体语言和着装风格。
*   **中景**：取景范围下沿到大腿或小腿中部。**切记裁剪线不能在关节处**，而应在躯干部分，否则人物会有被“肢解”的怪异感。中景能展示腿部的肢体语言和裤子等服装。
*   **大全景**：人物全身入镜。能完整展示身体、服装风格和全部肢体语言，大幅提高人物整体辨识度。对于颜值不突出的人来说，这能弱化面部特征，强化肢体辨识度。但大全景对背景要求较高，需要找到干净或大气的背景。

在了解了如何通过不同景别展示人物后，背景的处理同样关键。

### 背景处理要点
背景处理需根据景别调整。

*   **特写与近/中景**：背景务必干净整洁。可以选用修图软件将背景虚化，以突出人物。
*   **大全景**：环境可以相对复杂，但取景构图必须干净，人物在环境中不能显得突兀或低档。

为了处理背景，这里推荐两款实用的手机修图软件。

## 🛠️ 第二部分：修图软件实战演示

以下是两款用于背景虚化和人物突出的付费软件介绍，它们在Apple Store有售，总价约30元，是一项值得的投资。

1.  **Tada SLR**：主要用于精确选择人物边缘并进行背景虚化。
2.  **Facetune**：兼具背景虚化和面部修饰功能。

下面，我们来看看这两款软件的具体操作方法。

### 软件一：Tada SLR 使用教程
这款软件能帮助你精确地将人物从背景中分离并虚化背景。

1.  打开Tada SLR，导入照片。
2.  使用左下角的“Master”笔刷工具，并打开边缘识别功能（Edge按钮）。
3.  沿着人物轮廓涂抹，软件会自动识别相近颜色区域。对于识别不准的部分，需手动放大图片精细涂抹或擦除。
4.  涂抹完成后，点击“Next”。在虚化界面，选择“Linear”（线性虚化）模式。
5.  拖动虚化范围控制条，将人物部分保持清晰，背景部分设为虚化。**特别注意人物脚部与地面的接触要实，不能有悬空感**。
6.  调整上方的模糊强度滑块，控制虚化程度。
7.  点击完成并保存图片。
8.  将保存的图片导入“美图秀秀”进行进一步美化：可依次使用“HDR”增强清晰度、“LOMO”滤镜调整色调、“锐化”功能让细节更突出。最后用“裁剪”功能调整构图，并用“祛斑”功能处理人物边缘的瑕疵。

### 软件二：Facetune 使用教程
这款软件操作更直观，适合快速虚化背景。

1.  打开Facetune，导入照片。
2.  使用“散焦”工具，涂抹整个画面使背景模糊。
3.  使用“擦除”工具，小心地将人物部分擦出来，使其恢复清晰。**擦拭人物边缘时需格外仔细，避免留下生硬的清晰轮廓**。
4.  保存图片。
5.  将图片导入VSCO等滤镜软件，选择适合的滤镜（如公路照片适合蓝绿色调），并微调色温、饱和度和添加暗角。
6.  最后用美图秀秀进行裁剪，**注意裁剪时避免从人物关节处切断**，应选择小腿等部位。调整比例后保存。
