# 情商成长课程 01：认识情商 🧠

在本节课中，我们将要学习情商的基本概念、其重要性以及核心构成要素。理解情商是提升个人社交能力和情绪管理的第一步。

## 什么是情商？

情商，全称为情绪智力，是指个体识别、理解、管理自己和他人情绪，并利用这些信息来指导思维和行为的能力。它与智商不同，更多关注于情绪层面的智慧。

## 情商为何重要？

上一节我们介绍了情商的定义，本节中我们来看看它为何如此关键。高情商能帮助人们更好地应对压力、建立和谐的人际关系，并在工作和生活中做出更明智的决策。它是个人成功与幸福的重要基石。

## 情商的四大核心要素

情商的构成可以分解为四个相互关联的部分。以下是这四个核心要素的详细说明：

1.  **自我觉察**
    *   指识别和理解自身情绪的能力。例如，能明确感知到“我现在感到焦虑”。

2.  **自我管理**
    *   指基于自我觉察，调节和控制自身情绪与行为的能力。其目标不是压抑情绪，而是以建设性的方式表达。公式可表示为：**有效应对 = 识别情绪 + 选择反应**。

3.  **社会觉察**
    *   指感知、理解并回应他人情绪的能力，通常也称为同理心。这包括准确解读他人的非语言信号。

4.  **关系管理**
    *   指运用对自我及他人情绪的觉察，来有效处理互动、解决冲突并建立积极关系的能力。这涉及沟通、影响力与合作技巧。

---

本节课中我们一起学习了情商的基本定义，探讨了其重要性，并详细解析了构成情商的四大核心要素：自我觉察、自我管理、社会觉察和关系管理。理解这些基础是后续所有情商提升练习的起点。
