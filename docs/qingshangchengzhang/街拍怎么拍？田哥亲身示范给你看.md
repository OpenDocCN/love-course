# 街拍人像摄影教程 #01：从姿势、构图到后期的一站式指南 📸

在本教程中，我们将系统性地学习如何在公共或私密场合拍摄出具有“大片感”的人像照片。课程将涵盖从前期姿势摆拍、构图技巧，到后期修图的全流程，旨在帮助你摆脱“游客照”的尴尬，提升个人照片的表现力。

---

## 一、 前期准备与环境选择 🌳

上一节我们概述了课程目标，本节中我们来看看拍摄前需要做哪些准备。

许多人在户外拍照时，常常只是呆呆地站着，拍出的照片显得很“游客”。要拍出有质感的照片，需要专门的拍摄方法。本次示范地点在成都太古里，这是一个环境与装扮都不错的商圈。你可以在自己的城市找到类似的环境。

选择拍摄地点时，需注意避开杂乱的人群，以确保照片背景干净。例如，我选择了一个安静、无人但环境很好的通道。这里光线幽暗，泛着蓝光，能营造出我想要的氛围。

## 二、 着装与全身照拍摄要点 👕

在选择了合适的环境后，人物的着装和拍摄技巧同样关键。

着装方面，夏季男生可以穿着干净整洁。例如，我选择了一件白色T恤和一条黑色收脚裤（ZARA有售），搭配一双篮球鞋。这样的搭配简洁阳光。

拍摄全身照时，有以下几个核心要点：

1.  **构图底线**：相框底部千万不能切到脚，否则会有被“肢解”的感觉。同时，脚下留空也不能太多，否则会显矮。
2.  **最佳机位**：如果脚在画面底部，相框下边缘应刚好在脚底前方一点的位置。公式可以表示为：`画面下边缘 ≈ 脚底位置 - 微小偏移`。
3.  **姿势技巧**：可以尝试仰头看天的姿势，这样会显得身高更高。
4.  **视线管理**：拍摄文艺风格的照片时，眼睛尽量少看镜头。可以看左、右、上、下或后方。视线避开镜头能营造第三人称视角，让观看者感觉是在观察你的生活，而不是与照片中人对视。

以下是拍摄时的操作步骤：
*   让摄影师蹲下，采用低角度拍摄。
*   人物摆出仰头姿势。
*   摄影师确保构图符合上述要点，然后拍摄。

这样拍出的照片，人物显得高大，背景氛围感强。如果个子不高，可以穿高帮鞋和收脚裤来拉长腿部线条。

## 三、 后期修图实战：VSCO 调色 🎨

上一节我们介绍了前期拍摄的核心技巧，本节中我们来看看如何通过后期修图让照片质感更上一层楼。

在拍摄的多张照片中，选择一张最满意的进行后期处理。我将使用 VSCO 软件进行调色，目标是让照片看起来自然、干净。

以下是具体的调色步骤：

1.  **打开图片**：在 VSCO 中导入需要处理的照片。
2.  **调整对比度**：先将对比度增加到 **+2**，让颜色更鲜明。
3.  **调整清晰度与锐化**：将清晰度调到 **+2**（不宜过高，否则影响皮肤质感），并适当锐化，增加图片清晰度。
4.  **调整饱和度**：将饱和度向**左**（降低）微调 **2** 个单位，让照片色彩偏向素雅风格。
5.  **调整色温**：将色温向**左**（偏蓝）调整 **2** 个单位，强化场景的冷色调氛围。
6.  **微调色调与肤色**：将色调稍微向紫红色方向调整一点，让皮肤看起来更好。将肤色调整向红色方向微调，使皮肤呈现红润感。
7.  **添加暗角**：为照片添加暗角，可以突出画面中央的人物主体。
8.  **保存图片**：完成调整后，保存至相册。

通过以上调整，原图中“很黄、人很黄”的感觉被消除，取而代之的是“清晰、人很白、明亮、干净”的最终效果。代码式的调整思路可归纳为：
```python
# VSCO 调色参数示例 (非实际代码，仅为逻辑示意)
filter = “无”
contrast += 2
clarity += 2
saturation -= 2
temperature -= 2
tint += 0.5 (towards magenta)
skin_tone += 0.5 (towards red)
vignette = ON
```

## 四、 背面拍摄与透视矫正技巧 🏞️

刚才我们演示了如何拍摄和修饰正面照片，现在我们来学习如何拍摄背面照。

拍摄背面与正面一样，需要根据环境选择角度、站位和动作。例如，面对一片池塘（或大海）时，更好的选择是人物背对水面，让摄影师拍摄人物以及背后的全景。

以下是背面拍摄的要点：

1.  **站位**：可以站在场景的 V 字形角落或中央。
2.  **姿势**：切忌驼背或东倒西歪。可以尝试一只脚在前，一只脚在后，或者垫起脚尖。
3.  **拍摄角度**：摄影师同样需要蹲下，采用低角度拍摄。背面照更容易显人宽、显胖，因此拍摄时要有意识地将人物拍得修长一些。
4.  **构图**：可以尝试横构图拍摄全景，但要注意这样可能会让人物显得略扁。

拍摄完成后，可能会发现人物在画面中显得较胖或较矮。这可以通过后期软件进行矫正。

## 五、 后期修图实战：FaceTune 身形修饰 ✨

在拍摄背面或某些角度时，照片中的人物身形可能不尽如人意，本节我们使用 FaceTune 软件进行精细化调整。

我们使用 FaceTune 中的“重塑”功能来调整人物身形。

以下是操作步骤：

1.  **打开图片**：在 FaceTune 中导入需要修饰的照片。
2.  **使用“重塑”工具**：选择“重塑”功能（图标通常为曲线箭头）。
3.  **推动轮廓**：在人物身体边缘（如腰部、背部）向内轻轻推动，使轮廓看起来更瘦。**公式化理解**：`向画面中心方向的推力 → 使该区域变瘦`。
4.  **注意比例**：调整时要保持自然，避免过度修饰导致失真。
5.  **处理背景**：如果背景中有无关人物或杂物，可以使用“修补”工具进行处理。

通过“重塑”工具的向内推挤，可以让人物在全景中显得更修长、更瘦，同时保持背景的完整性。

## 课程总结 📝

本节课中，我们一起学习了街拍人像摄影的完整流程：

1.  **前期构思**：选择干净、有氛围的拍摄环境，注意着装搭配。
2.  **拍摄技巧**：
    *   **全身照**：注意构图底线，采用低角度拍摄，利用姿势拉长身高，管理视线方向。
    *   **背面照**：背对美景，注意挺直身姿，同样采用低角度拍摄以优化比例。
3.  **后期修图**：
    *   **VSCO 用于调色**：通过调整对比度、饱和度、色温等参数，让照片色彩干净、高级。
    *   **FaceTune 用于精修**：利用“重塑”功能调整身形，获得更理想的视觉效果。

记住，自然感和氛围感是拍出好照片的关键。多练习这些技巧，你也能在朋友圈晒出令人称赞的“大片”。
