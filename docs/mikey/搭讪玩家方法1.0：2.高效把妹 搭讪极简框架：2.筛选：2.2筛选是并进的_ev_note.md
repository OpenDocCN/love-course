# 搭讪玩家方法1.0 课程2：高效把妹 - 搭讪极简框架之筛选（2.2）：筛选是并进的

在本节课中，我们将要学习筛选过程中的一个核心原则：筛选的三个关键点——服从度、社交动态和资格赋予——是**同时并进**的，而非按顺序执行。理解这一点是达成高效互动的关键。

上一节我们介绍了筛选的基本概念，本节中我们来看看筛选在实战中是如何具体运作的。

## 🎯 筛选的三个核心点

筛选过程主要围绕以下三个核心点展开：
1.  **对方对我的服从度**
2.  **当时的社交动态**
3.  **我对对方赋予的资格**

这三个点共同构成了我们判断互动进程和对方兴趣度的坐标。

## 🔄 筛选是同时进行的

那么，这三个点之间存在先后顺序吗？答案是：**没有**。

这三个点是**同时进行**的。你可以在筛选对方社交动态的同时，也完成了对服从度的筛选。

以下是具体说明：

例如，当我可以自然地触碰到女方的手时，我可能会同时问她：“你今天和谁一起来的呀？” 如果她回答“我一个人来的”，我就可以接着回应：“哦，你一个人来的啊，挺好。” 或者更进一步：“我觉得跟你讲话好舒服，你挺会聊天的。”

**关键点在于**：在整个对话过程中，我的手可能依然握着她的手。这就是高效互动的体现——**三项筛选同时进行**。

它没有一个固定的流程顺序，三点是并行的。经过这次讲解，你再回去观看我的实战约会案例，无论是现场约会还是即时约会，你会发现我开场成功后，全程都在进行这种复合式筛选。

我一直在同步筛选服从度、社交动态以及我对她的资格赋予。当我发现所有这些条件都得到满足后，互动就会非常顺畅，仿佛可以自然地牵着手进入下一个阶段。

## 💡 并进筛选带来的优势

这种并进筛选的方式会带来两个显著效果：
1.  **展现真实性并传递价值**：互动流畅自然，不显刻意。
2.  **拥有明确路线图**：我知道每一步的目标和方向，这使得整个互动过程特别高效。

你现在如果再去看我的外部课程案例，就能看懂其中贯穿的这套逻辑了。
