# 搭讪玩家方法1.0 课程2：高效把妹之搭讪极简框架 🎯

## 第三部分：收尾

### 3.1 收尾的核心：保持一致性

在本节课中，我们将要学习搭讪流程中至关重要的收尾环节。收尾的核心原则是**保持一致性**。

保持一致性极其重要。有些人在收尾过程中，因为觉得自己即将成功，认为马上就能“拿下”这个女人，心态便开始发生变化。

这种心态会导致行为飘忽，让人分不清自我。有些人甚至会志在必得，提前在社交群里炫耀。例如，声称“今晚就能拿下这个女人，以后看我怎么对她”。这种行为非常不好。

你的浮夸表现会让女人察觉到异常。她会觉得“怎么和刚才那个人不一样了？”，然后你就会被拒绝。因此，必须稳扎稳打。

---

### 收尾的关键判断

这里需要讲到一个关键点：**非必要，不收号**。

我们的筛选逻辑是这样的：我们可以开场后直接进入收尾环节，而收尾就是收联系方式。

我在此补充说明一下，我们的A计划收尾信号，可以想象成“灭霸打一个响指”。这个响指能将你们传送到一个私密空间。如果在那里你们发生关系，对方不会大声说“不”，那就代表你可以进行收尾了。

很多人会困惑：“我不知道能不能收尾，我觉得女生挺喜欢我，我们相处也挺舒适。” 此时，你需要扪心自问一个问题：如果“灭霸打一个响指”，把我们传送到酒店的私密空间，我们要发生关系，女生会不会大声拒绝？如果她会大声拒绝，那就不要带她收尾。

---

### 收尾的实践话术

在实践收尾时，话术要自然流畅。例如：

“怎么会尴尬呢？来，主要是看看……这里我们走吧，我们一起玩吧。Let‘s go。”

“就我这身高能让你尴尬呀？哎，让我看看你的腰臀比……嗯，你的身材真的很不错。”

---

### 核心思维：换位思考

收尾时，必须站在女生的角度思考问题。不是站在当前社交环境的动态下思考，而是**从根本上思考这个问题**。

你需要判断的是关系发展的可能性，而不是当下的即时反应。

---

### 本节课总结

本节课中，我们一起学习了搭讪收尾环节的核心要点。

我们首先强调了**保持一致性**的极端重要性，指出心态膨胀和提前炫耀会导致失败。

接着，我们介绍了收尾的关键判断标准：**非必要，不收号**，并借助“灭霸响指”的比喻来形象说明何时可以安全收尾。

最后，我们探讨了收尾的实践话术，并总结了最核心的思维——**站在女生的角度进行根本性的换位思考**。

记住，稳扎稳打，保持一致性，是成功收尾的不二法门。
