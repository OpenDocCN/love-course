## 核心原则：印象优于遗忘

上一节我们介绍了课程目标，本节中我们来看看第一个核心原则。让女生对你产生印象，即便是负面的印象，也好过她完全不记得你是谁。

## 第一步：创造“无害”的接触机会

理解了印象的重要性后，我们进入实践环节。第一步的关键是创造频繁的接触机会，即使这些机会起初可能显得有点“笨拙”。

以下是创造接触机会的要点：
*   **频繁出现**：重要的是频繁地出现在对方面前。
*   **无伤大雅**：行为可以是“憨憨蠢蠢”的，例如影片中在地上画爱心这类举动。
*   **接受初期负面反馈**：初期对方可能会表现出不耐烦或责备，这是正常过程。

## 第二步：处理情绪爆发与“清空”

当你的频繁出现积累到一定程度，对方可能会产生情绪爆发。此时，同事或朋友甚至可能帮你“打助攻”，衬托出你的执着。

**关键操作**：等待对方将积累的负面情绪完全爆发出来。不要认为这是最终拒绝，实际上，这恰恰是关系转折的最佳时机。等她情绪宣泄完毕，心理上就完成了一次“清空”。

在对方情绪清空后，下一次相遇时，便是你操作的关键时刻。这个策略的核心是坦诚与反差。

2.  **展示笨拙的真诚**：“我不知道用什么方法让你对我产生好感，所以我只能用那些愚蠢的办法。”
3.  **突然放弃（纵）**：“我跟你说对不起，从今天开始，我再也不会骚扰你，我保证，再见。”

**心理原理**：你的离开制造了情感空缺。当你再次出现时，你就成了她情绪上的“救命稻草”。此时，便是升级关系、进行表白的最佳时机。
