## 场景引入与话题应对 🍽️

*   **顺着对方聊**：容易陷入一味迎合的境地。
*   **引导至自身熟悉领域**：郭靖采用了此方式，将话题从餐馆酒菜转向蒙古的“马奶酒”。

## 核心策略：话题引导与价值展示 🌄

上一节我们看到了郭靖如何转换话题，本节中我们来看看他如何深化这一策略并展示自身特质。黄蓉被勾起兴趣后，推测马奶酒可能更好，并评价当前的酒“充满了铜臭味儿”。此时，对话进入了一个关键转折点。

黄蓉提出了一个测试性问题：“是不是不管我喜欢什么，你都会给我啊？”郭靖的回答是：“当然了，你是我来到中土的第一个朋友嘛。”

以下是郭靖回应的核心要点：
*   **肯定回答**：直接且无条件地答应。
*   **强调关系定位**：将对方定义为“第一个朋友”，赋予了行动以情感基础。

随后黄蓉进一步测试：“如果我要你身上所有的银子呢？”郭靖的回应是：“没问题啊，拿去吧。”并且，当黄蓉问他为何不问要钱的理由时，郭靖说：“既然你要自然有理由了，你不说我何必问呢。”

这体现了郭靖沟通艺术中的一个关键公式：
**真诚付出 + 不加评判 = 建立深度信任**

## 策略总结与要点提炼 ✅

本节课中我们一起学习了郭靖与黄蓉对话中蕴含的沟通艺术。我们来总结一下其中的核心要点。
