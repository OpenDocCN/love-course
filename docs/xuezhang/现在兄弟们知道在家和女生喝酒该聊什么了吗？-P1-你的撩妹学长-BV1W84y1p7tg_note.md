## 核心技巧：合理化你的行为

当你说出一个令人惊讶的理由后，对方很可能会质疑。这时，需要为你的行为或状态提供一个合理的解释，避免被看作是在故意搭讪或说谎。

以下是合理化解释的关键点：

*   **将原因外部化**：例如，将“去酒吧”的行为解释为“那是我的心理医生让我去的”，把责任转移到第三方（心理医生）身上。

*   **引入专业术语**：使用如“急性焦虑症”这类听起来专业的词汇，能增加可信度。公式可以概括为：**令人惊讶的行为/状态 + 外部权威原因 + 专业术语包装 = 合理化解释**。
