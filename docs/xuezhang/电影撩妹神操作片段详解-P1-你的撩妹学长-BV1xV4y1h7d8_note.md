## 核心生物学原理 🧬

以下是这些动作的一个关键特征：
*   它们会发出低沉的声音。

人类作为高级动物，也会遵循这样的行为模式。在影片中，女主角低声细语地说了一句话。这是一个非常强烈的信号。
