# 电影里的撩妹片段-P1：studio_video_16

## 片段回顾与核心原理

上一节我们介绍了这个互动场景的概况，本节中我们来看看其背后的核心原理。整个魔术的关键在于一个预设的数学运算。

**核心公式**：
```
（你的手机号 + 0） * X = 你的手机号
```
实际上，更准确的描述是：你提前在计算器中输入了自己的手机号码，然后进行 `+ 0 *` 的操作。此后，无论对方输入什么数字，最终结果都会显示为你最初输入的手机号。

以下是这个公式在计算器上的操作步骤分解：

1.  输入你自己的手机号码（例如：`13800138000`）。
2.  按下 `+` 键。
3.  输入 `0`。
4.  按下 `*` 键。
5.  此时，计算器处于“等待下一个乘数”的状态。无论对方随后输入任何数字，计算结果都将是你的手机号。

## 互动流程拆解

理解了核心原理后，我们来看看如何将这个数学技巧融入一个完整的互动流程中。以下是引导对方完成互动的关键步骤：

首先，你需要创造一个轻松的游戏氛围，引导对方参与。

*   **发起游戏邀请**：提出玩一个数字游戏，降低对方的戒备心。
*   **引导输入随机数**：请对方随意输入一个四位数，然后一个五位数，生成一串“无意义的数字”。
*   **插入个人信息提问**：自然地提问一个关于对方的真实问题（例如：“你是几岁来到上海的？”），并将答案（如“18”）加在那串“无意义数字”的前面。
*   **完成“魔术”拨打**：让对方拨打这串组合后的长数字。由于你预先设置的计算器原理，拨出的号码实际上就是你的手机号。
