# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 什么是变量？📦

变量可以理解为一个存放数据的“盒子”或“标签”。在程序中，我们使用变量来存储信息，以便后续使用和修改。

例如，我们可以创建一个名为 `message` 的变量，用来存储一段文本信息。其基本语法是使用等号 `=` 进行赋值。

```python
message = "Hello, World!"
```

上一节我们介绍了变量的基本概念，本节中我们来看看Python中常见的数据类型。

## 基本数据类型 🔢

数据类型决定了变量可以存储什么种类的数据，以及可以对它进行哪些操作。Python有几个内置的基本数据类型。

以下是三种最常用的基本数据类型：

*   **整数**：表示没有小数部分的数字，例如 `10`、`-5`、`0`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`、`2.0`、`-0.5`。
*   **字符串**：表示文本信息，需要用单引号 `'` 或双引号 `"` 括起来，例如 `"Python"`、`'学习'`。

了解了基本类型后，我们来看看如何在实际中使用变量和这些类型。

## 使用变量与类型检查 🛠️

我们可以将数据赋值给变量，并通过变量名来使用这些数据。同时，可以使用 `type()` 函数来检查一个变量的数据类型。

以下是使用变量和检查类型的示例：

```python
# 将数字 100 赋值给变量 score
score = 100
print(score) # 输出：100

# 将文本赋值给变量 name
name = "小明"
print(name) # 输出：小明

# 检查变量 score 的数据类型
print(type(score)) # 输出：<class 'int'>

# 检查变量 name 的数据类型
print(type(name)) # 输出：<class 'str'>
```

在编程中，我们经常需要改变变量的值，或者将不同的变量组合起来使用。

## 变量的动态性与运算 ➕

Python中的变量是动态的，这意味着你可以随时改变一个变量存储的数据及其类型。此外，你可以对变量进行各种运算。

以下是关于变量操作的关键点：

*   变量可以重新赋值，甚至可以改变为不同的数据类型。
*   数字类型（整数和浮点数）之间可以进行数学运算，如加 `+`、减 `-`、乘 `*`、除 `/`。
*   字符串之间可以使用加号 `+` 进行连接。

```python
# 变量重新赋值
age = 18
age = 19 # age 的值变为 19

# 改变变量类型
value = 100     # value 是整数
value = "一百"  # 现在 value 变成了字符串

# 数字运算
a = 10
b = 3
sum_result = a + b  # 加法，结果为 13
product = a * b     # 乘法，结果为 30

# 字符串连接
greeting = "你好，"
user = "同学"
full_message = greeting + user # 结果为 "你好，同学"
```

本节课中我们一起学习了Python编程的基石：变量与数据类型。我们知道了变量是存储数据的容器，并认识了整数、浮点数和字符串这三种基本类型。我们还学会了如何给变量赋值、检查其类型、进行重新赋值以及基本的运算操作。掌握这些内容是开启后续所有编程学习的大门。
