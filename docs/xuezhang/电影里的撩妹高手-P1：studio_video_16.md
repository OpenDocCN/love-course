# 电影里的撩妹高手-P1：studio_video_16

在本节课中，我们将学习如何巧妙应对女生询问前任的棘手问题。通过分析一段电影对话，我们将拆解其中的核心技巧，帮助你掌握保持框架、转移问题的沟通策略。

## 概述

当女生问起你的前任时，常见的回答方式往往效果不佳。直接说“忘了”显得敷衍，说“没有前女友”可能显得不真实，而认真讲述又可能引发不必要的比较或情绪。本节教程将解析一种更有效的回应方法。

## 核心原则：前任即陌生人

上一节我们概述了问题的普遍性，本节中我们来看看回应的核心心态。关键在于树立一个清晰的态度：分手后，前任就是陌生人。

**核心态度公式**：`前任 = 陌生人`

这意味着对前任不怀念、不想念，她/他在你的人生中留不下任何痕迹。这种坦荡的态度本身就能传递出你的成熟和果断。

## 应对策略：破解“废物测试”

当你表现出上述坦荡态度时，女生可能会抛出所谓的“废物测试”，例如质疑你“这么冷酷啊”。这是一种测试你内心是否坚定、框架是否稳固的常见方式。

以下是应对此类测试的关键步骤：

1.  **识别测试**：首先意识到对方的评论（如“冷酷”）可能是一个测试，而非真正的批评。
2.  **拒绝辩解**：不要落入为自己“冷酷”态度辩解的陷阱。
3.  **重构选择**：将对方提出的单一负面选项，重构为一个二选一的选择题。

看影片中男主的示范回答：“要么冷酷，要么对不起你，你选一个。” 这个回答的精髓在于，它没有在对方设定的“你是否冷酷”的框架内回答，而是通过给出另一个选项（“对不起你”），巧妙地将问题抛回给对方。

**代码描述核心逻辑**：
```python
# 错误示范：在对方框架内回答
if question == “你这么冷酷啊”:
    response = “我不冷酷…” # 进入辩解模式

# 正确示范：跳出框架，重构选择
if question == “你这么冷酷啊”:
    response = “要么冷酷，要么对不起你，你选一个。” # 转移问题，保持主动
```

## 技巧总结：保持框架

上一部分我们拆解了具体话术，现在来总结背后的核心技巧——保持框架。这意味着始终把握互动的主动权，不被对方的问题带偏节奏。

“把问题再抛回给女生”是保持框架的直观体现。当对方感到你赢得了这次言语交锋（“行，你赢了”），恰恰证明你的框架足够稳固。每天学习并练习这种思维一分钟，都能有效提升你的沟通能力。
