# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 什么是变量？📦

上一节我们介绍了本课程的主题，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们使用等号 `=` 来给变量赋值。

其核心操作是赋值，公式如下：
**`变量名 = 值`**

例如，我们创建一个名为 `message` 的变量，并存入一段文字：
```python
message = "你好，世界！"
```
执行这行代码后，字符串“你好，世界！”就被存储在了名为 `message` 的“盒子”里。之后我们可以通过 `message` 这个名字来使用这个值。

## 基本数据类型 🔢

理解了如何存放数据后，我们需要知道Python中常见的数据类型。数据类型决定了数据可以进行的操作。

以下是Python中最常用的三种基本数据类型：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由字符组成的文本，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Python’`, `“学习”`。

## 如何查看和使用类型？ 🔍

在编程时，我们有时需要确认一个变量的数据类型。Python提供了一个内置函数 `type()` 来帮助我们。

其使用格式为代码：
**`type(变量名)`**

例如：
```python
age = 25
print(type(age))  # 输出：<class ‘int‘>
name = “小明”
print(type(name)) # 输出：<class ‘str‘>
```
`print()` 函数会将结果打印在屏幕上，这样我们就能看到 `age` 是整数类型，`name` 是字符串类型。

## 一个简单的实践示例 ✨

现在，让我们将以上知识点结合起来，完成一个简单的程序。

以下是创建变量、赋值并打印信息的步骤：

1.  创建一个存储你名字的字符串变量。
2.  创建一个存储你年龄的整数变量。
3.  使用 `print` 函数将它们一起输出。

参考代码如下：
```python
my_name = “张三”
my_age = 20
print(“我的名字是：”, my_name)
print(“我的年龄是：”, my_age)
```
运行这段代码，你将在屏幕上看到两条介绍自己的信息。
