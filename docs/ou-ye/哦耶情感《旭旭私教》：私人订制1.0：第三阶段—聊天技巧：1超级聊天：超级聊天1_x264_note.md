## 概述：聊天的根本作用

聊天的作用是什么？这是一个非常关键的问题。聊天的作用是交流、拉伸关系、邀约、引起舒适感。语言是表达内心的工具，它本身并不代表任何东西。真正起作用的是你通过语言所传递出去的信息。

很多人试图通过话术本身去吸引女生，这是错误的。女生不会因为你说了什么而喜欢上你，但会因为你说的话而感觉你是一个怎样的人。例如，女人不会因为你说脏话而讨厌你，但会因为你经常说脏话，觉得你是一个没素质的人而讨厌你。所以，语言是用来体现你到底是一个怎样的人的。

从技术层面来讲，聊天有三大作用：**吸引、拉伸、邀约**。你的聊天只有这三步。第一步要吸引她，第二步是拉伸关系，让关系突破朋友变得非比寻常，最后把她约出来。

如果你的聊天没有达到这三个作用中的任何一个，那么你说的话可能就是无效的。例如，你一上来就查户口或者说“今天天气不错”，这些话既没有吸引，也没有拉伸关系，更没有邀约，因此是无效的。

这三步的顺序不能颠倒。你必须先吸引，再拉伸，最后邀约。如果你一上来就说“我喜欢你”，在没有吸引的情况下拉伸关系，女生是不会接受的。女生只会接受她喜欢的人跟她拉伸关系。

## 吸引力的核心：态度与行为

上一节我们介绍了聊天的三大作用，本节中我们来看看如何产生吸引力。吸引力的产生并不取决于你说了什么内容，而取决于你说话时的**态度**和**行为**。

语言的内容本身并不产生吸引，语言的**态度**和**行为**才产生吸引。例如，你跟一个女生说：“我很牛逼，我昨天刚提了一辆保时捷911。”虽然语言内容价值很高，但你的行为是在证明自己，态度是自卑的，这反而会出卖你。

同样，给女生做饭这个行为，用不同的态度说出来，效果截然不同：
*   **低吸引力方式**：“你哪天有空我給你做飯吃，我做飯可好吃了呢，相信我。”（态度讨好，行为证明自己）
*   **高吸引力方式**：“這幾天忙完了，可以給你做飯吃。”（态度从容，行为是给予）

你说什么不重要，重要的是你怎么说。你展示价值的行为和态度才是关键。

以下是体现吸引力、需要你时刻保持的四大核心态度：

### 1. 嚴謹一該 (言简意赅)
记住，当你把所有话术都变短时，你就会显得很酷。话少不代表敷衍，敷衍是一种态度，而简短、酷是另一种态度。
*   **公式**：`回复 = 简短的核心词 + 酷的态度`
*   **示例**：
    *   女生问：“你愛不愛我？” 回答：“愛。”
    *   邀约：“週末我空，看電影啊。”

### 2. 高標準
女生会通过你标准的高低，判断你是否是一个优质男人。你的择偶标准一定要高，这体现了你的价值和稀缺性。
*   **示例**：

    *   **高标准（正确）回答**：“我是標準那麼低的人嗎？” 或 “一般人我都不會理的。”

### 3. 獨立的觀念
当你没有独立观念，只懂得附和对方时，你就会显得没有魅力。即使和女生的观点不一致，也要保持自己的观点，同时尊重对方。
*   **核心**：恋爱不是寻找完全合适的人，而是互相磨合。重点是谁能降服谁，吸引力是关键。不要为了迎合而放弃自己的原则。

### 4. 自尊自重
你不会为了讨好对方，去做一些自己不愿意做的事情，不会破坏自身的原则。
*   **示例**：女生以分手要挟你买包。
    *   **错误做法**：立刻妥协去买。这会让女生把你当成ATM，失去吸引力。
    *   **正确做法**：当时保持态度，拒绝威胁。之后可以在自己心情好或特殊日子时，以“我願意給你”的姿态送给她。这既体现了爱，又保持了自尊。

## 聊天实战：从开场到深入

了解了吸引力的原理后，我们来看看具体的聊天步骤。学习聊天的第一步就是要明白：**不是什么问题都能通过聊天解决**。聊天的前提是双方基本匹配。如果对方一开始就极度冷淡，可能连入场的机会都没有。

### 开场测试
开场永远先发一个表情或一个“嗨”。目的是测试对方的反应。
*   **反应好**（也回表情或“嗨”）：可以继续聊。
*   **反应差**（回“？”、“哦”、“有事吗”）：可能没有入场机会，不必强求。

### 一次追逐原则
如果女生反应冷淡，你只能再多追逐一次。如果第二次反应依然冷淡，必须切断对话，否则就变成跪舔。
*   **示例**：你说“周末有空吗，来我家吃饭啊？”，她回“为啥？”（冷淡）。
    *   你可以再追一句：“你怎麼這麼高冷呀？”
    *   如果她回复变好，继续聊；如果还是冷淡，就不要再回复。

### 激活右脑：凭空寻找话题
很多人找不到话题，是因为在用左脑“思考”该说什么，越想越空。正确的方法是激活右脑“看到”什么就说什么。
*   **方法**：打开女生的朋友圈或照片，看到什么就直接说出来。例如：皮肤很白、眼睛很大、项链很好看、定位在成都、喜欢旅游等等。周围的一切都可以是话题。

### 修炼左脑：把话说得好听
右脑负责找到话题，左脑负责用积累的词汇把话题说得有趣、好听。
*   **方法**：平时多积累形容词和表达方式。
*   **积累渠道**：
    1.  林夕的歌词
    2.  冯唐的诗
    3.  网易云音乐的热门评论
    4.  方文山的歌词
    5.  《奇葩说》、《吐槽大会》等节目
    6.  经典爱情电影的台词
*   **示例**：看到女生单纯，不要说“你真单纯”，可以说“你單純得像一張白紙”。

### 吸引力产生的三步流程
在聊天中，通过语言吸引女生有三个步骤：**引起好奇 -> 发现亮点 -> 保持神秘**。
对应的三个技巧是：**钩子 -> 展示 -> 释放**。

1.  **抛钩子**：将话题自然引导到你想展示的亮点领域。
    *   **示例**（想展示健身）：“好餓呀，但是又不能吃。”（引发好奇）
2.  **展示**：当女生好奇发问时，自然展示你的亮点。
    *   **示例**：女生问“為什麼？”，你回答“因為我在健身呢/要保持六塊腹肌呀。”
3.  **释放**：展示后不要穷追猛打，要保留神秘感。
    *   **示例**：女生问“哇，你還有腹肌哦？”，你回答“以後有機會給你看。”然后转移话题。

正经聊让话题深入，有趣聊让氛围开心。

由此，我们可以划分出聊天的四个象限，即“聊天四项线”：

| | **聊自己** | **聊女生** |

## 高阶技巧：抽离与测试

聊天不只靠语言。再会聊，如果始终是你主动，关系也不平衡。你需要用“抽离”技巧，让对方也开始主动。

### 什么是抽离？
聊着聊着，突然不回消息，或者延迟回复。目的是打破对方的预期，制造情绪波动，让关系趋于平衡。

### 抽离的方法与测试反应
1.  **突然不回**：在聊得不错时，尤其是对方提出要求时（如“我想看腹肌”），选择不回复。隔一段时间再联系。
2.  **拖延回复**：不定时地延长回复时间，观察对方反应。
3.  **发送无效信息**：回复“嗯”、“哈哈”或一个表情，观察对方是否会主动找新话题。
4.  **推开式回复**：发送“汗”的表情，或轻微否定对方（如“不是這樣的”），观察反应。

### 如何判断反应好坏？
*   **好反应**（说明对方在意你）：
    *   主动追问：“人呢？怎麼不回我了？”
    *   秒回你的消息。
    *   质问你：“你剛才幹嘛去了？”
    *   即使回复冷淡（如“嗯”），但**只要回复了**，就是好现象。
    *   在你发送无效信息后，主动找新话题。
*   **普通反应**：也延迟回复。
*   **无反应**：不回复。

**核心原理**：只要对方有反应（哪怕是质问你），都说明你引起了她的情绪波动，你在关系中占据了主动。即使对方没反应，你也通过“先抽离”保持了姿态平等，没有陷入被动追逐。
