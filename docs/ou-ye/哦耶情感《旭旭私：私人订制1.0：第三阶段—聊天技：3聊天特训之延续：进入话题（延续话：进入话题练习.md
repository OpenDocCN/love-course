# 哦耶情感《旭旭私教》1.0：第三阶段—聊天技巧（3）💬 聊天特训之延续话题：进入话题练习讲解

在本节课中，我们将要学习如何有效地“进入话题”，即在成功开启对话后，如何延续和深化话题，避免聊天陷入僵局。我们将通过具体示例分析核心要点，并指出常见的错误。

上一节我们介绍了如何开启话题，本节中我们来看看如何延续一个已经开始的话题。

假设我们已经成功打开了话题，现在进行最后一项练习：进入话题。例如，女生说：“哈哈哈，你也喜欢健身吗？”请问应该如何回复？

开场练习已经完成，现在你们找到了一个与健身相关的话题。那么，如何利用女生“你也喜欢健身吗？”这个问题，将话题延伸开来？请快速回答。还记得核心点吗？

核心点是：**不要询问，不要说屌丝的话**。同时，必须将话题延伸开，提供越来越多的信息。你的回复需要包含足够的信息点，才能让女生有回复的欲望。

以下是几个回复示例的分析：

*   “你这个是不是想和我一起啊？”——这个回答很好。重点在于它**与两个人都有关联**。
*   “还好了，只不过身负八块腹肌而已。”——这个回答有幽默感和自信，但**没有与女生产生关联**。女生可能只会回复“哈哈”。
*   “还好啦只不过八块腹肌而已，但是呢还是没有你身材好。”——这样修改后，**既展示了自己，又与对方有关**，对方会更愿意回复。
*   “哎一个人健身真枯燥，缺一个你这样的小可爱陪伴。”——这也是一个**同时关联双方的优秀范例**。

其他几个回复都很棒。所以，你们在“进入话题”这一环节的表现比较强。

但我发现了你们的弱点。你们的弱点在于开场阶段：找话题过于刻意，不够自信，缺乏奖品性心态。并且，开场时常犯一些漏洞式的错误。

以下是总结出的几个主要问题：

1.  **话题关联性单一**：你的话要么只跟对方有关，要么只跟自己有关，没有一句话能同时把两个人关联上。
2.  **姿态错误**：存在跪舔、询问、收取价值等行为，这些都是低姿态的表现。
3.  **缺乏奖品性**：说话语气过于屌丝，没有展现出自信和吸引力。

所以从今天开始，你们需要持续地去改变自身的这些问题。我已经指出了这几个关键点，你们需要不断利用它们进行练习。
