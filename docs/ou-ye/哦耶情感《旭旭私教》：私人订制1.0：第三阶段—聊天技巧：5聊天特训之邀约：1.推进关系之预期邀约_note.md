在本节课中，我们将要学习一种关键的邀约技巧——**预期邀约**。这是一种在正式见面之前，通过语言为对方营造心理画面和暧昧氛围的方法，旨在降低见面时的尴尬，并提高邀约成功率。

## 什么是预期邀约？

预期邀约，就是你要在跟他见面之前，就给他说一些，我跟你见面之后会发生的事情。这样的话会给他营造出非常强的暧昧感觉和心理预期。

例如，你可以这样发起邀约：
> “周末出来看电影。”

如果对方同意，你可以接着说：
> “我们要坐在最后一排。”

当对方问“为什么要坐在最后一排”时，你可以神秘地回答：
> “嘿嘿你不用管了，我今天要坐在最后一排，最后一排浪漫。”

你会发现，当你说这种话的时候，就会给对方内心营造出一种性张力。这个就是预期，给他一种心理预期，让他知道跟你见面之后会发生什么。

## 预期邀约的作用与原理

让他预想跟你见面之后会发生一些浪漫的事情。例如，你可以这样去跟他说：
> “周末我们一起逛街吧。”

他如果说“好啊”，你可以接着说：
> “手拉手逛街。”

这个时候他就可能就会说“才不要跟你拉手”。不过，就算他嘴上拒绝你，但是他心里已经对你产生了一种预期，就是他能预想到跟你逛街一定会发生这种事情。女生通常比较矜持，嘴上拒绝并不重要，重要的是她心里已经会产生这种性张力的幻想。

预期邀约的另一个作用是让对方提前做好心理准备。如果你说了这种预期，他还愿意跟你见面的话，他是做好了这种准备的。所以这个时候你去拉他的手，他一般拒绝的几率就会比较小。

**核心逻辑公式**：
```
成功预期邀约 = 营造暧昧画面感 + 让对方产生心理预期 + 降低实际互动时的突兀感
```

## 如何设计有效的预期？

以下是设计有效预期时需要遵循的几个要点：

1.  **结合场景，给出画面感**：预期要具体、有画面感。例如，“我们两个手拉手逛街”或“我做饭你在后面搂着我”，这些描述能让对方脑海中产生具体影像。
2.  **注意关系火候**：你们到什么关系，就给什么样的预期。不要过早给出过于亲密或越界的预期，如“周末我们见面去啪啪啪”，这会让对方感到压力并拒绝见面。
3.  **保持轻松调皮**：给一些浪漫的、好玩的、调皮的预期，别人才会觉得跟你聊天很有趣，很有暧昧的感觉。
4.  **可与“假定追逐”结合**：你可以反过来假定是对方在主动。例如，“逛街的时候你不许拉我的手”或“我做饭的时候，你可不许在后面偷偷调戏我”。这种说法增加了互动的趣味性。

## 预期邀约的注意事项

如果有人说你输出了一个预期之后，他就把你给拒绝了，他说“我不跟你见面了”，这说明什么？

说明他可能不喜欢你，或者不接受这种程度的暧昧。他微信上都不接受，何况见面以后。所以这个时候你需要继续回去，通过展示价值和吸引力重新吸引他。

如果他喜欢你，他嘴上都会象征性地矜持，实际上他还是会跟你见面的。
