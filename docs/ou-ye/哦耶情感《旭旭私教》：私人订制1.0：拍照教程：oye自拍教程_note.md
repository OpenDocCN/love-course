# 📸 课程一：机萌相机自拍基础教程

在本节课中，我们将学习如何使用“机萌相机”软件，从软件设置到拍摄手势与构图，完成一张高质量的自拍照。本教程专为初学者设计，步骤清晰，简单易懂。

## 软件下载与基础设置

首先，你需要下载名为“机萌相机”的软件。下载完成后，请打开应用进行初始设置。

以下是必须调整的设置项：

1.  **关闭水印**：点击右上角的设置图标，找到“水印设置”并将其关闭。这能确保你的照片没有水印，显得更专业。
2.  **开启网格线**：在设置中，找到并开启“网格线”功能。这能帮助你在拍摄时更好地构图。
3.  **选择滤镜**：在拍摄界面下方，选择“滤镜”。我们推荐使用“自然”分类下的“水嫩”滤镜。强度一般设置为 **`80`**。如果觉得皮肤效果不自然，可以调低至 **`60`**。

## 美颜功能精细调整

上一节我们介绍了基础设置，本节中我们来看看如何调整美颜参数，以达到自然又得体的效果。

进入“美颜”功能界面。首先，点击每个子项目（如美颜、媒体）下的“重置”按钮，将所有参数恢复为默认值。

对于男生而言，只需调整少数几项即可：

*   **磨皮**：建议将数值设置为 **`60`**。过高的磨皮会显得不真实。
*   **白牙**：此功能能让牙齿更洁白。建议将数值设置为 **`50`**。
*   **眉毛**（位于“美妆”分类中）：选择“平纸黑”款式，浓度根据自身眉毛情况，在 **`20`** 到 **`40`** 之间调整。

**注意**：“媒体”类（如瘦脸、大眼）和“美妆”类的其他项目（如口红、眼影）均无需调整，过度修饰效果反而不好。

## 正确的自拍手势与姿势

调整好相机后，正确的持机手势是拍好照片的关键。

以下是正确的手势步骤：

1.  用大拇指扶住手机左侧。
2.  用食指和中指扶住手机右侧。
3.  让手机的一个角顶在手掌心。
4.  保持手机竖直，甚至略微向外倾斜（有一个俯角），确保手机屏幕与你的面部大致平行。
5.  手臂伸直，将手机置于身体正前方，大约与额头齐平的位置。

**错误示范**：避免将手机举在身体侧面，或让手机屏幕过度向内倾斜对着自己，这样拍出的照片效果不佳。

## 构图技巧与拍摄实践

掌握了手势后，我们来学习如何构图。好的构图能让照片主体更突出。

构图的核心要点如下：

*   **头顶留白**：人物头顶与照片上边缘之间需留有空间，约占画面高度的1%。
*   **背景选择**：选择简洁、干净的背景。注意服装颜色与背景要有区分度，例如穿深色衣服站在浅色背景前。
*   **寻找角度**：微微转动头部，从正面到侧面45度之间，寻找自己最上镜的角度。
*   **使用道具**：如果觉得表情不自然，可以佩戴墨镜来减少尴尬感。
*   **多拍精选**：尝试拍摄20到50张照片，然后从中挑选出最满意的一张。

## 总结与后续

本节课中我们一起学习了使用机萌相机进行自拍的完整流程：从关闭水印、设置滤镜和美颜，到掌握正确的持机手势，最后运用构图技巧进行多角度拍摄。
