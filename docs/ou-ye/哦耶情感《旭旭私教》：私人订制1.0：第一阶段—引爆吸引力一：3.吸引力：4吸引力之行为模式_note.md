# 课程名称：情感私教课 1.0 - 第一阶段：引爆吸引力（三）吸引力之行为模式 🧠💼

在本节课中，我们将要学习吸引力的第三个核心层面：行为模式。我们将探讨行为模式如何体现个人价值，以及它如何影响他人对你的评价。

上一节我们介绍了硬价值和软价值，本节中我们来看看这些价值是如何通过具体行为展现出来的。所有的行为模式，本质上都是你内在硬价值和软价值的外在体现方式。你的行为会直接向他人传递信号。

## 行为模式：价值的体现与评判

行为模式决定了你在他人心中，到底是一个高价值的人还是一个低价值的人。他人会持续根据你的行为模式，在内心为你加分或减分。

为了帮助理解，以下是一个具体案例：

*   我曾有一位厦门的学员，他是一家公司的老板，硬价值很高。
*   他开着一辆不错的车，女性初次见面通常会因他的条件而对他产生好感。
*   但他的情商不高，一见到女孩就喜欢通过自黑和扮小丑来逗乐对方。
*   这种行为模式反而让他显得价值很低。

例如，他有一次见到一个女孩就说：“你看我的脸是不是很大，我的头是不是很大很可爱吧？”这种言行让女孩感到无语，觉得一个堂堂大老板不应如此，从而给他减分。

这个案例说明，他人会先根据初步印象（如硬价值）加分，但后续会根据你的行为模式重新评估。行为模式就是你实践自身价值的“机器”。

## 低价值行为模式示例

以下是常见的、会导致扣分的行为模式，请注意避免：

*   **约会举止不当**：在约会中毛手毛脚、乱动。
*   **眼神与气场不稳**：眼神飘忽不定，气场不稳，显得吊儿郎当。
*   **表现吝啬**：买单时反复向服务员索要优惠或打折。
*   **展示邋遢生活**：在朋友圈总是发布脏乱差的照片，显得生活邋遢。
*   **生活单调乏味**：生活模式是简单的两点一线（如上班、下班、吃饭、睡觉、打麻将），缺乏趣味。

许多人会在无意中做出大量扣分行为。他人（尤其是女性）会根据这些行为模式，来定义你是一个高价值还是低价值的人，这通常被称为“社交辨识”。

## 高价值行为模式培养

如果你想成为一个受欢迎的人，首先需要改变自己的行为模式。

你需要进行自我观察，识别并减少自己的扣分行为。同时，主动培养高价值的行为模式。一个有效的方法是观察那些受欢迎的人，学习他们的行为特点。

以下是受欢迎人群常见的高价值行为模式：

*   **坚定沉稳**：举止坚定、沉稳，不会轻易乱动或不分场合地哈哈大笑。
*   **认真专注**：在工作或做事时非常认真。
*   **自信酷感**：在异性面前保持自信和酷感，而非卑微或低三下四地恳求。
*   **富有主见**：有自己的想法和主见，不会事事都询问对方。
*   **互动自然**：懂得如何调侃对方，也懂得适时赞美。
*   **肢体语言沉稳**：肢体动作特别沉稳，眼神坚定。
*   **平等交流**：与他人进行平等交流，而非处处讨好或恳求。

这些行为模式都能有效地为个人形象加分。
