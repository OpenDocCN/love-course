# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**与**数据类型**。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于在计算机内存中存储信息。每个变量都有一个名字（即标签），我们可以通过这个名字来使用或修改盒子里存放的数据。

在Python中，为一个变量赋值非常简单，使用等号 `=` 即可。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```
例如：
```python
my_name = “旭旭”
age = 25
```
这里，我们创建了两个变量：`my_name` 存放了文本“旭旭”，`age` 存放了数字25。

## 2. 基本数据类型 🔢

仅仅把数据放进“盒子”还不够，我们需要知道盒子里装的是哪种类型的东西。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型介绍：

*   **整数 (int)**： 表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数 (float)**： 表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str)**： 表示文本数据，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `“Hello”`, `‘Python’`。
*   **布尔值 (bool)**： 表示逻辑值，只有两个：`True`（真）和 `False`（假），常用于条件判断。

你可以使用 `type()` 函数来查看任何变量或值的数据类型。
```python
print(type(10))      # 输出：<class ‘int’>
print(type(3.14))    # 输出：<class ‘float’>
print(type(“旭旭”)) # 输出：<class ‘str’>
print(type(True))    # 输出：<class ‘bool’>
```

## 3. 变量的命名规则与操作 ✏️

了解了数据类型后，我们还需要知道如何正确地给变量起名字，以及如何操作它们。

以下是变量命名的关键规则和常见操作：
1.  **命名规则**：
    *   变量名只能包含字母、数字和下划线 `_`。
    *   变量名不能以数字开头。
    *   变量名是区分大小写的（`age` 和 `Age` 是两个不同的变量）。
    *   应使用描述性的名字，例如用 `user_name` 而不是 `a`。

2.  **变量的重新赋值**： 变量的值是可以改变的。当你给一个已存在的变量赋予新值时，旧值会被覆盖。
    ```python
    message = “你好”
    print(message) # 输出：你好
    message = “世界”
    print(message) # 输出：世界
    ```

3.  **简单的运算**： 数字类型的变量可以直接进行数学运算。
    ```python
    a = 5
    b = 2
    print(a + b)  # 加法，输出：7
    print(a * b)  # 乘法，输出：10
    ```
