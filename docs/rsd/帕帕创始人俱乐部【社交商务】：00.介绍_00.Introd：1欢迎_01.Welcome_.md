# 帕帕创始人俱乐部【社交商务】：00.介绍_00.Introduction：1欢迎_01.Welcome 🚀

在本节课中，我们将要学习帕帕创始人俱乐部（RSC Founders Club）的概况与核心内容。这是一个专注于社交商务的会员项目，旨在通过丰富的资源帮助成员成长。

## 课程内容概述 📚

欢迎来到RSC创始人俱乐部。在这个会员项目中，你将能够接触到大量不同的学习资源。

以下是该项目包含的核心内容：

*   60个教学视频。
*   超过600个不同的实践项目。
*   涵盖60个主题。
*   总计10小时的视频内容，每个视频被分割为10分钟以内的片段。

上一节我们介绍了项目的整体资源规模，本节中我们来看看其中更具深度的部分。

## 深度访谈与个人经历分享 🎤

该项目还包括一系列由我主持的深度访谈。访谈对象是我心目中各个领域的成功人士。

这些访谈将带来多角度的洞察：

*   他们在自身行业内的专业能力与见解。
*   他们取得事业成功的经验。
*   他们的个人生活与约会生活经历。
*   他们关于当今社交经济的实际观点与实践经历。
*   我个人从零开始至今的生活与成功故事。

## 项目起源与特色导师 👥

让我们回到一切的起点。该项目始于我的第一个项目，合作者是我最亲密的朋友之一。

他最初是我的一名学生，是一位思维极具独特性的思考者，名叫“博弈论先生”（Mr. Game Theory）。他是一位世界级的冠军，这指的是他的球技。

除此之外，你还有机会向来自RSD的教练学习。在这个项目中，你将听到各种有趣的机会，这些机会专为该项目的成员设计。

## 专属社群与互动期望 🤝

我们为购买此会员项目的成员创建了一个专属社群。

以下是我们的核心期望：

*   希望你们能够积极进行自我沟通与反思。
*   希望你们能够与我保持交流与沟通。
*   希望你们能够在这个项目中，与我们一同探索和成长。

本节课中，我们一起学习了帕帕创始人俱乐部的丰富资源构成，包括视频、项目、深度访谈以及专属社群。该项目旨在从专业知识、成功经验和实践社群等多方面，助力成员在社交商务领域探索与成长。
