# 课程07：瓦伦蒂诺隐型游戏 - 5.成为满分的男人 - 3.价值观 💎

在本节课中，我们将要学习价值观的概念。价值观是我们人生中最重要的部分，它决定了我们的选择、行为和生活的方向。理解并明确自己的核心价值观，是塑造理想人生的关键一步。

## 概述

价值观是我们内心深处认为最重要、最值得追求的事物或情感状态。它像是一个内在的导航系统，指引着我们做出每一个决定。本节课将引导你识别并梳理自己的核心价值观。

## 什么是价值观？

价值观是我们所有人都值得在人生中拥有的东西。我们都值得成功、爱、健康、游乐、安全、确信、热情、信心、勇气，以及做出改变。这些都是我们想要感受到的美好情绪。

有些人可能会说“我值得家庭”或“我值得金钱”，但这里有一点关键区别需要理解。有些是具体的“车”（手段），而有些是真正的“目的地”（价值）。你真正想要的不是金钱本身，而是金钱能带给你的感受。例如，你可能想要钱，因为它让你感觉被爱，或者让你能为所爱的人买东西。但你内心深处真正渴望的其实是“爱”。认识到这一点，你可能会找到比“赚钱”更好、更直接的方法去获得“爱”。

我们值得生活中各种不同的东西，并且它们对我们有不同的重要性。因此，我们总有一个最重要的东西，比其他东西更重要。

## 价值观如何影响决策？

我们每时每刻都在做决定。无论是出门、开始约会、决定听某个节目、去健身，还是努力了解自己的心理，每一个决定都基于我们内心对事物重要性的排序。

例如，如果你正在工作，而你的女朋友打电话来说想念你。此时，你的决定就取决于你的价值观排序。如果你的价值观中“成功”高于“爱”，你可能会选择继续工作。如果你的价值观中“爱”高于“成功”，你可能会选择去陪伴她。我们所看重的事物，决定了我们在生活中选择的方向。

## 如何找出你的核心价值观？

以下是找出你核心价值观的步骤。请准备纸笔，跟随指引操作。

首先，请写下你认为人生中最值得拥有的五到十样东西。这至少要是五种你渴望体验的情绪或状态。例如：完美、安全、有趣、成功、爱、健康等。写下任何你认为重要的东西，不必追求完美。

**现在，请暂停阅读，花时间列出这个清单。**

写完后，看着这个列表，问问自己：我难道不值得拥有所有这些吗？谁不值得健康？谁不值得爱？谁不值得成功？

接下来，我们需要对这些价值进行排序。请按照以下方法操作：

1.  从你的列表中选出第一个价值（例如“成功”）和第二个价值（例如“爱”）。
2.  问自己一个关键问题：**“在过去，对我来说什么更重要？是感觉成功，还是感觉被爱？”** 请使用你自己的语言来描述这种比较。
3.  选择你认为更重要的那一个，把它记下来。这就是你当前价值观排序中的第一名。
4.  然后，用这个“第一名”去和列表中的第三个价值（例如“健康”）比较。再次问自己：“在过去，对我来说什么更重要？是[第一名]还是[健康]？”
5.  重复这个过程，用胜出的价值去和列表中下一个价值比较，直到筛选完所有项目。这样你就能确定对你而言最重要的核心价值是什么。

完成排序后，你可能会发现结果令人惊讶，甚至可能与你期望的理想生活不符。这很正常。许多成功、开心、满足的人，其价值观排序也可能有调整的空间。

## 关于核心价值的思考

在所有价值中，“爱”和“健康”是极其基础且重要的两项。

*   **健康**是基石。如果你的身体不健康，你就无法长久地享受生活，也会缺乏能量。
*   **爱**是我们能体验到的最高的情绪状态之一。人们许多行为背后的根本动机，常常是为了获得爱或表达爱，这通常源于我们早期与父母的关系模式。

此外，也存在一些我们不想体验的“负面价值”，例如失败、愤怒、羞耻、拒绝。本节课暂不深入探讨这部分，因为它涉及更深的层面，需要更多时间处理。

## 总结

本节课我们一起学习了价值观的核心概念。我们了解到价值观是驱动我们决策和人生方向的根本力量。通过列出并排序你的核心价值观，你可以更清晰地认识自己当前的状态。明确价值观是一个强大的工具，这个简单的练习足以改变你一生的轨迹，让你朝着自己真正想要的方向前进。
