## 如何开启并维持对话 💬

以下是开启和维持有趣对话的几个关键方法。

不要像面试一样逻辑性地一问一答。这会让氛围变得过于逻辑化。你希望氛围更情绪化、更愚蠢、更有趣。

**公式示例：**
```
她的回答: “我住在市中心。”
你的回应: “太好了！我正好需要个新室友，你睡沙发就行。”
```

很多男性不让女性做任何“工作”，他们不停地说话，因为害怕沉默会导致女性离开。你必须学会停止说话，看看她是否会重新开启对话。

### 给出“愚蠢”的答案

当她问你典型问题时，不要总是给出严肃真实的答案。可以给她一个愚蠢的答案，或者混合使用。很多男性在这点上过于紧张。

当男性学习交往技巧时，他们常忽略“校准”，即根据对方的反应调整行为。他们因为害怕受伤而机械地使用技巧，不顾结果。

**核心概念：校准**
> 根据互动对象的实时反馈（语言、非语言），灵活调整你的对话策略和行为强度，而不是固守一套模式。

### 共鸣与好奇心

从“好奇心”的角度出发去问问题，而不是从“索取”的角度。这有巨大区别。

**示例对比：**
- **索取角度:** “你是做什么工作的？” （潜台词：我想评估你）

好奇心非常强大，它让人们感觉可以对你打开心扉。

### 共同点与分歧

找到共同点很好，但有时制造一点分歧并戏剧化地表现它，能让她兴奋地试图说服你。例如，如果她说喜欢苹果，你可以说讨厌苹果。之后，当她阐述观点时，你再找到共识。
