# 帕帕创始人俱乐部【社交商务】：03.业务_03.Busine：08.投资者关系洞察_08.I

在本节课中，我们将要学习关于投资者关系、公司控制权以及创业初期资金管理的核心见解。课程内容基于创始人的真实经历，探讨了保持公司自主权、利用创造性金融工具以及理解不同类型投资者的重要性。

---

上一节我们介绍了业务运营的基本面，本节中我们来看看与资金和投资者相关的深层策略。

**保持控制权与自主经营**

从好朋友的角度来看，我曾经接受过投资。我有一位叫Tony的朋友，他投资了400万。但由于他作为投资者的干预和痛苦，他不断批评公司的经营方式。最终，我把所有的钱和利益都还给了他。

**公式**：`公司完全控制权 = 创始人自由经营 + 创始人期望的文化`

这让公司完全处于我的控制之下，使我能够自由地经营公司，并建立我想要的文化。

**关于经营公司的乐趣与挑战**

现在说到RSD（注：可能指公司或项目），如果我觉得在公司里没有乐趣，我宁愿不待在那里。经营公司当然不总是有趣的，这就是任何公司都会经历的痛苦和艰辛。

因为你必须努力工作和成长，其复杂性确实很复杂。但很多时候，这里也有很多快乐和兴趣。我欣赏这个世界上的这一面，并希望在未来继续拥有这份乐趣。

**构建共赢的成长模式**

目前RSD是我们自己的公司，我们在这里成长。但同时，我们也有教练的利益。他们有能力去发展自己的品牌，实现个人成长，因为他们都在管理自己的团队，就像经营自己的公司一样。

我觉得这很棒，因为这很成熟。这很棒，能让每个人成长。这就是我们所说的“利益”。

以下是利益的核心用途：
*   你被赋予的利益。
*   你被支付的利益。

我认为，任何公司的基本基础，都更有可能获得成长型投资和发展业务。

**不同类型公司的资金需求**

另一方面，有些其他公司需要更多资金来进行重建，特别是制造公司。我有朋友经营工厂做复苏产品，他们赚了很多钱，达到百万美元，但他们仍然需要吸引投资来让他们的计划成功。

因为他们需要支付复苏成本、建筑费用和大型工厂的运营。虽然他们最终可以赚回那些投资，但这需要时间。

**吸引真正相信你使命的投资者**

我们需要寻找的投资者，是那些相信你使命的人。他们甚至可能想要免费工作。实际上，很多教练和员工，大部分（如果不是所有人）都曾免费工作过。

在真实的社交经济中，这可能持续很长一段时间，甚至数年。

**创造性金融策略：以“黑卡”为例**

我们实施了疯狂的金融设计，因为我们没有外部投资。我们有一张美国特殊的“黑卡”。我们基本上创造了一棵“金钱树”。

我写过一篇金融计划文章，为了避免暴露身份（因为这很困难），我称自己为“神秘的Mr. X”。我们使用了美国特殊的黑卡，我们每两三个月花费一万多元。通过这张美国特殊黑卡，我们带来了超过十万人使用它，没有任何费用。

**代码逻辑示例**：
```pseudocode
if 使用特殊黑卡 and 在30天内付清:
    then 可以购买资产并经营生意
```

这意味着我们可以用这张黑卡购买资产，只要在30天内付清。我们要做生意，经营这家公司，我们需要时间，需要几年。

**发展过程中的挑战与应对**

在我们发展的过程中，我们经常没有预算，没有达成我们预期的成绩。如果没有这套预算，或者没有投资者的支出、家庭的支持，我们的资金会很紧张。

但我发现，有些人会给我短期的钱。例如，去“黑卡”的商店，让他们垫付，甚至去小型公司。如果你去小型公司，给他们你的信用卡，他们可能给你100元，让你做一些展示，还可以给你一些展示员。

但我不是去小型公司要100元，我要5万到10万元。用这笔钱来支付每月100到1000元的薪水，做这件事，持续30天，直到我被叫停。我们可以把这个收支平衡玩转，这很困难。

**家庭支持与内部资源**

如果你有家庭资金的支持，这无疑是最好的、最安全的方法。我曾经有非常成功的家庭成员作为员工，他们在我最需要的时候帮助我，不仅在信息资源方面，也在金融资源方面。

特别是当我们有大型投资时，例如我们的网站（像RSE Nation），我们投入了100到1000元。我们还有内部资源管理程序，以及制造业务，这些都能花费上亿元。这些技术在我们业务的背景下是非常棒的。

**警惕纯粹的财务投资者**

这比其他业务的前提要好，例如专业的商业投资者。他们做的很多事情，是为了我们业务做的，也是我们业务需要做的。当然，有些人有相关的兴趣，也有投资者。

但不仅仅是投资者，还有人对业务本身有兴趣。大部分投资者，例如某些类型的投资者，有兴趣教育你，促进你的业务成长，也对业务本身有兴趣。

如果你看像《鲨鱼坦克》这样的电视节目，你会看到很多那样的人。或是像鲨鱼和投资者，或是华尔街的、国内的投资者，他们会在那里，成为公司的投资者，投资你的公司，然后把它拆分、削减，然后提取公司里的所有价值去赚钱。

这很可怕，但人们会这么做，因为这是可行的。要小心。

**寻找信念一致的投资者**

另一方面，有些人真正相信你所在的行业，他们想投资公司，投资业务，因为他们认为教育非常重要。他们会给你钱，为了赚取回报，来交换教育和金钱。

教育可以为你打开大门，建立网络。如果你是一个没有大人脉网络的人，想扩展这个网络，想与其他成功人士分享这个网络，那么接受投资可能是对的。

**投资关系带来的稳定性与安全**

我知道有投资人这样做，因为这给他们带来很大的安全感。如果有公司拥有大量的现金流，还有投资人的关系，或者公司本身有，那么这些投资人会很开放地想投资。

如果他们的现金流下降，例如20%，他们有一年的缓冲期，整个公司就不会失败。所以我认为，拥有资金和赚钱能力是健康的。

管理现金流，尤其是在你成长中的公司，或者大公司里，有很多风险。你需要有安全的财务状况，需要有这些保护措施。

**保护公司利益与个人责任**

在公司利益方面保护自己很重要，因为你不只依靠自己，你还依靠很多人。

现在看这段影片的很多人，可能不太关心成长一个大型公司，或者管理复杂的经营金融、经营公司、国际银行等等。我学过管理方面的东西，但我认为，拥有一个知识，一个有利的知识，就是了解行业内的投资者关系。

这可以帮助人们成为一名企业家或经营者，明白他们应该做什么，或不应该做什么。

**个人经验：自力更生的资金路径**

就我个人而言，我从未获得过企业贷款、企业利益或银行的资金。因为我能够自己投资，我能够找到信用卡，能够找到其他方法。

更重要的是，我能够让客户为我们提供资金，像是通过预购物资等。我们可以走完行业的流程，并从最初开始经营公司。

我认为，可以让客户从未获得企业利益的情况下实现增长。但要非常小心，因为这会影响价值。

**认清价值的本质与人的关系**

通常，这个价值是一个难以知晓的价值，直到你真正能够了解某人。通常，你共事的人，你不会在最初见面或那种浅层关系中真正了解他。

这种关系不会深入增长，直到你真的能够知道那个人是谁。这需要很长时间，通过工作关系，甚至是你个人的朋友来观察。

你爱的、你相信的、你一直支持的个人朋友，当金钱介入时，可能会变得非常不同。

**统一目标，避免内部冲突**

现在，在企业利益方面，高杠杆的情况下，每个房间和房子都属于同一个企业，我们一起工作。但有时候，几个月后，每个房间都会产生相关的企业和利益，这导致了很多冲突。

后来，我会把房子里的每个人都打败，直到我的主要企业成为房子里唯一完全一体的企业，拥有统一的任务和目标。

后来，我们就能够在房子里和谐工作。但经过这么长时间，住在相关的房子里，我们学会了专注。

**核心教训与行动指南**

这教会我们，要专注于我们的交易。其中，矛盾一定要进行梳理和学习。不要每次都一定要争个输赢。

第一，真的，我的更大的目标是非常非常重要的。这让很多人有了新的生活和步伐。

当人们看待这些东西时，就要小心。他们不是那种会忘记你的朋友、忘记应该拥有共同目标和任务的人。

**本节课总结**

本节课中，我们一起学习了关于投资者关系的核心洞察：
1.  **保持控制权**：维护经营自由和文化塑造权至关重要。
2.  **资金策略**：可采用创造性金融工具（如特定信用机制），并优先考虑家庭或客户等内部资源。
3.  **识别投资者**：警惕纯粹财务收割型投资者，寻找认同使命并能提供附加价值（如教育、网络）的伙伴。
4.  **价值认知**：真正的价值和个人关系需要时间检验，金钱会改变关系的动态。
5.  **统一目标**：在团队或合作中，确保所有人朝向一个共同的、清晰的目标努力，是减少冲突、成功经营的基础。

关键始终在于**平衡资金需求与长期控制权及文化完整性**，并**与真正理解并支持你愿景的人合作**。
