# 马克 自然流《10天线上训练课》：10.【开始行动】进阶任务.045.3 搭讪步行者训练 🚶‍♀️💬

在本节课中，我们将要学习如何搭讪正在行走的女性。这是一个重要的进阶技能，能帮助你在任何场景下把握认识心仪对象的机会。

很多人更愿意去搭讪那些坐着的女性，例如在喝下午茶、喝咖啡、玩手机或发呆的女性。

大部分人不想去搭讪这些行走中的女性。他们觉得对方走得太快，不想打扰她。他们认为此刻上前搭讪不是给对方机会，而是造成打扰。

但问题是，如果你遇到一个你很喜欢的女性，她正匆忙地行走，如果你那时不去搭讪，你可能就永远失去了认识她的机会，很多时候一辈子都见不到她了。

搭讪这种行走中的女性有两个办法。

以下是两种核心方法：

1.  **拦截法**：如果她行走速度并不快，你可以将她礼貌地拦截下来。她可以停下脚步，与你聊一会儿。
2.  **同行法**：如果她走路很匆忙，没问题。你可以跟着她一起走，一边走一边聊。这并不会打扰她，而是给她一个机会。

上一节我们介绍了两种具体方法，本节中我们来看看为何这些方法有效。很多人以为如果女性很忙，她肯定不想和自己讲话。

但你换位思考一下，假如是你，你非常忙，要赶着去做一件事情。然后你非常喜欢的类型的女性过来跟你搭话，你会不愿意跟她聊一会儿吗？即使你非常忙，你也会跟她说：“我们一起走，边走边聊吧。”

女性也是一样的。如果你是个高价值的男人，你表现出的行为非常得体、自信且有高价值，她肯定愿意和你聊一会儿。没有人会拒绝一个有价值的、自信的人。

所以，去练习搭讪行走中的女性，这对你的未来非常有帮助。

让我们开始行动吧。

本节课中我们一起学习了搭讪步行者的重要性、两种核心方法（**拦截法**与**同行法**），以及从对方视角理解搭讪并非打扰，而是提供一个认识有价值的人的机会。掌握这项技能能让你在更多场景中把握机遇。
