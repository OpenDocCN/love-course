# 马克自然流《10天线上训练课》第八课：🚀 “搭讪焦虑”终极解决方案

在本节课中，我们将学习解决搭讪焦虑的终极方法：利用情绪杠杆。我们将深入探讨两种不同类型的痛苦，并帮助你做出更明智的选择，从而推动自己采取行动。

---

上一节我们探讨了调整心态的方法，本节中我们来看看如何利用你自身的情绪作为行动的杠杆。

比起搭讪时可能面临的短暂尴尬，不去搭讪所带来的长期自责是一种更深刻的痛苦。如果你不去接触喜欢的女性，你可能会在反复的自责中度过一整天。

这种由自我拒绝产生的痛苦，远比来自他人的直接拒绝更为难受。

过去，你可能将自我价值（ego）建立在外部反馈上。例如，搭讪时女性反应好，你就觉得自己很厉害；反应不好，就觉得自己很差劲。这是将自我价值建立在不可控的他人反应之上。

**核心概念公式：旧模式**
```
自我价值 (Ego) = 他人的反应 (不可控)
```

你应该将自我价值建立在自己可控的行为层面。即，评价标准在于“是否采取了行动”，而不在于“行动后得到何种结果”。

**核心概念公式：新模式**
```
自我价值 (Ego) = 是否采取行动 (可控)
```

以下是一个学员的案例，他长期受困于旧模式。

他常常在街上闲逛一整天，但遇到心仪的女性却不敢上前搭讪，因为他害怕被拒绝。他把自我价值完全寄托在女性的反应上，这导致他无法行动。

他人的反应是你无法控制的，你唯一能控制的是自己的行为。如果你行动后被拒绝，这种痛苦是可以解释和接受的，因为你无法控制他人。

但如果你不去行动，那么情况就变成了：并非女性拒绝了你，而是你拒绝了自己。你会因此看不起自己，陷入漫长的自责。

我相信你曾经历过这种因自我拒绝而产生的、难以言说的难受感觉。它比被女性直接拒绝更令人痛苦。

因此，你面临一个选择。

以下是两种痛苦的对比：

*   **可解释的痛苦**：源于行动后被拒绝。你可以告诉自己：“至少我尝试了。”这种痛苦不会损害你的自我价值感。
*   **不可解释的痛苦**：源于因害怕而拒绝行动。你会不断质问自己：“我为什么这么无能？”这种痛苦会导致长期的自我否定。

如果你是一个理智的人，你应该选择那种“可解释的痛苦”。选择行动，即使被拒绝，你也能在一天中保持更好的自我感觉。而选择不行动，你将在自责中浪费大量时间。

---

本节课中，我们一起学习了利用情绪杠杆克服搭讪焦虑的终极方法。关键在于转变评价体系：将自我价值从“他人的反应”转移到“自身的行动”上。在“可解释的行动之苦”与“不可解释的退缩之痛”之间，明智地选择前者，是推动你向前迈进、并赢得自我尊重的关键。
