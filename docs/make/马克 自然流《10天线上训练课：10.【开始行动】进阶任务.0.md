# 马克 自然流《10天线上训练课：10.【开始行动】进阶任务.0

在本节课中，我们将学习如何通过一系列针对性练习，突破个人在搭讪中的舒适区，以应对各种复杂或令人尴尬的社交场合。

欢迎来到搭讪焦虑挑战栏目。这个栏目主要为那些已经能够进行基本搭讪的人设计。你已经可以在路上走到一位女性旁边，并开始与她进行一段谈话。例如，在排队时，你可以自然地与旁边的人聊天并展开对话。

然而，每个人都有自己的短板。有时你可能在路上遇到一位正在打电话或戴着耳机的女性，你会感到不知所措。或者，当你遇到一个由五六个女性组成的团体时，你不敢上前搭讪。甚至，有时对方旁边有一位男性同伴，你也会感到胆怯。很多人只敢搭讪正在行走的人，却不敢搭讪坐着的人。也有人只敢搭讪坐着的人，不敢搭讪行走的人。每个人都有自己感到不舒适的区域和舒适的区域。因此，很多人有他们做起来很自然的事，也有很多让他们感到非常尴尬的事。同样，也存在一些他们觉得一辈子都不会去做、令他们极度尴尬、且处于舒适区之外的事情。

因此，我在这个系列中设计了**8个项目**供你练习。如果你能完全掌握这些项目，你就能应对大部分场合。在这个系列中，你不需要严格按照我设定的顺序1、2、3、4、5、6、7、8去执行。因为对有些人来说，他们或许已经能够搭讪女性团体，但却不习惯搭讪那些在店里坐着吃东西或喝东西的女性。如果你对某些事情感到尴尬，你的重点就是去挑战它。然而，如果有些任务对你来说相对简单，你也可以去完成。这无非是多花一天时间，但你也会在这个过程中变得更加流利。如果你对所有任务都感到尴尬，那么这些事情恰恰是你应该去做的。一旦你能够掌握这些，你就能应对大部分场合。这对你非常有益。
