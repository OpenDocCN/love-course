## 概述
在本节课中，我们将学习如何识别并避免与某些类型女生建立不良恋爱关系。

---

## 核心概念
**公式**：不良恋爱关系 = 自私女生 + 没有稳定工作女生

---

## 第一种女生：没有稳定工作，常年混迹在娱乐场所
以下是关于第一种女生的特点：

- **特点**：这类女生通常没有稳定工作，经常出现在酒吧、KTV等娱乐场所。
- **原因**：她们可能通过兼职主播、捞女等方式赚取收入。
- **风险**：这类女生的颜值普遍很高，但感情投入可能不稳定。

**代码**：
```python
class Entertainment_Girl:
    def __init__(self, has_job=False, income_source="entertainment"):
        self.has_job = has_job
        self.income_source = income_source
```

---

## 第二种女生：自私自利
以下是关于第二种女生的特点：

- **特点**：这类女生在约会时从不花钱，认为男生花钱是理所当然的。
- **风险**：她们很少回礼，即使收到大额礼物，也可能用小额礼物回礼。
- **建议**：这类女生可能不值得长期付出。

**代码**：
```python
class Selfish_Girl:
    def __init__(self, spends_money=False, reciprocates_gifts=False):
        self.spends_money = spends_money
        self.reciprocates_gifts = reciprocates_gifts
```
