## 1. 钝感力 🛡️

上一段我们提到了“撩”的本质是情绪调动能力，本节中我们来看看第一个核心特质：钝感力。

钝感是敏感的反义词，指听从自己内心，不受外界因素影响。通俗而言，可理解为脸皮厚、心态豁达。其行为模式是：**“这件事行不行，我先去做了再说”**。例如，遇到喜欢的女生，直接上前搭讪，即使失败也视为无损失的经验积累。

以下是钝感力与老实人思维的关键区别：

*   **钝感力者**：行动导向，不惧失败，对外界评价（如朋友嘲讽、女生打压）不纠结。
*   **老實人**：结果导向，无法接受不良结果，容易因他人一句话而患得患失，无法表达真实自我。

如果无法展现真实的自己，又怎能有效吸引女生呢？
