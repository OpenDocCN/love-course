2.  **别看你平时那么坚强，那是因为没有人真正去保护你，其实你内心特别容易受伤。**
3.  **有时候你也会有些倔脾气，有时候别人都不理解你，你渴望遇到那个真正理解你的人。**

这在心理学上被称为 **巴纳姆效应**：当有人直接指出我们某些性格特点并表示理解时，我们常会不假思索地认为对方说得对。

如此一来，对方就会为你打开心扉，认为你是真正懂她的人。当你似乎看清并读懂了他的内心，拉近关系还会困难吗？
