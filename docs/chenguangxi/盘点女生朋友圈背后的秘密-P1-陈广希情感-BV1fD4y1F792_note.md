## 真实生活女孩与九宫格女孩 📸

上一节我们介绍了课程主题，本节中我们来看看第一种对比：真实生活女孩与九宫格女孩。

一种女孩的朋友圈照片特别真实，没有明显滤镜，拍摄手法随意简单，图片之间也毫无关联。

另一种是“九宫格女孩”，她们发布的照片堪称一项大工程。

以下是九宫格照片的常见构成公式：

```
自拍 + 他拍 + 场景 + 美食 + 宠物
```

看到这些精致照片，你可能会感慨这些姑娘生活非常考究，反观自己则显得粗糙。但这里要指出一个秘密：九宫格女孩的自拍或他拍，通常都是精心设计过的。

因为自己的自拍素材可能不够，那些无关紧要的实物照片，找闺蜜提供即可。因此，她所呈现的大多数内容，未必完全真实。她究竟是真正精致，还是“伪精致”，你无从得知。

## 炫富女孩与炫自己女孩 💎

俗语说，做人不能太攀比。但有些女孩喜欢在朋友圈展示名牌包包、下午茶或化妆品。

你可能会感觉她物质或虚荣，但这不一定正确。她可能确实经济宽裕，拎着名牌包买菜。或者，她只是通过努力攒钱，买了心仪的包并分享喜悦，这无可厚非。

“炫自己女孩”与九宫格女孩有所不同。她们的朋友圈并非随意发布，而是倾向于晒出自己的优点。

以下是她们常见的展示逻辑：

*   **身材好**：秀大长腿。
*   **颜值高**：发正面自拍。
*   **局部好看**：例如鼻子好看就发侧脸照。

总之，女生发出来，你夸奖就对了，因为谁都喜欢听赞美。

有的女生给人一种特别健康活泼的印象。她的朋友圈充满各种段子、热点和琐事，一天发布数十条，笑容表情比谁都多。如果你说她发得太多，她可能会反驳：“我发朋友圈还要在乎别人感受？那我还发干嘛？”
