## 什么是“太监心态”？🤔

上一节我们介绍了课程主题，本节中我们来看看“太监心态”的具体含义。

“太监心态”指的是在追求异性过程中，男性表现出过度顺从、卑微和缺乏自信的状态。这种心态的核心是自我定位错误。

*   你表现得像一个需要讨好主人的仆人，而非一个平等的、有魅力的个体。
*   你主动放弃了自己的框架和原则，以对方的喜怒哀乐为唯一行动准则。
*   你内心默认自己价值低于对方，并通过行为不断强化这一点。

**公式：太监心态 ≈ 低自我价值感 + 过度讨好行为**
