女生会通过取笑或戏弄你来测试你的态度，但这绝非嘲笑。当她取笑你时，不必在意，这可能是她拉近关系的方式。

聊天是初步了解的重要途径。初期聊天应避免传递负能量或灰暗内容，以免吓跑对方。随着了解加深，可通过分享个人信息提升关系亲密度，例如爆料自己曾做过的搞笑蠢事。如果女生也分享自己的糗事，这是一个积极的兴趣指标。每个人都会对自己感兴趣的人透露更多私人信息。

如果女生身体斜对你（特别是90度转身），双手交叉胸前，身体后倾，这表示“不要靠近我”，即对你无兴趣。相反，如果她身体正面对你，手臂放松，前倾靠近你，则表示愿意交流。
