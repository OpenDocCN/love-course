那么，这究竟是为什么呢？上一段我们提出了这个普遍现象，本节中我们将具体分析背后的几个关键原因。

以下是导致这种现象的几个主要因素：

**第一，打扮时尚潮流。**
在我们身边总有一类男生非常热衷打扮自己。他们整天会给自己购买好看的衣服和漂亮的鞋子，还会经常打理发型。即便长相普通，经过收拾就显得非常潮流帅气。大部分女人都是视觉动物，很容易被这些男生的形象所吸引。女孩可能起初也没想到这个男生是否专一。在本来就有一定好感的情况下，加上男人的猛烈追求，很快便坠入了爱河。

相比“坏男孩”的外貌和情商，老实男人不仅不注重打扮，让自己输在起跑线上，很多时候也不善言辞，不懂得女孩子的内心。这导致他们被“坏男孩”拉开了距离。

而且，这些差异根本无关于人品的好坏。所以，我们要学习的是“坏男孩”身上的优点，而不是盲目地认为女人只是喜欢坏男人而已。
