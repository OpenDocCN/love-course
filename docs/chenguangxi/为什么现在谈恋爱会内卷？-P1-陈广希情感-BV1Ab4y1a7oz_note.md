# 情感经济学第一课：为什么恋爱变得“内卷”？ 🧠💔

在本节课中，我们将探讨一个普遍的社会现象：当代恋爱关系中的“内卷化”。我们将分析其表现形式、核心原因，并揭示信息传播在其中扮演的关键角色。

不知道大家有没有发现，现在谈恋爱都越来越内卷了。

## 男性视角的“内卷”表现 💼

上一节我们提到了恋爱内卷的现象，本节中我们来看看男性所面临的具体压力。男生的内卷是，追女生的成本越来越高。

以下是男性内卷的几个典型表现：
*   曾经吃饭看电影走完流程就好。
*   现在逢年过节都要礼物转账。
*   约会女生更是要高档轿车接送。
*   心机特别能餐。

## 女性视角的“内卷”表现 💄

了解了男性的处境后，我们再来看看女性所面临的竞争压力。那么女生的内卷也很夸张。

以下是女性内卷的几个典型表现：
*   用着昂贵的护肤品、化妆品。
*   服饰、鞋包都得上档次。
*   医美项目更是不能少。
*   肋骨、鼻、眼综合都是标配。

## 核心原因：信息传播效率的革命 📡

在观察了双方的表现后，我们不禁要问，驱动这一切的根本力量是什么？为什么现在谈恋爱会这么卷呢？本质的原因是，信息的传播效率越来越高了。

我们可以用一个简单的模型来理解这种变化：
```
过去：社交圈小 -> 比较范围有限 -> 竞争压力低
现在：社交圈极大（互联网）-> 比较范围无限扩大 -> 竞争压力剧增
```

曾经我们的社交只能维系几个亲朋好友，但进入移动互联网时代，每人至少有几百个好友。朋友圈这类产品所展示的，永远都是每个人想活成的样子。这把所有人择偶的需求，一下就拉高了。

## “超市理论”：一个生动的比喻 🏪

为了更直观地理解，我们可以引入一个“超市理论”。试想一下，曾经信息闭塞的时候，一个小区可以容纳两家不同的超市，一家在前门，一家在后门。大家互不影响，各自做生意。

而现在由于互联网飞速发展，优秀的那家超市，它的价格、品质、服务，能最快速传递给小区的每一个居民。慢慢的，小区就只能容纳一家最好的超市。

## 情感市场的“赢家通吃” 🏆

将这个理论映射到情感领域，结论就非常清晰了。所以在情感市场上，最优秀的那几小部分人，占据了绝大多数的择偶权。

我们现在变得非常容易，让别人的生活所影响。所有人的要求都变高了。大家都认为，那些朋友圈里让人羡慕的高富帅和白瘦美，就是自己向往的生活。

## 总结与反思 🤔

本节课中我们一起学习了当代恋爱“内卷化”的成因与表现。现在很流行一句话：“被狮子宠爱过的女人，怎么会爱上野狗。” 这正是内卷心态的写照。如果她每天在朋友圈里，看着狮子的生活，那么野狗自然就成了田狗。

其核心逻辑在于：**高效的信息传播（互联网）无限扩大了我们的比较范围，抬高了普遍期望，导致资源（注意力、择偶权）迅速向顶层聚集，从而加剧了竞争。** 理解这一点，是我们在复杂情感世界中保持清醒的第一步。
