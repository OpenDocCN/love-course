## 观察朋友圈动态 📱

上一节我们介绍了课程目标，本节中我们来看看具体如何操作。首先，你需要关注女生的朋友圈动态。以下是几个关键的观察点：

**核心逻辑**：`频繁自拍 + 深夜情感倾诉 ≈ 高概率单身`

## 处理“朋友圈三天可见” 🔒

如果对方设置了朋友圈“仅三天可见”，导致你无法观察到上述动态，那么可以采取一个试探性的策略。

你可以直接但略带技巧地向她提问：
```text
“跟你聊了这么久，你男朋友不会生气吧？”
```
这句话虽然听起来有些“绿茶”，但在实践中往往非常有效。

根据她的不同回答，你可以迅速判断出自己的机会大小。以下是可能的回应及含义：

*   **回应A**：“我们就是普通朋友，他不会介意的。”
    *   **含义**：这通常意味着她确实有男朋友，并且你的定位已被明确为“朋友”。很遗憾，这种情况下你的机会渺茫。

*   **回应B**：“我没有男朋友。”
    *   **含义**：这是一个明确的单身信号。此时，你可以果断推进关系。

    当得到这个回应时，你可以紧接着说：
    ```text
    “那从现在开始，你跟别的男生聊天，我可是会吃醋的哦。”
    ```
