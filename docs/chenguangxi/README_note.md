# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了课程目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，我们通过一个简单的等号（`=`）来给变量赋值。

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“Hello World”存储在其中：
```python
message = "Hello World"
```
这里，`message` 是变量名，`"Hello World"` 是存储的值。

## 2. 基本数据类型 🔢

仅仅知道如何存放数据还不够，我们还需要了解数据的类型。Python中常见的基本数据类型有以下几种。

以下是几种核心的数据类型及其说明：

*   **整数**：表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：表示文本信息，需要用单引号(`‘`)或双引号(`“`)包围起来，例如 `‘你好’`, `“Python”`。
*   **布尔值**：只有两种取值，`True`（真）或 `False`（假），常用于逻辑判断。

## 3. 如何查看和检查类型？ 🔍

在编程时，我们有时需要确认一个变量里存放的数据到底是什么类型。Python提供了一个内置函数 `type()` 来帮助我们。

例如，执行以下代码：
```python
number = 100
print(type(number))
```
屏幕上将会显示 `<class ‘int’>`，这表明变量 `number` 的类型是整数。

## 4. 一个简单的实践示例 💻

现在，让我们把以上概念结合起来，完成一个简单的实践。

以下是创建一个变量、赋值并打印其值和类型的完整步骤：

1.  创建一个变量存储你的名字。
    ```python
    my_name = “陈广希”
    ```
2.  创建一个变量存储你的年龄。
    ```python
    my_age = 25
    ```
3.  使用 `print` 函数将它们输出到屏幕。
    ```python
    print(“我的名字是：”, my_name)
    print(“我的年龄是：”, my_age)
    ```
4.  使用 `type` 函数检查年龄变量的类型。
    ```python
    print(“年龄变量的类型是：”, type(my_age))
    ```

运行这段代码，你将看到变量的内容以及`my_age`的类型信息。

## 总结 📝

本节课中我们一起学习了Python的变量与数据类型。我们知道了变量是存储数据的容器，并通过赋值操作符（`=`）来存放值。我们还认识了四种基本数据类型：**整数**、**浮点数**、**字符串**和**布尔值**。最后，我们掌握了使用 `type()` 函数来检查变量类型的方法。掌握这些基础概念，是后续学习更复杂编程操作的坚实基石。
