## 概述
在本节课中，我们将学习如何通过回答36个问题来建立亲密关系。这套方法旨在帮助人们在短时间内与陌生人建立信任和亲密感。

## 第一层：喜好和小趣味
以下是第一层的问题，主要涉及双方的喜好和兴趣。

**以下是喜好和小趣味的问题：**
1. 你最喜欢的季节是什么？
2. 你最喜欢的电影类型是什么？
3. 你最喜欢的食物是什么？
4. 你最喜欢的旅行目的地是哪里？
5. 你最喜欢的音乐风格是什么？

## 第二层：曾经和自我
第二层的问题涉及双方的过去和个性。

**以下是曾经和自我的问题：**
1. 你小时候的梦想是什么？
2. 你最难忘的一次旅行是哪一次？
3. 你最喜欢的老师是谁？为什么？
4. 你最喜欢的书籍是什么？为什么？
5. 你最喜欢的运动是什么？

## 第三层：当我变成我们
第三层的问题旨在探讨双方的关系和共同点。

**以下是当我变成我们的问题：**
1. 如果你可以改变一件事，那会是什么？
2. 你认为我们的关系中最重要的是什么？
3. 你希望我们的未来是怎样的？
4. 你认为我们的共同点是什么？
5. 你认为我们的不同点是什么？

## 第四层：分享尴尬及隐私
第四层的问题涉及双方的隐私和尴尬话题。

**以下是分享尴尬及隐私的问题：**
1. 你最尴尬的一次经历是什么？
2. 你最害怕的事情是什么？
3. 你有没有什么秘密？
4. 你有没有什么不为人知的爱好？
5. 你有没有什么特别的习惯？

## 第五层：底层的价值观
第五层的问题涉及双方的价值观和信念。

**以下是底层的价值观的问题：**
1. 你认为人生最重要的价值观是什么？
2. 你如何看待金钱？
3. 你如何看待家庭？
4. 你如何看待爱情？
5. 你如何看待事业？

## 总结
本节课中，我们一起学习了如何通过回答36个问题来建立亲密关系。这套方法可以帮助人们在短时间内与陌生人建立信任和亲密感。希望你在实际应用中能够取得成功。
