# 高手和傻子 - 课程P1 - 陈广希情感 🧠💔

## 核心概念解析

上一段我们介绍了课程的主题，本节中我们来看看“高手”与“傻子”的具体定义和行为模式。

**高手**是那些有原则、有骨气的人。他们遵循一个核心逻辑：**“你能伤害我一次，就是拿我不当回事，那么拜拜再见。”** 这意味着，一旦他们识别出对方不尊重自己，便会果断离开，绝不让自己在同一个地方跌倒第二次。

**傻子**则相反，他们是缺乏骨气的人。其行为模式可以概括为：**“他被女人被朋友伤害一次，他还跟她们玩，人家继续伤害他都没事。”** 傻子会允许自己反复掉入同一个陷阱，可能达到三次、四次甚至更多。

## 骨气与尊重的关系

理解了两种模式后，我们来看看骨气如何直接影响他人对你的态度。

一个男人必须有骨气，女人才会尊重你。如果你没有骨气，对方就会拿你不当回事。你的自我价值感，直接决定了你在关系中被对待的方式。
