## 概述
在本节课中，我们将学习如何在聊天中主动离场，避免不必要的尴尬和误解。

## 主动离场的重要性
在聊天过程中，主动离场是一种有效的沟通技巧，可以帮助你更好地控制对话节奏，避免过度投入。

### 分时间段聊天
**公式**：白天浅尝辄止，晚上深入交流

以下是具体的时间段安排：

- **白天**：简短交流，避免深入话题。
- **晚上**：重新建立沟通，深入交流。

### 主动离场的方法
以下是一些主动离场的方法：

1. **表达忙碌**：告诉对方你有其他事情要处理。
2. **转移话题**：将话题引向结束，如“今天聊得挺开心，我们下次再聊吧。”
3. **设置时间限制**：提前告知对方你只能聊多久。

### 例子
以下是一个例子，说明如何在聊天中主动离场：

我说我先起来吃东西了，有的兄弟会说，南哥我好不容易聊到这一块了，那为什么又聊到星座了，不继续乘胜追击呢，你们有没有看过第一集，这个女生也是刚起床，我也表达我刚起床，大家记住今天这集最后一个干货。

就是你在聊天的过程当中，一定要分时间段，白天浅尝辄止，刚聊到比较开心，自己先退出这个聊天氛围，自己主动的去离场，表达自己白天有忙碌的事情，而到晚上八九点，你再重新去建立沟通。

因为晚上一个人躺在床上拿着手机，他更愿意接受情感，所以大家不要说白天跟一个女生聊得不错，就一直聊下去，因为很可能她马上中午就要吃饭去了，她下午可能要逛街，甚至是开会了，还可能领导批评她。

会导致你白天说了很多优秀的话，正确的话，最后对方都给你一个冷脸。
