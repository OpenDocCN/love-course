# 聊天初期如何高胜算测试兴趣度（2） - P1 - 陈广希情感 - BV1AP4y1H7kV

## 概述
在本节课中，我们将学习如何在聊天初期通过简单的方式测试对方的兴趣度。

## 测试兴趣度的方法
### 1. 分享个人状态或感受
- **方法描述**：在聊天初期，分享自己的状态或感受，例如中午刚吃完饭很撑，或者晚上刚跑完步浑身疼。
- **公式**：`个人状态/感受 = 测试兴趣度`

### 2. 观察对方反应
- **方法描述**：观察对方是否愿意为你造句，或者回应你的分享。
- **公式**：`对方回应 = 兴趣度指标`

### 3. 分析对方回应
- **方法描述**：分析对方的回应，例如是否使用了表情包或者进一步的文字描述。
- **公式**：`表情包/文字描述 = 情感表达`

## 实例分析
- **描述**：你中午刚吃完饭很撑，对方愿意为你造句，并发出一个表情包。
- **分析**：对方愿意为你造句并使用表情包，表明她对聊天内容有一定兴趣。

## 总结
本节课中我们一起学习了如何在聊天初期通过分享个人状态或感受，观察对方反应，以及分析对方回应来测试对方的兴趣度。希望这些方法能帮助你更好地了解对方的兴趣，从而进行更深入的交流。
