## 概述
在本节课中，我们将探讨如何判断一个人是否在乎你。通过观察对方的行为和态度，我们可以迅速了解他们的真实感受。

## 判断标准
以下是判断一个人是否在乎你的几个标准：

### 1. 重视程度
**公式**：在乎程度 = 重视行为 / 忽视行为

- **在乎你的人**：会视你如命，忽略谁，也不会忽略你。他们会置顶你的名字，关注你的信息，牵挂你的悲喜，关心你的身体。
- **不在乎你的人**：拿你当风，不主动不热情，更不想把你心疼。你在付出在给予，照样对你无动于衷。

### 2. 行为表现
**代码**：
```python
def 主动程度(行为):
    if 行为 == "主动":
        return 1
    else:
        return 0

def 热情程度(态度):
    if 态度 == "热情":
        return 1
    else:
        return 0

def 关心程度(关心行为):
    if 关心行为 == "关心":
        return 1
    else:
        return 0
```

- **在乎你的人**：主动程度高，热情程度高，关心程度高。
- **不在乎你的人**：主动程度低，热情程度低，关心程度低。

### 3. 时间观念
**公式**：在乎程度 = 时间投入 / 忽视时间

- **在乎你的人**：想找你的人，时间总会有。
- **不在乎你的人**：不想找你的人，忙碌就是他的借口。

### 4. 情感需求
**代码**：
```python
def 情感需求(需求):
    if 需求 == "有需求":
        return 1
    else:
        return 0
```

- **在乎你的人**：心里有你，怎么样都可以。
- **不在乎你的人**：心里没你，怎么样都不行。
