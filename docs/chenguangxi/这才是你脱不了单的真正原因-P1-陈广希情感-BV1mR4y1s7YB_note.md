# 情感课程01：这才是你脱不了单的真正原因 🎯

在本节课中，我们将探讨阻碍多数人脱单的核心心理障碍——自卑。我们将分析其表现、根源，并提供具体的改善方法。

## 自卑是脱单的主要障碍

据我所知，90%来咨询的兄弟在感情方面最大的问题不是没钱，而是自卑。

我承认如果你有很多钱，脱个单真的不难。但一旦遇到你喜欢的女生，你就会开始想：我又长得不够高，我又不够帅，我又不是那么优秀。那我到底除了有钱，还有什么点是适合她的吗？

于是你要花更多的钱在女人身上，才能体现自己所谓的那一份自信。其实自卑是刻在你骨子里的，不是你有钱了就会消失的。

有些人没钱一样能追到妹子。甚至有些穷小子能让富婆倒追，有些穷渣男能让女人心甘情愿付出。

## 自卑的具体表现

如果你跟女孩子相处不敢主动，甚至退缩；聊天不知道聊什么；约会时像个孩子般被动、沉默；宅、消极、负能量，这就是自卑的表现。

## 克服自卑的方法

上一节我们认识了自卑的表现，本节中我们来看看如何克服它。

想要摆脱自卑首先要明白一点：每个人都有自己的闪光点。可能是内在的，可能是外在的，也可能是需要培养出来的。不管如何一定要找出来。

这样当你面对自己喜欢的女生时，内心深处就会有一股自信告诉自己：**我有某些方面的优点，她一定会喜欢我。**

其次，不要总待在自己的舒适区，一定要多看看外面的世界。你眼界越小，聊天的话题就越少，你就越觉得自己不会聊天，觉得自己情商低。

你就会越自卑，觉得不会有女孩喜欢这样的自己。反之亦然。

尝试着去改变自己，哪怕每天只有一点点。一年后的你也是完全不一样的你。永远不要低估自己的潜力。

相信自己真的值得拥有更美好的生活。

## 总结

本节课中我们一起学习了自卑是脱单的主要障碍，识别了其具体表现，并掌握了通过发现自身闪光点和走出舒适区来建立自信的方法。改变始于点滴，请相信你值得更好的感情与生活。
