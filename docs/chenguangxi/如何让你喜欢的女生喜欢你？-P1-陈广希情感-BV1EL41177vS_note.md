## 两性吸引的本质：情绪波动

上一节我们介绍了课程目标，本节中我们来看看两性吸引的根本是什么。男女交往过程中，好感是可以人为制造的。两性吸引的基础是**情绪**。情感的产生建立在**情绪波动**的基础上。

相处时，你让对方产生的情绪波动越大，对方喜欢上你的可能性就越高。反之，如果对方的情绪总是很平淡，那么对方对你很可能没有那方面的想法。

## 建立情绪波动的四个步骤

理解了情绪波动的核心作用后，我们来看看情绪是如何一步步建立的。以下是构建好感的四个关键要素，其中第四点是核心。

### 1. 接近性

当你看到一个心动的女生，可能会心跳加速并想立刻表白。但如果双方并不熟悉，莽撞的行为可能会吓退对方。

你可以多制造一些偶遇的机会。这类似于心理学中的**曝光效应**：当你以一定的频率出现在对方的视线里，对方就会在潜意识中对您产生熟悉感和好感。

**公式/概念**：`好感度 ∝ 曝光频率（在适度范围内）`

### 2. 相似性

在聊天中，如果女生发现你们喜欢同类型的电影、相同口味的美食，甚至三观都比较接近，她会觉得“跟这个男生挺投缘”，并认为“好久没遇到这么聊得来的男生了”。

这会使她开始向你分享她的生活状态和情感状态，从而产生吸引力。

你可以通过多观察女生的朋友圈动态，寻找共同话题，并培养共同的兴趣爱好来增强相似性。

当她身边的男性对她的好显得很刻意时，而你却能不经意地做到，效果会更好。

例如，约会时发现女生不时咳嗽。有些男生会不断询问是否感冒、是否需要买药。而你可以让她陪你去买水，结账时“顺便”拿一盒润喉糖递给她，并说：“看你刚才一直咳嗽，应该是嗓子不舒服了吧。”

这时，一股温暖的情绪会瞬间涌上她的心头。

### 4. 反差性（核心）

你需要有自己的态度和想法，不会对女人言听计从。如果对方让你不开心或感觉不被尊重，你应该拒绝她的要求。

例如，如果女生总是在出去玩完后让你开车去接，而不是一开始就邀请你同去，你可以反问：“所以我只是你的司机吗？什么时候给我结一下工资？”

她身边的讨好者通常言听计从，而你像纯爷们一样敢拒绝、不害怕失去。这种反差感会让女人不自觉地对您产生更深的情绪（视频中称为“牧墙情绪”）。
