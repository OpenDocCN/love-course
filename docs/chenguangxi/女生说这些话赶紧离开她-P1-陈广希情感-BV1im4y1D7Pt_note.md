# 情感课程 01：识别需远离的言语信号 🚩

在本节课中，我们将学习如何识别女性在交流中可能发出的、暗示关系不健康或对方极度自我的言语信号。理解这些信号有助于避免在无果的感情中过度投入。

上一节我们介绍了本课程的目标，本节中我们来看看具体的六种典型言语及其背后的含义。

以下是六种需要警惕的言语信号，其排列顺序大致体现了对方“自我中心”程度的递增。

1.  **看你表现吧。**
2.  **我很忙，没时间。**
3.  **不想打电话，打字说。**
4.  **你又开始了？说了没兴趣。**
5.  **有完没完？我不是已经说过了吗？**
6.  **我就是这样的人，你要想清楚。**

说过这些话的人，通常表明其性格比较自我。从第一句到第六句，这种自我中心的程度是逐渐加深的。

不要幻想你能够感化这样的人。即使你们最终在一起，关系也可能难以快乐。一个真心喜欢你的人，不会总是让你处于等待和猜测之中。

> 所谓“陪伴是最长情的告白”，有时只是安慰被动等待者的说辞。

本节课中我们一起学习了六种值得警惕的言语信号，它们揭示了对方可能存在的自我中心特质。识别这些信号是保护自己情感投入的第一步。请检查你是否有过类似的经历。
