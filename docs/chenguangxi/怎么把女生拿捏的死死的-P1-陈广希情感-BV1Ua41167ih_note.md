# 情感课程 P1：如何建立吸引力 🎯

在本节课中，我们将要学习建立情感吸引力的核心原则。关键在于调整自身心态与行为模式，避免因过度需求而丧失吸引力。我们将通过具体的心态指导和行动准则来展开。

上一节我们提到了课程主题，本节中我们来看看具体的第一条核心原则：降低需求感。

降低需求感是建立吸引力的基石。这意味着你需要控制自己主动联系的频率，避免表现出过强的目的性。具体行为包括：如果对方不主动联系你，你也无需主动寻找对方。在情感互动中，害怕失去会直接导致姿态降低，变得犹豫和讨好，而这种状态通常缺乏吸引力。

无所畏惧且无欲无求的心态，是实践这一原则的理想状态。

以下是实践降低需求感的具体行动列表：
*   控制主动联系的频率。
*   不因对方暂时冷淡而焦虑或过度反应。
*   将注意力更多地放在自身生活和成长上。

---

本节课中我们一起学习了情感吸引力的首要原则——降低需求感。其核心在于通过调整自身心态（**无所畏惧**）和行为（**减少主动**），来避免暴露过强的目的性，从而保持自身的吸引力和姿态。记住，**不害怕失去**是心态建设的关键。
