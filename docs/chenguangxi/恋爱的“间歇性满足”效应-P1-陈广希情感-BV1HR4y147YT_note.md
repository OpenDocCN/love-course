## 概述
在本节课中，我们将探讨恋爱中的“间歇性满足”效应，这是一种心理学现象，通过实验和案例来解释其对人际关系的影响。

## 实验背景
**公式：间歇性满足 = 不确定性 x 迷恋**
心理学家进行了一个著名的实验，将三只老鼠分别放入三个不同的笼子中：
1. 第一只老鼠按开关就有食物吃。
2. 第二只老鼠按开关随机有食物吃。
3. 第三只老鼠按开关没有食物吃。

## 实验结果
经过一段时间，只有第二个笼子的老鼠按开关频率最高，这表明间歇性满足可以增加对不确定事件的迷恋。

## 案例分析
**案例：公司高管与女朋友**
一位年薪百万的公司高管对女朋友百依百顺，但最终女朋友还是出轨了。这验证了实验中第一只老鼠的行为。

高情商的男人不会盲目对女人好，而是根据自身情况和女生的反馈来适时满足她。

- 不带女人去超出自己能力范围的消费场所。
