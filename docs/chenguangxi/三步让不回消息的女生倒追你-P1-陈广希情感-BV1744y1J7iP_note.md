1.  她很忙，确实没时间看消息。
2.  她觉得跟你聊天非常无聊、无趣，认为你很“直”，所以不想回复。

这样既能引起女生对你话题的兴趣，又能让她分享自己“排雷”或“种草”的经验，一举两得。

仅仅让女生对你的话题产生兴趣还远远不够。如果聊天节奏掌握不好，女生照样不愿意搭理你。
