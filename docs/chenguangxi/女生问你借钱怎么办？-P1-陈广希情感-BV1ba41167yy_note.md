上一段我们提到了女生借钱这一现象，本节中我们来看看为何需要谨慎对待此类请求。

女生跟你借钱怎么办？凭什么跟你借？怎么也轮不到跟你借。她没有钱可以找爸妈，不行可以找朋友、同事、闺蜜。还不行就找“马爸爸”，用花呗、借呗。怎么也不会找到你。她找到你，说明再也没有人愿意借给她了，她的信用已经破产。

既然她的信用已经破产，那就更不能借了。此外，她可能觉得你傻。这根本不是“借”，而是“要”。

有人可能会产生顾虑：可是不借，她不理我怎么办？

*   口头答应：“你等着吧，我等会给你转过去。”
*   立即执行拉黑删除操作。
