## 核心原则：感性情绪是关键

上一节我们介绍了课程主题，本节中我们来看看其背后的核心原则。女生如果喜欢上一个男生，通常不会进行客观分析，例如评估对方的外貌、财富或能带来的实际利益。

**公式：** 喜欢 ≠ 理性分析（外貌 + 财富 + 利益）

那种基于条件的衡量更像是交易，而非恋爱。女生对男生的喜欢，主要来源于她**感性的情绪**。当处于“恋爱上头”的状态时，她最不理智、最不清醒，甚至可能“稀里糊涂”就被打动。

因此，关键不在于在她理性时表白，告诉她你有多喜欢她。那样做，即使她对你有兴趣，也可能直接拒绝。

根据心理学的**吊桥效应**，在惊险、刺激的环境下，人更容易对身边的同伴产生心动的感觉。因为她会将环境引发的紧张、刺激和心跳加速，**误认为**是你带给她的感受。

**核心概念：吊桥效应**
> 在危险或刺激情境中产生的心跳加速等生理反应，容易被错误地归因于对在场者的浪漫兴趣。
