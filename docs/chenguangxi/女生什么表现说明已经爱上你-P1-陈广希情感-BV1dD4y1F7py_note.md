## 从陌生到爱上的四层表现

上一节我们概述了课程目标，本节中我们来看看核心的“四层表现”模型。女生对你的情感是一个渐进的过程，可以分为四个清晰的层次。

以下是四个层次的具体表现：

1.  **第一层：礼貌与客气**
    此阶段你们刚刚认识。女生的表现仅限于基本的社交礼仪，例如简单问候、礼貌性回复消息。她的态度中立，没有表现出特别的兴趣或亲近感。

2.  **第二层：友好与兴趣**
    关系进入朋友阶段。女生会愿意与你分享更多日常，主动寻找话题，并接受你的一般性邀约（如群体活动）。她对你有好感，但仅限于朋友范畴。

3.  **第三层：信任与亲近**
    这是关系深化的关键转折点。女生开始对你产生情感依赖，表现包括：分享个人秘密与烦恼，在你面前展现脆弱面，主动且频繁地联系你，并愿意进行单独约会。

4.  **第四层：爱慕与投入**
    这是最高层次。女生的表现充满爱意，例如：主动进行肢体接触，规划你们的未来，为你付出时间与精力，在你需要时毫不犹豫地提供支持，并且会因为你而吃醋。

## 如何判断你所在的阶段

上一节我们介绍了四个层次的定义，本节中我们来看看如何应用这个模型进行判断。关键在于观察对方行为的**一致性**和**主动性**。

以下是用于判断的核心方法：

*   **分析沟通主动性**：她主动发起聊天的频率和内容深度。公式可简化为：**情感深度 ∝ 主动分享的个人话题占比**。
*   **观察邀约反馈**：她接受邀约的类型（群体还是单独）以及赴约时的投入程度。
*   **留意特殊对待**：对比她对你和对他人的态度是否有明显区别。代码逻辑类似于：`if (她对你的行为 != 她对普通朋友的行为) { 你可能处于更高阶段 }`。
*   **感受情绪波动**：她的情绪是否会明显受到你的影响，例如因你而开心或沮丧。
