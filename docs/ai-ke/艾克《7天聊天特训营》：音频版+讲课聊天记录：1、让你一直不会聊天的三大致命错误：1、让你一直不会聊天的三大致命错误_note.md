# 艾克《7天聊天特训营》：音频版+讲课聊天记录：1、让你一直不会聊天的三大致命错误

## 概述
在本节课中，我们将探讨让你一直不会聊天的三大致命错误，帮助你规避这些错误，提升聊天技巧。

## 错误一：不会万未思考
**公式**： 万未思考 = 理解对方感受 + 考虑对方需求

**描述**： 
- 不会考虑对方的感受，只顾自己的需求。
- 例如，看到女生抽烟就劝她别抽，不考虑她抽烟的原因。

## 错误二：怎么舒服怎么来
**公式**： 舒服怎么来 = 理性思考 + 控制情绪

**描述**： 
- 只顾自己的感受，不考虑对方的感受。
- 例如，直接问女生约吗，不考虑对方的反应。

## 错误三：毫无标准可言
**公式**： 个人标准 = 了解自己 + 了解对方

**描述**： 
- 没有个人标准，对女生没有要求。
- 例如，看到美女就说美女你好，不考虑对方的性格、学历等。

## 总结
本节课中，我们学习了让你一直不会聊天的三大致命错误，分别是不会万未思考、怎么舒服怎么来和毫无标准可言。希望你能规避这些错误，提升聊天技巧。
