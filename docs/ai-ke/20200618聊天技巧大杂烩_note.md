# 20200618聊天技巧大杂烩教程

## 概述
在本节课中，我们将学习如何通过聊天技巧提升与异性的互动效果。我们将探讨开场白、维持对话、幽默技巧以及如何避免常见的聊天误区。

## 定位与开场
**定位**：确保你的定位准确，例如在成都。
**开场**：选择合适的开场白，例如从对方的头发入手，如“白发魔女，这个位置终于刷新了”。

## 维持对话
**维持对话**：如果对方不在聊天状态，说明她可能在做其他事情，如吃饭。
**竞争对手**：她不在聊天状态意味着竞争对手较少。
**真假话**：通过真假话的延伸，间接夸赞对方，例如“你肯定也很爱说真话，我代表着说，她说真话，所以她好看”。

## 角色扮演
**角色扮演**：通过角色扮演，如“换一种方式，说一下假话”，引导对方进入角色扮演的感觉。
**调侃**：在调侃中可以聊一些内心想法或好玩的事情，使聊天内容多样化。

## 信息交流
**信息交流**：描述分享你们的状态，如“我说你现在除了在跟我聊天，你还在干嘛”。
**引导**：通过引导女生追逐，如“我看你是否发心，故意套路她一下”，使对话更有趣。

**高标准**：以高标准的生活方式，如“按照更高标准的手法，去做一件事情”。
