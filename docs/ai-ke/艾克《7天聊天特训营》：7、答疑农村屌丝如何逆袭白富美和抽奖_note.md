我就总结了以下几类给你们做解答。在答疑结束后，我会通过四个故事告诉你，我是如何从一个屌丝逆袭成被一个白富美倒追的男生的。你可以当作故事来听，也可以当作干货来听。它多多少少能帮助你更好的实现情感自由、财富自由。故事结束后，我们还会进行抽奖。奖品一共有三个。

### 1. 如何展示价值

展示价值是为了获得女生的喜爱。女生在喜爱你后，你才有可能去跟她发展为亲密关系。展示分三种，按照可信度排行。

- **直接体验**：让她感官体验到你的价值，比如你要展示自己的厨艺好，你可以直接做好饭给她吃，让她尝一下。
- **讲故事**：在已经发生的事情中展示价值，比如你也是要去展示厨艺比较好，你可以告诉她，自从给家里人做了饭，他们就长胖了，每天都要多吃好几顿。
- **描述预期**：在还没发生的事情中展示价值，比如也是要去展示厨艺，你就说，你吃完我做的饭肯定会夸我。

### 2. 什么情况下可以暧昧

其实我在第四节课的前半节内容已经讲过什么时候可以暧昧。就是当女生欣赏你了。就可以尝试暧昧。那女生欣赏你的反应都是什么呢？

- 她的注意力一直在你身上。
- 她的回复之处很多，回复速度也很快。
- 会给你发很多表情，会表现出很多积极情绪。
- 主动找话题，符合你的观点。
- 撒娇，主动表达需求。
- 主动展示自己的优点，或暴露自己的缺点。
- 主动和你亲密，或者愿意配合你亲密。

### 3. 如何做到高姿态，怎么树立奖品性

首先，你不要谁都对她好，因为那只会得出一种结果。爱你的人会变成恨你，恨你的人会一直恨你。然后，其实你要做的事情非常简单，就是怎样去建立奖品性。其实就一句话，让爱你的人更爱你，让恨你的人更恨你。

### 4. 如何跟不同女生相处

这个问题的答案，归根结底就是万为思考出，每个女生内心到底在想什么，她想要什么。然后你去对症下药。如果非要展开来讲，不同的女生都该如何相处，那其实是可以写无数本书的，因为每个女生的各方面都不可能完全相同。当你看完A女怎么去聊，那你又遇到了B女，那你可能不知道怎么去聊。所以重点是要学会如何万为思考。当你能读懂任何人的需求时，你就能搞定任何人。

### 5. 见面后哪些点可以注意有什么干货

这里我给到大家五个干货。

- **55378理论**：人与人交流的信息传递过程当中，肢体语言、语气语调内容的比重分别是55378。这个理论重点表达了，说什么不是最重要的，语气语调和肢体语言更为重要。所以见面的时候不必太追求要说什么，而是保持放松的肢体语言和语气语调去互动。
- **推进关系**：你们出来约会都是为了推进关系的，否则就是在浪费生命。所以你是有义务在场地上、语言上、肢体互动上，逐渐朝着越来越暧昧的氛围推进。
- **注意力**：女生只要把注意力一直放在跟你有关的事情上，那就是对你有需求。所以你可以不管她说什么，或者是语言上怎么去推开你，你都可以持续的推进关系。
- **胆大心细**：整个过程中务必保持胆大心细、脸皮厚。说白了就是勇敢去突破舒适区。然后就算被推开了也要保持一种不在意的感觉，并仔细观察好每个细节，思考清楚应该怎么做。

### 6. 有哪些书是你觉得人生必读的？
