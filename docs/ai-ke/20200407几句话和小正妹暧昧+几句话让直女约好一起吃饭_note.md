# **20200407 暧昧聊天技巧教程**

在本节课中，我们将学习如何通过几句话和小正妹暧昧，以及如何用几句话约好一起吃饭。

## **一、开场白与互动**

以下是开场白和互动的技巧：

- **开场白**：可以采用“你的照片比本人还好看啊哈哈，我叫谁谁谁自我介绍一下就好了”这样的方式。
- **互动**：在聊天过程中，可以适时地调侃对方，例如“你长得应该挺帅的但你为什么不来个自拍呢”。

## **二、话题选择与引导**

- **话题选择**：选择与对方兴趣相关的话题，例如“你不是说今天匹配了很多妹子吗？”。
- **引导**：通过提问引导对方回答，例如“这个梗可以用在任何时间段，当女生说要唱歌的时候一般都可以用”。

## **三、暧昧技巧**

- **暧昧**：通过调侃和暗示来营造暧昧氛围，例如“你手机端听一下吧，我这里听不到”。
- **调侃**：可以调侃对方的外貌或行为，例如“你手机端听一下吧，我这里听不到，啊她说啥玩意儿，自己都养不好还养猫啊”。

## **四、邀约技巧**

- **邀约**：在聊天过程中，适时地提出邀约，例如“我不会胖的OK也迎合上来了，女生已经开始追了，已经开始追上来你已经说出来了，女生已经进入追逐状态”。
- **成功率**：通过营造暧昧氛围，提高邀约成功率。

## **五、总结**

本节课中，我们一起学习了如何通过几句话和小正妹暧昧，以及如何用几句话约好一起吃饭。希望这些技巧能够帮助你在社交场合更加得心应手。
