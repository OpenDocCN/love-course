# 20200925几个可复制开场+心态调整教程

## 概述
在本节课中，我们将学习如何进行有效的开场白，调整心态，并理解为什么高冷的态度比讨好更能吸引人。

## 一、开场白技巧
以下是几个可复制的开场白示例：

1. **调侃开场**：例如，“你的照片看起来很有趣，你是怎么做到的？”
2. **赞美开场**：例如，“你的笑容很迷人，让我想起了某个特别的人。”
3. **幽默开场**：例如，“你的名字听起来很有趣，是不是有什么特别的含义？”

## 二、心态调整
1. **高冷与讨好**：**高冷**意味着做自己，不管对方是否回应，都保持自信和独立。而**讨好**则意味着为了得到对方的认可而改变自己，这往往会导致失去自我和尊严。
2. **做自己**：不要为了迎合对方而改变自己，真实的你才是最有魅力的。

## 三、案例分析
以下是对学员聊天记录的分析：

1. **学员A**：开场白过于随意，缺乏吸引力。
2. **学员B**：开场白过于讨好，显得低价值。
3. **学员C**：开场白幽默风趣，能够引起对方兴趣。

## 四、总结
本节课中，我们学习了如何进行有效的开场白，调整心态，并理解了高冷与讨好的区别。记住，做自己才是最吸引人的。
