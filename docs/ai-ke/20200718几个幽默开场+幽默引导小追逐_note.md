# **20200718幽默开场与引导小追逐教程**

在本节课中，我们将学习如何通过幽默的方式开场并与女生进行有趣的互动，引导她们进入小追逐的状态。

## 一、开场白的重要性

**公式**： 开场白 = 好奇心 + 话题

一个好的开场白能够激发女生的好奇心，并引出有趣的话题。

## 二、幽默开场的方法

**1. 利用好奇心**

*   **例子**： “我猜不到哪一个是你，那就把帽子，那我，就喜欢，那我就喜欢你以外的一个东西吧——帽子。”

**2. 反转**

*   **例子**： “你的睡衣太好看了吧，你还有几个号？那换一个号吧，你把你的号打出來，分开打，我再开一个号。”

**3. 无中生有**

*   **例子**： “听说跟我聊天会漂亮，会变美，诶按错了，我就一直自圆其说，我说你要不试试调侃的方式。”

## 三、引导小追逐的技巧

**1. 让对方开心**

*   **例子**： “哎呦喂，一筒个锤子。”

**2. 让对方好奇**

*   **例子**： “我特别坏，当女生以为我要对她坏的时候，我就说，比如有人要来调戏你，我就会拿出水枪滋他。”

**3. 推拉**

*   **例子**： “长得帅就资本，他也不知道说啥吗？很正常一个不会聊天的人，他自然的反应就是这样。”

## 四、注意事项

*   **保持轻松的心态**： 聊天时不要过于紧张，放松自己，让对方感受到你的真诚。
*   **尊重对方**： 不要让对方感到不舒服，尊重对方的意愿和感受。
*   **持续学习**： 不断学习新的聊天技巧，提升自己的聊天能力。
