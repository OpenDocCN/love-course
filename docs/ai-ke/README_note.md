# 课程名称：艾克（Ekko）技能详解与实战教程 🕰️

在本节课中，我们将要学习《英雄联盟》中时间刺客“艾克”的技能机制、核心玩法与实战技巧。艾克是一位高机动性的AP刺客，擅长利用时间回溯进行高风险高回报的操作。

## 技能解析

上一节我们介绍了课程目标，本节中我们来看看艾克每一个技能的具体效果。

### 被动技能：Z型驱动共振
艾克对同一目标的第三次攻击或技能会触发额外魔法伤害。如果目标是英雄，艾克还会获得巨额移动速度加成。
*   **公式**： 额外伤害 = **30-140 (基于等级) (+ 80% AP)**。对野怪造成150%伤害。
*   **移动速度**： 触发时获得 **40/50/60/70/80%** (在1/6/11/16级时变化) 加速，持续2/2.5/3秒(在1/6/11级时变化)。

### Q技能：时间卷曲器
艾克投掷一个设备，对路径上的敌人造成魔法伤害。达到最大距离后，设备会展开，减速区域内的所有敌人，然后返回艾克身边，对返回路径上的敌人再次造成伤害。
*   **代码描述（伤害）**： 去程伤害：**60/75/90/105/120 (+ 30% AP)**。回程伤害：**40/65/90/115/140 (+ 60% AP)**。
*   **减速效果**： 被命中的敌人减速 **32/39/46/53/60%**，持续2.5秒。

### W技能：时光交错
被动：艾克的普攻对生命值低于30%的敌人造成额外魔法伤害。
主动：艾克在目标区域创造一个时空领域，在3秒延迟后，领域会展开。进入领域的敌人会被减速。如果艾克进入该领域，他会获得一个护盾并晕眩区域内的所有敌人。
*   **被动伤害公式**： 已损失生命值的 **3% (+ 每100AP提升1%)**。
*   **主动护盾公式**： 护盾值 = **80/100/120/140/160 (+ 150% AP)**，持续2秒。
*   **控制效果**： 晕眩区域内所有敌人 **2.25秒**。

### E技能：相位俯冲
艾克进行短距离冲刺，使他的下次普攻获得额外距离、造成额外魔法伤害并闪烁到目标身边。
*   **伤害公式**： 额外魔法伤害 = **50/75/100/125/150 (+ 40% AP)**。

### R技能：时空断裂
艾克回到4秒前的位置，治疗自身生命值，并对落地点的所有敌人造成高额魔法伤害。该技能对小兵和野怪造成较少伤害。
*   **治疗公式**： 回复过去4秒内所受伤害总值的 **100/125/150% (+ 600% AP)**。
*   **伤害公式**： **150/300/450 (+ 150% AP)**。

## 连招与技巧

理解了基础技能后，掌握连招是发挥艾克威力的关键。以下是几套核心连招。

**基础消耗连招：E-Q-A**
利用E技能拉近距离，接Q技能确保两段伤害命中，最后用普攻触发被动加速后撤。

**无视野W先手连招：W-E-Q-A**
在视野外预判释放W技能，利用E技能位移进入W范围触发护盾和眩晕，接Q和普攻打出爆发。

**终极反杀连招：R-W-E-Q-A**
在残血被追击时，向安全位置释放W，用R技能回到过去的位置并造成伤害和回血，然后立刻E进W的领域触发控制，完成反打。

## 玩法思路总结

本节课中我们一起学习了艾克的技能机制与核心连招。

*   **对线期**： 主要使用 **E-Q-A** 连招进行无伤消耗，利用被动加速安全脱离。W技能可用于保护自己免受Gank，或配合打野进行控制。
*   **团战期**： 艾克的定位是侧翼刺客。优先瞄准敌方后排C位。可以利用R技能进行开团、撤退或重置状态，打出第二套技能。
*   **核心要点**： 熟练计算W技能的延迟时间和R技能的影子位置是玩好艾克的基础。他的操作上限很高，需要大量练习来掌握时机判断。

总之，艾克是一位集爆发、机动、控制和生存于一体的独特刺客。通过精准的技能释放和对时间的巧妙掌控，你就能用他在战场上主宰时钟。
