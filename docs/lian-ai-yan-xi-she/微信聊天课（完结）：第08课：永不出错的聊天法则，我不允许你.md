# 微信聊天课（完结）：第08课：永不出错的聊天法则，我不允许你

在本节课中，我们将学习一个能显著减少聊天错误的万能法则。这个法则的核心是“以牙还牙”，即通过模仿对方的聊天行为来建立平衡、舒适的对话节奏，避免过度付出或错误回应。

很多人聊天时会犯各种错误。有些人会说很多话，比如对方只是发一个表情包或“嗯嗯”，他就回复一大段话，恨不得分享当天发生的所有事情。

有些人会过度地不断发表情包，或者说“嗯嗯”来回应。有些人会过度共情，例如对方只是分享加班时遇到的一些事，并没有说加班辛苦，他们却会安慰对方“你好辛苦，心疼你”。

这样对方其实会觉得非常奇怪，对方会想“我没说辛苦，你干嘛说心疼我，真是莫名其妙”。这些错误会影响聊天质量，会让对方感到尴尬、不知如何接话，或者觉得与你聊天体验很差，不值得来往。

那么有什么方法可以减少我们在聊天中出现的错误？当然有。今天我就教大家一个万能聊天法则。只要你掌握这个法则，就能大大减少你在聊天中出错的频率。这个法则就是“以牙还牙”。熟悉我的同学可能都知道。

这个法则是我一直倡导的，是与男生相处时必须遵守的法则。也就是说，和男生相处，不需要想他脑子里想的是什么，而是要复制对方的行为。他怎么对你，你就怎么对他。他不怎么理你，你也不怎么理他。他和你聊天很冷淡。

那你就要停止聊天，不要继续下去。他对你很热情，你也要很热情。总而言之，你根本不需要动脑子，只需要复制对方的行为就可以了。这样做的好处是，既不用担心对方冷淡时自己会过度付出。

也不用怕对方对你有意思，你却因心里糊涂而错失机会。这个法则用在聊天上，也非常合适。

那么具体要怎么运用呢？以下是三个核心的应用要点。

**第一，对方讲事你附和，对方吐槽你共情。**
很多人并没有发现，我们平常所说的话分为两部分：**事件 + 情感**。
*   有些话只有**事件**，例如：“我今天晚上又要加班，这一周每天都要加班到晚上九点。”
*   有些话只有**情感**，例如：“我真的太讨厌我老板了，天天这样加班，烦死了。”
*   有些话则是**事件加情感**，例如：“我今天晚上又要加班…烦死了。”

运用“以牙还牙”原则，处理方式如下：
*   **别人说事情（事件）**，你就要**附和这件事**。例如：“又加班了？我一年加的班都没有你一个月加的多。” 或者询问细节：“最近业务很忙吗？怎么天天加班？”
    *   此时，你**绝对不能共情**，因为对方没有表达情绪。如果你共情了，就犯了情感过度的错误，对方会觉得莫名其妙。
    *   如果你还是想共情，**可以先询问对方的感受**：“天天这样加班，会不会觉得老板很讨厌？” 只有对方表达了感受（如说很辛苦），你才能去共情。
*   **别人在表达情感（吐槽）**，那你就要**共情**。一定要看到对方的情绪。例如：“天天这样加班，谁受得了啊！” 或 “天天让你加班那么晚，你老板也太可恶了。”
*   如果是**事件加情感**，那核心还是在表达感情，你也要**共情**。

这个方法最关键的是，你能判断出对方是在讲事情，还是在吐槽。毕竟“以牙还牙”，你必须知道“牙”是什么，才能够“还牙”。不要误判对方是在讲事情而不是在吐槽，就自作聪明地瞎共情。如果真的想共情，最好先问一下对方：“你的感受是怎么样的？”“你会不会觉得很难过？” 等对方表达完情感之后，你再来共情。

上一节我们介绍了如何根据聊天内容（事件或情感）来回应，接下来我们看看聊天节奏的把握。

**第二，频率与回复时间要对等。**
“还牙”的另外一个要点是，要看对方聊天的频率和回复时间，这方面也要尽可能与对方一致。
*   如果对方隔二十分钟才回复，那你也不要立刻回复，也隔二十分钟左右再回。
*   如果对方秒回，你也要尽可能秒回。
*   千万不要对方隔好几个小时才回，你却立刻就回了。那么对方就会觉得你很容易被打发，不会把你放在心上。
*   如果对方解释“刚刚跑去忙了，来不及回”，且态度比较诚恳，那你可以酌情快点回，但**最好也不要秒回**，毕竟也要让他长点记性。
*   一般来说，回复时间可以参考：他隔一小时回，你二十分钟后回；他两小时后回，你半小时后回，以此类推。
*   如果他隔了一天才回你，又没有任何解释，那你可以直接不回复了。
*   另外，如果对方发表情包，那你也发表情包就好了。

总而言之，你就看对方是怎么对你的，你复制对方的行为就好。

了解了内容和节奏的匹配后，有同学可能会产生一个疑问，我们接下来就解答它。

**第三，“回复时多说点”与“以牙还牙”不矛盾。**
看到这里，或许有同学会问：“你之前的课程让大家多说点，这样对方才能看到你给出的信息，才能更好地接话。这不是和现在所说的‘以牙还牙’矛盾了吗？”

这并不矛盾。所谓“以牙还牙”，是指**对方聊事情，你就要聊事情；对方分享情绪，你就要共情**。而不是说你回复的字数必须和对方一样，这就太狭隘了。
*   特别是如果对方问你问题，这代表他对你感兴趣，你就要**多说一点**，多分享一下自己的情况。
*   当然，如果你分享了自己的事情，对方却不拓展话题，那代表他不太感兴趣。那你就要**减少自己要说的东西**，或者考虑是否需要结束对话。

本节课到这里就结束了。我们来总结一下。

有一个永不出错的聊天法则，那就是 **“以牙还牙”**。
*   如果对方聊**事情**，你就聊事情。
*   如果对方分享**情绪**，你就要共情。
*   如果对方回复得**慢**，你也不要秒回。

总而言之，要重复对方的行为。**对方怎么对待你，你就怎么对待他。**
