## 认知的四个层次

理解知识并付诸实践是一个渐进过程。人的认知分为四个层次：

1.  **知道**：你知晓某个信息。
2.  **理解**：你真正理解了其中的含义或原理。
3.  **做到**：你能将理解转化为行动。
4.  **做好**：你能熟练、出色地完成行动。

如果你仅仅停留在“知道”，而不理解其背后的原理，你根本无法“做到”。“做好”则是最难的一层。本节课的目标是帮助你真正“理解”两性相处之道。

## 从“术”到“道”的转变

上一节我们介绍了认知的层次，本节中我们来看看思维与行为的关系。之前课程所学的可称为“术”，即具体的行为和表现。今天我们要探讨的是“道”，即你的底层思维。

**思维支配行为**。就像鼻子痒了，你会伸手去挠，而不是揉眼睛。你的思维模式决定了你在互动中的具体表现。

课件内容基于过往四、五年的亲身经历及从女性身上学到的知识。它虽不完美，但已足够接近本质。我希望你不仅阅读文字，更要用心体会字里行间的原理，思考“为什么会这样写”。
