## 概述：什么是惯例？

惯例是预先设计好的对话片段、小游戏或故事，用于在互动中创造趣味、展示价值或推进关系。然而，生硬地“丢出”惯例往往效果不佳。本节课的核心在于理解：**惯例本身是工具，而你的表达方式和衔接能力才是决定效果的关键**。

上一节我们探讨了建立吸引的基础心态，本节中我们来看看如何用具体的对话工具来实践。

---

## 惯例使用的三个阶段

根据个人成长经验，使用惯例通常会经历以下三个阶段：

1.  **懵懂阶段**：盲目尝试。拿到一个惯例就随意使用，不关心语境和对方的反应，只追求“使用”这个动作本身。
2.  **理解阶段**：优化表达。开始意识到表达方式的重要性，会为惯例加入自己的语气、节奏和情绪，使其效果放大。
3.  **内化阶段**：成为本能。很少刻意使用“别人的”惯例。你自身的谈吐和故事已经具备了惯例的效果，或者说，你的个人风格就是你的最佳惯例。

## 核心原则：表达重于内容

同样一句话，用不同的情绪和语气说出来，效果天差地别。

**代码示例：同一句话的不同表达**
```python
# 平淡的陈述
tone_neutral = "你这个人怎么这么讨厌。"
# 真正的愤怒
tone_angry = "你这个人怎么这么讨厌啊！" # 配合重音和急促语气
# 打情骂俏的暧昧
tone_flirtatious = "哎呀~你怎么这么讨厌啊~" # 配合拖长音和微笑
```

**练习建议**：尝试用“你吃饭了吗”这句话，分别带入生气、害怕、开心、伤心、激动五种情绪来说。这个练习能极大提升你对表达的控制力。

## 如何自然地引入惯例：需要一个“影子”

你不能生硬地开启一个惯例。你需要一个自然的“影子”或引子，将对话平滑地过渡到惯例上。

以下是引入惯例“0到100之间有多少个9”的示例：

1.  **影子**：先聊一个相关的话题。“你小学时数学好吗？”
2.  **过渡**：无论对方回答什么，都可以自然衔接。“我上周遇到件趣事，我邻居家六岁的小孩问了我一个特别简单的数数题……”
3.  **引出**：“她问我，0到100之间，一共有多少个9？还给我5秒时间思考。”

这样引入，比直接说“我们来玩个猜数字游戏”要自然有趣得多。

---

## 惯例后的衔接与互动

说完惯例不是结束，如何承接对话同样重要。这能避免冷场，并将互动推向更深层次。

**以“0到100之间有多少个9”为例：**

*   **互动设计**：当对方猜错时，可以邀请她：“你用手指数数看你是怎么算的？” 在这个过程中，可以自然地轻触她的手指进行引导（轻微的Kino身体接触）。
*   **话题衔接**：惯例结束后，可以将话题引向与惯例相关的生活观或童年回忆。例如：“你小时候数学不好，是不是因为太调皮了？” 或者直接分享自己的童年趣事。

## 深度惯例示例：生命与价值观

惯例“人一生有多少天”是一个能引发深度共鸣的例子。

**使用流程：**
1.  给出选择题：“凭直觉猜，人一生大概有多少天？A. 两万七千多天 B. 六万多天 C. 十万多天。”
2.  揭示答案：“其实，如果活到80岁，大约只有**29200天**（80 * 365）。即使活到100岁，也才**36500天**。”
3.  表达感悟：“这听起来很少。这让我思考，我们有多少时间是真的在为自己而活？……”

**价值展示**：紧接着可以分享你由此产生的真实人生态度（DHV - 展示高价值）。例如：
> “这让我觉得，与其把时间和金钱全花在一场给别人看的婚礼上，不如用这些资源去体验世界，为自己活一段时间。很多人追求奢侈，但我认识一些很成功的朋友，他们也并不比我们快乐多少。快乐的定义，因人而异。”

这样的分享，没有炫耀财富，却展示了**独立思考、独特的生活观和真实的阅历**，能产生强大的吸引力。

## 其他惯例要点

以下是课程中提到的其他惯例及使用要点：

*   **惯例“洗澡时唱歌”**：适合社交场合破冰。“据说92%的女生洗澡时爱唱歌，8%的喜欢……你猜她们唱什么？”（暗示对方属于特别的8%）。**注意**：更适合面对面使用，而非微信文字。
*   **惯例“瘦的好看，胖的好用”**：适用于对微胖女生表达好感。当对方问“好用是什么意思？”时，这是一个废物测试。可以坦诚而大方地回答：“我得承认带点‘好色’成分。但我对性的态度很开朗，我觉得**宁愿坦荡荡地‘色’，也不做阴沉沉地‘淫’**。” 关键在于说这话时自己要**确信不疑**。
*   **回答敏感问题**：当女生问“你有几个前女友？”时，这通常是她感兴趣的信号。回答时：
    *   可以反问：“你是真的好奇，还是随便问问？”
    *   如实回答数字无妨，但**切忌详细描述或评价前任**。
    *   最佳回应是：“**那段经历让我成长了很多**”或“**从她身上我学到了很多**”，然后自然转移话题。这显得你成熟、不纠结于过去。

## 总结

本节课中我们一起学习了：
1.  惯例是工具，其效果取决于使用者的**表达阶段**（懵懂、理解、内化）。
2.  **表达方式**远比惯例内容本身更重要，不同的语气和情绪能传递完全不同的信息。
3.  使用惯例需要巧妙的**引入（“影子”）** 和顺畅的**后续衔接**，以避免生硬和冷场。
4.  一些深度惯例（如“人一生有多少天”）是展示你**独特思想、价值观和生活态度**的绝佳窗口，这种吸引力远比单纯展示物质更为深刻和持久。
5.  面对敏感问题（如情史）时，保持**坦诚、简洁、不贬低不纠缠**的态度，是成熟的表现。

记住，最终极的目标是让有趣的对话和吸引人的特质成为你的本能，那时，你将不再需要“惯例”，因为你本身就已足够迷人。
