## 首要原则：得体与合身

我们讲的首要且非常重要的一点，是服装要**得体又合身**。

一位法国男士曾评论，中国女性普遍懂得得体优雅地搭配服装，但身边的男性却常穿着T恤、短裤和拖鞋，显得邋遢。这让他不解，因为在国外，不修边幅的男士很难找到伴侣。

据统计，超过90%的中国男性不知如何打扮自己。这凸显了学习穿衣法则的重要性。

## 核心法则：颜色搭配

除了合身的服饰，穿衣最重要的就是**颜色搭配**。

试想一个人从远处走来，第一眼注意到的是其服饰的**颜色**，而非拉链或袖口等细节。

服饰颜色搭配应以**大色块**为主。例如，若对腿型自信，可穿颜色鲜艳的裤子来凸显。

一个人身上的颜色**最多不要超过四种**，否则会显得杂乱。但也**尽量不要少于两种**，否则会显得单调。

**颜色搭配公式：2 ≤ 服装颜色数量 ≤ 4**

## 形象改造：从“头”开始

观察一个人的顺序通常是从头到脚。因此，面部是否精神至关重要。

女性可通过化妆修饰，男性则更依赖一款合适的发型。一个好的发型能显著提升颜值。

数据分析表明，椭圆形脸最适合国人。但并非人人天生如此，我们可以通过发型弥补脸型的不足。

人的脸型大致分为四种：方形脸、圆形脸、倒三角形脸和长形脸。以下是每种脸型适合的发型建议。

### 方形脸
面部轮廓较大，应避免中分和过度蓬松的发型，以免显得脸更宽。同时，头发不宜留得过长。

### 圆形脸
为避免显得迟钝呆板，发型应尽量向上梳，露出前额，这样会显得精神、有冲劲。

### 倒三角形脸
容易让人产生距离感。应避免将所有头发向后梳，可将刘海打薄，并避免头发过于蓬松，以免上半部分脸型显得过宽。

### 长形脸
容易显得忧郁和老成。应留长前额刘海并从中间自然分开，同时避免将刘海打薄，否则会显得脸更长。

## 基础单品配置

如果说女性化淡妆是对他人的尊重，那么在外打拼的男性也应注意仪容仪表。

以下是一些必备的基础单品，它们能组合出多种得体造型：

*   **一件深蓝色西装外套**：可休闲可正式，非常百搭。
*   **两双优质皮鞋**：颜色尽量选择黑色或深棕色，易于搭配。所谓千里之行始于足下。
*   **两件白衬衫与一件蓝衬衫**：衬衫是衣橱基石。
*   **两件Polo衫**：适合休闲场合。
*   **一条细款领带与一条窄身皮带**：用于点缀和修饰。
*   **一些黑色袜子**：避免坐下时露出不协调的袜子颜色。
*   **一条直筒蓝色牛仔裤**：经典且实用。

```text
基础单品清单：
1. 深蓝色西装外套 x1
2. 皮鞋（黑/深棕） x2
3. 白衬衫 x2， 蓝衬衫 x1
4. Polo衫 x2
5. 细款领带 x1， 窄身皮带 x1
6. 黑色袜子 x若干
7. 直筒蓝色牛仔裤 x1
```

## 裤装穿着要点

关于裤子，需要注意以下几点：

1.  **腰围合身**：裤腰应与自身腰围相当，不能太松或太紧。裤子很松却勒紧皮带，会显得滑稽；过瘦的人穿过于贴身的裤子，观感也不佳。
2.  **避免紧身**：体型偏胖的人穿紧身裤，容易让人产生不雅的联想。
3.  **长度适宜**：裤子不能太长，过长的裤脚会显得人邋遢、不够利落。
