## 理解升级的挑战与测试目的

通常在升级的阶段，你都会感觉到面红心跳，因为这是人类最原始的欲望需求。关系升级总是那么让人期待，因为这意味着你们可以进入暧昧，可以进行更多的亲密行为。

但是，别以为关系升级会一帆风顺。在我以前带过的线下课程中，很多学员在之前做得很好，但往往在升级环节会受到很多拒绝。当然，也有可能女生只是假性拒绝。

所以，你必须得分清楚她是真的拒绝，还是口头上的拒绝。如果是后者，你要当真，那你就错失了一次机会。

所有服从性测试都应该先由浅到深，从小到大。先从移动开始，再到转场。

## 转场的艺术与情绪带领

在转场上，你需要很多的理由，并且要充分地告诉她为什么要跟随你。假设在转场的时候她的朋友也在，那么你要照顾好她朋友的感受，而不能冷落她。

当你转场的时候要记住，每一次转场，都会因为环境的不同而导致人的情绪不一样。所以你必须去照顾好对方的情绪，带领她，别让她的情绪回落。

而且，每一次转场，都应该把你们的关系推进得更进一步。假设没有这个效果，那么证明你这次转场可能就不成功了。

在转场的时候，你应该让她尽量多跟你有身体的触碰，因为这是升级之前的一个铺垫。身体触碰越多，证明她对你的接受度也可能会越高。
