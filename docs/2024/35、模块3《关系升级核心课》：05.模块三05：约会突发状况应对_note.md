## 应对约会中的突发情况

### 突发情况概述

**突发情况**指的是计划之外的事情，在约会中可能会让人感到不舒服。

### 四类突发情况及解决方法

1. **女生放鸽子**

   - **原因**：女生可能因为对男生的吸引力不够，或者临时有事而爽约。
   - **解决方法**：在约会前一天晚上确认约会，避免女生爽约。

2. **女生遇到突发情况**

   - **原因**：女生可能因为工作、家庭等原因无法赴约。
   - **解决方法**：理解女生，并敲定下次约会。

3. **遇到酒托**

   - **防范**：在约出来之前，通过社交软件资料、聊天内容等判断对方是否为酒托。
   - **解决方法**：选择熟悉地点约会，避免被酒托诱导。

4. **天气变化**

   - **预防**：查看天气预报，选择合适的约会地点。
   - **解决方法**：备好雨伞，确保女生舒适。

5. **等位**

   - **预防**：提前了解餐厅情况，选择合适的餐厅。
   - **解决方法**：准备两套计划，避免尴尬。

## 约会收尾

### 主动离场

- **目的**：避免女生掌握关系主动权。
- **方法**：在约会氛围高涨时，主动提出离场。

### 表达感受

- **目的**：帮助女生合理化关系。
- **方法**：表达对女生的喜欢和欣赏。

### 基本礼节

- **目的**：体现对女生的关怀。
- **方法**：送女生回家，并嘱咐其安全到家。

### 回去后发消息

- **目的**：巩固约会成果。
- **方法**：根据女生兴趣度，发送表达感受的消息。
