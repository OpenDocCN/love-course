# 凡哥私教课程：魅力男士3.0：011.11、聊天技巧之话题的开启与延续

🎼在之前的课程中，我们主要讲解了如何选择话题。选择话题并非仅仅基于你拥有的话题或女孩身上的话题，而是要根据当前情境选择合适的话题。

**核心概念**：选择话题应根据当前情境。

🎼举例来说，许多兄弟在刚加女生微信时，想要追求她，确认关系，但这往往无法得到理想结果。

**核心概念**：追求女孩应从展现吸引力、破冰开始。

🎼在聊天中，话题的选择也应遵循这一原则。以下是与女孩相处的三个阶段及相应话题：

1. **初识阶段**：可聊第三方话题，如共同认识的人、电影、美食、宠物、旅行等。
2. **熟悉阶段**：可聊个人信息类话题，如年龄、工作、家庭、兴趣爱好等。
3. **暧昧阶段**：可使用暧昧、调情技巧，快速升级关系。

**核心概念**：话题选择应根据与女孩的相处阶段。

🎼开启话题的方式有两种：

1. **关于你的话题**：通过图片、文字表达你的状态、感受，并询问对方状态。
2. **关于对方的话题**：使用冷读法，假设一个观点，询问对方信息。

**核心概念**：开启话题的方式包括表达自己、询问对方。

🎼延续话题的四个技巧：

1. **抓关键词**：找出对方聊天内容中的关键词，围绕关键词进行联想。
2. **上推**：将话题变得更加广泛、抽象、宽广，找到其属性分类。
3. **下切**：将话题进一步细分、具体化，如时间、地点、人物等。
4. **平行**：聊与话题相关或类似的事情。

**核心概念**：延续话题的技巧包括抓关键词、上推、下切、平行。

🎼使用这些技巧时，应注意以下几点：

1. 下切和平行方法较为常用，上推有一定难度，需谨慎使用。
2. 上推相当于冷读，需根据掌握的信息进行判断。
3. 初始阶段尽量使用抓关键词、下切、平行等方法。

**核心概念**：使用延续话题技巧时，注意方法选择和时机。

🎼总结：本节课介绍了话题的开启与延续技巧，帮助兄弟们在聊天中更好地展现自己，吸引女孩。希望兄弟们能够将所学知识应用到实际生活中，提升自己的聊天能力。
