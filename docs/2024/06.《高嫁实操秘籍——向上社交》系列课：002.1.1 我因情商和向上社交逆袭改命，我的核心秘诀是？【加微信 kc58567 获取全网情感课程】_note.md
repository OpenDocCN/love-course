# 06.《高嫁实操秘籍——向上社交》系列课：002.1.1 情商与向上社交的核心秘诀 🧠

在本节课中，我们将学习高情商与向上社交的核心秘诀。我将分享两个关键公式，它们是我实现人生逆袭的基础。课程将首先深入探讨高情商的本质与修炼方法，然后解析向上社交的必备素质与常见误区。

## 第一部分：高情商攻心计

上一节我们介绍了课程的整体框架，本节中我们来看看高情商的核心究竟是什么。

一个人的情商至关重要。哈佛心理学博士丹尼尔·戈尔曼在《情商》一书中写道：**你让人舒服的程度，决定你所能抵达的高度**。得到的创始人罗振宇也曾指出，在当今社会，最重要的能力之一是语言表达能力。高情商不仅是自我提升，更是决定人生高度与广度的关键技能。

许多人误以为高情商就是会说话、性格好或圆滑处事。事实并非如此。**高情商是能够满足自己又能够成全别人的能力**，这是一种双赢的能力。

### 高情商的底层秘诀：成为高能量的人 🪐

高情商的底层秘诀，是你要成为一个高能量的人。只有自身能量充足，才能在满足自己的同时成全他人。如果自身能量耗尽，便无法赋予他人能量。

如何培养高情商？首先需要纠正一个误区：高情商不等于好说话或迁就他人。**高情商是优雅且有个性的，它不等于老好人，更不是讨好型人格**。相反，高情商的人非常清楚自己的需求，可以说是聪明的利己主义者。

演员黄渤是公认的高情商典范。尽管其外貌并非传统意义上的帅气，但其人格魅力深受认可。他的应对方式总能让人感到舒适。

*   在讨论年龄时，他将“最老的”转化为“最大的”。
*   面对闫妮“这次跟你演夫妻，我知道自己要进入丑星行列了”的调侃，他回应：“那我的心态可跟你大不一样啊，我就是想靠你走进帅哥行列的，你可别拖我后腿。”这句话既化解尴尬，又赞美了对方。
*   被问及观众为何喜爱他时，他说：“可能我跟他们身边的朋友差不多，没有什么危险感跟压力感。观众看到我想到自己啊全是优越感，这不挺好的吗？”他将观众捧高，赢得了好感。

高情商的人懂得说话的艺术，能带给他人能量，让对方舒服的同时，自己也收获好感与尊重，实现双赢。**高情商从来不以牺牲自我为代价**。

### 修炼高情商的系统方法：萨提亚冰山模型 🧊

高情商是可以培养的后天技能。其核心在于理解彼此深层次的需求。那么，如何洞察自己与他人的内心世界呢？

这里需要引入著名的**萨提亚冰山模型**。该模型指出，我们的外在行为只是冰山露出水面的一角。水面之下，隐藏着我们的**应对方式、感受、观点、期待、渴望和自我**。简单来说，影响我们外在行为模式的，正是内在的关系模型。

这个模型可以解答大部分关系难题，例如伴侣沟通、高端场合发言或与行业大佬交流。

我们可以将人际关系想象为三层冰山：
1.  **表象关系层面**：外在行为。
2.  **需求层面**：行为背后的真实感受与观点。
3.  **脚本层面**：内心深处的渴望与自我认知。

用最简单的话说，就是**说话之前要过脑子，凡事三思而后行**。别人说出一句话，你的应对方式源于你的感受，但很多人只关注感受而脱口而出。

以下是一个运用冰山模型分析情侣吵架的案例：
*   **男生视角**：
    *   **行为**：因打游戏未及时回复女友。
    *   **应对方式**：道歉 (“对不起，我错了”)。
    *   **感受/观点**：认为女友大惊小怪，自己只是没听见；觉得男生爱玩游戏很正常。
    *   **渴望**：渴望被理解、被支持；认为男生需要有个人爱好。
*   **女生视角**：
    *   **行为**：因男友打游戏不理自己而生气。
    *   **应对方式**：生气、黑脸、质问。
    *   **感受/观点**：感到失望、沮丧；认为在男友心里游戏比自己重要，对方不够爱自己。
    *   **渴望**：渴望被爱、被重视，希望自己是对方的唯一。

通过模型拆解可见，男女双方的内心诉求不同。如果彼此了解对方深层次的需求（男生渴望理解，女生渴望被重视），矛盾可能就不会升级。

冰山模型的第二层（需求层面）是关键。**所有关系的表象都体现在沟通内容里，其本质是看人们的需求是否得到了满足**。高情商的底层逻辑，正是让**自己和对方的需求都能得到满足**，实现共鸣。

高情商也是融化冰山的过程。它让我们说话做事三思而后行，更能体会自身感受，了解内在观点，找到期待，接纳渴望，探寻深层自我，从而达成互相理解与成就。

### 修炼高情商的四大核心能力 🔧

了解底层逻辑后，我们来看看如何在短时间内修炼高情商。其核心要点是：**你的脑子要比嘴巴快，控制力要比行为强**。嘴比脑快的人容易得罪人而不自知。快速回应不等于不假思索地脱口而出，因为言语力量巨大，可能直接改变他人对你的看法。

这里的控制力，主要指**情绪管理能力**。情绪把控力越强的人，收获的机会和善意越多。情绪管理至关重要，我们将在后续课程中详细探讨其底层逻辑与实操技巧。

在修炼高情商的过程中，必须具备以下四大能力：

以下是高情商修炼的四大必备技能：

1.  **做那个懂他的人**：感知并揣摩他人内心情感的能力。目标是成为对方的知己，而非普通朋友，让对方产生相见恨晚之感。
2.  **让耳朵代替嘴巴**：学会沟通的第一步是学会闭嘴。学会倾听，才能了解对方的真实感受，有时倾听本身就能满足对方被尊重的需求。通过倾听，你才有时间辨别对方的真实需求与内心脚本，从而对症下药。
3.  **拥有“社牛”的觉悟**：这不是让你变成社交狂人，而是要具备人际关系适应与融入圈子的能力。在高端的社交场合，关注点不应仅限于菜肴或环境，而应观察他人、寻找可深度链接的对象、思考如何脱颖而出展示自己。要珍惜每一个机会。
4.  **开启你的“八卦雷达”**：这里的“八卦”指察言观色的能力。要懂得见什么人说什么话，提供不同的舒适度。例如，与伴侣相处提供甜蜜放松感，与同事相处提供信任合作感，与高层次的人相处则需谦逊、尊重并带有探索欲。开启雷达，才能精准打开与对方对应的互动模式。

### 核心秘诀公式一 📐

说了这么多关于高情商的话题，那么我的核心秘诀之一是什么呢？

**公式一：高情商的修炼 = 高能量 + 融化冰山 + 巧妙表达**

*   **高能量**：提升自身能量，才能赋予他人能量。
*   **融化冰山**：了解他人（及自己）的底层需求。
*   **巧妙表达**：需求反映在行为上，需要三思而行，好好说话。

掌握这个公式，你将能解决大部分人际关系问题。

## 第二部分：向上社交的进阶之道

上一部分我们深入探讨了高情商，接下来我们聊聊向上社交，其中也蕴含了我的第二个核心秘诀。

我们常说的向上社交，英文是“social climbing”，即为了提升社会地位而进行的努力。其必备技能正是高情商。向上社交不仅常见于职场交际场，在择偶与婚姻生活中也同样需要。

成年人的社交以价值为导向。圈层决定你的层次与格局。一个人的交际圈中最好有几位不同领域的佼佼者，这对提升自我和拓宽眼界是绝佳的捷径。有句话叫“依靠大树好乘凉”。与比你厉害的人建立良好关系，对你的人生大有裨益。选择大于努力，千里马常有，而伯乐不常有。自身有才能且勤奋，还需要遇到赏识你、愿意与你合作或提拔你的人，才能在成功的路上少走弯路。

### 向上社交的必备素质与条件 🧗

想要进行向上社交，你必须具备以下几个素质和条件：

以下是向上社交必备的三项核心素质：

1.  **看世界的能力**：这不一定要你 physically 走出去，但眼界必须打开，精神或脚步，总要有一个在路上。许多人知道要自我提升，却停留在“想”的层面，行动力不足。无论向上社交还是经营人生，行动都必不可少。终身学习能带来无穷可能，也能让你区别于他人。只有有趣的灵魂才能走入人心。拥有看世界的能力，才能不断打磨自己的实力与价值。你看过世界，听过世界，才能让别人觉得你值得交往，因为你有“量”。
2.  **每次沟通都要具有目的性**：正如在高情商部分提到的，每一次接触高层级圈子和机会都尤为重要。无论是简单聊天还是一顿晚餐，你都要200%打开你的雷达，将其视为吸收养分的机会。例如，巴菲特午餐拍出天价，精英们并非只为吃饭，而是为了收获能量、知识与潜在的机遇。因此，要多参加有档次的社交活动，接触有价值的人。每次交流都有价值，面对观点不同的人，可把自己当作海绵，吸收对方的能量为己所用。
3.  **把自己当做商品推销出去**：每一次亮相都是一次参展。你要放大自己的优势。如果不知道优势是什么，就去挖掘，将优点写在纸上并不断放大，直到别人无法忽略。电视剧《风吹半夏》中，赵丽颖饰演的许半夏便是典范。她身处男性主导的行业，凭借自身优势（如女性的细腻与坚韧）主动出击。在社交场合，她大方爽快、不卑不亢，勇于营销自己的专业与能力，让人觉得她靠谱、值得合作。千里马遇到伯乐的前提，是先让伯乐认识并了解你。遇到贵人的前提，是你有被帮助的价值。

### 向上社交的常见雷区 ⚡

我们说了三点向上社交的必备素质，接下来我来说说向上社交的一些雷点，你要记住，千万不要踩雷。

以下是两个必须避免的向上社交雷区：

1.  **“捞女”混不进高端圈**：向上社交的出发点是自我成长与协助成长。如果想通过向上社交实现躺平，那是不可能的。“捞女”只能作为高端圈子的调味剂或玩物，不可能成为平等的伙伴，更没有并肩的资格。向上社交的发心必须是正向的，抱着学习、赋能、价值交换的心态，才有可能实现圈层跨越。
2.  **“舔狗”没有春天**：感情中如此，向上社交亦然。正如开篇所说，高情商不等于委曲求全，向上社交也不意味着持续低姿态。与比你厉害的人打交道，无需卑躬屈膝。经济学中的“橱窗效应”指出，商品摆放越显眼、标价越高，人们越觉得其有价值。**你把你自己放在什么位置，你就拥有多大的价值**。所以，要让自己成为高端的展品，看起来很“贵”，让对方觉得你有利可图、有用。你们是平等的、相互价值交换的关系。越卑微的姿态，只会越拉低自身的价值。

### 核心秘诀公式二 📐

说了这么多关于向上社交的内容，大家有猜到我的第二个秘诀是什么了吗？

**公式二：向上社交 = 终身学习 + 目的性社交 + 营销自己**

*   **终身学习**：这是一切的基础，内涵才能支撑你走得更远。
*   **目的性社交**：每次交流都是吸收养分的过程，必须有收获。
*   **营销自己**：时时刻刻展示自己的优势，让别人记住你。千里马也要让伯乐看到，才有可能成为千里马。
