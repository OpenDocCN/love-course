# 34、米娅情感强势认知：02.本课程中“强势文化”的理解 📚

**概述**：在本节课中，我们将探讨“强势文化”的概念，并将其应用于情感关系中。

**强势文化与弱势文化**

1. **强势文化**：遵循事物规律的文化，强调在尊重人性的基础上，通过遵循情感发展的客观规律，合理评估自己的选择权，并在可操控的范围内主导自己情感的走向。

   **公式**：强势文化 = 尊重人性 + 遵循规律 + 合理评估 + 主导情感

2. **弱势文化**：依赖强者的道德，期望破格获取的文化，包括拯救、感动、纠缠等被动行为，期待与不匹配的伴侣。

   **公式**：弱势文化 = 依赖强者 + 破格获取 + 被动行为

**强势文化与弱势文化的区别**

1. **定位清晰**：具有强势文化思维的人对自己的定位清晰，思考的是我能得到什么样的伴侣，而不是我想得到什么样的伴侣。

2. **提升与改变**：为了接触更优秀的异性或进入更高级的圈层，需要做出提升和改变，来匹配自己的需求。

3. **能力与理想匹配**：现实往往是能力和理想不匹配，因此要承认自己的普通，并努力提升自己。

**总结**：本节课中我们一起学习了强势文化与弱势文化的概念，以及它们在情感关系中的应用。希望这能帮助你更好地理解自己在情感关系中的位置，并做出更明智的选择。
