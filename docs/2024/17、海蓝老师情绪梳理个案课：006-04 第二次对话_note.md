# 海蓝老师情绪梳理个案课：006-04 第二次对话

## 概述
在本节课中，我们将学习如何通过情绪梳理个案课帮助个体理解自己的情绪，并学会如何表达和处理这些情绪。

## 第一次对话回顾
**以下是第一次对话的回顾：**
- 个体表示自己不再干涉他人，对 baby 微说“没有了”。
- 个体表示听到妈妈说了很多后，感到平静，但只有2%理解。
- 个体表示对前男友的一些做法不认同，感到愤怒。
- 个体回忆起与妈妈的对话，表示自己总是适时的诸葛亮，不知道当时的情况。
- 个体表示自己与前男友分手后，妈妈曾预言他们不可能在一起，结果被言中。
- 个体表示妈妈总是认为自己是对的，别人是错的，干涉自己的恋爱。
- 个体表示妈妈不理解自己，总是以自己的老思想看待问题。
- 个体表示自己高中时成绩下降，妈妈曾建议让妹妹帮忙，但自己感到不被信任。
- 个体表示自己从小到大一直焦虑，朋友都没有轻松快乐地过一天。
- 个体表示自己因为初三时得了急性肠胃炎和窦异性动梗速，一直觉得自己有病。
- 个体表示自己小时候一直学习、做家务、照顾弟弟妹妹，感到很累。
- 个体表示自己与妈妈的关系紧张，妈妈总是干涉自己的事情。
