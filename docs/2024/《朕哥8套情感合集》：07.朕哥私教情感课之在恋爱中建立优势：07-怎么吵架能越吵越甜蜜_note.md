在本节课中，我们将探讨情侣间一个普遍却棘手的问题：吵架。我们将分析吵架的本质，并提供具体方法，帮助你将冲突转化为增进感情的契机，实现“越吵越甜蜜”。

## 一、吵架的本质：源于爱与需求

上一节我们介绍了建立优势的总体思路，本节中我们来看看如何具体处理冲突。几乎每一对情侣都会吵架，热恋期会吵，结婚后会吵，有了孩子还会吵。有些情侣床头吵架床尾和，也有些情侣吵着吵着就分手了。这其中存在奥秘。

首先要搞清楚一个问题：你们为什么会吵架？你们在生活中经常吵架吗？你们会跟父母、孩子、老公、闺蜜吵。你们会跟一个陌生人吵架吗？

你们会跟同事领导吵架吗？你肯定更容易选择离职而不是吵架。因为你跟他们的关系不够亲密。因为不够亲密，所以能以礼相待，所以不会吵架。这就是人的通病，对陌生人很有礼貌，对自己亲近的人却不管对方感受，把自己的想法强加给对方。

想一想你和老公相识相爱的过程。一开始相敬如宾，有需求会很礼貌地提出来。然后到了相爱会直接提出来。再到最后他不给你就吵着要。

我们现在至少知道了一点：**吵架来源于彼此的相爱**。如果你们不爱了，彼此就会变得冷漠，根本不可能吵架。吵架说明彼此之间有需求。吵架无非表达出来两个意思：
1.  **我需要你。**
2.  **我很难受。**

## 二、谁在吵架中更“凶”？往往是弱者

我再问大家一个问题：吵架时是谁声音高，谁就有理吗？不对。恰恰相反，不合理的一方往往声音更大。他想用声音来弥补自己的心虚。

更进一步地说，**吵架的发起者往往都是弱势的一方**。

以下是女生爱吵架的常见原因总结：
*   因为男生天天忙工作不理自己而吵架，那是你怕对方忘了自己。
*   因为婆媳关系而吵架，那是你怕他向着他妈而不向着自己。

所以说，每次吵架都是因为你在潜意识里认为自己是弱者，你觉得自己的地位比对方低，从而以吵架来提高自己的地位。吵架的结果，即便是男人哄你了，也只是表面上你的地位提升了，并不能实际解决问题。

当然很多姐妹都不承认这一点。那是因为人的心理有防御的本能。你的潜意识里是“我需要你，我害怕”，但是你表现出来的却是攻击性：“你为什么不来陪我？你不爱我了吗？你还是不是个男人。”

所以，**吵架就是自我防御的体现**。我想你了，我不会说“我想你”，因为你会觉得我很弱小。我会骂你凶你，这样你回头对我道歉了哄我了，我的目的就达到了，也没有丢面子。

有些男生在追女孩的时候，他喜欢女孩，反而会故意捉弄女孩，会让女人生气或者担心。就像小学和初中的时候，有些男孩会将毛毛虫放到女生的铅笔盒里。这其实是一种掩饰，他为了掩盖自己喜欢对方而故意捉弄对方的行为。

所以说，男女吵架，往往是最凶的那个人，反而是弱者。因为他需要用凶狠来掩饰自己内心的虚弱。所以当你跟男人对你特别凶的时候，你不要害怕，不要害怕他会抛弃你。他对你凶，说明他还爱你，说明他还需要你。他对你凶是想让你对他温柔。

## 三、吵架与爱的辩证关系

我之前听到一种说法，叫做“吵架就如同做爱”。因为这两者有很多相同之处，比如情绪都比较激动，都会面红耳赤。其实这两者的动机也都是相同的，都是因为爱。如果没有爱，那么男女就不会做爱。如果没有爱，两个人就更不会吵架。

所以每次当你们吵架时，一定要记得这一点：**你们在用伤害对方的方式索取爱**，忘了彼此应该是相爱的一对情侣。

打个比方，就好像两个人在做买卖，买的人想买，卖的人也想卖。一开始和和气气，结果为了折扣吵起来了。买的人就强行要求给自己更低的折扣。最后两个人打起来了，都受了伤。

情侣吵架大多数的情况也都是这样的，都是走得太远了，以至于忘了自己为什么要出发。当你明白了这个道理，你就会从根本上减少吵架。当男人找你吵架的时候，你也懂得包容。

## 四、让吵架不伤感情的六个“不要”

那么吵架要具体怎么做，才能不伤害彼此的感情？那就需要注意以下六点。

以下是六个需要避免的雷区：

1.  **不要上纲上线，扩大矛盾。**
    女人吵架经常加一些脑补的内容。比如男人没有回微信，她就会质问“你是不是不爱我了”。男人的社交聚会有异性，她就会威胁“你要是敢找小三，你就完了”。男人工作忙经常不陪女人，她就会抛出一句“既然不爱了，那我们就分手吧”。其实本来是一些小问题，他道个歉，你包容一下他，然后彼此拥抱一下就能解决。而你非要挑起矛盾，让彼此吵起来。你作为女人为什么不能温柔点呢？
    *   **核心原则**：无论争吵多么激烈，矛盾多么激化，都不要提“分手”或“离婚”。提分手最伤害感情。也许提一两次男人还能忍，但你提多了，他总有一天会累。当他说出“分手就分手”时，感情就无可挽回，你也没有台阶下了。
    *   **小矛盾不要上升到“爱不爱我”**。一个男人如果不爱你了，他会一言不发地转身离去。如果他还在你身边陪你哄你，就说明他还爱着你。你问他爱不爱你，其实是在质疑他。没有哪个男人喜欢被怀疑。
    *   **不要拉黑微信、电话**。这样做等同于说分手。你这么做让男人没有办法给你解释。而且你拉黑了过几天又要重新拉回来。有时候男人不过来哄你，你还得自己拉回来。你这样岂不是更没有面子、更尴尬？你又是何苦呢？女生大多数都是情绪化的，而且想象力丰富，尤其是在吵架情绪激动的时候。你就应该在吵架时格外注意控制自己的情绪。

2.  **不冷战，不逃避，双方找到解决问题的方法。**
    吵架后，女人会选择回娘家、搬到闺蜜那里住、和男人冷战、不回复信息、挂断电话。女人的想法是男人不道歉，自己就把问题拖着不解决。这样做只能让矛盾更加激化。
    *   **正确办法**：当天吵的架，当天解决。比如双方可以商量一个条约：吵架不能超过多长时间。比如吵了一个多小时，有一个人说“吵架的时间到了”，双方冷静后再解决问题。当天晚上两个人同床共枕、彼此拥抱亲密的时候，就要把吵架的事情解决了，不要让这件事留在彼此心里。

3.  **不要翻旧账，贴标签。**
    之前有过的事情都已经过去了。凡是男人道歉过的事情，你就不要再提。翻旧账只会让男人越来越讨厌你。人都是成长的，他以前什么样，不代表以后什么样，所以也别给对方贴标签。

4.  **不要触碰男人的底线。**
    吵架绝对不能触碰男人的底线。比如你骂他可以，但是不能骂他的父母和家人。也不能有侮辱他性能力等方面的内容，也不能侮辱到他的道德人格。经济能力也最好不要去嘲讽。

5.  **不要给自己拉外援。**
    吵架了别拉着自己或对方的父母、闺蜜、朋友等人一起对付男人。你这种拉外援的行为会将矛盾从两个人的对抗上升到两伙人的对抗。也不要把孩子拉扯进你们的矛盾里。

6.  **不要选择伤害自己或对方的方式。**
    吵架千万不要动手。如果他动手，你也不要还击，直接分手就好了。你吵架的时候也不要先动手。有些女人会掐、会捏，甚至会吐口水，这些都是很严重的破坏关系的行为。另外，你也不要通过自残的形式来对抗，比如割腕、绝食等行为。以这种行为去挽回男人，无异于饮鸩止渴。

## 五、如何让吵架增进感情：三个关键行动

姐妹们，你们要记住这样一句话：**男人和你吵架，你没有必要生气。因为他如果爱你，你不应该生他气；如果他不爱你了，对于一个不爱你的人，那就更没必要生气了。** 这句话可以说是金玉良言。

那么在现实中，很多女生和伴侣也经常吵架，但是他们却越吵感情越好，吵吵闹闹地结婚了，吵吵闹闹地生孩子了，到最后你都能感觉到他们吵架简直就是在撒狗粮。这些女生是怎么做到的呢？

吵架是两个人的事情。下面我讲的内容，你只能自己想办法做好自己的部分，而不是将这些讲给你的伴侣听然后让她做，那样你们的吵架可能就升级了。如果你能够按照我说的做法去做，你就可以潜移默化地影响你的伴侣。

如果你想让吵架能够加深你们的感情，请注意以下三点：

以下是三个将冲突转化为亲密感的行动指南：

1.  **就事论事，速战速决。**
    如果你们不可避免要吵架，就尽量在最短时间内吵出个结果来，即通过吵架找到解决问题的办法。这样，你们吵的就是事情本身，而不是彼此的态度。那些吵架反而能加深感情的情侣，都是吵一次架就解决一个问题，而不是一件事一直吵、越吵越凶。想想看，情侣之间的矛盾再怎么多，都是有限的。如果吵一次就能解决一个矛盾，那么矛盾只会越来越少。如果你不知道怎么吵是能够解决问题的吵架，有一个办法：你可以去观察那些相处了很久的情侣是怎么吵架的，尤其是那些三四十岁的中年人。他们也会偶尔吵架，但是同一件事情绝不会吵第二次，因为吵第一次的时候，问题就解决了。

2.  **用“反向表达”建立情感链接。**
    要让他知道，你对他凶其实是对他好。“打是亲，骂是爱”这个道理大家都知道，但现实中很难做到，情绪上来了，理智就糊涂了。你吵架时对他说“我说你是为你好”，基本等于白说。那该怎么说呢？你可以在平时不吵架的时候，用“骂”他的口吻去讲夸他的话。
    *   **代码示例（情感表达模式）**：
        ```plaintext
        日常场景 + 看似指责的语气 + 实际夸奖的内容
        例如：“你怎么每次说的都是对的，感觉我跟你在一起就变成了白痴。”
        “你说你怎么这么讨厌，嘴真甜，用甜言蜜语骗了多少小女生？真可惜我上了你这条贼船。”
        ```
    这样他就知道，你“凶”他其实是爱他。他跟你吵架的时候，心里就不会那么反感。与此同时，你说他的缺点、跟他提要求的时候要特别温柔。你这样夸他的时候“凶”，提意见的时候温柔，这样一个反转会有意想不到的效果。男生一定会觉得你古灵精怪，等吵架时，他的生气就会减少很多。

3.  **及时修复，将裂痕转化为加固点（最重要）。**
    就是要及时弥补吵架留下的伤痕。如果吵了一次架而且比较凶，你一定要及时弥补感情伤口。吵完架之后，一定要亲密一下。感情伤口愈合以后就不会有问题了。最简单的方法：小两口床头吵架床尾和。如果你和他同居，那么晚上亲密是愈合吵架的最好方法。其他的比如出去看个电影、吃个烛光晚餐，亲密你总是会了吧。
    *   **这样做另一方面也是提高了你们的感情免疫力**：吵架 -> 亲密 -> 再吵架 -> 再亲密。这样就像坐过山车一般的感觉，就好像吃了苦药之后再吃蜂蜜，蜂蜜就会变得更甜。那么你们的感情是不是就会更亲密了呢？而且这也相当于一次一次给你们的感情打预防针。之后你们发生更大矛盾的时候，你们也不会分手，相当于骨折之后的骨骼会更坚固，是一个道理。

本节课中，我们一起学习了如何理解和处理恋爱中的争吵。

最后，也是最重要的，我们掌握了三个将冲突转化为亲密感的积极行动：**就事论事速战速决、用反向表达建立情感链接、以及通过及时亲密的修复将关系裂痕转化为加固点**。
