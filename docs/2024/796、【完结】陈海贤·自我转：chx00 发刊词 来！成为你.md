# 796、【完结】陈海贤·自我转：chx00 发刊词 来！成为你

在本课程中，我们将开启一段关于“自我转变”的旅程。这门课程旨在帮助你在充满不确定性的生活中，找到并成为那个你真正想成为的自己。

你是否曾感到纠结？或许你想转换职业赛道，却不知如何迈出第一步；或许你考虑离职，却难以最终下定决心；又或者，你刚刚经历裁员，正在苦苦思索未来的方向。

关系的困扰也可能缠绕着你：是该继续一段关系，还是该选择离开？如果已经离开了一段不适合的关系，又该如何构建新的生活？

也许你正坚持着一个梦想，同时承受着它带来的痛苦；也可能你的日常生活充实幸福，但偶尔会觉得过于一成不变，在某个瞬间感到迷茫，不禁思考：什么才是自己真正想要的？自己的未来究竟会是什么模样？

这些故事的主角，可能是你身边的人，也可能就是你自己。

我是陈海贤，一名心理咨询师。我曾与许多面临上述困扰的来访者一同工作。这些烦恼为何会发生？表面上看，是他们遇到了具体的生活难题。

而更深层的原因在于：每个难题的背后，都有一个“旧的自我”不肯离去，同时一个“新的自我”渴望诞生。正是这对矛盾，推动着转变的发生。

如果你因为害怕而抗拒转变，或者因为不知如何转变而停止成长，你就会陷入停滞。从某种意义上说，许多我们熟知的心理问题与人生困境，都是“转变受挫”的产物。

此前，我在其他平台讲授过《自我发展心理学》、《亲密关系》及《家庭关系》等课程，累计有数十万同学参与学习。近年来，我不断遇到课程的同学，他们向我提出了许多关于人生转变的问题。

我也持续进行着与转变相关的工作：心理咨询、写作、举办转变主题的训练营。我与许多经历过转变的人深入交谈，汲取他们的经验，也帮助许多卡在转变期的人顺利度过难关。

现在，我将这些关于转变的理解与经验，整合成一门全新的课程呈现给你。这门课既是《自我发展心理学》的延续，也是一个全新的开始。《自我发展心理学》以“知识”为核心。

而这门《自我转变》课程，则以“转变的经验”为核心。它当然也包含知识，但这些知识是为了帮助你整理自身经验、寻找出路。

具体而言，本课程将聚焦于“转变”这一人生重要主题。如果你已经从宏观上了解人生发展，但仍对当下感到疑惑，并对更好的自己心怀期待，那么这门课程就是为你设计的。

在课程中，我会将转变发生的心理历程置于“显微镜”下，帮助你仔细审视转变过程中的每个关卡与每种出路。

我曾有一位学员，原在一家自媒体公司担任编辑，最近他辞职去追求自己一直想做的事。过程经历诸多曲折。他告诉我：“陈老师，当初我为是否辞职而严重内耗时，是您告诉我，为了养家糊口做一份不那么喜欢的工作，其实也是一种勇气。这让我在做准备的同时，也接纳了自己。所以现在走出来，我才没有那么慌乱。”

这正是本课程希望达成的目标之一：**如果你想转变，课程将帮助你了解转变的节奏与过程，在面对选择时减少内耗，从容应对。**

我还有一位学员，硕士毕业后进入一家研究所工作，这听起来是一份理想的工作。但他逐渐发现所从事的研究让他感到失望。经过漫长而痛苦的思考，他最终决定辞职去攻读博士学位。

常有好心人提醒他：“你辞职读博，毕业以后未必能再回到我们研究所。”但他内心非常坚定。他说：“我去读博，不是为了回来工作，而是为了做我自己想要做的那种研究。”

他完成了关键的一步：**新旧评价体系的切换**。这为他找到了内心的锚点。这是本课程希望达成的另一目标：**在你真正迈出转变步伐时，能够将人生的开关掌握在自己手中。**

转变为何如此艰难？许多人认为，这是因为决心不够坚定、没有找到正确的方法或方向。仿佛只要按下一个神奇按钮，就能换好工作、处理好关系，转变在一夜之间完成。

其实并非如此。如果深入观察，你会发现，在新自我取代旧自我的过程中，竟然发生了那么多事情。不同人的转变有其相似性：他们都不是仅仅解决了一个具体问题，而是走完了一条完整的道路。但很多人身在其中时，却很难意识到这一点。

因此，如今当有人问我关于人生转变的问题时，我不会再试图给出一个简单的答案。我会说：**“不要寻找答案，而要寻找道路。”**

在课程里，我为你梳理了这条道路上的**4个阶段**与**15个关键节点**，勾画出一幅详细的“自我转变地图”。

以下是转变的四个核心阶段：

第一个阶段叫**响应召唤**。我会告诉你，当你犹豫不决时，如何确定“内心的我想要”，如何抓住机会，走出转变的第一步。

第二个阶段叫**脱离旧自我**。我们会对过去的关系、过去的成就乃至过去本身有很多留恋，但只有与之告别，才能继续向前。课程将详细告诉你如何告别。

第三个阶段讲**踏上新征程**。站在新旧自我的交界处，你可能会感到迷茫甚至想要放弃。课程将给出答案：如何寻找和利用旧自我的资源，以及如何采取有效行动探索新的自我。

第四个阶段叫**形成新自我**。当我们受困于他人的目光时，如何把评价坐标从“他人”切换到“自我”？如何发展新的信念，塑造属于你自己的人生故事？课程会教给你方法。

你会发现，转变并非一夜发生，每一步都需要你踏踏实实地走出来。拥有这份“地图”，你就会了解自己要走的是一条什么样的道路，哪里有危险，哪里有资源，哪里有可能的出路。

过去几年，在与学员的交流中，我深切感受到这份地图对正处于转变中的人有何等帮助。有同学说：“以前我总是焦虑，现在我不会那么焦虑了。因为我知道自己正在经历什么、将要经历什么，以及能从这些经历中收获什么。”

还有同学说：“从来没有一门课，会这么走进我的内心，这么贴近和理解我的经验。”其实，这门课不只适用于正处于转变期的人，它适用于所有人。

我同意他们的说法。这是一门关于转变的课，但你不需要一定身处转变期才能学习。无论你是否愿意，“转变”是我们早晚会经历的事；而寻找理想的自我，获得自洽的人生，也是我们每个人毕生的功课。

自我转变是艰难的，但从某种程度上说，它又没那么难。因为你有一个最基本的资源可以利用：**那便是根植于你自身、一种与生俱来的生命力**。你只需要一个“容器”，将这股生命力培育成新的自我。

而我希望，我们的课程能够成为这样一个容器。

很高兴与你相遇。祝你通过这门课，成为你想要成为的那个自己。

我是陈海贤，我们课程里见。
