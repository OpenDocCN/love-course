# 戎震充电视频：《进入心流》：04：控制情绪，排遣欲望_note

在本节课中，我们将要学习进入心流状态的第四项关键技能：如何控制情绪与排遣欲望。情绪与欲望是人类的两大弱点，若控制不当，极易引发问题，甚至导致人生陷入负面循环。

## 情绪与欲望：两大人生弱点

上一节我们探讨了专注的重要性，本节中我们来看看阻碍心流的内部障碍。情绪与欲望是人类的两大弱点。控制不好就容易出问题。人一生中犯下的重大错误，往往源于情绪失控或被欲望驱使，做出违反伦理道德或法律的事情，从而导致人生进入负面循环。

对于身处社会底层并试图突破极限的人而言，满足的通常是人类比较原始的欲望。在这些方面，人们往往需要付出巨大代价。而进入心流状态，意味着进入一种高级的、理性的、有条理的生活，不再处于底层世界。拥有独立思想和判断力，是一种中等向上的状态。

一个拥有觉知、开悟、独立思想、判断力，且不受情绪与欲望支配的人，实际上就是社会精英分子。常说的贵族气质，其核心就是自律。**自律**并非指完全按纪律过军事化的生活，而是指对情绪的把控。

## 情绪的本质与控制

以下是情绪失控的常见表现及其后果。

*   青春期时，无论男女，情绪起伏受激素调节影响，很多人难以把握。
*   随着年龄增长至30岁左右，情绪控制通常会好转。
*   但也有人放任脾气，想发就发，想骂就骂。
*   这种人一生难有较大成就，常在社会较低层级徘徊。

例如，用暴力迫使伴侣屈服的人，其伴侣通常在社会中也难以取得高位。在底层互相攻击没有意义。过一种觉知的生活，必须能控制情绪。情绪是身体的产物，但你的身体、心灵与智慧同时在起作用。身心灵三位一体，在冥想过程中你能明确感受到。

身体的需求会产生欲望与情绪。你只需观察它，明白其发生机理，等待它自然滑落即可。情绪与欲望如一时激起的千层浪，终会慢慢平息。受到关键刺激时，作为生物本能，你必然会产生欲望。欲望促使你生存、繁衍，并发挥主观能动性改善生活条件。

而情绪，例如应急情绪，有其自我保护的目的。义愤填膺或紧张时害怕不动，都是进化遗留的原始本能。野兽倾向于捕捉移动快的动物，不动则不易被发现。因此，情绪与欲望都是原始进化而来、与生俱来的东西。

但人作为高级的社会化动物，必须能够控制原始欲望与情绪。如果看谁都来气，想弄死谁，这无法完成正常的社会化生活。

## 社会规则与自我控制

在一个社会中，占据优势的人，必定是深谙社会规则的人。社会规则的核心是赏善罚恶。谁越适应规则，谁就越能占据优先位置。

以下是无法控制情绪与欲望将导致的直接后果。

*   **法律与财务风险**：易怒好斗将招致赔偿金与牢狱之灾。
*   **人际关系恶化**：家庭与社会中，大家不愿接近，导致老年孤苦。
*   **价值认知错位**：年轻时凭外貌任性，年老色衰后无人接纳，却仍沉浸在过去的价值错觉中。
*   **人生代价高昂**：控制不住欲望（如声色犬马），会付出健康、社会声誉、法律与金钱的沉重代价。

成年人社会的巨大差距，往往就在于能否控制情绪与欲望。男性若总用下半身思考，天下没有免费的幸运，终将付出巨大代价。女性若总要求他人承受自己的脾气，结果要么找到特别懦弱的伴侣，要么孤家寡人。家人被迫承受，最终可能导致家庭分崩离析。

在网络社会中，或许可以凭借身体价值或对网友不客气来获得短暂满足，但这在社会现实中往往意味着是一个失败者、失意者。

## 通往心流与高阶生活的钥匙

所以，若你真正想进入心流状态，做恰如其分且成长率高的事情，实现阶级跃升，过富足生活，就必须控制并学会排解情绪与欲望。

你需要观察它们的升起与滑落，明白这只是身体的需要，是激素调节的结果。它可能控制你的大脑和身体，但**控制不了你的心**。你的心是至善至真的，不应去做违背人生准则的事情。

这是进入心流与冥想阶段的最大困难点：能否平衡底层能量（如海底轮）。如果基本的生存欲望得不到恰当满足，心智会出现问题；如果因处理不好人际关系而情绪波动巨大，也无法进入高能量、高智慧的状态。

只有处理好基础能量，控制并排解根本的情绪与欲望，看清它们只是生存与进化中的副产品，你才能掌控它们，从而迈向更高级的生活，追求精神与物质的丰盈。这才是开启完美生活的钥匙。

本节课中我们一起学习了控制情绪与排遣欲望对进入心流状态的重要性。我们认识到，情绪与欲望是原始本能，但通过观察、理解与自律，我们可以超越其控制，从而为理性、高阶的生活与个人成长铺平道路。
