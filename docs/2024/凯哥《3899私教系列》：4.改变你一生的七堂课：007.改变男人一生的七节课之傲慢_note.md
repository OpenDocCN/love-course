# 凯哥《3899私教系列》：4.改变你一生的七堂课：007.改变男人一生的七节课之傲慢

大家好，我是恋爱演习社的高级情感导师朱雀老师。在约会学中，降低需求感是常见理论。但有时会转化为傲慢。傲慢是天使犯了罪，变成了魔鬼。傲慢会让人满足于现状，出现心理膨胀，招惹他人厌恶，暴露心性，引来敌意，遭到异性鄙视。

## 傲慢的害处

以下是傲慢的几个害处：

1. **无法真实认知自己**：傲慢的人不知道真实的自己是什么样子，不学习，忽略成长，停滞不前。
2. **容易被屏蔽**：傲慢的人越容易被屏蔽，意识越不清楚，意识不到自己的问题，也就解决不了自己的问题。
3. **导致失败**：傲慢的男人在相亲节目中容易失败，被女嘉宾群起而攻之，拉入黑名单。

## 傲慢的表现

以下是傲慢的几种表现：

1. **礼仪缺失**：不够绅士，防卫性语言多，重心放在打压对方，忘记基本礼仪。
2. **装逼**：明明没有的东西，却要装作自己有，容易被识破，自食其果。
3. **轻视和贬低女人**：给女人打分，夸大她们的缺陷，不懂得欣赏女人。
4. **缺少平常心**：得到一个女孩后，产生傲慢情绪，很快就会摔跟头。
5. **约会学中毒**：沉溺于约会学理论，远离绅士，骄傲风趣变成骄傲，满嘴风凉话。

## 如何克服傲慢

1. **反思自己**：认识到自己的傲慢，并努力改正。
2. **多学习**：学习礼仪、沟通技巧等，提升自己。
3. **保持谦逊**：不要因为一点成就就骄傲自满。
4. **尊重他人**：尊重女性，欣赏她们的价值。

## 总结

傲慢是一种常见的心理问题，会给人带来很多负面影响。我们要努力克服傲慢，做一个更好的人。
