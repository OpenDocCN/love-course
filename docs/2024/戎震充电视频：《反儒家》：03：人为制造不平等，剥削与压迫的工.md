# 戎震充电视频：《反儒家》：03：人为制造不平等，剥削与压迫的工

在本节课中，我们将探讨儒家思想在历史上的局限性及其社会代价。我们将分析其如何成为一种制造不平等、服务于剥削与压迫体系的工具，并与现代社会的普遍价值观进行对比。

## 概述：发展的代价与历史局限性

上一节我们讨论了儒家思想的某些核心理念，本节中我们来看看其发展所付出的社会代价。这种代价主要体现在牺牲广大平民阶级的利益，以维护特定朝代或统治集团的力量。

很多人将古代的科举制度与今天的高考制度相提并论。这是一种非常荒谬的对比。

## 科举与高考：本质的区别

以下是科举制度与现代高考制度的几个根本不同点：

1.  **普遍性与义务性**：
    *   现代教育是**普遍义务制**。只要是中华人民共和国公民，适龄者都必须接受强制性义务教育。完成高中学历后，即可参加国家统一的高考。
    *   科举制度则**不普遍**。它并非面向所有平民子弟。首先需要家庭有经济能力支持读书，这意味着学习者需完全脱离生产劳动，对绝大多数家庭而言难以实现。

2.  **目的与结果**：
    *   现代高考制度的核心目的之一是**通过教育改变个人命运**，促进社会阶层流动。公式可简单表示为：`个人努力 + 公平教育机会 => 改变命运的可能性`。
    *   科举制度的核心目的是**为封建统治阶级选拔和吸收少数人才**，以稳固其统治。它并未意图解决阶级矛盾，也无法改变大多数底层人民（主要是农民）的地位。其逻辑是：`选拔少数人进入统治阶级 => 维持多数人的被剥削状态`。

因此，将现代教育制度与古代服务于剥削阶级利益的科举制度相比，是不公平且无意义的。

## 儒家的阶级分野：“君子”与“小人”

在儒家构建的秩序下，社会首先被分为“君子”与“小人”。

*   **君子**：指那些为君主出谋划策、拥有土地和生产资料的剥削阶级，或知书达理的读书人。
*   **小人**：指一般的劳动者、目不识丁的普通老百姓。

这种划分带有**阶级蔑视**，看不起种菜的农民、当兵的士卒等普通劳动者。它严格区分了有产者与无产者、有文化者与无文化者，并非一种具有普遍平等性的社会设计。

## 复古思潮的隐患

当前，有些人试图复兴包括《弟子规》在内的某些儒家规范，甚至赋予其类似邪教的性质，这令人匪夷所思。其目的若是为了**促进阶级固化、加剧社会矛盾**，则危害极大。

例如，所谓“乡贤”制度的变相提倡，即认为一个地方有资产、有学历的人就应拥有特权，这完全违背了“代表人民”的现代国家制度原则。儒家思想从起源上，便是服务于封建秩序的、具有倒退阶级性质的思想体系。

## 僵化的社会分层 vs 动态的现代流动

儒家将人分为“士农工商”，定下森严的等级。但在现代社会，情况截然不同：

*   农民可以进城打工，子女通过读书成为大学生，进入城市成为白领，购房定居，实现身份转换。
*   大学生也可以选择返乡，参与乡村振兴，担任村官或发展乡镇企业，为城乡发展做贡献。

现代社会的特点是**身份可以凭借努力和制度而改变**。古代则不然，普通百姓需要服徭役、纳重税，身份难以改变。土地私有制下，地主和官僚士绅对农民进行沉重盘剥（例如所谓的“四公六民”税制）。推翻儒家所维护的这套旧文化、旧制度，是历史的进步。

## 儒家的本质：求官的哲学与统治工具

为儒家唱赞歌者，常鼓吹“半部《论语》治天下”。但这套学说本质是什么？

历史上，一些儒家“大儒”在政权更迭时，常为新的暴力统治者辩护，称其“得了天命”。这暴露了其本质是一套**“拍马屁的求官哲学”**。它的一切设计，最终都以人民的牺牲为代价。如果你不懂、不遵从它那套“尊卑秩序”，你就需要白白付出劳动力，缴纳各种捐税。

**真正治理国家需要的是法家的制度、纵横家的策略以及农业等领域的实际科学，而非这套形而上的等级哲学。** 儒家的核心功能在于建立封建秩序：定义谁是尊、谁是卑（如男尊女卑），规定长幼次序、宗藩次序，划定“君子”与“小人”。这才是其封建本质所在。

因此，今天任何复兴儒家具体礼仪（如何行礼、如何侍奉领导、如何区别对待“君子”与统治者）的行为，都是在为封建糟粕“招魂”，是极不进步的。

## 总结与下期预告

本节课我们一起学习了儒家思想作为历史上人为制造不平等、服务于剥削压迫工具的本质。我们分析了其与现代教育制度的根本区别，揭示了其基于阶级分野的“君子小人”论，并批判了试图复古开历史倒车的思潮。儒家那套僵化的社会分层与求官哲学，与现代平等、流动的商业社会价值观完全格格不入。
