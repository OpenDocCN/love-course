## 概述：关系剧本的奥秘

我们可能都有一个疑问：为何许多人相爱容易，相守却异常艰难。从事心理情感行业五年后，我逐渐发现其中存在一个规律。

感情确实是不公平的。每个人的关系剧本，早在十几二十年前就已写好。对部分人而言，关系是简单游戏，他们能轻松找到合适伴侣，平稳结婚生子，过上美满生活。

但对另一些人而言，感情是地狱级难度的副本。他们内心渴望爱，却又忍不住恐惧爱。即便遇到喜欢的人并拼尽全力付出真心，结局却总是悲剧，在感情中反复受伤、痛苦挣扎。

情路尤为坎坷。为何存在如此巨大的差异？秘密在于每个人的**内核**以及**依恋类型**。

一个人的内核决定了他的所思所想，一个人的依恋类型决定了他在关系中的所作所为。你现在感情中的所有表现，都已被你的内核和依恋类型提前写好了剧本。你就像听话的演员，按此剧本反复上演自己的爱情片。

有的人剧本是超级英雄片，出场自带超能力和主角光环。而有的人剧本则是一场悲剧，或是一场唐吉诃德式的喜剧。

那应该怎么办？我们首先需要了解自己的剧本，找到对应方法，然后修炼出强大的内核来突破所有自我限制。这样你才能真正收获更强大的自己和更稳定的关系。

## 什么是依恋类型？如何测试？

首先，我们来看第一块内容：到底什么是依恋类型？又如何测试我自己的依恋类型？

依恋理论最早由心理学家约翰·鲍比在20世纪40年代提出，当时用于研究婴儿与父母之间的情感关系。到了上世纪80年代，许多著名心理学家发现这种依恋理论极其适用于成年人的关系。

根据一个人的**焦虑程度**和**回避程度**的强弱，人们可以将依恋类型分为四种：**焦虑型**、**回避型**、**恐惧型**和**安全型**。

那么，什么是焦虑程度？它指的是当你遇到困难时，是否着急要解决。如果不解决，是否会越来越紧张不安。

什么是回避程度？它指的是你在遇到事情时，是否第一时间想逃避。问题越难，是否越想远离这个问题。

根据这样的排列组合，我们可以得到四种依恋类型：

*   **高焦虑 + 低回避 = 焦虑型**
*   **低焦虑 + 高回避 = 回避型**
*   **高焦虑 + 高回避 = 恐惧型**
*   **低焦虑 + 低回避 = 安全型**

如果你不能确定自己或伴侣到底属于哪个类型，或者你们之间是否会因依恋类型互相伤害，那么接下来我们可以一起做一个测试。

读完了A组题目，你是否发现里面有几个重复出现的词？**害怕、担心、怀疑、否定**，这些都是带着负面情绪的关键词。正如题目里所说，你是否也有过自我贬低的表现？例如：“伴侣对我冷漠时，难道是我做错了吗？还是我不值得了？”

*   **如果A组最多**：那你的底色就是**焦虑型依恋**。你越害怕失去对方，就越容易焦虑。
*   **如果B组最多**：说明你是**回避型依恋**。你习惯保护自己，偏爱短期社交，容易回避深度关系可能带来的伤害。

*   **如果C组最多**：那么恭喜你，你是**安全型依恋**。在所有依恋类型中，这是最稳定的状态，也是自我提升最快的类型。

## 四种依恋类型的详细画像

接下来，通过看完四种依恋类型的画像，你将对自己有更清晰的认识。

### 焦虑型依恋画像 😟

结合刚才A组的表现，我们先说一说焦虑型依恋。

你有没有被人嘲笑过，说你是个“恋爱脑”？你非常渴望拥有另一半的一切，时时刻刻都想跟对方在一起。你可能会觉得这就是热恋的常态。

但如果过了六七个月，你仍无法回归正常生活，所有情绪依然拴在对方身上，有一点风吹草动就开始胡思乱想、消极暗示，怀疑对方对自己的爱，并且在接触中暴露出对对方的不信任，同时你的内耗会非常严重——难道这是关系的常态吗？

当然不是。如果一个人必须时刻掌握另一个人的动向，不允许有自己不知道的事发生，并且强迫对方表忠心，这并不是稳定的状态，而是在满足自己的依恋心理和依恋需求。这样的心理该如何改变？我们接下来的课程会详细告诉你。

焦虑型依恋往往最容易爱上的就是回避型的人，回避型也是最吸引焦虑型的依恋状态。焦虑型会喜欢上回避型的冷静和自信，但新鲜感一过，就会开始发展成为虐恋，开始互相伤害，并且进入恶性循环。那这是为什么呢？

### 回避型依恋画像 🏃‍♂️

现在让我们来看一看回避型依恋是什么样的。

回避型普遍具有理想主义，即便是亲密无间的伴侣，也要忍受他们挑剔的眼光。而且回避型是不“粘人”的，也更希望对方不要粘着自己。**自由独立就是他们的命**。

如果对方掌控欲太强、越界了，回避型会马上进入抗拒状态，会用尽方法想要摆脱这种关系里的被掌控感。

也正因为焦虑型和回避型在心理机制和底层逻辑上不同，这两种类型的关系往往就是相爱相杀。

### 恐惧型依恋画像 😨

焦虑和回避都是不稳定、不安全的依恋类型，它们有时还会同时存在于一个人身上。当这两种类型混合在一起，就会形成回避型的升级版——**恐惧型依恋**。

恐惧型依恋对长期关系更为挑剔。他们非常想获得长期关系的稳定感和安全感，但又特别害怕亲密接触，更讨厌处理关系中的矛盾和麻烦事。这类型依恋人格的恋爱和婚姻都很难长久，而单身的恐惧型甚至连开启一段关系都不愿意。

不过，恐惧型在人群中占比还是比较少的，不到5%。而且我们也有办法可以治愈这种情况。

### 安全型依恋画像 😊

不管是焦虑型、回避型还是恐惧型，问题的尽头，都是需要获得强大的自我内核和吸引力，从而转变成**安全型依恋**。

其实，所有人都可以通过学习和锻炼，获得极其强大的内核，把自己的依恋类型转变成安全型。一旦你获得了强内核和安全依恋，你会对焦虑型、回避型、恐惧型三种人格产生强大的吸引力。

并且，你对长期关系会有美好的憧憬和更强的掌控感。你也会获得稳定的情绪，既能与伴侣亲密无间，也能享受独处。

所以，只有你变成安全型，你才能进入一段良好的、掌控感强的长期关系，能在关系中有效沟通，妥善解决矛盾。
