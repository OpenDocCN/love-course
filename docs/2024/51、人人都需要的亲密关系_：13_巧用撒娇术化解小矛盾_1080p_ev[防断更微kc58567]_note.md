# 人人都需要的亲密关系_：13_巧用撒娇术化解小矛盾

## 概述
在本节课中，我们将学习如何利用撒娇术来化解与男生相处过程中可能遇到的小矛盾。

## 第一部分：妄自菲薄法
以下是使用妄自菲薄法化解小过失的步骤：

1. **情境**：遇到小过失，如物品零件丢失、弄脏衣服或损坏小物品。
2. **表现**：瞪大眼睛，嘴巴微张，看向对方的眉心，委屈地嘟嘴。
3. **反应**：男生会过来查看情况，你表现出自暴自弃的情绪。
4. **撒娇**：承认错误，然后给予男生一块小甜饼，让他撒个娇，捧起你。
5. **效果**：巧妙化解小麻烦，让他觉得你是离不开的英雄。

## 第二部分：责任嫁接法
以下是使用责任嫁接法化解小矛盾的步骤：

1. **情境**：约会迟到，对方可能不开心。
2. **表现**：承认错误，表达歉意，然后假装崴脚，表现出疼痛和自责。
3. **反应**：男生会关心你的疼痛，并意识到迟到的原因。
4. **效果**：将责任嫁接到自己身上，化解矛盾。

## 第三部分：处理争吵
以下是处理争吵的步骤：

1. **情境**：两人争吵，出现冷战。
2. **表现**：及时拉住冷战，控制情绪，放慢语气，放软态度。
3. **避免**：不要急躁，不要用沉默应对沉默，不要说出脱离撒娇范畴的话。
4. **行动**：表现出明事理和小委屈，为他做一件小事，给他台阶。
5. **效果**：快速化解矛盾，让他意识到你在闹脾气。

## 第四部分：冷战后的撒娇
以下是冷战后的撒娇方法：

1. **确认**：确定自己已经不生气，想要结束冷战。
2. **方法一**：直接抱他，表现出妥协和依赖。
3. **方法二**：走到他面前，低头垂眸，表现出纠结和勇气，请求拥抱。
4. **效果**：快速回暖关系。

## 总结
本节课中，我们一起学习了如何利用撒娇术化解小矛盾。通过妄自菲薄法、责任嫁接法、处理争吵和冷战后的撒娇，我们可以巧妙地化解与男生相处过程中的小问题。记住，撒娇是一种技巧，但也要注意不要过度依赖，遇到严肃问题时要保持端正的态度。
