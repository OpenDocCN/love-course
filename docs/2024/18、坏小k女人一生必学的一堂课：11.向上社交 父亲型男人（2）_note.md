# 18、坏小k女人一生必学的一堂课：11.向上社交 父亲型男人（2）

🌟 欢迎来到向上社交技巧篇。首先，我们需要理解优质男性分为七类，接下来我将讲解如何与这七类男性相处。以下是七类优质男性及其特点：

1. **皇帝型男人**：提供物质、平台和成就你的能力。
2. **父亲型男人**：给予物质、平台、支持你的事业，在感情中愿意服输。
3. **创一代男人**：具有创新精神，勇于开拓。
4. **富二代男人**：拥有财富和资源。
5. **生意型的人**：擅长商业运作。
6. **第六类型的人**：具体类型未提及。
7. **潜力股**：具有发展潜力的男性。

**以下是对父亲型男人的详细分析**：

**父亲型男人**：

- **特点**：属于成熟男性，是大多数女性难以追求的类型。
- **相处之道**：
  - **亲密关系**：不等于义务、责任或契约，不存在义务论。
  - **关系本质**：交换，满足彼此的需求。
  - **认知强者心态**：为自己的行为负责，不期望对方回馈。
  - **人生规划**：明确目标、自身条件、环境、行为、能力和信念。

**与父亲型男人沟通的话术**：

- “我觉得你是我认识的男人中最有能量的，你身边的所有人都会因为你而过得更好。我跟你在一起，即便是看看你的为人处事，对我来说都是一种成长。”

**注意事项**：

1. **提升自我**：学会增值自己，为事业、学历和未来规划投资。
2. **保持自我成长**：在无法改变的事情上向下看，在可以改变的事情上向上看。
3. **不要试图管理**：尊重对方，保持自己的独立性。
4. **创造自己的事界**：展现独特的世界观、精神、利益等。
5. **独立思考**：不要成为他人思想的奴隶，建立自己的世界观。

**总结**：

本节课中，我们学习了如何与父亲型男人相处，以及如何提升自我、保持独立思考。希望这些技巧能帮助你在向上社交的道路上取得成功。
