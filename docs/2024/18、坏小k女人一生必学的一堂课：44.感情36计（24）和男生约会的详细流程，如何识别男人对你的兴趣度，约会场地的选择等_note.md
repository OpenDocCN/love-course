# 18、坏小k女人一生必学的一堂课：感情36计（24）和男生约会的详细流程

🎼 感情36计第24个技巧和男生约会的详细流程。此流程包括如何识别男人对你的兴趣度、约会场地的选择、约会的频率、约会聊天话题的选择，以及如何迅速筛选出不符合目标的男人。约会的频率，每周一次即可。

### 约会流程

1. **识别兴趣度**：观察男人在约会中的言行举止，判断他对你的兴趣程度。
2. **场地选择**：根据约会阶段选择合适的场地。
3. **约会频率**：每周一次。
4. **聊天话题**：选择轻松愉快的话题，避免尴尬。
5. **筛选目标**：根据约会表现，判断男人是否符合你的目标。

### 第一次约会

1. **选择场地**：第一次约会可以选择吃饭，吃饭可以看出一个人的细节和素养。
2. **观察细节**：从吃什么到坐下来吃，是他的安排还是听从你的意见。
3. **了解背景**：从口味可以了解到他的成长背景和生活经历。
4. **注意细节**：点菜时是否询问你的喜好和禁忌，吃饭时是否照顾到你，如帮你倒水、递纸巾等。

### 社交软件认识的人

1. **初次见面**：不建议与社交软件上认识的人第一次见面就吃饭。
2. **时间控制**：一个小时的时间足够让你明白是否有必要与他吃饭。

### 第二次约会

1. **选择场地**：可以选择郊外一天可以来回的场地，如郊游。
2. **观察性格**：从他的驾驶中可以观察他的性格。
3. **消费观**：从他打车还是坐公共交通看观察他的消费观。

### 第三次约会

1. **选择场地**：可以接下班，下课加吃晚餐加散步。
2. **观察耐心**：不妨故意迟到观察他的耐心程度。

### 第四次约会

1. **选择场地**：可以选择看电影。
2. **推进关系**：通过看电影等机会推进关系。

### 约会核心时期

1. **破冰期**：安全第一，关系容错率较低。
2. **升温期**：调情行为导致的暧昧感产生。

### 约会场地选择

1. **流动型社交约会**：可以让两个人处于行进状态，有一个任务线，容易完成。
2. **共同记忆**：建立共同记忆，减少沟通压力。
