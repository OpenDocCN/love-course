# 嘴叔《聊天技巧及心态》：第八课，暧昧

## 概述
在本节课中，我们将学习如何在与心仪对象聊天过程中，通过暧昧的方式增进彼此关系。

## 暧昧的必要性
**公式**：暧昧 = 调侃 + 出乎意料 + 直接暧昧

1. **调侃**：通过轻松幽默的方式打破常规，拉近彼此距离。
2. **出乎意料**：在聊天中制造惊喜，让对方感受到你的用心。
3. **直接暧昧**：直接表达你对对方的喜欢和关心。

## 暧昧的程度
1. **初期暧昧**：表达“我想你了”、“越来越喜欢跟你聊天”等情感。
2. **避免过度暧昧**：不要过早涉及性话题或确定关系的话题。

## 暧昧的技巧
1. **真心话大冒险**：在微信上玩真心话大冒险，逐步升级话题。
2. **调侃**：适度调侃对方，但要避免过度。
3. **升级关系**：在暧昧的基础上，尝试升级关系。

## 避免的错误
1. **过度调侃**：避免在暧昧阶段过度调侃对方，以免让对方反感。
2. **过度幽默**：避免过度幽默，以免显得做作。

## 总结
本节课中，我们学习了如何通过暧昧的方式增进与心仪对象的关系。在聊天过程中，要注意暧昧的程度，避免过度，同时要把握时机，适时升级关系。希望这节课的内容能对大家有所帮助。
