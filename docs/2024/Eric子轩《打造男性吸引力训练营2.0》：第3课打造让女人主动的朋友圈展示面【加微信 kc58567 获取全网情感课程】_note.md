# Eric子轩《打造男性吸引力训练营2.0》：第3课打造让女人主动的朋友圈展示面

## 概述

在本节课中，我们将学习如何通过穿搭提升个人魅力，打造出吸引女性的朋友圈展示面。

## 避免穿搭错误

以下是几种需要避免的穿搭错误：

- **过于花哨的上衣**：避免选择颜色过多或图案复杂的上衣，以免显得没有品味。
- **格子衫**：格子衫容易穿出乡土风，不适合吸引女性。
- **三四十岁风格的衣服**：这类衣服容易显得老气，不适合年轻男性。
- **紧身裤**：紧身裤容易显得非主流，不适合大多数场合。
- **高饱和度鞋子**：避免选择过于鲜艳的鞋子，以免显得不协调。
- **紧身短袖**：避免选择紧身短袖，尤其是身材不是很好的人。
- **大字或夸张印花**：避免选择上面有大字或夸张印花的衣服。

## 打造个人风格

以下是四种常见的个人风格：

- **轻熟风**：以黑白灰为主色调，搭配简约的款式。
- **日韩风**：适合年龄在30岁以下的男性，风格清新。
- **商务风**：适合所有年龄和身材的男性，具有气场和性感。
- **网红风**：适合颜值高、身材好的男性，但年龄大于30岁的人不适合。

## 衣服颜色选择

- **基础色**：黑白灰是最不容易出错的颜色。
- **相邻颜色搭配**：选择深浅不同的颜色进行搭配，例如深灰裤子搭配浅灰色上衣。
- **低饱和度色系**：选择低饱和度的色系，例如灰色和亮色搭配。

## 配饰选择

- **手表**：选择一款贵一点的机械腕表，提升气场和审美。
- **戒指**：选择一款简约的戒指，与正装搭配。

## 男装购买建议

- **快时尚品牌**：如Zara、H&M等，价格便宜，风格出彩。
- **高级品牌**：如Armani、Gucci等，适合购买西服等正式服装。
- **基本款**：如优衣库、无印良品等，适合购买T恤、短裤等基础款。
- **日韩款型**：可以选择手店购买，质感较好。
- **折扣店**：如奥莱、品牌折扣店等，可以购买品牌奢侈品或轻奢品。

## 总结

本节课中，我们学习了如何避免穿搭错误，打造个人风格，选择合适的衣服颜色和配饰，以及如何购买男装。通过实践和尝试，你可以打造出吸引女性的朋友圈展示面，提升个人魅力。
