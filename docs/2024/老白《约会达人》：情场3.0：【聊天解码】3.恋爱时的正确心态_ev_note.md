# 老白《约会达人》课程3：恋爱时的正确心态 🧠

在本节课中，我们将要学习恋爱中最核心但也最容易被忽视的部分：心态建设。心态决定了我们如何开始、如何面对挫折以及如何最终获得成长。掌握正确的心态，是掌握恋爱能力的第一步。

## 概述：心态是脱单的最大挑战

我是老白。在脱单的道路上，我认为最大也最困难的问题，就是心态问题。许多同学觉得“心态”这个词很虚，认为心态建设非常复杂。但今天，我想与大家分享两个我认为最重要的核心理念。

## 核心理念一：恋爱是一种可以习得的能力

首先，情感本身是一种**能力**。如果你不把它看作一种能力，就会犯各种错误。

回想一下你小时候学习骑自行车的经历。你们是第一次就学会了吗？我们对于学自行车的预期，通常是决定学个三四次才学会。这是一个大家都能接受的设定。

但为什么到了谈恋爱这件事上，很多人就觉得必须一次性搞定、一次就学会呢？这里的核心在于：**任何技巧和知识点都需要不断练习**。例如聊天、例如修图，可能需要练习三五十次才能真正明白什么是恋爱。

**所以，恋爱是需要练习的。恋爱也是一种能力，一种技能。**

在技能学习过程中，有三个关键点必须注意。

### 1. 障碍区

障碍区是指你会跌倒、会感到困难的地方。就像第一次骑自行车，在转弯时摔倒了，下次再遇到转弯就会害怕。这就是一个障碍。

绝大部分人没有学会一项能力的核心错误，在于没有理解“人是存在障碍区的”。每一个障碍区的突破，大概需要**3到5次甚至更多次**的尝试，这个过程可能令人沮丧。

例如，你尝试社交搭讪却被拒绝，你可能会想：“是不是我不适合社交？是不是方法不靠谱？”其实不是，只是因为你正在经历**障碍区**。刚开始改变个人形象觉得奇怪、聊天总是聊不好、能把女生约出来但约会后就没了下文、恋爱超不过半年……在人生的道路上，一定会遇到各种各样、不同阶段的障碍区。

遇到障碍不是放弃的理由，而是要想办法去跨越。一次不行就两次，两次不行就三次，直到跨越为止。你看到别人追女神很轻松，自己却困难重重，那是因为你只看到了表面。任何表面上的轻而易举，都建立在背后无数次万分努力的基础上。

**所以，理解并接受“障碍区”的存在，是学习的第一步。**

### 2. 拐点

拐点是指你不再需要别人教导，获得了自主前行的力量的时刻。例如，健身不再需要教练指导，谈恋爱不再需要找我咨询，这都是你的拐点。

出现拐点，意味着你对这件事有了更大的积极性和掌控感，但绝不能骄傲。因为拐点也意味着你即将迈向**下一个起点**。突破社交恐惧去搭讪，恭喜，通过了一个拐点。展示面打造是拐点，进入暧昧是拐点，成功约会、牵手、接吻都是拐点。

每一个拐点都不能掉以轻心。但不断通过拐点，就意味着你获得了自动前行的力量。

**简单来说，当你出现了一个拐点，就意味着你成功通过了上一个障碍区。**

### 3. 状态回落

很多人在这里摔了跟头，就再也爬不起来了。好不容易突破了各种障碍区和拐点，终于找到称心如意的女朋友，然后开始得意洋洋、放松警惕，结果现实让你摔了个大跟头。

大家可能都有经验：往往淹死的都是会游泳的人。大家觉得自己没问题、很有把握，结果就在水里出事了。这就是**状态回落**，千万不能掉以轻心。尤其是在你刚刚突破一个拐点、状态最佳的时候，你的状态**一定会**回落。

就像开车的人说“我从来没出过事儿”，结果最后多少都会出点小事故。问题就出在这里。

**所以，学习恋爱、掌握恋爱能力是一个过程。它一定会经历以上三个重要阶段：障碍区、拐点、状态回落。** 包括现在的我，也还在学习，还在突破障碍区，通过拐点，应对状态回落。

其实学习恋爱真的不是一件太困难的事情。简单的说，**你就把每个阶段该犯的错误都犯一遍就好了。下一次你就知道你错在哪儿了。**

这是我分享的第一个核心理念。

## 核心理念二：用“成长型思维”看待恋爱经历

第二个方面，所有人看待事物，大致有两种模式。

第一种人，用**崇拜**或**宿命**的概念来看待事情。他们相信缘分，相信某些事情是上天注定的。

第二种人，相信**成长**，相信**事在人为**。

我对大家的要求是：**我希望你从今天开始，做第二种人。**

这很简单。如果你把事情仅仅看成“成与败”，你就会陷入一种宿命的心态。不要把恋爱经历仅仅看作成功与失败。

之前一个学员跟我说：“我觉得我需要一段感情，我上一段感情真的太失败了，我什么都没有得到。”我告诉他：错了，你得到了。你再也不用为这个女人浪费一秒钟时间。你从今天开始至少知道了这个人是不适合你的。**这就是你在这段关系中得到的最大的成功。**

所以，看待每件事情，悲观的人永远是消极的，乐观的人永远是积极的。我们从今天开始就做第二种人。

**核心是：从任何事情里得到成长。**

哪怕表面上没有得到任何东西，在这段过程中，你懂得了什么是教训，知道了下次不能再犯某些错误，这就是你得到的成长。

例如，当你与别人聊天、去社交、去约会、去谈恋爱时，一定要问自己：**“我从这件事情里得到了什么成长？”** 这是最重要的东西，也是心态建设的核心与关键。

## 总结与回顾

本节课我们一起学习了恋爱心态建设的两个核心方面。

**第一方面**，我们明确了恋爱是一种可以习得的能力。在学习过程中，我们不应建立过高的、不切实际的预期（如“一次成功”）。我们要理解并准备好穿越**障碍区**，重视并平稳度过**拐点**，同时警惕成功后可能出现的**状态回落**。这是一个学习者应有的基本心态。

**第二方面**，我们探讨了看待事物的两种模式。我们应摒弃以单纯的“成功/失败”来定义经历的**固定型思维**，转而拥抱**成长型思维**——从每一段经历中，无论结果如何，都主动寻找并肯定自己获得的成长与教训。

希望大家通过这堂课能够懂得，心态不是虚无缥缈的，而是可以通过具体方法建设和调整的。在脱单这条道路上，请务必把自己的心态调整到“成长”频道上来。
