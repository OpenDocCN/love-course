## 自我认知

### 自我认知的重要性

**公式**： 自我认知 = 接受自己的不完美 + 认识自己的优点和缺点

**代码**：

```python
class Self_Cognition:
    def __init__(self):
        self.perfections = []
        self.imperfections = []

    def add_perfection(self, perfection):
        self.perfections.append(perfection)

    def add_imperfection(self, imperfection):
        self.imperfections.append(imperfection)

    def show(self):
        print("优点：", self.perfections)
        print("缺点：", self.imperfections)
```

### 自我认知的步骤

1. **接受自己的不完美**： 认识到自己的不完美，包括情绪、性格、行为等方面。
2. **认识自己的优点和缺点**： 列出自己的优点和缺点，并挂在镜子上每天提醒自己。

## 边界性

### 边界性的定义

边界性是指在人际交往中，明确自己的底线和需求，并让对方了解自己的边界。

### 边界性的重要性

**公式**： 边界性 = 自我保护 + 和谐相处

**代码**：

```python
class Boundary:
    def __init__(self):
        self.boundaries = []

    def add_boundary(self, boundary):
        self.boundaries.append(boundary)

    def show(self):
        print("我的边界：", self.boundaries)
```

### 边界性的建立

1. **明确自己的底线和需求**： 列出自己不能接受的行为和需求。
2. **让对方了解自己的边界**： 与对方沟通自己的边界，并让对方尊重。

## 付出与妥协

### 付出与妥协的定义

付出与妥协是指在恋爱关系中，为了维持关系而做出的让步和牺牲。

### 付出与妥协的原则

1. **付出要有回报**： 在付出和妥协后，需要得到对方的认同和理解。
2. **尊重对方的选择**： 如果对方不愿意接受你的付出和妥协，要尊重对方的选择。
