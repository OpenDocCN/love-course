# 06.《高嫁实操秘籍——向上社：016.4.15 运用六度人脉

在本节课中，我们将要学习如何运用著名的“六度人脉理论”来系统地建立和拓展你的社交圈。我们将深入探讨该理论的分层结构、核心运用原则以及如何有效经营你的人脉网络，帮助你更高效地进行向上社交。

## 什么是六度人脉理论

上一节我们介绍了向上社交中建立人脉资源库的重要性，本节中我们来看看其背后的核心理论体系。

六度人脉理论指出，地球上任意两个陌生人之间，平均只需要通过不超过六个中间人就能建立起联系。该理论由美国心理学家斯坦利·米尔格兰姆在20世纪60年代首次提出，并得到了广泛验证。

通俗地讲，你与世界上任何你想认识的陌生人之间，所间隔的熟人数量不会超过六个。这意味着，通过有限的人际链条，你可以触达几乎任何目标人脉。

斯坦福大学研究中心的调查数据显示，一个人赚的钱，大约12.5%来自知识，另外87.5%来自人际关系。这印证了人脉拓展对个人财富积累的巨大助力。一个广为人知的成功公式是：**成功 = 70%的人脉 + 30%的努力**。积累优质人脉是获得成就的关键。

## 六度人脉的层级划分

理解了理论的基础后，我们来看看如何将你的人脉关系进行具体划分。

六度人脉以你为核心，向外可以划分为六个层级：
1.  **一度人脉**：你自己。
2.  **二度人脉**：有血缘关系的直系亲属。
3.  **三度人脉**：关系密切的亲戚好友。
4.  **四度人脉**：客户、邻居、普通朋友。
5.  **五度人脉**：旧同事、旧同学、不太熟悉的朋友。
6.  **六度人脉**：可能产生链接的新朋友或陌生人。

根据核心需求的不同，我们可以将上述人脉进一步归类到三个关系池中进行管理：

以下是三个人脉关系池的划分方式：
*   **核心关系池**：对你的生活、工作或个人发展起关键决定作用的人。此池中的人选需根据你不同阶段的目标进行更新和替换。
*   **紧密关系池**：能为你提供帮助或辅助搭建人脉体系的人群，例如亲密的合作者、同事、部门领导及亲密家人。
*   **备用关系池**：暂时用不到，但未来可能派上用场的人脉资源，只需定期维护即可。

你可以尝试用Excel表格来管理这些人脉，记录关键信息（如籍贯、家庭情况、住址等），并在重要节日定期问候或互动，以便在需要时能迅速采取行动。

## 如何精准拓展目标人脉

建立了人脉管理体系后，下一个问题是如何找到并链接你的目标人脉。

在向上社交过程中，并非总能直接找到对标人脉。你需要主动出击，进入目标人群聚集的场合。

以下是针对不同目标的拓展场景举例：
*   **目标：金融行业潜在客户（高净值人群）**
    *   **方法**：参与需要一定经济门槛的活动，如滑雪、冲浪、打高尔夫等。这些活动本身就是一个筛选机制。
*   **目标：商务合作或投资人资源**
    *   **方法**：参加高端展览、国际大秀、奢侈品品牌活动等。这些场合是品牌方、投资人、高影响力人群的聚集地。
*   **目标：链接特定品牌或资源**
    *   **方法**：主动研究目标公司的公开信息（地址、联系方式、邮箱），并毛遂自荐。例如，可以主动联系品牌方，提出以自身资源（如摄影、内容创作）进行免费合作，以此打开局面。

所有人脉资源都需要靠你自己主动想办法去链接，而非被动等待。

## 运用六度人脉的核心原则

前面我们分享了六度人脉的理论知识和分层方法，接下来我们聊聊运用该理论时需要掌握的核心原则。

六度人脉的概念是固定的，但在运用时，最适合自己的方式才是最好的，无需刻板照搬。

**第一点：区分人脉池，建立不同的价值体系。**

人脉需要物尽其用。除了上述三个基础关系池，你还可以根据领域建立更细分的关系网，例如：深度合作人脉、潜在客户人脉、政府人脉、金融人脉、技术人脉、媒体人脉等。网上有句话很有道理：你必须认识两个职业的朋友，一个是律师，一个是医生。

**第二点：注重人脉的投入产出比。**

用最小的人脉成本办最有利的事。所有投入都要用在刀刃上。伤害人脉关系的事不要做。能用钱解决的事是小事，动用“人情”的事才是大事。

例如，在饭局上认识一位大佬后，请求对方为你可能失业的丈夫介绍工作，就是一个典型的错误。这种请求不仅理由牵强，会让对方没有帮忙的动力，更会严重降低你在对方心中的价值和圈内形象。为了小事损失大人脉价值，得不偿失。

相比之下，请教事业困惑、投资理财建议，或为孩子教育问题寻求引荐，则是更合适、更能体现你格局的求助方向。

**第三点：抓住每一个帮助别人的机会。**

向上社交不是单方面索取，而是价值交换。即使对方目前看似比你强大，也总有他的知识盲区和需要帮助的时候。

如果你正好能提供专业帮助，一定要积极主动。例如，当你在高端饭局中，听到政府单位有发布新闻稿或主持活动的需求，而你恰好有相关资源和能力，就可以主动提出无偿协助。通过提供专业价值，你才能赢得尊重和更深层次的合作机会，对方也才更愿意将你引入他的核心人脉圈。

向上社交不是跪舔。在社交场合中，突出你的专业身份和价值，远比扮演陪吃陪喝的角色更重要。当对方认为与你的结识有价值时，你才有资格进入他的人脉储备库。

有时，主动“吃亏”、让别人占便宜，也是积累人情的好方法。人情债难还，对方在承了你的情之后，在你需要帮助时往往也难以拒绝。

**第四点：人脉储备需要循序渐进，切忌急功近利。**

过于热情和自来熟，可能会给人带来压迫感。人脉储备需要保持持续、平稳的接触，这样建立的关系才更值得信赖。

分享一个社交原则：**一回生，二回半生不熟，三回熟手**。第一次接触，双方通常都有戒备心。第二次接触，适合主动展现自我，进行初步了解。第三次接触，双方才会更加放松。循序渐进才是长久渗透的关键。

## 如何经营好你的朋友圈

说完了运用六度人脉的核心原则，接下来我们要聊一个非常重点的环节：如何经营好自己的朋友圈，为人脉建设加分。

**第一点：分层管理，针对性展示。**

针对不同的人脉分组，展示不同的内容。例如，将给政府单位做培训的照片，仅对相关领域的客户或领导可见；将精致的自拍，仅对潜在发展对象可见。在朋友圈，应多展示积极、专业的职场形象和精彩的活动参与经历，少发情绪化的感悟。

心理学上的“马太效应”指出，强者愈强。你展示的越多、越精彩，别人就越能感知你的价值。低调做人，高调做事，让你的好被别人看见。

**第二点：抓大放小，维护核心人脉。**

根据“二八定律”，20%的核心人脉为你带来的价值，远超其余80%的人。不必强求融入所有大圈子，有时小圈子能带来更大结果。将主要的人脉维护成本，投入到那些对你帮助最大、最能带来结果的目标人脉上。

**第三点：善用“熟人链接效应”，引发蝴蝶效应。**

善待你遇见的每一个人，尤其要重视你的“一度人脉”（最熟悉的人）。认识时间越久、彼此越了解的人，信任度越高，办事也越容易事半功倍，例如老同学关系。

要学会利用熟人引发蝴蝶效应。优质人脉的推荐更有可信度。在圈子里，要利用一切机会发挥你的最大价值，让人看见你，才有机会实现圈层的升级。

## 课程总结

本节课中，我们一起学习了以下内容：
1.  **六度人脉理论**：地球上任意两人最多通过六层关系即可链接。
2.  **人脉层级划分**：将人脉分为六度，并归类到核心、紧密、备用三个关系池进行管理。
3.  **拓展目标人脉的方法**：主动进入目标人群聚集的高端或特定场景进行链接。
4.  **运用核心原则**：包括建立价值体系、注重投入产出比、主动提供帮助、循序渐进储备人脉。
5.  **朋友圈经营技巧**：通过分层展示、抓大放小、善用熟人效应来提升社交形象和效率。
