# 课程01：理解变量与赋值 🔢

在本节课中，我们将要学习编程中最基础也最重要的两个概念：**变量**和**赋值**。它们是存储和操作数据的基石。

## 什么是变量？ 📦

变量是计算机内存中一个用于存储信息的“盒子”或“位置”。你可以把它想象成一个贴有标签的储物柜，标签就是变量的名字，而储物柜里存放的东西就是变量的值。

在编程中，我们通过一个名字（即**变量名**）来引用这个存储位置。例如，我们可以创建一个名为 `age` 的变量来存储一个人的年龄。

## 什么是赋值？ ➡️

赋值是将一个值放入变量中的操作。我们使用等号 `=` 来完成赋值，这被称为**赋值运算符**。

其核心概念可以用以下公式描述：
`变量名 = 值`

这意味着将等号右边的“值”放入等号左边的“变量”所代表的内存空间中。

---

上一节我们介绍了变量和赋值的基本概念，本节中我们来看看如何具体使用它们。

## 赋值操作详解

当你执行赋值语句时，计算机会完成以下步骤：
1.  在内存中为变量开辟一个空间。
2.  将等号右边的值计算出来。
3.  将这个值存入变量对应的内存空间。

例如，在Python中：
```python
age = 25
```
这行代码创建了一个名为 `age` 的变量，并将整数值 `25` 存储在其中。

---

理解了单个赋值后，我们来看看变量值是如何变化的。

## 重新赋值

变量的值并非一成不变。你可以随时通过新的赋值语句来改变变量中存储的值。新值会覆盖旧值。

以下是变量重新赋值的示例过程：
```python
score = 90    # 第一次赋值，score 的值为 90
score = 85    # 重新赋值，score 的值变为 85，90被覆盖
```

---

变量不仅可以存储简单的数字，还可以存储其他类型的数据，并参与到计算中。

## 变量在表达式中的使用

变量可以像具体的数值一样，用在数学表达式或其他操作中。当程序遇到变量名时，它会取出该变量当前存储的值来参与运算。

请看以下示例，它演示了变量如何参与计算：
```python
x = 10
y = 20
sum = x + y  # 计算时，取出x的值10和y的值20，相加后得到30，再将30赋值给变量sum
```

---

为了有效地使用变量，我们需要遵循一些命名规则和最佳实践。

## 变量命名规则与约定

给变量起一个好名字非常重要，它能提高代码的可读性。以下是需要遵循的主要规则和常见约定：

*   **规则（必须遵守）**：
    *   变量名只能包含字母、数字和下划线（`_`）。
    *   变量名不能以数字开头。
    *   变量名不能是编程语言的关键字（如 `if`， `for`， `while` 等）。
    *   变量名通常区分大小写（例如 `age` 和 `Age` 是两个不同的变量）。

*   **约定（建议遵守）**：
    *   使用描述性的名称，让人一眼就能明白变量用途（如 `user_name` 比 `un` 更好）。
    *   在Python中，通常使用小写字母和下划线来命名变量（蛇形命名法），例如：`student_count`、`total_price`。
