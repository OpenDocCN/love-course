# Python基础教程 1.1

## 概述
在本节课中我们将要学习Python编程语言的基础知识，包括安装、基本语法和变量。

## 安装Python
首先，你需要安装Python。Python可以从其官方网站下载：[https://www.python.org/downloads/](https://www.python.org/downloads/)。

### 安装步骤
1. 访问Python官网下载最新版本的Python。
2. 运行安装程序。
3. 在安装过程中，确保勾选“Add Python to PATH”选项。
4. 点击“Install Now”完成安装。

## Python基本语法
Python是一种解释型语言，这意味着你不需要编译代码，可以直接运行。

### 变量
变量是存储数据的容器。在Python中，你可以使用以下方式声明变量：

```python
x = 10
y = "Hello, World!"
```

### 输出
你可以使用`print()`函数输出信息：

```python
print("Hello, World!")
```
