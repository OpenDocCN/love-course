# 子阳《脱单实战聊天课》：1.聊什么：第十五节课：前两点总结

## 概述
在本节课中，我们将总结前两点关于如何与女生进行有效聊天的技巧。

## 第一点：锻炼自己的能力
**公式**：锻炼能力 = 理性话题 → 感性话题

当你与女生从普通朋友转变为好朋友时，你需要学会如何从一个理性话题迅速转换到感性话题，并表达对这些事物的看法。

## 第二点：处理分歧
**公式**：处理分歧 = 大事讲原则，小事讲风格

在讨论事物的看法时，如果产生分歧，你应该做到永远不要与她争论大事，坚持原则；对于小事，则可以灵活调整自己的风格。

## 第三点：避免盲目附和
**公式**：避免盲目附和 = 保持独立思考

永远不要盲目附和女生的观点，保持独立思考。

## 总结
本节课中，我们学习了如何与女生进行有效聊天，包括锻炼自己的能力、处理分歧以及避免盲目附和。希望这些技巧能帮助你更好地与女生交流。
