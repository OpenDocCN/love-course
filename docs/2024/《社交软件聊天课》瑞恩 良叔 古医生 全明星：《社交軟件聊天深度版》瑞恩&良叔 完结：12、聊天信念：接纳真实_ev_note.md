# 《社交软件聊天课》第十二课：聊天信念：接纳真实

## 概述

在本节课中，我们将深入探讨聊天信念中的“接纳真实”这一核心概念。通过分析案例，我们将学习如何在聊天中展现真实的自己，并理解真实表达对建立吸引力的意义。

## 一、案例回顾

**1. 开启话题**

*   **瑞恩**：听说今天成都天气很好。
*   **然**：没错哦，出太阳了。

**2. 共情与回应**

*   **瑞恩**：加上周末估计都是人。
*   **然**：是的，我在考虑出门不快回成都。

**3. 推拉与真实表达**

*   **瑞恩**：我怕我吃了你哦。
*   **然**：我会吃了你哦，就开玩笑嘛，就是你为什么要怕呀？
*   **瑞恩**：兄弟们在这里我插一个点啊，就是很多兄弟们看到这个聊天案你，看到这里呢，他会投射一件事情，就是他会投射我的强大。认为是降维打击，所以女生呢给你们感觉像一个大狼狗一样的来追我，但是呢通常呢我想讲一个就是很通俗的一句话，就是如果你想让一个女生对你这么的主动，甚至把你给吃掉。其实呢很多时候在我们跟女人聊天的时候呢。你不能去做这个大狼狗，你们要知道你都已经是这个大狼狗了，你们什么时候见过两。条大狼狗相互之间把相互想要吃掉的，这个是几乎不可能就是我刚才讲的我们自己定位和你想要去做哪把椅子的一个关系。所以如果你想让一个女人对你主动的想把你吃掉像当大狼狗的样子，你就不能是那个大狼狗。

**4. 打情骂俏与安全感**

*   **然**：哈哈你前面两句话，别人会信。
*   **瑞恩**：你这个人他真的别人说我还会信，你从头开始就不待见我。
*   **然**：哎，但是在这儿我看到了一个不一样的东西。不一样的在哪？那么女生当他前面讲啊，你前面两句话我会信，别人讲她会信对你前面两句话，别人会信，就是怕爱上我变成小舔狗。所以呢她就是默认什么。他默认一个，我不是会去追他，或者说是一个会跟他进入到那种关系里面的一个人。但是呢有一个重点啊，就是他后面第三句话他没有提，为什么？因为第三句话他是相信的，因为第三句话很真实，你吃了我我这倒不亏。这个其实是这三句话里面，我觉得最真实的一句话，最真实的一个自我的表达。

## 二、核心概念

**1. 接纳真实**

*   **公式**：真实表达 = 自我接纳 + 勇敢表达
*   **代码**：在聊天中，展现真实的自己，包括优点和缺点，并勇敢地表达自己的想法和感受。

**2. 推拉**

*   **公式**：推拉 = 主动表达 + 被动回应
*   **代码**：在聊天中，通过主动表达和被动回应，制造一定的紧张感和期待感。

## 三、总结

本节课中，我们学习了聊天信念中的“接纳真实”这一核心概念。通过分析案例，我们了解到真实表达对建立吸引力的意义，并学习了如何通过推拉和打情骂俏等方式，在聊天中展现真实的自己。希望同学们能够将所学知识应用到实际聊天中，提升自己的聊天技巧。
