# 课程18：感情36计（3）——如何保持情绪稳定以及利用他人情绪 🧠

在本节课中，我们将学习感情36计的第三部分核心技巧：如何建立内在情绪稳定性，以及如何利用他人的情绪来达成自己的目标。课程将分为两大板块进行详细解析。

## 一、如何做到情绪稳定 🧘‍♀️

上一节我们介绍了课程的整体框架，本节中我们来看看情绪稳定的具体内涵。情绪稳定并非指没有情绪，而是指能有效管理和运用情绪。它可分为四个层次，你可以对照评估自己所属的层次。

以下是情绪稳定的四个层次：

1.  **第一层：被情绪主导**
    *   描述：完全被情绪控制，无法进行理性判断。情绪不经思考，一点就炸。
2.  **第二层：无视负面情绪**
    *   描述：内心放松、平和，以正向情绪为主。很难被外界事物激起强烈的负面情绪。
3.  **第三层：主动寻找情绪**
    *   描述：以结果和利益为导向，主动选择展现何种情绪来推动事情向好的方向发展。例如，刻意表现出生气、笑容或威严来达成目的。
4.  **第四层：控制他人情绪**
    *   描述：能预判他人的情绪反应，并通过自身的状态、肢体语言和神情等，引导或控制对方的情绪走向。

### 情绪不稳定的核心原因

理解了层次，我们需要探究情绪不稳定的根源。主要原因有以下几点。

**原因一：与认知和经历有关**

让我们痛苦或情绪波动的，往往不是事件本身，而是我们对事件的**认知**和看法。例如，分手后，认知为“这辈子再也不能和这个人在一起”会带来巨大痛苦；而认知为“这只是关系的结束，并非感情的终结”则平和许多。

情绪系统有一套信息处理机制，它与你的认知能力自动对接。公式可以表示为：
**事件 (A) -> 认知 (B) -> 情绪与行为 (C)**

例如：
*   认知“蝙蝠很可怕” -> 见到蝙蝠 -> 情绪“害怕”。
*   认知“伴侣不回消息=不爱你” -> 两小时未收到回复 -> 情绪“悲伤”。

问题在于，我们的认知系统中存在许多**不合理的信念**，这些信念直接导致了负面情绪。

**原因二：轻易付出核心价值**

这涉及到关系中的价值交换失衡：
*   **男人的核心价值**：钱。
*   **女人的核心价值**：身体。
当一方过度付出自己的核心价值（如男人花太多钱，女人过早发生亲密关系），就容易因害怕失去或感到不公而情绪不稳定。

**原因三：不以结果为导向**

面对问题时，错误的反应模式是：问题出现 -> 情绪不稳定 -> 问题扩大 -> 情绪持续不稳定，形成**负向循环**。
正确的模式应是：问题出现 -> 忽略情绪，解决问题 -> 问题解决 -> 情绪平复，形成**正向循环**。

**原因四：关系失衡**

关系失衡本身并不可怕，可怕的是随之而来的**情绪失衡**。健康的态度是：可以接受关系失衡并选择离开，但要避免因情绪失衡而陷入又吵又闹的境地。

### 情绪的本质与作用

情绪是一个人对世界的反应，是自我与宇宙对话的形式。试想，如果将一个人与世界的所有联系切断，抛入虚无的黑暗宇宙，他将因记忆中的联系断裂而感到巨大痛苦。但如果连这些记忆也失去，他将不再有任何情绪，因为那个世界没有任何能引发情绪波动的要素。

反之，强烈的情绪正说明你与这个世界或他人有着激烈的链接。例如，看到“屌丝不配有优质男”的文章感到愤怒，这愤怒其实提醒了你内在可能将自己归为“屌丝”，并在恋爱中有受挫经历。文章只是挑起了你潜意识中的创伤或不合理认知。

## 二、如何利用他人的情绪达成目的 🎯

上一节我们深入探讨了如何管理自身情绪，本节中我们来看看如何将情绪作为一种策略工具，尤其是利用他人的情绪。

### 1. 主动运用情绪作为工具

核心思想是：不是让情绪控制你，而是你主动去**寻找**和**使用**情绪。情绪只是达成目的的工具，目的达到后，工具就应收起。

**操作示例（以吵架为例）：**
1.  **牢记结果思维**：明确你希望对方具体做什么。
2.  **释放情绪**：根据目的，表现出适当的情绪（如愤怒、失望）。
3.  **提出需求**：清晰说出你的要求。
4.  **修正行为与立规矩**：指明对方需要改变的地方，并设立规则。
5.  **给予台阶并收场**：在对方有悔意或妥协时，适时缓和气氛，结束冲突。

**职场示例（领导对下属）：**
*   错误做法1：不释放任何不满 -> 损害领导权威。
*   错误做法2：直接爆发情绪 -> 无法解决问题。
*   正确做法：先释放**部分**不满情绪，再提出具体的解决方案。

### 2. 提供情绪价值

情绪价值是人际交往中强大的“白嫖”工具。对于女性而言，虽可弱化其重要性，但需懂得以此来影响男性。优质异性身边不缺美女，因此**情绪价值**变得尤为关键。

情绪价值来源于波动，包括：
*   **正面情绪价值**：被赞美、被重视、被认可。
*   **负面情绪价值**：失落、焦虑、不被认可、危机感。

不同人群对情绪价值的需求比例不同：
*   **年轻男生（25岁以下）**：喜欢被“拉扯”的状态，正负情绪价值需求相当。
*   **优质男性（28岁以上）**：遵循 **80% 正向，20% 负向** 的原则。他们日常处理大量人际关系，更需要让其感到舒适的情绪价值。

#### 技巧一：接纳高阅历者的情绪

当对方暴露弱点时（如“我压力很大，没人懂我”），不要简单说“没事”。使用以下公式：
**接纳弱点 -> 顺着弱点 -> 表示接纳**

示例对话：
> “我知道，很多人会觉得你朋友那么多，可能不是一个特别靠谱的选择。（接纳弱点）但是我却依然愿意选择跟你相处，你知道为什么吗？（顺着弱点）因为你在我们面前偶尔示弱、偶尔崩溃的这一刻，我会觉得，你活得这么完美，其实挺累的。（表示接纳）”

#### 技巧二：寻找相同点建立链接

相同点不仅是表面的爱好，更是深层的**追求**。例如，两个原生家庭背景不同的人，可以找到本质的相同点。

示例话术：
> “其实我特别能理解你。我是在一个不缺爱但物质普通的家庭长大的。而你正好相反。所以走入社会后，你会非常渴求爱，而我会非常渴望物质回报。本质上，我们都是一样的，都在用一生追求那些最初缺失的东西。”

#### 技巧三：建立真正的信任

信任不等于交换秘密或过于热情。

*   **错误1：说朋友坏话**。你身边的人是你人设的一部分，请分享朋友**正能量**的故事。
*   **错误2：好为人师，过于热情**。牢记八字真言：**不问不答，不求不助**。不要让你的帮助显得廉价。

**建立信任的正确方式：**

1.  **共情**：
    *   核心：不是认同对方所有行为，而是理解“如果我经历了你的人生，面对你的选择，我也可能会这么做”。探讨一个人为何如此，比直接评判其行为更重要。
    *   代码式理解：`理解感受成因` 比 `理解感受本身` 更有效。

2.  **树立框架**：
    *   信任只建立在你有原则的基础上。学会拒绝是树立框架的关键。
    *   **拒绝理由**：
        *   **a. “我不喜欢/不想要”**：每个人都应学会直接拒绝，不留迂回空间。直接拒绝好过答应后拖延。
        *   **b. “你给我的东西不够”**：把拒绝当作交换筹码。
    *   **拒绝公式**：肯定对方 + 拒绝原因 + 合理化原因 + 间接提出要求。
        *   *示例1（拒绝表白）*：“我觉得你这个人哪哪都好（肯定），但我觉得你有时不够细心（拒绝原因）。我也很粗心，如果找个这样的男朋友，生活可能一团糟（合理化）。我觉得我更需要一个跟我互补的人（间接要求）。”
        *   *示例2（拒绝表白）*：“能被你这么优秀的人喜欢，我很开心（肯定）。但我们认识时间太短，还不够了解彼此（拒绝原因）。我觉得我更需要一个愿意花心思了解我的人（间接要求）。”

**建立信任的题外原则：传递“吃软不吃硬”的信号**

保持框架一致性，运用“三次妥协原则”：
*   信息发三条不回，绝不发第四条。
*   电话打三次不接，绝不打第四个。
*   道歉给三次机会不原谅，绝不给第四次。
*   请客吃饭三次，绝不清第四次。
*   对方冒犯你三次，绝不给他第四次机会。
