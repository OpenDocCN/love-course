# 礼仪派公开课精选：直播聊天：1对30成功邀约敲定7人

## 概述
在本节课中，我们将学习如何通过直播聊天成功邀约。我们将分析一位讲师如何在与30位女性聊天中成功邀约7位。

## 一、直播聊天技巧
以下是直播聊天的一些技巧：

**1. 聊天内容**：
* **真实性**：向对方说明自己的情况，包括来昆明的目的、个人情况等。
* **目的明确**：告诉对方自己为何发消息，以及邀约的目的。
* **提供选择**：提供喝酒、单约、白天或晚上的选项。

**2. 聊天节奏**：
* **简洁明了**：避免废话，直接表达自己的想法。
* **快速回复**：在有限的时间内回复对方，提高效率。
