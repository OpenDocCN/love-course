# 乐嘉7套：1.乐嘉-FPA性格色彩培训高清：乐嘉（二）

## 概述
在本节课中，我们将学习如何运用情感的力量影响不同性格的人，以增强演讲效果。我们将探讨如何根据观众的性格特点调整演讲策略，以及如何通过提问技巧影响观众。

## 核心概念
**公式**： 情感力量 = 真诚 + 共鸣 + 互动

## 教程内容

### 1. 情感力量的重要性
乐嘉老师强调，情感的力量在演讲中至关重要，它往往比理智的力量更具有影响力。因此，演讲者需要学会运用情感的力量去影响观众。

### 2. 不同性格的观众
乐嘉老师指出，不同性格的人对情感的反应不同。红色和蓝色性格的人情感丰富，容易受到情感力量的影响；而黄色和绿色性格的人情感相对较弱，需要采取不同的策略。

### 3. 影响黄色和绿色观众的技巧
对于黄色和绿色观众，乐嘉老师建议：
- **黄色观众**： 以实际行动和成果为主，避免过多的情感表达。
- **绿色观众**： 通过提问和互动，引导他们参与进来，激发他们的情感。

### 4. 提问技巧
乐嘉老师强调，提问技巧在演讲中非常重要。他建议：
- **选择合适的提问对象**： 选择愿意互动、信任并能够交流的观众。
- **避免负面提问**： 避免提问可能引起观众反感的敏感话题。

### 5. 案例分析
乐嘉老师通过一个案例，说明了不同性格的人对同一事件的反应不同。黄色性格的人更注重实用和实惠，而红色性格的人更注重情感和陪伴。

### 6. 成功的秘诀
乐嘉老师分享了他取得成功的秘诀：
- **热爱演讲**： 对演讲充满热情，并从中获得快乐。
- **不断学习**： 不断学习新的知识和技巧，提升自己的演讲水平。

## 总结
本节课中，我们学习了如何运用情感的力量影响不同性格的观众，以及如何通过提问技巧影响观众。通过了解观众的性格特点，我们可以调整演讲策略，提高演讲效果。
