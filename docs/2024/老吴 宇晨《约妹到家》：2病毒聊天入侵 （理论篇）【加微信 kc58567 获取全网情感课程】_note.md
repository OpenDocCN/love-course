# 老吴 宇晨《约妹到家》：2病毒聊天入侵 （理论篇）

## 概述
在本节课中，我们将学习如何进行有效的聊天，以实现与目标对象的深入互动和建立联系。

## 一、聊天分类
以下是四种主要的女性分类，以及相应的聊天策略：

### 1. 正经良家型（A类）
- **特点**：保守、传统，注重安全感。
- **聊天策略**：多聊、聊久，内容朴实。
- **公式**：`聊得多 -> 聊得久 -> 建立信任 -> 发展关系`

### 2. 精致文艺型（B类）
- **特点**：喜欢展示生活，追求高逼格。
- **聊天策略**：展示自我，简洁回应。
- **公式**：`展示自我 -> 简洁回应 -> 吸引注意 -> 拉近关系`

### 3. 玩家花蝴蝶型（C类）
- **特点**：喜欢展示颜值和身材，享受男性追捧。
- **聊天策略**：直接邀约，避免深入聊天。
- **公式**：`直接邀约 -> 避免深入 -> 享受追捧 -> 维持关系`

### 4. 闷骚型（D类）
- **特点**：介于A类和C类之间，工作稳定，内心渴望新鲜感。
- **聊天策略**：平衡正经与轻松，适时展示自我。
- **公式**：`平衡正经 -> 适时展示 -> 激发好奇 -> 发展关系`

## 二、资源管理
在添加女性后，进行以下步骤：

### 1. 分组与标签
- **分组**：根据分数（5、6、7分）和分类（A、B、C、D）进行分组。
- **标签**：包括城市、工作、兴趣爱好等。

### 2. 定期清理
- **目的**：保持微信列表整洁，避免僵尸粉占用空间。
- **方法**：定期删除不再联系的人。

## 三、聊天技巧
### 1. 双线聊天
- **定义**：时而正经，时而轻松。
- **目的**：避免过于严肃或过于随意。

### 2. 聊天核心
- **核心**：邀约和拉伸关系。
- **公式**：`聊天 -> 拉伸关系 -> 邀约 -> 见面`

### 3. 发散性思维
- **定义**：使用戏谑、夸张等手法，使聊天内容更具趣味性。
- **公式**：`关键词1 -> 关键词2 -> 意料之外 -> 情理之中`

## 四、邀约时机
- **时机**：聊天高潮时。
- **目的**：利用情绪化，促使对方做出决定。

## 总结
本节课中，我们学习了如何进行有效的聊天，包括聊天分类、资源管理、聊天技巧和邀约时机。通过掌握这些技巧，我们可以更好地与目标对象建立联系，实现我们的目标。
