# 钱进社课程60：如何摆脱“努力的废人” 🚀

在本节课中，我们将深入探讨一个普遍现象：为何许多人看似努力，却收效甚微，最终沦为“努力的废人”。我们将剖析其背后的核心原因，并提供一套清晰、可执行的行动框架，帮助你摆脱无效努力，实现真正的成长与突破。

---

## 概述：问题的本质

很多人常感到困惑：自己明明很努力，却似乎总是不见起色，于是将原因归结为缺乏机会或贵人相助。然而，机会与贵人的降临，更大程度上是基于你的**能力**，而非纯粹的运气。这里的能力，并非字面上的广义理解。如果你对“能力”的理解存在偏差，那么你努力的方向很可能从一开始就是错误的。

方向错误的问题会带来持续的内耗，却无法产生有效的解决方案。因为有效的方案必须基于一个**真实且正确**的问题本身。成年人最宝贵的资源是时间，若在错误的方向上消耗数年，这些时光便等同于白费。

我们常说重视过程，因为结果是过程的最终答卷。问题的关键就在于，每个人用怎样的思维、行为和方式来对待属于自己的这个过程。这就引出了我们今天要讨论的核心：你是否是一个“伪工作者”？

---

## 第一节：识别“伪工作”——你为何忙碌却无效？

“伪工作者”指的是那些终日忙碌、上蹿下跳，但最终业绩、认可度和结果都不理想，甚至很糟糕的人。他们的“努力”是一种自我消耗。

上一节我们指出了努力方向的重要性，本节中我们来看看，在你的工作过程中，是否也存在以下导致“伪工作”的典型问题。

以下是第一个关键板块：你是否在干不该干的事。

### 1. 干了不该干的事情

你很焦虑、很积极、也很忙，但这改变不了你在处理不属于自己分内之事的事实。在职场上，**必须优先做好分内之事**，并在此基础上有闪光点和结果产出，这是奠定你职场位置的基石。

*   **态度不等于结果**：老板可能注意到你加班的态度，但态度是抽象的，而结果是具象的。公司生存依赖的是能拿出老板所需结果的人。
*   **帮忙的前提**：无论谁请你帮忙，都必须以保证能更优质地完成自己本职工作为前提。插手他人工作，应基于两个现实且不丢人的出发点：
    1.  **增强自身锻炼**：为了学习或锻炼新技能。
    2.  **完善人际关系**：为了在平台上建立更好的人际网络。
*   **如何拒绝**：当他人提出请求时，需遵循两个原则：
    1.  **不帮违反原则或自己无能为力的事**：若帮不上，应直接说明。拖延回复会让对方产生期待，落空后反而损害关系。
    2.  **分清问题的归属**：判断哪些问题是对方自己能解决的，哪些是真正需要你帮忙的。例如，帮人代购价值高、复购率低的商品（如新款手机），能带来附加好感；而代购价值低、复购率高、品类繁多的商品（如奶粉），则应拒绝，因为这会成为无休止的负担，且你的帮助并非不可替代。

**核心要点**：要做一个**有用的人**，而非纯粹**好说话的人**。有用体现在你做事（无论是本职工作还是帮助他人）的层面和价值；而好说话则可能让你成为一个轻易被替代的、可有可无的人。对于职场新人，积极做事是证明自己“被需要”的重要途径。

## 第二节：时间与精力的管理误区

在明确了工作重心后，我们来看看另一个常见的陷阱：对时间和精力的错误管理方式。

以下是第二个关键板块：你是否在同时追逐太多目标，却无一精通。

### 2. 喜欢多线作战，却无一精通

很多人迷信“时间密度”，认为同时做很多事、把日程排满就是努力。这是彻底的误解。

*   **了解自我**：每个人都是不同的。盲目模仿他人的工作强度和模式，而不考虑自身底子和类型，很容易导致快速放弃。重要的是找到适合自己的节奏和方法。
*   **警惕自我感动**：不要用做事的“密度”（如每天5点起、12点睡）来衡量努力程度。这可能是**自我感动**。当你的内在能力无法支撑过载的任务时，就会开始犯错。
*   **小错的累积**：起初，你会为小错误找理由（“太累了”“下次注意”）。但当精力与脑力过度积压时，这些小问题会集中爆发成一个大问题。工作中不断犯小错且善于自我合理化的人，其最终结果必然不会好，且会越来越偏离正轨。

**公式**：**有效努力 ≠ 时间密度 × 事务数量**；**有效努力 = 专注深度 × 目标完成度**

## 第三节：摒弃投机心态，专注长期价值

解决了行为模式问题，我们还需要审视更深层的心态问题：你是否在追求不切实际的捷径？

以下是第三个关键板块：你是否迷信速成效应。

### 3. 迷信速成效应

世界总体是公平的。极不公平的机会（漏洞）一旦被大众知晓并涌入，就会迅速消失或价值被稀释。

*   **问题的本质**：当人们问“什么领域有更好发展/更能搞钱”时，其潜台词往往是“什么工作能更快赚钱且不那么辛苦”。然而，**轻松且回报率高的工作几乎不存在**。
*   **长线思维**：必须对标长线规划。现在回报高，不等于未来回报高。抱有投机取巧心理的人，是在与整个系统（宇宙）对抗，终将被现实击碎。
*   **速成陷阱**：速成一族极易成为“白忙”和“瞎忙”的代表。例如，被“9.9元，7天提高情商”这类广告吸引而购买课程，两周后却早已遗忘。这背后的逻辑是：你付费的初衷是“占便宜”（只需9.9元），而非真正“提高情商”。功夫下不够，不明确自身需求，任何低价或免费的东西都无法产生价值。

**代码描述速成心态**：
```python
if 用户动机 == “占便宜” or 用户动机 == “逃避长期投入”:
    行动 = “购买速成课”
    结果 = “遗忘课程，无实际改变”
elif 用户动机 == “认清需求，愿意持续投入”:
    行动 = “制定计划并坚持执行”
    结果 = “逐步提升，获得长期回报”
```

## 第四节：从思维到行动的系统改变

分析了所有问题板块后，最关键的一步是如何系统性地改变。这需要从思维到行为建立新的闭环。

以下是实现改变的行动框架。

### 4. 建立“思考-行动-复盘”的闭环

从现在开始，请列出一个清单，并遵循以下步骤：

1.  **先动脑，后动手——做减法**：
    *   不要急于往清单里添加新任务。首先审视清单上现有的所有事项（ABCD……）。
    *   启动你的大脑，开始**删除**那些非必要的事项。把“做减法”作为第一个动作。

2.  **审视每一个“开始”**：
    *   对于每一件你开始了却没有明确收尾的事情，必须总结：**你当初为什么选择开始它？**
    *   很多人不断开始新事情，却从不追问为何没有结尾，导致生活像一滩浆糊，不断开头却毫无积累。

3.  **强制复盘与打分**：
    *   每完成一件事（无论结果好坏），都必须进行复盘：你**得到了什么**，**失去了什么**？
    *   对这件事的最终效果**进行打分**。只有量化的打分，才能让你看清得失，建立完整的评判体系。
    *   通过复盘，你才能知道哪些经验可以借鉴到新的开始中，哪些老路可以避免。没有清晰的复盘，不断的“开始-要结果”循环，只会产生混乱。

**核心循环**：**计划（做减法） → 执行（审视动机） → 复盘（评估得失） → 优化（指导新的开始）**

## 总结与行动号召

本节课中，我们一起深入探讨了“努力的废人”背后的四大症结：
1.  **干了不该干的事**，忽略了本职工作的核心价值。
2.  **盲目追求多任务并行**，却无一精通，陷入自我感动。
3.  **迷信速成与投机**，缺乏长期主义的耐心和规划。
4.  **缺乏“思考-行动-复盘”的系统闭环**，不断开始却从不总结。

摆脱无效努力的关键，在于将思维从“盲目忙碌”转向“精准发力”。你需要先确保方向正确（做好分内事），然后管理好精力（专注而非贪多），接着摆正心态（追求长期价值），最后建立可持续的改进系统（复盘优化）。
