# 46、牵手丘比特《文小暖课程学习》：9.第7节课：读懂女人的真假兴趣及邀约技巧

## 概述
在本节课中，我们将学习如何识别女人的真假兴趣以及邀约技巧。

## 读懂女人的真假兴趣
### 真假兴趣性指标
在相处过程中，女人会释放出真兴趣和假兴趣。**公式**：真兴趣 + 假兴趣 = 兴趣性指标。

### 服从性测试
服从性测试是判断真假兴趣性指标的一种方法。**公式**：服从性测试 = 真兴趣性指标。

### 女人的心理
女人在被男人吸引时，会掩饰自己的兴趣，进行迂回战术，试图让男人先表达好感。

### 女人的兴趣性指标
1. **眼神**：认真被吸引的眼神会带有兴奋的光芒感。
2. **肢体接触**：偶然的肢体接触，如碰触、靠近等。
3. **话题参与**：主动参与话题，并有意调动节奏互动。
4. **服从性**：对语言调戏或肢体接触的反应。

## 邀约技巧
### 模糊性邀约
在邀约前，先进行铺垫，给出一个去的理由，并建立安全感。**公式**：铺垫 + 去的理由 + 安全感 = 模糊性邀约。

### 模糊性邀约的二次方法
模糊时间 + 模糊地点 = 明确性邀约。

### 邀约失败的处理
邀约失败时，用半认真、半开玩笑的语气表达失望，并为下一次邀约做铺垫。

## 总结
本节课中，我们学习了如何识别女人的真假兴趣以及邀约技巧。希望这些知识能帮助你在感情中更加得心应手。
