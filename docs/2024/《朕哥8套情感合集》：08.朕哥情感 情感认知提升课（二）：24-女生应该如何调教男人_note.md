# 《朕哥8套情感合集》：08.朕哥情感 情感认知提升课（二）：24-女生应该如何调教男人

🎼，🎼如何调教男人，明白了你就厉害了。人性相通，理论上学会调教一个男人，就能调教所有男人。学会调教男人，可以做很多事情，其实是在培养男人。

可能像训练宠物一样，但细节不同。理论上，做错时给予负面反馈，做对时给予正面反馈。但正负反馈又有所不同。

很多女生学完课程后，用方法调教男人，发现特别有效。男人会按照预设思路发展，预判男人的预判。

建立规则，男人会按照要求去做，这叫高端位。希望女生都能成为高段位，学会调教男人是必修课。

首先看男人做错时怎么办？很多女生会责骂、冷暴力，但男人不会因此屈服。男人吃软不吃硬，不能逼他叛逆。

真正高段位女人会先自责，转移话题，让男人陷入愧疚或失智状态，然后鼓励他，他会愿意听你的。

调教男人时，注意只有一个结局：皆大欢喜。不要与男人对抗，目的是共赢。

男人做对时，不要无底线夸他，否则他会膨胀。培养调教男人的过程，做对时表示感动，引导他继续做，做错时不要批评，惩罚。

男人没做对、没做错时，设立激励，告诉他怎么做你才会怎么对他。调教男人看似简单，难在实操。

控制情绪，男人做对时淡定，做错时也淡定。男人没做对、没做错时，让他有波动。

调教男人很重要，但记住选择大于努力。在调教前筛选优质男人，找到你最喜欢的类型。
