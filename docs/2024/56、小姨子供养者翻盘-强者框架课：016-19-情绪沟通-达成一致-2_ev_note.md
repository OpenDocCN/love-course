# 社交舞步法与情绪沟通法教程：达成一致

## 概述
在本节课中，我们将学习如何通过情绪沟通法中的“达成一致”步骤，有效地与对方建立联系并达成共识。

## 达成一致的重要性
**公式**：达成一致 = 兴趣 + 情绪共振 + 吸引 + 联系感

达成一致是社交舞步法中的关键步骤，它要求我们在与对方沟通时，加入一些美好的期待，从而更容易达成共识。

## 诱惑性的预期
在达成一致的过程中，植入一些有诱惑性的预期，可以增强诱惑性的情绪。

**代码**：
```python
# 植入预期
def add_expectation():
    # 添加美好的期待
    print("期待与您共度美好时光。")
```

## 例子分析
### 例子一：邀请对方吃火锅
- **小诱惑**：有机会一块出来吃火锅。
- **大诱惑**：我认识成龙，有机会出来，我带你去他家玩。

### 例子二：面试
- **小诱惑**：我会带着对咱们公司产品的修改建议来。
- **大诱惑**：我真是太喜欢咱们公司了。如果有机会合作的话，我也想在这个项目上投一点点钱。

## 注意事项
- 诱惑性不能太大，不能超过你本身对对方的吸引。
- 避免让对方为了诱惑而改变你们的关系本质。

## 总结
本节课中，我们学习了如何通过情绪沟通法中的“达成一致”步骤，有效地与对方建立联系并达成共识。通过植入有诱惑性的预期，我们可以增强诱惑性的情绪，从而更容易达成共识。
