# 课程01：Python基础入门 🐍

在本节课中，我们将要学习Python编程语言的基础知识，包括变量、数据类型和基本运算。这些是构建任何Python程序的基石。

## 什么是Python？🤔

Python是一种高级、解释型的编程语言。它以简洁的语法和强大的功能而闻名。Python的设计哲学强调代码的可读性。

## 变量与赋值 📦

上一节我们介绍了Python语言，本节中我们来看看如何存储数据。变量是存储数据的容器。你可以把变量想象成一个贴有标签的盒子。

在Python中，使用等号 `=` 进行赋值操作。等号左边是变量名，右边是要存储的值。

以下是变量赋值的基本语法：
```python
variable_name = value
```

例如：
```python
name = "Alice"
age = 25
```

## 基本数据类型 🔢

变量可以存储不同类型的数据。Python有几个内置的基本数据类型。

以下是三种最常用的数据类型：

*   **整数 (int)**： 表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数 (float)**： 表示带有小数部分的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str)**： 表示文本数据，需要用单引号 `' '` 或双引号 `" "` 括起来，例如 `"Hello"`, `'Python'`。

## 基本运算 ➕➖✖️➗

我们可以对数据进行各种运算。Python支持常见的数学运算。

以下是基本的算术运算符：

*   **加法 (+)**： `5 + 3` 的结果是 `8`。
*   **减法 (-)**： `10 - 4` 的结果是 `6`。
*   **乘法 (*)**： `6 * 7` 的结果是 `42`。
*   **除法 (/)**： `8 / 2` 的结果是 `4.0`。注意，除法运算的结果总是浮点数。
*   **取整除 (//)**： `9 // 2` 的结果是 `4`。它返回除法结果的整数部分。
*   **取余数 (%)**： `9 % 2` 的结果是 `1`。它返回除法后的余数。
*   **幂运算 (**)**： `2 ** 3` 的结果是 `8`，表示2的3次方。

## 字符串操作 🔤

字符串也可以进行特定的操作，最常用的是连接。

字符串连接使用加号 `+` 运算符。例如，`"Hello, " + "World!"` 的结果是 `"Hello, World!"`。

---

本节课中我们一起学习了Python的基础概念：我们了解了Python是什么，学会了如何使用变量存储数据，认识了整数、浮点数和字符串这三种基本数据类型，并掌握了如何对它们进行算术运算和字符串连接。这些是开始编写Python程序的第一步。
