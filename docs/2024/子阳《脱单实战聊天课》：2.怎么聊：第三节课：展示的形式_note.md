# Python基础教程 1.1

## 🌟 概述

在本节课中，我们将学习Python编程语言的基础知识，包括变量、数据类型、运算符和基本语法。

## 📚 变量和数据类型

### 什么是变量？

变量是存储数据的容器。在Python中，变量不需要声明类型，直接赋值即可。

**公式**：`变量名 = 值`

### 数据类型

Python中有多种数据类型，包括数字、字符串和布尔值等。

- **数字**：整数（int）和浮点数（float）
- **字符串**：文本数据（str）
- **布尔值**：True或False（bool）

## 📚 运算符

运算符用于对变量进行操作。

以下是Python中常用的运算符：

- **算术运算符**：加（+）、减（-）、乘（*）、除（/）、取余（%）、幂（**）
- **比较运算符**：等于（==）、不等于（!=）、大于（>）、小于（<）、大于等于（>=）、小于等于（<=）
- **逻辑运算符**：与（and）、或（or）、非（not）

## 📚 基本语法

### 注释

注释是代码中的非执行部分，用于解释代码。

**代码**：`# 这是注释`

### 输出

使用`print()`函数可以输出文本或变量的值。

**代码**：`print("Hello, world!")`

### 输入

使用`input()`函数可以接收用户输入。

**代码**：`name = input("请输入你的名字：")`

## 📚 总结

本节课中我们一起学习了Python的基础知识，包括变量、数据类型、运算符和基本语法。希望这些内容能够帮助你更好地理解Python编程。
