# 情绪价值课-【阿娇】：第19节 如何通过吃醋给对方情绪价值

## 1. 假装吃醋来调节情绪

当对方身边出现让你不舒服的人或产生负面情绪时，可以通过假装吃醋来调节情绪。例如，你的男朋友在公司有很多美女同事，你可以假装对所有美女都吃醋，而不是针对某一个人。这样既能转移注意力，又能让他觉得你很大度。

**示例代码**：
```python
# 假装吃醋
def act_jealousy():
    print("哎呀，你们公司美女好多啊，我要是个男的，我也愿意去上班。")
    print("要不这样，我去做你的秘书好不好？")
```

## 2. 假借吃醋之名来调节情绪氛围

当对方穿新衣服或得到夸奖时，你可以假装吃醋，实则是在调节情绪氛围。例如，当别人夸他新裤子好看时，你可以模仿夸奖他的方式，夸赞他的其他优点。

**示例代码**：
```python
# 假借吃醋夸奖
def act_jealousy_and_compliment():
    print("哎呀，你这个裤子好好看呢，看起来腿好长啊。")
    print("你这个衬衫好好看呢，看起来胸好大呀，让我摸一摸吧。")
```

## 3. 假借吃醋之名趁机敲打

当对方出现让你有危机感的行为时，可以假装吃醋来敲打他，提醒他注意自己的行为。例如，发现他在手机上与前女友聊天，你可以通过讲述自己的经历来提醒他。

**示例代码**：
```python
# 假借吃醋敲打
def act_jealousy_and_reprimand():
    print("你跟你前女友的关系好像不错，挺好的吧。我能理解你。")
    print("毕竟前任曾经在我们生命里很重要。有些时候，我们当下过得再好，也没有办法完全忘记以前的人。")
```
