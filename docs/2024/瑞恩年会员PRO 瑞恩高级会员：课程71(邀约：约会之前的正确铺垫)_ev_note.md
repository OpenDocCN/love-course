# 瑞恩年会员PRO 瑞恩高级会员：课程71(邀约：约会之前的正确铺垫)

## 概述
在本节课中，我们将学习如何通过有效的聊天铺垫，为约会创造良好的氛围。

## 案例分析
### 开启话题
**原文**：我说你说。人生的未完成情节有必要去完成吗？
**分析**：通过提出一个开放式问题，引导对方思考，并保持对话的创造性。

### 情感回应
**原文**：他说有必要去完成，但人生还是很长的。
**分析**：对方回应表明其愿意参与对话，并表达了对话题的理解。

### 话题深入
**原文**：我说想见一个人，他说在哪里呢？我说成都啊，他说我在巴中，就是他的老家啊，他是在南充那边的，在巴中那边的。然后呢我说我在广州。
**分析**：通过分享个人信息，拉近彼此距离，并了解对方的地理位置。

### 情绪调动
**原文**：我说一个说要来接我的死女人。我说搞半天他也不在，你看。我不在成都，我还没回去，我人在广州开开启话题，先跟他聊，看他人在不在，已经可以铺垫这个见面了。
**分析**：通过幽默的方式，调动对方情绪，并暗示见面可能性。

### 接纳起伏
**原文**：那么这个案例啊，从一开始加到我在国外跟他聊，到上飞机飞的时候跟他聊，到回国以后，在广州跟他聊，你会发现这四个来回当中啊，我们一直在保持一种状态。我们俩之间一直有情绪在流动着，一直在保持这种状态。只实这个状态。你看他有起伏啊，那么有起伏呢是一件很正常的事情。
**分析**：接纳关系中的起伏，保持情绪流动，有助于关系的稳定发展。

## 总结
本节课中，我们学习了如何通过有效的聊天铺垫，为约会创造良好的氛围。通过开启话题、情感回应、话题深入、情绪调动和接纳起伏等步骤，我们可以逐步拉近与对方的距离，为约会做好准备。

**本节课中我们一起学习了如何通过有效的聊天铺垫，为约会创造良好的氛围**。
