# 清风老师合集：09不委屈，做个真正高情商的人 第11期：098.第三课：如何通过高情商，做个“不委屈”的人 🧠

在本节课中，我们将要学习如何运用高情商的核心能力，避免在生活中产生“委屈”的感受。我们将探讨委屈感的来源，并重点介绍“自我调节”这一情商要素及其四个核心维度。最后，我们会将这些理论应用到情感和工作两个具体场景中，提供可操作的步骤。

在上一节课程中，我们介绍了良好情商能带来的益处和能避免的问题。本节中，我们来看看如何具体运用高情商，让自己成为一个“不委屈”的人。

为什么委屈的感觉会时常出现？我第一年工作时在一家上市公司。作为一名新人，我工作格外努力。我认为只要足够努力，领导就能看到我。领导确实认可我，但在任命项目负责人时，他选择了另一位同事。他的理由是，这位同事更加努力。当时我感到非常不服。明明这位同事平时只做领导重视的事情。而我无论领导是否重视，都努力完成所有工作。为什么我的付出没有得到回报？一气之下，我选择了离职。

很多年后我才明白，世界上有一个善良的谎言，叫做“付出就应该得到回报”。实际上，如果付出的方式方法不对，就得不到回报。或者有时方式对了但时机不对，也得不到回报。这需要我们用高情商去找到关键点，在付出与回报之间建立良好的桥梁。

第二个常见的观点是：我对他那么好，付出那么多，他为什么不理解我？我那么爱他，为他做了那么多，他为什么不喜欢我？前段时间，我看到一个小伙子在天台上用榴莲告白。他说：“女神，我爱你，你最爱吃的榴莲，我给你拉了一车来。”女神对他笑笑说：“对不起。”这种误解在男女关系和社交关系中经常存在。有时，信号的传递方式比信号本身更重要。如果没有情商，传递信号的过程极易出错。

那么，高情商的第二个要素——**自我调节**——就可以帮助我们避免这种委屈感。

以下是自我调节能力的四个核心维度：

**第一个维度：良好的自制力**
听到自制力，如果立刻想到要压抑自己、满足别人，那就错了。自制力是对自己有真实的判断，能识别自己的真实需求，并能通过行动去满足。这样我们就不必一味向他人索取。另一种情况是，我们能对自己的情绪负责，不需要向他人索取情绪上的满足和安慰。这才是真正的自制力，其本质是自己对自己负责的能力。

**第二个维度：强大的责任心**
对自己所做的事情有原则，这种原则不会因别人的看法而轻易改变。如果总是活在领导或伴侣的看法中，就会经常打破自己的原则，因而难以建立责任心。责任心之所以重要，是因为它与意义感挂钩。你所做的事情有其自身存在的意义。若缺乏这种意义感，在工作执行中会感到痛苦，觉得委屈了自己的才华。要避免委屈感，就要建立良好的责任心。

**第三个维度：优秀的适应力**
委屈感还有一部分源于对环境变化缺乏适应能力。我的一位女性来访者曾问我：“为什么我的丈夫总是欺骗我？明明答应今天陪我去迪士尼，突然有工作他就得去。是不是在他心里我根本不重要？”提出这个问题时，我能明显感觉到她的委屈。但这个意外并非她丈夫所愿，他也不得不去应付。这位女性在此缺乏适应力，缺乏应对意外的能力，因此将情绪迁怒于丈夫，导致双方在亲密关系中都很委屈，争吵一发不可收拾。

**第四个维度：出色的创新能力**
能够创造性地利用内在和外在资源解决问题。很多时候，我们并非没有资源，而是执迷于用老方法重复错误，从而忽略了解决问题的资源。我的一位来访者与男友分手，两人本已谈婚论嫁。但因她一直“作”和“闹”，让对方失望。她问我：“现在还有办法吗？我觉得怎么哀求都没用了。”我帮她梳理了她自身的优势，让她更好地认识到自己内外的资源。我告诉她：“你们像两个合作企业。你在工作上的能力对他帮助很大，他父母也很欣赏你的专业能力。一味哀求是用你最不擅长的方式去渴望得到爱。你平时是个硬气、有原则的人，不必放弃原则。做好自己优秀的样子，对方会主动靠近。”通过这番操作，她发现自己真正的优势是工作能力。于是她把注意力专注在工作上。结果，两人前些天给我发消息说成功领证了。

以上四种能力构成了高情商的核心能力之一——自我调节能力，能有效帮你避免委屈感。

接下来，我将帮你梳理两个核心的应用场景。

**第一个场景：在情感中如何避免委屈。**
以下是四个原则：

1.  **对方需要原则**：信号的传达可能出错，因为对方根本不需要。你给对方一车香蕉，对方却只想吃梨。这时你还问对方感不感动，其实你只是感动了自己。
2.  **自我认可原则**：你做这件事应该是你自己喜欢的。如果为了满足对方去做一件自己根本不认可的事（例如，为满足对方情感需求而买一个你承受不起的十万块钱的包），这会让你很累，让感情变糟，使你时常处于委屈中。
3.  **允许拒绝原则**：当对方拒绝你时，你能接受这件事。不会因为被拒绝就觉得自己糟糕，而是能客观看待：对方拒绝的是这件事，而不是你这个人。
4.  **表达需求原则**：在感情中能够大胆提出自己想要的东西，并在合理范围内让对方为你付出。这才能让你在情感中成为一个真正高情商的人。

**第二个场景：在工作中如何避免委屈。**
以下是五个步骤：

1.  **任务理解**：要理解到位。领导关注的点可能与你不同。一方面，领导掌握更全面的信息，存在信息差；另一方面，领导担负战略方向的重任，你要做的是交付结果。因此，理解清楚领导对任务的规划非常重要。
2.  **责任界定**：责任界定不是推诿，而是要明确什么事情该你做，什么不该做。我的一个学员曾帮同事完成方案，前面90%的内容是同事写的，最后10%由他完成并签字提交。结果领导因客户对前面方案不满意而严厉批评他，让他重写，否则走人。他非常委屈。这就是没有做好责任界定。你签字提交方案，就意味着要对全部内容负责，无论谁写的，结果都由你承担。
3.  **带着方案提问**：很多职场新人总是带着问题去问领导，这会让领导心烦。如果你能带着自己的解决方案去提问（例如：“领导，我这样做可不可以？您看有什么可以完善的？”），领导会觉得你有上进心。
4.  **关注领导的关注点**：在项目进展中，战略会不断调整。关注领导的关注点不是一味讨好，而是高情商的体现，有助于你了解事情的大方向、大发展和大战略。
5.  **建立互助关系**：同事是你需要依赖的合作伙伴，与他们互帮互助非常重要。但请注意，互帮互助要在第二步“责任界定”清晰的前提下进行。有同事的支持和理解，你在遇到问题时就能更好地解决。
