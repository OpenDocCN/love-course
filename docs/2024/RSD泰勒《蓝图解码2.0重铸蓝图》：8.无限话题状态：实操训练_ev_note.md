# RSD泰勒《蓝图解码2.0重铸蓝图》课程8：无限话题状态：实操训练 🎤

在本节课中，我们将学习如何通过“融入语言”来提升表达的能量与感染力，摆脱思维定势的束缚，释放内在的创造力。

如果说词汇是语言的血肉，那么节奏和情绪就是它的心跳。真正擅长表达的人能让观众随着他的声音起伏，随着他的情绪跌宕。这并非依靠华丽的词藻堆砌，而是源于对语言本身的驾驭。即使没有任何修饰，高手也能让人欲罢不能。就像一首无伴奏的说唱，仅凭节奏的魔力就能引爆全场。仅仅使用“我很无聊”这句话，不同的语音、语调、表达形式和能量场，效果就截然不同。

当然，这并非一蹴而就。这需要我们克服对语言的既有记忆，跳出思维的定势，去感受语句背后的韵律和能量。那么具体该怎么做？如何锻炼出来？接下来，我们开始蓝图解码2.0的重铸蓝图之旅。

## 从“头脑”到“身体”：两种表达状态对比

首先，我们来对比两种不同的表达状态：一种是从“头脑”出发，另一种是从“身体”出发。

我将先尝试从“头脑”出发说话，这是大多数人习惯的方式。当我试图回到那种状态时，感觉就像在进行**自我物理暴力**。那种感觉非常糟糕。我站在这里，努力回到那种受社会规训的“头脑空间”。这感觉很不好，我非常讨厌它。

在“头脑”状态下，我会想：“我现在看起来怎么样？我正在被评判，你们都觉得我很差劲。然后我还得向你们证明我很好。”这种思维方式非常疯狂，就像社会大多数人陷入的一种恍惚状态。但为了演示，我试试看。

我试着说一些自我标榜的话，也许能帮我进入状态。比如：“我超级有钱。我还有个大家伙，非常大，非常令人印象深刻。我是个猛男。”即使我觉得这很可笑，但我在努力思考大多数人希望展现的样子。“我有个酷房子，我有很多粉丝，我超级酷。”我在努力模仿人们通常的思维方式。

当我处于“头脑”中时，我需要观众认为我很酷。我感觉自己不够好。我不知道我是否能成为一个好榜样。我只是想为观众做个好示范。你能感觉到我大约有20%陷入“头脑”中了吗？你能感觉到能量的变化吗？

从“头脑”出发时，你会看到一个人困在自己的思绪里，他所说的话背后几乎没有力量。这会让你的表达变得非常缺乏说服力。

## 核心练习：融入语言

上一节我们看到了从“头脑”表达的局限，本节中我们来看看解决方案：**融入语言**。

现在，我想让你做的是：融入词语。让词语在你的身体里产生共鸣。我不是在对着麦克风演讲，我只是在对着摄像机说。看看内在循环。顺便说一句，我们稍后会深入探讨的部分，可能是本次活动最疯狂的部分。

你在那里看到的是他创造能力的物理显现。当他思考那个价值百万美元的想法时，那是一个“流产的胎儿”——百万美元的想法，就像“流产的胎儿”般的嗓音投射一样。

我表达的方式，完全符合我头脑中创造想法时的强度、专注、自由和解放的程度。我创造新内容、新商业想法的能力，与我此刻的说话方式如出一辙。原因在于，这不是一种技巧，而是一种**思维状态**。

## 释放被压抑的创造力

现在，我要告诉你，你比你意识到的更有创造力。我的意思是，想想你的梦有多富有创意？有些梦可能很平凡，就像日常生活。但有些梦，其想象力之广阔令人难以置信。如果你看过詹姆斯·卡梅隆的电影《阿凡达》，那可能只相当于我梦中景象的5%甚至1%。

我在梦中见过挪威峡湾与城堡，与巴西丛林交织，图案光怪陆离。如果我能画出这些东西，我能拍出最疯狂的电影或创作出最疯狂的艺术。梦中的情节、视角的转换、事物的涌现和变化，其创造力简直不可思议。

如果我在梦中能想到这些，如果你在梦中也能想到这些（也许不是峡湾城堡，但有你自己的天马行空版本），那意味着创造力就在你体内。只是你通过与所处的低振动能量社会的互动，被剥夺了这种创造力。我知道这一点，因为它也曾被从我身上剥夺。现在，我梦中的许多创造力回到了我身上，我可以用它来创造事物，包括赚钱、结识某人或获得约会。

此外，我还能用它来创造更深刻的东西。我创造的速度快得令人眼花缭乱，周围的人都目瞪口呆。但我并不是从“控制肌”去创造，我是从**释放**中创造。当然，这会很惊人。

你的问题在于，你试图通过收紧肛门来拉屎，然后还想把它推出来。所以，如果只射出一点点面条般的东西，那是正常的，对吧？但如果你只是释放……就像这样。

我公开演讲时，在座各位中我付出的努力最少。在社交、外出结识人、获得约会方面，我付出的努力也最少。我最不紧张，最不在乎，最漠不关心，就努力程度而言，我做的最少。然而，在外界看来，却是：“这家伙太有趣了！他怎么想出来的？这家伙太搞笑了！哦，他现在还和她亲上了？太疯狂了！”但在我的脑海里，我只是觉得……我什么都没做。你们太努力了。我看着你们，觉得你们这么努力，真可爱。就像你们把时间锁链拴在地上，却试图飞翔。

你们自己把锁链放在那里，所以你们必须解开它。那种力量，在家看视频的人无法完全感受到。我记得近距离看安东尼·罗宾斯演讲，当他走过我身边时，我能感觉到他身上散发出的原始电流。那是一种超现实的体验。遇到一个处于“释放”状态的人感觉很怪异，因为我们在社会中不习惯看到那样的人。我们不习惯看到一个不假思索、自由表达、不自我过滤的人。那里有一种创造力，我们不习惯。

人们会想：“他是怎么做到的？为什么他这样？这种电力从何而来？为什么他说话时，我能感觉到身体里有微小的脉冲？”正是这些微小的脉冲抓住了注意力，这就是吸引人的地方。你能在身体里感觉到它。

即使你很无聊，即使你很无聊，即使你很无聊……我也能抓住你的注意力五个小时，只是不断重复“即使你很无聊”。你以为必须有完美的话才能抓住注意力？你必须有一句很棒的话？你必须有一句不可思议的话？你必须知道正确的技巧？你必须把脚本写出来？你知道所有最厉害的人都有脚本吗？最吸引人的人，因为你必须做有效的事，所以你重复有效的事，得到相同的结果。这就像科学实验，这是科学，这是科学……即使你很无聊，这也是科学。你明白吗？这是科学。你必须控制一切。为了得到珍贵的东西，你必须控制。

即使你很无聊，你也必须控制一切，就像我现在做的一样。我真的在努力思考这个，我真的非常努力。我这么努力地想抓住你的注意力，太困难了，我正在因此失去能量。我的能量在下降，因为当你必须控制一切时，你的能量就会下降。你必须控制每一件小事，如果你不控制每一件小事……就不会成功。你永远无法抓住注意力。我永远无法得到你的注意力。我必须把每个字都选得完美无缺。如果我不说些深刻的话，我永远无法抓住你的注意力。我不知道如果没有完美的话说，我怎么能抓住你的注意力好几个小时。我可能会失去你的注意力。你可能会走出这里，你可能有更好的事情做，你可能开始看手机，你现在可能就渴望看手机。所以我必须把所有东西都脚本化。在过去的10分钟里，我大概只说了4个完整的句子。但这些句子挺不错的，对吧？所以，你最好有个系统。

我可以这样做一整天，而且，如果我这样做一整天，结束时我的能量会比开始时更多。为什么？因为我处于“释放”状态。大多数人处于一种压抑的意识状态，他们试图控制一切，因此他们正在耗尽能量，他们无法竞争。一旦你理解了这一点，你将碾压那些人。不仅碾压他们，而且你会像蒸汽压路机碾过一只虫子一样看待他们。虫子想：“你最好别惹我，我是只大虫子！”而你……你甚至没看到那只虫子，就直接碾过去了。你甚至没踩油门，你只是躺着。碾压一切。但有趣的是，当你碾压虫子时，你甚至不在乎。你只是开着压路机闲逛。那个在商业上与你竞争的人，你是他的整个世界，就像那只虫子，压路机就是它的整个世界。对你来说，它只是只虫子，你甚至不知道你在做什么。

我数不清有多少次，行业内的人来找我说：“你们是大佬，我们关注你们的一举一动。”而我的现实是，我根本没在想这个，我只是……哦，我在。

所以，要明白你有“控制肌”和“释放肌”。当你释放时，你**融入词语**，融入它。随之发生的是，当你说话时，你会感觉到微小的**电流脉冲**。砰，砰，砰。你正在把这种能量传递给对方。你会感觉到它。

我记得看到一些非常高水平的单口喜剧演员有这种特质，比如比尔·伯尔、乔·罗根，我看过他们的现场，充满电力。这些人是面对人群的大师级工匠。我的意思是，单口喜剧演员也是，水平完全不同。他们真的入错行了，如果他们转行做自助领域，会赚很多钱。但他们觉得，哦，在笑工厂舞台上赚200美元很酷。不过，有些人的确能通过Netflix特别节目致富。他们是世界级的演讲者，只是在一个报酬不高的细分领域。我们倾向于为娱乐支付较低的费用，我们为解决大问题支付更高的费用。尽管如此，他们中有些人也因此变得超级富有。

## 实践练习：从模仿到释放

理解了“放手，融入词语”的概念后，让我们做一些练习。现在，我为你设定的标准有点高，所以你可能感觉不太好。但你必须这样想：不要看我在这里做了什么，不要与我竞争，不要这样做。你必须处于一种**协作框架**中。

我会让你在这里走动一会儿。我想让你做的就是尝试对人群说“你会感到无聊”或类似的话。大致展示一下那会是什么样子。我们可以给你戴个小麦克风。展示一下，或者下一个人，如果不是他。现在下来。搞砸了也没关系。不要害怕搞砸。拿着这个，酷的是，如果你搞砸了，我们可以一起改进，无论如何，当你改进时会非常酷。也不一定非要面对整个人群。好的，兄弟，基本上你要做的是，用你自己的版本模仿我做的，并尝试玩转它，然后我会给你反馈并指导你。

他的投射怎么样？他是从头脑还是身体说话？继续。像这样不停地说：“你会感到无聊，对吧？想象一下听这个听四天。”他困在头脑里，他在乎，他在乎。像这样不停地说，一遍又一遍。我们能……你无聊了吗？你会感到无聊。我一直在说。所有的事情，所有的事情。现在尝试改变。这里有几件事可以改变：记得我们之前做的练习吗？改变节奏或表达方式的变化。试试那个。“你会感到无聊，像这样一直说，你会感到无聊。你无聊了吗？”好的，现在暂停。即使他还远未达到需要达到的水平，但比刚才容易观看一些了，对吧？他需要很多改进，但容易些了，对吧？继续做你正在做的事，你越来越接近了。“会无聊的，继续像这样，你会……”现在，顺便说一下，暂停。当你想出不同的节奏时，如果你试图记住节奏，它不会自然到来。要记的太多了，比如，唱歌式的、安静式的……太多了。这就像蜈蚣走路，如果蜈蚣开始思考，它会绊倒自己。在这个周末我们要带你达到的水平上，你无法通过控制来做到。你无法通过麦克风来修正。你做不到。所以，你唯一能做到的方式就是**放手**。你必须让它自然到来。其中一个练习就是如何让它自然到来。所以，让它自然到来。顺便说一句，当你看到我说话，我向你展示这些不同的方法时，你现在是否更清楚地看到我在做什么以及我是怎么做的？目前我说话的大部分内容，只是为了让你看到我在分解不同方面时的做法。所以，我现在说话不仅仅是为了说话。继续添加那个方面，继续尝试。你做得很好。

**更大声**，不过，要更大声，再大点声。好的，暂停一下，继续。更大声。再大点声。你们能听到他吗？现在再来一次。对付他们。你，你，你。用力点，兄弟，全力以赴。尽可能大声。你能看到自我审查吗？他害怕他会交配。这就是他这样做的原因。这背后有很多性方面的因素。有一种**交配恐惧**。他害怕他可能会与阿尔法男性的伴侣交配，从而被杀掉。这是一种人类穴居人机制。顺便说一句，这种穴居人机制也阻止你变得富有。你知道吗？你害怕如果你变得富有，你会有麻烦。有些人非常害怕发送……他们不学习阅读。你给他们看一本书，即使他们很聪明，他们也说：“不，不，不，不，不，不，我不想学阅读。”因为他们害怕如果他们学会阅读，他们会变得更聪明，然后他们变得更聪明，就会被杀掉。这是一种深层的恐惧。这是你可能处于的较低层次之一。好的，再来一次，大声点。

好的，你还是……展示一下你能发出的最大声音，就好像如果你的亲戚不大声说话就会死一样。现在，用那种音量，做你刚才做的事。不，用你能发出的最大声音，继续做你之前做的事。“你会感到无聊”等等。用你能发出的最大声音继续做那个。你不够大声，再大声点。“你知道，如果我继续像这样说话……你会感到无聊。这只是……”你只会……“如果我继续像这样说话，你会感到无聊，这很自然，你知道。”更多的变化，现在，“如果我继续像这样说话，你会感到无聊，你知道？”继续像这样说话，你会……更多的变化，更多的力量。尽可能地疯狂，像摇滚明星一样全力以赴地疯狂。继续。“那么，如果你会感到无聊，我为什么还要继续像这样说话呢？”越来越好了。是的，越来越接近了。我喜欢。好的，现在，顺便说一句。

在约会中，如果我站在一个所谓的“竞争者”旁边，另一个正在和某人说话的男人。如果我拥有更多能量，并且我向你展示的东西从我身上散发出来，我就是最有可能获得进展的人。这就是为什么即使我不是6英尺2英寸，我也能轻松地与6英尺2英寸的人竞争。非常非常容易。你会持续看到这一点。注意当我说话时，与一个新手说话时，你在能量上的感受有何不同。那种吸引力，有一个词来形容，你以前听过，叫做**吸引力**。吸引力不仅仅是异性相吸，也包括任何你被吸引的东西，无论性取向如何。不仅仅是性取向，而是你在能量上被吸引的东西。所以很多时候在约会中，我们认为我们必须获得身体上或性方面的吸引力，但它也可以是能量上的吸引力，随后再转化为身体上的亲密关系。它可以发展到那一步。

所以，我要和你一起走走，和你一起做一会儿，只是为了让你进一步热身，好吗？我……我只是重复：“你会感到无聊，一遍又一遍。”好的，你要和我一起做。“你会感到无聊，无聊，你会感到无聊。”和我一起说。“你足够好。”感到无聊，好的。那么我在那里做什么？我要让他同步，好吗？和我一起工作，或者好的，你得过来，好吗？好的，明白了。醒醒。说“醒醒”，醒醒。好的，醒醒，醒醒，醒醒，醒醒。你不会感到无聊。你不会感到无聊。好的，所以我想让他达到我的……我想让他跟上我的能量水平。好的，你不会感到无聊。你不会感到……现在我需要做的是融入词语。和我一起说，融入。融入词语。你习惯于从头脑说话。当你那样做时感觉如何？感觉有什么不同？我不需要思考，好的，很好。说“你感觉如何”，当我给你许可时，感受那种感觉，然后在你的身体里找到那种感觉。不要思考，当我停止时，你就会开始思考。很好。所以让我与你同步，但然后感受那种感觉，并在当我停止时保持它。好的，一个挑战：现在当我停止时，你是否感觉到自己有那么一瞬间变得紧张？是或不是？好的，这叫做社交意识。我能看出他没有告诉你。好的，所以，但现阶段这是件好事。好的，好的，好的，为那个鼓掌。请坐。

所以。好的，所以。哦，我可以拿过来。谢谢。和我一起做这个。融入。融入词语。开始！好的，现在当我们拉长声音时，用断奏更容易些，你会感觉到自己进出你的头脑。“怎么了？你好吗？你好吗？你好吗？怎么了？怎么了？你好吗？发生了什么？你要做什么？会怎么样？你好吗？”好的。所以你仍然……你会注意到，当你拉长声音时，融入其中会容易一点，但当它是断奏时，就更难了，所以只要让自己保持在外部，说“脱离我的头脑”。“在我的身体里，在我的身体里。在我的身体里。在我的和我的身体里。”好的，所以和你的小组一起，从那种范式开始练习说话。好的，就在那里，融入词语，融入世界。“你会感到无聊。你会感到无聊。你不会感到无聊。你不会感到无聊。”习惯于脱离你的头脑，融入世界。那会产生所谓的电力，和我一起说“电力”。好的，跳起来。我们现在就做，好的。

## 核心原理与总结

我们需要全身心地投入对语言的驾驭，关键是要学会**融入语言**，摆脱思绪的困扰，达到一种近乎忘我的状态。首先，我们要全神贯注，通过冥想式的练习清空杂念，专注于当下的语言。融入语言，意味着要像对待爱人一样，全身心地拥抱自己的语言，在节奏和韵律中达到身心合一的境界。

这种状态和性爱中的高潮相似，都需要人们暂时忘却自我，沉浸在当下的感受之中。虽然重复的动作可以帮助我们进入一种近乎催眠的状态，但我们一定要全身心地投入。正是因为在做爱做的事情中可以摆脱思绪的困扰，获得片刻的解脱和释放压力。要达到这种境界，肯定需要练习和勇气。我们要克服对于别人眼光的恐惧，克服对于自我的怀疑。其实每个人内心都有一个沉睡的表达者，渴望在言语中绽放。这需要我们一次次在融入语言的尝试中去寻找感觉。

我们可以看到，上台来被锤炼的小哥一开始沉溺于自我批判和他人眼光的焦虑状态中。在这种状态下，人们往往过度关注自己在他人眼中的形象，总是担心自己不够好，害怕被批评。这种担忧导致他在表达时小心翼翼，唯恐说错话或暴露弱点。他拼命想给人留下很酷的印象，希望获得他人的认可。但这种刻意的伪装实际上会削弱表达的力量，让人感到极其不真实和不自然，因为观众能够感受到背后的思绪以及他内心的不自信。

而泰勒告诉我们要跳出思维的桎梏，全身心地投入当下的自由状态。相比之下，当一个人能够放下自我批评，专注于当下，将身心完全投入到表达时，他的言语会充满感染力，能够带动氛围。因为这种状态下的表达不再是刻意的表演，而是发自内心的流露，表达者与当下的体验合为一体。语言不再是符号，而是情感的载体。

这不仅仅适用于演讲，也适用于我们生活中的方方面面。无论是在工作中、社交中，还是在追求梦想的路上，我们往往最大的阻碍就是自己的思维，被各种焦虑、担忧、自我怀疑所困，我们无法尽情挥洒自己的才华去展示自己。但如果我们学会不再被这些思想绑架，全身心地投入到当下的每一个瞬间，我们就能够发现，真正阻碍我们发挥创造力和社交潜能的，就是我们自己的心态和自我设
