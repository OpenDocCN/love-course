# 雨宸19套合集：07.《撩汉说明书，让你爱的人爱上你》：005.等价原则：让你牢牢抓住他的心

## 概述
在本节课中，我们将学习如何利用等价原则来建立和维持一段长期的亲密关系。

## 等价原则
等价原则指出，在一段恋情中，两个人品质越相当，步入婚姻的可能性就越大。

### 哪些品质会吸引优质的伴侣？
1. **外貌**：包括身材、长相、颜值等。
2. **人品**：无需多言。
3. **社交价值**：包括社交圈、人脉等。
4. **个人魅力**：情绪价值、情感价值、表达能力、情绪输出能力等。
5. **经济条件**：物质水平、家庭背景、收入、社会地位等。

### 如何利用等价原则抓住对方的心？
1. **兼具两性的特质**：
   - 女人会被心思细腻的男人打动。
   - 男人会被智慧理性的女人所吸引。
   - 追求男人时，可以谈论体育、政治或对方专业领域的话题，展现你的才智。

2. **提升自己的吸引力**：
   - 通过化妆、穿搭提升外貌吸引力。
   - 通过学习、提升自己的学识、性格、人品来吸引对方。
   - 保持积极向上的状态，对未来有规划。

3. **善于利用回声教育**：
   - 当你喜欢上一个比你优秀的人时，他就是你努力的最好目标和导师。
   - 在相处中，多观察他的习惯，倾听他的语言，模仿和重复对方的话，展现自己的高价值。

4. **利用知行一致性，暗示对方已经爱上你**：
   - 多制造偶遇的机会，和对方有实质性的接触。
   - 通过语言暗示对方你们是有缘分的，是同一类人会被吸引。
   - 让对方帮你一些小忙，送礼物，让对方觉得为你付出是合情合理的事。

5. **通过互动拉近距离**：
   - 通过找对方求助来让对方增加对你的投资。
   - 通过交换礼物来增加互动，增加对方在你身上的投资。

## 总结
本节课中我们一起学习了如何利用等价原则来建立和维持一段长期的亲密关系。希望这些方法能够帮助你牢牢抓住你爱的人的心。
