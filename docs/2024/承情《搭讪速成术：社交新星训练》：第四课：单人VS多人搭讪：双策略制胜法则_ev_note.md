# 承情《搭讪速成术：社交新星训练》第四课：单人VS多人搭讪：双策略制胜法则

## 概述
在本节课中，我们将探讨单人搭讪与多人搭讪的区别，并重点介绍如何控制多人搭讪的场面，以及如何利用僚机进行互动和交流。

## 单人与多人搭讪的区别
以下是单人搭讪与多人搭讪的主要区别：

- **交流深度与宽度**：单人搭讪的交流深度更深，而多人搭讪则更注重交流的宽度。
- **控场能力**：多人搭讪需要更高的控场能力，需要照顾到组合中其他成员的情绪。

## 多人搭讪的控场策略
以下是几种有效的多人搭讪控场策略：

1. **识别大姐大**：观察组合中谁处于中心位置，眼神交流频次和频率等，确定大姐大。
2. **与大姐大建立联系**：第一时间与大姐大进行眼神交流，并赢得她的好感。
3. **转向目标人物**：与目标人物进行强关联强对话，可以是夸赞或合理的借口。
4. **照顾组合情绪**：注意照顾到组合中所有人的情绪，进行微笑点头等互动。
5. **共同目标**：寻找共同目标，进行互动和交流。
6. **互相帮助**：提出请求或帮助对方解决问题。
7. **分工合作**：与僚机商定助攻和辅助策略，进行团队对团队的互动。

## 推荐策略
最推荐的控场打法是第一种，即与大姐大建立联系并转向目标人物。在了解情况下，第四种方法（分工合作）也是有效的。
