# 清风老师合集：05撒娇1.0底层逻辑篇11期：051.第四课：如何进入男人的内心，让他越来越

在本节课中，我们将要学习如何深入对方的内心，建立积极的沟通循环，从而让他越来越爱你。我们将分析关系疏远的常见原因，并掌握一套有效的四步沟通公式。

上一节我们了解了为什么爱有时会伤人，本节中我们来看看如何破解这个迷局，将互相攻击的负面循环转变为互相支持的正面循环。

以下是四个会让男人的心离你越来越远的核心原因。

**第一个原因：不断指责。**
很多时候争吵源于无关紧要的小事。例如，有学员分享，她与前男友曾因“蛋炒饭先放蛋还是先放饭”这样的小事争吵到砸东西。事情本身不值得，但在争吵过程中，不断的指责和攻击会使矛盾升级。

**第二个原因：不再倾听。**
当对方想与你聊聊工作压力或遇到的问题时，若未得到回应，他便会把话咽回去。社会往往要求男性独自承担压力，因此倾听对他们尤为重要。

**第三个原因：不再理解。**
当关系陷入拒绝沟通和长期冷战的循环时，小事争吵背后其实是无数次情绪的积累。

**第四个原因：道德绑架。**
常说“我为这个家付出这么多，你怎么能这样对我？”这类话。说者或许不觉伤人，甚至认为在陈述事实，但听者会感到委屈和难过。他用错误的方式（指责、施压）希望你改变，结果只会让你想逃避，关系从而陷入“越指责，越逃避”的恶性循环。

那么，如何正确表达想法，促使对方改变呢？接下来，我们学习一个核心的四步沟通公式。

**第一步：学会区分感受与评价/指责。**
我们通过三道题来练习区分。
1.  **“你怎么总是不理我。”** 这是指责，而非感受。对方听来会反驳：“我为你付出这么多，哪里不理你？”
2.  **“我为你付出这么多，难道你都感觉不到吗？”** 这是指责与质问。
3.  **“为什么你总是沉默，不能好好跟我讲呢？”** 这同样是指责。对方可能心想：“讲了有用吗？你又不理解我。”从而更沉默。

当你学会区分后，我们进入第二步。

**第二步：将指责转化为感受表达。**
以下是修改例句：
*   原句“你怎么总是不理我。”改为：**“你对我非常重要，我希望得到你的理解和关注。”**
*   原句“我为你付出这么多，难道你看不到吗？”改为：**“宝贝，我不希望你有压力，我希望你能看到我的付出。但这些都是我心甘情愿的，因为我很爱你。”**
*   原句“为什么你总是沉默，不能告诉我吗？”改为：**“宝贝，我想分担你的压力。如果你愿意，我愿意倾听你遇到的问题，和你一起找到解决方案。”**

接着，我们进入第三步。

**第三步：倾听对方的心里话与感受。**
倾听时需避免两大误区。
1.  **急于下定义给方法。** 例如：“你这点事算什么，只要……就可以了。”或“这有多难，我来弄。”这种回应会让对方觉得付出不被重视。
2.  **否定对方的感受。** 例如：“你这有什么呀，我遇到过更糟的……”这会让倾诉者感到不被理解，更加痛苦。

那么，什么是有效的倾听？以下是三个关键句式：
1.  我明白。
2.  我理解。
3.  我能感受到。

例如，当对方指责：“我每天辛苦打拼都是为了这个家！”你可以回应：**“宝贝，我知道你在外面打拼很辛苦，你为我们家做了很多努力，我能看到你的辛苦。可能我有时没表达清楚，没关系，我们可以慢慢沟通。”**

当对方表达压力时，用“我理解”、“我明白”、“我能感受到”来回应。这就引出了第四步。

**第四步：与对方的感受做回应。**
通过前三步——区分感受与指责、表达自身感受、倾听对方感受——最终实现对感受的积极回应与互动。

一段感情中，当双方都能舒适、坦诚地表达感受时，就仿佛回到了热恋期（例如：“宝贝，我想你了。”“亲爱的，我也想你。”）。此时感情自然会变得甜蜜。

记住，要让感受自然流露。在表达过程中，若遇到对方的指责，不要用指责回击，而要用**理解**去回应。理解是与对方深入沟通的高明方法。

本节课中我们一起学习了进入男人内心、让他更爱你的四步沟通法：区分感受与指责、表达自身感受、倾听对方感受、回应对方感受。当你们能坦诚交流感受时，关系将进入积极的循环。
