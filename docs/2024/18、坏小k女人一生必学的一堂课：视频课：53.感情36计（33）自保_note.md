# 18、坏小k女人一生必学的一堂课：视频课：53.感情36计（33）自保

🎼 **感情36计第33个技巧：自保**

在我们大家的印象中，自保就是自己保护好自己的意思，我们绝对不允许别人来伤害我们。但在感情中，自保的含义则更为丰富，它更多体现在给自己留下一条后路。

### 什么是感情中的自保？

试想一下，当我们去测试一个异性时，如果对方拒绝或忽视我们，我们是否应该有一条备选之路，以曲线救国的方式解决问题？因此，生活中的自保与感情中的自保存在区别。

### 自保的技巧

1. **同理心**：同理心是自保的一个常用技巧，它可以帮助我们避免完全陷入困境。
2. **预测回复**：在与异性聊天时，学会预测对方可能的回复。
3. **避免尴尬**：如果你预测对方会拒绝你，提前说出对方可能拒绝你的话，避免尴尬。
4. **测试感觉**：通过这种方式，你可以测试对方对你的感觉。
5. **建立默契**：让对方感觉到你了解他，增加彼此的默契。

### 例子

例如，很多同胞喜欢问异性：“你今天想不想我呀？”通常异性回答的都是“不想”，即使他们真的想，在没有确认关系之前也不会说出来。这时，你可以补一句：“你这么没良心，一定没想。”😊。

通过以上方法，你可以在感情中更好地自保，避免不必要的尴尬和伤害。

## 总结

本节课中，我们一起学习了感情36计中的第33个技巧——自保。希望这些技巧能帮助你更好地处理感情中的问题。
