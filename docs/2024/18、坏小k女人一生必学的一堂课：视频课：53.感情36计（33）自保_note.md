# 感情36计 第18课：自保技巧 🛡️

在本节课中，我们将要学习感情36计中的第33个技巧——自保。这个技巧的核心并非简单的自我保护，而是在感情互动中为自己巧妙地预留回旋余地，避免陷入尴尬或被动的局面。

## 自保在感情中的独特含义

上一节我们介绍了课程主题，本节中我们来看看“自保”在感情中的具体定义。在普遍认知中，自保意指保护自己不受伤害。然而，在感情语境里，自保更多地体现为**为自己留下一条后路**。

试想，当你试探一位异性时，如果对方拒绝或不予回应，你是否应该有条备选路径，从而迂回达成目标？因此，生活中的自保与感情中的自保存在区别。但两者的共通之处在于，它们都是一种**防止自己完全走入死胡同的常用补救技巧**。

## 自保技巧的核心操作方法

理解了概念后，我们进入实践环节。关键在于每次与异性交流时，都需进行预判与主动引导。

以下是实施自保技巧的三个步骤：

1.  **消息预判**：在每次发出消息后，预测对方可能如何回复。
2.  **主动陈述**：如果你预测对方会拒绝你，便抢先一步，将他可能拒绝你的话自己说出来。
3.  **达成效果**：这样做能同时实现三个目的。

## 自保技巧带来的三大好处

主动“自我拒绝”的策略能产生多重积极效果。以下是三个主要好处：

*   **避免尴尬**：抢先说出对方的拒绝，化解了直接被拒的难堪场面。
*   **测试感觉**：通过观察对方在你“自我拒绝”后的反应，可以试探他对你的真实态度。
*   **建立默契**：让对方觉得你非常懂他，因为你能预知他的想法，从而营造出默契感。

## 实战案例解析

理论需要结合实例来理解。这里有一个常见的对话场景。

许多人在聊天中喜欢问异性：“你今天想我了吗？”通常，即使对方真的想了，在关系未明确前也常会回答“不想”。

此时，自保技巧的应用方式是：在对方可能说出“不想”之前，你立刻补上一句：“你那么没良心，肯定没想。”

**代码逻辑模拟**：
```python
if 预测_对方回答 == “不想”:
    我方发言 = “你那么没良心，肯定没想。”
else:
    按原计划继续对话
```
通过这个方式，你提前说出了对方的潜在回应，将对话主动权掌握在自己手中。
