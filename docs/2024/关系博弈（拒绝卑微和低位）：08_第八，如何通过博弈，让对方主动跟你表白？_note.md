# 关系博弈（拒绝卑微和低位）：08_第八，如何通过博弈，让对方主动跟你表白？

## 概述
在本节课中，我们将学习如何通过博弈策略引导对方主动向你表白。

## 表白的重要性
无论关系再好，都应有表白的环节。这有助于避免双方对关系确认标准不同带来的问题。**公式**：关系确认 = 表白动作

## 暧昧期表白难题
在暧昧期，许多人不敢表白，担心关系变差或不知道如何表白。**公式**：表白风险 = 失去对方的风险

## 引导表白策略
以下为四个步骤的引导表白策略：

### 步骤一：释放单身信号
在相处过程中，明确表达自己是单身且想脱单的状态。**代码**：`表达单身状态 = 释放信号`

### 步骤二：让对方感受到被吃定
在聊天过程中，适当表达好感，让对方感受到自己被吸引。**代码**：`表达好感 = 增强吸引力`

### 步骤三：让对方重新体验不确定性
通过引入竞争者或改变聊天模式，让对方感受到自己不再被完全吃定。**代码**：`引入竞争者 = 增加危机感`

### 步骤四：暗示有效期
在适当的时候，暗示对方时间有限，制造压迫感。**代码**：`暗示有效期 = 制造压迫感`

## 总结
通过以上四个步骤，引导对方主动向你表白。本节课中我们一起学习了如何通过博弈策略，让对方主动向你表白。
