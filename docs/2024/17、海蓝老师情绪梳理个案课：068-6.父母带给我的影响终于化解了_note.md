# 海蓝老师情绪梳理个案课：068-6.父母带给我的影响终于化解了

在本节课中，我们将学习如何通过情绪梳理个案课来化解父母带来的影响。

---

## 情绪梳理过程

以下是情绪梳理个案课的具体步骤：

### 1. 确定情绪来源

那么多的恐慌、无助、不安全、恶心和厌恶无力。想要离开这个场景吗？想要把这种感受全部卸载掉吗？

**公式**： 情绪来源 = 恐慌 + 无助 + 不安全 + 恶心 + 厌恶无力

### 2. 选择卸载方式

想用什么样的方式把这些情绪卸载掉？可以用风、水、火、光、土，或者任何你能想象的方式。

**代码**： 卸载方式 = 风或水或火或光或土

### 3. 实施卸载

先进行到哪一步了？现在还打包好了。慢慢来。

**步骤**：
1. 打包情绪。
2. 烧掉情绪。
3. 吹走情绪。

### 4. 感受变化

在卸载了所有这些负担以后，现在从头到脚的感受是：喉咙没有那种不舒服的感觉，比较弱了。

**公式**： 感受变化 = 喉咙不适 - 不舒服

### 5. 吸收宁静

把宁静吸进来，放到心脏里面。感受一下宁静在心脏里面的感受。

**代码**： 宁静位置 = 心脏

### 6. 总结

本节课中我们一起学习了如何通过情绪梳理个案课来化解父母带来的影响。希望这个方法能帮助你更好地处理情绪问题。
