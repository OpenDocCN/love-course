# 高情商聊天话术 社恐变社牛：08.第四节，如何通过聊天的潜沟通建立高位

## 概述

在本节课中，我们将学习如何通过聊天的潜沟通建立高位，避免在聊天中处于低位，从而提升自己在女性心中的形象。

## 核心概念

**公式**： 高情商聊天 = 娱乐自己 + 尊重对方

## 教程内容

### 1. 明确聊天目的

*   **问题**： 在聊天中，你是在娱乐对方还是在娱乐自己？
*   **解决方案**： 确保聊天以娱乐自己为目的，即使偶尔损对方，也要让自己感觉好。

### 2. 调侃与尊重

*   **问题**： 如何在调侃对方时保持尊重？
*   **解决方案**： 将对方视为小朋友或小妹妹，用想象力去调侃，而不是降低自己的身份。

### 3. 语气助词的使用

*   **问题**： 语气助词过多会带来什么影响？
*   **解决方案**： 尽量减少语气助词的使用，避免让对方感觉到你过于主动寻求和谐关系。

### 4. 表情包的使用

*   **问题**： 如何正确使用表情包？
*   **解决方案**： 除非是调侃或暧昧的表情包，否则不要过多使用，尤其是表示情绪的表情包。

### 5. 赞扬的语气

*   **问题**： 如何赞扬对方？
*   **解决方案**： 使用认同和筛选的语气，避免居高临下的赞扬。

## 总结

本节课中，我们学习了如何通过聊天的潜沟通建立高位，避免在聊天中处于低位。通过明确聊天目的、调侃与尊重、语气助词的使用、表情包的使用以及赞扬的语气等方面，我们可以提升自己在女性心中的形象，成为她们心中的特殊人物。
