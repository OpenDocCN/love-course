# 老白《约会达人》课程27：电影院篇 🎬

在本节课中，我们将要学习如何规划一次成功的电影院约会。我们将从约会前的准备、观影过程中的互动技巧，到如何自然地推进关系，进行详细的讲解。电影院是一个经典的约会场所，通过恰当的安排和互动，可以有效拉近双方的距离。

## 约会前的预热与准备

上一节我们介绍了约会场所的选择，本节中我们来看看电影院约会的具体规划。在进入电影院之前，可以先进行一些预热活动。

*   可以带女生先去逛逛商场、抓娃娃或玩电动游戏。这些活动有助于把彼此之间的情绪先预热起来。
*   有了前面的关系铺垫作为基础，即使在看电影时双方不说话，也不会显得尴尬。

## 电影票与影厅的选择技巧

买电影票也是有技巧的。以下是几个关键点：

*   我一般会选择最大的放映厅看电影。因为大的放映厅位置较多，更容易选到像13、14这样连号且居中的座位。
*   当然，现在一些小影厅的视听效果也很好。但如果没有这类特殊影厅，尽量选择大的放映厅。
*   取完票之后，我一般会说：“今天选的这个电影座位号还挺好的，这可能是我们第一次看电影，很珍贵。”说完，可以把女生的手牵起来，把票放到她手里，并说：“记得保管好。”

## 影片类型与小道具准备

关于影片选择，爱情片是个不错的选择。如果没有合适的爱情片，那就从当季的热门电影里选一部。

*   例如，年底上映的《阿凡达2》就很好，我强烈推荐大家关注。
*   总之，选择喜剧片或惊悚片也可以，但不要选择那些单纯面向男性的电影。

看电影前，我们需要准备一些小道具。以下是需要准备的物品清单：

*   买一桶爆米花。
*   询问女生喜欢喝什么，购买两种不同口味的饮料，可以是奶茶或水。

## 进入影院与初期互动

等进入电影院的时候，因为环境比较黑且有台阶，这时一定要照顾女生。

*   可以打开手机灯照一下路，让女生感觉你比较体贴。
*   坐到座位后，不要一直牵着女生的手，需要释放一下。因为影院是一个相对陌生的环境，需要给女生一点适应的空间。这样做既不容易暴露需求感，也便于后面更好地推进关系。

电影开场10分钟之内，不要再尝试牵手。请认真观影，也不要说太多话。记住，在电影院里尽量少交谈。

## 利用爆米花推进互动

当大家都沉浸在电影情节中时，你可以开始互动。以下是利用爆米花进行互动的步骤：

1.  突然拿一颗爆米花放在女生的嘴前。
2.  如果女生用手来接，你要把她的手轻轻打开，然后继续把爆米花放在她嘴前。
3.  如果这时女生张嘴要吃，你可以把爆米花往前拿一下，逗她一下。
4.  接着，你转过头看着她说：“好了。”然后再把这颗爆米花塞到她嘴里。

有来有往，才叫礼尚往来。这时，你可以抓起女生的手，在爆米花桶里抓一颗，再抓紧她的手往自己嘴边送。

*   然后你认真地看着屏幕不说话。
*   或者说一句：“为什么感觉这颗比较甜？”

## 关系升温与试探

这时，你可以目不转睛地盯着屏幕，把头靠过去对她说：“上次是半自动，这次能换成全自动吗？”

*   **半自动**：指你拿着她的手喂你。

如果女生不理你，多半就会主动再喂你一颗。当她的手送过来时，你可以轻轻咬一下她的手指（注意别咬疼了）。然后边吃边调侃说：“嗯，这个味道比较特别，你的手上有一股奶味（或猪肉味）。”女生可能会反驳你，这就是一种轻松的调侃。这种小互动，是只有男女朋友之间才会有的。

等电影放到一些泪点或浪漫场景时，你可以默默地把女生的两只手捧起来，温柔地看着她，再慢慢把眼睛移到大屏幕上。记住这个动作一定要慢。

等电影散场，出门以后，自然地拉着女生走出大厅，就算彻底升级成功了。从此，你跟女生可能就进入了近似男女朋友的感情阶段。
