# 雨宸19套合集：10.《亲密关系必修课：遇见更好的自己》：003.识人心理学：如何快速判断TA是不是“对的人”？ 👫

在本节课中，我们将要学习如何快速判断一个人是否是你的“对的人”。我们将从排除错误选项、评估个人素质以及衡量双方匹配度三个核心方面入手，帮助你建立一套清晰的判断标准。

上一节我们探讨了如何快速走进一个人的内心，本节中我们来看看如何判断对方是否适合与你建立长期关系。恋爱关系的建立，有两个关键前提。

首先，双方的三观需要基本契合。在三观相合的基础上，相处的技巧和方法才能有效发挥作用。这就像开车一样，如果你自己没有驾照，给你任何一辆车你也无法驾驶。

一个没有驾照的人去开一辆性能优异的车，这种状态是危险的。因此，本节课的目标就是帮助你成为一个“有驾照的人”，让你有能力判断一辆“车”是否适合你驾驶。

恋爱的关键点之一，是明确自己的需求。就像买鞋一样，我需要知道自己的脚是多大码数。例如，我是44码的脚，我就需要在44码的鞋子里，挑选一双我最喜欢且买得起的。

很多人在恋爱中的状态是：第一，不知道自己“脚”的尺码（即不清楚自己适合什么样的人）；第二，不知道自己能“买”得起什么价位的“鞋”（即不清楚自己的情感承载能力和现实条件）。于是他们只能不断尝试。

尝试久了，发现很多“鞋”都不合适，就容易产生“我不配穿鞋”的错觉，从而一直“光着脚”。很多人单身的问题，并非没有适合他们的伴侣，而是他们根本不清楚自己的“尺码”和“预算”。

所以，这节课旨在帮助你明确自己的“尺码”和“预算”。

我们常说“不要去找对的人，当你自己对了，一切就都对了”。但同时也要承认，世界上确实存在相对“对”的人，这个“对”是相对于“错”的人而言的，它是一个相对概念。

我们将从以下三个方面，帮你定位那个“对的人”。

## 第一步：排除“渣男/渣女” 🚫

首先，我们需要排除那些不真诚的伴侣。他们或许很有吸引力，但必须远离。这里引入一个核心筛选模型：**投入模型**。

以下是投入模型的四个评估维度：

1.  **彼此对感情的投入**：包括精神上和物质上的投入是否对等、积极。
2.  **身边其他可选择的对象**：对方是否除了你之外，还与其他人保持着暧昧或超越普通朋友的关系。
3.  **对关系的满意度**：对方是否对你们目前的关系状态感到满意。
4.  **对关系的承诺度**：对方是否对你们共同的未来有明确的规划和承诺。

一段健康、真诚的感情，通常是以这四个维度为基础的。如果其中任何一点出现严重问题，那么这个人很可能别有用心。因此，筛选“渣男/渣女”，就从这四点入手。

## 第二步：评估对方的个人素质 🧐

排除了不合适的对象后，接下来需要评估对方个人的内在素质。这主要看以下三点。

### 1. 如何看人品

评估人品，首先要摘下“爱情滤镜”，保持客观。可以请朋友帮忙分析，或者使用一个有效的方法：**客观记录法**。

具体操作是：拿一张纸或打开一个文档，客观地记录下你们之间发生的一些关键事件。只描述事实，不加入主观感受。过一段时间后再以“上帝视角”回看这些记录，就像读一本关于别人的书，这样能帮助你更理性地判断对方的人品。

此外，李玫瑾教授曾提出一个深刻的观点：选择伴侣时，可以想一想，**你希望你的孩子继承对方身上的哪些品质**。用这个思路去思考，能帮你更清晰地看到对方的核心特质。

### 2. 看情绪稳定程度

情绪稳定是心理健康和成熟度的重要指标。一个情绪起伏极大、容易因小事暴怒或迁怒他人的人，其内心往往存在许多未解决的冲突和固着的执念。

**公式：情绪极端不稳定 ≈ 内心冲突固着 + 人格不完整**

与这样的人相处会非常疲惫，你可能会被迫卷入他的内心冲突，甚至开始怀疑自己。情绪稳定的人则具备更强的自控力和自我调节能力，在沟通时能以解决问题为导向，而非发泄情绪。

### 3. 看低谷期的表现

一个人在逆境和挫折中的反应，最能体现他的内在力量、责任感以及与世界的相处模式。

*   **健康的表现**：允许挫折发生，将其视为正常的一部分，并积极寻求解决方案。`态度 = 接纳 + 解决`
*   **不健康的表现**：不接纳挫折，要么逃避问题，要么将责任全部推给外界（包括你），以维持自我完美的假象。`态度 = 指责/逃避 + 推责`

在感情中，冲突和困难就是“挫折”。如果对方在低谷期是逃避和指责型的，那么在你们的关系出现问题时，他很可能采取同样的方式，导致你需要独自承担两个人的责任。

## 第三步：衡量双方的匹配度 ⚖️

最后，我们需要评估两个人是否“匹配”。就像游戏对战，等级相近的人在一起体验才更好。匹配度可以从以下五个方面考量：

以下是评估双方匹配度的五个关键维度：

1.  **原生家庭的相处模式**：观察你与对方异性父母的相处是否融洽。因为每个人都会不自觉地带入原生家庭的相处模式。如果你和他父亲（或她母亲）完全合不来，这可能是一个预警信号。
2.  **恋爱模式期待**：了解彼此对恋爱状态的期待。是喜欢轰轰烈烈，还是追求细水长流？这就像通过“期中考试”（过往恋爱经历）来预估“期末考试”（你们未来的相处）的成绩。
3.  **对未来生活的想象**：包括理想的生活地点、生活方式、是否与父母同住等重大规划。如果核心规划存在根本分歧且无人愿意妥协，长期关系将充满隐患。`妥协 = 我愿意` 而非 `我被迫`，后者易滋生怨怼。
4.  **受教育程度与包容度**：受教育程度在一定程度上反映了一个人的眼界、思维方式和三观。通常（非绝对），见识更广、内心更自信的人，包容度也更高，更有利于化解矛盾。`P(包容度) ≈ f(见识广度， 自信程度)`。
5.  **核心价值观念**：对金钱、家庭、事业、忠诚等根本性问题的看法是否一致。

---

本节课中我们一起学习了快速判断“对的人”的三步法：首先，用**投入模型**（感情投入、可选对象、关系满意度、关系承诺度）排除不真诚的伴侣；其次，从**人品**（客观记录法、品质继承法）、**情绪稳定性**和**低谷期表现**三个方面评估对方个人素质；最后，从**原生家庭模式、恋爱期待、未来规划、教育包容度及价值观**五个维度衡量双方的匹配度。掌握这套方法，能帮助你在感情中更加清醒和自信，找到真正适合你的伴侣。
