# 恋爱脑反转课：32节让男人离不开你 - 007.第六节 爱情价值判断：什么是不对的人？

在爱情中，我们都在寻找合适的伴侣，渴望与那个人共度余生。那么，什么是“不对的人”呢？以下是一些判断标准，供您参考。

## 一、情绪极度不稳定

这类人容易暴怒，生活在一起就像情绪的定时炸弹。例如，一个案例中的女性，她的丈夫因为一杯热水烫了她的嘴就暴怒，生活中随时可能被他的暴怒所影响。

**公式**：情绪不稳定 + 暴怒 = 恶劣的生活环境

## 二、不断打压和PUA

这类人无论你做什么，都无法获得支持。在他们眼中，你做什么都不对。例如，一个案例中的男性，他在婚姻中20多年从未得到过丈夫的一句肯定，无论做什么都被骂。

**代码**：打压 + PUA = 消极的婚姻关系

## 三、有不良嗜好

例如，爱赌博。赌博是一个万劫不复的事情，可能导致巨额债务。例如，一个案例中的男性，他迷恋网络赌博，欠下300万外债。

**公式**：不良嗜好 + 巨额债务 = 灾难性的后果

## 四、极度匮乏

这类人因为匮乏，所以会不断对你好，但有一天你可能会觉得不想要这些，却难以摆脱。

**代码**：极度匮乏 + 过度依赖 = 捆绑关系

## 五、极度控制

这类人会极度控制你，甚至限制你的人身自由。例如，一个案例中的男性，他在大学时代就认识了他的初恋，当着父母的面下跪求婚，但婚后却对妻子进行长时间的控制和限制。

**公式**：极度控制 + 限制自由 = 悲剧的婚姻生活

## 六、判断关系是否继续

判断一段关系是否继续，很简单。如果一个恋爱关系或婚姻关系滋养大于消耗，那么你可以继续。如果消耗大于滋养，且双方之间的清扯又没那么多的话，那么可以结束。

**公式**：滋养 > 消耗 → 继续关系；消耗 > 滋养 → 结束关系

## 总结

本节课中，我们学习了如何判断“不对的人”，以及如何判断一段关系是否继续。希望这些内容能帮助你在爱情中找到合适的伴侣，过上幸福的生活。
