# 雨宸19套合集：04.《冷暴力，别让爱情在沉默中死去》：006.学会反击：对抗冷暴力的自救方法

## 概述
在本节课中，我们将学习如何对抗冷暴力，通过学会反击来自救。

## 学会反击对抗冷暴力的自救方法

### 欣赏自己，培养自爱力
**公式**：积极自我对话 = 自我关怀 + 自我肯定

1. **积极自我对话**：用积极的方式与自己对话，改变自我批评的对话方式。
2. **自我关怀**：多夸赞自己，多自我总结，如“我怎么这么好，我怎么这么棒”。

### 放弃自我界定，看待真实的自己
**公式**：自我界定 = 自我认知 + 自我标签

1. **自我界定**：避免给自己贴标签，如“我是一个不优秀的人”。
2. **接纳自己**：认识到每个人都有归因属性，不要轻易给自己贴标签。

### 转移对伴侣的依赖性
**公式**：独立生活 = 兴趣爱好 + 社交圈子

1. **培养兴趣爱好**：不要将所有经历都放在关系中，有自己的工作、生活和朋友圈。
2. **独立生活**：感情只是生活的一部分，不是全部。

## 积极采取行动
### 重组思维
1. **认识到冷暴力不是你的错**：明确意识到冷暴力是对方的问题，而非自己的问题。
2. **接纳不完美**：接受对方的不完美，包括自己的不完美。

### 冷静分析处境
1. **冷静分析**：不要轻易相信对方的话，避免陷入自我怀疑。
2. **脱离思考模式**：避免陷入“我不行，他说的都是对的”的思考模式。

### 以暴制暴
1. **坚决反抗**：不要因为对方的态度影响自己的生活。
2. **维护合法权益**：坚决维护自己的合法权益，不要成为对方的木偶。

## 总结
本节课中，我们学习了如何对抗冷暴力，通过欣赏自己、放弃自我界定、转移对伴侣的依赖性以及积极采取行动来自救。希望这些方法能帮助大家走出冷暴力的阴影，勇敢地去爱。
