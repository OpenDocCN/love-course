# 小怡《情绪吸引力-人性篇》：014.14-聊天五步法-05达成一致

## 概述
在本节课中，我们将学习社交五步法的最后一步——达成一致。这一步是构建关系的自然延伸，通过前四步的铺垫，我们可以轻松地提出要求并达成目标。

## 达成一致
### 核心概念
达成一致是社交五步法的最终目标，它基于前四步的铺垫，包括破冰、了解、筛选和欣赏。

**公式**：
达成一致 = 破冰 + 了解 + 筛选 + 欣赏

### 步骤解析
1. **提出要求**：在前四步的基础上，提出一个具体的要求，这可以是建立关系、达成合作或其他任何目标。
2. **目标类型**：目标可以是最终目标，如建立男女朋友关系，也可以是阶段性目标，如达成一次邀约。
3. **能量强度**：根据前四步的能量强度，提出更接近最终目标的要求。

### 应用实例
- **情感关系**：与女生建立男女朋友关系，或达成一次约会。
- **面试**：成功谈成一个工作机会，或获得复试机会。
- **谈客户**：直接成交，或获取进一步沟通的机会。

### 注意事项
- 目标可以是深浅不一，具体取决于前四步的能量强度。
- 在吸引环节，对方兴趣越强，筛选强度可以越强。

## 社交舞步法在其他场景的应用

### 面试场景
1. **破冰**：与HR打招呼，进行简单的寒暄。
2. **了解**：自我介绍，展示自己的优势。
3. **筛选**：确认HR对你的兴趣是否真实。
4. **欣赏**：表达对公司和行业的看好，以及对岗位的期待。
5. **达成一致**：谈薪水，确定入职细节。

### 谈合作场景
1. **破冰**：夸赞对方，拉近关系。
2. **了解**：展示自己的价值，了解对方的需求。
3. **筛选**：确认对方对方案的兴趣。
4. **欣赏**：表达对对方的欣赏，以及对合作的期待。
5. **达成一致**：约定进一步沟通或合作。

## 总结
本节课中，我们学习了社交五步法的最后一步——达成一致。通过前四步的铺垫，我们可以轻松地提出要求并达成目标。社交舞步法在情感关系、面试和谈合作等场景中都有广泛应用。掌握社交舞步法，可以帮助我们更好地与他人沟通，达成目标。
