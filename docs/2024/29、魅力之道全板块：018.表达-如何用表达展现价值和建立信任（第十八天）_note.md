# 魅力之道全板块：018.表达-如何用表达展现价值和建立信任（第十八天）

大家好，今天我们要学习如何让表达为你的目标服务。首先，我们要明确表达的目的。最基本的是，表达是交流中我们自己的部分。我们说话、表达是为了与人交流。

但表达还可以承载更多作用。例如，倾诉以改善心情，或通过输出影响他人，通过观点教育他人。但本课程旨在提升魅力。

因此，表达在魅力中的作用是展现你的价值并建立更多信任感。尤其是讲故事时，因为分享独特经历，带有个人性，所以具有价值。

这有助于让别人深入了解你。这就是它能展现价值并建立信任的原因。接下来，我将介绍如何在表达中讲故事，以更好地展现价值和建立信任。之前已介绍过故事结构：开头、中间、结尾，以及至少一个高潮。

高潮通常在结尾或接近结尾处出现，必须具有价值，不能是低俗内容。它可以是一个有趣故事的搞笑点，也可以是独特经历的独特之处。例如，本板块第一节课分享的故事，高潮是我得到了心动女生的微信。

每个人生命中都有独特经历。如果你觉得自己生活匮乏，没有这样的经历，那么一定要多经历，无论是旅游、留学、结识新朋友、新异性等。你自己要主动去发生故事。

有魅力的人往往经历丰富，这两者互相促进。经历丰富让你更有魅力，而有魅力也会让你有更多经历。如果你现在的经历还不够丰富，也不要气馁。从现在开始，去经历、去体验，你会得到很多精彩故事。

尤其是当你学习了提升魅力的课程后，你会更加能够与周围的人建立关系。因为所有故事基本上都是人与人之间的故事，魅力能帮助你发生更多有趣故事。

我刚刚稍微扯开了一点，因为我觉得增加经历对大家来说很重要。现在我们回到本节课内容。当你有一个完整故事，且故事主角是你时，你可以将一些特质隐藏在故事中。

但放在的地方一定要是最不起眼的地方，你不能刻意展示某些特质。因为那样会像炫耀，而炫耀是一种低价值行为。例如，如果我说我分享的那个故事时，我会说：“我是一个很有勇气的人，然后我再去说那个故事。”

这样目的就很明显，就是一定要展示我是一个有勇气的人。但往往你越需要展示什么，你就越缺什么。而且会很刻意，给人一种炫耀的感觉，这并不好。但事实上，我分享那个故事时，明面上的目的是让大家更有勇气。

但在听故事的过程中，你也得知了我是一个有勇气的人，同时也是一个被女生喜欢的人。因为结局我说了，我和那个女生进入了一段关系，并且我们是从搭讪场景认识的。

你还知道了我是一个淡定临危不惧的人。因为当时无论是周围的人还是那个女生的朋友，我都承受了很多压力和阻力。你能够通过那个故事得到我很多信息。

这些信息隐藏在那个故事中，这些特质隐藏在其中。但同时又很自然。一方面，在听故事的人眼里，我的价值变得更高了。同时，因为听故事的你们知道了我的独特体验，你们也能更了解我这个人是谁，我有什么样的经历，从而你们会对我更加信任。

这就是讲故事的力量。它能同时提高你的价值和提高别人与你之间的信任感。那么今天我会为大家再讲一个故事。这个故事比较有趣，是我一段有趣的经历。

虽然我教大家魅力，但其实在我小时候，特别是上初中、高中时，我还是一个很内向的人。其实我心里一直有一个想法，就是我不想一直这样，我不想做那个被大家口中的害羞男孩。

我给自己当时设立了一个挑战，就是我很怕跟别人眼神对视。我就决定我要跟以后遇到的每一个人都进行眼神对视，至少坚持一会儿。那我就真的去做了。

即使我去零售店、杂货店买东西时，我跟人家买东西，我递给别人钱时，我都是说：“10块钱。”然后别人找给我零时，我也是看着对方的眼睛，说：“好的，谢谢。”然后我就离开了。

在那一段时间里，无论是跟遇到的任何人，只要我们有说话，我都会跟别人进行眼神交流。直到后来发生了一件事情，让我稍微修正了一下我这样一个行为。

😊，我在上大学时，本来是要去一个英联邦国家留学的，所以我去读雅思课程。在第一节课上，我们的口语老师走进来，他是一个非常壮的男人。我发现在上课的过程中，他一直盯着我看。

但那时我也是一个试恋期。我说了，我一直跟别人不断的眼神对视，那他看我，我也就反抗回去，我觉得我不能说很奇怪，他一直看着我，我们看了大概1分钟到2分钟的样子，终于终于他把他的眼睛给瞥过去了，他不再看我了。

可能我也不知道他是怎么想的。当时我心里想说挺好的，挺好的。那我还是赢了。中午我也就去吃饭了，因为下午还有课嘛，但是第一节课大家都加了老师的微信。

我吃面吃着吃的时候，我手机就叮收到一条消息，是来自于我那个雅思口语老师，消息是这么写的：“你长得很像我前男友。”就是他是一个同性恋者嘛。反正经过这次事情之后，我还是稍微修正了一下我这样一个行为。

我只对有吸引到我的人，或者说我很看重一场谈话，我要去说服别人的时候，我再去跟别人做眼神沟通。那么往往我在跟别人讲这个故事的时候，我的主题是我跟别人分享一个有趣的事情，或者是谈到我性格的转变。

我会跟他说，我以前其实挺内向的，谈我性格的转变，这些往往是我明面上的目的。但是呢，你注意一下，你在听这个故事的途中，你感受到了我的什么特质。

那作为一个还是学生的时代的人，我愿意去改变自己，去做这样跟别人对视的努力，说明我首先是一个很有勇气的人，以及我是一个追求自我提升的人，追求卓越。

同时呢，因为这件事情挺好玩的。你说一个小男孩怎么会想到说啊让自己从内向变得不那么内向，是通过跟所有的人去做眼神对视，所以又显得我是一个比较有趣的人，这些都是我能够在这个故事当中体现出来的我的隐藏的特质。

但是呢，他们又不是我这个故事的目的，所以说就显得我不是在炫耀，我又达到了我展示自己特质的效果，听故事的人，知道了我这样独特的经历之后，也会更加的信任我，因为他更加了解我。

好，那接下来是今天的作业。

**作业**：

1. 写下一个主角是你的故事，可以是你的独特经历、有趣故事或表面目的的故事。
2. 在故事中融入你身上的一些好的特质。
3. 为故事设计一个好的铺垫，抓住别人的注意力。
4. 编排好故事结构，使故事紧凑完整，而不是散。

希望你在练习讲故事的过程中，不要有完美主义。先开始做这件事。即使刚开始不那么好，甚至有些地方有点尴尬，有些地方有点尴尬，都是OK的。

慢慢慢慢，你会把讲好故事、练好自己的表达融入到你的血液中，成为你的本能。
