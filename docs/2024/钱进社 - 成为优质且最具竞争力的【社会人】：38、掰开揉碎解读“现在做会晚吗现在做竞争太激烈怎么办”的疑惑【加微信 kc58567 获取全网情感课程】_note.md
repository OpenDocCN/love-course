# 钱进社 - 成为优质且最具竞争力的【社会人】：38、掰开揉碎解读“现在做会晚吗现在做竞争太激烈怎么办”的疑惑

## 概述
在本节课中，我们将探讨关于“现在开始还来得及吗？”以及“竞争太激烈怎么办？”这类问题的疑惑，并分析如何成为优质且最具竞争力的社会人。

## 核心概念
**公式**：差距 = 行为和思维差异

## 教程内容

### 1. 疑惑的来源
以下是关于疑惑来源的列表：

- **身边人的赚钱信息**：大部分人的赚钱信息来自于身边人，导致焦虑和迷茫。
- **碎片式的信心**：部分人的信心来自于邻居、亲戚等，缺乏宏观的持续看待赚钱的方式。

### 2. 焦虑的原因
以下是关于焦虑原因的列表：

- **与大佬的差距**：与大佬的差距是可逆的，因为大佬也不会与普通人叫板。
- **竞争的误区**：焦虑总是往上看，却忘了身边的人还未竞争过。

### 3. 停滞不前的原因
以下是关于停滞不前原因的列表：

- **已知的问题**：困住你的不是未知的，而是已知的，如懒惰、拖延等。
- **不行动的问题**：问自己“我能做点什么吗？”却不会真正去改变。

### 4. 新人的优势
以下是关于新人优势的列表：

- **无包袱**：新人没有包袱，更容易行动。
- **放弃成本低**：新人放弃成本低，更容易尝试。
- **进取心强**：新人没有收获，更渴望收获。

### 5. 竞争与机会
以下是关于竞争与机会的列表：

- **竞争无处不在**：哪里都有竞争，不只是在网上。
- **机会需要寻找**：机会不会放在每个人面前，需要自己寻找。

### 6. 抓住机会的方法
以下是关于抓住机会方法的列表：

- **持续行动**：在平淡无奇的时候持续行动。
- **坚信机会**：坚信在自己不会的情况下，根本就没有晚了。

## 总结
本节课中我们一起学习了关于“现在做会晚吗？”以及“竞争太激烈怎么办？”这类问题的疑惑，并分析了如何成为优质且最具竞争力的社会人。希望这些内容能帮助你更好地面对生活中的挑战。
