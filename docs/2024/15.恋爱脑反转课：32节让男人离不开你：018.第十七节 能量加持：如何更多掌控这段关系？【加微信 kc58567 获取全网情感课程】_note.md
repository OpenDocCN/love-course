# 恋爱脑反转课 第15课：第17节 能量加持：如何更多掌控这段关系？⚡️

在本节课中，我们将学习如何通过能量加持，在两性关系中建立掌控感。我们将探讨掌控感与自信的关系，学习管理预期，并理解改变的核心在于动能而非结果。

上一节我们介绍了如何提升个人能量与取悦他人的能力。本节中，我们来看看如何运用这些能量，来更多地掌控一段关系。

在两性关系或具体的男女交往过程中，通常是谁能掌握关系的节奏，主动权就在谁手里。虽然前面讲过一些能量提升的办法，但如果最终要让关系令你舒适，它应该处于你的掌控之中。人的焦虑常源于事情脱离掌控。若事情本身在掌控之内，人就不会焦虑。那么，在一段两性关系中，你如何获得这种掌控感呢？

以下是获得关系掌控感的三个核心方法。

**第一，拆解目标，明白掌控感与自信相辅相成。**

我举个简单的例子。很多女人容易因男人的忽冷忽热而患得患失。男人发生关系后变得冷漠，女人便开始自我攻击，怀疑自身魅力。这是典型的缺乏掌控感和自信的表现。你越不自信，就越需要男人的热情来确认关系的价值。但如果你本身自信，无论男人忽冷忽热、热情似火或谈及婚姻，你都能保持自己的节奏。因为你对自己与对方的关系定位非常清晰。**公式：掌控感 = 清晰的自我定位 + 稳定的内在节奏**。当掌控感在你手中时，你就不会对关系有强烈的患得患失感。

**第二，建立并管理预期，明确自己在这段关系中需要什么。**

我接触过许多女性婚外关系案例。很多女人在这样的关系中，往往不清楚自己需要什么。例如，我接过许多无性婚姻的案例。一位33岁的女性因无性婚姻，与能力不错的领导走到一起。但随着关系深入，女人对关系的需求和欲望越来越大。**关键代码：`预期 = 初始需求`**。你在这段关系中，本只需要对方提供两性关系。但后来发展到既想要钱，又想要爱，甚至希望对方离婚娶你，将预期无限提升。男人能给予的仍是最初的两性关系，无法满足你膨胀的预期，这会导致你失望、患得患失并攻击自己和对方。因此，管理好自己的预期，才能明白关系中你需要什么，也知道自己该往哪个方向走。

**第三，明白改变是动能，而非结果。**

人永远处于动态发展的过程中。所有关系都是以“我”为中心展开的。如果你总是无法结束一段关系，说明你内在不够强大，对关系仍抱有期待。如果你强大到可以用自己的力量结束关系，就不会有强烈的内耗。**公式：关系主动权 ∝ 个人能量值**。所有的两性关系、婚姻、恋爱或情绪价值提供，都基于“我”展开。“我”才是最重要的。我的感受、身体、情绪和两性关系才是最核心的。因此，改变是持续的动能，而不是一个静止的结果。我们强调如何给自己增加能量，其实就是告诉你如何在变得强大后，去掌控关系。只有自己先强大起来，别人才会看见你并非好欺负。即使对方伤害了你，你也可以一笑而过，对方反而会更加敬佩你，因为你是一个能量强大的女人。

本节课中，我们一起学习了通过拆解目标建立自信、管理关系预期以及将改变视为持续动能，来增强对两性关系的掌控感。记住，关系的主动权源于强大的自我和清晰的认知。
