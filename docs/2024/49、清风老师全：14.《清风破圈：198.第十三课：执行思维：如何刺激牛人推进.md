# 49、清风老师全：14.《清风破圈：198.第十三课：执行思维：如何刺激牛人推进

## 第十三课：执行思维：如何刺激牛人推进合作

在本节课中，我们将要学习一种高级的社交策略，即如何通过引入竞争来刺激关键人物（“牛人”或“老大哥”）更积极地与你推进合作。我们将探讨人性的竞争本能，并介绍具体、可操作的方法，同时强调核心价值与感恩之心的重要性。

上一节我们介绍了聊天的基本结构与话题梳理。本节中我们来看看，当你已获得对方欣赏，但合作推进缓慢时，应如何应对。

你可能会遇到一种情况：对方对你很欣赏，但不愿意给你资源，也不愿意推进合作。这时，需要学习一种策略：刺激合作，引入竞争。这个过程是一种高段位的社交做法。

我们可以观察一个现象：抖音直播间里有榜一、榜二、榜三的榜单。在直播中，为什么要设置这样的榜单？按理说，各位观众凭心意打赏即可。其核心原因在于：**竞争是人性的本能**。

在我们的底色中，当发现别人比自己强时，往往会激发出两种状态。

以下是两种典型的心理反应：

1.  **奴性**：认为自己不行，选择臣服于对方脚下。
2.  **王者心态**：激发出极大的斗志和毅力，立志要超越对方。

大多数有成就的“老大哥”，其底色并非奴性。他们都想成为一方诸侯或君王。因此，如果你想让“老大哥”尽快与你推进合作，最好的方式是让他感觉到你是一个非常抢手的人物。

这里衍生出一个问题：你觉得“老大哥”是只有一个且对你忠心不二比较好，还是有越来越多的“老大哥”来帮你比较好？

仔细思考这个问题。答案一定是后者。有更多的“老大哥”想办法提携你，你才能快速成长。

打一个比方：就像一辆高速行驶的汽车，光有一个气缸是不够的。顶级的汽车都有六缸、八缸甚至十缸。越多的“老大哥”助力，会让你前进的速度和跃迁的节奏越快。

你可能会问：万一我的“老大哥”因为看到别的“老大哥”也在帮我，反而不愿意帮我了，怎么办？

首先，这需要你自身有足够强的价值。回到我们做事的初心：我们认识“老大哥”是为了更好地获得资源。**你本身的事情和能力才是最关键的**，这才是“老大哥”们愿意推进你的核心点。只有你具备踏实做事的能力，他们的帮助才能被极大放大。

那么，如何避免多位“老大哥”争风吃醋、最终都不愿帮你的情况，防止在圈子里口碑变坏呢？

其实，这个问题从一开始就想错了。“老大哥”们通常是有格局和心胸的。只要你在背后不去讽刺、贬低之前帮助过你的“老大哥”，几乎所有人都不会与你为敌。这里就要认识到**感恩之心**的价值。

层次越高的人，越懂得感恩带来的长远意义。哪怕某位“老大哥”没有实际帮你，当别人问起时，你也应该说：“大哥帮我了，在思想层面给了我很大启发。”“老大哥”们真正反感的是那些拆东墙补西墙、过河拆桥的小人。

当“老大哥”帮了你，你要常怀感恩之心。即使未来他不能再帮你，也要时常维系关系。这就是为什么当我的员工为了孝敬父母而预支工资时，我立刻同意了。在我看来，**回报每一个帮助过你的人，是一个人能快速成功和发展的关键**。如果你希望变得越来越优秀和成功，必须怀有感恩之心。

那么问题又来了：怀着感恩之心，又如何引入竞争者来刺激“老大哥”给我资源呢？

这里有一个小技巧分享给你：当你呈现出自身的竞争优势时，可以**通过第三方来呈现别人对你的关注**。

举个例子：假设有两位“老大哥”。一位是你主要想争取帮助的，但他迟迟没有动作；另一位对你也很关注，平时多有认可。你可以通过朋友圈截图（打上马赛克，但故意让人能辨认出是那位关注你的“老大哥”的头像）的方式，巧妙地展示你正被多人关注。马赛克不要打得太厚，目的是让人知道你同时被他关注。这样小小的刺激，往往能在关键节点上帮你快速推进合作。

回到社交的本质。我们有时确实需要一些小小的技巧来获得更大的回报。但要记住：**有道无术，术尚可求；有术无道，止于术**。千万不要认为刺激对方是最重要的事。重点永远要回到你自身能为“老大哥”提供的价值上。
