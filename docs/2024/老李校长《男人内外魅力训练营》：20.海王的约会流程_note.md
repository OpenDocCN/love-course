## 约会流程

### 当天见面，当天发生关系

1. **安排见面地点**：选择一个餐厅，如家楼下餐厅。
2. **结束用餐后**：邀请对方上楼喝两杯。
3. **营造氛围**：准备酒和香薰，创造轻松愉快的氛围。
4. **拉手与亲吻**：如果气氛合适，可以尝试拉手和亲吻。
5. **处理拒绝**：如果对方拒绝，不要过于紧张，让她离开即可。

### 见过几次面再发生关系

1. **初次见面**：确认长相，确保对方符合标准。
2. **送对方回家**：如果距离较近，可以送对方回家。
3. **第二次见面**：安排在中午，避免晚上过于猴急。
4. **第三次见面**：晚上用餐后，送对方回家。
5. **第四次见面**：邀请对方来你的住处。

## 海王的特点

1. **不舔狗**：海王不会过度讨好对方，保持自己的价值。
2. **自信**：海王对自己有信心，不会过于紧张。
3. **主导氛围**：海王能够主导约会氛围，让对方感到自在。

## 约会中的沟通技巧

1. **轻松自在**：将对方视为普通朋友，不要过于紧张。
2. **眼神交流**：适当的眼神交流，但不要过于盯着对方。
3. **话题选择**：不要刻意找话题，让对方主导对话。
4. **态度**：保持自信，不要过度关注对方的感受。
