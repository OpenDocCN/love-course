# 雨宸19套合集：10.《亲密关系必修课：遇见更好的自己》：009.拒绝的艺术：从不敢说“不”的人，到底在怕什么？

## 概述
在本节课中，我们将探讨为什么有些人不敢说“不”，以及如何学会拒绝。

## 为什么不敢说“不”？

### 1. 害怕留下不好的印象
**公式**：害怕拒绝 → 害怕留下不好的印象

### 2. 害怕受到惩罚
**公式**：缺乏自主意识 → 害怕拒绝 → 害怕受到惩罚

### 3. 害怕伤害对方
**公式**：担心伤害对方 → 不敢拒绝

## 如何学会拒绝？

### 1. 做好心理建设
**代码**：告诉自己拒绝是正常的，你有权拒绝。

### 2. 使用拒绝的公式
**公式**：拒绝的理由 + 明确的拒绝 + 替代方案

### 3. 修正不良预期
**代码**：不要把责任都推到自己身上，坚定自己的选择。

### 4. 寻找正面激励
**代码**：在信任的关系中尝试拒绝，并获取正面反馈。

## 总结
本节课中我们一起学习了如何克服拒绝的恐惧，包括了解恐惧的原因、做好心理建设、使用拒绝的公式以及寻找正面激励。希望这些方法能帮助你更好地与人交往。
