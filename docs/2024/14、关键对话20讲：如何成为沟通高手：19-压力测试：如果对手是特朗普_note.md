# 14、关键对话20讲：如何成为沟通高手：19-压力测试：如果对手是特朗普_note
## 第19讲：压力测试——如果对手是特朗普

在本节课中，我们将要学习如何应对风格激进、具有进攻性的对话对手。我们将分析这类对手常用的施压战术，并掌握一套“不怂不跑”、聚焦目标的应对策略，让你在高压对话中也能保持冷静，实现自己的沟通目标。

上一讲我们探讨了如何打破对话僵局。本节中，我们来看看当对话对手本身风格极具进攻性、给我们带来巨大压力时，应当如何应对。

压力下的对话，特指由对话对手激进的风格所带来的压力。这区别于因事情复杂或时间紧迫造成的压力。本节课的界限在于，对手的风格虽然激进，但尚未使用肮脏的伎俩，尚在可接受的范围内。对于“肮脏伎俩”，我们将在第22讲中具体分析。

面对风格激进的对手，常见的错误应对有两种：一是“以牙还牙，以眼还眼”，对方如何对待你，你就如何反击；二是始终保持温和，试图用教养和善意感化对方，甚至“别人打我的左脸，我把右脸也伸出去”。

然而，生活中的关键对话不是为了传道或教育对方，核心目标是实现利益和需求的满足。因此，上述两种方式都效果不佳。

今天，我们将介绍一种有明确目标的、针对进攻型对手的对话方式。本节课将以特朗普为例，假设你需要与他进行一场对话或商务谈判。目标是让你了解压力下对话的常见情况，识别进攻型对话者的常用战术，并掌握基本的应对策略。

### 核心原则：看表演，谈正事 🎭

应对夸张、有进攻性的对手，一个总的原则是：**我就看着你表演，表演完了回来说事情**。

当对方吹胡子瞪眼、拍桌子时，我们不能跟着他的节奏走。以牙还牙的思维方式并不可取，因为我们有自己的原则。同时，试图说服、感化对方，用善意转变其态度，往往也无效。

对方表现出夸张的进攻性，可能有两种情况：一是故意的策略（如特朗普），二是其本身的性格或状态使然。无论原因如何，我们的原则不变：**我不管你怎么成长，也不管你经历了什么，我们就是来说事情的**。

这种方式本身就是一种强硬。它体现在：无论对方如何表演、吵闹，都丝毫不能动摇你对对话目标的认定和追求，不能动摇你保持自己态度、解决问题的形式。这是一种健康的强硬。在这种态度下，对方反而可能动摇，因为他发现无法影响你。你的强硬有依据（你的利益和需求），而他的“狂风暴雨”则没有根基。

对于对手带来的压力，要做到“不怂也不跑”，这是一个高水平对话者的态度。我们等他表演完了，回到正事上说正事。

“不怂不跑”不代表不能暂时离席。那种“拂袖而去”、彻底关闭对话大门的方式不好，因为你的利益诉求还未解决。但你可以**暂停**对话。

以下是实施“暂停”的具体方式：
*   **提议休息**：例如，“我看你情绪比较大，我们休息一下，买杯咖啡，一会儿再谈。”
*   **使用“阳台策略”**：假设房间有阳台，你可以说“我去阳台上透会儿气”。这表示你并未离开对话环境，只是暂时中止，不接受对方当前的态度，但并未给对话画上句号。

### 识别常见施压战术 🔍

典型的进攻型对话者，最常使用的方式就是**威胁**和**虚张声势**。

#### 关于威胁

首先，我们自身**不要使用威胁**。威胁不是良好的对话方式。一方面，你的威胁可能根本吓不到对方；另一方面，即使对方暂时屈服，也会破坏关系利益，未来协议的执行将面临巨大风险。

我们不威胁别人，但别人可能会威胁我们。面对威胁，不要立刻服软或对抗，仍有办法周旋。

如果对方提出的威胁与你要求的事情完全不相称，或者后果你根本不害怕，可以一笑置之。

如果对方提出的威胁真实且严重，说明你们的谈判地位差距很大，你处于劣势。此时，首先要预见到可能无法达成目标。其次，在劣势下仍可采取**拖延和周旋**的策略。

以下是拖延和周旋的具体方法：
*   **多问问题**：例如，“你为什么这么生气？”、“这件事对你很重要吗？”、“你还有什么其他要求吗？”。通过聊天和提问来拖延时间。
*   **以拖待变**：事情可能因外部因素变化而出现转机。你可以说：“今天你这么激动，我们先不聊了，下周再约。”在这一周里，情况可能会发生变化。

#### 关于虚张声势

虚张声势可分为两种：**外表的虚张声势**和**对话条件上的虚张声势**。

**外表的虚张声势**，是指利用豪华的办公环境、昂贵的服饰、名车名表等来震慑你。例如，许多律师事务所会租用最贵的办公楼并豪华装修，就是为了在谈判中显示实力和主场气势。

**对话条件上的虚张声势**，是指在谈判中开出极端的高价或提出不可能的条件，让你在对比之下，觉得后续任何交易都“还不错”。这是一种心理战术。

对于这两种虚张声势，我们首先要**识别出来**，然后**不为所动**，忽略这些表面问题，坚定地回到我们既有的策略和利益轨道上来。

### 解析“特朗普式”战术 🃏

特朗普在《交易的艺术》一书中阐述了他的谈判理念，我们可以从中分析其常见战术。

1.  **极端高开价**：提出一个高到让你觉得交易不可能做成的价格（虚假锚点），然后在稍作让步时，让你觉得抓住了机会而成交。
2.  **展示“无所谓”姿态**：装作对谈判结果毫不在意，向你展示他拥有非常好的“最佳替代方案”（BATNA）。但请记住，只要双方能坐下来对话，就说明彼此有共同利益诉求，对方的“无所谓”多半是表演。
3.  **故作疯癫与出其不意**：喜怒无常，提出不可能的条件，在谈判中突然袭击（如一边谈着贸易，一边突然加征关税），或推翻已达成的共识。这种“不可琢磨”的状态是为了打乱对方节奏，迫使对方讨好和顺从自己。
4.  **持续宣传与刷存在感**：不停地讲同一件事，哪怕挨骂也不怕，目的是将自己塑造成在该问题上最有发言权的人，抢占话语权。

特朗普享受这种高压、不尊重的谈判过程，他开心，但你的情绪可能会被压制。**应对的关键是：他表演他的，你不要进入他的频率**。就像不要跳进泥塘和猪摔跤，因为猪很开心，而你只会不堪其扰。等他表演够了，我们回到正常的工作轨道，该谈什么谈什么。

### 总结与练习 📝

本节课我们一起学习了如何应对压力下的关键对话，重点是应对难缠、有进攻性的对手。

首先，我们确立了核心原则：**无论对手如何表演施压，我们都以“回到事情本身”的态度应对**。不拂袖而去，也不试图感化，而是采取一种“健康的强硬”——坚持自己的目标和轨道。必要时可以战术性暂停对话。

其次，我们分析了两种常见施压方式：**威胁**与**虚张声势**。对于威胁，我们自身应避免使用；面对威胁，可区分真假，对真实威胁采取拖延周旋策略。对于虚张声势（无论是外表还是条件上的），要识别并忽略，聚焦核心利益。

第三，我们解析了“特朗普式”战术，如极端高开价、展示无所谓姿态、故作疯癫、持续宣传等。看清这些策略背后的逻辑，它们就失去了神秘感和大部分威力。

**现在，我们来做一个练习：**

2.  **场景复盘与预演**：回想一个你与他对话的具体场景。当时他的哪些手段给你造成了压力？今后若再发生类似情况，如何运用本节课的原则（看他表演、适时暂停、回归正事）来应对？请思考并写下几个改进点。
