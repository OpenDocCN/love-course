# 【李大魔】《极致吸引》：012.十、乐观心态or悲观心态

🎼上节课结束后，有人可能会问，老师，我很悲观，怎样才能变得乐观？其实解决这个问题很简单，我们要从两个点去改善。

### 1. 改变时间长度

第一个点是把这个事情的长度，或者说这个事情所去影响的时间的长度，规划规定在此时此刻。你会发现，悲观的人总是会把一些事情的影响，看成是终身性的影响。所以，他们要去重新建立一个框架。

**例子**：一个人得了痛风，悲观的人会说：“我不能吃海鲜了，不能喝啤酒了，以后腿还要疼，怎么怎么的。”而客观的人会去说：“只是一个病而已嘛，我体的这种病以后注意就没事了。”

### 2. 缩小影响范围

第二个点是把影响的范围宽度把它缩小在一个点上面。

**例子**：一个悲观的人会说：“哎呀，我不是处女了，以后就没有人要我了。”没有人要我呢，我就没有法结婚了。无法结婚呢，就不会有孩子了，没有孩子呢，我就孤独终老了，以后一个人在一起了，最后呢，我最多养一只猫啊，我当我死了之后呢，猫还会把我去吃掉的。”

### 3. 乐观心态与悲观心态的区别

乐观的人是拥抱变化的，悲观的人是害怕变化的，他只能接受固定。所以，他们俩之间还有一个问题，其实很简单，就是乐观的人到最后是什么呢？我们改变我们所能改变的，接受那些我们所不能改变的事情，就是允许允许他不能改变，并且把接受的点呢控制在此时时刻，控制在这些事界上面。

### 4. 如何用乐观心态去解释事情

举个例子，之前有个师教志雄我，她这样评价她老公：“我对我老公啊很多时候是非常生气的，我一般都会发消息给他，和我老公讲道理，就前一跳，他给我拿了3万块。他说他是男人啊，这个钱他可以去做主的。而，我去说可以呀，你这个钱你可以拿走，但你这个钱你最起码要还给供应商呀。”

**悲观心态**：我以后怎么相信你啊？这就是一个典型的不存在这些事情发生的点上。今后呢你在我的心里面啊，你就是一个背着我挥霍这3万块的讨债鬼。

**乐观心态**：我们不要再谈今后了，我们要谈现在。第二个点呢，我们把这个事情的影响给扩散了。他原本的世界是我老公花了3万块，现在扩散到因为你花了3万块，所以我们要开始讨论供应商了，讨论完供应商之后呢，我要讨论你的人品。讨完人品之后呢，我们要去谈未未来。

### 5. 信念系统的更新

每个人的思想、情绪、行为都是他的内心的信念系统去支配。所以，你们想要内心强大，拥有乐观心态好的人生，就必须要去重新构建，更新那些限制你，阻碍你内心强大的消极信念。

**步骤**：

1. 刻意去察觉认识到自己的限制性的信念到底是什么。
2. 用新的积极信念来更新替代消极信念。
3. 用新的心念系统去指导你的新的行动。

### 6. 总结

本节课中我们一起学习了如何从悲观心态转变为乐观心态，以及如何通过更新信念系统来提升自己的心态。希望这些内容能够帮助大家拥有更加积极乐观的人生。
