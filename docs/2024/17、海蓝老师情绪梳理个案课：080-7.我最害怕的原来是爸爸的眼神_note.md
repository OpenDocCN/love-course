# 海蓝老师情绪梳理个案课：080-7.我最害怕的原来是爸爸的眼神

## 概述
在本节课中，我们将探讨一个关于孩子与父亲之间关系的个案，特别是孩子对父亲眼神的恐惧。

## 1. 父亲眼神的恐惧
**核心概念**：孩子对父亲眼神的恐惧源于童年时期的经历。

**内容**：
- 孩子回忆起10岁时，因金融危机导致父亲失业，家庭经济困难。
- 父亲因工作压力，对孩子态度变得严厉，眼神中透露出不耐烦和愤怒。
- 孩子因失眠被父亲责骂，感受到父亲的愤怒和严肃。

## 2. 父亲的愤怒与打骂
**核心概念**：父亲在愤怒时，会采取打骂的方式对待孩子。

**内容**：
- 孩子回忆起幼儿园时期，因不让母亲离开而遭到父亲打骂。
- 父亲在愤怒时，头部会出现火焰般的表情，眼神瞪大，表现出急躁和愤怒。

## 3. 孩子的感受与情绪
**核心概念**：孩子的恐惧和愤怒源于父亲的行为和眼神。

**内容**：
- 孩子对父亲的愤怒和严厉感到害怕，认为父亲的眼神充满愤怒和威胁。
- 孩子对父亲的打骂感到恐惧，认为父亲的行为是对自己的惩罚。

## 4. 情绪调节与成长
**核心概念**：孩子需要学会调节自己的情绪，并从经历中成长。

**内容**：
- 孩子意识到，父亲的愤怒和打骂并非针对自己，而是源于他的压力和焦虑。
- 孩子学会理解父亲的感受，并尝试与父亲沟通，表达自己的需求和感受。

## 总结
本节课中，我们一起学习了孩子对父亲眼神的恐惧，以及父亲的行为对孩子情绪的影响。通过了解这些，我们可以更好地理解孩子内心的感受，并帮助他们学会调节情绪，健康成长。
