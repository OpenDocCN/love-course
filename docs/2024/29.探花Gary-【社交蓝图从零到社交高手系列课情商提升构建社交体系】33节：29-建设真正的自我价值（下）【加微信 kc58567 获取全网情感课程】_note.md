# 社交蓝图从零到社交高手系列课情商提升构建社交体系教程：29-建设真正的自我价值（下）

🎼欢迎回来，我是Gary。上一节课我们讨论了内在价值，这节课我们将探讨如何构建外在价值。人的外在价值分为两个维度：生存价值和繁衍价值。

### 生存价值

生存价值与你的社会地位、房产、车辆、存款和社交圈子有关。然而，直接展示这些价值可能会显得庸俗。要体现高生存价值，你需要成为一个雄性领袖。

**公式**：雄性领袖 = 有朋友 + 提供价值

要成为雄性领袖，你需要：

1. **提供价值**：学会照顾他人，如组织聚会或解决问题时敢于做决定。
2. **扩大格局**：考虑团队利益，而非个人利益。
3. **锻炼领袖能力**：随着时间的推移，形成习惯，展现领袖能力。

### 繁衍价值

繁衍价值包括外貌、身材和体能等。以下是一些提升繁衍价值的方法：

1. **穿衣打扮**：选择合适的衣服，展现品味。
2. **发型打理**：去理发店设计适合你的发型，提升形象。
3. **身材管理**：锻炼身体，保持健康。

**代码**：身材管理 = 自律 + 坚持锻炼

### 课后总结

本节课我们学习了如何提升外在价值，包括生存价值和繁衍价值。通过成为雄性领袖和提升繁衍价值，你可以展现完整的自身价值。
