# 课程01：Python基础入门 🐍

在本节课中，我们将要学习Python编程语言的基础知识。课程将涵盖Python的安装、基本语法、变量、数据类型以及如何运行你的第一个程序。通过本课的学习，你将能够理解Python的核心概念，并开始编写简单的代码。

## 什么是Python？🤔

Python是一种高级、解释型的编程语言，以其清晰的语法和代码可读性而闻名。它广泛应用于Web开发、数据分析、人工智能、科学计算和自动化脚本等领域。

## 安装Python 💻

上一节我们介绍了Python的基本概念，本节中我们来看看如何安装Python。

以下是安装Python的步骤：

1.  访问Python官方网站（python.org）。
2.  下载适合你操作系统（Windows, macOS, Linux）的最新版本安装程序。
3.  运行安装程序，并确保在安装向导中勾选“Add Python to PATH”选项。
4.  完成安装后，打开命令行工具（如终端或命令提示符）。
5.  输入命令 `python --version` 来验证安装是否成功。如果安装正确，命令行将显示已安装的Python版本号。

## 第一个Python程序 ✨

现在，你已经成功安装了Python，让我们来编写并运行第一个程序。

1.  打开一个文本编辑器（如记事本、VS Code等）。
2.  输入以下代码：
    ```python
    print("Hello, World!")
    ```
3.  将文件保存为 `hello.py`。注意文件扩展名必须是 `.py`。
4.  打开命令行，导航到保存文件的目录。
5.  输入命令 `python hello.py` 来运行程序。
6.  你将在命令行中看到输出：`Hello, World!`

## 变量与数据类型 📊

在编程中，变量用于存储信息。Python中的变量不需要提前声明类型，解释器会自动推断。

以下是Python中的几种基本数据类型：

*   **整数（int）**：表示没有小数部分的数字，例如 `age = 25`。
*   **浮点数（float）**：表示带有小数部分的数字，例如 `price = 19.99`。
*   **字符串（str）**：表示文本，用单引号或双引号包围，例如 `name = "Alice"`。
*   **布尔值（bool）**：表示真（True）或假（False），例如 `is_student = True`。

你可以使用 `type()` 函数来查看任何变量的数据类型。例如：
```python
x = 10
print(type(x))  # 输出：<class 'int'>
```

## 基本运算 🧮

Python支持各种数学和逻辑运算。

以下是常见的运算符：

*   **算术运算符**：用于数学计算。
    *   加法：`+`
    *   减法：`-`
    *   乘法：`*`
    *   除法：`/`
    *   取模（求余数）：`%`
    *   幂运算：`**`
*   **比较运算符**：用于比较两个值，返回布尔值（True或False）。
    *   等于：`==`
    *   不等于：`!=`
    *   大于：`>`
    *   小于：`<`
    *   大于等于：`>=`
    *   小于等于：`<=`
*   **赋值运算符**：用于给变量赋值。
    *   基本赋值：`=`
    *   加法赋值：`+=` （例如 `x += 5` 等同于 `x = x + 5`）
