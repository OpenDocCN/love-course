# 老白《约会达人》：14、领袖小游戏--金字塔

## 概述
在本节课中，我们将学习一种名为“幻想拖拉机”的领袖小游戏。

## 游戏规则

### 幻想拖拉机
**定义**：幻想拖拉机是一种基于牌面组合和幻想的牌类游戏。

**玩法**：
1. **底牌**：游戏开始时，先亮一张底牌。
2. **个人牌**：每位玩家手中有一张牌。
3. **组合**：手中的牌加上底牌，再幻想一张牌。
4. **目标**：幻想出清一色顺子、瞬间或豹子即为拖拉机。
5. **换牌**：
   - 换手牌：喝半杯酒。
   - 换底牌：喝一杯酒。
   - 底牌最多可换三次。

### 清一色与顺子
- 清一色：所有牌面花色相同。
- 顺子：牌面数字连续。

### 豹子
- 豹子：三张牌面相同。

### 比牌
- 清一色通常大于顺子。
- 顺子的大小根据牌面数字决定。

### 喝酒规则
- 摇骰子决定喝酒量。
- 换手牌喝半杯，换底牌喝一杯。

## 游戏示例

### 开始游戏
1. **底牌**：3
2. **玩家A**：5
3. **玩家B**：幻想一张牌

### 换牌
- 玩家B认为手中的牌不好，幻想一张较小的牌，选择换手牌，喝半杯酒。

### 继续游戏
- 玩家B换底牌，喝一杯酒。
- 玩家B再换底牌，喝一杯酒。

### 结束游戏
- 玩家B换底牌，喝一杯酒。
- 玩家B不能再换底牌，只能换手牌。
- 玩家B换手牌，喝半杯酒。
