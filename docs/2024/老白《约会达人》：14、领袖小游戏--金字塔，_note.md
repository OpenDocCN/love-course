# 老白《约会达人》：14、领袖小游戏--金字塔，_note

在本节课中，我们将要学习一个名为“幻想拖拉机”的纸牌游戏。这个游戏也被称为“金字塔”，它结合了策略、胆量和一点运气，非常适合在社交场合中活跃气氛。我们将详细拆解游戏规则、流程和策略要点，确保初学者也能轻松掌握。

## 游戏概述 🃏

幻想拖拉机是一种基于“拖拉机”（又称“炸金花”）玩法演变而来的纸牌游戏。其核心在于，玩家需要利用一张公开的底牌、一张自己的手牌，并在脑海中“幻想”出第三张牌，从而组合成一个三张牌的牌型进行比大小。

## 游戏准备与基本规则 📋

游戏开始前，需要准备一副扑克牌。参与人数灵活，两人或多人均可，但人数越多通常趣味性越强。

以下是游戏的基本流程与核心规则：

1.  **发牌**：每位玩家获得一张手牌。另外，在桌面中央亮出一张牌作为“底牌”。
2.  **构成牌型**：每位玩家的最终牌型由三张牌组成：自己的手牌 + 公共底牌 + 自己幻想的一张牌。牌型大小遵循一般拖拉机规则，例如：豹子（三张相同点数）> 顺金（同花顺）> 金花（同花）> 顺子 > 对子 > 单张。
3.  **“幻想”环节**：这是游戏的精髓。玩家需要根据手牌和底牌，在脑海中构想出第三张牌，以组成尽可能大的牌型。

## 游戏核心环节：换牌与惩罚 🍺

如果玩家对当前的手牌或底牌不满意，可以通过喝酒为代价进行更换。这个机制增加了游戏的策略性和冒险性。

以下是换牌的具体规则：

*   **换手牌**：更换自己手中的那张牌。代价是喝**半杯**酒。
*   **换底牌**：更换桌面中央的公共底牌。底牌最多只能更换**三次**，且每次更换的代价递增：
    *   第一次换底牌：喝**1杯**酒。
    *   第二次换底牌：喝**2杯**酒。
    *   第三次换底牌：喝**3杯**酒。
*   **换牌顺序**：换牌按玩家顺序依次进行。每位玩家在自己的回合内，可以自由选择是否换手牌或换底牌（在次数限制内）。

## 胜负判定与最终惩罚 🎲

所有玩家完成换牌（或选择不换）后，进入亮牌环节。

1.  **亮牌与比大小**：所有玩家同时亮出手牌，并根据“手牌+底牌+幻想牌”的组合宣布自己的牌型。按照事先约定的规则（例如，顺子和金花谁更大）比较大小，牌型最小者为输家。
2.  **计算惩罚**：输家需要接受惩罚。通常的惩罚方式是摇骰子决定喝酒数量。例如：准备4个骰子，输家摇一次，骰子点数总和即为需要喝的“口数”。之后，根据“几口一杯”的约定（如3口一杯）换算成实际饮酒量。
3.  **总计饮酒量**：输家除了要喝掉比大小输掉的罚酒，还需喝掉自己在游戏过程中因换牌而承诺的所有酒。**总饮酒量 = 比大小输掉的罚酒 + 换牌过程中已喝的酒**。

## 实战演示与要点回顾 💡

上一节我们介绍了游戏的规则框架，本节中我们通过一段简化的实战对话来回顾关键要点，确保大家理解无误。

**场景**：四人游戏，底牌是6♠。
*   **玩家A**：（手牌10♣）觉得组合不大，选择“换底牌”（第一次），喝1杯，新底牌为K♥。
*   **玩家B**：（手牌6♦）看到公共底牌是6♠，幻想一张6即可组成豹子。选择“不换”。
*   **玩家C**：（手牌2♠）选择“换手牌”，喝半杯，换得新牌7♦。
*   **玩家D**：（手牌4♥）选择“换底牌”（第二次），喝2杯，新底牌为8♣。

所有玩家行动完毕，亮牌。
*   **玩家A**：手牌10♣ + 底牌K♥ + 幻想Q♦ = 散牌（K-Q-10）。
*   **玩家B**：手牌6♦ + 底牌6♠ + 幻想6♣ = **豹子**。
*   **玩家C**：手牌7♦ + 底牌8♣ + 幻想9♥ = **顺子**（9-8-7）。
*   **玩家D**：手牌4♥ + 底牌8♣ + 幻想5♦ = **顺子**（8-5-4，但花色不同）。

**胜负**：假设约定规则为豹子 > 顺子 > 金花 > 散牌。则玩家B（豹子）最大，玩家A（散牌）最小。
**惩罚**：玩家A是输家。他需要：
1.  摇4个骰子，假设点数和为9，则需喝9口罚酒。
2.  加上他之前换底牌承诺的1杯酒（若约定3口一杯，则1杯=3口）。
3.  玩家A总计需喝 9口 + 3口 = 12口酒。

本节课中我们一起学习了“幻想拖拉机”（金字塔）的完整玩法。记住核心流程：**以手牌和底牌为基础幻想组合 -> 通过饮酒代价换牌优化牌型 -> 最终亮牌比大小 -> 输家承担总罚酒**。这个游戏的关键在于胆识、策略以及对规则的清晰理解。在实际游戏中，请务必提前与所有玩家确认好牌型大小顺序、酒杯容量及骰子惩罚规则，以确保游戏顺畅进行。祝大家玩得开心！
