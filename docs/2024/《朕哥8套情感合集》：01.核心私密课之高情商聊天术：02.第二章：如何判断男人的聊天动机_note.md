# 《朕哥8套情感合集》：02.如何判断男人的聊天动机

在本节课中，我们将学习如何判断男人的聊天动机，并了解两个重要概念：兴趣指标和情感动机。

## 1. 男人的情感动机

**核心概念**：99%的男人的初始动机是找女人聊天，以满足自身欲望。

- **99%的男人的动机是找女人聊天，目的是满足自身欲望**。
- **男人想要的是钓鱼还是真心喜欢，取决于他是想短期关系还是长期关系**。
- **初始见面的场景决定了你们是浅入还是深入**。

## 2. 线上与线下聊天的区别

**核心概念**：线上聊天70%是语言，30%是展示面；线下聊天70%是肢体语言，30%是语言。

- **线上聊天重点在语言，线下聊天重点在肢体语言**。
- **男人更看重线下见面时的肢体动作和形象**。

## 3. 试吃原则

**核心概念**：用试吃原则扭转男人的情感动机，让他上钩。

- **试吃原则：给男人一点好处，让他觉得你很好得到**。
- **不要给太多好处，让他上钩，然后你就可以钓到更多的鱼**。

## 4. 兴趣指标

**核心概念**：兴趣指标是判断男人对你需求多少的重要标准。

- **兴趣指标分为真性兴趣指标和假性兴趣指标**。
- **真性兴趣指标：男人真的对你感兴趣，付出和兴趣度都是真实的**。
- **假性兴趣指标：男人为了得到你，或让你觉得他很认真，释放的兴趣指标**。

## 5. 如何利用男人的聊天动机拿捏男人

**核心概念**：了解男人的思维，利用他的思维来搞定男人。

- **钓鱼思维**：不要盯着鱼饵，你盯着鱼饵，你就是鱼。
- **责任感逃避思维**：把自己标榜成一个玩家，吸引男人。
- **试吃体验**：给男人一点甜头，让他追逐你。
- **白嫖画大饼**：展示你的价值，但不让他轻易得到。
