# 老白《约会达人》：情场3.0：【情场3.0】15.追求阶段三要素_note

在本节课中，我们将要学习追求阶段（L4）的核心三要素。在成功建立吸引、让女生对我们产生好感之后，关系便进入追求阶段。这个阶段是女生进行“废物测试”最频繁的时期，处理好这些测试是关系升级的关键。

上一节我们介绍了如何建立吸引，本节中我们来看看如何稳妥地通过追求阶段的考验。

## 追求阶段的挑战与核心目标

追求阶段女生会提出许多难以回答的问题，例如“我和你妈妈同时掉水里，你救谁？”、“你为什么喜欢我？”、“你和你前女友为什么分手？”。这些问题回答不好，容易导致女生疏远你并为你减分。

这些问题看似不同，但归根结底都指向两个终极疑问：
1.  **你是不是真的喜欢我？**
2.  **你是不是一个有价值的、靠谱的人？**

女生的底层需求在此阶段显现，即**安全感**与**确定性**。因此，我们追求阶段最主要的目的，就是塑造**靠谱**（DP）的形象，让她确信你的喜欢是真诚且认真的。

## 追求阶段三要素详解

为了达成“靠谱”（DP）的目标，我们需要掌握以下三个核心要素。

### 要素一：筛选（CG）

筛选（CG）是指通过提问或表达标准，让女生感觉到你也在认真考察她。这样做能传递一个关键信号：只有真心想谈恋爱的人才会仔细挑选。

以下是进行筛选（CG）的具体方法举例：

*   **询问恋爱史**：你可以问“你谈过几个男朋友？”。如果她回答“三个”，你可以说“比我多一个”，并配上一个委屈的表情。这表明你对此事有态度。
*   **表达个人介意点**：例如问“你和前男友同居过吗？”。如果她说“没有”，你可以回应“那还好，我有点介意同居”。如果她说“有，同居了两个月”，你可以说“哦，那还好（同居不介意，但时间长短我介意）”。关键在于，你没有直接说出的介意，她会感觉到，从而明白你是认真的。

**核心逻辑**：当你表现出在意她的某些“缺点”（如恋爱次数、身高、性格等）却依然选择她时，在她看来，这种“在意但包容”本身就是一种爱。

### 要素二：暖男行为

暖男行为是指在合适时机对女生表达关心、照顾和体贴。此时，你可以将那些真诚的、对女生好的方式拿出来。

关键在于，这些行为必须在**追求阶段（L4）** 才有效。在建立吸引前期过早使用，会被视为“跪舔”，效果适得其反。

以下是暖男行为的具体示例：

*   **留意细节**：例如，注意到她吃饭时挑出葱花，下次单独约会点菜时，主动告诉服务员所有菜都不放葱。这个细节会让她非常感动。
*   **日常照顾**：一起走路时，让她走在内侧（靠人行道一侧），自己走在外侧（靠机动车一侧）。上车时，用手为她护住头顶防止碰头。这些细微的照顾都能积累好感。

**核心逻辑**：女生当然希望男生对她好，这能有效提升关系。但时机是关键，在吸引建立后、追求阶段时的“好”，才会被解读为真诚的喜欢，而非别有所图。

### 要素三：走心（DT）与进退节奏

走心（DT）指的是情感上的真诚互动。在此环节，掌握主动与回应的节奏至关重要。

*   **男生的主动**：应是**靠近**。主动发起聊天、邀约，推进关系。
*   **女生的主动**：通常是**配合或不拒绝**你的靠近。例如，积极回复你的信息，接受你的邀约。

当女生偶尔主动时（例如主动找你聊天、轻微肢体接触），我们的策略是**先后退**。

这类似于逗猫的原理：一个物体轻轻晃动，猫会好奇地靠近并伸出爪子去挠。如果物体此时猛地扑上来，猫会吓得立刻跑开。但如果物体在猫挠的时候稍微后退，猫反而会更感兴趣，继续向前。

**具体应用**：当女生主动时（比如她调侃你或轻轻碰你），不要立刻猛烈升级关系。可以适当后退（比如开玩笑地躲开或转移话题），这会让她觉得有趣并更想“征服”，从而更主动。待她再次主动时，你再给予正面、温和的回应，关系推进就会非常稳固。

## 课程总结 🎓

本节课中我们一起学习了追求阶段（L4）的核心策略与三要素：

1.  **核心目标**：追求阶段的核心是塑造**靠谱（DP）** 的形象，满足女生的安全感需求。
2.  **三大要素**：达成目标需综合运用**筛选（CG）**、**暖男行为**和**走心（DT）** 三要素。
3.  **筛选（CG）的作用**：通过表达自己的标准和介意点，向女生传递“我正在认真考虑你”的信号，这是真心的表现。
4.  **暖男行为的时机**：对女生好是有效的，但仅在**追求阶段（L4）** 开始使用才有效，过早则无效甚至反效。
5.  **主动性的区别**：男人的主动是**靠近**；女人的主动是**配合**男人的靠近。
6.  **应对女生主动**：当女生主动时，策略是**先后退**，以维持吸引和好奇心，然后再寻找机会稳妥推进。

掌握好这三要素的平衡与时机，你就能平稳度过追求阶段，成功升级关系。
