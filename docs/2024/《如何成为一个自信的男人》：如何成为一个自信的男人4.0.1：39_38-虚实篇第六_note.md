# 《如何成为一个自信的男人》：如何成为一个自信的男人4.0.1：39_38-虚实篇第六

大家好，我是乃一。今天我们继续学习乃一品孙子兵法的虚实篇第六节课。

## 核心概念：虚实

**原文描述**：夫兵形向水，水之形避高而趋下。兵之形避时而积虚，水因地而制流，兵因敌而制胜，故兵无常势，水无常形。

**含义**：用兵的规律如同水，水流动避开高处流向低处。用兵打仗应避开敌人坚实之处，攻击其虚弱之处。根据地形变化调整流向，根据敌情制定取胜策略。用兵无固定模式，如同水流无固定形态。

**公式**：用兵策略 = 根据地形 + 敌情

## 利用虚实

**目的**：调动敌人，不被敌人调动。

**原文描述**：故善战者自人，而不至于人。

**含义**：善于指挥作战的人，能调动敌人，不被敌人调动。

## 孙子兵法四句话

1. 能使敌治治者利之也。能使敌不得志者害之也。敌亦能劳之。保能击之，安能动之，能使敌人自投罗网。
2. 出其所不趋，趋其所不意，出兵要指向敌人无法救援的地方，行动于敌人意料不到的地方。
3. 故行人而我无形，则我专而敌分，我专为一，敌人为实，是故以十攻其一也。则我纵而敌寡。
4. 故策之而知得失之计，坐之而知动静之理。行之而知死生之地，缴之啊而知有余不足之处。

## 撩妹指导意义

1. **避实击虚**：了解女性的弱点，如渴望爱情，利用这一点来吸引对方。
2. **调动对手**：学会调动对方，不被对方调动，掌握主动权。
