# 凡哥私教课程：长期关系经营课：009.09：爱情是勇敢者的游戏

🎼生物人类学家海伦费舍尔认为，爱是一种需求，一种渴望，是探寻生命中最大奖赏的驱动力。爱情难以抗拒，即使不相信爱情的人，一旦爱情来临，也会屈服。然而，爱情并非轻易可得。

**以下是关于爱情的几个常见问题：**

1. **渴望爱情却不得其门而入**
2. **在爱情中伤痕累累**
3. **曾经抓住过爱情，却又失去了他**
4. **总是在疑惑，到底什么是爱情**

人们常常将爱情中的不可得或失败归咎于个人运气或意志力的问题。然而，社会学学说认为，这可能是时代问题。

**现代社会与传统社会相比，最根本的不同在于不确定性。**

在传统农业社会，人们生活在同一片土地上，生活稳定。而在现代社会，人们可以自由选择生活、工作、结婚等，但每个选择的结果都可能不在意料之内。

**爱情的不确定性使得很多人焦虑。**

在这个不确定的时代去实践爱情，是勇敢者的游戏。需要相信爱情的存在，拥抱不确定性，愿意投入到一段不确定结局的时间流动中。

**想要经营一份成长治愈型爱情，需要智慧。**

我们需要守护底线，保护自己，反思自己的爱情脚本，跳出双标，学会让两个真实而独特的自我连接在一起。

**以下是爱情发展的四个阶段：**

1. **爱情的发生**：与很多人形成廉结，变成朋友，既能分享生活也能互相帮助。然后发现一个人，对他尤其好奇，想要靠近他。
2. **表白和热恋**：两情相悦，需要有人主动跨出一步确认关系。
3. **稳定关系**：爱情逐渐稳定下来，彼此习惯了对方的存在。
4. **彼此承担责任**：结婚，成为受法律保障的利益共同体、经济共同体。

**爱情是一步步发展的结果，不是一开始就设定的目标。**

每一步都需要勇气、信任和智慧，两个人才能实现更深入的连接。

**爱情不是人生的全部，但爱情最终的目的是个人的幸福。**

爱情让我们成长起来，拥有了幸福的能力。

**本系列课程的主题是爱情，但并不局限于爱情。**

当我们拥有处理爱情的能力时，我们会发现，这些能力在处理其他亲密关系和社会关系时同样适用。

**让我们勇敢出发，拥抱爱，拥抱自我，拥抱世界，助你幸福，让我们一起幸福。**
