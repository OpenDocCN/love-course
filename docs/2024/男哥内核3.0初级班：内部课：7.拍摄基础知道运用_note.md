## 🖼️ 核心构图技巧：九宫格法

了解了朋友圈需要什么内容后，我们进入拍摄环节。首先学习最基础的构图方法：九宫格构图。

在手机相机设置中打开“网格”或“九宫格”功能。九宫格将画面横竖三等分，产生四条分割线和四个交叉点。

**公式**：`画面被分为9个等份，产生4个视觉焦点（交叉点）`。

这四个交叉点最容易吸引视觉注意力。构图时，将拍摄主体放置在交叉点或分割线附近，可以有效突出主体，使成片更鲜明。

我们通过案例来理解。以下三张图片运用了相同的构图原理：

1.  将拍摄主体置于右侧的分割线或交叉点附近。
2.  主体所占面积约为画面的 **1/3** 或更小（即约占用九宫格中的 **1到4个小格**）。
3.  留出 **2/3** 或更多的空白区域（留白），用以展现背景环境。

这种构图能强烈突出主体，并利用留白营造氛围。类似的构图变化很多，例如将主体从右侧换到左侧，通过调整角度来更好地展现背后有线条、轮廓和光影结构的建筑背景。

**一个实用技巧**：在拍摄全身照时，无论怎么构图，人物的脚部位置通常不要离照片底部边框太远。这有助于在视觉上显高，是需要熟练掌握的技巧。

## 👤 人物特写与自拍构图

为什么需要特写照？因为如果所有照片都过于突出环境，就没有一张能清晰展示你个人形象的照片，这也会导致朋友圈元素缺失。

特写构图同样遵循上述原理：

*   **第一种（类似小四格构图）**：主体放大，但依然保留足够的留白空间来衬托整体效果。
*   **第二种**：主体占据画面约 **2/3**，留白约占 **1/3**。这样拍出的照片也不会显得拥挤。

**代码**（描述面积占比）：`主体面积 ≈ 2/3；留白面积 ≈ 1/3`。

大家在自拍时，可以以上述构图作为参考标准。回顾之前的案例照片，就能明白自拍构图的基础运用原理都是相通的。

## ⚙️ 常用构图：小两格法

我们接着来看另一种日常最常用的构图方式：小两格构图。

以下是运用小两格构图的四张图片。方法是调整拍摄角度，将主体固定在画面中间的两个格子区域内。

**注意**：构图无需绝对精确，主体占比大致符合 **1/3**、**2/3** 或两格区域即可，稍有出入没关系，但不要偏差太大。

采用小两格构图时，拍摄角度最好选择**仰拍**或**平拍**，尽量少用**俯拍**。俯拍角度难以驾驭，容易拍出不理想的效果。

**总结**：日常拍摄中，最常用、最实用的就是 **小四格** 和 **小两格** 这两种构图方式。熟练掌握它们，足以解决大部分展示面拍摄的构图问题。

## 🕴️ 人物姿势要领：轮廓感

掌握了构图，接下来我们学习如何摆出好看有型的姿势，让最终照片令人满意。

这里不讲解复杂知识，只分享最干货、最实用的核心要点，总结为两个字：**轮廓**。

大家可以看到，在示例图片中，都用线条标注出了人物的姿势轮廓。人的肢体有局限性，但要想照片有视觉冲击力，就需要让肢体呈现出清晰的轮廓感，而不是呆呆站立或双手叉腰的“游客照”姿势。

**建议**：将今天课程中的示例照片保存到手机。实地拍摄时，以这些姿势为参考标准，不断调整和练习，熟能生巧。

## 📐 拍摄角度的选择与调整

最后，我们来探讨拍摄角度的调整。一张照片包含主体和背景两个要素。

背景角度的选择，取决于你想突出表现什么。例如，在咖啡厅，你可以选择以咖啡杯为背景，也可以选择以旁边的钢琴为背景。**首先确定你想拍摄的背景内容和角度。**

找好背景角度后，再将人物主体加入其中。关于人物拍摄角度，并没有固定公式，因为每个人的颜值、穿搭和场地都不同，想突出的点也不同。这里只能提供大致框架，需要大家通过大量实操来找到最佳效果。

**练习方法**：
1.  **背景**：根据想表达的内容固定选择。
2.  **人物**：日常多练习自拍，找到自己最上镜、最能凸显优点的角度（如侧脸45度）。在后续拍摄中可经常使用这个角度，但也要有所变化，避免所有照片角度雷同导致审美疲劳。

## ❓ 常见个人问题解答

很多兄弟会提出一些与自身情况相关的具体问题。以下是一些例子及解决方案：

*   **问题：发际线高，拍出来脑门大。**
    *   **解决方案**：采用 **30度仰角** 拍摄，这个角度基本可以规避发际线过高的问题。

*   **问题：个子矮，拍出来不好看。**
    *   **解决方案**：
        1.  穿着有内增高的鞋子，或使用内增高鞋垫，可立即增高数厘米。
        2.  打理一个蓬松的发型，视觉上也能增加身高。
        3.  拍摄时运用 **小四格** 或 **小两格** 构图。
        4.  后期修图时，可适当拉长小腿比例，营造更高挑的即视感。
        *   **注意**：修图要适度，避免失真（如修成1米9），导致“见光死”。约会时也需通过穿搭和发型保持形象一致性。

*   **问题：拍照姿势僵硬、不自然。**
    *   **解决方案**：这通常是因为练习太少。没有人天生会拍照，都需要后天学习和大量练习。希望大家在学习课程后，能坚持实操，对自己的提升负责。
