# 男哥内核3.0初级班：内部课：7.拍摄基础知道运用

大家好！今天我们将学习如何拍摄出令人满意的照片。以下是本节课的主要内容：

## 朋友圈元素

朋友圈的元素丰富性是关键。添加日常、生活、自拍、学习、玩具等元素，避免过度自恋的印象。

**公式**：朋友圈元素 = 日常 + 生活 + 自拍 + 学习 + 玩具

## 拍摄技巧

### 九宫格构图

使用手机中的九宫格构图功能，将画面分为九个格子，将主体放在交叉点附近，突出主体。

**代码**：九宫格构图 = 交叉点 + 主体

### 人物自拍特写照

拍摄人物特写照，清晰展现自己，避免背景干扰。

**代码**：人物自拍特写照 = 清晰展现自己

### 小两格构图

使用小两格构图，调整角度，将主体放在中间两个格子中。

**代码**：小两格构图 = 中间两个格子 + 主体

## 摆姿势

摆姿势的关键在于轮廓感，避免僵硬的游客照。

**公式**：摆姿势 = 轮廓感

## 拍摄角度

根据想要突出的内容选择背景角度，加入人物主体。

**代码**：拍摄角度 = 背景角度 + 人物主体

## 个人问题解决

针对个人问题，如高发际线、矮个子等，提供解决方案。

**公式**：个人问题解决 = 仰拍 + 内增高 + 发型打理

## 总结

本节课我们学习了朋友圈元素、拍摄技巧、摆姿势和拍摄角度等知识。希望大家通过实操不断提升拍照能力。
