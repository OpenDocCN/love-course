**情绪价值**：一个人影响他人情绪的能力。

一个人越能给他人带来舒服、愉悦和稳定的情绪，他的情绪价值就越高，反之越容易让他人产生焦虑、低落和难堪的情绪，他的情绪价值就越低。情绪价值对彼此在亲密关系中能否有幸福体验感，起着非常重要的作用。😊。

上节课讲了提供情绪价值由浅入深的第一个层次，也就是倾听和积极反馈。想要做到这一层，可能更多的需要你的耐心和行动力。这节课我们将继续讲解接下来的两个层次。

### 第二层次：分享

你不仅需要有耐心和行动力，更需要你的收集和整理能力。它涉及到情感、思想、欲望、需求、梦想、恐惧等多个维度的输出。

**分享**：涉及到情感、思想、欲望、需求、梦想、恐惧等多个维度的输出。

现在网络上流行这样一句话来检测夫妻之间的亲密度。就是你和老公还会说废话吗？A不说，那你们已经接近纯洁的舍友关系了。也就是互相开启飞行模式。两个人明明没有关机，却互相没有信号。B以前说现在少量，那恩爱已经离你们很远了，有一点连接，但已经有了疏离。日常交流呢仅限于孩子，其他几乎没有C经常说，那么你们仍然亲密，你中有我。我中有你。这里的废话其实就是分享欲，就是我很想告诉你我的所见所闻所思所想。但如果我在最初和你说废话的时候，你就无视或制止。那慢慢的我也就不想和你分享了。

**分享欲**：我很想告诉你我的所见所闻所思所想。

分享一个看似简单的行为，实则是一种顶级的浪漫。蕴含着深远的含义。在亲密关系中，分享不仅是一种行为，更是一种沟通方式，一种深深的情感连接。

**分享的本质**：愿意让对方参与自己生活的表示，愿意让对方了解自己内心世界的信任。

它能帮助双方建立深厚的感情基础，增强彼此间的亲密度。当我们与伴侣分享自己的经历和感受。我们就有机会反思自己，了解自己的需求和期望。同时呢也能帮助我们了解和接纳对方。

**分享的意义**：帮助双方建立深厚的感情基础，增强彼此间的亲密度。

当我们倾听伴侣分享他们的经历和感受，就有机会了解他们的那些世界，理解他们的需求和期望。这样我们就能更好的满足彼此的需求，增强彼此的感情。

**倾听的意义**：了解对方的内心世界，理解对方的需求和期望。

### 分享的误区

1. **误区一：很多女生的分享仅限于物质**。

   在我的咨询当中，经常有丈夫们来吐口水，我老婆很有能力，我也很自豪这一点，但她太强势，脾气很暴躁。我有时候很心疼她工作忙，想和她聊一聊。她就说我没精力和你聊。

   我就说说我希望你可以和我分享，他就会说我挣的钱也有你的一半吧，这不算分享吗？大家看类似的女强人，在我们这个时代还是挺多的。有些女生会模仿那些只知道挣钱的男人，只知道分享物质资源。但夫妻的长期相处中。

   物质分享只是一部分，而非物质分享，比如感受和想法。这些往往更能拉近彼此的距离。

2. **误区二：分享就是毫无保留的倾诉一切**。

   一些女生可能认为在亲密关系中爱她就应该毫无保留。分享就应该是包括自己的所有想法，感受和经历，无论多私密。然而，分享应该是建立在相互信任和尊重的基础上的。

   不是毫无保留的倾诉一切。

3. **误区三：分享时带有情绪化的表达**。

   有时候你在分享时可能会带有情绪化的表达，如愤怒、悲伤或焦虑，也就是通常所说的带着气的。情绪化的表达可能会让伴侣感到困惑或不安，也可能导致沟通障碍。

4. **误区四：忽视分享的平衡性**。

   在亲密关系中，分享应该是双向的。然而。

   有时候你可能会更多的关注自己的分享需求，而忽视了与伴侣的互动和倾听。

5. **误区五：过度依赖对方的回应**。

   当你进行分享时，可能会期望伴侣给予特定的反应。

   如同情赞美或建议。如果对方没有及时回应，或者回应不如自己所期望，可能就会感到失望和不安。

### 如何进行分享

1. **选择合适的时间和地点**。

   分享不仅仅是说出自己的想法和感受。

   更是要让对方能理解和接受。因此，选择一个安静、轻松的环境和一个对方能专心倾听的时间。

2. **尊重对方的感受**。

   诚然我们分享的目的是为了获得有效的回应，但对方没有get到我们的点的时候，也需要我们能多一份理解和包容，尊重对方的感受。

3. **求同存异**。

   在分享中，我们需要理解我们和对方可能有不同的观点和感受，这是正常的。我们需要学会接受这些差异。

4. **坦诚和真实**。

   在分享时应该坦诚而真实的表达自己的想法和感受，不要掩饰或夸大。

### 第三层次：共情

在亲密关系中，共情指的是一种理解和体验对方内心世界的能力。

**共情**：一种理解和体验对方内心世界的能力。

它涉及到站在对方的角度去感受对方的情绪、想法、态度、动机和需要，并且将这些理解和感受，通过言语或非言语的方式表达出来，让对方感受到被理解和接纳。

### 共情的基础

1. **同情心**。

   你身边有没有这样的人。

   或者你是不是这样的人呢？男朋友和她母亲感情很好，一周一次啊吸收平常的分享，都会被你说成是妈宝男。老公因为工作不顺利，被领导训斥，和你分享，你却理解不了。你认为作为一个男人，抗压能力这么差，很不爷们儿。

   男朋友因为把一件重要物品弄丢了，伤心不已，你却觉得他很矫情。

2. **理解和接纳他人的情绪**。

   前面情绪管理的课程中讲到，不管我们自己出现了何种情绪，首要要做的并不是压制和排斥，而是接纳。

3. **将心比心，换位思考**。

   这需要你暂时抛开自己的立场和感受，真正站在对方的角度来思考问题。

4. **分析他人处境的能力**。

   这需要你使用情绪管理课程里的情绪管理ABC理论，对伴侣的情绪进行分析解读。

5. **提供走心的情感支持**。

   你需要用言语和行动告诉对方，我理解你，支持你，无论发生什么，我都会在你身边。
