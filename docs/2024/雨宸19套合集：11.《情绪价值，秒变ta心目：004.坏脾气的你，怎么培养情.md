# 雨宸19套合集：11.《情绪价值，秒变ta心目中最特别的人》：004. 坏脾气的你，如何培养情绪定力 🧘‍♀️

在本节课中，我们将要学习如何培养情绪定力。情绪化不仅难以给他人提供情绪价值，还可能让自己先陷入崩溃。管理好自己的情绪是提供价值的前提。

## 一、正确认识情绪

上一节我们提到了情绪管理的重要性，本节中我们来看看如何正确认识情绪。每种情绪的出现都有其原因，负面情绪往往反映了内心最软弱和无助的地方。

这种软弱和无助感源于我们的内在相处模式。外在言行是内在思维的投射，而内在思维则是在成长过程中形成的。

我们的性格，或者说内在相处模式，主要形成于6岁前与父母的互动中。以下是一个具体案例：

我曾有一位客户，她在感情中特别情绪化，其核心原因是无法接受任何否定。哪怕是轻微的批评，例如关于外貌或行为的评价，她都难以承受。她的男友在相处8个月后因无法忍受而分手。她自己虽想改变和讨好对方，但一旦再次面临否定，就会立刻崩溃。

这种反应源于她童年时期经常被父母否定和不认同的经历。她因此认为自己很差劲，并将任何批评都内化为对自我价值的否定。她努力追求完美，试图通过外部认同来肯定自己。

例如，初中时数学考了118分（满分120分），她因那2分的“误差”而痛哭。考钢琴十级时，仅错一个音符，回家后却自责到抽自己耳光。长期的不被认同，导致她对“否定”行为极度敏感。

她的本质问题是：**不接纳自己的不完美，不允许自恋受损**。她的情绪崩溃，实质上是无法面对内心那个“不完美”的自我在关系中被触及。

## 二、坏脾气的根源：习得性无助

刚才我们探讨了情绪背后的心理成因，接下来我们深入分析坏脾气的根源。这通常与一个心理学概念相关：**习得性无助**。

习得性无助是一种消极被动的行为模式，个体深受环境影响，习惯性崩溃。其形成往往可追溯至童年。

当孩子的负面情绪不被父母看见和接纳时（例如常听到“小孩子有什么可烦的”），他们不得不压抑这些情绪。长期压抑导致他们不知如何健康表达，最终在承受不住时崩溃。

为什么我们会压抑自己？因为在早期，当表达情绪换来的是否定和批评时，我们只能内化这种攻击，认为自己“很差劲”、“不对劲”。那位案例中的女孩就是如此，她通过否定自己来“适应”与父母的相处模式，因为“我是个垃圾，所以被批评是应该的”。这种扭曲的状态反而让她感到熟悉和“舒适”。

因此，当她在亲密关系中感受到类似的否定（哪怕只有一点点），就会立刻触发童年的痛苦记忆，导致情绪崩溃。此外，这还可能衍生出另一种情绪：**在亲密关系中寻求补偿**。

由于在原生家庭中缺乏认同，个体会希望在伴侣身上获得加倍补偿。他们会像孩子一样对伴侣发泄情绪，期待对方能给予父母未曾给予的接纳和包容。

但矛盾的是，因为他们早已习惯“不被认同”的模式，所以即使伴侣给予接纳，他们也会不断试探，直至将对方推远。对方的离开，又再次验证了“我不被爱”的信念，从而强化了这种错误的相处模式。

**我们在每一段关系中，都在无意识地重复与父母的内在相处模式。** 无论这个模式正确与否，我们都会努力让它重现。如果童年模式是“被批评、被否定”，我们也会在亲密关系中想方设法让对方批评、否定并最终离开自己，因为那才是我们熟悉的、“真实”的状态。

另一种情况是“巨婴”心态。被溺爱长大的孩子，缺乏“我”与“他人”的边界感，认为世界必须围着自己转。他们无法接受否定，一旦事情脱离掌控就会崩溃，并且从不为自己的负面情绪负责，就像婴儿一样，认为所有问题都该由别人（伴侣）来解决。

## 三、如何培养情绪定力

认识了问题根源后，我们来看看具体的解决方法。如何打破习得性无助，培养情绪定力？以下是三个核心步骤。

### 1. 改变归因模式：从内归因到外归因

首先，要打破习得性无助，关键在于将**内归因**转变为**外归因**。

**内归因**是指将他人的情绪和行为都归结为与自己有关。例如，案例中的女孩，当男友说“不想聊天”时，她立刻内归因为“他是不是不喜欢我了”。

而**外归因**则是指认识到，他人的情绪和行为可能源于其自身处境，与你无关。引导她问男友“是否心情不好”后，得到的回复是：“是我自己的问题，想静一静，不想把负面情绪传给你。”

心理学调查发现，幸福的伴侣大多善于外归因。例如，丈夫回家摔包，不幸的伴侣会想：“他一定是对我不满”；而幸福的伴侣则会想：“他今天可能遇到烦心事了”。

**公式：事件归因 = 他人自身原因 + 环境因素，而非“我的问题”。**

学会外归因，意味着理解：别人的批评、指责或拒绝，很多时候只是基于他们自身的视角和状态，并非针对你个人。建立稳定的自我认同后，外界的评价就很难动摇你。

### 2. 学会释放情绪

许多人长期压抑情绪，内心树立了一个“必须完美”的模板，这是社会或家庭要求的结果。但我们必须承认，**有情绪是正常的，不完美是人的常态**。

以下是释放情绪的步骤：
*   **首先，接纳**：从心态上承认“我有情绪是正常的”、“我不完美是正常的”。
*   **然后，负责**：为自己的负面情绪找到健康的宣泄渠道，例如独处、听音乐、运动等。
*   **重要原则**：**永远不要在情绪激动时做任何决定**，此时的决策往往是错误的。

### 3. 建立自我奖励机制

改变需要正向反馈。每成功应对一次情绪波动，都要及时鼓励自己。

**代码式思维：**
```plaintext
if (成功进行外归因 || 成功疏导自身情绪) {
    自我鼓励(“我今天真棒！”);
    强化新的行为模式;
}
```

通过每一次微小的成功来鼓励自己，逐步养成新的思维和行为习惯，一点一点打破原生家庭带来的负面影响。

## 总结 🎯

本节课中我们一起学习了如何培养情绪定力。

1.  **正确认识情绪**：负面情绪是内心软肋的暴露，其根源常与童年形成的内在相处模式有关，尤其是对“不完美自我”的不接纳。
2.  **理解坏脾气根源**：**习得性无助**是习惯性情绪化的关键，我们会在亲密关系中无意识地重复童年与父母的错误相处模式。
3.  **掌握培养方法**：
    *   **改变归因**：练习**外归因**，明白别人的情绪不一定与你有关。
    *   **释放情绪**：接纳自己的不完美，并为情绪找到健康的出口，避免在情绪中做决定。
    *   **自我奖励**：为每一个小小的进步鼓励自己，逐步建立新的情绪反应模式。

总结来说，不要将负面情绪妖魔化。当情绪出现时，抓住它，分析其成因，接纳它作为你的一部分。不要期待自己成为没有情绪的“完人”，而是以凡人的标准要求自己：我们可以有情绪，但要知道，别人的情绪是别人的事。不要用**内归因**重复原生家庭的错误，慢慢将自己引向健康、稳定的轨道。
