# 796、【完结】陈海贤·自我转变50讲：chx03 自我三阶段：“我想要”如何生长出来

欢迎来到自我转变五十讲。我是陈海贤。今天，我们将开始自我转变的旅程，第一站是“我想要”。这是自我转变的基本动力，也是所有转变的缘起。

### 我想要的重要性

“我想要”有很多层次，从简单的“我想吃什么”到更郑重的“我想成为什么样的人”。这些“我想要”的核心是我们想要成为什么样的人。

**为什么“我想要”对转变如此重要**？

因为我们的自我和外界的现实之间存在永恒的矛盾和张力。当我们想要突破现实的限制，实现自我时，“我想要”就变成了一个重要的信号。

### 人的自我意志和现实之间的矛盾

我听武汉大学的于峰教授讲过一个故事：一个孩子想要一箱可乐，妈妈为了满足孩子的愿望，不惜搬动一箱箱可乐。这个故事展示了人的自我意志和现实之间的矛盾。

**如何处理“我想要”**

当我们想要一个东西时，我们会如何对待它？是确认它，还是忽略它？这背后是两种相互矛盾的本能：自我成长和自我确立的需要，以及保护自己的本能。

### 自我转变的三个阶段

1. **自我是关系的背景**：在这个阶段，自我还很弱小，我们努力在社会和关系中找到一个位置，满足他人的期待。
2. **自我和关系相互矛盾和相互冲突**：在这个阶段，我们会感觉到“我想要”的声音，尤其是与现实相悖的声音。
3. **自我成为主体，关系变成背景**：在这个阶段，自我意识开始清晰，我们开始认真审视“我想要”。

### 自我转变的意义

自我转变意味着我们不再从属于特定的关系，而是从属于自己。这并不意味着我们完全脱离了关系，而是我们开始拥有关系。
