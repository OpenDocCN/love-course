## 概述

## 站桩入境

**以下是站桩入境的步骤**：

1. **手劳工叠放一肚脐**：右手叠放于肚脐，左手叠放于右手手背。
2. **两脚开立，与肩同宽，脚尖冲前**。
3. **膝关节微弯曲，臀部下坐**：注意下坐，不要翘屁股。
4. **上身保持直立，成间坠走，腋下虚空，含胸拔背**：不挺胸，含胸拔背。
5. **收河长喉**：喉结拉到颈部，再往后坐，充分感觉脊柱拉伸。
6. **坐跨收河长喉**：站上5到10分钟，注意提纲提稿，缩阴收腹。
7. **呼吸时，身体尽量不要动，用意用气头往顶**：头往上顶，感觉有人拉到头顶，百会穴往上顶。

## 呼吸吐纳

**以下是呼吸吐纳的步骤**：

1. **通过呼吸吐纳，让自己慢慢兴奋起来**。
2. **在站桩入镜的过程中，让自己慢慢慢慢苏醒**。
3. **练习无火吐纳**：无火吐纳练完后，可以去练习易筋经。
4. **根部垂钓**：如果没有时间，早晨可以练这么多。

## 无火吐纳

**以下是无火吐纳的步骤**：

1. **双手扶腰**：双手扶腰，与站桩入境动作相似。
2. **双脚与肩同宽，一条直线上**：脚不正，骨盆就歪了。
3. **鼻腔震动**：吸气时，尽量不要抬头，身体腰胯做中摆式摆动。
4. **配合盆底肌发力**：PD肌发力，提睾完，缩阴收腹，配合鼻腔振动。

## 易筋经

**以下是易筋经的步骤**：

1. **盘龙探海**：双脚分开，吸气，外展，尽量把气提起来去伸展。
2. **弯弓射雕**：双脚分开，与肩同宽，双臂在体侧向外伸展，伸展时，尽量往外伸。
3. **其他动作**：根据易筋经课程内容，进行其他动作练习。
