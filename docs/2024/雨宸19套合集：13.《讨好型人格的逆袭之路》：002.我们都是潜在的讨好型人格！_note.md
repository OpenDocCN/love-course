# 雨宸19套合集：13.《讨好型人格的逆袭之路》：002.我们都是潜在的讨好型人格！

## 概述
在本节课中，我们将探讨什么是讨好型人格，以及为什么我们都是潜在的讨好型人格。

## 讨好型人格的定义
**概念**：讨好型人格是指一味讨好他人，忽视自己感受的人格。

**核心概念**：**讨好型人格 = 一味讨好他人 + 忽视自己感受**

## 为什么我们都是潜在的讨好型人格
1. **社会生存需求**：为了更好地生存，我们或多或少都会做出一些妥协。
2. **维护社交关系**：为了维护社交关系，我们可能会做出一些正常的行为。
3. **成长环境**：与成长环境有着密不可分的关系。

## 讨好型人格的特点
1. **恐惧感**：在与人交往时，尤其是出现冲突时，会感到恐惧。
2. **不真诚的答应**：虽然答应别人的要求，但内心并不真诚。
3. **不清楚自己的底线**：即使觉得为难，也无法拒绝别人的要求。

**核心概念**：**讨好型人格特点 = 恐惧感 + 不真诚的答应 + 不清楚自己的底线**

## 讨好型人格与正常社交行为的区别
1. **偶尔出现**：正常社交行为是偶尔出现，而讨好型人格是将这种行为变成常态。
2. **错误的行为模式**：讨好型人格是一种错误的行为模式，需要改变。

## 如何识别和改变讨好型人格
1. **认识错误的行为模式**：了解讨好型人格的特点和危害。
2. **自我认同**：认识到自己的价值，不要过度牺牲自己。
3. **寻求帮助**：如果需要，可以寻求心理医生的帮助。

## 总结
本节课中，我们一起学习了讨好型人格的定义、特点以及如何识别和改变。希望你能认识到自己的行为模式，并做出积极的改变。
