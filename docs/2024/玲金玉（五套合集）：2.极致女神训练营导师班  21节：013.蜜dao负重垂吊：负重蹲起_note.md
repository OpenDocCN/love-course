# 玲金玉（五套合集）：2.极致女神训练营导师班  21节：013.蜜dao负重垂吊：负重蹲起

🎼OK，接下来我们将开始负重练习，特别是带着蛋蛋的负重练习。今天的第一小节课是负重蹲起。

**负重蹲起**：
- 建议购买水晶球作为负重工具，因其美观且可逐渐增加重量。
- 新学员在连续训练7天后，特别是盆腔吐那呼吸训练后，可以开始负重练习。
- 从一个球开始，逐渐增加至两个或三个球。
- 在进行负重训练前，先进行身心合一的冥想、俯式呼吸和盆腔吐那静心抖动等热身活动。

**负重蹲起步骤**：
1. 将负重物体挂至膝盖下方位置。
2. 双脚比肩略宽，脚掌向前。
3. 在练习过程中，用手托住物体，避免其直接挂在身体上。
4. 使用呼吸维持盆腔堵那呼吸：鼻子吸气，鼻子吐气，舌头顶住上颚。
5. 吸气时起身，吐气时放松，同时进行阴道收紧和放松练习。
6. 每组15个深蹲，共三组。

**注意事项**：
- 膝盖面向正前方，蹲下时不要外扩。
- 可以尝试双手插腰或放在膝盖上，以增加着力感。

🎼坚持训练，相信自己的身体，完成三组负重蹲起。
