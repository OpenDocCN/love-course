# 课程03：超硬核沟通课：关于说话的一切 - 3.3 问题之锤：用最纯粹的方式打开世界 🔨

在本节课中，我们将要学习“问题”在思考和语言中的核心作用。我们将探讨问题如何成为认知的起点，以及如何通过提问来穿透我们思维中的盲区与成见，从而更有效地理解世界并推动个人成长。

---

前面我们提出了一个问题：深入思考的顺序应该是什么？我们得出的结论是从最基本的概念开始推演。但这个答案本身存在问题。因为当你开始思考“思考的问题是什么”时，你的思考过程就已经启动了。

因此，一切答案都始于问题。思考只能从问题开始。为了更清晰地阐述这一点，我们需要从一个更基本的概念进行推演。我们来关注一个更基本的问题：话语的基本逻辑结构是什么？

前面我们提到，在说理中，结构是**前提**和**结论**；在叙事中，结构是**起因**和**结果**。催生出这种结构的基本作用力是“提供”和“回答”。驱动说理的问题是“**这是为什么？**”，这是对“因”的疑惑。驱动叙事的问题是“**然后怎么了？**”，这是对“果”的追究。

有一个前提很容易被接受：我们复杂精妙的语言系统，最初都源于生存问题对我们的逼迫。当语言的信息传递功能满足了基本生存需求后，我们就需要语言的意义生成功能来回应更深层的存在问题。

我们的语言如此强大，既能极尽抽象议论之能，也能极尽具象写实之能。它能够构建前提之前的前提，挖掘经验之下的经验。我们所有的探索，无非是为了回答那几个难以回答、甚至永远无法被彻底回答的问题：**我是谁？世界是什么？心与物的关系是什么？这一切有什么意义？**

我们之前提到的“代码效率高”的概念组合之所以来自各个学科，是因为学科本质上就是一个特定“问题域”的答案集合。最能启发人的概念组合之所以经常来自哲学，是因为最原初、最基本的问题都是由哲学提出的，然后分派给各学科去研究。当遇到最深奥无解的问题时，人们依然会回归哲学，让哲学提出更基本的问题。

这就是我们这些“符号动物”面临的终极困境：**问题是语言发展的起点，但答案却不是语言发展的终点**，因为答案之后总会引出更多的问题。海德格尔曾精辟地指出：“**语言是存在的居所**”。这就是我们的生存处境：我们通过语言这个“居所”进入世界、存在于世。但这个居所并非结构清晰的建筑，而是一个首尾相连、无穷扩张的“怪圈”。

我们之所以能在这个怪圈中正常行事而不至于精神崩溃坠入虚无，要特别感谢各种**框架**和**成见**封印了我们的好奇心，也要感谢各种**信仰、传统和世俗教义**阻断了我们无尽的追问。

为什么要说这些呢？我只是在阐明一个浅显的道理：**那些让我们存活下来的东西，往往也是阻碍我们发展的东西**。例如习惯、情绪、肥胖的脂肪，还有我们的语言本身。妨碍我们发展语言组织能力的，正是我们日积月累习得的语言。而破除这种障碍的唯一方式，就是退回到语言被层层限定或过度发挥之前的状态，用一种更原始、更纯粹的方式来使用语言。没错，这种方式就是**提问**。

爱因斯坦曾说：“如果给我一个小时解决一道难题，我会花55分钟找出恰当的问题。因为一旦知道关键点在哪，只需5分钟就能解决问题。”

下面，让我们站在“问题”的视角，理解语言世界如何与心理世界相互作用，以及这种作用又如何进入现实世界，影响你我的生存处境。

前面我们提到了“特定问题域下经过思辨处理的抽象概念组合”。这里的“**问题域**”是一个关键词。我们可以将符号世界粗略地分为三个部分：**已知域**、**问题域**和**未知域**。在结构上，它可以对应我们心理世界的**舒适区**、**成长区**和**未知区**。某种意义上，进入问题域就是进入成长区。

如果此时我说“人通过解决问题获得成长”，那就落入了俗套。我们永远关心的是：问题是什么？它是如何产生的？我们应该如何解决？但在今天这个课题下，我们要问得更加微妙一些。

我们要问：**为什么提出一个正确的问题，能帮我们说清楚甚至想清楚一件事？为什么有时想清楚或说清楚一件事，本身就已经解决了一个问题？** 换句话说，我们要探究“问题”这个经验范畴，如何在我们的现实世界、心理世界和符号世界之间“流窜”，其作用原理是什么？它如何影响我们的生活？我们又如何通过解决它来获得成长？

为了说清这件事，我们暂时离开语言世界的弯弯绕绕，聚焦于心理世界和现实世界。

我们知道，心理世界的认知过程是“完形性”的。我们总是需要结合背景经验来处理对象。符合经验原型的，表现为**理解**；不符合的，表现为**问题**。而在现实世界，“问题”具体指那些构成生存障碍的事件。当生存无碍时，我们会觉得“没有问题”。

由此，两个世界的两种状态两两交叉，能获得四种认知状态，或者说四个经验范畴的标记：

以下是四种认知状态的详细说明：

1.  **理解**：一个经验对象既能被当下的心理框架容纳，又不在现实世界中表现为障碍。
2.  **问题**：一个经验对象与我们的已有解释框架产生冲突，并且妨碍了我们在现实世界中达成目标。**问题是我们明确知道的“不知道”**。
3.  **成见**：在框架之内，是我们习以为常的隐含假设和默认前提。我们用它来处理世界，但它本身可能导致我们遭遇各种障碍。例如，我们以为自己知道什么是真实、理解或知识，但我们并不真的知道。**成见是我们以为知道的“不知道”**。
4.  **盲区**：在框架之外。它与“问题”的区别在于，盲区并不直接妨碍我们的生活（注意，这里的“不妨碍”指的是我们没有感知到它的妨碍）。它就像经验疆域之外的巨大暗海，大陆的气候生态都与它有关，它以一种间接且宏观的方式影响着我们的命运。**盲区是我们以为“不是障碍”的障碍，是“不知道的不知道”**。

盯着这个框架，我们会有一个惊人的发现：**我们似乎只能在符号活动中，才能够揭示成见和盲区的存在**。在这个活动之前，它们是看不见的，否则也就不叫成见和盲区了。通过有意识的符号组织活动，我们把它拎出来、贴上标签，这个标签就映射出了我们心智中某些经验范畴。

这样的符号组织活动之所以可能，是因为组织者具有“**问题意识**”。

注意，下面这句话是重点：**问题意识之所以特别重要，是因为“问题”是中介。我们必须通过“问题”，才能够触达我们以为理解的不理解（即成见），和我们以为不是障碍的障碍（即盲区）**。对于由成见和盲区导致的问题，我们只能通过“问题”来反向追溯它们。问题是唯一的线索。因为在逻辑上，我们只能够通过“已知的未知”（问题），才能够触达“未知的已知”（成见）和“未知的未知”（盲区）。

我们必须在这个意义上，才能理解“人永远是通过问题成长的”这句话，也必须在这个意义上，才能理解为什么我们要拿起“问题之锤”。

我们的成长无非两条路径：

以下是两条成长路径的说明：

1.  **实践进路**：我们透过行动进入真实世界，在舒适区的边缘面对最真实的困境和障碍，然后在最真切的经验中问出最真实的问题。
2.  **思辨进路**：抽身而出，在符号世界中去归纳、推演，找到具体问题背后的那个抽象问题，小问题背后的那个大问题。只有这样，我们才能增加核心知识的“复用率”。

如果你认同前面的一系列假设和推演，会发现一个有意思的细节：成长过程的图示，并非我们所认为的“舒适区扩张”的过程，而是一个“问题域双向膨胀”的过程。通过问题域，已知域中的成见被挖掘出来，未知域中的盲区被辨认出来，它们都被包含进了一个巨大的“问题空间”中。

我们用“问题之锤”来强调问题意识的重要性，是为了创建一个相对陌生的心理标尺。拿着锤子看什么都是问题，问题背后一定还有问题。你可能会觉得这样活着太累，到处都是问题，永远解决不完，为什么要把人生和事业搞得这么复杂？

我分享一个切身体会：**我们真正的麻烦不是问题太多，而是问题太少**。

注意，下面这句也很关键：**在符号世界中成为“命题”的问题太少，就会导致在现实世界中成为“事实”的问题太多**。出于思维的惰性和心智的“最经济原则”，我们一定倾向于轻松生活。但有句话叫“你有你的计划，世界另有计划”。这不是你搞不搞复杂的问题，世界本来就是复杂的。各种问题一定会顺着我们的盲区、顺着我们的成见，以各种莫名其妙的方式刺入我们的生活。

我们必须直面这个巨大的问题域，直面作为一个会反思的物种必然会面临的“知道得越多，不知道的也越多”的思考窘境。但是，我们要时刻在心里放一把“**抽象之梯**”——这是我要给大家的另一个重要心理标记。

---

本节课中，我们一起学习了“问题”的核心地位。我们认识到，问题是思考和语言的起点，是连接已知与未知、穿透成见与盲区的中介。通过培养强烈的问题意识，拿起“问题之锤”，我们才能在实践与思辨的双重进路中，更有效地拓展认知边界，应对复杂世界，实现真正的成长。记住，我们真正的风险不在于问题太多，而在于对问题视而不见。
