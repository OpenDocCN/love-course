# 清风老师合集：13做自己的情感咨询师：情感咨询师培育计划24期：174.第十四课：自体心理学的应用

## 概述
在本节课中，我们将学习自体心理学在咨询中的应用，了解如何帮助来访者建立内聚的字体，以及共情在治疗过程中的重要性。

## 自体心理学的概念
自体心理学是20世纪由美国精神分析学会主席科胡特提出的重要观点，旨在解释和理解自体的形成，以及在人际交往中自体对个体心理健康的影响。

### 核心概念
**公式**：自体心理学 = 自体发展 + 人际交往 + 心理健康

## 自体障碍与治疗
自体障碍是由于自体在发展过程中没有形成内聚的字体，导致个体在现实生活中遇到各种问题。

### 核心概念
**公式**：自体障碍 = 内聚字体缺失 + 现实问题

## 自体心理学治疗
自体心理学治疗的目标是帮助来访者完善自体，将其转化为内聚的字体。

### 核心概念
**公式**：自体心理学治疗 = 完善自体 + 内聚字体

## 共情的作用
共情是自体心理学治疗中帮助来访者建立内聚字体的主要方式。

### 核心概念
**公式**：共情 = 沉浸体验 + 感受理解 + 积极情绪

## 咨询师的责任
咨询师在自体心理学治疗中承担着重要的责任，包括指出内在的字体发展、未被满足的需要，以及引导来访者进行自我觉察。

### 核心概念
**公式**：咨询师责任 = 指导发展 + 满足需求 + 自我觉察

## 总结
本节课中，我们学习了自体心理学在咨询中的应用，了解了共情在治疗过程中的重要性，以及咨询师在治疗中应承担的责任。希望这些知识能够帮助你在未来的咨询工作中更好地帮助来访者。
