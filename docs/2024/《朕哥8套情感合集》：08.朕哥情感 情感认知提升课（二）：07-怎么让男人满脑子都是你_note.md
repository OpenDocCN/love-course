# 《朕哥8套情感合集》课程08：朕哥情感认知提升课（二）07 - 怎么让男人满脑子都是你 💭

在本节课中，我们将要学习如何通过一系列心理学技巧，让一个男人在脑海中时常想起你。核心在于理解并运用几种关键的情感认知概念。

如果你能理解并应用接下来讲解的图示与原理，就能有效地占据一个男人的思维。我们先来介绍几个基础概念：**失去感**、**危机感**、**重心毛**、**独家回忆**、**陌生感**和**凸显价值**。

## 1. 制造失去感 🫳

人们通常在真正失去某物时，才会格外在意它。例如，一件漂亮的衣服，平时你不会总想着它，只有当你丢失它时，你的脑子里才会全是这件衣服。

男人和女人之间的感情也是如此。如果男人从未体验过可能失去你的感觉，从未想过你会离开，他就不会满脑子都是你。他会觉得你**可有可无**，因为他确信**不会失去你**。

因此，适时地让他感受到“可能失去你”是必要的。

## 2. 制造危机感 ⚠️

上一节我们介绍了失去感，本节中我们来看看危机感。危机感与失去感不同，它是指让男人觉得你**很抢手**，或者他存在许多**潜在竞争者**。

换句话说，要营造出一种氛围：如果他不要你，你瞬间就会被其他更优秀的男人接手。这种感觉能有效激发他的紧张和关注。

危机感相对的是安全感。很多女性会犯一个错误，就是给予男人过多的安全感。女性和男性的逻辑不同：女性通常需要安全感，安全感越多，爱意可能越深；但男性若感到**过于安全**，则可能觉得你离不开他，从而不再用心对你好。

所以，我们需要学会制造危机感，本质上是**抽离一部分安全感**，不让他觉得一切尽在掌握。请注意，这指的是**不给予过度的安全感**，而非对感情不专一。专一与制造危机感并不矛盾。

## 3. 学会“重心毛” 🪡

以下是关于“重心毛”的核心介绍。这个概念非常重要，指的是你有意识地为男人植入一些独特的心理印记或触发点。

例如，在我们的课程中专门讲到，当感情出现问题时，你曾种下的“心毛”会发挥巨大作用。它能帮助你们重温美好，比如回忆起曾一起去过的餐厅，重新勾起当初的快乐相处。

我们发现，有些女性即使恋爱三五年，也**没有种下任何“心毛”**，没有创造任何能让男人瞬间想起的独家记忆。当需要挽回或增进感情时，就找不到可以勾起对方情感的切入点。

因此，平时就要注重如何有意识地为男人“重心毛”。

## 4. 创造独家回忆 🎞️

接下来，我们谈谈如何创造独家回忆。这与“重心毛”略有不同：“重心毛”是**刻意为之**的心理锚点，而“独家回忆”则是**自然发生**的共同经历。

你要带他去体验一些他未曾经历，或你们共同未曾做过的事情。比如一起坐旋转木马、登山、玩滑梯、滑雪等。目标是创造出**专属于你们两人的共同记忆**。

让男人意识到，和你在一起时很快乐，你们拥有许多回忆，一起去过很多地方，看过不同的风景，经历过各种事情。这样，他会特别依赖你，他的脑子里会充满与你的回忆。即使你暂时离开，他也会不断想起这些片段。

我之前讲过，爱的一个基础是**习惯**。如果你们的爱里没有形成习惯，男人不习惯生活中有你，那么感情是存在很大问题的。独家回忆正是培养这种习惯和依赖的重要方式。

## 5. 营造陌生感 🎭

每次与男人见面，或隔一段时间不见，你都需要带给他一种**不一样的体验感**，营造一种陌生感。

不要让他觉得对你熟悉得像“左手摸右手”。如果你学会了营造陌生感，男人会觉得和你在一起充满新鲜感、生命力，以及对未来探索的欲望。

这样的话，他的脑子里就会时常期待和想象**下一次见到你的场景**。

## 6. 凸显自身价值 💎

最后，也是至关重要的一点，是学会凸显你的价值。你必须提高自己的“展示面”。

以下是提升展示面的具体方法：
*   **经营社交媒体**：精心打理你的朋友圈等社交平台。
*   **抓住展示机会**：利用各种场合和机会展示自己，不要羞涩或觉得不好意思。
*   **学习他人**：观察那些“名媛范”充足的女生，她们通常很擅长展示自己。
*   **巧妙展示高价值**：例如，可以通过分享你与朋友有趣、有深度的聊天记录（截图），来委婉地凸显你自己的**有趣**和**高价值**。给男人的感觉是：“我本身是一个很有趣、灵魂高级的人，只是你不懂如何与我相处罢了。”

凸显价值的方法有很多，但很多女性要么不会做，要么不想做，觉得没必要。你必须努力展示你的高价值和吸引力，让男人来追逐你。
