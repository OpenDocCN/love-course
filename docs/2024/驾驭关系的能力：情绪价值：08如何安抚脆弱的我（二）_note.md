# 驾驭关系的能力：情绪价值：08如何安抚脆弱的我（二）_note

在本节课中，我们将学习如何识别并安抚他人（以及自己）的脆弱部分。我们将深入探讨脆弱背后的心理机制，并通过具体的生活实例，掌握通过共情与换位思考来有效安抚他人的方法。

## 发现与记录脆弱点

上一节我们介绍了脆弱点的概念，本节中我们来看看如何具体操作。首先，我们需要去发现并记录这些脆弱点。记录对象不仅包括他人，也包括自己。

以下是记录时需要注意的要点：
*   记录他人的脆弱点。
*   同时记录自己的脆弱点，避免只关注他人而忽略自我成长。

## 理解脆弱：意识与潜意识

去发现这些之后，我们要做到的就是安抚。那么，如何安抚？我们需要先理解什么叫脆弱。

在心理学，特别是精神分析法中，常提到“意识”和“潜意识”。
*   **意识**：指我们能够觉察到的心理活动。公式表示为：**意识 = 可被觉察的思维/情绪**。例如，“我知道我正在发脾气”就是一种意识。
*   **潜意识**：指我们未能觉察到的心理活动。公式表示为：**潜意识 = 未被觉察的思维/情绪**。例如，有时我们“不知道自己在发脾气”，但行为上却在发怒，这时的怒气就存在于潜意识中。

脆弱的点，也可以理解为“情绪按钮”。每个人都有多个情绪按钮（即脆弱点），一旦被触碰，就容易引发强烈的情绪反应或心理防御。

## 脆弱背后的核心：内心创伤

在脆弱的这个部分，背后往往体现着我们内心的创伤。例如“孤独感”这种脆弱，可能源于成长过程中的伤害。

举例来说，童年时因学习不好被老师罚站门外，可能会产生“被世界抛弃”的孤独感，这是一种童年创伤。

再以婚姻为例，假设丈夫回家后如坐针毡。这可能并非因为他人，而是因为家庭氛围——孩子的白眼、妻子的责骂、父母的叹息——让他感到被亲人排斥，从而激发了深层的孤独感。

换位思考，如果你处于这种被排斥的境地，也会感到孤独。此时，他最需要的是有人能理解他，帮他说句话。如果此时外界有人给予关怀，他很容易产生依赖和逃离的冲动。

## 安抚的关键：共情与换位思考

我们想疗愈别人，就要去体会别人内心的这份感受。这需要两种核心能力：**共情能力**和**换位思考能力**。

*   **共情**：指感受他人感受的能力。
*   **换位思考**：指站在对方角度理解问题的能力。

这些是情绪管理和关系处理的基础能力，需要进行系统训练。情绪价值是建立在这些基础之上的更高维度的能力。

## 安抚实战：针对“孤独感”的话术示例

我们为什么要去体会对方？因为只有设身处地地体会之后，才能有效地进行安抚。

假设你的丈夫因感到家庭压抑而孤独，你可以这样进行安抚：

> “老公，我能看得出来你在家里边其实感觉到挺压抑的。这里边肯定有我们的问题。之前我跟你吵了太多次，造成了孩子、我，还有爸妈，都在无形中给你压力。我也想过，如果我是你，自己的孩子对自己有意见，父母不断提要求，老婆不断控制，那我进这个家门也会有很大压力。所以我现在很能理解你现在的处境。如果你感觉真的很压抑，也可以出去走一走。但我现在能做的是，我尽可能不再像以前那样哭闹解决问题了。我也会找机会跟孩子好好解释，我知道你是爱孩子的，是我之前的解释工作没做好。事情因我而起，就由我来解释吧。我希望通过我们的调整，你回到家能感到更轻松，不再像今天这样孤立无援。我希望把话说开，我们一起往前推进。我会尽最大努力做好我能做的事。如果你觉得家里气氛越来越好，我非常希望并欢迎你每天都能早一点回家陪我们。”

这段话术做到了：
*   **尊重**对方感受。
*   **允许**对方有情绪和空间。
*   **准确描述**对方内心的感受。

对方未必立即转变，但能感受到你的真诚和理解，会觉得你是一个通情达理、值得信赖的人。这就是通过共情能力去疗愈对方，让对方感到被允许、被理解、被安抚。

## 安抚实战：针对“控制感”（不被尊重）的话术示例

我们再举一个例子，以便大家有更深的体会。

例如，丈夫在接到妻子“你在哪儿？几点回？”的电话时容易暴怒。这背后隐藏的脆弱点可能是“感到被控制”或“不被尊重”。

学习后，你的思维方式应变：从“他气死我了”转变为“我触发了他哪个情绪按钮？”。通过分析，你可能发现这是他童年时不被父母尊重的创伤在成年关系中的重现。

找到脆弱点后，便可对症下药进行安抚。等他回家后，可以这样沟通：

> “老公，今天给你打电话，你劈头盖脸吼了我一顿。你是不是觉得我打电话是在控制你，觉得我不尊重你、让你没有自由，所以才发火的呀？……我猜也是。这事肯定有我的责任，也许我当时的语气引起了你的这种看法，我首先向你道歉。但我真的没有控制你的意思，我只是想问你今天几点回来，因为我做了红烧肉想给你留点。可能是我过去的习惯，或者近期的吵架，让你对我有了防卫之心，这很正常。但我想坦诚地告诉你，最起码今天我没这个意思。我后面也会注意。我知道每个人都不希望被控制，包括你控制我，我也不愿意。所以我能够理解你当时的心情，这也是我今天没生你气的原因。我不是可以接受你吼我，而是我能看到你当时的心理变化。当然，如果你就是故意想气我，那我肯定在意。但我相信你不是故意的，从心理学角度看，可能是你潜意识里一下子觉得被控制，自己也没意识到就发了脾气。我现在越学习，越能理解你的一些表现。我也有暴躁、不尊重你的时候。我今天想谈的是，告诉你我今天没那个意思。同时，我也希望你能为吼我这件事道个歉。当然，我不强迫你。”

通过共情，把他的内心感受表达出来并表示理解，会让对方觉得你通情达理，从而可能主动道歉。

## 核心策略：记录、规避、安抚与保护

想驾驭关系，不要害怕对方出现情绪。关键在于：

以下是处理脆弱点的核心策略：
1.  **记录**：智慧的人懂得收集对方脆弱的点。
2.  **规避**：最重要的是有意识地避开这些脆弱点。例如，他不喜欢被控制，我们就不控制；他爱面子，我们就不当众批评。
3.  **安抚**：当脆弱点不小心被触发时，我们要能够进行有效安抚。
4.  **保护**：当别人无意中踩到他的脆弱点时，你要能站出来为他圆场、保护他。例如，当朋友开玩笑触及他介意的事，你可以立刻站出来维护他。

如果你能做到**记录脆弱点、规避脆弱点、安抚脆弱点、保护脆弱点**，你就会成为非常懂对方的人。对方会感到与你相处舒适，愿意与你亲近，并且舍不得离开你，因为你既懂得如何让他舒适，又懂得如何保护他。
