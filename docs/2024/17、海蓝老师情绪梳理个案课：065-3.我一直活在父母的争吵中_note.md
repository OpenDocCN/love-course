# 海蓝老师情绪梳理个案课：065-3.我一直活在父母的争吵中 🧘‍♀️

## 概述
在本节课中，我们将探讨个案中个体如何长期生活在父母争吵的环境中，以及这种环境对其情绪和价值观的影响。

## 第一部分：父母争吵的日常
以下是父母争吵的日常情况：

- **妈妈和爸爸的争吵**：妈妈痛苦，爸爸坚持，争吵持续了一段时间。
- **妈妈的反抗**：妈妈的反抗剧烈，可能到了受伤的程度。
- **爸爸的变化**：爸爸在争吵中表现出与平时不同的强势和不考虑他人的一面。

## 第二部分：价值观的混乱
以下是价值观混乱的描述：

- **爸爸的价值观**：白天善良、不强势、考虑他人，晚上变得强势、不考虑他人。
- **妈妈的价值观**：白天强势，晚上变得无能为力。
- **个体的感受**：个体对爸爸和妈妈的变化感到恐惧和混乱。

## 第三部分：恐惧和不确定性
以下是恐惧和不确定性的描述：

- **恐惧来源**：个体对爸爸和妈妈的变化感到恐惧，担心爸爸会像对待妈妈一样对待自己。
- **恐惧程度**：个体对爸爸和妈妈的变化感到恐惧的程度差不多，但爸爸的变化可能稍微多一点。

## 总结
本节课中，我们一起学习了个体如何长期生活在父母争吵的环境中，以及这种环境对其情绪和价值观的影响。通过了解这些，我们可以更好地理解个体的心理状态，并为其提供相应的帮助。
