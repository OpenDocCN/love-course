# 清风老师合集：01解读孙子兵法智慧8期：007.第五课：作战篇 上

大家好，欢迎来到清风带你解读孙子兵法智慧。😊 在之前的课程中，我已经带领大家解读了孙子兵法中的开篇，即最重要的G篇。接下来，我们将学习的是作战篇。

很多人认为孙子兵法就是关于战争和冲突的，但实际上，孙子兵法最强调的是“不战而屈人之兵”，即在不战的情况下就能使敌人屈服。这意味着，如果可能，我们尽量避免战争，但如果我们的利益受到侵犯，我们必须坚决反击。在敌人有巨大利益可以被我们夺取时，我们要迅速行动，权衡利弊。

作战篇将融入我们与伴侣的互动过程，通过作战篇的智慧，让我们在现实生活中拥有更好的行动力和解决问题的能力。

### 原文解读

孙子曰：“凡用兵之法，驰车千四，割车千乘，士十万人，千里馈粮，则内外之费，宾客之用，车甲之奉，日费千金，然后十万之师举矣。”

这段话的意思是，孙子说：“用兵打仗的方法，需要有千辆驰车，千乘割车，十万士兵，千里之外运送粮食。这样，内外费用、宾客开支、车甲供应，每天耗费千金，才能动员十万大军。”

### 核心思想

孙子在这里告诉我们，战争动员的过程是劳民伤财的，使用战争这把利器风险极大。

### 生活启示

在现实生活中，很多宝宝在撒娇成功后，喜欢频繁向对方索要礼物，这是错误的。撒娇得到对方的付出，其实就像一次博弈，这种博弈的核心是“先胜后战”。在G篇中我们已经讲过，我们要有足够的积累，才能开口索要。如果没有积累，开口只会让对方感到尴尬，甚至破坏感情基础。

### 不战而屈人之兵

孙子兵法中的核心思想之一是“不战而屈人之兵”，即我们可以战斗，但必须确保胜利。在分离二加一的情况下，如果无法让对方付出足够的代价，彻底屈服，那么问题仍然会再次出现。

### 等待时机

孙子在原文中提到：“夫兵久则国利者谓之游也。” 这句话的意思是，战争持续时间过长，国家却兴旺发达，这是不可能的。孙子提醒我们，在不能取得结果之前，要耐心等待，积极提升自我。

### 主动出击

孙子在原文中提到：“故兵贵神速。” 这句话的意思是，用兵打仗，贵在迅速。在现实生活中，无论是与伴侣的互动，还是职场竞争，都要主动出击，但在此之前，要做好充分的准备。

### 总结

本节课中，我们一起学习了孙子兵法作战篇的核心思想，包括不战而屈人之兵、等待时机、主动出击等。希望这些智慧能够帮助我们在现实生活中取得更好的结果。我们下节课再见。😊
