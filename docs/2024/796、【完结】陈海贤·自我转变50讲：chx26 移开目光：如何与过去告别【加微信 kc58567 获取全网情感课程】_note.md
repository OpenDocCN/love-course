# 课程26：移开目光 🧭 如何与过去告别

在本节课中，我们将学习如何与过去告别，从而为新的未来腾出空间。告别过去是自我转变的关键一步，但往往也是最困难的一步。我们将探讨三个核心思路，帮助你从过去的牵绊中解脱出来，面向新的开始。

上节课我们探讨了为何告别过去如此艰难，核心在于过去承载着我们的失去、伤痛和难以割舍的自我。正是这些牵绊，让我们难以开启新的篇章。

本节中，我们来看看如何主动“移开目光”，从过去转向未来。这不仅是简单的注意力转移，更是为新的可能性创造心理空间。

以下是帮助你与过去告别的三个具体思路。

## 1. 把结束当开始 🚀

转变期是新旧自我的边界。站在这个节点上，我们常常只看到确定的失去，却忽略了它同时也是新未来的起点。过去的失去是确定的，但新的未来充满不确定性。用“开始”的视角替代“结束”的视角，能让我们更好地面对未来。

心理学家埃斯特·佩雷尔在处理夫妻出轨危机时，会这样说：
> “无论是好是坏，从出轨这件事被发现开始，你们上一段的婚姻实际上已经结束了。现在你们要一起来决定是不是要开始一段新的婚姻。”

这句话揭示了一个关键节点：**结束即开始**。当你面临转变危机时，可以告诉自己：
> “无论过去是好是坏，从某一刻起，过去的某件事、某段关系、某个状态已经结束了。现在，你要如何选择和决定新的未来？”

如此思考，你会发现结束从来不只是终点。例如：
*   你可以说“我结束了一份工作”，也可以说“我开始了新的职业探索”。
*   你可以说“我结束了一段婚姻”，也可以说“我开始经营一个人的生活”。

正如哲学家刘擎所言：“每一天都是余生里的第一天。”每一个结束，都是一个新开始。既然是新开始，你希望如何度过它？

有时我们不愿告别，是不想失去过去的自己。例如，一位放弃梦想转行成功的男演员，虽获财富却深感背叛了过去的自己。应对梦想结束的最好方式，是**重新树立一个新的梦想**。这不是背叛，而是与过去的和解。把结束当开始，能让我们看到面向未来的出路。

## 2. 与自我和解 🤝

关于过去的遗憾，常让我们陷入后悔与自责，其背后隐藏着一个不现实的假设：**“如果我当时更明智，就能避免错误。”** 这种幻觉让我们更难告别过去。

例如，一位从舒适高管职位跳槽互联网大厂的学员，因不适应新环境而长期后悔当初选择。在转变中，后悔是危险的，它引诱你逃回过去，却无法带你走向未来。

我与他进行了这样的对话：
*   **问**：当初为什么想出来？
*   **答**：那时年轻，不想被限制，想到外面闯荡。
*   **问**：后面的经历有价值吗？
*   **答**：有，我成长了很多，阅历也丰富了。
*   **问**：如果没有后面的经历，你还在原公司，现在会怎么想？
*   **答**：我可能还是会不甘心，会想出来。

所以，现在的后悔不代表你做了错误的选择，只代表**那时你没有足够的切身经验来理解这个选择**。你现在的看法（觉得第一份工作好）本身，就是后续经历带来的，是成长的一部分。

就像我们觉得童年美好却不想回到孩童状态一样，我们只能接受经验带来的改变。这不是错误，是成长必然的经历。如此思考，便能放下过去，与自我和解。

## 3. 重新整理过去 📦

没有过去，就没有历史；但只有过去，就没有将来。过去是我们的来处，是自我重要的一部分。告别过去，也需要重新整理它。

例如，那位与原生家庭分离的男生，他面临一个难题：若要保留与父母有关的美好记忆，就要承认父母的重要性；若删除这些记忆，就会失去一部分过去的自己。

这时需要明白：**保留过去的美好记忆，不代表你要改变对他们的看法或原谅他们。它只代表人和关系都会变。** 从分离开始，所有事（包括历史）都只与你有关了。这是**你的记忆**，不是他们的记忆。

既然是你的过去，你就有权利用自己想要的方式来整理它。整理过去的目的，是为了**从关系中解脱，独自理解自己是谁**，并面向未来。

对于放不下过去的来访者，我常使用这个比喻：
> “我知道失去很痛苦，所以你宁可停留在过去。这就像你遭遇灾害，原来的房子变成了废墟。你当然怀念过去的家，但你也需要去看看废墟里，还有什么东西是你能用的。也许，它能成为你自我重建的基础。”

本节课中，我们一起学习了面对过去、开启未来的三条思路：**把结束当开始**、**与自我和解**、**重新整理过去**。你和过去是怎样的关系？你认为哪种思路更适合你？

然而，有时我们不愿放下过去，是因为不知道如何面对未来的不确定性。自我转变的下一环节，我们将讨论如何面对转变中的不确定，如何面对未来。我们下节课再见。
