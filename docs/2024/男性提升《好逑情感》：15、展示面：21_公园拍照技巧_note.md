# 男性提升《好逑情感》：15、展示面：21_公园拍照技巧

🎼如何在公园拍摄出独特感，以下提供4个简单拍照技巧。🎼

### 1. 公园草坪拍摄技巧

- 将镜头翻转，使用草坪作为前景，人物作为背景，增强意境感。
- **公式**：前景（草坪）+ 背景（人物）= 独特意境

### 2. 公园长椅拍摄技巧

- 将机位放在左侧，人物移动到最右侧，蹲下来仰拍，抬头看向天空。
- **公式**：仰拍（天空）+ 长椅（左侧）= 高级感

### 3. 公园树林拍摄技巧

- 避免让兄弟后退镜头砍人物的腿部，注意构图，调整曝光，避免画面单调。
- **公式**：构图 + 曝光调整 = 避免单调

### 4. 公园栏杆拍摄技巧

- 让兄弟走到正面背对着他仰拍，淡入天空，手刹兜仰头，增加高级感。
- **公式**：仰拍（天空）+ 手刹兜仰头 = 高级感
