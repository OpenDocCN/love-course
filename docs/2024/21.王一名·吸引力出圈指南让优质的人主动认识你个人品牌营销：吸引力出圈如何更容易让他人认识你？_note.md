# 21. 王一名·吸引力出圈指南：如何让优质人主动认识你？个人品牌营销

欢迎来到吸引力出圈指南。我是王一鸣。之前的课程，我们探讨了吸引力的秘密及其重要性。它是人与人之间拉开距离的核心关键。我们也学习了如何找准自己的优势。在传播自己的优势之前，首先要确定自己的优势是什么。通过四个关键词来协助我们确定优势：成就、热爱、擅长和学习。当你通过这四个关键词确定好自己的优势之后，接下来就进入两大步骤中的第二个步骤：展示和传播你的优势。

在展示和传播之前，首先我们要给自己创建一个载体，也就是如何展示你的优势。比如抖音、视频号、小红书等常规主流的三个平台，基本上就足够了。

### 起一个好名字的重要性

第一步是先有一个简单又容易传播的账号名字，也就是你的昵称。这一节我们就讲一下如何起一个自带流量的好名字。

**定位时代，你唯一能做的最重要的营销决策就是给产品起名字**。这句话中用了两个关键词，唯一和最重要，来强调一个好名字对于品牌打造的重要性。

在实际辅导学员的过程中，我们发现有的人不重视个人品牌的名字，起得很随便；有的人很重视，但是不知道该怎么取。其实一个人的名字本身就是一个品牌，而好的名字就是一个钩子，会把你想要连接的人吸引过来。

### 如何起一个自带流量的好名字

以下是从四个方式给你详细讲解：

1. **突出人物**：比如“海阔天空”、“章鱼小丸子”、“披荆斩棘的我”、“阳光正好”等，看到这样的名字，你的第一感受是什么？你会知道这个名字背后是一个什么样的人吗？你愿意认识一下他们吗？

2. **突出优势**：比如“喜茶”、“汉堡王”、“瑞幸咖啡”等，光看名字，你大概就能知道他是做什么的。所以说能反映品牌定位的名字，有很多都是好名字。

3. **简单好辨认**：如果一个名字简单好辨认，就更好传播，有了传播，我们才能吸引更多的潜在粉丝。

4. **保持一致**：在不同平台的名字要尽可能的保持一致。

### 总结

好的名字一定要简单和辨认，容易传播。好的名字都是为了让更多的目标人群更好的靠近你，了解你，从而给你带来更多的机会和更多的好运气。
