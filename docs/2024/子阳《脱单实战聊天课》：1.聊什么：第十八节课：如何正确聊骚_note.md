# 子阳《脱单实战聊天课》：1.聊什么：第十八节课：如何正确聊骚

## 概述
在本节课中，我们将学习如何正确地进行聊骚，包括如何引导和暗示，以及如何处理对方的反应。

## 聊骚技巧

以下是聊骚的一些关键技巧：

### 1. 引导和暗示
- **公式**：通过提问或描述情境，引导对方进行联想。
- **代码**：`"你今天能喝冰的吗？"` 或 `"那你有没有兴趣跟我一起看个月亮啊？"`

### 2. 避免直接
- **公式**：不要直接表达你的意图，而是通过暗示让对方产生联想。
- **代码**：`"我喝多了有可能会做一些坏事哦，你能你还能做什么坏事呢？"`

### 3. 处理反应
- **公式**：如果对方反应不好，可以及时收尾。
- **代码**：`"今天晚上没有月亮，那我们就不去看月亮了。"`

### 4. 一拉一扯
- **公式**：在聊天的过程中，通过一拉一扯的方式，增加情绪波动。
- **代码**：`"我说的是我新买的普天鞋子，对不对？怎么你一个死渣你，你脑子里就只缠我的身子嘛。"`

## 总结
本节课中我们一起学习了如何正确地进行聊骚，包括引导和暗示、避免直接表达、处理反应以及一拉一扯等技巧。希望这些技巧能够帮助你在聊天中更加得心应手。
