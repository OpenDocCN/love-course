# 雨宸19套合集：02.《好好吵架，好好爱》💑：005. 好好吵架经验篇：先处理情绪，后掰扯道理

在本节课中，我们将学习如何有效地处理亲密关系中的争吵。核心原则是：**先处理情绪，再平衡差异**。我们将探讨男女情绪的不同特点，并提供具体的方法来缓和冲突，从而将争吵转化为加深理解的契机。

## 争吵的本质：情绪与事理的比例

上一节我们介绍了课程主题，本节中我们来看看争吵的本质构成。

你需要首先知道，争吵中**90%是情绪，10%是事儿**。吵架的原则是**先处理情绪，再平衡差异**。很多人特别喜欢讲道理。

尤其在对方需要的不是道理，而是被关注、被安慰的时候，还跟对方讲道理，效果会很差。在情绪正强烈的情况下，人是很难听进任何道理的。

如果你发现讲道理行不通，对方不听，那一定是你方法有问题。

## 理解男女情绪的差异性

了解了争吵中情绪的主导地位后，我们来看看男女处理情绪的天然差异。

你要知道男女情绪的一个特点。男性的情绪是**内向化**的，需要独立的空间去消化情绪。这就是为什么很多情侣吵架时，男生不愿说话，他需要一段时间去消化情绪。遇到事情，男生喜欢一个人待着，比如喝闷酒、在车里待着或抽根烟。

女生的情绪是**外化**的，需要发泄，需要说出来，需要被理解和安慰。女生有情绪的时候，会找闺蜜、找朋友倾诉。

你要学会理解这两种思维的差异性。女人一直在讲感受，男人一直在讲事儿。

吵架后的情绪差表现为：吵完架之后，男人往往气都消了，女人往往更生气。情侣间最常见的情况是，男生跟没事人一样，女生还在气头上，而且越想越气。

你必须要抓住男女情绪特点的不同。如果你不在意这些特点，没有抓住重点，没有好好了解伴侣的性别特征，恋爱就谈不好。

## 如何正确处理情绪

认识到差异后，我们来看看针对不同性别，该如何正确应对情绪。

重点是我们该如何正确地去处理这个情绪。

对于男人来说，男人往往更快恢复理性。因此，不妨让他先发泄完情绪，再去表达自己的想法。男人的情绪是内向化的，即自己消化。你等他消化完之后，再跟他沟通就好了。先让他表达，先安抚情绪。

**公式：应对男性情绪 = 给予空间 + 等待消化 + 事后沟通**

跟女人吵架，要先安抚完情绪再讲道理，不要上去就讲道理，没有用。你把情绪安抚住，说什么对方才能听进去。如果情绪安抚不住，你那不叫讲道理。

对于女人来说，要给对方独立的空间去消化情绪，不要抓着对方不放，逼对方继续争吵。也不要把对方的回避解读为不爱。女人有时候生闷气，想自己静静，男生可能受不了，觉得“你凭什么要安静，一定是不爱我了”，从而纠缠对方。其实对方只是想安静一会儿。

对于两个人来说，不要把对方的情绪归咎到自己身上，不要有负罪感。不要认为“对方生气都是我的错”。共情和倾听，是我对你的付出和负责，而不是因为我承认自己错了。

## 讲道理前的关键步骤：表达爱

处理完情绪，准备沟通事理时，有一个至关重要的前置步骤。

讲道理之前，先表达爱。有一句话说得很好：一个人只有在感觉不被爱的时候，脾气才会变得很差；人只有在感觉安稳和被爱的时候，才会是温柔的。

以下是表达爱的示例方式：

*   “我觉得你特别没有责任心，什么事都我管。其实我挺想要帮你的，我可以成为你的依靠。”
*   “我非常需要你，我觉得你很好。但你需要我的时候，能跟我说吗？你从来都不说，所以我也不知道。我以为你能行，但你跟我说过之后，我就会帮你。”
*   “我以为你是一个很独立的人，但我没想到你也有脆弱的时候。”

你要记住，听话一定要听对方在表达情绪时，真正想表达的是什么。不要只听他的表面感受，要听他表达什么。当对方说“我觉得你没责任心”时，背后往往是焦虑，需要的是安慰。

## 学习缓和氛围的小技巧

道理讲通了，关系还需要润滑。以下是一些能快速缓和争吵氛围的实用技巧。

学习一些缓和氛围的小技巧很有帮助。

以下是几个有效的小技巧：

*   **幽默与微笑**：在气氛紧张时，尝试笑呵呵地去说，不要过于认真。女生往往更看重这种感受。
*   **肢体接触**：给对方一个拥抱或亲吻。女人在跟你吵架时，你给个吻、给个抱抱，比讲什么都重要。即使她把你推开了，但你拥有想去拥抱她的想法和行动，也会让女人内心感到踏实。
*   **行动服务**：主动做一些对方希望你做的事，比如洗衣服、打扫卫生等。

当对方在发泄情绪时，如果你用一种充满爱意的感受去回应，她会觉得很踏实、很舒服。

## 课程总结 🎯

本节课中，我们一起学习了如何“好好吵架”。

总结一下这堂课：大部分的争吵，只是我们用来跟对方加强联系、缓解内心不安的一种方式。尤其是在恋爱阶段，两个人没有太多利益相关时，最在乎的就是“你是否在意我”。

当你主动去关心对方，关注对方的情绪以及情绪背后的情感需求时，很多问题就迎刃而解了。

分手的本质，往往是双方都觉得对方不爱自己了。“我觉得你不爱我，我也觉得你不爱我，既然都不爱，那就分开吧。”但这真的是因为不爱了吗？还是我们表达爱的方式有问题？

通过先处理情绪，后掰扯道理，我们不仅能解决当下的冲突，更能让彼此感受到爱意，让关系更加牢固。
