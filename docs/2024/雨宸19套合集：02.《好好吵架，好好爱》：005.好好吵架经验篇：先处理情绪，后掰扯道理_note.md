# 雨宸19套合集：02.《好好吵架，好好爱》：005.好好吵架经验篇：先处理情绪，后掰扯道理

## 概述
在本节课中，我们将学习如何处理争吵中的情绪，以及如何通过正确的方法来解决问题。

## 核心概念
**争吵中情绪占比**：90%情绪，10%是事儿。

**吵架原则**：先处理情绪，再平衡差异。

## 情绪处理
以下是处理情绪的步骤：

1. **识别情绪**：了解争吵中情绪的占比，认识到情绪的重要性。
2. **理解男女情绪特点**：
    - **男性**：内向化，需要独立空间消化情绪。
    - **女性**：外化，需要发泄和被理解。
3. **处理情绪**：
    - **男性**：给予独立空间，让其发泄情绪。
    - **女性**：给予安慰和理解，避免过度纠缠。

## 沟通技巧
以下是沟通的技巧：

1. **表达爱**：在讲道理之前，先表达对对方的爱。
2. **倾听**：认真倾听对方的表达，理解其情绪和需求。
3. **缓和氛围**：使用一些小技巧，如笑、拥抱等，缓解紧张气氛。

## 总结
本节课中，我们一起学习了如何处理争吵中的情绪，以及如何通过正确的方法来解决问题。记住，争吵只是加强联系、缓解内心不安的一种方式，关键在于关注对方的情绪和需求。
