# 《全套课程合集》：leon 3.0：14.开场白

Hello，这里是第十四节课。本节课我们将讲解开场白。在开场白中，许多兄弟们会遇到一个误区，那就是在开场时，你可能会想太多，比如第一句话该说什么。但想得越多，首先会增加你的行动成本，导致你错失一些搭讪机会。其次，想得越多，会让你在开场过程中能量下降，反而无法吸引女生的注意力。你可能会想出一句话，但女生已经离开了。因为开场只有一个最核心的目的，那就是获得女生的注意力。而你想得越多，其实你越在压抑自己的能量，这反而会导致你在表达出来后适得其反，女生反而不会给你她的注意力。所以开场只有一个目的，那就是获得注意。

最好的方式就是把你想到的第一句话大大方方地表达出来，不要过多地思考。接下来，开场白共有三种方式，分别是直接开场、间接开场和推拉开场。

### 直接开场

直接开场适合新手朋友，因为你在开场的过程中经常容易遇到问题。问题是你在开场时没有很好地表达清楚你的意图，反而导致女生离开了。所以对于新手朋友，你不需要想太多，直接去用，直接看就可以了。例如：“我觉得你很漂亮，可以过来认识一下你吗？”或者“你单身吗？我觉得我怎么样？我想跟你加个微信。”

直接开场适用于所有情景和时间，无论是白天、晚上、夜店还是喝咖啡的新手都可以。但是，如果你是一个中阶选手，那么直接开场可能就不那么适合你了。因为在这个过程中，你的状态是要去认识这个女生，所以你的姿态是低于她的。如果你一直以这种姿态去保持互动，会让你在后续的关系中处于追逐女生的状态，会让你聊得很累。所以如果你想要成为一个好的gamer，那么在刚开始的时候就要做好这个姿态的建设。至少你们是平等的，最好是你们是高于她的。

间接开场是以非男对女的前提去打开对话，可以问一个路，可以问女生一个建议，可以以社交需求为开场。我自己比较喜欢用的是以社交需求为开场去做间接开场。例如：“Hello，我刚刚在那边看到你，想过来跟你打招呼。今天是我这个月第一次出门，工作挺忙的，出来透透气。我觉得打扮挺符合审美的，我觉得过来跟你讲话，应该是能够满足我想要的需求。同时我也希望能够给你打一些情绪价值。”然后你可以从间接转到直接，因为在刚刚那段10到15秒的表达过程中，你可以抓住女生身上某个点进行复歌，例如：“哎，你的眼睛还挺漂亮的。”或者“哎，所以你现在在干嘛？”

推拉开场在保证男对女的前提下，多了一个否定的元素，可以否定自己，也可以否定女生。例如：“本来我是不准备过来给你搭讪的，因为挺累的，今天工作一天的。但你刚刚那个笑容觉得非常治愈，然后还是决定拖着疲惫的身躯过来给你打个招呼啊。”或者“本来我不准备过来认识你的。因为我觉得你的这个比如说身高啊，或者说你的外形啊，不是我喜欢的类型。但是啊再找到一个点去复合他，可以是她的这个笑容，可以是他刚刚跟闺蜜聊天的一个语言的内容，也可以是他给你这样一个感觉，或者说她的某一个行为都有亮点。”

这是三种不同的开场元素。所以你如果是新手，就直接用直接开场。如果是中阶到高阶，建议你可以开始练推拉和间接开场。
