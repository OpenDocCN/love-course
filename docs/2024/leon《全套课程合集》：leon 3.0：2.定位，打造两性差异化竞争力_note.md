# 《leon全套课程合集》：leon 3.0：2.定位，打造两性差异化竞争力

## 概述
在本节课中，我们将学习如何通过定位和打造人设来提升两性关系中的竞争力。

## 定位的重要性
定位是特劳特在商业中提出的一个概念，用于帮助企业在竞争中找到自己的差异化优势。在情感领域，定位同样重要，它可以帮助我们梳理自身优势，建立与异性的联系感。

## 竞争与价值匹配
无论是情感还是商业，我们都在竞争中。在情感中，我们需要明确自己的价值，并锁定与之匹配的对象。

## 目标群体与竞争对手
锁定目标群体后，分析竞争对手的优势，明确自己的差异化竞争点。

## 塑造人设
在人设塑造上，我们需要关注以下9个方面：
1. 年龄
2. 职业
3. 性格
4. 原生家庭
5. 情感经验
6. 社交生活
7. 痛苦矛盾
8. 思想价值观
9. 人生追求

## 例子
以下是一些塑造人设的例子：
1. 年龄：适当包容自己的年龄，让女生觉得你更成熟。
2. 职业：包装自己的职业，提升自己的社会地位。
3. 性格：总结自己的优势，用故事的形式展现出来。
4. 原生家庭：适当包装自己的家庭背景，展现自己的优势。
5. 情感经验：不要说前女友的坏话，展现自己的成熟。
6. 痛苦矛盾：将痛苦转化为成长，展现自己的坚强。
7. 思想价值观：展现自己的价值观和人生追求。
