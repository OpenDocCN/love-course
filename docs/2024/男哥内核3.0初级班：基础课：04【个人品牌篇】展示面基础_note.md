# 男哥内核3.0初级班：基础课：04【个人品牌篇】展示面基础

## 概述
在本节课中，我们将学习如何构建个人品牌，特别是展示面的基础。我们将探讨如何通过照片和配文来展示自己的个性和价值。

## 执行力的重要性
**公式**：执行力 = 知识 + 行动
- 执行力是成功的关键，它需要将知识转化为行动。
- 许多人在学习知识后没有付诸实践，导致无法取得成果。

## 展示面基础
### 照片布局
- **公式**：照片布局 = 美观 + 个性化
- 照片布局应注重美观和个性化，以吸引他人注意。

### 配文
- **公式**：配文 = 简洁 + 有趣 + 有价值
- 配文应简洁、有趣且有价值，以增强照片的吸引力。

### 示例分析
- **示例**：一位消防员学员使用单反拍摄工作照片，展示了他的职业属性。
- **分析**：照片展示了学员的职业，配文应强调其专业性和责任感。

### 配文技巧
- **技巧**：使用马南微博中的短句，以增加配文的吸引力。
- **注意事项**：避免使用过于负面或极端的配文，以免给人留下负面印象。

### 反差感
- **概念**：反差感是指通过对比来增强吸引力。
- **示例**：一位学员在朋友圈中分享了一张正装照片，与之前的浮夸风格形成对比。

### 钓鱼配文
- **概念**：钓鱼配文是指容易引起共鸣并引发回复的配文。
- **示例**：一张狗狗照片的配文：“这只狗在等我回家，你呢？”

### 作业
- 熟悉马南微博，并尝试将其中的短句用于配文。
- 分析自己的照片，看看哪些可以改进。

## 总结
本节课介绍了展示面基础，包括照片布局、配文技巧和反差感。通过学习这些内容，你可以提升自己的个人品牌，吸引更多人关注。
