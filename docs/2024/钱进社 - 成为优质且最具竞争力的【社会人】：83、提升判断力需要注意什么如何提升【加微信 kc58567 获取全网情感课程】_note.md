# 钱进社 - 成为优质且最具竞争力的【社会人】：83、提升判断力需要注意什么如何提升

## 概述
在本节课中，我们将探讨如何提升判断力，包括需要注意的事项和提升方法。

## 提升判断力的关键点

### 1. 按步骤行事
**公式**：判断力提升 = 按步骤行事
- 避免东一榔头西一锄头的做法，确保判断力提升的系统性和连贯性。

### 2. 逻辑和认知
**公式**：判断力 = 逻辑 + 认知 + 行事顺序
- 判断力与智商不直接挂钩，而是与逻辑、认知和行事顺序相关。

### 3. 正确的思维方式
**公式**：正确判断 = 正确思维方式
- 避免错误的思维方式，如追求非黑即白的答案。

### 4. 判断流程
**公式**：判断流程 = 发现问题 + 搜集信息 + 确定对策 + 调整 + 决定
- 通过一系列步骤进行判断，确保判断的准确性和全面性。

### 5. 聚焦流程而非结果
**公式**：正确判断 = 聚焦流程而非结果
- 关注判断过程而非结果，以便优化方案和改善对策。

## 提升判断力的步骤

### 1. 发现问题
- 多角度看待问题，避免片面性。

### 2. 搜集信息
- 收集信息并佐证假设，避免道听途说。

### 3. 确定对策
- 锁定范围进行比较挑选，最终具体化。

### 4. 调整
- 面对突发状况和阻碍，进行必要的调整。

### 5. 决定
- 用匹配的方式传递最终决定，坚定信念。

## 注意事项

### 1. 不要只看表面问题
- 深入思考问题的本质，避免表象判断。

### 2. 不要忽视前提条件
- 明确前提条件，提前做好准备。

### 3. 短期判断与长期判断
- 考虑短期和长期因素，避免一步到位。

### 4. 寻找其他选项
- 避免只有一个选项的情况，扩大判断范围。

### 5. 从小事做起
- 通过日常生活中的小事锻炼判断力。

## 总结
本节课中，我们学习了如何提升判断力，包括关键点、步骤和注意事项。通过刻意练习和不断积累，我们可以提高自己的判断力，成为更具竞争力的人。
