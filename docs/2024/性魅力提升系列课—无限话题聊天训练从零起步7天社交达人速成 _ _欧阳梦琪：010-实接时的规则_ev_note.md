2. **避免寻求认可**
   - **公式**：从他人认可转变为自我认可，建立内在自信。
   - **代码**：肯定自己的价值，不依赖他人给予的认可。

3. **避免选择不方便的见面地点**
   - **公式**：选择方便的地点，构建主场优势。
   - **代码**：规划路线，控制成本，避免额外开销。

4. **避免所有选择都询问对方**
   - **公式**：展现主见和领导力。
   - **代码**：做出选择，引导对方，而非依赖对方。

5. **避免时阶结束后过度纠缠**
   - **公式**：保持果断，避免给对方留下低价值印象。
   - **代码**：避免询问过多后续问题，保持轻松氛围。

以下是5个需要重点去做的事情：

   - **公式**：进入最佳状态，放松心情。
   - **代码**：听音乐放松，避免焦虑和紧张。

3. **选择人少的地方坐下来**
   - **公式**：营造私密社交氛围，深入交流。
   - **代码**：选择人少的地方，避免公共空间带来的尴尬。

   - **公式**：提高成功率，观察对方情绪。
   - **代码**：在对方情绪高时提出升级或要求。
