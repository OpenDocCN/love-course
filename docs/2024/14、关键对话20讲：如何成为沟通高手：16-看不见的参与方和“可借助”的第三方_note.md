# 14、关键对话20讲：如何成为沟通高手：16-看不见的参与方和“可借助”的第三方

🎼大家好，欢迎来到关键对话，如何成为沟通高手的课堂。我是史新月。今天是我们这门课的第十六讲，我们将探讨关键对话中的多方对话。

🎼许多关键对话涉及多方，我们需要更广阔的视角和更强的控制力来掌握这类对话。在第七课中，我们学习了对手识别，第八课则讲述了作为组织的对手识别。今天这一讲可以看作是对手识别的进阶课程。

🎼对手识别不仅是一对一的识别，还包括在更多对手和对话参与者中识别每个参与方的利益和需求，从而更好地处理多方对话。有人认为多方对话并不难，只需将对话的基本能力和技术应用于每个一对一的关系中即可。但这种观点可能只知其一，不知其二。

🎼多方对话离不开我们已经讲过的那些基本能力，但其复杂程度并非简单的叠加。两方对话只是一组关系，一旦增加一方，三方对话就是三组关系，四方对话则是六组关系。关系越多，越难以把握。

🎼何况还有那些未到场的利益相关者和看不见的参与方。如果我们没有看到他们的存在，而他们又可能对关键对话产生重要影响，我们又如何运用我们的各种能力呢？这堂课的目的就是要建立一个新的认知。

🎼多方对话是一个更加复杂的对话场景，不是简单的一对一对话的叠加。我们更要树立一个意识，那就是看不见的参与方。只有脑子里有看不见的参与方这根弦，我们才称得上是进入高阶的关键对话者。

🎼今天这堂课，希望你能掌握多方谈话的几个基本场景，深入了解那些没有参与到谈话现场的参与者，也就是我们常说的看不见的参与人。最后，希望你能对常见的影响对话的参与人的类型有比较熟练的把握。

🎼这门课之后，当你进入一场关键对话，出现多方参与时，你知道如何应对。即便有未到场的，你也能清楚地知道谁没来。在对话中，时刻考虑到那些没有出现的人的态度和利益。

🎼一对一是常见的对话场景，但多方的对话也并不少见。例如，你去租房时，很多时候是和中介、房东一起谈的，这就是一个三方的对话场景。中介并不属于你的团队，也不属于房东的团队，他有他独立的利益诉求，所以这不是一个团队谈判，这是一个多方谈判。

🎼多方对话的场景识别和对手识别你首先得做对了。我曾经陪一个外国客户去租一个四合院，他想租个四合院做他公司的办公地址。我们和房东以及中介见了面，这里面有4个人，但是三方。我和租客这个外国客户是一个团队，因为在这个场景里，我并没有独立于我客户的利益。

🎼中介自称是agent，但外国客户问他你是谁的agent，你是我的agent呀，你还是房东的agent。这一问把中介给问晕了，他解释了半天也没说明白。问题就出在他自己介绍自己的时候，用了一个错误的词儿，就是agent。

🎼房屋中介其实是broker，他是居间人，他说自己是agent，他误导了别人，也绕晕了自己。在这个场景里，我才更像是agent，因为我在代表客户在谈，客户是本人，我是他的代理人。

🎼像中介这样的居间角色并不完全等同于代理。所以当一方有代理人时，并不是多方的对话。但是如果有中介或居间角色在其中，那么这个场景就是一个多方的对话。大家搞明白这两种的区别。

🎼多方对话有两个基本场景：一个是全体对话，另一个是分别对话。全体对话就是所有人都出席的对话都到场。例如，一个公司要出售，这公司有三个股东，三个股东一起来和买家谈，这就是都到场的全体对话。

🎼分别对话可以发生在任意一对参与者之间或部分参与者之间。我们看到国际上开大会，无论是联合国还是气候大会，总会有立场相似的国家坐一起，先协调立场。这就是分别对话。分别对话的场景有两种：有规则安排的分别对话和私下的分别对话。

🎼规则安排的分别对话是根据已经定好的流程，有一些只有部分参与方参加的对话，这是规则就这么定的。例如，三个卖方股东说，我们三个自己去商量一下。

🎼这是对方买方知道的，卖方要去商量，他们举行了一场单独的分别对话。但是如果一个卖方私下去联络了买方，在另外两个卖方不知情的情况下沟通了一回，那这就是私下的分别对话。

🎼有些情况下，私下的分别对话是被禁止的，最明显的场景就是投标招投标的过程中是禁止投标者之间对话的，所有对话都发生在发标方和投标方之间。投标者之间的对话那叫串标，这是不允许的。

🎼一旦出现了多方的对话，博弈的情况就会变得更加复杂。有优势的那一方，有能力制定规则的那一方，就会利用自己这个优势创造一些对自己有利的博弈机制。

🎼接下来，我们讲一讲看不见的参与方。为什么会有看不见的参与方呢？走进一个对话现场的参与者总是有限的。无论是他们跟这件事的关系所限，有直接关系的来了，只有间接关系的没有来，或者说整个会场能够容纳的人数有限，参与者是有一定范围的。

但是对话的内容到底和谁的利益相关，并不会因为会场的人数有限，或者说有的相关方没有来到现场，你就直接能够把他们排除在外。

🎼比如兄弟姐妹几个人在商量自己家的一些事儿，可能涉及到财产的分配，老人的赡养等问题。那他们作为兄弟姐妹，当然是坐在一起的相关方，但是他们各自的配偶、男女朋友、子女可能都会跟这些安排发生一些联系，产生一些利益上的关系。

但是这些人未必到了现场，或者说他们正在商量赡养老人的事儿，但是老人可能因为正在养老院或者正在医院里治疗，他也没有到现场。

但是呢他们商量的却是跟这个老人切身相关的事情。所以这也是一个很重要的相关方没有到现场。🎼没有到场的相关方在很多场合下是比较明显的。比如我们说啊安排老人的事情或者孩子的事情啊，老人孩子本人没在。

这个我们都知道，他不在，但是最终他接受不接受，这也是一个非常重要的因素是吧？是本人接受不接受，我们要在对话里考虑到。

🎼我们能够意识到谁没来以及没来的这个人的利益，是因为我们对这个谈话的结构和谈话的文化背景十分熟悉，所以我们能够明确准确地意识到，哎，不在场的那些相关方都有谁。

🎼当我们换一个文化环境，尤其是说走进了一个不熟悉的国家，不熟悉的地方，有另外一套法律体系，人民的生活习惯、思维方式也不同的时候，这些看不见的相关方就没有那么明显。

这个时候啊就需要开启我们那个6.0的雷达。在一个不熟悉的环境里，发现那些没有出现的参与者。

🎼没有到场的参与方不见得不重要，有的时候恰恰是没到场的人能够决定这场对话，最终是否能够成功能否可行。有的时候不到场的人往往有最后的决定权，我们可以回顾一个案例啊，大家都知道。

瑞典过去有一个非常有名的汽车叫萨博，这个萨博呢是一个高性能的跑车，很多人都喜欢对于萨博的技术啊，和萨博的车型啊，很多车迷都津津乐道。

那后来萨博呢陷入了生产经营的困难，好几家中国车企都和萨博接触过，像他抛出过橄榄枝。比如说北汽就买了萨博的一些技术，但是并没有把萨博整个买过来，吉利汽车也跟萨博有过接触，最接近把萨博整个购买过来的是当时的庞大汽车联合了青年汽车，两家共同收购萨博。

整个交易推进的深度啊接近于完成。但这个时候没想到的是，一个没有参加这场对话的第三方站出来了，它就是通用汽车。

🎼那通用汽车为什么在这里有发言权呢？因为萨博的很多技术啊，包括它的生产平台都是由通用许可给萨博使用的。也就是说那不是萨博的自有技术。萨博是在通用的平台上做出了他自己品牌的汽车，通用站出来表示不同意这场收购。

如果庞大和青年的这个联合体收购了萨博，通用是不会再继续许可这些关键技术了。🎼那为什么呢？因为通用在中国也有非常重要的合资产，也有它重要的市场利益是吧？如果萨博这个广受人们喜爱的品牌被中国企业收购了。

那对通用现有的市场和现有的合资公司可能会造成一些冲击啊，这是最明显最表面的一些原因，也是当年媒体报道。那通用在这个时候就是那个最关键的没有出现在谈判桌上的谈判方，他一出手，这个几乎谈定的交易就死了。

🎼所以在复杂的关键对话中啊，我们必须要用到在第五课学到的啊对复杂事物的理解能力。那节课我们讲了基辛格啊，他用广角镜头去看整个对话这个全局有哪些相关方，还有哪些相关方呢可以影响这场对话。

包括还有哪些其他的力量可以影响这场对话。我们就要像基辛格做的那样，既要看得到大的全景，哪些是支持的力量，哪些是阻碍的力量，同时呢又努力把阻碍的力量转化成支持的力量。

首先我们要用广角镜头观察到，然后呢再通过利益的调整和利益的交换，满足各方的需求，把反对力量变成支持的力量。🎼这样的能力啊，既是对复杂事物理解能力的体现，又是外交能力的具体体现。

所以在这里啊我要提醒各位同学，到了一个我们不熟悉的环境里，要特别注意看不见的参与方，这个不熟悉的环境啊，也不见得是另外一个国家，另外一个文化，他有可能是你不熟悉的行业，有可能是你从学校到了职场，也有可能呢你是在职场工作多年。

现在又回学校去做学生工作，比如招聘，那你回去了又是一个什么环境，这里面有哪些你看不见的相关方，有哪些还没有发言的，但是跟他们的利益有重大关系的人，我们应该啊做一些前期的研究。

用全景的广角的镜头去看这个途径。

🎼我们来看看啊，常见的那些不在场的参与方，或者叫看不见的参与方，都有谁呢？我们可以从大类上捋一捋，以后遇到不熟悉的领域，不熟悉的场合，你在分析看不见的参与方的时候呢。

可以从这几个大的框架上去找一找。🎼首先啊就是有权利管理的人，比如像政府。有权利发放许可或者决定不许可的单位。

🎼比如啊我是做并购交易的律师，那我们每涉及到一个新的国家或者地区的并购交易的时候，很重要的一个前期工作就是要看一看在这个并购中有哪些政府许可我们需要拿到才能让这个交易生效。

然后我们要评估一下这些许可。我们拿到它的可行性。🎼同时呢我们还要把这个交易啊放到一个特定的行业领域里去看。因为不同的行业，它的许可的要求肯定是不一样的。

在不同的行业里，又有那个行业里特定的看不见的相关方。比如你开一个连锁的餐厅和你开一个发电的电厂。

那他所涉及的许可和他许可背后那些并不会到你会议室来的相关方肯定是不一样的吧。虽然他们不来，他们不参与你交易的直接的谈话。但是你把材料报上去的时候，这个交易如果被否决了，你前面的工作都是白做的。

所以要提前把这些事情啊都研究好安排好。🎼权力许可的部门单位啊，这些看不见的参与方有可能也不是政府啊，他就是有这权利。

🎼比如我我想去学校里做个活动，我和学生会，我和组织者都安排好了。结果学校说有突发情况。这外面过来的人啊，活动讲座交流暂时都不能办。

那这个学校他虽然不是政府，但是校园这块地儿就归他管。那他也是一个非常重要的看不见的参与方。🎼那么第二类看不见的参与方呢。

我们可以管它叫舆论。这个舆论具体说啊，包括媒体啊，传统媒体啊、电视报纸也包括很多的新媒体啊，微博、朋友圈、公众号、视频网站、B站、短视频，他们影响力都非常大。

那舆论呢也包括周围人的意见。我们不把它放到一个全民都能阅读的媒体的层面啊，就是你要做一个事儿，放到你们公司里，放到你班级里，放到你学校或者放到你们家去讨论，大家是不是支持这些人会不会给你造成巨大的压力或者阻碍，让这件事办不成。
