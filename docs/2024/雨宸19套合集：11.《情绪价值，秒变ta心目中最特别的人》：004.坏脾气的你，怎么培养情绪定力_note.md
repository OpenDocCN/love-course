# 雨宸19套合集：11.《情绪价值，秒变ta心目中最特别的人》：004.坏脾气的你，怎么培养情绪定力

## 概述
在本节课中，我们将探讨如何培养情绪定力，帮助你管理自己的情绪，从而为他人带来价值。

## 培养情绪定力的方法

### 1. 正确认识情绪
每种情绪的出现都有其原因，负面情绪反映了你内心中最软弱、最无助的地方。

**公式**： 负面情绪 = 内心软弱 + 无助

### 2. 坏脾气的来源
坏脾气与习得性无助有关，即习惯性崩溃和情绪化。

**公式**： 坏脾气 = 习得性无助

### 3. 习得性无助的形成
习得性无助是一种消极被动行为，受环境影响很大。例如，小时候负面情绪不被认可和接纳，导致长大后无法表达情绪，最终崩溃。

**公式**： 习得性无助 = 消极被动行为 + 环境影响

### 4. 如何打破习得性无助
1. **改变情绪归因模式**： 不要将所有情绪归因于自己，学会外归因。
2. **释放情绪**： 找一个特殊的途径释放负面情绪，如听音乐、打游戏等。
3. **奖励机制**： 每次成功对抗情绪，都要给自己鼓励。

**公式**： 打破习得性无助 = 改变情绪归因模式 + 释放情绪 + 奖励机制

### 5. 接纳不完美的自己
负面情绪是内在潜意识和软肋的自我暴露，要学会接纳自己的不完美。

**公式**： 接纳不完美的自己 = 接纳负面情绪 + 认识到不完美是正常的

## 总结
本节课中，我们学习了如何培养情绪定力。通过正确认识情绪、改变情绪归因模式、释放情绪、接纳不完美的自己等方法，我们可以更好地管理自己的情绪，为他人带来价值。
