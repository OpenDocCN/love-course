# 清风老师合集：07关系修复，破镜重圆11期：078.第七课：挽回核心技术（下）让他回到你身边

## 概述
在本节课中，我们将学习如何通过正确的方式，使用三种核心技术——危机术、助攻术和诱惑术，让对方主动挽回你，回到你身边。

## 危机术
危机术的核心在于通过虚拟的危机刺激对方主动挽回。以下是危机术的三个层次：

**浅层次**：
- 展示朋友圈，例如：“又被同事告白了，不知道该怎么拒绝，好烦恼啊。”
- 这种方式告诉对方，除了你以外，还有其他异性对你感兴趣。

**中层**：
- 通过朋友传达信息，例如：“我最近有一个朋友喜欢谁谁谁，你们现在到底怎么样了，是不是真的分手了？”
- 这种方式给对方施加危机，促使对方想要重新稳定关系。

**深层**：
- 直接对话，例如：“我有一个朋友最近在追求我，我不知道该怎么样拒绝。”
- 这种方式让对方表现出负面情绪，如吃醋，从而促使对方主动告白。

## 助攻术
助攻术通过朋友制造助攻接触的机会。使用条件是在怀念期之后，避免在排斥期使用。

**使用建议**：
- 环境选择：KTV、聚餐、酒吧等场合。
- 利用酒水让男性放松，增加安全感。
- 朋友正常聊天，让对方产生错觉，觉得你们没有分手。
- 触发损失厌恶机制，促使对方在聚会结束后想要告白。

## 诱惑术
诱惑术的本质是放松，让对方想要和你更多接触交流，深入你的生活。

**要点**：
- 通过最有魅力的那一面，调动对方的欲望。
- 三招技巧：
  1. **氛围营造**：香水、灯光、着装。
  2. **眼神交流**：运用害羞的眼神，练习眼神走曲线。
  3. **肢体接触**：调整衣领、整理衣物，制造肢体触碰。

## 总结
本节课中，我们学习了如何使用危机术、助攻术和诱惑术，让对方主动挽回你。希望这些技巧能帮助你挽回心爱的人。
