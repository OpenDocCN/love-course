## 社交能力

社交能力又分为社交内向和社交外向。

社交内向的女人，社交圈往往会很小，小到只有家人、亲戚、同学、同事、闺蜜，她们很少有机会和他人进行互动，所以交际能力会比较弱。

**公式**：社交内向 = 社交圈小 + 交际能力弱

**公式**：社交内向 = 不知所措 + 思想开放

因为交际圈小的缘故，他们有更多的时间花在书籍，报刊网络上，他们的思想也可以是开放的，他们其实也可以是段子手。

**公式**：社交内向 = 读书 + 报刊网络 + 段子手

对于理性的女人也是如此。

**公式**：社交内向 = 理性

思想上他未必是保守的。

**公式**：社交内向 = 思想开放

所以你会碰到这样一种情况。

**公式**：社交内向 = 情况

**公式**：社交内向 = 正常现象

不必大惊小怪。

**公式**：社交内向 = 不必大惊小怪

在感情方面，他们会无私的奉献他们的所有，包括时间、精力、金钱、身体。

**公式**：社交内向 = 无私奉献

但这并不代表他们是爱情里的傻子。

**公式**：社交内向 ≠ 爱情傻子

他们会通过直接或间接的方式要求或是暗示，你也对他做出付出或是改变。

**公式**：社交内向 = 要求或暗示

一旦他们付出的越来越多，也就会陷得越来越深。

**公式**：社交内向 = 付出越多，陷得越深

情感的破裂，往往会让他们窒息，需要很长一段时间才能缓过来，甚至会做出异常的举动，比如威胁对方轻生人格分裂等等。

**公式**：社交内向 = 情感破裂 + 异常举动

那么社交能力的另一部分就是社交外向。
