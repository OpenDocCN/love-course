# 雨宸19套合集：02.《好好吵架，好好爱》：004.人间疑惑之：为什么有的人一吵架就像世界大战？

## 概述
在本节课中，我们将探讨为什么有些人一吵架就像世界大战，分析其背后的心理原因和应对策略。

## 为什么会像世界大战？

以下是导致人们吵架时像世界大战的原因：

### 1. 灾难化思维
- **定义**：将小事扩大化，导致极度焦虑。
- **例子**：因为对方没有及时回复消息，就认为对方不再爱自己。

### 2. 强迫性思维
- **定义**：强迫对方与自己保持一致，期望过高。
- **例子**：要求对方必须按照自己的意愿行事，否则就感到愤怒。

### 3. 内心安全感缺失
- **定义**：小时候没有得到足够的关爱和认同，导致内心缺乏安全感。
- **例子**：因为对方没有及时回复消息，就认为对方不再爱自己。

### 4. 过于自我
- **定义**：以自我为中心，忽视他人的感受。
- **例子**：认为自己的感受最重要，而忽略对方的感受。

### 5. 缺乏平等关系经验
- **定义**：在原生家庭中，没有建立起平等的关系，导致在亲密关系中缺乏平等意识。
- **例子**：认为对方应该无条件地满足自己的需求。

## 应对策略

### 1. 降低期待
- **定义**：不要对对方期望过高，接受对方的不足。
- **例子**：不要期望对方能够完全满足自己的需求，接受对方的缺点。

### 2. 设置底线
- **定义**：明确自己的底线，学会尊重对方。
- **例子**：不要让对方做出违背自己底线的事情。

### 3. 提高行为目的的察觉
- **定义**：在生气时，要清楚自己的目的，并思考对方是否能够接受。
- **例子**：在吵架时，要明确自己的目的，并思考对方是否能够接受自己的方式。

### 4. 好好吵架
- **定义**：在吵架时，要保持冷静，尊重对方，寻求解决问题的方法。
- **例子**：在吵架时，不要使用攻击性语言，而是尝试理解对方的观点。

## 总结

本节课中，我们学习了为什么有些人一吵架就像世界大战，以及应对策略。希望这些内容能够帮助大家更好地处理亲密关系中的矛盾和冲突。
