# 马克搭讪教程 398：如何与明星陈卓璇搭讪 🎬

在本节课中，我们将学习马克在视频中演示的与明星陈卓璇搭讪的具体开场白和互动技巧。我们将拆解他的语言和非语言沟通策略，帮助你理解如何开启一段对话。



## 概述

本节教程基于马克的一个搭讪实例。我们将分析他使用的问候语、表情和简单的互动策略，这些都是开启对话的基础步骤。



## 开场白与问候

上一节我们介绍了课程的整体目标，本节中我们来看看具体的开场白。一个友好且自然的开场是成功搭讪的关键。

以下是马克使用的核心问候语组合：

*   **Hello, hi, hello**：重复且简单的问候，能快速吸引对方注意并建立初步连接。
*   **What‘s up**：一句随意的寒暄，旨在让对话氛围变得轻松。
*   **😊**：使用微笑表情符号，代表在实际交流中应保持友好、开朗的面部表情，这是传递善意的重要非语言信号。



## 互动推进策略

在打完招呼之后，需要一些简单的策略来推进互动，避免对话戛然而止。

以下是马克演示的两种推进方法：

*   **出て**：这个词意为“出来”，在情境中可理解为邀请对方“出来聊聊”或“走到前面来”，是一种主动创造互动空间的提议。
*   **领的。还有一个。到了**：这些短语表明了一种带领和分享的过程。可以理解为“我领路。这边还有一个（地方/东西）。我们到了。”，这展示了通过带领和介绍周围环境来延续对话的自然方式。



## 核心技巧总结

本节课中我们一起学习了与明星陈卓璇搭讪的实例分析。核心技巧在于：**使用多层友好问候（Hello, hi, what‘s up）结合积极非语言信号（如微笑😊）来破冰**，随后通过**主动提议（“出来”）和带领分享（“领的，还有一个，到了”）** 来自然地推进互动，使对话得以延续。

关键是将这些简单的元素流畅地组合起来，形成一次完整的互动开场。
