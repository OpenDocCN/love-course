第二个错误是**主动示好、讨好过度**，这源于不自信和迷失自我。

**关键点**：主动权必须掌握在自己手里。你可以选择对你好，也可以选择收回。要让她明白，你的行为是出于自我选择，而非卑微的讨好。

这一点与第一点一脉相承。**你的行为本身也是一种间接表白**。始终保持“自我”状态，坚信自己并不差。在两性关系中，可以假设“她喜欢我，但她不说”。这个思维框架能让你在互动中占据心理优势。
