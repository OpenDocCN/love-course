# 陈哥私教《高情商培训3.0》：03 2023私教3.0——突围 第二节：认知吸引

## 概述
在本节课中，我们将学习如何通过认知吸引来提升个人魅力，从而在人际交往中更加得心应手。

## 认知吸引的核心概念
认知吸引是指通过特定的行为和沟通方式，激发对方的好奇心和兴趣，从而建立吸引力的过程。

### 错误的心理和行为
以下是几种常见的错误心理和行为，它们可能会阻碍你吸引他人：

1. **过早暴露目的性**：过早地表达自己的意图，如表白，会让人失去兴趣。
2. **过度讨好**：过度讨好对方，会降低自己的价值感。
3. **行为低级**：缺乏情商，如强迫对方做不愿意做的事情。
4. **缺乏主见**：为了取悦对方而放弃自己的观点和感受。
5. **毫无幽默感**：过于正经，缺乏轻松愉快的氛围。
6. **表现软弱**：在交往中表现出软弱，会让人失去信心。

### 正确的做法
以下是一些提升认知吸引的方法：

1. **保持神秘感**：不要过早地暴露自己的意图，让对方保持好奇心。
2. **自信大方**：保持自信，不要过度讨好对方。
3. **展现个性**：保持自己的观点和感受，不要为了取悦对方而放弃自己。
4. **幽默风趣**：在交往中保持轻松愉快的氛围。
5. **敢于挑战**：敢于表达自己的观点，敢于挑战自己的恐惧。

## 总结
本节课中，我们学习了认知吸引的核心概念和提升方法。通过避免错误的心理和行为，并采取正确的做法，我们可以提升自己的吸引力，从而在人际交往中更加得心应手。
