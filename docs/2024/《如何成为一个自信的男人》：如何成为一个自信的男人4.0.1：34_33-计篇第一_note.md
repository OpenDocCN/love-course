# 《如何成为一个自信的男人》课程4.0.1：34_33-计篇第一 🧠⚔️

在本节课中，我们将要学习如何将《孙子兵法》的核心思维运用到两性关系中。我们将重点解读“计篇第一”，理解“先计后战”与“攻其不备，出其不意”这两个核心原则，并探讨它们如何指导我们在情感博弈中占据主动，建立健康的长期合作关系。

## 课程概述与结构

上一节我们介绍了本课程第五板块的核心目标：将博弈的最高境界——合作，应用于两性关系。本节中，我们来看看实现这一目标的具体方法论。

《孙子兵法》全文六千余字，共十三篇，分为五大板块。本课程将聚焦于前八个章节的核心思维，因为它们对情感关系的指导意义最为直接。后面的章节主要涉及军事地理和特殊战法，在此不做解读。

本课程共八节课，每节课解读一个章节。本节课，我们首先学习“计篇第一”。

## 核心思维一：先计后战 📝

孙子兵法在本篇开宗明义，强调了谋划的重要性。其第一个核心思维是“先计后战”，即在行动前必须进行周密的研究和筹划。

原文有言：“兵者，国之大事，死生之地，存亡之道，不可不察也。” 意思是，战争是国家的大事，关系到生死存亡，不能不进行详尽的考察。这告诉我们，在感情中采取重大行动前，也必须深思熟虑。

那么，具体从哪些方面进行“计”呢？孙子提出了“五事七计”的框架。

以下是“五事”的具体内容：
*   **道**：指道义、民心。
*   **天**：指天时、气候等时机条件。
*   **地**：指地利、环境。
*   **将**：指将领的才能。
*   **法**：指法规、制度。

以下是“七计”的具体内容，用于对比敌我双方：
*   **主孰有道**：哪一方君主更正义？
*   **将孰有能**：哪一方将领更有才能？
*   **天地孰得**：哪一方占据更多天时地利？
*   **法令孰行**：哪一方法令能切实执行？
*   **兵众孰强**：哪一方兵力更强大？
*   **士卒孰练**：哪一方士兵更训练有素？
*   **赏罚孰明**：哪一方赏罚更公正严明？

原文总结道：“多算胜，少算不胜，而况于无算乎！” 公式可以概括为：**胜利概率 ∝ 筹划的周密程度**。筹划越周密，胜利条件越多，则胜算越大；反之则小；毫无筹划则必败。

## 核心思维二：攻其不备，出其不意 🎭

在充分谋划的基础上，孙子提出了第二个核心思维：“攻其不备，出其不意”。这是取得优势的关键战术。

原文指出：“兵者，诡道也。……攻其无备，出其不意。此兵家之胜，不可先传也。” 意思是，用兵之道在于诡诈多变。要在敌人没有防备的地方进攻，在他意料不到的时候行动。这是取胜的奥秘，必须灵活运用，无法事先规定。

这强调了策略的灵活性和隐蔽性。在代码逻辑上，这类似于一个条件判断：`if (target.isUnprepared() && action.isUnexpected()) { return advantage; }`。

## 在两性关系中的运用 💑

理解了以上两个核心思维后，我们来看看它们如何具体指导我们的情感实践。

**第一，应用“先计后战”：追求“诛心为上”。**
追求异性如同用兵，消耗巨大（时间、金钱、情感）。因此必须“先计后战”，最好能通过沟通让对方产生好感甚至倾心，再考虑见面。这样见面后关系推进顺利，成本极低。然而，许多人的做法恰恰相反：急于见面、吃饭送礼，投入大量资源却可能毫无进展。问题根源在于没有谋划，急切暴露需求，导致对方利用你的急切心理引导你不断付出，使你陷入被动。所以，从长期关系角度看，核心应是 **`追求精神共鸣 > 追求物理推进`**。

**第二，应用“诡道”：避免无效“坦白局”。**
许多人在情感博弈中吃亏，是因为误信“沟通万能论”，试图通过完全坦露内心来解决问题。这就像兵法中所说：“故能而示之不能，用而示之不用”。现实中，当你坦诚一切时，对方往往不会对等回应，反而可能利用你的坦诚来拿捏你。有效的策略不是单方面坦白，而是保持一定的神秘感和策略性，引导关系向有利方向发展。

**第三，应用“利而诱之”：懂得给予与获取。**
孙子曰：“利而诱之，乱而取之。” 要想在博弈中让对方跟随你的节奏，占据高位，你必须懂得“给利”和“取利”。然而现实中，很多男性容易走向两个极端：要么只想索取不愿付出；要么盲目付出后不知如何争取应有回报，被动等待。正确的做法是，付出时要有策略、有目的，并在合适的时机，以自然的方式获取关系的进展或对方的承诺，形成良性互动。

## 总结

本节课中，我们一起学习了《孙子兵法·计篇第一》的两个核心思维：“先计后战”与“攻其不备，出其不意”。

*   **“先计后战”** 教导我们，在情感中行动前需像战略家一样周密筹划（五事七计），评估形势，追求“诛心为上”，而非盲目投入。
*   **“攻其不备，出其不意”** 则提醒我们，关系推进需要策略和灵活性，避免直白暴露所有意图，要善于创造时机和优势。

将这些原则应用于两性关系，能帮助我们从一开始就建立健康、互利的互动模式，避免陷入被动付出的困境，从而更快地导向真诚的合作关系。下一节课，我们将进入“作战篇第二”，学习如何高效管理我们的“情感资源”。
