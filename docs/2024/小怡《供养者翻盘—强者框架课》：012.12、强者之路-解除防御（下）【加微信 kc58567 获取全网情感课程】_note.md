### 1. 欲望的力量

上节课提到，男人的松弛状态对女人有吸引力。但许多人表示，即使知道这一点，也难以做到。

#### 欲望的偏见

许多人在看到喜欢的女生时，会因为害怕拒绝而犹豫不决。这种犹豫源于对自身欲望的偏见。

**公式**：欲望 = 欣赏 + 欲望

**代码**：欲望 = like + want

#### 大胆表达

记住，任何人都不需要为自己的社交愿望感到羞耻。大胆地表达欣赏，自信地传递你的喜欢，这并不会让你显得猥琐。

### 2. 表白的技巧

有些人在表白时，会采取一些有退路的方式，例如酒后表白或愚人节表白。这些行为在现实中是大忌。

#### 驾驭欲望

驾驭欲望的核心是不要被结果绑架。记住，你只负责表达，不负责结果。

**公式**：表达 = 欣赏 + 欲望

**代码**：expression = appreciation + desire

### 3. 允许结果

允许你的表达带来任何结果，然后大方微笑继续前行。

#### 修成正果

感受就是正果，过程就是正果，不为结果即真自由。

**公式**：正果 = 感受 + 过程

**代码**：true fruit = feeling + process

### 总结

本节课中，我们学习了如何在喜欢的女生面前不扭捏，放得开。记住，大胆表达你的欲望，不要被结果绑架，允许结果的发生。
