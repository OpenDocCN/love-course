# 清风老师合集：14链接贵人，让贵人帮你的实操方法论21期：188.第三课：展示思维：找到你的舞台，快速接触大佬 🎭

在本节课中，我们将要学习“展示思维”的核心方法。上一节我们介绍了定位思维，帮助你明确自身方向。本节中我们来看看如何正确地向“老大哥”展示自己，以获得他们的关注与帮助。

很多人反映在社交场合感到尴尬和焦虑，不知该说什么。问题的根源不在于聊天技巧，而在于“不自信”。不自信的原因，是在社交场合总想“证明自我”，把自己放在了主角的位置上。

这是一个大忌。进入新社交场合，尤其是有“老大哥”在场的场合，我们不应是最重要的人。我们应是最不起眼、甚至是为他人提供服务的人。

俞敏洪校长曾在其演讲中说过一段话：“同学们别看我在北大的校友群里面，你们把我当老大哥。我和马云交朋友的时候，我就是给他们提供服务的那个人。” 这段话揭示了在不同场合中，我们的位置和状态应随之调整。

例如，在我的课程和线下交流会中，我是中心。但与身家过亿的“老大哥”们在一起时，我就是需要谦虚学习的小弟。越谦虚，他们愿意教导的就越多。

因此，关键点在于：不要在“老大哥”面前试图证明自己。保持谦虚、低调、真诚和学习的态度即可。抛开证明自我的负担后，心理压力会小很多。

另一点需要破除的思维误区是：认为只有立刻展现特别优秀，才能获得关注。

这是错误的。在一个场合中，需要的不是你立刻表现得多么优秀。因为一上来就过度展现，会显得“事出反常必有妖”。“老大哥”反而会认为你是带着目的故意表演，从而产生戒备。

重点不在于第一次接触就让他觉得你多么优秀，而在于后续交流中，一次次展现你的行动和成果。

例如，“老大哥”教你一个商业模式，你将其运用到实践中并取得了成果。然后你带着这个成果再去请教他。在一次次的请教与反馈中，“老大哥”看到你通过他的建议获得了成功，他会越来越关注你，并真心想要帮助你。

为什么？因为你能成事。这才是真正的“展示思维”。

现在我们来总结一下核心要点：

以下是两个关键行动指南：
1.  在“老大哥”面前无需过度展现自我。要做的是谦虚学习和低调思考。
2.  在“老大哥”面前无需一上来就展现优秀。真正有效的方式是通过持续的行动结果，一次次赢得信任。

“老大哥”们看人，不是看你表现出来的东西，而是用心、用时间去看。只有一次次展现你的价值，才能真正得到帮助。

将以上两点结合，你会发现这是一个循环往复的过程。你已经开始掌握关键了。

我们向上攀登的过程，不是一条笔直上升的直线，而是一条**曲折螺旋上升的路线**。在我们一次次遇到困难时，往往需要“老大哥”资源和智慧的加持来帮助我们突破。

那么，我们是在巴结“老大哥”吗？

错了。带着这种思想的人注定失败，因为他们的出发点心术不正。他们只是把“老大哥”当成了攀附的对象。

真正能得到帮助的人，一定是有自己项目、能落地、脚踏实地做事的人。“老大哥”的价值在于：当你为一件事努力到极点，达到了99%，他来为你加持那最后的1%，事情就成了。

贵人真的帮你做成了事吗？不是。是因为你足够努力，贵人恰好看到了。

学习本课程，希望你有一个正确的发心：我们向上攀登，不是为了迎合或讨好谁，而是让资源为我们所用，让我们努力的过程加速，让我们成功的结果提早到来。

在你30岁开上一辆豪车，和50岁开上的感觉是不同的。希望各位同学能带着正确的发心，脚踏实地做事，让自己努力到极致。这时，贵人才能够真正帮助到你。

本节课中我们一起学习了“展示思维”的正确方法：破除证明自我和急于表现的心理，通过谦虚学习和用持续成果建立信任，在螺旋上升的道路上获得贵人的关键助力。
