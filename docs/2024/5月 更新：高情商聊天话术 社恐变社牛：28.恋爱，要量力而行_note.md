成为恋爱高手的关键在于心态调整。当你不在乎这个女人时，你自然会成为一个高手。当你在乎时，你几乎所有的策略和动作都会出错，人格魅力也会丧失。

## 三、 如何处理欺骗与谈判 🤝

上一节我们讨论了错误心态，本节中我们来看看一个具体问题及其解决方案。

有学员遇到女友经常骗钱的情况，例如以做美甲为名要钱却并没去。他投入了大量金钱，既无法放弃，又痛苦于继续被骗。

对此，有两种方案：

**方案一：隐忍**
一个成熟的男人需要懂得隐忍。能够忍受他人的欺骗甚至侮辱，是一种能力。懂得隐忍的人，通常能获得更好的结果。

**方案二：设置条件（推荐）**
如果你不愿被骗却想继续交往，关键在于改变付款模式。

**核心策略代码**：
```python
if 对方提出要求（如“我要去美甲”）：
    我的回应 = “可以，美甲后给我看照片，我给你报销。”
```
即：**不见兔子不撒鹰**。不要提前支付，要在对方完成对应行为后再兑现。

你之所以被看不起，就是因为对方还没给你什么，你已经付出了很多。高手的方式是：**你必须做出对应的动作，我才会对你好**。

女人也喜欢挑战，她需要“赢得”你。例如，她想要礼物或买单，就必须不断做出一些讨好你的动作或话语。这是一个长期的 **“拉锯战”** ，目的是让对方心理产生波动，最终通过谈判达成一致。

**谈判心法**：永远不要直接说“不”或拒绝。所有回应都应附带条件。例如，“我是要给你买包的，但是你必须先……”。通过不断提出条件进行拉锯，直到对方服软。

如果谈崩了怎么办？**崩了就崩了**。拥有这种“不怕失去”的心态，你才会有魅力。不谈崩的诀窍就在于永远进行“有条件”的谈判，而非直接否定。

## 四、 平等思维与人格魅力 ⚖️

在人际关系中，必须使用**平等思维**，而非奴才思维。

*   **平等思维**：你对我好，我就对你好；你不对我好，我就不对你好。
*   **奴才思维**：你骂我，我也尊重你，但我不会对你好。（例如，对于不尊重自己的人，保持礼貌但不搭理。）

你必须拥有这种态度，才有人格魅力。应用到聊天中：如果女生不回消息，你就过几天再发一条，无所谓。即使面对高分美女，也应保持这种姿态。你冷落我，我也冷落你。谁先忍不住，谁就失去了主动权。

父母把你生下来，不是让你去求女人的。男人应该是主心骨，顶天立地，以自我为中心。不要被网络上批评“大男子主义”的言论欺骗，男人就应该有男人的样子。媒体上那些男人下跪、上交工资卡的视频，都是在把你教成傻子和奴才，削弱你的男性魅力。

## 八、 识别伴侣类型：表达型 vs. 行动型 🧩

人分为两种恋爱感受类型：

1.  **表达型**：通过语言表达来感受爱。无论你做了多少，如果听不到你说“爱她”，她就感觉不好。
2.  **行动型**：通过具体行动来感受爱。比如你帮她做事、送礼物。她不太在意你说什么。

在与女人相处时，要先探索她的观念。通常：
*   **非常漂亮、家境富裕的女生**，多是行动型。她们需要看到具体付出。
*   **家境普通、自身条件一般的女生**，更容易被语言和简单的关心打动，属于表达型或混合型。

追求漂亮女人的代价非常高，且性价比低。她们要求高收入、房子、奢侈品，性格往往高傲，且可选择对象多。很多漂亮女人要到二十七八岁，价值下降后才会放低心态，但那时其吸引力也已降低。

**核心结论**：**追求漂亮女人的代价相当高昂，而且划不来**。最好的选择是找“六分妹子”，相处更舒适。
