# 凡哥私教课程：恋爱成长必修课之七大思维：001.1.吸引思维_note

在本节课中，我们将要学习恋爱思维系列的第一课——吸引思维。我们将探讨为什么传统的“追求”方式往往无效，并理解“吸引”才是建立健康恋爱关系的核心。

## 概述

在我从事情感领域工作的八年时间里，学员最常问的问题包括：“为什么我对她这么好，她还是不答应和我在一起？”、“我追求她很久了，她还是把我当朋友，该怎么办？”、“我天天嘘寒问暖、送礼物、请吃饭，但她却说对我没感觉，我该怎么办？”

之所以出现这些问题，根本原因在于恋爱思维存在误区。解决问题需要治本。现在我要告诉所有兄弟们：**女人不是追来的，女人是吸引来的**。

## 社会背景与思维困境

首先，分享一个有意思的数据。2015年末的人口普查显示，全国总人口13.7亿，其中男性7亿，女性6.7亿，相差0.3亿。当时每出生100个女孩，就有105个男孩出生。到2016年，男性超出女性人口30%以上，总“光棍”人数从2012年的1000万上升至2022年的3000多万。如今，每出生100个女孩，就有约116个男孩出生。

在中国，娶妻问题正变得越来越困难。在男多女少、“狼多肉少”的大环境下，男性面临巨大压力，因此容易产生“快速追求一个女生以达到繁衍后代目的”的想法。

然而，许多男人从小到大学习到的唯一吸引女性的方式就是“追求”。他们认为，对她好、给她买东西、送礼物，最终就能感动她，让她将心比心地和自己在一起。但结果往往是女生说“你是个好人，但我们不合适”，或者说“我对你没感觉，还是做朋友比较好”。

## 核心概念：感动 ≠ 爱情

这里必须明确一个核心概念：**感动并不是感情，也不是爱情**。

那么爱情是什么？爱情源于你自身的价值。例如：
*   你的事业心很强。
*   你很有主见。
*   你拥有成熟的恋爱观。
*   你非常稳重。

因为你拥有这些特质，女人与你交往时会欣赏你、崇拜你、对你心动。这才是爱情的开始，也就是**互相的吸引**。

## “追求”行为错在哪里？

接下来，我们分析“追求”这件事到底错在哪里。我们可以把“追求”二字拆开理解。

我们先看看“追”这个字，通常会有哪些表现。

以下是“追”的典型行为链条：

1.  **调查户口**：例如，“你是哪里人？”、“你在哪工作？”、“你做什么工作？”、“你喜欢吃什么、做什么？”。同时也会介绍自己：“我是哪里人，做互联网工作，兴趣爱好是……”。
2.  **间隔性定点查岗与关心讨好**：例如，“在干嘛？”、“吃饭了吗？”、“睡觉了吗？”、“吃药了吗？”、“天冷了多喝热水”、“有快递给你买了个裙子，记得查收”。
3.  **邀约**：觉得时机差不多后，开始发出邀请。
4.  **“求”的行为**：邀约的具体内容，例如“我们一起吃饭/看电影/爬山/去公园吧”。如果邀约不成功，则进入“间隔性定点查岗 + 邀约”的循环，例如“在干嘛？我们一起去吃饭”、“在干嘛？我们一起去看电影”。
5.  **放大招——表白与哀求**：当所有“追”的招数用尽后，开始放大招，直接表白：“做我女朋友好不好？”、“不要不理我好不好？”、“跟我说句话好不好？”、“求求你再给我一次机会”。

你之所以追不上女人，是因为在“追”和“求”的观念指导下，你的行为通常基于以下六种错误的前提假定。

### 六种错误的思维假定

以下是六种常见的错误思维假定及其分析：

1.  **错误假定一：只要我对她足够好，她总有一天会被感动**
    *   **表现**：如果她没吃饭，你就主动买；她工作没做完，你就免费帮她做；她家电脑坏了，你马上上门修。你认为这些行为能让她看到未来交往时你会对她很好，从而喜欢你。
    *   **真相**：如果你只会对她好、帮她做事，那么她和你在一起会觉得很没劲。因为你和其他追她的男人没有区别，对她而言已毫无挑战性。就像玩一个总能在一分钟内轻松获胜的游戏，你不会再有兴趣。

2.  **错误假定二：只要我同意她的一切，我们就能避免矛盾，成为完美一对**
    *   **表现**：她说“我很傲娇，脾气不好”，你回答“没关系，我宠着你”。她说“你也太逊了，连我都比你强”，你回答“你还挺厉害的”。她说“我喜欢能容忍我坏脾气的男人”，你立刻说“这不就是我吗”。
    *   **真相**：如果你同意她的每一件事、每一个观点，你就是在违背自我、讨好对方。在女人眼里，你会变成一个没有主见、虚伪、见风使舵的男人，她反而会因此对你失去信心。实际上，当你坚持自己的不同观念时，女人表面可能不快，心底却会产生敬佩和信任，从而更喜欢你。

3.  **错误假定三：只有多联系、多关心，让她多了解我，她才会喜欢我**
    *   **表现**：每天准时发早安晚安，叮嘱她天冷加衣、多喝热水、早点睡觉，恨不得24小时都和她说话。
    *   **真相**：问题的根源在于男性“一次做到位”的思维。但让女人爱上你的核心，**不是你对她投资了多少时间，而是她对你投资了多少时间和心思**。成功的关键不是你脑子里想她最多，而是你能占据她头脑最多的时间。

4.  **错误假定四：只要我勇敢表白，展现强烈的爱意，她就会同样爱上我**
    *   **表现**：直接对她说：“我喜欢你很久了，做我女朋友吧。”
    *   **真相**：试想，如果一个你完全不喜欢的女人发疯似的说喜欢你，你会因此爱上她吗？显然不会。那么，你为什么会期待女人因为你表白而喜欢你呢？

5.  **错误假定五：只要我给她买礼物、送东西，她就会喜欢我**
    *   **表现**：省吃俭用两个月给她买爱马仕包包，或者送她心仪已久的卡地亚腕表。
    *   **真相**：这么做会在她潜意识里建立一种思维定式：你需要用物质来打动和争取她，否则她不会喜欢你。这背后的潜台词是你认为自己本身不值得她喜欢，所以需要外在物品来赢得她的心。这相当于把选择权完全交给了对方。

6.  **错误假定六：只要我长得帅或很有钱，她就会喜欢我**
    *   **表现**：认为吸引女生主要靠外貌和财富。
    *   **真相**：长得帅或有钱确实不是阻碍，但它们只占吸引力的很小一部分，甚至可以忽略不计。对比一个有钱/帅的男人和一个极具人格魅力、吸引力的男人，女人通常会被后者吸引。你会看到有钱人搂着小明星却无爱情，也会看到其貌不扬的穷小子搂着你梦寐以求的女人，并且这个女人深爱着他，甚至愿意倒贴。

## 思维解放与课程预告

说了这么多，核心在于大部分男人并不清楚什么最吸引女人。以往认为吸引女人就是“对她好”，就像电视剧里演的那样，最终女生被感动，两人幸福地在一起。但你从未想过，那是编剧为迎合观众期望而故意安排的情节。

在现实中，这么做只会换来一句：“谢谢你，你是个好人，但我们真的不合适。”

因此，如果你想学习如何吸引女人，第一课我必须让你**解放思想**。

下一节课，我将分享**步骤思维**。这也是非常干货的内容，能帮助很多兄弟少走弯路，用最短时间找到最适合自己的女生，从认识到步入美满婚姻。很多兄弟在恋爱道路上缺乏正确的步骤。

举个例子，就像玩《王者荣耀》一样，你需要有步骤。你不可能一级就去攻打敌方水晶。正确的步骤是：第一步发育，拿野区资源；第二步参加团战，拿人头；第三步推塔；最后才能一步步推进，打到对方水晶。

恋爱同理，不是凭一腔热血往前冲就能搞定。有了正确的步骤，你和女生确立关系的几率会大大增加。**恋爱是需要谋略的**。下节课，我将分享正确的恋爱步骤有哪几步。
