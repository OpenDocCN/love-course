# 凡哥私教课程：恋爱成长必修课之七大思维：001.1.吸引思维

🎼应广大学员要求，本次课程将为大家带来关于恋爱思维的系列专题课程，帮助学员建立正确的聊天思维，从而少走弯路。

🎼在从事情感领域工作的8年中，我遇到学员问得最多的问题是：为什么我对他这么好，他还是不答应跟我在一起？追求她这么久，她还是把我当朋友，我该怎么办？我喜欢一个女生5个月了，天天嘘寒问暖，送礼物，请吃饭，发消息，她生病了还给她炖汤，但她就是没有感觉，我该怎么办？

🎼这些问题出现的原因在于你的恋爱思维存在问题。解决问题需要治本，所以我要告诉所有兄弟们：女人不是追来的，是吸引来的。

🎼今天先给大家讲恋爱思维的第一节课——吸引思维。首先给大家分享一个有趣的数据：2015年年末，我国总人口13.7亿，其中男性7亿，女性6.7亿，人口数量相差0.3亿。也就是说，每出生100个女生，会有105个男生出生。2016年，男性超出女性人口的30%以上，光棍人数由2012年的1000万上升至2022年的3000多万。所以，每出生100个女孩，会有116个男孩出生。在这样的环境下，娶老婆的问题将越来越困难。

🎼男多女少，狼多肉少，我们作为男人，面对的压力非常大。所以，我们想要快速追求一个女生，达到繁衍后代的目的。从小到大，我们学习到的唯一吸引女人的方式可能就是追求她，对她好，给她买东西，送礼物，最后她会被我感动，然后心甘情愿地跟我在一起。但结果往往是女生说：“你是一个好人，但我们并不合适”或“我对你没有感觉，我们还是做朋友比较好”。

🎼所以，一定要明确一个概念：感动不是感情，不是爱情。爱情是你身上有价值，比如事业心强、有主见、有成熟的恋爱观、稳重等。因为你有这样的特质，女人会欣赏你、崇拜你、对你心动，这就是爱情的开始，也就是互相的吸引。

🎼接下来，我们先来聊聊追求这件事情错在哪里。把“追求”这两个字拆开理解一下。

🎼首先，你肯定会去调查户口：你是哪里人？在哪工作？做什么工作？平时喜欢吃什么？喜欢做什么？

🎼然后，你开始说：“我是哪里人？我在哪工作的啊？我是做什么互联网工作的，我平时兴趣爱好是什么？”

🎼接下来，你会去做间隔性的定点查岗，去关心讨好：在干嘛？吃饭了吗？睡觉了吗？吃药了吗？这两天天气比较冷，要多喝热水啊，天气已经凉了，记得多穿衣服，有快递给你买了个裙子，你记得查收一下，然后觉得差不多了，又开始邀约。

🎼邀约之后，你就会产生球的行为：比如说我们在一起去吃饭、看电影、爬山、公园。如果邀约没有成功，就进入下一个阶段，也就是间隔性的定点查岗加邀约。

🎼最后，你把所有追女人的招数都用光之后，开始放大招：直接对这个女生表白，做我女朋友好不好？哎，不要不理我好不好？跟我说句话好不好？我就是想听一听你的声音，求求你再给我一次机会。

🎼所以，你为什么追不上女人？就是因为你在追求的过程中，在“追”和“求”这两种观念的指导下，很多行为都会不约而同地产生以下6种前提假定：

🎼1. 错误的假定：如果我对他特别好，他会帮我做很多事情，那么他就会看到我们交往或者说结婚之后，我会对她很好，所以说她会因此喜欢我。女人总有一天会被我感动的。

🎼2. 错误的假定：如果我同意他的每一件事情，每一个观点，那么我们之间就会避免矛盾，他就不会不理我，他会觉得我很宠着她，因此而感激我，这样我们才能是完美的一对，女人才会爱上我。

🎼3. 错误的假定：追女生，我就要多给她打电话，我就要多关心她，给她发微信、发短信，我只有多跟她接触，让她对我有更多的认识，还有了解，她才会喜欢我。

🎼4. 错误的假定：如果我喜欢她，我就要去跟她表白。我只有表现出来我对他强烈的爱，他才会因此喜欢上我，爱上我。

🎼5. 错误的假定：我给她买东西，我给她送礼物，她就会喜欢上我跟我在一起。

🎼6. 错误的假定：如果我长得帅、很有钱，她就会喜欢我。

🎼其实，大部分男人都不清楚什么东西才是最吸引女人的。以前就认为怎么样吸引女人，也就是那么回事，只要是对她好，就像电视剧里演的那样，男人对女人好，女人就会被感动，然后两个人就幸福的在一起了。但是，你从来都没有想过，这都是编剧根据观众们希望的结果，他故意安排的情节。

🎼所以，在现实生活当中，你这么做了，女人她会跟你说：“谢谢你，你是一个好人，但我们真的不合适。”所以，你如果想要学习怎么样去吸引女人，那么第一课我一定要让你去解放你的思想。

🎼在未来，我会给所有兄弟们分享刚货的形式，就是理论加上实战。那么，从今天开始，我会给兄弟们分享的课程是恋爱思维的系列的专题课，分别包括了吸引思维、步骤思维、奖品思维、前提思维、拒绝真命思维、女性思维，还有博弈思维。

🎼下面一节课，我会给大家分享步骤思维，同样也是非常干货的内容，可以让很多兄弟们少走很多弯路，可以用最短的时间找到最适合自己的女生。从认识再到美满的婚姻，很多兄弟们在恋爱的道路上，没有正确的步骤。

🎼我给大家举个例子，就跟你玩王者荣耀一样，你玩王者荣耀呢，你会有一个步骤。比如说你上来才一级，你不可能就去攻打敌方的水晶。你会在第一步先发育，然后去拿一下野区的资源，然后去参加团战，拿一拿人头，然后才能完成推塔，最后才能一步一步地去推进，去打到对方的水晶。

🎼其实恋爱也是一样的，不是说你凭着自己的一腔热血往前冲就能搞定的。你有了一个正确的步骤，你跟女生确立关系的几率就会增大很多很多。所以说恋爱是需要谋略的。
