# 21、凯男的社交情商课：第一节课：爱商提升的7个思维逻辑：002.2、步骤思维

🎼今天为大家带来颠覆恋爱思维的第二节课，即恋爱思维的步骤思维。那么，从认识女生到成为男女朋友，需要经历哪些步骤呢？其实，让一个女生从陌生到爱上你，成为你的女朋友，只需要三个步骤：吸引、建立舒适感和升高关系。

### 吸引

吸引分为两个步骤：激起兴趣和建立吸引。

**激起兴趣**的目的是让女生对你感兴趣，愿意与你聊天。如果你做不到这一点，那么后续的一切都免谈。在微信上与女生聊天之前，你需要通过朋友圈建设来激发她的兴趣，让她看到你丰富多彩的生活，看到一个有趣、立体的男人。

**建立吸引**主要是指在情绪上建立吸引。你需要将女生的情绪聊到飞起，让她感受到你的魅力。

### 建立舒适感

建立舒适感包括两个阶段：建立浅层舒适感和建立深层舒适感。

**建立浅层舒适感**可以理解为基本信息交换，包括你的工作、家庭、年龄、兴趣爱好等，让女生对你有一个初步的安全感。

**建立深层舒适感**是指在合适的时机聊你们的三观，包括恋爱观、婚姻观、人生观等，用你的三观去吞噬掉她的三观。

### 升高关系

升高关系是指关系一步步变得亲密，最终达到与女生确立关系的目的。升高关系需要加入暧昧元素，例如称呼、约会等。

### 跨步骤操作的风险

如果不按照步骤操作，会出现以下几种情况：

1. 没有建立吸引和舒适感，直接约女生出来。
2. 没有建立吸引，只会建立舒适感，没有升高关系。
3. 建立了吸引和舒适感，但聊天平淡无趣，不敢升高关系。
4. 在特殊场所快速与女生发展关系，没有建立舒适感。

以上四种情况都可能导致失败。

### 总结

本节课介绍了恋爱思维的步骤思维，包括吸引、建立舒适感和升高关系。只有按部就班地完成这三个步骤，才能提高成功的概率。下一节课将继续分享恋爱思维的第三课：奖品思维。
