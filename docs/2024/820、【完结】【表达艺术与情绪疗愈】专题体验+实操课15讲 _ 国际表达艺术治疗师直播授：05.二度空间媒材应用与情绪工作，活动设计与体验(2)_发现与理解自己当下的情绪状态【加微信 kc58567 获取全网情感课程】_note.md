# 【完结】【表达艺术与情绪疗愈】专题体验+实操课15讲 _ 国际表达艺术治疗师直播授：05.二度空间媒材应用与情绪工作，活动设计与体验(2)_发现与理解自己当下的情绪状态

## 概述
在本节课中，我们将学习如何发现和理解自己当下的情绪状态，并通过表达艺术的方式释放压力。

## 课程内容

### 1. 物资准备
以下是本次体验课所需的物资：
- A4白纸
- 四种颜色的纸张
- 白纸杯
- 固体画具（蜡笔、马克笔等）
- 轻粘土（白色、红色、蓝色、黄色）
- 拼贴材料

### 2. 表达艺术发展阶段
- **12-15个月**：随机标记，探索力度和线条控制。
- **15-24个月**：受控涂鸦，形成不同形状。
- **3-4岁**：基本图案，开始模仿和创作。
- **5-6岁**：完全复制或创作，关注背景和人物联想。
- **6-7岁**：理解象征意义，开始使用颜色和动物表达情绪。
- **9岁以上**：自然掌握绘画能力，区分写实和抽象。

### 3. 压力与情绪
- 压力与情绪密切相关，正面适当的压力有助于生活动力，过度压力则可能影响身心健康。
- 压力应对方式：冻结、逃跑、迎战。

### 4. 体验活动：压力火山
- 选择一张不喜欢的颜色纸，代表压力来源，撕碎或处理。
- 选择一张纸，用文字或绘画表达压力来源。
- 选择第三张纸，用涂鸦表达与压力相关的情绪。
- 创作压力火山，装饰火山和环境。

### 5. 分享与反思
- 分享作品，分享感受和想法。
- 反思压力来源、应对方式、情绪表达。

## 总结
本节课通过表达艺术的方式，帮助我们发现和理解自己当下的情绪状态，释放压力。希望大家在创作过程中享受过程，感受情绪，并从中获得启发。
