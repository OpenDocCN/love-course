# 瑞恩年会员PRO 瑞恩高级会员：课程63(经典废测：在你眼里我是什么人)

## 概述
在本节课中，我们将学习如何通过情感流动与对方进行有效沟通，并了解为何复制粘贴话术往往无效。

## 情感流动的重要性
以下是几个关键点：

1. **情感流动**：在聊天中，情感流动比话术本身更重要。**公式**：情感流动 > 话术。
2. **避免逻辑思考**：当回复对方消息时，如果感到头疼或烦躁，说明你的方向可能错了。**公式**：逻辑思考 → 头疼/烦躁。
3. **感受的重要性**：有效的回复往往来自于感受，而非逻辑。**公式**：感受 > 逻辑。

## 案例分析
以下是一个案例分析，展示了如何通过情感流动与对方进行有效沟通：

1. **女生提问**：女生问：“在你眼里我是什么人？”
2. **回应**：回答：“女人。”并附上微笑表情。
3. **情感流动**：继续聊天，让情感在两人之间流动，例如：
   - “女人就应该有备胎，这不很正常吗？”
   - “我还可以万变，其实在下只有七十二遍。”
4. **主动行为**：女生表现出主动行为，例如：“我还可以万变。”

## 复制粘贴话术的无效性
以下是几个原因说明为何复制粘贴话术往往无效：

1. **情感不匹配**：复制粘贴的话术无法与当下情感匹配，导致牛头不对马嘴。
2. **缺乏感受**：复制粘贴的话术往往缺乏感受，无法引起对方共鸣。
3. **无法应对不同回复**：当对方给出不同回复时，复制粘贴的话术无法应对。

## 总结
本节课中，我们学习了情感流动的重要性以及为何复制粘贴话术往往无效。通过关注情感流动和感受，我们可以更好地与对方进行沟通。
