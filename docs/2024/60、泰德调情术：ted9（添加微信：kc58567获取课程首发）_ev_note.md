## 复格技巧
### 什么是复格？
**公式**：复格 = 赋予女人资格

复格是指通过给予女人某种特质或能力，从而提升她的自我价值感。

### 复格的实战应用
**代码**：
```python
# 假设女人在游戏中反应木讷
if 女人在游戏中反应木讷:
    # 结合客观事实进行复格
    复格("你玩游戏有点呆，但这也说明你很专注。")
```

### 复格的注意事项
- 结合前后语境和客观事实。
- 让女人觉得你的话非常合理。

## 自我证明陷阱
### 什么是自我证明陷阱？
**公式**：自我证明陷阱 = 为了证明自己而赚钱或改变自己

### 如何避免自我证明陷阱？
- 不要为了迎合女人而改变自己。
- 不要为了证明自己而赚钱。
- 关注自己的内心需求，而不是他人的评价。

## 输出标准
### 什么是输出标准？
**公式**：输出标准 = 表达自己的价值观和标准

### 如何输出标准？
- 结合女人的好奇点进行复格。
- 例如：你觉得你可能更适合公务员类型，因为你看起来很板正。
