# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：42、一份亲密关系，两份选择，如何选择留还是退

## 概述

在本节课中，我们将探讨如何在一段亲密关系中做出是否继续留存的决策。我们将分析四个选择板块，并探讨如何通过自我反思和行动来做出最佳决策。

## 核心概念

**选择板块**：

1. **离开**：评估离开后生活质量的提升。
2. **留下并改变可改变的**：改变可改变的方面，构建信任。
3. **留下并接纳不可改变的**：接纳无法改变的方面，关注自身价值。
4. **留下并破罐子破摔**：放弃改善关系，专注于自我成长。

## 教程内容

### 一、离开

1. **评估生活质量**：思考离开后生活质量的提升，包括收入、居住地、个人状态、年龄、健康、社会认可度等。
2. **分析原因**：明确离开的原因，如三观不合、性格不合、底线问题等。

### 二、留下并改变可改变的

1. **改变可改变的方面**：挑选可改变的方面，如沟通方式、行为习惯等。
2. **构建信任**：从自身内心开始，强化信任感。
3. **明确底层价值**：清晰自己在亲密关系中的底层价值，如成为什么样的伴侣，为何坚持这段关系。

### 三、留下并接纳不可改变的

1. **接纳无法改变的方面**：放下批判、敌意、绝望，接纳自己的烦恼和担忧。
2. **关注自身价值**：放大自身价值，聚焦于自身成长。
3. **活在当下**：不要过度担忧未来，专注于当下。

### 四、留下并破罐子破摔

1. **放弃改善关系**：不再尝试改善关系，专注于自我成长。
2. **关注自身价值**：放大自身价值，聚焦于自身成长。

## 总结

本节课中，我们探讨了在一段亲密关系中如何做出是否继续留存的决策。通过分析四个选择板块，我们可以更好地了解自己的内心需求，并做出最佳决策。希望这节课能帮助大家更好地处理亲密关系中的问题。
