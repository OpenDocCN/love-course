# 51、人人都需要的亲密关系_：13_巧用撒娇术化解小矛盾_1

在本节课中，我们将学习如何运用“撒娇术”来巧妙化解与伴侣相处时可能遇到的小摩擦和过失。撒娇并非示弱，而是一种高情商的沟通技巧，能有效缓和气氛、增进亲密感。

## 化解自身小过失

上一节我们介绍了课程概述，本节中我们来看看当你不小心犯下一些小过失时，如何用撒娇来化解尴尬。核心思路是：**承认过失 + 示弱自嘲 + 给予对方价值感**。

以下是两种具体的方法：

**1. 妄自菲薄法**
当你不小心弄坏小物件、弄脏对方衣服时，可以使用此法。
*   **步骤一：装傻示弱**。睁大无辜的眼睛，微微张嘴看向对方眉心，委屈地嘟嘴说：“怎么办啊？我好像把它弄坏了。”
*   **步骤二：自暴自弃**。在对方查看时，哼唧一声，用仰天长啸般的绝望语气补充：“哎呀，完了完了，要被嫌弃/讨厌了啊。”
*   **步骤三：给予甜饼**。看着对方忙碌修复的样子，由衷感叹：“我真是太笨了，没有你可怎么办呢？”

此方法的**核心公式**为：**制造问题 → 示弱求助 → 赋予对方“英雄感”**。先承认错误，再通过自嘲降低对方责怪的预期，最后通过夸奖让对方获得价值感，从而巧妙转移对过失本身的关注。

**2. 责任嫁接法**
当你因自身原因（如迟到）引发对方不快时，可使用此法。核心是**自然地将问题源头与对方进行弱关联**，引发其关心。
*   **步骤一：诚恳道歉**。首先认真表达歉意：“亲爱的，我来晚了，对不起。”
*   **步骤二：自然嫁接**。拍拍腿或脚，带着疼痛且自责的表情说：“哎呀，我这鞋/路也不知道怎么回事，走得好好的突然就不顺了。”**注意**：点到为止，切勿夸大痛苦。
*   **步骤三：反客为主**。自然地坐下或挽住对方，小心地问：“你到很久了吧？你不会生气吧？”

此时，对方的注意力会从“你迟到”转移到“你因赴约而遭遇小意外”上，从而主动关心你，忘记不快。

## 化解争吵与冷战

上一节我们学习了如何为自己的小过失善后，本节中我们来看看当双方发生争吵或陷入冷战时，如何用撒娇打破僵局。

**1. 及时控制，软性撒娇**
当气氛紧张、对方明显生气时，应立即停止闷头撒娇或争吵。
*   **做法**：放慢放软语气，但不要油腻。例如，可以说：“我们都先冷静一下再谈好不好？我先去给你倒杯水/做点吃的。”
*   **核心**：表现出**明事理**（主动提议冷静）、**带委屈**（语气柔软）、**显关心**（为他做件小事）的态度。这本身就是一种“软撒娇”，能为对方铺好下台阶。

**2. 主动破冰，结束冷战**
如果你已消气，想主动结束冷战，有两种直接的撒娇方式。

以下是两种破冰方法：

**方法一：肢体行动（不说话）**
*   **做法**：直接从他背后环抱。双手上下分开贴住他的腰部，侧脸贴住他的后背，保持沉默。
*   **后续**：通常对方会握住你的手表示和解。此时你可贴着他后背点头，再将手臂收紧一些。**代码式步骤**可简化为：`拥抱 → 等待回应 → 点头 → 收紧手臂`。

**方法二：语言沟通（说话）**
*   **做法**：走到他面前，对视两秒后低头，手指搅在一起，用低声委屈的语气说：“我想好了，我不生气了，你能抱抱我吗？”
*   **氛围加持**：说完后抬眼看他，若眼眶含泪或轻轻嘟嘴、吸鼻子，效果更佳。主动服软并索要拥抱，几乎能确保对方心软。

## 总结与重要原则 🎯

本节课中，我们一起学习了如何用“妄自菲薄法”和“责任嫁接法”为自己的小过失撒娇解围，也掌握了在争吵时“软性撒娇”控制局面、在冷战后通过“肢体拥抱”或“语言服软”主动破冰的技巧。

最后必须强调的核心原则是：**撒娇术适用于化解无伤大雅的小矛盾与小过失**。它本质是一种情感润滑剂。如果遇到严肃的原则性问题，则必须端正态度，勇于承担自己应负的责任，用真诚的沟通去解决。聪明的做法是看清问题本质，用撒娇偷个小懒、躲个小麻烦，但在大事上始终保持担当。
