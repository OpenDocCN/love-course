# 17、海蓝老师情绪梳理个案课：002-个人采访

🎼大家好，我是吴维伟，27岁，来自湖北武汉，是一名护士。本次课程将梳理我在工作中的焦虑情绪。

### 工作焦虑的体现

我在工作中经常感到焦虑，例如去喝水或离开病人，哪怕只是一分钟，我也会感到焦虑。

**公式**：焦虑 = 离开病人 + 时间短

### 焦虑带来的影响

这种焦虑让我在工作中感到非常疲惫，有时还会出现烦躁的情绪。一忙起来，这种情绪会更加糟糕。

**公式**：疲惫 = 焦虑 + 工作繁忙

### 对工作的态度

这种焦虑让我对工作充满了负能量，感到倦怠。每天下班后都非常疲惫，上班时感觉像战场。

### 解决焦虑的期望

如果能够解决这种焦虑，我相信上班会变得更加轻松，下班也不会那么累。也许我的睡眠也会好一些。

### 工作的意义

我对这个工作可以更多地享受成就感，而不是被焦虑情绪困扰，感受不到工作的意义。

### 与病人的交流

如果没有焦虑，我上班时可以面带微笑与病人交流。当他们有痛苦情绪时，我可以运用所学心理知识来安慰他们，缓解他们的焦虑。

### 生活的影响

下班后，我可以更好地享受自己的生活和其他事情，不会被这种情绪困扰。

### 总结

本节课中，我们学习了吴维伟护士在工作中遇到的焦虑情绪及其影响。通过梳理和解决焦虑，她希望能够更加轻松、愉悦地享受工作，并改善生活质量。
