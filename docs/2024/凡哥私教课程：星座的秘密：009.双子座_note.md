# 凡哥私教课程：星座的秘密 009.双子座

🎼大家好，我是木子老师。今天我们来探讨一下双子座。众所周知，双子座被认为具有双重人格，这一特点非常显著。在老师看来，一切皆有因有果。

不可能有人天生就是双重性格，这必然是成长过程中某些性格因素导致的。最主要的原因是他们不想麻烦任何人。

“麻烦”有多种含义，无论哪种含义，都会导致他们很少说实话。这不是骗人，而是一种欺瞒，他们不想让别人担心或看不起。

他们可以对自己的事情闭口不谈，同时认为没有人能真正了解他们，他们自己也享受这种感觉。这种性格源于自卑和胆怯。尽管双子座不记仇，但他们会记住所有人对他们的批评，不会听也不会承认。但这些批评对他们的影响是潜移默化的。

例如，双子座年轻时可能会因为喜欢开玩笑而讨人厌，因为他们经常把握不好度。如果他们无意中让别人生气，他们也不会道歉。但他们会记住对方生气的表情和难听的话。下次开玩笑时，他们会更加小心翼翼。

然而，由于他们的本性并未改变，他们只是因为知道这样做不好，所以强迫自己不要这样做。这导致他们活得特别压抑。因此，在一个人时，他们可能会原形毕露。这就是我们所说的两种人格。

另一种人格是他们内心阴暗面的体系。准确地说，双子座的两种人格是一种是他们原本的样子，另一种是他们活成的样子，或者说是被迫活成的样子。

很多人都会有这样的经历，但为什么只有双子座变成这样呢？主要还是因为他们的性格原因。他们觉得自己聪明、幽默、主观意识强、骄傲。

这五个点加起来就能造成双重人格的原因，就是他们的主观意识特别强。在正常情况下，双子看到与自己不同的想法时，第一反应就是别人错了。然后他们会跳下来与对方争辩。

他们可能没有恶意，只是想要别人夸奖自己，找到存在感。任何人被攻击后，肯定不会老老实实地夸奖对方，对吧？这时，双子座的性格使他们不会善罢甘休。

由于他们聪明和幽默，他们一般不会选择少说一句，而是会一直说到对方闭嘴为止。这时，如果局面失控，双子座心里没有底。因为他们没有恶意，也不想把局面弄成这样，但他们也不知道如何收场。

由于他们骄傲，他们不允许自己低头。因此，在争吵后，他们往往会自责很久。因为他们知道这是他们自己挑起来的。下次遇到这种情况，他们还是会这样做，因为他们骄傲。

一般情况下，他们不会道歉，只会装作若无其事地在对方身边转悠，试图找台阶下，但找不到机会。这时，我们称之为“怂”。

如果这种情况发生多次，他们可能会学聪明，知道是自己的问题，但不会去改变。更多是因为他们骄傲和不想麻烦别人，不想让你觉得他们为你改变，因为他们觉得这样不酷。

因此，面对双子座时，很多人都会有这样的感觉：这个人太特立独行了，好赖话都听不进去。其实他听进去了，只是不知道如何去做，也不想丢脸。

甚至他可能觉得你们的关系在他眼里还不配，所以他不想去自讨没趣，为你改掉这些小毛病。接下来该怎么办呢？

他们会继续伪装，只能继续伪装下去。现在的问题是，他们这样的行为，几乎身边的朋友都知道了。伪装有什么意义呢？并没有。于是，他们想出了一个办法：等我熬过这段时间，到了一个新的环境，我才改变。

这种改变其实也是一种伪装，因为他们换了一个新的环境，大家不知道他们以前是什么样子的。这只是一种伪装，只是看上去他好像变好了，变成熟了，但其实并没有。

因此，他们实际上会感到很孤僻，因为好像没有人能走进他们的内心，因为他们早就把那道门关上了。这就是老师之前提到的那句话：这是一种发自肺腑的孤僻，因为他们知道这么多年他们已经改变太多了。

变得自己都快要不认识了。身边的朋友可能和他们相处得还不错，但他们知道那些人眼中的自己根本就不是真正的自己，或许他们也没有想那么多，只是本能地抗拒。

但是我想说的是，如果一个人不坦诚，带着伪装去和别人相处，别人肯定也不会坦诚。因为这个世界上有多少傻瓜呢？这个道理双子也明白，但他们还没有办法，因为他们害怕自己再犯同样的错误。

因为他们总是记得，当年因为和你争吵，就离开了一个这样的人，见人说人话，见鬼说鬼话，那其实也变成了他们的一种生活生存方式。

如果你觉得你身边的双子每天都嘻嘻哈哈的，告诉你，并不是他们无忧无虑，没有烦恼，而是他们不想增添新的烦恼。他们选择用这种方式来掩盖。

因此，我经常用双子的一些形容词，比如“花心”、“三分钟热度”、“没有耐心”、“不踏实”、“不靠谱”等。其实也不是，因为双子表现出来的样子确实是这样的。

但你也可以说这是他们刻意为之的。因为他们觉得那个专一、有耐心、靠谱的自己太蠢了。因为那样的自己只会得罪人，还不如嘻嘻哈哈的开心。

其实，双子对任何人都没有太大的要求，他们会把所有的错都归于自己无能。因为他们的原则性，他们很难改变自身的一部分性格。要改变就全部都改变。

举个例子，任何性格都有两面性。专一在某种程度可以说是偏执，因为他们想要变得更圆滑一点，但又做不到专一。即使双子知道这个道理，他们也不会改变自己的特性去迎合别人。

对他们来说，只有专一而偏执，或者伪装成花心而圆滑，就可以了。能接受这样的人，才有资格让双子对他敞开心扉。

其实我解释了这么多，已经很能说明双子座为什么是这样的性格了。因为他们所有的完美、成熟、浪漫都是假的，都是他们伪装出来的。

如果他真心喜欢一个人，想要付出真心，这些伪装也就用不上了。因为他用这样的伪装方式去真心追求一个人的时候，他其实是很难做到的。

这就是为什么双子很孤僻，很难找到真爱的原因。但这个性格与金牛座的性格特别相似。但金牛没有双子那么聪明，也没有那么有原则，而且金牛不会委屈自己。

所以我才说双子不是真的花心。因为他们其实根本花心不起来。包括在大家眼里，双子很渣的行为，在他自己看来却很自然。

因为他们是典型的有贼心，没有贼胆的代表。也可能是因为他们双重人格的原因，他们心里始终住着一个人。

这个人没有离开的时候，或者没有心的人愿意住进来的时候，他始终都会是那个样子。的温暖。

其实我告诉你，双子其实不需要理解，他其实就是说他需要理解，但他不需要同情，他更不会被感化，明白吗？

他需要的不是别人的同情和怜悯，而是理解。这边老师只给你提三点建议。

第一点，你跟双子在一起的时候，千万不要作。不是他没有耐心，而是他会自卑，怀疑自己会退缩。

第二点，千万不要跟双子提分手，除非你是铁了心想要分手。

第三点，千万不要对他失去信心。他可能现在有很多毛病，不是他不想改，而是他不知道怎么去改，或者他还没有动力去劝自己改。

你不需要去包容他，不需要去宠爱他。你只需要去理解他，给他机会和动力，他就会为你们的未来考虑好一切，不需要任何东西。

大家心里其实都明白的，明白吗？所以老实讲，这边就是想要很清楚地告诉你，双子的双重人格，包括他们展现出来在你面前的样子，可能都不是最真实的他们。

所以如果你能够看到一个你觉得不太OK的双子的时候，可能就是他最真实的样子。老师希望你能够珍惜这一点，能够理解他，能够包容他这些。

等你能够完完全全接受他的时候，我相信你那个时候也是非常幸福，非常OK的一个状态。
