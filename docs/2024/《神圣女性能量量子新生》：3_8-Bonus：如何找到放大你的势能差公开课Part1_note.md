## 什么是势能差？⚡

上一节我们介绍了课程主题，本节中我们来看看核心概念“势能差”究竟是什么。

势能差是一个能量层面的概念。你可以将其通俗地理解为“降维打击”。当你在某个情境、领域或关系中，所具备的能量、格局、视野或位置明显高于对方或市场基准时，你就拥有了势能差。

**公式**：`你的势能 > 对手/环境的势能 = 势能差`

这种能量优势让你能够更轻松地影响他人、引领潮流或达成目标。例如，在商业中，一个品牌因其更强的品牌势能而获得消费者青睐；在知识付费领域，势能高者向势能低者传递知识与经验，其本质也是势能差的传递。

**核心要点**：势能传递并非满分者教导零分者，而是遵循`90分 → 80分 → 70分 → 60分`的梯度模式。找到比你现阶段略高一层级的导师，是最有效的成长路径。
