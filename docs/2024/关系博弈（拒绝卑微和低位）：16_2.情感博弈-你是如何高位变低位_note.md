# 关系博弈（拒绝卑微和低位）：第16课 - 情感博弈：你是如何从高位变低位的 👑

在本节课中，我们将要学习在感情关系中，个体是如何从看似优势的“高位”逐渐滑落到被动“低位”的。核心在于理解情绪波动和被动接受心态如何导致关系主导权的丧失。

## 概述：高位为何会变低位？

即使你自身价值很高，也可能在关系中处于低位。除了价值不匹配，另一个核心原因是**情绪拉扯**。对方通过引发你的情绪波动，让你“上头”，从而投入更多情感筹码。这种持续的拉扯和投入，最终导致你变得被动。

在感情里，导致低位的常见现象就是**情绪波动导致的高开低走**。当你处在低位时，主要体现为你非常被动。明明对方可能并不出众，却能牢牢掌控你。而且对方的行为特别具有迷惑性。

你无法分辨他的真实目的，尤其是在情绪拉扯的情况下，你分不清他是想分手还是想控制你。说白了，他的目的就是让你绝对顺从，打破你的一切原则和底线。最终的结果是，他在感情中占据绝对的主导地位。

## 被动心态：卑微的根本原因

上一节我们介绍了低位关系的表现，本节中我们来看看导致这种结果的个人心态。关于那些在感情里被动的女生，如果你在感情里是个被动的女生，你就容易被高开低走，很容易卑微。

以下是“被动女生”的特征描述：
*   **等待追求**：你等着对方过来追你，等着对方主动付出，你全程都是被动接受。
*   **被动沟通**：对待问题的态度也是被动的。例如，你不想表达，希望他可以主动发现你的需求。如果他没发现，或经你提醒后才做到，你会觉得这不是他主动做的。
*   **错误等式**：你希望男生是主动的，并认为“主动用我喜欢的方式对我 = 爱我”。

这种女生在感情前期可以享受新鲜感的红利，可以在恋爱前期被动地高高在上，享受对方的付出。然而，基于你高高在上的权利，是对方主动给你的，自然他也可以收回。当你接受并习惯之后，他若收回对你的付出，你会无比痛苦。所以你看似主动，实际上是被动。这是你被动卑微的根本原因。

## 三个阶段：从高位滑向低位

下面我从三个阶段告诉你，你是怎么一步步从高位变低位的。

### 第一阶段：关系的开始与“假性高位”

两个人是怎么开始的？对方在这个阶段只有降低你的防备心，获得你的信任，才能获得一个可以跟你在一起的名额，才具备被你喜欢上，或是跟你在一起的条件。

**你看似主动高位，实际被动低位，因为对方在暗，你在明。**

在刚开始的时候，这个男人总体来说不如你，或者你看不上。你身边有很多这样的追求者，就是他的物质条件不如你，或者比他优秀得多的男生大有人在。你考虑跟他谈恋爱，都是觉得不怎么喜欢，就随便谈谈。

因为这种高位开始的感情最容易被逆转。本质就是当你身为一个强者，你会下意识看不起弱者。所以自然对他这种人没有防备心，不自觉地习惯接受，就陷入进去了。但是一个人不可能一直持续的高位，特别是，你又是一个在感情里被动接受的人。对方可以选择对你好，也可以对你不好。恋爱前期高位在你，后期等你依赖之后，高位自然在他那边。

### 第二阶段：前期相处的两种策略

前期在一起，主要看对方采取的两种策略。

**第一种情况：让你不设防的“白手套”**

他们在没有利益关系的人面前，极度随和，看起来非常好相处，非常人畜无害，极其有服务意识，完全让你感受不到他内心的目的性。其实随和只是他们的外表，像他们这样的人，其实往往对人际关系的要求是最高的。

他们的底层社交逻辑是：
```python
# 伪代码展示其策略逻辑
if 对你表现得极度柔和与友善:
    你的防备心降低
    你展现真实自我
    他深入了解你的弱点和敏感面
    if 你的价值符合他的标准:
        针对你的弱点展开进攻
    else:
        对你冷淡
```

他表面的随和不代表他骨子里真的随和。其实他们是有非常强的标准的。因为他们都非常有能力，所以他们的人际交往成本太高了。在他过往经历中，他深刻了解到，只有了解到这个人内心最真实的样子，他才能够去信任这个人，或是决定要不要跟他合作等等，这样才能达成他最终筛选的目的。

这种人在感情里，在前期跟你相处的时候，一定是对你非常友善，看起来特别好说话。等消除你对他的防备心之后，你对他展现真实的自我时，他就会开始针对你的弱点，或者是你内心敏感的一面去对你展开进攻，一拿一个准。其实他一开始通过各个方面已经了解到你这个人的价值是可以的，他才会对你进攻。他对你柔和，只是为了去了解你的内心弱点。

**第二种情况：扮猪吃老虎，抓住为数不多的机会**

你在刚认识他的时候，是没有想过要跟他在一起的，或者是他一开始在你面前的价值特别低。而且像他这种情况，在你身边一抓一大把。他在你这里是价值低的，自然而然你不会对他有任何的防备心。

因为你压根不会觉得他会给你带来什么伤害，你会被他的外表所欺骗，就会觉得这个人也就那样。你也觉得你根本不可能会喜欢上他。慢慢的，你就对他越来越不设防备心。

其实他想让你喜欢上他，或是想你上头，第一件事情要做的就是让你对他没有防备，没有隔阂，更不会有恶意。在此前提下，他才能有办法跟你更进一步。所以他对你的心态就是那种“无所谓，但友好”——就是你拒绝我也无所谓，我对你依然很友好，我也愿意对你一直主动。

他就是一直在等待一个机会，等待一个你有情感窗口，或是你比较脆弱的时刻。你平时在他面前或者跟他聊天，都是对他无所谓。但是在你有情感空窗期的时候，或者是你刚失恋在感情里受伤，需要陪伴疗伤时，你就会发现，其实他这个人还蛮不错的，发现他身上还是有很多优点的。于是你开始对他产生好感。

这是因为，一开始在你面前满分的人，慢慢的相处过程中，会有一些生活的细节，会慢慢的扣分。相反，那些一开始在你面前就不是那么优秀的人，或者是你没有看上的人，他其实是会突破你的心理预期的。就是你跟他相处久了之后，你发现他有越来越多的优点。

## 总结

本节课中，我们一起学习了情感博弈中从高位滑向低位的机制。我们首先分析了**情绪波动**和**被动接受心态**是导致地位丧失的核心。接着，我们剖析了“被动女生”的特征，指出等待和沉默并不能换来真爱。最后，我们通过关系开始的三个阶段，详细拆解了对方可能采用的两种策略——“白手套”的深入洞察和“扮猪吃老虎”的耐心等待——其根本目的都是**降低你的防备，深入了解你，并在关键时刻影响你的情绪和决策**。理解这些模式，是避免在关系中陷入被动卑微的第一步。
