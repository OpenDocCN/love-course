# 关系博弈（拒绝卑微和低位）：16_2.情感博弈-你是如何高位变低位

## 概述
在本节课中，我们将探讨如何在一段感情中从高位变为低位，以及如何避免这种情况的发生。

## 情感博弈中的低位现象
**以下是低位现象的常见表现：**
- 情绪波动导致的高开低走。
- 在关系中处于被动地位，对方能轻易控制你。
- 对方的行为具有迷惑性，难以理解其真实目的。

## 被动女生的特点
**以下是被动女生的特点：**
- 等待对方追求，被动接受对方的付出。
- 不主动表达需求，希望对方主动发现。
- 期望男生主动，认为主动等于爱。

## 从高位变低位的三个阶段
### 第一阶段：关系的开始
- 对方需要降低你的防备心，获得你的信任。
- 你看似主动高位，实际被动低位，因为对方在暗，你在明。

### 第二阶段：前期的相处
- **第一种情况：让你不设防的白手套**
  - 对方在无利益关系的人面前随和，让你感到舒适。
  - 消除你的防备心后，对方会针对你的弱点展开进攻。
- **第二种情况：扮猪吃老虎抓住机会**
  - 对方在你面前价值低，让你不设防备。
  - 逐渐展现优点，让你对他产生好感。

### 第三阶段：关系的深入
- 当你习惯对方的付出后，对方可能会收回，让你感到痛苦。

## 总结
本节课中我们一起学习了如何从高位变为低位，以及如何避免这种情况的发生。希望这些内容能帮助你更好地理解情感博弈，并在关系中保持高位。
