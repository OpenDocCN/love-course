# 【女生版-视频课】爱的吸引力：021.19.如何解决亲密关系中两个人无话可说的困境

🎼Baby think me。🎼How want you to be my best friend love。🎼的好子。大家好，我是米娅。对于刚进入一段亲密关系的情感小白或者是正在热恋的女生们。

看到这节课的主题，可能会不以为然，相爱的两个人怎么会无话可说呢？事实上，爱情开始的时候总是美好的，你们可能是这样的。宝贝儿，你今天的口红真好看，比昨天的还要美。或者亲爱的我想和你一起去看流星雨。然而。

随着时间的推移，两个人从激情期步入了平淡期，原本无话不谈的两个人渐渐变得无话可说。如果是情侣，每天可能说早安晚安吃了吗？这些日常问候，如果是夫妻成了最多聊一下孩子的生活搭子。

这个平淡期就像是我们亲密关系的一道分水岭，有些人选择了停滞不前，让感情在悄无声息中慢慢冷却，而有些人则选择了主动出击。他们明白，度过这个阶段并不仅仅靠惯性维持，更需要双方共同的努力和经营。

在了解平淡期之前，我们需要了解两个人之间的无话可说是怎么产生的。

## 一、生活节奏快，工作压力大

身处在激烈竞争的社会，大家的生活节奏普遍较快，工作的压力也很大。因此，回到家里更多的希望伴侣能够体谅自己，不要再给自己精神压力。

但生活中难免会产生矛盾。而解决矛盾，唯一的途径就是沟通。但累了一天的两个人似乎没有多少耐心来梳理矛盾产生的原因，尤其是女生往往更加情绪化，而面对女生的细节控和不满的情绪，男生渐渐失去了解决矛盾的动力。

久而久之，要么选择回避，要么挣个面红耳赤，因此就会变成无话可说。

## 二、经济能力增加选择

随着越来越多的女性走入职场，有了能养活自己的能力。那个嫁汉嫁汉穿衣吃饭的年代已经过去了，这样就造成了一种家庭责任的模糊感。男主外挣钱，女主内顾家的单一模式已不再是全部的婚姻模式。

所以很多女人对婚姻是女人的避风港这个信念有了新的要求，因为赚钱比伴侣多的女人不在少数，他们能自己买房子，因此，避风港不只是生存保障，更是一种精神的交流。

一旦伴侣表现出精神的懈怠，无话可说就来得快乐。

## 三、对男人的期望值过高

信息传播的速度越来越快，越来越便捷。看了网络上的那些完美老公带给女性的爱情体验，对标身边这个直男既没有生活的高品质，又没有格调，喜悦，荡然无存。

想改造这个直男，发现难上加难，无话可说，似乎已是必然。

## 四、同居让人提早进入准婚姻状态

在我的咨询案例中，时常会出现女生和刚认识几天的男生早早走入到同居的行列。由于没有法律的保护，很多都没有来自伴侣的经济和资源共享。

这种恋爱模式并不能像婚姻中的两个人一样，把逐渐褪去的激情，化为一种建立信任，紧记责任的爱情，而是早早的体验到了婚后的柴米油盐，激情很快被一地鸡毛代替。

重点是没有更多的价值来维系。比如共同的孩子，共同的利益。因此离开不甘心，继续过不好。久而久之，没有动力再去积极的解决无话可说的矛盾了，就都摆烂了。

## 五、和外界的诱惑相比，现实生活更显得索然寡味

面对网络海王虚拟的诱惑，甜言蜜语，堆砌如山，造成公式化的爱与泛滥，爱心变得不再真诚无价。网恋一夜情成了爱情快餐的当家菜，总会让人有尝鲜的冲动。

而家里的爱人则成了回锅菜，只有解宝之功，却无回味之能。对比之下，自觉乏味，看看也就厌了。当然也就无话可说了。

## 六、伴侣是否精神出轨

如果你发现伴侣仿佛断崖式的开始梳理自己，比如不再关心自己，回信息不再像以前那么及时，不管自己怎么引导，都不想和自己交流，回避话题，甚至吵架都懒得吵，但他和其他人去聊的热火朝天。

尤其开始和某个女生频繁互动。那么这就说明他起码是精神出轨了。

## 七、是否吸引力降低了

我们在前面的课程中反复强调过，两性之间靠吸引，吸引力使得双方在爱情初期产生了对彼此进一步了解和接触的欲望，从而关系升级，最终确认男女关系。

在你们的相处过程中，也会更加亲密和包容，但吸引力会随着时间的推移而发生变化。一旦女生的吸引力不足时，那么男生就可能表现出对你的无视和无语。

## 八、无话可说的两种情况

1. 两个人之间的矛盾呢还没有上升到原则性问题。在激情退去之后，生活开始变得规律而平淡。两个人都更了解对方，对方身上的光环，开始退去，暴露出来一些缺点。

话题开始变少。不再像以前那样频繁交流，不再像以前那样亲密的关心对方的生活情感。双方对彼此的热情呢有所减退，不再像以前那样充满激情和兴奋。

不再或许更多的考虑对方的感受，称呼对方的语气呢，也不像最初那般亲昵。这样的无话可说是可以被改变的，需要给你们的爱情加一些保鲜剂，让激情重燃。

2. 两个人之间感情基础薄弱，一方或者双方体验了新鲜感之后，发现并不合适，或者因为生理需求而在一起，没有真爱。这样的无话可说，就需要重新考量你们的关系是否需要继续，你要不要及时止损。

## 九、维持爱情的保鲜计

1. 控制激情的弥漫速度。

2. 制造一些属于你们的记忆。

3. 学会双向付出。

4. 要有彼此独立，也要有相互依靠。

5. 保持学习心态，一起制定共同的小目标。

6. 找到专属你们的沟通模式。

7. 爱一定要挂在嘴上。

感情在不断变化，但两个人在一起的初衷不会改变，永远记住你是为了让自己更幸福，让双方更幸福。所以想要继续经营这份感情，而绝不是你为了挽留他而委屈求全，经常反思复盘这份感情，时时督促自己提升进步，默默引导对方改变，相信你们一定可以重新找回那份最初的激情和浪漫。
