# 钱进社课程67：真正好的出路是“深度”，其他都是过眼云烟 🎯

在本节课中，我们将探讨在信息爆炸的时代，为何“深度学习”能力是个人成长与竞争的核心关键。我们将剖析浅层学习的陷阱，并提供一个清晰的学习金字塔模型，帮助你构建深度学习的路径。

## 概述：信息便利时代的认知陷阱

现在获取知识和信息的渠道非常多，也非常轻易。无非是你是否愿意去交付金钱和时间罢了。也正因为是这样，大批量的观点就会扭曲成：你如果是一个时间管理的强者，如果你是一个好学不放弃自己的人，你就要疯狂的接纳大批量的知识信息。数量成了最基本的1个PK。比如“我一年听365本书，你一年才两本，你太弱了，你不好学”。

而这种思维架构对于进阶和好学产生的只是错觉。人根源上的学习机制并没有随着时代变得更快。即便现在小孩越来越聪明，但是他们也不是超人。不会因为长在这个时代之下就可以跳过很多步骤，直接到达终点。我们大脑的运作模式并没有产生根源上的变化。

而更可悲的是，就因为现在有了更快捷的信息方式，所以逐渐在损耗人类深度学习的能力。并且这种倾向越来越明显。就像当你习惯性的在开车的时候开始听书，而到底转化了多少呢？可能是未知数。你只知道“我不要浪费碎片化的时间，我要学习”，听书更方便，不耽误做其他事，所以我就选择放着。到底有没有听进去，听进去了多少，转化了多少，可能都是未知。

各种事实都验证了，快捷、便利、轻松的方式，让人类的天性得到更好的释放。比如说急于求成，比如说逃避困难，本能地去接近更简单的。所以离“深度”这两个字就越来越远了，因为深度的能力几乎全部依赖于高级的理智。

上一节我们介绍了信息便利带来的认知误区，接下来，我们将深入探讨精英与大众在学习逻辑上的根本区别。

## 精英逻辑：直面核心困难

真正的精英和成事的人，做事的逻辑依旧没有发生太大的变化。依旧是需要直面核心的困难，努力并且进行深度的耕耘。而大多数信息获取者呢，开始逐渐依赖和享受轻度的学习，消费内容，昙花一现。

如果你是那个想要笑到最后的人，你靠什么后劲去跟人家拼持久爆发力？“轻松学习”是一件特别扯淡的事儿，这是一种特别扯淡的说法。学习就是苦的，它就是乏味的，它就是没有那么快的。你锤炼不出深度学习的能力，你不会逆流向上的，你更不可能成为那个最后稀缺的人。

我们看过很多信息，那么多知识内容，你有没有想过，在那些播放量的背后，是不是80%的人都在接近越来越平庸，但是他们不那么认为。而这里的“平庸”指的不是你的消费模式，不是指的是你住什么样的房，你开什么样的车。而是你的思维在逐渐的平庸。并且你不认为这是一种平庸，这才是最可悲的。每个人都要想清楚自己到底要过什么样的人生，这是很关键的。

理解了深度的重要性后，我们来看看“深度学习”具体由哪两部分构成。

## 深度学习的两个核心：主动 vs. 被动

那么什么是深度学习？主要有两个点：主动学习和被动学习。

被动学习指的是什么呢？比如说听演讲、听阅读、看视频等等。而这一套操作下来，真正的知识留存率可能只有 **10% 到 20%**。

主动学习是什么？结合了被动学习的方式，你是否通过讨论、互动、实践等等行为，让你吸纳的知识产生转化。那么留存率起码它至少得超过 **60%**。

这又要回归到我总是说的一句话了：你没有懂就是没有懂，不存在“道理都懂，但是我做不到”的说法。听的层面只有10%的留存，那你说你懂什么？你是超人吗？“听过很多大道理，我们依旧过不好这一生”，当然了，听过很多大道理，怎么可能过好这一生呢？听是最浅层的，只有10%的留存，你就有10%的留存，你居然想过好这一生，你想什么呢？

为了更直观地理解学习的层次，下面我将介绍一个五层学习金字塔模型。

## 学习金字塔模型：从浅层到深度

举个例子来说，就拿我们的音频来说，如果是一个金字塔式的学习模型，分5个层面。

以下是这个模型的五个层级，从最浅层到最深层：

1.  **第一层：获取 (听/看)**
    听或看这个层面，它的占比是最小的，也就是说在最上面的一层。你要先听到、先获取信息。获取的这个层面，是在金字塔的最上面一层。

2.  **第二层：搬运 (笔记和摘抄)**
    占比比第一层大一点，就是笔记和摘抄。因为你会为此做一些笔记和摘抄，等同于你做了一些知识的搬运。同时你在搬运的这个过程中，你加深了知识的记忆。

3.  **第三层：梳理 (制作思维导图)**
    你可能会为此做一批思维导图，来更清晰化的了解和细分知识。

4.  **第四层：内化 (独立输出笔记)**
    你自己输出一篇独立的笔记，完全是你个人吸纳角度的、对这个音频（或书本、演讲、视频等）的理解和延伸。理解不够，一定要有延伸的东西。你延伸不出去，就说明你的思维没有真的被打开，你一定不具备举一反三的能力。你对这个知识的转化可能只有一个及格分。如果你输出不出来，“深度”这一关，你在第四层就卡住了。你只能停留在第三层，比如搬运、思维导图这些。第四层开始就是要对标真正的转化，也就是说及格分以上的东西了。

5.  **第五层：实践**
    如果你上面第四层关卡卡住了，你会发现你无法带入实践。或者说到了实践，你脑子里面空的就像什么都没有一样，你的思维导图作废，你的笔记作废，然后你听的、看的、吸纳的全部作废。在第五层的时候，就直接反弹给你一个最终的反馈：以上全部作废。

那么我们倒推来看等于是什么？你这一系列的学习操作就失败了。因为最重要的、积淀最深的，是第五层。第五层的占比是最大的。所以你反推来看，可能你这一系列的学习操作就失败了，你只停留在浅层。即便我每天给你8个音频，你依旧是没有戏的。

了解了模型后，我们来看看这个模型在现实中的具体表现。

## 现实差距：为何同样的输入，结果不同？

这种现象很常见。就像我们在群内交流，或者说我在直播的时候，我说到某一个点，我们群内有的小伙伴能马上精准的说出音频中的内容，而有的小伙伴一脸懵逼。他此刻的脑子可能是快速运转的，但啥也想不起来。

大家都在一个群里面，如果大家都领取了音频，那么大家应该是都听了。无非是有的人听了十遍，有的人只听了一遍；有的人是认真地听，有的人是洗脸刷牙、上厕所的时候听。无非就是这些差距。可是最终的结果是，有的小伙伴是可以精准的马上说出音频的内容，而有的小伙伴一脸懵逼，仿佛他刚刚才待到这个环境里面。

同样，大家都完成了最浅层的“听”，为什么差距这么大？难道是大家的耳朵大小长得不一样造成的吗？就是“深度”这个环节大家是截然不同了，造成的结果是截然不同的。同样的时间，同样的音频，在不同的人身上就是会有不一样的结果。

所以为什么我总是说，大家如果都是同事，大家都在一个起跑线上，别人跟你的差距越拉越大，你不要总是去找别人其他的原因。你就看一下你能做到的层面，你跟别人的差距有多大。不要赖他是不是会拍领导马屁这些外部的因素。就说大家同样是在做一件事情的这个单纯的PK上面，你跟人家的差距是多少？

就像大家都在听音频，为什么有的小伙伴可以精准的说出音频的内容？有的小伙伴会一脸懵逼呢？那如果说这是一个职场，能快速找到音频中的内容并且说出来的人，他升职了；一脸懵逼的人，你没有升职，你凭什么怪他？你应该怪你自己啊，怎么你就一脸懵逼呢？

认识到差距后，我们必须警惕一种流行的、却有害的学习观念。

## 警惕“低水平勤奋”

所以说，我根本不学那种，就包括小红书上面那种什么“我是如何快速高效阅读的”、“我是如何一年完成365本阅读量的”。我也希望我们所有群里的小伙伴不要学这种。被动学习是最浅层的，不可能让我们成为真正稀缺的人。能让我们成为真正稀缺的人是“深度”。

当你默认为“多听快学”这种模式是一种非常勤奋的象征的时候，你就更分不清好坏了。因为你觉得你在做一件非常正向、非常正确的事情。但更可怕的是，低水平的勤奋，投入越多，你的损失就会越大。

那么为什么我刚才罗列的这个学习的金字塔架构里，构建思维导图，它也只在第三层呢？虽然这属于主动学习的一部分，但是呢，还是停留在“陈述”的层面。

就像有的小伙伴他其实还蛮依赖思维导图的。我更见过有一些读书社群，里面的小老师拆一本书之后，把这本书做成一个思维导图，然后发给他的付费学员，告诉他们：“你们看这个思维导图就OK了。”然后所有的学员就会觉得他们自己赚大了，他们可以用最少的时间成本、最少的精力成本，完整的吸纳到这本书的精华，他们觉得自己简直太赚了。而这种东西是推动人性的，就很多人真的会觉得自己赚大了。

所以为什么我会把思维导图也加进这个学习的金字塔里边？因为有的小伙伴他确实是挺依赖于这个东西的。但是你看，我把思维导图这个东西，也只放在金字塔学习模型中的第三层。第四层才开始最重要，它才第三层，也就是说它在一个中庸地带。虽然这属于主动学习的一部分，但是还是停留在陈述的层面。它很有必要，但是他不决定最终的稀缺性，也不决定最终的结果。离转化还间隔很大的一个台阶。

为了真正达到深度，我们需要进行一个关键的训练：从“知道”到“能陈述”。

## 关键训练：从“知道”到“能陈述”

我们试想一下，如果你只停留在陈述的层面，它能转化吗？就像我在工作中，就会让我的同事做两个步骤：第一，陈述一遍我说的；第二，附上他自己的想法和他自己独立的东西。这两个步骤，每一个步骤我自己就可以捕捉到不同的反馈。

为什么我们做事的最终结果有本质的区别？就是在深度上面，人和人有本质区别。以及，我会刻意的让我的同事觉得他自己很明白的事情，清楚的陈述一遍给我听。得到的结果就会存在：他明明心里面觉得他自己完全明白了，特清楚了。但是如果他要把这个步骤走完——清晰的陈述一遍给我，以及说服我的时候，可能他就会卡住。

这不是一个小概率的现象。他自己很明白，很清晰。但说“好，你陈述一遍给我，然后你通过你陈述的方式说服我”，可能他就会卡住了。可能有小伙伴说，那是不是口才的问题？并不一定，不全是。你就算不说，你写出来可能照样会卡住，或者说一动笔的时候就已经卡住了。

这个“卡住”，因为我们知道，把一件事情陈述出来，或者说把一件事情通过笔的形式写出来，它都是需要思维做支撑的。所以，“你自己知道”，那只是在你心里面知道了。为什么他陈述出来，或者说他写出来、说出来他就会卡住？输出的这一部分卡住，那就说明输入的这一部分，并不是你想象中的“你都懂了”。

而往往这两步操作，很多小伙伴是合二为一的。它不是两步走的，它是一步走的。这就会造成很多小伙伴“我都懂了，我都懂了”。好，你告诉我你怎么懂了？哎，卡住了。他可能自己都怀疑自己了：“我没懂？我明明都懂了呀。”这就是两个步骤，不能合二为一的。

为什么会存在这种现象呢？因为“交付”和“自身过滤”本身就是两个体系。就像“听”和“实践转化”也是两套体系一样。听多简单啊。而想要和“深度”这两个字挂钩，你先问问你自己过程中，你觉得有没有难度？如果你觉得好像没啥大难度，那么只存在两个可能：一、你智商确实是高，堪比超人；二、你只在浅层，你当然觉得没难度了。

那么，如何将知识真正内化呢？核心在于“缝补”。

## 深度核心：知识的“缝补”与串联

如果你能把新吸纳的知识解释清楚的时候，就等于吸纳了你自己的知识体系。你是归类、融合了你自己的知识体系。既然已经纳入了你自己的知识体系，你在表达给他人的时候，是不是就应该可以很顺畅、很有逻辑了？

深度的前提是，我们每个人都要学会做“缝补”，和你的过往认知做一个缝补，把它串联起来。碎片化的思维最大的诟病是什么？就是无法顺利的缝补，东一榔头西一锄头。我们是成年人了，我们如果还在用这种方式的话，这是小朋友的惯性思维。如果你只侧重在获取新知识，却没有完成缝补，你就没有办法达成“深度”这两个字。而缝补的过程也是痛苦的过程。你如果完成了缝补，你就在金字塔占比最重的那一层了。

所以，“一年365本书”，你还能把它缝补好，你还能串联起来，你确定你是超人中的超人吗？

我其实在过往的音频里面说过，如果你只是把群里面的音频当做“听说软件”一样去对待，一年以后，或许你是不需要再继续待着了。我给到所有音频内容都是经过一轮又一轮的深度缝补，结合了实践、试错等等，再反输出的内容。而到了你这儿如果变成了浅层输入，那么等于价值没有被放大，反而被缩小了。对于你来说，你是亏大了。

如果你听的所有的东西不能落地，不能深度思考，形成你自己的输出，所有的知识在你这里就会烟消云散，留不下什么痕迹的。

基于以上分析，我们可以总结出深度学习的核心公式。

## 深度学习的公式与实践方法

所以，深度学习的公式是什么？
**深度学习 = 获取高质量知识 + 深度缝补与串联 + 独立语言总结与复盘**

如果你只知道复盘笔记是远远不够的。笔记之后需要有你完全独立的一份笔记。如果说你只是复盘，重复复盘的笔记是属于搬运类型的，不是独立输出类型的。那么你只复盘这一份搬运的笔记是不够的。这一份搬运的笔记只能在你人生的不同阶段，对应起到不同的启发和警醒。但是你真正复盘，是需要附上你自己“脱稿式”的笔记一份的。独立输出的笔记要有一份的，这才是完整的复盘。

为什么我那么多年保持手写习惯？你一旦开启了独立缝补和串联的这个行为方式，你就会发现，“快学”、“多学”根本就是没有安全感的。你需要花费更多的时间去面临更难的处境。那如果说你本身自己的积淀还没有达到能承载这么多的话，你会发现你越学越乱，简直是一团乱。然后你发现你好像啥也没学会，但啥都好像也知道点，那有什么用呢？

现在便捷的学习方式是在迎合人的天性的，所以才会发展的如此迅速，包括了商业变现。但是，真正有用的、深度的交织是反人性的。也就是你觉得你越不想做的，它越有用；你觉得越麻烦的，它越有用；你觉得越不能见缝插针去做的，越有用。

那么，这种困难、不舒适，我们如何通过具体方法去逐一改善呢？以下是三个核心方法。

### 方法一：获取一手知识
获取一手的，而不是二手的。比如我们的音频，你自己去听，不要看其他小伙伴的1比1的还原笔记。二手知识必然是被“剥削”过一层的，你觉得轻松愉快并不是一件好事情。即便是一比一的还原笔记，和你一手的拿到一个独立的音频去磨、去听，也是两回事。

### 方法二：用自己的话脱稿输出
用自己的话把所吸收的知识写出来。很多小伙伴有一个误解，以为复盘就是只是重复看搬运式的那一份笔记。而复盘很重要的环节是什么？是脱稿。就是你自己要写一篇总结出来的，不依托于搬运的笔记，不依托于音频、书本，不依托于任何东西，完全用你自己的理解，写一篇东西出来。你今天写出这一篇，不要修改。一个月以后，两个月以后，落地到你的实践，再去复看你自己写的这篇东西，再写一篇出来，形成对比。这才是复盘的迭代。

就像我会问我的同事问题，我说：“我不想要听你再告诉我一遍，我是怎么说的。我想要听一听的是，你听了我说的之后，你想怎么说，你想怎么做。”这就是在走复盘的路，是要脱稿的。人一定要形成自己独立的思维方式，然后进行陈述。你可以用笔陈述，也可以用嘴陈述，可以打字、手写都OK。但是必须是得脱稿的、独立的、依附于你自己的思维方式的。这才是复盘的迭代，而这个环节是不可以遗漏的。

### 方法三：将反思落地输出
反思。我个人认为我是属于将反思这个事儿贯穿进生活的一个人。而我的婚姻、我的个人生活以及我的工作能不胡乱跑偏，我觉得和反思息息相关。我相信很多小伙伴也会反思，为什么你的反思不能起到决定性的作用？因为你只在脑子里“兜”。而我是什么？我是落在输出上面。因为我做的事情需要我输出——我的思维输出，我的行为输出，我的语言输出。

重点是在于“输出”你的反思的每一个细节，不然你是看不到事实的。如果你只是在脑子里面、心里面兜的话，多半来说，特别是女性小伙伴，你反思的就是个“情绪”。如果你不落地，不落在纸笔或电脑键盘上，你的反思就是在心里面和脑海里面兜一兜而已。你觉得你在给你自己清理负面数据，但其实清了半天，我们重的是“事实”，但是可能在你这儿清了半天，就是“情绪”。

人和人的差距如果不是直观的，那么必然是潜在的。直观的指的是智商、年龄、社会阅历，这些差距是非常直观的。潜在的指的是什么？反思以后得到的升华。一步升华怎么可能？顿悟的前提，咱们先说是不是反思。一个人没有反思这一步，他不可能顿悟。**反思 -> 顿悟 -> 升华**，三个步骤，反思在第一顺位。反思没到位，顿悟不可能，升华想也别想。你一步到位到升华了？你连顿悟都没做好，你还升华？三个步骤，一个都不能跳。

为什么很多小伙伴饭也没少吃，日子也没白过，但是依旧感知力非常弱？我相信反思这一关，要么不做，要么一定就是坐在云端飘着的、不落地的。就觉得“好像我平时自己也挺爱琢磨的”。但是你问他：“你具体怎么琢磨的？你这个以琢磨为目标的行为，最后落地的行为反馈是什么？”——“我就是发发呆的时候想一想”，或者“我坐车的时候想”。有什么用啊？没有用的。我的感知力的培养，一大部分来自于日复一日的反思行为，这一点都不夸张。所以重点又来了：**日复一日**这四个字，而不是短期内的“我去做了，我反思了”。一个星期能有一次？好像有吧。就呈现这种状态，然后就没有然后了，那有什么用啊？

## 总结：深度是稀缺性的来源

你如何去接近“深度”呢？“深度”这个词我相信很多进阶的小伙伴都想达到的，但是里面有很多很多的步骤。感知力也是深度的一部分，学习的方式也是深度的一部分，反思的方式也是一部分，复盘的侧重也是一部分。深度是直接对标我们是不是能真正进阶的这两个字。也就是说，真正的稀缺性。不是所有的人都有深度的，那么有深度的人是不是就有稀缺性呢？你有了稀缺性，那么最后的进阶是不是就能跟你有关，你就可以甩开你的对手呢？
