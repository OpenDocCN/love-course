# 男哥内核3.0初级班：内部课：20.飞龙在天系统延伸拓展（五）红绿灯系统

## 概述
在本节课中，我们将学习飞龙在天2.0系统的红绿灯系统，这是一个基于窗口识别的进阶技巧，旨在帮助学员在恋爱和婚姻中提升交流能力和表达个人魅力。

## 红绿灯系统核心概念
红绿灯系统通过评分机制，将女生的行为分为红灯、黄灯和绿灯，从而判断其对你的动力程度。

### 动力识别系统
动力识别系统是红绿灯系统的核心组件，它通过以下因素评估对方对你的动力：
- **直接问你的信息**
- **回复速度**
- **发照片**
- **撒娇、调侃、任性、抱怨等情绪表达**
- **字数和行数变化**
- **表情包**

### 评分标准
- **字数和行数**：根据对方之前与你的交流情况，进行相对值评分。
- **表情包**：根据表情包的类型和情感表达进行评分。
- **回复速度**：根据回复速度的快慢进行评分。
- **其他行为**：如发照片、撒娇等，根据行为出现的频率和强度进行评分。

### 红绿灯状态
- **红灯**：表示对方对你的动力不足，需要采取措施提升。
- **黄灯**：表示对方有一定的动力，但需要进一步引导。
- **绿灯**：表示对方对你的动力充足，可以大胆推进关系。

## 实战技巧
- **病毒聊天**：通过引导对方多聊话题，激发其行为上的反应。
- **引导行为**：通过提问、描述情境等方式，引导对方表达更多情感和行为。
- **案例分析**：通过分析聊天记录，学习如何进行评分和判断。

## 总结
本节课介绍了飞龙在天2.0系统的红绿灯系统，通过评分机制帮助学员判断对方对你的动力程度，从而在恋爱和婚姻中提升交流能力和表达个人魅力。学员需要掌握评分标准，并通过实战练习提升应用能力。
