## 概述：什么是情绪价值？

## 如何理解情绪价值？

## 情绪价值的深层逻辑

上一节我们看了具体问题，本节中我们来看看背后的心理机制。现代许多女性的状态是：渴望一天24小时中，有人填补作为个体的空虚感。

**情绪价值** 的核心公式可以概括为：
**情绪价值 = 我和你在一起时感到真切的开心与满足**

当一个女生问你“你会一直陪着我吗？”，其潜台词是：“你让我很开心，所以我想一直拥有这种开心。”
