## 73855法则
**核心概念**：7%说什么，38%怎么说，55%谁说的

1. **说什么**：内容占比7%，即信息传递。
2. **怎么说**：技巧占比38%，即表达方式。
3. **谁说的**：身份占比55%，即个人形象和角色。

## 高分男生的定义
**核心概念**：根据外貌、经济条件、社交能力等因素，将男生分为青铜、白银、黄金、钻石、铂金、星耀、王者七个段位。

1. **青铜**：外貌不佳，社交能力差。
2. **白银**：外貌普通，社交能力一般。
3. **黄金**：外貌顺眼，但经济条件一般。
4. **钻石**：外貌顺眼，有一定经济基础。
5. **铂金**：身材好，五官精致，但不会聊天。
6. **星耀**：帅哥，有点小腹肌，有点小富有。
7. **王者**：高富帅，明星脸。
