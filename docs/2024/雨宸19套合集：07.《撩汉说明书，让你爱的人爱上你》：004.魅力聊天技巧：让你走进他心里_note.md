# 雨宸19套合集：07.《撩汉说明书，让你爱的人爱上你》：004.魅力聊天技巧：让你走进他心里

## 概述
在本节课中，我们将学习如何运用魅力聊天技巧，让对方对你产生兴趣，走进他的心里。

## 如何选择话题
**以下是选择话题的步骤：**
1. **观察身体语言**：注意对方的身体语言，如面部表情、头部动作和身体转向，判断对方是否感兴趣。
2. **转移话题**：如果对方对当前话题不感兴趣，可以尝试转移话题或做其他事情，如看手机。
3. **寻找共同话题**：对方会主动找话题，如果没有，可以尝试寻找共同兴趣。

## 自我暴露
**自我暴露的四个层次：**
1. **兴趣爱好、生活习惯**：分享你的兴趣爱好和生活习惯，让对方了解你。
2. **态度**：分享你对某些事物的看法和态度，让对方了解你的价值观。
3. **自我意识和个人人际关系**：分享你的家庭和朋友关系，让对方了解你的生活背景。
4. **隐私**：分享一些不为大众所知的秘密和想法，拉近彼此的距离。

## 操纵对方的自动揭示
**以下是一些操纵对方自动揭示的方法：**
1. **无声的赞美**：通过眼神和肢体接触，让对方知道你认同他并理解他的观点。
2. **共情**：在对话结束后，表达你理解对方的心情，并创造共同笑点。
3. **分析对方的理想自我认同并赞美他**：支持并剖析对方的理想自我状态，并赞美他。
4. **斯波克法则**：主动告诉对方你喜欢并欣赏他的哪些地方，让他继续做让你喜欢的事情。

## 聊天原则
**以下是聊天的原则：**
1. **情感独立，有界限感**：建立自己的框架，保持情感独立，不要过度干涉对方的事务。
2. **降低欲望**：不要期望一次聊天就能让对方非你不可，循序渐进地建立关系。

## 总结
本节课中，我们学习了如何运用魅力聊天技巧，让对方对你产生兴趣，走进他的心里。通过选择合适的话题、自我暴露、操纵对方的自动揭示和遵循聊天原则，我们可以更好地与对方沟通，建立良好的关系。
