## 什么是情感？

情感是人类对内部或外部刺激产生的复杂心理与生理反应。它不仅仅是感觉，更涉及认知评估和身体变化。例如，当遇到危险时，我们不仅感到“害怕”，身体也会进入“战斗或逃跑”的应激状态。

上一节我们介绍了课程的整体目标，本节中我们来看看情感的具体定义与构成。

情感通常包含三个核心成分：
1.  **主观体验**：个人内心的感受，如快乐、悲伤。
2.  **生理反应**：身体伴随情感发生的变化，如心跳加速、出汗。
3.  **行为表达**：情感的外在表现，如微笑、哭泣。

## 为什么情感如此重要？

理解情感的重要性是运用它的第一步。情感对我们的决策、人际关系和身心健康有着深远的影响。

以下是情感的主要功能列表：
*   **行动指南**：情感如恐惧能促使我们避开危险，是快速的生存信号。
*   **沟通桥梁**：通过表情和语气，情感帮助我们传递意图、建立连接。
*   **决策依据**：情感为理性判断提供价值参考，例如“直觉”或“好感”。
*   **内在动力**：强烈的情感（如热爱、渴望）能驱动我们追求目标。

## 如何识别自己与他人的情感？

识别情感是情感管理的基础。这需要我们对内进行自我觉察，对外进行观察与共情。

上一节我们了解了情感的功能，本节中我们来看看如何具体地识别情感。这可以通过两个主要途径实现。

**1. 自我觉察**
关注自身的生理信号和思维模式。可以定期进行“情感检查”，问自己：“我现在的身体感觉如何？我脑子里反复出现的想法是什么？” 这有助于将模糊的感觉转化为明确的情感标签。

**2. 观察与共情**
通过观察他人的非语言信息来识别其情感。请注意以下几点：
*   **面部表情**：微表情往往能透露真实情感。
*   **肢体语言**：姿势、手势和动作的紧张或放松程度。
*   **语音语调**：说话的音量、速度和语气变化。

> 共情不是猜测，而是基于观察，尝试理解对方情感背后的原因。

## 核心情感模型：普拉奇克的情感之轮 🌈

为了更系统地理解情感，我们可以借助一些心理学模型。罗伯特·普拉奇克的情感之轮是一个经典工具，它将核心情感分为八种，并展示了情感之间的对立与强度关系。

其核心关系可以用一个**公式**表示：
**基本情感 + 强度变化 = 复杂情感**
例如：**喜悦（高强度） = 狂喜**， **喜悦 + 信任 = 爱**。

以下是该模型的八种基本情感配对：
*   喜悦 😄 与 悲伤 😢
*   信任 🤝 与 厌恶 🤢
*   恐惧 😨 与 愤怒 😠
*   惊讶 😲 与 期待 🧐
