# 性魅力提升系列课—无限话题聊天训练从零起步7天社交达人速成 _ _欧阳梦琪：033-自信的塑造

🎼欢迎回来。在大学里观看辩论比赛时，我发现有些人讲话时身体微微前倾，处于紧张状态，连续10分钟仍无法表达清楚。有些人眼神躲闪，不知看向何处，这些都是不自信的表现。吸引是人际交往的前提，而吸引的前提是价值、能量和自信。自信是魅力之源，是打开吸引力开关的必要条件。

**建立自信的8个方法**

1. **直面恐惧**：每个人都有难以逾越的障碍，可能是童年的阴影或过去的创伤。接纳过去，活在当下，未来才会更幸福。
2. **擅长之事**：优秀的人不是全能的，而是做自己擅长的事情的人。明确自己的优势，做自己擅长的事，你就已经赢了90%的人。
3. **不过于敏感**：不要怕丢脸，很多事只要你做了，就会有意想不到的回报。不做的话，就永远只能看着别人成功。
4. **外在影响内在**：从小事做起，如每天洗头、保持形象清爽、抬头挺胸、与人四目相对，你会发现自己的眼神越来越自信，心态也越来越平和。
5. **挖掘潜力**：通过行动发现自己更多的潜能，不断更新对自己的认知，你会知道你比想象中的自己更强大。
6. **增长见识**：多看书，多走出去看看这个世界，了解自己没去过的地方，而不是困在自己的小圈子里。
7. **学会承担责任**：敢于负责任，把原本准备推卸的责任收回来，放到自己肩上。主动承担责任越多，心智成长的速度就越快，影响力就越大。

**课后总结**

本节课我们学习了如何建立自信。自信是建立吸引的必备条件，所以建立自信就成了实现精神与社交自由的必要条件。我们给出了7个方法，分别是直面恐惧、擅长之事、不过于敏感、外在影响内在、挖掘潜力、增长见识、学会承担责任。当你拥有了自信，你身上的吸引力自然会一打，社交自由之路也会走得更加顺畅。
