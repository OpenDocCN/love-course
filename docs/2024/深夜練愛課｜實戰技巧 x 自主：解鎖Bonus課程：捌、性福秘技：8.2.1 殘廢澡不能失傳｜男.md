## 上半身清洁：从牵手到若有似无的触碰

上一节我们介绍了课程概述，本节中我们来看看如何开始。首先，让伴侣坐着或站着，站立姿势通常更方便操作。由于示范使用的是无法独立站立的娃娃，因此以坐姿进行演示。

以下是上半身清洁的具体步骤：

1.  **起始于牵手**：可以先自然地牵起对方的手。
2.  **延伸至手臂与腋下**：从手部开始，用沾有沐浴露的手或沐浴球，缓慢向上移动至腋下进行清洁。
3.  **清洁颈部与耳后**：接着清洁脖子，务必注意清洗耳后，这是许多男性容易忽略的部位。
4.  **胸前的技巧性触碰**：清洗胸前时，可以一只手用沐浴球清洁，另一只手的虎口可以**似有若无地轻轻夹碰或抚过乳头**。许多男性喜欢这个部位的触碰。
5.  **制造“意外”触碰**：技巧在于，让触碰看起来像是清洗过程中的自然接触。例如，手掌下滑时手心“不经意”拂过乳头，或再次清洗时用虎口轻轻夹住。

在清洗上半身的过程中，手部动作可以偶尔向下，仿佛快要触碰到私密区域，再自然收回，以此营造期待感。

## 下半身清洁：视角与触感的双重享受

完成了上半身的清洁，接下来我们聚焦于下半身。为保护腰部，建议采用蹲姿而非弯腰进行清洗。

以下是下半身清洁的具体步骤：

1.  **从脚踝与脚趾开始**：许多男性洗澡时会忽略脚部细节。请从脚踝开始，仔细搓洗，并清洁脚趾缝。
2.  **小腿的清洁与抚摸**：清洗小腿时，一只手用沐浴球清洁，另一只手可以同时上下滑动抚摸皮肤，两种触感交错进行。

## 背部清洁：结合按摩的放松时刻

在进行了正面细致的清洁后，我们转到背面。背部清洁是帮助对方放松的绝佳机会。

以下是背部清洁的具体步骤：

1.  **环抱式清洁**：一只手使用沐浴球清洗背部，另一只手可以环抱住对方的腰部以保持稳定和亲密。
2.  **重点搓背与肩颈按摩**：男性通常很喜欢搓背。在清洁时，可以重点搓洗背部，并顺势按摩脖子和肩颈部位。
3.  **提供个性化服务**：可以轻声询问对方是否需要针对某个背部区域进行按摩。无需用力，短暂的按压就能带来显著的放松感。
4.  **臀部清洁注意事项**：清洗臀沟时，动作要轻柔，用沐浴球轻轻带过即可。如果对方害羞，这个部位可以交给他自己处理。臀部两侧等较大面积处可以多清洗几下。

3.  **清洁阴茎与龟头**：

    *   包皮褪下后，重点清洁**冠状沟**。此处易积累尿垢和分泌物。可用双手大拇指指腹在冠状沟周围滑动清洁，也可使用食指或小指辅助，直至没有滑腻感。
4.  **龟头的刺激与清洁**：

    *   右手手掌以画圈或类似写字的方式摩擦龟头。公式为：`手掌画圈摩擦 = 清洁 + 刺激`。

5.  **结合按摩**：此时，清洁可转化为前戏按摩。一只手继续刺激阴茎，另一只手可按摩阴囊及阴囊后方的会阴区域，交替反复进行。
