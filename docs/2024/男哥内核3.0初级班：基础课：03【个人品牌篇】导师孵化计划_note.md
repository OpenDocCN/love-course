# 男哥内核3.0初级班：基础课：03【个人品牌篇】导师孵化计划

## 概述
在本节课中，我们将学习如何打造个人品牌，特别是展示面的重要性。我们将探讨如何通过照片和视频展示自己，以及如何利用光线和线条来吸引他人的注意。

## 展示面核心
**核心概念**：展示面类似于产品的包装，其目的是吸引他人的注意并给予时间探索你。

**公式**：展示面 = 包装 + 价值传递

## 搭讪与展示面
**以下是搭讪与展示面相关的内容**：

- 搭讪是认识新朋友和潜在伴侣的重要途径。
- 搭讪的数量和质量都很重要，建议每月至少搭讪70人。
- 展示面应体现出你的个性和价值，避免千篇一律。

## 公交车上的展示面
**以下是公交车上的展示面相关的内容**：

- 在公交车上搭讪时，首先要考虑自己的形象和状态。
- 如果形象良好且自信，可以尝试进行眼神交流。
- 下车后可以上前打招呼，表达认识一下的意愿。

## 展示面的变化
**以下是展示面变化相关的内容**：

- 展示面从过去的华丽和虚假，逐渐转变为真实和有价值。
- 现在的展示面更注重传递价值和个性。

## 如何吸引他人
**以下是吸引他人相关的内容**：

- 通过模仿女性用户的使用习惯，了解她们喜欢什么样的展示面。
- 选择清晰度高、光线充足的照片，避免过度使用滤镜。
- 在照片中寻找光和线条，使照片更具吸引力。

## 照片拍摄技巧
**以下是照片拍摄技巧相关的内容**：

- 使用高像素手机，确保照片清晰度。
- 尝试在光线充足的地方拍摄，如夜晚的CBD或步行街。
- 利用对角线拍摄，使照片更具动态感。

## 微信朋友圈布局
**以下是微信朋友圈布局相关的内容**：

- 微信朋友圈是展示个人品牌的重要平台。
- 在朋友圈中发布高质量的照片和视频，展示你的生活状态和价值观。

## 总结
本节课中，我们学习了如何打造个人品牌，特别是展示面的重要性。通过学习如何拍摄吸引人的照片和视频，以及如何利用光线和线条，我们可以更好地展示自己，吸引他人的注意。希望同学们能够将所学知识应用到实际生活中，提升自己的个人品牌。
