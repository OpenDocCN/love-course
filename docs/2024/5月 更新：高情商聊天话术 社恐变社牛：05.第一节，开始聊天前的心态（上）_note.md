## 心态准备：平等与筛选

上一节我们介绍了课程的整体框架，本节中我们来看看聊天前最关键的心态准备。许多人在聊天中无法取得良好进展，根本原因在于心态的错位。

### 1. 建立平等心态，而非讨好

首先需要明白，要让对方喜欢你，双方的位置必须是平等的，甚至你可以略高于对方。许多聊天失败的原因在于，整个聊天的出发点只是为了讨好对方。

**错误心态公式**：`聊天 = 讨好 + 打动 + 迎合`

如果你的聊天只是为了打动或迎合对方，把自己放在一个很低的位置上，那么无论你学到多少技巧，失败的概率依然非常高。你不能把自己放在一个需要对方“搭理你”的乞求者位置上。

这就像消费者选择商品。人们选择苹果手机，并非因为其服务最好或功能最全，而是因为它在人们心中代表了**高价值**和**稀缺性**。同样，在人际吸引中，价值感至关重要。

**核心心态转变**：
*   **目的不是**：通过聊天去打动、跪舔或感动对方。
*   **目的应是**：双方处于**相互平等**，并**互相了解**的状态。

聊天只是为了互相了解，而不是去做很多事情试图感动对方。后者透露出的潜台词是“我配不上你”，这一定会破坏吸引力。因此，切勿努力讨好或自我贬低。如果初始行为就错了，后续学习再多话术技巧也收效甚微。

### 2. 明确聊天核心目的

当我们接受了上述平等心态后，接下来要明确早期聊天的具体目的。

请问，当你刚认识一位姑娘时，聊天的目的是什么？答案不是让她立刻喜欢上你。

**早期聊天核心目的**：`把她约出来`

在你们尚未能经常单独约会之前，你的唯一目的就是创造机会约她见面。许多人错误地认为，前期聊天就要让对方喜欢上自己。这个想法会导致你过于努力地展示自己。

**过度展示的潜在含义**：`你需要她 > 她需要你`

这传达出你把自己放得比她低，需要去讨好她，从而破坏了我们强调的**平等**基础。只要关系不平等，聊天就很难顺畅。

回想你过去失败的聊天经历，无论是没能约出对方，还是聊天气氛很差，基本上都犯了把自己摆得太低，或太想展示自己以求对方喜欢的错误。

在对方眼中，这样的你显得缺乏价值、缺乏异性选择、甚至有些饥渴和黏人。一旦对方形成这种感受，再多的聊天技巧也难以弥补，因为基本面已经错了。

### 3. 建立“筛选”心态

那么，真正需要的心态是什么？不仅是互相了解，甚至你也在**筛选**她。因为她同时也在筛选你。

你要认识到，你心仪的漂亮姑娘通常不乏追求者。她会在众多追求者中筛选出她认为最合适、最有价值的人。如果你表现得像个“来者不拒”的追求者，你就会被筛掉。

**关键心态**：`你也在筛选她`

你不能抱着“是个女的就行”的心态。你需要有自己的标准和框架。

举例来说，一个真正不缺追求者的美女在社交资料上，不会罗列自己的优点（如温柔、会做饭、有房有车）。因为她不缺，所以她只会做一件事：**提要求**。

她会列出筛选标准，例如：“35岁以上勿扰”、“年收入低于XX万勿扰”、“身高低于XXX勿扰”。处于强势地位的人（无论男女），行为模式都是**筛选**而非**展示**。

所有不缺异性资源的男人也是如此。以我为例，我明确知道自己想要什么样的伴侣：外貌过得去，但最重要的是心智成熟、善于沟通、愿意为感情切实付出。正因为我有这些标准，遇到不符合的异性，我会保持距离或直接表明不合适。结果，对方反而可能更想吸引我的注意。

**心态对比公式**：
*   **有标准**：`你 = 高价值、不缺选择`
*   **无标准**：`你 = 低价值、饥渴、易讨好`

当你越有标准，在对方眼中你越是不缺选择的人。反之，如果你只要对方漂亮就无底线迎合，那么再高深的聊天技巧也救不了你。

### 4. 如何传递“筛选”信号

你需要让对方知道你有自己的标准，并且你曾因这些标准拒绝过一些人。这能立刻将你与那些无底线讨好的追求者区分开来。

这考验你“编故事”的能力。你可以分享一个过往的经历来传递你的标准和原则。

**示例故事（可改编）**：
> “之前有个女生跟我聊得还不错，但她有个行为踩到了我的原则。有一次我问她一件很正常的事，她一直没回，却发了一条朋友圈。我觉得这样很不尊重人，后来就没再继续了。”

通过这样的分享，你让她知道：1. 你是有标准的；2. 你曾因标准拒绝过别人；3. 你和那些只知道跪舔的男人不一样。

## 本节总结

本节课中我们一起学习了开始聊天前必须夯实的两个核心心态：

1.  **平等心态**：聊天是为了**互相了解**，而非讨好、迎合或打动对方。双方地位平等是有效沟通的基础。
2.  **筛选心态**：你与对方是**互相考察**的关系。你需要有自己的标准和框架，并让对方感知到这一点。这能有效提升你的价值感和吸引力。

记住，妹子时刻都在筛选你。如果只有她在筛选你，而你对她没有要求，那么在她心中你就是没有价值的。建立平等、带筛选的心态，是开启一切有效聊天的第一步。
