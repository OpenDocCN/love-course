# 高情商聊天话术 社恐变社牛：05.第一节，开始聊天前的心态（上）

## 概述
在本节课中，我们将探讨在开始与女生聊天之前，你需要具备的心态和认识上的转变。

## 核心概念
**公式**：平等心态 + 筛选心态 = 高情商聊天

### 1. 聊天前的认识转变
在聊天之前，你需要明白以下几点：

- **平等地位**：你和女生应该是平等的，或者你略高于她的地位。
- **目的明确**：聊天的目的是为了互相了解，而不是为了讨好或迎合对方。
- **筛选心态**：在了解对方的同时，你也在筛选对方是否符合你的标准。

### 2. 聊天前的准备
在聊天之前，你需要做好以下准备：

- **心态调整**：放下讨好和迎合的心态，以平等和了解的心态开始聊天。
- **明确目的**：明确聊天的目的是为了互相了解，而不是为了约对方出来。
- **筛选标准**：明确自己的筛选标准，了解自己需要什么样的女生。

### 3. 聊天中的注意事项
在聊天过程中，需要注意以下几点：

- **避免跪舔**：不要过分展示自己，以免让对方觉得你低价值。
- **保持平等**：与对方保持平等地位，不要让对方觉得你低人一等。
- **筛选对方**：在了解对方的同时，也要筛选对方是否符合你的标准。

## 总结
本节课中，我们学习了在开始与女生聊天之前，你需要具备的心态和认识上的转变。在后续的课程中，我们将继续探讨如何运用这些心态和技巧，让你在聊天中更加自信和从容。
