# 15、关键认知：关键关系训练营（学浪）：1-01节人是关键关系的乘积

大家好，欢迎来到关键关系第一讲。在正式开始之前，提醒所有购买课程的学员一定要联系客服，领取我们的社群。因为这个课程严格意义上他的名字叫做关键关系训练营。所谓的训练营就是要把人集中在一起，去集中解决你生活中面临的问题。不光是我的课程内容，还有我在训练营之中设置的主题讨论的主题，以及每名其他学员，还有我们的助教老师进行的引导发言，以及通过别的同学分享的案例，你能够学到的东西，可能是远大于你在面对屏幕看着我对着一块白板讲到的内容的。

## 1. 马克思关于人的论述

马克思提出最著名的一句关于人的论述，就是人事所有社会关系的总和。我们就记住这个“总”字是我们今天要解读的关键。为什么说人是所有社会关系的总和呢？因为人本身的存在性在社会维度上，他就表现为关系。你说这个人在社会中他有什么成就，或者他是谁，都是以关系的形式呈现的。

## 2. 社会关系的分化

那么这个关系的和指的就是很多关系加在一起。那为什么我说人是所有社会关系的总和在我们这个时代可能不像之前那么适用了呢？因为人和人之间的关系竟在进行分化。什么叫做分化，就是人群被分的更加细了。有的人之间的关系是更加紧密的，而有的人之间其实没有什么关系了。

## 3. 关键关系的沉积

在这个沉积之中呈现的。就有一句话说，你最常交流的5个人的平均的那个财富值就是你的财富值。就是你打开你的手机，你看一看你最常联络的5个人是谁，然后你看他们5个人大概都处于什么样的一个财富规模，你加起来一除可能就是你个人的一个财富水平。

## 4. 经营关键关系

你就要意识到一件事儿，就是你在经营关系上，你需要先把握住哪些人是你的关键关系，就是他作不作为你那5个平均值中的一员，你可以反向去思考嘛。如果你相信一个人最常交流的5个人的财富的平均值，就是你的财富值的话，如果你追求财富。那围绕这个关键关系，你怎么跟他们去交往，怎么去沟通感情，怎么样去交换信息，怎么样让他们愿意为了你的成长，为了你的发展而贡献自己的智慧，贡献自己的力量，甚至牺牲自己的利益呢？
