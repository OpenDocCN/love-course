# 课程32：中心自我与边缘自我：如何切换 🔄

在本节课中，我们将学习整理旧自我的第二条思路：中心自我与边缘自我的切换。我们将探讨边缘自我如何从中心自我中生长出来，并最终可能演变为新的核心身份。通过具体案例，我们将理解这一转变的过程，并学习如何识别和培育你自己的边缘自我。

---

上一节我们介绍了“旧能力的新应用”，本节中我们来看看整理旧自我的另一条路径：中心自我与边缘自我的切换。

几乎每个选择背后都有一个可能的自我。它们不断呼喊着“选我选我”。在特定选择中，你只能选择一个。选择它，就有机会将其带入现实；不选它，它就会消失。

人有很多个自我。在某些时段，某些自我占据舞台中心，另一些自我则成为边缘配角。如果说**中心自我**代表了最适应现实的那个自我，那么**边缘自我**则寄托了“现实以外的我”的愿望。这些边缘自我让我们的自我变得更完整。

然而在转变期，无论是出于主动选择还是现实无奈，长期占据中心位置的旧自我可能忽然消失。这时，某个边缘自我就有潜力重新成长为新的自我。

让我来举个例子。我们工作室有位家庭治疗师，是从儿科医生转行而来。这听起来是一个不太常见的转变。他是如何完成新旧自我更替的呢？

原来，他在儿科门诊时，常遇到过敏、高热、惊厥的孩子。他发现治疗效果与父母的表现有很大关系。如果父母镇静，孩子很快镇静，症状缓解；如果父母慌张，孩子更慌乱，症状维持。这是他第一次发现父母与孩子之间存在一种**情绪链**，会影响疗效。

发现这个有趣效应后，他开始关注父母的情绪。这个关注把他带到了一个更大的世界。他发现父母情绪背后常是多年未解决的矛盾，这些复杂矛盾是他不理解的。于是，他开始关注家庭治疗领域。

最初，他的**中心自我是儿科医生**，对家庭治疗的关注只是一个服务于儿科工作的**边缘自我**。这个边缘自我是从中心自我长出来的。一旦长出来，他就舍不得丢掉。他觉得只开药、不管家庭系统的做法太简单了，但又没能力治疗孩子的家庭。

于是，他开始不停加班调休，用调休时间参加各种培训。他构造了一个**容器**，想要培育那个新自我。儿科医生本就很忙，加上培训奔波，他慢慢发现自己吃不消了，好几年没有假期。科室领导也开始有意见，他的生活受到影响。

在关于容器的内容里，我们讲过容器的作用是维持新旧自我共存，让我们一边培育新自我，一边避免做出不可逆转的选择。但当新自我逐渐成长，新旧自我无法共存时，选择的契机就来了。他必须在儿科医生和家庭治疗师之间做出选择。

当选择来临时，他发现那个原本只是边缘的“家庭治疗师自我”变得不可放弃了。事实上，从那个自我冒出来开始，他就知道无法放弃。于是他选择离开医院，成为一名家庭治疗师。

当然，边缘自我变成中心自我绝非一蹴而就。它需要一个完整过程，甚至可能在遭遇现实挫折后被放弃。作为家庭治疗师，他并非一开始就顺利。离开医院后，他经历了很长时间的失落和痛苦。更糟糕的是，他没有足够客源支持新角色。靠着原医院同事的介绍，他才挨过最难阶段，逐渐成为一名家庭治疗师。

从他的转变道路，我们可以重新思考中心自我和边缘自我的切换是如何完成的。

以下是边缘自我生长的关键步骤：

首先，**边缘自我从哪里来？** 答案是：边缘自我是“我想要的自我”意愿，从现实这个容器中溢出来的。如果说我们灌溉的中心自我是一棵大树，那边缘自我就是树荫下的花花草草。它最初伴随大树而生，但随着生长，它会变得五颜六色、越来越显眼。当最后这棵大树不在时，我们发现还可以把这一片灌溉成比原来更美的小花园。

想想刚才案例中的转变。这位儿科医生最初关注家庭，看起来只是本职工作的补充和延续，但这并非本职工作要求，而是回应了他内在的某种兴趣和价值取向。他不喜欢只是开药，对人的情感本身就有兴趣。随着这个自我进入现实，不断添加新材料，它开始有自己的演进过程，最终慢慢变成一个现实中可以规划的方向。

让我来举另一个例子。我有一个朋友叫彩彤。他心理学博士毕业后到一家IT公司做用户研究。那时他的兴趣是写作和研究，他一直想当大学老师。但当他进入另一种现实时，这种研究和写作的需要有了另一种表达。他会把在用户研究领域的经验整理成心得分享。一段时间后，他成了业界小有名气的“大神”。这种写作和分享的习惯迁移到了一个刚兴起的分享网站，分享领域也逐渐从用户研究扩大到学习心得和职场感悟。

最初，这种分享欲望是从工作中长出来的。但写作和分享本身脱离了原来工作，有了自己的生命力，直到新的现实需要他在原来工作和专职写作之间做出选择。于是他开始辞职写《精进》系列的书。写作帮他完成了从边缘自我向中心自我的切换，让他从一个爱分享的IT人变成了作家。

从这些人的故事中，你可以看到一个边缘自我如何在“现实”和“我想要”之间长出来，又如何不断演进，拥有自己独立的生命力。它演进的动力就来自我们自身内在的需要。当它变成中心自我时，你就更愿意投入时间和精力，更有行动的可能。而这些行动是新自我最关键的因素。

说到这儿，也许你会想：现在我的旧自我也逐渐不合时宜了，我要从哪里去寻找边缘自我，看看它是否有潜力发展成新自我呢？

这是我们要讲的第二个问题：**如何寻找边缘自我？**

以下是提示边缘自我的线索：

*   你曾经的梦想。
*   你所做的最有成就感的事。
*   你最敬佩、最向往的人。
*   你原来工作中最让你享受的部分。

在此，你可以问自己两个问题：
1.  在我曾经做过的事情里，哪些事是就算一时看不到回报，我仍然愿意去做的？
2.  这个事情背后有一个怎样的我自己？

也许你会发现，新自我的种子早已在旧自我中种下，就在那些你在原来的自我中最热情、最愿意投入的地方。

可是我也想提醒你，对很多处于转变期的人来说，边缘自我并不直接代表新自我。它们只是值得探索的新自我的起点。就像那位转行的家庭治疗师，会面临受训不足、客源不足、他人信任等难题；半路出家的彩彤，也会面临写作技法和题材的难题。边缘自我只是新自我一个值得探索的开始。想要把它变成新的自我，还需要克服很多难题。但是对于身处转变的人来说，拥有一个开始就已经足够了。
