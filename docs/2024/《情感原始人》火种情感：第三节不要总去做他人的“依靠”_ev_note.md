# 《情感原始人》火种情感：第三节不要总去做他人的“依靠”_ev

## 概述

在本节课中，我们将探讨自卑与超越的主题，帮助大家理解自卑的两种表现形式，并学习如何克服自卑，成为真实的自己。

## 自卑的两种情况

**以下是自卑的两种情况：**

1. **害怕接触和被否定**：这种情况下，个体在面对女生或其他情境时，会产生自我否定的情绪，这与原生家庭、经历和思维方式有关。
2. **恐惧展示脆弱和爱他人的能力**：这种情况下，个体在他人面前表现出强大的形象，但内心却害怕展示脆弱，害怕被他人否定。

## 克服自卑的方法

**以下是克服自卑的方法：**

1. **认识到自卑的两种情况**：了解自卑的两种表现形式，有助于我们更好地认识自己。
2. **学会爱自己**：只有先爱自己，别人才会爱你。不要因为他人的评价而自卑。
3. **尊重自己**：尊重自己的感受和选择，不要活成他人眼中的自己。
4. **改变原有的想法**：认识到自卑的根源，并努力改变自己的想法。
5. **远离错误的圈子和环境**：跳出错误的漩涡，寻找适合自己的圈子。

## 总结

本节课中，我们一起学习了自卑的两种情况以及克服自卑的方法。希望大家能够认识到自卑的根源，学会爱自己，尊重自己，勇敢地追求真实的自己。
