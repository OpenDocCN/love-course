## 依恋类型的起源：母婴关系的关键作用

著名心理学家阿德勒曾指出：“幸运的人一生都被童年治愈，而不幸的人一生都在治愈童年。”这句话深刻揭示了早期经历对个人心理与行为的深远影响。

原生家庭，特别是从出生到3岁期间与母亲（或主要抚养者）的关系，对我们性格和依恋类型的形成至关重要。这是我们人生中最初的亲密关系，我们从中从零开始学习如何被爱以及如何与人建立联结。因此，母婴关系会在我们内心塑造一个基本的关系模型和框架，此后的所有亲密关系都会以此为蓝本。

0至3岁正是心理安全感和婴儿依恋类型形成的关键时期。这个阶段被如何对待，直接决定了你亲密关系的底色，对你一生的情感和行为模式有着至关重要的影响。

接下来，我们将逐步拆解0至3岁的母婴关系，看看它是如何具体影响我们的。

## 婴儿依恋发展的四个阶段

根据英国心理学家、依恋理论之父约翰·鲍比和美国心理学家玛丽·安斯沃斯等人的研究，婴儿依恋的发展过程可分为以下四个阶段。

以下是婴儿依恋发展的四个关键阶段：

1.  **0至3个月：无差别的社会反应阶段**
    *   此阶段婴儿对人的反应最大特点是不加区分。他们对所有人的反应都一样，同时，任何人对他们的影响也无差别。婴儿不会对任何人表现出偏爱，包括自己的母亲。

2.  **3至6个月：有差别的社会反应阶段**
    *   此时婴儿对人的反应开始出现区别和选择。他们对母亲更为偏爱，面对母亲时会表现出更多亲近反应，如微笑或依偎。对陌生人或其他家庭成员，这些反应则相对较少。但此阶段的婴儿还不懂得害怕，其情绪反应主要源于陌生感。

3.  **6个月至2岁：特殊的情感联结阶段**
    *   从6个月起，婴幼儿对母亲的存在更为关切，特别愿意与母亲在一起。当母亲离开时，他们会哭闹试图阻止。陌生人无法替代母亲的位置。这表明，婴幼儿已形成对母亲的明显依恋和专属情感联结。同时，他们对陌生人的态度变化很大，常会表现出紧张甚至哭闹，此时的情绪已由陌生感转变为由陌生驱动的害怕。

4.  **2岁以后：目标调整的伙伴关系阶段**
    *   两岁后，幼儿开始能够认识并理解母亲的情感、需要和愿望。他们知道母亲爱自己且不会抛弃自己，并能在互动中考虑母亲的需要来调整自己的行为。此时，孩子将母亲视为交往伙伴，认识到对方有自己的需求。孩子与母亲在空间上的临近性变得不那么重要。

由此可知，在0至3岁的母婴关系中，**6个月至2岁**这个阶段最为关键。大多数婴儿都是在此阶段内形成某种稳定的依恋类型。

## 如何判断婴儿期的依恋类型：“陌生情境实验”

经过上节课的测试，如果你属于焦虑型、回避型或恐惧型这三种不安全的依恋类型，那么这很可能源于你在婴儿期与母亲的关系中受到了或多或少的心理创伤。这些创伤未被治愈，并延续至今，影响着你在长期关系中的行为模式。

那么，母亲的哪些行为会导致心理创伤，从而形成不安全的依恋类型呢？心理学家玛丽·安斯沃斯通过“陌生情境实验”揭示了其中的奥秘。

该实验创设了一个有适度压力的情境：在一个不熟悉的房间里，让婴儿与母亲短暂分离，以激活婴儿的依恋系统。实验过程是：婴儿和母亲一起待在房间 → 母亲离开 → 陌生人进入与婴儿共处 → 母亲返回。研究者全程观察婴儿在不同情境下的反应。

你可能从未想过，人类在婴儿时期就已能表现出四种依恋类型中的一种。以下是各类型在实验中的典型表现：

以下是四种依恋类型在婴儿期的行为表现：

*   **安全型依恋**
    *   母亲在场时，能自由玩耍，情绪积极平稳，常与母亲进行目光交流或微笑。
    *   陌生人靠近时，能较快与之相处，反应较为积极。
    *   母亲离开时，会表现出明显的焦虑、苦恼和不安，并试图寻找母亲。
    *   母亲返回后，会立刻寻求接触，并很容易被安抚，平静下来继续玩耍。
    *   他们将母亲视为安全基地和避风港。安全型婴儿的母亲通常对孩子的情绪反应敏感，能及时回应需求，并鼓励孩子探索。社会中约有20%的人属于安全型。

*   **焦虑型依恋**
    *   非常在意母亲的一举一动，显得警惕，无法全心投入玩耍。
    *   母亲离开时，会表现出极度痛苦和反抗，甚至可能歇斯底里地哭喊。
    *   母亲返回时，态度矛盾，既想接触又表示抗拒（如拒绝拥抱、扔玩具）。
    *   对陌生人非常敌视，难以接近。
    *   他们的母亲可能经常误解孩子需求，自身情绪也不稳定，行为前后不一致，常按自己的理念或情绪而非孩子的真实需求来照顾孩子。焦虑型在人群中约占30%-35%。

*   **回避型依恋**
    *   对母亲的态度显得无所谓。
    *   母亲离开时，不表示反抗，直接忽略。
    *   母亲返回时，可能短暂迎接，随即又忽略母亲的存在。
    *   陌生人在场时，也较少感到紧张。
    *   回避型婴儿的母亲通常缺乏耐心，对孩子行为反应迟钝。当孩子出现状况（如打翻食物、闹情绪）打乱其计划时，容易情绪失控、指责孩子。当婴儿期待互动时，母亲常心不在焉或消极反馈。婴儿为免受反复伤害，启动了自我保护机制，隔离了对母亲的期待。回避型约占40%。

*   **恐惧型依恋**
    *   面对母亲时，表现出自相矛盾、不可预测的反应。
    *   情绪难以控制，时而平静冷漠，时而激动愤怒。
    *   在人群中约占5%-10%。

## 影响依恋类型的其他因素

除了母亲照顾的质量，还有其他因素也会影响婴儿依恋类型的形成。心理学家研究发现以下四点：

以下是影响依恋类型的四个关键因素：

1.  **稳定的情感联结机会**
    *   孩子是否有一个稳定的主要照顾者，以获得建立感情联结的机会。如果照顾者频繁更换（如上个月是外婆，下个月送到舅舅家），孩子便无法建立稳定的情感连接。

2.  **养育的质量**
    *   高质量的养育远不止管吃喝和安全。英国精神分析学家温尼科特提出，安全型依恋的孩子需要一个“足够好的母亲”。她能**全神贯注**于婴儿，敏锐地察觉并**及时满足**其需求，并随着孩子成长，有意识地调整依赖程度。当母亲能真正体验到孩子的感受，并让孩子知道这一点时，孩子会感受到无比的亲密与安全。公式化表达即：**敏感回应 + 及时满足 → 安全感**。

3.  **孩子的性格特征**
    *   每个婴儿天生具有不同的性格气质（如易照养型、难照养型）。面对不同气质的孩子，需要家人持续给予足够的耐心。对于难照养的孩子，家长若容易焦虑急躁，这种养育方式会加剧亲子间的不和谐关系，加深孩子的不安全感。

4.  **家庭环境**
    *   此处指家庭成员之间的关系网络，如父母关系、婆媳关系等。和谐互助的家庭关系能营造安全稳定的氛围，孩子的需求和行为更容易被接纳和发现。

## 总结与展望

本节课中，我们一起学习了依恋类型的起源与原因。我们发现，**成年人在恋爱中的表现与婴儿在母婴关系中的表现基本一致，恋爱关系本质上是母婴关系的翻版**。我们了解了母婴关系如何塑造最初的依恋类型，并认识了影响依恋形成的多种因素。

一个和谐稳定的原生家庭环境，加上能够敏感发现和满足孩子需求的母亲，大概率能为孩子的人生建立良好基础，塑造出强大的内在安全感。
