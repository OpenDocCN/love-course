# 海蓝老师情绪梳理个案课：036-查不出病因的病人第五课完成版 🎼

## 概述
在本节课中，我们将探讨情绪梳理个案中的心理创伤处理过程，以及如何通过理解和接纳来疗愈创伤。

## 过程解构
**健康与死亡的恐惧**  
当我们听到“健康”和“死亡”这些词时，会感到心跳加速，出现窒息感，身体僵住。这种僵住状态意味着恐惧到极点时，人体会停止动作。

**惊恐发作**  
很多人经历过惊恐发作，觉得自己快要死了，去医院检查却什么问题都没有。这通常与创伤有关。

**创伤回顾**  
对于有抑郁和焦虑症状的人来说，回顾过去的经历非常重要。特别是抑郁的人，往往有过极其无助的时刻。

**创伤画面**  
小孩子的前脑逻辑部分尚未发达，他们只能凭本能感受世界。因此，他们会将世界想象得非常危险。

**孤独与冷漠**  
小孩子会感到孤独和冷漠，这种理解会持续很多年。

**抑郁与焦虑**  
这种视角和滤经会导致抑郁和焦虑。

**回到场景**  
我们需要从孩子的角度回到那个场景，理解当时的情况。

**创伤的复杂性**  
创伤是复合性的，包括爸爸、姥姥、奶奶和老师的反应。

**孤独与无助**  
在最需要帮助和理解的时候，孩子感到孤独和无助。

**特殊照顾与批评**  
老师对孩子的特殊照顾和批评，实际上是一种特殊照顾。

**错位与理解**  
人与人之间有很多错位，我们需要理解和接纳。

**情绪表达**  
我们需要给情绪一个表达的机会，让它们被听到、看到和理解。

**创伤化解**  
通过梳理创伤点，我们可以理解爸爸、姥姥、奶奶和老师当时的行为和感受。

**卸载负担**  
我们需要卸载思想包袱、心理负担和身体负担，注入力量、爱和勇气。

**内在世界**  
我们的内在世界是一个军团，每个情绪和思想都有其脉络。

**智慧力量与爱**  
我们需要用智慧、力量和爱来引领我们的世界。

**焦虑恐惧的价值**  
焦虑、恐惧、抑郁等都是内在世界的一部分，它们有价值，关键是我们如何理解和接纳它们。

**身体感受与心理感受**  
我们需要尊重身体的感受和心理的感受，它们是最靠谱的。

**疗愈与幸福**  
疗愈是为了让身体舒服、心情舒服，这是幸福的要素。

## 总结
本节课中，我们一起学习了情绪梳理个案中的心理创伤处理过程，以及如何通过理解和接纳来疗愈创伤。希望这些内容能够帮助大家更好地理解自己的情绪和感受，从而过上更幸福的生活。
