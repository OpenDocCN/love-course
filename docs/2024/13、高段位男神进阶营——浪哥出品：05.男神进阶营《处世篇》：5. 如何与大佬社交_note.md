# 高段位男神进阶营——浪哥出品：05.男神进阶营《处世篇》：5. 如何与大佬社交

欢迎来到高段位男神进阶营的这节课程。本节课程我们将探讨如何与大佬沟通。面对身份地位远高于我们的人，我们可能会感到不自然，甚至拘束，这是很正常的。

然而，这其实是一个机会。在与大佬接触的过程中，我们需要留下好印象，以便从他们那里获得我们想要的东西。以下是与身份地位高、社会资源丰富的人沟通的技巧和方法。

### 常见错误

在与大佬沟通时，以下三个错误是常见的：

1. **彰显才华**：在大佬面前炫耀自己的才华是不明智的。这会让大佬觉得你自负，难以成大事。
2. **溜须拍马**：拍马屁会让大佬觉得你是个工具人，不值得信任。
3. **观点不一致时辩解**：与大佬在观点上产生分歧时，不要辩解。辩解只会让你失去机会。

### 正确方法

以下五个正确的方法可以帮助你与大佬沟通：

1. **角色定位**：将自己定位为学生的角色，表现出对大佬的尊重和求知欲。
2. **尊重者**：尊重大佬，不要谄媚，让对方感受到被尊重的快乐。
3. **自然态度**：保持自然的态度，不要拘谨，让对方感到舒适。
4. **巧托会配**：巧妙地衬托大佬，不要过分表现自己。
5. **主动真诚**：保持真诚的态度，让对方感受到你的诚意。

### 总结

本节课中，我们学习了如何与大佬沟通。通过避免常见错误，并采取正确的方法，我们可以从大佬那里获得更多的资源和机会。希望这些技巧能帮助你更好地与大佬沟通。
