#### 1. 游戏互动
以下是马克使用的一种游戏互动方式：

- **剪刀石头布**：通过这个游戏，马克可以轻松地与对方互动，并赢得一个快速的吻。

```python
# 剪刀石头布游戏
def rps_game():
    import random
    choices = ["剪刀", "石头", "布"]
    user_choice = input("选择剪刀、石头或布：")
    computer_choice = random.choice(choices)
    print(f"你的选择是：{user_choice}")
    print(f"电脑的选择是：{computer_choice}")
    if user_choice == computer_choice:
        print("平局！")
    elif (user_choice == "剪刀" and computer_choice == "布") or \
         (user_choice == "石头" and computer_choice == "剪刀") or \
         (user_choice == "布" and computer_choice == "石头"):
        print("你赢了！")
    else:
        print("你输了！")
```
