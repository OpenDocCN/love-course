# 情感博弈课 04：对方是如何掌握关系主动权的？🤔

在本节课中，我们将深入探讨一段关系中，主动权是如何被对方一步步获取的。我们将分析“高开低走”现象背后的心理机制与行为逻辑，帮助你理解为何有时会陷入被动，以及对方使用了哪些策略。

## 概述：主动权是如何转移的？

关系的主动权转移，往往始于一方无条件的付出与包容。对方通过赋予你极大的权利，让你在感情初期占据高位。然而，这种权利的给予并非永久，其背后隐藏着后续收回权利、使你陷入被动的策略。

## 1. 权利赋予：制造高位假象

上一节我们介绍了情感博弈的基本概念，本节中我们来看看对方是如何通过“赋予权利”来制造开局优势的。

对方在关系初期，会通过无条件的包容、接纳、付出和理解来行动。这让你感觉被深爱，并认为自己掌握了关系的至高权利。例如，一个你可能最初看不上的人，会通过极度谦卑和持续的付出来追求你。

**核心行为公式**：
```
初期行为 = 无条件付出 + 持续讨好 + 降低你的防备心
```

这个过程会让你享受其中并逐渐“上头”，开始习惯这种被全方位关注和照顾的状态。

## 2. 权利收回与底线打破

当你习惯了拥有所有权利后，关键的转折点便会出现。对方开始逐步收回之前赋予你的部分权利。

以下是权利收回后发生的连锁反应：

*   **感到痛苦**：权利被收回，你的舒适感消失。
*   **打破底线**：为了缓解痛苦、挽回之前的“利益”（即对方的好），你开始妥协，主动降低自己的标准。
*   **陷入被动**：你的主动妥协和付出，标志着关系主动权的悄然转移。此时，**谁痛苦，谁改变**。

**核心逻辑**：
```python
if 对方收回权利():
    你感到痛苦()
    if 你想逃离痛苦():
        你开始妥协(打破自身底线)
        你开始主动付出()
        关系主动权转移(向对方)
```

## 3. 试探底线：掌握主动权的关键步骤

那么，对方具体是如何操作，来测试并扩大自己的权利范围的呢？高段位者掌握主动权的核心方法是：**试探并突破你的底线**。

他们并非直接询问，而是通过具体行动观察你的反应。

以下是试探底线的典型过程：

1.  **发起试探**：对方做一件可能让你不快的事。
2.  **观察反应**：观察你是否接受，即使你难受，但最终是否没有施加实质性的“惩罚”（如冷淡、离开、设立边界）。
3.  **得出结论**：如果你没有惩罚，对方则判定“此事可为”，且**犯错成本为零**。
4.  **扩大战果**：基于此，对方会得寸进尺，继续突破你的其他底线。

最终，感情的天平会彻底倾向对方，对方获得极大的权利。这一切的基础是：**你的行为默许**。即使你口头反对，但行为上的容忍和接受，便是对方眼中的“许可”。

## 4. 高开低走的本质与目的

综上所述，我们可以总结“高开低走”策略的本质。

**高开低走的本质是一种“以小博大”的情感博弈**。其根本目的是争夺感情中的**绝对权利**，从而支配关系中的资源，使关系走向符合自身利益的方向。

**核心路径公式**：
```
高开（全力付出）→ 让你习惯依赖 → 低走（收回权利、试探底线）→ 你痛苦妥协 → 对方获得权利 → 对方支配资源（舒适感、金钱、服从等）→ 实现自身利益
```

每个人都是自私的，都会优先满足自己。这场博弈的终点，就是让对方成为那个优先被满足的人。

## 总结

本节课我们一起学习了关系中主动权转移的全过程。我们了解到，对方通过 **“先赋予权利，再收回权利”** 的策略，引发你的痛苦与妥协；并通过 **“不断试探和突破你的底线”** 来扩大自己的权利范围，最终实现“高开低走”，掌握关系的绝对主动权。关键在于，这一切都建立在**你行为上的默许**之上。认识到这些模式，是避免陷入被动、守护自身边界的第一步。
