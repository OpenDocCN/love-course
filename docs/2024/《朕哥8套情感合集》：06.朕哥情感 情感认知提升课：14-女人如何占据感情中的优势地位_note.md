# 课程01：理解变量与赋值 🔢

在本节课中，我们将要学习编程中最基础也最重要的两个概念：**变量**和**赋值**。它们是所有程序的基石，就像盖房子需要砖块一样。

## 什么是变量？ 📦

上一节我们介绍了课程的目标，本节中我们来看看什么是变量。

变量是计算机内存中的一个**存储位置**。你可以把它想象成一个带标签的盒子。这个盒子（变量）可以存放数据，比如数字、文字或更复杂的信息。每个盒子都有一个独一无二的名字，这就是**变量名**，我们通过这个名字来找到并使用盒子里的数据。

在代码中，我们这样声明一个变量：
```python
my_variable
```
这行代码的意思是：“计算机，请为我预留一个名叫 `my_variable` 的存储空间。”

## 赋值操作：给变量“放入”数据 ➡️

仅仅有盒子（变量）还不够，我们需要往里面放东西。这个过程就叫**赋值**。赋值使用等号 `=` 来完成。

它的核心公式是：
**`变量名 = 值`**

这个等号 `=` 不表示“等于”，而是表示“将右边的值，放入左边的变量中”。例如：
```python
my_number = 10
```
这行代码的意思是：将数字 `10` 放入名为 `my_number` 的变量中。

## 变量的使用 ✨

一旦变量被赋值，我们就可以在程序中使用它。使用时，计算机会找到对应的“盒子”，取出里面的值来参与运算。

以下是使用变量的一个简单例子：
```python
x = 5      # 将5赋值给变量x
y = 3      # 将3赋值给变量y
sum = x + y # 计算x和y的和，并将结果赋值给变量sum
```
执行后，变量 `sum` 中存储的值就是 `8`。

## 变量命名规则 📝

为了代码清晰且能被计算机正确理解，给变量起名需要遵循一些规则。

以下是主要的命名规则：
*   **可以包含**：字母（A-Z, a-z）、数字（0-9）和下划线（_）。
*   **不能以数字开头**：`1variable` 是无效的，但 `variable1` 是有效的。
*   **区分大小写**：`Age`、`age` 和 `AGE` 是三个不同的变量。
*   **不能使用保留字**：不能使用像 `if`、`for`、`while` 这样的语言内置关键字作为变量名。
*   **建议使用描述性名称**：使用 `user_age`、`total_price` 这样的名字，比 `a`、`x1` 更能让人理解变量的用途。

## 重新赋值 🔄

变量的值不是一成不变的。我们可以随时给同一个变量赋予一个新的值，新值会覆盖旧值。

```python
message = “你好”  # 变量message中存储了“你好”
print(message)    # 输出：你好
message = “再见”  # 现在，变量message中的值被更新为“再见”
print(message)    # 输出：再见
```
这个特性让变量变得非常灵活和强大。
