# 雨宸19套合集：10.《亲密关系必修课：遇见更好的自己》：001.人际关系研究：让TA喜欢上你的一百个理由！

## 概述
在本节课中，我们将学习如何通过社会心理学原理，运用相互性原则、交换性原则、情绪调动、外貌、价值、品质、情绪表达和曝光效应等方法，让对方喜欢上你。

## 一、社会心理学原理
### 1. 相互性原则
**公式**：付出 → 回报 → 认同 → 重视

**描述**：在人际交往中，人们期望得到认同和成就感。当你为别人付出时，对方也会期望得到相应的回报，以获得认同和重视。

### 2. 交换性原则
**公式**：价值 → 吸引 → 价值提升

**描述**：个体期待人际关系对自己有价值。通过展示自己的价值，让对方感受到与你在一起会变得更有价值。

## 二、情绪调动
### 1. 情绪的本质
**公式**：喜欢 = 情绪

**描述**：喜欢一个人，本质上是因为对方的情绪和感觉。因此，要调动对方的正面情绪，如愉悦感、满足感、兴奋感等。

### 2. 如何调动情绪
**公式**：外貌 → 吸引 → 价值 → 品质 → 情绪

**描述**：通过外貌、价值、品质等因素吸引对方，让对方产生好奇和愉悦感，进而产生正面情绪。

## 三、建立关系
### 1. 建立社交关系
**公式**：相识 → 朋友 → 熟人 → 情侣

**描述**：通过社交方式或社团活动等途径，与对方建立基础关系，为后续发展奠定基础。

### 2. 合理的身份
**描述**：在接近一个人时，要有一个合理的身份，让对方感受到安全感。

## 四、应用原则
### 1. 相互性原则
**公式**：对方付出 → 你回应 → 认同 → 重视

**描述**：对方为你付出时，你要积极回应，让对方感受到认同和重视。

### 2. 交换性原则
**公式**：你满足对方需求 → 对方满足你需求 → 价值提升

**描述**：通过满足对方的需求，让对方感受到在这段关系中有收获。

## 五、刺激情绪
### 1. 外貌
**描述**：保持干净整洁的外貌，以吸引对方。

### 2. 价值
**描述**：展示自己的优势和价值，让对方感受到你的吸引力。

### 3. 品质
**描述**：展现真诚、诚实、可信赖等优秀品质，让对方感受到你的可靠性。

### 4. 情绪表达
**描述**：学会正确表达自己的情绪，让对方感受到你的真实感受。

## 六、曝光效应和心理暗示
### 1. 曝光效应
**描述**：多制造与对方偶遇的机会，让对方感受到你们的缘分。

### 2. 心理暗示
**描述**：在沟通过程中运用心理暗示，让对方感受到你们的想法一致。

## 总结
本节课中，我们学习了如何通过社会心理学原理和多种方法，让对方喜欢上你。在日常生活中，要注重细节，学会运用这些方法，提升自己的吸引力，建立良好的人际关系。
