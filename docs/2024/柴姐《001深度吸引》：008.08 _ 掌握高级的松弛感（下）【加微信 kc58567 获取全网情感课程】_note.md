# 柴姐《001深度吸引》：008.08 掌握高级的松弛感（下）

## 概述
在本节课中，我们将学习如何解除攻击和压迫，从而在社交中展现出高级的松弛感。

## 解除攻击
### 意识层面
- **核心概念**：一个斤斤计较、开不起玩笑、过于敏感的人是吸引力很弱的。相反，一个大度宽容的男生才是一个高大的存在。
- **公式**：吸引力 = 宽容度 - 敏感性

### 行为层面
- **降低辩论感**：当观点不和时，避免辩论，表达观点后接受差异。
- **以德报怨**：以礼貌真诚的态度回应对方的不礼貌。
- **减少被动型攻击**：避免使用平静的消极的摆烂方式攻击他人。

## 解除压迫
### 压迫感的特点
- **核心概念**：压迫感是一种不安甚至恐惧的感觉，与攻击不同，压迫感是不可预测的。
- **公式**：压迫感 = 不确定性 - 安全感

### 压迫感的来源
- **以自我为中心**：只关注自己的利益和需要，不顾及其他人的权利和感受。
- **缺乏同理心**：不尊重他人，不关心他人的感受。
- **冷嘲热讽**：使用尖酸刻薄、阴阳怪气的话语。
- **太心急**：做超越了现有关系的事情。

## 总结
本节课中我们一起学习了如何解除攻击和压迫，从而在社交中展现出高级的松弛感。希望大家通过训练能够解除紧绷、解除攻击、解除压迫，离开防御状态，散发出高级的松弛感。😊
