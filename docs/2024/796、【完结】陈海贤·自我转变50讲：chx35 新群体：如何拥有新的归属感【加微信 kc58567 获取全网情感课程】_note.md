# 796、【完结】陈海贤·自我转变50讲：chx35 新群体：如何拥有新的归属感

欢迎来到自我转变五十讲，我是陈海贤。上节课我们探讨了两种常见的角色认同——榜样和导师。今天，我们将探讨第三种认同对象——一群人。当你与这群人在一起时，你会深刻感受到与他们之间有一种情感的共鸣，感觉自己属于他们，并希望以这种角色和身份继续生活。这就是我们今天要讲的寻找新群体。

人需要一种归属感，哪怕是一种抽象的归属感，它也能提示你是谁，让你安心。在前面的课程中，我们把新旧自我的更替形容为新旧部落之间的迁徙，并详细讲述了如何应对旧身份和旧关系的脱离。然而，这种脱离并不意味着我们一直要孤单下去。虽然关系对人有很多限制，但关系仍然很重要。它塑造着我们的情感反应和价值观。因此，在离开原有的关系后，我们会渴望加入一个新的群体来获得新的归属感，以重新确认自己是谁。

**新群体与旧群体的区别**

也许你会问，同样是归属一个群体，为什么我好不容易离开了一个群体，还要加入新群体呢？新群体和旧群体的最大区别在于，旧群体通常基于旧的规则，这种规则常常与我们内心自我的需求相矛盾。而我们要寻找的新群体，会为我们留下更多的空间。可以说，寻找新群体的过程也是寻找新自我的过程。

榜样和导师成为认同对象时，新群体为你提供了更多新世界的信息，也提供了更具体的情感支持。道理很简单，当你一个人走一条路时，你会怀疑。但如果你看到一群人一起走一条路，你就会相信这条路是可行的，你对转变的旅程也会有更多的信心。有时候，榜样和导师是在进入一个新群体之后才找到的。新群体支撑了一个新的自我。

**如何寻找一个新群体**

如何寻找一个新群体呢？这是一个漫长而曲折的过程。有时候你会走很多弯路。有时候当你真正进入这个群体时，你会发现这个群体不是你想要的。有时候你真的很想加入某个新群体，却发现怎么也找不到加入的门在哪里。从我所接触的转变比较成功的同学看，我发现他们都遵循了以下几条重要的原则：

1. **尝试接触**：这是一个很直白的原则，但很多人做不到。很多人被困在自我怀疑中，他们会不停想：“这真是我想要的吗？我配吗？他们这么忙，会搭理我吗？如果我被拒绝了怎么办？”很多时候，进入了以后，你就会发现这个新群体并没有那么高不可攀。你有你自己的能力和资源，你的自我怀疑只是因为你对这个新群体陌生，只是因为你觉得你的我想要是微不足道的。

2. **从错误的群体中吸取正确的信息**：当你进入一个新的领域时，你很难一开始就找到让你认同的新群体。这是一个曲折探索的过程。虽然错误的群体没有办法带给你身份认同，但他们常常也会提供一些正确的信息。

3. **不断问自己，这是不是我想要成为的样子**：无论是榜样、导师还是新群体，它们都是我们寻找自我认同的过程。我们寻找它们就是在寻找一个自己生活的样子。所以当你与新的人群接触时，你会不停问自己：这些人是不是我要活成的样子？如果是，他们的状态在哪里吸引我？如果不是，那我要活成的样子，跟他们的不同在哪里？这样所有的信息都有了一个校准的基础。
