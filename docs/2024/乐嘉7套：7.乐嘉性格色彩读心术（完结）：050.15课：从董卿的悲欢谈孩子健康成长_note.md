# 乐嘉7套：7.乐嘉性格色彩读心术（完结）：050.15课：从董卿的悲欢谈孩子健康成长_note

## 概述

在本节课中，我们将通过主持人董卿的成长案例，探讨不同性格色彩的家长会采用何种教育方式，以及这些方式如何影响不同性格的孩子。我们将学习性格色彩的“钻石法则”——因人而异、因色施教，并思考如何运用这一法则促进孩子的健康成长。

---

最近，朋友圈的关注点发生了变化，讨论家长里短八卦的人变少了。

许多人开始对中国诗词大会产生浓厚兴趣。在节目中，主持人董卿表现出色，不逊色于任何选手，她对唐诗宋词信手拈来，不使用题词器，与所有人互动时，开口便能引经据典。

董卿走红后，网络上的宣传铺天盖地。许多父母因此让孩子背诵诗词，希望孩子长大后能“腹有诗书气自华”，成为像董卿那样谈吐不凡的人。有的家庭要求孩子一天背三首，有的甚至要求一天背五首。

但不久后，网上出现了另一种声音。有帖子提醒所有家长，不应只看到董卿的辉煌，而忽视她悲惨的、缺乏玩乐时间的童年，不能因为父母的期望而毁掉孩子的一生。帖中指出，董卿的成功仅仅是个案。

于是，大人们又感到困惑。不让孩子背诗，担心他长大后落后于人；让孩子背诗，又怕他失去快乐童年，将来被孩子怨恨。父母们既想严格管教，又不想成为恶魔，不知该如何是好。

网络上支持培养“董卿式”孩子的人，大多是采用类似教育方式的父母，或者自己就是被这样教育成才的。他们的论据是：董卿在台上的卓越表现，正是源于她童年的刻苦。

而网络上反对培养“董卿式”孩子的人，大多可能是在这种教育模式下受过伤害。他们挖掘出董卿的童年经历，公之于众。

那么，董卿的童年究竟如何？她的父母是复旦大学的高材生。读书时，父亲对她威逼利诱，要求学习成绩必须优秀。因为父亲喜欢文科，便让她大段抄写成语，无论懂不懂，都必须不停地抄。寒暑假时，她必须勤工俭学，天不亮就到马路上跑步。父亲不允许母亲给她做新衣服，因此董卿小时候在同学面前感到非常自卑。父亲还喜欢当众数落她，指责她哪里做得不好、吃相难看等。

在一次节目访谈中，董卿亲口对主持人说：“我爸是一个特别特别特别严厉的爸爸。”她用了三个“特别”，并感叹自己经常觉得好像不是父亲亲生的。

这番话引起了那些反对成为“董卿”的人的共鸣。于是，“成为董卿”和“不要成为董卿”两种观点展开了拉锯战，让许多年轻父母感到迷茫，不知该何去何从。

然而，如果你稍微了解一点性格色彩，就能从董卿的案例中发现，她的父亲是一位**黄色**性格特征非常明显的家长。

## 不同性格色彩家长的教育方式

上一节我们分析了董卿父亲的黄色性格特质。本节中，我们来看看其他性格色彩的家长通常会采用怎样的教育方法。

以下是四种主要性格色彩家长的教育特点：

*   **黄色性格家长**：有强烈的改造欲和控制欲。他们倾向于用直接了当的方式，让孩子朝他们认为正确的方向发展。他们通常不认可、不表扬孩子，坚信只有批评和施加压力才能使人进步。他们不关注孩子的感受，坚持让孩子做他们认为对的事。
    *   **核心行为模式**：`目标导向 + 施加压力 + 忽视感受`

*   **红色性格家长**：与黄色性格家长不同。就像电视剧《虎妈猫爸》中的爸爸罗素，他们富有童心，会趴在地上和孩子一起玩积木，看到电视里有趣的内容会和孩子一起模仿玩耍。他们的首要目标是让孩子开心。红色性格的父母通常很受孩子欢迎，互动性强，他们认为人生最重要的是当下的快乐，孩子长大后总会有办法，说不定还有好事等着，不必现在搞得那么累。
    *   **核心行为模式**：`快乐导向 + 高度互动 + 关注当下`

*   **蓝色性格家长**：规划性极强。他们对孩子穿什么颜色的衣服、牙刷如何摆放都有自己的规矩。他们不仅要求结果符合目标，还要求过程的每一步都按他们的计划进行。但他们不会像黄色性格家长那样直接批判，而是会晓之以理、动之以情，慢慢地磨，直到孩子屈服。
    *   **核心行为模式**：`计划导向 + 注重细节 + 持久说服`

*   **绿色性格家长**：他们的特点是能为孩子营造宁静、安详的家庭氛围。孩子想玩就先去玩，晚点吃饭也没关系；考试不及格也不会发脾气；孩子被欺负了，会告诉孩子“吃亏是福”。他们没有挫折教育，也没有严格的规矩约束，让孩子拥有一个温顺的童年。这也解释了在绿色性格父母养育下长大的孩子，压力感通常最低。
    *   **核心行为模式**：`平和导向 + 低要求 + 避免冲突`

世界上不同性格的父母，教育方法各异，施加在孩子身上会产生完全不同的效果。

## 不同性格孩子的不同反应

了解了家长的教育方式后，本节我们来看看，当不同性格的孩子面对这些教育方式时，会作何反应。我们继续以董卿为例。

董卿本人的性格是**红色**。红色性格不喜欢巨大的压力。因此，当她那**黄色**性格的父亲给她带来巨大压迫感时，她内心会陷入痛苦。

但如果董卿是**黄色**性格，情况就不同了。最典型的例子是钢琴家朗朗。如果你看过朗朗和他父亲的采访，会发现朗朗的父亲也是标准的黄色性格，与董卿父亲一样。朗朗不好好弹琴，父亲就拿皮带抽，直到抽断为止，从小就以目标为导向。而朗朗本人抗压能力非常强，父亲越逼迫，他越要练好琴，在这条道路上坚定前进，并没有像董卿那样描述内心的痛苦。

那么，如果黄色家长的教育方式用在**蓝色**性格的孩子身上，会怎样呢？我建议你看一部电影，片名叫《向日葵》。这是一部文艺片，但非常值得一看。简单来说，电影中的父亲梦想成为画家，但因文革破灭，于是将全部梦想寄托在儿子身上，逼迫儿子学画。但儿子根本不喜欢画画。父亲用尽各种方法强迫他。儿子是蓝色性格，反抗无果。终于有一天，在国庆游行时，儿子手握一个“二踢脚”鞭炮，点燃后死死捏在手中不放，最终炸掉了半个手掌，鲜血直流。但那个孩子却露出了微笑，因为他用这种极端的方式，终于可以不再画画了。

你会发现，当蓝色性格的孩子内心深处想要反抗时，他们会采用这种沉默但极端的方式，来表达对父母试图控制他人生的强烈愤慨。

## 钻石法则：因色施教

当我们了解了不同性格家长的教育方式，以及不同性格孩子的反应后，问题来了：父母到底应该使用什么样的教育方法？他们的方法是对是错？如果我们的父母用错了方法教育我们，我们是否应该恨他们？

我理解，有时你可能对父母心怀怨恨。但我想说，你未必真正理解他们。

如果我们的父母那一辈早早懂得“因人而异，因色施教”，明白不同性格的孩子需要不同的教育方法，他们也许就能知道，如何让孩子既能快乐成长，又能取得成功，像董卿那样，同时保有快乐的童年。

我有一个学生是典型的红色性格。我想特别说明，绝不是只有打压和批判才能成功。对于红色性格而言，他们喜欢的是**表扬和鼓励**。

我这个学生特别喜欢画画。有一次他心血来潮画了一幅挂历，他的父母立刻称赞道：“哎呀，画得真好，都可以卖钱了！”第二天，父母从兜里掏出十块钱，告诉孩子这幅挂历被单位里的一位阿姨看中并买走了。孩子特别兴奋，得到了口头和金钱的双重激励，于是开始拼命画画，一连画了十多幅“卖”给父母，“挣”了不少钱。长大后他才知道，那些挂历一幅都没卖出去，都是父母藏起来后，自己掏钱给他的。但正因为孩子全身心投入画画，并得到了巨大激励，最终在小学五年级时，获得了澳大利亚举办的全球少儿水彩画大赛金奖，也是唯一一名来自中国的小朋友。

那时他的父母没有学过性格色彩，但无形中已经运用了性格色彩的**钻石法则**——用适合这个孩子性格的方法去激励他。所以这个孩子从小到大都对父母充满感激，因为他拥有一个特别快乐的童年。

这个人就是我办公室的蒋佳琦，他是《超级演说家》第二季我团队中的队员。你可以在网上搜索到他当年的演讲视频。他每次跟我讲起这个故事，都对父母充满感恩。

## 理解与宽恕：教育的前提

网络上还有一种声音，很多人批判董卿的父母，认为他们对孩子太严苛。但我想说，在董卿内心深处，从她站在中央电视台舞台的那一刻起，她是感激父母的。因为她不仅理解了父亲当年的艰辛，也明白了父亲那么做，是希望她能养成吃苦耐劳的习惯，拥有面对生活困难和挫折的信心与勇气。

然而，还是有很多人心中有恨，并非所有人都能原谅父母。

有一部电影叫《思悼》，讲述了朝鲜王国历史上真实的悲剧。朝鲜第21代国王英祖，是一位望子成龙的父亲。他对唯一的儿子思悼太子从小进行严格的精英教育。但思悼太子是一个像“小燕子”一样的红色性格孩子。最终，这种严苛的方式激怒了孩子，导致其造反。失败后，国王将儿子关在米柜里五天，活活饿死。儿子死后，这位父亲也痛苦不堪。

我想转述这对父子在临终前的对话，让你了解他们内心的真实想法：

父亲说：“我四十岁才老来得子，你知道我有多开心吗？从你两岁起，我就让你接受帝王教育。但小时候的你玩刀剑、画画偷懒、不学习时，我觉得天都要塌下来了。我为了让你成为好皇帝，才对你如此严厉。可每次你犯错，你知道我有多担心……”

儿子说：“我知道，我知道这些，我才努力去理解你。可你这种强求的做法让我窒息，我无法忍受。学习有那么重要吗，爸爸？穿衣着装就有那么重要吗，爸爸？”

父亲说：“如果一国之君不好好学习，就算系歪了一个裤脚，将来也会被臣子轻视。”

儿子最后说：“我不喜欢当王，也不喜欢权谋，我只希望过自己的人生。我要的只是你一个温柔的眼神和一句亲切的话啊……”

说完这句话，儿子就咽气了。那位黄色性格的老国王在黑夜中对着儿子的尸体，一动不动。

皇宫之中，大家勾心斗角。老皇帝无法理解儿子的感受，只能用他以为正确的冷酷方式逼迫儿子变得强大；而红色性格的儿子也无法理解黄色父亲的做法，只希望父亲能多表扬他，给他一个微笑。遗憾的是，两个人都没有学过性格色彩，理解得太晚，早已没有回天之力。

## 总结

所以，在开始引导你的孩子正确成长之前，我的朋友，请你先**理解和原谅你的父母**。因为如果你不能理解父母教育你的方式是对是错，你将无法用正确的方法去引导你的孩子。

学会理解，会让我们减少很多痛苦，会让我们知道人生下一步该如何正确行走。

你完全可以用正确的方法，做到董卿已经做到的那些事。关键在于运用性格色彩的钻石法则：**识别性格，因色施教**。

本节课中，我们一起学习了：
1.  通过董卿的案例，分析了黄色性格家长的教育特点。
2.  了解了红、黄、蓝、绿四种性格色彩家长的典型教育模式。
3.  探讨了不同性格孩子对同一种教育方式可能产生的截然不同的反应。
4.  学习了性格色彩的“钻石法则”——因色施教，并通过实例看到了鼓励对红色性格孩子的巨大力量。
5.  认识到理解与宽恕父母是实施正确教育的重要前提。

希望这节课的内容，能对所有为人父母、即将为人父母，以及在父母教育方式下感到痛苦的朋友们有所帮助。
