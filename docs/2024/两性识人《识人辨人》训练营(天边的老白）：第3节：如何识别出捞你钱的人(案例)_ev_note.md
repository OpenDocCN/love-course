# 两性识人《识人辨人》训练营(天边的老白）：第3节：如何识别出捞你钱的人(案例)

大家好，我是老白。上节课我们了解了人性的多面性和环境、成长经历对人性造成的影响。这节课，我将通过一个案例，帮助大家识别那些只想捞你钱的人。

## 案例分析

有一个姑娘爱上了一个夜场工作的小哥哥，她想知道这个小哥哥是真心还是只想捞钱。我了解到姑娘的情况：照片一般，身高1米6，体重110左右，小哥哥身高1.84米，很帅。

**问题**：小哥哥夜场工作，认识美女那么多，你觉得你在他的微信里属于什么档次？除了钱，你还有什么能吸引他的？

小哥哥说：“我知道我长得不那么好看，但我比那些人对她好，我会给她做吃的。”

姑娘很感动，觉得从来没有人对她这么体贴。她27岁，年龄不小了，这个工作总不能一直做下去吧。

**测试方法**：观察小哥哥在你不花钱后的表现。

姑娘给我发了一些聊天记录，可以看出这个男人在套路她。

**关键对话**：

- 小哥哥：“宝贝，你说啥？你说哈，今天我很困，然后因为想见你了，没去工作，有没有小补偿？”
- 姑娘：“你想怎么补偿？”
- 小哥哥：“比如一个小红包呀，最近花钱地方真是太多了，我约了鼻子手术，然后房子也在装修，所以才天天上班。”
- 姑娘：“你的金库守护者，你做吗？”
- 小哥哥：“不用，最近这几天难一点而已，你看一边哭穷，一边又表现出不用不用不用，这就是啥？一边伸着碗，一边说不用不用啊，不要给我钱。但是碗一直往你面前伸。”

姑娘上头了，觉得小哥哥很在乎她。

**案例分析**：

这个案例中，小哥哥一边哭穷，一边又表现出不需要钱，这就是典型的“一边伸着碗，一边说不用不用”。他只是想捞钱。

## 如何识别捞钱的人

1. **筛选**：通过一个人的生活环境、原生家庭来推测他人性中最大的需求，得出一个相对客观的结论。
2. **验证**：通过对方的行为而非语言来验证你的判断。
3. **独立判断**：不要被别人的影响下得出判断，要自己独立判断。
4. **验证说法**：判断他说的话是真是假。
