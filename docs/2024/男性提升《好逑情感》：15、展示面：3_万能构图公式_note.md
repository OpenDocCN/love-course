# 男性提升《好逑情感》：15、展示面：3_万能构图公式 📸

## 概述
在本节课中，我们将学习如何运用四个潮男拍照常用的万能构图公式，以及如何通过这些公式提升照片的视觉效果。

## 1. 三分之一构图法
**公式**：将画面分为三个等分，人物置于画面的右三分之一或左三分之一位置，头顶控制在上三分之一处。

**代码**：
```markdown
|------------------|
|   人物           |
|------------------|
|                   |
|------------------|
```

## 2. 四格构图法
**公式**：将画面分为四个等分，人物置于右下四格或左下四格，形成对角线构图。

**代码**：
```markdown
|------------------|
|                   |
|   人物           |
|------------------|
|                   |
|------------------|
```

## 3. 对角线构图法
**公式**：利用场景的左右对称或上下对称，形成对角线构图。

**代码**：
```markdown
|------------------|
|                   |
|                   |
|------------------|
|                   |
|------------------|
```

## 4. 对称构图法
**公式**：利用场景的左右对称或上下对称，形成对称构图。

**代码**：
```markdown
|------------------|
|   人物           |
|------------------|
|   人物           |
|------------------|
```

## 总结
本节课中，我们一起学习了如何运用四个潮男拍照常用的万能构图公式，包括三分之一构图法、四格构图法、对角线构图法和对称构图法。希望这些技巧能够帮助你在拍照时提升照片的视觉效果。
