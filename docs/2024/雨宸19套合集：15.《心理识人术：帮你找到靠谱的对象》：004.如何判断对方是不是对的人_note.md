# 心理识人术：判断对的人 🧐

## 概述

在本节课中，我们将学习如何判断对方是否是合适的人选。我们将探讨亲密关系中的三个关键标准：亲密度、匹配度和承诺度。

## 亲密度

**定义**：亲密度是指与对方在一起时感到的舒适和放松程度。

**判断标准**：

1. **放松与舒适**：与亲密的人在一起时，你会感到放松和舒适，卸下心防，展示自己最脆弱的一面。
2. **自然身体接触**：与对方有自然而然的身体接触，如牵手、拥抱等。
3. **双向沟通**：沟通是双向的，而不是单方面的找话题。

## 匹配度

**定义**：匹配度是指双方在性格、价值观和经历等方面的相似性。

**判断标准**：

1. **原生家庭**：与各自父母，尤其是异性父母的相处方式相似。
2. **三观**：对世界的看法、对自己的看法和对关系的看法相似。
3. **未来规划**：对未来的规划一致。

## 承诺度

**定义**：承诺度是指对关系的投入程度和愿意为关系付出的程度。

**计算公式**：承诺度 = 满意水平 + 沉默成本 - 替代伴侣的质量

**判断标准**：

1. **满意水平**：对方对你是否认可，是否愿意把你介绍给亲朋好友，是否鼓励和肯定你。
2. **沉默成本**：在关系中的投入程度，如时间、精力、感情和金钱。
3. **替代伴侣的质量**：替代伴侣的吸引力程度。

## 总结

本节课中，我们学习了如何判断对方是否是合适的人选。通过关注亲密度、匹配度和承诺度这三个关键标准，我们可以更好地了解自己和对方，从而找到合适的人。记住，感情需要经营，学会经营才能拥有幸福的爱情。😊
