# 《朕哥8套情感合集》：02.朕哥高阶私教课之保卫爱情：18-实操二：从被男人嫌弃，到成为男人的女神

🎼被分手的爱情又回来了。大家好，我是震哥。我们有很多女学员啊，在谈过一场相处时间很长，但是无疾而供的恋爱之后呢，都感觉自己再也没有办法投入同样多的精力去经营新的感情了。

**概述**：本节课将探讨如何挽救一段感情，分析两种常见的分手类型：价值缺失型和情感疲惫型。

**价值缺失型**：

- **案例**：婷婷和男友在一起6年，因工作原因相处时间减少，导致沟通不畅，误会产生，最终分手。
- **分析**：男生因聚少离多，难以把握女生情绪；女生因工作忙碌，对男生关心减少。
- **挽回建议**：
  1. 利用朋友圈传达信息，暗示男生还爱他。
  2. 改变形象，焕然一新，吸引男生。
  3. 初步试探，观察男生反应。
  4. 沟通联系，约男生见面，消除误会。
  5. 聊天过程中，适当回忆过去，注意沟通技巧。

**情感疲惫型**：

- **案例**：小平和小峰恋爱多年，因工作忙碌，沟通减少，感情平淡，产生误会，分床睡。
- **分析**：男生因习惯彼此付出，忽略女生情绪；女生因担心男生变心，产生怀疑。
- **挽回建议**：
  1. 冷静思考，停止付出，保持联系。
  2. 重新吸引男生，改变形象，激发危机感。
  3. 与共同好友聚会，展示新形象。
  4. 深度沟通，表达顾虑，解决问题。
  5. 和好后，注意沟通，相互理解，寻找新鲜感。

**总结**：本节课中，我们学习了如何挽救一段感情，分析了两种常见的分手类型，并给出了相应的挽回建议。希望同学们在遇到类似问题时，能够冷静思考，找到问题的关键点，最终挽回幸福美满的爱情。
