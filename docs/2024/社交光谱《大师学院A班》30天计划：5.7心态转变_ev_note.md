# 社交光谱《大师学院A班》30天计划：5.7 心态转变 🧠

在本节课中，我们将要学习心态转变的核心原理。我们将探讨情绪的两面性，理解恐惧背后的深层动机，并学习如何将行动的驱动力从恐惧转变为爱。

---

上一节我们介绍了情绪的基本概念，本节中我们来看看一个重要的哲学观点：祸福相依。所有你感受到的事物，无论是好是坏，都具有两面性，并隐藏着相应的回报。

例如，对于接近女生的恐惧。

这种恐惧带来的阻力，恰恰能帮助你理解恐惧的本质。真实的情绪只是一个信号，它会很快消失。但虚假的恐惧会持续占据并依附你的思维。

我们之所以成为现在的自己，很大程度上是童年时期面对各种处境时，那个孩子所选择的最优解。

很多人的内在是千疮百孔的，包括许多外貌出众的女生。

以下是她们可能经历的情况：
*   在成长经历中，尤其在初中高中阶段，她们可能经常受到骚扰或言语侮辱。
*   在生活中，她们会遇到许多男性的搭讪。
*   许多女生因此缺乏安全感，从而形成一种“厌男”的自我保护机制。

她们形成了一种核心信念：**这种恐惧保障了她们的安全，她们本该如此。**

她们沉浸于恐惧之中。

对于每件事都想得很多，时刻处于缺乏安全感的状态，因为不想将自己置身于危险中。

尤其是许多具有回避型依恋的人，他们不想依恋或相信任何人。

只有恐惧会让他们觉得安全。

这就是为什么原生家庭导致缺乏安全感的女生，在亲密关系中常常表现出“作”的行为。但如果你真心想要发展长期关系，建议你用心去体会她内心深处的猜忌与回避。

你是在用拥抱生活的手来拥抱你的伴侣，这是长期关系的核心之一。不过这部分内容将在后续的S班课程中详细展开。

恐惧会让我们感到安全，这即是其好的一面。

当然，如果你认为恐惧能保障安全，你就会紧紧抓住它。

这引出了我们的下一个知识点。

---

上一部分我们看到了恐惧的“好处”，接下来我们学习一个关键的转变方法：在心里提醒自己，**你可以出于爱去采取保护措施**。

在我们的自我意识尚未唤醒时，我们会本能地认为，保护自己的唯一方法就是体验恐惧。

简单的逻辑公式如下：
`我担心[某事发生] -> 所以我采取[某个行动]`
例如：
*   我担心破产 -> 所以我努力赚钱。
*   我担心被抛弃 -> 所以我努力索取他人认同。
*   我担心被伤害 -> 所以我最好总是感到害怕、质疑一切、处于提防状态。

但是，你可以出于爱去采取同样的行动，而不仅仅是恐惧。恐惧本身并非问题所在，因此试图解决恐惧是无效的。问题的根源在于，恐惧出现的地方，正是你内心对于爱的匮乏。

于是我们得到了一个新的行动公式：
`我可以为了[爱/尊重/健康] -> 去采取[某个行动]`
例如：
*   我可以为了我的健康去努力运动，而不仅仅是害怕生病。
*   我可以为了自我成长去提升，而不仅仅是为了得到某人青睐。

你可以做些什么，因为你爱自己、尊重自己。你可以从对自己的爱中获得任何你想要的东西，比如成功或赚钱的机会。你可以摸索出与自己的边界。这是出于爱和自尊，而不仅仅是恐惧所驱动的。

你可以问问自己：你所经历的恐惧，其好的一面是什么？（例如，它保证了你的安全。）

一旦你确定了这个问题的答案，你就可以在精神上做出一个转变，或进行一些释放。**这才是解决任何恐惧的关键所在。**

---

本节课中，我们一起学习了心态转变的路径。我们认识到任何体验都有两面性，恐惧可能被用作一种安全策略。更重要的是，我们学会了将行动的出发点从“逃避恐惧”转变为“拥抱爱与自我尊重”，这才是实现真正心态转变和解决深层恐惧的核心。
