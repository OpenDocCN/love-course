# 21.王一名·吸引力出圈指南让：好口才的秘密开口就能讲出被人喜
## 个人品牌营销：好口才的秘密——开口就能讲出被人喜欢的观点

在本节课中，我们将学习如何创作一篇有潜力、受欢迎的短视频文案。核心内容包括两个关键步骤：**确定选题**与**构建文案结构**。我们将通过一个简单有效的公式，帮助你系统地完成创作。

---

### 第一步：如何确定选题 📝

上一节我们介绍了什么样的内容才是大家喜欢的内容。本节中，我们来看看如何为你的内容找到一个成功的主题。

选题是内容的灵魂，决定了内容的成败。结构再完美，画面再精致，如果主题不明确，观众将无法理解你想表达什么，更不会点赞。

确定选题的关键在于选择大众本身关心的主题。不要闭门造车，而应通过调研，发现你所在领域中点赞最多的内容主题。

这好比开一家咖啡馆。正确的做法不是空想位置和产品，而是调研成功的咖啡馆开在哪里、卖什么。例如，在星巴克旁边开瑞幸咖啡，就是基于现有消费数据的选择。

反之，在一个没有咖啡馆的区域开店，很可能因为该区域缺乏咖啡消费人群而失败。创作短视频同理，盲目尝试未经市场验证的新颖创意，成功率往往很低。

因此，要在大概率上成功，就必须从大众关心的主题中进行创作。

以下是确定选题的具体方法：

1.  **寻找对标账号**：首先，找到至少5个你所在领域的优秀博主（对标账号），分析他们的内容。
2.  **分析爆款主题**：其次，找出这些对标账号中点赞最多的视频，分析它们讨论的主题。
3.  **确定你的主题**：最后，你可以围绕相同的主题进行创作，或从书中提炼有价值的观点作为主题。

以读书博主为例：
*   你发现对标账号中，视频《蛤蟆先生去看心理医生》点赞很高。
*   这本书的副标题“该不该看心理医生，这本书会给你答案”就可以直接作为你的视频主题。
*   书中的关键内容，如“心理咨询的体验”或“心理咨询师与心理医生的区别”，也可以作为主题。

总结来说，书中有价值的观点都可以成为推荐理由和视频主题。这种方式成本低、易操作，且能帮助你塑造有吸引力的个人品牌。

此方法可举一反三，适用于任何领域的博主。通过分析爆款、借鉴主题，并用你自己的角度表达，是打造个人品牌、提升吸引力的有效途径。

### 第二步：构建文案结构——PREP公式 🏗️

在确定了视频主题之后，接下来我们需要为这个主题搭建一个清晰有力的表达框架。本节将介绍一个经过验证的万能文案结构。

一篇好的文案有其固定的结构。今天介绍的 **PREP结构** 是经过众多演讲者、专家及知识博主长期实践总结出的经典框架，非常适合用于短视频表达。

**PREP** 是四个英文单词的首字母，分别代表：
*   **P**oint (观点)
*   **R**eason (理由)
*   **E**xample (案例)
*   **P**oint (再次强调观点)

这本质上是一个“总分总”结构，应用范围极广。

以下是PREP结构四个部分的详细说明：

1.  **P：抛出观点 (Point)**
    *   开头第一句话必须直接、清晰地抛出核心观点。短视频的“5秒完播率”至关重要，必须在最初5秒内让观众抓住重点，否则他们就会划走。

2.  **R：阐述理由 (Reason)**
    *   提出观点后，紧接着需要给出支撑这个观点的理由。理由通常需要2-3个，以确保观点有说服力。

3.  **E：列举案例 (Example)**
    *   在理由之后，通过举例子来进一步解释和印证你的观点。案例能使内容更生动、更易懂，通常建议使用2-3个例子。

4.  **P：重申观点 (Point)**
    *   在文案结尾，再次强调你的核心观点。这能帮助观众强化记忆，并提高他们对视频内容的认同感和点赞意愿。

**公式总结：观点 → 理由 → 案例 → 再次强调观点**

这个结构不仅适用于短视频文案，也适用于职场汇报、演讲、自我介绍等多种需要清晰表达观点的场合。理解了PREP结构，你就能看懂很多知识博主为何显得有理有据、非常专业。

### 课程总结 📚

本节课我们一起学习了创作优质短视频文案的两个核心步骤。

首先，我们探讨了**如何确定选题**，关键在于从市场验证过的、大众关心的主题入手，而非凭空想象。我们以读书博主为例，介绍了通过分析对标账号爆款内容来确定主题的方法。

其次，我们详细讲解了构建文案的**PREP结构公式**：**观点 (Point) → 理由 (Reason) → 案例 (Example) → 再次强调观点 (Point)**。这个结构能帮助你条理清晰、有说服力地表达任何观点。
