# 钱进社 - 成为优质且最具竞争力的【社会人】：22、什么样的工作人际重要，什么样的工作能力重要

## 概述
在本节课中，我们将探讨不同类型工作中人际关系和工作能力的重要性，帮助大家更好地理解职场生存之道。

## 工作类型与人际关系、工作能力侧重点

### 1. 公司规模
- **大公司**：人际关系更重要。大公司人员复杂，需要处理各种人际关系。
- **小公司**：工作能力更重要。小公司人员较少，更注重员工的工作能力。

### 2. 事业单位
- **事业单位**：人际关系更重要。事业单位规则固定，需要哄好领导。

### 3. 销售运营市场类
- **技术含量偏低**：人际关系更重要。这类工作需要处理人际关系，如销售、运营等。

### 4. 文员、文案等文科类
- **工作无数据衡量**：人际关系更重要。这类工作能力较难量化，需要上司重用。

### 5. 技术门槛高、创意性工作
- **技术门槛高**：工作能力更重要。这类工作需要专业技能，如技术类岗位。
- **创意性工作**：工作能力更重要。这类工作需要创意和创新能力，如自媒体、教师等。

### 6. 老板
- **小老板**：工作能力更重要。小老板需要具备工作能力。
- **大老板**：人际关系更重要。大老板侧重于战略方向和人际关系。

## 总结
本节课中，我们学习了不同类型工作中人际关系和工作能力的重要性。在职场中，我们需要根据自身情况和工作类型，合理分配精力，提升自身能力，同时处理好人际关系。
