## 三、 认识你的身形与穿搭原则

理解了美的本质后，我们需要通过穿搭来优化和展现它。首先，必须了解自己的身形。

以下是常见的五种身形及对应的穿搭要点：

*   **沙漏型**：肩宽与胯宽基本一致，腰部纤细。穿搭重点是**永远强调并勾勒出腰线**，避免上下装体积过大破坏平衡。
*   **长方形/直筒型**：肩、腰、胯宽度差异不大，腿往往较细。需要通过服装（如A字裙、腰带）**人为制造腰线**，并利用版型增加曲线感。
*   **梨形**：肩窄于胯。穿搭目标是**加宽肩部视觉**，可穿泡泡袖、露肩装等，同时也可强调腰线。
*   **倒三角型**：肩宽于胯。需**增加下半身体积感**，如穿浅色或蓬松的下装，上半身宜穿深色、修身的款式。
*   **椭圆形/苹果型**：腰部曲线不明显。关键是**找到并标记出躯干最细的部位作为腰线**，避免夸大上半身，并简化下半身设计。

确定身形后，可遵循 **“BECE”** 造型四步法：
1.  **Balance**：平衡你的身形。
2.  **Enhance**：加强你最漂亮的部位（如锁骨、细腰、长腿）。
3.  **Conceal**：巧妙隐藏你认为不足的部位。
4.  **Elongate**：根据身高拉长整体造型线条（如娇小个子善用高跟鞋）。
