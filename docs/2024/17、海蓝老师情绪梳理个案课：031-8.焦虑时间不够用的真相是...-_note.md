# 海蓝老师情绪梳理个案课：031-8.焦虑时间不够用的真相是...

## 概述
在本节课中，我们将探讨焦虑时间不够用的真相，并学习如何通过情绪梳理个案来理解并解决这一问题。

## 核心概念
**焦虑时间不够用**的真相是源于**潜意识中的恐惧和焦虑**，这种恐惧和焦虑往往与**过往的经历和模式**有关。

## 案例分析
以下是一个情绪梳理个案的案例分析，帮助我们理解焦虑时间不够用的真相。

### 案例背景
一位名叫红的女士，经常感到焦虑，觉得自己时间不够用。她非常努力地工作，但仍然觉得自己无法满足自己的期望。

### 情绪梳理过程
1. **身体承载位置**：红女士感到肩膀和胸口有压力，这是她焦虑的承载位置。
2. **过往经历**：通过引导，红女士回忆起初中时父母争吵的画面，以及妈妈经常抱怨家里没有钱的经历。
3. **潜意识逻辑**：红女士意识到，她潜意识中的逻辑是：只有有能力赚钱，才能不被瞧不起，才能获得幸福。
4. **父母关系**：红女士与父母的关系紧张，她觉得自己必须努力工作，才能证明自己的价值。
5. **夫妻关系**：红女士与丈夫的关系也出现问题，她担心丈夫瞧不起自己。

### 解决方案
1. **情绪释放**：通过情绪梳理，红女士释放了与父母和丈夫之间的负面情绪。
2. **认知调整**：红女士意识到，她不必活成父母的样子，也不必满足所有人的期望。
3. **放松和接纳**：红女士学会了放松和接纳自己，不再焦虑时间不够用。

## 总结
本节课中，我们学习了焦虑时间不够用的真相，并通过一个情绪梳理个案的案例分析，了解了如何通过情绪梳理来解决这一问题。希望我们能够从中获得启发，学会放松和接纳自己，过上更加幸福的生活。
