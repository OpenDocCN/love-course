# 20.心碎小k-让男人恋爱脑去吧！：006.第五课 如何防止被对方情感操控

🎼第五课，如何防止被对方情感操控？🎼

### 情感操控的特征

以下是情感操控的几个特征：

1. **情感绑架**：以付出为要挟，对方一开始体贴细致，满足你的各种愿望和要求，之后开始对你有所诉求。
2. **拒绝社交**：切断外界的评判，除了他之外，其他人对你都是虚情假意，都是利用，都是不怀好意。
3. **自恋否定打压**：自恋的人永远都认为他自己是对的，需要你的认同和赞扬，本质上这类型的男人自尊心非常强。
4. **自私以自我为中心**：自私自利的，他的决策是希望你为他付出，让他得到更多，最后的结果是你会发现你始终都是付出的那一方。

### 被情感操控的表现

以下是被情感操控的一些表现：

1. **以他人为中心**：逐渐开始特别听他的话，认为他的话就是对的，如果不按照他说的做，会害怕他生气、骂你，甚至打你。
2. **没有朋友**：和他在一起之后，发现身边一个能说话的人都没有了，没有其他社交，身边的朋友都不喜欢，他也不希望你们在一起。
3. **对自己没信心**：觉得自己配不上对方，当和一个人在一起的时候，没有变得越来越好，反而变得自卑敏感，没自信。
4. **不快乐**：在一段关系里，如果没有特别的快乐，也没有随心所欲的感觉，反而要尽力去讨好对方，时常感觉到自己不快乐。

### 如何防止被情感操控

1. **明确的自我认知**：了解自己是谁，想要成为什么样的人，具备什么以及欠缺什么，可以为了目标付出怎样的努力。
2. **独立自信**：对自己的评价，I don't care，我就是自信的，同时我也是独立的，我能够自己过得很好，有另一半可以，没有另一半我也可以过得照样精彩。
3. **兴趣广泛**：生活一定要很精彩，不要把自己局限在一个眼对眼的小世界里，去结识更多的朋友伙伴，去解锁更加广阔的世界。

### 总结

本节课中我们一起学习了如何防止被对方情感操控，希望大家能够从中受益，保护好自己的感情。
