# 18.辰薇魅力女性训练营：06打造无瑕底妆 🌟

## 概述

在本节课中，我们将学习如何打造无瑕且透亮的底妆，让肌肤看起来像灯泡一样光彩照人。

## 粉底工具

以下是打造完美底妆所需的工具：

1. **隔离**：隔离可以保护皮肤免受粉底中的铅和汞的伤害。
2. **遮瑕**：用于遮盖瑕疵和均匀肤色。
3. **修容**：用于修饰脸型，打造小V脸。
4. **粉底**：根据皮肤状况选择合适的粉底类型。
5. **定妆产品**：包括定妆喷雾、定妆粉和定妆粉饼。

## 隔离

以下是隔离产品的选择方法：

1. **紫色隔离**：适合皮肤发黄的人群。
2. **绿色隔离**：适合皮肤发红、有红血丝的人群。
3. **白色隔离**：适合皮肤白皙的人群。

## 遮瑕

以下是遮瑕产品的选择方法：

1. **橘色遮瑕**：用于遮盖黑眼圈。
2. **浅色遮瑕**：用于遮盖眼袋。
3. **深色遮瑕**：用于遮盖痘印。

## 粉底

以下是粉底产品的选择方法：

1. **BB霜**：含水量较高，适合皮肤较干的人群。
2. **CC霜**：含水量更高，适合皮肤较干的人群。
3. **粉底液**：含水量较低，适合皮肤肌底较好的人群。
4. **粉底霜**：含粉底量更高，适合皮肤瑕疵较多的人群。
5. **粉底膏**：含水量更低，适合舞台妆。

## 定妆

以下是定妆产品的选择方法：

1. **定妆喷雾**：让皮肤更加透亮，固定妆容。
2. **定妆粉**：用于定妆和补妆。
3. **定妆粉饼**：用于定妆和补妆。

## 实操

以下是打造无瑕底妆的实操步骤：

1. 护肤：使用爽肤水和玻尿酸。
2. 遮瑕：使用紫色修颜液遮盖瑕疵。
3. 打粉底：从中间开始打向四周，斜向上提升。
4. 定妆：使用定妆粉、定妆喷雾和定妆粉饼。

## 总结

本节课中，我们学习了如何打造无瑕且透亮的底妆，让肌肤看起来像灯泡一样光彩照人。希望同学们能够掌握这些技巧，打造出属于自己的完美妆容。
