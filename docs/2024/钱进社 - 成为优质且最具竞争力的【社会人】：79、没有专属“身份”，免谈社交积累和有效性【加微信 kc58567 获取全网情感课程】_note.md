# 钱进社 - 成为优质且最具竞争力的【社会人】：第79课、没有专属“身份”，免谈社交积累和有效性 🎯

在本节课中，我们将要学习社交中的一个核心概念：专属“身份”。我们将探讨为什么身份定位对于有效的社交积累至关重要，并分析四种关键的社交角色类型，帮助你找到并强化自己在社交网络中的位置。

## 概述

社交的有效性并非偶然，它建立在一个清晰的身份定位之上。这个“身份”并非指你已经取得的卓越成就，而是指你在特定社交环境中所扮演的、能够创造价值的独特角色。没有这个专属定位，社交行为就容易流于表面，难以积累真正有效的人脉资源。

上一节我们介绍了身份定位的重要性，本节中我们来看看如何具体找到并塑造自己的社交身份。

## 身份定位：社交的基石

很多人在社交中感到困难，除了性格差异，根源在于缺乏一个清晰的、专属的社交身份。这个身份是带引号的，它不要求你已经功成名就，而是指你在互动中能提供的独特价值点。

自身具备一定能力的人，社交起来会相对简单，但这并不保证能把社交做好。因为“有点东西”不代表能找准自己专属的位置。如果找不到能让你发光发热、并与他人需求相匹配的定位，你的能力也可能被埋没。世界上从不缺少“有点东西”的人。

真正的社交高手，除了外在性格，更关键的是能清晰地在特定环境和圈层中给自己定位，并将这个定位的价值最大化，从而成为一个难以被替代的人。宏观上，没有人是不可或缺的，但在特定的圈层中，通过精准定位，你的价值可以实现最大化。

在任何圈子和环境中想要站稳脚跟，都需要找到自己的不可替代性。这种不可替代性并非顶尖精英的专利，因为再厉害的人也有短板。选择社交这条路，本质是希望为自己积累生产资料。真正乐在其中的人是少数，大部分人是为了现实目标而进行社交。

## 从现有社会角色开始定位

对于工作需要社交或职业发展依赖人脉积累的人来说，定位可以从你眼下的社会角色开始。

**定位是可以调整的**。它会随着你获取的价值、个人职业走向以及社交本身的积累过程而演变。社交是一个渐进的过程，而非一蹴而就。没有人天生就拥有超高价值，社交能力本身——包括表达方式、情商、识人能力——也是在实践中积累的。

你需要深度自问：是逃避心态让你觉得自己不适合社交，还是你确实尝试了但做不好？前者需要勇气去突破，后者则需要迭代和复盘的方法。只有通过实践和结果，你才能认清自己更适合后端支持工作，还是前端开拓角色。

如果既不愿迭代复盘，又缺乏尝试的勇气，却整天抱怨社交困难、羡慕他人资源，那么自然难以在前端社交中有所建树。

## 社交中的四种关键角色类型

每个想要建立自身社交蓝图的人，都需要先了解社交中大概率存在的角色组合。我使用一些形容性质的标签做了以下分类，方便大家更有代入感地去感受和定位。

在不同的场合和圈层中，每个人的角色是变化的。例如，在家是子女/父母，在公司是员工/领导。无论面对何种情况，首要任务是忠于当下的角色，这是人际关系的基础。然而，忠于角色这个听起来简单的基础，在实际操作中却并不容易，因为人的本能是追求更多。

在做好“忠于角色”这个基础之后，更重要的是**定位自己现有角色的差异化**。你需要思考自己能带给别人什么，以及你提供的价值与他人有何不同。这种差异化不一定需要超级高的能力，每个层级都需要专业和靠谱的人。

大多数人对社交的定位存在偏差，认为社交就是即时获利或必须按自己设想的节奏推进。社交的本质是与人打交道，人是流动的，很难完全按照预设剧本发展。因此，你需要放弃不切实际的规划，从细微处（如形象、性格）到宏观处（如专业度、靠谱程度）找到自己的差异化价值。

不同的价值定位决定了不同的位置定位，进而产生不同的反馈。选择一个适合自己且有相对差异化的定位，你就能掌握更大的优势。

从社交本质出发，若要稳固且深入地经营关系，可以参照以下四种互相拉扯、彼此保障的类型：

以下是四种关键的社交角色类型：

1.  **专家型**
    *   这类人在社交圈中拥有知识、学识、信息或核心资源，代表权威，对圈子贡献最大。
    *   他们因拥有核心价值而具备话语权、说服力和影响力。
    *   专家型角色是长期积累形成的，难以被轻易替代，需求最大，通常别人会主动寻求交流。他们的聚众效果往往是被动产生的，主动权较强。
    *   **核心公式**：**稀缺知识/资源 + 权威性 = 被动吸引力与话语权**

2.  **融通型**
    *   这类人如同桥梁，擅长将不同的人、资源连接在一起，实现价值最大化。
    *   他们自身专业能力不一定极强，但拥有高说服力和社交情商，具备“曲线价值”。
    *   他们是稀缺人才，通常头脑清晰且懂得中庸之道，性格圆融、富有弹性，能自然地在不同圈层中穿梭、搭建关系。
    *   他们善于发现他人的优势，并能忽略对方的缺点，通过广结善缘不断积累人脉基数。
    *   **核心代码**：`function connectResources(personA, personB) { return synergyValue > (personA.value + personB.value); }` （连接资源产生的协同价值大于个体价值之和）

3.  **明星型**
    *   这类人不一定提供实际价值或专业顶尖，但拥有极高的煽动性和表达力。
    *   他们善于拿捏情绪（先掌控自己的，再影响他人的），通过情感链接引导他人，人格魅力突出。
    *   他们能快速将他人带入自己设定的“优势场”，高效推进目标。
    *   **核心公式**：**高超情绪感染力 + 人格魅力 = 快速建立信任与引导**

4.  **助理型**
    *   这是服务性质的类别，但不可或缺。它通常是许多人被贵人扶植的起点。
    *   这类人热心、主动、任劳任怨，将“勤劳”、“靠谱”等基本技能做到极致。
    *   虽然地位看似相对较低，但他们是信息流通和事务协调的关键节点，甚至他人需要先与他们搞好关系。在这个位置上可以锻炼能力、积累人脉。
    *   **核心代码**：`while(!expert) { accumulateExperience(servingOthers); buildNetwork(); }` （在成为专家之前，通过服务他人积累经验和人脉）

## 如何应用与迭代

这些分类没有绝对的好坏或面子之分。你需要做的是：

1.  **结合自身评估**：深刻评估自己的能力、性格（是温柔、强势还是有棱角），找到与你当前状态最匹配的、最容易切入的定位。性格特征是一种捷径。
2.  **先卡位，后迭代**：不要空想。先选择最容易接近的定位切入，在实践中锻炼自己。
3.  **动态调整**：定位不是固定的。随着你的成长、经验积累和价值变化，你可以调整自己的定位，向其他角色类型演进或深化。
4.  **行动优于空想**：对于没有醒目强项的人，可以从“助理型”或强化“靠谱”特质入手，先找到一个位置参与进去，在实战中学习、碰撞、获得反馈，这远比在脑中空转有效。

本节课中我们一起学习了社交中专属“身份”的核心重要性。我们明白了有效的社交始于清晰的自我定位，并深入分析了专家型、融通型、明星型和助理型这四种关键的社交角色。关键在于，要结合自身实际情况，选择一个可切入的定位开始实践，并在过程中不断迭代和调整。记住，社交是积累的过程，行动和复盘远比空想和焦虑更有价值。
