# 乐嘉性格色彩读心术（完结）·第014课：RC美学（笑死人不偿命）🎭

在本节课中，我们将跟随乐嘉老师一段特殊的个人经历，探讨在极端不便的生理条件下，如何通过“意念控制”来完成日常任务。这段经历虽然围绕“排便”展开，但其背后蕴含的专注、控制与解决问题的思路，却是一种独特的生活哲学。

## 🎼 术后恢复的挑战

你也许很脆弱。你也可以更坚强。听乐嘉亲身讲述因男人的脆弱引发的刀光剑影和内心独白。

大便的艺术手术以后，第一天不让吃东西，第二天和第三天必须吃流质，然后才能正常。因为前三天基本上啥都没吃，所以小便比较容易，但大便几乎没有。

护士小姐每次进到房间里面，都会特别关心的问我，今天排便了吗？今天排便了吗？我连续三天回答她没有，我内心特别内疚，我觉得对不起小护士。不过护士很善解人意，看到我这么羞愧的表情，她就安慰我，她说不要多想，有了告诉她就行了。

所以我看她这么安慰我，内心下定决心，我一定要争取早排多排，以报答护士姑娘的大恩大德。

## 🚽 如厕的困境与哲学

所谓世间贞洁烈女入内宽衣解带，天下英雄豪杰到此忍气吞声。前面我讲过了，因为我受伤的位置比较糟糕，所以小便一开始只能蹲坐式。不像以前有伤的时候，大便的时候可以顺带小便。但现在因为我小便一上来就把我大便的姿势给占据了，所以我也只能耐心等待。

我开始有第一次想要拉粑粑的欲望以后，才发现尿尿的麻烦不值一提。太用力气，我怕伤口被崩裂，虽然不是痔疮，但是毕竟是隔壁邻居，我担心城门失火殃及池鱼。但你不用力气，你根本啥都排不出来。

马桶上呆的越久，心情越烦躁，还要担心时间长了，本来没痔疮，结果搞出来个痔疮。我反复的寻找到底是用力和不用力，这两者的平衡点在哪里？

探索了半天以后，轰隆一声，天门开，一条黄龙冲下来。因为并没有那种便秘之苦，所以我也没有那种千呼万唤始出来、排空郁气奔如电的气势，生理快感有限。但是我只要一想到自己拉出来了，我终于可以对得起我的小护士了，我就心底上少了很多愧疚感。

## 🎯 “意念排便法”的诞生与要诀

在我坐在马桶上的时候，你知道我最害怕什么吗？我最怕的就是我下坠的宝物，直击马桶轻弹，激起涟漪，水波兴起，浪花四溅，正中纱布。让我的伤口感染，那就麻烦大了。尤其是如果马桶里的水位太高的话，你的蛋蛋就更容易沾到便池里面的水。

所以呢，在这种特别时期如厕，我用的都是一种**意念排便法**。这种意念排便法是有一些要诀的。

以下是“意念排便法”的核心操作步骤：

1.  **坐下马桶，身体前倾。**
2.  **想尽方法，让你的黄金宝物盘旋在马桶四周的陶瓷白壁上。**
3.  **暗暗用力，用意念让你的粑粑可以蜿蜒伸展，延绵不绝。**
4.  **当粑粑触及到谷底的时候，千万不能让它断裂**，因为万一撞击到水面，那可不是闹着玩的。

总之，你要想象，当你的粑粑从体内出来的时候，它的走势不能轻易的倒在水潭当中。一开始我认为这只是个运气，后来我才发现它是一个可以训练的技术活。总之我宁可在出宫以后多冲洗几次马桶，虽然麻烦，也比你的伤口被弄湿了后遗症要好得多。

## 🔗 从“意念排气法”到“意念排便法”

我创造的这个意念排便法，我认为很多人可能也会有，但这个是我自己给它起了个名字。这个方法它的起源，跟**意念排气法**有异曲同工之妙。

比方说你刚交了个男朋友、女朋友，两个人刚谈恋爱。谈恋爱的时候，你突然肚子不舒服，想要放屁，那你怎么办呢？你要憋屁，因为初恋的时候为了留下好印象，绝不能轻易放屁。万一响屁憋不住，你就要大声说话把它给掩盖住。但是最要命的是闷屁，没有声音，但是瞬间熏倒一片。

这种情况下只能用一招，要用嘴巴的功夫，要嘴形外松内紧，外面看不出来，装作若无其事，但实际上你里面在暗自吹气如兰，你又不好意思让别人发现是你自己干的，又想欲盖弥彰，想要让味道尽快的散去。

不管是响屁还是闷屁，为了避免这样的尴尬，万一憋不住的时候，人们最常见的最雅致的做法就是先行离开，放完再回。可是如果你万一当时不能离开，你就必须要憋屁。可是憋屁有可能会憋坏五脏哦。所以憋了半天没憋住，最后最要命的事情还一柱擎天到底，那这个时候就是赔了夫人又折兵，你就完蛋了。

长期实战过程当中，人们用的就是**意念排气法**。就是控制括约肌，一点一点地向外排放，做到无声无息，然后出来一点，就用嘴技让这个气消弭于无形之中，循环反复。

你看，意念排便法就是吸取了意念排气法的精华。它的差别是在于：排气的时候不能多，每次只能一丢丢；但是排便的时候必须要一触到底，而且速度不能太快，免得撞击太强烈。关于这方面的技术如何操作，需要结合避免尿液四溅的三大原则。如果你能够完美的结合在一起，我相信各位听众朋友很快你也可以掌握意念排便法的精髓。

在有限的资源和条件下，我既要完成小护士交给我的任务，又要确保我的伤口干净整洁，没有污染，这是我能想到的唯一的方法。可是如果你不用这个方法，你就只能祈祷，你没有在医院，你没有在马桶上，你换一个繁花似锦、青草葱葱的美景就可以解决了。

## 🏞️ 古今中外的“如厕风雅”

日本有一个著名作家谷崎润一郎，在他的《阴翳礼赞》当中，曾经描述过他一生当中见过的最美的拉粑粑的地方。那是有一次在乡野饭店里面如厕的经历，当他跨开两腿向下张望，只见几十米之下遍布野草，粉蝶飞舞于下坠的粪便之间，还有着油菜花盛开。真是便便如此，夫复何求啊。

但是这种描绘的方法，和我们国家元代的时候有个著名画家叫倪云林先生相比，还差得太远。这位倪大师每次上厕所，都会用大量的飞蛾的翅膀来代替细沙，放在壶中垫底，粪落其中不露痕迹呀。

在我看到的现代著名诗人百华老师有一本诗集《一点墨》当中，他提到了倪大师更夸张的一件事。他会用大量雪白的鹅毛来掩盖自己拉出来的便便。这古语是这么说的：“殿下鹅毛起覆之，童子在其旁俟亦去，不闻有晦气。”

所以，我只能一方面羡慕古人的风雅，另外一方面感慨，风雅是需要很多很多的**money**做代价的。

## 📜 如厕文化的考据与现实的窘迫

再回到我自己吧。反正我手术以后的第一次便便快要结束了。最悲催的事情是，在卫生间里面找不到擦屁屁的纸。我的卷筒纸的卷轴空空如也，马桶盖上啥也没有。我本来想趁住院的时候写点东西，所以陪护的人员早就被我赶走了。如果我打铃铛叫护士来送擦屁股纸给我，我会觉得在护士面前不好意思。

其实我想了半天，我也没想明白。护士每天看到我的身体，我都不会觉得羞愧，可是他送手指给我，我自己就会羞愧。什么原因呢？我觉得有一种可能的原因，就是在我的潜意识里面，我觉得我的身体是干净的，便便是不干净的。

所以如果我自己都有这么想法的话，我就更加有足够的理由相信，正在看我这本书的读者、正在听我讲这段话的听众里面，一定会有一部分的朋友不适应，认为我为什么可以如此大言不惭的讨论关于屎屁尿的问题呢？其实我再次想跟大家报告和重复的事情是，这些东西都是在每个人的身体当中最真实的部分，我其实只是想做一个忠实的历史的记录者而已。

我在马桶上面待了10分钟，想尽各种方法都不行。我想撅着屁屁移到房间里面取纸巾，如果在家里面我就这么干了，但现在因为我刚做完手术，行动不便。

在古代的笔记里面，我特别想看到有哪个古代的著名的文人如厕无纸，用手指头擦完以后洗干净。但在现实生活当中，我相信一定会有。但是在古人的记载当中，我是没看到。这种行为感觉上会让自己觉得像原始人返祖，不到万不得已、无计可施，我想一般人是不会用的。你想想这个画面，都会挺抓狂的。

你知道古代的时候，古代人没有纸啊，在元朝以前大家都是用竹片或者木头片来擦屁股的，皇帝老儿也不例外。但是竹子的毛刺太多，有的时候会把屁股给伤了。在南唐书里面就有记载，南唐后主李煜，他会亲自动手削竹片，供僧徒如厕使用。而且你知道他是一个多么虔诚的信徒啊，他为了不让这个和尚们的屁股被竹刺所伤，他会用他自己的脸来检验竹子是否光滑。

在一本佛经《毗尼母经》卷里面也详细记载了如来佛制定的比丘如厕的规矩。其中有一段是这么说的：便便时要慢慢的撩起衣服，蹲下，不可以突然唰的一下子就把身体暴露出来。拉完了以后，只能够用木头、竹子或者竹尾刮干净，不允许你用石头、青草、土块、软木或者树叶来擦。

你看，这种真是长了好多知识跟学问。我看了看我的卫生间，石头、青草、土块、软木、树叶，这都对我来讲是奢侈品，啥都没有。我又看了看这个洗脸台上没有开封过的新牙刷，我后来在想，我如果拿那个牙膏的纸壳来刮会不会太疼了。我看了看挂着的毛巾，也许我可以把毛巾擦干净以后，再把它消毒消干净，但是又觉得太不道德了。我把我可不可以把毛巾给扔掉，这样我就当我花钱买一条毛巾好了。

## 🦢 文学中的终极解决方案与现实的救星

这个时候我突然就在梦想该怎么办的时候，我就最想，我想你一听，有一只小鹅在我的面前。

在16世纪，法国有一个杰出的小说家叫拉伯雷，拉伯雷写过一部著名小说《巨人传》。主人公说过一段话，我到现在都记得，原话是这么说的：“哦，所有擦屁股的东西什么都比不上一只绒毛丰满的小鹅。不过拿它的时候，你必须把它的头关照它的两腿当中。我以我的名誉向你担保，你完全可以相信，因为肛门会感受到一种非凡的快感，既有绒毛的柔软，又有小鹅身上的温暖，热气可以植入你的大肠和小肠，上灌到你的心脏和大脑。别以为极乐世界的那些神仙的享受，只是百合花、仙丹或者花蜜，它们的享受其实真正就是用小鹅来擦屁股。”

在我犹豫要不要打铃，不知道该怎么办，以及在做梦小鹅的这个档口，咣，我的门被推开了，护工阿姨的声音响起来了，啊，我的救星来了。我和阿姨要纸一点都没觉得有啥不好意思。

我突然想起来，为什么前面我向那个护士要纸会不好意思呢？我终于找到了一个原因，根深蒂固的原因，我不敢承认的原因，发自人心原始的原因，一个让我自己都觉得特别兴奋的原因。那是因为，我对阿姨毫无企图心，但是我对小护士少不了有一些暧昧的想要去勾搭的想法。虽然很遗憾，我一直没有什么实质性的举动，但是我心里面有鬼，总觉得像一个姑娘要那个卫生纸，既没有儒雅，又没有豪情。但在老阿姨面前全然没有了这样的顾虑，所谓无所求则无所惧，有所求则有所忧。

## 💰 “艺术家的屎”与自我欣赏

我看着马桶里面自己排出来的便便，我觉得和我心目当中真正的好粑粑还是有很多距离的。我所理解的好粑粑最高境界要具备四大指标：金光条条、貌似焦黄、粗细均匀、微臭无熏。因为这几天我也没吃啥荤的东西，粑粑闻起来的味道呢就像刚刚腌过的咸带鱼一样，味道差不多，指标勉强过关。但是在色泽、软硬度和形状这三个指标上，我觉得都还有巨大的空间可以进步。但是毕竟这是我手术以后的第一场粑粑，我必须要鼓励一下自己，毕竟还是有了一个好的开始。

1961年，记住哦，1961年，意大利有一个艺术家叫皮埃罗·曼佐尼。他把自己拉出来的粑粑装到90个罐头里面密封好，每个罐头装了30克他的粑粑，罐头上还有他自己的签名和编号。他创作出来90罐作品，作品的名字叫《艺术家的屎》。他打算自己死了以后，把这些大便按照黄金的市场价格来卖。

结果这个人30岁的时候就死翘翘了。在他死了以后44年，也就是在2007年的时候，他当时那90罐粑粑编号18号的《艺术家的屎》在苏富比的拍卖会上拍出了12.4万欧元，也就是150万人民币。换算一下，它的一克粑粑就等于200克黄金的价格。

我记得当时那个艺术博物馆的发言人是这么说的：“他的这件作品对20世纪艺术的许多问题进行了探索……是一件人类历史上开创性的作品。”我真的觉得收藏者的心态我是不懂的，我们这种穷人不搞收藏也不懂。我就看有些艺术家在白纸上随便画了三根线条，就卖个几千万美金。我看看我女儿画的那个作品，再怎么看都比那三根线条要美，这说明什么呢？说明我不懂艺术，要么就说明名人效应很牛逼，还有一种我在嫉妒别人。

我在想，如果有人愿意出十分之一的价格买我的粑粑，啊不，百分之一好了，不，千分之一我也卖。对，如果有人愿意出那个艺术家的屎的千分之一的价格买我的粑粑，我现在马上就卖，绝不食言。我会从我拉的便便里面精挑细选，颜色不是暗黄的不要，形状不成条的不要，太粗的不要，太细的不要，味道太臭的不要，无味的也不要。我会按照特级粑粑的标准，绝不偷工减料，全球只此一份，绝不复制，这样才对买家才是公平的买卖。

## 📸 历史的纪念与最终的告别

我仔细看了自己拉出来的粑粑，看了好一会儿。我想把它冲掉，但是转念一想不行。就算我的粑粑卖不出来什么好价钱，不可能名垂青史，可是别人可以瞧不起我的粑粑，我自己一定要瞧得起，对不对？好歹我要给自己留下一个念想。所以我强忍住我的疼痛，一步一步的移到床头旁边，把我的手机拿回来，然后重新再拖回自己到马桶旁边，就那么三四米的路，我走了有一个世纪那么漫长。

我再次回到马桶旁边，看到我的粑粑在那里，巍然不动，神采依旧，他并没有因为在水潭里面被稀释的散开来，仍旧很坚挺的在那里屹立着。我努力站着，双腿颤抖，拿起手机对准马桶，我就拍了三张极其富有历史意义的照片。我就想把它冲掉，不行，就这样冲掉了，谁知道这是我拉的粑粑呢？口说无凭，立照为据，对不对？好歹也要发个朋友圈。

我呢就继续，然后，我就扶着墙壁把身体转了180度，慢慢的蹲下来，我把我的头靠近马桶，左手死死的抓着输液架，右手举起手机，瞄准镜头，我露出了特别真心的微笑，咔嚓。人便合一，一次成功，我终于拍下了我这一生当中最最最最变态的一张自拍照。

我再次从痛苦当中缓慢的爬起来，恋恋不舍的看了我的宝贝粑粑，狠心的按下马桶的按钮。我目睹着旋转的水流，连同这个了不起的粑粑烟消雾散。良久我都沉浸在自拍的喜悦之中。终于，我可以向我的护士小妹妹有个交代了。

## 📚 本节课总结

在本节课中，我们一起学习了乐嘉老师在术后特殊时期的一段真实经历。我们从**术后排便的生理与心理挑战**开始，探讨了在极端不便下发明的 **“意念排便法”** 及其操作要诀。接着，我们通过对比**“意念排气法”**，理解了这种控制技巧的思维来源。然后，我们跨越古今中外，领略了从日本作家谷崎润一郎到中国元代倪瓒的**如厕风雅文化**，并考据了从南唐后主到佛教经典的**如厕规矩**。最后，从法国文学《巨人传》的幻想，到意大利艺术家曼佐尼的天价粪便作品《艺术家的屎》，我们以一种幽默而深刻的视角，重新审视了身体、隐私、价值与自我接纳的关系。这段经历的核心在于：**在限制中创造方法，在窘迫中保持幽默，在真实中寻找意义**。
