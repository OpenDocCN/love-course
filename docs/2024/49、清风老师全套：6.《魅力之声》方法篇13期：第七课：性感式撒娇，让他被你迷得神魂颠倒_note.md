# 清风老师全套：6.《魅力之声》方法篇13期：第七课：性感式撒娇，让他被你迷得神魂颠倒

## 概述
在本节课中，我们将学习如何通过撒娇技巧，提升与暧昧对象、男朋友或老公的关系，使其更加亲密和吸引人。

## 教程内容

### 1. 撒娇的目的
通过撒娇，拉近与暧昧对象、男朋友或老公的距离，使其更加渴望与你亲近。

### 2. 撒娇技巧
以下是撒娇的几个关键技巧：

#### 2.1 制造场景和节奏
通过声音和肢体动作，营造浪漫氛围，激发对方兴趣。

#### 2.2 释放信号，激发兴趣
通过言语和行为，向对方传递你的兴趣和好感。

#### 2.3 后撤，引导对方追求
在表达兴趣后，适当后撤，引导对方主动追求你。

#### 2.4 表达兴趣，激发兴趣后撤
循环表达兴趣，激发对方兴趣，然后后撤，形成闭环。

### 3. 撒娇案例解析
以下是对视频中撒娇案例的解析：

#### 3.1 案例一：打羽毛球
通过夸赞对方身高，拉近物理距离，激发心动感觉。

#### 3.2 案例二：换衣服
打破常规思维，表达出对对方的兴趣，拉近心理距离。

#### 3.3 案例三：打球
通过询问对方是否有腹肌，激发对方兴趣，并引导对方追求。

#### 3.4 案例四：学习打羽毛球
通过示弱，激发对方保护欲，并给予正向反馈。

#### 3.5 案例五：比赛
通过提出条件，引导对方做出承诺，并逐步增加投入。

### 4. 撒娇公式
释放信号 → 激发兴趣 → 主动追求 → 提出要求 → 完成要求 → 正向反馈 → 提出下一个要求

### 5. 登门槛效应
在完成一个要求后，对方会倾向于满足下一个要求，因为要保持一致性。

## 总结
本节课中，我们学习了性感式撒娇的技巧和案例，通过撒娇提升与暧昧对象、男朋友或老公的关系。在下一节课中，我们将学习需求式撒娇，结合hoke上瘾模型，形成完整的闭环。
