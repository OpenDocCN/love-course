# 钱进社 - 成为优质且最具竞争力的【社会人】：第95课、如何快速加深印象，建立连结 🎯

在本节课中，我们将要学习如何快速与他人建立连接并加深印象。核心在于理解人性，找到共同点，并通过提供精准的价值来达成目标。我们将分步骤解析具体方法，并辅以实例说明。

## 概述

建立连接的核心是找到与对方的**共同点**。许多人误以为良好的交流是关键，但在现实中，没有共同点的交流往往缺乏基础。本节将探讨如何识别并利用共同点，以及如何根据不同人群的特点调整策略。

## 找到共同点：连接的基础

上一节我们概述了核心思路，本节中我们来看看如何具体操作。与他人建立连接时，唯一不变的核心是找到**共同点**。共同点分为显性和隐性两种。

*   **显性共同点**：例如，双方都来自上海。
*   **隐性共同点**：例如，双方私下里都最喜欢吃泡面。

“物以类聚，人以群分”的底层逻辑不会改变。相似的人在一起会形成天然的交集和安全感。每个人身份中自认的标签，就是识别“自己人”的关键。

以下是常见的身份标签类别：
*   年龄、时代
*   兴趣、审美、文化
*   地域、家庭
*   工作、民族、国籍
*   观点、信仰
*   社会层次
*   情绪、美食
*   性取向

例如，在海外见到华人会感到亲切；在都是上海人的工作环境中，老乡身份能快速建立连接。

## 顺应人性：获取好感的关键

想要和陌生人建立连接，首要目标是获得对方的好感。人性最关注的是自己是否得到重视和认可。对方的视角是以自我为中心的。

你的任务是：
1.  允许对方以自我为中心。
2.  找到对方希望被关注、重视和认可的点。
3.  给予对方更多的认可和重视。

这是顺着人性来的方法。想要快速加深印象，必须顺应人性。

## 快速分类法：四种人群的应对策略

一个快速但不绝对严谨的方法是，初期根据年龄和性别将人分为四大类。以下是针对每类人群的具体策略。

### 1. 面对年长男性：给予极度尊重

年长男性最需要的是**尊重**，甚至是极度的尊重。他们最骄傲的是自己的人生阅历。

切入点应是**请教**：
*   请教人生经验、做人道理、职场心得。
*   让他们扮演老师、教授的角色。
*   关心政治，与他们有共同话题。

即使你的观点略显稚嫩也没关系，年长者通常会给年轻人成长空间。但如果你对时事一无所知，只关注娱乐八卦，则可能被归为“口水型年轻人”，被他们看不起。

此外，称赞他们的身体和精力旺盛，例如说“您精力真好，我们年轻人都比不上”，会让他们非常受用，这满足了他们“宝刀未老”的心理。如果对方提及子女，积极称赞其子女，并补充一句“这都是您教导有方”，效果更佳。

### 2. 面对年长女性：给予关爱与认可

年长女性内心深处渴望的是**关爱**。她们可能面临被伴侣和孩子忽视的中年危机，自我意识容易偏向负面。

你需要做的是唤醒她们的**认可和自信**。具体方法是**夸赞细节**：
*   夸赞保养得好、气色佳。
*   夸赞审美在线、穿搭有气质、配饰点睛。

例如：“您今天这个胸针特别显气质，很有职业范儿。”这些生活中的细节可能已无人关注，你的关注就成为了稀缺的**无形价值**。女性需要被语言滋养，年长女性尤其缺乏这种滋养，你的赞美就是一种无形的保养品。

### 3. 面对同龄男性：营造“好相处”感

同龄男性通常对运动、旅行、工作、电影、游戏等话题感兴趣。如果有相似的教育或地域背景，更容易拉近距离。

关键是要营造一种“**好相处**”的感觉。性格上可以显得大咧咧、不拘小节、接地气，让他们觉得你虽然性别为女，但没有太多情绪，不矫情，不抠细节。这会让他们感到轻松，不累。

**重要提醒**：如果你情商不足或情绪不稳定，**不要**释放出可能将私生活与工作混为一谈的信号（如暧昧）。一旦开启错误模式又处理不好，会适得其反。人对自我的认知很重要，并非所有捷径都适合每个人。

### 4. 面对同龄女性：放下身段，甘当绿叶

与同龄女性打交道，最忌讳“端着”。即便对方很端着自己，你也要尽量打开自己，表现得接地气、粗线条，甚至可以用自嘲。

同龄女性间容易产生不易察觉的嫉妒和攀比。因此，你需要**甘当绿叶**，把红花角色让给对方。但请注意，你的个人档次不能太低（用公式表示即 `个人档次 >= 对方感知档次`）。例如，对方是爱马仕，你可以是LV或香奈儿，但不能是小CK。

如果你本身档次不低却愿意衬托对方，对方会觉得“这是我应得的”；如果你本身档次较低，对方则会认为“你本来就该当绿叶”。这个尺度需要把握。

## 深化连接：让对方记住你

在主动寻找共同点的同时，我们也可以同步操作，让对方主动记住我们。以下是三个关键步骤。

### 1. 强化共同点

共同点是第一顺位。如果这一点很模糊或完全没有，后续连接会非常生硬，任重道远。

### 2. 打造视觉与需求记忆点

你需要打造属于自己的亮点。

*   **视觉记忆点（聚焦颈部以上）**：在人群密集的场合，人们的视线通常只能扫到颈部以上。可以利用醒目的配饰（如亮色头箍、独特发型）来制造视觉焦点，让人快速记住你。
*   **需求记忆点（判断并满足深层需求）**：通过细心观察，判断对方的潜在需求并提供价值。
    *   **实例**：作者每次见一位忙碌的医生都会戴黄色头箍、穿固定衣服（视觉点），并附带一杯鲜榨果汁和一份点心（需求点）。作者判断医生作为母亲，可能无暇顾及孩子，这些健康零食可能被转交给孩子。这种持续、独特且充满关怀的举动，让医生深深记住了她。

### 3. 提供精准价值

当你还没有具体资源进行置换时，**服务价值**和**情绪价值**就是最重要的价值。

*   **观察细节**：留意对方办公室的摆设（乐器、运动器材）、手机屏保（全家福、孩子、宠物），这些都能揭示其兴趣和生活中最重要的部分。
*   **提供对应价值**：
    *   对方感冒，顺手带感冒药。
    *   对方有乐器，邀请听音乐会。
    *   对方屏保是宠物，分享宠物相关话题。
*   **判断核心需求**：例如，面对一个急性子，他的核心需求是**不要浪费他的时间**。在提问前，可以先说“耽误您3分钟，我有两个问题”，给出具象的数字和框架，这比直接提供情绪价值更重要。

心要细。成功人士在关键事务上往往心细如发。你可以不必处处心细，但在达成目标的关键环节上，必须调动起细致的观察力和判断力。

## 总结

本节课中，我们一起学习了快速加深印象、建立连接的完整方法。

1.  **核心基础**是寻找并利用**共同点**。
2.  **根本原则**是**顺应人性**，满足对方被重视、被认可的需求。
3.  **实践策略**是采用**快速分类法**，针对年长男性、年长女性、同龄男性、同龄女性四类人群，采取不同的尊重、关爱、共情、衬托策略。
4.  **深化技巧**在于同步打造**视觉记忆点**、通过细心观察满足对方**深层需求**，并提供精准的**服务与情绪价值**。

记住，所有方法都需要结合耐心、细心和对需求的判断。初次建立连接时，避免贸然深入家庭话题，可通过试探观察。深入的邀约应建立在已发现共同点的基础上。通过有意识的练习，你可以更有效地与他人建立深度连接。
