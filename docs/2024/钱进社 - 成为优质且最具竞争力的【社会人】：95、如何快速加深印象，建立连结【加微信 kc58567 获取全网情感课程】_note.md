# 钱进社 - 成为优质且最具竞争力的【社会人】：95、如何快速加深印象，建立连结

## 概述

在本节课中，我们将学习如何快速加深印象，与他人建立连接。通过找到与对方的共同点，我们可以更有效地进行交流，并建立更深层次的联系。

## 找到共同点

**核心概念**：找到和对方的共同点。

**公式**：共同点 = 显性特征 + 隐性特征

**步骤**：

1. **显性特征**：例如，共同的兴趣爱好、地域、职业等。
2. **隐性特征**：例如，相似的价值观、生活经历等。

## 顺着人性来

**核心概念**：顺着人性来，找到对方想要关注的东西。

**步骤**：

1. **了解对方**：了解对方的兴趣、需求、价值观等。
2. **提供价值**：提供对方感兴趣的内容或帮助对方解决问题。

## 不同年龄、性别划分

**核心概念**：根据年龄和性别划分，找到合适的沟通方式。

**步骤**：

1. **年长的男性**：尊重、请教、关心政治。
2. **年长的女性**：关爱、认可、夸奖细节。
3. **年龄相仿的男性**：运动、旅行、工作、电影、音乐等。
4. **年龄相仿的女性**：落地、粗线条、自嘲、不low。

## 释放信号

**核心概念**：主动释放信号，让对方记住你。

**步骤**：

1. **共同点**：强调共同点，让对方感受到共鸣。
2. **亮点和特点**：展示自己的亮点和特点，让对方印象深刻。
3. **为对方提供价值**：提供对方感兴趣的内容或帮助对方解决问题。

## 总结

本节课中，我们学习了如何快速加深印象，与他人建立连接。通过找到共同点、顺着人性来、根据年龄和性别划分、释放信号等方法，我们可以更有效地与他人建立联系。希望这些方法能帮助你在人际交往中取得更好的效果。
