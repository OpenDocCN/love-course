# 课程01：Python基础入门 🐍

在本节课中，我们将要学习Python编程语言的基础知识。我们将从最核心的概念——变量开始，了解如何存储和操作数据。然后，我们会学习Python中的基本数据类型，它们是构建所有程序的基础。最后，我们将通过一个简单的示例，将所学知识串联起来，完成一个完整的程序。

## 变量：数据的容器 📦

上一节我们介绍了课程的目标，本节中我们来看看什么是变量。变量是编程中用于存储数据的容器。你可以把变量想象成一个贴有标签的盒子，标签是变量的名字，盒子里存放的是数据。

在Python中，创建一个变量并给它赋值非常简单。你只需要写出变量名，然后使用等号 `=` 将值赋给它即可。

以下是创建变量的基本语法：
```python
variable_name = value
```

例如，我们可以创建一个名为 `message` 的变量，并将字符串 `"Hello, World!"` 存储在里面：
```python
message = "Hello, World!"
```

## 基本数据类型 🔢

上一节我们学会了如何创建变量，本节中我们来看看Python中有哪些基本的数据类型可以放入这些“盒子”里。数据类型决定了数据可以进行的操作。Python有几个内置的基本数据类型。

以下是三种最常用的基本数据类型：

1.  **整数 (int)**
    *   表示没有小数部分的数字，例如：`-10`, `0`, `42`。
    *   代码示例：`age = 25`

2.  **浮点数 (float)**
    *   表示带有小数点的数字，例如：`3.14`, `-0.001`, `2.0`。
    *   代码示例：`price = 19.99`

3.  **字符串 (str)**
    *   表示文本信息，需要用单引号 `' '` 或双引号 `" "` 括起来。
    *   代码示例：`name = "Alice"`

## 一个简单的程序示例 ✨

现在，让我们把前面学到的变量和数据类型结合起来，编写一个简单的程序。这个程序将完成以下任务：存储一个用户的名字和年龄，然后打印出一条问候语。

以下是完整的程序代码：
```python
# 定义变量
user_name = "张三"
user_age = 30

# 打印问候语
print("你好，" + user_name + "！")
print("你今年" + str(user_age) + "岁了。")
```

在这个示例中：
*   我们创建了两个变量：`user_name`（字符串类型）和 `user_age`（整数类型）。
*   我们使用 `print()` 函数将文本输出到屏幕。
*   在打印年龄时，我们使用了 `str()` 函数将整数 `user_age` 转换为字符串，以便能与其他字符串连接（用 `+` 号）。

运行这个程序，你将在屏幕上看到：
```
你好，张三！
你今年30岁了。
```
