# 婚姻经营+创伤治愈私教项目：013.Week 3-4 Lesson 2 安全型依恋模式与内在运作模式

在本节课中，我们将探讨安全型依恋模式及其内在运作模式，了解它们如何影响我们的情绪表达、认知解读和行为表达方式。

## 1. 内在运作模式

**定义**：内在运作模式是指个体在成长过程中形成的情绪表达、认知解读和行为表达方式。

**公式**：内在运作模式 = 情绪表达 + 认知解读 + 行为表达

**关联**：内在运作模式与依恋理论密切相关，它反映了个体在早期成长环境中的依恋类型。

## 2. 依恋理论

**定义**：依恋理论是研究个体与主要照顾者之间情感联系的理论。

**核心概念**：

* **安全型依恋**：个体在成长过程中与主要照顾者建立了稳定的情感联系，能够信任他人，并在亲密关系中感到安全。
* **焦虑型依恋**：个体在成长过程中与主要照顾者之间的情感联系不稳定，容易焦虑，担心被抛弃。
* **回避型依恋**：个体在成长过程中与主要照顾者之间的情感联系较少，倾向于独立，不太依赖他人。

## 3. 安全型依恋模式

**特点**：

* **情绪表达**：能够自由地表达自己的情绪，包括正面和负面情绪。
* **认知解读**：对自我和外界有积极的认知，相信自己值得被爱，世界是安全的。
* **行为表达**：愿意与他人建立亲密关系，并能够照顾他人的情绪。

**公式**：安全型依恋模式 = 自由的情绪表达 + 积极的认知解读 + 亲密关系

## 4. 焦虑型依恋模式

**特点**：

* **情绪表达**：容易焦虑，担心被抛弃，需要他人的关注和照顾。
* **认知解读**：对自己和他人的认知较为消极，容易怀疑他人的意图。
* **行为表达**：在亲密关系中容易表现出依赖和焦虑。

**公式**：焦虑型依恋模式 = 焦虑的情绪表达 + 消极的认知解读 + 依赖的行为表达

## 5. 回避型依恋模式

**特点**：

* **情绪表达**：压抑自己的情绪，不太表达自己的需求。
* **认知解读**：对自己和他人的认知较为消极，容易感到自卑。
* **行为表达**：在亲密关系中倾向于独立，不太依赖他人。

**公式**：回避型依恋模式 = 压抑的情绪表达 + 消极的认知解读 + 独立的行为表达

## 6. 亲密关系中的治愈

**核心概念**：

* 亲密关系是治愈依恋模式的重要途径。
* 通过练习脆弱、分享情感，可以逐渐改善依恋模式。

**公式**：亲密关系 = 治愈依恋模式

## 总结

本节课中，我们学习了安全型依恋模式及其内在运作模式，了解了它们如何影响我们的情绪表达、认知解读和行为表达方式。通过了解自己的依恋模式，我们可以更好地理解自己的行为，并在亲密关系中寻求治愈和成长。
