# 乐嘉7套：7.乐嘉性格色彩读心术（完结）：16课：怎样从办公桌和饭桌一眼读懂同事性格？_note

在本节课中，我们将学习如何通过观察同事的办公桌和饭桌上的行为，快速判断他们的性格色彩。掌握这些技巧，可以帮助你更好地理解职场中的合作伙伴，提升沟通与协作效率。

---

## 一、通过办公桌判断性格色彩 🖥️

办公桌的布置与整理方式，能直观反映一个人的性格特点。以下是四种性格色彩在办公桌上的典型表现。

### 1. 红色性格
红色性格不喜欢一成不变的工作环境，总想在可支配的领域内搞些新花样。  
如果公司不严格要求整洁，红色性格的办公桌常堆满个人物品，例如零食、装饰品等。  
即使公司规定办公桌不能放置与工作无关的物品，红色性格也会设法用彩色羽毛笔、特殊徽章或奇特装饰品来凸显自己的与众不同。  
若职位较高、拥有大办公桌，红色性格最可能摆放奖杯和奖状，因为获得他人认可和称赞对他们至关重要。

### 2. 蓝色性格
蓝色性格注重规则与秩序，即使公司没有要求，他们也会将桌面整理得整齐简约。  
蓝色性格对物品的摆放位置非常精确，若被移动能迅速察觉并归位。  
他们同样关注物品细节：铅笔需削好、圆珠笔需有墨水、水杯会及时清洗，不会积攒污垢。  
例如，一位蓝色性格的伞用了十年仍如新的一般，折叠痕迹整齐、一尘不染。

### 3. 黄色性格
黄色性格重视效率，抓大放小，办公桌上物品很少。  
他们只放置每日必需品，如电脑、笔筒、个别文件，以最大限度降低干扰，快速完成工作。  
当然，“必要”的定义因人而异：例如，一位相信风水的黄色性格可能在桌上放招财树，此外仅留电脑和电话。  
黄色性格批判性强，不认可同事在桌上放置不必要物品，因为他们随时处于“战斗状态”，认为轻装上阵对达成目标至关重要。

### 4. 绿色性格
绿色性格以人际关系为导向，欲望较低。  
若有独立办公室，绿色性格的桌面也较简单，可能摆放小型多肉植物，营造温馨氛围。  
若在敞开办公区与同事共用空间，绿色性格的桌上物品会逐渐增多——这些物品往往不属于他们自己。  
例如：同事分享食物，绿色不好意思拒绝便收下直至发霉；红色性格物品太多放不下，占用绿色桌面空间，绿色也不会提醒或拒绝。这体现了绿色性格的“懒”、不愿得罪人、以及对他人的包容。

## 二、通过饭桌行为判断性格色彩 🍽️

同事聚餐是观察性格的绝佳场合。不同性格色彩在饭桌上的表现差异明显。

### 1. 红色性格
红色性格在饭桌上通常活跃，主动发起话题，谈论工作以外的事情。  
他们最担心冷场，会努力找话、招呼他人、主动倒水、推荐菜品，并乐于分享个人生活点滴。  
红色性格容易成为饭桌主角，因为他们喜欢讲故事。但心情不好时，他们会显得沉默寡言，情绪变化明显，容易被人察觉。

### 2. 蓝色性格
蓝色性格在饭桌上较为谨慎，少言寡语，多是对他人话题做出必要回应。  
即使周围人聊得热火朝天，蓝色性格也可能显得置身事外。  
例如点菜时，蓝色不会主动揽活，因为他们考虑因素太多，不愿轻易承担。  
但用餐结束后，蓝色会主动核对账单，检查菜品是否上齐、价格是否正确，体现其谨慎细致的特质。

### 3. 黄色性格
黄色性格讨厌浪费时间，因此不会参加无意义的饭局。  
工作聚餐中，除非有重要人物出席，否则黄色性格可能不会参与。  
即使参加，他们也会在用餐后率先离开，不会参与闲聊。  
黄色性格对“上菜慢”容忍度最低，可能因米饭上得慢而直接离席，不顾已上桌的菜品。

### 4. 绿色性格
绿色性格以“磨蹭”闻名，反应较慢且重视人际关系，可能导致用餐时间延长。  
例如：看到好吃的菜，绿色会等别人夹完再动筷；若红色性格夹来不喜欢的菜，绿色为照顾对方感受也会慢慢吃完。  
聊天时绿色不主动发言，但会聆听，并在冷场时随口附和，因此往往最后吃完。

## 总结 📝

本节课中，我们一起学习了如何通过办公桌和饭桌上的行为快速判断同事的性格色彩。  
- **办公桌**：红色爱展示、蓝色重整洁、黄色求效率、绿色易被占用空间。  
- **饭桌**：红色活跃、蓝色谨慎、黄色匆忙、绿色磨蹭。  

掌握这些观察技巧，可以帮助你更深入地理解职场伙伴，促进更有效的沟通与合作。
