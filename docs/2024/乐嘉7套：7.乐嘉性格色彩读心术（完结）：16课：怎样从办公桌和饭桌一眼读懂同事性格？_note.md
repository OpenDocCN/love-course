所以今天我教你两招来快速判断你职场上周围人的性格色彩。第一招看办公桌。

## 办公桌观察法

### 红色性格

- 不喜欢一成不变的工作环境
- 办公桌上堆满个人物品，如零食
- 可能使用彩色羽毛笔或特殊装饰品
- 高职位者可能放奖杯和奖状

### 蓝色性格

- 注重规则和秩序
- 办公桌整洁简约
- 物品放置位置精确
- 注重物品细节，如铅笔和饮料杯

### 黄色性格

- 重视效率，抓大放小
- 办公桌上物品很少，仅放必需品
- 批判性强，不认可不必要的东西
- 可能放置招财树等风水物品

### 绿色性格

- 以人际关系为导向
- 办公桌上物品较少，喜欢温馨氛围
- 可能放置小型植物
- 在大空间办公时，桌上物品可能增多

## 饭桌观察法

### 红色性格

- 活跃，主动发起话题
- 分享个人经历，如电影、书籍、家庭事件
- 喜欢讲故事，喜怒哀乐易察觉

### 蓝色性格

- 谨慎，少言寡语
- 跟随他人话题回应
- 结束用餐时检查账单，确保无误

### 黄色性格

- 饭桌上时间最少
- 讨厌浪费时间，不参加无意义的饭局
- 容忍度最低，对上菜慢最不能忍受

### 绿色性格

- 磨磨唧唧，反应慢
- 重视人际关系，聆听他人
- 可能成为最后一个吃完的人

## 总结

通过观察办公桌和饭桌上的表现，可以快速判断同事的性格色彩。这有助于职场中更好地与人相处，提高工作效率。
