# 重开计划（大橙2.0）2499：8、升纬展示面打造方式

## 概述

在本节课中，我们将学习如何通过朋友圈和照片展示自己，以提升个人魅力和吸引力。

## 如何展示自己

### 展示原因

通过朋友圈和照片展示自己，可以让不经常见到你的女生或网友了解你的生活、爱好和价值观，从而更容易产生好感。

### 展示方向

1. **生活日常**：打球、卡丁车、做饭、朋友聚会、听音乐等。
2. **工作事业**：加班、出差、开会、学习、看书、下午茶、客户拜访、公司年会等。
3. **兴趣爱好**：养猫、养狗、唱歌、打球、卡丁车、画画、滑雪、旅游、练字等。

### 展示技巧

1. **反差**：展现与平时形象不同的爱好或特点，例如文静的老师喜欢唱歌、说唱、街舞等。
2. **立体**：通过照片展示多个方面的自己，例如工作努力、爱好广泛、有爱心等。

## 负分展示面

### 负分展示面类型

1. **文章转发**：频繁转发文章，尤其是工作相关的文章。
2. **鬼混照片**：经常去KTV、蹦迪等场合的照片。
3. **负能量动态**：发布大量负能量动态，例如抱怨、痛苦等。
4. **纯文字**：发布大量纯文字动态，例如“今天怎么样”、“今天干了什么”等。
5. **重复照片**：重复发布同一张照片。
6. **随手一拍**：发布未经修图的照片或原图。

## 展示面拍摄

### 取景

1. **发现生活中的美**：学会发现生活中的美好瞬间，并记录下来。
2. **背景选择**：选择有标志或特色的背景，避免单调的背景。

### 构图

1. **九宫格构图**：利用手机拍照软件的九宫格功能，将人物放置在九宫格的交叉点上。
2. **人物与景色的关系**：根据个人特点，调整人物与景色的比例，突出优势，遮挡劣势。

### 拍照软件

1. **无他相机**：一款带有美颜效果的拍照软件，可调整美颜参数。
2. **醒图**：一款抖音开发的修图软件，功能全面，操作简单。

## 总结

本节课介绍了如何通过朋友圈和照片展示自己，以及如何避免负分展示面。通过学习本节课的内容，你可以提升个人魅力和吸引力，更容易获得他人的好感。
