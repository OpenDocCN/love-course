# 18、坏小k女人一生必学的一堂课：05.框架思维 （2） 养成框架思维体系

## 概述
在本节课中，我们将学习如何养成框架思维体系，这是一种高情商的表现，有助于我们在生活中做出更明智的决策。

## 思维框架
### 什么是框架思维？
框架思维是指通过设定一个特定的框架来影响他人的思考方式和决策过程。

### 框架思维效应
以下是几个关于框架思维效应的例子：

1. **赌注问题**：
   - 问题一：有10%的概率赢得95块钱，90%的概率损失5块钱。
   - 问题二：有10%的概率赢得100块钱，90%的概率什么也得不到。
   - 尽管这两个问题的本质相同，但由于框架不同，人们可能会做出不同的选择。

2. **电影邀请**：
   - 当你问某人“我们什么时候去看电影？”时，你实际上是在假设对方愿意和你一起去看电影。

3. **面馆点餐**：
   - 当你问某人“你今天要点一个鸡蛋还是两个鸡蛋？”时，你实际上是在假设对方会点鸡蛋。

### 如何运用框架思维？
以下是几个运用框架思维的例子：

1. **恋爱关系**：
   - 当男人问“你敢不敢看我的手机？”或“你愿不愿意给我花钱？”时，这些问题都与框架思维有关。

2. **框架思维博弈**：
   - 当男人说“我不喜欢发展太快”或“你还不了解我”时，这些问题实际上是在设定框架。

3. **打破框架思维**：
   - 当男人说“我不喜欢娇生惯养的女生”时，你可以通过回答“我会做蛋炒饭”来进入他的框架。
