# 古法易筋经六式教程 🧘‍♂️ 第43课：彭祖强肾功

在本节课中，我们将学习古法易筋经的六式练习。易筋经是古人通过呼吸导引，来撑拔身体筋骨的方法。这套练习旨在使关节打开，身体柔软，从而达到“筋长一寸，寿延十年”的养生效果。

## 概述

易筋经通过特定的动作和呼吸配合，拉伸身体的筋络。中医的“筋”概念广泛，包含肌肉、肌腱、筋膜、血管和神经。练习易筋经可以改变筋的状态，进而影响骨、脉、髓、血、气和发，长期坚持能明显改善身体健康。

## 第一节：盘龙探海 🌊

这个动作主要拉伸膀胱经，以及手臂的手三阳经和手三阴经。

以下是动作步骤：

1.  双脚分开，与肩同宽，站正。
2.  双臂在体侧向外伸展，想象有人拉着你的手，尽量去撑拔。
3.  吸气时用力撑拔，呼气时放松。每个动作停住三个呼吸。
4.  目视双掌，脚跟抬起，继续吸气撑拔，呼气放松。
5.  俯身，向前缓缓折腰，把气慢慢呼掉，吸气撑拔，呼气放松。
6.  缓缓起身，胸前翻掌，目视前方，脚跟吊起，停住三个呼吸。
7.  再次俯身，向前缓缓折腰，停住三个呼吸后缓缓起身。
8.  最后，双手在胸前翻掌向上，并步收势。

## 第二节：弯弓射雕 🏹

上一节我们拉伸了身体后侧，本节中我们来看看如何拉伸两侧的肝胆经。

以下是动作步骤：

1.  双脚分开，与肩同宽。
2.  双臂在体侧向外伸展，停住深呼吸。
3.  吸气用力撑拔，呼气放松，重复三次。
4.  上身向左侧旋转，双掌缓慢落于左侧腰间。
5.  右脚向右前方跨出一大步，成右弓步，同时翻掌向右前方推出，双臂伸直，目视双掌，停住三个呼吸。
6.  缓缓起身，扣脚。
7.  上身向右侧旋转，重复上述动作，推出左掌。
8.  左右各重复一遍后，收脚并步。

## 第三节：太虚发音 ☁️

拉伸完躯干后，本节动作将锻炼我们的平衡能力和下肢力量，同时撑拔上身。

以下是动作步骤：

1.  双脚分开，与肩同宽。
2.  双臂在体侧向外伸展，撑拔三次。
3.  目视双掌，脚跟抬起，膝盖微屈，臀部下坐。
4.  吸气撑拔，呼气放松，重复三次。
5.  五指变虎爪，力开天门，想象把天拉开一道口子，停住三个呼吸。
6.  双手下落并步。
7.  重复上述流程，共做四遍。注意脚跟抬起，膝盖微屈，臀部后坐，敛臀收腹，上身保持直立。

## 第四节：天圆地达 🌍

本节动作主要拉伸身体前侧的任脉和肾经。

以下是动作步骤：

1.  双脚分开，一肩半宽。
2.  双臂在体侧向外、向上、向后伸展。
3.  双手下落，向后反叉，握拳置于腰间，做开胸夹脊的动作。
4.  保持开胸夹脊，向后缓缓折腰，停住三个呼吸。注意幅度慢慢打开，手撑腰时用力顶着，以减少腰部发力。
5.  缓缓起身，双手下落。
6.  重复以上步骤，共做四次。

## 第五节：黔息昆吐（一式） 🌀

前面四式活动了躯干和四肢，本节开始我们将活动颈部和头部。

以下是第一式动作步骤：

1.  双脚分开，与肩同宽，双手自然下垂。
2.  缓缓吸气，头向左后方旋转，吸到中间后，开胸夹脊，头向远端极限拉伸，然后呼气还原。
3.  接着向右后方重复相同动作。
4.  左右交替进行，吸气时吸到自己吸不动为止。

## 第六节：黔息昆吐（二式） 🌀

接下来是颈部的第二式练习，进一步拉伸颈侧筋络。

以下是第二式动作步骤：

1.  双脚分开，与肩同宽，双臂自然下垂。
2.  缓缓吸气，头向左侧旋转，眼睛余光关注尾椎，停住深呼吸。
3.  吸气时用力撑拉颈部，呼气时放松，重复三次。
4.  慢慢回正头部。
5.  向右侧旋转，重复相同的撑拉呼吸过程。
6.  左右各重复两遍。
7.  收功：双手打开吸气，上举撑登天，呼气下落，重复一次后并步。

## 总结

本节课中我们一起学习了古法易筋经的六式练习。从“盘龙探海”拉伸膀胱经，到“弯弓射雕”舒展肝胆经，再到“太虚发音”锻炼平衡，“天圆地达”拉伸任脉，最后通过“黔息昆吐”的两式活动颈项。这套练习的核心在于 **呼吸导引** 与 **动作撑拔** 相结合，遵循“筋柔则骨正，骨正筋柔，气血自流”的原则。坚持练习能使身体变得柔软、协调，起到祛病强身、延年益寿的作用。记住，简单的动作只要勤加练习，就能产生巨大的健康效益。
