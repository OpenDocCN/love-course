# 清风老师合集：14链接贵人，让贵人帮你的实操方法论21期：186.第一课：向上破圈：用破圈思维带你跨越阶层！

## 概述
在本节课中，我们将学习如何通过破圈思维实现阶层跨越，了解如何获得贵人的帮助，以及如何在社会中实现向上社交。

## 课程内容

### 1. 个人经历与启示
**清风老师**从21岁开始白手起家，通过自己的努力获得了千万级体量的成功。他强调，对于普通人来说，没有资源、没有背景，没有任何优势，但可以通过以下方式获得贵人的帮助：

- **寻找资源与人脉**：向上社交，抓住资源和人脉，攀登社会影响力。
- **执行力**：果断行动，抓住机会。
- **初心**：坚持做有价值的事情，帮助他人，为社会提供价值。

### 2. 向上社交的比喻
向上社交可以比喻为抓住一块石头，这块石头是资源和人脉，是社会影响力。同时，还要向下用力，踩的是自己的执行力，或是在创业中支持你的团队。

### 3. 清风老师的创业经历
清风老师在大学期间参加配音大赛，认识了一位创业老师，并抓住了创业机会。他通过路演，获得了投资人的建议，并最终获得了百万级天使投资。

### 4. 破圈的关键点
- **选择相信**：相信世界上有人比我们更优秀，向他们学习。
- **执行力**：果断行动，抓住机会。
- **初心**：坚持做有价值的事情，帮助他人，为社会提供价值。

### 5. 反思与建议
清风老师反思了自己在获得百万级天使投资后，陷入傲慢无知和自满的教训。他提醒创业者，要避免自我封闭，不断学习，实现更大的突破。

## 总结
本节课中，我们学习了如何通过破圈思维实现阶层跨越，了解如何获得贵人的帮助，以及如何在社会中实现向上社交。希望这些知识能够帮助大家实现自己的梦想，改变命运。
