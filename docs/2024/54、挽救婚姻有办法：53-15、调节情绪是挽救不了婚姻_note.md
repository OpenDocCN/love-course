# 54、挽救婚姻有办法：53-15、调节情绪是挽救不了婚姻 📚

## 概述
在本节课中，我们将探讨如何正确处理婚外情带来的情绪问题，以及如何通过合理表达情绪来挽救婚姻。

## 核心概念
**婚外情导致情绪失控**：婚外情事件会对夫妻双方造成心理创伤，引发情绪波动。

**情绪表达的重要性**：正确表达情绪是解决问题的关键，而非压抑或忽视。

## 情绪来源
以下是婚外情导致情绪失控的来源：

- **心理创伤**：婚外情事件对夫妻双方的心理造成创伤。
- **应急反应**：情绪波动是身体对创伤事件的自然反应。

## 错误的情绪处理方式
以下是一些错误的情绪处理方式：

- **不理不睬**：压抑情绪，不利于解决问题。
- **超理性**：忽视情绪，可能导致内耗和夫妻关系恶化。
- **一哭二闹三上吊**：过度情绪化，可能加剧夫妻矛盾。

## 正确的情绪表达方式
以下是正确的情绪表达方式：

1. **表达感受**：将内心的感受告诉对方，如痛苦、愤怒等。
2. **避免攻击**：在表达情绪时，避免指责、抱怨和攻击对方。
3. **合理沟通**：以平和的心态与对方沟通，寻求解决问题的方法。
