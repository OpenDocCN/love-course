# 课程25：李想-性格识人课 🎭
## 017-16 7号快乐型 职场角色

在本节课中，我们将学习如何识别7号快乐型人格，并深入了解他们在职场中的典型表现、优势、劣势以及适合的角色。

7号快乐型人格的核心特征是追求快乐、新鲜感和自由，他们像永远长不大的孩子，充满活力与创意。接下来，我们将从多个维度来认识他们。

### 如何识别7号快乐型？ 🔍

上一节我们概述了7号的基本特征，本节中我们来看看如何通过外在表现识别他们。

**身体语言与表情**
7号快乐型的人通常坐立不安，身体会不断转动，伴有各种小动作，例如翘二郎腿或扭动身体。他们没有“偶像包袱”，姿态随意。他们的表情丰富，常常大笑或不笑，很少微笑，有时会露出不屑的神情。

**语言风格**
他们的语言特点是“语不惊人死不休”，说话一针见血，有时甚至显得刻薄。但他们也能在刻薄与甜言蜜语之间快速切换。

**常用词汇**
以下是他们的一些口头禅：
*   **管他呢**
*   **爽了再说**
*   **用了再说**
*   **吃了再说**
*   **做了再说**

从这些词汇可以想象，他们在消费上倾向于即时满足，难以存钱。

### 7号快乐型的职场表现 💼

了解了他们的外在特征后，我们来看看他们在工作环境中的具体表现。

**工作偏好**
他们特别喜欢多元化、有趣、多变的工作环境，无法忍受单调和重复。在一家公司工作两年后，可能会因为“无聊”而非薪资原因辞职。他们追求新鲜感，因此工作稳定性较差。

**适合的岗位**
基于其性格特点，他们非常适合以下角色：
*   **公关**：享受与人打交道，能用公司的钱“陪客户吃喝玩乐”，并且自己乐在其中。
*   **导游**：符合其喜欢变化、探索和与人交流的特质。
*   **销售（开发新客户/新市场）**：热衷于开拓，但不喜欢维护老客户或旧市场。
*   **创意、综艺、旅游相关行业**：这些领域能满足他们对新奇和快乐的追求。

**领导风格**
作为领导，他们通常是“快乐式领导”。例如，开会可能选择在饭局上进行，三言两语聊完工作后便开始娱乐。其管理公式可以概括为：**工作 = 社交 + 娱乐**。

### 性格的深度剖析：优势与挑战 ⚖️

上一部分我们看到了7号在职场中的角色，现在我们来深入分析其内在的性格优势和面临的挑战。

**核心优势**
他们的优势主要体现在思维和心态上：
*   **多元开放与创新**：思维发散，充满金点子。
*   **聪明且一心多用**：能同时处理多项事务。
*   **积极乐观**：永远保持快乐的状态。
*   **资源整合能力强**：擅长连接不同的人和事。

**主要缺点与挑战**
然而，他们的性格也带来一些明显的缺点：
*   **不专注与虎头蛇尾**：难以深入和坚持，项目容易“烂尾”。
*   **缺乏承诺感**：经常说完就忘，给人不靠谱的印象。
*   **财务大进大出**：赚钱能力强，但消费冲动，难以积蓄。
*   **关系难以稳定**：在婚恋和爱好上容易厌倦，追求新鲜刺激。

**成长方向**
7号需要学习的功课是 **“专注”与“扎根”** 。他们需要练习安住于当下，无论是工作、爱好还是亲密关系，都需要发展深度和稳定性。其成长代码可以表示为：
```python
if 追求新鲜感:
    导致 浮于表面
else:
    练习 专注扎根 # 这是成长的关键
```

### 典型代表与外在风格 👗

最后，我们通过一些典型人物和外在装扮，来更立体地认识7号快乐型。

**外在装扮**
他们的穿衣风格如同“花蝴蝶”，色系丰富，风格无定式，追求创新和突破，有时在常人看来显得“雷人”或奇特。其装扮逻辑是：**造型 ≠ 传统审美，造型 = 新奇刺激**。

**代表人物**
1.  **李敖（作家）**：体现了7号极致的自恋与跳脱思维。他曾说：“我人生最大的烦恼就是没有办法做一个读者读李敖的书。我人生最大的遗憾就是没有办法做一个听众听李敖的演讲。”
2.  **马云（企业家）**：展现了7号强大的资源整合能力与侠客般的快乐文化情怀。阿里巴巴的“快乐文化”和注重奖励、美食的特点，深深烙印了他的性格印记。

### 总结 📝

本节课中，我们一起深入学习了7号快乐型人格。
*   我们学会了通过**好动**的身体语言、**惊人**的言语和**追求即时快乐**的消费观来识别他们。
*   我们了解到他们在职场中适合**公关、开拓类**角色，其领导风格是**快乐式**的，但存在**虎头蛇尾、缺乏承诺**的缺点。
*   我们剖析了其**创新乐观**的优势与**难以扎根**的挑战，成长关键在于练习**专注**。
*   最后，通过李敖和马云等代表，我们看到了7号人格在自恋、创新与文化塑造上的极致表现。

7号快乐型是一群追求自由、创意与多彩人生的人，理解他们有助于更好地与之共事或相处。
