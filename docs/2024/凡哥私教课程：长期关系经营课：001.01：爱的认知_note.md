# 凡哥私教课程：长期关系经营课：001.01：爱的认知

🎼常有人感叹，道理都懂，但亲密关系处理不好。🎼原因可能在于只懂表象，不懂底层逻辑。🎼要真正理解道理，首先需认识到。

🎼无道理能放之四海皆准。🎼需了解道理背后的规律和事物本质。社会学能帮助大家理解这些。布尔迪厄认为，社会学说明群体如何发挥作用，以及如何利用规律控制群体。通过社会学，了解世界或规律如何影响你，从而更好地规划人生，遇见爱情，维护爱情。

🎼情感非本能反应，受社会文化影响。爱情作为重要情感，仅有约100年历史。从54时期至今，爱情经历降维过程。最初，爱情挑战社会隔离，与自由意志相关，具有解放力。如今，已无解放诉求。爱情成为自我独特性和价值的确认。

🎼爱情更接近纯粹关系，但更艰难。需处理自我与他者、感性与理性、激情与长久等冲突关系。

🎼爱情被视为超越理性，其迷人之处在于感性与激情。爱上一个人可能吃醋、不自信、讨厌自己，甚至失控。爱情带来的不确定性，如不确定伴侣是否永远爱你，不确定自己是否值得被爱，不确定两人能否持续走下去等。

🎼过去父母之命媒妁之言的婚姻排斥爱情，成为今日爱情和婚姻关系的障碍。过去社会稳定，追求爱情是获得不确定性和非凡人生途径。然而，不确定性是现代社会重要特征。社会每时每刻都在变化，年轻人身处不确定性强的环境中，尤其是大城市年轻人，工作压力和对未来的担忧，使他们不想挑战不确定性，而是希望在日常生活中抓住确定性的东西，获得安全感。

🎼面对爱情的不确定性，研究发现，年轻人会通过选择单身或在恋爱时用量化指标对抗。选择单身的年轻人强调不婚不育，保平安，享受独处生活。他们通过消费获得自我独特性，解决自我认同问题。一些年轻人通过虚拟恋爱，如追星、QCP、二次元人物恋爱，取代真实恋爱。这样做既能感受到恋爱的快乐，又避免了自我被评价。对于想要进入或身处婚恋关系的人，他们试图通过量化指标增加婚恋的确定性，如交换学历、户籍、财产状况，确保未来婚姻生活质量。一些恋人之间更喜欢用转账等仪式性方式衡量对方的爱。虽然感情不确定，但这些量化指标确定。既然不能保证感情长久，那就先保证当下能确定的东西。

🎼为了避免不确定性，年轻人把爱情变成确定性的东西，这与爱情本质背道而驰。越是这样，越难获得理想中的浪漫爱情。

🎼随着互联网发展，网络充斥着难以辨别的毒鸡汤，如“家不是讲道理的地方”、“爱你的人一定会秒回”。这些鸡汤通常只描述现象，有时适合你，但更多时带偏你。如“家不是讲道理的地方”观念被广泛接受。但这观念有害，因为无理取闹只能消耗爱，不能积攒爱。不断妥协背后是爱的疲惫。如果不能同时增加爱，那么当爱被消耗完时，关系也就结束了。那时，可能连呼吸都是错的。所以，感性的世界需要理性来保驾护航。

🎼“爱你的人一定会秒回”的观念可能导致错失有事业心的人。因为认真工作的人无法秒回，他们可能很专心，包括玩游戏，这意味着他们和你恋爱时也很认真。所以，如果你觉得秒回的人才是真的爱你，以后也不要抱怨对方没有上进心。

🎼类似这样的鸡汤只描述现象，有些鸡汤甚至极端化某些现象，但没有真正把握事情发展的逻辑，也没有全面认识人性。爱情问题本质是认知问题，而非技巧问题。只有认清背后的规律，改变认知，才能打开对爱情的想象，才能经营好爱情。

🎼我们需要明白，爱情在今天这个时代越来越不合时宜，有很多不确定性，很难追求效率，甚至不能用成败论英雄。所以，今天爱情越来越成为一个勇敢者的游戏。但我依然鼓励大家尝试去做这样的勇敢者，因为绝对的安全意味着没有任何挑战和发展。在一个不确定的社会里，如果你要追求确定性，那么只有一条路，就是尽量保持不变，减少和外部的连接。但这样的话，你的世界就会变得很小很单一。在这个充满不确定的时代，爱情的真正目的是让你踏上寻找真正自己的旅程。然后通过碰撞，通过不断的行塑自己，去与他人建立连接，然后打开你的世界，打开你对生活的想象，体验有趣的人生。爱情是某种意义上的奢侈品，这种奢侈品不会区分你的背景、经济实力等。它仅仅重视的是你的勇敢程度，就是你是否愿意把自己打开，去与他人建立连接。

🎼更重要的是，亲密关系对今天的我们而言越来越重要。社会学有一个关于幸福拐点的研究，随着物质水平的提高，物质能带来的幸福感和边际效应逐步减弱。贫穷时，一件大牌衣服、一件暖和的大衣都能让人感到幸福。而富裕时，美食、仙医不会再带来巨大的幸福感。甚至人们开始通过断舍离来寻求内心的平衡。这时，最让人感受到幸福的是人与人之间的亲密关系，而爱情就是最让人感到快乐和幸福的途径之一。所以，爱情虽然很难，但依然有人在追求。爱情虽然麻烦，但依然有人在乐此不疲。爱情虽然不稳定，但依然具有永恒的意义。爱情是需要学习的，而具体学什么，怎么学是随之而来的新困惑。

🎼这次的课程更新，我想要把爱情背后的底层逻辑和大家说明白，也想把具体的方法带给所有人。在后续的课程里，我会带着你用社会学的思维重新解读爱情。从理论到实践，从问题到方法，从认知到技巧，我们来一一破解爱情的迷局。好的爱情不是没有扭活不下去，而是因为爱情我们变得更好。

🎼让我们一起来认识新时代的爱情途景，我们一起来成长为更好的我们。
