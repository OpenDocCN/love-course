# 05.对视无惧【新课首发，针对：002.方法：终结对视恐惧，这个方法就够了_

## 002. 方法篇：终结对视恐惧的唯一方法

在本节课中，我们将学习解决对视恐惧的核心方法。我们将首先剖析关于对视恐惧成因的常见误解，然后揭示其根本原因，并介绍“欲求满足”这一核心解决思路。

上一节我们介绍了课程的整体结构，本节中我们来看看解决对视恐惧的具体方法。

### 关于对视恐惧成因的常见误解

以下是关于对视恐惧产生原因的三种常见但片面的理解，我称之为“三个大坑”。

1.  **身体虚弱**。许多人将包括对视恐惧在内的各种恐惧或强迫思维归因于身体虚弱，例如手淫过度或脾虚肾虚。这种说法有一定道理。
2.  **专注力不足**。注意力不集中可能导致个体更容易关注到不该关注的事物，例如开始怀疑自己的目光是否导致了不良的社交结果，从而引发焦虑和强迫循环。提升专注力是正确的方向。
3.  **精神能量场弱**。这在心理学上指个体表现得自卑、胆小、弱势。这种状态下，个体更容易受到外界负面信息的影响，并可能产生额外的负面思维，进而试图控制自己的行为。这与对视恐惧有关联。

然而，这些原因与对视恐惧之间并无必然联系。身体虚弱、专注力不足或精神能量场弱的人很多，但他们并非都患有对视恐惧。同样，许多精神能量场弱的人也能平稳度过一生，或实现性格逆转。这中间缺失了关键的一环。

### 对视恐惧的根本原因：欲求不满

缺失的关键一环是**欲求不满**。

每次与他人目光交汇时，个体强烈地控制、伪装和压抑自己。最底层的原因是想看但不敢看，因此才压抑和控制自己，最终以失控收场。最根本的原因就是欲求不满。

解决欲求不满的方法非常简单，即**欲求满足**。

佛陀说过，人类所有的痛苦，无论是恐惧、难耐、煎熬还是抑郁，本质上都起源于欲望。如果没有欲望，痛苦也就不复存在。因此，当我们感受到巨大恐惧时，本质就是欲求不满。解决的思路就是满足欲求。

### 为何必须尽早满足欲求

满足欲求必须尽早进行。长期处于压抑和扭曲的状态会导致质变，这就是我们常说的“泛化”。

这与身体类似，小病长期积累会变成大病。长期处于对视恐惧或社交恐惧状态，个体会逐渐失去深层社交的能力，社交圈会越来越小，最终可能感受不到生活的快乐。因为人类大部分的存在感和快乐都来自社交。当个体无法感受到存在感、生存的意义和快乐时，就可能进入抑郁的范畴。

### 如何实现欲求满足

实现欲求满足的方法很简单。

个体之前的状态是“想看但不敢看”，处于压抑状态。现在转变为“直接看”，释放所有狰狞、扭曲的欲望。当欲望被满足后，个体甚至会变得“懒得看”或“不想看”。

最初的眼神闪躲和抗拒，本质上还是因为想看。普通人对此的理解是“大大方方看”，他们认为这很简单。但他们不知道，对于有对视恐惧或其他视线恐惧问题的人来说，“大大方方看”已经不是一个能轻松介入的行为了。在这之前有一个最大的障碍：**恐惧感**。

因此，下一节课我们将探讨如何通过具体方法消除这种恐惧感。届时，个体就能轻松地、大大方方地看，从而满足欲求。
