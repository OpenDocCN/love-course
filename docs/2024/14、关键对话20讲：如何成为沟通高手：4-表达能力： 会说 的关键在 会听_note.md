# 14、关键对话20讲：如何成为沟通高手：4-表达能力：会说 的关键在 会听

🎼你好，欢迎来到关键对话，如何成为沟通高手的课堂。🎼这是我们课程的第四讲，也是六大能力模块的第一讲。🎼首先，让我们回顾一下六大能力。

- 表达能力
- 思维能力
- 对人的认知能力
- 策略运用能力
- 交换和创造价值的能力
- 外交能力

🎼今天，我们将集中讨论表达能力。🎼表达能力的核心是听的能力和思考的能力，说和谐建立在听和思考的基础上。一味强调说和谐，容易本末倒置。

🎼在现实生活中，有些对话友好顺利，有些则对抗性强。如何进行对抗性对话，甚至将其转化为友好对话，需要过硬的表达能力。

🎼例如，你向客户陈述完一个工作方案，客户不以为然，甚至指责你的方案。这时，批评或妥协都不是最佳选择，因为它们都关闭了对话的大门。

🎼我们应该从利益和需求的发现上重新梳理倾听和表达的能力，让关键对话实现我们的目标。

🎼“会说的不如会听”是真理。正确理解别人的意思，听出需求是表达的前提。

🎼明确对话双方的关系是理解对方意图的关键。不同关系下，简单的一句话可能有不同的含义。

🎼例如，“你看起来有点累”这句话，你妈、领导、女朋友说的含义可能完全不同。

🎼“在关系里听需求”是理解对方意图的口诀。不同关系有不同的需求，如亲子关系、恋爱关系、上下级关系、客户关系等。

🎼例如，客户不认可你的方案，可能是因为个人关系或公司关系出现问题，而不仅仅是业务内容本身。

🎼关系和需求是倾听别人说话的解码器。没有关系意识，没有需求意识，我们听到的就都是乱码。

🎼关键对话的技能可以实时锻炼，即使不参与对话，也能从中获得收获。

🎼接下来，我们谈谈意思表达的各种类型及其恰当用法。

🎼表达可以分为三对概念：

1. 观察和事实
2. 感受和判断
3. 请求和要求

🎼观察不是事实，而是观察者认为的事实。不同的人看到不同的事实。

🎼感受是主观的，判断也是主观的。要区分感受和判断，避免引起争论。

🎼判断有三种主要形式：评价、比较和规则。尽量少用判断，尤其是评价和比较。

🎼请求和要求是表达中最重要的部分。请求要具体，不能使用惩罚。

🎼良好的对话表达公式是：观察 + 感受 + 请求。

🎼在表达时，应紧扣观察、感受、请求这三个词，对应需求，满足需求。

🎼不好的表达方式是谈论立场，违反了谈利益的原则。

🎼在倾听时，可以用这种方式判断对方说的是观察、感受、评价还是请求。

🎼当对方没有使用正确的表达方式时，可以引导对方走上正确的表达方式，发现对方的实际需求。

🎼这节课我们讲了关键对话能力中的表达能力。🎼首先明确了听重于说，听的核心是听需求，听的关键是明白对话者之间的关系。

🎼然后讲了表达能力中的三对概念：观察和事实、感受和判断、请求和要求。

🎼最后讲了如何将关系中的倾听和观察、感受、请求的表达法结合起来应用。

🎼今天的课后练习是关于邻居装修噪音的场景。🎼请用观察、感受、请求表达你的需求，并假设邻居用不恰当的表达方式回应，然后设计问题，发现邻居的需求。

🎼多练习恰当的表达和有效的倾听，就会在你身上生根发芽。
