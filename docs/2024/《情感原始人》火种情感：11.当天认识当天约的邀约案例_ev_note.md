# 《情感原始人》火种情感：11.当天认识当天约的邀约案例_ev

## 概述

在本节课中，我们将学习如何通过聊天和情绪感受，快速达成邀约。通过分析一个实际案例，我们将了解如何识别女生的窗口，以及如何通过聊天技巧和情绪刺激，引导女生接受邀约。

## 背景介绍

学员在丽江旅行时，通过搭讪认识了一位女生。两人相识仅一分钟，但情绪愉快。本期重点将强调，真正的聊天应该是灵活多变，结合现场情况和情绪，进行大量的推拉和情绪刺激，而不是生搬硬套所谓的聊天公式。

## 案例分析

### 自我介绍

学员和女生互相介绍了名字，女生没有继续说话。

**技巧**：不要期望女生会一直说话，大部分中国女孩都会保持一定的矜持。

### 万金油技巧

学员使用火种常用的“万金油”技巧，即随手拍下当前场景并发送给女生，不附加任何文字。

**技巧**：通过这种方式，可以测试出女生是真窗口、假窗口还是红灯。

### 情绪刺激

学员通过聊天，逐步提升情绪，从冷到热，从弱到强。

**技巧**：通过情绪刺激，可以引导女生产生约会的欲望。

### 邀约

学员在聊天过程中，逐渐引导女生接受邀约。

**技巧**：邀约时，不要过于急切，要保持轻松、自然的语气。

### 反推

女生询问学员是否有女朋友，学员反问女生有多少人。

**技巧**：通过反推，可以测试出女生的真实想法。

### 语言抽离

学员通过语言抽离，暗示自己可能会离开。

**技巧**：语言抽离可以增加女生的紧迫感，促使她做出决定。

### 最终邀约

学员邀请女生一起喝酒，女生同意。

**技巧**：在邀约时，要注重语气和语调，保持轻松、自然的氛围。
