# 正向吸引高阶成长课：07_5·照片美学_note

在本节课中，我们将要学习如何通过照片来塑造和提升个人线上形象，掌握照片美学的核心技巧，从而更有效地吸引他人，展现个人魅力。

## 概述：线上形象的重要性

你的头像有多好看，就等于你有多好看。你的朋友圈有多么的丰富，那么你的生活就有多吸引人。所以，**头像 + 朋友圈 + 真实的你 = 别人眼中的你**。

不要认为在现实生活中好看就足够了。在互联网社会，尤其在早期散发魅力的过程中，更多的交流是在线上进行的。你说的每句话都是由你的头像发出来的。对方想你的时候，经常打开的是你的朋友圈，看的是你的照片和视频，甚至分享给朋友、父母的也是你的线上信息碎片。因此，头像和朋友圈的重要程度完全不亚于现实生活中的真实形象。当然，一定不要走极端，我们也不要把自己活成只存在于线上的样子。但是，线上状态的美好呈现，一定是非常加分的。

我的很多学生就通过这种方式，在他们的真实交往过程中受益。这有一个很有趣的心理现象：哪怕你长得不好看，但是你经常去发你的美照，经常去展示你在长相或者身材上的优点，在对方的心里面就会认为你就是这样的。哪怕他在见到真实的你以后，他脑子里浮现出来的画面也是照片中的你。

## 如何通过照片吸引人

那么，如何通过照片去吸引人走进你的内心世界呢？要知道，照片是我们生活的一个个切片，它是信息的单元，是我们表达交流的一个能量的载体，也是我们人生的一个印记。每张照片都是一个单位的能量，它能够给人带来吸引的力量。

照片上面传递的信息远远不止照片本身。无法一览无遗的照片，拥有更强大的耐看性，画外的因素也可以是表达的重点。所以不要认为看到我们照片的人，他们只会看我们照片内所表达的信息，他们也会思考和琢磨我们照片外想要表达的东西。

有个最简单不过的例子：我们想要炫富，想要展现我们的身材，想要展现我们的内在，难道一定要通过赤裸的表达吗？可以在照片中添加相关的元素去表达我的生活习惯方式和我的生活环境。这样的照片更具有吸引力。

所以要知道，如果我们的对手不是一个简单的人，不是一个初级选手，那么我们就需要通过照片去更懂得表达怎么去吸引他，怎么让他在我的照片上停留，去发现那些我所预埋的细节，去找到那些可以沟通和聊天的话题。

## 照片的四大类别

以下是我们在线上社交中经常要用到的四种照片类型。

1.  **头像**：它是个人形象的直接折射。
2.  **朋友圈封面**：它反映你的心情、状态和向往。
3.  **人物照**：包括自拍、他拍、抓拍、摆拍或合影聚会等。
4.  **空景图**：不含人物元素的风景、城市、动物或物品照片。

在发朋友圈的时候，也是在讲故事，它其实跟摄影和拍电影是同一个底层逻辑。有时需要用空景图来转化一下大家的视觉审美疲劳，同时去衬托我们想要表达的事情、故事、环境或心境。好的空景图不仅能够遮挡我们可能原本并不想展示的那些外貌或者身材上的不足，它还可以更加衬托出我们的美好感觉。

## 照片美学的核心概念

了解完照片的种类，接下来我们进入照片美学的主要内容。

### 1. 景别

景别是指画面中人物与环境的比例关系，主要分为：
*   **远景**：人物较小，强调环境和意境。
*   **中景**：拍摄人物膝盖以上，兼顾人物动作和空间关系。
*   **近景**：拍摄人物胸部以上，突出面部表情和情绪。
*   **特写**：聚焦于人物的局部，如面部或身体某个细节。

景别是用来控制观众注意力以及心理距离的工具。景别越大，传递的环境信息越多；景别越小，对人物情绪和细节的刻画越强。

### 2. 构图

构图是安排画面元素的方式。以下是几个关键概念：
*   **前景**：通过遮挡物（如门框、树叶）创造空间感和故事感。
*   **后景/背景**：衬托主体，交代环境。
*   **视觉重心**：画面中最想突出、吸引观众第一视线的元素。
*   **画面留白**：在主体周围留有空白区域，可以突出主体，营造意境。
*   **二次裁剪**：后期通过裁剪来重新构图，强化主题或情绪。

常见的构图方法有：
*   **三分构图法**：将画面横竖分为三等分，将主体放在交点或线条上。
*   **对角线构图**：主体沿画面对角线分布，富有动感和张力。
*   **三角形构图**：画面中的主要元素构成三角形，具有稳定感和平衡感。

### 3. 光圈与畸变

手机镜头会产生光学畸变。**脸越靠近画面中心，畸变越小，越接近真实**；身体和四肢越靠近画面边缘，会被拉长。

因此，最接近真实长相的是镜子里的自己，其次是相机，最不像的是手机前置摄像头。我们可以利用这种畸变让人物拍得更好看、身材更修长。但有两种人需注意：
*   **五官离散型**：在手机镜头中五官会显得更分散。
*   **面部过于立体型**：在手机镜头中立体感会被夸张。

### 4. 光影

光影是照片的灵魂。
*   **光源**：优先使用自然光。硬光（如正午阳光）会产生强烈明暗对比；柔光（如阴天）则更柔和。
*   **光位**：
    *   **顺光**：面部受光均匀，但较平。
    *   **侧光**：增强面部立体感。
    *   **逆光**：勾勒人物轮廓，但面部较暗。
    *   **顶光**：容易在眼窝、鼻下产生浓重阴影（“死亡顶光”）。
*   **影调**：好的照片需要有丰富的明暗层次（全影调）。避免画面整体过亮（过曝）或过暗（欠曝）。

### 5. 色温与色调

*   **暖色调**（如橙、黄）：给人温暖、亲近、热烈的感觉。
*   **冷色调**（如蓝、青）：给人冷静、疏远、高级的感觉。
*   **冷暖对比**：在画面中同时运用冷暖色调，可以增加视觉冲击力和空间感。

### 6. 调色与滤镜

*   **调色**：通常调整**饱和度**（色彩鲜艳度）和**鲜明度**（对比度与清晰度）。降低饱和度可得到黑白或高级灰效果。
*   **滤镜**：谨慎使用。流行的“冷白皮”效果可通过**降低色温、提高色调**实现。选择与照片风格匹配的滤镜，如复古、港风等。

### 7. 美颜与修图原则

追求自然真实的高级感。
*   尽量使用**原相机**拍摄，保证清晰度。
*   **慎用磨皮**：保留皮肤质感，可局部处理痘痘、斑点。
*   **适度调整**：眼睛、脸型、身材的调整要符合美学比例，避免失真。
*   **别忘了背景**：修图时注意背景线条是否歪斜，保持协调。
*   **拼图技巧**：朋友圈发图数量建议为1、2、3、6、9张。组合时，避免相同景别相邻，可插入空景图增加故事感。

## 实用拍照技巧

上一节我们介绍了美学概念，本节中我们来看看如何在实际拍摄中运用。

### 场景与角度选择
*   **光线**：寻找光线充足、采光好的地方。
*   **背景**：选择有层次（明暗、冷暖）、与个人风格匹配的背景。户外可找景深，室内可利用白墙留白。
*   **角度**：
    *   **平拍**（镜头与眼睛同高）：客观冷静。
    *   **仰拍**（从下往上）：使人显得高大，有气势。
    *   **俯拍**（从上往下）：使人显得小巧，带点“上帝视角”。

### 姿势与情绪
*   **姿势**：身体尽量舒展，利用手部修饰脸型或构造线条（如三角形构图）。借助物品（沙发、门框）增加画面丰富性。
*   **情绪**：这是照片吸引力的核心。避免面部僵硬，可以微笑，或通过眼神表达情绪。自然流露的情绪最有魅力。

## 进阶美学应用

### 景别心理学
不同的景别营造不同的心理距离：
*   **特写**：拉近心理距离，强调情绪与亲密感。
*   **中景**：呈现自然的社会交流距离，展示人物关系与动作。
*   **远景**：拉开心理距离，营造疏离感或强调环境氛围。

### 用光提升颜值
*   **约会或拍照时，主动寻找有利光源**，避免背光和死亡顶光。
*   确保手机镜头干净，设置高清参数（如4K）。
*   室内光线不足是拍照模糊的主因，可尝试靠近窗户或使用补光。

### 构图讲故事
*   **端平手机**是基础。
*   **画面简洁**：清除无意义的元素。
*   **善用引导线**（如道路、栏杆）将视线引向主体。
*   **画外之音**：不必拍全，适当的裁剪（如从大腿处裁切）给人更多想象空间。

### 遮挡、留白与对比
*   **前景遮挡**：增加空间纵深感与故事感。
*   **留白**：聚焦视线，烘托情绪。“白”可以是任何纯色区域。
*   **明暗/冷暖对比**：是提升照片高级感的有效手段。**控制曝光**，让该暗的地方暗下去，才能突出光亮部分。

### 美食拍摄小贴士
美食是分享生活的重要部分。
*   保持桌面干净，可用桌布增加氛围。
*   多采用**俯拍**角度。
*   构图要有主次和层次。
*   调色可适当增加**鲜明度**和**饱和度**，让食物看起来更诱人。

## 如何提高审美：案例学习

最后，我们通过看图来学习如何提高审美和鉴赏能力。

在看任何照片时，尝试从摄影师的角度思考：
1.  模特用了什么姿势和情绪？
2.  光线从哪里来？为什么用这种光？
3.  为什么这样构图？画面中的元素有什么作用？
4.  摄影师想通过这张照片讲述什么故事？

要相信，好的拍摄和后期可以弥补长相上的不足，放大身材和气质上的优点。审美需要在大量鉴赏和实践中逐步提高。
