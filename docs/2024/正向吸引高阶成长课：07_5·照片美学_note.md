# 正向吸引高阶成长课：07_5·照片美学 📸

## 概述

在本节课中，我们将学习如何通过照片美学来提升个人魅力和吸引力。我们将探讨头像、朋友圈封面、人物照和空景图等不同类型的照片，以及如何通过构图、光影、色温等技巧来提升照片的审美价值。

## 头像心理学

**公式**：头像 = 自我意识 + 自信度 + 气质 + 能量

以下是头像类型及其所传递的内心世界：

1. **近景怼脸自拍**：自我迷恋，自信，较少在意他人目光。
2. **包含背景的他拍照片**：含蓄温和，客观认知，善于扬长避短。
3. **背影或遮挡面部照片**：内心矛盾复杂，渴望认可，不够自信。
4. **肢体语言或物品头像**：独特个性，热情自由，追求轻松惬意的生活方式。
5. **宠物或卡通形象**：沉浸于虚拟世界，渴望归属感，缺乏自信。
6. **明星或网红头像**：颜控，浪漫主义者，内心丰富，渴望爱情。

## 照片美学

**公式**：照片 = 景别 + 构图 + 光影 + 色温 + 调色 + 滤镜

以下是照片美学的主要内容：

1. **景别**：远景、中景、近景、特写。
2. **构图**：前景、后景、视觉重心、人物方向、画面留白、二次裁剪。
3. **光影**：光源位置、光圈大小、曝光度、明暗对比。
4. **色温**：暖光、冷光、冷暖对比。
5. **调色**：饱和度、鲜明度。
6. **滤镜**：冷白皮、高级灰、复古质感。

## 拍照技巧

1. **场景选择**：光线充足、背景层次丰富、景深合适。
2. **拍摄角度**：仰角、俯角、水平视线。
3. **拍照姿势**：舒展身体、利用手和腿构图、凸显曲线和线条。
4. **情绪表达**：微笑、眼神、表情。

## 美颜修图和拼图

1. **美颜修图**：保留画质、清晰度，避免过度磨皮、大眼、小脸、瘦脸。
2. **拼图**：单张、两张、三张、六张、九张，避免相邻、添加空景照片。

## 总结

本节课中，我们学习了照片美学的相关知识，包括头像心理学、照片美学、拍照技巧、美颜修图和拼图等。通过学习这些知识，我们可以提升自己的照片审美能力，并通过照片展现自己的魅力和个性。
