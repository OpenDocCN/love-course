# 雨宸19套合集：03.《婚姻心理课：做自己的情感咨询师》：009.引导付出，要内外兼修_note

在本节课中，我们将学习如何通过“内外兼修”的方式来引导伴侣在感情中付出。核心在于提升自身心理地位，并学会以正确的方式表达需求与认可，从而建立健康、平衡的亲密关系。

## 一、提升心理地位 🧠

上一节我们探讨了付出的基础概念，本节中我们来看看如何从心理层面建立平等、舒适的关系。提升心理地位是引导付出的内在基础。

**核心公式：舒适感 = 成就感 + 认同感 - 压力**

以下是具体方法：

1.  **停止过度付出，提高参与度**
    本质是学会隐藏自己的过度需求感，避免因单方面过度付出而导致心理失衡。

2.  **学会以参与代替索取**
    不要让对方在“满足你”和“满足他自己”之间做选择。例如，当伴侣打游戏时，质问“游戏重要还是我重要”会给予对方压力，挤压其舒适感。人本能地会逃离让自己不舒服的关系。

3.  **将“撒娇”转化为给予对方成就感的行动**
    正确的撒娇不是拽着对方陪自己，而是能让对方感受到成就感。具体有两种参与方式：
    *   **实际行动参与**：尝试和对方一起玩游戏，共同体验。
    *   **语言行为参与**：如果你不玩游戏，可以坐在旁边陪伴，端茶送水，并以好奇、崇拜的语气互动（例如：“哇，你好厉害！”）。这能用行动表达认同，满足对方的虚荣心和成就感，让他感觉像英雄。

4.  **保持正常社交圈与自身吸引力**
    最不明智的做法是生活中只剩下对方，失去独立自我。这会导致“浆糊关系”，丧失吸引力。一个人被爱，正是因为他/她是一个独立、有自己生活圈子和空间的个体。因此，务必保持自己的爱好、社交圈和私人空间。

## 二、学会表达认可与赞美 🗣️

在建立了坚实的心理基础后，我们需要学习如何通过外在表达来巩固关系。本节将重点学习如何有效地表达认可与赞美。

**核心思路：认同不是嘴上说说，而是用实际行动参与对方的世界。**

以下是具体方法：

1.  **通过共同参与事件来表达认同**
    例如，当伴侣对美妆、衣服感兴趣时，你可以参与挑选和讨论，并给予真诚的赞美。感情正是在一件件共同参与的事情中培养出习惯、惯性与美好回忆的。这些积极的共同记忆会成为未来化解矛盾的有力证据。

2.  **发生冲突时，通过示弱与撒娇来化解**
    思路是：**满足对方的需要感与价值感，并为自己的行为做合理化解释**。
    *   **错误示范**：（因对方考完试未回电话而生气）“你为什么不给我回电话？”
    *   **正确示范**：（描述事实+表达感受+示弱）“我刚考完试，感觉考砸了，心情特别糟糕，一直在想后续怎么办。没给你打电话是因为我不知道怎么开口，我不想把坏情绪带给你。”
    这样就从宣泄不安全感，转向激发对方的保护欲，从而化解冲突。

3.  **想要对方做什么，自己先去做**
    人很难被改变，但容易被影响。如果你希望对方在生日送你礼物、为你制造浪漫，不要只是要求（这是在改变对方，会引发不满）。正确的方法是：**你先为他/她这样做**。用你的实际行动去影响和带领对方。
    *   **核心原则**：感情遵循平等原则，付出是相互的。摒弃“因为我是女性/男性，对方就应该如何”的单向索求思维，这种不平等的要求终将导致关系崩溃。

---

本节课中我们一起学习了引导伴侣付出的“内外兼修”之法。**内在**，我们通过停止过度付出、学会参与、保持独立来提升心理地位；**外在**，我们通过行动表达认同、用示弱化解冲突、以及以身作则来影响对方。记住，健康感情的基础是平等与相互满足，而非单方面的要求和压力。
