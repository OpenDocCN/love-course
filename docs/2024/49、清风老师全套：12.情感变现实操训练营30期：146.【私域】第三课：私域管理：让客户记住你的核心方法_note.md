# 清风老师全套：情感变现实操训练营30期 - 私域管理：让客户记住你的核心方法 🌟

## 概述
在本节课中，我们将学习如何通过私域管理，利用标签化用户信息，提高客户转化率和满意度。

## 核心概念
**私域管理**：一种用户管理方式，通过收集用户信息，进行精准营销和客户关系维护。

**标签化**：对用户进行分类，以便于后续的精准营销和服务。

## 用户信息收集
以下是用户信息收集的标准模板：

```
您好，很高兴能够添加到我的微信。为了能够更加高效地解决问题，方便后期赠送一些适合的资料福利，请您补充以下信息：

- 姓名
- 所从事的行业
- 个人诉求
```

## 标签化用户信息
以下是用户信息中三个核心要素及其代表的意义：

1. **称呼**：建立熟悉度，方便后续沟通。
2. **行业**：精准推荐资料或信息，提供个性化服务。
3. **个人诉求**：了解用户需求，提供针对性解决方案。

## 精细化标签系统
为了方便群发和精准触达用户，我们需要对客户进行精细化标签管理。

1. **未成交**：分为C1、C2、B1、B2四个档次，分别对应领了资料但未说话、领了资料但意向不强、收入较高且有一定意向、收入较高且意向强烈。
2. **成交**：根据客户成交金额进行分类，如399元、1980元、9800元等，以便于后续的跟单和客户维护。
