# 情感指针：005.第 5 课：通过合盘，找到给你不动产的“送房哥”

## 概述
在本节课中，我们将学习如何通过合盘分析，识别那些愿意在不动产或房产上给予你真金白银的“送房哥”。

## 核心概念
- **合盘分析**：将两个人的星盘合并，分析彼此的关系和互动。
- **第四宫**：代表家、不动产和根基。

## 教程内容

### 1. 记录金星和木星位置
首先，记录下你星盘中的金星和木星的位置。可以使用笔和纸，或手机备忘录。

### 2. 打开合盘功能
打开测色APP，点击主页的“合盘”按钮。

### 3. 选择合盘对象
选择你和你男嘉宾的档案，得到一张心牌。

### 4. 分析合盘
- **金星落在第四宫**：对方可能愿意为你购买房产或支付房租。
- **木星落在第四宫**：对方可能愿意为你提供购买房产的资金或机会。

### 5. 引导对方付出
- 分享你的原生家庭故事，表达对家的渴望。
- 随着感情发展，可以提出希望对方在不动产上的付出。

### 6. 实战案例
依依老师分享了自己的案例，她的男朋友木星落在她的第四宫，最终为她购买了一套价值1000多万的房子。

### 7. 注意事项
- 如果对方经济条件不佳，即使金星或木星落在第四宫，也不一定能够满足你的期望。
- 综合考虑对方的经济条件和能力，再做出判断。

## 总结
本节课中，我们一起学习了如何通过合盘分析，找到愿意在不动产或房产上给予你真金白银的“送房哥”。希望你能运用所学知识，找到属于自己的幸福。
