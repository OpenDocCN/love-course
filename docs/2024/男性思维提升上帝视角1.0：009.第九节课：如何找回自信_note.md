# 男性思维提升上帝视角1.0：009.第九节课：如何找回自信

🎼本节课将探讨如何找回自信，而非如何变得自信。自信是与生俱来的，尤其对于男性而言，自信更是骨子里的特质。然而，由于原生家庭和成长环境的不同，我们可能会在某个时刻失去这份自信。

🎼同样，我们与生俱来的还有自卑。自信和自卑是相伴而生的。要了解自信，首先要了解自卑。每个人内心深处都有或多或少的自卑情节。然而，心理医生在治疗病人时，若只告诉病人不要自卑，要自信，这并不能解决问题。

🎼这种与生俱来的自卑情节可以转化为内在的前进动力。人类社会正是因为这种自卑情节，才会不断进步。自卑感意味着我们意识到自己的不足，从而不断进步，超越自我，成就更好的自己。

🎼以下是一些例子：

- 一个人小时候被伤害，感到自卑，于是努力出人头地，拼命赚钱，甚至不择手段。
- 另一个人受到欺负，感到自卑，于是采取暴力方式解决问题。

🎼这些行为源于无法合理处理自卑感。暴力并不能解决问题，反而会带来更多伤害。

🎼本节课将探讨如何正确处理自卑感，找回自信。

## 正视自卑感

🎼要找回自信，首先要正视内心的自卑感。自卑感从何时开始呢？从我们出生的那一刻开始。人类是出生后无法独立生存的，所有生存需求都寄托在父母身上。

🎼我们的生存本能促使我们迎合父母，听从他们的安排。这导致我们最原始的自主权被剥夺。

🎼随着成长，我们受到父母和外部世界的标准影响，抑制了最本能的选择。

🎼然而，父母的标准并不一定是正确的。我们需要建立自己的价值观。

## 找回本我

🎼要找回自信，首先要找回本我。本我是真实的自我，而小我是为了迎合外部世界标准而抑制真实的自我意识。

🎼以下两个思路可以帮助找回本我：

1. **用心感受**：体验经历，倾听内心真实的声音。
2. **建立自己的价值观**：忽略外部世界的标准，遵从内心的标准。

🎼要突破小我，找回本我，需要发现小我，用知行合一的觉悟打破外部世界的标准。

🎼以下是一些例子：

- 小龙女被侵犯后，杨过被世人的眼光束缚，而当他突破小我，找回本我时，他们成为了神仙眷侣。

🎼本节课中，我们学习了如何找回自信，找回本我。希望对大家有所帮助。
