## 第五节：心态建设——自我预言的实现 🔮

你的期待会塑造你的结果。这在心理学上叫“自我实现的预言”。

**心态影响行为公式：**
```
你认为自己是什么样的人 → 你会做出相应的行动 → 你最终成为那样的人
```
如果你总说“我自卑”、“我做不到”，你就在为自己的不行动找借口，结果必然如此。如果你说“我很自信”，即使现在不是，你也会为了维护这个“人设”而朝这个方向努力。

**具体建议：**
1.  **永远不说负面词语。** 语言塑造思维。
2.  **“骗”自己向好。** 既然没人把你往好了“骗”，那就自己“骗”。相信自己能成功，相信未来会更好。
3.  **梦想是未来的投射。** 人活成自己梦想的样子。拥有希望和梦想至关重要。
