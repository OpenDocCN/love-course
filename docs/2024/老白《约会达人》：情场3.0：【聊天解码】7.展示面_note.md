# 老白《约会达人》：情场3.0：【聊天解码】7.展示面_note

在本节课中，我们将要学习“展示面”的核心概念。展示面是追女生的基础，其本质是通过你的社交媒体（如朋友圈）来“扬长避短”地展示自己，告诉别人你是谁、你有什么、你的生活是什么样的。一个优质的展示面能吸引女生，让她愿意与你聊天和深入了解。

## 展示面的重要性

我们和女生聊天时，女生不回复或态度冷淡，有时并非因为不会聊天，而是对方根本不给机会。客观来讲，打造好展示面，才能让女生愿意跟你聊天，愿意去了解你。

## 展示面的五大组成部分

展示面主要由以下五个部分组成：昵称、头像、背景墙、签名和常规朋友圈。昵称用于初步判断品味，头像展示真实自我，背景墙体现价值观，签名传达三观，常规朋友圈则让人更深入地了解你。

以下是每个部分的具体打造方法：

### 1. 昵称：简单即高级

昵称应避免使用“追风少年”、“哥是个传说”等非主流或暴露年龄的词汇。简单就好，可以是英文，或英文加汉字的组合，例如“Missr白”。无论何种形式，简单就是高级。

### 2. 头像：第一印象的关键

头像至关重要。我们来做一次换位思考：一个优质的头像能让人自动进入“追求状态”；而一个糟糕的头像，即使对方人品好、有才华，也可能让人因“第一眼没有火花”或“感觉不是同一个世界的人”而拒绝接近。

在学习如何拍摄头像前，先了解哪些照片不能用：
*   **风景照**
*   **动物照**
*   **证件照**
*   **卡通照**
*   **网图或画质很差的照片**

正确的头像应该怎么拍呢？核心是营造氛围感和神秘感。
*   **公式**：`优质头像 = 人物 + 有格调的环境/道具 + 意境`
*   例如，人物缩小，视觉中心不在人物本身，而在于人物所处的环境（如网红背景），突出整体文艺感。
*   可以借助宠物、娃娃、手机、围巾、帽子等道具，避免全脸出镜，制造神秘感和好奇心。

### 3. 朋友圈封面：你的黄金广告位

如果说头像是吸引女生点击的“引流器”，那么朋友圈封面就是你的“黄金广告位”。使用不当，等于告诉别人你是一个很low、没有品位的男人。

以下几种类型一定不能用：
*   **网图**
*   **大头自拍**（无论帅否，都显得自恋）
*   **旅游照**（特别是比剪刀手的）
*   **普通的风景照**（除非拍得非常艺术）

那么应该选用什么样的照片呢？封面必须展示你自己的独特价值。
*   **核心**：展示**别人拍不出来**或**大多数人接触不到**的高逼格生活。
*   **例如**：潜水、飞机头等舱、冲浪的照片。
*   如果缺乏这类生活，一张有设计感的纯文字图、在网红地点拍摄的环境照，甚至一张从网上找到的高逼格图片，也是不错的选择。

**重要原则**：拍好的照片一定要修图。坚持“不修图，不发朋友圈”的原则，你的展示面就成功了一半。总之，封面图要告诉看你朋友圈的人：我是一个精致、生活丰富、有高价值和品味的男人。

### 4. 签名：一句话体现品位

签名可以是一句意味深长的话，也可以是一句英文。总之，目的是让女生觉得你是一个有品位的男人。

### 5. 日常朋友圈：你经营的产品

头像是“引流器”，封面是“广告牌”，日常朋友圈就是你为自己经营的“产品”。头像和封面再好，如果日常朋友圈质量跟不上，也是徒劳。

首先，要明白哪些朋友圈不能发：
*   **第一，转发的心灵鸡汤或无病呻吟的文章**。这类内容无人感兴趣，会把你定义为“没有自己生活的屌丝”。
*   **第二，纯文字**。文字提供的信息价值有限，女生无法从中了解你。即使文笔好，女生也不会有兴趣一直看。
*   **第三，随意和朋友玩闹吃宵夜的照片**。“物以类聚，人以群分”，把朋友拍得很屌丝，或在很low的环境聚餐，会拉低你的自身价值。给朋友拍照也要认真，并选择环境不错的地方。
*   **第四，工作广告推销信息**。不是不让做生意，但请善用微信“分组可见”功能，把广告留给客户，把精彩生活留给女生。
*   **第五，太过专业的学术性内容**。如果你是摄影师、音乐人，可以发作品展示；但如果你是程序员或金融从业者，经常发晦涩难懂的专业内容，大多数人看不懂，也不会因此觉得你很厉害。

那么，什么样的朋友圈才是正确的呢？以下是优质朋友圈的几种类型：
*   **第一，美食**。注意是“美食”，不是兰州拉面、沙县小吃或街边烤串。目的是打开聊天窗口，让女生有兴趣问“这是哪里？看起来好好吃”，而不是记录一日三餐。
*   **第二，丰富的娱乐生活**。没有女生不喜欢生活丰富多彩的男生。健身、骑马、旅游、游泳、射箭、桌球、家庭聚会、看画展/摄影展等，都能体现高价值。传达的信息是：“我是一个愿意上进、愿意给自己投资的男人，和那些整天在家打游戏的屌丝不一样。”
*   **第三，宠物或小朋友**。女生对可爱的东西抵抗力很低。展示宠物或婴儿，是测试中女生打开聊天窗口最多的内容之一，同时也能体现你的爱心。
*   **第四，预选展示**。即在朋友圈晒你和**其他女生们**的合照。注意两点：1）不要单独和一个女生合照，容易让人误会是女朋友；如果非要发，配文需说明关系（如“好朋友”）。2）合照女生的颜值不能太低，否则会拉低你的价值。这能引发女生思考：“为什么他身边这么多女人？”答案只有一个：因为你优秀。
*   **第五，工作学习**。例如，配文“大小黄鸭熬夜陪我赶方案”，发布时间在凌晨4点多。简单展示自己是一个爱学习、努力工作、有上进心的男人。
*   **第六，朋友圈配文**。当有一张好照片时，配文原则是“宁愿少，不要多”。如果讲得太清楚，女生就没有好奇心和提问的欲望了。如果不知道怎么写，可以参考相关论坛的配文板块。
*   **第七，音乐分享**。分享你听过的好听的、有感触的歌。歌曲选择很关键，如果欠缺这方面知识，可以下载网易云音乐，参考“每日推荐”。

## 课程总结与行动指南

本节课我们一起学习了展示面的核心概念与打造方法。最后，给大家三个行动总结：

1.  **立即清理**：删掉你朋友圈中所有不合格的内容，并按照课程建议，马上去拍摄一套展示面素材。

3.  **调整心态**：不要把打造展示面当成负担。在这个过程中，你本身也在提升自己，从没有这些生活到开始体验全新生活，这些经历都将成为日后聊天中的宝贵谈资。
