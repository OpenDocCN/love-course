# 老白《约会达人》：情场3.0：【聊天解码】7.展示面

🎼兄弟，你好，我是老白。这节课我们来讲讲展示面的概念，正确的展示面，展示自己属于。🎼追女生最基础的东西，简单来说就是扬长避短的。通过你的朋友圈告诉别人我是谁，我有什么，我的生活是什么样的？

**展示面概念**

展示面是展示自己，扬长避短，通过朋友圈告诉别人你是谁，你有什么，你的生活是什么样的。

**展示面组成部分**

1. 昵称头像背景墙签名
2. 常规朋友圈

**昵称**

昵称是初步判断你的品味，简单明了，可以是英文或英文加汉字的组合。

**头像**

头像告诉别人你是谁，你是真实的人，背景墙告诉别人你的价值观，签名告诉别人你的三观。

**头像拍摄**

头像不要放风景照、动物照、证件照、卡通照、网图或画质很差的照片。

**朋友圈封面**

朋友圈封面是黄金广告位，展示出你自己的价值，例如逼格高的照片、网红地点的照片等。

**朋友圈内容**

1. 不要转发心灵鸡汤或无病呻吟的文章
2. 不要纯文字，提供价值信息
3. 不要随意和朋友玩闹吃宵夜的照片
4. 不要工作广告推销信息
5. 不要太过专业的学术性东西

**正确的朋友圈内容**

1. 美食
2. 丰富的娱乐生活
3. 宠物或小朋友
4. 预选展示
5. 工作学习
6. 配文
7. 音乐分享

**总结**

删掉不合格的朋友圈内容，拍一套展示片，定期更新朋友圈展示面，不要把展示面当成负担，提升自己。
