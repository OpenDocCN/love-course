在本节课中，我们将通过一段街头采访的对话实录，学习如何自然地开启并深入一个关于私人话题（如性生活）的对话。我们将分析对话结构、提问技巧以及如何应对受访者的回避，这些技巧同样适用于日常社交与搭讪。

上一节我们介绍了对话开场，本节中我们来看看如何通过一系列问题逐步深入话题。

以下是采访者提出的核心问题序列：

1.  **确认关系状态**： “你有男朋友没有？”
2.  **询问亲密行为时间**： “你最近一次跟男朋友接吻到什么时候？” “你们多久没接吻了？”
3.  **探讨思念与欲望的时机**： “你觉得你最想他的时候是什么时候，白天还是晚上？” “一个月中，你欲望最强的是什么时候？”
4.  **询问性生活的具体时间**： “你们最近是啪啪啪在什么时候？”
5.  **了解关系中的其他活动**： “你们除了做爱还能做什么？”
6.  **询问性生活起始年龄**： “那你是从几岁开始有性生活？”
7.  **了解恋爱经历数量**： “你交过几个男朋友了？”
8.  **探讨性生活的益处**： “你觉得性生活给你带来的好处是什么？”

---

在对话中，受访者有时会表现出犹豫或回避。以下是采访者应对这些情况的技巧：

*   **面对模糊回答（如“两瓶吧”）**： 采访者没有深究这个可能的口误，而是自然地推进到下一个问题，保持了对话的流畅。
*   **面对直接拒绝（如“这个不能说吧”）**： 采访者接受对方的边界，使用“朋友吧”进行缓和，并换用更宽泛的提问方式（“上周…上上周吧”）来获取近似信息。
*   **面对不愿详述（如“这可以不说了”）**： 采访者尊重对方意愿，果断结束该问题线，并礼貌致谢，完成对话。

---

通过这段对话分析，我们可以总结出深入私人话题的对话模型：

**核心流程 = 开启话题 → 渐进深入 → 尊重边界 → 礼貌结束**

其中，**渐进深入** 是关键，问题设计需像阶梯一样，由浅入深：
```
问题[n] 的私密程度 < 问题[n+1] 的私密程度
```
