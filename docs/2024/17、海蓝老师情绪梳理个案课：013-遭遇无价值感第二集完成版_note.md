# 17、海蓝老师情绪梳理个案课：013-遭遇无价值感第二集完成版_note

在本节课中，我们将跟随一位来访者的叙述，学习如何梳理由童年创伤事件引发的羞耻感与无价值感。我们将看到，一个看似“微小”的事件如何通过错误的解读和内化，深刻影响一个人的自我认知和人际关系。

## 概述：事件的起源与羞耻感的产生

来访者回忆了童年时期（约8-9岁）遭遇邻居男孩不当身体接触的经历。事件本身并未造成严重后果，来访者当时也成功保护了自己，但后续的心理反应却带来了深远影响。

**核心事件**：`一个平常一起玩的、大2岁的邻居男孩，在某次独处时试图拥抱我。我立刻感到不对劲，僵住后坚决地将他赶走。`

## 事件后的心理变化：内化与自我攻击

上一节我们了解了事件的经过，本节中我们来看看事件发生后，来访者的内心世界发生了怎样的变化。

来访者将此事视为“羞耻的秘密”，不敢告诉任何人，包括父母。这种压抑在后来以一个梦的形式爆发。

**关键转折点**：`我梦见与一位有好感的男同学有亲近的画面。醒来后，我感到自己无耻、下流、淫荡、卑微。`

这个梦成为了自我批判的催化剂。来访者开始将外部事件与内在道德品质错误地挂钩，形成了负面的自我认知公式：

**错误认知公式**：`一次意外的接触 + 一个自然的性萌动梦境 = 我是一个无耻下流的人`

以下是这种错误认知导致的行为模式：

*   **社交回避**：感觉无法与班级男生正常接触。
*   **投射性猜想**：认为别人看自己的眼神都充满厌恶，能看穿自己的“秘密”。
*   **核心信念**：认为自己不值得被喜欢。

## 回到创伤场景：重新评估与资源发现

当我们被情绪淹没时，往往看不到事件的全貌。现在，让我们引导来访者以成年人的视角，重新审视那个8-9岁的场景。

在引导下，来访者清晰地回顾了当时的情景：她迅速识别危险，态度坚决，行动果断，成功保护了自己，并在母亲回家前妥善处理了局面。

**重新评估后的发现**：`我看到那个8-9岁的小黄金，在那种情况下头脑非常清晰，行动力很快，非常勇敢，她成功地保护了自己。`

以下是来访者在事件中展现出的、曾被忽略的内在品质：

*   **警觉与清晰**：能瞬间察觉不对劲。
*   **果断与坚定**：立即采取行动，坚决要求对方离开。
*   **勇敢与自我保护**：在可能的风险面前，全力以赴保护自己。

## 与内在小孩对话：倾听与道歉

看到了那个勇敢的小黄金后，我们需要倾听她当时未被听见的心声。

那个8-9岁的小女孩想对现在的自己说：

*   她在求救，她感到害怕。
*   她希望不被贴上“无耻下流”的标签。
*   她希望在害怕时能得到支持和安慰，有人告诉她“没事”。
*   她希望被接纳，希望有一个安全的环境。
*   她希望父母能给予保护，让她感到安全。

听到这些，成年的来访者想对内在小孩说：

*   **道歉**：对不起，我不应该那样评判你。
*   **正名**：你不是没有判断力，你当时处理得很好。
*   **肯定现状**：你现在也有伴侣，并向他倾诉过，他并未因此讨厌你。你现在有能力保护自己。

## 未被满足的渴望：对父母的诉求

内在小孩的恐惧很大程度上源于家庭支持系统的缺失。让我们看看她对父母，尤其是母亲，有哪些未曾表达的诉求。

来访者想象对母亲诉说，核心诉求是渴望被看见、被接纳、被保护：

*   **渴望关注**：感觉父母只关注哭闹的妹妹，自己像空气一样被忽略。即使曾躲起来测试父母是否会寻找，也害怕找到后被责骂。
*   **渴望理解**：不敢说出被欺负、在学校吃不饱、穿脏衣服被嘲笑等事，因为预感到父母的回应会是“这有什么”、“别没事找事”。
*   **渴望支持**：希望在自己害怕、需要帮助时，父母能成为后盾，而不是感到“丢人”。特别希望父母不要再提起那家人，因为那让她感到事件被轻视。
*   **渴望无条件的接纳**：希望无论自己出现什么情绪或状态（如焦虑），都不会被评判，而是能被接纳和安慰。

**核心诉求公式**：`被看见 + 被理解 + 被保护 + 无条件的接纳 = 安全感与价值感`

## 总结：从羞耻感到自我和解

本节课中，我们一起学习了如何层层深入地梳理由童年创伤引发的无价值感。

1.  **追溯起源**：我们找到了当前自我否定感的源头——一次童年事件及后续的梦境。
2.  **识别错误认知**：我们看到了来访者如何将事件内化为对自我道德的全面否定。
3.  **重新评估资源**：我们引导来访者回到场景，发现了自己被忽略的勇气、清晰和自我保护能力。
4.  **倾听内在小孩**：我们倾听了内心那个受伤小孩的恐惧与渴望，并给予了道歉和肯定。
5.  **表达未满足的渴望**：我们代她向父母（象征性的）表达了对于关注、理解和无条件接纳的深层需求。

这个过程的核心在于，将“我不好”的**整体自我批判**，转化为对**特定事件**和**特定环境**（如缺乏支持的家庭）的理解，从而拿回自我定义权，看见自身的内在力量与资源。疗愈始于看见真相，并与那个被误解的、勇敢的自己重逢。
