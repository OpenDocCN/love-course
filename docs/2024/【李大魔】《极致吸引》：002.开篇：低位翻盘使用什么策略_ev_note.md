在本节课中，我们将要学习男女关系中的两种核心策略，并重点探讨如何从“供养者”的低位状态中翻盘，掌握“情人策略”以建立更具吸引力的关系。

## 概述：两种择偶策略

在男女关系中，由于基因选择的影响，女性对男性存在两种择偶策略。一种是**供养者策略**，另一种是**情人策略**。请记住，这是策略，与您当前是什么样的人无关。

在这里，请不要简单地用定义去划分某人属于哪种类型。

## 第一节：理解“供养者策略”

上一节我们介绍了两种策略的概念，本节中我们来看看什么是“供养者策略”。

**供养者策略**是指：男性通过物质与资源的付出，以换取女方进入一段情感关系。

例如，在一段恋爱中，如果总是男生花钱，女生一分钱不花，那么男生就是物质上的供养者。这与花费金额大小无关，核心在于是否形成了一种**交易关系**。

接下来，我们谈谈**情绪供养者**。

情绪供养者是指：您持续为对方提供情绪价值，包括哄对方、照顾对方、给予体贴关心、包容任性、并承担对方所有的负面情绪。

绝大多数男性既是物质供养者，也是情绪供养者。大部分男性在感情中处于低位，正是因为其供养者属性。并非说供养者不好，相反，供养者为社会稳定做出了很大贡献。

但如果您找到并打开了这门课程，说明依靠供养者方式，您无法在当下的感情中获得幸福与真正想要的东西。因此，您需要一种全新的策略来改变关系现状。您接下来所学的所有知识，都将帮助您实现低位翻盘，逆转关系格局。

## 第二节：为何需要学习“情人策略”

我们为什么要学习这门课程？因为世间的爱并非无条件。您想得到什么，就需要相应地付出什么。您所接受的馈赠，暗中早已标好了筹码。

男性如果陷入供养者的深渊，就可能遭遇诸如被背叛或被索取高价彩礼等悲惨下场。改变这种关系的唯一机会，就是觉醒自己的**情人策略**。

从这节课开始，我将教您掌握情人策略，真正具备极致吸引力。

首先，我们要明白情人策略的核心是什么？

**核心是：打开您的基因选择开关，让对方本能地对您产生原始欲望、冲动、情绪波动和占有欲。**

当您掌握了情人策略，您会发现关系发展会非常迅速，尤其是亲密关系的门槛会降低。对方不会给您制定所谓标准，因为标准是由掌握情人策略、占据感情高位的人来制定的。

## 第三节：掌握情人策略的必备能力

无论是供养者还是索取者，在感情中都会非常累，因为您一直在单向提供价值。但如果您成为掌握情人策略的人，对方与您聊天会非常积极，无需您费尽心思找话题，对方会主动体贴和迎合您。

以下是掌握情人策略需要具备的几项核心能力。

### 1. 情绪体验

请注意，这里并非让您去提供所谓的“情绪价值”。您首先要明白，让一个人对您“上头”，靠的不是理性判断，而是**情绪波动**。

因此，您绝对无法通过讲道理或表白等方式说服一个人爱上您，那只是强行推销。**情绪体验**的核心在于制造情绪波动，让对方开心、难过、失落、嫉妒、感激。通过不同的情绪刺激，让对方感觉与您在一起像坐过山车，从而为您着迷，觉得与您在一起是一种荣幸和奖励。

这里必须谈一个误区：所谓的“提供情绪价值”。举个例子，当对方生气时，您是否想着去哄？当对方冷淡时，您是否想方设法找话题？这些行为，包括挽回时主动联系对方，都是误区，源于您本能地想提供过多情绪价值，最终只会让您成为情绪供养者。

真正掌握情人策略的人，只需做自己。对方会自行调节情绪，他们爱的是您这个人本身，而不是因为您对他们有多好。

### 2. 内核与核心自信

这个点非常重要。绝大多数人在长期关系中，都输在没有稳固的内核上。具体操作会在后续“觉醒的力量”中更新，这里先分享两个培养自信、完善内核的有效方法，这也是我从自卑走向自信的核心理论。

**第一点：自发自信**
无需与他人对比或参考，您自己就可以重新找回自信。具体有两个步骤：

*   **优点总结**：拿出一张白纸，罗列出您身上所有的优点。书写的过程就是不断强化自我潜意识。保存好这张纸，每当内耗或不自信时，拿出来看看，您的自信便有所依仗。这叫做“拐杖自信”。
*   **成就真理**：将您过往做成的事一一列出，强化潜意识。例如：高考前只能考200分，突击一个月增分140+；从前无人喜欢，现在被很多女性倒追；刚毕业月入2000，通过技能提升后月入60万。请从事业、荣耀、生活、爱好等各方面总结您的成就。

**第二点：通过正反馈获得自信**
这个关键点在于：**一定要建立异性的社交圈，多和其他异性接触。**

如果您与异性接触只是为了“搞定”对方，您的社交圈会变得非常窄。当您很少与女性正常交流时，您的社交与自信需求无法得到满足，因为您的互动带有强烈的功利性和目的性。

但如果您不抱有此目的，例如认识一位优秀女性，只是单纯将其视为朋友，您会发现她能为您带来更优质的社交资源。当然，前提是您自身的价值要达标。

举个例子，我在与女性相处中，常被夸赞帅气、有才华、有气场等。要知道，男性的魅力与自信，往往建立在女性给予的正向反馈之上。如果您之前很少获得这种夸赞，不要着急，学完课程，您获得的正反馈会越来越多，您的内核也会越来越稳定和强大。

此外，您的**行为与肢体语言**也至关重要。您内心所有的状态都会投射到外在行为上。如果您的行为表现出大方、自然、放松，那么您的内核也会显得很高，您会变得非常有吸引力。

### 3. 框架博弈能力

在长期关系中，框架博弈能力决定了您的话语权以及在对方心中的心理位置。

对方对您冷淡，就是一种博弈。如果您此时过分主动，您在心理位置上就会变成低位。女生对您说“你爱我就要为我付出/花钱/对我好”，这也是一种框架博弈。如果您进入并默认了她的框架，您就陷入了被动。

后续课程我会教您如何提升框架能力，这点必须好好学习。否则，在长期关系中被冷淡、被甩、被背叛甚至被分手等问题，您一定会遇到。

### 4. 引导投入

一个人能爱您多久，取决于他/她为您投入了多少物质、情绪、时间和精力。

想一想为什么您舍不得一个人？本质就是您为他/她投入了太多，这种投入就是**沉没成本**。

如果您的投入是100，对方投入是1，那么投入比例就是 **`100:1`** 。可想而知，这种状态下肯定是您更舍不得对方，对方离开您的成本太低，而您因投入过高难以抽身，心理也难以平衡。

合理的投入比例是 **`7:3`**、**`6:4`** 或 **`5:5`**。举个例子，您为这个女人花了7000元，她应该为您花3000或4000元，这才是合适的投入比。如果您花了7000元，对方一毛不拔，那您就陷入了不利境地。

请注意，对方的投入不仅指物质，她的时间、精力、情绪投入都应纳入考量。

## 总结

本节课中，我们一起学习了：
1.  男女关系中的两种核心策略：**供养者策略**与**情人策略**。
2.  处于低位往往源于**供养者属性**，需要通过觉醒**情人策略**来翻盘。
3.  情人策略的核心是激发对方本能的**原始欲望与情绪波动**。
4.  掌握情人策略需培养四大能力：**情绪体验**（制造波动，非单向提供价值）、**内核自信**（自发总结与获取正反馈）、**框架博弈**（掌握关系主动权）以及**引导投入**（平衡双方的付出比例）。
