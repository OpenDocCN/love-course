# 课程03：沟通课 丨 成为说话高手 🎤
## 第34节：有趣夸张——大胆的夸张 ✨

在本节课中，我们将要学习如何运用“夸张”这一技巧，让你的表达变得生动有趣、令人印象深刻。夸张并非扭曲事实，而是一种富有想象力的表达方式。

我们经常被告知说话不要太夸张。然而，如果运用得当，夸张能带来特别出色的效果。这正是许多优秀的演讲者、主持人和脱口秀演员经常使用夸张的原因。

那么夸张具体会带来什么效果呢？我们先用三个小例子来对比一下。

假设在一场同事或朋友的聚会上，大家都在介绍自己的喜好。你想说自己喜欢美食，但又希望介绍得特别一点。

以下是两种介绍方式的对比：

*   **方式A**：我这个人特别喜欢美食。
*   **方式B**：我这个人特别喜欢美食，到什么程度呢？就是凌晨两点，如果你给我打电话说在楼下送我200块钱，我是不会行动的。但是如果凌晨两点给我打电话说带我去吃好吃的，不管你在哪儿，两点10分我肯定出现。

和A相比，B的说法显然有趣多了。这个说法乍一听像是在开玩笑，但它正是通过夸张，让别人对你的特点印象非常深刻，比单纯说“我喜欢美食”要有趣得多。

再举一个例子。在一次自我介绍中，你想自嘲一下，说自己脑子反应比较慢。但如果只是平淡地说“我这个人脑子不太好使”，效果并不有趣。

那么尝试夸张一下，可以这样说：“我这个人就是脑子不太好使。我朋友都说吃核桃能补脑，我心里想，如果这样的话，我得吃一卡车核桃才行？” “一卡车核桃”这个夸张的说法，让大家立刻明白了你的意思，并且会觉得你很有想象力、好玩有趣。

第三个例子，假设朋友讲了一个你已经听过的笑话。当着其他人的面，你不好直接说“我早听过了”，但你和这个朋友比较熟，想调节一下气氛。

这个时候可以夸张地说：“这个笑话，恐龙应该也听过。”“恐龙也听过”这个说法非常夸张，因为恐龙几千万年前就灭绝了。要的就是这种效果。这么一说，大家就明白了你的意思，笑一笑之余，还会觉得你很幽默。

通过以上例子，我们可以看到夸张并非黑白颠倒或扭曲事实。它是一种顽皮的表达，把本来要说的东西进行拉伸。尤其是当大家听多了“好”、“大”、“快”、“壮观”这些单调的形容词后，已经对这些词免疫了。夸张则可以激活别人的注意力。

这就好像舞蹈老师为了让学生把动作做到位。如果只是说“把胳膊伸展出去”，力度是不够的。而如果说“把胳膊伸到窗户外面去，体会把胳膊从北京伸到巴黎的感觉”，效果就完全不同。胳膊当然不能伸到巴黎，但这样的夸张描述能让学生有更深刻的体会。

所以，对于夸张，别人明明知道不是真的，但可以理解你说的意思，并且感受更深刻。同时还会觉得你很有想象力、很幽默，这种说法很好玩。因为它脱离了现实，是一种超现实主义的表达，给听众的感觉就像突然从地面飞到了外太空，眼前不再是家长里短，而是银河繁星，因此会觉得有趣。

当然，并非随便夸张就会有趣。夸张得不好，反而会显得笨拙和尴尬。

上一节我们介绍了夸张的效果和原理，本节中我们来看看具体如何操作才能让夸张变得有趣。

首先，说话时要加入夸张，我们需要有意识且很自然地将一个事物本来的程度、范围、尺度或数量等，扩大10倍、100倍甚至1000倍。

以下是几个不同维度的夸张示例：

*   **体积的夸张**：如果说“一个人肚子大”，不如说“肚子可以放下10个西瓜”或“肚子可以装下一头牛”有趣。这里的“10个西瓜”或“一头牛”就是把体积扩大了若干倍。
*   **时间的夸张**：如果说“我想好好赞美你”，这是很普通的说法。但如果说“赞美你需要100年”，就显得浪漫而有趣。这里是把时间放大了若干倍。这个例子其实来自诗人安德鲁·马维尔的夸张手法。
*   **速度的夸张**：形容小李说话慢、爱停顿，可以说：“一般小李说完第一句话之后，如果你上了厕所再回来，他第二句话还没想好说什么。”这是把时间无限拉长。形容小猪说话快，可以说：“他讲话巴拉巴拉特别快，你听他讲话的时候，花钱打一个喷嚏的功夫，可能就直接漏掉一部《荷马史诗》。”这是把时间无限缩短。
*   **重量的夸张**：你骑自行车，朋友坐后面，你觉得他特别重。这时可以开玩笑说：“拉着你的感觉，就好像背后拉着一辆卡车一样。”
*   **外貌的夸张**：形容某人嘴巴大，只说“嘴巴大”没意思。可以这样说：“刘总嘴巴大得，一笑起来，感觉嘴角都能连到耳朵上。”我们古人也是夸张的高手，比如成语“汗牛充栋”，就是形容某人家里书特别多，多到运书的牛累出汗，书堆得顶到房梁，这是多么有趣的夸张。
*   **虚拟状态的夸张**：情绪、气势等虚拟的东西也可以夸张。比如你去见了一个气势威严的大客户王总，回来可以这样描述感受：“我当时去见王总，一进会议室，那气势，整个会议室直接刮起一阵风，所有人都打了个冷战，我当时差点就吓跪。”

举了这么多例子，希望大家能感受到如何从各个角度、对各种程度进行放大。这是夸张的第一个关键点。

那么夸张的第二个关键点，就是要加入一些“无厘头”的元素，这样会更加有趣。

什么意思呢？我们来举个例子。假设一个人这样介绍自己：“我的特点，大家应该也看出来了，就是皮肤黑。这也是为什么我晚上走路的时候经常被车撞，因为我的肤色和夜色会融为一体。”皮肤会黑到那个程度吗？当然不会。真的被车撞过吗？大家也知道不是真的。这就是无厘头。他这么一说，夸张效果就非常好。

再比如，你想形容某家餐厅的火锅特别辣。但如果只说“辣”，别人感觉不到到底有多辣。但如果你说：“辣得这么说吧，你要是吃上一口，你肯定想把舌头给拔下来扔出去。”“把舌头拔下来”怎么可能？这就是无厘头。你这么一说，关于那个火锅辣到了什么程度，感觉就很到位了。

所以，总结夸张的两个关键：
1.  对事物的程度进行**放大、放大、再放大**。
2.  敢于加入一些**无厘头**的元素。

如果我们再深入一点，为什么夸张会带来有趣呢？我认为高级的夸张是一种浪漫主义。“赞美你需要100年”、“打个哈欠会错过一部《荷马史诗》”，这都是脱离现实、极具想象力的描述，多么浪漫。

关于夸张，我们今天就讲到这里。

本节课的练习就是：寻找一个人或者一个事物，然后尝试描述它。
先用普通的方式描述，随后把其中用到的形容词（如大小、快慢、轻重、形状、颜色、状态等）进行夸张处理。如果能再加入一些无厘头元素就更好了。试试看这样的描述效果怎么样。

**本节课中我们一起学习了**：夸张并非简单的夸大其词，而是一种通过有意识地放大事物特征（程度、体积、时间等）并巧妙融入无厘头元素，从而使表达变得生动、有趣、令人印象深刻的沟通技巧。掌握好夸张的度，能让你的语言充满想象力和感染力。
