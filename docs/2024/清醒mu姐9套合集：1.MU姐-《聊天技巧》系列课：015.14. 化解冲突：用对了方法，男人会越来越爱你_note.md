# 清醒mu姐9套合集：1.MU姐-《聊天技巧》系列课：015.14. 化解冲突：用对了方法，男人会越来越爱你

🎼 欢迎来到聊天技巧系列课。大家好，我是木姐。上节课我们讲解了如何与另一半磨合，以及磨合期的沟通技巧，旨在帮助大家减少冲突和矛盾。为什么说是减少而非防止？因为即使是亲密的伴侣，也难免会有各种小摩擦。观察自己的父母就能发现，尽管他们相处多年，依然会因为琐事争吵。这是因为亲密关系是动态发展的，人也在不断变化，因此碰撞在所难免，就像健康的人偶尔也会生病一样。冲突是变化的警示灯，表明其中一方在关系中感到不舒服，需要调整。我们不必担心冲突，它只是亲密关系的一种体现。既然无法避免，我们就正面应对，学会如何处理冲突，并从中获得自我修复的能力。很多人面对冲突时缺乏处理能力，只会用暴力沟通的方式激化矛盾，最终导致关系破裂。因此，今天我将与大家分享解决冲突的正确方法。

**以下是解决冲突的步骤**：

1. **了解冲突形成的原因**：包括责任分配、观点分歧、情绪表达、期望磨合、付出不均衡和第三者介入等。
2. **识别引发冲突升级的沟通方式**：如批评、鄙视、辩护和冷战。
3. **控制双方情绪**：通过内心暗示、离开现场等方法平复情绪。
4. **正确化解冲突**：包括营造安静的环境、表达感受、找出原因、倾听、讲述经过、评估责任、道歉和接受道歉等。
5. **非暴力沟通**：观察事件、表达感受、阐述需求、提出请求。

**以下是具体操作方法**：

1. **约定冲突底线**：明确哪些话不能说、哪些事不能做，如提分手、离婚、拉黑、断联、动手打人、冷战、找异性倾诉等。
2. **处理情绪**：通过内心暗示、离开现场等方法平复情绪。
3. **情绪回应**：理解对方的情绪，并引导对方回到沟通正轨。
4. **解决问题**：认识到双方是一伙的，共同解决问题。
5. **非暴力沟通**：观察事件、表达感受、阐述需求、提出请求。
6. **达成共识**：求同存异，重视共同点。
7. **制定冲突协定**：明确规则、奖惩制度，并共同遵守。

**总结**：

本节课中，我们学习了如何化解冲突，包括了解冲突原因、识别沟通方式、控制情绪、正确化解冲突、非暴力沟通、达成共识和制定冲突协定等。希望这些方法能帮助大家在亲密关系中更好地处理冲突，增进感情。
