# 49、清风老师全套：9.不委屈，做个真正高情商的人：098.第三课：如何通过高情商，做个“不委屈”

在本节课中，我们将要学习高情商如何帮助我们避免生活中常见的“委屈感”。我们将探讨委屈感产生的根源，并重点介绍“自我调节”这一核心能力及其四个关键维度。最后，我们会将理论应用于情感和工作两大具体场景，提供可操作的行动原则与步骤。

---

在上一节课程中，我们介绍了良好情商能带来的益处与能避免的问题。本节中，我们来看看如何具体运用高情商，来消除那种令人不快的“委屈”感受。

为什么委屈的感觉会时常出现？我第一年工作时在一家上市公司。作为一名新人，我工作格外努力。我认为只要我足够努力，领导就会看到我。领导确实对我很认可，但在任命项目负责人时，他选择了另一位同事。他的理由是，这位同事更加努力。当时我感到非常不服。明明这位同事平时只做领导重视的事情。而我无论领导是否重视，都努力完成。为什么我的付出没有得到回报？一气之下，我选择了离职。

很多年以后，我才明白世界上有一个善良的谎言，叫做“付出就应该得到回报”。实际上，如果付出的方式方法不对，就得不到回报。或者有时方式对了，但时机不对，也得不到回报。这需要我们用高情商去找到关键点，在付出与回报之间建立有效的桥梁。

第二个常见的观点是：我对他那么好，付出那么多，他为什么不理解我？我那么爱他，为他做了那么多，他为什么不喜欢我？前段时间，我看到一个小伙子在天台上用榴莲告白。他说：“女神，我爱你，你最爱吃的榴莲，我给你拉了一车来。”女神对他笑笑说：“对不起。”这种误解在男女关系和社会交往中经常存在。有时，信号的传递方式比信号本身更重要。如果没有情商，传递信号的过程极易出错。

那么，高情商的第二个核心要素——**自我调节**——就可以帮助我们避免这种委屈感。它包含以下四个核心维度：

以下是自我调节能力的四个核心维度：

1.  **良好的自制力**
    听到自制力，如果立刻想到要压抑自己、满足别人，那就错了。自制力是能真实判断自己的需求，并通过行动去满足。这样我们就不必一味向他人索取。另一种情况是，我们能对自己的情绪负责，不需要向他人索取情绪上的满足和安慰。这才是真正的自制力，其本质是**自己对自己负责的能力**。

2.  **强大的责任心**
    这是对自己所做的事抱有原则，且不因他人看法而轻易改变。如果总是活在领导或伴侣的看法中，就会经常打破自己的原则，难以建立责任心。责任心之所以重要，是因为它与意义感挂钩。你所做的事有其自身存在的意义。若缺乏这种意义感，工作过程就会变得痛苦，觉得委屈了自己的才华。要避免委屈感，就需要建立良好的责任心。

3.  **优秀的适应力**
    委屈感也有一部分源于对环境变化缺乏适应能力。我的一位女性来访者曾问我：“为什么我的丈夫总是欺骗我？明明答应今天陪我去迪士尼，突然有工作就要去。是不是在他心里我根本不重要？”我能感觉到她的委屈。但这个意外并非她丈夫所愿，他也不得不去处理。这位女性缺乏适应意外发生的能力，因此将情绪迁怒于丈夫，导致双方都感到委屈，争吵一发不可收拾。

4.  **出色的创新能力**
    即能创造性地利用内在和外在资源解决问题。很多时候，我们并非没有资源，而是执迷于用老方法重复错误，从而忽略了已有的资源。我的一位来访者与男友分手，两人已谈婚论嫁。因她一直“作闹”，让对方失望。她问我：“我已经用尽方法哀求，但依旧无法复合。”我帮她梳理了自身优势，让她认识到自己内外的资源。我告诉她：“你们像两个合作企业。你在工作上的能力对他帮助很大，他父母也很欣赏你的专业能力。一味哀求是用你最不擅长的方式索取爱。你平时是个硬气、有原则的人，不必放弃原则。做好自己优秀的样子，对方自然会主动靠近。”她发现真正的优势是工作能力，于是专注工作。后来两人成功领证。

这四种能力构成了高情商的核心能力之一——自我调节能力，能有效帮你避免委屈感。

接下来，我将理论应用于两个核心场景，并提供具体原则与步骤。

### 场景一：在情感中避免委屈

在情感关系中，遵循以下四个原则可以避免委屈感：

以下是情感关系中的四个原则：

*   **对方需要原则**：信号的错误传达，常因对方根本不需要。你给对方一车香蕉，对方却只想吃梨。这时你还问对方感不感动，其实你只感动了自己。
*   **自我认可原则**：你做这件事应是出于自己喜欢。如果为了满足对方而做一件自己根本不认可的事（例如，超出承受能力买一个十万块的包），这会让你很累，让感情变糟，使你时常感到委屈。
*   **允许拒绝原则**：即能够接受对方的拒绝，并不因此认为自己糟糕。要能客观看待：对方拒绝的是这件事，而不是你这个人。
*   **表达需求原则**：在感情中能大胆提出自己的需求，并在合理范围内让对方为你付出。这才能真正成为情感中的高情商者。

### 场景二：在工作中避免委屈

在工作中，遵循以下五个步骤可以帮助你避免成为委屈的人：

以下是工作中的五个步骤：

1.  **任务理解**：要理解到位。领导关注点可能与你不同。一方面，领导掌握更全面的信息，你只是任务中的一环。另一方面，领导负责战略方向，你要做的是交付结果。因此，理解领导对任务的规划至关重要。
2.  **责任界定**：这不是推卸责任，而是要明确什么事该你做，什么事不该做。我的一位学员曾帮同事完成方案（他仅负责最后10%），并签字提交。结果因前90%的内容客户不满意，领导批评他并责令重写。他感到委屈。这就是没有做好责任界定：签字提交，就意味着对全部内容负责。
3.  **带着方案提问**：职场新人常带着问题去问领导，这会让领导心烦。高情商的做法是带着解决方案去提问：“领导，我这样做可不可以？您看有什么可以完善的？”这体现了上进心。
4.  **关注领导的关注点**：在项目进程中，战略会不断调整。关注领导的关注点不是讨好，而是了解事情的大方向、大战略，这是高情商的体现。
5.  **建立互助关系**：同事是需要依赖的合作伙伴，与他们互帮互助非常重要。但请注意，互助必须在第二步“责任界定”清晰的基础上进行。有同事的支持与理解，你就能更好地解决问题。
