# RSD德里克-游戏十诫：3 特维斯解读：游戏十诫3.2 特维斯第二节

## 命令2：你必须回答三个神圣的问题

**核心概念**：通过回答三个神圣的问题，建立清晰的个人愿景，从而在追求女性和生活方式成功时打下坚实的基础。

### 1. 我是什么样的男人？

**公式**：自我认知 = 真实自我 + 核心特质

**步骤**：

- **自我认知**：客观地评估自己的性格、价值观、喜好和厌恶。
- **核心特质**：识别那些无论何时都保持不变的真实特质。
- **行动**：摒弃虚假的自我，拥抱真实的自我。

### 2. 我想过什么样的生活？

**公式**：生活愿景 = 个人目标 + 使命

**步骤**：

- **个人目标**：确定你想要实现的具体目标。
- **使命**：找到驱动你的核心目标或动机。
- **行动**：制定实现这些目标的计划。

### 3. 那种生活的理想伴侣是谁？

**公式**：理想伴侣 = 与你的生活愿景相匹配的人

**步骤**：

- **生活愿景**：回顾你在问题2中确定的愿景。
- **理想伴侣**：确定那些能够与你共同实现这些目标的人。
- **行动**：寻找并吸引那些与你相匹配的人。

## 总结

本节课中，我们学习了如何通过回答三个神圣的问题来建立清晰的个人愿景。通过这个过程，我们可以更好地了解自己，找到适合自己的生活方式，并吸引那些与我们相匹配的人。希望这些知识能够帮助你实现自己的目标。
