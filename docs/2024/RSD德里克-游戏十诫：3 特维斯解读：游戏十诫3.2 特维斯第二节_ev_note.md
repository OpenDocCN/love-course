完美主义的解药是完整性，而不是完美。我们必须理解并拥抱我们天性的所有方面，无论是好是坏，我们所有的好恶、倾向，并接受我们无法控制的事实的真相。

我已经为回答“你从根本上和个体上是谁”奠定了基础，以使你从其他男人中脱颖而出，并轻松地行走于世，了解你真实的个体天性。引用传奇钢琴家塞隆尼斯·蒙克的话：“天才是最像自己的人。”不可能成为别人。唯一的前进道路是变得更像你自己，并在未来拥抱那种个性。

### 问题二：我想过什么样的生活？ 🗺️

通过问题一，你已经设定了你象征性被发到的“牌”的参数，即你实际上是谁的不可控参数。由此，你获得了大多数男性人口永远无法获得的清晰度。通过问题二，你可以利用这些信息开始前进。

现在，宇宙，无论出于何种原因（如果有的话），决定给你你现在居住的这个身体，连同它所有独特的长处、缺陷、怪癖和限制。作为一个成年人，接受这一点。

将你的身体看作你居住的某种东西会很有帮助，就像你是某种飞行器的飞行员。你只是看着它做它该做的事，所有的动作、所有的想法、所有的情绪，一切。你只是在观察。如果你熟悉埃克哈特·托利（如《当下的力量》）的作品，你就会明白这种方法为何有用。世界上有许多这样的身体，但这个是“我的”。

我们在问题一中，本质上是识别了这个“容器”是什么，即你的身体以及它如何运作。现在，你必须问：我们拿这该死的东西做什么？如果它有轮子，那么最好用它来做类似驾驶的事情。如果它有翅膀，那么用它来做类似飞行的事情会对这个容器很有益。巧合的是，这些容器最擅长执行的功能，也是让它运行最高效的功能。

我认为，**生命的意义就是去做你被造就去做的事**。你有手，所以你会抓东西。你有腿，所以你会走路和跑步。你有一个阴茎，天哪，你也要用它。但如果手抓东西，腿走路，你可能想知道你应该做什么。而“你应该做什么”这个问题的答案，正是**你想做什么**。
