# Chris《智选伴侣》：007.识女系统课第4节

## 概述

在本节课中，我们将学习如何根据不同维度将女性分为八种原型，并了解每种原型的特点和在婚恋市场中的表现。

## 一、影响女性择偶倾向的三个维度

1. **经验维度**：从多到少。
2. **身份维度**：从实现个体价值到实现合作或家庭价值。
3. **SMV维度（时间维度）**：从20到25岁为SMV最高黄金阶段，25到30岁以上为低SMV阶段。

4. **2B（社工大龄女青年）**：在男性主导体制中基层工作的女性，如女博士、社工。
5. **3A（传统保守乖乖女）**：经验维度少，身份维度偏向家庭，SMV高，如潮汕、莆田、闽南地区中产家庭女性。

1. **排除左派女性**：左派女性崇尚个体价值，容易将男性视为工具人。
2. **考虑右派女性**：右派女性更受男性主导，可作为婚姻伴侣考虑。
3. **注意右上派女性风险**：右上派女性容易成为捞女。
4. **寻找右下派女性**：右下派女性稀缺，但适合作为终身伴侣。
