# 思阳情感训练营--弱者、强者_高低位逆转课：003.3.高低位逆转-强者思维_note

在本节课中，我们将要学习一个贯穿所有人际关系，尤其是亲密关系的核心思维——**强者思维**。我们将探讨如何通过这种思维模式，在关系中占据主动，获得更健康、更愉悦的情感体验。

## 概述：给予者才是高位者

上一节我们探讨了关系中的基本位势概念，本节中我们来看看强者思维的核心。首先需要纠正一个普遍误解：**给予者才是高位者**。因为强者具备向下兼容的能力，而被兼容的一方，才是更难以离开这段关系的人。

许多人在亲密关系中患得患失，是因为他们的心态停留在“我能获得什么样的体验，全看对方怎么做”的层面。这是一种寄生型关系，将自己的喜怒哀乐完全寄托在对方身上。

而内核稳定、具备强者思维的人，在情感中拥有足够的主场意识。他们处于“对方能获得什么样的体验，由我说了算”的层面。这是一种共生式关系，自己能够产生大部分情感养分。

**公式：情感自给率 > 70%**
这意味着，自身能提供70%-80%的情感需求，他人只需补充30%，便能达到共生共赢的状态。这并非嗷嗷待哺地索取，而是有能力给予，至于给或不给，则视对方表现而定。

## 第一点：情绪价值的给予与掌控

以下是强者思维在提供情绪价值时的具体表现。

强者清楚，情绪价值是成本最低的价值。正因为自身情感充足，给予与否完全取决于自身意愿。在亲密关系中，他们从不吝于哄对方开心，例如说：“你在我心里就是个小男孩，我得疼你哄你。”

然而，这段关系能否维持，完全取决于“我”是否愿意继续哄你。你要我哄你，也得证明你值得被爱。这才是对方真正离不开的表现，使其处于完全的被动状态。一旦“我”离开，突如其来的失去感会让对方难以承受。

因此，具备强心态意味着：哄对方并无不可，因为一切尽在掌握。由此揭示一个真相：**给予的一方，才是真正的高位者**。因为他既能提供价值，也能果断撤回价值。而提供或撤回，取决于另一半的表现。主动权始终在给予者手中。

## 第二点：区分“给予”与“讨好”

理解了给予的价值后，我们需要明确一个关键区别：给予不是讨好。

在亲密关系里，能够给予的人占据高位。但很多人所做的是讨好，而非给予。以下是两者的核心区别：

*   **给予**：你说一些话、做一些事，目的是让对方享受“被喜欢”的感觉。出发点是自己“想给”才给，这是在提供情绪价值，让对方逐渐上瘾。
*   **讨好**：你说一些话、做一些事，目的是让对方记住你的好、你的付出。出发点是为了换取回报才给，这是在索取情绪价值，只会让自己内心拧巴。

例如，伴侣工作压力大时，你发送关心信息：“亲爱的你太辛苦了，我一直都在你身边。”
*   如果你的目的是得到他“你真体贴、好感动”的回应，且得不到就会生气，这便是**讨好式索取**。你的行为本质是取悦对方，并索取情绪反馈。
*   如果你的目的仅仅是表达关心，他回应与否都不影响你的心情，这才是**给予**。

**代码式区分：**
```python
if 行动目的 == “让对方享受” and 心情不受对方反馈影响：
    行为 = 给予
elif 行动目的 == “换取对方回报” and 心情因反馈差而变糟：
    行为 = 讨好
```
因此，给予与讨好的本质区别在于：讨好者总期望得到回馈和爱；而给予者虽然做同样的事，却不会因对方反馈不佳而影响自我情绪。他们做事凭自己心情，有给予的能力，也有随时收回的能力。付出时不强求回报，但能在底线被触及时及时止损。这份清醒，才是强者思维。

## 第三点：爱得起，也放得下

接下来，我们探讨强者思维在付出时的心态：没有牺牲感。

在感情中，请记住：所有你多出来的付出，都是自愿的。当你发现自己对关系的需求远超过对方，你应该做的是降低爱情在生命中的比重，而不是将多出的付出转化为怨念，去索取更多。

一旦产生怨念，就必须停止付出。否则只会越索取，越失望。对伴侣要求过多的恋爱，很难幸福长久。

强者都是“爱得起”的。他们认为：“我爱你，是我自己的事。我付出是因为我乐意，我想给才给，而不是为了让你更爱我。”他们不会因对方反馈不好而影响心情，行动只遵从自己的意愿。

**强者思维公式：我的付出 = 我乐意**
如果有一天对方让你不爽，或你觉得这段感情不值得继续付出，你能立刻撤回所有的好。他们不纠结，始终以自我感受为先去相处：习惯早睡就不会熬夜聊天，吃不了辣就不会勉强吃火锅。他们不会让付出变成“我都为你……了，你怎么还不……”的悲情记账本。

因此，强者的快乐是自由的。与这样的强者恋爱或结婚，伴侣也会感到快乐和自由。

## 第四点：嘴上哄，行动狠

在明确了内在心态后，我们来看外在行为准则：不卑不亢，嘴甜心硬。

聪明的女人在感情中不卑不亢，心中有明确的相处规则。她们 **“说软话，做狠事”** ——嘴甜，但心硬。而有些姑娘则活得又卑又亢，计较拧巴，她们 **“把话说绝，做事却怂”** ——嘴硬，但心软，最终容易受伤。

我们来对比两种模式：

以下是两种典型的行为模式对比：

1.  **嘴甜心硬者**：平时身段软、嘴巴甜，提供极佳的情感体验。但当男人表现不好时，该惩罚绝不手软，该分手绝不拖沓。不圣母心，不抱侥幸，及时止损。嘴可甜，心得狠。
2.  **嘴硬心怂者**：嘴上爱说各种办不到的狠话，逞一时之快。说完又沉不住气，对方稍一冷淡就扑上去道歉。平时在小框架上纠结（如消息秒回、早晚安），但遇到大矛盾（如犯错、劈腿）时却不敢惩罚、不敢离开。

请思考，男人会念念不忘哪一种？又会完全吃定哪一种？

在感情中，我们要抓大方向，别总在小细节上拧巴，消耗对方耐心。越是在小事上纠结消耗，未来就越容易在大事上丧失主动权。

## 第五点：少琢磨男人，多关注自己

最后，也是最重要的一点：强大自身，才是永恒的高位。

比起琢磨男人，多关注自己的感受和标准更重要。具备强者思维的女人，会守住自己的情感主场，减少内心戏，不去反复纠结“他是不是不爱我了”之类的问题。

她们会用考察的心态淡定观察：**这个男人在我这里是否合格？我是继续付出，还是收回我的好？是否要淘汰他？**

如果考察发现对方把你当备胎，或感情不再，那就不要把他当成唯一选项。此时应做的是丰富自己的选择，积极接触新人。

在稳定的关系阶段，强者思维的女人只抓大框架，而非揪小细节。

**大框架定义：这段关系能否成就更好的你？**（包括精神快乐与社会成长）
如果相处开心、能带来资源，那就真诚享受。时刻关注自己在这段关系中是否开心、是否有所得。

这是一个强大的爱情心法：**比起琢磨“如何让他更爱我”，不如思考“他能带给我什么”以及“这段感情如何成就更好的我”。** 不要成为在“少说早安、没秒回”等细节里挣扎的小女人。

比起总琢磨对方想法，永远优先考虑：“我怎么想？我喜不喜欢？我接不接受？我要不要继续？”同时，坚守自己的原则：出轨不原谅、表白要官宣、怠慢要后撤、犯错有代价。

**总结：**
只有持续关注自己、强大自己，这样的女人才是真正的强者，也最具魅力。请记住，你的价值与快乐，根源在于自身。

本节课中，我们一起学习了**强者思维**的五大核心：**1. 给予者才是高位者；2. 区分给予与讨好；3. 爱得起，付出无怨念；4. 嘴甜心硬，行动果断；5. 少琢磨对方，多关注和强大自我。** 掌握这些思维，你将在任何关系中都能保持从容与主动。
