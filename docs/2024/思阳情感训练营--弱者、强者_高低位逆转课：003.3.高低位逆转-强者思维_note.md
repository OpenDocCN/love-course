大家好，我是思阳，欢迎收听我们的高低位逆转课程。这一节课我们要讲解一个重要的思维——强者思维。希望你们在生活中，尤其是在亲密关系中，能够运用这种思维。

## 一、给予者才是高位者

**公式**：给予者 = 高位者

恋爱和婚姻中，给予者才是真正的高位者。强者向下兼容，被兼容的人才是离不开的那一个。

## 二、强者思维在亲密关系中的应用

**1. 给予而非索取**

**公式**：情绪价值 = 成本最低的价值

强者清楚情绪价值是成本最低的价值，她们能够自给自足，给予对方30%，而自己也能获得70%到80%的满足。

**2. 给予不是讨好**

**公式**：给予 = 提供情绪价值；讨好 = 索取情绪价值

给予是为了让对方享受被喜欢的感受，讨好是为了让对方记住你的好。

**3. 爱得起，付出无怨无悔**

**公式**：付出 = 自愿；怨念 = 索取

强者不会因为付出而感到牺牲，她们认为付出是因为自己乐意，而不是为了让对方更爱自己。

## 三、强者思维的表现

**1. 嘴上哄，行动上不能丑**

**公式**：强者 = 不卑不亢；弱者 = 不卑不亢

聪明的女人在感情中不卑不亢，她们清楚自己的相处规则，说软话做狠事儿，嘴甜心硬。

**2. 少琢磨男人，多关注自己**

**公式**：关注自己 = 强者；琢磨男人 = 弱者

强者思维的女人会关注自己的感受和标准，而不是纠结于男人的想法。
