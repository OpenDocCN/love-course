## 概述
在本节课中，我们将学习如何通过改变思维方式和提升展示面来提升情商，从而在情感关系中取得成功。

## 移动互联网思维
**公式**：0到1的思维，即追求卓越，不做第二。

**核心概念**：
- 展示面：展示你的安全性和吸引力。
- 自带窗口：展示面能够激发女生的兴趣，但不会直接导致不可描述的事情发生。

## 展示面的重要性
**核心概念**：
- 展示面是安全性的体现，女生更愿意了解你，因为你没有攻击性。
- 展示面不好，女生会认为你只对她感兴趣，从而产生防备心理。

## 聊天技巧
**核心概念**：
- 聊天是吸引的工具，展示你的人格魅力。
- 聊天要传递能量，无欲无求的状态更容易吸引女生。

## 添加异性资源
**核心概念**：
- 每天添加3到6个女生，扩大异性资源。
- 通过朋友圈互动，筛选出对你有兴趣的女生。

## 长期关系维系
**核心概念**：
- 长期关系需要付出和经营，设定目标，共同为感情努力。
- 付出要有智慧，不要一味索取。

## 答疑环节
**问题1**：公司里长得最漂亮的女生，如何应对？
**回答**：扩大异性资源，关注其他女生，不要过于关注一个女生。

**问题2**：彩礼问题如何应对？
**回答**：夸赞彩礼，表达对女生的尊重，同时表达自己的观点。

**问题3**：情感上驾驭不了女生怎么办？
**回答**：反思自己的言行，改变自己的态度，提升自己的魅力。

**问题4**：农村人是否需要去大城市打工？
**回答**：取决于个人性格和追求，风险与回报并存。

**问题5**：搭讪女生后，女生不回信息怎么办？
**回答**：发送简单问候，进行基本信息交换，邀请女生出来喝杯东西。

**问题6**：女生一碰就拉下脸怎么办？
**回答**：慢慢来，不要急于求成。

**问题7**：约会时想跟女生发生关系，但没成功怎么办？
**回答**：表达自己的立场，维护自己的形象，与女生保持距离，让她有时间消化。
