# 海蓝老师情绪梳理个案课：015-遭遇无价值感第四集完成版

在本节课中，我们将学习如何通过情绪梳理个案课来处理无价值感。

## 情绪梳理个案课概述

情绪梳理个案课是一种通过引导个体深入探索自己的情绪，从而实现情绪释放和自我成长的方法。

## 案例分析

### 梦境与希望

同学做了一个梦，梦见自己看到了一点希望。这表明他渴望被关注和理解。

**公式**： 梦境 = 希望被关注

### 情感表达

同学表示，他希望好好践行，并询问小黄精想对他说什么。小黄精表示，他看到了同学的变化，并鼓励他继续努力。

**代码**： 好好践行 = 继续努力

### 自我认知

同学表示，他不再戴着面具，而是更加真实地面对自己。他希望通过践行和学习来疗愈过去的创伤。

**公式**： 真实面对自己 = 疗愈创伤

### 过去事件

同学回忆起过去发生的事情，认为那不是他的错，而是对方的自我保护能力。他看到了对方的勇气和判断力。

**代码**： 自我保护能力 = 勇气和判断力

### 个性特质

同学认为，他具备善良、乐于助人、为别人着想和清晰判断的能力。这些特质使他能够保证自己的安全。

**公式**： 安全 = 善良 + 乐于助人 + 为别人着想 + 清晰判断

### 未来展望

同学相信，通过不断的努力，他的未来会越来越好。

**公式**： 未来 = 不断努力 + 越来越好

## 情绪梳理

### 卸载负面情绪

同学希望卸载掉无价值感、自卑、羞愧等负面情绪。他想象自己在一个湖边沙滩上，通过风、火、水、空气等力量来卸载这些情绪。

**代码**： 卸载 = 风力 + 火力 + 水力 + 空气力

### 接受温暖

同学在卸载负面情绪后，开始感受到阳光和温暖。这表明他已经进入了一个接纳和放松的状态。

**代码**： 接纳 = 阳光 + 温暖

### 拒绝挑剔

同学意识到自己内心有一个挑剔的部分，并将其归因于自己的期望和父母的挑剔。他尝试将这部分情绪还给自己和父母，最终化解了挑剔的情绪。

**代码**： 拒绝挑剔 = 还给自己 + 还给父母

## 总结

本节课中，我们通过情绪梳理个案课，帮助同学处理了无价值感。他学会了如何卸载负面情绪，接受温暖，并拒绝挑剔。相信在未来的日子里，他能够更加自信地面对生活。
