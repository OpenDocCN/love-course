# 《自然方法》搭讪自然人：2.8 共振的三个重点

🎼共振是搭讪过程中至关重要的一环，它包括三个关键点：舒适感、投资和复格。

### 舒适感

**舒适感**是指与对方产生强烈的共鸣，如同二人合一，同频共振。这种状态表现为：

- **专注享受过程**：在与对方交流时，完全沉浸其中，忘记时间与环境。
- **情绪共鸣**：与对方产生强烈的情绪联系，共同经历喜怒哀乐。

**舒适感的来源**：

- **共同经历**：与对方共同经历各种事件，如爬山、旅行等。
- **环境变化**：带对方体验不同的环境，如星巴克、餐厅等。
- **独处时间**：与对方共度更多时间，增进了解。

**提升舒适感的技术**：

- **共谋理论**：与对方共同策划一件事情。
- **角色扮演**：模拟情景，探讨问题。
- **未来计划**：向对方承诺未来的活动。
- **私密笑话**：分享只有你们才能理解的笑话。

### 投资

**投资**是指让对方在你身上投入更多的时间和情感，使其在你心中更具价值。

**投资的意义**：

- **提高价值**：投资越多，对方在你心中的价值越高。
- **增强服从性**：投资越多，对方越可能服从你的要求。
- **调整要求**：根据对方的投资水平调整你的要求。

**提升投资的技术**：

- **投资窗口**：故意制造机会让对方产生情绪投资。
- **条件型投资**：提出需要服从的条件。
- **投资问题**：故意挑战对方，刺激其情绪。

### 复格

**复格**是指向对方说明你为什么喜欢她，强调其独特性。

**复格的意义**：

- **表达兴趣**：让对方知道你对她感兴趣的原因。
- **平衡价值与资格**：保持价值与资格的平衡。
- **激发好奇心**：让对方对你产生好奇心。

**复格的技术**：

- **情感无法解释法**：用情感无法解释的方式表达喜欢。
- **表露脆弱面**：分享自己的脆弱经历。
- **性格复格**：强调对方的性格特点。

### 总结

共振是搭讪过程中的关键环节，包括舒适感、投资和复格三个要点。通过掌握这些技巧，你将能够更好地与对方建立联系，最终实现成功搭讪。🎉
