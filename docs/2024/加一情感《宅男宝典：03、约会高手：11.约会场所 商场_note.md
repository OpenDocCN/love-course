# 加一情感《宅男宝典》课程03：约会高手 - 第11课：商场约会场所 🛍️

在本节课中，我们将要学习如何利用商场作为约会地点，通过一系列自然的互动和技巧，与约会对象建立联系、制造暧昧氛围，并顺利过渡到后续的约会环节（如看电影）。商场是一个充满机会的动态环境，关键在于如何引导和互动。

## 见面与破冰

上一节我们介绍了商场作为过渡场所的优势，本节中我们来看看如何完成初次见面并打破尴尬。

约会见面地点最好定在商场大门。商场门口人流通常较多，对方很难一眼看到你。这为初次见面创造了一个可以进行小互动的机会。

以下是见面破冰的具体步骤：

1.  双方到达约定地点后，可能彼此没有立刻认出对方。此时，可以先通过微信进行互动。
2.  你可以在微信上询问：“你穿什么衣服？”。如果对方回答，例如“白色大衣加牛仔裤”，你便可以据此寻找她。
3.  当你看到她时，可以先拍一张她的照片。
4.  然后，绕到她身后，轻拍她左边的肩膀，并出现在她的右侧。
5.  当对方转头看向左边时，你可以在右边说：“哎，在这边呢。”
6.  顺势进行调侃，可以说：“你这个人这么笨，会不会走丢啊？要不要牵着你走？”

如果你对约会节奏有较强的把控力，或对自己比较自信，也可以选择更直接的方式。见面后可以直接说：“走吧，这边我带你逛。”然后观察对方的反应。如果对方以“你才笨呢”这类话语回应，说明反馈不错，后续在商场内可以寻找牵手的机会。如果对方反馈一般，则可以先进入商场逛逛，稍后再尝试推进关系。

## 商场互动技巧

成功破冰后，接下来便是在商场内进行互动。逛商场很容易涉及“是否要为对方购物”的问题。

许多男生担心成为“买单侠”，害怕被骗财。不买又担心显得小气，被女生看不起。最好的方法是避免给自己挖坑，即不要主动约女生去可能产生高消费的店铺。正常的女生在第一次见面时，通常不会要求你为她购物。如果一个女生一见面就索要物品，建议你尽早结束约会。

如果你们已经确立了男女朋友关系，为女朋友买东西则是表达爱意的一种方式。但在初次约会阶段，应避免涉及购物。

商场里有许多设施可以用来增进关系。例如，乘坐扶梯时，你可以先上一个台阶，然后转头对女生说：“你上来，我看看你到底有多高。”

以下是针对不同情况的互动话术：

*   **如果女生个子较高**：当她站到同一台阶后，你可以开玩笑说：“你还是下去吧，这样我比较有气势，看起来比较大个。”
*   **如果女生个子较矮**：你可以让她站到上面的台阶，然后说：“这样咱俩的高度就比较搭了。”

这是一个轻微的“小打压”，可能会引起对方情绪上的波动。当她表现出轻微不高兴时，你可以立刻哄她，说：“这样你就跑不出我的视线了，万一走丢了这么可爱的小姑娘，我上哪里去找？”

上了扶梯后，通常有两个方向。你让女生走前面，她可能不知道往哪走。这时，你可以过去牵起她的手说：“别乱跑，我带你走。”对方可能会有轻微反抗，此时你需要表现得果断一些，看着她说：“外面人多。”然后比较霸道地拉着她往一个方向走。

聊天话题可以围绕商场里有哪些好吃的展开，因为“吃”是一个永恒的话题。当然，前提是你对这个商场比较了解。

## 利用娱乐设施升温关系

除了闲逛和聊天，商场内的娱乐设施是制造互动和暧昧情绪的绝佳场所。

你可以带女生去夹娃娃。如果你夹娃娃技术不错，能为她夹到喜欢的娃娃，这次约会就成功了一大半，女生的情绪会变得非常好。因为在我接触过的女生中，很少有人自己能夹起娃娃，她们非常享受娃娃被夹起的那一刻。而且夹娃娃的过程本身就能促进两人互动。

以下是夹娃娃时的互动示例：

1.  如果你没夹起来，可以说：“哎呀，今天我手气不太好，借你点手气用一用。”然后顺势轻轻拿起女生的手“借用”一下手气，再继续尝试。
2.  如果还是没夹起来，可以说：“你的手气也不行嘛，要不你来试试？”
3.  如果女生也没夹起来，可以说：“要不我们两个一起来试试，说不定就能夹起来了。”

即使最终没有夹到娃娃也没关系，一起完成某件事的过程，就足以在你们之间制造出暧昧的情绪。

如果商场里有游戏厅，在时间充裕的情况下，可以一起去玩游戏，例如投篮、开赛车，或者一起点歌唱。只要你们两人不停下来，一直在做一些不同的事情，你们之间的能量就会越来越高，关系也会变得越来越亲密。

大部分人约会失败，是因为只是干坐着，绞尽脑汁地想话题，导致气氛尴尬。商场作为一个过渡场所，不需要在这里进行过多深入的交谈，重点是偶尔制造一些小动作、小温馨和暧昧感。

## 暧昧测试与肢体接触

在商场互动中，可以通过一些轻微的肢体接触进行关系测试，并制造暧昧氛围。

例如，在乘坐直梯时，可以用小手指轻轻勾一下女生的手（而不是完全握住），同时眼睛不要看她。这样可以制造一种若即若离的暧昧气氛。等电梯到达后，可以顺势牵起女生的手去取电影票。

取完票后，通常会上厕所，因为一场电影将近两个小时。这时可以利用一些小道具或动作进行测试。

以下是不同季节的测试方法：

*   **冬天**：如果你戴着围巾，可以取下来围在女生脖子上，然后拉住围巾轻轻将她拉近，看着她的眼睛说：“我去去就回，乖乖在这里等我哦。”
*   **没有围巾时**：可以轻轻拉住女生大衣的衣领。
*   **夏天**：如果穿着单薄，可以像对待小朋友那样轻轻摸摸她的头。

这些动作都可以产生暧昧的感觉，并作为测试，观察对方是否会拉开与你的距离，从而判断她对你的好感程度。
