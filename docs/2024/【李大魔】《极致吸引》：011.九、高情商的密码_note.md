# 【李大魔】《极致吸引》：011.九、高情商的密码_note

在本节课中，我们将要学习如何通过改变沟通模式，将破坏性的“负面三角”关系转化为建设性的“正面三角”关系。核心在于理解并运用“只谈感受，不谈评价”的原则，并建立乐观的解释风格。

## 从负面三角到正面三角

上一节我们介绍了负面三角关系（负面评价 → 负面沟通 → 负面结果）是如何导致关系破裂的。本节中我们来看看如何打破这个循环。

**负面三角的根源在于“评价”**。当我们对一个人或一件事做出负面评价时，就会引发防御和对抗，沟通陷入漩涡，双方互相指责，无法修复关系。

## 核心方法：只谈感受，不谈评价

化解负面三角的关键，是将沟通中的“评价”替换为“感受”。评价是对他人或事物的判断，容易引发对立；而感受是表达自我的情绪和体验，更容易获得理解。

以下是评价与感受的对比示例：

*   **评价示例**：“你这个月才赚3000块，真不如我朋友做抖音轻松赚1万。”（充满比较和价值打压）
*   **感受示例**：“换成我是你的话，做这样一份工作，我会感觉很累的。”（表达自我的感受和共情）

在聊天中，持续的评价会导致话题终结，变成尬聊。例如：
*   **评价式聊天**：“今晚湖边风好舒服。”“是的。”“这氛围让人放松。”“是的。”
*   **感受式聊天**：“今晚湖边风好舒服，让我想起了在大明湖期待遇到夏雨荷的感觉。”（分享个人感受和联想）

感受不仅包括情绪（开心、难过），也包括期望和念头。练习察觉情绪，是为了能在沟通中清晰表达自己。

## 高情商的本质

高情商的本质，就是正确表达真实的感受，并让情绪为目标服务。它需要两项核心技能：

1.  **察觉情绪的能力**：能感知自己和他人的情绪，做到“见人说人话，见鬼说鬼话”。
2.  **表达真实情绪的能力**：你想让对方感受到什么（如真诚、愤怒、边界），就准确表达出相应的情绪。公式为：**目标情绪 → 真实表达 → 对方感知 → 达成目标**。

## 建立乐观的解释风格

在生活中我们不可能完全不评价。评价的背后是“归因机制”和“解释风格”。我们如何归因（把结果归结于什么原因）以及如何解释这个原因，共同形成了我们的评价。

**将负面三角转为正面三角，需要从评价体系下手，建立乐观的心态和解释风格。**

### 1. 调整归因机制
归因是把责任归于外部（外归因）还是自身（内归因）。它是一把双刃剑。
*   **消极外归因**：“都是他的错，我是受害者。”这推卸了所有责任，让人无法进步。
*   **积极调整**：在承认外部因素的同时，也寻找自身可改善的部分（如沟通方式）。对于自卑者，适当理解外部客观原因也有助于建立自信。

人类常会“错误归因”，例如将坐过山车的心跳加速归因于对同伴的好感（吊桥效应）。因此，不要总认为自己认定的原因绝对正确。

### 2. 培养乐观的解释风格
解释风格是我们如何看待一件事。面对半杯水：
*   **悲观解释**：“我只剩下半杯水了。”（关注失去）
*   **乐观解释**：“我还有半杯水呢！”（关注拥有）

以下是悲观与乐观解释风格的核心区别：

*   **关注点不同**：悲观者看**成败**，乐观者看**成长**。悲观者定义“一段失败的感情”，乐观者视其为“一段带来成长的经历”。
*   **心态不同**：悲观往往是**固定心态**，相信宿命；乐观则是**成长心态**，相信事在人为。

改变解释风格，就是学习不轻易用“成功”或“失败”固化经历，而是问自己：“我从中学到了什么？”

---

本节课中我们一起学习了高情商的沟通密码。关键在于**用表达“感受”替代给出“评价”**，从而跳出负面沟通的漩涡。同时，通过建立**乐观的解释风格**，调整我们的归因方式，将焦点从“成败”转向“成长”，就能从根本上将负面三角扭转，构建积极、顺畅的人际关系。记住，**情绪是为你目标服务的工具**，而真实的感受是最高效的沟通语言。
