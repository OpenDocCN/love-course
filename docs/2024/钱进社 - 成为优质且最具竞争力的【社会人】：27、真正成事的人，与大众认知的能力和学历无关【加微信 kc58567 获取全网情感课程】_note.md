# 钱进社课程27：真正成事的人，与大众认知的能力和学历无关 🎯

在本节课中，我们将探讨一个核心观点：在社会上取得真正成就的人，其关键因素往往与大众普遍推崇的高学历或特定能力无关。我们将通过具体例子和逻辑分析，揭示成事背后的底层思维和心态。

## 概述：重新定义“成事”的标准

首先需要明确，这里讨论的“成事”指的是在社会上取得可衡量的结果，例如高收入、行业影响力或实现个人价值。这与大众通常认为的“拥有高学历或漂亮外表就能成功”的观念存在本质区别。

## 核心误区：大众标准与真实成就的偏差

上一节我们概述了课程主题，本节中我们来看看大众认知中常见的几个误区。

许多人将自己的处境归咎于外部条件，但问题往往出在思维层面。以下是几个典型的认知偏差：

1.  **圈层隔离与精神同频**：当一个人开始走向成功时，其思维和行为的转变可能导致他与原有社交圈层格格不入。一个成事者的基础操作是，在精神层面与无法同频的人保持距离，这无关情感亲疏，而是聚焦目标的必然选择。
2.  **“条件优越却结果不佳”的困惑**：部分人困惑于为何自身条件（如学历、外貌）看似优于他人，但社会成就（如收入、婚恋结果）却不如人。这源于对“优质条件”的错误评分。大众圈层中的“略微优秀”并不等于社会层面的核心竞争优势。
3.  **学历与能力的过度自信**：除了顶尖院校（如清华北大），大部分学历在社会竞争中已属普遍。将自信建立在“985/211”学历或“比普通人好看”的外形上，是一种基于自我想象的迷之自信，无法有效变现。

**核心公式**：
```
社会成就 ≠ 大众眼中的优质条件（高学历、好外貌）
社会成就 = 可衡量的结果（高收入、影响力、价值实现）
```

## 成事的两大核心：心态与延展性

理解了误区所在，我们再来剖析成事者真正依赖的两大核心支柱：心态与延展性。这两点听起来抽象，却是区分普通人与成事者的关键。

### 第一核心：扎根的心态

心态是所有行动的起点。许多人追求具体的“答案”，例如哪个行业赚钱、该转行去哪里，但这恰恰是最空的追求。正向的逻辑是向内审视。

以下是心态偏差的具体表现：

*   **逃避现实的“切换赛道”思维**：认为自己在本行业未能成功，换一个行业就能解决所有问题。这忽略了个人做事逻辑和态度具有惯性，换汤不换药。
*   **沉溺于“怀才不遇”的情绪**：对现状不满，依据却是自我想象（如“我学历高就该拿高薪”），而非真实的市场价值与个人付出。这种情绪导致无法在任何领域扎根。
*   **划水式的“完成任务”心态**：将“完成老板布置的任务”等同于努力工作，这是典型的学生思维。在社会竞争中，这远远不够。

真正的成事心态是：**在可以匹配的领域沉下心来，接受起点，拒绝一边工作一边抱怨“不公平”，通过长期深耕积累价值**。正如三只松鼠的创始人，在坚果领域扎根十年，才抓住了电商风口。

### 第二核心：长期的延展性

延展性指的是在选定的行业里，付出极致努力，目标是成为该领域前5%的顶尖者。

以下是实现延展性的关键：

*   **付出与配比**：需要付出90%的努力，去争夺5%的顶尖位置和资源。大部分人不愿或不能付出这样的代价。
*   **时间与积累**：没有行业会轻易赚钱，也没有行业会完全落败。核心在于你是否通过时间和深耕，逐步接近行业的顶部资源。频繁换赛道只能永远在底层争夺残羹冷炙。
*   **“撑下去”的韧性**：以讲师丈夫为例，在一个辛苦、非高薪的行业坚守十年，承担巨大工作强度，最终换来行业名望、头衔和收入的提升。成功往往属于那5%能“撑下来”的人。

**核心逻辑**：
```
普通人的逻辑：寻找不辛苦、高回报的捷径。
成事者的逻辑：在选定的赛道付出90%的努力，用时间和韧性生根，目标是成为前5%，从而获得超额回报。
```

## 工作能力的真实公式

如果你认为能力主要指学习能力，那仍是学生思维。在社会上，真正含金量的工作能力由以下公式决定：

```
真正的工作能力 = 工作经历与经验 + 时间深耕 + 真实积累
```

1.  **工作经历与经验**：包括具体的项目经验、积累的人脉资源、行业敏锐度、专业度及风险预估能力。
2.  **时间深耕**：在一个行业或公司持续投入的时间，是获得信任、接触核心资源的基础。老板不会将核心交给一个频繁跳动的人。
3.  **真实积累**：指全力以赴的耕耘，而非“完成老板任务”式的划水。抱有后者心态，永远只能做普通的、易被替代的职业人。

## 向内耕耘与向外利他

当一个人通过向内耕耘（专注自身成长，咬牙坚持）成就了“1”之后，才会产生向外的良性循环。

*   **向内（成就自己）**：所有成就的起点。感谢自己“撑过来”了，贵人只会帮助这个已经成立的“1”。
*   **向外（利他谦卑）**：当自己成为“1”后，以谦卑姿态对待外界（贵人、家庭支持），懂得利他，整合资源，形成合力。

大部分人的问题在于：**向内是空白的**，所有焦点都在向外索取、比较、抱怨，却不肯先向内付出艰辛的耕耘。

## 总结

本节课我们一起学习了如何重新审视“成事”的逻辑。

1.  **破除误区**：社会成就与大众看重的学历、外貌等条件无必然联系。
2.  **抓住核心**：成事依赖于 **“扎根的心态”** 和 **“长期的延展性”** ，即愿意在一条赛道上付出极致努力，坚持深耕。
3.  **理解能力**：真正的工作能力是 **经历、时间与真实积累** 的总和，而非单纯的学习能力。
4.  **理顺逻辑**：先 **向内耕耘**，成就自己这个“1”；而后才能 **向外利他**，获得良性循环。

成事者的路径并非秘密，但需要对抗人性中追求轻松、不断自我说服的惯性。坚持的理由或许只有一个，但放弃的理由可以有无数个。你的选择，决定了你的终点。
