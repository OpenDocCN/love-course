# 男哥内核3.0初级班：内部课：0.初级班开学典礼

## 概述

在本节课中，我们将学习初级班开学典礼的内容，了解课程结构、学习方式以及初级班和中级班的目标和区别。

## 课程结构

以下是课程结构：

*   **录制课程**：面向大量学员，聚焦聊天、约会和挽回三个方面，包含基础知识和技巧。
*   **群内课程**：面向小班学员，为期一个月，包含基础课程和实战指导，重点讲解病毒聊天和线下活动知识。

## 初级班和中级班

### 初级班

*   **目标**：在一个月内，每天至少获取15个女生资源，为终身学习打下基础。
*   **内容**：个人品牌建设、心态管控、聊天技巧、约会技巧等。
*   **要求**：认真学习基础课程，积极参与实战练习。

### 中级班

*   **目标**：快速邀约、快速聊天，以快速建立关系为目的。
*   **内容**：高级聊天技巧、约会技巧、快速建立关系技巧等。
*   **要求**：具备初级班基础，能够灵活运用所学技巧。

## 群内学习

*   **重点**：积极参与群内讨论，记录问题和答疑，积累经验。
*   **要求**：不要在群内发无关信息，保持学习氛围。

## 课程学习

*   **直播**：可看可不看，主要用于巩固基础知识和学习实战技巧。
*   **课程**：认真学习，做好笔记，为实战打下基础。

## 恋爱三板斧

*   **动力**：源源不断的异情资源，每天至少添加5个女生。
*   **地图**：清晰的步骤和话术库，帮助学员找到适合自己的聊天方式。
*   **技术**：催化剂，帮助学员更快地提升聊天技巧。

## 总结

本节课介绍了初级班开学典礼的内容，包括课程结构、学习方式、初级班和中级班的目标和区别，以及恋爱三板斧。希望学员能够认真学习，积极实践，早日找到心仪的另一半。
