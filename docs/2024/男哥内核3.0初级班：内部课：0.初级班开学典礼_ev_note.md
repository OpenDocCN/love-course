## 课程结构与学习形式 🏗️

上一节我们介绍了课程概述，本节中我们来看看课程的具体结构与学习形式。

有学员会问，为什么既有录播课程，又有群内课程。

录制课程是基础课程。我们将所有学员需要解决的聊天、约会、挽回等问题的共同核心知识，提炼并录制下来。这部分是通识知识，学习周期为两个月。

群内课程是进阶与实践指导。学习周期为一个月。群内课程不讲基础，而是针对你在实践基础课程时遇到的具体问题，进行答疑和深化教学。不同的女生、不同的时间点，实操方法都不同，这需要群内实时指导。

以下是群内课程的两个核心重点：

1.  **病毒聊天**：这是保证学员权益的前沿内容。其目标是让你在掌握基础后，实现从“你撩妹”到“妹子撩你”的转变。
2.  **前沿聊天风格与步骤**：我们会教授最前沿的、灵活可变的聊天框架与步骤，帮助你在不同情境下应对自如。

如果你错过了群内直播，可以通过以下方法查找历史内容：

1.  点击群聊右上角三个点。
2.  选择“查找聊天内容”。
3.  输入关键词（如“开学典礼”）即可搜索到对应内容。

## 初级班与中级班的区别 🎯

了解了课程形式后，我们需要明确你所在班级的核心目标。初级班与中级班的要求和目的完全不同。

**初级班的核心目的**是为你打下终身受用的基础，核心是建立“流量池”。你必须在一个月内，达到每天能稳定新增5-15个女性资源。公式可以表示为：**每日新增流量 ≥ 5**。拥有足够资源，你才能清楚自己的喜好，并缓解“需求感”，避免因资源少而过度关注某一个体。

以下是初级班的具体要求：

*   **个人形象建设**：包括发型、穿衣搭配等，我会逐一检查指导。
*   **聊天能力**：前50句聊天就要让女生对你产生兴趣，将你标记为“A类男生”。
*   **时间利用**：一个月时间非常紧凑，必须高效利用，杜绝拖延。

**中级班的核心目的**是“效率与深化”。学员已具备一定基础，因此着重于快速邀约、快速建立吸引，以及解决实战中遇到的高阶问题，例如一次约会实现四次进挪、化解重大障碍等。

中级班的学习建立在初级班成果之上。因此，中级班学员不应再询问头像、微信没人加等基础问题，而应专注于高阶技巧的实践与优化。

## 群内学习与个人要求 📝

上一节我们介绍了班级目标，本节中我们来看看如何在群内高效学习，并对自己提出要求。

关于群内学习，我希望营造一个纯粹的知识库环境。群里应只有“问题”和“感谢”，避免闲聊。别人的问题与我的答疑，很可能就是你下周会遇到的情况，因此必须记录。

关于学习安排，直播课是给内行人看的，用于分析病毒聊天和化解障碍的技巧。如果你的基础课没看完或没实操，看直播意义不大。你必须先有“地图”（基础步骤），再追求“催化剂”（高阶技巧）。

现在，请在笔记本第一页写下以下两点要求：

1.  **（写下日期与姓名）** 我参加了本课程。我人生中没有几个能像这样，专注于提升自我、寻找更好自己的一个月。
2.  在这一个月里，对于我要求的事情，必须想尽一切办法、不择手段、不惜花费时间与财力去完成。

课程安排上，前两三周是严格执行要求阶段，最后一周是放开实战与精准答疑阶段。

## 恋爱的核心：“三板斧” 🪓

明确了学习要求后，我们来剖析恋爱的核心框架。我将它称为“三板斧”，你需要记在笔记本上。

**第一板斧：动力（油）**
动力就是源源不断的异性资源。公式是：**成功 = 持续的动力输入**。你必须每天完成新增流量的任务，这是所有事情的基础。没有动力，一切技巧都是空谈。

**第二板斧：地图（步骤）**
地图就是行动的步骤和流程。它告诉你每一步该做什么，但不会规定每一句话。你需要根据地图的指引，结合自身情况，建立自己的“话术库”，这过程就是“打破舒适区”。

**第三板斧：技术（催化剂）**
技术能加速化学反应，但不能引发反应。同理，聊天约会技术能让你更快成功，但前提是你拥有足够的“动力”和清晰的“地图”。技术很重要，但不是首要追求。主次顺序错误会导致“泡学中毒”，即永远在追求技巧的轮回中，无法形成自然吸引。

## 答疑与核心问题聚焦 ❓

在课程最后，我们针对一些常见问题进行集中解答，并再次强调看待问题的核心方法。

**关于挽回**：挽回的核心在于对方对你重新产生好奇。但分手意味着对方已否定你。此时不应纠缠，而应立刻专注于“动力”和“地图”建设。当你拥有新生活和新资源时，对方的需求感会变化，挽回契机才会出现。

**关于“个人展示面与本人有差异”**：差异主要不在五官，而在“肢体语言和眼神”是否自信。女人需要被带领，自信是关键。约会时女生提出“你跟照片不一样”，往往是“刁难测试”，看你如何应对，这反而是机会。

**关于“没时间发朋友圈/工作特殊”**：如果工作严重限制你展示生活，请立即申请小号专门用于社交。人生最快的发展是“螺旋式上升”，即同时兼顾主业与自我提升，虽然辛苦，但这是最快的捷径。不要找借口，机会触手可及时没抓住，就永远错过了。

**关于各种具体情境问题（如社团女生、私密空间）**：很多问题都是“次要问题”。核心方法是：**先解决主要问题，次要问题会随之解决或消失**。如果你资源充足（主要问题），就不会纠结于某个特定场景的女生（次要问题）。必须看到问题的核心。

**最后提醒**：如果已经将女生“聊死”，挽回成本极高。请对自己负责，严格按照课程指导操作，避免这种情况。
