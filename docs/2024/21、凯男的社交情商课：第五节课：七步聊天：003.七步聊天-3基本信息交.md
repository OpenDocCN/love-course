# 21、凯男的社交情商课：第五节课：七步聊天：003.七步聊天-3基本信息交

## 003. 七步聊天-3：基本信息交换 🔄

在本节课中，我们将要学习“七步聊天法”的第三步：基本信息交换。上一节我们介绍了如何通过“距离感过渡”让互动升温，本节中我们来看看如何通过交换基本信息，解除对方戒备，建立更深层的联系感。

### 概述：第三步的重要性

在前两节课中，我们分别讲解了开场白和距离感过渡。当你成功调动女生的情绪后，她会觉得你聊天有趣，双方很投缘。然而，这仅代表“聊得来”，并不代表熟悉或信任。因此，第三步“基本信息交换”至关重要。它的核心目的是让女生解除对你的戒备，产生安全感，让她知道你是一个立体、真实且安全的男人。完成这一步，你才能为后续发展奠定基础。

### 基本信息交换的核心方法

进行基本信息交换时，你需要一边自然地交换彼此信息，一边穿插有趣的话题，避免对话变得枯燥。关键在于将信息传递融入轻松对话中，而非机械式问答。

以下是两种建立联系感的方式：

**1. 建立浅层联系感**

这种方式涉及交换基础的个人信息，旨在初步了解对方，营造熟悉氛围。

*   彼此的年龄、职业。
*   兴趣爱好。
*   家庭背景或童年趣事。

**2. 建立深层联系感**

这种方式涉及交流彼此的观念、价值观与经历，旨在建立精神层面的共鸣与连接。

*   彼此的人生观、价值观、世界观。
*   对感情的看法与理想。
*   个人重要的成长经历与感悟。

### 正确的话术与时机

正确的话术需要结合价值展示，并在合适的时机使用。生搬硬套或在不恰当的阶段使用，效果会适得其反。

**正面话术示例（结合价值展示）：**

*   **交换兴趣时**：“我觉得你说话的样子蛮可爱的。我以前也学过播音主持，我感觉你跟我差不多，你很像我以前的同学。” **（在赞美中间接透露自己信息）**
*   **了解年龄时**：“我觉得你的声音蛮好听的。哎，对了，你多大呀？” **（先给予认可，再自然提问）**
*   **探讨爱情观时**：当女生分享后，你可以说：“小时候想当英雄，想变超人，想成为被光环围绕的很厉害的人。现在呀，只想做一个普通人，养一只猫，一只狗，还有一个心爱的女人。” **（传递积极、有画面感的价值观）**
*   **分享经历时**：“我住过地下室，过年的时候，水龙头都冻成冰柱了。那一刻我觉得这个城市不属于我……正是因为这些经历，才有了现在的我。我相信每个人都因为一段经历而蜕变。” **（展示坚韧、深刻的个人成长）**

**错误的话术示例（低价值传递）：**

“我之前住地下室，水龙头都冻成冰柱了。不过现在住的地下室好多了，不像以前那么惨了。” **（这种表述只传递了窘迫，缺乏积极转化）**

### 关键细节与常见误区

进行基本信息交换时，时机和方式同样重要。以下是必须注意的关键点：

*   **切忌在互动初期进行**：如果在开场阶段就查户口式提问（“你做什么工作？”“喜欢干嘛？”），会使对话变得无聊、易终止。
*   **应在吸引建立后进行**：完成前两步，当女生觉得你有趣、愿意聊天后，她甚至可能主动询问你的信息。此时再进行信息交换，氛围更自然，你也能掌握更多主动权。
*   **每句话都应传递能量**：与女生聊天不是索取能量（不断提问“在干嘛？”“吃了吗？”），而是传递能量。调动情绪传递幽默能量，交流三观传递精神能量。持续传递价值，对方才愿意与你深入交流。
*   **不要生搬硬套**：理解方法精髓后，应结合自身经历DIY属于自己的话术，并确保传递积极、正向、高价值的内容。

### 第三步后的行动指引

成功完成第三步，并与女生建立了深层次联系感后，你将拥有更多主动权。

*   **如果窗口很大**：当女生对你非常认可，窗口很大时，你可以直接尝试邀约。
*   **如果处于新手期**：建议大多数初学者继续稳扎稳打，按部就班地走完后续步骤，为关系打下更坚实的基础。

### 总结

本节课中，我们一起学习了“七步聊天法”的第三步：基本信息交换。我们明确了它的核心目的是**解除戒备、建立安全感**。我们掌握了通过**浅层信息交换**建立熟悉感，以及通过**深层价值观交流**建立精神共鸣的两种方法。更重要的是，我们学会了必须在**建立初步吸引后**进行信息交换，并且要使用**能传递自身价值**的话术，避免陷入无聊的问答模式。记住，在正确的时间做正确的事，每一步都不可或缺。
