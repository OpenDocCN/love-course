# 高情商聊天话术 社恐变社牛：22.不要通过送东西追女人

## 概述

在本节课中，我们将探讨在恋爱关系中，不应通过送礼物来追求女性的原因，以及如何以更有效的方式建立关系。

## 不要在确认关系前送礼物

**公式**：在确认关系前送礼物 → 女方感到压力 → 关系紧张

**原因**：

1. **吓到女方**：就像女方突然送礼物给男方一样，男方送礼物也可能让对方感到压力和不安。
2. **要挟关系**：送礼物可能被视为一种要挟，迫使对方接受关系。
3. **金钱交易**：如果礼物不贵重，可能会让对方觉得关系变成了金钱交易。

## 自信与行动

**公式**：自信 → 主动行动 → 建立关系

**建议**：

1. **保持自信**：在任何时候都要自信，不要因为对方的不理睬而失去自信。
2. **主动行动**：通过实际行动，如牵手、约会等，来建立关系。
3. **不要过度追求**：不要试图通过语言或礼物来打动对方，而是要展现自己的魅力。

## 搭讪与互动

**公式**：搭讪 → 互动 → 建立联系

**建议**：

1. **搭讪**：通过问路、询问意见等方式搭讪对方。
2. **互动**：与对方进行有趣的互动，如聊天、分享兴趣等。
3. **建立联系**：通过互动建立联系，为后续发展关系打下基础。

## 总结

本节课中，我们学习了在恋爱关系中不应通过送礼物来追求女性的原因，以及如何以更有效的方式建立关系。记住，自信、行动和互动是建立良好关系的关键。
