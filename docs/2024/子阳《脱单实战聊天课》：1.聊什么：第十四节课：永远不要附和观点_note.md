# 子阳《脱单实战聊天课》：1.聊什么：第十四节课：永远不要附和观点

## 概述
在本节课中，我们将学习如何在与女性交流时避免附和对方的观点，从而避免成为“舔狗”。

## 核心概念
**公式**：永远不要附和女性观点

## 1. 不要附和女性观点
永远不要附和女性的观点。很多男性会错误地认为，只要不花钱就是不爱对方，这种观点是不正确的。

## 2. 高挺商回复
所谓的“高挺商回复”是指对方说什么，你都接得住，不让他掉到地上去。但这种做法容易让人成为“舔狗”。

## 3. 为什么不要成为“舔狗”
不要成为“舔狗”的原因在于，女性不喜欢与“舔狗”在一起。这是因为“舔狗”在女性眼中代表着一种低价值的表现。

## 4. 女性的“木墙”心理
女性的“木墙”心理是指，她们倾向于寻找强者来延续自己的基因。因此，不要附和女性的观点，以免被视为“舔狗”。

## 5. 不要在三观上附和女性
在与女性交流时，如果发现三观上的东西与对方不同，不要去附和对方。

## 6. 举例说明
有一个兄弟在群里问问题，他说：“我不这么认为，我觉得喜欢一个人就应该心甘情愿的为他付出。”这句话中，他犯了两个错误：一是与女性争辩，二是附和女性。

## 总结
本节课中我们一起学习了如何在与女性交流时避免附和对方的观点，从而避免成为“舔狗”。记住，不要在三观上附和女性，尊重自己的内心想法。
