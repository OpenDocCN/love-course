# 任亚《人间清醒后半生》：第23节 遭遇困境面对外面的“蝴蝶”，该怎么办？

大家好，我是任呀。欢迎来到我的课堂。这节课我想和大家分享的是，面对习惯性出轨的伴侣，我们该怎么办？

## 1. 习惯性出轨的本质

**习惯性出轨是一个没有办法治疗的问题。** 如果你认为他是一个病的话。

但是我个人其实对习惯性出轨，我没有评判，我只是把它界定为作为一位专业人士。你知道的，我不会轻易的用道德去评判一些人。我的评判，我的界定就是有某一类人，他不适合一对一的婚姻。

## 2. 如何识别容易出轨的人

**选择比努力更重要。** 你要学会识别哪些是容易出轨的人，哪些是不太适合一对一婚姻关系承诺的人，就是承诺也做不到的。

**识别方法：** 了解他的生命观和价值观。对那些在生命中想要去体验，更喜欢冒险的人，而且他总是想要从错误中学到的人，其实他的他就其实他就本身就不太适合一对一的婚姻。

## 3. 沟通的重要性

作为一位女性，我可能会觉得啊我的伴侣有外遇，嗯，习惯性的出轨。是不是因为我不够优秀，是不是因为我表现不好？

**沟通不畅，会让伴侣更有机会出轨。** 因为沟通不畅，他就有情绪无处表达，那么他可能会也有可能会到外面去倾诉跟别人去表达。

但是这也不足以让一个人成为一个出轨的一个理由，对不对？

## 4. 情绪管理

**情绪不带着情绪性的沟通。** 这时候你需要专业人士的帮助，而不是你自己单枪匹马的独斗。

## 5. 寻求专业人士的帮助

很多人在关系中出现问题。其实其实家丑不外扬更应该找专业人士，而不是找亲朋好友。

## 6. 做出选择

**你有权利选择待在这段关系里继续或者离开。** 但是我要非常清楚知道这是我的选择，主动权在我手里。

## 7. 衡量婚姻的价值

你要好好衡量对方跟你之间关系的价值。这个价值就是我们说婚姻为我们提供的以下几个价值。

**婚姻的N大功能：**

1. 精神上的共鸣和理解。
2. 精神上的共鸣以及共同成长。
3. 情感。
4. 信。
5. 基因好的基因的传递。
6. 经济互助。
7. 资源共享。
8. 共同养育孩子。

## 8. 总结

本节课中我们一起学习了面对习惯性出轨的伴侣，我们应该如何应对。希望对大家有所帮助。
