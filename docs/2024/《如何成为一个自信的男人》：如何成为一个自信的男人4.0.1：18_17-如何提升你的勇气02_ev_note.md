以下是三个非搭讪的小技巧，帮助你提升在两性关系中的勇气：

2. 如果发现美女旁边有空位，直接坐下。
3. 低着头玩手机，或偶尔用余光观察美女。
4. 将此习惯养成。

### 技巧二：眼神接触
1. 在地铁站出入口，尝试与路过的每个女生进行眼神接触。
2. 不需要微笑或说话，只需盯着对方看，直到对方转移视线。
3. 重复此过程。

### 技巧三：试错思维

2. 承担一定风险，积累经验和教训。
3. 及时止损，尝试下一个机会。

## 长期关系中的勇气
在长期关系中，以下几种勇气至关重要：
