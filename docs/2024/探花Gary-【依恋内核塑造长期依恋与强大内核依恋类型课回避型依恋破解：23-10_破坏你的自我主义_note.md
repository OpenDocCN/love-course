## 自我主义的根源与危害

**自我主义**源于我们自恋的一部分，表现为认为自己与众不同，不能走传统路线，是例外。这种自我主义会隔绝你与所有能够真正帮助你的东西，忽略社交或长期关系中的真实反馈。

**公式**：自我主义 = 自恋 + 认为与众不同

**代码**：selfishness = narcissism + uniqueness

## 自我主义的表现与影响

以下是自我主义的一些常见表现：

- 忽视他人反馈
- 认为自身情况特殊，无法接受一般建议
- 过度关注自尊和面子
- 忽视现实，扭曲现实

自我主义会导致以下影响：

- 隔绝高价值和有效帮助
- 忽视他人反馈，导致关系破裂
- 阻碍个人成长和成功
- 导致心理问题，如焦虑、抑郁

## 如何克服自我主义

1. **接受现实**：认识到自己并非特殊案例，与其他人一样，面临相似的问题。
2. **放下自尊和面子**：不要过度关注自尊和面子，学会接受失败和挫折。
3. **尊重他人**：倾听他人意见，接受他人反馈。
4. **放下自我主义**：时刻提醒自己放下自我主义，以开放的心态面对问题。

**公式**：克服自我主义 = 接受现实 + 放下自尊 + 尊重他人 + 放下自我主义

**代码**：overcome_selfishness = accept_realities + let_go_of_pride + respect_others + let_go_of_selfishness
