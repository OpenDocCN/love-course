# 乐嘉7套：5.乐嘉性格色彩全能训练营（入：13 父爱如山——如何理解不同

在本节课中，我们将要学习如何识别和理解四种不同性格色彩（红、蓝、黄、绿）的父亲，并掌握与他们有效沟通、化解矛盾的具体方法。通过学习，你将能更好地与自己的父亲相处，改善家庭关系。

在中国家庭中，父母扮演的角色通常有显著差别。母亲更多主内，操持家务；父亲更多主外，家庭大事多由他做主。因此，与母亲的关系和与父亲的关系往往不同，出现的问题和矛盾也不一样。

本节内容将告诉你，与不同性格的父亲可能产生何种矛盾，以及如何正确相处并化解这些矛盾。

对于尚未接触过“红蓝黄绿”性格色彩体系的朋友，接下来我会先简要阐述四种性格父亲的特点，然后详细告诉你，面对不同性格的父亲时，应采用何种交流方法以达到最佳效果。

## 第一类：红色性格的父亲 🎈

红色性格的父亲富有童心。在你小时候，他可能会趴在地上陪你骑大马、玩积木、打游戏。如果只有他在家时家里总是有些凌乱，他很可能就是红色性格。因为红色性格在生活中相对随意，他们认为东西能找到、住得开心就行，人生说到底就是图个开心。

在家庭聚会时，他常会夸赞你，分享你的好事和好成绩，甚至招呼你：“来，给叔叔阿姨唱首歌”或“站起来让大家看看你长多高了”。这类父亲最懂得追求和分享快乐。

如果你也是红色性格的孩子，你与父亲会像小伙伴一样，很少发生矛盾。但对于其他性格的孩子，你可能会觉得这样的父亲不够靠谱、不够成熟。

例如，电影《内布拉斯加》中的红色性格父亲，收到一条虚假的中奖短信后深信不疑，并兴奋地与他人分享这个“好消息”，甚至添油加醋，结果遭到全村人嘲笑。他的儿子为了维护父亲的尊严，卖掉自己的车，买来二手豪车并装满“奖品”让父亲在村里炫耀，最终为父亲赢得了面子。

这个故事说明，红色性格的父亲对快乐和他人认可的追求非常高。作为子女，你只需做好两件事：

以下是与他相处的两个核心方法：

1.  **经常汇报好成绩**：你需要有声有色地向父亲汇报你的好消息。当他接收到这些积极信息时，会乐于分享给朋友，从而获得巨大满足感。
    *   **公式**：`汇报好消息 = 详细描述 + 积极情绪`
2.  **给予足够关注和认可**：密切关注父亲的生活，并及时给予真诚的赞美。例如，当他发老照片时，你可以说：“爸，我同事都说您当年是汤姆·克鲁斯级别的帅哥！” 当他尝试新事物时，你可以说：“爸，您真时髦，很多同龄人都跟不上潮流。”
    *   **代码逻辑**：
        ```python
        if 父亲分享了新动态：
            发送真诚且具体的赞美
        ```

这种方法几乎无需金钱成本，仅凭语言就能让父亲非常开心。懂得说话的艺术，往往比物质馈赠更能事半功倍。

上一节我们介绍了如何与追求快乐和认可的红色父亲相处，本节中我们来看看截然不同的蓝色性格父亲。

## 第二类：蓝色性格的父亲 📐

如果你父亲是蓝色性格，你会深刻体会到什么是规矩。例如，他可能规定：家里的筷子必须按材质和用途严格分类；厨房有四块功能各异的抹布整齐排列；卫生间洗护用品的标签必须全部朝外。

蓝色性格的人坚持规则与秩序，拒绝松散和混乱。他们有强烈的意愿将无序变为有序，且一旦制定规则，任何人都必须严格遵守。他们对自己和他人要求都很高。

因此，如果你是蓝色性格父亲的孩子，可能会被他琐碎的要求和频繁的管束所困扰。例如，电影《帕丁顿熊》中的父亲，或演讲《中国式父亲》中描述的父亲：孩子考了90分，他会问为什么不是100分；孩子达到了他的要求，他又会设立更高的标准。这种持续的不认可可能导致子女叛逆，关系破裂。

其实，蓝色性格的父亲与红色父亲一样，情感丰富且需求很高，但表达方式不同：
第一，他不愿轻易表扬，认为严格要求才能使人进步。
第二，他不善表达情感，即便想念你，也绝不说出口，而是通过其他方式默默关心。

例如，一位学员的蓝色性格父亲总是批评他的工作，但母亲告诉他，父亲其实非常关注他的生活，甚至偷偷比划着照片为他买合身的西装。蓝色父亲的内心像一堵厚城墙，希望子女走进来，却摆出拒人千里之外的姿态。

最正确的相处方法不是进攻或逃离，而是**无言的相守和陪伴**。你只需静静待在他的“城墙”边，他说什么你都听着，他不说你就等着。时间久了，他知道你始终在那里，安全感会让他内心的城墙逐渐打开。

理解了追求秩序与默默关怀的蓝色父亲后，我们再来看看目标明确、作风强硬的黄色性格父亲。

## 第三类：黄色性格的父亲 🏆

如果你父亲是黄色性格，你与他最大的矛盾可能源于他强烈的控制欲。与蓝色父亲不同，黄色父亲的控制更加直接，喜欢“以暴制暴”，信奉“棍棒底下出孝子”。若未经过修炼，他们可能对不按自己要求行事的孩子进行严厉批判甚至体罚。

例如，董卿的父亲、韩国电影《思悼》中的老皇上、电视剧《虎妈猫爸》中的杜峰、《摔跤吧！爸爸》中的父亲，都是黄色性格的典型。他们通过严格甚至苛刻的训练，希望子女变得强大和成功。

黄色性格一生追求成功，因此也望子成龙。他们认为现在的严厉是为了让孩子未来在社会上少犯错。但这种压迫感，尤其对红色性格的孩子来说，可能难以承受，甚至导致关系决裂。

理解父亲的苦衷是第一步。面对他的压迫，你需要运用 **“三步法”** ：

以下是应对黄色性格父亲压迫的三步沟通策略：

1.  **站队表态**：首先缓和态度，表明你与他在目标上是一致的。例如：“爸，我理解您的苦心，其实我也希望……”
2.  **分析利弊**：客观、理性地分析按他的方式做与按你的计划做，各自的好处与坏处。
3.  **说明计划**：给出你深思熟虑后的具体计划和步骤，让他看到你的主见和可行性。

例如，一位听友用此法应对父亲催生：先表态自己也想要孩子（站队），再分析目前生育对事业和经济的压力，以及推迟两年后的优势（分析利弊），最后说明未来两年的具体提升计划（说明计划）。父亲看到他有理有据的规划后，便不再逼迫。

最后，我们来看与黄色性格完全相反的绿色性格父亲。

## 第四类：绿色性格的父亲 🌿

绿色性格的父亲毫无控制欲，崇尚和谐，习惯妥协。简单说，就是“你爱干啥干啥”。他们知足常乐，以他人为中心，例如《阿甘正传》中成为父亲后的阿甘。

这种性格是一把双刃剑。好处是家庭氛围宁静和谐；坏处是当你遇到巨大困难时，他可能无法为你拿主意，显得“软弱无能”，甚至因为不懂拒绝而给家人添麻烦（例如轻信推销而购买不需要的东西）。

与绿色性格父亲相处，只需做好两件事：

以下是与他和谐相处的两个要点：

1.  **提供舒适享受**：既然他喜欢舒适平和的生活，你就为他创造这样的条件，例如购买按摩器、改善饮食等。他会享受并感念你的心意。
2.  **明确告知后果**：不要简单阻止他做某事，而要**强调其行为可能对家人造成的负面影响**。例如：“爸，我赚钱很辛苦，如果您被骗了，我和妈妈的生活都会受影响，我也没法攒钱结婚让您抱孙子了。” 绿色性格非常在意亲人感受，听到这会主动避免麻烦。

本节课中我们一起学习了与四种不同性格色彩父亲的相处之道。

## 总结 📝

对四种性格的父亲，核心相处要点如下：
*   **红色父亲**：多关注、多赞美，满足他对快乐和认可的需求。
*   **蓝色父亲**：保持稳定沟通，注重无声的陪伴，给予他安全感。
*   **黄色父亲**：尊重其意见，通过“站队、分析、计划”三步法，展现你的主见和能力。
*   **绿色父亲**：为其创造舒适生活，并通过告知对家庭的潜在影响来引导其行为。

只要你理解并运用这些方法，就能更好地理解你的父亲，并与之建立更有效、更和谐的亲子关系。
