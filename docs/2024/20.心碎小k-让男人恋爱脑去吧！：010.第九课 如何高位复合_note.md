# 20.心碎小k-让男人恋爱脑去吧！：010.第九课 如何高位复合

🎼第九课如何高位复合。首先声明，不是所有人都可以复合。只有对方对你仍有感情，才有可能复合。

**误区解析**

以下是挽回的常见误区：

1. **纠缠**：分手后，对方需要独立空间。过度纠缠只会让对方更想摆脱你，不利于建立联系。
2. **过度关注**：分手后，给对方一些空间。不要总是关心对方，以免影响情绪和关系。
3. **低姿态求复合**：爱情不是求来的，而是相互吸引。低姿态求复合只会让对方更厌烦。

**复合步骤**

1. **道歉加神秘阶段**：向对方道歉，表达自己的改变，但不要提出复合诉求。消失一段时间，让对方思考。
2. **舒适感，建立联系阶段**：见面时只谈风月，不谈感情。通过行为和语言让对方回忆美好，感受你的变化。

**男人心理**

男人分手后，会经历一个空虚寂寞的阶段。了解男人心理，过好自己的生活，顺其自然。

**小作文模板**

小作文的目的：让对方忘掉你的不好，只记得你的好。

**总结**

本节课介绍了如何高位复合，包括误区解析、复合步骤、男人心理和小作文模板。希望对你有所帮助。
