# 小怡《情绪吸引力-人性篇》：013.13-聊天五步法-04欣赏

🎼hello，你好，我是小怡。今天我们来学习聊天五步法的第四步——欣赏。

### 欣赏的重要性

上节课我们提到筛选是关系的分水岭。如果对方没有通过筛选，我们需要回到了解阶段继续努力。如果对方通过了筛选，并且你对他的兴趣是真实的，那么我们就需要进入欣赏的阶段。

为什么这个时候要进入欣赏的阶段呢？我们来回顾一下前三步：

1. **破冰**：打破陌生感，建立基础的安全感。
2. **了解**：充分交换信息，通过合理自我展示建立吸引。
3. **筛选**：确认对方对你的兴趣是真实的。

经过这三步后，你们双方的关系状态是什么？对方已经对你产生了兴趣，但你还没有回应。那么，如果你对对方产生了浓厚的兴趣，你最希望得到的是什么？当然是希望对方也表示出对你的兴趣。

### 欣赏的目的

欣赏这个环节的目的是让对方知道你们是彼此感兴趣的，把你的真实想法传递给对方，这样你们才会产生结盟的感觉。欣赏环节的目的很简单，就是准确的表达你的意图，表达你对对方的欣赏，向对方示好，让对方明确知道你对他也是很感兴趣的。

### 欣赏与暴露需求感的区别

要注意，这里的表达意图和破冰环节的意图声明有本质的区别。在破冰环节，我们要做的意图声明是表达你对当下这段关系未来发展方向的期待。而这个环节的意图声明是直接表达你对对方的认可和欣赏。

### 如何欣赏

欣赏最重要的就是找准角度。夸人要夸细节，这样才显得真诚。但这只是一个小技巧，可以帮你的话语锦上添花，但很难解决本质的问题。本质的问题是对方希望你欣赏他的地方是什么。

我们要欣赏一个人，最重要的是先要找准欣赏的角度。万事万物，我们都可以从内在和外在两个角度去进行划分。

**内在**：学识、性格、心态、生活态度、价值观等。

**外在**：外貌、身材、长相、学历、家庭状况等。

除了人之外，物品也分内在外在。比如一栋房子，它的外在是房龄的新旧、面积的大小、装修的好坏、小区的美化等等。而内在是地理位置的价值、是否学区房、周边的便利程度、包括建筑质量等等。

所有的东西我们都可以结合内在和外在两个角度去欣赏。记住一句话，内在价值高的，多欣赏外在，外在价值高的，多欣赏内在。

### 欣赏的流程

1. 发现对方身上不易被别人察觉的地方。
2. 用心的去观察，找到对方的优点。
3. 进行夸赞。

千万呀不要为了夸赞而说一些假话，做一些不真诚的行为。因为如果你不真诚的话，对方是能直接感受到的。

### 欣赏的练习

想要做好欣赏这个环节，首先你要练习，拥有欣赏别人的能力。无论是人或事物，你都可以从内在外在两个角度去挖掘对方身上的优点。

当你掌握了这个能力之后，你会发现欣赏别人就不是一件难事儿，同时呢也对你厌就温柔的心态有很大的帮助。

### 欣赏的反馈

当你表达欣赏之后，我们要观察对方给你的反馈。如果前面这些环节啊，你都处理得当的话，你表达欣赏之后，对方给你的回应啊，应该是非常积极正面的，会有那种啊两个人一拍即合的感觉。

反之啊如果对方给你的回馈是很谦虚的，不太接纳你的欣赏。那很有可能啊是你在了解阶段建立的吸引还不够，需要回到这个阶段继续下功夫。

### 总结

欣赏是在恰当的时机，让对方知道你对他也非常的感兴趣。从而达成互相的认可，产生结盟的感觉，为最后一个环节达成目标做铺垫。

本节课中我们一起学习了欣赏的重要性、目的、与暴露需求感的区别、如何欣赏以及欣赏的练习和反馈。希望对你有所帮助。
