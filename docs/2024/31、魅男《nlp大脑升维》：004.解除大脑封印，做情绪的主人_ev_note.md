# 31、魅男《nlp大脑升维》：004.解除大脑封印，做情绪的主人

## 概述

在本节课中，我们将学习如何解除大脑封印，成为情绪的主人。我们将探讨负面思维和情绪的来源，以及如何通过改变思维方式和建立信念系统来提升情绪管理能力。

## 一、负面思维与情绪

### 1. 负面思维

*   **负面词语**：例如“我不行”、“我做不到”、“这样行不通”等，这些词语会阻碍思考，限制行动。
*   **思维漏洞**：例如“这不可能”、“我不可能做到”等，这些思维会让人止步不前。

### 2. 负面情绪

*   **来源**：接触外在的人事物产生的负面影响，或他人侵犯了你的利益。
*   **解决方法**：
    *   **抽离出来**：站在上帝视角思考问题，避免情绪放大。
    *   **提升解决问题的能力**：通过学习和实践提升解决问题的能力，减少负面情绪。
    *   **建立信念系统**：建立一套信念系统，看待世界的方式，提升包容性。

## 二、情绪管理

### 1. 情绪的本质

*   **得失心**：情绪的本质是得失心，得到东西会开心，失去东西会痛苦。
*   **吸引**：只有吸引才能产生情绪波动，才能产生正面情绪和负面情绪。

### 2. 情绪管理的公式

*   **吸引 + 需求 = 价值**
*   **正面情绪 + 负面情绪 = 情绪波动**

### 3. 情绪管理的技巧

*   **提升吸引力**：通过提升自身价值，提升吸引力。
*   **控制情绪**：通过抽离出来、提升解决问题的能力、建立信念系统等方式控制情绪。

## 总结

本节课中，我们学习了如何解除大脑封印，成为情绪的主人。通过改变思维方式和建立信念系统，我们可以提升情绪管理能力，更好地应对生活中的挑战。
