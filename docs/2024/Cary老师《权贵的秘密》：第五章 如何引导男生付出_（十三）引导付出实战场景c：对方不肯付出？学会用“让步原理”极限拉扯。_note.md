# Cary老师《权贵的秘密》：第五章 如何引导男生付出_（十三）引导付出实战场景c：对方不肯付出？学会用“让步原理”极限拉扯。

## 概述
在本节课中，我们将学习如何使用“让步原理”来引导男生付出，特别是在对方不愿意付出时如何操作。

## 让步原理
当男生没有兑现承诺时，我们可以通过以下步骤来测试他的付出底线：

1. **学会沟通**：与男生进行沟通，逐渐缩小自己的要求。
2. **测试底线**：通过逐步让步，观察男生的反应，以确定他的付出底线。
3. **心理影响**：让步会促使男生产生愧疚感，从而更愿意接受要求。

## 实战案例
以下是一些实战案例，展示如何应用让步原理：

### 案例一：要求购买宾利
- **初始要求**：要求男生购买一台宾利。
- **让步操作**：提出预算问题，引导男生降低期望。
- **最终结果**：可能得到一辆保时捷。

### 案例二：欧洲旅游
- **初始要求**：要求下个月一起去欧洲旅游。
- **男生反应**：表示预算不足，可能推迟计划。
- **让步操作**：提出去三亚旅游，降低期望。
- **最终结果**：男生可能同意去三亚。

## 行为引导
为了使男生更愿意付出，我们可以从以下方面进行行为引导：

1. **从小到大**：从小的付出或习惯开始引导，例如称呼、行为习惯等。
2. **及时回馈**：在男生完成具体行为或付出时，给予积极的语言回馈。
3. **绑定情感**：将男生的行为与爱你的想法绑定。

## 终极目标
在引导过程中，我们要明确自己的终极目标，并不断提醒男生。

1. **播种**：在日常生活中，不断向男生展示你的目标。
2. **长期铺垫**：通过长期的赞美和良性回馈，引导男生产生付出意愿。
3. **提醒目标**：在接近目标时，提醒男生你的期望。

## 总结
本节课中，我们学习了如何使用“让步原理”来引导男生付出，并从行为引导和终极目标两个方面进行了详细讲解。希望这些方法能帮助你更好地引导男生付出。
