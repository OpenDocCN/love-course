# 男性思维提升上帝视角1.0：007. 第七节课：不同视角的差异 👁️

在本节课中，我们将要学习人类集体意识如何从原始的“动物性视角”演变到农耕文明的“父权视角”，并分析这两种视角背后的核心逻辑与权力结构。理解这些差异是提升认知、建立“上帝视角”的关键一步。

## 从动物性视角到父权视角的演变 🏹

在原始社会，人类的生存环境非常凶险。一切活动都以生存为前提。男女分工协作的模式，是一种被自然规律筛选后的必然形态。我们可以理解这是一种幸存者偏差。

原始人类不得不利用男女生理特性上的优势，各司其职。此时，人类的集体意识还处于生存模式下的动物性视角。人们头脑里思考的问题和猴子没有多大区别，例如温饱、安全与种族的繁衍。

人与人之间并没有太多的精神交流和需要。在这个阶段，由于分工上男女相对平等，甚至存在以母权为主导的社会形态。这种结构我们在很多动物群体上也可以看到，例如蜜蜂、蚂蚁等。

因为部落的高度群居生活，所以照顾族群的女性，相当于部落结构里的CEO管理者。男性则有点像蜂群里外出觅食的工蜂。动物性视角的底层逻辑就是**丛林法则**。于是，我们带着这样的思维惯性，一路进化发展。

## 农耕文明与父权视角的崛起 🌾

上一节我们介绍了原始的动物性视角，本节中我们来看看定居的农耕文明如何催生了新的视角。

进入定居的农耕文明后，男性的生理结构导致他们的体力优势，变成了生产力的绝对优势。因此，在这个时期，男性群体掌握和控制了话语权。阳性能量的思维是进攻性的，是战争的思维。

他们通过力量的角逐来分高下。就像武林高手见面，要先过招，成王败寇，打输的人心甘情愿地臣服于强者。在人类心智发育的初期，这种简单粗暴的逻辑能够被简单的集体意识所理解。

在这场性别战争里，男性靠着武力值暂时成为了赢家，这就是父权社会的集体意识环境。征服者自然有一种傲慢，是不屑去学习被征服者的思维模式的。男人是制定游戏规则的人，如果想参与这场游戏，就必须学会和他们说话的方式。

于是，阳性能量的思维模式就成为了官方语言。这就仿佛多年前，世界霸主是美国，于是英语就成为了世界的通用语言，这就是文化的霸权。语言背后是思维方式，而性别文化的霸权也伴随着男性的思维方式，这就是权力的语言。

就像中国古代将“君君臣臣父父子子”作为伦理纲常，充满了服从与奴役、征服与控制的权力斗争和博弈。

## 个体心智成长中的视角冲突 💥

我们在回过头来观察个体心智的成长过程。到了生理性激素的发育阶段，男性的生理结构导致他们拥有体力优势。

他们听不懂又理解不了。女孩子们很可爱，可又似乎有一种魔法，可以让人神魂颠倒，不用打架分出胜负，就能让他们缴械投降，乖乖听话被控制。这不符合男性“成王败寇”的逻辑。

*   **攻击方式的错位**：在父权社会的视角里，他们习惯了用武力这种物理攻击。因此，女性力量在他们看来是一种“魔法伤害”。

中国的圣人孔子，他老人家也对此束手无策，说出了“唯女子与小人难养也”的话。“小人”指的是心智尚未成熟的未成年人，他们还没有学会驾驭内心的小野兽，因此无常且不可控。而“女子”说的那套情绪语言，则令人无从揣测，直接困惑。
