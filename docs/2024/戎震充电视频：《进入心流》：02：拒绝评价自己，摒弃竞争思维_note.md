# 戎震充电视频：《进入心流》：02：拒绝评价自己，摒弃竞争思维_note

在本节课中，我们将要学习如何通过“拒绝评判自我”来进入心流状态。核心在于防止内心陷入竞争性思维，并学会接纳当下的自己。

## 什么是竞争性思维？

上一节我们探讨了心流的基础，本节中我们来看看阻碍心流的一种常见心态：竞争性思维。这种思维模式往往源于成长环境。

许多人从小就被父母拿来与“别人家的孩子”比较。这种现象在教师或军人家庭中尤为常见。这些家庭往往强调事事争第一，无论是学习还是体能，都处于一个高度比较的环境中。

教师家庭的孩子有时面临更大压力，因为父母可能将工作中的情绪和要求带入家庭，认为“教师的孩子学习怎么能差”。然而，养育孩子本身充满随机性。

以下是关于人生与竞争的几点事实：
*   寒门可能出贵子，富裕家庭的孩子也可能表现平平。
*   孩子的未来走向是多元的，有人走学术路线，有人经商，这存在随机概率。
*   每个人都有自己的命运轨迹，我们当前所处的位置，或许正是我们当下应该所在的位置。

## 接纳自我：心流的前提

要进入心流状态，关键在于接纳自我。而接纳自我的前提，正是停止评判自我。这种自我评判往往源自我们内心深处的竞争思维。

需要明确的是，**拥有竞争思维**与**进行竞争活动**是两回事。拥有竞争思维意味着内心总想着“我必须比别人强”。然而，人不可能在方方面面都超越所有人。

所谓的“第一”是相对的。你是全县的高考状元，但换一个县、一个国家或一个领域呢？比较的对象一旦改变，“第一”的意义也随之改变。这种“什么都得比别人强”的思维是有害的。

## 方向比努力更重要

在三维空间中看待人生，无论一个人向前、向后、向左、向右奔跑，时间都以平稳的速度向前流动。这里存在一个关键悖论：**你的目的地可能并非你当前努力的方向，而你真正适合的方向，你自己未必知晓。**

例如，如果一段婚姻注定分离，那么无论多么努力经营和挽回，都可能没有意义。作为无法掌控时间的个体，我们容易陷入当下的矛盾，过度批判自己，认为自己不够好、不够努力。这本质上是一种对预期的错判。

下一期我们将探讨“时间的权实性效应”。为了便于理解，我们可以将人生简化看待：

*   设 **时间** 为 **Y轴**。
*   设 **人生的努力方向** 为 **X轴**。

你的努力会影响人生轨迹在坐标图上的曲率和倾斜度。例如，你非常努力，可能达到一个75度的仰角，抵达坐标上的某个点。**但这个点，未必是你真正应该停留的位置。**

因此，进入心流状态最重要的是：**明白你为了什么而努力，为了什么而活。** 首先，你必须明白自己是谁。

## 认识生命的本质

你是时间与时代的产物，是一个机缘巧合下随机产生的生命体。这个生命体的基本任务是维持自身的生存与繁衍，在为社会贡献能量后，生命便会终结。

“吾生也有涯，而知也无涯，以有涯随无涯，殆已。” 我们不应以有限的生命去追逐无限的外物。不要总想着与他人竞争，因为“人外有人，山外有山”，你不可能永远匀速领先。

如果用做生意的心态经营人生——一味追逐利益，且认为利越大越好——是不可取的。因为利益大到一定程度，你需要考量自己是否承受得住。对于普通人而言，年入百万可能是一个极限；而要超越这个数字，则需要经营上市公司、大型工厂或成为顶级网红，这需要不同的能力和机遇。

人生需要了解并接纳自己、父母、孩子的平庸。生命的本质包含着平庸，没有人能永远站在舞台中央。各领风骚数百年，每个人都会衰老、死亡。但终点是死亡，并不意味着过程没有意义。死亡可以看作一种约定好的休息，因为一生已经努力过。

宇宙能量物质不灭，生命形式可能只是相互转换。没有人能穿透生死回来告诉我们真相。因此，最重要的不是来世，而是**今生该如何度过**。把每一天过好，一生自然就好了。

## 放下与命运的对抗

不要以时间为维度去竞争，例如设定“明年一定要挣多少钱”的目标，因为人无法完全控制命运。常言道“三分命，七分运”，与命运对抗通常是徒劳的。

但在命运允许掰手腕的领域，比如求学阶段，努力可以改变命运——“少壮不努力，老大徒伤悲”，知识确实能改变命运。要想提升社会地位，需要遵循社会学的基本原理，即**权力、财富、声望**构成的三角关系。

更快进入心流状态，需要更快地接受自己，并明白当前状态的目的。你的行动不应是为了升官发财或满足他人预期，而是因为你认为有必要这么做——为了增强自己，或找到一种放松状态，最终是为了**自我表达**。

自我表达，即厘清你活在世上的意义，你对人类社会的作用。你需要清晰了解自己的命运和终点。当经历过无数灵魂的暗夜后，你可能会进入一种纯粹的服务状态：在服务他人与社会的同时，也满足自身及家庭的生存需要（挣钱吃饭）。这两者并不矛盾。

## 专注自己的人生轨迹

孔子说“三十而立，四十不惑，五十知天命”。人生只能做自己，你无法成为别人，也无法完全体会他人的喜怒哀乐，因此没有必要羡慕别人。

即使有一个和你一模一样的人，若生于不同时代，境遇也会完全不同。所以，不要羡慕别人的状态。

把你的人生画成一条线或一个面，专注于当下的每时每刻。不要总想着“努力十年后就能怎样”，因为十年后你未必过上想要的生活，即使过上了，也未必快乐。

人生有两大遗憾：一是未实现理想；二是实现了理想，却发现不过如此。因此，最重要的是**知晓当下的喜乐在哪里**。

## 心流的核心：停止自我批判

进入心流状态最关键的一点，就是**不能批判自己**。一旦开始批判，你就会觉得当下的自己不完美。

实际上，人时刻处于完美与不完美的状态中，关键在于你如何定义“完美”。难道要身上没有任何分泌物、完全健康、所有指标都正常？这种状态只能维持一瞬，并非长久状态。

长久的状态如同钟摆，始终在摇摆中，不会停留在顶点或终点。达到势能最大值（顶点）后，它就会返回。人生也是如此，如同太极八卦或DNA的双螺旋结构，**没有绝对的终点，而是在滚动中向前**。

明白了这个基本原理，进入心流就不难了。你学习、阅读，应该是为了取悦自己，让自己达到更好的存在状态，而不是为了让他人（如母亲）高兴。

只有在这种状态下，你才会认识到：**当下的我就是完美的，值得拥有现在的一切，并且一切都会变好。** 拥有这种“视死如归”、“向死而生”的心态，反而能活得更好。

## 心流是一种心境

心流（Flow）完全是一种心境。儒家文明与英美文明（盎格鲁-撒克逊）的一个不同点在于：我们这边的人往往能力很强但自信心不足；美国人则可能能力参差，但普遍自信心很强。

信心需要被激发，而不是依赖他人的看法。儒家文化中“富贵不还乡，如锦衣夜行”等观念，追求的是外在的认可和名分，这在佛家看来是“虚妄”。追求虚幻的东西没有意义。

我们应该“去虚就实”，追求人生真正的快乐——即你的体验和感受。过上不焦虑、心灵富足、精神满足的生活。觉得“现在的状态就很好”，即使未来可能拥有更多，但深知每个阶段都有其独特的价值，不会一味认为“下一个阶段才会快乐”。

## 总结：拥抱当下的完美

本节课中我们一起学习了如何通过摒弃竞争思维和停止自我评判来进入心流。

真正的匮乏感，往往是与他人比较时产生的心理状态。实际上，生存所需无非是水、阳光、空气和食物。我们常常夸大了自己的真实需求，这反而阻碍了心流的产生。

要想进入心流，你需要明白：**当你放下一切，心无旁骛时，才是进入心流的最佳时机。**

人生的完美状态并不存在，或者说，**人生的每一点都是完美的**。如果你总认为考上研、找到工作、结婚、退休后才会快乐，那你永远得不到快乐。真正的快乐在于觉得**当下就是快乐的**。

一百年后，我们或许都不存在了，但这并不意味着我们现在的生命没有意义。活着的意义在于心灵的体验与满足。正如王阳明所说，心的境界达到了，状态也就达到了。幸福、完美、快乐，都是内心的创造物，需要你自己去领悟和调节。

当你悟到“自己的一生每一刻都是完美的”，你就会进入一种“三美”状态：**过去是完美的，现在是完美的，未来也是完美的。** 你的人生因而完整。
