# 《进入心流》教程：02 拒绝评价自己，摒弃竞争思维

## 概述
在本节课中，我们将探讨如何拒绝自我评价和竞争思维，以进入心流状态。

## 拒绝评判自我
**核心概念**： 
- **公式**： 心流状态 = 接纳自我 - 评判自我

**内容**：
- 拒绝评判自我是进入心流状态的关键。
- 竞争思维源于从小被父母与他人比较，如“别人家的孩子”。
- 竞争思维导致过度批判自己，认为自己不够好。

## 竞争思维的影响
**内容**：
- 竞争思维使人陷入矛盾，过度批判自己，认为自己不够好。
- 竞争思维导致对预期的错判，认为人生有终点，追求终点。
- 竞争思维使人陷入焦虑，追求物质满足，忽视精神满足。

## 接纳自我
**内容**：
- 接纳自我是进入心流状态的前提。
- 每个人都有独特的命运，应接纳自己的位置。
- 不要以时间为维度去与时间竞争，接受命运的安排。

## 自我表达
**内容**：
- 自我表达是人生的目的，了解自己的命运和方向。
- 服务他人，满足自己的生存需求。
- 不要羡慕他人，专注于自己的生活和体验。

## 完美与不完美
**内容**：
- 人生没有终点，每个阶段都是完美的。
- 不要追求完美，接受自己的不完美。
- 每一点都是完美的，才能构成完整的线。

## 总结
本节课中，我们学习了如何拒绝自我评价和竞争思维，以进入心流状态。通过接纳自我、自我表达和接受不完美，我们可以更好地享受生活，实现人生价值。
