# 清风老师合集：10情绪自控力，不再发脾气11：110.第四课：如何规划休息，摆脱熬夜，让你的

在本节课中，我们将要学习如何科学规划休息，摆脱熬夜的困扰，从而获得饱满的精力与良好的自控力。

在上一节课程中，我们介绍了如何规划饮食和休息以提升自控力状态。本节中，我们来看看如何具体规划休息，摆脱熬夜。

你是否感觉晚上总是难以入睡，翻来覆去，一看手机时间已晚，想到明天的事情便更加焦虑，结果依然无法入眠。

我们渴望的自控力，通常表现为三个方面。第一，拥有良好的执行力。第二，对自己的行为有良好的控制能力，能做出有远见的决策，考虑未来规划。第三，拥有良好的情绪，待人温和。然而，睡眠不足会严重损害我们的自控力表现。

休息不足对自控力有三个致命影响。

以下是具体影响：

1.  **执行力降低**。想到第二天要汇报工作，结果凌晨三点还未睡着，昏昏沉沉的汇报会让你觉得自己表现不佳，领导也可能不满意。回到家后，你会特别后悔，不明白为何总是无法控制自己。
2.  **促使短期选择**。2019年诺贝尔经济学奖得主在《贫穷的本质》一书中发现，穷人难以摆脱贫穷，是因为他们常做短期决定。在休息不足时，我们也倾向于做出许多短期选择。例如，为了对抗疲劳，大量饮用咖啡、茶等含咖啡因的饮料，导致第二天晚上继续失眠，形成恶性循环。或者，因为状态差、很累，为了提振精神而摄入一些平时不吃的垃圾食品。这些短期决策，会让我们离真正的自控力越来越远。
3.  **情绪暴躁易怒**。在休息不足的情况下，我们往往会变得更加暴躁易怒，难以用平和的心态处理遇到的问题。

那么，我们为何会休息不足呢？

以下是三个主要原因：

1.  **无法处理好焦虑情绪，被焦虑绑架**。在第二节课程中，我们学习了两个方法：改变内在对话模式和通过情绪释放法。情绪释放法的本质是心理学中的一种疗法，叫做意向对话，即将我们的情绪具象为某个物体甚至动物，然后将其释放掉。
2.  **总是被手机软件绑架和唤醒**。例如，手机软件常在睡觉时推送通知。晚上十点弹出一个通知，你一刷手机不知不觉就到了十二点。这时又来一个通知，一看时间又过去了。对此，我有一个建议：尽量把手机放远一点，特别是十点以后尽量不要看手机。如果无法做到，十点后请打开勿扰模式，关闭通知声音。不要让手机不断打扰你，而是由你主动控制注意力。这样可以创造一个不被打扰的环境。第三个建议是床头放一本大部头的书。例如，我之前在床头放了《人类简史》，晚上睡不着时看一看，很快就入睡了。这种大部头的书特别适合催眠，你也可以试试。
3.  **缺乏合适的睡眠环境**。例如光线特别刺眼，即使拉上窗帘，外面仍有光线干扰休息。再比如噪音多，楼下汽车驶过或邻居的声音都会干扰我们，并容易在睡眠中惊醒。你可以选择购买一个遮光眼罩，并用手机下载一个白噪音软件，播放海浪、虫鸣或下雨等声音，这些声音容易助人入睡。白噪音的原理是：用一个持续性的噪音让我们的听觉适应，这样当其他噪音出现时就不会显得突兀，让我们的听觉感觉更舒适。

现在，我们对休息不足的原因有了一些了解。接下来，我们来认识一下什么是睡眠。

本节课内容非常超值，因为我将分享世界顶级的睡眠方法。提出这套理论的人是英超曼联和NBA的睡眠教练尼克·利特尔黑尔斯，他是一位睡眠专家。他将最前沿的脑神经科学与心理学结合，提出了自己的睡眠理论。

我们一起来了解睡眠的5个阶段。他将睡眠比喻为从楼梯上到一个滑梯，最后又回到楼梯的循环过程。就像一个小朋友慢慢走上楼梯，然后滑下滑梯，滑梯末端又回到起点，形成一个循环。这听上去很像电影《盗梦空间》。

以下是五个阶段：

1.  **上楼梯**：我们刚躺下，还很清醒，容易被吵醒。
2.  **在楼梯上**：我们开始进入浅睡眠，大声响几乎可以吵醒我们，但一些小声音不会。
3.  **在楼梯下**：我们进入深睡眠。
4.  **旋转滑梯**：我们已在滑梯中滑下，进入非常深度的睡眠。此时脑电波放慢，大脑得到修复和休息。
5.  **重回楼梯**：在深睡眠后，我们重复第一个“上楼梯”的过程。两者相似之处是都容易被吵醒，但在第五阶段，我们处在梦境中。

他提出了一个 **90分钟休息法**，即睡满这5个周期（一个周期90分钟），在第五个周期结束时被唤醒。这时我们会感到精力充沛，状态非常舒适。

如何应用到日常生活中呢？举个例子：一般成年人需要睡满5个周期，即 `5 * 90 = 450` 分钟，也就是7.5小时。如果你晚上12点半入睡，可以定早上8点的闹钟起床。当然，这取决于你的入睡时间。如果你比较难入睡，可以12点入睡，早上8点起来。这样大约能睡满7.5小时，即完整的5个周期。醒来后你不仅会感到精力充沛，还会觉得非常舒服。以此类推，如果你凌晨2点入睡，要睡满7.5小时，就需要在早上9点半起床。

你或许有过越睡越累的经历。这就是在错误的睡眠周期被唤醒导致的结果。例如，在第三或第四个深睡眠周期被唤醒，人会感到非常疲倦甚至无力。因此，请尽量用90分钟睡眠法来规划你的睡眠。

接着，我将分享提高睡眠质量的5个步骤。

以下是五个步骤：

1.  **找到自己的睡眠规律**。如果你习惯12点半睡，尽量保持这个稳定规律。不要突然有一天两点睡，另一天为了早睡又调到11点。不规律的睡眠对身体伤害比晚睡更大。
2.  **建立睡前仪式感**。以我为例，睡前首先关闭手机，避免干扰。其次，我会选择听一些白噪音，用音箱播放海浪声等，然后看会儿书，看完后舒服地睡觉。
3.  **降低干扰因素，创造良好睡眠环境**。我特别喜欢戴眼罩睡觉，在睡眠过程中不易被外界干扰。而且我经常需要去各地演讲，在飞机、汽车上补觉，戴眼罩已成为我的生活习惯。如果你的环境也易受打扰，请尽可能戴一个眼罩。
4.  **建立起床规律**。起床后应有一套固定动作让你得到唤醒。我通常这样做：先取一片VC泡腾片泡杯水，然后蒸两个鸡蛋作为早餐，接着刷牙洗脸。这时人慢慢进入清醒状态，喝完水后身体被完整唤醒，整个人就调整过来了。
5.  **做一些起床后的小动作，让大脑更快进入工作状态**。我一般会在起床后做一个10分钟的冥想，进行最简单的呼吸冥想。即坐在椅子上，开始感受自己的呼吸，让大脑尽快进入良好状态。
