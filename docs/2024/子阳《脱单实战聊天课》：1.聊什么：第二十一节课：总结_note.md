# 子阳《脱单实战聊天课》：1.聊什么：第二十一节课：总结

## 概述
在本节课中，我们将总结之前所学的聊天技巧和关系定位方法。

## 聊天的三大目的
1. **互相了解**：通过聊天了解对方的背景、兴趣和价值观。
2. **培养感情**：通过共同话题和互动增进彼此的感情。
3. **达成邀约**：将聊天转化为线下见面。

## 五步定位法
1. **陌生人**：交换姓名，打招呼，交换基本信息。
2. **朋友**：聊感性话题，避免原则性争执。
3. **网友**：逐步试探，从普通朋友到暧昧关系。
4. **好朋友**：进一步深化关系，增加互动。
5. **暧昧关系**：逐步推进关系，直至确定关系。

## 注意事项
- **关系进展**：每一步关系都不能跳过，即使进展速度可以加快。
- **避免快速进展**：避免因冲动而跳过关系步骤，导致关系不稳定。

## 总结
本节课中，我们学习了聊天的三大目的和五步定位法，强调了关系进展的重要性。通过掌握这些技巧，我们可以更好地与异性建立联系，并最终达成邀约。
