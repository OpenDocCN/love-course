# 老白《约会达人》情场3.0：第五课 - 缺少步骤会带来什么结果？🚫

在本节课中，我们将要学习恋爱关系发展过程中，如果跳过或做错了某个关键步骤，会导致什么样的后果。我们将通过具体案例，结合上节课介绍的“恋爱六个阶段（LS6）”理论，来分析问题根源，帮助你诊断并解决自身的情感困惑。

---

上一节我们介绍了恋爱关系发展的六个阶段理论。本节中，我们来看看如果在实践中缺失了其中某个步骤，具体会产生哪些问题。

我们经常在和女生关系彻底结束后，反复思考自己到底错在哪里。百思不得其解。去询问女生时，对方通常不会直接告知，或者只会委婉地表达。

有时对女生而言，这也只是一种感觉。她们也说不清你具体哪里不对，只是感觉不对，因此不想继续在一起。下面我们分析几个案例，尝试用LS6理论来解释。

分析导致这种结果的原因，在于缺乏其中某个步骤、做错了某个步骤，或步骤间顺序错误会导致什么结果。未来你可以基于现有问题，判断是哪个步骤没做好导致的。然后我们就能针对性地解决问题。

以下是几个典型的错误案例及其分析：

**一、社交软件中的直男思维**
*   案例：在社交软件上，你对女生说“你好”，女生回复“你好”。你接着说“约吗？”，女生回复“滚”。这是多数人会犯的错误，也是女生在社交软件资料里明确写明“不约”的原因。或者，你说“好近啊”，女生说“是啊”。你紧接着说“我们一起去吃点东西/看个电影/来我家看剧/喝一杯吧”。女生回复“叔叔我们不约”。
*   问题分析：这里的行为是直接从 **认识（L1）** 越级跳到了 **追求/诱惑（L4/L5）**，完全缺乏了 **熟悉（L2）** 和 **吸引（L3）** 的中间过程。

**二、错误表白与浪漫追求**
*   案例：在普通的同学、同事关系中，平时交流不多，在对方眼里你们只是普通朋友，即关系处于 **熟悉（L2）** 阶段。有一天你发了很长的表白信息，对方告诉你“我不是你想象的那种人，我有很多缺点，你了解后就不会喜欢我了，你会遇到更好的”，或者直接拒绝你。你开始展开公式化追求：送礼物、买东西、不停邀约，模仿电视剧中的浪漫桥段。女生因感到压力而拒绝，你认为她可能觉得你不是真心的，于是变本加厉地对她好，想证明自己的真心。最后女生把话挑明，关系变得尴尬，她开始躲着你。
*   问题分析：这里是把 **吸引（L3）** 和 **追求（L4）** 的阶段搞反了。顺序应该是先有吸引，再有追求。追求女生不是先让她知道你可靠，而是先让她知道你是“猎人”（有吸引力）。如果对你没有兴趣，可靠与否根本不重要。如果没有吸引力，对方根本不在乎你是否靠谱。

**三、陷入友谊区**
*   案例：你们是无话不谈的好朋友，可能是同学、朋友或关系特别好的同事。彼此非常了解，相处了很长时间。你知道自己喜欢对方，但一直以朋友身份相处，害怕表白后连朋友都做不成。这看起来与上一个例子相反，但同样不正确。因为你脸皮薄，你们最终进入了“友谊区”，对方在你身上再也找不到恋人的感觉。如果尝试暧昧，两人都会觉得别扭。
*   问题分析：这是因为关系长期停留在了 **认识（L1）、熟悉（L2）、吸引（L3）** 阶段，只有L1-L3，而没有推进到 **追求（L4）、诱惑（L5）、长期交往（L6）**。

**四、社交达人的烦恼**
*   案例：由于机缘巧合与一个女生发生了一夜情。事后你喜欢上她，但女生说“那天我喝多了/心情不好，我们不可能，你找个适合你的人”。也许那天的灯光、音乐氛围很好，或者你当时表现得很温柔，但关系没有后续。
*   问题分析：因为你们只有 **认识（L1）、吸引（L3）、诱惑（L5）**，跳过了 **熟悉（L2）** 和 **追求（L4）**，且后续也没有补上。在对方看来，你的行为不合理、不靠谱。你没有给她一个合理化的理由去接受这件事，也没有在后续表现中补上“可靠”这个点。很多学员初期展示面打造得好，约会时氛围和操作也得当，短暂在一起后很快分手，就是因为缺失的步骤没有补上。我在复合攻略里详细讲过这种情况。

接下来我们系统性地看看，缺失LS6中的任何一个步骤，分别会导致什么样的后果。

**认识（L1）**
如果“认识”这个窗口无法打开，这是最致命的。心态会崩溃，很容易喜欢上一个女生，很容易对她过度上心，患得患失。所有不会追女生的男生可能各有不同，但有一个共同点：**都不认识几个女生**。我记得上学时，我所在城市有个女生特别多的学校，男生中流传一句话：“我很丑，但是我很抢手”。还有一句是“科大的饭，师大的妹子，工大的狗”。这是因为有些学校女生比例远高于男生，导致那里的男生即使条件一般也很“抢手”。原因在于女生基数足够大。

**熟悉（L2）**
有效资源指的是进入熟悉阶段后的女生。如果没有达到熟悉，那就是无效资源。例如，加了微信后聊天，女生一开始就很冷淡，那么就算加100个，这100个也是无效资源。我们衡量自己认识多少女生，最终看的是有效资源数量。所以，“你有多少资源”意思就是有多少女生和你到了熟悉阶段。没到熟悉阶段的暂时不算。大家是否有过这样的体会：和一个女生网上聊得很好，但约她出来她不愿意，理由是“我们还不熟”？因为她不知道和她聊天的人是不是坏人或骗子。这就是安全感，女生需要的最基本的安全感、踏实感，就是在这里建立的。你要让她感觉你是一个不错、正常的普通人，而不是让她感到危险的人。熟悉感就是让她了解你这个人安全可靠。

**吸引（L3）**
这是重中之重。我们喜欢一个女生，往往会直接去“追”。我们看看其他生物，它们在交配前都会做一件事：**吸引异性**。比如孔雀开屏、雄鹿打架、海豚吐泡泡、小鸟搭窝，都是把自己最好的一面展现出来。就像一些小鸟，谁搭的窝好，谁就能吸引异性来它的巢穴。而我们作为最高级的人类，看上一个女生直接去“追”，就好像繁殖季节来临时，一只公麋鹿跑到母麋鹿面前说“我喜欢你”。你觉得结果会改变吗？不会。它需要做的是去和其他麋鹿打架，证明自己强壮，来吸引母麋鹿追着它。

**追求（L4）**
如果只有吸引就有用，那么世界上大部分男生都找不到异性，因为女生都会选择最有魅力的男人。为什么女生追星，却嫁给普通男生？因为那些男明星对她们来说并不现实，不可靠。在第一节课里我讲过，女生有两个底层需求：第一个是“猎人”（吸引力），第二个是“靠谱”。当你吸引她、让她喜欢你之后，她此时更看重的是你这个人靠谱不靠谱。如果发现你不靠谱，她会停止喜欢你。尽管你吸引力很强，但如果她觉得你不靠谱，认为和你在一起后你会甩了她，她也会选择尽快结束这段关系以保护自己。

**诱惑（L5）**
这个阶段很好理解。你想要一个女生结婚那天才可以进洞房吗？在现代恋爱中，没有发展到最后一步，也不算严格意义上的恋爱。没有这个阶段在初中高中很常见，但在我们成年后就比较少见了。

**长期交往（L6）**
这个不用多说，是我们每个人都必须经历的阶段。

本节课中我们一起学习了缺失恋爱六个阶段（LS6）中任一环节可能导致的后果。从社交软件的越级冒进、错误表白的压力施加，到陷入友谊区的停滞不前，以及因跳过关键步骤而导致的关系短暂，其核心都在于步骤的缺失或顺序的错乱。记住，关系的健康发展需要循序渐进，**先建立熟悉与安全感（L2），再创造吸引力（L3），然后才是真诚的追求（L4）与关系的深化（L5、L6）**。理解每一步的作用，才能有效诊断问题，并针对性修复你的情感关系路径。
