# 40、纳兰女性终身成长视频课nlnx：第22课如何面对孩子的手机成瘾（下）—亲子关系

🎼接下来我们来分享今天的最后一个部分，孩子已经这样了，具体我该怎么办？首先第一个用文字的方式沟通，为什么要这样说呢？因为我们会发现手机成瘾的孩子，大部分现在他可能跟家长的亲子关系都不好，为什么？因为他很冷漠，他天天在这玩手机，他没时间跟你交流，或者说因为他手机成瘾，我们的家长给他没收手机，控制他用的时间他也会很烦，所以亲子关系不好。

**文字沟通的优势**：
- 家长在写信时进行自我梳理，梳理担忧、改变决心和陪伴成长的愿望。
- 孩子收到信会感到被尊重，觉得妈妈很用心。
- 青春期孩子用文字交流更平和，会认真阅读。

🎼第二个具体的方法是多带孩子进行户外运动，我们会发现孩子在成长过程中精力非常旺盛，需要释放。周末孩子特别希望出去玩，但家长往往选择在家休息，这会产生矛盾。聪明的家长会选择让孩子舒服一些，带孩子去植物园、博物馆、美术馆等文化场所，或者简单地在楼下遛弯、骑自行车。

🎼第三点是要培养孩子的体育项目，因为爱运动的孩子不容易手机成瘾。爱运动的孩子没有时间玩手机，而且很少有自杀的。长期体育运动的孩子会锻炼忍耐力，不会因为困难或挫折而放弃。

🎼第四点是要让家成为一个有趣的地方，让家长自己变成一个有趣的家长。孩子对手机上瘾，但对家长不上瘾。让孩子对家长上瘾，你就成功了。让孩子在宽松的环境下长大，他就不容易对某一事物成瘾。

🎼以下是针对不同年龄段孩子的建议：
- 3岁以下：多带孩子玩游戏，转移注意力。
- 3-6岁：让孩子在幼儿园或小区里与同龄小朋友玩耍。
- 7-9岁：根据孩子的兴趣引导，成为孩子的好朋友。
- 10岁以上：引导孩子进行体育运动、观看纪录片、参与义工活动等。

🎼第五个方法是试着在孩子玩游戏时让他关掉声音，因为游戏的声音会刺激孩子，减少声音后游戏的乐趣会减少一半。

🎼第六个方法是跟孩子约法三章，根据孩子手机成瘾的程度制定规则和使用时间，规则和时间要循序渐进。

🎼第七个方法是“时停止六必须”：
- 停止吼叫、讽刺挖苦、打骂、忽视、说教、羞辱、刺激挑衅、否定感受、停留在过去、不切实际的愿望。

🎼第八个方法是“六必须”：
- 每天微笑、抱抱或拍拍孩子、说“我爱你”、有目光接触、点头、表扬孩子。

🎼最后，要慢慢来，因为习惯的形成需要时间。给孩子和父母时间，认识到孩子有反复是正常的，调整计划是完全可以的。

🎼爱比教育重要1万倍，因为爱的力量是世界上最伟大的力量。无论我们面对孩子手机成瘾、叛逆还是其他育儿问题，都要给孩子真诚的发自内心的爱。
