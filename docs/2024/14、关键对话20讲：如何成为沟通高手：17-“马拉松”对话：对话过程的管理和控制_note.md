# 关键对话20讲：如何成为沟通高手 🎤
## 第17讲：“马拉松”对话：对话过程的管理和控制 ⏳

在本节课中，我们将从时间的维度来探讨关键对话。我们将学习如何应对时间压力巨大的对话，以及如何管理那些持续时间很长的“马拉松”式对话。理解时间因素如何影响对话的进程和结果，是成为沟通高手的重要一环。

上一讲我们完成了基础课程的学习。从今天起，我们将进入更具挑战性的进阶内容。

### 时间维度带来的挑战

以前我们从场景、对手、准备工作和交涉阶段等角度分析过关键对话。本节中，我们来看看时间这个维度。我们将重点关注那些在时间上特别有挑战性的对话，并学习如何把握它们。

这种时间的挑战主要体现在两个方面：
*   时间紧、任务重，可能出现24甚至48小时连续作战的对话。
*   对话时间拖得非常长，一年两年都谈不下来。

我们可能会有一种简单化的思维，认为关键对话的基本技能和结构不变，只是时间长度有变化。这种想法是把长时间跨度或时间压力大的对话看简单了。时间维度一变，许多事情都变了。时间本身就是运动的标志，在对话过程中，各方面的因素，包括我们自身的状态，都在不断变化。当时间足够长时，许多内外部因素的变化就会发生。如果我们还用最初的思路去看问题，就难免有刻舟求剑之嫌。

能够在宽松时间内完成关键对话的对话者，也需要增加在时间压力下或长时间跨度内掌控对话的能力。就像打球，打10分钟好球和打一个赛季好球，在普通比赛里打好球和在决赛里有关键表现，这是两个不同层次的能力。

因此，我们需要好好拆解一下时间紧、任务重的对话和超长的“马拉松”式关键对话。

### 本节课的核心目标

今天这节课的目标是让大家明白：
1.  在时间压力下，如何把握对话节奏，不被带乱，不仓促成交，不在压力下妥协。
2.  如何以正确的态度，用耐心和合作的方式推动一场长时间跨度的关键对话顺利进行。
3.  认识到时间是最宝贵的资源，对于不该进行的超长对话，该离场就要离场。

### 一、时间压力下的关键对话

时间是生命中最重要的维度。在我们的生活和工作中，许多关键对话都有其时间标记。有的对话可以在相对从容的时间内完成，但很多关键对话因其复杂性，无法拥有合理宽松的时间，或不可能在短期内完成。这时，时间维度就会增加对话的复杂程度。

有一类关键对话就是在时间压力下举行的，必须在某个时间点之前谈出一个结果。

以下是几种常见的时间压力来源：

**人为设定的截止日期**
*   例如：开学前必须租到房子；建设项目开工前必须谈好合同。
*   在工作中，我们经常为交易设定日期（如签约日、交割日），必须在此之前完成规定动作，否则可能面临合同失效或违约责任。

**法律或监管设定的日期**
*   例如：上市公司涉及重大交易，信息泄露导致股价异动后，监管机构可能要求其在下一个交易日开市前发布公告，澄清是否进行该交易。这可能导致双方必须在极短时间（如48小时）内完成所有谈判。

在这样的时间压力下，人的情绪容易波动，可能出现近乎崩溃、想拍桌子走人或干脆妥协的情况。应对这种高压谈判，需要更高阶的对话把控能力、情绪控制能力以及对各方利益的清晰洞察。

### 二、马拉松式的长期对话

有些对话的时间跨度非常长。无论是在国际事务还是个人生活中，一些复杂或艰难的事情（如感情破裂后的财产分割、遗产继承）都可能导致谈判旷日持久。

以下是几个国际上的长期谈判例子：
*   **中国加入WTO**：谈判历时15年（1986-2001年）。
*   **英国脱欧**：主要谈判持续了4年多（2016-2020年），至今仍有遗留问题。
*   **朝鲜停战谈判**：谈了两年多（1951-1953年），几乎与战争同步。
*   **中俄边界谈判**：跨度超过40年（1964-2008年）。
*   **巴黎气候协定**：谈了将近5年。

在工作中，投资并购、商务谈判或争议解决等对话，持续一两年也很常见。

一场对话时间特别长，可能由不同因素造成：
*   **议题特别复杂**。
*   **参与方过多**，需要单独沟通的事情太多。
*   **工作风格造成**，例如有的公司代表每月只来谈两天，导致一个合资项目能谈上一年。

当然，我们并非提倡闪电式成交。我们在核心理念中讲过，要避免仓促成交，应清晰自己的目标并为之耐心努力。对话需要一定时间，多谈几轮往往是更理性的选择。同时，也不应把对话拖得特别长，一个合适的时间尺度是双方应共同追求的。

### 三、应对时间挑战的核心态度与策略

在第九课我们讲过，要做一个乐观有耐心的对话者。面对时间压力或超长对话，耐心尤为重要。耐心源于认识到世界的复杂性，理解每个人、每个机构都有其特点和不同视角。认识到这一点，你就不会急于强加自己的观点给别人。

以下是应对时间挑战的几个核心策略：

**1. 识别真假时间压力，警惕谈判伎俩**
我们要分清是否真的有必要进行马拉松式谈判。有些时间压力是人为制造的，例如一方声称“今天不谈好就不结束”，或谎称自己次日必须离开，以此制造紧迫感。

故意给对方制造时间压力，逼迫其在紧迫下做出选择，是一种谈判伎俩。还有一种叫“最后一分钟”策略，即在一切谈妥后，突然提出额外要求（如再降价）。这时，对方可能因前期投入巨大而陷入两难。

对于这种不良手段，我们应：
*   **提前做好心理建设**，预判对方可能使用此类手段。
*   **坚守立场**，对于不合理的突然袭击，可以说：“我不接受这种最后一分钟的要求，这不是好的谈话方式。如果你坚持，我们就不谈了。”
*   **始终回到根本利益上**，判断哪些点才是最重要的。

**2. 建立合作共赢的“同事”关系**
克服时间压力的一个重要态度，是把对话对手视为共同解决问题的“同事”。本着合作的心态一起工作，相互配合，能有效提高对话效率，减轻时间压力带来的精神消耗。

真正的谈话高手可以放下对“正确”的固执坚持，关注事情本身。这回到了第三课讲过的“向外扫射的雷达系统”。双方作为队友，集思广益，共同寻找解决方法。这样的对话，即使时间长，过程也是专注、有效率的。

**3. 善用对话间歇期**
在多轮对话的间隙，对话并未停止。只要还有下一轮，我们就应将其视为持续进程的一部分。

间歇期是施展能力、更新策略的关键时期。就像足球比赛的中场休息、篮球比赛的暂停、F1赛车的进站，都是战术组成部分。

在间歇期，我们应该：
*   **更新对手识别、场景识别和策略推演**。
*   **主动与对手保持良好沟通**，例如发信息问候、在社交媒体互动等。维持友好的关系对推进对话至关重要，而间歇期正是轻松建立日常联系的好时机。

### 四、评估时间价值，学会适时离场

我们的时间都是特别宝贵的。如果你预见到这将是一场非常耗时的对话，请评估是否要参与。

关键对话何时可以离场，也是一项关键能力。我们需要评估：不谈的结果，与继续谈下去（或调整目标后）的结果，哪个利益更大？这时就要用到 **BATNA** 的概念。

我们常常意识不到时间的宝贵，花费大量时间在琐事上。例如，为了省2块钱手续费，宁愿花15分钟过街取钱。如果用逆向思维思考：“如果有人给你2块钱，让你跑这15分钟，你愿意吗？”答案往往是否定的。

一场关键对话何时该离场？时间因素是一个重要考量。当我们花费的时间可能已超出所能得到的利益时，就该考虑离场。问题的关键在于，有时思维局限让我们无法认清真正的利益所在。

事情的价值一定要加上时间的维度再评估。对于已经发现不划算的事情，最好的结束时间就是现在。因为过去的时间无法追回，及时“割肉”优于等待或幻想回本。

### 总结与练习

本节课中，我们一起学习了如何在时间的维度下管理关键对话。

**核心要点总结：**
1.  **识别时间挑战**：对话可能面临时间紧、任务重的压力，也可能是长时间的马拉松式多轮对话。我们需要提前预判并做好相应准备。
2.  **保持核心态度**：
    *   **做乐观有耐心的沟通者**：理解世界的复杂性，避免仓促成交。
    *   **建立合作共赢的“同事”关系**：与对手共同解决问题，提高效率，减少内耗。
3.  **善用间歇，持续准备**：在多轮对话的间隙，积极更新策略，并与对手保持良好关系。
4.  **珍视时间，敢于离场**：时间是宝贵资源。在投入耗时对话前需三思，一旦发现得不偿失，应果断离场，避免陷入“沉没成本”误区。

**课后练习：**
1.  **反思最长对话**：回想你参加过的时间最长的对话（会议、谈判或私人讨论）。你当时是否预见到了它的长度？如果预见到了，你做了哪些准备？现在回看，你觉得应该做哪些准备？
2.  **识别低价值对话**：列出你是否参加过那些根本不值得花时间的对话，花费了大量精力却所得甚少。分享这样的经历，思考是哪个思维盲点导致了这种决策。
