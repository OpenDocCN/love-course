# 凯哥《3899私教系列》：2.情商提升3.0：006.情商3.0课程-吸引的核心

## 概述
在本节课中，我们将学习如何提升吸引力，特别是如何通过展示面和聊天技巧来吸引对方。

## 步骤一：激发回复欲望
**核心概念**：激发对方的回复欲望等同于激发好奇。

* **错误做法**：直接表白或过度追问。
* **正确做法**：筛选开场白的思路，避免直接表白或过度追问。

## 步骤二：展示面的重要性
**核心概念**：展示面是吸引力的基础。

* **错误做法**：忽视展示面，只依赖聊天技巧。
* **正确做法**：做好展示面，包括朋友圈、形象改造等。

## 步骤三：一致性
**核心概念**：展示面和聊天要保持一致性。

* **错误做法**：展示面和聊天不一致，给人留下不稳定的印象。
* **正确做法**：保持展示面和聊天的一致性，给人留下稳定的印象。

## 步骤四：避免成为“矬男”
**核心概念**：避免在聊天中表现出“矬男”特质。

* **错误做法**：见一个爱一个、说话没有逻辑等。
* **正确做法**：保持自信、有主见，避免在聊天中表现出“矬男”特质。

## 步骤五：情绪投入
**核心概念**：通过情绪投入激发好奇。

* **错误做法**：过度依赖表情包或恭维。
* **正确做法**：在适当的时候使用表情包或恭维，但不要过度依赖。

## 步骤六：需求感
**核心概念**：避免暴露需求感。

* **错误做法**：过度追求对方，让对方感到压力。
* **正确做法**：保持自信，避免过度追求对方。

## 步骤七：个性化冲突
**核心概念**：通过个性化冲突吸引对方。

* **错误做法**：与周围人没有反差，缺乏吸引力。
* **正确做法**：与周围人形成反差，展现独特的个性。

## 步骤八：反差感
**核心概念**：通过反差感吸引对方。

* **错误做法**：与周围人没有反差，缺乏吸引力。
* **正确做法**：与周围人形成反差，展现独特的个性。

## 总结
本节课中，我们学习了如何提升吸引力，特别是如何通过展示面和聊天技巧来吸引对方。记住，吸引力是一个综合性的概念，需要我们在各个方面都做到位。
