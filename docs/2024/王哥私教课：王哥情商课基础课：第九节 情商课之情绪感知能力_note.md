# 王哥私教课：情商课基础课 第九节 🧠 情绪感知能力

在本节课中，我们将要学习情商的一个核心组成部分：情绪感知能力。我们将探讨它的定义、重要性，并通过具体例子来理解如何在实际生活中运用这项能力。

你是否曾在工作、感情或生活中，因为说错话或做错事，导致与朋友的聚会、约会或某件事的结果非常糟糕？我相信有人经历过。

这听起来可能有些宽泛。让我们举一个更确切的例子：你是否曾被告知“你得罪人了”或“你的人缘不好”？或者，你是否曾被评价为“不懂社交”，例如在他人需要安慰（如女性生理期时）却不知如何回应？当别人向你指出这些时，你是否想过，为什么这些问题总是发生在你身上，而不是别人身上？

许多人的答案可能是：“我的情商不够”。那么，情商不够具体指什么呢？这里给出一个确切的答案：**情绪的感知能力不行**。

## 什么是情绪感知能力？

所谓情绪感知能力不行，我们常能看到一种现象：面对同一件事、同一个人用同样的语气讲述，两个不同的人在听取并转达时，会得出不一样的结果。你相信吗？至少我的经历是这样的。

举个例子，王哥早年生活在农村时，常听父亲说起一句话：“芝麻大的事儿，从村东头传到村西头，就变成了跟大象一样大的事儿。”父亲常以此嘲笑村里人喜欢以讹传讹。但这真的只是以讹传讹吗？我认为未必，这本质上就是**情绪感知能力**的缺失。

## 情绪感知能力的实际案例

为了更具体地说明，我来分享一个亲身经历。在我从事现在这个行业之前，曾在一家企业上班。我们有一位女领导，她情绪特别不稳定，所有同事都很害怕她，唯独我与她相处得特别好。甚至后来这位女领导还对我表示过好感，但由于年龄差距（我当时二十三四岁，她已三十三四岁，且因工作压力显老），我并未接受。

为什么这位女领导会赏识和喜欢我呢？因为我的**情绪感知能力**。具体表现是什么？这位女领导因为相貌出众，常被更大的领导叫去应酬。应酬完第二天，宿醉后身体必然很不舒服。其他同事去汇报工作时，都只专注于讲工作本身。而我则不同，我的工作甚至可能没做好，但她从不责备我。

**我的做法是**：第二天，我会为她泡一杯茶，或者至少倒一杯温水送过去。你感受到了吗？这就是情绪感知能力。她当时不一定是对工作不满意，而是**心情不好，需要安慰和关心**。她需要的不是下属汇报这天的工作——毕竟大家都是在为老板打工，完成分内事而已——而是在工作之外，有人能关心她因应酬醉酒后的难受，体谅她一个人在陌生城市的孤独。当一个讲话温柔（至少上班时是）的男下属适时递上一杯热水，效果自然不同。我明白当时应该这样做。

因此，每次我这样做（倒茶倒水是在她办公室内完成，其他同事看不到），甚至让其他同事后来误以为我和女领导有特殊关系。其实哪有什么特殊关系，这只是一个**情绪感知能力**的体现。大家总想着工作的事、常规的事，却从未站在她的角度思考：“此刻她最需要什么？”答案显然是**关心**。

## 情绪感知能力的重要性

当你拥有良好的情绪感知能力时，这就是别人所说的**高情商**。不要认为那些讲话侃侃而谈、口才好的人就一定情商高。记住，在一场关于某事的讨论或辩论中，最厉害的那个绝不是说得最多的人，一定是**懂得倾听**的人。

这里的“懂得倾听”不是指用耳朵听——谁都会听——而是指**听完之后会思考对方情绪和状态**的人。这就叫情绪感知能力。

## 如何培养情绪感知能力？

我希望你听完这节课后，从今天开始，在与他人沟通讲话时，先不要急于关注对方说话的内容。

**你应该先关注**：他讲这句话时的精神状态和情绪。这是一个很微妙的东西。先关注这一点。

然后，以这种感知到的情绪状态为出发点，思考：“如果是我本人遇到同样的事，我会希望别人怎么做？”从这个角度出发，再结合对方谈话的具体内容去回应他。

你会发现，这样往往会得到一个非常完美的效果。这就是所谓的**情商，即情绪感知能力**。
