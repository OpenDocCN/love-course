# 戎震《魅力提升课程》：戎震魅力提升课程7 - 营造亲密氛围

## 概述
在本节课中，我们将学习如何通过营造亲密氛围来拉近与对方的距离，快速创造有效环境，增加友好度。

## 营造亲密氛围
**以下是营造亲密氛围的步骤：**

1. **创造友好氛围**：在约会或社交场合，让对方感到愉悦。这可以通过聆听、点头、微笑等方式实现。
2. **以对方为中心**：聊天时，始终以对方为主体，围绕对方的兴趣、擅长领域、家乡、家人等进行交谈。
3. **引导对方说话**：避免过多自我表达，而是引导对方讲述自己的故事和经历。
4. **避免质问和反问**：不要过多使用质问句和反问句，以免显得心虚。
5. **创造轻松愉快的氛围**：避免使用质问句和反问句，不要过分展示自己的智慧和知识。

## 核心概念
- **友好氛围**：通过聆听、点头、微笑等方式创造愉悦的氛围。
- **以对方为中心**：围绕对方的兴趣、擅长领域、家乡、家人等进行交谈。
- **引导对方说话**：避免过多自我表达，而是引导对方讲述自己的故事和经历。

## 总结
本节课中，我们一起学习了如何通过营造亲密氛围来拉近与对方的距离，增加友好度。记住，聊天时要以对方为中心，引导对方说话，创造轻松愉快的氛围。
