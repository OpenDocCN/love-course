# 🌟 Python基础教程 1.1：变量与数据类型

## 概述

在本节课中，我们将学习Python中的变量和数据类型。变量是存储数据的容器，而数据类型则定义了数据的种类和操作方式。

## 变量

变量是用于存储数据的标识符。在Python中，变量不需要声明类型，直接赋值即可。

```python
x = 10
y = "Hello, World!"
```

## 数据类型

Python中有多种数据类型，包括数字、字符串、列表、元组、字典和集合。

### 数字

数字用于表示数值。Python支持整数（int）、浮点数（float）和复数（complex）。

```python
a = 5  # 整数
b = 3.14  # 浮点数
c = 2 + 3j  # 复数
```

### 字符串

字符串用于表示文本。在Python中，字符串被引号包围。

```python
name = "Alice"
message = 'Hello, "World!"'
```

### 列表

列表是用于存储一系列元素的容器。列表中的元素可以是不同类型的数据。

```python
fruits = [1, "apple", 3.14, True]
```

### 元组

元组是类似于列表的容器，但不可变。

```python
coordinates = (10, 20, 30)
```

### 字典

字典是用于存储键值对的容器。

```python
person = {"name": "Alice", "age": 25}
```

### 集合

集合是用于存储不重复元素的容器。

```python
numbers = {1, 2, 3, 4, 5}
```

## 总结

本节课中，我们学习了Python中的变量和数据类型。变量是存储数据的容器，而数据类型则定义了数据的种类和操作方式。在下一节课中，我们将学习Python中的运算符和表达式。
