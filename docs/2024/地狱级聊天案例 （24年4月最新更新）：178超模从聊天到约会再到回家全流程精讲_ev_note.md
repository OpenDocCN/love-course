在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。理解它们是编写任何程序的第一步。

## 1. 什么是变量？ 📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。你可以把变量想象成一个贴有标签的盒子，用于存放数据。在Python中，创建一个变量并给它赋值非常简单。

核心操作是使用等号 `=` 进行赋值，其基本公式为：
**`变量名 = 值`**

例如，我们写一行代码：
```python
name = "小明"
```
这行代码创建了一个名为 `name` 的变量，并将文本“小明”存放了进去。之后在程序中，我们就可以通过 `name` 来使用这个值。

## 2. 基本数据类型 🔤

仅仅知道如何存放数据还不够，我们还需要了解数据的“类型”。Python中有几种基本的数据类型，它们决定了数据可以做什么操作。

以下是三种最常用的基本数据类型：

*   **整数**：没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：由文本组成的数据，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘你好’`, `“Python”`。

## 3. 查看与理解类型 🧐

在编程时，有时我们需要确认一个变量的类型。Python提供了一个内置函数 `type()` 来帮助我们。

其使用代码格式为：
**`type(变量名或值)`**

例如：
```python
age = 20
print(type(age))  # 输出：<class 'int'>，表示整数类型

price = 9.99
print(type(price)) # 输出：<class 'float'>，表示浮点数类型

greeting = “Hello”
print(type(greeting)) # 输出：<class 'str'>，表示字符串类型
```
通过 `type()` 函数，我们可以清晰地了解正在处理的数据是什么类型。

## 4. 变量命名的规则 📝

给变量起名字需要遵循一些简单的规则，这能让你的代码更规范、易读。

以下是命名变量时需要遵守的几个关键规则：

*   名字只能包含字母、数字和下划线。
*   名字不能以数字开头。
*   名字是区分大小写的（`myVar` 和 `myvar` 是两个不同的变量）。
*   不能使用Python的保留关键字（如 `if`, `for`, `while`）作为变量名。

本节课中我们一起学习了Python编程的基石。我们了解了**变量**就像一个存放数据的盒子，并通过 `变量名 = 值` 的公式给它赋值。我们认识了三种**基本数据类型**：整数、浮点数和字符串。我们还学会了使用 `type()` 函数来检查数据类型，并掌握了变量命名的基本规则。牢记这些概念，是开启你Python编程之旅的关键第一步。
