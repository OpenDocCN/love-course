# 清风老师合集：06撒娇2.0方法篇13期：063.第五课：夸奖式撒娇，让他对你主动_note

在本节课中，我们将要学习一种强大的沟通技巧——夸奖式撒娇。通过正确、具体的夸奖，你可以强化伴侣的正确行为，让他更愿意为你和家庭付出，从而改善关系，让他对你越来越主动。

## 概述：夸奖的魔力 ✨

夸奖之所以有巨大魔力，是因为男人在与女人相处时，需要不断得到正向鼓励和反馈，才肯继续行动。他们害怕自己的行为不被喜欢。夸奖本质上是一种**正向强化**工具。

在之前的课程中，我们分享过学员通过正向强化，让原本从不付出的前夫一次性转账20万的案例。这之所以能做到，是因为夸奖让对方感受到了被看见和被爱。特别是对于常年强势的女性，一句恰到好处的夸奖，威力巨大。

## 夸奖的最大误区 🚫

以下是女人在夸奖时最担心的一点：夸奖会让男人飘起来、不尊重自己或去找第三者。

请放心，越是懂得夸奖的人，获得的尊重反而越多。这就像老师夸奖学生，学生会更尊重老师一样。遇到冲突时，我们本能地想争吵，这是一种动物性反应。但夸奖是在强化对方正确的行为，效果更持久。

## 夸奖的核心原则 🎯

夸奖的核心是：**夸奖对方的行为或正确的态度，而非空洞的赞美**。

错误的夸奖示例：`“我老公是全天下最棒的男人。”`
这种夸奖可能导致对方去别处验证自己是否“最棒”，反而可能让他离开你。

正确的夸奖原则是：夸奖那些**他可以通过自己行动获得具体反馈**的事物。
例如：`“老公，你今天把地拖得好干净，我觉得我有一个很愿意为家庭承担责任的老公。”`
这样的夸奖会让他越做越好。

## 六种有效的夸奖方式 📝

接下来，我们介绍六种具体的夸奖方式，结合撒娇的语气和眼神，效果更佳。

### 1. 夸奖态度

当对方，特别是习惯冷暴力的伴侣，愿意沟通时，要及时夸奖他的态度。

*   `“老公，你今天跟我沟通的态度真的好棒。”`
*   `“老公，感觉你今天说话特别温柔/温暖，我听着好舒服。”`

即使他嘴上否认（如说“你瞎说”），但他的嘴角可能在笑。要观察他的肢体语言，身体往往比嘴巴更诚实。

### 2. 夸奖责任

当对方为家庭付出时，例如上交工资卡或转账，要夸奖他的责任感。

*   `“老公，你对我们这个家真的好负责呀。”`
*   `“每次收到老公的转账，我就觉得我的老公真的是一个很爱家、很有责任感的男人。”`

这样的夸奖会让“为家庭付出”在他的潜意识里与“快乐”产生正向关联，从而更愿意投入。

### 3. 夸奖担当

当对方为你或家庭做出牺牲或改变时，要夸奖他的担当。

*   `“老公，你真的是一个很有担当的男人。”`
*   例如他放弃了与朋友的聚会选择陪你，这时给他一个拥抱并说：`“哇，老公说不去就不去，真是男子汉，好有担当。”`

这会让他的付出被看见，得到情绪上的回应，避免委屈积累成未来的报复行为。

### 4. 夸奖善良

当对方表现出同理心或恻隐之心时，要夸奖他的善良。

*   `“哇，我老公真的好善良，好有同情心。”`

他对外的同理心，也需要被引导到对你身上。如果他对别人比对你更好，往往是因为他在你这里付出时没有获得意义感（如夸奖）。这时不要打击他，而是找到他的闪光点去夸奖。

### 5. 夸奖具体行动

对他做的具体家务或事情，要大力夸奖。

*   `“老公，这个地拖得真干净，比你脑门都亮！”`
*   `“老公，你做的饭真的太好吃了！”`

**案例**：一位姐妹的富二代男友第一次下厨做了蛋炒饭，但蛋和饭是分离的。她夸奖道：`“哇宝宝，我听说最高级的蛋炒饭就是蛋饭分离，你做出来了！”` 这一夸让男友开始热衷研究厨艺，越做越好。这背后的心理是：他需要自己的行动被看见和认可。

### 6. 夸奖“拒绝诱惑”（最重要）

要让对方觉得“专一”比拥有很多异性更高级。从小事开始培养这种观念。

*   当他拒绝一次工作跳槽机会或别的诱惑时，可以说：`“老公，你真的很优质。你的专注和认真让我敬佩。这个时代好多人都经不起诱惑，但我发现你是经得起考验的。”`

这会让“专一”在他心中产生一种神圣感和高级感。你可以转述这样的观点：`“我很认可的一位老师说，专一才能体验到感情中最深入美好的部分，而不是重复浅层的行为。我觉得我的老公也是这样的人，嫁给你真的很幸福。”`

当他在外面面临诱惑时，心里想起你的这句夸奖，就可能帮助他抵御那次诱惑，从而避免后续一系列问题的发生。

## 总结 📚

本节课我们一起学习了夸奖式撒娇的核心心法与六种具体操作方式。

记住，夸奖是强化对方正确行为的强大工具。关键在于夸奖**具体的行为和态度**，而非空洞的奉承。通过夸奖他的**态度、责任、担当、善良、具体行动以及拒绝诱惑的品格**，你可以有效地引导他更爱你、更愿意为你付出。

运用好夸奖式撒娇，你就能成为一个真正好命又高级的女人。
