# 清风老师合集：06撒娇2.0方法篇13期：063.第五课：夸奖式撒娇，让他对你主动

## 概述
在本节课中，我们将学习如何通过夸奖式撒娇，让伴侣更加爱你，并对你更好。

## 夸奖式撒娇的魔力
**公式**：正向强化 = 正向鼓励 + 反馈

男人在相处过程中需要得到正向鼓励和反馈，才能继续付出。夸奖式撒娇正是这种正向强化的体现。

## 夸奖的误区
**误区**：夸奖会让男人飘、不尊重你。

**事实**：懂得夸奖的人获得的尊重更多。

## 夸奖的原则
- 夸奖对方的行为或态度，而非夸大其词。
- 夸奖具体可反馈的事物。

## 夸奖的方式
1. **夸态度**：夸赞对方沟通态度、温柔、温暖等。
2. **夸责任**：夸赞对方对家庭的责任感。
3. **夸担当**：夸赞对方为家庭做出的牺牲和改变。
4. **夸善良**：夸赞对方内心的善良和同情心。
5. **夸行动**：夸赞对方的具体行动，如家务、做饭等。
6. **拒绝诱惑**：夸赞对方拒绝诱惑，保持专一。

## 案例分析
- **案例一**：学员通过正向强化，让前夫转了20W。
- **案例二**：学员夸赞老公拒绝跳槽，获得老公的尊重和爱。

## 总结
本节课中，我们学习了夸奖式撒娇的技巧和原则。通过夸奖，我们可以让伴侣更加爱你，并对你更好。希望姐妹们都能运用好这些技巧，成为好命又高级的女人。
