# 16、海蓝老师情绪沟通管理（线上视频课）：012-买前必看_note

在本节课中，我们将学习如何观看已购买的“海蓝老师情绪沟通管理”线上视频课程。本课程为虚拟数字产品，请注意其使用方式与实体书籍不同。

## 课程观看指南 🎬

上一节我们明确了课程的商品属性，本节中我们来看看具体的观看步骤。您需要在抖音APP内完成课程的学习。

以下是具体的操作步骤列表：

1.  打开手机上的抖音APP。
2.  点击右下角的【我】，进入个人主页。
3.  在个人主页的右上方，找到并点击【抖音商城】。
4.  进入商城后，点击左上方的【我的订单】。
5.  在订单页面，点击【全部】以查看所有订单。
6.  在订单列表中找到您购买的“海蓝老师情绪沟通管理”课程。
7.  点击该订单下的【去学习】按钮，即可开始观看视频课程。
