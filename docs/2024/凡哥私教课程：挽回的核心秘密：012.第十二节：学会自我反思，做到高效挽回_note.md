# 凡哥私教课程：挽回的核心秘密 012.第十二节：学会自我反思，做到高效挽回

🎼本节课将探讨一个关键理论——自我反思。🎼有些想要挽回的人，在支付咨询费后，会将所有愿望强加给情感导师，并表现出急切的态度。

🎼这种行为在心理学上被称为退行。成年人的行为是理性的，有容忍度和谦让，但退行时则变得像孩子。🎼即使导师像魔术师一样将对象变回，这种退行行为也会让对方难以忍受。🎼因此，我们需要克服退行行为，采取自我反思的策略。

🎼自我反思意味着从人生经验和教训中学习。那么，为什么遇到这种情况会变得急切呢？🎼在中国，如果一个人没有自我反思的能力，他无法成长，无法活到现在。只是当情况发生巨大变化时，他忘记了这种能力，变得急切，忘记了自我反思。

🎼因此，我们首先要恢复自我反思的频率。情感导师会帮助你进行自我反思，并引导你提高效率。🎼以下是提高自我反思效率的三个维度：

1. **恩情维度**：你们的关系中，对方是否对你有恩？
2. **爱情维度**：在这个过程中，你是否让对方感到甜蜜，失去了信心和兴趣？
3. **交情维度**：作为朋友，你是否对得起对方，是否讲究？

🎼以下是基于这三个维度的问题，请自行回答：

1. 如何用最简短的描述呈现你们的关系？
2. 在这段关系中，你付出了什么，得到了什么？
3. 对方在这段关系中得到了什么，付出了什么？
4. 在与对方打交道的过程中，你是否有过私心或算计？
5. 你是否有过情绪失控的情况？
6. 当对方出现人性的一部分时，你有多大的包容和接纳？

🎼通过这些问题，你可以进行自我反思。🎼自我反思的目的是将自己放在时间轴上，回顾过去几年的感情，客观地面对这段关系。🎼通过拉长时间轴，你可以明心见性，看到自己和对方，心理学上称之为“看见即意味着改变”。
