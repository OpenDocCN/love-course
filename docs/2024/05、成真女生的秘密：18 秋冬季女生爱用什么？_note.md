# 课程05：成真女生的秘密：18 秋冬季女生爱用什么？🧴

在本节课中，我们将要学习秋冬季女生护肤流程中的最后一个关键步骤——面霜。我们将了解它的作用、质地、使用顺序以及它在基础护肤中的定位。

上一节我们介绍了乳液，本节中我们来看看护肤流程中的最后一步：面霜。

面霜的作用与乳液相似，核心功能是**保湿**。它们都含有基本的动物油脂或人工合成的油脂成分。面霜的保湿效果通常比乳液更强。

面霜的质地比乳液更加厚重。在秋冬季节，部分女生会在涂抹乳液之后，再涂抹一层面霜，目的是为了获得更好的保湿效果。

对于最基础的护肤流程，通常只包含化妆水和乳液两样产品。

以下是基础护肤步骤的顺序：
1.  首先使用化妆水。
2.  然后使用乳液。

本节课中我们一起学习了面霜在秋冬季护肤中的角色。我们了解到面霜的核心作用是**保湿**，其质地比乳液厚重，保湿效果也更强。在基础护肤中，正确的使用顺序是先化妆水，后乳液。
