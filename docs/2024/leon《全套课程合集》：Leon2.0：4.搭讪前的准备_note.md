# Leon《全套课程合集》：Leon2.0：4.搭讪前的准备 🤝

在本节课中，我们将学习如何在搭讪前做好充分的准备，以便更加自信和有效地进行交流。

## 1. 仪容仪表的重要性

**公式**：良好的仪容仪表 = 良好的第一印象 + 增强自信

首先，确保你的仪容仪表整洁。保持良好的发型，清洁面部，穿着合身的衣服。这些细节可以帮助你给对方留下良好的第一印象，并增强自己的自信心。

## 2. 心态调整

**公式**：正确的心态 = 减少焦虑 + 提高成功率

很多人在搭讪前会感到焦虑和不安。要明白，搭讪是一种给予对方价值的行为。你通过真诚的赞美和交流，为对方提供了一种积极的情绪体验。

## 3. 理解女生的需求

**公式**：女生的需求 = 被认可 + 认识新朋友

女生打扮得漂亮，是为了得到别人的认可。你的搭讪行为正是对她们价值的认可，同时也是她们认识新朋友的一种途径。

## 4. 预判和应对

**公式**：预判 = 提前了解 + 减少焦虑

在搭讪前，观察女生的打扮、气质和举止，预判她们可能的活动和状态。这样可以帮助你更好地应对可能出现的情况，减少焦虑感。

## 5. 任务与总结

**任务**：

1. 在每次搭讪前，在心里默念以下三个点：
   - 我的搭讪行为是有价值的，我在提供价值给面前的女人，她是希望被我搭讪的。
   - 女人她需要一个男人，她很孤独，她很被动，她平常没有什么朋友，她需要被我搭讪，我会去搭讪她她会很开心的。
   - 如果她出现了不好的反应，那不是因为我，是因为她的经验过往被现，乱七八糟的人给搭讪过销售我，或者说尾尾随尾随过。所以她，如就算如果他都有出现不好的反应，不是因为我是因为他的经验，那么出现了这些反应怎么办呢？我去打消他的顾虑，我去化解这些情绪上的不是我去建立舒适，但不是因为我所以这三个点，每次搭讪前，可以在心里默念一遍，知道养成习惯啊。

2. 养成观察和预判的习惯，猜测女生现在在干嘛？接下来要干嘛？她平常的生活状态是怎么样子？她是谁，他大概是个怎么样的人。大概几岁啊，是不是经常有男人搭着他他平常他们觉得他的圈子怎么样，通过观察他的面相，觉得他是他是一个缺乏安全感的人，还是一个比较自信的种活泼的人，内向的人面前的人等等，进行一个猜测。

**总结**：

本节课中，我们学习了如何在搭讪前做好充分的准备。通过调整心态、理解女生的需求、预判和应对可能出现的情况，我们可以减少焦虑感，提高搭讪的成功率。希望这些内容能帮助你更好地进行社交。
