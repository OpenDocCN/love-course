# 富山山女性成长课程：18_得到付出如何正确给予情绪价值 🌟

## 概述
在本节课中，我们将探讨如何在人际关系中正确给予情绪价值，特别是在得到对方付出之后。

## 适当赠送小礼物
**公式**：每次收到大礼物时，送出价值为10分之1的小礼物。

**代码**：
```markdown
每次收到大礼物 -> 送出价值为10分之1的小礼物
```

## 引导对方付出
**公式**：在表达需求时，先提出一个大件，然后引导对方付出。

**代码**：
```markdown
提出大件需求 -> 引导对方付出
```

## 提供情绪价值
**公式**：在对方付出后，提供足量的情绪价值。

**代码**：
```markdown
对方付出 -> 提供情绪价值
```

## 保留空间
**公式**：在引导对方付出时，保留一定的空间。

**代码**：
```markdown
引导对方付出 -> 保留空间
```

## 感激与回报
**公式**：在得到物质宠爱后，表达感激并回报。

**代码**：
```markdown
得到物质宠爱 -> 表达感激 -> 回报
```

## 举例说明
以下是具体操作步骤：

1. **提出需求**：向对方提出一个大件需求，如钻戒。
2. **引导付出**：在对方付出后，表达感激并回报。
3. **提供情绪价值**：通过写信、送花等方式，让对方感受到你的用心。
4. **展示价值**：将得到的物质与爱画等号，让对方知道你的感激之情。

## 总结
本节课中，我们一起学习了如何在人际关系中正确给予情绪价值。希望这些方法能帮助你更好地处理人际关系，获得更多的幸福。
