# 17、海蓝老师情绪梳理个案课：035-查不出病因的病人第四课完成版_note

在本节课中，我们将跟随海蓝老师的引导，学习如何通过情绪梳理技术，处理因童年重大创伤（如亲人遭遇车祸）而积压的恐惧、无助与悲伤情绪，并学习为内心注入新的积极品质。我们将通过一个完整的个案对话，展示从识别情绪、释放情绪到转化内在资源的全过程。

## 概述：从创伤中释放与重建

上一节我们探讨了如何识别情绪在身体上的反应。本节中，我们来看看如何具体引导来访者卸载这些负面情绪，并用积极的品质重新滋养内心。

## 第一步：识别与定位情绪

首先，需要引导来访者（娜娜）识别并定位与创伤事件相关的情绪在身体上的具体位置。

以下是引导步骤：

1.  **确认情绪**：引导来访者说出与事件相关的核心感受，如无助、孤独、悲伤、害怕。
2.  **定位身体承载点**：询问这些情绪通常储存在身体的哪些部位。

> 老师引导：“孤独、恐惧、无助，在过去一般都在身体的什么位置上承载着？”
>
> 娜娜回答：“胸口、肩膀、胳膊还有大腿。”

## 第二步：卸载负面情绪

在明确了情绪的“存放地”后，下一步是引导来访者运用想象，将这些情绪从身体中清除。

以下是具体方法：

*   **选择清理方式**：来访者可以自由选择象征性的清理方式，如风、火或水。
*   **执行清理**：引导来访者想象选定的元素（如火焰）将指定部位的情绪彻底冲刷或燃烧干净。

> 老师引导：“那可不可以把它全部卸载掉？卸的方式，风啊、火啊都行。”
>
> 娜娜选择用火清理。老师随后引导：“好的，那全部把它冲掉，烧完了会告诉我。”

## 第三步：注入积极品质

清理掉负面情绪后，内心会留下空间。此时，需要引导来访者主动邀请积极的品质进入身体，并锚定在特定位置。

以下是需要邀请的品质及其锚定位置：

1.  **力量**：邀请“力量”进入身体，并锚定在**肩部以上、肩臂上**的位置。
2.  **爱**：邀请“爱”进入身体，并充满**整个胸腔**。
3.  **温暖**：邀请“温暖”进入身体，让其住在**腹部、腰部**。
4.  **勇气**：邀请“勇气”进入身体，注入**腿部和胳膊**。

引导来访者充分感受这些品质在身体内的感觉，进行稳固的锚定。

## 第四步：回顾创伤场景并评估变化

带着新注入的积极品质，引导来访者再次回顾最初的创伤场景（6岁时妈妈出车祸的画面），观察感受和身体反应的变化。

以下是评估对比：

*   **初始反应**：提到“死亡”时，心跳加快、双手发麻、全身僵硬、有窒息感。
*   **梳理后的反应**：再次看到妈妈躺在地上的画面时，情绪强度从0-10分评估，**仅剩1分**。来访者描述自己形象变得“很有力量、很大”，第一反应是“要帮助妈妈，迅速救治”，没有难过、伤心和害怕。身体**没有**僵硬、发麻等不适反应。

这个对比清晰地展示了情绪梳理带来的内在转变。

## 第五步：与内在部分对话与和解

创伤往往会在内心留下一些长期存在的“部分”，比如“焦虑”和“恐惧”。引导来访者与这些部分进行对话，理解它们的初衷，并达成和解。

以下是对话要点：

*   **与焦虑对话**：感谢它从小到大像家人一样24小时在线保护自己，理解它是因为害怕“失去”而如此警觉。同时告诉它，自己现在看到了身边的支持系统，身体也很强大，邀请它适当休息。
*   **与恐惧对话**：承认它像“大编剧”一样编造恐怖片来保护自己，源于对“生死离别”的深刻记忆。告知它自己现在会妥善照顾身体，请它放心。

通过这样的交流，来访者实现了与内在情绪的和解。

## 第六步：回顾收获与整合

在梳理结束时，引导来访者回顾整个过程，整合收获。

以下是娜娜分享的最有帮助的几点：

1.  **看到支持系统**：意识到在危急时刻，并非孤独一人，会有很多人给予帮助。
2.  **理解身体反应**：认识到在极端危机下，身体的某些保护机制（如暂时麻木）并非纯粹的痛苦。
3.  **感受到被爱**：重新看到并感受到在妈妈缺席的时期，爸爸、奶奶、姥姥、老师等人给予的爱，自己并非想象中那样孤独。
4.  **与内在和解**：通过交流，理解了“焦虑”、“恐惧”等内在部分存在的意义，并与之和解，感受到了对他们的爱。

## 总结：释放、重建与和解

本节课中，我们一起学习了处理深层创伤情绪的一个完整流程。我们从**识别并定位身体上的情绪**开始，接着**运用想象卸载这些负面负荷**，然后**主动邀请并锚定积极品质**来重建内心力量。随后，我们**回顾创伤场景以检验变化**，并最终通过**与内在部分对话**达成深刻的理解与和解。这个过程的核心在于，不抗拒情绪，而是通过看见、释放和转化，将创伤经历转化为内在资源，最终实现与自己的和解。
