# 《二月老师的核心自信》第36课：潜意识与意识 🧠

在本节课中，我们将探讨潜意识与意识的关系，理解潜意识如何影响我们的日常决策和行为，并学习如何通过增强意识来更好地掌控自己的生活。

---

上一节我们介绍了潜意识的基本概念，本节中我们来看看潜意识与意识的具体关系。

精神分析学家荣格曾提到，你的潜意识操控着你的人生，而你却称其为命运。潜意识真的有这种操控人生的力量吗？

以下是几个常见的例子，说明潜意识如何影响我们的行为：

*   **工作场景**：当出现新任务时，你只是看了一眼就选择忽略，不敢主动争取。这是因为你的潜意识害怕得不到期望的结果，从而选择了逃避。
*   **情感场景**：在大学校园里见到心仪已久的对象，你只是叹了口气便将目光移开。这是因为你的潜意识认为“我配不上他/她”或“他/她不可能喜欢我”。
*   **学习场景**：你花很久读完一本书，感觉收获满满，但当朋友问及内容时却忘得一干二净。这是因为你的潜意识告诉你“读完了就等于读懂了”，但实际上你并未真正理解。

那么，意识和潜意识之间的关系是什么呢？

意识好比是**船长**，而潜意识好比是**船本身**。当你的意识足够强大、思路清晰时，你能与“船”良好配合。但如果你的意识弱小、思路混乱，那么“船”就可能带你步入困境。

我们所有的学习都需要用到意识，也就是注意力。上节课我们举过学开车的例子：刚开始学车时，我们注意力高度集中，不敢与人交谈；熟练之后，我们可以边聊天边开车，遇到紧急情况也能本能反应。

为什么会这样？这种本能由谁管理？答案就是**潜意识**。当你熟练掌握一项技能后，它就会下沉成为习惯，由潜意识接管。

**本能、习惯、直觉和敏感度都属于潜意识的工作范畴。**

世界上有位潜能大师曾说过，**潜意识的力量比意识要强大3万倍以上**。可以说，任何潜能开发或目标的实现，都需要依靠你的潜意识。

根据科学研究，整个意识世界就像浮在水面的冰山。意识是水上部分，而潜意识是水下部分。冰山下的部分是水上部分的9倍，而潜意识的力量更是意识的3万倍。

当然，并非所有人都会受到潜意识的困扰，这取决于你意识的强度。一般来说，如果你的意识足够强大，能够随时保持觉知，那么潜意识对你的控制就越小。

潜意识既给我们力量，也会让我们举步维艰。因为潜意识是长久以来，我们依据对事物判断结果的积累所产生的本能反应。

以下是潜意识如何形成的例子：

*   **情感创伤**：你用情至深却被异性拒绝，伤心许久。这次经验会下沉到潜意识，导致下次见到心仪异性时，你再也不敢尝试。
*   **童年阴影**：你小时候被混混欺负，留下了心理阴影。下次远远看到某个混混时，即使意识上知道距离安全，身体仍会本能地紧张。

基本上，我们的信念和生活理念都存在于潜意识中。我们的注意力只能在一个时间点聚焦于一件事，不可能把所有理念都放在意识层面，否则将无法正常生活。因此，需要由潜意识来接管它们。

当你看到一个心仪的异性却不敢上前搭讪时，最终取决于你潜意识中的信念和以往的经验。所以说，潜意识比较强势。

如果你能让自己相信可以做成某件事，并确保这件事有可能发生，那么无论多困难你都能做到。相反，如果你潜意识里认为连最简单的事都做不到，那你就真的无法完成。

假设地上放了一块9米长、30厘米宽的厚木板，每个人都能从一端走到另一端，且保证不踩到边缘。但如果把这块木板放到20米高的空中，又有谁能走几米？走的时候还能听到别人说话吗？大概不能。迈出几步后，腿就会开始颤抖。尽管你想着“我不能掉下去”，但最终还是会摔下来。

为什么木板在地面时你不会掉下来，升到高空就不行了？仅仅是因为在地面时，你的潜意识觉得“我可以轻松走过去”。你相信自己能做到。但升到高空时，你的潜意识觉得“我做不到”。你的潜意识在害怕，你自己都不相信自己能做到，所以你绝对做不到。

**注意**：你的意识没有让你前进的能力。如果你的潜意识觉得你做不到，那你绝对不可能做到。但木工和瓦匠可以轻松完成这件事，因为他们认为自己能做到。在他们的潜意识里，高空和地面是一样的。

在日常生活中，自己的意志受另一种力量摆布的情况非常多。这种力量就是潜意识。就连不知道“潜意识”这个词的小学生，也能感觉到一股从内心深处打败自己理智的力量。

当我们的意志和潜意识产生冲突时，意识往往只能甘拜下风。

以下是几个例子：

*   你的意识说“我要减肥”，为何结果管不住自己的嘴？
*   你的意识说“我要存钱”，为何管不住自己花钱的手？
*   你的意识说“我要看书”，为何买了书却没翻几页就丢到一边？
*   你的意识说“不要迟到，不要熬夜”，为何结果还是像往常一样？

原因就在于你的意识被你的潜意识打败了。我们前面说过，潜意识比意识更为强势。

一般来说，当你的意识和潜意识发生冲突时，如果没有极强的意志力，意识基本都会败下阵来。我们上一节课也说过，真正操控和指导我们生活的，是我们的潜意识。因为潜意识里有我们过去二三十年人生经验的积累，那是我们默认的模式。

很多时候，我们跳跃性的思维或未经思考直接做出的决定，都是因为潜意识在做出决定。你看到一个异性不敢上前，可能你的意识没有在思考，但本能地就不敢上去。为什么？因为你的潜意识在思考、在做决定。

它根据你过去二三十年经验的积累来做决定。如果你之前认识异性多次失败，潜意识积累了这些经验，再碰到心仪对象时，它就会阻止你上前。如果你之前总是成功，那么看到一个心仪对象时，你甚至不会思考就会直接上前认识，因为潜意识积累的经验都是成功的。这就是潜意识在操控我们的生活。

我们的意识很多时候是停止的，除非在必要的时候。例如，当我们接触全新的事物、行业或工作时，意识会接管。就像学开车，从未开过时，你的意识和注意力必须高度集中。

但在我们生活中大部分时候（没有一个具体数字），我们是不思考的。我们都是通过人生的默认模式，也就是潜意识来进行生活的。能理解吗？就是用你的本能来回应很多事情。

你的人生默认模式就是这样。基本上，你在跟别人聊天时，你的默认模式是怎样的，后续的聊天方式就会按照以前的模式进行。

所以我们说，潜意识远远要比意识更强势、更强大。
