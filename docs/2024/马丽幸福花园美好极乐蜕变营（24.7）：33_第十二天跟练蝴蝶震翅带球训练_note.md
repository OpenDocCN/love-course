# 马丽幸福花园美好极乐蜕变营（24.7）：33_第十二天跟练蝴蝶震翅带球训练

🎼欢迎参加今天的练习。今天的练习重点是带球向上提的动作。建议大家在练习前已经准备好球，以免浪费额外时间。🎼

### 热身准备

🎼首先，复习昨天的动作。然后，学习新动作。🎼

#### 热身步骤

1. **抖动膝盖**：原地抖动膝盖，微弯膝盖。
2. **抖动手臂和腿部**：抖动手臂和腿部，左右各一次。
3. **抖动膝盖**：将手放在膝盖上，上下抖动，找到舒适的节奏。
4. **前后左右运动**：在瑜伽垫上做前后、左右和绕圈运动。
5. **绕圈运动**：顺时针和逆时针绕圈，热身整个部位。

### 动作练习

#### 第一个动作

1. **呼气收紧，吸气放松**：重复此动作，每组动作结束后休息。
2. **越夹越紧的动作**：呼气收紧，越收越紧，直到最紧，吸气放松。

#### 第二个动作

1. **呼气收紧，吸气放松**：重复此动作，每组动作结束后休息。
2. **加紧3秒**：呼气收紧，屏住呼吸3秒，吸气放松。
3. **加紧5秒**：呼气收紧，屏住呼吸5秒，吸气放松。
4. **加紧8秒**：呼气收紧，屏住呼吸8秒，吸气放松。

#### 第三个动作

1. **呼气收紧，吸气放松**：重复此动作，每组动作结束后休息。
2. **一步到位**：呼气收紧，一步到位，吸气放松。

### 新动作：一步到位

1. **四组练习**：每组10次。
2. **收紧放松**：呼气收紧，吸气放松，重复此动作。

### 音乐练习

1. **放松收紧**：在音乐中做放松收紧动作。
2. **挺胸哈腰**：体会肌肉的收紧和放松。

### 总结

🎼今天的练习到此结束。希望大家能够坚持练习，达到更好的效果。🎼

### 注意事项

1. 练习结束后，及时取出球，清洁并存放好。
2. 每天带球运动时间不超过1小时，以免产生副作用。
