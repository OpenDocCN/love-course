# 老白《爱上大学生》第六课：学生中最大的不公平 👔💇♂️🧴

在本节课中，我们将学习校园男生如何通过提升外在形象来增强吸引力。课程将从服装搭配、护肤基础以及发型管理三个方面展开，提供具体、可操作的建议。

## 概述

许多校园男生不擅长打理自己的形象。原因主要有两点：一是缺乏相关概念，二是不知道具体方法。一个男生可以不帅，但不能显得低俗。做好形象管理，颜值至少能提升两个档次。

## 一、服装搭配：告别“路人感” 🧥

上一节我们概述了形象的重要性，本节中我们来看看如何通过服装搭配实现蜕变。颜值很大程度上由穿衣搭配和配饰决定。

许多人的日常穿搭可能显得普通，拍照也不好看，这通常与穿搭过于随意有关。

以下是目前在校园中最受女生欢迎的日系搭配风格要点：

*   **风格核心**：日系风格注重整体色彩与舒适感，以简约为主，色调温柔，营造干净、舒服的邻家男孩感。
*   **色彩选择**：以暖色、浅色为主。可以尝试加入一些时尚、清新的元素，避免总是穿着灰、卡其、姜黄、土绿等显老气、无特色的颜色。
*   **搭配原则**：整体搭配不能太单调，也不宜过于花哨。应避开特别夸张的色调，选择浅色或素色，并搭配一丝时尚元素。

接下来是关于“叠穿”的注意事项。天气变化时，叠穿很常见，但要注意细节。

以下是叠穿的关键点：

*   **层次感**：从贴身衣物到外套，应穿出内浅外深的层次感，颜色从浅到深递进。
*   **长度**：外套内的衣服长度不能超过外套，否则会显得邋遢。
*   **裤子**：裤子不宜过长，避免盖住脚踝，以保持整体利落感，并可以拉长视觉比例。

此外，配饰能起到画龙点睛的作用。

可以搭配的配饰包括：

*   项链
*   戒指
*   背包

这些小物件能为整体风格增添点缀，避免单调。

除了日系风，另一种受欢迎的休闲穿搭风格也值得了解。

以下是该风格常见的单品与配色原则：

*   **常见单品**：休闲衬衫、套头卫衣、牛仔外套、牛仔裤、工装裤、运动外套、老爹鞋等。
*   **配色法则**：若非搭配高手，建议整体色系统一。大面积颜色不超过三种，并按**主色（60%）+辅助色（35%）+点缀色（5%）**的比例进行搭配。

以上是针对大多数男生的常规建议。如果你思想新潮，不在意他人眼光，可以尝试以下核心方法。

**核心方法**：在拼多多等平台搜索LV、巴宝莉（Burberry）、巴黎世家（Balenciaga）等国际大牌的仿版（俗称“复刻”）。这些衣服价格不贵，夏季几十元，冬季一百多元。先购买一件，挑选其中你认为最合适的三四件留下，其余退掉。

**原理**：国际大牌的设计师是顶级的，我们借鉴的是其设计美感。相比之下，国内许多杂牌的设计可能不尽如人意。此外，国外品牌官网的模特通常是普通人，这证明衣服本身好看；而国内模特往往非常帅，容易让人忽略衣服本身是否适合自己。

对多数男生而言，最有效的穿搭就是尝试大牌风格的设计。起初可能不适应，但长期坚持能极大提升你的审美能力，超越身边95%审美滞后的同龄人。

当然，如果你接受不了明显的品牌标识，按照前面讲的常规搭配方法即可。

## 二、基础护肤：清洁与保湿是关键 🧴

上一部分我们探讨了服装，现在转向护肤。男生护肤无需像女生一样复杂，做好基础清洁保湿即可。

很多男生长痘是因为不注意护理和皮肤缺水。面部清洁与护理对男生同样重要。

基础的男士护肤三件套是：

*   洗面奶
*   乳液（或面霜）
*   面膜

关于洗面奶和乳液的选择，有以下建议：

*   **有条件可选**：碧欧泉（Biotherm）水动力套装或SK-II男士系列。
*   **大众平价可选**：妮维雅（Nivea）、巴黎欧莱雅（L‘Oréal Paris）男士套装。

**注意**：不建议每次洗脸都用洗面奶，以免破坏皮肤角质层和正常油脂分泌。通常每晚睡前使用一次即可。

关于面膜的使用，推荐以下产品：

*   皇后的秘密（QUALITY FIRST）
*   曼丹（Mandom）婴儿面膜

面膜一周敷2到3次即可。如果黑头多、毛孔大，可使用科颜氏（Kiehl‘s）亚马逊白泥面膜进行清洁，一周两次，清洁后务必敷补水面膜。

个人清洁同样重要，特别是体味和口腔卫生。

需要注意的个人清洁细节包括：

*   **口腔**：早晚刷牙，口气重者可随身携带漱口水。
*   **体味**：学生党可选用CK one或万宝龙（Montblanc）等平价男士香水，出门前喷两三下即可，切忌过量。

## 三、发型与体毛管理：细节决定成败 💇♂️

最后，我们来处理“头顶大事”和其他毛发细节。发型不是随心所欲，需要结合整体风格设计。

改造发型前，应与造型师充分沟通你的穿衣风格和期望路线。选择理发店也有讲究。

关于理发店的建议：

*   **地点**：不要选择学校内的理发店，应去市区或大型连锁店。
*   **平台**：可在美团或大众点评上选择。
*   **发型师**：选择首席或店长为你设计发型。完成后务必询问日常打理方法并让他帮你拍照记录，以后在学校理发店就可按此修剪。

如果理发师推荐烫发，请谨慎。他们的目的常是推销高利润项目。其实，一款几十元的发泥，其塑形效果和持久度可能优于仅维持两个月的烫发。

男士发型打理需要三件工具：

1.  大功率吹风机
2.  发泥
3.  定型喷雾

**基本流程**：洗完头后用吹风机吹干并吹出基本造型，再用发泥抓出最终造型，最后用定型喷雾固定。（具体手法可参考相关课程内容）

鼻毛和眉毛等细节也不容忽视。

需要定期管理的毛发细节：

*   **鼻毛**：定期修剪，切勿让其长出鼻孔。可在淘宝购买鼻毛修剪器。
*   **眉毛**：大部分男生的眉毛杂乱或淡疏。理发时可请理发师修理，或去纹绣店进行微调。眉毛淡又不想纹的，可购买眉笔学习简单描画。
*   **胡须**：建议在校学生不要留胡须，干净的面容是校园主流审美。从高中毕业起就应养成刮胡子的习惯，尤其在见女生前务必保持面部清洁。
