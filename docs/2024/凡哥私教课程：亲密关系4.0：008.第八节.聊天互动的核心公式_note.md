## 聊天互动的核心公式
**公式**：女生LOI → 男生CT → 女生服从 → 奖励、升高关系；女生不服从 → 化解

## 互动的五大绝招
### 1. 冷读
**描述**：通过猜测女生的性格或特点，快速拉近彼此关系。
**应用场景**：舒适感阶段，建立在吸引和互动的基础上。

### 2. 骄傲风趣
**描述**：以自我为中心进行自夸，展示自信。
**应用场景**：基本信息交换阶段，对方对你有兴趣且有窗口时。

### 3. 角色扮演
**描述**：扮演特定角色或场景，进行对话互动。
**应用场景**：拉近关系阶段，已有基本信息交换和互动。

### 4. 推拉
**描述**：通过示好和推开，制造情绪波动，打开女生窗口。
**应用场景**：IOI较多时，平衡需求感。

### 5. 曲解
**描述**：曲解女生字面或语境背后的意思，进行新的解释。
**应用场景**：互动过程中，化解女生的防御机制。
