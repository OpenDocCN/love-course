# 高情商聊天话术 社恐变社牛：31.泡妞的9个重点

在本节课中，我们将学习如何通过提升个人魅力和社交技巧，从社交恐惧者转变为社交达人，并掌握泡妞的九个关键点。

## 第一点：展现市场价值
**公式**：有市场价值 = 周围有女性朋友 + 朋友圈活跃

- 确保周围有女性朋友，包括微信、QQ、公司等。
- 尽量选择美女，但丑女也可。
- 让女性感觉到你是受欢迎的。

## 第二点：成为领导者
**公式**：领导者 = 主动决策 + 活跃气氛 + 吸引目光

- 在生活中各方面展现领导力，如决定吃饭地点、电影类型等。
- 主动倒酒、活跃气氛、组织活动。
- 成为他人注视的焦点。

## 第三点：展现责任心
**公式**：责任心 = 完成承诺 + 孝敬父母 + 感恩他人

- 完成承诺，如按时完成代练任务。
- 孝敬父母，表达感恩之情。
- 感恩他人，如感谢老师、公司等。

## 第四点：没有企图心
**公式**：无企图心 = 顺手送礼物 + 顺手约会 + 不表白

- 礼物、约会等行为应自然，不要带有目的性。
- 不要表白，而是通过行动展现关心。

## 第五点：独立性
**公式**：独立性 = 不受他人影响 + 冷静沉着

- 不受他人情绪影响，保持冷静。
- 具有领袖气质，不受情绪波动。

## 第六点：不断社交
**公式**：社交 = 建立关系网 + 搭讪 + 帮助他人

- 不断建立关系网，包括搭讪、帮助他人。
- 交女性朋友，扩大社交圈。

## 第七点：不夸赞外貌
**公式**：不夸赞外貌 = 提高要求 + 调动情绪

- 不要过度夸赞女性外貌，提高要求。
- 调动女性情绪，让她不爽，才能带来爱。

## 第八点：有自己的框架
**公式**：框架 = 坚持原则 + 不动摇理想

- 有自己的原则和理想，不轻易被他人动摇。
- 成为有魅力的男人。

## 第九点：幽默感
**公式**：幽默感 = 有趣的生活常识 + 幽默故事

- 学习有趣的生活常识和幽默故事。
- 提升个人魅力。
