# 老白《搞钱课》：5、别妄想，做好吃苦的准备

## 概述
在本节课中，我们将探讨副业创业过程中需要付出的努力和牺牲，以及如何正确看待副业和创业的过程。

## 1. 不要对副业抱有太美好的幻想
**公式**： 副业 ≠ 轻松赚钱

*   副业并非轻松赚钱，需要投入大量时间和精力。
*   不要期望通过副业轻松获得高收入。

## 2. 做好吃苦的准备
**公式**： 成功 = 精力投入 × 时间投入

*   创业过程中，尤其是0到1的阶段，需要做好吃苦的准备。
*   创业失败率高达90%以上，不要轻视这个过程。

## 3. 太珍惜自己的人无法创业成功
**公式**： 成功 = 放弃 × 承受

*   创业过程中，需要放弃一些东西，如娱乐、社交等。
*   承受压力和孤独，才能取得成功。

## 4. 不要把副业想的太苦逼
**公式**： 副业 = 收入 × 成长

*   副业可以带来收入和成长，但需要付出努力。
*   与普通工作相比，副业更具挑战性。

## 5. 做困难但长远有利的事情
**公式**： 护城河 = 累积优势

*   做困难但长远有利的事情，积累优势，形成护城河。
*   避免投机取巧，脚踏实地。

## 6. 三天打鱼，两天晒网无法成功
**公式**： 成功 = 持续努力

*   副业需要持续努力，不能三天打鱼，两天晒网。
*   草根创业者需要顽强的生命力。

## 总结
本节课中，我们学习了副业创业过程中需要付出的努力和牺牲。要想成功，必须做好吃苦的准备，放弃一些东西，持续努力。希望这节课能帮助你更好地理解副业创业的过程。
