# 男性思维提升上帝视角1.0：011. 第十一节课：如何走出低谷 🧗♂️

在本节课中，我们将要学习如何面对人生中不可避免的低谷与失去。我们将探讨一个心理学模型，帮助你理解伤痛的过程，并学习具体的方法来缩短痛苦的时间，更快地走向接受与成长。

人生是一场不断经历失去的过程。每个人都会遇到感到彷徨、无助和痛苦的时刻。如果长期无法走出这种状态，最终能帮助你的，可能只是几句简单而深刻的话。

## 理解伤痛的五个阶段 🧠

上一节我们提到了面对失去时的普遍感受，本节中我们来看看心理学是如何系统描述这个过程的。无论是重大失去还是日常挫折，伤痛通常会经历五个阶段。这是一个心理学上的共识。

以下是伤痛的五个阶段：

1.  **否认**：拒绝接受已经发生的事实。
2.  **愤怒**：对失去感到生气、不公。
3.  **挣扎**：试图通过行动挽回或改变现状。
4.  **抑郁**：意识到无力改变后，陷入难过与消沉。
5.  **接受**：最终平复情绪，接纳现实。

为了让你更清晰地理解，我们举一个常见的例子：丢失自行车。

*   首先，你会否认：“我的自行车真的丢了吗？是不是我记错位置了？”
*   确认丢失后，你会愤怒：“谁偷了我的车？让我找到他！”
*   接着，你会挣扎，四处寻找，希望找到它。
*   寻找无果后，你会感到郁闷和难过。
*   最后，随着时间的推移，你接受了自行车已丢的事实，并选择步行或购买新车。

即使是面对像确诊癌症这样的重大打击，人们也普遍会经历这五个阶段。瑞士一家临终关怀医院的研究证实了这一点。人与人之间的区别在于，**每个人在每个阶段停留的时间长短不同**。如果我们能找到正确的方法和视角，就能有效缩短在痛苦阶段（尤其是愤怒、挣扎和抑郁阶段）的时间，更快地走向“接受”。

## 聚焦：如何走出失恋 💔

理解了伤痛的普遍过程后，我们将其具体应用到一种常见的痛苦经历——失恋中。面对爱情的失去，我们需要建立正确的核心概念。

**最核心的概念是：每一次分手都能带来疗愈，因为它让你从过去的错误中学习。** 当你将关注点从“失去”转向“自我成长与疗愈”时，就能更快地走出痛苦。

然而，失恋的过程非常复杂，常伴随“没有他/她，我活不下去”等极端想法。因此，你需要一些工具来帮助自己。

以下是几句有效的“正向自我肯定”，建议写在你能经常看到的地方，时刻提醒自己：

*   **我为自己而存在。**
*   **他/她会爱上我，也会离开我，但我会一直爱我自己、支持我自己。**

如果你不能接受伤痛并进行正向的自我肯定，就很容易陷入怀疑、自责和试图挽回的恶性循环，这反而会让你失去吸引力。因为恋爱中有一个重要原则：**自爱**。

## 自爱：你的情感水槽 💧

上一节我们强调了正向肯定的重要性，本节中我们通过一个比喻来深入理解“自爱”的核心作用。

你可以把自己想象成一个**水槽**，里面的**水位**代表你爱自己的程度。

*   如果你的水槽原本就有80%的水（即你足够自爱），当外来的“爱的洪流”注满水槽时，你会感到幸福。
*   当这股洪流退去，你的水槽里**依然保有80%的水**。因为你本自具足，所以比较容易恢复和走出伤痛。

相反，如果你把恋爱当作救命稻草，意味着你的水槽本身是**接近干涸**的（自尊水平低）。当爱的洪流涌入时，你会感到极度美好；一旦它退去，你的水槽将**空空如也**，你会感到彻底崩溃。

**公式：承受失去痛苦的能力 ≈ 自爱程度（水槽水位）**

因此，**自爱是走出失恋的基石**。如果我们自己不爱自己，那么无论别人给予多少爱，一旦他们离开，我们都将无法承受痛苦。

## 疗愈与支持自己 🌱

明白了自爱是基础之后，我们来看看如何主动进行疗愈并在过程中支持自己。走出情感上的失去，主要依靠三个东西。

走出失恋需要依靠三个核心支柱：

1.  **自爱**：确保你的“情感水槽”有足够高的水位。
2.  **疗愈过去**：主动对过去进行释怀。可以反复告诉自己：“**我爱我自己，我原谅我自己。我完全放下了过去的一切，我自由了。**”
3.  **支持自己**：在失恋过程中，自尊水平容易下降，导致自责，而自责又会进一步降低自尊。你必须打破这个循环，学会在内心支持自己，而非批判自己。

生命有其自己的节奏与真谛。它不会让你永远平稳，而是会带来未曾预料的挑战和改变。

> 生命总是朝着疗愈的方向前进。它往往不以你期望的方式展开，充满迂回曲折。一旦你允许自己感受改变的痛苦，接受失去并克服伤痛，你就能体会到生命的真谛。

本节课中，我们一起学习了如何走出低谷。我们了解了**伤痛的五个阶段模型**，并将其应用于失恋场景。我们认识到**自爱（情感水槽水位）** 是抵御痛苦的根本，并学习了通过**正向自我肯定**、**疗愈过去的话语**和**主动的自我支持**来缩短痛苦时间，走向接受。真正的强大在于，无论经历什么，你都能疗愈自己的内心。
