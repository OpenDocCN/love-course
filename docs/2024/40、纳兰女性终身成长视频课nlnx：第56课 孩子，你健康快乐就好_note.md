在本节课中，我们将探讨教育认知的第二步：如何为孩子构建新的成长蓝图。我们将从长远视角出发，思考我们希望孩子20年后成为什么样的人，并学习如何以这个目标来指导当下的教育。

上一节我们讨论了如何“脱下旧衣服”，即摒弃过时的教育观念。本节中，我们来看看如何“穿上新衣裳”，建立面向未来的新教育认知。

## 一、以终为始：规划20年后的孩子

你现在是成年人。听课的姐妹，至少是二十几岁、三十几岁、四十几岁的妈妈。你希望你的孩子成年以后，像你这样的年龄，他是一个什么样子的人。那么你就可以按20年后的标准来教育你的孩子。

20年的时代可能时过境迁。例如，我们早在5年之前，谁知道抖音是什么。而现在抖音改变了我们的生活。它改变了你获取信息的渠道，改变了你的消费方式，甚至改变了一个人的思维和认知方式。我们无法精准预测未来，但可以把握不变的核心。

很多东西一年一年都在变，但有很多东西是亘古不变的。例如，我们当下有钱的，也有焦虑不安的；没钱的，也有焦虑不安的；有钱的也有非常平稳的，没钱的也有平稳的。你会发现，孩子内在的品质、价值观层面、精神的富足，这些东西永远都是一个孩子最大的内在动力源和精神滋养。这些东西有了，他以后无论是在什么时代，面对什么样的挑战，从事什么行业，他都能够独当一面，都是独立和完整的。

## 二、新教育认知的六个核心要点

以下是构建新教育认知的六个重要方面，我们将逐一展开。

### 1. 健康比成绩重要

大家不要以为孩子的身体健康是理所当然的。一个高级动物的健康是身心都健康才算的。身体健康的确是我们大部分父母都幸运拥有的，但心理是否健康，是需要用智慧来链接的。所以你不能只关注你孩子的身体。很多孩子长大后的叛逆或各种各样的问题，都跟他心理营养缺乏有关。他的心理未必健康，他所表现表达的未必是他内在真实的东西。

在电影《你好，李焕英》里，贾玲的母亲说：“你健康快乐就好。”很多妈妈转发说“我的孩子健康快乐就好”。大家老觉得“健康快乐”这四个字是唾手可得的，太简单了，是一个最简单的基本的培养目标。恰恰相反，一个健康快乐的孩子才是一个人的最终极的目标。如果你能够以“健康快乐就好”培养孩子，其实你的使命和责任是非常大的，你要做的功课也非常多。

很多事情就像生病，只要健康，你就不觉得健康珍贵。只要我们的孩子是健康的，你就不觉得健康重要。你的孩子发个高烧，你在病床前那一刻，你都觉得“你考多少分，我都接受”。人最悲哀的是，我们在生这个孩子的时候，目的非常简单：“孩子健康长大就好了”。但随着孩子成长，学龄段提升，妈妈的欲望就像气球一样吹起来。这很可怕。

**核心公式：终极目标 = 身体健康 + 心理健康**

### 2. 方向比努力更重要

这个道理对大人也适用。你觉得那么辛苦，在这个行业里辛苦多年，却好像老无所成，这跟你的方向不对有关。当你方向不对的时候，你越努力，离你最想去的地方越远，南辕北辙。你做什么选择，比你有多努力更重要。所有刻意的、违背本心的努力背后，都可能导向相反的目标。父母的首要功课是帮孩子找到方向。

例如，一个美国女孩在父母规划下成为工程师，但她从小喜欢画画。一场车祸让她失去双腿后，她重新拿起画笔，成为一名成功的画家。她说：“感谢上天，让我失去了双腿，让我重新找回了我自己的人生的方向和兴趣爱好。”你真的不知道什么是真正适合孩子的。

全红婵是跳水天才，张继科是打球天才，成龙是武打明星。如果都以成绩来定论，他们是“优秀的孩子”吗？全红婵自己说“我不大喜欢学习”。如果你非得觉得你的孩子非要以学习定论，孩子的其他天分也许就在你的要求下被扼杀。

**核心概念：选择 > 努力**

### 3. 如何帮孩子发现天赋

以下是帮助孩子发现天赋的两个关键方法。

**第一，提供探索空间。** 从小帮助孩子去探索，给到孩子自由成长和选择的空间。很多孩子的“兴趣爱好”，其实是家长想为升学加分。这个兴趣是孩子的兴趣，还是家长的兴趣？如果孩子是咬紧牙关去坚持，他的灵魂是紧张的，而所有的兴趣和天分都需要自由的灵魂去连接。家里经济条件富足，可以带孩子多尝试；条件一般，也可以支持他打球、踢足球等。关键是尊重他的自由和选择。

**第二，学会接纳与不干预。** 如果你第一条做不到，请回到第二条：你要接纳他，不要过多干预，不要强行给他定位定方向，不要在孩子年龄小的时候剥夺他尝试和体验的机会。全红婵最大的天然优势，是她出生在一个普通的农村家庭，家里有多个孩子，父母没有把全部精力和期望捆绑在一个孩子身上。父母“不干预”的态度，反而成全了天才。

### 4. 能力比知识更重要

知识是我们学到的概念理论，能够背下来、讲出来的东西。能力是你完成一项目标或任务所体现出来的综合素养。例如，给你一个陌生领域的命题作文，我能讲上半小时，自圆其说，这是一种能力。能力总与完成一定的实践联系在一起，能力来自实践。

知识可以学到，而能力只能“习得”。“习”在古语里是练习、实践的意思。很多孩子高分低能，记忆力好，但应用到生活和关系中就不行。知识是交通规则，能力是驾驶技术。你懂再多规则，不会开车，上路就被贴罚单。

在我父母那一代，知识极度匮乏，读过书本身就是一种能力。但在当下，知识储备非常丰富，能力就更重要了。好的知识是通过实践得来的。你从书本获得知识，再通过实践获得“真知”。从知识到真知的过程，就是能力的体现。没有能力的人无法把知识转化成真知。

**核心关系：知识 → (实践) → 能力 → 真知**

### 5. 情商比智商更重要

智商是硬实力，基本由基因决定。情商是软实力，是后天可以无限发展的空间。情商通俗讲，是一种能够让人舒服的能力，一种让你的爱能够到达的能力。情商决定了你的自我关系，也决定了你与他人的关系。

高智商的人可能在职场或学术上有成就，但回了家，可能是一个连爱人都不了解的人。这也是另一种形式的“残疾”——你不能跟周围的人好好连接。

以下是孩子获得情商的三个方法。

**第一，家庭关系稳定。** 稳定的家庭关系是孩子情商的底色和土壤。父母情绪的自我关照能力强，家庭能量稳定，才能给孩子更好的情绪滋养。

**第二，接纳孩子的情绪。** 允许孩子有情绪，做孩子情绪的容器。允许和接纳、倾听和觉察，让孩子情绪不积压。孩子突然的情绪爆发，往往跟长期的压抑有关。

**第三，丰富社会与自然链接。** 让孩子有自己的小伙伴圈子，在社会关系中提高情商。同时，不要忽略让孩子接触大自然。大自然的花草虫鱼、风雨雷电，都能培养孩子的感知力、悲悯心和爱的能力。

### 6. 生活比学习更重要

我们拼命工作、努力赚钱是为了什么？为了生活。我们学习的目的是为了什么？也是为了生活。但大多数人一生疲于奔命而忘记了生活。

很多年前，记者采访放牛娃的梦想。放牛娃说：放牛、赚钱、娶媳妇、生孩子，孩子长大了再放牛。大家觉得可笑。请问我们考大学是为了什么？找个好工作、娶媳妇、生娃。然后你的娃长大了，你再让他考更好的大学……我们和放牛娃的本质区别有多大？那个放牛娃很幸福，但城里的白领可能没有他幸福。

一个人最根本的幸福感的缺失，在于你没有链接当下的能力了。你所有的拥有都不再被珍惜。我们一辈子的努力都是为了离开农村，现在的目的又是想回去。人生仿佛一个轮回。

**核心反思：所有努力都应回归生活本身。**

## 三、总结：我们要培养什么样的孩子？

今年两会期间，一位中学校长唐江澎的发言很有启发性。他说：“好的教育应该是培养终身运动者、责任担当者、问题解决者和优雅生活者，以孩子们健全而优秀的人格赢得未来的幸福。”

分数是重要的，但分数不是教育的全部内容，更不是教育的根本目标。我们家长应该培养的是一个心智健全、人格健全的孩子。至于他能否在不同行业释放天赋，这要看机缘和孩子的慧根。

## 四、给家长的心态建议

教育认知的第三步，是从家长角度保持好心态。

每一个孩子都是一粒种子，只是花期不同。当你的孩子老是不开花的时候，不要以为他是废物，他有可能是一棵大树。庄子说“无用方为大用”。一棵树因为“没用”而没被砍伐，反而能存活千年，为乡里乡亲遮阳纳凉，成就了更大的用途。

教育本身就是一个缓慢的过程。教育的本质是“一棵树摇动另一棵树，一朵云推动另一朵云，一个灵魂唤醒另一个灵魂”。这是一个慢下来、尊重自然规律的过程。成长是终身课题，我们现在四十多岁都坐在这里听课，何必苛求一个六岁的孩子呢？

以下是给父母的三个行动关键词：

1.  **相信**：永远相信你的孩子内心是向上的。没有一个孩子希望自己糟糕。
2.  **接纳**：学会接纳、了解、陪伴，以包容开放的姿态面对孩子。
3.  **等待**：让孩子主动思考，等待他展开自己生命的思考。他的自我探索可能很早开始，只是需要时间显现。

## 五、什么是好的教育？

最后，我们来总结一下什么是好的教育。

1.  **好的教育是让孩子成为他自己**，成为一个精神独立的人。例如“国民才女”武亦姝，小时候也是“差等生”，是母亲的耐心引导，在生活中培养了她的诗词爱好。
2.  **好的教育是父母做自己**，发挥榜样的力量。高晓松的家庭教育就是如此，代代相承的是做自己的态度。
3.  **好的教育是培养孩子独立生活和走向社会的能力**。包括基本生活技能、吃苦耐劳、解决问题、情商心智等。
4.  **好的教育是用价值观来教育孩子**。陶行知校长用三块糖教育打架学生的故事，体现了理解、包容、尊重和价值观引导。父母的善良、正直、利他之心，每天都在通过言行示范给孩子。孩子的价值观天然就有，需要你来强化。在一个好的价值观和稳定能量的家庭里，孩子品行不会差。

**核心总结：好的教育 = 健全人格 + 正确价值观 + 独立能力**

本节课中，我们一起学习了如何以长远眼光看待教育，认识到健康、方向、能力、情商和生活本身的重要性远超过单纯的成绩与知识。教育的最终目的，是培养一个心智健全、人格独立、能够感受幸福并创造价值的人。父母最好的角色，是成为孩子价值观的稳定示范者和成长路上的耐心陪伴者。
