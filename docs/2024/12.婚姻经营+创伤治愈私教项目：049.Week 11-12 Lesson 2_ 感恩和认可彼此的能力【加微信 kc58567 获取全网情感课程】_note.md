# 课程12：婚姻经营与创伤治愈私教项目 - 第049课：第11-12周 第二讲 🙏 感恩与认可彼此的能力

在本节课中，我们将学习如何真正地接纳彼此，掌握在日常生活中认可彼此的具体方法，并更好地表达感恩，从而培养感恩的心态。心理学研究表明，拥有感恩之心的人往往生活得更快乐。因此，接纳、认可和感恩是心理学中非常重要的三种正面情绪。

## 第一步：接纳是基础 🌱

上一节我们介绍了本模块的核心目标，本节中我们来看看接纳的重要性。在心理咨询中，咨询师首先需要培养的能力就是接纳来访者——接纳他们的情绪、想法和行为，不去责怪或对抗。接纳意味着承认事实的存在，并试图去理解和弄明白它。

我想问问你：你接纳自己吗？你喜欢自己吗？在中国的文化背景下，“接纳”这个词并不常用。特别是对于70、80、90年代成长起来的人，我们往往习惯于用对错来评判事物，倾向于只做“正确”的事，而难以接纳所谓的“错误”。

在咨询工作中我发现，那些不接纳自己、经常自我攻击的人，在婚姻中也更容易不接纳并攻击伴侣，从而导致更多的沟通问题。因此，请你们暂停一下，相互分享一下彼此的自我接纳程度和对自己的喜爱程度。

### 接纳的治愈力量

我认为，接纳具有强大的治愈能量。在咨询中，当一方哭泣时，如果另一方能够接纳这些情绪，并说出“我看到了你的悲伤”，那么被看见和被接纳的情绪往往就能得到治愈和放下。反之，如果在对方痛苦时讲道理或无视其痛苦，只会加剧伤痛。

不接纳会延长负面情绪或痛苦。这同样适用于自我接纳。在个体咨询中，我会评估一个人是接纳还是排斥自己。排斥自己的人，治愈过程通常更长，因为他们厌恶自己；而自我接纳的人，进步则更快。

对于你们的关系而言，无论发生什么，请不要争论对错或比较优劣。试着从彼此的角度去接纳事实：**他/她此刻就是有这样的情绪和想法，暂时无法改变**。那么该怎么办呢？可以问一些更深入的问题：
*   你为什么这么想？这个想法从何而来？
*   你为什么会有这种情绪？

通过更深层次地理解彼此，就像咨询师所做的那样——即使不完全听懂，但首先选择接纳，才能更好地理解对方。

### 关于接纳的常见疑问

有些学员会问：“如果有些事情我就是无法接纳怎么办？”需要明确：**接纳不等于赞同**。接纳只是承认这是一个既存的事实，我不与之对抗，我去看见并试图理解它。这并不意味着你必须接受未来一直如此，或者你必须赞同它。接纳的核心是：**OK, it's there. Let it be.**（好吧，它就在那里。让它存在。）

另一些学员会问：“我们的性格截然不同，对方有很多糟糕的缺点，我实在无法接纳。”然而，你越不接纳，你们的关系就会越糟。以我和我先生为例，我们在金钱观念上因成长背景不同而有巨大差异。我来自80年代相对拮据的中国家庭，对金钱有不安感；他成长于美国中等家庭，对金钱没有概念。我们无法改变彼此的底色。

**解决方案是：首先接纳彼此的不同。** 这些不同没有对错，只是成长环境塑造的结果。有些反应是你的潜意识或创伤应激，你不能要求对方和你一样。接纳彼此的存在状态——无论是情绪表达、金钱观念还是事业追求上的差异。不要因为自己认为“好”而去挑剔对方。试着看到，自己的一些特质在社会文化中被认为是“好”的，但未必绝对正确。

先接纳彼此，再进行深入沟通，了解彼此的成长经历。或许你们可以相互影响，共同变得更好。

以下是需要你们共同完成的练习：
请思考并分享：你觉得需要接纳对方的哪些方面？你们彼此身上有哪些因成长环境不同而产生的差异，需要更多的接纳而非挑剔？即使真是缺点，也可以坦诚沟通，通过相互影响慢慢改善。

## 第二步：认可彼此的能力 👍

接纳讲完后，我们来看看第二个重点：认可。无论是在咨询室内外，都有太多伴侣在“乞讨”对方的认可。先生渴望太太的认可，太太也渴望先生的认可。

为什么我们在婚姻中如此渴望被对方看见和认可？因为在我们成长过程中，常常缺乏认可。父母、老师的打击和批评，让我们内心产生羞耻感和自我怀疑（“我作为太太/先生做好了吗？”）。因此，婚姻中的批评很容易触发这些旧的负面感受。

请反思一下：你们是否会相互指责、批评、否定对方？如果对方回答“有”，那么请先接纳这个感受：“他/她就是真实地这么觉得”。然后去弄明白，对方为什么感觉被否定。如果伴侣间持续出现攻击性批评，另一方可能会出现应激反应，如激烈反击、冷战回避，甚至自我伤害行为（如撞墙）。

思考一下，你批评的情绪从何而来？是因为对方不听你的，还是源于你过去的经历？

我曾有位来访者，太太产后忙于孩子，忽略了先生。先生极度渴望被看见和认可，当工作中出现一位欣赏他的女性时，他深受感动，进而发生了越界行为。这提醒我们，当内心缺失某些东西时，我们可以在婚姻中去治愈它。

总体而言，中国的先生们可能不会直接说“请鼓励我”，但他们内心非常渴望听到太太的表扬和鼓励。如果你希望得到表扬，也可以直接表达。另一种情况是，有些人内心渴望表扬，但因从小不习惯被表扬（父母老师很少表扬），刚开始被表扬时会不习惯甚至不好意思，这本身也是个人需要成长的课题——学习接纳赞美。但请不要放弃，要多学会表扬和认可。

### 如何进行有效的“具体表扬”

我不太喜欢“你好棒”、“你好厉害”这类肤浅的表扬。它们过于空泛，缺乏信息量。我想教你们一种更有效的认可方式：**具体的表扬**。

具体的表扬包含丰富的信息量，明确告诉对方你到底认可什么，给予对方清晰的信号。这样，对方下次就知道该多做些什么。“你好棒”则没有提供这样的具体信息。

在心理学上，这被称为 **“标记性赞扬”**。具体操作步骤如下，你的表扬应包含以下要素：
1.  **描述行为**：具体描述对方做了什么事情。
2.  **指明细节**：指出他/她做法的哪几个点特别突出。
3.  **表达影响**：说明这个行为对你产生了什么具体影响、帮助。
4.  **分享感受**：表达你当时的心情和感受。
5.  **阐述意义**：讲述这对你意味着什么。

**公式：具体表扬 = 描述行为 + 指明细节 + 表达影响/帮助 + 分享感受 + 阐述意义**

这种表扬方式的作用在于**提供进步的路线图**。你这次表扬这个点，他记住了；下次表扬那个点，他也记住了。对方就会有一个清晰的进步方向和地图。

**应用场景示例：性生活**
在性生活中，如果这次某个环节感觉特别好，就具体告诉对方。下次尝试又有好的体验，再次具体指出。经过多次尝试和反馈，你们就能总结出哪些方式让彼此愉悦，以后的性生活就会更愉快顺畅，无需过多沟通。因此，请收集那些好的瞬间，并具体地表扬出来。

具体的表扬可以应用在生活的方方面面：日常生活、亲密关系等。它能给予对方具体的指引，避免一方默默付出却始终无法满足另一方的需求。常见的情况是，先生觉得自己付出很多，太太却感觉不到爱；太太则觉得先生是按自己的方式爱，并不懂她。这时，太太可以帮助对方，在他做得好时，给予具体的反馈和“地图”。

以下是需要你们进行的练习：
请太太转向先生，具体表扬他最近做的一件事情。按照上述步骤，描述他做了什么、哪几个点做得好、给你带来了什么帮助和感受、对你有什么意义。太太讲完后，再由先生同样具体地表扬太太最近为他做的一件事。通过这个练习，你不仅在表达认可，其实也在表达自己的需求。

## 第三步：培养感恩的心 ❤️

最后，我们来探讨需要培养的第三种情绪：**感恩**。感恩的情绪不太容易训练，它更多是一种个人体验。当我放下更多负面情绪、更多表达爱时，我才最终能够到达感恩的状态——由衷地感恩生活、感恩对方的存在。

感恩需要你主动练习。例如，我每天都会感谢一些重要的事物。这是我在学习心理学时培养的练习：关注正向的事物并感恩，你会更快乐。心理学上对感恩情绪有大量研究。

中国的太太先生们其实非常愿意为彼此付出。我们经常听到的一句话是：“我并不需要他/她为我多做多少，我只希望他/她能感激一下我，看到我所做的。” 如果对方能表达感谢和看见，我们会更愿意付出。

总的来说，感恩的情绪需要平时多加练习。请经常相互说说感谢的话。

以下是本节课最后一个练习：
请向对方说一句彼此感恩的话，感谢对方所做的某件事。

多练习感恩的情绪。当你们处理了创伤，学会了表达接纳和认可，最后一步就是多表达感激。请摒弃“对方做这些是应该的”这类想法，多跟对方说感谢。

本节课中，我们一起学习了婚姻中三种重要的正面情绪：**接纳**（承认事实，不对抗，是治愈的基础）、**认可**（通过具体表扬提供进步地图，满足被看见的需求）以及**感恩**（主动练习，表达感激，深化连接）。通过理解和练习这些方法，你们可以更好地经营彼此的关系，促进共同成长。
