# 《Leon全套课程合集》：Leon2.0：5.搭讪基本框架流程 🎬

## 概述

在本节课中，我们将学习如何进行有效的搭讪，包括基本框架、应对不同情况以及常见错误。

## 搭讪基本框架

搭讪的基本框架分为四个点：

1. **表明意图**：明确表达你的搭讪目的，让女生知道你的意图。
2. **解决问题**：解决女生的心理防备和情绪问题。
3. **话题延伸**：与女生进行互动，建立舒适感。
4. **收尾**：结束搭讪，留下联系方式。

### 表明意图

- **核心**：让女生知道你的意图，避免被误认为是销售人员。
- **方法**：用坚定、自信的语气表达你的意图，例如：“嘿，你好，我刚刚看到你，想认识一下你。”

### 解决问题

- **原因**：女生可能因为不安全感而表现出防备。
- **方法**：再次表明你的意图，安抚她的情绪，例如：“别慌，我不是坏人，只是想认识一下你。”

### 话题延伸

- **目的**：建立舒适感，让女生了解你。
- **话题**：现状、住址、年龄、家乡、职业等。

### 收尾

- **方法**：留下联系方式，例如：“那我们加个微信吧。”

## 应对不同情况

- **行走的目标**：走到女生旁边或前面，拍一下她，让她停留。
- **静止的目标**：走到女生面前，与她打招呼。
- **组合**：提前与女生的闺蜜打招呼，避免被拉走。

## 常见错误

1. **开场不够坚定**：导致女生误认为是销售人员。
2. **没有解决女生的情绪问题**：导致女生感到不安全。

## 总结

本节课中，我们一起学习了搭讪的基本框架、应对不同情况以及常见错误。希望这些知识能帮助你更好地进行搭讪。
