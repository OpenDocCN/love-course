# 《自然方法》搭讪自然人：1.3四层沟通 🎤

## 概述

在本节课中，我们将学习如何通过四层沟通技巧来提升搭讪和约会的效果。

## 四层沟通

以下是四层沟通的详细解释：

### 1. 说话内容（表层沟通）

这是沟通的第一层，指的是你所说的具体内容，即你的表层意思。

**公式**： 表层沟通 = 说话内容

### 2. 说话方式（浅沟通）

第二层是说话方式，包括你的语调、表情、肢体语言等，这些都是浅沟通的组成部分。

**公式**： 浅沟通 = 语调 + 表情 + 肢体语言

### 3. 背后的意图（框架）

第三层是背后的意图，即你的出发点，这决定了你的沟通方式和目的。

**公式**： 框架 = 出发点

### 4. 自身水平（心态）

第四层是自身水平，即你的心态，这是沟通中最重要的一层。

**公式**： 心态 = 心情 + 自尊水平

## 沟通顺序

以下是四层沟通的先后顺序：

1. **自身水平（心态）**：这是最重要的，如果你的心态不好，那么其他沟通技巧都将无效。
2. **背后的意图（框架）**：如果你的心态没问题，那么你需要检查你的框架是否正确。
3. **说话方式（浅沟通）**：如果你的框架没问题，那么你需要注意你的说话方式和肢体语言。
4. **说话内容（表层沟通）**：这是最不重要的，如果你的前三层沟通都做得很好，那么你只需要确保你的说话内容有趣、有逻辑。

## 减压和施压

在沟通中，我们可以通过以下方式来减压和施压：

- **说话内容**：通过改变说话内容的方式，我们可以实现减压或施压。
- **说话方式**：通过改变语调、表情和肢体语言，我们可以实现减压或施压。
- **背后的意图**：通过改变沟通的目的和出发点，我们可以实现减压或施压。

## 总结

本节课中，我们学习了四层沟通技巧，以及如何通过这些技巧来提升搭讪和约会的效果。记住，沟通的顺序很重要，先从自身水平开始，然后是背后的意图、说话方式和说话内容。希望这些技巧能帮助你更好地与人沟通。
