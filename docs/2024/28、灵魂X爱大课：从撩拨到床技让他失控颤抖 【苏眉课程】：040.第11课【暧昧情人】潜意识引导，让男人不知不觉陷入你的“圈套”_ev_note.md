## 潜意识价值观植入

### 直接植入

直接植入是指强行说服对方，例如：“你不要抽那么多烟嘛，这个对肺非常不好。”

**公式**： 直接表达 → 对方潜意识 → 反感

### 潜意识价值观植入

潜意识价值观植入是通过语言暗示，让对方的潜意识自主地想到你要的结果。

**公式**： 语言暗示 → 对方潜意识 → 想要的结果

**案例**： 通过展示与优秀异性在一起的照片，让对方感受到你的抢手。

## 潜意识

潜意识是指人类心理活动当中没有被察觉的部分，是人类已经发生了，但是还没有达到意识状态的那个心理活动。

**案例**： 同一瓶酒，因为潜意识的影响，口感会有所不同。
