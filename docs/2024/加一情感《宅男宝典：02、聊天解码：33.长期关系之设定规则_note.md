# 加一情感《宅男宝典：02、聊天解码：33.长期关系之设定规则🎼

嗨你好，欢迎来到情圣方法，我是班长alen。本节课我们将探讨长期关系相处模型的第三点——设定规则。

**设定规则是任何一段情感关系的开始。** 

**规则** 是你自己的框架、态度和标准。例如，你不喜欢女生抽烟，要求女生不能超过12点回家，不希望她八卦他人，这些都是你的标准。

**只有拥有足够多的、内心深处的标准，规则才能让感情越来越好。** 许多兄弟不敢设定规则，以为讨好女生就能赢得她的爱。但事实上，设定规则会让你显得更有价值，女生会更喜欢你。

**特别是在异地恋或情感维系中，如何设定规则是关键。**

**设定规则分为三个步骤：**

1. **制造意外**：例如，女生不接电话，你可以打第二个，她还是不接。
2. **冷处理升级意外**：不要轻易放弃，要让她意识到你重视这个问题。
3. **颠覆价值观**：解释为什么你这么在意，给出合理的价值观。

**以下是一个例子：**

- **意外**：女生不接电话。
- **冷处理**：不立即道歉，而是让她意识到你很在意。
- **价值观**：强调相互尊重是感情的基础。

**设定规则时，要注意以下几点：**

- **价值观**：确保你的价值观是正确的，对方会认可。
- **底线**：明确你的底线，让对方感受到压迫感。
- **合理性**：将规则与价值观相结合，使其合理。

**总结：**

- 设定规则是感情关系的开始。
- 通过制造意外、冷处理升级意外和颠覆价值观来设定规则。
