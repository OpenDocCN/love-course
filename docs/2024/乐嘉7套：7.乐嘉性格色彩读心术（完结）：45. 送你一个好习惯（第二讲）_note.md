# 乐嘉性格色彩课程（45）：送你一个好习惯（第二讲）🎁

在本节课中，我们将学习如何利用性格色彩学的原理，针对不同性格特点来培养和坚持一个好习惯。课程将提供四种具体方法，帮助你突破自我，建立持久的好习惯。

上一节我们介绍了习惯养成的重要性，本节中我们来看看如何运用性格色彩学的智慧，为不同性格量身定制习惯养成策略。

## 利用红色性格培养坚持力

你可能会感到奇怪：红色性格的人通常被认为目标性差、兴趣容易转移、缺乏持久力，如何能用红色性格来培养坚持呢？关键在于红色性格的核心特点——**喜欢互动和受到关注**。

以下是具体方法：

*   **公开打卡**：每天完成任务后，在朋友圈等社交平台打卡记录。红色性格渴望获得他人的赞美（如“真的好努力”）和关注，这种正向反馈能转化为持续行动的动力。
*   **对抗负面评价**：红色性格在意他人看法。可以利用“不希望被他人视为肤浅、只有三分钟热度的人”这种心理，用持续的实际行动来证明自己的坚持，从而将外在压力转化为内在动力。

因此，对于红色性格而言，坚持的方法核心在于**获得足够的评价、认可或关注**。利用这一特点，在亲朋好友的反馈中启动并维持你的习惯。

## 学习黄色性格培养进取性

在培养习惯时，许多人倾向于从简单、有趣的内容开始。但如果你想借鉴黄色性格的优点，就需要调整策略。

黄色性格的特点是**目标明确、追求效用、敢于挑战极限**。他们可能认为看武侠小说是“无用的消遣”，而更倾向于阅读如《资治通鉴》、商业或技能类书籍，因为这些能带来新思想、新技能并立即改变生活。

因此，在培养习惯时，应学习黄色性格，为自己设定有挑战性、有实际价值的目标，而不仅仅满足于轻松入门。

## 借鉴蓝色性格培养计划性

当你有了动力和目标后，下一步是制定周密的计划。这正是蓝色性格的优势所在。

所谓计划，就是明确**每天何时做、做多久**。以晨跑为例，蓝色性格会注重细节和科学性：
*   跑前需做热身运动（如压腿、下蹲）。
*   注意呼吸方式（用鼻而非口）。
*   控制时长与强度（如25分钟即可，避免马拉松式慢跑加快走）。
*   了解注意事项（如晨跑前不宜吃早餐）。

对于随性的红色性格，在开启习惯培养前，务必学习蓝色性格的严谨。可以上网搜索相关注意事项，学习他人经验，并严格执行计划。

## 模仿绿色性格让习惯成为自然

最后，你需要学习绿色性格，让习惯最终内化为本能。

就像绿色性格每天自然而然地刷牙洗脸一样，这些行为已成为他们“打娘胎出来就有的”本能反应。我们之前介绍的方法（如利用红色对赞美的需求）是帮助启动习惯的“拐杖”。

但核心目标是：**在习惯养成后，要努力使其像绿色性格的本能一样，融入日常生活，成为无需刻意坚持的部分**。一旦习惯成为你血液的一部分，你将终身受益。

2.  学习**黄色**的目标性与进取心，挑战有意义的任务。
3.  借鉴**蓝色**的计划性与严谨性，科学执行每一步。
