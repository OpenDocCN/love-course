在本节课中，我们将学习如何使用Python的Pandas库进行基础的数据处理。我们将通过一个简单的案例，了解如何创建、查看和操作数据。

## 概述

Pandas是Python中一个强大的数据分析库，它提供了快速、灵活且表达力强的数据结构，旨在使数据清洗和分析工作变得简单快捷。本节课将介绍其核心数据结构DataFrame的基本操作。

## 1. 创建DataFrame

上一节我们介绍了Pandas的基本概念，本节中我们来看看如何创建一个DataFrame。DataFrame是一个二维的、大小可变的、可以存储多种类型数据的表格结构。

以下是创建一个简单DataFrame的步骤：

1.  首先，需要导入pandas库。通常我们使用`pd`作为其别名。
    ```python
    import pandas as pd
    ```
2.  准备数据。数据可以是一个字典，其中键是列名，值是列表形式的数据。
    ```python
    data = {
        ‘姓名‘: [‘张三‘, ‘李四‘, ‘王五‘],
        ‘年龄‘: [25, 30, 35],
        ‘城市‘: [‘北京‘, ‘上海‘, ‘广州‘]
    }
    ```
3.  使用`pd.DataFrame()`函数将字典转换为DataFrame。
    ```python
    df = pd.DataFrame(data)
    ```

## 2. 查看与探索数据

创建好DataFrame后，我们需要查看数据的基本信息以了解其结构。

以下是几种常用的查看数据的方法：

1.  使用`df.head()`可以查看数据的前几行，默认显示前5行。
    ```python
    print(df.head())
    ```
2.  使用`df.info()`可以获取数据的简明摘要，包括索引数据类型、列数据类型、非空值数量和内存使用情况。
    ```python
    print(df.info())
    ```
3.  使用`df.describe()`可以生成描述性统计信息，包括计数、均值、标准差、最小值、四分位数和最大值。这对于数值型列非常有用。
    ```python
    print(df.describe())
    ```

## 3. 数据选择与过滤

在数据分析中，经常需要从DataFrame中选择特定的行或列。

以下是几种基本的数据选择方法：

1.  选择单列。可以通过列名直接选择，结果是一个Series（一维数组）。
    ```python
    ages = df[‘年龄‘]
    ```
2.  选择多列。需要将一个包含列名的列表传递给DataFrame。
    ```python
    subset = df[[‘姓名‘, ‘城市‘]]
    ```
3.  按条件过滤行。可以使用布尔索引来选择满足特定条件的行。
    ```python
    # 选择年龄大于28的行
    older_than_28 = df[df[‘年龄‘] > 28]
    ```
