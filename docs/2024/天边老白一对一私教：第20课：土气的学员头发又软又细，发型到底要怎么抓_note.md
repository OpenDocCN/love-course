# 天边老白一对一私教：第20课：土气的学员头发又软又细，发型到底要怎么抓

## 概述
在本节课中，我们将学习如何打理软细稀疏的头发，打造出蓬松自然的发型。

## 头发打理步骤

以下是打理软细稀疏头发的具体步骤：

### 1. 洗发
首先，用洗发水将头发洗净。

### 2. 吹发
将头发吹至半干，注意不要吹得太干，以免头发变得脆弱。

### 3. 蓬松处理
对于头发稀疏的人，不要考虑偏分，而是将头发向前吹，使其蓬松。

### 4. 使用发泥
使用哑光发泥，如欧莱雅哑光发泥，轻轻涂抹在头发上，使头发变得稍微硬一点，避免头发趴在头皮上。

### 5. 定型喷雾
喷上定型喷雾，以保持头发的形状。

## 定型喷雾的好处

使用定型喷雾可以：

- 保持头发的形状更久
- 防止头发被风吹散

## 定型喷雾的缺点

使用定型喷雾的缺点是：

- 容易导致头发掉落

## 总结
本节课中，我们一起学习了如何打理软细稀疏的头发，打造出蓬松自然的发型。通过使用发泥和定型喷雾，我们可以轻松打造出明星般的发型，而不必花费大量金钱烫发。
