# 天边老白一对一私教：第20课：发型打理教程 🧑‍🏫

在本节课中，我们将要学习如何为头发细软、稀疏且容易出油的男士打理发型。我们将重点介绍如何通过正确的吹发技巧和使用特定造型产品，打造出持久、蓬松且不显油腻的发型效果。

## 概述：发型问题的核心

很多学员面临的主要问题是发型打理。对于男士而言，发型对整体形象至关重要。许多人的头发与我相似：细软、稀疏、易出油且发际线较高。这类头发容易紧贴头皮，显得发量更少且缺乏精神。

上一节我们介绍了发型问题的普遍性，本节中我们来看看具体的解决步骤。

## 第一步：吹干头发，打造蓬松基础

首先，洗完头后需要正确吹干头发。目标是让头发蓬松起来，尤其是对于发量少和发际线高的人士。

**核心技巧**：不要采用传统的偏分或中分。应让头发整体向前、向上蓬松。
以下是吹发的具体方法：
*   将左侧头发向中间吹。
*   将右侧头发向中间吹。
*   将后侧头发也向中间吹。
*   重点将头顶区域的头发吹得竖立、蓬松。

这样操作后，头发在视觉上会显得更多，因为它拥有了体积感和支撑力。

## 第二步：使用哑光发泥进行塑形

吹干后的蓬松感并不持久，细软头发很快会再次塌陷。接下来我们需要使用造型产品来固定形态并增加发丝的支撑力。

**产品选择**：只推荐使用**哑光发泥**（例如欧莱雅品牌）。避免使用发蜡、亮面发胶或啫喱水，这些产品容易让细软头发显得油腻、结块。
以下是使用发泥的步骤：
1.  取适量发泥（约黄豆大小），在掌心彻底揉搓均匀，直至完全乳化、呈透明状。
2.  将带有发泥的双手手指轻轻插入发根，然后向外、向上轻抓头发。
3.  重点处理需要蓬松的区域，动作务必轻柔，以避免拉扯导致脱发。

发泥的作用是让细软贴服的头发变得稍硬挺，从而能够维持蓬松的造型，不会轻易塌陷。

## 第三步：（可选）使用定型喷雾加固

为了使发型在风吹或长时间活动中保持更久，可以使用定型喷雾进行最后加固。

**注意事项**：定型喷雾虽能增强持久度，但过量使用可能增加脱发风险。请根据自身需求和发质酌情使用。
以下是关于定型喷雾的要点：
*   在抓好的发型上均匀喷洒少量定型喷雾即可。
*   使用后，发型可保持一至两天。
*   在风大的天气，它能有效防止发型被吹散。

至此，我们已经完成了从吹干到定型的全部步骤。这个造型方法能创造出类似烫发后的蓬松效果。

## 总结与核心建议

本节课中我们一起学习了针对细软稀疏发质的完整打理流程：**吹蓬松 -> 哑光发泥塑形 -> (可选)定型喷雾加固**。

**核心公式**：`细软塌发型 = 逆向吹风 + 哑光发泥 + 轻柔手法`

最后给出重要建议：**不要轻易烫发**。对于发量少、发质软的人，烫发损伤大且效果未必理想。理发师可能出于盈利目的推荐烫发。我本人已多年未烫发，仅靠正确的吹发技巧和一瓶几十元可用数月的哑光发泥，就能维持日常所需的蓬松有型效果，这与许多明星造型的原理是一致的。

**成本对比**：
*   **本方案**：几十元的发泥可使用数月。
*   **烫发**：每次花费数百元，且存在损伤发质的风险。

记住，打理的关键在于让头发**脱离头皮**，只要发型整体蓬松，无论从正面或侧面看，效果都不会太差。
