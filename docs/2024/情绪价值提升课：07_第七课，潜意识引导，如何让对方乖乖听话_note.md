# 情绪价值提升课：07_第七课，潜意识引导，如何让对方乖乖听话 🧠

在本节课中，我们将要学习“潜意识引导”的核心原理与操作方法。你将了解到如何通过引导对方为你投资和付出，来掌握关系中的主动权，并让对方越来越离不开你。

上一节我们介绍了情绪价值的基础概念，本节中我们来看看如何通过具体的行为引导来影响对方。

## 引导投资的原理

在感情的世界里，只要你能够引导对方为你付出，你就能掌控这段感情。这句话的含义是，只要你学会正确的引导对方为你投资付出，你就能在这段感情里做一个主导者，控制感情的走向，同时也能让对方越来越爱你。

人是一种有了投资就会想有回报的动物。在恋爱中也是一样，有付出就希望获得自己想要的回报。引导对方投资就是利用对方想要回报的心态，去引导对方付出。最终，对方会因为付出太多而彻底离不开你。

例如，你读高中时非常想考一所好大学，于是付出了大量时间学习，废寝忘食。如果高考发挥失常，考了很差的分数，你会非常痛苦难过，因为你所有的投资付出都白费了。

## 投资的五种类型

在两性相处中，投资并非仅指金钱。以下是投资的五种主要类型：

*   **时间投资**：对方为你花费的时间。
*   **金钱投资**：对方为你花费的金钱。
*   **物质投资**：对方赠予你的礼物等物质。
*   **肉体投资**：亲密的身体接触与关怀。
*   **感情投资**：对方投入的情感与心思。

## 引导投资的两个方法：奖励与惩罚

引导投资有两个核心办法：**奖励**和**惩罚**。引导男生投资是一个训练的过程，目的是让他不断对你付出，用这些付出来讨好你，让你开心。

这个方法的本质原理是：你身上必须有男生需要的价值。男生喜欢你身上的某些优点，或者你们之间存在感情，你才能引导他投资。如果男生对你毫无兴趣，你无法吸引他投资。也就是说，你们之间需要存在好感，男生才会对你有投资行为。

所以，引导投资就是在男生对你投资时，你需要奖励他，让他开心，他才会愿意继续投资。在这个过程里，他是心甘情愿的，而不是被逼迫的。

而惩罚的目的，是让他不断对你好。当他没有对你投资时，你需要轻轻惩罚他，让他感到不开心，并告诉他“你没有做到这些事情，我很失望”。惩罚的目的是让他改正错误行为，继续按照你指定的方向前进。

## 引导投资的两个步骤

关于引导投资，具体可分为两步操作：

1.  **制定规则**：你需要为你们的相处制定一个规则。
2.  **引导付出**：当男生顺从你的规则时，你要引导他对你付出，这样男生就会被你折服。

今天的课程就像引导小孩子的过程。在小孩子不听话时，你要告诉他什么能做、什么不能做。这就是让男生听话的过程。但无论多听话的小孩子都会有叛逆阶段，怎样才能让小孩子不叛逆呢？所以就有了第二步引导投资。

当一个人对你付出了很多时间、金钱、精力和感情时，他就不会随便离开你了。因为他付出得越多，就越舍不得离开。

## 一个训狗的类比

下面用一个训狗的故事来说明。

比如你养了一只不听话的小狗，到处拉屎拉尿，甚至咬伤了别人。这难道是小狗的错吗？不是，归根结底是主人的问题。

如果小狗在做错事时没有受到惩罚，它就会继续犯错。因为小狗不知道对错。当小狗到处撒尿时，如果你当场抓住它，告诉它这么做是错的，并惩罚它一天没饭吃，它大脑里就会形成一种意识：知道做这件事是错的，会受惩罚。下次它就不敢再犯了。

把这件事应用到你伴侣身上，道理相同。当对方不听你话时，你要立刻告诉他：“你这样做是错的，我很不开心，你快点来哄我。”

当你辛苦下班回家，你的小狗很乖地把拖鞋叼到你面前，你摸了摸它的头，并给它喜欢吃的东西。这时小狗会非常开心，因为它会记住这样做得到了奖励。于是它每做对一件事，你就给它一次奖励。久而久之，它就会越来越听话。

同样，当你的伴侣对你付出时，你应该奖励他，告诉他：“你做的这件事让我非常开心。”夸奖之后，你可以陪他做一些他喜欢的事，比如陪他打游戏。这样他才会继续开心地对你好。

上面的小故事有两个重点：

*   **犯错要惩罚**：当小狗犯错时，我们给它惩罚，让它意识到错误，知道再犯错会受惩罚，它就不敢再犯了。
*   **做对要奖励**：当小狗做对时，我们要及时奖励它，让它开心，并告诉它这样做是对的、会得到奖励。小狗就会乐意去做正确的事。

通过这个故事可以发现，并不是小狗不听话，而是没有人告诉它什么才是正确的事。其实跟训练小狗一样，你的伴侣也可以用这样的方式去引导。

当你的伴侣犯错时，我们要告诉他：“你这样做是错误的。”接着要进行适当的惩罚。

## 奖励环节不可或缺

到这里，可能有女生会问：“为什么当我惩罚老公、冷淡他时，他还会继续犯错呢？”

因为你缺少了一个关键环节：**奖励**。如果你在相处中只有惩罚而没有奖励，你的伴侣就会越来越不听话。当对你好的行为没有得到奖励，谁还愿意继续对你好呢？

所以，当伴侣对你好，或做了一些让你开心的事时，你就要及时给他奖励，或告诉他：“你这样做是对的，你真的对我很好。”夸奖之后，要进行一些行为奖励，比如亲他一下，或陪他打一把游戏。

讲到这里，我们就可以知道引导的方法：当伴侣做了对的事，或让你感到开心时，你可以对他说“你对我真好”。通过这句话让他知道这个行为是正确的。之后，我们就要陪对方做一件事，让他感到开心。这样他就知道，下次有动力继续对你好了。

## 一个实用的小技巧

接下来再分享一个实用的小技巧。

操作起来很简单：你只要经常在他面前对他说：“你对我最好了”、“你怎么可以那么好”。你的伴侣就会对你越来越好，因为他已经开心地走入了一个“应该要对你好”的标签设定中。

但如果你每天重复说：“你对我不够好”、“跟你在一起真的不开心”、“跟你在一起很郁闷”，那么他对你也会越来越差，因为他也走进了一个“对你不好的标签设定中。

这个逻辑就像训猴子一样，做对了事，你就给一根香蕉。小猴子就会形成一种条件反射，不停地去做这件事。而且香蕉每次都要给，不能中断。不断反复，这样的反射才强烈、才有效、才会记忆深刻。

本节课中我们一起学习了潜意识引导的核心方法。我们明白了引导对方投资需要利用奖励与惩罚机制，并通过制定规则和及时反馈来强化对方的行为。关键在于，在对方做对时给予正面奖励，在做错时进行适当惩罚，并保持奖励的持续性，从而让对方心甘情愿地为你付出，并越来越离不开你。记住，关系的引导是一个建立正向条件反射的过程。
