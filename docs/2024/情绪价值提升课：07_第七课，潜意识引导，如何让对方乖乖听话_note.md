# 情绪价值提升课：07_第七课，潜意识引导，如何让对方乖乖听话 🎙️

## 概述
在本节课中，我们将学习如何通过潜意识引导让对方乖乖听话，掌握感情中的主导权。

## 潜意识引导的核心
潜意识引导的核心在于引导对方为你付出，从而掌控感情走向。

### 投资与回报
**公式**：投资 → 付出 → 获得回报

在感情中，人们希望获得回报，因此会愿意为对方付出。

### 投资的类型
以下是投资的不同类型：
- 时间投资
- 金钱投资
- 物质投资
- 肉体投资
- 感情投资

## 引导投资的原理
引导投资有两个方法：惩罚和奖励。

### 惩罚
惩罚的目的是让对方改正错误的行为，并按照指定的方向前进。

### 奖励
奖励的目的是让对方开心，从而愿意继续对你投资。

## 训练小狗的故事
以下是一个关于训练小狗的故事，用以说明惩罚和奖励的原理。

- 当小狗犯错时，给予惩罚，让它意识到错误。
- 当小狗做对时，给予奖励，让它开心并记住正确的行为。

## 引导老公的方法
以下是如何引导老公的方法：

1. 当老公犯错时，告诉他错误并给予适当的惩罚。
2. 当老公对你好时，及时给予奖励，并夸奖他的行为。
3. 经常对老公说“你对我最好了”，让他形成对你好的标签设定。

## 总结
本节课中，我们学习了潜意识引导的原理和方法，以及如何通过投资和回报来掌控感情。希望这些知识能帮助你在感情中取得成功。
