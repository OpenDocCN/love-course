# 东方情韵-林金玉极致女神训练营：009.脱垂训练-预防脱垂训练

在本节课中，我们将要学习脱垂训练的第二部分，这是一个针对预防以及改善轻微宫颈脱垂状况的练习动作。我们将详细讲解动作的姿势、呼吸方法以及核心要领。

上一节我们介绍了脱垂训练的基础概念，本节中我们来看看一个具体的调整动作。这个动作的呼吸方法与上一节相同，但身体姿势进行了调整。

特别是对于已经出现轻微宫颈脱垂情况的学习者，必须练习这个动作。完成这套动作后，再进行能量弹弹弓或负重练习会更安全有效。

以下是这个训练动作的具体步骤。

1.  准备姿势：在家中找到瑜伽垫躺在地上，或将身体支撑在沙发或墙面上。建议支撑在沙发上更为方便。
2.  调整体位：将身体靠近支撑面（如沙发），使整个盆骨区域向上抬起。这个姿势能让宫颈自然地向身体内部回缩。
3.  放置工具：在阴道内放入能量弹。能量弹的重力将在后续呼吸中发挥作用。
4.  进行呼吸：共进行三次完整的呼吸，方法与上节课一致。用舌头顶住上颚，鼻子吸气，鼻子吐气。
    *   **第一次呼吸**：吸气时，想象大阴唇、小阴唇整体向内卷入小腹；吐气时慢慢放松。
    *   **第二次呼吸**：吸气时，瞬间将气吸到胸腔；吐气时放松。
    *   **第三次呼吸**：吸气时，将气吸到头顶；吐气时放松。

在吸气的过程中，需要集中意念。核心要点是，在瞬间吸气时，要运用大脑的想象力，配合能量弹的重力，感觉将轻微脱垂的宫颈“吸回”或“拉回”身体内部。

本节课中我们一起学习了针对宫颈脱垂的预防与改善训练法。通过特定的体位调整，结合专注的呼吸与意念引导，可以帮助强化相关肌群，维护阴道内环境的健康。请持续练习以获得最佳效果。
