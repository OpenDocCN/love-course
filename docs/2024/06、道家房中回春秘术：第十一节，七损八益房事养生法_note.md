# 06、道家房中回春秘术：第十一节，七损八益房事养生法_note

在本节课中，我们将要学习源自古代经典《素女经》的“七损八益”房事养生法。这套方法详细阐述了八种对身体有益的房事做法，以及七种可能造成损伤的做法。通过学习与实践，我们可以了解如何通过房事活动来补益身体、治疗小疾，同时避免不当行为带来的伤害，从而达到养生与延年益寿的目的。

## 概述：何为“七损八益”？

“七损八益”的概念最早记载于我国古代经典《素女经》中。书中记载，素女向黄帝指出，男女在房事交合中，有七种做法对身体有损伤，而有八种做法对身体有补益。本节课程将首先介绍对身体有益的“八益”之法。

## 八益：八种有益的房事养生法

以下是八种对身体有补益作用的房事方法，每种方法都有其特定的姿势、动作次数与养生功效。

### 一、固精法 💎
**做法**：令女子侧卧，张开双腿。男子侧躺在女子两腿之间，将玉茎刺入阴道。抽送18次后，停止动作。
**功效与要求**：此法能固守精气。若结合之前所讲的“九浅一深”之法，每日交合两次，约十五日内可见成效。

### 二、安气法 🌬️
**做法**：女子正面仰卧，垫高枕头，伸直并张开双臂。男子跪于其两腿之间，将玉茎刺入阴户，抽送27次后停止。
**功效与要求**：此法能使男子气血平和，缓解烦躁、上火或寒症。对于女子，则可治疗阴道虚寒、手脚冰凉、小腹凉及不孕之症。每日如此性交三次，约十五天左右，上述小毛病可消失。

### 三、利脏法 🫀
**做法**：女子正面躺卧。男子横卧于女子大腿上，从背后将玉茎向上刺入。采用“天龙八部”步伐抽送36次后停止。
**功效与要求**：此法能调和五脏（心、肝、脾、肺、肾）之气，使人脏气平和。同时治疗女子虚寒、手脚冰凉之症。每日交合四次，约二十天病愈。

### 四、强骨法 🦴
**做法**：女子侧卧，左膝弯曲。男子伏在女子身上，采取从正面攻击侧面的姿势，将玉茎刺入抽送45次后停止。
**功效与要求**：此法能调和关节，缓解房事后腰酸腿疼。亦可治疗女子月经不通、不调、紊乱等毛病。每日如此交合五次，约十天病愈。

### 五、调脉法 🩸
**做法**：令女子侧身躺卧，弯曲右膝，伸开左臂。男子撑地，从正面攻女子侧面，刺入玉茎抽送54次后停止。
**功效与要求**：此法能使男子血脉通利，对高血压、高血脂、血液粘稠及冠心病有辅助调理作用。动作需缓慢。同时治疗女子痉挛、受寒、风湿骨病。每日交合六次，约二十天病愈。这符合“少量多次”的养生原则。

### 六、蓄血法 🩸➡️💪
**做法**：男子正面仰卧，让女子臀部跨坐在男子胯骨上。此时男子玉茎正对女子阴门，采取男下女上的被动姿势。女子以阴门下迎玉茎，一坐一起，进行64次后，男子不泄精。
**功效与要求**：此法能令人气力强盛，并治疗女子月经不利。每日如此交合七次，约十天病愈。

### 七、益液法 💦
**做法**：天气炎热时，于凉爽时进行，避免大汗。令女子正面朝下趴着，高抬臀部。男子站于女子身后，玉茎从上方刺入，抽送72次后停止。
**功效与要求**：此法能令人气血充盈。需注意，此类房事活动次数多但每次时间短，意在不断调动性兴奋、勃起，促进荷尔蒙与睾酮分泌，但需收住不泄，避免长时间消耗。

### 八、道体法 🌟
**做法**：女子正面仰卧，两腿弯曲，双脚压在臀下。男子双膝跪于床席，双掌撑席，将玉茎从女子大腿下刺入，抽送81次后停止。
**功效与要求**：此法能让人筋骨气血充实，并治疗女子带下、异味等症。每日做九次，约九天病愈。

## 七损：七种有害的房事做法及矫正

古代先贤对房事认识清晰，认为不宜久战。我们修炼的目的并非追求每次“金刚不倒”，而是讲究顺应自然、养生延寿。若身体存在不适，不妨先利用“八益”之法恢复健康，再图延长房事时间。因为延长需要身体机能和谐，必须遵循从少到多、从短到长的养生原则。

以下是七种不当的、可能损伤身体的房事做法及其矫正方法。

### 一、厥气（精气厥逆） 🥶
**成因与症状**：心中本无性欲而勉强交合。非发自内心喜爱，而是出于占便宜等心态，在身体无反应或勃起不坚时强行行事。导致大汗淋漓、精疲力竭、气短、心烦热、眼睛昏花，损伤肾气。
**矫正方法**：女子仰卧，男子抬起其双腿，将玉茎深深刺入，让女子自己摇动身体，直至阴精流出为止。男子不得泄精。每日如此交合九次，十日病愈。根本在于平时常练“九浅一深”、“天龙八部”及“七窍无归固精法”。

### 二、溢精（精液溢泄） 🫗
**成因与症状**：心理贪求性爱，阴阳之气未调和便急于交合（前戏不足），或中途泄精，或大醉大饱后交合。导致呼吸急促、气息紊乱，伤及肺脏，令人咳嗽、气喘、消渴（血糖紊乱）、易怒、口干身热、久立不稳。
**矫正方法**：女子正面仰卧，男子浅刺女阴，纳入玉茎止半寸（不完全进入）。令女子自己摇摆身子，待女子阴精一出即停止，女子应有高潮。男子以不泄精为原则。每日如此交合九次，十日病愈。

### 三、杂脉（脉气杂乱） 🌀
**成因与症状**：玉茎不坚硬而勉强交合，半途强行，或饱食之后行房。这会伤脾，令人食物不消化，玉茎阳痿且不能射精，脾气不振，食欲低下。
**矫正方法**：女子正面仰卧，用脚勾住男子臀部。男子跪于女子两腿中间，向阴道纳入玉茎，让女子自己摇动身子，待女子阴精一出即止。男子不泄精。每日如此交合九次，十日病愈。

### 四、泄气（气机外泄） 💨
**成因与症状**：劳累疲倦、大汗未干时行房，或大怒、大醉后同房。导致精气衰竭，令人腹中燥热、唇焦口干。
**矫正方法**：令男子正面仰卧，女子骑跨在男子身上，面向男子脚的方向（男子看到女子背部）。女子在男子两腿之间，男子刺入玉茎，让女子自己摇动身子，阴精一出即止。男子不可泄精。每日如此交合九次，十日病愈。

### 五、厥伤（气机厥逆损伤） ⚠️
**成因与症状**：大小便后身体未平定便硬行交合，或仓促间突然行事，动作持续急速。这会伤肝及劳损经络，耗伤气血。导致目光模糊、旧病复发、血脉枯竭，日久可能得偏瘫、中风。
**矫正方法**：令男子正面仰卧，女子骑跨在男子大腿上，双手撑席，面向男子头部。徐徐纳入玉茎，勿让女子摇动身体，等待女子自己达到高潮。男子不可以泄精为快。每日交合九次，十日病愈。

### 六、百闭（百脉闭寒） 🚫
**成因与症状**：与女子交合只为取乐，不能节欲惜精，多次交合而无节度，耗竭精气。即使想强力泄精，也已无精可出。这是过度欲望、力不从心的结果。导致百病齐生、喉干、尿道不通、双眼昏花，严重者可危及生命。
**矫正方法**：道家修炼讲究道法自然、延年益寿，追求的是“金枪不倒”的能力，而非显摆功能。应深藏不露，避免过度强求。具体矫正需从根本上节制欲望，回归养生正道。

### 七、血竭（阴血枯竭） 🩸🔥
**成因与症状**：用力劳作或快跑后大汗，心慌慌张张（恐伤肾），在此情况下勉强行房。导致疾病爆发，连连泻精不止，甚至遗精、尿精。表现为精力不振、免疫力低下、易感冒、腰酸腿疼、脸色发青发黑、皮肤松弛、阴囊潮湿，严重者精液带血。
**矫正方法**：令女子面向上仰卧，高举臀部，伸开双腿。男子跪于其两腿之间，深深纳入玉茎，让女子自摇身子，待其阴精一出即停止。男子不可泄精。每日交合九次，十日观察能否康复。此法在现代需更长时间坚持。

## 总结与核心原则

本节课中，我们一起学习了“七损八益”房事养生法的具体内容。

**核心原则总结如下**：
1.  **目的不同**：“八益”是**养生治病**之法，核心在于 **“男子不泄精”** 与 **“女子阴精出（达到兴奋）”** ，通过调动自身气血与荷尔蒙来调理身体。
2.  **方法差异**：这与追求快感的普通房事**程序不同**，重在次数（少量多次）与特定的姿势、节奏。
3.  **避免损伤**：“七损”列举了各种不当情境（如勉强、疲劳、饱醉、情绪不佳时）下行房的危害，务必避免。
4.  **循序渐进**：身体恢复需过程，应从短时间、少次数开始，以养生为首要目标，逐步增强能力。
5.  **道法自然**：最高境界是顺应自然、延年益寿，而非刻意追求或炫耀房事能力。

总而言之，正确的房事养生应有益健康，避免损伤。结合天气、情绪、身体状况等综合因素，谨慎实践“八益”，远离“七损”，方能在愉悦中达到强身健体、延年益寿的目的。
