# 课程08：夫妻道歉的七种语言之改造语言 🔄

在本节课中，我们将要学习夫妻道歉七种语言中的“改造语言”。这种语言的核心在于，当你意识到自己的行为对伴侣造成了伤害后，除了表达歉意与悔意，最关键的一步是付诸实际行动进行改正。我们将探讨如何从认知到行为实现真正的改变。

---

上一节我们介绍了道歉中表达诚意的重要性，本节中我们来看看如何通过“改造”来落实道歉。

所谓改造的语言，是指当你意识到你的行为对你的爱人造成了伤害。那么你除了要表达出你的诚意，表达出你认识到对对方的伤害，最关键的是你要改正。这种改正要求你对你所做的这些事情感到忏悔。对你所做的这个事情感到后悔。对你所做的事情，真正的要有一个承诺。真正的要有一个落地的行为，就是类似的这种行为不再犯。

同时，就在你内心深处，你感到伤害你的爱人是一件非常错误的事情。伤害你的爱人会导致夫妻之间的感情崩溃。伤害你的爱人会导致爱人可能会出轨，甚至可能会提出来离婚。那么我们说改正的这个语言，其实是对你承诺的一种执行。其实是对你承诺的一种去执行。

以下是现实中常见的几种无效道歉行为，它们缺乏真正的改造：

*   一部分油嘴滑舌的男人，或者一部分女人，老是会说“对不起，这事情我做错了，我以后一定会改的”。有这样的一部分人总是会去这样子去讲，讲多了，别人就不相信你了。
*   还有一部分人，你打死他他也不跟你道歉。他什么事情都不做，就像我辅导的大量案例里面，他出轨了，他就一句话“我回来了”，然后呢咱们啥也别说，咱们好好过日子。结果你发现他没有好好过日子，他还跟外面的女人在那里联系。
*   还有一部分人，他就会说“你看我未来的表现吧，啥也别说了”。你只要一跟他提出轨这个事情，他马上火冒三丈。那你面对这样的人，你会相信吗？你不会相信的。

像我辅导的那个案例，女生自己出轨了，她要取得她老公的谅解。那她要去采取行为。她在这个过程中也会有反复，甚至她不认为她出轨对老公造成了伤害，因为她认为是她老公对她不好，所以她出轨了。我们说确实你可以有各种理由出轨，但是不论你是何种理由出轨，结局都是一样的。一点都不会减轻对方的痛苦。对方就是痛苦。一点都不会说让这个人觉得跟你在一起，未来的日子还要继续这样过。他是希望你改正的。他是需要你改造的。

那么我们说这种改正，这种改造，首先是什么？首先是意识层面的，也就是说认知层面。认知层面你要得到一种改正，得到一种改造。如果你对于这件事情，或者说对于这一类事情，或者说对于这样的一个行为（因为我们说对对方的一些伤害，其实往往是自尊，还有一部分是对于对方的关爱关心缺乏导致），你的认知层面没有达到，你是永远都会犯同样的错误的。

**核心公式：真正的改造 = 认知改变 + 行为落实 + 习惯强化**

就好像我们有一些男性，他就是不尊重女性。跟他老婆做爱，他不做前期的，上来扒拉裤子直接干的。像这一种，认知层面没有发生变化，你说他在行为上面会发生变化吗？不会的。所以说改正，首先就是认知层面发生了一个改正，发生了一个改造。

那么然后呢再是什么？再是你具体的这个行为。你具体去做这件事情，从行为层面来去落实你的认知。否则的话，你认知得再好，你做不到，也等于白认知。当然大量的人其实是认知没有做到，所以说后面的行为更加就谈不上。

那么还有一个叫做强化，也就是说固定化，把你这个良好的行为把它固定住。这个行为意味着要变成习惯。也就是说要重复反复持续的做一阵子。所以说这样子的话才是真正的一个改正。否则的话，你其实没有达到这样的一种改正。

大家要知道，如果是说你真的把它改正了，其实你的爱人是会觉得非常好的。真正的改正了，你的爱人是会真的感受到你的道歉是有意义的。而且他会觉得他能够宽恕你，他能够原谅你。他觉得心里有底。

---

本节课中我们一起学习了夫妻道歉语言中的“改造语言”。我们了解到，真正的道歉不止于言语，更需要深刻的认知转变、具体的行为落实以及长期的坚持强化，从而形成新的习惯。只有这样，你的伴侣才能感受到道歉的诚意，并重建对关系的信任与安全感。
