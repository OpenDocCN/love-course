# 《二月老师的核心自信》课程27：行为模式3_ev - 掌控意识焦距 🎯

在本节课中，我们将要学习如何通过控制意识的“焦距”来主动调整我们的情绪和状态。上一节我们介绍了通过生理控制情绪的方法，本节中我们来看看如何通过改变注意力的焦点来改变我们的感受。

## 概述：意识如何影响感受

我们体验到的身体感觉，正是我们集中注意力的部分。我们将焦点放在哪里，我们的意识捕捉画面就会自动停留在那个焦点上。其次，我们还能够控制如何调整意识的焦点，例如心中画面的范围、脑中最明亮的色彩、声音的大小。如果我们改变注意事物的方式，就能够立刻改变自己的状态。

## 意识如同照相机 📷

如果你的脑子是一台照相机，而你要去参加一个宴会，你只会对你所注意到的事情有所感觉。整个宴会中有很多活动、情绪、对话以及许多事物在同时进行。但是，你对这个宴会的感觉，却是根据你脑子里的照相机所对到的焦点所产生的感觉。

以下是几个例子，说明焦点如何决定感受：

*   **焦点在冲突**：如果现在焦距在两个起冲突的人身上，宴会结束后，你会说整个宴会的气氛很紧张，很不好。
*   **焦点在美好**：如果你的焦距是在两个相恋的年轻人热情相拥，那么你会说这个宴会的气氛很棒。
*   **焦点在无聊**：如果你的焦距放在那些觉得无聊的人身上，那么你会说没有一个人会喜欢这个宴会。

问题就在于，你的相机（意识）的焦距并未同时捕捉到所有的事情，你只能捕捉一个画面。所以说，你的意识焦距决定了你的感受。

## 我们如何筛选信息

因为我们无法注意到每一件事情，并且有意识地观察所有事物，所以我们都会成为删除信息的动物。我们会自动删除一些每天看到的事情，而挑选一些对我们更有利，或是我们更为感兴趣的事情，这是进化所致。

以下是意识筛选信息的倾向：

*   在任何聚会里，我们会自动把焦距投放到那些高价值的人身上，而忽略那些低价值的人。
*   如果一个高价值的人和一个低价值的人跟你同时说着同一件事情，那么你的注意力基本上都会放在高价值的人身上。

更糟糕的是，照相机（意识）还会近距离对焦，它会让物体比实际来得更大、更亮。你的注意力也是如此，在你生命的宴会当中，总会有一些事物让你觉得很不高兴。很多人似乎养成了一个习惯，就是专门盯着不好的地方去看，注意点就永远在那里，然后不断地给自己的宴会添堵。

相反的，在任何一个宴会、任何一段时光，总是有一些人度过美好的时光，因为他们有意识地控制了自己的焦点。哪怕是一些很小的事情，他们会让那些很小的事情变大变亮，然后让自己心情变好。

## 如何控制意识焦距：提出好问题 ❓

决定你把焦点放在哪里，是你如何评价事物的方式，这将决定了我们的感受。如果我握着拳头，回到离你脸不到2米的距离，你的脑子就在评估：这什么意思？我要怎么做？我要离开吗？我要跟他打一架吗？你的脑子就在评估，并开始搜寻“这是什么意思”的信念。也就是说，你问的问题，决定了焦点所在。

每个人的大脑都是问题与答案的集合体。如果你问你自己“为什么这些事情老是发生在我身上？”，那么你的脑子会搜寻所有的资料（一生中什么人对你说过什么话、你对自己说过的话、你读到的信息、你经历的事情等等），从中搜寻出一个答案。紧接着你会觉得很糟糕，于是你的情绪在不断下降。

**核心公式：`你问的问题 -> 大脑搜寻答案 -> 产生对应的感受和状态`**

换句话说，我们的问题决定了我们的感受。

*   **坏的问题示例**：“为什么我这么胖？” 你的脑子会启动，寻找你肥胖的原因（因为你不断地吃，你毫无节制等），接着你会感觉自己很没用，情绪变得低落。
*   **好的问题示例**：“我如何既能享受美食又能变得苗条呢？” 这个问题会引导大脑寻找解决方案。

这里就是更改你的神经联想模式。你的生活品质，与你每天问自己的问题的品质有关。

> 这不是打鸡血，也不是“积极思维”。积极思维试图催眠自己进入一种不同的状态（如“我很棒”），这有点像秃头看着镜子念“我有头发”。我们提倡的是，在你继续沉沦之前，提出合理的问题来调整状态。这是引导自己的大脑在正确的文件抽屉里搜寻文件，这样你会找到促使你前进的答案，而不是阻碍你前进的答案。

## 好问题的三大作用

以下是提出好问题能带来的三个关键改变：

1.  **改变意识焦距与状态**：好的问题办到的第一件事是改变我们的意识焦距，以及能够立刻改变我们的状态。例如问：“什么是很棒，而我没有注意到的部分？” 如果你抱着诚挚的态度问自己，你一定会得到一个答案。

2.  **改变我们删除的信息**：好的问题能够做到的第二件事是它改变了我们想删除的事情。当你要觉得不高兴，你会将焦点放在你不高兴的事情上，并把生活中发生所有的好的事情都删除掉。你可以把焦点放在生活的美好、事业的顺利、爱你的人、正在呼吸新鲜空气、正在晒太阳等等。而唯一的挑战是你必须用问题去控制。问题改变我们删除的事物，人类不会停止删除事情，我们为什么不去删除那些不好的事情呢？

3.  **激发内在潜能**：问题的第三件事就是帮助你获得你内在的能源，相信你所要达成的每一件事，你早已经有内在的能源可以让它们实现。例如，问“我能做得到吗？”会造成怀疑的感觉，这种感觉会让你远离成功。而问“我要怎么去实现这件事，并且有趣地完成它？”，你的脑子就会挖掘你更多的能力。当然，事情最终是否能够完成，考量的是多方面的因素（天时、地利、人和），但是这样的问题不仅会保持你更好的状态，也会让你在不断成长。

**代码示例：两种问题导向的思维模式**
```python
# 消极的问题循环
question = "我能完成吗？"
brain.search(question) # 大脑搜寻关于“怀疑”和“不可能”的证据
result = "犹豫、恐惧、停滞"
state = "低落"

# 积极的问题循环
question = "我如何才能完成？"
brain.search(question) # 大脑搜寻关于“方法”和“步骤”的答案
result = "步骤一、步骤二、步骤三..."
state = "专注、有动力"
```

所以，好的问题能控制你的意识焦点、情绪和状态。坏的问题也会控制你的焦点、情绪和状态。如果你想要更好地控制自己，那么就要学会问自己好的问题，而不是问一些负面的问题。同时，问题需要现实可行，例如“我如何完成这个学习任务？”而不是不切实际的幻想。

## 总结

本节课中我们一起学习了行为模式中的关键一环：通过控制意识焦距来管理情绪。我们了解到，感受取决于注意力的焦点，而焦点可以通过我们向自己提出的问题来主动引导。记住，提出如“如何实现？”、“有哪些好的部分？”这类建设性问题，能够将我们的注意力转向解决方案和积极面，从而立刻改善我们的状态，并激发内在的行动力。
