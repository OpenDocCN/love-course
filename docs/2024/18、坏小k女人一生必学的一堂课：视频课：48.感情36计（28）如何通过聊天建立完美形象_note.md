# 18、坏小k女人一生必学的一堂课：视频课：48.感情36计（28）如何通过聊天建立完美形象

🎼感情36计是第28个技巧，聊天的核心是塑造形象，而非产生交流。如何通过聊天建立一个完美的形象。

### 一、聊天细节

1. 给对方起一个别人没有叫过的昵称。
2. 在适当的时机说出“咱俩都多大人了，还玩这个你可千万不能说出去丢人”这类话，效果爆炸。
3. 多次提到“咱们例句咱俩都多大人了”。
4. 深层反向预测，很管用。根据对方在人前的显性性格来反向压住，不管事实上有没有猜对，但对方一定会觉得你比别人更了解他。

### 二、频繁叫对方的名字

频繁的叫对方的名字，或是你之前给对方起的代称，而不是用爱来代替。

### 三、学会高价值离场

聊不下去就尝试高价值离场，而不是一味地没话找话去尬聊。这个就是当你感觉自己聊不下去了，不知道说什么了，就说一件比较高价值的事情，借机离开。

### 四、不解是不证明不对抗

框架：不解是不证明不对抗。

### 五、同频率沟通

和对方聊天的时候，保持同频的沟通，包括回复时间、次数、情绪和投入。

### 六、聊天沟通时的误区和注意事项

1. 不要无端释放关心，不要以关心对方的方式去进行开场。
2. 不要迷恋对方的认同，在与对方相处的时候，一定要有底气，不能虚。
3. 尽量别向自己喜欢的人征求意见，反而要在自己喜欢的人面前传递出独立自强的品质。
4. 社交软件沟通的时候，要掌握对应原则。
5. 尽量少问问题，如果有必要用疑问句代替陈述句。
6. 能用一个字表达清楚的，绝不使用两个字。
7. 一次只发一条消息，切记追命连。
8. 当得不到男生及时回复时，千万别去追问。

### 七、无限话题制造

以十字纵横思维，比如男人说“我很喜欢旅游，曾经在海边旅居了很长一段时间，算得上是我度过的最美妙的时光了”。这句话包含了哪些内容呢？

1. 旅游：你喜欢旅游还是冒险，还是喜欢去某个地方，亦或只是喜欢大海带给你的感受。
2. 海边：你最喜欢大海的什么地方，是蔚蓝的海水，还是细软的沙滩感受。
3. 感受：待在海边，让你感觉到心是宁静的。

通过这种思维，聊天的内容无所不包。
