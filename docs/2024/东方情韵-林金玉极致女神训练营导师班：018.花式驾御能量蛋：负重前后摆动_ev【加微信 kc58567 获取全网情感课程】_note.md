   **公式**：盆底肌收缩 = 意念想象 + 收紧尾骨 + 提起蛋蛋

🎼2. **摆动控制**：随着盆底肌的收缩，负重物会前后摆动。此时，跟随节奏进行摆动。🎼

🎼3. **练习方法**：找不到感觉时，可以轻轻晃动负重物，使其前后摆动，然后收缩盆底肌进行训练。🎼
   **代码**：晃动负重物 + 收缩盆底肌
