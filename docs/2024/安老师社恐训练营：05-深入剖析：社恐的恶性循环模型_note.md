# 安老师社恐训练营：05-深入剖析：社恐的恶性循环模型 🌀

在本节课中，我们将要学习社交恐惧中一个核心的心理模型——恶性循环模型。理解这个模型是打破社交恐惧、实现改变的第一步。

## 概述：从生活例子到社恐模型

我们都有过类似的经历：计划周末读书，却刷手机停不下来，最终因未完成计划而自责。这种“定目标-未完成-更自责”的模式，就是一种恶性循环。潜意识会逐渐接受“计划可以不完成”的信号，形成牢固的坏习惯。

改变的方法很简单：**打破恶性循环**。例如，刷手机时立刻停止，转而去看书。完成计划的行为会在潜意识里种下“我能完成目标”的种子。重复多次后，你会获得成就感，而非愧疚感。

**一个动作就可以让结果截然不同，这个动作就是打破之前的恶性循环。**

回归到社交恐惧，同样的道理。因为存在恶性循环，所以每次面对同类社交场景时，你都会紧张、焦虑、恐惧。我们要做的，同样是**打破恶性循环**。但首先，必须弄清楚这个循环到底是什么。

**要想解决一个问题，首先得把问题弄清楚、分析清楚。**

我们今天的目标，就是深入解析社交恐惧的恶性循环模型。

## 思维的核心作用：认知的三个层次

从小到大，我们常听到“思维”这个词。在社交恐惧中，个人想法起了核心作用：**想法决定你的感受，决定你的行为**。

那么，什么是思维？思考今天吃什么、今天的工作是什么、怀疑“那个人是不是看我了”，甚至很多朦胧的、我们未意识到的想法，都算是思维。

其实，思维或认知可以分为由浅入深的三个方面：
1.  **注意力水平**
2.  **自动化的思维水平**
3.  **深层次的潜在信念与猜想**

之所以这样细分，是为了更好地解决问题。这里提供一种重要的思维模式：**将复杂问题拆解成多个简单问题**。例如，编写一个网站不能只想着最终漂亮的页面（结果），而应拆解为搭建结构、设计头部、主体、底部等可执行的小步骤。这种分解思维至关重要。

好，让我们回到认知的三个层次。

### 1. 注意力水平：你的心在何处？

人们注意到的事情，就是心里想的事情。对于社恐者来说，更倾向于**将注意力集中在自己身上**。

例如，一个有社交恐惧的人（假设叫刘云）向老板汇报工作时，突然感觉自己脸红、发热、出汗。他认定自己脸红了，非常紧张焦虑，于是用头发遮挡脸部。整个过程中，他无法听清老板的话，也不知道自己是否做错了事。

我们发现，当刘云开始焦虑时，他注意到的是自己“脸红”的内心感受。**当人的注意力在哪儿的时候，他的心就在哪。**

### 2. 消极的自动化思维：不自知的悲观滤镜

自动化思维，指任何发生在我们身上的事都会自动引发的思考，很多时候我们并未意识到。

更重要的是，这种不自知的想法会影响我们如何评估遇到的事情。有时想法正确，但有时会过于悲观、消极。

以下是消极自动化思维的几个例子：
*   陌生人冲你笑，通常认为对方友善，但悲观者会觉得对方只是做做样子。
*   聚会时，会想“他们是不是觉得我很傻、很无聊”。
*   总是负面看待自己，觉得自己总做错事，把自己贬得一无是处。

**这些并不是你主动这么想的，是你自动化思维这么做的。**

### 3. 深层次的潜在信念与猜想：指导生活的“潜规则”

这可以简单理解为潜意识里对自己、他人和世界的看法，是人长久以来的生活理念或态度，无需用语言描述。

例如，你觉得一个人“人品不错，值得信任”，这不是通过语言描述得出的，而是经过对其言谈举止、态度经历等大量信息综合后，在深层次形成的看法。

**猜想**则是在此基础上构成的生活准则。例如：
*   **信念**：这类人人品好。
    *   **猜想**：如果我真诚对待他们，他们也会真诚待我。
*   **信念**：这类人人品差。
    *   **猜想**：如果我过多暴露自己，他们会抓住并利用我的缺点。
*   **信念**：我无法改变自己，我很愚蠢。
    *   **猜想**：如果一段对话很糟糕，出错的一定是我。
*   **信念**：人们都不喜欢紧张、不安、害羞的人。
    *   **猜想**：我必须尝试多说话，否则别人会觉得无趣、不喜欢我。

**信念不是你主动去想的，而是深层次的潜在想法。这些猜想就像生活的规则一样指导着你。**

## 安全行为：饮鸩止渴的自我保护

人是趋利避害的。在思维的引导下，社恐者在面对令自己焦虑的社交场景时，会做出**安全行为**，以获取安全感。但这种安全感是通过逃避和避免完成的。

以下是安全行为的例子：
*   与人相处时，避免眼神交流，注视地板。
*   会议一结束就立刻离开，避免会后闲聊。
*   不经常发表个人意见，怕出错。
*   不谈论有风险的事，唯唯诺诺，讨好他人。
*   家庭聚会时去刷碗，避免留在饭桌上闲聊。
*   只跟感到安全的人谈话，或只聊安全的话题。

**安全行为是社交恐惧者做出的、让自己感到安全的行为，其本质是试图避免引起别人的注意。**

## 社交焦虑模型详解：恶性循环如何运转

1995年，临床研究者戴维·克拉克和阿德里安·威尔斯提出了一个**社交焦虑模型**。该模型有可靠研究支持，能很好地说明如何克服社交恐惧。我们理解即可，无需死记硬背。

该模型的核心就是我们刚才讲的**三种层次的思维**：
*   **自我关注**：社恐者在意或关注的事（注意力水平）。
*   **消极自动化思维**：主观认为社交场合可怕，或将其解读为具有危险性。
*   **潜在的信念和猜想**：在特定场景被触发。

当一个人主观认为某个场景具有危险性时，会产生两种直接后果：
1.  做出**安全行为**。
2.  表现出**社交恐惧的迹象**（如手抖、出汗、声音发颤）。

### 安全行为如何加剧恶性循环？

模型指出，安全行为主要通过三种方式引发并加剧恶性循环：

**1. 强化错误信念，掩盖事实**
安全行为会使同一场景反复引发恐惧，**肯定一个人的潜在错误信念和猜想**（如“我果然不行”），并加剧“必须采取安全措施，否则会出事”的想法。你一直使用安全行为，甚至无法认识到那些所谓的“威胁”其实完全无害，你的安全行为毫无意义。

**2. 增强自我关注，放大危险感**
当一个人将注意力放在自己身上（内部）时，他的自我觉知会增强，这会使社交场合看起来更可怕。例如，你通过低头避免眼神接触（安全行为），但这意味着你将注意力放在了内心的焦虑上，忽视了外部实际情况。**自我意识越强，你越会觉得场合危险**，因此安全行为反而让情况更糟。

**3. 加剧恐惧表现，形成负面印证**
安全行为会让你更有压力、更紧张。例如，当你努力想让自己的表现不沉闷时，反而更难想出幽默的话。安全行为加剧了紧张，导致你的表述更不自然。同时，你表现出的紧张迹象（如手抖），会进一步**增强你的自我意识**（“别人一定看到了”），从而觉得场合更加危险，形成负面循环。

**有趣的是，这些迹象和表现恰恰是社恐者最关注的。**

## 模型应用实例

下面用两个例子揭示这个模型在实际生活中如何起作用。

### 实例一：参加聚会
*   **潜在信念**：我不属于他们，我与他们不同。
*   **自动化思维**：我到时候一定想不出说什么，我没办法很好地接近别人。
*   **注意力**：感到威胁，将注意力放到自己身上，发现脑子一片空白，非常注意谈话时的沉默，觉得尴尬。
*   **安全行为**：避免眼神接触，不敢直视别人，尽量不多说话，确保自己不出错。
*   **身心反应**：内心紧张不自在，焦虑加剧，感到害怕、惊慌、出汗，无法保持思路清晰，甚至开始说胡话。
*   **结果**：无论过程如何，最终都强化了“类似场景有潜在危险”的印象，并再次证实了“我不属于他们”的信念。

### 实例二：与半生不熟的人聊天
*   **潜在信念**：我很内向，不擅长聊天。
*   **自动化思维**：我不知道说什么，好尴尬。
*   **注意力**：关注自身，感觉大脑混乱，担心表情不自然。
*   **安全行为**：眼睛时不时看地板，不敢眼神接触，身体做出小动作（如来回动），潜意识想撤离场景。
*   **身心反应**：更加紧张、出汗，无法放松自己。
*   **结果**：安全行为再次证实了“我很内向，不擅长聊天”的信念。

## 总结与下节预告

本节课中，我们一起学习了社交恐惧的**恶性循环模型**。

我们首先了解了思维在社恐中的核心作用，并将认知分解为**注意力水平、消极自动化思维、深层次潜在信念与猜想**三个层次。接着，我们认识了社恐者为获取安全感而采取的**安全行为**，但其本质是逃避。

最后，我们深入剖析了**社交焦虑模型**，看到“威胁感知 -> 安全行为/恐惧表现 -> 强化错误信念与自我关注 -> 加剧威胁感知”这一完整的恶性循环是如何在聚会、聊天等具体场景中运转的。

**想要克服社交恐惧，必须打破这个恶性循环模型，避免问题不断发展。**
