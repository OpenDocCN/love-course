# FreeMan计划【做情感自由的真男人】课程07：高维情商聊天课 - 读懂对方潜意识 🧠

## 概述

在本节课中，我们将要学习聊天回复的核心思考流程。很多人在聊天时，面对对方的一句话，常常不知道该如何回复。我们将通过一套系统化的四步法，帮助你摆脱依赖，学会在任何情况下都能独立思考并给出恰当的回应。

## 第一步：明确当下关系

上一节我们介绍了聊天回复需要系统思考，本节中我们来看看第一步——明确当下关系。

在我进行咨询时，首先会询问学员与对方的认识过程、经历和近期聊天内容。这样做的唯一目的是判断双方当前的关系处于哪个阶段。必须先判断关系，否则无法进行有效沟通。

在中国，有一个词叫“交浅言深”，指的是两人关系尚浅时，不宜谈论过于深入的话题，否则会让对方感到冒犯。这是聊天中的最大禁忌之一，适用于所有社交关系。

例如，在聚会上刚认识的新朋友突然问你：“兄弟，你一个月赚多少钱？” 你会感到别扭，因为收入是隐私话题，刚认识就聊属于“交浅言深”。

但如果换成多年未见的大学挚友问同样的问题，你可能只会调侃回去，不会感到被冒犯。同样的话，在不同关系中效果完全不同。

再比如，网上统计显示，女生最反感的话之一是“发张照片来看看”。这句话本身没问题，但如果在刚加上好友、彼此不熟时提出，就会让人感到冒犯，这也是“交浅言深”。

通过以上例子，你应该明白了为何不能“交浅言深”，以及明确当下关系的重要性。你们当下的关系决定了你说话的尺度和边界。

那么，如何判断关系呢？很简单，我将男女关系分为四个阶段。

以下是四个关系阶段的定义：

1.  **初期认识阶段**：刚加上联系方式或开始聊天，彼此不熟。聊天目标是消除陌生感，话题通常轻松，如天气、路况、对彼此的印象。
2.  **建立吸引阶段**：彼此已有些熟悉，开始交换更深入的信息，如工作、兴趣爱好、生活习惯。目标是展示魅力，建立吸引。
3.  **升高关系阶段**：对方已对你产生基础兴趣。话题需更深入，以推进关系、制造暧昧气氛，例如聊感情观、未来期待。处理得好甚至可确立关系。
4.  **维护关系阶段**：关系已达标志性节点（如确定关系、成为普通朋友或保持暧昧但暂时无法见面）。目标是进行长期关系的维护与巩固，保持愉快聊天气氛。

所有聊天回复之前，你都需要先对照以上阶段，判断你们处在哪个关系阶段。这是聊天回复的第一步。

## 第二步：读懂对方意图

明确了关系与聊天尺度后，我们进入第二步：读懂对方意图。

两性思维存在差异，女性说话有时会拐弯抹角。能适应这种差异、理解对方言外之意的人，往往在社交中更具优势。因此，想要聊好天，必须具备读懂对方话语背后含义的能力。

例如，互有好感的两人聊天，一方问“在干嘛？”，其潜台词往往是“我想你了”。再比如，吵架后女友说“下班不用来接我了”，真实意图是“还不快来哄我”。

在这些场景中，只要能读懂意图，就能做出正确回应。如何迅速读懂对方意思呢？我教你四个方法。

以下是四个读懂意图的方法：

1.  **反推测法**：当你猜不到对方潜在意思时，可以反推其真实意图。例如，朋友对你说“我觉得还行，如果你喜欢的话，可以买”，这通常意味着他并不真心推荐。如果真心觉得好，他会说“挺好看的，适合你”。再如，第一次约会你提议吃全聚德，女生说“换一个吧，我觉得全聚德也没那么好吃，而且还有点贵”。字面意思是嫌弃贵和难吃，但反推测一下：若她真这么想，会更直接地说“全聚德特别坑人”。她的真实意图可能是觉得第一次见面吃这个太贵，有些不好意思。
2.  **找目的法**：思考对方做这件事的根本目的或利益所在。例如，面试官问“你喜欢出差吗？”，你不必猜测他考察什么，而是想：在这个情境下，他的根本利益是找到能胜任工作（包括出差）的人。从根本利益推测通常不会错。
3.  **先后顺序判断法**：注意对方话语的前后顺序，通常后面部分是重点。例如，评价“你平时虽然抠门，但是到大事上从来不含糊”是夸赞；反过来说“你在大事上其实还行，但是平时太抠了”则是批评。约女生看电影，若她说“好呀，我有点想看了，但是我现要帮我妈做卫生……等我做完再联系吧”，肯定在前，否定在后，赴约概率低。若她说“哎，我正在帮我妈做卫生呢，等我做完再联系，我也有点想看电影了”，顺序调换，赴约概率就高。
4.  **反问法**：当你实在不明白对方意图时，可以用反问来确认，以避免出错。例如，女生说“亲爱的，我肚子有点疼”，如果你不确定该做什么，可以问“亲爱的你是希望我能帮你做点什么吗？”，这比直接说“多喝热水”要好得多。

学会这四个方法，能有效帮你读懂对方意图。这是摆脱“直男”思维的关键一步。读懂意图需要经验积累和练习，随着后续学习，你会对女性的聊天习惯有更深理解。

以上是第二步，读懂对方意图。

## 第三步：认清回复方向

明确了关系，读懂了意图，接下来我们进行第三步：认清回复方向。

很多时候，当你搞清楚前两步后，回复方向会自动浮现。例如，女生问“我和你妈同时掉水里，你先救谁？”。能问出这话，说明关系处于升高或维护阶段。她的意图是测试你是否爱她、关心她。明白了这两点，回复方向就很清晰。

如果你想推进关系，回复就要让她感受到关心和偏爱，例如：“如果必须牺牲一个，那我就牺牲我自己，把你们俩都救上去。因为你和我妈一样，都是我生命中最重要的人。”

如果你不想推进关系，可以让她觉得你“直男”一些，例如：“胡说什么呢？你能不能别有事没事咒我们。”

所以这一步通常不难。但在某些特殊情况下，我们会陷入困境。

例如，女生说：“哎，这两天又被老板骂了，感觉我要失业了。” 她的意图显然是寻求安慰、支持和鼓励。但如果你了解她工作能力确实差，一味安慰可能导致她真的失业；若直接讲道理，她又会觉得你直男。

面对这种困境，回复方向是：既要照顾情绪，又要引导解决问题。你可以说：“你们老板就是有眼无珠，没事儿啊，等你失业了，我陪你一起要饭。” 先共情、调侃，刺激她的焦虑感。等她情绪发泄、理性恢复后（例如她回应“我才不去要饭呢”），你再引导她理性看待工作问题，就不会显得直男了。

这种特殊情况较少见。大多数时候，读懂意图后，回复方向自然出现。这一步相对简单。

## 第四步：斟酌回复措辞

接下来是四步法的最后一步：斟酌回复措辞。

这一步有时完全不需要，有时却至关重要。处理不好，可能让前功尽弃。这取决于你平时的聊天素养。如果你还是新手，一定要特别注意这个环节。

例如，同学聚会聊到家里老人，一位同学说他爷爷90多岁身体很好。另一位同学脱口而出：“我靠，你爷爷竟然还活着呢！” 大家哄堂大笑。他本意是想感叹长寿，但因没斟酌措辞闹了笑话。

再如，我和朋友聊工作，我说自己在写剧本。朋友说：“连你都写上剧本了。” 我有点尴尬，问他是否觉得我写得烂。他赶忙解释：“不是啊，我是觉得你时间管理太强了，又要做课程还要做咨询，竟然还能挤出时间写剧本。” 这也是典型的措辞不当。

这两个例子本意都是好的，回复方向也对，但就因为没斟酌措辞，造成了误会或笑话。

所以在聊天时，我们不仅要找对方向，还要思考如何避免误会，把话说到对方心坎里。斟酌措辞并不难，重点是要花时间思考。大多数错误都是想到了方向就脱口而出，完全不思考导致的。

有些朋友可能觉得词汇匮乏，不知如何措辞。这通常不是积累问题，而是思维出现了阻碍。在后续课程中，我会带你突破这个阻碍。只要你能给自己一点时间思考，就能更好地措辞。

## 总结

本节课中，我们一起学习了聊天回复的四步思考流程：

1.  **明确当下关系**：判断双方关系阶段（初期认识、建立吸引、升高关系、维护关系），确定聊天尺度。
2.  **读懂对方意图**：运用反推测、找目的、先后顺序判断、反问四个方法，理解对方话语背后的真实含义。
3.  **认清回复方向**：基于前两步，确定回复是推进关系、维护关系、提供情绪价值还是引导解决问题。
4.  **斟酌回复措辞**：花时间思考，确保表达准确、得体，避免误会，把话说到对方心里。

和任何人在任何情况下聊天，只要你按照这四个步骤思考，就都能得出想要的答案。抓紧练习起来吧。我们下节课见。
