# 乐嘉7套：5.乐嘉性格色彩全能训练营（入门）【勇略】：33 TA会不会记仇，从性格就能看出来！

## 概述
在本节课中，我们将探讨不同性格的人是否容易记仇，以及记仇对个人和社会的影响。

## 记仇与性格
以下是不同性格的人记仇的特点：

### 绿色性格
- **特点**：宽容大度，不易记仇。
- **例子**：广东省江门一位108岁的老太太，心地善良，不争强好胜，不记仇。
- **公式**：绿色性格 + 宽容 = 不记仇。

### 黄色性格
- **特点**：记仇，但会等待机会反击。
- **例子**：美剧《纸牌屋》男主角Fci，被算计后开始连环复仇计划。
- **公式**：黄色性格 + 仇恨 = 等待机会反击。

### 蓝色性格
- **特点**：记仇，一旦记仇，烙印在心里，很难磨灭。
- **例子**：韩国电影《老男孩》，男主角因同学乱伦事件记仇，多年后实施复仇计划。
- **公式**：蓝色性格 + 仇恨 = 难以磨灭的烙印。

### 红色性格
- **特点**：记仇，但时间不长，心态乐观。
- **例子**：鲁迅与胡适，鲁迅曾批评胡适，但胡适仍愿意帮助鲁迅出版作品。
- **公式**：红色性格 + 仇恨 = 时间不长，心态乐观。

## 总结
本节课中，我们学习了不同性格的人记仇的特点。建议大家在生活中多学习红色和绿色性格的做法，不计较仇恨，让自己过得更加从容。
