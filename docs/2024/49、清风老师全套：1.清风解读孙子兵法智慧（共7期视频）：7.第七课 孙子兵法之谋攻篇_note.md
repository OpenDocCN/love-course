# 课程49：孙子兵法智慧解读 - 第七课 谋攻篇 🎯

在本节课中，我们将要学习《孙子兵法》中极为重要的“谋攻篇”。这一篇的核心思想是“不战而屈人之兵”，强调通过谋略和智慧取得胜利，而非直接对抗。我们将深入探讨这一战略概念，并结合实际生活案例，帮助大家理解如何将其应用于情感、家庭和人际关系中。

---

## 原文解读与核心概念

孙子曰：“夫用兵之法，全国为上，破国次之；全军为上，破军次之；全旅为上，破旅次之；全卒为上，破卒次之；全伍为上，破伍次之。”

这段话的意思是：用兵打仗的方法，能够完整地保全敌国并使其屈服，是最上等的策略；通过攻破敌国来取胜，则是次等的策略。同样，保全敌军、全旅、全卒、全伍并使其屈服，也比击破它们更为高明。

**核心公式**：  
**胜利的最高境界 = 不通过直接战斗而使对方屈服**

---

## “不战而屈人之兵”的战略思想

上一节我们介绍了“谋攻篇”的基本理念，本节中我们来看看其核心战略思想。

孙子提出：“故百战百胜，非善之善者也；不战而屈人之兵，善之善者也。”

这句话的意思是：即使百战百胜，也不是最高明的；能够不通过战斗就使敌人屈服，才是最高明的。

**代码描述**：  
```python
if 百战百胜 == True:
    print("非善之善者也")
elif 不战而屈人之兵 == True:
    print("善之善者也")
```

在情感关系中，这意味着：  
- 频繁争吵并取得表面胜利（百战百胜），并非真正擅长维护关系。  
- 通过自身价值和魅力，让对方主动关心和付出（不战而屈人之兵），才是最高境界。

例如，在分离第三者时：  
- 不断指责和抱怨（百战百胜）可能暂时有效，但无法根本解决问题。  
- 掌握家庭资产，提升自身价值，让第三者因无利可图而主动退出（不战而屈人之兵），才是持久胜利。

**核心观点**：忠诚不是人品的体现，而是权衡利弊后的选择。要让对方持续忠诚，必须确保他留下的收益足够大，离开的成本足够高。

---

## 实现“不战而屈人之兵”的四层策略

了解了核心思想后，我们来看看如何实现“不战而屈人之兵”。孙子给出了四层策略：

“故上兵伐谋，其次伐交，其次伐兵，其下攻城。攻城之法，为不得已。”

以下是这四层策略的具体解释：

1.  **上兵伐谋**：通过谋略让对手放弃想法。  
    - 在情感中，这意味着提升自我价值，掌握资源分配权。例如，掌握家庭资产，让对方因成本过高而不敢背叛。

2.  **其次伐交**：通过外交手段解决问题。  
    - 例如，在婆媳关系中，先给予好处（如发红包），再在对方态度不好时撤回，以此调整双方地位。

3.  **其次伐兵**：与对方正面冲突。  
    - 这是次等选择，消耗大且效果有限。例如，在感情中不断争吵，即使赢了一时，也可能损害关系。

4.  **其下攻城**：直接攻击对方核心，代价最大。  
    - 这是最后的手段，应尽量避免。例如，在分离第三者时，公开对抗或激烈冲突往往两败俱伤。

**关键点**：要成为资源的分配者，而非被分配者。通过提升自我价值，掌握主动权，才能在关系中立于不败之地。

---

## 避免持久战的智慧

上一节我们探讨了四层策略，本节中我们来看看孙子对持久战的警告。

“修橹轒辒，具器械，三月而后成；距堙，又三月而后已。将不胜其忿而蚁附之，杀士三分之一而城不拔者，此攻之灾也。”

这段话的意思是：准备攻城器械需要三个月，堆筑攻城的土山又要三个月。将领因愤怒而命令士兵像蚂蚁一样爬城进攻，即使士兵伤亡三分之一，城仍攻不下，这就是攻城的灾难。

在情感和生活中，这意味着：  
- 避免被烂人烂事拖住，消耗时间和精力。  
- 如果当前关系无法改善，应迅速决断，抓住自己能掌握的资源，及时止损。

**行动准则**：不要为不值得的人或事浪费一秒钟。能抓住的现有资源，远比虚无的未来承诺更重要。

---

## 谋攻的最高境界

最后，我们来看看谋攻的最高境界是什么。

“故善用兵者，屈人之兵而非战也，拔人之城而非攻也，毁人之国而非久也。必以全争于天下，故兵不顿而利可全，此谋攻之法也。”

这段话的意思是：善于用兵的人，使敌人屈服而不通过战斗，夺取敌人的城池而不通过强攻，毁灭敌人的国家而不通过持久战。一定要以全胜的姿态争夺天下，军队不受挫而利益得以保全，这就是谋攻的方法。

在实际应用中：  
- 通过提升自我价值（上兵伐谋），让对方主动退出或屈服。  
- 例如，在分离第三者时，掌握家庭资产并提升自身魅力，让第三者因无利可图而主动离开。

**案例分享**：  
一位学员通过婚内财产协议掌握家庭资产，再通过自我提升和撒娇，让第三者主动退出。这就是“不战而屈人之兵”的成功实践。

即使在困境中（如全职妈妈面对丈夫出轨），也可以通过提升自我价值（如上社交软件、努力打扮、认真工作）实现“上兵伐谋”，吸引新的机会，从而跳出困境。

## 总结与下节预告

本节课中，我们一起学习了《孙子兵法·谋攻篇》的核心思想：

1.  **最高战略**是“不战而屈人之兵”，强调通过谋略而非直接对抗取胜。  
2.  **四层策略**（上兵伐谋、其次伐交、其次伐兵、其下攻城）指导我们如何在不同情境中选择最优方案。  
3.  **关键行动**是提升自我价值，掌握资源分配权，成为人生的主导者。  
4.  **避免持久战**，迅速决断，抓住现有资源，及时止损。

记住，无论是商业竞争还是情感关系，都要“先算后战”，通过阳谋而非阴谋取得胜利。成为有谋略的人，把价值和资源牢牢掌握在自己手中，才能“以全争于天下”。
