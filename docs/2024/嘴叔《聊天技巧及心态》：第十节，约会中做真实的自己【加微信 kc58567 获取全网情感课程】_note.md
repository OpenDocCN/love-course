# 嘴叔《聊天技巧及心态》：第十节，约会中做真实的自己

## 概述
在本节课中，我们将探讨如何在约会中保持真实自我，以及如何通过聊天技巧和心态调整来提升约会成功率。

## 约会案例分享
**以下是嘴叔分享的一个约会案例：**

嘴叔曾用所学聊天技巧与一位姑娘聊天，持续了两三个月，但姑娘不愿见面。尽管姑娘回复信息不多，嘴叔并未在意，因为他有其他选择。他分享自己的生活趣事，让姑娘了解他。三个月后，姑娘主动提出吃宵夜，嘴叔趁机见面。

**案例分析：**
- **嘴叔的心态**：嘴叔深知自己没有犯错，且拥有其他选择，因此不急于求成。
- **嘴叔的行为**：分享生活趣事，让姑娘了解他，建立信任。

## 约会技巧与心态
**以下是嘴叔总结的约会技巧与心态：**

1. **心态调整**：
   - **不急于求成**：避免在前期聊天中过于急切，给对方压力。
   - **保持自信**：相信自己的价值，不因对方的态度而动摇。
   - **心态平和**：面对挫折和拒绝时，保持冷静，不轻易崩溃。

2. **聊天技巧**：
   - **轻松有趣**：营造轻松愉快的聊天氛围，让对方感受到你的魅力。
   - **展示真实自我**：让对方了解你的真实性格和价值观。
   - **保持耐心**：给对方足够的时间了解你，建立信任。

3. **约会意义**：
   - **拉伸关系**：通过约会，增进彼此了解，拉近关系。
   - **肢体接触**：约会中的肢体接触有助于加深感情。
