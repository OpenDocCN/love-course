# 35、模块3《关系升级核心课》：01.模块三01：约会准备篇

大家好，欢迎来到模块3《关系升级核心课》的第一节——约会准备篇。本节课将分为三个部分，帮助大家更好地准备约会。

## 第一部分：约会的目的

**核心概念**：约会的主要目的是升级你们之间的关系。

**公式**：约会 = 关系升级

**内容**：

1. 约会是为了推进关系，如果关系没有进展，则约会是浪费机会。
2. 避免孤岛陷阱，选择便于转场的约会地点。
3. 通过四种好感信号判断关系升级程度：
   - **回应**：普通朋友，保持联系，提升自己，等待机会。
   - **倾诉**：好朋友，巩固关系，等待依赖，适时表白。
   - **关注**：好朋友，积极回应，表达心意，保持自信。
   - **亲密关系**：情侣关系，大胆提出要求，享受美好体验。

## 第二部分：约会前的准备

**核心概念**：

1. 选择合适的约会时间：下午2点左右。
2. 准备谈资，确保话题不断。
3. 保持微笑，增加自信。
4. 设定转场地点，避免无聊。
5. 安装微信、大众点评、打车软件等APP。
6. 准备口香糖，保持口气清新。

**内容**：

1. 第一次约会时间建议下午2点左右，避免中午和晚上。
2. 准备谈资，确保话题不断，避免尴尬。
3. 保持微笑，增加自信，提升价值感。
4. 设定转场地点，避免无聊，增加舒适感。
5. 安装微信、大众点评、打车软件等APP，方便沟通和转场。
6. 准备口香糖，保持口气清新，提升约会体验。

## 第三部分：约会中的禁忌

**核心概念**：

1. 不要过度展示自己，避免需求感。
2. 不要盲目效仿他人，思考自己的独特性。
3. 不要犯错，高手从不犯错。
4. 学会控制内心，保持气定神闲。

**内容**：

1. 不要过度展示自己，避免需求感，引起反感。
2. 不要盲目效仿他人，思考自己的独特性，展现个性。
3. 不要犯错，高手从不犯错，保持冷静。
4. 学会控制内心，保持气定神闲，提升吸引力。
