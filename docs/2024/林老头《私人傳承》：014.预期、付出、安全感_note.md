# 林老头《私人传承》：014.预期、付出、安全感

🎼hello，大家好，本节课我们将详细讲解关于预期、付出和安全感的知识。

## 概述

在本节课中，我们将学习如何通过预期、付出和安全感的理解来提升人际关系。

## 预期

预期是人行动的原动力，是人们追求目标的原因。**公式**：预期 = 动力 + 缘由。

### 预期的两个关键点

1. **对方利益**：了解对方的利益需求，从对方的角度出发制定预期。
   - **例子**：对方想要结婚，你可以从婚礼、蜜月等方面制定预期。

2. **奖励机制**：人们渴望奖励，适当的奖励可以增强预期效果。
   - **例子**：请对方吃饭、给予夸赞等。

## 付出

付出是人际关系中不可或缺的部分，它代表着机会成本和反馈。

### 机会成本

机会成本是指放弃其他选择而选择当前行为所付出的代价。

- **例子**：你选择和某人约会，放弃了其他约会机会。

### 付出与反馈

人们付出是为了获得反馈和回报。

- **例子**：牵手、聊天、亲吻等行为都是付出。

## 安全感

安全感来源于对风险的把控。

### 风险控制

通过换位思考，降低对方的风险，提升安全感。

- **例子**：约会时给对方一个台阶下，降低对方的风险。

### 控制与框架

控制力越强，安全感越强。适当的放框架，可以增强自己的安全感。

- **例子**：主动放框架，让对方感受到你的价值。

## 总结

本节课中，我们学习了预期、付出和安全感这三个关键点，以及如何通过它们来提升人际关系。记住以下关键词：

- 对方的动力、原由、机会、需要、风险、控制
- 对方的利益角度、奖励机制、随机性、限制、反馈、降低对方的风险、主动放框架

通过将这些知识点应用到实际生活中，相信你的人际关系会得到提升。
