# 王哥私教课：王哥形象改造课：第二节 发型设计 🎨

## 概述

在本节课中，我们将学习如何设计一款适合不同脸型和头发的发型，以及如何解决秃顶问题。

## 核心概念

**公式**： 发型设计 = 脸型 + 头发长度 + 发型风格

## 发型设计步骤

### 1. 选择合适的发型

*   **脸型**： 圆脸、方脸、椭圆脸等。
*   **头发长度**： 至少5厘米。
*   **发型风格**： 前次发型。

### 2. 寻找发型店

*   **大城市**： 通过抖音搜索所在城市男士发型店。
*   **小城市**： 通过抖音搜索“男士前次发型”视频，并保存视频。

### 3. 与发型师沟通

*   向发型师展示保存的视频，说明想要设计的发型。

## 解决秃顶问题

### 1. 选择假发

*   在淘宝搜索“男生假发 秃顶假发”。
*   选择价格几百元以上的假发。
*   可定制假发。

### 2. 定制假发

*   在二线以上城市商场寻找定制假发的地方。
*   前台工作人员会指引你定制假发。

## 总结

本节课中，我们学习了如何设计适合自己的发型，以及如何解决秃顶问题。希望对你有所帮助。下节课我们将学习如何打理和维护发型。
