# 关系博弈（拒绝卑微和低位）：19_1.情感博弈-底层逻辑-什么是人性

## 概述
在本节课中，我们将探讨人性的本质，以及它如何影响我们的情感关系。

## 人性的本质
**人性**是满足自身欲望的过程。所有行为背后都有一套特定不变的规律，这些规律构成了人的行为动机。

**公式**：人性 = 欲望满足

人跟动物都有需求，生存和繁衍。有需求就会产生欲望。与动物的区别在于，人类在满足欲望时，会进行理性思考。

**代码**：人类行为 = 理性思考 + 欲望满足

情绪是欲望满足之后的产物，需求与情绪相挂钩。人的行为动机基于情绪，而情绪基于认知。

**公式**：行为动机 = 情绪
**公式**：情绪 = 认知

认知决定事情给你的情绪，而情绪会指导你的行为。

## 社交关系的本质
社交关系的本质是争夺权利的过程。只有在关系中拥有的权利，才能最大限度满足自身的欲望。

**公式**：社交关系 = 权利争夺

感情中的权利就是话语权，即对一段感情或一件事情的决定权。

## 情感中的权利
以下是情感中的权利：

1. **话语权**：在一段感情中，听谁的，或者对一件事情的决定权。
2. **痛苦控制权**：谁痛苦谁改变，痛苦为驱动力，让你不得不听。
3. **选择权**：在两个痛苦的选择下，选择一个让自己不那么痛苦的关系。

## 权利的类型
关系中的权利有三种：绝对权利、平等权利以及权衡利弊的权利。

1. **绝对权利**：我说什么就是什么，你不能反抗，你只能听我的。
2. **平等权利**：在双方平等的基础上，互相尊重彼此。
3. **权衡利弊的权利**：你的权利基于你对我的付出，你的权利范围基于你提供的利益和价值。

## 总结
本节课中，我们一起学习了人性的本质、社交关系的权利争夺以及权利的类型。希望这些知识能帮助你更好地理解人际关系，并在情感中保持独立和自信。
