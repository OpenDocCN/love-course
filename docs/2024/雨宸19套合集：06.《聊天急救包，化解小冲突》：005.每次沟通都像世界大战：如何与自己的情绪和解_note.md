# 雨宸19套合集：06.《聊天急救包，化解小冲突》：005.每次沟通都像世界大战：如何与自己的情绪和解

## 概述
在本节课中，我们将探讨如何与自己的情绪和解，以改善沟通质量。

## 情绪与沟通
**以下是沟通中情绪处理的关键点：**

1. **情绪与事件的占比**：90%的情绪，10%的事件。**公式**：情绪占比 = 90%，事件占比 = 10%。
2. **情绪放大冲突**：情绪放大了冲突，使沟通更加困难。
3. **男女思维差异**：男性倾向于内化情绪，女性倾向于外化情绪。

## 情绪处理方法
**以下是处理情绪的步骤：**

1. **接纳情绪**：认识到情绪的存在，并描述感受。**代码**：描述感受 -> 提出需求。
2. **分析情绪**：识别情绪背后的不合理认知。
3. **不被对方情绪影响**：理性看待对方的情绪，关注其背后的需求。

## 总结
本节课中我们一起学习了如何与自己的情绪和解，以改善沟通质量。通过接纳、分析和理性看待情绪，我们可以更好地处理沟通中的冲突。
