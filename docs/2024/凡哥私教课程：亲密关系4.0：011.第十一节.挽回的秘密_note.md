# 凡哥私教课程：亲密关系4.0：011.第十一节.挽回的秘密

🎼hello兄弟们，我是你们的情感导师林一。今天这节课程的主题是挽回。首先，我们需要明白分手的根本原因，才能进行有效的挽回。

## 分手的原因

以下是导致分手的常见原因：

* **缺乏沟通**：缺乏共同话题，沟通不在同一个层面上。
* **个人问题**：抽烟、喝酒、赌博、邋遢、不爱卫生、说话不知轻重等。
* **生活方式**：生活方式差，社交价值低。
* **缺乏上进心**：没有事业心，没有上进心。
* **缺乏爱心**：不懂得关爱他人，不懂得爱幼。
* **为人处事**：不懂得为人处事，社交能力差。
* **不稳重**：不稳重，不坚定。
* **花天酒地**：经常花天酒地，不顾及伴侣的感受。
* **负能量**：过于悲观，缺乏自信。
* **占有欲强**：占有欲强，没有给对方私人空间。
* **不尊重不理解**：不尊重对方，不理解对方。
* **不懂得哄女人**：不懂得哄女人，不知道女人的需求。
* **不懂浪漫和惊喜**：不懂浪漫和惊喜，无法给对方带来感动和惊喜。

## 真假性分手的判断

以下是判断真假性分手的方法：

* **通讯方式**：假性分手不会拉黑或删除联系方式，真性分手会。
* **情绪指标**：假性分手会有情绪波动，真性分手会沉默。
* **社交圈子**：假性分手不会告诉亲友，真性分手会。
* **网络状态**：假性分手会发状态或照片，真性分手会沉默。
* **邀约情况**：假性分手会答应见面，真性分手会拒绝。
* **是否彻底消失**：假性分手不会彻底消失，真性分手会。

## 挽回策略

### 假性分手

* **穷追不舍**：保持联系，表达决心和真诚。
* **建立安全感**：痛改前非，用行动证明爱。
* **改变相处模式**：改变沟通和相处模式，巩固爱情。

### 真性分手

* **空间抽离**：停止纠缠，抽离一段时间。
* **改变自己**：提升自己，让自己更有魅力。
* **恢复联系**：制造偶遇，让对方感受到你的变化。
* **感情升温**：做浪漫的事情，让对方感受到恋爱的美好。

## 二次吸引

二次吸引是通过自我提升，提升价值和魅力，重新吸引对方。

## 挽回中的错误做法

* **和女生辩论**
* **不断承诺**
* **死缠烂打**
* **情绪低落**

## 挽回中的正确做法

* **认同她的情绪**
* **不要提感情**
* **像普通朋友一样相处**
* **冷冻**
* **断联**

## 总结

挽回需要耐心和决心，需要改变自己，提升自己，才能重新吸引对方。希望兄弟们能够从中受益，挽回自己的爱情。
