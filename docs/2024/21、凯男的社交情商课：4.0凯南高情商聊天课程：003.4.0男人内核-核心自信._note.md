## 四、什么是自信？🤔

接下来，我给兄弟们讲解第二个要点：什么是自信？

我现在问所有兄弟们：什么是自信？你觉得怎样才能有自信？怎样才能成为一个自信的男人？

我猜一下兄弟们的答案：

以下是兄弟们可能认为的“自信”来源：

1.  **外在条件**：我长得帅、有钱，我就自信了。
2.  **社会地位**：我人脉广、有权力，比别人有优越感，我就自信了。
3.  **技巧成功**：我背诵了一些话术套路，并且试用成功了，我就有自信了。
4.  **身体条件**：我健身，有八块腹肌，身材好，我就自信了。

我刚才举的这些例子，都属于 **“情景自信”** ，也叫 **“伪自信”** 。这是别人的标准，是社会的假象和肤浅标准。

如果你相信这些，你永远不会有真正的自信。你永远会觉得不够。有钱了会觉得不够帅，又帅又有钱了会觉得身材不够好，全都拥有了又会觉得话术技巧不好……如果是情景自信，你永远都需要更多。

为什么有些兄弟认为的自信都是情景自信、伪自信？因为他们过得很迷茫，缺乏自我价值观，不知道自己追求什么，只会追随别人的脚步。他们缺少独立想法，不知道自己的价值，生活没有明确目标。

这样的人，会把思想交给社会，让社会给你价值观、决定和观念。但社会给的观念，很多是肤浅甚至扭曲的。

## 五、价值观与独立思考 🧠

关于价值观，我要给兄弟们讲一下。一个人的学习方式有两种：

1.  **亲自体验**。
2.  **从外部获得经验**。

例如，你没从10楼跳过，但你会给出“会骨折但不会死”的答案。这个答案就是你从外部（朋友、家人）获得的经验。

大多数情况下，从外部获得经验是有帮助的。社会告诉我们不能碰电门，否则会触电，所以我们习惯从别人那里获取知识。

但最后，我们容易把别人的想法当作绝对正确，并深信不疑。你会毫无根据地相信一些事情，你的大脑乐于被周遭观念影响，比如“有钱就有自信”、“长得帅就有妹子”。

社会的观念有两面性，有好有坏。不好的观念会把你拖入深渊。你要学会自己思考并做出决定，而不是听别人告诉你怎么做。如果你能做到这一点，吸引女人对你来说会变得非常轻松。
