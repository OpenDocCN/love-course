# 林老头《系统认知》：014.4、爱情阶段，打破最后的隔阂，开启一段感情 🌹

## 概述

在本节课中，我们将学习爱情阶段的关键技巧，帮助您打破最后的隔阂，开启一段美好的感情。

## 爱情阶段概述

当您经历了势能转换，关系可能偶然突破后，就来到了爱情阶段。这个阶段需要特别注意，因为稍有不慎，原本要在一起的关系可能会破裂。

## 关键技巧

以下是爱情阶段的关键技巧，做好这些，您的胜率将高达90%：

### 1. 主动与可得性

*   **主动**：您的主动会带动可得性，让对方感受到您是可得到的。
*   **可得性**：指的是可以得到，而不是直接在一起。

### 2. 筛选与假设

*   **筛选**：筛选出您想要的性格、价值观等，并让对方知道您的期望。
*   **假设**：假设一些可能发生的情况，并思考如何应对。

### 3. 快速

*   **快速**：在爱情阶段，要快速给予反馈，快速推进关系。
*   **及时性反馈**：就像打游戏一样，及时获得反馈才能保持兴趣。

### 4. 建立信期待

*   **信期待**：通过描述浪漫场景，让对方对未来的感情生活充满期待。

### 5. 再次放大需求

*   **二次确认**：不断确认对方对您的喜欢，并放大对方的需求。
*   **服从惯性**：人会越来越认可自己曾经认可的东西。

### 6. 注重感受体验

*   **感受体验**：不要只关注结果，要注重过程中的感受和体验。

## 总结

本节课中，我们学习了爱情阶段的关键技巧，包括主动与可得性、筛选与假设、快速、建立信期待、再次放大需求以及注重感受体验。掌握这些技巧，将有助于您在爱情阶段取得成功。希望您能够将这些技巧应用到实际生活中，开启一段美好的感情。
