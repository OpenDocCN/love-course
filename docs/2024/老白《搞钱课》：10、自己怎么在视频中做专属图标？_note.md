# 老白《搞钱课》教程：10、如何在视频中制作专属图标？

## 概述
在本节课中，我们将学习如何在视频中创建一个专属图标。这将有助于提升视频的专业度和品牌形象。

## 1. 准备工作
在开始制作专属图标之前，您需要以下准备工作：
- **图像编辑软件**：如Adobe Photoshop、GIMP等。
- **视频素材**：您想要添加图标的视频文件。

## 2. 设计图标
以下是设计图标的步骤：

### 2.1 打开图像编辑软件
打开您选择的图像编辑软件。

### 2.2 创建新文件
- **分辨率**：确保图标分辨率足够高，以便在视频中清晰显示。
- **格式**：通常选择PNG格式，因为它支持透明背景。

### 2.3 设计图标
- **形状**：选择合适的形状，如圆形、方形或自定义形状。
- **颜色**：使用与视频主题相符的颜色。
- **文字**：如果需要，添加文字，如公司名称或品牌标志。

### 2.4 保存图标
将设计好的图标保存为PNG格式。

## 3. 添加图标到视频
以下是添加图标到视频的步骤：

### 3.1 打开视频编辑软件
打开您选择的视频编辑软件。

### 3.2 导入视频和图标
- **视频**：将视频文件导入到软件中。
- **图标**：将保存的图标文件导入到软件中。

### 3.3 添加图标到视频
- **位置**：选择视频中的合适位置添加图标。
- **持续时间**：设置图标的显示时间。

### 3.4 预览和导出
- **预览**：检查图标在视频中的显示效果。
- **导出**：导出带有图标的视频文件。

## 总结
本节课中，我们一起学习了如何在视频中制作专属图标。通过添加图标，您可以提升视频的专业度和品牌形象。希望您能将所学知识应用到实际项目中。
