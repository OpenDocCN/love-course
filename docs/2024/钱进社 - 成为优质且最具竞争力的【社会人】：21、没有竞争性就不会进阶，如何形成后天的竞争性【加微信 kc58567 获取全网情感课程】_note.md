# 钱进社课程21：没有竞争性就不会进阶，如何形成后天的竞争性 🚀

在本节课中，我们将探讨“竞争性”对于个人进阶的重要性，并学习如何在成年后培养这种关键特质。我们将分析缺乏竞争性的根源，并提供四个具体、可执行的建议，帮助你从思维到行动上构建起内在的竞争力。

---

## 概述：竞争性是进阶的核心引擎

成功是一个极小概率的事件。因此，我们应该通过做正确的事并投入时间，逐步向成功推进，而非幻想立刻成功或被对结果的渴望所绑架，陷入焦虑。

但必须承认，那些能接近成功或有所作为的人，都具备一种超级强的内在性格，即**竞争性**。你可能因为成长环境顺遂，或社会对性别的固有规划，而缺乏这种性格。这便成为你觉醒后试图进阶时最大的短板。如果无法在后天培养内心的竞争性，真正的进阶将非常困难。

---

## 第一节：理解“竞争性”与常见误区

上一节我们概述了竞争性的重要性，本节中我们来看看人们对“竞争”的常见误解。

我反感一种论调：那些没做成过什么事的人，听到任何机会，第一反应总是问“竞争很大吧？”。这种人眼中的竞争“大”或“不大”，其实源于内心对竞争的潜意识恐惧和抗拒。其本质仍是希望坐享其成。在当今社会，做什么事都存在竞争。

事情的成果如何？归根结底是人与人之间的竞争。而这种未做成事的人，总在担忧竞争大小，一旦觉得“大”便立刻打退堂鼓。本质上，他们不具备任何竞争力。

反之，做成过一些事情的人对竞争性的考量，才具备参考价值。对于前者而言，**所有的竞争都很大**，根源在于其内心根本没有竞争性。

**过渡**：认识到问题所在是第一步。那么，竞争性从何而来，又该如何后天培养呢？

## 第二节：培养后天竞争性的四个核心建议

我的内在具备强竞争性，源于家庭经历：父亲经商，天天竞争；家道中落后，我若不竞争就会被踩在脚下。这形成了我的竞争性人格，成为我做事的基石。

对于改造成本很高的成年人，若想进阶，必须在内心培养起竞争性人格。以下是四个建议，请目标单一化地执行，避免再次陷入迷茫和无法坚持的循环。

### 1. 培养男性思维 🧠

你想做成事，想拥有竞争性人格，必须具备**男性思维**。大家口中的“女强人”，实质是女性的皮囊贴上了男性思维的标签。我认为，女性拥有男性思维比男性本身更具竞争力，唯一的短板是职场对女性家庭与生育的歧视。但女性皮囊结合男性思维，完全可以所向披靡。

**以下是培养男性思维的具体方法：**

*   **潜心研读《三国演义》**：不要只将其当作名著消遣。从我个人的角度，这是一部培养男性思维的绝佳教材。请带着目的性去阅读。
*   **与伴侣探讨军事**：如果另一半喜欢军事，这是很好的机会。军事领域充满男性思维，是快速培养的途径。
*   **关心时事政治**：社群里的女生应多关心时事新闻、政治，而非对每部韩剧如数家珍。从单纯追剧视角（非商业视角）看，大部分韩剧剧情容易将现实中的正常女性引向“自毁前程”的思维模式。痴迷于此，你的进阶思维必然存在问题。

**过渡**：思维模式是内在的驱动，而外在状态同样重要。接下来，我们看看如何从身体层面激活竞争状态。

### 2. 选择对抗性运动 🥊

内在驱动与外部状态通常保持一致。观察那些能成事的女性，少有柔柔弱弱、木讷呆滞的，多是风风火火、果敢利落的。女性要强在骨子里，而非表面。外部状态不应是强势的，但也不能是绵羊般傻气、木讷的状态。

你必须让身体先激进起来，内心才会跟进。如果你认为自己是个“蠢女人”，就不会有任何境界。进行对抗性运动的最终目的，是激发内心，改掉软弱、柔弱和娇气。

**以下是选择运动的原则：**

*   **避免慢节奏运动**：如果你本身性格软绵、娇气，就不要再报那种慢悠悠的运动班（如某些舒缓瑜伽）。越慢、越停顿的运动，越难改变你的性格，你甚至可能享受这种与性格契合的“舒适区”，这对进阶毫无帮助。
*   **选择与速度或对抗相关的运动**：
    *   **速度类**：如跑步、游泳。有意识地培养速度感，速度感是内心激进的一种外化。
    *   **对抗类**：如拳击。现在有很多拳击班供女性学习。重点不在于能否打飞对手，而在于**参与对抗，找到对抗的状态**。竞争性就是一种对抗。你需要在身体上先找到这种感觉，从而激发内在的竞争意识。

**过渡**：当内在思维和身体状态都准备好后，我们需要强大的工具来表达和影响他人，这就是表达力。

### 3. 锤炼表达力 💬

表达力好等于**渲染力强**和**逻辑能力强**。渲染力不是打鸡血，而是一开口就能抓住眼球。同样的文案，不同的人通过形象、表述方式和语言结构呈现，效果天差地别。人生结果越来越好的人，其表达能力也必然越来越好，因为这融合了其阅历、底蕴和魅力。

表达能力可以练习，并非完全天生。如果你的知识储备（脑子）不够，填充它需要时间，但表达力这项技能可以同步锻炼提升。

**以下是每日练习表达力的“公式”：**

```plaintext
每日录音练习 = 概述事件 + 细分原因/细节 + 总结反思
```

**具体操作步骤：**

1.  每天打开手机录音，**不写草稿**，直接说。
2.  **概述**：简述今天发生的一件关键事。（例如：“今天我被老板批评了，他说我工作不走心。”）
3.  **细分**：反省并细分原因。（例如：“我反思可能有以下几点原因：第一，……第二，……第三，……”）
4.  **总结**：提出后续改进方向。（例如：“反省后，我认为日后工作中需要及时跟进和定期反省。”）

**为何要这样做？**

*   **复盘工作与生活**：这是对一天的回顾。
*   **切换视角**：回放录音时，以“外人”角度聆听，能更客观地评判自己的表达逻辑和渲染力。如果自己听着都觉无语，外人感受更甚。
*   **克服心态弱点**：表达力弱的人常因“说不好”而“不说”，形成恶性循环。这个方法让你“硬上”，先说给自己听，压力小，易于坚持。
*   **手写辅助**：如果口述逻辑混乱，就用笔写下来。手写比打字更能让大脑专注，有助于理清框架。写完后，再依据文字录一遍。我强烈建议大家培养手写复盘的习惯。

**过渡**：最后，也是最有效的方法，就是将自己置于一个能逼迫你成长的环境中。

### 4. 进入高挑战环境 ⚔️

最快的提升方法往往最折磨人，那就是**环境**。你必须进入一个人际关系复杂，或每天都有人逼迫你前进的环境。

试想，如果你的环境里，人人都在聊八卦，毫无压力，这种没有竞争性的环境只会让你越来越废。例如，有些销售岗位，领导要求严苛，达不到业绩就走人；或有些工作底薪极低，全靠提成。这类环境会极大地逼迫你培养竞争性：你必须从不善沟通变得善于沟通，从内向变得外向，从拖拉变得激进。

当一个人的内心拥有了竞争性，他面对工作的状态就会彻底改变。关于环境选择，过往音频有更详细阐述，在此不再赘述。
