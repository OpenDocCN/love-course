# 钱进社 - 成为优质且最具竞争力的【社会人】：21、没有竞争性就不会进阶，如何形成后天的竞争性

## 概述
在本节课中，我们将探讨如何形成后天的竞争性，这对于个人进阶至关重要。

## 核心概念
**竞争性**：指个体在面对挑战和竞争时，所表现出的积极应对和超越自我的能力。

## 教程内容

### 1. 竞争性的重要性
**没有竞争性就不会进阶**。成年人的教育和改造成本极高，一旦开始有意识改变，可能会陷入怪圈，最大的敌人是自己。

### 2. 形成后天竞争性的方法
以下为培养后天竞争性的四个建议：

#### 2.1 培养男性思维
- **男性思维**：指独立、果断、敢于挑战的性格特征。
- **方法**：阅读《三国演义》，关注时事新闻、政治，培养快速思维。

#### 2.2 选择对抗性运动
- **目的**：激发内心，改变软弱、柔弱、娇气的性格。
- **方法**：参与跑步、游泳、拳击等对抗性运动。

#### 2.3 提升表达力
- **表达力**：指渲染力强和逻辑能力强。
- **方法**：每天打开手机录音，总结细分再总结，培养表达能力。

#### 2.4 创造竞争环境
- **环境**：人际关系复杂、竞争激烈的环境。
- **目的**：逼迫自己改变，提升竞争力。
