# 正向吸引高阶成长课：03_1·第一印象

### 第一节 第一印象

**首先，我们需要认识到，0.1秒的第一印象往往决定了50%的情感成败。** 这意味着在情感交流中，第一印象的重要性不言而喻。**第一印象良好，可以决定50%的成功率**。

**概念解析：**

1. **首映效应**：即第一印象效应，指的是人们对于初次接触的人或事物形成的印象往往会影响后续的认知和评价。
2. **初恋效应**：指的是第一任伴侣对一个人的爱情观影响最大，后续伴侣的影响逐渐递减。
3. **自我验证偏差**：指的是人们倾向于寻找支持自己观点的证据，从而影响对他人行为的解释和判断。

**快速判断的准确性**：

- 0.04秒判断陌生人面部表情（喜悦或愤怒）。
- 0.1秒判断对方长相是否具有吸引力。
- 5秒判断性格和人格。

**人际距离与第一印象**：

- **恋人距离（0.5米以内）**：主要关注面部长相、微表情和情绪。
- **朋友距离（0.5-1.5米）**：关注面部长相和腰部上方的身材线条。
- **商务距离（1.5-3米）**：关注面部长相和大腿上方的身材线条。
- **公共距离（3米以上）**：关注身材线条，不太关注面部长相和情绪。

**打招呼的策略**：

- 根据自身优势和对方特点，选择合适的打招呼方式。

**大众审美与男性视角**：

- **大众审美**：脸对称、身材比例协调、腰臀比适中。
- **男性视角**：自律程度、气质、身材比例。

**脸部的吸引力**：

- **成长印记**：童年经历、性启蒙等。
- **性格特质**：保守或开放。
- **容貌焦虑程度**：自信或焦虑。

**印象管理策略**：

1. **高举高大**：展现自信魅力。
2. **投其所好**：迎合对方喜好。
3. **拉低预期**：制造惊喜。
4. **持续反差**：创造多变风格。

**彩蛋部分**：

- 具有强烈感官词典和魅力标签的人更容易留下深刻印象。

**总结**：

本节课介绍了第一印象的重要性、影响因素以及印象管理策略。希望同学们能够掌握这些知识，在人际交往中更好地展现自己，提升吸引力。
