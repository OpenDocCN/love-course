# 27、林夕-高质量关系课程：1.14-第14课《送礼物的学问》🎁

## 概述

在本节课中，我们将学习男女之间送礼物的学问，包括如何识别男人的投资心理、谈钱的时机、金额、话术以及注意事项等。

## 核心概念

**公式**： 
- 金额 = 投资习惯 + 感情浓度
- 方法：让步原理、以退为进、登门槛效应、潜意识价值观植入

## 1. 识别男人的投资心理

**公式**：
- 敞量型：钱来得容易，花得也容易
- 谨慎型：钱来得不容易，花得谨慎

## 2. 谈钱的时机

以下是谈钱的几种时机：

1. 节假日送礼
2. 对方做错事情，感到内疚
3. 男人爱面子，需要回礼
4. 共同做一件事情，如旅行、逛街
5. 示弱，表现出喜欢的东西
6. 确定关系后，引导男人长期投资

## 3. 金额

**公式**：
- 金额 = 投资习惯 + 感情浓度

## 4. 话术

以下是几种话术：

1. 让步原理：先提一个大要求，然后提出真正的要求
2. 以退为进：激发男人的愧疚感
3. 登门槛效应：先提一个小要求，再增加要求的分量
4. 潜意识价值观植入：建立人设，如不物质、舍得付出、注重仪式感

## 5. 注意事项

1. 学会包装自己，提升自身价值
2. 认识自己的需求，勇敢表达
3. 接受礼物时，保持平常心，珍惜使用
4. 继续用心经营感情，保持良性互动

## 总结

本节课中，我们学习了男女之间送礼物的学问，包括识别男人的投资心理、谈钱的时机、金额、话术以及注意事项等。希望各位女性能够通过学习，学会聪明地谈钱，享受高质量的关系。
