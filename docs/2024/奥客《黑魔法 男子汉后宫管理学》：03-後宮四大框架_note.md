# 奥客《黑魔法 男子汉后宫管理学》：03-后宫四大框架 🏛️

在本节课中，我们将要学习管理多重关系的四大核心框架。理解并运用这些框架，可以帮助你更好地平衡不同关系中的互动模式，避免关系失控。

上一节我们介绍了亲密焦虑光谱图，并学习了如何以25%和75%为界来把握关系比例。本节中，我们来看看构成后宫管理的四大具体框架。

## 朋友框 👬

朋友框是四大框架中的第一个。如果之前有人不理解“兄弟炮”的含义，这个框架就是最好的例子。至于相关问题，我们会在讲解“女友框”时详细叙述。

以下是朋友框的一个对话示例。在《列女课》中，我讲过如何复盘对话：你需要将两人的对话演绎出来，揣测他们说话时的表情和语气，从而理解他们所处的氛围。

```
女：你上次约会那个女生怎么样？
男：还不错啊，挺聊得来的。你呢，最近有认识新人吗？
女：有啊，但感觉差点意思。还是跟你聊天舒服，像兄弟一样。
```

这个对话本身，如果不告诉你左边是女性，右边是男性，是否很像两个兄弟在讨论各自的约会心得？这就是所谓的“兄弟炮”，是朋友框的一种体现。在这种框架下，双方可以像朋友一样，以平等主义的姿态聊天，甚至可以像讨论电玩游戏一样，轻松谈论彼此的上床过程。

朋友框的“兄弟炮”，也是我之前提到的，唯一一种在亲密值可能爆表的情况下，仍能维持炮友关系的框架。

你可能会问，为何要采用朋友框的管理模式？答案是：朋友框可以用来“刹车”。它能让关系稳定在某个阶段，防止其像失控的火车一样不断滑向更深的习惯区。我们之前讨论的平等主义、自我揭露等技巧，都是在将互动框架推向朋友框。

## 师长框 🧑‍🏫

接下来我们看看师长框。在这个框架中，你像一位老师一样与你的“盘子”（指关系对象）互动。

以下是师长框的一个案例。左边女性在向右边男性进行感情咨询，她最近有一个想认真发展的男性对象。右边采用师长框架的男性，则像老师一样点醒她，告诉她该如何做。

```
女：我觉得他好像对我忽冷忽热，我该怎么办？
男：首先，停止过度分析他的行为。把你的注意力放回自己的生活上，提升自己的价值。当他发现你不再围着他转时，态度自然会改变。
```

在《列女课》中，我提过一个概念：尽量不要当女生的“恋爱军师”。因为那种像朋友一样相处的恋爱顾问，通常是闺蜜的角色。虽然以师长框架进行这类对话会稍好一些，但无论如何，女性多半只愿与闺蜜或同性深入讨论感情问题。因此，即使以师长框架聊感情，也难免沾染上“男闺蜜”的印象，与闺蜜形象产生重叠。

所以，我并不建议用师长框来聊感情事。但是，你可以从其他领域建立师长框架，例如职场指导或生活品味教学。这就是为什么许多职场主管在与女性下属互动时能得心应手——主管与下属本身就是天然的师长框架。

## 女友框 👩‍❤️‍👨

现在，我们进入女友框。这个框架管理的是“竞争焦虑”。之前有人问：为什么焦虑值上升时就一定会有“悱恻”（指情感纠葛）？答案就在这里。

当女性发现你越来越不受控，对你的行踪越来越无法掌握时，她们会产生“竞争焦虑”，担心你随时会和别人跑掉。请记住：只要你的自身状态好，把自己打理得当，女性自然会产生竞争焦虑。

可以这样理解：你甚至无需对她做什么特别的事。只要保持打炮关系，中间甚至可能不频繁联络，这种焦虑也会在她心中慢慢累积。当然，我并非提倡你什么都不做。关键在于，只要你确定自己对她而言是有价值的，你就要“摆着”，让她对你有幻想，让她去猜测你在做什么。

焦虑相较于亲密，本身更不消耗你的资源。它不是你必须有意识付出资源的行为。只要你拥有价值，并过着丰富多彩的生活，对她而言，你就是一个“焦虑的发光体”。

以下是一个女友框的案例。左边是一位已婚女性，右边是一位有女友的男性。

```
人妻：心情有点复杂…不开心，但又想见你。
男：怎么了？
人妻：但我不想在你和女友约会后的一两天内跟你进一步互动（比如打炮）。因为…你会把“量”用掉，我会很不爽。
```

现在你明白为什么女友会检查“量”了吧？对于她们而言，“量”的减少也是一种资源投注的减少，是基因层面的资源竞争。

这种情况是很多人会犯的。后来这两个人吵架了。一个人妻和一个有女友的男人，照理说各取所需，怎么还会吵架、还会走心、产生独占欲呢？答案很自然：他们之间产生了竞争焦虑，都想要独占对方的资源。所以，即使本是炮友关系，一旦有人走心，这种情况就会发生。

我的建议是，如果对方走心，你要“电”她，要“飙”她。你可以说：“你有事吗？你已婚，我有女友，我们不可能有任何发展，就这样爽爽过就好了。”通常，你飙完之后，对方会变乖。案例中的男生只是稍微安抚（“没事，抱抱”），但实际上，你应该去“电”她才行。

## 主奴框 ⛓️

最后，我们来到很多人可能最喜欢的——主奴框。我本人也很喜欢讲这一部分。

主奴框的对话最常出现在两人刚在一起的热恋期。因此，我的城市建议是：如果你想尝试主奴框，关系初期的热恋期是最好的时机。因为此时彼此欲望还很强烈。

主奴框管理的是“真实欲望”。你要在这个阶段去建立你希望的框架，对方为了满足你的需求，可能会做出妥协，并表现出服务精神。这包含各种性幻想。

以下是一个主奴框案例（双方当时尚未见面，仅在交友软件聊天）：

```
男：你以后就是我的小宠物了，知道吗？
女：嗯…（幻想被绑起来）你会怎么对待你的宠物？
男：（错误示范）哈哈哈，你想我怎么对待你呢？
```

我认为这个男生处理得并不好。他在主奴框中使用问句、征询语气，甚至用“哈哈哈”来缓和情绪，这是错误的。在《练语课》中我讲过如何处理：你的语气要彻底转变。下面提供一个更好的模板：

```
男：（正确示范）好，下次见面时，我会用丝巾蒙住你的眼睛。记住，主人说话没有疑问句。
```

右边是另一个案例。女生有老公，男生是她后来找到的偷情对象。当女生说“想跟你保持距离”时，男生处理如下：

```
男：你是我的女友，也是我的玩具。听我的，知道吗？
女：好吧…
男：说“是，主人”。
女：是，主人。
```

这就是主奴框的完美呈现：使用命令句。在床上的互动也应符合这种形式。请注意，请务必遵循合法、双方合意的原则。

曾经有大学生问我能否看这门课。我的回答是：你可能要十年后才能真正理解这些内容。因为我不认为一个大学生能完全理解这种对话模式。这不仅仅是“千人斩”的技巧，更是许多Beta炮男、M炮男性所缺乏的。主奴框是一个男人可以去追寻的互动境界。

---

本节课中，我们一起学习了后宫管理的四大框架：
1.  **朋友框**：用于建立平等、轻松的关系，并能有效“刹车”。
2.  **师长框**：用于在特定领域（如职场、生活）建立指导与被指导的关系。
3.  **女友框**：用于管理对方的“竞争焦虑”，通过自身价值引发对方的不安与投入。
4.  **主奴框**：用于在欲望高峰期建立命令与服从的关系，满足彼此的深层幻想。

掌握这四大框架，并学会在不同关系和不同阶段灵活运用，是管理复杂情感关系的关键。
