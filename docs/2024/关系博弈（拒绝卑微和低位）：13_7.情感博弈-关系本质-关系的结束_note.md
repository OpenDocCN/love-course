# 关系博弈（拒绝卑微和低位）：13_7.情感博弈-关系本质-关系的结束

## 概述
在本节课中，我们将探讨关系中的本质，包括关系的开始、发展、结束以及挽回的可能性。

## 人性底层的欲望
**公式**：欲望 → 行为 → 满足 → 新欲望

人天生带有欲望，这些欲望驱动着我们的行为和思想。满足欲望后，新的欲望又会产生，导致欲望不断升级。

## 恋爱需求升级
**代码**：
```python
# 初始需求
initial_demand = "陪伴"

# 需求升级
upgraded_demand = "每天陪伴，无聊时一起出去玩，想吃好吃的时带去吃"

# 需求进一步升级
further_upgraded_demand = "所有时间都属于我"
```

随着时间的推移，恋爱中的需求会不断升级，从简单的陪伴到更深入的情感需求。

## 关系破裂核心点：关系失衡
**公式**：满足对方需求 → 感情失去平衡 → 关系破裂

当一方无法满足另一方的需求时，感情就会失去平衡，最终导致关系破裂。

## 挽回的两个核心动机
1. **不甘心**：分手后，内心失去平衡，想通过挽回前任来找回平衡。
2. **舍不得**：对对方有深厚的情感投资，不愿放下。

本节课中，我们学习了关系中的本质，包括人性的欲望、恋爱需求的升级、关系破裂的原因以及挽回的可能性。希望这些内容能帮助你更好地理解关系中的问题，并找到解决之道。
