# 驾驭关系的能力：情绪价值：07如何安抚脆弱的我（一）

## 概述

在本节课中，我们将探讨如何识别和安抚伴侣的脆弱部分，从而提升关系质量。

## 1. 脆弱的我

每个人都有脆弱的部分，这部分通常源于心理创伤，如孤独、无价值感、不被爱、不被尊重等。这些创伤背后伴随着各种情绪，如恐惧、无助、悲伤、自责、委屈、羞愧、失望和迷茫。

## 2. 发现脆弱点

为了安抚伴侣的脆弱部分，首先需要发现这些脆弱点。以下是一些发现脆弱点的步骤：

### 2.1 敏感点

- **记录敏感点**：拿一个小本子记录下伴侣的敏感点，如对否定、控制、骂人等敏感。
- **细化敏感点**：将敏感点进一步细化，如对否定敏感的具体表现。

### 2.2 脆弱字典

- **记录脆弱点**：将发现的脆弱点记录下来，形成一个脆弱字典。
- **细化脆弱点**：将脆弱点进一步细化，如孤独感背后的原因。

### 2.3 防御机制

- **反推脆弱点**：通过观察伴侣的防御机制，反推其脆弱点。

## 3. 安抚脆弱点

发现脆弱点后，需要采取行动安抚这些脆弱点。以下是一些安抚脆弱点的建议：

### 3.1 疗愈

- **提供情绪价值**：通过赞美、肯定等方式提供情绪价值。
- **疗愈**：通过深入了解伴侣的人格和需求，提供针对性的疗愈。

### 3.2 幸福相处

- **免责声明**：本课程旨在帮助人们幸福相处，而非对付他人。
- **成为可爱的人**：通过提升自己，成为别人眼中最可爱的人。

## 总结

本节课介绍了如何发现和安抚伴侣的脆弱部分，从而提升关系质量。通过记录脆弱字典、反推脆弱点、提供情绪价值和疗愈，我们可以更好地理解和支持伴侣，共同创造幸福的相处模式。
