# 驾驭关系的能力：情绪价值：07 如何安抚脆弱的我（一）🧩

在本节课中，我们将要学习如何识别并安抚伴侣内心“脆弱的部分”。上一节我们介绍了如何发现并放大对方的“真我”，本节中我们来看看如何面对和处理那些隐藏的创伤与敏感点。

## 回顾与作业：持续放大“真我”

在深入探讨“脆弱的我”之前，我们先对上一节内容进行补充。上一节我们讲到如何发现对方的“真我”部分，即善良与优秀的品质。

以下是给大家的实践任务：无论你们当前关系如何，请尝试每天找出伴侣三个做得好的地方。这些地方可以体现在生活琐事中，例如他对家庭、孩子的付出。我们要做的就是将这些“真我”的部分放大并给予认可。

你可能会觉得，伴侣的优秀品质就那么几个，很快就会“夸完”。其实不然，关键在于将品质与具体事件相结合。例如，“自信”是一个品质，但今天可以结合他的一次**演讲**来夸赞，明天则可以结合他与客户的**谈判**来肯定。通过这种方式，赞美可以变得无穷无尽。

**任务公式：**
`每日赞美 = 发现伴侣的积极品质 × 关联具体事件`

请坚持完成这个任务，每天找到三个点并进行真诚的赞美与肯定。这能有效促进你们的关系升温。

## 认识“脆弱的我”：每个人的内在创伤

现在，我们进入本节课的核心：认识“脆弱的我”。每个人都有脆弱的部分，这源于我们成长过程中可能形成的心理创伤。

这些脆弱点通常表现为一些核心感受，例如：
*   孤独感
*   无价值感
*   不被爱的感觉
*   不被尊重的感觉

在这些感受背后，往往伴随着复杂的情绪，例如：
*   恐惧
*   无助
*   悲伤
*   自责
*   委屈
*   羞愧
*   失望
*   迷茫

面对“真我”，我们的主张是**放大**；而面对“脆弱的我”，我们的核心主张则是**安抚**。这需要我们先**发现**，再**安抚**。

## 第一步：如何发现伴侣的脆弱点——创建“脆弱字典”

要安抚脆弱，首先需要精准地发现它。在长期共同生活中，你其实已经知晓对方的一部分敏感点。

以下是创建属于你伴侣的“脆弱字典”的具体方法：

1.  **准备工具**：拿出一个笔记本（或电子文档），专门用于记录。
2.  **记录已知敏感点**：写下你已经知道的、伴侣容易受伤或情绪波动的领域。例如，“敏感”是一个大类。
3.  **进行细化分类**：在每个大类下，记录具体触发场景。这是关键步骤。
    *   例如，在“敏感”大类下：
        *   **被否定**：当他的决定被反驳时，容易暴怒。
        *   **被控制**：感到被支配时会激烈反抗。
        *   **被辱及家人**：当争吵中涉及对其家人的侮辱时，反应特别强烈。
4.  **持续观察与补充**：在生活中不断观察，将新发现的脆弱点及其具体表现补充到你的字典中。
    *   例如，“孤独感”可能细化为：
        *   **需要陪伴**：害怕独处，这背后可能是安全感缺失。
        *   **害怕被抛弃**：源于童年未被充分关爱，极度渴望爱又害怕失去。
    *   例如，“不被尊重”可能细化为：
        *   **公开场合被驳面子**：在外人面前被大声指责。
        *   **在孩子面前形象受损**：被伴侣在孩子面前贬低。
        *   **社交场合不配合**：在朋友聚会时拒绝一同出席。
5.  **探究行为背后的根源**：一些外在行为可能反映了内在的脆弱。尝试记录并分析。
    *   例如，**过度追求物质（开好车、穿名牌、抢着买单）**，可能根源是**自卑**或“怕被人看不起”的**羞耻感**。
    *   例如，**莫名的恐惧**，可能包括恐高、害怕与体格强壮者冲突、恐惧健康问题或权力丧失后的落差等。

**核心方法：**
`脆弱字典 = 观察 + 记录（大类 -> 具体触发场景 -> 可能根源）`

这个“脆弱字典”是你理解伴侣内心世界的密码本。通过它，你能对他有更深刻、更系统的了解，而非停留在表面争吵。记住，我们建立这个字典的目的不是为了攻击，而是为了下一步的**理解与疗愈**。

## 课程总结与前瞻

本节课中，我们一起学习了情绪价值中关于“脆弱部分”的第一课。我们明确了“脆弱的我”是什么，并掌握了通过创建“脆弱字典”来系统发现伴侣脆弱点的方法。

我们再次强调，发现脆弱点的最终目的是为了**安抚**与**疗愈**，这是提供深层情绪价值的关键。拥有这本“字典”，意味着你掌握了通往伴侣内心的地图，但这地图是用来修复关系，而非发动战争。

下一节课，我们将继续探讨“如何安抚脆弱的我（二）”，学习当伴侣的防御机制被触发时，如何反向推导其脆弱点，并开始讲解具体的安抚方法与话术。这将是我们从“了解”迈向“疗愈”的实践步骤。
