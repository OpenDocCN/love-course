# 彭祖强肾功课程 43：014.14. 武火吐纳 🔥

在本节课中，我们将学习“武火吐纳”的具体练习方法。这是一种通过特定的呼吸与身体动作相结合，来强化肾脏功能的功法。我们将从预备姿势开始，逐步分解动作与呼吸的配合要点。

## 预备姿势 🧍

首先，我们需要摆好正确的身体姿势，这是所有后续动作的基础。

以下是预备姿势的具体步骤：
*   双脚平行分开，与肩同宽。
*   双膝微曲，略向外展。
*   脚底虚空，重心稳固。
*   臀部下坐，如坐高凳。
*   双手扶腰，上身自然直立。
*   舌顶上颚，牙齿轻叩，下颌微收。
*   目视前方，采用鼻吸鼻呼的方式。

## 呼吸与核心动作 💨

摆好姿势后，我们进入核心的呼吸与动作配合练习。上一节我们建立了稳定的身体架构，本节中我们来看看如何在此架构上进行“武火吐纳”。

呼吸与动作的配合是本节的关键，其核心流程可以概括为以下公式：
**吸气 -> 提肛缩肾 -> 呼气 -> 胸胯开裆**

以下是详细的配合步骤：
1.  **吸气**：用鼻子吸气，同时提肛、收缩会阴（缩肾）。
2.  **动作**：吸气时，髋关节做横向的中摆运动。
3.  **发声**：吸气过程中，震动鼻腔，发出“嗯”、“嗡”一类的声音，以震动气息。
4.  **呼气**：用鼻子呼气，用意念将气压送到下腹部。
5.  **动作**：呼气时，胸部自然放松，胯部微微打开（开裆）。

## 练习关键点 ✨

掌握了基本动作后，为了保证练习效果，必须时刻注意以下几个要点。这些要点贯穿练习始终，是“武火吐纳”的精髓所在。

以下是必须牢记的五个关键点：
*   **头上顶**：感觉头顶百会穴微微上提，保持脊柱中正。
*   **收下颌**：下巴微微内收，有助于畅通督脉。
*   **意念下注**：将意念集中于会阴部位。
*   **收小腹**：呼气时，轻轻收缩小腹。
*   **舌顶上颚**：始终保持舌尖轻抵上颚，连接任督二脉。

## 深化与收功 🧘

在练习的最后阶段，我们需要深化动作感受并妥善收功，让气息归元。经过前面的练习，身体已经活跃起来，现在通过更精细的意念引导来强化效果。

在呼气收小腹时，可以加入以下意念：
> 想象小腹尽可能向后贴近脊柱，以此轻柔地拉伸脊柱，增加其柔韧性。

同时，确保全身放松，感受气力能够通达脚底，与大地连接。
练习结束时，需缓慢收势，静立片刻，调和呼吸，即为收工。

---

本节课中，我们一起学习了“武火吐纳”功法的完整流程：从**双脚平行、膝微曲**的预备姿势开始，掌握了**吸气提肛、呼气开裆**配合髋部摆动的核心动作，明确了**头上顶、收下颌**等关键要点，最后通过意念引导深化练习并完成收功。坚持练习，有助于增强肾气，提升活力。
