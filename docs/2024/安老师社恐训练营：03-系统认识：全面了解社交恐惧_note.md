## 1. 如何判断自己是否有社交恐惧

社交恐惧者会在社交场合产生显著持久的恐惧心理。以下是一些特征：

- **过分关注他人看法**：内心戏多，担心他人对自己的负面评价。
- **退缩回避行为**：在社交场合中，如聚餐、聚会等，会主动承担一些工作，避免尴尬。
- **极度焦虑**：在社交场合中，会感到心跳加速、脸红、口干舌燥等。

**区分内向、害羞和社交恐惧**：

- **内向**：喜欢独处，但不害怕社交。
- **害羞**：想躲避社交，与人交往时感到不自在。
- **社交恐惧**：对社交或公开场合感到强烈的恐惧或忧虑。

## 2. 社交恐惧的原因

造成社交恐惧的原因有很多，以下是一些常见原因：

- **过度警觉**：对自身反应过度关注，导致越想放松越紧张。
- **完美主义**：担心犯错，导致焦虑。
- **家庭原因**：父母的行为和焦虑想法可能影响孩子。
- **逃避**：逃避社交场合，导致问题加剧。

## 3. 社交恐惧的分类

社交恐惧的行为和表现因人而异，以下是一些常见分类：

- **赤面恐惧**：害怕脸红。
- **视线恐惧**：害怕与人对视。
- **表情恐惧**：担心面部表情不自然。
- **口吃恐惧**：说话结巴。
- **异性恐惧**：害怕与异性交往。
- **权威恐惧**：害怕与权威人士相处。
