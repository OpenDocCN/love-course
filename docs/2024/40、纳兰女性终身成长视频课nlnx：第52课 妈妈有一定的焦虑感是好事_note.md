```
社会因素 = 教育变革 + 剧场效应 + 家庭结构变化

```
家庭因素 = 教育认知偏差 + 单一的教育认知 + 不和谐的夫妻关系 + 隔代带养
```

个人因素 = 感性 + 自我认知 + 不相信孩子
```
