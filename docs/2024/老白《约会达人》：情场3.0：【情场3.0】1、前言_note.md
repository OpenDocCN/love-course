# 老白《约会达人》：情场3.0：【情场3.0】1、前言_note

在本节课中，我们将要学习如何正确开启情感能力提升之旅。课程将剖析常见的情感困境根源，并介绍后续学习所需的核心心态与正确方法。

许多人在感情生活中遇到了各种问题，例如表白失败、分手后想挽回，或是因为自卑内向而困扰。

你或许不在意恋爱方法或技巧，只是单纯想和对方在一起。然而，我们需要反思，正是因为缺乏正确的方法才导致了这些结果。我们首先需要抚平内心，找出问题究竟出在哪里。

这些问题的答案都包含在后续的课程中。我们也能通过课程找到正确的方法，纠正过去犯下的错误。人们常常会沉浸在某种情绪中无法自拔。

为此，我准备了疗愈心态的课程。当你内心感到难受时，可以打开聆听。

这能舒缓你的心情。人是世界上情感最丰富的生物，而爱情是其中最重要的情感之一。当你思考为何自己付出了全部却换来无情对待时，当你疑惑仅仅因为做错一件事对方就要离开时，当你对一段关系感到绝望并决心改变自己时，请相信，你经历过的这些感受我都曾体会。

这一切的付出，终将在你看到课程结语的那天开始获得回报。在这里你并不孤独，这是一个温暖的大家庭。

有许多和你一样在情感中受伤、渴望改变的人。不要再沉溺于过去，当你决定改变时，就应彻底地与过去的自己告别。

还有一点需要说明。恋爱在我心中是一种感觉、一种体会，是一种需要亲身经历才能理解其分量的东西。我试图用语言将这种体会、感觉和分量表达出来。

但我们的表达能力是有限的，未必能完美描述出我的全部感受。同时，大家听我讲述时，也存在一个理解的过程，理解有时会出现偏差。

因此，最终每个人形成的感受、体会以及对一件事的理解，其深刻程度都会有差异。

请大家先试着理解我能表达出来的意思，进而去体会那些我甚至无法用语言表达的、真实存在于我脑中的意思与感受。在恋爱中，有些经历我们有过，有些则没有。

为了将大家带入这种特定的感觉中，我尝试了各种方法。也希望大家在学习时，能带着“空杯心态”认真观看。最后，也是最重要的一点。

学习过程中切记不要“动心”。随着我们能力提升，女生对我们的反馈会开始变好，让我们体验到前所未有的感觉。

这时我们特别容易动心，从而陷入状态越来越差的死循环。必须在学习过程中做到不动心，只专注于练习。

在学习过程中犯遍所有错误，熟悉各个环节步骤之后，再开始认真寻找恋爱对象。这个过程里，我们也会遇到各种各样的女生，从而真正明白自己喜欢的是什么类型。

也才能真正知道该如何去做。学习过程中，我们常以为自己非常喜欢某个女生，但那未必是真正的喜欢。这就像我小时候去大城市，觉得城市真好，灯红酒绿。

直到去过很多大城市后，我才明白自己喜欢哪个城市，哪个城市更适合我。道理是相通的。

本节课中，我们一起学习了正视情感问题、准备正确学习心态以及“练习期不动心”的核心原则。这是你情感成长之旅的坚实起点。
