# 清风老师全套：情感变现实操训练营30期 第十课：课程发布系统：快速过审核攻略 📚

## 概述
在本节课中，我们将学习如何在平台上架课程并快速通过审核。

## 课程内容

### 1. 平台规则注意事项
- **平台忌讳**：不要将客户引导至其他平台，避免在平台上出现其他平台的账号和内容。
- **敏感词汇**：避免使用如“挽回”、“快速脱单”、“约炮”、“精神操控”等敏感词汇。
- **内容方向**：对于敏感内容，建议以防诈骗的方式呈现，如教授识别和反精神操控。

### 2. 小额通与学浪平台对比
- **小额通**：由腾讯投资，服务于私域用户，宽容度较高，适合多种类型课程，但无法带来公域流量和价值，支持国际收款。
- **学浪平台**：审核严格，但能带来直播间流量和更高的转化率。

### 3. 平台选择建议
- 新手教师建议先尝试小额通，积累经验后再尝试学浪。
- 学浪审核门槛为本科学历及以上，可通过海豚之道上架课程。

### 4. 课程销售方式
- **私域销售**：通过直播和社群引流至私域，搭建私域体系并变现。
- **直播销售**：在直播中推广课程。
