# 老李校长《男人内外魅力训练营》：81.保持框架

🎼保持框架。🎼情感的道路充满挑战，充满人性的贪婪与邪恶。只有勇敢面对，才能成为真正的英雄。跟随我，一起探索情感的奥秘，看透人性的善恶。

**以下是本节教程的主要内容：**

### 1. 框架的定义
框架，源自心理学，是一种原则和策略。它要求我们在情感交往中保持自我，不轻易妥协。

### 2. 框架的体现
- **拒绝不合理要求**：即使做错了事情，也要敢于批评、翻脸、吵架，坚决拒绝不合理的要求。
- **合理要求**：对于合理的要求，要积极回应，勇于承担责任。
- **情绪稳定**：在遇到困难或挫折时，保持情绪稳定，不轻易崩溃。

### 3. 框架的重要性
- **提升魅力**：敢于拒绝、翻脸、吵架，让对方感受到你的男人味。
- **情绪稳定**：在情感中保持情绪稳定，不轻易被外界因素影响。

### 4. 如何修炼框架
- **经历与心理**：通过积累经验，提高心理素质，更好地理解人性。
- **关键词战略**：敢于反击、翻脸、拒绝，时刻保持情绪稳定。

**总结：**

本节课中，我们学习了如何保持框架，包括框架的定义、体现、重要性以及修炼方法。希望同学们能够将所学知识应用到实际生活中，提升自己的魅力和情商。
