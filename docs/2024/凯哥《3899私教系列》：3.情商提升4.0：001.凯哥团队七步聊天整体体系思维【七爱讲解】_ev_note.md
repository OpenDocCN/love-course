## 七步聊天法

### 1. 开场白

* 目的：激发对方回复欲望
* 标准：开场白是否能让对方产生回复的欲望
* 注意事项：
    * 避免夸大其词、暴露需求感
    * 从观察对方朋友圈入手，激发对方兴趣
    * 根据对方特质进行评价

### 2. 基本信息交换

* 目的：了解彼此基本信息
* 内容：工作、年龄、兴趣爱好、生活等

### 3. 拉近距离

* 目的：通过互动拉近彼此距离
* 方法：
    * 冷读、推拉、表演性人格、画面感、植入等
    * 一句逗逼，一句正经的聊天方式

### 4. 塑造自己的形象，揭露对方

* 目的：塑造男神形象，揭露对方
* 方法：
    * 揭露对方的话术
    * 承担责任，揭露对方

### 5. 升高关系

* 目的：升高关系，进入暧昧阶段
* 方法：
    * 画面感植入、暧昧、调情等

### 6. 植入预期

* 目的：让对方对未来产生期待
* 方法：
    * 分享对未来生活的想法

### 7. 邀约

* 目的：邀请对方见面
* 方法：
    * 直接邀约、模糊性邀约
