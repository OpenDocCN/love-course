# 线上聊天法，学会如何和喜欢的人聊天：第11课：聊天四大阶段-了解阶段

🎼hello，大家好，欢迎来到咱们的第十一节。这节课我们将探讨聊天的四大阶段中的第二阶段——了解阶段。

## 了解阶段概述

了解阶段是从打破陌生到走向熟悉的过程。在这个阶段，我们需要设定基调，自然地展开话题，并带动整个聊天的节奏。

## 了解阶段的目标

了解阶段的目标是试探和挖掘双方，建立吸引。

## 了解阶段的周期

了解阶段分为三个周期：

1. **试探周期**：刚认识的前3到5天，主要是磨合价值观点，让女生了解你。
2. **中期**：邀约前，需要更多了解和建立吸引，建立安全感。
3. **后期**：第一次约会后，增强联系感。

## 如何了解对方

在了解阶段，我们如何了解对方或让对方了解自己，分为以下两个部分：

### 展示自我

1. **主动展示**：主动分享关于自己的信息，但初期应保持浅层。
2. **对话中展示**：在对话中不经意地描述自己，如分享美食经历。
3. **中层对话**：在女生追问下，进入更深层次的对话。

### 主动了解

1. **询问状态**：了解女生的当前状态和活动。
2. **关键词寻找话题**：通过关键词引导话题。
3. **深入话题**：在自己熟悉的领域深入讨论。
4. **引导女生多说**：引导女生分享自己的想法和经历。

## 了解阶段的聊天步骤

1. **开场**：通过外围话题（如天气、事件）或状态感受开启对话。
2. **话题选择**：由点到线再到面自然展开，根据关键词选择话题。
3. **充分互动**：包含情绪和话题延续法，使话题丰富。

## 总结

本节课中，我们学习了了解阶段在聊天中的重要性，以及如何通过展示自我和主动了解对方来建立吸引。下节课我们将继续探讨聊天的其他阶段。
