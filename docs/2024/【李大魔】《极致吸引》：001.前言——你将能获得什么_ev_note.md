# 【李大魔】《极致吸引》：001.前言——你将能获得什么

## 概述
在本节课中，我们将了解李大魔的《极致吸引》课程，以及你将在这门课程中获得哪些知识和技能。

## 你将能获得什么
这套课程旨在帮助你在感情中低位翻盘，重塑核心自信，塑造强者框架，最终成为一个让女人心动的男人。

### 以下是课程的主要内容：

1. **心态篇**：建立核心自信
   - 通过10节视频课程，帮助你打开核心自信的认知，提升抗压能力，达到核心自信的目的。

2. **强者框架篇**：塑造高位置
   - 掌握12个核心框架，了解真正的强者如何应用策略，构建强大的吸引力。

3. **技巧篇**：释放吸引力
   - 通过技巧的举例，帮助你倒推拥有核心自信和强者框架的人如何应对各种场景，释放你的吸引力，产生更强大的威力。

## 核心概念

**公式**：心态 × 技巧 × 强者框架 = 极致的吸引力

### 1. 心态篇

**建立核心自信**

- **技巧**：情绪管理、自我认知、目标设定
- **强者框架**：坚定、果断、有责任感

### 2. 强者框架篇

**塑造高位置**
