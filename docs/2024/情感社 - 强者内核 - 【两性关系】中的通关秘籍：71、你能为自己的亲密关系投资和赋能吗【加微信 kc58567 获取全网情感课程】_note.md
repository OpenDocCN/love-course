# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：71、你能为自己的亲密关系投资和赋能吗

## 概述
在本节课中，我们将探讨如何为自己的亲密关系投资和赋能，以及如何建立强大的内核。

## 核心概念
**投资逻辑**：将亲密关系视为一种投资，关注自身实力和长期回报。

**内核**：指一个人的内在素质，包括认知、性格、价值观等。

## 投资逻辑与内核
以下是投资逻辑与内核的关键点：

### 1. 自身实力
- 投资者本身需要具备一定的认知和行为能力。
- **公式**：自身实力 = 认知能力 + 行为能力

### 2. 贪心必输
- 过度追求回报会导致关系破裂。
- **公式**：贪心 = 输掉一切

### 3. 内核修补
- 强者需要修补自身的认知和行为缺角。
- **代码**：内核修补 = 认知提升 + 行为调整

### 4. 价值观迭代
- 当代社会价值观迭代较快，导致人们容易陷入自我怀疑。
- **公式**：价值观迭代 = 自我怀疑

### 5. 外强中干
- 一些独立女性看似强大，实则内心脆弱。
- **公式**：外强中干 = 内心脆弱

### 6. 分寸感
- 分寸感是内核的体现，也是情商和社会智慧的体现。
- **公式**：分寸感 = 内核素质

### 7. 松弛感
- 强者具备松弛感，而弱者则会拧巴。
- **公式**：松弛感 = 内核强大

### 8. 非黑即白
- 非黑即白的认知逻辑会导致人际关系问题。
- **公式**：非黑即白 = 认知缺角

### 9. 亲密关系相处能力
- 具备理性表达不满和接受反馈的能力。
- **公式**：亲密关系相处能力 = 理性表达 + 接受反馈

### 10. 独处能力
- 独处能力对于维持亲密关系至关重要。
- **公式**：独处能力 = 内心强大

### 11. 内心延续
- 不要非黑即白，保持内心的平和。
- **公式**：内心延续 = 平和心态

## 总结
本节课中，我们学习了如何为自己的亲密关系投资和赋能，以及如何建立强大的内核。通过提升自身实力、修补认知和行为缺角、培养分寸感和松弛感，我们可以更好地经营亲密关系，获得幸福和快乐。
