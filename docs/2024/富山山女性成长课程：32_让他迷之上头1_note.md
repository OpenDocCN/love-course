# 富山山女性成长课程：32_让他迷之上头1

## 概述
在本节课中，我们将学习如何通过高情商的对话技巧，提升恋爱和人际交往中的互动感和趣味性。

## 接话的艺术
**核心概念**：高情商对话 = 互动感 + 生动性 + 趣味性

以下是接话的艺术要点：

- **避免单口相声**：确保对话中有你来我往，避免一个人讲，另一个人听。
- **打破距离感**：在聊天中迅速拉近与对方的距离，避免主动制造距离感。
- **避免心不在焉**：在交流过程中，不要掏手机或表现出不感兴趣的态度。

## 如何接话
以下是接话的几个例子：

1. **无聊话题**：
   - **问题**：你吃了吗？
   - **回答**：自从第一次跟你见面，吃了那个葱油爆虾之后呢，我之后就天天特别想吃这个菜。我今天吃的还是咱们俩第一次见面的时候，吃的那个葱油爆虾。

2. **激发想象**：
   - **问题**：你吃了吗？
   - **回答**：我最近都没有胃口，我还是觉得吃饭的时候，有一个可爱的人坐在旁边会比较有意思。

3. **构建场景**：
   - **问题**：你吃了吗？
   - **回答**：我最近呢特别想吃红烧茄子啊，我觉得找半天也没有好吃的，好像你朋友圈发的那个你做的，看起来会比较美味。

## 聊天禁忌
以下是聊天中需要避免的禁忌：

1. **单口相声**：避免一个人讲，另一个人听。
2. **主动制造距离感**：在聊天中，不要主动制造距离感。
3. **心不在焉**：在交流过程中，不要表现出不感兴趣的态度。

## 总结
本节课中，我们一起学习了如何通过高情商的对话技巧，提升恋爱和人际交往中的互动感和趣味性。希望这些技巧能够帮助你在人际交往中更加自信和得体。
