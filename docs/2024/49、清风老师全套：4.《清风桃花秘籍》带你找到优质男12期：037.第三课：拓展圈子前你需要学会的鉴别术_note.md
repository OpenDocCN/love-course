# 清风老师全套：4.《清风桃花秘籍》带你找到优质男12期：037.第三课：拓展圈子前你需要学会的鉴别术

## 概述
在本节课中，我们将学习如何拓展社交圈子，并学会鉴别渣男，确保情感幸福。

## 社交圈子的误区
**以下是关于拓展圈子的常见误区：**
- 认为认识的人越多越好。
- 认为微信好友数量越多越好。

## 社交圈子的本质
**社交圈子的本质是固定数量中质量的变化。**
- 人的社会关系上限数为150人，这是由心理学家罗宾·邓巴提出的邓巴数。
- 社交中的人不是越多越好，而是在合理的数量范围内，将认识的朋友进行流动变化。

## 渣男的三大特点
**以下是渣男的三大特点：**
1. **缺乏行动**：只说不做，例如承诺换手机却迟迟不行动。
2. **不愿公开关系**：不愿公开朋友圈，不愿承认关系。
3. **缺乏共同兴趣**：与你的灵魂没有碰撞，只能向你索取身体。

## 社交的三个分层
**社交就像一个夹心的饼干，分为三层：**
1. **至友知己**：数量少，但稳定提供帮助和支持。
2. **好朋友**：数量适中，有相似的兴趣和共同话题。
3. **朋友**：数量较多，有一定接近的兴趣和共同话题。

## 如何避免遇到渣男
**以下是一些避免遇到渣男的方法：**
- 从第二层加层，即好朋友中拓展社交。
- 通过共同的兴趣、相似的爱好、共同的话题找到值得交朋友的人。
