# 乐嘉性格色彩读心术（完结）·第25课：男男话题与自我接纳 🫂

在本节课中，我们将探讨男性之间关于身体形象与自我接纳的隐秘对话。通过一个关于“蛋蛋”的私密故事，我们将理解言语如何深刻影响一个人的自信与亲密关系，并学习打破这种心理枷锁的方法。

***

上一节我们探讨了人际沟通中的普遍现象，本节中我们来看看一个在男性间尤为敏感的具体话题。

你也许很脆弱。你也可以更坚强。听乐嘉亲身讲述因男人的“蛋蛋”引发的内心波澜和独白。本章题为“男男”，让我们谈谈同病相怜，四弹相对。

闺蜜之间情到深处，经常会同住一室，相互逗笑打闹。她们会笔墨调侃彼此胸部的大小。通常情况下，胸部较小的女生会羡慕胸部较大女生的“命好”，认为其在就业求职、洽谈客户时占尽便宜，惹得无数男人侧目。而胸部较大的女生则常感慨：“姐姐你有所不知，妹妹我是重心不稳，行动不便，买衣穿衣尴尬，而且还容易招惹骚扰。天下世人只看到我的‘事业线’，不见我的事业心。这一双活物就是个大累赘。”于是，这两人一个高呼“你得了便宜还卖乖”，另一个大喊“你不信我跟你换就好了”。

可奇怪的是，两个兄弟之间，即便交情过命，在一起洗桑拿共浴时，也很少会去比拼“蛋蛋”的大小。好像对男人来讲，讨论“蛋蛋”并不是一件重要的事。即便在全世界都有的阳具崇拜文化中，你常会看到艺术家精心打磨的“擎天柱”造型，但对于“蛋蛋”几乎都是草草带过，甚至不提。

有一个我相知多年的好兄弟，在我“蛋碎”后，发现我在网上发的相关文章，便发了一条长短信给我。短信内容让我非常震惊：“大哥，读完你‘淡淡’的文章，勾起我儿时回忆，日后唯有更加厚待。我现在的真实情况，天下恐怕无人知晓，包括爹娘也不知实情。看完大哥的文章，这么悲催的事儿都能被写得那么轻巧，是个真汉子。我看了以后，突然心底轻松了。这20多年以来，小弟我的两个蛋蛋形态一直定格在一大一小。值得深刻记忆的是，小时候淘气贪玩，试图用榔头砸碎玻璃珠，结果不小心弹珠打到了我的裆部。大哥，你居然受伤后忍了一个半小时才发现，你太牛了。我记得那时刚刚发育，被击中的一瞬间，天旋地转，坐在地上双手紧握，恨不得咬断自己的舌头，疼得全身不能动。不过确实是自己贪玩搞成这样，所以我无处可说，这么多年以来，一直都难以启齿。”

你想想看，我跟他这么多年的好兄弟，他突然开始跟我“青梅煮酒论蛋蛋”，这让我陡然觉得我们俩的关系已超越普通兄弟。我对他的“蛋蛋”居然也油然而生一种爱护之情，这是一种多么难以想象的兄弟情谊。我自己的“蛋蛋”还在水深火热中备受煎熬，尚未脱离苦海，可我却已开始关怀另一个兄弟“蛋蛋”的命运。我觉得我们之间的兄弟情，在那一瞬间突然得到了升华，进入了一种崭新的境界。

既然我的小兄弟开始跟我讨论“蛋蛋”的问题，我就以一个过来人的身份，装模作样地表现得似乎很老练。我问他：“嘿，兄弟，你难道没有去医院看过吗？有病就得赶紧治，千万别不好意思，男人的这种事万一耽误了就麻烦大了。”小兄弟跟我说：“大哥，这么多年以来用起来没啥问题，所以一直没去看过，不过就是看起来和摸起来好像不那么对称，不那么美观。”

当我听他这么说的时候，我无比惊讶。我说：“不美观、不对称，是你自己的感觉，还是你女友对你的评价？”我难以想象，我这个兄弟到底找的是如何神奇的女朋友，居然会评价自己男人的“蛋蛋”不美观。天哪，这个女人得见过多少男人的“宝贝”，才能得出“不美观”这样的评价？我只知道对于男人的“金枪”，古往今来有无数的文人墨客诗词赞颂，但在历史文献里很少看到有哪个著名诗人对“蛋蛋”的美予以描述。作为男人最重要的“战略支援部队”和“战斗后勤保障部队”，“蛋蛋”们经常被提及，但从未被重视。所以我完全无法想象，“蛋蛋不美观”的说法从何而来。我想了半天，我此生是否见过“美丽”的蛋蛋，但不管怎么启发自己，也无法挖掘到丝毫记忆。

我非常怀疑我这个兄弟到底找的是怎么样一个神奇的女朋友，他居然会评价自己男人的“蛋蛋”不对称。对天下所有的男人来讲，两个蛋蛋都是一大一小、一低一高。我在澡堂子里从来没见过有哪个“神圣”的蛋蛋是左右护法，如出一辙。我还特别担心自己会不会见识短浅，所以我就把我所有认识的泌尿科、男性科的专家全部问了一遍。当他们听了以后，纷纷表示：“哦，两个蛋蛋对称一样大，这种神奇真的是世界罕有啊。”并且他们叮嘱我，如果将来看到这种蛋蛋，一定要赶紧告知，给他们一个参详观摩的机会。所以我完全不知道男人蛋蛋不对称这种说法从何而来。

结果我这个小兄弟告诉了我一句让我当场下巴落地的话：“大哥，我没有给任何女人看过我的蛋蛋，我也不允许她们仔细看。我记得这是我娘跟我这么说的。”我差点晕倒，是他娘说的，是他老妈对他说的。天哪，他老妈的一句无心之语，就让我这个兄弟在如此巨大的心理阴影之下晃荡了十多年，让他在面对上天赐予人类最美好的性爱之时，不能坦然，无法坦荡，羞于和他所爱的女子一起浑然天成。

天下的老妈呀，你们真的不知道，就这么随意的一句话，也许只是一个小小的玩笑，但是对于自己孩子的终生幸福，你知道影响会有多大吗？我想起我的另外一个兄弟。我记得以前他跟我提过他内心最深处的自卑。他和他女朋友在行云雨之欢时，正运动得特别欢快，他女朋友突然说了一句话：“哎，你进去了没有啊？”就这一句话，他当场差点吐血，自尊心荡然无存，马上就一蹶不振。更要命的是，他那个女朋友还马上鼓励他：“没关系没关系，下次会更好的。”本来不说还好，越说越完蛋。从此以后，这个兄弟就“废掉”了。

还有更让人要命的事情是，我这个兄弟还特别喜欢看欧美的A片，总想着从当中琢磨出一些“奇技淫巧”。但欧美人本来就跟我们这个民族在尺寸长短上有巨大差别，再经过A片的“后天训练”，本就没有可比性。可怜我那个小兄弟，以己之短，比他人之长，长他人志气，灭自己威风。时间长了以后，连战斗的勇气也没了，还没“出家”，基本上相当于已经把自己给“自宫”掉了。一条活生生的好汉，就被他的女友给这样“扑灭”了。

我看着那个一直纠结在“不美观”、“不对称”当中的小兄弟，我决定要帮他一把，给他加个码，要给他人生重重地敲一记响锤。我要帮他彻底打碎那久久不能释怀在他心头的噩梦。

我很清楚，我现在如果告诉给他“你好厉害哟，你好棒哟”，这种话屁用都没有。因为这种话应该是他女朋友跟他实战完了以后跟他说的话，关我一点干系都没有。所以我如果那么跟他说，除了表示虚伪，就是派送鸡汤，没有任何作用。

我想了一想，我应该怎么办呢？突然我想到一个点子。我选了两张我自己“蛋蛋”的图片发给他：一张是手术以前的狰狞惨烈，第二张是手术以后的静仰安息。两张图一对比，天差地别，宛若隔世。但是我相信，即便只是手术后的那张图片，也足以让他彻底颠覆对于“对称”和“美观”这两个词的定义，让他重新解读人生。

果然没过多久，他就发了一条短信过来，短信上面写的是：“大哥大哥，嘿嘿，我看了你的图片，我突然发现我自己的蛋蛋好美哦。”善哉善哉，救人一命，胜造七级浮屠。
