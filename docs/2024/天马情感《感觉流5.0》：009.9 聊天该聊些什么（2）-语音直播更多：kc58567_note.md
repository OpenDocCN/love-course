## 核心概念一：不可避免的价值判断

我希望你们能记住并反复理解下面这段话：**价值判断**不仅是男生追女生时会发生的环节，更是你在社会上与他人社交时，必定会经历且无法逃避的过程。

为什么无法逃避？
首先，如果你不敢表达自己，甚至嫌弃自己的工作而不敢提及，那么你传递给别人的感觉就是负面和充满负能量的。价值判断的依据并不仅限于你**说了什么**，它同样包括你的**情绪**以及你对别人的**态度**。你畏缩的态度和焦躁的情绪，同样会向别人传递出你是一个社交价值较低的人。

其次，价值判断并没有那么可怕。无论你是工人、销售还是银行职员，这都只是一份职业，你的工作都有其价值。在社会中，如果总是与人比较金钱和地位，你永远无法胜出，因为总有比你更富有、更出色的人。

你需要学会的是：**分享自己的价值，并与他人进行价值交换**。

例如，如果你是一名装修工人，你应该告诉别人你的职业。当别人需要装修时，他首先会想到你。当别人明白了你的价值所在，并且他有相应需求时，他就会找到你，而你也能帮助他。你们之间是完全平等的，能够进行价值的互相交换。

**公式：有效社交 ≈ 主动展示自我价值 + 寻求平等价值交换**

如果你不敢表达、不敢展示自我的价值，带来的后果就是别人不知道你的价值所在。那么，你对这个社会而言就是没有价值的，你就会被淘汰。

追女生同理。如果你不透露自己的信息，那么你对这个女生而言也是没有价值的，不具备值得接触的可能性。除非，你只想成为一个“工具人”。

## 警惕最大误区：沦为备胎与工具人

在从陌生人发展到朋友这个“红灯区”，还存在一个巨大的误区，即沦为“备胎”、“好人”或“工具人”。很多人聊着聊着就会掉进这个区域。

你要明白：**朋友关系**与**备胎/工具人关系**是两种不同的情况。成为朋友不代表你就是备胎。如果你能按照正确的指引正常交流，你不会成为工具人，而会成为她的朋友。

出现备胎情况通常有两点原因：

以下是第一点原因及对策：
*   **心态问题**：你的心态本身就有偏差。解决方法请回顾第七节课的内容，那里详细讲解了追求女生时应有的正确心态。

以下是第二点核心原因：
*   **无条件付出价值**：这违背了前面讲的价值交换原则。

**代码示例：错误的关系模式**
```plaintext
// 错误模式：无条件单方面付出
if (女生提出请求) {
    你无条件付出(时间、精力、金钱);
    返回值 = 女生的一句“谢谢”;
    你的情绪状态 = 开心、荣幸;
}
```

什么叫做无条件的付出价值？例如：女生叫你帮忙搬家，你开心地去了，辛苦一天后，她连一杯水都没给你买，反而奇怪你怎么还不走。你付出了时间和精力，只得到一声“谢谢”，却还感到开心。

又或者，女生在几十公里外，来例假了，你煲了红糖水坐两小时公交送去，她对你说“你人真好”，你便激动不已。你觉得这没问题吗？

这完全不符合常理。如果主角换成你的朋友或家人，你会因为能请朋友吃饭、送朋友上班而感到无比激动和荣幸吗？正常的情绪可能是觉得麻烦，甚至想收取一点劳务费。这才是正常的社交态度。

你把为另一个人付出、请他吃饭、送他上下班当作一件让你感到开心和光荣的事，这本身就是非常可悲的行为。只有奴才和被所谓“爱情”冲昏头脑的人才会这样。如果你们已经是情侣关系，那另当别论。但在女生还不是你女朋友的情况下，这么做是完全错误的。

你是在付出你的金钱和精力，却觉得自己很荣幸。这与工人刷墙、雇主付钱的等价交换性质完全不同。你这叫做**无条件的付出和讨好**。你一直在输出价值，却没有要求对方给予回报。那么你成为备胎，该怪谁呢？

## 如何避免：践行等价交换原则

你不想再被女生当备胎，不想再收到“你是个好人”的卡片，其实很简单：管好你自己。大部分备胎现象都是你自己一手造成的。

**停下无条件的讨好，学会等价交换。**

例如：
*   你送女生上班或上学，让她请你吃顿饭，这不过分。
*   你唱歌好听，她让你唱给她听，你就让她给你画幅画，这很正常。
*   你们一起看电影，你买了电影票，那就让她请吃晚饭。

**公式：健康互动 ≈ 你的付出 + 对方相应的回报**

你们是互相平等的。你的时间同样值钱，女生愿意陪你看电影，并不代表你低她一等。当然，等价交换需要讲究方式和情商。例如，如果说“我帮你通马桶，你得让我抱一下”，这就显得很低级，像一场赤裸的交易。但如果说“我帮你写作业，你给我揉揉肩”，这就自然很多。

如何更高情商地引导女生付出，进行等价交换，而不显得尴尬和僵硬，这其中涉及各种话术和技巧。这些内容会在私教一对一的高阶课程中详细教授。

但在这里，我想让你明白一个核心道理：**追女生不一定非要请她吃饭、给她买礼物。甚至，一旦你开始这么做，反而容易被她当作备胎和“好人”，你永远得不到她，她也永远不会喜欢上你。**

这与你想象中的“追求流程”不同。你想象中“追女生就是要感动她、给她安全感、告诉她我很喜欢她”的做法，往往是错误的。
