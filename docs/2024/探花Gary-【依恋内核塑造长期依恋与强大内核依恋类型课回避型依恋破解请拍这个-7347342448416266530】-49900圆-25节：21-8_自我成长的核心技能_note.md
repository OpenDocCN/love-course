# 探花Gary依恋类型课第21-8节：自我成长的核心技能 🧠

在本节课中，我们将要学习“自我洗脑”这一核心技能，它是获得长期稳定提升、打破枯燥无趣生活的关键。我们将探讨大脑的运作模式、为何改变如此困难，以及如何通过正向思维重塑自己，从而建立理想的生活。

## 大脑的流程化与舒适圈

上一节我们介绍了自我提升的总体框架，本节中我们来看看阻碍我们改变的根本原因——大脑的固化思维。

我们活在这个世界上，是由大脑控制着我们的日常生活和时刻需要做出的选择。但大脑也有自己的习惯，这些习惯形成了我们固定的思维模式。这个思维模式决定了我们如何思考、从什么角度看待问题、脑海里容易产生什么想法，并最终导致我们做出何种选择。

然而，大多数人完全没有控制自己大脑意识的能力。你会发现，每一天百分之八九十的时间里，你的日常行为和选择都是流程化的。也就是说，你的大脑不需要任何思考，也不愿意去思考，全部都是你的意识根据形成的习惯在自动反应。

以下是流程化行为的例子：
*   从早上起来洗漱。
*   决定吃什么。
*   决定去哪里、做什么。
*   决定和谁说话。

举个例子，我以前常去一个汉堡店，每次都会点一个汉堡，并要求放西红柿和酸黄瓜。而每一次做汉堡的人都会照做。反反复复，这就形成了他的日常流程习惯，也是他对我的条件反射。这就是一个人的大脑日常的流程化程式，他每天到了这个场景后，就失去了独立思考的能力，只能执行流程化操作。

大部分人其实很少需要启动大脑，去思考日常流程以外的东西。久而久之，你的大脑不习惯，也不愿意去引入任何超出日常的新思维和新东西。这就是每个人给自己精神上设置的限制，也可以理解为是我们的舒适圈。任何舒适圈以外的东西，我们的大脑都会不适应。

## 大脑的抗拒与借口机制

理解了舒适圈的概念后，我们来看看当试图突破它时，大脑会如何反应。

我们的内心会对任何习惯性的改变产生排斥情绪。同时，你的大脑会给你找无数的借口，就是为了不让你打破原来的日常流程。

比如，每当你跟身边的朋友、家人、同事或同学提起任何超出他们认知或舒适圈以外的新东西时，几乎每一次他们的第一反应都是排斥。然后你会看到他们会找任何借口或负面点来否定你提出的新东西。这就是最好的证据和证明，他们的大脑在排斥超出其设定局限以外的东西。

这个大脑的固化设定，导致他们的选择性焦点只会放在否定新事件的借口和负面点上，而不会深入思考新事物。再加上所有人从小到大，还不停地被社会洗脑。每个人都形成了一种固化的思维模式和同样的想法。哪怕你自己从来没有通过一手经验验证过什么，但因为身边所有人都这么想，所以你坚信这样的一定是对的。

我们大脑的这个流程化固定设置，是导致我们在试图改变自己、让自己提升的道路上无法坚持前进的最大原因。这就是为什么你会发现，学了很多内容，看了很多课程，但在现实生活里，你的行动、日常和结果没有丝毫改变或改变不大的核心原因。

## 改变为何如此困难：实例分析

现在，让我们通过一个具体例子，来看看大脑的抗拒机制在现实中如何运作。

以下是超出大多数人日常流程和舒适圈的例子：
*   外出约会。
*   每天健身。
*   定期外出社交。

每当你尝试去打破并改变当下这个枯燥无趣、没有结果的生活流程时，你的大脑就会开始劝退你自己。各种借口、各种自我破坏、各种低能的想法会马上引入到你的脑海中。

比如，你想出去认识一个新的人，或想出去社交认识新朋友，这些都不是你日常生活流程中的东西。那你前期去执行的时候，遇到的阻力会非常大，因为你的大脑在抗拒这些新东西。你的大脑不想思考，它很懒惰，它在劝退你。

你的大脑会跟你说：
*   `算了，太难了，别浪费时间。`
*   `还不如在家打游戏多舒服。`
*   `我不够好看/不够有钱/不够好，谁会搭理我。`
*   `我要工作，我太忙了，我没时间。`

这就是为什么99%的人没有任何结果，并且还浑然不觉。他们自愿接受大脑负面的洗脑，屈服在找借口的思维陷阱里面。我曾经也是这样。学生时代的我了解到了社交技巧，一开始我自己出门实战，用行动试图改变自己的时候，阻力非常大。在长期关系里，我要改变自己的讨好和低自尊，遇到的阻力也非常大。

## 破解之道：认识高我与小我

既然大脑的抗拒是主要障碍，那么唯一的破解方法就是重新给大脑洗脑，重新设置大脑的模式和习惯。只有这样，才能让你的日常行为和流程改变，才能让你持续坚持做能获得结果的事。

当你试图去改变一些当下生活流程，去做新的可能带来结果的事情（如日常外出社交或健身）时，你的大脑肯定会反抗，给你制造无数借口。这时，我们要如何让大脑停止找借口？现在我们必须意识到，我们的大脑里其实有两个自我：**高我** 和 **小我**。

*   **高我**：代表更高维度的自我，是宏大的、伟大的、智慧的，有点像你的宏观意图和理想。
*   **小我**：代表低维度的自我，是小气的、关注当下的，有点像你的微观意图和即时需求。

我们每个人都有高我和小我。举一个例子，假设你是一个程序员，是某个互联网公司的软件工程师。你每天的工作就是开会、敲代码，然后再开会、敲代码。

*   你的**高我**会觉得：`我做这份事业是为了改变世界，做出一份贡献。我写的代码能改变平台功能，造福用户，这体现了我的价值和使命。`
*   你的**小我**会觉得：`我每天工作10小时，是为了挣1万或2万块的工资养活自己。如果表现好被提拔，我就能管人、涨工资、存更多钱。`

你会发现，不管是脑子里的小我还是高我，只要你把他们两个拉到同一个方向，用正面理由介入，让你自己前进，哪怕这个理由或借口很小气、来源于你的小我，但只要是为了你想要的目标去前进，你就能持续坚持并获得结果。

相反，大部分人也能想很多负面借口来劝退自己。比如你可能会想：`我算老几，就是个普通老百姓，还轮得到我来改变世界？每天累得要死，回家打游戏时间都不够，工资还没朋友多，干嘛还要做这个工作？`不出意外，你很快就会辞职或被辞退。

现在你可以看出来，其实是你大脑的思维决定了一切。思维模式决定了你人生中所有的事情，包括学习、自我提升、事业、长期关系等。这就是为什么我们要给自己正向洗脑的原因。因为到最后，你会赢，你会拿到结果，你会赢得人生。这个自我洗脑的操作，同时也是为了激发你大脑独立思考的能力，让你从日常固化流程中解脱出来。

## 实践方法：正向自我洗脑练习

理论已经清晰，本节我们将学习具体的实践方法，即正向洗脑的操作步骤。

正向洗脑的操作是根据我们每天的状态，去激活大脑里面的高我和小我。对于大多数人来说，日常的小我可能会更多，因为状态低迷、缺乏激情和动力的时候比较多，而高我则相对较少。

以下是正向自我洗脑的练习步骤：

1.  **写下目标**：把你的目标、想要的结果具体地写出来。
2.  **分列原因**：在目标下面分两栏，一边是“高我原因”，另一边是“小我原因”。
3.  **填充内容**：把你能想到的所有原因，从高我的角度和从小我的角度全部写下来。
    *   **关键**：哪怕有些小我的想法很难看、很自私，但要明白，这是在低迷状态下，能让你不偏离道路的正向洗脑，是支持你前进的思维逻辑。
4.  **定期回顾**：每当你发现自己开始找借口、想放弃、逃避或感觉累的时候，把写好的纸拿出来看一看、念一念。
5.  **动态补充**：在原来写好的基础上，再添加几个当下想出来的新的高我或小我的原因。

## 应用实例：社交与两性关系

让我们将上述方法应用到一个实际场景中，以便更好地理解。

每当你自己要出去认识新的社交对象，或者想要和伴侣严肃解决问题的时候，一旦你开始给自己找借口、劝退自己，这时就可以启动你的高我和小我思维。

*   **高我思维**可以这样想：
    *   `我每次社交/表达，是为了改变我的人生。`
    *   `为了成为我自己想要成为的样子。`
    *   `为了让我的关系变得更好。`
    *   `为了我的下一代，为了我的家庭可以更好。`
    *   **做法**：把这些高我的思维模式和想法写下来，反复给自己巩固。
*   **小我思维**也可以这样想：
    *   `我每次出门是为了认识一个人，满足当下的新鲜感。`
    *   `我的每次表达，都是为了满足自己的表达欲。`
    *   `反正已经陷入负面情绪，没有更好玩的事可做。`
    *   `心情不好，出门找个人吐槽一下，释放坏情绪。`

很明显，你的这个小我虽然很小气、很自私，但是它会和你的高我在一起，把你往你想去的方向推。当大我和小我都能在同一个方向努力时，我们的大脑就能跳出原有的固化思维和模式，并且停止找借口把你拉回原来的日常流程中。这个就是正向的自我反洗脑。

## 总结：重塑人生的起点

从出生到现在，我们每个人本来就被社会洗脑了。这些社会的洗脑和影响，打压了我们的自尊自信，限制了我们很多的人生结果。就算你发现了新的人生可能性，这个洗脑的固化思维也会让你无法开启新的模式和人生新的现实。

这时，我们就只能利用自己大脑本身的机制去给自己反洗脑，从而达到自身想要的目的和结果，让自己在另外一条人生道路上、在你真正想要的改变中去坚持。而这一切，都要从掌握自己的大脑开始。

本节课中，我们一起学习了：
1.  **大脑的流程化本质**：我们的日常行为多由习惯驱动，形成了舒适圈。
2.  **改变的阻力来源**：大脑会抗拒舒适圈外的变化，并制造借口。
3.  **思维的双重结构**：我们拥有关注理想的高我和关注当下的小我。
4.  **破解的关键方法**：通过正向自我洗脑，将高我和小我的力量引导至同一方向。
5.  **具体的实践步骤**：写下目标，分别从高我和小我角度列出行动原因，并定期回顾强化。

掌握自我洗脑的技能，是自我成长和塑造强大内核的核心。我们下节课再见。
