# 40、纳兰女性终身成长视频课n：97.97、模特妆容之画眉+腮

在本节课中，我们将学习一个完整的日常妆容实操流程，涵盖画眉、打腮红、遮瑕、提亮、修容和涂口红六大核心步骤。我们将通过模特兰星的实例演示，讲解每个步骤的原理、技巧和注意事项，帮助你掌握打造自然精致妆容的方法。

## 一、画眉：塑造清晰眉形

上一节我们介绍了妆容的整体思路，本节中我们来看看如何画好眉毛。画眉的关键在于拥有一个清晰的眉形。如果脑中无清晰眉形，画眉就会变得困难，常出现高低不一或形状不对称的问题。

眉毛大致可分为两类：
*   **平眉**：眉形平缓，弧度较小。
*   **弯眉**：眉形带有明显弧度，只是弯曲程度不同。

以下是针对不同眉毛条件的建议：
1.  **眉毛颜色浅、毛发量少**：建议直接做半永久纹眉。成熟的技术能为你提供一个清晰的眉形，省去日后80%的修眉和画眉精力。
2.  **眉毛条件好但无清晰形状**：建议找专业修眉师傅，修出适合你的眉形。之后只需填补空缺即可，同样能节省大量精力。

**画眉实操步骤：**
我们以模特兰星为例。她的眉形基础好，仅局部有颜色不均和毛发空缺。日常妆的目标是让人感觉“长得好看”而非“画得好看”，因此只需填补空缺，使其均匀自然。

1.  先用眉梳梳理眉毛。
2.  使用眉笔（如植村秀砍刀眉笔或其平价替代品），在毛发稀疏、颜色浅淡的位置填补颜色。填补时需边画边照镜子，及时修正。
3.  填补完成后，眉形已大致完整。为显得秀气，可加强眉尾的尖细部分。
4.  最终效果是眉毛更完整精致，但仿佛自然生长。

**核心要点：** 将专业的事交给专业的人。把控眉形是画眉最关键的一步，可求助修眉师傅或选择半永久。省略自己摸索眉形的步骤后，画眉会变得非常简单。

## 二、打腮红：营造自然气色

眉毛完成后，我们进入下一步——打腮红。腮红能提升气色，营造氛围感。若无专用腮红，可用喜欢的口红颜色替代。

**腮红打法有三种：**
1.  **圆形腮红**：打在眼睛下方、苹果肌偏上的区域。此法不挑脸型，能显得人可爱。
2.  **横向腮红**：横向打在脸颊区域。适合长形脸，能从视觉上拉宽脸型。
3.  **斜向腮红**：从颧骨最凸点向内约一指的位置，沿脸部斜向上下晕染。此法有一定修容效果，也可在下眼睑后1/3处晕染，营造氛围感。

**腮红实操步骤：**
根据兰星的整体感觉，我们选择**圆形腮红**打法。
1.  将膏状腮红涂在手背，均匀推开，如同匀粉底。
2.  采用**少量多次**的原则，将腮红拍在眼下目标区域。
3.  **关键注意**：腮红位置**不能低于鼻底（中庭底线）**，否则会拉垮面部。
4.  为增强氛围感，可在眼下稍作叠加。
5.  若边缘明显，用手指轻轻晕开，达到“仿佛皮肤自然泛红”的干净匀称效果。

## 三、遮瑕：精准遮盖瑕疵

打完腮红后，我们处理遮瑕。在第一遍粉底后，大部分瑕疵已被遮盖。遮瑕的目标是处理局部明显瑕疵，并进行面中提亮。

**遮瑕实操步骤：**
1.  **遮盖黑眼圈**：
    *   黑眼圈常带青、紫色，需用偏橘调的遮瑕膏校正。若颜色过深，可与浅色遮瑕调和，调出适合肤色的颜色。
    *   用遮瑕刷蘸取少量产品，**在手上匀到几乎看不见**后再上脸。
    *   只点在有色素沉淀的位置，用刷子背面或轻转刷头的方式，将颜色轻轻点按晕染均匀。
    *   **遮瑕一定要少**，看似无用，效果实则明显。
2.  **遮盖法令纹**：
    *   法令纹是细长的阴影。选用亮色遮瑕，**立起刷子，用刷子边缘**蘸取产品。
    *   同样在手上匀开后，精准提亮法令纹的阴影线，使其与周围肤色融合。
    *   **正确预期**：法令纹、眼袋是动态纹，化妆只能**减弱，无法完全消除**。
3.  **定妆**：
    *   遮瑕后需**立即定妆**，防止移位。
    *   用散粉刷均匀蘸取散粉，轻轻按压在遮瑕部位。

**遮瑕原则：** 哪里有瑕疵遮哪里。大面积瑕疵可用美妆蛋，小面积瑕疵用刷子。

## 四、提亮：塑造面部立体感

遮瑕让面部趋于平整，接下来通过提亮增强立体感。大面积提亮建议使用**哑光**产品，避免满脸“金角大王”似的亮片。

**提亮实操步骤：**
1.  **哑光提亮**：使用浅肤色粉底、偏白色散粉或粉饼。
    *   用美妆蛋蘸取产品，在手上匀到很薄后再上脸。
    *   **常用提亮点**：眼下三角区、山根、鼻尖、眉骨转折点、嘴角。
    *   依次在这些位置少量多次按压提亮。
2.  **高光提亮（可选）**：如需更强妆感，可小面积使用粉质高光。
    *   **三个位置**：鼻头、山根、颧骨内侧。
    *   少量点缀并晕染自然，在转头时呈现若隐若现的光泽。
3.  **局部定妆**：提亮后，若T区或额头出现油光，可再次少量定妆。

**灵活调整**：鼻部提亮点可根据脸型调整。中庭长，则将山根和鼻尖提亮点向中间靠拢，视觉缩短中庭；中庭短，则向外扩展，视觉拉长中庭。

## 五、修容：收敛轮廓

提亮后，我们进行修容，让面部轮廓更精致。修容产品可用专用修容盘，也可用眼影盘中的深色。

**修容核心思想**：面积要小，位置要精准。
**修容实操步骤：**
1.  正面照镜子，找到脸部**最宽的点**（如颧骨凸点、下颌角）。
2.  将修容粉**精准按压在这个点上**，不要大面积铺开。
3.  先确保颜色附着在目标点，再用手指或美妆蛋将周围颜色向四周晕染开，但最深的颜色应始终集中在最初的那个点上。
4.  通过以点为中心的扩散式晕染，达到自然的收缩效果。

## 六、口红：点睛之笔

最后一步是涂口红。日常妆可选择浅淡、元气的颜色。

**口红实操步骤：**
1.  涂口红时可微微微笑，便于勾勒唇形。
2.  直接涂抹或用口红刷上色。
3.  **关键步骤**：用棉签将唇线边缘仔细晕染模糊，**避免出现明显的口红边界线**。这样涂出的口红自然减龄。
4.  可在唇部中央稍作叠加，增加层次感。

至此，半张脸的完整日常妆容就完成了。快速补齐全妆后，整体效果是模特依然是她自己，但变得更加动人、精致。
