## 核心概念
- **BC类女生**：指对你态度一般，回复慢、少且缺乏情绪的女生。
- **窗口识别**：通过五个方面来判断女生对你的态度，包括字数、语音、回复条数、表情和话题接住能力。

- **年龄**：23岁以上，有丰富的感情经历。

## 列表：窗口识别的五个方面
1. **字数**：字数少且与话题无关，窗口小；字数多且与话题相关，窗口大。
2. **语音语气**：通过语气判断女生的态度。
3. **回复条数**：回复条数多且带表情，窗口大；回复条数少，窗口小。
4. **表情**：表情丰富，窗口大；表情单一，窗口小。
5. **话题接住能力**：能够接住话题，窗口大；无法接住话题，窗口小。

1. **通过完整的心理历程说明**：在女生给你不好的反馈时，告诉她你为什么会这么做。

3. **学会示弱**：展示自己的脆弱一面，让女生产生共鸣。

2. **细节+心路历程**：通过细节和心路历程化解女生的冷漠态度。
