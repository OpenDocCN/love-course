# 13、高段位男神进阶营——浪哥出品：11.男神进阶营《处世篇》：11. 破解强势

欢迎来到高段位男神进阶营的课程学习。本节课程的主题是破解强势。在日常生活中，我们经常会遇到一些强势的人。有些人可能很厉害，而有些人可能并不出色，但他们同样表现出强势。

**如何与强势的人沟通**？

首先，我们需要明确一个基础认知：强势并不一定是一件坏事。只要我们选择正确的沟通方式和技巧，强势的人其实都是“纸老虎”。

**强势心理分析**

1. **内心脆弱**：强势的人内心往往比较脆弱，他们通过外表的强硬来支撑自己。
2. **不与人交流**：有些人根本不给你说话的机会，只要你开口，他们就会打断你。
3. **无理由的强势**：有些人没有任何理由地表现出强势。

**应对强势的人**

1. **分析强势原因**：了解对方强势的原因，分为以下几种类型：
   - **有底气**：例如，有钱或有丰富的社会资源。
   - **装出来的**：通过强势来掩盖自己的弱点。
   - **无理由的**：性格中本身就带有强势。
2. **应对策略**：
   - **地位高的人**：以学生或请教者的身份与他们沟通，满足他们的“好为人师”心理。
   - **虚荣心强的人**：多几分同情，维护他们的尊严。
   - **无理由强势的人**：摆明道理，坚持原则。

**应对强势的人的四个方法**

1. **自信**：保持内心强大，并足够自信。
2. **明确拒绝不合理的要求**：坚持原则，不被人摆布。
3. **尽量不发生冲突**：避免正面冲突，保持风平浪静。
4. **独立思考**：不要被强势的人控制，坚持自我。

**总结**

面对强势的人，我们要坚持自己的底线，根据不同情况采取不同的应对策略。本节课中，我们一起学习了如何破解强势，希望对大家有所帮助。
