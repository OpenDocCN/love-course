# 小怡《情绪吸引力-人性篇》：012.12-聊天五步法-03筛选

## 概述
在本节课中，我们将学习如何使用筛选技巧来判断对方对你的兴趣是否真实。

## 筛选的定义
筛选是一个确认对方对你兴趣是否真实的方法。它类似于使用筛子分离粗细物料的过程。

### 核心概念
**公式**：筛选 = 表达态度 + 观察反应

## 筛选的步骤
1. **破冰和了解阶段**：与对方充分交流，了解彼此。
2. **观察兴趣信号**：注意对方的兴趣信号，如询问信息、称赞、邀约等。
3. **表达态度**：通过言语或行为表达你的态度。
4. **观察反应**：观察对方对你的态度的反应。

## 筛选的注意事项
1. **使用时机**：在破冰和了解阶段之后，对方对你表达兴趣之后使用。
2. **筛选的能量与兴趣信号强度匹配**：根据对方的兴趣信号强度选择合适的筛选方式。
3. **筛选的标准要准确**：确保筛选标准能够准确区分对方的真假兴趣。

## 筛选的误区
1. **使用时机不对**：过早使用筛选或在没有兴趣信号之前使用。
2. **筛选的能量与兴趣信号强度不匹配**：使用过强或过弱的筛选方式。
3. **筛选的标准无法准确区分意图**：筛选标准不能准确判断对方的真实意图。

## 筛选的例子
1. **健身房场景**：通过表达不想花钱买课，但想进一步接触的态度来筛选。
2. **男女关系场景**：通过表达择偶态度来筛选。

## 总结
筛选是聊天五步法中的重要环节，可以帮助我们判断对方对你的兴趣是否真实。在使用筛选时，要注意时机、强度和准确性，避免误区。如果筛选未通过，不要强求关系升级，继续在了解阶段下功夫。
