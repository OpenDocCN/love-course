# 《二月老师的核心自信》（综合版）：41-附加：生命的内耗_ev

🎼好，这节课我们将探讨生命的内耗。若自我接纳是自信的基础，则生命的内耗便是自信的最大杀手。何为生命的内耗？简言之，即自我搏斗与自我消耗。例如，后悔、纠结等。

🎼为何条件相似的两人，一个轻松自在，另一个却郁郁寡欢？很多时候，我们感到痛苦，并非外界问题，也非事情本身，而是内心的冲突。许多人对于事情的反应过于敏感，外界任何风吹草动都会激起情绪波澜。处于这种境况中，即使一天无所事事，也会感到无力与疲惫。这便是自我消耗。内耗如同用勺子慢慢挖空自己，因此内耗是自信的最大杀手，也是最不必要的消耗。

🎼我们每天都要应对生活、工作、爱情等琐事，为何还要与自己对抗？这一点值得我们深思。抑郁症的形成，便是内心不断自我搏斗。那么，如何停止内耗？

🎼以下是一些建议：

1. **停止后悔**：精神病学家指出，许多病人将时间花在缅怀过去、后悔过去的事情上。这种反复的懊悔只会让自己陷入精神消耗。人生难免有遗憾，一时的反思可以推动进步，但长久的懊悔只会让我们举步维艰。

2. **活在当下**：我们每个人的世界里都存在着三条时间线：过去、现在、未来。然而，大多数人活在过去或未来。活在当下，感受现在的一切，才是唯一的真实。

🎼以下是关于活在当下的详细解释：

- **三条时间线**：过去、现在、未来。
- **活在当下**：感受现在的一切，而非过去或未来。
- **避免后悔**：活在当下，感受自己的真实。

🎼如何避免后悔类的自我内耗？

- **活在当下**：感受现在的一切，而非过去或未来。
- **以旁观者角度看待过去**：回忆过去时，以旁观者角度，避免情绪干扰。

🎼总结：本节课中，我们学习了生命的内耗及其危害，以及如何停止内耗。希望这些内容能帮助大家更好地理解自己，活在当下，享受生活。
