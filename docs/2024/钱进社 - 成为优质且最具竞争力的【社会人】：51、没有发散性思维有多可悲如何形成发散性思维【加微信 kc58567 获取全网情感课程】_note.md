# 钱进社 - 成为优质且最具竞争力的【社会人】：51、没有发散性思维有多可悲如何形成发散性思维

## 概述
在本节课中，我们将探讨发散性思维的重要性以及如何培养这种思维方式。

## 发散性思维的重要性
发散性思维对于个人成长和问题解决至关重要。**公式**：发散性思维 = 创新思维 + 多角度思考。

## 如何形成发散性思维
以下是培养发散性思维的步骤：

1. **思维开拓**：相信任何事情的逆转基础是思维的开拓。
2. **打开思维**：勇敢地打开自己的思维方式，即使有些想法看似天马行空。
3. **过滤想法**：过滤掉纯粹是扯淡的想法。
4. **细化整理**：将天马行空的想法进行细分化的整理，形成完整式的方案。
5. **落地实施**：将想法付诸实践。

## 发散性思维的应用
以下是发散性思维在解决问题中的应用：

1. **面对问题**：将问题细分，记录与问题相关的信息和自身可以做出的反馈。
2. **提问**：向自己提问，例如“为什么”、“什么时期”、“如何”等问题。
3. **逆向思维**：从相反的角度思考问题。
4. **盲点思维**：将盲点思维嫁接到熟悉的事件和场景中，辅助思维。

## 性格与发散性思维
拥有发散性思维的人性格具有兼容性，能够兼容不同的经历和场景。

## 教育与发散性思维
教育对发散性思维的形成有重要影响。从小父母和老师会给孩子贴标签，限制孩子的思维。

## 自我突破
要培养发散性思维，首先要摆脱社会对某一性格特征的狭隘定义，撕掉标签，接纳自己的多样性。

## 总结
本节课中我们一起学习了发散性思维的重要性以及如何培养这种思维方式。希望这些内容能帮助你提升自己的思维能力和问题解决能力。
