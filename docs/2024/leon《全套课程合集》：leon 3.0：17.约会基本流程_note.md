# 《leon全套课程合集》：leon 3.0：17.约会基本流程

🎉 欢迎来到第十七节课。本节课我们将探讨约会的基本流程。

### 一、明确共识

首先，我们需要明确一个共识：搭讪、微信聊天和约会是一个整体，并非分割的。按照我们的撩眉五步曲，即打开前提、评估、叙事、收尾。

### 二、约会阶段

约会大概在评估和叙事阶段。如果你在微信上使用5句话邀约，其实还没有进入评估环节。

女人之所以邀约你，是因为她对你好奇，想从好奇的角度接触你。在约会过程中，你需要进行大量提问，挑战她的框架，并在这个过程中，她也会反过来挑战你。

### 三、价值感建设

在这个过程中，你需要处理悱恻，并植入人设价值感。当价值感拉到60分，女生对你有好感时，我们再进行叙事。约会的核心是评估环节。

### 四、拉伸价值感的方式

1. **评估**：提前准备好5到10个问题去挑战她的信念，当女生开始自我解释、寻求认同时，你的目的就达成了。
2. **挑战**：女生会主动来挑战你评估你，因为女人最擅长评估和挑战男人。
3. **人设植入**：在互动过程中，暴露你的年龄、职业、家乡、成就、性格、思想、人生追求和价值观等，以拉伸价值感。

### 五、叙事环节

在价值感建设到位后，我们进入叙事环节。让女生了解你的整体轮廓、性格思想、价值观等。

### 六、升级动作

1. **肢体测试**：肩胳膊是C级普通测试，捏头发、捏脸是B级测试。通过这两个指标后，你可以带她去私密空间。
2. **言语测试**：了解她对你的看法，进行深层次交流。你可以问：“到现在为止，你觉得我对你印象怎么样？有60分吗？”
3. **升级**：体验听力的感觉，握手，然后推一下，以维持张力。

### 七、转场私密空间

在肢体和言语指标通过后，你可以带她去私密空间，如公园、电影院、家或酒店。

### 八、约会流程

1. 第一次约会：设计好私密空间，了解她的安排，以转场带入私密空间为导向。
2. 第二次约会：如果第一次没有结果，不要动手动脚，只需简单升温，刺激欲望。
3. 第三次约会：在信任感和私密空间的基础上，可以带她去家里或酒店。

### 总结

本节课中，我们一起学习了约会的基本流程。希望你能按照这个模板去运用，让你的约会更加简单。😊
