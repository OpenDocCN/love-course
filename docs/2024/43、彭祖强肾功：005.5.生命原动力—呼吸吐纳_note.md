# 43、彭祖强肾功：005.5.生命原动力—呼吸吐纳_note

在本节课中，我们将要学习呼吸吐纳的专业知识。吐纳本质上是练习呼吸、锻炼肺功能的功法。掌握呼吸，就掌握了生死的关键，因为呼吸是每个人的长寿密码。

很多人不了解呼吸的奥秘。古人云：“圣人之息在于踵，众人之息在于喉。”观察婴儿便可发现，他们呼吸时腹部自然起伏，此为腹式呼吸。随着年龄增长，呼吸逐渐变为胸式呼吸。生命临终时，呼吸则变得急促、表浅。由此可见，从生到死的过程，正是呼吸逐渐变浅的过程。

那么，“圣人之息在于踵”是什么意思呢？许多练习周天呼吸法的修行者，能够从脚底吸气，推动气息在体内循环运转，从而将呼吸拉得更深长。这引出了一个核心概念：**让呼吸变得深长、缓慢、均匀**。

## 放松身心，夯实基础

上一节我们介绍了呼吸深浅与生命状态的关系，本节中我们来看看如何调整身体以优化呼吸。“虚其心，实其腹”是重要原则，意指在生活中要让上身放松下来。

现代人常受压力、焦虑等不良情绪困扰。身体紧张时，肩膀会不自觉地耸起并紧绷。长期如此，呼吸就会变得非常浅。现在，请坐直身体，尝试放松。

放松，再放松，让整个肩膀自然下沉。你需要学会观察自己，很多人已养成肩膀长期紧张的模式。

“实其腹”是指腹部不能松松垮垮。许多练功者的腹部都非常紧实，这是因为通过训练，他们的腹部力量得到了增强。气息在体内运行，应如拉风箱般绵长不断，气血得以充分吐纳荡涤，这才是正确的呼吸方法。

## 吐纳的本质与生活化练习

吐纳就是练气、练呼吸，以此调节肺功能。许多人会选择在公园、山林等空气好的地方修炼。

但吐纳其实可以融入日常生活，并非必须在特定场所练习。行走坐卧，皆可练习。只要你想起，随时都可以进行吐纳呼吸。要明白，人的生命就在一呼一吸之间，长寿的密码也藏在呼吸里。

以下是不同生物的呼吸频率与寿命的对比，清晰地揭示了呼吸与寿命的关联：

*   乌龟每分钟呼吸2到5次，寿命可达300到500年。
*   人类每分钟呼吸16到18次，寿命约为80岁。
*   小狗每分钟呼吸约30次，寿命通常为10到20年。

由此可见：**呼吸越深长，寿命往往越长**。因此，应避免过快过急的呼吸。

听完分享后，你可以用钟表计算一下自己一分钟的呼吸频率。你会发现，呼吸急促的人，其身体反应往往也快。例如，有些男性时间较短，测量其呼吸通常也较快。呼吸快的男人，在床上的表现也可能较快。

反之，许多练习了固精功、房中术的人，能在同房时持续一两个小时，正是因为他们掌握了正确的呼吸模式。他们的呼吸深长，从而提升了持久力。所以，练习呼吸不仅关乎长寿，也能显著增强男性的战斗力。

当你调整了呼吸模式，整体状态会变得更好。

## 逆腹式呼吸：核心功法

在练习吐纳时，整个骨盆会随之摆动。骨盆前后摆动，本身也是对盆底肌和膈肌的一种对抗训练，能无形中拉长呼吸。

在此过程中，我们运用的是**逆腹式呼吸**。所有练功者都采用此法，因为“顺则成人，逆则成仙”。通过逆腹式呼吸，你能体会到呼吸被拉长的感觉，甚至感觉到一呼一吸之间，气息能直达会阴。

在后续的训练和带练中，老师会详细讲解。坚持练习一个月，你再对比之前的呼吸，会发现呼吸明显变长、变慢、变深。

呼吸变深后，强化的不仅是肺和肾的功能，而是整个脏腑系统。所有脏腑都需要氧气，当呼吸更深长时，供氧更充足，脏腑功能会变得更好。

## 吐纳：强肺之本

吐纳是真正锻炼肺功能的功法。许多人的肺泡并未被充分激活，导致肺功能逐渐下降。通过呼吸训练，增加每次呼吸的空气量，就是在锻炼你的肺。

若要养护肺，靠吃或喝什么效果有限。真正让肺功能变好，只有一个方法：**通过呼吸去训练**。

传统的胸式呼吸，一吸气肩膀就容易紧张。而普通的腹式呼吸，其作用和力量来得较慢。逆腹式呼吸则将肾脏与肺部功能贯通起来。

你会发现呼吸更深长，吸入的氧气量更大。练习一段时间后，很多人面色会发生改变，皮肤变得更白皙。这是因为身体经过充分补氧，脏腑功能改善，气血更通畅，肺功能也更强。

训练后你可以自行测评，会发现状态完全不同。特别是训练一段时间后，在与伴侣亲密时，你会感觉状态越来越好，不再那么容易疲劳。这是因为你懂得了深长呼吸，从而能更好地控精止漏。

我们的每一步功法都相辅相成，各有其重要作用。请务必跟随接下来的训练坚持练习。
