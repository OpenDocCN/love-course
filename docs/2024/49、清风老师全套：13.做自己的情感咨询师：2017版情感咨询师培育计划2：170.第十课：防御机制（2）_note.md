# 课程49：防御机制（二）—— 在咨询中的应用 🛡️

在本节课中，我们将深入学习防御机制在情感咨询中的具体应用。上一节我们介绍了防御机制的基础知识，本节我们将重点探讨如何识别、理解并处理咨询中遇到的防御机制，并学习相关的咨询技术。

## 概述：防御机制的核心概念

防御机制是精神分析理论中的一个重要概念。在人格结构中，它属于**自我（ego）**的功能。当自我觉察到来自本我的冲突时，就会以预期的方式体验到焦虑，并尝试用一定的策略去阻止它。这个过程就是防御，或称为自我的防御。防御主要是针对能引起个体焦虑的欲望或攻击性。一般来说，防御是在潜意识里进行的，因此个体并不会意识到它在发挥作用。

## 防御机制的五个特点总结

通过上一节的学习，我们对防御机制有了一定认识。以下是其五个核心特点的总结：

1.  **防御机制不是蓄意使用的**。它们是**无意识的或部分无意识的**。许多来访者的问题并非故意隐瞒，而是无意识的自我保护。例如，“狐狸吃不到葡萄说葡萄酸”就是一种合理化防御，个体通过这种想象来缓解痛苦。

2.  **防御机制是依靠自尊或自我美化来保护自我，免受伤害**。从性质上可以分为积极与消极两大类。积极的防御机制能带来好的体验，增强自信（例如，失败后告诉自己“下次会成功”）。消极的防御机制则可能引发焦虑（例如，预想下次也会失败）。

3.  **防御机制具有自我欺骗的性质**。它会掩盖或伪装自我真正的动机，甚至否认可能引起的焦虑、冲突或行为记忆。防御机制本身是一种阻断心理过程，使自我免受焦虑之苦的保护方法。因此，在分析来访者的防御机制时，很可能会遇到**阻抗**。来访者可能无法意识到或不愿承认自己的问题。理解这一点至关重要，我们不应轻易下定义，而应通过识别防御机制来更好地理解对方。

4.  **防御机制本身并非病态**，相反，它在维持正常心理健康方面起着重要作用。但是，它的**过度使用**可能会导致病理性的后果。防御机制可以单一使用，也可以重复使用。

5.  **防御机制通常是多种同时进行的**。例如，一对情侣吵架，原因是男友玩游戏输了，未完成本该做的工作。面对质问，男生可能将游戏失败的情绪**投射**到质问中，并对问题本身采取**回避**态度。防御机制总是站在对个体有利的角度进行的。

## 咨询中的应用：探索、面质与澄清

认识到防御机制的特点只是第一步。更重要的是将这种理解应用到咨询过程中。防御机制可能会阻碍咨询的顺利推进。那么，我们该如何做呢？

在咨询中，我们需要建立良好的咨询关系，甚至是更深入的联盟关系。这种关系需要咨询师有意识地去建立和维护。如果没有这种联盟，分析工作将无从下手。

以下是一个案例：
来访者安安抱怨，每当她想与男友深入沟通或谈心时，男友总是回避问题，让她感到抓狂和无助，甚至不敢继续聊天。

我问安安是否询问过对方回避的原因。安安表示问了，但对方不说，她就没办法了。我向安安解释，对方的回避和闪躲是一种**自我保护**的方式，可能因为他觉得问题令他为难，回避在某种程度上也是在保护感情。这类似于善意的谎言。要解决问题，需要在生活中观察原因，并通过平和的沟通来解决。

安安听后仍感困惑，询问具体做法。我告诉她，如果在观察中感觉到对方的回避和自卑，可以在交谈时间接询问：“亲爱的，你为什么不正面回答我的问题呢？是感到不舒服吗？如果你不相信自己能处理好，我们可以先放一放。”问完后观察对方的反应。

这就是今天要教给你的方法：**探索 + 面质 + 澄清**。

**探索**：作为咨询师，我们需要对防御机制有理解和认识，尝试找到一些防御来诠释对方的行为。可以通过长期观察（如伴侣间），或在咨询中通过提问来实现。我们也可以将此方法教给来访者，让其在有条件时与伴侣进行探索。

**面质**：即当面质询，把问题放到台面上与来访者进行深入、充分的探讨，帮助其认识到问题所在。在安安的例子中，就是把“对方因焦虑而回避关系”这个问题摊开来谈。

**澄清**：当我们没有理解清楚对方的意思时，可以当面说明，请对方做出解释，避免猜测意图。在亲密关系中，澄清极为重要。澄清的语句通常是：“我刚刚理解你的意思是……，你觉得是这样吗？”例如，安安可以问对方：“你不想面对我，是因为感到难堪或没有自信吗？是这样吗？”这个动作能帮助关系得到非常好的修复。

## 技术的延伸：如何进行诠释

在通过探索、面质、澄清找到主要问题后，我们需要对这些问题的防御进行解释和诠释。许多新手咨询师或普通人能看到对方问题的表现，但不知如何深入分析或入手。

为此，老师准备了诠释的几个方面，可以帮助你建立更好的咨询关系：

**1. 动力学诠释**
心理动力学由弗洛伊德创立，认为行为由强大的内部力量（本能或生物驱力）驱使，并试图解决个人与社会间的冲突。从动力学看，心理问题是个体发动、指引或维持某种活动的心理原因。弗洛伊德将本能（即**力比多**）视为基本心理动力。本能来自身体内部刺激，驱使人通过活动来满足需求，宣泄紧张和焦虑。
在咨询中，我们可以向来访者解释他们未曾看见的防御，并指明被真实情感遮掩的部分。例如，在安安男友的例子中，男友采用**回避**防御来避免冲突。对于回避型依恋者，回避常被用来保存亲密关系。我们可以向安安解释：男友并非讨厌她，而是在很大程度上希望保存两人的关系。这样的解释能帮助安安完成认知上的整合，并保存关系。
人的内驱力可分为两类：由饥饿等生理需要产生的**第一内驱力**（原始、低级）；由责任感等社会性需要产生的**第二内驱力**（社会、高级）。高级内驱力对低级内驱力起调节作用。每个行为背后都有内驱力，即行为动力，这可能是解决问题的真正关键。

**2. 起源性诠释**
很多当前行为可能是过往经历的投射。起源性诠释就是要找到问题产生的根本起源。例如，一位来访者因初恋被全心全意付出后却遭背叛抛弃，此后的恋爱中，只要男友稍有不好，她就担心会被抛弃。她因男友未陪逛街而吵架，并怀疑对方爱上别人。通过探讨她的初恋经历，我解释道：“你这样想，很大程度上源于初恋被抛弃的恐惧。那段经历的痛苦被你带到了现在的生活中，将当前行为与过去联系在了一起。”那段痛苦经历就是她问题的起源，也是其防御的真实面目。

**3. 移情诠释**
移情是精神分析用语，指将对父母或过去生活中某个重要人物的情感、态度或属性转移到别人身上，并做出相应反应的过程。例如，来访者小鱼总觉得男友不在乎自己，但又感觉对方很认真，因而怀疑自己多疑。探讨其童年经历发现，小鱼的父亲常常忽略她。在与男友相处中，小鱼将父亲忽略自己的情绪**转移**到了男友身上，认为男友也不在意她，这就是一种移情。
在咨询中发生移情时，咨询师就成了求助者某种情绪体验的替代对象。因此，当来访者频繁向你发出求救信息或将负面情绪转移给你时，不必过度担心，这说明他正试图与你建立一种移情关系。此时，咨询师可以说：“你认为我会和你父亲一样不注意你。如果在你说话时，我与你互动，让你重复重点，你会感觉怎么样？”了解移情的原因，能帮助我们与来访者进行更好的互动。
移情有直接和间接之分，也有正面和负面之分。出现负面情绪正说明咨询在起作用。

**4. 重现诠释**
重现是记忆的重要过程，指以前记过的事情，在当下其他事物的刺激下重新呈现出来。重现是我们对事物感受再次出现的感觉。诠释时需要思考，在早期的生命经历中可能发生过什么，造成了当前的困境。过往经历影响很大，而防御机制会让人忽略这种记忆，从而在生活中造成很大影响。
需要注意的是，重现某些记忆时可能不准确，来访者很可能在此刻产生其他防御机制，例如过分热心地附和咨询师的意见，违背内心真实想法以符合权威等。

## 情感咨询与心理咨询的界定

最后，老师要简要界定一下情感咨询与心理咨询的区别。
*   **情感咨询**中，咨询师扮演的角色更像一个**引路人**，是来访者黑暗中的火把。其作用往往在于**认知的更新**。我们通过解释、面质等技术，帮助来访者了解存在的问题，认识自身的特质。
*   **心理咨询**则更像一面**镜子**，让来访者在与咨询师的关系中，将其潜藏的动力呈现出来。在呈现过程中，通过咨询师的互动给予其新的体验，从而达成治愈。
