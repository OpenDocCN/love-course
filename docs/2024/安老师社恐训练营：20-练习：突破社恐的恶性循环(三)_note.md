# 安老师社恐训练营：20-练习：突破社恐的恶性循环(三)_note

在本节课中，我们将要学习如何通过改变行为模式来突破社交恐惧的恶性循环。我们将聚焦于一个核心练习：识别并改变你在焦虑场景中的习惯性行为。

改变行为模式的核心在于，对于引发紧张和焦虑的社交场景，我们需要有意识地采取与以往不同的行动。这需要我们主动打破旧有的回避模式。

## 练习步骤详解

以下是改变行为模式的具体练习步骤。请准备纸笔，在心中构建一个特定的、让你感到焦虑的社交场合，并将其详细写下来。

**第一步：辨别当前行为模式**
写下你在该场景中通常会做出的行为。例如：
*   回避眼神接触。
*   逃避参加集体活动或会议。
*   在对话中保持沉默。

**第二步：联系内在想法**
将上述行为与你当时的想法联系起来。例如：
*   想法：“人们对我缺乏兴趣。”
*   想法：“如果我和他们说话，他们会扭头走开，根本不会听。”

**第三步：决定做出改变**
基于以上分析，决定一个具体、可操作的新行为。目的是调动你的勇气和好奇心。例如：
*   改变行为：“我会主动和一位同事分享一件周末的趣事。”

**第四步：评估结果与反思**
在尝试新行为后，及时评估结果并进行反思。请保持对自己的包容态度，并检验最初的想法是否正确。例如：
*   结果：“我跟小刘说了我周末看的电影，他不仅听过，还很感兴趣。后来另一位同事也加入讨论，并询问我的看法。”
*   结论：“他们确实在听我说话，并没有将我排除在外。我最初‘没人会听’的想法可能并不准确。”

## 案例示范

上一节我们介绍了练习的四个步骤，本节中我们来看一个具体的应用案例。

假设小明的核心担忧是：“人们在深入了解我之后，就不会再喜欢我了。” 针对这个担忧，他可以进行如下练习：

1.  **焦虑场景**：周一上班，与同事闲聊周末生活时。
2.  **当前行为模式**：避免谈论任何关于自己的活动、经历或意见，通过保持安静来保护自己。
3.  **联系想法**：想法是“人们对我缺乏兴趣，如果开口说话，他们会扭头走掉”。
4.  **决定改变**：主动与一位同事分享周末看电影的趣事。
5.  **评估结果**：同事不仅回应，还表现出兴趣，甚至引发了更多人的积极讨论。
6.  **反思结论**：他们认真倾听并参与了对话，证明“没人会听”的想法是错误的。

这个练习可以记录在卡片或想法记录表上，形式不限，核心目标是**将逃避行为转变为积极面对**。
