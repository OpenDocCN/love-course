## 开宗筋、通八关之垂吊起式

### 1. 中金位置及揉搓
**中金**位于人体特定位置，是垂吊起式的重要穴位。在练习垂吊前，需对中金位置进行深层次的揉搓，以疏通气血。

**公式**：揉搓中金位置 → 疏通气血 → 提升垂吊承受力

### 2. 持股点揉搓
**持股点**位于中金上方，有许多荆棘。需由轻到重揉搓，以疏通气血。

**代码**：
```plaintext
轻揉持股点 → 感觉咯噔咯噔 → 疏通气血 → 缓解痛点
```

### 3. 拨筋与揉搓
**拨筋**需由轻到重进行，寻找金筋组织，并配合揉搓。

**代码**：
```plaintext
轻拨筋 → 感觉一股股组织 → 揉搓 → 提升气血循环
```

### 4. 拨筋与揉搓（续）
继续拨筋，揉搓时配合呼吸，将气血引导至痛点。

**代码**：
```plaintext
拨筋 → 揉搓 → 配合呼吸 → 缓解痛点
```
