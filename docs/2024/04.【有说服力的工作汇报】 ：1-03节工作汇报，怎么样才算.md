# 04.【有说服力的工作汇报】 ：1-03节工作汇报，怎么样才算

## 概述
在本节课中，我们将探讨如何判断一个工作汇报是否优秀。我们将分析汇报好坏由谁评判，并学习优秀工作汇报的三个核心特点。

上一节我们解决了“工作汇报到底在汇报什么”的问题，并介绍了工作汇报的五种重点内容。本节中，我们来看看另一个关键问题：工作汇报怎样才算汇报得好。

要解决这个问题，首先需要知道工作汇报的好坏由谁来判断。答案很简单，就是听汇报的人来判断。这个答案往往越简单越容易被忽略。很多人在准备工作汇报时，会忘记站在汇报对象的角度去思考问题。

如果你在各种会议上听过领导或汇报对象说过类似的话，例如“这情况我怎么知道？”、“你挑重点说吧”、“所以你需要我做什么？”，很可能就是因为汇报者忘了从听者的角度思考。如果在一次汇报中频繁听到这些话，那么这个汇报肯定算不上好。

## 好的工作汇报像一则促销广告 🎯
说了这么多，听汇报的人到底想听到什么样的汇报？怎么汇报才能减少这些问题？根据过去的经验，我发现好的工作汇报应该像一则促销广告。

为什么这么说？上节课我们提到，工作汇报的核心目的是**推动决策**。从这个角度看，工作汇报和促销广告非常相似。无论何种促销广告，最终目的只有一个：推动消费者做出购买决策。大白话说就是“买它”。

许多优秀的促销广告都站在消费者的角度思考。如果不这样做，就会出现有趣的情况：消费者会对广告提出和领导在会上提的一模一样的问题。例如：“这信息我怎么不知道？”、“有促销活动你早点说啊”、“这么一大堆，你到底哪个最优惠？”、“所以你想不想让我买啊？”

接下来，我们看看好的工作汇报应该是什么样的。概括下来有三个特点：**主动及时**、**简洁清晰**、**便于决策**。

## 特点一：主动及时 ⏰
我们先说第一个特点：主动及时。正常情况下，你不太愿意主动看促销广告。但如果你买东西时，商家没有主动及时地告诉你有促销活动，而是等你问了或事后才说，你心里反而会不舒服，因为担心错过了最低价。工作汇报也是类似的。

例如，我曾遇到一位同事（假设是小朱）。他刚到公司就接手了一个项目，工作特别忙，对流程也不熟，就一门心思埋头苦干，总想着等做完工作、把项目结果漂漂亮亮地再汇报给领导，中间就不耽误时间了。结果过了一段时间，领导觉得小朱这边一直没动静，就问了他项目情况。这一问才发现，小朱的工作方向彻底跑偏了，而且此时离项目结束已没多少时间，搞得整个团队陪他加班加点，项目最终还是延期了。

所以汇报一定要主动及时。如果等着领导问你工作情况你再汇报，多半情况下领导心里已经有点不舒服，甚至已经耽误事了。领导最怕的就是该了解的进度没了解，该发现的问题没发现，导致该有的决策和判断没有及时做出来。

那么，在工作的什么阶段汇报才算是主动及时呢？有以下五种一定要汇报的时刻。

以下是五种需要主动汇报的关键时刻：

1.  **工作准备开始时**：此时要向领导汇报你的工作准备情况，看看领导还有什么补充建议或执行中需注意的事项。例如，很多项目性工作在开始执行前会有一个启动会或通气会，同步项目准备情况和执行计划。
2.  **工作产生阶段性成果时**：此时汇报是为了让领导知道工作方向正确，一切在有条不紊地推进。如果有跨部门协同，阶段性成果也能及时让其他部门用上。例如，你正在执行一个市场推广方案，所有宣传渠道都沟通了一遍，这时就可以进行一次汇报，反馈沟通结果，看看接下来有什么需要重点跟进或团队配合的工作。不要企图攒个大成果一股脑汇报，因为在你看来是惊喜，在领导看来可能是惊吓。
3.  **工作遇到问题时**：这个问题无需过多解释。问题是否会导致工作延误、是否会产生连锁反应，都需要及时让领导知道，以便尽早应对和解决。
4.  **产生创意方案时**：有了好的想法，特别是合理的改进优化建议，及时向领导汇报，可能帮助整个团队提高工作效率。
5.  **工作长期持续进行时**：比如持续跟进某个平台的销售数据。这类工作可能你和领导都很熟悉，日常推进即可。但并非绝口不提，仍要定期汇报，让领导知道一切正常，因为“照常进行”本身也是一种决策。

## 特点二：简洁清晰 ✨
说完了主动及时，我们说一下好的工作汇报的第二个特点：简洁清晰。

很多人在工作汇报时，就怕自己说得不够全面、有遗漏，于是不管讲什么都扯一堆细枝末节。本来一句话就能说明白的事，非要说上三五句甚至三五十句。或者思维极具跳跃性，在好几个问题上反复横跳，让听汇报的人越听越头疼。

例如，我就听过这样一段汇报演示：“大家好，我想说一下我负责的那个项目的情况。前几天咱们和客户沟通的时候，客户说对咱们这个方案挺满意的。哎，那个方案不全是我做的，说起来还得感谢小李，他那边有个设计工具还挺好用的。原来咱们那个工具有点问题，我觉得这个工具应该更新一下。不过这个一会再说吧，还是说客户…那个客户…那个方案…方案…接下来该做宣传材料了…”这个时候，我相信在场所有人心里都在默念：能不能简单点、清晰点儿。

所以说，在做工作汇报时，要对内容进行整理和归纳。领导最关心什么内容？你最希望向他传递什么信息？就用最清晰的方式呈现出来，不然对方很容易对你的汇报失去耐心。

那么，具体怎么做汇报才能做到简洁清晰呢？我建议你做到以下四点。

以下是实现简洁清晰的四个技巧：

1.  **结果量化**：工作完成的情况，能用数字呈现的就用数字。说“这项工作完成了60%”，就比说“这项工作推进得还行”要清晰。这是让你的表达更清晰的第一种方式。
2.  **结论先行**：订单签了还是没签？问题解决了还是没解决？你认为A方案更好还是B方案更好？先把结论说出来。千万别绕半天圈子，说了好多“我们做了这个、我们做了那个”，最后才告诉领导“客户不同意”，那就浪费了领导的时间，最后还给个惊吓。
3.  **分条叙述**：一条一条地把信息进行分类，列出1、2、3，不要一大堆内容混在一起说。如果你说的时候是一团信息，那么听的人脑子里也是一团，他很难把你的一团信息分解成一条一条的。
4.  **重复重点**：很简单，重要的事情多说几遍。一个信息你重复了好几遍，领导自然就知道这个信息很重要了。

## 特点三：便于决策 🤔
最后，我们来说一下好的工作汇报的第三个特点：便于决策。

那么，怎么汇报才便于决策呢？很简单：让领导做**判断题**或**选择题**，而不是**问答题**。

我见过一个做市场营销的员工，他汇报工作时是这么说的：“领导，您上次让我调研了几个宣传平台，我调研完了，他们大致的情况是这样的……（此处省略一堆描述）。那么接下来您看一下咱们这个方案该怎么做呀？哦还有，咱们合作的一个平台不给咱们推广了，您觉得怎么办呢？”后来听说，这个营销方案确实做出来了，平台不给推广的问题也解决了，但这个员工也被老板“解决”了，因为老板实在不想再帮他回答问题了。

听懂了吗？所以在汇报工作时，最好是让领导做这样的决策：“行”或者“不行”；选“供应商A”还是选“供应商B”。别忘了，汇报的核心目的是**推动决策**。领导做决策越容易，他就会觉得你这个汇报做得越好。

## 总结
本节课我们一起学习了如何才算一个好的工作汇报。

好的工作汇报应该像一个促销广告，它越有利于你的听众（汇报对象）产生决策，就越是一个好的汇报。

好的汇报有三个特点，分别是：**主动及时**、**简洁清晰**和**便于决策**。

*   **主动及时**就是在五种时刻一定要汇报：工作开始时、有阶段性成果时、遇到问题时、产生创意方案时以及工作持续进行时。
*   **要做到简洁清晰**，有四个技巧：结果量化、结论先行、分条叙述以及重复重点。
*   **让领导便于决策**的方法，就是让领导做判断题或选择题，而不是问答题。

**本节课的小练习**：回顾一下你自己最近做的一个汇报，或者你观察到的一个汇报。然后基于我们课上讲的这些要点，想一想，如果给自己或你看到的这个汇报打分，是60分、80分、100分还是不及格？
