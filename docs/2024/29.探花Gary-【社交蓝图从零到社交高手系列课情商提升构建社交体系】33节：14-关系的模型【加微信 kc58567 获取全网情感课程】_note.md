# 社交蓝图从零到社交高手系列课：关系模型构建教程 🎯

## 概述
在本节课中，我们将学习如何构建社交关系模型，了解关系的六个阶段，并掌握如何在不同阶段提升自己的社交技巧。

## 关系模型概述
关系模型由六个阶段组成，分别是：认识阶段、了解阶段、吸引阶段、试好阶段、拉升阶段和稳定阶段。

### 认识阶段
**公式**： 认识 = 新环境 + 互动

在这个阶段，我们通过社交软件、工作、聚会等方式认识新的人。关键是要表现出足够的善意，如得体的穿搭、良好的形象和恰当的待人接物。

### 了解阶段
**公式**： 了解 = 深入交流 + 互动

在认识之后，需要一定时间来相互了解。这个阶段是关系能否进一步深入的前提和基础。了解是一个持续的过程，贯穿整个关系的构建。

### 吸引阶段
**公式**： 吸引 = 价值展示 + 信号释放

吸引是让别人关注你的关键。通过自身建设构建价值，并通过合理的方式展示出来，形成价值感。

### 试好阶段
**公式**： 试好 = 逐步认可 + 可得性赋予

在了解和吸引之后，开始对对方进行不同程度的认可，赋予对方可得性，并正确推进关系。

### 拉升阶段
**公式**： 拉升 = 张力提升 + 情绪调动

通过语言或行为提升张力，如塑造氛围和调动情绪，将关系从普通朋友提升到更深入的关系。

### 稳定阶段
**公式**： 稳定 = 长期关系维护

在稳定阶段，需要维持健康的长期关系。

## 总结
本节课中，我们学习了关系模型的六个阶段，并了解了如何在每个阶段提升自己的社交技巧。按照这个模型推进关系，可以确保你的推进健康、有条不紊地进行。🎉
