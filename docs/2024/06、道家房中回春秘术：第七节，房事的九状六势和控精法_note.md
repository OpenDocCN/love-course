# 道家房中回春秘术：第六课，房事的九状六势与控精法 🧘‍♂️

在本节课中，我们将要学习房中术的核心实践技巧，即阴茎的“九状”与性交的“六势”，并掌握一种在紧急时刻减少精液损失的控精方法。这些技巧旨在帮助修炼者更好地控制射精，实现“久交不泄”的养生目标。

## 九状详解：阴茎的九种运动状态 🌀

上一节我们介绍了固精的基础功法，本节中我们来看看在具体房事中，阴茎可以运用的九种不同运动状态，即“九状”。掌握这些状态，能让你在固精的前提下，给予伴侣更丰富的刺激。

以下是九状的具体描述：

1.  **左撞右撞**：像猛将一样左右冲击。这不仅是深浅刺激，也是对阴道壁左右两侧的摩擦。
2.  **上撞下撞**：由上至下撞击，如野马跃过河心。此法刺激强烈，需在“七窍乌龟法”运用纯熟后方可使用。
3.  **液压状**：快速进出，如波浪涌动。这是日常活塞运动，但也需在能熟练运用固精法时进行，以促进雄性激素分泌。
4.  **深注浅排**：深深进入后，仅浅出少许，再深深进入，不完全抽出。此法有助于在接近高潮时，通过短暂停留和固精法来避免射精。公式可理解为：`深度插入 -> 浅度退出（不完全）-> 深度插入`。
5.  **深冲浅刺**：深深进入后完全退出，再进行浅浅的刺入，然后再深深进入。如同大石投入大海，需根据伴侣反应灵活运用。
6.  **缓进状**：缓慢推进，如睡眠之蛇入洞。适用于前戏不足或伴侣阴道干涩时，也适用于双方临近高潮时，避免因猛烈运动导致失控。
7.  **速放状**：快速冲击数次，如受惊老鼠窜过洞穴。在伴侣第一次高潮后，自身精已固住时使用，以引发伴侣后续的快感。
8.  **正标状**：改变体位，龟头向下，如鹰击长空，自上而下深度刺激。用于在伴侣首次高潮后，为其带来新一轮强烈愉悦。
9.  **上顶状**：改变体位，自下而上顶撞，如狂风吹帆。与第八状方向相反，共同构成完整的深度刺激变化。

**核心前提**：九状的所有应用都必须建立在能够运用“七窍乌龟法”固住精、锁住精的基础上，目的是为了养生和让伴侣获得多重愉悦，而非追求射精。

## 六势详解：性交的六种姿势与技巧 🧘‍♀️

了解了阴茎的运动状态后，我们再来看看配合这些状态的六种基本性交姿势与技巧，即“六势”。它们与九状相辅相成，重点在于通过不同角度的摩擦达到最佳效果。

以下是六势的具体说明：

1.  **下摩式**：阴茎向下，龟头摩擦阴蒂。女性的阴蒂在兴奋时会如同男性的微型阴茎，刺激此处能迅速带来愉悦。
2.  **上提式**：阴茎由下向上提顶，如抛开石头取宝石。龟头沿阴蒂下方或阴蒂系带向上刺激，是第二状“上撞”的具体应用。
3.  **正捣式**：自上而下冲击，如捣药铁杆入臼。这是第八状“正标状”对应的姿势，需在能固精的前提下进行。
4.  **左右击式**：将阴茎抽出少许，进行左右方向的连续刮擦。如同铁匠打铁，确保对阴道壁进行全方位的按摩刺激。
5.  **斜磨式**：阴茎以斜角往来摩擦，而非直进直出。通过交叉斜向的运动，刺激不同的深度与位置，带来多样化的感受。
6.  **斜贯式**：从后方或下方进行斜向的贯通式刺入。这是角度更为特别的刺激方式，构成姿势的完整体系。

**核心要点**：六势与九状的核心目标一致，均需以“七窍乌龟法”固精为前提，通过对“天龙八步法”的熟练运用，达到自身不泄、伴侣满足的境界。

## 紧急控精法：会阴按压术 🚨

尽管我们强调固精为先，但初学者在兴奋或练习不纯熟时，仍可能面临失控边缘。本节将介绍一种亡羊补牢的紧急控精法，用于最大限度减少精液损失。

当感到射精欲望强烈，精液已迫近尿道，常规固精法难以挽回时，可采取以下步骤：

1.  **立即停止**：尽可能停止或大幅减缓抽插动作。
2.  **定位按压**：用右手（或常用手）的食指和中指，用力按压**会阴穴**。该穴位位于肛门与阴囊（阴茎根部）之间的凹陷处。
3.  **持续施压**：在整个射精过程中持续用力按压此部位，直至射精感完全结束。

**作用与效果**：此法通过物理压迫射精时收缩的关键肌肉（球海绵体肌），能减少约三分之一到三分之二的精液流出量，从而降低身体损耗。

**重要提醒**：此方法仅为应急补救措施，适用于酒后、久别重逢等极度兴奋而暂时失控的场景。修炼的根本仍在于熟练掌握并时刻不忘运用“七窍乌龟法”与“天龙八步法”进行主动固精，应尽量避免使用此急救法。
