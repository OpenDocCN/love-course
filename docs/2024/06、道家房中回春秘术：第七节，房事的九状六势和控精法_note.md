# 道家房中回春秘术：第六课，房事的九状六势与控精法 🧘‍♂️

在本节课中，我们将要学习房中术的两个核心实践部分：玉茎的“九状”与“六势”，以及一个在紧急时刻减少精液损失的控精方法。这些技巧旨在帮助修炼者更好地掌控自身，实现“久交不泄”的养生目标。

上一节我们介绍了固精锁精的基础方法，本节中我们来看看如何在实际房事中运用更丰富的技巧。

## 玉茎的九状 🪷

“九状”指的是男性玉茎在房事中可以采取的九种运动状态或方法。这些方法并非孤立使用，而是建立在熟练掌握“七窍乌龟固精法”与“天龙八步法”的基础之上，目的是在不泄身的前提下，带给伴侣更多愉悦。

以下是玉茎的九种状态：

1.  **左撞右撞**：像猛将一样左右冲击。这不仅涉及深浅刺激，还包括对阴户周边区域的摩擦。
2.  **上撞下撞**：由上至下撞击，如同野马跃过河心。此法刺激强烈，需在功法纯熟后方可使用。
3.  **一出一入**：像浸入波浪中有一种压力，即常规的活塞运动。此法也需在能熟练运用“七窍乌龟法”时使用，以获取更大摩擦刺激，促进激素分泌。
4.  **深注浅排**：深深进入后，仅浅浅退出一点，随即再次深入，不完全抽出。此法有助于在接近高潮时，通过短暂停留并运用“七窍乌龟法”来固精。
5.  **深冲浅刺**：深深进入后完全退出，再进行浅浅的刺入，接着再次深冲。如同大石投入大海，需根据伴侣反应灵活运用。
6.  **缓进徐入**：像睡眠的蛇入洞一样缓慢推进。适用于前戏不足或伴侣阴道干涩时，以及双方临近高潮时，避免因猛烈动作导致泄身。
7.  **速放疾刺**：像受惊的老鼠快速穿过洞穴。在伴侣第一次高潮后，自身能量已通过“搬精”固住时使用，以快速有力的刺入引发伴侣后续高潮。
8.  **正刺（鹰击）**：龟头向下，如老鹰俯冲抓兔，从上至下深度刺激。用于伴侣首次高潮后，为引发其再次高潮而采取的强烈刺激。
9.  **顶刺（帆扬）**：从下方向上顶刺，如狂风鼓帆。与第八状方向相反，需配合体位变化使用。

**核心前提**：所有“九状”的应用，都必须建立在通过“七窍乌龟法”能牢固锁住精关、实现“久交不泄”的基础上。练习时应循序渐进，由少到多，切忌急于求成。

## 房事的六势 🧘‍♀️

“六势”指的是房事中可以采用的六种姿势或角度，旨在通过不同方式的摩擦与刺激，全面激发双方快感，同时服务于固精养生的目的。

以下是六种姿势的要点：

1.  **下摩式**：玉茎向下，龟头摩擦女性阴蒂区域。女性的阴蒂在兴奋时类似男性玉茎，对此区域的刺激能迅速带来愉悦。
2.  **上提式（顶法）**：玉茎由下向上提顶，龟头沿阴蒂下方或阴蒂茎部运动。如同抛开石头取宝石。
3.  **正刺式（捣药法）**：玉茎从上方向下刺入，如捣药杆杵入药臼。需确保在能固精的基础上进行。
4.  **左右击式（打铁法）**：将玉茎抽出，从左右两侧连续攻击摩擦。如同铁匠打铁，确保对阴户360度全方位按摩刺激。
5.  **斜摩式**：玉茎以斜角往来摩擦，而非直进直出。通过左右交叉的斜向运动，刺激不同深度与位置。
6.  **斜贯式**：从后方或下方进行斜向贯穿刺入。这是另一种角度的斜向刺激。

**核心要点**：“六势”与“九状”有部分重叠，但侧重点不同。“六势”更强调体位与角度，其核心同样依赖于“七窍乌龟固精法”与“天龙八步法”的纯熟运用，目的是在不泄身的前提下，多次满足伴侣。

## 紧急控精法：会阴按压法 ⚠️

对于部分初学者或在极度兴奋、饮酒等特殊情况下，可能暂时无法用“七窍乌龟法”锁住精关，面临泄身风险。以下是一个“亡羊补牢”的应急方法，可在精液已至尿道、射精不可避免时，尽量减少损失。

**操作步骤**：
1.  在感到即将射精的瞬间，立即停止抽插运动。
2.  用右手（或惯用手）的**食指**与**中指**用力按压**会阴穴**。
3.  会阴穴位于肛门与阴茎根部之间的凹陷处（即骨盆底肌群区域，是控制射精的肌肉所在）。
4.  持续按压直至射精感完全结束。

**作用与注意**：
*   此法可减少约 **1/3 到 2/3** 的精液射出量，从而最大限度保全精气，减少身体损耗。
*   这仅是应急的补救措施，并非养生正道。核心仍在于平日刻苦练习“七窍乌龟法”，达到收发自如的境界。
*   适用于初学未稳、酒后或久别重逢等自控力较弱的特殊情况。
