# Ricky基础私教：03.吸引第三课：阴阳

## 概述
在本节课中，我们将学习如何通过阴阳状态来提升个人吸引力。

## 阴阳状态
### 阴
- **定义**：不受外界影响，情绪稳定，不轻易流露强烈情感。
- **公式**：\[ \text{阴} = \text{情绪稳定} + \text{不受外界影响} \]
- **例子**：面对领导的批评，感到难过；面对侄女的嘲笑，一笑置之。

### 阳
- **定义**：积极乐观，主动接触，影响他人，增加自身价值。
- **公式**：\[ \text{阳} = \text{积极乐观} + \text{主动接触} + \text{影响他人} \]
- **例子**：与女生约会，无论自己喝还是与朋友喝，都感到开心；面对迟到，保持乐观，并调侃对方。

## 阴阳在吸引中的作用
- **阴**：降低共振，提升吸引力。
- **阳**：增加价值，提升吸引力。

## 阴阳在人际交往中的应用
- **面对迟到**：保持情绪稳定，既不讨好也不抱怨。
- **面对拒绝**：无所谓，保持积极乐观。

## 总结
本节课中我们一起学习了如何通过阴阳状态来提升个人吸引力。通过掌握阴和阳，我们可以更好地影响他人，提升自身价值，从而吸引更多的人。
