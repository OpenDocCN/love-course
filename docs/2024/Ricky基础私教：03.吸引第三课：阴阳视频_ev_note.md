# Ricky基础私教：03.吸引第三课：阴阳 🌀

在本节课中，我们将要学习吸引力三部曲的最后一章：阴阳。我们将探讨如何在与人（尤其是与女性）相处时，通过调整自身状态来输出吸引力。同时，我们也会了解如何在需要时反向操作，以快速降低吸引力。

## 吸引力由状态决定

上一节我们介绍了吸引力的基础概念，本节中我们来看看吸引力的核心来源。你的吸引力并非由特定技巧决定，而是由你的**状态**决定的。

你的状态直接反映了你是智慧、勇敢、强大，还是懦弱、小气和无能。这种状态会通过你的言谈举止传递给他人，改变你对事物的理解、你的行为举止以及与他人互动的方式。有吸引力的行为会从这些态度中自然流露出来。

## 阴阳：吸引力的双面状态

为了描述这种有吸引力的状态，我们使用“阴”和“阳”这两个概念。它们共同构成了你人格的底色，使你在社交和两性关系中区别于普通人。

### 阴：不为所动的内核 🧘

阴，指的是你**不受外界影响**的状态。任何事情都不能动摇你的情绪，你对外界没有过多的情绪波动，不会流露出过于强烈的情感。

为了更好让你理解“阴”的这一面，我们来看一个例子：
*   你的领导批评你，你会感到难过，产生情绪共振。共振过多，你就是低价值的一方。
*   你5岁的小侄女说你笨，你只会哈哈一笑，因为它无法影响到你，你是更高价值的一方。

这完全取决于你的情绪反应。在与女性或他人交往时，**你不被影响，有着更低的情绪共振，这意味着你是更有吸引力的一方**，对方就会更喜欢你、尊重你。

### 阳：积极影响外界的光 🌞

阳，形容你是一个**积极正面、散发出正面情绪**的人。阳是你影响他人的方式，也是通过削弱目的性来增加自身价值的方式。

你在自己的世界里感到舒服、安全和开心。因此，你不会轻易被别人影响或改变。你很乐观、积极、幽默，无论和谁在一起。

以下是说明“阳”的例子：
你今天约了一个女生喝酒，你开心**不是因为她在**，而是因为无论你自己喝还是和谁喝，你都感到开心。你本身就是一个开心且乐观的人。如果她迟到，你也应该保持开心，并可以调侃她：“迟到要买单哦。”而不是被她影响，苦兮兮地坐在那里等她。

这两种感觉截然不同。这个例子可以说明阴和阳对你吸引力的共同影响。

## 阴阳在实践中的应用

理解了概念后，我们来看看如何在具体场景中应用阴阳状态。

### 场景一：应对对方迟到

现在你遇到一个女生迟到30分钟，普通人通常有两种错误做法：
1.  讨好型：“没关系，我也刚到，女生迟到很正常。”
2.  抱怨型：“你怎么迟到那么久？”

拥有阴阳状态的人会这样说：“怎么迟到那么久？迟到的人要买单哦。” **他迟到的行为不会影响我们的情绪，我们既不要讨好，也不要抱怨。**

以下是两种错误做法的本质：
*   **讨好**：会导致不被尊重，严重降低吸引力。因为只有携带强烈目的性的人，才会在不被尊重的情况下仍然讨好。这如同“没有交配权的公狗”。
*   **抱怨**：是非常无能的表现，是目的性强且解决不了问题时的愤怒。

如果你确实无法接受长时间等待，可以平静地表达：“多久能到？太晚的话我们要不改天？” 这体现了《教父》中的原则：**女人和小孩可以粗心大意，但男人不行**。在两性交往中，女人可以犯错，但男人必须保持稳定。

### 场景二：尝试接吻被拒绝

阴阳是你人格的一部分，它让你在任何行为中保持吸引力。例如，在你和女人接吻时，如果对方拒绝（无论是真拒绝还是矜持），普通人会有以下糟糕的反应：
1.  不知所措，愣住。
2.  害怕，不敢再做，反而去讨好对方。
3.  着急，质问对方“为什么不行？”

这些都是非常降低吸引力的行为。

假设你拥有阴和阳的心态，你应该表现得**无所谓**。你可以说：“我只是突然想接吻了，能不能接上，没有关系。” **公式：你的态度 = 无所谓 (阴) + 积极自我表达 (阳)**。

这种状态会让对方觉得你没有强烈的需求感和目的性。那么在下一次你想接吻时，对方可能比你更想要接吻。

## 总结与行动指南 🎯

本节课中，我们一起学习了吸引力构成的终极状态：阴阳。

*   **阳**：是你本身作为一个积极乐观的人，主动且乐观地去接触外界。
*   **阴**：是你保持内核稳定，对方给予的任何反馈（拒绝或接受）对你来说都**无所谓**。

你的吸引力就蕴含在这种状态里。在前期，你可能需要**伪装成已经做到这种状态**，并最终通过实践把自己变成那样。

就像我早期的经历一样：我先是扮演着这样的角色，最终我成为了这样的人。这使得女人很想和我在一起。很多导师曾问我，为什么女人见我一次就会产生强烈的喜欢？如何才能拥有这样的吸引力？

**答案就在这吸引力三部曲的三节课里。** 核心在于：**吸引力是一种可内部修炼、可外部呈现的状态**。通过理解和练习阴阳之道，你就能由内而外地掌控它。
