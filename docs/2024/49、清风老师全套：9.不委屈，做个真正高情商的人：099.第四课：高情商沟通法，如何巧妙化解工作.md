# 情商提升课 09：高情商沟通法，巧妙化解工作危机 🛠️

在本节课中，我们将学习如何运用高情商沟通技巧，巧妙化解工作中可能遇到的危机。我们将分析三种常见的工作困境，并学习如何通过激励自己和有效沟通来应对。

## 三种常见的工作困境

上一节我们介绍了如何避免成为委屈的人，本节中我们来看看工作中因缺乏情商而容易陷入的三种典型困境。

### 困境一：只做好分内事

第一种困境是认为“我只要做好自己的事情就可以了”。  
一位朋友倾诉苦恼：他在公司工作多年，渴望升职未果，于是决定只完成本职工作，绝不插手他人事务。结果，老板以他“工作不努力”为由将其辞退。

**核心问题**：员工与老板之间存在“心理契约”的认知差异。  
老板期望用1万元换取你12小时的工作价值，而你只愿意付出8小时。这中间的4小时差异形成了一种博弈。高情商的做法是，让老板满意，同时自己也不感到委屈。

### 困境二：过度热心帮助他人

第二种困境是“在职场里特别喜欢帮助别人”。  
刚入职场时，我热心帮助同事解决问题，却反被埋怨“不务正业”。后来发现，在帮助过程中，我不断指出对方的问题，导致对方压力巨大，最终反而指责我来回避问题。

### 困境三：付出未被看见

第三种困境是“我做得很好，但领导没看到我的成绩”。  
这与上一课提到的“默默付出却不被重视”类似，也需要用高情商的方式去化解。

---

## 运用激励能力完成工作

现在，我们来利用高情商的第三种核心能力——**激励**，来更好地完成工作。激励包含三个维度。

以下是激励的三个维度：

1.  **强大的成就驱动力**  
    找到内在驱动力，即做某件事的真正原因。这能让你在处理问题时自然进入高情商状态。  
    **方法**：问自己——“如果所有工作收入相同，我会选择做什么？为什么？”  
    例如，寿司之神小野二郎一生钻研寿司，因其**工匠精神**而获得认可。当工作本身带来快乐时，你自然能激发出高情商状态。

2.  **卓越的使命感**  
    使命感是驱动力的进阶。追问自己为何想做这份工作。  
    **方法**：使用**偶像法**——找出三位你最崇拜的人，以他们的定位和所做的事情作为参考，来探索自己的使命。

3.  **百折不挠的乐观精神**  
    乐观不是盲目，而是在挫折中看清自己却不放弃。例如，马云创业初期被27家机构拒绝，却对团队说“今天我又拒绝了一家投资机构”。这种在逆境中不否定自我价值的精神至关重要。

## 高情商沟通在工作场景中的应用

那么，如何将情商的激励能力应用于实际工作沟通呢？首先，我们需要了解社交的三种基本模式。

### 模式一：向上沟通（对领导）

与领导沟通时，展现高情商可分为四个步骤。

以下是四个具体步骤：

1.  **关注领导的需求**  
    理解领导真正想要什么。例如：“领导，您看这个方案是否符合您的要求？”
2.  **主动分享进度**  
    领导注意力有限，主动汇报工作进展（无论好坏），能有效吸引其关注。
3.  **提出问题与改善方案**  
    带着发现的问题和解决方案去请示领导。领导欣赏能发现问题并提出方案的人。
4.  **反馈真实情况**  
    切忌报喜不报忧。工作中问题是常态，只汇报好事会导致领导产生不合理的期待。

### 模式二：平行沟通（对同事）

与同级同事沟通时，应注意以下步骤。

以下是沟通的四个要点：

1.  **聆听同事的吐槽**  
    同事通常先吐槽而非直接求助。此时应专注聆听，而非一起抱怨。例如：“我理解你付出了很多却未被看见，这确实让人委屈。”
2.  **理解而非攻击**  
    避免附和攻击他人（如领导），而是表达理解并引导思考解决方案。
3.  **判断职责范围**  
    对于超出职责的请求，应礼貌澄清界限。例如：“我理解你的困难，但这不在我的职责范围内。或许你可以试试X方法与他人沟通。”
4.  **基于真实需求提供帮助**  
    像心理咨询一样，只在对方明确提出诉求时才提供帮助，而非强行介入。这有助于建立健康的同事关系。

### 模式三：向下沟通（对下属）

与下属沟通时，体现高情商的方法如下。

以下是四个指导步骤：

1.  **帮助下属认识自己**  
    询问他为何喜欢或选择这份工作，帮助他找到工作的价值感。即使是基础工作也有意义，例如，清洁工的工作是“让人感到轻松和愉悦”。
2.  **拆解任务**  
    帮助下属将大任务分解为可执行的小步骤。
3.  **设置错误底线**  
    明确告知什么样的错误可以犯，什么样的错误绝不能犯。
4.  **通过错误促其成长**  
    利用犯错的机会，帮助下属总结经验，获得成长。
