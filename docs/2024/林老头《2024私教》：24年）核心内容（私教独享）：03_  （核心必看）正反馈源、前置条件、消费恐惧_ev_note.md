# 林老头《2024私教》：24年核心内容（私教独享）03_ 正反馈源、前置条件、消费恐惧

## 概述

在本节课中，我们将学习林老头《2024私教》中的三个核心概念：正反馈源、前置条件和消费恐惧。这些概念不仅适用于情感领域，也适用于工作、事业和人生规划。

## 正反馈源

**核心概念**： 正反馈源是推动人们行动和取得结果的动力。

**公式**： 正反馈源 = 预期结果 + 阶段性正反馈

**解释**：

* 正反馈源是推动人们行动和取得结果的动力。
* 正反馈源包括预期结果和阶段性正反馈。
* 阶段性正反馈是指在每个阶段取得的成果，这些成果会进一步激发人们的动力。

**案例**：

* 一个兄弟在参加线下活动时，第一天就认识了一个漂亮的女生，并与之建立了良好的关系。这个经历为他提供了正反馈源，让他更有动力去追求自己的目标。

## 前置条件

**核心概念**： 前置条件是达成目标所必需的基础条件。

**公式**： 前置条件 = 基础知识 + 实践经验

**解释**：

* 前置条件是达成目标所必需的基础条件。
* 前置条件包括基础知识和实践经验。
* 基础知识是指对某个领域的了解和认知。
* 实践经验是指通过实践获得的经验和技能。

**案例**：

* 一个兄弟想要提升自己的聊天技巧，但他没有学习过相关的知识，也没有进行过实践。这种情况下，他很难取得进步。

## 消费恐惧

**核心概念**： 消费恐惧是指对消费的恐惧和抵触。

**公式**： 消费恐惧 = 配得感低 + 风险恐惧

**解释**：

* 消费恐惧是指对消费的恐惧和抵触。
* 消费恐惧包括配得感低和风险恐惧。
* 配得感低是指认为自己不配拥有某些东西。
* 风险恐惧是指害怕承担风险。

**案例**：

* 一个兄弟明明能够负担得起五星级酒店的消费，但他却觉得自己配不上这样的生活，因此选择不去消费。
