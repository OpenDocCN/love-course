# 钱进社 - 成为优质且最具竞争力的【社会人】：32、现实论下，不可小觑的12条弱者心态，你准备弱到什么时候

## 概述

在本节课中，我们将探讨现实论下不可忽视的12条弱者心态，帮助大家识别并克服这些心态，从而成为更具竞争力的人。

## 1. 推卸责任

**公式**：弱者心态 = 推卸责任 + 永远没问题

弱者心态的人在面对问题时，总是将责任推卸给他人，认为自己永远没问题。

## 2. 怯懦

**公式**：弱者心态 = 怯懦 + 无法独立操作

弱者心态的人在面对实际问题时，总是寻求他人的陪伴，无法独立操作。

## 3. 分不清是非，不讲道理

**公式**：弱者心态 = 只讲情理 + 忽略本质

弱者心态的人在面对问题时，只关注情感，忽略问题的本质。

## 4. 在乎面子和虚无缥缈的感受

**公式**：弱者心态 = 在乎面子 + 虚无缥缈的感受

弱者心态的人过于在乎他人的评价，追求虚荣。

## 5. 不敢直面痛点和现实

**公式**：弱者心态 = 逃避现实 + 认不清现实

弱者心态的人面对问题时，总是逃避现实，无法认清现实。

## 6. 侥幸心理

**公式**：弱者心态 = 侥幸心理 + 习惯性侥幸

弱者心态的人习惯于侥幸心理，无法认真对待工作。

## 7. 作

**公式**：弱者心态 = 作 + 终身没戏

弱者心态的人过于作，无法获得成功。

## 8. 不会主动思考

**公式**：弱者心态 = 不会主动思考 + 缺乏思考能力

弱者心态的人缺乏主动思考的能力。

## 9. 注意力在云端

**公式**：弱者心态 = 注意力在云端 + 无法脚踏实地

弱者心态的人总是关注心理层面，无法脚踏实地。

## 10. 不要轻易下结论，而是反向思考

**公式**：弱者心态 = 轻易下结论 + 缺乏思考

弱者心态的人总是轻易下结论，缺乏思考。

## 11. 挑别人毛病比看别人优点多

**公式**：弱者心态 = 挑别人毛病 + 缺乏包容心

弱者心态的人总是挑剔他人，缺乏包容心。

## 12. 计较蝇头小利

**公式**：弱者心态 = 计较蝇头小利 + 缺乏大局观

弱者心态的人过于计较小利，缺乏大局观。

## 总结

本节课中，我们学习了现实论下不可忽视的12条弱者心态。希望大家能够识别并克服这些心态，从而成为更具竞争力的人。
