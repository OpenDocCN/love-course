在本节课中，我们将学习如何通过控制盆底肌，让负重的能量蛋实现前后摆动。这是一个检验前期练习成果、提升身体感知与控制力的重要训练。

上一节我们介绍了基础的热身与呼吸练习，本节中我们来看看如何具体操作负重摆动。

## 准备姿势 🧍‍♀️

双脚打开，与肩同宽。请注意，双脚间距不宜过宽。对于初学者，建议将幅度收小一些，以便更好地控制。

## 负重选择 ⚖️

前期尝试时，负重物体不需要太重。以下是几种选择：
*   可以挂一个小球。
*   也可以尝试两个或三个小球。

## 核心动作：摆动控制 🔄

本节的核心是学习用盆底肌控制负重物体前后摆动。关键在于用意识引导肌肉收缩。

**动作要领如下：**
1.  意念集中于盆底肌。想象连接肛门与尾骨区域的盆底肌。
2.  尝试让这部分肌肉向前、向上收紧，感觉像向内卷起。这会使能量蛋轻微上提。
3.  当蛋被提起时，它会自然产生前后摆动的趋势。
4.  顺应这个节奏，配合盆底肌的“收”与“放”，控制摆动。

**可以用代码来理解这个收缩与放松的循环：**
```python
while 进行训练:
    盆底肌.收紧()  # 意念引导，向内向上卷起
    蛋.轻微上提并摆动()
    盆底肌.放松()  # 有控制地放松
```

## 辅助练习方法 💡

如果一开始找不到控制摆动的感觉，可以尝试以下方法：

以下是两个有效的辅助练习步骤：
1.  **弯腰练习**：将头和腰弯下来，在这个姿势下收缩盆底肌，可能更容易找到发力感。
2.  **先动后控**：先用手或身体轻微晃动能量蛋，让它前后摆动起来。然后，尝试用盆底肌的收缩去契合和引导这个已经存在的摆动节奏。

**关键诀窍**：先让蛋摆动起来，再用盆底肌去跟随和控制这个摆动。经常练习以找到前后摆动的感觉。

本节课中我们一起学习了负重前后摆动的技巧。这个练习非常考验盆底肌的基础功底。核心在于通过意念精确控制盆底肌（尤其是肛门尾骨区域）进行向内向上的卷缩，从而带动能量蛋摆动。对于初学者，可以从轻负重开始，并利用“先晃动、再控制”的辅助方法来寻找感觉。坚持练习是掌握此技巧的关键。
