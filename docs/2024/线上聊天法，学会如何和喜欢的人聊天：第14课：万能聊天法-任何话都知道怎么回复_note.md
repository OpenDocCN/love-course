# 线上聊天法，学会如何和喜欢的人聊天：第14课：万能聊天法-任何话都知道怎么回复

🎼大家好，欢迎来到第十4集。本节课我们将教授一个万能聊天法，适用于任何场合，帮助你在难以回应时找到合适的回复。特别适合在争吵等需要冷静的时刻使用。

**万能聊天法步骤**：

1. **确定关系阶段**：了解你和对方目前处于何种关系阶段（破冰、了解、熟悉或舒心），不同阶段回复策略不同。
2. **识别情绪**：判断对方情绪是正向（高兴、期待）还是负向（怀疑、不安全）。
3. **解读意图**：分析对方话语意图，是测试你、期待你做某事，还是其他。
4. **斟酌回复**：根据前三步分析，运用话题转移、追加信息、调侃共情等战术进行回复。

**示例**：

- **了解阶段**：女生问你“你喜欢什么样的女生”，这是正向情绪，她期待你做出令她满意的回答。你可以简单表达个人想法，如“我喜欢贤惠的”。
- **熟悉阶段**：女生同样问你“你喜欢什么样的女生”，此时情绪为负向，她可能想测试你是否真的爱她。你可以哄她，给她安全感，说一些甜言蜜语。

通过万能聊天法，你可以轻松应对各种棘手的回答。本节课我们就到这里。
