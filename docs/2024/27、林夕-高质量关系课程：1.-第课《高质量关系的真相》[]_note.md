# 27、林夕-高质量关系课程：1.-第课《高质量关系的真相》[]

大家好，我是林夕。今天是我们高质量关系课程的第三节课，主题是《高质量关系的真相》。这节课旨在探讨一段感情为何开始、为何出现问题以及为何分手。

**以下是课程内容：**

### 1. 感情关系的本质

我们普通人往往只看到感情关系的表面，而真正的情感专家则会深入分析价值、需求和本质。双方因价值或利益冲突导致关系变化。

**公式**：关系变化 = 价值/利益冲突

### 2. 梳理感情关系

首先，梳理你们感情的时间线，包括认识时间、在一起时间、发生性关系时间、确认恋爱关系时间、发生大矛盾时间、见家长时间或分手时间。

**代码**：时间线 = [认识时间, 在一起时间, 发生性关系时间, ...]

### 3. 找到关系转折点

根据时间线，回想当时双方处境，找到关系转折点。转折点即双方价值发生变化的时间点。

**公式**：转折点 = 价值变化

### 4. 格局失衡

格局是指两个人在情感中的位置，由自身价值和需求感共同决定。关系开始时，双方格局平衡，但随着时间推移，格局可能发生变化。

**公式**：格局 = 自身价值 + 需求感

### 5. 自身价值

自身价值分为硬价值和软价值。硬价值包括外形条件、收入、家庭、学历、户口、国籍等；软价值包括情绪价值、情商、性能力、性格等。

**公式**：自身价值 = 硬价值 + 软价值

### 6. 择偶范围

自身价值决定了择偶范围。提升自身价值，自然能吸引更优质的伴侣。

**公式**：择偶范围 = 自身价值

### 7. 感情走势中的格局变化

感情走势中的格局变化体现在冲突事件中。思考以下三个关键点：

1. 为什么在一起？
2. 为什么出问题？
3. 为什么分手？

### 8. 常见的价值变化

女性和男性在恋爱和婚姻中常见的一些价值变化，如身材走样、整容整形、黄金择偶期变化、事业变化、解决户口或国籍问题、生完孩子、出轨等。

### 9. 择偶需求的变化

择偶需求主要包括情感、身体和利益三个方面。

**公式**：择偶需求 = 情感 + 身体 + 利益

### 10. 复盘感情关系

梳理感情关系的时间线，分析问题所在，找到解决方法。

**总结**：

本节课我们学习了高质量关系的真相，包括感情关系的本质、梳理感情关系、找到关系转折点、格局失衡、自身价值、择偶范围、感情走势中的格局变化、常见价值变化、择偶需求的变化以及复盘感情关系。希望大家通过学习，能够更好地理解感情关系，提升自身价值，找到合适的伴侣。
