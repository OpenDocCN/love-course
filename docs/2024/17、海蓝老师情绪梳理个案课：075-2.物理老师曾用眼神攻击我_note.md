# 海蓝老师情绪梳理个案课：075-2.物理老师曾用眼神攻击我

在本节课中，我们将学习如何通过情绪梳理个案课来处理过去的负面情绪体验。

## 情绪梳理个案课概述

情绪梳理个案课是一种心理治疗方法，旨在帮助个体识别、理解和处理负面情绪。通过以下步骤，我们可以深入了解个案的情绪体验：

1. **识别情绪**：识别个案所经历的情绪，如恐惧、愤怒、悲伤等。
2. **追溯记忆**：追溯导致这些情绪的具体事件或经历。
3. **理解情绪**：理解情绪背后的原因和意义。
4. **处理情绪**：学习如何有效地处理和释放这些情绪。

## 案例分析

以下是案例中个案的情绪梳理过程：

### 1. 识别情绪

个案描述了在初中时期，一位物理老师用眼神攻击她的经历。她感受到了恐惧、愤怒和恶心。

**公式**：恐惧 + 愤怒 + 恶心 = 负面情绪体验

### 2. 追溯记忆

个案回忆起当时的情况：她因户口问题在学校没有学籍，物理老师对她有偏见，并在全班面前用眼神攻击她。

**代码**：户口问题 + 偏见 + 眼神攻击 = 负面记忆

### 3. 理解情绪

个案的情绪源于被误解、被歧视和被攻击。她感到恐惧是因为害怕再次受到伤害，愤怒是因为感到被不公正对待，恶心是因为感到羞耻和痛苦。

**公式**：误解 + 歧视 + 攻击 = 恐惧 + 愤怒 + 恶心

### 4. 处理情绪

个案在情绪梳理过程中，表达了对老师的愤怒和不满，并想象了报复的场景。随后，她意识到自己不应该将负面情绪带入现在的生活，并学会了放下过去。

**代码**：表达愤怒 + 想象报复 + 放下过去 = 情绪处理

## 总结

本节课中，我们通过情绪梳理个案课，了解了如何识别、追溯、理解和处理负面情绪。通过案例分析，我们看到了个案如何从负面情绪中走出来，重拾自信。希望这节课能帮助大家更好地面对生活中的挑战。
