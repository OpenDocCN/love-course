# 老白《约会达人》：13、领袖小游戏--金字塔

## 概述
在本节课中，我们将学习一种多人饮酒游戏——金字塔。

## 游戏规则

以下是金字塔游戏的详细规则：

### 1. 准备阶段
- 参与人数：4人、5人、6人或更多。
- 准备一副扑克牌，去掉大小王。

### 2. 堆积金字塔
- 将扑克牌堆叠成金字塔形状，每层比下一层多一张牌。

### 3. 发牌
- 将剩余的牌平均分给每位玩家。

### 4. 游戏进行
- 按顺序从金字塔顶部开始，每位玩家抽取一张牌。
- 抽到的牌代表饮酒的口数，具体如下：
  - 1口：最底层
  - 2口：下一层
  - 3口：再下一层
  - 4口：再下一层
  - 5口：再下一层
  - 6口：最顶层

### 5. 喝酒与翻牌
- 抽到牌后，根据牌面数字饮酒。
- 如果玩家手中有一张“圈”牌，可以指定其他玩家饮酒，并翻倍。
- 如果被指定的玩家手中也有“圈”牌，可以反指定，此时指定玩家需饮酒翻倍。
- 如果玩家手中没有“圈”牌，则需饮酒。
- 如果玩家选择不饮酒，可以出一张牌，指定其他玩家饮酒。

### 6. 游戏结束
- 当所有牌被抽完时，游戏结束。
- 计算每位玩家剩余的牌数，每张牌代表一口酒。

## 总结
本节课中，我们一起学习了如何玩金字塔游戏。这种游戏适合多人聚会，能够增加游戏的趣味性和互动性。希望你在聚会中尝试这个游戏，享受欢乐时光。
