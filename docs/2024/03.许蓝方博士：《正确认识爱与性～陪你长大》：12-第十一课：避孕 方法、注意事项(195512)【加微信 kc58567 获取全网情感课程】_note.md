# 课程03：正确认识爱与性 - 第十一课：避孕方法与注意事项 🛡️

在本节课中，我们将要学习避孕的重要性、各种避孕方法的原理与操作，以及如何根据个人情况选择合适的方式。正确避孕是保护身心健康、规划未来的关键一步。

## 概述

长大成人是一个循序渐进的过程。认识并了解自身变化，才能成为更好的自己。在青少年发育阶段，对性产生好奇后，就必须开始注意避孕。避孕的重要性不仅在于防止意外怀孕，更在于避免其带来的身心伤害。

## 避孕为何如此重要？

与其强调避孕本身重要，不如说忽视避孕的后遗症非常伤身。不避孕会伤害身体与心理，其损害并非一时，可能残留脑海与身体一辈子，造成阴影与后遗症。

### 堕胎的身心影响

假设在不适合的时机怀孕，很可能走上堕胎之路。堕胎会产生以下影响：

**生理影响**：
现代医疗技术进步，流产手术（刮除子宫内膜）伤害子宫的几率已减小。但无论药物或手术堕胎，仍可能并发子宫颈损伤、阴道出血或子宫穿孔等情况。尤其堕胎超过两次的女性，术后感染风险增加，更可能伤害骨盆腔导致发炎，影响未来生殖能力。在不适合的年纪多次堕胎，未来到30、40岁想怀孕时，受孕可能性会降低。

世界卫生组织研究指出，女性若进行两次以上人工流产，未来较易生出早产儿或体重过轻的胎儿。因为子宫已习惯被刮除，胎儿较易早产。此外，堕胎次数过多易导致日后惯性流产，或衍生其他妇科疾病，甚至造成不孕。

**心理影响**：
对女性而言，堕胎常是重大创伤。这种心理创伤与悲伤类似亲人离世，毕竟体内曾有一个生命。尤其被迫堕胎或因胎儿不健全需终止妊娠时，女性易受良心谴责，怀疑是否因自身原因失去孩子，严重时需心理医生介入协助。

精子与卵子一旦受孕成为受精卵，就是一个生命。堕胎意味着一个生命的逝去。这对女性身心伤害难以言喻。当然，以上所指是明知应避孕却未避孕而怀孕的堕胎。若女性在不情愿情况下怀孕而必须堕胎，则另当别论。

## 常见避孕误区

避孕并非只是男性的责任，女性也需懂得保护自己。切勿相信男性“戴套没感觉”、“不戴没关系”、“不射在里面就好”等说法。也勿认为“现在不是排卵期/安全期，不戴套没关系”。

计算排卵期与估算安全期是一段期间，代表你无法精确知道何时排卵，任何时候都有可能怀孕。无论是否戴套、月经是否来临，都有可能怀孕。这是最不安全且危险的方式。

## 避孕方法详解

以下是几种主要避孕方式及其原理、使用方法与注意事项。

### 1. 口服避孕药 💊

口服避孕药需每日服用，让身体摄取定量女性荷尔蒙（含雌激素与黄体素）。主要作用是抑制排卵，防止卵子释放；干扰子宫内膜生长；并增加子宫颈粘液稠度，使精子不易进入。

**服用方法**：
从月经来潮第五天开始吃第一颗（有的医生建议从月经第一天开始）。连续服用三周后，第四周为月经期，月经结束后的第七天继续服用下一盒。

**副作用与注意事项**：
可能副作用包括体重增加、水肿等，但随着医学进步，副作用已降低。务必每日定时服用。若忘记服用：
- 8小时内想起：立即补吃。
- 超过12小时忘记：次日固定时间吃两颗（例如固定早8点服药，次日早8点吃两颗）。

**常见误解**：
口服避孕药并非有害。现代第四代避孕药副作用更小。国外女性使用常规避孕药避孕比例高达一半，亚洲比例仍低。口服避孕药好处多于坏处，能改善青春痘、预防某些妇科疾病、调节月经。长期使用5年可降低25%子宫内膜癌风险。

有人担心服用会导致不孕。每日服用的避孕药不会导致不孕，停药后反而更容易受孕（因月经排卵更规则）。短期服用者，停药一年内受孕几率与未服药女性相同。但仍有轻微副作用可能，如短暂少量阴道出血或性交后出血，只要出血量不大即无妨。

**禁忌人群**：
35岁以上、吸烟、有心血管疾病、脑血管疾病或血栓疾病者，不建议使用口服避孕药。最安全方式是咨询妇产科医生是否适合使用。

### 2. 避孕环 🌀

阴道避孕环是一个柔软有弹性、含药物的透明环，会定时定量释放低剂量荷尔蒙，原理与口服避孕药相同。荷尔蒙经阴道上皮内膜吸收。

**使用方法**：
女性在月经来潮五天内，自行将环放入阴道深处（近子宫颈处）。放置三周后取出，取出的那一周为月经期。月经第五天再次放入，循环使用。

**优点**：
一次放置可维持三周，减少经血过多，让月经规律、减少经痛。可自行取出，无需医生操作。

**缺点与副作用**：
若不小心掉出，可能造成怀孕。副作用包括头痛、恶心呕吐、乳房胀痛、体重增加、浮肿等，停用后副作用会消失。

### 3. 子宫内避孕器 🏥

避孕原理是引起子宫内膜变化，干扰受精卵着床。

**原理说明**：
若有排卵，身体感应到可能有精子与宝宝出现，子宫会让子宫内膜增厚，“铺床”准备迎接受精卵。子宫内避孕器就是让子宫内膜产生变化，“不铺床”，使受精卵无法着床。避孕效果高达95%。

**适用人群**：
较适合已生产过的女性。未生产过的女性不建议使用，可能因装置导致出血、经期改变、经血量变化、下腹酸痛等不适。

**使用方法**：
需医生在月经结束后放入，或在性交后五天内作为紧急避孕放入。不可自行执行，否则易感染甚至造成死亡。

**优点**：
可长期使用，取出后仍可怀孕。

### 4. 结扎 ✂️

**女性结扎**：
将两条输卵管结扎切断，使卵子无法通过输卵管，避孕效果100%。但结扎后若想再接通，需再次手术，非常麻烦。未生育过的女性非常不建议结扎。若已生育且确定不再生育，结扎是好选择。

**男性结扎**：
相比女性结扎，男性结扎手术更安全、省时、快速。若真要结扎，男性结扎是更好选择。

### 5. 子宫内投药系统 💉

长期缓慢释放黄体素，比一般避孕器效果更高，且有减少经血与经痛的好处。未来若有怀孕计划，可请医生取出。统计显示，80%以上女性在取出一年内可顺利怀孕。但需专业医生操作。少数人可能出现乳房胀痛、头痛、青春痘等副作用。

### 6. 事后避孕药 ⚠️

性交后服用一颗即可避孕。但个人非常不推荐，这不是正规避孕方式。事后避孕药含高浓度荷尔蒙，只能干扰卵子与精子结合，且不代表它们不会结合。高浓度荷尔蒙会干扰经期，长期服用可能影响内分泌系统导致不育。

## 正确避孕实践

若已在法定年龄开始性行为，请在彼此尊重意愿下进行。性行为前务必采取避孕措施。

### 双重避孕策略

最推荐的方式是双重避孕：男性戴避孕套，女性服用口服避孕药。有人问，女性服药后男性是否可不戴套？仅就避孕而言可能可行，但避孕套还有另一重要功能：防止性传染病。只有避孕套能做到这点。因此，安全性行为需两件事都做：男性戴套，女性服药。

### 如何选择避孕套？

选择合适避孕套至关重要。不合适（太紧或太松）会不舒服，导致不愿使用。合适则可以从第一次使用持续到计划怀孕时。

以下是选择要点：

**1. 制造日期与有效期**：
检查有效期，勿使用过期产品。过期会破坏材质，影响避孕效果。

**2. 成分**：
分乳胶与非乳胶（聚氨酯）。聚氨酯套通常超薄、无味；乳胶套弹性较好、稍厚、有味道。若对乳胶过敏，选择聚氨酯套即可。勿以过敏为理由不戴套。

**3. 阔度（宽度）**：
购买适合阴茎大小的阔度。测量方法：阴茎勃起时，用软尺测量圆周，除以2即为阔度（单位：毫米）。例如，勃起圆周10公分（100毫米），阔度为50毫米。若不确测量，可购买标有“52±2毫米”范围的产品。

**4. 保存方式**：
勿将避孕套长期放在钱包并置于口袋（贴近身体），体温会破坏材质。应存放在阴凉干燥处。

### 口服避孕药服用要点

务必每日定时服用。常见有两种包装：
- 21颗装：连续服用21天后，停药7天（月经期），第8天开始下一盒。
- 28颗装：含21颗有效药片与7颗无药性片（仅为帮助记忆，防止忘记）。每日一颗，吃完最后一颗直接开始下一盒。

## 根据个人情况选择避孕方式

虽然避孕方式多样，但需根据身体状况与当下情况选择。平衡条件，选择最适合自己的方式，并弄清原理与目的，避免错误方式影响未来生育能力。

### 年龄与健康状况的影响

不同年龄与健康程度的人需选择不同方式：
- 35岁以上不吸烟女性：戴避孕套与服用口服避孕药是安全方法。
- 35岁以上吸烟女性：避孕药可能增加心脏病与中风风险，不建议使用。
- 有偏头痛、血栓、心血管或脑中风问题者：强烈建议经医生评估后再使用口服避孕药。

### 男性避孕方式

男性目前最有效的避孕方法是结扎与避孕套。所谓“中途拔出”、“不射在里面”等方式均不正确。避孕套除避孕外，还能避免性传染病。选择品质好、合适阔度的避孕套，不会让男性不舒服。男性结扎相比女性更简单易行，通常当天即可回家，无需住院。
