# 18、坏小k女人一生必学的一堂课：视频课：34.感情36计（14）如何建立真正自信，普通人可实操的方法

🎼 感情三十六计之第十四大技巧，如何在社交上真正建立自信，普通人可实操的内核修炼方法。

### 概述
在本节课中，我们将学习如何建立真正的自信，并探讨普通人可实操的方法。

### 身份与自信的关系
**公式**：身份 → 行为 → 自信

你的身份限制了你的言行，等于你对自己的潜意识限制了你的言行。例如，现在的你们，我让你去大街上搭讪你心仪的男孩子。

### 身份限制的表现
以下是身份限制的表现：

1. **他人给你的回应**：比如你试着成为一个有魅力的女人，某个人却突然嘲笑你，于是你意识到自己好像并不是很有魅力。你的内心就会极具发生变化。开始根据他人的回应来调整自己的行为。
2. **环境给你的回应**：比如一个女神，原来在学校里备受欢迎，趾高气扬，当她考入北影这样的学校时，到处都是漂亮女神，她的内心就会极剧发生变化。开始根据环境来调整自己的行为。

### 伪自信与真自信
**公式**：伪自信 → 环境变化 → 自信崩塌

伪自信，如果你的自信是跟随环境，他人给你的回应，皆称为伪自信。如果你是靠伪自信吸引他人，那么你随时会崩塌。

### 真正自信的来源
**公式**：自信 → 自我认同 → 行为

真正自信，来源于你对自己身份坚定信念，不跟随任何环境，语言发生变化，你就是你世界上只有一个你，你是唯一经历了你自己的人生，并且有权改变决定你的将来的人。

### 总结
本节课中我们一起学习了如何建立真正的自信，并探讨了普通人可实操的方法。希望这些内容能帮助你在社交中更加自信地展现自己。
