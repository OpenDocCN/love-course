## 搞定客户性格，提升销售业绩

### 红色性格客户

**特点**：爱说话，以体验为导向，容易被创意广告或个人喜好吸引。

**应对方法**：

*   **赞美**：多赞美客户，给予特权和小礼物。
*   **描绘美好画面**：通过语言描述产品使用后的美好画面。

**公式**：赞美 + 描绘美好画面 = 提高推销成功率

### 蓝色性格客户

**特点**：内敛，注重细节，完美主义。

**应对方法**：

*   **有条理介绍产品**：客观介绍产品细节，避免猜测客户隐私。
*   **等待客户开口**：耐心等待客户参与对话，销售成功一半。

**公式**：有条理介绍 + 等待客户开口 = 销售成功

### 黄色性格客户

**特点**：主见强，追求有用，利益大于损失。

**应对方法**：

*   **分析利害关系**：分析项目或产品带来的好处和坏处。
*   **避免催促**：让客户自己内心理解，改变决策。

**公式**：分析利害关系 + 避免催促 = 客户同意方案

### 绿色性格客户

**特点**：随和，易相处，缺乏主见。

**应对方法**：

*   **主动出击**：主动与客户交流，引导客户购买。
*   **关注家人**：告诉客户产品对家人有益，提高购买意愿。

**公式**：主动出击 + 关注家人 = 客户购买
