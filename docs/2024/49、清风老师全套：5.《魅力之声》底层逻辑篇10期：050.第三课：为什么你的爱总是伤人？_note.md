# 清风老师全套：5.《魅力之声》底层逻辑篇10期：050.第三课：为什么你的爱总是伤人？

## 概述
在本节课中，我们将探讨为什么在磨合期中，爱情往往会伤害到彼此。我们将分析磨合期中的问题、矛盾以及恶性循环，并探讨其内在和外在原因。

## 磨合期的特点
以下是磨合期的三大特点：

1. **问题重复出现**：问题似乎总是得不到解决，例如，夫妻一方沉迷手机，另一方感到愤怒和痛苦。
2. **深层的委屈和愤怒**：误解和委屈不是一天两天形成的，而是多年积累的结果。
3. **冲突升级**：夫妻双方互相指责，任何小事都可能引发争吵。

## 磨合期的四种方式
以下是磨合期中最常见的四种方式：

1. **互相指责**：一方指责另一方，认为对方不好。
2. **互相伤害**：一方伤害另一方，认为对方伤害了自己。
3. **不停道歉**：一方不断道歉，但问题并未得到解决。
4. **无力解决**：双方无法解决问题，陷入恶性循环。

## 磨合期伤人的爱的原因
磨合期中伤人的爱分为内在和外在原因：

### 内在原因
1. **关系中的表演逐渐失去**：随着时间的推移，人们不再刻意表现自己，而是展现出真实的需求和期待。
2. **双方开始寻找关系中的位置**：在关系中，一方可能开始控制另一方，导致矛盾和冲突。
3. **通年伤痛的重复**：人们在关系中重复童年的伤痛，例如，指责和谩骂。

### 外在原因
1. **下沉变平行**：原本条件较好的一方与伴侣平起平坐，感到不舒服。
2. **平行变下沉**：原本条件相当的一方变得自卑和懦弱。
3. **平行变上升**：原本条件相当的一方努力提升自己，而另一方原地踏步，导致差距越来越大。

本节课中，我们一起学习了磨合期中爱情伤人的原因。在下一节课程中，我们将开始实操课，学习如何进入男人的内心，以及如何用正确的方式让他越来越爱你。
