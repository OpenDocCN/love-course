# 《大迎私教》：1_18-18.三个核心观念

## 概述
在本节课中，我们将学习三个核心观念：目标与接纳、努力与学习、信念与使命。

## 目标与接纳
**目标与接纳**是指设定清晰明确的目标，并在实现目标的过程中接纳可能出现的情绪干扰。

**公式**：目标 + 接纳情绪 = 成功

**示例**：挽回女友的目标是明确的，但在过程中可能会遇到负反馈，此时需要接纳情绪，继续努力。

## 努力与学习
**努力与学习**是指通过不断努力和学习来达成目标。

**代码**：努力 + 学习 = 成果

**示例**：想要获得100个短期结果，需要每天积极搭讪、聊天、邀约，并参加酒会等活动。

## 信念与使命
**信念与使命**是指对自己所追求的目标和人生意义有坚定的信念，并为之努力奋斗。

**公式**：信念 + 使命 = 人生意义

**示例**：设定使命为成为可以影响他人的导师，并为之努力学习和输出。

## 总结
本节课中，我们学习了目标与接纳、努力与学习、信念与使命三个核心观念。这些观念将帮助我们更好地实现目标，找到人生意义。
