# 婚姻心理课：做自己的情感咨询师 - 010.遇事不要慌：婚姻危机公关手册

## 概述
在本节课中，我们将学习如何处理婚姻中的危机情况，以及如何避免常见的错误处理方式。

## 常见的错误处理方式
以下是婚姻中常见的两种错误处理方式：

### 1. 冷战
- **被动攻击**：消极地发泄情绪，如沉默或消极回应。
- **期待对方读懂内心**：不向对方表达自己的需求和感受，期待对方能猜到。

### 2. 忽略矛盾
- **轻飘飘地打断沟通**：在对方想要沟通时，轻描淡写地打断，如说“没什么大不了的”。
- **产生逃避心理**：因为缺乏安全感或不懂得解决问题，选择逃避矛盾。

## 正确的处理方式
以下是处理婚姻危机的五个原则：

### 1. 权威认证原则
- **了解情况**：在做出结论前，先了解清楚情况。
- **非暴力沟通**：通过描述现象和表达感受进行沟通，避免情绪宣泄。

### 2. 速度第一原则
- **及时解决问题**：有问题不隔夜，及时沟通解决问题。

### 3. 承担责任原则
- **反思自身**：在沟通过程中，先自我批评，展示自省。
- **树立标杆**：为对方树立标杆，影响对方沟通方式。

### 4. 真诚工作原则
- **先表达爱，再表达需求**：在表达需求前，先表达对对方的爱。
- **站在对方和两个人立场上说话**：用上帝视角观察问题，客观看待关系。

### 5. 系统运行原则
- **避免与异性讨论个人话题**：避免与伴侣之外的异性讨论个人话题。
- **与女人讲感受，与男人讲道理**：根据男女思维差异，选择合适的沟通方式。
