🎼大家好，欢迎大家继续收看海澜的情绪梳理法。今天我们谈谈失爱，什么是失爱呢？就是失恋、分手、离婚等等，都属于失爱。失爱就是失去一段感情，离开一个人，或者结束一段生活，不管是主动还是被动，首先。

**什么是失爱**？

失爱是指失去一段感情，离开一个人，或结束一段生活，无论是主动还是被动。

**失爱的原因**

1. **不再相爱**：在爱的关系中，最令人心碎的是你依然爱着他，而他的心却已经离你远去。例如被拒绝、被抛弃、被离婚等。被动失去往往让人们对自己产生巨大的否定。

2. **性格不合**：很多恋人或夫妻分手，经常把原因归结为没有找到对的人，认为分开的原因是性格不合。

3. **家庭及环境的阻碍**：有时两人情投意合，但由于父母不赞成、想优先发展事业不在一个城市、住房彩礼、户口等问题压力最终没有走到一起。

**失爱后的阶段**

1. **怀疑、不相信、惊讶、悲伤、愤怒**：有的发展成怨恨、抑郁。

2. **长期的自我价值贬低和无力感**：认为都是自己不够好。

3. **心灰意冷，不想开始新的感情**。

4. **有心无力，想开始，但又总是开始不了**。

**如何应对失爱**

1. **梳理和成长自己**：从根本上疗愈自己，补足自己内心需要的各种养分。

2. **练习自我关怀**：学会自己关爱自己。

3. **柳暗花明法**：找一安静的地方闭上眼睛，做几个缓慢深长的呼吸，让自己的心绪和思绪安宁下来，然后在脑海中把曾经的伴侣带到面前，回想一下你和他在一起感觉到爱的1个场景，列成清单，看一下这个清单，这些场景都体现了对方的什么品质，然后问自己就。🎼这个人具备这些素质和能力吗？其他人有吗？你对自己的发现是什么？你在亲密关系中需要改变什么。成长什么？

4. **积极开始改变自己修复破损的关系**。

5. **寻找新的伴侣**。

**总结**

失去一个人一段感情，不管你多么撕心裂肺，不管你觉得多么难以割舍，无法替代，一切都终将过去。你会发现你所需要的所爱的品质特征，并不是只有一个人有，而实际上你真正爱的并不是某个人，而是他身上具备的你需要的品质和他带给你的感受。慢慢的你就会发现，他带给你的这种感觉，很多人都可以给我们带来。当你发现很多人都可以带来这种感受的话，就会有一种释然。寻找爱人，实际上是寻找生命中缺失的部分，和哪个人的关系并不很大，和他是否具备你所缺失的部分有关。
