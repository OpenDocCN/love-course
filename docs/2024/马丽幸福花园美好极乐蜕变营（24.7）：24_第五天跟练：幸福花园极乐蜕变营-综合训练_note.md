在本节课中，我们将学习如何将之前掌握的核心技巧融入动态的身体律动中，从站立过渡到床上，完成一次综合性的练习。课程分为两个部分：第一部分是跟随指令的分解练习，第二部分是伴随音乐的自由发挥。

## 概述

上一节我们介绍了多种静态的收紧与放松技巧。本节中，我们来看看如何将这些技巧与身体的前后、左右及绕圈运动结合起来，让练习融入更生活化、更具美感的动态中。

## 第一部分：跟随指令练习

以下是今天的练习流程，我们将从站姿开始，逐步过渡到跪姿、趴姿和躺姿。

### 1. 站姿练习

首先，我们以站姿开始。双腿稍分开，膝盖微弯，手臂自然放松，找到一个灵活、动感的身体状态。

**前后运动**
想象耻骨（身体前侧骨盆下方）有一根线向上提，带动骨盆向前卷动，此时**盆底肌自然收紧**。想象尾骨（脊柱末端）有一根线被向后拉，带动骨盆向后摆动，此时**盆底肌自然放松**。动作公式可理解为：**向前 = 收紧，向后 = 放松**。我们重复这个节奏：收紧、放松、收紧、放松。

**左右运动（摆胯）**
这是类似舞蹈中的摆胯动作。将重心移到身体一侧时，尝试让同侧的**盆底肌收紧**，移向另一侧时则**放松**。你可以探索适合自己的节奏和幅度，例如：左移收紧，右移放松；或者只在移动到最左侧或最右侧时收紧。

**绕圈运动**
以盆底区域为圆心，用胯部画圈。在顺时针绕圈时，当胯部运动到最前方，尝试**收紧**；运动到最后方时，尝试**放松**。逆时针绕圈时同理。你可以自由探索在绕圈路径的不同位置进行收紧与放松，让控制变得更加灵活。

### 2. 跪姿练习

接下来，我们转移到床上或垫子上，采用跪姿。这个姿势能提供不同的发力感受。

在跪姿下，重复站姿中的**前后运动**、**左右摆动**和**绕圈运动**。请专注于感受盆底肌在不同方向运动时的状态变化。例如，在左右摆动时，探索是移动到一侧时收紧更自然，还是在摆动到中间时放松更舒适。目标是让盆底肌在任何动态中都能根据需要自如地**收紧**与**放松**。

### 3. 趴姿练习

现在，让我们采用趴姿，双肘支撑，类似平板支撑的起始动作。这个姿势能更深入地感受核心与盆底的联动。

**前后运动**
轻轻将臀部向天花板方向抬高，此时盆底及臀部肌肉处于**放松**状态。然后将耻骨向下压向床面，此时会感受到从大腿根部到臀部、再到盆底区域的肌肉整体**收紧**。重复：放松（抬高）、收紧（下压）。

**左右滚动**
保持趴姿，让胯骨贴着床面进行左右的轻微滚动。例如，滚动向左侧时，尝试**收紧**盆底肌，滚动回右侧时**放松**。同样，可以尝试反方向练习。这个动作像婴儿一样自在，能帮助身体更放松地感知肌肉控制。

### 4. 躺姿练习

最后，我们以最放松的躺姿结束。平躺，双脚平放于床面，或屈膝踩床。这是与伴侣亲密时可能用到的主要姿势之一，因此掌握其中的动态控制很重要。

**上下运动（骨盆卷动）**
吸气时，将骨盆向上抬起，同时**收紧并上提**盆底肌。呼气时，将骨盆放下，同时**放松**盆底肌。可以尝试在提到最高点时保持收紧数秒。代码式指令如下：
```
抬起 -> 收紧上提
放下 -> 放松
```

**左右摆动**
在躺姿下进行骨盆的左右摆动。例如，摆向左时**收紧**，摆向右时**放松**。你也可以探索在摆向任何一侧时都收紧，只在回到中间时放松。

**单腿辅助上提**
将一条腿抬起，再进行骨盆上提的动作。这会改变发力角度，能更针对性地练习单侧控制。左右腿交替进行。

完成这套动作后，你可能会感到有些疲惫。这是正常的，可以平躺放松休息。
