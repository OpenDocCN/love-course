**公式**：高情商 = 明确目标 + 抓大放小

情商高的人会充分思考问题，明确自己的目标，并专注于实现目标。他们不会因为一些小事而分散注意力，从而提高自己的情商。

**公式**：高情商 = 自我提升 + 服务他人

情商高的人会不断提升自己，学会服务他人。他们乐于助人，能够为他人带来快乐，从而提高自己的情商。

**公式**：高情商 = 认输 + 谦逊

情商高的人会学会认输，不会因为一些小事而争斗。他们保持谦逊，懂得尊重他人，从而提高自己的情商。

本节课中，我们学习了什么是情商以及如何提高情商。通过认真倾听、明确目标、学会放弃、自我提升和学会认输，我们可以提高自己的情商，成为一个更加优秀的人。
