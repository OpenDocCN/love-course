## 概述
在本节课中，我们将学习如何建立强大的内核，彻底摆脱供养者属性，从而实现个人蜕变。

## 认知层面
**以下是认知层面的要点：**
1. **接受现状**：首先，你需要接受自己目前的状态，认识到自己还有提升的空间。
2. **剖析不足**：分析自己不足的原因，明确哪些方面需要改进。
3. **制定计划**：根据分析结果，制定改变的计划。
4. **活在计划中**：避免陷入消极情绪，将注意力集中在实现计划上。

**公式：**
- 认知改变 = 接受现状 + 剖析不足 + 制定计划

## 心智模型层面
**以下是心智模型层面的要点：**
1. **潜意识改变**：从潜意识层面开始改变心智模型。
2. **行为与心智模型的关系**：人的行为是过往生活经历所积累的心智模型的呈现。
3. **心智模型突破**：通过训练，改变潜意识习惯，呈现优秀状态。

**公式：**
- 心智模型改变 = 潜意识改变 + 行为改变

## 技术层面
**以下是技术层面的要点：**
1. **技术学习时机**：在认知和心智模型层面有所突破后，再学习技术。
2. **技巧与认知的关系**：技巧是认知的延伸，没有深刻认知的技巧是拙劣的。
3. **核心强者框架理论**：学习如何维护自己的框架，不被欺负，不做生活中的妥协者。

**公式：**
- 技术学习 = 认知 + 心智模型 + 技巧
