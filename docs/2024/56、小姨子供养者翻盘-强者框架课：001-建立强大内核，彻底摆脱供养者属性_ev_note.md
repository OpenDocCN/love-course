# 强者框架课 001：建立强大内核，彻底摆脱供养者属性 🧠💪

在本节课中，我们将要学习如何通过三个核心层面的突破，来建立强大的内在核心，从而彻底改变你在社交关系中的姿态与处境。

## 课程概述

这套课程是男性魅力提升体系的核心课程之一。它能帮助你建立起一个男人该有的强者姿态，从此改变在社交中不被尊重的处境，让你在关系中真正获得自由。

## 核心内容：三个突破层面

整套课程会在以下三个方面带你进行突破，以实现真正的蜕变。

### 1. 认知层面：接纳与规划

想要改变现状，第一步是接受自己目前还不行的事实。当你能接纳这个现实，就可以静下心来剖析自己哪里不行、不行的原因是什么，以及改变哪里就能让自己行。这些都是在认知层面重新认识自己的现状，并做出改变的计划。

要知道，人要么活在计划中，要么活在情绪中。如果你不认清自己并做出改变的计划，就会持续被“我不行”的情绪困扰，持续陷入自卑情绪当中。

### 2. 心智层面：重塑潜意识模型

解决了认知层面的问题，我们就要动手解决自己的问题了。我们要从潜意识层面开始改变自己的心智模型。因为人的每个行为都是过往生活经历所积累的心智模型的呈现。

所以，我们要一个心智模型、一个心智模型地突破，一个心智模型、一个心智模型地训练，最终重新改变自己的潜意识习惯，让自己变成一个不费吹灰之力就能呈现出优秀状态的人。

这个过程占据了整个体系的大部分时间，这也是雕刻自己的过程。要记住，每一个伟大的作品都经历过无数次精细的雕琢。

### 3. 技术层面：巩固与运用

当我们认清自己的现状，也进行了足够多的心智模型训练以后，最后要巩固的就是技术层面的知识了。

有些人可能会好奇，为什么最后才学技术，而不是直接学技术？要知道，想做成一件事，绝不仅仅是一些技巧这么简单。技巧只是认知的延伸。如果你没有足够深刻的认知，对事情的底层逻辑没有足够的思考和练习，只是想走捷径、直接使用成熟技巧，那结果就是东施效颦，所使用的社交技巧也会让人觉得很拙劣。

正所谓“有道无术，术尚可求；有术无道，止于术”，就是这个道理。在这个层面，我会跟你分享核心强者框架的理论，例如如何维护好自己的框架不被欺负，不做生活中总是妥协的那个人。

## 课程总结

本节课中，我们一起学习了“强者框架课”的三个核心突破层面：**认知层面**、**心智层面**和**技术层面**。只要你在这三个部分完成了突破，你会发现你已经拥有了完全不同的人生。身边人对你的态度，也会因为你的转变而发生转变，再也没有人觉得你好欺负。你收获的将是更多的爱慕与尊重。
