# 良叔《强大内核工程》高阶版：007.7.强_内核之语_构建_note

在本节课中，我们将要学习语言如何塑造我们的思维、行为乃至命运。语言不仅是沟通的工具，更是构建我们内心世界和外在现实的基石。理解并重塑你的语言，是走向强大的关键一步。

## 概述：语言的双重力量

你的语言构建了你的行为和思维。你的语言，也可能阻碍你走向强大。这背后的语言构建了你的行为和思想，这个潜意识可以改写你的人生。

## 语言如何塑造现实：从个人经历说起

在开始之前，我们分享一个亲身的例子。在我最早进入情感行业之前，有人问我做什么，我说工作。后来去了北京发展，还有同学问我在做什么，我说挣钱。再后来，说在北京创业。现在直接说在搞点热爱的事情。每改一次说法，感觉都非常不一样。

就如同我在22年的时候，各种疫情新闻和视频都让人人心惶惶。我每一天都特别担心自己阳了，然后把病毒带给孩子。后面我觉察到这个语言建构出来不好的状态之后，我意识到了，决定重新用语言构建一下。担心也是一天，为家人祈福也是一天。开始把刷新闻担忧的时间全部改成了预防问题和祈福的时间。当这样子做了以后，我的状态就发生改变了。

大家有没有发现，我正在用我的语言去构建我热爱的工作。因为这个工作让我很有成就感。

## 语言的陷阱：你如何构建了自己的“命运”

很多人会在不知不觉当中陷入到一种轮回里面。在那些方面看起来很有魅力的人，无论在哪里、什么场景下，他都可以吸引到女生。在什么情况下，他都能把事情做得很好。而那些看起来会失败的人总是很沮丧痛苦。我们把这种经历叫做命运。

以下是几种常见的、由语言构建的“命运”陷阱：

*   **构建低价值的自己**：很多人在面对困境时会说：“梁叔，我这个人就是很社恐，我太紧张了，所以我没有办法和女生好好约会。” 他把不能和女生好好约会，当成了他的命运。那说明他其实已经构建了一个低价值的自己。
*   **构建赚钱太难**：有人评价自己的时候会说：“哎呀梁叔我其实很普通，很平庸的。赚钱太难了。” 他把赚钱太难当成了命运，她已经构建了自己赚不到钱，而且还很自卑。
*   **构建被抛弃的受害者**：有些人和我说：“梁叔，这个女人抛弃了我。为什么我每一段感情都这么认真，都是全心全意的，为什么这些女人都负我。” 她已经构建了自己是低价值的。因为她用的词是“抛弃”。她抛弃了我，所以我没价值。没价值代表没有吸引力，没吸引力，我就吸引不了其他女人，所以我要挽回她来证明我的价值。挽回不了过后就会产生焦虑，越焦虑越想控制，越想控制就越陷入到死循环。
*   **构建不平等的关系框架**：一个学员哪怕表达需求的时候，哪怕是正常的需求，她也会用乞求的语气和女生说：“明天我生日了，我有一个小小小小的心愿，能不能祝我生日快乐。” 在这个学员的命运里面，貌似女人始终都是高高在上的，而她是永远卑微的，她构建的就是不平等的感情框架。

大家意识到了吗？很多时候，我们会听到这样子的语言：“梁叔，我都三十了，还是个处男，我改变不了了。算了，就这样吧，摆烂吧。”“我感觉怎么学习都是这样子。没办法，我就是一个软弱的人，没有那么多的力量去面对别人的攻击、嘲讽和欺骗。所以我不敢进入到一段关系里面。”“我没有那么多的能力去承担工作的压力，我就喜欢吃垃圾食品。”总而言之，这些人都构建了一种自己无法解决问题。

大家发现了吗？这些都是他们的语言当中传达出的信息。但其实正是这些语言构成了他们的命运，正是这些语言构成了他们没法强大。

## 语言的起源：我们如何学会“看”世界

从出生到现在，我们说了无数句话，无论是对女人、对老板、同事、朋友、亲人，对陌生人还是对自己。但是我们从来没有想过，我说出这句话的时候，我在想什么？我为什么会这么说？我每天都要接触的语言，其实都有巨大的魔力，他足以改变我们的人生。

语言是在我们的经历当中被塑造的。之前我在学育儿知识的时候，看到一个知识点，讲述的是婴儿的时候，你眼中的世界是什么样子的。你可能会想，这和我们看到的世界没什么区别啊，花是花，树是树，高楼是高楼，汽车是汽车。但其实并不是这样的。

在婴儿眼中的世界，只是杂乱的一片混沌的。只有等他经过教育，还有了基本的认知，这些事物才能在他的眼里面逐步清晰。比如父母会一边抱着他，一边指给他看：“这是花呀，这是树呀，这是房子啊，这是人哪。”虽然婴儿当时听不懂这些语言，但是日复一日的累积，他就能把这些和语言对应起来。

需要注意的是，当婴儿在学习这些的时候，并不只是在学习这些事物到底叫什么，而是婴儿在通过学习这些语言，学习和事物相处的规律。他知道花和树是不能吃的，知道房子意味着家，知道大家都很喜欢钱，钱可以去买一些东西。

也就是说，语言只不过是我们一个代号。在语言背后，我们怎么和一个事物产生联系的，我们怎么看待这些事物的，我们的认知是什么？这个是更加重要的，也是最核心的。所以与其说我们在通过语言理解这个世界，不如说我们是不断经历学习，然后有了某个认知，然后再把这个认知变成了我们的语言。理解了吗？

## 语言如何限定世界观：以“分手”为例

当语言被塑造出来之后，会发生什么呢？语言会潜移默化地限制我们的世界观。

举个例子，一个学员在女生提分手的时候，他会这么跟女生说的：“说啊，是我太粘人了吗？所以你又要抛弃我吗？删除我。” 那我们会发现，他说的不是分手，不是结束，而是“抛弃”。

这个学员这样子说的时候，就等同于把自己当成了一个没有价值的男人，而女人是高高在上的，有着主导权的。好像她的一念之间就能决定你们的关系走向。

但是事实是什么？就算是女生提的分手，就只能叫做被抛弃吗？有没有可能她只是觉得跟你在一起，你的控制欲太强了，导致压力太大，要离开。或者呢，两个人不合适。又或者是她认为你的吸引出了问题。那分手的原因有很多种，有多种因素，不是单一因素。

**公式：事件（分手） ≠ 单一解释（被抛弃）**

当你说出“我被抛弃”这句话的时候，你就定义了这段关系，也定义了你自己。当用这种角度看待时，她就会注定会痛苦。而且就算进入到下一段感情当中，就算你会面对新的人，这份“被抛弃”的语言，也会让你更敏感、更加胆小懦弱。就算是发生一件很小的事，你也会觉得：“她这句话是什么意思？这个女的是不是也要抛弃我？”因为之前这个事情带给你的负面影响，当你自己的语言没有构建好，等下一个场景再次出现的时候，你的语言又会不断地去构建一些不平等的框架了。

语言在两性当中的影响是很大的，因为它是认知层面的，而且它很隐蔽。当我们说什么的时候，我们认知就已经产生了。这个无意识的东西，它会影响着我们是否真的强大。**当你控制了语言，你就控制了思想。**

## 如何重塑语言：打破旧语境，构建新认知

看到这里，你可能会想：我的确了解到了语言对我的认知有多大影响啊，那我能怎么办？对我的人生有什么用啊？当然有用。首先我们要改变自己的习惯性语言，就可以打破原来的语境，才能形成新的认知，看到更多的可能性。同样的，你也可以通过判断别人的语言，看到他们的伪装和语言背后的意图，这样你才不会被操控和伤害。

一个很有意思的实验，就是让你不要想一只大象，你做不到。因为你告诉自己“不要去想”的时候，本身就已经在想了。这就是一种非常隐蔽的操控。不管你具体说了什么，你的用词本身就已经让对方按照你说的去理解了，明白了吗？

这也告诉了我们一个关于语言的基本原则，也就是你在和对方争论的时候，不要使用他们的语言。他们的语言确立了一个世界观，但绝不会是你想要的世界观，理解了吗？

我的一个亲身经历，就是一次做咨询的时候，遇到过这样子的一个学员。不管我说什么，他都很阻抗，拒绝交流，拒绝改变。后面呢，他直接爆发了一次，跟我说了一长串的指责的话，说：“啊，梁叔，你根本就不行。看起来你是在帮我指导，其实你就是在忽悠。”你巴拉巴拉说了很多很多。

那你在想，如果我同意了他的敌意，产生情绪，那接下来的咨询就没法继续了。因为你知道这是阻抗和防御，需要时间来缓冲，也是需要被允许的。就像守卫城门的人拿起刀和枪。那一般人会认为说：“啊，这个梁叔他还在挑战你，这是对抗。”但是我更加愿意把“对抗”换成了“防御”，把“攻击你的人”换成了“守卫城门的人”之后，换个说法就构建了不一样的关系了，理解了吗？

**代码示例（思维转换）：**
```
旧思维：用户语言 = “攻击” -> 反应 = “对抗/反击”
新思维：用户语言 = “防御/守卫” -> 反应 = “理解/允许/引导”
```

## 总结与行动

在最后我们再总结一下，要想真正的走向强大，就得知道你自己用了什么样的语言，就代表着你会有什么样的世界观。你说出了什么，也就意味着你是什么。

那在最后你也可以思考一下，在过去的感情当中，在困境当中，在你的人生当中，你都用什么样的语言来形容你自己？用什么样的语言来形容你们的原生家庭、你们的关系？或者你有没有掉进什么样的语言陷阱？你又是如何构建这个语言的？

如果用上今天所学的内容，当你用不同的语言去描述它，你会有什么样的改变呢？大家可以好好感受一下。

本节课中，我们一起学习了语言作为强大内核构建工具的核心原理。我们了解到语言并非中立的符号，它源于我们的经历和认知，反过来又深刻地塑造并限制着我们的世界观和行为模式，甚至构建了我们所谓的“命运”。通过识别并改变那些自我设限、构建不平等关系或固化负面认知的习惯性语言，我们可以打破旧有的思维框架，看到更多可能性，从而真正掌控自己的思想和人生走向。重塑语言，即是重塑现实的第一步。
