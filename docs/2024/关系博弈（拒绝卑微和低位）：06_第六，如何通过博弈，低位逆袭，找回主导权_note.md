# 关系博弈（拒绝卑微和低位）：06_第六，如何通过博弈，低位逆袭，找回主导权

大家好，今天我们来探讨如何在关系中转变自己的主导权，如何从低位状态回到高位。这一切的关键在于理解关系中的主导权来源。那么，在一段关系中，主动权取决于什么？

**主动权取决于对关系的依赖程度。依赖程度越高，主动权越低，越被动。** 例如，如果你是一个生活乏味的人，遇到一个可爱的女生，她给你的生活带来了乐趣，那么在你与她建立的关系中，你会更依赖她的快乐，从而变得被动。即使她做了触及你底线的事情，你也会选择容忍，因为你害怕失去她，以及她给你带来的乐趣。

**如果你是一个生活充满乐趣的人，有自己的社交和爱好，那么失去这个女孩的恐惧就不会那么强烈。** 即使失去了她给你带来的乐趣，你也有其他的乐趣可以自足，因此你不会那么被动。

**想要在关系中找回主导权，可以从以下三个方向调整：**

### 1. 融入对方的习惯

**习惯是导致分手后痛苦的原因之一。** 习惯了对方的存在，一旦失去，就会非常痛苦。因此，要让对方的习惯中包含你，让她体会到失去你的感受，从而更加依赖你。

**具体步骤如下：**

1. **观察对方的习惯**：观察对方在每个时段会做什么事情，例如早上、下午、晚上。
2. **拆分对方的习惯流程**：分析对方固定行为动作之前和之后分别是什么。
3. **融入对方的流程**：找到固定动作之间的间隙，将自己融入其中。

### 2. 增加自己的快乐来源

**当一个人能够增加自己快乐的来源时，他会更容易在生活当中感受到满足。** 他自己能够满足自己的时候，就会减少对伴侣、对别人的依赖程度。

**以下是一些增加快乐来源的方法：**

1. **培养兴趣爱好**：选择有创作成分的爱好，例如种花、画画、写作等。
2. **写成功日记**：记录每天的成功和成就，增强自信心和自我价值感。
3. **保留人际关系**：不要因为恋爱而冷落朋友，保持良好的社交圈。
4. **锻炼身体**：提升幸福感，降低对别人的依赖程度。
5. **改变生活环境**：获得新鲜感和刺激感，增加快乐来源。
6. **养成日常习惯**：在固定的时间区间内做固定的事情，增加生活的确定性。

### 3. 加大自己在对方心里的快乐来源占比

**以下是一些加大自己在对方心里快乐来源占比的方法：**

1. **提供支持和鼓励**：在对方情绪低落时，给予关心和安慰。
2. **共同参与一件事**：参与对方的生活，共同完成一些事情，提升关系浓度。

**通过以上方法，你可以在关系中找回主导权，从低位逆袭，成为关系中的主导者。**
