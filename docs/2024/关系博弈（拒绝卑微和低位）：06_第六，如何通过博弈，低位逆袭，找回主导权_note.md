# 关系博弈（拒绝卑微和低位）：06_第六课，如何通过博弈，低位逆袭，找回主导权 🎯

在本节课中，我们将要学习如何在关系中转变主导权，从一个被动的低位状态回到主动的高位状态。核心在于理解主导权的来源，并掌握三个实用的调整方向。

## 主导权的来源：依赖程度 🔑

在一段关系中，主动权取决于谁对关系的依赖程度更高。越依赖关系的人，主动权越低，越被动。

例如，一个生活苦闷无聊的人，遇到一个能带来巨大乐趣的伴侣。他会更需要对方提供的快乐，从而变得被动。即使对方触及他的底线，他也会选择容忍，因为害怕失去对方和对方带来的乐趣。

反之，一个生活本身充满乐趣、拥有自己社交和爱好的人，失去伴侣的恐惧感会弱很多。因为他可以自足一部分乐趣，对关系的依赖程度较低，因此也就没那么被动。

## 找回主导权的三个方向 🧭

上一节我们介绍了主导权源于依赖程度，本节中我们来看看如何通过三个方向进行调整，以降低依赖、找回主导权。

### 1. 融入对方的习惯

很多人分手后感到痛苦，离不开对方，习惯是主要原因。习惯了对方每天的陪伴，一旦习惯被打破，人就会非常痛苦。习惯像空气，存在时不珍惜，失去时才觉珍贵。

因此，要让对方的习惯里有你。然后，找一个月或一周的时间，通过出差或与朋友约会等方式，让对方短暂体验你不在其习惯中的感受。这种“失而复得”的体验会让他更加需要和依赖你，从而让你获得更多主导权。

以下是融入对方习惯的三个具体步骤：

**第一步：找到对方的习惯**
观察对方在每个时段（如早上、下午、晚上）固定会做什么事情。观察一周，你就能大致掌握他的习惯。例如，某人习惯早上写作，中午喝茶。

**第二步：拆分对方习惯的流程**
找到固定行为动作（即习惯）前后的环节。例如，习惯是“喝茶”，其流程可能是“吃饭 -> 喝茶 -> 午休”。拆分出完整的流程是为了下一步的融入。

**第三步：把自己融入对方的流程中**
在固定动作之间的间隙，把自己放进去。例如，在“吃饭”到“喝茶”之间，可以融入“帮我冲茶”；在“喝茶”环节，可以融入“一起喝茶”。当你融入对方多个习惯后，一旦你短暂离开，他会因为习惯被打破而感到别扭和想念你。

千万不要小看融入习惯的力量。当一个人80%甚至90%的习惯里都有你的影子时，他离开你的成本会变得非常大，这正是许多长期关系破裂后痛苦不堪的原因。

### 2. 增加自己的快乐来源

当一个人能增加自己的快乐来源时，他更容易在生活中感到满足。自我满足能力提升，对伴侣或他人的依赖程度自然就会降低。同时，人在快乐充实时会更有信心，能更好地处理烦恼，从而进一步减少依赖感。

以下是增加自身快乐来源的六种方法：

**第一，培养有创作成分的爱好**
爱好最好能产生一个“作品”，例如种花种草（种出一盆植物）、画画（创作一幅画）、写作（写出一篇文章或笔记）。这类需要沉浸精力并产生结果的爱好，能带来深刻的满足感，减少对关系的依赖。

**第二，写成功日记**
成功日记旨在帮助增强自我价值感。每天记录自己的小成就和优点，将注意力分配到做得好的地方。这能有效提升自信心，自我价值感越强，对关系的依赖就越低。

成功日记可以这样写：
```
日期：2023年4月14日
今天的完成了什么？
1. 完成了一份工作报告，获得了同事的好评。
2. 成功解决了一个技术难题，为公司节省了成本。
3. 顺利完成了一项运动训练，提高了身体素质。
靠什么品质完成的？
1. 聪明才智：在工作中发现问题，并迅速解决。
2. 动手能力：运用自己的技能，成功完成了一个任务。
3. 坚韧不拔：在运动训练中坚持不懈，没有放弃。
```

**第三，保留并扩展人际关系**
不要因为恋爱而冷落朋友。良好的人际关系能提供快乐和幸福感，降低对单一伴侣的依赖。保持与朋友的约会频率，甚至扩展交友圈。应将伴侣定位为“丰富生活的人”，而非“成为生活的人”。

**第四，坚持锻炼身体**
锻炼能促使体内释放内啡肽，提升幸福感。身体健康时，情绪更乐观，活力更充沛。从心理层面看，身体健康会让你觉得自己是强壮的，从而不会轻易依赖他人。

**第五，偶尔改变生活环境**
通过改变生活环境获得新鲜感和刺激感。例如，定期调整书房或卧室家具的布局。这种简单的改变能有效增加快乐来源。

**第六，养成日常习惯**
将时间简单划分（如早上、下午、晚上），在每个时段做固定的事情。例如，每天早上9点写作，下午玩游戏。这种日常习惯能带来**生活的确定性**和安全感，减少因外界不确定性而产生的依赖需求。

### 3. 加大自己在对方心里的快乐来源占比

本节我们来看看如何通过正向的相处方式，增加你作为伴侣能提供的快乐价值，从而提升你在对方心中的重要性。

以下是两种核心方法：

**第一，在生活中提供支持和鼓励**
当伴侣情绪低落（如因工作压力大）时，主动关心并提供支持。最有效的支持往往是“共情”，例如，当对方吐槽某人时，你可以和他一起吐槽。避免像一些人那样，在对方需要安慰时反而进行二次指责。

**第二，共同参与一件事**
在亲密关系中，共同参与比单纯陪伴更能培养感情。不要痴迷于每天电话或微信聊天。更有效的方式是参与进对方的生活，例如：
*   一起爬山（共同目标：登顶）。
*   一起做饭（从买菜到洗碗全程协作）。

这种共同完成事情的生活体验，能极大提升关系浓度，让对方感受到真正的快乐，从而加大你在他快乐来源中的占比。

## 总结 📝

本节课我们一起学习了如何在关系中通过博弈找回主导权。关键在于降低自身对关系的依赖。我们探讨了三个主要方向：
1.  **融入对方的习惯**，增加对方对你的需求。
2.  **增加自己的快乐来源**，通过培养爱好、写成功日记、保持社交、锻炼身体、改变环境、建立日常习惯来提升自我满足感和安全感。
3.  **加大自己在对方心里的快乐来源占比**，通过提供支持鼓励和共同参与事件，提升你作为伴侣的价值。

记住，主导权来自于自身的独立与丰富。当你自己的生活足够精彩时，你便能在关系中进退自如，掌握主动。
