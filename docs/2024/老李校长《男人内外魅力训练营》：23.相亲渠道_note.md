# 老李校长《男人内外魅力训练营》：23.相亲渠道

🎼 相亲渠道 🎼

情感的道路充满挑战，充满了人性的贪婪与恶。只有勇敢面对，才能成为真正的英雄。勇敢的男人，跟我一起揭开情感的神秘面纱，洞察人性的善恶，体验世间真挚的情感。

**互联网相亲平台与自媒体大V**

如今，基于互联网平台的和基于自媒体大V的相亲机构特别多，他们承诺通过支付几万元，就能匹配到优质的异性。以下两件事，供大家体会和评估：

1. **化妆师相亲经历**：一位化妆师朋友参加过一次线下相亲，却因为未完成高中教育而遭到鄙视。尽管他是北京一线明星的化妆师，月收入三四万，长相也不差，但在相亲市场上，他的资料被忽视，因为人们只关注他的教育背景。

2. **年轻男士相亲经历**：一位20多岁的男士参加过线下相亲，发现大部分参与者都是外貌出众的人。他的一位朋友在活动中被一位女士拒绝，原因是他外貌不佳。这位朋友表示，他不会再参加任何相亲活动。

**付费相亲平台**

以下是付费相亲平台的几个特点：

- **适合人群**：适合资料和数值特别好的人，如存款过亿、房产多套、豪车等。
- **不适合人群**：数值一般的人可能会在相亲市场上感到羞辱和耻辱。
- **相亲博主**：一些相亲博主表示，他们不缺付费的中年女性客户，而是缺少优质男性和大哥以及年轻漂亮的姑娘。

**总结**

相亲渠道的选择取决于个人的情况和需求。如果你年轻、素质高，可以尝试参加；如果你年龄较大、素质一般，则需要慎重考虑。
