**聊天本质**： 提供情绪价值，展现自我价值的渠道。

**目的**： 升级关系，增加情感连接。

**聊天原则**：

1. **欣赏**： 先欣赏，后表达喜欢，避免过早暴露。
2. **建立信任**： 增加了解，建立信任。
3. **产生共情**： 展示自己，引导对方展示自己。
4. **调动情绪**： 通过语言、肢体动作等方式调动女生情绪。

**认知雷区**：

1. **技巧误区**： 过度依赖技巧，忽视真实自我。
2. **推进关系**： 过早暴露需求感，让对方感到压力。
3. **聊天频率**： 过度频繁，让对方感到不适。
4. **开场方式**： 使用生硬的打招呼方式，拉远关系。

**开场技巧**：

1. **信息联想法**： 根据对方信息进行合理联想，开场亲切。
2. **夸赞开场法**： 找到对方亮点，真诚夸赞。
3. **表情包开场**： 发送可爱表情包，调节气氛。

**展示价值**：

1. **侧面展示法**： 在对话中展示优点，让对方感受到你的优秀。
2. **四感联想法**： 通过感官感受、感觉感想等方式，增加聊天舒适度。
