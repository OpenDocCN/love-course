# 课程012：烽暴聊天法（中）💬

在本节课中，我们将继续深入探讨“烽暴聊天法”的核心步骤。上一节我们介绍了聊天流程的前四点，本节中我们来看看具体聊什么话题，以及如何通过话题推进关系、提供情绪价值。

## 聊什么话题

以下是开启对话时可以使用的几个关键问题，它们能帮助你快速判断对方的基本情况和互动方向。

1.  **询问是否本地人**
    询问此事是前期判断操作方向的标准。目标是长期关系还是短期关系，是持久战还是闪电战。你在本地接触到的对象，不一定都在此常住。如今在大城市间往返的女性很多，总占比可能超过20%。她们可能是来旅游、参加活动、访友、探亲或出差，只停留几天或表示会常来。如果你默认所有资源都是本地人，那是有问题的。也可以根据城市特性分析打法。在一线城市，例如上海、北京等大城市，资源多以“砖十三”（指高价值展示）为基调，或者你确实有超能力进行降维打击。否则，在这些城市进行“低打高”比较吃力。因此，建议先打好展示面这张第一张牌。**公式：吸引力 = 优质展示面**。你有了明信片，才会有资源青睐你。

2.  **询问平时休息喜好**
    这一点可以分辨对方平时的娱乐性质，是爱玩型还是自律健康型。如果她爱玩夜店，说明其开放性程度较高，可能接受快速推进关系。如果对方是学生或外来务工人员，带他们领略都市繁华，也可能快速拿下。如果她是朝九晚五的上班族，且休息时内容积极向上，说明她的部分生活可能需要稳定发展的对象。

3.  **查看对方定位**
    熟悉我聊天风格或常看我直播的兄弟都知道，这个环节在聊天中出现次数较多。但很多兄弟不知道为什么一定要这么做。这是聊天步骤里最重要的一点。首先，问她位置是为了判断她对你的前期防备心理强弱。其次是了解你们之间的距离，以便为邀约做铺垫。有些女生说在本地，但那可能只是她平时的状态，当前未必在。所以前期必须了解对方情况才好出击，而不是打无准备之仗。盲目进攻只会一败涂地。女生给出位置后，你可以趁机升高关系。例如回复：“看看你的位置。”如果对方定位显示30分钟或更远的距离，她觉得有些远，你可以说：“这点距离不算远，心远才是问题。”或者“想见的人，跋山涉水都想待在一起。”如果她情绪好，没有明显反感，就可以继续进攻。如果她表示疑惑或反应一般，我们就转移话题，或根据个人状态顶着压力聊下去。不懂的话，可以发在群里艾特我解答。

4.  **测试兴趣窗口及邀约变现**
    在聊天前期，第四个步骤“分辨窗口”已详细讲解。大家可以根据上面列举的情况，对照自己在聊天中的表现。判断出窗口后，若对方反应不差，可以直接表达好感和欣赏。例如，女生有明显窗口时，可以适当踩油门，快速升高关系。测试小话术：“最近都在休息吗？”、“平时都什么时候休息？”，以具体判断她的休息情况。有的上班族只有周末休息，或只能调休。所以在聊天中要捕捉关键信息，而非长篇大论没有重点。如果她当时忙，就多保持联系，等她下班或隔天休息时再提前邀约。有些玩家的作息不稳定，困了就睡，醒了才活动，没有规律。这种就需要追加消息，敲定见面时间，以防被放鸽子。他们常用的借口是“睡过头了”或“临时有事”。因此，对于玩家，不要说过两天再约，因为他们的心情飘忽不定。情绪不稳定时就不会赴约，需要当下立即敲死，紧紧跟住这条线。然后，一定要表达好感和欣赏：“跟你聊天感觉挺久违的，好久没有这种契合的感觉了。”、“看你照片也是我喜欢的感觉。”注意，一定要说“感觉”，而不是“类型”、“风格”这种能具象表达的词语。否则女生会注意一些你答不上来的点，导致对话直接结束。

## 邀约前的预期植入

我们接着讲聊天步骤的第六点：在邀约之前做好预期，让见面后推进关系的效率更高。如果没有在聊天中植入一些你们即将发生的事情，或向那层关系引导，对方可能以为只是普通见面，最多算不熟的朋友。然后你一见面，没打招呼就直接动手动脚，她会觉得你在耍流氓，直接给你警告牌甚至红牌。因此，聊天中的重中之重，是要时不时植入一些具有暧昧属性的话，让彼此关系升高成为暧昧对象。这样她出来见面时，自然也明白会发生什么。话术举例：
*   “我们待一起肯定很虐狗。”
*   “星座上说我们很般配。”
*   “你这个小身板，我一下就能把你抱起来。”
*   “跟喜欢的人待在一起，怎么样都是种浪漫。”
*   “感觉很久都没有跟谁这么聊得来了。”
*   “无论多晚都等你。”
*   “熬了这么多的夜，甜甜的感情原来是在这等我呢。”
*   直接表达好感等唯一性表达。

这种表达能让她明白你聊天的来意。有人是为谈合作，有人是为结交人脉，有人是为扩列，有人是为谈恋爱。所以，选择引导至哪种关系取决于你。通常，我会在表达好感后提出邀约。但对于新手学员，可以先聊一段内容再做表达，这样更稳妥。不懂如何延续话题的，可以看“接触资源后的战略布局”，里面有分享式聊天和共鸣式聊天的公式。开启话题一般通过朋友圈、热点、时事、天气等第三方话题，或者描述自己的状态和感受来开启。这样聊天不显刻意，对方也好接话。因为第三方话题或描述自身状态的方式不带攻击性，对方也不会觉得有压力。除非你感觉她情绪很好，才可以猛踩油门，进攻性强一些。但要懂得在何时踩刹车或提速。我们参考的点是：在升高关系中，对方给予我们什么样的反应，我们就做出什么样的回应。反应好（亮绿灯）就加速；反应一般（亮黄灯）或产生疑问时，可以减速并破解问题。所以，升高关系的关键在于聊天中植入预期的说辞，要往彼此暧昧关系上引导。

## 升高关系的核心要点

以下是实现关系升级的几个核心要点。

1.  **低需求感**
    这是排在首位的关键点。换句话说，就是使自己的心态放稳，不急不躁。不要对她有过量的占有欲，别管她与其他男生的聊天或吃饭等。别有玻璃心，因为你们只是熟一点的关系而已。过于关心或急着推进关系，会把对方吓跑，也会让你显得不够稳重，从而丧失吸引力。对方只会觉得你很奇怪。

2.  **表达感受与想法**
    让对方了解你，表达你对彼此关系定义的期盼和想法。这有助于你们更好地了解彼此，从而推进关系。敢于表达自己已经优于其他男生，不遮遮掩掩。敢做自己，自信是吸引女生的重要法则之一。没有自信，她也看不见你的光芒，就不会想与你有下一步发展。

3.  **以目标关系模式相处**
    什么关系的模式相处，就会结出什么样关系的果实。你把对方当成自己人聊，相处也会很融洽，没有隔阂。平时多分享生活状态给对方。例如，“现在在干嘛”，给他拍一张照片，或者遇到趣事就分享。这种相处模式通常只有情侣才有。她也会潜移默化地被这种氛围拉进暧昧当中，默认彼此是在接触、交往。聊天中的关系升高也是如此。如果连分享生活这么简单的事都不愿意做，女生会觉得不够了解你，进入不到你的生活中。当你表达好感和进攻时，她会觉得很突兀、冒昧。但在我们的聊天中，主打“汕头局”的情况就是把唯一性表达得很好，让她觉得好像哪里不对，但又无迹可寻。这种道不清说不明的情况，就是暧昧的魅力之处。

4.  **进行深度对话（攻心）**
    在生活中，我们有时遇到很重要或很有趣的人，想与他们建立更深层次的联系，了解对方的想法、感受、经历和憧憬等。但有时不知如何开口，或者怕说错话。也并非所有话题都适合深度对话。有些话题太敏感或太私密，可能让人觉得关系没到位，聊这些会让对方不舒服、不信任你。有些话题太肤浅或无聊，人家会觉得你没意思，或对你没兴趣。两个人之所以会在一起，也是因为彼此有共鸣，能相互理解和照顾，或者对方能带领你。那么，应该选择什么样的话题进行深度对话呢？
    *   **与价值观、信念、理想相关的话题**：例如，你可以问：“在人际交往中，你最看重什么？”、“你最崇拜哪类人？”、“你渴望实现什么目标？”。这些问题可以帮助你了解对方内心世界的核心部分。
    *   **与兴趣爱好、乐趣相关的话题**：例如，可以问：“平时都喜欢做什么？”、“喜欢哪种音乐？”、“最想去哪里旅游？”。这些问题可以帮助你了解对方在生活中享受和经历的事情。
    当然，并非选好话题就能进行深度对话。还要注意你的语气和表情。讨论敏感或私密事情时，一定要用温柔和尊重的语气，保持微笑和真诚的眼神交流，让对方觉得你愿意倾听。还要及时给予反馈和鼓励。听完对方讲述后，一定要给予反馈，并提出一些延展性或探索性的话题。这样可以让对方知道你真的理解并关注其内容，激发她分享更多想法和感受。然后，你可以适当分享一些自己与话题相关的经历和看法。这能让对方感受到你们之间的共同点或不同点，增加彼此的信任度和亲密度。

5.  **提供情绪价值**
    很多学员在刷视频或听课时都听过“情绪价值”这个词。那到底什么是情绪价值？情绪价值是在人际关系中，通过关心、理解和支持他人的情感与感受，让对方感受到被关注、被重视、被爱护的能力。它包括倾听他人需求、给予鼓励和安慰、分享喜悦和困扰等行为。提供情绪价值可以加深人与人之间的亲密度和信任，让关系更稳固和幸福。古时皇帝后宫佳丽三千，每个都经过层层筛选。所以美貌在皇帝面前不是稀缺资源。那为什么被筛选出的美女还分宫女、皇妃等级别？除了自身硬实力，差别就在于是否懂得提供情绪价值。也就是说，让大佬开心，让大佬觉得你最懂他，最能安抚他的情绪，为他分忧解虑，这才是你留在皇帝身边的核心竞争力。这也是《封神》中纣王独宠妲己的原因。面对“低打高”的路，只有一条：提供情绪价值，让对方觉得跟你在一起很舒服，有从未有过的体验。在大多数人看来，情绪价值就是一味的夸赞和顺从。但在我看来，情绪价值有更系统的表现方式，分为正面和反面。简单来说，情绪价值就是给对方正向情绪，影响对方情绪波动的能力。大部分人都喜欢听好话，被赞美、认同、帮助和支持，这些属于正面情绪价值。但有一小部分人喜欢别人逆着他来，俗称“刀子嘴豆腐心”。这种逆反心理叫做负面情绪价值。我们自己不能受负面情绪干扰，要为自己的情绪负责，也不要把负能量发泄在别人身上。懂得调节自己的情绪，让对方感到稳定和有安全感。即使在体验负面情绪时，也能保持理性，不说伤人的话，不做过激行为，避免无法挽救的场面。容易生气也是导致过激行为的反应。可以想一想：为什么要生气？是因为别人的错误，还是自己做得不够好？如果是别人的错误，没必要生气，因为做错的是对方，何必内耗自己的能量？如果是自己的失误，更不应该生气。错误已成事实，下次吸收经验，避免重蹈覆辙就行。你不会的事情才会令你成长，生活也会一遍遍教会你。就像一场没及格的考试，这些题多练习，下次通关后，难题就只是下酒菜。所以，气自己或气别人都是最傻的行为。与对方交流时，可以多表达关心、关注，制造幽默感，进行夸奖赞美。如果对方不是有意放你鸽子或做了不顺你意的事，可以表示善意的理解，有时给予支持，满足对方小需求。落落大方，不计较鸡毛蒜皮，也是一种提供情绪价值的方式。举个例子：对方借了你的充电宝，但没拿稳砸到自己的脚。有些直男学员会先着急他的充电宝，忽略了女生。这种行为不好。最好的反应是：无论什么状况，先安抚女生，然后再去拿东西。这样的排序给人一种很舒服的处理效果，“芳心收割机”的细节就在于此。同时，平时社交也要做好倾听准备。如果对方想倾诉，就耐心倾听；如果她不想多说，给予足够空间。表示“你愿意说的话，我在听”，既给足空间又表达暖心。有兄弟担心变成情感垃圾桶。其实只要做好引导和表达好自己，就不会出现这种情况。如果遇到难解决的问题，也可以发给我解答。有时在伴侣伤心难过时给予支持，也是提供情绪价值必不可少的方式之一。两个人在一起，争论不可避免。但在爆发前，可以软化你开启对话的方式。通过清晰而温和的讨论，对方能感知到你的尊重和关心。避免使用责备、攻击的语气，因为这可能令她更愤怒，相当于增加对方不满的情绪成本。在沟通时，常表达自己的支持和信任。如果她总是什么都不跟你说，很可能是因为你不太懂得怎么去支持和信任她。如果对方在表达想法和决定时，得到的是否定和打击，那谁都不愿再面对你。对方做得不好时，则需要你表达对她的理解和包容，这样她才有足够动力自发改正不完美之处。简而言之，就是让伴侣感受到自己在你身边的价值感和存在感，让你们的关系更亲密。否则，她会觉得做什么都感受不到被需要，慢慢也就不愿多付出了。但不要敷衍评价，例如“真好”、“真棒”、“不错”、“厉害”等让人感觉不真诚的回复。她会觉得你是在敷衍话题。你应该就她说的事情展开来聊，就像亲密关系一样，对她的话、她做的事都用心聆听，让她感受到你是真心的。

## 如何针对不同目标提供情绪价值

在实操过程中，如何知道自己展现的是正确的情绪价值？这与你目标的年龄有较大关系。

*   **年轻目标群体（如学生）**：她们可能更青睐负面情绪价值，也就是我们常说的“作”。她们喜欢在感情中被拉扯的状态，因为现阶段无压力、无忧无虑。所以她们大多处于闲散状态，会觉得男生有态度、有意思，能够带来刺激感和带领感。这也是很多年纪小的女生形成“作闹”模式的原因之一。
*   **优质成熟女性（有一定阅历）**：你会发现负面情绪价值这套已经不管用了。因为目标对象的经历、年龄和认知导致其喜好偏向发生改变。这时，你需要提供正向情绪价值。语言和状态上的热情至少要给到80%的标准量，行动上的肢体语言只需做到20%。用语言传递情绪作为主导，是因为你面对的优质女生，她们的心智被很多事物（如工作压力、复杂人际关系）占据。她们想要的关系一定是更让她们舒服、开心的。只有让她们觉得舒服，她才会把剩余未被占领的心智部分交给你。

那么，具体如何操作，了解她的需求并给予正向情绪呢？首先，可以运用赞美法。用得好，对方会觉得是恰到好处的夸赞，听起来舒服、悦耳；用不好，就会觉得你在拍马屁，不真诚、虚伪。我们的赞美不要模棱两可、含糊其词，给人一种敷衍不走心的态度。好的赞美一定包含真诚和细节。真诚就是让对方感受到你是专注在她身上，眼里都是她，认真看着她。即使简单的夸赞也能有极好效果。一定不要怯场，不要害羞地表达。细节描述就是要体现出对方的独特性，有什么不一样。每个人都想要独一无二、与众不同的心理。所以我们一定要把对方和别人在她身上有什么不一样的点具体指出来。举例：在与目标聊天中，彼此有一定好感时，就可以表达欣赏：“你知道我就欣赏你身上哪一点吗？我最欣赏的是，你明明知道自己长得天生丽质，过着不错的生活，也知道自己是个非常美艳端庄的女生，可以受很多男生欢迎。但你却不喜欢非常肤浅的帅哥，你喜欢的是能够跟别人深度交流的感觉。所以在这点上，我也觉得你跟其他只是单纯长得好看的女生不一样。”这样的说辞会让对方觉得你懂她，而且说得很真诚、表达得很细节，对方也觉得你是真的专注在她身上，从而好感加倍。切记，千万不要说一些烂大街的冷读话术，例如：“别人都认为你很坚强，但我觉得你内心一定非常脆弱吧。”、“你看起来好像很活泼开朗，性格很好，但内心肯定非常寂寞吧。”、“你看起来善于沟通，但懂你的人应该很少。”就是这种老套、没有含义的概括总结话术。如果你面对的是高分女生，人家只会表示无语：“这些话你对多少人说过？你真的了解我吗？拜托，也没说出什么不一样的新花样，讲不出我和别人的不同处。那你就是在广撒网。”对方会直接给你在这张评分卷上打个低分。
