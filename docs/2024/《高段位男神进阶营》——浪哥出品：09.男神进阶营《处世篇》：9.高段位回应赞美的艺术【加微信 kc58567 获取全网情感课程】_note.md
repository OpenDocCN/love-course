**生活中，我们常遇到他人的赞美，但如何分辨赞美是否真诚，以及如何恰当回应，是本节课的核心内容。以下是详细教程：**

### 一、错误心态识别

在回应赞美时，以下三种心态需避免：

1. **矫情**：明明接受赞美，却假装谦虚，让人感觉做作。
2. **刻意**：过度谦虚，试图引导对方继续夸奖，显得不真诚。
3. **较真**：对赞美过于认真，甚至质疑，显得不自信。

**公式**：避免矫情、刻意、较真心态。

### 二、高段位回应方法

1. **认真聆听**：倾听赞美，并表示感谢。
2. **区分赞美对象**：针对不同对象，采取不同回应方式。
3. **直接回夸**：直接赞美对方，拉近彼此距离。

**代码**：

```python
def respond_to_compliments(compliment, person):
    if person == "elder":
        return "感谢您的夸奖，我会继续努力，向您学习。"
    elif person == "peer":
        return "谢谢你的认可，我们共同进步。"
    elif person == "subordinate":
        return "你已经做得很好了，我们一起努力。"
    else:
        return "谢谢你的赞美，我很高兴。"
```

### 三、具体回应示例

1. **长辈赞美**：表现出谦虚，感谢夸奖，并表达学习意愿。
2. **平辈赞美**：积极回应，表达自信，不必过于低调。
3. **下属赞美**：肯定下属，不抢功，共同进步。
