# 情绪价值课-【阿娇】：第3节 情绪价值有哪几种类型

大家好，我是娇姐。今天我们来探讨情绪价值的几种类型。根据日常生活中的常见情绪价值，我们将其划分为六个类型，以便大家更直观地了解。

### 情绪价值的6个类型

1. **带领型情绪价值**：给别人带来快乐的情绪价值。**公式**：饱满的情绪状态 → 感染他人 → 获得不同情绪体验。
2. **治愈型情绪价值**：在对方沮丧、压力大或遇到困难时给予安慰和开导。**公式**：倾听 → 提供建议 → 缓解焦虑和情绪。
3. **分享型情绪价值**：与对方分享日常琐事，增进互动和情感交流。**公式**：倾听分享 → 分享自身经历 → 增进了解和满足感。
4. **陪伴型情绪价值**：陪伴对方，提供温暖和支持。**公式**：陪伴 → 提供安全感 → 增进关系深度。
5. **惊喜型情绪价值**：为对方提供新鲜体验和惊喜。**公式**：探索新事物 → 带来新鲜感 → 增加快乐和乐趣。
6. **认可型情绪价值**：给予对方足够的认可和支持。**公式**：赞赏 → 支持 → 提升自信和存在感。

### 总结

本节课中，我们学习了情绪价值的六种类型，包括带领型、治愈型、分享型、陪伴型、惊喜型和认可型。希望大家能够认识到自己在不同类型中的表现，并努力提升，使自己的情绪价值更加全面，从而更好地与他人建立和维持关系。
