# 清风老师合集：08彻底打败2+1，保护孩子保：094.第十二课：放手以后，如何开始新的生活_n

在本节课中，我们将要学习在决定分开后，如何调整心态、避免常见误区，并开始积极的新生活。我们将探讨三个需要警惕的错误观点，并了解“爱的七个等级”这一成长框架。

在上一节课程中，我们讨论了如何通过法律途径维护自身最大利益，以及如何向孩子和父母传达分开的决定。本节中，我们来看看如何真正地放过自己，为未来铺路。

出现第三者这件事，你可能已经从最初的愤怒，经历了深深的自责与后悔，到达了现在相对平静的阶段。我担心你会陷入一种误区，觉得一切都不再重要，甚至想立刻逃离现实去寻求心灵的绝对平静。我把这称为“心灵的僻静”。这条看似安静的小路，实则是你想为自己寻找一个安全空间。

但这是一条错误的道路。人生总会遭遇各种意外。有时我们独自面对非常困难，甚至会对自己感到极度失望。但请相信，未来会有一个人，他或许没有驾着五彩祥云，但愿意为你披荆斩棘，与你共同经历风雨。

分开以后，常常有三个观点可能影响你的一生。以下是需要警惕的三个误区：

**误区一：我都可以自己来，不需要男人。**
这种观点在分开后很常见，认为对方是累赘。这是错误的。两个人在一起，并非需要对方为我们做什么。我们不是乞丐，不需要施舍。我们需要的是共同成长，去探索生命与爱情，而不是找一个物质ATM机或情感奶瓶。

**误区二：我再也不想接触男人了，男人都一样，都是骗子。**
携手一生的感情需要努力维护。我们羡慕那样的感情，正因其稀少。欺骗或许是人性的一部分，但请不要因此失去对爱情的期待。爱有时也是一种美好的幻觉，就像我们会被虚构的电视剧情感动。真与假并非绝对。如果因一次欺骗就放弃对未来的期待，那是对世界复杂性的低估。我们每天都在欺骗自己，爱人为了让我们活在幸福的幻觉里，也可能付出努力。这种“欺骗”也是爱的一部分。请保留对美好未来的期待，并努力将携手一生的幻觉变为现实。

**误区三：我自己不想找了，就为孩子找个爸爸吧。**
请多为自己考虑。如果抱着这种心态寻找伴侣，你依然不会幸福。因为你并不爱对方，他只是你情感和生活上的替代品。一旦对方察觉，关系仍会出问题。请学会从自己的角度出发。孩子需要照顾，但你也需要爱。

我希望你能认真开始新生活。通过新生活，我们可以反思并见证自己的成长；通过爱人的陪伴，我们可以获得温暖与力量；新生活能让你感到舒适，看到真正的进步。

我知道这条路很艰难。你受了伤，正在默默舔舐伤口，不敢向前。我想陪你一起向前看。中国有句古话：“塞翁失马，焉知非福”。今天失去的，未来或许会有更好的来弥补。虽然无法保证它何时出现，但请一起保持相信，因为“相信”本身就是一种奇迹和力量。

最后，我想与你分享“爱的七个等级”。以下是具体内容：

**第一级：爱自己。**
能够满足自己的需求，不向他人情感乞讨。**公式：自足 = 不索取**。这时，你便具备了爱的能力。

**第二级：爱生活。**
拥有经营精彩独立生活的能力。当有人到来时，你是分享美好，而非索取爱。

**第三级：爱事业。**
拥有自己热爱的事业，找到人生的使命。

**第四级：爱伴侣。**
尊重伴侣感受，平等地付出与接受，共同浇灌爱的土壤。

**第五级：爱亲人。**
接纳亲人以过激方式表达的伤害，化解恨意。改变“等待父母道歉”的思维，与亲人共同成长，让爱充满家庭。

**第六级：爱孩子。**
对孩子进行有条件的教育，提供无条件的支持、引导与认可，帮助孩子成为爱自己、有使命的人。

**第七级：爱众生。**
平等对待、接纳并理解他人，真心提供帮助。

这是我眼中爱的七个等级。你可以对照一下，自己目前正处于哪一级。学会爱自己，是我们都需要成长的路径。

我恳请你认真思考，自己究竟希望过怎样的人生。我也希望在这条路上能陪伴你，一起欣赏风景，共同解决问题与成长。
