# 【李大魔】《极致吸引》：015.低位翻盘核心框架：服从框架②

## 概述
在本节课中，我们将深入探讨低位翻盘的核心框架——服从框架，并学习如何通过建立服从性来提升自己在关系中的地位。

## 服从框架的重要性
**公式**： 服从性 = 高位者姿态 + 逐步建立

服从框架是低位翻盘的关键，它关乎你在关系中的地位和权力。以下是一些关键点：

### 1. 情绪不反应
**公式**： 情绪不反应 = 真正的情绪不反应

情绪不反应是使用情人策略的基础。只有做到真正的情绪不反应，你才能更好地运用策略。

### 2. 借力试探
**代码**： 行为习惯 = 微信二维码直接丢给对方

在关系初期，通过潜默化的方式试探对方的服从性。

### 3. 培养服从度
**代码**： 逐步培养 = 借力超越有益的关系

在社交环节中，逐渐培养对方的服从度，以便超越有益的关系。

### 4. 服从度与关系
**公式**： 服从度 = 0到100的过程

恋爱从0到1的过程，就是一个女人对你的服从从0到100的过程。

### 5. 服从性的培养
**代码**： 逐步建立 = 高位者姿态

培养一个人对你的服从，就是在一段关系中逐步建立你是一个高位者的姿态。

## 低位者思维
**代码**： 低位者思维 = 承受情绪压力

很多人在关系中变成供养者，变成低位者的局面，是因为他们扛不住来自于对方的情绪压力。

## 投射性认同
**代码**： 投射性认同 = 操控别人以潜意识设定的方式对待他

一个人会在与别人互动的过程中潜意识里操控别人，让别人以他潜意识里设定的方式来对待他。

## 供养者与投射性认同
**代码**： 供养者 = 用金钱证明爱

男人通过金钱来证明自己对女人的爱，通过买买买的方式让女人感知到这个行为模式。

## 逆转格局
**代码**： 逆转格局 = 从服从框架开始

想要逆转格局，想要低位翻盘，从服从框架开始进行。

## 总结
本节课中，我们一起学习了低位翻盘的核心框架——服从框架。通过建立服从性，我们可以提升自己在关系中的地位和权力。记住，情绪不反应、借力试探、培养服从度是关键。
