# 倪《高段位主导关系》：014.百万网红聊天案例_ev

## 概述
在本节课中，我们将通过分析一个百万网红的聊天案例，学习如何在社交场合中打破预设关系框架，建立平等和谐的两性关系。

## 聊天案例分析

### 1. 聊天背景
**场景**：商业课程结束后，学员之间互相添加微信进行交流。
**人物**：A（本文作者）、B（百万网红）

### 2. 聊天过程

#### A的聊天策略
1. **打破预设关系框架**：A通过开玩笑的方式，将自己与B的关系从商业社交转变为普通朋友。
2. **转移话题**：A将话题从商业领域转移到个人兴趣，避免陷入商业竞争的陷阱。
3. **建立平等关系**：A通过赞美B的价值，同时表达自己与B在价值上的平等，避免建立地位差。

#### B的聊天策略
1. **展示价值**：B在聊天中多次展示自己的身份和成就，试图建立地位差。
2. **回应A的赞美**：B对A的赞美表示感谢，但仍然保持商业社交的态度。

### 3. 聊天总结
通过这段聊天，A成功地将B从商业社交的框架中拉出来，建立了平等和谐的两性关系。这为后续的交往奠定了良好的基础。

## 核心概念

**公式**：打破预设关系框架 → 建立平等关系 → 营造和谐氛围

## 总结
本节课中，我们一起学习了如何在社交场合中打破预设关系框架，建立平等和谐的两性关系。通过分析百万网红聊天案例，我们了解到，在聊天过程中，要注重转移话题、建立平等关系，并营造和谐氛围。希望同学们能够将所学知识应用到实际生活中，提升自己的社交能力。
