# 魅力之道全板块：010.自信-如何提高自己的自尊（第十天）

大家好！今天我们将探讨如何提高自尊。这实际上是昨天内容的深化。昨天我们强调的是，你的言行应与内心感受保持一致。高自尊的关键在于相信自己内心的想法。

**自尊**是指你对自我的看法，你是否认为自己重要、有能力和拥有各种良好品质。它是自我认可的基础。

**低自尊**的典型表现是觉得自己不够好，面对美好事物时觉得自己不配。这源于缺乏自我认同。

**如何提高自尊**？尊重内心感受，做真实的自己是一个方面，但更重要的是让行为、信念和自我认知保持一致。

不一致的行为会损害自尊，而一致的行为则会滋养自尊。这听起来可能有些抽象，接下来我会分享一个个人故事。

**故事分享**：大约两年前，我已决定从事魅力教学，创业开设频道，帮助想要改变的人。然而，我在一家高强度互联网公司工作，内心矛盾。我口头宣称要创业，但迟迟未行动。这种不一致性让我失去自信。

直到有一天，我意识到言行不一致，决定改变。我开始学习剪辑、PS，制作线上课程。2022年4月，我正式开始行动，频道增长，课程制作，重拾自尊。

**自尊的关键**是依据价值观生活，听从内心声音，避免做自己不想做的事。行为与内心声音一致时，才能拥有自尊。

**练习**：每天填写5个句子，找出不一致性，并针对它们采取行动。

**自我接受**：接受现状，才能进步。不接受现状，就是自我欺骗。

**价值观**：相信并践行自己的价值观，听从内心声音。分辨他人期望与内心声音，追求自己真正想要的。

**作业**：找出句子填空中多次出现的内容，如创业、换工作、结束亲密关系等，制定改变计划。

**总结**：本节课我们学习了如何提高自尊，通过保持行为与内心声音一致，践行价值观，听从内心声音。希望对你有所帮助！
