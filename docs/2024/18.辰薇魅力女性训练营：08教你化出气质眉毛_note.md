# 18.辰薇魅力女性训练营：08教你化出气质眉毛 🌸

## 概述

在本节课中，我们将学习如何画出气质眉毛，包括眉毛的构成、不同眉形的搭配以及化妆工具的使用。

## 眉毛的构成

首先，我们需要了解眉毛的构成。以下是眉毛各部分的名称：

* **眉头**：眉毛的起始处，也是眉毛中最粗的部分。
* **眉腰**：眉头和眉峰之间的部分，颜色最深。
* **眉峰**：眉毛的最高点，位于眉毛的3/2处。
* **眉尾**：眉毛的末端，颜色较浅。
* **眉凹**：眉峰正对下的部分，与眉峰在同一垂直延长线上。

## 不同眉形的搭配

根据脸型选择合适的眉形，以下是四种常见的眉形：

* **标准眉**：眉头最粗，眉峰在眉毛的3/2处。
* **平眉**：眉头和眉峰之间的线条基本平行。
* **挑眉**：眉峰向上扬起，适合圆脸。
* **上扬眉**：眉峰和眉尾都向上扬起，适合提升气质。

## 化妆工具的使用

以下是画眉毛所需的工具：

* **眉粉**：用于画出眉毛的大致形状。
* **眉笔**：用于画出仿真眉线条。
* **眉刷**：用于晕染和梳理眉毛。
* **染眉膏**：用于染眉和梳理眉毛。
* **修眉刀**：用于修剪杂眉。

## 实操演示

以下是一个实操演示，展示如何根据脸型选择合适的眉形并画出气质眉毛：

1. **确定脸型**：首先确定自己的脸型，例如菱形脸。
2. **选择眉形**：根据脸型选择合适的眉形，例如标准眉或上扬眉。
3. **画眉**：使用眉粉和眉笔画出眉毛的大致形状，然后用眉刷晕染和梳理。
4. **染眉**：使用染眉膏染眉和梳理眉毛。
5. **修眉**：使用修眉刀修剪杂眉。
