# 老白《约会达人》：情场3.0：【情场3.0】17.追求阶段之预期_note

在本节课中，我们将要学习“预期”（FP）这一核心概念。预期是在与女生进行肢体升高（BR）前，通过沟通让她做好心理准备的关键技巧。掌握它，能有效避免贸然行动导致的尴尬，并测试对方的接受程度。

## 理解预期（FP）的重要性

上一节我们介绍了追求阶段的不同层级，本节中我们来看看在L4阶段如何顺利推进关系。很多人反映，即使和女生认识很久、关系不错，一旦尝试肢体接触（如牵手）也会被拒绝。

其实，大多数遇到此问题的人，关系并未真正达到L4（吸引阶段），而是停留在L2（熟悉阶段）。在未跨越L3（舒适感阶段）时，尝试牵手确实困难。**真正进入L4后，进行肢体接触是必要步骤**。牵手本身不难，但对毫无好感的女生实施则非常困难。

当关系进展到L4阶段，正确的做法是什么？如果女生对我们有好感，在约会时直接进行肢体接触（CE），女生仍可能因矜持，特别是在毫无心理准备时，下意识地拒绝。然而，她事后回想，内心未必真正排斥这种行为。

因此，在进行肢体升高（BR）之前，我们需要让女生有一个心理准备，让她知晓并期待这件事。这就是**预期（FP）**。预期的主要好处有两点：
1.  让女生提前做好心理准备。
2.  其表达过程本身也是一种测试。如果她不拒绝，表明进行BR没有问题；如果她很排斥，则说明时机尚未成熟，从而避免贸然行动导致双方尴尬。

## 如何设定预期（FP）

知道了预期的好处，那么问题来了：具体该如何操作？我们主要通过微信聊天，以开玩笑或幽默的方式委婉传达。通常，在约会前进行设定，可以在确定约会时或模糊邀约之后。

以下是设定预期（以S1牵手为例）的两种方法：

**方法一：反向调侃**
你可以说：“明天见面了可不能吃我豆腐哦。” 如果女生回复“你不吃我豆腐就好了”或“放心吧，我不会的”，你可以接着说：“那我就放心了，最多就能拉拉手。” 这样预期就成功设定了。只要对方反馈不特别差，见面时就可以尝试S1。

**方法二：自我陈述**
另一种方式是：“我这个人其实还是比较矜持的，一般见面还是有点害羞的，牵手估计要等到见面的时候缓一会儿才敢做。” 这样也委婉地表达了意图。

## 设定预期的注意事项

关于设定预期，有以下几点需要注意：

1.  **预期内容应简单直接**：聊天时可以暧昧，但设定预期时要简单明了。因为聊天可以漫无边际，甚至深入探讨，毕竟只是言语交流。但预期关乎下一次见面要落实的行动，必须谨慎。
2.  **考虑女生的内心感受**：设定预期时，需充分考虑女生的接受程度和内心想法，避免给她造成压力或不适。

本节课中，我们一起学习了“预期（FP）”在恋爱关系推进中的关键作用。我们明白了在L4阶段进行肢体接触前，通过幽默、委婉的方式提前沟通，能让女生做好心理准备，同时测试对方的接受度。记住，操作时要简单直接，并始终顾及对方的感受。掌握好预期，能让关系升级更加自然顺畅。
