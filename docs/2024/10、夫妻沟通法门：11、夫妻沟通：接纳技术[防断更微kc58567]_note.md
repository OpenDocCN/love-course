# 夫妻沟通法门 10：接纳技术 🫂

在本节课中，我们将要学习夫妻沟通中的一个核心技巧——接纳技术。我们将探讨为何沟通会陷入僵局，并学习如何通过接纳对方的感受来建立更顺畅、更亲密的沟通桥梁。

## 沟通为何陷入僵局？😔

上一节我们介绍了沟通的重要性，本节中我们来看看为何夫妻间的对话常常会“聊不上”。许多伴侣反映，起初沟通良好，但近年来却难以交流。

以下是常见的沟通障碍模式：

*   **模式一：证明自己正确**。一方在对话中总是试图证明自己是对的，无论对方说什么，都要进行批评或反驳。
*   **模式二：纠正对方想法**。当伴侣表达观点或抱怨时，另一方会立刻指出“你这样想不对”，并试图提供“正确”的思维方式。例如，当丈夫抱怨领导时，妻子会劝说“领导有领导的难处，你这样背后抱怨不对”。
*   **模式三：寻求对方支持**。沟通的目的变成了要求对方认同和支持自己，而非真正的信息与情感交流。

这些模式会导致一个结果：抱怨的一方不再向伴侣倾诉，转而向外界寻求共鸣。其背后的心理根源，往往不是性格强势，而是**自我认同的匮乏**。一个自我内在力量较弱的人，可能不由自主地通过反对对方、彰显自我来确认自己的价值。

## 什么是真正的沟通？

许多人误将“审问”当作沟通。例如，“今天在公司怎么样？”“和同事关系如何？”这类问题更像盘问，而非沟通。

真正的沟通包含以下要素：

*   有信息或感受需要**分享**。
*   有共同的问题需要**探索**。
*   心中有**疑惑**需要讨论。

沟通是双向的平等交流，而非单向的质询。

## 接纳技术的核心：关注感受，而非真相 🎯

接纳技术的核心在于：**重视对方的感受，而非执着于事情的“真相”**。

我们常常急于追求客观事实，并以此纠正对方。但世界没有绝对的真相，只有每个人基于自身认知产生的**真实感受**。即使事情并非如对方所想，他/她所产生的感受也是真实存在的。误解之所以产生，正是因为双方感受到了不同的“真实”。

因此，沟通的关键是接住对方的情绪、判断和认知。例如，当伴侣抱怨“我们领导就是个王八蛋”时，接纳的回应不是纠正，而是尝试共情：

**“听你这么说，他做的这些事确实挺让人难受的，感觉他有点不近人情。”**

这个回应公式可以概括为：**“重复/描述对方的话 + 认可其感受”**。

## 如何实践接纳技术？🛠️

以下是实践接纳技术的具体步骤：

1.  **倾听与接住**：首先，全神贯注地倾听，不打断。然后，用你自己的话复述对方的核心内容和情绪，表示你听到了也理解了。例如：“你是说……这件事让你觉得……对吗？”
2.  **共情而非评判**：放下对错判断，尝试站在对方的角度去理解他/她的感受。使用表达感受的词汇，如“这一定让你很沮丧/委屈/生气”。
3.  **通过提问引导**：如果你希望帮助对方调整认知，不要直接指正。而是通过提问的方式，引导对方自己思考。例如：“那你觉得，如果换一种方式处理，可能会有什么不同？” 这比“你应该……”的说法更有效。
4.  **给予支持**：在夫妻沟通中，**支持、理解与肯定**远比纠正错误更重要。让对方感觉到“我们是一边的”，是牢固关系的基石。

## 总结

本节课中我们一起学习了夫妻沟通中的**接纳技术**。我们了解到，沟通障碍常源于急于纠正对方和自我证明。真正的沟通始于接纳——接纳对方的**真实感受**，而非争论事实真相。通过**倾听、共情和肯定**，我们可以接住对方的情绪，让沟通渠道重新打开，从而在夫妻之间建立起更深层的理解与支持。

记住这个原则：**你想要得到对方的沟通，就要先付出沟通的意愿与接纳的态度。**
