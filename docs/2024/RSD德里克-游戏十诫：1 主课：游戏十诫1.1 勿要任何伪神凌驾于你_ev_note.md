# RSD德里克-游戏十诫：1 主课：游戏十诫

## 1.1 勿要任何伪神凌驾于你

**核心概念**：

* **公式**： 个人责任 = 主动创造生活 + 追求梦想
* **代码**： `PersonalResponsibility = ActiveLifeCreation + PursueDreams`

### 教程内容

**概述**：

在本节课中，我们将学习RSD德里克的第一个游戏十诫：“勿要任何伪神凌驾于你”。这个诫律强调了个人责任和自我驱动的重要性。

**以下是本节课的主要内容**：

1. **伪神的定义**：
    * 伪神指的是那些凌驾于你之上，让你依赖其决策和期望的人或事物，如政治家、营销人员、机遇主义者等。
    * 伪神会利用你的不确定性和犹豫不决，让你追求他人的梦想，而不是自己的。

2. **不遵守诫律的后果**：
    * 如果你不去追求自己的梦想，你将陷入追求他人梦想的困境。
    * 如果你没有足够的技能，你将成为社会底层的一员，可能无家可归或依赖社会福利。
    * 这可能让你感到满意，但如果你接受这种现状，你就不会听到这个诫律。

3. **诫律的解释**：
    * 不要让任何伪神凌驾于你，意味着你要为自己的生活负责。
    * 这意味着做你自己，追求你想要的东西。
    * 你是自己的CEO，无论你是否愿意。

4. **个人责任的重要性**：
    * 你比任何人都更关心你的人生结果。
    * 你比任何人都更了解什么能让你快乐。
    * 你必须深入思考能带来持久幸福的驱动因素，并开始构建你的未来。

5. **面对现实**：
    * 生活中会有许多不可预测的事件，如自然灾害、疾病、死亡等。
    * 尽管如此，你仍然可以通过改变自己的思想和行为来控制你的生活。

6. **行动的重要性**：
    * 不要找借口，要采取行动。
    * 将你的生活分解成10分钟的时间单位，并尽可能少地将它们浪费在无意义的活动上。
    * 停止拖延，开始做那些你知道能为你的生活增添价值的事情。

7. **纪律的重要性**：
    * 纪律等于自由。
    * 培养纪律，才能获得自由。

8. **承担责任**：
    * 承担责任并不意味着责备自己，而是接受现在的自己，并努力成为更好的自己。

9. **寻找机会**：
    * 寻找承担额外责任的机会，以帮助你实现当前和未来的目标。
    * 培养现在就采取行动的习惯。

10. **建立声誉**：
    * 你的声誉取决于你过去和现在的行为，而不是你将要做什么。

11. **持续进步**：
    * 每天寻找小方法来提升你的技能、知识和能力。
    * 不要满足于现状，要不断进步。

12. **接受不完美**：
    * 没有人是完美的，不要对自己要求过高。
    * 专注于成为更好的自己，而不是追求完美。

**总结**：

在本节课中，我们一起学习了RSD德里克的第一个游戏十诫：“勿要任何伪神凌驾于你”。这个诫律强调了个人责任、自我驱动和行动的重要性。通过承担责任，追求自己的梦想，并采取行动，你将能够创造一个更美好的未来。
