## 男女画饼的差异

上一节我们介绍了课程主题，本节中我们来看看男女在“画饼”这件事上的根本差异。

不得不承认，男性在画饼方面更具天分。这是因为在雄性竞争过程中，例如想成为老板或管理者时，画饼是考验一个男性能否成为好领导的基本门槛。能够为团队描绘一个清晰的愿景，让团队拥有信念和方向，最终取得成果，这就是一个好领导。因此，几乎所有男性在初期都不会排斥学习画饼。

而女性则恰恰相反。对女性而言，主动向男性提出要求已经是一件困难的事。提出“画饼”则更为艰难。

女性不擅长画饼，主要原因是练习得少，并且女性通常更重视承诺，因此不敢轻易开口。大多数男性在画饼时则没有这种心理负担。因此，学会画饼的第一步，是放下一个执念：**说到就必须做到**。说到并能做到当然是好事，但在大多数情况下，无论是团队还是家庭，想要取得一番成绩，往往需要先描绘一个愿景。能否完全成功并非首要，重要的是让大家感觉跟着你有方向。一个家庭也是如此。

## 画饼与欺骗的本质区别

在深入公式前，我们必须明确画饼与欺骗的本质区别。

反向画饼和欺骗是有区别的。我们所说的画饼，是给对方一个美好的愿景，让对方先建立信任并付出努力，最终我们共同将这个愿景实现。

而欺骗，则是以单纯获取对方资源为目的。例如，我本不打算与你结婚，却天天承诺会与你结婚，这就是诈骗。最终对方付出了车子、房子、票子却一无所获。我们给人画饼，是为了共同的未来，是为了实现一个美好的期待，绝不是以此作为手段来骗取不合理的收益。

## 画饼四步公式 🧩

明确了概念和原则后，现在我们来详细拆解画饼的核心公式。该公式分为四个清晰的步骤。

### 第一步：可验证的小事实

画饼的前提绝不是空泛的愿景，而是当下的现实。你需要一个可以被确认的小事实作为起点。

**公式：画饼起点 = 可验证的现有事实**

例如，如果我说：“我不仅会讲情感课，我还会造飞机。”你不会相信，因为情感和造飞机毫无关联。但如果我说：“我不仅会讲女性情感课，其实我也很精通男性情感。”这可信度就高很多，因为两者属于同一领域。

在生活应用中，你可以这样说：“老公，我们这个月加在一起，我挣了8000，你挣了1万，我们总共18000。明年我们努努力，争取全家年收入达到24万。我每月多挣1000，你每月也多挣1000，我们一起完成这个目标。”这就是通过一个可以验证的小事实（当前月收入），引出接下来的共同行动。

另一个经典例子是男人给女人画饼：“你看我们都见过父母了，下一步是不是该推进结婚了？关于彩礼，我很爱你，也很在乎你，但这个金额我们是不是可以再协商一下？”这是通过“见过父母”这个小事实，让女方觉得关系有结果，从而顺势压低彩礼。

### 第二步：事实背后的共同价值

基于小事实，我们需要将其价值升华，从个人层面上升到双方共同的利益层面。

**公式：价值升华 = 个体事实 → 共同利益**

你给对方画的饼，必须是对方也能吃到的。如果饼画得太大太远（例如“我们一下子要月入10万”），对方会觉得不切实际而失去动力。

承接上例，在说完收入目标后，可以补充：“老公，我们一起把这个家经营得更好。未来我们有孩子了，多出来的这部分收入也可以作为孩子的共同储备。”这样就把对方拉入了家庭整体规划中。当他觉得挣钱是为了“我们”的家，而不仅仅是为“你”挣时，动力就会不同，会觉得这个饼真实、自然且舒适。

在向上社交时，也可以运用：“哥哥，你看我今年自己干企业一年有几十个收入。在跟你学习的过程中，我收获很多，明年我争取能干到100个。希望明年我能有更多实实在在的东西回报给哥哥。”对方不一定直接给项目，但听到这样的话会开心，你的资源和机会也就变大了。

如果对方已经给了你项目或经济上的帮助，一定要学会感谢并继续画饼：“哥哥，今年谢谢你。这个100万的项目，没有你我根本没机会做。我们净利润是50个W，我给你准备了一份礼物，虽然不贵只有十几个，但也是我的一片心意。明年我希望再接再厉，给哥哥创造更多价值。”懂得感恩的人，自然会获得更多机会。

### 第三步：基于事实的可行计划

价值描绘完成后，需要提出一个具体的、可落地的计划，让对方看到实现的路径。

**公式：可行计划 = 具体行动方向 + 时间节点**

例如：“老公，从3月份开始，我打算学习一项新技能，多学点管理知识。争取到今年年底公司调薪时，我能晋升为我们部门的小组长。”这就提出了一个清晰的计划，让对方基于这个事实和计划，来考虑如何支持你。

### 第四步：计划的具体拆分与单一请求

这是将计划付诸实施的关键一步，重点是聚焦于一个清晰、明确的小目标，一次只提一个要求。

**公式：有效请求 = 一次只提一个明确、可执行的小要求**

千万不要同时提出多个要求。例如，不要同时说：“你既要给我工资卡，又要每月给家里生活费。”谈工资卡和生活费，一次只谈一个。同样，跟“大哥”不要同时说：“你今年既要给我项目，又要让我出去进修。”项目和进修也只谈一个。

我们只聚焦一件事。当目标聚集在一个点上时，画饼才有意义。重点不是画饼本身，而是画完之后，对方愿意按照我们说的去做，给我们资源和支持。只有集中到一个点，对方才会觉得这个计划可行。毕竟对方不傻，他拥有资源，条件可能更优，他心里会权衡给你支持需要付出什么代价。聚焦一点能最大程度提高他答应的概率，也能有效提升我们在这个议题上的话语权。

以下是提出具体拆分请求的示例：
*   “老公，我打算花1万块去报一个管理课程，为了咱们家更好，我要在工作上更努力。”（请求经济支持以完成学习计划）
*   “老公，原来我们的工资卡是分开的。现在为了家庭年收入突破20万这个共同目标，我们把工资卡放一起管理，一起努力吧。”（请求财务共管以实现共同目标）

## 当对方拒绝时怎么办？🔄

上一节我们介绍了如何提出具体请求，但计划可能遭遇拒绝。本节中我们来看看如何应对。

如果对方听完画饼后，没有直接答应你的诉求，不必着急。因为基于计划的拆分是逐步推进的：先提出目标并获得认可，再提具体计划。

如果对方拒绝，关键在于询问“为什么”，而不是质问“凭什么”。

**有效应对 = 询问“为什么” > 质问“凭什么”**

例如，你说：“我们一年想多接一个项目，多挣100万。”他说不愿意。这时你应该问：“为什么？是觉得我业务能力还需要提升吗？”如果他给出原因（例如业务能力），你就有了新的落地方向：“好，那我去报名参加一个业务能力提升班。”只要你提出了改进方向，结果一定会比现在更好。

再比如，你说：“老公，把工资卡给我管吧。”（这是基于共同理财计划的拆分请求）。他说：“不行。”你应该问：“为什么？是担心我管不好账吗？”如果他承认是担心账目管理，你就可以回应：“老公，我专门做了一个家庭账目规划表，你看看。我们可以先试运行一个月。”通过问“为什么”，对方会觉得你很真诚，是真心希望两个人好。当他讲出原因，你就有机会解决这个障碍，从而最终拿到想要的结果。

## 课程总结 📝

本节课中，我们一起学习了“画饼式撒娇”的四步核心公式。

1.  **可验证的小事实**：这是画饼的起点，必须基于你们当下已经拥有的、可确认的结果，由此引申出更大的目标。
2.  **事实背后的共同价值**：将个人目标升华到双方共同利益。这个饼不能只有你一个人受益，必须是双赢，你只是顺势推动了这件事。
3.  **基于事实的可行计划**：提出一个让对方感觉可行的具体行动计划。目标要基于当下情况，有规律可循。例如收入增长目标，设定在20%-30%的增幅是合理的，超过100%则显得不现实。
4.  **计划的具体拆分与单一请求**：将计划拆解为一个具体的、可执行的请求，一次只聚焦一个点。这能让对方付诸实际行动，使我们拿到实际结果。

通过这四步，你可以自然、真诚地描绘共同愿景，引导关系向更积极、富足的未来发展。
