# 老公出轨心理27堂课：23 男人出轨求原谅后，为什么再次出轨？

## 概述

在本节课中，我们将探讨为什么有些男人在出轨后求原谅，却仍然会再次出轨。我们将分析背后的心理动力，并探讨如何应对这种情况。

## 翘尾现象

**翘尾现象**是指在婚外情生命周期中，出轨行为先上升后下降，然后再次上升的现象。这种现象在90%以上的男人中都会出现。

## 出轨类型

根据案例数据统计，出轨类型主要有两种：

1. **与同一个人产生多次婚外情**：例如，与某人在分手后再次复合，或与不同的人分手后再次与同一个人复合。
2. **与不同的人产生婚外情**：例如，与一个情人分手后，又与另一个情人开始婚外情。

## 出轨心理驱动力

以下是导致男人再次出轨的七个心理驱动力：

1. **婚姻矛盾**：例如，婆媳矛盾、房事矛盾等。
2. **婚姻倦怠**：例如，夫妻生活乏味，缺乏激情。
3. **事业因素**：例如，事业成功或失败，需要帮手或合作伙伴。
4. **人生升级**：例如，结婚、生子、升职等。
5. **特定需求激发**：例如，为了传宗接代或满足特定需求。
6. **基因家族基因问题**：例如，家族中存在出轨现象。
7. **社会风气**：例如，社会道德观念的影响。

## 求原谅的原因

以下是男人出轨后求原谅的原因：

1. **婚姻没有死**：在出轨之前，婚姻关系并未破裂。
2. **婚外情不是替代婚姻**：婚外情只是对婚姻的补充，而非替代。
3. **内心软弱，外在依靠老婆**：男人无法独立面对小三，需要依靠老婆。
4. **求生的本能**：不想离婚，不想家庭破裂。
5. **内心认识到婚外情的不道德**：知道婚外情会伤害婚姻，但无法抗拒诱惑。

## 再次出轨的原因

以下是男人再次出轨的原因：

1. **没有决定分手**：分手只是策略，并非真心。
2. **婚外情果汁太美味**：婚外情带来的激情、轻松、愉快等感觉难以抗拒。
3. **婚外情分手没有三个阶段**：意识层面、物理层面、依恋关系破裂。
4. **策略分手后没有持续的压力**：婚外情没有见底，没有打穿。
5. **婚外情没有惩罚**：没有心理层面和物质层面的惩罚。
6. **婚姻和婚外情的创伤没有疗愈**：婚姻和婚外情的防御机制仍在。
7. **婚姻没有重建**：婚姻的规章制度、信任、日常生活管理没有重建。

## 总结

在本节课中，我们学习了男人出轨后求原谅，却仍然会再次出轨的原因。了解这些原因有助于我们更好地应对这种情况，并采取措施防止再次发生。
