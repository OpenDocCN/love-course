## 四个基本动作

### 1. 初始动作
**公式**：呼气收紧 → 吸气放松 → 呼气收紧 → 吸气放松

**操作**：
- 找到目标肌肉，跟随指导进行呼气收紧和吸气放松。
- 初始动作是最简单的收紧和放松过程。

### 2. 持续收紧
**公式**：呼气收紧 → 屏住呼吸 → 持续收紧（3秒/5秒/8秒/10秒/15秒/20秒） → 吸气放松

**操作**：
- 在收紧过程中保持呼吸，逐渐增加持续收紧的时间。

### 3. 越加越近
**公式**：呼气收紧 → 越夹越紧 → 吸气放松

**操作**：
- 逐渐增加夹紧的力度，直至达到极限。

### 4. 蝴蝶正次
**公式**：呼气收紧 → 吸气放松 → 呼气收紧 → 吸气放松 → ... → 提速收紧 → 回到原速

**操作**：
- 通过改变收紧和放松的速度，实现变速训练。

## 练习提示
- 在未来的三个月或六个月内，专注于基本动作的练习。
- 在日常生活中，如等地铁、办公时，随时进行初始动作的练习。
- 持续练习，即使只练习一个动作，也能取得效果。

## 总结
本节课中，我们一起学习了蜜道训练的四个基本动作，包括初始动作、持续收紧、越加越近和蝴蝶正次。通过持续练习，这些动作将有助于提升蜜道训练的效果。
