# RSD泰勒《蓝图解码2.0重铸蓝图》课程0：预览：为什么自恋的人具备一定吸引力 🔍

在本节课中，我们将探讨一个社会心理学现象：为何具备自恋特质的人，往往能展现出某种吸引力，并能在特定领域（如创业）中取得显著成就。

我们的社会中总会存在一定程度的自恋者。许多自恋者能成为卓越的创始人。如果你看过“火人节”或相关纪录片，就会明白这一点。自恋者通常是优秀的创始人和项目发起者。

然而，他们也可能把事情弄得一团糟。自恋者把事情搞砸的程度可能超乎想象。有时，你需要一个这样“疯狂”的人站出来说：“我们要做这个。”而你的反应可能是：“这根本说不通，你不可能成功。”他们的想法听起来可能很荒谬。

---

上一节我们提到了自恋者在发起项目时的特质，接下来我们具体分析其吸引力来源与潜在问题。

以下是自恋者可能具备吸引力的几个关键点：

*   **愿景与行动力**：他们通常是项目的“种子”和“发起者”，拥有强烈的启动意愿。
*   **自信与感染力**：其强烈的自信能吸引追随者并推动项目初期发展。

以下是自恋者可能导致的问题：

*   **管理混乱**：他们很可能“把事情弄得一团糟”。
*   **决策风险**：其想法可能“根本说不通”或显得“荒谬”，导致项目脱离实际。

---

本节课中，我们一起学习了自恋特质的双面性。我们了解到，自恋者因其强烈的自信、愿景和行动力，在作为创始人或发起者时具备独特吸引力。但同时，他们的特质也可能导致管理混乱和决策脱离实际，从而带来风险。理解这一现象，有助于我们更全面地看待个人特质与成功之间的关系。
