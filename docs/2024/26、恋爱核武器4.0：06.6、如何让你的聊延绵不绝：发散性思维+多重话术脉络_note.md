# 恋爱核武器4.0：06.6 如何让你的聊天延绵不绝：发散性思维+多重话术脉络

## 概述

本节课将学习如何通过发散性思维和多重话术脉络来延续与女生的聊天，使对话更加流畅和有趣。

## 发散性思维

### 微观视角

1. **拆分句子**： 将女生的话拆分成不同的词语，例如“我”、“闺蜜”、“厨艺”。
2. **联想词语**： 从词语出发，联想到相关的话题，例如“闺蜜”可以联想到“单身狗”、“想有闺蜜”等。

### 宏观视角

1. **确定话题**： 确定当前聊天的话题，例如“美食”。
2. **寻找相关话题**： 从当前话题出发，寻找相关的话题，例如“啤酒”、“饮料”、“健康”等。
3. **使用思维导图**： 利用思维导图来启发话题，例如“美食”可以扩展到“厨艺”、“具体食物”、“美食节目”等。

## 多重话术脉络

1. **往回聊自己的话题**： 当聊天陷入困境时，回顾自己之前的话题，寻找可以继续聊的方向。
2. **往回聊他的话题**： 从女生的话中寻找可以继续聊的话题，例如从“喝酒”回到“闺蜜”。
3. **一次只说一句话**： 避免一次性说太多话，保持聊天简洁。
4. **避免捡垃圾**： 不要重复之前被女生放弃的话题。
5. **升级关系**： 在聊天中适当升级关系，例如约女生见面、称呼对方为“宝贝”等。
