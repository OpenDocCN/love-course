# 猫咪外卖案例两则：闪电战攻略

## 案例一：深夜聊天，快速升温

**以下是案例一的关键步骤：**

1. **时间选择**：选择深夜时间发起聊天，增加对方的好奇心。
2. **表情测试**：发送萌萌的表情，测试对方的反应。
3. **简单开场**：询问对方是否已睡，以轻松的方式开启对话。
4. **了解信息**：通过聊天了解对方的基本信息，如工作、居住地等。
5. **情感升温**：从理性客观的条件聊天，过渡到情感化的聊天，拉近彼此的距离。
6. **邀约见面**：在合适的时候，提出邀约，观察对方的反应。

**案例一的核心概念**：

* **公式**：深夜聊天 + 表情测试 + 简单开场 + 了解信息 + 情感升温 + 邀约见面
* **代码**：`深夜聊天 -> 表情测试 -> 简单开场 -> 了解信息 -> 情感升温 -> 邀约见面`
