# 清风老师合集：12情感变现实操训练营30期：145.【私域】第四课：私域话术：让客户主动买单的秘密

## 概述
在本节课中，我们将学习如何在私域中建立人设，提升用户关系，并最终实现成交。

## 人设建立
**公式**：用户认知 = 个人形象 + 个人简介 + 个人故事

以下是建立人设的关键步骤：

1. **个人形象**：拍摄一张专业且具有故事感的个人形象照。
2. **个人简介**：在简介中明确自己的工作、资历和能为用户解决的问题。
3. **个人故事**：分享个人经历或成功案例，使人设更加立体。

## 关系升级
**公式**：用户持续关注 = 优质内容 + 持续价值

以下是关系升级的关键步骤：

1. **优质内容**：输出有思想、能解决用户需求的内容。
2. **持续价值**：不断更新和升级引流内容，如破圈指南等。

## 成交
**公式**：用户付费 = 信任 + 高价服务

以下是成交的关键步骤：

1. **信任**：通过建立人设和关系升级，让用户信任你。
2. **高价服务**：服务定价至少500元/小时，初期可设定较低价格，如699元或999元。

## 总结
本节课中，我们学习了在私域中建立人设、提升用户关系和实现成交的三个关键时期：人设建立、关系升级和信任成交。希望大家能够将所学知识应用到实际操作中，实现私域变现。
