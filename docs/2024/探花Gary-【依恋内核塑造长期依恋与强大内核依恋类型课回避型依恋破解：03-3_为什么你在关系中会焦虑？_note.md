## 安全感的内核：自信

上一节我们介绍了母婴关系对依恋的影响，本节中我们来看看焦虑型依恋的具体表现。许多焦虑型的人常问：为什么我总是没有安全感？要回答这个问题，我们首先需要理解安全感的本质。

安全感的内核根源是**自信**，它分为对外和对内两个维度：
*   **对外自信**：相信这个世界和其他人都是值得被爱的。
*   **对内自信**：相信自己是值得被爱的。

焦虑型依恋的问题根源，在于**对内自信**出现了问题。思维中存在着创伤性的**负向核心信念**，继而演变成“**觉得自己不值得被别人爱**”。这个信念在潜意识中影响着所有的社交和亲密关系行为，例如经常要求对方用很多行动来证明爱意。

## 案例：教养方式与依恋创伤

为了更具体地理解，我们来看一个案例。一位学员在亲密关系中，因对伴侣提出过分要求（如严格限制回家时间、社交需报备）而导致对方离开。其根源可追溯至童年：作为留守儿童，父母每年短暂的归来与离开，让他反复体验“浓烈的爱”与“被抛弃的焦虑”。这种不稳定的成长环境造成了心理创伤，导致他在成年后的关系中，潜意识里不断需要确认伴侣是否会像父母一样离开。

这个案例说明，除了婴儿期的依恋模式，**父母的教养方式**同样深刻影响孩子的依恋模式，可能造成自尊体系不完整，并影响与他人的关系。

## 四种教养类型及其影响

原生家庭早期的教养方式如何影响我们的依恋模式呢？我们可以从父母对孩子的“要求性”和“反应性”两个维度来观察，共有四种主要类型。

以下是四种教养类型的简要介绍：

1.  **专制型**
    *   **特点**：高要求，低反应。父母采用专权态度和强制措施。
    *   **影响**：孩子易形成畏惧、逃避情绪，缺乏自信，难以进行亲密表达。容易形成**焦虑型或恐惧型依恋**，并可能无意识效仿父母的专制行为对待他人。

2.  **忽视型**
    *   **特点**：低要求，低反应。父母既不管教也不关心，孩子处于“野蛮生长”状态。
    *   **影响**：可能发展出两种极端：一是极度独立，演变成**回避型依恋**；二是因缺乏存在感，害怕被忽视，演变成**焦虑型依恋**。

3.  **溺爱型**
    *   **特点**：低要求，高反应。父母极度关怀但缺乏原则，放任孩子行为。
    *   **影响**：孩子容易迷茫，缺乏规则概念。在亲密关系中可能敏感，渴求伴侣像父母一样溺爱自己，也容易演变成**焦虑型依恋**。

4.  **权威型**
    *   **特点**：高要求，高反应。父母进行正面管教并积极回应孩子需求。
    *   **影响**：这是最值得推荐的教养方式。在此环境下成长的孩子，成年后状态更健康稳定，易于形成强大内核，构建**安全型依恋**。

除了权威型，其他三种教养方式都可能造成不安全的依恋风格，核心原因在于父母未能给孩子灌输足够的信任感。在孩子的潜意识中，“父母的不信任”等同于“我不值得被爱”，这种**负向核心信念**会伴随成长，并在所有关系中如影随形。

## 焦虑型依恋的六大特征

受到负向核心信念的影响，焦虑型的人在恋爱中通常表现出以下六大特征。你可以对比自己或伴侣的情况，以便针对性调整。

以下是焦虑型依恋的六个典型表现：

1.  **自我价值感低**
    *   不相信自己值得被爱，自我脆弱，认为必须依赖他人。其情感“养分”几乎全部来自外部关系，**自爱能力几乎为0**，像寄生虫一样向外索求爱。

2.  **情绪容易失控**
    *   内心常产生难以自控的“情绪海啸”，极易被负面情绪淹没，无法独立面对和安抚自己，需要依靠他人来“拯救”。常将中性事件曲解为威胁（如：不接电话 = 不爱我 = 抛弃我）。

3.  **害怕并难以拒绝**
    *   潜意识认为“被拒绝=不被爱=被抛弃”。难以区分对方是拒绝事情还是拒绝自己这个人，容易因此产生挫败和愤怒。

4.  **渴望融合，导致越界**
    *   核心恐惧是“被抛弃”，因此会过度追求亲密来对抗恐惧。这常常侵犯对方的心理边界，不尊重他人意愿，给对方造成压力。

5.  **敏感多疑，表达需求不当**
    *   因童年被忽冷忽热地对待，不擅长直接表达需求，害怕被拒绝。更倾向于用“投射性引导”让对方猜测并满足自己，若对方未做到则心生不满。

6.  **掌控欲强，患得患失**
    *   内心有一套严格的恋爱标准，过度关注对方是否做到，并对此非常敏感。当付出未得到预期回报时，容易感到愤怒、焦虑和怀疑，让负面情绪充斥关系。
