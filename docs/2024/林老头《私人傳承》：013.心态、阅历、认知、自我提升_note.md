# 林老头《私人傳承》：013. 心态、阅历、认知、自我提升 🧠

在本节课中，我们将要学习构建稳固心态的四个核心支柱：阅历、处理当下、接纳与身份认同。这些要素共同构成了个人成长与认知提升的基础。

## 阅历：心态的基石 🧱

上一节我们介绍了课程主题，本节中我们来看看第一个核心点：阅历。任何心态都是上层建筑。上层建筑的牢靠、美观或实用程度，都取决于其底层建筑——阅历。

让一个从未接触过大城市繁华与复杂环境的孩子去经历这些，他的心态是难以稳定的。没有阅历支撑的心态是空中楼阁。

我在前几年，心态初步形成时，曾在全国各地游历。这段经历让我见识了许多，也经历了许多。

以下是关于积累阅历的一些建议：

*   旅行与探索：去你从未去过的地方，例如香港、台湾、澳门。打破已知的舒适圈。
*   尝试新事物：报名你从未接触过的课程，即使它可能价值不高。你花钱买到的不仅是知识，更是经历。
*   保持开放：永远不要封闭自己。敢于尝试，永远处于“进行时”的状态。

去做那些你没做过的事情，这对于提升你的整体认知有巨大帮助。阅历会成为你认知提升的底层支撑。

## 处理当下：专注眼前之事 🎯

上一节我们探讨了阅历的基础性作用，本节中我们来看看如何专注于当下。许多人存在一个问题：目光放得太远。

例如，尚未开始学习，就幻想自己已成为大师；尚未开始工作，就幻想自己年薪百万。这是一个错误。

专心做好当下的每一件事情才是最重要的。许多人学习了大量理论后，便认为自己很厉害，不屑于实践，这是可笑的。

以下是处理当下的具体方法：

*   立足现实：以你当前的经济实力、颜值和条件，或许只能吸引到“6分”的女生。不要看不起，就去实践。
*   循序渐进：当你完全掌握与“6分”女生相处后，再去考虑“7分”的女生。永远不要想跨维度打击。
*   从基础做起：大部分人都高估了自己。处理好当下你能处理好的每一件事。

如果你听了这节课却没有实战，那就去实战。不要幻想自己天赋异禀，可以跳过基础步骤。这样做会摧毁心态，无法建立良好的心理基础。

这就像没有从基层做起就想直接担任CEO，你完全无法掌握公司运营。扎实的经历会成为你面对未来一切事情的基石。

具体是否去执行那些简单、重复甚至低效的事情，取决于你自己。

## 接纳：包容一切发生 🤝

上一节我们强调了专注当下的重要性，本节中我们来看看心态的第三个关键：接纳。讲到接纳，最关键的一点是接纳一切已经发生的事情。

接纳你的出身不好，接纳你的工作一般，接纳你的先天条件一般，接纳你的认知水平一般。接纳所有负面，同时也接纳所有正面。

这就是“宠辱不惊”。接纳了所有，无论是夸奖还是批评，都能一笑而过，然后反思自己。

这样的心态是成为任何行业顶尖人物的前提。我做情感内容这么多年，也经历过低谷。

那时我认为我的粉丝都是高认知群体，因此做了一些不接地气的内容。结果发现大家并未到达那个阶段。那段时间很艰难，但你必须接受它。

正是这样的过程和经历，让我更理性地看待市场，看待事物。这些都会成为你成长的基石，成为你认知和心态的基石。

因为你经历过，你知道这样做的结果是什么。你知道即使带来负面影响，这些影响反而会让你变得更好、更强、认知到更多。

## 身份与自我认同：定义你的轨迹 🪐

上一节我们讨论了接纳的重要性，本节我们来看最后一个点：身份与自我认同。这个点我们在直播课里讲过。你会有一个社会身份。

例如，你去健身房，怕别人看不起你。那么，你就去接纳自己就是健身房最底层的那批人。

**公式：接纳身份 = 承认现状（“我肌肉小，我最弱”）+ 无惧评价（“你看不起我无所谓”）+ 专注目标（“我正在变强”）**

那么，自我认同的前提是什么？自我认同最关键的在于：你先给自己假设一个身份，然后去认同这个身份。

例如，假设你的身份是“健身房里最底层的人”。你可以给自己假设一个像“一拳超人”那样的身份：一开始很弱，但坚信自己会成为强者。

**代码描述（心态转变逻辑）：**
```python
current_identity = “健身房底层弱者”
assumed_identity = “正在蜕变中的未来强者”
if accept(current_identity) and believe(assumed_identity):
    focus_on_action(“努力训练”)
    new_identity = “健身房最上进的人”
```

他认同了自己“正在变强的路上”这个身份，并努力朝这个方向行动。于是，他的身份就从“最菜的人”转变成了“最上进的人”。

这个概念是：你原本有一个身份（例如月入2000）。你给自己假定一个新身份（我应该月入10万），并坚信自己能做到。

你现在月入2000，所以要朝10万的方向努力。于是，你的身份就变成了“一个月入2000但正在向10万努力发展的人”。

你假设并认同了这个点，在实际行动中，你就完全实现了自我认同。这会极大地巩固你的心态。

这样的心态让你能接受更多负面反馈和失败，在面对成功时能戒骄戒躁，并能清晰地看到是哪些关键点促成了你的跨越。

## 总结 📝

本节课中我们一起学习了构建强大心态的四个支柱：

1.  **阅历**：通过不断尝试新事物、打破舒适圈来积累，作为一切心态的底层支撑。
2.  **处理当下**：专注做好眼前力所能及的每一件事，避免好高骛远，脚踏实地进步。
3.  **接纳**：包容所有已发生的正面与负面，达到宠辱不惊的状态，从中汲取成长养分。
4.  **身份与自我认同**：主动设定并坚信一个更高阶的自我身份，以此指导行动，实现心态的巩固与提升。

这些内容可能相对抽象，如果你能理解，请记下这些要点。在实际行动的过程中，再仔细品味其中的深意。
