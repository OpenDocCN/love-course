# Simon2024课程【新升温核心】：9.有效搭讪、自然流、技术流_ev

## 搭讪技巧

**概述**：本节将介绍如何进行有效搭讪，包括建立熟悉感、联系感和好感。

**有效搭讪**：

*   **建立熟悉感、联系感和好感**：通过聊天中的情绪指标（如哈哈哈、表情包）和主动提问，展示对对方的兴趣。
*   **主动询问和夸奖**：通过询问对方工作、夸奖对方外貌等方式，表达兴趣。
*   **约会的可能性**：有效搭讪后，应能轻松约对方见面。

**搭讪类型**：

*   **技术流**：注重话术和话题，关注自己的标准和拉近关系。
    *   **标准**：在搭讪时注重事物，有明确的标准。
    *   **拉近关系**：通过寻找共同点或创造机会，拉近与对方的距离。
*   **自然流**：自然地进行社交，如东北人唠嗑。

## 聊天极简框架

**概述**：本节将介绍聊天极简框架，包括试探、建立联系和展示兴趣。

**聊天极简框架**：

*   **试探**：在搭讪后，试探对方的热度，判断基准线。
*   **建立联系**：通过共同点或兴趣，与对方建立联系。
*   **展示兴趣**：通过表达感受和好奇，展示对对方的兴趣。

**总结**：本节课中，我们学习了如何进行有效搭讪、自然流和技术流搭讪技巧，以及聊天极简框架。希望这些技巧能帮助你在社交场合更加自信和成功。
