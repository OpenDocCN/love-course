# 富山山女性成长课程：11_特性筛选人1

Hello，大家好！今天我们要学习如何通过观察一个人的外在形象，如穿着和整体风格，来快速识别这个人是否是我们想要链接的优秀资源。以下是一篇关于如何通过穿搭识人的教程。

## 概述

在本节课中，我们将学习如何通过观察一个人的穿着和整体风格来了解其人格、性格、审美、喜好、生活环境和社会阶层。

## 穿搭识人

### 1. 正装与休闲装

- **正装**：通常在工作或职场中，男性需要穿着正装以展现威严和严肃的形象。如果一个人不愿意穿正装，而是选择休闲装，那么他可能是一个大气、不羁、洒脱的人，心态年轻，灵活变通。
- **休闲装**：喜欢穿休闲装的人可能比较随性，不太拘泥于条条框框。

### 2. 紧身衣与运动装

- **紧身衣**：喜欢穿紧身衣的人可能比较吹毛求疵，对细节要求严格，对别人要求较高。
- **运动装**：喜欢穿运动装的人可能比较外向，包括女生也是如此。

### 3. 颜色与配饰

- **明亮颜色**：喜欢穿明亮颜色的人可能性格开朗，爱炫。
- **暗沉颜色**：喜欢穿暗沉颜色的人可能性格沉稳，古板。
- **超大配饰**：喜欢戴超大配饰的人可能性格开朗，出手阔绰。
- **小巧配饰**：喜欢戴小巧配饰的人可能心思细腻，对细节要求较高。

### 4. 个性与风格

- **夸张风格**：喜欢夸张风格的人可能性格古灵精怪，爱炫。
- **卡通图案**：喜欢穿卡通图案的人可能比较童趣，不太在意衣着。
- **潮流风格**：喜欢赶潮流的人可能性格活泼，情绪和爱情变化较大。

## 总结

本节课中，我们学习了如何通过观察一个人的穿着和整体风格来了解其人格、性格、审美、喜好、生活环境和社会阶层。希望这些方法能帮助大家在日常生活中更好地识人。
