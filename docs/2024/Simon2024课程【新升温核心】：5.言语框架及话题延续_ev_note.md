# Simon2024课程【新升温核心】：5.言语框架及话题延续_ev

## 概述

在本节课中，我们将学习如何构建有效的言语框架和延续话题，以提升搭讪技巧。

## 言语框架

**什么是好的搭讪？**

搭讪可以分为几个部分，其中之一是言语框架。言语不仅仅是你说的话，还包括你的动作、情绪、肢体语言、表情、语调和离场反应。

**语言是一种符号**

语言是一种符号系统，例如中文就是一种符号。在搭讪中，女生通过你的肢体表情、语言语调、临场反应等感知你的价值。

**肢体语言**

肢体语言也是语言的一部分。例如，教父中的黑帮老大形象就是通过肢体语言展现的。

**放松与自信**

在搭讪时，保持放松和自信非常重要。紧张会让人显得拘谨，而放松则能展现你的自然魅力。

**掌控与确信感**

在搭讪时，可以通过肢体语言和语调展现确信感。例如，当你说“你好，我喜欢你”时，可以加上一些肢体动作，如眼神交流或微笑。

**眼神接触**

在酒吧等环境中，可以通过眼神接触来吸引对方的注意。

## 话题延续

**话题选择**

搭讪时，可以选择一些简单的话题，如年龄、工作、是否单身、来自哪里、口音等。

**话题延续技巧**

1. **联想与观察**：通过联想和观察，将话题延续下去。例如，在谈论年龄时，可以询问对方的兴趣爱好。
2. **深度表达**：每个话题都可以聊很久，关键在于深度表达。例如，在谈论年龄时，可以分享自己的看法和经历。
3. **多表达自己**：在聊天中，多表达自己的观点和感受，让对方了解你。

## 搭讪技巧

**打开话题**

对于新手来说，搭讪时可以采用简单的话题，如“你好，你叫什么名字？”或“你好，这里有什么好玩的？”

**结束语**

本节课中，我们一起学习了如何构建有效的言语框架和延续话题，以提升搭讪技巧。希望这些技巧能帮助你更好地与人交流。
