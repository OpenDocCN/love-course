# 56、小姨子供养者翻盘-强者框架课：008-前言-3_ev_note

在本节课中，我们将了解本套课程的创作初衷、核心价值以及它所涵盖的完整知识体系。这套课程旨在系统性地提升你的聊天能力，从而彻底改变你的社交关系与生活面貌。

---

正是因为我非常清楚聊天在社交中的重要性，所以在这些年做咨询的过程中，我不断总结和提炼大家问题的共性。我将过去的解决方案沉淀成方法论，最终制作成了你眼前的这套课程。

接下来，我也会不遗余力地让更多人学到这套方法。因为我非常清楚一件事：**如果一个人能掌握聊天的底层逻辑，突破思维的桎梏，他们将会开启一个与过去完全不同的新生活。** 他的整个社交关系都将发生改变。

这种体现在生活中的变化之大，可能是你无法想象的。这套课程是一整套系统的聊天训练体系，它主要讲解我们如何变成一个会聊天、懂聊天、能把握好分寸的人，成为一个说话恰到好处、掷地有声、能轻易调动情绪并推进社交关系的人。

以下是本课程内容的核心模块介绍：

*   **聊天前的心态准备与目标明确**：为有效沟通奠定基础。
*   **聊天的8个黄金法则**：构建高质量对话的基本原则。
*   **快速积累谈资的三个维度**：让你在任何场合都有话可说。
*   **让话题变饱满的三层递进法**：深入展开对话，避免尬聊。
*   **从语义沟通转为情绪沟通**：核心公式：**有效沟通 = 内容传递 + 情绪共鸣**。
*   **任何情况都能应对的万能回复思维**：提供应对各类对话的思维模型。
*   **永不枯竭的话题挖掘思维**：像雷达一样持续发现新话题。
*   **从认识到邀约的聊天全流程讲解**：完整覆盖社交关系推进的各个阶段。
*   **通过植入情绪快速推动关系的方法**：加速社交关系的升温。
*   **11个聊天技巧的应用**：即学即用的实战技巧。
