# 49、清风老师全套：10.《情绪自控力》不再发脾气：110.第四课：如何规划休息，摆脱熬夜，让你的

在本节课中，我们将学习如何通过科学规划休息来摆脱熬夜，从而提升自控力与精力水平。我们将探讨休息不足的危害、原因，并介绍一套世界顶级的睡眠方法。

上一节课程中，我们介绍了如何规划饮食与休息以改善自控力状态。本节中，我们来看看如何具体规划休息，摆脱熬夜的困扰。

许多人晚上难以入睡，反复查看手机，导致睡眠时间推迟，进而影响第二天的状态与情绪。

我们渴望的自控力有三个表现。第一，拥有良好的执行力。第二，对自己的行为有良好的控制能力，能做出有远见的决策，考虑未来规划。第三，拥有良好的情绪，待人温和。

睡眠不足会对自控力产生三个致命影响。

以下是睡眠不足的三个主要影响：

1.  **执行力降低**：休息不足导致第二天昏昏沉沉，表现不佳，事后产生后悔情绪。
2.  **促使短期选择**：在休息不足时，人们倾向于做出短期、不理智的决策，例如依赖咖啡因或垃圾食品来提神，形成恶性循环。
3.  **情绪暴躁易怒**：睡眠不足会降低情绪管理能力，使人难以平和地处理问题。

那么，导致我们休息不足的原因有哪些呢？

以下是三个常见原因：

1.  **无法处理焦虑情绪**：被焦虑感绑架，难以放松入睡。在第二课中介绍过的“改变内在对话模式”和“情绪释放法”可用于应对。情绪释放法的本质是心理学中的意向对话疗法，其核心是将情绪具象化后释放。
    > **情绪释放法示例（伪代码）**：
    > ```
    > 1. 识别当前情绪（例如：焦虑）。
    > 2. 想象这种情绪的形状、颜色、质地。
    > 3. 想象将它放入一个容器（如盒子），或看着它飘远、消散。
    > ```
2.  **被手机软件绑架**：睡前手机通知不断，导致无意识地刷手机，拖延入睡时间。
3.  **缺乏合适的睡眠环境**：例如光线刺眼或噪音干扰，影响入睡并容易导致睡眠中断。

针对这些原因，可以尝试以下解决方案。

以下是三个实用建议：

*   **管理手机使用**：睡前将手机放远，或至少开启勿扰模式、关闭通知声音，创造不被打扰的环境。
*   **借助书籍助眠**：在床头放置一本内容较深或篇幅较大的书籍，睡前阅读有助于放松和入睡。
*   **优化睡眠环境**：使用遮光眼罩阻挡光线，并通过白噪音软件（播放海浪、雨声等）来掩盖环境噪音。白噪音的原理是用一个持续性噪音让听觉适应，从而降低其他突发噪音的干扰。

现在，我们对休息不足的原因有了一定了解。接下来，我们深入认识一下睡眠本身。本节课非常超值，因为我们将分享一套由英超曼联和NBA的睡眠教练尼克·利特尔黑尔斯提出的世界顶级睡眠方法。

他将睡眠分为5个阶段，并将其比喻为一个循环过程：从楼梯走上滑梯，滑下后再回到楼梯。

以下是睡眠的五个阶段：

1.  **上楼梯（入睡期）**：刚躺下，意识清醒，易被惊醒。
2.  **在楼梯上（浅睡期）**：进入浅睡眠，较大声响可能唤醒，小声响则不会。
3.  **到楼梯下（深睡期过渡）**：进入更深层的睡眠。
4.  **旋转滑梯（深睡期）**：进入深度睡眠，脑电波放慢，大脑进行修复与休息。
5.  **回到楼梯（快速眼动期）**：容易惊醒，此阶段常伴随梦境。

他提出了 **“90分钟睡眠法”** 。其核心公式是：
**一个完整的睡眠周期 ≈ 90分钟**
**优质睡眠 ≈ 睡满5个周期（即7.5小时）**

该方法建议在第五个周期结束时（即快速眼动期）被唤醒，此时人会感觉精力充沛。

如何应用呢？例如，若你计划晚上12点半入睡，可以设定早上8点的闹钟，这样大约能睡满7.5小时（5个周期）。如果你入睡较晚，比如凌晨2点，则应计划在早上9点半起床，以保证完整的周期。

许多人有过“越睡越累”的经历，这通常是因为在深睡眠阶段（第3、4阶段）被强行唤醒。因此，尽量用90分钟周期来规划睡眠时长。

接着，我将分享提高睡眠质量的五个具体步骤。

以下是五个步骤：

1.  **找到并保持睡眠规律**：尽量固定每天入睡和起床的时间，不规律的作息比单纯晚睡伤害更大。
2.  **建立睡前仪式感**：例如，关闭手机、听白噪音、阅读书籍等一系列固定动作，向大脑发出“准备睡觉”的信号。
3.  **降低干扰因素**：使用眼罩、耳塞等工具，为自己创造黑暗、安静的睡眠环境。
4.  **建立起床规律**：起床后执行一套固定的唤醒动作，例如喝水、准备简单早餐、洗漱等，帮助身体和意识清醒。
5.  **进行起床后的小练习**：例如进行10分钟的呼吸冥想，帮助大脑快速进入清晰、专注的工作状态。
    > **简易呼吸冥想**：坐在椅子上，专注于感受一呼一吸的过程，持续10分钟。
