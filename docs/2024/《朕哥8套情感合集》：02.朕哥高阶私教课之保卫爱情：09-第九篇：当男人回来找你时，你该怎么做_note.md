# 《朕哥8套情感合集》：02.朕哥高阶私教课之保卫爱情：09-第九篇：当男人回来找你时，你该怎么做

🎼当男人回来找你时，你该做什么？大家好，我是郑哥。很多女生都会有一个困惑，就是和男友已经分手了，但是他又回来找我。那这个时候我该怎么办呢？我该重新接受他吗？他这次是真的意识到自己的问题了吗？

还是直接拒绝他更好一些呢？那么今天呀我们就来讲一下，当男人回来找你时，你该怎么做。首先这种时候呀，我们要问自己三个问题。

### 1. 能否继续
- **了解对方本性**：判断对方是否真的改变，本性是否难以改变。
- **分手原因**：分析分手原因，是否已解决。
- **对方动机**：考虑对方回来找你的原因。

### 2. 为何回来
- **未找到新女友**：满足心理和生理需求。
- **与现女友争吵**：寻求安慰。
- **工作或社交压力**：寻求解压。
- **意识到你的重要性**：重新开始。

### 3. 是否真心
- **关注你的动态**：了解他是否真心爱你。
- **行为表现**：观察他的行为，判断他的诚意。

当思考过这三个问题后，我们可以按照以下步骤来做：

### 1. 内心标准
- **设定标准**：确定可以接受对方哪些改变。
- **面对面交谈**：了解对方这段时间的感情变化。

### 2. 观察期
- **普通朋友关系**：避免肢体接触。
- **多次见面**：判断对方的诚意。

### 3. 初步抉择
- **思考一个月**：对对方有一个新的判断。
- **表达真实想法**：根据内心的答案表达自己的想法。

### 4. 行为验证
- **抛出橄榄枝**：给对方机会。
- **侧面暗示**：通过行为观察对方的反应。

### 5. 把握节奏
- **控制约会次数**：避免太容易得到。
- **了解对方**：相处时间越长，默契度越高。

### 6. 延迟满足
- **满足自己的需求**：将对方的需求后置。
- **兑现承诺**：让对方尽快兑现承诺。

### 7. 保留选择权
- **潜意识**：让对方明白你是优势方。
- **制造危机感**：展示你的选择权。

本节课中我们一起学习了当男人回来找你时，你该怎么做。希望对你有所帮助。
