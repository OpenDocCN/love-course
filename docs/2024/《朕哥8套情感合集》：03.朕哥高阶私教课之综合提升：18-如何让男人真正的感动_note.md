# 《朕哥8套情感合集》：03.朕哥高阶私教课之综合提升：18-如何让男人真正的感动

🎼 如何让男人真正感动？大家好，我是震哥。许多咨询的姐妹不知道如何让男人感动，她们认为付出很多，对男人好，男人就会感动。但实际上并非如此。

**核心概念**：理解男人的感情世界，通过日常细节让他感动。

**课程结构**：
1. 男人感情世界是怎样的？
2. 什么行为会让男人感动？
3. 如何不失去自己的地位？

### 1. 男人感情世界是怎样的？

- **外表坚硬，内心柔软**：男人在任何时候都表现出强大，但内心也有柔软的一面，恋爱中会体现出来。
- **不肯主动寻求帮助**：男人在意面子，很少向别人求助，尤其是喜欢的女人。
- **爱你时，一切都是美好的**：男人爱你时，你的缺点也认为是优点，不爱你时，优点也变成缺点。
- **不善于表达，但渴望被理解**：男人的言语表达能力不如女人，渴望被理解。
- **付出不计成本，离开绝不回头**：男人在热恋期会付出很多，但一旦离开，就不会回头。

### 2. 什么行为会让男人感动？

- **在他得意时，为他高兴，为他鼓掌**：让他觉得你懂他，欣赏他。
- **在他失意时，无理由的支持和坚定的陪伴**：让他觉得你是他的真爱。
- **夸他不起眼的小优点，表现出努力，帮他提高进步**：让他觉得你与众不同。
- **适当的作，极致的体贴照顾**：让他更加重视你。
- **在他迷茫失落时，发挥你的聪明才智，帮他一把**：让他明白你的能力。
- **男人遭受不公平时跳出来保护他**：让他觉得你值得依靠。
- **把自己收拾打扮的越来越漂亮**：让他觉得更有面子。

### 3. 如何不失去自己的地位？

- **既能跟得上他的步伐，又只甘心做他的配角**：让他觉得你值得依靠。
- **完全相信他的能力，且努力向他学习**：让他觉得你信任他。
- **用心感受他的情绪和处境**：让他觉得你关心他。

**总结**：通过理解男人的感情世界，掌握合适的方法，你就能让男人真正感动。本节课中我们一起学习了如何让男人感动，希望对你有所帮助。
