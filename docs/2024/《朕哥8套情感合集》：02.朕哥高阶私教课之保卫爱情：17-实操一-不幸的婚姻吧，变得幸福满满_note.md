4. **引导男生付出**：减少亲密，引导男生为自己付出。
5. **制造交集**：一起旅游、参加活动、运动，培养共同兴趣爱好。
6. **提升内在价值**：降低发生关系频率，拒绝男生要求，提升自己在感情中的地位。
