## 概述

把妹的核心本质就是吸引。见面时不推进关系、不进入男女框架、不做肢体接触，原因很简单。这些都是索取的行为，非常容易导致吸引力下降。仔细品味这句话。所以你们看到我们的实战视频时，会感觉整个过程超级平和。我们通常选择在到家之后再推进关系。

## 关系推进受阻的根本原因

之所以你们会遇到很多问题，例如牵手、接吻等关系推进不动，是因为吸引力下降了。

上一节我们介绍了吸引力下降是关系受阻的核心，本节中我们来看看具体哪些行为会导致吸引力下降。

以下是导致吸引力下降的六种常见行为：

1.  你很想“操操”，也就是你很想操作，想进攻。
2.  你想粘着对方。
3.  你给予对方过多的注意力。
4.  没有一致性（言行不一）。
5.  没有推开（不会适时制造距离感）。
6.  一直取悦对方。

当你们遇到所有女生都推进不动的时候，只有一个原因。这个原因就是吸引力不够，或者你在互动中破坏了已有的吸引力。

## 吸引力的两个阶段

理解了破坏吸引力的行为后，我们需要系统地认识吸引力是如何建立和起作用的。吸引力的产生分为两个关键阶段。

### 第一阶段：见面前的筛选

在你们见面之前，对方会通过你踏实的外表和状态，再加上你朋友圈所展示的你是一个怎样的人，来判断是否要见你。这相当于你拥有一份合格的简历。

**公式：见面机会 ≈ 外在价值展示（形象 + 社交展示面）**

例如，你要去Google上班，那你必须有比较强的学历。而这个学历也决定了你能不能获得面试机会。你的外在价值决定了对方会不会跟你见面。当对方觉得你还不错时，通常就到了“可以见一下”这一步。

### 第二阶段：见面后的核心评估

最重要的东西从你们见面的那一刻开始。前面的外在价值就已经没有那么重要了，简历审核阶段已经过了。这时你已经在公司上班了，没有人在意你的履历和学历，而是开始看重你的业务能力。

**核心概念转换：外在价值 → 内在价值**

也就是说，从见面的那一刻开始，外在的价值已经退居次席，内在价值占领了高地。

为了更清晰地说明这一点，我们不得不再用B来举例描述。B的外在价值非常好，但是只要女生见他一面，很快就会把吸引力消耗完。女生会开始磨磨蹭蹭地改变邀约时间，没那么想见他。

B一直理解不了，为什么女生见完我和Elon之后，会不断地给我们发消息，主动提出见面的时间。看懂了吗？你的外在价值只决定了你够不够资格，而你的内在价值决定了对方喜不喜欢你。这个“喜不喜欢”，就是吸引。

## 总结与行动指南

本节课中，我们一起学习了吸引力的核心本质和两个关键阶段。

首先，我们明确了所有关系推进困难的根本原因只有一个：**吸引力不足或遭到破坏**。我们列举了六种常见的“掉吸引”行为，如过度需求、粘人、过分关注等。

其次，我们剖析了吸引力的两个阶段：**见面前依靠外在价值（形象、展示面）获得机会**；**见面后则完全依靠内在价值（气场、互动模式、人格魅力）产生真正的吸引**。两者关系可以概括为：**外在价值是门票，内在价值是演出本身**。

因此，如果你们遇到任何瓶颈和问题，应该思考的是：你的吸引力建设是否做得不够，或者你是否在互动中做出了破坏吸引力的行为。始终围绕“创造和维持吸引”这一核心来展开你的所有互动。
