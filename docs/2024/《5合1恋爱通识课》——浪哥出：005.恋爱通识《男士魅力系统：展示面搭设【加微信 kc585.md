# 课程01：Python基础入门 🐍

在本节课中，我们将要学习Python编程语言的基础知识。我们将从最核心的概念——变量开始，了解如何存储和操作数据，然后学习如何使用`print`函数在屏幕上显示信息。这些是编写任何Python程序的第一步。

## 什么是变量？ 📦

变量是计算机内存中用于存储数据的“盒子”或“标签”。你可以把变量想象成一个贴有名字的储物柜，里面可以存放任何东西，比如数字、文字或更复杂的数据。

在Python中，创建一个变量非常简单，只需要给它起一个名字，然后用等号`=`给它赋值。

**核心概念**：变量赋值
```python
my_variable = 10
```
这行代码创建了一个名为`my_variable`的变量，并将数字`10`存储在其中。

变量名可以包含字母、数字和下划线，但不能以数字开头。Python是区分大小写的，所以`age`和`Age`是两个不同的变量。

## 使用`print`函数 📤

上一节我们介绍了变量，它用于存储数据。本节中我们来看看如何将这些数据显示出来。`print`函数是Python中最基本的输出工具，它可以将括号内的任何内容显示在屏幕上。

**核心概念**：`print`函数
```python
print("Hello, World!")
```
这行代码会在屏幕上输出文本：`Hello, World!`。

`print`函数不仅可以打印文字，还可以打印变量的值。以下是`print`函数的一些常见用法。

### `print`函数的常见用法

以下是`print`函数的一些基本用法示例：

*   **打印字符串**：直接输出一段文本。
    ```python
    print("欢迎学习Python！")
    ```

*   **打印变量**：输出变量中存储的值。
    ```python
    name = "小明"
    print(name)  # 输出：小明
    ```

*   **打印多个项目**：用逗号分隔，可以一次打印多个内容，它们会以空格连接。
    ```python
    age = 20
    print(name, "的年龄是", age, "岁。")
    # 输出：小明 的年龄是 20 岁。
    ```

*   **进行简单计算**：`print`可以直接输出表达式的结果。
    ```python
    print(5 + 3 * 2)  # 输出：11
    ```

## 让我们动手实践 ✍️

理论需要结合实践。现在，让我们将学到的变量和`print`函数结合起来，完成一个简单的任务。

假设我们要编写一个程序，存储你的名字和年龄，然后打印一句自我介绍。

**示例代码**：
```python
# 创建两个变量来存储信息
my_name = "张三"
my_age = 25

# 使用print函数输出自我介绍
print("大家好，我叫", my_name)
print("我今年", my_age, "岁了。")
```

当你运行这段代码时，屏幕上会显示：
```
大家好，我叫 张三
我今年 25 岁了。
```

你可以尝试将`my_name`和`my_age`的值改成你自己的信息，然后再次运行程序。
