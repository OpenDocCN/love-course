# 林老头《私人传承》：002.吸引的逻辑（建立吸引攻略）

大家好，我是老林。今天我们来探讨如何建立吸引攻略。

## 一、侧面分析

**核心概念**：女生通过侧面分析了解一个男人，而非直接听他说什么或看他的直接表现。

**步骤**：

1. **了解女生期望**：女生通常期望优秀男人是低调、谦虚、不浮夸、简约，生活自由。
2. **展示生活细节**：通过朋友圈、聊天细节展示你的生活状态，让女生侧面分析你。
3. **模仿优秀男人**：模仿优秀男人的行为，如衣品、生活方式等，与优秀男人产生链接。

## 二、放缓节奏

**核心概念**：优秀男人不急于得到女生，而是慢慢推进关系。

**步骤**：

1. **不急于追求**：不要急于追求女生，保持冷静和自信。
2. **展示无目的性**：在聊天中保持热情，但不过度主动，让女生觉得你无目的性。

## 三、热情，但不主动

**核心概念**：在聊天中保持热情，但不过度主动，让女生觉得你无目的性。

**步骤**：

1. **积极迎合**：在聊天中积极迎合女生，创造话题。
2. **无目的性**：不要过度追求女生，让她觉得你无目的性。

## 四、短期与长期策略

**短期策略**：

1. **多约会**：多与女生约会，增进感情。
2. **暧昧**：在约会中保持暧昧，让关系升温。
3. **表达好感**：在适当的时候表达你的好感。

**长期策略**：

1. **持续吸引**：通过侧面分析、放缓节奏、热情但不主动等方式持续吸引女生。
2. **建立信任**：与女生建立信任，让她觉得你是一个可靠的人。

## 总结

本节课中我们一起学习了如何建立吸引攻略。通过侧面分析、放缓节奏、热情但不主动等方法，我们可以有效地吸引女生，并建立稳定的关系。希望这套逻辑能帮助你更好地追求心仪的女生。
