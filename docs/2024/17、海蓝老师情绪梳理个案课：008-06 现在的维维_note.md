# 海蓝老师情绪梳理个案课：008-06 现在的维维 🧘‍♀️

## 概述
在本节课中，我们将跟随海蓝老师一起，深入了解个案维维的情绪状态，并学习如何通过情绪梳理来帮助她改善睡眠和生活质量。

## 维维的现状
以下是维维目前的情况：

- **失眠**：晚上经常失眠，下班后也感到焦虑，晚上会做噩梦。
- **疲惫**：上班时感到特别疲惫，需要检查两遍工作记录。
- **担忧**：喝水时担心病人出现意外，害怕出错。

## 情绪梳理过程

### 1. 觉察情绪
- **感受**：心疼、嫌弃。
- **行动**：靠近、拉手。

### 2. 深入了解
- **回忆**：维维回忆起初三时失眠的经历，感到害怕和焦虑。
- **反思**：认识到睡不好并不是生命的全部，生活依然可以继续。

### 3. 改变认知
- **接受**：接受自己现在的状态，并相信会慢慢变好。
- **寻找出路**：了解睡眠的两种方式，并尝试静观来改善睡眠。

### 4. 聚焦问题
- **工作焦虑**：担心辐射、病毒、细菌，以及昼夜颠倒对身体的影响。
- **睡眠焦虑**：担心失眠导致身体出现问题。

### 5. 解决方案
- **调整工作**：考虑换一份工作，减少工作压力。
- **改善睡眠**：尝试静观和自然睡眠，提高睡眠质量。

## 总结
本节课中，我们一起学习了如何通过情绪梳理来帮助维维改善睡眠和生活质量。通过深入了解她的情绪状态，我们找到了问题的根源，并提出了相应的解决方案。希望维维能够早日摆脱失眠的困扰，过上幸福的生活。
