# 凡哥私教课程：长期关系经营课：005.05：关于长期关系（一）

🎼随着爱情脚本从缘分婚姻转变为爱情婚姻，处理你我我们的关系成为亲密关系中核心话题。在初婚关系中，恋人寻找共鸣。找到越多一致之处，两人距离越近，爱情感觉越强烈。而在长期关系中，角色互补性尤为重要。差异大的两人可能配合得更好，但差异也导致双方在许多事情上有不同想法和解决方案，容易产生冲突。因此，处理好差异和冲突成为维护长期关系的核心能力。甚至可以说，双方处理冲突的方式，是否能磨合好，决定关系能否长久。

**核心概念**：长期关系中，处理差异和冲突的能力是维护关系的核心。

🎼初婚关系转变为长期关系主要取决于两人在一起的时间和性格。两性关系的重点是相似性。研究表明，18个月后，互补性变得更重要。在互补过程中，我们要重新认识差异，享受差异带来的好处，让处理差异的能力成为处理长期关系的核心能力。

🎼当然，这个时间长度不是必然的。两个天天黏在一起的恋人和身处异地的恋人，情况不同，实践中不存在明确的节点。让我们感受到我们从初期关系走到长期关系。

🎼当爱情持续一段时间后，双方会发现早期的激情不再存在。这时，需要意识到处理关系的方式和原则可能需要改变。

**核心概念**：长期关系中，处理关系的方式和原则可能需要随着时间改变。

🎼很多人会忽略长期关系中爱的感受这种体验的重要性，也没有重视初期关系和长期关系的不同。在面对冲突原因时，容易简单归结为性格不合。这背后其实存在很多误区。

**误区一**：用初期关系的逻辑处理长期关系。

🎼爱情最早开始时，激情发挥重要作用，一见钟情，一日不见如隔三秋。生物学上用多巴胺解释这种激情，但多巴胺有一个重要特征，即对预期外的东西产生强烈反应。一旦体验变成预期内，多巴胺释放变弱。

🎼那么，从生物学上来讲，爱情就必然是短暂的吗？《他婪了多巴胺》告诉我们并非如此。激情之爱也可能转化为更持久的东西，成为陪伴之爱，虽然可能不像多巴胺那样让人兴奋，但基于当下神经递质，如催产素、血管加压素和脑内啡的作用，会为你提供长期的幸福感。

🎼感情就像流水，既想要河水很深，又想要它奔流不息，澎湃汹涌，就需要消耗很多能量。所以激情之爱会消耗大量能量，期望值不断加大，直到无法满足。在长期关系中，除了激情，还需要安全感和确定性。

🎼心理学家艾斯特佩瑞尔在《亲密陷阱》中发现，一方面，我们希望爱情能提供一种超然的激情，让我们有超越常规生活的体验；另一方面，我们又希望爱侣是个稳定可靠的依靠，能为我们提供安全感和稳定性。这两种需求本身是冲突的。

🎼安东尼罗宾斯说，激情和不确定性是正相关的。关系越不确定，有时我们就越有激情。这也解释了为何廊乔一梦这个婚外情故事会成为经典。因为人们常常是享受舒适的同时，又渴望激情；在享受激情的同时，又需要安全感。

🎼爱情刚开始时，竞争对手争夺爱人会激发爱的斗志，让彼此更珍惜在一起的机会。但在长期关系中，出现竞争者会损害两人之间的信任感和安全感，反而破坏爱情。

🎼那么，两人之间如何既保持激情，又有相对的安全感和信任感呢？大量研究为我们提供了一系列解决方案。

**解决方案一**：用发展的眼光看爱情。

🎼爱情是慢慢走向成长治愈性的，它在不断变化，成长本身就带有不确定性，能提高新鲜感。虽然我还是过去的我，但我想不断进步，不断改变。你看这不就有新鲜感了吗？在追求成长的过程中，我们也接受了这种不确定性，减弱了不确定带来的恐慌，与对安全感的追求之间的对立，这就是一个很好的认知改变。

🎼在长期关系中，除了爱情，还有亲情、事业等很多事情。一个人怎么可能一辈子都在谈恋爱呢？爱情很重要，但绝不是生活的全部。工作忙碌时，每天只需拿出30分钟，集中能量增加爱情的体验，让激情之爱时不时的再现。这种方法也能很好地平衡我们对激情与稳定的需求。

**解决方案二**：创造仪式感。

🎼仪式感是指郑重其事地做一些非必要、非功能性的事。比如玩游戏，游戏的定义是按照一定规则完成一个非必要的目标。比如俄罗斯方块的目标是把不同的形状堆起来，但我们为什么要做这件事呢？它的意义在哪里呢？其实是没有的，但它就是让我们很快乐。仪式感也是这样。

🎼仪式感可以是日常的特殊约定，也可以是特殊日子的惊喜。比如两个人坚持一定要一起吃早餐，坚持多久一定要一起去看一场电影，这就是日常的特殊约定。而每年一次结婚纪念日的互赠礼物，或者每年两天两个人的旅行，就是特殊日子的惊喜。

🎼这些事情在日常生活中都是非必要的。你不看电影又怎样呢？早餐分开吃又怎样呢？不旅游也可以啊。很多人的人生不都是这么过的嘛。但如果你想要维持爱情的体验，就要郑重其事地去做这些事。

🎼这才是仪式感。因为这些事情会不断地再现爱的承诺，帮助我们不断确定和体验爱情关系，让我们感受到我们是一个整体，感受到背后依然存在的激情。

🎼为什么要郑重其事呢？因为虽然是做一些非必要的事情，但彼此交换的却是各自的稀缺资源。一个日常活动如果没有涉及稀缺资源的交换，那往往很难有仪式感。

🎼爱情小说里的霸道总裁如何体现对女主角的爱呢？他会拿出一天的时间陪女主角做旋转木马，女主角和观众都会很感动。而一个穷小子如何表达爱呢？常常是把他的一个月工资，甚至是一年的收入结余，买一条昂贵的项链送给女主角。

🎼如果反过来，霸道总裁送一条项链或穷小子陪女主角玩一天，我们都不会感动。因为两个人的稀缺资源是不一样的。霸道总裁缺的是时间，穷小子缺的是金钱，只有把稀缺资源拿出来才能感动我们。

🎼礼物的意义也在于此。而我们不仅需要拿出稀缺资源，创造仪式感，还需要想象力。可想象力不够怎么办呢？没关系，我这里提供三个技巧给你。

**技巧一**：模仿电影或电视剧的桥段。

🎼比如，有一年我和我的爱人在恋爱纪念日的时候，正好看了一部电影，是有关于情书的，里面有个场景是这样的。男女主角认识了几年，男主角每年一发生重要的事情就会写一封信，积累起来，最后见到女主角的时候，把这些信交给她，并且说道：“你不在我身边的这些日子里，我把每一年的思念都写了下来。”

🎼我跟我爱人天天在一起，那当然不能照搬这个了，但他给了我启发。我每年都会一起经历一些重要的事情，很有价值。所以呢，我就买了很多明信片，从我们认识的那一年开始，把每一年一起经历的最重要的事情记录下来。

🎼所以看爱情剧其实也是有用的。偶尔抄个作业稍微改变一下，男女主角的浪漫就很有可能变成了你我的浪漫。

**技巧二**：注意积累生活中的浪漫灵感。

🎼商业社会给我们提供了很多灵感。比如你可以去网上搜索情侣一定要做的100件事情，里面就有一些可以成为你仪式感的素材。你也可以关注一些商家，看他们推出的活动，比如主管晚餐、海边散步等，都可以成为创造仪式感的内容。

🎼又比如说在5月20号这天转520块钱给他，这真的是非必要行为。明知是商业炒作，可女孩子呢还是会很高兴。不要嫌弃仪式感很俗，要看对方吃不吃这套，如果对方吃，那么再俗都有用。

🎼当然了，背后有没有真心，也是能互相感受得到的。只要真心到位，就不怕套路。但是呢，如果认为520这天必须转钱，不转就是不爱，那就马上脱离了仪式感的概念，变成功能性作用了。

**技巧三**：根据家庭特色做一些可以不断重复的设计，时间会给浪漫加持。

🎼比如我们家就特别喜欢每年全家人一起去穿亲子装去拍全家福，看着孩子成长，看着家庭一点点变化，这让我们特别的快乐。这些可以重复的日常行为一旦持续数年，其意义就会显现出来。

🎼所以说日子要想过得有滋有味，靠的不是日复一日的琐碎，而是这些闪亮的瞬间。当然，它的频率不能太高，如果频率太高了，那么大家会很累，保持一定的节奏就好。

**总结**：本节课中，我们学习了长期关系中的差异和冲突处理方法，以及如何创造仪式感来维持爱情体验。希望这些内容能帮助你更好地经营长期关系。
