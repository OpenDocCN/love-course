# 清风老师合集：13做自己的情感：情感咨询师培育计：164.第四课：如何持续学习_n

在本节课中，我们将要学习情感咨询师如何进行有效的自我提升与持续学习。我们将探讨学习的本质、核心步骤，以及如何将学习转化为解决实际问题的能力。

上一节我们介绍了咨询师培育计划的背景，本节中我们来看看咨询师自我提升的具体方法。

作为一名情感咨询师，我已经从业5年。在这5年时间里，我一直在实时地为自己复盘和回顾。今天上午，我查阅了自己从2016年到2017年8月的所有咨询记录和档案。我发现自己的来访者当中存在一些特征和现象，这让我对这个工作有了新的认知。这是我作为一个咨询师在自我提升的一个过程。

自我提升往往包含三个核心步骤。

以下是这三个步骤的详细说明：
*   **输入**：学习某一样东西。就像现在老师给大家讲课是给你输入。
*   **整合与消化**：当你接收信息之后，就要进行整合和消化。
*   **输出**：整合消化完了之后，就要输出一些东西。例如给你的小伙伴做咨询，或者跟老师交流，在群里面跟大家分享。

完成了这一遍之后，还需要加上一个反馈，再进入下一个循环。

反馈之后，我们就能够从复盘过程中收获巨大的收益。在思考和总结的过程中，我们会发现：第一，过去自己哪里做得不好；第二，自己有什么地方可以做得更好；第三，我们还应该往什么样的方向去努力。实时的复盘与反馈，是一个经验之谈，也是在我的自我提升过程中非常重要的事情。

那为什么要强调反馈呢？因为反馈就像是《西游记》里的定海神针。它是一个支点，一个锚点，是在信息流的过程中，我们可以跳出简单信息冲刷的一个点。

为什么这么说？因为我们已经处在一个社会快速发展、信息爆炸的时代。例如现在你正在听课，等会儿手机微信弹出一条消息，你就可能把注意力转移到微信上。你去看看线下的大课，或者我们参与的会议过程中，有多少人是这样的？我可以说有80%以上的人都会被外界的信息吸引注意力，然后不专注于当下这件事。

前段时间我一直在学习正念疗法，也跟大家分享过。正念的意思不是一个积极快乐的“正能量”念头，而是**专注当下**。为什么专注当下？因为专注当下可以让我们获得注意力上的集中。只有专注当下，我们才能更好地去解决问题，找到问题的本质，然后把问题突破掉。

咨询师也是如此。咨询师最重要的能力不是“咨询”，你可能会听着很奇怪。咨询师最重要的能力是**学习**。在我的线下课程中，我给同学们说过，咨询师99%是学习，0.5%是咨询，0.5%是输出。

咨询为什么没有学习重要？因为我们所经历的时代，是一个极具快速变化的时代。有可能你在3年前进大学时学到的知识，到3年后出大学时已经没有用了，已经被市场淘汰了。那么这种情况下，我们该怎么办？我们只有练就一生不断自我提升、不断学习的本领，才能让我们在这个时代，作为一名优秀的咨询师和自由职业者，创造自己的价值。

所以在线下课程中，我给同学们讲了咨询师的“三板斧”。

以下是这三项核心能力：
*   **心理学**
*   **营销学**
*   **如何学**

今天，就给线上的各位同学分享一下“如何学”这个内容。

在我们上初中、高中、小学的过程中，所有的老师都只会强调一件事情：学习很重要，但却忘了告诉我们该怎么样学习。学校里的方法往往是上课听讲，下课做作业，但是这样的效果好吗？并不好。甚至很多时候我们非常讨厌做这些作业，但在义务教育中，我们没有办法去改变。

学习本身是一个**解决问题的过程**。它不是一个让我们不停制造麻烦的过程，相反，学习是为了帮助我们解决麻烦。

听到这个定义，我相信大家就会很疑惑了。例如，我觉得有一些东西我感兴趣，我也想去学，比如烘焙、插花、学高尔夫球或射箭。这叫兴趣，这不叫学习。兴趣往往用来放松我们自己的状态，让我们能有一个更好、更享受的状态。它有点像看电影，有点像听音乐。有的人说，我看电影就学会了口语。那我请问各位同学，你看电影学会了英文口语吗？学会了怎么样跟外国人交流吗？想必很难。因为你所处的那个状态是放松的状态，而学习是一个解决问题的状态。

解决问题的状态放松吗？肯定不是的，但它也不一定是非常痛苦的。因为解决完了这个问题，会有一种快感，一种极大的成就感。就像玩游戏一样，我们接到一个任务，然后把这个任务完成，获得经验、奖励和技能提升，都会让我们有一种快感。学习本身也是如此。

所以说“学习让我快乐”这件事是不错的，但是它的快乐在于**结果**，而非在于过程。不是“快乐的学习”就能够“学习让我快乐”，这是两个概念。“快乐的学习”是没有意义的，因为你放松了，在学习过程中没有去打磨自己的技能、去解决问题，是学不到有意义的知识的。而“学习使我快乐”是因为你用学习、用解决问题的思维把问题解决了，获得了巨大的成就感。在这个过程中，你找到了自己的价值，所以你感到了快乐。

那作为一名咨询师，大家最常遇到的问题是什么？是“老师，这个来访者提的这个问题，我不知道该怎么办”。现在，你可以通过学习的方式去解决。

以下是针对不同问题的学习方向：
*   **当心理学咨询技术缺乏时**：来找老师，老师给你推荐书籍。
*   **当想在朋友圈推广自己做情感咨询，不知如何包装自己时**：营销学的知识，来找老师，老师告诉你该怎么样进行自我定位和包装，然后把我们推广出去。
*   **当在学习过程中感到困惑时**：也可以来找老师，分享你具体遇到的困惑是什么，以及你希望得到什么样的帮助。

只有我们带着**解决问题的思维**去学习，才能够在学习以后收获到有效的成果，而这个成果也让我们感到快乐。

这是一个非常焦虑、非常孤独的时代。我们已经处在一个陌生人的社会当中。所以，成为一名情感咨询师，我希望各位同学都是一个**终身的学习者**。只有我们保持终身学习的态度，才能够在学习过程中不断实现自我超越、自我进步，然后在这个时代拥有一个全新的自我。

本节课中我们一起学习了持续学习的核心方法。我们明确了学习是解决问题的过程，其关键在于“输入-整合-输出-反馈”的循环，并需要以专注当下和解决问题的思维为导向。希望各位能成为终身学习者，在实践中不断自我提升。
