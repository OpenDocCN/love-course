# 课程03：沟通课 丨 成为说话高手 🗣️
## 第23节：如何循序渐进展开话题 🎯

在本节课中，我们将学习如何通过四个步骤，循序渐进地展开一个话题，使沟通更顺畅、更有效。

你是否遇到过这样的情况？由于性格太直，经常被别人说说话太直。或者想向别人提出建议，却不知如何开口，担心直接说会显得唐突，但不进入主题又无法推进事情。其实这只是缺乏沟通技巧，把谈话变得过于直接或严肃。这时就需要我们循序渐进地沟通。

本期的逻辑框架是教大家通过四个步骤，循序渐进地展开一个话题。

### 四个核心步骤

以下是展开话题的四个步骤：

1.  **轻松开场**：先不聊主题，但慢慢将话题引向主题，用一个轻松的开场开启谈话。
2.  **步入主题**：明确地和对方聊到今天的主题是什么。
3.  **提出建议**：在主题上，提出你的看法和建议。
4.  **积极收尾**：给这段谈话画上一个完满的句号。

接下来，我们通过例子来看如何应用这四步。

### 示例应用：安排挑战性任务

假设你想让团队里的一名下属接手一个有挑战的项目。你知道这个项目适合他，也能锻炼他，但同时他手头已有多个项目。你会如何沟通让他接受任务？

直接说“小朱，这个任务交给你了”太过直接，不合适。这时可以用一个轻松的话题开场，让对方放下戒备。当然，轻松话题最好与主题有一定相关性，否则会很唐突。

**第一步：轻松开场**
可以这样说：“小朱，你上次的发言稿太棒了，成功化解了公司的信任危机，让大家对你刮目相看。你是怎么想到从那个角度切入的？”

通过这个话题与小朱聊完后，就可以逐渐进入主题。

**第二步：步入主题**
例如：“其实在公司发展，最好的锻炼就是时不时尝试有挑战性、能发挥潜能的事情。我记得我入职第二年，本是纯技术岗位，结果加入一个售前项目，需要大量与客户沟通，这是我以前没做过的。没想到它帮我发现了新的可能性，做到了现在的位置。”

这样，话题就逐渐转到了要聊的主轨道上。

**第三步：提出建议**
接着可以说：“现在我们有一个新项目，背景是这样的……（交代项目）。我觉得你是非常适合的人选，因为你的表达能力和文案能力可以在这个项目中得到充分展现。当然，这个项目也有一定挑战，你可以考虑一下有没有兴趣接手。”

这样就点出了重点建议。

**第四步：积极收尾**
最后鼓励对方，或表示对对方决定的支持与理解。例如：“我其实很看好这个项目，也相信你能在这个项目中做出漂亮成绩。” 或者：“不管你愿不愿意接受，我都希望你能把自己的潜能在公司里发挥出来。”

这样，我们就通过四个步骤逐渐展开了重点话题。

### 另一个示例：推荐工作机会

假设朋友小朱非常适合你另一位朋友老刘刚在外地开的公司。你想帮忙问问小朱是否愿意跳槽，但觉得直接说不太合适，可以通过循序渐进的方式展开谈话。

**第一步：轻松开场**
聊一些轻松话题，例如：“你最近怎么样？工作忙不忙？” 或者：“听说你们行业最近有些动态，你们公司受影响了吗？” 再或者：“我们过去组织了几次球赛，你后来忙什么呢？” 总而言之，聊一些表示关心或想了解他近况的话题，让双方放松并同步最近状况。

**第二步：步入主题**
可以聊你对现在各行业的看法，然后提到你朋友公司所在的行业（假设是新媒体）。例如：“这个行业最近发展确实不错。我朋友开了家公司，最近增长和盈利性都很好，公司也在迅速发展壮大。” 你也可以基于小朱聊的内容进一步探讨，比如如果小朱说他最近在忙与新媒体相关的事，你可以问：“为什么你这个事做得这么好？” 或“你跟新媒体相关的这些事到底做了些什么？” 第二步的目的是把对方拉到你想要谈的话题上，让他觉得谈话与他的特点、能力相关。

**第三步：提出建议**
如果小朱过往的工作经历或专长是内容策划，这时可以说：“小朱，因为我朋友的公司发展非常快，很有前景，而且我知道你在内容策划这块很有经验，所以想看看你有没有兴趣去他那里试一试。虽然是在异地，但现在这个行业的发展带来的机会很难得，你可以考虑一下。” 这里就明确提出了你的看法和想法。

**第四步：积极收尾**
例如：“如果你愿意的话，我约我朋友，你们两个一块聊一聊，看看能不能碰出什么火花。” 或者：“你不愿意也可以告诉我，没关系。总而言之，我希望你知道有这样一个机会，因为你身上的能力和闪光点确实值得更大的舞台。”

这样，我们就可以比较缓和地进入主题，让对方明确知道你的意思，同时无论结果如何，对话都有一个积极的收尾。

### 课程总结 🎓

本节课中，我们一起学习了如何通过四个步骤循序渐进地展开话题。

最后，我们再来总结一下这四个步骤。首先，我们通过一个轻松的话题跟对方先聊起来，当然不是漫无目的地乱聊，而是慢慢转到我们要聊的第二步，也就是主题上来。在主题中，我们点出对这个事情的看法，加上对对方的建议，这是第三步。最后，我们再进行一个积极的收尾，让这个谈话是一个圆满的、带着理解与支持的收尾。

**本节课的练习**
想象一个你正在面临或过去面临的、想和对方说但又觉得不太适合直接说的问题。对象可以是任何朋友、家人或自己的另一半。然后，应用课程学过的四个步骤，设计一段沟通。
