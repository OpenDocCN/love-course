# 15、关键认知：关键关系训练营（学浪）：1-19节拒绝“抱怨”看向长期

大家好，欢迎来到关键关系第十九讲。这一讲，我将分享一个亲身经历，以及作为关键关系的被围系方所经历的一些事情。

**核心概念**：关键关系构建中，避免抱怨，看向长期利益。

### 1. 抱怨的本质

当你想和一个人构建关键关系时，记住，从那一刻开始，对他不要有任何抱怨。抱怨是一种情绪化、非理性的情绪宣泄，通常是对预期未达成的情绪反应。

**公式**：抱怨 = 预期未达成 + 情绪化反应

### 2. 抱怨的负面影响

在交易过程中，我们可以通过情感和精神手段逼迫对方满足自己的要求。但在长期关系中，逼迫对方是没有意义的。长期关系应该是基于自愿和主动的行为。

**代码**：长期关系 = 自愿 + 主动

### 3. 利益共同体

在关键关系中，双方应视为利益共同体。你寻求帮助，是因为你们是利益共同体。只要你好，也能让他分享到你的好处。

**公式**：利益共同体 = 互相帮助 + 共享利益

### 4. 强者心态

在向上社交过程中，强者不会认为自己比对方低，而是认为在更高维度上是平等的。他们相信，现在需要对方，将来也有需要对方的时候。

**代码**：强者心态 = 平等 + 互相需要

### 5. 年龄差

关键关系的年龄差大约在15到25岁左右。在这个年龄差内，对方不会存在自己的后代与你竞争。

**公式**：关键关系年龄差 = 15-25岁

### 6. 抱怨的后果

抱怨会破坏关系，甚至可能导致关系断裂。如果你没有得到对方的帮助，不要抱怨。因为抱怨对你构建这个关系没有任何益处。

**代码**：抱怨 = 关系断裂

### 总结

在本节课中，我们学习了在关键关系构建中，避免抱怨，看向长期利益的重要性。希望这些内容能帮助你更好地理解和构建关键关系。
