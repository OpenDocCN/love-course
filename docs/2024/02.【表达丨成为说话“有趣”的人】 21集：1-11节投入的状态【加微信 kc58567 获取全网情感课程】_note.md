# 课程02：成为说话“有趣”的人 🎤
## 第11节：投入的状态 🎭

在本节课中，我们将要学习一种让说话变得有趣的隐性方法——**投入的状态**。这种方法虽然不易察觉，却是许多有趣表达者的共同特质。

上一节我们探讨了表达的基础，本节中我们来看看如何通过“状态”为话语注入生命力。

### 一、状态差异的对比

我们通过两个对比案例来放大差异，请观察哪一种讲述更有趣。

**案例一：讲述被老板批评的经历**

*   **讲述者A**：“那天老板骂我说我笨，我根本没往心里去。”
*   **讲述者B**：“那天老板骂我：‘你怎么这么笨啊！’我心里其实是……（耸肩、轻松一笑）其实没往心里去。”

**案例二：讲述餐厅点餐的经历**

*   **讲述者A**：“我那天去餐厅吃饭，点了一个三文鱼，服务员说没有。我在那想怎么办？结果服务员接着说还有一个推荐，结果我就尝了尝。”
*   **讲述者B**：“我那天去餐厅吃饭，心里想：‘今天吃点啥呢？吃个三文鱼吧！’然后我就跟服务员说：‘嗨，来个三文鱼。’结果服务员（模仿服务员抱歉的表情和语气）：‘哎呀先生，三文鱼没有了。’哎，我正在想：‘这个怎么办呀？要不要走呢？’这个时候服务员马上接着说：‘先生，我们这里还有一款鱼更好吃，推荐叫深海石斑，今天刚到货，肉质鲜美，这个季节吃味道最好，您算来得是时候了！’我想想说：‘那好啊！’”

通过对比，我们发现第二个人的讲述明显更有趣。两者的核心内容相似，区别在于**说话时的状态与口吻**。

### 二、核心原理：代入感创造趣味

为什么投入状态会让表达更有趣？

试想，你是更愿意听朋友转述《西游记》的故事，还是更愿意穿越到孙悟空与猪八戒身边，亲眼目睹他们的一言一行？答案显然是后者。

因为人们都喜欢更真实、更鲜活、更有**代入感**的表达。表达者越投入，就越能还原真实的场景、人物情绪与细节，听众就越容易被吸引。这就像歌手或演员，他们越投入，我们就越相信他们所演绎的角色。

**核心公式：**
**投入的状态 → 增强代入感 → 还原真实细节 → 表达变得有趣**

### 三、经典案例：田老师的讲述

我小学时在交响乐团学单簧管。一次排练《梁山伯与祝英台》前，指挥田老师并未平铺直叙地介绍剧情。

他把自己完全投入故事中：一会儿化身梁山伯，一会儿化身为祝英台。讲到“十八相送”时，他用略带羞涩的女声口吻说：“梁兄……祝英台若是女红妆，梁兄愿不愿……”；讲到祝英台哭坟时，他带着哭腔呐喊：“没有你，叫我如何面对这一世的孤寂！”

他的讲述超越了词语本身，带来了全方位的冲击，将我们所有人紧紧拉入了剧情之中。这就是表达状态的力量。

### 四、如何练习：掌握投入的状态

以下是两组核心练习，旨在帮助你在表达时能快速、自如地调动饱满情绪，贴近真实人物状态。平时可多做练习，也可在重要发言前进行热身。

**第一组：打开情绪开关**

这个练习的目的是激活你的情绪表达范围。请跟随练习，确保情绪到位。

1.  **起床打哈欠**：`“Oh... Yeah... 啊...”`
2.  **大笑**：`“哈哈...哈哈哈...哈哈哈哈！”`
3.  **大哭**：`“呜... 啊...”`
4.  **发怒**：`“气死我了！气死我了！”`
5.  **失望**：`“唉...完了。”`
6.  **窃喜**：`“嘿嘿，老板今天给我加了个鸡腿。”`

**第二组：人物角色扮演**

这个练习的目的是让你体验并模仿不同角色的说话方式。请尝试演绎以下角色：

*   **老人说话**：“你这年轻人不懂规矩，吃那么大西瓜都不给我尝一口。”
*   **小孩说话**：“奶奶，你说为什么有时候天空是红色的呢？”
*   **发飙的人说话**：“你什么情况？那么大西瓜自己吃了啊！”
*   **温柔的人说话**：“那么大西瓜，你怎么就舍得自己独享啊？”

你也可以模仿任何你喜欢的电影、喜剧或小品中的人物。模仿能极大地帮助你丰富表达时的状态、语气和神态。

### 五、本节总结与课后练习

本节课中，我们一起学习了让说话变得有趣的关键：**投入的状态**。通过**代入角色、还原细节、饱满情绪**，你的表达将变得鲜活、生动，充满吸引力。

**课后小练习：**
请给身边的人讲述一件你最近亲眼目睹的事情。**关键要求是**：不能以第三人称旁观的方式讲述，而必须使用当事人（即你看到的那个主角）的语气、神态和动作来演绎。在开始演绎前，不妨先做一遍上面的情绪与角色扮演练习。
