在本节课中，我们将要学习如何通过观察孩子的日常行为，快速识别其天生的性格色彩。我们将通过生动的案例，理解不同性格色彩的孩子在相同情境下的不同表现，并探讨性格形成的先天因素。

## 性格与个性的定义

上一节我们介绍了性格色彩的基本框架，本节中我们来看看性格与个性的核心区别。在FPA性格色彩学中，有两个重要的概念：**性格**与**个性**。

*   **性格**：指的是“原来的你”，即与生俱来的、天生的部分。
*   **个性**：指的是“现在的你”，即经过后天环境、教育、经历等影响后形成的部分。

因此，关于性格是先天还是后天的问题，在性格色彩学中，**性格**几乎无一例外是先天的。

## 从婴儿时期看性格差异

性格的差异在个体很小的时候就已经显现。以下是不同性格婴儿在需求表达上的典型区别：

*   **绿色性格婴儿**：即使过了吃奶时间感到饥饿，哭声也较小，表现为“嗯嗯嗯”的低吟。
*   **黄色性格婴儿**：即使刚吃完奶，也会用响亮、持久且极具穿透力的哭声来吸引注意，将其作为影响和控制周围人的武器。

## 四种性格的孩子如何“买玩具”

要快速看懂孩子的性格，观察他们在特定情境下的行为模式是关键。以下通过“孩子跟爸爸上街买玩具”的场景，来剖析四种性格孩子的典型表现。

### 1. 红色性格的孩子 🎈

红色性格的孩子情绪外露，变化迅速。

以下是红色性格孩子买玩具的典型过程：
1.  走在路上东张西望，对新鲜事物充满兴趣。
2.  看到喜欢的变形金刚，会直接、兴奋地提出要求：“爸爸，给我买这个好不好？”
3.  被拒绝后，会当场嚎啕大哭，试图用情绪达成目的。
4.  若发现哭闹无效（爸爸不理并走远），会沮丧地放弃，跟随爸爸回家。
5.  回家后，如果妈妈用其他礼物（如小猪佩奇小人书）安抚，能很快忘记之前的不愉快，恢复开心。

**核心特点**：情绪来得快，去得也快，容易哄，但相对闹腾和喜欢折腾。

### 2. 蓝色性格的孩子 🔵

蓝色性格的孩子内敛、含蓄，注重承诺与规则。

以下是蓝色性格孩子买玩具的典型过程：
1.  走路中规中矩，不吵不闹。
2.  看到喜欢的变形金刚，内心喜欢但绝不直接说，而是站在橱窗前纹丝不动地凝视。
3.  在爸爸催促时，会用含蓄的方式暗示，例如问：“爸爸，你觉得我这段时间表现好吗？……表现好的孩子是不是应该有奖励？”
4.  若被爸爸以“没带钱”为由拒绝，不会哭闹，会默默跟随回家，表面若无其事。
5.  此事会一直记在心里。可能几周后，当爸爸询问为何不开心时，他会旧事重提：“反正你上次说话就不算数。” 若爸爸忘记承诺，他会感到失望并不再轻易信任。

**核心特点**：不直接表达，善于暗示；极度看重他人对自己的承诺，一次失信可能影响深远。

### 3. 黄色性格的孩子 💪

黄色性格的孩子目标明确，为达目的善于运用策略。

以下是黄色性格孩子买玩具的典型过程：
1.  走在前面领着爸爸，看到目标后，立即采取行动。
2.  首先使用“迷魂汤”策略：“爸爸，你是全世界最好的爸爸！” 然后提出要求。
3.  若被拒绝，会转换策略，陈述利害关系：“我们班同学都有，我不会玩，你也没面子。”
4.  若软的不行，则来硬的，甚至可能“威胁”：“你不买，我就告诉妈妈你今天跟陌生阿姨要电话。”
5.  如果当场哭闹，其持久力和强度远超红色性格，会哭到“山崩地裂”为止。若当场未果，回家后会另寻途径（如缠着爷爷奶奶）直至达成目标。

**核心特点**：天生懂得如何达成目标，软硬兼施，意志坚定，是最有主见也最叛逆的类型。

### 4. 绿色性格的孩子 🌿

绿色性格的孩子温和、被动，欲望较少，最好抚养。

以下是绿色性格孩子买玩具的典型过程：
1.  上街时紧紧抓着父母的衣角，跟随左右。
2.  即使看到喜欢的玩具，如果父母不主动问，很少会自己提要求。
3.  一切听从父母安排，不愿给他人添麻烦。

**核心特点**：关照他人感受，被动顺从，是四种性格中最容易抚养的孩子。

## 总结与展望

本节课中我们一起学习了如何通过观察孩子的行为模式来快速识别其性格色彩。我们看到，**红色**孩子情绪多变、直接奔放；**蓝色**孩子含蓄内敛、重承诺；**黄色**孩子目标导向、坚定执着；**绿色**孩子温和被动、易于相处。这些差异在很小的时候就已由先天性格决定。

现在你可以明白，孩子“乖或不乖”、“好动或安静”、“喜欢表达或沉默寡言”，其背后主要是性格的力量在起作用。

在后面的学习中，我们会继续探讨：当我们长大成人后，**环境**、**经历**、**教育**以及**身边人的性格**，是如何影响我们的个性发展，有时甚至会让我们越来越不认识自己。

认清自己是我们每个人一生的功课。我们越清楚地知道自己是谁，就越有可能避免不必要的麻烦和代价，更好地发挥自身优势，并有意识地控制天生的短板。
