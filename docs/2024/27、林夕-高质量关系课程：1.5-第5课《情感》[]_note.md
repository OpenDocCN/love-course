# 27、林夕-高质量关系课程：1.5-第5课《情感》🌟

大家好，我是林夕。今天我们将进入高质量关系的实操环节。前几节课我们主要讲解了心态和理论，从今天开始，我们将学习如何真正地去做，如何实操。

**高质量关系**指的是你与自己以及与对象的关系。接下来三节课，我们将快速梳理我与我自己的关系，因为只有处理好与自己的关系，才能更好地处理与对象的关系。

**以下是三个环节**：

1. **我与我的情感关系**：很多人并不清楚自己真正想要什么样的伴侣。以下是一个识别择偶需求的标准，帮助你明确自己想找什么样的人。

   - **步骤**：
     1. 列出你理想伴侣的五项特征。
     2. 评估你遇到过的男人，找出分数最高的。
     3. 分析你的情感历程，了解自己一路以来都在寻找什么样的价值。

   - **注意事项**：
     - 如果最高分不是你最爱的人，可能是评判标准有问题，或者你真正在乎的标准没有写进去。
     - 如果你的情感历程是每任伴侣分数越来越高，说明你非常了解自己，情感生活顺利。
     - 如果每任伴侣分数越来越低，说明你的价值退步了，需要调整。

   - **总结**：
     - 总结你曾经爱过的伴侣，找出他们的共性，这可能是你最在乎的价值。
     - 分析分手原因，了解自己真正想要什么样的人。

2. **自我认知与接纳**：
   - 通过反思，找出你真正在乎的价值。
   - 接纳自己的情感需求，才能用正确的方式与他人相处。

3. **案例分析**：
   - 通过一个会员案例，说明如何通过分析择偶需求，帮助学员找到适合自己的伴侣。

**作业**：

1. 列出识别自己真实情感需求的步骤。
2. 按步骤写出你想要什么样的男人，并发送给课程总监审核。

**总结**：

本节课中，我们一起学习了如何识别自己的择偶需求，以及如何通过自我认知和接纳，找到适合自己的伴侣。希望每个学员都能享受高质量的爱情。
