# 56、小姨子供养者翻盘-强者框架课：005-第一节-4_ev

🎼情绪价值是易变的，可以提升也可以降低。心情好时，情绪价值高，社交舒适；心情差时，情绪价值低，社交压力大。情绪价值直接影响基础价值的吸引力。

**公式**：吸引力 = (生存价值 + 反衍价值) × 情绪价值

情绪价值是放大或缩小基础价值的变量。😊

**情绪价值的杠杆作用**：训练恋爱社交能力主要针对此。生存价值和繁衍价值短期内不可改变，情绪价值是突破口，能撬动巨大社交资源。

**案例**：一位富二代学员，家产丰厚，但缺乏核心吸引力，通过提升情绪价值，提升人格魅力，改变社交困境。

**提升情绪价值的方法**：带着目的和方法论进入实战，这是改变社交状况的唯一方法。不懂原理盲目社交无区别，只懂原理不实战则纸上谈兵。

**实战思路**：第三章至第五章将讲解聊天、约会、长期关系的基础实操思路，体会情绪价值在关系中的关键作用。

**关键在于个人**：学习者满足于懂道理，修炼者将道理应用于生活。私人定制学员通过针对性指导快速成长。

🎼准备好实战课件，让我们一起提升情绪价值，改变社交状况。😊
