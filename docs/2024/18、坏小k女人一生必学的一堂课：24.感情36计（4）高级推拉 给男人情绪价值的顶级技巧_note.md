# 18、坏小k女人一生必学的一堂课：24.感情36计（4）高级推拉 给男人情绪价值的顶级技巧

🎼 感情36计第四大技巧，四种高级推拉技巧，给男人情绪价值的顶级技巧。首先，了解最低层次的推拉技巧。推即推开对方，可利用打压、调侃、挑战，开开玩笑，展现灵性去指标。拉则是将对方的情绪波动掉下来的做法，赞美、保护、肯定、认同他。

以下是四种高级推拉技巧：

1. **先肯定再否定**：例如，对方问你这车怎么样，你可以先肯定再否定：“你这车坐的挺舒服的。不过音响效果一般。”

2. **混合信号**：一时表达对他感兴趣，一时表达对他不感兴趣，不断微调，让他不断经历情感过山车。

3. **显示与隐藏**：展示高价值，然后隐藏高价值，如展示你经常旅行，但原因是喜欢吃麦当劳的冰淇淋。

4. **好人和坏人**：让你跌入谷底，最后给你一线生机，如说：“虽然你看起来很热情，但我觉得你很难接近。”

**核心概念**：

- 推拉技巧：通过推拉让对方产生情绪波动，从而产生好感。
- 显示与隐藏：展示高价值，然后隐藏高价值，增加神秘感。
- 好人和坏人：让你跌入谷底，最后给你一线生机。

**总结**：

本节课中我们一起学习了高级推拉技巧，通过这些技巧，你可以更好地掌握男人的情绪，从而在情感中占据主动。
