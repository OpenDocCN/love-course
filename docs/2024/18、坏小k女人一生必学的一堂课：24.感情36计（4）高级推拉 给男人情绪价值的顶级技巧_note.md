# 课程18：感情36计（4）高级推拉——给予情绪价值的顶级技巧 🎭

在本节课中，我们将学习“感情36计”的第四项核心技巧：高级推拉。这是一种通过制造情绪波动来给予对方情绪价值的顶级方法。我们将从基础概念讲起，逐步深入四种具体的高级技巧，并通过实例帮助你理解如何在实际互动中运用。

## 概述：什么是推拉？

推拉技巧的核心在于制造情感上的张力和波动。“推”是指将对方推开的言行，例如通过打压、调侃或挑战来制造距离感。“拉”则是指将对方拉近的言行，例如通过赞美、肯定或认同来表达好感。

**公式表示**：互动 = 推（制造距离） + 拉（表达亲近）

一个基础的推拉公式是：**先肯定（拉），再否定（推）**。例如，当对方问你“这车怎么样？”，你可以回答：“你这车坐得挺舒服的（拉），不过音响效果一般（推）。”

上一节我们介绍了推拉的基本概念，本节中我们来看看如何将这种技巧升级，以产生更强烈的情绪价值。

## 四种高级推拉技巧

以下是四种能够有效调动对方情绪的高级推拉技巧。

### 1. 混合信号法

推拉技巧的本质是发出混合信号。即一时表达对他感兴趣（拉），一时又表达对他不感兴趣（推），通过不断的微调，让对方经历情感上的“过山车”。

例如，在约会中，你有时可以像对待男神一样赞美他、对他好（拉）；有时又可以像对待普通人一样，对他爱理不理或开他玩笑（推）。这种态度的切换会使他的情绪产生微妙变化。

**核心操作**：敢于让对方的情绪在“笑与哭”、“笑与气”之间转换。当他开始感动时，你的态度可以再次变得强硬。有拉就可以有推，有推就可以有拉，关键在于打破他当下的情绪状态，切入到另一种状态中。

### 2. 制造后悔法

你是否遇到过这种情况：一个男孩先对你示好，让你心动，而当你主动回应时，他却立刻退缩？这种“先拉后推”的策略，会让你因为自己的主动行为而后悔，从而对他投入更多情绪。

我们可以对男人使用同样的方法，并且同样有效。

以下是两种制造后悔的具体方式：

*   **后悔行为**：先释放可得性，让男人觉得你对他有兴趣（拉）。但当男人开始有所行动时，你便开始退缩（推），让他后悔自己的行动。
    *   **示例**：“本来我是对你挺有好感的，可是你这么快表白，让我觉得你是不是对其他人都这样？我突然很没有安全感。”

*   **后悔语言**：通过语言设置陷阱，让对方陷入自证或解释的境地。
    *   **示例1**：先一起笑话路人的穿着，然后说：“你怎么能笑话他人呢？”（推）等他认真时再说：“不过我也承认这确实太有趣了。”（拉）
    *   **示例2**：告诉一个男孩你欣赏他的三个方面，但只说出两个。当他着急问第三个时，可以说：“本来第三个是耐心，但是你现在着急知道答案的样子，让第三个特质有待考证。”（先拉后推）

### 3. 显示与隐藏法

顾名思义，此技巧在于交替展示和隐藏你的高价值。

*   **展示高价值**：透露你具备的吸引力特质或成就。
*   **隐藏高价值**：将展示高价值的原因，归结到一个非常平凡、甚至“俗气”的理由上。

**核心要点**：切记不要持续地“装备”自己。所有的高价值展示，最后都要“落地”到平凡的话题上，这样才显得真实、生动，不具攻击性。

以下是具体示例：

*   你经常全世界旅行（**展示高价值**），原因是你喜欢品尝各国麦当劳的冰淇淋（**隐藏高价值**）。
*   你高尔夫打得特别好（**展示高价值**），原因是你单纯喜欢晒太阳（**隐藏高价值**）。
*   你弹钢琴特别好（**展示高价值**），原因是你小时候为了逃课才学的（**隐藏高价值**）。

### 4. 好人与坏人法 + 故意推翻法

这个技巧组合旨在制造巨大的心理落差，从而牢牢抓住对方的注意力。

**好人与坏人法**：核心是“让你跌入谷底，最后给你一线生机”。
*   **示例**：“你知道吗？虽然你看起来像个很热情的人，但事实上很多时候我觉得你很难接近，可能你比较直男。我想你并不是一个外向、懂浪漫的人。我并不喜欢像你这样性格的男孩。”（**推，使其跌入谷底**）
*   “不过，这段时间的相处，我在你面前不知道为什么很放松。所以我有的时候在想，你会不会是我评价标准中的一个例外？”（**拉，给一线生机**）

这时，男人通常会试图做各种事情来证明自己，以得到你的认可，从而在情感层面开始追逐你。

**故意推翻法**：先装作关心或认同对方的回答，然后突然推翻。
*   **示例1**（推翻关心）：
    *   男：“约会结束后你准备干嘛？”
    *   女：“你为什么想知道？”（**设局**）
    *   男：“问问呀。”
    *   女：“一会儿准备和闺蜜去逛街。既然我告诉你我的计划了，你也得告诉我你的。”（**拉，建立交换**）
    *   等男人准备说时，打断他：“其实我不关心这个，我只想知道你会不会告诉我而已。”（**推，推翻之前的表现**）
*   **示例2**（推翻缺点，释放可得性）：
    *   “你知道吗？在我眼里你蛮大男子主义的。可能因为我自己也比较强势，我之所以还和你接触，可能是两个强势的人之间总有奇怪的欣赏（**拉，将缺点转化为连接点**）。不过我俩在一起的话，估计周围的人都会被吓跑，所以我们适合当一辈子的朋友（**推，设定边界**）。”
    *   此举意在**释放有限的可得性**，让对方做出选择：要么服从你设定的框架（例如，改变或证明自己），要么离开。

## 总结

本节课中，我们一起学习了四种高级推拉技巧：
1.  **混合信号法**：通过交替表达兴趣与冷淡，让对方情绪起伏。
2.  **制造后悔法**：通过“先拉后推”的策略，让对方为自己的行动或言辞后悔，增加情绪投资。
3.  **显示与隐藏法**：巧妙展示高价值后，用平凡的理由进行化解，显得真实而不做作。
4.  **好人与坏人法+故意推翻法**：制造巨大的心理落差，并通过推翻预期来掌控互动节奏，引导对方追逐。

掌握这些技巧的关键在于灵活运用与平衡，目的是创造积极的情感张力，而非真正的伤害。通过练习，你可以更自如地给予对方深刻而难忘的情绪价值体验。
