# 05、成真女生的秘密：46 月经杯怎么用？_note

在本节课中，我们将学习一种可重复使用的经期产品——月经杯。我们将了解它的外观、结构、工作原理以及使用注意事项。

## 概述与外观结构

上一段我们提到了经期产品的多样性，本节中我们来看看月经杯的具体构造。

月经杯的外观类似一个钟形。钟形主体部分用于置入阴道内，收集经血。其底部带有一个短柄。

短柄的主要作用是保持月经杯的平衡，并便于取出。其功能类似于卫生棉条的拉绳。

## 核心特性与工作原理

了解了基本结构后，我们进一步探讨其核心特性。

一个月经杯可以重复使用很长时间，这是其环保和经济的特点之一。

其核心功能是收集而非吸收经血。使用时，钟形部分在阴道内展开，贴合阴道壁，形成密封以收集经血。取出时，通过捏压杯体释放密封，并借助短柄将其拉出。

## 市场现状与使用注意

在掌握了工作原理后，我们来看看它在市场上的情况。

目前，在国内的超市或商场中，最常见的经期产品仍是卫生巾。卫生棉条和月经杯相对较少见，使用也不普遍。

许多女性可能对月经杯不习惯。因此，若考虑购买，需注意选择适合自己的型号。
