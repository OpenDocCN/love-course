# 课程05：月经杯使用指南 🩸

在本节课中，我们将学习一种可重复使用的经期用品——月经杯。我们将了解它的外观、结构、工作原理以及使用注意事项。

## 月经杯的外观与结构

上一节我们介绍了课程主题，本节中我们来看看月经杯的具体样貌。

月经杯的形状类似一个钟形。钟形主体部分用于置入阴道内，收集经血。其底部连接着一个短柄。

以下是短柄的主要作用：
*   保持杯体平衡。
*   便于取出杯子，其功能类似于卫生棉条的拉绳。

## 月经杯的耐用性与普及度

了解了基本结构后，我们来看看它的使用特点。

一个月经杯可以重复使用很长时间。目前，在国内的超市或商场中，最常见的经期产品是卫生巾。卫生棉条和月经杯相对较少见，使用也不普遍。许多女性可能不习惯使用月经杯。

因此，如果您打算购买，请注意区分。不过，估计也不太容易买错。
