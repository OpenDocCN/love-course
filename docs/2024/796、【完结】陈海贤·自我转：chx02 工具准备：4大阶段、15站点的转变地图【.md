# 796、【完结】陈海贤·自我转：chx02 工具准备：4大阶段、15站点的转变地图【

在本节课中，我们将学习自我转变的完整心理模型。这是一张结合了多领域智慧与真实人生经验的“转变地图”，它将帮助你理解转变的必经阶段、关键节点以及应对方法。

上一讲我们初步探讨了转变的本质与方向。本节中，我们将为你“打包行囊”，详细介绍这张涵盖**4个阶段**和**15个关键节点**的转变路线图。

## 地图的起源与价值

这张地图并非凭空产生。它融合了神话学、成人发展心理学、存在主义心理学、关系研究及职业发展心理学等多个领域的理论精华。

然而，它最重要的来源是当代人鲜活真实的人生经验与心路历程。许多人常困惑于转变为何如此艰难，我的观察是，人们对转变的理解往往过于窄化。他们误以为转变是一夜之间、靠一个决心或简单方法就能完成的事。

但深入观察便会发现，新自我取代旧自我的过程复杂而完整。转变者并非解决单一问题，而是走完了一条完整的道路。不同人的转变之路存在共性，正是这些共性被提炼、整理，形成了献给你的这份地图。

我们所构建的转变心理模型，核心结构可概括为：

**转变 = 响应召唤 + 脱离旧自我 + 踏上新征途 + 获得新自我**

接下来，我们将逐一审视这四个阶段及其包含的节点。

## 第一阶段：响应召唤 📣

转变始于第一个问题：**到底要不要变？** 我称之为“响应召唤”阶段。这是一个准备期，你的自我意志开始觉醒，并与现实环境产生剧烈矛盾，生活平衡被打破。新的要求与挑战，成为了新自我的种子。

在正式做出决定前，你可能会经历以下4个节点：

以下是“响应召唤”阶段的四个关键节点：

1.  **缘起**：在此环节，你的自我意志开始凸显。你开始重新思考一个根本问题：**“我到底想要什么？”**。发现这个“我想要”，是你主导转变的关键力量。
2.  **冲突**：虽然隐约感觉到了“我想要”，但你却发现自我与环境、或自我内部的不同需要之间发生了激烈冲突。在此环节，你需要探索自我的核心，并以此为依据思考选择。
3.  **容器**：新自我尚未形成，需要一个“容器”来培育。这个容器允许你在不立即放弃旧自我的情况下，滋养新的可能性。新自我的培育会推动下一个节点的到来。
4.  **契机**：当新旧自我无法共存时，一些看似偶然的生活事件会成为推动选择的关键转折点。在此环节，你需要学习如何理解、接纳甚至推动这些偶然，以促成重要转变。

当契机发生，我们开始脱离熟悉的生活，转变便进入了第二个阶段。

## 第二阶段：脱离旧自我 🚪

在“脱离旧自我”阶段，我们开始与过去告别。你在转变中体验到的许多伤痛、后悔、迷茫和恐慌都源于此。这些复杂情感常诱使你逃回“转变尚未发生”的过去。

但恰恰在这个阶段，你需要完成告别，才能为新的自我腾出空间。为什么这个阶段充满纠结？因为你可能面对以下几种情况：

以下是“脱离旧自我”阶段需要面对的几种典型情境：

*   **面对失去**：你可能失去一个重要目标或身份。你的任务是突破思维窄化，训练自己超越失去，去看更大的世界。
*   **面对放逐**：你可能从一个重要群体或一段重要关系中脱离，开始面对自我的孤独，学习重新依靠自己。此节点的任务是重新找回自我。
*   **面对告别**：你常常想要回到过去，尚未接受自己已从旧自我中脱离。此时你要做的是整理过去，与之正式告别。
*   **面对迷茫**：不确定性会引发焦虑，甚至让人失去探索的勇气。但如果你能迈过这一步，接受了失去与告别，就将开始探索新的可能性。

一旦你接受了失去并与过去的自己告别，转变便进入了第三个阶段。

## 第三阶段：踏上新征途 🚀

许多人认为转变从这里才算开始。但若无前面的经历，很难直接踏上新征途。在此阶段，你会发现新旧自我并非一刀两断。你开始从旧自我中寻找资源，寻找更认同的群体和关系，并通过实践探索可能性，摸索出新自我。

在这个阶段，你可能会经历以下三件事：

以下是“踏上新征途”阶段的三项核心任务：

1.  **整理旧自我的遗产**：虽然失去了旧自我，但它留下的许多重要资源（如真实品质、深刻体验）成为了新自我重建的基础。
2.  **寻找新的守护者**：你的转变如同从一个旧部落迁徙到新部落。你需要领路人，他们提供信息、途径，或帮助你理解正在经历的转变，给予宝贵指导。
3.  **掌握寻宝图**：你需要遵循 **“尝试 -> 行动 -> 获得反馈 -> 调整”** 的实践循环。通过不断探索，获得新体验，逐渐构建和摸索出新的自我。

你可能会问，如何知道新自我出现了？从外部看，你可能获得了新工作、新关系或新身份。但更重要的变化发生在心灵层面。此时，你就进入了第四阶段。

## 第四阶段：获得新自我 🏆

在“获得新自我”阶段，你所经历的磨难逐渐凝聚为人生智慧。你开始拥有新的评价坐标，以创造者身份面对现实，形成自己的信念与人生故事。转变本身，会成为新故事中最浓墨重彩的情节。

这个阶段包含以下关键转变：

以下是“获得新自我”阶段的三个标志性转变：

*   **指南针（坐标切换）**：你开始从外界评价坐标，切换到以**自我的感受、想法、意志**为判断标准的内部坐标。
*   **炼金术（面对现实）**：你以更成熟的态度面对现实，从融合、顺从或反抗中解脱出来，成为现实的**理解者、接纳者、利用者和改变者**，即现实的创造者。
*   **新故事（信念与叙事）**：你从困境中长出新的信念，并将所有经历转化为新的人生故事。在这个故事里，你拥有一个勇敢经历转变的新自我。

## 总结与提醒

本节课中，我们一起学习了自我转变的完整地图：从**响应召唤**，到**脱离旧自我**，再到**踏上新征途**，最终**获得新自我**。这四个阶段、十五个节点，勾勒出了你即将踏上的转变之旅。

但必须提醒：**地图不是道路**。你根据自身经历所走的路径，不会与地图完全相同。并非所有转变都需要以离职、离婚等激烈方式完成。转变形式多样，探索属于你自己的道路才是关键。

此外，地图的节点不一定每个都要亲历。例如，“容器”环节可能更适用于职业探索，而“告别”环节更适用于脱离一段关系。如果某些环节与你当前现实关系不大，也建议将其作为学问来学习，这能加深你对人、对转变的理解，未来或可帮助他人。

如何找到“我想要”？如何实现潜能？如何走过十字路口？如何从逆境中长出智慧？如何找回自己？这些重要问题都没有简单答案，需要一个完整的探索过程。这份转变地图，将成为你探索路上的路标。
