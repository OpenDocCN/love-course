# 21.王一名·吸引力出圈指南让：如何展示你的优势大家希望你展示

在本节课中，我们将学习个人品牌营销的核心一步：如何准备你的视频内容。我们将探讨什么是“好文案”，以及如何确保你发布的内容正是你的目标受众所希望看到的。

---

上一节我们介绍了如何起一个容易传播的好名字。本节中，我们来看看账号准备好之后，第一步具体该做什么。既然我们决定通过短视频来展示自己，那么第一步就是准备拍摄视频的文案，即你视频中要表达的具体内容。

无论你是想成为读书博主，还是分享某个领域观点的知识博主，一篇好的文案是视频成功的关键。

那么，什么是好的文案呢？

**好的文案，就是对你的目标受众有价值的文案。**

如何判断什么文案对你的受众有价值？在视频平台上，**点赞多的内容**，就是这个受众群体认为有价值的内容。

以前今日头条有句广告语叫“你关心的，才是头条”。在短视频平台上同理，大家关心的内容才是好内容。每个人只会给对自己有价值的内容点赞。这里的“价值”包括**情绪价值**和**实用价值**。

*   **情绪价值**：指你说出了观众的心声，或替他们表达了不会说的话，让他们产生共鸣、开心、安慰、佩服等正向情绪。
*   **实用价值**：指你的文案内容具有可参考和学习性，例如教人做菜、化妆技巧或提升吸引力。

因此，大家愿意点赞的内容，就是他们认为对自己有价值的内容。

既然我们知道了点赞多的内容就是大家认为有价值的，那么方法就简单了。如果你想在抖音上发布能获得大量点赞的内容，你就需要提前进行调研（具体调研方法后续会讲），找出哪些内容大家点赞多，然后去做类似的内容。

也就是说，你只要知道大家喜欢什么主题类型，然后围绕这个主题创作，就更容易获得大量点赞。这句话听起来简单，但需要深刻理解其背后的逻辑。

总结一下：你必须在抖音上发布**大家本身就喜欢的内容**，大家才会点赞。

很多知识博主经常提到一个观点：**爆款是重复的**。凡是能火的内容，有很大概率会再次火爆。因为这就是大家关心的。以抖音为例，其日活跃用户近7亿。一条拥有1000万播放量的视频，意味着仍有超过95%的用户没有刷到过。因此，同类型的火爆内容，换一种表达方式再做一次，同样可能火爆。

**请注意**：是创作**同一个主题**，但使用**不同的表达方式**，而不是简单地复制抄袭别人的文案。抄袭不会带来流量。

有一个重要的注意事项：你不能看到什么类型的内容火，就去发布什么类型。因为那些内容可能与你自身无关。你必须通过调研，找到**你自己领域内**点赞多的主题，这才是符合你账号定位的内容。

例如，如果你是一个美妆博主，你就不应该发布可爱的宠物视频、搞笑段子或其他随手拍的内容。因为这与你计划持续深耕的领域无关。即使某天有一条视频意外火了，获得了许多点赞，也不会吸引太多人关注你成为粉丝，因为你发布的内容太杂乱。

观众不会相信你后续会继续发布他们喜欢的同类内容。试想一下，你关注一个账号，通常是因为这个账号持续产出同一种类型的内容。这意味着你能在这个账号看到许多同类视频，并且确信它未来会继续发布此类内容，所以你才会关注。

反之，如果你刷到一个非常喜欢的视频并点了赞，但点进该账号主页发现内容杂乱无章，你就会明白这个博主并非专注于此领域，因而不会选择关注。因为你清楚这个账号后续不太可能继续发布你喜欢的同类视频。

所以，策略不是“什么火就发什么”，而是**持续发布你自己领域内的视频**，这样才会有人关注你。

内容创作是一个长期的过程。所有能成功的人，都是因为能在正确的方向上坚持。本节课就是告诉你，**正确的方向就是从最开始就创作大家本身就喜欢的内容**。只要方向对了，成功就只是时间问题。
