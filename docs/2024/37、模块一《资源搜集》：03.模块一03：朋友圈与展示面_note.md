# 37、模块一《资源搜集》：03.模块一03：朋友圈与展示面

大家好！今天我们将学习如何打造一个吸引人的朋友圈和展示面。以下是本节课的主要内容：

## 头像选择

**核心概念**：选择合适的头像可以提升个人形象，增加吸引力。

**步骤**：

1. **避免性相关头像**：避免低俗、性暗示的头像。
2. **避免非主流头像**：避免过于悲伤、忧郁的头像。
3. **避免美颜过度头像**：避免过于虚假、娘炮的头像。
4. **避免大头照**：避免过于普通、没有特色的大头照。
5. **选择有逼格的摄影作品**：展示个人品味和审美。
6. **选择体现兴趣爱好的图片**：突出个人形象，增加吸引力。
7. **选择特色鲜明的人物照**：展示个人追求和内在品质。
8. **选择本人的feel照**：展示个人感觉和气质。

## 昵称与背景墙

**核心概念**：昵称和背景墙可以提升个人形象，增加吸引力。

**步骤**：

1. **选择有品味的昵称**：与个人特点相关，易于交流。
2. **避免非主流或自恋的昵称**：避免低俗、矫情的昵称。
3. **避免工作职位加名字**：避免过于直接、缺乏吸引力。
4. **选择中文名字加英文名字，加emoji符号**：提升个人品味。

## 展示面

**核心概念**：展示面可以展示个人价值，增加吸引力。

**步骤**：

1. **展示生活有趣的一面**：展示个人生活丰富多彩的一面。
2. **展示优良价值**：展示个人特长、爱好等。
3. **展示个人品味**：展示个人审美和品味。
4. **避免虚假、夸大**：避免虚假、夸大的内容。
5. **避免负面内容**：避免抱怨、负能量的内容。
6. **避免广告、视频链接**：避免过于直接、缺乏吸引力。

## 总结

本节课我们学习了如何打造一个吸引人的朋友圈和展示面。通过选择合适的头像、昵称、背景墙和展示内容，可以提升个人形象，增加吸引力。希望同学们能够将所学知识应用到实际生活中，打造一个有魅力、有品味的个人形象。
