# 正向吸引高阶成长课：10_8·反差多变

🎼今天我们将学习一个重要的外在唤醒策略——反差多变。它不同于点对点的刺激，而是一种线性且强有力的刺激张力。

🎼这种策略能将对方从对你的刻板印象中抽离出来，带来惊喜和惊艳的感觉，使原本有吸引力的当下在对比中显得更加吸引人。

🎼这种反差多变会给对方带来无限遐想，因为你在对方心中已经贴上了一系列标签。

🎼在正式学习之前，我们先了解一个心理学效应——初丑效应。才能平庸的人可能不会受到太多喜爱，但毫无缺点的人也未必讨人喜欢。最讨人喜欢的往往是那种完美中带有点小缺点的人。

🎼举个例子，比如在酒吧遇到三个男人，第一个很帅、有钱、自信，一切都很完美；第二个比较普通，但跟你说话时很紧张；第三个很帅、有钱，但在喝酒时不小心碰倒了杯子。你可能会更容易喜欢上第三个男人，因为他的小瑕疵让他显得更加接地气、可爱。

🎼再比如，梁朝伟之所以受到很多人喜爱，除了他在电影中的表演外，还有他在现实生活中的害羞和社恐。这种反差萌让他更加生动、丰富。

🎼以下是一些关于反差多变的策略：

1. **暴露缺陷**：在对方面前不必总是逞强，适当示弱、犯错，卸下偶像包袱，反而可能起到更好的效果。

2. **短效反差**：适用于两个人互相了解的初期，短时间内风格不断跳换的视觉张力会产生一种惯性，预示着未来你可以有无限的可能。

3. **长效反差**：适用于关系的修复，比如挽回或重新唤醒。在一段时间内进行锻炼或分居，进行默默的成长和改变。

🎼以下是一些关于内外反差的例子：

1. **外在风格与内在性格的反差**：外表高冷，内心软萌；外表可爱，内心强大。

2. **长相风格与谈吐气质的反差**：长相高冷，说话软萌；长相可爱，说话霸道。

3. **行为上的反差**：身份不同带来的反差；在不同的环境下，有着完全不一样的言行举止。

🎼最后，我们可以通过MBTI性格测试和形象风格的人设标签对照图来判断自己的性格类型和形象风格，了解自己与他人的反差异。
