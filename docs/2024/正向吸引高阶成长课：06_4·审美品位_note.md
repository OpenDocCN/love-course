# 正向吸引高阶成长课：06_4·审美品位 👁️‍🗨️

在本节课中，我们将要学习“审美品位”这一核心概念。我们将探讨审美品位的定义、其重要性，以及如何通过具体的方法来提升我们的审美能力，从而增强个人魅力与社交竞争力。

## 什么是审美品位？ 🎨

审美品位并非一个被严格定义的概念，但它确实存在高低之分。审美品位是什么？它是基于你过往的认知，反映的是你当下的生活，表达的是你对未来的期许。

因此，审美品位可以用一个公式来概括：
**审美品位 = 过往认知 + 当下生活 + 未来期许**

首先，我们需要知道，审美品位是最直接、最长久且最容易展现的魅力表达方式。品味不仅是两个人共同话题的源泉，也是一个人对另一个人散发的致命吸引力。

百度百科将人的气质分为六大类：涵养、鉴赏、审美、眼界、品味和仪式感。当然，每个人对“气质”二字都有不同的理解。

## 气质与审美的深层联系 💫

上一节我们介绍了审美品位的定义，本节中我们来看看什么是“气质好”。它不仅是外在气质状态和形象的呈现，有时也是你内心内在的反应。因此，审美能力将成为未来社交生活中一项非常重要的竞争力。

因为审美品位是我们表达感性素质的重要元素。例如，一个审美能力弱、品味差的人，通常也很难有浪漫的诉求。在第二章的内在提升课程中，我们也会继续探讨感性素质的话题及其重要性，它是一种非常重要的情感唤起策略。

审美品位是一件很有意思的事情。它与情感、关系以及两人之间的相互吸引到底重不重要？传统意义上，我们说一个人外在好看，很有吸引力。但我们都知道这种吸引力不长久，再好看的人也会看腻，都会失去新鲜感。

所以，我们需要用对美的理解和表达，去延续我们的外在美，让对方感知到的美不仅停留在外表，更多是我们对生活的理解和对未来的规划。这是审美品位能反映出的最重要的东西。

## 审美品位的现实例证 🏠

为了更具体地理解，让我们来看一个例子。前段时间我去一个朋友家吃饭。这对夫妻中，男方在体制内上班，朝九晚五，看似很无聊。被体制内工作打磨后，人可能已没了棱角，回家后很累，不再有更多生活上的想法。

但那天我去到他家后特别惊讶。他们家是三年前装修的，放到今天的小红书上看，装修风格依然不过时，非常有情调，是法式浪漫风格，充满细节和艺术品装饰。尽管他们家有小孩，但家里从桌面、沙发到地上，没有一丝脏乱。

我很好奇，吃饭聊天时聊到当年装修的事情。每一个细节，每一件软装家居，甚至单凭一把椅子，都是他亲手去选的。听他分享装修过程，我感觉到，他把自己所有对生活的热爱、在工作里未必能满足的对美的追求，全部放到了生活中，用这种热爱去表达对生命的理解。

因此，他们夫妻经常在家里营造浪漫的仪式感，逢年过节有时也不去外面吃饭。这位男性朋友在工作中可能不太能表达对感性的追求和对审美情趣的理解，但在生活中却愈发想去表达。在交流过程中，我觉得他特别有魅力，特别可爱，形象瞬间更加立体，让人很想跟他交朋友，有更多交集。当然，这也是因为我们频率相同，谈得来。

假如我是一个对美没有自己理解的人，可能就无法发现他的这些优点。很多人只关注一个人（女生）好不好看，或（男人）会不会赚钱，但其实不是这样的。有时我们在散发魅力、表达自我价值时，可以通过审美品位和对生活的理解去侧面表达。这虽然不是最直接的方式（肯定没有长得美或很有钱那么直接），但一定更加深刻。

再比如电视剧《狂飙》里，高启强第一次主动去见大嫂（陈书婷）时，大嫂正陪孩子在幼儿园玩。高启强特意打扮，穿着西装去见她，本想第一句话就起范儿，占据主动权。但反过来，高叶扮演的大嫂指出了他的问题：衣服不合身，枪驳领西装不该那样穿，衬衫领子应该翻出来。瞬间，高启强的气势就落下去了，他不知道该说什么，整个人很有压力、非常紧张。

正是因为在这第一次见面中，大嫂表达了自己对品味、对男人品味的要求，瞬间拉高了她在高启强心中的位置，从而让高启强迷上她。吸引高启强的，不仅是这个女人的美、钱或背景，更多是这个女人对生活的理解。

就像一句俗话所说：男人娶了什么样的女人，就选择了什么样的生活。因为大部分男人不会花那么多时间去研究生活，除非他很闲。如果你娶了一个很懒的老婆，你的生活就很难丰富、过得很细致；如果你娶了一个勤快、对生活有追求的老婆，也许你们没有那么多时间在外面做大事，但你们一定有一个很温馨的小家。这就是一种吸引力。

当然，这是传统意义上“男主外女主内”的理解。现在也有很多夫妻共同布置家庭。所以，让一对情侣感情升温、或让他们有结婚、推进关系意愿的最好方式，就是一起去旅行，或者一起装修。

因为在这个过程中，双方会有更多关于审美品位的交流和探讨。你们能跟对方碰撞，能用你对审美的理解、对生活的追求去感化对方。当你们进入一个完全陌生的环境，一切你对美和新鲜事物的感知与认知，都基于你的经验和内心真实的需求与渴望。这时，如果一个人对美没有追求，对品味要求不高，观念就可能跟你不一致。这个过程也是双方三观互相交换、交流的过程。一起装修也是同理。

## 如何提升审美？四个方向 📈

我做了总结，提高审美可以有四个方向。

以下是审美提升的四个具体方向：

1.  **内在与外貌的审美**：例如护肤、美甲、妆容、发型、穿搭，以及如何选择配饰和奢侈品品牌。这些都是你审美品位的体现。
2.  **视觉上的审美**：可以通过摄影、绘画、看电影、看生活纪录片来找到视觉风格。例如近几年流行的莫兰迪色、莫奈风格，包括赛博朋克美学等比较有个性的美学形式，都能在视觉上反映你的审美。
3.  **艺术上的追求**：例如乐品（听音乐）、学习舞蹈、看话剧、听音乐剧，以及参观展览。
4.  **设计上的理解**：包括平面设计、室内布置（如刚才提到的装修），以及对建筑的理解。它更偏向结构和几何，与我们的生活息息相关。我们能想到的应用场景就是发朋友圈，去表达自己对美的欣赏，比如看到好看的建筑、图片或展览后做生活分享。

很多同学平常不知道发什么朋友圈，总来问我。其实不是你不会发，是你没有一双善于发现美的眼睛。当你拥有这样一双眼睛，随时随地都会产生想分享的情绪。就像在树林里看到一只孤单觅食的小猫，你就能捕捉到那个美的瞬间，那就是生活细节。

如果你每天忙碌奔波于上班和回家，对生活完全失去了感知能力，就很难发现这些细节之美，也很难产生分享的欲望。当你不能分享生活时，对方也无法感受到你的魅力。

当然，除了朋友圈这种形式，我们对审美的更多需求是内在的。我们真的被这些美好事物治愈，我们的一天被它们填充，内心是富足的，不再匮乏、饥渴、空虚。哪怕独自一人，也可以经营得很好。我认为这是审美和品味对个人而言更重要的意义。比起发朋友圈分享和展示自己，更重要的是修炼我们的内心。

## 如何提升品味？四个方向 🍽️

那么，提高品味也有四个不同的方向。

以下是品味提升的四个具体方向：

1.  **饮食品味**：如何对美食表达理解和喜爱，如何品酒、品茶，如何展示厨艺。这表达的是你在食物上的品味。
2.  **生活品味**：例如注意生活细节，如何布置房间，日常是否熨烫衣服，接人待物时的礼仪，从而散发精致的贵气。
3.  **兴趣品味**：培养一些有意思的爱好。例如，不管会不会画画，都可以定期通过画画疗愈自己。现在城市里有很多油画馆，一个下午可以自己去布置颜色，表达情绪和对颜色搭配的理解。还有手工DIY、插花、养宠物、练瑜伽、普拉提、冥想、做雕塑、练书法等，这些都能在兴趣上提升你的品味。
4.  **收藏品味**：常见的如收藏礼品、字画、文玩、花鸟、珠宝、古董，包括一些老CD唱片。

当我们在日常生活中有了这些积累，才能够在与对方沟通、与陌生人拉近距离时，找到更多话题，让对方对我们产生好感，通过我们的价值光环去吸引对方，从而拥有更多主动权。

我们是不是经常发现，在聊天或约会时，某一方讲更多关于自己的生活，另一方更多附和？其实是那个主动分享的人占据了主动权。如果你自己在生活上没有理解、没有追求、没有对美好事物的执念，就很难在聊天中占据主动权，只能被对方的话题带走。

## 提升审美的具体方法 🛠️

上一节我们介绍了审美的四个方向和品味的四个方向，本节中我们来看看具体如何操作。首先第一点，我们要**多看**，要训练自己的眼睛，要去“养眼”。眼睛是可以训练出来的，你对美的理解也能通过不停地看而提升。

以下是“多看”的具体途径：

*   利用碎片时间，大量观看摄影图片，尤其是摄影师作品和美好事物。
*   去看画展、时尚设计展、有意思的博物馆。
*   看电影和纪录片。但请从创作角度去看，例如电影如何构图、如何打光、如何让每一帧都很美。（课程最后会分享一些养眼的电影和纪录片。）
*   浏览相关网站和杂志，例如豆瓣。

豆瓣是一个兴趣社区，会针对很多话题聚集爱好者进行讨论。我自己有两个习惯：
第一是看电影前，会先看豆瓣评分以及大家对电影的不同理解和声音，带着这些思考和理解再去看电影。因为豆瓣上很多影评人比较专业，他们会从剧情、构图、色彩、形式风格、画面美感、镜头语言等角度分析电影和导演实力。不管看电影前还是后，我都会刷很久豆瓣，看看自己对电影的理解与这些领域内有积累的人的理解是否一致，差距有多大。
第二是听音乐。每听到一首喜欢的音乐，我就会去分析、幻想、构思音乐背后的情绪和故事，再点开评论区，看大家分享的情绪和经历是否与我产生共鸣。我就是通过这两个简单方式，在日常生活中训练自己对艺术表达和艺术形式的品味。

除了多看，第二是要**会看**。例如，对外貌而言，一个人除了长得好看，我们还需要判断他的长相风格。

以下是“会看”需要关注的细节：

*   **观摩外貌**：观察轮廓、五官比例、妆容类型、发型，以及他如何通过化妆修饰脸型、如何变美，他对穿搭、服装、首饰的理解。今天很容易找到很多优秀的、对审美有独特追求的博主，以及平面模特照片，学习他们如何通过拍照抒发情绪，用什么样的肢体语言传递故事。
*   **分析摄影**：看构图、视觉重心、画面平衡感；看图片中的搭配，人物与景的关系；看图片的层次——一个平面的2D图片如何通过3D的前景后景来表达故事和寓意。好的摄影作品没有多余元素，就像好的电影镜头里，每个道具的出现都有其价值和意义，我们要去揣摩。
*   **欣赏画展**：首先看一幅画的注释，了解画家创作时的背景、心境及想表达的故事。再看笔触、用的颜料、纹理细节，笔触是深是浅、是快是慢，这些动作都与画家当时的心境直接相关。最后看整幅画的构图、配色及所要表达的信息。这就是为什么有人看画不知道看什么，看一眼觉得不好看就不想看了；而有人能在画前看一小时甚至一天，与作者产生共鸣，感受到作者的喜怒哀乐，甚至产生快乐或悲伤的情绪。

在“看得多”以及“懂得怎么看”之后，第三要学会**思考**。例如，为什么他适合这种风格？为什么我适合那种风格？摄影师为什么要这么拍？他想表达什么？摄影师眼里的世界和我眼里的世界有什么区别？只有我们学会了看，懂得导演或摄影师创作时的心境，才能运用到自己的创作中。我们要把每一张照片、每一个视频都当作创作，只有懂了创作者的视角，才能在之后自己拍摄或表达时学以致用。

第四是**收藏**。首先要有概念，要建立自己的“审美字典”，包括不同类型风格的名词、不同视觉内容的元素。你脑海里拥有的概念和知识越多，对审美的理解就越丰富，这会直接反映到你的表达中。当然，我们可以通过转发朋友圈的方式，分享那些我们认为美好的照片、图片或视频，这也是一种常见的记录美好的方式。

第五是**表达**，这也是我认为最重要的。例如，我们自己认为不错的妆容或穿搭，光自己觉得美不够，要把它表达出去、输出出去，让周边的人看到，并从他们那里得到反馈。我们才能通过这种交流和反馈，让自己的审美更上一个台阶。

除了自身的美好，也可以将我们认为美好的事物分享给周边的人，比如自拍或发朋友圈。当我们获得点赞、收藏、转发后，同样能从社交网络得到足够多的反馈。因为我们要明白，现在无论是朋友圈还是外界信息都是爆炸的。我们如何让朋友、好友或感兴趣的人看到我们的信息？我们就必须分享足够有吸引力的信息、足够让人想点进去的照片、足够让人想看完的一段话。这一切都需要我们有品味。只有我们发的内容、作品、感想足够有吸引力，才会得到更多人的共鸣和关注，才会吸引到他们，并因此产生交流和情感。

除了表达和分享，我们还要学会**赞美**。看到美好的事物，要学会去赞美它，用我们所学到或收藏到的概念去表达。我们正是通过这种赞美，去表达对别人的喜欢，从而获得别人对我们的好感。

最后一点，要去**帮助他人**，尝试帮助周边的人提高审美。只有周边人的审美品味也在提高，我们的审美品味才会得到更多提升。试想一下，如果你跟一群不爱干净、对美没有追求的人生活在一起，一个人也很难坚持对美的追求。

所以，从今天开始，要养成对审美有追求的习惯，拒绝邋遢，拒绝敷衍，拒绝平庸。
