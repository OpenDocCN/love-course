# Simon2024新课【情感蓝图】「更新完毕」：3. 价值衡量

各位内部群的兄弟们，大家好。今天我们来学习第三节课，主题是“如何衡量你的价值”。

### 一切有违法如梦幻泡影

金刚经中有句话：“一切有为法，如梦幻泡影，如露亦如电，应作如是观。” 这句话告诉我们，一切事物都是因果变幻，都是暂时虚幻的一面。在分析事物时，不要执着于表面，要看穿事物背后的本质。

### 价值衡量

在课程中，我们了解到关系建立的最底层逻辑是价值。分析任何情感问题时，都要透过现象看本质，分析背后的价值。今天我们主要讲解的内容就是价值衡量。

### 价值衡量是什么？

价值衡量是指在做事之前，先衡量你所提供的价值和你收获的价值是否匹配。在进行良性实操博弈时，要学会评估价值。

### 价值评估

如果你的价值较高，女方做出的行为让你不高兴，你可以用一些手段和套路去让她顺从，提高她的接受度。这时，你的博弈是有效的。如果你的价值过低，不如女方，你需要尽可能提高自己的筹码，或者不作为，尽量减少自己的错误。

### 框架

框架是一个社交行为学概念，很多人在情感关系中梳理自己的原则和框架，表明自己是一个有立场、有棱角、有原则的男人。但对于小白来说，盲目设立原则和框架反而不好。当你的价值整体低于女方时，需要在前期放低姿态，妥协一些事情。等慢慢提升了筹码，引导了更多沉默成本后，你才会有更多机会去逆转框架。

### 价值碾压

当你的价值整体高于女方时，你可以尽情展示你的价值，树立框架与边界。如果你没确定这个女孩子是否适合常则的方式相处，就尽量不要利用常则的方式与她相处。

### 进攻性

在关系当中，合理利用自己的进攻性，向女方提出我们的需求和要求。在提出期望和要求时，要循序渐进，在女生投入成本不多，你本身的价值又没有完全显露时，不适宜提出超出你自身价值范围的要求。

### 总结

本节课中，我们一起学习了价值衡量、框架、价值碾压和进攻性等概念。希望对大家有所帮助。
