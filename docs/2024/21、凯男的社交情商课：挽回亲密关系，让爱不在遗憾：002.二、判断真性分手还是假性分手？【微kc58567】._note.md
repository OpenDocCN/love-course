# 21、凯男的社交情商课：挽回亲密关系，让爱不在遗憾：002.二、判断真性分手还是假性分手？

## 概述

在本节课中，我们将学习如何判断亲密关系中的分手是真性分手还是假性分手，并了解相应的挽回策略。

## 判断真性分手还是假性分手

### 1. 通讯情况

- **假性分手**：对方不会拉黑你的电话、微信或QQ，即使删了也能很快加回。
- **真性分手**：对方会拉黑你的所有通讯方式，完全把你当陌生人。

### 2. 情绪指标

- **假性分手**：对方可能生气、责怪，但还会接电话、回短信。
- **真性分手**：对方沉默、无视你，对你绝望。

### 3. 社交圈

- **假性分手**：对方没有告诉亲友，还在纠结。
- **真性分手**：对方告诉亲友，已下定决心分手。

### 4. 网络状态

- **假性分手**：对方可能在社交网络上发状态，或发刺激你的照片。
- **真性分手**：对方可能什么也不发，一片沉默。

### 5. 邀约情况

- **假性分手**：对方可能答应见面。
- **真性分手**：对方拒绝见面，或提出条件。

### 6. 是否彻底消失

- **假性分手**：对方可能还和你有交集。
- **真性分手**：对方彻底消失，不再联系。

## 挽回策略

### 假性分手

1. **穷追不舍**：主动表达感情，展现挽回决心。
2. **建立安全感**：痛改前非，用行动证明爱意。
3. **改变相处模式**：改变沟通和相处方式，巩固爱情。

### 真性分手

1. **空间抽离**：停止纠缠，消失一段时间，淡化负面印象。
2. **改变自己**：反省错误，提升魅力。
3. **恢复联系**：重建吸引，让对方感受到你的变化。
4. **感情升温**：用浪漫和实际行动让感情升温。

## 二次吸引

分手后，需要通过提升自我价值，提升自我魅力，重新吸引对方。

## 总结

本节课中，我们学习了如何判断真性分手和假性分手，以及相应的挽回策略。希望这些知识能帮助你在感情中更加从容。
