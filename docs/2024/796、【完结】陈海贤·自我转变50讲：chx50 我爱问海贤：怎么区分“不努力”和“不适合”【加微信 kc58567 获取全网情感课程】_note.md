# 796、【完结】陈海贤·自我转变50讲：chx50 我爱问海贤：怎么区分“不努力”和“不适合”

欢迎来到自我转变五十讲，我是陈海贤。我们已学完第四阶段课程，获得新自我。今天为第四次问答环节，前两个问题来自蒋行佳老师。

**问题一：走出改变人生的重要一步后后悔怎么办？**

**回答：**

1. 走出改变人生的重要一步可能会后悔，但不出这一步也可能后悔。
2. 哈佛心理学家丹尼尔吉尔福特认为，没有选择而错失一件事比做了选择更让人后悔。
3. 选择新道路时，用新评价标准思考，而非旧标准。
4. 选择新道路后，不断向前，建立新的评价坐标。

**问题二：对于电影《世界上最糟糕的人》的看法及如何处理自我选择可能带来的伤害？**

**回答：**

1. 电影中朱莉的选择逃避了更深的亲密关系，背后是对承诺的恐惧。
2. 选择要跟承诺相匹配，没有承诺的选择只是逃避。
3. 对于自我选择可能带来的伤害，尊重自己的选择，但也要理解对所爱之人的伤害。
4. 面对恐惧，要面对和克服，包括理解自己的选择对所爱之人的伤害。

**问题三：如何区别在探索过程中是自己的努力不够，还是真的在这个领域不合适？**

**回答：**

1. 努力不够还是领域不合适没有特定标准，取决于你的选择。
2. 内心感受和外在反馈是判断标准，两者相互影响。
3. 如果一件事对你很重要，会调整努力方式；如果不重要，会考虑换领域。
4. 不要在选择阶段停留太久，要不断尝试。

**问题四：为什么很多人希望通过比别人强来获得尊重？**

**回答：**

1. 竞争和合作是人类的本能，希望比别人强是本性。
2. 人不仅要追求比别人更强，还要超越自己的限制。
3. 转变意味着从外在标准中解脱出来，更好地发展自己。
4. 很少有人能完全不在意别人的评价，但可以做到不总是想着别人的评价，追随自己的内心。

**总结：**

本节课中我们一起学习了如何区分“不努力”和“不适合”，以及如何面对自我选择可能带来的伤害。希望这些内容能帮助你更好地认识自己，找到属于你的道路。
