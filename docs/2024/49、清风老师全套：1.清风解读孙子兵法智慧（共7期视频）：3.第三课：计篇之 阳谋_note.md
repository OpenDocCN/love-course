# 清风老师全套：孙子兵法智慧解读（第三课：计篇之阳谋）

## 概述
在本节课中，我们将深入探讨孙子兵法中的“计篇”，特别是其中的“阳谋”策略。我们将学习如何通过信息搜集和战略计算来预测胜负，并理解孙子兵法中的“先胜后战”和“先算后战”的核心思想。

## 信息搜集与战略计算
以下是孙子兵法中关于信息搜集和战略计算的原文内容：

**原文**：故教之以技而所其情曰煮熟有道。

**翻译**：因此，通过教授技艺并了解其具体情况，可以说有方法。

**解释**：这段话强调了在战斗或竞争中，通过学习和了解对手的情况，可以找到取胜的方法。

**公式**：信息搜集 + 战略计算 = 胜利

## 先胜后战与先算后战
孙子兵法中的“先胜后战”和“先算后战”是关键的战略思想。

**原文**：将听武技，用之必胜，留之将不听武技，用之必败去也。

**翻译**：听从我的武技，使用它必胜；不听从我的武技，使用它必败。

**解释**：这句话表明，只有当有必胜的把握时，才应该进行战斗。

**公式**：必胜把握 + 行动 = 胜利

## 强势文化与弱势文化
孙子兵法中的“强势文化”和“弱势文化”是理解人性和社会规律的重要概念。

**原文**：所谓强势文化就是遵照客观规律的文化，不以人的意志为转移，按照客观规律去办事。

**翻译**：所谓的强势文化就是遵循客观规律的文化，不随人的意志转移，按照客观规律行事。

**解释**：强势文化强调遵循客观规律，而不是个人喜好。

**公式**：客观规律 + 行动 = 成功
