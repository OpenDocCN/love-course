# 倪《高段位主导关系》：007. 场外权力之三：无处不在的自我PUA 🧘

## 概述

在本节课中，我们将要学习第三种场外权力：自我满足的权利。这是一种由自己赋予自己、能够带来内在能量与自由的重要能力。

---

所有的三种场外权力都有一个共同点。那就是它们都是你自己给自己的。这些权力不是别人、社会、职称或头衔赋予的。它们属于你可以自己去创造的私人化权力，是当你把自己作为人生的主角后，本身就会获得的。

上一节我们介绍了场外权力的共同特征，本节中我们来看看最后一种，也是看起来最微不足道的一种权力：自我满足的权利。自我满足就是爱自己、满足自己、取悦自己。

我不会讲太长这节课。我知道自我满足的反面——否定自己的欲望——对某些人来说，已经形成了自动化的思维，根植于骨肉深处，很难拔出。我遇到过太多兄弟，每天都在否定和压抑自己，从生活中的任何一件不起眼的小事开始。

我也知道，一旦一个人掌握了自我满足的能力——注意，这是一项可以练习的能力——他的人生将会开始真正的自由。自由是一个大词，我不展开讲，因为它不接地气。我换一个词：你会开始真正的无所畏惧，不再攀比，快乐自足，享受属于自己的人生。这听起来可能还是不够接地气。那么，更具体地说，你会在跟女生的相处中没有需求感。

自我满足就是自己满足自己的诸多欲望，并且不带评价地去满足。有些欲望满足起来很轻松，一点都不会内耗。比如去吃一顿好的，花几千块钱吃顿米其林。在我们这个年代的人看来，这不会有太大阻力。但对于生于五六十年代、从物资匮乏年代过来的人看来，享受对他们来说就是困难的。

但是，有一些欲望的满足，对很多人来说就是困难的。比如攻击欲。攻击欲也是人的本能欲望。有人侵犯你了，你骂他“狗娘养的”，这就是攻击欲。我们的学员里面有很多老实人、老好人，很难去满足自己的攻击欲。整个人看起来就很疲惫、很虚弱。因为攻击欲不对外释放，就会对内产生内耗。他们说话总是客客气气、有礼貌、有教养，但是没什么活力。

有一次我跟一个兄弟聊天。我问他：“你有骂过人吗？”因为我听他的感觉是非常礼貌客气的。他说：“我骂过。”我说：“你怎么骂的？”他说：“哎，你怎么能这样啊。”我说：“这不是骂人，确切地说这叫指责。”后来我让他骂人，我让他骂“草泥马”。一阵强迫之后，他终于憋着一股劲骂了出来。骂完之后，他说他浑身充满了力量。

因为他在生活中是极其压抑的。压抑的来源是，当他有任何想要表达的东西，他都会先在内心里面去考虑：这样说会不会不太好？这样说是不是很没礼貌？这样说是不是显得不稳重？天天这样去折磨自己。这就是攻击欲的压抑。

那么如何去表达攻击欲呢？除了去骂人，你还可以去做一点暴力的运动，把攻击欲释放出来。比如拳击、摔跤等等。

那么还有什么样的欲望满足会困难一些呢？比如表达欲和展示欲。明明想说，明明想要去表达，明明在人群当中想要去说自己的观点，但是这个时候会不去满足自己，告诉自己说：“这样会显得太秀了，太炫耀了，太争强好胜了，话太多太好为人师了。”发个朋友圈也是：“哎，这样发会不会太装逼了？”于是这个欲望就在一瞬间被否定、被压抑回去了。

还有一个非常重要的欲望，表达起来也是困难的，那就是性欲。性欲当然有它的特殊性，它通常跟一些禁忌有关。但满足自己的性欲却是极其重要的。我认为所有的需求感都是因为性欲的不满足而来的。甚至我认为，所有生活中的不愉快都是因为性欲的不满足。比如有些中年妇女脾气暴躁，不用问，那肯定是没有性生活的。更年期就等于没有性生活了。

这里我想说一个概念，叫做“撩拨自己的欲望”。这是很多人处理自己欲望的一个方法。就是说，当自己的一个欲望出现时，不是直接去满足，他用的方式是去撩拨自己的欲望，让欲望悬浮在空中。

举个例子。比如你在某个APP上看上一双鞋，价格略高。这时候你想买，但是你既不是咬咬牙直接买了，也不是直接就不买了。而是没事就打开APP去看一看，看一会儿舍不得下单，然后晚点又去看。一直让自己的欲望处于得不到又被激活的状态。我觉得这是对自己最大的残忍。不断地、重复地去否定自己的欲望。欲望起来了，放在这儿否定它；过一会儿它起来了，又否定它。就是不满足它，就是不让它落地。

再举个例子。比如有的时候有点欲望了，就看个小黄片，看了呢又不干点什么，过一会儿又看。这就是撩拨自己的欲望，痛苦。不满足自己的欲望，还有比这个更残忍的事情吗？

那怎么才能够不撩拨自己的欲望呢？答案是：**立即行动，立即决策**。比如你喜欢某双鞋，要么马上买，要么关掉APP。如果太贵了，就换一双便宜的，马上行动。想要表达攻击欲，如果不是在有利益相关的场景下，那么马上骂出来（如果不是面对你的老板）。或者说马上背包去健身房去释放。不要让那个欲望停留在体内伤害自己。

《道德经》里面说过，人修炼的最高境界是“复归于婴儿”。尼采也说过，人从骆驼到狮子到婴儿的三个境界（或叫成长阶段），都是“婴儿”。你看，他们都提到一个词叫“婴儿”。婴儿是什么特征？在我们今天的语境下，我愿意叫他“自我满足，自我滋养”。想吃就吃，想哭就哭，想笑就笑。

自我满足最大的障碍，是成长中的限制性信念，是清理内在的被内化的那些规训戒律，是清理掉道德感，是释放真实的自我，充分地自我满足。老子和尼采都透露出了一个重要的信息：这件事他妈的不容易。

你现在就是一个骆驼，背负着各种道德压力、社会训诫活着。你经常感觉不到自己的存在，丧失乐趣，都快要抑郁了。做点什么都要去想：“这样会不会不太好？”而不是马上做。我想喝可乐，会不会糖太高？吃烤肉会不会太油？一个不快乐的人，怎么有能量去做生活中的其他事情呢？你甚至会怀疑：我苦哈哈的、咬牙切齿的努力工作，然后赚了钱之后，我也不会真正的享受。那潜意识里面就根本不会想要去赚钱。

所以，不是你没钱，所以不会享受；而是你不会享受，所以没钱。这样的人通常贬低一切会让自己爽的事情。

所以，在我们的社群里面，我会有一个板块，专门叫做“立即满足”，也可以叫“立即行动”，或者说取一个其他的名字。就是说，不要在头脑里面盖楼，也不要在头脑里面拆楼，在头脑里面去骂人。而是回到现实中，真实的去碰一碰，立即行动。想都是问题，行动才是答案。因为行动才会把假问题变成真问题。

另外，你需要成长，需要系统知识的支持，去粉碎掉那些限制我们的思想重负。这些都需要在我们的社群里面浸泡。

我们继续。最后我们要提一下，当我们开始自我满足之后，为什么就会转变成主导关系的权利呢？我相信，就算这之间没什么关系，你也会这么做，你也要这么做。因为它的意义远大于主导关系这件小事情。但是课程还是要交代一下。

当我们开始自我满足之后，我们的内心会产生一种自我认同的感觉，会产生强大的能量。你知道需求感是怎么来的吗？你想象一下，一个一年都没有做过爱的男人，你要让他没有需求感，这是不可能的。她就是看女人一眼，那个女生都会觉得这个男人的目光像是激光一样，会把她衣服都烧烂。
