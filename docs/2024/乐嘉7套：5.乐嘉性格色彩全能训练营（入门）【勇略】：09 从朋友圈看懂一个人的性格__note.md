# 乐嘉性格色彩全能训练营（入门）【勇略】09：从朋友圈看懂一个人的性格 👁️

在本节课中，我们将要学习如何通过观察一个人的社交媒体（如朋友圈或微博）内容，快速判断其核心性格色彩。这种方法无需深入接触，仅从日常分享的细节即可洞察人心。

## 概述

人的性格可以在生活的方方面面，甚至很多细节里体现出来。有时，一个人只要发了一条朋友圈或者说了一句话，你就能推断出他的性格。本节课将重点教授如何通过社交媒体内容快速判断性格。

## 不同性格的社交媒体表现差异

上一节我们概述了通过社交媒体洞察性格的可能性，本节中我们来看看不同性格色彩的人在发朋友圈时，具体有哪些差别。

### 1. 红色性格：外放与分享

典型红色性格的人发朋友圈，最喜欢发三样东西。

以下是红色性格的典型分享内容：

*   **美美的自拍照**：在典型红色性格看来，展示大量美美的自拍照是件快乐的事，获得点赞和夸奖会让他们开心。即便知道个人信息有暴露风险，他们也可能为了“及时享乐”而继续分享，并花费大量时间修饰照片。
*   **个人生活细节**：例如去哪里玩、看了什么电影、减肥成果、孩子的成绩、收到的礼物等。红色性格希望第一时间与朋友分享快乐，并期待获得羡慕和赞美的留言。他们的天性外放、不设防，认为暴露私生活的风险概率远低于分享带来的即时快乐。
*   **负面情绪**：例如公开抱怨工作太累、与人吵架、客户难缠等。典型红色性格的人，如果自身力量不够，特别希望从外部世界获得力量。通过表达痛苦并获得他人的鼓励，他们能更快恢复开心状态。

因此，如果你看到一个人的朋友圈里有大量自拍、频繁分享生活点滴、并偶尔夹杂负面情绪，基本可以判断他是典型的红色性格。

### 2. 蓝色性格：内敛与含蓄

上一节我们介绍了外放分享的红色性格，本节中我们来看看与之完全相反的蓝色性格。蓝色性格不希望受到过多关注，他们的内心像有一堵厚厚的城墙，追求“我不出去，你也不要进来”的安全感。

在发社交媒体这件事上，蓝色性格是发帖频率最低的。即便发，也会用非常低调、含蓄的方式表达。

以下是蓝色性格的典型表现：

*   **内容以转发为主**：可能只转发一首歌、一篇文章，而不配或只配极少文字。这些内容往往承载着个人重大但私密的情感或纪念意义（例如，转发一首定情曲在分手纪念日），即所谓“以物言志”。他们不追求被所有人理解，只在乎特定的人能读懂。
*   **原创内容晦涩难懂**：如果发布原创内容，经常显得含蓄、晦涩，需要特定“钥匙”（即了解其内心的人）才能解读。蓝色性格精心包装的信息，层层包围，外人难以窥见其核心。

所以，如果一个人朋友圈多是转发、缺乏个人直白表达，或其原创内容让你感到晦涩难懂，那他很可能是内敛含蓄的蓝色性格。

### 3. 黄色性格：实用与成就

接下来我们再来分析一下黄色性格。黄色性格做事的核心标准只有一个：**有用**。如果经营社交媒体对自己没用，他们会认为是浪费时间。

因此，黄色性格不会耗费大量精力刷朋友圈，但社交媒体对他们有两个重要作用。

以下是黄色性格利用社交媒体的两个主要方向：

*   **凸显个人成就**：他们愿意分享事业突破、完成的高大上项目、见到的大人物、参加的高级会议，以及家人取得的亮眼成绩。因为黄色一生追求成就与强大，社交媒体是凸显这些价值的有效平台。他们**绝不会**在朋友圈分享内心真实的感受、情感和情绪。
*   **吸收与分享实用知识**：他们乐于转发如“马云送给创业者的19句话”、“如何有效管理时间”、“5分钟看懂伟大公司商业模式”这类有用信息。这对他们价值连城，发朋友圈就像记笔记一样是为了留存纪念。

因此，如果你（尤其是面对客户时）发现一个人的朋友圈充满了彰显成功与实用的信息，基本可以判断他是黄色性格。

### 4. 绿色性格：平淡与平和

最后让我们来聊一聊绿色性格。绿色性格的人发朋友圈也很少，因为他们觉得看别人发就很快乐，自己似乎没什么可发，生活平淡，发了也没人看，所以觉得没必要发。

绿色性格无欲无求，日子过得平淡。即便发朋友圈，内容也如“清汤寡水”。

以下是绿色性格的分享特点：

*   **内容平淡简单**：例如“妈妈来看我了，好开心”、“养的花开了”、“看的电影让我睡着了”。他们不太会雕琢语言，句子像小学生写的，照片也不会精心修饰，觉得麻烦且没必要。
*   **情绪平稳**：绿色性格脾气好，不易陷入负面情绪，也很快能排解。因此，你基本不会在他们的朋友圈看到夸张的心情描述或负面情绪的表露，似乎他们的人生永远由简单的日常构成。

在很多人看来，绿色性格活得“没劲”，但恰恰因为他们无欲无求、安于现状、心态平和，反而是活得最轻松的人。他们平淡如小学生作文般的朋友圈，充满了一种“平淡是美”的生活氛围，这或许是许多人一生打拼后所追求的方向。

## 总结

本节课中，我们一起学习了如何通过社交媒体内容快速判断四种性格色彩：
*   **红色**：乐于分享自拍、生活点滴与情绪（包括负面情绪），追求关注与互动。
*   **蓝色**：发帖极少，内容含蓄、晦涩或以转发代言，追求深度理解而非广泛关注。
*   **黄色**：内容围绕“有用”原则，旨在凸显成就或分享实用知识，避免流露个人情感。
*   **绿色**：发帖少，内容平淡简单，情绪平稳，反映其无欲无求、轻松平和的生活态度。

认清一个人的性格并没有那么难，不一定需要深入接触和长谈。类似朋友圈、微博这样的日常信息，就足以暴露一个人的性格底色。下回你想了解一个人，不妨先从性格色彩的角度看看他的社交媒体。善用这个工具，你会发现洞察人心在日常生活中无处不在。
