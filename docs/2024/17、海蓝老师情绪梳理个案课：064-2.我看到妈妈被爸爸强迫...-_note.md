# 海蓝老师情绪梳理个案课：064-2.我看到妈妈被爸爸强迫...

## 概述
在本节课中，我们将探讨一个关于儿童心理创伤的案例。通过分析一个6岁孩子小海鹰的回忆，了解他在家庭冲突中的感受和经历。

## 案例背景
小海鹰回忆起自己6岁时，看到父母之间发生冲突的场景。他感到非常厌恶和恶心，对这种感受的起源和影响进行梳理。

## 案例分析
### 1. 感受的起源
小海鹰最早感受到这种厌恶和恶心的情绪是在6岁左右。当时，他看到父母之间发生推搡和争吵，感到非常惊恐。

**公式**： 感受 = 惊恐 + 家庭冲突

### 2. 感受的影响
这种感受对小海鹰的年龄来说，可能不是很强烈，但他对这种事情有恐惧感。当时，他可能还没有完全理解发生了什么，但已经感到不安。

**代码**： 恐惧 = 不理解 + 家庭冲突

### 3. 场景描述
小海鹰回忆起父母之间的争吵，其中一方不愿意，另一方强迫。他感到恶心、恐怖和无助。

**公式**： 恶心 = 强迫 + 不愿意
**公式**： 恐怖 = 强迫 + 不愿意
**公式**： 无助 = 强迫 + 不愿意

### 4. 小海鹰的感受
小海鹰感到自己无法改变现状，非常无力。他试图劝阻父母，但没有人理会他。

**公式**： 无力 = 无法改变 + 尝试劝阻

### 5. 与父母的关系
小海鹰感觉与父亲的关系更近一些。在冲突中，他感受到母亲是痛苦的。

**公式**： 父子关系 = 近
**公式**： 母亲感受 = 痛苦

## 总结
本节课中，我们一起学习了小海鹰在家庭冲突中的感受和经历。通过分析他的回忆，我们了解到儿童在家庭冲突中的心理创伤，以及他们所面临的困境。
