# 恋爱核武器4.0：03.3、如何不见面，就让女生喜欢上你

## 概述

本节课将教你如何在不见面的情况下，通过朋友圈和拍照技巧吸引女生。

## 一、朋友圈运营

### 1. 发朋友圈的时间

*   晚上6点至8点
*   午饭以后
*   晚上11点至12点

### 2. 更新朋友圈的频率

*   三天一更，灵活调整

### 3. 朋友圈内容

*   **60% 自我展示**：兴趣爱好、吃喝玩乐、吸引女生的价值
*   **30% 不带自己的生活**：刷存在感，提醒女生你的存在
*   **10% 不感兴趣的内容**：分享音乐、书籍、观点等

### 4. 配文方式

*   **照片高大上，配文逗比**：展现自信，拉近与女生的距离

### 5. 不在乎他人看法

*   **提升价值，展现酷**：不在意他人看法，提升自身价值

## 二、拍照技巧

### 1. 人像三要素

*   **人**：摆姿、神态
*   **构图**：居中构图、对角线构图、三分法构图
*   **光线**：避免逆光，选择合适的光线

### 2. 摆姿

*   **控制眼神**：脸朝向哪儿，眼睛看向哪儿
*   **神态**：自然、舒适、自信
*   **道具**：矿泉水、报纸、可乐、香烟等

### 3. 构图

*   **居中构图**：人物放在画面中间
*   **对角线构图**：人物偏离画面中心
*   **三分法构图**：将画面分为九宫格，人物放在四个焦点之一

### 4. 光线

*   **避免逆光**
*   **选择合适的光线**

## 三、总结

本节课介绍了如何通过朋友圈和拍照技巧吸引女生。通过运营朋友圈，展示自己的价值，并通过拍照展现自己的魅力，可以有效地吸引女生的注意。
