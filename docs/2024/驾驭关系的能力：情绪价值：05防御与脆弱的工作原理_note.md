# 驾驭关系的能力：情绪价值：05防御与脆弱的工作原理_note

在本节课中，我们将要学习一个人内在“防御的自己”与“脆弱的自己”这两个核心部分的工作原理。理解它们如何相互作用，是掌握情绪价值、改善人际关系的关键。

上一节我们介绍了人的内在构成。本节中，我们来看看防御与脆弱这两个部分具体是如何运作的。

## 什么是“防御的自己”？🛡️

我们先说“防御的自己”，也可称为防御部分。每个人内心当中都会有一个防御的自己。

从名词上解释，“防御”在心理学上常被称为“防御机制”或“防御面具”。它指的是，人在面对人和事时，往往不是展现全部真实的自己，而是戴上一个“面具”。这个面具可能是愤怒、指责、冷漠等。

他为什么要有这样的防御面具呢？其实防御的是不想被别人发现的点。从“防御”二字去体会，它指的是保护起自己不想被别人看到的东西。

以下是防御的两个主要类型：

*   **无意识的防御**：指当事人自己意识不到的防御行为。例如，当被伴侣指责时，对方瞬间暴怒，但他自己并未意识到暴怒是因为自尊心受到了伤害。这种反应源于潜意识，是下意识的。
*   **有意识的防御**：指当事人故意采取的防御行为。最常见的例子是“冷战”，通过持续的不理睬来宣泄情绪或报复对方。

我们需要明白，防御其实是有意和无意两种，它们是不一样的。

生活中有太多防御面具，无法一一列举。皱眉、抱臂沉默、不表达等，其实都是一种防御面具。

所以，我们学习的目标是培养一种能力：**发现和觉察别人（以及自己）防御面具的能力**。

你最应该先识别的不是别人，而是自己。大家应该先记录自己的防御面具。

以下是建立自我觉察的建议：

*   记录自己何时发脾气（指责面具）。
*   记录自己何时想静静独处、拒绝沟通（回避/冷漠面具）。
*   通过记录，建立属于自己的“葵花宝典”，更了解自己。
*   在了解自己的基础上，再用家人作为观察素材，你对防御面具的觉察能力会大大提高。

我们先解释清楚了“防御面具”或“防御部分”的概念。

## 什么是“脆弱的自己”？💔

接下来我们讲第二个部分：“脆弱的自己”。一个人分为真正的自己、脆弱的自己和防御的自己三部分。

脆弱的自己，往往是我们潜意识里的东西，自己通常看不到。每个人都有脆弱的自己，它往往源于原生家庭或成长环境中形成的心灵创伤。

为什么叫脆弱呢？因为一个防御，一个脆弱，它们其实是相互呼应的。**防御防的是脆弱，是保护脆弱**。因为它脆弱，所以要保护。而这些保护往往处在无意识状态，是潜意识里的东西。

脆弱的自己，指的就是这些潜意识里的创伤。例如，自卑就是一种常见的脆弱部分，它由创伤形成。

这些创伤是如何形成的呢？它源于我们的成长经历。

例如，童年时期缺乏父母的情感回应（如“哭就不管”的粗放养育），可能让孩子形成“我不值得被爱”的信念，未来可能发展为回避型人格。而过度的溺爱，剥夺了孩子尝试和获得成就感的机会，也可能导致其产生深深的无能感和自卑。

而这些脆弱的部分，我们通常是不认可的。谁能认可自己的自卑、无价值感或恐惧呢？当你不愿意接受时，你会做什么？**你会做掩盖。这会激发你的防御面具。**

## 防御与脆弱的工作原理 ⚙️

现在，我们来看看防御部分和脆弱部分的关系。它们的核心关系是：

**脆弱的创伤（脆弱部分）通过潜意识，瞬间激发防御部分。**

例如，当我们感到不被尊重（脆弱的羞耻感被触动），会立刻启动防御，用发脾气或攻击对方的方式表现出来。你看到的是攻击（防御面具），但内在的起因是不被尊重的羞耻感（脆弱部分）。

它们之间是有对照关系的，但并非简单的一一对应。需要结合具体事情来分析。

以下是它们的工作原理：

1.  **触发**：当**人和事**刺激到我们内在潜意识里的**脆弱部分**（如羞耻、恐惧、委屈）时。
2.  **反应**：我们**本能地**会启动**防御面具**（如愤怒、吵架、对抗）。
3.  **目的**：通过防御面具，来**掩盖**我们内在的**脆弱部分**。

这是一个连带关系：**脆弱部分驱动防御面具**。

我们为什么要学这个原理？因为日常生活中，我们只关注别人表现出来的愤怒（防御面具），却忽略了他内在更想被保护的脆弱部分。

每个人都希望被尊重、被爱、被认可、被包容。我们最想保护的正是自己的内在脆弱。因此，别人要想靠近我们，必须**穿透我们的防御面具，读懂我们的脆弱**。能做到这一点的人，才会被我们认为是懂我、尊重我、爱我的。**情绪价值的核心点就在于此。**

如果我们不明白这个工作机制，就会一直被对方的防御面具（如发脾气、不讲道理）挡在外面，觉得无法解决问题。本课程的目的，就是教你**穿透别人的防御面具，进入其脆弱的部分**，从而建立深层连接。

## 人际冲突的“尾巴与獠牙”游戏 🐾

我们用一个比喻来理解人与人之间的冲突。把一个人**脆弱的部分**叫做 **“尾巴”** ，把**防御的部分**叫做 **“獠牙”** 。

每个人都有脆弱的尾巴和防御的獠牙。

冲突过程如下：
1.  伴侣做了某事让你不满（他的行为/獠牙刺激了你的尾巴）。
2.  你的脆弱尾巴感到疼，立刻激发你的防御獠牙（你生气并攻击他）。
3.  你的攻击（獠牙）又踩中/咬疼了伴侣的脆弱尾巴。
4.  伴侣感到疼，立刻也激发他的防御獠牙回头攻击你。

如此循环，就像两个人在互相追着咬对方的尾巴。这就是为什么婚姻中的问题会越来越复杂。

学习这个原理后，送你一个法宝：当你不想吵架时，**下意识地“摸摸自己的尾巴”**（想想自己脆弱的点是不是被踩到了）。这个动作能帮你转移注意力，甚至笑出来，从而让理性回归，压下即将爆发的防御獠牙。接着，你便可以冷静地去觉察对方的防御面具，并反推其脆弱的尾巴。

**请注意**：找到对方的尾巴，不是为了去踩它、报复他，而是为了理解他，从而穿透防御，走入他的内心，成为他最重要的人。这个能力可以用于驾驭关系、增进亲密，而非欺负他人。

## 总结与学习方法 📚

本节课中，我们一起学习了：
1.  **防御的自己（防御面具）**：分为有意识和无意识两种，是用来保护脆弱、掩盖不想被看到部分的表面行为。
2.  **脆弱的自己**：通常是潜意识中的心灵创伤，如自卑、恐惧、羞耻感等，我们不愿面对它。
3.  **工作原理**：**人和事 → 刺激脆弱部分 → 本能激发防御面具**。防御是脆弱的“保护壳”。
4.  **应用意义**：要改善关系，需要**穿透对方的防御面具，理解和接纳其脆弱部分**，这是提供情绪价值的核心。
5.  **实践方法**：先从**自我觉察**开始。记录自己的防御面具，并探究每个面具背后对应的脆弱感受（如恐惧、羞愧、委屈、孤独等）。在自己身上搞懂这种连带关系后，才能更好地反推和理解他人。

学习心理学工具，最应该先研究的实验对象是自己。先把自己分析透彻，明白自己每天的防御与脆弱，才能将心比心，真正掌握洞察他人、经营关系的能力。如果一遍没听懂，请带着反思自己的态度，多听几遍。
