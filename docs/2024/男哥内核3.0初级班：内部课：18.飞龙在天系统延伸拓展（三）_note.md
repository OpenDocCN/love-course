# 男哥内核3.0初级班：内部课：18.飞龙在天系统延伸拓展（三）

## 概述
在本节课中，我们将深入探讨飞龙在天系统的延伸拓展，学习如何通过正确的思维和心态，提升个人魅力和人际关系。

## 正确思维的重要性
**公式**： 正确的思维 = 正确的心态 + 身份转变

### 1. 高价值认知
- **事实**： 世界上不存在真正的高价值人。
- **原因**： 认为对方条件优越，实则只是主观认知。
- **行动**： 保持平等心态，认识到每个人都有价值。

### 2. 价值来源
- **来源**： 来自于个人如何看待自己和对方。
- **行为**： 仰望对方，给予对方价值。

### 3. 仰望行为
- **行为**： 不停发问、提供无用话题、过度关注。
- **结果**： 不会有结果，降低自身价值。

### 4. 高分女生认知
- **事实**： 认为的高分女生并非真正高分。
- **原因**： 人类肤浅，始于颜值。
- **行动**： 关注内在，提升自身价值。

### 5. 女生与闺蜜话题
- **事实**： 女生与闺蜜的话题可能更开放。
- **原因**： 男女思维差异，女生更愿意分享。
- **行动**： 保持开放心态，理解女生。

### 6. 女生不会告诉你的真相
- **事实**： 女生喜欢意中人，性格和外在都强大的男人。
- **行动**： 提升内在和外在，成为意中人。

## 实战技巧
### 1. 情绪管理
- **原则**： 当情绪被影响时，采取以下步骤：
  1. 选择性加人，分散注意力。
  2. 持续自身建设，获得正向反馈。
  3. 练习魅力男士动作，提升社交技巧。

### 2. 选择与计划
- **原则**： 同样的时间、成本，选择不同的人和事，方法论决定结果。
- **行动**： 改变选择，提升自身价值。

### 3. 聊天中不能犯的错误
- **错误**： 秒回、展示内向脆弱、需求感过强、贬低自己取悦女生、解释和纠结、自我陶醉、话痨、规划人生、过度装逼、突兀升高关系。

### 4. 聊天目的
- **目的**： 让对方投资，逆向合理化，激发探索欲望。
