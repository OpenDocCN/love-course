## 微信基础设置：名字与头像

上一节我们指出了微信人设的普遍问题，本节中我们来看看如何从最基础的名字和头像开始改造。

以下是设置微信名字和头像的核心原则：

1.  **微信名字**
    *   **公式**：`中文名/昵称` 或 `英文名`
    *   中文名字不一定用全名，可以用你名字的最后一个字。例如，最后一个字是“熊”就叫“熊哥”或“熊军”；最后一个字是“涛”，就可以叫“涛哥”或“大涛仔”。即用你最后一个字再起个昵称。这比任何稀奇古怪、表达情绪的名字强百倍。

2.  **微信头像**
    *   在大家提升完自己的性张力之后，一定要去小红书上约一个摄影师给你拍。
    *   **要求**：他拍，有场景感（例如开车、在隧道里），既能认出是你，又特别有氛围感和文艺感。
    *   **核心目的**：头像必须让女人“下得去口”。如果你的头像让别人下不去口，那么后面再做任何动作都已经输了。

## 朋友圈加分素材

说完名字和头像，第三部分是加分的朋友圈素材。朋友圈是你个人价值的展示窗口，需要精心经营。

以下是四类能够为你加分的朋友圈素材：

1.  **他拍形象照**
    *   让别人拍你的帅照，包括侧面、背影，要有场景。每月发一两张即可，发多显得自恋。

2.  **高大上场景**
    *   **国外旅游**：在小红书上搜索当地摄影师，付费拍摄。
    *   **高级活动**：出席的高级会议、高级场所。注意拍下场景和地理位置。
    *   **高级消费场所**：高级餐厅、雪茄吧、设计感强的酒吧。如果无人帮你拍人像，务必露出场所名称。

3.  **硬价值展示**
    *   **财富**：例如股票账户资产（如2000万），可以适当展示。钱是男人魅力的一部分，但不能只用钱吸引人。
    *   **资产**：好车（如保时捷、跑车）、好房子（窗外景色、别墅、大平层、知名豪宅）。拍摄时要“不经意”，避免刻意。

4.  **高价值社交**
    *   与社会名流、明星、知名网红或有来头人物的合影。这是凸显你社交圈层和硬价值的有效素材。

## 朋友圈减分行为

了解该做什么之后，我们同样需要知道应该避免什么。以下是必须杜绝的朋友圈减分行为：

1.  **设置三天可见**
    *   除非你是王思聪或吴彦祖本人，否则普通人不要设置三天可见。

2.  **发布长篇“小作文”**
    *   避免撰写关于见了哪个企业家、得到什么人生感悟的长篇大论。

3.  **发泄牢骚与讨论政治**
    *   不要骂人、抱怨或发表政治立场议论。这主要显得“土”。

4.  **展示负面情绪**
    *   避免在朋友圈发布慌张、紧张、痛苦等负面情绪。

5.  **发布低质量内容**
    *   不要发手机自拍。可发可不发的内容（如拍得不好的羊肉串）就不要发。宁可少发、发得精，也不要多发、发得土。

**额外提醒**：大家也不用刻意购买某些机构提供的“朋友圈健身图”，因为有些拍得过于刻意，反而过犹不及（例如刻意在书架前摆拍），容易给人一种矫揉造作和油腻的感觉。
