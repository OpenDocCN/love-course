# 任亚《人间清醒后半生》：第14节 男孩养育与女孩养育的区别和侧重点 👨‍👩‍👧‍👦

在本节课中，我们将探讨男孩与女孩养育的异同。核心观点是：**首先应将孩子视为“人”来培养，而非过分强调性别差异**。我们将分析性别刻板印象的弊端，并讨论“精神富养”对所有孩子的重要性。

## 破除性别刻板印象 🚫

上一节我们探讨了养育的基础理念，本节中我们来看看性别议题。首先需要明确的是，男孩与女孩在养育上并无本质区别。每个孩子首先都是独立的“人”。

我们现今观察到的许多性别差异，实际上源于养育过程和社会环境中的刻意区别对待。例如，社会常默认男孩应该玩枪、玩车，女孩应该玩布娃娃。这是一种典型的**性别刻板印象**。

因此，我想强调的第一点是：**不要区别对待男孩和女孩**。优秀的个体往往是“雌雄同体”的，兼具多种特质。若能秉持此观念，你便是在更高维度上养育子女。目标是培养一个“很棒的人”，性别带来的主要是生理与生命角色的不同，例如女性拥有子宫，男性没有；男孩的发育阶段可能略晚于女孩（但这并非绝对）。

关键在于，不要将刻板的性别印象强加给孩子。男孩也可能喜欢布娃娃，女孩也可能喜欢枪炮。女性可以在商界、政界表现出色，男性也可以将家庭照顾得细致入微。职场中，那些细心、体贴的男性同样富有魅力。因此，我们不应给孩子灌输诸如“男儿有泪不轻弹”或“女孩才能哭”之类的观念。

**在养育中，应避免刻意制造性别刻板印象。**

那么，男孩与女孩的主要区别何在？区别主要在于生理结构与发展过程。例如，女性拥有子宫，可以生育，这使她们可能拥有与生命共呼吸的独特体验，进而可能增强其共情能力。但请注意，这并非绝对。

**每个生命都是独一无二的，切勿陷入刻板印象。**

## 精神上皆需“富养” 💖

讲完避免刻板印象，我们来看看男孩与女孩养育的共同侧重点：**精神上都需要“富养”**。

常有人问：“为什么主张男孩要穷养，女孩要富养？” 这是一个非常糟糕的命题，根本不应存在。有人认为女孩富养后，长大才不会因金钱跟人走。但男孩因金钱跟人走就好了吗？这是一个伪命题。

我认为，**精神上都要富足，物质上量力而行**。

“量力而行”指根据家庭实际经济条件提供相应生活。没有必要刻意“装穷”。孩子是聪明而敏感的，并不好欺骗。即便能骗过孩子，也难以骗过其同学与环境。

例如，我曾为读初中的儿子买了一辆较贵的碳合金自行车。他非常喜爱。当被门卫问及是否是“富二代”时，他回答“不是”。我告诉他，这并非因为他是富二代，而是妈妈愿意在这件事上给予他快乐。我计算过，自己买包买鞋的快乐，不如他拥有这辆自行车的快乐。我赠予的是一份快乐，而非单纯的物品。

**富养的本质，是给予孩子精神上的丰盛与快乐，而非在物质上刻意匮乏。**

我有一位学员，曾是单身妈妈，经济优渥却信奉“男孩穷养”，从不给孩子买好东西。听完我的课程后，她决定改变，为儿子买了一双限量版球鞋。儿子欣喜若狂。对孩子而言，贫富是相对的。若有经济条件，为何一定要给孩子劣质物品？美好的事物生来就是为了被欣赏与享受。

当然，这并非鼓吹“只买贵的，不买对的”。贵总有贵的道理，但核心是购买“好东西”，而非盲目追逐名牌。**过度崇尚名牌，往往源于早年的匮乏感**。真正富足的人，只购买自己所需。

## 关于“吃苦”与逆商的培养 🏃

许多家长想通过“穷养”或刻意制造逆境来培养孩子的吃苦精神与逆商（抗挫折能力）。我的观点是：**完全不必刻意让孩子吃苦**。

然而，有两件事至关重要：
1.  **经常进行户外运动**
2.  **坚持做家务（自己的事情自己做）**

优秀的运动员若品行良好，日后创业往往更易成功。原因在于，他们的身体承受能力更强。成功常在于比别人多坚持一点，而这份坚持依托于身体。**身体能做到，生命才能做到**。

家长无需刻意让孩子吃苦。推荐阅读《上海的金枝玉叶》，书中讲述了一个从小富养的孩子，长大后并不觉得生活苦楚。反之，从小被刻意训练吃苦的孩子，容易认为世界本是苦的，并会放大苦难。

**一个未曾感知深度苦楚的孩子，长大后对苦的耐受力，反而可能强于那些从小沉浸在苦痛恐惧中的孩子。**

这是因为，成年人能理性看待事实本身，而孩子容易放大想象的恐惧。小时候吃一点苦，若被刻意强调，恐惧就会被放大。成年后遇到小挫折，便易退回童年的痛苦感受中。这如同我们面对孩子情绪时，也会放大自身情绪一样。

若童年未曾遭受不必要的苦楚，长大后便会觉得：“这件事就这么回事，撑过去就好了。” 以我自身为例，许多朋友认为我“娇生惯养”、“从未吃苦”。我深呼吸后并未辩解。事实并非如此，我经历的许多苦楚是我主动选择去体验的。我从未觉得生命是苦的，而是将其视为生命的闯关游戏与丰富阅历。正因这些经历，我才能胜任现在的工作。

**因为懂得，所以慈悲；因为经历，所以更爱这个世界。同时，我深刻明白：未经他人苦，莫劝他人善。**

## 界定真正的“穷养”与“富养” ⚖️

最后，我们来界定何谓真正的穷养与富养。

**富养，核心是精神上的富足感。** 精神富足的人，只有“需求”，少有“欲望”。欲望常源于童年被压抑的需求。如果幼时微小的需求长期得不到及时满足，便会进入头脑，演变为无法满足的欲望。

**公式：被压抑的童年需求 → 成年后膨胀的欲望**

因此，**欲望是无法满足的需求，而需求是可以满足的欲望。** 真正的富养，是及时、合理地满足孩子的正当需求，从而培养其内在的精神丰盛与满足感。

---

本节课中，我们一起学习了：
1.  应摒弃性别刻板印象，将孩子首先作为“人”来培养。
2.  男孩与女孩在精神上都需要“富养”，物质上则应量力而行，不必刻意“穷养”。
3.  无需刻意制造苦难来培养逆商，但应鼓励户外运动与家务劳动。
4.  真正的富养在于满足精神需求，避免将需求压抑成未来的欲望。

养育的终极目标，是培养出一个精神富足、人格完整的“人”。
