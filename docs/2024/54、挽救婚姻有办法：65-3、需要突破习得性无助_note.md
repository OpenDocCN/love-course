# 54、挽救婚姻有办法：65-3、需要突破习得性无助 📚

## 概述

在本节课中，我们将探讨如何突破习得性无助，以挽救婚姻。

## 案例分析

以下是一个网络案例：

> 老公出轨多次，她知道但没有采取行动。这一次出轨严重，小三怀孕，逼着离婚。男方同意离婚，但女方担心离婚后无法获得应有的财产。她尝试过各种方法，但都无济于事，感到无助。

## 什么是习得性无助

习得性无助是一种心理状态，当个体在经历连续的失败和挫折后，会感到无法改变现状，从而放弃努力。

**公式**：

习得性无助 = 连续失败 + 挫折 + 放弃努力

## 如何突破习得性无助

1. **认知婚姻的必然性**：

> 婚姻就像生命一样，有生有死。我们要认识到婚姻的消亡是必然的，就像熵定律所描述的，世界万物最终都会归于消亡。

2. **相信他人的能力**：

> 当我们感到无助时，要相信他人有能力帮助我们。我们可以寻求专业人士的帮助，例如心理咨询师或婚姻辅导师。

3. **相信行动的力量**：

> 只要我们愿意去行动，就一定会有所改变。我们要相信自己的能力，并付诸行动。

## 总结

在本节课中，我们学习了如何突破习得性无助，以挽救婚姻。通过认知婚姻的必然性、相信他人的能力和相信行动的力量，我们可以找到挽救婚姻的方法。
