# 老白《吸引力法则》：第5课：营造被动吸引力激活她对你的热情_ev

## 概述
在本节课中，我们将学习如何通过营造被动吸引力来激活对方对你的热情。我们将探讨如何区分感情和关系，以及如何通过引导对方来提升彼此的关系。

## 关系与感情
**关系由感情决定**。真正的恋爱高手不会过分关注关系本身，而是关注当下的感情。

## 错误案例
**对方说你好可爱啊，当事人说谢谢还好了**。这种回答缺乏引导，且让对方难以继续话题。

## 正确引导
**对方说你好可爱啊，你可以这样回应**：
```
所以你会为可爱这个词儿着迷吗？
```
这样的回答可以引导对方承认对你有好感。

## 举例说明
以下是几个引导对方承认你优秀或喜欢你的例子：

**对方夸你做饭**：
```
你可以就是回会做饭的男生在你眼里是怎么样的啊？
```

**对方夸你工作努力**：
```
你怎么看待工作努力的男生？
```

## 反向引导
当对方抱怨你时，也可以用类似的方法引导：

**对方说你好高冷啊**：
```
你是希望我不那么高冷啊？
```

## 男生视角
**对方说感觉你很会撩呃，你会不会是海王呃**：
```
感觉你很会撩，不会是海王吧。我可能会回，你是希望我不要撩你嘛，先把问题抛给对方。
```

## 总结
本节课中我们一起学习了如何通过营造被动吸引力来激活对方对你的热情。通过引导对方，我们可以让对方自然而然地顺着我们的节奏，从而提升彼此的关系。
