# 海蓝老师情绪梳理个案课：048-09 结束

🎼在课程结束后，回顾今晚的梳理，总结三大收获是什么？🎼

### 1. 目标清晰

**公式**：目标清晰 = 当下目标任务明确

**代码**：清晰目标 = 立即行动

### 2. 意识到自我恐惧

**公式**：自我恐惧 = 长期自我吓唬

**代码**：自我觉醒 = 停止自我吓唬

### 3. 相信转化能量

**公式**：转化能量 = 将负面情绪转化为正面情绪

**代码**：转化过程 = 静心倾听、释放、转化

🎼当遇到愤怒、害怕等情绪时，这些都是等待被转化的能量。需要静下来倾听，了解需要化解的问题，并慢慢释放出来。

### 4. 巴西状态

**公式**：巴西状态 = 淡定、清晰、好奇、关怀、创造力

**代码**：巴西状态 = 保持积极心态，面对生活

🎼当处于巴西状态时，我们会感到自信、充满力量，不再有恐惧。

### 5. 内在力量

**公式**：内在力量 = 认识到内在没有不好的部分

**代码**：内在力量 = 增强自信，面对挑战

🎼当我们认识到内在没有不好的部分时，我们会更有力量，不再有恐惧。

### 6. 不再自我吓唬

**公式**：不再自我吓唬 = 相信自己，不再自我怀疑

**代码**：不再自我吓唬 = 坚定信念，勇往直前

🎼要相信不能再自己吓自己，牢牢记住这些是来提醒你的。

### 7. 身体感受

**公式**：身体感受 = 轻松、满足

**代码**：身体感受 = 保持积极心态，享受生活

🎼此时此刻，从头到脚身体感到轻松，也觉得挺傻的。但这也是一种美好的体验。

### 8. 担心父母

**公式**：担心父母 = 过度担心

**代码**：担心父母 = 正确面对，理性处理

🎼担心父母是正常的，但过度担心会影响日常生活。要正确面对，理性处理。

### 9. 未尽的意愿和孝道

**公式**：未尽的意愿和孝道 = 完成心愿，尽孝道

**代码**：未尽的意愿和孝道 = 努力实现愿望，关心父母

🎼人一定会离开，关键离开之前，有好多未尽的意愿和未尽的孝道。要努力实现愿望，关心父母。

### 10. 尽力满足父母愿望

**公式**：尽力满足父母愿望 = 尽孝道，关心父母

**代码**：尽力满足父母愿望 = 努力实现愿望，关心父母

🎼要尽力满足父母的愿望，关心父母，让他们感到幸福。

### 11. 担心减少

**公式**：担心减少 = 清晰面对害怕

**代码**：担心减少 = 勇敢面对，减少恐惧

🎼当把害怕都清楚后，担心会减少很多。

### 总结

本节课中我们一起学习了如何梳理情绪，认识到内在没有不好的部分，相信转化能量，不再自我吓唬，关心父母，尽力满足父母愿望，减少担心。希望这些内容能帮助大家更好地面对生活，拥有积极的心态。
