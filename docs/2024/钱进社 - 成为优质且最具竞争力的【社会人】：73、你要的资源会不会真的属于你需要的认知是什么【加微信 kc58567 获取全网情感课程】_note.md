# 钱进社 - 成为优质且最具竞争力的【社会人】：73、你要的资源会不会真的属于你需要的认知是什么

## 概述
在本节课中，我们将探讨如何正确评估资源是否真正属于你所需要的，以及如何避免在追求资源时陷入误区。

## 定位资源
**公式**：资源评估 = 实用性 + 可变现性 + 对方性格

### 实用性
- **问题**：你确定对方的资源，你用得上吗？
- **分析**：考虑对方资源是否与你的需求相符，以及你是否有能力利用这些资源。

### 可变现性
- **问题**：对方资源是否真的可以落地以及变现？
- **分析**：考虑对方资源是否具有实际价值，以及是否能够为你带来实际利益。

### 对方性格
- **问题**：对方是否值得你去跪舔？
- **分析**：考虑对方的性格是否自负且自恋，以及你是否能够从对方那里获得实际帮助。

## 案例分析
### 小张与王姐
- **背景**：小张是一位销售业务员，希望做大单。
- **问题**：小张过度追求王姐的资源，却忽略了对方的真实实力和资源变现能力。
- **结论**：小张应该更加理性地评估资源，避免盲目追求。

## 资源变现的误区
- **误区1**：认为所有有资源的人都是值得合作的。
- **误区2**：只看重对方的地位和名望，而忽略对方的实际能力。
- **误区3**：过度追求资源，而忽略了自身能力的提升。

## 总结
本节课中，我们学习了如何正确评估资源是否真正属于我们所需要的。在追求资源时，我们要理性思考，避免陷入误区。最终，我们要关注的是对方的为人，以及资源是否能够为我们带来实际利益。
