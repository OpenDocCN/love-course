# RSD德里克-游戏十诫：第9诫 - 帮助他人实现目标

**Commandment 9， I shall help others accomplish their goals。**

**不遵守这条诫命会发生什么？你的社交影响力将大打折扣。**

想象一个无家可归且吸毒的人，没有人会在意他。但如果你帮助他实现目标，情况就完全不同了。

**诫命的解读：**

这条诫命源于一句古老的谚语：“物以类聚，人以群分。”这个观点在德里克之前就已经存在。1900年前，T.O.G. Kim Eteitis就知道了这一点。他说：“最重要的是，永远不要让自己与旧相识和朋友过于紧密相连，以至于被他们拉低到他们的水平。否则，你将一事无成。”

你选择是否让这些朋友爱你，同时保持你原本的样子，或者为了这些朋友而成为一个更好的人。如果你试图两者兼得，你将既无法进步，也无法保持你曾经拥有的东西。

**以下是践行这条诫命的几个要点：**

1. **积极建立由聪明、有资源、有抱负的人组成的部落。**
2. **同时，也要积极远离那些分散你注意力的人。**
3. **如果你给自己设定高标准并大量社交，你将结识到许多优质的朋友。**
4. **随着时间的推移，你的生活将充满优质和积极的事物，你将没有时间去做任何不完美的事情。**
5. **要充满伟大的事物，将没有更多的时间去陪伴那些优秀的人。**

**关于对老朋友的忠诚：**

邀请他们加入你追求更好生活的旅程，就像他们陪伴你并跟上你的步伐一样，如果他们做不到，你已经走上了结交更好朋友的道路。

给他们提供几次机会，让他们迈出步伐。不要进行自我斗争。尽可能培养他们，给予真诚的鼓励，设定目标，创造怨恨。

只是为了帮助你的朋友。随着时间的推移，一些人会加入你，与你一起做一些了不起的事情。其他人可能只是自己做一些酷的事情，或者他们可能选择不做什么。那些找到有成效的道路的人，也许你以后会再次联系他们，如果你会联系他们。也许你会在以后的生活中与他们一起做一些更酷的事情。

对于那些没有抱负或无法实现他们尝试的目标的人来说，你将看到你的对话逐渐变成过滤你生活中最好的东西，这样我们就不会伤害这些人的感情。你的对话将全部关于回忆过去的事情，或者只是听他们抱怨自己的生活，通常都是责怪别人。

**以下是践行这条诫命的几个具体例子：**

1. **帮助女孩实现目标：**
   - 了解她真正兴奋的事情以及你如何可以做出贡献。
   - 给予她鼓励。
   - 阅读有关信息，帮助她的事业。
   - 介绍她认识你社交网络中的人，这些人可以帮助她的事业。
   - 以身作则，这意味着她应该看到你朝着自己的目标努力并实现它们。
   - 与她一起参加学校或其他教育活动。
   - 与她一起参加TD Talks、Ideas Week、自助研讨会。
   - 指出她与那些不能帮助她实现目标的人在一起的地方。
   - 与她一起旅行，了解更多关于世界、历史及其文化。
   - 带她参加刺激的冒险，如跳伞、与鲨鱼游泳、浮潜、射击机枪。
   - 带她去赛车场，体验赛车。
   - 与她一起骑摩托车。
   - 一起去健身房。
   - 一起吃健康食品。
   - 做任何能让她的生活变得更好的事情，并给她一个更亲近你的理由，帮助她与你建立更紧密的联系。

2. **帮助其他优质男性：**
   - 商业也是团队支持。做出有用的社交介绍，帮助他们的事业。
   - 邀请他们参加有趣的活动，无论在哪里发生有趣的事情，都有女孩在场，男孩都想在那里。
   - 冷门接触女孩，并介绍一些女孩给他。
   - 许多男性都会回报你，并尽最大努力以类似的方式帮助你，或者以其他方式，这取决于他们拥有的资源。

**总结：**

在本节课中，我们一起学习了RSD德里克-游戏十诫中的第9诫：帮助他人实现目标。这条诫命强调了建立积极的人际关系和实现个人目标的重要性。通过帮助他人，我们可以提升自己的社交影响力，并为自己创造更多的机会。
