# 《leon全套课程合集》：leon 1.0开撩蓝图：3 开场前三秒

## 概述
在本节课中，我们将学习如何有效地开场，特别是在前三秒内吸引女生的注意。

## 开场类型
**直接开场**：直接表明你的意图，例如：“嗨，你好，我注意到你，想和你认识一下。”

**间接开场**：以寻求意见或帮助为开始，例如：“你好，你知道星巴克在哪吗？”

## 静止目标与行走目标
**静止目标**：例如在等人、在商场门口、在餐厅吃饭的女生。

**行走目标**：例如在步行街或商场中行走的目标。

## 静止目标的开场技巧
1. 从背后走过去，走到她旁边，保持一个手臂的距离。
2. 大大方方地打招呼，例如：“嗨，你好。”
3. 微笑着走过去，表明你的意图。
4. 直接告诉她你在等人，例如：“我在这边等人，觉得你挺正的，过来给你打个招呼。”

## 行走目标的挑战
行走中的目标更容易被打枪，因为她们可能认为你是销售人员。

## 提升开场成功率的三点技巧
1. **开场操作**：从背后走过去，走到她旁边，保持一个手臂的距离。
2. **状态**：保持坚定的眼神和肢体语言，不要像销售人员一样。
3. **意图**：在第一时间内表明你的意图，让她知道你是来搭讪的。

## 总结
本节课中我们一起学习了如何有效地开场，特别是在前三秒内吸引女生的注意。记住，从背后走过去，保持坚定的状态，并在第一时间内表明你的意图。这样，你的开场成功率将会得到显著提升。
