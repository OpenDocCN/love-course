# 《朕哥8套情感合集》：06.朕哥情感 情感认知提升课：23-如何让你喜欢的男生倒追你

🎼 如何让你喜欢的男人倒追你，这个方法很厉害，核心如下。想要优质男人追求你，首先你得认识他。陌生情况下，不认识的男人不可能追求你。因此，第一步是邂逅。

### 邂逅方法

1. **社交邂逅**：在社交场合偶遇，如公司团建、酒会、游乐场等。
2. **介绍邂逅**：通过朋友介绍认识。

### 社交场合邂逅技巧

- **释放意愿度**：用眼神、微笑等方式表达好感。
- **给予空间**：避免周围都是朋友，给对方接近你的机会。
- **兴趣指标**：通过聊天、动作等传达对他有好感。

### 建立亲密关系

- **刷存在感**：通过聊天、见面等方式让他对你熟悉。
- **避免陷入友谊区**：保持吸引力，不要只成为朋友。

### 建立吸引力

- **精神魅力**：独特、匹配他的空缺。
- **身体魅力**：身材、外貌、穿着等。

### 推拉技巧

- **推**：保持不可得性，让他觉得你不容易得到。
- **拉**：释放可得性，让他知道你有一定可得性。

通过以上方法，让他追逐你，最终追到你，在感情中占高位。

🎼 本节课中我们一起学习了如何让你喜欢的男生倒追你，掌握了邂逅、建立亲密关系、建立吸引力以及推拉技巧。希望对你有所帮助。
