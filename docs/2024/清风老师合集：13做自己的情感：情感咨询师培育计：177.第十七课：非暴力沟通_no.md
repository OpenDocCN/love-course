# 清风老师合集：13做自己的情感：情感咨询师培育计：177.第十七课：非暴力沟通_no

在本节课中，我们将要学习一种重要的沟通方法——非暴力沟通。我们将探讨其核心概念、四个基本要素，以及如何将其应用于实际生活和咨询场景中，以帮助我们建立更和谐、充满尊重与爱的关系。

上一节我们介绍了叙事疗法，学习了如何通过故事重写来发现积极能量。本节中我们来看看，在运用这些方法时，沟通扮演着何等关键的角色。今天要学习的“非暴力沟通”，正是一门关于如何有效、平和沟通的学问。

我们常听到肢体暴力、家庭暴力等概念，这些伤害直观可见。但实际上，语言暴力和冷暴力对关系的破坏性往往更大。

我们很少认真分析语言暴力究竟是什么。通常认为辱骂是语言暴力，但更多时候，它源于污蔑、嘲笑和诋毁等行为，这些行为会让人感到极度不适。

例如，家庭中可能出现这样的对话：妻子对丈夫说：“你怎么总是把家里弄得这么乱，跟个垃圾堆似的，我怎么待？” 这种情况下，丈夫会感到被污蔑和错怪，进而可能用他自己的方式回击，导致关系恶化。

真正的语言暴力常表现为不恰当的评价。家庭问题常因缺乏良好沟通而产生。言语上的指责、嘲讽、否定、说教、随意打断、不予回复、以及武断的评价和结论，给我们带来的精神与情感创伤，有时比肉体伤害更令人痛苦。

这些无心或有意的语言暴力，使我们与最亲密的人之间变得冷漠、隔阂甚至敌视。对于这种暴力，非暴力沟通是一个非常重要的解决手段。

## 什么是非暴力沟通？ 🤔

非暴力沟通是英文“Nonviolent Communication (NVC)”的中文翻译。它认为，在交流过程中，通过专注于自己和他人的**观察**、**感受**、**需要**和**请求**，致力于满足彼此生命的需要，可以减少退缩、申辩、对抗等造成隔阂的反应。培养彼此的尊重与爱是其核心宗旨。

非暴力沟通的创始人是马歇尔·卢森堡博士。由于在促进人类和谐共处方面的突出成就，他于2006年获得了地球村基金会颁发的和平之桥奖。卢森堡博士早年师从心理学大师卡尔·罗杰斯，后来发展出极具启发性和影响力的非暴力沟通原则与方法。

圣雄甘地曾用“非暴力”一词，指代暴力消退之后自然流露的爱。非暴力沟通不只是一种沟通方式，更是一种持续不断的提醒，帮助我们更好地找到自己真正需要的东西。

## 非暴力沟通的四要素 📝

在马歇尔·卢森堡博士创建的非暴力沟通中，有四个基本要素：**观察**、**感受**、**需要**和**请求**。

以下是这四个要素的具体说明：

*   **观察**：仔细观察正在发生的事情，并清楚地说出观察结果，而不掺杂评价。
*   **感受**：表达对于所观察到的现象，我们内心的真实感受。
*   **需要**：说出是哪些需要（或价值、愿望等）导致了那样的感受。
*   **请求**：提出具体的、正向的请求，希望对方如何行动来满足彼此的需要。

让我们用之前的例子来说明。妻子说：“你把家里搞得像垃圾堆一样。” 如果用非暴力沟通来表达，可以是：“我看到咖啡桌上有两只脏袜子，电视机旁边有三只脏袜子（**观察**）。我挺不高兴的（**感受**），因为我看重整洁（**需要**）。你是不是愿意把袜子拿到房间或者放进洗衣机呢（**请求**）？”

记住，一个完整的非暴力沟通包含：**观察**、**感受**、**需要**和**请求**。

## 是什么蒙蔽了我们的爱？ 💔

马歇尔·卢森堡博士说过：“我相信人天生是热爱生命的，乐于互助。可是，究竟是什么使我们难以体会到心中的爱，以至于互相伤害？又是什么，让有些人即使在充满敌意的环境中也能心存爱意？”

博士认为，人的语言沟通本为满足某种愿望，但若倾向于忽视人的感受和需要，就会造成彼此的疏远和伤害。他称这种沟通为“异化的沟通方式”。

以下是造成这种异化沟通的几个主要原因：

*   **道德评判**：用我们自己的标准来评价别人。如果对方行为不符合我们的价值观，我们就可能从道德上认为对方是不道德的。例如：“你这个人太自私了！”
*   **进行比较**：比较是评判的另一种形式，也是生活中诸多烦恼的开始。例如在亲密关系中：“我前任就不会像你这么粗心。”
*   **回避责任**：我们常使用“不得不”、“你让我”等词汇，淡化了自己应负的责任。例如：“你这样做让我很痛苦。”
*   **强人所难**：对别人的要求常常暗含着威胁。例如：“你如果继续这样，我就不理你了。” 这会让对方感到被胁迫。

正如马歇尔博士所言，人天生热爱生命，乐于互助，但异化的沟通方式让我们感受不到这种爱和需要。

## 如何实践非暴力沟通？ 🛠️

还记得我们提到的四个要素吗？接下来，我们逐一深入探讨如何实践它们。

### 1. 区分观察与评论 👀

我们要仔细观察正在发生的事情，并能够清楚地表述观察的结果。但很多时候，我们说出的是自己的评价，而非观察。

例如，一位来访者说：“我觉得他越来越不爱我，也越来越懒。” 当被问及具体表现时，她说：“他从来都不关心我……他不注意东西的摆放，弄得到处乱糟糟的。”

不难发现，这里很多是她的评论（如“懒”、“不关心”），而非观察（如“把袜子扔在沙发上”、“最近几次做饭时他在看电视”）。非暴力沟通强调区分观察和评论的重要性。将两者混为一谈时，听者容易产生逆反心理和情绪冲突。

印度哲学家克里希那穆提曾说：“不带评论的观察是人类智力的最高形式。” 学会区分观察和评论并不容易，但至关重要。

### 2. 体会与表达感受 💗

在客观观察之后，第二步是正确表达自己的感受。我们常遇到的问题是，将**感受**和**想法**混淆起来。

我们的大脑每天产生无数想法，在沟通中，我们常常直接把想法说出来，却很少关注内心的感受。例如，我们常说：“我觉得你不爱我了。”（这是**想法/评价**）而不是说：“你最近在我做饭时总是看电视，我感到很难过，这让我觉得你好像不那么在乎我。”（这包含了**观察**和**感受**）

“我觉得”常常不是在表达感受，而是在表达想法或评价。每一个想法背后都隐藏着我们的感受。我们可以学习使用更直接的词汇来表达感受，如：高兴、伤心、害怕、沮丧、温暖、孤独等。

### 3. 看见感受背后的需要 🔍

在我们学会表达感受之后，就需要探寻感受的根源——我们内心哪些需要得到了满足或未被满足。

很多时候，我们说不出自己真正的需要。例如，一位疲惫的女性可能说：“我一整天都没休息，回家还要洗衣做饭，你都不在乎我！” 但她真正的需要可能是：“我需要休息”或“我需要你分担一些家务”。说不出的需要，会让对方感到困惑。

非暴力沟通鼓励我们直接、清晰地表达自己的需要，这更容易让对方理解并做出积极的回应。

### 4. 提出明确的请求 🙏

非暴力沟通的前三步是关联的：观察行为，表达感受，梳理需要。第四步就是基于这些，向对方提出具体的、正向的**请求**。

什么样的请求更容易得到积极回应呢？关键在于：**具体、正向，并给予对方选择的权利。**

以下是两种表达方式的对比：
*   **命令式/模糊请求**：“你周末要陪我逛街。” 或 “我希望你对我好一点。”
*   **非暴力沟通式请求**：“亲爱的，最近工作挺忙的，我想周末放松一下，你可不可以陪我去逛街？” 或 “最近你回家很晚，我一个人在家感到害怕，明天你可不可以十点前回家？”

前者听起来像命令或模糊的要求，容易引发抵触。后者则具体、正向，并尊重了对方的意愿。非暴力沟通的宗旨不是为了改变他人来迎合我们，而是帮助我们在诚实和倾听的基础上，与人建立更好的联系。我们希望人们的改变和行动是出于对彼此的爱。
