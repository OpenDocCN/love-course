# 钱进社课程07：职场与生意场，“我要什么”的思维误区 🚫

在本节课中，我们将探讨一个在职场和生意场中常见的思维误区——过度关注“我要什么”，而忽略了“他要什么”。我们将通过具体的商业案例和职场情境，分析这种思维如何导致弯路，并学习如何通过换位思考来达成目标。

## 概述：从“我要”到“他要”的思维转变

人的本能是从自身出发看待问题和判断事物。我们对职场或生意产生怨气，往往是因为过于侧重自我意识，而现状未能满足预期。实际上，要达到“我要什么”的结果，必须以“他要什么”为过程。过程走不好，结果就达不到。

上一节我们介绍了思维误区的根源，本节中我们来看看如何在具体场景中应用正确的思维。

## 一、 生意场：从客户角度出发

许多人在做生意时，害怕广告过于明显会引起客户反感。这个想法本身没有错，但关键在于是否真正分析过客户群体。

以下是需要自我审视的几个问题：

*   客户从哪里来？（微信、淘宝、电商平台等）
*   他们为什么留在你身边？
*   他们的阶层和消费观是什么？
*   你是否通过实际数据验证过对客户的判断？

例如，在小红书、抖音等平台，有些内容直接展示产品，没有软性植入，却吸引了大量精准客户咨询。这说明部分群体并不讨厌硬广，反而认为简单直接的方式节省了时间。

然而，并非所有产品采用这种方式都有效。你需要分析以下三点：

1.  **产品风格**
2.  **产品价位**
3.  **产品定位**

分析必须站在客户角度，做到精准落地，而非主观臆断。例如，高端客户通常不会花整晚时间抢直播秒杀，但他们也需要购物。他们的需求可能是一对一的专属服务，以及便捷、省时的购买体验。

许多商家内心只想让产品“爆炸”和“疯传”，目的是赚更多钱。但如果没有分析客户真正需要什么情境和服务，所谓的“爆款”可能正在扼杀客户的消费意愿，因为他们厌烦你的营销模式。

## 二、 案例分享：微商经历的思维实践

接下来，我将分享一段微商经历，重点在于其中的逻辑框架内核。

我当时运作一个单品，在不到一年时间里实现了近400万的现金流。这个结果印证了我对整个过程的判断。

几年前，一款名为 **`SP-68`** 的打底裤在微商圈很火。在它通过品牌定位和宣传爆火之前，我已经在运作这款产品，但我选择的是**走量和整合商家**的路线，并注册了商标。

我为什么不做品牌？

1.  **市场环境**：中国市场对原创的保护不足，而品牌的根基在于持续的原创能力。在一个复制性很强的环境里，谈品牌意义不大。
2.  **成本与时间**：建立被认可的品牌感需要时间和资本辅助。销量高不等于品牌，那可能只是商标。
3.  **产品特性**：当产品**可复制性**很强时，不适合投入大量资本做品牌，否则极易陷入同质化竞争和价格战。

因此，我的定位非常明确：**快速整合商家资源，赚一笔快钱，短平快**。

我为什么确信能整合商家？因为我也是生意人，我清楚生意人在各个阶段的瓶颈和突破点。我的产品能帮助他们快速吸引更多客户，而且品控好，售后问题少。

这款打底裤的卖点是弹力大，**一个尺寸**适合所有人，没有设计感反而让它能“通吃”。对商家而言，这意味着：

*   **无压货压力**
*   **无尺码退换压力**
*   **客户顺带购买，消耗品属性复购率高**
*   **不影响主业务现金流**

“我要什么”的思维是赚钱。“他要什么”的思维是：**获得能辅助打开市场、赢得口碑、且无资金压力的产品**。

许多微商代理模式要求先交钱拿货，这会让新手担忧。关键在于洞察并解决他人对资金流转的担忧。无论金额大小，解决了这个问题，才能快速整合资源。

所以，“他要什么”就是：**让别人赚钱，让别人感到爽快和惊喜**。这是你首要的思考方向。

当时，我整合的不仅是素人商家，还包括一些视频网站KOL、微博小网红，甚至上海电视台美妆节目的嘉宾。我能整合他们，不仅仅因为产品靠谱，更关键的是：

*   我提前洞悉了不同群体的不同需求（如供应链、代发货、贴牌等）。
*   在对方开口前，我已针对不同群体分类，并准备好了解决方案。
*   在沟通中能精准切入要点，展现专业性。

**专业性是给人最大的安全感**，也是促成合作的基础。赚快钱需要**洞察力、执行力、做事逻辑**三者缺一不可，一旦延缓就会错过商机。

后来，`SP-68`果然有资本进入想做品牌，但在此之前我已赚到现金流。我并不气恼，因为：

1.  我们不是同一赛道。
2.  我预判这款产品不适合大资本长期运作，利润空间会被竞争压缩。
3.  高投入、高流水但净利润低的模式，并非我追求的，它还会占用发展多元生意的精力。

## 三、 思维应用：职场与自我审视

这一套“他要什么”的思维同样适用于职场和其他领域。

**对于职场晋升：**
总是思考“我为什么没晋升”的人，往往忽略了上级的需求。如果你认为晋升的同事只会拍马屁，那恰恰说明你的领导需要这样的人。你需要观察和分析：
*   上级需要什么样的人成为左膀右臂？
*   晋升的同事具备哪些特质和行事方式？
一味坚持自我，情绪化地看不起他人，就无法达成“我要晋升”的结果。

**对于面试：**
自以为优质却面试失败，很可能是因为没有满足用人单位的真实需求。对公司而言，找到一个合适的人并不容易。如果你真的非常匹配，对方不会拒绝。问题可能出在**你没有展现出对方需要的能力或特质**。

**对于自媒体与营销：**
许多内容是“自嗨型”的。发布前，请换位思考：如果你是观众，会想看这个内容吗？你是否认真研究过同行数据，分析过爆款内容与普通内容的区别？机械堆砌、不动脑筋，然后抱怨没流量，是行不通的。

**对于商家与微商：**
请定期以消费者视角审视自己：
*   你的朋友圈作为消费者想看吗？
*   你的产品和展示能让自己动心吗？
*   你的服务能感动自己吗？
如果没有这种换位思考，就很难真正调整进步。

## 总结与核心要点

本节课我们一起学习了“我要什么”思维的局限性，以及如何通过“他要什么”的思维过程来达成目标。

**核心公式：**
> **我要什么 (结果) = 他以他要什么为基础的过程 (成功执行)**

**关键要点总结：**

1.  **思维转变**：将焦点从自我需求 (**`Self.Want`**) 转移到对方需求 (**`Other.Need`**)。
2.  **生意本质**：想客户所想，甚至想客户所未想，这是专业能力的体现。
3.  **底层逻辑**：商业架构需要积累，非一蹴而就。它包括持续的研究、实践和换位思考。
4.  **通用性**：此逻辑适用于生意、职场、人际交往等各个方面。
5.  **行动建议**：在追求任何需要他人协作或认可的目标前，先问自己：“**对方的需求和顾虑是什么？我如何解决？**”

情商的关键不在于油嘴滑舌，而在于**习惯性地站在对方角度思考其需求**。通过满足“他要什么”的过程积累，你自然会更接近“我要什么”的结果。

记住，如果你要的东西需要他人给予或赋能，不先搞定“他要什么”，你的“我要什么”就无从实现。
