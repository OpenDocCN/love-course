# 钱进社 - 成为优质且最具竞争力的【社会人】：07、职场和生意场，“我要什么”的思维，会逼你上大弯路

## 概述

在本节课中，我们将探讨“我要什么”的思维模式在职场和生意场中的局限性，并介绍如何通过关注“他要什么”的思维模式来提升竞争力。

## 核心概念

**公式**：成功 = 关注“他要什么” + 关注自身需求

## 1. 职场思维

**以下是职场思维的一些要点**：

*   **关注自身需求**：许多人在职场中只关注自己的需求，而忽略了上级和同事的需求。
*   **缺乏换位思考**：没有站在上级和同事的角度思考问题，导致沟通和合作困难。
*   **结果导向不足**：只关注短期目标，而忽略了长期发展。

**以下是职场思维的改进方法**：

*   **关注上级需求**：了解上级的期望和目标，并努力满足他们的需求。
*   **换位思考**：站在上级和同事的角度思考问题，提高沟通和合作效率。
*   **结果导向**：关注长期发展，制定并实现长期目标。

## 2. 生意场思维

**以下是生意场思维的一些要点**：

*   **关注自身需求**：许多人在生意场中只关注自己的利益，而忽略了客户的需求。
*   **缺乏市场调研**：没有深入了解客户的需求和喜好，导致产品或服务无法满足客户。
*   **营销方式单一**：只采用一种营销方式，而忽略了多种营销手段的结合。

**以下是生意场思维的改进方法**：

*   **关注客户需求**：深入了解客户的需求和喜好，开发满足客户需求的产品或服务。
*   **市场调研**：定期进行市场调研，了解市场趋势和竞争对手情况。
*   **多种营销手段结合**：采用多种营销手段，提高品牌知名度和市场占有率。

## 3. 案例分析

**以下是一个案例分析**：

**案例**：某微商品牌在推广过程中，只关注自身利益，大量发送硬广，导致客户反感，最终导致品牌口碑受损。

**分析**：该品牌在推广过程中，没有关注客户需求，只关注自身利益，导致客户反感，最终导致品牌口碑受损。

**改进建议**：

*   **关注客户需求**：了解客户需求，开发满足客户需求的产品或服务。
*   **软性植入**：采用软性植入的方式，提高品牌知名度和美誉度。
*   **客户服务**：提供优质的客户服务，提高客户满意度。

## 总结

本节课中，我们探讨了“我要什么”的思维模式在职场和生意场中的局限性，并介绍了如何通过关注“他要什么”的思维模式来提升竞争力。希望同学们能够将所学知识应用到实际工作和生活中，取得更好的成绩。
