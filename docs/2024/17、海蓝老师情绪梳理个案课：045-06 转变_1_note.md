# 海蓝老师情绪梳理个案课：045-06 转变_1 🌟

## 概述

在本节课中，我们将学习如何通过情绪梳理个案课来转变心态，面对生活中的挑战。

## 核心概念

**公式**： 情绪梳理 = 自我认知 + 行动计划 + 持续陪伴

## 教程内容

### 1. 自我认知

**以下是自我认知的步骤**：

- **钉住最坏的结果**： 将最坏的结果牢记在心，以此激励自己珍惜当下。
- **反思自身**： 深入思考自己的内心世界，了解自己的需求和愿望。
- **积极行动**： 制定行动计划，将想法付诸实践。

### 2. 行动计划

**以下是制定行动计划的步骤**：

- **安排妈妈的旅游**： 为妈妈安排一次愉快的旅行，让她感受到关爱。
- **成家立业**： 关注自己的事业和家庭，努力实现人生目标。
- **与朋友沟通**： 与朋友分享自己的想法，寻求支持与帮助。

### 3. 持续陪伴

**以下是持续陪伴的步骤**：

- **关注小清晨**： 陪伴小清晨，给予他关爱和支持。
- **定期联系**： 每周至少与小清晨联系一两次，建立稳定的联系。
- **勇敢面对**： 用自己的勇敢、智慧、爱和判断力去陪伴小清晨，帮助他度过难关。

## 总结

本节课中，我们学习了如何通过情绪梳理个案课来转变心态，面对生活中的挑战。通过自我认知、制定行动计划和持续陪伴，我们可以更好地应对生活中的困难，实现人生目标。希望同学们能够将所学知识应用到实际生活中，不断提升自己。
