# 情绪价值课-【阿娇】：第1节 情绪价值是什么

在本节课中，我们将探讨情绪价值的概念及其在人际关系中的作用。

## 情绪价值定义

情绪价值是指一个人在一段关系中提供的正面情绪与负面情绪之间的差额。我们可以用以下公式来解释：

**情绪价值 = 情绪收益 - 情绪成本**

其中：
- **情绪收益**：指在一段关系中获得的正面情绪，如快乐、满足、信任等。
- **情绪成本**：指在一段关系中感受到的负面情绪，如争吵、冷漠、指责等。

## 情绪价值的影响

### 情绪价值高的影响

- **正面影响**：高情绪价值有助于巩固感情，使人们在面对矛盾时更愿意解决问题，而不是选择分开。
- **生理影响**：感受到正面情绪时，大脑会产生多巴胺，使人感到快乐和兴奋。

### 情绪价值低的影响

- **负面影响**：低情绪价值可能导致关系破裂，使人们感到压力和疲惫。
- **三种情况**：
  1. 对方看不上你，不愿意与你相处。
  2. 你没有足够的能力和情商让对方开心。
  3. 你提供了负面情绪或过度索取，导致对方感到压力。

## 如何成为高情绪价值的人

- 提供情绪收益，减少情绪成本。
- 减少让对方情绪不好的事情，减少互相伤害的话。
- 减少情绪消耗和感情消耗。
