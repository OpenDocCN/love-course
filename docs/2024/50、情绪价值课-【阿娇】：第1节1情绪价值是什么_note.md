# 情绪价值课 - 第1节：情绪价值是什么？💡

在本节课中，我们将要学习情绪价值的核心定义、其构成要素，以及它如何影响我们的人际关系。我们将通过一个简单的公式来理解情绪价值，并分析高情绪价值与低情绪价值分别带来的不同影响。

## 什么是情绪价值？🤔

上一节我们介绍了课程主题，本节中我们来看看情绪价值的具体定义。

情绪价值是当今社会经常被提及的一个概念。我们可以用一个公式来解释它：

**情绪价值 = 情绪收益 - 情绪成本**

### 情绪收益

情绪收益是指你在一段关系中获得的正面情绪体验。

以下是常见的情绪收益示例：
*   开心
*   快乐
*   鼓励
*   体贴
*   彼此信任
*   彼此理解

这些都能为你提供满足感和积极的情绪。

### 情绪成本

情绪成本是指你在一段关系中感受到的负面情绪，以及为获得情绪收益所需付出的代价。

以下是情绪成本的构成部分：
1.  **负面感受**：例如争吵、冷漠、嫌弃、互相质疑、互相指责等让人不舒服的感受。
2.  **付出的代价**：包括时间、精力、金钱以及感情投入。

了解了情绪收益和情绪成本后，用前者减去后者，得到的结果就是你为他人提供的情绪价值。

## 情绪价值的影响🔍

知道了情绪价值是什么以后，我们再来讲讲它会给我们的生活带来什么样的影响。

### 高情绪价值的积极影响

在我们的大脑中，感受到正面情绪时会分泌一种叫多巴胺的物质，它会让人感到快乐、亢奋，甚至产生期待和上瘾的感觉。

大脑为了维持这种喜悦感，会需要更多的多巴胺，从而使你对提供这种情绪价值的人产生依赖、期待和信赖。你会觉得和对方在一起的感受很好，甚至希望一直如此。

这有助于巩固你们的感情。即使未来遇到矛盾或产生分开的念头，你们也会因为曾经相处的美好感觉和快乐回忆，而愿意主动解决问题、修复关系，不愿失去这份美好。这就是高情绪价值带来的积极作用。

### 低情绪价值的负面影响

情绪价值低的情况通常分为三种。

以下是三种常见类型：
1.  **主观不愿提供**：你根本看不上对方，不想提供情绪价值，甚至不给好脸色。对方自然会知难而退，无法从你这里获得开心和满足。
2.  **能力不足或不被认可**：你自身能力或情商有限，无法让对方开心。或者对方一开始就看不上你，不在意你的付出。即使你费尽心思讨好，对方也可能根本不领情。
3.  **收益被成本抵消**：你有能力让对方快乐，对方也很受用。但之后你又提供了大量负面情绪，或不断索取陪伴和物质，导致对方认为从你身上获取情绪价值的成本太高了。与你相处虽然开心，但也很累。

一正一负，功过相抵，对方会觉得不赚不赔。但随着时间推移，你能提供的新鲜感越来越少。当感情从热恋进入平淡期，彼此的探索欲和好奇心下降，又恰逢磨合期，争吵和情绪消耗增多。

于是，对方开始感受到，从你身上获得的快乐已经远远小于承受的痛苦。他觉得你提供的情绪价值在减少，而他付出的情绪成本却在增加，开始觉得相处“不划算”。

如果你在此阶段仍不知如何控制和调整状态，一方面提供的情绪收益有限，另一方面彼此情绪消耗巨大，对方很快就会感到压力，想要逃离，甚至拒绝你再提供的任何情绪价值，也就是“烦了”。

这种情况很常见。例如，很多受原生家庭影响的人，小时候做对事很少被表扬，做错事却总被批评指责。在这种环境下长大，容易变得过分敏感或自卑，对负面情绪反应激烈。

他们在亲密关系中，更容易关注双方的负面情绪和问题。对方做得好，他们觉得理所当然；对方稍有没做好或彼此有点问题，他们就容易抓着不放。

时间久了，你给别人带来的情绪收益可能没有增加，但你们之间的情绪成本却越来越高。

大家一定要注意，当我们与他人相处时，负面体验越来越多，就很容易陷入不断争吵、彼此伤害或产生不可调和矛盾的漩涡。这种情况又会产生更多负面情绪，加深彼此的消耗，形成一个恶性循环，最终导致关系破裂。

## 如何成为高情绪价值的人？✨

说到这里，相信大家已经或多或少明白了其中的道理。

就像我们前面说的，一方面要管理好情绪收益和情绪成本，另一方面也要提升所提供的情绪价值的数量和质量。

核心在于，在亲密关系中，少做一些让对方情绪不好的事，少说一些互相伤害的话，减少彼此的情绪消耗和感情消耗。

## 总结📝

本节课中，我们一起学习了情绪价值的核心公式 **（情绪价值 = 情绪收益 - 情绪成本）**，并详细探讨了高情绪价值与低情绪价值分别对人际关系产生的深远影响。关键在于增加正面情绪体验，同时减少负面情绪消耗。
