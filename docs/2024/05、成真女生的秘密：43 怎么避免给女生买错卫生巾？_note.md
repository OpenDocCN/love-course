# 课程05：成真女生的秘密 - 如何避免买错卫生巾？🩸

在本节课中，我们将要学习卫生巾（俗称“姨妈巾”）的基本知识，包括其材质、类型、尺寸及适用场景。掌握这些信息，可以帮助你在需要时为女生选购合适的卫生巾，避免买错。

## 卫生巾的常见材质🧵

上一节我们介绍了课程概述，本节中我们来看看卫生巾的常见材质。卫生巾的表面材质主要分为两种，不同女生对材质的偏好和适应性不同。

以下是两种主要的表面材质：

*   **棉质表面**：触感柔软，透气性较好，适合皮肤敏感的女生。
*   **网制表面**：通常更干爽，但部分女生使用后可能出现皮肤过敏现象。

因此，在购买前询问对方“你习惯用棉质的还是网制的？”是很有必要的。

## 卫生巾的类型与尺寸区分📏

了解了材质选择后，我们来看看卫生巾如何按使用时间和尺寸进行分类。卫生巾主要分为日用和夜用两种类型，其核心区别在于长度和设计，以适应不同的身体姿态。

以下是具体的分类说明：

*   **日用卫生巾**：适用于白天站立或坐着的直立状态。长度通常为**240毫米**左右，尺寸相对较小。
*   **夜用卫生巾**：专为夜间躺着睡觉设计。由于平躺时更容易发生侧后方渗漏，因此夜用款更长，尾翼也更宽大。长度一般从**280毫米**起，常见的有290、350、400、420毫米等规格。多数女生会选择最长的一款以确保安全。
*   **迷你卫生巾**：长度很短，例如171、180、190毫米。通常在生理期即将结束、流量很少时（如第7天或第8天）作为保险使用。
*   **裤型卫生巾**：形状类似内裤，穿脱方便，用完即扔。适用于流量特别大的日子，或在进行户外等大运动量活动时使用。

## 总结与核心要点🔑

本节课中我们一起学习了卫生巾的基础知识。关键在于理解：**材质（棉质/网制）** 关乎个人舒适与过敏可能；**类型与尺寸（日用/夜用/迷你）** 则对应不同的使用场景和身体姿态（直立/平躺）。记住这些要点，并在购买前进行简单沟通，就能有效避免买错，更好地照顾对方的需求。
