# 05、成真女生的秘密：43 怎么避免给女生买错卫生巾？_note

在本节课中，我们将学习卫生巾（俗称“姨妈巾”）的基本知识，包括其材质、类型、尺寸及适用场景。掌握这些信息，可以帮助你在需要时为女生选购合适的卫生巾，避免买错。

## 卫生巾的常见材质🧵

卫生巾的表面材质主要分为两种：棉质和网制。不同女生对材质的偏好不同，有些女生使用网制卫生巾可能会出现皮肤过敏。

因此，在购买前，最好询问对方：“你平时用的是棉质的还是网制的？有没有材质偏好？”

## 卫生巾的类型与尺寸区分📏

上一节我们介绍了材质，本节中我们来看看卫生巾如何按使用时间和尺寸进行分类。卫生巾主要分为日用型和夜用型。

*   **日用卫生巾**：适用于白天活动时使用。人在白天通常处于站立或坐姿（`直立状态`），因此日用卫生巾的长度一般较短，常见规格为**240毫米**。
*   **夜用卫生巾**：专为夜间睡眠设计。人躺下后可能翻身，液体容易向后流动。因此，夜用卫生巾更长更宽，以提供更好的后方防护，防止泄漏。其长度通常从**280毫米**起，还有290、350、400、420毫米等更长的规格。

以下是夜用卫生巾更长的核心原因：
> 夜用卫生巾的尾翼通常又长又大，目的是防止夜间睡眠时发生侧漏。

大部分女生会选择最长的一款夜用卫生巾，以最大程度防止泄漏到衣物或床单上。

## 其他特殊类型的卫生巾🩲

除了常见的日用和夜用型，还有一些特殊设计的卫生巾以满足特定需求。

*   **裤型卫生巾**：形状类似内裤，穿完后可直接丢弃。适用于月经量特别大的日子，或在进行户外等运动量较大的活动时使用。
*   **迷你卫生巾**：长度很短，常见规格有171、180、190毫米。适用于生理期即将结束的日子（例如第7天或之后），量很少但为确保安全而使用。
