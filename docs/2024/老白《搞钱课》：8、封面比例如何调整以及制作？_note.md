# 老白《搞钱课》：8、封面比例如何调整以及制作？_note

在本节课中，我们将学习如何为短视频平台（如抖音）制作符合要求的3:4比例封面，并利用修图软件完成封面的设计与模板化，以提高后续内容发布的效率。

## 🖼️ 封面比例与基础画布设置

上一节我们介绍了封面制作的重要性，本节中我们来看看具体的操作步骤。首先，需要明确平台对封面比例的硬性要求。

抖音博主主页的封面显示比例为3:4。因此，我们制作封面的第一步是创建一个3:4比例的黑色画布作为背景。

以下是具体操作步骤：

1.  打开修图软件（如“醒图”）。
2.  点击“导入”，选择“添加画布”。
3.  在画布颜色中选择“黑色”。
4.  在画布比例中选择“3:4”。
5.  点击“下载”保存这个黑色背景图。

完成基础画布设置后，我们就可以开始添加图片素材了。

## 🎨 封面图片的选取与处理

接下来，我们需要为封面选择一张有“氛围感”的主图。可以从网络平台（如小红书）搜索“现实主义照片”或“超现实主义”等关键词来寻找灵感。

假设我们已经将选好的图片保存到手机相册。现在，将其导入并调整至合适的位置和大小。

以下是图片处理步骤：

1.  在软件中再次点击“导入”，选择“添加画布”。
2.  向右滑动，找到并点击“导入图片”。
3.  从相册中选择你准备好的主图。
4.  选中图片后，点击下方工具栏的“构图”选项。
5.  将构图比例设置为“正方形”，这样裁剪后效果更佳，点击“确定”。
6.  使用双指在屏幕上 pinch（捏合）手势，将图片放大或缩小，并拖动到你觉得合适的位置。

图片处理完成后，封面的视觉主体就确定了。下面，我们需要为其添加文字信息。

## ✍️ 封面文字的添加与排版

一个完整的封面通常包含标题和账号信息。我们将分步添加并调整这些文字元素。

首先，添加主标题。例如，标题可以是“最近看到的醍醐灌顶的一段话”。

以下是添加与调整标题的步骤：

1.  点击工具栏的“文字”按钮，输入你的标题内容。
2.  点击“样式”，在“排列”选项中选择“居中”对齐。
3.  将文字拖动到封面底部合适的位置。
4.  点击“字体”，根据个人喜好选择字体样式，例如“新青年体”或“思源黑体”等。
5.  如需修改文字颜色，在“样式”中选择“颜色”进行更改。

如果标题较长，可以将其分为两行。点击文字框旁的“复制”（或“加一”）按钮，可以快速创建一个格式相同的文字层，然后修改其内容即可。

接下来，添加代表账号信息的副标题，通常是英文或中文的“破折号”格式。

以下是添加账号信息的步骤：

1.  新建一个文字层，输入你的账号名，例如“一盒阅读”。
2.  在账号名前加上破折号，格式如“—— 一盒阅读”。
3.  调整该文字的大小和位置，通常将其放在主标题下方。
4.  英文字体可以保持默认，或选择一款简洁的字体。

所有元素调整完毕后，一个完整的封面就设计好了。为了提升未来制作封面的效率，我们可以将其保存为模板。

## 💾 封面模板的创建与复用

将设计好的封面保存为模板，之后只需修改文字内容即可快速生成新封面，无需重复调整布局和样式。

以下是创建与使用模板的步骤：

1.  点击右上角的“存为模板”按钮。
2.  为模板命名并保存。
3.  下次需要制作封面时，直接导入最初的3:4黑色画布。
4.  在软件内向左滑动，进入“我的”页面，找到你保存的模板。
5.  点击使用该模板，你只需要修改其中的文字内容（如标题和日期）即可。

对于不再需要的旧模板，可以在模板库中长按并选择删除。

封面图片制作完成后，最后一步是将其正确添加到视频中。

## 🎬 将封面添加至视频

在视频剪辑软件中，需要确保封面以正确的3:4比例显示，并检查封面文字与视频内字幕的协调性。

以下是添加封面到视频的步骤：

1.  在剪辑软件中，先将视频比例设置为“3:4”并确认。
2.  点击“设置封面”功能，选择“从相册导入”。
3.  选中你刚刚制作好的封面图片，点击“确认”并“保存”。
4.  此时，视频画面会变为3:4比例。你需要将视频比例改回常规的“4:3”或“9:16”（根据平台要求）。
5.  返回后你可能会发现，视频开头的字幕字体变小了。这时，需要手动调整：
    *   点击“文本”，选中第一句字幕。
    *   点击“编辑”，进入“样式”调整字体大小。
    *   **关键点**：务必**取消勾选**“应用到所有字幕”的选项，这样调整就只会应用于当前选中的字幕，而不会影响封面上的文字。

调整完毕后，播放预览一遍，确保封面显示正常，且视频开头的声音与画面同步。

本节课中我们一起学习了短视频封面的完整制作流程。我们从明确3:4的平台比例要求开始，逐步完成了从创建画布、选取处理图片、添加排版文字，到最终保存模板并正确添加到视频中的全部操作。掌握这套方法，你可以高效地制作出风格统一、符合要求的视频封面，为你的内容增添专业吸引力。
