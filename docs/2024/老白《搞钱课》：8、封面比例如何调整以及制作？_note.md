# 老白《搞钱课》教程：8、封面比例调整及制作

🎬 在本节课中，我们将学习如何调整封面比例以及制作封面。

## 封面比例调整

以下是封面比例调整的步骤：

1. **选择图片**：在小红书搜索“现实主义照片”，选择一张氛围感强的图片。
2. **导入图片**：打开图片编辑软件，导入图片。
3. **设置比例**：选择3比4的比例，这是短视频封面的标准比例。
4. **下载图片**：下载调整后的图片。

## 封面制作

以下是封面制作的步骤：

1. **添加画布**：导入图片后，添加一个3比4比例的黑布。
2. **导入封面图片**：导入你选择的封面图片。
3. **调整图片**：将封面图片调整成正方形，并放置在合适的位置。
4. **添加文字**：在封面添加文字，如标题和作者名。
5. **设置字体和样式**：选择合适的字体和样式，并调整文字颜色和大小。

## 封面模板

以下是封面模板的制作步骤：

1. **保存模板**：将制作好的封面保存为模板。
2. **编辑模板**：下次制作封面时，可以直接导入模板进行编辑。

## 添加封面

以下是添加封面的步骤：

1. **选择比例**：选择3比4的比例。
2. **设置封面**：从相册选择封面图片。
3. **确认并保存**：确认封面设置，并保存。

## 总结

本节课中，我们一起学习了如何调整封面比例以及制作封面。希望这些知识能帮助你制作出精美的封面。
