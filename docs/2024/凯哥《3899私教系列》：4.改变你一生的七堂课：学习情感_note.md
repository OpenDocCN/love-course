在本节课中，我们将要学习“情感”这一核心概念。情感并非不可捉摸的感觉，而是一种可以被理解、分析和管理的心理与生理反应。掌握情感运作的规律，能帮助我们更好地认识自己、处理人际关系并做出明智决策。

## 情感是什么？

上一节我们介绍了课程的主题，本节中我们来看看情感的本质定义。

情感是生物体对外部或内部刺激产生的复杂反应模式。它包含**主观体验**、**生理唤醒**、**认知解释**和**行为表达**四个基本成分。可以用一个简化的公式来描述其产生过程：

**刺激 -> 认知评估 -> 生理反应 + 主观感受 -> 行为**

## 核心情感类型

了解情感的本质后，我们需要认识几种基本的情感类型。心理学中常将情感分为几种核心类别。

以下是六种普遍存在的基本情感：

*   **快乐**：一种愉悦、满足或欣喜的感受。
*   **悲伤**：与失去、失望相关联的痛苦感受。
*   **愤怒**：当目标受阻或受到不公对待时产生的强烈情绪。
*   **恐惧**：对感知到的威胁或危险产生的警觉反应。
*   **厌恶**：对令人不快的事物产生的排斥感。
*   **惊讶**：对突发或意外事件产生的短暂反应。

## 情感的功能

认识了基本类型，你可能会问：情感对我们有什么用？本节将探讨情感的核心功能。

情感不是生活的干扰，而是至关重要的生存与社交工具。

以下是情感的三个主要功能：

*   **适应功能**：情感帮助我们快速评估环境（如恐惧促使躲避危险），是进化留下的生存机制。
*   **动机功能**：情感驱动行为（如渴望引发追求，愤怒激发反抗），为行动提供能量和方向。
*   **社交功能**：情感通过表情、语调传递信息，促进人际沟通、建立联结与理解。

## 如何管理情感？

理解了情感的价值，接下来我们将学习如何有效地管理情感，而不是被情感所控制。

情感管理不是压抑情感，而是有意识地感知、理解并调节情感反应的过程。一个常见的初级管理方法是“情绪标注”，即在心里用语言描述当前情绪。

```
# 情感管理步骤示例（伪代码）
if 感知到强烈情绪触发：
    暂停反应
    识别情绪名称（“这是愤怒”）
    分析情绪背后的想法（“我认为我的边界被侵犯了”）
    选择建设性应对行为（“我可以平静地表达我的诉求”）
```

## 实践练习：情感日记

理论需要结合实践。为了提升情感觉察与管理能力，本节介绍一个简单有效的练习方法。

撰写情感日记是增强情感认知的经典方法。每天花几分钟记录以下内容：

*   **情境**：记录引发情绪的事件。
*   **情感**：准确描述你感受到的情绪（尝试使用具体词汇，如“沮丧”、“焦虑”，而非简单的“不好”）。
*   **身体反应**：注意身体的感受（如心跳加速、肩膀紧绷）。
*   **想法**：记录情绪产生时脑海中的自动想法。
*   **行为**：记录你实际采取的行动。
