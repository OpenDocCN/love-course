# 43、彭祖强肾功：052.洗髓功秘训！

🧘‍♂️ 在本节课中，我们将学习洗髓功的秘训内容，特别是根部垂钓、呼吸吐纳、拍打和易阴经等技巧。

## 一、根部垂钓

以下是根部垂钓的步骤：

1. **准备**：使用一条长条布，清洗晾干后折叠成合适长度。
2. **热身**：进行充分的热身，特别是耻骨、两侧复沟和金属等部位。
3. **绑法**：将布条开口朝前，卡住根部，用绳子固定，并调整松紧度。
4. **负重**：选择合适的重量，进行根部垂钓。
5. **呼吸**：采用心如止水呼吸法，吸气时提重物，呼气时放松。

## 二、呼吸吐纳

以下是呼吸吐纳的步骤：

1. **揉通鼻腔**：揉搓鼻翼，确保鼻腔通畅。
2. **站姿**：站立，双脚与肩同宽，双手放在腰间。
3. **呼吸**：采用心如止水呼吸法，吸气时骨盆卷曲，呼气时放松。
4. **震动**：在吸气时震动鼻腔和喉腔。

## 三、拍打

以下是拍打的步骤：

1. **热身**：进行充分的热身，特别是耻骨、两侧复沟和金属等部位。
2. **拍打**：用手掌拍打全身经络，特别是根部垂钓部位。

## 四、易阴经

以下是易阴经的步骤：

1. **打坐**：选择舒适的打坐姿势。
2. **呼吸**：采用心如止水呼吸法，配合意念引导。
3. **动作**：进行易阴经的特定动作。
