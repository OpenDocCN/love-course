## 概述

在本节课中，我们将学习如何通过高情商的接话技巧，使对话更加生动有趣，增强互动感，从而提升人际关系。

## 接话的重要性

**公式**： 互动感 = 主动接话 + 场景化回答

往往，如果无法接住对方抛出的话题，对话很快就会陷入冷场。因此，学会接话是至关重要的。

## 接话的艺术

**代码**：
```python
def 接话(问题):
    if 问题 == "吃了吗？":
        return "自从第一次跟你见面，吃了那个葱油爆虾之后，我之后就天天特别想吃这个菜。"
    elif 问题 == "睡了吗？":
        return "最近都没有胃口，我还是觉得吃饭的时候，有一个可爱的人坐在旁边会比较有意思。"
    elif 问题 == "在干什么？":
        return "最近呢特别想吃红烧茄子啊，我觉得找半天也没有好吃的，好像你朋友圈发的那个你做的，看起来会比较美味。"
    else:
        return "关你屁事，我老公追我的时候，他经常问我问题，我说老资在忙。"
```

通过这种方式，我们可以给对方新鲜的答案，构建场景，激发对方的想象。

## 聊天禁忌

1. **单口相声**： 没有你来我往的互动。
2. **主动制造距离感**： 将对方拉到与自己相同的层次。
3. **心不在焉**： 在交流过程中不断掏手机，显得不尊重对方。
