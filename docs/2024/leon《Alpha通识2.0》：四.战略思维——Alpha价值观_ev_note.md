# 《Alpha通识2.0》：四.战略思维——Alpha价值观

## 概述

在本节课中，我们将学习Alpha价值观中的战略思维，包括如何优化自身形象、财富观以及如何通过刻意练习提升链接意识、影响力思维和表达能力。

## 一、优化形象

**颜值的重要性**

颜值在个人形象中占据重要地位。无论是在选举、面试还是销售过程中，颜值都能给人留下深刻印象。以下是一些提升颜值的方法：

* **饮食控制**：保持健康的饮食习惯，避免过多摄入高热量、高脂肪的食物。
* **健身**：定期进行体育锻炼，增强体质，塑造良好体型。
* **形象改造**：根据自身特点，选择合适的服装和发型。

**90天提升颜值**

90天的时间可以让一个人的颜值从6分提升到7.5分甚至8分。以下是一些提升颜值的建议：

* **了解网络资料**：学习相关知识和技巧。
* **请教他人**：向身边的朋友或专业人士请教。
* **坚持锻炼**：保持良好的生活习惯。

## 二、优化财富观

**财富模型**

真正的财富模型包括产品、销售、营销和渠道四个板块。以下是一些提升财富观的方法：

* **专业选择**：选择适合自己的专业，并深入学习相关知识。
* **中介能力**：掌握营销和销售技巧，成为中介人。
* **商业认知**：提升商业认知，了解商业运作规律。

**必备技能**

* **链接能力**：与人建立良好关系，拓展人脉。
* **意识**：具备敏锐的洞察力，发现商机。
* **影响能力**：影响他人，达成共识。
* **认识及表达能力**：清晰表达自己的观点，说服他人。

## 三、提升表达能力

**表达结构**

任何能抓住你注意力的短视频，其结构都是激发你的好奇、需求或痛苦。以下是一个表达模型：

* **理解**：理解他人的需求和痛苦。
* **破定**：输出自己的价值观。
* **定**：建立共识，引导关系建立。

**表达模型的应用**

通过运用表达模型，可以建立良好的人际关系，实现财富增长。

## 四、结果导向的系统性思维

**系统思维**

系统思维是指将事物看作一个整体，分析其各个部分之间的关系，从而找到解决问题的方法。

**系统思维的应用**

* **尖刀思维**：专注于一件事情，避免分散精力。
* **先胜后战**：找到最强的人，学习其经验。
* **轻装上阵**：远离影响你的人，靠近能帮助你的人。
* **刻意练习**：强化关键能力，避免失败。

## 总结

本节课中，我们学习了Alpha价值观中的战略思维，包括优化形象、财富观以及提升表达能力。通过刻意练习和系统思维，我们可以实现财富增长和人生目标。
