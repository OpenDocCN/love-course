# 雨宸19套合集：16.《雨宸复合课：透视关系破裂的根源，挽回丢失的爱情》：003.安全感缺失的人，最需要的不是安全感

## 概述
在本节课中，我们将探讨安全感缺失对恋爱关系的影响，以及如何建立自我安全感。

## 安全感不是他人给予
**公式**：安全感 = 自我认知 + 自我肯定

安全感并非他人赋予，而是源于自我认知和自我肯定。许多人误以为安全感来自伴侣，实则不然。

## 安全感缺失的表现
以下是安全感缺失者在恋爱中可能出现的几种情况：

### 1. 焦虑型
- **内在模式**：托付心态
- **问题**：过度依赖伴侣，将安全感寄托于对方。

### 2. 过于期待完美的猎人
- **问题**：期待伴侣弥补自己的不足，却忽视自身问题。

### 3. 改变对方
- **问题**：试图改变伴侣，却忽视自身不足。

### 4. 缺乏安全感
- **问题**：内心缺乏安全感，无法感受到爱。

## 建立自我安全感
要建立自我安全感，首先要认识到以下几点：

1. **相信爱情存在**：相信自己是值得被爱的。
2. **接受自己**：接受自己的优点和缺点。
3. **学会爱自己**：每天对着镜子说“我很可爱”。
4. **客观认识自己**：了解自己的优势和不足。

## 总结
本节课中，我们探讨了安全感缺失对恋爱关系的影响，以及如何建立自我安全感。希望你能从中获得启发，建立自信，收获美满的亲密关系。
