# 《二月老师的核心自信》（综合版）：38-38：改写潜意识

## 概述
在本节课中，我们将学习如何通过改写潜意识来提升自信。

## 自我暗示的第三规律：替换
根据心理学家的研究，我们的潜意识只能在同一时间内主导一种感觉，即一个积极正面的思想。通过反复灌输给大脑中的潜意识，我们可以逐渐替换掉原有的思想，让新的思想占据上风。

### 暗示的积极与消极意义
以下是一些例子，说明暗示的积极和消极意义：

- **积极暗示**：当孩子参加高考时，父母叮咛“千万不要紧张”。
- **消极暗示**：当孩子学习成绩不好时，父母斥责“笨得不行，你怎么这么蠢啊”。

### 正面暗示与负面暗示的影响
- **正面暗示**：能够使我们积极向上，而负面的暗示则阻碍我们成功。
- **负面暗示**：容易让孩子形成自卑感，陷入心灵的阴影，走向苦难和犯罪。

## 改写潜意识的方法
以下是一些改写潜意识的方法：

1. **重复目标**：不断重复你的目标，如“我正在变得越来越好”。
2. **放松思想**：在重复时，保持心平气和以及心理顺从的状态。
3. **视觉化**：在你的头脑里放映你的愿望已经实现的情景。
4. **正向肯定句**：为自己设计简短的正向肯定句，暗示你已经实现了你的目标。
5. **写下肯定句**：每天拿出来不停地重复。
6. **回想成功经历**：回想过去成功的经历，体验成功的积极体验。
7. **谈论愿望**：像小孩子一样去运用想象力，以假为真地谈论你的愿望。
8. **重复重复再重复**：通过重复形成新的习惯和信念。
