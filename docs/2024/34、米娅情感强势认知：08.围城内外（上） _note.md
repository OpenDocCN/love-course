# 34、米娅情感强势认知：08.围城内外（上）

**概述**：本节课将探讨婚姻与爱情的关系，从人类历史发展的角度分析婚姻制度和社会属性的变化。

**核心概念**：
- 婚姻与爱情的关系
- 社会属性
- 人类历史发展

## 1. 婚姻与爱情的关系

**以下是关于婚姻与爱情关系的讨论**：

- 许多婚姻的结合并非基于爱情。
- 许多所谓的爱情最终无法走入婚姻。
- 人们口中的相爱终究敌不过现实，还是我们总是习惯把本质是供需关系的两性关系披上一层浪漫的所谓爱情的面纱。

## 2. 婚姻的社会属性

**以下是关于婚姻社会属性的讨论**：

- 当代婚姻制度是一夫一妻制。
- 家庭是社会组织的最小单位，因此婚姻具有社会属性。
- 安格斯认为，只有真正取消私有制，才谈得上爱情。

## 3. 人类历史发展

**以下是关于人类历史发展的讨论**：

- 旧石器时代，社会组织的基本单位是家庭，由父亲、母亲和子女组成。
- 男女之间的关系在旧石器时代比以后任何时候都更为平等。
- 女性在旧石器时代对食物供应的贡献甚至大于男性。
- 新石器时代，经济平等和社会地位相同是村社的明显特征。
- 农业革命导致人口增长，进而引发社会变革。

## 4. 古代文明

**以下是关于古代文明的讨论**：

- 古代文明的特征包括城市中心、国家形式的建制化政治权威、税收、文字、社会分层、阶级或等级等。
- 农业发展使食物来源更有保障，人口增加，进而引发社会变革。
- 古代文明出现社会关系和性别关系的不平等。

## 5. 女性地位的变化

**以下是关于女性地位变化的讨论**：

- 随着农业技术的发展，女性在经济和社会上的独立地位被破坏。
- 女性逐渐成为家内活的主要承担者，处于从属地位。
- 男性垄断了经济、政治和军事权利，女性成为弱势性别。

**总结**：本节课从人类历史发展的角度分析了婚姻与爱情的关系，以及婚姻的社会属性。通过学习，我们可以从更宏观的角度审视自己的婚姻和爱情。
