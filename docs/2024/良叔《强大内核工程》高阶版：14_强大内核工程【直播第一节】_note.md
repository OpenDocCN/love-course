# 良叔《强大内核工程》高阶版：14_强大内核工程【直播第一节】

## 概述

在本节课中，我们将学习如何克服焦虑，建立强大的内核。良叔将分享五个内部训练方法，帮助我们在面对焦虑时保持冷静和理性。

## 五个内部训练方法

### 1. 避免灾难性想象

**公式**： 灾难性想象 = 担心未来可能发生的事情

**代码**： `灾难性想象 = 担心未来可能发生的事情`

**解释**： 我们常常担心未来可能发生的事情，例如约会时女生会不会看不上我们。这种担忧会让我们感到焦虑和不安。我们需要意识到这些担忧只是我们的想象，而不是现实。

### 2. 不要用特写镜头看待问题

**公式**： 特写镜头 = 放大问题，忽略其他因素

**代码**： `特写镜头 = 放大问题，忽略其他因素`

**解释**： 我们常常用放大镜看待问题，只关注问题的负面影响，而忽略其他因素。这种思维方式会让我们感到焦虑和不安。我们需要学会从更长远的角度看待问题，不要被问题本身所困扰。

### 3. 不要让外界阻挡自己

**公式**： 受害者心态 = 将责任推给外界

**代码**： `受害者心态 = 将责任推给外界`

**解释**： 我们常常将责任推给外界，认为自己无法掌控自己的命运。这种思维方式会让我们感到无助和沮丧。我们需要学会承担责任，相信自己可以掌控自己的命运。

### 4. 前进推进

**公式**： 前进推进 = 专注于解决问题，不要被复杂问题所困扰

**代码**： `前进推进 = 专注于解决问题，不要被复杂问题所困扰`

**解释**： 我们常常被复杂问题所困扰，不知道如何解决问题。我们需要学会将问题简化，专注于解决问题，不要被复杂问题所困扰。

### 5. 不要被自动思维所困住

**公式**： 自动思维 = 负面想法，不受控制

**代码**： `自动思维 = 负面想法，不受控制`

**解释**： 我们常常被负面想法所困扰，这些想法不受控制，会让我们感到焦虑和不安。我们需要学会察觉这些想法，并对其进行评估，不要被它们所控制。

## 总结

在本节课中，我们学习了如何克服焦虑，建立强大的内核。通过避免灾难性想象、不要用特写镜头看待问题、不要让外界阻挡自己、前进推进和不要被自动思维所困住，我们可以更好地应对生活中的挑战，建立强大的内核。
