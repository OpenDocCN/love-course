# 驾驭关系的能力：情绪价值：06如何发现对方的真我

## 概述

在本节课中，我们将学习如何发现对方的真我，并通过肯定、赞美和认可来驾驭关系。

## 真我的定义

一个人的内在由三个组成部分构成，即真我、防御自我和潜意识。真我指的是一个人纯净善良、美好的一面，包括情绪稳定、自信、阳光、关怀、思路清晰、好奇心、勇敢、创造性和建立良好人际关系的能力。

## 发现真我

1. **观察和相处**：通过观察和相处，发现对方骨子里的善良和美好。
2. **肯定和赞美**：发现对方真我后，给予肯定和赞美，让对方感受到被认可和肯定。

## 案例分析

### 案例一：丈夫背叛

1. **发现优点**：在丈夫背叛时，跳出自己的受伤点，从多个角度寻找他的优点，如责任心、关爱孩子等。
2. **肯定和赞美**：针对优点进行肯定和赞美，如“虽然我们关系不好，但我对你这个人非常认可，因为孩子不管什么时候有事，你都能够亲力亲为。”

### 案例二：夸奖公式

1. **积极态度**：在夸奖时，保持积极的态度，如微笑、热情等。
2. **具体事实**：提供具体的事实依据，如“你看这个红烧排骨，外焦里嫩，颜色通红透红。”
3. **正面评价**：给出正面评价，如“真的很香啊，而且脱骨。”
4. **夸奖依据**：说明夸奖的原因，如“你看啊，每一次做肉，你做的都比我好吃啊，你看今天这个红烧排骨如果我做，我就只能炖了。”

## 总结

通过发现对方的真我，并给予肯定、赞美和认可，我们可以驾驭关系，让对方愿意接近我们。智慧的女人懂得如何运用情绪价值，成为最强大的人。
