## 概述
在本节课中，我们将深入探讨烽暴聊天法的第五点：聊什么话题。我们将学习如何通过提问和话题选择来更好地了解对方，并提升聊天效果。

## 聊天话题的选择
以下是几个关键问题，用于引导话题：

1. **对方是否本地人**：判断对方是否为本地人，有助于了解其生活状态和操作方向。
2. **平时休息时做什么**：了解对方的娱乐方式和生活方式。
3. **对方的定位和聊天风格**：判断对方对你的防备程度和距离感。
4. **测试兴趣窗口和邀约变现**：通过提问了解对方的兴趣和邀约意愿。

## 聊天步骤解析
1. **展示面**：在一线城市，展示面是关键，要有明信片才能吸引资源。
2. **表达好感**：在明显有窗口时，适当表达好感和欣赏。
3. **做好预期**：在邀约前，植入暧昧属性的话，让见面后推进关系更高效。
4. **选择话题**：通过朋友圈热点时事、天机气候的第三方话题来开启话题。
5. **攻心尽力**：进行深层次的对话，了解对方的想法和感受。
6. **给予情绪价值**：通过关心、理解和支持，让对方感受到被关注和重视。
