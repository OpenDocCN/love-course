# 《反儒家》视频教程：02 努力就有好结果？勤劳最无用！谷贱伤农，卖炭翁、养蚕人、蜂，告诉我们一个什么样的道理？

## 概述
在本节课中，我们将探讨勤劳是否总是有益的，并分析古代社会和现代社会中勤劳的不同影响。

## 勤劳的定义与古代社会
**勤劳**是指辛勤劳动、不懈努力。在古代中国社会，勤劳被视为美德，但实际情况可能并非如此。

### 古代社会的勤劳与科举制度
- **儒家与法家**：古代中国社会表面上推崇儒家思想，实则内里实行法家制度。
- **科举制度**：通过科举考试选拔官员，控制民众。
- **世袭贵族**：世袭贵族无需考试，世代传承。

### 勤劳的负面影响
- **阶级固化**：科举制度导致阶级固化，勤劳的读书人难以改变命运。
- **勤劳的局限性**：勤劳并不能保证成功，尤其是在资源稀缺的情况下。

## 现代社会中的勤劳
### 金针菇效应
- **金针菇种植**：某地农民种植金针菇成功，其他农民纷纷效仿，导致价格下跌。
- **资源稀缺**：在资源稀缺的情况下，勤劳并不能保证成功。

### 剧场效应
- **海淀妈妈焦虑**：中国中产阶级家长过度焦虑，认为孩子必须努力学习。
- **城市化与劳动力流动**：大量劳动力涌向城市，但并非所有人都能留在城市。

### 基因与认知
- **基因与认知**：人的基因和认知水平影响其努力程度和成功概率。
- **勤奋的局限性**：勤奋并非万能，成功需要多种因素。

## 创业与事业
### 创业与事业
- **创业**：通过创业实现财富积累。
- **事业**：事业属于个人，职业属于他人。

### 商业社会属性
- **商业社会**：商业社会强调个人奋斗，但并非所有人都能成功。

## 总结
本节课中，我们探讨了勤劳是否总是有益的，并分析了古代社会和现代社会中勤劳的不同影响。勤劳并非万能，成功需要多种因素，包括资源、机遇、基因和认知等。
