在本节课中，我们将要学习女性在择偶过程中的基本需求逻辑，理解其与男性需求的根本差异，并分析短期与长期择偶策略的不同。

而协调性的选择，是在女人考虑到生存资源，需要权衡利弊的时候才会去考虑的。所以，协调性的选择是第二选择。

理解了价值构成后，我们进一步探讨女性在不同情境下的择偶策略。

女性择偶的策略其实是分为两种情况的：一种是长期，另一种是短期。如果仅仅只是谈个恋爱，那么女性一定是以本能的选择为优先。而短期的恋爱，如果相处融洽，也是很有可能发展为长期的关系的。

最后，我们来分析一个关键误区：仅靠物质供给能否赢得女性的爱慕。

**本节课总结**

本节课中，我们一起学习了女性择偶的基本需求框架：
