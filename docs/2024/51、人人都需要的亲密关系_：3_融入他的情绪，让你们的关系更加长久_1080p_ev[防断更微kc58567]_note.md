# 人人都需要的亲密关系_：3_融入他的情绪，让你们的关系更加长久

## 概述
在本节课中，我们将探讨如何通过融入对方的情绪，使亲密关系更加长久。我们将学习如何理解并回应对方的情感，以及如何通过情感共鸣来加深彼此的联系。

## 什么是迷恋？
迷恋是一种非理性的情感状态，它使人对某个人或事物产生强烈的、无法控制的吸引力。**公式**：迷恋 = 情感投入 + 非理性吸引。

## 如何让自己看起来值钱？
1. **展示高标准**：明确你的期望和需求，让对方知道你值得被珍惜。
2. **独立性**：不要完全依赖对方，保持自己的生活和兴趣。
3. **背书**：通过其他优质男性的追求来展示你的价值。

## 如何触发他的欲望？
1. **戏谑和挑逗**：通过幽默和轻松的方式激发对方的兴趣。
2. **欲望的承诺**：暗示或许下承诺，但不一定履行，以保持对方的兴趣。
3. **喂食无分饱**：给予对方一部分满足，但不要完全满足，以保持对方的欲望。

## 如何获得他的信任？
1. **营造家的感觉**：支持对方，成为他的后盾。
2. **精神鼓励**：给予对方情感上的支持，成为他的精神支柱。
3. **重复**：持续地给予鼓励，让对方感受到你的真诚。

## 总结
本节课中，我们学习了如何融入对方的情绪，使亲密关系更加长久。通过让自己看起来值钱、触发对方的欲望以及获得他的信任，我们可以建立更加稳固和持久的亲密关系。希望这些方法能够帮助你更好地理解和维护你的亲密关系。
