# 林老头《私人传承》：017.「关系概念」与进阶必备，导师和普通人的差距 🎓

## 概述

在本节课中，我们将探讨关系概念的重要性以及如何建立进阶必备的沟通技巧。我们将深入分析关系营销、合理化沟通和建立信任背书等关键概念。

## 关系营销

**以下是关系营销的关键点：**

- **关系的重要性**：与任何人建立关系时，首先要明确你们的关系类型和发展方向。
- **个人价值与可得性**：个人价值（如高富帅）并非建立关系的唯一因素，关系中的可得性更为关键。
- **关系营销策略**：将重点放在建立关系上，而非个人价值的展示。

**公式**：

\[ 关系 = 个人价值 + 可得性 \]

## 合理化沟通

**以下是合理化沟通的关键点：**

- **认可与幸福**：合理化沟通的前提是让对方认可你或你的理论。
- **无底线接受**：在认可的基础上，可以接受无底线的合理化。

**公式**：

\[ 合理化沟通 = 认可 + 理论认可 \]

## 建立信任背书

**以下是建立信任背书的关键点：**

- **展示面**：展示面应包含他人认可和追求幸福的证据。
- **侧面发现**：信任背书应通过侧面发现，而非直接告知。

**公式**：

\[ 信任背书 = 展示面 + 侧面发现 \]

## 总结

本节课中，我们学习了关系概念、合理化沟通和建立信任背书等关键概念。通过掌握这些技巧，我们可以更好地与他人建立关系，提升沟通效果。

- 关系营销的重要性
- 合理化沟通的技巧
