# 子阳《脱单实战聊天课》第1课：聊什么？🤔

## 第二节课：聊天的三个目的 🎯

在本节课中，我们将要学习聊天的核心目的。许多学员不知道和女生聊天该说什么，本节课将首先解决这个问题，明确我们每一次对话究竟是为了达成什么。

在开始正式课程之前，我们先要搞明白一个根本问题：聊天的目的是什么？你们希望通过聊天来达到什么目的？

以下是学员们常见的回答：
*   邀约。
*   增进感情。

这里需要统一说明一下。很多兄弟报名课程，是因为遇到了一个非常喜欢但搞不定的女孩子。但首先需要明确一点：我们不可能仅通过聊天就让一个女生爱上自己。聊天只能帮助我们达成以下三个具体目的。

### 1. 相互了解 👥

上一节我们提出了聊天的目的问题，本节中我们来看看第一个核心目的：相互了解。

这个目的指的是，我们需要在聊天中与女生大量交换彼此的基础信息和三观。这个过程本质上是一个**筛选机制**。

**公式：聊天（交换信息）→ 相互了解 → 筛选合适对象**

大部分兄弟的情感问题，根源往往不是技巧不足，而是**人没选对**。相互了解就是解决这个根本问题的第一步。

### 2. 培养感情 🤝

了解了筛选的重要性后，我们来看看第二个目的：培养感情。

请注意，这里所说的“感情”并非特指男女之间的爱情。它指的是双方通过共同投入时间成本所磨合出的情感联结。

我给你们举个例子：
假设你住了一套30年的老房子，市值50万。你的邻居用他市值70万的新房子来换，你很可能不愿意换。因为你对自己的老房子产生了感情，这是长期居住投入的情感。

**代码描述：感情 = 累计投入(时间, 经历)**

一个人或一件物品，接触久了自然会产生感情。聊天中的“培养感情”，指的就是通过持续的互动，累积这种基于时间投入的情感基础。

### 3. 达成邀约 📅

明确了长期互动能培养感情后，我们最终要导向一个实际结果：达成邀约。

这是我们第三个，也是最终的目的。无论聊天中使用了多少看似花哨的技巧，例如推拉、上推下切等，其最终目标都是为了实现**邀约**。

**核心逻辑：所有聊天技巧 → 为邀约服务**

不必过于纠结复杂技巧，聊天的终端结局就是创造线下见面的机会。

---

本节课中我们一起学习了聊天的三个核心目的：**相互了解**、**培养感情**和**达成邀约**。首先，聊天是筛选与了解彼此的过程；其次，它是通过时间投入培养情感联结的方式；最终，所有对话都应导向线下邀约这个实际目标。请兄弟们务必明确，你们在聊天中说的每一句话，都应围绕这三个目的展开。
