# 《朕哥8套情感合集》：02.朕哥高阶私教课之保卫爱情：05-第五篇：如何跳出恋爱劣势

🎼如何跳出恋爱劣势？大家好，我是震哥。本节课震哥将教授女生如何摆脱恋爱劣势。大多数女生容易被比自己优秀的男人吸引。然而，由于男人比自己优秀很多，她们担心男人会被其他女人抢走，也害怕男人不再爱自己。因此，许多女生在恋爱中表现得非常顺从，生怕男人对自己不满意，离开自己，最终导致自己在感情中处于被动地位，没有地位。

如果女生在恋爱中处于劣势，整天提心吊胆，这种感情是无法长久的。遇到这种问题，女生该怎么办呢？接下来震哥用一个例子来讲解你们之间的价值匹配度，帮助同学们更好地理解这个问题，并平衡自己的心态。

**以下是本节课的主要内容**：

1. **价值匹配度**：将感情比作一个缘，如果两个人都是大半圆，那么他们更多的是相似，但无法互补，不会长久。如果两个人都是小半圆，那么他们拼到一起也无法成为一个圆，会有很多不足与空白，反映到现实中就是物质生活不丰富，或者两个人在一起平淡无味，价值不够撑起一个家。如果在这个圆里，他占大半个，你占小半个，你觉得你处于劣势，那么这个圆可以从很多方面去理解。他的大部分是社会价值，而你的小部分是情绪价值。有了你之后，他才更加完整。

2. **提升非专属价值**：非专属价值是指不仅仅对你男友有用，对其他男人也有用的价值。例如，你对男人温柔体贴，这就是你对男人的专属价值。但是，你每天无论去上班还是去约会，都可以打扮得很漂亮，街上的男人看到你可多两眼发直，这就不是你男友的专属价值了。非专属价值包括衣着气质、化妆等等外表方面，还有会说话、高情商、幽默等等。具体提升方法包括重新重视化妆打扮、全面提升自己的气质、报一个关于提高情商的培训课等。

3. **增强不可替代性**：提升自己的专属价值，例如在男人精疲力尽回到家的时候，给他准备可口的饭菜。另外，要增强不可替代性，例如你很会理财，男人放在你这里的钱都得到了增值。

4. **引入竞争性嫉妒**：让男人有一些潜在的情敌，让男人觉得你不可拿捏，不可掌控，琢磨不透。具体做法包括多参加朋友聚会、同事聚会、行业年会等，让自己的照片里可以有几个帅哥或者行业大佬等更优秀的男人，在自己的朋友圈里发自拍，吸引异性朋友点赞评论，有自己的社交圈，定期和朋友们聚会。

5. **找到男人的性格弱点**：了解他的性格，找到他的性格弱点，并帮他一把，那他一定会深深的意识到你对他的价值。

6. **生活上的间歇性脱钩**：偶尔出乎他的意料，脱离他的控制。具体做法包括微信聊天不要秒回，偶尔在他想和你聊天的时候，说自己有事拒绝和他聊天，不要让他完全掌握你的作息规律和行动轨迹，不要每次他下班啊，你都在家里迎接他，一周有那么个一两次不回家和闺蜜逛街去吃饭。

7. **将你们的爱固化**：当男人开始琢磨不透你的时候，他就会想要求你，要求你微信要及时回复，下班要准时回家等。那么这个时候呀，你就可以跟他说你自己的想法了。至少要求他也做到以上几条。那这样的交流呢就是可以将你们的爱固化。

**总结**：

本节课中，我们学习了如何跳出恋爱劣势，包括价值匹配度、提升非专属价值、增强不可替代性、引入竞争性嫉妒、找到男人的性格弱点、生活上的间歇性脱钩、将你们的爱固化等。希望同学们能够将这些方法应用到实际生活中，改善自己的感情状况。
