# 54、挽救婚姻有办法：1-学员学前准备 📚

各位学员，我是朱生勇，婚姻管理学派的创始人，欢迎大家加入我们的婚姻管理系统学习。在开始学习之前，我为大家列出十个要求。

1. **自主学习**：请确保您独立学习，不要强迫您的伴侣一起学习。有些学员尝试拉着自己的伴侣学习，结果导致对方愤怒。请记住，严禁拉伴侣学习。

2. **学习三遍**：每个课程都要学习三遍。先听三遍课程，然后做笔记。有些学员听一遍就上战场，结果发现效果不佳。

3. **学以致用**：我们的婚姻管理系统和思想侧重于应用和解决问题。不要期望朱老师能教您高大上的东西，如身心灵的提升。朱老师教的是实操性的方法。

4. **一点一滴谈**：在解决婚姻问题时，要一点一滴地谈，不要期望一次就能解决问题。

5. **无事才谈**：不要在发生冲突时谈论问题。在对方放松的状态下谈论事情，才能更好地沟通。

6. **分享学习笔记**：在学员群中分享您的学习笔记，但只分享其中的10%。分享可以增加您的使命感，并督促您每天学习。

7. **坚持学习**：即使您的伴侣在家，也要坚持学习。治理婚外情需要五个阶段，不要以为伴侣回来就万事大吉。

8. **学习婚姻管理学派**：不要与其他学派比较，我们的婚姻管理学派是一个系统，无法单独比较。

9. **课程无法替代辅导**：课程无法替代辅导，辅导是个人化的，可以更具体地解决您的问题。

10. **心怀感恩**：感恩自己的付出，感恩伴侣、孩子、父母以及朱老师的付出。

**总结**：本节课中，我们学习了如何进行婚姻管理系统学习。希望每位学员都能从中受益，改善自己的婚姻关系。
