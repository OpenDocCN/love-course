# 情感指南针：003.第 3 课：通过本命盘，找到带你搞大钱的男人 🌟

## 概述
在本节课中，我们将学习如何通过本命盘识别那些能够帮助你赚钱的男人。我们将探讨不同类型的男人及其在命盘中的特征。

## 第一类：创一代型男人
这类男人的特征是金星或群星落在六宫，代表他们热爱工作，事业心强。**公式**：金星/群星落在六宫。

- **工作能力**：这类男人工作能力强，能够教你如何赚钱。
- **情绪价值**：在工作时不要打扰他，工作结束后随叫随到。

## 第二类：资源型男人
这类男人的特征是木星落在十一宫，代表他们拥有丰富的人脉资源。**公式**：木星落在十一宫。

- **人脉资源**：这类男人能够为你提供实质性的资源和人脉。
- **情绪价值**：肯定他在朋友圈的地位，展示你自己的资源，进行资源互换。

## 第三类：大官员型男人
这类男人的特征是木星落在十宫，代表他们社会地位高，有影响力。**公式**：木星落在十宫。

- **社会地位**：这类男人通常名声显赫，能够帮助你很多。
- **情绪价值**：营造小白兔的人设，让他在你面前放下戒备。

## 第四类：高认知型男人
这类男人的特征是木星落在九宫，代表他们思维层级高，有学识。**公式**：木星落在九宫。

- **认知提升**：这类男人能够给你带来很多认知上的提升。
- **情绪价值**：与他进行思想交流，展示你的内涵。

## 总结
本节课中，我们学习了四种可以带你赚钱的男人的配置。这些特征可以作为初步筛选，但更准确的信息需要看全盘。希望这些内容能帮助你擦亮眼睛，准确挑选出最好的男人。
