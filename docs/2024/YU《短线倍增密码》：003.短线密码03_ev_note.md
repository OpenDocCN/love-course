## 什么是变量？📦

上一节我们介绍了本课程的目标，本节中我们来看看什么是变量。变量是编程中用于存储数据的容器。你可以把变量想象成一个贴有标签的盒子，标签是变量的名字，盒子里装的是数据。

在Python中，创建一个变量并为其赋值非常简单，只需要使用等号 `=`。

**代码示例：**
```python
name = "小明"
age = 25
height = 1.75
is_student = True
```

## 基本数据类型 🔢

变量可以存储不同类型的数据。Python有几种内置的基本数据类型，它们决定了数据可以执行的操作。

以下是Python中最常用的几种基本数据类型：

*   **整数 (int):** 表示没有小数部分的数字，如 `10`, `-5`, `0`。
*   **浮点数 (float):** 表示带有小数点的数字，如 `3.14`, `2.0`, `-0.5`。
*   **字符串 (str):** 表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，如 `“你好”`, `‘Python’`。
*   **布尔值 (bool):** 表示逻辑值，只有两个：`True` (真) 和 `False` (假)。

## 如何查看和转换数据类型？ 🔄

有时我们需要知道一个变量的类型，或者将一种类型转换为另一种类型。

**查看类型：** 使用 `type()` 函数。
**代码示例：**
```python
print(type(age))    # 输出：<class ‘int’>
print(type(name))   # 输出：<class ‘str’>
```

**类型转换：** 使用 `int()`, `float()`, `str()` 等函数进行转换。
**代码示例：**
```python
num_str = “123”
num_int = int(num_str)  # 将字符串“123”转换为整数 123
print(num_int + 7)      # 输出：130
```
