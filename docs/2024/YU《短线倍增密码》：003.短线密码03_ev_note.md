# Python基础教程 1.1

## 概述
在本节课中，我们将学习Python编程语言的基础知识，包括安装Python、基本语法和变量。

## 安装Python
首先，您需要安装Python。Python可以从其官方网站下载：[https://www.python.org/downloads/](https://www.python.org/downloads/)。

### 安装步骤
1. 访问Python官方网站。
2. 选择适合您操作系统的Python版本。
3. 下载并运行安装程序。
4. 在安装过程中，确保勾选“Add Python to PATH”选项。

## 基本语法
Python是一种解释型语言，这意味着您可以直接在终端或命令行中运行代码。

### 变量
变量是存储数据的容器。在Python中，您可以使用以下方式声明变量：

```python
x = 10
y = "Hello, World!"
```

### 输出
要输出变量，可以使用`print()`函数：

```python
print(x)
print(y)
```

### 运行代码
在终端或命令行中，输入以下命令运行代码：

```bash
python filename.py
```

其中`filename.py`是您的Python文件名。

## 总结
本节课中，我们一起学习了如何安装Python、基本语法和变量。在下一节课中，我们将学习Python的控制流和函数。
