# 27、林夕-高质量关系课程：1.0-第0课《婚姻中如何让对方依赖》

大家好，我是林夕。今天我们将探讨如何在婚姻中让对方产生依赖感。

## 概述

在本节课中，我们将学习如何在婚姻中实现以下目标：
1. 持续共同成长，变得优秀。
2. 拥有单身力，保持独立性。
3. 给彼此的生活留白，保持信任。

## 第一部分：共同成长

**观点**：两个人必须共同成长，一起变得优秀。

**例子**：郭晶晶与霍启刚的婚姻。

**共同成长的方法**：
1. 保持良好的身体状况，提高繁衍价值。
2. 简单的生活经历，保持清白。
3. 保持良好的外貌，提升个人价值。

**共同成长的益处**：
- 缩短差距，实现平等。
- 携手共进，和谐相处。

## 第二部分：拥有单身力

**观点**：拥有单身力，让对方更加依赖。

**单身力的定义**：
- 即使结婚，也要保持独立意识。
- 具备照顾自己的能力。
- 有分手和离婚的底气。

**单身力的益处**：
- 更好地经营关系。
- 拥有更多，而不是失去更多。

## 第三部分：给彼此留白

**观点**：给彼此的生活留白，保持信任。

**留白的益处**：
- 避免过度限制对方。
- 发现生活的美好。
- 尊重彼此的空间。

**信任的重要性**：
- 规则内的信任。
- 规则外的自由。

## 总结

本节课主要讲述了婚姻中让对方依赖的三个关键点：
1. 持续共同成长，变得优秀。
2. 拥有单身力，保持独立性。
3. 给彼此的生活留白，保持信任。
