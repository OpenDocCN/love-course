# 11.钻石男神婚恋辅导提升营：069.3聊天技巧，如何有趣的

在本节课中，我们将学习如何让聊天变得有趣，掌握实用的表达技巧。这些技巧不仅适用于现场约会，也适用于网络聊天。在网络使用时，请注意结合上一节课的内容进行适当调整。

## 一、 聊天心态

上一节我们介绍了课程概述，本节中我们来看看聊天的核心心态。良好的心态是愉快交流的基础。

以下是关于聊天心态的两个要点：

*   **避免过强目的性**：聊天不应总以接吻或开房为目标。应投入并享受与女孩交流的过程。在一个没有目的性、没有“追求者”姿态的人面前，女孩更愿意敞开心扉，也更愿意了解你。这种良好氛围能快速拉近双方的心理距离。
*   **保持真诚，避免过度表现**：不要总想着表现自己或迎合女孩。应表达真实想法。这种真诚态度在每次谈话中都是必要的。展示欲过强、不停说话，可能让女孩觉得你不够自信，只是在炫耀。时间久了，女孩没有说话空间，会感到无聊。

## 二、 基本交谈技巧

掌握了正确的心态后，我们来看看具体的交谈技巧。良好的技巧能让对话顺畅进行。

以下是三个基本的交谈技巧：

*   **学会享受并主导聊天**：与女孩聊天时，应专注并进入状态，这是基本礼貌。若感觉话题无聊，应主动切换。若谈话无趣，应及时结束、转场或礼貌离开。聊天时玩手机、东张西望等行为很不尊重对方。聊天是双向交流，而非个人独白。
*   **不要随意发表观点**：每人都有擅长的领域和考量。在不完全理解女孩想法时，不要随意发表观点。尤其当女孩分享个人经历时，这本身是表达兴趣，希望你能发现她的闪光点。若你像“找茬”一样对她的故事评头论足，她自然不会有好感。涉及她的专业领域时，更不要随便评论。
*   **善用开放式话题**：开放式话题是引导女孩聊天的好方法。问题不应太详细，将剩余部分交给女孩讲述。例如，问“你为什么喜欢吃草莓？”即可。很多人会画蛇添足，在后面加上自己的猜测（如“是不是觉得酸酸甜甜的？”），如果猜中，女孩只会回答“是”，这会将开放式问题变成封闭式问题。

## 三、 高级表达技巧

在掌握了基本交谈技巧后，我们来学习一些更高级的表达技巧，让你的聊天更具吸引力。

以下是三个高级表达技巧：

*   **避免重复**：不要一直重复某个话题、某件事或某句话。可能这是你的“奖项”，或能提高你的价值。但重复本身没有意义。一件高价值的事说一遍是高价值，说两遍是装逼，说三遍就是自卑了。
*   **学会倾听**：这是所有技巧中最难的一项。每个人都有表达和倾诉的欲望。倾听时其实不需要太多语言。你可以长时间不说话，静静让女孩讲述，看着她的眼睛或递上一杯水。即使女孩讲的只是生活琐事，你们之间的氛围也会很快升温，因为她能感受到你的认真和真诚，觉得在进行心灵沟通。
*   **说话简单明了**：和女孩聊天切忌拖泥带水，否则会显得不自信且语言表达能力不到位。说话要像超短裙：足够短以留住别人注意力，也足够长以涵盖所有重要部分。

## 四、 如何实现有趣表达

了解了高级技巧后，本节我们聚焦于如何让表达本身变得有趣。有趣的表达能极大提升聊天魅力。

实现有趣表达需要做到以下两点：

1.  **清晰的表达是基础**：基本表达能力是必备的。口齿不清、语速太快或发音不准会破坏聊天氛围。平时应锻炼基本表达能力。可以找一本白话小说，逐字清晰、语速平缓地念出来。练习久了，可尝试用胸腔发音（气从下往上升），使声音更浑厚有力。聊天时说话要有停顿，不要不停讲。注意与女孩眼神交流，讲述时可使用幅度不大的肢体语言。语句音调可稍微降低，不要一直提高。
2.  **掌握有趣的技巧**：有趣的技巧能让表达出彩。
    *   **反转**：故事结局与常人思维不同。例如：法官问罪犯为什么用假钞，罪犯说：“因为我不会用真钞。”
    *   **夸张**：从片面夸大以达到搞笑效果。例如：女孩说完一件趣事后，你可以面无表情地说：“嗯，这个发现挺好，我觉得可以带你去获得诺贝尔奖了。”
    *   **荒谬**：将两个截然不同的东西联系到一起。例如：“母狮子在家庭里任务很重，负责打猎养家。你知道公狮子负责做什么吗？他负责长鬃毛。”

有趣的聊天需要长期积累和练习。建议平时多听脱口秀和相声，为你的幽默积累素材。
