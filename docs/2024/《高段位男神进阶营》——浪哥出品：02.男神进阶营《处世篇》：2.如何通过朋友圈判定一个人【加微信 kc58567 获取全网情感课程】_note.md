## 昵称识别
以下是几种常见的昵称类型及其特点：

**1. 真姓真名**
- **特点**：为人靠谱、光明磊落、做事严谨、真诚、值得信赖。
- **公式**：昵称 = 姓名

**2. 名 + 状态**
- **特点**：开朗、洒脱、年龄小、不在乎他人看法、精力旺盛、喜欢挑战。
- **公式**：昵称 = 姓名 + 状态

**3. 英文名**
- **特点**：有品位、傲娇、接受新鲜事物快、冒险、独当一面。
- **公式**：昵称 = 英文

**4. 小名/乳名**
- **特点**：豁达、不设防备、容易满足、希望快速融洽的关系。
- **公式**：昵称 = 小名/乳名

**5. 二次元名字**
- **特点**：幼稚、关注小事、宅、不喜互动。
- **公式**：昵称 = 二次元元素

**6. 四字成语**
- **特点**：年龄偏大、强势、渴望被尊重、情感丰富。
- **公式**：昵称 = 四字成语

**7. 职业信息**
- **特点**：努力、自律、渴望被认可、目的性强。
- **公式**：昵称 = 职业信息 + 姓名

**8. 字母/数字**
- **特点**：直销/微商、上进心强、目的明确、行动果断。
- **公式**：昵称 = 字母/数字 + 内容

**9. 表情符号**
- **特点**：冷静、理智、严谨、真诚、值得信赖、年龄小、年少老成。
- **公式**：昵称 = 表情符号

**10. 无关词组合**
- **特点**：内心有火、渴望被点燃、渴望被看见、渴望被尊重、能力一般。
- **公式**：昵称 = 无关词1 + 无关词2
