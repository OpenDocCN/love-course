# 49、清风老师全套：8.彻底打败2+1，保护孩子保：089.第七课：寻求亲人的支持（下篇）获得力量

在本节课中，我们将学习如何向亲人（包括自己的父母、对方的父母以及孩子）寻求支持，以获得力量来应对伴侣出轨的危机，并为全面反击做好准备。

上一节我们介绍了如何寻求朋友的支持，本节中我们来看看如何向亲人寻求帮助。在与亲人沟通前，充分的准备至关重要。

以下是沟通前必须完成的四个准备步骤：

1.  **搜集足够多的证据**：这是后续所有行动的基础。
2.  **确定对方出轨的事实和对象**：在掌握证据后，明确问题的核心。
3.  **客观地看待自我**：这不是一味自责或将责任全部推给第三者，而是**正面评估自己做得好的地方（满足对方需求的部分）和负面评估自己需要改善的问题**，以便在关系中做出客观调整。
4.  **准备好合理的诉求**：明确你的核心需求，例如**财产分配方案、对方的具体承诺，或在对方父母见证下对关系的某种肯定**。诉求需结合实际情况，保持合理。

在这里需要特别说明法律层面的关键信息：在法律上，出轨通常不影响离婚财产的基本分配原则（一般为平分）。法律认定的“外遇”通常指**有配偶者与他人同居3个月以上**。一夜情、嫖娼等行为一般不被认定为法律意义上的外遇。若对方行为构成法律认定的外遇，你可以主张精神损害赔偿，**一般最高不超过20万元**。因此，提出诉求前，务必了解全面信息，结合各方面条件，确保诉求合理有利。

完成以上准备后，我们就可以探讨如何向三类亲人寻求支持：自己的父母、对方的父母以及你们的孩子。父母的爱有时表现为指责，但他们始终是你最坚定的后盾。而对方的父母，其首要利益是保护自己的孩子，需注意沟通策略。孩子是无辜的，我们的目标是尽量减少事件对他们的伤害，而非将其作为工具。

## 如何向自己的父母寻求支持 🤝

首先，寻求他们的理解，表达自己当前的脆弱和需要。出示证据时，需考虑父母身体状况，酌情告知。

以下是沟通要点：

*   **表达脆弱与困难**：明确告知父母你需要他们的理解与支持，不要独自承受一切。
*   **寻求精神支持**：你可以这样说：“爸妈，我知道你们也很重视我的这段关系。现在走到这一步，大家都不愿意看到。我希望你们能多给我一些宽容和精神上的支持，现在是我最需要你们的时候。”
*   **核心是精神鼓励**：现阶段，父母在精神上的鼓励和支持，远比物质或经济援助更为重要。

## 如何向对方的父母寻求支持 🧓👵

切记，对方的父母不是你的父母。沟通时需保持策略，站在“希望家庭更好”的共同立场。

以下是沟通的三个步骤：

1.  **传达反思与歉意**：主动谈及自己在关系中做得不好的部分，展现反思态度。
2.  **传达改善与未来计划**：说明你自身正在做出的积极改变以及对未来的打算。
3.  **传达对孩子的愧疚**：表达因为夫妻矛盾让孩子受到影响的愧疚之情，这容易引发对方父母的理解与共鸣。

采取理解、反思的方式沟通，比愤怒、攻击的方式，更能赢得对方父母可能的支持与内疚感。

## 如何与孩子沟通并寻求支持（减少伤害） 👶🧒🧑

孩子是无辜的，我们的目标不是让他们评判对错，而是最小化他们所受的伤害，并根据年龄采取不同沟通方式。

以下是针对不同年龄段孩子的沟通方法：

*   **5岁以下**：保持常态，通过日常关怀向孩子传递爱和安全感。
*   **5到10岁**：坦诚沟通。如果孩子问“爸爸妈妈是不是要离婚？”，不要欺骗。可以说：“爸爸妈妈遇到了一些问题，如果解决不了，可能会分开。” 如果孩子追问，可以用他能理解的语言解释：“妈妈发现爸爸不爱妈妈了，妈妈很努力想让爸爸回来，但不知道结果会怎样。不过，爸爸和妈妈都永远爱你，这份爱永远不会变。” 重点是多陪伴，安抚其情绪。
*   **10到17岁**：孩子可能已较为懂事。当他们说出“爸爸对你不好，离婚吧”这类话时，可以借此引导他们思考。例如反问：“你觉得一段感情中什么最重要？”或“你认为和伴侣相处应该怎么做？”。这既是交流，也是引导孩子学习如何正确去爱和相处的机会，并能拉近亲子关系。
