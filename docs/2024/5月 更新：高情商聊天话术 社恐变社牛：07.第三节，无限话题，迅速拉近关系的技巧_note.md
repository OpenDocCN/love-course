# 高情商聊天话术 社恐变社牛：07.第三节 无限话题，迅速拉近关系的技巧

## 概述
在本节课中，我们将学习如何在聊天中找到无限话题，并运用这些技巧迅速拉近与对方的关系。

## 核心概念
**无限话题能力**：指在任何事物中都能找到话题进行聊天。

## 找到话题的技巧
### 1. 发散思维
- **观察周围环境**：例如，看到车厘子可以讲述第一次吃车厘子的经历。
- **联想和发散**：从身边的事物联想到相关的故事或经历。

### 2. 利用对方提供的话题
- **关注对方回应中的关键字**：从对方的话语中找到可以回应的关键词，进行话题延伸。
- **联想和发散**：例如，看到钱包可以联想到大学时父亲送钱包的情景。

### 3. 利用朋友圈
- **关注对方的朋友圈内容**：了解对方的生活经历和兴趣爱好。
- **讲述自己的故事**：与对方分享相关的故事，拉近彼此的距离。

### 4. 避免成为负能量的接收桶
- **不要在对方开心时打扰**：避免在对方开心时询问负面问题。
- **不要成为负面情绪的垃圾桶**：避免在对方抱怨时过度关注。

### 5. 放入对方过去和未来
- **讲述与对方经历相关的故事**：例如，对方提到考试，可以讲述自己高中时的经历。
- **分享未来计划**：例如，对方提到旅游计划，可以表达愿意陪伴的意愿。

### 6. 针对性话题
- **根据对方兴趣爱好选择话题**：例如，对方喜欢旅游，可以分享旅游经历。
- **根据对方需求选择话题**：例如，对方正在备考，可以提供相关资料。

## 总结
本节课中，我们学习了如何找到无限话题，并运用这些技巧迅速拉近与对方的关系。通过发散思维、利用对方提供的话题、关注朋友圈、避免成为负能量的接收桶、放入对方过去和未来以及针对性话题等方法，我们可以更好地与对方沟通，建立良好的关系。
