# 乐嘉7套：5.乐嘉性格色彩全能训练营（入门）【勇略】：28.客户千差万别，方法怎么能从一而终？

## 概述
在本节课中，我们将学习如何运用性格色彩工具来识别和应对不同性格的客户，从而提高销售业绩。

## 第一类：红色性格客户
**特点**：爱说话，以体验为导向，容易被创意广告或个人喜好吸引。

**应对方法**：
- 尽可能赞美客户。
- 通过语言赞美、认可和描绘产品使用后的美好画面来提高推销成功率。

## 第二类：蓝色性格客户
**特点**：内心有想法，不直接表达，注重细节，完美主义。

**应对方法**：
- 用有条理的语言介绍产品，避免猜测客户隐私。
- 等待客户主动开口参与对话，销售成功一半。

## 第三类：黄色性格客户
**特点**：主见强，赞美无效，追求有用。

**应对方法**：
- 分析利害关系，让客户自己意识到产品的好处。
- 不需要过多语言介绍，客户会自己做出决策。

## 第四类：绿色性格客户
**特点**：随和，容易相处，对产品选择没主见。

**应对方法**：
- 主动出击，与客户交谈。
- 如果客户与家人一起来，要搞定他周围的人。

## 总结
本节课中，我们学习了如何运用性格色彩工具来识别和应对不同性格的客户。了解客户性格，运用相应的方法，将有助于提高销售业绩。
