# 加一情感《宅男宝典》课程02：聊天解码 🧠

## 12. 聊天时必须具备的心态

在本节课中，我们将要学习与女生聊天前应具备的核心心态。心态是成功沟通的基础，正确的态度能让你在互动中更加从容和有效。

很多兄弟知道聊天需要自信和坦然，但具体如何做到却感到困惑。感情经历少、面对女生不自然，这些都是常见问题。仅仅知道概念是不够的，我们需要具体、可操作的心态建设方法。

上一节我们介绍了心态的重要性，本节中我们来看看三种具体且关键的聊天心态。

### 一、 去做心态

传统的心态建设方法，例如自我暗示，效果往往有限。告诉自己“我很优秀”、“我很自信”，但若没有实际行动支撑，这种暗示在遭遇现实挫折（如搭讪被拒绝）时很容易崩塌。世界的普遍价值观（例如女生可能更青睐高富帅）不会因个人想象而改变。

因此，关键在于**去做心态**。这意味着你需要通过实际行动来改变自己，从而获得真实的自信。自信来源于你切实做出的努力和取得的进步，例如改造形象、坚持健身、广泛阅读、增加社交经验等。当你通过行动真正变得优秀，并获得他人正面的反馈时，自信便会自然建立。

从“去做心态”出发，可以衍生出两个对聊天至关重要的具体心态。

### 二、 提供价值的心态

在跟女生聊天之前，你需要明确一个核心原则：尽量多地去提供价值，而不是索取价值。

这个关系可以用一个简单的公式表示：**提供价值 = 产生感觉**。只有你提供的价值足够多，女生对你的感觉才会更好。

以下是提供价值的两种具体形式：

**1. 情绪价值**
情绪价值并非指一个人不停说话，而是指你能影响对方的情绪，制造情绪波动，让聊天过程充满趣味。例如，让对方一会儿笑，一会儿紧张，一会儿期待。

以下是三种可以制造情绪波动的聊天技巧：

*   **推拉**
    推拉技巧是指一下将对方推开（制造轻微否定或挑战），一下又将对方拉近（给予肯定或奖励），从而调动情绪。
    **代码示例（聊天对话）:**
    ```
    女：发来一张和闺蜜的合影（她更漂亮）。
    男：一看就知道右边的是你。（拉——暗示她好看）
    女：因为我最美是吗？哈哈。
    男：不是，右边的胸大一点。（推——开玩笑地否定）
    女：打死你。
    男：但我喜欢胸小点的，穿衣服好看。（再拉——表达偏好并夸奖）
    ```

*   **曲解**
    曲解是故意以幽默、自恋的方式误解对方的话，营造轻松有趣的氛围。
    **代码示例（聊天对话）:**
    ```
    女：昨天竟然梦到你了。
    男：占我便宜没？（曲解为对方对自己图谋不轨）
    女：（发表情）
    男：完了，肯定被你占了。
    ```

*   **打压**
    打压通常用在女生态度突然变冷淡、姿态变高，或为了维持自身框架时。注意，打压不是人身攻击，而是态度上的轻微反击。
    **代码示例（聊天对话）:**
    ```
    （当女生回复敷衍时）
    男：你平常都这样和人聊天的吗？
    （当女生在关系升温时突然退缩）
    女：…可能我也不是你喜欢的长相吧。
    男：不靠谱你还聊，是有多无聊？我反正没时间和我觉得不靠谱的人闲聊。（展示强硬框架）
    ```
    打压也可以与曲解结合使用，在拉伸关系时调节对方过高的自我感觉。

**2. 实际价值**
实际价值指女生可以直接通过视觉感知到的你的生活状态、品味或能力。可以通过分享高质量的照片或小视频来展示。
*   **示例**：在环境优美的餐厅吃饭时，分享美食照片；健身时，分享训练瞬间；开车时，分享驾驶座视角；做饭时，分享烹饪成果。

**索取价值**的反面例子则是不断提问，如“在干嘛？”、“吃了吗？”，这只会消耗对方的耐心。记住，提供价值的前提是你自身拥有价值，例如朋友圈需要精心经营，展示的照片或视频需要有品质。

### 三、 游戏者心态

游戏者心态能帮助你保持轻松，降低需求感，避免因过度紧张而表现失常。它主要包含两种子心态：

**1. 一边玩一边撩的心态**
聊天不应是你生活的全部。就像打游戏时来了信息，你会先完成当前操作再回复。与女生聊天时，也应如此：回复一条消息后，可以将手机放在一边，继续处理自己的事情，过段时间再查看回复。这能有效降低你的需求感和得失心。

**2. 有选择性的心态**
你不能只专注于一个女生。如果所有精力都集中在一个人身上，你会变得异常紧张，对方的一举一动都让你患得患失。这毫无吸引力。
这就像打游戏，只有一条命时你会非常紧张且容易失败；但当生命值满格时，你就能更从容地应对。拥有更多社交选择（即积累更多聊天资源）本身就是一种健康的心态和状态。

最后需要强调的是，即使你做得很好，也可能有女生不理你。我们不是人民币，无法让所有人都喜欢。你的精力应该放在那些对你反馈不错的女生身上，而不是纠结于一开始就对你冷淡的人。在自身没有犯错的前提下，初期就不友好的人，后期很难通过聊天扭转印象。

本节课中我们一起学习了与女生聊天的三大核心心态：**去做心态**是基础，它催生了**提供价值的心态**（包括情绪价值和实际价值）和**游戏者心态**（一边玩一边撩、有选择性）。掌握这些心态，能让你在聊天前做好充分准备，保持自然、自信的状态，从而更有效地建立吸引。记住，选择与那些给予你积极反馈的人互动，是高效社交的关键。
