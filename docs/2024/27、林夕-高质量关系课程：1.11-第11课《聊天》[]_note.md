# 27、林夕-高质量关系课程：1.11-第11课《聊天》🗨️

## 概述
在本节课中，我们将学习如何与男性进行有效的聊天，包括聊天的价值、分寸感、主动方式以及如何通过聊天建立高质量的关系。

## 聊天的价值
**公式**：聊天 = 中心话题 + 能量 + 放松

每个人希望自己是聊天的中心，避免成为情绪垃圾桶。通过让对方成为聊天的中心，可以让他感到能量和放松，从而增加聊天的兴趣。

## 聊天的分寸感
**公式**：分寸感 = 聊天次数 + 频率 + 离场时机

1. **聊天次数**：避免成为最后说话的人，让男性有发挥空间。
2. **聊天频率**：根据对方的回复速度调整自己的回复速度，保持分寸感。
3. **离场时机**：通过高价值离场，展示自己的高价值，同时给对方留下想象空间。

## 如何主动
**公式**：主动 = 小事主动 + 大事被动

1. **小事主动**：在小事上主动，如主动聊天、约局等。
2. **大事被动**：在重要事情上，如约会、求婚等，由男性主动。

## 主动聊天的话题内容
**公式**：高质量聊天内容 = 提问 + 分享 + 投其所好

1. **提问**：提问对方擅长的问题，展示自己的关注和兴趣。
2. **分享**：分享新鲜事物、有趣视频、个人成就等。
3. **投其所好**：根据对方的喜好分享内容。

## 如何提问
**公式**：黄金三部曲 = 赞美 + 细节疑问 + 个性点评

1. **赞美**：通过眼神、点头等方式表达赞美。
2. **细节疑问**：针对对方的话题进行提问，展示自己的关注和思考。
3. **个性点评**：对对方的回答进行个性化点评，展示自己的独特见解。

## 如何讲故事
**公式**：讲故事 = 人物 + 地点 + 时间 + 反转

通过讲述故事，展示自己的优点和经历，让对方更了解自己。

## 总结
本节课中，我们学习了如何与男性进行有效的聊天，包括聊天的价值、分寸感、主动方式以及如何通过聊天建立高质量的关系。希望大家能够通过练习，掌握这些技巧，享受高质量的感情。
