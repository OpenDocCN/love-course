# 男性提升《好逑情感》：3、吸引女生的7个思维：3_7--七、博弈思维

## 概述

在本节课中，我们将学习如何运用博弈思维来吸引女生。博弈思维是指在与女生交往中，保持自己的原则和底线，不轻易屈服于对方。

## 博弈思维的定义

博弈思维是指两个人相互较量，相互PK的意思。你有你的思维想法，我也有我的原则，还有框架，我不会轻易服从你设定的规则。

## 博弈思维在恋爱中的应用

在恋爱的过程当中，博弈思维同样也是非常重要的思维。以下是对比奖品思维和博弈思维：

- **奖品思维**：我是奖品，你来追我。
- **博弈思维**：我有原则，我有底线。我有想法，我有我自己的三观，我不服从于你。

## 博弈思维的例子

以下是一些运用博弈思维的例子：

1. **女生说“男人没有一个好东西”**：
   - **错误回答**：我是一个好男人，请不要这样想。
   - **正确回答**：那看来你要削发为尼了呀。那我知道有一个地方，那个地方呢在招尼姑，而且和尚有很多可以给你涟漪一下。

2. **女生说“你的这些招数呢对小女生有用”**：
   - **错误回答**：套路是我学的，但撩你是真心的。
   - **正确回答**：看来你是个有经历，有故事的女人呢，我不会嫌弃你经验丰富的。

3. **女生说“最近有一个男生总是缠着我”**：
   - **错误回答**：是谁呀，他干嘛的呀？他为什么纠缠你呀？告诉我去收拾他。
   - **正确回答**：哎呀，你这么有魅力，人家想跟你一结金兰，也是理所当然的，哈，一波六走起来。

## 博弈思维的注意事项

1. 女生有时候打压你的时候，让你难堪的时候，让你服从他的时候，他有的时候是无意识的。
2. 博弈思维是一门艺术，你不能生硬的博弈。要学会在不服从的时候，把幽默的元素加进去。
3. 博弈思维要灵活运用，不要让技巧成为自己的绊脚石。

## 总结

本节课中，我们学习了如何运用博弈思维来吸引女生。博弈思维是指在与女生交往中，保持自己的原则和底线，不轻易屈服于对方。通过运用博弈思维，我们可以更好地吸引女生，建立稳定的关系。
