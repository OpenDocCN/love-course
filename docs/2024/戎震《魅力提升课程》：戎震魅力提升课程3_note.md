# 戎震《魅力提升课程》：戎震魅力提升课程3_note

在本节课中，我们将要学习人际交往中的一个核心原则：如何通过引导他人谈论他们自己，来有效提升你的个人魅力和社交关系。理解并运用这一原则，是建立良好人际关系的关键。

上一节我们介绍了基础社交概念，本节中我们来看看一个具体且强大的实践技巧。

## 核心原则：人都喜欢谈论自己

人都喜欢谈论自己。人对自己最感兴趣。即使一个人今天早上父亲去世了，但如果另一个人今天早上智齿发炎了，后者一整天思考的也绝对是他智齿发炎这件事。

因为自己的事情与自己更相关。如果人类都只关心自己以外的事物，那么在自然灾害面前，原始人类早就灭绝了。人只有先关心自己，先关心自己的生存与喜怒哀乐，才能更好地在地球上生存。这是一种天然的自然倾向。

人类是一种高级动物。在研究人类时，我们需同时考虑其社会性（人的部分）和生物性（动物的部分）。我们有时过度强调社会性，却忽视了生物性。因此，人本质上是自私的，人在意自己的喜怒哀乐、利益和所有物。

## 实践应用：如何提升关系与魅力

基于上述原则，若想提升与别人的关系和自身魅力，需记住一个核心方法：**引导别人谈论他自己**。

这同样适用于赠送礼物。你要送别人感兴趣或擅长的东西。例如，若某人家里陈列古董，你可送古董相关礼品；若陈列高尔夫球杆，则可送高尔夫球类产品。通过外在物品判断其喜好是关键。

但需注意，表象可能具有欺骗性。例如，我家里书很多，但若你送我书，我可能并不喜欢，因为我读书多为工作所需，我个人最大的乐趣是打游戏。因此，深入分析对方真实喜好最为理想。

## 沟通中的常见误区与正确做法

人最大的乐趣在于研究、表达和发展自己。因此，喜欢与他人讨论自己的人，最容易受人欢迎。

但这里的“讨论自己”有一个关键前提：**讨论的对象是对方，而不是你自己**。因为你对别人而言可能无足轻重。在对方看来，你的私事可能远不如他们自己的小事重要。

许多人交往时犯的错误是滔滔不绝地谈论自己的事。这种将别人当作情绪“树洞”的行为是错误的。例如，我常收到许多陌生私信，对方不分青红皂白就将自己的问题倾倒给我，期望免费获得解答。这在没有建立任何利益关联或友好度的前提下，是很难发生的。

人与人之间通常是在产生利益关联后，才会建立连接、提升友好度并愿意互相帮助。因此，在萍水相逢时，最重要的一点是：**你需要先对他人有所施予或恩惠**，别人才会有所领受。

## 具体行动指南：说话的藝術

说话是最简单的实践艺术。你无需立即送礼，但在例如饭局等场合，你可以围绕主人感兴趣或擅长的话题展开交流。

以下是具体操作步骤：

*   **寻找话题**：例如，他当过兵、某项运动特别出色、历史知识丰富等。
*   **跨越自尊心障碍**：这是对自己的要求。大部分普通人难以跨越自尊心，而你的社交策略可以是“喂养”他们的自尊心。
*   **提供情绪价值**：为了让对方自尊心得到满足，你可以说对方爱听的话。你的目的不是评判对方的自尊、自负或自恋，而是为了提升与他的友好度。因此，说话可以怎么让对方舒服就怎么说。

戴尔·卡耐基《人性的弱点》的核心观点之一便是如此：如何记住并称呼对方的名字，如何引导对方谈论他们自己。

## 如何留下深刻好印象

若想给别人留下好印象，就诱导对方谈论以下内容：

*   他自认为骄傲的事。
*   他自认为做得最成功的事。
*   他自认为家族里光荣的事。

哪怕这些事情在旁人看来不值一提。但如果你能成为那个引导他讲述这些事的人，你就会给他留下非常深刻的印象。

因为对方在与你聊天时，会感到十分舒适。原因在于，你的聊天过程始终在围绕“他”展开。当然，对方可能会反问你的情况，这时简略回应即可，这本身就是人际交往中最佳的润滑剂。
