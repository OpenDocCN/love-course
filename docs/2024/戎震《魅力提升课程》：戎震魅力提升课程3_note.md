# 戎震《魅力提升课程》：魅力提升课程3

## 概述
在本节课中，我们将探讨人类的天性，特别是人们对自己话题的兴趣，以及如何利用这一点来提升个人魅力。

## 人类天性：谈论自己
**公式**：人都喜欢谈论自己，因为自己的事情与自身更相关。

- 即使遭遇重大不幸，人们也会优先关注自身的事情。
- 这种倾向是生物发展的规律，符合自然界的生存法则。

## 利用人性提升魅力
**公式**：引导别人谈论自己，可以提升人际关系的友好度和连接感。

- 送礼物时，选择对方感兴趣或擅长的东西。
- 分析对方喜好，送出合适的礼物。
- 例如，对方喜欢古董，则送古董；对方喜欢高尔夫，则送高尔夫相关产品。

## 交流的艺术
**代码**：说话是一种艺术，要学会引导对方谈论自己。

- 在饭桌上，围绕对方感兴趣的话题进行交流。
- 例如，对方擅长体育，则谈论体育；对方英俊，则谈论外貌；对方知识丰富，则谈论历史。
- 越过自尊心障碍，哄骗对方谈论自己。

## 留下深刻印象
**公式**：诱导别人谈论自己的引以为傲之处，可以留下深刻印象。

- 引导对方谈论自己认为最自豪的事情。
- 即使这些事情在别人看来不值一提，但对方会感到舒适。

## 总结
本节课中我们一起学习了如何利用人类天性，引导别人谈论自己，从而提升个人魅力。记住，关注对方，引导对方谈论自己，是建立良好人际关系的关键。
