# 课程03：成为说话高手 🗣️
## 第26节：活学活用——成为有逻辑的沟通高手 🧩

在本节课中，我们将学习如何将之前课程中介绍的各种逻辑框架灵活应用于实际沟通场景。核心在于理解框架并非一成不变的公式，而是可以根据具体情境进行识别、调整和创新的工具。

不知不觉，我们的课程已接近尾声。最后一节，我想探讨一个贯穿前面所有小节的核心话题：如何更好地应用所学的逻辑框架，以及如何灵活运用它们。

我将围绕以下三点展开讲解。

### 一、识别沟通场景 🎯

在之前的课程中，我们介绍了许多不同的沟通逻辑与框架。然而，仅仅记住它们是不够的，关键在于用对场景。识别场景是选择正确沟通逻辑的第一步。

场景如同开关，只有打开正确的开关，我们才能取出合适的钥匙，打开沟通的大门，即找到合适的逻辑框架。

以下是识别场景的关键点：

*   **明确沟通目标**：当前的场景是说服对方采纳建议，还是提供反馈，或是希望对方改正某个错误？
*   **理解逻辑的相通性**：聪明的你可能已经发现，某些场景下所需的沟通逻辑内容存在相通之处，部分模块是重合的。例如，多个框架都涉及“给出建议”，多个框架都涉及“分析现状”或“分析问题原因”。
*   **灵活选择逻辑**：因此，面对同一场景，可能有多种逻辑或内容模块都适用。在实际沟通时，我们可以基于具体场景，灵活选择最合适的内容与逻辑。

### 二、灵活运用逻辑框架 🧠

上一节我们介绍了如何识别场景，本节中我们来看看如何在实际运用中保持灵活性。所谓“活学活用”，是指每个逻辑框架的运用顺序并非固定不变。

我们在课程中讲解的顺序，通常是普适性最强、最普遍的一种，在默认状态下这样沟通比较可靠、安全且合理。但大家完全可以根据实际情况，调整模块的顺序。

以下是一些调整顺序的实例：

*   **“3-1框架”的顺序调整**：在相关课程中，我们建议的顺序是先说“愿景”，再说“分析评估”，最后说“建议”。但有时也可以调整，例如先提出“建议”，然后阐述支持该建议的“分析”，最后描绘“愿景”。这取决于沟通对象更关注什么，以及我们对场合的判断。
*   **“B.I.E.R框架”的灵活运用**：在提出期许时，我们介绍了包含**行为**、**影响**、**期望**、**结果**四个部分的框架。这个顺序同样可以打乱，甚至只抽取其中两三个模块。例如，当与对方信任基础良好时，可以直接说“期望”和“结果”，必要时再补充“行为”和“影响”。

总之，课程中讲解的逻辑是一个重要参考。大家既可以按照原有顺序沟通，也可以保持灵活性，不被框架束缚。我相信，随着大家对框架掌握得越来越熟练，也能找到自己认为最合适的逻辑顺序。

### 三、总结与创新框架 🚀

无论我们讲解多少种沟通逻辑，总会遇到新的情况与场景，需要我们思考如何应对。我希望大家学完课程后，能够自己总结出梳理沟通逻辑的方法。

当你练习和思考得足够多，会发现许多逻辑框架之间存在共通之处。

例如：

*   在某些场景下，沟通总会涉及“展望未来”，再“回到当下要做什么”。
*   在另一些场景下，沟通总会以“陈述客观事实”作为建议的基础。
*   还有一些场景，沟通总会包含“表达对对方的理解”。

以这些观察为基础，当我们遇到不同的场合时，就可以创造属于自己的新框架。即利用课程中学到的内容模块，对它们进行重新组合，发明出适合自己或特定场景的新逻辑框架。

### 课程总结

本节课中，我们一起学习了成为有逻辑沟通高手的三个关键：**识别场景**、**活学活用**以及**总结创新**。核心在于理解沟通逻辑是工具而非枷锁，通过灵活调整与组合，让表达更清晰、更有力，从而帮助我们在工作与生活中更有效地达成目标。
