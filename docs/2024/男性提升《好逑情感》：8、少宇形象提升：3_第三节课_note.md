# 男性提升《好逑情感》：8、少宇形象提升：3_第三节课_note

在本节课中，我们将要学习体型偏瘦的男生应如何进行穿搭。课程将分析偏瘦体型的常见缺点，并提供具体的单品选择与搭配技巧，帮助你通过着装改善视觉形象。

体型偏瘦的男生常被认为穿衣服包容性强。但偏瘦容易让女生感觉缺乏安全感，甚至给人弱不禁风的印象。因此，学习针对性的穿搭方法非常重要。

## 分析偏瘦体型的缺点

上一节我们概述了课程目标，本节中我们来看看偏瘦体型主要存在的两个缺点。

### 缺点一：肩膀较窄

肩膀窄是偏瘦者穿衣服时最头疼的问题。窄肩会显得撑不起衣服，容易显得没有气质。若同时存在圆肩、驼背、探颈等不良体态，更容易显得猥琐。因此，矫正体态是首要任务。

以下是纠正不良体态的建议：
*   避免耸肩、驼背和探颈。
*   可在网上搜索针对性的矫正运动视频进行练习。
*   这些矫正运动通常每天只需几分钟，坚持一段时间即可看到体态改善。

回到穿搭话题，针对肩膀窄的问题，主要有三种解决思路。

以下是三种具体的解决方式：
1.  **修饰肩膀线条**：穿着材质**硬朗**、**偏厚重**的衣服，或通过**叠穿**增加厚度，以拉直和延伸肩线。
2.  **转移视觉注意力**：穿着**有领子**的衣服，将他人视线吸引至颈部和头部，从而忽略肩窄的缺点。
3.  **模糊真实肩线**：选择**落肩款**的衣服。落肩款指肩线设计比正常位置更靠外的款式，能拉伸肩部视觉宽度。

落肩款衣服能使肩膀看起来明显变宽，整体造型也更显潮流。若找不到合适落肩款，购买**大一码**的普通T恤也能起到类似效果，但需注意衣服长度不宜过长。

### 缺点二：身材单薄

偏瘦者身材通常单薄，侧面看如纸片人，显得弱不禁风。因此，应避免穿着过于紧身的衣服，那会加剧单薄感。

为解决身材单薄问题，通常有两个方法。

以下是两个具体的解决方法：
1.  **选择合身或宽松版型**：**合身**的衣服不强调缺点；**宽松**的衣服能掩饰身材，让他人以为你只是偏爱潮流风格。
2.  **穿出层次感**：通过**两件或以上**单品叠穿营造层次。
    *   **夏季**：可使用**假两件**T恤。
    *   **春秋季**：在T恤外加一件外套，如**牛仔夹克**、**工装外套**或**连帽卫衣**。
    *   **冬季**：可尝试内搭衬衫，外加**羊绒针织衫**。

## 提升整体清爽感

在分析完主要缺点后，我们来看看提升整体形象的两个基础要点。干净整洁是赢得好感的关键。

以下是两个基础的变帅思路：
*   **保持清洁感**：衣物需勤洗勤换。可使用**淡香型洗衣液**，让衣物散发清爽气味。
*   **注重面部整洁**：进行基础护肤。每晚使用**水**和**乳液**；白天可使用**控油产品**，并可学习画**眉毛**、涂**粉底**等基础妆容技巧。

## 偏瘦男生的单品选择指南

在了解缺点和基础要点后，我们来具体探讨偏瘦男生应如何选择各类单品。

### 1. T恤

T恤是最常见的单品，用于单穿或内搭。选择时需注意三点。

以下是T恤选择的三个要点：
1.  **领口**：瘦人通常脖子细长，**不建议选择V领**，因为V领会拉低上半身视觉顶点，显矮。**更推荐圆领**。
2.  **厚度（克重）**：克重指面料每平方米的重量（单位：`g/m²`），衡量厚度。
    *   克重过低（如`<180g/m²`）易透色。
    *   克重越高，轮廓感越强，越能修饰肩线，但透气性和柔软度会降低。
    *   **打底**：推荐`180-200g/m²`。
    *   **单穿**：浅色推荐`>190g/m²`（防凸点），深色推荐`>180g/m²`。
3.  **肩线**：单穿时，**推荐尝试落肩款**，能有效模糊真实肩线，掩饰肩窄缺点。需注意落肩款同样有轻微显矮效果，但可通过整体搭配平衡。

颜色方面，**黑、白、灰**最经典不易出错。想显年轻活力可选**牛仔蓝、藏青、烟灰、米色**；想显沉稳可选**墨绿、豆沙色、咖啡色、暗红色**。

### 2. 衬衫

衬衫是提升气质的利器。选择合身版型并按推荐颜色选择，通常不会出错。

**白色衬衫**是极佳选择，它不挑肤色和身材，单穿、内搭、外搭皆可，搭配方便，建议男生衣柜常备。

若将衬衫作为**外搭**，内配T恤是不错的搭配。注意内搭T恤颜色不宜过深，否则透出会显得色彩脏乱，影响整体效果。

### 3. 卫衣

卫衣宽大舒适且百搭。对于偏瘦者，**推荐连帽卫衣**。

连帽设计能增加领口层次感，吸引视线，改善肩窄的视觉问题。同时，帽子能从侧面增加身体厚度，掩饰身材单薄。

颜色首选经典的**黑、白、灰**。连帽卫衣可单穿、内搭或外搭。常见搭配如：内搭T恤，外穿连帽卫衣，再外搭牛仔夹克。

### 4. 牛仔夹克

牛仔夹克是最推荐的外套单品。其**搭配门槛低**，不挑身高颜值，且能与几乎所有单品搭配。

硬朗的材质能很好地修饰身体线条，**让瘦人显壮，胖人显瘦**。颜色推荐**浅蓝、深蓝、灰、黑**等常规百搭色。

内搭T恤、衬衫或连帽卫衣皆可。在春秋微凉天气，**内搭连帽卫衣**尤其推荐，层次感丰富。

### 5. 裤子

最后推荐几条基础百搭的裤子。

以下是推荐的裤装类型：
*   深色牛仔裤
*   蓝色牛仔裤
*   西装裤
*   休闲裤

这些裤子实穿且百搭，建议每位男生衣柜至少备有一条。
