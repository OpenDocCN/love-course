# 社交情商训练营：07_2-1、高情商聊天口才修炼3 🎤

## 概述
在本节课中，我们将学习如何通过日常练习和技巧提升聊天口才，从而提高社交情商。

## 锻炼方法
### 1. 说话机会
**公式**：多说话 = 好口才
- 不要放过任何可以说话的机会，无论是与家人闲聊，还是与司机、理发师、售货员或同事交流。
- 口才的提升需要多说话和多练习。

### 2. 聊天内容
- 初始阶段，可以从衣食住行、工作、爱好、理想等话题开始。
- 如果不知道聊什么，可以像查户口一样提问。
- 不断提问并配合聊天课中的反馈技术。

### 3. 讲故事
- 准备一些自己的故事，并学会用说话展示自己的价值。
- 分享故事以建立信任，并持续锻炼讲故事的能力。
- 讲故事时，可以加入幽默元素。

### 4. 总结与反思
- 总结自己的错误，并反思如何改进。
- 通过实战和总结反思，最多不超过3个月，口才将得到显著提升。

## 总结
本节课中我们一起学习了如何通过日常练习和技巧提升聊天口才，从而提高社交情商。通过不断练习和总结反思，我们都可以成为说话的高手。
