# 重开计划（大橙2.0）2499：19、聊天中的思维脑图梳理（上）

## 概述

本节课将介绍重开计划2.0课程中关于聊天思维脑图的梳理方法，帮助学员更好地应对不同类型的女生，提高聊天成功率。

## 课程梳理

### 第一部分：重开计划2.0课程梳理

1. **认知觉醒、价值观重塑、思维开拓、恋爱规划学习**
    - 认知觉醒：打破固有观念，开拓思维。
    - 恋爱规划：了解自身需求，规划恋爱过程。
    - 思维开拓：学习新知识，提升思维能力。
    - 恋爱规划学习：学习恋爱技巧，提高恋爱成功率。

2. **深度了解自己、了解女生、了解男生**
    - 深度了解自己：认识自身优点和不足，明确提升方向。
    - 了解女生：了解女生心理，提高沟通效果。
    - 了解男生：了解男生心理，增进彼此了解。

3. **了解当下使命、接受自己的无能、热爱生活、获取能量后带领女生**
    - 了解当下使命：明确人生目标，提升自信。
    - 接受自己的无能：接受自身不足，专注于提升。
    - 热爱生活：培养兴趣爱好，丰富生活。
    - 获取能量后带领女生：在熟悉领域带领女生，展现自信。

4. **个人提升**
    - 快速增值：提升自身价值，提高脱单效率。
    - 金钱管理：合理分配金钱，提高生活质量。
    - 时间管理：合理安排时间，提高效率。
    - 个人品牌建设：打造个人品牌，提升吸引力。

5. **高效脱单**
    - 核心思维：运用大数据思维，提高脱单效率。
    - 看上白五大核心思维：识别女生类型，选择合适聊天方式。
    - 避开聊天中错误做法的核心方法：避免聊天失误，提高成功率。

### 第二部分：聊天中的思维脑图梳理

1. **开场白**
    - 开场白不能犯错：避免聊天失误，提高成功率。
    - 看上白：识别女生类型，选择合适聊天方式。

2. **不同类型女生聊天方式**
    - 对你态度好的女生：填充安全感，判断接受程度，逐步升级关系。
    - 对你态度一般的女生：化解误解，填充安全感，判断接受程度，逐步升级关系。
    - 对你态度不好的女生：不解释，不纠结，填充安全感，否定负面标签，逐步升级关系。

3. **认识很久的女生聊天方式**
    - 透朋圈：展示自身魅力，吸引女生注意。
    - 聊天：了解女生想法，寻找共同话题。

4. **不聊天不想恋爱的女生聊天方式**
    - 测试：了解女生想法，寻找合适聊天方式。

本节课介绍了重开计划2.0课程中关于聊天思维脑图的梳理方法，帮助学员更好地应对不同类型的女生，提高聊天成功率。学员需要掌握开场白、看上白、不同类型女生聊天方式等技巧，并结合自身情况进行灵活运用。
