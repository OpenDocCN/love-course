# leon《全套课程合集》：leon《Alpha通识》：四.Alpha心脏正视恐惧并超越_ev

## 概述

在本节课中，我们将学习如何正视恐惧并超越它，从而获得真正的勇气和成功。

## 1. 人为什么活着？

**问题**： 人为什么活着？

**答案**： 生存和产业。

**讨论**： 为什么很多人会是虚无主义者？因为他们没有证实自己的攻击性和支配性，压抑支配性的人才会虚无主义。

## 2. 支配与不被满足

**问题**： 支配与不被满足的副产物是什么？

**答案**： 喜欢攻击别人，包括低级的攻击形式（如谩骂）和高级的攻击形式（如装逼）。

**讨论**： 装逼的危害：消耗自己、容易与人起冲突、在人际关系中失去亲密性、得不到尊重。

## 3. 勇气的边界

**问题**： 一个人勇气的边界是什么？

**答案**： 一个人勇气的边界就是一个人生命的边界。

**讨论**： 勇气与边界和压迫斗争思想的内核。

## 4. 学习观

**问题**： 如何构建一个学习的认知？

**答案**：

1. 找对标：找有结果的人学习。
2. 系统性学习：学习有结果人的知识结构。
3. 阶段性学习：每个阶段只学一个概念，几个概念就能把一个学习弄穿了。

## 5. 冒险

**问题**： 为什么冒险很重要？

**答案**： 冒险才能获得新鲜的一手经验，从而才能精准的形成自信。

**讨论**： 冒险的成本和收益、如何做好措施、如何适应痛苦、如何刻意练习、如何复盘、如何借助教练咨询反馈。

## 6. 获得成功

**问题**： 如何获得成功？

**答案**：

1. 搭讪：处男单身的第一个任务是找女朋友。
2. 影响别人：支配你的父母、老师、同学。
3. 攻击：学会攻击漏洞，攻击错误的思想，替换成正确的、更有效的、更有统筹性的思想。

## 总结

本节课中，我们学习了如何正视恐惧并超越它，从而获得真正的勇气和成功。希望这节课能够帮助你成为一个更勇敢、更自信的人。
