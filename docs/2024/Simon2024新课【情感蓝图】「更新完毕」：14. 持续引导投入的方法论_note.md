# Simon2024新课【情感蓝图】「更新完毕」：14. 持续引导投入的方法论 📚

在本节课中，我们将学习如何通过持续引导投入的方法论来增强人际关系中的吸引力。

## 1. 引导投入的底层原因

**公式**：引导投入 = 增加对方筹码 + 提高沉默成本

引导投入的目的是让对方加大投入，从而无形中增大我们的价值，让对方在离开时面临较大的损失，减少其抽离的可能性。

## 2. 引导投入的形式

以下是几种常见的引导投入形式：

- **金钱投入**：对方为了得到你而为你投资金钱。
- **时间投入**：对方为了你而牺牲自己的时间。
- **机会成本**：对方为了你而放弃其他更好的机会。
- **精神投入**：与对方在一起时产生强烈的情绪体验，为对方投入大量的时间、经历和筹码。

## 3. 精神投入的指标

- **情绪体验**：与对方在一起时是否有过强烈的情绪体验。
- **时间、经历和筹码投入**：为对方投入了多少时间、经历和筹码。

## 4. 引导精神投入的时机

当对方开始在意你的回复速度、朋友圈内容和未来规划，并主动询问你问题时，是引导精神投入的最佳时机。

## 5. 注意事项

- **避免过早表明底牌**：在对方尚未达到精神投入阶段时，过早地表明底牌只会适得其反。
- **正确引导**：在引导对方投入时，要耐心等待时机，并正确引导。

## 总结

本节课中，我们学习了持续引导投入的方法论，包括其底层原因、形式、指标和注意事项。通过掌握这些方法，我们可以更好地增强人际关系中的吸引力。
