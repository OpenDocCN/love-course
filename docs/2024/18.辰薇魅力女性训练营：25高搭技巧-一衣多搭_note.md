# 18.辰薇魅力女性训练营：25高搭技巧-一衣多搭

## 概述

在本节课中，我们将学习如何通过一衣多搭技巧，用有限的衣物打造多样化的风格，节省开支并提升形象。

## 一衣多搭的精髓

**公式**：色彩、风格、场合、服装的款式与质感融合

**代码**：选择主色调 → 考虑场合与角色 → 搭配饰品

## 买衣服的思路

1. 以精良的基础款为主，个性款为辅。
2. 选择多场合能穿的衣服，即基础款。
3. 个性小搭配作为辅料。

## 衣橱管理

1. 对衣物进行分类：T恤、毛衣、衬衫、配饰等。
2. 色彩区隔。
3. 搭配好几套服装，挂起来陈列。

## 搭配思路

1. 寻求时间、地点、场合和角色。
2. 选择主色调。
3. 搭配饰品。

## 搭配案例

1. 黑色短款羽绒服 + 绿色卫衣 + 牛仔裤 + 老爹鞋
2. 白色卫衣 + 牛仔裤 + 洋气小帽子 + 切尔西靴
3. 黑色夹克羽绒服 + 裙装
4. 西装套装 + 白衬衫 + 裙装
5. 西装套装 + 白T恤 + 牛仔裤
6. 西装套装 + 衬衫 + 短裤

## 小技巧

1. 针织衫当成外搭或饰品。
2. 注意风格的统一性。
3. 使用饰品辅助搭配。
4. 选择合适的饰品，避免繁复。

## 总结

本节课中，我们学习了如何通过一衣多搭技巧，用有限的衣物打造多样化的风格。希望同学们能够将所学知识应用到实际生活中，提升自己的形象和气质。
