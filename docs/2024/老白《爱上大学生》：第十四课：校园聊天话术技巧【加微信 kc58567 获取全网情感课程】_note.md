# 老白《爱上大学生》：第十四课：校园聊天话术技巧

## 概述
在本节课中，我们将学习如何运用校园聊天话术技巧，提升与女生聊天的效果。

## 安全系数高的回复方式
**安全系数高**，即不会让女生感到尴尬或不适。

### 常见场景及回复
以下是针对女生不同回复的回复模板：

1. **女生说在上班**：
   - 你可以说：“这么勤劳吗？😊”
   - 或者：“给你一朵小红花。”
   - 或者：“看来是一只勤劳的小蜜蜂。”
   - 或者：“上班玩手机不怕被领导抓吗？”
   - 或者：“这么辛苦，带你去私奔好了。”
   - 或者：“周末还在加班，你们老板这么喜欢虐待小动物。”

2. **女生说宅着**：
   - 你可以说：“嗯周末在家宅着，就是对周末的不尊重，对周末的浪费，就是在犯罪，你知道吗？”
   - 然后加上一个偷笑的表情，这样就好像你在开玩笑。

3. **女生说在看电视**：
   - 你可以说：“嗯我好想变成一个电视加一个害羞的表情。”
   - 这句话适合在你和女生暧昧的时候去聊。

4. **女生说在躺着**：
   - 你可以说：“😊，哎，躺成了一个大字吗？”
   - 如果女生反应很好，你可以说：“哎挪一挪腾个地儿，我来陪陪你。”

5. **女生说在趴着**：
   - 你可以问：“你是吃太多了还是亲戚来了？”

6. **女生说在准备考试**：
   - 你可以说：“选择题前五兆。😊，ABCDD不用谢我嗯，我的名字叫雷锋。”
   - 或者：“考得好，奖励你吃好吃的，趁机呢去邀约一下这个女生。”

7. **女生说在吃饭**：
   - 你可以说：“多吃点宝宝还在长身体的时候，或者说替我多吃点，我减肥，你吃饱了有力气了，给我加油。”

## 特殊情况应对
1. **女生说“我不温柔”**：
   - 你可以说：“没关系。😊，会有天使替我来爱你。”
   - 或者：“哎呀今天是个大喜的日子啊，终于帮你找到了你单身的原因了。”

2. **女生说“切”**：
   - 你可以说：“你想切哪里，我这里有贴身的指甲剪，可以借你用一下。”

3. **女生说“滚去死”**：
   - 你可以说：“你再说一遍。”
   - 如果女生再说一遍，你可以说：“哎宝宝真乖，或者说宝宝真听话就知道咱家我说了算。”

4. **女生说“不”**：
   - 你可以说：“哎，那你要听话乖乖的，我吓唬你了。”

5. **女生夸自己**：
   - 你可以说：“天生的美少女一般都是这么安慰自己的。”
   - 或者：“就喜欢你自恋的样子。”
