## 教程概述

在本节课中，我们将学习亲密关系技巧中的伞状收缩动作，该动作由三个部分组成，旨在增强亲密关系中的身体接触和情感交流。

## 动作一：散状收缩

**以下是散状收缩的步骤：**

1. 用双手中指按压阴茎海绵体两侧。
2. 手指向上移动，指尖逐渐贴合阴茎体，直至手根部。
3. 从手根部向下，手指从龟头方向缓慢向下移动。
4. 手指向上移动，整个手指贴合阴茎和龟头。

**公式描述：**

```
散状收缩 = 按压海绵体 + 手指向上移动 + 手指向下移动 + 手指贴合阴茎和龟头
```

## 动作二：阴茎根部托举

**以下是阴茎根部托举的步骤：**

1. 右手托住阴茎根部。
2. 左手轻轻缓慢地向下压阴茎，根据对方情况调整力度。
3. 缓慢顺时针打圈。
4. 缓慢逆时针打圈。

**公式描述：**

```
阴茎根部托举 = 托住根部 + 向下压阴茎 + 顺时针打圈 + 逆时针打圈
```

## 动作三：包皮系带按摩

**以下是包皮系带按摩的步骤：**

1. 左手握住阴茎根部。
2. 右手大拇指在包皮系带人字处打圈。
3. 缓慢、轻柔地按摩。

**公式描述：**

```
包皮系带按摩 = 握住根部 + 打圈 + 按摩
```
