# Ricky基础私教：08.第八课：实际操作与推开

## 概述
在本节课中，我们将学习如何在实际操作中运用推开技巧，以及如何通过改变自己的行为来控制场面。

## 实际操作案例分析
**例子**：我吃完饭准备坐电梯回家，那天我穿着拖鞋短裤，头发也没整理。我小区有平台，一楼到五楼再上自己的楼层。我本来是要上三楼，这时一个女生进了电梯，她按了五楼。我知道她肯定穿着特别高贵，被爱马仕的包吸引。这是我的目标。我继续坐到五楼，下了电梯，我问她：“你也住这里吗？”
**重点**：
- 她穿着非常上流，很高贵。
- 我那天穿着拖鞋短裤，头发很短，如果搭讪她时有一点虚，之后就没有后续了。
- 如果你有一点点兴趣，她就会觉得你不配，然后躲你。

## 推开技巧
**理论**：如果你怕她，她就会天然觉得你是弱者，你不配她。所以，即使我们的外形差距很大，我还是成功留了她的微信。这是非常重要的。你需要把它看待得很普通，并且一点也不能心虚。

## 约会实践
**第二天**：我们开始约会，第二天就约了个咖啡。他在一楼大堂等我，这次背的也是爱马仕的包，然后我们互相打招呼。面对这种每只包都超过10万的女生，搭讪时的broing要30万。

**注意**：
- 在大堂见面后，我扭头就走，大概和他隔了1.5米，在路上我也只是简单的聊了两句。
- 注意，在走路的时候，不要看着她，因为这样意味着你需要她的反馈和看到她的表情，这是掉价的行为。
- 面对一些比较有吸引或者价值没那么高的女生，你需要更nice一点，它的价值越高，你的keep层级越高。

## 咖啡厅约会
**场景**：我到了咖啡厅后，和服务员说：“你好，我要一个拿铁。”然后问他：“你要什么？”点完之后我们就坐下了。我的内心很想和她亲近，她白净、身材好、漂亮，身上的香水很好闻。

**注意**：
- 面对价值这么高的女生，一定要更keep。我描述一下我当时的状态比你们在youtube看到的要更keep一点，更冷一点。我没有那么把它当一回事儿，因为我偶尔也会玩手机。
- 我们所说的keep，不是消极，不是不理她是nice，但是我们是有距离感的。

## 接吻尝试
**场景**：聊到一个人住的情况，他给我看了他家的猫，我顺势和他说：“好久没玩猫了，我去你家玩一下吧。”他犹豫了一下，我立马打断他：“你不会家里有人吧？”他解释说：“他家里没有人，想去就去吧。”
**注意**：
- 当你提出质疑，女生是会自己解释的。他会主动拉近你，前提是你前面没掉吸。
- 如果他躲开，你迎上去准备接吻，这是冲，而往后坐远离他，这是退。这会让你回到原来的位置上。

## 总结
本节课中我们一起学习了如何在实际操作中运用推开技巧，以及如何通过改变自己的行为来控制场面。希望这些技巧能够帮助你在现实生活中更好地与人交往。
