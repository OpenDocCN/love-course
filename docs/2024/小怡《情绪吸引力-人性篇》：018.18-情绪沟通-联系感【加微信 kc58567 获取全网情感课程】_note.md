# 小怡《情绪吸引力-人性篇》：018.18-情绪沟通-联系感

## 概述
在本节课中，我们将学习情绪沟通法中的第四个情绪——联系感。联系感是建立深度关系的关键，它不同于简单的联系，而是基于情感纽带的深层沟通。

## 为什么需要联系感
**问题**：为什么有了吸引后，不能立即进行邀约，而是要进入联系感的阶段呢？

**解答**：
1. **吸引与邀约的关系**：吸引能让对方靠近，但不足以保证成功邀约。
2. **价值展示的误区**：展示价值不等于对方会感受到价值，可能只是单方面的炫耀。
3. **建立联系的重要性**：让对方感受到有机会得到你的价值，增强付出动力。

## 联系感的本质
1. **联系感与普通联系的区别**：常联系不等于有联系感，而是精神层面的沟通。
2. **联系感的建立**：通过情感纽带，在内心留下深刻的情绪体验。

## 进入联系感的方法
1. **欣赏**：表达对对方的欣赏，让对方感受到认同和自我实现。
2. **真实**：展示真实的一面，建立信任和情感纽带。
3. **分享脆弱**：分享柔软脆弱的一面，建立更深层次的信任和联系。
4. **共谋**：建立只有双方才懂得的小秘密，增强联系感。

## 总结
本节课中我们一起学习了联系感的概念和建立方法。通过欣赏、真实、分享脆弱和共谋，我们可以与对方建立更深层次的联系，为后续的沟通和关系发展打下坚实的基础。
