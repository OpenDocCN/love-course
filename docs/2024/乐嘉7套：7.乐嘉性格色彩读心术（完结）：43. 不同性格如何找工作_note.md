# 乐嘉性格色彩课程（七）：第43课 不同性格如何找工作 🎯

在本节课中，我们将学习如何根据四种核心性格色彩——红、蓝、黄、绿——来选择最适合自己的职业。我们将逐一分析每种性格的潜能与局限，并探讨其对应的理想工作领域。

## 概述：性格与职业的匹配

每个人性格不同。了解自己的强项与短板，找到能发挥强项、规避短板的工作，最适合你的工作就能让你的人生价值最大化。所以，今天要聊的就是每一种颜色的性格，大学毕业以后最适合的职业是什么。

如果你已经参加工作，也可以通过本课内容分析现在的工作是否符合你的性格特点。如果你是老板或领导，也可以借此分析下属员工的性格，以及他们现在的岗位是否能最大限度地发挥其特长。

## 红色性格：追求快乐与认可 🌟

上一节我们概述了性格与职业匹配的重要性，本节中我们来看看红色性格的特点。红色性格一生中最重要的追求就是快乐。他们渴望每天收到他人的关注和赞美。他们富有表现力，愿意和大家分享自己的心情。

这就是为什么演艺界人士大多为红色性格。若将一个特别喜欢互动的红色性格安排在无人搭理、天天伏案工作的岗位，两三天尚可，时间一长他就会崩溃。但如果工作能让红色性格获得足够多的互动和他人认可，红色就会特别开心。

以下是适合红色性格的职业举例：
*   **婚礼创意师**：适合有创意的红色性格。把别人的婚礼场面设计得特别美，获得新人及家人的赞美，会激励他们做出更美的场景。
*   **销售**：很多红色性格喜欢销售，因为他们喜欢表达，可以无拘谨地与不同客户交流。
*   **作家、导演、编剧**：他们能用天马行空的想象力，带领读者或观众进入全新世界。

此前热播剧《人民的名义》中，饰演陈岩石的白志迪老先生演技精湛，经常一条就过，观众评价很高。他就是红色性格，虽年长却常与剧组年轻人打成一片。他曾表示，演戏上瘾，能把角色演出花来，别人说演得好，不给钱也干。对他而言，幸福就是一辈子干自己喜欢的事。

我早前节目说过，红色性格的人喜欢跳槽。那是因为他们发现现在干的事没意思、不开心，或容易受他人言论影响（如“这工作没出息”、“你看别人混得多好”），就可能撂挑子不干。那么，红色性格真的无法坚持吗？**不**。只要工作让红色感到快乐并充满意义，他就会坚持。

我本人在发现性格色彩之前也换过十多份工作。但传播性格色彩、教别人演讲，在帮助他人改变的同时，我自己也得到成长和巨大满足感。看到人们的变化，我觉得自己有价值、很开心，在做有意义的事，所以坚持至今。

在《超级演说家》第三季，我做过一篇题为《成功，只需要一件事》的演讲，核心就是“热爱”二字。这两个字对红色性格特别重要。所以，红色性格的朋友不要因为觉得自己不够细心、没耐心就否定自己。如果你能找到一份让自己感到无比快乐和开心的工作，没人督促，你依旧会努力拼命。时间长了，成绩自然显现。

## 蓝色性格：内敛严谨与追求完美 🔍

相比较刚才提到的外向热情的红色性格，蓝色性格就内敛低调了很多。他们不太愿意受到别人关注。只要能有一件事让自己安安静静做研究，就再好不过。他们从小就有钻研精神，非常享受经过几个小时解出一道数学题的快感。

他们最大的特点是细心、做事一丝不苟、追求完美。所以他们最适合的职业，你自己也可以想出来——那些特别追求细节、不允许出错的工作。

以下是适合蓝色性格的职业举例：
*   **建筑设计、财务、法务、工程师**：这些领域需要严谨的计划、程序和研究。
*   **医生**：在这些行业里，一旦出错就可能人命关天，或蒙受巨大损失，因此蓝色性格来做就很棒。

纪录片《我在故宫修文物》中，有人为修复一个部件就待上一整天。为了让颜色与原有颜色一模一样，需用大量颜料反复调配，这需要极大的耐心。而足够强大的耐心，是蓝色性格天生具备、无需训练的特点。

电影《十二公民》讲述了一个案件：父亲被杀，所有证据都指向其子。12名陪审员中，11名红色或黄色性格者认为证据确凿，应判有罪。但一名蓝色性格者认为“总觉得哪儿不对劲”，要求重新梳理案件。根据规则，一人不同意则需继续讨论。经过一整夜辩论，更多证据浮出水面，最终说服其他11人，无法证明孩子是凶手，避免了冤案。影片最后揭示，这位蓝色性格者的真实身份是一名法院的律师。

由此可见，蓝色性格可以及时发现工作中出现的问题。如果你不具备蓝色性格天性中的细心、严谨和耐心，就可能在工作中频繁犯错，导致重大损失甚至悲剧。

## 黄色性格：目标坚定与勇往直前 ⚡

黄色性格雷厉风行，敢想敢干。他们从小就想当带头老大。如果工作能让他们冲锋陷阵、力争上游、勇夺第一，那便是其天性中的乐趣。许多运动员具备黄色性格特质，就是因为他们追求更快、更高、更强。

在许多政坛和商界人物中，也有非常多黄色性格，因为他们目标坚定，抗压能力强。对黄色性格来讲，**要么不做，要做就做到第一**。

以马云为例。从专业上讲，马云应属典型的红+黄性格。但为了在本课中让大家易懂，我暂时将其假定为黄色性格。他两次高考失败，曾从事秘书、搬运、蹬三轮等工作。20岁考入普通大学，毕业后想做互联网，召集24位朋友商议，23人反对，仅一人说可试试。马云想了一夜，决定“所有人都反对，我也要干”。到32岁时，他仍寂寂无名，在1996年互联网尚未普及时，甚至被人称为“骗子”。直至2000年，阿里巴巴横空出世。

这便回到了我开篇提到的问题：你到底适不适合创业？我的回答是：如果你性格中有黄色性格的优势，创业这条路相当不错。因为黄色性格若想做大事，他们需克服的最大麻烦是容易逃避困难。而蓝色性格则过于犹豫纠结，不太敢冲敢闯。但若你性格中有很多黄色，天性中就足够勇往直前。

我说这段话不代表性格中没有黄色的人就不能成功。世界之大，凡事皆有可能，且不同性格的人创业途径也不同。但我想讲，黄色性格在天性中的特点，使其在取得“名、权、利”的事业成功上，概率比其他性格要高。

## 绿色性格：平和包容与耐心服务 🍃

最后一种绿色性格无欲无求，对工资要求不高，只要饿不死就行。绿色性格也没什么抛头露面的欲望，觉得一辈子过过小日子就够了。

那绿色性格到底适合干什么工作呢？我觉得服务型行业对绿色不错，因为他们善于耐心倾听、包容他人，且天性具备缓和矛盾的特点。

美国短片《缓慢的莱纳》（仅9分钟）讲述了一个角色，其说话做事速度比普通人慢一半以上。结果他工作无法按时完成，表白被抢先，客户也被抢走，事事不顺，最终因公司要求高效而被开除。像莱纳这种慢腾腾的绿色性格，似乎没人需要。

但有趣的是，片末他找到一份新工作——在敬老院做护工。那里全是行动缓慢的老人，舀一勺饭送入口中需几十秒，从床上到客厅也要十分钟。其他性格的人做几天就跑了，觉得既无耐心又无成就感。但莱纳作为职业看护，其行动速度完美契合了老人的节奏。他由此重获自信，工作非常快乐。后来因表现极其出色，被升为全院主管，带领团队并分享经验。

由此可见，我们每个人的行为模式不一样。但如果你找到足够适合的事情，就可以很快乐地做，并体现价值。

以下是适合绿色性格的职业举例：
*   **客服、幼教老师、护士、空乘**等。
在许多需要有耐心、流程相对简单不复杂的职业中，你会发现，对社会而言，这部分职业对绿色性格来说，简直是老天爷专门为他们设计的，能完全发挥其特长。

## 总结：找到属于你的位置

我今天谈找工作这个话题。如果你在社会上历练已久，会发现有些人挣了很多钱，工作也特别高大上，但就是不快乐。人和人差别如此之大，一个重要原因在于：你是否了解自己的性格，能否找到一个可以相守一生、适合自己性格特点的工作。

希望今天所讲的话题，能给你一点小小的启发。祝你早日找到符合自己性格特长的工作，开启顺风顺水的人生。
