# 18、坏小k女人一生必学的一堂课：06.情感框架训练 （3）在亲密关系中赢得尊重

## 概述
在本节课中，我们将学习如何在亲密关系中运用情感框架训练，以赢得尊重并维护自己的权益。

## 训练框架思维的重要性
**公式**： 框架思维 = 自我认知 + 目标设定 + 行动策略

### 为什么需要训练框架思维
在现实生活中，许多女性喜欢解释自己，但往往没有足够的身份去解释。她们试图说服他人，尤其是在情感上，试图说服男性喜欢自己。然而，这种本质上是一种以下对上的框架，男性沟通，即使你说得再累、再有道理，做的再好，男性也不会买单。

### 框架思维与个人魅力
一个有框架、有态度、有标准、有规则的人，我们会联想到雄心领袖、女王、家庭化制人、团队领导等词语。这些人我们内心愿意去接近，因为他们的生存价值很高，潜意识里我们认为这些人对我们有利。

### 框架思维与人际关系
一个没有框架、没有态度、没有标准、没有底线原则的人，我们会联想到努力、丫鬟、穷人、卑贱者等词语。这些人我们内心不愿意去接近。

## 框架思维在亲密关系中的应用
### 例子1：新朋友交往
当我们认识一个新朋友时，为了留住这个所谓的人脉，人们会想方设法表现自己，为他做点什么。然而，很少有人思考如何在交往中留下好印象，让对方觉得我们是一个怎样的人。

### 例子2：追求者抱怨
当一个追求者不停地抱怨自己如何如何不好，抱怨自己配不上你时，有框架思维的人会告诉你，这是树立框架的时候。

### 例子3：恋爱关系中的底线
在恋爱关系中，如果对方总是说“没事儿，我不介意”，那么你的底线就在这一句“没事儿”中被瓦解。

## 框架思维在生活中的应用
### 例子1：同事帮忙
当同事找你帮忙带饭时，你可以提出交换原则，即先让对方接受自己的要求，然后再去完成对方的要求。

### 例子2：原生家庭
在原生家庭中，如果父母总是阻碍你成长，那么你可以尝试舍弃原则，即舍弃与他们的沟通。

## 总结
本节课中，我们一起学习了如何在亲密关系中运用情感框架训练，以赢得尊重并维护自己的权益。希望你在今后的生活中，能够运用所学，成为一个有框架、有态度、有标准、有规则的人。
