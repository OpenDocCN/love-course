# 老白《搞钱课》：7、动画视频制作完整流程 🎬

## 概述
在本节课中，我们将学习如何制作动画视频，包括文案准备、配音、音频处理、字幕添加、动画特效以及图片插入等步骤。

## 文案与配音
1. 准备文案。
2. 使用配音软件为文案配音。
3. 分离音频，删除不必要的滤布。
4. 添加黑幕或白幕，根据背景选择。

## 音频处理
1. 调整音频音量，一般设置为15。
2. 试听音频，检查是否有卡顿。
3. 根据需要调整音频倍速。
4. 选中音频，调整间隙，确保流畅。

## 字幕添加
1. 点击文本，使用智能字幕功能。
2. 识别并调整字幕位置，确保舒适阅读。
3. 修改中文断句，确保读起来自然。
4. 选择合适的字体和颜色。

## 字幕动画
1. 添加英文字母作为水印。
2. 设置字体、大小、位置和颜色。
3. 添加动画特效，如打字机效果。
4. 设置入场和出场动画，如向左滑动、向左擦除等。

## 图片插入
1. 选择合适的图片，如与文案相关的图片。
2. 下载图片，并将其插入视频中。
3. 调整图片位置、大小和比例。
4. 添加入场和出场动画。

## 效果调整
1. 确保入场和出场动画时长一致。
2. 选择合适的特效，避免花里胡哨。
3. 调整字体大小和颜色，突出重点。
4. 添加背景音乐和配音。

## 总结
本节课中，我们一起学习了如何制作动画视频，包括文案准备、配音、音频处理、字幕添加、动画特效以及图片插入等步骤。通过学习这些步骤，你可以制作出吸引人的动画视频，提高你的播放量。希望你在实践中不断摸索，早日成为百万播放博主。
