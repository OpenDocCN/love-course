# 23、老公出轨心理27堂课：05 挑明男人出轨后，为什么也

在本节课中，我们将要学习男人出轨后，为何多数情况下并不选择与原配离婚。我们将通过案例分析，深入探讨其背后的心理需求、出轨驱动力，以及男人在离婚与不离婚之间的权衡逻辑。

## 概述

我叫朱生勇，是一名婚姻管理师，也是婚姻管理学派的创始人。我从事婚姻管理工作已有十年，辅导了上万个家庭修复夫妻关系、重建婚姻。我的工作包括协助挽救婚姻、治理婚外情、进行离婚辅导以及复婚辅导，同时也帮助大量第三者从婚外情的陷阱中走出来。

## 案例分析：漏洞百出的出轨事件

上一节我们介绍了课程的整体框架，本节中我们来看看一个具体的案例，以理解男人出轨后的典型心理。

我曾辅导过一个案例。一位女士的丈夫出轨，在78年间与4个女人有染。妻子直到第四个第三者打电话来才发现。第三者说：“你是某某的妻子吗？我告诉你，我是你老公的小三。你不是一直不肯离婚，还说离婚就去死吗？你怎么不去死？”妻子感到莫名其妙，因为她从未提过离婚。于是她去质问丈夫，丈夫否认，并称那个女人疯了，是因为借钱未遂而污蔑他。

妻子半信半疑，又联系了第三者。第三者让她添加联系方式，并发送了照片和视频作为证据。证据确凿，妻子再次找到丈夫。丈夫辩称是醉酒后的一次意外，并强调自己从未想过离婚，多年来对家庭尽心尽力，是对方在胁迫他。

妻子又致电第三者求证，第三者却说：“根本不是这样！明明是他主动追求我。而且我告诉你，他前面还有三个女人。”妻子彻底崩溃。第三者解释自己36岁跟了他，现在38岁，被家人逼婚，而丈夫谎称自己已离婚，家人才没嫌弃。第三者因受不了这种欺骗和压力，才选择联系原配。

这个案例中，丈夫的谎言漏洞百出。在第三者那里，所有的漏洞都可能被揭穿。最终，通过第三者的求证，真相浮出水面。从这个案例可以看出，男人出轨后往往编织复杂的谎言来维持现状。

## 是真爱吗？评估男人出轨的性质

像今天辅导的一位客户问我：“朱老师，你说我老公和这个小三是真爱吗？他们都在一起三四年了。”如何判断是否为真爱？我们可以通过评估标准来分析。

如果两个人恋爱三四年考虑结婚，通常会涉及见家长、买房、规划共同未来等实际行动。如果这些通通没有，怎么能称为真爱呢？

所以，这种男人出轨又不愿离婚的情况，往往是：**真情、真性、假爱**。
*   **真情**：真的激情、情调和感情。
*   **真性**：真实的性关系。
*   **假爱**：并非真正的、有承诺和未来的爱。

一个男人是否真的爱一个女人，有六个评估标准。我们需要审视男人出轨究竟是为了什么。

## 男人出轨的核心驱动力

以下是男人出轨常见的几种心理需求，它们构成了出轨的核心驱动力。

### 1. 心理需求
纯粹为了玩弄女人的比例很低，大约在5%左右，不会超过10%。第一个层面是心理需求。他需要肯定、认可、赞同、关注、陪伴、归属感以及爱与被爱。很多人在婚姻中无法满足对方的这些需求。

**哪些阶段最容易出问题？**
婚姻经营不好的一个核心原因是“有了孩子，忘了郎”。很多女性从怀孕开始，就将80%甚至90%的精力放在孩子身上。这就是为什么我们说孩子有时会成为婚姻的一大杀手。妻子全心带孩子，但第三者不会。第三者可以把孩子托付给别人，去和男人约会、旅游。如果丈夫讨厌孩子吵闹，让妻子别带孩子，妻子往往做不到，但第三者可能做到。这是很残酷的现实。

### 2. 情感需求
所谓情感需求，常常源于夫妻间的情感疲劳或亲密关系出现间隙。此时，男人可能到外面寻求与异性谈情说爱、花前月下的浪漫，进行一些夫妻间早已不再进行的活动，从而获得情感上的满足。

### 3. 生理需求
生理需求包含好几个方面：
*   **夫妻运动**：即性生活。
*   **口腹之欲**：一起做饭、享受美食。
*   **皮肤接触的满足**：抚摸等肢体接触的渴望，这一点很多人会忽略。

### 4. 价值需求
很多男人通过照顾第三者、帮助对方解决困难来体现自身价值。例如，第三者来找他做项目、拿工程，他会觉得“我这个男人有价值”。另外，当男人事业进入低谷或经济有压力时，也可能通过婚外情寻求价值肯定，因为他可能觉得妻子瞧不起他。很多老板在事业低谷时，喜欢找娱乐场所的女性，就是这个原因。

### 5. 物质需求
这种情况相对较少。指男人自身经济能力一般，通过有钱的女性（富婆）来获得物质满足，比如名牌包包、衣服、鞋子等。他以前可能也有这些渴望但买不起，第三者的给予让他很满足。

### 6. 个体发展需求
这也是很多人出轨的一种需要。个体发展需求是指：
*   觉得自己人生遇到瓶颈，需要别人来激发“第二春”。
*   个体发展需要仰仗某人。
*   觉得自己的个体发展被妻子束缚了，从而想去追求。

### 7. 繁衍需求
这个很容易理解，就是想要孩子，特别是想要儿子，或者想要“更优秀”的后代。

以上是男人出轨的主要驱动力。在《婚情之痛》这本书里，我总结了大约16到17条，大家可以参考。

## 男人出轨的具体驱动力分析

了解了出轨的需求，我们再来看看哪些具体情境会成为男人出轨的驱动力。

### 1. 夫妻关系紧张或疏远
夫妻关系的好坏、是否亲密、是否放松，决定了丈夫出轨的可能性。大家一定要注意这一点。

### 2. 家庭矛盾解决不了
家庭矛盾包括：
*   **夫妻矛盾**：各种小摩擦，如饮食习惯、作息习惯、卫生习惯、吵架等。深层次看，这可能是自我认同出了问题。也包括常说的“三观不合”。
*   **婆媳矛盾** 或 **翁婿矛盾**：与对方家庭的矛盾。
*   **亲子矛盾**：因孩子教育问题产生的矛盾。
*   **经济矛盾**：家庭经济出现紧张，也常常导致婚姻出轨。

### 3. 夫妻运动不和谐
这是很多男人出轨的强劲驱动力。不和谐的原因可能包括：
*   双方疲劳。
*   一方过于保守，不愿尝试新姿势、新体位。
*   两人生物钟不匹配。
*   时间长短、大小、身体气味等也会影响。

夫妻之间在这件事上要敞开来、放开来、解放。这应被视为夫妻间的一种修行，是“打通任督二脉”的课程。

### 4. 个体发展的压力
包括事业压力、企业压力、人到中年的心理危机，以及一些五六十岁男人面临的“个人存在危机”。有一种现象叫“59现象”，原指公职人员，但普通人也会遇到。即临近退休或感觉个人发展可能中断时产生的焦虑和压力。即使事业发展很好，也会有压力。

### 5. 圈子的风气
如果你丈夫所处的圈子中，很多人都有出轨行为，那么他十有八九早晚也会出轨。人是社会性动物，有羊群效应，容易跟风。

### 6. 事故
指家庭发生重大变故，例如家人生重病或去世、朋友同学出事等。这类事件会让人产生巨大的心理波动，可能思考人生意义，从而放纵自己；或者因为妻子无法给予有效安慰，而转向外界寻求慰藉。家里如果有重大事件，请好好陪伴你的丈夫。

### 7. 阶段性发展失调
指夫妻双方成长不同步，聊天聊不到一个平台。例如，丈夫谈事业，妻子听不懂或没兴趣。这也会成为强大的出轨驱动力。

除了这7点，还有其他驱动力，我们无法一一展开。以上是日常生活中较常见的情况。

## 男人在什么情况下会考虑离婚？

我们进入男人离婚这个话题。男人离婚分两种情况：一种是在搞婚外情之前就考虑离婚；另一种是因为婚外情才想离婚。

### 婚外情之前就考虑离婚的情况
为什么想了却没离？可能有各种盘算。他会考虑父母、子女、财产，但常常不是主要考虑财产。他可能对妻子有一种悲悯之心，觉得妻子多年付出，没有特别大的过错。

**什么情况下，他会在婚外情之前就动离婚念头甚至执行？**
以下是几个关键点：

1.  **对妻子绝望**：觉得妻子无药可救、无法沟通、再不改变也无用。认为妻子在婆媳关系、个人发展、教育子女或房事等方面完全不称职，看不到妻子的未来。
2.  **对婚姻无望**：觉得两人关系无法改变，自己已尽最大努力仍无法推动。觉得婚姻好不了，看不到头，看不到未来。
3.  **对自己渴望**：觉得自己的人生不能这样过，渴望过得幸福、成功、自在、人前风光。他对自己并未绝望，而是充满渴望。
4.  **对财产和子女有妥善安排**：这里指的是正当安排，而非恶意转移财产。恶意转移财产涉及人品问题，不在我们的一般考量范围内。
5.  **对过去有客观（或自认为公正）的看法**：能看到妻子、自己和婚姻的问题所在，对过去持否定态度。对现实没有留恋，觉得没法过、不想过。对未来有新的展望。

### 因婚外情而想离婚的情况
我们要着重解决的是这一部分人。如果是在搞婚外情之前就想离婚的，治理起来超级难。

1.  **性格冲动或偏执**：人格上有冲动或偏执、边缘型特点。一旦产生念头，不管后果就要去做。
2.  **婚外情对婚姻产生二次创伤**：例如，与第三者有了私生子、分居、搬出去住、分房、冷战、不给钱等。这些行为会对婚姻造成进一步伤害。
3.  **与第三者关系和谐且有共同成就**：两人有很多火花，能共同解决矛盾，一起做成事情，特别是**一起创业**且事业有起色。但专注于婚外情的人，往往很难真正搞好事业，因为精力分散了。要让婚外情“成功”，往往需要“干掉”家庭、妻子和婚姻，而不仅仅是专注于婚外情本身。
4.  **理性分析后认为第三者优于妻子**：不是激情下的“灵魂伴侣”、“真爱”结论，而是像当初娶妻一样，进行理性分析：欣赏第三者的哪些特点、能力，了解其缺点和原生家庭等。同时也会理性分析妻子的优缺点。当理性分析得出第三者具有决定性优势时，就可能选择离婚。
5.  **婚姻本身确实糟糕且持续恶化**：婚外情发生前，婚姻体验就很差；或者婚外情暴露后，妻子采取极端行为（如一哭二闹三上吊、离家出走、刷光家产等），导致婚姻状况急剧恶化。**发现丈夫出轨后，千万不要轻易离家出走或采取极端财务行动，这无异于“作死”。**
6.  **婚外情后认知变化，认为“妻子不是自己想要的人”**：有些男人通过婚外情变得成熟，发现妻子不是自己想要的。特别是当第三者对他有“创伤疗愈”作用，而妻子没有时，这非常麻烦。
7.  **对未来过度乐观（高估）**：觉得离婚后跟第三者在一起，未来一切都会更好（经济、婆媳关系、个人发展等）。如果对未来预估不那么乐观，就不容易离婚。
8.  **对现状的差评**：搞婚外情后，对妻子和婚姻各种挑三拣四。一部分是故意找茬，一部分是真实感受，需要注意区分。

## 男人为什么不离婚？

男人不离婚，心里也有一杆秤。这杆秤其实就是挽救婚姻的5个标准（满意度、婚姻发展、矛盾评估、感情状况、离婚后果）。他会进行以下评估：

1.  **评估婚姻满意度**：他会想婚姻里哪些地方是满意的，哪些不满意。也会评估妻子为他做了什么，没做什么。
2.  **评估婚姻整体状况**：结婚后过得怎么样？婚姻发展如何？自己被照顾得如何？身体、财富、子女、人际关系怎么样？
3.  **评估婚姻矛盾**：婚姻里到底有哪些矛盾？他会想“如果妻子改了某个地方，我就愿意过”，如果一直不改，就不想好好过。
4.  **评估感情状况**：他和妻子的感情到底好还是不好？
5.  **评估离婚后果**：离了婚我会变成什么样子？财产损失多少？找的第三者怎么样？一对比，可能发现第三者学历不如妻子、不是良家妇女等。还会考虑离婚带来的社会关系、家庭关系的严重损失。

很多男人对婚姻的看法是很现实的。千万不要以为只有女人现实。实际上，女人对婚姻可能更幻想、更魔幻，而男人很现实。他现实地考虑：和你在一起会过成什么样子？你能“伺候”成什么样子？离了婚又会过成什么样子？这些他都在心里盘算好了，尽管可能没有条理清晰地梳理出来，但潜意识里一直在运行。这就是很多男人不离婚的原因。

## 总结

本节课中，我们一起学习了男人出轨后为何多数不愿离婚。我们通过案例分析了出轨的谎言与漏洞，探讨了出轨行为背后的心理、情感、生理、价值、物质、个体发展及繁衍等多重需求。我们详细剖析了促使男人出轨的具体驱动力，包括关系紧张、家庭矛盾、性生活不和谐、个人压力、圈子风气、重大变故及成长不同步等。同时，我们深入区分了男人在婚外情前后考虑离婚的不同心理动机，并解释了男人在权衡利弊后选择不离婚的深层评估逻辑。理解这些，有助于我们更客观地看待婚姻危机，并为后续的应对和修复提供思考基础。
