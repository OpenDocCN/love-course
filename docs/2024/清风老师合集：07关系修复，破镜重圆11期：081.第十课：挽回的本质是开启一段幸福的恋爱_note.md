# 清风老师合集：07关系修复，破镜重圆11期：081.第十课：挽回的本质是开启一段幸福的恋爱_note

在本节课中，我们将深入探讨挽回的本质，帮助你审视自己的挽回动机，并学会区分健康的挽回与盲目的执念。我们将分析导致“爱的执念”的根源，并提供清晰的思考框架，引导你做出更理性、更有利于未来幸福的选择。

上一节课程我们讨论了挽回成功后该如何维护关系。本节课程，我们将进行一次深入的对话。

或许在听完一系列课程后，你现在对于挽回有些举棋不定。你不知道自己的挽回究竟是否正确、是否有意义。你甚至在想，如果自己真的能把心态调整好，这个人好像也不太值得。如果你正在怀疑这些问题，请你冷静下来思考：你究竟因为什么才开始挽回？

错误的观点认为，爱是一种执念，只要我还爱着这个人，我就一定要和他重新在一起。那么，你为什么会有这种执念呢？如果你被这个问题问住了，请你认真思考一下。

产生这种对爱的执念，往往有三个原因。

以下是三个主要原因的详细分析：

1.  **童年的伤痛**：在你的童年经历中，感受到父母的亲密关系不和谐，婚姻关系不健康。于是在成长过程中，你非常害怕在情感关系中像自己的父母一样，成为别人眼中不被看好、不被认可的关系。所以当面临分手时，你急于想要证明“我是对的，我没有错，我没有像我父母一样”。但殊不知，你越是想要去证明，就越是变得和他们一样，在感情里毫无安全感，不断控制和绑架你的爱人。

2.  **自我的缺失**：因为你特别渴望从对方身上获得某种满足。可能是他有着英俊的外表，而你对自己的长相不够自信；也可能是他拥有高额财富，而你对自己的收入不太满意。当你不断试图通过对方来满足自身的缺失时，你会得到一个不好的结果：原先互相成长、互相认可的爱，变成了紧紧抓住对方、让对方窒息的绑架。

3.  **对未来的不自信**：觉得自己的人生已经毁了，没有任何机会了。认为和这个人分开以后，自己的人生没有任何其他的可能性。

如果你因这三个原因产生了爱的执念，我恳请你停下来，不要着急。挽回是一种选择，我们要做的是找到一个对的人和一份好的爱，而不是简单地与对方复合。如果对方是一个长期酗酒、对生活和未来没有任何规划能力的人，即使你和他挽回，在两人的亲密关系中，你仍会时常处于颠沛流离、担惊受怕的状态。

我的一位学员，她因丈夫出轨而分开。她问我：“老师，我该不该去挽回他呢？挽回他是不是显得有点贱？”我问她：“你真的是因为自己想要去挽回他吗？还是因为孩子、因为家庭、因为各种各样的压力想要去挽回他？”她回答：“老师，我不得不承认，我已经不爱他了，我对他没有爱了。但是为了孩子，为了家庭，甚至为了别人眼中对我的看法，我要去挽回他。”这时我告诉她：“姑娘，你很优秀，你很值得被爱。不要在一个不爱你的男人身上不停花功夫，让他证明他是爱你的。那既让他痛苦，也让你自己痛苦。”

在你挽回之前，请你问问自己：**这个人真的是一个好的对象吗？** 如果是，请你大胆勇敢地往前走，我教你的方法都可以帮助你。如果不是，请你冷静一下，不要因为爱的执念，让你的生活陷入水深火热之中。

接下来，我们再思考一个问题。如果在挽回过程中，出现了更优秀的异性，你也很喜欢对方，但似乎还不能完全放下前任，这时该如何选择呢？不用着急回答，我想你心中一定非常纠结，一方面放不下过去，另一方面也不能很好地迎接新欢。这正是我这堂课的目的。

我恳请你给你的未来一些宽容，给你的未来一些新的可能性。特别是如果你社交圈子很小，又不爱社交，当有新机会出现时，你可以先进行一些尝试。不用急着和对方牵手拥抱，至少可以先给对方一个机会约会体验一下。

体验之后，请你先别拿“爱不爱对方”作为唯一的衡量标准。盲目考虑“爱”会把你带入爱的执念和漩涡中。因为**爱是所有体验的最终结论，而不是一个简单可衡量的标准**。你和前任经历了那么多，付出了那么多，最终得出结论说你很爱他，这没有问题。但和这个新出现的人，你们还没有经历那么多，请你千万不要急于下定论说“我对他还没有感觉”。

我理解我们都渴望那种从身体到心灵完全心意相通的爱，渴望得到对方的关注、关心和全身心的爱，也渴望全身心地爱对方。但人生不总是完美的，并不是每时每刻都能得到我们想要的。当你遇到不够完美的问题时，请你多一些耐心。

**把条件放到前面，而不是简单地把爱放到前面。** 爱很重要，但爱不能作为唯一的考量标准。或许你会觉得，我说要看对方的条件、工作、身高、长相，这很功利。是的，爱是要基于功利的基础上的。如果你只是一门心思去纯粹地爱一个人，最后有可能爱的不是一个人，而是你的幻想，是你对于爱情这种美好的追求和幻想，而不是眼前这个真实的、有七情六欲、有缺点的男人。

我的一位学员，是知名企业家的女儿，经历很像三星集团的长公主。第一段婚姻非常不幸，离婚后她带着孩子生活。前任对她很不好，分开的原因是前任好赌酗酒。大家都不明白，为什么如此优雅美丽、出身高贵的女人，会爱上这么一个看上去像登徒浪子的男人。她和我说：“老师，我实在放不下他。我从小到大经历了那么多段别人看上去很棒的感情，只有我知道那些男人爱慕的不是我，是我的家庭。只有他对我是纯粹的。虽然他脾气很不好，但他是真的爱我的。”听到这样的话，我真的很心痛。这个姑娘被自己对于爱情的理解洗脑了。

**爱不是毒药，也不是让你的生活毁掉的可怕灾难。爱是美好的。** 如果你发现和一个人在一起好像相爱，但互相折磨，请你别骗自己了，那就是在互相折磨。这时冷静一下，重新选择或许并不是一件糟糕的事。它能让你看到这个世界有更多生命的可能性，让你看到未来有更多可能性。

姑娘们，请你们在挽回的道路上，不要失去理性，不要失去对于爱的追求，也不要失去对于未来的期待。因为在追求爱的过程中，你有可能遇到更美好的风景。
