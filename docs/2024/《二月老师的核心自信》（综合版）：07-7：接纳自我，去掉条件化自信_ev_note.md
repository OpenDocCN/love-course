# 《二月老师的核心自信》（综合版）：07-7：接纳自我，去掉条件化自信_ev_note

在本节课中，我们将要学习如何通过**接纳自我**来摆脱**条件化自信**的束缚。我们将探讨为何人们难以接纳先天特质，以及如何区分社会标准与个人标准，从而建立起不依赖于外界评价的内在自信。

## 难以接纳的先天特质

上一节我们介绍了自信的不同层面，本节中我们来看看阻碍真正自信的第一个障碍：对先天特质的不接纳。

除了生命本身，每个人的身高、长相也都是无法选择的先天特质。很多人难以真正接纳这些部分。

以下是常见的自我质疑：
*   为什么我这么矮？
*   为什么我的五官长得不够立体？
*   为什么我不够帅或不够美？

然而，外界的成功与审美标准是由社会制定的。那些标准并非属于你个人，并且会随着时代变迁而不断变化。

## 重新定义“成功”标准

上一部分我们看到了对外貌的不接纳，类似的困扰也存在于对“成功”的定义上。

社会常将成功与金钱、地位划等号。但拥有快乐富足的人生、乐于助人并帮助他人，这些难道就不是成功吗？本节的目的并非重新定义成功，而是强调：你应该找到自己真正有热情的事物，完成它对你而言就是成功。即便没有完成，尽力尝试本身也是一种成功。

你的成功不应由外界制定。你不应活在外界的标准里，而应活在自己的标准里。必须想清楚这一点，否则你的人生就将被外界控制。

## 打破多变的审美桎梏

理解了成功的个人化定义后，我们再来审视同样多变的审美标准。

审美标准也是如此。以前流行大眼睛，现在可能欣赏小眼睛；过去以丰腴为美，现在则以瘦为美。在20世纪以前，全球男性平均身高可能低于1.7米，那时男性基本没有身高困扰。为什么现在男性低于1.7米就可能感到不自信？

你为什么要活在别人定义的生活里？别人的标准就是真理吗？当你坚持认为自己的身高不是问题，那么它对你而言就不再是问题。

说一句题外话，如果没有人指出你的身高是问题，你可能根本察觉不到这个问题。但如果有三个人、十个人、成千上万个人都说你矮，说你身高有问题，你可能就会崩溃。

但我希望你能做到，哪怕整个世界都在嫌弃你的身高，你依然能够爱你自己。你不需要去对抗这个世界，但在接纳自我这件事上，你必须拥有面对世界的勇气。

## 从接纳自我到爱自己

面对外界的标准，真正的力量源于内在的自我接纳与爱。

很多人都说我眼睛小，建议我去割双眼皮，但我至今没有这个想法。因为我很清楚，如果我自己都不能接纳自己、爱自己，那么我永远都找不到真正的自信。

所以，这里多说一句题外话。对于那些喜欢整容的人，其背后往往是对自我的不接纳。那是一个无底深渊：今天割了双眼皮，明天可能就想垫下巴，后天又想削骨。颜值及审美是时代的产物，你不应陷入无止境的自我摧毁中。

而且前面也说过，广告不断宣传所谓的完美形象，是想让你陷入自卑，从而去追逐他们定义的完美，因为他们需要从中赚取利润。然而，他们所说的就是真理吗？

所以，哪怕你的外表不符合现代大众审美，你也应该学会欣赏自己的独特，学会爱自己。

能理解吧？因此，对于我个人来说，我不推荐整形。但是我推荐微调。例如，龅牙、满脸痘痘、秃头这些问题，进行调整是OK的。这就像我推荐你学会护肤一样，不属于自我不接纳，而属于自爱。但只要涉及严重的动刀手术，我一概不推荐，我认为那才是对自我的不接纳。

## 条件化自信的陷阱

在不接纳自我的基础上，很多人衍生出嫌弃自己情绪的问题。这就是造成你自卑的根本原因。

很多人的内心有一个强大的错误假设：**只有各方面优秀的人才有资格自信**。

对于那些现实中不够优秀但却很自信的人，有自卑情节的人可能会感到愤怒：“为什么他那么差，什么都不是，却还自我感觉那么良好？”

问题的原因恰恰出在这个愤怒之处。那个自我感觉良好的人，他的良好感觉不太受外在条件限制。它更多的是一种弥漫性的个体状态。外在条件的好坏、成就的多寡，很难威胁到他内心的核心自我。这就是剥离了外界的一切条件。

你是不是觉得这种现象很不正常、不合理？而你之所以认为这不合理，正是因为你内心有那个强大的假设。这个假设是：**人只有表现良好、取得成就、各方面优秀，才有资格自我感觉良好，才有资格自信**。

很多人都是这样想的。然而，真正决定你自信的是你的**核心自我**。

## 自信的本质在于自我接纳

当一个人把自我感觉良好的状态**条件化**时，他就永远无法做到真正的自信。因为自信的本质在于**自我接纳**。

自卑的人之所以自卑，是因为他们内心无法做到完全地接纳自己。能理解吗？

因为这种不能自我接纳，会导致他们认为自信是需要条件的。这些条件可能表现为：必须达到某些标准，或必须在某些方面超过同龄人，他们不允许自己在某些方面不行等等。

有自卑情节的人在取得一定成就后，短时间内会表现出自信，有时甚至是无与伦比的自信。但这个自信不能持久。过一段时间后，他又会返回自卑状态，或者他需要持续保持成就来维持自信。

这样他就会很累，做不到真正的相信自己。很累的根源，就在于对“不优秀”或“没有成就”的防御，也就是对于那个不够优秀、缺乏成就的**自己**，不能做到完全接纳。

尽管这个不够优秀、缺乏成就的自己，通过努力可能已经不存在了，但你对他的**担心**依然存在。如果不能理解，请写下来多看几遍。

只要这个担心存在，一个人就需要付出更多能量去抗拒这个担心，那么他就不会获得真正的自信。一个对自己“不够优秀”、“缺乏成就”存在防御的人，怎么可能有真正的自信呢？对吧？防御就说明他对自己充满怀疑。

即便他外在的条件改变了，但他内心那种糟糕的**核心自我假设**并没有随之改变。如果难以理解，请写下来反复看几遍。

## 建立无条件的自信

而且，我们前面也说过，真理和技能是难以穷尽的，这个世界上也不存在全能。优秀也只是一种支撑，即**条件化**。

我们前面就说过，依靠外界，你无法建立起真正的自信。同时，当你依赖外界的成功来获得自信时，你依然是活在别人的世界里，活在外界的标准里。

因为当你拥有绝对的自信时，你就不会依赖外在支撑。只有当你不够自信的时候，你才需要依赖外界的支撑。而那些支撑的标准，是不是外界制定的？所谓的成功，是不是外界认为的，而非你自己认定的？对吧。

所以，你必须能够认识你自己，明白你是谁，**无条件地接纳自己**。如何做到呢？别忘了我们关于自尊的课程，这里就不多重复。最终，你应该能够做到随遇而安，乐得其所。

**本节课中，我们一起学习了：**
1.  认识到对身高、长相、成功标准等先天或社会性特质的不接纳，是阻碍自信的常见障碍。
2.  理解了审美与社会成功标准是外界制定的、可变的，不应成为个人价值的标尺。
3.  区分了自我不接纳（如追求完美整形）与自爱行为（如改善健康问题的微调）。
4.  剖析了**条件化自信**的陷阱：即认为必须优秀才有资格自信，这会导致自信短暂且使人疲惫。
5.  明确了自信的本质在于**核心自我**的**无条件接纳**，而非依赖外在成就或评价。
6.  最终目标是摆脱对外界标准的依赖，建立基于自我认知和自我接纳的、稳固的内在自信。
