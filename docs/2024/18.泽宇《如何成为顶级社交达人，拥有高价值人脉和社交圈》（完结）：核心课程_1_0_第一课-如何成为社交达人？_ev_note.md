# 18.泽宇《如何成为顶级社交达人，拥有高价值人脉和社交圈》（完结）：核心课程_1_0_第一课-如何成为社交达人？

## 概述

在本节课中，我们将学习如何从零开始成为一个社交达人，并最终实现情感自由。

## 核心概念

**社交的本质是连接**。 

**人脉公式**： F = P * (1 + I)^N

其中：

* F 代表人脉
* P 代表初始人脉
* I 代表每天新朋友的数量
* N 代表天数

## 第一部分：社交达人之路

### 1. 社交知识架构

* 社交也是一个知识架构，需要系统学习。
* 从基础理念开始，逐步提升。

### 2. 认知差异

* 人与人之间最大的差别是认知。
* 改变认知差异，才能摆脱现状。

### 3. 社交概念

* 社交的本质是连接，无论目的如何。
* 快速建立连接，是社交的关键。

### 4. 建立关系网络

* 将自己视为关系网络中的中心点。
* 与更多点建立连接，扩大关系网络。

### 5. 每天认识一个新朋友

* 从零开始积累人脉的最佳方式。
* 一年下来，可以认识365个人。

## 第二部分：打破误解

### 1. 外在因素

* 不需要长得好看或很有钱。
* 提升内涵，自信交流。

### 2. 性格和个性

* 不需要改变性格和个性。
* 挖掘优点，改正缺点。

### 3. 社交时间

* 不需要花很多时间进行社交。
* 用最少的输入获得最大的输出。

### 4. 交友标准

* 不需要放低交友标准。
* 认识的人越多，标准越高。

### 5. 互联网时代

* 互联网连接所有人，社交更高效。

## 第三部分：社交达人衡量

### 1. 人脉

* 认识多少人，包括亲朋好友和弱连接。
* 人脉分为三层：亲朋好友、普通朋友、弱连接。

### 2. 资源

* 通过人脉触碰到的一切资源。
* 社会力量：调动除自身以外的社会力量。

### 3. 六度人脉理论

* 你和任何陌生人之间相隔的人不会超过6个。
* 通过6个人，可以认识世界上的任何陌生人。

### 4. 33%人脉原则

* 导师：在某个领域有丰富经验的人。
* 战友：志同道合的朋友。
* 门徒：愿意学习的人。

## 第四部分：行动起来

### 1. 每天认识一个新朋友

* 从零开始积累人脉的最佳方式。
* 一年下来，可以认识365个人。

### 2. 复利公式

* F = P * (1 + I)^N

### 3. 开始行动

* 不要等到准备好再开始，立即行动。
* 从今天开始，每天认识一个新朋友。

### 4. 跨出舒适圈

* 从打招呼开始，逐渐增加交流量。
* 尝试赞美，开启话题。

### 5. 内部群

* 加入内部群，与其他学员交流。
* 在线练习，提升能力。

## 总结

本节课中，我们学习了如何成为社交达人，并最终实现情感自由。通过每天认识一个新朋友，建立人脉关系网，我们可以获得更多的资源和社会力量，实现人生目标。
