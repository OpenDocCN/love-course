## 第三部分：术——执行细节 🛠️

上一节我们明确了通过“经历”来获得熟悉度的核心思路，并介绍了四大基本功。那么，在具体执行这些基本功时，会遇到无数细节问题，这就是“术”的层面。

“术”指的是具体的技术和操作方法，即落到实处该怎么做。例如：女生说“你好”，我该回“你好”还是一个表情包？这就是一个细节。

细节层面的学习规律是：**叠加与迁移**。

*   **细节叠加**：你会遇到大量细节问题。有些你可以自己处理，有些则需要导师帮助（如代聊、具体指导）。解决问题的过程，就是积累细节数据库的过程。
*   **知识迁移**：当你积累了足够多的细节经验（例如1000种），再遇到第1001种新情况时，你的大脑可以基于之前的数据库进行联想和迁移，从而找到解决方法。

**细节学习流程**：
```
遇到细节问题 -> 自行尝试或寻求指导 -> 解决问题 -> 经验存入数据库 -> 数据库扩大 -> 解决新问题的能力增强
```

因此，在私教陪跑过程中，初期（尤其是前半年）通常是学员细节问题最多的时候。随着课程学习、实践增加和问题被逐一解决，你的细节处理能力会自然“躺平”，变得娴熟。

对于学员而言，在“术”的层面，你需要做的就是：
1.  **听话照做**：遵循导师制定的计划和指导。
2.  **主动提问**：在实践中遇到问题，主动向导师提问，获取针对性的解决方案。
