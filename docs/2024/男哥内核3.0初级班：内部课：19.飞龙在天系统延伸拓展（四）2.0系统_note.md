# 男哥内核3.0初级班：内部课：19.飞龙在天系统延伸拓展（四）2.0系统

## 概述
在本节课中，我们将深入探讨飞龙在天2.0系统，学习如何通过朋友圈建设、阶段模型、退二进一策略和双线理论来提升个人魅力和社交技巧。

## 朋友圈建设
### 价值展示
- **核心概念**：价值 = 过去价值 + 当下价值 + 未来价值
- **具体内容**：社会地位、收入、阅历、圈子、预选（如健身、上进心）、价值观、厨艺、留学、学位、车、朋友多、事业、看书等。

### 朋友圈故事
- **核心概念**：通过分享个人经历和成长故事，展现个人魅力。
- **具体内容**：设定目标、记录生活、分享旅行经历、展示厨艺等。

### 旅行经历
- **核心概念**：通过旅行展示个人视野和生活方式。
- **具体内容**：计划旅行、分享旅行经历、展示旅行照片等。

### 经历的女生多
- **核心概念**：通过展示解决问题的能力和对婚姻、爱情的看法，提升个人魅力。
- **具体内容**：分享解决问题的案例、表达对婚姻、爱情的看法等。

## 阶段模型
### 阶段划分
- **陌生阶段**：基本信息交换，激发动力。
- **熟悉阶段**：基本信息交换，激发动力。
- **暧昧阶段**：约会，深化关系。
- **恋人阶段**：夫妻话题，共同生活。

### 阶段模型应用
- **核心概念**：根据当前阶段，采取相应的沟通策略。
- **具体内容**：陌生阶段以基本信息交换为主，熟悉阶段以激发动力为主，暧昧阶段以约会为主，恋人阶段以夫妻话题为主。

## 退二进一策略
### 退二进一
- **核心概念**：在当前阶段遇到问题时，退回上一个阶段，然后再进入下一个阶段。
- **具体内容**：例如，在暧昧阶段遇到问题，退回熟悉阶段，然后再进入暧昧阶段。

## 双线理论
### 双线理论
- **核心概念**：在沟通中，既要关注对方，也要关注自己。
- **具体内容**：在关注对方的同时，也要关注自己的感受和需求。

## 错误案例分析
### 案例一
- **错误原因**：直接跳过暧昧阶段，进入恋人阶段。
- **正确做法**：退回熟悉阶段，然后再进入暧昧阶段。

### 案例二
- **错误原因**：没有激发对方动力，直接提出要求。
- **正确做法**：先了解对方需求，再提出要求。

### 案例三
- **错误原因**：没有关注对方感受，只关注自己。
- **正确做法**：在关注自己的同时，也要关注对方感受。

## 总结
本节课中，我们学习了飞龙在天2.0系统的核心概念和应用方法。通过朋友圈建设、阶段模型、退二进一策略和双线理论，我们可以提升个人魅力和社交技巧，更好地与他人沟通和相处。希望同学们能够将所学知识应用到实际生活中，不断提升自己。
