# 梵公子 情绪价值核心课【向上社交破圈升思维】：价值交换语音包

## 第一节：价值交换概述

姐妹们，当你们听完公子的课程后，可能还会觉得云里雾里，不知道自己的价值体现到底是怎么样的。很多姐妹说我的先天价值很高，但她现在对我还是一般，她只会在前期对我有所付出。但后期不知道为什么，现在他对我完全没有动力了。现在反而是我天天主动联系他，天天粘着他等等等等一系列的问题。

### 先天价值

先天价值是一个道底是什么东西呢？有的人说是长相，有的说是身材，其实不是的，它是一个整体的概括。它是一个不单单说是形象的一个问题，它是整体的一个形一个概括。比如说他的一个从脸到脚，从头到手的一个整体形象，包括他的气质，他的一个为人处事，他的一个姿态。

### 物质价值

物质价值呢，你可以总结的归结为你的出身，你是的生活环境，你的居住条件，你的这个日常的一个吃穿用度，比如说你的爸妈是老师，你爸妈是医生，你的爸妈是一个企业的老总，这样你家里有几套房子一辆车，这个是你的物质价值。

### 情绪价值

情绪价值是你对一个人的一个状态的一个及时把握，一进行一个正面的反馈。你要求对方能够给你进行一个足够多的一个及时的一个反馈。你要求对方能够及时的察觉你的一个任何的一个情绪。

### 三者关系

他们三者呢是够能够相互转化的。啊，而且呢是能够巨大的变变化的。就是说我们这个社会为什么那些大佬能够一掷千金。为什么那些深夜的女主播能够给他们每天能进行大量的刷礼物，这就是他们需求的不同的一个体现。

### 关系义务论

觉得很多人觉得姐妹觉得我既然已经是他老婆了。她已经是我老公了，她对我付出不是理所应当的吗？啊，他对我好，不是应该的吗？我告诉你姐妹，没有应该没有。

### 总结

感情从来不是一个人的游戏。不是说你们单方面付出就能够得到一个很好的收候，它是需要两方共同维护，共同经营的这么一个东西。他是一定要有来有往的，是两个人的一个共同的一个游戏。
