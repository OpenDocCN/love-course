# 27、林夕-高质量关系课程：1.20-第20课《你和伴侣的最优相处模式》👫

## 概述
在本节课中，我们将探讨如何与伴侣建立最优的相处模式，并了解PAAC理论及其在亲密关系中的应用。

## PAAC理论
PAAC理论是描述亲密关系中双方状态的体系。它包括三种状态：**P（父母）**、**A（成年人）**和**C（孩子）**。

- **P（父母）**：优越感、统治感、话语权和权威性。
- **A（成年人）**：客观理性地看待问题，分析问题不情绪化。
- **C（孩子）**：服从、依赖和任人摆布。

## 伴侣相处模式
基于PAAC理论，两人相处可以形成六种模式：

1. **PP模式**：双父母模式，两人都处于统治者的状态。
2. **PA模式**：父母和理性者模式，一方权威统治，另一方理性冷静。
3. **PC模式**：孩子和父母模式。
4. **AA模式**：双成年人模式，两人都理性。
5. **AC模式**：理性者和孩子模式。
6. **CC模式**：双孩子模式。

## 模式解析
- **PC模式**和**AA模式**是最融洽的模式。
- **PP模式**、**PAA模式**、**CC模式**和**AC模式**容易出现问题。

## 如何找到适合的相处模式
- 根据不同情境和角色，灵活调整相处模式。
- 当对方处于某种状态时，以相应的模式应对。
- 通过观察和训练，找到适合双方的相处模式。
