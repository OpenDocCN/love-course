# 54、挽救婚姻有办法：夫妻吵架要遵守“四不一没有”原则

大家好，我是朱生勇。挽救婚姻，没有方向，挽回爱人没有办法，请听红外线治理。今天我们要跟大家讲的这个话题呢，是关于夫妻应该怎么吵架，对吧？夫妻到底应该怎么吵架，才能够把这个婚姻吵好。

才能够把夫妻关系吵圆满，才不至于出大的事情。那么我们说这是一个案子，一个杭州小情侣酒后吵架，他很跳楼，结果呢他的女友真的跳下去了，死了。审理的罪名是以故意杀人罪。故意杀人。

**以下是夫妻吵架时应该避免的情况**：

1. **醉酒情况下**：醉酒后情绪失控，容易做出非理性的事情。
2. **危险情况下**：如开车、乘坐缆车等，容易发生意外。
3. **夫妻生活时**：会影响夫妻生活和谐。
4. **吃饭时**：会影响食欲。
5. **妻子坐月子时**：会影响妻子恢复。
6. **教育孩子时**：会影响孩子成长。
7. **对方父母在场时**：容易引起矛盾。

**以下是夫妻吵架时应该遵循的原则**：

1. **不分手，不离婚**：避免用威胁的方式处理问题。
2. **不升级**：避免情绪激动，不断升级争吵。
3. **不扩大范围**：避免将争吵范围扩大到其他方面。
4. **不伤人格**：避免攻击对方的人格尊严。
5. **不重复争吵**：避免重复已经解决过的问题。

**吵架时的注意事项**：

1. **管控情绪**：避免情绪失控，做出非理性的事情。
2. **不要吵真相**：真相无法吵出来，要找感受。
3. **不要解决问题**：吵架很多时候无法解决问题，但可以释放情绪。
4. **吵架后要修补**：吵架后要进行解释和补偿，修复关系。
