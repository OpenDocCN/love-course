# 关键对话20讲：如何成为沟通高手：15-红脸白脸：“角色扮演”和“战术配合”

🎼大家好，欢迎来到关键对话，如何成为沟通高手的课堂。我是史新月。🎼这是我们第十五课，今天我们将探讨关键对话中的多人团队对话。🎼在进入对话实际操作的模块三之后，我们已经讲解了对话的准备和对话的进程。🎼接下来的几节课，我们将针对不同的对话场景和对话结构，逐一讲解对话中的难点。🎼今天这节课，我们将拆解一个常见的关键对话场景——团队对话。🎼团队的对话依然是两方的对话，并非多方的对话。我们将在下一讲专门讲解多方的对话。🎼有些人认为团队的关键对话仅限于工作场合，但实际上，它也出现在生活的各种场景中。🎼有些人认为，如果不是领导，团队对话与自己无关。但实际上，即使不是领导，团队中的人也需要明确团队对话的目标和分工。🎼对于团队的对话，本节课希望大家掌握基本的组织方法，以及团队配合的基本策略和战术，同时了解团队对话的缺陷，并学会克服这些缺陷。

🎼不是每场对话都需要团队参与，生活中有很多事情需要个人完成，但许多关键对话需要团队共同参与。因为问题和过程非常复杂，一个人无法独自完成对话，或者对话涉及到不同的利益，需要共同组成团队参与对话。

🎼有时候，我们是被给定一个团队，有时候我们可以在一定范围内选择队友参与对话。无论是给定的还是选择的，团队组织就像组织一个球队一样，有角色、有分工、有位置。一个对话或谈判团队中大概有以下几个角色：

1. 决策者：负责制定目标，确定何时可以接受对方的条件，何时停止谈判。
2. 发言人：在对话过程中与对方交涉，代表团队发言和回应对方。
3. 专家：负责把控专业问题，如购房时，对户型和位置信息了解较多的人就是团队的专家。
4. 辅助者：负责信息收集、安排会面、记录、编辑文档或翻译等工作。

🎼组织好团队后，整体团队需要充分做好对话前的准备。准备工作怎么做呢？按照我们在第十三课中讲解的那样做。团队准备起来应该更容易，因为人手多，可以分工，但团队之间一定要互相通气，让大家的信息都在一张纸上。

🎼团队完成的对话或谈判在过程中有一个角色分配的问题。这不仅仅是发言人、专家这样的功能角色，还有一种角色叫情绪角色，就像俗话里说的“一个唱红脸，一个唱白脸”。

🎼红脸代表好说话的那一方，能够与对方配合推进对话；白脸代表不好说话的那一方，可以让对方做出更多的让步和利益交换。红脸就像踩油门的，白脸就像刹车的，一红一白才能控制好车速。

🎼情绪角色和功能角色如何结合呢？一般来说，发言人是唱红脸，因为他需要与对方对话，推进对话进程；专家的角色一般是唱白脸，他总能从专业的角度提出问题，或者阻止发言人说这个方案不行。

🎼在团队的对话中，还可以考虑分组。有时候双方团队的人数较多，坐在一起谈没有效率，可以考虑分组，先坐下谈好框架，框架定了之后，双方各自的专业分组坐在一起谈。

🎼团队一起完成的对话优点很多，可以弥补个人的短板，可以产生出多种的战术策略。但是团队对话也有一些缺点。

🎼这是我们在团队对话中要意识到的问题，并尽量做出防范和补强。第一个缺点是团队内部的利益冲突。人一多，利益就多了。一个团队里，每个人、每个小团队的利益可能都不一样。

🎼第二个缺点是羊群效应。羊群效应说的就是大家都盲从于一个领导，人云亦云，不动脑筋。🎼第三个缺点是不容易保密。信息的保密在很多关键对话中起到重要的作用。人越多，就越不容易保密。

🎼总结一下今天的内容。这堂课我们着重讲了团队对话。关键对话因为它复杂，因为它关键，很多时候是由团队一起完成的。团队也不见得一定是工作单位里的同事团队，你们家你和邻居们，你和买了同一个产品的消费者，都可以构成一个团队。

以团队出面的对话，可以增强我们对话的综合能力，拓展了我们应用各种策略和战术的空间和可能性，但是也会增加对话的复杂程度。

🎼今天我们来做一个练习题，我们假设一个关公战秦琼的场景。水泊梁山要和大观园谈判，他们两家要合作投资一个红楼水浒联名的主题乐园，就比照着迪士尼和环球影城那种。

🎼请你分别给水浒一方和红楼一方组织两个团队来谈判，每个团队4到6个人。🎼水浒队里有水浒传中出现的人物，不一定是梁山好汉，书里有就行，比如高俅。🎼红楼也是，不一定是大观园贾府的人，书里有就行，比如贾玉村。

🎼你说我不熟悉水浒和红楼没关系，找你熟悉的作品，人物多一点的奥特曼、七龙珠、小猪佩奇都可以。你来组个队，练习的第一个问题就是你的队员都有谁，水浒队有谁，红楼队有谁。

🎼要说明每个团队里谁是决策者，谁是发言人，谁是专家，谁是辅助者。🎼练习的第二个问题，你怎么安排整个团队的情绪角色，谁是白脸，谁是红脸？🎼第三个问题，你预计这两个团队里会不会出现利益冲突、羊群效应和保密问题，你准备怎么防范？
