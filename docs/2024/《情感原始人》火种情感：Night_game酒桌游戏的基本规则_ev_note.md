# 《情感原始人》火种情感：Night_game酒桌游戏的基本规则_ev_note

在本节课中，我们将要学习Night_game（夜店游戏）中的几种核心酒桌游戏规则。这些游戏是社交破冰、营造氛围和关系升级的实用工具。我们将从最基础的骰子游戏开始，逐步介绍其他流行玩法，并了解如何将游戏与惩罚机制结合，为后续的情感互动打下基础。

## 🎲 骰子游戏（吹牛）

上一节我们介绍了课程概述，本节中我们来看看最经典的酒桌游戏之一——骰子游戏，也常被称为“吹牛”。这个游戏的核心在于猜测和虚张声势。

游戏的基本设定如下：通常使用两个骰盅，每个骰盅内有五颗骰子。游戏双方摇骰后，各自查看自己的点数，但**不向对方公开**。游戏目标是通过喊出某个点数的总数量来“吹牛”，迫使对方判断真假。

以下是游戏的核心流程与规则：

1.  **起始叫法**：由先手玩家开始，喊出“X个Y”（例如“3个2”），其中X代表骰子数量，Y代表骰子点数（1-6）。
2.  **叫法规则**：下家必须喊出比上家**更大的**点数组合。规则是：新的“X个Y”中，**X（数量）或Y（点数）至少有一个要比上家大**。
    *   例如，上家叫“3个2”，下家可以叫“4个2”（数量变大）或“3个3”（点数变大），但不能叫“2个2”或“3个1”。
3.  **数字“1”的特殊规则**：在大多数玩法中，点数“1”可以**当作万能数字**，代表任何点数（2-6）。例如，当你喊“3个5”时，对方骰子中的“1”也可以算作一个“5”。
4.  **“斋”与“非”的进阶规则**：在某些地区玩法中，喊数时可以加上“斋”。喊“X个Y斋”意味着本轮**数字“1”失去万能属性**，不能代表Y。如果下家想恢复“1”的万能属性，则必须喊“非”，但规则通常是**数量（X）需要翻倍**。例如，上家叫“3个2斋”，下家若叫“6个2非”，则“1”恢复为万能数字。
5.  **“顺子”的特殊情况**：当一方摇出的五颗骰子点数为“1、2、3、4、5”时，称为“顺子”。处理方式因地区而异，常见有两种：**视为0个任何点数**，或**要求玩家重摇**。具体规则需在游戏前协商一致。
6.  **“开盅”与胜负**：当一方认为上家所喊点数不可能达到时，可以选择“开盅”。双方公开骰子，清点全场（双方骰子加起来）实际包含所喊点数的骰子数量（计算“1”的万能规则）。如果实际数量**大于等于**所喊数量，则喊数者赢；如果**小于**所喊数量，则开盅者赢。

总而言之，骰子游戏是一个基于概率、记忆和心理博弈的游戏。关键在于通过对方的叫法习惯，推测其可能的点数，并灵活运用“吹牛”和“开盅”来取胜。

## ✋ 数字游戏：十五二十

了解了需要道具的骰子游戏后，我们来看一个更简单、只需双手就能玩的游戏——“十五二十”。这是一个考验反应和默契的猜拳类游戏。

游戏由两人进行。每只手可以代表两个数字：握拳为**0**，张开手掌为**5**。双方同时出手，并**喊出一个0、5、10、15、20中的数字**。所喊数字是**预测双方出手所代表数字的总和**。

以下是判断输赢的规则：

*   **猜中即胜**：如果你喊出的数字，恰好等于你出手代表的数字与对方出手代表的数字之和，则你获胜。
*   **均未猜中**：如果双方喊出的数字都与实际总和不符，则游戏继续，进入下一轮。
*   **同时猜中**：如果双方都喊中了实际总和，则为平局，游戏继续。

例如：
*   你出**一拳（0）一掌（5）**，总和为5。如果你喊“5”，对方喊“10”，则你猜中，你赢。
*   你出**两拳（0+0）**，总和为0。如果你喊“20”，这显然不可能实现（因为最大总和为20，需双方都出两掌），因此你必输。

这个游戏规则简单，但非常考验瞬间计算和反应速度，几轮下来就能快速上手。

## 👆 互动游戏：抓手指

在掌握了两人对决的游戏后，我们来看看适合多人参与、能快速炒热气氛的群体游戏——抓手指。这个游戏的核心是制造注意力差和进行群体挑战。

游戏的基本流程如下：
1.  一人作为**组织者**，将手掌平放于桌面中央。
2.  其他所有参与者各伸出一根手指（通常是食指），轻轻抵在组织者的手掌上。
3.  组织者倒数“3、2、1”，然后**迅速将手掌握拳，只竖起大拇指**。
4.  其他参与者需立即反应，去**抓住那个竖起的大拇指**。反应最快的第一人抓住组织者的大拇指，第二人抓住第一人的手指，以此类推，形成一个“手指链”。
5.  **最后一名**没有抓到任何手指（即手指处于链条最顶端无人可抓）的参与者，接受惩罚。

这个游戏的变体和趣味性在于组织者发起的**“挑战”**。组织者可以在倒数后不立即竖起拇指，而是提出一个条件或挑战。

以下是两种流行的挑战玩法：

*   **条件挑战**：组织者喊出一个条件，例如“穿牛仔裤的人赢！”或“戴眼镜的人安全！”。满足条件的人可以立即收回手指，视为安全；不满足条件且反应最慢者受罚。
*   **添酒挑战**：准备一个酒杯。从组织者开始，每人轮流往杯子里倒入少量酒。**目标是尽可能多倒酒而不让酒溢出**。轮到谁倒酒时导致酒溢出，谁就失败受罚。这个玩法充满紧张感和恶作剧的乐趣。

抓手指游戏能有效调动全场注意力，是进行轻度惩罚和开启大冒险的绝佳前置游戏。

## 💋 暧昧挑战：撕纸巾

最后，我们介绍一个在男女社交中常用于关系破冰和暧昧升级的游戏——撕纸巾。这是一个需要近距离接触的挑战游戏。

游戏规则如下：
1.  游戏发起者用嘴唇含住一张完整的纸巾。
2.  在不使用双手的情况下，将纸巾传递给下一个人。下一个人也必须用嘴唇接过，并**撕下一部分纸巾**。
3.  撕下部分后，由撕纸者用嘴唇含着剩余纸巾，继续传递给再下一个人。
4.  游戏如此循环，纸巾越变越小。
5.  当轮到某人时，如果纸巾已经太小而无法用嘴唇撕下（或在此过程中双方嘴唇发生接触），则该人失败并接受惩罚。参与者也可以在轮到自己前，因觉得纸巾太小而主动认输。

这个游戏通过逐渐缩小的纸巾和不可避免的近距离接触，自然而然地制造了紧张和暧昧的氛围。

## ⚖️ 游戏的核心：惩罚机制

在学习了具体游戏玩法后，我们必须认识到，游戏的灵魂往往在于其惩罚机制。单纯的喝酒惩罚效果有限。

强烈建议为所有游戏设立一个**三选一的惩罚规则**：
1.  **喝酒**：传统的惩罚方式。
2.  **真心话**：通过提问深入对方内心，是植入性幻想和了解窗口的有效方式。
3.  **大冒险**：要求对方完成一个指定行为，这是进行**服从性测试**的绝佳机会。

将惩罚多元化，能极大地提升游戏的互动深度和情感张力，为关系升级创造更多可能性。
