# 王哥私教课：王哥形象改造课 第17节 鞋子的改造 🚶‍♂️

## 概述
在本节课中，我们将学习如何根据身高选择合适的鞋子，以提升个人形象。

## 个子矮的兄弟
**公式**：个子矮的兄弟（1.8米以下）→ 选择厚底切尔西鞋

以下是几种推荐的切尔西鞋款式：
- **厚底切尔西鞋**：例如王哥推荐的厚底切尔西鞋，底厚约4厘米，脚背高，不易磨脚。
- **仿制品牌切尔西鞋**：如仿制Zaara品牌的切尔西鞋，价格亲民，质量可靠。

## 个子高的兄弟
**公式**：个子高的兄弟（1.8米以上）→ 选择多种切尔西鞋款式

以下是几种推荐的切尔西鞋款式：
- **经典切尔西鞋**：如高帮切尔西鞋，适合多种场合。
- **潮流切尔西鞋**：如布洛克、英伦风、潮流款式的切尔西鞋，展现个性。

## 增高鞋垫
**公式**：个子矮的兄弟（1.75米以下）→ 选择增高鞋垫

**注意事项**：
- 不要选择过于夸张的增高鞋垫，以免显得不自然。
- 选择质量可靠的增高鞋垫，确保舒适度和耐用性。

## 鞋子选择要点
- **质量**：尽量选择二三百元以上的鞋子，避免质量差的鞋子。
- **材质**：询问卖家是否为牛皮材质，确保鞋子质量。
- **款式**：选择适合自己的款式，展现个性。
