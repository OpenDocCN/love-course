# 玲金玉（五套合集）：1.亲密关系技巧 56节：14.口腔技巧_note

在本节课中，我们将学习口爱前的12个基础动作，这些动作分为嘴巴的6个动作和舌头的6个动作。掌握这些技巧有助于提升亲密体验。

## 嘴巴的六个常用动作 🫧

上一节我们介绍了课程概述，本节中我们来看看嘴巴可以完成的六个基础动作。这些动作主要利用嘴唇和口腔的配合。

以下是嘴巴的六个动作及其描述：

1.  **作**
    *   在说的过程中，可以适当发出声音。

2.  **磨**
    *   用对方的阴茎在嘴巴上，或者在其生殖器、睾丸、整个阴茎上进行摩擦。公式可理解为：**磨 = 嘴唇/口腔表面 + 与生殖器的持续接触与移动**。

3.  **滑**
    *   用嘴唇在对方的生殖器上做滑动的动作。

4.  **套**
    *   用嘴巴和口腔含住阴茎，进行套弄。代码逻辑类似：`while(含住) { 上下套弄 }`。

5.  **吸**
    *   吸的时候可以分为大力吸和轻轻吸。轻轻吸可以在阴茎体上进行。
    *   大力吸时，感觉像是口腔将空气全部吸入，整个口腔将其生殖器向内吸入。

6.  **夹**
    *   用嘴唇去夹住对方。

## 舌头的六个常用动作 👅

了解了嘴巴的动作后，我们接下来看看舌头可以完成的六个精细动作。舌头因其灵活，能带来更丰富的刺激。

以下是舌头的六个动作及其描述：

1.  **撩**
    *   用舌尖轻轻地去撩。可以用舌尖的三分之一处去撩对方的整个生殖器。也可以画Z字形撩。

2.  **擦**
    *   让对方的龟头在舌头上做一个摩擦的动作。

3.  **绕**
    *   用舌头去环绕阴茎。

4.  **扫**
    *   用整个舌面去扫它的整个生殖器。扫与撩的区别在于：撩是用**舌尖**轻轻进行，扫是用**舌背**大面积进行。

5.  **拍**
    *   用舌头像拍打一样动作。

6.  **拨**
    *   用舌尖去挑拨，例如将龟头从下往上挑起来。

本节课中我们一起学习了口爱前的12个基础技巧，包括嘴巴的6个动作（作、磨、滑、套、吸、夹）和舌头的6个动作（撩、擦、绕、扫、拍、拨）。熟练掌握这些动作的组合与变化，能有效提升亲密互动的体验。
