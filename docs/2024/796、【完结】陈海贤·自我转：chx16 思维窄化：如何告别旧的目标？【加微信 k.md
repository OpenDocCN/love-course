## 概述：转变的艰难阶段

上一节我们介绍了转变的第一个阶段。本节中我们来看看转变的第二个阶段——告别旧自我。转变之所以艰难，是因为它不可避免地伴随着失去。追求新梦想的人，首先会失去经济保障；离开旧关系的人，首先要面对关系结束的痛苦。这些失落和悲伤并非错误，而是转变必须经历的历程。就像装修房子需要先腾空旧家具，我们也需要告别旧自我，才能为新自我的诞生腾出空间。

## 目标对自我的意义

在探讨如何告别旧目标之前，让我们先理解失去一个目标对自我意味着什么。目标是自我很重要的一部分。它是尚未兑现的自我，是自我组织生活的方式，也是意义感的来源。它连接着过去的经验，也寄托着对未来的期望。因此，失去一个目标，就等于失去了一个定义自我的机会。

**公式：** `自我 ≈ 过去经验 + 当前目标 + 未来期望`

我有一个朋友曾是一位风光的创业者。公司失败后，他停止了创业，过上了看似自由的环球旅行生活，但他并不快乐。他唯一感到快乐的时刻，是在与创业者交流的场合中，他感觉自己仍是其中一员。创业失败对他而言，失去的并非金钱或世俗成功，而是一个定义自己、成为理想中那个自己的机会。当目标失去，我们就需要暂时告别目标背后的那个自我，去别处重新寻找“我是谁”。

## 为什么难以走出目标失去的困境？

也许你会问，只是失去一个目标，为何如此难以走出来？这是因为重要的目标会塑造我们，让我们的生活围绕着追求它而组织起来。一旦目标失去，自我就会散架，需要找到新目标才能重新拼凑起来。

这背后有一个关键的心理机制：**思维窄化**。

不知道你是否看过赛马比赛。赛马在比赛时会戴上一种特殊的眼罩，遮住两侧视线，让它只盯着眼前的赛道，心无旁骛。这个眼罩的优点是聚焦，缺点也是聚焦。它把成功与失败牢牢地与自我绑定在一起。

**代码隐喻：**
```python
# 思维窄化就像给视野加上过滤器
def view_world(goal):
    if sight in goal.related_things:
        return perceive(sight)  # 只关注与目标相关的事物
    else:
        return ignore(sight)    # 忽略目标之外的世界
```

当你为一个目标努力时，你也给自己戴上了这样的“心理眼罩”。你通过它来定义成功、失败、顺利与不顺。当目标失去，就像比赛结束的赛马，你仍习惯性地盯着空荡的跑道，只觉得自己输掉了比赛，却不知该往何处去。除非摘下眼罩，否则你很难发现目标之外还有一个更大的世界。

## 摘下眼罩：电影《心灵奇旅》的启示

让我举一个例子。电影《心灵奇旅》讲述了一位名叫乔伊的黑人，他一心梦想成为爵士钢琴家。他将全部自我寄托于此，以至于觉得梦想之外的现实生活（如稳定的工作、人际关系）都平庸且令人失望。

一个悲剧性的转折后，他的灵魂“偷渡”回人间。这次经历迫使他摘下了“梦想”这个眼罩。他忽然发现，自己一直生活在富足之中——披萨的香味、普通人的善意、母亲的爱。此前，他的生活被目标分割为“实现目标前”（匮乏无意义）和“实现目标后”。而摘下眼罩后，他意识到生命本身、活着本身就是一种富足。

这个例子并非否定追求目标。专注目标能调动资源实现它。问题在于，失去目标后，如何突破**思维窄化**，重新看见并珍视自己所拥有的，重新定义自己。

## 应对失去：从“有限游戏”到“无限游戏”

说到这，就要探讨如何面对失败。哲学著作《有限与无限的游戏》提出了两种游戏：
1.  **有限游戏**：有明确规则、起点和终点。玩家的目标是尽快结束游戏并成为赢家。
2.  **无限游戏**：没有明确的胜负、起点和终点。玩家的首要目标是让游戏继续下去。

书中的一个核心观点是：**人生是一个无限游戏**。因此，单个目标的失败不算什么，重要的是让游戏继续下去。

这两种游戏提示了不同的自我定义方式：
*   在**有限游戏**中，你用目标的成败来定义自我：`自我 = 成功者 或 失败者`。
*   在**无限游戏**中，你定义自我的方式改变了：`自我 = 玩游戏的人`。这个身份不会因短暂的成功或失败而失去。

因此，应对目标失去的最好方式，就是更新你对自我的定义。你要知道，以成败论英雄是目标带来的**思维窄化**的产物。现在，你要学习用新的方式定义自己：

以下是几种可能的新定义：
*   追求梦想的人
*   不惧失败的人
*   接纳现实的人
*   正在转变的人

当你摘下思维窄化的眼罩，你就会发现自己身处一个更广阔的世界，拥有更多可能性。这些可能性正等待你去创造新的自我。

1.  失去目标是自我转变中必然且艰难的部分，因为它意味着失去了一个定义自我的机会。

3.  应对目标失去的关键是突破思维窄化，从“有限游戏”的成败思维，转向“无限游戏”的玩家思维，用更丰富的方式重新定义自己。
