# 好述情感《破除世俗价值观》：009.2.9吸引的实质

## 概述
在本节课中，我们将探讨吸引的实质，包括行为线索、好情绪和认同感三个关键要素。

## 行为线索
**以下是行为线索的介绍：**
- 行为线索体现在男人与女人之间的讲话和做事情背后的前沟通中。
- 前沟通指的是说话和做事的方式以及背后的前台词。
- 真正吸引女人的行为线索包括决断力、控制力和风趣骄傲。

**公式描述：**
- 行为线索 = 决断力 + 控制力 + 风趣骄傲

## 好情绪
**以下是好情绪的介绍：**
- 好情绪指的是男人在与女孩互动时能自然产生的好感觉。
- 男人首先需要在自己的社交圈中感到开心、兴奋和有趣。
- 通过提供情绪价值，如好情绪，可以吸引女孩。

**公式描述：**
- 好情绪 = 男人自身的快乐 + 情绪价值

## 认同感
**以下是认同感的介绍：**
- 认同感指的是女人在社交团体中判断自己在社交阶层的位置。
- 如果女人认为与某个男人在一起能提高自己的社交地位，她可能会选择与他在一起。

**公式描述：**
- 认同感 = 社交地位提升

## 总结
本节课中我们一起学习了吸引的实质，包括行为线索、好情绪和认同感三个关键要素。这些要素共同作用，使男人能够吸引女人。
