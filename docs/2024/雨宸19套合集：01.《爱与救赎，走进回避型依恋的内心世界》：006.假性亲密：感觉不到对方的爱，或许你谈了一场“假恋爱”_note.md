# 雨宸19套合集：01.《爱与救赎，走进回避型依恋的内心世界》：006.假性亲密：感觉不到对方的爱，或许你谈了一场“假恋爱” 💔

在本节课中，我们将要学习什么是“假性亲密关系”，它与真正的亲密关系有何区别，以及它对回避型依恋者有何影响。我们还将探讨如何打破假性亲密，建立真诚的情感连接。

## 什么是假性亲密关系？ 🤔

上一节我们探讨了回避型依恋的某些行为模式，本节中我们来看看一种常见的关系状态——假性亲密。

假性亲密是一种形式上的亲密关系。它是一种为了避免关系中的不确定性而建立起来的防御机制。处于假性亲密中的双方缺乏对彼此的真心和信任，一点冲突和改变都有可能让他们分开。

简单来说，就是你感觉与对方之间隔着一层看不见的玻璃。你想靠近对方，但没有动力，也害怕受伤害。对方也有同样的想法，想离开又舍不得，于是就一直维持这种状态。

## 什么样的人容易陷入假性亲密关系？ 🧐

了解假性亲密的成因，有助于我们识别和改变它。以下是三类容易陷入假性亲密关系的人：

1.  **不知道自己想要什么的人**：回避型依恋者在爱的意识形成时，并未得到过真正安全的爱。因此，他们不知道自己理想的爱情是什么样子，也不确定想从一段感情中获得什么。他们不清楚自己的情感需求，对一段恋爱是否合适、为何舍不得、为何要谈恋爱都感到迷茫。
2.  **不清楚伴侣需求的人**：回避型依恋的人不会主动走进别人的情感空间，因此很难意识到作为伴侣应该为对方做什么。他们不了解对方的需求，也不知道该如何去满足。
3.  **在情感认知上有障碍的人**：许多人错误地认为确定关系就是恋爱的终点，而忽略了关系确定后，亲密关系仍需发展和升华。他们觉得确认关系后就可以“放飞自我”，不再维护感情，最终导致关系破裂。这就像买车后从不保养，随便乱开直到车坏掉一样。

## 真假亲密关系的区别 🔍

理解了假性亲密的特征，我们再来对比一下真假亲密关系的核心区别。

**在沟通方式上：**
*   **假性亲密**：缺乏坦诚、真挚的深度沟通，抗拒分享内心真实想法。彼此更像是在相互利用、相互依靠，就像荒野中两只靠在一起取暖的狼，太阳出来就可能分开。
*   **真正亲密**：善于自我暴露，能够坦诚分享自己的经历和感受。

**在情感状态上：**
*   **假性亲密**：关系的稳定像是按照设定好的“剧本”或“人设”在发展，很多时候是表演给外人看的，并非真实状态。
*   **真正亲密**：信任、亲密和爱是关系的第一位。双方能主动认识并探求彼此的需求，并相互满足。

**在互益性上：**
*   **假性亲密**：几乎不会让你得到成长，反而可能让你患得患失、经常被否定或打击，感觉自己越来越差。
*   **真正亲密**：对双方的价值和自我认知都有提升，让两个人都能获得进步。双方都能在对方眼中看到更好的自己。

## 如何打破假性亲密？——DREAM五步法 🌈

既然假性亲密让人痛苦，我们该如何打破它，成为真心爱人呢？心理学家提出了一个名为“DREAM”的五步法。

以下是实施DREAM五步法的具体步骤：

1.  **发现（Discovery）**：发现彼此在关爱中的异常表现，例如抗拒分享、抗拒沟通。关键是找到关系中不和谐的矛盾点，并意识到问题的存在。
2.  **修复（Repair）**：主动说出自己的感受，但要避免传递焦虑。应提出解决问题的方式，而不是把问题扔给对方。正确的表达是客观陈述事实和当下的心理状态，例如：“你刚才做的XX事，让我感到很难受。” 然后给出你认为可行的解决方案。
3.  **赋予（Empowerment）**：摒弃负面的交流方式。当对方需要时间思考或采取回避时，不要逼迫对方立刻改变或做决定，而要学会赋予自己力量。这意味着把伴侣看作平等的人，在提出诉求后，给予对方消化和回应的空间。
4.  **关爱（Alternately）**：尝试理解回避型依恋的深层需求，做那个主动坦诚感受和暴露真实自我的人。拉近关系的关键在于“分享秘密”。当你敢于展示自己真实的生活状态，甚至是一些“糗事”时，对方会觉得与你更熟悉、更亲近。喜欢的程度，与了解的程度成正比。
5.  **豁达（Mettle）**：不要执着于感情中的得失。与回避型依恋者相处，需做好付出可能得不到同等回报的准备。关键在于：**在不强迫对方的同时，保证自己的情感不受伤害**。感情中没有绝对的平等，要学会享受为感情付出的过程，而不是一味期待即时回应。如果在一段关系中，你持续感到痛苦、压抑和不舒服，那么及时止损是必要的。评判感情好坏的唯一标准，永远是你自己的感受是否良好。

## 总结 📚

本节课中，我们一起学习了“假性亲密关系”的核心概念。我们了解到，假性亲密是一种缺乏真心与深度连接的形式关系，常见于不清楚自我需求、不了解伴侣或对关系认知有偏差的人群。它与真正亲密关系在沟通、情感状态和互益性上存在显著区别。

最后，我们介绍了打破假性亲密的“DREAM五步法”：**发现**问题、**修复**沟通、**赋予**彼此空间、主动**关爱**与分享、以及保持**豁达**的心态。记住，建立真诚关系的关键在于坦诚的自我暴露、平等的沟通，以及最重要的——始终关注并尊重自己在关系中的真实感受。
