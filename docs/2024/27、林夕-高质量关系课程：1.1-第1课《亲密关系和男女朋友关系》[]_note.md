# 27、林夕-高质量关系课程：1.1-第1课《亲密关系和男女朋友关系》👫

## 概述

在本节课中，我们将探讨亲密关系和男女朋友关系的建立，以及如何正确处理和定位这些关系。

## 亲密关系的理解

**核心概念**：亲密关系是两性关系的核心吸引力，对性的态度决定了关系的和谐程度。

**公式**：亲密关系 = 正确的性态度 + 相互享受

**要点**：

* 亲密关系不会因为发生性关系而产生很大变化。
* 不要过度看重性关系，避免心理落差。
* 可以明确表达自己的性观念，但不要强迫对方。

## 亲密关系的最佳时间点

**核心概念**：亲密关系的最佳时间点取决于关系的定位。

**公式**：

* 短期关系：明确认知后，随时可发生。
* 长期关系：从认识到暧昧阶段，1-3个月左右。

**要点**：

* 短期关系：避免沉迷，保持清醒。
* 长期关系：避免草率，循序渐进。

## 关系定位的方法

**核心概念**：通过不同渠道认识、相处场合等因素判断关系的定位。

**要点**：

* 社交软件、酒吧等场合：短期关系可能性较大。
* 相亲、朋友介绍等场合：长期关系可能性较大。

## 确定关系的方法

**核心概念**：通过关键谈话和操作确定关系。

**要点**：

* 发生关系后进行确认谈话，了解对方的感受和期待。
* 通过沟通了解对方的未来规划，判断是否被纳入规划。
* 通过服从性测试判断对方的投资程度。
* 通过情感博弈引导对方付出，确认关系。

## 关系润色和推进

**核心概念**：通过提高亲密关系的和谐程度、了解对方的频率需求、调整期待值等方式推进关系。

**要点**：

* 了解对方的感受，进行复盘和沟通。
* 了解对方的频率需求，进行沟通和调整。
* 调整期待值，循序渐进地接受。

## 短期关系转长期关系

**核心概念**：通过停止无成本的性行为、引导植入长期观念等方式将短期关系转变为长期关系。

**要点**：

* 停止无成本的性行为，让对方付出。
* 引导植入长期观念，表达自己的认真态度。
* 提升自己的价值，吸引对方。

## 识别渣男信号

**核心概念**：通过观察对方的沟通方式、态度变化、回避关键问题等方式识别渣男。

**要点**：

* 不谈未来，只谈当下。
* 主次颠倒，只回复心爱话题。
* 发生关系后态度变化，只聊邀约。
* 回避关键问题，引导你投资。
