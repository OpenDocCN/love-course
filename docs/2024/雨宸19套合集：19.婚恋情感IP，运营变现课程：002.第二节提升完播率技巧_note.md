# 雨宸19套合集：19.婚恋情感IP，运营变现课程：002.第二节提升完播率技巧

## 概述
在本节课中，我们将学习如何提升短视频的完播率，这是影响视频在抖音平台热门的重要因素之一。

## 完播率与平均播放时长
**公式**： 完播率 = 完整播放次数 / 总播放次数

**核心概念**： 平台更在乎的是哪个内容创作者创造出来的哪一条视频能够帮助平台把用户的时间更多的消耗在抖音上。

## 提升完播率的方法
以下是几种提升作品平均播放时长，从而提高完播率的方法：

1. **爆点前置**： 在视频开头就放一个爆点，引发观众的好奇心。
2. **价值预告**： 在视频开头就表达出视频的核心价值，引发观众的期待。
3. **干货秒过**： 视频中包含大量干货，播放速度快，引发观众复看。
4. **层层递进**： 情节或内容层层递进，让观众欲罢不能。
5. **节奏紧凑，逻辑紧扣**： 视频节奏快，避免观众产生不想看的感觉。
6. **内容不够，画面埋凑**： 如果内容不够强，可以适当丰富画面。

## 提升点赞率的方法
以下是几种提升点赞率的方法：

1. 引发用户的认同、喜欢或趣味。
2. 满足用户的猎奇心理。
3. 内容正能量。
4. 内容有共鸣。
5. 内容有收藏价值。

## 总结
本节课中我们一起学习了如何提升短视频的完播率和点赞率，以及如何进行账号定位和标签优化。希望这些知识能够帮助你在抖音平台上取得更好的成绩。
