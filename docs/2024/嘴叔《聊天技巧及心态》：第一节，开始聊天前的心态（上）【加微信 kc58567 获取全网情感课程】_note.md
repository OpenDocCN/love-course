# 嘴叔《聊天技巧及心态》：第一节，开始聊天前的心态（上）💡

在本节课中，我们将要学习与异性开始聊天前，必须建立的核心心态和认知转变。这是所有后续聊天技巧的基础，心态摆正，技巧才能发挥作用。

## 概述：心态是聊天的基石

很多人在与心仪的女生聊天时，无法进入良好状态，甚至让女生感到厌烦，其根本原因往往在于错误的心态。本节课程将详细剖析两种必须建立的基础心态，帮助你从起点就走在正确的道路上。

## 一、建立平等关系，而非讨好

上一段我们概述了心态的重要性，本节中我们来看看第一个核心心态：平等。

很多兄弟聊天无法进入良好状态，或者无法让女生真正喜欢自己，是因为他们的整个聊天出发点只是为了讨好对方。他们的目的单一，总是把自己摆在很低的位置，认为需要去迎合和打动女生，对方才有可能搭理自己。

如果用这种方式去聊天，无论后续学到多少技巧，只要位置没有摆对，失败的概率仍然会非常高。你不能把自己放在一个很低的位置上。

**核心概念**：吸引的本质在于价值感知，而非服务或讨好。就像人们选择苹果手机，往往不是因为其服务有多好，而是因为其品牌价值高、显得稀缺和珍贵。公式可以表示为：
**吸引力 ∝ 感知价值**

因此，在聊天之前必须明白的第一点是：你并不是要通过聊天去打动、跪舔或感动妹子。最基础的心态应该是双方相互平等，并且互相了解。

**关键点**：聊天只是为了互相了解，而不是去做很多事情打动对方。背后透露“你配不上对方”信息的讨好行为，必须避免。

如果一开始就有讨好行为，那么学习再多的后续话术和技巧，都没有太大用处。

## 二、明确聊天目的：邀约而非让她喜欢上你

理解了平等心态后，我们接下来看看聊天的核心目的应该是什么。

当刚认识一个姑娘，还没有能经常单独约她出来之前，你的目的只有一个：把她约出来。而不是让她喜欢上你。

这个想法错误的点在于：如果你努力想让妹子喜欢上你，你就会出现一种表现——太过于努力地去展示自己。而这会给女生传达一个潜在信息：你把自己放得比她低，你需要去讨好她。这打破了我们之前说的最基本的一点：平等。

只要关系不平等，那么怎么聊都不行。处在平等、互相了解的心态下，才可能开始一段早期的聊天。

回顾过去错误的聊天经历，例如没能把妹子约出来，或者聊天气氛很差，你会发现基本上都犯了把自己摆得太低，或太想展示自己让对方喜欢上你的错误。

在对方眼中，这样的你是没有地位与价值的，会显得很缺乏异性关注、很饥渴甚至很黏人。如果对方对你产生这种感受，学再多的聊天技巧都无法弥补，因为基本面已经错了。

## 三、拥有筛选心态，而非被动接受

明确了平等和以邀约为目的后，我们还需要一个进阶心态：筛选。

你真正需要的不仅是一种互相了解的心态，甚至你还在筛选她，因为她同时也在筛选你。你一定要有一种你在筛选她的心态。

你要知道，你认识的漂亮姑娘通常有很多人追，她会在众多追求者中筛选最有价值的人。为什么她会筛选掉你？因为她的选项多。

而你要让她知道，你其实也在筛选她。你至少要有这样一种心态：你不是“是个女的就行”。

以下是理解筛选心态的关键点：

*   **强势方的行为模式**：真正漂亮、不缺追求者的女人，她只会做一件事：筛选男人。她不会刻意展示自己（如“我会做饭、很漂亮”），因为她不缺。她的典型行为是 **“提要求”**。
*   **举例说明**：例如在相亲网站上，她会直接列出标准：“35岁以上勿扰”、“收入低于XX勿扰”、“身高低于XX勿扰”。当她处在不缺的、强势的地位时，就会采用这种筛选模式。
*   **男性的对等心态**：所有不缺女人的男人，心中也会有筛选标准。例如，明确知道自己想要的女人是心智成熟、可以沟通、会对感情切实付出的。正因为有了标准，遇到不符合的妹子时会保持距离，或直接表明“你不是我喜欢的类型”。这样反而会让对方觉得你特别不缺异性，从而更想吸引你的注意。

当你越有标准时，在妹子眼中，你反而是一个特别不缺女人的人。当你没有标准，“只要是个漂亮的妹子就行”就开始跪舔时，再高深的聊天技巧都救不了你。

**核心操作**：你可以通过分享经历（考验编故事的能力），让她知道你有自己的标准，并且曾因标准拒绝过一些妹子。
**示例代码（话术框架）**：
```
“之前有个妹子聊得还不错，但有次我发消息问正事，她没回却发了朋友圈。我觉得这很不尊重人，后来就没再联系了。我很看重互相尊重和及时回应。”
```

让她知道你和那些毫无底线、只知道跪舔讨好的男人是完全不一样的。这就是前期聊天必须拥有的心态。

## 总结与回顾

本节课中我们一起学习了开始聊天前必须建立的两个核心心态：

1.  **平等心态**：聊天是为了互相了解，而非讨好或打动对方。你的价值不通过迎合来体现。
2.  **筛选心态**：你与对方是互相考察的关系。拥有并适当展示自己的择偶标准，能有效提升你在对方眼中的价值感。

记住，在妹子时刻筛选你的同时，你也要筛选她。如果只是她在筛选你，而你对她没有标准要求，那么你在他心中就是没有价值的。前期聊天的正确心态，是建立在平等和互相筛选的基础上，目的是互相熟悉以达到邀约。
