**公式**：平等心态 + 筛选心态 = 有效沟通

## 开始聊天前的心态

### 1. 理解平等的重要性

在聊天之前，你需要明白，你和女生应该是平等的，或者你略高于她的位置。很多兄弟在与女生聊天时无法进入良好状态，原因在于他们没有摆正自己的位置。

**代码**：`平等心态 = 互相尊重 + 价值对等`

### 2. 聊天的目的

聊天的目的不是为了讨好女生，而是为了互相了解。记住，你聊天只是为了让她更多地了解你，同时也让你更多地了解她。

**代码**：`聊天目的 = 互相了解`

### 3. 筛选心态

在聊天过程中，你需要有一种筛选的心态。因为女生也在筛选你，所以你也需要有自己的标准。

**代码**：`筛选心态 = 自我价值 + 标准明确`

### 4. 建立自己的标准

所有不缺男人的漂亮女生都有自己的标准和条件。你应该明确自己想要的类型，并根据自己的标准进行筛选。

**代码**：`建立标准 = 自我认知 + 条件明确`

### 5. 表达自己的标准

在聊天过程中，你可以向女生表达自己的标准，让她知道你并不是随便见女人就舔的男人。

**代码**：`表达标准 = 自信表达 + 价值体现`
