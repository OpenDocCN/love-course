# 凯哥《3899私教系列》：2.情商提升3.0：003.情商3.0课程-转变思维

## 概述

在本节课中，我们将学习如何提升情商，转变思维，从而在情感和事业中取得成功。

## 核心概念

* **艾宾浩斯遗忘曲线**：人会在第二天忘记第一天学习内容的80%。因此，学习时需要记笔记，反复听课和复习。
* **内核**：指一个人的内在品质，包括价值观、思维方式、行为习惯等。
* **价值**：指一个人在情感和事业中所展现出的吸引力。

## 第一部分：转变思维

### 1. 情感小白

* **问题**：很多人停留在情感小白的阶段，不知道如何与异性交往。
* **原因**：缺乏动力，没有学会爱自己，没有原则和底线。
* **解决方案**：学会爱自己，设定原则和底线，提升自身价值。

### 2. 女人的驯化

* **问题**：女人倾向于驯化男人，让他成为自己的仆人。
* **原因**：女人需要征服和安全感。
* **解决方案**：建立自己的原则和底线，展现自己的价值，成为女人的领导者。

### 3. 话术与价值

* **问题**：很多人依赖话术，却忽略了自身的价值。
* **原因**：缺乏自信，没有认识到价值的重要性。
* **解决方案**：提升自身价值，学会在正确的时间做出正确的行为。

## 第二部分：实战案例

### 1. 七步聊天法

* **目的**：通过七步聊天法，提升聊天效果，吸引异性。
* **步骤**：
    1. 开场白：用有趣的方式开场，引起对方兴趣。
    2. 提供价值：分享有趣的话题，展现自己的价值。
    3. 聊天节奏：一句逗逼，一句正经，保持聊天兴趣。
    4. 升温关系：通过互动，拉近彼此距离。
    5. 推进关系：在合适的时候，提出邀约。
    6. 模糊邀约：用模糊的方式邀约，给对方留下想象空间。
    7. 约会细节：注意约会细节，展现自己的魅力。

### 2. 实战案例

* **案例**：凯哥通过七步聊天法，成功吸引了一位女生，并与其发展成恋爱关系。
* **关键**：展现自己的价值，学会在正确的时间做出正确的行为，保持自信。
