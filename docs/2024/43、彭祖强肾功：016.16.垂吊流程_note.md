# 43、彭祖强肾功：016.16.垂吊流程_note

在本节课中，我们将学习彭祖强肾功中垂吊环节的准备步骤，核心内容是掌握湿数布的正确折叠方法，为后续的垂吊练习打下基础。

## 准备材料

首先，垂吊之前需要准备一块湿数布。这块湿数布会提供给学习彭祖强肾功的学员。

大家收到这块湿数布以后，可以用手清洗并晾干，然后再进行使用。

## 折叠步骤详解

上一节我们介绍了所需的材料，本节中我们来看看具体的折叠流程。以下是湿数布的折叠步骤。

1.  **对折布条**  
    首先，将布对折成一条长条状。

2.  **铺设与对齐**  
    找一个桌面将布铺平，两边一定要对齐。有字的一面朝上。

3.  **折出四指宽度**  
    将布对折，折出大约四个手指并拢的宽度。

4.  **向前折边**  
    用食指和拇指捏住对折后的边，再次向前折大约一指的宽度。这个动作一共重复三次。折的时候要向下按压。

5.  **整理成长条状**  
    在折叠过程中，同时向两边拉扯布条，逐渐将其整理为平整的长条状。注意，不要将布条卷成圆筒状。

6.  **完成对折**  
    最后，将湿数布对折，使其呈现为规整的长条状。

本节课中我们一起学习了垂吊前湿数布的完整折叠流程。正确的折叠是确保垂吊练习安全、有效进行的重要前提，请务必按步骤仔细操作。
