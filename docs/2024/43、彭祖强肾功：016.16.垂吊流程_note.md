# 🎼彭祖强肾功：016.16.垂吊流程

大家好，我是陈伟民。这节课给大家讲解一下垂钓。首先垂钓之前咱们需要准备。

**以下是垂钓前的准备工作：**

1. 准备一块湿数布，施筑部会为彭祖强肾功学员准备，届时会送给你们。
2. 收到湿数布后，可以用手清洗晾干，再进行使用。

**湿数布折叠步骤：**

1. 将湿数布对折，形成一条长条。
2. 将对折后的布铺在桌面上，确保两边对齐。
3. 将布对折，形成四指宽度。
4. 用食指和拇指将对折的边再次往前折一指宽，重复三次，同时向下按压。
5. 拉扯布条，使其逐渐变为长条状，注意不要卷成圆形。
6. 将布条对折，使其呈长条状。

**总结：**

本节课中，我们学习了彭祖强肾功的垂吊流程，包括准备工作及湿数布的折叠方法。希望对大家有所帮助。
