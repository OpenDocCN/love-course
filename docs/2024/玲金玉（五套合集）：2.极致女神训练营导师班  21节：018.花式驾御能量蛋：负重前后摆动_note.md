# 玲金玉（五套合集）：2.极致女神训练营导师班  21节：018.花式驾御能量蛋：负重前后摆动

🎼经过前面的训练，现在我们将学习如何控制负重的前后摇摆。🎼

## 概述
在本节课中，我们将学习如何通过控制盆底肌来使负重物体进行前后摆动。

## 准备工作
- 双脚打开与肩同宽。
- 负重物体（如小球）。

## 步骤
1. **摆动准备**：
   - 负重物体可以不需要太重，初期可尝试挂一个小球或两个小球。
   - 脚不要太宽，幅度可适当收窄。

2. **盆底肌控制**：
   - 使用盆底肌向前收。
   - 用意念想象盆底肌连接肛门尾骨位置，向前收紧并向上卷起。

3. **摆动过程**：
   - 当盆底肌收紧时，负重物体将前后摆动。
   - 随着摆动，跟随节奏进行收缩和放松。

4. **练习方法**：
   - 如果找不到感觉，可以轻轻晃动负重物体，使其前后摆动。
   - 然后收缩盆底肌进行训练。

5. **技巧**：
   - 先让负重物体摆动，再用盆底肌控制。
   - 经常练习，找到前后摆动的感觉。

## 总结
本节课中，我们一起学习了如何通过控制盆底肌来使负重物体进行前后摆动。通过不断的练习，我们可以提高盆底肌的控制能力，达到更好的训练效果。🎊
