# 男性提升《好逑情感》：5、斌哥—吸引力法则：7_吸引力7

## 概述
在本节课中，我们将学习斌哥关于吸引力法则的讲解，包括建立信念、价值认知以及个人标准期望和原则的重要性。

## 建立信念
**信念**是指对一件事情的执着，即使受到冲击也不会被击溃。你必须相信自己强大且具有吸引力。

**公式**：信念 = 执着 + 不被击溃

**例子**：优秀的销售人员会坚信自己的产品，因为他们相信产品的价值。

## 价值认知
**价值**包括生存价值、繁衍价值和情绪价值。你必须认识到自己的价值，并相信它。

**公式**：价值 = 生存价值 + 繁衍价值 + 情绪价值

**例子**：一个没有学历和技能的人，即使幻想也能创造情绪价值，但这种价值是没有意义的。

## 个人标准期望和原则
**标准期望**和**原则**是筛选和测试他人的基础。你必须清楚自己想要什么，不能容忍什么。

**公式**：标准期望 = 想要的 + 期待的 + 不能容忍的

**例子**：一个想要一个能够辅助自己的女人，并期望她在关系中保持健康、有野心，不能容忍懒惰和愚蠢。

## 总结
本节课中，我们学习了斌哥关于吸引力法则的讲解，包括建立信念、价值认知以及个人标准期望和原则的重要性。通过这些方法，我们可以提升自己的吸引力，更好地筛选和测试他人。
