# 倪《高段位主导关系》：003.三、你错在哪？为什么你陷入被支配的关系中了？

🎼为什么很多兄弟会逐渐陷入被支配被主导的关系里呢？他们的性格中或者认知里哪些固化的思维导致了这样的结果呢？这是今天这节课我们要解决的问题。

## 关系中的权利意识缺失

凡事会陷入在关系里被支配被主导的地位的兄弟啊，是因为他们不具备关系中的权利意识，他没有要主导关系的观念。

## 错误思维导致的结果

他没有什么啊，先说他拥有什么，他有什么才造成了现在的情况。他拥有了什么样的错误的思维才导致了这个结果。

### 例子：女生要看你手机

女生要看你手机的问题啊，女生说啊要看手机，因为这是爱的体现，这是信任的体现啊，要不你就是心里有鬼。

### 妨碍关系稳定的错误逻辑

这个错误的逻辑是只要我妥协就能维护好关系。

## 权力博弈与关系稳定

暂时的妥协是会获得暂时的关系和谐，但随着他的权利变成一种理所应当的时候，不是他的要求会越来越过分，而是他会变成你的权威。

## 关系中的权利地位

在一段关系里，谁该占主导地位？有人会说，当然是男人啊，但是呢现在越来越主流的一种声音是宣扬女性在关系中的权利。

## 权利意识的植入

这个观念是什么时候被植入到我们头脑里的呢？也许呢是在你还小的时候，面对过于严苛和暴躁的父母，你只要表现的听话顺从，就能得到他们的关注。

## 更新思维模式

妥协不能够换取关系的和谐，妥协只会失去关系，会让女人蔑视你，不珍惜你和背叛你。

## 主导关系的重要性

主导关系才能让关系更长久。而高位置是通过博弈而来的，这就涉及到了我们这门课程的核心理念了。
