# 27、林夕-高质量关系课程：1.-第课《情绪》[]

大家好，我是林夕。今天我们探讨高质量关系中的自我关系，重点落在情绪管理上。情绪管理至关重要，因为它关乎你能否掌控自己的情绪，进而影响他人。

**情绪管理的核心**

情绪管理并非压抑，而是通过高效的方法快速冷静，不感到委屈。关键在于提升认知，拓宽视野，避免钻牛角尖。

**情绪的分类**

情绪分为积极情绪和负面情绪。积极情绪包括喜悦、感激、激动、自豪等，而负面情绪包括愤怒、伤心、恐惧、痛苦、委屈、遗憾等。

**情商的三个等级**

1. **情绪管理**：初级水平，学会管理自己的情绪。
2. **情绪价值**：中级水平，不仅管理情绪，还能在对方情绪波动时提供价值。
3. **冲突管理**：高级水平，高效解决冲突，实现双赢。

**情绪管理的重要性**

情绪影响行为，行为导致结果。管理好情绪，避免做出错误的行为，从而掌控命运。

**情绪归因**

情绪归因分为内归因、外归因和中立归因。了解情绪的来源，有助于更客观地看待问题。

**情绪分类与沟通**

情绪分类包括初始反应（原生情绪）和后续反应（次生情绪）。有效沟通应表达原生情绪，避免次生情绪。

**情绪疗法**

1. **ABC理论**：通过调整信念，快速冷静。
2. **情绪支付理论**：在关系中保持正面情绪，避免负面情绪。
3. **情绪记录**：了解自己，取悦自己。

**作业**

1. 分析负面情绪事件，包括事件、信念、结果、原生情绪和次生情绪。
2. 制作情绪记录表，了解自己的情绪变化。

**总结**

本节课我们学习了情绪管理的重要性、情绪的分类、情商的三个等级、情绪归因、情绪分类与沟通、情绪疗法等内容。希望每位女性都能享受高质量的情感关系。
