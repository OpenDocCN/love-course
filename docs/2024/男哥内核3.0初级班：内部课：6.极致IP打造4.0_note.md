# 男哥内核3.0初级班：内部课：6.极致IP打造4.0

## 概述
在本节课中，我们将学习如何打造极致IP的4.0版本，包括选景、拍摄技巧、修图方法等。

## 选景
### 选景原则
- 明确拍摄主题，例如拍摄咖啡场景。
- 利用美团、小红书等平台搜索网红打卡地。
- 学习他人的拍摄姿势和角度。

### 搜索与选择
- 打开小红书，搜索相关主题，如“北京下午茶网红打卡”。
- 选择合适的场景，注意场景与服装的搭配。

## 拍摄技巧
### 自拍
- 利用手机录屏模式练习拍照姿势。
- 展现最帅的一面，如正脸、小心机等。
- 拍摄日常照片，避免过度刻意。

### 他拍
- 了解自己适合的角度，如侧脸、45度角等。
- 注意拍摄场景与服装的搭配。
- 避免做作的动作，如看手表、玩手机等。

## 修图方法
### 软件选择
- 使用醒图等修图软件。

### 修图步骤
- 裁剪、矫正图片。
- 调整亮度、对比度、饱和度等。
- 修饰脸型、眼睛、鼻子、嘴巴等。
- 调整滤镜、色彩等。

## 总结
本节课中，我们学习了如何打造极致IP的4.0版本，包括选景、拍摄技巧、修图方法等。希望同学们能够将所学知识应用到实际操作中，提升自己的形象和影响力。
