## 一、场景选择与前期准备 🏙️

上一节我们介绍了个人形象的基础要素，如服装、发型等。本节中，我们来看看如何为拍摄选择合适的场景。

选择场景前，首先要明确你想拍摄的主题。例如，如果你想拍摄符合人设的喝咖啡场景，就需要寻找合适的咖啡厅。

以下是寻找和评估场景的具体步骤：

1.  **使用平台搜索**：打开美团或小红书，搜索关键词，如“北京下午茶网红打卡”。小红书通常有用户分享的打卡照片、姿势和构图，可供参考。
2.  **参考与模仿**：点开感兴趣的帖子，观察模特的姿势、拍摄角度和位置。你可以直接学习并模仿这些姿势和角度进行拍摄。
3.  **留意场景信息**：帖子下方通常配有文字说明，包括营业时间、具体位置等，这些信息非常重要。
4.  **服装与场景搭配**：选择的服装必须与场景氛围协调。例如，在高端下午茶场景穿着随意鲜艳的服装会显得格格不入。如果不确定如何搭配，可以联系课程客服或在群内提问。

关于服装，需要再次强调品牌辨识度的重要性。许多具有影响力的公众人物都会选择有品牌效应的服装。课程推荐的服装均经过严格把关，确保版型、颜色和质量，并提供售后保障，你可以放心搭配和购买。

## 二、自拍技巧与参数设置 🤳

了解了如何选景后，我们来看看如何进行拍摄。首先从自拍开始。

自拍的核心在于找到自己好看的角度，并让动作看起来自然。以下是自拍时需要注意的要点：

*   **避免错误姿势**：避免手机角度过高或过低等不自然的拍摄方式。
*   **利用构图线**：在相机设置中打开“网格”或“井字构图”功能。将人物主体放在网格线的交叉点或三分之一处，而不是正中间，这样构图会更美观。
*   **注意肢体与空间**：拍摄时，头部上方和侧面应留出一些空间。手部动作要自然展现，不要刻意遮挡。
*   **眼神与表情**：眼神可以目视斜前方45度，这样会使眼神更有神采。表情应放松自然。
*   **日常感与写真感的区别**：朋友圈照片应体现生活日常感，避免过于刻意的“写真”式摆拍。可以尝试一些自然的动作，如思考、玩手机等。

为了获得更好的自拍效果，推荐使用“轻颜相机”等App，并参考以下参数进行设置（可根据个人情况微调）：

```plaintext
美颜强度：57
磨皮效果：适中（参考竖线位置稍退后）
均匀肤色：适中
肤色选择：白皙（第一个选项）
清晰度：调高一些
瘦脸效果：根据脸型开启（圆脸用圆脸专属，长脸用长脸模式，自然脸型用自然模式）
小头功能：轻微调整，缩小头部比例
隆鼻/鼻翼/鼻梁：根据需求微调，使鼻子更精致
眼睛大小/瞳孔大小：适度调整，以自然有神为准
```

课程结束后，请完成每日练习作业：**每天拍摄正脸、左侧脸、右侧脸各10张照片，从中选出每个角度最满意的一张，发到课程群中**。通过持续练习，你将更快找到自己最佳的上镜角度。

## 三、他拍技巧与角度示范 📷

当你通过自拍练习熟悉了自己的最佳角度后，他拍就会更有方向。

他拍的关键在于与拍摄者沟通，并利用环境营造氛围。以下是他拍的要点：

*   **明确告知角度**：提前告诉拍摄者你哪个侧脸或角度更好看，指导对方进行拍摄。
*   **利用环境叙事**：他拍照片应能交代“谁、在哪儿、干什么”。例如，在办公室场景，可以结合电脑、咖啡杯等道具，展现工作状态。
*   **尝试多角度**：
    *   **平拍**：相机与脸部平行，是最常见的角度。
    *   **俯拍**：从稍高角度拍摄，可以带来不同的视觉效果。
    *   **仰拍**：从低角度拍摄，容易拍出大长腿效果，但需注意脸部变形。
*   **服装与场景强化人设**：服装应与场景共同塑造你想展现的形象。例如，在独立办公室，西装或夹克可以塑造成功人士形象；整洁的衬衫则能体现阳光、干练的气质。
*   **注意细节**：拍摄前检查服装是否平整，配饰（如手表、袖扣）是否到位，这些小细节能极大提升精致感。

## 四、后期修图教程 🖌️

拍摄完成后，后期修图是提升照片质感的最后一步。我们将使用“醒图”App进行演示。

修图的总体思路是：**先调整图片整体色调和构图，再精细修饰人像**。

以下是修图的核心步骤：

1.  **基础调整（调色与构图）**：
    *   首先调整**亮度**、**对比度**、**饱和度**和**氛围**，让照片色彩更鲜明、有层次。
    *   利用**高光**和**阴影**工具，提亮暗部，压暗过亮的部分。
    *   根据照片感觉，微调**色调**（偏暖或偏冷）。
    *   使用**裁剪**和**矫正**工具，修正构图，让画面更平衡。

2.  **人像精修**：
    *   使用 **“一键美颜”** 或 **“原生皮”** 功能，初步改善肤色、去除明显瑕疵。
    *   进入 **“面部重塑”** 功能，这是修脸的关键：
        *   **小头**/**瘦脸**：适度调整头部大小和脸部轮廓。
        *   **太阳穴**/**颧骨**/**下颚**：让脸型线条更流畅。
        *   **眼睛**：调整大小、眼距、瞳孔大小，让眼睛更有神。
        *   **鼻子**：精细调整鼻翼、鼻梁、山根、鼻尖，让鼻子更挺拔精致。
        *   **嘴巴**：调整嘴唇厚度、宽度，可添加一点微笑。
        *   **眉毛**：加粗或调整眉形，增加精神气。
    *   使用 **“妆容笔”** 工具，为嘴唇添加自然唇色。
    *   使用 **“瘦身瘦腿”** 工具，**主要拉长小腿**以优化身体比例，注意不要过度导致变形。
    *   最后，可以添加一个合适的**滤镜**（如“冷白皮”、“摩登”等），统一并提升照片风格。

**修图核心原则**：所有调整都应遵循“自然”原则，避免修图痕迹过重。修图前后务必对比，确保背景线条没有因推拉脸部而变形。
