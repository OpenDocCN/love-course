# 社交情商训练营：02_1-2 感知自我情绪_ev

## 概述
在本节课中，我们将学习如何感知和管理工作中的情绪，这是提升情商的关键一步。

## 自我情绪管理的重要性
**公式**： 情绪管理 = 认知情绪 + 控制情绪

一个人最大的敌人是他自己。战胜自我，才能成为优秀的人。情绪如快乐、生气、悲伤、恐惧等，若无法控制，会导致后悔和惭愧。

## 认知情绪
**代码**： `情绪认知 = 观察情绪 + 分析情绪`

认知情绪是管理情绪的第一步。从小事做起，观察并分析自己的情绪。例如，开心、生气等。

## 控制情绪
**代码**： `情绪控制 = 小情绪控制 + 大情绪控制`

控制情绪分为小情绪控制和大情绪控制。小情绪如轻微生气或悲伤，可以通过自我暗示来控制。大情绪则需要更深入的锻炼。

## 情绪爆发与智商
情绪爆发时，智商为零。此时所说所做，未经大脑思考，结果往往不好。

## 管理情绪的原理
**公式**： 管理情绪 = 防止情绪扩大 + 防止情绪爆发

管理情绪的原理是，在小情绪出现时，就要控制它，防止其扩大和爆发。

## 总结
本节课中，我们一起学习了如何感知和管理工作中的情绪。通过认知情绪、控制情绪，我们可以更好地管理自己的情绪，避免后悔和惭愧。
