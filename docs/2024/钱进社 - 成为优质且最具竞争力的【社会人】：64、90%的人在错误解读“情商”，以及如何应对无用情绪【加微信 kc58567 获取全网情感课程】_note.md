# 钱进社 - 成为优质且最具竞争力的【社会人】：64、90%的人在错误解读“情商”，以及如何应对无用情绪

## 概述

在本节课中，我们将探讨90%的人对情商的错误解读，以及如何应对无用的情绪。

## 1. 情商的真正含义

**公式**：情商 = 情绪识别 + 情绪利用

情商是一种能力，它使我们能够识别自己和他人的情绪，并利用这些情绪信息来指导决策和行为。

## 2. 情绪与态度的区别

**代码**：情绪 = 本质；态度 = 表象

情绪是本质，态度是表象。态度可以理解为刻意塑造过的情绪，即利用情绪来传递你想传递的东西。

## 3. 情商的核心要素

以下是情商的四个核心要素：

1. **自我意识**：了解自己的情绪和行为。
2. **控制情绪**：管理自己的情绪，使其不会影响决策和行为。
3. **自我鼓励**：鼓励自己面对挑战和困难。
4. **认知他人的情绪**：理解他人的情绪，并据此调整自己的行为。

## 4. 提升情商的方法

### 4.1 认识自己的情绪

1. **认可情绪的客观存在**：不要逃避或压抑情绪，而是直面它们。
2. **避免暴力式的压抑情绪**：寻找良性的出口来释放情绪。

### 4.2 觉察情绪

1. **区分情绪和行为**：情绪可以驱动行为，但行为也可以反过来引领情绪。
2. **利用行为来引领情绪**：通过改变行为来调整情绪。

### 4.3 装备自己

1. **找到自己的道具**：找到可以增强自信和气势的物品或行为。
2. **利用道具来强化行为**：通过行为来引领情绪。

## 总结

本节课中，我们探讨了情商的真正含义、情绪与态度的区别、情商的核心要素以及提升情商的方法。希望这些内容能够帮助你更好地理解情商，并提升自己的情商水平。
