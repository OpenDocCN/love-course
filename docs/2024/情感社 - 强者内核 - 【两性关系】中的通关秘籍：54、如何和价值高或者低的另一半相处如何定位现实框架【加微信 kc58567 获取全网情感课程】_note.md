# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：54、如何和价值高或者低的另一半相处如何定位现实框架

## 概述

在本节课中，我们将探讨如何与价值高或低的另一半相处，并了解如何定位现实框架下的价值。

## 世俗价值与主观价值

**公式**：世俗价值 ≠ 主观价值

**代码**：世俗价值 = 具象指标，主观价值 = 评判者个人决定的价值

以下是关于世俗价值和主观价值的详细解释：

* **世俗价值**：指具象的指标，评判一个人在社会中的标准估值，如年龄、收入、职位等。
* **主观价值**：指评判者个人决定的价值，包括物质、安全感、情绪价值、体验等。

## 价值波动与认知

**公式**：价值波动 = 认知偏差

**代码**：价值波动 = 评判者个人决定的价值波动

以下是关于价值波动和认知的详细解释：

* 价值波动是由于评判者个人认知偏差导致的。
* 价值波动会影响亲密关系的走向。

## 如何与价值低的另一半相处

* **明确价值定位**：认识到价值高低只对评判者本人有意义。
* **注重成长性价值**：通过学习、提升认知等方式缩小与对方的差距。
* **尊重对方**：避免因为价值不对等而产生控制或打压行为。

## 如何与价值高的另一半相处

* **保持平衡**：避免因为对方价值高而感到不平衡。
* **承担更多家庭责任**：通过承担更多家庭责任来弥补差距。
* **注重个人成长**：通过学习、提升认知等方式提升自身价值。

## 总结

本节课中，我们学习了如何与价值高或低的另一半相处，并了解了如何定位现实框架下的价值。通过明确价值定位、注重成长性价值、尊重对方等方式，我们可以建立更加稳固的亲密关系。
