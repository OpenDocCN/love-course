# 乐嘉性格色彩读心术（完结）第47课：如何过好面试这一关（红色篇&蓝色篇）🎤

在本节课中，我们将要学习如何快速判断面试官的性格色彩，并掌握与不同性格面试官有效沟通的技巧。我们将重点分析红色与蓝色性格面试官的特点，并提供针对性的应对策略。

---

面试官通常都受过专业训练，因此无论其性格如何，在行为上都会表现出一定的共性。在洞察面试官性格时，切忌仅凭单一细节贸然下结论。然而，如果你能与面试官交谈超过15分钟，就有机会从其言谈细节中捕捉到性格特质。

由于时间关系，本讲将详细讲解红色与蓝色性格面试官的特点及沟通方法。下一讲中，我们会继续分析黄色与绿色性格的面试官。

## 第一类：红色性格面试官 😊

红色性格的面试官通常亲和力较强，交流时不会让人感到太多距离感。他们思维跳跃，富有创意，在面试中可能会提出一些看似刁钻古怪的问题。

例如，他们可能突然指着墙上的画问你：“看了这幅画你有什么感受？” 或者问：“你能估算全世界每年消耗多少张卫生纸吗？” 如果你的性格比较严谨，且对红色性格了解不多，遇到这类突发问题可能会感到发懵甚至憋屈。

上一节我们介绍了红色面试官的基本特点，本节中我们来看看与他们沟通时需要注意的三个核心要点。

以下是应对红色性格面试官的三件最重要的事情：

1.  **表达热忱的工作愿望**：红色性格非常看重他人的态度。相比能力出众但态度傲慢的应聘者，你的诚恳和热情能获得巨大加分。这满足了他们内心深处希望帮助他人并获得认可的欲望。
2.  **用故事或具体例子展示能力**：在陈述过往成绩和能力时，避免抽象叙述，要善于讲故事。例如，当被问及“如何处理客户投诉”时，不要只回答步骤，而是讲述一个具体的成功案例，让对方感性地认识到你的责任心与沟通技巧。
3.  **巧妙表达认可**：认可是影响红色性格的有效方法，但需避免显得虚假。较好的做法是表达对公司和行业的认同，说明自己的工作动力来源。面试后，可以发送一封热情的补充邮件，并对面试官的专业度与亲和力表示认可。

## 第二类：蓝色性格面试官 😡

蓝色性格的面试官逻辑极其严密，善于抽丝剥茧地盘问每一个细节。他们发自内心地认为“一屋不扫，何以扫天下”，因此非常注重应聘者表现的条理性和精确性。

例如，有应聘者因提到“推崇巴菲特的价值投资理念”，而被蓝色面试官连续追问关于巴菲特投资细节的多个问题，场面十分狼狈。另一位应聘者自称“概率论学得不错”，结果被当场要求写下**伯努利方程**和**贝叶斯公式**，最终面试失败。

了解了蓝色面试官追求严谨的特点后，我们来看看如何做好充分准备并应对他们的考察。

以下是应对蓝色性格面试官的三件非常重要的事情：

1.  **做好充分准备，引导至熟悉领域**：准备简历和各种材料时务必精确细致，能提供数字的地方尽量提供。针对每一项成绩和经历，准备好可能被追问的证明材料。在面试中，递上一份翔实的材料比千言万语更有力。
2.  **有条理、沉稳地回答问题**：听清问题后想好再说，尽量采用“第一、第二、第三”或“因为…所以…”等逻辑结构进行回应。即使受到质疑，也要保持情绪稳定，与对方在同一频道对话。
3.  **只谈确定之事，切忌信口开河**：对于不确定的事情，宁可坦诚承认不懂。蓝色面试官特别看重诚信，信口开河会引发他们对你的诚信产生严肃质疑。

本节课中我们一起学习了如何识别并应对红色与蓝色性格的面试官。面对红色面试官，关键在于展现热情、讲述故事并巧妙认可；面对蓝色面试官，核心在于准备充分、回答有条理且诚实严谨。

需要记住，面试是双向选择的过程，你同样可以通过面试了解对方，判断这份工作是否真正适合自己。
