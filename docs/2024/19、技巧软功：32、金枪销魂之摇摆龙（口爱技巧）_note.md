# 课程19：技巧软功之三十二——金枪销魂摇摆龙 🐉👄

在本节课中，我们将学习一项名为“摇摆龙”的口爱技巧。这项技巧的核心在于利用舌头的柔软特性，通过左右摇摆的动作来刺激敏感部位，从而提升体验。我们将分步解析其动作要领与练习方法。

## 动作原理与核心概念

上一节我们介绍了口爱的基本概念，本节中我们来看看“摇摆龙”的具体实现方式。其核心是舌头的**左右横向摆动**，而非旋转。

这个技巧的力学原理可以简化为一个往复运动。假设舌头中点为原点，其左右摆动的幅度为 **A**，频率为 **f**。那么龟头在水平方向上受到的力可以类比为一种简谐运动的刺激模式。

## 分步动作详解

理解了核心是左右摆动后，以下是具体的执行步骤，请按顺序练习。

1.  **基础含入**：将目标部位含入口中，保持嘴唇放松。
2.  **舌头定位**：将舌头置于龟头下方或环绕其底部，作为支撑和摆动的基础点。
3.  **发起摆动**：利用舌头的柔软性，开始做**左右方向的横向摆动**。动作代码示意如下：
    ```python
    while engaging:
        tongue.position = “left”  # 舌头摆向左
        tongue.position = “center” # 经过中点
        tongue.position = “right” # 舌头摆向右
        # 循环往复，形成摇摆
    ```
4.  **控制力度与节奏**：摆动应保持轻柔且富有节奏感，让龟头在口腔内自然弹动。
5.  **持续与变化**：保持摆动动作的持续性，并可根据反应轻微调整摆动的幅度与速度。

## 练习要点与常见误区

在掌握了基本步骤后，以下是练习时需要注意的关键点和应避免的错误。

*   **重点**：确保是纯粹的左右摇摆，动作轨迹呈“之”字形。
*   **重点**：利用舌头的柔软性来缓冲和传递动作，而非僵硬地用力。
*   **误区**：将“左右摇摆”与“旋转”混淆。摇摆是横向运动，旋转是圆周运动。
*   **建议**：初次练习时可放慢速度，专注于动作的准确性和节奏感。
