# 加一情感《宅男宝典：02、聊天解码：16.聊天中的语言情绪调动🎶

嗨你好，欢迎来到勤圣方法，我是班长alen。本节课我们将探讨聊天中的语言情绪。首先，解释一下什么是语言情绪，即那些自带情绪的话语。心理学上，我们在与人沟通时，语言占7%，语气占38%，肢体语言占55%。由于我们讨论的是微信聊天，是纯文字，因此不必过多考虑肢体语言。

本节课中，如果我们仅通过微信聊天，那么语言就占了20%，语气占比38%。这表明在微信聊天中，语气非常重要。语气用得好，情绪自然流露。给女生发信息时，更多的是情绪的沟通，而非直接的意思。许多兄弟在与女生聊天时，话里完全没有情绪。

例如，在开场介绍自己时，可以说：“你好，我是羞涩的徐动。”这样的表达会让女生感受到一种俏皮的、轻松的有趣的聊天氛围。而如果只是简单地说：“我叫徐动，你呢？”则缺乏情绪的力量。这就是我们所说的没有情绪的对话。女生本身就是情绪的动力，语言情绪到位了，什么都好说。

那么，如何学会这种语言情绪呢？或者说女生喜欢怎样的语言情绪？这就是今天要解决的问题。我总结了以下4种女生喜欢的语言情绪：

1. **日常用语**：日常用语与正式用语或书面语相对。例如，表达愤怒时可以说“我靠”或“操”，称呼对方可以叫“猪头”，使用方言等。这样的对话不仅情绪丰富，还能拉近彼此的距离。

2. **网络用语**：网络用语可以让聊天充满情绪。例如，称呼对方为“老铁”、“老铁没毛病”，用“蹦迪喝酒”代替“喝酒”，用“很佩奇”表示佩服等。但要注意不要使用过时的网络用语。

3. **非逻辑语言**：女生不跟你讲逻辑。例如，调侃对方时可以说：“你看你洗澡和我不穿衣服的样子有毛线关系？”这样的对话可以引发女生的好奇和反驳，从而调动情绪。

4. **表情**：现在手机聊天中，表情非常重要。动态表情可以不用文字就能传递情绪。例如，用贱贱的表情来表达情绪，或者使用表情包来拉近彼此的距离。

总结：在与女生微信聊天中，只有20%是文字本身的意思，剩下80%都是情绪的传递。在情绪聊天模式中，主要通过四种方式传递聊天中的情绪：日常用语、网络用语、非逻辑语言和表情。
