## 🖼️ Snapseed 基础修图案例实操

上一节我们介绍了苹果自带的编辑工具，本节中我们来看看功能更强大的第三方修图软件 Snapseed。我们将通过一个具体的草原风光照片案例，演示如何利用 Snapseed 进行从修复瑕疵到色调调整的全流程处理。

首先，在苹果手机上打开 Snapseed 软件，并从相册中导入一张待处理的照片。本例中的照片拍摄于草原，包含马匹、人物、天空和远山。照片意境不错，但背景中存在多余的电线杆和其他马匹，影响了画面的简洁感。后期调色的方向是使画面更加明亮、对比更强烈、整体更通透。

### 第一步：修复画面多余元素

以下是使用“修复”工具去除杂物的步骤：

1.  点击底部工具栏中的“工具”。
2.  在工具列表中找到并选择“修复”工具。
3.  进入修复界面后，用双指放大照片，以便精确操作。
4.  用手指直接涂抹需要去除的电线杆，软件会自动将其清除。对画面中其他多余的马匹也进行同样操作。
5.  在涂抹时需仔细，可放大到最大视图。如果一次修复效果不自然，可多次涂抹直至画面融合顺畅。

修复完成后，照片构图变得更加简洁，意境得以突出。

### 第二步：使用滤镜确立色调基调

接下来进行调色。建议先使用一个滤镜来快速确立整体的色调方向。

1.  在工具栏中找到“粗粒胶片”滤镜并点击进入。
2.  在滤镜列表中，推荐使用 **X01 至 X05** 这最后五项。
3.  逐一尝试，选择最符合个人审美倾向的滤镜。本例中选用的是 **X02**。
4.  选定滤镜后，在屏幕上**上下滑动**可以调整两个参数：
    *   **“粒度”**：为画面添加颗粒感。建议通常将其调整为 **0**，以保持画面干净。
    *   **“样式强度”**：控制滤镜效果的强弱。根据画面效果调整到一个合适的数值，例如 **70-80** 之间。

通过这一步，我们为照片设定了一个基础的色调风格。

### 第三步：调整基础参数

滤镜应用后，我们再进行精细化的参数微调。

1.  点击工具栏中的第一项“调整图片”。
2.  在屏幕上**上下滑动**，可依次调整七个参数：
    *   **亮度**：本例照片亮度适中，可暂不调整。
    *   **对比度**：适当增加（例如+20），以强化明暗对比。
    *   **饱和度**：轻微增加（例如+10），让色彩更饱满，但切忌过度。
    *   **氛围**：相当于“鲜明度”，可轻微增加（例如+15），提升画面通透感。
    *   **高光**/**阴影**：分别控制亮部和暗部细节。本例中天空未过曝，山坡阴影有细节，故可暂不调整。
    *   **暖色调**：相当于色温。可稍向右滑动（例如+5），为画面增添一丝暖意。

### 第四步：应用曲线工具增强影调

曲线工具可以更精细地控制画面的影调关系。

1.  点击工具栏中的“曲线”工具。
2.  点击底部第三个按钮，会弹出预设曲线效果。
3.  对于初学者，建议直接套用预设。推荐尝试“柔和对比”效果。
4.  若想微调，可拖动曲线上的控制点：**左上方的点控制暗部**，可稍向下拉以压暗；**右下方的点控制亮部**，可稍向上提以增亮。这便构成了一个简单的“S”型曲线，能增强画面对比又不失自然。

经过以上四步调整，照片的后期处理基本完成。长按屏幕可以对比原图：原图色彩平淡、对比不足、杂物较多。调整后，画面变得简洁、通透，色调更具吸引力。

**总结Snapseed修图思路**：先分析原片问题（如杂物、影调、色彩），然后使用修复工具清理画面，接着用滤镜快速定调，最后通过调整图片和曲线等工具进行精细化微调。

## 🎞️ 剪映 基础视频剪辑案例实操

上一节我们完成了照片的修图，本节中我们来看看如何使用剪映软件对视频进行基础的剪辑与合成。我们将学习如何导入素材、添加背景音乐、调整视频速度、添加文字等核心操作。

首先，在苹果手机上打开剪映APP，点击“开始创作”，从手机相册中选择需要剪辑的视频素材导入。

### 第一步：添加与调整背景音乐

视频素材导入后，时间线上会显示视频片段。接下来为其添加合适的背景音乐。

1.  点击底部工具栏中的“添加音频”，然后选择“音乐”。
2.  在剪映自带的音乐库中试听并选择一首符合视频氛围的音乐。
3.  音乐添加后，时间线下方的绿色条即为音频轨道。**长按**音频条，将其拖动至与视频开头对齐，确保整个视频都有配乐。
4.  由于我们后续会调整视频速度，视频总长度会变化。处理完后，将播放指针（白色竖线）移动到视频结束的位置。
5.  选中音频轨道，点击底部“分割”，将多余的音频部分切分开，然后选中后半部分并“删除”。

为了让音乐开始和结束更自然，可以添加淡入淡出效果：
*   选中音频轨道，点击“淡化”。
*   设置“淡入时长”和“淡出时长”（例如各设置为1秒），使音乐音量平缓变化。

### 第二步：调整视频速度与排序

如果某些视频片段节奏过慢，可以通过变速来调整。

1.  选中需要加速的视频片段。
2.  点击底部工具栏中的“变速”，选择“常规变速”。
3.  将速度滑块向右拖动，例如调整为 **2.0倍**（即两倍速）。此操作特别适用于让延时摄影的动态感更强。
4.  对多个片段进行变速后，视频总长度会缩短。

若需要调整视频片段的播放顺序：
*   在时间线上，**长按**某个视频片段，即可将其自由拖动到其他位置。

### 第三步：为视频添加文字标题

为视频开头添加一个文字标题，能有效提升观感。

1.  将播放指针移动到视频开头。
2.  点击底部工具栏中的“文本”，然后选择“新建文本”。
3.  输入标题文字，例如“手机延时摄影作品”。
4.  在预览窗口中，可用**双指缩放**调整文字大小，**单指拖动**调整文字位置。
5.  点击“样式”，可以更改字体、颜色、添加描边或背景（“花字”和“气泡”选项）。
6.  点击“动画”，可以为文字设置“入场动画”（如“收拢”）和“出场动画”（如“弹出”），并调节动画时长。
7.  在时间线上，拖动文字轨道两端的白框，可以控制文字在视频中显示的时长。

完成上述基础编辑后，可以点击播放按钮预览整体效果。一个包含背景音乐、变速处理和文字标题的简单视频就剪辑完成了。

**总结剪映剪辑流程**：导入视频素材 -> 添加并修剪背景音乐 -> 对视频进行分割、变速、排序 -> 添加文字、转场等效果 -> 导出成品。

对于更复杂的操作（如关键帧、曲线变速、音效等），剪映提供了丰富的工具，大家可以通过实践或观看进阶教程进一步学习。
