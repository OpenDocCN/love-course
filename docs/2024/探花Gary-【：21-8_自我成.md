# 探花Gary-【：21-8_自我成

在本节课中，我们将要学习一项核心技能：自我洗脑。我们将探讨如何通过重新设定大脑的思维模式，来获得长期、稳定的自我提升，从而打破枯燥、无趣且缺乏成果的生活循环。

## 大脑的流程化与舒适圈

上一节我们介绍了自我提升的总体目标，本节中我们来看看阻碍我们前进的第一个障碍：大脑的固有模式。

我们活在这个世界上，是由大脑控制着我们的日常生活和时刻需要做出的选择。但大脑也有其自身的习惯，这些习惯形成了我们固定的思维模式。

这个思维模式决定了我们如何思考事情、从什么角度看待问题、脑海里容易产生什么样的想法，并最终导致我们容易做出什么样的选择。然而，大多数人完全没有控制自己大脑意识的能力。

你会发现，在每一天百分之八九十的时间里，你的日常行为和选择都是流程化的。你的大脑不需要任何思考，也不愿意去思考，全部都是你的意识根据形成的习惯在自动反应。

以下是日常流程化行为的例子：

*   从早上起来洗漱。
*   决定吃什么。
*   决定去哪里、做什么。
*   决定和谁说话。

举个例子，我以前常去一个汉堡店，每次都会点一个汉堡，并要求放西红柿和酸黄瓜。而每一次，做汉堡的人都会把西红柿和酸黄瓜放进去。反反复复，这就形成了他的日常流程习惯，也是他对我的条件反射。

这就是一个人的大脑日常的流程化程式。每天到了特定场景后，他就失去了独立思考的能力，只能执行流程化的操作。大部分人其实很少需要启动大脑，去思考日常流程以外的东西。

久而久之，你的大脑不习惯，也不愿意去引入任何超出日常的新思维和新东西。这就是每个人给自己精神上设置的限制，也可以理解为是我们的舒适圈。

任何舒适圈以外的东西，我们的大脑都会不适应。我们的内心会对任何习惯性的改变产生排斥的情绪。同时，你的大脑会给你找无数的借口，就是为了不让你打破原来的日常流程。

比如，每当你跟身边的朋友、家人、同事或同学提起任何超出他们认知以外，或超出他们舒适圈以外的新东西时，几乎每一次他们的第一反应都是排斥。然后你会看到他们会找任何借口或负面点来否定你提出的新东西。

这就是最好的证据和证明。他们的大脑在排斥超出其设定局限以外的东西。这个大脑的固化设定，导致他们的选择性焦点只会放在否定新事件的借口和负面点上，而不会深入思考新事物。

## 固化思维的阻力与改变困境

了解了大脑的舒适圈后，我们来看看它是如何具体阻碍我们改变的。

我们从小到大，还不停地被社会洗脑。每个人都形成了一种固化的思维模式和同样的想法。哪怕你自己从来没有通过一手经验验证过什么，但因为身边所有人都这么想，所以你坚信这样的一定是对的。

我们大脑的这个流程化固定设置，是导致我们在试图改变自己、让自己提升的道路上无法坚持前进的最大一个原因。

这就是为什么你会发现，学了很多内容，看了很多课程，但在现实生活里，你的行动、你的日常、你的结果没有丝毫改变，或者改变不大的核心原因。

来举一个具体的例子，比如外出约会、每天健身或是定期外出社交。放眼整个社会，我刚才讲的这些事情，基本上都超出了绝大多数人的日常生活流程和他们大脑愿意接受的舒适圈范围。

每当你尝试去打破并改变当下这个枯燥、无趣、没有结果的生活流程时，你的大脑就会开始劝退你自己。各种借口、各种自我破坏、各种低能的想法会马上引入到你的脑海中。

比如你想出去认识一个新的人，或者你想出去社交认识一些新朋友，这些都不是你日常生活流程中的东西。那你前期去执行的时候，遇到的阻力会非常大，因为你的大脑在抗拒这些新东西。

你的大脑不想思考，它很懒惰，它在劝退你。你的大脑会跟你说：“算了，太难了，别浪费时间，还不如在家打游戏多舒服”，或者“我不够好看、不够有钱、不够好，谁会搭理我”，“我要工作，我太忙了，我没时间”等等。

这就是为什么99%的人没有任何结果，并且还浑然不觉。他们自愿接受大脑负面的洗脑，屈服在找借口的思维陷阱里面。我曾经也是这样。

学生时代的我了解到了社交技巧，一开始我自己出门实战，用行动试图改变我自己的时候，阻力是非常大的。在长期关系里，我要改变自己的讨好和低自尊，遇到的阻力也非常大。

## 破解之道：认识高我与小我

面对大脑的顽固抵抗，我们该如何应对？唯一破解自己大脑固化思维和打破借口陷阱的方法，就是重新给自己的大脑洗脑，重新设置你大脑的模式和习惯。

只有这样才能让你的日常行为和流程改变，才能让你持续坚持做能获得结果的事。当你试图去改变一些当下生活流程，用新的可能给你带来结果的事情（比如日常外出社交或者健身）时，你的大脑肯定会反抗，给你制造无数的借口。

那这个时候，我们要如何让自己的大脑停止找借口？现在我们必须要意识到，我们的大脑里其实有两个自我，其中一个叫**高我**，另外一个叫**小我**。

意思就是有一个更高维度的自我和一个低维度的自我。**高我**是宏大的、伟大的、智慧的，有点像你的宏观意图。而**小我**是小气的、关注当下的，有点像你的微观意图。那么我们每个人都有高我和小我。

举一个例子，比如你的工作、你的职业。假设你是一个程序员，是某个互联网公司的软件工程师，那么你每天的工作就是在开会，然后敲代码，然后继续开会、敲代码，其实也挺无聊的。

而你对自己这个事业的看法，会根据你当下的情绪和流入大脑里的想法变动。你的**高我**会觉得：“我做这个伟大的事业，是为了改变世界，做出一份贡献。通过互联网平台，我写的代码可能改变某个功能或界面，这是为了造福用户，是体现我的价值、存在和完成我自己使命的，这是我的理想。”

但你的**小我**会觉得：“我每天工作10个小时，为了挣这个月1万或2万块钱的工资。我为了养活我自己，我要吃饭，我默默地付出。有一天可能因为表现好，我会被领导提拔，我能成为一个管理者，这样我就能开始管人了，我的工资也会涨，我能存更多的钱。”

## 正向洗脑的操作方法

理解了高我和小我的概念后，我们来学习如何运用它们进行正向的自我洗脑。

你会发现，不管是脑子里的小我还是高我，只要你把他们两个拉到同一个方向，用正面介入、让你自己前进的理由，哪怕这个理由或借口很小气，哪怕是来源于你的小我，但只要是为了你想要得到的目标去前进，那你就能持续坚持下来，并且获得结果。

那么相反，大部分人也能想很多负面的借口来劝退自己。比如你有可能会想：“我算老几，我就是个普通老百姓，还轮到我来改变这个世界吗？”“我每天都已经累得要死了，我回家打游戏的时间都不够，每个月就这么点钱都没有我朋友多，干嘛还要做这个工作呢？”

不出意外的话，你很快就会辞职或者被辞退了。你现在可以看出来，其实是你大脑的思维决定了一切。思维模式决定了你人生中所有的事情，包括学习、自我提升、事业、长期关系等等。

这就是为什么我们要给自己正向洗脑的原因。因为到最后你会赢，你会拿到结果，你会赢得人生，所以你必须要给自己洗脑。这个自我洗脑的操作，同时也是为了激发你大脑独立思考的能力，让你从日常固化流程中解脱出来。

那正向洗脑的操作，是根据我们每天的状态去激活大脑里面的高我和小我。对于大多数人来说，日常的小我可能会更多一些，因为状态低迷的比较多，没有什么激情和动力。而高我可能就会相对少了很多。

以下是具体的操作步骤：

1.  **写下目标**：你可以把自己的目标、想要的结果写出来，写具体一点。
2.  **分列原因**：在你这个目标的下面分两栏，一边是“高我”，另外一边是“小我”。
3.  **填充想法**：把你能想到的所有原因，从高我的角度和从小我的角度全部写下来。就像我刚刚给你举的例子。
4.  **接纳小我**：哪怕有些小我的想法很难看、很自私，但是你要明白，这是在低迷的状态下，能让你不偏离道路的正向洗脑，是支持你前进的思维逻辑。

为了能持续在改变的道路上前进，每当你发现自己开始找借口、想放弃、逃避或者累的时候，把你写的这张纸拿出来看一看、念一念。甚至在原来你已经写好的基础上，再添加几个当下想出来的新的高我或小我的原因。

## 实战应用示例

让我们将理论应用于实践，举一个实际提升社交和两性关系的例子。

每当你自己要出去认识新的社交对象，或者想要和伴侣严肃解决问题的时候，一旦你开始给自己找借口、劝退自己，这时候你的**高我**就可以想一想：“我每次社交、每次表达，是为了改变我的人生，为了成为我自己想要成为的样子，为了让我的关系变得更好，为了我的下一代，为了我的家庭可以更好。”

这就是你可以参考的高我的思维模式和想法。并且你可以把它写下来，反复地给自己去巩固新的想法和理念。

那你的这个**小我**也可以想想：“我每次出门是为了认识一个人，或者我的每次表达，都是为了让自己手，满足一下我当下的这个新鲜感或者表达欲。我反正也已经陷入到负面情绪里了，反正也没有别的更好玩的事情可以去做。还有就是我心情也不好，出门找个人吐槽一下，释放一下我的坏情绪等等。”

那很明显，你的这个小我虽然很小气、很自私，但是会和你的大我在一起，把你往你想去的方向推。只要我们能确保大我和小我都能在同一个方向，我们的大脑就能跳出原有的固化思维和模式，并且停止找借口把你拉回原来的那个日常流程当中。这个就是正向的自我反洗脑。

## 总结与核心要义

本节课中，我们一起学习了自我成长的核心技能——正向自我洗脑。

从出生到现在，我们每个人本来就被社会洗脑了。这些社会的洗脑和影响，打压了我们的自尊自信，限制了我们很多的人生结果。就算你发现了新的人生可能性，这个洗脑的固化思维也会让你无法开启新的模式和人生新的现实。

那这个时候，我们就只能利用自己大脑本身的机制去给自己反洗脑，从而达到自身想要的目的和结果，让自己在另外一条人生道路上、在你真正想要的改变中去坚持。而这一切，都要从掌握自己的大脑开始。

**核心公式**：
`正向结果 = （高我原因 + 小我原因） > 大脑的借口与阻力`

通过有意识地区分并联合你的“高我”（宏大、理想）和“小我”（现实、当下），将它们的目标对齐，并书面化、反复强化这些正向理由，你就能逐步覆盖大脑原有的负面自动化程序，建立起支持你持续行动和获得成果的新思维模式。
