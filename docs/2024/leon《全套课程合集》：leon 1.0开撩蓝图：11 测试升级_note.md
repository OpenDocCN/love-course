# 《leon全套课程合集》：leon 1.0开撩蓝图：11 测试升级

## 概述
在本节课中，我们将学习如何进行测试和升级，这是约会中建立舒适感、价值和吸引的关键步骤。

## 核心概念
**动力 > 阻力**：女生与你发生关系的动力来源于对你的吸引和好感，阻力则来自于她的道德束缚和价值观。

## 测试的目的
- 测试女生对你的兴趣指标。
- 测试女生的道德感。

## 观察能力
- 观察女生在微信聊天中的情绪和主动性。
- 观察约会中女生的肢体语言和情绪反应。

## 肢体测试
以下是几种肢体测试方法：
- **摸头测试**：在女生调皮或乖巧时，轻摸她的头，观察她的反应。
- **捏脸测试**：轻捏女生的脸，观察她的反应。
- **过马路搀扶**：过马路时，轻搀扶女生的腰，观察她的反应。
- **抓胳膊测试**：过马路时，不经意地抓起她的胳膊，观察她的反应。

## 言语测试
- 表达对女生的赞美。
- 主动靠近女生，并询问她的感受。
- 用眼神长时间注视女生，观察她的反应。

## 道德感测试
- 通过询问女生对某些性方面事情的看法，了解她的性观念。
- 询问女生对出轨、暧昧关系的看法。

## 总结
本节课中我们一起学习了如何进行测试和升级，通过观察和测试女生的兴趣指标和道德感，我们可以更好地把握升级的时机，从而建立更深入的关系。
