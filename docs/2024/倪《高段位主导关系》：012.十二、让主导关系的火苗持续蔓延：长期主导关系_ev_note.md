# 倪《高段位主导关系》：012.十二、让主导关系的火苗持续蔓延：长期主导关系_ev_note

在本节课中，我们将要学习如何在长期关系中维持主导地位。我们将探讨长期关系与短期关系的核心区别，并揭示一个确保关系长久稳定的关键观念。

---

上一节我们介绍了短期关系中的主导策略，本节中我们来看看如何将这种主导延续到长期关系中。长期关系的目的与短期关系截然不同。短期关系的目的是**占领关系中的高位**以确定关系。而长期关系本身已经处于高位，其目的是**让彼此关系更亲密**，享受幸福感。关系中的高位在这里只是一个手段。

因此，长期关系中“主导关系”本身并不重要，重要的是两个人越来越信任，陪伴质量越来越高，实现 **1 + 1 > 2** 的目标。这个目标非常困难。许多缺乏爱的能力的情侣会陷入互相索取与指责的恶性循环，就像两个没水的人在沙漠中叫对方给自己水一样。为了避免人性的恶在关系中滋生，我们必须从源头开始，将一切负面因素扼杀在萌芽中。

这里分享一件最重要的事情，它不是一个技巧，而是一个核心观念与心法：**变态级的筛选**。

什么是变态级的筛选？它指的是至少需要**6个月到两年**的时间对伴侣进行考察，时间越长越好。在这段筛选期内，如果发现彼此有重大不合，应随时准备分开。这与许多热烈开始、惨烈结束的关系模式相反。我主张保持积极的悲观主义，不轻易上头，不轻易承诺。就像盖高楼需要深地基一样，漫长的筛选期就是长期关系稳固的地基。

你选好了一个优质的对象，比后续任何努力都重要。不要试图在相处中去改变对方。这条法则可以概括为：**只筛选，不教育；只选择，不改变**。如果按照二八定律，我坚持把**80%的精力放在最初的选择上**，而后续的相互影响和改变只占**20%**。改变一个人太难了。

因此，要拥有一段高质量的长期关系，就要花最长的时间在筛选上。闪婚、一见钟情如同赌博开盲盒。从这个意义上说，**筛选对了，你就几乎获得了高质量的长期关系**，也可以说你主导了长期关系。

以下是**变态级筛选**中需要考察的一些重要项目：

*   **第一，闺蜜圈**。筛选她的闺蜜圈，圈子决定一个人的基本信念和价值观。
*   **第二，原生家庭**。例如她是否来自离异家庭，家庭是否幸福，这会决定她的家庭观和幸福感。
*   **第三，金钱观**。考察她对花男人钱的看法。
*   **第四，事业观**。考察她对拥有自己事业的看法。
*   **第五，是否具有成长思维**。这是非常稀缺的品质，指她是否愿意提升自己，对探索自我有兴趣。如果没有这种思维，她很容易被外界信息洗脑，随着年龄增长可能变得肤浅、焦虑且不可预测。
*   **第六，是否具备和解的能力**。考察吵架后多久能和解，是否会反思自身，当对方伸出和解的橄榄枝时是否能大方接受。这背后体现了她的教育、家庭、自尊感以及对关系的理解。

其他关于共同成长、共同目标等因素，都没有以上这些根基性的筛选条件重要。**不要冲动**，不要以任何理由进入一段长期关系，除非对方通过了你的所有暗中筛选。

这样的男人，配得上一段高质量的关系。因为第一，你具备以始为终的思维模式，考虑长远。第二，你始终相信自己不缺选择，能坚定地筛选掉不合适的目标。第三，你了解自己想要什么，并形成了自己的情感哲学。

最终，这一切还是落到了那个终极问题上：**你有没有搞清楚自己是谁，自己要什么？**

---

本节课中我们一起学习了长期关系的主导之道。核心在于理解长期关系的目的已从“占领高位”转变为“滋养亲密”。实现这一目标最关键的一步，是在关系开始前进行**变态级的筛选**，将绝大部分精力投入在选择合适的伴侣上，而非后期的改造。记住，认识你自己，明确所需，是经营一切高质量关系的起点。

最后，送给大家一句刻在希腊德尔菲神庙上的箴言：**认识你自己**。
