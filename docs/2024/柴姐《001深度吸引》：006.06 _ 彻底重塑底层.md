# 柴姐《001深度吸引》：006.06 _ 彻底重塑底层

在本节课中，我们将继续学习如何彻底重塑底层自信。上节课我们探讨了外在价值的不稳定性，并指出真正的自信源于内心稳固的高价值感。本节我们将聚焦于建立这种高价值感的具体方法，涵盖对能力、情绪、财富及人际关系的全面掌控。

---

## 改变核心认知：从追求快乐到追求骄傲 🎯

上一节我们介绍了外在价值无法成为自信的基石，本节中我们来看看如何从内部构建自信。首先，你需要改变一个根本的认知。

每天起床后，不要思考今天如何让自己快乐，而应思考今天如何让自己感到骄傲，如何让他人（如母亲、伴侣、朋友）为你感到骄傲。一次直播连线的兄弟询问如何获得动力改变懒散状态。如果每天唤醒你的是使命感和为自己骄傲的欲望，你将充满动力。你之所以缺乏动力，恰恰证明你的思维由追求快乐和舒适驱动，这与世界上99%的庸碌之人相同。

我从不刻意追求快乐，但我非常快乐。因为我的思维由骄傲驱动，我拥有使命感。完成使命后，快乐自然产生。**快乐是结果，而非目的**。建立自信，就是在“让自己骄傲”的使命感下，全面掌控自己的人生。

---

## 全面掌控人生的四个维度 🔑

要想变得自信，需要在使命感下对自己的人生进行全面的掌控。我将其分为四个维度：对能力的掌控、对情绪的掌控、对财富的掌控以及对人际关系的掌控。

### 1. 如何掌控自己的能力 💪

对自己能力的掌控是自信的基础。以下是两个核心方法。

**第一，对自己言而有信，说到做到。**

这意味着你承诺的事情必须执行，而不是内心知道自己不会去做。例如，你告诉自己今晚12点睡觉，就必须在12点躺上床，即使睡不着。而不是在深夜喝咖啡、点宵夜，然后说“算了，睡不了”。要么不承诺，承诺就必须做到。

有些粉丝惊讶于我既能玩乐又能保持优异成绩。答案很简单：我对自己言而有信。想玩？看完这页书再玩。想先玩？玩完后回来把书看完，不睡觉也要看完。

大三在台北做交换生时，周围同学都尽可能少修学分以便游玩。但我身负学业使命，必须修满学分以便回大陆专心备考。我告诉自己：玩可以，书也要看。我经常在宿舍楼下自习室通宵，保安大爷最后送我手工礼物，说我是他见过对自己最狠的女生。最终我以11门课平均绩点93.3毕业。

对自己言而有信，不意味着压制所有欲望。如果计划明早拍视频，今晚有朋友邀约喝酒，我会去。我可能喝到清晨才回家，但我一定会确保3小时后起来拍视频。如果没起来，虽然能在睡梦中舒服几小时，但醒来后会因对自己失信而难受。在我的人生里，不存在对自己失信。当我告诉自己要做某事，我就知道自己一定能做到。

**第二，通过努力实现一个目标。**

对自己言而有信是日常任务，只关乎执行。而目标则与现有能力不匹配，需要努力才可能实现。

近期我参加了基金从业资格考试。作为零基础的纯文科生，许多朋友劝我做好补考准备。了解难度后，我购买了资料并制定了严格计划：4月底到5月底每天学习2小时，6月初到考前每天学习5小时。我将所有学习环节前置规划，雷打不动。五一假期搬家时，我坐在一片狼藉的箱子中间继续学习。最终我以84和83的高分通过考试。

在我的人生里，不存在补考。只要设定目标，我就会用尽全力实现它。我有个朋友曾重200斤，通过3个月每天跑步游泳减了50斤。他说这是他做过最酷的事，此后遇到任何挫折都告诉自己：“你连减50斤都做到了，还有什么做不到？”

考试、升职、加薪、减肥等，只要你努力就一定能实现。如果没实现，只能证明你不够努力。高中时，我从年级900多名冲到平行班第一，最终录取全省只招两人的第一志愿。这靠的不是智商，而是努力。高三周末我常留校学习，别人好奇为何本地人不回家。我说回家会分心。很多人害怕被说“努力”，仿佛努力等于愚笨。但我要告诉你：**没有任何成功是轻而易举的**。所有看似毫不费力的背后，都是无法想象的精力投入。

运动员为何自信？因为他们每日刻苦训练，每一滴汗水都对应一点进步。他们掌控了自己的能力，才能更高、更快、更强。我想做的事没有做不到的，我喜欢的人没有得不到的。我不怕别人说我努力，只怕自己不够努力。我一直相信：**当你非常想要某种东西时，整个宇宙都会助你实现愿望**。

### 2. 如何增强对情绪的掌控 🧠

我们在第二章说过，情绪是助你达成目标的工具。你不能被工具控制，而需驾驭它。我们都知道七宗罪：傲慢、嫉妒、暴怒、懒惰、贪婪、暴食和色欲。增强情绪掌控，需在日常生活中严格管理这些负面情绪。

**首先，不要沉迷社交软件。**

社交软件会让你充满嫉妒、懒惰、傲慢和怨恨。互联网让我们看到不曾见过的世界：有人拥有多辆顶级跑车，有人20多岁财富自由，朋友晒出新买的手表。对比之下，我们觉得自己很失败。但我们不知道：跑车可能是租的；财富自由可能是微商噱头；手表可能是朋友攒了三年钱买的生日礼物。很多社交软件上的富足都是假装的，目的是传播嫉妒和怨恨。真正的富豪很少晒生活，因为很少人的财富是绝对干净的。能铺天盖地晒的，通常是想收割你的人。

社交软件上还有“伪正能量”内容：女生发提车、提包、提房照片，配文“努力的结果”，其实是薅男人的结果；发书籍照片配文“提升自己”，其实两页书都看不下去；发健身房自拍配文“坚持运动三年”，其实视频是PS的，身材是抽脂的。我不想诋毁任何人，只想告诉大家：社交软件上的内容很可能不真实。

社交软件还是“精神毒品”，让你分泌大量多巴胺，以致无法回到现实。精良的搞笑视频、跳舞的小姐姐让你产生虚假的操控感，多巴胺不断分泌，阈值不断提高。但当你走出房门面对真实世界时，发现无法操控任何东西，觉得一切平静无聊。你不快乐了，感到难受焦虑，于是又拿出手机。为什么外卖小哥、网约车司机常播放短视频？因为他们多巴胺上瘾了，这和吸毒一样。

我身边自信有魅力的男生，没有时刻拿手机的，聚会中也少看手机。因为他们的现实生活丰满立体，能让他们分泌多巴胺。社交媒体提供太多廉价快乐，让我们对多巴胺高度上瘾。如果你已养成一看手机的习惯，那一定散发着低能量和低价值感。

**建议**：去和真实的人与事物互动，发现每件小事的美。例如，下班等电梯时，强迫自己不看手机，尝试与同事聊天，赞美对方穿搭。走出公司，看看夕阳和云的形状。少刷手机一段时间后，你的情绪会更稳、更快乐。

**控制负面情绪的第二个点：远离色情网站。**

我不是让你从此不看，但不要过度。《雄性衰落》一书重点讨论了色情片的危害。色情片成瘾不像酒精毒品需要“更多”，它需要“不断翻新”，你必须不断加强刺激强度。

这么做的恶果：首先，降低你在现实中对女性的兴趣，难以找到女朋友。因为你在色情网站上看过“最好的”，就像刚吃完蛋糕再吃生菜，索然无味。其次，色情片让你逃避竞争，不再渴望在真实世界得到女性，变得消极，不去搭讪推进。因为虚拟世界总有“温柔港湾”。最可怕的是，看片过多可能导致实战时“不行”，造成巨大自信心打击，使你陷入看片→自卑的负循环。

此外，沉迷色情网站浪费巨大时间。每次打开六七个网页，东看西看，清理战场后已过一小时。有人说可以10分钟解决然后精力满满工作。我们承认吧，你做不到。每周浪费3小时，一年浪费144小时。这些时间本可用于读书、锻炼、提升自我，做让自己骄傲的事。

为什么打飞机后会有罪恶感？那是**对自己没有女人的愤怒、对浪费时间的愧疚和无力改变的恐惧**。

**增强情绪掌控的另一个习惯：情绪记录。**

我们每天经历很多事情，情绪像袋子，你不断往里装东西，最后一件可能让袋子爆掉，但你不能只怪最后一件。例如：早起没买到早餐，等电梯被人踩脚未获道歉，周会被老板指责，晚上约会女生迟到20分钟。这些累积导致整场约会不适。

回家后，建议你打开备忘录或Word文档，细细复盘这一天。从头梳理是哪几件事让情绪变坏，当时发生了什么，你的反应是什么。这个习惯帮我理解和梳理情绪。很多时候，当我把事情一点一点写出来，与自己对话，就能理顺情绪，意识到暴躁的原因。意识到后就不烦了，下次遇到类似场景也能更好驾驭。

我们之所以烦恼生气，是因为不能理解和掌控自己的情绪。

### 3. 如何增强对财富的掌控 💰

对财富的掌控不意味着短期挣多少钱，但你必须清楚自己的现金流和资产增值计划。

**建议习惯：制定财富目标。**

以下是具体步骤：
1.  **盘点现有资产**：清楚自己有多少钱。列出所有银行卡、微信、支付宝余额及外债，一分不能少。
2.  **计算月度支出**：清楚每月用多少钱。列出房租、网购、通勤、吃喝玩乐等所有开销。
3.  **结合收支制定目标**：制定短期和长期的财富目标。例如半年后、一年后、三年后、五年后，你希望账户里有多少钱。
4.  **规划实现动作**：思考如何实现目标。是否需要节省开支（如每天少买一杯咖啡）？是否需要努力工作拿更高年终奖或换工作？投资理财收益是否稳定？能否向金融领域朋友请教？是否需要找副业？

我2018年来上海，入职大厂年薪20万出头。从那时至今，我记录了13次财富目标。最神奇的是，每次目标都实现了。就连2018年写下的“5年后（2023年）想有多少钱”也实现了。虽然实现方式可能从职业晋升变成了做个人IP和投资，但结果殊途同归。

钱是流水，如果你不规划，它就会流走。如果你认真吸引它，它就会为你所用。最妙的是，当你盘点财富发现每次目标都实现甚至超额完成时，没有比这更爽的感觉。

### 4. 如何掌控人际关系 🤝

一定要勇于向上社交，并切断低价值人脉。

整天和失败者在一起，不会增强你的价值感，只会让你戾气加重，觉得世界没有希望。和认知高、能力强的人在一起，你在观察学习过程中会发现：其实大家都一样。他们在专业领域很强，但其他方面可能并不比你强多少。当你撕下他们的神秘面纱，发现他们也是拥有七情六欲的普通人，你自然会增加自我价值感。

很多兄弟问如何在向上社交中克服自卑。答案是：起初你一定会难受。看到闪闪发光的人，内心充满仰望和失落，这是正常的。适度的自卑是人类正常感觉，也是激励我们进步的动力。但你知道吗？越厉害的人越能发现你身上独特可贵的闪光点，这些是你身边低价值朋友发现不了或不会告诉你的。

以清酒盲品为例：我们买了从几百到几万的不同清酒。朋友们基本都猜出最贵的那只，但纷纷表示差别不大。然而后来大家都承认还是最贵的好喝。有钱人的世界是：**5%的差别，价格却是50倍**。他们会为细微差别支付高额溢价。对人也是如此，只要他们发现你身上小小的闪光点，会更愿意给你机会，并为你的优点支付高额溢价。

国外一个拥有数十辆顶级跑车的网红讲过故事：他在停车时，一个小男孩来拍照。小男孩说觉得车很酷，拍照后会email给他，如果觉得好可以转4.99美金。网红觉得小男孩聪明有商业头脑，便录用他进入团队，小男孩由此改变命运。

和低价值的人在一起，就是鸡毛蒜皮、生存挣扎和垃圾负面情绪，会让你的能量越来越低。和厉害的人在一起，你能感受到对方的思想深度和格局，会有动力，觉得人生充满各种可能性。

我们无法选择原生家庭，很多哥们还受其折磨。我的建议是：只要这个人给你负能量，阻止你变自信、过上想要的人生，就切断他，不管他是谁。

## 总结 📝

本节课中我们一起学习了彻底重塑底层自信的具体方法。我们探讨了如何通过改变核心认知，从追求快乐转向追求骄傲。接着，我们深入分析了全面掌控人生的四个维度：

1.  **对能力的掌控**：通过对自己言而有信和努力实现目标来建立可靠感。
2.  **对情绪的掌控**：通过远离社交软件与色情网站、进行情绪记录来驾驭情绪工具。
3.  **对财富的掌控**：通过制定并执行清晰的财富目标来规划财务未来。
4.  **对人际关系的掌控**：通过勇于向上社交并切断低价值人脉来优化环境。

只要你能掌控这四个方面，就能掌控自己的人生，从而由内而外地散发出高价值感。与他人相处时，你强大的自信将自然折服对方。

自信这个主题我分三节课讲解，因为它至关重要，请大家认真感悟。我很喜欢一句话：**Get comfortable with getting uncomfortable.** 因为你所有不舒适的时候，就是你进步的时候。

记得一次看近景魔术，魔术师用扑克牌做成“扑克牌雨”。当漫天落下扑克牌时，他说：“梦想的魅力，永远不是它本身，而是追逐它的过程。”我想借用这句话结束本章课程：我们很自信，不是因为我们有好工作、好身材、开好车；我们很自信，是因为我们靠自己的努力，一步一步实现它们的过程。
