# 正向吸引高阶成长课：08_6·视频美学

🎼第六节：视频美学

在本节课中，我们将学习如何通过视频放大照片之外的美。视频相较于照片，承载的能量是百倍千倍。

### 常用视频种类

以下是常用的视频种类：

- 朋友圈小视频
- 抖音、小红书等平台的短视频
- 视频通话或直播

### 视频美学基础概念

1. **画幅比例**：
   - 竖屏9比16：强调视觉冲击力，符合手机观看方式，带来沉浸式体验。
   - 横屏16比9：适合文案驱动的类型，有距离感，适合专业内容。

2. **景别切换**：
   - 全景、中景、近景、空镜头、特写、画外镜头等景别要来回切换，给人舒适的转场感受。
   - 同一景别之间切换不舒服，差别很大的景别之间切换会唐突但具有视觉冲击力。

3. **关键帧效果**：
   - 缓慢放大的画面会将注意力引导到焦点上，起到聚焦效果。

4. **慢动作**：
   - 延缓时间，增加情绪持久渲染力，提升画面高级感。

5. **暖光布景**：
   - 丰富画面，降低饱和度，增加明暗对比。

6. **音乐**：
   - 辅助氛围渲染和情绪释放，增强观感。

### 基础拍摄技巧

1. **运镜**：
   - 推进：深入人物内心，增加故事和情感感受。
   - 拉远：故事终结，关注大环境。
   - 摇晃：增加紧张氛围。
   - 横移：拍摄移动物体。
   - 上下移动：代表人物内心波澜。

2. **夹角关系**：
   - 镜头与人的夹角代表心理距离，正面夹角更能感受情绪，侧面夹角更像是旁观视角。

3. **俯仰关系**：
   - 与小孩平行镜头：进入小孩视角。
   - 仰角：拍大人。

4. **表演**：
   - 在规定情境中表达正在发生的事情，诠释人物关系，设定听众。

5. **发声和交流感**：
   - 把镜头当成说话对象，控制说话节奏停顿，让表达更自然。

6. **情绪感染力**：
   - 微笑、兴奋、紧张等感觉，打动观看者的关键。
