# 柴姐《001深度吸引》：004.04 _ 彻底重塑底层自信（上）_ev

## 概述
在本节课中，我们将探讨自信的深层构建，了解自卑的根源，并学习如何通过重塑自信来提升个人魅力。

## 自信的核心
**自信**是吸引力的核心。它像一块磁铁，吸引着周围的人。自信并非一蹴而就，而是需要经历多个阶段。

### 自信的四个阶段
1. **小学阶段**：自卑源于家庭教育和不适应新环境。
2. **高中阶段**：通过努力学习和奋斗，建立初步的自信。
3. **大学阶段**：在新的环境中，再次感受到自卑，但开始学会防御型社交。
4. **美国阶段**：通过观察多元文化，认识到内在能量的重要性，开始建立真正的自信。

### 自信的公式
自信 = 内在能量 + 自我认可

## 自卑的根源
自卑源于多种因素，以下是常见的几个原因：

### 原因一：原生家庭
- 父母的教育方式、经济状况等都会影响孩子的自信心。

### 原因二：社会性动物
- 人类天生会关注他人的评价，容易陷入比较和嫉妒。

### 原因三：单一的评价体系
- 过度关注成绩、外貌等单一维度，容易导致自卑。

### 原因四：过度聚焦自己的短板
- 过度关注自己的缺点，忽视自己的优点。

### 原因五：样本量不足
- 与少数人交往，容易受到负面反馈的影响。

### 原因六：低价值社交圈
- 长期处于低价值社交圈，容易埋没自己的优点。

## 重塑自信
要重塑自信，我们需要：

1. **了解自卑的根源**：分析自己的自卑来源，找到解决问题的方法。
2. **关注内在能量**：通过健身、学习、尝试新事物等方式提升自己的内在能量。
3. **自我认可**：认识到自己的优点，并给予自己正面的评价。
4. **扩大社交圈**：与不同的人交往，了解自己的优势。
5. **忘记社会规则**：不要过度关注他人的评价，专注于自己的成长。

## 总结
本节课中，我们学习了自信的四个阶段、自卑的根源以及如何重塑自信。希望你能通过这些知识，找到自信的力量，提升自己的魅力。
