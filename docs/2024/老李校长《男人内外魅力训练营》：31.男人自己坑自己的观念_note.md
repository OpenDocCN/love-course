# 老李校长《男人内外魅力训练营》：31.男人自己坑自己的观念

🎼男人自己坑自己的观念🎼

情感的道路并非一帆风顺，而是充满了挑战和困难。在这个世界中，人性的贪婪和邪恶无处不在。只有那些敢于面对挑战的人，才能成为真正的英雄。勇敢的男人，让我们一起揭开情感的潘多拉魔盒，洞察人性的善恶。

### 品尝人性的情与物

只有通过体验人性的情感和物质，我们才能达到人生的圆满。一个人之所以不成功，赚不到钱，并非因为他的学历或能力，而是因为他被过去的观念所束缚。要赚到钱，必须打破自己，重新塑造自己，不断突破自己。这个过程虽然痛苦，但却是必要的。

### 错误的观念和认知

在情感方面，许多男人没有意识到自己过去的认知是错误的，并且越走越远。以下列举了五个主要的错误观念：

1. **责任感**：认为男人应该对女人负责，这种观念会导致男人在情感上陷入被动。
2. **听话**：过度听话，缺乏主见，会让人认为你没有种。
3. **专一**：过早地表明自己的专一，会给对方造成压力，并增加自己的期望。
4. **不在乎形象**：忽视自己的形象，会让人认为你只注重金钱。
5. **绅士风度**：过度追求绅士风度，反而会暴露自己的需求感。

### 责任感

责任感是给那些双向奔赴的人准备的。在年轻的时候，你不需要有责任感。责任感不仅会害了你，还会害了和你交往的女人。你可以和女生发生关系，但不要因为责任感而束缚自己。

### 听话

在两性交往中，你不是奴才，你是领袖。你应该坚持自己的观点，而不是盲目顺从。

### 专一

追女人是一场消耗战，你需要集中所有的注意力和时间。过早地表明自己的专一，只会让对方感到压力。

### 不在乎形象

形象非常重要，它决定了别人对你的第一印象。不要忽视自己的形象，否则你只能依靠金钱和物质来吸引女人。

### 绅士风度

过度追求绅士风度，反而会暴露自己的需求感。真正的海王不需要绅士风度。

### 总结

从今天开始，把责任感、听话、专一、不在乎自己形象、绅士风度这些观念抛在脑后，按照正确的方式去实践。你会发现自己变得更加有魅力，更容易吸引到心仪的女人。
