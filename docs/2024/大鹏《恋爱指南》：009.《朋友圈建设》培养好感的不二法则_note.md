# 大鹏《恋爱指南》：009.《朋友圈建设》培养好感的不二法则

🎼。🎼

**概述**： 本节课将深入探讨如何通过建设朋友圈来提升个人魅力，降低聊天难度，并最终吸引心仪的女生。

**一、为什么要建设朋友圈**？

*   **降低聊天难度**： 通过展示丰富的经历和兴趣爱好，让女生主动找话题，降低聊天门槛。
*   **社交名片**： 朋友圈就像个人名片，展示你的生活品质和个性，给女生留下深刻印象。
*   **展示自我**： 通过朋友圈展示你的优秀一面，吸引更多人靠近。
*   **打造个人品牌**： 通过运营朋友圈，打造个人品牌，提升个人价值。

**二、如何建设朋友圈**？

1.  **头像**：
    *   画质高清，半身照为主，尽量露脸。
    *   背景干净大气，避免杂乱。
    *   穿搭、发型、气质到位。
2.  **个性签名**：
    *   简洁有力，凸显个性。
    *   可参考：“我不擅长甜言蜜语，连温柔都带着粗暴。”、“等你音讯全无，我就去爱世间万物。”、“品味没办法伪装，岁月都是修行。”
3.  **背景墙**：
    *   使用电影画报、风景照、喜欢的车、自己画的画等。
    *   避免使用自己的照片，避免过度自恋。
4.  **内容**：
    *   以自己的照片为主，包括帅气照片、吃喝玩乐、旅行等。
    *   少量转发文章，要求有深度或与行业相关。
    *   避免转发营销内容。
5.  **配文**：
    *   简单表达状态或感受。
    *   避免过于矫情或做作。
6.  **频率**：
    *   朋友圈建设中，每周2-3次。
    *   朋友圈建设完成后，每月1-2次。

**三、朋友圈注意事项**

*   避免营销内容。
*   避免刷屏。
*   避免掉身份的照片。
*   避免引起反感的内容或言论。
*   避免与女生亲昵的照片。
