# 49、清风老师全：13.做自己的情：2017版情感咨：168.第八课：送你一个时光机器

在本节课中，我们将探讨一个有趣的概念——“时光机器”，并学习如何将其应用于爱情与情感咨询的实践中。我们将介绍四种具体的方法，帮助你和你的来访者超越当下的困境，创造更理想的未来。

我们经常在科幻小说中听到“时光机器”这个词，它是一种能帮助人们回到过去或前往未来的装置。今天，我们将讨论爱情中存在的“时光机器”。

我相信你一定有过让自己后悔的事情，比如一句说错的话，或一件没做好的事。对这些事，我们常想：如果有个时光机可以重新选择就好了。你也常说“如果世界上有后悔药就好了”。这潜意识里是希望拥有一个时光机器，给自己重来一次的机会。如果可以重来，你或许会拥有美满的爱情，好好爱对方，绝不分手，还能做自己喜欢的工作。利用时光机器回到过去，把所有不圆满的事重做一遍，那该多好。

但时光机器真的能实现吗？未来或许有，但现在确实没有。俄罗斯理论物理学家诺维科夫在上世纪80年代提出了“诺维科夫自洽性原则”。这个原则指出，即使你能回到过去，也无法改变历史的进程，因为我们的世界就是已被改变过的最终结局。2010年，胡歌主演的电视剧《神话》中，就有穿越到过去的场景。后来有人在知乎提问：古代和现代的易小川如何共存？有人回答：只要穿越一次，就会永远在那个时间点循环，形成一种死循环。因此，等待时光机器来解决现实问题，是不靠谱的。例如，你和爱人之间有问题，难道要等时光机器出现再去改变吗？爱情等不起。

说到这里，你可能会问：清风老师，你说了半天没有时光机器，为什么还说爱情里有时光机器？前面讨论的是我们想象中的时光机器。接下来，我将告诉你几种可以实现时光机器效果的方法。

## 第一种时光机器：自证预言

自证预言是第一种爱情的时光机器。心理学家早就注意到一个现象：我们说出某段话后，甚至会不自觉地朝那个方向靠拢。例如，有的孩子小时候说长大要当警察，后来真的成了警察。有时，你在争吵中对伴侣说：“你这么不关心我，一定外面有人了。”最后对方可能真的出轨了。我们存在的世界不仅有客观存在，还有主观存在。你如何对待世界，世界就如何对待你。它甚至能证明你说出的观点是正确的。

对于爱情，你给自己一个爱情的期待和预言，把爱情想象成你最想要的样子，然后自我证明，朝这个预言努力。简单来说，你要坚信自己未来会有一份美好的爱情，那么未来你就真的会拥有。这听上去像心灵鸡汤，但在心理学中，这被称为自我暗示，是自证预言的一部分。它不断提醒你朝期待的方向前进，最后很可能实现。心理暗示有着不可抗拒的巨大力量。心理学家普拉诺夫认为，暗示能改变人的心境、兴趣、情绪等方面，从而使某些生理功能和健康状态在无意中产生变化。

这种自我暗示本质上是唤醒我们的潜意识。我们的内在有意识和潜意识。有时我们能意识到如何塑造人生，但更大程度上是潜意识在塑造。心理学家荣格从精神分析角度指出：潜意识指引着我们的人生，我们却称之为命运。当潜意识被呈现，命运就被改写了。通常，意识层面的选择我们容易感受到，所以会说“这是我做的选择”。而潜意识层面的选择，因无法清晰感受，我们会觉得是外在力量推动的结果。

例如，有的女孩常说：“我就是遇不到好男人，好男人都死完了，男人都是狼心狗肺的。”她就在实现这种自证预言，将自己与坏男人绑定。即使遇到好男人，她也容易走向极端。心理学家武志红分享过一个案例：一位女士总是遇到坏男人，离了两次婚，都是因为家暴。后来她找了一位温和顺从的男士结婚，结果不到一个月又出现家暴。咨询中发现，在争吵时，女士不断高喊：“你是不是想打我？男人没一个好东西，你打呀！”结果男士大脑一片空白，真的打了她。这就是可怕的自证预言过程。

你可能会问：这个预言有理论支撑吗？有。1968年，美国两位博士在一所中学进行智商测试，随机抽取一些学生告诉老师他们智商很高。实验结果惊人：那些被老师认为高智商的学生，成绩果然突飞猛进。其实这些学生是随机抽取的。原理很简单：老师在自证预言影响下，给这些“高智商”学生更多感情投入、学习机会和课堂反馈。学生觉得自己高智商，不断被提醒，于是更自信，学到更多知识，成绩自然提升。

在爱情中也是如此。你首先要明白自己想要什么样的爱情，自己应该是什么样子，才能得到这样的爱情。这样你就可以朝期待的方向努力，让自己变成所期待的样子。同时，你身边也会逐渐出现你所期待的人。当然，如果两人已在一起，这种自证预言需要双方共同努力，方向应是彼此都能接受的。自证预言可以帮助你去到未来，拥有更好的未来。最重要的是你要明白自己想要什么。就像给你一台时间机器，但你要清楚回到过去或去到未来，究竟想要一个什么样的未来。

如果你希望得到一份平淡而恒久的爱情，可以在早上经常对伴侣强调：“亲爱的，我们的爱情一定是平静而美好的。”通过这种不断提醒，强化彼此潜意识中对爱情的期待。

## 第二种时光机器：个体差

接下来，我们看看爱情中的第二种时光机器，它叫做“个体差”。你可能会问：什么是个体差？个体差是指每个人之间的感情存在一种时间差异。这意味着，很可能别人现在的爱情，就是你期待的未来爱情的样子。

例如，你身边有一对相爱8年的伴侣，你希望自己的感情也能像他们一样稳定甜蜜。而你刚刚开始谈恋爱，这就存在一个时间差。你可以借鉴他们经营感情的经验，应用到自己的关系中，去掉不合适的部分，这就完成了一次“时光旅行”。他们的现在就是你的未来，你可以借鉴。同时，你也可以把自己的经验告诉有类似想法的人。比如，你有一段不太顺利的感情经验，可以分享给朋友，在他们困惑时给予建议。

作为一名情感咨询师，很重要的一点是从经验出发，但不要只看到经验。这种个体差异在国家之间特别明显。例如，我们都知道美国硅谷是计算机最发达的地方。中国在努力跟上美国，两国在计算机产业上的差距就是一种时间差。把美国市场的做法拿到中国，就相当于运用了时光机器，抢先获得机会。很多成功的中国公司就是这么做的，比如美国有eBay，中国有淘宝；美国有ICQ，中国有QQ。

在感情中也是如此。当你发现某些个体有非常好的经验值得借鉴，而他所处的感情阶段又是你感情的下一个阶段时，不妨记住这种经验，让它成为你未来爱情的指导。

## 第三种时光机器：十倍速

接着，老师要介绍第三种爱情时光机器，它叫做“十倍速”。什么是十倍速？这是谷歌联合创始人拉里·佩奇在商业周刊采访中提到的一种思维方式：把一件事情做到比原来好十倍，比做到好10%更容易。为什么呢？因为要比原来好10%，通常的方法是用更多资源、更多努力去改善，但实现方法没有本质变化。

例如，两个人在一起维护爱情，你希望对方更在乎你一点，于是想改进10%，花很多时间和精力去改善。但如果你们本身存在的问题是对方最近压力很大，你不停地想改善关系，反而让他压力更大，最后关系恶化。我相信这种经验每个人都有。就像两个人经常吵架，你希望每次比上次少吵一点，但本质上还是在吵架，改进一点还是坏的。

我们有一个成语叫“矫枉过正”，意思是要通过超过原来预期的方式，来改变遇到的问题。传统观念认为改变是一点一点进行的，这没错，任何改变都是点滴积累的。但就像增高一栋房子，在原来基础上增加不可能太高，因为地基不稳。而如果把固有观念撤销，重新开始，就可能达到以前达不到的高度。我不是说让你跟对方一吵架就马上重新开始，而是从观念上重新开始。既然原来引起争吵和伤害的观念有问题，就不要只想着改善它，而要想如何从根本上让这个观念改善十倍。

因为如果你一开始的目标就是改善十倍，那过去的方法肯定没用了，你必须逼自己去做根本上的创新，打破一些基本假设。这款时光机器告诉你要尽可能不按过去的思维模式做事，因为那样最多只能逐渐改善。你想获得巨大的、美好的恋爱，就要从根本上换方法。爱情中，两人之间的牵连并不简单。要大刀阔斧地改变关系，使其达到新高度，需要双方共同努力和互相理解。但对于一些名存实亡的爱情，如果只有你一个人在坚持努力，甚至不如重新开始，这也是让事情变得好十倍的方式。

总结一下，这个时光机器的核心思想是：改善要从根本出发，从让事情变得比原来好十倍的角度去思考问题，才能创造真正的改变。

## 第四种时光机器：未来回望法（重启法）

第四款送你的时光机器叫做“未来回望法”，也叫“重启法”。重启这个动作我们很熟悉：电脑死机了，我们就重启；手机打不开了，我们也重启。未来回望法就是假设你站在未来，回头看看现在，思考未来的你会怎么做。如果未来你不会再这样做，那么现在就不要做了。

老师举一个著名的例子：1985年，英特尔公司危机四伏，其内存业务受到日本厂商的巨大冲击。英特尔的CEO安迪·格鲁夫当时想退出内存行业，进军微处理器。他问合伙人：“假如换个CEO来干，他们会怎么办？”合伙人回答：“当然是进军微处理器市场。”那一刻可能是商业史上最有智慧的时刻。安迪·格鲁夫说：“好，那为什么我们不这么干呢？”这个例子虽与爱情无直接联系，但它是未来回望法（重启法）的经典案例。

在爱情中，我们也应该用这个角度想一想。比如，我们打算做什么事，就要想一想：这么做，未来我们可能会变成什么样？未来那样是不是我们希望的？如果不是，那么我们最好不要去做。很多时候，爱情中我们是不理智的，甚至有人说爱情中的人智商等于零。我们有时只追求一时快感，让自己后悔，却没想过这些事最终产生的结果。在你犹豫不决时，不妨用这个方法重启一遍，问问自己：如果是未来的我站在这个时间点上，会怎么做？

## 课程总结

本节课中，我们一起学习了四种爱情的“时光机器”：

1.  **自证预言**：这是我们心理学中的一种自我暗示、自我强化方法。通过给自己积极的爱情期待并朝之努力，你可以创造更理想的未来。
2.  **个体差**：别人现在爱情的样子，可能是你未来所期待的样子。通过观察和借鉴他人的成功经验，你可以加速自己的感情成长。
3.  **十倍速**：改善十倍比改进百分之十往往更容易，因为它迫使你从根本上解决矛盾，进行创新性的改变。
4.  **未来回望法（重启法）**：当你犹豫不决时，站在未来的角度审视现在，思考未来的你会如何选择，从而做出更明智的决定。
