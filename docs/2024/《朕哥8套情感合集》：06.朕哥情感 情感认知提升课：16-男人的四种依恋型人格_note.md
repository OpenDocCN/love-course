# 《朕哥8套情感合集》：06.朕哥情感 情感认知提升课：16-男人的四种依恋型人格_note

## 概述
在本节课中，我们将要学习男人的四种依恋型人格。理解这些依恋类型，能帮助你分析伴侣的性格特点，从而在未来与男性的相处中获得更大的帮助。

一个人的性格形成，绝大多数都与他的原生家庭有关。这与先天性的关系不大。你从小在什么样的家庭环境中长大，接触什么样的父母，如何成长，直接决定了你未来的性格。例如，家里非常宠爱你，这样的小孩长大后可能会很自我。如果在原生家庭中你很缺爱，长大后就会很渴望别人来爱你，渴望家的温暖，对爱有一种缺失感。因此，你是什么性格的人，与你的原生家庭有很大关系。

## 四种依恋型人格
上一节我们介绍了性格与原生家庭的关系，本节中我们来看看具体的四种依恋型人格。它们分别是：**安全型依恋**、**疏离型依恋**、**恐惧型依恋**和**痴迷型依恋**。

### 1. 安全型依恋 😊
安全型依恋是最好的性格类型，相处起来充满爱且舒适。

这种人格的形成源于原生家庭**既和谐又亲密**。家庭关系很好，不会吵架，家人之间没有隔阂，大家关系融洽、亲密。因此，拥有安全型依恋人格的男性，在关系中会带来和谐与融洽。他不会回避问题，能给你满满的舒适感。

**核心公式**：`安全型依恋 = 低焦虑感 + 低回避感`

### 2. 疏离型依恋 🏃‍♂️
疏离型依恋的特点是追求和谐，但倾向于逃避问题。

这种人格源于原生家庭**和谐但存在高回避感**。家庭可能不吵架，但你能感觉到他与父母之间有隔阂，缺乏亲密的舒适感，关系疏远但表面和谐。这样的男性性格可能比较柔和，以和为贵。但最大的问题是，他喜欢逃避。当你想要吵架或解决问题时，他可能会选择不理你或逃跑，但也不会主动解决问题，只是保持一种疏离而和谐的关系。

**核心公式**：`疏离型依恋 = 低焦虑感 + 高回避感`

### 3. 痴迷型依恋 💬
痴迷型依恋与疏离型相反，不回避问题但容易陷入争吵。

这种人格源于原生家庭**亲密但存在高焦虑感**。家人之间可能很亲密，但经常吵架，关系不和谐。因此，这类男性不会回避问题，但会倾向于跟你大吵大闹，直面冲突而不懂委婉解决。他会很清楚地知道自己需要什么，并思考如何与你相处、解决问题甚至说服你。

**核心公式**：`痴迷型依恋 = 高焦虑感 + 低回避感`

### 4. 恐惧型依恋 😨
恐惧型依恋是较为极端的一种类型，可能伴随强烈的控制欲。

这种人格源于原生家庭**同时具有高回避感与高焦虑感**。家庭关系不和谐，家人一见面就吵架，同时又互相疏远、不想见面。这类男性会害怕亲密关系。他可能会非常爱你，但同样可能控制欲极强、脾气暴躁，非常想要掌控你和这段感情。一旦得不到满足，可能会做出一些极端的行为。

**核心公式**：`恐惧型依恋 = 高焦虑感 + 高回避感`

## 如何与不同依恋类型的人相处
理解了四种人格的原理后，我们来看看如何与他们相处。并没有一个具体的万能办法，但关键在于明白其内在逻辑。

以下是核心相处原则：**看他缺什么，就补什么**。

*   **如果他原生家庭缺爱**，你就多给他家的温暖和爱情。
*   **如果他原生家庭不稳定**，你就带给他稳定的感觉和安全感。

总之，当你与一个性格有缺陷的人相处时，要观察他到底缺失什么，然后给予他相应的补偿。这是一个简单的逻辑。

## 总结
本节课中我们一起学习了男人的四种依恋型人格：安全型、疏离型、痴迷型和恐惧型。我们了解到，这些性格主要根植于原生家庭所塑造的“焦虑感”和“回避感”水平。理解伴侣的依恋类型，并遵循“缺什么补什么”的原则，能帮助你们更好地相处。这也解释了为何在择偶时，大家会如此看重对方的原生家庭背景。
