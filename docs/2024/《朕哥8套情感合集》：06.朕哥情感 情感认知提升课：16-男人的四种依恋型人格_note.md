# 《朕哥8套情感合集》：06.朕哥情感 情感认知提升课：16-男人的四种依恋型人格

🎼 了解你的男人，分析他的依恋型人格，这对未来与男性相处大有裨益。人的性格形成大多与原生家庭有关，与先天性等因素关系不大。以下是原生家庭对性格的影响：

- **原生家庭和谐**：孩子性格形成中低焦虑感，与父母关系和谐，不会经常吵架。
- **原生家庭缺乏关爱**：孩子长大后渴望被爱，渴望家的温暖，对关爱有缺失感。

**依恋型人格分类**：

1. **安全型依恋**：原生家庭和谐，亲密无间，关系融洽。
2. **疏离型依恋**：原生家庭和谐，但与父母关系疏远，缺乏亲密感。
3. **痴迷型依恋**：原生家庭亲密，但关系紧张，经常吵架。
4. **恐惧型依恋**：原生家庭关系紧张，经常吵架，彼此疏远。

**依恋型人格特点**：

- **安全型依恋**：和谐相处，关系融洽，不会回避问题。
- **疏离型依恋**：和谐相处，但喜欢逃避问题，不解决问题。
- **痴迷型依恋**：亲密相处，但关系紧张，经常吵架，直面问题。
- **恐惧型依恋**：关系紧张，极端，控制欲强，脾气暴躁。

**与依恋型人格相处**：

- 了解对方的性格缺陷，给予关爱和支持。
- 根据对方的缺失，给予相应的补充。

**总结**：

本节课中，我们学习了男人的四种依恋型人格及其特点。了解这些，有助于我们更好地与男性相处，建立和谐的关系。
