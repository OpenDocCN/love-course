# 课程17：海蓝老师情绪梳理个案课：073-个人采访 🎤

在本节课中，我们将通过周海英女士的个案分享，学习如何识别和梳理在特定关系（尤其是亲密关系）中产生的复杂情绪。我们将看到她如何描述自己的矛盾感受，并开始探索面对这些情绪的初步意愿。

## 个案背景介绍

周海英女士来自兰州，今年36岁，在金融行业工作。她前来梳理的情绪主要围绕对性关系的恐惧。

## 核心情绪与矛盾

上一节我们介绍了周女士的基本情况，本节中我们来看看她所描述的具体情绪状态。

她感到在两性亲密关系中存在障碍。她希望关系能顺畅，从而能更敞开地面对亲密关系与自我。矛盾之处在于，她对性关系既感到不安全、不舒服和恶心，同时又怀有期待。这种矛盾导致两种力量相互抵消，最终没有任何行动。

## 过往的归因与新的决定

在认识到上述矛盾状态后，周女士回顾了过去的想法。她曾将原因归结为多种因素，例如“不够喜欢”或“考虑过多”。

既然意识到这些想法可能与现状有关，她决定要面对这个问题。她认为，在他人帮助和自我探索下面对这个问题，对自身是一种进步。因此，她决定通过此次机会进行具体探索。

## 总结

本节课中，我们一起学习了周海英女士的情绪个案。我们看到，核心矛盾可以概括为：**期待 vs. （不安全感 + 不适感 + 厌恶感） = 行动力抵消**。她从过去的模糊归因，转向了主动面对和探索的决定，这标志着情绪梳理的第一步。
