在开头，泰勒提出了一个又一个核心思想。他列举了RSD导师中的一些人，比如卢克——一个300斤的胖子，和一个矮个子的型男，这两个人都能做出一些超乎常人的事情。他们能够很好地社交，并且交到很多女朋友，并且现在依然在传授相关经验。这时候，我们就要跳出一种叫做“名人光晕”的东西。

人们常常陷入名人的光环里，认为取得成就的人与自己不同，而没有意识到，其实我们每个人都有同样的潜力。**一个人能做到的，另一个人也能做到**。正是因为我们都是人，所以我们没有任何区别。别人行，你也行。别人没有比你多长一个脑袋、一条胳膊或一条腿。在我们的研讨会群里，你也可以见证其他《蓝图2.0》的学员是如何一步步从零到一、从无到有成长起来的。

思维上的卡点，有时候你想几年都想不通，但有可能群里的大佬轻飘飘的一句话，你就顿悟了。我们可以看一下抖音的一些新晋网红，比如李子柒，她有没有模仿者？小杨哥，他有没有模仿者？这些网红前赴后继，就像雨后春笋一般。我并不是说让大家去做一个网红，而是说泰勒也一直在强调：**一个人能做到的，另一个人也能做到**。

很多人在面对他人的非凡成就时，常常会感到不可思议，甚至认为这是因为对方拥有某种特殊的天赋，或者他有着比别人优秀的背景。事实并非如此。真正阻碍我们的，往往是我们自己的心理障碍。我们将成功人士视为名人，认为他们之所以能够取得今天的成就，是因为他们跟我们不一样，他们和我们拥有不同的特质。但事实上，他们和我们一样，都是普通人。他们能做到的，我们也能够做到。**一个人能做到的，另一个人也能做到**，这应该成为我们的信念和行动指南。
