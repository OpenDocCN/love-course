## 概述

在本节课中，我们将学习Python编程语言的基础知识，包括安装Python、基本语法和变量。

## 安装Python

以下是安装Python的步骤：

1. 访问Python官方网站：[https://www.python.org/](https://www.python.org/)
2. 下载适合您操作系统的Python版本。
3. 运行安装程序并按照提示操作。

## Python基本语法

以下是Python的基本语法：

```python
# 单行注释
print("Hello, World!")  # 输出Hello, World!
```

## 变量

在Python中，变量用于存储数据。以下是创建变量的方法：

```python
x = 10  # 整数
y = 3.14  # 浮点数
name = "Alice"  # 字符串
```
