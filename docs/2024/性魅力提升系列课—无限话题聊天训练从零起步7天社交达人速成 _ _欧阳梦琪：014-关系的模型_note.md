# 性魅力提升系列课—无限话题聊天训练从零起步7天社交达人速成 _ _欧阳梦琪：014-关系的模型

🎼我是欧阳梦琪，欢迎来到我们从零起步系列课的理论篇。本篇主要讲解社交关系中的一些原理和实操，掌握这些心理学原理，能帮助你更清晰地看到自己的弱项和进步方向。现在，我们开始今天的课程。

很多人渴望拥有稳定的关系，却对方法一无所知，这种感觉就像在蒙眼过马路，不知往哪走，也可能遭遇车祸。实际上，任何事如果没有掌握要领，想要得到期望的结果都很难。建立关系并非无规律的行为，而是一系列已被验证的过程叠加。我们不知道这种模糊的感性概念背后的逻辑联系。我在阅读大量心理学理论并结合自身实践后，将人与人建立关系的过程总结为六个阶段：

1. 认识阶段：指我们与一个人相识的过程。生活中，我们可以通过社交软件、工作、相亲、朋友聚会、旅游等途径认识新人。只要进入新环境，就有机会认识新人。表现出足够的善意，如得体的穿搭、良好的形象、恰当的待人接物，容易结识新人并给他们留下好感。结识新人后，最重要的是表达出“我是一个友好的人”的信号，作为自己的基础印象。在此阶段，最好不要表现出过度的兴趣，以免给对方带来社交压力。

2. 了解阶段：指两个人相互了解和筛选的过程。认识后，需要一定时间来彼此了解，这是关系能否进一步深入的前提和基础。了解阶段贯穿于构建关系的全过程，即使成为朋友或伴侣，了解的过程也不会结束。真正了解一个人需要几年甚至更长的时间。

3. 吸引阶段：指吸取别人的目光和注意力，吸引到自己身上。没有吸引别人，就不会关注你，构建关系也就无从谈起。吸引别人需要通过自身建设来构建自己的价值，并通过合理的方式展示出来。吸引和了解一样，不是一劳永逸的，贯穿整个关系始终。如果吸引降到0，关系很大几率会破裂或根本不会开始。

4. 示好阶段：在认识、了解、吸引后，才能开始示好。示好阶段的重点是认可对方，赋予对方可得性，正确推进关系。

5. 拉升阶段：将关系从普通朋友、同事、同学等关系提升为更深入的关系，让关系越过鸿沟，变成深度关系。核心技巧是通过语言或行为拉升张力，如塑造氛围、调动情绪等。

6. 稳定阶段：代表你和对方或群体的关系正式进入长期稳定的关系。此时，你需要维持健康的长期关系。

以上就是关系建立的六个阶段，就像一张地图，要随时拿出来看看自己到了哪个阶段，什么该做什么不该做。一旦不按照地图走，就可能会犯错误。

例如，小帅和小美有好感，一起上课、吃饭，互相了解，共同经历了很多事情。但小帅胆子小，始终没有勇气主动开启示好和拉升阶段，导致关系一直停留在友谊区，时间一长就固化了。小帅和小美在经过认识、了解、吸引三个阶段后，因为没有开启示好和拉升阶段，导致关系无法推进，最后一直原地踏步。

本节课中，我们学习了关系建立的六个阶段，按照这个模型推进关系，可以确保你的推进健康、有条不紊。
