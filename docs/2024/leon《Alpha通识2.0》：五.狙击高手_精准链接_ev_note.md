# 《Alpha通识2.0》：五.狙击高手_精准链接_ev

## 概述
在本节课中，我们将学习如何与高手建立联系，获取宝贵的信息和资源。

## 信息的重要性
信息是其次的，人是第一位的。链接的意义非常重要。

## 链接高手的重要性
链接高手可以获取优质的信息和资源，避免盲目学习。

## 链接高手的方法
1. **识别高手**：高手通常情绪稳定，以结果为导向，逻辑性强，语言体系完善。
2. **提问技巧**：提出优质问题，如询问对方的挑战和思维框架。
3. **提供价值**：通过优质信息和认知为高手提供价值。
4. **建立联系**：通过社交媒体、社群、线下活动等方式与高手建立联系。
5. **尊重和感恩**：尊重高手的宝贵时间和知识，感恩他们的帮助。

## 链接高手的步骤
1. **自我介绍**：展示你的上进心、追求和现状。
2. **建立联系感**：通过共同兴趣、经历等方式增加熟悉度。
3. **情绪推拉**：通过幽默、调侃等方式让高手放松，建立信任。
4. **价值观输出**：展示你的价值观和人生观。
5. **升级关系**：通过电话、线下见面等方式建立深度关系。
6. **索取价值**：学习高手的思维框架和经验。
7. **细节学习**：通过实操和反馈学习细节。
8. **结果获取**：通过前六步获取结果。
9. **强化链接**：与高手建立深度合作，获取更多资源。

## 总结
本节课中，我们学习了如何与高手建立联系，获取宝贵的信息和资源。通过以上步骤，我们可以更好地提升自己，实现人生目标。
