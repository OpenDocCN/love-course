# 796、【完结】陈海贤·自我转变50讲：chx38 我爱问海贤：现在的爱好，可以作为边缘自我吗？

欢迎来到自我转变五十讲。我是陈海贤。😊，不知不觉，我们已经完成了关于自我转变第三阶段的学习。在这个阶段，我们踏上了新的征途，一共走过三个站点，分别是整理旧自我的资源，获得新世界的信息，以及创造新的经验。今天是第三阶段的答疑环节，在开始行动之前，你可能或多或少都会有一些顾虑，我要做的事情究竟是不是正确的呢？我收到很多这样的疑问，今天就来回答一些比较有代表性的问题。

### 边缘自我判断

**问题**：如何判断边缘自我是一时冲动，还是确实是自己想要的呢？

**回答**：

- **判断标准**：看是否有一种超乎寻常的热情，愿意深入研究，克服困难，即使暂时看不到回报。
- **热情来源**：背后有一个喜欢的自我，这个自我对他意义重大，愿意为他付出和牺牲。
- **一时冲动**：需要有一个爱好，因为这个爱好有好处可以把他带往成功，有时还能帮他逃离现实的困境。

### 边缘自我与学习

**问题**：喜欢做手工，但担心不能维持生活，又去学别的东西。想做的事情很多，为之学习也花费了大量的时间，结果都只学到了皮毛。感觉是在以学习为借口，逃避真正的问题。

**回答**：

- **反省**：感觉现在是以学习为借口，逃避真正的问题。
- **边缘自我**：最初不是负担，而是一种消遣，是沉重生活中为自己保留的空间。
- **选择**：要么勇敢面对艰难，掌握一样真正的技能，要么接受自己的平庸，享受生活。

### 寻找榜样和导师

**问题**：喜欢的人很多，但不知道如何寻找榜样和导师。

**回答**：

- **主动寻找**：像找对象一样，要主动，但不是病急乱投医。
- **心态准备**：有想加入某个群体的冲动，对某一份职业、某一类人、某一种特质有憧憬。
- **潜意识指引**：带着冲动去接触很多人，问自己是否希望成为他们那样的人。

### 救自我

**问题**：经历过职业转换，不知道旧自我中有什么是新自我可以整合的。

**回答**：

- **自我是持续发展的过程**：从旧自我继承的不仅有具体的工作经验，还有抽象的情感情怀、判断和职场经验。
- **继承的资源**：我想要、原能力（克服困难的能力、学习的能力、分析判断的能力等）。

本节课中我们一起学习了如何判断边缘自我、如何寻找榜样和导师以及如何整合旧自我到新自我中。希望这些内容能帮助你更好地进行自我转变。
