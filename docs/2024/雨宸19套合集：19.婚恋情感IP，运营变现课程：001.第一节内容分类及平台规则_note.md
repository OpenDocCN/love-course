# 雨宸19套合集：19.婚恋情感IP，运营变现课程：001.第一节内容分类及平台规则

## 概述

在本节课中，我们将学习婚恋情感IP的内容分类及不同平台的运营规则。

## 内容分类

内容分类分为三大类：

1. **泛情感**：用于涨粉，内容涉及情感八卦、心理问题等，关注人群广泛。
2. **分手离婚分析**：吸引精准咨询客户，内容涉及分手、离婚的心理成因，适合售卖课程。
3. **挽回追回**：可进行一对一变现，内容涉及挽回方法、处理技巧等，收费较高。

## 平台规则

以下是四个主要平台的运营规则：

### 知乎

- **内容要求**：3000字以上，避免抄袭。
- **推广方式**：自然流量为主，可充值推广，但刷赞无效。
- **注意事项**：针对挽回方案不限流，但需注意抄袭问题。

### 小红书

- **内容形式**：图文+视频，可搬运内容，但需注意抄袭问题。
- **推广方式**：自然流量为主，广告投放几率不高。
- **注意事项**：挽回方案易被限流，可替换精准词，与泛情感、分手分析内容混发。

### 抖音

- **内容形式**：情绪+干货，视频需有情绪和干货，避免搬运抄袭。
- **推广方式**：自然流量为主，可投放抖家内容。
- **注意事项**：挽回方案不允许，需通过泛情感、分手分析引流。

### B站

- **内容形式**：长视频，内容丰富，避免搬运抄袭。
- **推广方式**：自然流量为主，可找渠道刷数据。
- **注意事项**：挽回方案不受限，后期以精准内容为主。

本节课中，我们学习了婚恋情感IP的内容分类及不同平台的运营规则。了解这些规则有助于我们更好地进行内容创作和平台运营。
