# 15、关键认知：通识哲学【50条关键认知课】——线上录播课：1-28节28、金钱观

金钱观是一个关键问题。在传统道德文化中，金钱常被视为肮脏之物，尤其在商业文明中，商人的社会地位较低，这与历史原因有关。然而，金钱对现代文化价值标准的影响深远。

这导致年轻人对金钱的态度暧昧，内心渴望却不敢直言。80后、90后相对坦然，00后则更理性地承认自己想赚钱。我们应树立怎样的金钱观？

### 存钱还是不存钱？

**结论**：**出**就要存钱。即使工资低，也要存下部分收入。

**原因**：

1. **收入低**：当你产生存不存钱的疑问时，说明你的收入较低。
2. **时间不值钱**：收入低的原因是时间不值钱，你出卖时间换取收入。
3. **时间增值**：存钱是为了将来有一段不为生计奔波的时间，让时间增值。

### 花在哪？

**1. 改变自己**：不要学习工具性知识，而是做那些暂时没有价值的事情，如学习非遗技艺。
**2. 喜欢自己**：让自己喜欢自己，找到自己的价值。
**3. 重新上阵**：当你喜欢自己后，重新进入商业世界，以不败的姿态赚取利润。

### 总结

金钱观的核心是拥有一段不为生计奔波的时间，找到自己真正喜欢的事情，实现人生目标。通过努力，让自己立于不败之地，获得现实价值。
