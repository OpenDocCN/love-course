# 《leon全套课程合集》：leon 3.0：3.内核建设，打开核心自信

## 概述
在本节课中，我们将探讨如何建立强大的内核，从而提升自信。内核，即自信的根源，与原生家庭、早期环境、个人能力等因素密切相关。

## 内核是什么？
**内核**可以理解为自信的源泉。自信的最初来源是源于你的原生家庭和早期环境。

### 原生家庭与早期环境
- 在一个充满安全感、被接纳和爱护的环境中成长，你会感到自信。
- 相反，在打压和指责的环境中成长，即使成绩优秀，也可能缺乏自信。

### 个人能力
- 提升个人能力，成为有价值的人，能够解决他人问题，满足他人需求，从而获得认同和喜欢。

## 获得自信的方法
### 环境因素
- **改变环境**：加入线上社群、寻求私教和门度，获得长期的支持和鼓励。
- **改变他人**：通过沟通，让家人、朋友和同事了解你的需求，改变他们的行为。

### 实力提升
- **外形**：通过健身、改变穿着等方式提升外形。
- **能力**：通过学习、接触优秀的人、向上链接等方式提升个人能力。

### 短期提升
- **学习与执行**：学习理论知识，并在实践中应用，获得正反馈。

## 总结
本节课中，我们学习了如何建立强大的内核，从而提升自信。通过改变环境、提升实力、学习与执行等方法，我们可以逐步建立自信，成为一个有价值的人。
