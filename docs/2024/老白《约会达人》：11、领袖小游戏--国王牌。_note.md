# 老白《约会达人》：11、领袖小游戏--国王牌

## 概述
在本节课中，我们将学习一种名为“国王牌”的小游戏，这是一种适合四人参与的互动游戏，旨在增进彼此的了解和亲密感。

## 游戏规则
以下是“国王牌”游戏的详细规则：

### 参与人数
最少需要三人参与游戏。

### 牌的数量和类型
使用5张牌，其中一张为K（国王牌），其余为1至4。

### 游戏流程
1. **发牌**：庄家发牌，每人选择一张牌，留下两张。
2. **确定国王**：亮出国王牌，确定谁是国王。
3. **制定规则**：国王制定游戏规则，其他人必须遵守。
4. **执行规则**：根据规则执行相应的动作。

### 核心概念
- **国王**：拥有制定规则的权力。
- **底牌**：玩家不能查看自己的底牌。

## 游戏示例
以下是游戏的一个示例：

1. **发牌**：庄家发牌，每人选择一张牌，留下两张。
2. **确定国王**：假设玩家A拿到国王牌，成为国王。
3. **制定规则**：国王A说：“1和2要拥抱3秒钟。”
4. **执行规则**：玩家B和C拥抱3秒钟。

## 游戏变体
- **女生制定规则**：如果女生拿到国王牌，她可以制定规则。
- **底牌不可见**：玩家不能查看自己的底牌。

## 总结
本节课中，我们一起学习了“国王牌”游戏。这种游戏适合在私密空间进行，可以增进彼此的了解和亲密感。希望这个游戏能为你的约会增添乐趣。
