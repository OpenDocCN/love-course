# Simon2024课程【新升温核心】：6.搭讪开始_ev

## 概述
在本节课中，我们将学习如何开始搭讪，包括如何打开话题、如何避免吓到对方、如何表达意图以及如何延续话题。

## 打开话题
**打开**是搭讪过程中的关键步骤，它意味着让对方将你视为一个正常人。以下是一些提高打开率的技巧：

- **避免销售形象**：不要背包、戴帽子等可能让人联想到推销者的装备。
- **使用饮品**：拿一杯饮品，如星巴克，可以降低被误认为销售的可能性。

**公式**：`打开 = 正常人交流 - 销售形象`

## 避免吓到对方
在叫停对方之前，要注意以下几点，以避免吓到对方：

- **声音先行**：先打招呼，再靠近。
- **避免突然动作**：不要突然伸出手，以免让对方感到惊吓。

**代码**：`避免吓到 = 声音先行 + 避免突然动作`

## 表达意图
表达意图有三种方式：直接、间接和推拉。

- **直接表达**：直接告诉对方你喜欢她。
- **间接表达**：通过赞美或描述来暗示你的意图。
- **推拉表达**：先贬低对方，再赞美，以增加互动性。

**公式**：`表达意图 = 直接 + 间接 + 推拉`

## 话题延续
在搭讪过程中，要避免为了聊话题而聊话题。以下是一些延续话题的技巧：

- **展示价值**：通过讲述你的经历、成就或兴趣来展示你的价值。
- **话题主导**：引导话题到你擅长或感兴趣的领域。

**代码**：`话题延续 = 展示价值 + 话题主导`

## 总结
本节课中，我们一起学习了如何开始搭讪，包括如何打开话题、如何避免吓到对方、如何表达意图以及如何延续话题。希望这些技巧能够帮助你更好地与人交流。
