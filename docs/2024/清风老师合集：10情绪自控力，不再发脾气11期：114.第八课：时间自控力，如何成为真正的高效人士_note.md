# 清风老师合集：10情绪自控力，不再发脾气11期：114.第八课：时间自控力，如何成为真正的高效人士

## 概述
在本节课中，我们将学习如何建立时间上的自控力，成为真正的高效能人士。

## 工作困境
以下是三个常见的工作困境：

1. **无法专注**：容易被打扰，导致工作效率低下。
2. **缺乏有效目标**：对工作感到迷茫，不知道下一步该做什么。
3. **没有规划**：做事无序，对未来感到迷茫。

## 目标的重要性
目标是我们前进的方向。以下是一个弓箭手的例子：

**公式**：弓箭手 + 目标 = 有意义的行动

## 目标的SMART原则
SMART原则包含五个维度：

1. **具体**：目标要有明确的结果。
2. **可衡量**：目标可以用数字或其他标准衡量。
3. **可达到**：目标要现实可行。
4. **相关性**：目标与其他目标相关联。
5. **时间限制**：目标要有明确的时间限制。

## 设计目标的步骤
以下是设计目标的七个步骤：

1. **调研**：了解真实情况，收集信息。
2. **制定目标**：根据SMART原则制定目标。
3. **拆解目标**：将大目标拆解为小目标。
4. **执行**：逐步实现小目标。
5. **预案**：制定应对意外情况的预案。
6. **反馈**：定期进行反馈和复盘。
7. **优化**：根据反馈优化目标。

## 总结
本节课中，我们学习了如何建立时间上的自控力，成为真正的高效能人士。通过设定明确的目标，并按照SMART原则进行拆解和执行，我们可以提高工作效率，实现个人成长。
