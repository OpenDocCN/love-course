# 清风老师情感幸福课合集 第10期·第8课：⏳ 时间自控力，如何成为真正的高效人士

在本节课中，我们将学习如何建立时间上的自控力，从而成为真正的高效能人士。我们将探讨工作中常见的困境，并学习一个经典的目标管理原则及其具体应用步骤。

在上一节课程中，我们介绍了什么是真正的正能量，以及建立自我接纳和接纳他人的三个标准。本节中，我们来看看如何通过管理时间来提升个人效能。

以下是三个常见的工作困境，请看看你是否遇到过。

*   **困境一：无法专注。** 工作时总被各种事情打扰，无法专注于单一任务。本想好好工作，却不断被打断，导致一小时能完成的事被拖延至三、四小时甚至更久。
*   **困境二：缺乏有效目标。** 对所做之事感到迷茫，完成眼前任务后不知下一步该做什么，或觉得长远目标遥不可及，从而怀疑工作的价值与意义。
*   **困境三：没有规划。** 做事缺乏条理，东一榔头西一棒子，前进过程中常感到迷茫，无法预知后续进展。

为什么我们需要建立目标？试想，如果一个弓箭手没有目标，他手中的箭还有意义吗？答案是没有。目标就是我们真正想要抵达的地方。

从长远看，目标是**你想成为什么样的人、过什么样的生活**。从短期事业角度看，你可以问自己：**如果所有工作收入相同，我愿意做什么？** 这能帮你明确工作目标。再具体到项目，则要问：**这个项目需要取得什么成果？我们的目标究竟是什么？** 有了目标，前进才有方向。

一个正确的目标包含五个维度，这遵循一个非常经典的 **SMART原则**。

*   **S - 具体的**：目标需要有明确、清晰的反馈结果。例如，“我要变好”不具体，而“我要学习自控力课程”则很清晰。
*   **M - 可衡量的**：目标应能用数字关联，便于衡量进度。例如，将“一天打100个电话”转化为“每打完一个电话，就从100个图钉盒中取出一个”，这就建立了可衡量的机制。
*   **A - 可达到的**：好目标应是“跳一跳能够得着”的，而非需要“踩着火箭”才能完成。不切实际的宏伟目标一旦失败，会带来糟糕的自我感受。
*   **R - 相关性**：目标应与其他目标相关联，形成正向循环。例如，学习自控力课程可与改善情感关系、提升工作能力相关联；学习英语可与获得更高职称、增加收入相关联。
*   **T - 有时限的**：目标必须有明确的截止时间，否则会被无限拖延而失去意义。为自己设定一个“死亡线”能有效督促行动。

以上就是SMART原则的五个标准：**具体的、可衡量的、可达到的、相关的、有时限的**。

那么，我们该如何设计一个具体的目标流程呢？我将它拆解为七个步骤，这也是我在上市公司带团队时分享的方法。

1.  **调研**：了解真实情况，通过询问客户、专家或自我审视，收集足够信息。
2.  **制定目标**：运用SMART原则，制定一个符合标准的目标。
3.  **拆解**：将大任务或复杂目标拆解为具体、清晰的小目标。例如，将“拥有好身材”拆解为：第一周改善饮食，第二周去健身房，第三周学习健身知识，第四周加入健身社群。
4.  **执行**：逐个完成小目标，最终汇聚成大目标的达成。
5.  **预案**：为目标执行中可能出现的意外制定应对方案。例如，计划运动却下雨，需有室内替代方案；工作忙碌却想改善社交，需提前规划时间。
6.  **反馈**：在关键节点进行复盘与反馈。有反馈的目标才有意义，它能促进沟通与自我成长。
7.  **优化**：根据反馈，总结优点并记录成功经验，发现缺点并补充预案，不断完善流程。
