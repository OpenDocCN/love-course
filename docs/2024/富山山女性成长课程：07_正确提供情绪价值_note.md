# 富山山女性成长课程：07_正确提供情绪价值 💖

在本节课中，我们将要学习如何正确地为他人提供情绪价值。很多人对情绪价值的理解是单一且片面的。许多女性认为提供情绪价值很简单，就是夸奖、赞美和欣赏。但事实并非如此。

很多人并不接受单一的赞美。有些人反而喜欢被拉扯、被“虐”的心痛感。因此，我将情绪价值分为正向和负向两种。在与男性相处的整个过程中，你需要根据对方的“口味”来调整你所提供的情绪价值类型。

## 正向情绪价值

上一节我们介绍了情绪价值的分类，本节中我们来看看正向情绪价值的具体构成。正向情绪价值包含几个核心部分。

以下是正向情绪价值的几个关键组成部分：

1.  **赞美**：这是最基础的一项。
2.  **接纳弱点**：包容对方的缺点。
3.  **安抚与鼓励**：在对方需要时给予支持。
4.  **成为精神支柱**：在更深层次上提供情感依靠。

### 如何有效赞美

那么，我们到底该如何有效地赞美一个男人呢？真诚度、细节感和独特视角都是必不可少的要素。有时，我们甚至需要运用更高级的技巧，例如“拉踩”（通过对比抬高）、先抑后扬、或带有反讽意味的夸奖。这些都是人际交往中的必修课，而非单一的夸赞。

以下是结合了真诚、细节和独特视角的赞美示范：

*   **夸能力**：不要直接说“你好棒，工作能力好强”。可以这样说：“宝宝，昨晚我看到你和员工对话时的状态，以及你处理某件事的具体方式（**描述细节**），我真的好佩服。什么样的男人能想出这样的方法？你可以教教我吗？或者让我跟在你身边学习？” 这种方式包含了时间、地点、人物和具体事件，并结合了请教的口吻。
*   **夸品味**：不要简单说“你真有品位，衣服好看”。可以这样说：“经过这段时间相处，我由衷觉得你是个很有品位的人。比如你昨天穿运动鞋时把袜子堆起来穿（**细节**），只有很细心的男生才会注意这种搭配亮点。上次去你家，我看到你的花瓶摆法和墙面颜色太搭了（**细节**）。我就在想，到底是什么样的环境和经历，让你拥有这么独到的眼光和艺术感？你和大部分粗枝大叶的男生真的很不一样。”
*   **夸礼物**：收到对方送的礼物时，可以先肯定礼物，再肯定对方的用心。例如：“你怎么知道我最喜欢这款包？你是我肚子里的蛔虫吗？还是说你太关注我，把我的一举一动都放在心上？（**肯定关注度**）” 戴上手表后可以说：“你看这个秒针，像你大腿肌肉的线条（**独特联想**）。这块表非常配我的脸型和手型，和我那件白色皮衣是绝配。这么百搭经典的选择，一定是很有品位的男人才能挑出来的。”

赞美可以继续深化。你可以联系他的家庭背景或未来潜力。例如：“你妈妈是不是也特别有审美，常带你去博物馆，所以你眼光这么好？” 或者：“你要是不做事业，去做艺术家或鉴赏师也一定能成功。这说明你有洞悉事物的能力和敏锐的直觉。” 甚至可以用略带夸张的方式说：“你这样的基因，不生十个八个孩子可惜了。” 这样的赞美能激发他在日常生活中对你的保护欲。

## 日常互动与人设建立

在学习了如何具体赞美之后，我们来看看如何在日常互动中巩固这种情感连接。日常生活中的昵称和互动也是提供情绪价值、帮助建立个人形象的重要方式。

当对方在日常生活中为你点餐或带你吃美食时，你可以称呼他为“我的小饭票，你最懂我了”。这种称呼实际上是从侧面帮助你建立人设。你需要在生活中时常提及“价值”和“现实”，但要以可爱的方式表达。

例如，在交往初期，可以称呼对方为“我的小提款机来了”、“我的小饭票”。长期下来，当你提前这些现实话题时，对方就不会排斥。这种既现实又可爱的人设就能够稳固地建立起来。
