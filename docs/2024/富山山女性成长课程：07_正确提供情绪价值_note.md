# 富山山女性成长课程：07_正确提供情绪价值 🌟

## 概述

在本节课中，我们将学习如何正确地提供情绪价值。情绪价值不仅仅是赞美和夸奖，它包括更广泛的情感支持和理解。

## 正向情绪价值

### 赞美

**公式**：真诚 + 细节 + 独特视角

1. **真诚**：赞美要发自内心，不要虚假。
2. **细节**：关注对方的细节，例如工作状态、处理问题的方法。
3. **独特视角**：从独特的角度赞美，例如对方解决问题的方法。

### 接纳弱点，包容缺点

**公式**：理解 + 支持 + 鼓励

1. **理解**：理解对方的不足，不要轻易批评。
2. **支持**：在对方遇到困难时给予支持。
3. **鼓励**：鼓励对方克服困难。

### 安抚与鼓励

**公式**：同理心 + 鼓励

1. **同理心**：站在对方的角度思考问题。
2. **鼓励**：鼓励对方面对困难。

### 精神支柱

**公式**：信任 + 支持 + 鼓励

1. **信任**：信任对方，给予对方安全感。
2. **支持**：在对方需要时给予支持。
3. **鼓励**：鼓励对方追求梦想。

## 赞美的示范

以下是一个赞美男人的示例：

> 宝宝昨天晚上我看到你跟你的员工对话。我看到你工作时候的状态，我看到你是怎么怎么样处理某一件事情的。我真的觉得好佩服呀。什么样的男人能想出这样的方法呀？你可以教教我嘛，可以把我带在你的身边，让我跟你好好学习吗？

## 总结

本节课中我们一起学习了如何正确地提供情绪价值。通过赞美、接纳弱点、安抚与鼓励以及成为精神支柱等方式，我们可以更好地与他人建立情感联系。
