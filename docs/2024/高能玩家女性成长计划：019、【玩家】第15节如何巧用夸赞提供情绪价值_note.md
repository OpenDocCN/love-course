# 高能玩家女性成长计划：019、【玩家】第15节：如何巧用夸赞提供情绪价值 🎯

在本节课中，我们将学习如何通过巧妙的语言夸赞，为他人提供有效的情绪价值。我们将探讨不同心理议题下的夸赞策略，并提供可直接套用的公式，帮助你快速建立积极的人际链接。

## 概述：理解情绪价值与夸赞的核心

每个人都有其核心的人生议题。夸赞是向他人输送能量的最佳方式之一。

无论一个人身处“权威议题”还是“依恋议题”，其天性中都存在对“自恋”的需求。人的心理发育始于依恋议题，从婴儿时期起，潜意识里就渴望获得“全能自恋感”。

除了面对面的肢体互动，语言夸赞是能在人际交往中快速实现能量传递的方法。这正是“情绪价值”的重要组成部分。“情绪价值”一词常被提及，但其内涵可能令人困惑。

情绪价值并非简单的嘘寒问暖、关怀或不断恭维。这些理解是片面的。本节课将具体讲解如何通过语言夸赞，为互动对象提供真正的情绪价值。

## 第一步：区分核心议题

首先需要学会区分对方的核心心理议题。

若不具备足够的心理学知识，可掌握一个基本规律：多数男性拥有“权威议题”，多数女性拥有“依恋议题”。部分女强人因身处“权威矩阵”，也拥有权威议题。当然，也存在两种议题兼具的男性和女性。

理解这一点是应用后续方法的基础。

## 核心方法：两种万能夸赞公式

本节将提供两种针对不同议题的夸赞公式。这些公式旨在让你能快速从大脑中调用，建立积极链接与正向互动。

无需深入学习心理学理论去探测对方内心。在运用语言提供情绪价值时，若实在无法区分对方议题，可轮流使用“权威夸法”和“依恋夸法”，基本不会出错。

语言能快速为他人提供情绪价值。相比之下，通过行为传递价值周期较长，而语言能更直接地走入他人内心。

### 公式一：针对“权威议题”的夸赞

对于权威议题重的人，应运用以下公式：
**夸他强 + 细节论证 + 自己的受益**

权威矩阵的人努力攀登，是为了实现“强者”的感受。

普通的夸奖如“你好厉害”、“领导真有远见”略显空洞。虽然积极主动的姿态可能对部分权威议题者生效，但并非总是有效。

高能玩家的夸赞示范如下：
> “您真的太厉害了。我前几天躺在床上思索人生困惑，刚才听您一番话，瞬间被点醒，醍醐灌顶。如果我几年前听到，能少走几年弯路，真是相见恨晚，您是我的贵人。”

这种夸赞没有虚无浮夸的臣服姿态，但实现了对对方逻辑脑的“降维打击”。

**此公式的精妙之处在于：**
1.  **对比衬托**：用“饭前不会，见面后学会”来衬托对方的厉害程度。
2.  **评价自己，而非对方**：许多权威议题者内在讨厌被过度关注和评价，即使正面评价也会令其不适。因为直面夸奖与直面审判的底层逻辑相似，都让对方感到在被评价。而我们的方法是在评价自己，将对方置于安全的“上帝视角”，极大满足了其“审判者”的心理需求。这个技巧结合了心理学与高能玩家意识矩阵，非常巧妙。
3.  **细节作为无可辩驳的证据**：用“前几天的我”和“现在的我”的感受变化作为细节论据，对方的逻辑系统无法推翻这些个人感受。我们用自身巨大的、迅速的变化，喂养了对方“好为人师”的强者自恋，同时避免了其“被审判”的感觉。
4.  **营销自身高价值**：通过展现自己悟性高、学习能力强，我们给权威议题者的逻辑系统留下了“高价值”的印象。被优秀的人夸赞一次，优于被平庸者恭维百次。“孺子可教”远比“油盐不进”高级。这让对方感到能量涌向我们，其潜意识会将我们视为“自己人”，因为能量已完成勾连。人对付出过心血（哪怕是心理能量）的对象难以忽视。

使用此公式必须发自内心，真诚地看到对方优点。

此技巧不仅适用于向上社交或职场，也适用于亲密关系，因为90%以上的男性伴侣拥有权威议题。尤其在亲密关系中，不建议使用直接夸人的方法。

**反面案例：**
*   饭局上，有人敬酒说：“领导您身材真好，一定很自律。”领导听后不太高兴：“公仆嘛，当然要对自己要求高点。”此人后续评价一般。
*   女孩带男友见父亲，男友恭维：“叔叔您车技真好。”女孩父亲私下不满意：“我车技好不好，轮得到你来评价？”

权威议题者非常抗拒被评价。为什么有时有人说“领导您真棒”却没引起抗拒？可能因为恭维者具备其他特质，如年轻美貌（代表强生命力），或是有背书、人品好的人（如董宇辉），其话语可信度高。

**切记**：不要轻易评价权威议题者“自律”、“上进”、“会管理”。我们并非官方评价系统，直接评价缺乏说服力。权威议题的出现，往往源于其原生家庭中已存在“审判者”，他们对此已深感厌倦。生硬的评价会像“爹味审判”一样令人不适。

最稳妥的方式就是高能玩家公式。将“领导您身材真好，一看就很自律”改为：“领导您身材保持得真好，看了您我很受触动，我在想如何才能成为像您一样优秀的人。”主要描述放在自己身上。

**公式扩展应用（已有交情基础）：**
对于已有交情的领导或上位者，可不直接夸“强”，而是通过多个细节论证来衬托，并强调自身受益。姿态上可保持“受造福的晚辈”姿态。

例如拜年信息：送上祝福与感恩 -> 用自身实例证明领导的能量之强 -> 最后加上祝福。全程通过描述自己的受益来衬托对方，而非一味褒奖。这种方式成效显著，且能获得对方高度认可。

**注意**：并非所有权威矩阵者都只需被满足强者自恋。但在所有公众场合，务必使用此公式，因为公众场合遵循丛林法则。

### 公式二：针对“依恋议题”的夸赞

对于依恋议题重的人，应运用以下公式：
**夸特别/不俗之处 + 细节论证 + 自己的受益**

低能情感矩阵中的人（无论男女）长期处于低能量状态。他们既有被“看见”的需求，又因孤立无援而需要支持以增强能量感。长期低能量可能导致大脑前额叶功能受影响，逻辑性欠缺。高敏感特质也使其容易“自我策反”（自我怀疑和否定）。

因此，我们不仅要输送高能量，还要改善其低配得感和低价值感，让其感受到自己是美好而强大的，同时为其大脑提供论证佐证。

为什么要夸“特别之处”？因为极度依恋缺损的人，内在对丛林法则不屑一顾。他们不希望自己是普罗大众认可的“强”，而是追求标新立异、另一个赛道的“强”，即“不俗的强”。这源于情感矩阵的特性：对现实主义不屑，向往梦幻主义。

因此，与低能情感矩阵者打交道时，“你好厉害”、“你能力好强”等夸赞力度不够，需遵循我们的公式。

**示范**：
> “你今天黄色上衣配咖啡色半裙的搭配让我惊呆了。我只会用黄色配黑色，总觉得差点什么。今天见到你终于知道差哪儿了。我觉得多跟你见几次，我都能成搭配小天才。”

通过挖掘对方身上不俗的特别之处，来提升其自恋感、价值感，改善低自尊、不自信的部分，并用细节佐证，说服其大脑接纳夸赞。

这考验我们挖掘他人特别之处的眼光。

## 备用方案：通用与进阶技巧

如果你觉得区分议题和记忆公式仍显复杂，本节提供更通用和进阶的夸赞技巧。

### 技巧三：万能玄学夸法

一个夸赞任何议题者都适用的方法是：**往玄学上靠拢**。

比起普通夸赞，输送能量更强的方式是将其与“好运”、“福气”等玄学概念关联。

**示范**：
> “都说下巴长痣的人福泽深厚，很多伟人像毛爷爷、明星孙俪都这样。天哪，我得多沾沾你的喜气。”

对方会瞬间感受到高能量，且无法用逻辑策反（谁不想有好运呢？），从而快速拉近距离、产生交互。

### 技巧四：借第三人之口夸赞

这属于高能玩家中的“借势”技巧，借助他人的能量。

**示范**：
> “上次张三就跟我说，咱们公司新来的姑娘身材很好，今天一见果然名不虚传，给你美到了，我心情都好了起来。”
> 大文人胡适给妻子回信写道：“你这封信写得很好，我念了几段给钱瑞生、张子高两位听，他们都说，‘胡太太真能干，又有见识。’”

借第三方之口，既能说服逻辑系统，又能借到能量的势能。

### 核心原则：实事求是

无论使用哪种基于细节的夸赞，都必须**实事求是**。

不要夸对方没做过的事，也不要夸明显有漏洞的地方。例如，领导大腹便便，却夸其自律身材好；美女搭配一般，却硬夸其会搭配。这非常不妥。

**反面案例**：
某女性在局上恭维商业大佬Y：“上次我和领导A吃饭时，听她提起过您，说您能力很强。”当时领导A也在场，听后瞄了她一眼。事后了解到，该女性并未与A吃过饭，也未谈论过Y。这种不实的夸赞，虽然暂时恭维了Y，却伤害了A的逻辑系统，令A产生负面感受，认为她不实事求是，后续带来诸多麻烦。

**大家务必谨记此原则。**
