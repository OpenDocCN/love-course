# 45、牵手丘比特--姜姜课程学习：2.很重要！聊天想法以及应对方法_ev

## 概述

在本节课中，我们将学习如何进行有效的聊天，包括聊天框架、逻辑、记录以及如何提供优质服务。

## 聊天框架

1. **恋爱模型**：建立清晰的恋爱模型，迅速建立恋爱认知。
2. **熟悉感**：建立熟悉感，成为朋友，展现高价值。
3. **打破和谐感**：打破和谐感，突破友谊区，避免长时间停留在友谊区。
4. **构建吸引力**：构建吸引力，包括讲话、事业成就、疼人、乐观向上等。
5. **升级关系**：通过约会、牵手、拥抱、接吻等方式升级关系。

## 聊天逻辑

1. **开场白**：根据时间、地点、场合等因素选择合适的开场白。
2. **话题**：根据女孩的关键字延伸话题，或通过朋友圈主动提供话题。
3. **好感度指标**：通过主动提供话题、提问、文字加表情、语音聊天、逗笑等方式测试女孩的好感度。
4. **安全感**：打破不安全感，展现真实的自己，同时保持框架和原则。

## 聊天记录

1. **朋友圈**：展示朋友圈，展现自己的价值。
2. **讲故事**：通过聊天学会讲故事，展现自己的优势。
3. **约会**：通过约会展现自己的安全感以及价值所在。

## 总结

本节课中，我们学习了如何进行有效的聊天，包括聊天框架、逻辑、记录以及如何提供优质服务。希望同学们能够将所学知识应用到实际生活中，提升自己的聊天技巧，获得美好的爱情。
