# 林老头《私人传承》：016.长期关系深度解析，技巧的最上层使用就接近了“道”本身

各位团承班的兄弟们，大家好。本节课我们将深入探讨长期关系的维护技巧。以下内容将围绕七个核心点展开，涵盖系统课认知的30%左右。

### 1. 价值与获得感

**公式**：获得感 = 价值 / 付出

价值与获得感是长期关系中的关键因素。如何合理运用价值，提升对方的获得感，是本节的重点。

### 2. 系统课内容与解决问题能力

**公式**：解决问题 = 确定目的 + 满足需求/合理化 + 情绪沟通

在长期关系中，矛盾是不可避免的。学会分析矛盾目的，满足需求或进行合理化，以及情绪沟通，是解决问题的关键。

### 3. 预期价值

**公式**：预期价值 = 对未来憧憬 / 动力

预期价值为关系注入动力。通过共同目标、项目合作等方式，提升对方的预期价值。

### 4. 底层诉求

**公式**：底层诉求 = 与你在一起的原动力

了解对方的底层诉求，是实现长期关系的关键。

### 5. 套牢效应

**公式**：套牢效应 = 控制权 + 信息引导 + 信任建立

通过控制权、信息引导和信任建立，让对方在关系中越陷越深。

### 6. 新鲜感的经营

**公式**：新鲜感 = 重新追求 + 故事重现

通过不断提供新鲜感，提升对方的付出动力和关系稳定性。

### 7. 外界对比

**公式**：外界对比 = 优秀点对比 + 生活对比

通过对比，突出自身优势，提升关系稳定性。

本节课中，我们学习了长期关系的维护技巧。希望这些内容能帮助大家建立稳定、幸福的长期关系。下一节课，我们将继续探讨相关话题。谢谢大家！
