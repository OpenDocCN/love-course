# 课程05：对视无惧 · 010. 心法：解决闯入性思维 🧠

在本节课中，我们将要学习如何理解和应对“闯入性思维”。这是一种常见的心理现象，也是引发某些强迫症的直接原因。通过学习其本质和应对方法，我们可以避免被这些不请自来的想法困扰。

## 什么是闯入性思维？ 🤔

上一节我们介绍了课程主题，本节中我们来看看闯入性思维的具体含义。闯入性思维是指在正常思考过程中，突然出现的、不受主观意愿控制的杂念或想法。

人脑每秒都在进行数百万次的化学反应。在这些反应中，会产生各种思维碰撞和想法，这完全正常。我们的思维不会只产生生活所需的主线思维。

**主线思维**指的是我们正常生活和工作中的思考路径。例如：
*   思考中午做什么菜。
*   考虑与领导或同事沟通的内容。
*   规划如何撰写一份PPT。

然而，在数百万次的化学反应中，我们无法控制所有过程，因此总会产生一些杂乱无章、突然出现的想法。

## 闯入性思维为何会困扰我们？ 🌊

以下是闯入性思维产生影响的两种不同情况：

对于非高敏感人群，某个闯入性想法可能像蜻蜓点水一样，出现后即刻消失，不会引发后续的深度思考。

但如果你属于高敏感人群（易感人群），某个闯入性想法就可能在思维中泛起很大水花，甚至形成涟漪不断影响你。你会开始陷入深度联想，不断思考“我为什么会有这种想法”、“这是不是我的错”、“该如何控制和改正”，从而进入强迫思维的漩涡。

需要明确的是，闯入性思维本身受到我们正常意识的制约。例如，你突然想到“从楼上跳下去”，但你并不会真的去做。因为我们具备思维底线、自我保护意识、道德和法律意识。因此，这种思维本身通常不会造成实际后果。

但是，如果你陷入上述思维的漩涡，不断进行强迫性思考，形成了强迫症，那么最终就会造成一种实际的伤害。这种伤害更多是内向的自我摧残过程。

## 如何应对闯入性思维？ 🛡️

应对闯入性思维的方法其实很简单，核心就是**转移注意力**。这个方法非常管用，因为闯入性思维在刚进入我们思维体系时，是很弱小和轻薄的。只要让它失去我们的注意力，不再专注于此，它通常会自动消退。

以下是具体的应对策略：

如果这个突如其来的想法对你本就不会产生太大影响，你内心非常确信它很荒谬。那么，你完全可以把它放在那里，不需要刻意转移注意力，也不用管它。总之，不去关注，问题就会自动消失。

如果这个想法对你产生了一点影响，你开始想要深入思考它。那么，请立刻引入另外一件事情。这件事情最好是能让你感兴趣或感到开心的，当然，能引发你愤怒的事情也可以（因为强烈的情绪也能转移焦点）。这样，闯入性思维就会失效。

**举例说明**：
有一种比较轻微且容易破解的强迫症叫“口水强迫症”。有一次我骑车时，脑子里突然闪出“口水强迫症”这几个字。我当时是这么做的：我开始想中午要吃什么，考虑路上是否需要买点东西回家，然后打电话询问家人是否需要带东西。等我把这一串事情想完并做完后，我几乎已经忘了最开始在想什么。过程就是这么简单。

## 课程总结 📝

本节课中，我们一起学习了闯入性思维的本质与应对方法。

闯入性思维是大脑正常活动中产生的杂念，本身无害且受到意识制约。应对的关键在于**不与之纠缠**。如果它不影响你，就忽略它；如果它开始吸引你的注意，就立刻用另一件能吸引你的事转移注意力。记住，**不关注**是让这类思维自动消退的最有效方式。
