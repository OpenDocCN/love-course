# 《朕哥8套情感合集》：02.朕哥高阶私教课之保卫爱情：03-第三篇：判断男人对你的爱还剩多少

🎼大家好，我是郑哥。本节课将教授大家如何判断男人对你的爱还剩多少。郑哥在为学员提供情感咨询后，发现一个普遍问题：人们仅通过表面现象分析男人是否爱自己，而忽略了男人的真实感受和实际情况。

**核心概念**：判断男人对你的爱还剩多少，不能仅凭表面现象，而应关注男人的内心感受和实际付出。

**以下是本节课的主要内容**：

### 1. 情绪表达与感情深度

- **情绪表达形式**：正向表达情绪和负面表达情绪。
- **情绪表达对感情的影响**：正向表达情绪使感情甜蜜稳定，负面表达情绪使感情不愉快。
- **判断方法**：观察男人是否愿意为你改变，是否愿意为你付出。

### 2. 男人爱你的表现

- **情绪被你牵动**：男人会因你的喜怒哀乐而受到影响。
- **为你改变自己**：男人愿意为了你改变自己的性格和习惯。
- **很少拒绝你的要求**：男人会尽力满足你的需求。

### 3. 男人在爱你时的具体行为

- **吵架**：吵架说明你们之间仍有感情，但要注意控制情绪，解决问题。
- **管着你**：男人关心你的安全，希望保护你。
- **任何事情都陪着你**：男人愿意与你共同面对困难和挑战。
- **邀请你共同出入公开场合**：男人希望与你建立更深的联系。
- **为你付出金钱精力**：男人愿意为你付出时间和金钱。

### 4. 男人爱你的减少表现

- **偶尔以玩笑的形式提分手**：男人对待感情认真，玩笑式的分手要引起警惕。
- **经常性消失**：男人不再主动联系你，说明他对你的感情已经减弱。
- **对你的态度逐渐平淡**：男人不再关心你的感受，说明你们之间的问题已经严重。

**总结**：判断男人对你的爱还剩多少，需要关注男人的内心感受和实际付出。通过观察他的情绪表达、行为表现和付出程度，可以初步判断他对你的爱是否还在。
