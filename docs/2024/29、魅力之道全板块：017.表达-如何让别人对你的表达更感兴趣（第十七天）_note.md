# 魅力之道全板块：017.表达-如何让别人对你的表达更感兴趣（第十七天）

大家好，今天我们来学习如何让你的表达更加吸引人。首先，大多数情况下，别人对你的兴趣源于你本人，这也是魅力课程的目标。在这种情况下，你的个人价值至关重要。

即使你的表达技巧不多，只要你能引起别人的兴趣，内容本身就能吸引人。这也是为什么表达板块是整个魅力课程的辅助板块，而1到3个板块则专注于建立你的强大内核。

有时，你对别人说的话可能很感兴趣，但并不关心这个人。例如，你去看一个脱口秀演员的表演，一开始你可能不认识他，也不感兴趣。但表演结束后，如果演员来找你聊天，你会对他产生更多兴趣，因为他通过表达赢得了你的兴趣。

因此，我们的重点应放在对人的兴趣上。如果别人对你感兴趣，你的表达技巧将如虎添翼，让他们对你更加感兴趣。就像拆开一个精美的盒子，里面装着有价值的东西，他们会享受这个过程。你的表达就是你的包装，之前的课程介绍了许多提升个人价值的方法，这些方法同样适用于提升你的软价值。

例如，马斯克虽然社交技巧不佳，但他的演讲总是吸引人的，因为他具有个人价值。我们不是他，但我们可以通过提高硬价值和软价值来提升自己。软价值包括主见、执行力、价值观和情绪价值等。同时，建设硬价值也很重要，如身材、穿搭等，这些都能在说话前给人留下好印象。

你可以根据自己的兴趣和方向来建设硬价值，而且硬价值通常需要长期建设。当你既有硬价值又有软价值时，你将无敌于世。因此，你需要同时提高硬价值和软价值，进行长期自我建设，这样你说什么别人都会感兴趣。

当别人认为你有价值时，你的表达就像奢侈品，精美的包装让好东西更有格调，更吸引人。当别人还不了解你时，你的表达可以帮助提升你在他们心中的价值。

接下来，我们将介绍如何让别人对你的表达更感兴趣。第一种方法是铺垫，或称为抛钩子。在此之前，先思考一下，你如何在表达前铺垫，以引起别人的注意力和兴趣？让对方感受到从你的表达中获得价值，他们就会感兴趣。以下是一些方法：

1. **分享新奇、有趣或具有情绪价值的内容**：例如，分享你人生中最惨的一次约会经历或本周发生的倒霉事。
2. **与对方相关的内容**：例如，告诉你的约会对象，你对他/她的印象与预期相符的比例。

铺垫的重点在于兑现别人之前建立的预期，确保你的内容不low，不与预期背离太多。

接下来，介绍第二个方法：说话节奏和语气。在表达故事时，每次停顿都会引起对方的期待。以下是一些技巧：

1. **朗读练习**：通过朗读故事来练习停顿。
2. **语气变化**：根据表达的内容调整语气，例如，用不同的语气表达“我发生了超级倒霉的事情”。

最后，今天的作业是修改你之前写的故事，并尝试用上述方法讲述它。

本节课中，我们学习了如何让别人对你的表达更感兴趣。希望这些方法能帮助你提升个人魅力。明天再见！
