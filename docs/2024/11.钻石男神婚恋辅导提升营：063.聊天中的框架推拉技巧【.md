## 什么是框架？🛡️

上一节我们介绍了课程主题，本节中我们来看看什么是框架。

框架是一个人为人处事的原则。每个人都应该有自己的框架。

当我们和女生相处时，难免会遇到框架碰撞的情况。碰撞的结果无非有两个：事情最后是按照你的意愿去做，还是按照我的意愿去做。如果我们在和女生的相处过程中，永远服从女生的框架和意愿，按照女生的要求来，那么渐渐的你的框架就会越来越弱。你的硬框架就会越来越少，软框架会越来越多。那你的可得性就会越给越高，你的吸引力会越来越小。

如果说你对他百依百顺，倾尽所有，也许在他的理智里面认为你确实是对他很好，他应该同样去爱你。可是在他的荷尔蒙系统里面，他会发现自己渐渐已经不爱你了。因为你的吸引力实在是在不经意间就流失掉了。

那可能会有人问，难道我这样对他还不够好吗？很遗憾，你认为这是爱他的一种方式，其实是以爱之名来掩盖自己的窝囊和懦弱。因为你的脑子里面没有框架的意识，所以说你会流失掉吸引。他不喜欢你是一个必然的过程。

**核心公式：吸引力 ∝ 硬框架强度**

男人必须要有自己的框架。而发生框架碰撞的时候，我们也要能守住自己的框架。守住自己框架的方式，就是使用推拉。

## 推拉技巧的运用：先推后拉 🔄

当“推”展示了你的框架后，需要适时地“拉”回来，以平衡关系并给予奖励。以下是推拉的具体步骤：

**第一步：推（展示框架，设立边界）**
男生对女生的高高在上姿态表示不认同，明确了平等相处的原则。代码可以类比为设定一个条件判断：
```python

    response = “请以平等身份相处”
```

**第二步：拉（给予奖励，表达亲近）**

这个时候，女生已经完全进入到了男生的框架里面：我是你的女朋友，而且我很欣赏你，我觉得和你在一起非常的有趣。
