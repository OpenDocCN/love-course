# 16.隔壁王司机情绪自由的真相：第七课：人的情绪能量大不相同

**概述**：在本节课中，我们将通过一个真实事件，探讨人的情绪能量差异以及如何面对恐惧。

---

**事件回顾**：

2005年，作者在大连讲课，结束后被邀请游玩。在游玩过程中，作者被邀请尝试蹦极。起初，作者对蹦极的安全性表示担忧，但在朋友和教练的鼓励下，最终决定尝试。

**蹦极经历**：

1. **准备阶段**：作者在蹦极地点签字，并被告知需要绑上绳索。
2. **恐惧来临**：站在跳台上，作者感到极度恐惧，身体发抖，额头和手心冒汗。
3. **克服恐惧**：在朋友和教练的鼓励下，作者最终鼓起勇气跳下。
4. **空中经历**：作者在空中翻滚，感到极度恐惧，但最终绳子停住，安全落地。

**恐惧的根源**：

作者在回顾自己的经历时，发现自己对高处和跌倒的恐惧源于童年时期的一次意外。这次经历让作者对恐惧有了更深的理解。

**同理心的重要性**：

作者认为，同理心是理解他人情绪的关键。在面对他人恐惧时，我们应该尊重并接纳他们的感受，而不是轻易否定。

**总结**：

本节课通过一个真实事件，让我们了解到人的情绪能量差异以及如何面对恐惧。在面对恐惧时，我们需要勇敢地面对，并学会尊重他人的感受。

---

**核心概念**：

* **恐惧**：一种情绪体验，通常与危险或威胁相关。
* **同理心**：理解并感受他人情绪的能力。
