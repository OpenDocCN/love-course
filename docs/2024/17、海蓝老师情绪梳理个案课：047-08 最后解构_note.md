# 海蓝老师情绪梳理个案课：047-08 最后解构

🎼明确当前首要任务，不是工作。以下是对个案的详细分析。

**案例分析**

当青晨提到特别害怕妈妈离开时，他最初的恐惧程度评分为2分（满分10分）。这种恐惧难以衡量，因为一想到这件事，青晨就会感到极度恐惧，甚至流泪。

**童年经历**

青晨6岁时，妈妈吵架后离家出走。对青晨来说，这意味着失去妈妈，甚至死亡。当时，他拼尽全力去追妈妈，因为害怕失去她。这种恐惧和死亡的感觉一直伴随着他。

**恐惧形成**

青晨对妈妈离开的恐惧，以及对婚姻的恐惧，源于童年经历。他认为，父母吵架意味着生命的终结，甚至可能导致死亡。这种恐惧一直延续到成年。

**情绪梳理**

为了帮助青晨释放情绪，我们需要回到童年，了解当时发生了什么。在梳理过程中，青晨意识到，妈妈并没有离开，而是追了回来。这让他意识到，自己在当时的环境中是有能力的。

**认知调整**

青晨需要调整自己的认知，认识到父母吵架是正常的，不必过度恐惧。通过对话，他了解到父母只是吵架，并不会导致死亡。

**恐惧释放**

在梳理过程中，青晨的情绪得到了释放。他意识到，妈妈如果去世，他会感到难过，但不会那么害怕。他更担心的是妈妈意外身亡，因为他觉得还有两件事没有完成：好好对待妈妈和成家立业。

**成家立业**

青晨内心深处害怕妈妈离开，因为他知道这是妈妈最大的愿望。他意识到，成家立业是解决妈妈心病的关键。

**总结**

通过情绪梳理，青晨意识到自己内心的恐惧和需求。他开始积极寻找伴侣，并努力实现成家立业的愿望。
