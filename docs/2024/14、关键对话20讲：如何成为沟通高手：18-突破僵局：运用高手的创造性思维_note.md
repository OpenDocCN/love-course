# 14、关键对话20讲：如何成为沟通高手：18-突破僵局：运用高手的创造性思维

🎼大家好，欢迎来到关键对话，如何成为沟通高手的课堂。今天我们进入第十八讲，主题是如何突破对话中的僵局。

## 1. 僵局的认识

**僵局是关键对话中常见的难点，但并非对话的结束。它是由利益交织、问题复杂导致的正常现象。**

**公式**： 僵局 = 利益交织 + 问题复杂

## 2. 避免僵局

**以下方法可以帮助我们避免进入僵局：**

* **避免情绪化**： 使用雷达扫描系统，关注对方情绪和需求，而非自身情绪。
* **避免人身攻击**： 针对个人攻击不利于对话进行。
* **避免设计可能导致僵局的制度**： 如股权分配、否决权等。

## 3. 化解僵局

**以下方法可以帮助我们化解僵局：**

* **改变环境**： 改变会场、座位、参会人员等。
* **改变谈话内容**： 谈其他问题，增加了解和互相理解。
* **引入第三方**： 如市场研究人员、咨询顾问等。
* **换人**： 换一个更合适的人进行对话。

## 4. 让步与妥协

**化解僵局最终需要双方或多方妥协和让步。以下原则可以帮助我们进行让步和妥协：**

* **不做单方面的让步**： 双方都要让步。
* **让步幅度递减**： 符合心理学和营销学原则。
* **高举共同利益旗帜**： 寻找共同利益，创造新的价值。
* **使用“如果”思维**： 寻找新的可能性，创造新的价值。

## 5. 总结

**本节课我们学习了如何突破对话中的僵局。僵局是关键对话中常见的难点，但并非不可逾越。通过避免僵局、化解僵局、让步与妥协，我们可以有效地解决僵局，达成共识。**
