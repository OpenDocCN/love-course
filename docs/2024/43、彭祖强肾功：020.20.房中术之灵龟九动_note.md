# 43、彭祖强肾功：020.20.房中术之灵龟九动_note

在本节课中，我们将要学习“灵龟九动”这一房中术技巧。该技巧旨在通过一系列特定的动作与呼吸控制，有效延长房事时间，提升双方体验，并帮助男性更好地控制射精冲动。

---

上一节我们介绍了房事中前戏的重要性。本节中，我们来看看如何通过“灵龟九动”的具体动作，在房事过程中进一步调动双方状态。

**灵龟九动**包含九个连贯的动作与呼吸练习。以下是其具体步骤：

以下是第一个动作“灵龟饮水”的做法。
*   在阴茎充血勃起的状态下，手持阴茎在女性阴道口外部画圈。
*   先正向画圈，再反向画圈。
*   此动作可重复进行七七四十九次。

完成“灵龟饮水”后，我们进行第二个动作“灵龟伸腰”。
*   将整个阴茎置于女性阴道外部，用阴茎刺激其阴蒂及整个阴户门口区域。
*   此动作同样可重复七七四十九次。

接下来是第三个动作“灵龟撼动”。
*   将阴茎置于阴道洞口。
*   缓慢地进行试探性插入，随后收回，如此反复。
*   此动作可重复七七四十九次。

完成插入试探后，我们进入第四个动作“灵龟调息”。
*   将整个阴茎完全插入阴道，但保持静止不动。
*   同时进行“阴跷呼吸”法，即吸气时收缩会阴部肌肉。
*   进行三十到五十次这样的呼吸，以降低射精感。

调息完毕，我们开始第五个动作“灵龟摆尾”。
*   使阴茎在阴道内进行上、下、左、右方向的抽插。
*   此动作可重复七七四十九次。

摆尾之后是第六个动作“灵龟搅海”。
*   使阴茎在阴道内旋转画圈。
*   正向与反向旋转可各进行七七四十九次。

完成内部画圈后，我们进行第七个动作“灵龟浅深”。
*   在抽插过程中，采用“行九浅一深”或“行八浅二深”的节奏。
*   浅与深的变化应避免规律，富有变化为宜。

浅深变化之后是第八个动作“灵龟扶首”。
*   在抽插时，用阴茎冠状沟部位重点刺激女性的G点。
*   此动作可重复七七四十九次。

最后是第九个动作“灵龟冲浪”。
*   此时可以进行完整的抽插，节奏与深度应根据女方的喜好灵活调整，可快可慢，可深可浅。

---

在“灵龟冲浪”或整个过程中，若出现强烈射精感，需运用之前课程所学的控精方法。以下是应对射精感的步骤：

若在冲撞过程中产生射精感，首先运用“阴跷呼吸法”。
*   **方法**：通过呼吸配合，收缩会阴肌肉。
*   **操作**：进行三十到五十次呼吸，待射精感下降后再继续抽插。

若几分钟后射精感再次出现，可运用“气运小周天”法。
*   **方法**：深吸一口气，停止抽插，引导气息沿任督二脉循环。
*   **操作**：连续进行三次以上，体会射精感是否消退。消退后继续。

若射精感第三次出现，可运用“气运大周天”法。
*   **方法**：深吸一口气，引导气息沿全身主要经脉循环。
*   **操作**：进行三到五次，体会射精感是否下降。下降后继续。
