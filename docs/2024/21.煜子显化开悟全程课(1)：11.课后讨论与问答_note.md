# 21.煜子显化开悟全程课(1)：11.课后讨论与问答

## 概述
在本节课中，我们将探讨如何在显化过程中进入空心状态，以及如何通过放下执念和提升自信来增强显化能力。

## 进入空心状态
以下是进入空心状态的步骤：

1. **意识分离**：意识到意识与身体是分离的，意识可以独立于身体存在。
2. **观察者视角**：以观察者的视角看待自己的身体和周围环境。
3. **呼吸法**：通过关注呼吸来引导意识，达到空心状态。
4. **音频辅助**：使用特定音频（如963赫兹音频）帮助进入空心状态。
5. **等待法**：通过不断重复“快出来”来引导念头，最终进入空心状态。

## 放下执念
以下是放下执念的方法：

1. **决绝放下**：对于在意的事情，要果断地放下，不再纠缠。
2. **恐惧消除**：放下后，恐惧会随之消失。
3. **选择权**：意识到自己有选择权，可以选择是否继续在意某事。

## 提升自信
以下是提升自信的方法：

1. **抛硬币实验**：通过抛硬币实验来测试自己的信念和信心。
2. **写话练习**：通过写话练习来增强对显化能力的信心。
3. **避免外求**：不要过度依赖他人或外部力量，相信自己内在的力量。

## 总结
本节课中，我们学习了如何进入空心状态、如何放下执念以及如何提升自信。这些方法将有助于我们在显化过程中取得更好的效果。
