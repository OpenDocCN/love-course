# 男性思维提升上帝视角1.0：002.第二节课：人际关系的本质

🎼关系是社会学的概念，其存在是为了建立共识标准，确保社会高效运作。🎼时间原本不存在，是我们制定的标准。

**关系定义**：明确人际边界。🎼模糊的人际边界导致问题，形成暧昧。了解社会共识边界后，知道进退分寸。

**明确立场和行为准则**：知道该做什么，不该做什么。🎼这带来安全感，避免困扰。情商高即尊重边界。

**中国式关系**：个人边界模糊，如逼婚现象。🎼结婚是个人选择，属人权。边界是动态的，共识缺失导致冲突。

**心理认同**：关系存在需心理认同。🎼名义上的朋友可能只是熟人。要求对方履行义务只会引起抵触。

**恋爱关系**：相处模式自然发展到某一步，无需刻意确定。🎼金庸《射雕英雄传》中郭靖与黄蓉的例子。

**关系边界**：关系确立后，双方对边界有共识。🎼熟悉后，拥有自己的相处模式，不再拘泥于标准。

**关系类型**：亲情、友谊、名义关系等。🎼不同关系有不同的心理认同。

**恋爱**：最高境界，紧密连接，暴露人性，了解自我。🎼爱情、性、恋爱、婚姻是独立概念。

**修行**：在处理关系中学习和成长。🎼爱情、性、恋爱、婚姻定义和逻辑上独立。

**总结**：本节课中我们一起学习了人际关系的本质，包括关系定义、边界、心理认同、不同关系类型以及爱情、性、恋爱、婚姻的区别。
