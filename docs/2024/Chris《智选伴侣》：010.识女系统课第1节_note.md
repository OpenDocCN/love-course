# Chris《智选伴侣》：010.识女系统课第1节

**概述**

在情感行业中，有一句话：“男人是一杯红酒，女人是一块奶酪。”本节将探讨女性在两性市场价值（SMV）中的变化及其对择偶倾向的影响。

**SMV与年龄的关系**

- **横轴**：年龄（20-40岁）
- **纵轴**：SMV（两性市场价值，0-100）

- **男性SMV**：从20岁开始最低，25-30岁上升，30-35岁达到顶峰，之后逐渐下降。
- **女性SMV**：20岁时最高，20-25岁为黄金阶段，之后逐渐下降，30-40岁继续下降，40岁时跌至谷底。

**女性SMV高峰期特点**

- **派对时间**：20-25岁，女性倾向于筛选并接触大量男性，探索不同类型的关系。
- **直觉判断**：根据第一印象迅速判断男性吸引力。
- **追求刺激**：不惧怕短暂或非传统恋爱关系，情感升温快。
- **非物质筛选**：更倾向于兴趣爱好和相融性。
- **社交媒体活跃**：利用现代社交渠道认识新人。
- **非固定关系**：接受非传统或非单一伴侣关系，进入和退出关系快。
- **慎重承诺**：对未来话题持有保留态度，排斥过度承诺。
- **生活态度**：以享乐主义为导向，重视休闲活动。
- **个性表达**：彰显独立个性。

**女性SMV下降期特点**

- **理性选择**：考虑潜在伴侣的资源和承诺能力。
- **筛选标准高**：寻找有结构化生活和稳定提供能力的伴侣。
- **寻求稳定关系**：对潜在伴侣有明确和严格的标准。
- **寻求深度**：寻找能提供情感深度和精神支持的伴侣。
- **社会期望**：在意社交评价，相处过程中更端着自己。
- **经验总结**：参考过去关系的经验和教训。
- **生活兼容性**：重视价值观、生活方式和目标的兼容性。
- **传统模式**：倾向于回归更传统的关系模式和家庭价值观。

**总结**

本节课中，我们学习了女性在SMV高峰期和下降期的特点及其对择偶倾向的影响。了解这些特点有助于我们更好地理解女性的心理和行为，从而在情感关系中取得成功。
