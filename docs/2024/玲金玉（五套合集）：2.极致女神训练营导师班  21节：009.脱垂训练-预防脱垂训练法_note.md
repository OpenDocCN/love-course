# 玲金玉（五套合集）课程2：极致女神训练营导师班 第21节：009.脱垂训练-预防脱垂训练法 🧘‍♀️

## 概述
在本节课中，我们将学习针对脱垂问题的具体训练动作。这个动作是上一节预防脱垂训练的进阶版，主要调整了身体姿势，特别适用于已有轻微宫颈脱垂情况的学习者。我们将详细讲解动作要领、呼吸配合及核心训练理念。

## 课程内容

上一节我们介绍了基础的预防脱垂训练，本节中我们来看看如何通过调整姿势来加强效果。

脱垂训练的第二节课内容，其呼吸方法与上一节的预防脱垂训练动作完全一致，我们仅对身体动作进行细微调整。

这个调整尤其重要。如果你已经存在宫颈轻微脱垂的情况，必须练习这个动作。完成这套动作后，我们应先处理预防脱垂以及脱垂已呈现的状况，之后再练习能量弹弹弓和负重训练。

因此，有轻微脱垂的学习者，请务必掌握并练习此动作。

以下是该动作的具体练习方法。

你可以在家中使用瑜伽垫，躺在地上进行。也可以利用沙发，将身体支撑在墙上或沙发上。我个人认为撑在沙发上更为方便。

撑在沙发上之后，必须让身体靠近沙发，使整个盆骨向上抬起。此时，我们的宫颈会向身体内部移动。同时，我们会在阴道内放置能量弹。

能量弹具有重力。在吸气时，利用能量弹的重力现象，通过用力吸气，将轻微脱垂的宫颈收回体内。此过程与上一节课程相同，共进行三次呼吸。

呼吸步骤如下：
1.  舌顶上颚，用鼻子吸气，鼻子吐气。第一次呼吸，想象大阴唇、小阴唇随吸气向内卷入小腹，吐气时慢慢放松。
2.  第二次呼吸，吸气瞬间到达胸腔，吐气放松。
3.  第三次呼吸，吸气到达头顶，吐气放松。

现在，请跟随我一起练习。

在吸气时，请再次注意，我们需要瞬间发力。用大脑想象你的宫颈，仿佛用气息瞬间将其收回体内的感觉。

理解这个意思吗？即在脑海中构想：你的宫颈脱垂了，轻轻一吸，将其吸回身体内部，拉回脱垂的宫颈。

## 总结
本节课我们一起学习了针对轻微宫颈脱垂的专项训练动作。关键在于结合特定的身体姿势（盆骨向上）、使用能量弹的重力辅助，并通过三次有意识的呼吸，用气息和意念引导将脱垂组织收回。请持续练习，以维护健康的阴道内环境。
