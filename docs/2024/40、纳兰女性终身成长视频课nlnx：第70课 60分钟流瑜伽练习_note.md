## 呼吸与热身

### 坐姿调整

1. 选择一个舒适的坐姿，双腿小腿的2/3处交盘。
2. 基础练习者可进行散盘。
3. 双手轻轻拨动臀肌，确保坐骨坐在垫子上。
4. 双手轻轻放在双膝上，感觉坐骨向下扎根，肩膀自然下沉。

### 呼吸练习

1. 觉察呼吸，接纳原本的呼吸状态。
2. 学会做呼吸的观察者，不要着急。
3. 通过练习，让自己的呼吸越来越深远绵长。
4. 关注呼气，让吸气像闻花香一样柔和均匀顺畅。
5. 在呼吸的末端留一点余地，不要贪心。
