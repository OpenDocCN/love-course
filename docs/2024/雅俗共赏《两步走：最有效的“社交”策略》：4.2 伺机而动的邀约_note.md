# 雅俗共赏《两步走：最有效的“社交”策略》教程：4.2 伺机而动的邀约

## 概述
在本节课中，我们将学习如何根据不同的氛围和场景，采取伺机而动的邀约策略，以实现社交目标。

## 伺机而动的邀约

### 氛围好的场景

1. **线上敲定**：在第一次见面后，线上聊天非常顺畅，关系可能朝着恋爱方向发展，可以直接在线上确定关系。
2. **线下共赏**：线下见面两次后，开始聊骚或暧昧发言，可以顺水推舟，进行下一步行动。
3. **实际性行为**：已经确定进行实际性行为，可以直接安排见面。

### 氛围差的场景

1. **话题聊不深**：话题逐渐变少，可能需要拓展新话题或推进关系。
2. **精力转移**：对方可能将精力转移给其他人，需要果断行动。
3. **向下兼容**：在沟通交流层面，总是感觉在兼容对方，需要警惕温柔乡的陷阱。
4. **单方面投入**：在互动过程中，单方面投入增加，可能培养出不好的意识。

### 邀约时间

- 第一次邀约通常在第一次见面后的一周左右进行。

### 邀约策略

- **氛围好的场景**：顺水推舟，自然进行邀约。
- **氛围差的场景**：果断、势如破竹，釜底抽薪。

## 实操案例

1. **莲花山女孩**：通过聊天内容引导话题，最终邀约见面。
2. **张博士**：通过分享日常，逐步推进关系，最终邀约见面。
3. **其他案例**：根据不同场景，采取不同的邀约策略。

## 总结
本节课中，我们学习了伺机而动的邀约策略，以及在不同场景下的应用。通过掌握这些技巧，我们可以更好地实现社交目标。
