# 乐嘉7套：5.乐嘉性格色彩全能训练营（入门）【勇略】：10 从表达看懂一个人的性格_

## 概述
在本节课中，我们将学习如何通过观察一个人的表达方式来了解其性格特点。

## 1. 天生不会说话的人？
### 1.1 性格与表达
每个人的性格特点都会影响其表达方式。例如，红色性格的人喜欢用故事表达，蓝色性格的人注重逻辑，黄色性格的人直截了当，绿色性格的人亲和力强。

### 1.2 性格优缺点
每种性格都有其优点和缺点。例如，红色性格的人感染力强，但可能话多且无重点；蓝色性格的人逻辑性强，但可能过于严肃；黄色性格的人直截了当，但可能过于强硬；绿色性格的人亲和力强，但可能缺乏明确立场。

## 2. 不同性格的表达特点
### 2.1 红色性格
- **优点**：感染力强，有画面感。
- **缺点**：话多且无重点。

### 2.2 蓝色性格
- **优点**：逻辑性强，条理清晰。
- **缺点**：过于严肃，缺乏感染力。

### 2.3 黄色性格
- **优点**：重点明确，简练不啰嗦。
- **缺点**：过于强硬，不关注他人感受。

### 2.4 绿色性格
- **优点**：亲和力强，让人感觉舒服。
- **缺点**：缺乏明确立场，不利于事情推进。

## 3. 如何说出别人想听的话？
### 3.1 四句话秘诀
- 红色性格：故事
- 蓝色性格：道理
- 黄色性格：结果
- 绿色性格：什么都愿意听

## 总结
本节课中，我们学习了如何通过观察一个人的表达方式来了解其性格特点。了解性格特点有助于我们更好地与他人沟通，提高沟通效果。
