# 恋爱脑反转课：32节让男人离不开你 - 第30节 撒娇吸引术：激发男人对你的怜爱情绪

大家好，我是火氧。上一节课我们讲解了如何在关系中表达需求，以及如何让男人满足你的需求。那么，本节课我们将探讨撒娇的技巧。因为有时候，满足需求需要通过撒娇来实现。

**撒娇的必要性**

我发现，在现实生活中，许多女性不会撒娇，她们担心撒娇会让自己显得软弱。然而，在两性关系中，不应有太多的规矩。任何能促进关系的行为都是可行的，包括撒娇。

**克服心理障碍**

首先，要克服对撒娇的心理障碍。不要担心撒娇会破坏关系或让你丢脸。撒娇只是表达情绪的一种方式，并不代表你不独立。

**撒娇的本质**

撒娇的本质是向你所爱的人表达“我很爱你”。记住这一点，你才会更加发自内心地撒娇。

**撒娇的时机和场合**

撒娇要选择合适的时机和场合。例如，当对方强大时，你可以表现得柔弱一些；当对方脆弱时，你可以表现得坚强一些。撒娇也可以通过拥抱、淘气、幽默和恶作剧等方式表达。

**避免在错误的时间和场合撒娇**

不要在对方能量低或情绪不佳时撒娇，因为这可能会适得其反。同时，要确保对方能够满足你的需求。

**保持分寸**

在撒娇时，要确保你的需求在合理范围内，让对方能够轻松满足。不要让对方感到压力过大。

**总结**

在本节课中，我们学习了撒娇的技巧和注意事项。撒娇是一种表达爱的方式，但要注意时机、场合和分寸。通过撒娇，你可以更好地与伴侣建立亲密关系。
