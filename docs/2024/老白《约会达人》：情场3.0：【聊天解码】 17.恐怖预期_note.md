# 老白《约会达人》：情场3.0：【聊天解码】 17.恐怖预期_note

在本节课中，我们将要学习聊天中的一个核心技巧——“预期”。我们将了解预期的定义、重要性，并分阶段学习如何在舒适感、暧昧及走心阶段，恰当地向女生植入预期，以提升关系发展的流畅性与成功率。

## 概述：什么是预期？

聊天可分为三个主要阶段：舒适感阶段、暧昧阶段以及走心阶段。这三个阶段相互联系、相辅相成，而非完全割裂。

上一节我们介绍了聊天的基本阶段，本节中我们来看看连接这些阶段的关键技巧——预期。

如果跳过某个阶段直接进入下一阶段，往往会导致失败。例如，舒适感未建立就进入暧昧，女生会感到唐突和不靠谱；暧昧未做足就进入走心，则会显得过于突兀。

预期，简单来说，就是向女生描绘一个**未来与你相关的画面**。这样做的目的是，当未来你们真的去做这件事时，她不会感到尴尬和突然，反而会觉得“这件事我们曾经说过”。

在聊天中植入预期，能**提升女生对某件事的兴趣**，并**增强日后行动的流畅性，降低女生的抵触心理**。预期贯穿于每一个聊天阶段，不仅是为了方便约会中的亲密行为，更是每个阶段都必须进行的铺垫。

## 第一阶段：舒适感阶段的预期

在舒适感阶段，聊天以**相互熟悉**为主要目的。应尽量以幽默风趣的方式探寻对方的基本信息，如职业、爱好等。

以下是该阶段常见的几种预期类型：

1.  **植入见面可能性**：当女生提到某个共同兴趣时，可以自然引出未来见面的画面。
    *   **示例**：女生说“我超喜欢吃火锅，而且超能吃。” 你可以回复：“别吹牛，以后见面的时候，我可能会点一大桌子菜，到时候别自己打自己脸。”

2.  **暗示延续聊天**：当女生表现出某些特质时，可以暗示未来需要更多交流。
    *   **示例**：女生说“我比较慢热。” 你可以回复：“那看来以后得多找我聊聊天了。我平易近人的小秘密，只和你一个人说过。”

3.  **侧面建立自身框架**：通过评论其他现象，间接展示自己的价值观和标准。
    *   **示例**：女生说“闺蜜的男朋友又给她买了个包包。” 你可以回复：“我每次看到女生发什么‘好男友标准’——不打钱、不转账，只会说甜言蜜语已经过时了。我只想说，估计这些女孩也找不到这样的男人，因为好男人也很抢手。整天围着你转的那些，基本上都是屌丝。”

该阶段预期的核心方向是：向对方植入 **“我们聊天会延续下去”** 以及 **“我对聊天对象有标准”** 的概念。

## 第二阶段：暧昧阶段的预期

进入暧昧阶段，目标是**升高关系**。此时，曲解和推拉是经常使用的技巧，可以很好地带动女生的情绪。

以下是该阶段常见的预期类型：

1.  **委婉表达兴趣**：通过曲解对方问题，测试女生态度。
    *   **示例**：女生问“你喜欢什么样的女生？” 你可以回复：“你再努力一下，我就快喜欢上你了。”

2.  **植入未来生活画面**：抓住女生话语中的细节，描绘带有亲密感的未来场景。
    *   **示例**：女生说“我昨天不小心把手切破了。” 你可以回复：“拍个照给我看看严重不严重。”（女生发照片后）“怎么这么不小心，手伤了以后还怎么帮我洗内裤洗袜子？”（加入推拉）“逗你玩的，明天带你去吃好吃的补补。”

3.  **植入见面后的亲密预期**：在约会前，为可能的牵手、接吻等行为做铺垫。
    *   **示例**：约好见面后，女生说“明天见。” 你可以回复：“先说好，明天我会收拾得很帅，你到时候不能因为我长得帅，就趁我不注意偷偷牵我手。” 女生若否认，你可接着说：“反正你要是太漂亮了，我可是会忍不住牵你手的。”

4.  **一本正经地植入性话题**：应对女生的“废物测试”，隐晦表达自己的性观念。
    *   **示例**：女生问“你有没有约过炮？” 你可以回复：“我不约炮，我只制造爱情。” 若女生追问，可接着说：“也许我外表不羁，但内心脆弱。我希望倒下去时，是在一个充满爱的怀抱里，而不是一个我想立刻赶走的人。”

在这个阶段，有一个重要法则：**只要女生不是明确拒绝，就等于默认**。在约会中，可以更大胆地推进关系。

## 第三阶段：走心阶段的预期

走心阶段的核心是**情感深度连接**。预期的方向是描绘**美好、平淡而真实**的未来共同生活画面，这能触动女生内心柔软的部分。

以下是该阶段常见的预期类型：

1.  **描述与另一半的未来生活**：当女生试探未来关系时，具体描绘日常生活细节。
    *   **示例**：女生问“你会对你未来的女朋友很好吗？” 不要直接回答，可以说：“我要的爱情很简单，一起手拉手逛街、吃饭、看电影，一起窝在家里看剧。我做饭你夸好吃，分享喜怒哀乐。偶尔吵架，我会哄你，而你很好哄。”

2.  **描述与另一半的未来家庭**：将女生规划进更长远的人生蓝图，如养育孩子。
    *   **示例**：女生问“你喜欢男孩还是女孩？” 你可以回复：“我想要一个男孩一个女孩，男孩是哥哥，女孩是妹妹。我们一家四口穿家庭套装出门，哥哥照顾妹妹，我来照顾你。我做饭你洗碗，两个小宝贝帮我们拿碗…我们看着他俩慢慢长大，他们守着我俩慢慢变老。”

这样的画面**真实可感，易于实现**，能给予女生强烈的安全感和归属感。

## 总结

本节课中，我们一起学习了“预期”这一核心聊天技巧。

首先，我们明确了在聊天中向女生植入预期的重要性：它能**提升兴趣、增强行动流畅性、降低抵触心理**，并且需要在**任何一个阶段**持续进行。

其次，我们分三个阶段学习了如何操作：
*   **舒适感阶段**：以轻松幽默的方式植入延续聊天和未来见面的可能性。
*   **暧昧阶段**：运用曲解、推拉等技巧，委婉表达兴趣并植入亲密预期，记住“不拒绝即默认”的法则。
*   **走心阶段**：描绘美好、真实、平淡的未来共同生活画面，进行情感上的深度绑定。

最后，关键原则是：**必须分阶段做适合的预期，切勿跨越阶段**，否则极易引起女生反感，导致关系倒退。

掌握好植入预期的时机与方法，能让你的聊天如虎添翼，自然流畅地推动关系向前发展。
