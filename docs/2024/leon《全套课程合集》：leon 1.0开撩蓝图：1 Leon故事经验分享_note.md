**核心准则公式：**
`遇到不敢上的目标 = 必须上`

正是这一点，让我不断突破自己。有的人出去实战，可能在外面逛一天只出手两三次。遇到真正心动的目标时，却会找各种借口，比如“她走得太快了”、“她旁边有朋友”、“好像不是我最喜欢的类型”。因此，进步很慢。

这是我分享的一个价值巨大的经验。你越是不敢上的目标，越要逼自己上。这是我和99.9%的人做得不一样的地方，也是我成功的一个巨大秘诀。

所以，最后一点是：跟比你牛逼的人玩。如果你遇到了这样的人，放下你的面子，虚心地向他请教。人家花了时间精力，在这个领域有所建树（不单单是把妹，任何领域），如果你想要这个东西，无论是英语、创业、运营还是销售，如果他比你牛逼，如果他很厉害，你就应该请他吃饭，应该跟他去玩。

其实很多人有机会改变自己、取得进步，但他们就是把握不到机会。比如，说实话，你们和Leon接触，和我的老师接触，你们就能改变自己。但是你没有那个野心，没有那个魄力。你会想：“我付这个钱学习，我付这个钱线下见面，有必要吗？我能改变吗？”

亲爱的，一堆人想跟我们一起吃饭，一堆人想认识我们。你在犹豫“有没有必要，会不会改变”的时候，别人已经得到这个机会了。当你在犹豫时，别人已经在尝试了。为什么呢？因为你觉得不知道有没有用，但别人在尝试。你连这点魄力都没有，怎么改变，怎么成长？

我也不知道我努力有没有用，但我还是在努力。为什么呢？因为我太想成功了。如果方法没用，那我就继续努力；如果这个人不行，那我就换下一个人跟着他学。所以，这是一个思维导向：一定要跟着牛逼的人一起玩。

因为牛逼的人很有能量，他认识很多有能量的人。你认识一个牛逼的人，就能认识一群牛逼的人。所以，你想认识那样的人，你自己的性格一定要做一个善良、正直、诚实、懂得感激、愿意付出、懂得如何提供价值的人。这样，牛逼的人才愿意跟你一起玩。

所以第四点非常重要：你要想尽一切办法去认识牛逼的人。只要你认识这样的人，他就能带着你飞，带着你玩，你就能在这个领域快速进步。如果你想把妹，跟着我们玩，你就能实实在在得到结果。如果你会想“会不会没用呢”，那是不可能的。只要你跟着我们玩，你绝对能够改变。
