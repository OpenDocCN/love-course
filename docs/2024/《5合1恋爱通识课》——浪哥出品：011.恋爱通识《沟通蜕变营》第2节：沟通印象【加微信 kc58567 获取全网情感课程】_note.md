# 《5合1恋爱通识课》——浪哥出品：011.恋爱通识《沟通蜕变营》第2节：沟通印象

欢迎来到沟通蜕变营的第二节。今天主要讲解沟通印象。在聊天过程中，女人会本能地规避带有目的性和需求感的男人。以下是从女人的角度出发，将男人分为六种类型，帮助大家在线上聊天时进行判断。

### 1. 奇葩男

- 初步交流中，感觉你脑回路不正常，在不恰当的时候说出出格的话。
- 查户口式聊天或炫耀自己认为很牛逼的地方。
- 与你谈论问题时，根本聊不到一个频道，自顾自地说一堆废话。

### 2. 猥琐油腻男

- 初期沟通时，没有对你产生吸引，但你表现出强烈的需求感。
- 经常发土味情话、要照片、开黄腔等。

### 3. 老好人，备胎

- 聊天还不错，接受你持续提供的价值。
- 但不会把你当成潜在的恋爱对象。

### 4. 情人定位

- 开放式聊天，充满暧昧气氛。
- 双方有吸引，有目的性，可以达成情感关系或其它目的。
- 不想让生活有太多交集，对未来评估也没有太多考虑。

### 5. 情侣定位

- 有目的性地建立男女朋友关系，有吸引、可得性和对未来的展望。
- 符合社会道德观念，男人应该对女人忠诚，愿意持续投入。

### 6. 婚姻定位

- 女人把你归类为婚姻对象，潜在的婚姻对象。
- 双方生活、人脉资源、交际圈和家庭等方面有长期可能性。

### 总结

通过聊天，女人对你的定位会影响你们的关系。建议将自身定位为开放式关系，循序渐进地进入男女朋友关系，避免给双方带来压力。
