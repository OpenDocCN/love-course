# 安老师社恐训练营：07-幡然醒悟：跟任何人相处都不紧张的秘密

同学们大家好，很多人在与他人相处时会感到不自在和紧张。通常情况下，人们在面对重要事件时，如考试、面试或公开演讲，会感到紧张。但对于社交恐惧症（社恐）患者来说，紧张的情况可能更为普遍。

例如，在遇到陌生人或与不熟悉的人交谈时，可能会感到手不知道放在哪里，表情不自然，肢体动作紧张。所有这些问题如何解决？其实，如果前面的课程按照要求进行，基本上就可以解决。前面的课程是从根本上解决社恐问题。但本节课将提供几个常用的、非常重要的技巧，帮助大家在与人相处时不再紧张，能够从容自在地生活。共有6个技巧。

### 技巧一：降低期待

降低期待是至关重要的。在人际交往中，我们对自己的期望过高，或者总是抱有期待。正是因为这样，导致我们产生各种紧张的行为，担心自己的表现，担心对方对我们的看法，担心结果不好，担心自己的穿着、长相、学识等受到他人的嘲笑等。

我们之所以担心，是因为总是想从对方那里获得所谓的认可。在人际交往中，我们不必给所有人留下好印象，但也要记住，并不是说要给所有人留下坏印象。我们不要理解错了，不是非黑即白的思维。

这里说的是在人际交往中，我们不必太在意对方对我们的评价。如果你很在乎，你会非常拘谨和紧张。人际交往的一个很大障碍就是过分拘谨。另外，我们要对结果降低期待。你不必在别人面前表现得完美，能相处就相处，不能相处就不相处。要放松自己，只有放松自己，你的人际关系才能变得顺畅。

### 技巧二：把对方看成是你熟悉的人

不要高估对方，不用考虑对方的身份、地位、年龄，就把他看成是你的老朋友。老朋友自然感情会更近一点，这样你们相处时才会自由自在，不会紧张，不会拘谨。

如果我们考虑对方的身份、地位、年龄，自然而然会有一种比较，而一比较就容易产生自卑感。比如你去相亲时，看到对方无论是穿着打扮还是工作上的身份，都是上市公司高管。如果你考虑这些，你自然会感到紧张，说话会小心翼翼。要记住，只有自己从容放松的时候，你才有可能表现好。

### 技巧三：提前做好准备

有人可能会疑问，不是说降低期待吗？不在乎结果吗？为何还要提前准备？我们所说的出发点是削弱紧张感，不是摆烂。提前做好准备是什么？我们要做最好的准备，但并不一定要期待最好的结果。你可以提前准备好你说话的思路，想好再说。很多人紧张是因为没有想好就说说的时候又忘了说什么了，进而造成紧张。所以在表达时，要逻辑清楚，言之有误。在有了准备之后，你自己也会有信心。

### 技巧四：私下多锻炼

与别人相处时，紧张的最大的表现是眼神不敢看对方，总是飘忽不定，表情不自然，还有肢体动作，有时候紧张的手都不知道往哪里放。所以平时的时候，自己一个人在家就可以去锻炼，主要从三个方面：眼神、表情、肢体动作。当你一个人在家的时候，可以把家里的任何一件物品当成是人，你跟他聊天，聊天的时候你的表情是什么样的？眼神是什么样的？肢体动作是什么样的，直到找到一种你熟悉的感觉，放松的感觉。记住它。比如你在家跟你的冰箱说：“嗨，哥们儿，你挺能装啊，肚子里全是货，给我一瓶饮料来，你不会介意吧？”就是类似的练习。大家不要觉得好像是神经病一样，它只是一种前期的训练。当你训练好了之后，无论是你的表情、眼神、肢体动作，在与人相处的时候都会让人感觉到很舒服，你自己也不会紧张，是那种很从容很放松的感觉。

### 技巧五：适应不确定性

很多时候我们紧张是因为很多事情无法确定，无法把控未来的不确定性太多了。但是我们可以想想，你在过去担忧的很多事情，当时是很焦虑紧张的。那现在再来看的话，过去让你焦虑紧张的事情是否发生了呢？同理，现在我们也可以为未来去焦虑。但是呢，你为未来的一些事情紧张，你要考虑这些紧张、焦虑的事情可能也不会发生。既然有很多的不确定性，那么我们就摆了，随而安行吗？当然不是的，我们能做的是什么？抓住当下，把我们当下的事情做到目前我们认知之内的最好就可以了。既问心无愧，又不会紧张。

### 技巧六：不怕出丑

不怕出丑，不是让你故意出丑。这点要明白，每个人都有出丑的时候。出丑的时候会感到瞬间的紧张脸红，因为怕别人嘲笑自己嘛，怕别人关注自己嘛。出丑并不可怕。出丑这件事情本身是正常的。只不过出丑之后的表现才可能引起别人的嘲笑。另外，可以尝试退货联系。什么是退货联系呢？就是平时你去超市买东西，拿着东西去前台结账的时候，发现东西很贵，你后面又有很多人排着，你又不好意思不买，你就忍受的价格买回家了。现在可以试着这样做。同样的东西拿到前台结账的时候，当售货员跟你说了这个价格之后，你就说：“哎呀，太贵了，买不起，这个不买了。”此后不是此时你后面有很多人会排的队，这个很多人看着你，你不要慌，不要紧张，不要去想其他的，你要顶住你内心的压力。为什么说要顶住内心的压力呢？因为对于没有社恐的人来说，这只是很平常的事情。我就是显贵，我不想买了怎么了，而且呢你后面排队的人也不会有什么奇怪的想法。但是有社恐的人呢会很在意别人的看法和眼光，会胡思乱想，觉得自己出丑了，觉得别人肯定是评价自己。哎呀，就是怎么评价呢？就是觉得自己很穷又又装之类的这个东西，对吧？反正内心心非常多。进而呢会给自己造成非常大的心理压力。平时的时候可以去这个尝试退伙练习，锻炼自己。另外呢要敢于冲突，冲突并不是剑拔弩张，吵架之类的。而是不同的意见碰撞，有社恐的人呢往往不敢表达自己真实的想法，怕引起冲突，怕有不同的意见，怕别人对自己有这个不好的反应，尤其是大庭广众之下，更不敢。比如你坐公交车，司机这一站，并这个司机这一站没有停车，你也不敢喊停。你想着要不下一站下车算了。其实你可以直接跟司机说的，到了站为什么不停车呢？你可以听听司机的理由，看看到底为什么不停车。平时在生活中也是啊要敢于发表自己的意见，要敢于面对冲突，只有敢于面对冲突，你的内心才会变得很强大。

本节课就到这儿了，大家可以复习回顾一下重点，做下笔记之类的，努力提升自己，加油。
