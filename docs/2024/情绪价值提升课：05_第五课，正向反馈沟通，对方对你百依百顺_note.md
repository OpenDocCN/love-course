# 情绪价值提升课：05_第五课，正向反馈沟通，对方对你百依百顺

大家好，欢迎收听本次课程。本节课将讲解正向反馈沟通的重要性。人与人的关系破裂往往源于沟通问题，尤其是爱人之间。以下是一些常见场景。

**场景一**：男人凌晨一点才回家，女人很生气。女人质问男人是否还关心家庭，男人解释是陪客户。女人反驳，认为男人在外留恋。男人感到被误解，情绪失控。

**问题分析**：

1. **不客观描述**：沟通时夹杂个人情绪，随意贴标签，扭曲表达。
2. **上纲上线**：将个别行为放大，如男人晚归等同于不负责任。
3. **以偏概全**：夸大事实，如每次都晚归。
4. **情绪化表达**：抱怨、指责，而非客观描述问题。

**解决方案**：

1. **包容理解**：理解对方情绪，耐心等待情绪平复。
2. **提醒对方**：以幽默的方式提醒对方沟通方式。
3. **明确问题本质**：清晰表达问题，避免情绪化。
4. **疏解情绪**：安慰对方，缓解情绪。

**沟通技巧**：

1. **冷静沟通**：急事冷静处理，避免情绪化。
2. **幽默提醒**：用幽默方式提醒对方。
3. **谨慎评价**：不轻易评价，避免误解。
4. **事实为依据**：避免无端指责。
5. **履行承诺**：确保承诺可实现。
6. **避免伤害**：不使用言语伤害他人。
7. **保护隐私**：不随意谈论他人。
8. **倾听他人**：谦虚接受他人意见。
9. **共同商讨**：情侣或夫妻共同处理问题。

**总结**：

沟通是维护关系的关键。通过正向反馈沟通，我们可以避免争吵，增进感情。好好说话，让彼此更加亲密。😊
