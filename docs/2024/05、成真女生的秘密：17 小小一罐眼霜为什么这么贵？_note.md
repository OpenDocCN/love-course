# 课程05：成真女生的秘密：为什么一小罐眼霜这么贵？ 💰

在本节课中，我们将探讨眼霜的核心作用、其价格高昂的原因，并通过具体品牌和价格实例来加深理解。

## 眼霜的基本作用

上一节我们介绍了课程主题，本节中我们来看看眼霜的具体功能。眼霜是用于眼睛周围皮肤的护肤品，主要作用是保湿。护肤的许多环节本质就是保湿。眼霜可以防止眼睛周围出现细纹，因为眼睛周围皮肤的角质层非常薄。此外，人们每天都要频繁眨眼。

因此，对于女生而言，眼角周围皮肤非常容易长皱纹。眼睛一旦长出皱纹，会立刻显得苍老、不好看。所以大部分女生都害怕眼睛部位长皱纹和细纹。

于是，她们会使用眼霜来进行保养和保湿。

## 眼霜的价格定位

了解了眼霜的基本功能后，我们来看看它的市场定位。眼霜的价格在所有女性护肤品中属于单价最高档。比较贵的品牌产品，有时仅十几毫升，一小管就要七八百元。

甚至上千元的产品也存在。如此小的容量却价格高昂。

以下是两个具体的品牌与价格示例：

*   **雅诗兰黛小棕瓶**：容量为15毫升，价格大约在500元左右。
*   **海蓝之谜眼霜**：容量同样为15毫升，一小瓶价格大约需要一千七八百元。

当然，一个品牌旗下通常不只有一种眼霜，可能包含其他系列，其中会有更高端的产品。

例如，雅诗兰黛品牌下可能就有售价约2000元的15毫升装眼霜，以及约1000元的产品，分为不同的档次。

本节课中我们一起学习了眼霜的保湿抗皱功能，并分析了其作为高端护肤品价格昂贵的原因，还了解了几款具体产品的市场定价。理解这些有助于我们更理性地看待护肤品的价值。
