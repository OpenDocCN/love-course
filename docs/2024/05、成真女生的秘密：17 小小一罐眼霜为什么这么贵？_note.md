# 05、成真女生的秘密：17 小小一罐眼霜为什么这么贵？_note

在本节课中，我们将探讨眼霜的核心作用、其定价高昂的原因，并通过具体品牌和价格实例进行分析。

眼霜是专门用于眼周区域的护肤品，其主要功能是**保湿**。护肤的基础之一就是保持皮肤水分。眼霜可以防止眼周细纹的产生，因为眼周皮肤的**角质层非常薄**。此外，我们每天都会频繁眨眼，这进一步加剧了该区域的皮肤负担。

因此，对女性而言，眼周是极易生长皱纹的部位。眼睛一旦出现皱纹，会立刻显得苍老、不美观。所以，大多数女性都非常担忧眼周长出皱纹或细纹。

于是，她们会使用眼霜来进行保养和保湿。眼霜的价格，可以说是所有女性护肤品中单价最高档的。一些较贵的品牌，有时仅**十几毫升**的一小管，售价就高达七八百元。

甚至上千元的眼霜也存在。例如，比较热门的**雅诗兰黛小棕瓶眼霜**，容量为15毫升，价格大约在500元左右。

我记得曾有一位女性向我展示她梳妆台上的化妆品，她指着那瓶很小的眼霜说，这一瓶的价格抵得上后面好几瓶其他护肤品。这让我对“小棕瓶”印象很深。但实际上，还有更昂贵的眼霜。

比如**海蓝之谜（La Mer）** 的眼霜，同样15毫升的一小瓶，价格大约需要一千七八百元。当然，一个品牌旗下通常不只有一款眼霜，可能还有其他系列，定位更为高端。

例如雅诗兰黛品牌下，可能就有售价约2000元的15毫升眼霜，以及约1000元的不同档次产品。品牌通过不同系列和成分组合来划分产品档次。

---

本节课中，我们一起学习了眼霜的基本功能是保湿，并了解了其价格高昂的原因在于针对眼周皮肤薄、易衰老的特性进行专项护理，同时品牌溢价和高端成分也推高了成本。
