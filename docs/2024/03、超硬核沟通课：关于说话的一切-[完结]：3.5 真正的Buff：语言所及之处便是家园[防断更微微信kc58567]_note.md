# 课程03.5：真正的Buff：语言所及之处便是家园 🏠

在本节课中，我们将深入探讨如何通过语言和思考，获得面对问题的主动性，并构建属于自己的精神家园。我们将理解“区分问题”的深层含义，以及如何通过建立清晰的观念系统，在复杂的世界中找到自主与笃定。

---

上一节我们探讨了语言与经验的关系，本节中我们来看看如何运用这种认识，获得一种宝贵的主动性。

这种来来回回区分真假问题的过程多了，你才能在另一个维度上区分问题，即把问题区分为**你想解决**的和**不想解决**的。这使你获得一种宝贵的主动性。

例如，我在语言世界中，借助各种材料澄清了社交问题的本质，也理清了自己心理世界中许多含混的经验。但我就是**不想解决**现实世界中的社交问题。因为我认识到人格具有多种特质，实现价值的路径也是多元的。这些认识曾是我的盲区。

我的陈旧观念告诉我：不擅长社交，你就是一个没有价值的人；积极的社交是人生不得不做的事情。我之所以敢说那是盲区，是因为我以一种比以前更有序的方式整理了我的经验。

在这样的经验系统和观念系统中，消极社交是合情合理的事情，是对我个人而言合情合理的选择。即便我知道参与社交活动有助于我的人生和职业发展，我也能坦然选择忽视这个问题。这也是一种解决，但并非掩耳盗铃，而是我**主动选择的忽视**。所以，有时想清楚一个问题，本身就已经解决了这个问题。

这种区分“可以解决”与“不可以解决”的宝贵主动性，其最宝贵之处在于：对于那些我们认为可以解决的问题，会生出一种**盲目的自信和乐观**。这种自信和乐观，与我们之前讨论过的科学主义的自信与乐观特别相似。

我们认为，但凡是我们想要解决的问题，只要花功夫、下精力，就一定能解决。之前我们是以批判的态度去讨论科学主义的无限进步论和乐观主义。但从个人层面而言，我们太需要这种盲目的自信和乐观，去对冲我们那些盲目的自卑和悲观了。

我现在的思考让我能武断地说：一个人如果拥有问题意识，再配上这种盲目乐观，他一定不会混得太差。也许有一天你终会撞上天赋和运气的那堵墙，但这**绝不是问题**。他已经有能力活得足够好了。大多数人根本连这堵墙都没见过，就直接躺平了，躺平后还抱怨这堵墙又高又硬，撞得自己好疼。

更重要的是，这种信念让我们在过往的经历中，面对天赋和运气的局限时，不会自怨自艾太久。这个“Buff”其实是一种**盲目的信念**，它让我们觉得自己永远有的选，永远拥有一种**确认的选择感**，从而产生一种深层次的自信。

我们用一种被习得的笃定心态去面对世界，因此依然能抵达舒适区，只不过它和我们想象的不太一样。

有人会觉得这个讨论跑题太远，也有人会觉得恰到好处。这取决于你和我是否有类似的现实障碍与心理困惑。但作为作者，我必须这样写，因为在做概念推演时，逻辑必然会带你来到这个地方。而且真实经验告诉我，只有这样写，才是在说真实、真诚的东西。

我敢这样写，是因为这些理解和经验，也特别符合一些久经考验的厉害理论模型，比如著名的**自我决定论**。我经常引用它，因为它特别符合我自己的经验感受。

自我决定论认为，我们的幸福感来源于三种基本心理需求：
*   **归属感**：在与人的关系中获得。
*   **胜任感**：在与事的互动中获得。
*   **自主感**：在与整个世界的互动中获得一种彻底确认的自主感。

我们不妨问，最理想的人生境遇是什么样子的？我现在几乎可以断言：是在一种**有所归属的自主状态中，去遇见不确定性**。你会发现这三个要素相互矛盾，但正是这种综合，让我们获得了最正向、最积极也最丰富的人生体验。

我们面对的风险和惩罚，其上限是清晰的；但我们可能收获的奖赏，却是上不封顶的。这有点像一种**绝对安全状态下的冒险**。我们在小时候的游乐园、少年时期的网吧、成年后的旅行中，经常能收获这样的体验，但它们都只是商业社会的人造景观。我们要想办法把自己的真实生活也变成这个样子。

我们前面引用了海德格尔的“语言是存在的居所”，并用“怪圈”这个悖论意象描述了居所的形态。如果你愿意，不妨这样想：如果语言是我们的居所，我们通过语言进入世界，那我们就可以用语言最纯粹的方式去发问——问“远处是什么？”，再问“远处的远处是什么？”。我们的语言所及之处，都会成为我们的圈地。最终，这个语言的居所会无限扩大，成为我们世界本身。

在这个过程中，我们居然可以为之取乐，闻得忘乎所以。偶尔会遇到从小地方来的人，他们总是一脸严肃地谈论着一些无聊的事情，你稍微补充两句，他们就会惊为天人：“哇，你说到点子上了！”

---

下面让我们从问题的角度，回顾一下这些内容。

首先，把我们聚到一起的初始问题是：**在现实世界中，我们应该如何把话说到点子上？**

这个问题把我们引向一个更抽象但也更具体的问题：**描述经验的语言与被描述的经验本身之间的关系。**

为了处理这个问题，我们引入了**三个世界（现实世界、心理世界、语言世界）** 的概念模型，并推演出了“现实 → 心理 → 语言”逐级简化的认识机制。

接着，将这个认识机制作为新的前提和框架，问题就具体化为：**为什么有些话能克服这种简化，准确地映射出那些被简化、被遮蔽的经验？**

我们通过一个类比，引出了**经过思辨的框架级概念**。它们是突破旧框架的新框架，帮助我们整理复杂的经验，比如刚刚提到的“三个世界”。

此时，我们关心的问题变成了：**人类知识系统中那些宝贵的框架级概念从何而来？我们又应该如何思辨才能得到这些框架级概念？**

这两个问题可以合在一起来回答：**通过问题。一切思考都是从问题开始的。** 我们在“三个世界”的框架下重新理解了“问题”。问题关联着“陈见”和“盲区”，并且贯通这三个世界。我们在语言世界中，基于真问题思辨得来的好答案，帮助我们调和了现实世界与心理世界的不对称。

最后，我们给出了一个比较终结的答案或观念。从个人经验出发，我试图让大家理解乃至接受这样一种人生观和问题观：**所有的真问题都是未来的假问题。问题是解决不完的。** 我们关注问题、解决问题，是为了获得一种与世界互动的方式，准确说是为了习得一种与“问题”、“障碍”、“麻烦”、“挑战”这类经验范畴互动的心智模式。

之所以要说出这样的观念，是因为我们意识到了表达乃至思考的双重困难：
1.  **第一重困难**：把话说明白的本质是要交代清楚因果关系，但预设的框架和前提会遮蔽这种关系。
2.  **第二重困难**：当我们寻找更好的框架来克服第一重困难时，会发现“因外有因，果外有果”。我们谈论的永远是某个局部。随着阅历增长，你总能看见更大的局部，不断受启发，又不断觉得“当初幼稚”。这迫使我们必须在哲理层面达成一种共识：所谓因果、所谓更大的局部，都是一种人为的建构，一种符合心理-语言结构的因果造型而已。

我特别喜欢“**意义炼金术**”这个说法。这里的“意义”指的是那个可以无限演绎的意义；“炼金术”指的是这个意义无法在物理层面被建构出来，其中一定有某些神奇的“化学反应”在发生。

当你在三个世界中来回穿梭，在语言世界中反复琢磨，就会发现语言世界充满了“Bug”，它本身就是一个巨大的Bug。同样一件事情可以从无数个侧面去理解。所谓的理解和领悟，只是我们在语言世界人为建构的结果，具有明显的任意性和随意性。

同一个对象、相似的经验，有无数种描述和表达的方式。这种任意性和随意性的极致表现，就是所谓的**诗文和诗意**。所以，诗是真正的人类之光。

人类发明了**数学和逻辑**这样的形式语言来消除这种任意性，获得了绝无歧义的命题和必然为真的推理。这是另外一个频谱的人类之光。语言在这个时候变成了**真理指南针**。

你们这些理科生用指南针去发现定理，我们这些文科生用炼金术来提炼意义。我们都是真实世界的黑客，我们都有光明的未来。

**本节课总结**：
本节课我们一起学习了如何通过语言和思辨获得面对问题的主动性。我们认识到，区分“想解决”与“不想解决”的问题是一种关键能力，它能带来宝贵的自主感。通过构建清晰的观念系统（如自我决定论），我们可以在矛盾与不确定性中找到归属、胜任与自主的平衡，从而将生活本身构建成一种“安全地冒险”。最终，语言不仅是工具，更是我们拓展认知边界的居所，让我们得以在意义的海洋中航行，无论是通过诗意的炼金术，还是逻辑的指南针，都能抵达属于自己的家园。
