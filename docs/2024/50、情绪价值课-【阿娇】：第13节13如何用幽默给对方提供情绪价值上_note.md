# 情绪价值课-【阿娇】：第13节 如何用幽默给对方提供情绪价值

大家好！娇姐又来和大家分享啦。今天我们来探讨如何通过幽默来为他人提供情绪价值。

## 概述

幽默是一种调节氛围、拉近人与人距离的有效方式。在亲密关系中，幽默更是不可或缺的元素，它能增加乐趣，让双方更加轻松愉悦。

## 核心概念

**公式**： 情绪价值 = 幽默 + 沟通

## 6种幽默方式

以下是亲密关系中的6种幽默方式：

1. **偷换概念**：将事情从原本的层次转换到另一个不相干的层面上，达到意想不到的效果。
2. **欲扬先抑**：先通过贬低别人，再发现原来是在赞美，制造反差效果。
3. **反夸**：当对方夸你时，不要谦虚，而是学会反夸，提升幽默感。
4. **正话反说**：将正面的话用反面的方式表达，更加生动有趣。
5. **比喻加联想**：通过比喻和联想，让对方回味无穷。
6. **自我嘲讽**：营造轻松洒脱的形象。

## 方法讲解

### 1. 偷换概念

**例子**：

- “有人在找工作吗？来我这里吧，我最近在找喜欢的人找我。”
- “一个人走路真的害怕天这么黑，我又这么漂亮，真怕别人看不见。”

### 2. 欲扬先抑

**例子**：

- “你就说他，你根本就不像个男人。男人哪有你这么温柔的？”
- “我特别讨厌你。讨厌你把我宠的像个小公主，让我越来越离不开你。”

### 3. 反夸

**例子**：

- “没想到呀，你这个人啊不断情商高，那眼光很很高嘞，说话还这么重听。”
- “我如果没有这副傲人的身材，又怎么能配得上你这个糟糕的颜值呢？”

本节课我们学习了如何用幽默为他人提供情绪价值的6种技巧。通过练习和运用这些技巧，我们可以更好地与亲密关系中的对方沟通，增进彼此的感情。下节课我们将继续探讨剩余的3种幽默方式，敬请期待！
