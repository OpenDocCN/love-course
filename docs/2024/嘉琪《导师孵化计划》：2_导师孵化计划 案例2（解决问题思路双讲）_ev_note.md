# 嘉琪《导师孵化计划》：2_导师孵化计划 案例2（解决问题思路双讲）

## 案例分析

在本案例中，我们将分析导师如何处理与一位高冷女生的聊天，并从中提炼出解决问题的思路。

### 问题判断

1. **问题走向判断**：导师首先需要判断聊天中的问题是否需要解决。例如，女生提出的问题是否需要立即解决，还是可以暂时搁置。
2. **讲解逻辑判断**：如果问题不需要解决，导师需要考虑如何讲解逻辑，使对方能够理解。

### 案例回顾

**开场**：女生通过P图、化妆、增高鞋垫等方式展示自己，并询问导师的看法。

**导师回应**：导师没有将P图、化妆、增高鞋垫视为问题，而是认为这是合理的，并以此为契机展开话题。

**问题处理**：

1. **测试**：女生通过提问测试导师是否能够解决问题。
2. **情绪优先**：导师首先关注情绪，通过传递情绪使聊天继续进行。
3. **话题权重**：导师在回答问题时，避免直接回答问题，而是通过提问引导话题。
4. **情绪控制**：导师通过提问和回答，引导女生表达情绪，并使其情绪得到释放。

### 解决思路

1. **情绪优先**：在解决问题之前，先关注情绪，确保聊天能够继续进行。
2. **话题权重**：在回答问题时，避免直接回答问题，而是通过提问引导话题。
3. **情绪控制**：通过提问和回答，引导对方表达情绪，并使其情绪得到释放。
4. **框架构建**：在适当的时候，通过提问和回答，构建框架，引导对方思考。

## 导师讲解

### 案例本质

案例本质上是强调女生一方，而不是强调导师一方。导师需要关注女生表达的意思，并分析其可能带来的威胁。

### 对立面讲解

1. **框架**：女生通过提问和回答，构建框架，对导师进行命令和要求。
2. **对立面**：导师需要找到机会，站在高位，产生变现。

### 导师讲解方法

1. **案例分析**：分析案例中女生表达的意思，以及可能带来的威胁。
2. **对立面讲解**：讲解框架和变现的逻辑。
3. **情绪调动**：通过幽默、互动等方式，调动观众情绪。
4. **知识点讲解**：讲解情绪优先、话题权重、情绪控制、框架构建等知识点。
