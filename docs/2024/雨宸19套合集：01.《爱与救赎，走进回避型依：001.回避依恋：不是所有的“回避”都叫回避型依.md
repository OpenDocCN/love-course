# 雨宸19套合集：01.《爱与救赎，走进回避型依恋的内心世界》：001.回避依恋：不是所有的“回避”都叫回避型依恋 🧠

在本节课中，我们将要学习什么是回避型依恋人格。我们将明确其定义，了解其核心特点，并探究其背后的内在信念。最重要的是，我们将学会如何区分真正的回避型依恋与单纯的不喜欢或逃避行为。

## 什么是回避型依恋？ 🤔

上一节我们介绍了课程主题，本节中我们来看看回避型依恋的具体定义。

回避型依恋是情感咨询中最常见的一种人格状态。但需要记住：**不是所有的回避行为都叫回避型依恋**。

在感情中，你可能会遇到这样的人：他们个人能力、素质和外界的评价都不错，但无论你如何表达，他们似乎都不相信，总是与你保持距离。即使在一起了，他们也很少主动找你，不主动约会，推进关系特别费劲，甚至可能直接消失。

遇到这种情况，先别急着放弃感情，因为对方很可能是一种回避型依恋人格。本节我们就来剖析回避型依恋到底是怎么回事。

首先，要明白一个核心概念：依恋理论。它最早由约翰·鲍尔比提出，描述的是婴儿与主要抚养者（通常是母亲）之间最初的情感连接方式。这种早期形成的依恋类型，会深刻影响儿童成年后与外界和他人的关系模式。

也就是说，回避型依恋源于个体小时候与母亲的相处模式，主要形成于**婴儿时期（1-3岁）**。在1到4岁这个关键阶段，我们与母亲的互动会形成我们的**潜意识**。

*   **意识**：你能意识到的问题。
*   **潜意识**：你下意识做出的反应。例如，有人对你大声说话，你本能地闭嘴、想逃避，事后却不明所以。这就是潜意识在保护你远离潜在威胁。

潜意识很大程度上是在0-4岁与母亲的相处过程中形成的。

## 四种依恋类型 📊

了解了定义，我们来看看依恋理论中总结的四种主要类型。

以下是四种基本的依恋类型：

1.  **安全性依恋**：母亲离开或陌生人在场都不会引起强烈反应。这类人在社交中爱与被爱、接近与被接近、表达爱与拒绝都非常正常，能清晰表达感受，内心有安全感。公式：**安全感充足 = 健康的情感表达**。

2.  **焦虑性依恋**：时刻警惕母亲的离开，对分离非常反抗。但矛盾的是，当母亲回来安抚时，他们又会拒绝。在恋爱中表现为：抱怨伴侣不爱自己，但当伴侣试图安慰时，又将其推开。他们时刻怀疑自己被抛弃，内心充满焦虑，这种焦虑会蔓延到生活的方方面面。

3.  **回避性依恋（本节重点）**：母亲在场或离开都无所谓，婴儿自己玩自己的。这表明婴儿与母亲未形成亲密的感情连接。这类人长大后，情感表现淡漠，仿佛所有行为都是理智思考的结果（“我该做”而不是“我想做”）。他们的**边界感特别强**，不喜欢他人靠得太近。

4.  **混合型依恋**：同时具备以上安全型、焦虑型、回避型的特点，但都不特别明显，属于综合状态。

## 回避型依恋的核心特点 🔍

上一节我们介绍了四种依恋类型，本节中我们重点分析回避型依恋的具体特点。

以下是回避型依恋的四个主要特点：

1.  **习惯独立，情感淡漠**：他们往往从小就很独立，习惯一个人和孤单，特别强调个人边界感。这种独立已成为其社交认知和安全感的一部分。突然改变会引发不适和抗拒，因为改变意味着动摇其安全感基础。

2.  **抗拒亲密，否认真实感受**：与常人相反，他们特别慢热，不喜欢快速发展的关系，反感“自来熟”。有时越喜欢一个人，越不敢靠近，会否认自己的真实感受和需求，甚至抗拒亲密行为。关键在于，他们需要**绝对的掌控感**——关系必须由他们主动一步步推进。如果对方过于主动，他们会因不习惯而恐慌、逃避。

3.  **害怕面对冲突**：他们很少与人发生正面冲突，尤其在亲密关系中，往往选择冷战，把事憋在心里。因为他们幼年缺乏与人相处的经验，不知道如何处理人际冲突。为了维持独立性、完美感和安全感，他们选择逃避。

4.  **相信“有条件的爱”**：他们也渴望被爱，但有人爱他时又会怀疑。他们需要明确对方爱自己的“理由”（如身材好、声音好听、能提供情绪价值），才能感到踏实。他们不相信无条件的爱，因为从未体验过。他们需要确认：“我有什么优点？你为什么需要我？你为什么会一直需要我？”

5.  **随时准备抽身离开**：在他们的认知里，关系的本质就是相遇后终会分开。由于幼年与不会离开的玩具相处，他们缺乏“关系会变化”的认知。当发现人都会离开时，他们会恐慌，害怕受伤，因此随时准备从关系中抽离，作为一种自我保护。

**总结**：回避型依恋者对亲密关系充满不信任和怀疑，但这本质上是一种**自我保护**。他们像剥壳鸡蛋一样脆弱，努力维持自我完整，避免受伤。

## 回避型依恋的内在信念 💭

了解了外在特点，我们进一步探究其内在驱动力。任何行为背后都有思维和认知的支撑。

回避型依恋通常由以下三种内在信念支撑：

1.  **信念一：没有人是爱我的**。婴儿时期的基本需求未得到满足，未感受过足够的温暖和爱，因此觉得自己不值得被爱，认为爱都是假的。他们追求一种过于纯粹的爱，但不了解真实人际交往的规则，因求而不得受过挫折，从而不敢再信。

2.  **信念二：我不好，所以没有人会爱上真实的我**。小时候被忽视或打压，导致对自我价值缺乏正确认知。他们往往伪装成别人喜欢的样子，很少展现真实的自己。内心深处认为只有“完美的人”才配被爱，因此对自己要求极高，活得很累。除非学会接纳不完美的自己，否则很难解脱。

3.  **信念三：真实但不完美的我，不会被接受**。他们始终否定自我价值，认为只有具备外在价值（如能力、财富）才会被爱。因此不断追求完美，学习各种技能来伪装自己，弥补内心自认为的空虚。本质上是**不接纳真实的、自认为卑微的自我**。

## 如何判断：回避型依恋 vs 单纯的不喜欢 ❓

最后，我们必须厘清一个关键点：不是所有的不理睬和逃避都是回避型依恋。

有的人只是单纯不喜欢了，或不想直接提分手，于是采用回避的方式。不要轻易给对方贴上“回避型依恋”的标签。

以下是两者的核心区别：

*   **回避型依恋**：
    *   **针对人群**：无针对性，是**习惯性行为**。对所有人都这样，不仅是对你。
    *   **否定对象**：通常**否定自己**，认为自己不配。
    *   **行为模式**：态度一贯比较平淡、疏离，**变化不大**。回避行为具有**持久性**。

*   **非回避型（单纯不喜欢/逃避）**：
    *   **针对人群**：有针对性，是**短暂出现的行为**，通常抱有特定目的（如想脱离关系）。
    *   **否定对象**：通常**否定别人**，觉得对方不好。
    *   **行为模式**：可能开始时很热情，后来才慢慢变淡。逃避是**间歇性、短暂**的。

**本节课中我们一起学习了**：回避型依恋的定义、四种依恋类型、回避型依恋的五大特点、其背后的三种内在信念，以及如何区分回避型依恋与单纯的不喜欢行为。核心在于理解，回避型依恋是一种源于早期经历、以**自我保护和情感疏离**为特征的持久行为模式，而非针对某个人的临时态度。
