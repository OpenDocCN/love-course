# 雨宸19套合集：16.《雨宸复合课：透视关系破裂的根源，挽回丢失的爱情》：009.长期的亲密关系，是一种互相的满足

## 概述
在本节课中，我们将探讨长期亲密关系的维护，特别是如何了解并满足对方的需求，以维持一段关系的稳定和幸福。

## 长期亲密关系的维护
### 了解并满足对方的需求
**公式**：动力 = 满足需求

首先，你需要了解并满足对方的需求，这是维持一段关系的关键。一个人是否有动力去维持一段感情，取决于对方是否满足他情感价值排中位于首位的一个需求。

### 需求的变化
**公式**：需求 = 时间 + 经验 + 社会环境

对方的首要需求是不断变化的。刚开始谈恋爱时，对方的需求可能与你满足他的恋爱需求有关。但随着时间的推移，需求会不断变化。

### 例子
以作者与爱人的关系为例，刚开始他们满足的是生存需求，但随着生存需求的满足，他们开始更加关注情感需求。

### 需求的变化与伴侣的选择
**公式**：伴侣选择 = 需求匹配

随着时间的推移，如果两个人的社会生活认知层次不一致，他们的需求也可能不一致。如果一方不能满足另一方的需求，可能会导致分手。

### 维护关系的秘诀
**公式**：关系维护 = 进步 + 学习 + 充实

为了维护关系，我们需要不断进步，强化自己，不断学习和充实自己，以跟上对方的脚步，满足对方的情感需求。

## 人格分类与需求
### 表演性人格
**特点**：内外表现不一致，内心渴望被认同。

**需求**：被认同，缺爱。

**满足方法**：给予鼓励，帮助对方找到真实的自己。

### 工具型人格
**特点**：坦诚、诚实、公正，喜欢稳定。

**需求**：有规划，安全感。

**满足方法**：帮助对方规划，提供稳定性。

### 现任性人格
**特点**：社交高手，情绪家值高，容易喜新厌旧。

**需求**：没有需求，缺乏原则。

**满足方法**：帮助对方找到需求，建立原则。

## 总结
本节课中，我们学习了如何维护长期亲密关系，特别是了解并满足对方的需求。通过人格分类和需求分析，我们可以更好地理解对方，从而更好地维护关系。记住，关系维护需要观察力、细节、放弃自我和共情。
