# 《全套课程合集》：leon 1.0开撩蓝图：8 低需求感

## 概述

在本节课中，我们将学习如何通过控制需求感来提升自己在女生心中的价值感。

## 需求感是什么？

需求感，即你对女生的欲望和渴望。过强的需求感会导致“舔狗”现象，表现为：

*   微信聊天秒回
*   约会时身体前倾
*   热情高涨地迎合女生
*   百分百服从女生
*   围绕女生转，对她好

## 低需求感

低需求感，即与女生相处时保持轻松、自在的态度，不急于表达自己的需求，不失去自我。

### 低需求感的表现：

*   微信聊天不急于回复
*   不急于升级关系
*   不油腻，不围着女生转
*   不为了迎合女生而失去自我
*   对女生的行为不斤斤计较
*   不急于证明自己

### 低需求感的价值

*   体现你的价值
*   让女生觉得你值得追求
*   提升你在女生心中的价值感

## 如何做到低需求感？

*   **情绪上的淡定**：不要因为女生的言行而情绪波动过大。
*   **言语上的淡定**：不要过度迎合女生，做自己想说的。
*   **行为上的淡定**：不要为了迎合女生而失去自我。
*   **保持真实**：做自己，不要为了讨好女生而改变自己。

## 总结

本节课中，我们学习了低需求感的重要性以及如何做到低需求感。记住，低需求感等于高价值，做自己，才能在女生心中留下深刻的印象。
