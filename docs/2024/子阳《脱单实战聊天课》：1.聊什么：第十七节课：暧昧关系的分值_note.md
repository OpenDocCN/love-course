# 子阳《脱单实战聊天课》：1.聊什么：第十七节课：暧昧关系的分值

## 概述
在本节课中，我们将学习如何与女孩子建立暧昧关系，并了解不同暧昧程度的表达方式。

## 踩油门技巧
**公式**：暧昧关系分值 = 沟通深度 × 情感投入

### 暧昧关系分值获取
要达到70分，你需要掌握以下技巧：

1. **沟通深度**：不要突然表白，而是逐渐增加沟通深度。
2. **情感投入**：不要非零即一，要像刻度一样丝滑。

### 暧昧关系程度
以下是不同暧昧程度的表达方式：

1. **0.2的暧昧关系**：简单的问候，如“早安”或“晚安”。
2. **0.3-0.4的暧昧关系**：给对方取亲昵的外号，如“小懒猪”、“小狗狗”。
3. **0.5的暧昧关系**：使用土味情话，如“被子太轻，压不住想你的心”。
4. **0.7-0.8的暧昧关系**：开车或聊骚。

## 总结
本节课中我们一起学习了如何建立暧昧关系，并了解了不同暧昧程度的表达方式。希望这些技巧能帮助你更好地与心仪的人沟通。
