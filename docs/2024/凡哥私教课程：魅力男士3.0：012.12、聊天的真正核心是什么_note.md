## 共鸣型聊天方式

那么，聊天的过程中，聊天的时候吸引女孩的核心到底是什么？在讲终极的核心之前，我额外再讲一个比较厉害的聊天方式，叫**共鸣型聊天方式**。共鸣也是跟女生聊天的一个非常重要的技巧之一。

共鸣型的聊天方式大概分为以下几个步骤：
1.  倾听她当下的语言。
2.  了解她背后的情绪。
3.  对她的情绪表示理解。
4.  用更强的情绪去和她共鸣。

这叫共鸣式聊天。我在之前的课程中也讲过，女生的情绪怎么去调动。她高兴，你比她更高兴；她生气，你比她更生气。其实我之前讲的就是共鸣式聊天的模型。你要想跟这个女生产生聊天的共鸣，想跟她去深入地推进关系、升级关系，首先你要做的是倾听她当下的语言，好好听她跟你说了什么。然后呢，了解她背后的情绪是喜、是怒、是哀还是乐。了解她的情绪之后呢，对她的情绪表示理解、表示认可，之后用更加强烈的情绪去跟她共鸣。这是比较重要的一种方式。

但这种方式也有个缺点，是什么缺点呢？当你用共鸣聊天的模型跟她聊天的时候，跟女生去聊天的时候，就算你能够共鸣到这个女孩心底里，那又怎么样？即使你能把她撩哭，那又怎么样？这个女生会因此爱上你吗？会因此喜欢上你吗？还是更加有可能把你当做一个倾诉对象？

大多数情况下会怎么样？你们共鸣完了之后，这个女生会说：“你真好，觉得你很懂她。”然后你自己心里就想：“他妈的就这样完了吗？就这样结束了吗？然后各回各家各找各妈吗？”有这样的吧。

## 核心的揭示：结果导向

咱们来总结一下。其实刚刚我讲了三个非常重要的点，兄弟们自认为非常重要的点，觉得跟一个女生通过聊天成功吸引到她，要靠三个点。很多兄弟会这样认为：
1.  源源不断的话题跟幽默。
2.  高价值的展示。
3.  情绪上的共鸣，懂女生、走进女生心里。

这是咱们之前认为的非常重要的三种方式，对不对？通过聊天吸引女生的非常重要的方式：源源不断的话题跟幽默、高价值的展示、情绪上的共鸣。

其实这三个点错了吗？没错，非常重要，这三个点绝对非常重要。但是这**绝对不是**聊天吸引女孩最核心的东西。这仅仅是表面，这仅仅是皮毛，是表象。

其实我讲到这里，你会发现什么？当你掌握了以上的所有技能之后，当你有了源源不断的话题、当你会幽默、当你能够展示你的高价值、当你能够跟这个女生产生共鸣，当你拥有了这些所有技巧技能之后呢，你会发现，你还是根本无法通过聊天让这个女生喜欢上你，更别说爱上你了。

不管你有多么能聊，为什么？这是我之前课程中经常会强调的一个点：这些华丽的句子、这些华丽的话术、华丽的技巧，往往会让你自己忘记你跟她聊天、你跟她约会的目的。你跟他聊天的目的是什么呢？聊天、约会，是为了获得一段亲密关系，有了亲密的关系，然后才能够让她爱上你。这是一个结果导向。

所以说，咱们很多兄弟把你的侧重点放在哪里了？真的是放在聊天上了，放在了话题上，放在了幽默上，放在了高价值上，放在了跟女生共鸣上。
