## 聊天的核心

但是，这节课我们将更深入地探讨聊天的思维、方法、思想、方向，以及聊天的真正核心。聊天的过程中，有一个非常核心的东西，那就是如何通过聊天吸引女生。

**公式**： 聊天吸引力 = 内在吸引力 + 外在吸引力

## 聊天误区

很多人羡慕那些聊天话题源源不断的人，觉得他们是聊天高手。然而，真正吸引女生的秘密武器并不是话题和幽默，而是你自身的内在和外在吸引力。

**代码**： 聊天吸引力 ≠ 话题 + 幽默

## 聊天目的

初期聊天的目的不是吸引女生，也不是查户口，而是建立熟悉感，为后续的深入吸引和见面做准备。

**公式**： 初期聊天目的 = 建立熟悉感

## 朋友圈的作用

朋友圈是展示你物质生活、内在品质的重要渠道。一个充满垃圾转发和营销号的朋友圈，会让人感觉你生活无聊，难以吸引女生。

**代码**： 朋友圈 = 展示生活

## 聊天技巧

1. **共鸣型聊天**： 倾听女生的语言，了解她的情绪，表示理解，用更强的情绪与她共鸣。
2. **展示价值**： 展示你的外在形象、物质生活、内在品质。
3. **情绪共鸣**： 与女生产生共鸣，调动她的情绪。

**代码**： 聊天技巧 = 共鸣型聊天 + 展示价值 + 情绪共鸣

## 聊天核心

聊天的核心是只聊两件事情，只做一件事情。

**公式**： 聊天核心 = 发现女生值得喜欢的地方 + 告诉她为什么喜欢 + 升级关系

## 结果导向

聊天要以结果为导向，即获得亲密关系。

**公式**： 聊天结果导向 = 聊天 + 邀约 + 肢体接触 + 关系升级 + 确认关系

## 总结

本节课我们学习了聊天的真正核心，以及如何通过聊天吸引女生。记住，聊天的目的是为了获得亲密关系，而不是沉浸在话题和技巧上。
