# Chris《智选伴侣》：008.识女系统课第3节

## 概述
在本节课中，我们将探讨现代社会男女价值观的变化，特别是女性身份认同的演变。

## 价值观变化
**现代社会男女价值观相比过去已经大不相同。**我们可以看到一些有别于过去的心式恋情，例如姐弟恋等。因为今天社会不少女人的价值观已经脱离了过去传统社会，对于女性的那种身份认同的形象。

**传统赋权**定义女性通常是客体，是家庭当中依赖丈夫、母亲的角色，是被动的柔弱的温顺的，需要男性保护照料的角色。然而今天社会的价值观宣传的是男女平等。

## 女性身份认同的演变
**许多女人已经脱离了这种过去对于女性身份认同的形象。**他们认为女人也可以是一个主体，可以跟男人一样赚钱，在情感中也可以像男人一样主动啊，甚至给男人花钱，所以才会演变出一些姐弟恋这种新的情感模式。

**在身份维度上，我们可以把女性分为**：
- **认同过去女性定义的右派传统女性**：以家庭的合作，去构建自己的价值观。
- **认同现代女性定义的左派独立女性**：以个体的自主性去构建自己的价值观。

**这两类女性最核心的区别在于**：
- **右派女性**：更注重家庭观念，以与男性合作的利益优先。
- **左派女性**：更自私，以女性自我利益为优先，很少考虑家庭关系。

## 左派女性的特点
以下是左派女性的一些特点：
1. **时尚风格**：在穿衣打扮上更前卫，崇尚穿衣自由。
2. **行为更主动**：遇到喜欢的男人会主动出击。
3. **多元化关系**：更愿意尝试和探索有别于传统男强女弱的关系模式。
4. **身体自主性**：认为身体的自主性不应该被男性去定义。
5. **进步思想**：追求更多元化、性别平等等等的新观念和思潮。

## 右派女性的特点
以下是右派女性的一些特点：
1. **时尚风格**：在穿衣打扮上更符合传统审美。
2. **更为温柔顺从**：认为女人是被动的，这样才符合传统女性的形象。
3. **传统关系**：更接受传统关系。
4. **保守态度**：在一些方面更倾向保守的态度。
5. **家庭观念**：更注重家庭生活，会把家庭稳定和幸福放在一个更高的优先级上。

## 总结
本节课中我们一起学习了现代社会男女价值观的变化，特别是女性身份认同的演变。通过了解不同类型女性的特点，我们可以更好地理解和与他们相处。
