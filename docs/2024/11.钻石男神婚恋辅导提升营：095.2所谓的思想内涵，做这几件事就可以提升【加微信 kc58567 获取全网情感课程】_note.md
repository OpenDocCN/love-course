# 11.钻石男神婚恋辅导提升营：095.2提升思想内涵的方法

大家好，我是陈楠老师。提升思想内涵，可以通过以下几件事来实现。

## 1. 丰富知识内涵

长期沉迷于游戏和影视作品会扰乱正常思维。现在世界信息量巨大，各种价值观冲击，需要丰富自己的知识内涵，以保持独立的思想和价值观。

**公式**：知识内涵 = 阅读量 + 思考深度

## 2. 避免碎片化信息

不要过多阅读洗脑文章和营销文，专心整理知识，避免以偏概全的思维。

**代码**：避免碎片化信息 = 避免阅读洗脑文章 + 专心整理知识

## 3. 培养读书习惯

读书不仅为展示面和聊天提供素材，还能体验伟大人物的灵魂。经典书籍内涵丰富，能学到更多东西。

**公式**：读书习惯 = 碎片化时间 + 有深度有趣的书

## 4. 多看有营养的节目

推荐观看康熙来了、逻辑思维、一周立波秀等节目，以及纪录片，以丰富知识。

**代码**：有营养的节目 = 康熙来了 + 逻辑思维 + 纪录片

## 5. 演讲能力

TED系列演讲是碎片时间的必需品，能提高演讲能力和品味经典故事。

**公式**：演讲能力 = TED系列演讲 + 经典故事

## 6. 定期看电影

观看有深度的电影，如晚秋、全民情敌、撒娇的女人最好命等。

**代码**：有深度的电影 = 晚秋 + 全民情敌 + 撒娇的女人最好命

## 7. 参加社交活动

参加社交活动，认识女孩，拓展朋友圈。

**公式**：社交活动 = 认识女孩 + 拓展朋友圈

## 8. 多运动

培养爱好，锻炼身体，如射箭、慢跑、骑行等。

**公式**：运动 = 锻炼身体 + 培养爱好

## 9. 定期旅行

尽量一年两次以上旅行，丰富生活，约女孩同行。

**公式**：旅行 = 丰富生活 + 约女孩同行

## 10. 尝试和女孩做朋友

与女孩交往时，尝试少一些功利性，多一些真诚，培养友谊。

**公式**：与女孩交往 = 少功利性 + 多真诚 + 培养友谊
