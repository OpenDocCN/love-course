# 34、米娅情感强势认知：03.解读社会交换论 📚

## 概述
在本节课中，我们将从社会交换论的角度来探讨爱情，理解人们如何选择伴侣，以及爱情发展的不同阶段。

## 社会交换论概述
社会交换论认为，人们选择伴侣是基于理性考量，即选择能带来更多价值和幸福的对象。以下是对该理论的详细解读。

### 利益与价值
**公式**：价值 = 利益 - 成本

所有导致爱情的因素均可归结为利益和价值，包括物质、经济、社会和心理因素。

### 爱情发展的四个阶段
1. **取样与评估**：男女双方在选择交往对象时，主要考虑交往的成本与收益，以及相互抵消后的盈余。
2. **互惠**：双方尽可能交换收益，降低成本，如聊天、互赠礼物等，避免进入对方私密领域。
3. **承诺**：双方认为从对方那里得到的收益大于其他异性，停止与其他异性的交往，关系相对稳固。
4. **制度化**：随着亲密关系的增强，双方通过契约形式将关系制度化，如订婚、结婚等。

## 价值的多维度
**代码**：value = {观赏价值, 情绪价值, 生命价值, 养育价值}

男女感情关系中的价值具有多个维度，如观赏价值、情绪价值、生命价值、养育价值等。

## 认知陷阱
当听到“对方有价值”时，我们往往会将其与金钱挂钩，陷入认知陷阱。实际上，价值是多维度的，包括观赏价值等。

## 择偶规律
我们的择偶行为实际上遵循社会交换论的规律。当我们对某人一见钟情时，实际上对方已经提供了价值，而我们被这种价值所吸引。

## 总结
本节课中，我们一起学习了社会交换论在爱情中的应用，以及价值的多维度。希望这能帮助我们更好地理解爱情和择偶行为。
