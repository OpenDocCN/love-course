# 大鹏《恋爱指南》：020.《废物测试》正确破解女生刁难

## 概述
在本节课中，我们将学习如何正确破解女生的“废物测试”，这是一种女生用来筛选不合格追求者的测试。

## 定义
**废物测试**是指女性为了筛选不合格的追求者，而产生的测试。目的是让不合格的男生知难而退。

## 为什么会有废物测试
女生在情感中是被动被挑选的，她们会通过设置难题来筛选追求者。

1. **打压型**：女生会故意打压你的自信心，比如质疑你的身高、体重或收入。
2. **劳工型**：女生会要求你做一些体力劳动，比如帮你搬家或拿快递。
3. **提问型**：女生会通过提问来测试你的反应，比如询问你是否喜欢她。
4. **暗示型**：女生会通过暗示来测试你的反应，比如提到她的闺蜜或朋友。
5. **花心测试**：女生会故意说你是一个花心的男人，然后观察你的反应。

## 应对方法
1. **平衡要求**：当女生提出要求时，你可以反向提出一个要求，以保持关系的平衡。
2. **合理调侃**：用幽默的方式化解女生的测试。
3. **巧妙转移**：通过转移话题来避免废物测试。
4. **尊重自己**：避免做让自己不舒服的事情。

## 总结
本节课中，我们学习了什么是废物测试，以及如何正确应对。希望这些知识能帮助你更好地理解和应对女生的刁难。
