## 一日三餐的具体建议

上一节我们指出了需要改变的旧习惯，本节中我们来看看如何构建新的、有助于减肥的三餐结构。

### 早餐建议

早餐的建议结构是：**主食 + 蛋白质 + 蔬菜**。

以下是具体的食物推荐和分量建议：
*   **主食**：建议吃大约一个拳头的量。无论是选择哪种主食，都控制在这个分量。
*   **蛋白质**：可以选择各种蛋白质来源。
*   **蔬菜**：推荐吃蔬菜，虽然可能口感比较寡淡。实际上，吃其他烹饪方式的蔬菜也可以，比如炒菜。

我们毕竟要过日常生活，不需要每天带着秤。很多减肥专家建议随身带秤，但外出吃饭时这样做太尴尬了。你应该对自己吃多少有基本概念，能少吃就少吃。

### 午餐建议

午餐的推荐结构也是：**主食 + 肉 + 蔬菜**，本质上同样是**主食 + 蛋白质 + 蔬菜**。

以下是具体的执行建议：
*   **蛋白质选择**：这里推荐吃动物蛋白。根据我的经验，午餐吃虾、牛羊肉、鸡胸肉、去皮的肉类，吃多少都不会长胖。
*   **核心好处**：午餐多吃肉的好处是，到晚餐时你都不会饿。因为动物蛋白饱腹感特别强，消化慢，能在身体里持续提供饱腹感，这样晚上就特别容易控制饮食。如果中午就开始严格控制，晚上很容易饿。
*   **现实挑战**：尤其是忙碌一整天后，晚上回家面对一桌丰盛的饭菜，很难忍住不吃。很多人午餐控制得很好，但晚上回家就放松了，这正好吃反了。应该倒过来，午餐可以多吃（尤其是肉类），晚餐则要控制。

### 晚餐建议

晚餐大家需要注意。

以下是晚餐的核心原则：
*   **原则一：简单**：晚餐能简单就简单，千万不要吃反了（即晚餐吃得比午餐丰盛）。
*   **原则二：早吃**：晚餐越早越好。例如，如果晚上10点睡觉，建议晚餐在6点前吃完，这样有4小时消化，形成空腹状态。否则晚上肾脏和消化系统会有负担。我个人有时七八点吃晚饭，是因为睡得晚，也有足够的消化时间。
*   **原则三：清淡**：建议少油少盐。因为油盐很难量化，所以要在烹饪前端就控制。我通过提醒家里的阿姨减少油盐用量，她也发现了这是我瘦下来的妙招之一：饮食清淡，烹饪方式简单。只要孩子和老公不在家，我就尽量简单吃。

这些建议包含了品类推荐和大概的量。一个身高1.77米、140斤的姑娘和一个身高1.55米、100斤的姑娘，食量肯定不同。所以用“拳头”作为参考量，而不是具体的克数，你可以对自己的主食和蛋白质摄入量有一个大致的概念。

## 食物选择的优化

上一节我们规划了三餐的框架，本节中我们来看看如何在每一类食物中做出更优的选择。

### 荤菜的选择

在荤菜的选择上，有一些指导建议。

以下是优化荤菜选择的要点：
*   **选择低脂部位**：要吃不容易致肥的肉，不要吃带皮的。猪、牛、羊肉、兔肉、海产品都可以吃，但要吃瘦的部分，其他部分要少吃。
*   **替换比例**：在瘦肉和海鲜的替换上，如果你吃一个拳头的瘦肉，就可以吃两个拳头的虾。

### 蔬菜的选择

在蔬菜的选择上，需要注意区分。

以下是蔬菜选择的注意事项：
*   **识别“伪蔬菜”**：土豆、山药、板栗、莲藕这些其实是高淀粉食物，属于主食范畴。所以吃的时候要把它划归到主食类，不要当成蔬菜。因为吃炖土豆带来的满足感，跟吃米面是一样的。
*   **推荐绿叶蔬菜**：其他蔬菜都可以吃，尤其推荐绿叶蔬菜。绿叶蔬菜膳食纤维丰富，能帮助肠胃蠕动，且不会被吸收转化成热量。

### 主食的选择

主食的选择要尽量优化。

以下是主食选择的建议：
*   **替换精制碳水**：尽量不吃精白的米、面及其制品，替换成粗粮，比如玉米、红薯、燕麦等。

### 烹饪方式的选择

烹饪方式的选择同样重要。

以下是烹饪方式的推荐与不推荐：
*   **推荐方式**：蒸、煮、炖、快炒、凉拌。
*   **不推荐方式**：红烧、油炸、糖醋、干锅。

### 水果与饮水的建议

关于水果和饮水，也有明确的建议。

以下是具体建议：
*   **水果**：推荐番茄、黄瓜、柚子、苹果等低糖水果。
*   **饮水**：尽量避免喝酒或少喝；不喝含糖饮料和碳酸饮料；可以喝白开水、无糖柠檬水、淡茶或花茶。建议每天饮水2000毫升或以上。

如果真的想减肥，尤其是低糖低碳饮食部分，建议回听一遍课程。因为第一次听可能对GI值、升糖原理等概念一时绕不过来，再听一遍就没问题了。

## 减肥启动前的关键关注点

在开始你的减肥计划之前，有一些关键点需要大家关注。

以下是需要提前关注的四个要点：
1.  **避开生理期**：尽量避开生理期，尤其是生理期较长的姐妹。在生理期前后，由于激素水平和身体储水的变化，体重会不稳定且容易增加，这时开始减肥容易造成混乱。
2.  **避开高压期**：尽量避开频繁出差和高压工作安排的阶段。如果无法避免，就需要极强的意志力。例如，我在工作应酬时坚持不吃东西，靠同事帮忙解释。减肥需要克服困难，如果因为应酬中断，很容易前功尽弃。能随时中断减肥的人，在其他人生目标上也容易放弃。
3.  **清理厨房与冰箱**：在减肥前，清理一下你家厨房和冰箱里不该有的食物。如果家里没有零食，你在饿的时候就不会有“挣扎爬起来煮饺子”的诱惑。这个习惯非常重要，自律应该发生在购买环节——不要随便往家里买零食。
4.  **养成记录习惯**：养成记录的习惯。记录身体的变化和每天的饮食。
    *   **体重记录**：可以每三天或每周记录一次，选择同一时间（如早上上厕所后）。
    *   **饮食记录**：这个要每天记。好处是能让你清晰地看到自己到底吃了什么。很多姐妹觉得自己吃得少但胖，可能是因为吃了一小块蛋糕，其热量比一大筐蔬菜还高。记录让你有仪式感，不敢隐瞒，从而能发现问题所在。

### 易瘦体质的核心原理

大家特别关注的“易瘦体质”，其实就是**基础代谢率高**。

通过健康合理的饮食，让身体由内而外发生变化，从“吃什么都胖”慢慢变成“吃了也不胖”，这就是易瘦体质。它源于你的饮食习惯加上合理运动燃脂，提升了肌肉量，改变了代谢。

**基础代谢率（BMR）** 可以简单理解为我们静止状态下，维持呼吸、生命体征所消耗的最低热量。公式上，它决定了你每天什么都不做的“躺瘦”能力。

举个例子，如果你的基础代谢是1400大卡。即使你整天躺着，也消耗了1400大卡。如果你今天只摄入400大卡，那么热量差就是1000大卡，你自然就会减重。

减肥无非是处理**摄入**与**消耗**的关系。
*   **摄入**：就是你吃进去的食物热量。
*   **消耗**：由两部分组成：
    1.  **基础代谢**：提升它需要增加肌肉量，这需要通过运动来改善肌肉与脂肪的比例。
    2.  **日常活动消耗**：包括所有活动，如走路、工作、运动等。增加活动量就能增加这部分消耗。

为什么有人容易瘦，有人好像不吃也胖？根本区别就在基础代谢和总消耗上。

### 对易瘦体质的错误认知

关于易瘦体质，有一些常见的错误认知。

以下是需要厘清的两个观点：
1.  **你看到的可能只是假象**：生活中充满假象。你以为别人天天吃零食也不胖，可能他那包薯片放了一周才吃两片；而你虽然不常吃，但一天就能吃两包。或者别人虽然聚餐多，但只吃清淡的蔬菜和低油烹饪的食物。很多人的自律发生在你看不见的地方。
2.  **运动习惯至关重要**：经常运动的人更不容易胖。这有两个原因：第一，运动过程本身消耗热量；第二，经常运动的人肌肉含量高，而肌肉含量高会提升基础代谢。这就是为什么我坚持运动，吃得比别人多却不会更胖的原因。肌肉含量高，意味着即使不运动时，基础代谢也高，消耗更多热量。这才是真正的“躺瘦”或“易瘦体质”，其他声称躺瘦的都是骗子。

### 如何养成易瘦体质

要养成易瘦体质，需要从多个生活习惯入手。

以下是养成易瘦体质的五个关键行动建议：
1.  **培养良好的饮食习惯**：早餐一定要吃，有说法称早餐启动基础代谢。保持合理的饮食结构，尽量戒掉加工零食，选择天然食材。**细嚼慢咽**非常重要，它能让大脑及时接收到饱腹信号，避免过量进食。细嚼慢咽也是对食物的尊重和链接。
2.  **坚持规律运动**：运动不仅消耗热量，长期坚持还能增加肌肉量，提升基础代谢。如果很难有完整时间，可以利用碎片时间多动，比如爬楼梯、多站立。避免久坐，每隔一段时间就起来活动一下。
3.  **保证充足睡眠**：充足的睡眠（建议每天7-9小时）是新陈代谢和身体排毒的前提。高质量的睡眠至关重要。
4.  **保持情绪稳定**：情绪不稳定会导致内在能量失衡，容易让人在饮食上“破罐破摔”，破坏减肥计划。保持情绪稳定有助于维持持续的基础代谢状态。
5.  **将以上习惯融入生活**：养成良好的习惯能改善全身循环和代谢，加快脂肪燃烧，最终促进易瘦体质的养成。易瘦体质就是这么来的，并不神秘。
