# 奥客《黑魔法 男子汉后宫管理学》：06-资源调度与分配

## 概述
在本节课中，我们将学习如何进行资源调度与分配，以优化后宫管理。

## 资源调度与分配
以下是资源调度与分配的几个关键点：

### 1. 资源类型
- **金钱**：包括直接的花费和机会成本。
- **心力**：包括时间、精力等。
- **物质**：包括礼物、消费等。

### 2. 资源占比
- **正宫**：认真交往，投入心力。
- **正工**：与正宫类似，但可以有两个。
- **炮友**：频率为2-3周一次。
- **奴**：资源等同于一个以上的正宫。
- **盘子**：资源等同于2-3个炮友。

### 3. 安全系数
- **两倍年薪**：确保女人安心，不会抱怨。
- **五倍年薪**：相对安全，外敌难以入侵。
- **十倍年薪**：高度安全，绝大部分外敌可以抵。

### 4. 量入为出
根据自身状况和格局，选择合适的女人。

### 5. 组合推荐
- 正宫+正工
- 正宫+炮友
- 正宫+奴+炮友
- 正宫+盘子
