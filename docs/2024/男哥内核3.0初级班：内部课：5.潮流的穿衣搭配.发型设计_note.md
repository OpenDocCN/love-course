# 男哥内核3.0初级班：内部课：5.潮流的穿衣搭配.发型设计 🎨

## 概述

在本节课中，我们将学习如何打造个人形象，包括发型设计和穿衣搭配，以提升个人魅力和社交吸引力。

## 学习方法

* **系统学习**：初级班旨在帮助学员建立完整的个人形象体系。
* **形象建设**：首先提升个人形象，包括品牌建设、发型和穿衣搭配。
* **实战应用**：在形象建设的基础上进行实战聊天。

## 发型设计

### 发型的重要性

* 发型占据整个颜值的70%，对形象至关重要。

### 发型设计要点

* **剪短干净**：避免板寸或毛寸，保持简洁干净。
* **后脑勺搓起**：打造立体感，避免平板头。
* **放射状造型**：根据个人喜好调整，打造个性化造型。

### 发型打理

* **使用发泥和定型喷雾**：打造干燥、有空气感的发型。
* **吹风机吹干**：避免头发打绺。
* **适量使用发泥**：打造一撮一撮的感觉。

## 穿衣搭配

### 穿衣原则

* **个人品牌建设**：服装代表个人生活态度，体现个人品味。
* **消费能力**：根据自身消费能力进行搭配。
* **搭配建议**：可寻求专业人士帮助。

### 服装选择

* **避免过时服装**：保持时尚感。
* **避免卡通头像**：提升形象气质。
* **避免山水画头像**：避免与自然景观混淆。

## 作业

1. 观察照片，保存照片，仔细观察。
2. 准备打理头发的干胶和发泥。
3. 着重进行服装搭配。
4. 选择合适的理发店。
5. 拍摄自拍，进行修图练习。

## 总结

本节课学习了个人形象建设的重要性，包括发型设计和穿衣搭配。通过学习，学员可以提升个人魅力，增强社交吸引力。
