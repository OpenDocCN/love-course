# 子阳《脱单实战聊天课》：1.聊什么：第五节课：陌生人如何聊成网友_ev_note

在本节课中，我们将学习如何将陌生人关系推进为网友关系。这是建立初步联系的关键一步，核心在于完成两个简单的动作。

上一节我们探讨了聊天的基础心态，本节中我们来看看如何迈出从陌生到相识的第一步。一个陌生人和一个网友之间最大的区别是什么？主要有两点：**第一，双方拥有彼此的联系方式；第二，双方交换了姓名并有了备注**。

因此，要实现从陌生人到网友的转变，你只需要完成以下两个核心步骤。

以下是实现这一转变的两个核心步骤：

1.  **交换真实姓名**
    你需要大方地告知对方你的真实姓名，并主动询问对方的真实姓名。目标是给彼此一个真实的备注，而不是网名或昵称。如果对方最初只给出昵称，你应该礼貌地追问其真实姓名。

2.  **主动开启第一句对话**
    你需要主动、热情地向对方打第一个招呼。这个开场白不需要复杂，只需符合三个原则：**真诚、礼貌、热情**。例如，简单的“Hi，[对方姓名]”或“很高兴认识你，[对方姓名]”加上一个表情即可。

在实践这两个步骤时，有两个关键问题需要你特别注意。

以下是执行过程中必须注意的两个要点：

*   **务必使用真实姓名**
    在交换姓名环节，请务必使用你的真实姓名。避免使用“某某地彭于晏”或“某某区吴彦祖”这类看似有趣实则轻浮的称呼。你的目标是建立真诚的连接，一个真实的姓名是信任的基础。同样，你也要努力获知对方的真实姓名。

*   **开场白只需简单得体**
    你的第一句打招呼（或称“开场白”），无需绞尽脑汁设计得花哨或与众不同。只需确保你的问候是**真诚的、礼貌的、热情的**。一个简单的“你好”，一句“很高兴认识你”，或是一个友好的表情包，只要符合这三点，就完全足够了。

本节课中我们一起学习了将陌生人发展为网友的核心方法。关键在于完成**交换真实姓名**和**发起一次真诚礼貌的问候**这两个动作。记住，在初始阶段，简单、真诚远比技巧和套路更为有效。掌握了这一步，你就为后续的深入交流打下了坚实的基础。
