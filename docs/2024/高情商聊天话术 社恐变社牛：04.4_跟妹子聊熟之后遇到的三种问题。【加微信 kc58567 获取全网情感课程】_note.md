# 高情商聊天话术 社恐变社牛：04.4_跟妹子聊熟之后遇到的三种问题

🎼在情人节因分手而困扰，多久后去找她合适？要解决问题，先了解原因和过程，然后有针对性地解决。

**核心概念**：理解问题原因和过程，针对性解决。

🎼分手通常有原因，要么是对你不满意，要么是出现了更好的选择。

**核心概念**：分手原因分析。

🎼前三天介绍了如何与妹子聊天，今天讲的是聊开后如何继续。

**核心概念**：聊天技巧。

🎼聊开后，如何继续沟通？

**以下是三种情况及应对方法**：

### 1. 正面反应

**情况**：妹子对你反应积极。

**应对方法**：升级关系，暧昧。

**核心概念**：正面反馈，积极互动。

### 2. 中立反应

**情况**：妹子愿意聊天，但不接受升级关系。

**应对方法**：摊牌，亮剑。

**核心概念**：中立反馈，明确立场。

### 3. 负面反应

**情况**：妹子突然冷淡。

**应对方法**：换人，展示，筛选。

**核心概念**：负面反馈，果断放弃。

**总结**：

本节课介绍了与妹子聊天后遇到的三种问题及应对方法，帮助大家更好地处理人际关系。希望对大家有所帮助。
