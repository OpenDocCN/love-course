# 《朕哥8套情感合集》课程01：核心私密课之高情商聊天术 🗣️
## 第11章：和男人聊天的黄金十八招（1）✨

在本节课中，我们将要学习高情商聊天术的核心技巧——黄金十八招的前三招。这些方法旨在通过改变语言表达方式，有效提升聊天的趣味性和主导权，从而建立更舒适的沟通关系。

上一节我们介绍了聊天的基础逻辑，本节中我们来看看具体可操作的方法技巧。首先，我们要理解一个核心原则：所有高级的聊天技巧都服务于让对话更有趣、更有效，而不是机械地背诵话术。关键在于将逻辑内化，形成自然的表达。

---

### 第一招：正话反说与反话正说 🔄

这一招的核心在于转换表达角度，以达到不同的沟通目的。**正话反说**是为了调情，让聊天更有意思；**反话正说**则是为了解决问题，而非单纯释放情绪，它能将对方的思维从对抗方向拉回。

以下是具体操作方法：

*   **正话反说示例**：
    *   **正话**：“我饿了，你帮我点个外卖。”
    *   **反说**：“可怜的小姑娘饿了，也不知道有没有人给我点个外卖。”
    *   **核心公式**：将直接的请求或夸赞，转化为带有调侃或撒娇意味的间接表达。例如，想夸对方时，不说“你真优秀”，而说“你别这么优秀，我怕我追不上你”。

*   **反话正说示例**：
    *   **反话**（情绪化）：“你为什么跟那个女人暧昧？你真不要脸！”
    *   **正说**（解决问题）：“你是不是碰见同事了？或者有什么事需要她帮忙才聊这么久？”
    *   **核心逻辑**：当内心产生负面情绪和指责（反话）时，冷静下来，将其重新构建为带有关心或寻求解释的正面询问。这能避免争吵，引导对方反思并照顾你的感受。

**练习思路**：在聊天中，当你想说一句正话（如“我爱你”）时，先思考如何反着说（如“你不要再对我这么好了，不然我会越来越离不开你”）。同样，当想指责对方（反话）时，练习如何正向、建设性地表达出来。

### 第二招：调侃打压，占据高位 🎯

当男人对你示好或试图推进关系时，适当的调侃和轻微打压可以打破他的节奏，让你从被动的接受者转变为互动的引导者，从而占据心理高位。

以下是具体应用场景：

*   **场景一：对方示好**
    *   **他的话**：“老婆，你喜欢什么礼物？我买给你。”
    *   **错误回应**：“不用了”或“谢谢亲爱的”（完全顺从，失去张力）。
    *   **调侃打压**：“原来你连我喜欢什么都不知道呀~（加一个沮丧表情）”
    *   **效果**：这不会让他真正生气，反而会促使他急于解释和进一步表现，同时隐含了“你应该更了解我”的期望。

*   **场景二：对方夸赞**
    *   **他的话**：“你的声音真好听。”
    *   **错误回应**：“谢谢。”（被对方的夸赞带领节奏）。
    *   **调侃打压**：“只有声音好听吗？”或“你怎么这么肤浅~”
    *   **效果**：通过玩笑式的打压进行推拉，避免自己因一句夸赞就完全满足，从而保持互动中的趣味性和主动权。

**核心逻辑**：不要轻易顺着对方的思路进入他的“套路”。他示好，你就完全接受；他打压，你就陷入低位。这会导致你产生“服从动能”，最终丧失自己的节奏。调侃打压的关键在于**度**，要用撒娇、可爱的语气，让对方觉得有趣而非被冒犯。

### 第三招：推拉艺术，掌控情绪过山车 🎢

推拉是聊天中制造情绪波动、保持吸引力的核心技巧。“推”是指用相对消极或开玩笑的方式将对方推开；“拉”是指用积极、赞美肯定的方式将对方拉回来。

#### 基础推拉

通过一推一拉，让对方的情绪随之起伏，从而爱上与你聊天的感觉。

*   **示例对话**：
    *   男：“我在你心里能打几分？”
    *   你：“120分！”（**拉**）
    *   （对方开心时）你：“不过满分是200分哦~”（**推**）
    *   男：“看来我还要继续努力。”
*   **代码式逻辑**：
    ```python
    if 对方寻求认可：
        给予超出预期的肯定（拉）
        if 对方情绪上升：
            随即设置一个更高的标准或进行轻微调侃（推）
    ```

#### 高阶推拉

高阶推拉能产生更强烈的情绪效果，用于引导对方行为。

1.  **制造后悔**：
    *   **方法**：先给予一个承诺或希望（如答应某件事），但因对方某个错误行为而合理取消。
    *   **示例**：“本来今天很想见你，但你刚才那句话让我觉得你并不在意我的感受，现在不太想去了。”
    *   **效果**：让对方为自己的错误行为感到后悔和反思，经历情绪过山车，从而更想弥补你。**关键**：理由必须自然、合理。

2.  **推翻既往评价**：
    *   **方法**：对对方过去的优点表示认可，但因其现状未达预期而“收回”或“降低”评价。
    *   **示例**：“你以前真的很自律、很帅，但现在好像有点松懈了。不过胖了点也挺可爱的。”
    *   **效果**：制造心理落差，促使对方为了维持在你心中的高评价而付出努力。

3.  **引导解释**：
    *   **方法**：在对方做出承诺或表达好感时，不断追问“为什么”，引导其进行解释和承诺。
    *   **示例**：
        *   男：“我喜欢你。”
        *   你：“真的吗？你喜欢我什么？”（引导解释）
        *   男：“因为你善良、可爱……”
        *   你：“那我可要看看你以后怎么对我哦~”（将解释转化为未来承诺）
    *   **效果**：在感情中，经常解释的一方往往处于心理低位。通过引导对方解释，你能掌握对话节奏，并使对方的解释成为一种无形的承诺。

本节课中我们一起学习了黄金十八招的前三招：**正话反说与反话正说**以转换角度提升情趣或解决问题；**调侃打压**以在对方示好时占据心理高位；**推拉艺术**以基础和高阶技巧掌控聊天中的情绪节奏。记住，所有技巧都需要练习以把握分寸，核心目标是让沟通更流畅、更有趣，而非机械套用。在后续课程中，我们将继续深入探讨更多实用的聊天招式。
