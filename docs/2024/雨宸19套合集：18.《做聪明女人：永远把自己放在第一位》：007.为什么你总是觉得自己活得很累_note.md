# 雨宸19套合集：18.《做聪明女人：永远把自己放在第一位》：007.为什么你总是觉得自己活得很累

## 概述
在本节课中，我们将探讨为什么有些人会觉得自己活得很累，并分析内耗的原因及其危害。

## 为什么你总是觉得自己活得很累

**以下是导致你感到疲惫的原因：**

1. **内耗**：当你不再爱自己时，没有人会真心爱你。
2. **高敏感加低自尊**：过分在乎别人的看法，容易将责任和错误内化。
3. **高评价加低自尊**：对自己要求过高，不切实际，背负着父母的期待和同伴的压力。

**内耗的危害：**

1. **心理上的疲惫和压力**：一直在与自己较劲，当被指责或目标未完成时，会自责甚至讨厌自己。
2. **伤害身边的人**：情绪不稳定，身边的人也会感到难过。

## 如何停止内耗

**以下是停止内耗的方法：**

1. **接纳自己**：喜欢自己优秀的时候，愉快享受；不优秀的时候，愉快接纳。
2. **不参与过度竞争**：人生不是给别人看的，别人的评价不会影响你的生活。
3. **正视自己的能力**：承认在某些方面不如别人优秀，不必给自己设定过高目标。
4. **关注过程**：将大任务分解成小目标，关注过程，逐步改善内核。

**总结**
本节课中我们一起学习了为什么有些人会觉得自己活得很累，以及如何停止内耗。希望你能从中受益，学会接纳自己，追求自己的优秀，同时也不排斥那个不优秀的自己。
