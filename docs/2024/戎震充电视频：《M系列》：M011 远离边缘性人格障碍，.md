# 戎震充电视频：《M系列》：M011 远离边缘性人格障碍 🚫

在本节课中，我们将要学习边缘性人格障碍的核心特征、识别方法，以及为何必须与这类人群保持距离。课程内容基于心理学知识，旨在帮助你保护自己的心理健康。

## 概述：什么是边缘性人格障碍

边缘性人格障碍是一种复杂的人格障碍。其成因与个体0到3岁期间的成长经历密切相关。许多患者在幼年时未能得到父母的充分保护，或未能与父母形成良性的早期互动。这种障碍也具有一定的遗传概率。

上一节我们介绍了障碍的成因，本节中我们来看看它的核心特征。

## 核心特征：情绪的两极波动

这种障碍与双向情感障碍有相似之处。主要特点是情绪只有两极。正常人的情绪变化是一个渐变过程，例如从1到10或从-1到-10。他们的情绪会缓慢地变坏或变好。

但边缘性人格障碍患者的情绪模式截然不同。他们要么处于极度兴奋、有魅力和学习能力强的状态，要么在瞬间跌入低谷，开始砸东西并涌现各种负面情绪。

## 识别身边的BPD：操纵与煤气灯效应

如果你长期与这类人共同生活，例如他们是你的父母、子女、配偶或恋人，你会感到非常被动和难受。长时间内，你可能都无法正确调整自己的心态。

以下是他们常用的操纵手段：

*   **无意识的操纵**：许多操纵行为并非其有意为之，而是由其人格类型驱动。
*   **煤气灯效应**：他们会不断否定你、打击你的自信。其根本原因是害怕你选择离开，这是他们最恐惧的事情。
*   **矛盾的自我认知**：他们日常可能认为自己不够好、配不上你，或认为自己足够好、比你更强，但又总是担心你的变化。

这让正常人感到无所适从，因为你永远无法准确知道他们在想什么。他们可以伪装得特别好。

## 行动准则：远离与自我保护

基于以上特征，我们的核心建议是：一定要远离所有这类人格障碍者。让他们与自己同类相处，例如BPD找NPD（自恋型人格障碍）的情况就很常见。

以下是具体的行动步骤：

1.  **鼓励就医**：这类情况本应寻求心理医生的专业治疗。他们不属于完全正常的人格类型。
2.  **避免自我诊断**：由于人口基数大，有特质的人很常见。不要轻易给自己贴上人格障碍的标签。从抑郁状态到抑郁症，有轻度、中度、重度之分，并非一时想不开就是重度抑郁。不要过度联想，就像胳膊疼不等于骨癌。
3.  **进行专业鉴定**：如果你怀疑自己有人格障碍，务必去医院精神科做系统鉴定。请注意，甚至权威医院的鉴定结果也可能每次不同，例如有人被诊断为双向情感障碍，有人被诊断为边缘人格障碍，还有人被诊断为抑郁症与焦虑症交替发作。
4.  **掌握识别知识**：学习这方面知识是为了判断他人是否属于这类人格障碍。例如：
    *   **表演性人格障碍**：总觉得自己最好，认为别人都应喜欢自己。
    *   **偏执性人格障碍**：永远觉得自己正确，是反驳型人格，总在辩驳和贬低他人，实则也怕被离开。
    *   **自恋型人格障碍**：会通过各种方式寻找你的关注。
    *   **边缘人格障碍**：情绪交替，时而对你极好、显得多才多艺（虽然往往什么都干不好），时而情绪暴躁让你无所适从。他们正是通过这种方式来控制你，希望你一直维持其好的状态。

## 深入分析：为何必须远离

当这类人用暴烈的情绪对待你时，你会因为害怕其再次爆发，而在他们状态好时，也时刻担忧其变回原样。但这类人通常极端自私和虚伪。在他们的世界里，只有自己的喜怒最重要。

你是一个正常人。正常的行为模式是：你对我好，我就对你好，知恩图报。你不能在一个人天天打你骂你时，还持续对其好。

不要因为对方某些表面特质（如NPD长得帅，或BPD初期显得健谈、多才多艺）而抱有幻想。有些BPD甚至可能在初次见面就发生亲密关系。日后你会发现，与这种人纠缠简直是人生的至暗时刻。不要自以为能摆脱他们，要在第一时间跑开。

**记住，这是连心理医生通常都不敢轻易接洽的人群。** 你凭什么认为自己可以应对？你是救世主吗？你有这种能力吗？

## 总结与最终建议

本节课中我们一起学习了边缘性人格障碍的识别与应对。面对任何人格障碍、心理障碍或心理疾病，你的行动顺序应该是：**劝其就医，然后自己尽快远离**。最好删除联系方式，或仅保持最普通的泛泛之交。

一旦与这种人建立恋爱、肉体或其他亲密关系，你的一生都可能与之纠缠不清。虽然患病本身令人痛苦，但如果你无故招惹上这种人，就是自己倒霉了。照顾好自己的情绪，保证自己的家园不失火，不被殃及池鱼，这非常重要。

因为身边有这样的人，他们会不断干扰你的心情，让你也对世界陷入绝望。这些人实质上是**用成年后的自己在惩罚自己，以医治童年创伤**。不要期望一个人能真正变好，永远不要这样赌。能抛弃就尽量抛弃。一个人能变好的概率实在太低。俗话说“狗改不了吃屎”，人很难远离自己的童年状态。

要么换项目，要么换人。换一个人很简单。广袤的地球上有许多人，选择不同的人就是选择不同的人生，选择不同的合作伙伴也是如此。人要仔细鉴别，不要总想着修复别人，不要总抱着“我对谁好，别人就一定对我好”这种幼稚的老实人想法。

人际关系不是做买卖，不是自动售货机——投入50元就必定得到50元的商品。现实往往是：你给他5元，他可能还你150元；你给他500元，他可能只还你2元。总是有亏有赚。

所以，不要用自己那颗圣母心去揣度、去衡量天下这些亘古不变的各种神经病与精神障碍。不要试图用自己的“好”去感化别人。不要学我走的这条路线，这不值得。
