## 020-什么是社交中的兴趣指标？

### 概述
在本节课中，我们将学习社交中的兴趣指标，这是判断对方是否被我们吸引的重要概念。

### 兴趣指标
兴趣指标是指对方间接表示已被吸引或对某方面感兴趣的一种暗示，也称为“窗口”。这种暗示有时是无意识的。

**公式**：兴趣指标 = 吸引 + 价值

**代码**：
```python
def interest_indicator(attractiveness, value):
    return attractiveness * value
```

#### 兴趣指标的表现
- 对方和你说话时，嘴角自然上扬、撩头发。
- 对方在交流时，脚尖、膝盖、胸膛、额头四个点指向你。

#### 兴趣指标的分类
- **轻度兴趣指标**：对方对你有一些兴趣，但不足以构建关系。
  - 对方稍微注重形象，但不会刻意表现。
  - 对方语速放慢，注意坐姿。
  - 无意识拨头发、调整衣着。
- **中度兴趣指标**：对方对你产生好奇。
  - 注意你在面前的言行举止。
  - 对你的个人信息好奇，如问名字、经历。
  - 找理由称赞你，拉近距离。
- **高度兴趣指标**：对方清楚你的价值，并被吸引。
  - 频繁与你互动，希望得到更多关注。
  - 愿意进行肢体接触。
  - 对你感兴趣的话题表现出兴趣，并回应。

### 无兴趣指标
无兴趣指标是指对方表现出不感兴趣的迹象和信号。

#### 无兴趣指标的表现
- **语言上的敷衍**：对方在对话中不透露信息，简单回复问题，不提问、不表达观点、不表露情绪。
- **回避交往**：对方刻意避开与你相处或交流的机会，如不与你眼神接触、用身体语言表示躲避。
- **切断联系**：对方避免与你产生关联，如不回复微信消息、不接电话、甚至直接冲突。

#### 无兴趣指标的原因
- **价值展示方式不对**：如在不合适的场合展示价值，或打破他人隐私。
- **没有及时给出可得性**：如对方对你感兴趣，但你没有及时回应。

### 总结
本节课中，我们学习了兴趣指标和无兴趣指标，这两个指标是社交路口上的红绿灯，指引着我们是前进还是暂停。兴趣指标是对方被你吸引的标志，无兴趣指标是对方对你不感兴趣的标志。当发现无兴趣指标时，要反思原因并改变做法，让绿灯亮起来。
