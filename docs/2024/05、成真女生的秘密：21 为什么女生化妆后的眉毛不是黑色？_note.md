# 课程05：成真女生的秘密 - 为什么女生化妆后的眉毛不是黑色？ 🤔

在本节课中，我们将要学习眉笔的作用、颜色选择以及为什么化妆后的眉毛通常不是纯黑色的。我们将通过简单的解释和对比，帮助你理解这个常见的化妆细节。

## 眉笔的作用 ✏️

眉笔是用来画眉毛的工具，它的样子细细的，像一根笔一样。

为了直观地理解画眉毛的效果，以下是画眉毛与不画眉毛的照片对比。

*   画了眉毛的人看起来精神一些。
*   画了眉毛的人精气神有一点不一样。

上一节我们介绍了眉笔的基本作用，本节中我们来看看眉笔颜色的多样性。

## 眉笔的颜色选择 🎨

眉笔有不同的颜色，例如黑色、灰色、棕色、咖啡色以及浅棕色等。

听到这里你可能会觉得奇怪，为什么眉毛还会有不同的颜色？眉毛不就是黑色的吗？我们都是中国人，黄皮肤黑头发，怎么会是其他颜色呢？

实际情况并非如此。有的女生平时可能会染发，例如染成棕色或浅色的头发。

不同颜色的眉笔就是为了和发色进行搭配。而且，纯黑色的眉笔画起眉毛来就是两条黑黑的眉，挂在脸上有些女生可能会觉得太生硬、不好看。

所以，如果你仔细观察女生的眉毛，大多数都不是那种很黑很深的颜色，而是带有一些其他色调。

本节课中我们一起学习了眉笔的功能和颜色选择的原理。我们了解到，眉笔不仅能修饰眉形、提升精神面貌，其颜色的选择更是为了与发色协调，避免纯黑色带来的生硬感，从而打造更自然、和谐的妆容效果。
