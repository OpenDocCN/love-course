# 19.心灵成像第二期：09_9、心灵成像：意念投射

## 概述

在本节课中，我们将学习如何使用心灵成像的方式传递信息给他人，并尝试用意念去影响他人的思想。

## 1. 意念投射的原理

心灵成像是一种通过心灵屏幕进行视觉化思考的能力。当我们进入天才状态时，我们的心灵屏幕会变得清晰，我们可以将我们的想法和感受投射到心灵屏幕上，并通过心灵屏幕传递给他人。

## 2. 意念投射的方法

### 2.1 视觉化

将你想传递的信息或画面投射到心灵屏幕上，并观想对方接收并理解。

### 2.2 传送感觉

在你的内心中酝酿出你想传递的感觉，然后通过心灵传递给对方。

## 3. 意念投射的练习

### 3.1 请某人为你做一件事儿

选择一个小事儿，例如让对方帮你买一杯咖啡，然后使用意念投射的方式让对方产生这个念头。

### 3.2 用意念传递对某人的欣赏

选择一个你欣赏的人，使用意念投射的方式传递你的欣赏之情。

### 3.3 传递爱给你的宠物、孩子、伴侣或父母

选择一个你关心的人，使用意念投射的方式传递你的爱。

## 4. 总结

本节课我们学习了如何使用心灵成像的方式传递信息给他人，并尝试用意念去影响他人的思想。通过练习，我们可以提升自己的感知力和意念投射能力，从而更好地与他人沟通和交流。
