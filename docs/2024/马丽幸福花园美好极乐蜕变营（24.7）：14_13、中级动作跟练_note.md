在本节课中，我们将要学习一系列中级训练动作。这些动作建立在基本动作之上，旨在帮助学员进一步提升核心肌群的控制力与协调性。课程将包含五个核心动作的详细跟练指导。

## 安全须知与训练原则 ⚠️

运动前请仔细阅读安全须知。

关于中级训练，需要提醒的是，必须遵循循序渐进的训练方法。基本动作必须练习到非常娴熟的程度后，才能开始中级训练的动作。

不要只练习了基本动作三五天，就急于尝试更高级的内容。如果前面的基础没有打扎实，后续的动作将无法正确掌握，发力方式也会不明确。最终会导致技术不扎实，训练效果不佳，停滞在基础不牢、进阶不顺的阶段。这是许多学员曾经犯过的错误，在此提前提醒大家。

## 动作一：拔草练习 🌱

动作要领如下：
*   **呼气时**：上提。
*   **吸气时**：放回原处。

以下是拔草练习的跟练步骤，请跟随指令进行：

第五次呼气上提，吸气放回。
第六次呼气上提，吸气放回。
第七次呼气上提，吸气放回。
第八次呼气上提，吸气放回。
第九次呼气上提，吸气放回。
第十次呼气上提，吸气放回。
第十一次呼气上提，吸气放回。
第十二次呼气上提，吸气放回。

## 动作二：爬楼梯练习 🪜

完成了基础的拔草练习后，我们进入第二个动作——爬楼梯。这个动作要求在一次呼气中，分阶段、有控制地向上提升。

动作分解如下：
*   **呼气时**：分三个阶段上提（可理解为上提一点、再上提一点、上提到最高处）。
*   **吸气时**：一次性放回原处。

动作模式可以概括为：**呼气：上提(阶段1) -> 上提(阶段2) -> 上提(阶段3，最高)；吸气：放松**。

以下是爬楼梯练习的跟练步骤：

爬楼梯练习准备开始。
第一次呼气上提，越提越高，提到最高，吸气放回。
第二次呼气上提，越提越高，提到最高，吸气放回。
第三次呼气上提，越提越高，提到最高，吸气放回。
第四次呼气上提，越提越高，提到最高，吸气放回。
第五次呼气上提，越提越高，提到最高，吸气放回。

爬楼梯练习结束。

## 动作三：一次性提到最高 🚀

接下来，我们练习第三个动作。它与爬楼梯动作相关，但要求简化过程，在一次呼气中直接提到最高点。同时，可以配合身体前后的轻微摆动来帮助找到发力感觉，这个摆动类似于拉丁舞的基本动作。

动作要领是：**一次呼气，直接提到最高**。

以下是该动作的跟练步骤，请尝试配合身体前后摆动：

准备开始。
一次。
两次。
三次。
四次。
五次。
六次。
七次。
八次。
九次。
十次。

完成练习。

## 动作四：三级收紧与爬楼组合 💪

掌握了独立的上提和收紧动作后，第四个动作是将两者进行整合。为了便于理解，我们将收紧的过程也分为三个阶段，并与三阶段上提同步进行。

动作可以分解为以下三步：
1.  呼气，开始收紧并开始上提。
2.  继续呼气，进一步收紧并进一步上提。
3.  呼气末，收紧到最紧，同时上提到最高。
4.  吸气，完全放松放回。

以下是组合动作的跟练步骤：

三级收紧、三级爬楼训练开始。
第一次呼气，收紧同时上提，越收越紧，越提越高，收到最紧，提到最高，吸气放回。
第二次呼气，收紧同时上提，越收越紧，越提越高，收到最紧，提到最高，吸气放回。
第三次呼气，收紧同时上提，越收越紧，越提越高，收到最紧，提到最高，吸气放回。
第四次呼气，收紧同时上提，越收越紧，越提越高，收到最紧，提到最高，吸气放回。
第五次呼气，收紧同时上提，越收越紧，越提越高，收到最紧，提到最高，吸气放回。

三级收紧、三级爬楼训练完成。

## 动作五：八爪鱼练习 🐙

最后，我们练习传说中的八爪鱼动作。这是前面所有技巧的综合与强化，包含强力收紧上提和爆发式推送两个部分。

完整动作流程如下：
1.  **第一次呼气**：一次性夹到最紧，同时提到最高。
2.  **吸气**：保持收紧状态3秒。
3.  **第二次呼气**：将肌肉猛烈地向阴道口方向推送。
4.  **吸气**：完全放松。

以下是八爪鱼练习的跟练步骤：

八爪鱼练习准备开始。
第一次呼气，收紧同时上提。吸气，保持握住3秒（1, 2, 3）。呼气，大力推出，推至阴道口。吸气放松。
第二次呼气，收紧同时上提。吸气，保持握住3秒（1, 2, 3）。呼气，大力推出，推至阴道口。吸气放松。
第三次呼气，收紧同时上提。吸气，保持握住3秒（1, 2, 3）。呼气，大力推出，推至阴道口。吸气放松。
第四次呼气，收紧同时上提。吸气，保持握住3秒（1, 2, 3）。呼气，大力推出，推至阴道口。吸气放松。
第五次呼气，收紧同时上提。吸气，屏住呼吸3秒（1, 2, 3）。呼气推出，大力推至阴道口。吸气放松。

完成八爪鱼训练。
