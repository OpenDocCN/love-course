# 加一情感《宅男宝典：01、情圣方法：15.追求阶段三要素_note

在本节课中，我们将要学习在建立吸引之后，如何顺利进入并度过“追求阶段”。这个阶段是女生对你进行“废物测试”最频繁的时期，处理得当则关系升温，处理不当则前功尽弃。我们将深入剖析这个阶段的核心目标与三个关键行动要素。

上一节我们介绍了如何建立吸引，本节中我们来看看，当女生对我们产生好感后，关系将进入“追求阶段”。这个阶段，是女生提出“废物测试”最多的时候。

她会问诸如“我跟你妈掉水里了，你只能救一个，你救谁？”、“你为什么喜欢我？”、“你和你前女友为什么分手？”等问题。这些问题难以回答，若回答不好，女生就会疏远你。

我们来看看这些问题背后的含义。“我跟你妈掉水里”这个问题，她想了解在你心中，她和你的母亲谁更重要，是在拿自己与你生命中最亲近的女性进行比较。“和前女友为什么分手”这个问题，她想了解分手的原因，以此判断你是否是一个喜新厌旧的人。

其实，所有这些问题归根结底，都指向一个终极问题：**你是不是真的喜欢我？** 她问这些问题的根本目的，是想确认你对她的喜欢是否真诚。这实质上关联着女生两个最底层的需求之一：**靠谱**。

我们追求阶段的核心目的，就是要完成 **DP**。

在吸引阶段之后，女生更在意的便是你是否靠谱，是否真的喜欢她。那么，DP到底是什么？又该如何让女生觉得你是真的喜欢她呢？我们需要从以下三个要素入手。

以下是实现DP的三个核心要素：

1.  **筛选**
2.  **暖男行为**
3.  **走心**

首先，我们来看第一个要素：**筛选**。具体做法是，你要让她感觉到你也在考察她。

为什么要这么做呢？因为只有当你认真对待一段感情时，才会仔细挑选伴侣。如果只是抱着玩玩的心态，就不会如此在意。所以，当你进行筛选时，她会明白你是认真的。

举个例子：我们去商店买东西，挑来挑去，那代表真的想买。如果只是随口说“东西挺好，我们再看看”，往往意味着购买意愿不强。同样，如果你在不了解她时就要确立关系，女生会觉得你不是真的喜欢她。

那么，具体该如何操作呢？以下是几个示例：

*   **示例一**：你问女生：“你谈过几个男朋友？”她回答：“三个。”你可以说：“比我多一个。”然后发一个委屈的表情。接着再问：“你有和男生同居过吗？”她说没有，你回应：“那还好。”她若反问：“你很介意同居吗？”你可以回答：“有点。”
*   **示例二**：如果女生回答曾与男生同居过，你可以问：“多久？”她答：“两个月。”同样的问题“你很介意同居吗？”，你可以回答：“也还好。”

关键在于，此时女生能感知到你是在意的，但你并没有直接说出来。这会让她觉得你是真的喜欢她。能理解吗？其实我们喜欢一个人时，确实会在意对方的某些过去或特质，但若完全不说，会给女生一种很假、像是“骗炮”的感觉，因为只有目的不纯的人才表现得完全不在意。

你在意她的一些事情（例如情史、身高、体型、是否擅长家务、性格是否温柔），但最终选择了接受，这在她看来，就是**真的喜欢**。

接下来，我们谈谈第二个要素：**暖男**。这个行为其实很多人都懂，但关键在于时机。到了这个阶段，你可以将之前追求女生的一些方法运用起来。

例如对她好、嘘寒问暖、关心照顾、赠送小礼物、把她的事放在心上。

分享一个案例：我曾追求一个女生，大家聚餐时我注意到她喝汤会把葱花挑出来。后来我单独约她吃饭，点完菜后，我跟服务员补充了一句：“所有的菜里都不要放葱。”她问我：“你也不喜欢吃葱吗？”我回答：“不是呀，是你不喜欢吃。”她很惊喜地问：“你怎么知道的？”我说：“上次一起吃饭，我看你把葱都挑出来了。”后来，她成了我的女朋友。

你关心她的小细节，会让她非常感动，她会确信你是真的喜欢她。

我们很多人在学习了情感方法后，会认为“跪舔”是不对的，觉得“百般真情留不住，唯有套路得人心”。

有这种转变是一种进步，但理解尚不全面。对女生好本身没有错，错的是时机。一开始就“跪舔”是无效的。女人当然希望男生对她好，这也能促进关系，但我们必须到**这个阶段**才开始做。此时做，才有效果。

女生相对男生更为矜持。这种矜持可以延长男生追求她的时间，让她有更多机会观察这个男生，确定自己是真正被吸引，而不是被欺骗。这个过程也能增加男生的投入，而男生投入越多，就会越在意，后续的关系也会更稳固。

即使女生喜欢你，通常也不会非常主动。在这个过程中，男生需要主动一点。请注意：**男生的主动是“靠近”，女生的主动则是在男生靠近时给予“配合”**。

当女生主动时，我们千万不要直接去升高关系或推进关系。正确的做法是：**后退**。

因为当女生主动时，我们若直接升高，她会觉得我们很随便，或者怀疑自己这样做是否正确。而当我们选择后退时，她会觉得很有意思，挑战性更强。

很多时候，女生会故意挑逗你。如果我们立刻主动推进，她会觉得“这个人真饥渴”。而当我们先后退，之后再找机会推进关系时，整个过程就会显得更稳健、更自然。

**本节课总结**

本节课中，我们一起学习了追求阶段的核心策略与行动要点。

1.  追求阶段的核心目标是完成 **DP**，即让女生感觉你是“真的喜欢”她。
2.  实现DP有三个要素：**筛选**、**暖男行为**和**走心**。
3.  **筛选**之所以有效，是因为只有当我们真心喜欢一个人时，才会认真考察对方。
4.  对女生好（**暖男行为**）是有效的，但必须在建立吸引后的**追求阶段**进行。
5.  在互动中，**男生的主动是“靠近”，女生的主动是在你靠近时给予“配合”**。
6.  当女生主动时，正确的应对策略是**先后退**，以保持框架和吸引力。

掌握这三点，你便能更从容地应对追求阶段的挑战，稳步推进关系。
