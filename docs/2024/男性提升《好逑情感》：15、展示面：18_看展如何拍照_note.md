# 男性提升《好逑情感》：15、展示面：18_看展如何拍照

🎼如何拍摄展览照片？🎼

在本节课中，我们将学习如何在参观展览时拍摄出有吸引力的照片。

## 1. 站在画旁构图

**公式**：构图 = 站位 + 视角

站在画作旁边，选择合适的视角，利用画作的线条和形状来构图。

## 2. 使用手机拍摄

**代码**：手机拍照

使用手机拍摄，确保手机稳定，避免抖动。

## 3. 隐藏面部

**公式**：隐藏面部 = 背对镜头

如果不想在照片中露脸，可以背对镜头拍摄。

## 4. 大场景构图

**公式**：大场景构图 = 利用环境

利用展览的环境，如墙角等，进行大场景构图。

## 5. 拍摄意境感

**公式**：意境感 = 遮挡 + 实况

利用墙角遮挡，拍摄出意境感。打开实况模式，捕捉人物走过时的瞬间。

## 6. 长曝光

**公式**：长曝光 = 慢速快门

选择长曝光模式，拍摄出模糊的背景，增强照片的意境。

## 总结

本节课中，我们一起学习了如何在参观展览时拍摄出有吸引力的照片。通过站在画旁构图、使用手机拍摄、隐藏面部、大场景构图、拍摄意境感和长曝光等技巧，我们可以拍摄出更加精美的展览照片。希望这些技巧能够帮助你在下一次参观展览时拍出满意的照片。
