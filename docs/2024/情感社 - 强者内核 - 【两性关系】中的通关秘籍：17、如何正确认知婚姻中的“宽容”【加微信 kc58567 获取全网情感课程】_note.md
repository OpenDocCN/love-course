# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：17、如何正确认知婚姻中的“宽容”

## 概述
在本节课中，我们将探讨如何在婚姻中正确理解和实践宽容。

## 核心概念
**宽容**：在婚姻中，宽容意味着在对方犯错或行为不符合期望时，能够保持理解和接纳的态度。

### 1. 宽容的内核
- **公式**：宽容 = 内心中的“怂” + 得过且过的态度
- **代码**：`courage = fear + tolerance`

### 2. 宽容的实践
以下是实践宽容的几个步骤：

#### 2.1 了解自己和对方
- **步骤**：
  1. 明确自己能够容忍什么，反感什么。
  2. 了解伴侣的脾气秉性、智商情商。
  3. **公式**：了解 = 自我认知 + 伴侣认知

#### 2.2 降低期待
- **步骤**：
  1. 将期待细化到生活的细节中。
  2. 分批次融入生活。
  3. **公式**：降低期待 = 细化期待 + 分批融入

#### 2.3 划分大事和小事
- **步骤**：
  1. 大事：不经常发生、突如其来的、涉及他人。
  2. 小事：每天发生的细节。
  3. **公式**：大事 = 不经常发生 + 突如其来 + 涉及他人
  4. **公式**：小事 = 每天发生 + 细节

### 3. 宽容的底线
- **步骤**：
  1. 明确自己的底线。
  2. 理解并接受现实。
  3. **公式**：底线 = 自我认知 + 现实认知

## 总结
本节课中，我们一起学习了如何在婚姻中正确理解和实践宽容。希望这些知识能够帮助你在婚姻中找到自己的平和状态。
