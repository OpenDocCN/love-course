# 男哥内核3.0初级班：内部课：15.升高关系全流程—飞龙在天

## 概述

在本节课中，我们将学习如何通过“飞龙在天”系统构建恋爱关系，并探讨如何通过“过程大于结果”的原则来提升关系温度。

## 核心概念

**过程大于结果**：在沟通中，注重过程而非直接表达结果，让对方在参与过程中逐渐接受和认同。

## 1. 聊天技巧

**聊他聊我聊我们**：在聊天中，先聊对方，再聊自己，最后聊彼此。

**病毒聊天**：通过幽默、夸张等手法，让对方在轻松愉快的氛围中接受你的观点。

## 2. 升温关系

**升温感情**：通过聊天技巧，让对方逐渐对你产生好感。

**升温感情失败的原因**：

* 不知如何升温感情，害怕出错。
* 知道如何升温感情，但对方拒绝。

**解决方法**：

* 过程大于结果：注重过程，而非结果。
* 楼梯变成抛物线：避免直接升温，而是循序渐进。

## 3. 过程大于结果

**例子**：

* 男人：兄弟，今天晚上要不要喝酒？
* 女人：宝贝在干嘛呢？

**分析**：

* 男人倾向于直接表达结果，而女人更注重过程。

## 4. 升高关系流程

**陌生阶段**：

* 刚开始觉得你蛮高冷的，没想到你这么逗逼。

**熟悉阶段**：

* 我们有要看过三国呀，那个。诶。张飞总是在说。俺也是俺也是。

**暧昧阶段**：

* 对了，你多高啊？电梯脚尖刚刚好。

**情人阶段**：

* 你会做饭吗？有空一起做饭。

## 5. 总结

本节课中，我们学习了如何通过“飞龙在天”系统构建恋爱关系，并探讨了“过程大于结果”的原则。通过掌握这些技巧，你可以更好地提升关系温度，走向成功。
