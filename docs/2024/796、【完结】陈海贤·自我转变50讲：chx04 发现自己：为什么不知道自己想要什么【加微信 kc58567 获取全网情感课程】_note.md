# 796、【完结】陈海贤·自我转变50讲：chx04 发现自己：为什么不知道自己想要什么

欢迎来到自我转变五十讲。你好，我是陈海贤。上节课我们介绍了自我转变的缘起，是你内心有个我想要需要去表达和实现。虽然在转变最初你不一定认得他，或者就算认得他，你仍然会抱有一种不合时宜的羞愧。

**转变过程**：让我想要逐渐清晰并引导现实的过程。

**常见问题**：在转变期，人们经常会说“我不知道自己想要什么”。

**原因分析**：除了自我尚未成型，这背后还有什么样的心理动力？

**案例说明**：通过一个女士的访谈，说明环境、标签和努力如何影响一个人的自我认知。

**环境因素**：成长环境、原生家庭、社会观念等。

**标签影响**：穷、女性等标签对个人认知的影响。

**努力与挑战**：努力挣钱、追求梦想、面对现实等。

**转变过程**：从模糊到清晰，从迷茫到坚定。

**核心概念**：

* **自我认知**：通过清晰的我想要勾勒出自我。
* **适应矛盾**：自我与环境之间的永恒矛盾。
* **选择与代价**：在知道而可能求不得的痛苦和不知道的痛苦之间，选择后者。

**总结**：

* **自我转变**：让我想要从模糊变得清晰，从而发现自己的过程。
* **新自我建立**：通过努力在现实中为我想要寻找出路，建立新自我。

**思考**：

* 你不知道自己想要什么，还是不敢知道自己想要什么？
* 如果有两个选项，第一个选项是我不知道自己想要什么，第二个选项是我知道自己想要什么，却需要去面对更多的烦恼、冲突和痛苦，你会做什么选择？
