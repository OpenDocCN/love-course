# 27、林夕-高质量关系课程：1.25-第25课《挽回流程及关键时刻》📚

## 概述
在本节课中，我们将学习挽回关系中的关键时刻，以及如何正确处理这些关键时刻。

## 关键时刻识别
**公式**：关键时刻 = 冲突 + 关键节点

### 排斥期
1. **每一次联系都是关键时刻**：找准冲突，用对方希望的方式联系。
2. **见面**：分手后的见面是关键时刻，表现好可能直接复合。
3. **再度排斥**：时间是最好的缓和剂，根据排斥程度计算复联时间。
4. **对方突然回应**：从排斥期过渡到朋友期，保持平常心沟通。

### 朋友期
1. **自述**：自述不要多于对方。
2. **说话频率**：与对方持平。
3. **主动沟通**：遵循“进二推一”法则。
4. **内容相关**：分享与对方需求相关的延展话题。
5. **暧昧话题**：出现暧昧话题，谨慎对待，懂得调情。

### 暧昧期
1. **发生亲密关系**：正确看待亲密关系，避免成为炮友。
2. **关键谈话**：发生前进行关键谈话，避免默认为炮友。

### 情侣期
1. **重塑恋爱格局**：挽回成功后的三个月内，学习情感关系维系技能。
2. **重新引导投资**：重塑价值，学会冲突管理。

## 总结
本节课中我们一起学习了挽回关系中的关键时刻，以及如何正确处理这些关键时刻。希望这些知识能帮助你在挽回关系中取得成功。
