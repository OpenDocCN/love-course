# 梵公子 情绪价值核心课【向上社交破圈升思维】：10.自身定位模型及人生规划

## 概述
在本节课中，我们将学习如何进行自身分析以及目标的确立，以便在向上社交中找到最适合的对象和规划人生。

## 自身分析
在进行自身分析时，我们需要考虑自身的价值和年龄，以选择最适合的对象。

### 自身分析分类
以下是四种自身分析分类：

1. **价值高，年龄小**：这类人群在婚恋市场上拥有绝对的话语权，可以选择自己喜欢的人。
2. **价值高，年龄大**：这类人群应追求有恋爱体验感的婚姻，考虑与条件相当或更差的男生建立稳定关系。
3. **价值低，年龄小**：这类人群应追求物质与结果导向，选择能给予自己大额付出的男生。
4. **价值低，年龄大**：这类人群应尽快结婚，不要对男生要求过高，以尽快找到合适的伴侣。

## 人生规划
在明确自身情况后，我们需要规划人生，以实现开心快乐的目标。

### 人生规划要点
1. **目标导向**：人生追求的意义是为了开心快乐，每个人在不同阶段的需求不同。
2. **结合实际情况**：根据自己的条件，选择最适合自己的东西。
3. **放弃不切实际的想法**：如果什么都想要，最终可能什么都得不到。

## 总结
本节课中，我们一起学习了如何进行自身分析以及人生规划。希望同学们能够结合自己的实际情况，找到最适合的对象，规划出属于自己的幸福人生。
