# 海蓝老师情绪梳理个案课：034-查不出病因的病人第三课完成版 🎨

## 概述
在本节课中，我们将通过个案分析，学习如何通过情绪梳理帮助病人理解自己的情感，并找到解决问题的方法。

## 奶奶的情感梳理
**以下是奶奶的情感梳理：**

1. **奶奶的照顾**：奶奶对我照顾很多，画面中有很多是和奶奶在一起，奶奶哄我睡觉。
2. **奶奶的爱**：奶奶也是爱我的。
3. **奶奶的感受**：奶奶觉得我认可她，感谢我能记得她。
4. **奶奶的表达**：奶奶可能不太善于表达。

**公式描述**：
\[ \text{奶奶的照顾} + \text{奶奶的爱} = \text{奶奶的认可} + \text{感谢} \]

## 姥姥的情感梳理
**以下是姥姥的情感梳理：**

1. **姥姥的忙碌**：姥姥一直在忙前忙后。
2. **姥姥的强大**：姥姥在我心里是一个特别强大的人，内心非常强大。
3. **姥姥的担忧**：妈妈发生事情，姥姥最着急。
4. **姥姥的努力**：姥姥努力去做事情，照顾我。
5. **姥姥的坚强**：姥姥坚强地撑起这个家。
6. **姥姥的担忧**：姥姥担心这件事对我造成创伤。

**公式描述**：
\[ \text{姥姥的忙碌} + \text{姥姥的强大} + \text{姥姥的担忧} = \text{姥姥的努力} + \text{姥姥的坚强} + \text{姥姥的担忧} \]

## 老师的情感梳理
**以下是老师的情感梳理：**

1. **老师的责任**：老师知道家里发生了严重的事情，知道妈妈很危险。
2. **老师的无奈**：爸爸送我迟到频率太高，老师无法理解。
3. **老师的沟通**：老师私下里和爸爸沟通过，希望他不要迟到。
4. **老师的期望**：老师希望我能够得到家长的照顾。
5. **老师的关心**：老师每次说我的时候，也是希望能够尽到责任。
6. **老师的误解**：老师从来没有觉得我是一个没有妈的孩子。

**公式描述**：
\[ \text{老师的责任} + \text{老师的无奈} + \text{老师的沟通} + \text{老师的期望} + \text{老师的关心} = \text{老师的误解} \]

## 总结
本节课中，我们一起学习了如何通过情绪梳理帮助病人理解自己的情感，并找到解决问题的方法。通过分析奶奶、姥姥和老师的情感，我们了解到每个人都有自己的情感和想法，我们需要尊重和理解他们，才能更好地帮助他们解决问题。
