# 《朕哥8套情感合集》：01.核心私密课之高情商聊天术：05.第四章：最佳聊天姿势（下）

## 概述
在本节课中，我们将学习如何通过高情商的聊天技巧，提升与男性的沟通效果，建立更紧密的关系。

## 聊天姿势一：往后想三步
**公式**：明确聊天目的 → 引导话题方向 → 达成预期效果

**步骤**：
1. 明确你想要达到的目的和效果。
2. 根据目的，引导话题方向。
3. 通过话题引导，达成预期效果。

## 聊天姿势二：正反话交错，推拉结合
**公式**：正话 + 反话 → 推拉 → 模糊态度

**步骤**：
1. 使用正话夸赞对方，例如：“你很帅气，很有男人魅力。”
2. 在对方得意时，突然使用反话，例如：“哈哈，从来没有女人这么夸过你吧。”
3. 通过委婉的推拉方式，让对方无法确定你的真实态度。

## 聊天姿势三：共谋
**公式**：共同秘密 + 独特经历 → 亲密关系

**步骤**：
1. 与对方分享私密笑话、绰号等，建立共同秘密。
2. 一起做一些独特的事情，例如吐槽共同朋友、分享过去趣事等。
3. 通过共同经历，拉近彼此关系。

## 聊天姿势四：撬动对方更多回应
**公式**：倾听 + 带窗口的夸赞 + 引导选择 + 激发好奇心 → 增强互动

**步骤**：
1. 倾听对方说话，让对方多说话。
2. 使用带窗口的夸赞，例如：“你果然没有让我失望，我就知道你不是一般的男人。”
3. 引导对方做选择，例如：“你觉得我穿哪件衣服好看？”
4. 激发对方好奇心，例如：“你猜我刚才看见谁了？”

## 聊天姿势五：避免触碰男人的反供养者机制
**公式**：避免物质要求 + 尊重对方付出 → 建立基因提供者形象

**步骤**：
1. 避免直接要求对方物质支持。
2. 尊重对方的付出，例如为他买的东西表示感谢。
3. 表达对对方基因的欣赏，例如：“我觉得你很帅气，很优秀。”

## 聊天姿势六：高能量姿势
**公式**：避免逻辑争辩 + 激发荷尔蒙 → 提升吸引力

**步骤**：
1. 避免与对方争对错，例如：“你喜欢萝卜不喜欢梨，那就吃萝卜吧。”
2. 激发对方荷尔蒙，例如：“我想吃个梨，和你的舌头分享。”

## 总结
本节课中，我们学习了五种最佳聊天姿势，包括往后想三步、正反话交错、共谋、撬动对方更多回应、避免触碰男人的反供养者机制和高能量姿势。通过掌握这些技巧，我们可以提升与男性的沟通效果，建立更紧密的关系。
