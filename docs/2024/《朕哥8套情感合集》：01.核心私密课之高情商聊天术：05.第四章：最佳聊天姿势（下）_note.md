# 《朕哥8套情感合集》：01.核心私密课之高情商聊天术：05.第四章：最佳聊天姿势（下）


在本节课中，我们将继续探讨高情商聊天的核心姿势。我们将学习如何通过“往后想三步”、“正反话交错”、“共谋”等策略，以及如何撬动对方更多回应、避免触碰男人的“反供养者机制”，并最终掌握“拼荷尔蒙而非拼逻辑”的高能量聊天心法。这些技巧将帮助你引导对话方向，建立深度连接，并牢牢掌握关系中的主动权。


## 1. 往后想三步：引导对话方向

上一节我们介绍了聊天的基本心态，本节中我们来看看如何有策略地引导对话。聊天不应只局限于当下，你需要具备“往后想三步”的心态，明确每次对话的最终目的。

就像下棋一样，你需要提前规划好后续的步骤。你的每一句话都应服务于你最终想达成的目标。

以下是具体应用方法：

*   **明确目标**：在聊天前，先想清楚你希望将话题引导至何处。例如，如果你不想与对方发展过快，就应避免主动谈及同居或亲密关系等话题。
*   **正向引导**：如果你想达成某个具体目标，比如希望对方约你吃饭，就可以在聊天中向这个方向引导。当对方问“在干嘛”时，你可以回答：“正在想晚上吃什么好呢。” 然后顺势问：“你准备吃什么？让我参考一下。”
*   **话题靠拢**：如果你想引导对方表白，就可以多聊与爱情相关的话题。例如，探讨“什么是爱”，并自然地问：“你觉得向一个人表白是什么感受？”
*   **高级技巧**：更高明的方式是引导对方自己说出你想听的话。例如，如果你希望对方多陪你聊天，可以聊“灵魂伴侣”的状态，并问：“你有没有遇到过那种特别聊得来的人？”

掌握这种引导能力需要大量的实战经验，但首先要在脑中建立“想远一步”的意识，而不是只顾当下聊得开心。


## 2. 正反话交错：推拉结合的艺术

仅仅明确方向还不够，我们还需要让对话充满张力。聊天不能全是好话（“拉”），也不能全是打压的话（“推”），需要将正反话交错使用，这就是“推拉结合”。

推拉的目的是让对方情绪产生波动，无法完全拿捏你的态度，从而对你保持好奇和探索欲。

以下是推拉结合的例子：

*   **先扬后抑**：你可以先说：“你真的很帅气，很有男人魅力。”（拉）在对方得意时，紧接着说：“不过，估计从来没女生这么夸过你吧？毕竟审美达不到我这个水平，可能看不出你的帅。”（推）
*   **制造模糊**：通过这种委婉的推拉，让对方搞不清你是在夸他还是调侃他，也搞不清你对他到底有多喜欢。这其实也表达了好感，但方式更含蓄，容易让男人浮想联翩。

我们后续会有专门课程详细讲解推拉技巧。这里你只需先建立一种心态：在聊天中要有意识地去“推”和“拉”。缺乏这种意识，对话会显得无趣，也难以达到理想效果。


## 3. 建立共谋：创造独特联结

除了语言上的技巧，关系上的独特绑定也至关重要。我们可以通过“共谋”来建立一种私密的、战友般的关系。

共谋是指与对方共同拥有一些私密的、专属的体验或秘密，这能快速拉近两人的心理距离。

以下是建立共谋的方法：

*   **创造专属记忆**：可以是一起给同事或朋友起只有你们懂的外号，一起吐槽某个共同认识的人，或者分享自己以前的糗事。
*   **一起“做点坏事”**：这里的“坏事”是带引号的，指的是一些无伤大雅但有点刺激、打破常规的小事。比如，同事一起偷偷溜班几分钟，同学一起谋划逃一次无关紧要的课。
*   **拉近关系的本质**：拉近两人关系的最好方式，就是一起去“干一点坏事”。这能建立一种亲近的、同盟般的友谊。日后提起这些事，都会让彼此感觉感情很近。

如果对方回复慢或聊天冷场，可以尝试构思一点这样的“小坏事”，这往往能触动对方内心想要释放压力、寻求刺激的一面，让他更乐意与你互动。


## 4. 撬动对方回应：让对方多说

在互动中，谁说得更多，谁往往就更被动。高段位的聊天者懂得如何撬动对方更多地表达。

核心原则是：**让自己话少，让对方话多**。对方回应越多，投入的精力与情感就越多，你们的关系也就越紧密。

以下是三个撬动回应的技巧：

*   **带窗口的夸赞**：不要只说“你真棒”，而是说：“你果然没让我失望。” 或“我就知道你不是一般的男人。” 这种夸赞会留下“窗口”，引发对方好奇（“为什么是‘果然’？”），从而促使他追问，延伸话题。
*   **让对方帮你选择**：利用男性倾向于为女性做决定的心理。你可以说：“我特别怕做选择，以后你来帮我选好不好？” 然后具体化，比如发两件衣服图片问他：“你觉得我穿哪件好看？”（这也是一种撩拨）。
*   **激发对方好奇心**：用悬念开头。例如：“你猜我刚才看见谁了？” 或“你知不知道我在你眼里看到了什么？”（线上线下均适用）。又或者：“你今天是不是忘了什么事？”（他问什么事后，你可以说“比如给我发晚安”）。这些话都能有效撬动对方回应。

此外，聊天中要“敢于尴尬”。不要一冷场就急着找话题，有时可以坦然接受沉默，让对方来打破尴尬。


## 5. 避开雷区：不要触碰“反供养者机制”

聊天中有些雷区绝对不能踩，最重要的就是避免触发男人的“反供养者机制”。

从男性心理底层逻辑看，男人通常希望自己是“基因提供者”（凭借魅力、外表、性格吸引异性），而非“物质提供者”（仅凭金钱和资源被需要）。触发后者机制会让他感觉被工具化，非常反感。

以下行为会让他觉得被当作“供养者”：

*   **直接索取**：开口要钱、要贵重礼物。
*   **价值贬低**：说他“穷”、“没用”，或拿他与经济条件更好的男人比较。
*   **无视付出**：对他送的东西嫌不够好、不够贵。

**核心心法**：无论你真实看重的是什么，在互动中都要让他感觉，你看重的是他这个人本身（他的基因、魅力、性格），而不是他的物质条件。多表达对他个人的欣赏，例如：“你穿这衣服真有品味，显得特别帅气。”


## 6. 高能量心法：拼荷尔蒙，而非拼逻辑

最后，我们来到最高阶的聊天心法。低端的聊天是和男人讲道理、拼逻辑；高端的聊天是和男人拼荷尔蒙，调动他的情绪与感觉。

男性思维偏重“荷尔蒙思维”（感性、冲动）。不要陷入与他争对错的逻辑辩论中，你的目标是让他“荷尔蒙上头”。一旦他情绪上头，你的观点就更容易被接受。

**应用场景（注意分寸）**：
*   **在已确立感情关系中**，可以大胆使用。例如，不说“你为什么不喜欢梨”，而是说：“我想吃个梨，然后和你分享。” 或者：“真想念你身上的味道。”、“我昨晚梦到你了，梦到一些害羞的事情。”
*   **在未确立关系或暧昧初期**，需要委婉化、减半使用。例如，只说：“我昨晚居然梦到你了，梦到我们一起在……” 留下想象空间即可。

**核心原则**：少讲道理，多营造氛围和感觉。只要他喜欢你，甚至只要他不讨厌你，这类调动情绪的话语都能产生强大效果。关键在于根据你们的关系阶段，精准拿捏说话的“度”。


本节课中我们一起学习了高情商聊天的多项核心姿势：从有目的地“往后想三步”引导对话，到运用“推拉结合”制造吸引力；从通过“共谋”建立独特联结，到巧妙“撬动对方更多回应”；我们明确了要避开“反供养者机制”的雷区，并最终掌握了“拼荷尔蒙而非拼逻辑”的高能量心法。记住，聊天不仅是信息交换，更是情感和关系的编织。将这些心态与技巧融入实践，你将成为对话的主导者。
