## 增加曝光率的方法

**方法描述**：多参加婚礼，尤其是当地有影响力的人的子女婚礼，这些场合是大型社交现场，可以结识各色人等。

**核心概念**：
- **公式**：增加曝光率 = 参加婚礼 + 结识新人脉

### 2. 参加品酒活动
**方法描述**：报名参加酒庄品酒活动团，这些活动通常吸引精致的小资人群，可以认识活动组织者，拓展人脉。

**核心概念**：
- **代码**：`expand_network = join_wine_tasting`

### 3. 加入高端社群
**方法描述**：加入跑车俱乐部、游戏群等高端社群，这些社群成员通常条件较好，可以拓展人脉。

**核心概念**：
- **公式**：拓展人脉 = 加入高端社群 + 拓展社交圈

### 4. 参加高端插花局
**方法描述**：参加高端插花局，通过认识年纪大的姐姐，链接他们身后的资源。

**核心概念**：
- **代码**：`connect_resources = join_flower_arranging`

### 5. 参加高端造型工作室
**方法描述**：参加高端造型工作室，认识其中的知名人士，拓展人脉。

**核心概念**：
- **公式**：拓展人脉 = 参加高端造型工作室 + 认识知名人士

### 6. 参加知名餐厅活动
**方法描述**：参加知名餐厅活动，留下经理联系方式，参与活动，拓展人脉。

**核心概念**：
- **代码**：`expand_network = join_restaurant_event`
