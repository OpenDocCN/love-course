# 张院长《技巧软功》课程22：体位课高级版之躺姿 🛌

在本节课中，我们将要学习高级体位课程中的“躺姿”系列。躺姿是一种非常浪漫的姿势系列，既适合在前戏中使用，也适合在激烈运动后的舒缓阶段，或在缠绵的后戏中运用。本节课将介绍两种核心的躺姿及其变式，帮助你与伴侣建立更深的连接。

## 基础姿势：海星式 ✨

上一节我们介绍了躺姿的概述，本节中我们来看看第一种基础姿势——海星式。这个姿势结合了剪刀式与海星式的特点，旨在为双方提供稳定且亲密的接触。

以下是海星式的具体步骤：

1.  男生首先平躺在床上，双腿打开，为女生留出进入的空间。
2.  女生进入后，男生的一条腿在上，女生的一条腿在下，形成交叠状态。
3.  女生的另一条腿放在男生的另一条腿上面。
4.  此时，女生的右手抓住男生的左手，通常可以十指相扣。
5.  下体结合后，女生躺下，可以进行前后的抽动。
6.  女生可以在基础上，腿部稍用力，进行旋转摩擦。

**核心要点**：女生动作应轻柔，只需活动胯部和私密处进行旋转摩擦，避免整个身体大幅移动，以防男生的阴茎滑出。这样可以让男生的阴茎在体内有旋转的感觉，同时也能刺激到睾丸。

## 变式姿势：勺子式 🥄

掌握了基础的海星式后，我们来看看一种更侧重亲密与配合的变式——勺子式。这个姿势要求双方侧躺，如同两只勺子叠在一起，能实现深度的身体接触与情感交流。

以下是勺子式的具体步骤：

1.  女生躺在男生前方，男生侧身从后方贴近。
2.  男生的手可以自然平放，女生则舒适地躺在这只手臂上。
3.  男生的腿弯曲，贴合女生的身体曲线。
4.  在抽动过程中，配合是关键。当男生向前插入时，女生的臀部应向后送，形成对抗动作。
5.  如果双方同时向前，男生会缺乏抽插的感觉。

**核心要点**：女生在过程中可以回头与伴侣对视。男生的手不要闲着，可以抚摸女生的腿部、胸部，或轻抚耳朵、颈脖，也可以进行接吻，这些都是很好的情感链接与交流方式。通过这种配合，双方能达到很好的默契。
