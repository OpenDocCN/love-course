# 《高段位男神进阶营》——浪哥出品：17.男神进阶营《内核篇》：17 唤醒你的社交自信

## 概述

在本节课中，我们将深入探讨社交自信的概念，了解其形成过程，并学习如何正确建立社交自信。

## 社交自信的定义

**社交自信**是指一个人不向外界寻求自我价值的认可，而是保持理性，在不受外界影响的情况下接受反馈并调整自我。

**公式**： 社交自信 = 自我认知 + 理性反馈 + 自我调整

## 社交自信的形成过程

1. **自我认知**： 了解自己的优点和不足，接受自己的不完美。
2. **理性反馈**： 接受外界的反馈，并从中学习。
3. **自我调整**： 根据反馈调整自己的行为和态度。

## 如何建立社交自信

### 认知层面

1. **接受自己的不完美**： 不要过分追求完美，接受自己的优点和不足。
2. **理性看待反馈**： 不要将外界的评价作为衡量自己的标准，理性看待反馈。
3. **自我调整**： 根据反馈调整自己的行为和态度。

### 执行层面

1. **直面恐惧**： 不要逃避恐惧，勇敢面对。
2. **打破自恋**： 不要过度关注自己的形象，认识到自己并非世界中心。
3. **提升能量现实**： 提升自己的认知层次和表达能力，让自己更具吸引力。

## 能量现实

**能量现实**是指在自我认知维度下的现实世界，通过和一个人的接触，我们可以感受到他的能量现实。

**公式**： 能量现实 = 认知层次 + 表达能力 + 气场气质

## 总结

本节课中，我们学习了社交自信的定义、形成过程以及建立方法。希望同学们能够通过学习和实践，不断提升自己的社交自信，成为一个更加自信、从容的人。
