# 钱进社 - 成为优质且最具竞争力的【社会人】：81、如何抵御会产生消耗的诱惑

## 概述

在本节课中，我们将探讨如何抵御生活中各种诱惑，从而保持专注和目标导向。

## 1. 诱惑无处不在

诱惑存在于生活的方方面面，从小到克服懒惰和贪玩，大到面对人生重大选择。**公式**：诱惑 = 懒惰 + 贪玩 + 重大选择。

## 2. 抵御诱惑的方法

### 2.1 设定长远目标

抵御诱惑最根本的方法是设定一个长远的目标。**公式**：目标 = 长远 + 实操 + 有意义。

### 2.2 自我定位

明确自己的强项和弱项，避免盲目跟风。**公式**：自我定位 = 强项 + 弱项 + 实践检验。

### 2.3 选择合适的环境

与三观正的人合作，避免不良环境的影响。

### 2.4 保持理性

面对诱惑，保持理性，避免盲目跟风。

## 3. 抵御诱惑的益处

### 3.1 提高专注力

抵御诱惑有助于提高专注力，更好地实现目标。

### 3.2 增强安全感

通过抵御诱惑，建立安全感，避免内耗。

### 3.3 提升能力

抵御诱惑有助于提升自身能力，实现人生价值。

## 总结

本节课中，我们学习了如何抵御诱惑，保持专注和目标导向。通过设定长远目标、明确自我定位、选择合适的环境和保持理性，我们可以更好地应对生活中的诱惑，实现人生目标。
