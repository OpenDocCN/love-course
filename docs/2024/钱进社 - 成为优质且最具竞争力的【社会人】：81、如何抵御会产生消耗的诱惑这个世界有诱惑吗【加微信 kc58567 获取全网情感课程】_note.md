# 钱进社课程81：如何抵御会产生消耗的诱惑 🛡️

在本节课中，我们将要学习如何识别并抵御生活中那些会消耗我们能量和时间的诱惑。诱惑无处不在，从微小的拖延到重大的人生选择，理解其本质并建立有效的防御机制，对于个人成长和目标的实现至关重要。

## 诱惑的本质与影响

诱惑以各种形式分布在我们的生活中。小到克服懒惰和贪玩，例如想多睡一会儿、多玩一会儿游戏或多网购一会儿。这些行为本身没有根本性错误，但如果控制不当，积累起来的负面影响会消耗巨大的能量。

好事的积累能量很大，坏事的积累能量同样很大。一件小事看似不会对人生产生本质改观，但积累起来的能量就不同了。在自我要求层面，如果每一件小事都给自己找理由放纵，等待你的只会是越来越松散的状态。

我们其实没有那么多时间可以浪费。每一次浪费都会加倍地让自己失去更多的选择权。你无法克服某种模式，就等同于自动进入了另一种模式。例如，你无法克服勤奋，就等同于自动进入了懒惰。

顺应人性的所有思想和行为都无法让你超越别人，让你显得与众不同。这一点很难，因为顺应人性是舒服的、本能的。如果你不能反其道而行，变成一个勤奋的人，那么你就只可能变成人云亦云、很普通的那个，最终导致相应的结果。

## 抵御诱惑的根本方法

摆脱诱惑是我们每个人都要做到，而且必须做到的事情。因为诱惑可大可小。往小说，只是摆脱一些消费性质的诱惑，例如美食、垃圾食品、好笑的综艺等。往大说，是面对人生重大课题以及左右你选择、失去和得到的事情，这会直接影响你的人生走上完全不同的轨迹。

每一件小事之下的反思和选择，整体来看，会让每一个人生真正走上不同的轨迹。每个人都可以回想一下自己：有的人做前端工作，有的人做后端工作，有的人的工作持续在进阶，拥有不同的选择权，慢慢积累不同的资源。而有的人从工作那一刻开始，就基本上再也没有什么新的可能了。

回想自己是如何走上这条工作之路，如何变成这样的工作氛围，以及如何得到这样的结果。是不是在很多忽略的细节和小事里，以及有很多外人对你的评价和三观上的左右？你选择听还是不听，选择接纳还是屏蔽他们的意见？所有这一切小事的细节，最终汇总成你选择了什么样的工作，走上了什么样的路，得到了什么样的结果。

抵御诱惑最根本的方法是：**有一个长远的大目标**。这一点非常非常重要，也非常非常致命。

90%，甚至99%的人都想要过上幸福的生活。所谓幸福的生活，就是没有什么可以焦虑的，没有让自己害怕的。例如家庭幸福、恋爱顺利、工作顺利、还能定时定点升职加薪、没有过多的人际纷扰等。

但为什么现在90%的人都反过来了？焦虑很多，恐惧很多，内耗很多。为什么大家都反过来了？而且为什么这么多人的焦虑、恐惧、内耗不会有任何正向答案？就是因为这一部分人他们只有**设想**，但是没有**目标**。

我反复强调目标和设想的区别就在这里。目标和设想，谁不会想？99%的人都想要有一个幸福可控的人生。那为什么这些99%的人都不能如愿以偿呢？关键点就在这里。

“我想要升职。”“我想要换个环境，我觉得这个环境太差了。”“我想要这个公司怎么怎么样。”“我想要老板怎么怎么样。”“我想要学历怎么样。”“我想要同事怎么样。”然后他会发现现实全是反着来的。他越想要什么，越得不到什么；越想要接近什么，越什么跟他无关。

为什么会这样？他的每一步可能都是七拐八拐的，每一步踩的都不是精准的，最终可能就形成这样的结果。当他意识到这样的结果让人非常懊恼、手足无措的时候，他就开始想了。但是他所有的想，只要停留在设想层面，再过十年，他依旧是一年前今天自己这副样子。十年后的自己再照镜子，可能是放弃挣扎了，但不会有本质的改变。

就是因为他没有一个长远的大目标做积淀。这是非常重要的，是不能忽略掉的一个环节。因为只有长远的大目标才需要你持续地去推动。

而大部分人想的是什么？“我有目标啊，我今天定明天能实现啊，最多一个月以后就要实现，不然我就撑不住了，我绝对是要内耗了，我心态要崩了，我可太难了。”就是这样的。那你怎么可能拥有更多的东西呢？你这么万能吗？机器猫吗？

而且必须是**目标**，而不是**设想**。为什么是这个点？其实我很多音频和视频都已经说过了，目标跟设想的区别，我在这里就不重复说了。

你可以把你长远的大目标定成什么？比如说“我要暴富”。很多人都想暴富，手机壳是暴富，手机桌面也是暴富，电脑桌面也是暴富，iPad桌面也是暴富，封面全是暴富。但是这个“暴富”，很多人是当目标来推进的，但是有的人是当设想来推进的。就是“啊我想暴富我想暴富，我想暴富，某一天搞不好，我就暴富了。”这种暴富是天方夜谭，这种概率比买彩票还低。

但是当你第二天醒来以后，发现你自己的这种设想、这种痴心妄想，看到你的手机暴富桌面，看到你的iPad暴富桌面，然后以及你面对工作中的种种困难，你比谁都想要逃避，你比谁都想要叹气，你比谁都想要失忆，恨不得一巴掌把自己拍晕。那么你最基础的这一关就已经卡住了。你想要去抵御诱惑，对于你来说是天方夜谭。小的诱惑，大的诱惑，你都抵御不了。因为你只停留在设想层。

“暴富”想要暴富没有问题，也不是好高骛远。你只要把“想要暴富”这件事情定成**目标**，就没有那么可耻，没有任何人应该要耻笑你。但是如果你把“暴富”定为**设想**，你就自己跟自己玩吧，你也别怪别人看不起你，你也别怪别人觉得你在扯淡，因为你可能就是在扯淡。

## 如何设定有效的长远目标

所以我们定的长远的目标必须是具备实操价值的，且是有意义的。

例如，你想赚100万。这个就是具备实操价值的。因为你要为赚这个100万放弃很多，以及你需要把这个100万拆碎，放到你的每个月、每一天、每一个季度、每分每秒的努力和收获上面。你需要学习很多，你需要克服很多，你需要为之付出很多，你需要吸纳很多才能输出很多，最终离这个100万越来越近。

当然，每个人不一样。例如，你已经有100万了，那100万对于你来说不是一个大的目标。所以每个人定的这个长远的目标，我们是根据每个人不同眼下的能力来定的。你现在眼下能赚50万，你可以把你的目标定成100万。你眼下能赚100万，你可以把你的目标定到1000万，没有什么多大问题。这里只是用100万来举个例子。

那么你必须要通过时间来完成这个100万的目标，不可能是短期内的目标。因为需要时间来做基础，所以你就要持续地去推进。你的精力和关注点就需要放在“持续为了赚到100万而不断努力”的这个点上。

人的精力是有限的，人的专注度也是有限的，人所有的一切都是有限的，不可能无限扩容。以及，人有了非常明确的目标的时候，应该是很投入的，是很忘我的。如果你不投入，那么问题不在这里，就是说明你的目标是假的，是没有形成。

例如，“我确定了我的目标真的是要赚100万，我非常确定这是目标。”但如果你投入不进去，你时常飘，时常想来想去，“哎呀，那我要赚100万呢，要不我再认真点？还是我今天晚上先去嗨一嗨？”经常出现这种内心波动。问题就不在这儿了，而是在于你定的这个目标可能是有水分的，你没有真的把它定成目标，所以它没有办法形成这种目标感。那么你的精力、你的集中力、你的专注力就抓不住，还是会涣散掉。

## 目标如何帮你抵御小诱惑

如果面对消费性质的小诱惑，很快你就会因为精力不够以及专注度聚焦在某一个目标上面，逐渐去忽略这个诱惑。

就像有的人他喜欢去蹦迪，但是因为有一笔大单子，如果完成了，他就可以拿到非常丰厚的回报，可能他就会去忘记要蹦迪这个事情。因为他确定他优秀地完成了以后，可以拿到一个很丰厚的回报。那么他是可以自动地去屏蔽蹦迪这个事情的，完全聚焦在“把这个大的单子搞定”的这件事情上面。小的诱惑对于他来说，可能都没有办法真正地定义为诱惑了，因为他的脑子不在这儿了。他有强大的使命感和目标感。

只有你内心很波动的时候，很多事情都会成为对于你来说是诱惑。从小到大，你们可以想一下，是不是当你的内心波动特别大的时候，就会发现很多事情他好像都是一个诱惑。

例如，你最近没什么钱，你会觉得人家吃一个人均1000块钱的午餐，都是一种诱惑。你可能单身很久了，然后你发现走在路上，你也不知道人家的感情到底是什么状况，你只是看到一个表象，大家看起来好像比较甜蜜的，你认为好像也是一种诱惑。

所以只有你内心很波动的时候，才会认为是一种诱惑。例如，你想要赚100万，但是面对困难，你又开始犹豫要不要赚这个100万，你会给自己灌负能量。“我想这样，但是我很怕我不行。”“我想这样，但是我怕我做了也没有用，也没有结果，那怎么办？”就给自己习惯性地灌负能量，觉得这么难的事情是不是可能并不会发生的，是不是我的一种痴心妄想，是不是我自己努力也没有用，但其实自己从来就没有真正的努力过。然后就开始给自己灌负能量，说：“哎，是不是我努力也没用？我努力努力的也不会有好的结果的。”诸如此类。

那么这种情绪出现的时候，这些小的诱惑就会成为很多很多击垮你的致命一步，直到把你击垮。就比如说你已经很动摇了，你已经给自己灌负能量了，你也不相信自己，甚至你根本就没有付出努力半分，然后就开始质疑努力了是不是也不会有回报。那么今天晚上正好有个人邀约你去蹦迪，邀约你一起追剧，一起玩剧本杀。那么你马上就可以自我说服，就会动摇，说“那就去吧，反正努力也不差一顿蹦迪。”“那就去吧，反正努力也不差追几集剧。”你就去了。

而这种所有的小的细节性的诱惑，就会成为最后压死你的稻草。就是会这样。是从思维认知、集中力、专注度先把你压到无可救药，然后你的行为就自然反弹回来了。但你的焦虑是不会少的，因为你的诱惑本身是没有抵消掉的。你看到别人年薪比你高的时候，你依旧会感叹：“哇塞，我怎么就没有这么高的年薪呢？哇塞，人家又升职了，我怎么就是5年了也完全原地不动呢？”你“哇塞”的东西多了去了。

但为什么又绕回来了？这个循环本来是往前绕的，现在又绕回来了。点就在这儿。

所以，抵御小的诱惑只可能在你有长远的目标之下才可能成立，不然只可能成为伪命题。因为你只有拥有了长远的一个大的目标，你才需要把你这个大的目标切切切，一直切到你的每天、每时、每分去做推进，你才有可能将这个长远的大目标去实现。而就是因为它需要你持续地去推进，所以他要持续地去拉你的集中力和你的专注度，那么这种小小的诱惑，你自然而然就没有精力了，你就自动把它屏蔽掉了。

所以当你没有办法有一个长远的目标设定的值的时候，你永远不可能抵御小的诱惑。例如，“我怎么能不犯懒呢？”“我怎么能少看点剧啊？”“我为什么一看剧就是看8小时啊，都耽误了什么什么？”“这怎么有解啊？”长远目标没有，以及有，但它不是目标，它是一个设想，那么就等同于你没有。

对一件无法对你产生持续帮助的事情上瘾，是因为你没有真正想要达到的目标，你也不是真的需要收获。你的行为是直接反映你的内心的，嘴巴是假的，行为是真的。

而消费性质的活动，它也并不是不能做，而是你是可以说到做就做，收就收。例如，“我这个星期已经高强度的加班，高强度去推进我的设想和目标，已经完成了多少多少事情。我周末准备给自己两个小时的时间去看综艺，放松一下。”没有问题，完全没有问题，就应该投入了去放松，去看综艺。但是时间一到，我该干嘛就干嘛，因为我的目标是不会轻而易举地改变的，不会过多地想要去逃避。这种节奏是完全没有问题的，这是劳逸结合。是需要给自己打开这样的一个渠道，让自己劳逸结合。

我为什么要两个小时去看综艺？是为了后面的两个小时，我放松过了，我喘气过了，我可以更集中精力地去做我应该做的事情，而不是被诱惑牵着鼻子走。

## 有回报的事与消费诱惑的本质区别

有回报的事情，往往和“上瘾”没有强关联。因为有回报的事情必须要以时间为基础，以及必须要付出努力，以及跟欲望背道而驰。这一系列和消费性质的诱惑是完全冲突的。

有回报的事，收获它通常都比较慢，但是成就感、真正的获得感又比较持久。这才叫真正有回报的事。

例如，你玩游戏，几分钟后其实你就能赢。你赢了以后，这就是回报。你现在在几分钟咔咔咔玩游戏，几分钟以后你就赢了，那马上就回报给你了一个“赢”。但你的付出是有限的，因为你只付出了几分钟，所以赢得的这个快感，它很快就消失了，它是无法产生真正的迭代的，因为下一局你有可能输。

但是真正有回报的事，你在获得快感的同时，还能提升你真正的能力，让你持续的获得更多。这种感受和回报它是非常非常扎实的一个东西。它只要非常扎实，就需要时间，不会来得那么快。

就比如说你玩游戏几分钟能赢，几分钟也能让你输，因为不扎实，所以它的结果非常的波动。有的时候你手感好一点就能赢，手感差一点就输了。你只有获得扎实的东西，你的安全感才会真正的建立起来，才能真正的落地，不容易波动和产生内耗。

我们去看为什么很多小伙伴在工作中、生活中会很容易波动，很容易内耗，是因为他根本就没有把自己真正的安全感建立起来。而这种安全感为什么建立不起来？因为不扎实。

拥有学历就一定扎实了？不是吧。在学历机制下，你拥有了学历，它扎实，可是放到社会机制下，你就不扎实了。这就是为什么我们可能提升了自己很多学历的部分，我们依旧感觉不安全，安全感依旧是缺失的。因为他换框架了，他换框了。

为什么很多小伙伴说：“哦，我这个工作很稳定，但是我没有发展了，我好焦虑，我怎么办？我是不是就这么废了，我年纪还轻呢，二十几岁就感觉自己要废了，但是工作就挺稳定的。”为什么？因为不扎实。他的工作稳定是依托于他的工作性质框架，例如体制内。这是他人给你的，不是你自己积累起来的、扎实的。你只要到了那个框架中，你就能稳定。已经完成这个稳定了，但这个东西不是你自己扎实地拥有的，是他人给你的、国家拥有的这样一个框架。那么你待到这个框架里面，你就稳定了，就是这么回事。所以他即便他有一份稳定的工作，他依旧没有安全感。点就在这儿，因为不扎实。

随时波动、随时内耗、持续波动、持续内耗的人，都是因为手上的牌他不扎实。就是因为这个。所以他的安全感很难建立起来。

包括在我们的亲密关系中也是这个样子的。为什么我在直播课的时候跟大家说要“盖楼盖楼盖楼”？因为你要把你自己的安全感首先主动地攥在自己的手中，垃圾就直接踹。你是扎实的，而不是说你明明知道它是一个垃圾，你也不知道应该怎么去做，甚至你也不敢离开。你的安全感怎么可能建立起来？你是不扎实的。

## 如何抵御重大的人生诱惑

上一节我们讨论了如何通过目标抵御日常小诱惑，本节中我们来看看如何抵御那些直接与你人生选择和走向有关的重大诱惑。

例如，谁让你去做经理，谁让你去做什么样的项目，谁让你合伙创业一起发财等等，这些是属于人生走向选择方面的诱惑。

以下是抵御重大诱惑的三个关键点：

**第一，自我定位。** 你必须非常清楚自己的强项和弱项，以及你通过时间和做事已经测试过自己了。

我并不那么赞同一个小白，不给别人打工，马上去创业。例如现在说“哦创业，打工都是没有出路的，打工永远都是天花板，你就应该一毕业马上去创业。”意思就是说你完全不要走打工这条路，你就一步就是去创业做老板。我是不那么赞同的。为什么？我认为是浪费青春，浪费时间，是在给自己做内耗，不把自己的时间当价值来看。我个人认为是这个样子的，除非你极具天分，以及你虽然没有真正的去打过工，但是你的这个环境它是一个商业环境。例如富二代，他从小就听他爸他妈在做生意，他二舅他大姑全在做生意，他从小耳濡目染，这个环境是不一样的。那你说一开始就要去小试牛刀，去创个业试试，那也可以试试，只要家里和自己能兜底，也能试试。因为他的这个底子是不一样的。有的是属于要么就是极具天赋，你的家里面的人可能都不是做生意的，但是你就属于老天赏饭吃，极具天赋。但这个毕竟是非常非常小概率的事件。所以我不是那么赞同一个小白，他从来没有给人打过工，他马上就一股脑的、一步登天地要去创业做老板去了。

核心点在什么地方？就是你无法精准定位自己，这是最致命的。例如，你就是一个做木匠的，你的核心价值是做木匠，哎，你非要去给人家剪头发。那你说你怎么可能发光发热呢？我举例。对不对？就是这样的。就人一旦无法给自己精准定位，你走的每一步路都是七拐八拐的。

为什么我一直强调说，你不要管别人，你先把你自己的定位给定清楚。你定清楚你自己，你再去定别人。你自己都定不清楚，别人被你定出来的那个位也是错的。你说你定不清楚自己，你怎么定别人？你觉得你给别人定位了，但因为你定不清楚你自己，你给别人那个定位也是定错的。那好了，完了，你不错。

自我定位可能来自于一种设想。例如，“我长大想当科学家，所以我把我自己定位成我长大要去做科学家。”那我这个定位哪来的？是来自于一种设想。这个设想是来自于哪呢？可能我看了某一个新闻，我看了某一本书，我特别羡慕科学家，我觉得他们很聪明，很智慧。所以这是来自一种设想。而设想是会产生极大的偏差的。设想如何不产生极大的偏差？就是要在实践中得到结果。就是这样的。

那如果说你看不明白你自己，你怎么可能分辨得出这个是非？对吧？“我设想我想当科学家。”结果呢？你不适合当科学家。那个“不适合当科学家”的这个结论是怎么来的？那你肯定是奔着这条路去走的喽。对不对？但是人有多少时间可以去浪费？没有多少时间可以去浪费。尤其是我们，没有那么多那么多的时间可以拱手让给你说：“去吧，你就去浪费时间吧，没事。”这是极少数的事件。

那你当你已经凭着你的这个不精准的个人定位，溜了一圈又一圈，发现不是这么回事的时候，你想要切换赛道，这也是要付出代价的。对不对？

人贵有自知之明。例如，你不是统筹的料，别人让你去做统筹息息相关的工作，即便说得很让你心动，你觉得是一个极大的诱惑，那你也要有这个意识：自己承载不住的东西，它就是一个作废的东西。

为什么我说小白就是要去先打工？因为你可以快速地向你的定位靠拢。但创业不是，创业是什么？是凭着自己的设想、自己的热情。就是这个样子的。
