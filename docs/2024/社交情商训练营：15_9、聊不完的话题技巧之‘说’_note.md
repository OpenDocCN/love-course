# 社交情商训练营：15_9、聊不完的话题技巧之‘说’

## 概述
在本节课中，我们将学习如何通过“说”的技巧来提升社交情商，特别是在与异性交流时如何有效地展示自己的价值。

## 核心概念
**公式**：感受 + 价值 = 真诚分享

## 教程内容

### 1. 技术介绍
**说**的技巧，即通过表达感受和展示价值来与对方建立联系。

### 2. 技术运用方向
- **通过关键词延伸**：利用聊天中的关键词，结合个人感受，展开话题。
- **直接表达**：直接说出自己的想法，并运用“下钩子”技术吸引对方好奇心。

### 3. 关键词延伸示例
以下是使用关键词延伸的示例：

**示例**：在聊天中提到“旅游”，可以分享自己的旅游经历和感受。

- **步骤**：
  1. 聊天中出现“旅游”关键词。
  2. 回忆并整理与旅游相关的经历和感受。
  3. 用一句话表达出来，例如：“说到旅游，我就想起了我以前去云南大理，真是郁闷死我了，就呆了两天就把我晒得像个黑人一样。”

### 4. 直接表达示例
以下是直接表达并运用“下钩子”技术的示例：

**示例**：展示自己做饭的能力。

- **步骤**：
  1. 聊天中提及与做饭相关的话题。
  2. 使用“下钩子”技术，例如：“哎呀，排队好无聊啊。”
  3. 观察对方反应，继续引导话题：“你有没有想过，排队的时候可以做一些有趣的事情呢？”
  4. 展示价值：“比如，我可以给你展示一下我最近学会的做饭技巧。”

### 5. 图片或视频辅助
在展示价值时，配以图片或小视频，可以增强吸引力。

### 6. 总结
本节课中，我们学习了如何通过“说”的技巧来提升社交情商，特别是在与异性交流时如何有效地展示自己的价值。记住，真诚分享和适当展示是关键。
