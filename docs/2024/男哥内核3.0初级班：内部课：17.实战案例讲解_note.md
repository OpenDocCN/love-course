## 实战案例一：用户登录系统

### 案例背景
假设我们需要开发一个用户登录系统，用户可以通过用户名和密码进行登录。

### 核心概念
```python
# 用户登录系统核心代码
def login(username, password):
    # 检查用户名和密码是否正确
    if check_credentials(username, password):
        return "登录成功"
    else:
        return "登录失败"
```

### 步骤
1. **定义登录函数**：创建一个名为`login`的函数，接收用户名和密码作为参数。
2. **检查凭证**：调用`check_credentials`函数来验证用户名和密码是否正确。
3. **返回结果**：根据验证结果返回相应的信息。
