# 男哥内核3.0初级班：内部课：2.飞龙在天系统延伸拓展二

## 概述
在本节课中，我们将深入探讨沟通心态和价值的传递，帮助学员改善心态，提升沟通技巧。

## 一、沟通心态
### 1. 心态问题分析
- **自信不足**：认为自己是被动的，而非主动的。
- **不够坦然**：害怕说错话，过于在意对方的反应。
- **不够自然**：过于刻意，缺乏真实感。

### 2. 解决方法
- **接受现实**：认清自己的优势和不足。
- **规划生活**：丰富自己的生活，提升自我价值。
- **正向反馈**：从他人那里获得认可和鼓励。

### 3. 乐天学习法
- **形象改造**：提升个人形象，增强自信。
- **生活规划**：明确自己的人生目标，规划未来。

## 二、价值的传递
### 1. 价值传递误区
- **无目的无需求的提供价值**：为了对方而改变自己，显得卑微。
- **有需求的提供价值**：期望对方给予回报，容易陷入被动。

### 2. 正确的价值传递
- **无需求的传递价值**：从自身出发，提升自我价值。
- **关注自身**：不为对方改变，保持自我。

### 3. 销售思维
- **明确目标用户**：了解对方的需求，提供合适的价值。
- **拓展用户群体**：不要将所有精力集中在一个人身上。
- **保持框架**：不要过度索取，保持自我价值。

## 三、总结
本节课中，我们学习了沟通心态和价值的传递，帮助学员改善心态，提升沟通技巧。通过接受现实、规划生活、正向反馈等方法，学员可以更好地与他人沟通，传递价值。同时，我们要学会无需求的传递价值，保持自我，避免陷入被动。希望学员们能够将所学知识应用到实际生活中，提升自己的沟通能力和价值。
