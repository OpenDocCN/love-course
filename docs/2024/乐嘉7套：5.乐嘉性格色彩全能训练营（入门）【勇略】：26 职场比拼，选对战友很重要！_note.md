# 乐嘉性格色彩课程5：职场比拼，选对战友很重要！👬

在本节课中，我们将通过经典小说《大唐双龙传》的故事，学习如何根据性格色彩理论，在职场中选择和搭配你的“战友”。理解搭档的性格类型，并运用“钻石法则”与之相处，是团队成功的关键。

## 概述：两种搭档模式

上一节我们介绍了性格色彩的基本概念，本节中我们来看看它在团队协作中的应用。通过分析《大唐双龙传》中两组核心搭档，我们可以总结出两种主要的搭档模式：**互补型**与**相似型**。

寇仲与徐子陵是典型的**互补型搭档**。寇仲具有**红+黄**性格，积极热情、目标感强、急于求成。徐子陵则具有**蓝+绿**性格，内敛谨慎、随和包容、淡泊名利。徐子陵的冷静与自律，多次帮助冲动的寇仲化解危机，例如在他因练“邪帝舍利”而走火入魔时，徐子陵及时找到了“和氏璧”来解救他。

与之形成对比的是李世民与李秀宁，他们属于**相似型搭档**。李世民是典型的**大黄色**性格，理性、以大局为重、不受情绪左右。李秀宁在小说中也更偏向**黄色**性格，为了家族事业可以将儿女情长放在一边。他们目标高度一致，行动默契，最终成就大业。

这两种模式并无绝对优劣。互补型搭档需要**取长补短**，而相似型搭档则需要**价值观高度一致**。当价值观一致时，无论性格相似还是互补，都需要运用“钻石法则”来优化合作。

**钻石法则的核心是**：用适合对方性格的方式对待他，而非你喜欢的方式。

## 如何运用钻石法则与不同性格搭档合作

以下是针对四种不同性格色彩的搭档，运用钻石法则的具体方法。

### 1. 与红色性格搭档相处 🎯
红色性格需要情感互动与认可。徐子陵能说服走火入魔的寇仲，正是基于他们深厚的情感基础和信任。同样，寇仲的恋人宋玉致（红色性格）为他付出从不求回报，只需寇仲一句赞美或一个小礼物就能让她非常开心。

与红色搭档相处时：
*   **多进行情感交流**，建立信任。
*   **及时给予真诚的赞美和认可**，尤其是在他们取得成就时。
*   **避免忽视他们的贡献或泼冷水**，这会破坏关系。

### 2. 与蓝色性格搭档相处 📘
蓝色性格注重规则、原则和谨慎。剧中的师妃暄是典型蓝色性格，她严守对师傅的承诺（戴面纱），即便在救人这种“不可抗力”下破例，事后也强烈自责。徐子陵（蓝+绿）因理解并尊重她的原则，最终与她走到一起。

与蓝色搭档相处时：
*   **尊重他们的规则和原则**，不强迫其做不愿之事。
*   **在做重大决策时，务必与他们共同全面评估风险**。若独自决定，会引发他们的不安全感。
*   **一起坐下来，理性分析所有负面可能性**，达成一致后再行动。

### 3. 与黄色性格搭档相处 👑
黄色性格享受决策和控制感。李世民是典型代表，他为了大局可以牺牲个人情感，甚至对恩人寇仲、徐子陵下手。李秀宁作为他的搭档，始终扮演着优秀的辅助者角色。

与黄色搭档相处时：
*   **让他们主导决策**，你从旁协助分析，帮助制定计划。
*   **目标一致时，默契配合，无需过多交流**。
*   **若想说服或纠正他们，避免正面冲突和否定**。应遵循核心原则：**平静、坚定、就事论事地指出其做法可能对结果造成的损害，并亮明你的底线**。理性客观的讨论能赢得黄色的尊重。

### 4. 与绿色性格搭档相处 🌿
绿色性格随和、缺乏主动性，需要被带动。徐子陵性格中有显著的绿色成分，对争霸天下很淡然，但在好兄弟寇仲的主动引领下，他愿意配合并一起成就事业。

与绿色搭档相处时：
*   **你需要主动带动和引领**，切忌双方互相等待，导致一事无成。
*   **避免一次性下达宏大目标**，应分解任务，逐步推动。
*   **参考寇仲带动徐子陵的方法**：主动规划，邀请参与，共同前行。

## 总结

本节课中，我们一起学习了如何根据性格色彩选择与搭配职场战友。我们分析了**互补型**（如寇仲与徐子陵）和**相似型**（如李世民与李秀宁）两种搭档模式。关键在于，无论哪种模式，都需要运用 **“钻石法则”**——用适合对方性格的方式与之合作。

记住以下要点：
*   对**红色**，重情感、多赞美。
*   对**蓝色**，守规则、共评估。
*   对**黄色**，让决策、理性谏。
*   对**绿色**，要主动、善带动。

理解并应用这些原则，将帮助你在职场中找到对的战友，并与之形成强大的合力，共同达成目标。
