## 搭讪概述

在本节课中，我们将学习如何有效地搭讪女生，特别是高分女生。搭讪是一种主动与陌生人攀谈的行为，目的是结识对方。它是一种包容开放，想要结交朋友的生活态度。

## 搭讪认知

以下是关于搭讪的几个重要认知：

1. **搭讪不是可耻的**：搭讪是主动与陌生人攀谈，借此结识对方的一种行为。搭讪的目的可能是交友、调情、开发顾客或合作伙伴。
2. **女生喜欢被搭讪**：女生是活在attention的世界，时时刻刻都希望获得外在的关注，被搭讪就代表着被关注被欣赏。
3. **搭讪很简单**：只要你能正常的友善的轻松的站在女生面前，并正常的友善的轻松的说几句话，你已经战胜了99.9%的男生。
4. **搭讪的好处**：搭讪是效率很高的交友方式，能让你对自己的原始魅力有更清醒的认知，提升自己的交际能力，甚至让你接触到非常有价值的人脉。

## 搭讪步骤

搭讪分为四步：启、成、转、合。

1. **启**：搭讪的开场白，也就是你接近女生时说的第一句话。搭讪只有三种开场方式：直接搭讪、间接搭讪和趣味搭讪。
2. **成**：搭讪的对话环节，也是你给女生建立信任的阶段。
3. **转**：搭讪的转折，也就是你告诉女生，你无法继续对话。
4. **合**：最后添加好友的行动，结束整个搭讪。
