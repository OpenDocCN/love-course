# 米娅《女人一生幸福的必修课》：02第1节：男人的“猎人属性”秒懂思维差异

在本节课中，我们将要学习一个核心概念：男性的“猎人属性”。我们将从进化心理学的角度，剖析男性思维模式与行为习惯的根源，帮助你理解两性思维的根本差异，从而避免感情中的许多误解与冲突。

我们经过漫长的观察与经验积累，必须承认一个基本事实：男人和女人天生是两种不同的物种。这不仅体现在生理结构上，更体现在思维方式上。然而，正是这种不同，构成了异性相吸的基础。

我们首先要承认这种不同，然后顺应这种不同，最后驾驭这种不同。很多女性在感情中感到被动，或遇到问题不知如何处理，常常陷入僵局，甚至让事情发展到不可挽回的地步，感到委屈。其实这大可不必，问题往往源于不了解男性的思维方式。

如果你用女性思维去衡量和要求男性，必然会发现他的表现与你的心理预期存在巨大落差。本节课的内容非常关键，它是你进入男性大脑思维的第一步。所谓“知己知彼，百战不殆”，无论你是要改善夫妻关系，还是要吸引喜欢的男性，这一步都至关重要。

以下是生活中一些常见的问题，你可以对照思考：

*   **问题一：他是否特别不喜欢陪你逛街？** 当你购物并希望他提供意见时，他常常表现敷衍或不耐烦。你们经常高兴出门，却憋着一肚子气回家，最终引发争吵，升级为“他不爱你”的争论。
*   **问题二：他是否一忙工作就冷落你？** 你跟他讲话，他好像听不见，或左耳进右耳出。你因此感到困惑和生气，越想唠叨他，结果又引发冲突，他觉得你烦，你觉得他不在乎。
*   **问题三：他是否显得固执，听不进劝？** 你觉得自己的建议很有道理，但他就是不采纳，甚至明知你的建议更好也不照做。争吵后，你常无奈地将此归结为“三观不合”。
*   **问题四：他回家后是否不爱说话？** 他不是玩手机就是打游戏。你因此发牢骚，甚至怀疑他有外遇或不爱你了。他觉得委屈，认为只是想放松一下；你也觉得委屈。两人逐渐从无话不谈变得相视无言。
*   **问题五：他是否总不在意细节，不善于观察？** 即使你表现得很明显，他也没反应，无法在你需要时及时提供情绪价值。你总觉得他不懂你。
*   **问题六：你是否总觉得他听不懂你说话？** 你常常感到无法与他沟通，不知道他在想什么。

女性的思维是**网状发散式**的，容易将许多事情最终归结为“男人爱不爱你”。你可能会认为，他做不到这些事就是不爱你，爱你的男人一定会注意细节、懂你、陪你。

其实，男性的许多本能反应与“爱不爱”并无直接关系。真正爱你的男人，也不一定能让你完全满意；而看似面面俱到的男人，也未必是真爱，可能另有所图。能够提供超高浓度情绪价值、让你如沐春风的男人，几乎都经过训练。未经“驯化”的直男和老实人，总会在不自觉中惹你生气。

这里我们不讨论对错，而是先深度剖析男性行为背后的原因。了解真相后，或许能避免许多无谓的争吵。

我们这节课的标题是“男人的猎人属性”。顾名思义，在历史上，男性与猎人角色密不可分。尽管男性看起来很复杂，但如果你足够了解他们，就会发现他们是一个非常简单的物种，有时简单到不可思议。一个男性的主要特点，几乎都能在一个猎人身上充分体现。你了解了猎人，就相当于了解了男性。

这要追溯到几百万年前。那是人类历史上非常漫长的一段时期，男性和女性在危险的世界中艰难生存。男性充当家庭的**供养者**和**保护者**，女性则负责抚养孩子、采摘果实、操持家务。男性每日的主要工作是外出为家庭寻找食物。

根据生物进化论的观点，为了适应狩猎工作，男性在狩猎中自觉养成了种种习惯。在漫长的时间里，这些习惯逐渐变成自发的本能，存在于男性基因之中，对其思维方式和行为模式产生了深远影响。

为了生存，男性需要获取更多猎物，因此需要更快的速度、更大的力量等。在长期的狩猎生涯中，男性大脑的特定区域自然发生了变化，以使他们能更成功地完成任务。这个特定区域叫做**视觉空间中枢**，它主要负责测量**速度**、**角度**、**距离**和完成**空间配合**。这些都是大脑的“狩猎区域”。

因此，我们会发现许多男性的空间感和方向感比女性强得多，而女性的空间中枢活动则不如男性发达。

这里我们可以解释前面的第一个问题：为什么男性通常厌烦陪女性逛街？如果需要一件衣服，他们会**直奔目的地 -> 快速挑选 -> 结账走人**。他们真的是为了“买衣服”而“买衣服”。但女性不同，女性更享受“挑选”的过程，可能逛一天却一件不买。如果女性了解男性的狩猎本性，就不会对此感到奇怪。

作为猎人，他的任务简单明确：必须快速决策并快速拿下猎物。犹豫会导致失去目标或最佳捕猎时机。因此，男性在购物这件事上大多简单直接，这是他们大脑基因决定的，使他们无法对在此事上浪费时间产生兴趣。如果非要将其上升到“爱不爱”的高度，确实有些冤枉。

猎人必须独立解决打猎过程中遇到的各种问题，例如去哪儿打猎、用什么工具等。当猎物出现时，他通常没有机会与人商量，也不需要任何人指示。他所做的就是在最短时间内，利用自己的经验与直觉快速得到结果，然后把猎物扛回家交给女性。在整个过程中，猎人很少需要别人帮助。

这就可以解释为什么男性通常不喜欢别人对自己指指点点，而更喜欢独立解决问题。

本节课中，我们一起学习了男性“猎人属性”的起源及其对现代男性思维与行为的影响。我们了解到，男性的直接、目标导向、注重独立解决问题等特质，根植于漫长的进化历史。理解这一点，是读懂男性思维、改善两性沟通的第一步。在下一节中，我们将继续探讨“猎人属性”如何具体影响亲密关系中的其他互动模式。
