# 课程05：成真女生的秘密：卫生棉条长什么样？🧐

在本节课中，我们将学习卫生棉条的基本物理构造、主要材质、尺寸规格以及其核心工作原理。了解这些基础知识是正确和安全使用卫生棉条的第一步。

---

## 构造与材质

卫生棉条是一种棉质的圆柱体。它的形状大致呈圆柱形。

其主要材质是棉、人造纤维，或者是这两种材质的混合。不同产品的混合百分比有所不同。

## 尺寸与细节

卫生棉条有不同的尺寸规格。有的直径是1厘米，有的可达1.9厘米，接近2厘米。

在它的尾端，附有一条小棉线。这条小棉线是一个拉绳。使用完毕后，可以通过拉动这条棉线将其取出。

## 使用方法与原理

上一节我们介绍了卫生棉条的物理构造，本节中我们来看看它的使用方法和核心工作原理。

卫生棉条需要被置入阴道内使用。因此，它不需要配合护垫使用。

其核心功能是吸收经血。卫生棉条通过其材质（棉或人造纤维）的吸水/吸血特性，将经血吸收并储存在内部。
