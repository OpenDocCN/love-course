# 课程05：卫生棉条长什么样？🧐

在本节课中，我们将要学习卫生棉条的基本物理构造、主要材质、尺寸规格以及其核心工作原理。了解这些基础知识是安全、正确使用卫生棉条的第一步。

## 构造与材质 🧬

卫生棉条是一种棉质的圆柱体。它的主要材质是**棉**、**人造纤维**，或这两种材质的**混合材质**。

## 尺寸规格 📏

卫生棉条有不同的尺寸。以下是常见的直径范围：
*   直径约为1厘米。
*   直径最大可达约1.9厘米，接近2厘米。

## 核心组件与工作原理 ⚙️

上一节我们介绍了卫生棉条的外形和尺寸，本节中我们来看看它的关键部件和如何工作。

卫生棉条的尾端有一条**小棉线**，即一个**拉绳**。使用完毕后，通过拉动此绳即可将其取出。

它需要被置入阴道内。因其直接置于体内吸收经血，所以无需额外使用护垫。它的核心功能是**吸水（血）**，将经血吸收储存。
