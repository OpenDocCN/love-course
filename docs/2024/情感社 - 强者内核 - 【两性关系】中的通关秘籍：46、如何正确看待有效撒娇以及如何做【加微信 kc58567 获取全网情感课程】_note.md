## 正确看待撒娇

### 定位撒娇

**公式**：撒娇 = 技能 + 内心深处的爱

撒娇不仅仅是一种技巧，更是一种内心深处的爱。它不是套路，而是通过调频技能，在不同的人、不同的场景下，做出相应的行为反馈。

### 调频技能

**公式**：调频 = 对不同的人、不同的场景做出相应的行为反馈

调频技能是指在不同的人、不同的场景下，能够自然地做出相应的行为反馈。这种技能可以通过认知和实践来培养。

## 如何做

### 装傻

**方法**：在对方犯错时，不揭穿，不质问，而是以温柔的方式表达自己的感受。

### 行为姿态

**方法**：在轻松的氛围中，运用猫的行为方式，通过肢体语言和语言表达撒娇。

### 女性化肢体接触

**方法**：运用女性特有的肢体接触，如轻拍、扭动等，传递温柔和爱意。

### 语言结构

**方法**：

1. 擅用“人家”这个词，表达乞求的情绪。
2. 使用拖尾音，传递幼态的情绪。
3. 使用叠字，增强亲密感。
4. 善用语气词，如“哼”等，表达撒娇的情绪。

### 注意事项

1. 不要过度使用撒娇，以免产生腻味感。
2. 见好就收，不要没完没了。
3. 公众场合不要撒娇。
4. 对方心情不好时，先闭嘴，观察。
