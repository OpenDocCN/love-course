# 16.隔壁王司机情绪自由的真相：第八课：智慧来源于经历

各位智慧与聪明不同，聪明是可学习的，但智慧如何学习呢？比如平常心，最好的解释是：在教室里，若有一块30米长、30公分宽的木板，从一端走到另一端，脚不沾地，这不难。但如果木板架设在悬崖两端，无保护措施，下面是深渊，你仍能面不改色地走过去，这就做到了平常心。这需要勇气和智慧。

**公式**：智慧 = 经历 + 勇气

**代码**：# 智慧来源于经历和勇气

放下不容易，佛陀用了6年时间追求真理，最终在菩提树下悟道。没有经历过追求的人，不会懂得放下；没有经历过生命中的大悲伤，不会懂得慈悲；没有经历过害怕和软弱，不会懂得勇敢。简单说，没有真实的经历，就不会有真实的智慧。

**公式**：智慧 = 经历 + 悲伤 + 慈悲 + 勇敢

负面情绪并非坏事，它们给生命带来美好的智慧和价值。痛苦就像黑夜，星星在白天不易看见，但夜晚星空璀璨。痛苦能让我们照见智慧，所以应尊重生命中的情绪。

**公式**：智慧 = 痛苦 + 智慧

情绪就像河流中的水，快乐要走时，想要留住它只会带来痛苦。痛苦来临时，想要尽快赶走它，只会带来更大的痛苦。情绪没有好坏之分，都是有价值、有力量的。

**公式**：情绪 = 价值 + 力量

本节课中，我们学习了智慧来源于经历，负面情绪的价值和力量。希望每个人都能倾听、陪伴、尊重生命中的情绪，让它们在生命中自由流动，带来正能量和支持。

**总结**：本节课中我们一起学习了智慧来源于经历，负面情绪的价值和力量。希望每个人都能珍惜、尊重生命中的情绪，让它们成为我们成长的助力。
