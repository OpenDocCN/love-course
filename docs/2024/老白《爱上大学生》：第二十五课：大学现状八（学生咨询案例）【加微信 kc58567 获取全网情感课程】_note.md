## 学生咨询案例

### 自卑感与社交
有时候，我也会感到自卑，就像抖音上的美女一样，即使我打招呼聊天，但总觉得自己不如那些擅长社交的人。

**公式**： 自卑感 = 比较自己与他人 + 自我评价低

### 网络与现实
那些抖音上的美女，看起来精致，但实际上可能很贫穷，甚至可能住在1000元的出租屋里。

**公式**： 网络形象 ≠ 现实生活

### 主播与网红
很多主播和网红看起来光鲜亮丽，但实际上很多都是贫穷的，没有赚到多少钱，生活开销很大。

**代码**：
```python
# 穷主播
class PoorInfluencer:
    def __init__(self, followers, income, expenses):
        self.followers = followers
        self.income = income
        self.expenses = expenses

    def is_poor(self):
        return self.income < self.expenses

# 示例
influencer = PoorInfluencer(followers=100000, income=100000, expenses=150000)
print(influencer.is_poor())  # 输出：True
```
