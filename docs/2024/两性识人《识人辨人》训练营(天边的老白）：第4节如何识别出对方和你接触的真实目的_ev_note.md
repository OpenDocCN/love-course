# 两性识人《识人辨人》训练营(天边的老白）：第4节如何识别出对方和你接触的真实目的

大家好，我是老白。上节课我们讲解了如何通过一个人当下的环境判断他和你在一起的目的。本节课，我们将深入探讨如何识别对方和你接触的真实目的。

## 1. 人性的多面性

人性就像水，它可以是滋润你的甘泉，也可以是冲击你的洪水。人性本身没有绝对的善恶，关键在于当前的状态和展现形式。

## 2. 立场与目的

在与人交往中，了解对方的立场和目的是至关重要的。立场不同，目的不同，会导致行为上的冲突和误解。

### 2.1 案例分析

以下是一个案例，让我们更直观地了解立场和目的的重要性：

**案例**：一位女性在相亲时遇到了两个男生，A男和B男。A男明确表示想要结婚，而B男则不确定自己的立场。

**分析**：

*   **A男**：立场明确，目的明确，追求结婚。
*   **B男**：立场模糊，目的不明，可能只是想恋爱。

在这种情况下，女性应该选择与自身立场和目的相同的人，即A男。

### 2.2 筛选与判断

为了找到与自身立场和目的相同的人，我们需要进行以下筛选和判断：

*   **了解对方的立场和目的**：通过观察、沟通等方式，了解对方在当前情境下的立场和目的。
*   **判断对方是否真诚**：判断对方是否真诚地追求与自身相同的立场和目的。
*   **排除立场和目的不同的人**：将那些立场和目的与自身不同的人排除在外。

## 3. 立场不同导致的冲突

当立场不同时，对方的行为可能会在你看来是伤害你的，但在他看来却是正确的。这时，不要试图改变对方，而是接受这种差异。

## 4. 应对方法

*   **了解对方**：通过观察、沟通等方式，了解对方的立场和目的。
*   **调整自己的期望**：根据对方的立场和目的，调整自己的期望和态度。
*   **保持理性**：在处理与立场不同的人的关系时，保持理性，避免情绪化。

## 5. 总结

本节课我们学习了如何识别对方和你接触的真实目的。了解对方的立场和目的是至关重要的，这有助于我们找到与自身立场和目的相同的人，避免不必要的冲突和误解。
