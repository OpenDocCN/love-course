# 海蓝老师情绪梳理个案课：044-06 转变 🌟

## 概述

在本节课中，我们将学习如何通过情绪梳理个案课来转变自己的心态，更好地面对生活中的挑战。

## 核心概念

**公式**： 情绪梳理 = 自我认知 + 行动计划 + 持续陪伴

## 教程内容

### 1. 自我认知

**以下是自我认知的步骤**：

- **钉住最坏的结果**： 将自己最坏的结果清晰地钉在心中，以此激励自己更加珍惜当下。
- **看到更好的自己**： 通过反思和努力，看到自己更好的样子，懂得珍惜。
- **面对恐惧**： 当面对恐惧时，思考自己可以做什么，想要做什么，并努力去实现。

### 2. 行动计划

**以下是行动计划的步骤**：

- **安排妈妈的旅游**： 先安排妈妈的旅游，让她享受生活。
- **积极行动**： 积极行动起来，成家立业。
- **制定详细清单**： 与朋友分享自己的想法，并制定详细的行动计划。

### 3. 持续陪伴

**以下是持续陪伴的步骤**：

- **陪伴小清晨**： 每天陪伴小清晨，让他感受到关爱。
- **倾听小清晨**： 倾听小清晨的想法，了解他的需求。
- **勇敢面对**： 用自己的勇敢、智慧、爱和判断去陪伴小清晨，帮助他度过难关。

## 总结

本节课中，我们一起学习了如何通过情绪梳理个案课来转变自己的心态，更好地面对生活中的挑战。希望同学们能够将所学知识应用到实际生活中，不断提升自己。😊
