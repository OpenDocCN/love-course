# 瑞恩年会员PRO 瑞恩高级会员：课程66(抽离：以退为进的聊天是怎么样的)

## 概述
在本节课中，我们将学习如何通过“抽离”技巧，在聊天中以退为进，激发对方的情感和吸引力。

## 技巧解析
以下是“抽离”技巧的具体应用步骤：

### 1. 以退为进
- **场景**：女生表示要来机场接你。
- **行为**：告知女生你可以不用来接，因为时间对不上，让她先忙好自己的事情。
- **目的**：通过不让她来接，激发她的好奇心和渴望。

### 2. 抽离
- **场景**：女生表示想要的男人没了。
- **行为**：表示缘分未到，你打算回香港后继续出差。
- **目的**：通过抽离，让对方感受到你的独立和忙碌，从而增加吸引力。

### 3. 情感释放
- **场景**：女生表示内心焦急。
- **行为**：表示幸好问了一下，她真接你。
- **目的**：让对方感受到你的关心和在乎，从而释放出她的情感。

### 4. 激活内心
- **场景**：女生表示落地后空落落的。
- **行为**：表示你落地后她还空落落的，表示出一点失望。
- **目的**：挑动她的情绪，激活她内心的孩子和情感。

## 总结
本节课中，我们学习了如何通过“抽离”技巧，在聊天中以退为进，激发对方的情感和吸引力。通过以上步骤，我们可以让对方更加投入情感，从而建立更深的联系。
