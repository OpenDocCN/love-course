# 驾驭关系的能力：情绪价值：12如何提供情绪价值（一）

## 概述

在本节课中，我们将深入探讨如何通过观察对方的防御机制，找到其脆弱部分，并给予对方情绪价值。我们将重点讲解提供情绪价值的三个步骤，并通过案例分析帮助大家更好地理解和掌握这一能力。

## 提供情绪价值的三个步骤

**公式**：观察防御机制 → 找到脆弱部分 → 给予情绪价值

1. **观察防御机制**：首先，我们需要观察对方的防御机制，即对方在面对问题时表现出的防御行为或态度。
2. **找到脆弱部分**：在观察防御机制的基础上，我们需要找到对方的脆弱部分，即对方内心深处的敏感或易受伤害的地方。
3. **给予情绪价值**：最后，我们需要用共情的表述方式或行为方式，给予对方情绪价值，满足对方内心的需求。

## 真我部分的提供情绪价值

除了防御机制，我们还需要关注对方的真我部分，即对方的优点、闪光点等。对于真我部分的提供情绪价值，步骤如下：

1. **觉察真我部分**：观察对方的优点、闪光点等。
2. **给予认可、崇拜、鼓励、夸奖等**：通过认可、崇拜、鼓励、夸奖等方式，强化对方的优点和闪光点。

## 案例分析

以下是一个情感关系中的案例分析，帮助我们更好地理解如何运用提供情绪价值的方法。

**案例**：妻子发现丈夫出轨，打电话质问丈夫为何不接电话，引发争吵。

**分析**：

1. **观察防御机制**：妻子在打电话时，丈夫表现出愤怒、指责等防御行为。
2. **找到脆弱部分**：丈夫的脆弱部分可能是不被妻子理解、害怕失去妻子等。
3. **给予情绪价值**：妻子可以通过共情的表述方式，表达对丈夫的理解和关心，帮助丈夫缓解情绪。
