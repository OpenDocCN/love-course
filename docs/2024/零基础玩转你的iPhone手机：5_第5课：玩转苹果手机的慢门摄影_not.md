# 零基础玩转你的iPhone手机：5_第5课：玩转苹果手机的慢门摄影_not

在本节课中，我们将学习如何使用苹果手机进行慢门摄影。慢门摄影能帮助我们拍摄出具有创意和视觉冲击力的画面，例如拉丝的流水、虚化的车流以及梦幻的光绘效果。我们将从慢门的基本原理讲起，并分别介绍使用手机自带相机和专业软件进行拍摄的具体方法。

## 📖 慢门摄影原理

上一节我们介绍了慢门摄影的创意效果，本节中我们来看看其背后的原理。

慢门摄影的核心原理是**延长单张照片的曝光时间**。公式上可以理解为：**曝光量 = 光圈 × 快门速度 × ISO**。通过降低快门速度（即延长曝光时间），画面中移动的景物（如行人、车辆、水流）就会因持续曝光而记录下运动轨迹，从而形成拉丝或虚化的效果。

## 📸 使用苹果自带相机拍摄慢门

了解了原理后，我们来看看如何利用iPhone自带的“相机”应用实现慢门效果。这个功能隐藏在“实况照片”模式中。

以下是具体操作步骤：
1.  打开“相机”应用。
2.  确保顶部工具栏中的“实况照片”按钮（一个同心圆图标）已开启（显示为黄色）。
3.  手持手机保持稳定，对准拍摄场景（如流水、车流、走动的人群）进行拍摄。
4.  拍摄完成后，进入“照片”应用，找到刚才拍摄的实况照片。
5.  向上滑动这张照片，在效果选项中选择“长曝光”。

**拍摄要点**：拍摄时手机务必拿稳，并确保画面中希望保持清晰的主体（如静止的模特、石头）完全不动，这样才能与移动的景物形成虚实对比。

### 案例展示与技巧

以下是几个使用此方法拍摄的典型场景：

*   **走动的人群**：使用iPhone XS Max的实况模式拍摄。画面中站立不动的人物与周围走动的人群形成虚实对比，增强了照片的动感和故事性。
*   **拉丝的流水**：打开实况模式，拍摄浪花拍打岩石的场景。选择长曝光后，水流会变得如丝般顺滑。拍摄时手机越稳，静态的岩石等物体就越清晰。
*   **虚化的车流**：在街头使用实况模式拍摄。通过适当变焦拉近构图，并使用长曝光效果，地面的车流会虚化成绚丽的光轨。

## 🔧 使用专业软件 ProCam 拍摄慢门

虽然自带相机方便快捷，但使用专业软件能提供更长的曝光时间和更丰富的控制选项。我们将使用之前介绍过的ProCam软件进行拍摄。**请注意，使用软件拍摄慢门必须使用三脚架来稳定手机。**

### 1. 拍摄拉丝流水

上一节我们看了自带相机的效果，本节中我们来看看如何用ProCam拍出更细腻的流水。

在ProCam中，我们使用 **“低速快门”** 模式下的 **“动态模糊”** 功能。建议曝光时间设置为**10秒到30秒**，并选择光线柔和的清晨、傍晚或阴天进行拍摄，以获得更柔和的色彩。

以下是具体操作流程：
1.  将手机固定在三脚架上。
2.  打开ProCam，切换到“低速快门”模式。
3.  选择“动态模糊”子模式。
4.  将曝光时间设置为15秒。
5.  按下快门，等待曝光完成即可得到丝滑的流水照片。

**构图变化**：你可以尝试竖拍构图来表现瀑布的高大，或使用长焦镜头（如2倍或2.5倍变焦）拉近画面，捕捉水流更细微的细节和背景的虚实对比。

对于平静的湖面或海面，同样可以使用“动态模糊”模式，并将曝光时间延长至**30秒**左右，这样可以将水面的波纹雾化，拍出平整、静谧、富有意境的画面。

### 2. 拍摄夜景车轨

接下来，我们学习如何拍摄夜晚车流的轨迹。这需要使用ProCam的 **“车轨”** 模式。

拍摄车轨的关键是寻找**视野开阔的高处机位**（如楼顶天台），避免在过低的天桥或路边拍摄，以免车灯过强导致画面过曝。曝光时间建议控制在**5秒到15秒**之间（如4秒或8秒），时间过长可能导致车轨过于密集和过曝。

以下是具体操作步骤：
1.  在开阔的高处架设好三脚架。
2.  打开ProCam，进入“低速快门”模式下的“车轨”子模式。
3.  根据车流速度和密度，将曝光时间设置为8秒。
4.  按下快门，即可捕捉车辆行驶留下的光轨。

### 3. 拍摄创意光绘

最后，我们来探索充满创意的光绘摄影。光绘是在黑暗环境中，用光源在空气中“绘画”并被相机记录下来的艺术。

拍摄光绘需要在一个**全黑的房间**里进行，使用一盏有颜色的灯（或手机屏幕显示彩色图片）作为“画笔”。我们使用ProCam的 **“光轨”** 模式，曝光时间通常设为**10秒到30秒**。

以下是创作光绘的步骤：
1.  在黑暗环境中架好三脚架，布置好静态背景（如一个瓶子）。
2.  打开ProCam，选择“低速快门”下的“光轨”模式。
3.  将曝光时间设置为15秒或30秒。
4.  对背景物体（如瓶子）对焦。
5.  按下快门，同时用彩色光源在镜头前、物体周围描绘你想要的图案（如圆形光环）。
6.  曝光结束后，即可得到一张光绘作品。

**技巧提示**：你可以自由控制“画笔”的运动轨迹，创作出各种图案。使用长焦镜头（如2.5倍）并让手机离物体稍远一些，可以避免广角镜头带来的物体变形。

2.  使用专业软件 **ProCam**，通过 **“动态模糊”**、**“车轨”** 和 **“光轨”** 模式，配合三脚架进行长时间曝光，创作出更细腻的流水拉丝、夜景车轨和创意光绘作品。

掌握慢门摄影的关键在于理解其原理，并在合适的场景（如动态的水流、车流、黑暗环境）中灵活运用稳定设备和曝光控制技巧。希望大家课后多加练习，将慢门摄影融入你的创作中。
