# 富山山女性成长课程：34_让他迷之上头3

## 概述
在本节课中，我们将学习如何通过高情商的沟通技巧，让对方对你产生浓厚的兴趣。

## 第一部分：抓住对话中的关键词
**公式**：关键词 = 兴奋点

1. **提炼关键词**：当对方说一大段话时，找出其中的关键词，这些关键词可能是他的兴奋点。
2. **从兴奋点切入**：围绕这些兴奋点进行回应，使对话更加生动有趣。

## 第二部分：引导对方说出更多
**代码**：引导 = 提问 + 倾听

1. **提问**：通过提问引导对方说出更多，例如询问对方对照片的看法，或对某个话题的深入了解。
2. **倾听**：认真倾听对方的回答，从中获取更多信息。

## 第三部分：理解弦外之音
**公式**：弦外之音 = 表象 + 潜台词 + 暗藏玄机

1. **理解表象**：理解对方话语的表面意思。
2. **理解潜台词**：理解对方话语背后的真正含义。
3. **理解暗藏玄机**：理解对方话语中的隐含信息。

## 第四部分：引导对方说真话
**代码**：引导 = 共情 + 接受

1. **共情**：站在对方的角度理解他的感受。
2. **接受**：接受对方的真实想法，不要轻易评判。

## 第五部分：面对质疑的声音
**公式**：接话 = 真诚 + 义正言辞

1. **真诚**：以真诚的态度回应对方的质疑。
2. **义正言辞**：坚定地表达自己的观点。

## 总结
本节课中，我们学习了如何通过高情商的沟通技巧，让对方对你产生浓厚的兴趣。通过抓住关键词、引导对方说出更多、理解弦外之音、引导对方说真话以及面对质疑的声音，我们可以更好地与他人沟通，建立良好的人际关系。
