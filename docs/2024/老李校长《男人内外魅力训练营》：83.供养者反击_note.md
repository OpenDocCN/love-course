# 老李校长《男人内外魅力训练营》：83.供养者反击

🎼 情感之路充满挑战，充满人性的贪婪与邪恶。只有勇敢面对，才能成为真正的英雄。跟随我，一起揭开情感的潘多拉魔盒，洞察人性的善恶。

🎼 品味人性的情感与物质，才能达到人生的大圆满。实际上，“供养者”这个词并不能完全描述作为供养者的痛苦。他们常常感到自卑，分为轻度供养者和重度供养者。其中，重度供养者最为痛苦。

### 重度供养者的困境

重度供养者分为两种情况：一种是未发生关系的，只是牵手亲吻；另一种是偶尔发生关系的。但即使如此，他们也不愿意被拉黑，甚至希望对方能承认他们的关系。然而，当他们为对方花钱，投入感情时，却难以割舍这段感情。这时，他们最为痛苦。

### 如何停止花钱

1. **停止花钱**：首先要停止对对方的物质投入。
2. **要回钱财**：可以通过法律途径，如报警、曝光、找律师等，要求对方归还钱财。

### 轻度供养者的反击

轻度供养者已经与对方在一起，但对方的付出超出了他们的承受范围。这时，他们应该：

1. **拒绝不合理要求**：坚决拒绝对方的所有不合理要求。
2. **保持底线**：不要被对方调教成重度供养者。

### 重度供养者的反击

作为重度供养者，他们应该：

1. **寻求专业帮助**：如果不知道如何解决，可以寻求有经验的人士帮助。
2. **保持底线**：坚决反击，不要被对方的精神控制。

### 总结

本节课中，我们学习了如何应对供养者的困境。作为供养者，我们要勇敢面对，坚决反击，保护自己的权益。
