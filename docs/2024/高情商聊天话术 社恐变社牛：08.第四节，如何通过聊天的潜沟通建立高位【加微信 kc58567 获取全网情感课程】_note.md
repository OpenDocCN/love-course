# 高情商聊天话术 社恐变社牛：08.第四节，如何通过聊天的潜沟通建立高位

## 概述

在本节课中，我们将学习如何通过聊天的潜沟通技巧建立高位，避免在聊天中处于低位，从而提升自己在对方心中的形象。

## 核心概念

**公式**： 高情商聊天 = 娱乐自己 + 尊重对方 + 控制语气词 + 适度表情包

## 教程内容

### 1. 明确聊天目的

*   **思考**： 在聊天中，你是为了娱乐对方还是娱乐自己？
*   **原则**： 以娱乐自己为主，偶尔损对方，让自己感觉好。

### 2. 调侃与尊重

*   **案例**： 当对方做错事时，不要一味地安慰，可以调侃对方，但要保持尊重。
*   **原则**： 将对方视为小朋友，调侃她，但不要让自己显得幼稚。

### 3. 控制语气词

*   **现象**： 过多使用语气词，如“呢”、“哦”等，会让人感觉你过于主动寻求和谐关系。
*   **原则**： 尽量减少语气词的使用，避免暴露需求感。

### 4. 适度使用表情包

*   **原则**： 
    *   避免使用卖萌、装可爱等表情包。
    *   适度使用调侃、暧昧等表情包。
    *   语气包数量不要超过对方。

### 5. 赞扬与认同

*   **原则**： 
    *   使用认同对方的语气，如“做得不错，没想到你还可以做成这样”。
    *   避免居高临下的赞扬，如“哇，好厉害，果然你是最棒的”。

## 总结

本节课中，我们学习了如何通过聊天的潜沟通技巧建立高位，避免在聊天中处于低位。希望同学们能够将所学知识应用到实际聊天中，提升自己在对方心中的形象。
