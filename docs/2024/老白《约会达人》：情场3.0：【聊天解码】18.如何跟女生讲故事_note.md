# 老白《约会达人》：情场3.0：【聊天解码】18.如何跟女生讲故事

🎼大家好，我是老白。本节课将教授如何与女生讲故事。这或许对一些兄弟来说有些困惑，与女生讲故事究竟有何意义？实际上，无论是线上聊天还是约会，最终目的都是推进彼此关系。

在推进关系时，有些想法可能不便直接表达，或直接表达显得过于刻意。这时，我们可以通过讲故事来传达，让女生从故事中领会你的意图。

### 第一个表达

**核心概念**：我是一个有吸引力、有原则、专一的男人。

**故事示例**：我通常与朋友、闺蜜保持一定距离。之前，我前女友的一个久未谋面的闺蜜从国外留学回来。她拒绝唱歌，让我陪同。那天晚上我们玩得很开心，她晚上一直想和我喝酒。后来，大家喝多了，就去酒店开了房间。我睡沙发，她对我说觉得我挺好，很羡慕我前女友能有我这样的男朋友。后来，她不时找我聊天，说一些暧昧的话。我怕她发现后误会，于是告诉她这件事，他们俩就彻底闹掰了。虽然我没做错什么，但心里有点过意不去。所以，我一直和女朋友的闺蜜保持一定距离。

### 第二个表达

**核心概念**：我是一个努力上进、有理想、有目标、正能量的男人。

**故事示例**：年轻时，我热衷于各种圈子、行业的人。后来，我发现，无论别人多么厉害，都与我无关。只有自己强大了，才是真正的强大，别人也会尊重你。所以，我要把更多时间花在提升自己上，我经常是公司最后一个走的人。虽然工作很辛苦，但我觉得很充实。我觉得人得有追求的目标，这样才会不断成长和进步。

### 第三个表达

**核心概念**：我现在想认真谈恋爱，未来的女朋友会和我很幸福。

**故事示例**：以前，我确实有点爱玩，但现在玩够了。我想安安心心地找一个我喜欢的、也喜欢我的女生，好好过日子。我也不小了，到了该结婚的年龄了。所以，我现在很努力赚钱，也是为了以后和我在一起的那个人，我不想让我的老婆孩子跟着我过苦日子。可能不会特别富裕，但肯定会很幸福。

### 讲故事技巧

1. **练习**：在讲故事之前，可以先自己练习一下，试着写一些，看看自己是否能被自己感动。
2. **题材**：故事题材有很多，不能一一列举。要根据你想传达的意思去发挥。
3. **引导**：讲故事时，要逐步引导，不要直接讲，这样更真实。
4. **分段**：不要一口气讲完，要分段讲，观察女生的反应。
5. **内容**：不要讲自己卑微、弱小的事情，也不要讲太多人物，最好有1-2个转折或笑点。
6. **升级**：可以将故事升级，将女生带入故事中，暗示暧昧。

通过本节课的学习，我们了解到，通过讲故事可以委婉地推进彼此关系，具体讲什么故事要根据你想传达的意思来定。希望对大家有所帮助。
