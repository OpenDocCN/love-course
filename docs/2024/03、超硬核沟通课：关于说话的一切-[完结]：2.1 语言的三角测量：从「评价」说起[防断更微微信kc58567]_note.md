# 03、超硬核沟通课：关于说话的一切 - 2.1 语言的三角测量：从「评价」说起 🔺

在本节课中，我们将要学习语言活动中的一个核心机制——三角测量，并理解为何沟通不能仅停留在“把事情说清楚”的内容层面，而必须深入到权力与关系维度。

上一节我们探讨了好好说话为何困难，揭示了权力关系对语言的深层支配。本节中，我们来看看如何应对这种复杂性，并找到有效沟通的起点。

## 从基本模型到三角结构

现在，让我们清空桌面，从最基本的概念开始。一个简单的沟通模型包含发送者和接收者。发送者组织语言符号传达意义，接收者则解释这个意义。

但这里存在一个问题：意义从何而来？如果世界上只存在这两个人，他们将无话可说。因此，我们必须引入第三个维度——**事物**。它可以泛指语言之外，能被双方共同认知的一切对象。

有了这个“事物”，便产生了“意义”。然而，你的意思和我的意思可能不同，交流便由此发生。在反复交流中，人们发现，**意义最终取决于权力**。权力结构预先支配了词语的内涵和谈话的走向，权力流动的规律界定了言说场景的类型。

由此，我们可以方便地将语言活动分为两个维度：
*   **关系维度**：涉及人与人之间的权力关系。
*   **内容维度**：涉及人与事之间的信息传递。

语言活动在关系维度的目标是建立亲和关系，在内容维度的目标是形成深度理解与共识。理想的目标是“既说到点子上，又说进心坎里”。

## 为何必须从关系侧入手？

许多人希望从内容侧入手，认为把事说清是一个如何组织语言的结构工程学问题，似乎存在许多可操作的套路与公式。

但结果往往是，学了诸多沟通技巧，依然说不清一件事，或者即便说清了，也未能达成沟通的原始目的。沟通的目的并非单纯“说清一件事”。为了追求这种“帅”，我们可能搞砸很多事情。

**注意，这是一个三角结构，我们必须从关系这一侧入手。** 为什么？让我们以“评价”为例。

评价的本质是什么？语言学家指出，**评价的本质是精神世界的一种三角测量法**。其公式可概括为：

```
【评价】= 以第三方（事物/对象）为参照，反向明确【我】与【你】之间的相对位置与关系。
```

当你我在虚空中相对时，无法真正理解彼此。但一旦你我之间出现哪怕一块石头，情况就会不同。我说它来自天上，你说它来自地下。这块石头可以是物品、人物或事件，但本质不变：**当我们评价你我之外的任何对象时，正是在建构你我之间的关系**。

一致性评价拉近关系，分歧性评价则相反。但总存在某种裁决分歧的知识系统，能够援引它们的人就能占据更高的权力位置。无法对事物进行评价，则会因丧失参照点而无法理解自身与他人、与社会的关系，从而失去权力。

因此，你会发现网络上充斥着“如何评价XXX”的讨论。这正是无数人在进行的“精神世界的三角测量”。

## 语言的功能与权力的核心地位

按照语言学家的说法，语言主要有三个功能：**评价**、**告知**和**请求**。如果你对关系维度不闻不问，只关心在内容层面如何组织得体的评价、告知和请求，那便是舍本逐末。

我们总是在**权力关系**中说话：作为孩子、学生、员工、公民……我们能理解一句话及其背后的深意，并非仅仅因为懂逻辑和语法，更是因为我们作为社会动物，天然敏感于自身与对象之间的权力关系。

**运用权力、建立关系，是语言活动动机背后的动机，目的背后的目的。** 忽视这一层，语言活动就会呈现出无法理解的复杂性；反之，一旦重视它，语言活动便会呈现出简单甚至枯燥的规律。

## 本节核心：建立亲和关系的机理

我们今天的目标正是揭示这种规律。从哪里开始？回顾关系维度的目标：通过语言与他人建立亲和关系。

因此，我们的问题可以细化为：为什么有些语言能够建立亲和关系？其作用原理是什么？换句话说，那些能“说进别人心坎里”的话，作用机理是怎样的？

直接揭晓答案：从结果上看，这些话都通过某种特别的方式，制造了一种深层次的**相互尊重**。尊重是我们作为社会动物的一种姿态，通过它我们承认对方权利的存在。

重点在于“特别的方式”。我们用八个字概括：**打破预期，顺应张力**。即打破特定权力结构下的交往预期，又顺应特定权力结构带来的欲望张力。这里需要重点关注三个名词：**特定权力结构**、**交往预期**、**欲望张力**。下面将一一解释。

所有交往场景背后都有特定的权力结构。在微观个体层面，我们可以将其分为三类：
*   优势权力场景
*   弱势权力场景
*   平等权力场景

我们参与到这些权力场景中，总是带着各自的**欲望张力**，即主体想要实现自身认同的一种心理趋力。由于互动双方都具备这种趋力，为避免冲突，我们逐渐习得了在各类权力场景下的交往原则，形成了所谓的**交往预期**。

**交往预期**指的是双方基于对彼此权力关系与欲望张力的察觉，对即将发生的交往活动持有的预判、期许和成见。我们都是带着各自的预期入场展开沟通的。没有这种预期，交往活动将变得不可理解；但也正是这些预期，常常让我们陷入沟通困境。

在接下来的课程中，我将通过一连串例子来说明，什么是“打破预期，顺应张力”，以及它何以能制造那种深层的相互尊重。

本节课中，我们一起学习了沟通的三角结构模型，明确了“评价”的本质是一种精神世界的三角测量，其核心在于通过第三方对象来建构关系。我们认识到，有效的沟通必须从权力与关系维度入手，而不能局限于内容组织。最后，我们引出了建立亲和关系的核心机理——“打破预期，顺应张力”，并定义了**特定权力结构**、**交往预期**和**欲望张力**这三个关键概念，为后续的具体分析打下了基础。
