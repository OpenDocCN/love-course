# 清风老师合集：14链接贵人，让贵人帮你的实操方法论21期：191.第六课：鉴别思维：鉴别真假牛人，快速识别真伪

## 概述
在本节课中，我们将学习如何通过鉴别思维来识别真假牛人，快速识别真伪。

## 识别大佬的方法
以下是清风老师分享的识别大佬的方法：

### 1. 逆向思维
- **核心概念**：逆向思维是一种重要的思维方式，通过反向思考来识别真假大佬。
- **公式**：逆向思维 = 常规思维的反面
- **示例**：一个有钱的大佬潜伏到竞争对手公司，应该低调侦查，而不是炫耀财富。

### 2. 理解利益诉求
- **核心概念**：理解对方背后的利益诉求，建立思维模型。
- **公式**：利益诉求 = 背后动机 + 目标
- **示例**：大佬炫耀的不是财富，而是圈子，因为圈子不能造假。

### 3. 二度人脉认证
- **核心概念**：通过二度人脉认证或社交认证来验证大佬的真实性。
- **公式**：二度人脉认证 = 初步背景调查 + 圈子调查
- **示例**：通过起查查、天眼查等软件查询公司情况，通过百度、抖音等了解真实故事。

### 4. 行为一致性
- **核心概念**：通过观察大佬的行为一致性来鉴别真假。
- **公式**：行为一致性 = 表达 + 行为
- **示例**：大佬说有实力，但饭钱却要打折，行为和表达不一致。

## 总结
本节课中，我们学习了如何通过鉴别思维来识别真假牛人，快速识别真伪。通过逆向思维、理解利益诉求、二度人脉认证和行为一致性等方法，我们可以更好地识别大佬，找到适合自己的贵人。
