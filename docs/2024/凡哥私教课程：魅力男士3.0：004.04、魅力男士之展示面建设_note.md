# 凡哥私教课程：魅力男士3.0：004.04、魅力男士之展示面建设

🎼本节课将为大家讲解魅力男士建设的另一方面，即展示面建设，也就是吸引力建设的第二个层面。主要讲解的是微信朋友圈的建设，以及其他社交软件APP上的朋友圈或空间建设。🎼

## 一、朋友圈建设的重要性

朋友圈建设是我们对自己的一种定位，也是女人对我们的定位。我们需要清楚自己是一个怎样的人，有什么闪光点和亮点，以便展现给对方。对方看到这些关键信息后，也会给予我们一个定位。

## 二、朋友圈建设的目的

朋友圈建设需要像经营一个庄园一样，重点是要规划的。我们的目的是塑造一个个人形象，制造一个个人吸引力最神力的地方。别人一看你的朋友圈就能一目了然，你是一个怎样的男人，有什么兴趣爱好，有什么特长，业余生活又是怎样的。

## 三、朋友圈内容建设

### 1. 删除内容

- 引用链接的文章
- 长篇大论的政治见解
- 心灵鸡汤
- 没必要的抱怨

### 2. 建议发布的内容

- **宠物**：展现爱心、勤奋工作、情感状态等价值。
- **美食**：展现高品质生活、高价值。
- **风景**：展现美好瞬间、个人感受。
- **自拍**：展现亲近感、无距离感。

## 四、朋友圈建设技巧

1. **思考价值**：先思考要展现什么价值，再进行拍摄和发布。
2. **收集素材**：多收集美图，利用手机备忘录记录好的句子和照片。
3. **使用软件**：美颜相机、美图秀秀、激萌等。
4. **展示高价值**：展现生活中高质量的部分，高价值的社交圈、工作等。

## 五、核心要点

- 品味
- 审美
- 思想
- 灵魂

## 总结

本节课中我们一起学习了魅力男士之展示面建设，了解了朋友圈建设的重要性、目的、内容建设以及技巧。希望兄弟们能够认真思考，将所学知识应用到实际生活中，打造一个高价值的展示面。
