# 21、凯男的社交情商课：第五节课：七步聊天：010.破解废物测试

在本节课中，我们将学习如何破解女性在社交过程中抛出的“废物测试”，这是一种测试男性是否为高价值男性的方式。

## 什么是废物测试？

**公式**：废物测试 = 女性无意识发出的测试男性价值的问题

废物测试通常以问题形式出现，女性通过这些问题来测试男性是否具备高价值。

## 破解废物测试的重要性

1. **快速吸引女性**：不会破解废物测试，将无法快速吸引女性，也无法推进关系。
2. **通过高分女性的筛选**：高分女性会通过废物测试筛选男性，无法通过测试将失去机会。

## 破解废物测试的方法

1. **夸张到无限大**：通过夸张的回答，达到幽默的效果，例如：“我已经聊了第999个了，恭喜你，你是第1000个，你将会得到一个神秘礼物。”

2. **曲解**：曲解女性的用意，例如：“你太可爱了，你觉得你跑得过我吗？”

3. **进入女性框架，再建立自己的框架**：先进入女性的框架，再重新建立自己的框架，例如：“你去找我大舅吧，我大舅还单身，他是钻石王老五，我可以给你介绍一下。”

4. **直接保持自己的框架**：直接保持自己的框架，但可能显得生硬，例如：“我也不喜欢比我大的。”

5. **直接无视**：直接无视女性的问题，例如：“你今天吃饭了吗？”

6. **直接废测回去**：直接对女性的废物测试进行回应，例如：“你是不是觉得你老了？”

## 高级废物测试

**公式**：高级废物测试 = 女性通过特殊方式测试男性反应

例如，女性会说：“其实我很丑的，你还是不要出来见我了吧。” 高价值的男性不会轻易上钩。

## 总结

破解废物测试的核心是不解释、不迎合、不跪舔，同时展现出自信不为所动的男神态度。在回答问题时，要让自己处于奖品性的位置。
