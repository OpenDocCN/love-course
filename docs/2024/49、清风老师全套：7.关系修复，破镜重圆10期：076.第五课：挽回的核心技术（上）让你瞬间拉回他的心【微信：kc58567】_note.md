# 清风老师全套：7.关系修复，破镜重圆10期：076.第五课：挽回的核心技术（上）让你瞬间拉回他的心

## 概述
在本节课中，我们将学习如何运用挽回中的核心技术，包括冷冻术、后撤数和客观理由测试术，来修复关系并重圆。

## 冷冻术
**定义**：冷冻术是一种应对排斥期的策略，通过语言和行为上的冷冻来缓解对方的情绪。

**步骤**：
1. **语言冷冻**：承认分手事实，给予对方空间，避免过度讨好。
2. **行为冷冻**：主动离开，给对方时间冷静和怀念。

**公式**：
```
冷冻术 = 语言冷冻 + 行为冷冻
```

## 后撤数
**定义**：后撤数是在感情危机时主动撤退，避免危机扩大的策略。

**步骤**：
1. 主动发现危险，撤退。
2. 留下联系理由，如物品、书信等。

**公式**：
```
后撤数 = 主动撤退 + 留下联系理由
```

## 客观理由测试
**定义**：通过无关话题测试对方在后撤后的反应，了解排斥程度。

**测试理由**：
1. 个人物品
2. 账号密码
3. 软件设置
4. 课程推荐
5. 业务对接
6. 共同朋友
7. 物品采买

## 总结
本节课中，我们学习了冷冻术、后撤数和客观理由测试术，这些技术可以帮助我们在挽回过程中更好地应对排斥期，修复关系。在下一节课中，我们将继续学习其他挽回核心技术。
