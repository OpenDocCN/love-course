# 绅士派《迷情烽暴2023》：007.第五集 展示面到底怎么

在本节课中，我们将要学习如何拍摄高质量的展示面照片，核心在于掌握构图思维。我们将从设备选择、前期准备到具体的拍摄构图方法进行详细讲解，帮助你拍出自然、真实且具有吸引力的照片。

## 设备选择与前期准备

上一节我们介绍了课程主题，本节中我们来看看拍摄前的准备工作。首要任务是选择合适的拍摄设备。

我们的拍照设备在这个时代已经更新换代多次。如果个人展示中没有两三组拿得出手的照片，已经说不过去了。毕竟现在已不是远古时期，而是人手一部手机的时代。

每个人都用自己的手机记录生活。手机拍照携带轻巧，后期修图也很方便。因此，拍摄展示面使用手机即可。但需要说明，不要使用自带美颜功能的手机，其拍出的效果往往失真、不自然，后期修图痕迹也会过重。

这是我们一直强调的事情。设备选择的标准建议是苹果12及以上型号。有兄弟会问三星、华为、OPPO等品牌。说实话，我自己也使用华为等品牌的手机，系统好用并不代表相机成像效果也好。

当前展示面的效果要求越来越真实和自然。拍照和修图越不刻意，成片效果越好。下面以苹果设备为例进行说明，其他品牌手机也可参考。

以下是手机相机的基础设置步骤：
1.  打开“设置”。
2.  找到“相机”。
3.  打开“网格”。
4.  打开“镜头矫正”。

## 人物全身照构图法

完成了基础设置，接下来我们学习具体的人物全身照构图方法。

拍摄人物全身照时，请遵循以下步骤：
1.  将手机镜头倒置拍摄。
2.  把人物脚部放置于九宫格最下面一条线上。
3.  人物身体占据画面约三分之二的位置。
4.  头顶预留约三分之一的空白，这在艺术角度上称为“留白”。

## 服装、配饰与造型建议

构图之外，人物的整体造型也至关重要。本节我们来看看服装、配饰与发型的搭配建议。

首先是服装建议。搭配应根据场景色调进行。不懂挑选的话，无脑选择白色即可，白色是百搭色。

有服装就少不了画龙点睛的配饰。以下是推荐的必选配饰单品：
*   戒指
*   手表
*   手链
*   项链
*   墨镜

冬季则可以考虑：
*   手套
*   帽子
*   围巾

请根据个人风格和审美挑选意向的服装与配饰。都可以发给我参考，也可以找我帮你搭配。

最后，发型是最关键的一步。发型是人的第二张脸，不重视会显得邋遢，毫无气质。

如果自己不懂如何打理，可以在大众点评的同城服务中，寻找排名前五的理发店，挑选一家评价好、顺眼的，让设计师为你设计。

做完发型后，请发型师用手机围绕你的新发型，从各个角度录制一段视频。以后无论去哪家店，都可以直接出示这段视频给发型师看，让其按照视频中的造型为你设计。

此外，有些兄弟皮肤状态一般，建议打一层粉底以提升精气神，但无需过于夸张。男生化妆的核心在于：
*   遮瑕
*   修眉
*   提亮肤色

## 场景拍摄构图实战

前期准备就绪后，我们开始行动去拍摄展示面。下面教大家餐厅和车辆的拍摄构图法。

我们知道自己哪边脸或哪个拍照角度好看，就告诉跟拍的兄弟，让他多拍你熟悉的角度。有些兄弟喜欢侧脸，显得棱角分明，脸部比较立体；有的兄弟喜欢拍正脸，方便后期修图。大家需要多交流、多拍，才能得到自己想要的效果。

如果是两人或三人以上的合照，可以更好地凸显中心位（C位），场景的烘托性也会更强。这里不展开讲，线下会给大家示范。

接着我们去拍车。市面上车型很多，我们在线下也会为大家准备豪华车、超跑、商务车等车型。拍车有一个细节：要把车灯或车门打开，这表示车是自己的物品，而不是蹭来的。
