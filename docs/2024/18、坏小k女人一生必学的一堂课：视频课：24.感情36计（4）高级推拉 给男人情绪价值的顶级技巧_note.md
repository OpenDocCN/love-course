# 18、坏小k女人一生必学的一堂课：视频课：24.感情36计（4）高级推拉 给男人情绪价值的顶级技巧

🎼 感情36计第四大技巧，四种高级推拉技巧，给男人情绪价值的顶级技巧。首先了解最低层次的推拉技巧。

**推**：字面意思，推开对方，可利用打压、调侃、挑战，开开玩笑，展现灵性。

**拉**：将对方情绪波动下来，赞美、保护、肯定、认同他。

例如，对方问车怎么样，可用公式“先肯定再否定”进行回复。

推拉用得好，男人会感到困惑，你到底喜不喜欢我？

**案例**：绿茶收到帅哥信息，给了他一巴掌，然后主动打电话，发出混合信号，让他经历情感过山车。

**技巧**：

1. **情感过山车**：有时赞美、对他好，有时冷淡、开玩笑，让他情绪产生微妙变化。
2. **打破情绪状态**：让他从一种情绪状态切换到另一种，让他感到愉悦。
3. **后悔感**：让他产生后悔行为，例如主动追求，然后退缩，让他后悔。
4. **后悔语言**：故意谈论不得体的话题，让他后悔自己的行为。

**高级技巧**：

1. **显示与隐藏**：展示高价值，同时隐藏高价值，例如展示旅行经历，但原因是为了吃麦当劳冰淇淋。
2. **好人和坏人**：让他觉得你难以接近，然后给他一线生机，让他证明自己。
3. **推翻**：故意关心他的回应，然后推翻，例如询问他的计划，然后打断他，表示不关心。

**核心**：

- 推拉技巧要灵活运用，不要持续装备。
- 不要让对方觉得你在玩弄他，而是伺候他。
- 让他感到愉悦，而不是耻辱。

**总结**：

本节课中我们一起学习了高级推拉技巧，这些技巧可以帮助你更好地理解男人，与他们建立更深入的关系。
