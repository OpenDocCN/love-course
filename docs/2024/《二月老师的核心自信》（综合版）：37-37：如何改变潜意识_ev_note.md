# 《二月老师的核心自信》（综合版）：37-37：如何改变潜意识

## 概述
在本节课中，我们将学习如何通过改变潜意识来提升自信。

## 改变潜意识的方法

### 自我暗示
**公式**：自我暗示 = 思想灌输

自我暗示是指一个人对自身进行的思想灌输。以下是如何进行有效的自我暗示：

1. **寻找优点**：每天寻找自己身上的优点。
2. **积极思考**：多想一些好的事情，多微笑。
3. **力所能及**：自我暗示必须是力所能及的事情。

### 重复
**公式**：重复 = 信念 → 坚定不移

重复是改变潜意识的关键。以下是如何重复：

1. **设定可实现的目标**：目标必须是可实现的。
2. **细化步骤**：将目标分为几个可实行的步骤。
3. **增加负面意义**：思考未达成目标的负面后果。
4. **增强正面意义**：思考达成目标的正面意义。

### 内模拟
**公式**：内模拟 = 内心想法 → 表情 → 状态

内模拟是指当一个人内心在想什么事情时，他的表情会不由自主地模拟。以下是如何进行内模拟：

1. **心理预演**：想象成功的景象。
2. **清晰具体**：梦想要越清晰、越具体越好。

## 总结
本节课中我们一起学习了如何通过改变潜意识来提升自信。通过自我暗示、重复和心理预演，我们可以改变潜意识，从而提升自信。
