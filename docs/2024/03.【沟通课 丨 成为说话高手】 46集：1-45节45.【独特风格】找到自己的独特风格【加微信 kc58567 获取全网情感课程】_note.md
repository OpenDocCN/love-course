# 课程03：找到你的独特说话风格 🎭

在本节课中，我们将探讨如何将之前学习的各种有趣说话技巧，整合并发展成属于你自己的独特风格。当你已经掌握了一些方法并进行了实践后，下一步就是思考如何形成个人特色。

上一节我们介绍了在各种场景中应用有趣技巧，本节中我们来看看如何将这些技巧内化，形成你独一无二的表达风格。

## 理解个人风格

那些优秀的喜剧大师、脱口秀演员或演讲者，他们的风格各不相同。学习有趣的说话方法，关键在于将前面讲过的所有方法和技巧视为你的**工具**或**武器**。例如，你可以选择使用夸张（工具A），也可以选择把故事讲得生动（工具B），这是你的自由。

**核心公式：你的风格 = 你的选择 + 你的偏好 + 你的性格特质**

因此，成为说话有趣的人，不需要面面俱到，而是需要逐渐找到并强化自己的风格。

## 如何找到自己的风格

以下是三个帮助你探索和确立个人风格的步骤。

### 1. 探索个人喜好

首先，需要问自己一个核心问题：我发自内心喜欢什么样的有趣方式？

以我自己为例，我特别喜欢**比喻和类比**。当我读书或看演讲时，非常钦佩能将一个事物巧妙比喻成另一个事物的能力。因此，我会刻意积累和练习这种方式。此外，我也喜欢在讲话时加入**表演和演绎**。

也有人，比如喜剧演员凯文·哈特，他的风格是**调侃式的自夸**，他经常说：“你们都知道，我是个大人物。”

所以，你可以这样做：回顾你看过的有趣表达，对应到课程中学过的方法，思考哪个最让你欣赏、最想学习、最让你着迷。

```text
行动指南：
1. 列出你欣赏的有趣表达者。
2. 分析他们主要使用了哪些技巧（如夸张、反转、类比等）。
3. 标记出你最想模仿和学习的技巧。
```

### 2. 对听众反馈保持敏感

当你与他人沟通或完成一段讲话后，尝试回忆：自己说的哪句话、哪种方式获得的反馈最热烈（如笑声、鼓掌、互动）？

如果自己无法准确回忆，可以直接询问他人。在我过去的培训中，我会一边讲课，一边观察学员的反应。大家反馈好的部分，例如将商业知识换成生活小例子，我会加强。反馈一般的部分，我会避免或改进。

课后，我经常与助教、培训负责人或学员交流，了解他们对哪些内容更感兴趣。这些反馈常常带来惊喜，让我发现一些自己未意识到的亮点或不足。

**反馈就像一面镜子**，能真实地反映你哪种风格最有效、最受认可，帮助你不断调整。

### 3. 寻找自己的舒适区

我们常听到“跳出舒适区”，这本身没错。但最终要形成稳定的个人风格，你需要在**自己的舒适区**内才能做到。

以我为例，十多年前我开始在会议和讲台上发言。起初完全没有舒适区，全是“不舒服区”。后来我慢慢发现，当与听众有更多**互动和调侃**时，我感到比较舒适。再后来，当我**讲故事并掺杂表演**时，我感到非常放松和自然。这些方式就逐渐变成了我的风格。

**舒适区的判断方法**：在你进行了大量尝试、练习并获得反馈，并经过调整之后，反思自己在什么状态下最舒适、最自然。是在自嘲时最放松，还是在讲故事时最享受？某次特别放松的分享，是因为内容感兴趣，还是用了特定的互动方式？

你的风格会在你的舒适区里绽放。

## 课程总结

本节课中，我们一起学习了如何找到自己的独特说话风格。

首先，**向内探索**，询问自己真正喜欢的风格是什么，你喜欢的观看、聆听和阅读内容都会给你答案。

其次，**向外观察**，重视并分析听众的反馈，这是一种客观且有价值的输入，能帮你判断风格的有效性。

最后，**定位舒适区**，在尝试过各种讲话方式和场合后，找到那个让你感到最自然、最放松的表达状态。

祝你成功找到自己的风格，成为说话有趣的人，让有趣为你带来成功与幸福。
