# 情绪价值课-【阿娇】：第15节 如何送礼物给对方情绪价值

大家好，本节课我们将探讨如何通过送礼物为对方提供情绪价值。

### 1. 礼物与情绪价值

通常，女生收到礼物时会感到开心和愉悦，觉得自己被重视。同样，男生收到礼物时也会有类似的感受，尽管他们对礼物的要求可能没有女生那么高。

### 2. 送礼物的原则

- **避免过度昂贵**：礼物不应超出你的能力范围，以免显得卑微或缺乏自我认同感。
- **巩固感情**：送礼的目的是巩固感情，让对方意识到你在意他。

### 3. 不同关系阶段的礼物选择

#### 3.1 暧昧期

- **新鲜感和刺激**：选择互动性强的玩具，增强对方的期待感。
- **小惊喜**：送食物、水果、书籍等小物品，让对方感到被在意。
- **文案和诗歌**：让对方意识到你在他心中的位置。
- **定制礼物**：如刻有名字的钥匙扣、手机壳等。
- **暗示约会**：邀请对方尝试新餐厅，增进关系。

#### 3.2 异地恋

- **身份认同感和归属感**：送功能性强的物品，如包包、装饰品等。
- **视频和语音**：让对方在想你时能听到你的声音，看到你的身影。
- **精美装饰品**：包装精美的礼物，让对方感受到你的用心。
- **数码产品**：加强你在他身边的存在感。
- **定制旅行**：提前订票和酒店，带他去未去过的地方。

#### 3.3 热恋期

- **表达在意和依赖**：通过礼物表达你对他的在意和依赖。
- **适当占有欲**：通过礼物展现合理的占有欲，增加感情。
- **高档内衣**：表达你对他贴身衣物的挑选权。
- **护肤品**：为他准备护肤品，表达关心。
- **锦理牙刷**：提醒他口腔卫生，展现关心。
- **体验式礼物**：如健身卡、演唱会门票等，提供情绪体验。
- **支持爱好**：根据他的爱好准备相关礼品。

#### 3.4 平淡期

- **新鲜感和回忆感**：通过礼物保持生活的新鲜感，巩固感情。
- **回忆性礼物**：代表你们共同经历过的过往，表达美好回忆。
- **情趣用品**：提供服务性的行为，增进感情。
- **实现梦想**：帮他实现小梦想或购买一直想买的东西。
- **纪念型礼物**：如情侣照片、记录生活点滴等。
- **未来期待**：询问他的目标，为他实现目标提供支持。

### 4. 总结

本节课我们学习了如何通过送礼物为对方提供情绪价值。礼物不仅可以升温感情，还是表达爱意的媒介。在下一节课中，我们将探讨如何引导对方回馈礼物，保持关系的平衡和长久。
