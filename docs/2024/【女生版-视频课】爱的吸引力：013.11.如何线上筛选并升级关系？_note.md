# 【女生版-视频课】爱的吸引力：013.11.如何线上筛选并升级关系？

🎼Baby think me。🎼How want you to be my best friend love。🎼的好子。大家好，我是米娅。成年人只做筛选，不做改变。

**概述**：本节课将教授如何通过线上聊天筛选合适的伴侣，并升级关系。

**线上筛选伴侣**

*   **线上渠道优势**：提供随时随地沟通的平台，覆盖更广泛的人群，增加认识男生的机会和选择范围。
*   **线上聊天问题**：虚拟性导致了解和信任有限，难以判断对方真面目。
*   **筛选原则**：
    *   保持自我框架，不卑不亢。
    *   观察对方一致性、真实性。
    *   观察对方是否愿意深入交流。
    *   观察对方是否付出实质行动。
    *   保护个人隐私信息。
    *   保护钱包，不轻易转账。
    *   相信直觉，果断拒绝不适感。

**筛选阶段**

*   **观察越级行为**：越级聊天、暴露需求感、急切关系升级等。
*   **观察真诚度**：回避问题、不提出见面、身份造假等。
*   **观察金钱观**：借钱投资、拉入伙等。
*   **观察尊重程度**：不尊重意见、不支持鼓励等。

**铺垫见面**

*   **释放善意和真诚**：让对方感受到友好和关注。
*   **注意聊天雷区**：避免深入感情、三观、收入等话题。
*   **引导见面**：自然轻松友好地表达期望。

**关系升级阶段**

*   **常见问题**：尬聊、例行公事、无法深入话题、内耗瞎想等。
*   **聊天真相**：
    *   男生喜欢你的前提是对你有探索欲和分享欲。
    *   摆正心态，避免患得患失、控制欲强。
    *   留白产生爱情，避免过度分享。
    *   提升沟通能力。
*   **建立规矩**：让对方明白需要为你做什么，哪些事不能做。
*   **案例**：通过引导对方付出行动，让对方明白付出越多，越不容易离开。

**总结**：线上聊天可以帮助我们认识更多人，拓宽社交圈子。筛选合适的伴侣后，适时推进关系是最佳选择。
