# 林老头《框架课》：聊天中的方向下（视频）

## 概述

在本节课中，我们将学习如何在聊天中有效地展示自己，从而吸引对方并建立更深层次的关系。

## 第一目标的重要性

**公式**：设定清晰的第一目标 → 提升动力 → 完成目标 → 获得成就感

在开始聊天之前，设定一个清晰的第一目标至关重要。这有助于你保持动力，并确保你不会在聊天过程中迷失方向。

**以下是一些设定第一目标的方法**：

*   **数量目标**：例如，先加三个好友，先聊成朋友，先跟他们聊5天等。
*   **质量目标**：例如，提升聊天技巧，建立更深层次的关系等。

## 聊自己的重要性

**公式**：展示自己 → 建立连接 → 吸引对方

在聊天中，更多地聊自己而不是对方，可以帮助你建立连接并吸引对方。

**以下是一些聊自己的方法**：

*   **分享你的经历和想法**：例如，分享你的兴趣爱好、工作经历、旅行经历等。
*   **展示你的个性和特点**：例如，分享你的幽默感、乐观态度、独特见解等。
*   **分享你的生活琐事**：例如，分享你今天遇到的有趣事情、你最近买的有趣东西等。

## 身份定位

**公式**：明确身份 → 强化特点 → 增加魅力

在聊天中，明确自己的身份并强化自己的特点，可以增加你的魅力。

**以下是一些身份定位的方法**：

*   **游戏玩家**：将聊天视为一场游戏，将对方视为NPC，专注于提升自己。
*   **探险家**：将聊天视为一场探险，不断探索新的话题和经历。
*   **艺术家**：将聊天视为一种艺术创作，用你的语言和表达方式吸引对方。

## 实际操作

**以下是一些实际操作的建议**：

*   **分享你的生活琐事**：例如，分享你今天遇到的有趣事情、你最近买的有趣东西等。
*   **展示你的兴趣爱好**：例如，分享你喜欢的电影、音乐、书籍等。
*   **分享你的观点和见解**：例如，分享你对某个话题的看法、你的生活哲学等。

## 总结

本节课中，我们学习了如何在聊天中有效地展示自己，从而吸引对方并建立更深层次的关系。通过设定清晰的第一目标、聊自己、明确身份定位，并采取实际操作，你可以提升自己的聊天技巧，并吸引到更多有趣的人。

*   聊自己的重要性
