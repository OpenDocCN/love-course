# 凡哥私教课程：情绪挽回法：004.挽回黑科技四（掌握两大技巧，让她主动找你复合）

## 概述

在本节课中，我们将学习如何通过激发对方的正面情绪来挽回感情，并掌握两大技巧：内疚情绪激发和责任感激发。

## 内疚情绪激发

**核心概念**：

*   **内疚情绪**： 当一个人伤害或意图伤害他人后，产生的道德情绪产物。
*   **激发内疚情绪**： 通过让对方感受到伤害，从而产生内疚情绪，推动其接受复合。

**方法**：

*   **长信法**： 写一封长信，描述过去美好的回忆和分手带来的伤害，让对方感受到内疚。
*   **点滴可怜法**： 通过朋友圈等渠道，分享一些不太好的生活场景，让对方感受到你的不易。

**注意事项**：

*   **使用对象**： 道德感强、自尊心高的人。
*   **避免过激行为**： 自残、自杀威胁等行为不可取。
*   **时机把握**： 选择合适的时机分享，避免让对方反感。

## 责任感和自我满足激发

**核心概念**：

*   **责任感**： 女生同情弱者，认为帮助别人自己的存在才有价值。
*   **自我满足**： 通过尊重和自我实现来组成，是激发责任感的内驱力。

**方法**：

*   **寻求帮助**： 通过第三方或直接方式，请求对方帮忙，例如照顾宠物、挑选礼物等。
*   **强化尊重和自我满足**： 在帮助过程中，表达感激和依赖，让对方感受到被尊重和满足。
*   **发起复合总攻**： 在激发起对方的正面情绪后，通过暗示或直接方式，提出复合。

## 总结

本节课介绍了如何通过激发对方的正面情绪来挽回感情，并掌握了内疚情绪激发和责任感激发两大技巧。在实际操作中，需要根据具体情况灵活运用，并注意时机和方式，以提高挽回成功率。
