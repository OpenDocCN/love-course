如果你的时间非常特殊，短于一分钟，那么应立即去医院进行生殖器相关检查，确认器官是否有病变，并听从医生的安排。

*   **青春期的频繁自慰**：追求快速刺激，形成了条件反射。
*   **对视觉刺激反应过度**：一看到兴奋画面就忍不住。

正是由于高频、过快的自我刺激，让兴奋度阈值变高，形成了快速结束的习惯。因此，我们只需要改变这个习惯，将兴奋度降下来，就可以延长时间。

通常，我们把发射过快分为两大类型。以下是具体分类，请看看你属于哪一种：

**1. 敏感性（忍耐力差）**

## 延时的核心原理

想要增强对时间的把控力，就需要了解延时的原理。射精分为三个阶段：
1.  **快感期**
2.  **射精紧迫期**
3.  **射精期**

其中，我们能够自主控制的只有**快感期**。射精紧迫期和射精期都是一瞬间的事情，不受大脑控制，属于植物神经行为。

我们来看一下这两张示意图。在你不自主发射的情况下，射精控制区非常窄。因此，当你感觉到快感积蓄后，到临界点就立刻喷射了，这是绝大多数男性的过程。

而延时，是一个可以**自主调节快感期时间**的过程。目标是让射精控制区变宽，反复在快感和临界点之间徘徊，延长快感幅度，最后再结束。这样，时间自然就延长了。

所以大家明白了吗？**延长快感期才是控制时间的好方法。**

## 延时训练方法

现在，我们一起进行具体的延时训练。训练的核心是降低敏感度和增强PC肌控制力。

#### 1. 腹式呼吸法（降低兴奋度）

首先，我们要改变以往的呼吸方式，采用腹式呼吸法。具体做法如下：

1.  将手轻轻放在腹部。
2.  **吸气**时，让腹部向外鼓起。
3.  **吐气**时，让腹部轻轻向内收紧。
4.  连续重复，一定要缓慢地吸气、吐气。

如果站立时感受不深，可以躺下模仿睡眠时的呼吸方法，因为我们在睡着时会自动切换成腹式呼吸模式。你可以在躺着时，在腹部放一本书，观察书是否随着呼吸抬起落下。如果有，说明你学会了。

**作用**：腹式呼吸有助于降低兴奋度，缓解敏感，调节大脑的基础兴奋等级。在过程中感觉太兴奋时，可以停下来做两次腹式呼吸。

#### 2. PC肌组合训练（增强控制力）

预备：排空大小便，避开饭前饭后一小时。平躺在瑜伽垫上，调匀呼吸，放松。

**热身训练（夹球练习）**
1.  双腿分开，双手放身体两侧，将小球放在两腿之间夹紧。
2.  **吸气**，同时往腹腔方向收紧PC肌，嘴巴发“嘶”音，保持3秒。
3.  **吐气**，放松。
4.  注意：放在小腹的手不应感到腹部用力；放在会阴的手应感到四周肌肉向内收缩。这才是正确的PC肌收缩。
5.  进阶：吸气收紧PC肌，坚持5秒，最后一声干脆的“哦”音时提住，然后吐气放松。
6.  每次练习时，尝试比上一次夹得更久，目标每次能夹紧10秒，呼气时放松5秒。每次做三组。

**臀桥练习**
1.  平躺，双脚踩地，夹住小球。
2.  抬起臀部，稍微离地。
3.  **吸气**，收紧（PC肌）不动。
4.  **呼气**，抬升臀部，保持住。
5.  **吸气**不动。
6.  **呼气**，放下臀部。
7.  做3到5组。

**腰肌左右侧移训练**
1.  小腿自然弯曲，脚掌贴地，夹住小球。
2.  用腹部力量支撑起上半身，双手自然放在左右两侧臀部位置，头微扬。
3.  此时夹紧PC肌，以臀部为中心保持不动。
4.  利用腰腹力量牵引上半身，左右侧移。
5.  左右移动一次后，放松PC肌。
6.  一组做3次。

**骨盆灵活训练**
1.  小腿自然弯曲，脚掌贴地，双手放在骨盆上。
2.  做前后移动：以肚脐和耻骨为参照点。
3.  **呼气**，用耻骨去找肚脐（向上向前顶）。
4.  **吸气**，用肚脐向后向下找耻骨（向后翘）。
5.  重复20次。

**骨盆侧移与画圈训练**
1.  姿势同上，双手放在骨盆上做左右移动。
2.  **呼气**，提拉PC肌回到中间；**吸气**，提拉另一边。
3.  左右循环，一个循环做20次，做2到3组。
4.  加入画圈运动：先向左画一圈，再向右画一圈，做20次。

**侧位灵动练习**
1.  侧躺，右手枕着头，左手放在骨盆。
2.  右脚伸直，脚背勾起；左脚放在前方。
3.  抬升右脚，保持自然。
4.  反过来再做一次。
5.  做3到5组，每组10次。

**太空漫步训练**
1.  双手枕在后脑勺，腰腹以上保持不动。
2.  以臀部及大腿为重心。
3.  **吸气**，夹紧PC肌，引导左脚踏出（像走路），停留在半空。
4.  紧接着右脚踏出，与左脚并齐，随后放下。
5.  **呼气**，放松PC肌。
6.  一组做10次。

**仰卧提腿**
1.  双肩放松，双手平放于腰侧，夹住小球。
2.  抬起双腿，悬空。
3.  **吸气**，双腿像钟摆一样左右移动。
4.  **呼气**，放下。
5.  做10次。

**狗鸟式**
1.  双脚跪地，双手撑地（类似四足跪姿）。
2.  伸直左臂和右腿（左右交叉）。
3.  伸直时夹紧PC肌，收回时放松。
4.  左右交叉各做10次。

#### 3. 坐立卷腹训练（改善盆骨后倾）

绝大多数男士存在盆骨后倾的情况。此训练在椅子上进行。

**基础动作**
1.  坐在椅子前三分之一处，坐直，双脚分开。
2.  用头顶找向天空的感觉，双脚向中间，双手外翻放在大腿内侧（靠近骨盆）。
3.  骨盆状态要向上。
4.  **呼气**，手的感觉向后向上挖（骨盆后倾）。
5.  **吸气**，向前翘（骨盆前倾），想象用凳子边去触碰耻骨位置。
6.  每次缓慢移动时，发“哦”音，最后一下“哦”要收紧PC肌。
7.  做3到5组，每组10次。

**坐骨左右抬起**
1.  保持坐姿，双手放在大腿内侧。
2.  身体向左、向右抬起坐骨。
3.  左右各一次为一组。
4.  做3到5组，每组10次。

**360度旋转**
1.  保持坐姿，用骨盆做360度画圈旋转。
2.  转一圈后，最后往上提一下PC肌（盆底肌）。
3.  做3到5组，每组10次。

**坐姿深蹲起落**
1.  双手放在骨盆位置。
2.  做类似浅深蹲的起落训练，臀部离开凳子。
3.  做3到5组，每组10次。可以变化速度，先快后慢。

#### 4. 训练后放松

完成所有动作后，闭上眼睛，进行冥想呼吸，放松肌肉，保持自然呼吸状态。
