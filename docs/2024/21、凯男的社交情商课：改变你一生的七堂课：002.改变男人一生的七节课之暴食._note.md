# 21、凯男的社交情商课：改变你一生的七堂课 002.改变男人一生的七节课之暴食

## 概述
在本节课中，我们将探讨恋爱和情感中常见的“暴食”现象，分析其成因和影响，并提供相应的解决策略。

## 核心概念
**暴食**：在恋爱和情感中，过度追求关系进展，不顾对方感受，导致关系失衡的现象。

### 发力过猛，不留空白
1. **男性化思维**：男性追求极致，追求完美，导致在恋爱中过度投入，不留空白。
2. **初恋影响**：初恋时过度投入，导致后续恋爱中难以自拔。
3. **连环炮式攻势**：过度追求信息交流，不留任何缝隙。

### 盲目的暴食，无理的控诉
1. **展示DHV**：在酒吧夜店等场合过度展示自己，反而效果适得其反。
2. **释放信号**：用一点点信号吸引对方，让对方主动发掘自己。
3. **稳健的作战风格**：观察、了解、测试、收集反馈信息，调整作战方案。

### 深情不受
1. **过度投入**：对爱情有不合理的期待，导致过度投入。
2. **安全感缺失**：缺乏安全感，导致在爱情中过度依赖对方。
3. **自我成长**：爱情不是逃避现实的借口，而是自我成长和个人努力的奋斗。

### 一口吃狗胖子，心太急
1. **改变需要时间**：改变需要付出时间和努力，不是简单的上几节课就能实现的。
2. **科学体系**：通过丰富的任务、训练和共同学习成长，实现真正的改变。

## 总结
本节课中，我们学习了恋爱和情感中的“暴食”现象，分析了其成因和影响，并提供了相应的解决策略。希望这些内容能帮助大家更好地理解和处理恋爱关系。
