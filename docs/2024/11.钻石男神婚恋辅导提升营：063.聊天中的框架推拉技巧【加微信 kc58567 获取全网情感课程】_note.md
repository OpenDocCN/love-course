**核心概念**：**框架 = 个人原则与立场**。男人必须要有自己的框架。发生框架碰撞时，我们也要能守住自己的框架。

你可以去想想，如果当这个女生说“你快点帮我教训他”的时候，这个男生立马就跳过去教训他了，那么女生的心里会有怎么样的反应呢？她会觉得这原本就是你应该做的。你做得好是正常的，你做不好，你就会掉分。

**核心操作**：**推（拒绝/树立框架） -> 间隔时间（情绪冷却） -> 拉（意外满足/奖励）**。

## 课程总结 📝

本节课中，我们一起学习了框架与推拉技巧。

1.  **框架**是个人为人处事的原则，是吸引力的基础。失去框架意味着失去吸引力。
2.  **推拉**是维护自身框架、调节可得性、制造情绪波动的核心技巧。
