# 【李大魔】《极致吸引》：004.二、没有吸引的原因

🎼大家好，欢迎来到第二节课，我是张家华。本节课我们将探讨为何你无法吸引他人。

## 一、情绪控制

**核心概念**：情绪控制能力

**公式**：情绪控制能力 = 情绪认知 + 情绪管理

**内容**：

1. **情绪认知**：了解自己的情绪，包括情绪的来源、类型和强度。
2. **情绪管理**：学会调整情绪，避免被情绪支配，做出理智的决策。

## 二、心理地位差

**核心概念**：心理地位差

**公式**：心理地位差 = 价值感知 + 注意力分配

**内容**：

1. **价值感知**：认识到自己的价值，并尊重他人的价值。
2. **注意力分配**：将注意力集中在自己身上，而不是过度关注他人。

## 三、注意力心理地位差

**核心概念**：注意力心理地位差

**公式**：注意力心理地位差 = 注意力分配 + 价值感知

**内容**：

1. **注意力分配**：将注意力集中在自己身上，而不是过度关注他人。
2. **价值感知**：认识到自己的价值，并尊重他人的价值。

## 四、价值对比

**核心概念**：价值对比

**公式**：价值对比 = 价值感知 + 行为表现

**内容**：

1. **价值感知**：认识到自己的价值，并尊重他人的价值。
2. **行为表现**：通过行为展现自己的价值，避免刻意炫耀或低调。

## 五、总结

**内容**：

1. **行为决定环境**：你的行为决定了你所处的环境，进而影响他人对你的看法。
2. **思维决定行为**：你的思维决定了你的行为，进而影响你的价值感知。
3. **改变从“我是谁”开始**：想要改变自己，首先要改变对自己的认知。

**本节课中我们一起学习了如何提升吸引力，包括情绪控制、心理地位差、注意力心理地位差、价值对比等方面。希望你能将这些知识应用到实际生活中，成为一个更有吸引力的人**。
