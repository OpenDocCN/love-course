# 钱进社 - 成为优质且最具竞争力的【社会人】：68、如何不被“富人思维”坑害

## 概述

在本节课中，我们将探讨如何避免被“富人思维”误导，并学习如何建立正确的认知体系和评判体系。

## 认知体系和评判体系的重要性

**公式**： 认知体系 + 评判体系 = 格局

认知体系和评判体系是构成个人格局的基础。只有当这两个体系稳定且成熟时，个人才能在更高维度上思考和行动。

## 格局的误区

**误区**： 认为格局可以通过简单的学习或阅读提升。

**真相**： 格局是认知体系和评判体系沉淀后的结果，需要长时间的积累和沉淀。

## 提升格局的正确方法

1. **大胆设想，小心求证**： 在设定目标时要有远见，但在行动时要谨慎。
2. **勇于落地**： 将想法付诸实践，并不断调整和优化。
3. **保持警惕**： 对自己的成就保持谦虚，避免自满。
4. **关注未来潜在价值**： 不要只关注眼前的利益，要考虑长远发展。
5. **有可控的现金流**： 确保财务稳定，才能有更多的精力去追求更高目标。
6. **接受不同的认知**： 保持开放的心态，学习新的思维方式。

## 总结

本节课中，我们学习了如何避免被“富人思维”误导，并了解了提升格局的正确方法。只有建立正确的认知体系和评判体系，才能在更高维度上思考和行动，实现人生目标。
