# 《人间清醒后半生》第18节：情绪控制——对方脾气暴躁，情绪失控怎么办？

大家好，我是任亚，欢迎来到我的课堂。这节课的主题是：如何在婚姻中应对情绪不稳定的伴侣。

## 情绪失控的表现

情绪不稳定和失控的表现包括：骂人、口不择言，严重时甚至动手，如砸东西或打人。以下是一些具体表现：

- **骂人**：口不择言，言语攻击。
- **砸东西**：情绪激动时砸毁物品，不伤害人。
- **动手打人**：情绪失控时直接对伴侣动手。

## 选择比努力更重要

在婚姻中，选择一个情绪稳定的伴侣比努力去改变对方更重要。婚前应筛选掉有情绪失控和暴力倾向的伴侣。因为情绪失控是难以改变的，除非对方主动寻求治疗。

## 如何选择情绪稳定的伴侣

- **观察家庭背景**：了解伴侣父母的相处模式，观察伴侣是否在特定情况下失控。
- **避免找理由**：不要为情绪失控找理由，如“他是因为生气才失控”或“他是因为在乎我才失控”。
- **认识到情绪失控的本质**：情绪失控是一种能力问题，而非简单的情绪表达。

## 如何应对情绪失控的伴侣

1. **自我觉察**：观察自己是否是情绪压抑的人，并学习如何让情绪流动。
2. **观察伴侣的情绪来源**：判断伴侣的情绪是否来自自身还是周围环境的影响。
3. **鼓励伴侣学习**：如果伴侣情绪失控，鼓励他参加相关课程学习。
4. **学会不受影响**：当周围人情绪失控时，学会保持冷静，不被影响。

## 总结

本节课中，我们学习了如何应对情绪不稳定的伴侣。选择一个情绪稳定的伴侣、自我觉察、观察伴侣的情绪来源、鼓励伴侣学习以及学会不受影响，都是应对情绪失控伴侣的有效方法。希望这些内容能对大家有所帮助。
