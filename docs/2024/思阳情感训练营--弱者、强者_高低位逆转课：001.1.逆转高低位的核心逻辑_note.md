# 思阳情感训练营--弱者、强者_高低位逆转课：001.1.逆转高低位的核心逻辑

大家好，我是思阳，欢迎收听我的高低位逆转训练营。本节课我们将探讨爱情中的高低位关系，帮助大家正确理解并逆转这一关系。

## 理解高低位关系

**以下是关于高低位关系的核心概念：**

* **关系不平等**：在大多数亲密关系中，双方的位置不可能完全平等，总有一方处于主导地位或优于另一方。
* **高低位产生**：由于付出多少、个人条件、相处后的变化等因素，高低位关系可以改变。
* **误区认知**：误区包括认为高位者付出多、情绪化、物质给予多的一方是低位等。

## 高低位关系的原因

**以下是导致高低位关系的原因：**

1. **自我价值认知不足**：自卑和缺爱导致无法肯定自我价值，容易陷入糟糕的亲密关系。
2. **光环效应**：对对方形成某种印象后，影响对其其他特质的判断，导致自惭形秽。
3. **资源差异和打压**：资源丰富的一方更容易走向高位，资源价值不匹配导致心态变化。

## 逆转高低位的方法

**以下是逆转高低位的方法：**

1. **塑造个人价值**：提升外在、生存价值和情绪价值。
2. **人格独立**：拥有自己的事业、兴趣、朋友圈和精神世界。
3. **保持异性资源**：拓展社交、提升自我，给对方带来紧迫感。
