# 课程05：对视无惧 · 第11课：心法——恐惧与焦虑的终极解决方案：临在状态 🧘

在本节课中，我们将要学习如何通过“临在状态”这一核心心法，从根本上解决对视恐惧、焦虑等负面情绪。临在状态，即“活在当下”，是一种通过自我观察将注意力锚定在此时此刻的方法。我们将探讨其原理，并学习一个快速进入此状态的实用技巧。

## 临在状态：活在当下的本质

上一节我们介绍了应对恐惧的方法框架，本节中我们来看看其背后的核心心法。临在状态，简单来说就是“活在当下”。人所经历的所有痛苦，无论是焦虑、抑郁、强迫、恐惧、担心还是失落，从时间线上看，根源都在于思维要么纠缠于**过去**，要么焦虑于**未来**。

如果你真正处于当下这一刻，你会发现能做的事情只有一件：**观察**。所有事物对你而言都是一幅画面，你只能去感受和观察，除此之外，什么都做不了。

## 思维如何制造痛苦

以下是思维偏离当下、制造痛苦的典型过程：

1.  **纠缠过去**：例如，面对一盘美食，你思量“它的原材料可能不干净”。这盘菜端上来之前发生的事情已是**过去**。你消耗当下的能量去思量过去。
2.  **担忧未来**：接着，你担心“我吃完可能会拉肚子”。这是在担忧**未来**尚未发生的事情。

你的思维只要开始活动，就必然指向过去或未来，因为“当下”是一个极短的时间点。在这个点内，你只能感受和观察。既然在纯粹的当下只能观察，那么焦虑、恐惧等负面情绪便无从生起。

> 注意：这并非否定回顾过去与规划未来的价值。合理的反思能避免重蹈覆辙，合理的规划能增加人生掌控力。问题在于“过分消耗”，这种无意义的能量消耗本身就是痛苦的来源之一。

## 如何进入临在状态：自我观察法

既然临在状态如此有效，如何快速进入呢？最快的方法是**自我观察**，也被称为内观修行。冥想训练中的“观呼吸”便是此法：细致体察呼吸时气流在鼻腔、肺部、腹部的感觉，乃至汗毛随气流晃动的感觉。这种细微的体察，就是为了让你安住于当下。

进入自我观察有一个小窍门：**向自己提出一个问题**。这个问题如同一个提醒的铃声，让你瞬间觉知到自己已陷入纷乱的思维，从而跳出来，回到观察者的位置。

## 自我观察的实践技巧

以下是我个人推荐并经过实践的方法要点：

*   **核心问题**：我常用的问题是“**我在干嘛？**”。这个问题简短有力，易于使用。
*   **提问要诀**：提问时，**语速一定要慢**。例如，缓慢地自问：“我……在……干……嘛？”。这能增强提醒的清晰度和力度。
*   **观察原则**：**只观察，不评价**。这是至关重要的一点。就像观察那盘菜，只感受其颜色、气味、温度。一旦开始评价（如“原料干不干净”、“吃了会怎样”），你就立刻跳出了当下状态，脱离了自我观察。
*   **效果体验**：当你通过提问习惯性地进入自我观察状态，你会站在一个更高的**观察者视角**。此时，所有负面情绪（害怕、担心、焦虑、紧张等）都如同在台下演戏的小丑。无论台上如何闹腾，作为看戏的观察者，这些情绪都无法真正困扰你。

## 理性看待修行与现实

我们总结一下：临在即当下，进入当下最好的方式是通过一个提问，快速切换到高维的**观察者视角**，俯瞰一切。

许多成就者如李连杰、赵文卓都长期进行内观（即临在）训练。赵文卓曾表示，即使在接受采访时，他也能保持内观状态，清晰看待事物而不被其困扰。

然而，达到这种稳定的境界通常需要长时间的修行。反观他们年轻时，同样面临事业、生活与各种欲望的纠缠，那时很难静心修行并取得成果。这解释了为何许多人在事业巅峰后选择出家——他们已从欲望中解脱。

但我们普通人不同，我们需要面对柴米油盐、车房、子女教育、养老等现实问题与欲望。在这种状态下，很难“轻松”地达到深度活在当下的状态。即便是修行得道的弘一法师，也是在中年**抛弃家庭、遁入空门**，避开世俗欲望与纠缠后，才得以潜心修行达成大成。

我并非打击大家实践的积极性，练习进入临在状态绝对有意义。但我们必须**理性看待自己能达到的高度及其实际效果**。本课程更像一个“教程”，提供方法让我们知道如何**快速进入**临在状态以缓解痛苦。

如果你渴望达到“心如止水、不被痛苦欲望萦绕”的境界，这本身可能是一种**妄念**。真正的放下，如同弘一法师的选择，当你能真正放下许多时，修行成果自是水到渠成。

本节课中我们一起学习了恐惧与焦虑的终极心法——**临在状态**。我们明白了痛苦源于思维对过去未来的执着，而通过**自我观察**（尤其是借助“我在干嘛？”这样的慢速提问），我们可以快速将注意力拉回当下，切换到**观察者视角**，从而有效剥离负面情绪。请记住核心原则：**只观察，不评价**。同时，我们应理性看待此方法的作用，将其作为应对当下情绪困扰的实用工具，在现实生活中加以练习和运用。
