# 老白《搞钱课》：11、抖音文案开始前5秒

## 概述
在本节课中，我们将学习如何选择抖音视频的文案，特别是关注视频开始前5秒的文案，因为这是决定观众是否继续观看的关键。

## 文案选择原则

以下是选择抖音文案时需要遵循的原则：

### 1. 播放量与文案选择
- **播放量是关键**：文案的选择应以提高播放量为目标。
- **高播放量案例**：分析播放量高的视频文案，如“对我很好，应不应该在一起我的建议”和“男生会嫌弃女生家里穷或者拖后腿吗？”等。

### 2. 5秒完播率
- **5秒完播率**：5秒内的完播率是衡量视频吸引力的关键指标。
- **完播率标准**：40%以上的完播率算合格，40%以下则不合格。

### 3. 文案开头形式
- **三种开头形式**：
  1. **吊人胃口**：通过提出问题或悬念吸引观众。
  2. **戳痛点**：直接触及观众的心理痛点。
  3. **平淡叙述**：直接陈述事实或观点。

### 4. 痛点与共鸣
- **痛点**：针对观众普遍关心的问题或困扰。
- **共鸣**：与观众产生情感共鸣，引起共鸣。

### 5. 新颖题材
- **新颖题材**：选择新颖的题材，避免与常见题材重复。

## 文案案例分析

以下是几个文案案例的分析：

- **案例1**：“对我很好，应不应该在一起我的建议”：戳痛点，引发观众共鸣。
- **案例2**：“男生会嫌弃女生家里穷或者拖后腿吗？”：戳痛点，引发观众好奇。
- **案例3**：“到底什么时候姐妹们情侣到底什么时候见家长”：戳痛点，引发观众共鸣。
- **案例4**：“导致焦强烈的占有欲会导致焦虑，焦虑会导致患得患失”：平淡叙述，缺乏吸引力。

## 总结
本节课中，我们学习了如何选择抖音视频的文案，特别是关注视频开始前5秒的文案。通过分析高播放量视频的文案，我们了解了痛点、共鸣和新颖题材的重要性。在今后的创作中，我们要注重文案的质量，以提高视频的播放量和完播率。
