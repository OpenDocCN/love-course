## 富二代的四大类别

富二代根据原生家庭的模型，可以分为以下四个核心类别：

1. **暖男款**：这类富二代家庭中，母亲是全职太太。他们提供的核心价值是极高的生育价值和生存价值。所在行业通常是商业地产、餐饮、工业制造等，岗位多为总裁、高管、连锁企业创始人等。他们的核心价值观是奋斗加影响或影响加精彩。

2. **奶狗款**：这类富二代家庭中，母亲资源大于父亲。他们的核心价值是高声誉和高观赏。所在行业通常是大型企业、集团公司、家族企业等，岗位多为稳定的事业单位编制、技术人员或管培生。他们的核心价值观是安全加精彩或传统加精彩。

3. **冷漠款**：这类富二代家庭中，长时间与父母分居，渴望得到别人的爱，但不敢投入和付出。所在行业通常是金融、游戏、计算机、投行、律所等，岗位多为律师、医生、职业经理人等。他们的核心价值观是奋斗、独立和精彩。

4. **海王款**：这类富二代家庭中，母亲是小三上位或二婚。他们的核心价值是高观赏和高情绪。所在行业通常是文娱、学术、咨询、新兴产业等，岗位多为网红、艺术家、自由职业者等。他们的核心价值观是精彩加博爱或博爱加仁慈。

## 与不同类型富二代的相处策略

### 暖男款

- **初期**：展示自己的优秀，让对方眼前一亮。
- **中期**：展现自己的柔弱，让对方感受到你的依赖。
- **后期**：认可对方的责任心，让他把更多的时间和精力放在你和家庭上。

### 奶狗款

- **初期**：展现自己柔弱的一面，让对方感受到你的依赖。
- **中期**：展现自己的成长和变化，让对方看到你的进步。
- **后期**：鼓励对方找寻自己的生活和爱好，让他看到自己的成长。

### 冷漠款

- **初期**：展现出自己独立的一面，不要表现出依赖。
- **中期**：逐步展现对对方的依赖，让对方感受到你的关心。
- **后期**：与对方探讨分享，鼓励对方接受亲密关系。

### 海王款

- **初期**：展现自己丰富的生活，让对方感受到你的魅力。
- **中期**：体现一些依赖，偶尔有点小幼稚。
