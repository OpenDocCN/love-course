# 19、技巧软功：4、韩式风情 🌸

在本节课中，我们将学习韩式风情的玩法。韩式风情的关键在于掌握S型刮痧技巧，以下是一步一步的教程。

## 概述

韩式风情是一种通过S型刮痧来挑逗和吸引对方的方式。它不仅能够增加亲密接触的乐趣，还能提升双方的参与感。

## S型刮痧技巧

1. **了解S型刮痧**：S型刮痧是指用刮痧板或手指在皮肤上画出S型线条，以产生吸引力和挑逗感。
2. **实践S型刮痧**：用舌头或手指轻轻吸引对方的皮肤，像种草莓一样，然后画出S型线条。

```markdown
# S型刮痧公式
S型刮痧 = 吸引皮肤 + 画S型线条
```

## 韩式风情流程

1. **耳部挑逗**：从耳部开始，轻轻挑逗对方的耳头，然后沿着耳垂向下至脖子。
2. **颈椎S型刮痧**：在颈椎突出处，用S型刮痧技巧吸引并刮痧。
3. **排骨S型刮痧**：在排骨两侧，分别用S型刮痧技巧。

```markdown
# 韩式风情流程
韩式风情流程 = 耳部挑逗 + 颈椎S型刮痧 + 排骨S型刮痧
```
