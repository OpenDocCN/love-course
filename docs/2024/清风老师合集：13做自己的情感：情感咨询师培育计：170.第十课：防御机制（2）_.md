# 清风老师合集：13做自己的情感：情感咨询师培育计：170.第十课：防御机制（2）_

在本节课中，我们将要学习防御机制在咨询实践中的具体应用。我们将回顾防御机制的核心概念，并学习如何识别、分析及应对来访者的防御行为，从而更有效地推进咨询过程。

上一节我们介绍了防御机制的基础知识。本节中我们重点来看看防御机制在咨询当中的应用。

防御机制是我们在应对自认为危机或危险的情况下，进行自我防御的一种自我保护机制。它是精神分析理论中的一个重要概念。在人格结构当中，它属于自我（ego）的功能。当自我觉察到来自本我的冲突时，就会以预期的方式体验到一定的焦虑，并尝试用一定的策略去阻止它。这个过程就是防御，或称为自我的防御。防御主要是针对能引起个体焦虑的欲望或攻击性。一般来说，防御是在潜意识里进行的，因此个体并不会意识到它在发挥作用。

通过上一次的学习，我们对防御机制的特点有了一定的认识。以下是老师总结的五个要点：

首先，防御机制不是蓄意使用的。它们是无意识的或部分无意识的。就像我们面对的很多来访者，他们可能并没有说出最真实的答案，但很多并不是故意这样，而是无意识的保护自己的一种手段。例如，狐狸吃不到葡萄而说葡萄是酸的，这在某种程度上，是通过合理化的想象来缓解自己吃不到的痛苦。

第二，防御机制是依靠自尊或自我美化来保护自我免受伤害。从性质上可以分为积极与消极两大类。积极的防御机制让我们得到好的体验，变得更加自信有力。例如，我们做一件事情失败了，告诉自己“下一次一定会成功”，这就是积极的。如果我们想到“下一次也可能失败”，并陷入自我焦虑当中，这种防御就是消极的。

第三，防御机制有自我欺骗的性质。也就是说，它会掩盖或伪装自我真正的动机，甚至否认可能引起的焦虑、冲突、行为或记忆。防御机制本身是一种阻断心理过程，使自我免受焦虑之苦的保护方法。所以，我们在分析来访者的防御机制时，很有可能遇到阻抗。也就是来访者并不想承认自己确实有这样的问题。有时并不是他不想承认，而是连他自己也无法意识到。因此，在讨论防御机制时，我们一定要理解，不要轻易地去给对方下定义，而是能够识别对方应对的防御机制，帮助我们更好地理解和认识对方。

第四，防御机制本身并不是病态的。相反，在维持正常心理健康方面有重要的作用。但是，它的过度使用可能会导致病理性的后果。防御机制主要是针对能引起个体焦虑和攻击性的情绪。任何防御机制本身的目的都是好的，但很多时候，有一些防御机制往往掩盖了事实的真相。

第五，防御机制可以单一使用，也可以重复使用。大多数时候，我们的防御并不是单一进行的，而是几种防御同时进行。例如，在咨询中遇见一对吵架的情侣，起因是男友玩游戏输了，然后本该属于他的工作没有做。在女友质问时，男生把玩游戏失败的情绪**投射**到质问中，并对质问本身**回避**，不做任何解答。防御机制本身都是站在个体有利的角度进行的。

对于我们来说，认识到这些防御机制及其特点是第一步。更重要的是要把这种技术应用到咨询过程当中。而这些防御机制可能会让我们的咨询受到一定的阻碍，无法顺利推进。

那么你也许会问：防御机制有好有坏，我该怎么做呢？

在咨询当中，我们需要建立咨询关系。很多时候我们还要建立更深入的联盟关系。这种关系是我们作为咨询师，在咨询或日常聊天中有意识地去建立的。如果没有这种咨询联盟的建立和分析，我们会有一种无从下手的感觉。

给你们讲一个案例。我的一个来访者安安，她和我说了她和男朋友沟通的情况。每当她想和男友谈论一些问题，或者想说心里话的时候，男友的一句话就让安安不知道下一句该怎么说了。这种情况让安安都不敢和男友继续聊天了。安安找到我问：“清风老师，我有时候就是想知道他心里想什么，或者他是什么看法，但他总是不正面说，顾左右而言他。每次都让我很抓狂。平时聊天这样没什么，但有些时候我想解决我们之间的问题，他竟然也这样逃避，我该怎么办呢？”

我问安安：“那你想问过他为什么这样吗？是不是他有什么顾虑呢？”安安说：“老师，我问了，但是他也没有说，我就没办法了。”

接着我告诉她：“安安，对方的回避和闪躲只是一种自我保护的方式，或者他认为这样对于他来说并不好。这个问题是让他非常为难的，他之所以回避你的问题，其实某种程度上也是为了保护这段感情。当然了，这更多的是他自我保护的方式。这就像是善意的谎言。要解决你们之间的这种问题，你要在你们的生活当中进行观察，看看是什么原因造成了他的回避和躲闪。更重要的是，你们还需要通过平和的沟通来解决。”

安安听了有一些迷糊。她问我说：“那清风老师，我该怎么做呢？”

我说：“如果在观察的时候感觉到了他的回避，你也感觉到了他的为难。那么在和这样交谈的过程当中，你可以这样问他：‘亲爱的，你为什么不正面回答我的问题呢？你是感觉到不舒服吗？如果你不相信自己可以处理的很好，那我们可以放一放。’问过之后看看他的反应。”

这就是老师今天要教给你的方法：**探索 + 面质 + 澄清**。

所谓**探索**，就是我们作为咨询师对于防御机制有所理解和认识，找到一些防御来诠释对方的行为。如果是长期生活在一起，我们还可以通过观察来实现。当然，如果是在咨询或聊天当中，我们可以通过一些问题来和我们的来访者进行探索。同样的，我们也可以把这个方法交给我们的来访者，让他在有条件的情况下和自己的伴侣进行一些性质上的探索。

**面质**就是当面质询的意思。或者说，把这个问题放到台面上来和来访者进行深入或充分的探讨。在这个过程当中帮助他认识到这个问题。在前面安安的例子当中，我们就把对方的回避情绪拿来放在台面上谈。对方回避她，有可能是因为没有信心处理好这段关系的焦虑导致的。

最后就是**澄清**。有时我们没有理解清楚对方的意思，可以当面说明，让对方去做出解释，不用去猜测对方的意图。在亲密关系当中，澄清也是极为重要的。澄清的语句往往是这样的：“我刚刚理解你的意思是说……，你觉得是这样吗？”例如，安安可以跟对方去做澄清：“你不想面对我，是因为你感觉到面对我很难堪、没有自信，是这样吗？”这就是一个澄清的动作，能够帮助两个人的关系进行非常好的修复。

接着，老师要在刚刚教你的技术当中进行延伸。我们再通过探索、面质、澄清的方式寻找到主要的问题之后，就需要对这些问题背后的防御进行解释。

曾经有一个我的来访者问我说：“老师，有时候我看到了他的问题，但我不知道该怎么样理解他。我看起来感觉他好像很悲观，但我又不知道怎么样安慰和帮助他。”其实很多人都会有这样的感觉，明明看到了对方问题的表现，但是就是不知道怎么样给他深入的分析，不知道该怎么样入手。或者说，作为一名新手的咨询师，你也不知道该怎么样入手，给你的来访者进行分析。

那么接下来，老师给你准备了诠释的几个方面。从这几个方面都可以帮助你建立更好的关系。

首先，是**动力学的诠释**。心理动力学是由我们所熟知的弗洛伊德创立的。他认为行为是由强大的内部力量驱使或激发的，是从继承来的本能或生物驱动力当中产生的，并且试图解决个人和社会之间发生的冲突。从心理动力学来看，心理问题就是个体发动、指引或维持某种活动和事业的心理原因。弗洛伊德将本能视为人类的基本心理动力，也就是我们所说的**力比多**（libido）。本能是来自于身体的内部刺激，它驱使人通过活动来满足由于内部刺激所产生的心理或生理的要求，宣泄和消除由于刺激所带来的紧张、痛苦和焦虑。

在咨询的时候，我们向别人解释他们所没有看见的防御，然后表明哪些情感被真实的遮掩住了。那么这个时候，我们咨询师就可以做一个解释性的动作：他们为什么会有这样的防御机制？产生这样防御机制的动力是什么？例如，在我们刚刚举的安安的男友的例子当中，男友采用了回避的防御机制来回避两个人之间可能产生的冲突。对于回避型依恋关系来说，很多的回避型依恋都会用回避来保存一段亲密关系。我们就可以解释给安安听，告诉她，男友并不是讨厌你，并不是不喜欢你，而是在很大程度上，希望保存你们两个人的关系。通过这样的解释，能够使两个人的关系得到保存。那么通过这样的解释，在跟安安咨询的过程当中，我们就帮她完成了认知上面的整合。

人的内驱力可以分为两个大类：由饥饿等生理需要而产生的内驱力称为**第一内驱力**，又可以称为基本的、原始的或者低级的内驱力。由责任感和后天所形成的社会性需要所产生的内驱力，可以称为**第二内驱力**，或者称为社会的或者高级的内驱力。一般来说，高级的内驱力对低级的内驱力起调节作用。我们的每个行为背后都有一个内驱力，也就是有一个行为的动力。而驱使我们行为的这个动力，可能就是解决问题的真正关键。

第二个方面，**起源性诠释**。什么是起源性诠释？就是我们很多行为可能都是过往的一些经历替换到了现在。所以我们要找到最初的那个起源，抓住问题产生的根本。我有一个来访者，25岁才开始初恋。初恋的时候，全心全意地对男朋友好，自己有100块钱，90块钱都会花在男朋友身上。但是，男朋友却在这种过度的好当中习惯了，最后出轨并抛弃了她。这段经历给她造成了很大的影响。后来和男友相处的过程当中，对方稍微有点不好，她就会觉得对方会不会也要抛弃自己。找到我咨询的时候，她就说：“老师，基本上我现在谈恋爱都很短，两三个月一次。这次这个男生我真的挺喜欢他，不想离开他。但是昨天因为他答应我陪我逛街，没有陪我，我就和他吵了一架。你说他是不是爱上别人了呀？”这个姑娘因为男友没有陪她逛街，就想到男友是不是爱上别人，要抛弃她。后来谈到了她的初恋之后，我通过解释的方式帮她进行了一个整合。我告诉她说：“其实啊，你这样想，很大程度上是因为你第一次恋爱的经历，对初恋，你害怕他像初恋那样抛弃你，即使你对他比对自己都要好。初恋抛弃他的那种恐惧，被他带到了以后的生活当中，让他把现在的行为和过去所联系在了一起。那段经历的痛苦，就是她问题的起源，也是她进行这种防御的真实面目。”

第三个方面，**移情诠释**。移情是精神分析的一个用语，是指把对父母或对过去生活中某个重要人物的情感、态度或属性转移到了别人身上，并且相应地做出反应的过程。小鱼向我咨询的时候跟我说：“清风老师，我现在和他相处的过程当中，我感觉到他一点都不在乎我。但是我问他的时候呢，又能感觉到他很认真。老师，你说我是不是有心理疾病啊，我很多疑呀。”和小鱼聊到小时候经历的时候，我问小鱼：“小时候你和自己的原生家庭和家人是怎么样相处的呢？”小鱼说：“老师，我和母亲相处很好，但是和我父亲，很多时候他都不注意我。有时候我话说了好几遍，他都没听到。后来我不想和他说话了。”在和男朋友相处的过程当中，小鱼把父亲对于自己忽略的情绪转移到了男友身上，以为男友也和父亲一样不会在意她，这就是一种移情。

在咨询过程当中，发生移情时，咨询师就成了求助者某种情绪体验的替代对象。所以不要担心你的来访者频繁地向你发出一些求救的信息，或者把一些负面情绪转移到你的身上。包括这两天在社群当中，我也看到同学在提问说：“老师，有些人我真的很不讲理。”那么这就说明他在试图跟你建立一种移情关系，他把对某个人的需要转移到了你的身上。在这种情况下，作为咨询师，我们可能会说：“好，你是认为我会和你父亲一样，不注意你。如果在你说话的过程当中，我和你互动，让你重复说一些重点的部分，你会觉得怎么样呢？”在咨询的这种防御过程当中，我们知道了移情的原因，这些方面就能够跟我们的来访者进行更好的互动。而我们平常和别人的聊天当中也会出现移情。移情有直接和间接的。前者就是一些来访者，会直接地表达自己的感受，例如：“哎呀，老师我和你聊天特别愉快，让我想起了谁谁谁。”那么有的人会比较间接地表达自己的感觉，例如：“老师我觉得你态度很好，我觉得很宽松。”除了间接和直接，还有正面和负面。前面提到了负面，那么不用担心出现了负面的情绪，这正说明你的咨询在起作用。

第四个方面称之为**重现**。重现是记忆的一个重要过程。以前记过的事情，不在当下，在其他事物的刺激影响下，把它重新呈现出来，叫做重现。重现是我们对于一个事物的感受再次出现的感觉。重现需要思考在早期的生命经历当中可能发生过什么，造成了当前的困境。过往的经历对于他们的影响很大，而防御机制会让他们忽略这种记忆，因此，在生命或生活当中造成很大的影响。还有，我们在重现某些记忆的时候会有一点不准确。因此，我们的来访者很可能会在这个时候产生其他的防御机制。比如说过分热心地重视，让他们符合我们的意见；让他们违背自己内心真实的想法，符合咨询师的权威，等等。

上面是老师给你介绍的四个方面。那么如果你还想对于防御机制有更深入的了解，上一次我给你推荐了一本书，这本书叫做《心灵的面具》。

在这里，我要跟大家界定一下情感咨询和我们心理咨询的一些区别。情感咨询中，咨询师扮演的身份更像是一个引路人，像是来访者黑暗当中的火把。它的起作用往往在于认知的更新。也就是我们通过解释、面质这样的技术，帮助我们的来访者了解到他们所存在的问题，帮他们认识自己身上所存在的一些特质以及特点。那么心理咨询呢，更像是一面镜子，让他们在跟咨询师的关系当中，把他们潜藏的一些动力呈现出来。在呈现的过程当中，通过咨询师的互动来给他们新的体验，从而治愈他们。

本节课中，我们一起学习了防御机制在咨询中的应用。我们回顾了防御机制的五个特点，学习了通过“探索、面质、澄清”三步法来应对防御，并深入探讨了动力学诠释、起源性诠释、移情诠释和重现这四种诠释角度。理解并善用这些知识，能帮助我们更好地理解来访者，建立咨询联盟，并有效地推进咨询进程。
