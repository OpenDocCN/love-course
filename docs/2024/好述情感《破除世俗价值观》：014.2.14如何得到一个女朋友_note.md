# 好述情感《破除世俗价值观》：014.2.14如何得到一个女朋友_note

在本节课中，我们将要学习如何得到一个女朋友。我们将探讨一个核心方法，并解释其背后的逻辑，帮助初学者理解两性关系中的关键步骤。

## 核心方法：关系建立的捷径

上一节我们介绍了关系建立的基础，本节中我们来看看具体如何操作。想要与一个女孩发展关系，无论是成为情人还是男女朋友，都需要先与她发生性关系。

最快速度发生性关系是最有效的方法。

## 成功公式解析

基于上述方法，我们可以推导出一个公式。这个公式描述了在发生关系后，如何将关系稳定发展为男女朋友。

公式很简单：**发生关系后，继续保持情人特质，并且自愿成为供养者**。

这种情况意味着，在保持情人特质的基础上，主动承担供养者的角色。具体行为包括买礼物、送东西、请客吃饭、带她看电影等，表达出愿意为她提供资源的感觉。这是得到一个女朋友最快的方法。

我们可以用一个更简单的公式来概括：
**女朋友 = 性爱 + 牛排 (即：发生关系 + 提供资源/约会)**

## 方法背后的逻辑

我们回来深入探讨这一点，再重复一遍核心逻辑。

女孩与你发生性关系是她最大程度的投资。当她冒了这么大的风险后，意味着她已经愿意为你释放卵子。她已认定这个男人足够优秀，并认为与他生下的孩子也会同样优秀。此时，你想将她变成自己的女朋友。

绝大部分女孩在发生关系后，都会渴望与你在一起，并会考虑这个男人能否继续为她提供资源。你无需先提供资源就能与她发生关系。

但是，如果你希望与她好好相处，成为男女朋友乃至将来结婚，就必须提供资源。此时，你真正的资源，例如金钱，才开始慢慢起作用。

然而，金钱或长相等被社会看重的肤浅价值观，对于最初吸引女孩过来是完全没有用的。它们只是在女孩已经喜欢你、愿意与你在一起后，用于供养她时的要素。

## 关于维系关系的提醒

供养行为本身并不会留住女孩。这一点需要明确。

大家可能会困惑一件事：女孩一开始和我在一起并发生了关系，但后来又被别人拐跑了，该怎么办？

一旦你开始担心女孩会被别人拐跑，就会产生不安全感、嫉妒心，害怕她会离开，并失去自信心。此时，女孩反而会因此离开。当你怀疑女孩是否出轨时，通常她就会开始出轨。当你不怀疑她时，即使她有出轨的机会，也很难去实施。

因为她能感觉到你强大的性吸引力和男性魅力，而这一点与金钱无关。她能感觉到的这种吸引力，是你男性特质的一部分，是你最终释放出的男性魅力。

## 实践步骤与关键点

那么，如何得到一个女朋友呢？先发生性关系，然后再陪她吃饭、逛街、看电影即可。

通常来说，想要得到一个女朋友，仅仅发生一次性关系是不够的。你至少需要发生第二次、第三次性关系。这是非常关键的。

连续发生两三次性关系后，你还需要与她调节到一种很好的相处感觉上。

以下是需要达成的状态：
*   不论在床上还是床下，都要有很好的相处感觉。

这种状态才有利于你慢慢获得一个女朋友。
