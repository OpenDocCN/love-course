# 高情商聊天话术 社恐变社牛：19.12浪漫就是仪式感

## 概述
在本节课中，我们将学习如何通过高情商的聊天技巧，从社交恐惧者转变为社交达人，并深入探讨浪漫的本质——仪式感。

## 成就介绍
**成就**是指你过去获得的奖牌、职位、财富等。在与人交流时，展示你的成就可以增加对方对你的好感。

**公式**： 成就 = 职位 + 财富 + 荣誉

## 自信的来源
自信源于熟练度和内心强大。熟练度越高，你的实力越强，自信也就越强。

**公式**： 自信 = 熟练度 + 内心强大

## 不要在女人面前说负面话
永远不要在女人面前说自己的负面话和负面情绪，这会让对方觉得你是一个 loser。

**公式**： 负面话 = 失败 + 不自信 + 没有魅力

## 只顾自己
在社会上，只顾自己，先把顾己把自己顾好，才是最高明的境界。

**公式**： 最高境界 = 只顾自己 + 不受他人影响

## 讲真话
讲真话是最高明的，通过欺骗讲假话迎来的客户会阻碍你真正做强做大。

**公式**： 最高明 = 讲真话 + 不欺骗

## 永远不要受挫
永远不要觉得妹子拒绝了你或伤害了你，这会降低你的男性魅力。

**公式**： 受挫 = 失败 + 不自信 + 没有魅力

## 女人永远不想当荡妇
永远要告诉女人，她很纯洁单纯、可爱、可爱。哪怕对方是妓女，也要这样讲。

**公式**： 荡妇 = 不纯洁 + 不单纯 + 不可爱

## 以退为进
永远要表示你是一个没有目的的人，你不会要求她做什么，你不会强迫她做什么，永远是以退为进。

**公式**： 以退为进 = 不给对方压力 + 不让对方焦虑

## 肢体接触
男女交往的过程是找一切机会进行肢体接触的过程。

**公式**： 肢体接触 = 亲密关系 + 情感升温

## 多转换场地
约会时，一定要去多个地方约会，不能在一个地方约会，不能约老是约到一个地方。

**公式**： 多转换场地 = 熟悉感 + 情感升温

## 仪式感
浪漫就是仪式感，制造仪式感，对方就会觉得浪漫。

**公式**： 仪式感 = 浪漫 + 惊喜 + 感动

## 总结
本节课中，我们学习了如何通过高情商的聊天技巧，从社交恐惧者转变为社交达人，并深入探讨了浪漫的本质——仪式感。希望这些技巧能够帮助你更好地与人交往，找到属于你的浪漫。
