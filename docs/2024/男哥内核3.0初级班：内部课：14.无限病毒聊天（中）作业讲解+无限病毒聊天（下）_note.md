## 三短一长

**概念**：三短一长是指交流中，短句和长句的比例。

**实战运用**：

1. 提供话题价值，让对方参与。
2. 当对方参与度高时，减少字数，增加参与度。
3. 当对方参与度不高时，增加字数，激发对方参与。
4. 通过“三短一长”的节奏，引导对方聊天。
