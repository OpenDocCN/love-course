## 个案演示过程
1. **第一步：明确目标**
   - 不一开始就问案主要解决什么问题。
   - 记住，这种问题已经发生在案主身上很长时间了。
   - 为了能够有一个新的东西可以进来，第一步也需要安顿下来，不要陷在头脑的挣扎里。

2. **第二步：明确意图**
   - 清晰按住个案里，不是把目光只是放在那些他不要的东西，而是他想要的是什么。
   - 在这个过程中，我们并不是简单地用语言说我想要温暖，温暖来自这里。

3. **第三步：解决问题**
   - 如果这个意图是共赢的，我们能够感受到他想要的，那是什么障碍了他。
   - 问题并不是一个问题，问题里面包含着很多正向的动机。
   - 把问题也邀请进来，但我们邀请问题的方式并不是和你去分析问题。

4. **第四步：改变关系**
   - 给问题一个安身之地，给问题一个家，给问题一件隐形美丽的披风。
   - 改变和问题的关系，改变保持他的方式。

5. **第五步：进入家庭动力系统**
   - 和孩子之间分清楚，我是大的，你是小的，生命的河流经由我传给你。
