# 老白《约会达人》：21、喝酒小游戏--骰子

## 概述
在本节课中，我们将学习一种在酒吧常见的喝酒游戏——骰子游戏。这个游戏简单易学，适合在约会时与朋友一起玩。

## 游戏规则
以下是骰子游戏的规则：

- **骰子**：每个玩家使用两个骰子，每个骰子有5颗骰子。
- **万能牌**：骰子中的“1”是万能牌，可以代表任何数字。
- **单闪为0**：顺子（如1256）的骰子计为0。
- **起叫**：两人开始喊数，例如喊“三个3”，下一个玩家可以喊“四个3”或更高的数。
- **喊数**：玩家根据手中的骰子喊出数字，数字由骰子上的点数组成，1可以是任何数。
- **开牌**：当玩家认为对手的喊数不可能时，可以选择开牌。

## 游戏示例
以下是游戏的一个示例：

1. 玩家A喊“三个3”。
2. 玩家B喊“四个3”。
3. 玩家A喊“五个3”。
4. 玩家B喊“六个3”。
5. 玩家A喊“七个3”。
6. 玩家B喊“八个3”。
7. 玩家A选择开牌。

## 游戏技巧
- **观察对手**：注意对手的喊数和骰子，判断其可能持有的数字。
- **合理喊数**：根据手中的骰子合理喊数，避免喊出不可能的数字。
- **开牌时机**：当认为对手的喊数不可能时，选择开牌。

本节课中，我们一起学习了骰子游戏的规则和技巧。希望你在约会时能够运用这些知识，与朋友一起享受游戏的乐趣。
