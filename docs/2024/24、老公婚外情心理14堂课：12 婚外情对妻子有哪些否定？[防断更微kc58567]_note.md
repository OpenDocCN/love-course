# 24、老公婚外情心理14堂课：12 婚外情对妻子有哪些否定？

🎼大家好，我是朱生勇，婚姻管理师，手把手教你管理婚姻。今天来跟大家谈的是老公背叛对于妻子的否定。

## 一、婚外情对妻子的否定

### 1. 对过去的否定

- 老公的背叛会让妻子怀疑过去是否有真正的爱情和婚姻价值。
- **公式**：过去爱情 = 背叛行为 + 真实承诺

### 2. 对现状的否定

- 妻子可能会因为外貌、身材等因素产生自卑感，认为自己不再有魅力。
- **代码**：自信 = 外貌 + 身材 + 自我认知

### 3. 对未来的否定

- 妻子会怀疑婚姻的未来，认为婚姻已经没有意义。
- **公式**：婚姻未来 = 背叛行为 + 信任重建

## 二、对男人的否定

- 妻子会对背叛自己的男人产生厌恶、愤怒等负面情绪。
- **代码**：男人形象 = 背叛行为 + 道德责任

## 三、总结

面对老公的背叛，妻子会产生对过去、现状和未来的否定，以及对男人的否定。为了经营好婚姻，夫妻双方需要共同努力，重建信任，并始终将忠诚作为核心经营资产。
