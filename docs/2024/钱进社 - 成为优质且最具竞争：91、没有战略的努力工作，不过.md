# 钱进社 - 成为优质且最具竞争：91、没有战略的努力工作，不过

在本节课中，我们将要学习为什么单纯的努力工作在职场中往往不足以带来晋升和成功。我们将探讨“战略”与“战术”的区别，并学习如何通过构建职业蓝图和搭建人脉，将努力转化为实际成果。

---

## 课程概述

在职业生涯中，我们常常发现一个现象：并非所有有能力的人都能获得晋升或更多机会。现实情况是，努力工作只是基础，而真正的突破往往依赖于清晰的战略规划。本节课将解析这一现象背后的逻辑，并提供一套可操作的策略。

---

## 战略与努力的关系

上一节我们概述了努力与结果之间的脱节现象，本节中我们来看看其根本原因。

努力工作是在职业生涯中立足的基础。然而，这个基础的价值高度依赖于你所处的公司体系、平台环境和工作模式。

*   **在小团队中**，努力容易被看见。团队规模小，每个人的贡献和问题都显而易见，老板能直接感受到。
*   **在中型或大型团队中**，情况则不同。架构复杂，人员众多，老板很难具体感知到每个人的努力程度。尤其是在一些体系稳固（如国企、外企）的大型组织中，“努力”这个词对个人职业发展的定位变得非常模糊。

因此，仅仅做好眼下的工作，在中大型团队中并非核心竞争力。它只是一个必要但不充分的条件。

---

## 什么是战略方针？

理解了努力作用的局限性后，我们需要认识什么是真正的“战略方针”。

战略方针是为自己晋升之路所做的**长线谋划**。它与即时生效的“战术”（如说话技巧、行为方式）不同。战略是方向，战术是达成战略的具体步骤。**没有战略的战术，很可能是无效的。**

战略的核心在于建立你的**外部价值**。在工作环境中，最大的悲哀不是被人利用，而是**没有人想要利用你**。这意味着你缺乏被他人需要的价值。如果你的价值仅限于完成手头工作，而无法被更高阶的领导或同事“利用”，那么你在晋升名单上就容易被忽略。

许多人能得到认可和鼓励，却得不到升职，正是因为他们的价值未被纳入更高层面的“利用”链条中。顾好眼下工作是战略中的一颗棋子，但绝非整盘棋局。

---

## 如何构建你的职业战略？

明确了战略的重要性，接下来我们学习构建个人职业战略的具体顺序和方法。

正确的顺序应该是：先规划，再行动。以下是为自己构建职业蓝图的步骤：

1.  **向上看：识别关键决策者**
    *   找出能直接影响你晋升的领导。
    *   分析他的性格、喜好以及他身边的“红人”具有何种处事风格。

2.  **平行看：组建协同同盟**
    *   在平级同事中，寻找能与你高效协同、提高整体效率的伙伴，结成同盟。

3.  **合理分配：置换资源，为自己减负**
    *   识别谁能帮你处理一些常规杂事。
    *   用你拥有的、对他而言稀缺的资源进行置换。例如，你可以利用向领导汇报的机会，**在背后**具体、真诚地赞扬这位同事的贡献。
    *   代码示例（资源置换逻辑）：
        ```python
        # 你拥有的资源：接触领导的机会（曝光度）
        your_resource = "exposure_to_leader"
        # 同事需要的资源：在领导面前被认可
        colleague_need = "recognition_from_leader"
        # 同事拥有的资源：时间和精力处理杂务
        colleague_resource = "time_for_chores"

        # 置换条件：如果你的资源正好是对方稀缺的
        if your_resource == "scarce_for_colleague":
            # 进行置换：你获得时间，同事获得认可
            gain_time_for_strategy(your_resource)
            colleague_gain_recognition(colleague_need)
        ```
    *   这种“背后赞美”不仅能帮你置换出时间，还能在领导面前树立你格局大、善于团队协作的形象，是一种双赢。

**关键点**：这不是欺负同事，而是**合理的资源置换与分工**。目的是为你自己减负，腾出时间和精力去聚焦核心战略部署，而不是被杂事淹没。

## 心态调整与长期主义

掌握了具体步骤后，心态的调整同样至关重要，尤其对于职场女性。

在自身不够强大时，首要任务是站稳脚跟，而不是空谈情怀或“义气”。**先让自己成为“大侠”，才有能力回馈帮助过你的人**。合理的资源整合与分配是聪明人的组局方式，无关道德高低。

许多男性更善于此道，因为他们目标直接，脸皮较“厚”。而部分女性可能因脸皮薄、想证明“不比男人差”而拒绝求助或合作，导致自己成为最累却最不被看见的“冤大头”。

**你需要的是骨子里的要强和清晰，而非表面的强硬**。匀出时间搭建人脉是战略的一部分，这绝非偷懒。真诚是基础，别人能感受到你是否在真心合作。

## 战略等待与盲目等待的区别

最后，我们要区分积极的战略准备与消极的原地踏步。

高层领导的核心工作之一是**物色能为公司带来更大发展和可能性的人才**。当你决心构建职业蓝图时，就必须提前谋划，这是一个需要**时间沉淀**的长期过程。

*   **战略等待**：在认知清晰、人脉搭建好、能力准备好的基础上，主动观察平台变化，**等待“天时地利”的时机出现**（如对手离职、业务调整）。这是积极的准备。
*   **盲目等待**：脑子空空，没有规划，只是被动地待在舒适区，幻想机会降临。这是消极的消耗。

**公式：职业成功 ≈ 清晰战略 (人和) × 时间沉淀 × 平台机遇 (天时地利)**

你所做的一切，都是在为自己创造一个能发挥更大才能的环境。不断追求可实现自我价值的过程，就是不断获得合理自我利益的过程。

## 课程总结

本节课中我们一起学习了：

1.  **努力的必要性与局限性**：努力是基础，但在中大型组织中，单纯努力不足以构成核心竞争力。
2.  **战略的核心地位**：战略是为晋升所做的长线谋划，其关键是建立被更高层面“利用”的价值。
3.  **构建战略的步骤**：向上识别关键人，平行组建同盟，通过资源置换合理减负，聚焦核心部署。
4.  **必备的心态调整**：摒弃无谓的“玻璃心”，在站稳脚跟前优先关注自我成长，学会聪明地“组局”。
5.  **贯彻长期主义**：区分“战略等待”与“盲目等待”，在谋划清晰的基础上，耐心等待并抓住“天时地利”的机遇。

记住，**没有战略的努力工作，不过是一场空**。请从现在开始，不仅埋头干活，更要抬头看路，为自己的职业生涯绘制清晰的战略蓝图。
