## 推拉技巧解析

### 推拉的定义
推拉是一种通过打压和拉回的方式，让对方产生情绪波动，从而消除负面印象的技术。

### 推拉要点

#### 1. 先拉后推
在推之前，一定要先拉，即先认同对方，再进行调侃或开玩笑。

**公式**：
```
认同 → 调侃/开玩笑
```

#### 2. 避免自我提升
不要在推拉中制造自我提升的感觉，避免让对方产生反感。

**公式**：
```
避免：认同 → 打压 → 自我提升
```

#### 3. 把握度与退让
如果调侃对方导致其生气，要立刻退一步，道歉并提出补偿。

**公式**：
```
调侃 → 生气 → 退一步 → 道歉 + 补偿
```

#### 4. 避免道德评判
不要用道德观或价值观去评判对方，避免失格或打压。

**公式**：
```
避免：认同 → 打压 → 道德评判
```

#### 5. 主动找台阶
在调侃对方时，要主动帮对方找台阶下，避免沾沾自喜。

**公式**：
```
调侃 → 生气 → 主动找台阶
```

#### 6. 推拉一体
推拉要相互配合，推中有拉，拉中有推。

**公式**：
```
推 → 拉 → 推 → 拉
```
