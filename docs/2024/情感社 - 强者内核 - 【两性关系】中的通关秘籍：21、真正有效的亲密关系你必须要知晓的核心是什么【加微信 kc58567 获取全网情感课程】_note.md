# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：21、真正有效的亲密关系你必须要知晓的核心是什么

## 概述

在本节课中，我们将探讨如何建立和维护一段真正有效的亲密关系。我们将分析核心要素，并给出具体的建议和技巧。

## 核心要素

### 1. 呼应

**定义**：在平淡的生活中，彼此有呼应，无需衡量标准，只要双方觉得没问题即可。

**公式**：呼应 = 平淡生活中的默契

### 2. 积极性

**定义**：增加愉快的相处机会和时间，减弱情绪冲突的相处机会。

**公式**：愉快相处 = 质量而非次数

### 3. 开放性

**定义**：坦诚地直面关系，无论好坏，并找到合适的沟通方式。

**公式**：开放性沟通 = 坦诚 + 沟通方式

### 4. 语言和行为

**定义**：巧用语言表达情感诉求，展现真诚，并引导对方给予回馈。

**公式**：语言推进 = 真诚 + 期待 + 回馈

### 5. 社交权

**定义**：彼此独立的社交圈是共同的，互相认识并肯定。

**公式**：社交权 = 互相认识 + 互相肯定

### 6. 分担任务

**定义**：共同面对家庭琐事，培养“我们”意识。

**公式**：分担任务 = 共同面对 + 共同协商

### 7. 回避

**定义**：在不影响亲密关系的情况下，给予对方一定的空间和隐私。

**公式**：回避 = 空间 + 隐私

### 8. 变相的幽默

**定义**：开彼此的玩笑，吐槽缺点，展现亲密无间的关系。

**公式**：变相的幽默 = 开玩笑 + 吐槽缺点

## 总结

本节课中，我们学习了建立和维护真正有效亲密关系的核心要素。通过理解并应用这些要素，我们可以更好地与伴侣相处，建立更加稳固和幸福的关系。
