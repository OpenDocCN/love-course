# 玲金玉（五套合集）：1.亲密关系技巧 56节：23.搔首弄姿_note

在本节课中，我们将学习一个名为“搔首弄姿”的亲密技巧。这个技巧的核心在于运用手部和舌头的协调动作，旨在为伴侣带来独特的感官体验。我们将分步解析其动作要领与变化方式。

## 核心动作解析 🎼

“搔首弄姿”这个动作要求用手握住对方的阴茎。

同时，用舌尖在龟头、马眼以及包皮系带等部位进行挑弄。

舌尖的动作应轻柔地扫过龟头区域。

动作也应涵盖包皮系带部位。

在操作时，舌尖可以沿着“Z”字型或“S”型路径移动。

**核心路径公式：** 舌尖轨迹 ≈ 画“Z”或“S”型

## 动作的两种实现方式 🌀

上一节我们介绍了核心的动作路径，本节中我们来看看实现这一路径的两种具体方法。你可以选择不同的身体部位主导动作。

以下是两种主要的操作方式：

1.  **仅用舌头**：保持头部相对静止，完全依靠舌头的灵活运动来画出“Z”字型。
    *   **代码描述：** `action = move_tongue_in_pattern(“Z”) while head_stationary`

2.  **结合头部转动**：在舌头运动的同时，轻微地转动头部，以扩大动作的覆盖范围或改变力度感受。
    *   **代码描述：** `action = move_tongue_in_pattern(“Z”) + rotate_head_slightly()`
