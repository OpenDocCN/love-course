# 【李大魔】《极致吸引》：016.框架逆转2：提要求

**概述**： 本节课将探讨在情感关系中，如何通过提要求来引导对方投入，以及如何通过提要求来提升自己在关系中的地位。

**一、提要求的本质**

要求本质是一种引导对方投入的过程。在许多感情关系中，对方之所以轻易离开，甚至不需要付出任何代价，是因为违约成本非常低。很多人在感情中不会索取，不会提要求，导致对方在情感关系中的沉默成本很低。

**二、违约成本**

违约成本一般发生在刚确定关系时。例如，第二天对方突然反悔，不想搭理你；或者约会后，对方不再出现。如果你交往的女朋友足够多，就会经常遇到这种情况。这是因为对方在情感中的违约成本较低，可以随意背叛恋爱契约关系。

**三、强者心态**

强者不会害怕女人逃跑、提分手或找其他人。因为强者在关系中将其视为一种体验，是一种活在当下的感觉。他们会享受这种体验，而不是为过去和未来的事情而内耗焦虑。

**四、提要求的技巧**

1. **明确引导投入**： 通过提要求，明确引导对方为你投入，例如让对方为你倒水、拿快递、夸你等。
2. **等价交换**： 当对方提出要求时，可以提出等价交换，例如让对方请你吃饭作为你帮忙搬家的回报。
3. **软框架**： 使用软框架，例如调侃式地提出要求，而不是生硬地命令对方。
4. **引导服从的谢顶性**： 通过提供价值，引导对方进行小的服从行为，例如让对方夸你、叫你爸爸等。

**五、案例分析**

1. **搬家案例**： 当对方提出帮你搬家时，可以调侃式地提出要求，例如“那你请我吃什么大餐呀，不然我没有力气搬家哦”。
2. **购物案例**： 当你为对方花了钱后，可以要求对方为你提供相应的服务，例如让对方给你按摩、推背等。

**总结**： 本节课中，我们学习了如何通过提要求来引导对方投入，以及如何通过提要求来提升自己在关系中的地位。希望这些技巧能够帮助你更好地处理情感关系。
