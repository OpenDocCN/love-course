# 25、李想-性格识人课：004-03 你是哪一种人格

## 概述
在本节课中，我们将学习如何通过144道题的九型人格测试来了解自己的性格类型。

## 测试说明
### 测试版本
- 使用144道题的专业版本进行测试。

### 测试原则
1. **二选一**：每题选择一个选项。
2. **快速回答**：凭第一感觉快速作答。
3. **真实选项**：选择最符合自己的选项，而非渴望的选项。
4. **柱状图分析**：完成测试后，形成9种性格类型的得分柱状图。
5. **关注高、低分**：关注得分高和低的类型，代表性格优势和劣势。
6. **确定性格类型**：通过最高分和前三高分的类型确定性格类型。

## 测试步骤
1. **选择选项**：在每题的两个选项中选择一个。
2. **快速作答**：不要长时间思考，快速选择。
3. **真实作答**：选择最符合自己的状态，而非理想中的状态。
4. **完成测试**：完成所有题目后，形成柱状图。
5. **分析结果**：分析得分，确定性格类型。

## 性格类型确认
1. **了解特质**：通过线上课程了解每种性格类型的特质。
2. **对号入座**：根据课程内容，确定自己的性格类型。
3. **了解他人**：了解身边人的性格类型，如老板、同事、家人等。

## 总结
本节课中，我们学习了如何通过144道题的九型人格测试来了解自己的性格类型。希望这个课程能帮助你更好地认识自己，了解他人。
