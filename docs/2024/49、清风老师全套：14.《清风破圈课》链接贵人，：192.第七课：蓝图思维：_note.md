# 49、清风老师全套：14.《清风破圈课》链接贵人，：192.第七课：蓝图思维：_note

在本节课中，我们将学习如何与“贵人”（即资源丰富的大佬）建立深度信任，并最终促成合作。我们将分阶段拆解这一过程，核心在于通过建立舒适感、提供情绪价值并展现自身价值，来赢得对方的青睐与支持。

上一节我们介绍了如何鉴别真假大佬。本节中，我们来看看如何获得对方的信任并推进合作。

## 绘制合作蓝图：三个阶段

我将整个过程分为三个阶段来讲解。我们已经学习了曝光思维、转化思维和产品思维。在与大佬的接触中，第一阶段“曝光”已完成，即成功出现在对方面前。接下来，我们将进入第二阶段：如何“转化”他们，使其从信任走向合作，甚至是持续合作。

这三个阶段再次浮现：曝光、建立信任、促成合作。如何让大佬信任你，是接下来要讨论的关键。

## 信任的基石：建立舒适感

所有的信任都建立在舒适感之上。即让大佬在与你的相处过程中感到放松甚至亲切。

在这个过程里，你接触的持续时间越长，舒适感越强，推进合作的机会也就越容易。

当然，如果只是一味建立舒适感，最终你可能只会变成大佬的酒肉朋友。因此，在合适的时间推进信任、推出项目、转化大佬，让他的资源为你所用，也非常关键。

## 把握推进的尺度

首先，我们必须在舒适感阶段给对方做好足够的铺垫。

以下是关于舒适感的三个核心问题：

*   **什么是舒适感？**
*   **怎么建立舒适感？**
*   **如何强化舒适感？**

接下来我会一一拆解。

### 舒适感的本质

舒适感的本质，是在与对方沟通交流的过程中，**顺着对方的话题往下走**，而不是拼命用你的话题去带节奏。

大佬们通常是自恋的，特别是当他们拥有强烈的成功经验时。他们的生活和社会阅历往往比我们更丰富。我们要做的是贴着对方的话题深入。

但难点来了：如果一味认同，大佬不会觉得你这个人有想法。那么，如何有技巧地聊天呢？

### 有技巧的聊天方法

当大佬抛出一个话题时，我们既要认同肯定，但同时要进行自己的思考。

**核心要点：千万别问是非题。**

这句话很重要。以下是一个错误示范：

```text
“大哥，我做这件事情对不对？”
```

这种问题是糟糕甚至愚蠢的，只会让大佬觉得你缺乏独立思考能力，对你们的关系有损伤。

更不要问“这件事我该怎么做？”。大佬通常不会教你具体做事的方法，他们拥有的是资源和人脉。你要听懂的是背后的底层逻辑。

那么，如何才能与大佬愉快地聊天呢？在后续课程中我会详细拆解。这里先帮你排除底层雷区。

### 正确的聊天姿势：倾听

有了正确的聊天方法，在顺着对方聊、对方愿意听的情况下，跟大佬正确的聊天姿势其实就是**倾听**。

倾听是一个动作，而不是一个话术。例如：

*   **质疑**：这是一个动作。
*   **认同**：这是一个动作。
*   **倾听**：这是一个**肢体语言**，而不仅仅是说“大哥，你说得太好了”（这显得浮夸）。

要让对方感觉到你在认真听他讲话。可以倾听对方的表达，并时不时用眼神和肢体语言与对方交流，让对方感受到你的肯定。

## 深化信任：寻求完善而非答案

大佬会向你表达具体想法。你该如何让对方建立更多信任呢？

教你一个句式：

```text
“大哥，我最近遇到一个问题。我理解后，用了这样的方式去解决，你能不能帮我完善一下？”
```

请大佬帮你**补充和完善**，往往能让对方快速建立认同，并给予你更多帮助。

这里还有一个小技巧分享：**随身携带录音笔**。

我自己就有一个习惯，随身带录音笔。它可以记录我与大佬的对话，特别是在回去的路上（如开车时），能提供一个深入交流的私密空间。在这种空间里把对话录下来，既体现了尊重，也体现了认可，大佬会更愿意持续输出。

因为大佬的生活也非一帆风顺。“人生不如意事十之八九，可与人言者无二三”。大佬的日子也有很多问题。马云曾说过“我对钱没兴趣，最羡慕一个月挣五万块钱的人”，我相信这是真话，因为他背负的压力太大。

此时，如果有人愿意听大佬持续输出观点和想法，大佬会非常愿意倾诉。所以请记住：**大佬们更需要情绪价值**。

## 促成合作的关键：展现自身价值

有了舒适感，提供了情绪价值，信任的建立只是时间问题。这时，只要你不急于求成，不迫切地向大佬索要资源，机会早晚会出现。

回到做事本身，回到做人本身，回到做项目本身，你才能让大佬持续为你投入。

那么，让大佬促成合作的关键是什么？是在他已经信任你的基础上，**建立在你足够努力的前提之上**。

为什么我与小徐只聊了十分钟，就愿意花一万二买两张线下课门票？后来仅聊了一小时，我就给他转了二十九万八？小徐比我还小两岁，是97年的。我为何如此信任他？

道理很简单：因为在过往的抖音商业周期中，以及他所表现出的努力里，我非常认可和欣赏他。他的努力让我看到了他身上有一种不服输的勇气、一股倔强的精神，以及不断努力向上攀登的意志。这都是我欣赏和认可他的地方。

反过来讲，大佬也一定会在我们身上寻找这些**踏实、努力、肯于吃苦**的状态。看到之后，才会给我们机会。

中国绝大多数成功的企业家并非来自互联网，很多是实业老板、传统行业出身。他们最欣赏的品质就是四个字：**艰苦奋斗**。

在努力的过程中，你往往能获得更多大佬的青睐。而在互联网上，大家比拼的往往是**更快的速度、更深刻的逻辑以及更强的执行力**。这与“艰苦奋斗”矛盾吗？不矛盾。它们的本质是一样的，都是在告诉年轻人：在向上攀登的过程中，要向大佬展现我们自身的价值，同时持续努力做事。

## 合作的发生方式

因此，合作一定不是出现在我们祈求大佬时，而是出现在**大佬主动关心我们**的过程中。我们不断抛出当下遇到的困难、现状和问题，让大佬来帮我们提供解决方案。

听到这里你可能困惑了：清风老师，你前面不是讲不能问“大哥怎么办”吗？

是的，我们不能问他，而应该让他来问我们需要什么资源。我们要描述自己当下遇到的困难，让大哥来主动帮我们解决。当你完成了这一步，就叫做**境界**。

## 警惕一个误区

有一句话是最大的“毒鸡汤”，请你一定要警惕，特别是在我们向上攀登的过程中，千万不要陷入这个误区。

这句话是：
```text
“有钱人根本不想认识我们这种普通人，因为他们只想认识更有钱的人。”
```

这句话是废话，也是陷阱。因为有钱人更多的社交其实是**向下**，而不是向上。想想看，他们本身的圈子已经很小，顶级大佬就那么多，如果还是同行，交流机会更少。

但是，大佬们愿意关注年轻人正在做的事情。他们也很焦虑，看到新行业时，往往需要更多年轻人去拼命干，他们更愿意投资，也需要新的思想。**大佬们在布局未来，而这就是我们年轻人的机会。**

所以请你记住：要向上破圈，因为大佬们也在等待像你这样**优秀、努力、充满上进心和斗志**的年轻人。
