# 19.心灵成像第二期：07_7、心灵成像：链接高维智慧

## 概述
在本节课中，我们将学习如何通过心灵影片技术链接高维智慧，获得指引和帮助。

## 核心概念
心灵影片技术是一种通过心灵影片与高维智慧沟通的方法，它利用德尔塔脑波作为桥梁，将我们的愿望和问题传递给高维智慧，并接收其指引。

## 心灵影片技术步骤
以下是使用心灵影片技术的具体步骤：

1. **收集信息与分析**：在清醒状态下，思考你的问题，并进行分析。
2. **睡前进入天才状态**：在睡觉前，使用倒数法或抓住天才状态，进入天才状态。
3. **检视心灵影片**：在天才状态下，重新检视清醒状态下记录下来的心灵影片。
4. **转化为计划或项目**：将问题转化为计划或项目，并使用想象力将其制作成另一部心灵影片。
5. **传递给高层智慧**：带着将心灵影片传递给高层智慧的意念入睡。
6. **观察与反馈**：在接下来的三天内，仔细观察周围发生的事情，并指示如何解决问题。

## 注意事项
1. **遵循心灵成像法则**：在制作心灵影片时，遵循心灵成像法则，如己所不欲勿施于人、解决方法必须让整个世界变得更美好等。
2. **问题转化为计划**：将问题转化为计划，以便高维智慧能够提供反馈和帮助。
3. **持续使用**：定期使用心灵影片技术，以便持续接收高维智慧的指引和帮助。

## 总结
本节课中，我们学习了如何使用心灵影片技术链接高维智慧，获得指引和帮助。通过心灵影片，我们可以将我们的愿望和问题传递给高维智慧，并接收其指引，从而实现我们的目标。
