# 钱进社 - 成为优质且最具竞争力的【社会人】：34、职业转型要注意的重点是什么

## 概述
在本节课中，我们将探讨职业转型时需要注意的重点，帮助大家更好地规划自己的职业发展。

## 职业转型的重要性
**公式**： 定期职业定位 = 状态和能动性保持
- 每隔3-5年，重新定位自己的职业路线，以保持状态和能动性。

## 转型前的自我评估
**列表**：
- **厌烦工作原因**： 具体分析厌烦工作的具体原因。
- **工作努力程度**： 评估自己在当前工作中的努力程度和提升空间。
- **转型客观理由**： 分析想要转型的客观理由，而非情绪化因素。
- **热爱行业**： 是否有热爱且能支撑不计回报的行业。
- **个人知点**： 评估个人在当前领域的知点，以及可变现的技能。
- **领域竞争力**： 分析当前领域内的竞争情况。

## 转型风险与应对
**公式**： 转型风险 = 新领域风险 + 坚持与度过方式
- 转型到新领域存在风险，关键在于如何坚持和度过。
- 分析自身对当前工作的不满和厌烦，明确转型原因。

## 转型成功的关键
**公式**： 知点 = 可跨领域应用的核心技能
- 在原有行业内积累的核心技能，可应用于其他领域。
- 评估个人支点，确保其可跨领域变现。

## 总结
本节课中，我们学习了职业转型时需要注意的重点，包括定期职业定位、自我评估、转型风险与应对以及转型成功的关键。希望这些内容能帮助大家更好地规划自己的职业发展。
