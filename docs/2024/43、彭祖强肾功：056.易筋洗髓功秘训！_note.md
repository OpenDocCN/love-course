洗髓功主要由吐纳呼吸、根部垂钓、拍打、易筋经和打坐组成。本节课的核心是吐纳呼吸法，以及为安全有效进行根部垂钓所做的准备工作，包括热身和捆绑。功法练习贵在坚持，只有持续练习，身体才能获得明显改善。

## 一、 吐纳呼吸法详解 🌬️

吐纳是洗髓功的基础，旨在通过特定呼吸方式调动内气，连接身体关键部位。正确的姿势和发力点是练习的关键。

### 1. 核心姿势与要领
练习吐纳时，需保持身体中正：
*   **双脚**：与肩同宽，站正，**切忌外八字**。脚外八会导致髋关节外旋，促使臀大肌发力，影响气的提升。
*   **双手**：扶于腰部。
*   **上身**：保持直立，沉肩，含胸拔背，不要挺胸或耸肩。
*   **头部**：收下颌，舌顶上颚，保持虚灵顶劲，拉伸脊柱。
*   **动作**：微屈膝，做胯，进行骨盆的卷曲运动（前倾与后倾）。

### 2. 两种核心呼吸法
以下是两种针对不同部位的呼吸震动方法：

**鼻腔震动法**
此法连接鼻腔与生殖器，旨在提升睾丸。
*   **动作**：吸气时，提纲（收缩肛门）、提睾丸、缩阴、收腹，**同时震动鼻腔**。呼气时，全身放松。
*   **呼吸**：**鼻吸鼻呼**。呼气时，一部分气从鼻呼出，大部分气应向内、向下灌注，为后续功法蓄力。
*   **代码示意发力顺序**：
    ```python
    # 吸气过程
    def inhale():
        vibrate_nasal_cavity()  # 震动鼻腔
        lift_perineum()         # 提纲
        lift_testicles()        # 提睾丸
        contract_abdomen()      # 收腹
    # 呼气过程
    def exhale():
        relax_all()             # 完全放松
    ```

**喉腔震动法**
此法连接喉腔与肛门，有助于改善痔疮等问题。
*   **动作**：姿势同上，吸气时**震动喉腔**，同样体会提肛的感觉。
*   **关键**：练习时上半身肌肉应保持放松，看似静止，实则内在运转。

### 3. 常见问题与过渡
上一节我们明确了吐纳的姿势和两种方法，练习初期可能会遇到一些反应，这很正常。
*   **发力点无感**：因相关肌肉（提睾肌）薄弱所致。坚持练习，尤其在后续进行根部垂钓负重训练后，感觉会逐渐明显。
*   **头晕耳鸣**：初期因用力吸气可能导致缺氧不适，属正常过程。感到不适可暂停休息，循序渐进。
*   **鼻子不通**：如有鼻炎，可先揉搓鼻翼两侧帮助通气。鼻子的通畅对性功能和健康都至关重要。

掌握了吐纳的基础，接下来我们需要为更深度的练习——根部垂钓做好准备。

## 二、 垂钓前准备：热身与捆绑 🪢

根部垂钓是洗髓功的重要环节，充分的热身和正确的捆绑是安全练习的前提。

### 1. 根部热身手法
垂钓前，必须进行5-10分钟的充分热身，以活络气血，避免损伤。
以下是热身步骤：
1.  **揉搓**：揉搓耻骨上方、两侧及腹股沟区域。
2.  **揉搓金锁（睾丸）**：轻柔地揉搓睾丸。
3.  **牵拉**：用手卡住阴茎根部，向前、上、左、右四个方向缓慢牵拉。**牵拉到有轻微痛感时停住，吸气并向该处努气（灌注意念）**，重复几次。
4.  **旋转**：卡住根部，进行正向和反向的旋转。

### 2. 捆绑布折法与绑法
正确的捆绑能确保力量作用在正确位置，并保证安全。

**折布方法**
1.  将专用布平铺，按**四指宽度**进行对折。
2.  折时向两侧拉平，折成长条状（非圆筒状），使布有一个开口。
3.  **开口一端朝前**。

**捆绑步骤**
1.  热身完毕后，一手卡在耻骨前阴茎根部。
2.  将折好的布条（开口朝前）置于卡住的手指下。
3.  一手压住布条一端，另一手开始缠绕勒紧。**务必绑紧**，避免滑脱。
4.  打第一个结，勒紧。检查是否靠近根部，可适当旋转调整。
5.  打第二个死结，再次勒紧。
6.  捆绑完成后，可轻轻下拉测试是否稳固。

### 3. 重量与时间建议
*   **起始重量**：新手建议从1-3斤开始，根据自身感受调整。
*   **增加频率**：如无不适，可每日增加1斤。若感到吃力，则在当前重量维持3天再增加。
*   **垂钓时间**：初期以能坚持为准，目标逐步达到每次8-10分钟，约320次呼吸摆动。

## 三、 垂钓练习与收功 🔄

现在，我们将吐纳、捆绑和垂钓动作结合起来。

### 1. 垂钓过程
1.  将捆绑布末端的绳套挂于重物（如水桶）上。
2.  提起重物，使其离地约10公分，先轻轻摆动，让身体适应重量。
3.  开始配合呼吸进行垂钓摆动：**吸气时上提，呼气时放松或向下灌注**。身体尽量保持稳定，减少晃动，主要依靠呼吸和盆底力量带动。
4.  过程中如有痛感，需忍耐。通常坚持一二百次后，痛感会适应。

### 2. 重要收功流程
垂钓结束后的收功至关重要，能巩固练习效果，促进恢复。
1.  **垂钓中收气**：垂钓结束后，先提着重量深吸几口气，将气灌注于睾丸区域。
2.  **解开重物**：然后解开绳套，放下重物。
3.  **拍打放松**：**必须**进行全身拍打，特别是酸痛的点，以疏通经络，促进气血循环和恢复。
4.  **甩动**：最后可进行阴茎的甩动（左右或前后），进一步放松。

## 四、 核心原则与季节养生 🌱

功法练习需遵循自然规律，并与生活相结合。

*   **持之以恒**：方法是基础，但唯有通过持续练习（“时间变成功夫”），身体才能发生根本改变。
*   **冬季养生**：冬季主收藏。此时性欲收敛、勃起不如春夏频繁，是身体在自我修复和蓄能的表现，**是好事**。应顺应季节，减少泄精，固守精气，以待来年春天生发。
*   **饮食配合**：练习者，尤其是体重偏大者，晚餐不宜过饱。过量的营养若无法转化，反成负担。
*   **正心正念**：练习时需专注呼吸与身体感受，减少杂念。达到高层次时，可做到“一念不生，唯有勃起”，固摄能力极大增强。
