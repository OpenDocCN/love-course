# 老白《约会达人》：情场3.0：【聊天解码】8.如何认识更多的女生_note

在本节课中，我们将要学习如何拓展你的社交圈，认识更多的女生。这对于实践聊天技巧、积累经验至关重要。

很多兄弟自身条件不错，但面临的主要问题是身边可以聊天的女生太少。学习了一些技能，却无处实践。对于技巧尚不熟练的初学者而言，这尤其可怕，因为很容易把天聊死。因此，拥有足够多的女生资源来支撑我们不断试错，就显得尤为重要。

我们认识女生的传统方式主要依赖天生缘分，例如家人朋友介绍、单位同事、聚会等等。

然而，通过这些方式认识的资源是有限的。不可能天天都有人给你介绍，也不可能天天都有聚会可参加。如今，我们已经进入互联网新时代，日常生活越来越离不开网络。

互联网在不断改变我们的生活方式，包括交友方式。因此，大量的异性交友软件应运而生，比如早期的陌陌、世纪佳缘。

以及现在主流的探探、积目等等。互联网必将成为未来主要的交友方式。

因为它让两个陌生人相互认识的成本变得很低，省去了大量线下见面的时间。通过几张照片，就可以让对方对你产生基本印象。喜欢的话就聊。

不喜欢就换下一个。社交软件最大的优势就在于此：平台上的女生资源几乎是取之不尽、用之不竭的。很多兄弟有一个错误观念，认为网上认识的女生都不靠谱。这主要是因为你**不懂得筛选和辨别女生的类型**。

以及**不懂互联网交友的游戏规则**。所以你会遇到很多不靠谱的女生。这些方法我也会在接下来的课程中全部教给你们。我要告诉你们，通过社交软件认识女生，是当下最靠谱的资源收集方式。

过去你收集女生的方式，只能局限于你的固定圈子里。而社交软件能让你认识各种年龄、性格、职业和圈子的女生，你的选择范围会扩大很多。你接触到的女生质量，通常也会比你身边圈子的女生质量高很多。

另外，假设你在一次聚会上，同时觉得两个或更多女生都不错。但你不好意思上前都认识一下，因为你害怕别人觉得你花心。

而社交软件完美地避免了这一点。你可以大方地去认识更多你感兴趣的女生。

最后挑选一个最喜欢的，与她好好相处。这有点像我们常说的“闷声发大财”。有些兄弟会说，我只想老老实实地和一个女生交往接触，社交软件不适合我。说到这里，我必须指出一个关键问题。

你怎么能确定你想好好相处的那个女生，是否也想和你好好相处呢？感情是相互的，只有在对方愿意搭理你的前提下，你才有与她好好相处的必要。因此，我们应该先去认识大量的女生，从这些女生中慢慢筛选出那些愿意搭理你的。

然后你再从中选择，与她好好相处。我们无法做到让每一个女生都对我们感兴趣，但我们可以努力让大多数女生对我们感兴趣。当然，除了社交软件，就没有其他面对面认识女生的方式了吗？答案当然是有的，比如搭讪。

搭讪和社交软件的性质是一样的，核心都是**筛选自己喜欢的女生**。

只是把筛选的方式，从线上搬到了线下。而搭讪最大的优点是，你可以最直接地认识让你最满意、最心动的女生。关于搭讪的详细方法，我将在接下来的课程中详细讲解。

以下是本节课的总结：

第一，社交软件是当下最流行、最主流，也是最快速认识女生的途径。它能让你认识到更多不同类型的女生资源。

第二，认识很多女生并不代表你不专一。真正的策略是在这些女生中，选择一个合适你的，去和她好好相处。

本节课中，我们一起学习了拓展女生资源的重要性，并重点介绍了社交软件这一高效、主流的认识女生的途径，同时也简要提及了线下搭讪作为补充方式。关键在于转变思维，主动扩大选择范围，为后续的深入交往奠定基础。
