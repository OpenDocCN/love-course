# 沟通课03：超越语言：I See You 👁️

在本节课中，我们将要学习沟通中超越字面含义的核心：如何通过“看见”对方，建立真正相互尊重的关系。我们将探讨尊重、主体性以及“I See You”的深刻内涵。

---

上一节我们探讨了语言在关系中的功能，本节中我们来看看，如何超越语言本身，实现真正的相互理解和尊重。

不管是主动握手的权利者，出人意料的赞美者，还是冒犯自己的求助者，他们都通过某些语言和行动打破了一个预期，然后释放了一个信号。这个信号叫“我选择尊重你”。这个预期叫做“我应该尊重你”。

“应该”和“选择”之间有天壤之别，是“猫”和“人”的区别。“我应该尊重你”意味着对象迫于形势，不得不去尊重。“我选择去尊重你”意味着主体基于自己的意愿，主动选择去尊重你。一言以蔽之：**我通过主动选择**。

在你眼中展现为一个有主动性的人，一个有尊严的主体。被一个有尊严的主体尊重，而非被一个对象当成预期中的另外一个对象去应对，才会顺应你的欲望，才会让你感觉被尊重，才会让你有成为主体的感觉。

这样，互为主体的关系就建立起来了。这是相互尊重的终极本质，也是我们走出“地狱”的唯一方法。你会发现，今天我们绕来绕去，结果只干了一件特别简单的事情：把我们概念图中的“对象”变成了“主体”。

这似乎就是我们常说的那句：**你把别人当人看，别人才会把你当人看**。

说到“把人当人看”，我想起一个特别有意思的命题。以下是两种完全相反的心理或价值取向：

*   一种人对抽象之人充满了漫无边际的慈悲和善意，但是对眼前的人却常常感觉很厌恶。
*   另一种人完全相反，他对抽象的人性毫无信心，但是对眼前的血肉之躯却充满了包容和善意。

抽象和具体的区分，并不在于他是否出现在你眼前的物理空间中，而在于他有没有作为一个真正的人，出现在你的心理世界里面。

我以前去外面办业务，对接工作人员心情会很暴躁，觉得对方不专业、效率低。你知道，那是在系统的流程中、在权力关系中、在对于“对象”的预期中才会有的反应。这个时候，我一旦仔细地看一下他，看他的眉眼，看他脸上的痘痘，看衣服上的褶皱，看鞋子上的灰尘，看他面前的工作牌，看上面的姓氏和名字，我的心里就会生出很多善意，面目也会舒展很多。我心想：原来他和我一样，他也是有名字的。

我们生活中的理性太多，觉性太少。给你一个建议：下次和朋友见面，不要总想着说点什么，你就认真地看着他。哪怕你只是认真地看着他，然后一边看一边傻笑，他就会问你：“你傻笑什么呀？”然后你说：“没什么，I See You。”

我们还真的得在《阿凡达》的意义上理解什么是“I See You”。有没有想过，为什么一个商业科幻大片的名字不叫《决战潘多拉》，要叫《阿凡达》？为什么整个片子的落点叫“I See You”？

在语言世界中，“I See You”是一个开天辟地的句子，因为它包含了这个宇宙中最原初的主语、谓语和宾语。但是这个句子的灵性已经被我们的符号秩序给遮蔽了。“I See You”已经没有了，只剩下一个化身（Avatar）看着另外一个化身而已。

这种秩序遮蔽了很多重要的东西，又衍生出来很多诡异的概念，比如前面讲到的“赞美”，和我们接下来要讲到的“自信”。

“自信”也是一个需要重新理解的概念。我们认为自信是一种在与他人交往的过程中，保持一种自我肯定状态的能力。当我们说“我们要自信”的时候，其实是我们要主动展现出一种交往姿态。这种姿态告诉对方：“我在这里，我不逊色于你。”

但是自信的诡异之处在于：在理性上，我们认为它一定是来自于自己的；但是在经验中，它又总是来自于外部的反馈。这种抽象理解和具体经验之间的错位，会让我们心中的“他人”变成一个非常扭曲的存在。他人既是一个有待超越的卑劣的“对象”，又是权倾天下的凝视的“目光”。

所以我们领到这样的诡异命题，就只能造出下面这种听起来非常奇怪的句子：“我根本不在意你怎么看我，总有一天我会证明这一点。”这句话虽然自相矛盾，但却是一句真心话。意思是：**你根本不重要，但是你的看法却很重要**。

这种“你不重要，你的看法却重要”的诡异自信观，就会让我们在“卑”与“亢”之间来回摆荡。原因其实特别简单：加诸于他人的目光，终有一天会被自己所承受。所以，“你不重要，你的看法却很重要”一定会变成“我不重要，但是我的看法却很重要”这样的认知。

所以，三角关系中的“你”和“我”最终一定会被抽空。剩下的就是各种“看法”和“评价”拼贴而成的“阿凡达”（Avatar）。“阿凡达”除了“化身”之外，还有一个意思是“符号化的现象”，我觉得更加贴切：你和我是没有生命的，符号化的评价和看法才是我们的命。

每次发视频我都不敢看评论，下面那一堆符号。看见一个差评，哎呦，不行不行，要死了要死了。那这个时候你要赶紧给我看一个好评，我马上生龙活虎地活过来。

所以，“I See You”的深刻含义是：**我的目光透过了符号化的形象，真正的看见了你**。因为这种“看见”，我才成为了“我”——不是那个符号秩序中的、不是那个权力结构中的那个化身，而是真实的、具体的、亲密关系中的那个“我”。

如果真的有一种东西叫“不卑不亢的自信”，那么它一定是从别人的目光中反射回来、被你感知到的一种“全神贯注的看见”。已经看见“你”和“我”了，不需要“看法”了。“你”和“我”的位置都已经确定了，我们不需要再搞什么三角测量了。

所以我们对于自信的那种模糊笼统的感知，其实是很准确的：在源头上，它一定是由你自己创造的；但是在方向上，它一定来自于对方，只不过中间有一个“互为主体”的反射而已。

---

本节课中，我们一起学习了沟通中超越语言的核心。

这期我们聊的是语言活动中的“关系侧”，探讨的是一句话到底是如何说进人的心坎里的。但我们今天提纲挈领，聊的是交往活动中，发自内心的相互尊重是如何被制造出来的。

我们今天的主题深化是由三组、每组八个字推进的：
*   在技巧层面，八个字是：**打破预期，顺应欲望**。
*   在原理层面，八个字是：**主动选择，互为主体**。
*   在哲理层面，八个字是：**你中有我，我中有你**。

而且你发现了吗？三种权力关系下的三个案例，分别对应了语言的三个功能：告知、评价和请求。

讲到这些，你还认为我们如何在物理层面去遣词造句、去组织一个语言是多么致命的事情吗？它很重要，但是它不致命。

下一期，我们要带着对“关系侧”的理解进入“内容侧”，带着今天的认识去看见字词句篇之下的万物生长，敬请期待。
