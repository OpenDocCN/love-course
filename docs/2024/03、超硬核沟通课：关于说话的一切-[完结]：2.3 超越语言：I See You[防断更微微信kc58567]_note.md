# 沟通课03：超越语言：I See You 👁️

在本节课中，我们将探讨沟通中超越字面含义的深层关系。我们将理解，真正打动人心的并非华丽的辞藻，而是通过语言和行动传递出的尊重与看见。

## 从“应该”到“选择”：尊重的本质

上一节我们探讨了沟通中的权力与姿态。本节中，我们来看看尊重是如何产生的。

不管是主动握手的权利者、出人意料的赞美者，还是那个冒犯自己的求助者，他们都通过某些语言和行动打破了一个预期，然后释放了一个信号。这个信号叫“我选择尊重你”。这个预期叫做“我应该尊重你”。

“应该”和“选择”之间有本质区别。“我应该尊重你”意味着对象迫于形势，不得不去尊重。“我选择去尊重你”意味着主体基于自己的意愿，主动选择去尊重。一言以蔽之：**我通过主动选择，在你眼中展现为一个有主动性、有尊严的主体**。

被一个有尊严的主体尊重，而非被一个对象当成预期中的另外一个对象去应对，才会顺应你的欲望，才会让你感觉被尊重，才会让你有“成为主体”的感觉。

这样，互为主体的关系就建立起来了。这是相互尊重的终极本质，也是我们走出“地狱”的唯一方法。你会发现，今天我们探讨的核心，就是把我们概念图中的“对象”变成了“主体”。

这似乎就是我们常说的那句话：**你把别人当人看，别人才会把你当人看**。

## 抽象与具体：两种价值取向

说到“把人当人看”，这里有一个值得深思的命题。

以下是两种完全相反的心理或价值取向：
*   一种人对抽象之人充满漫无边际的慈悲和善意，但对眼前的具体个人却常常感到厌恶。
*   另一种人对抽象的人性毫无信心，但对眼前的血肉之躯却充满包容和善意。

抽象和具体的区分，并不在于他是否出现在你眼前的物理空间，而在于他有没有作为一个真正的人，出现在你的心理世界。

例如，以前去办事时，面对效率低下的工作人员可能会很暴躁。那是在系统的流程、权力关系和对“对象”的预期中产生的反应。一旦我仔细看他——看他的眉眼、脸上的细节、衣服的褶皱、鞋上的灰尘、工作牌上的名字——心里就会生出许多善意。我心想：原来他和我一样，是有名字的。

我们生活中的理性太多，觉性太少。给你一个建议：下次和朋友见面，不要总想着说点什么，你就认真地看着他。哪怕你只是认真地看着他，然后一边看一边傻笑。他可能会问你：“你傻笑什么呀？”你可以说：“没什么，I See You。”

## I See You：看见的灵性

我们还真的得在电影《阿凡达》的意义上理解什么是“I See You”。为什么一个商业科幻大片的名字不叫《决战潘多拉》，而叫《阿凡达》（Avatar）？为什么整个片子的点睛之句叫“I See You”？

在语言世界中，“I See You”是一个开天辟地的句子，因为它包含了宇宙中最原初的主语、谓语和宾语。但这个句子的灵性已被我们的符号秩序所遮蔽。“I See You”已经没有了，只剩下一个化身（Avatar）看着（looking at）另外一个化身而已。

这种秩序遮蔽了很多重要的东西，又衍生出很多诡异的概念，比如前面讲到的“赞美”，以及接下来要讲的“自信”。

## 重新理解“自信”

我们认为自信是一种在与他人交往中保持自我肯定状态的能力。当我们说“要自信”时，其实是我们要主动展现出一种交往姿态。这种姿态告诉对方：**我在这里，我不逊色于你**。

但自信的诡异之处在于：在理性上，我们认为它一定来自于自己；但在经验中，它又总是来自于外部的反馈。

这种抽象理解与具体经验之间的错位，会让我们心中的“他人”变成一个非常扭曲的存在：他人既是一个有待超越的卑劣对象，又是权倾天下的凝视目光。

于是，我们只能造出下面这种听起来奇怪却真实的句子：
> 我根本不在意你怎么看我，总有一天我会证明这一点。

这句话自相矛盾，却是一句真心话：**你根本不重要，但是你的看法却很重要**。

这种“你不重要，你的看法却重要”的诡异自信观，会让我们在自卑与亢奋之间来回摆荡。原因很简单：加诸于他人的目光，终有一天会被自己所承受。所以，“你不重要，你的看法却很重要”一定会变成“我不重要，但是我的看法却很重要”这样的认知。

三角关系中的“你”和“我”最终会被抽空，剩下的就是各种看法和评价拼贴而成的“化身”（Avatar）。化身除了指代身体，还有一个意思是“符号化的现象”，这更加贴切：你和我是没有生命的，符号化的评价和看法才是我们的命。

就像每次发视频都不敢看评论，看见一个差评就觉得“要死了”，但赶紧看一个好评，马上又“生龙活虎”地活过来。

所以，“I See You”的深刻含义是：**我的目光透过了符号化的形象，真正的看见了你**。因为这种看见，我才成为了“我”——不是那个符号秩序或权力结构中的化身，而是真实的、具体的、亲密关系中的那个“我”。

如果真的有一种东西叫“不卑不亢的自信”，那么它一定是从别人的目光中反射回来、被你感知到的一种“全神贯注的看见”。已经看见“你”和“我”了，就不需要看法了。“你”和“我”的位置都已经确定，不需要再搞什么三角测量了。

所以，我们对于自信的那种模糊笼统的感知其实很准确：在源头上，它一定是由你自己创造的；但在方向上，它一定来自于对方。只不过，中间有一个“互为主体”的反射而已。用公式表示这个关系可以是：
**自信 ≈ 自我创造 × 来自对方的看见（反射）**

## 总结与展望

本节课中，我们一起学习了语言活动中的“关系侧”，探讨了一句话到底是如何说进人的心坎里的。我们今天的主题深化是由三层含义、各八个字推进的：

*   **在技巧层面**：打破预期，顺应欲望。
*   **在原理层面**：主动选择，互为主体。
*   **在哲理层面**：你中有我，我中有你。

你会发现，三种权力关系下的三个案例（握手、赞美、求助），分别对应了语言的三个基本功能：告知、评价和请求。

讲到这些，你还认为我们在物理层面去遣词造句、组织语言是多么致命的事情吗？它很重要，但它不致命。
