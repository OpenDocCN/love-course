# 18.辰薇魅力女性训练营：12高衣Q帮助你远离穿衣误区

欢迎来到薇姐的魅力女性训练营的第十三讲。今天，我们将探讨如何提升个人衣品，远离穿衣误区。

## 一、穿衣误区

### 1. 追求流行

误区：盲目追求流行，认为流行什么就穿什么。

**解析**：流行与美学认知不同，并非所有流行都适合自己。选择服装时，应考虑是否适合自己，而非盲目跟风。

### 2. 满身大牌

误区：认为满身大牌可以提升形象力。

**解析**：过度追求品牌logo，反而会给人一种杂乱无章的感觉。着装目的是展现品味，而非彰显品牌。

### 3. 穿给自己看

误区：认为穿衣服只是为了取悦自己。

**解析**：着装不仅是为了取悦自己，还要在合适的场合展现自己，让对方感受到舒适度。

## 二、四维穿搭法

**公式**：四维穿搭法 = 场合维度 + 风格维度 + 色彩维度 + 品味层级

### 1. 场合维度

**解析**：根据不同场合选择合适的服装，如职场、休闲、社交等。

### 2. 风格维度

**解析**：找到适合自己的风格，如时尚、简约、休闲等。

### 3. 色彩维度

**解析**：根据肤色选择合适的色彩，如浅色、深色、中性色等。

### 4. 品味层级

**解析**：提升品味，追求高级感。

## 三、TPOR原则

**公式**：TPOR = 时间 + 地点 + 场合 + 角色

**解析**：在穿衣搭配中，充分考虑时间、地点、场合和角色，展现最佳形象。

## 四、色彩与肤色

### 1. 皮肤颜色

**解析**：根据皮肤颜色选择合适的色彩，如白皙、暗沉、黄种人等。

### 2. 肤色对比度

**解析**：根据肤色对比度选择合适的色彩，如弱对比、强对比等。

## 五、品味层级

### 1. 穷富贵雅素

**解析**：着装品位分为五个层级，从穷到雅，追求更高层次的品味。
