# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：51、真正稳固的亲密关系依托的是什么逻辑

## 概述

在本节课中，我们将探讨真正稳固的亲密关系所依托的逻辑，以及如何通过理解和应用这种逻辑来建立和维护长久的关系。

## 核心概念

**公式**： 稳固的亲密关系 = 深厚的友谊 + 相互尊重 + 低期待 + 接纳缺点

## 教程内容

### 1. 亲密关系的矛盾

*   大部分亲密关系中的矛盾都是无法解决的，因为它们源于主观意识和缺乏深度认知。
*   人们往往凭借个人主观意识去理解和行使男女之情，导致关系失控。

### 2. 友谊与亲密关系的差异

*   友谊的稳固性源于留白、松弛、没有过多期待和接纳对方的缺点。
*   亲密关系则相反，人们往往对伴侣有更高的期待和苛求，导致矛盾和不满。

### 3. 友谊逻辑在亲密关系中的应用

*   将友谊逻辑应用于亲密关系，可以建立稳固且美好的关系。
*   例如，薇娅和她老公的婚姻关系，就是以友谊逻辑为基础，取得了成功。

### 4. 孩子与亲密关系

*   孩子是夫妻双方的连接体，但并非生命的延续。
*   夫妻双方应保持独立的责任和意愿，避免陷入不满和期待的循环。

### 5. 双打概念

*   双打概念指的是两个独立个体之间的互动，例如性生活。
*   双打概念并非像机器一样一键锁定，需要双方共同努力。

### 6. 友谊与亲密关系的记忆

*   人们更容易记住与朋友之间的美好时光，而与伴侣之间的痛苦经历。
*   这是因为对伴侣有更高的期待和规则，导致更容易记住负面事件。

### 7. 规则与亲密关系

*   规则的产生会限定行为和语言，导致无法好好沟通。
*   修补破裂的亲密关系成本很高，因此需要智慧地看待关系。

## 总结

本节课中，我们学习了真正稳固的亲密关系所依托的逻辑，以及如何通过理解和应用这种逻辑来建立和维护长久的关系。希望这些内容能够帮助大家更好地理解和处理亲密关系。
