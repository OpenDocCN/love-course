# 清风老师合集：01解读孙子兵法智慧8期：005.第三课：计篇之 阳谋

## 概述
在本节课中，我们将学习孙子兵法中的“计篇”，特别是关于阳谋的策略。

## 原文解读
### 1. 教之以技而所其情曰煮熟有道
**公式**：计算五个科目，了解具体情况，进行实力博弈对比。

**解释**：通过计算和对比，了解双方实力，为战略决策提供依据。

### 2. 将孰有能，天地孰得，法令孰刑，兵重孰强，士卒孰练，赏伐孰民，无以此之胜负矣
**公式**：将领能力、天地条件、法令执行、兵力强弱、士兵训练、赏罚制度。

**解释**：通过以上因素判断双方胜负。

### 3. 将听武技，用之必胜，留之将不听武技，用之必败去也
**公式**：听从战略，必胜；不听从，必败。

**解释**：孙子强调战略的重要性，只有遵循战略才能取得胜利。

## 强势文化与弱势文化
**公式**：强势文化 = 遵循客观规律；弱势文化 = 以个人喜好做判断。

**解释**：强势文化注重客观规律，而弱势文化容易受情绪影响。

## 先胜后战与增量
**公式**：先胜后战 = 必胜把握；增量 = 创造额外价值。

**解释**：孙子兵法强调先确保胜利，再进行战斗，同时注重创造额外价值。

## 总结
本节课中，我们学习了孙子兵法中的阳谋策略，包括计算实力、判断胜负、强势文化与弱势文化、先胜后战与增量等核心概念。通过学习这些策略，我们可以更好地应对生活中的各种挑战。
