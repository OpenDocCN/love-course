# 魅力之道全板块：002.第一印象-信任（第二天）

欢迎来到第二天的课程。首先，在课程开始前，我要问你是否完成了昨天的作业？如果没有，请先完成作业，因为只听课不实践，你提高的只有认知。

想要获得结果，就必须实践。只有这样，你才能内化知识，将其转化为行为，帮助你成为一个更有魅力的人。

### 信任的重要性

如果只有积极的情绪价值而没有信任，没有人会愿意与你深交。例如，街边的小丑或脱口秀演员，他们能让你开心，但不会因此与你深交，因为你缺乏信任。

建立信任的重要性在于，别人与你在一起时，无法放心地打开自己，没有足够的舒适感，这会制约你们关系的发展。

### 如何建立信任

信任的基础是诚实。如果你让别人感到你是真诚的、诚实的，他们会更加信任你。但在某些情况下，你会面临选择，比如在说话时，大多数人说的都是真话，但能建立信任的诚实，取决于在某些时刻，你牺牲自己的利益来选择诚实地说真话。

#### 不诚实的情况

1. **不诚实的情况一**：例如，你今天过得很糟糕，但别人问你过得怎么样，你却回答“我好极了”。这是为了做正能量的人，从而让别人喜欢你。但这样会失去信任。
2. **不诚实的情况二**：意图不清晰。例如，在约会时，你害怕问出问题，导致意图不清晰。

### 诚实与信任

诚实是建立信任的基本盘。在行为上，说话要习惯脱口而出，不要犹豫。诚实并不意味着全盘托出，你可以选择点到为止地透露信息。

### 触碰

触碰能拉近你和别人的距离，让别人对你产生信任感。例如，男人之间的初次见面可以选择握手或碰拳，熟悉的人可以选择拥抱。

### 今天的作业

今天的作业是在与人见面时，用一种触碰方式和对方形成触碰。你可以选择与同性的人开始，比如在公司，你可以把手放在同事的肩膀上问个好。

### 总结

本节课中，我们学习了信任的重要性以及如何建立信任。希望你能将所学知识应用到实际生活中，成为一个更有魅力的人。
