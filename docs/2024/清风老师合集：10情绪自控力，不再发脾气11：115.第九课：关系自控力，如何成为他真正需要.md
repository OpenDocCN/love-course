# 清风老师合集：10情绪自控力，不再发脾气11：115.第九课：关系自控力，如何成为他真正需要

在本节课中，我们将要学习如何在关系中建立自控力，避免因过度付出或不当索取而破坏关系。我们将探讨良好关系的要素，并学习“付出”与“提需求”的具体原则和步骤。

上一节课程中，我们学习了如何控制自己的时间，建立时间自控力。本节中，我们来看看如何在关系中进行自控。

前段时间我接到一位来访者的咨询。他向我哭诉说，他为对方付出了很多，甚至跨越千里去挽回爱情，但对方却非常冷漠和绝情。如果你在关系中有类似的困惑，试图通过付出来打动对方，那么你需要学习今天的课程。因为爱并非完全建立在付出上。爱的种子需要**互相吸引**的土壤，付出只是浇水施肥。没有土壤，只有水，植物无法生长。在关系中盲目付出，最终可能导致对方的忽视甚至厌恶。

前段时间有一个社会新闻，一名大学生为追求女神组织了盛大的告白仪式，结果被泼了一盆冷水。这种告白感动了自己，却没有打动对方。

除了过度付出的困境，关系中还存在另一个典型问题。例如，新闻中一位河南女孩的父母在婚礼前夜要求增加彩礼，导致新郎悔婚。女方的父母缺乏关系中的自控力，对未来的亲家采取掠夺性、索取性的态度，这自然会令对方害怕和离开。

那么，在一段关系中如何实现自控呢？我们首先需要了解一段良好关系是怎样的。

在我们的正向沟通课程中，我总结了一段良好关系的宏观和微观四大要素。

以下是宏观四大要素：
1.  **互相包容**：不会因为小事就导致关系破裂。
2.  **互相支持**：彼此关心、照顾，在困难时坚定站在对方身后。
3.  **允许犯错**：接受对方的不完美，允许关系中存在磕碰。
4.  **带有底线**：即付出的自控力。过度付出也会让人害怕，因为接受方会感到需要回报的压力。

以下是微观四大要素：
1.  在社交圈公开彼此关系。
2.  彼此真心投入。
3.  拥有共同的兴趣和爱好。
4.  保持持续的学习和成长。

建立了这八大要素，你们的关系会比较稳定。

今天我们要深入了解如何在关系中建立两种自控力：**付出的自控力**和**提出需求的自控力**。

前面我们讨论了过度付出的后果。正确的付出有三个原则。

以下是三个付出原则：
1.  **对方需要**：给予对方真正需要的东西，而不是你想给的东西。例如，对方需要香蕉，你却给了一车苹果。
    *   `错误做法：给予对方不需要的“苹果”。`
    *   `正确做法：观察并了解对方真正的喜好和需求。`
2.  **自己能够轻松承受**：付出应在自己经济或情感能力范围内。例如，月入一万却透支购买三万的包包，会给自己和对方都带来巨大压力。
3.  **不带有目的**：付出应是发自真心，不要求即时或对等的回报。这样的付出让对方感觉更真诚。

接下来我们说一说如何提出需求。

各位男性同学听到“提需求”时可能会紧张，认为男人不应该向女人提需求。但好的感情是互相付出的。把问题憋在心里反而会让伴侣困惑。女性也同样不应害怕提出需求。真正爱你的伴侣，愿意倾听并满足你的合理需求。

提出需求有四个步骤。

以下是提出需求的四个步骤：
1.  **先入为主（可选）**：可以先为对方付出，再顺势提出自己的需求。例如，想要一双AJ球鞋，可以先送对方一个等价值的包包。
2.  **直接提出**：明确、直接地表达你的需求，不要藏着掖着。同时，**允许对方拒绝**。例如：“老婆，如果你送我一双AJ球鞋（型号XXX），我会非常高兴。如果你没空买也没关系。”
3.  **再次争取**：可以尝试再次温和地表达你对这个需求的喜爱。
4.  **表达感谢或表示理解**：如果对方满足了你的需求，一定要表达开心和感谢。切忌通过“作”来获得爱，即对方付出后仍表现出不开心，这会对长期关系造成损害。如果对方未能满足，表达理解即可，争取两次后便不再强求。

这里你可能会问：前面说付出不要带有目的，但“先入为主”的提需求方法似乎带有交换目的，这矛盾吗？

这两者并不矛盾。我们付出礼物时，出发点应是让对方开心，我们收获的是**满足感**，而不是对方的回礼。而我们提出需求时，可以直接大方地提出，并允许对方拒绝。你甚至可以不用“先入为主”的方法，直接提出需求。

请你记住，好的关系是相互的：`他为你付出，你也为他付出；他珍惜你，你也支持他。`

最后，我想聊一聊在关系中被人拒绝这件事，例如告白被拒、挽回被拒或礼物被嫌弃。这都是关系中常见的失控。

失控的出现是必然的。我们人生中多数情况并非井然有序，但我们努力营造有序的假象以获得稳定感。就像智能手机电池会老化，手机无法永久使用，但在其使用寿命内是稳定的，我们便能接受。

面对失控，我们要找到其中的着力点，不要沉溺其中。如果你不知道如何走出极端情绪，可以回听第六课《情绪自控力》，避免过度放大、过度自责、过度要求这三类极端想法。

你可以向自己提出以下5个问题，以转化为成长型思维：
1.  我**学到了**什么？
2.  我可以继续**改善**什么？
3.  未来我可以通过什么**方式**，避免同一类型的问题？
4.  如果写一本自传，我如何**看待**现在经历的这件事？
5.  未来的我会如何**感谢**现在的我？

转化为成长型思维后，失控便不再可怕。失控恰恰意味着机会，让你看到之前忽略的点，发现对方未暴露的需求。
