# 富山山女性成长课程：06_情绪价值到底该怎么给？_note

在本节课中，我们将要学习情绪价值的第二和第三个层次：如何接纳对方的弱点，以及如何提供安抚、鼓励乃至必要的“负面”情绪价值。掌握这些技巧，有助于建立更深层、更健康的关系。

## 第二层次：接纳弱点与缺点 🤝

上一节我们介绍了情绪价值的基础，本节中我们来看看如何实践更高阶的部分——接纳对方的弱点。包容他人的短处是一件反人性的事，因此需要清晰的自我认知作为前提。

女生在关系中必须清楚自己的核心诉求是什么。明确最终目的后，再去接纳对方的弱点就不会感到委屈。许多人在关系中不断抱怨和不甘，往往是因为自己没想清楚到底要什么，甚至没考虑对方能否给予。

人都有弱点。当一个男人的弱点并非“抠门”或“家暴”这类致命问题时，大多属于边缘化弱点。只要我们心中有明确目的，这些都可以包容。

面对他人展现缺陷、不堪过往或弱点时，我们不应使用“忍耐”或“接纳”这类词汇。“忍耐”隐含不甘，我们应选择“忽视”，且不期待对方改变。许多人会指责对方“你哪里错了，你要改正”。但所谓“正确”并非以个人为标准。成年人性格很难改变，若无重大生活变故或打击，缺点很难修正。这不是不想改，而是改不了。

我们自己清楚自身缺点。当一个人能全盘接纳我们时，感受是被疼爱、被理解、信赖、放心和安全感。因此，面对男性的弱点和缺点时，也应给予同样态度。告诉他：“我全盘接纳你。”此刻是展现女性如母亲般胸怀的时机。目的可以是让关系更幸福长久，也可以是引导对方付出。

**明确自身目的永远是第一步。**

## 第三层次：安抚、鼓励与必要的“负面”价值 🔄

接下来我们探讨情绪价值中的安抚、鼓励，以及如何适时提供“负面”反馈。

### 安抚与鼓励：成为他的能量源 🔋

与高经济价值的高阶男性相处意味着什么？意味着他在外拼搏，在职场和社会中摸爬滚打。他需应对上级，管理下属，自然会有委屈和烦心事。

此时，我们需要给予他能量，成为他的“电池”充电。一个女人能与高阶者相处，需能作为一个“器皿”，接得住对方的情绪。若对方倾诉苦水、同事矛盾或社会委屈，你必须让他感到你是值得信任的人，是他的解语花和温暖怀抱。

有时只需轻抚其头说：“有我在。”你可以说：“乖，没事，来我怀里。”抚摸头发这一动作极具母性。给予信任的眼神和微笑，告诉他：“我相信你。”“我相信你能度过难关。”“我陪着你。”“我理解你的难处。”“我相信你的能力，我会一直在你身边。”

以下是我常用的几句话：

*   “这么多年一个人摸爬滚打，是不是很难？你需要我做什么，随时告诉我，粉身碎骨我也会去做。”
*   “其实我相信你可以解决，但这一刻我觉得你像个孩子，我非常心疼。”
*   “别人看到的是你坚强的一面，这种脆弱时刻只有我能看到。我既欣慰又心疼。”
*   “谁欺负了你？我不许别人欺负你。告诉我，谁惹了你，我去替你出头。”

你甚至可以陪伴他落泪，眼眶湿润地说：“谁敢欺负你？谁敢欺负我的心肝宝贝？”“你受了那么多委屈，我都帮不上忙。我嘴笨，但心是炙热真诚的。”

还可引导他看到事情两面性：“老天爷给你这么多磨难，究竟有多大的福报在后面等着你？”

这些都属于非常正向的情绪价值。

### 必要的“负面”情绪价值：适时按压与边界设立 ⚖️

现在我们来讲如何操作必要的“负面”情绪价值。你需要适时按压对方，防止他过于膨胀。

可通过侧面阐述或间接话术让他明白不能骄傲。例如，我的另一半身家几十个“小目标”。在日常相处中，他一直清楚我对此不屑一顾。在他眼中，我是一个温柔可爱、古灵精怪，但又非常独立、有自己精神世界和事业的女强人形象。他从不觉得我觊觎他的财富，或佩服他能赚这些钱。我夸赞的永远是他其他能力。

我总让这类男人觉得我“见过世面”。有一次他与我的员工相处后，因他气质谈吐不俗，员工走后我说：“我的员工觉得你很有内涵，他们以为你特别有钱呢。还问我：‘你老公是不是超有钱？’”

这句话的潜台词是：你那几十个小目标，我觉得还好，不算“超有钱”。我常说：“你这打扮，人家真会以为你是超级有钱的人。”通过各种侧面描述按压他，不让他因身家而狂妄。虽然我常提供情绪价值、夸他，但他从不因此骄傲。

我的夸赞有时带有战略性，难以一言蔽之，这需要一定天赋。我会以夸张方式夸他，或一句话蕴含两三种情绪。

何谓夸张地夸？许多人在初级阶段学夸人追求“真实”，但我追求“假”。这点以后有机会可分享。因为这种方式并非一般人能操作，需像我这样带有狂野、不屑气质的人讲出才像样。我教授的内容是降维打击，但只要学好，面对身家几十个“小目标”的男性轻而易举。

初学夸人可先求“真”。到更高层次后，便知夸人也可“很假”。我有时夸人就故意追求假，不让你觉得真，但情绪到位。尤其在公众场合，旨在调节氛围，无需让他信以为真，能做到大家哄堂一笑即可。

言归正传，继续讲负面情绪价值。除了在他骄傲时需按压，在你自己感到极致失望时，或他做了踩踏你原则边界的事时，都需给予负面情绪价值。这不代表它不好或不必要，对关系健康长远发展而言，其实是必要的。

提供负面情绪价值需注意什么？以下是关键点：

*   **注意时效性**：不要等事情过去十天半个月再提，那就成了翻旧账。应在事情发生当下立即提出。
*   **理解其意义**：当他收到负面情绪价值时，未必认为不好。他可能反向认为：“这是因为这个女人特别在意我，看重这段关系。”
*   **监督惩罚执行**：除了言语按压，若需惩罚，必须监督执行到底。
*   **感知关系重要性**：男性有时需在你的适度负面情绪中感知你对关系的看重程度及其重要性。若是不想长期发展的关系，我们无需鞭策。
*   **引导付出的时机**：提供负面情绪价值的时机，也是引导对方付出的好机会。女孩子们应聪明把握。

本节课中我们一起学习了如何提供正向情绪价值（接纳弱点、安抚鼓励）以及必要的负面情绪价值（适时按压、设立边界）。掌握这些方法，能帮助你在关系中更加从容、智慧，促进关系健康持久发展。
