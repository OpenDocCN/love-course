# 《如何成为一个自信的男人》课程4.0.1：06_05 - 人性总是爱撒谎的 👤

在本节课中，我们将探讨一个关于人性的底层逻辑：人为什么以及如何撒谎。理解这一点，对于你与女性相处或与其他男性打交道至关重要。

## 课程概述 📖

上一节我们讨论了其他的人性特点，本节中我们来看看“撒谎”这一普遍行为。我们将分析撒谎的根源、男女在撒谎上的差异，以及几种常见的撒谎方式。掌握这些知识，有助于你在人际交往中保持清醒，避免被误导。

## 人性为何爱撒谎？🤔

人喜欢撒谎，根源在于人性是趋利避害的。为了获取利益或避免伤害，人们会选择性地陈述事实。

## 男人与女人，谁更爱撒谎？⚖️

在说谎这件事上，男人和女人半斤八两，双方都非常擅长，只是方向有所不同。

以下是具体的区别：

*   **女性**更擅长**把没有的东西说成有**。例如，一个女生可以说“我特别喜欢你”，但内心可能对你毫无感觉。
*   **男性**更擅长**把有的东西说成没有**。例如，当被问及是否对其他女性有好感时，绝大多数男人会平静地回答“没有，我只爱你一个人”，尽管内心可能波澜起伏。

## 常见的撒谎方式 🎭

除了上述区别，人类还有几种常见的撒谎策略。

以下是三种典型的撒谎方式：

1.  **断章取义（以局部代替整体）**
    *   **含义**：只陈述部分事实（A），而隐瞒另一部分关键事实（B）。真实情况本是 `A + B`。
    *   **特点**：撒谎者可以坚称自己没说谎，因为他说的“A”是真实的。这种方式常见于女性。

2.  **浑水摸鱼（真话中掺杂假话）**
    *   **含义**：在大量真话（例如99句）中，夹杂一两句关键的假话。
    *   **特点**：极具迷惑性，让人难以分辨。许多“毒鸡汤”或误导性言论都采用此方式。`真话1 + 真话2 + ... + 假话 + 真话99`

3.  **连环套（谎言套谎言）**
    *   **含义**：用一个谎言去圆另一个谎言，形成连环套。
    *   **特点**：复杂度高，容易让人思维混乱，失去判断力。例如，为一件荒谬的事（如“吃屎”）包装上比赛、荣誉和奖励，就会有人趋之若鹜。

## 核心原则与应对策略 🛡️

讲了这么多，核心是想分享一个重要的成年人处世观。

**成年人的世界观，应该是默认每一句话都是谎言。** 知而不信，才是君子之道。

不要对任何一句话抱有不切实际的幻想。你需要：

*   **多观察对方的行为**，而非只听其言。
*   **多用常识去检验你遇到的问题**。这是非常重要的一点。不要被漂亮的话术或复杂的概念绕得头脑发昏。
