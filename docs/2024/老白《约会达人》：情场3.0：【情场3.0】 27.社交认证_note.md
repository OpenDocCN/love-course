# 老白《约会达人》：情场3.0：【情场3.0】 27.社交认证

🎼社交认证是吸引范畴的概念，其重要性不言而喻。社交认证的核心在于展示个人价值，让女生认识到我们是一个猎人（HT猎人）。实现这一目标的方式包括高价值展示（DHV）和两种感觉。

社交认证不同于这两种方式，但效果显著。例如，带女生参加聚会，如果周围的人都对你表示尊重和友好，这实际上是在向女生暗示你是一个受大家认可的人，从而增强你的吸引力。

以下是社交认证的几种方式：

1. **朋友认可**：与朋友一起外出时，注意互相捧场，避免与损友为伍，因为这可能影响你的社交认证。
2. **交往的朋友优秀**：你的交往圈子中朋友都很优秀，侧面反映出你也很优秀。
3. **交往过的女生或身边的女性朋友优秀**：这被称为“玉选”，意味着其他女生已经对你做出判断，你她会忽略自己的判断。

此外，以下因素也属于社交认证：

- **穿着**：你的穿着打扮。
- **出入场所**：你常去的场所。
- **座驾**：你的交通工具。

以下是一个例子：

一个男生花2000元给女生买手机，另一个男生花1500元租豪车带女生吃麻辣烫，结果后者成功吸引了女生。虽然这是个笑话，但说明了钱要花在刀刃上。

以下是一个线下活动的例子：

在酒吧活动中，9个男生和3个导师共12人，用两个卡。由于女生不是同时到达，可以看到一个明显的现象：先到的两个形象气质一般的女生起初不愿交流，只是玩手机。当其他女生陆续到来时，这两个女生的社交窗口突然打开，她们不再玩手机，而是积极参与聊天、喝酒、蹦迪和玩游戏。

从这个例子中可以看出，当男生数量多且质量高时，女生会感到这些男生没有吸引力，不受欢迎。而当女生数量多且质量更高时，男生会变得礼貌、客气，并享受愉快的氛围。

总结：

1. 朋友对你的认可。
2. 交往的朋友都很优秀。
3. 交往过的女生或身边的女性朋友都很优秀。

这些都是社交认证的关键因素。
