# 林老头《系统认知》：024.14、引导与投入，关系的“动力”全部来自于投入_ev

## 概述
在本节课中，我们将学习如何通过引导和投入来增强人际关系中的动力。

## 关键概念
**投入**：指在关系中投入情绪、时间和身体接触，是关系动力的来源。

## 投入的重要性
任何关系没有投入，都等于没有了动力。人往往会越重视自己投入过的付出，因此，投入是关系发展的关键。

## 投入的三大板块
1. **情绪投入**：为对方开心、着急等。
2. **时间投入**：将空余时间花在对方身上。
3. **身体接触**：牵手、接吻等。

## 引导
引导不是命令或请求，而是通过陈述事实和依据来引导对方。

### 引导方法
1. **陈述**：事实 + 依据 = 升高
   - 例如：我们牵过手了，我觉得我们性格合适，可以试试在一起。
2. **讲故事**：目的 + 为什么 = 升温
   - 例如：我觉得我们在一起会非常幸福，因为你的某个行为让我想起了我的过去。

## 夸张的反馈和奖励
通过夸张的反馈和奖励来提供爽感和让对方上头。

### 执行方法
1. **夸**：夸奖对方，具体、真诚。
2. **词**：用词做作、浮夸。
3. **打击痛点**：强化对方的痛点，如责任心、生活品质等。

## 总结
本节课中，我们学习了如何通过引导和投入来增强人际关系中的动力。通过情绪、时间和身体接触的投入，以及合理的引导和夸张的反馈，我们可以使关系更加顺畅和充满动力。
