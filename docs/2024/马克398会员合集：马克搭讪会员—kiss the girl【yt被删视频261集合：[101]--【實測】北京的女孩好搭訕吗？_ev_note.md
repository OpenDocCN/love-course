## 教程：如何进行有效的搭讪

### 概述
在本节课中，我们将学习如何进行有效的搭讪，包括开场白、了解对方、建立联系等步骤。

### 开场白
**公式**：你好，我叫[你的名字]，你叫什么名字？

**代码**：
```python
print("你好，我叫[你的名字]，你叫什么名字？")
```

### 了解对方
**公式**：你有什么兴趣爱好？

**代码**：
```python
print("你有什么兴趣爱好？")
```

### 建立联系
**公式**：我觉得你手很好看，可以加个微信吗？

**代码**：
```python
print("我觉得你手很好看，可以加个微信吗？")
```
