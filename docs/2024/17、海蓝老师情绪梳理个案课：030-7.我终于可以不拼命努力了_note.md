# 海蓝老师情绪梳理个案课：030-7.我终于可以不拼命努力了

## 概述
在本节课中，我们将学习如何通过情绪梳理个案课，帮助学员释放焦虑，接纳自我，并找到轻松自在的生活状态。

## 一、学员背景
学员从小到大一直背负着焦虑和紧张，害怕被人瞧不起，无法放过自己。通过情绪梳理个案课，学员成功卸下了这些负担，感受到了轻松和自在。

## 二、情绪梳理过程
以下是情绪梳理个案课的具体过程：

### 1. 卸载负面情绪
学员通过想象的方式，将过去所有负面情绪（如焦虑、紧张、害怕被人瞧不起等）全部卸载掉。

### 2. 吸引积极品质
学员希望将轻松、自在等积极品质吸引到身体里，让它们住在心里。

### 3. 允许和接纳自我
学员学会了不再强迫自己，允许和接纳自己的懈怠、放松等状态。

### 4. 与家人相处
学员希望与家人建立良好的关系，将精力放在与家人的相处上。

## 三、核心概念
**公式**：情绪梳理 = 卸载负面情绪 + 吸引积极品质 + 允许和接纳自我

## 四、学员收获
通过情绪梳理个案课，学员成功释放了焦虑，接纳了自我，并找到了轻松自在的生活状态。以下是学员的收获：

### 1. 感受到轻松和自在
学员在情绪梳理过程中，感受到了轻松和自在，内心变得更加平静。

### 2. 接纳自我
学员学会了允许和接纳自己的懈怠、放松等状态，不再强迫自己。

### 3. 与家人建立良好关系
学员希望与家人建立良好的关系，将精力放在与家人的相处上。

## 五、总结
本节课中，我们一起学习了如何通过情绪梳理个案课，帮助学员释放焦虑，接纳自我，并找到轻松自在的生活状态。希望学员能够将所学应用到实际生活中，过上更加幸福、自在的生活。
