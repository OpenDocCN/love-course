# 17、海蓝老师情绪梳理个案课：030-7.我终于可以不拼命努力了_note

在本节课中，我们将跟随一位学员的个案，学习如何识别、卸载长期积压的焦虑与自我强迫，并学习如何邀请轻松、接纳的品质进入内心。整个过程将展示一种内在对话与情绪转化的方法。

---

上一节我们介绍了课程的整体目标，本节中我们来看看具体的梳理过程。学员描述了一种长期存在的感受。

我想把从小到现在，所背负的焦虑、紧张、觉得自己不够好、怕被人瞧不起、不能放过自己等所有这一切，全都卸载掉。只留下轻松和自在。这些感受原本存在于头脑和心口。

那么，是否想将它们都卸载掉？卸载的方式可以有风、水、光、火、图画或其他任何方式。你希望用哪种方式，将过去所有这些“时间不够、着急、怕被人瞧不起、放不下”的东西全部卸载？

以下是学员选择的卸载方式：
*   **方式**：用“火”烧掉。
*   **过程**：想象父亲在帮忙。父亲帮我烧掉了这些东西。我过去“不要活成爸爸那样”的沉重负担也随之烧掉。

卸载完成后，我们转向下一个步骤。上一节我们卸载了沉重负担，本节中我们来看看需要邀请什么进来。

希望把什么品质、素质或状态吸引并邀请到身体里面？答案是：轻松自在。

那么，让“轻松”进来，并倾注在身体的什么地方？让轻松住在心里。当轻松住下后，身体感到发热。

还需要邀请什么品质？答案是：不再强迫自己。这本质上是**允许和接纳自己**。

那么，把“接纳和允许”放在身体的什么位置？放在心里。充分地放在心里。

还希望什么品质？通过深呼吸，充分感受这种轻松，让它留在身体的每一个位置。

因为“放不过自己”这个部分已经跟随你49年了。虽然它挺想休假，你也准备带它去巴厘岛（或它带你去）。相信这49年的“工作”它也挺辛苦、不容易。也许以后它还会再出来。

那么，它接下来希望你怎么与它相处？这个“放不过自己”的部分（被比喻为“小东西”）说，如果再出来，希望你能劝它回去继续休假。它是功臣，需要倾听它的愿望。

它说，以后可以把努力放在跟家人的关系上。听到这里，学员流下了眼泪。它建议我们换一种工作方式，不要把精力放在别处，而是放在地上，多去看看妈妈，处理好和家里人的关系。学员认为这个建议很对，愿意听从，并称赞其智慧。

现在，感受一下脑袋和心口的状态。最开始脑袋是懵的，心口是堵的。现在感觉头脑清晰了，但心里有点疼。

询问内心为何有点疼。回答是，它有点“舍不得”。但我们已经决定把时间精力不往那儿（强迫自己）放了，要回去多看看妈妈。学员表示知道了，会回去落实。

现在再感受一下。心里的疼感消失了，感觉心变大了，是一种敞亮、能容纳很多东西的感觉。

在结束之前，看看内在是否还有其他需要交流或安抚的部分。可以到此结束。

吸气，呼气，准备睁开眼睛。学员有些舍不得睁眼。可以随时闭上眼睛，重新进入这种轻松或接纳自己的状态。

回顾整个过程，分享此时此刻的感受。学员的感悟转向了与父母的关系。

我不知道对我妈妈的怨恨，以及对我爸爸的看法，是否真的如我所想。我今天一下子觉得卡住了，看到了他们之间那种离不开的情感，并不是我以前看到的样子。原来都是互相惦记。我以前觉得妈妈烧纸是出于内疚，现在我知道，那也是因为爱。没有爱，不会有内疚。

回去要好好照顾妈妈。这些年来，其实没有好好数算过。我一直跟妈妈对着干，虽然没吵过，但心里确实如此。我知道妈妈为什么那样了，她也能感受到我的对抗。

学员觉得梳理的主题似乎从“焦虑”跑题到了“妈妈的问题”。但引导者指出，从来不会跑题。

那么，看到“时间不够”这件事的背后，到底是什么？学员现在看到，其实是想怕别人瞧不起，希望别人能瞧得起。为此非常费劲，总觉得时间不够。每天必须完成作业，在群里也怕表现不好，一直觉得一定要争第一，怕让人看不起。

今天看到这背后的根源是：**不接纳自己**。是和一个像妈妈一样要强、停不下来的自己在一起。内心又有着像爸爸那样（自认为）被瞧不起的部分。实际上，妈妈心里也是怕人瞧不起，他们的努力源于此。

今天整个过程中，对你最有帮助的几个点是什么？学员总结道：
*   **核心收获**：我允许我自己，也能够接纳我自己了。
*   **转变过程**：当引导者提问“谁在要求你”时，我发现，老公没说我，其他人也没要求我做成什么样子。只有我自己在要求我自己。所以我现在觉得，应该接纳自己的懈怠或放松，原先我觉得这是不可以的，不允许自己比别人差。

那么这个“怕人瞧不起”的根源，最深的地方在哪儿？学员认为，最深的根源是害怕活成爸爸的样子，觉得爸爸活得太窝囊、太吓人了一辈子。

但实际上，爸爸的感觉和你并不是一回事。你只看到他挨骂的时候，没看到他好的时候。他并非什么都没做，他没有“拼命努力”，但他在家做饭，好好爱家人了。这也是好的。

最后，回顾整个案例，还有没有想补充的帮助点？学员提到，最核心的部分就是觉得“时间不够”。

**本节课总结** 🎯

本节课中，我们一起学习了一个完整的情绪梳理个案。我们从**识别并卸载**长期的焦虑与自我强迫开始，通过象征性的方式（如“火”）处理了这些负担。接着，我们学习了主动**邀请和安置**“轻松”、“自我接纳”等积极品质到内心。通过与内在那个“放不过自己”的部分对话，我们理解了它的辛苦，并协商了新的相处方式：将努力的方向转向**改善与家人的关系**。在回顾中，学员洞察到“时间不够”和“必须优秀”背后的深层根源是**对父母的复杂情感与自我不接纳**，并最终找到了**允许和接纳自己**的钥匙。整个过程展示了如何通过内在对话，将压力转化为对自我的理解与关爱。
