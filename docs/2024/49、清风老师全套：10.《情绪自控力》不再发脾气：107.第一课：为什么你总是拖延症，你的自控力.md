# 49、清风老师全套：10.《情绪自控力》不再发脾气：107.第一课：为什么你总是拖延症，你的自控力

在本节课中，我们将探讨自控力的本质，识别影响自控力的关键因素，并学习如何开始培养和提升它。

你是否在日常生活中遇到过以下困境？

以下是三种常见的困境：

1.  想要改善关系，却因无法控制情绪而争吵，或在挽回时做出下跪、疯狂联系等过激行为。
2.  面对重要紧急的任务，总是拖延，最终导致被批评或自我否定。
3.  决心学习或运动，却无法集中注意力或坚持，轻易放弃并抵消努力成果。

这些困境均由缺乏自控力导致。要改善现状，需要首先纠正对自控力的错误理解。

以下是关于自控力的三个常见误区：

*   **误区一：自控力是无限的。** 人们常认为自控力取之不尽。事实上，自控力像精力一样，有高峰、低谷和极限。
*   **误区二：自控力是天生的。** 有人认为意志力强弱是天生的。事实上，自控力是可以通过后天学习和练习获得的。例如，讲师本人通过学习和实践，已持续基金定投超过 **1665天**，改变了过去在金钱上缺乏规划的习惯。
*   **误区三：自控力就是对抗自己。** 自控常被等同于痛苦地强迫自己，像苦行僧。真正的自控力是顺应人性，运用科学方法找到良好状态，而非陷入痛苦。

那么，真正的自控力是什么？可以将其比喻为**一块肌肉**。它既需要训练来增强，也需要休息来恢复，使用时需避免透支。良好的自控力是**顺应并利用人性**，而非简单地违背它。

理解了自控力的本质后，我们来看看哪些因素会影响它的状态。

以下是影响自控力的三个关键因素：

1.  **情绪**：在情绪低落、愤怒或失望时，自控力会变得薄弱，更容易做出冲动行为（如情绪化饮食）。因此，正确处理情绪是建立自控力的基础。
2.  **饮食**：良好的饮食能显著提升自控力。在饥饿状态下，自控力会下降，可能导致非理性决策。
3.  **休息**：疲劳时自控力表现较差；精力充沛时则更强。休息不足容易陷入“休息差 -> 情绪低落 -> 自控力差 -> 行为失控 -> 情绪更差”的恶性循环。

了解了影响因素，我们该如何系统地培养自控力呢？本专栏将分为三部分讲解。

以下是培养自控力的三个核心部分：

*   **第一部分：认识自控力**。首先要了解自控力的极限。就像运动时身体达到极限会发出“我不行了”的信号一样，自控力也有极限。通过正确训练，不仅可以把握极限，还能突破它，进入心理学家米哈里提出的 **“心流”** 状态。在这种状态下，人能极度专注于当前任务，高效达成目标并获得成就感。
*   **第二部分：正确规划自控力**。制定计划时常见的错误是要求绝对化（如“一个月绝不发脾气”）或排满日程不留休息时间，这都高估了自控力，会导致失败。好的规划应**合理且有弹性**。此外，避免孤军奋战。自控力的提升不是单机游戏，找到伙伴（伴侣、同事、家人）共同成长会更有效。以身作则比言语说教更能影响他人。
*   **第三部分：锻炼自控力**。自控力肌肉需要方法锻炼。以下是五件可以帮助你快速入门的小事：
    1.  **吃早饭**：保障良好饮食与休息。
    2.  **花3分钟认真刷牙**：通过小习惯提升对健康的重视和自控管理。
    3.  **睡前将手机放到别的房间**：改善睡眠质量。
    4.  **每天做10分钟冥想**：显著提升自控力。
    5.  **独自吃饭时不看手机**：避免通过刷手机掩饰内心孤独或焦虑，防止自控力下降。

在本专栏的十节课中，我将帮助你提升自控力，具体包括：管理焦虑情绪、规划饮食与休息、克服拖延、走出情绪低谷、成为高效能人士、改善亲密关系等。最终目标是让你掌握时间与精力，过上想要的生活。
