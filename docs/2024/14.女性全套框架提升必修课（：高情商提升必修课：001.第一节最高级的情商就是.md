## 高情商提升必修课：001. 最高级的情商就是说他想听的话

在本节课中，我们将学习高情商沟通的核心原则：说对方想听的话。我们将探讨如何通过有效的倾听和回应，在异性交往及各种人际关系中建立良好互动，提升个人魅力。

### 一、 沟通差异的根源

上一节我们介绍了课程的整体目标，本节中我们来看看男女沟通差异的根本原因。

我们必须承认，男人和女人的思维方式生来不同。在现代社会快节奏的生活中，认识异性的途径有限，多通过朋友介绍或碎片化社交活动（如聚会、剧本杀）。在这些场合，双方都希望在有限时间内达成目的——挑选心仪的异性。

以下是双方行为模式的差异分析：

*   **男性的筛选模式**：男性首先被女性的**外表**（颜值、穿着打扮）吸引。符合其口味的女性会获得他们进一步的关注和互动（如主动交谈、索要联系方式）；不符合的则会被第一时间淘汰，以便集中精力。
*   **女性的筛选模式**：女性虽对外表有偏好，但通常不会表现得过于明显。她们更倾向于让自己处于“被选择”的位置，希望吸引更多男性的注意，最终从选择自己的人中，挑选综合评分最高的一位。

经过分析可以发现，在交往初期，男性往往将自己置于**主动选择方**，而女性则习惯将自己置于**被动被选择方**。双方若未意识到这种差异，良好互动的基础从一开始就可能丧失。

### 二、 如何开启有效对话

了解了差异后，我们来看看如何成功开启一段对话。有效的沟通是相互的过程，而非单方面的盘查。

许多女性在与男性交谈时，容易陷入“盘查式”的低效沟通。例如：
```plaintext
女：“你多大了？”
男：“26。”
女：“哦。那你平时喜欢干什么？”
男：“打打游戏，运动一下。”
女：“那你家里几口人？”
```
这样的对话没有信息交流，也无法留下好印象。

在一个有效的沟通中，我们应该关注的不是自己，而是**听你讲话的那个人**。多思考对方希望听到怎样的问题或回应。每个人都喜欢谈论自己的兴趣，并希望获得积极反馈。

以下是基于对方兴趣点开启话题的方法：

1.  **给予正面评价，引导深入**：当对方分享兴趣后，首先给予正面评价。
    *   **示例**：对方说“我喜欢打游戏和运动。”你可以回应：“我觉得爱打游戏的人通常很有趣，爱运动的人很自律。那你一定是个既懂生活又自律的人。”接着追问：“你最喜欢玩什么游戏？”或“你平时做什么运动？”

2.  **主动分享，满足好奇心**：对于主动与你聊天的男性，他希望能了解你。如果你也有好感，可以先大方分享关于自己的事，满足他的好奇心，再将话题引回对方。
    *   **示例**：约会时可以说：“你也喜欢喝咖啡呀？我平时也天天喝，不过不是为了提神。”（停顿，等他询问）然后接着说：“因为喝咖啡能消肿，对我比运动还有效。对了，你平时爱喝什么咖啡？”

### 三、 提升接话能力：改变思维方式

成功开启话题后，接下来面临如何接话的挑战。很多人在对方侃侃而谈时，反应不过来，只能憋出“你说的对”之类的话，让气氛冷却。

接话反应慢，通常是因为**思考方式**有问题。根据《慢思考》一书，人脑可分为三部分：
*   **反射脑**：对感官世界快速、无意识反应。
*   **思考脑**：负责逻辑分析、总结预测。
*   **存储脑**：负责信息筛选、分类与存储。

所谓反应慢的人，常在沟通中下意识使用**思考脑**，对问题进行过度逻辑分析，以确保回答无误，但这降低了沟通速度。

**解决方案**：将需要快速回答的问题交给**反射脑**和**存储脑**处理，绕开思考脑，避免过度思考。

具体步骤如下：
1.  **感受与想象**：听到问题后，先用感性思维感受。
2.  **回忆类似经历**：用存储脑寻找类似情景或做法。
3.  **提炼新观点**：根据经验提炼观点。
4.  **整合成回答**：将以上整合成最终回应。

**示例分析**：记者问走清纯路线的女明星：“你对吻戏什么看法？”
*   **标准回答**：“吻戏是影视剧常见戏码，是推动剧情的重要因素。我小时候看的很多经典剧，吻戏也很自然。作为演员，如果我接到有吻戏的角色，会认真完成，这是演员的本分。”
*   **话术逻辑**：未直接引向自己，先回忆相关经验（经典剧），对“吻戏”做正面评价，最后回归自身职业，说明做法并提升形象（职业道德）。

### 四、 运用金字塔原理组织语言

有时我们觉得说话没逻辑，可能是因为缺少结构。这里可以运用**金字塔原理**来组织语言，它包含两个逻辑方向：
*   **纵向**：自上而下，结论先行。
*   **横向**：按演绎或归纳推理分组，相当于归类。

**应用示例**：当男性问你：“你觉得现在的渣女多吗？”
你可以这样组织回答：
1.  **结论先行**：花不花心是因人而异的。
2.  **分论点一**：朋友小A不花心。
    *   **论据**：她虽在虚拟世界“老公”无数，但现实中恋爱时，会主动与其他男性保持距离。
3.  **分论点二**：朋友小B很花心。
    *   **论据**：她不追星，却在实际生活中同时交往多人，备胎无数。
4.  **回归自身并总结**：“所以说，女人好色是天生的，但渣与否因人而异。对我来说，一对一是对恋爱最起码的尊重。喜欢看帅哥美女是人之常情，但需把握分寸。”最后可加一句玩笑缓和气氛：“所以说啊，男人也得擦亮眼睛。”

### 五、 抓住重点：利用注意力法则

健谈不等于会说话。有时多说话不如少说话，关键在于抓住重点。心理学上的**注意力法则**指出，良好沟通应在最短时间内最大程度获取对方注意力，精准表达重点。

**避免抓不住重点的沟通**：例如，想约心仪的男生吃饭，如果说：“哎，其实我有点事…因为你最近有空…我也忙完了…以前怕你忙…明天一起吃个饭吧…你胃不舒服…我们去吃周蹄火锅…那家店在你公司对面…你觉得怎么样？”这段话絮叨冗长，关键信息（时间、地点）不突出。

**改进方法：使用3W结构**
传递信息时，可使用**3W结构**：
*   **What**：核心观点/事件是什么。
*   **Why**：原因或重要性。
*   **How**：需要怎么做。

**优化后的约饭说辞**：“明天你下班后，6点半，我们在你公司对面的某某周蹄火锅约个晚饭吧。因为你之前说胃不舒服，这家比较养生。我们也好久没见了，正好聊聊。时间OK的话，下班前告诉我一声。”
这样表达清晰、准确，便于对方快速理解和决策。

### 六、 精准回应与倾听技巧

抓不住重点的另一个原因，是未关注对方说话内容，特别是**关键词**。男性尤其希望得到关注，如果你的回应总是偏离重点，他会觉得不被重视。

**示例**：男朋友说：“最近工作好忙，头都抬不起来，经常想回你消息时又被新任务打断。”
*   **错误重点**：聚焦于“忙”，并给出解决方案（如“让同事分担”、“午休时回消息”）。
*   **真实重点**：他因“忙”而“没空回消息”，潜台词是希望你不要因此生气。
*   **正确回应**：“没关系，我知道你工作忙，不会怪你的。你安心工作，别分心。”这样回应表明你理解他，关系会更亲近。

**两个提升倾听与回应效果的小技巧：**

1.  **有效重复**：在肯定对方的基础上提出见解，而非机械重复。
    *   **示例**：对方说：“今年上映的几部科幻片真让人失望。”你可以回应：“是的，最近这几部科幻片质量确实一般。不过有几部科幻电视剧倒挺不错的，你看过吗？”这样既肯定了对方，又延续了话题。

2.  **避免“偷话题”**：男性要强且享受被关注，特别讨厌话题被偷走。
    *   **错误示例**：对方正兴致勃勃谈论自己的成功项目，你突然插话：“我们公司也有个人很厉害，他之前帮我…”然后开始大谈自己的事。
    *   **正确做法**：当对方讲述自己的高光时刻时，仔细聆听，给予充分肯定和关注即可。
