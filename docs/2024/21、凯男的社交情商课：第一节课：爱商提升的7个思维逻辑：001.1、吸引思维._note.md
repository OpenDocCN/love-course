# 21、凯男的社交情商课：第一节课：爱商提升的7个思维逻辑：001.1、吸引思维._note

在本节课中，我们将要学习恋爱思维的第一课——吸引思维。我们将探讨为何传统的“追求”方式常常失败，并揭示建立真正吸引力的核心逻辑。

## 概述：为何“追求”往往无效？

在多年的情感咨询工作中，我遇到学员最常问的一个问题是：“为什么我对他/她这么好，对方还是不答应和我在一起？” 例如，有学员提到，他喜欢一个女生五个月，期间天天嘘寒问暖、送礼物、请吃饭、对方生病时还炖汤，但最终只换来一句“没有感觉”。

之所以出现这种问题，根源在于恋爱思维存在误区。解决问题需治本。现在，我要告诉所有朋友一个核心观点：**人永远不是追来的，而是吸引来的**。记下这句话。

## 社会背景与思维困境

首先，我们来看一组有趣的数据。2015年末人口普查显示，中国总人口约13.7亿，其中男性约7亿，女性约6.7亿，男性比女性多出约3000万。这意味着每出生100个女孩，就有约105个男孩出生。到2016年，男性人口超出女性30%以上，而“光棍”总数也从2012年的1000万上升至2022年的3000多万。目前，每出生100个女孩，对应的男孩数已达到116个。

在这种男多女少的大环境下，男性面临巨大的婚恋压力。因此，许多人倾向于思考如何“快速追求”一个女生以达到目的。然而，我们从小到大可能学到的唯一吸引女性的方式就是“追求”：对她好、买礼物，认为最终能感动她，让她和自己在一起。

但结果往往是，对方会说“你是个好人，但我们不合适”或“我对你没感觉，还是做朋友吧”。所以，你必须明确一个概念：**感动不是爱情**。

## 什么是爱情与吸引？

爱情的本质是相互吸引。它源于你自身的价值，例如：
*   你的事业心强。
*   你有主见。
*   你拥有成熟的恋爱观。
*   你性格稳重。

因为你拥有这些特质，女性在与你交往时会欣赏你、崇拜你，从而对你心动。这才是吸引。

## 剖析“追求”：行为与错误假定

接下来，我们深入分析“追求”行为到底错在哪里。首先，我们把“追求”二字拆开理解。

“追”通常表现为哪些行为呢？以下是典型过程：

1.  **查户口式提问**：例如，“你是哪里人？”、“你在哪里工作？”、“你喜欢做什么？”，然后紧接着介绍自己。
2.  **间歇性定点查岗与关心讨好**：例如，频繁发送“在干嘛？”、“吃饭了吗？”、“记得多喝热水”、“我给你买了个礼物，记得查收”。
3.  **邀约**：例如，“我们一起吃饭/看电影/爬山吧”。
4.  **循环与升级**：如果邀约失败，则可能陷入“查岗+邀约”的循环。当所有常用招数用尽后，便会放出“大招”——**直接表白**，甚至带有哀求语气，例如“做我女朋友好不好？”、“求你再给我一次机会”。

为什么这些做法难以成功？因为在“追”和“求”的观念指导下，你的行为通常基于以下六种错误的前提假定。

以下是六种常见的错误思维假定：

1.  **错误假定一：对她好就能换来爱情**
    *   **想法**：如果我对她特别好，帮她做很多事，她就会看到未来交往或结婚后我也会如此对她，从而被感动并喜欢我。
    *   **例子**：她没吃饭，你主动买；她工作没做完，你免费帮忙；她家电脑坏了，你立刻去修。
    *   **真相**：如果你只会通过付出和帮忙来对她好，她会觉得和你在一起很无趣。因为你和其他采用同样方式的男性没有区别，对她而言已毫无挑战性。就像玩一个总能轻松赢的游戏，你很快就会失去兴趣。**轻易得到的东西不会珍惜**。

2.  **错误假定二：永远同意就能避免矛盾**
    *   **想法**：如果我同意她的每件事、每个观点，我们就能避免矛盾，她会觉得我宠她、感激我，这样我们才是完美的一对，她才会爱上我。
    *   **例子**：她说“我脾气不好”，你回答“没关系，我宠着你”；她说“你太逊了”，你回答“嗯，你厉害”；她说“我喜欢包容我坏脾气的男人”，你回答“你说的不就是我吗？”
    *   **真相**：这其实是在违背自我、讨好对方。如果你为了讨好女性而放弃自己的观点和信念，在她眼中，你会成为一个没有主见、虚伪、见风使舵的男人，从而对你失去信心。事实上，当你坚持自己不同的观念时，女性表面可能不快，但心底里反而会产生敬佩和信任，从而更喜欢你。

3.  **错误假定三：联系越多越好**
    *   **想法**：我要多打电话、多发信息关心她，多接触让她更了解我，她才会喜欢我。
    *   **例子**：每天准时发早安晚安，叮嘱她穿衣喝水，试图24小时保持对话。
    *   **真相**：这种心态如同鲨鱼闻到血腥味，急切地想一次做到位。但关键在于，成功的标志**不是你为她付出了多少时间和精力，而是你占据她头脑的时间有多少**。让一个女人爱上你的核心，在于**她为你投入了多少**，而不是你为她投资了多少。

4.  **错误假定四：表白就能赢得芳心**
    *   **想法**：如果我喜欢她，就要表白。只有表现出强烈的爱意，她才会因此喜欢上我、爱上我。
    *   **例子**：直接说“我喜欢你很久了，做我女朋友吧”。
    *   **真相**：请想象一下，一个你完全不感兴趣的女性（可以具体想象一个不符合你审美或喜好的形象）疯狂向你示爱，你会因此爱上她吗？**永远不会**。那么，你为什么期待女性会因为你的表白而喜欢你呢？

5.  **错误假定五：送礼就能打动她**
    *   **想法**：我给她买东西、送礼物，她就会喜欢上我、跟我在一起。
    *   **例子**：省吃俭用两个月为她买名牌包或手表。
    *   **真相**：这种行为在她潜意识里建立了一个思维定式：**你需要用物质来争取她，否则她就不会喜欢你**。这背后的根源是你认为自己本身不值得她喜欢，所以需要借助外在物品。这样做，等于提前把选择权完全交给了对方。

6.  **错误假定六：有钱或帅就足够了**
    *   **想法**：如果我长得帅或者很有钱，她就会喜欢我。
    *   **真相**：帅气和财富确实是有利条件，不会阻碍吸引，但它们只占吸引力的很小一部分，甚至可以忽略不计。对比一个有钱/帅的男人和一个有吸引力、有人格魅力的男人，女性明显更容易被后者吸引。现实中，你可以看到有钱人搂着女伴却未必有爱情，也可以看到其貌不扬的“穷小子”拥有令人羡慕的伴侣，且对方愿意为他付出一切。

## 核心总结与课程预告

综上所述，许多男性并不清楚什么才是最吸引女性的。过去，我们可能受影视剧影响，认为只要拼命对女方好就能感动对方、终成眷属。但那只是编剧为迎合观众期望而设计的情节。

在现实中，如果你只懂得“追求”，很可能只得到“你是个好人，但我们不合适”的回答。如果你想学习如何真正吸引女性，那么第一课就必须**解放你的思想**。
