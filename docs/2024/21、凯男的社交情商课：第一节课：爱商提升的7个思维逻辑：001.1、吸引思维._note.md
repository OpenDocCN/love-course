# 21、凯男的社交情商课：第一节课：爱商提升的7个思维逻辑：001.1、吸引思维

## 概述
在本节课中，我们将学习如何通过吸引思维提升爱商，从而在恋爱中少走弯路。

## 吸引思维
### 数据分析
2015年年末，中国总人口为13.7亿，其中男性7亿，女性6.7亿。2016年，男性人口超出女性人口30%以上，光棍人数由2012年的1000万上升至2022年的3000多万。

### 吸引思维的重要性
在男多女少的大环境下，男性面临巨大压力。因此，我们需要转变思维，从追求转变为吸引。

### 吸引思维的核心
1. **价值吸引**：拥有事业心、主见、成熟的恋爱观等特质，使女性欣赏、崇拜并心动。
2. **相互吸引**：爱情是相互的吸引，而非单方面的追求。

### 追求思维的误区
1. **过度付出**：只会对对方好，帮助对方，让对方觉得无聊，失去挑战性。
2. **迎合对方**：同意对方的所有观点，失去主见，让对方失去信心。
3. **过度联系**：频繁联系对方，让对方感到压力，而非吸引。
4. **表白过早**：过早表白，让对方感到压力，而非吸引。
5. **物质吸引**：用物质去打动对方，让对方觉得你本身不值得喜欢。
6. **外貌和财富**：外貌和财富只是吸引的一部分，人格魅力更重要。

### 吸引思维的正确方法
1. **展现自身价值**：展示自己的优点和特质，吸引对方。
2. **建立联系**：与对方建立良好的沟通和互动，让对方了解你。
3. **保持神秘感**：不要让对方完全了解你，保持一定的神秘感。
4. **共同成长**：与对方共同成长，让对方感受到你的价值。

## 总结
本节课中，我们学习了吸引思维的重要性以及追求思维的误区。通过转变思维，我们可以更好地吸引对方，提升爱商。在下一节课中，我们将学习步骤思维，帮助大家更好地掌握恋爱技巧。
