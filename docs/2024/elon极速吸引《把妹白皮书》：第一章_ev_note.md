# Elon极速吸引《把妹白皮书》：第一章

## 概述
在本节课中，我们将学习把妹的先决条件，即满足把妹的两个价值：外在价值和内在价值。

## 第一节：把妹的先决条件
把妹的先决条件是满足两个价值：外在价值和内在价值。

### 外在价值
外在价值包括身高、长相、学历、职业等因素，这些因素决定了你能否接触到对方。

### 内在价值
内在价值指的是你的态度、状态、谈吐和世界观等，这些决定了你是否具有吸引力。

## 第二节：吸引力的原理
吸引力的原理是：外在价值决定你是否能接触到对方，而内在价值决定你是否能吸引对方。

### 案例分析
以下是一个真实的案例：

- B是一位外在价值很强的男士，毕业于美国名校，有钱有品位，长相帅气。在与女生见面之前，女生对他非常感兴趣。但在见面之后，女生的兴趣明显下降。
- Elon和一龙虽然外在价值不如B，但在见面后很快获得了女生的强烈吸引。

### 吸引力的判断
如何判断自己的吸引力是否足够？如果女生总是以各种理由推脱约会，即使愿意牵手或接吻，但推进关系困难，那么说明吸引力不足。

## 第三节：女性的标准
女性的标准是留给不喜欢的人，如果她喜欢你，自然就没有标准。

### 案例分析
以下是一个案例：

- 一位女士认为自己的条件很好，但男生却不喜欢她。这位女士对待男生和对待其他朋友的态度有很大区别。

## 第四节：吸引力的来源
吸引力的来源是内在价值，而不是外在价值。

### 案例分析
以下是一个案例：

- 西罗和一位巴西超模在一起，西罗非常喜欢这位超模，但超模却对她百般嫌弃。

## 总结
本节课中，我们学习了把妹的先决条件，即满足外在价值和内在价值。吸引力的来源是内在价值，而不是外在价值。通过提升内在价值，我们可以获得更强的吸引力。
