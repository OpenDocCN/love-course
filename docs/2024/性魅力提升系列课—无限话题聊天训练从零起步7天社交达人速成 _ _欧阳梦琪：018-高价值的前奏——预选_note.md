# 性魅力提升系列课—无限话题聊天训练从零起步7天社交达人速成 _ _欧阳梦琪：018-高价值的前奏——预选

🎼大家好，我是欧阳梦琪，欢迎回来。不同人拥有的资源存在两极分化，即马太效应。社交资源丰富的人，其资源质量也越好。缺少社交资源的人，遇见的多数是低价值的人。但有些人长相一般、没有钱，却能吸引社交资源。这是因为雄性看重繁衍价值，雌性看重生存价值。繁衍价值易见，如身材外形等。而生存价值不易察觉，如权势、社会资源调动能力等。了解一个人需要时间和精力。若有一种方式能让一个人快速判断另一个人的生存价值，那么对方可能会迅速被吸引。这种能够让对方快速预估生存价值和性格价值的标志，就是**预选**。

**预选**意味着预先被选择。当一个人周围有一些高价值的人时，就相当于这个人已经被高价值群体选择。即使你和这些人没有发展关系，他们的存在也能让你在陌生人面前产生竞争压力，认为自己需要更加优秀才能引起你的兴趣。这就像应聘职位，如果应聘者众多，你会感到压力，同时也会认为这个职位更有吸引力。

**预选**会让别人把你身边人的高价值转移到你身上，形成对你的价值认可。这是一种无需求的高价值展示，能迅速建立吸引。当一个人看到你身边环绕着很多竞争对手时，他会认为你的选择范围很广，你已经见识过各种各样的人。这时，你的吸引会达到一个制高点，因为你的局势不再是追逐资源，而是谁有办法赢得你的关注。

**预选**会形成别人对你价值的基本认知，这种认知往往会在他脑子里根深蒂固。如果你觉得自己没有资源可以做预选，那么你需要通过自身建设来积累预选。多参加活动，结交朋友，多为朋友提供便利，展示自己的生活，释放人格魅力和自信。当你与外界交流增多，你会懂得如何为别人提供价值，你的圈子会越来越丰富，预选自然而然就会产生。

**课后总结**：

今天我们讲的是**预选**。**预选**是指预先被选择的意思。当一个人周围有一些关系亲密的高价值群体时，就证明了这个人已经通过了高价值群体或其他人的预选。在现实世界里，一个人的生存价值无法让对方一下子发现，而你拥有了预选，就相当于已经有其他人对你的生存价值和性格价值做出了认可。别人会通过你的预选来评估你的价值。**预选**也是一个人身上最强的吸引开关之一。因此，我们需要通过自身建设来积累预选，多参加活动，多和不同的圈子交流，不要抱有任何需求感，用交朋友的心态去做这件事。但也不要把重心放在别人身上，而是放在提升自己的生活质量和展示自己的样本上，渐渐你会发现你开始有了自己的预选，你也是一个可以展现出生存价值的人。
