# 老白《约会达人》：1、如何认识更多优秀异性

欢迎来到约会高手系列课程。本节课我们将探讨如何获取大量资源，以便认识更多优秀异性。

## 获取资源的方式

首先，我们将收集资源分为两种形式：线上和线下。

### 线上资源

线上资源主要通过网络APP软件获取，例如：

- **探探**：一款社交APP，大量女生曾集中在此，但随着时间推移，优质女生逐渐减少。
- **积木**：类似探探的社交APP，女性用户质量较高，但存在假人和微商。
- **送**：社交APP，功能与探探类似。
- **牵手**：以相亲为目的的社交APP，适合寻找结婚对象。

### 线下资源

线下资源主要通过以下方式获取：

- **搭讪**：主动与陌生人交流，认识异性。
- **朋友介绍**：通过朋友介绍认识异性。
- **聚会**：参加聚会认识异性。
- **相亲**：通过相亲认识异性。

## 软件使用技巧

### 探探和积木

- **探探**：在寒暑假期间，海外留学女生和学生党较多。
- **积木**：女性用户质量较高，但存在假人和微商。

### Sosooul

- 用户众多，类型多样，包括语音聊天、文字聊天和群聊派对。
- 附近人功能随机配对聊天，质量参差不齐。

### 牵手

- 以相亲为目的，适合寻找结婚对象。
- 不允许提微信，可多聊一些其他方式。

## 资源管理

### 窗口大小

- **大窗口**：女生对你反应良好。
- **小窗口**：女生对你反应一般。
- **无窗口**：女生对你无反应。

### 资源维护

- **朋友圈**：通过发朋友圈吸引女生关注。
- **互动**：保持与女生的联系，避免长时间不聊天。
- **邀约**：及时邀约窗口大的女生。
- **评论**：在朋友圈评论女生，而非点赞。

### 私聊转移

- 尽量将聊天转移到微信，避免在社交软件上聊天过多。

## 总结

本节课我们学习了如何获取和利用资源，以及如何管理资源。希望这些技巧能帮助大家认识更多优秀异性。
