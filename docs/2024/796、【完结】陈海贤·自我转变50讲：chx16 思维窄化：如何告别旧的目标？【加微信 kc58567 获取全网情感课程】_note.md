## 告别旧的目标

### 目标的意义

目标是自我重要的一部分，它连接着我们的过去和未来，定义着我们的生活方式和意义感。

**公式**：目标 = 过去经验 + 未来期望

### 思维窄化

当我们将所有注意力集中在目标上时，我们就会陷入思维窄化，只看到与目标相关的事物，而忽略了生活的其他方面。

**代码**：思维窄化 = 目标聚焦 - 生活多样性

### 突破思维窄化

为了突破思维窄化，我们需要重新定义自我，不再以成功或失败来衡量自己。

**公式**：新自我 = 追求梦想的人 + 不惧失败的人 + 接纳现实的人

### 例子

电影《心灵奇旅》中的主角，通过经历失去梦想的痛苦，最终意识到生活中还有许多其他重要的东西。
