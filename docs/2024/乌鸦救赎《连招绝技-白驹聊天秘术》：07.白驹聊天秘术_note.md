# 乌鸦救赎《连招绝技-白驹聊天秘术》：07. 白驹聊天秘术 🚀

在本节课中，我们将要学习“连招三”中的核心策略——意愿锁定。我们将深入探讨其背后的哲学原理、具体功能以及实际操作方法，帮助你掌握高效建立恋爱框架的技巧。

## 概述：学习的根本原理

上一节我们介绍了连招的基本概念，本节中我们来看看如何通过掌握原理来精通任何领域。

学习的原理只有两个。第一个是搞懂这个领域的原理。第二个是你要专注一个月到两个月的时间，只研究这个东西。你需要把这个领域的核心基本论，也就是他们的第一性原理，花两个月时间反复钻研透彻。

当你把基本功学扎实后，那些所谓的高阶理论，你一听就会明白。因为你用基本理论一套用，就知道它是哪些基本方法的重新组合。所谓的高招和牛逼的技术，很多时候就是基本方法的重新组合。所以，一旦你的基本功很扎实，高阶方法你一听就懂，因为它是通的。如果它不通，那就说明它不是同一套方法。

## 核心策略：意愿锁定 💡

意愿锁定是连招三的第二招。连招三的第一招叫做前价值展示，我们在第一节课已经讲过，这里不再赘述。我们直接讲连招三的策略部分。

展示面属于“升学”，而意愿锁定属于策略。为什么叫意愿锁定？“意愿”指的是恋爱意愿，即锁定对方的恋爱意愿。换句话说，在聊天最开始的时候，我们就要把聊天的氛围和框架锁定到谈恋爱的方向上。这样可以解决非常多的问题。

比如女人耍贱、不认账、很难聊，或者假装对你没兴趣。很多人没用连招的方式去聊，只是瞎扯，结果聊不到点子上。因为对方有无数种理由调整框架，避免暴露需求感，并想办法让你落入她的框架，让你看起来是在追逐她。

如果将对话锁定在以恋爱为目标的方向上，很多问题就迎刃而解了。这个策略就叫做意愿锁定，也是我们白源体系的一个伟大发明。

## 哲学基础：只打顺风局 🎯

你可以理解为只打顺风局。这为你提供了打顺风局的基础条件。你进框架，我就跟你聊；不进框架，我就不跟你聊。以前没有这套方法论，你被迫去打逆风局，因为你根本看不出来这是逆风局还是顺风局，可能300句对话后才能看清楚。

意愿锁定可以通过上来第一句话就帮你判断出这是顺风局还是逆风局。你可以理解为不打逆风局，也没有必要打。我放着大量的好车、新车不开，为什么要去修破车？

这里穿插一些新学的内容。很多男人喜欢修破车，可以说是很贱。这不是说人很贱，而是人性很贱，因为这种心理是普遍性的。比如，昨天群里有兄弟说，连着用白驹搞定了四五个，都不想聊了，甚至不想泡妞了。这很有意思，因为你之前为什么那么信誓旦旦地想去聊天？

你会发现，人有一种常态就是不满足，叫做完整性心理，希望所有事情得到圆满满足。执念就是希望圆满满足，所以经常处于不满足的状态。因为很多事情天然就不可能被满足。

人就很贱，总是关注“哪壶不开提哪壶”。人们在讨论事物时，几乎对于过去的成功只是顺带一提，或者是在装逼时才聊。人类的话题内容几乎都是围绕问题本身的，就是哪有问题聊哪。运行通畅的，比如红绿灯通畅，没人会聊；但哪出事故堵了，人们就聊哪。

人类的这种心态，天然是一种“哪壶不开提哪壶”的状态。你可以理解为贱，有句话说是“会哭的孩子有奶喝”。人类的注意力思维逻辑，就被设定成了关注问题，哪儿不好、哪儿还没有完满、哪儿还有缺乏，就会关注哪。

这种心理被带入到情感中，就会变成：你搞定那个都叫你老公了，你不去跟人家聊后面怎么深刻筛选、建立内在连接，规划下一场约会。他在那干啥呢？他在那修车，修那个死活不叫老公的。“老子今天一定要把你拿下来。”即便很多人学会了方法，他也是这样。

你的思维把你玩了，一种惯性，你就放着那个好的不要。所以很多男人婚姻不好、恋爱不幸福，其实是活该。当然，这不是骂人话，是很客观的描述。在佛教中就叫做自作自受，都是你自己作出来的。你放着那个顺服的，不去好好跟他谈恋爱，你觉得没意思，觉得没有挑战感，一定要找个难的。

什么样的叫做难？很简单，就是不好搞定的。所以为什么很多男人的婚姻以及恋爱不幸福？因为他天然在认识一个女人的时候，当他决定要脱单，首先能够赢得他注意力的就是那个难搞的。主动上来投怀送抱的，他就觉得没意思，不想搞。所以他最终注意力就被锁定在了这个不好搞、难搞的身上。

费尽千辛万苦，终于把这个女人搞回家了，问题才开始了。因为女人的这种属性是具有一致性的。你在恋爱期间追她的时候不好搞的女人，在恋爱之后依然不好搞，那就是你倒霉的日子开始。

很多男人的婚姻不幸，就是纯属自作自受，就是你自己活该，一点都不冤枉，因为就是你自己挑的。你偏要挑那个难度，因为男人有一种雄性荷尔蒙，睾酮激素，他希望得到征服感。而这种征服感最大的满足就是征服一个难以搞定的东西。换句话说，就是你完成一个难度系数越高的事情，你就越有成就感。

所以这就是你的情绪心理性设计上的一个bug。这个心理性显然不是一个正确的心理性。所以你总是想要去挑那种有难度系数的，还觉得自己怪牛逼的。其实就是你活该，就是因为又菜又爱玩，你本身技术还不行，然后还就要挑那个男的。

比如很多人一听到基本功，就不愿意学了。为什么？他都没学，但是当他听到“基本功”这三个字，在他的心智层面，他就会认为这东西很简单，没啥意思，没啥难度，心里面就有心毛了，就会认为这个东西没什么价值。

事实上，客观层面当然不是这样。你连那个东西都没有接触过，你怎么知道它没有价值？你怎么就知道它简单呢？有很多领域的基本功也不简单。它只是讲的年头久了，老师们不断迭代表述方法，所以可以用一种非常简单的方式把基本功陈述清楚，给了人一种错觉，觉得这个东西简单。而事实上是，它在表述层面变简单了，这个东西本身一点都不简单。

难度高不等于回报高。你没有错，你说对了。其实在历史上，很多真正的大雄、吉雄，他们都得出过类似的人生经验。比如，虽然是《三国演义》里的，诸葛亮说他平生从不弄险。什么叫从不弄险？就是不挑难度系数大的事情干。

还有巴菲特，他说过一句话，绝对不会去投资自己看不懂的行业。这又是什么意思呢？也很简单，就是难度系数大的，我不干，我不投。因为你这个东西我很难理解，说明我要控制它、驾驭它的成本就会很高。

我自己创业这么多年，22岁开始创业，我挑战过很多新的方式方法、新的商业模式，有很多成了，有很多败了。但是，在我的成和败的规律当中，我发现了一件事情：如果你想做成一件事情，那么你在已经成功的那件事情上再去创新，它就很容易。因为再去在成功的事情上创新，它就是个微创新，很容易做成。

但如果你搞颠覆性创新，完全就是新东西，从来没有出现过，那么就只有两个结局：要么特别成功，要么特别失败，而且大概率是特别失败的，因为它没有基础。虽然说每一个能够提出全新颠覆性方案的人或颠覆性创新的人很伟大，但是他所担负的成本是非常高昂的，甚至是以身家性命。

如果你很想好好经营自己的人生，希望自己能够最终登顶，或去到一个很高的社会地位，那么在你的方法论体系中，你就必须要学诸葛亮，那就是人生从不弄险，什么容易做得到，那我就去做哪一个。

你会发现，成功的男性都会有一个特征，就是他会挑那个自己有把握的事情去干，而那个没有把握的事情，他是不会去做的。起码成熟的男人是不会去做的。但是你会发现，年轻人，比如十八九岁的小孩，不知道天高地厚，他以为这个世界很简单，最后被这个世界给操翻了。为什么？因为他就喜欢挑那个有难度系数大的。

他的心态就是两极反转，从信心满满到一贫如洗。没干的时候觉得自己挺牛逼，干完之后觉得自己是傻逼。很多没有经历复系教育的男性会存在这样的问题。为什么？因为任何一个男性，他来到这个社会上，从家庭走入社会，都能看到先前男性所创造的丰功伟绩，那些牛逼的东西。

讲这么多例子，我想让大家明白一件事情：从孙子兵法，再给大家讲一个例子。在孙子兵法的观念中，历史总会铭记那些打了绝地反击战的人，就是以少胜多的战役，比如5000人干了5万人。这种战役在历史上比较少，但这样的人就很容易被普通人认为是神将、飞将军。

事实上，在孙子兵法中，对于这种人的论述就差直接骂他是傻逼了。理由也很简单：你作为一个将军，怎么会把自己的军队带到这种死地里去？你就不应该去，你怎么可能会让5万人给围了？你看起来是破围成功了，但是问题是你居然蠢到会进这么一个圈套。

所以在孙子兵法中，他是绝对的追求稳中求胜。你看他的哲学观点：先求不败，而后求胜。我们也沿用了这种思想体系，我先做到不输，然后我再去试图求胜。而很多人上来就是我要大圆满，这种人就是无知。我不说傻逼，就是无知，你对于这个世界缺乏正确而客观的认知。
