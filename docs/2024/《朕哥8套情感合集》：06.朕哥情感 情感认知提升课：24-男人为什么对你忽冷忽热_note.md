# 《朕哥8套情感合集》：06.朕哥情感 情感认知提升课：24-男人为什么对你忽冷忽热

🎼，🎼男人几天不联系，如何掌握主动权。若男人连续几天不联系你，可能意味着他已将你置于较低位置，认为可以离开你，而你却无法离开他。

**男人忽冷忽热的原因**主要包括以下几点：

1. **养鱼心态**：男人将你视为众多选择之一，如同鱼塘中的一条鱼，有事找你，无事则找其他人。
2. **冷暴力操控**：通过冷暴力让你屈服，听从他的话。
3. **不爱但舍不得分手**：男人不爱你，但又担心找不到更好的人，暂时不分手。
4. **想逼你分手但愧疚**：不想分手，但又想站在道德制高点，通过冷暴力让你提出分手。
5. **情绪问题**：男人本身情绪不稳定，容易忽冷忽热。

**解决方法**：

1. **养鱼心态**：扩大交友圈，了解更多男人，筛选出合适的人选。
2. **冷暴力操控**：不要屈服于冷暴力，保持平和态度，不表现出恐惧或屈服。
3. **不爱但舍不得分手**：寻求专业咨询，学习挽回技巧。
4. **想逼你分手但愧疚**：保持冷静，不要与其置气，让他意识到冷暴力无效。
5. **情绪问题**：关注男人情绪变化，寻求专业帮助。
