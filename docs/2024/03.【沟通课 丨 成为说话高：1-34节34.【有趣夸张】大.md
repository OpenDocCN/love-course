# 课程03：沟通课 丨 成为说话高手 🎤
## 第34节：有趣夸张——大胆的夸张 ✨

在本节课中，我们将要学习如何运用“夸张”这一技巧，让你的表达变得生动有趣、令人印象深刻。夸张并非扭曲事实，而是一种富有想象力的表达方式。

我们经常被告知说话不要太夸张。然而，如果运用得当，夸张能带来特别出色的效果。这正是许多优秀演讲者、主持人和脱口秀演员经常使用夸张的原因。

那么，夸张具体会带来什么效果呢？我们先用三个小例子来对比一下。

假设在一次同事或朋友的聚会上，大家正在介绍自己的喜好。你想说自己喜欢美食，但又希望介绍得特别一点。

以下是两种介绍方式，我们看看哪种更有趣。

*   **方式A**：我这个人特别喜欢美食。
*   **方式B**：我这个人特别喜欢美食，到什么程度呢？就是凌晨两点，如果你给我打电话说在楼下送我200块钱，我是不会行动的。但是如果凌晨两点给我打电话说带我去吃好吃的，不管你在哪儿，两点10分我肯定出现。

与A相比，B的说法显然有趣多了。这个说法乍一听像是在开玩笑，但它正是夸张。这么一说，别人对你的印象就非常深刻，比单纯说“我喜欢美食”要有趣得多。

再举一个例子。在一次自我介绍中，你想自嘲一下，说自己脑子反应比较慢。但如果只是平淡地说“我这个人脑子不太好使”，效果并不有趣。

那么尝试夸张一下，可以这样说：“我这个人就是脑子不太好使。我朋友都说吃核桃能补脑，我心里想，如果这样的话，我得吃一卡车核桃才行？” “一卡车核桃”就是夸张。这么一说，大家就明白了你的意思，还会觉得你很有想象力、好玩有趣。

第三个例子。假设朋友讲了一个笑话，其实你已经听过了。当着其他人的面，你不好直接说“我早听过了”，但你和这个朋友比较熟，想调节一下气氛。

这个时候可以夸张地说：“这个笑话，恐龙应该也听过。” 恐龙不是几千万年前就灭绝了吗？这就是夸张。这么一说，大家就明白了你的意思，笑一笑，还会觉得你很幽默。

通过以上例子，我们可以看到，夸张并非黑白颠倒或扭曲事实。它是一种顽皮的表达，把本来要说的东西进行拉伸。尤其是当大家听多了“好”、“大”、“快”、“壮观”等单调的形容词后，已经对这些词免疫了。夸张则可以激活别人的注意力。

这就好比舞蹈老师为了让学生把动作做到位。如果只是说“把胳膊伸展出去”，力度是不够的。而如果说“把胳膊伸到窗户外面去”，甚至“体会把胳膊从北京伸到巴黎的感觉”，虽然胳膊不可能真的伸到巴黎，但这样的描述能让学生有更深刻的体会。

所以，对于夸张，别人明明知道不是真的，但可以理解你说的意思，并且感受更深刻。同时，还会觉得你很有想象力、很幽默，因为这种说法脱离了现实，是一种超现实主义的表达，能给听众带来新奇有趣的体验。

当然，并非随便夸张就会有趣。夸张得不好，反而会显得笨拙和尴尬。

上一节我们介绍了夸张的效果和原理，本节中我们来看看具体如何操作，让夸张变得有趣。

以下是夸张的两个关键方法。

首先，要有意识且很自然地将一个事物本来的程度、范围、尺度或数量进行大幅放大，例如放大10倍、100倍甚至1000倍。

*   **描述体积**：说一个人“肚子大”，不如说“肚子可以放下10个西瓜”或“肚子可以装下一头牛”有趣。这里的“10个西瓜”或“一头牛”就是把体积扩大了若干倍。
*   **描述时间**：如果说“我想好好赞美你”，这很普通。但如果说“赞美你需要100年”，则浪漫有趣。这里是把时间放大了若干倍。
*   **描述速度**：形容小李说话慢、爱停顿，可以说：“一般小李说完第一句话之后，如果你上了厕所再回来，他第二句话还没想好说什么。” 这是把时间无限拉长。形容小猪说话快，可以说：“他讲话巴拉巴拉特别快，你听他讲话的时候，花钱打一个喷嚏的功夫，可能就直接漏掉一部《荷马史诗》。” 这是把时间无限缩短。
*   **描述重量**：朋友坐你自行车后座，你觉得他特别重，可以开玩笑说：“拉着你的感觉，就好像背后拉着一辆卡车一样。”
*   **描述外貌**：形容某人嘴巴大，只说“嘴巴大”没意思。可以说：“刘总嘴巴大得，一笑起来，感觉嘴角都能连到耳朵上。” 古人也是夸张高手，成语“汗牛充栋”就是形容家里书特别多，多到运书的牛累出汗，书堆到房梁。
*   **描述虚拟事物**：情绪、气势、状态也可以夸张。例如，你去见一个气势威严的大客户王总，回来后可以这样描述感受：“我当时去见王总，一进会议室，那气势，整个会议室直接刮起一阵风，所有人都打了个冷战，我当时差点就吓跪了。”

以上例子展示了如何从各个角度对事物进行夸张放大。这是夸张的第一个关键点。

夸张的第二个关键点，是要敢于加入一些“无厘头”的元素，这样会更加有趣。

什么意思呢？我们来举个例子。

假设一个人这样介绍自己：“我的特点，大家应该也看出来了，就是皮肤黑。这也是为什么我晚上走路的时候经常被车撞，因为我的肤色和夜色会融为一体。” 皮肤会黑到那个程度吗？当然不会。真的被车撞过吗？大家也知道不是真的。这就是无厘头。他这么一说，夸张效果就非常好。

再比如，你想形容某家餐厅的火锅特别辣。但如果只说“辣”，别人感觉不到到底有多辣。但如果你说：“辣得这么说吧，你要是吃上一口，你肯定想把舌头给拔下来扔出去。” “把舌头拔下来”怎么可能？这就是无厘头。你这么说，关于火锅辣的程度就很到位了。

所以，总结夸张的两个关键：
1.  对事物的程度进行**放大、放大、再放大**。
2.  敢于加入一些**无厘头**的元素。

如果我们再深入一点思考，为什么夸张会带来有趣呢？我认为，高级的夸张是一种浪漫主义。“赞美你需要100年”、“打个哈欠会错过一部《荷马史诗》”，这都是脱离现实、充满想象力的描述，非常浪漫。

本节课中我们一起学习了如何运用“夸张”来提升表达的趣味性和感染力。我们了解了夸张的效果，并掌握了两个核心方法：大幅放大程度和加入无厘头元素。

**本节课的练习**：寻找一个人或一个事物，尝试描述它。
1.  先用普通的方式描述。
2.  随后，把其中用到的形容词（如大小、快慢、轻重、形状、颜色、状态等）进行夸张处理。
3.  如果可能，加入一些无厘头的元素会更好。

试试看，这样的描述效果会怎么样。
