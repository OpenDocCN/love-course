# 清风老师全套：8.彻底打败2+1，保护孩子保护自己13期：090.第八课：直面第三者，全面反击第四步

## 概述
在本节课中，我们将学习如何直面第三者并开始全面反击。我们将分析第三者的心理特点，讨论在直面过程中可能遇到的困难和问题，以及如何正确操作。

## 第三者的心理特点
以下是第三者的三个心理特点：
1. **不敢主动进攻**：大多数第三者知道伴侣有妻子或伴侣后会选择退缩，因为他们缺乏社会地位的认可，不敢轻易挑战你的位置。
2. **过度重视男人的态度**：在关系中，第三者往往将男人的态度视为一切，因此要抓住资源，利用家庭活动展示你的重要性。
3. **缺乏感情以外的牵绊**：第三者为了维护关系，不敢轻易向男人索要礼物或付出，因此展示你的物质条件可以激发对方的嫉妒心理。

## 直面第三者的错误观念
在直面第三者之前，以下三个错误观念需要破除：
1. **冷处理会慢慢离开**：第三者的出现是一个重要冲突，逃避无法解决问题，只有勇敢面对才能解决问题。
2. **必须打架**：打架违法且降低价值，应使用心理战，让对方害怕，这才是真正的胜利。
3. **第三者无价值**：不要带着敌视情绪看待第三者，要反思自己，了解对方的优点，用其优点来回击。

## 与第三者见面的前期准备
以下是与第三者见面前要做的四件事情：
1. **找亲友陪同**：在暗处陪同，在关键冲突时出来镇住对方，给你带来底气。
2. **隐藏实力**：透露的信息越少，后续打击越有利，给对方误导信息。
3. **约在熟悉地点**：展示你对情况的把握，进行威慑。
4. **全程录音**：注意不要剪辑，避免录音无效。

## 与第三者谈话的结构
### 玩具型第三者
1. **认可对方的价值**：表达对对方年轻、漂亮和梦想的认可。
2. **了解对方的需求**：询问对方的需求，并表示愿意商量。
3. **提出条件**：提出解决条件，如金钱。
4. **提出威胁**：强调自己的能力和资源。

### 上位型或玩具型第三者提感情
1. **认可对方的价值**：表达对对方青春和爱情的认可。
2. **提出男人的共性**：指出男人都会说这些话，但未来不可预测。
3. **提出自己的筹码**：强调自己掌握经济大权，知道伴侣的罪行。
4. **强调坚定态度**：表示不会轻易出卖伴侣，但会为了面子断绝关系。

## 总结
在本节课中，我们学习了如何直面第三者并开始全面反击。通过了解第三者的心理特点，正确操作，我们可以有效地应对第三者。记住，勇敢面对冲突，抓住资源，用心理战赢得胜利。
