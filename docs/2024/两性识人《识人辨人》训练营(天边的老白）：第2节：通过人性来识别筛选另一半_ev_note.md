# 两性识人《识人辨人》训练营(天边的老白）：第2节：通过人性来识别筛选另一半_ev_note

在本节课中，我们将要学习如何通过理解人性来识别和筛选潜在的伴侣。课程将围绕两个核心重点展开：首先是对人性本质的理解，其次是环境与经历如何塑造和改变人性。掌握这些知识，有助于我们在两性关系中做出更明智的判断。

## 理解人性的视角 🌱

上一节我们探讨了识人的基础，本节中我们来看看人性的本质。对于人性的理解是千人千面的，没有任何一个人或理论能够完全概括人性的全部面貌。

这就像盲人摸象，每个人都只能从局部去了解人性。因此，我只能分享我个人对人性的理解和观念，这些观点不一定绝对正确，因为人性本身并无对错之分。

没有人能够讲全人性，所以不要期望从任何一位老师或作者那里获得关于人性的完整答案。在本课程中，我将带你了解的，仅限于两性情感这个细分领域中，人性的各种展现形式。

有句话需要理解：**人终其一生都在用自己的视角去看世界**。正如许志远节目中所言，人是主观的，不可能是客观的。人总是带着偏见去看世界，无论你是否承认。

人无法以上帝视角去看待他人，因为你自己就是局中人。因此，我们都是在以自己的视角去观察和理解人性。

近两年我有一个深刻的感悟：员工永远无法以老板的视角看待整个项目，除非他成为老板。同样，女人无法完全以男人的角度理解男人，男人也无法完全与女人的视角产生共鸣。

关于人性，我想分享的核心观点是：**人都是从自己的角度去看待他人和人性，哪怕自认为很客观**。

## 人性的种子与成长 🌳

每个人都是一颗种子，每颗种子内部都蕴含着善、恶、贪婪、欲望、恐惧、嫉妒、虚荣等所有人性特质。种子起初相似，但成长的过程和所处的环境（即“场域”或“命数”）不同，最终长成的树也就不同。

有的树更偏向善良，有的则更偏向邪恶。但每颗种子的起点是相同的。

阐述这些是为了说明，**人是多面性的，并非只有一面**。例如，你遇到一个“好人”，但他可能也会做“坏事”；你喜欢一个善良、孝顺、上进的人，但他可能边界感很差，是“中央空调”，这同样会让你难受。

在社交中，我们最容易踩的第一个坑就是：**因为对方某个优点很突出，而忽略了他的缺点**。因为太喜欢对方的某个特质，而容忍了另一个可能伤害自己或自己无法接受的特质。

因此，看人要看全面、看多面，不能只看一点。

## 如何学习与判断人性 🔍

我认为人性就像一颗种子。大多数种子看起来相似，但它们的成长经历和原生家庭（即“土壤”）差别巨大，在不同的土壤中会长出不同的样子。

以我自己为例，我是河南人。小时候，河南人的名声不好，常有“骗子多”、“偷井盖”之类的新闻。这种地域环境会影响外界对个体的初步判断，虽然这不公平，但它是现实的一部分。

**想知道一个人是什么样的人，预测与他恋爱或合作的结局，就需要从他的成长经历和原生家庭开始拆解**。虽然不一定百分之百准确，但八九不离十。

你不仅要了解人性这颗“种子”里包含什么，更要了解它是如何长大的，生长在什么样的环境里。生长在淤泥中，大多会满身污泥。“出淤泥而不染”之所以流传千古，正是因为其万中无一。

例如，人性中都有贪婪。但一个经济条件差的人谈恋爱，可能对物质的贪婪表现得更明显；而一个经济条件好的人，对物质的贪婪可能就少一些。这就是成长环境和经历导致的不同展现。

## 客观分析与行为验证 📊

我有很多学员，当我分析指出他们喜欢的人可能动机不纯时，他们会很抗拒。他们会列举许多相处中对方做得好的细节，来向我证明这个人很好、很爱他，并非我想象中那么不堪。

这是因为你看人是从他当下的动作去主观判断的，并且自己身在局中。而我是从整体出发，进行相对客观的分析。当然，我也无法做到绝对客观，正如前文所说，人很难绝对客观。

我能做的，是比你相对客观一些。因为你的结论一定是主观的。

**正确的判断方式是：通过对方的行为去推断，而不是听信他的言辞**。具体来说，就是通过分析他的原生家庭、成长经历、当前工作以及他过往的行为，来判断他是一个什么样的人。

他告诉你他是一个不物质、慢热的人，这没有用。他告诉你的，只是他想让你知道的。例如，他说“这事不是钱的事”，那这事往往就是钱的事。他说“我是个慢热的人”，可能恰恰相反。

所以，不要听他说什么，要自己去判断。通过他在与你接触过程中，不经意间说的话和做的事，来验证我们的猜想和判断是否正确。

## 案例分析：环境如何塑造行为 💸

我曾有一个朋友，家庭特别贫穷，初中辍学，很早就外出闯荡。通过分析可以得出结论：他的原生家庭注定了他特别缺钱，非常看重利益。在利益和感情之间，他会无条件选择利益。

他有一个认识了十几年的小学同学，关系一直很好。但因为一笔3万元的口罩生意款项，两人闹掰了。他卖了同学的口罩，却以各种理由不还钱。我知道他手头有钱，但他就是不想还。

在很长一段时间里，我无法理解，为什么十几年的关系抵不上3万元，而且还是同村乡亲。后来我明白了：**他的成长经历让他形成了“利益至上”的观念**。友情、爱情在他眼里没那么重要。当他觉得一段关系即将结束时，他选择最后“捞一把”然后断交。

这种情况并不少见。在恋爱或友情中，一方准备分手前，可能会借一笔不大的钱（几千到一万），然后开始疏远、冷战，最终默认分手。

我自己也曾遇到类似情况。几年前谈过一个做微商的女生，她在分手前一个月，以备货差11200元为由向我借钱，说周转几天就还。钱借出后，她便开始疏远我，再也不提还钱的事，最终我们默认分手。

你要明白，**如果一个人的原生家庭、成长经历和当前环境都很“不对头”，那么你们的友情或爱情，在他眼里可能连一万块钱都不值**。反之，一个家庭和睦、教育完整、重感情的人，绝不会做出这种事。

**一个人二三十年形成的人性观念，不会因为短暂的恋情而改变**。最终，他仍会回归到他经历所塑造的样子。当他认为一段关系可能无法持续时，他可能会选择“最后坑一把再断交”。

## 本节总结 📝

本节课中我们一起学习了如何通过人性来识别筛选另一半。我们来回顾一下核心要点：

1.  **人性是一颗种子**：起初，每颗种子都包含贪婪、恐惧、自私、多情等所有人性特质。公式可表示为：`人性种子 = {善，恶，贪婪，恐惧，自私，……}`

2.  **成长过程决定结果**：种子在不同的环境（土壤）中成长，最终长成的“树”各不相同。有的树偏向善良重感情，有的树偏向自私重利益。这个过程可以简述为：`最终人性 = f(初始种子， 成长环境， 个人经历)`。

3.  **提前推断与筛选**：你要做的，是在认识一个人之初，就通过他的成长环境和经历，推断出他的“人性之树”长成了什么样子。如果推断结果可能给你带来伤害或不好的结果，就应该果断放弃，或将其降级为普通朋友（点赞之交），不要过多纠缠。

下一节课，我将通过一个具体的聊天案例，让大家更清楚地明白“不同的环境对应不同的人群”这条铁律。凡是与这条规律抬杠的人，下场往往都不会太好。
