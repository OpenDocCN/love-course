# 两性识人《识人辨人》训练营(天边的老白）：第2节：通过人性来识别筛选另一半_ev

大家好，我是老白。本节课将重点讲解两个内容：人性的理解以及环境经历对人性的影响。

### 人性的理解

**1. 人性的多面性**

人性是复杂且多面的，每个人都是从一个局部去了解和理解人性的。因此，没有一个人可以完全总结出人性的万象。

**2. 主观视角**

人终其一生都在用自己的视角去看世界，带着偏见看待世界。即使你认为你很客观，实际上每个人都是带着偏见去看世界的。

**3. 人性的种子**

每个人都是一颗种子，每个种子中都蕴含着善、恶、贪婪、欲望、恐惧、嫉妒、虚荣、物质等特质。每个人的成长经历和场域不同，最终长成的树也不同。

### 环境经历对人性的影响

**1. 原生家庭**

原生家庭对一个人的成长影响很大，不同的原生家庭环境会塑造出不同的人性。

**2. 成长经历**

成长经历也会对人性的形成产生影响，不同的成长环境会让人展现出不同的特质。

**3. 当下环境**

当下所处环境也会对人性的表现产生影响，不同环境会让人展现出不同的行为。

### 如何识别筛选另一半

**1. 全面了解**

在认识一个人时，要全面了解他的成长环境、经历和当下环境，从而推断出他的人性树长成了什么样子。

**2. 避免偏见**

不要只看到对方的优点，而忽略了他的缺点。要通过他的行为来判断他的真实性格。

**3. 理性判断**

不要轻易被对方的言论所影响，要通过自己的观察和分析来判断对方的真实性格。

**4. 避免纠缠**

如果发现对方的人性树长成了可能给你带来不好结果的模样，要及时放弃，不要纠缠。

本节课中，我们学习了人性的多面性、环境经历对人性的影响以及如何识别筛选另一半。希望大家能够将这些知识应用到实际生活中，找到真正适合自己的人。
