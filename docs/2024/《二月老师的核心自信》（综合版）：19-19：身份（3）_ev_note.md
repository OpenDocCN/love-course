# 《二月老师的核心自信》课程19-19：身份（3）🔍

在本节课中，我们将要学习身份概念的第三个核心部分：身份是如何形成的，以及社会反馈如何塑造和影响我们的自我认知。我们将探讨身份的来源、社会反馈的作用，以及两种主要的身份赋予方式。

---

上一节我们介绍了身份的基本概念，本节中我们来看看身份的具体来源。你的身份是从哪里来的？这是一个至关重要的认识。自我的概念一直是通过你与世界的互动发展起来的，它往往是你从其他人身上得到的社会反馈。

例如，3岁的小孩子如何判定对错？根据外界的反馈。一个3岁的小孩子在地上捡脏东西吃，旁人看到后露出厌恶的表情。于是这个小孩子根据外界的反馈，就知道这个东西不能吃。

这样来说，如果你试图采取更高价值的身份，并且你接受了它，你的身份就会增强为更高的价值，你的行为举止也会自然发生改变。但如果你试图采取更高价值的身份，而你却没有接受它——例如因为别人取笑你，你想变得更酷，你的朋友说你“臭美”或“装逼”，你无法承受那份社交压力——你可能会不敢再尝试，于是你的身份会回到原来的状态。

作为人类，我们天生就是群居动物。作为社会性动物，我们总是处于征求社会反馈的过程中，而外界的反馈则会进一步影响我们对自己身份的调整。

例如，你在一个团体中尝试去做雄性领袖，外界的反馈都是支持你的，你自然会加强自己身份的构建。但如果外界都不支持，你则会对自己的领袖身份感到怀疑。

反馈在心理学中指的是我们从周围环境中得到的关于自己的信息，因为它能丰富并指导我们的行动。反馈可以使我们根据所得到的信息，逐步调整自己的思维方式以及行为方式。我们的思想不断重复着一个参考和验证的过程：将我们的认知与他人的认知进行比较。

比如，你对这个世界的认知设定是美好的，而你朋友对这个世界的认知是邪恶的，他也告诉了你很多原因。于是你对他的认知产生了参考。最终如果你能够坚持自己的立场，那是因为你的现实感足够强，你会找到更多的方式去验证自己的价值观。如果你的价值观变成了和他一样——从美好变成了丑恶——那么你也会找到更多的方式去验证新的价值观。

通过这种微妙和持续的过程，我们的心灵得到社会反馈。从极其微妙的亚交流到最明显的反应，这两者都在不断地塑造和重塑我们的想法和我们生活的世界。

因此，在社交反馈的作用下，我们就拥有了两种赋予身份的方式。

以下是第一种方式：

*   **别人赋予我们的身份**。
*   一般来说，我们每个人的身份都是别人赋予的。
*   外界会根据你的自信程度、你的肢体语言、你的影响力，以及你的整体外在表现，来决定他们将以什么样的身份和你互动。
*   例如，如果你超级自信（并非伪装），你所有的行为方式都是由内而外散发出来，你没有受到外界的影响，那么你的这种态度将会让外界把你当成高价值的人对待。这就是外界赋予你的身份。
*   当外界给你好的反馈，你的“雷达”一定会接收到。因为他们都尊重你，而好的反馈则会进一步加强你身份的上升。
*   所以，你所有的表达都应该充满权威感和笃定，也就是对自己所说的话都应该深信不疑。同时，对外界的反馈完全不在意，即没有想着如何给别人留下更好的印象，也没有想着如何在别人身上索取价值。
*   当你能够达到这样的态度时，你的表达就会开始充满权威，你会开始深信自己所说的都是完全有价值的。哪怕你所说的没有价值，你也无所谓。就像你在跟外甥女互动一样，你是真实的表达自己。

大部分人之所以难以做自己，无疑就是两个阻碍。

以下是这两个阻碍：

*   **第一，担心别人怎么看自己**。希望能给别人留下好印象，希望别人能够看到自己的优秀等等。这属于进化心理学的范畴，因为你想要拥有更多更好的联盟。但这是活在别人的世界里。
*   **第二，想在别人那里索取某项价值**。这是极其缺乏独立性，并让自身价值不断下滑的行为。现在已经是21世纪，但很多人依然处于“巨婴”状态，总是想着去依赖他人，想着能够在别人那里获得更多更好的资源，无法依赖自己。如果你也是这样，那么你将注定无法做自己。

当然，可能有人会说，你总是有求于其他人的时候，总是有受制于人的时候。确实会有。但每次当我碰到这样的难题时，我都会选择利益交换。我求别人帮我做事，都会给他相对应的好处。也就是说，我所有的诉求都不会让他白帮忙，我至少不会让别人吃亏。所以如果我有求于别人的时候，我会不卑不亢，更不会低三下四。但说句实话，我基本上不会向外界索取价值，和别人互动时就更加不会。

其实真正的独立，说的就是能够做自己。当然，这里绝对没有说让你从此以后不再向外界寻求合作，要单打独斗。绝对没有这个意思。我之所以强调这一点，是担心有些人钻牛角尖。可能有些人又会说，那你向外界寻求合作的时候，比如某个代理商一家独大，你就是需要陪着笑脸去求他。

首先，我说说我自己。我可能会尽最大的努力去达成这个合作。但是，你要让我牺牲尊严、丧失自己才能达成，那么我宁愿不合作。当然，这关系到价值观排序。因为之前我们就讲过，在我的个人价值观里面，个人成长是前几位的，金钱是较为靠后的。

说点心里话，其实在这个社会上，任何人都想鱼和熊掌兼得。我们当然会尽一切努力去达成自己目标的完成。只是无论你怎么做，总是会碰到一些较为棘手的事情。如果你是以下跪为代价达成的某项合作，既然你的选择是如此，那么你就不要怪别人不尊重你。反正我是不会这样，所以看你们个人抉择。

好吧，那回到我们的人际交往当中，又或是和异性互动当中。你之所以不能做自己，无非就是上面所讲的两个因素在作祟：你的得失心，还有你的需求感。所以你应该好好地想想这个问题。

本节课中我们一起学习了身份形成的关键机制。我们认识到，身份很大程度上源于我们与社会互动中接收到的反馈。这种反馈从细微到明显，持续塑造着我们的自我认知。我们探讨了身份被赋予的两种方式，重点分析了“他人赋予身份”这一普遍现象，并指出了阻碍我们“做自己”的两个核心因素：过度在意他人看法和试图向外界索取价值。理解这些，是建立稳固、独立身份认同的基础。关于身份的更多内容，我们下一节课再见。
