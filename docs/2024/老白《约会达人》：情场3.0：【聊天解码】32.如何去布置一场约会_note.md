# 老白《约会达人》：情场3.0：【聊天解码】32.如何去布置一场约会

## 概述
在本节课中，我们将学习如何布置一场成功的约会，包括目的、流程设计、环境选择和座位安排等。

## 约会目的
**目的：** 升高关系。

**公式：** 约会 = 信任 + 感觉

## 约会流程
### 1. 前半小时互动
**目的：** 在前半小时内建立交集，避免女生情绪降温。

**步骤：**
- **10分钟建立交集：** 选择开放环境，肯定女生，让她感到安全。
- **10分钟建立吸引：** 聊天中关注生活看法和感情态度，确认共同点。
- **10分钟主动升级：** 推动约会进行，引导女生接受关系拉近的行为。

### 2. 约会环境
**原则：** 避免孤岛陷阱，选择有转场配套的场所。

**示例：**
- 甜品店 → 小吃街 → 酒吧
- 小吃店 → 超市 → 家 → 宠物
- 甜品店 → 展览 → 咖啡厅 → 溜冰场

### 3. 座位选择
**原则：** 选择相邻或并排座位，避免面对面，选择小桌子和小板凳。
