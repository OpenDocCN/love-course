# 老白《约会达人》：情场3.0：【聊天解码】9.社交软件的玩法

## 概述
在本节课中，我们将学习如何使用社交软件探探，提高匹配率，并了解一些使用技巧。

## 探探界面及操作
探探界面简单明了，左滑表示不喜欢，右滑表示喜欢。若出现匹配成功的界面，即可开始聊天。

## 提高匹配率的关键
**核心概念**：匹配率 = 照片质量
要提高匹配率，关键在于优化照片墙中的6张照片。

### 照片选择原则
1. **人帅**：选择最能展现你魅力的照片。
2. **生活帅**：展示你的生活精彩瞬间。

### 照片排序建议
1. **大头照或半身照**：清晰展现你的面部特征。
2. **另一个角度的半身照**：展现不同角度的身材。
3. **美食照片**：展现你的生活品味。
4. **健身照片**：展现你的健康生活方式。
5. **西装照或工作照**：展现你的职业形象。
6. **小动物合影或环境照**：展现你的兴趣爱好。

## 照片下方信息
1. **公司信息**：不建议填写，以免显得low。
2. **个人签名**：建议有趣或文艺。
3. **标签**：可以是调侃语句或身高信息。

## 使用技巧
1. **探探主要用于收集资源，而非聊天**。
2. **先简单聊天，再提出加微信**。
3. **避免在探探上与女生聊得太久，以免显得花心**。
4. **不要发送低俗信息或私密照片**。
5. **同时使用多款社交软件收集资源**。
6. **注意社交软件上的骗子、假人等**。

## 总结
本节课中，我们学习了如何使用探探提高匹配率，并了解了一些使用技巧。希望这些内容能帮助你更好地使用社交软件，找到心仪的另一半。
