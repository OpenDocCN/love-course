# 21、凯男的社交情商课：第一节课：爱商提升的7个思维逻辑：005.五、拒绝真命思维【微kc58567】

🎼这节课非常重要，我将讲解第五个思维——拒绝真命思维。首先，让我们了解什么是真命思维。真命思维是指认定一个女性，认为她是你一生的挚爱。

这种情感状态与许多中国男性的情感现状密切相关。大多数男性认识的女性有限，且可能得不到她们的回应，导致他们感到饥渴和慌乱。当遇到一个对她们有积极回应的女性时，他们就像抓住救命稻草一样，即使自己并不那么喜欢这个女性。

这种自欺欺人的模式使他们认为这是命中注定，从而形成了真命思维。形成真命思维的原因在于男性缺乏选择自由和权利。😊

🎼现在，我提出一个现实问题：如果你拥有吸引任何女性的能力，你会选择和哪个女性相爱？思考一下，回到现实中，你有多少女性可以选择？

在思考这个问题时，我们意识到，在当今社会，有多少男性能够实现情感自由？答案可能令人沮丧。许多男性将真命思维视为专一的表现，认为自己是真心实意的。

然而，我要告诉你，这并不叫专一，这叫情商低。专一是指，在有多种选择的情况下，你只选择与她在一起。如果你没有选择的权利，却非得追求她，这并不叫专一，这叫情商低。

🎼但我要强调，拒绝真命思维并不意味着你要变得花心，不再真心对待女性。我们当然要一心一意对待一个女性。然而，前提是她也喜欢你。

在她没有喜欢上你之前，你的真命思维会对追求她造成很大的困扰。

接下来，我将讲述真命思维会给你们的恋爱带来哪些不良后果。

**以下是男生角度的后果：**

一旦你拥有真命思维，你会疯狂追求她，为她付出。你总想着如何搞定她。从最初的早安晚安，到后来的红包、礼物，再到更大的投资，这些都是金钱的投资，是可以计算的。

更可怕的是，一些投资是无形的，是对她的经历、情感的投资。你对她投入越多，就越在意她的反馈。你的情感电路会告诉你，你好像喜欢上了她，但实际上，只是你的投资变多了。

因为你需要一个结果，有时你并不是真的喜欢她，而是希望她喜欢你。因为你投入了这么多努力，真命思维让你投入了太多。

然而，你并不明白这些道理，也控制不住自己的情绪，只会加大投资，以换取你想要的回报。这会导致赌徒心态。

**以下是女生角度的后果：**

1. **需求感暴露**：每天早安晚安地追求她，会让她感到压力很大，让她无法呼吸。她可能一开始对你有兴趣，但你的真命思维导致的这种行为会给她带来太多压力，她可能会直接拒绝你。

2. **价值索取**：她可能会认为你会追求她，非她不娶，她会像无底洞一样一味地索取你的价值。今天让你买包，明天让你送表。你买了之后，她会给你奖励，然后像训练狗一样训练你。

3. **备胎沦陷**：你的真命思维会让你害怕失去她，因此你会说“好好好，那不提感情了，只要你开心就好”。这种思维会让你沦为一个十足的备胎。

**总结：**

在女性不喜欢你之前，你不必一心一意地对她做这些事情。她不会因为你这样做而感动，她可能会拒绝你，或者利用你。这也会给你带来不同程度的情绪困扰。

当有人喜欢你，与你在一起时，你的真命思维就显得非常重要。你会一心一意地爱她，对她好，一心一意地专一。这是她非常看重的。

然而，在她喜欢你之前，她不仅不看重，还会利用你。你何必这样做呢？

**结论：**

当你还不确定这个女性是否喜欢你时，请不要出现真命思维。当你确定她喜欢你，并且你们已经在一起时，这时你才要一心一意地对她。

**吸引女性的最佳思维是奖品思维。这在上节课中也有提到。同时，你也要对女性呈现出一种有兴趣无需求的状态，这是她最喜欢的。有兴趣无需求的状态是指，我会去撩她，但不会给她任何压力。例如，我会邀请她去一家有真火必炉、轻柔古典音乐、柔和光线和舒适氛围的餐厅，喝点红酒，享受当下。**
