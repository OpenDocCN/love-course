# 男哥内核3.0初级班：内部课：23.飞龙在天系统延伸拓展（八）发散性思维练习方法

## 概述
在本节课中，我们将学习如何通过发散性思维来提升沟通能力，特别是与女性的交流。我们将探讨发散性思维的概念、练习方法以及如何将其应用于实际沟通中。

## 发散性思维的重要性
**公式**： 发散性思维 = 更丰富的词汇量 + 更灵活的联想能力

发散性思维在沟通中至关重要，它可以帮助我们更好地理解对方，并产生更多有趣的话题。

## 解决大脑空白问题
**公式**： 大脑空白 = 紧张 + 害怕失去 + 缺乏发散性思维

要解决大脑空白问题，我们需要：
1. **大量获取资源**： 通过阅读、学习等方式增加知识储备。
2. **个人品牌建设**： 建立自信，减少害怕失去的感觉。
3. **学习发散性思维**： 通过练习提高联想能力和词汇量。

## 男人与女人的思维差异
**公式**： 男性思维 = 线性思维（结果导向）  
女性思维 = 发散性思维（过程导向）

男性倾向于关注结果，而女性更注重过程和感受。

## 发散性思维的练习方法
以下是几种练习发散性思维的方法：

### 1. 拆字法
将词汇拆分成更小的部分，并尝试联想其他相关词汇。

### 2. 归类法
将事物按照不同的类别进行分类，并尝试找到它们之间的联系。

### 3. 联想法
根据对方的话题，联想到其他相关的事物或经历。

### 4. 感受评价法
在沟通中，关注自己的感受，并将其表达出来。

### 5. 红绿灯法
根据沟通的情境，判断是否继续聊天。

## 总结
本节课中，我们学习了发散性思维的概念、练习方法以及如何将其应用于实际沟通中。通过不断练习，我们可以提高自己的沟通能力，更好地与异性交流。
