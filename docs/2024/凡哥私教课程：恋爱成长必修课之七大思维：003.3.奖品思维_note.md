# 凡哥私教课程：恋爱成长必修课之七大思维：003.3.奖品思维

🎼在前两节课中，我们分别讲解了吸引思维和步骤思维。吸引思维告诉我们，女人不是追来的，而是吸引来的。步骤思维则强调恋爱需要策略，不要只凭热情冲动。

🎼今天，我们将探讨颠覆传统恋爱思维的第三节课——奖品思维。这节课不仅会教授如何正确恋爱，还会分享一些让女人主动追求你的秘诀。

🎼在讲解这节课之前，我想分享自己的成长经历。初中时，我是一个书呆子，暗恋班上的班长。暗恋了三个月后，我给她写了一张小纸条表达心意，结果被她拒绝了。这件事让我很伤心，直到上大学后，我才开始疯狂追求女生，但都以失败告终。

🎼我开始思考，为什么我这样一个普通男生没有女生喜欢？是不是我的追求方式有问题？我发现追求女生真的很累，花费时间、精力和金钱，最后却一无所获。

🎼那么，有没有一种方法可以让女生来追求我呢？于是，我开始研究自己的理论——奖品思维，让自己成为奖品，让女生来追逐我。

🎼首先，我们要明白，大部分女生天生就知道如何让自己成为男人的奖品，让男人来追逐她们。她们内心深处认为，只要有机会，男人都想要和他们发生一些不可描述的事情，她们总认为自己在选择或接受男人。

🎼以下三个例子可以帮助你理解：

1. 男人试图约会女生，女生有权利选择接受或拒绝。
2. 男人请女生吃饭看电影，女生有权利选择接受或拒绝。
3. 男人向女人表白或求婚，女人有权利选择接受或拒绝。

🎼这就是为什么即使女生对某个男人感兴趣，她也不会主动追求他的原因。她会试图让这个男生来追求自己。

🎼举个例子，你会发现，当一个女生被一个男生吸引后，她不会主动接近这个男生，而是会做出一些事情，让男生注意到自己，然后期待男生上钩来追求自己。

🎼那么，什么是奖品思维呢？奖品思维表达的是一种态度，一种自信，相信别人会接近自己，喜欢自己。同时，奖品思维也是一种有利于自己获得框架、得到窗口并占据主动权的互动形式。

🎼用直白的话来说，在男女之间，我们要做奖品，而不是让对方做我们的奖品。

🎼以下是一个例子：

🎼在一个酒吧里，你旁边坐了一个肤白貌美大长腿的辣妹。她发现你一直在盯着她看，然后走过来问你：“你要请我喝一杯吗？”
🎼如果你回答：“好啊，我请你。”那么你在这个女人眼里，就像其他男人一样，没有丝毫挑战性。她会利用你的价值，让你请她喝酒，然后离开。

🎼但如果你运用奖品思维，你会这样回答：“你都是用这种方法在酒吧搭讪男生的吗？”

🎼这样的回答不仅没有迎合她，还用奖品思维回答了她，让她觉得你与众不同，从而增加了吸引她的机会。

🎼在与女人相处时，你的回答可以体现你是一个有奖品性、自信的高价值男人，还是一个没有奖品性、低价值的男人。

🎼如果你总是迎合她，顺从她的回答，那么你就是在告诉她：“你对我来说真的很重要，你是一个女神，你是奖品，我要通过我的努力去追上你，拜托你给我一点机会。”

🎼女人知道了这样的潜台词，她会认为这个男人就像其他男人一样，没有挑战性，勾不起她的兴趣。

🎼所以，兄弟们，这就是你一步步地没有追上女生，反而沦为一个备胎的原因。

🎼但如果拥有一个好的奖品思维，你就能让女生反过来追逐你。

🎼以下是一些例子：

🎼1. 女生说：“你今天的发型好帅呀。”
🎼如果你回答：“过奖了。”那么你就在告诉她：“我平平淡淡，勾不起你想要跟我继续聊天相处的欲望。”

🎼但如果你运用奖品思维，你会这样回答：“你是不是每天晚上都对着我的照片发呆呀？”

🎼这样的回答在告诉女生：“我是奖品，是你在追着我跑的，每天你在对着我的照片发呆，是你已经被我的帅气迷住了。”

🎼2. 女生说：“你长得有点像彭于晏哎。”
🎼如果你回答：“你这样说，我会害羞的，万一你爱上我了怎么办呢？”
🎼这样的回答在告诉女生：“我是奖品，我是大牌，我在给你追逐我的机会，你有一个和我亲密接触的机会。”

🎼普通男人是不会做这些事情，也不会说这些话的。对于一个普通男人来说，好像有一个女生到他身边，他就要谄媚，就要去追求这个女人，就要去夸她，把这个女人当做奖品。

🎼但这样的男人，女人肯定不会理他。因为像这样的没有骨气、没有吸引力的男人，每天女神都会遇到好多，每天她都会被骚扰很多次。

🎼所以，兄弟们，这就是你一步步地没有追上女生，反而沦为一个备胎的原因。

🎼但如果拥有一个好的奖品思维，你就能让女生反过来追逐你。
