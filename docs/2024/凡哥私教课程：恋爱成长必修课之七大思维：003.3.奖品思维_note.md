# 凡哥私教课程：恋爱成长必修课之七大思维：003.3.奖品思维 🏆

在本节课中，我们将要学习“奖品思维”。这是一种核心的恋爱心态，旨在帮助你建立自信，扭转在关系中的被动地位，并引导对方来追逐你。

上一节我们介绍了“步骤思维”，强调了恋爱需要策略和规划。本节中，我们来看看如何通过调整自身定位，来掌握互动的主动权。

## 我的亲身经历与思考

我曾是一个不擅长处理感情的“书呆子”。初中时暗恋班长三个月，鼓起勇气表白后被狠狠拒绝。

上大学后，我产生了强烈的恋爱欲望，开始疯狂追求女生。我把她们当作女神，送礼物、写情书、表白，但无一例外都遭到了拒绝。

我开始反思：自己各方面条件尚可，为何没有女生喜欢？追求过程耗费大量时间、精力和金钱，最后却一无所获。是否存在一种方法，能让女生来主动追求我呢？

## 什么是奖品思维？

奖品思维表达的是一种态度，一种对自己的自信与满意，相信别人会接近自己、喜欢自己的态度。同时，奖品思维更是一种极其有利于你自己获得框架、得到窗口，并且占据主动权的互动形式。

用直白的话来说：**在男女互动中，我们要去做对方的奖品，而不是让对方成为我们的奖品。**

大部分女生天生就懂得如何让自己成为男人的奖品，让男人来追逐。她们内心常认为，是男人在追求她们，而她们拥有选择或接受的权利。

以下是三个常见的例子，帮助理解这种动态：

1.  通常都是男人试图约会女生，女人有权选择接受或拒绝。
2.  通常都是男人请女人吃饭看电影，女人有权选择接受或拒绝。
3.  通常都是男人向女人表白或求婚，女人有权选择接受或拒绝。

因此，即便女生对某个男人感兴趣，她通常也不会主动追求，而是会设法让男生来追求自己。

## 奖品思维的实战应用

一句话、一个回答就能体现你是一个有奖品性的高价值男人，还是一个低价值的追逐者。事事迎合对方，潜台词就是在告诉对方：“你对我来说很重要，你是奖品，我要努力追上你。” 这会让对方觉得你缺乏挑战性，从而失去兴趣。

相反，运用奖品思维，能让你引导对方来追逐你。以下是几个对话案例，展示了普通回应与奖品思维回应的区别。

**场景一：酒吧搭讪**

女生说：“你要请我喝一杯吗？”

*   **普通回应**：`“好啊，我请你。”`
    *   **逻辑**：顺从对方，把自己放在追逐者的位置。对方会认为你与其他男人无异，没有挑战性，你可能只是被利用来换取一杯酒。
*   **奖品思维回应**：`“你都是用这种方法在酒吧搭讪男生的吗？”`
    *   **逻辑**：不附和，反而指出是对方在主动搭讪你。潜台词是：“我是奖品，你是被吸引过来的一方。” 这提供了挑战性，让你与众不同，大大增加了吸引对方的机会。

**场景二：收到夸奖**

女生说：“你今天的发型好帅呀。”

*   **普通回应**：`“过奖了，过奖了。”`
    *   **逻辑**：谦虚回应，缺乏奖品性和张力。对话可能就此终结。
*   **奖品思维回应**：`“你是不是每天晚上都对着我的照片发呆呀？”` 或 `“你是不是每天都对着我的照片流口水？”`
    *   **逻辑**：以幽默、自信的方式接话，暗示对方已被你吸引，正在“追逐”你。这延续了对话，并建立了你的奖品框架。

**场景三：被说像明星**

女生说：“你长得有点像彭于晏哎。”

*   **普通回应**：`“没有没有，差远了。”`
    *   **逻辑**：否定夸奖，降低自身价值。
*   **奖品思维回应**：`“你这样说，我会害羞的，万一你爱上我了怎么办呢？”` 或 `“那给你一次和大牌亲密接触的机会，你还不赶紧亲亲我。”`
    *   **逻辑**：坦然接受夸奖并加以发挥，将自己比作“大牌”（奖品），暗示对方获得了与你互动的珍贵机会。这巩固了你的高价值地位。

普通男人常会不自觉地谄媚、追逐，把女人当作奖品。但这样做，只会让你沦为众多无趣追求者之一，无法引起对方真正的兴趣。

## 总结

本节课中我们一起学习了“奖品思维”。其核心在于**建立自信，将自己视为值得被追逐的“奖品”**，而非一味地去迎合和追逐对方。通过语言和态度的转变，你可以在互动中占据主动权，引导对方来投资和追逐，从而建立更有吸引力的关系。
