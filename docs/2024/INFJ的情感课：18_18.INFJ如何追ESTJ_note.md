# INFJ的情感课：18_18.INFJ如何追ESTJ_note

在本节课中，我们将探讨INFJ人格类型如何理解并追求ESTJ人格类型。我们将分析ESTJ的核心特点、思维模式，以及INFJ在与他们相处时可以运用的策略和心态调整方法。

## ESTJ的性格特点 🧠

上一节我们介绍了课程主题，本节中我们来看看ESTJ的核心性格。ESTJ的优势功能是**Te（外倾思考）**和**Si（内倾感觉）**。这使得他们处理事务时，倾向于使用一套可靠、高效的标准去衡量一切。

ESTJ非常要强，这种特质既体现在生活作风上，也体现在话语上。一个显著特点是他们话特别多。在与ESTJ交流时，你可能会发现几乎没有插话的余地，他们可能占据了95%的对话内容，只留给你表示赞同或附和的空问。这种沟通方式本身可能不是INFJ所喜欢的，因为INFJ自己也需要倾诉。

## 理解ESTJ的深层动机 💡

了解了ESTJ的外在表现后，我们进一步探讨其内在动机。如果你能看透ESTJ，你会发现与他们相处其实是一件特别的事。关键在于理解他们外在言行下的情感诉求。

ESTJ在情感中不断评论事物好坏、对错时，其内在诉求其实是情感诉求。例如，他说“你怎么来这么晚”，潜在意思是“我想早点见到你”。他说“这件事做得差”，潜在意思是“我可以帮你，因为我想帮你”。这些言行背后包含了许多潜在的情感表达。ESTJ很多时候需要情感支持，但只是不想示弱而已。

在与ESTJ相处过程中，你可以更多地去发现他性格里面的闪光点。近距离观察时，你会发现他不像外表表现得那么愤世嫉俗。很多时候他只是对事情有自己的看法，在表达时更像一个直接说出内心想法的小孩，这对INFJ而言其实是非常真诚的。

## ESTJ的社交行为模式 🤝

理解了他们的动机，我们再来看看ESTJ在不同社交场合的行为模式。ESTJ在生活中，只跟他关系近的人去说那些“不好听”的实话。在工作场合中，他一般只对自己的下属说这些话。在面对领导、上级或需要处好关系的人时，他完全可以做到表现得非常听话、和颜悦色、脾气好、端庄大方。他不是没有“演技”，只是会选择性地使用，决定在何种情况下应该表示真诚。

这一点是ESTJ能够带给INFJ最大的学习之处。INFJ在生活中经常遇到的一个问题是：跟不该直白的人直白，跟应该直白的人反而不直白。例如，在与朋友相处时，最应该做的就是直白表达，如“我生气了，你今天别过来”。但INFJ往往做不到，反而会编造理由。但在面对阻力时，比如家长催相亲，INFJ又容易在不该直白的时候产生逆反心理，直接顶撞。与ESTJ相处多了，INFJ能在社会上更吃得开，因为INFJ很多时候缺乏一个“保护者”的角色。

## 与ESTJ建立关系的核心要点 🔑

那么，与ESTJ建立和发展关系的核心是什么呢？如果你喜欢上ESTJ，最需要注意的一点是：你在他眼里首先必须是一个靠谱的人。这一点对ESTJ格外重要。

以下是“靠谱”的具体体现：

*   **信守承诺**：你答应他的事，必须做到。如果做不到，就别答应。ESTJ对承诺看得很重，他会用一个人能否对自己负责，来衡量这个人能否对他负责。
*   **接纳他的直率**：当ESTJ敢说你、敢挑你小毛病时，那是一种亲密行为。你需要理解，在生活中能被ESTJ直言，其实是特别被他看得起的情况。
*   **适当示弱**：ESTJ的软肋是对弱小者非常有同情心。面对小孩子、小动物或弱势群体时，他充满同情。你可以让他体会到，你一方面很靠谱，另一方面在生活中也有需要被保护的脆弱面。你越适当示弱，你的优势可能越大。

ESTJ在情感中通常不会多变，只要你们真的在一起，他愿意走完一生，这符合其有一定保守色彩的思维。如果ESTJ成为你的伴侣，他往往会承担“保护者”的形象，让你感受到被保护，并鼓励你在该直白时直白，该顶撞时顶撞，该顺从时顺从。

## 课程总结 📝

本节课中，我们一起学习了INFJ如何理解并追求ESTJ。我们探讨了ESTJ依靠**Te**和**Si**的思维模式、其外在直率下的深层情感动机、在不同场合的社交行为，以及与他们建立关系时需要注重的“靠谱”原则和适当示弱的智慧。记住，没有绝对的优势或劣势人格，关键在于相互理解和调整相处方式。希望本课能帮助你更好地理解ESTJ，无论是在情感关系还是日常交往中。
