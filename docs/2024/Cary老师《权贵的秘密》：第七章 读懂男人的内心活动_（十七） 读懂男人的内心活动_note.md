# Cary老师《权贵的秘密》：第七章 读懂男人的内心活动_（十七） 读懂男人的内心活动

大家好，欢迎来到第十七节课。今天我们将学习如何判断男人的类型，即如何根据男人的行为和言语来了解他们的内心想法。

## 了解男人的行为惯性

首先，我们要了解男人的行为惯性。每个人的行为习惯、兴趣爱好、价值观和经历都会影响他们的择偶观。例如，一个人可能谈过多次恋爱，但每次都偏好某种类型的女性。了解这些可以帮助我们预测男人未来的行为反应和判断。

### 方法论

1. **看情史**：了解男人过去喜欢的女性类型，以及他们曾经经历的情感经历。
2. **看惯性**：观察男人在恋爱中的相处模式，例如是否经常给对方花钱。
3. **看分手原因**：了解男人以往分手的原因，以及他们是否容易因为相似的原因再次分手。

## 读懂男人的言外之意

除了了解男人的行为惯性，我们还需要学会读懂他们的言外之意。这需要我们具备一定的洞察力和判断力。

### 原则

1. **不必读懂建议**：当对方给你建议时，表明他们对你有真实想法，可以直接按照建议去做。
2. **读懂夸奖的含义**：当对方夸奖你时，要注意他们是否在用中性词，以及是否正面回答了问题。

### 方法论

1. **注意对方是否在兜圈子**：例如，对方没有正面回答问题，而是绕圈子。
2. **观察对方是否在用中性词夸奖你**：例如，对方说你今天打扮得“挺别致的”，而不是“很好看”。

### 常见场景

1. **男人说“我很花心，但是我只爱你一个”**：言外之意是，他还会找其他女人，但不会爱上她们。
2. **男人说“我现在没打算要结婚”**：言外之意是，他不想跟你结婚，只是想玩玩。
3. **已婚男人隐瞒自己有家庭**：言外之意是，他想低成本地拥有你。
4. **男人说“我可以主动给，但是你不能主动要”**：言外之意是，你已经明码标价，他只会在你表现好之后才给你一些小费。

## 总结

本节课中，我们学习了如何判断男人的类型，以及如何读懂他们的言外之意。希望这些知识能帮助大家更好地了解男人，建立更健康的人际关系。
