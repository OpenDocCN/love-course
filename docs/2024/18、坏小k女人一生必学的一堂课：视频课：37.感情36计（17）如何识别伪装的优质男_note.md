# 18、坏小k女人一生必学的一堂课：视频课：37.感情36计（17）如何识别伪装的优质男

🎼在情感三十六计的第十七计中，我们将探讨如何识别那些伪装成优质异性的个体。以下是一些具体方法和流程。

### 1. 识别一级问题

**以下是识别伪装优质男的方法：**

- **一级问题**：这类问题通常以“为什么”开头，后面跟着单一的行为，例如“为什么他不理我”、“为什么主管没有给我答应”等。频繁提出这类问题的人，往往在关系中寻求成为受益者，缺乏深度思考，性格直接率真。

### 2. 识别二级问题

- **二级问题**：这类问题涉及关系冲突等议题，例如“为什么我看到弟弟失望之后，我就立刻给他补上了一个奥特曼，可他回家后依然哭了”等。频繁提出这类问题的人，在关系中倾向于成为平衡者。

### 3. 识别质疑者

- **质疑者**：在讨论一个事件时，只有约15%的人会提出质疑，例如“你跟我说的这个事情，真的是事实吗？”这类人通常不易随波逐流，自我意识强，可能是潜在的引领者。

### 4. 测试经济状况

- **测试方法**：
  1. **房价态度**：询问他对房价的看法，了解他对房价的态度。
  2. **时间价值**：观察他的时间是否值钱。
  3. **公积金**：聊聊公积金，了解他的收入水平。

**公式**：月工资收入 = 公积金金额 / 公积金比例

**例如**：如果公积金是工资收入的12%，公积金金额为700元，公积金比例为0.12，则月工资收入约为5800元。

通过以上方法，你可以更好地识别伪装的优质男，保护自己免受伤害。
