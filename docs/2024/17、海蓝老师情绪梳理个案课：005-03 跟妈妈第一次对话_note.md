# 17、海蓝老师情绪梳理个案课：005-03 跟妈妈第一次对话

## 概述
在本节课中，我们将学习如何通过情绪梳理个案课，与母亲进行第一次对话，表达内心的感受和想法。

## 对话内容整理

### 1. 对妈妈的抱怨
- **抱怨内容**：认为妈妈无知，什么都不懂。
- **具体表现**：觉得妈妈不理解自己的压力，总是找借口。
- **情绪表达**：😔

### 2. 对妈妈的期望
- **期望内容**：希望妈妈能够理解自己的辛苦，不再找借口。
- **具体表现**：希望妈妈能够站在自己的角度思考问题。
- **情绪表达**：😊

### 3. 对妈妈的指责
- **指责内容**：认为妈妈过于强势，总是要求自己道歉。
- **具体表现**：觉得妈妈对自己的要求过高，造成很大的压力。
- **情绪表达**：😔

### 4. 对妈妈的回忆
- **回忆内容**：回忆起小时候妈妈对自己的打骂，以及自己的焦虑情绪。
- **具体表现**：觉得妈妈过于关注自己的学习，忽视了自己的其他方面。
- **情绪表达**：😔

### 5. 对妈妈的感激
- **感激内容**：感激妈妈为自己付出的一切，包括经济上的支持。
- **具体表现**：觉得妈妈辛苦赚钱，供自己读书。
- **情绪表达**：😊

### 6. 对妈妈的建议
- **建议内容**：希望妈妈能够理解自己的压力，给予更多的支持和理解。
- **具体表现**：希望妈妈能够关注自己的心理健康，而不是只关注学习成绩。
- **情绪表达**：😔

## 总结
本节课中，我们通过情绪梳理个案课，与母亲进行了第一次对话，表达了自己的内心感受和想法。通过这次对话，我们希望能够增进与母亲的沟通，改善彼此的关系。
