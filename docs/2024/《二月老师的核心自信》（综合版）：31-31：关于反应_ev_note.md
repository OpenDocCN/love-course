# 《二月老师的核心自信》课程31：关于反应 📡

在本节课中，我们将要学习社交互动中的一个核心概念——“反应”。我们将探讨什么是反应，它如何影响你的社交价值，以及如何通过管理你的反应来提升自信和社交地位。

## 概述：什么是反应？

上一节我们介绍了社交互动中的基本动态，本节中我们来看看“反应”这个概念。当你被某个人吸引时，你的正常状态会受到影响。你的想法、感受和行为在不同程度上会受到干扰。

例如，在心仪的对象面前，你会感到紧张。你在思考自己要说的话。在喜欢的人面前，你很难完全做自己。这其实就是被对方影响。在社交时，总会有一方的反应多于另一方。

## 反应与价值的关系

以下是反应如何体现价值的关键点：

*   **反应多的一方价值较低**：如果你被一个异性吸引，你可能会畏首畏尾或瞻前顾后。当你碰到一个高价值的人时，你的内心也会产生反应。产生过多反应的一方，基本上是价值较低的一方。
*   **反应是绝对的价值指标**：这个世界上不存在绝对的物质或社会价值。你有钱，但可能碰到比你更有钱的人。你颜值高，但可能碰到颜值更高的人。但**反应就是绝对的价值**。
*   **反应决定感知价值**：举例来说，在一个房间里，有两个你不认识的外国人。我坐在中间一言不发，但他们却一直在我面前卑躬屈膝。从你的角度看，我的价值最高。即使事后得知他们一个是印度首富，一个是阿拉伯首富，他们对我的反应也只能进一步衬托我的价值。
*   **反应失衡导致价值失衡**：当你被一个人“拿捏”时，也就是你对他的反应超过他对你的反应时，无论你有多少钱或多高颜值，你的价值还是低于他。这就是为什么有些白富美在一无所有的男孩子面前“什么都不算”。

**核心公式：**
**他人对你的反应 > 你对他的反应 → 你的价值更高**
**你对他的反应 > 他人对你的反应 → 他的价值更高**

因此，当别人对你有反应时，你的价值就能凌驾于他，哪怕你的财富、社会地位都不如他。如果一群人都对你有反应，那么你无疑是整个人群中最亮眼的一位。

## 反应的具体表现

当人们对你有反应时，他们可能会表现出以下行为：

*   应声开启：你随便开启一个话题，他们就会立马加入谈话。
*   高度警觉：他们会观察你的行为举止，哪怕是很细微的方面。
*   格外关注：他们会珍惜你的接受、认可和触碰。
*   声音较弱：通常会用比你弱以及不太确定的声音与你对话。
*   更容易发笑：对你讲的笑话更容易发笑。
*   内心纠结：更纠结于对你的时间和空间的占有，担心自己说的东西不能引起你的兴趣。
*   心理波动：当你和他们靠近或离远时，他们都能在心里产生波动。
*   小心措辞：更加小心地考虑自己所说的是否重要，能否得到你的注意。

当男生喜欢一个女生，或女生喜欢一个男生时，基本上也是这样的情况。低价值的人在高价值的人面前，基本上都是如此。显然，**反应更少的人拥有更高的价值**。

## 反应过度的后果与“此消彼长”

一定要记住这句话：你对一个人过多的反应，只会让你在社交中处于劣势地位。

在社交力学中，最重要的一个概念就是**此消彼长**。

举个例子，如果你和某个人互动时，你的反应过于激烈，且你对他的反应远超他对你的反应，这时对方就能通过社交微观动态，捕捉到你内在价值的匮乏。他的社交地位会因为你过多的反应而自动被提升。

也就是说，你的反应在不断地堆砌他的价值。

我们社会中大部分人都活在别人的反应当中。他们根据外界的变化而变化。外界的反应好，他们的反应也不错；外界的反应不好，他们的反应就不好。甚至他们在高价值的人面前，会用更多的反应来堆砌对方的价值。

此消彼长，当你对别人有更多的反应时，他能捕捉到你的迎合以及你对他的需要。于是你的这些需求会进一步增加他自身的价值。因为他感觉到了自己对于你而言的重要性。在整个相处中，你对他的反应进一步增加了他的整体价值。

## “活在反应中”的人有何特点？

活在别人反应中的人都有什么样的特点？以下是他们的主要特征：

1.  **不清楚自己是谁**。
2.  **不知道自己的价值**。
3.  **缺少独立的想法，不知道赞同什么**。
4.  **生活没有明确的目标**。
5.  **不懂情绪是如何运作的**。

说到这里，你是否感觉有点熟悉？除了第五点（我们将在后续自信课程中讲到），我们的“内在工程”前面几节课都是在讲这些内容。

活在别人反应中的这一类人，就像芦苇一样随风摇摆。他们的思维是散乱的，没有自己的核心，通过外界来定义自己，而不是自己定义自己。

## 如何避免过度反应？——练习“无应”

那么我们该如何避免过度反应？其实就是两个字：**无应**。

无应意味着在你的内心里，你的行为主要不是你对另外一个人的反应。这并不是说你得完全不动。

*   **强忍不是无应**：一个人可能会因为担心遭人非议而强忍着什么都不做，这其实还是活在别人的标准里，是低价值行为。
*   **强忍不反应也是一种反应**：如果你的内心受到了极大波动，只是强忍着没有发作或表达出来，这依然是一种反应。你的行为举止及面部表情等所有“亚交流”信号，都会透露出你的不自然或不安。

当然，新手前期都只能从“强忍不反应”开始练习。

**核心逻辑：**
当你的反应不明显时，外界很难捕捉到你的内心动态，从而无法从你对他们的需求中得到筹码，进而无法提升他们的社交地位。

但这并不是终极目标。我们前面说过了，虽然你强忍着，外界还是可能看得出来。你应该训练自己的中心定力，开始**练习减少自身的反应**。这是一个练习的过程，不可能一开始就做得很好。

对于新手来讲很难，必须通过不断的练习来减少自身的反应。当你经过不断练习，使“不反应”成为你的一种本能时，你就会减少练习中的那种刻意，变得自然。

就像学车，一开始很刻意。但当这项技能完全学会并下沉到潜意识后，它就成为本能，你可以一边聊天一边开车。但在刚开始时做不到。

所以，在一刚开始练习不反应时，一定会有刻意，一定做得不是很好，这是正常的。直到有一天你把它练成本能。那么本能的你就反应比较少。这时，你就不会再以自己的反应去堆砌别人的价值。

相反，当别人看到你过少的反应时，他会把你当作更高价值的人对待。

一般来讲，反应过多的人都是低价值，外界能看得出来。当你反应过少时，外界也能看到，于是你会被当作高价值的人对待。

## 总结

本节课中，我们一起学习了“反应”这个核心概念。

我们明白了**反应是衡量社交价值的绝对标尺**：反应更少的一方通常价值更高。我们探讨了过度反应如何通过“此消彼长”的机制降低自身价值、抬高对方价值。我们也分析了“活在别人反应中”的人的几种特征。最后，我们找到了应对方法——通过持续练习达到“无应”的状态，即减少本能反应，从而稳定自身的核心价值，在社交中占据更有利的位置。
