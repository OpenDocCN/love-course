# 野哥《聊天特训营》：第二节聊天准备_ev_note

在本节课中，我们将学习如何为成功的聊天做好充分准备。我们将探讨提升基础吸引力的方法，如何积累和管理社交资源，以及如何避免聊天中常见的错误。做好这些准备，是开启高质量对话的第一步。

## 一、 基础吸引力：你的社交名片 🖼️

上一节我们提到了社交的本质，本节我们来看看如何构建吸引他人的基础。机会总是留给有准备的人。你需要反思自己为聊天做了哪些准备。女孩子精心拍照是为了增加社交机会，扩大自己的圈子。当你寻找聊天对象时，也会先看对方的照片。如果对方不符合你的审美，你通常不会选择聊天。同理，女孩子在选择异性聊天时，首先会看照片。

因此，我们必须为自己的基础吸引力做好准备。基础吸引力主要包括三个方面：**外形**、**物质**和**情绪价值**。你的照片可以充分展示外形；你的选景、拍摄内容和环境可以反映物质水平；情绪价值则来源于配文和互动方式。所以，**展示面**是一种高效且全面的展示基础吸引力的方式。

## 二、 资源积累：量变引起质变 🔄

除了个人形象，聊天还需要资源基础。资源获取主要分为两种途径。

以下是两种主要的社交资源获取方式：
*   **个人社交（远见）**：从展示面运营开始，主动寻找和吸引你喜欢的对象。
*   **群体社交（社群/共同圈）**：这需要你进行社群建设。个人社交可以转化为群体社交，因为在男性群体中，异性资源通常是稀缺的。如果你能利用好这一点，就能建设出自己的社交群体。

本次课程附带的社群不仅用于学习答疑，也是为了帮助大家找到志同道合的伙伴，更好地建设并利用圈子，实现资源共享，接触更优质的人脉。

量变产生质变。如果你希望接触的异性质量越来越高，心态越来越稳，就必须从积累数量开始。这也会积极影响你的心态。在我们的内部课中有一个“钱包概念”：当你兜里只有1000元时，花800元会像割肉一样疼；但当你有1万元时，花800元可能只像被牙签刺了一下。当你的资源足够多时，与异性聊天就不会患得患失，因为失去一个，还有很多其他选择，并且你收集新资源的难度会变得很低。

## 三、 如何打造展示面 📸

前面提到，展示面是低成本、高收益的展示方式，同时也能让女孩子在你的朋友圈中找到聊天话题。很多时候，女生反应平淡、不主动找话题，是因为你这个人本身缺乏话题点。如果对方点开你的朋友圈，发现一片空白，或者全是琐碎无意义的内容，聊天的欲望会瞬间消失。反之，如果你的朋友圈内容丰富多彩，话题自然就打开了。你的展示面元素越多，对方能参与讨论的话题点就越多，总有一个能引起共鸣。

综合以上三项因素，在拍摄展示面时需要注意以下几点：
*   **充分准备外形**：因为它直观地展现了你的外貌和颜值。
*   **精心选择拍摄内容**：内容选择非常重要。不要在一个地方拍过多照片，显得没见过世面。要更生活化，显得自然。
*   **注重配文和互动**：你可以添加我们的微信，我们会提供一系列朋友圈互动案例和配文素材供你选择。

关于展示面如何拍摄，我们提供了学习资料。同时，你可以在我们的社群中找到搭档互相拍摄，甚至可以直接参加线下活动，由老师亲自帮你拍摄。

## 四、 聊天中的常见问题与核心准则 ⚠️

了解了如何准备，我们再来看看聊天中需要避免哪些问题。以下是学员中最常见的几个问题：

**1. 言多必失与漫无目的**
很多兄弟错误地认为，和妹子聊得越多，对方就会越有好感。其实并非如此。你越唠叨，对方可能越没兴趣看下去。例如，发送60秒的语音，对方可能听两秒就关了，或者看到转文字后密密麻麻的文字就不想细读。不要说没有营养的话，说多了容易语无伦次。在基础不牢时，价值密度不够高，女生就没有兴趣继续聊下去。其次，聊天漫无目的，天天扯淡，没有推进关系的话题，很容易导致进入“友谊区”。听完第一节课后，大家应该已经有了明确目标，不要再犯这个错误。

**2. 话题不当与低俗趣味**
话题不符合时宜或对方兴趣。例如，曾经有兄弟和妹子吟诗作对，虽然反馈尚可，但无法有效推进关系，这属于自娱自乐。要选择符合当下热点、符合你个人设定的话题，不要没话找话。另一个问题是低俗趣味和低级土味情话。例如，在社交软件上不停说“八嘎”，会让女生觉得奇怪和傻气。低级的趣味和流传过多次的土味情话，作用有限，且会暴露你贫乏的业余生活。

**3. 追加消息与舔狗行为**
很多人发现妹子没及时回复，就会着急地追加消息。但每个人都有自己的生活，可能在忙或不方便。在不合适的时候硬聊不是好选择，除非你明确意识到自己说错了话，可以适当补救。此外，要避免“舔狗”行为，即无条件迎合女生的所有要求。越是没有难度、伸手可得的东西，在对方眼中价值就越低。女性通常会向上选择伴侣。一旦你在对方眼中失去价值，在一起的成功率就非常低。

**4. 缺乏逻辑性与话术误用**
聊天时前言不搭后语，缺乏逻辑。很多学员会生硬地套用记下的话术，但常常用错阶段，不知道在什么情况下说什么话更好。话术本身是加分项，但必须用在正确的时机和场景。

## 五、 核心聊天准则总结 ✅

总结下来，**不犯错是第一步**。人与人相处，无论是同性还是异性，都是“减分容易加分难”。不犯错就是加分的开始。

以下是聊天的核心基本准则：
*   **准则一：传递有效信息，推动关系**。所有对话都应旨在分享有价值、有营养、有回复点的信息，并推动双方关系向前发展。对方会因为你们关系的转变而更愿意与你聊天。
    *   `对话目标 = 传递价值信息 + 推动关系发展`
*   **准则二：技巧旨在降低社交成本**。所有聊天技巧都是为了降低后续社交行为的难度。关系在网上聊得越深入，后续线下见面、关系升级或在一起的难度就越低。这比什么都没聊就直接见面，需要你在现场投入更多金钱和精力（如请客吃饭、去高档场所）要高效得多。
*   **准则三：懂得包装与展示价值**。这一点在展示面部分已经阐述得很明白，即构建自身吸引力。
*   **准则四：建立社交圈子的概念**。建设自己的社交圈子，既能让你学习成长，也能收集更好的谈资。例如，在社群里，兄弟们会分享聊天记录，导师会解答问题，你可以从中学习。社群还会组织线下活动，为你提供社交机会。
