# 对视无惧 05：006. 真相：你在错误的道路上，狂奔了太久 🚫

在本节课中，我们将要学习两个关于对视恐惧及相关问题的核心真相。了解这些真相本身，就能为你带来深刻的启发，帮助你调整方向。

## 第一个真相：你的影响范围与“成本” 💡

上一节我们探讨了恐惧的表现，本节中我们来看看关于“影响他人”的真相。你是否总感觉自己的紧张、恐惧会给他人造成困扰，甚至认为别人在针对或议论你？

实际上，你的状态确实会对一部分人产生影响。但请注意，这不是为了制造焦虑，而是为了让你在透彻理解后，不再为此事心惊胆战，从而更坦然地面对并更快地解决问题。

这个影响并非针对所有人，它主要作用于与你相似的**高敏感人群**。这类人群通常具有敏感、自卑等特质，他们同样处于高度警觉状态，容易受到他人眼神、语言等变化的困扰。

认识到这一点后，我们无需对“影响他人”这件事本身产生新的焦虑。我们需要认清一个清晰的现实：**没有人能让周围所有人都喜欢和认同自己**。

以下是关于人际评价的一个普遍事实：
*   据统计，一个普通人背后，超过95%的亲戚朋友都可能议论或说过他的不是。可能只有最亲近的直系亲属例外。
*   这说明，人与人之间的负面评价是普遍存在的，只是通常不被说破。

你看，普通人之间尚且如此。因此，你努力伪装、掩饰，强迫自己做违心的事，担心影响他人——**即便你不这样做，讨厌你的人依然可能讨厌你**。

我常对训练营的学员说，你应该将同事、亲戚等视为治愈过程中的**“沉没成本”**。在经济学中，沉没成本是指已经发生且无法收回的支出。将他们看作必须付出的“代价”或“损耗”后，你的心态会发生变化：看你就看你了，得罪就得罪了，误会就误会了。

以我本人为例，我性格比较特别。喜欢我的朋友是真喜欢，讨厌我的亲戚同事也是真讨厌。我的态度是：**“我吃你喝你了？我靠你了吗？如果你客气，大家情分还在；如果你在我面前摆架子，那就离我远点。”** 话虽直白，但这确实是我真实的心境。当你建立起这种心态，恐惧对他人的影响自然会减弱。

## 第二个真相：药物的角色与局限 💊

了解了人际影响的真相后，我们再来看看另一个无处不在的话题：药物能否解决问题？什么情况下该吃药？

我曾于2017年因其他问题尝试过两种药物：**佐洛复（舍曲林）** 和 **盐酸帕罗西汀**。以我个人的体验，服药约十天后，我整天昏昏沉沉，异常困倦，频繁打哈欠。那种状态下，我无法正常工作、学习或处理任何事务。

当然，我也听过一些朋友反馈，服药后症状减轻了不少。那么，药物到底能否根治我们的问题呢？

**答案是不能。思维导致的问题，必须通过思维的方式来解决。**

药物是用来干什么的呢？它只在一种情况下适用：**当长期处于强迫、抑郁或焦虑的高压状态，导致身体出现器质性病变时**。

其原理很简单：长期的高压状态可能导致严重的神经衰弱（医院检查可能给出的结论）。此时，即使没有外界刺激，你的身体也比常人更容易陷入紧张和惊恐。这不再是心理影响生理，而是**生理问题反过来加剧了心理症状**。

如果你确信自己已处于这种难以自控的生理性惊恐状态，那么去医院寻求医生帮助并开具药物是合理的。

但是，请务必记住：**药物永远只是辅助治疗强迫、焦虑等问题的工具**。如果你将其视为救命稻草，那就是本末倒置。心病终需心药医。

将药物作为主要治疗手段，本身也说明了一个问题：**你在逃避**。而逃避，只会带来更大的恐惧和更深的弱势人格，让你离真正的治愈越来越远。

## 总结 📝

本节课中我们一起学习了两个关键真相：
1.  你的恐惧主要影响的是同类高敏感人群，且人际间的负面评价本是常态。将部分人际关系视为治愈的“沉没成本”，有助于你建立更坦然的心态。
2.  药物无法解决根源性的思维问题，它仅适用于缓解因长期心理压力导致的严重生理症状。依赖药物作为主要手段是一种逃避，会阻碍真正的康复。

了解真相，是为了停止在错误道路上的狂奔，从而为转向正确的解决之道奠定基础。
