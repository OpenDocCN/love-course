## 高段位接话的典型错误

以下是高段位接话中常见的几个错误：

1. **心不在焉**：在沟通中表现出不专注，让对方感到不被尊重。
2. **各说各的**：话题不相关，导致双方都感到不感兴趣。
3. **抢话**：在对方还未说完时插话，让对方感到不被尊重。
4. **话里带着对抗情绪**：总是抬杠，让人感到不舒服。
5. **过分谦虚**：过度谦虚显得虚伪，让人感到不真诚。

## 高段位接话的技巧

1. **从合格的聆听者开始**：让对方把话说完，保持倾听状态，并给予回应。
2. **恰如其分的思考**：在聆听过程中思考，让对方感到你在认真思考。
3. **肢体语言**：通过肢体语言表现出你的兴趣，让对方感到你在认真聆听。
4. **找点**：找到对方的痛点、痒点、兴奋点、共鸣点、兴趣点，再开始接话。
5. **引导对方继续说**：让对方越说越开心，越说越打开，越说越释放。
