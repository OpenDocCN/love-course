# 情绪价值提升课：04_第四课，自尊滋养，收获对方的无限宠爱

大家好，欢迎收听本次课程。本节课将介绍自尊滋养素，它是指给予你良好的自我感觉，让你觉得自己独特、聪明，值得被关注。你是一个伟大的英雄，是勇敢和智慧的化身，是地球一切美好的来源。

**自尊滋养素的核心概念**：

- 自尊滋养素：提升自我感觉，增强自信。
- 自尊滋养：理解并支持对方的自我定位，用心支持。

**以下是自尊滋养的具体方法**：

1. **无声赞美法**：
   - 停止手头的事情，认真倾听。
   - 保持眼神接触，微笑点头。
   - 轻柔的肢体接触，如拍肩膀。

2. **充分认同对方**：
   - 了解对方的理想自我形象。
   - 关注对方的表情、眼神和笑容。
   - 温柔抚慰，给予理解。

3. **共情赞美**：
   - 表达认同、理解和共情的语句。
   - 使用共情短语，如“我明白你的感受”。

4. **疯狂的夸奖对方自我陶醉的点**：
   - 找到男生的兴趣点和狂热的爱好。
   - 及时赞美，增强自信心。

5. **学会倾听**：
   - 虔诚的倾听状态。
   - 运用固定句式，如“你怎么会有这么高的天分？”、“你是怎么做到的？”、“真的假的？你真的是太有才了”。

**总结**：

本节课中，我们学习了自尊滋养素的概念和具体方法。通过理解和支持对方的自我定位，用心支持，我们可以提升对方的自尊，收获对方的无限宠爱。希望大家都能够掌握这些方法，在人际交往中更加得心应手。
