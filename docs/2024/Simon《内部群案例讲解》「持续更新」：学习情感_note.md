## 概述 📋

情感分析是理解文本背后情绪状态的关键技术。本节课程将引导你通过实际案例，掌握分析情感的基本方法。

---

## 1. 情感分析的核心概念

上一节我们介绍了课程的整体目标，本节中我们来看看情感分析的核心是什么。

情感分析旨在判断一段文本所表达的情感是**正面**、**负面**还是**中性**。其核心在于量化文本的情感倾向。

一个基础的量化公式可以表示为：
**情感得分 = (正面词汇数 - 负面词汇数) / 总词汇数**

---

## 2. 情感词汇库的构建

理解了核心概念后，我们需要一个工具来识别正面和负面词汇，这个工具就是情感词汇库。

以下是构建基础情感词汇库的步骤：

1.  收集常见的情感词汇，如“高兴”、“优秀”列为正面词，“悲伤”、“糟糕”列为负面词。
2.  为每个词汇赋予一个基础的情感强度分值。
3.  将词汇与分值存储为键值对，便于程序调用。

一个简单的代码示例可以是：
```python
sentiment_lexicon = {
    “优秀”: 2,
    “高兴”: 1,
    “一般”: 0,
    “糟糕”: -1,
    “悲伤”: -2
}
```

---

## 3. 案例文本分析步骤

有了词汇库，我们就可以对真实的群聊文本进行分析了。分析过程遵循一系列清晰的步骤。

以下是分析单条群聊消息的步骤：

1.  **文本预处理**：清除无关符号，并将文本拆分成独立的词汇。
2.  **词汇匹配**：将每个词汇与情感词汇库进行比对。
3.  **分数计算**：累加所有匹配词汇的情感得分。
4.  **结果判定**：根据总分判断情感倾向（如总分>0为正面）。

---

## 4. 处理复杂情况

基础分析能解决大部分问题，但实际对话中常出现否定、转折等复杂情况，这会影响情感的最终判断。

例如：
*   “这个方案**不是不好**。”（否定词削弱了负面情绪）
*   “产品很好，**但是**服务太差。”（转折词使情感由正面转向负面）

处理这些情况需要在算法中加入规则，例如：当检测到“但是”时，优先考虑其后半句的情感。
