# 米娅《女人一生幸福的必修课》：23.第22节：选择结婚对象一定要规避的十三个

在本节课中，我们将继续探讨选择结婚对象时需要规避的雷区。上一节我们介绍了部分需要警惕的性格与行为模式，本节我们将深入分析另外几个关键雷区，帮助你更清晰地识别潜在风险，为未来的婚姻幸福打下坚实基础。

## 第四个雷区：大男子主义、自私自利与言语刻薄 🚫🗣️

上一节我们讨论了情绪不稳定等特质，本节中我们来看看另一种对婚姻伤害极大的组合：大男子主义、自私自利与言语刻薄。

恋爱期间感情浓度的最高值，就是婚后感情浓度的最高值。如果在恋爱期间，你感觉这个男人不怎么在乎你，不考虑你的感受，那么结婚后这种状况只会加剧。婚姻不会解决任何问题，反而会因身份角色转变和家庭关系复杂化，催生新的问题。在你年轻貌美时他尚且对你不好，年老色衰、满心怨气时更不可能改善。

当男人自称自私时，通常有两层含义。第一层是陈述事实，他确实自私，并且实际程度往往比口头承认的更甚。第二层是试探与免责声明，他提前告知，若你接受，日后他便有理由推卸责任。例如婚后若你指责他自私，他可能回应：“对呀，我早就告诉你了。”

以下是自私男人的常见表现：

*   **忽视伴侣感受**：相对于你的感受，他更在乎自己的感受。有时并非未察觉你不开心，而是不愿提供情绪价值。
*   **从不认错**：吵架后从不认错，永远需要你哄着他，骨子里非常自私。
*   **利益优先**：面临重要选择时，优先考虑自身利益。其胜利果实不愿与你分享，且很少让女人掌握经济大权。

自私自利的男人，大多伴随言语刻薄。吵架时口无遮拦，用难听的话侮辱你，用扎心的话刺激你。言语是内心的侧面反映。内心纯真善良、温暖有爱的男人，绝对说不出恶毒的话。只有内心阴暗、自私自利、不顾及他人感受的人，才会在争吵时用最恶毒的语言重伤你。

不要看一个人好的时候对你有多好，要看你们关系不好时，他对你有多差。不好的时刻才能真正考验一个人的本性。如果这个人好的时候让你感觉是世界上最幸福的女人，吵架时又将你贬损得体无完肤，这就不是健康的感情。

大男子主义的男人，控制欲强且常双标。例如，他可以与兄弟打牌夜不归宿，但你与闺蜜逛街回来晚就要被盘问。这类男人骨子里崇尚男尊女卑，自己可能并无大本事，却觉得自己很了不起。在家颐指气使，在外可能卑躬屈膝。这类男人也可能是甩手掌柜，认为只要往家里拿钱，其他事情都与他无关，缺乏同理心和共情能力，几乎不可能提供情绪价值。只有当你铁心离开时，他或许才会意识到问题。

## 第五个雷区：不良嗜好、大龄负债或超额透支消费 🚫🎲💸

接下来我们探讨与个人习惯和财务状况相关的雷区。

首先是**不良嗜好**，常见的主要是赌博和嫖娼。如果在恋爱期间发现对方有此类问题，务必远离。不要相信“结婚后会改”的承诺，也不要盲目自信认为自己能改变对方。尤其对于赌博成性者，其承诺一句都不要信。成年人应只做筛选，不做改变。应筛选能带来健康生活方式的人，而非试图改变有不良嗜好者，否则你不仅无法改变他，反而可能被拖入消极负能量的漩涡。

有不良嗜好的人通常存在认知观念问题，可能源于原生家庭影响或交友圈子差。这类人往往难以在事业上取得成绩。赌博心态是不劳而获、妄想一夜暴富、走捷径，这违背自然逻辑。**人永远无法赚到认知以外的钱，却可能因有限的认知将已赚到的钱挥霍一空。** 嫖娼通常也难以戒除，尤其当其身边都是此类朋友时。俗话说物以类聚，人以群分，换人比改变他更靠谱。

其次是**大龄负债**。所谓大龄通常指38或40岁以上。普通人的大额负债主要是房贷，这没有问题。但若发现他年纪很大却有非房贷的大额负债，则可能是投资亏损或赌博所致。

对于投资亏损，需审视这个男人是否有过成功赚钱的经验，或在某一行业做出过成绩。如果年纪已不小却从未赚过钱、总是亏损、也未在任一行业有所建树，那么不管他如何用语言包装，都很难再赚到钱。其问题在于不踏实，能力与欲望不匹配。在当今时代，踏实肯干、勤奋努力过上小康生活并不困难。若一个男人40多岁仍无成功经验，却每天幻想创业成功，这是很差的选择，大概率不会带来踏实日子。

若是赌博导致的负债，请参考前述关于不良嗜好的内容。

最后是**超额透支消费**。这明显属于缺乏财务规划能力，且可能导致征信有问题。因此，在结婚前非常有必要查询对方征信。这是互相的要求，你也可以出示自己的征信。如果对方反抗情绪强烈、拒绝要求或用奇怪理由搪塞，大概率其征信有问题。

## 第六个雷区：恋爱期间感情浓度低，感受不到被爱 🚫💔

最后，我们审视关系中最核心的要素：爱的感受。

一个男人如果爱你，你一定能感觉到。如果感觉不到，通常真相就是他不爱你。一个宠爱你的男人，眼神都会不同。**恋爱期间的感情浓度就是婚后的最高值。** 无论你多喜欢这个人，如果他不爱你，也绝非良配。婚后，你会感受到许多冷漠、忽视、不关心与不重视，最初的喜欢也会在琐碎生活中消磨殆尽。不管这个男人有多少优点，他不爱你，这些都与你无关。

如果明确感受到对方不喜欢、不爱你甚至不尊重你，却仍贸然进入婚姻，婚后出轨风险极高，且可能长期被嫌弃打压，家庭矛盾必不可少。婚姻的基本盘很重要。一个男人喜欢你和不喜欢你，你在婚后能得到的待遇相差极大。婚前感情浓度低或互相凑合的婚姻，几乎无法幸福。

男人不爱你却愿意娶你，可能有以下几种原因：年纪大被家里催婚想完成任务；自身条件差找不到对象，有人愿嫁就将就；对婚姻持消极态度，认为跟谁结婚都差不多；或是比较渣，认为你老实、好控制，放在家里安全不影响他外出乱搞；还有可能是“女才男貌”，男方希望利用女方少奋斗二十年。但最后这种，你只能期待对方永远赚不到钱，一旦他赚到钱，大概率就会“上岸第一步，先斩意中人”，因为他本来就不爱你。

综上所述，在选择结婚对象时，对方对你的感情浓度至关重要，直接影响婚后待遇。即使你没有很爱他，但只要他很爱你、对你好、也尊重你的父母，也好过选择一个你很喜欢但他不待见你的男人。你不仅仅是在为自己选丈夫，也是在为父母选女婿，为未来孩子选父亲。**一个男人爱自己的孩子，不见得会爱孩子的母亲。但如果一个男人不爱自己的孩子，那他一定也不爱孩子的母亲。** 一个男人若能尊重你的父母，不代表一定爱你；但他若不尊重你的父母，那么他一定不爱你，他更爱他自己。

本节课中我们一起学习了选择结婚对象时需要规避的另外三个关键雷区：大男子主义、自私自利与言语刻薄的性格组合；不良嗜好、大龄负债或超额透支消费所反映的生活态度与财务问题；以及恋爱期间感情浓度低所预示的婚姻风险。识别这些雷区，有助于你做出更清醒、更有利于长期幸福的选择。
