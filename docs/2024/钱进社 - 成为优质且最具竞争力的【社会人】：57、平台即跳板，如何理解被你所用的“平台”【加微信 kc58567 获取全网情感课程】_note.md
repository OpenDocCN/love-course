# 钱进社 - 成为优质且最具竞争力的【社会人】：57、平台即跳板，如何理解被你所用的“平台”

## 概述
在本节课中，我们将探讨如何理解被你所用的“平台”，以及如何通过平台实现个人成长和提升。

## 核心概念
**平台**：所有可以帮你达到另一个高度的都可以称为平台，包括工作单位、父母、老师、博主、领导、同事等。

## 平台的理解
### 自我建立的平台
1. **规划和布局**：通过自己的规划和布局，即思维方式，来建立平台。
2. **认知和谋略**：认知不足会导致错误的行为和预判，谋略不足会导致行动吃力。
3. **自我宣传**：通过自我宣传，提高曝光度，加深他人对你的印象。
4. **创造和拥有有价值的方案**：拥有比他人更好的方案，成为你的核心竞争力。

### 他人建立的平台
1. **依托于公司**：找到一个可以提供所需资源的公司或单位。
2. **依托于老板**：依靠内在价值、情商、真诚、忠诚和运气，得到老板的栽培。
3. **依托于伙伴**：与共同成长的朋友建立深厚的情感基础。
4. **依托于家庭**：家庭背景对个人成长的影响。

## 总结
本节课中，我们学习了如何理解被你所用的“平台”，以及如何通过平台实现个人成长和提升。希望这些内容能对你有所帮助。
