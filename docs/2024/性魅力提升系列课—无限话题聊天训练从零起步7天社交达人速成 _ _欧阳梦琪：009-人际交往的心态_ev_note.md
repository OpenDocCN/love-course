# 性魅力提升系列课—无限话题聊天训练从零起步7天社交达人速成 _ _欧阳梦琪：009-人际交往的心态

🎼本节课我们将探讨人际交往中所需的心态和思维。心态是人际交往的核心，它往往在事情发生之前就决定了结果。在日常生活和人际接触中，心态影响着我们的情绪状态和自信，进而影响我们的表现。

**核心概念**：心态是人际交往的核心，影响情绪、自信和表现。

以下是时阶中需要的三种心态：

1. **筛选心态**：在初期阶段，我们需要判断对方是否符合我们的标准，同时也要了解对方的需求。
2. **伴侣心态**：在中期阶段，我们需要将对方视为潜在的伴侣或合作伙伴，尊重对方，建立舒适感和亲密度。
3. **双赢心态**：在后期阶段，我们需要帮助对方做出推进关系的决定，建立信任感和安全感。

### 筛选心态

在初期阶段，我们需要保持平等的心态，避免迎合对方和成为老好人。以下是一些注意事项：

- **避免迎合**：不要为了取悦对方而失去自我。
- **保持平等**：将对方视为平等的个体，不要有高低之分。
- **降低预期**：不要对结果抱有过高的期望，以缓解紧张和焦虑。

### 伴侣心态

在中期阶段，我们需要将对方视为潜在的伴侣或合作伙伴，尊重对方，建立舒适感和亲密度。以下是一些注意事项：

- **尊重对方**：将对方视为重要的个体，给予尊重。
- **建立舒适感**：通过肢体接触和言语表达，让对方感受到舒适和自在。
- **比对方先进入状态**：带领对方进入舒适的状态，让对方感受到你的自在和舒适。

### 双赢心态

在后期阶段，我们需要帮助对方做出推进关系的决定，建立信任感和安全感。以下是一些注意事项：

- **同理心**：从对方的角度思考问题，给予对方安全感和信任感。
- **建立信任感**：通过言行一致，让对方感受到你的真诚和可靠。
- **安全感和信任感**：这是获得理想结果的关键。

**总结**：本节课中，我们学习了人际交往中所需的三种心态：筛选心态、伴侣心态和双赢心态。通过掌握这些心态，我们可以更好地与人交往，获得理想的结果。
