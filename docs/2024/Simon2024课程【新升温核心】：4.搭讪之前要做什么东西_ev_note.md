# Simon2024课程【新升温核心】：4.搭讪之前要做什么东西_ev

## 概述
在本节课中，我们将学习在搭讪之前需要做哪些准备工作，以及如何克服焦虑，提高社交技巧。

## 打开社交状态
以下是搭讪之前需要做的几点：

1. **打开社交状态**：在日常生活中，我们都会进行一些社交行为，如问路或寻求帮助。这些都是打开社交状态的好方法。
2. **寻找雇佣者**：去商店或餐厅时，店员会主动与你交流，这可以帮助你练习社交技巧。
3. **让自己快乐**：当你感到快乐时，你会更愿意与人分享，这有助于打开社交状态。
4. **热身练习**：尝试进行一些简单的社交互动，如打招呼或询问对方的情况，以减轻焦虑。

## 克服焦虑
以下是克服焦虑的方法：

1. **目标性焦虑**：对于目标性焦虑，我们需要认识到，一个人的价值不仅仅在于外表，而在于其内在品质。在与他人互动时，要关注对方的内在价值。
2. **非目标性焦虑**：非目标性焦虑通常源于对他人眼光的担忧。要意识到，大多数人并不会关注你的行为，即使他们看到了，也会很快忘记。

## 搭讪的核心部分
搭讪可以分为以下几个部分：

1. **言语框架**：在搭讪时，要注意自己的语言表达，避免使用过于直接或冒犯性的话语。
2. **互动了解**：在搭讪过程中，要积极与对方互动，了解对方的兴趣和想法。

## 总结
本节课中，我们一起学习了搭讪之前需要做的准备工作，以及如何克服焦虑，提高社交技巧。希望这些内容能帮助你更好地进行社交互动。
