## 认识生理性别与第一性征

上一节我们介绍了青少年的整体发育。本节中，我们来看看区分男女生理性别的第一性征。

第一性征是指与生俱来的性器官。男生拥有外生殖器，女生则拥有外阴部和隆起的胸部。这些特征通常在男生约12岁、女生约10岁时开始伴随第二性征出现明显变化。

## 外生殖器的构造与认识

了解自身结构是避免接收错误信息的第一步。以下是男性外生殖器的主要组成部分及其功能。

### 阴茎的构造

阴茎内部并非由骨骼构成。通过触摸，你可以感受到以下结构：
*   **尿道海绵体**：位于阴茎下方，内部包含尿道，是排尿和射精的共同通道。
*   **阴茎海绵体**：左右各一，位于尿道海绵体两侧。它们是阴茎勃起的主要结构。

### 阴茎的发育与形态

当青春期开始，男性荷尔蒙分泌会促使阴茎海绵体生长。其生长速度和最终长度因人而异，且左右两侧的生长可能不同步。

这会导致勃起时阴茎可能偏向左侧、右侧或保持正中。**公式：勃起方向 = f(左右海绵体发育长度差)**。所有方向都属于正常范围，就像人的身高脚长各有不同，关键在于健康成熟地成长。

### 影响发育的因素与护理

1.  **健康饮食**：发育期间应减少高油脂食物（如薯条、炸鸡）的摄入。因为过量油脂中的某些成分可能类似女性荷尔蒙，理论上可能干扰男性荷尔蒙作用，从而影响阴茎在遗传决定范围内的充分发育。成年后，阴茎长度将固定。
2.  **温和清洁**：清洗时切勿用力搓揉，避免损伤。

### 阴囊与睾丸

阴茎下方的囊袋称为**阴囊**，其内部容纳着**睾丸**。

阴囊通常呈现一高一低的状态，这是为了避免行走时两颗睾丸相互摩擦，属于正常生理结构。睾丸是人体唯一外露的实质性器官，阴囊通过调节温度，为精子提供适宜的存活环境。

### 包皮

包皮长度因人而异，可能覆盖或部分覆盖龟头。是否需要割包皮，取决于个人具体的生理状况（如是否反复发炎或包茎），并非必需。

## 至关重要的清洁习惯

无论是否有性行为，从学会自己洗澡开始，都应养成清洁生殖器的习惯，这能预防许多感染和疾病。

清洁方法如下：
1.  洗澡时，将包皮轻轻向后推开，露出龟头及**冠状沟**（龟头后方的一圈凹陷）。
2.  使用清水冲洗冠状沟区域，清除积存的污垢。
3.  **避免**使用肥皂或沐浴露直接清洗该处粘膜，以免因过度刺激或干燥导致发痒、破皮乃至感染。
4.  清洗动作应轻柔，可用手指蘸水轻轻拂过即可。

## 内生殖器的功能

内生殖器位于体内，负责精子的生成、成熟、储存和运输。

以下是内生殖器的主要部件与功能：
*   **睾丸**：核心器官。功能一：分泌**男性荷尔蒙（睾酮）**；功能二：**制造精子**。
*   **附睾**：附着在睾丸上，是**储存成熟精子**的地方。
*   **输精管**：负责将成熟精子从附睾输送出去。
*   **精囊**：储存精液的部分成分，射精前精子会在此与精囊液混合。
*   **前列腺**：分泌**碱性液体**，构成精液的一部分，中和阴道酸性环境以保护精子。
*   **尿道球腺**：在性兴奋时分泌少量透明液体（**尿道球腺液**，常被误称为“前列腺液”），起润滑作用。**注意**：射精前分泌的兴奋液中可能混有尿道内残余的少量精子，因此为安全起见，勃起后即应使用保险套以避孕和防病。

## 每月一次的睾丸自检 🩺

与关注阴茎长度相比，定期检查睾丸健康更为重要。睾丸负责维持男性特征和生育能力。

建议每月固定一天（如每月1号），在洗澡时进行自检，因为温水让阴囊松弛，易于触摸。

自检步骤：
1.  用拇指和食指、中指轻轻捏住睾丸。
2.  以不引起疼痛的力度，轻轻滚动睾丸，感受其整体状况。
3.  正常睾丸应触感均匀、有弹性、无硬块或明显突起物。

大小参考：用手比划一个“OK”手势形成的圈，睾丸应能刚好通过该圈。左右睾丸大小通常略有差异，在2厘米以内的差别均属正常。若发现硬块、大小异常或剧痛，应及时咨询泌尿科医生。

## 射精与排尿的生理机制

为什么射精和排尿不能同时进行？这与身体的精密控制有关。

*   **排尿时**：膀胱颈口打开，尿液经尿道排出。
*   **射精时**：膀胱颈口关闭，精液经尿道射出。

**代码逻辑类比：**
```plaintext
if (动作 == “排尿”) {
    膀胱颈.状态 = “打开”;
} else if (动作 == “射精”) {
    膀胱颈.状态 = “关闭”;
    启动射精流程();
}
```
两者路径在尿道交汇，但通过膀胱颈的开关控制实现分离。若射精时有快感但无精液射出（逆行射精），可能是膀胱颈未正常关闭，导致精液逆流入膀胱，此时需就医检查。
