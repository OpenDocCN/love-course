# 《朕哥8套情感合集》：01.核心私密课之高情商聊天术——第八章：聊天节奏的把控

🎼Yeah。🎼本章我们将探讨聊天节奏的把控，这是非常重要的一个环节。聊天其实是有节奏的，下面我们来详细讲解。

### 渗透法

对于外向型的女生，建议使用渗透法。这意味着在与男生聊天时，采取进三退二的策略，例如，主动热情三天，然后冷却两天，再进二退一，如此循环。渗透的过程中，通过热情和冷淡的交替，不要让男生摸清你的节奏。

**公式**：渗透法 = 热情 - 冷淡 - 热情 - 冷淡 - ...

### 爆破法

对于内向型的女生，建议使用爆破法。集中力量，在短时间内爆发大量热情，例如，在一周或两周内频繁联系男生，给他发送大量信息。

**代码**：爆破法 = 长时间聊天 - 短时间内大量热情

### 引导情绪付出

当男生需求度很高时，引导他表达爱意和付出情绪。例如，当男生说“我好想你陪陪我聊天”时，你可以反问“你有多想我，我怎么没有感觉到呢？”让他表达对你的爱。

### 撩拨与刺激

当男生需求度不高时，通过文字、听觉或视觉刺激他，让他对你产生需求。例如，描述对过去激情的怀念，或对未来的期待。

### 相当原则

在聊天节奏中，遵循相当原则，包括回复字数、投入的真诚、主动性、地位和价值等。

**公式**：相当原则 = 回复字数相当 + 真诚相当 + 主动性相当 + 地位相当 + 价值相当

### 总结

本节课中，我们学习了聊天节奏的把控，包括渗透法、爆破法、引导情绪付出、撩拨与刺激以及相当原则。通过掌握这些技巧，我们可以更好地与男生交流，建立更深入的关系。
