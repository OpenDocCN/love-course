# 富山山女性成长课程：36_破圈链接姐姐

## 概述
在本节课中，我们将学习如何通过链接优秀的姐姐来拓展社交圈，提升个人魅力。

## 方法
以下是一些拓展社交圈的方法：

### 1. 寻找优秀的姐姐
- **寻找目标**：寻找在商业、媒体等领域有影响力的姐姐。
- **分析环境**：电视台等地方是结识各色人等的理想场所。

### 2. 主动发起链接
- **节日祝福**：在节日或周末发送祝福，如“周末愉快”。
- **重新联系**：主动联系常年不联系的人，如“好久不见，姐姐，改天我们一起喝个下午茶”。

### 3. 注意事项
- **识人**：避免与充满“词净”味道的姐姐交往。
- **选择姐姐**：选择性格大大咧咧、有大哥气质的姐姐。

### 4. 与姐姐交往
- **主动买单**：在饭局或活动中主动买单，避免尴尬。
- **分享资讯**：分享新鲜资讯，提升姐姐的形象。
- **送礼物**：根据姐姐的喜好送礼物，如餐具、水晶杯、饰品等。

### 5. 深层次交往
- **利益交换**：让自己有价值，提升个人魅力。

## 总结
本节课中我们一起学习了如何通过链接优秀的姐姐来拓展社交圈，提升个人魅力。希望这些方法能帮助你在社交场合更加得心应手。
