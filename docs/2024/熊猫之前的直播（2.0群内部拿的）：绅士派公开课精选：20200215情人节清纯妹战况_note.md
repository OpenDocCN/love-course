## 核心概念解析：价值评估与关系本质

上一节我们概述了课程内容，本节中我们来看看关系的核心本质。一段关系的建立，本质上是一种价值交换。对方提出任何要求时，重点不在于要求本身是否“合理”，而在于**提出要求的人自身价值几何**。例如，林志玲要求20万彩礼，多数人不会觉得贵，因为她的个人价值足够高。

一个人选择与你建立关系，必然有所图谋。可能是你的外貌、相处时的舒适感、你对女性的理解，或是经济实力。你不太可能与一个和你价值完全对等的人恋爱，通常会选择价值略高于你的人。因此，当对方“开价”时，许多男性吃亏的点在于：**对方看准了你愿意为其支付高价，而她的实际价值可能并不匹配这个价格**。
