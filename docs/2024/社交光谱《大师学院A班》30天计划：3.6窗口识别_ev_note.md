# 社交光谱《大师学院A班》30天计划：3.6窗口识别

## 概述

在本节课中，我们将学习如何识别社交互动中的窗口，包括好奇、兴趣和好感标志，以及如何根据这些标志进行有效的社交互动。

## 好奇标志

**定义**： 发出疑问，回复多，回复快，有情绪。

**公式**： 好奇标志 = 发出疑问 + 回复多 + 回复快 + 有情绪

**示例**： 当你提出一个有趣的问题，对方积极回应并表现出兴趣和情绪时，这表明他们对你好奇。

## 兴趣标志

**定义**： 从被动变得主动，主动撒娇调侃，主动连续发问，主动问你的情感经历，废物测试，对你进行夸赞和认可，表示崇拜。

**公式**： 兴趣标志 = 主动行为 + 情感表达 + 互动行为

**示例**： 当对方开始主动提问，分享个人经历，并对你的某些特质表示赞赏时，这表明他们对你有兴趣。

## 好感标志

**定义**： 占有欲，主动开启性话题，或者是去主动邀约你，寻求安全感，积极的接受你的框架。

**公式**： 好感标志 = 占有欲 + 主动行为 + 安全感需求

**示例**： 当对方表现出对你强烈的占有欲，主动邀请你外出，并寻求你的陪伴时，这表明他们对你有好感。

## 社交上钩点

**定义**： 对方对你产生好奇，并主动与你互动。

**公式**： 社交上钩点 = 好奇 + 主动互动

**示例**： 当你在街上搭讪一个女孩，她停下来与你交谈，这表明她对你产生了好奇，并愿意与你互动。

## 投资与舒适感

**定义**： 通过投资和创造舒适感来吸引对方。

**公式**： 投资与舒适感 = 投资行为 + 舒适感创造

**示例**： 通过分享个人经历，倾听对方的故事，并营造轻松愉快的氛围，可以吸引对方并建立舒适感。

## 升高关系

**定义**： 将关系提升到更高层次。

**公式**： 升高关系 = 深入话题 + 情感表达

**示例**： 通过分享个人观点，表达情感，并引导对方进行深入的对话，可以将关系提升到更高层次。

## 总结

在本节课中，我们学习了如何识别社交互动中的窗口，包括好奇、兴趣和好感标志，以及如何根据这些标志进行有效的社交互动。通过理解这些概念，你可以更好地与他人建立联系，并提升你的社交技巧。
