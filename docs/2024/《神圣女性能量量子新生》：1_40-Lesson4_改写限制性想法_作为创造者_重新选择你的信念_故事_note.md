# 《神圣女性能量量子新生》：1_40-Lesson4_改写限制性想法_作为创造者_重新选择你的信念故事_在3D世界的身份

## 概述

在本节课中，我们将学习如何改写限制性想法，成为创造者，重新选择你的信念故事，以及你在3D世界的身份。

## 改写限制性想法的前提

以下是改写限制性想法的五个前提：

1. **意识到过去的累积**：认识到你的过去累积了来自3D世界的创伤、编程和过时的限制性信念。
2. **与自我分离**：能够将自我从过去的形象、故事小我以及性格中剥离出来。
3. **意识到选择权**：意识到作为永恒的意识，你在3D世界有选择权，可以选择新的故事、信念和身份。
4. **注意到反面故事**：在现实生活中注意到限制性信念的反面故事。
5. **决定改变**：决定做出改变，并承担起个人成长和内在工作的全部责任。

## 改写限制性想法的六步

以下是改写限制性想法的六个步骤：

1. **找到陈旧的编程和模式**：识别出过时的、不再适用的限制性想法。
2. **陈辅祷告**：借助高维力量帮助你找到想要拥抱的新故事。
3. **做出神圣的决定**：在日记中写下新的故事，为自己选择新的故事。
4. **导演人生的新电影**：闭上眼睛，想象自己活出新故事，注意所有细节。
5. **通过肯定句肯定新的信念**：例如，“我是值得的”、“我是足够的”、“我是富有的”等。
6. **重复数百万次**：直到潜意识记住新的真相和新的系统。

## 主动性改写和被动性改写

主动性改写：察觉到自己目前的生活中有卡点，需要突破，但不知道如何下手。

被动性改写：在日常生活中被某个人的某句话、某个场景触发，身体有紧缩感或未被消化的情绪。

## 常见的学习错误

1. **一知半解**：接受新概念后，没有深入学习和实践。
2. **没有显化性想法**：不了解自己，离自己核心部分较远。
3. **手高眼低**：接收信息后，不做练习，仅仅在头脑里做。
4. **坚持一段时间没有效果**：没有意识到改变需要时间。

## 小我让你分新的把戏

1. **提出不必要的问题**：让你陷入思考，无法采取行动。
2. **显化出一些 drama 码或 toxic 的人或情形**：让你陷入情绪中，无法成为观察者。
3. **生病或疲劳**：让你无法进行内在工作。
4. **对复杂和困难上瘾**：让你沉迷于挣扎，无法轻松解决问题。

## 如何释放羞耻和内疚

1. **祈祷**：向宇宙、天使等寻求帮助，释放内疚和恐惧。
2. **改写限制性信念**：将羞耻和内疚转化为爱。

本节课中，我们学习了如何改写限制性想法，成为创造者，重新选择你的信念故事，以及你在3D世界的身份。希望你能将所学知识应用到实际生活中，不断成长和进步。
