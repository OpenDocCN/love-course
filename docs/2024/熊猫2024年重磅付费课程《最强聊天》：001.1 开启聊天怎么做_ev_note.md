# 熊猫2024年重磅付费课程《最强聊天》：001.1 开启聊天怎么做

## 概述

在本节课中，我们将学习如何开启聊天，包括表情包的使用、分享状态、描述状态以及如何应对各种情况。

## 第一步：表情包

**公式**： 表情包 = 可爱表情包

**步骤**：

1. 使用可爱表情包作为开场。
2. 如果没有表情包，扫描二维码获取。

## 第二步：分享状态

**公式**： 分享状态 = 我在干嘛？ + 对方状态

**步骤**：

1. 使用“我在干嘛？”询问对方状态。
2. 根据对方回答，了解对方状态。

**注意事项**：

* 避免使用自我中心的表达，如“我在吃饭”。
* 分享状态应是一个闲适的状态，如“我在发呆”。

## 第三步：描述状态

**公式**： 描述状态 = 我在干嘛？ + 我很无聊

**步骤**：

1. 根据对方回答，描述自己的状态。
2. 表达自己的无聊，如“我很无聊”。

**注意事项**：

* 使用“好吗？”强调自己的无聊。
* 避免使用“无聊”等负面词汇。

## 第四步：应对各种情况

**公式**：

* 对方回答：我在看书
  * 我：我知道你在看书，太无聊了。
* 对方回答：我在看剧
  * 我：我知道你在看剧，太无聊了。
* 对方回答：我在躺着
  * 我：我知道你在躺着，太无聊了。
* 对方主动开启话题
  * 我：根据对方话题进行回应。

## 总结

本节课中，我们学习了如何开启聊天，包括表情包的使用、分享状态、描述状态以及如何应对各种情况。通过学习这些技巧，我们可以更好地与对方交流，提升聊天效果。
