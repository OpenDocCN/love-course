# 21、凯男的社交情商课：第一节课：爱商提升的7个思维逻辑：003.3、奖品思维【微kc5

在本节课中，我们将要学习“奖品思维”这一核心概念。这是一种颠覆传统恋爱观念的态度，旨在帮助你建立自信，在互动中占据主动，并引导对方来追逐你。我们将通过定义、实例分析和具体话术，让你理解并掌握这一思维。

---

上一节我们介绍了“步骤思维”，强调了恋爱中谋略与规划的重要性。本节中，我们来看看如何通过改变自身定位来彻底扭转互动模式，这就是“奖品思维”。

奖品思维表达的是一种态度。它表达了对自己的满意，并相信别人会因你的价值而主动接近你。同时，奖品思维是一种极其利于你自己获得框架、得到窗口并占据主动的互动形式。

用直白的话来说，在男女互动中，**我们要做对方的奖品，而不是让对方做我们的奖品**。

为了理解这个概念，我们先看看大多数女性的天然思维模式。

其实大部分女生天生就明白如何让自己成为男人的奖品，并让男人来追逐她们。在很多女生的内心深处，都有这样一个想法：她们认为只要一有机会，男人都想要和她们发生关系。她们总认为是她们在选择或者接受男人，而男人需要为此付出代价。

以下是三个常见的例子，帮助理解这种动态：

*   通常都是男人试图约会女生，女人有权利接受或者拒绝。
*   通常都是男人请女人吃饭或者看电影，女人有权接受或者拒绝。
*   通常都是男人向女人表白或者求婚，女人有权利接受或者拒绝。

这也是为什么，即便一个女生对某个男人感兴趣，她也不会主动追求，而是会试图让男人来追求自己。例如，她会做出一些事情让男人注意到自己，然后期待对方“上钩”。

她们很好地把自己摆在了“奖品”的位置上。因此，作为男性，我们需要学习同样的思维。

现在，我们通过一个场景来具体分析。

假设你在酒吧喝酒，旁边坐了一位肤白貌美的女生。她发现你看了她几眼，然后走过来对你说：“你要请我喝一杯吗？”

以下是常见的错误回答及其分析：

**错误回答**：`“好啊，我请你。”`

这是一个非常典型的错误。女生说“你要请我喝一杯吗”，其背后的逻辑是：“你要来取悦我，我是奖品，你要来请我喝酒来追我。”如果你顺从并迎合她，在女人眼里，你跟其他大部分男人都一样，没有挑战性。她可能只是利用你的价值喝一杯酒，然后就会离开，你不会有任何吸引到她的机会。

那么，运用奖品思维应该如何回答呢？

**正确回答（示例）**：`“你都是用这种方法在酒吧搭讪男生的吗？”`

这句话背后的逻辑不仅没有符合她，反而运用了奖品思维去回应：**“是你来主动搭讪我，我是奖品，你被我吸引。”** 这样的回答把自己当成了奖品，引导女生来追逐，并给了对方极大的挑战性，让她觉得你与众不同。此时，你吸引到她的机会就大大增加了。

你与女人相处时，一句话、一个回答就能体现你是一个有奖品性、自信、高价值的男人，还是一个没有奖品性、低价值的男人。如果你事事选择迎合和顺从，你所有的潜台词都是在告诉对方：“你对我来说真的很重要，你是一个女神，你是奖品，我需要努力追上你。”

女人如果感知到这种潜台词，她会想：“这个男人也就这样，跟其他男人都一样，丝毫不具备任何挑战性，也勾不起我的兴趣。就让他追着我跑，免费给我送东西吧。” 这就是很多人一步步沦为备胎的原因。

但是，如果你拥有很好的奖品思维，就能让女生反过来追逐你。

以下是更多运用奖品思维的回答示例，供你体会其威力：

**场景一：女生夸赞你**
*   **女生说**：`“你今天的发型好帅啊。”`
*   **错误回答（谦虚/低价值）**：`“过奖了”` 或 `“没有没有”`。这样的回答在女人眼里毫无奖品性，会让对话平淡结束。
*   **奖品思维回答（示例1）**：`“你是不是每天晚上都看着我的照片发呆啊？”`
*   **奖品思维回答（示例2）**：`“呦，是不是每天都看着我的照片流口水？”`
*   **逻辑分析**：这两种回复背后的逻辑是告诉女生：**“我是奖品，是你在追我，你已经被我的魅力迷住了。”**

**场景二：女生说你像某个明星**
*   **女生说**：`“我发现你长得有点像彭于晏。”`
*   **奖品思维回答（示例1）**：`“你这样说我会害羞的，万一你爱上我了怎么办？”`
*   **奖品思维回答（示例2）**：`“那给你一次和大牌亲密接触的机会吧，还不赶紧抱抱我？”`
*   **逻辑分析**：背后的表达逻辑依然是：**“我是奖品（大牌），我在给你追逐我的机会。”**

普通男人不会做这些事情，也不会说这些话。对于普通男人来说，好像有女生到身边，他就要去谄媚、去追、去夸，把对方当奖品。这样做，女人自然不会感兴趣，因为这样没骨气、没有吸引力的男人，她每天会遇到很多个。
