# Leon2.0：7.搭讪质量提升 🤝

在本节课中，我们将学习如何提升搭讪的质量和效率，以便更有效地获取女生的联系方式并安排约会。

## 搭讪的本质：价值的互换

搭讪的本质是连接，而连接的本质是价值的互换。为了提升搭讪质量，我们需要释放更多价值。

## 女生在意的价值

女生在意的价值分为两个方面：性价值和社交价值。

### 性价值

**外在性价值**：包括身高、长相、骨架、身形等天生因素，以及身材、穿搭等可改变因素。

**内在性价值**：雄性魅力，体现在自信、沟通、坚定、强势等方面。

### 社交价值

女生无法直接感受到你的社交价值，而是通过以下线索来判断：

1. **外形线索**：打扮、穿搭、衣品等。
2. **健身线索**：体现对自我的要求和社会价值。
3. **言语线索**：讲话的逻辑、停顿、节奏、语调、语言习惯等。
4. **浅沟通线索**：坚定、放松、自信、有力等。
5. **社交线索**：自信、放松、幽默、情绪等。
6. **社交认证线索**：社交场合、朋友、朋友圈等。

## 提升搭讪质量的细节

1. **节奏**：放慢讲话节奏，加入停顿和重音，显得更加放松和自信。
2. **坚定**：保持关注，全身心投入互动，避免眼神游移和手机依赖。
3. **情绪**：表达情绪，无论是正面还是负面，以建立真诚的连接。
4. **浅沟通**：从意识层面改变内心认知，释放积极、健康的能量。
5. **穿搭**：选择适合的服装，体现对生活的品味和自我要求。

## 任务

1. 有意识地压慢讲话节奏，并不断优化。
2. 在互动过程中保持关注，避免手机依赖。

4. 观察自己的浅沟通，并改变内心认知。

本节课中，我们学习了如何提升搭讪的质量和效率。通过释放更多价值、关注细节，我们可以更好地与女生建立连接，并提升自己的吸引力。
