# 清风老师合集：07关系修复，破镜重圆11期：077.第六课：挽回核心技术（中）让他主动联系你

## 概述
在本节课中，我们将学习如何通过展示术、高情商聊天术、反思术和情感理由测试术来挽回关系，并让对方主动联系你。

## 展示术
**核心概念**：通过展示方式让对方从排斥期脱离，进入怀念期。

**展示内容**：
- **自私**：展现付出和奉献，如礼物、未来规划。
- **不成长**：展现学习成长和改变，如健身、学英语。
- **不好看**：提升穿着打扮、气质，如学舞蹈。

**展示方式**：
1. 社交软件（微博、抖音、朋友圈）。
2. 共同朋友转述或照片。
3. 直接对话。

**展示频率**：建议一周3次。

## 高情商聊天术
**核心概念**：在怀念期制造聊天循环，让聊天不停下来。

**聊天步骤**：
1. 提出话题。
2. 获得反馈。
3. 关注情绪。
4. 等对方抛出话题。

**注意事项**：
- 避免过于小心翼翼。
- 关注对方真正喜欢的东西。
- 聊过去感兴趣的话题。

## 反思术
**核心概念**：将变化呈现到对方面前。

**反思层次**：
1. **浅层**：反思过去的不足（自私、学习、气质形象）。
2. **中层**：反思成长缺失和存在的问题。
3. **深层**：反思对未来期待和规划。

## 情感理由测试术
**核心概念**：通过情感理由测试拉近关系。

**推荐话题**：
1. 对未来情感的规划。
2. 对过去感情的看法。
3. 对自己为人、工作的看法。
4. 对性的看法。

## 总结
本节课中，我们学习了展示术、高情商聊天术、反思术和情感理由测试术，这些技术可以帮助你挽回关系，并让对方主动联系你。希望你在实践中能够灵活运用，取得成功。
