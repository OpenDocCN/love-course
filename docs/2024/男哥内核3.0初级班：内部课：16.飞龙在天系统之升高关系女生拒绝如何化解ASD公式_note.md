# 男哥内核3.0初级班：内部课：16.飞龙在天系统之升高关系女生拒绝如何化解ASD公式

## 概述

在本节课中，我们将学习如何化解女生在升高关系过程中出现的拒绝，特别是针对ASD（矜持拒绝）的情况。

## ASD与IOD

**ASD**：对方对你有较高的兴趣度，但你的话过于敏感或对方过于矜持而拒绝你。

**IOD**：对方对你没有兴趣，即使话多或使用表情包。

**公式**：

- **ASD**：对方矜持，可通过激发ASD公式化解。
- **IOD**：对方无兴趣，需退回舒适阶段。

## 激发ASD

**步骤**：

1. **表达属性**：通过表达不纠结、不纠缠、不解释等属性，让对方感受到你的框架。
2. **主线任务**：在互动过程中，通过聊自己来引导对方关注你。
3. **病毒聊天**：通过进一退二或进一退一的方式，调整话题节奏，让对方围绕你。

## 激发ASD的具体操作

1. **表达属性**：
    - **例子**：你说“谁要跟你牵手啊”，她说“以亿谁要跟你碰啊”，你可以说“今天心情好，谁要跟你牵手啊”。
    - **关键**：表达不纠结、不纠缠、不解释的态度。

2. **主线任务**：
    - **例子**：你说“总算下班了”，她说“哼”，你可以说“可算到家了，总算下班了”。
    - **关键**：聊自己，引导对方关注你。

3. **病毒聊天**：
    - **进一退二**：在升高关系过程中，如果对方拒绝，可以聊他聊我聊他聊我，然后退两步。
    - **进一退一**：在化解ASD后，可以退一步，然后继续升高关系。

## 总结

本节课介绍了如何化解女生在升高关系过程中出现的拒绝，特别是针对ASD的情况。通过激发ASD公式，可以有效地化解矜持拒绝，并继续升高关系。
