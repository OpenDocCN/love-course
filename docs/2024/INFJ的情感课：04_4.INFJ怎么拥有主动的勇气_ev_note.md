# INFJ的情感课：04_4.INFJ如何拥有主动的勇气_ev

🎼大家好，我是升级创始人理事。本期我们将探讨如何拥有主动的勇气。在前几讲中，我们介绍了灵活伴侣的标准，许多同学可能在评论区找到了与自己相符的伴侣类型，或者在生活中感受到了一些希望。然而，如何主动追求心仪的对象，是许多INFJ所面临的难题。

**主动性对INFJ的重要性**

对于INFJ来说，主动性在情感中尤为重要。他们往往在特别信任和喜欢一个人时才会主动，一旦有所顾虑，便不会主动出击。然而，等到彻底放心时，对方可能已经有所归属。因此，尽早出击，主动追求，是关键。

**增强勇气的途径**

在了解了前几讲的内容后，你可能已经增加了遇到灵魂伴侣的几率，并提高了对方了解你的可能性。但更重要的是，你需要增强自己的勇气。勇气来源于底气，而底气则来源于知识和信息。通过学习更多知识，了解不同类型的人，你会更有底气，从而带来更多的勇气。

**《被讨厌的勇气》的重要性**

我强烈建议INFJ阅读《被讨厌的勇气》这本书。它讲述了阿德勒心理学的思想，对于INFJ来说，这是一本必修的心态之书。

**被拒绝的勇气**

为什么INFJ需要被拒绝的勇气呢？因为生活中，我们往往围绕回避被拒绝和拒绝他人这两种情况而生活。我们害怕被拒绝，也害怕拒绝他人。面对拒绝，我们更多采取的是恐惧和躲避的态度。但只有面对拒绝，我们才能真正获得勇气。

**如何面对拒绝**

以下是一些面对拒绝的方法：

1. **否定自己**：在内心里否定自己，承认自己的普通，从而减少自我怀疑。
2. **做好被否定的心理准备**：不要给自己过高的评价，做好被拒绝的心理准备。
3. **否定自己的优势**：将你认为的优势点否定掉，从而减少被拒绝的伤害。

**总结**

在本节课中，我们学习了如何拥有主动的勇气。通过增强底气、阅读《被讨厌的勇气》以及面对拒绝，INFJ可以更好地追求心仪的对象。希望这节课对你有所帮助。我们下期再见，拜拜！
