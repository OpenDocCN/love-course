# 海蓝老师情绪梳理个案课：024-1.我要抓紧每一分钟，不能停-

## 概述
在本节课中，我们将探讨时间流逝感带来的焦虑情绪，并通过个案分析，学习如何理解和处理这种情绪。

## 案例分析
### 1. 时间流逝感与焦虑
**主题**：焦虑，时间过得特别快，感觉学不完。

**感受**：心口堵，头蒙，感觉无从下手。

**核心概念**：
- **时间流逝感**：感觉时间过得飞快，无法把握。
- **焦虑**：对未来的担忧和不安。

### 2. 回顾过去
**主题**：19年前，刚开始做公司时的焦虑。

**感受**：焦虑，紧张，顾不上孩子和公司。

**核心概念**：
- **压力**：来自工作和家庭的双重压力。
- **责任感**：对孩子和公司的责任感。

### 3. 情绪体验
**主题**：看到19年前的自己，感受是什么？

**感受**：心疼，忙碌，想靠近又想远离。

**核心概念**：
- **同理心**：对过去的自己产生共鸣。
- **距离感**：意识到过去的自己已经改变。

### 4. 情绪处理
**主题**：如何处理焦虑情绪？

**感受**：温暖和关怀，评判和不满。

**核心概念**：
- **情绪调节**：学会调整自己的情绪。
- **自我接纳**：接受自己的情绪，并找到解决问题的方法。

## 总结
本节课中，我们一起学习了时间流逝感带来的焦虑情绪，并通过个案分析，了解了如何理解和处理这种情绪。希望同学们能够将所学知识应用到实际生活中，更好地应对各种情绪困扰。
