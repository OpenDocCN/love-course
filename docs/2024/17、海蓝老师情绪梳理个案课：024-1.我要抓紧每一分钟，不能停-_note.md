# 17、海蓝老师情绪梳理个案课：024-1.我要抓紧每一分钟，不能停-_note

在本节课中，我们将跟随一位学员的个案，共同探索由“时间紧迫感”引发的深层焦虑。我们将学习如何识别情绪在身体上的反应，追溯情绪的早期根源，并通过与内在自我对话的方式进行梳理。

---

## 概述：焦虑的浮现

学员提出了一个明确的主题：感到特别焦虑，觉得时间过得飞快。尤其是在加入一个为期两年的进修项目后，一打开课程列表，看到繁多的内容，情绪瞬间涌上心头。她担心两年时间肯定学不完，并且感觉一个月不知不觉就过去了，自己却好像什么也没学到。

这是一种由时间压力和对学习成果的担忧混合而成的焦虑感。

## 第一步：定位身体的感受 😌

当强烈的情绪出现时，它往往会在身体上留下印记。引导学员闭上眼睛，去感受“时间赶不上”这种情绪在身体哪个部位最不舒服。

学员的描述是：心口特别堵，头也觉得发蒙。在进一步区分后，头蒙的感觉比心口堵更强烈一些。这种身体的感觉，是情绪最直接的信号。

## 第二步：追溯情绪的根源 🔍

为了理解当下的情绪，我们需要回到过去，寻找最早出现类似感受的时刻。询问学员：“最早有这种感觉是什么时候？还有哪些时候会有同样的感觉？”

学员回忆道，大约在19年前，她30岁时，和爱人一起创办公司。那时她需要独自处理公司内部所有事务，如做标书、人员储备、整理资料等，事情非常多，让她感到无从下手。

## 第三步：与过去的自我对话 👥

引导学员在想象中“看到”19年前那个刚刚创业、忙碌不堪的自己（我们称她为“小红”）。

**以下是引导对话的关键步骤：**

1.  **初始感受**：当看到“小红”时，学员的第一感受是“觉得她挺可怜的”。因为那时孩子还小，公司刚起步，她既顾不上孩子，公司的事情又不得不做，感到孤立无援。
2.  **剥离评判**：当学员想靠近“小红”时，发现内心混杂着“温暖关怀”和“评判不满”（如想劝她、建议她找人帮忙）。我们引导将评判、建议、批评的部分暂时“退后”，只留下纯粹的关怀。
3.  **建立连接**：学员带着温暖和关怀，静静站在忙碌的“小红”身后。她发现“小红”停不下来，也听不见外界的声音，这让她感到难过和心疼。
4.  **倾听“小红”**：当“小红”终于抬起头，感受到关怀时，她开始表达自己的压力。她说公司的事情必须做，资质必须不停升级，否则就无法参与招投标。
5.  **探索恐惧核心**：通过一连串的提问，层层深入“小红”行为背后的恐惧。
    *   **问**：不做工程会怎么样？
    *   **答**：只能往外拿钱，公司做不下去。
    *   **问**：公司做不下去意味着什么？
    *   **答**：两人会没有收入。
    *   **问**：没有收入会怎么样？
    *   **答**：孩子和老人都照顾不了。
    *   **问**：那意味着什么？
    *   **答**：意味着“穷”。
    *   **问**：“穷”对你来说，最深的感受是什么？
    *   **答**：**被人瞧不起**。

## 总结：看见情绪的深层逻辑

本节课中，我们一起学习了一次完整的情绪梳理过程。

1.  **从触发事件开始**：当下的时间焦虑是情绪的导火索。
2.  **连接身体感受**：将模糊的情绪转化为具体的身体感觉（头蒙、心口堵）。
3.  **回溯生命经历**：找到早期形成类似情绪模式的源头事件（19年前创业初期的超负荷压力）。
4.  **与内在自我对话**：通过剥离评判、表达关怀，倾听内在那个“停不下来”的自己的心声。
5.  **抵达核心信念**：最终发现，驱动所有焦虑和忙碌行为的底层恐惧，是一个关于**生存尊严**的核心信念——“不能穷，否则会被人瞧不起”。

这个深层的信念，像是一个隐藏的指令，一直在无形中驱动着行为，导致一旦面临压力（如繁重的学业），就会自动激活“必须抓紧每一分钟，不能停”的焦虑模式。看见它，是理解和转变的第一步。
