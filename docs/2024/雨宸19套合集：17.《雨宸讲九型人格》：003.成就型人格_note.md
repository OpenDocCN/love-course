# 雨宸19套合集：17.《雨宸讲九型人格》：003.成就型人格_note

在本节课中，我们将要学习九型人格中的第三种类型——成就型人格。这种人格也被称为实用型，其核心特征是追求成功、关注结果与现实利益。我们将详细解析其性格特点、成因、在亲密关系中的表现，并提供与之相处的实用建议。

## 概述

成就型人格者以目标和结果为导向，他们渴望成功、名誉与认可。他们的注意力集中在“如何达成目标”上，核心价值观是“我必须努力成为成功的人”。接下来，我们将深入探讨这种人格的各个方面。

## 性格特征

根据资料描述，成就型人格具有以下明确的性格倾向：

*   **基本倾向**：外向、主动、善于交际。
*   **关注点**：关注任务与结果，而非过程的意义。
*   **动力来源**：渴望通过成绩获得认可与接受，喜欢竞争，通过超越他人建立优越感。
*   **行为模式**：坚持目标，效率高，有时为追求效率而牺牲完美或走捷径。
*   **潜在问题**：可能疏忽自身感受，为了事业成功、声望与财富，有时会牺牲情感、婚姻、家庭或朋友。

他们的核心驱动力可以用一个简单的公式表示：**成功 = 认可/地位**。他们的努力很大程度上是为了达成这个等式。

## 性格成因

成就型人格的形成往往与童年经历有关。小时候，当他们完成某些事情时，父母给予了称赞和夸奖，这让他们体验并享受到了被掌声和荣誉包围的感觉。长大成人后，他们可能会将大部分精力用于持续追求这种被认可的状态。

## 在亲密关系中的表现

上一节我们介绍了成就型人格的基本特征，本节中我们来看看他们在亲密关系中的具体表现。

在亲密关系中，成就型人格者可能会显得比较冷漠，伴侣常常看不到他们过多的表情或情绪。你可以将其理解为，他们通常不会轻易相信别人，仿佛总是戴着面具生活。他们表现出来的状态与内心真实感受往往不相符。

因此，在夜深人静时，由于白天长时间“戴面具”，他们可能会被一种强烈的空虚感所笼罩。他们的伴侣可能会感到格格不入，因为成就型人格者将大量时间投入工作、事业和目标追求中，容易忽略家庭和另一半。他们有时会对伴侣的情感诉求视而不见，更愿意逃避情感带来的压力，不愿在情感中付出更多或承担更多责任。

用一个现实的例子来说，在情感中，这类性格的人倾向于用钱解决问题，而非付出过多的精力和情感。无论是对于伴侣还是孩子，他们常常用物质补偿来代替情感的付出。因此，与这类人谈恋爱，伴侣可能会有“找了一个假男朋友/假女朋友”的感觉。

他们的性格在亲密关系中呈现两极化状态：时而得意洋洋、自信满满，表现出极度的自恋；一旦遇到挫折和磨难，又会陷入深深的自卑与沮丧。在沮丧时，他们可能不想说话、不想沟通、不想参与任何事。

## 核心思维模式

成就型人格者的思维模式是彻底的结果导向。他们做任何事都以达成目标为最终目的，结果对他们至关重要。如果结果未能达成，他们会觉得整个过程都失去了意义。这与许多更看重过程的普通人不同。

他们的行为逻辑可以概括为以下代码模式：

```python
if 达成目标(成功、认可、地位):
    感到满足与自豪
else:
    感到挫败、沮丧，认为过程无意义
```

例如，追求一个人时，如果没有追到，他们会认为之前花费的时间都是浪费的。他们通常不会花费大量时间去做一件完全不确定的事，思维中缺乏梦想主义的成分。他们的努力，无论是工作还是感情，往往并非出于真正的热爱或喜欢，而是为了获得某个成就或结果（如升职加薪、证明自己有魅力）。

## 相处与应对策略

了解了成就型人格的特点后，我们来看看如何与他们相处，特别是在伴侣关系中。

以下是与之相处的三个核心原则：

**1. 保持相对独立性**
不要过于黏人或依赖对方。如果你过于亲近和依赖，首先，他会认为你已经完全属于他，无需再追求；其次，他会将你视为负担。对于已经达成的“恋爱”或“婚姻”这一结果，他们可能不愿再为维护关系付出更多努力，除非关系出现问题。保持一定的距离和独立性，让他觉得关系尚有不确定性，或你身上仍有值得征服和探索之处，他才会愿意继续付出情感。

**2. 成为其依靠并真诚互动**
当他陷入沮丧时，你要成为他最强的依靠。同时，针对他“戴面具”的状态，可以适当地、直接地揭穿，然后给予理解与关爱。成就型人格者内心通常比较强大，且原则边界可能不如其他类型清晰，真诚的揭穿与接纳反而可能赢得他的信任。获得这种人的信任非常难，一旦成功，他可能会向你袒露所有不为人知的想法，建立起真正的亲密唯一性。

**3. 展现现实价值**
如果需要进行情感修复或长期维系关系，仅靠情感价值很难打动他。因为他已经将“拥有这段关系”视为一个达成结果，不愿再为已定局的事过多付出。此时，重新展示你的现实价值至关重要。例如，如果他在创业，你能在事业上提供切实帮助；如果他找工作，你能给出独到的专业建议。只要你能在现实层面提供他人无法替代的帮助，你们的关系就很难被拆散。因为对他们而言，**现实利益 > 感情**，这是其性格塑造过程中形成的固有状态。

## 总结

本节课中，我们一起学习了成就型人格的核心特征。他们是以目标和结果为导向的现实主义者，追求成功与认可，在亲密关系中可能显得冷漠且以物质补偿情感。与他们相处的关键在于保持独立性、在其低谷时给予支持并以真诚互动赢得信任，同时务必注重展现自身的现实价值。理解这些特点，能帮助我们更好地与这类人建立和维护关系。
