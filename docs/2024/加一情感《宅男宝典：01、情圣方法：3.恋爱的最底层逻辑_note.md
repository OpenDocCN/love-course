# 加一情感《宅男宝典：01、情圣方法：3.恋爱的最底层逻辑_note

在本节课中，我们将要学习恋爱的最底层逻辑。我们将从进化心理学的角度出发，剖析人类在择偶行为背后的根本原因，特别是女性行为逻辑的根源。理解这些底层逻辑，将帮助你解开“感觉”之谜，看清两性关系的本质。

恋爱在今天是一个复杂的概念，其结果受许多因素影响。有些人喜欢帅的，有些人喜欢有钱的，有些人喜欢踏实的。有些人喜欢有才华的。有些人对女生很好，但女生却不喜欢你。有些渣男对女生很差，女生却深深爱上他。女生经常对我们说“没有感觉”。“感觉”这个东西让男生很头疼。如果说个具体的事情，我们知道如何去改进。但你说“感觉”，我们如何去捕捉呢？这节课我将为大家打开这个谜团。

我们将搞清楚女生最底层的需求到底是什么，还有她的行为逻辑。

大约在2万年前，地球上的人大部分生活在非洲草原上，生活在部落社会。我想问一下，大家觉得那个时候的人跟现在的人差异有多大呢？你可以想象一下，那个时候的人跟现在的人在基因上的差异是千分之一。那个时候的婴儿，如果放在今天的社会环境去生长，和今天的人是一样的。现在的社会很复杂，我们很难看清楚男女恋爱的本质。那么我们就回到那个部落时代，来看爱情到底是怎么样的。

进化心理学告诉我们，人类的身体适应的是远古的环境。我们生活在21世纪，却仍有着石器时代的大脑。近2万年，社会发展得太快。人类的进化远远跟不上社会变化的速度，也就是我们的好多思维模式和潜意识还停留在部落时代。

上一节我们提到了人类的思维模式仍停留在部落时代，本节中我们来看看具体的例子，方便大家去理解。

以下是几个进化心理学在生活中的体现：

*   **对甜食的偏好**：今天好多人喜欢吃甜食，特别是在欧美国家。结果呢把自己吃成了个大胖子，导致一系列的疾病。据统计，在2015年因肥胖导致的死亡就达到400万。喜欢吃甜的人，其实也不知道他为什么喜欢吃甜。但进化心理学家找到了原因。喜欢吃甜的、油腻食物的人，是因为在部落时代，吃了甜的果子能快速补充能量，而不喜欢吃甜的人会很容易被淘汰。虽然喜欢吃甜食的人不知道为什么自己会喜欢吃甜的，但是他保留了这个特点。对他来说，这也是一种“感觉”。
*   **对女性胸部的偏好**：我曾经无数次的问学员一个问题：“你为什么喜欢女生胸大？”知道回答最多的答案是什么吗？是“感觉”，说不出来原因。但是我们绝大多数的男生都喜欢胸大的女生。有些人说可能和哺育下一代有关系。但其实研究表明，胸部的大小和产奶量并没有关系。同样，进化心理学家给了我们答案。在部落时代，我们还没有文胸这个东西的时候，女生胸部大，只要年纪稍微大一点，她就会下垂得很厉害。胸小的就不会。而胸大并且坚挺的，说明她在繁殖的黄金年纪。喜欢它的原因就是为了更好的繁殖。大家看一下附属文稿，我有份原始部落的图片，脸看年纪真的看不出来，但是胸就很直观。
*   **对女性长发的偏好**：那你知道为什么男生都喜欢女生长发吗？人还是那个人，什么都没变，为什么头发剪短了就不受欢迎了呢？剪短发的女生有一个算一个，没有一个不后悔的，因为大多数的男生不喜欢。我们能给出的答案也是两个字：“感觉”。女生对这个“感觉”是不是也很无奈？女生头发一年长15到20厘米。如果一个女生想拥有一头健康的长发，她要3到4年的时间。那就要求女生3到4年的时间没有疾病，因为在那个时候没有婚前检查。头发是一个判断健康的重要标准。
*   **对金发碧眼的偏好**：包括欧洲人喜欢金发碧眼，因为随着年纪的增大，欧洲人的发色、眼睛的颜色都会变深。所以呢，如果这个女生是金发碧眼，那么她一定是在最佳的繁殖年纪，所以他们喜欢。但是对我们亚洲人来说，金发碧眼是没有感觉的。

我们可以看到，我们的“感觉”是有最底层的原因的，只是放在今天的社会，社会环境变了，我们不理解了。

那么，了解了这些普遍偏好背后的逻辑后，我们来看看在两性关系的底层是怎样的。女生又是怎么样考虑的呢？

首先大家想一个问题：如果一个女生要生10个小孩，要多长时间，要经历些什么？如果一个男生要生10个小孩，要多长时间，要经历些什么？

女生差不多要10年时间，而且这十年她一直在怀孕。生活不便，没有生存能力，而且众多的小孩怎么吃饭养活，都是要投入巨大的精力跟时间的。再来看看男生。理论上，他们要生10个小孩，只要一个晚上。

可以看到，男人跟女人在繁殖成本上有着巨大的差异。这种差异呢，就决定了他们有着完全不同的择偶方式。

**男性的繁殖策略是：播种。**
**女性的繁殖策略是：选种。**

男性会不停地找繁殖价值高的、健康的女生去交配，从而达到他广泛传播基因的目的。所以男人的繁殖方式是播种。而女人的生育成本高，他们选择的方式就不是播种，而是选种。所以划重点了：**男人的繁殖方式是播种，女人的繁殖方式是选种。** 在今天，这也是为什么男人花心好色。

接下来我们来看看，在女生这种“选种”的繁殖方式下，她们的择偶观是怎样的。

以下是女性在部落时代形成的两大核心择偶标准：

1.  **找优秀的男人（猎人HT）**：在部落时代，什么样的男人是优秀的呢？答案是猎人：身材强壮、会打猎的。这样的人会让她衣食无忧，特别是在怀孕期间、环境恶劣的时候不至于饿死。
2.  **找靠谱的男人（靠谱DP）**：如果这个男人会打猎，但是不可靠，交配完就不管她了。那么她跟她的小孩一样要饿死。

而没有这样选择的女生会怎样？他们会在漫长的进化过程中被慢慢地淘汰。所以保留下来的女性基因，就是物质现实的。所以呢，不是“哪一个女生物质现实”，而是“女生的底层潜意识，都是物质现实的”。

所以，要画这节课的重点了：**女生的最底层的需求是：猎人HT（优秀/有能力）和靠谱DP（可靠/有担当）。**

猎人的定义在今天有变化，后面的课程我们会细讲，但简单的理解就是优秀的男人，有能力的男人。

那么女生所谓的“没感觉”是什么呢？她自己可能说不出来，其实就是跟这两条违背的。她感觉你不够优秀，或者感觉你不够靠谱。

我们来看看那些大家传说的女生喜欢的特质，比如说有钱、乐观、幽默、上进、努力、拼搏，哪一条不是优秀男生的品质呢？包括你说的高和帅。但这里高和帅呢，我要先卖个关子，我们在讲吸引力的时候再讲。

本节课中，我们一起学习了恋爱的最底层逻辑。我们从进化心理学的视角出发，理解了现代人的许多“感觉”其实源于远古的生存与繁殖策略。核心结论是：由于繁殖成本的巨大差异，男性进化出了“播种”策略，而女性进化出了“选种”策略。这导致了女性最底层的两大需求：**寻找优秀的“猎人”（HT）和可靠的“伙伴”（DP）**。所有让女生产生“感觉”的特质，几乎都指向这两个核心。理解这一点，是理解两性互动、提升吸引力的根本起点。
