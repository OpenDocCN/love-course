# 老白《搞钱课》：9、手机端动画剪辑课_note

在本节课中，我们将学习如何使用手机剪辑软件（以剪映为例），从文案配音到添加字幕、动画、图片和背景音乐，完整地制作一个动态文字视频。整个过程将分为清晰的步骤，确保初学者也能轻松跟上。

## 概述：从文案到成片的流程
制作一个动画视频主要包含以下几个核心环节：准备文案与配音、导入素材与对齐、添加字幕与调整、设置文字动画、插入匹配图片、最后添加背景音乐并导出。我们将一步步拆解。

---

## 第一步：文案准备与配音 🎤

首先，需要准备好视频的文案。文案通常以无标点符号的短句形式呈现，方便后续断句和配音。

**操作流程如下：**
1.  将文案按语义断句，每行为一个短句。
2.  打开配音软件（例如“魔音工坊”普通版），将处理好的文案粘贴进去。
3.  生成配音后，点击下载。通常选择**第一个“带绿幕”**的选项进行下载。

> **核心概念**：绿幕（Chroma Key）是一种后期技术，通过去除特定颜色（如绿色）背景，方便替换为其他画面。这里我们用它来分离音频。

---

## 第二步：导入素材与基础对齐 ✂️

上一节我们准备好了带绿幕的配音视频，本节我们将其导入剪映并进行基础处理。

**操作流程如下：**
1.  打开剪映，点击“开始创作”，导入刚才下载的绿幕配音视频。
2.  选中视频轨道，点击“抠像”，选择“色度抠图”去除绿幕。
3.  点击“分离音频”，将音频分离出来，然后删除绿幕视频轨道。
4.  点击“添加” -> “素材库”，添加一个**黑色背景**素材。
5.  将黑色背景的长度拉至与音频轨道完全对齐。
6.  点击背景轨道，在编辑菜单中选择“防抖”功能。
7.  点击“比例”，选择 **4:3** 作为视频画面比例。

接着调整音频。选中音频轨道，将音量调整到合适大小，例如 **250**。

---

## 第三步：添加与校准字幕 📝

音频和背景对齐后，接下来需要为视频添加字幕。

**以下是添加字幕的步骤：**
1.  返回主菜单，点击“文本” -> “识别字幕” -> “开始匹配”。
2.  等待几十秒，系统会自动生成字幕。
3.  检查字幕与音频是否同步。将时间轴拉到最左端开始播放，逐句聆听。
4.  如果发现字幕与语音不匹配，双击错误字幕进行修改。字幕间用**空格**分隔，无需标点符号。
5.  拖动字幕条，确保其时间范围与对应的语音片段完全对齐。

校准后，统一调整字幕样式。选中任意字幕，点击“编辑”。
*   **字体**：选择你喜欢的字体，例如“凌慧体”。
*   **字号**：设置为 **10** 号。
*   **位置**：将字幕拖动到画面中间（参考蓝色十字光标）。

---

## 第四步：设置文字动画 🎞️

字幕样式调整好后，为了让视频更生动，我们需要为文字添加入场和出场动画。

首先，根据视频总时长规划动画。例如，一个39秒的视频，可以大致分为前、中、后三个13秒的部分，每个部分使用一种主要的入场动画。

**以下是设置动画的步骤：**
1.  **入场动画**：选中一个文字片段，点击“动画” -> “入场动画”。为视频前、中、后三段分别选择不同的入场效果，例如“打字机Ⅰ”、“向右滑动”、“向左擦出”。动画时长默认 **0.5秒**。
2.  **出场动画**：同样选中文字片段，点击“动画” -> “出场动画”。为每一句字幕单独设置一个简洁的出场效果（避免过于花哨），例如“溶解”。对于短句，出场动画时长可设为 **0.3秒**。
3.  **批量操作**：对于多句连续使用同一种入场动画的文字，可以使用“批量编辑”功能提高效率。

> **核心提示**：手机端操作需耐心，熟能生巧。为保持统一，画面上的水印文字动画应与上方主字幕的入场动画一致。

---

## 第五步：插入与匹配图片 🖼️

上一节我们为文字添加了动画，本节我们来看看如何插入图片来丰富画面。

为了让图片与每一句文案精确对应，我们需要先对黑色背景轨道进行分割。

**操作流程如下：**
1.  选中黑色背景轨道，在时间轴上放大。
2.  在每一句字幕开始和结束的位置，对背景轨道进行“分割”。这样，每一段背景就对应了一句字幕。
3.  打开浏览器，访问免版权图片网站，搜索与每句文案关键词（如“领悟”、“观众”）相关的图片并下载。
4.  返回剪映，点击“画中画” -> “新增画中画”，导入下载好的图片。
5.  将图片轨道拖动到对应句子的背景片段之上，系统会自动吸附对齐。
6.  调整图片大小，使其与文字宽度相近，并放置在文字上方，保持间隔一致。
7.  为图片设置动画：选中图片轨道，添加“入场动画”（如“向下甩入”）和“出场动画”，时长均为 **0.5秒**。

---

## 第六步：强调色与背景音乐 🎵

所有元素添加完毕后，可以进行最后的润色：为关键词标色和添加背景音乐。

**以下是最终调整的步骤：**
1.  **强调关键词**：双击字幕，将光标移动到要标色的词语前，拖动选中该词。点击“编辑样式”，选择一个强调色（如棕色）。图片不一定需要与标色词严格对应，但与整句话相关即可。
2.  **添加背景音乐（BGM）**：点击“音频” -> “音乐”，从“抖音收藏”或其他渠道选择一首合适的音乐。
3.  将音乐轨道拖入，对齐视频开头。删除音乐前奏的空白部分，使其从视频开始时响起。
4.  调整音乐音量，使其不遮盖人声（例如调至 **30** 左右）。将音乐轨道尾部与视频末尾对齐并分割删除多余部分。

---

## 第七步：设置封面与导出 📤

视频制作完成，最后一步是设置封面并导出成品。

**操作流程如下：**
1.  点击“比例”，暂时切换为 **3:4**（更适合做封面的比例）。
2.  点击“设置封面”，从视频帧中选择一帧作为封面，保存。
3.  将比例改回 **4:3**。此时封面可能会影响第一句字幕的样式，只需单独调整第一句字幕的字体大小（如改回10号）即可。
4.  点击右上角“导出”，分辨率选择 **1080P**，帧率选择 **60帧**，然后点击导出。

---

## 总结与回顾 🎉

本节课中，我们一起学习了在手机端制作一个动态文字视频的完整流程。我们从**文案配音**开始，经历了**导入对齐**、**添加校准字幕**、**设置文字动画**、**插入匹配图片**、**标注强调色和添加BGM**，最后完成**封面设置与导出**。

**核心操作公式可归纳为：**
`准备素材 -> 对齐基础轨道 -> 添加字幕 -> 赋予动画 -> 插入图片 -> 润色调音 -> 导出成品`

只要按照这些步骤耐心练习，你就能熟练运用手机剪辑软件，制作出吸引人的动画短视频。关键在于细节的校准和反复的预览调整。
