# 老白《搞钱课》：9、手机端动画剪辑课 🎬

## 概述
在本节课中，我们将学习如何在手机端使用剪辑软件进行动画剪辑。我们将从文案准备、音频录制、字幕添加、动画效果设置、图片插入以及背景音乐添加等方面进行详细讲解。

## 文案准备
首先，我们需要准备好文案。确保文案格式清晰，没有标点符号。

```markdown
**文案格式示例：**
- 最近看到的醍醐灌顶的一段话，你其实没有那么多观众活着的时候没有多少人关注你。
- 一旦离世，也没有几个人会记得你，不妨大胆一些，勇敢的人最先享受世界的美好。
```

## 音频录制
使用配音软件录制音频，选择魔红声普通版。

```markdown
**操作步骤：**
1. 选择配音软件。
2. 按照文案格式录制音频。
3. 点击下载，选择第一个下载。
```

## 字幕添加
将音频导入剪辑软件，添加字幕。

```markdown
**操作步骤：**
1. 点击绿布，选择分离音频。
2. 删除绿布，添加黑布。
3. 调整画面比例，设置为4比3。
4. 调整音频音量。
5. 点击文本，识别字幕。
6. 调整字幕与音频同步。
```

## 字体设置
调整字体样式、字号和颜色。

```markdown
**操作步骤：**
1. 点击文本，编辑字体。
2. 选择字体样式、字号和颜色。
3. 调整字体位置和行间距。
```

## 水印添加
添加个人水印。

```markdown
**操作步骤：**
1. 点击新建文本，输入水印内容。
2. 调整水印样式、字号和颜色。
3. 将水印与文案对齐。
```

## 动画效果设置
为文字和图片添加动画效果。

```markdown
**操作步骤：**
1. 选择文字或图片。
2. 点击动画，选择入场或出场动画。
3. 调整动画时长。
```

## 图片插入
插入图片并与文案对应。

```markdown
**操作步骤：**
1. 打开浏览器，搜索图片。
2. 下载图片并导入剪辑软件。
3. 将图片与文案对齐。
4. 为图片添加动画效果。
```

## 背景音乐添加
添加背景音乐。

```markdown
**操作步骤：**
1. 点击音频，选择背景音乐。
2. 调整音乐音量。
3. 删除多余的音频部分。
```

## 导出视频
导出视频并设置封面。

```markdown
**操作步骤：**
1. 点击导出，选择分辨率和帧率。
2. 设置封面比例和字体大小。
3. 导出视频。
```

## 总结
本节课中，我们学习了如何在手机端使用剪辑软件进行动画剪辑。通过文案准备、音频录制、字幕添加、动画效果设置、图片插入以及背景音乐添加等步骤，我们可以制作出精美的动画视频。希望同学们能够熟练掌握这些技巧，创作出更多优秀的作品。
