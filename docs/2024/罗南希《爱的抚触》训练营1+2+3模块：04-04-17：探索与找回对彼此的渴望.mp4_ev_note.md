## 🌟 爱的抚触第十七课：探索与找回对彼此的渴望 🌟

### 概述
在本节课中，我们将学习如何通过觉知和行动来探索和找回对彼此的渴望。

### 一、知行合一，觉知可望

**核心概念**：
- **知行合一**：在爱抚中感受到什么，就表现什么，不排斥自己的感受。
- **觉知可望**：在爱抚中，感受自己的渴望，并积极期待。

### 二、知行不合一的表现

以下是知行不合一在爱抚场景中的几种表现：

1. 当对方询问想要怎样的爱抚时，明明身体渴望，但第一反应是“没有怎样啊，都可以”。
2. 女性在爱抚过程中有其他想法，但害怕被对方视为荡妇或索取无度，因此表现得无知和纯真。
3. 在爱抚过程中有其他想法，但担心伤害对方的自尊，所以假装没事发生。
4. 在爱抚中，既不排斥也不接受，总是兴趣平平，无法安定下来。
5. 在爱抚时，身体紧绷或昏睡过去。

### 三、渴望的重要性

**核心概念**：
- 渴望是对身体生命的允许，对情绪感受的允许，对鲜活体验的允许。

### 四、练习：探索与找回对彼此的渴望

**步骤**：

1. 找一个安静不被打扰的时间和地点，准备好计时器、笔和纸。
2. 进行1到3分钟的腹式深呼吸。
3. 感受此刻的心情状态和周遭环境。
4. 挑选可以独立分辨的感觉，用声音描述出来。
5. 写下此时此刻，你想要怎样的爱抚。
6. 分析写下的渴望，思考哪些想法吓到了自己，哪些很有感觉。
7. 分享你的感受，并尝试与伴侣一起实践。

### 五、练习：建立脑内支持系统

**步骤**：

1. 回忆第十六课中学习的如何分辨脑内小声音和真实发生的事情。
2. 在纸上写下你渴望实现的爱抚场景中，有哪些阻碍。
3. 区分脑内声音和真实的东西，帮助理清头脑是如何欺骗你的。
4. 想象一下，一旦你渴望的爱抚实现了，你或你的伴侣关系会是什么样子的。
5. 将这个意识深深植入脑海，脑波向宇宙发出订单。

### 六、练习：深入探索彼此的渴望

**步骤**：

1. 与伴侣一起选择一个安静不被打扰的时间和地点，准备好计时器、笔和纸。
2. 一起做三次深呼吸。
3. 感受自己的身体、空间、心情和对方。
4. 写下此时此刻，你想要怎样的爱抚。
5. 交换笔和纸，进行面对面的分享。
6. 关注自己的身体感受，并明晰地告知对方你喜欢怎样被抚摸。

### 总结
本节课中，我们学习了如何通过觉知和行动来探索和找回对彼此的渴望。通过练习，我们可以更好地了解自己的需求，并与伴侣建立更亲密的关系。希望同学们能够将所学知识应用到实际生活中，让爱抚变得更加美好。🌹
