在本节课中，我们将深入探讨焦虑型依恋模式的底层逻辑。我们将分析其核心成因，并详细拆解焦虑型依恋在亲密关系中的六大典型特征，帮助你理解自身或伴侣行为背后的心理动因。

上一节我们介绍了母婴关系对依恋模式的深远影响。本节中，我们来看看焦虑型依恋的具体表现与形成原因。

许多焦虑型依恋者常有一个疑问：为什么我总是没有安全感？现在让我们思考一个问题：安全感是什么？它是一种稳定踏实的感觉，还是长久的陪伴？其实，两者都不完全是。安全感的内核根源是**自信**，分为对外和对内两个维度。对外自信是相信这个世界和其他人都是值得被爱的；对内自信则是相信自己是值得被爱的。

焦虑型依恋的问题根源，其实是对内的自信出现了问题。思维里存在着创伤性的**负向核心信念**，继而演变成“觉得自己不值得被别人爱”。这个信念悄悄地在你的潜意识中影响着你的所有社交和亲密关系行为，例如经常要求对方用很多行动来证明爱意。

## 一、案例剖析：依恋模式如何影响关系

前段时间，一位学员私信说，他好不容易遇到一个想发展长期关系的人，却不小心把对方逼走了。在沟通中我发现，因依恋模式破裂的关系，最需要的不是挽回行为，而是要转变焦虑的依恋模式。

我了解他小时候的经历：他是留守儿童，父母每年只回来一次。每年过完年父母离开时，他都特别难受，觉得自己再次被抛弃。虽然现在长大了，但只要伴侣不在身边，他也会特别失落，甚至会对伴侣提出过分要求（如晚上8点前必须回家、周末约朋友需提前报备）。一旦对方未做到，他就发脾气。最终，对方因受不了高强度管制而分手搬离。

上节课我们说过，父母的教养方式会影响孩子的依恋模式。成年后在亲密关系中的依恋模式，与你小时候和父母的依恋模式是一样的。从这个案例中我们也能感受到，除了婴儿时期，父母后来的教养方式也会深刻影响孩子的依恋模式，造成孩子的自尊体系不完整，无法独立解决问题，甚至影响与他人的关系。

该学员从小与父母分隔两地，成长环境不稳定。父母每次回来都给予浓烈的爱和礼物，但又在短暂相处后离开。这让幼小的他直接感受到：那种浓烈的爱是**焦虑**的，而且这种焦虑最后总会变成现实（即分离），给他造成心理创伤，产生失落、愤怒等应激情绪。因此，他用苛刻条件拴住身边的人，是因为潜意识需要确认伴侣是否会像父母一样突然离开，能否对他不离不弃。

## 二、原生家庭教养方式与依恋类型

原生家庭早期的教养方式，会怎样影响我们的依恋模式呢？我们可以从父母对孩子的“要求性”和“反应性”两个维度来看，共有四种教养类型。

以下是四种教养类型的简要介绍：

*   **专制型**：父母要求高，反应低。采用专权态度和强制措施。孩子易形成畏惧、逃避的情绪特点，甚至引发过激行为。长大后可能非常不自信，难以进行亲密表达，容易形成**焦虑型或恐惧型依恋**。也可能无意识效仿父母的专制对待他人。
*   **忽视型**：父母要求和反应双低。对孩子既不管教也不关心，任其野蛮生长。案例中的学员便是典型。长期得不到积极回应，孩子可能走向两种极端：一是变得独立，演变成**回避型**；二是因没有存在感，害怕被忽视，演变成**焦虑型**。
*   **溺爱型**：父母反应高，要求低。对孩子极度关怀但缺乏原则，放任其行为。这样的孩子虽不缺爱，但容易迷茫，缺乏规则概念，可能无意识做出越界行为。在亲密关系中可能比较敏感，渴求伴侣像父母一样溺爱自己，也易演变成**焦虑型依恋**。
*   **权威型**：父母要求高，反应也高（双高）。对孩子有正面管教，并能积极回应其需求。研究表明，这是最值得推荐的教养方式。在此环境下成长的孩子，成年后状态更健康稳定，易于形成强大内核，构建**安全型依恋**。

以上四种方式具有普遍性，但也存在小概率偏差（如专制型家庭也可能出自信的孩子）。但不可否认，父母是孩子的启蒙老师，你从父母身上习得的能力，至今仍在无形中影响着你的处事风格。

## 三、焦虑型依恋的六大特征

为什么除了权威型，其他三种教养方式都可能造成焦虑型等不安全依恋？答案是：父母没有灌输足够的信任感给你。在你的潜意识中，“父母的不信任感”就等于“我不值得被父母爱”。这种**负向核心信念**伴随你成长，并在所有关系中如影随形。

受此影响，焦虑型依恋者在恋爱时通常有以下六大特征。请对比自己或伴侣是否有以下情况，以便针对性调整。

以下是焦虑型依恋的六个典型表现：

1.  **自我价值感低，不相信自己值得被爱**：焦虑型的自我很脆弱，觉得自己无法独立生活，必须依赖某人。在他们的认知里，关系中的爱像养分，只有源源不断供给，他们才能成长。其根源在于**自爱能力几乎为0**，内在的爱占比极低，极度需要外部爱来填补情感匮乏，像寄生虫一样拼命索爱。
    *   公式：`焦虑型的爱需求 ≈ 外部他人的爱 (占比极高)`
2.  **内心易产生情绪海啸，被情绪淹没**：经常感受到负面情绪过载，无法独立面对和安抚情绪，全靠身边人拯救。这种情绪海啸给对方带来巨大压力。焦虑型容易将中性事件曲解为威胁事件（例如：恋人不接电话 -> 他不想理我 -> 他抛弃我 -> 他不爱我了），从而引发夺命连环Call等行为。
3.  **害怕被拒绝，也不擅长拒绝别人**：潜意识里认为“被拒绝 = 不被爱 = 被抛弃”。拒绝会引发挫败和愤怒，且难以分清对方拒绝的是事情还是你这个人。例如，送女友口红被嫌色号不合适，内心便上演“她拒绝我的礼物 = 拒绝我的爱 = 不爱我了”的幻想剧。
4.  **渴望融合，导致心态错位与越界**：核心情绪是对被抛弃的恐惧，因此会尽可能追求亲密来对抗。但这种渴望融合的状态，常过度侵犯对方心理边界，不懂尊重他人意愿。例如，无论对方多忙，都要求随时报备行踪。
5.  **内心敏感多疑，不懂正确表达需求**：因从小被忽冷忽热对待，直截了当表达需求成了一种奢望。害怕提出需求会被拒绝，导致价值感更低。因此更愿意用“投射性引导”让对方主动满足自己。例如，嘴上让对方选电影，若对方真选了你不爱看的，便会觉得对方不考虑你的感受。
6.  **有极强掌控欲，容易患得患失**：心中有一套严格的恋爱标准，过度关注另一半是否做到，并对此非常敏感。例如，经常送小礼物希望对方开心，但若对方没有主动回礼或表达感谢，便会特别生气，觉得付出被忽视，让焦虑、害怕等负面情绪充斥关系。
