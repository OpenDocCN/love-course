# 感觉流6.0：核心自信-天马情感恋爱课堂：004.第四课：聊天无限话题的技巧（附案例）

## 概述

在本节课中，我们将学习如何通过保存和积累过往有趣的事物来创造无限的话题，从而在聊天中更加自信和有趣。

## 保存和积累

**保存**：将生活中有趣的事物记录下来，例如拍照、写日记等。

**积累**：将保存的内容整理成话题库，以便在聊天中随时使用。

## 无限话题库的应用

1. **过往话题当作现在发生**：将过去有趣的事情当作现在发生的事情来聊，例如展示照片时，不要说明是去年的照片，而是说“我现在跟朋友在划船，给你看看”。

2. **展示价值**：通过分享自己的经历和知识，展示自己的价值。

3. **寻找共同点**：在聊天中寻找与对方的共同兴趣，例如书籍、电影、游戏等。

## 案例分析

以下是一个案例，展示了如何运用话题库和聊天技巧：

1. **展示价值**：分享自己的社交圈子，例如“我几个朋友也是做设计，一个在贪玩蓝月，一个在虎牙”。

2. **寻找共同点**：询问对方是否喜欢玩游戏，并展示自己对这个行业的了解。

3. **主动沟通**：在对方冷淡回应时，主动沟通，例如“那肯定是因为你不重要，不然怎么会难约”。

4. **提供情绪价值**：在对方服软时，提供好的话题，例如“被你这么一说，发现我才是个王者”。

5. **模糊邀约**：在约会时，使用模糊邀约，例如“那到广州可以约个饭”。

## 总结

本节课中，我们学习了如何通过保存和积累过往有趣的事物来创造无限的话题，并在聊天中展示自己的价值。通过运用话题库和聊天技巧，我们可以更加自信和有趣地与他人交流。
