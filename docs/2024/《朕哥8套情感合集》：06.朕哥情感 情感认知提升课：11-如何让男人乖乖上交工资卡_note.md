# 《朕哥8套情感合集》：06.朕哥情感 情感认知提升课：11-如何让男人乖乖上交工资卡

🎼 如何让男人乖乖上交工资卡？这个问题需要分情况考虑。因为有的人上交工资卡时处于高位，有的人则处于低位，高位和低位的情况不同，明白吗？

**核心策略**：
- 分析问题：让男人感觉到工资卡放在你这里对双方更有利，并非为了占便宜，而是为了家庭的最大收益。
- 陈述原因：明确告诉男人上交工资卡的原因，让他明白这是出于家庭整体考虑。
- 无形压力：通过冷淡的态度给予男人无形压力，让他意识到不交工资卡的不妥。
- 巧施恩惠：用他的钱为他买礼物，让他感受到你的付出和关心。

**高位策略**：
1. 分析问题：指出家庭开支过大，缺乏理财规划等问题。
2. 陈述原因：解释上交工资卡的原因，让他明白这是为了家庭整体利益。
3. 无形压力：通过冷淡的态度给予男人无形压力。
4. 巧施恩惠：用他的钱为他买礼物，让他感受到你的付出和关心。

**低位策略**：
1. 引导话题：将话题引导到工资卡上，让他意识到家庭财务问题。
2. 展现能力：展示你的理财能力和为家庭考虑的态度。
3. 植入想法：通过提问引导他思考上交工资卡的问题。
4. 欲擒故纵：在男人提出上交工资卡时，不要直接接受，而是以其他理由拒绝，再巧妙地为他花钱。

**总结**：
本节课中我们一起学习了如何让男人乖乖上交工资卡的方法。无论是高位还是低位，都需要根据具体情况采取不同的策略。通过分析问题、陈述原因、给予无形压力和巧施恩惠，可以有效地让男人上交工资卡。
