# 43、彭祖强肾功：041.11、12经络通拍排毒法！

## 概述
在本节课中，我们将学习彭祖强肾功中的拍打排毒法，通过拍打身体特定穴位和经络来促进气血流通，达到排毒养生的目的。

## 拍打工具与部位
以下是拍打时使用的工具和需要拍打的部位：

**工具**：
- 手
- 拍打棒（可选）

**部位**：
- 关元穴
- 会阴穴
- 百汇穴
- 腋窝鸡全穴
- 肩颈穴
- 手背、前胸、后背
- 腿部（前外侧、内侧、后侧）
- 膀胱经、肾经
- 钣金（腹股沟）
- 中线（胸部、两肋、后背）
- 腰部

## 拍打方法
以下是拍打的具体方法：

1. **关元穴**：用空心拳拍打，左右手各十次。
2. **会阴穴**：一手护着睾丸，一手空心拳拍打，左右手换手。
3. **百汇穴**：双手握拳敲击20下。
4. **腋窝鸡全穴**：用空心拳拍打，左右手换手。
5. **肩颈穴**：用空心拳拍打。
6. **手背、前胸、后背**：用手背拍打，重点拍打关节点。
7. **腿部**：从前外侧开始，拍打至脚踝，再从内侧拍打至腹沟。
8. **膀胱经、肾经**：从后侧膀胱经开始，拍打至比侧，再从后内侧肾经拍打至服务后。
9. **钣金**：在腹股沟处拍打。
10. **中线**：用甩打法拍打胸部、两肋和后背。
11. **腰部**：用空心拳拍打。

## 注意事项
1. 拍打时力度要适中，避免用力过猛。
2. 拍打时注意呼吸，保持呼吸均匀。
3. 拍打后不要立即洗澡，避免受凉。
4. 拍打时注意保护自己，避免拍打脆弱部位。
