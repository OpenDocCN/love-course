# 雨宸19套合集：19.婚恋情感IP，运营变现课程：010.第十节 如何做咨询指导4-常用的提升方法和二次付费

## 概述
在本节课中，我们将学习如何通过提升咨询指导的方法和技巧，以及如何引导客户进行二次付费。

## 常用的提升方法

### 期望管理
**公式**：满意度 = 事情的结果 - 期望值

**步骤**：
1. 了解自己的价值及对方的能力。
2. 有意识地设置和管理期望。
3. 选择合适的对象。

### 提升情绪察觉能力
**步骤**：
1. 了解自己的价值及对方的能力。
2. 通过自我察觉和他人评价进行客观评价。

### 共情技巧
1. 对非语言行为的反馈。
2. 对深层情感的反馈。
3. 指出冲突性的感觉和想法。
4. 运用类比。
5. 有目标性的自我帮助。
6. 运用媒体反馈。

### 指导客户聊天
1. 聊天节奏：遵循对等原则，保持主动性和内容对等。
2. 打开话题：上堆、下切、平移。

## 二次付费

### 客户二次付费意愿
**公式**：二次付费意愿 = 用心程度 + 与客户拉近距离

### 引导客户进行二次付费
1. 描述接下来的指导计划。
2. 发朋友圈宣传优惠活动和新服务模式。
3. 吸引老客户了解二次付费。

### 注意事项
1. 做人要有底线，避免过度推销。
