# 子阳《脱单实战聊天课》：2.怎么聊：第七节课：女生反应的两个原则

## 概述
在本节课中，我们将学习如何根据女生的反应来调整自己的聊天策略，特别是关注两个重要的原则。

## 原则一：坏反应的度
**公式**：坏反应度 = 女生坏反应度

1. **理解坏反应度**：如果你不确定女生的坏反应程度，可以参考以下方法判断。
2. **回应策略**：根据女生给你的坏反应程度，给予相同程度的坏反应。
3. **判断依据**：使用尺子作为参考，将女生的反应与尺子上的刻度对应。
4. **调整策略**：如果个人吸引力和价值远大于对方，可以适当提高回应的坏反应度。

## 原则二：线上与线下坏反应的度
**公式**：线上坏反应度 < 线下坏反应度

1. **线上聊天**：在微信等线上聊天时，给予较轻的坏反应。
2. **原因**：女生接到坏反应后，可能会直接退出聊天，因为转换聊天对象成本较低。
3. **线下约会**：在约会中，可以给予较重的坏反应，因为对方无法轻易离开。
4. **初期聊天**：在刚认识时，给予较轻的坏反应，因为缺乏情感基础。

## 总结
本节课中，我们学习了如何根据女生的反应来调整自己的聊天策略，特别是关注坏反应的度。通过掌握这两个原则，我们可以更好地应对女生的反应，提高聊天效果。
