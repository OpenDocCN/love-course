# 社交情商训练营：01_1-1、低情商的雷区 😬

在本节课中，我们将要学习什么是低情商，并详细剖析几种常见的低情商表现。了解这些“雷区”是提升社交能力的第一步。

---

情商不高的人，去到哪里都惹人讨厌。情商高的人，去到哪里都很受人欢迎。那么，什么是情商低呢？以下是几种常见的现象。

上一节我们概述了低情商的影响，本节中我们来看看具体表现。低情商行为通常体现在以下几个方面。

**1. 不尊重他人感受**
这类人自认为是心直口快、没有心机。他们喜欢有话直说，不考虑说话后的结果，也不在乎他人是否开心。其行为本质是缺乏礼貌和教养，不懂得换位思考。核心问题可以概括为：**行为 = 只顾自我表达，忽略他人反应**。

**2. 读不懂潜台词**
别人已经给出明显暗示，但低情商者无法理解。他们不深入思考对方话语的潜在含义，只理解表面意思。这经常导致他们做出令人反感的事情。关键缺陷在于：**理解层次 = 字面意思**。

**3. 过分情绪化**
他们的高兴、开心、生气全部表现在脸上。有时，别人开个小玩笑，他们也会当真，甚至当众发脾气。他们习惯把情绪放在第一位，不能理性分析问题，也不能冷静处理问题。这是非常不成熟的表现。其行为模式是：**反应 = 情绪 > 理性**。

**4. 说话不分场合**
在不同场合，说话方式应与平时不同。例如，在婚礼上不能开新郎新娘的玩笑；在很多同事面前，不能开领导的玩笑。在不同的场合，我们需要根据具体情况斟酌哪些话能说，哪些话不能说。核心原则是：**表达内容 ∝ 场合适宜性**。

**5. 其他常见表现**
除了以上四点，还有一些常见行为也是低情商的表现。例如：
*   爱表现，行为浮夸。
*   喜欢埋怨，怨天尤人。
*   做事古板，不懂灵活变通。
*   喜欢指手画脚，好为人师。

你可以审视一下自己是否有以上五种表现。如果有，就需要认真学习本节课的内容。

---

本节课中，我们一起学习了低情商的五大常见“雷区”：不尊重他人感受、读不懂潜台词、过分情绪化、说话不分场合以及其他如爱表现、爱埋怨等行为。识别这些问题是改善社交能力、提升情商的重要基础。
