# 课程二：向上社交系列课 003.1.2 学会这五招你也可以成为情绪管理达人 🧘‍♀️

在本节课中，我们将要学习情绪管理的核心方法与实用技巧。情绪管理是向上社交和高情商修炼的关键环节，掌握它，你就能更好地驾驭人际关系，避免因情绪失控而错失良机。

上节课我们介绍了高情商的核心与向上社交的基础。本节中，我们来看看如何成为情绪管理达人。

都说性格决定命运。搞不定情绪的人也搞不定自己的人生。你的情绪稳定吗？你真的了解情绪吗？不了解就无法掌控，不理解就不能驾驭。这里有一个自测可以给你答案。

以下是情绪自测的十个要点，如果你基本认同，说明你对情绪的认识并不客观，有必要认真学习本节课。

1.  情绪不好时，会忍不住对亲近的人发脾气。
2.  经常后悔自己说过的话或做过的事。
3.  遇到压力就想逃避或崩溃。
4.  别人的一句话能影响你一整天的心情。
5.  总是担心未发生的事，感到焦虑。
6.  开心和悲伤的情绪都来得很快、很强烈。
7.  不擅长表达自己的感受，常常生闷气。
8.  在冲突中，总是先情绪上头，而非解决问题。
9.  事后常常觉得自己反应过度。
10. 认为“做自己”就是想怎样就怎样。

我们倡导真实地做自己，但做自己不代表无所顾忌、肆意妄为。成年人的世界有规则，毕竟不是所有人都会宠着你、让着你。情绪管理非常有必要。你有多少能量约束自己，就能收获多少善意。

心理学有个著名理论叫做“踢猫效应”。它是指对弱于自己或等级低于自己的对象发泄不满情绪而产生的连锁反应。“踢猫效应”是一种典型的负面情绪传染。当我们把不满情绪传递给身边的人，就会产生一系列连锁反应，到头来伤人伤己。你想再去获得别人对你的好印象，就非常困难了。

我以前参加电视台比赛时，认识一个学播音的女孩，身材长相都很好。她男朋友是个富二代，对她很好。他们交往小半年后，女孩怀孕了。富二代积极推动结婚，带她回家见父母。男方母亲态度比较一般。女孩从男方家里出来后就开始闹情绪、吵架，并索要高额彩礼。她情绪失控，说了很多难听的话，甚至以打掉孩子相威胁。男方后来选择分手，给了分手费。复盘来看，如果女孩能稍微控制情绪，“带球上位”的可能性很大。但正因为年纪小、情绪不稳，导致了失败。

这个案例让我明白，要去修炼自己的情绪管理。你要记住一句话：**我发完这个火之后能给我带来什么样的结果？** 在情绪不稳定、想要发火时，就要记住这句话。记住，不要让你的情绪先行，而是要让你的目的先行。当你手上没有足够的“牌”时，是没有发火资格的。

那么我们如何修炼情绪管理呢？下面给大家几招能够快速帮你重建情绪框架的方法，直接可以拿来用。

你有没有遇到以下情况？遇到说话不友好的人，让你瞬间下头，然后控制不住地说出一些不理智的话，事后后悔。画了美美的妆去见男朋友或闺蜜，却因为等了对方很久，妆都晒花了，心情变差，忍不住对对方发脾气。遇到紧急且繁琐的工作，顿时失去方寸，效率低下，心情烦躁。遇到挫折和难题时，感觉喘不过气，谁跟你说话都不爱搭理，一碰就炸毛。

这样突如其来的情绪失控，举不胜举。这就是我们要说的第一部分：遇到事情容易情绪化，要怎么高情商地处理。其中蕴含了三招情绪管理和修炼的技巧。

**第一招：改变心态，自我疗愈。**

关于情绪失控，下面说一个生活中最常见的场景来拆解。你和对方已经发生冲突，这时如果你再继续，一定会口不择言，伤人伤己。那应该怎么办呢？

以下是三点解决方案。

1.  **当下的逃离不是坏事。** 如果你正处于正面冲突中，感觉马上就要发飙，记得给自己一个指示或暗示。比如掐一下自己，或摸摸耳朵，给自己约定一个“暂停键”暗号。让自己从紧张气氛中暂时抽离，深呼吸，快速转移话题或离开现场，冷静下来。
2.  **凡事都要善后。** 如果正面冲突发生，你仍怒气未消，可以通过看电影、听音乐、散步等自己喜欢的方式使紧张情绪松弛下来。想办法通过合适方式将情绪化解掉，不要让这种情绪长时间占据内心。你可以生气，但不能“越想越气”。冲突发生后，要及时善后。当矛盾已发生，若还想留住对方，就要把情绪放一放，以解决问题为导向。和对方冷静沟通，可以说：“刚刚是我激动了，抱歉，我情绪不太好。我的出发点是想解决问题，我们可以好好聊一下怎么解决。” 你要为你做过的事情善后。
3.  **定期处理负面能量和磁场。** 每个人都会有情绪不好的时候，这很正常。但不能让它持续蔓延，影响你的判断。定期给自己做情绪清理很重要。美国心理学家艾利斯的“情绪ABC理论”这时非常有效。该理论认为，人的情绪困扰（C）不是由诱发事件（A）直接引起的，而是由个体对事件的信念和看法（B）引起的。**公式：A (事件) → B (信念) → C (情绪/结果)**。例如，看到伴侣就想发脾气（C）。诱因（A）可能是他最近不关心你。你的信念（B）可能是“他有二心了”或“他不爱我了”。当你知道后果可能是大吵一架，这个情绪还有必要发出去吗？其实可以有更好的解决方案：暂时转移视线，稳定情绪后再心平气和地沟通，以解决问题为导向。

情绪管理的核心不是没有情绪，而是情绪稳定。

说完了第一招，下面我们再回到自身。我们常常会因为情绪而产生内耗，怎么完成自我修炼，不沦为情绪的奴隶呢？

**第二招：拒绝情绪内耗，学会用“反派思维”生活。**

我们常常看电视剧，剧中的女二、女三多是反派角色。他们的思维行为，我们常常气得跳脚，但冷静下来想想，会发现这些反派人设都有一个共同点：非常自私，行为的出发点以自己的感受为导向。今天就跟大家说，最好拒绝情绪内耗的方式就是拥有“反派思维”。当然不是让你学习反派的目中无人和自私，而是他们思考事情的方式值得借鉴。

前段时间很火的韩剧《黑暗荣耀》里的女二朴妍珍，网友总结她的座右铭是：“与其精神内耗自己，不如发疯指责别人。”虽然极端，但她的心态确实是很多女生都紧缺的。反派的心态就是在任何困难和刁难面前都可以自信地做自己。朴妍珍在剧中遇到女主报复时说：“每个人都会犯错，但是解决问题的办法不在后面，它永远在前面。就算有人死了，那也是昨天的事情，今天我们就要做好今天的事情。”反派之所以永远清醒，就是因为他们永远以目标为导向，目标明确，不达目的誓不罢休。

很多女孩遇到问题就会想：“是不是我不够好？”“是不是我做错了什么？”情绪内耗的本质，是希望终结一件未完成的事情，因为没有完成而产生的一系列负面情绪。反派为什么不会精神内耗？就是他们不会让事情过夜，有问题解决就好。

以下是几句“反派思维”的心理暗示语句，在你遭遇道德绑架或伴侣PUA时可以使用：
*   `这件事没毛病，我为什么要卑微？`
*   `我为什么要委屈我自己？`
*   `那是他的主观想法，与我无关。`
*   `我没有做错什么。`
*   `我值得更好的，有什么大不了？`

记住，不要用别人的错来惩罚自己。当你觉得没有信心时，在向上社交或寻找优质对象的路上遇到阻碍时，就要给自己一个信念：**没有什么是我做不到的。** 提升自信力，我必须尽快提升自己，我必须要拿下他。无论是面对工作、向上社交，亦或是拿下优质对象，都需要快、准、狠，心中要有绝对的目标感。

有人说：“我这个性格就是很难改啊。”这就要说到第三招。

**第三招：性格无法速变，但可练习改变沟通方式。**

有人说：“我就是这样的性格，没办法。我就是个急性子，在跟伴侣沟通时，我就是忍不了，总是不假思索就把不好听的话说出来。”这时你就想想我们上节课分享的：高情商说话一定是双赢的局面，有效沟通才有意义。

性格不是一朝一夕可以改变的，但是习惯可以被改变。《刻意练习》中有句话：杰出不是天赋，而是一种人人都能学会的技巧。优秀的情绪管理者、情绪稳定的人不一定是与生俱来的，而是刻意练习的结果。上节课我也强调过很多次，所有的沟通和社交都是为了能够使双方利益最大化，而不是把时间消耗在无聊的掰扯上。

很多人情绪不能自控，还有一个非常重要的原因：太把自己当回事了。宝贝，有的时候你的意见真的没有那么重要。你以为你说了很多，其实真正能被别人记住的，也许一句话都不到。话可以少，但要说到点子上。尤其是在跟优质对象、大佬等高质量人脉接触的过程中，多了解对方感兴趣什么，听别人说什么，关注对方的情绪波动，对症下药才是最有利的。

一个小技巧就是在沟通的过程中，你可以多用“你”、“我们”、“你们”，少用“我”、“我觉得”、“我认为”等主观性词语开头。这样才会让对方感受到你是以他为导向、以他为中心的聊天，自然会把目光多在你身上停留。

情绪管理不是一朝一夕的事情，有些底层逻辑的东西，你必须要学会。我们来聊聊第四招。

**第四招：拉高心智的波峰，少做蠢事。**

什么是心智？心智就是你的认知力、行动力和自控力的结合。很多人的情绪容易产生波动，有很大的情绪起伏，其实最主要的原因就是你的“心智波峰”太低。一点点事情就会大惊小怪，觉得超出承受范围，从而在行为上表现得过激。

人类在面对未知时，很容易出现恐惧、抗拒以及逃避。你的认知水平越高，自信力就越强，情绪也越稳定。当你心智的波峰越高，认知水平一定越广，遇到事情时，你的恐惧和抗拒就会越少，就不容易被影响，也不会轻易做出不理智的事情或说出不理智的话，因为你已经看到事物背后的真相了。

当然，拉伸心智的波峰并非一朝一夕。就像上节课提到的一样，高情商需要日复一日修炼。在这里，同样可以给大家三点建议来提升。

以下是提升心智的三点建议。

1.  **保持目标和感知力，少就是多。** 有时候我们容易不开心、沮丧、愤怒，很大一部分原因是因为贪心。例如找对象，本来想找有钱的，找到了又想要情绪价值；找到了提供情绪价值的，又觉得不够帅。很多情绪内耗就来源于贪心。人不要既要、又要、还要。目标必须明确，要什么就以这个为导向。有时候少就是多，贪心只会让你失去更多。
2.  **大悲大喜之前不要做任何决定，保持情绪的觉知。** 当你的情绪已经到达一个临界点时，不要轻易做出任何决定。比如很多女生在盛怒之下就会说“我要分手”、“我要离婚”。凡是冲动之下的决定，你都会后悔。感知到情绪失控，这时闭嘴是最好的选择。
3.  **适当的“摆烂”很有必要。** 不要让自己的精神一直处于极度紧绷状态，适当给自己放松和调整的时间。情绪也需要假期。比如我每周工作很忙，但周末一定会给自己放空的时间，去做SPA、散步或爬山，接触大自然，什么都不想，给自己心情放个假。

我们说完了情绪失控、情绪内耗。最后一点，我们来聊聊负面情绪。负面情绪不是洪水猛兽，有时利用得好，反而会柳暗花明，成为你的解药。

**第五招：走出非黑即白的世界。**

当你的心智波峰还在成长中时，有人会说：“那我当下的负面情绪就是来了，我该怎么办？”很多人容易愤怒、生气、抑郁、忧愁，很大部分原因就是因为太在乎，觉得这个世界要么是白，要么是黑；要么是对，要么是错；要么是好，要么是坏。这就是非常机械的思维模式。因为世界上多的是灰色地带。

例如择偶，优质对象的选择一定是综合考虑的。你能说他不喜欢年轻漂亮吗？是个男的都喜欢。但作为终身伴侣来说，漂亮根本就不是最重要的，因为他需要的是势均力敌。这就是为什么很多男人找伴侣，未必会选择最喜欢的，但一定会选当下最合适的。我常常和女生们说，用男性思维思考问题，才能拿下优质男。男性思维的一大特点就是：什么都可以衡量，没有绝对。

那么面对负面情绪也一样。如果你觉得愤怒，就可以考虑是不是这件事超出了你的承受范围。这时也是一个警钟，提醒你可以停一下、思考一下。重视情绪背后的真相才是王道。所以遇到负面情绪时，其实可以不用过度放大它。

美国心理学家佩蒂教了一招：把心理的不愉快、烦恼、压力写在纸上，再把纸张扔进垃圾桶，就可以舒缓负面情绪。我前面说情绪过度浮动是因为在乎。其实只要你不在乎，你就不会有所谓。前几天有部很火的动画片《哪吒》中有句台词：“人心中的成见是一座大山，任你再怎么努力，也无法搬动它。”不喜欢你的人就是不喜欢。所以不要企图改变别人。有那个时间还不如用在自己身上。面对偏见跟误解，有时时间是最好的打脸工具。

最后一点，亲爱的们，负面情绪很正常。人生不可能只有快乐，不断地解决问题和烦恼才是我们的常态。就像再相爱的两个人都有磨合期、都会争吵、都会有相看两厌的时候。所以记住，别指望一劳永逸。太顺遂只能说明你的生活没有盼头。一味的发泄情绪没有用，只有解决问题才有意义。

好了，以上就是我们本节课的全部内容。

本节课中，我们一起学习了成为情绪管理达人的五招核心方法：

1.  **改变心态，自我疗愈。** 学会暂停、善后和定期清理情绪。
2.  **拒绝情绪内耗，学会用“反派思维”生活。** 以目标为导向，减少自我消耗。
3.  **性格无法速变，但可练习改变沟通方式。** 通过刻意练习，养成有效沟通的习惯。
4.  **拉高心智的波峰，少做蠢事。** 提升认知，明确目标，保持觉知，适当放松。
5.  **走出非黑即白的世界。** 接纳灰色地带，正确看待和疏导负面情绪。

情绪管理是你向上社交和找到优质对象必须要掌握的技能，大家一定要重视起来。
