# 富山山女性成长课程：03_精准破圈3 🌟

## 概述

在本节课中，我们将学习如何通过精准破圈，结识优秀人士，拓展社交圈。

## 一、结识优秀人士的途径

1. **寻找优秀姐姐**：寻找身边优秀的姐姐，如会做生意的姐姐、在电视台工作的姐姐等。
2. **翻看人际交往圈**：回顾自己的人际交往圈，寻找合适的姐姐。

## 二、与姐姐建立联系

1. **主动发起联系**：逢年过节，发送祝福，表达想重新联系的意愿。
2. **重新热络关系**：询问姐姐最近在做什么，重新建立联系。

## 三、注意事项

1. **识人**：选择那些大方、乐于分享的姐姐。
2. **主动买单**：与姐姐外出时，主动买单，展现懂事和大气。
3. **分享资讯**：分享新鲜资讯，帮助姐姐提升形象。
4. **送礼物**：根据姐姐喜好，送餐具、水晶杯、饰品等礼物。
5. **订花**：为关系好的姐姐订花，表达关心。

## 四、利益交换

1. **提升自身价值**：让自己有价值，才能在社交中立于不败之地。

## 总结

本节课中，我们一起学习了如何通过精准破圈，结识优秀人士，拓展社交圈。希望同学们能够将所学知识应用到实际生活中，拓展自己的人际关系。
