# 18、坏小k女人一生必学的一堂课：37.感情36计（17）如何识别伪装的优质男

🎼 感情三6集第十七大技巧，如何识别伪装的高质量异性具体方法及流程。你大概能从一个人问的问题中反推出他是怎样的一个人。分享一个比较快速、常用到的方法。

### 一级问题

- **定义**：为什么后面加的是单一行为，如为什么他不理我。
- **特征**：频繁提问，表明在多关系中总想成为受益者。
- **表现**：缺乏深度思考，脾气直接率真。
- **风险**：可能不够可靠，容易情绪化，孩子气。

### 二级问题

- **定义**：涉及关系冲突议题，如为什么团建失败。
- **特征**：在关系中寻求平衡。
- **表现**：关注关系问题，寻求解决方案。

### 三级问题

- **定义**：提出质疑，如东方真选为何爆火。
- **特征**：不随波逐流，意识强。
- **表现**：关注事实，寻求真相。

### 测试经济状况

1. **房价看法**：通过询问对房价的态度，判断是否有房产。
2. **时间价值**：观察时间管理能力，间接了解收入水平。
3. **公积金**：通过公积金数额和公司购买比例，估算月收入。

**公式**：月工资 = 公积金 / (公积金比例 / 12)

**示例**：公积金为700元，公司购买比例为12%，则月工资约为5800元。
