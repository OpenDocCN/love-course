# 任亚《人间清醒后半生》：第21节 相处智慧——对方小气抠门，总是舍不得花钱

大家好，我是任亚，欢迎来到我的课堂。这节课我将与大家探讨如何应对另一半小气抠门、不舍得花钱的问题。

## 概述

在本节课中，我们将探讨如何处理与伴侣在金钱观和消费观上的差异，以及如何设定界限和沟通。

## 婚前了解的重要性

**公式**：婚前了解 → 婚后沟通 → 婚后生活品质

其实，这些问题都应该在婚前就了解清楚。婚前不认真选择伴侣，婚后面对问题时想要补救，往往为时已晚。

## 金钱观和消费观

**公式**：金钱观 + 消费观 = 生活品质

金钱观和消费观是两个人在一起生活的一个非常核心的底层东西，很难改变。而这些特质在婚前就很容易识别出来。

## 自我消费

**代码**：自己赚钱 → 自由消费 → 不受对方干涉

如果对方不舍得为我花钱，那就自己给自己花钱。自己赚钱买花带，自己喜欢的东西，对方选择不吭声或管我，我觉得都是OK的。

## 设定界限

**代码**：设立界限 → 明确沟通 → 提出止损条件

如果对方既舍不得花钱，还影响我购买，还不停的不允许我购买，还不停的嘚嘚嘚，我觉得这种关系真的是要重新考虑一下。

## 家庭消费

**代码**：共同消费 → 明确需求 → 沟通协商

涉及到共同消费时，如买房、装修风格、耐用品牌等，需要与对方进行沟通协商。

## 沟通与止损

**代码**：清晰表达 → 提出期待 → 设定止损条件

在沟通时，要清晰表达客观事实、感受、期待和要求，并提出止损条件。

## 婚姻与合伙人

**公式**：婚姻 = 合伙人公司

婚姻可以比作一个合伙人公司，核心是合伙人能否共同为这个公司赚钱，以及如何分钱和花钱。

## 金钱与婚姻

**代码**：金钱 = 婚姻核心

钱是婚姻中非常底层和核心的东西。谈恋爱可以不谈钱，但婚姻必须谈钱。

## 结论

**代码**：自我消费 → 设定界限 → 评估价值

如果对方很抠门，不愿意为我花钱，那我的底线就是我为自己花钱，你不要管我。如果对方非要管，我的耐受力、抗压力、情绪管理能力能否抗大？如果对方不同的嘚嘚嘚，我依然能够保持情绪稳定，我觉得留下他也无妨。

本节课中我们一起学习了如何应对伴侣小气抠门、不舍得花钱的问题。希望对大家有所帮助。
