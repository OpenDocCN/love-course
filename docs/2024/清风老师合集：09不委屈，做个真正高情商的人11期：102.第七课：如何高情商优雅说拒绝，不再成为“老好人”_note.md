# 清风老师合集：09不委屈，做个真正高情商的人11期：102.第七课：如何高情商优雅说拒绝，不再成为“老好人”

## 概述
在本节课中，我们将学习如何用高情商优雅地说拒绝，不再成为“老好人”。

## 为什么我们会害怕拒绝他人？
**公式**： 爱恐惧 → 拒绝恐惧 → 失去爱恐惧

我们害怕拒绝他人，往往是因为我们害怕拒绝了别人，就没有办法得到爱了，没有办法去维系这段关系了。

## 拒绝的负面影响
1. 自我价值感低
2. 害怕社交

## 如何正确拒绝
### 场景一：如何拒绝提出过分要求的领导
1. 表达实际情况
2. 提出可能的方案
3. 传达歉意

### 场景二：如何拒绝总是伤害你的父母
1. 表达自己的委屈
2. 寻求理解
3. 提出谈话主旨
4. 找到平衡点

### 场景三：如何拒绝总是指责自己的伴侣
1. 表达对对方的理解
2. 提出自己的观点
3. 描述发生的事实
4. 寻求对方的支持

### 场景四：如何拒绝黏人的孩子
1. 询问孩子的感受
2. 表达自己的感受
3. 提出方案
4. 阐明原因

## 总结
本节课中，我们学习了如何用高情商优雅地说拒绝，不再成为“老好人”。希望这节课能帮助你更好地处理人际关系，获得情感幸福。
