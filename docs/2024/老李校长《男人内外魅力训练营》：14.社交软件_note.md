# 老李校长《男人内外魅力训练营》：14.社交软件_note

在本节课中，我们将要学习如何高效利用社交软件，打造吸引人的个人形象，并掌握关键的线上互动策略，以提升社交成功率。

情感的道路并非总是坦途。情感世界充满挑战，唯有克服挑战的人，方能成为赢家。勇敢的男人，让我们一同探索社交软件这个“潘多拉魔盒”，洞察人性，最终实现个人魅力的提升。

## 选择适合你的社交软件 🗺️

上一节我们提到了探索社交软件的重要性，本节中我们来看看如何选择适合你的平台。

我身边的情场高手最常使用的社交软件，除了简单高效，最重要的是能发挥他们的优势。这些高手通常是高颜值、有车的男性。具备这些条件，在网上往往能吸引他人主动接近。现在社交软件种类繁多。

以下是选择社交软件的几个步骤：

大家可以去小红书等平台搜索社交软件评测。不同软件适合不同类型的人，这类似于打游戏：有人擅长赛车游戏，有人擅长射击游戏，有人则喜欢节奏缓慢的战略游戏。情场高手们也有各自偏好和擅长的社交软件。

有些人在某个软件上就是感觉不舒服、没感觉。小红书等平台会描述不同社交软件的特点、用户画像。你可以结合自己的喜好进行选择，然后注册试用。如果发现不适合，应立即注销。

因为某些社交软件上骗子和虚假账号很多。这些软件表面上是去认识真人，实际上首先是展示你的照片。我认识的那些情场高手，就特别喜欢向我要高质量的照片。

## 打造核心吸引力：高质量素材 📸

选择好平台只是第一步，接下来我们需要准备展示自己的核心素材。

我给大家讲讲我周围这些情场高手朋友，包括如何弄出好照片。大家不必专门去学拍照。你可以去小红书上搜索本地的摄影师。在去某个城市前三四天，就在小红书上把本地摄影师搜一下。

以下是获取高质量素材的步骤：

查看摄影师的作品集，了解他的价格。然后预约时间，让他为你拍照。**切记不要用自拍，一定要他拍，并且最好使用单反相机**。拍完后他通常还会为你修图。这些照片可以重复使用。因为你在社交软件上用的照片，和微信朋友圈等私域用的照片是通用的。

这方面的投资是必要的。优质女性通常会关注那些有经济实力、有品位的男性。一个男性想找一个漂亮女朋友，现实就是如此。社会现实，人性亦然。而且我提到的这些情场高手，很多人会去做微整形。

## 构建高价值人设与线上互动策略 💬

有了高质量的素材，我们还需要通过线上互动来构建并维持一个高价值的人设。

他们还会花钱购买二手豪华品牌汽车，因为他们特别懂得展示。大家有钱不一定非要买全新品牌。去旅游、吃好东西、拍照，这些活动表面是去玩，实际上是为了不断打造自己的人设，不断收集素材。这些素材至关重要。重视素材，用于发布在你的社交软件上。花几千几万精心挑选，与别人匹配时，购买那些付费浏览功能是值得的，不要花钱买那些乱七八糟的功能。

那么，在社交软件上，即使你的人设和照片过关了，还有几个关键点。这些点是老手才会告诉你的，就是在社交软件上聊天的方法。

以下是社交软件聊天的核心原则：

**第一，不要秒回。第二，不要急着要微信。**

在社交软件上匹配成功后，甚至不需要当天就聊天。比如匹配后，简单打个招呼，过几天再聊两句。你可以说“那天在忙”。如果你经常泡在软件上，有求必应，会显得你无所事事。即使照片很高大上，别人刷到这个社交软件也会觉得你人设是假的，你的难度就降低了。一般不急着要微信，你会发现别人会主动提出：“要么我们加微信继续聊？”

然后，即使转移到微信上，也是简单聊聊。觉得可以了就去见面，但不要在匹配当天就要微信。一定要延迟满足，一定要把你的人设衬托得高。你是一个真正松弛、真正的高富帅，没有任何需求感。其实在社交软件上打造人设，和视频打造人设是一模一样的。

## 持续经营与心态调整 ⚖️

构建人设和互动策略后，长期的经营和正确的心态是成功的关键。

我们要让他也看起来是全世界各地飞、吃好东西、坐豪华汽车、去美国、日本、法国、英国经商游览的人。一般人设拉高了，很多粉丝才愿意为你买单。如果你表现得很土、很穷，没有粉丝会给你买东西。所以会玩的人，平时就积累去英国、法国等国旅行的素材，随时拍照。小红书随时随地都在积累素材。朋友圈等私域也要保持素材的顺滑衔接。

加微信后，并不着急聊天，也断断续续地聊。这样经常会激发对方特别想认识你的感觉。人们担心容易匹配到的美女会趁机马上聊天、马上要微信。那么，以这种节奏聊，也有些匹配永远停留在软件上。敢于去匹配，但不要急急忙忙要微信，否则成功率基本很低。反过来，永远是价值强的一方掌握主动权。就拿故事来举例，就像男明星。

他们剩下很多粉丝，但没有女明星会“收割”这些粉丝。女明星只是“收割”那些企业家、圈内更高级的男性。这就说明一个道理：大家要不急不躁，不慌不忙。对于社交软件，这个渠道在当前环境下效率最高，并且你在上面的投资回报率高。

很多素材能变成资产，经常使用，所以重点推荐给大家。尤其大家挣钱这么忙，根本没空经常参加线下活动。社交软件这块是排在首位的，也非常非常重要。希望大家听完这一章之后，积极行动起来。

## 总结与行动指南 🚀

本节课中，我们一起学习了社交软件的完整使用策略。

把自己的社交软件资料，在照片素材、视频素材方面改进起来，把自己的优势如学历、公司、车、颜值都展示出来。这节课希望大家多听几遍，一定要掌握。

**核心行动清单：**
1.  **研究选择**：在小红书等平台研究各社交软件特点，选择适合自己“游戏风格”的平台试用。
2.  **投资素材**：聘请本地摄影师拍摄高质量他拍照片（**禁用自拍**），建立可重复使用的素材库。
3.  **打造人设**：通过旅行、美食、高端场景等素材，在社交资料和朋友圈构建高价值、松弛感人设。
4.  **掌握节奏**：聊天时**不要秒回**，**不要急于索要微信**，采用“延迟满足”策略，保持神秘感和高姿态。
5.  **保持心态**：不急不躁，专注于持续展示高价值，吸引对方主动推进关系。

记住，社交软件是展示和筛选的工具，核心在于通过优质素材和策略性互动，高效传递你的内在与外在魅力。
