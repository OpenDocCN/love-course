# 老白《约会达人》：情场3.0：【聊天解码】30.预期

## 概述
在本节课中，我们将学习如何运用预期技巧，使聊天更加顺畅，提升约会成功率。

## 什么是预期？
预期，即在聊天中描述一个虚拟的画面，让女生提前幻想与你的见面或生活场景。

### 预期的作用
预期可以：
1. 让女生提前了解见面后的可能活动，减少尴尬。
2. 增强女生对你的期待感，提高约会成功率。

## 预期的三个模式

### 模式一：确定邀约后植入见面场景
**公式**：描述一个亲密行为，将其与你们两个人联系起来。

**示例**：
- “见到我，如果你喜欢我，两分钟后我会跟你牵手。”
- “你不能因为我长得帅，趁我不注意偷亲我。”

### 模式二：长期关系描述
**公式**：用文字描述一个浪漫的画面，让女生感受到美好的未来。

**示例**：
- “我不喜欢花言巧语，更喜欢实际一点的生活。比如我们一起养只狗，每天推开门，它们就扑到我们的怀里。我们俩人一起窝在沙发里，你在我怀里，我们一起看电视。”

### 模式三：坚定长期关系
**公式**：展示你对未来的目标，让女生感到与你在一起有安全感。

**示例**：
- “即使我现在没有钱，但我是个潜力股。和你在一起，你会感到安全，因为我有理想，有理智，是个睿智的男人。”

## 总结
本节课中，我们学习了如何运用预期技巧，使聊天更加顺畅，提升约会成功率。通过三个模式，我们可以根据不同情况，选择合适的预期方式，让女生提前了解我们的想法，减少尴尬，增强期待感。
