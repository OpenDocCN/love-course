# 安老师社恐训练营：第13课 - 一去不返：社交恐惧再也不来了 🚀

在本节课中，我们将要学习如何巩固成果，并掌握三种应对社交恐惧可能复发的核心策略。这些方法旨在帮助你建立长期的心理韧性，确保社恐真正地“一去不返”。

经过前面课程的学习，我们已经基本克服了社交恐惧。社交恐惧对我们来说可能已经变得陌生。但是，如果社交恐惧再次出现怎么办？虽然概率非常小，但如果它来了，我们有什么应对措施？或者说有什么好的策略？今天将讲解三个策略。

## 策略一：向上的练习 📈

上一节我们介绍了课程的总体目标，本节中我们来看看第一个策略：向上的练习。什么是向上的练习？就是每天发现并记录生活中的小确幸。

以下是具体操作方法：
*   **记录形式**：可以写日记，或者尝试录制视频。现在很多应用有私密功能，录制的视频只能自己看见。
*   **记录内容**：用语言描述每天遇到的开心事。切记不要使用笼统的感觉和意向，要去找到事情积极的方面。
*   **核心要求**：所有思维上的、行为上的、感受上的描述，都必须是向上的、向前的。

这个方法有多么大的作用？它能带来爆炸性的改变。只这样做一两天可能没什么效果，但是坚持一个月两个月，效果就会非常明显。你的人生态度和心态会超越基本上90%的人，整个人的精神状态会变得非常好。做很多事情时，你会积极面对，积极思考解决问题的方法，变得非常睿智。时间久了之后，就会有脱胎换骨的感觉。

## 策略二：渐进式放松练习 🧘

好，这是第一点向上的练习。接下来是第二个策略：渐进式放松练习。其实放松也是需要方法的。当整个人感觉到很紧张的时候，可以主动调动身心去放松。

具体怎么做？分三个步骤。以下是操作流程：
1.  **施压**：向身体某个部位施压。施压不是按住，而是让这个部位使劲。例如，让手臂肌肉用力绷紧。
2.  **保持**：保持肌肉绷紧状态约5秒钟，用心体验紧张的感觉。
3.  **放松**：5秒钟之后，彻底放松肌肉。然后将注意力集中在身体该部位放松后的感觉上。

一个部位一个部位地练习，直到全身放松为止。这个方法其实随时随地都可以练习。

## 策略三：调整心情的呼吸法 🌬️

这是第二个渐进式放松练习。现在来看第三个策略：呼吸法。这个也是随时随地都可以练习的，核心是有意识地放慢呼吸。

练习分为两步。以下是具体步骤：
1.  **呼气**：一边在脑海中默念“放松”，一边缓慢呼气，持续约3秒钟。
2.  **吸气**：接着，慢慢地吸气，同样持续约3秒钟。

第一步和第二步交替循环进行。练习时，将注意力完全放在呼吸本身上，慢慢地你就会放松下来。

其实，无论是渐进式的放松练习，还是调整心情的呼吸方法，根本目的都是为了让自己能够主动地放松下来。**放松不仅仅是一项技能，更是一种态度和习惯**。学会放松之后，我们就可以淡定从容地应对社交和生活中的紧张感。
