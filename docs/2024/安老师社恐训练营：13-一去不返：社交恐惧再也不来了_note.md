# 安老师社恐训练营：13-一去不返：社交恐惧再也不来了

同学们，大家好。经过前面课程的学习，我们已基本克服社交恐惧。社恐对我们来说可能已陌生。但若社恐再次出现，我们该如何应对？概率虽小，但应对措施是必要的。

以下是三个应对策略：

### 1. 向上的练习

**定义**：每天发现一些小确幸，并用语言描述。

**方法**：
- 写日记或录视频（私密功能）
- 描述每天遇到的开心事
- 找到事情积极的方面
- 所有思维、行为、感受都向上、向前

**效果**：
- 爆炸性改变
- 一个月两个月后效果明显
- 超越90%的人
- 精神状态良好
- 积极面对问题，变得睿智

### 2. 渐进式放松练习

**定义**：主动调动身心放松

**步骤**：
1. 向身体某个部位施压（非按住）
2. 保持5秒绷紧肌肉，体验紧张感
3. 5秒后放松肌肉，集中注意力在放松感觉上
4. 逐部位练习，直至放松

**适用**：随时随地

### 3. 呼吸法

**定义**：有意识地放慢呼吸

**步骤**：
1. 念“放松”，呼气3秒
2. 吸气3秒
3. 交替进行

**效果**：
- 放松身心
- 应对社交和生活中的紧张感

**总结**：

本节课中，我们学习了如何应对社交恐惧。通过向上的练习、渐进式放松练习和呼吸法，我们可以更好地应对紧张感。希望同学们能将所学应用于实践，相信行动的力量。

最后，若仍有社恐困扰，可单独联系安老师寻求帮助。祝大家克服社交恐惧，人生一帆风顺！加油！
