# 17、海蓝老师情绪梳理个案课：038-清晨个人采访

🎼大家好，我是清晨，来自上海，主要在资讯行业工作。今天我想梳理的是关于健康方面的焦虑情绪，尤其是对家人的健康担忧，这给我的生活带来了一些困扰。

**以下是清晨的个人情况：**

1. **居住地与家庭关系**：我住在上海，但妈妈的老家在浙江。我希望妈妈能在我身边，避免回老家，因为担心她在老家会有健康问题。
2. **日常行为**：每天晚上我都会给妈妈打电话，当她在我身边时，我会感到安心和踏实。如果不在身边，我会感到不安，可能影响睡眠和焦虑。
3. **影响**：这种焦虑情绪可能影响睡眠、工作效率和生活质量，导致注意力不集中，甚至出现白发。
4. **目标**：我希望通过梳理，能够减轻焦虑，感到轻松，不再担心妈妈的健康问题，能够正常生活和工作。

**以下是清晨的期望：**

- 通过梳理，达到轻松的心态，不再焦虑。
- 当妈妈回老家时，能够放心，正常生活和工作。

**总结：**

本节课中，我们学习了清晨的个人情况和她希望通过梳理减轻的焦虑情绪。希望她能够通过情绪梳理，达到轻松的心态，过上正常的生活。
