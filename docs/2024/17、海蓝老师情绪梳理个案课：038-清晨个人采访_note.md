# 17、海蓝老师情绪梳理个案课：038-清晨个人采访_note

在本节课中，我们将通过学员“清晨”的真实个案，学习如何识别与梳理因家人健康问题而产生的焦虑情绪。我们将分析她的困扰、行为模式及期望，并从中提炼出情绪梳理的核心步骤。

---

我叫清晨，来自上海，主要从事资讯行业方面的工作。

我主要想梳理健康方面的情绪，具体是对家人健康的焦虑情绪。这种情绪对我的生活有一定困扰，因为我总是担心家里人的身体出状况。

我住在上海，但我妈妈的老家在浙江。我希望她留在我身边，不要回老家。一旦她回老家，我就会担心。我基本每天晚上都会打电话给她。她来到我身边时，我就会感到放心和踏实。

她不在我身边时，我心里就不踏实，可能会睡不好，会感到焦虑。虽然焦虑程度并非特别严重，但我感觉它确实有影响。它可能会影响我的睡眠，影响工作效率和整体生活质量，导致注意力不集中。我觉得自己这两年白头发都多了不少。

我梳理完了。我希望可以达到自己很轻松、不再那么焦虑的状态。我希望如释重负，能够放下这方面的焦虑，感觉很轻松。例如，即使妈妈回老家了，我也可以不用那么担心，能够正常地生活和工作。

---

上一节我们倾听了清晨的困扰描述，本节中我们来看看如何将这些信息整理成可操作的梳理步骤。以下是基于此个案总结的情绪梳理要点：

**核心情绪公式：**
`特定事件（家人离开）` -> `触发焦虑情绪` -> `导致生理反应（睡眠差、注意力不集中）`与`行为模式（频繁打电话）` -> `影响生活质量`

**核心目标代码：**
```python
if 家人不在身边:
    旧模式: 感到焦虑，影响生活
    目标新模式: 感到轻松，生活如常
```

首先，需要**识别情绪触发点**。清晨的焦虑直接由“母亲回老家”这一具体事件触发。

其次，要**觉察情绪带来的连锁反应**。焦虑情绪引发了生理上的睡眠问题、注意力不集中，并固化为每晚打电话确认安全的行为模式。

最后，明确**梳理的终极目标**。并非消除对家人的关爱，而是将焦虑情绪转化为一种平静的关心，从而恢复个人正常的生活与工作状态。

---

本节课中，我们一起学习了如何通过一个具体个案来梳理焦虑情绪。我们识别了情绪的外部触发事件，观察了其引发的身心与行为反应，并最终明确了将消耗性的焦虑转化为平和关心的调整目标。这个从“识别”到“觉察”再到“目标转化”的过程，是情绪梳理的通用路径。
