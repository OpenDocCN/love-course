# 课程03：超硬核沟通课 🗣️ – 1.0 宣传片：如何说清楚一件复杂的事

在本节课中，我们将探讨如何清晰表达复杂事物。我们会分析表达不清的常见原因，并介绍一个系统的表达框架，帮助你理解并提升自己的语言组织能力。

---

人人可学的工具或许是最没有价值的部分。有人能谈论绿茶的“茶清质量”和乌龙茶的“揉捻程度”，而你只能评价“好不好喝”。但如果你不了解背后的知识，你就无法说清自己为何说不清一件事。

我是汤质，本期节目的主理人。你现在看到的是我的说话课程的推广视频。这门课程旨在帮助你提升语言组织与表达能力，让你能说清楚一件件复杂的事情。

但你知道吗？推广这样的课程是极难的事。

因为如果我自己都无法说清你为什么需要这门课，那我就成了一个笑话。我不想，也不会成为笑话，因为对我而言，清晰表达似乎很容易。我有太多工具可以应对。

例如，要让听众对一个观点感兴趣并愿意听下去，可以使用“背景-冲突-问题-解决”这样的表达框架。在刚才的开场中，我就使用了“背景”与“冲突”的框架。

仅有兴趣还不够，我们还需让听众信服。要让听众的态度从质疑转向信服，就需要用到**图尔敏论证模型**。这个经典模型告诉我们，要论证一个主张（例如“我做了一门非常精彩的说话课”），你需要先列举事实（如走心的好评、热情的点赞）。

仅有事实不够。你最好还能讲出**理据**，说明这些事实如何支撑你的主张。也就是说，我们需要回答：“美誉度在多大程度上与课程质量相关？”

老实说，我回答不了这个问题。大家都说好的东西不一定适合你，更何况说好的人可能是水军。因此，我提供了足够长且完整的公开试看内容。好不好，你需要自己判断，谁说都不好使。

上面这番说辞，在图尔敏模型中称为**反驳**和**限定**。事实和理据能强化你的主张，反驳和限定则能软化你的主张。因为没人愿意被粗暴说服，我们需要软硬兼施。

这些都是非常好用的技巧。但我真正想说的是，上述技巧或工具并不能真正帮助你掌握说话的要领。

技巧人人可学，工具人人都能掌握。它们或许是最没有价值的部分。这些技巧在我们的课程框架中只占很小比重，属于“结构化”和“命题化”环节。这个环节决定了如何将网状的语料整理成线性的语流，决定了应先说什么、后说什么。

但真正的问题是：你到底想说什么？你有什么值得说的？有洞察力、启迪人心的观点从何而来？要知道，平庸的观点永远无法组织成发人深省的话语。

所以，我们要在这个框架中向前追溯，你会发现更具决定性的因素。

以下是决定表达能力的核心要素：

**第一，是经验值。** 一切语料都源于经验。好的表达能力，其基本前提是拥有足够丰富且细致的感知经验。如果我要与你谈论眼前这杯茶，我就需要有喝茶的经验。有人能区分冲泡前后龙井口感的差异，有人只能区分红茶和绿茶。

**第二，是词汇量。** 丰富的经验内容需要有与之匹配的词汇网络来标记。一个领域的专家往往掌握大量专门词汇来描述经验细节。同样是品茶，有人能谈论绿茶的“茶清质量”和乌龙茶的“揉捻程度”，而你只能评价“好不好喝”。

**第三，是知识量。** 谈论“茶清质量”和“揉捻程度”或许能唬住外行，但要真正说清它们与口感的关系，就需要大量原理性知识。否则，你无法经受刨根问底的追问，也无法为自己的观点辩护，在别人眼中不过是个夸夸其谈的“知道分子”。

有了经验、匹配经验的词汇以及阐述原理的知识，你才需要考虑用何种结构组织你的言谈——是以理入室、娓娓道来，还是开门见山、结论先行。差异其实并不大。如果你是真正的专家，自然会有无数角度来展开一个话题。

因此，我们说：**先有“脑才”，再谈“口才”。**

当然，即使以上要素都齐备，你可能仍显得嘴笨。那很可能是因为你的**短时记忆容量**不够，无法同时在脑中缓存足够多的语料以供取用。此时，你最好通过反复演练来加强记忆。很多流畅自如的言谈，其实都是“强准备”的结果——正如这杯茶是事先泡好的，我现在说的每句话都有逐字稿。

我们都知道说清楚一件事很难。但到底难在哪呢？

你会发现，说不清一件事，至少有五种可能：

1.  **经验匮乏型**：因经验不足或分辨力不够，导致“无米之炊”。
2.  **语料匮乏型**：因词汇（心理标记）不足，导致“意无穷，词太穷”。
3.  **知识匮乏型**：因原理性知识不足，导致成为“知道分子”。
4.  **结构缺失型**：因结构化工具缺失或整合能力不足，导致“整不明白”。
5.  **记忆容量型**：因工作记忆容量太低，导致“阿兹海默型”表达。

或许还要加上因口语表现力不足导致的“嘴里拌蒜型”。

当你不知道上述知识时，你无法说清自己为何说不清一件事。

学完这节课后，你可能仍然说不清某些复杂的事。这节课没有魔法，它只是前面提到的、用于澄清原理的知识而已。你不能期待仅靠知识就能轻松说清所有事，这是虚妄的期许——世界上不存在能说清所有事的人。

但我至少敢保证，通过这些知识，像“如何好好说话”、“如何清晰表达”、“如何说服他人”、“如何倾听与回应”这类问题，将不再是含混不清、令人畏惧的课题。它会变得清晰、通透、条理分明。

我希望你就像拿到了地图的旅者，因此有了上路的动力和信心。

关于为什么要加入我这堂说话课，不知道我说清楚了吗？

---

**本节课总结**：我们一起学习了清晰表达复杂事物的系统性框架。我们认识到，表达不清可能源于经验、词汇、知识、结构或记忆等多方面不足。真正的表达能力始于丰富的经验、精准的词汇和扎实的知识（“脑才”），而后才是表达结构与技巧（“口才”）。本课程旨在为你提供一张清晰的“地图”，帮助你理解并系统地提升自己的沟通能力。
