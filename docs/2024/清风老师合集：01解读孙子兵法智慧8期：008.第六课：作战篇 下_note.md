# 清风老师合集：01解读孙子兵法智慧8期：008.第六课：作战篇 下

在本节课中，我们将继续探讨孙子兵法中的作战篇，深入理解其智慧，并将其应用于我们的工作和生活中。

## 善用兵者亦不在己，粮不三在

孙子兵法中提到，善于用兵的人不需要频繁征召士兵，粮食也不需要储备过多。这意味着一次性的调动和快速作战是关键。

**公式**： 善用兵者 = 一次性调动 + 快速作战

## 取用于国，阴良于敌

孙子提出，在战争中，首先要满足自己的需求，然后从敌人那里获取资源，以消耗敌人。

**公式**： 取用于国 = 自我需求满足 + 从敌人获取资源

## 国之贫与师者远殊，远殊则百姓贫，近于师者贵卖

战争会导致国家贫困，远离军队的地方百姓贫困，靠近军队的地方物价上涨。

**公式**： 国家贫困 = 远离军队的百姓贫困 + 靠近军队的物价上涨

## 力区才单，中原内需于家，百姓之肺失去其七，公家之废，破车霸马

战争会导致国家资源枯竭，军队战斗力下降。

**公式**： 国家资源枯竭 = 军队战斗力下降

## 古制驾务食于敌，食敌一中，当午二十中，岂敢一旦当午二十旦

孙子建议从敌人那里获取粮食，以减少自己的负担。

**公式**： 从敌人获取粮食 = 减少自己的负担

## 故杀敌者怒也，取敌之利者祸也

孙子认为，战胜敌人需要愤怒，但仅仅依靠愤怒是不够的，还需要利益驱动。

**公式**： 战胜敌人 = 愤怒 + 利益驱动

## 古车站得车十成以上，赏其先得者，而更其精奇，车杂而诚职之足善而养之，是为胜敌而一强

孙子建议奖励先取得胜利的人，并培养他们的战斗力。

**公式**： 奖励先取得胜利的人 = 培养战斗力

## 故兵贵胜不贵酒，古之兵之家生明之司命，国家安危之主也

孙子强调，战争的目标是胜利，而不是消耗。

**公式**： 战争目标 = 胜利
