## 串联练习前的四个要点 🔑

在开始串联练习之前，有四个关键点需要注意。这些要点是串联练习安全、稳定和有效的基础。

以下是四个核心要点：

1.  **双腿的扎根感**：双腿力量要均匀和稳定，为身体提供坚实的基础。
2.  **脊柱的延展空间**：在练习中，始终要为脊柱创造向上延展的空间。
3.  **动作转换的觉知**：在动作与动作转换之间，要细腻地感受身体是如何缓慢、有控制地移动的。
4.  **呼吸的顺畅感**：保持呼吸的顺畅是串联练习中最重要的部分，呼吸应引导动作。

## 串联练习第一部分

上一节我们介绍了串联练习的四个要点，本节中我们来看看第一组串联的具体步骤。请准备两块瑜伽砖，放在垫子前端两侧。

### 准备姿势与热身

首先，屈膝跪在垫子中间。

以下是准备姿势的细节：

*   十个手指大大张开。
*   双手打开，与肩膀同宽。
*   双膝分开，与骨盆同宽。

若不确定距离，可将手肘抵住膝盖，手掌压地后，让掌根压在中指位置，这个距离基本就是标准的四肢支撑准备姿势。

可以选择勾脚趾（骨盆更稳定）或绷脚背（释放膝盖压力）。现在开始调整呼吸，并让脊柱做柔和的热身。

**脊柱热身练习**：
随着吸气，手推地，提胸口，抬头。代码描述：`吸气 -> 手推地 + 提胸口 + 抬头`
随着呼气，手推开，拱背，低头，腹部内收，看向肚脐。代码描述：`呼气 -> 拱背 + 低头 + 收腹`
重复此流动3-4次，感受背部肌肉的启动。

### 下犬式及站姿体式串联

热身结束后，我们进入下犬式（Adho Mukha Svanasana）。

确保十个手指大大张开，勾脚趾。吸气准备，呼气时推臀部向上，脚跟向下踩，停留5个呼吸。公式：`下犬式 = 手推地 + 坐骨上提 + 脚跟下踩`

在下犬式保持中，关注：
1.  虎口压实地面。
2.  双肩远离双耳，颈部放松。
3.  推送坐骨向上。
4.  保持坐骨高度，脚跟向下踩。
5.  聆听顺畅的呼吸。

接下来，我们将从下犬式进入一系列站姿体式。

**序列示例（以右侧开始为例）**：
1.  从下犬式，转左脚朝外，右脚向前跨到双手之间。
2.  两脚踩稳，吸气，手臂上举，进入战士一式（Virabhadrasana I）。保持5个呼吸。要点：前膝对准第二脚趾，后腿伸直，脊柱延展。
3.  呼气，双手落回右脚两侧，撤右脚回到下犬式。保持几个呼吸。
4.  换左侧重复（转右脚朝外，左脚向前跨）。

随后，我们加入三角伸展式（Utthita Trikonasana）的变体。从下犬式开始，转脚、向前跨步后，用外侧手推砖，内侧手臂上举，进入三角伸展式。保持呼吸，感受双腿稳定和侧腰伸展。

之后，我们加入扭转三角式（Parivrtta Trikonasana）的预备练习。从下犬式跨步后，将砖放在前脚内侧，手推砖，另一手臂上举，进入扭转。保持稳定呼吸，初学者可先看地板，稳定后再尝试转头看上方手指。

最后，从扭转回到下犬式，并进入战士二式（Virabhadrasana II）。从下犬式跨步，先进入战士一式，再打开髋部与手臂，进入战士二式。保持5个呼吸，关注骨盆摆正，手臂向两侧延展。

完成一侧练习后，换另一侧重复所有站姿体式序列。

### 休息与过渡

完成站姿序列后，回到下犬式调整呼吸。然后屈膝，臀部坐向脚后跟，进入婴儿式（Balasana）休息。手肘落地，额头轻触地板，放松手臂和手腕，调整呼吸。

这就是我们串联练习的第一部分。

## 新体式讲解：斜板式与四柱支撑

在开始第二组串联前，我们需要学习两个新体式：斜板式（Phalakasana）和简易四柱支撑。它们是串联中重要的过渡体式。

**斜板式（Phalakasana）**：
从四肢支撑准备式开始，手在肩正下方，勾脚趾，伸直双膝，双脚分开与骨盆同宽。脚跟向后蹬，头顶和脚跟向相反方向延伸，手推地，腹部微收。公式：`斜板式 = 身体呈一条直线（肩、背、臀、腿）`
常见错误是骨盆下沉。纠正方法是：可将脚稍向前走，让臀部略微推高，找到腹部内收的感觉。

**简易四柱支撑（Chaturanga Dandasana 变体）**：
在斜板式基础上，屈膝跪地，然后屈手肘，让手肘夹住身体肋骨两侧。感受手臂和背部肌肉的发力。然后推起身体回到斜板或进入下犬式。
对于初学者，做到屈膝屈肘这一步即可。随着练习深入，可以尝试胸腔、下巴点地，然后穿出进入上犬式或眼镜蛇式，但本课暂不要求。

## 串联练习第二部分

上一节我们学习了斜板式和简易四柱支撑，现在我们将它们融入第二组更流畅的串联中。

首先，站到垫子前端，山式（Tadasana）站立。调整几组呼吸，让心绪平静。

**热身流动**：
吸气，双手高举过头顶合十。
呼气，手臂打开，微屈膝，身体向前折叠（前屈式，Uttanasana）。腿后侧紧的学员可保持屈膝。
重复此流动2-3次，作为热身。

### 串联序列详解

热身结束后，开始正式串联。

**基础串联单元**：
1.  **前屈式** 呼气进入。
2.  **斜板式** 吸气，手压稳，双脚向后跳或走到斜板式。保持5个呼吸。
3.  **简易四柱** 呼气，屈膝，屈手肘夹肋向下。吸气，推起。
4.  **下犬式** 呼气，勾脚趾，推臀进入下犬式。保持5个呼吸。

这个“斜板-四柱-下犬”的单元将作为后续串联的固定过渡。

以下是第二组串联的部分序列示例：

1.  从**下犬式**，转左脚朝外，右脚向前跨，进入**战士一式**。
2.  呼气，双手在背后食指交扣，进入**谦卑战士式**变体（保持前腿弯曲，上身向前折叠）。保持呼吸。
3.  呼气，双手落回脚两侧，后脚后撤，回到**斜板式**。完成**简易四柱**单元后，进入**下犬式**。
4.  换左侧重复战士一式和谦卑战士式。
5.  从**下犬式**进入**战士二式**，然后进入**侧角伸展式**（Utthita Parsvakonasana）。保持呼吸。
6.  回到**斜板式**，完成**简易四柱**单元后，进入**下犬式**。
7.  换另一侧重复战士二式与侧角伸展式。
8.  后续串联中，加入**三角伸展式**、**扭转三角式**等体式，每个站姿体式后都通过**斜板式**与**简易四柱**单元连接，最后回到**下犬式**。

### 结束序列

完成所有站姿串联后，进行舒缓的结束练习。

1.  从**下犬式**屈膝跪地，手臂向前伸展，进入**大猫伸展式**（Marjaryasana）。胸腔贴地，额头或下巴点地，放松呼吸。
2.  慢慢推起身体，回到**下犬式**短暂调整。
3.  最后，进行**倒箭式**预备练习或**双角式**（Prasarita Padottanasana）变体：屈手肘，小臂贴地，头顶离地。勾脚趾，腿后侧紧可提脚跟。保持呼吸，感受肩背的伸展。
4.  缓慢屈膝，以婴儿式放松，调整呼吸。

## 总结 📝

本节课中我们一起学习了瑜伽的串联练习。我们从四个核心要点（根基、延展、觉知、呼吸）开始，逐步练习了两组完整的串联序列。第一组串联侧重于基础站姿体式的连接与呼吸配合；第二组串联引入了斜板式和简易四柱支撑作为动态连接，增加了序列的流畅性和力量元素。

记住，串联练习的精髓在于呼吸与动作的同步，以及动作之间平稳、有觉知的过渡。初学者务必根据自身情况调整，使用辅具（如瑜伽砖），并始终将呼吸的顺畅放在首位。定期练习将帮助你建立力量、柔韧性和整体的流动感。
