## 概述

在本节课中，我们将学习如何通过理解人性，运用特定的方法，实现情感关系的逆袭挽回。

## 关键概念

**公式**： 挽回 = 理解人性 + 方法技巧

## 案例分析

**案例**： 一位会员从被对方转了20万元，到彻底转变关系，对方主动付出，开始收到5200元的转账。

**逆袭关键**： 

1. **排斥期**： 冲淡负面影响。
2. **怀念期**： 建立舒适的聊天。
3. **求证期**： 重新建立框架，让对方投入付出。

## 情感挽回的三个阶段

1. **排斥期**： 对方讨厌你，不理你。
2. **怀念期**： 对方开始思考你们关系的美好。
3. **求证期**： 对方想要与你重新回到这段关系。

## 核心战略

- **排斥期**： 冲淡负面影响。
- **怀念期**： 建立舒适的聊天。
- **求证期**： 重新建立框架，让对方投入付出。

## 技术方法

1. **冷冻术**： 主动关闭话题，给对方处理情绪的时间。
2. **后撤术**： 关闭朋友圈，设置黑头像，让对方看到你不会打扰他。
3. **测试术**： 通过客观话题测试对方反应，如询问帮助。

## 案例指导

**案例**： 会员与弟弟闹矛盾，弟弟主动求加回微信。

**指导**：

1. 不要心软，等对方提出解决方案。
2. 给出坚定态度，要求解决方案。
3. 转移话题，避免只聊加微信。
4. 确定见面时间地点，明确原则。

## 危机术与诱惑术

1. **危机术**： 透露有其他人在追求你，让对方有危机感。
2. **诱惑术**： 展现性感妩媚的一面，增加对方投入和付出。
