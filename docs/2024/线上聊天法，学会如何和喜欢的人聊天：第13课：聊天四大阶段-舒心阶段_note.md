# 线上聊天法，学会如何和喜欢的人聊天：第13课：聊天四大阶段-舒心阶段

🎼大家好，欢迎来到咱们的第十三节。这节课我们将探讨四大阶段的舒心阶段。经历前三个阶段，尤其是熟悉阶段后，你们的聊天和关系将变得更加亲密和暧昧。

### 舒心阶段概述

那么，此时我们将进入舒心阶段，这是一个准恋人的阶段，其中包含了一些恋爱感觉。在这个阶段，我们的特点如下：

1. **自由自在的气氛**：你可以随意表达自己的想法，例如在玩游戏时，女生可能会找你聊天，并发送大量表情或撒娇。
2. **暴露需求感，强联系感**：你可以表达想见她或想和她见面的愿望。
3. **历程跟语气的暧昧化**：可以使用“宝贝”、“亲爱的”等亲昵词汇。
4. **进入彼此的生活**：分享彼此的私事，如去医院或和朋友聚餐。
5. **结盟，共同付出**：共同购买物品或送礼物。
6. **考虑未来**：讨论旅游计划、职业安排或生活变动。

### 舒心阶段分期

舒心阶段分为两个阶段：

1. **意向期**：表达爱意，接受对方的爱意。这是一个表白的好时机。
2. **测试期**：女生内心犹豫，可能因为不确定你的真心或对恋爱期望的不确定性。

### 测试期应对策略

1. **言行一致**：真诚地表达你的爱意，并付诸行动。
2. **询问女生的期望**：了解她想要的恋爱类型，并做出相应的改变。

### 总结

本节课我们学习了舒心阶段的特点和分期，以及如何应对测试期。希望这些知识能帮助你更好地与喜欢的人聊天。
