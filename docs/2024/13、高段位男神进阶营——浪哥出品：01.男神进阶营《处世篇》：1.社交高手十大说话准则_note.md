# 13、高段位男神进阶营——浪哥出品：01.男神进阶营《处世篇》：1.社交高手十大说话准则

大家好，欢迎来到高段位男神进阶营的课程。提升个人段位，摆脱内耗，提高效率，把握机会，成为更优秀的人。

**课程概述**： 本节课将围绕社交高手的十大说话艺术原则展开，帮助大家提升社交技巧，更好地融入社会。

### 1. 有关别人的事情，不要说

**原则**： 来说是非者，必是是非人。避免在社交场合谈论别人的是非，尤其是背后议论他人。

**公式**： 避免谈论他人是非 → 避免引起误会 → 维护人际关系

### 2. 有关自己的事情，要听别人说，不反驳，不解释

**原则**： 保持谦卑，接受他人的评价，不要过多解释。

**公式**： 听取他人评价 → 保持谦卑 → 提升自我认知

### 3. 别人家的事，多听少说，以情绪疏导为主

**原则**： 尊重他人隐私，避免评价他人家庭。

**公式**： 尊重他人隐私 → 避免评价他人家庭 → 维护人际关系

### 4. 遇到急事，慢慢说

**原则**： 保持冷静，避免情绪化。

**公式**： 保持冷静 → 沉着应对 → 解决问题

### 5. 凡是小事，幽默点说

**原则**： 用幽默化解尴尬，缓解气氛。

**公式**： 幽默化解尴尬 → 缓解气氛 → 提升人际关系

### 6. 没有把握的事，谨慎的说

**原则**： 避免添油加醋，传播谣言。

**公式**： 避免添油加醋 → 避免传播谣言 → 维护人际关系

### 7. 没发生的事，不要胡说

**原则**： 避免猜测，传播未经证实的信息。

**公式**： 避免猜测 → 避免传播未经证实的信息 → 维护人际关系

### 8. 做不到的事，不要乱说

**原则**： 避免乱承诺，失去信任。

**公式**： 避免乱承诺 → 保持诚信 → 提升人际关系

### 9. 伤害别人的话，不能说

**原则**： 尊重他人，避免伤害他人。

**公式**： 尊重他人 → 避免伤害他人 → 维护人际关系

### 10. 自己的伤心事，不要见人就说

**原则**： 保护自己，避免传递负能量。

**公式**： 保护自己 → 避免传递负能量 → 提升人际关系

**总结**： 本节课介绍了社交高手的十大说话艺术原则，希望大家能够认真学习并应用到实际生活中，提升自己的社交技巧，成为更优秀的人。
