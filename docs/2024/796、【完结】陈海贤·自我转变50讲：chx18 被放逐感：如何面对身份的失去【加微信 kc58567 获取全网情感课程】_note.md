# 796、【完结】陈海贤·自我转变50讲：chx18 被放逐感：如何面对身份的失去

欢迎来到自我转变五十讲。我是陈海贤。上节课我们探讨了目标带来的思维转化以及失去目标后的痛苦。这种痛苦源于我们无法获得一直渴望的身份。

然而，转变有时意味着另一种情况：原本是他人羡慕的群体成员，拥有令人瞩目的身份。但出于某种原因，你放弃了它，这同样会带来巨大的失落。

**本节课我们将探讨如何面对这种身份的失去。以下是一些关键点：**

### 1. 身份的失去与放逐感

- **身份的失去**：指的是失去一个优越的地位和与他人达成的关于你是谁的共识。
- **放逐感**：在转变期失去身份的心理落差，源于潜意识的本能恐惧。

### 2. 身份失去的原因

- **追求优越地位**：人类作为社会动物，总是寻找更优越的地位。
- **限制自我**：原有的身份限制了自我中与身份不符的部分。

### 3. 身份失去的痛苦

- **被放逐的痛苦**：失去原有的群体归属感，感到被排斥。
- **寻找新的意义**：离开后需要寻找自己的意义。

### 4. 如何面对身份的失去

- **接受失去**：接受失去身份的现实，不要逃避。
- **寻找新的身份**：寻找新的群体和身份，获得新的归属感。
- **自我觉醒**：依赖自己，信任自己，成为自己的指南针。

**以下是一些例子，帮助你更好地理解：**

- **从高校辞职创业的朋友**：失去高校教师的身份，感到失落。
- **从体制内辞职的来访者**：失去体制内的工作，感到被放逐。
- **从大厂离职的朋友**：失去在大厂工作的身份，感到迷茫。

**总结**

本节课我们探讨了身份的失去、放逐感以及其对新自我的意义。希望你能从中获得启发，勇敢面对身份的失去，寻找新的自我。
