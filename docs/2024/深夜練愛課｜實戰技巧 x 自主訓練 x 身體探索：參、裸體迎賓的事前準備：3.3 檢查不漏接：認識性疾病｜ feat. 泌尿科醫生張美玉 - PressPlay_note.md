## 1. 檢查不漏接：認識性疾病
### 1.1 阴蟲
- **陰蟲**是通過接觸傳染的，例如衣物、毛巾或床單。
- 症狀包括瘙癢、皮疹和出血點。
- 治療方法包括剃除陰毛和塗抹專門殺蟲的藥膏。

### 1.2 疱疹
- 疱疹分為單純疱疹和帶狀疱疹。
- 單純疱疹是幾個小水泡，常見於免疫力低下的人。
- 帶狀疱疹是水泡，常見於中年人，並伴有疼痛。
- 治療方法包括藥物治療和休息。

### 1.3 菜花
- 菜花是尖圭濕疣，由人類乳頭狀病毒引起。
- 症狀包括肉芽狀物和皮疹。
- 治療方法包括藥物治療和外科手術。

### 1.4 淋病
- 淋病是由淋病雙球菌引起的性病。
- 症狀包括尿道分泌物、瘙癢和疼痛。
- 治療方法包括抗生素治療。

### 1.5 梅毒
- 梅毒是由梅毒螺旋體引起的性病。
- 症狀包括潛伏期、潰瘍和淋巴结腫大。
- 治療方法包括抗生素治療。

### 1.6 喉痘
- 喉痘是由猴痘病毒引起的性病。
- 症狀包括皮疹和疼痛。
- 治療方法包括藥物治療和休息。

### 1.7 传染性軟疣
- 传染性軟疣是由人類乳頭狀病毒引起的性病。
- 症狀包括皮疹和瘙癢。
- 治療方法包括藥物治療和外科手術。

## 2. 預防性病
### 2.1 安全性行為
- 使用保險套可以降低感染性病的風險。
- 選擇固定的性伴侶可以降低感染性病的風險。

### 2.2 定期檢查
- 定期檢查可以發現和治療性病。
- 女性應該每年進行一次妇科檢查。

### 2.3 疫苗接種
- 疫苗可以預防某些性病，例如人類乳頭狀病毒和梅毒。
