# 老公出轨心理27堂课：19 挑明老公出轨后，他玩失踪，怎么应对？

## 概述
在本节课中，我们将探讨老公出轨后玩失踪的情况，以及如何应对这种情况。

## 案例分析
以下是几个案例，用以说明老公出轨后玩失踪的情况：

1. **案例一**：妻子发现老公出轨后，老公否认，随后消失。最终，老公与小三分手，与妻子和解。
2. **案例二**：老公欺骗小三，声称妻子出轨，小三离婚后与老公在一起。一年多后，老公消失，小三发现老公有家庭和孩子。
3. **案例三**：小三怀孕后，老公消失。小三发现老公有家庭和孩子，陷入困境。

## 老公玩失踪的原因
老公玩失踪可能有以下几种原因：

1. **激情状态**：老公与小三陷入热恋，认为离开小三无法生活。
2. **小三逼迫**：小三逼迫老公必须在一起。
3. **边缘型人格**：老公没有婚姻约束，以自我为中心。
4. **对妻子愤恨**：老公对妻子有各种不满，想离家出走。
5. **经济压力**：老公遇到经济问题，被迫离家出走。

## 应对策略
面对老公玩失踪的情况，可以采取以下策略：

1. **评估婚姻**：评估婚姻满意度、婚姻发展、婚姻矛盾冲突。
2. **评估老公**：评估老公是否经常离家出走、是否付出型或掠夺型、婚姻压力。
3. **明确要求**：要求老公回家，否则报警或通知单位。
4. **报警**：如果老公无音讯，报警并协助定位。
5. **面对小三**：与小三沟通，了解情况。
6. **保持理性**：保持理性，避免情绪失控。
7. **设定截止日期**：设定截止日期，如果老公未归，采取法律手段。
