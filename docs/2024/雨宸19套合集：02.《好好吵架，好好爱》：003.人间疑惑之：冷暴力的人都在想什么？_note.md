# 雨宸19套合集：02.《好好吵架，好好爱》：003.人间疑惑之：冷暴力的人都在想什么？

## 概述
在本节课中，我们将探讨冷暴力现象，分析冷暴力者的心理状态和需求，以及如何应对冷暴力。

## 冷暴力的类型
以下是冷暴力的两种主要类型：

1. **情绪上头型**：争吵后不理睬对方，表现出冷漠的态度。
   - **内心状态**：愤怒、不满，通过冷暴力表达情绪。
   - **常见于**：男性，争吵时选择沉默。

2. **回避性依恋型**：认为对方是施暴者，通过冷暴力保护自己。
   - **内心状态**：不满、恐惧，缺乏勇气提出分手。
   - **常见于**：依赖性强、需求度高的个体。

## 冷暴力的成因
以下是导致冷暴力的几个原因：

1. **内化父母的相处模式**：从小未得到正确情绪表达的机会。
2. **缺乏情绪认同**：表达情绪得不到认同和满足。
3. **自我保护**：通过冷暴力避免冲突和负面情绪。

## 应对冷暴力的方法
以下是应对冷暴力的几个方法：

1. **积极沟通**：主动表达自己的感受和需求。
2. **打破预期**：努力过好自己的生活，保持独立。
3. **以暴制暴**：以对方的方式回应冷暴力。
4. **引导对方表达**：让对方了解你的期待和要求。

## 总结
本节课中，我们学习了冷暴力的类型、成因和应对方法。了解冷暴力者的心理状态和需求，有助于我们更好地应对这一现象。
