# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：55、亲密关系中，不断产生分歧，不想分开，该如何做

## 概述
在本节课中，我们将探讨如何在亲密关系中处理不断产生的分歧，以及如何避免分手或离婚。

## 核心概念
**公式**： 重新框定状况 = 新的认知 + 共同的积极面

## 教程内容

### 1. 分歧的根源
以下是关于分歧的根源：

- **价值观不协调**：价值观的改变非常困难，尤其是当试图改变他人时。
- **主观价值体系**：人们倾向于为自己的价值观辩护，并期望对方接受。

### 2. 处理冲突的方法
以下是处理冲突的方法：

- **找到共同支持的积极面**：即使存在分歧，也要寻找双方都认同的积极方面。
- **重新框定状况**：更新对状况的理解和认知，以促进不同的行为反馈。

### 3. 换框架的步骤
以下是换框架的步骤：

- **记录情绪和事件**：养成记录情绪和事件的习惯，以便更清晰地看到问题。
- **设想最糟糕的结果**：考虑争执对亲密关系的影响，以及是否值得继续。
- **确定框架**：识别局限行为、观念和态度的框架。
- **找到框架的积极意义**：寻找框架中积极的方面，并达成共识。
- **找到其他方式达到积极目的**：避免旧框架的影响，寻找新的解决方案。
- **审视自己**：确保自己真正接受新的框架，并持续体验其带来的积极变化。

### 4. 框架的重要性
以下是框架的重要性：

- **改变视角**：不同的框架会导致不同的视角和解决方案。
- **降低冲突**：通过重新框定状况，可以降低冲突的可能性。

## 总结
本节课中，我们一起学习了如何在亲密关系中处理不断产生的分歧。通过重新框定状况，找到共同的积极面，我们可以降低冲突的可能性，并找到解决问题的方法。
