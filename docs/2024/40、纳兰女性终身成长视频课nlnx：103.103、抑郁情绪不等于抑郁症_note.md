**情绪病的定义**

情绪病是由于过度或短暂的兴奋、抑郁、焦躁等情绪引起的情绪波动或精神状态不佳，是一种普遍的心理疾病。类似于身体亚健康状态，情绪也会有亚健康，就像身体会感冒，情绪也会“感冒”。

**抑郁情绪的普遍性**

抑郁情绪是普遍现象，每个人都会有或曾经有过，或正在疗愈，或已经疗愈，或接下来将要出现。长期情绪不佳需要咨询医生，确诊是否属于疾患。

**抑郁情绪与抑郁症的区别**

1. **普遍性**：抑郁情绪普遍存在，但不是每个人都有抑郁症。
2. **严重性**：抑郁症的严重性在于多发和极端行为，如自杀倾向。
3. **人群**：抑郁症不分人群、年龄，非常普遍。

**抑郁情绪的表现**

1. 低能量：外在表现无力，内在表现无精气神，不愿交际，做事效率低。
2. 低自我价值感：自我怀疑，自责，内疚，看不到自己的优点。
3. 情绪低落：时常陷入委屈和痛苦，无法自拔。

**抑郁情绪的原因**

1. 工作失误、失业、商场失利、人生挫败等。
2. 个人经历、原生家庭、特殊事件等带来的伤痛。

**抑郁症的表现**

1. 思维认知：持续性痛苦，不可改变。
2. 负面性：所有事情都很糟糕，抵触积极思维。
3. 自我扭曲：严重的自我贬低、自我怀疑、自我攻击。
4. 生活态度：对任何事情都没有兴趣，感到生活没有意义。
5. 躯体感受：表情僵硬麻木，身体沉重，动作迟缓，头昏头疼，记忆力衰退，睡眠障碍。
6. 行为反应：害怕、拒绝与外界接触，反复出现自杀念头和行为。

**抑郁症的成因**

1. 外部环境：个人经历、原生家庭、特殊事件等带来的伤痛。
2. 人际关系：没有亲密的人际关系，或在重要关系中感到冷漠、不理解、不被尊重。
3. 长期的压抑环境：长期浸泡在负面消息中。
4. 个人因素：生物科学生理因素（遗传、性格、基因等）、情志失调、体质虚弱。

**总结**

本节课中，我们学习了抑郁情绪与抑郁症的区别、表现、成因等。希望大家能够正确认识抑郁情绪，及时调整心态，保持身心健康。
