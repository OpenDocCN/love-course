# 雅俗共赏《两步走：最有效的“社交”策略》：3.3 节奏把控及重点事项 🎯

## 概述

在本节课中，我们将学习如何在线下第一次约会中掌握节奏，并注意一些关键事项，以确保留下良好的印象。

## 节奏把控

**1. 可做可不做**

*   在约会中，应避免过度行为，以免给对方留下不好的印象。
*   在不确定是否应该做某事时，最好不做。

**2. 互动程度和内容**

*   使用行为来表示互动程度和内容。
*   从日常工作和兴趣爱好等泛泛而谈开始，逐步过渡到深入对话和雅俗共赏。

**3. 分值量化**

*   使用分值来量化互动程度。
*   注意分值不是等距的，例如10到45和45到60。

**4. 关系程度和实际发生的事情**

*   使用两根柱子来表示关系程度和实际发生的事情。
*   关系程度应高于实际发生的事情。

**5. 合适的区间**

*   在合适的区间内进行互动，以留下良好的印象。

## 重点事项

**1. 约会地点和时长**

*   第一次约会推荐选择咖啡厅、citywork或餐厅。
*   约会时长控制在60到90分钟。

**2. 不卑不亢**

*   保持平常心，避免过度投入。

**3. 基础目标**

*   留下好印象，让对方意犹未尽。

**4. 距离和见面方式**

*   如果距离较远，建议让对方来找你。

**5. 花钱**

*   钱应该花在两个人一起的事情上。
*   在对方交了底牌之后再花钱。

**6. 关系期待**

*   根据对方的性格和自己的诉求，决定释放什么样的信号。

**7. 速通**

*   速通需要承担风险，建议稳妥行事。

**8. 呼应对方情绪**

*   在适当的时候呼应对方的情绪，并给出反馈。

**9. 第一次约会成功的标准**

*   能做到雅俗共赏。

**10. 分开后指标**

*   对方是否会主动找你、分享话题并敲定第二次邀约。

## 总结

本节课中，我们学习了如何在线下第一次约会中掌握节奏，并注意一些关键事项。通过掌握这些技巧，你将能够留下良好的印象，并提高约会成功的几率。
