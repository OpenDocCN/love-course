# 雨宸婚姻心理课03：婚姻中，你的底气是什么？💪

在本节课中，我们将探讨维持婚姻关系的核心“底气”究竟是什么。我们将分析为什么外在条件靠不住，并学习如何通过内在建设，建立稳固且能动态发展的亲密关系。

上一节我们讨论了婚姻中的常见误区，本节中我们来看看，维系婚姻的真正力量源自何处。

## 需求是动态变化的 🔄

你可以问自己：在婚姻中，我凭什么去维持一段感情？是靠金钱、外貌，还是对方的诺言？首先你要知道，这些东西都是靠不住的。人和人的关系在不断变化，只有保持一种动态的平衡，才有可能真正拥有长久的相伴。

因为需求这个东西是会变的。即使当年互相满足了对方的核心需求，能够成为彼此的唯一，人的需求也会随时间变化。你可以想象一下，5年前的需求和5年后的需求还一致吗？5年前的择偶观和5年后的择偶观是否还一致？答案一定是否定的。

也就是说，人对伴侣的要求、对恋爱的需求在不断变化。所以你要学会保持动态平衡，不断跟上变化。如果对方在变而你没变，最后被关系抛弃的一定是你。这就是“七年之痒”的根源：即使当初相互满足，但随着时间推移，对方的需求变了，而你没变。

## 如何建立动态平衡？ 🛠️

既然知道了需求会变，我们要学会保持动态平衡，那具体该怎么做呢？以下是四个核心方法。

### 1. 明确婚姻目的与自我探索

首先，你要明确自己进入婚姻的目的是什么，自己想要的是什么。说白了，就是完成一个自我探索。

为什么很多人会出轨、跟异性暧昧，而有些人不会？花心是人的一种本能需求。人分两种驱动：本能需求和理智。为什么有的人能用理智克制本能？因为他清楚自己的需求与追求，知道“我是一个什么样的人”、“我在追求什么”。一旦发现行为与追求背离，他会及时止损。

例如，很多人不想暧昧、不愿出轨，因为他认为这不符合“我给自己规定的人设”，不是我真正想要的。所以用理智克制了本能。所有人看到帅哥美女都可能心动，但有些人能用理性克制，原因在于：**我知道我想要什么，而本能驱使的事并非我真正的发展方向**。所以他会及时制止自己。

你要明确在婚姻中“我想要什么”，以及“要成为什么样的人”。在整个过程中，完成自我探索。罗振宇曾说过：过去的婚姻是成年后第一件事，结婚才被社会认可；现在的婚姻是把自己的人生都整理好后才去做的事。这句话很有道理。

在婚姻之初如果方向错了，很难回到正轨。所以开始时就要清楚：我要做什么样的人？要在婚姻中得到什么？这些都要清楚。至少要有明确的概念，为后续行为提供正确的指导。**方向错了，再怎么努力也是错的。**

### 2. 具备处理问题的能力

第二点是具备处理婚姻问题和自身问题的能力。

之前我回答过一个问题：“为什么觉得周围没有人的婚姻是幸福的？”有一个回答特别经典：**因为大多数人难以过上幸福的人生，却把问题栽到婚姻头上**。这句话非常对，能引起共鸣。

很多时候我们把不快乐、不幸福栽到某个人身上，习惯性逃避、甩锅。人有一种技能就是甩锅，好像只要甩了锅，自己就舒坦了。谈恋爱时也常这样：明明是自己的问题，却推卸给别人，好像推卸后就能证明不是我的问题，能保持自我良好印象。

例如碰瓷的老人，网上有两种说法：道德败坏或讹钱。但有一部分实际情况是：老人摔倒，不想承认是自己身体控制不住，一定要把锅甩给别人——“是这个小伙子推了我”。把问题推卸给别人，来达到内心认可自己的状态，不接受意外。

很多人在恋爱中也是这种状态：只要发生事情，就推到对方身上，归因到某一件事上。好像这样就没那么自责难受，问题没解决，但“把提出问题的人解决掉了”。就是这么一种心理。

但为什么要逃避、甩锅？本质是不愿接受意外，包容不了生活中的意外。这是我们推卸责任的根本原因，也是自我认知的问题：没有一个强大的“本我”去接受和包容意外，永远只接受好的一面，不接受坏的一面。这其实是一种责任感的缺失。**责任感是能接受优点和美好，也一定要能接受缺点和糟糕。**

婚姻中会有各种难缠的问题。你跟一个人在一起是因为他的优点，但记住：**你是在跟一个人的缺点谈恋爱**。婚姻中不只有美好，美好被赞颂是因为它是个例；大多数情况是平淡甚至糟糕。面对这些时，你如何处理？推给别人、推给关系，你就满足了吗？问题并没解决。

所以想解决问题，首先要知道问题出在哪。很多问题解决不了，是因为无视、逃避、搁置问题。我相信每个人进入婚姻前都经历过恋爱。在我的咨询中发现：每个人在情感中犯的问题其实是一种常态，在每段感情里都会出现。但问题为什么每次都出现？因为每次他都把问题甩给伴侣、甩给这段关系，归因为“我选错了人”。我知道有问题，但我不接受，结果问题一直没被解决，最后拖到婚姻里。

**想解决问题的关键是正视问题。** 只有正视问题，知道问题在哪儿，才能去解决。

### 3. 建立平等相互尊重的关系

第三，你要学会建立平等相互尊重的关系。亲密关系的原则就是平等原则和相互尊重原则。

**平等原则** 是你我之间平等，没有谁比谁高，没有主从关系，没有“主体-客体”工具化关系。你我都是为了维护这段关系而存在，没有个体之分。如果婚姻或恋爱里没有平等，那么一定有人受委屈、不断妥协，最后这段关系只能是悲剧。

**尊重** 是什么？举个简单例子：两个人一起看电影，怎么才能看得开心？两个人都想去。如果其中一个人不想看，你硬拽他去，结果就是一个人牺牲自己满足另一个人喜好。一个人通过牺牲自己使另一个人快乐，这段关系平等吗？尊重吗？不尊重，也不平等。

所以只要是两人关系，就无法凭一个人的喜好决定。我之前在课里说过，在亲密关系中无法完全维持自我，但前提是你要知道“自我”是什么。为什么一直提“自我”？因为真正有自我，你才是内心独立且强大的人。真正内心独立强大，你才能接受生活中的种种变故和变化。因为生活中不只有开心，更多的是不开心和平淡。你用什么样的心态去面对这些平淡和问题？

真正强大的心态是：对方不想陪你看电影，你不会认为他是不爱你，仅仅是他真的不喜欢这么做。我们能用一个客观、平等、尊重的原则去看待一件事，很多矛盾就不会发生。

很多时候矛盾发生是因为我们错误解读，把内心的自卑放大、具象化，放到对方身上。我们认为“对方这样是不爱我”，因为自卑，所以对方一切拒绝我的事，我都觉得是一种伤害。把自己内心的空虚和无力具象化、现实化，这才造成很多情感问题。

情感中有大事吗？没有。我接到的案子里，最后分手离婚都没有任何大事，都是小问题、小误会不断积累，最后爆发。其实都是鸡毛蒜皮的小事。我们如何应对这些小事？需要一个平等和尊重的关系。其前提是拥有强大的自我认知和真正独立的自我，你才能真正客观看待每一件事。

当然，人不可能完全客观，听完课也不可能立刻变成那样的人，包括我自己都做不到。但就像这节课开头说的：**我们至少给自己的行为提供一个正确的规范，再按照这个规范去努力就可以了。**

### 4. 具备离开的勇气和实力

第四点是随时具备离开的勇气和实力。

有的人不敢离婚，害怕离婚后活不下去，或害怕承受道德压力，所以委屈求全。男人女人都有：害怕离婚后活不下去的多是女人；害怕道德压力、委屈求全的多是男人。

但有一个前提点：我们活着是为了什么？为了自己开心快乐，为了愉悦。你会说：老师，你这么想太自我、太自私。我觉得自私是以伤害别人为前提来为自己获利；自我是“我自我感觉良好，我让自己先舒服”，这没错。我们都不是圣人，我自己都不舒服，看谁都跟仇人一样，还怎么给别人施舍？那不叫善良，叫傻。

**所谓的付出，都是在照顾自己之后，有余力才为别人去奉献。** 有多余的力量分享给别人，这叫付出。先照顾好自己，连自己都照顾不好，怎么照顾别人？

我们怎么才能具备离开的勇气和实力？很多时候为什么放到最后讲？因为我们常发现伴侣或这段关系并不是我想要的。常说“要及时止损”，但很多人没有及时止损的勇气，他觉得“我离开对方就活不下去了”。但这仅仅是你认为，是你的主观臆断，你把主观臆断当成了客观事实。

我记得前年有个客户，是位姐姐。我跟她聊天时一直在夸她，说的也是事实：她事业做得很好，聊感情时状态特别丧，但一聊她正在做的事，眼神就泛光。她老公是渣男，多次出轨，但她就是舍不得，习惯了这个人，觉得离开后生活就空了，自己带不好孩子、生活不好、没能力完成事业。当时我跟她说：姐姐，你真的没问题，你可以，你是我见过为数不多真正非常有实力的女人。她后来真的打破关系，离婚了，现在过得非常开心。

所以很多时候，我们所认为的并不是真正的事实，而是长期以来的不自信、自卑、懦弱，或内心不够强大，难以改变一个习惯，难以跨越自己设置的未来。很多时候困难并不可怕，你认为他很困难，这才可怕。我觉得没有什么事情是离不开的，这世界没有谁离不开谁活不了，只是有时你自己这么认为。我们习惯了依赖别人，却忽略了相信自己。这个心态很重要。

但离开不仅需要勇气，还需要实力。就像刚才讲的姐姐，她是有实力离开的，只是缺乏勇气。在我给她勇气后，她大胆转身，现在有大把追求者。但前提是她有这个实力。

这个实力是什么？对于女人来说：**身材管理、兴趣爱好、工作能力、金钱**。很多女人热衷于做全职母亲，真的好吗？有没有想过，当伴侣有一天做出踏破底线的事，你用什么实力去说“不”？我觉得人永远要给自己留后路，永远要保留说“不”的能力。这个能力是：**我没有你，一样能过得非常好。** 我和你是因爱相拥依靠，但并不是共生。共生关系是“我没有你就活不下去”，就像寄生虫和人一样。你要做什么？我当然不喜欢这种共生关系，我希望每个人都是平等的。

**平等不仅仅是说说，平等需要靠实力。** 就像现在的中美关系，美国能跟中国打贸易战，是因为中国的实力让美国打不起热战；美国跟中国冷战，是因为美国清楚热战是他玩不起的游戏。这就是实力。那么你要想转身、想维护自己的感情，就需要这份实力。这份实力就是：我没有你一样可以活得非常好。

那么对于男人呢？我觉得大部分男人在要离开时，并不是实力问题（这跟中国传统观念“男人要挣钱养家”有关），大部分男人其实有实力转身，但他为什么不转身？缺乏的是勇气。很多男人抱着一种拯救对方的态度，老是期待伴侣有一天会变好。但记得我之前讲的：**人不可能被改变，只能被影响。** 当有一天你发现这个人你影响不了他的三观、行为、做法，他会继续让你在这段关系里痛苦下去，你有什么理由不能转身？

很多人害怕：我这么大岁数，都有孩子了才转身，我是不是个渣男？为什么这么在意别人的看法？你这么在意别人的看法而忽略自己的感受，那所有委屈只能自己忍受，就不要再跟别人抱怨“我有多难受”，这是你自己选的。想要自己开心，就不要怕别人说什么。我觉得这世界上别人说什么都没用，都是假的，只有我自己的感受是真的。

很多时候别人会站在道德制高点批评你，但不会设身处地理解你。张张嘴批评太简单，但去经历、去做、去感受却很难。有句话叫“未经他人苦，莫劝他人善”，但很多人不懂，只是喜欢站在道德制高点抨击。那么我为什么要做所谓的道德模范，去成为你们心目中那个人呢？很多时候我们是为自己活着的。

就像小时候父母教育我们要做一个好人，很多人就按好人标准生活，活得非常累。好人标准是什么？我觉得好人标准首先一点是：**我让自己活得舒服，我有多余的能力再为别人去奉献。** 我自己都过得不开心，所谓的好人又有多少价值？

所以无论男女：女人要有离开的勇气，也要有离开的实力；男人除了有实力，更要有转身的勇气。想保护对方是对的，但对方真的值得你这么做吗？

---

本节课中我们一起学习了维系婚姻的真正“底气”。它不依赖于外在条件，而源于内在建设：**明确自我追求、具备解决问题的能力、建立平等尊重的关系，以及保有离开的勇气与实力**。掌握这四点，你就能在动态变化的婚姻关系中，保持平衡与主动，真正成为自己情感的主人。
