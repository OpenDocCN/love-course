# 43、彭祖强肾功：017.17.易筋洗髓之垂吊 🎼

## 概述

在本节课中，我们将学习彭祖强肾功中的垂吊技巧，这是一种通过特殊绑法和使用重物来锻炼生殖器，进而达到强身健体的方法。

## 绑法步骤

以下是垂吊的绑法步骤：

1. **绑法位置**：将绳子绑在阴茎和睾丸根部的耻骨连接处。
2. **热身运动**：
    - 用双手搓揉腹沟，使其微微发热。
    - 握住根部，揉搓睾丸的正反方向。
    - 揉捏阴囊两侧的金属。
3. **卡住根部**：双手虎口卡住根部，进行各个方向的牵拉，如上下左右，各牵拉9次。
4. **旋转**：握住根部进行旋转，正反各旋转9次。
5. **导气**：吸一口气，尽量往下切拉，缓缓呼气，以导气至根部。
6. **捆绑**：
    - 左手虎口卡住根部，右手将湿毛巾从左手和耻骨之间放下。
    - 左手中指夹住湿毛巾，用右手拉到左边，打结。
    - 拉紧湿毛巾，调整绳结方向，勒紧。
    - 打结，调整绳结方向，勒紧。

## 绑法注意事项

1. 绑紧根部，确保安全。
2. 第一次拉紧时，大力伸吸气，同时收紧PCG提高，让根部收缩。
3. 绑法过程中如有不适，可调整湿毛巾或重新绑。
4. 绑法时绳结要在根部的中间，不要偏移。
5. 初期捆绑时，注意不要挤压睾丸，以免产生胀痛感。

## 垂钓实操

1. 站立姿势：两脚分开，鼻尖宽一点，双腿微弯曲，脚趾抓地，双手叉腰。
2. 摆动重物：深吸气，双腿弯曲，双手拢气吸气，从头顶百汇下到睾丸，再吸气，做三次。
3. 前后摆动：根部由于重量拉扯，吸气往前摆动。
4. 坚持时间：一般坚持320下，约10分钟。
5. 重量选择：从一公斤开始，每次增加一瓶水，逐步增加。

## 收工

1. 收工：深吸气，沁入下丹田，吸气，气入下丹田，吸气，气入下丹田。
2. 解开湿毛巾，甩动阴茎，按摩放松睾丸和阴茎。
3. 拍打身体，将气血分布到全身。
