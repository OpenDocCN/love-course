# 03.【沟通课 丨 成为说话高：1-02节2.【了解逻辑】逻辑

在本节课中，我们将要学习“逻辑”在沟通中的核心定义与重要性。理解逻辑的本质，是提升我们说话条理性和说服力的第一步。

## 概述：什么是逻辑？

逻辑到底是什么？在开始学习具体的逻辑框架之前，我们必须先搞清楚这个根本问题。只有了解了逻辑的本质，才能将其有效地贯穿到我们的日常沟通与表达之中。

与人沟通时，会说话的人会首先在大脑中快速建立逻辑，然后再进行表达，因此他们的发言自然有条理。相反，不会说话的人往往是想到哪儿说到哪儿，导致表达一片混乱。

## 感受逻辑：有与无的对比

以下是几个简单的案例，帮助我们具体感受有逻辑和没有逻辑的表达，其区别到底是什么。

**案例一：描述书柜**

假设你问朋友小猪，他的书柜里有哪些书。

*   **无逻辑的表达**：“我有一本瑞·达利欧的《原则》。我还有什么哦，还有稻盛和夫的《心》，我还有叔本华的《人生的智慧》哦，我还有本漫画西方史吧，还有一本画册，妈妈买的画册。还哦，还有本《小丸子》啊，还有朱老师出的《有趣》这本新书，还有书…”

*   **有逻辑的表达**：“我的书柜里一共有大概80多本书，主要分为心理学、商业和哲学三大类，再加上几本杂书。心理学的书有22本，其中有《XXX》。商业相关的书有30多本，里面有《YYY》。哲学类的书大概有十来本，里面有《ZZZ》。最后还有几本杂书，包括一本漫画和一本画册。”

第二种表达听起来清晰多了，因为它对信息进行了分类整理。

**案例二：分享自媒体经验**

假设有人问：“你是怎么把自媒体做起来的？”

*   **无逻辑的表达**：“嗯…这个肯定得能写东西啊。写的…写生活写的要有特点，当然也不只是写啊，还有视频的形式。嗯，我拍摄不太专业其实啊，不过我喜欢啊我喜欢的内容呢其实很多哈，做自媒体嘛，还是做我擅长的东西我这个人就擅长写啊。不过你不擅长写也别着急啊，因为先制那个…啊对了，你要找准自己的定位，你看看那些做的好的哈，都有清晰的定位。哦，对了，输出一定要规律哦。嗯，不管你擅长是什么啊，哪怕你不擅长，你不擅长怎么办呢？不擅长啊…对了，不擅长要学嘛…”

*   **有逻辑的表达**：“我认为做好自媒体分为三步。**第一步**，找准内容角度，想好自己要呈现什么内容，最好是自己擅长的话题。**第二步**，确定你擅长的传递形式，可以是文字、视频或者其他形式。**第三步**，确保持续性的输出，并根据用户的反馈和数据的表现，调整你的输出。这是我认为做好自媒体三个基本的步骤。”

第二种表达比第一种更清晰、更有条理，因为它按照步骤进行了结构化阐述。

**案例三：提出商业建议**

假设公司新开的便利店客流少，老板问你有什么建议吸引顾客。

*   **无逻辑的表达**：“老板，要不咱试试发兑换券，或者我们试试…抽奖？”

这种说法太随意，像是临时拍脑袋想出来的，缺乏说服力。

*   **有逻辑的表达**：“老板，我认为现在客流少的主要原因，是很多人不知道我们这个店，或者知道但没有动力来。我了解到一家类似的便利店，他们当时通过在附近办公楼发兑换券吸引消费者，并用朋友圈集赞鼓励传播。这样一让附近的人有了来店的动力，二让不知道店的人成了潜在消费者，效果很好。既然效果这么好，我觉得我们也可以尝试一下。”

第二种表达听起来更有说服力，因为它分析了**因果**关系（为什么做）并提供了**证据**（成功案例）。

## 逻辑的核心定义

通过以上三个案例，我们是否能感受到有逻辑的表达所带来的不同沟通效果？它更清晰、更高效，也更有说服力。

那么，怎样才能让我们的沟通更有逻辑呢？答案是：**必须先有逻辑的思考，才能有逻辑的表达**。

现在回到最初的问题：到底什么是逻辑？观察我们的案例：
*   第一个案例中，我们将书籍按照**相似性**进行了分类。
*   第二个案例中，我们按照**时间顺序**将过程分为了几步。
*   第三个案例中，我们分析了事情的**因果**关系。

逻辑还可以有其他许多种更复杂的形式。但从这些例子中，我希望大家体会到我们说话的底层逻辑是什么：**那就是把你想要表达的内容进行整理，找到它们之间的关联**。

无论是相似性、先后顺序还是因果关系，其实都是“关联”。再通俗一点说，逻辑就是**事物之间的关系**。在本课程中，我们特指**你说话内容之间的关系**。

因为人们无论是记忆、理解还是认同一件事，最好的方式就是让他们知道你所说的事物之间有什么样的关联，是什么样的来龙去脉和因果关系。这就是我对“说话逻辑”最通俗的定义。

## 过渡：从底层思维到应用框架

上一节我们探讨了逻辑的本质是建立内容间的关联。接下来，让我们进入三种非常基本且重要的底层思维模式。

这三种底层思维是基石，无论你在什么场合下说话，无论后面我们将学习到多么复杂的逻辑框架，它们都是基于这三种思路演变而来的。别看它们基础，但根据我的经验，包括我给许多企业管理者授课时发现，很多人并未真正掌握。

讲完这三种底层逻辑思维之后，我们将进入十多种实际沟通场景，看看有哪些具体的逻辑框架可以帮助我们。

## 总结

本节课中，我们一起学习了：
1.  **逻辑在沟通中的重要性**：有逻辑的表达更清晰、高效、有说服力。
2.  **逻辑的本质**：是整理说话内容，并找到内容之间的**关联**（如分类、顺序、因果）。
3.  **核心公式**：**逻辑 = 内容之间的关联**。
4.  **实现路径**：**逻辑思考 → 逻辑表达**。

理解了这个基础，后续学习具体的逻辑框架时就会轻松很多。下节课，我们将开始学习三大底层逻辑思维。
