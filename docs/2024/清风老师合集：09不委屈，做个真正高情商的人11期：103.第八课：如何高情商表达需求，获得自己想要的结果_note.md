# 清风老师合集：09不委屈，做个真正高情商的人11期：103.第八课：如何高情商表达需求，获得自己想要的结果

## 概述
在本节课中，我们将学习如何通过高情商表达需求，以获得自己想要的结果。

## 认识需求
### 为什么害怕表达需求
1. **需求是丢人的**：我们觉得需求是很丢人的，但实际上，提出需求证明你遇到了真正需要的事情，这不是丢人的。
2. **害怕被拒绝**：我们都害怕被拒绝，但拒绝是正常的。你害怕的是被拒绝后的无价值感。

### 自卑情绪
1. **自卑的表现**：
   - 经常觉得自己做得糟糕。
   - 对负面评价过度放大，全面否定自己。
   - 反复查看赞美，难以接受赞美。
2. **自卑的原因**：小时候的经历，父母的严格要求和否定。
3. **破解自卑**：建立一套对自己有效的客观评价体系。

## 建立自我评价体系
1. **人生目标**：投资智慧、社交魅力、经营的哲学。
2. **走出自卑**：通过努力践行自己的目标，建立自信。

## 高情商表达需求
### 场景一：工作
1. **自我评估**：评估工作成果和工资匹配度。
2. **明确需求**：明确涨工资的数额和职位。
3. **提出申请**：向领导提出申请，允许被拒绝。
4. **根据反馈**：根据反馈再次争取。

### 场景二：伴侣
1. **满足对方需求**：在提需求前先满足对方的需求。
2. **直接提出**：直接告诉对方你的需求。
3. **再次争取**：对方拒绝时，再次提出理由。
4. **理解对方**：如果对方无法满足需求，理解对方，因为改变需要时间。

### 场景三：父母
1. **表达感受**：说出自己的感受，例如压力太大。
2. **拉回主旨**：表达父母的爱。
3. **明确需求**：直接提出你的需求。
4. **重视需求**：爱你的人会重视和理解你的需求。

## 总结
本节课中，我们一起学习了如何通过高情商表达需求，以获得自己想要的结果。希望你能成为一个敢于提出需求、面对拒绝、在拒绝后继续提出需求的人。这是真正的高情商。
