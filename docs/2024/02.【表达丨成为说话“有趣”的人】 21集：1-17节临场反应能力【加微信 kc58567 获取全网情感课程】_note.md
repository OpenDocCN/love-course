# 课程02：成为说话“有趣”的人 🎤
## 第21集：1-17节 临场反应能力

在本节课中，我们将要学习如何锻炼临场反应能力，以便在各种场合下能够自然、有趣地应对突发状况。

许多人学习了说话技巧后，仍然面临一个实际问题：当现场发生尴尬或遇到他人突然的言辞时，他们无法迅速做出有趣的反应。这就像学会了骑自行车，但在下雪天遇到小狗时，仍然会因紧急刹车而摔倒。临场反应能力并非一蹴而就，它需要通过反复经历各种场景来培养。

上一节我们介绍了临场反应能力需要锻炼的本质，本节中我们来看看具体如何通过五个步骤来提升这项能力。

以下是锻炼临场反应能力的五个关键点：

**第一点：及时开口**
无论何种场合，都必须尝试开口说话。例如，在会议中主动发言，或是在尴尬时尝试用自嘲或故事来化解。即使最初说得不够好，也必须说出来。只有通过实际表达，才能获得反馈，从而了解何种方式有效、何种方式无效。说话能力无法仅靠倾听来完全提升。

**第二点：减少自我过滤**
我们常常在说话前犹豫，将有趣的想法过滤掉。但越是如此，越难说出有趣的话。许多有趣的说法正是在这种过滤中消失的。我们应该像孩子一样，减少这层心理过滤（当然，伤害他人的话除外）。例如，当被称赞时，可以像孩子一样直接承认，而不是过度谦虚。

**第三点：保持轻松，不反击、不辩解**
临场反应时，保持轻松心态至关重要。如果对他人的言论（例如“你这衣服真土”）进行严肃反击或辩解，对话就会变得对抗，趣味全无。正确的做法是第一时间告诉自己不要严肃，放下防卫心理，不辩解，转而运用有趣的思维方式来回应。

**第四点：培养玩耍心态**
玩耍心态意味着在对话中保持轻松、调侃和嬉戏的状态，甚至可以打破常规。例如，将对话视为游戏，赢了开心，输了也无妨，说说笑笑。在这种心态下，更容易做出有趣的反应，比如用“我人太洋气了，穿个土衣服往下拉一拉”来幽默回应批评。

**第五点：多参与社交场合**
通过参加各类社交活动，如聚会、会议、兴趣小组等，与不同的人在多样情境下交流、开玩笑、讲故事，可以逐步提升临场反应能力。锻炼说话能力不是闭门造车，而是需要在开放环境中不断实践，就像学习语言需要与母语者日常交流一样。

以上五点建议，通过持续锻炼，将有效帮助你提升临场应变能力。

本节课中我们一起学习了提升临场反应能力的五个方法：及时开口、减少过滤、保持轻松、培养玩耍心态以及多参与社交。最后，请思考在近期（如明天或下周）的某个场合（如聚会或会议）中，如何运用这些方法大胆尝试，倾听反馈，并无重复锻炼，直至能力提升。
