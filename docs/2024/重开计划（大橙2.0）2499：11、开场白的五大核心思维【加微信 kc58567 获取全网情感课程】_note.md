# 重开计划（大橙2.0）2499：11、开场白的五大核心思维

## 概述

在本节课中，我们将学习开场白的五大核心思维，这些思维将帮助你更有效地与女生进行开场白，并建立良好的第一印象。

## 五大核心思维

### 1. 时间、地点、话题、状态

**公式**：聊得来 = 时间 + 地点 + 话题 + 当下状态

- **时间**：了解女生一天的习惯，选择合适的时间进行聊天。
- **地点**：选择合适的地点进行聊天，避免距离过远或环境不适宜。
- **话题**：选择女生感兴趣的话题，避免无关话题。
- **状态**：了解女生的当下心情和恋爱欲望。

### 2. 筛选有效资源

**公式**：有效资源 = 时间 + 地点 + 话题 + 状态

- 筛选出符合以上四个条件的女生，作为聊天对象。

### 3. 不超越现有关系

- 在聊天过程中，避免说出超越现有关系的话，以免让对方感到不舒服。

### 4. 可回复性

- 选择具有可回复性的话题，让对方能够轻松回复。

### 5. 与其相关

- 回复内容要与聊天内容相关，避免让对方感到无聊。

## 总结

本节课介绍了开场白的五大核心思维，包括时间、地点、话题、状态、筛选有效资源、不超越现有关系、可回复性和与其相关。通过掌握这些思维，你可以更有效地与女生进行开场白，并建立良好的第一印象。
