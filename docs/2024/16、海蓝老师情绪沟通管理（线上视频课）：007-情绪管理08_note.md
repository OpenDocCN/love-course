# 课程十六：情绪沟通管理（线上视频课）🎭 007-情绪管理08：怕拒绝与怕被拒绝

在本节课中，我们将学习如何理解和应对“害怕拒绝别人”与“害怕被别人拒绝”这两种普遍的情绪困扰。我们将探讨其背后的心理根源，并通过一个具体案例的梳理过程，学习实用的解决方法。

## 概述：两种普遍的“害怕”

在工作和生活中，许多人担心或害怕拒绝别人，或害怕被别人拒绝。

害怕拒绝别人的人，不敢说“不”，习惯性地说“好的”。
害怕被别人拒绝的人，则尽量自己去完成所有事情，不敢开口请求帮助。

这两种心态都会产生一系列情绪和行为问题。

## 一、 “害怕”带来的情绪与行为困扰

以下是两种“害怕”心态可能引发的具体困扰：

*   **自己工作繁重，却难以拒绝额外请求**：即使手头工作很多，当领导或同事提出帮忙的请求时，内心想拒绝，话到嘴边却说不出口。因为怕引起对方不高兴，最终还是接下任务，导致自己情绪低落。
*   **害怕麻烦他人，宁愿自己硬撑**：即便面对熟人，也不好意思开口求助。有时宁愿自己崩溃，也不愿意求助于人。

无论是“说不”还是“说出一个‘帮’字”，都变得异常艰难。

## 二、 我们究竟在害怕什么？

那么，人为什么害怕拒绝或被拒绝？到底在怕什么？通过课前调研发现，无论职业与性别，这种现象普遍存在，大家的感受也非常相似。

**害怕拒绝别人**，通常是因为：
*   怕别人不高兴、不舒服。
*   怕破坏彼此关系，让关系疏远。
*   怕自己不被认可，甚至被孤立、被边缘化。

**害怕被别人拒绝**，则是因为被拒绝通常让我们认为：
*   自己做得不够好，考虑不周全。
*   对方不认可我们，意味着自己不够好、不值得、没有价值。
*   感到没面子、羞愧。

## 三、 案例呈现：娜娜的困扰

娜娜来自成都，从事人力资源管理工作。她的困扰正是害怕被拒绝和不敢拒绝别人。

**具体表现如下：**
*   **不敢拒绝**：当同事或朋友请求帮忙时，即使时间、精力或能力上做不到，她也不敢拒绝，硬着头皮接下，做的时候心里很委屈。
*   **害怕求助**：当自己需要帮助时，又怕说出请求后被拒绝，内心纠结后往往选择自己硬撑。
*   **带来的后果**：找她帮忙的人越来越多，而她不愿求助他人，与大家产生疏离感，感到孤独、被忽视，内心渴望被看见却常感失落。

娜娜希望自己能勇敢地说出“我做不到”。

**一个工作场景的具体例子：**
领导安排她主导一个额外的培训项目，她尝试拒绝，但只是面带微笑、不够坚定地说了一句“我手头事也挺多的”。领导依然将任务交给了她。接下任务后，本职工作非常繁杂，导致她情绪委屈、能量耗竭，甚至影响了对家人的耐心。

她曾尝试“温柔而坚定地拒绝”等方法，但话到嘴边又咽回去，总觉得没有力量。这种困扰占据了她工作烦恼的80%，对整个生活也造成了60%的影响。

## 四、 情绪梳理：看见内心的“小孩”

在引导下，娜娜闭上眼睛，回想最近一次害怕拒绝领导的场景。她感受到恐惧集中在身体上半部分。深入感受后，她看到了一个**躲在角落、蜷缩着、无助的自己**。

她想去陪伴和拥抱这个内在的自己。那个“小孩”告诉她，他害怕的是不被认可、不被看见。娜娜承诺会看到他、在乎他、认可他。

追溯这种无助感的根源，回到了**高中时期**。那时，她被迫与优秀的哥哥比较，放弃了擅长的文科而选择理科，学得非常痛苦。父母和老师的期望、与哥哥的比较，让她压力巨大，形成了“自己不够好”的深刻信念。

在梳理中，娜娜以现在成人的视角，对当时那个无助的“小娜娜”和比较她的老师说出了心里话：
*   **对自己**：肯定她的努力、细心和英语好的优势，告诉她每个人都有自己的路，不必复制哥哥。
*   **对老师**：坚定地表达“我和哥哥是不同的，不要拿我和哥哥比较，我有自己要走的路”。

当这些被压抑的话说出来后，那个“小娜娜”站了起来，挺直了腰杆，离开了那个被比较的环境，走向了一个有绿树、湖水、草坪和花朵的、充满阳光的地方。她露出了笑容，显得自信而有力量。

最后，娜娜引导“小娜娜”用**光**的方式，将几十年来积累的恐惧、害怕被否定等负担全部卸载掉。那个“小孩”终于被看见，对未来充满了信心。

## 五、 核心概念剖析：三个关键假设

从娜娜的案例中，我们可以梳理出三个常见的、却不合理的心理假设。

**1. 假设一：委屈自己 = 求全**
近一半的人怕拒绝别人，是担心破坏关系，于是用委屈自己来换取和谐。但委屈会发酵，导致内伤。**委屈真的能求全吗？**
*   **公式**：`委屈求全 ≈ 效果打折 + 关系隐患`
*   心存委屈时，工作效果会打折扣。
*   委屈会滋生自责、怨恨等情绪，破坏真正的亲密关系。
*   你的非语言信息（表情、姿态）会透露委屈，让他人感到你表里不一，反而成为关系障碍。

**2. 假设二：拒绝/被拒绝 = 我不够好**
“好”与“不好”是一种价值判断，与我们最深层的安全感需求相连。从小，我们就通过外界的评价来形成自我认知：事情做得好、别人满意，就是“好”；反之就是“不够好”。
*   **公式**：`(他人不满意 ∪ 被拒绝) ⇒ 自我评价“不够好”`
*   娜娜高中时期的经历，让她根深蒂固地形成了“自己不够好”的心态。因此，成年后仍将拒绝与被拒绝简单归因为“自己不够好”，导致害怕。

**3. 假设三：现在的我 = 小时候的我**
许多人的恐惧源于过去的**心理创伤**。创伤不一定是大事，可能只是一个眼神、一句评论。
*   **代码描述**：
    ```python
    if 当前场景触发类似感受:
        唤醒(被卡在过去的“内在小孩”) # 那个带着恐惧、无助感受的自我部分
        反应模式 = 沿用过去的应对方式 # 如：讨好、硬撑
    ```
*   娜娜在面对权威要求时，主导反应的是高中时代那个无助的自己。虽然身体长大了，但那个受伤的部分仍被卡在过去。

事实上，**我们已经长大了**。我们内心都有一个**成熟、智慧、有勇气的自己**。我们需要让这个部分来主导应对。

## 六、 解决方法：如何不再害怕？

上一节我们剖析了问题的根源，本节我们来看看具体的解决方法。

### （一） 解决“害怕拒绝别人”

**1. 了解根源，放下过去**
像娜娜一样，许多方法（如心理演练）无效，是因为没有触及根源——那个被锁住的、承载着过去恐惧的“内在部分”。
*   **关键动作是“看见”与“理解”**：娜娜看到了高中时代那个不容易的自己，理解并关怀了她，从而化解了那份固着的恐惧。
*   **提示**：如果自我梳理时情绪过于强烈，请停下来，这可能是需要寻求专业帮助的信号。

**2. 调用“成熟的自己”来应对**
我们内心由许多“部分”组成：
*   **受伤的部分**：承载恐惧、羞愧等记忆。
*   **管理者的部分**：如讨好、控制等，是保护我们的方式，但也有副作用。
*   **成熟的自己 (True Self)**：我们原本的样子，拥有**平静、清晰、智慧、自信、勇气、创造力与爱**。
*   **公式**：`有效应对 = 调用(成熟的自己)`
*   当娜娜剥离恐惧后，成熟的她就出现了，能清晰、坚定、平静地思考如何回应。

**3. 实践演练：如何回应领导？**
娜娜在梳理后设想如何回应领导派发新任务：
*   **步骤**：
    1.  **列出现有工作**及所需时间精力。
    2.  **清晰表达**：目前时间和精力有限，无法接受，请领导安排他人。
    3.  **提出替代方案**：完成本职工作后，愿意做一些辅助工作。
*   **心态**：平静、坚定、柔声细语，不带自责和犹豫。她预想领导会理解，自己也感到轻松。
*   **对同事**：同样，若的确没时间，就提前明确拒绝，这对双方都负责。勉强答应反而会散发负能量，损害关系和印象。

### （二） 解决“害怕被别人拒绝”

当我们梳理了害怕拒绝的根源，对于害怕被拒绝通常也会有帮助。
*   **核心认知转变**：
    1.  **被拒绝 ≠ 我不好**：这只是对方当下状态或能力所限，不一定是你的价值判断。
    2.  **被拒绝 ≠ 破坏关系**：坦诚的拒绝有时能明确边界，让关系更健康。
    3.  **适当麻烦别人，是在拉近距离**：互相求助和帮助是人际关系的粘合剂。
*   **行动指南**：做真实的自己，做利人利己的事，尊重双方边界。这样的人，反而更受欢迎。

## 七、 娜娜的收获与总结

回顾整个梳理过程，对娜娜帮助最大的三点是：
1.  **找到根源 (80%)**：看到高中时期的创伤是自己力量的源泉，看清后获得了清晰感和力量。
2.  **用成人视角应对 (15%)**：用长大了的、有能力的自己去面对问题，内在力量、自信和坚定由此而生。
3.  **对工作负责的态度 (5%)**：能做就做好，不能做就坚定拒绝，这对双方都有好处。

她感到现在的坚定与以往在脑中排练的犹豫截然不同，是一种一气呵成的、有力量的感觉。
