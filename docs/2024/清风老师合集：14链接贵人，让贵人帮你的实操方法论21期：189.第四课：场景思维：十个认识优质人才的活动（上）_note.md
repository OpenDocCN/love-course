# 清风老师合集：14链接贵人，让贵人帮你的实操方法论21期：189.第四课：场景思维：十个认识优质人才的活动（上）

## 概述
在本节课中，我们将学习如何通过不同的社交场合认识优质人才，并建立高端人脉。

## 认识优质人才的活动

### 1. 抖音
抖音不仅仅是一个内容平台或娱乐平台，它也是一个顶级的社交平台。通过抖音，你可以认识许多公众人物，包括创业者、大老板等。

**公式**：抖音 = 社交平台 + 公众人物

### 2. 上课
每年投入大量资金参加高端课程，可以筛选掉没有实力的人，并结识有经济实力和向上心的人。

**公式**：高端课程 = 经济实力 + 向上心

### 3. 参加路演
参加路演可以认识优秀的投资人，并获得宝贵的意见。

**公式**：路演 = 投资人 + 宝贵的意见

### 4. 参加艺术展和画展
参加一线城市的艺术展和画展，可以结识高端人士。

**公式**：艺术展 = 一线城市 + 高端人士

### 5. 车友聚会
参加车友聚会可以结识有资源、有量级的人士。

**公式**：车友聚会 = 资源 + 量级

## 总结
本节课中，我们学习了五个可以认识优质人才的活动。通过这些活动，你可以建立高端人脉，并实现阶层的跨越。
