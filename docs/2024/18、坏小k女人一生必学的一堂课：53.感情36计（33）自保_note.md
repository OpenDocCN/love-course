# 🎼感情36计（33）自保技巧教程

在本节课中，我们将学习感情36计中的第33个技巧——自保。

### 什么是自保？

自保，简单来说，就是保护自己，给自己留一条后路。在情感交往中，自保不仅仅是避免受到伤害，更重要的是在遇到拒绝或冷漠时，能够灵活应对，找到其他解决问题的方法。

### 自保的技巧

以下是自保的一些具体技巧：

1. **预测对方的反应**：在与异性交流时，要学会预测对方可能的回复。如果预测对方会拒绝，可以提前准备应对的话语。

2. **避免尴尬**：如果对方拒绝，提前说出对方可能拒绝的理由，可以避免尴尬。

3. **测试对方感觉**：通过预测对方可能的回复，可以测试对方对你的感觉。

4. **展现默契**：如果你能准确预测对方的想法，这表明你们之间有默契。

### 实例分析

以下是一个例子：

**场景**：你问一个异性：“你今天想不想我呀？”

**对方回答**：不想。

**你的回应**：“你这么没良心，一定没想。”

通过这样的回应，你既避免了尴尬，又测试了对方对你的感觉，同时也展现了你们之间的默契。

### 总结

本节课中，我们一起学习了感情36计中的自保技巧。通过这些技巧，我们可以在情感交往中更好地保护自己，找到解决问题的方法。希望这些技巧能够帮助你在感情的道路上更加顺利。
