# 17、海蓝老师情绪梳理个案课：033-查不出病因的病人第二课完成版_note

在本节课中，我们将通过一个名为“小娜娜”的个案，学习如何识别、感受并梳理童年创伤事件所遗留的情绪与身体感受。我们将看到，通过有意识地回顾和理解过去的经历，可以释放积压的负面情绪，达成与自我及家人的和解。

## 识别内在小孩的情绪 😔

上一节我们介绍了情绪梳理的基本概念，本节中我们来看看如何具体操作。首先，我们需要找到并识别那个承载着过去情绪的内在自我。

引导者让个案“小娜娜”去看见6岁时因妈妈车祸住院而感到孤独的自己。个案看到了那个孤独、害怕的小女孩，并产生了心疼的感受。引导者鼓励她去拥抱那个内在小孩。

以下是引导对话的关键步骤：
*   引导者询问内在小孩是否有话想说。
*   内在小孩感到“很懵”，不知道发生了什么。
*   她看到爸爸回家会哭，姥姥也常往外跑。
*   她感觉自己“被冻住了”，好像没有妈妈一样，无人专心照顾自己。

## 连接情绪与身体感受 🧘

当我们识别出情绪后，下一步是感受这些情绪在身体上的具体反应。情绪不仅存在于心里，也会以躯体感觉的形式表现出来。

个案描述了当时的身体感受：
*   **整个手臂**：特别沉重。
*   **胸部**：空荡荡的。
*   **腿部**：非常紧绷。
*   **心脏**：心跳加快，有憋闷和窒息感。

引导者让她保持呼吸，好好感受这些伴随着无助、无望情绪的身体感觉。

## 重构认知：区分“感受”与“事实” 🔄

仅仅感受情绪还不够，我们需要用当下的、成熟的视角重新审视过去的事件，区分当时的“感受”和客观的“事实”。

引导者帮助个案认识到：
1.  **当时的感受**：是深切的无助、无望和恐惧。
2.  **后来的事实**：妈妈恢复得很好，是一个医学奇迹，那段艰难的时期已经过去了。

当个案意识到“过程其实已经过去”后，身体感受开始发生变化：手臂、胸部、腿部逐渐放松，心跳变平稳，窒息感消失。引导者让她通过呼吸，将这种放松和踏实的感觉吸入身体。

## 表达未说出口的话 💬

情绪积压往往源于未被表达的需求和话语。给予内在小孩一个安全表达的机会，是疗愈的关键一步。

个案对爸爸表达了积蓄已久的话语：
*   **陈述事实**：我知道妈妈受伤严重，大家的精力都在妈妈身上。
*   **表达委屈**：我感觉没人照顾我。我上学总迟到，书皮也没人包，被老师批评，我感到很委屈。
*   **渴望关爱**：那段时间你很少笑，很少跟我说话，家里气氛阴沉，我感觉不到家的温暖，觉得自己像个没妈的孩子，特别孤独。

## 聆听与理解：换位思考 🤝

表达之后，聆听对方的立场同样重要。这能帮助我们理解事件的另一面，打破“受害者”的单一视角。

在引导下，个案听到了“爸爸”的回应：
*   **承认困难**：爸爸能力有限，面对大事压力巨大，不知如何是好。
*   **解释行为**：并非不爱你，而是精力透支，疏忽了细节（如起床晚导致迟到、不知包书皮）。
*   **表达爱与无奈**：爸爸也非常需要帮助，感到无人可依靠，但始终非常爱你。

## 达成和解与接纳 ✨

当表达被听见，理解得以建立，和解便自然发生。这带来了情绪的释放和新的认知。

个案听完爸爸的话后：
1.  **产生理解**：知道了爸爸的不容易，也很心疼他，确信爸爸是爱自己的。
2.  **表达原谅与爱**：对爸爸说“我非常爱你”，并希望自己当时能成年，帮助爸爸分担压力。
3.  **身心转变**：做完深呼吸后，感觉“轻松了很多”。

---

**本节课总结**：我们一起学习了情绪梳理的完整过程。从**识别**内在小孩的情绪，到**连接**身体感受，进而通过**重构认知**区分感受与事实，再通过**表达**与**聆听**完成与过去事件的对话，最终实现情绪的**释放**与内心的**和解**。核心在于给予过去那个受伤的自己以关注、理解和接纳，从而让卡住的情绪能量重新流动。公式可以概括为：**觉察情绪 → 感受身体 → 重构认知 → 表达倾听 → 接纳释放**。
