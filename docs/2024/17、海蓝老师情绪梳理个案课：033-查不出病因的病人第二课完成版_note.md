# 海蓝老师情绪梳理个案课：033-查不出病因的病人第二课完成版

## 概述
在本节课中，我们将通过一个案例学习如何进行情绪梳理。我们将跟随一个六岁小女孩娜娜的经历，了解她在母亲遭遇车祸后的心理变化和情绪体验。

## 案例背景
6岁的娜娜的母亲遭遇车祸住院，娜娜感到非常孤独和无助。

### 情绪体验
1. **孤独感**：娜娜在医院感到孤独，无法感受到母亲的陪伴。
2. **无助感**：娜娜不知道发生了什么，感到无助和无望。
3. **悲伤感**：娜娜看到父亲和奶奶的悲伤，感到悲伤和不知所措。

### 身体感受
1. **手臂**：娜娜感到手臂紧张。
2. **胸部**：娜娜感到胸部空荡荡。
3. **大腿**：娜娜感到大腿紧绷。
4. **心跳**：娜娜感到心跳加快，有窒息感。

## 情绪梳理过程
1. **感受身体**：娜娜感受自己的身体，体验无助和无望的感觉。
2. **回忆过去**：娜娜回忆起母亲受伤时的画面，以及自己当时的感受。
3. **理解父母**：娜娜理解父母在母亲受伤期间所承受的压力和困难。
4. **表达情感**：娜娜向父母表达自己的感受和需求。

## 情绪梳理结果
1. **放松**：娜娜感到身体和心情都放松了。
2. **理解**：娜娜理解了父母的难处，并表达了对他们的爱。
3. **成长**：娜娜在经历中成长，学会了如何面对困难和压力。

## 总结
本节课中，我们通过娜娜的案例学习了如何进行情绪梳理。通过感受身体、回忆过去、理解父母和表达情感，我们可以更好地处理自己的情绪，并学会如何面对生活中的困难和挑战。
