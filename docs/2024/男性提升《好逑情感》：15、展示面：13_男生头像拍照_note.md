# 男性提升《好逑情感》：15、展示面：13_男生头像拍照

🎼

## 概述

在本节课中，我们将学习如何为男生拍摄具有网感十足的头像照片。

## 照片上滑比例与构图

首先，将照片上滑比例调整为正方形。将人物放置在左下角，四肢伸展，戴上帽子，拉低曝光，这样可以使照片瞬间具有完感。

**公式**：照片上滑比例 = 正方形

**代码**：将手机相机的照片比例设置为方形

## 背影拍摄

采用三角构图拍摄背影，一个肩高一个肩低，转头拍摄，这样可以营造出神美感。

## 镜头放大与实况拍摄

将镜头放大1.4倍，打开实况走动拍摄，这样可以增加照片的美感。

## 假装低头喝咖啡

假装低头喝咖啡，突出下颚线，这样可以增加照片的质感。

以下是具体步骤：

1. 调整照片上滑比例为正方形。
2. 将人物放置在左下角，四肢伸展，戴上帽子，拉低曝光。
3. 采用三角构图拍摄背影，一个肩高一个肩低，转头拍摄。
4. 将镜头放大1.4倍，打开实况走动拍摄。
5. 假装低头喝咖啡，突出下颚线。
