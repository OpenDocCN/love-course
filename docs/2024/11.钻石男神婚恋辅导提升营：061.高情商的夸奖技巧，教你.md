# 11.钻石男神婚恋辅导提升营：061.高情商的夸奖技巧，教你

在本节课中，我们将学习如何有效地赞美他人，特别是如何通过高情商的沟通技巧，将夸奖说到对方心里去，从而建立良好的人际关系。

## 概述

我们都喜欢被赞美，因为这让我们感到被重视。美国哲学家威廉·詹姆斯曾指出，人类天性中最深层的本质是渴望得到别人的重视。因此，**被赞美 = 被重视**。然而，不恰当的赞美可能适得其反。本节课将介绍四个核心技巧，帮助你真诚、具体、差异化地赞美他人，并满足其最高层次的需求。

## 一、 赞美要真诚 🤝

上一节我们概述了赞美的核心是满足被重视的需求。本节中我们来看看赞美的第一个，也是最重要的原则：真诚。

赞美的基石是发自内心。不被接受的赞美不能称之为真正的赞美。生活中，许多自以为会说话的人常在赞美时“翻车”。例如，一位实习生夸赞45岁、穿着朴素的领导“裙子真好看”，或对看似疲惫的领导说“今天气色真好”，这类不基于事实的赞美只会让人感到虚伪。

心理学认为，你的内心世界是怎样的，你的外在世界就是怎样的。为了赞美而赞美，会给人留下虚伪、不可信的印象。真诚是人际交往中获取信任和好感的关键。

如果你暂时没有高情商的表达套路，那么回归真诚就是最好的策略。正如罗振宇所说：“没有任何道路通向真诚，真诚本身就是通往一切的道路。”当你由衷欣赏对方的某个优点时，就坦诚地表达你的感受，这样既生动又可信。

## 二、 赞美要具体化 📝

仅仅真诚还不够，抽象的赞美往往让人不知所云。本节我们来学习如何让赞美落到实处。

过于抽象的赞美，例如夸女生“很有气质”，在现代年轻人听来可能效果不佳，甚至可能产生反效果。只有具体化的赞美，才能直击人心，赢得真正的好感。

以下是几种具体化赞美的有效做法：

*   **借美好事物表达赞美**：用一个高价值的事物来衬托对方。例如：“你长得好像刘亦菲啊。”（前提是对方也喜欢这个明星）。
*   **借别人的嘴表达赞美**：引用他人，特别是有影响力者的积极评价。例如：“上次XX就说你唱歌好听，今天一听果然如此。”
*   **在比较中表达赞美**：通过对比来突出对方的独特。例如：“你真人比照片上好看多了。”或“你很像某个明星，但我感觉你比她更有气质。”
*   **借具体行为或物品表达赞美**：赞美对方的选择，就是在赞美其品味。例如：“你今天的色彩搭配和妆容很有高级感，时尚品味真好。”

事实证明，具体化的赞美能显著提升对方的好感度。所谓高情商，就是会说话。

## 三、 赞美要有差异化 🎭

学会了具体化赞美后，我们还需要注意避免赞美变得千篇一律。本节探讨如何通过差异化赞美脱颖而出。

使用陈词滥调或重复的赞美同样会引起反感。经济学中的“边际递减效应”在此同样适用：第一次被夸漂亮会开心，第一万次被夸漂亮就可能无感甚至厌烦。

对于一个被众多追求者环绕的美女，重复夸其外貌效果甚微。正确的做法是展示你的个性，进行差异化赞美。

*   **寻找独特切入点**：当别人都夸外貌时，你可以关注她特别的爱好、需求或内在特质。例如：“我觉得你是个特别有想法、有趣的人。”
*   **极端做法：适度批评**（需谨慎使用）：当所有人都在夸奖时，你适度的、有道理的批评反而会让她印象深刻。她会觉得“只有你能看见真实的我”。但请注意，这种方式对自身价值和情商要求极高，使用不当会彻底失败。**价值低 + 甩头 = 被淘汰**。就像领导不一定听下属建议，但下属必须听领导的一样。

差异化赞美的最高级形式，是为对方量身定制。

## 四、 赞美要体现对方最高需求 🏆

前面我们学习了赞美的形式，现在我们来探讨赞美的深度——如何满足对方更高层次的需求。

根据马斯洛需求层次理论，人的最高需求是“自我实现”。称赞外在不如称赞对方对最高需求的追求。我们每天都有被认可的需求。

以乔布斯为例，他在产品发布会上并不直接赞美电脑性能，而是赞美“富有创造力的人们能用电脑做什么”。他通过赞美人的创造力，消除了人们对技术替代的恐惧，赋予了产品更深层的意义。

在日常生活中，做到这点的有效方法是：**多鼓励，少表扬**。
*   **表扬** 针对结果和成效（如“考了100分，你真棒！”），可能带来压力，让人感觉结果无法掌控。
*   **鼓励** 针对态度和过程（如“考了100分，这是你努力的结果，继续加油！”），给人以掌控感和持续进步的动力。

伟大的音乐家莫扎特也曾表示，他更喜欢听到对其努力的鼓励，而非对其天才的泛泛表扬。人性渴望被欣赏，但欣赏不等于泛泛的表扬。

## 不同关系阶段的赞美策略 💑

掌握了四大核心技巧后，我们来看看如何在不同的人际关系阶段中灵活运用。夸赞女生并非张口就来，需根据亲密程度选择最合适的方式。

### 1. 相识阶段
此阶段过度赞美会显得虚假且有目的性，易引发对方戒备。
*   **避免**：频繁夸“漂亮”；通过自降身价（如“我不如你”）来博取好感。
*   **建议**：进行适度、轻松的外貌或性格夸奖。例如：“你的声音很温柔。”“你也喜欢XX的歌？很有品味。”“这个发型很适合你，很淑女。”

### 2. 吸引阶段
建立好感后，可通过赞美巩固和升级关系。若对方好感明显，赞美可以更大胆热情。
*   **建议**：“你笑起来好迷人。”“你好温柔，是我的理想型。”“今天的穿搭特别适合你，可爱又温柔。”
*   **注意**：若你已有所表示，对方仍若即若离，建议及时抽身。没有回应代表没有吸引。

### 3. 恋爱阶段
即使已成为恋人，赞美依然不可或缺，这是表达爱意的重要方式。
*   **建议**：把她当成宝贝，在各种情境下自然表达。例如，素颜时可夸“我家小仙女素颜也这么美”；吃饭时可夸“看你吃得香，我都饿了，真可爱”；即使她生气也可夸“小仙女生气也这么好看吗？”
*   **核心**：女生更在意你心里是否有她。赞美是爱的表达，甚至能在未来矛盾时成为“修复关系的暗号”。可以辅以亲昵的称呼，如“宝宝”、“小宝贝”。

### 4. 稳定阶段（婚后或热恋后）
感情趋于平淡时，容易忽略对方的付出。此时的赞美是关系的润滑剂。
*   **建议**：
    *   **指出进步**：“宝宝做饭进步好大！”“你居然会缝衣服，太厉害了！”
    *   **感恩付出**：“每次吃到你做的饭，就觉得一切辛苦都值得。”
    *   **增加情趣**：在足够熟悉后，可以适当说一些亲密调侃的话语来调节气氛。

## 总结

本节课我们一起学习了高情商赞美的四大核心技巧：**真诚是基石**，**具体化让赞美有力**，**差异化使你脱颖而出**，**满足最高需求让赞美走心**。同时，我们探讨了在不同关系阶段（相识、吸引、恋爱、稳定）应用这些技巧的具体策略。记住，赞美的本质是看见并尊重对方的独特价值，在适合的时间，用适合的方式，说出对方最需要听到的肯定。
