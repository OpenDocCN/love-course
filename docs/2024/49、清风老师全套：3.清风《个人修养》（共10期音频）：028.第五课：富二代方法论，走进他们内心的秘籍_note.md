## 概述
在本节课中，我们将探讨如何理解富二代的内心世界，以及与他们相处的核心策略。

## 富二代的四大类别
富二代根据原生家庭的模型，可以分为以下四个核心类别：

### 1. 暖男款
- **核心价值**：高生育价值 + 高生存价值
- **行业**：商业地产、餐饮、工业制造、新消费、新零售
- **岗位**：总裁、高管、连锁企业创始人、品牌创始人
- **核心价值观**：奋斗 + 影响 或 影响 + 精彩
- **相处策略**：
  - 初期：展示优秀的一面
  - 中期：展现柔弱
  - 后期：认可对方的责任心

### 2. 奶狗款
- **核心价值**：高声誉 + 高观赏
- **行业**：大型企业、集团公司、家族企业
- **岗位**：稳定的事业单位编制、技术人员、管培生
- **核心价值观**：安全 + 精彩 或 传统 + 精彩
- **相处策略**：
  - 初期：展现柔弱
  - 中期：展现成长和变化
  - 后期：鼓励对方找寻生活和爱好

### 3. 冷漠款
- **核心价值**：高生存 + 高观赏
- **行业**：金融、游戏、计算机、投行、律所
- **岗位**：律师、医生、职业经理人
- **核心价值观**：奋斗 + 独立 + 精彩
- **相处策略**：
  - 初期：展现独立
  - 中期：逐步展现依赖
  - 后期：探讨分享，鼓励接受亲密关系

### 4. 海王款
- **核心价值**：高观赏 + 高情绪
- **行业**：文娱、学术、咨询、新兴产业
- **岗位**：网红、艺术家、自由职业者
- **核心价值观**：精彩 + 博爱 或 博爱 + 仁慈
- **相处策略**：
  - 初期：展现丰富的生活
  - 中期：体现依赖
  - 后期：关注对方的成长
