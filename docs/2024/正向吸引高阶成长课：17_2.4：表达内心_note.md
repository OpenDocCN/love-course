# 正向吸引高阶成长课：17_2.4：表达内心 🗣️

## 概述
在本节课中，我们将学习如何清晰、有效地表达内心感受与需求。这不仅关乎日常沟通能力的提升，更在于掌握在关键时刻通过表达扭转局面的能力。我们将从底层逻辑和训练方法入手，帮助你成为更会表达的人。

## 第四节：表达内心——关键的表达力量
表达内心也可称为输出表达或关键表达。本节课程不仅学习如何在日常中提升表达能力，也学习如何在关键时刻清晰地表达自己，以起到关键的扭转作用。

在情感中，如果你不擅长表达，就等于有了一座高墙挡在你和对方之间。你有想法说不出，他想了解你又走不进来，这是非常糟糕的。好的人设故事可以将情感初期的吸引拉到很高。但是你们在一起之后，如果感情出现了问题，或者有一方的需求未被满足，又或者遇到了外部的挑战，我们该怎么办呢？

例如，对方让你失望，你不知道怎么说，感觉说出来很矫情，你只能憋在心里生闷气。结果对方还觉得是你在无理取闹，这完全是反向的效果。

再比如，你想提分手，但是想给对方一个挽回的机会。其实你并不是想分开，只是希望对方对你好一点，这该如何表达？稍有不慎，就会变成一场大型争吵，甚至被误解为PUA。

再比如，你们是姐弟恋或者门不当户不对，对方的父母很强势，不同意你们在一起。但是你并不想放弃，该如何去赢得对方父母的喜欢？

还有异地恋的情况。当对方想要你去到他的城市，而你却不想放弃自己的事业，该如何表达才能让对方觉得你是爱他的，让他可以理解你的选择，甚至愿意迁就你？

同学们，想想会表达是不是太重要了。如果你们见面时说不出口，或者不想再有情绪上的争吵，你也可以选择用小作文或电话的方式去表达。

虽然我们这节课最后会有关于朋友圈文案、情感小作文或电话沟通的教学，但这只不过是我们不同的运用场景。我们更需要的是掌握一套完整的底层逻辑，以及表达的训练方法。

会表达内心的人，可以将他此时此刻内心的感受和需求表达得非常清楚，让对方准确地接收到所有想表达的信息。甚至我们可以用表达的方式去感染对方，去感动对方，去快速拉近关系的距离。

当然，好的表达也可以遮盖我们心理上的不足或行为上的犯错，让对方原本高高在上的姿态一下子软弱下来，让我们可以化被动为主动。

同学们还记得在前面的时候，老师说过这样一句话：聪明的女人都很擅长言胜于行，用语言去征服男人，用男人去征服天下。这并不是说女人要靠男人怎样，而是说这是一项很高情商的社会技能。

所以，如何表达内心，如何快速提升自己的表达能力，如何输出大量的情绪信息，让情绪价值成为我们在情感中交换的筹码，成为我们在感情中进退自如的资本，这便是本节课学习的重点。

## 两个核心成长公式
那么首先，我们要记住两个黄金公式，一个是关于成长的，一个是关于情感的。如果你能够完完全全掌握和理解这两个公式，那么你的人生一定可以开挂。

### 公式一：快速成长
**`快速成长 = 输入 + 深度思考 + 输出`**

如果你只从外界输入，从来不对外输出，很容易内耗，憋出内伤。如果你只输出从来不输入，内心会非常空洞，很容易成为别人眼里的笑话。快速的成长方式就是输出略大于输入，锻炼的是我们的思考转化能力。

### 公式二：情绪价值
**`情绪价值 = 倾听 + 同感共情 + 正向反馈`**

如果你在聊天时只是在那倾听，这只能叫做情绪的垃圾桶。另外，如果你的反馈不是建立在倾听的基础上，那对方也很难从你这汲取到跟他有关的正向价值，也就很难对你上头。所以，不管在亲密关系、社会关系还是客户关系里，在倾听后给予正向的反馈，一定是推进关系的关键步骤。

## 深入学习第一个公式：如何通过输入与输出快速成长
上一节我们介绍了两个核心公式，本节中我们来看看如何具体应用第一个公式来实现快速成长。

当我们还是一个初级倾听者的时候，一般情况下输入和输出的比例大概是 **7:3**，也就是要做到多听少说。这个阶段我们的内心是匮乏的，总会说不好、说错话。这个时候一定要去控制自己的表达欲。

以下是这个阶段需要注意的要点：
*   **找准定位**：在向上社交的场合里，一定要对自己有清晰的认知，找准自己的定位，虚心倾听。
*   **非语言沟通**：微笑注视对方，点头表达尊重，呈现出一副渴望学习和成长的样子。这样大家都会很喜欢你。

当我们已经成为了一个初级的表达者，大概日常生活中的输入输出比例在 **5:5**，也就是能够很好地传递信息，准确无误，是一个合格的信息搬运工。

老师我在10年前就开始做上市公司的总裁助理，公司每年几百亿的规模，那个总裁的能量非常大。他说话做事也非常严谨。我跟在他身边的时候，每天如履薄冰，很多时候不敢说话，不敢表达。领导说话，我也只能洗耳恭听。但是至少我勤动手，勤动笔，领导说的每句话我都会记下来。每一次会议纪要，我都尽量准确地记录领导的工作指令，并且在会后去督办到位。

就这样跟了总裁半年之后，我开始慢慢有了一层管理者的视角，开始学着领导去讲话、去说话、去做事，也开始慢慢越来越像他了。我就从一个信息的搬运工慢慢变成了一个更加成熟的表达者，也就进入了我们成长的第三个阶段，输入输出的比例在 **3:7**，然后就不停地在社会中去磨练自己的嘴皮。

所以，选择一个好的领导、老师和贵人，或者说伴侣，非常重要。因为你跟在他身边就是最好的输入。

除此以外，我们还需要去输入哪些信息和知识呢？当然还是要多看书，多去拓宽你的知识面，丰富你的词汇量，让你的储备充足起来。

以下是推荐的输入方式：
*   **多元化吸收**：看电影、看语言综艺节目，以及有深度的纪录片。
*   **向高人学习**：多去听音乐、听课，听比自己认知高的人讲话。
*   **积累素材**：可以把自己看到的、听到的一些经典的句子、经典的台词或者经典的故事背诵下来。下回你在聊天的时候就可以活学活用，举一反三，会让对方觉得你是一个爱学习、爱思考、有想法、有内涵的人。

如果你感觉自己每天的大脑都在被充实，你的认知都在被挑战，至少输入这个步骤是没有问题的。

### 第二步：深度思考
接下来是深度思考环节。大家只需要记住一个黄金圈法则，就是通过寻找问题的本质来锻炼自己的思考能力。

任何决定和事情都不要先想“做什么”、“怎么做”，而是要多思考“为什么”、“凭什么”。大众思维都是在想表面那一圈的“做什么”，当碰到了问题，才有可能去早补去思考“为什么要这样做”。典型的就是手脚勤快，脑子懒惰。这种方式会让你兜兜转转又回到原点，一切都是徒劳。

大部分在感情中被动的人也是这样，从来不去表达自己的想法，不表达自己的需求，就等着看对方怎么做。对方追你，你也接受；对方后撤了，你就落空。完全没有自己的一个节奏，这是不行的。这种情况下，你最后也只可能是空欢喜一场，你也只能天天在那抱怨自己遇人不淑。

真的喜欢深度思考的人，他一定是有目标感、有计划的，知道自己要什么、为什么需要，然后知道自己该怎么做才能够达到目标。所以某种意义上讲已经达到了一种自我觉醒的状态，是需要我们去花时间静下心来思考的。

---

### 第三步：输出
最后一步是输出。首先我们要打开自己，多去走走，多去表达，敢于表达。一逮到机会就要说话，在说话中学习说话。如果你跟不熟的人说话让你感到紧张，说明你正在这种不舒适的环境中积累经验和快速地成长。

以下是几种有效的输出练习方法：
*   **自我录音**：你可以就一个话题去随便说个两分钟，然后录下来，听听自己讲的到底怎么样。刚开始你可能会觉得自己的声音和表达很不舒服、很不自然。正因为如此，你才会不断地去纠正、去改善，久而久之就肯定会有进步。当然，我们在语音聊天的时候，也可以反复回听自己的声音，看看自己有没有说清楚，情绪是不是表达到位了。
*   **写作练习**：如果你没有一个固定的交流对象，那么我们也可以练习表达，那就是通过写作。既然我们每天都会有输入，那么每天也需要及时的输出。比如每天去写日记、写总结，养成一种自我反思的习惯。写备忘录时，要列好序号，逻辑清晰，做好笔记和碎片化的知识积累。
*   **两个宝藏习惯**：
    1.  **写影评**：可以先自己写，然后再去看豆瓣，看看别人是怎么表达的。
    2.  **写乐评**：在听音乐的时候，写下自己真实的感受，再去看那些高赞的评论。
*   **成为博主**：最后一个是很真诚的建议，建议大家有时间的话，一定要去做一个博主。哪怕最后粉丝不多，但是这是一个很好的练习输出表达的过程。博主是最需要去表达内心的，最需要去呈现真实的。而且通过网络分享观点，这种公开的环境会给自己带来一定的压力，适当的压力会更加激发出很强烈的表达欲望。如果有的同学已经是某些领域的专家，完全可以多去分享一些这些专业类的科普知识内容，练习表达的同时，还可以给自己带来客户和收入，一举两得，何乐而不为。

## 深入学习第二个公式：如何提供情绪价值
快速成长之后，我们来看第二个公式：如何提供情绪价值。

首先就是耐心的倾听，认真地注视对方，要用眼神去正面的回应。然后呢就是同感共情了。这个步骤需要我们去放弃自己的视角，跳出一个上帝视角，暂时忽略因为听到对方的话而影响到的自我情绪，才有可能说去带入对方的角色去感同身受，再去替对方描述出他的内心所想，甚至你比对方的情绪波动还要大。

比如，比对方体会到的快乐还要快乐，比对方感受到的痛苦还要痛苦。那对方的内心就会觉得“哇，你好懂我啊”，跟你在一起特别特别的舒服。等对方的心情愉悦了，说明你的情绪价值已经提供到位了。

有同学问过老师这样一个问题：老师，为什么我在工作中还挺能讲，在感情里面却不善表达？同学们，这很正常。因为我们在日常工作中的表达是去情绪化的，它更注重的是逻辑要清晰；而情感中的表达呢，它更需要感染力、共情力还有抒情。

## 不同场景的表达应用
我们可以先来学习一下朋友圈的文案写法，去感受一下不同维度的差别。

很多同学在发朋友圈的时候，经常会问老师怎么去配文案。其实呢，不是你不会配，是你太不愿意思考。如果你好好梳理过自己的三观，有一个明确的故事人设，再具体落实到怎么去配文案，那就太简单了。

同学们还记得我们在第二节“三观碰撞”里面学习的四个人生方向吗？高度、深度、温度和宽度。同样，朋友圈就是我们人生方向的最好体现。文案呢也是我们三观的缩影，是最好体现我们人格魅力的广告位。

以下是针对不同人生方向的文案写作建议：
*   **凸显人生高度**：我们可以强调自己的外在和内在的价值很高，我们的地位成就，还可以去转发一些链接来表达明确的正向价值观。
    > 例如文案：“真正的价值，在于创造价值本身。”
*   **凸显语言深度**：需要去重逻辑，多用一些专业的词语，看起来深奥又高级，也可以多用一些“虽然…但是…”、“不仅…还是…”这样的关联词，多用排比、对比和转折。
    > 例如文案：“逻辑是骨架，情感是血肉，二者兼备，表达才有生命。”
*   **凸显生活广度**：那就要涉猎更多的兴趣爱好，多去探索诗和远方。你的文字呢就需要写实一点。把我们今天去了哪里，做了什么，感受到什么都可以记录下来。记得多用动词，少用形容词，再可以配上精美的图片或视频。
    > 例如文案：“今天徒步穿越了山谷，每一步都踏在松软的泥土上，听见风穿过树林的声音。”
*   **凸显情绪温度**：哪怕我们只是想立一个人设，没有关系，我们可以多去表达快乐、自由、浪漫、潇洒这些正向的能量，去做到正向的吸引。那么这些文字呢就需要用优美的、抽象的、有意境的。
    > 例如文案：“愿你我都能活成一束光，温暖而不炙热。”

朋友圈文案相对来说还是比较简短的。但有的时候我们还是要去锻炼长句的表达，毕竟长句还是非常考验你的情感递进和逻辑的。

### 情感小作文的写作
我们在还不太会说之前，可以先练习去写，毕竟写的时候，你可以反复斟酌。比如说像小作文。

大家不要觉得男人一看到小作文就会头疼。其实更让人头疼的是你疯狂的打电话，是你连着几条的60秒语音，是与你见面后的无休止的争吵。老师我一直坚信，小作文是能够帮助我们在关键的时候去扳回一局的。

小作文不要天天写，要在关键的时候写。只要你写得好，就算对方再忙、再不想理你，他也会用心地看完。实际的情感辅导过程当中，我几乎所有的同学都需要用到小作文，不管是在暧昧期，还是说遇到了问题，或者说想要去挽回，都需要用一段这样的文字来进行破冰。哪怕只是把双方的关系拉回到一个能正常聊天的程度，那已经很厉害了。

那么，如何去写情感的小作文，如何写一段让对方愿意看完，并且非常容易产生共鸣的小作文呢？这里有一个经典的公式，大家可以记一下，叫做 **“情-理-路”**：动之以情，晓之以理，让之以路。

以下是“情-理-路”公式的具体步骤：
1.  **情**：表达认同、同感共情、理解、感恩、赞美或者示弱等等。让对方以一种愉悦平和的心态看下去。
2.  **理**：抛出问题，列举观点，有理有据，描述细节。一定要降低自己的主观情绪。
3.  **路**：提出解决问题的思路，细致地表达诉求，去营造一个美好的愿景。也可以叫“画饼”，让对方去做决定，比如说二选一。这样让对方觉得你说的不只是废话，可以进行有价值的回复。所以他基本上看了就一定会回复你。

### 小作文应用案例
我们现在来看两个案例。

**案例一：想分手但给对方挽回机会**
这种情况一般发生在你想后撤，但是对方对你还有感情的时候。这种小作文该怎么写？
1.  **第一步用“情”**：树立一个深情的人设，感恩过去，感恩相遇，感恩我们之间发生的一切。
2.  **第二步用“理”（陈述对方）**：客观陈述对方的过错，千万不要再去争吵那些事情，不要再去指责、批评和否定对方，不要再用主观的情绪去表达。记住，这里要用“理”，而不是“情”。用“情”的是第一步，但那个情也不是负面的争吵，而是正面的情绪。
3.  **第三步还是“理”（陈述自己）**：表达自己在感情里面的投入和付出，让对方感受到自责和内疚。客观陈述就行，不需要加入太多自我感动的东西。
4.  **第四步就是“路”**：首先给出我的选择，也就是第一条路：虽然我不舍，但是我坚决要离开你。其次就是让步，再给对方留一条路：我写了这么多，其实是想等你的答复。我虽然已经下定决心放弃你了，但是最后的决定权其实在你的手里。如果你能够清晰地意识到之前的过错，并且做出改变，我们也许还有机会。等你答复。

**案例二：对方父母不同意在一起**
这种情况下，一般有几个原因：要么是你的家境不好，要么是你的年龄太大，要么是你性格太强势，要么是你职业不稳定。这些跟对方父母原本的期望相差都太大。

如果对方的父母是社会圈层比较高的，他们一般都会说一不二。你在一开始就需要树立一个价值感高的形象。如果对方只是一对普通的父母，那么你的情感的打点只有一个，就是要认同他们的儿子。在普通父母的眼里，儿子大概率都是全家人的骄傲，他们为人父母自己有强烈的想要被认同的欲望，想图名声、图面子、图功劳，比如说教育得好。

这种情况下的小作文结构：
1.  **第一步是“情”**：首先要肯定伴侣的优秀，比如孝顺，也就是肯定了对方父母的教育。另外还要肯定对方父母不接纳自己的合理性，让他们觉得你还是挺懂事的。人都是渴望被认同被看见的，所以自然也就降低了对方的攻击性，打破了原本的对立面。
2.  **第二步还是“情”**：通过示弱来争取到同情，降低对方的防备意识，让对方也能够理解你的痛苦和不容易。
3.  **第三步是“理”**：如何把自己的缺点转化为优点，让缺点合理化。比如：虽然我家境不好，但是我比较踏实，能吃苦；虽然我年龄大，但是我性格比较好，性情比较稳定，在情绪上能照顾对方；虽然我的性格比较强势，但是我能够在家里顶起半边天；虽然我的职业不稳定，但是他可能还是很有前景的，只是想再拼过几年。
4.  **最后一步是“路”**：可以让路，把最后的决定权交给对方的父母。毕竟他们那一辈的人还是比较想拥有权威感的，只要不要强加意识给他们，相信两位长辈还是会心软同意你们在一起的。

### 电话沟通与课后作业
电话抒情其实跟小作文是一样的，大家一定要牢牢记住“情-理-路”这样的框架。在打电话的时候，可以选择先去倾听对方的声音，感受对方的情绪，然后安静下来。先对对方的情绪和行为表达理解，再去陈述自己的道理，最后给出对方选择。这种情况，对方一般也很难拒绝你的心意。

这里给大家留三个课后作业，请运用所学进行练习：

**作业一：异地恋沟通**
如果对方想让你去到他的城市，而你却不愿意，希望他能够迁就你多一点、爱你多一点，该怎么表达？
> **提示**：第一步肯定还是要表达理解对方的需求；第二步是解释自己的困难，合理化自身的行为；第三步是找到双方共同的目标，营造一个美好的愿景，让对方相信你还是很憧憬未来的。

**作业二：推进暧昧关系**
假设你在暧昧阶段就跟一个男生发生了关系，但是对方并没有把你当成是女朋友。同样也没有放弃这段关系，还在日常的维护。你该怎么去推进关系的确立？这是不是很多来咨询的同学们经常会遇到的情况？
> **给个建议**：可以先表达“我知道你还没有做好恋爱的准备，只是因为我很喜欢你，才愿意突破自己的原则。”然后可以说“我身边也有很多的追求者，所以我等不了你太久。”最后是退一步还是进一步，决定权在你手里。

**作业三：体面说分手**
如何体面地说分手，分手后还是希望对方能够继续给你生活或事业上的帮助？按照“情-理-路”的公式可以怎么说？
> **老师来举个例子**：“我不是很想提分手，分手是很难过的，但是分手好像对我们双方来说都是更好的选择。而且分手并不代表关系的结束，而是意味着关系的转化。我们依然可以是很好的朋友。以后有机会了，也许还会有缘再见，那个时候我们都已经成为了更好的自己了，不是吗？所以，是继续消耗彼此，还是走一条新的路，你有什么建议？”
