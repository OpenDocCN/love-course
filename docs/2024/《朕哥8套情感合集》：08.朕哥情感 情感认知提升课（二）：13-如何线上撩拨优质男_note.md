# 《朕哥8套情感合集》：08.朕哥情感 情感认知提升课（二）：13-如何线上撩拨优质男

🎼如何线上撩拨优质男。线上沟通时，制造反差感是关键。例如，形象御姐风格的人可以发可爱表情包，与形象形成反差，容易打动男人。

**核心概念**：制造反差感

以下是提升线上吸引力的方法：

1. **利用表情包**：收集并使用表情包，增加聊天趣味性，拉近与男人的距离。
2. **掌握聊天主动权**：避免让男人主导聊天，学会在聊天高潮时转移话题或结束聊天，运用“高价值离场”技巧。
3. **学会发语音**：自然地发送语音，尤其是声音好听的女性，可以发送娇滴滴但正经的语音，增加吸引力。
4. **不经意间发美照**：展示颜值，刺激男人的听觉和视觉神经，拉近彼此距离。
5. **掌握聊天节奏**：避免过冷、过热或秒回，保持自然轻松的聊天氛围。

**核心概念**：高价值离场、聊天节奏

**总结**：本节课中，我们学习了如何在线上吸引优质男。通过制造反差感、掌握聊天主动权、学会发语音、不经意间发美照以及掌握聊天节奏，我们可以提升自己在男人心中的吸引力。
