# 课程03-02：了解逻辑的本质 🧠

在本节课中，我们将要学习“逻辑”到底是什么。只有理解了逻辑的本质，才能将其有效地运用到我们的沟通与表达中，使我们的说话更有条理、更具说服力。

## 逻辑在沟通中的重要性

在与人沟通时，会说话的人会首先在大脑中快速建立逻辑，然后再进行表达，这样自然就会条理清晰。相反，不会说话的人往往是想到哪说到哪，导致表达混乱。

## 什么是逻辑？

在深入学习后续的二十多种逻辑框架之前，我们必须先搞清楚逻辑的本质。这个概念至关重要，理解透彻后，后续内容的学习和应用都会轻松很多。

让我们通过几个具体案例，来感受一下有逻辑和没有逻辑的表达，其区别到底是什么。

以下是第一个案例：

*   **无逻辑的表达**：当被问及书柜里有哪些书时，回答是：“我有一本瑞·达利欧的《原则》，哦，还有稻盛和夫的《心》，我还有叔本华的《人生的智慧》，哦，还有本漫画西方史吧，还有一本画册，嗯……还有本《小丸子》，还有朱老师出的《有趣》这本新书，还有书……”
*   **有逻辑的表达**：“我的书柜里大概有80多本书，主要分为心理学、商业和哲学三大类，再加上几本杂书。其中，心理学的书有22本，包括……；商业相关的书有30多本，包括……；哲学类的书大概有十来本，包括……；最后还有几本杂书，包括……”

显然，第二种表达听起来清晰得多。

上一节我们通过书籍分类的例子看到了逻辑带来的清晰感，本节中我们来看看另一个场景。

以下是第二个案例：

*   **无逻辑的表达**：当被问及如何做自媒体时，回答是：“嗯…肯定得能写东西啊，写得要有特点。当然也不只是写啊，还有视频的形式。我拍摄不太专业，不过我喜欢…我喜欢的内容其实很多。做自媒体嘛，还是做我擅长的东西，我这个人就擅长写。不过你不擅长写也别着急，因为…对了，你要找准自己的定位，看看那些做得好的，都有清晰的定位。哦，对了，输出一定要规律。嗯，不管你擅长是什么，哪怕你不擅长…不擅长要学嘛…”
*   **有逻辑的表达**：“我认为做好自媒体分为三步：**第一步**，找准内容角度，想好自己要呈现什么内容，最好是自己擅长的话题。**第二步**，确定你擅长的传递形式，可以是文字、视频或其他形式。**第三步**，确保持续性的输出，并根据用户反馈和数据表现进行调整。”

第二种表达比第一种更清晰、更有条理。

我们已看到逻辑如何让描述和计划变得清晰，接下来，我们看看逻辑在提出建议时如何增强说服力。

以下是第三个案例：

*   **无逻辑的表达**：当被老板问及如何提升新便利店客流时，直接回答：“老板，要不咱试试发优惠券，或者我们试试抽奖？”
*   **有逻辑的表达**：“老板，我认为现在客流少的主要原因，是很多人不知道我们店，或者知道但缺乏来店的动力。我了解到一家类似的便利店，他们通过在附近办公楼发兑换券吸引消费者，并用朋友圈集赞鼓励传播，效果很好。既然这个方法有效，我觉得我们也可以尝试。”

第二种表达听起来更有说服力。

## 逻辑的本质是什么？

三个简单的例子讲完了，你是否感受到了有逻辑的表达所带来的不同沟通效果？它更清晰、更高效，也更有说服力。

那么，怎样才能让我们的沟通更有逻辑呢？答案是：**必须先有逻辑的思考，才能有逻辑的表达**。

回到最初的问题：到底什么是逻辑？让我们复盘一下案例：

1.  在第一个案例中，我们将多本书按照**相似性**分为了几类。
2.  在第二个案例中，我们按照**时间顺序**把做自媒体分为了几步。
3.  在第三个案例中，我们分析了事情的**因果关系**。

逻辑当然还有其他许多种，也可以更加复杂。但通过这些例子，我希望大家体会到我们说话的底层逻辑是什么：**那就是把你想要表达的内容进行整理，找到它们之间的关联**。

无论是相似性、先后顺序还是因果关系，其实都是“关联”。再通俗一点说，逻辑就是**事物之间的关系**。在本课程中，特指你说话内容之间的关系。

因为人们无论是记忆、理解还是认同，最好的方式就是让他们知道你所说的事物之间有什么样的关联，是什么样的来龙去脉和因果关系。这就是我对说话逻辑最通俗的定义。

## 课程总结与预告

本节课中，我们一起学习了逻辑的本质——即**整理内容并找到其内在关联**。我们通过三个案例对比，直观地感受了有逻辑的表达在清晰度、条理性和说服力上的优势。

接下来，我将先带领大家进入三大非常基本且重要的底层思维。无论你在什么场合，无论后面讲到多么复杂的逻辑框架，它们都是基于这三种底层思路演变而来的。请不要因为它们基础而轻视，根据我的经验，很多企业的中高层管理者也未必完全掌握。

讲完这三种底层逻辑思维之后，我们再进入十多种实际场景，看看有哪些逻辑框架可以帮助我们有效表达。
