# 关系博弈（拒绝卑微和低位）：07_第七，如何通过博弈，让对方离不开你？

## 概述
在本节课中，我们将探讨如何在关系中通过博弈策略，让对方更加珍惜和离不开你。

## 关键概念
- **博弈策略**：通过一系列的行为和决策，影响对方的认知和行为，从而实现自己的目标。
- **低位卑微**：在关系中处于弱势地位，感受被忽视和不被珍惜。

## 问题分析
你是否经常感到在关系中处于低位卑微的位置，你的伴侣似乎对你的感受漠不关心，不珍惜你的存在。这种被忽视的感受，常常令你痛苦不已。然而，你又难以割舍这段关系。

## 解决方案
### 第一步：让伴侣意识到失去你的风险
1. **增加伴侣对失去你的风险感知**：
   - **方案一**：让伴侣的亲朋好友都认识你，当分开时，对方会承担被他人指指点点询问的风险。
   - **方案二**：创造属于两个人的专属体验，当分开时，伴侣会承担只能依靠回忆而没有真实体验存在的风险。
   - **方案三**：融入对方的习惯，当分开时，伴侣会承担习惯中没有你的痛苦。

2. **增加伴侣对失去你的可能性感知**：
   - **方案一**：资金合并，当分开时，伴侣会承担资金损失的风险。
   - **方案二**：保持独立性，能自己养活自己，让伴侣认识到，你不是离开了，他就活不了的。
   - **方案三**：有自己的独立时间，给自己一些独立的空间和时间，展开自己的兴趣爱好，与朋友相聚。

### 第二步：改变关系中的动态
通过上述解决办法，你可以增加伴侣珍惜你的可能性，改变关系中的动态，获得更多的关注和重视。

## 总结
本节课中我们一起学习了如何通过博弈策略，让对方离不开你。通过增加伴侣对失去你的风险感知和可能性感知，以及改变关系中的动态，你可以让对方更加珍惜和离不开你。
