# 长期疫情时代下的网聊流程 📱

## 概述
在本节课中，我们将探讨在长期疫情时代下，如何进行有效的网络聊天。我们将分析聊天流程、互动方法以及如何根据不同情况调整策略。

## 一、疫情背景下的聊天环境
### 1. 大学未开学
- 学生群体主要活跃在网络上，进行直播等活动。
- 白领群体失业或未开工，生活节奏放慢。

### 2. 女生推荐软件
- 推荐使用社交软件进行聊天。
- 不推荐使用搭讪软件和社交圈。

### 3. 朋友圈观察
- 由于疫情，朋友圈内容普遍平和，缺乏新鲜话题。
- 通过朋友圈内容判断女生级别和开放程度。

## 二、聊天预设
### 1. 基本分数和开放程度
- 根据朋友圈内容判断女生级别和开放程度。
- 设定基础聊天预设。

### 2. 聊天空间
- 疫情期间，聊天空间相对较大。
- 女生空闲时间较多，更容易进行聊天。

## 三、互动方法
### 1. 高价值展示
- 在聊天中展示自己的高价值，吸引女生注意。
- 避免谈论日常琐事，如吃饭、工作等。

### 2. 判断兴趣
- 通过女生主动给予的兴趣或升高关系来判断。
- 避免盲目追求，降低跪舔风险。

### 3. 无兴趣处理
- 若女生无兴趣，可切换项目邀约，如轰趴、喝酒等。
- 避免直接放弃，保持联系。

### 4. 有兴趣处理
- 升高情感，进入约会流程。
- 可以尝试各种话题，如游戏、网红、网络公司等。

## 四、现实见面环节
### 1. 第一印象
- 长相帅气占75%，聊天技巧或态度占25%。
- 避免过度依赖物质条件，如豪车等。

### 2. 互动环节
- 展现需求和价值互换，明确双方期望。
- 避免因需求不匹配而导致的误会。

## 总结
本节课中，我们学习了在长期疫情时代下如何进行有效的网络聊天。通过了解疫情背景、设定聊天预设、掌握互动方法以及应对现实见面环节，我们可以提高聊天成功率，找到心仪的对象。
