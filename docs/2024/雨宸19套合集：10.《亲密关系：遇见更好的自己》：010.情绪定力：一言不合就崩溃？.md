# 雨宸19套合集：10.《亲密关系：遇见更好的自己》：010.情绪定力：一言不合就崩溃？

在本节课中，我们将要学习如何管理自己的情绪，培养情绪定力。很多人容易情绪化，遇到挫折就崩溃，这严重影响生活和工作。本节将探讨情绪化的原因，并提供培养情绪定力的具体方法。

## 概述

情绪化是许多人面临的共同挑战。本节旨在帮助你理解情绪化的本质，停止自我责备，并学会通过有效的方法管理情绪，从而建立更稳定、健康的心理状态。

## 第一点：停止自我责备，正确认识情绪

上一节我们介绍了课程主题，本节中我们来看看如何正确看待情绪。

首先，每个人对情绪的感受力存在天然差异。有的人情绪起伏较大，有的人则比较平稳。这种差异是正常的性格特质，情绪化不一定完全是坏事。情绪化意味着你对世界有独特的感知力。

随着人的成长和成熟，情绪管理能力自然会提升。6岁、16岁和26岁的人，其情绪管理能力完全不同。经历增多后，能力也会增强。不要把情绪化单纯视为缺点。人其实没有绝对的优缺点，所谓优缺点只是他人眼中的看法。

社交的前提是做真实的自己。为了讨好他人或维护关系而伪装自己，这种伪装终有一天会暴露。暴露后，关系也难以维持。真实地做自己，反而能吸引那些能接纳真实你的人。因此，他人不接受你的情绪化是他的问题，不是你的问题。你只需要做自己，并选择能接受你的人。

不要一味责备自己情绪化是一种缺陷。很多时候，情绪化是一个人最真诚的表现，也是你独特的特质。坚持做真实的自己。

## 第二点：探寻情绪化的原因

理解了情绪的本质后，我们来看看情绪化通常有哪些具体原因。

以下是情绪化的几个常见原因：

1.  **习得性无助**：从小没有学会正确处理负面情绪，导致负面情绪被压抑和否定。积累到一定程度，自身无法承受且无法表达时，就容易突然崩溃。例如，小时候向父母表达诉求时，经常被打压且不被允许解释，就会形成“解释了也没用”的无助感。这种无助感习得后，成年后便不知如何正确表达，只能通过崩溃来引起注意，就像婴儿用哭闹表达需求一样。

2.  **挫折-侵犯模型**：当目标受阻（挫折）时，可能产生侵犯行为。例如，向父母表达诉求（如身体不适不想上学）却被打击驳回。长期诉求不被理解、遭受打击，会导致两种状态：
    *   **向外侵犯**：直接对施加挫折的对象（如父母）爆发愤怒。有些人长大后只对特定对象（如母亲）易怒，就是对童年积压情绪的回馈。
    *   **替代侵犯**：不敢对原始对象（如父母）发泄，转而将情绪发泄到伴侣或亲近的朋友身上。你会发现，当关系变得亲密时，对方可能突然变得情绪化，这可能是你在替他承担其内心深处对父母的不满。
    *   **向内侵犯**：情绪被极度压抑，无法表达，可能导致通过自我伤害（如自杀）的方式来释放压力。

3.  **社会化程度不足**：情绪化的人，往往有情绪化的“资本”，即有人愿意为他的情绪买单，允许他继续像孩子一样发泄。例如，一些企业老板随着地位升高，脾气可能越来越大，因为身边人都顺从他。社会化的过程是学习调整自己以适应社会，包括管理情绪。当一个人意识到情绪化无人买单、毫无用处时，就会开始学习用理性控制情绪。反之，如果一直被纵容，就会养成情绪化的习惯。

## 第三点：如何培养情绪定力 🛡️

分析了原因，接下来我们探讨培养情绪定力的具体方法。

以下是培养情绪定力的几个关键步骤：

1.  **提升情绪觉察能力**：很多人无法管理情绪，首先是因为不清楚自己当下的情绪是什么以及为何产生。解决方法是事后反思自己的情绪和行为，当下次类似情绪出现时，就能有意识地控制。例如，曾因客户质疑而暴怒，通过反思发现，愤怒源于联想到童年时被父亲质疑的羞辱感。意识到这一点后，就能区分当下与过去，避免为过去买单。公式可以表示为：**情绪觉察 -> 识别触发点 -> 区分当下与过去 -> 有意识控制**。

2.  **锻炼情绪表达能力，打破习得性无助**：要释放被压抑的侵犯性，可以采用“替代侵犯”的合理方式，例如跑步、摔打枕头等，将超负荷情绪释放出来。然后，尝试向对方表达你的感受，争取理解。多次练习后，就能用更平和的方式处理事情。表达情绪时，应遵循“描述感受-解释原因-提出期望”的三步法，而不是直接发泄怒火。代码如下：
    ```python
    # 错误的情绪发泄
    直接怒吼：“你简直不可理喻！”

    # 正确的情绪表达
    描述感受：“我现在感到非常愤怒和受伤。”
    解释原因：“因为你刚才在大家面前否定我的努力，这让我觉得不被尊重。”
    提出期望：“我希望下次你可以私下跟我沟通你的不同意见。”
    ```

3.  **学会合理释放情绪**：培养健康的爱好来释放情绪，例如运动（武术、跑步）、射击、射箭等带有一定“破坏性”或消耗性的活动。这能帮助合理宣泄内心的负面情绪和侵犯性，而不是将其倾泻到人际关系中。

## 总结与行动建议 📝

本节课中我们一起学习了情绪定力的培养。我们明白了不要因情绪化而过度苦恼，尤其要避免在情绪激动时进行“情绪推理”（让情绪影响认知）。

当你情绪激动时，一个非常有效的建议是：**冷静十分钟**。这十分钟能帮助你从过去的习惯性反应中抽离，冷静分析当下事件本身。养成这个习惯。

同时，多参与社会交往，勇敢地表达和锻炼自己。人际交往是一所很好的学校，能教会你如何沟通。情绪化往往是因为我们不知道还有其他方式可以获得想要的。爆发情绪通常得不到好结果，反而会引发对抗。

学会用“表达感受”的新方式，替代旧有的“情绪爆发”模式。慢慢地，你会发现一个全新的、更平和的自己。
