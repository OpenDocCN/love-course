# 雨宸19套合集：11.《情绪价值，秒变ta心目中最特别的人》：002.情绪认知篇：什么是情绪价值

## 概述
在本节课中，我们将探讨情绪价值的概念、其在人际关系中的作用以及如何提升情绪价值。

## 什么是情绪价值
情绪价值是一种通过表达情绪，使自身和对方的情绪产生共鸣，从而增加彼此亲密感和和谐度的能力。它类似于高情商，是一种后天可习得的能力。

### 情绪价值与情绪的关系
情绪价值的提供必须有情绪的高参与，只有当自身的情绪相对稳定时才有可能提供给对方情绪价值。

## 情绪价值的类型
以下是情绪价值的八种类型：

1. **分享性价值**：一次高质量的沟通或一堂有深度的课。
2. **猎奇性价值**：一次非常新奇的尝试，如玩狼人杀、探秘古堡或旅游。
3. **引导性价值**：在迷茫时，对方给予的有用建议。
4. **支持性价值**：在失落时，别人的开导和安慰。
5. **陪伴型价值**：多付出时间陪伴对方。
6. **炫耀性价值**：来源于另一半的优秀，如帅气或漂亮。
7. **环境性价值**：长期相处中形成的一种默契。
8. **优越性价值**：来源于自身的优秀。

## 低情绪价值的表现
低情绪价值的表现包括：

1. 抱怨，不顾时间地点随意发泄负面情绪。
2. 指责，不考虑对方的感受随意批评。
3. 敷衍对方，在安慰时不耐烦。
4. 否定对方，经常否定对方的人际交往。

## 总结
情绪价值是亲密关系中非常重要的一种能力，能够带给别人情绪价值的人，一定会非常受欢迎，并具有人格魅力。本节课中我们一起学习了情绪价值的概念、类型以及如何提升情绪价值。
