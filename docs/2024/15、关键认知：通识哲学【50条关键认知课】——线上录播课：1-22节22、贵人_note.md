# 15、关键认知：通识哲学【50条关键认知课】——线上录播课：1-22节22、贵人

## 贵人

**概述**：本节课将探讨如何识别和结交贵人，以及如何与贵人相处。

### 1. 贵人的定义

* **公式**：贵人 = 帮助你、给你利益、回报大于付出的人
* **代码**：`贵人 = 帮助 + 利益 + 回报 > 付出`

### 2. 如何识别贵人

* **公式**：贵人 = 冷漠 + 对你示好
* **代码**：`贵人 = 冷漠 + 示好`

* **解释**：贵人通常对他人冷漠，但对你表现出特别的关心和帮助。

### 3. 如何与贵人相处

* **公式**：利益共同体 = 利益绑定
* **代码**：`利益共同体 = 利益绑定`

* **解释**：与贵人建立利益共同体，即表达你愿意与他分享利益，共同成长。

### 4. 与贵人相处的技巧

* **表达意愿**：直接表达你想要与贵人成为利益共同体。
* **展示价值**：通过行动展示你的能力和价值。
* **感恩回报**：用私利去感谢贵人的帮助，用大义去归因。

### 5. 与贵人相处的注意事项

* **避免误解**：不要将本节课的内容误解为行贿受贿。
* **把握规则**：了解人际关系的潜在规则，并遵守。
* **保持真诚**：真诚地对待贵人，建立长期关系。

**总结**：本节课介绍了如何识别和结交贵人，以及如何与贵人相处。通过建立利益共同体，用私利去感谢，用大义去归因，我们可以与贵人建立长期、稳定的关系，实现共同成长。
