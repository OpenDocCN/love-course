# 《自然方法》搭讪自然人：2.7 共振

🎼共振是社交过程中的关键阶段，它指的是与对方建立情感共鸣，共同享受互动的过程。

以下是共振阶段的关键点：

### 1. 共振的重要性
共振是建立情感连接和判断是否合适的重要阶段。它可以帮助我们判断是否与对方在同一频道上，是否适合长期相处。

### 2. 共振的步骤
- **到达性商沟点**：在到达性商沟点之前，对话比例从9比1变为1比15比5。
- **营造氛围**：将注意力放在创造暧昧的氛围上，享受与对方的互动。
- **无目的性**：共振的目的只是为了共振，而非追求结果。
- **高价值体现**：共振能力是高价值的体现，缺乏共振能力说明社交能力存在问题。
- **时间感知**：共振会让时间感觉过得很快，反之则说明没有共振。

### 3. 共振的注意事项
- **男女框架**：男女框架是基础，有性张力的共振才是合格的共振。
- **真诚**：真诚是共振的关键，装逼者难以产生共振。
- **判断**：通过共振可以判断是否与对方合适，及时止损。

### 4. 共振与收尾
共振是发生关系的必要过程，完成共振后，下一步是收尾。下一节我们将讲解收尾阶段。

本节课中我们一起学习了共振的重要性、步骤和注意事项。共振是社交过程中的关键阶段，希望大家能够掌握并运用到实际中。
