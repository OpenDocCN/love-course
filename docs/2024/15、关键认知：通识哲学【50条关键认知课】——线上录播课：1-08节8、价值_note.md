# 15、关键认知：通识哲学【50条关键认知课】——线上录播课：1-08节8、价值

## 价值

**概述**：本节课将探讨价值的本质，包括其主观性、一般性、特殊性以及与时间和金钱的关系。

### 价值观与价值论

**价值观**：价值观是我们三关中的第二关，是主观的，不同主体对同一事物的价值判断可能不同。

**价值论**：价值论研究好坏对错的标准问题，伦理学是其中的一个学科。

### 价值的主观性

**主观性**：价值具有相当强的主体性，不同主体对同一事物的价值判断可能不同。

**广告**：广告是一种统一价值的行为，通过广告让消费者认为产品有价值。

### 一般等价物

**一般等价物**：钱是一般等价物，是大多数东西的衡量标准。

**特殊物品**：有些特殊物品无法用钱衡量，如爱情。

### 时间与金钱的交换

**时间效率**：金钱和时间之间的交换效率决定了个人能够拥有多大的价值。

**劳动时间**：商品背后代表的是劳动时间，金钱代表的是个人的劳动时间。

### 价值交换的本质

**价值交换**：任何价值交换背后都是时间，商品和金钱的交换实际上是时间和时间的交换。

### 消费行为准则

**消费行为准则**：只购买真正蕴藏时间的东西，拒绝可以标准化批量化生产的东西。

### 思考题

**思考题**：如何用自己的时间交换到更多的金钱？如何提高自己的效率？为什么效率提不上去？

**总结**：本节课中我们一起学习了价值的本质，包括其主观性、一般性、特殊性以及与时间和金钱的关系。理解价值背后的时间，有助于我们更好地进行消费和人生规划。
