# 31、魅男《NLP大脑升维》：002.NLP高效学习法

在本节课中，我们将学习如何通过NLP高效学习法提升学习效率。

## 短期记忆与长期记忆

**概念**：人的大脑记忆系统分为短期记忆和长期记忆。

**公式**：

* 短期记忆：遗忘曲线呈指数衰减，2小时内遗忘50%，12小时内遗忘约80%，48小时内遗忘约90%。
* 长期记忆：大脑的数据库，通过短期记忆和大脑处理器转换而来。

**总结**：短期记忆遗忘速度快，长期记忆需要通过复习和转换才能形成。

## 学习模型

**概念**：接收信息 -> 遇到难题 -> 行动 -> 结果 -> 反馈 -> 接收信息

**总结**：通过不断循环这个过程，将知识内化为自己的能力。

## 学习层次

1. **无意识的无能力**：不知道自己不会。
2. **有意识的无能力**：知道不会，但无法运用。
3. **有意识的有能力**：知道并能运用。
4. **无意识的有能力**：潜意识支配，无需思考。
5. **应用自如**：将知识应用于不同领域。

## 学习方法

1. **输出输入法**：学习后立即输出，加深记忆。
2. **带着问题学习**：带着问题去学习，更有针对性。
3. **降低预期**：降低目标，更容易坚持。
4. **收集正反馈**：收集正反馈，增强动力。

## 总结

本节课中，我们学习了NLP高效学习法，包括短期记忆与长期记忆、学习模型、学习层次、学习方法等内容。通过掌握这些方法，我们可以提高学习效率，将知识内化为自己的能力。
