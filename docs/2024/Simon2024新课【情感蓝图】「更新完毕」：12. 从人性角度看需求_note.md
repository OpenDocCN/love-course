# Simon2024新课【情感蓝图】「更新完毕」：12. 从人性角度看需求 🧐

## 概述

在本节课中，我们将探讨从人性角度分析需求，了解人的基本需求及其在人际交往中的作用。

## 1. 人的基本需求

人的需求分为两大类：

* **生存需求**：来源于人的生物性，如吃喝拉撒、舒适地活着。
* **精神需求**：来源于社会，被社会驯化程度越高，精神需求越高。

**公式**：人的需求 = 生存需求 + 精神需求

## 2. 生存需求与精神需求的关系

* **生存需求优先**：在人际交往中，生存需求优先于精神需求。
* **尊重生存需求**：在与他人相处时，要尊重对方的生存需求，避免挑战对方的基本需求。
* **满足精神需求**：通过满足对方的精神需求，建立深度吸引。

## 3. 如何满足精神需求

* **赋予特殊性**：通过副格赋予对方特殊性，让对方感受到自己的价值被认同。
* **精准的副格**：副格要精准地反映对方的需求，而非自己的渴望。
* **互动与深挖**：通过互动和深挖话题，了解对方的需求，并给予满足。

## 总结

本节课中，我们学习了从人性角度分析需求，了解了人的基本需求及其在人际交往中的作用。通过尊重生存需求、满足精神需求，我们可以建立深度吸引，实现人际关系的和谐发展。
