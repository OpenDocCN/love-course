# 正向吸引高阶成长课：12_第二章：内在·精神的愉悦

🎼同学们，今天我们开始学习第二章，主题是内在灵魂的契合。我们将对正向吸引进行更高级的定义。

**正向吸引的定义**：不是吸引那些仅仅喜欢我们的人，而是吸引那些我们喜欢的人，让他们也喜欢我们。

**复习第一章内容**：

*   **外在魅力**：不仅是视觉上的好看，还包括听觉、嗅觉等感官愉悦，以及营造的第一印象。
*   **身材风格和长相风格**：了解并找到适合自己的提升计划。
*   **动态颜值**：关注表情、情绪、体态等动态表现。
*   **审美品味**：提升穿搭、拍照、视频等技能，获得更多选择机会。
*   **外在吸引策略**：感官刺激、反差策略、性张力等。

**提升外在价值**：

*   **容貌提升**：调整三庭五眼比例，建立抗衰保养计划。
*   **身材训练**：制定减肥计划，进行卷腹、深蹲等锻炼。
*   **表情训练**：训练眼神、微笑等。
*   **审美提升**：观看经典电影、艺术展，积累美图。
*   **拍照技术提升**：学习构图、色调调整等。

**内在吸引力的构成**：

*   **人格品性**：性格好、三观正。
*   **三商素养**：情商、智商、逆商。
*   **灵魂思想**：有趣的灵魂、深度的思想。

**本章节学习内容**：

1.  **性格好**：了解性格类型，找到适配的另一半，避免性格不和。
2.  **三观碰撞**：交换人生观、价值观，输出有价值观点。
3.  **人设经营**：设计人设，讲好故事，提升吸引力。
4.  **表达内心**：准确表达内心想法，传递爱意，提供情绪价值。
5.  **情商**：掌握聊天技巧，提升沟通能力。
6.  **感性召唤**：提供情绪价值，满足对方情感需求。
7.  **策略**：用情绪能量感染对方，向上社交，社交媒体运营。

**总结**：

本节课我们学习了内在吸引力的构成和提升方法。内在吸引力是情感吸引力的关键，需要我们从人格、情商、感性等方面进行提升。希望同学们能够通过学习，找到适合自己的内在价值呈现之路。
