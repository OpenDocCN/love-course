# 老白《爱上大学生》：第十二课：聊天中的实用小技巧小套路

## 概述
在本节课中，我们将学习如何在微信聊天中运用实用的小技巧和套路，以提升聊天效果。

## 建立融洽感
**公式**：建立融洽感 = 消除隔阂 + 互动意愿

1. **消除隔阂**：通过开启话题，如赞美对方的穿搭或询问对方状态，打破初次见面的尴尬。
2. **互动意愿**：避免使用话术和怪点，以真诚的聊天方式吸引对方。

## 建立安全感和联系感
**公式**：安全感和联系感 = 基本信息互换 + 共同点发现

1. **基本信息互换**：交换名字、星座、学校、专业等信息，让对方了解你不是坏人。
2. **共同点发现**：通过共同兴趣或经历，快速建立信任。

## 构建吸引力
**公式**：吸引力 = 展示优势 + 价值传递

1. **展示优势**：在聊天中展示自己的独特之处。
2. **价值传递**：通过语言优化自己的价值。

## 关系升高
**公式**：关系升高 = 确定兴趣 + 技巧运用

1. **确定兴趣**：观察对方聊天反馈和情绪，判断其是否对你感兴趣。
2. **技巧运用**：通过曲解或推拉技巧升高关系。

## 深交关系
**公式**：深交关系 = 挑逗 + 调戏 + 调情

1. **挑逗**：通过言语挑逗对方，如赞美对方或表达关心。
2. **调戏**：在对方无聊或吐槽时，表达关心或调侃。
3. **调情**：通过暧昧的言语或行为，让对方知道你们不只是朋友。

## 总结
本节课中，我们学习了如何在微信聊天中运用实用的小技巧和套路。通过掌握这些技巧，你可以更好地与对方交流，提升聊天效果。
