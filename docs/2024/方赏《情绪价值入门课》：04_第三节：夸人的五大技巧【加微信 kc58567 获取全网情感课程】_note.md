# 方赏《情绪价值入门课》：04_第三节：夸人的五大技巧

好的，我们进入第三节课，主题是夸人。夸人在情绪价值中占据重要位置。

## 夸人的误区

以下三种夸人方式是常见的误区，应避免：

1. **空洞乏味的夸奖**：仅用简单词汇如“你好棒”、“你好厉害”等，缺乏具体细节。
2. **敷衍应付式的夸奖**：看似赞美，实则心不在焉，只是为了社交应酬。
3. **功利式的夸奖**：夸奖背后隐藏着个人需求，不真诚。

## 夸人的五大技巧

### 技巧一：细节

夸人时需提供具体细节，避免空洞。例如，夸赞课程时，不仅要表达“好”，还要说明具体哪些内容受益匪浅。

### 技巧二：品质

夸赞对方品质而非外在表现。例如，夸赞情商高时，应强调其为人着想、乐于助人的品质。

### 技巧三：意外

夸赞对方不常见的优点，例如，夸赞餐厅设计独特，或夸赞孩子懂礼貌。

### 技巧四：反差

先贬低对方，再夸赞，制造反差效果。例如，先说对方过去表现一般，再夸赞其现在的进步。

### 技巧五：重复

多次夸赞，让对方感受到真诚。例如，多次夸赞对方做饭手艺好，表达关爱。

## 总结

本节课介绍了夸人的五大技巧，帮助大家避免误区，学会真诚夸赞。通过练习和运用这些技巧，我们可以更好地传递情绪价值，建立良好的人际关系。

本节课中我们一起学习了夸人的五大技巧，希望这些方法能帮助大家在日常生活中更好地表达赞美，传递正能量。
