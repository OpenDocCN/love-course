# 凡哥私教课程：挽回的核心秘密 009.第九节：正确的情绪应对术

🎼今天这节课我们将探讨如何以高情商的方式挽回感情。在挽回过程中，对方可能会出现哪些情绪，我们又该如何应对这些情绪呢？🎼首先，我们来分析一下对方可能出现的情绪。🎼这些情绪由浅入深，依次为：烦、不愿意理、逃避、厌烦、厌恶和决绝。🎼在挽回过程中，当一方采取挽回行为时，往往会被对方反弹。此时，对方也处于一种情绪和情感状态中。你会感到他们烦你，甚至可能不愿意搭理你，甚至想把你踢离地球。🎼这时，我们需要梳理对方的情绪和情感状态，并对其进行等级化。🎼以下是详细解释：

### 烦
- **概念**：烦有两种相反的力量，一种是期待拉近，另一种是推开、讨厌、厌恶。
- **例子**：分手后，你找对方好好谈谈，但结果却让对方感到不舒服，从而产生烦的情绪。

### 不愿意理
- **概念**：情绪升级，不想接收信号，无力处理，感到不开心。
- **升级**：逃避（回避、沉默以对）和逃避（离开、搬离）。

### 厌烦
- **概念**：开始讨厌，收到关于你的信号就感到烦，不想见到你。
- **产生**：厌恶。

### 厌恶
- **概念**：一切信号都会让你产生生理反应，不想收到任何消息。

### 决绝
- **概念**：承受不了厌恶，关闭所有信号，拉黑情感链接。

### 解决方法
- **信念**：让时间来解决，放弃急切的心态。
- **过程**：随着时间的流逝，负面情绪会逐渐消失。
- **行动**：少犯错，不做错事。

### 调整自己
- **原因**：对方在情绪状态中时，你也会有一个对应的情绪状态。
- **方法**：放下自己的情绪，进行自我成长。

### 总结
- **核心**：让时间来解决，少犯错，不做错事。
- **信念**：相信时间会带走负面情绪。
- **行动**：调整自己，进行自我成长。

本节课中，我们学习了如何应对挽回过程中的情绪问题。希望这些方法能帮助你更好地挽回感情。我们下节课再见！
