# 14、关键对话20讲：如何成为沟通高手：3-【核心理念】：别让立场掩盖了利益

🎼你好，欢迎来到关键对话，如何成为沟通高手的课堂。🎼这是我们课程的第三讲，也是总论的最后一讲。从下一节课开始，我们将进入六大能力模块的内容。

🎼在讲解具体能力模块之前，我们要强调两件事。首先，关键对话是目标导向的复合能力，这一点在上一节课中已经详细讲解。🎼今天我们要讲的是第二件事，关键对话是围绕利益进行的。

必须清楚地认识到利益所在，才能实现一场成功的关键对话。🎼本节课的目标是，我们一起彻底搞清楚什么是利益，什么是立场，并学会如何确定自己的利益和他人的利益。🎼我们还要树立一个理念，那就是共同利益在关键对话中的引领作用。🎼同时，作为关键对话的高手，我们的脑子里还得有一根弦，那就是第三方利益。

🎼我们见过太多的对话，都是在强调立场。比如这种做法不符合我们公司的一贯政策。比如到底是听你的还是听我的？🎼比如这是原则问题，没什么可谈的。🎼或者说在这个问题上，我们必须和其他人保持一致。🎼把话说成这样的时候，这个对话差不多也就结束了，没什么可谈的。🎼但是真的是这样吗？

🎼对一个关键对话的高手来说，可谈的事多着。🎼你越是这么说，他就越能谈下去。这就是我们今天要讲的重要内容，抛弃立场式的对话，从立场中挖掘真正的利益。

🎼今天的第一个部分，我们要仔细讲解什么是利益，以及与此相对的，什么是立场。分不清利益和立场，谈话就难以取得成功。🎼关键对话的高手都是从利益和立场的分水岭中间走出来的。记住这句话。

🎼一开始我们举的几个生活中的例子，告诉你这都是立场的表达，而不是利益。我们就拿这句时不时想在耳边的名言“到底是听你的还是听我的”来分析一下，为什么这句话是立场，而不是利益。真正的利益又是什么呢？

🎼那你上一次听到“到底是听你的还是听我的”这句话是什么时候？是你女朋友或者男朋友跟你说的，还是你的合作队友跟你说的，还是你跟你的小孩说的呢？😊

🎼假设是你女朋友跟你说的。🎼你女朋友国庆节想去云南旅拍。🎼你不想去，因为你父母要来你住的城市看你，你去云南就没人招待你父母了。那你和你女朋友沟通了几回，互相都没有说服对方，说来说去还说气了。女朋友说“到底是听你的还是听我的”。🎼这句话一出口，完了。🎼大有僵持的态势。

🎼如果这会儿你问一个情感专家，他会告诉你这句话代表着你们的关系进入了权力斗争这样一个不健康的模式。🎼那作为我们关键对话课堂的同学来说，你要明白这句话是一个立场宣誓。

🎼女朋友说，她的立场是你要听她的，她说了算。🎼你要是横一点，你可能说凭什么都听你的呀，完了，这权力斗争升级了。🎼你要是怂一点，你可能会说：“这不是听不听你的问题。”

🎼哎，你这句话说得真好，不是这个问题，它到底是什么问题呢？😡。

🎼这就是你从立场谈话开始向立场谈话的起点。🎼那如何拨开立场，看到利益呢？最常见的方法就是问问题。🎼用真正的探求的口吻和想解决问题的口气。🎼以真想和对方一起搞清楚问题所在的态度来问问题。🎼不是反问，也不是质问。

🎼这个提问的技术部分，我们在后面也会提到，这里假设你通过了正确的提问搞清楚了几件事。🎼第一，你女朋友的闺蜜最近去了云南旅拍了，发了朋友圈，你女朋友已经羡慕了。🎼第二，你们两个人已经很久没有一起出去旅游了，你女朋友想和你出去玩。

🎼你懂了你女朋友的利益所在了？🎼拍一组可以发朋友圈的好照片和出去玩。🎼这两个利益就是对应的她的两个需求，一个是在朋友圈获得认可，超过至少匹配一下闺蜜发的侣拍，得到那200多个赞是吧？这是一个需要认可的需求。

第二是和你出去玩，这是增加陪伴，增进感情的需求。🎼你搞清楚了这两个需求，你就好办了呀。🎼你女朋友的需求里没有非得去云南，也没有排斥你父母来看你。

🎼说到这儿了，你已经知道该怎么办了吧，而且解决的方案不只是一个。🎼那前面我们从立场开始，逐渐剥开了这个立场的包裹，找到了利益所在，又把利益转化成了需求。

需求明确了之后，解决问题的方案他自己就冒出来。🎼这就是我今天给你的第一个模型，我们叫它“拆开立场的包裹模型”。那你可以看如何拆开立场这个包裹呢？😊，立场是一个外壳。

🎼利益是它的内容，而需求是利益的目的。🎼利益的实现就是需求的满足。🎼对方向我们表明了一个立场，就好像给我们发来了一件快递，是包裹着的物品。

我们要小心地拆开这个包裹。🎼开箱的刀子就是用合适的问题把这箱子打开，然后找到里面的内容，这个内容就是利益所在。而对方为什么需要这个内容，这就是他的需求，对吧？

🎼你拆开一个箱子里边是个杯子，那他的需求就是他渴了吗？他要喝水，或者他要喝酒。🎼这是显而易见的。

🎼这个拆开立场包裹的模型，需要你熟练地掌握。🎼听见一句话，先去判断这是立场还是利益。🎼如果是立场，那么它包裹的利益是什么？利益反映的需求又是什么？

🎼这种思维方式要成为你头脑中的自动调用的一套程序，就像你看见树在剧烈摇动，你就知道要刮风，你看看天色就知道要下雨，就得成这么自然的反应，你才是一个在利益和立场之间能够区分的谈话高手。

🎼明确了关键对话的第一个重要原则，也就是我们关注的是利益，而不是立场。🎼我们接下来讲第二个重要原则，关键对话要始终坚持共同利益。

🎼坚持共同利益，首先你得看到共同利益。🎼有些共同利益特别明显，有些则没有那么明显。那在合作和交易的对话中，共同利益是显而易见的。

双方可以达成协议，那就各取所需。比如说就像下班路上，我在街边花店买花，我买到了花，老板拿到了钱，那成交就是我们的共同利益。如果这会儿已经比较晚了，甚至说快下雨了，我可能还用比较少的钱买到许多花。

因为老板想清仓嘛，这花过夜他就不新鲜了，所以他乐意降价，让我多买。那这个故事里的共同利益就非常的明显。😊

🎼但是呢，在另外一些对抗性的对话的场景下，共同利益就没那么明显了。🎼比如我们为了解决某个争议而谈判，那表面上看起来呢这是个零和游戏。

🎼你拿走了多少，就是我少了多少，那看起来没有什么共同利益。😡

🎼可是真的是这样吗？🎼我举个例子。🎼有一个电商的网站发布了一组商品的推广照片。

其中有一个照片呢是一个明星穿着某个品牌的衣服。🎼但是明星呢跟这个电商之间跟那个品牌之间都没有合作关系啊，他只是出席活动，随便穿了那么一件他喜欢的衣服，这并不构成一个代言。

🎼这张照片呢品牌看到了就先发出来了。然后电商从品牌那把这照片拿过来，直接就拿过来用。🎼明星就认为啊是电商网站侵犯了他的肖像权，对吧？我也没有给你授权，你为什么用我照片来推销这个商品？

🎼他就发律师函，给这电商网站要求赔偿。🎼我在这儿插一句题外话，可能有同学问说，为什么这明星不告服装品牌，而是告电商的？🎼因为明星现在虽然跟这个品牌没合作啊，但是不排除以后会有合作哈。

所以明星一般不会告品牌，而且会告商业平台。这是给你分享一个可能没用，但是有趣的小知识。🎼我们说回来啊，在这场明星和电商的索赔谈判当中，仿佛是没有共同利益的对吧？

🎼电商好比说给明星赔了10万块钱，那电商就少了10万块钱。🎼即便是这样，双方还是走进一个会议室里谈起来，为什么？就是因为他们有共同利益。

🎼同学，请你记住这句话。🎼只要是在对话中的双方永远都有共同利益。哪怕这个对话看起来是个零和游戏，哪怕双方是对抗性的，甚至是有敌意的。

只要他们还谈着，有时候甚至是互相高声指责着，他们之间还是有共同利益。在明星和电商的这场索赔的谈话里，共同利益是什么呢？

🎼最明显的一点就是避免诉讼。对明星来说，这一类照片侵权不得不打，因为不打就会更多。但是真的去法院打，一是时间长，二是赔偿数额也没有他要的那么多。

第三，你还得花律师费，不是吗？🎼对电商来说呢，也得尽量避免诉讼。诉讼时间长，需要耗费的管理资源也多，需要多花律师费。

而且如果他被判侵权的话，还会影响他的声誉。🎼所以啊双方都有在诉讼之前和解的需要，这就是共同利益。就是因为这一点让双方坐在了一起。

🎼所以啊请你记住了。🎼就假设别的什么共同利益都没有，避免损失也是共同利益。🎼这个案子我们假设一下，说双方不谈不对话，那各自有什么损失？

🎼这个损失一定比谈的成本要高，双方才会谈。🎼那寻找共同利益的时候呢，有个基本的思维方法就是逆向思维。🎼这个思维方式就是让你来填个空。
