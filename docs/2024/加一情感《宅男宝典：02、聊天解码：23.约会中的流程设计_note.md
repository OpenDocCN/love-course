# 加一情感《宅男宝典：02、聊天解码：23.约会中的流程设计_note

在本节课中，我们将要学习如何为约会设计一个合理的流程。一个流畅且符合关系发展阶段的流程，是营造良好约会氛围、避免尴尬并成功升高关系的关键。

上节课程我们介绍了在约会中营造良好氛围的重要性。本节中我们来看看，如果氛围始终无法建立，问题可能出在哪里。

很多时候，约会本身没有明显错误，但气氛就是无法活跃起来。这很可能是因为约会流程设计得不够合理，导致女生在某个环节感到不适应，从而不愿配合互动。

例如，一个常见的情况是约女生出来看电影。看电影的过程中，双方全程无交流。电影结束后，女生的情绪可能就一直无法高涨，导致后续约会变成“垃圾时间”；或者她会找借口先行离开，然后就没有然后了。

这节课我来讲解关于约会流程的设计。整个约会其实是贯穿在一个又一个不同场景中的。前面的课程也提到了转场的重要性。我们同样可以利用在不同场景下的互动来提升约会氛围。

但是，大多数兄弟对于约会场景的选择，其先后顺序是错误的。

以下是正确的约会场景顺序，应依次为：破冰场景、主线场景、暧昧场景、私密场景。

接下来我逐一为大家解释。

首先，**破冰场景**。这类场景沟通难度低、自带氛围，例如电玩城、游乐场、滑雪场、艺术展等。这些场景有一个共同点：即使是陌生人也能自然地玩到一起。在约会前期，彼此还比较陌生时，我们应该选择这类场景进行互动，而不是那些需要刻意寻找话题的场景。

上一节我们介绍了破冰场景，本节中我们来看看**主线场景**。这类场景以真正的约会氛围为主，例如看电影、浪漫的餐厅、溜冰场、陶艺馆等。这些地方的特点是：和好朋友一起去不会尴尬，但和陌生人去就会感到不自在。因此，它们比较适合与已经有了一定熟悉感的人前往。

在建立了基本熟悉感之后，可以进入**暧昧场景**。这类场景暧昧气息浓厚，例如鬼屋探险、逛内衣店、一起做瑜伽、逛超市买菜、一起做鱼疗或足疗等。这些场景适用于已经有了一定暧昧基础的人，大多数升高关系的机会也出现在这里。不知你是否发现，这些场景是情侣之间常去的。女生会下意识地觉得和你的关系越来越像情侣。同时，你能带她去这些场景，也是你区别于她过往经历的关键。

最后是**私密场景**。这类场景以营造私密氛围为主，例如烛光晚餐、私人影院、你家、她家等。在这些场景中，你们可以一起做饭或相拥看恐怖片。这就像老夫老妻一样，能给女生营造一种“回家”的感觉，同时让她感到安全，因为这种空间只属于你们两个人。

那么，我之前比较常用的一套流程也可以推荐给大家，这也是搞定“班嫂”的流程，兄弟们可以参考。

以下是这套流程的步骤：
1.  首先，我可能和女生约在商场楼下见面，然后在商场里简单逛逛，彼此熟悉一下。
2.  接着，去打电玩，中间会夹娃娃。在这个过程中，我会进行一些升高关系的互动。
3.  然后，可能会一起看场电影。
4.  如果时间还早，我会带她去一趟鬼屋，这是我着重升高关系的一个场景。
5.  最后，一起去买菜，然后回家一起做饭。
