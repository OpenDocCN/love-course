# 正向吸引高阶成长课：14_人格特质：原生家庭只是起点 （下）

🎼在了解依恋人格之后，我们进一步探讨特殊的人格障碍类型。在情感交往中，需提前识别潜在风险，即使对方再优秀，也应尽量远离自恋型人格障碍（NPD）和双向情感障碍（BD）。

🎼感兴趣者可查阅具体特征和症状。接下来，学习人格特质的第二个重要模型——性别角色。🎼从性别角度看，分为三种人格特质：工具性、表达性和双性特质。传统观念认为，男性应具备男子气概，自信、独立、勇敢；女性则应展现女性特质，热情、细腻、敏感、温柔和善良。

🎼工具性特质关注任务和使命，更多关注事物本身；表达性特质关注社交和情感，更多关注人情和关系。低表达性的人缺乏情绪感染力和同理心，不易表现出热情和温柔；工具型的人则缺乏自信和魄力，自尊和适应能力较低。

🎼现代社会中，超过三分之一的人并非完全的工具性或表达性。他们展现出自信和温柔、独立坚强又多愁善感的双性特质。并非只有具有强烈男子气概的男性和具有女人味的女性才最合适。研究表明，至少一方具备双性特质，感情会更加融洽。

🎼从吸引力的角度看，最具魄力的男性是那些最懂女人心的男人。他们知人冷暖，懂你所想，深情又温柔，很少有女人能抵挡。同样，最具魄力的女性也是最懂男人的女人，上得厅堂，下得厨房，独立又不过分依赖。

🎼当男性接受女性的吐槽和唠叨，并愿意倾听，不带反驳，只是微笑，就能获得更多快乐和幸福。当女性接受男性的心思，避免逃避麻烦，理解男性行为，就不会感到太多苦恼。

🎼双性特质的男性能更好地洞察女性细腻的心思，避免陷入大男子主义；双性特质的女性能更好地工作、社交，避免过度依赖和胡思乱想。

🎼回顾一下周围在感情中游刃有余的人，是否大多具备双性特质？而那些一味抬杠、做作的人，在感情中是否举步维艰？

🎼以下案例：女性追求年轻健身教练，对方对其不太感冒。在聊天过程中，双方意见不一，女性开始摆道理，语气强势，解读对方，激化矛盾，讽刺对方，最终挑衅对方。

🎼因此，不要解释自己，不要与男性讲道理，不要挑战男性的胜负欲。对付工具型男性，要调动表达性特质，如适当撒娇。这些技能将在后续章节中学习。

🎼女性在工作中可以强势，但不要将这种习惯带入生活。刀子嘴豆腐型的女性往往命苦。

🎼只有表达性的人才会更具有同理心。了解直男通常受母亲保护较好，可能长期处于母亲的语言和精神操控之下，要找到对方的软肋对症下药。

🎼任何人在成长中的缺失，都会渴望在新的关系中找回，构成一定的内在吸引力。

🎼Big5人格模型包含五个绝对正向的人格特质：开放性、外情性、尽责性、疑人性、情绪稳定性。这些特质构成英文单词ocean，代表人格的海洋。

🎼一项研究发现，婚姻幸福的程度至少有50%的因素由双方婚前的神经质特质决定。神经质越弱的夫妻越容易感到幸福。

🎼接下来，将分享一些有趣的研究成果，如Big5人格与依恋人格的联系，不同依恋人格对情感态度和爱情观的不同心态。

🎼此外，还有一些有趣的人格测试，如九型人格，了解人的本能、理性、感受主导。

🎼本节课程最后一个重要内容：两个人在一起，性格相似好，还是性格互补好？

🎼DIS性格分析模型将人的性格分为两个维度：是否愿意主动追求，是否更理性关注事物，还是更感性关注人情。得到四个不同的象限，即老鹰型、狐狸型、刺猬型、兔子型。

🎼将DIS四大类型展开，得到九种不同的性格方向。性格相似好，还是互补好？记住口诀：中轴要相似，对角要互补，中间可百搭。

🎼横竖中轴两端的人性格差异太大，难以相处；处于四个象限极端性格的人，如果太相似，也会出现性格冲突。

🎼支配型与顺从型、感召型与观察型互补，更合适。综合型人格最常见，但缺乏个性。

🎼性格会影响人生价值观，形成九种不同的人生规划，即成长人格、宿命人格、稳定人格和多变人格。

🎼越渴望成长、追求多变的人，越容易被互补性格所打动；越追求安逸、害怕变化的人，越容易被相似性格所打动。

🎼在一段关系中，有两个不同的需求面：向内探索和向外摄入。

🎼阿德勒生活风格模型反映从原生家庭中养成的行为模式，包括宠溺型、讨好型、自我型和粘人型。

🎼原生家庭养成的生活风格和亲密关系中不断进化出的依恋人格相互交叉作用，形成多重人格。
