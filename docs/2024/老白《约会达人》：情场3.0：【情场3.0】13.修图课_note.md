# 老白《约会达人》：情场3.0：【情场3.0】13.修图课

## 概述

在本节课中，我们将学习如何使用美图秀秀、VSCO和SNAPSEED等软件进行照片修图，提升照片质量。

## 修图软件介绍

以下是几款常用的修图软件：

* **SNAPSEED**：一款加滤镜的软件，支持安卓和苹果手机。
* **美图秀秀**：一款功能丰富的修图软件，支持安卓和苹果手机。
* **VSCO**：一款付费软件，但免费版本也足够使用，支持安卓和苹果手机。

## 美图秀秀修图步骤

1. 打开美图秀秀，选择要修的照片。
2. 使用“美容”功能进行美颜、磨皮、美白等操作。
3. 使用“编辑”功能进行瘦脸、瘦身、修容等操作。
4. 使用“滤镜”功能添加滤镜效果。
5. 保存修好的照片。

## VSCO修图步骤

1. 打开VSCO，导入要修的照片。
2. 使用“工具”功能调整曝光、对比度、锐化、清晰度、饱和度等参数。
3. 使用“滤镜”功能添加滤镜效果。
4. 保存修好的照片。

## SNAPSEED修图步骤

1. 打开SNAPSEED，导入要修的照片。
2. 使用“工具”功能调整曝光、对比度、锐化、清晰度、饱和度等参数。
3. 使用“滤镜”功能添加滤镜效果。
4. 使用“样式”功能调整样式强度、阴影强度等参数。
5. 保存修好的照片。

## 总结

本节课中，我们学习了如何使用美图秀秀、VSCO和SNAPSEED等软件进行照片修图。通过练习，你可以掌握不同的修图技巧，提升照片质量。
