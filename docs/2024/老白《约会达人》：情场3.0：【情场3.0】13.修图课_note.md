# 老白《约会达人》：情场3.0：【情场3.0】13.修图课_note

在本节课中，我们将要学习如何通过修图软件，将照片修饰得更加出色。我们将介绍三款常用软件，并详细讲解从人物瑕疵修复到滤镜风格调整的完整流程。

## 软件介绍

以下是本节课将要用到的三款修图软件。

*   **Snapseed**：一款功能强大的免费滤镜与图片编辑软件，图标为树叶形状。
*   **美图秀秀**：一款广为人知、功能全面的图片美化软件，尤其擅长人像处理。
*   **VSCO**：一款以高质量滤镜和胶片质感著称的软件，部分功能收费，但免费功能已足够使用。

> **核心概念**：`Snapseed` 和 `VSCO` 主要用于调整图片整体色调和质感（加滤镜），而 `美图秀秀` 主要用于修饰人像细节（修人形）。

## 修图流程概述

我们的标准修图流程分为三步，以下是具体步骤。

1.  **第一步**：使用 **美图秀秀** 处理人像瑕疵，调整身形与面部轮廓。
2.  **第二步**：使用 **VSCO** 进行精细的滤镜和质感调整。
3.  **第三步**：使用 **Snapseed** 添加最后的复古滤镜风格，并微调强度。

## 实战演练一：人物与场景照

上一节我们介绍了修图流程，本节中我们来看看如何修一张带场景的人物照片。

### 第一步：美图秀秀 - 基础人像调整

首先，我们打开美图秀秀，导入照片。我们的目标是修饰人物身形上的小缺陷，使其看起来更完美。

以下是美图秀秀“美容”功能下的主要操作点。

*   **瘦脸瘦身**：此功能最常用。可以推拉肩膀使其更厚实，或微调脸部轮廓。注意调整时不要使背景线条弯曲，否则会显得不自然。如果不小心推弯了，可以使用撤销键。
*   **祛斑祛痘**：如果脸上有斑点或痘痘，可以使用此功能点掉。
*   **去皱**：用于去除脸上的皱纹或抬头纹。
*   **增高**：如果觉得身高不够，可以使用“增高塑形”功能拉长腿部。
*   **不推荐功能**：“一键美颜”、“美妆”、“面部重塑”等功能效果通常过于夸张或不自然，不建议使用。

完成基础调整后，点击保存。

### 第二步：VSCO - 质感与滤镜调整

接下来，我们使用VSCO为照片添加质感。打开VSCO并导入上一步保存的照片。

以下是VSCO中需要调整的核心参数。

*   **曝光**：向右滑动增加亮度，向左滑动降低亮度。
*   **对比度**：向左滑动会降低色彩对比，使画面柔和；向右滑动会增加对比，使色彩更鲜明。
*   **锐化**：建议适当调高，可以让图片细节更清晰。调整时可以放大图片观察效果。
*   **清晰度**：不建议调太高，否则会显得生硬。轻微增加即可提升质感。
*   **饱和度**：控制色彩鲜艳程度。向右滑动更鲜艳，向左滑动更清淡。
*   **颗粒**：适当添加颗粒感可以增加照片的胶片质感。
*   **色调分离**：
    *   **阴影色调**：一般添加蓝色调。
    *   **高光色调**：一般添加黄色调。

调整完毕后，点击保存。此时照片的质感已经得到很大提升。

### 第三步：Snapseed - 最终风格定型

最后，我们使用Snapseed为照片添加一个复古滤镜。打开Snapseed并导入VSCO处理后的照片。

具体操作如下：点击 **工具** -> **复古**。在复古滤镜中，7、8、9、10号色系都较为常用，可以选择一个喜欢的。

如果觉得滤镜效果过重，可以在屏幕上**上下滑动**，选择“样式强度”并调低。同样，上下滑动选择“晕影强度”可以调整照片四角的暗角效果，突出画面中心。

调整满意后，导出保存。至此，一张带场景的人物照就处理完成了。

## 实战演练二：正面人像照

处理完场景照，本节我们专注于如何修饰一张正面人像照片。这类照片的核心在于面部细节的精致处理。

我们继续使用美图秀秀。导入一张正面人像照片后，进入“美容”模块。

以下是修饰正面人像的具体步骤。

1.  **祛斑祛痘**：使用此功能，选择合适大小的笔刷，点掉脸上的痘痘、斑点或疤痕。处理时左上角会出现放大镜，便于精确定位。
2.  **瘦脸瘦身**：放大图片，微调脸部轮廓。例如，如果觉得脸型较宽，可以向内轻轻推压脸颊边缘。也可以微调鼻型或发型。
3.  **去黑眼圈**：如果眼周有黑眼圈或暗沉，可以使用此功能进行淡化。
4.  **眼睛放大**：使用“眼睛放大”功能，将“+”号对准黑色瞳孔后松手，可以自然放大眼睛。

处理完所有瑕疵并调整好轮廓后，点击保存。对于精致的正面人像，完成美图秀秀的步骤后通常已足够，无需再进行复杂的滤镜调整，以保持自然感。

## 课程总结

本节课中我们一起学习了系统的手机修图方法。

我们首先认识了 **Snapseed**、**美图秀秀** 和 **VSCO** 这三款核心软件及其分工。随后，我们掌握了 **“先修人，后调色”** 的标准流程：先用美图秀秀处理人像瑕疵与身形，再用VSCO调整照片的曝光、对比度、质感等基础参数，最后用Snapseed添加风格化滤镜并微调强度。

记住，修图的目的是弥补小缺陷、提升质感，而非改头换面。关键在于多练习，用不同的照片尝试，最终你会找到得心应手的感觉，让每一张分享出去的照片都更具吸引力。
