# 《二月老师的核心自信》（综合版）：44-附加：生命的内耗（4）_ev

🎼本节课我们将探讨内耗的第五点：停止反复犹豫。

**1. 停止反复犹豫**

你是否经常犹豫不决？比如想去减肥、旅行或创业，但一想到其中的困难就犹豫不决。最终，你发现自己什么也没有做成。作家托布花曾说，人生总是有很多左右为难的事。

**解决方案**：

- 不要反复推演，立即行动。
- 放弃脑海中左右摇摆的想法。
- 不要总是想着如何得到万无一失的方法。

**2. 自我攻击**

自我接纳是自信的前提。如果你无法接纳自己，那么你永远都别想拥有真正的自信。

**自我攻击的表现**：

- 总是觉得自己不够好。
- 自我贬低，认为自己差劲。

**解决方案**：

- 完全接纳自己不能逆转的一切，包括长相、身高等。
- 找到自己身上的闪光点，欣赏自己。

**3. 对抗事实**

对抗事实只会增加内耗，而且对你没有任何帮助。

**解决方案**：

- 接纳事实，直面结果。
- 将精力放在可以改变的事情上。

**4. 内耗**

内耗是最愚蠢且最无意义的自我消耗，与情感、社交和自信息息相关。

**解决方案**：

- 停止内耗，让你的内心处于稳定平和的状态。

**总结**

本节课中，我们学习了如何停止反复犹豫、自我攻击和对抗事实，以及如何应对内耗。希望这些内容能帮助你提高自信，过上更好的生活。
