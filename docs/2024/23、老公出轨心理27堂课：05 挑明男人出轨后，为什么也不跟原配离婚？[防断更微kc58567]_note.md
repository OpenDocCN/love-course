# 23、老公出轨心理27堂课：05 挑明男人出轨后，为什么也不跟原配离婚？

我叫朱生勇，婚姻管理师，婚姻管理学派的创始人，从事婚姻管理工作十年，辅导上万个家庭修复夫妻关系，重建婚姻。协助大家挽救婚姻，治理婚外情，做离婚辅导，还有复婚辅导，协助大量小三从婚外情中走出来。

## 主题概述

本节课将探讨为什么男人出轨后，多数情况下不愿意与原配离婚。

## 案例分析

**案例**：一位妻子发现丈夫出轨，第四个小三打电话威胁，妻子询问丈夫，丈夫否认，称小三想借钱未借，污蔑自己。妻子发现真相后，丈夫解释称酒后与小三发生关系，但从未想与她在一起，并称自己从未想过离婚，多年来一直对她好。

## 出轨原因

1. **心理需求**：男人出轨多数是为了满足心理需求，如肯定、认可、关注、陪伴、归属、爱和被爱。
2. **情感需求**：夫妻情感疲劳，寻求新鲜感。
3. **生理需求**：夫妻运动不和谐，口腹之欲，皮肤抚摸的满足。
4. **价值需求**：照顾小三，帮助小三解决问题，满足事业成就感。
5. **物质需求**：有钱女人帮助满足物质需求。
6. **个体发展需求**：寻求个人发展，突破人生瓶颈。
7. **房子需求**：想生孩子，想生儿子。

## 离婚原因

1. **性格冲动**：冲动型人格容易冲动离婚。
2. **婚外情二次创伤**：婚外情导致婚姻二次创伤，如私生子、分居、冷战等。
3. **与小三和谐**：与小三关系和谐，共同解决问题。
4. **理性分析小三取代妻子**：理性分析小三和妻子，认为小三更优秀。
5. **婚姻不想要了**：婚姻状况恶化，对妻子失望。
6. **妻子不是自己想要的人**：妻子无法满足他的需求，小三有疗愈作用。
7. **对未来高估**：对未来有良好预期，认为与小三在一起会更好。
8. **对现状差评**：对婚姻现状不满，挑三拣四。

## 不离婚原因

1. **满意度评估**：评估婚姻中满意和不满意的地方。
2. **婚姻生活评估**：评估婚姻生活、身体状况、财富、人际关系等。
3. **矛盾评估**：评估婚姻中的矛盾和问题。
4. **感情评估**：评估夫妻感情。
5. **离婚后果评估**：评估离婚后的财产损失、社会关系和家庭关系损失。

## 总结

本节课中，我们学习了男人出轨后不愿意离婚的原因，以及男人离婚的原因。希望大家能够了解这些原因，更好地处理婚姻问题。
