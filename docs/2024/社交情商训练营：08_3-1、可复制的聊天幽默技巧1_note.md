# 社交情商训练营：08_3-1、可复制的聊天幽默技巧1 😄

在本节课中，我们将要学习如何通过系统性的方法来培养幽默感，特别是制造幽默的能力。幽默不仅能让你更受欢迎，还能为你的人际关系和人生增添色彩。

## 概述：幽默的价值与构成

幽默之所以吸引人，是因为人类是情绪的生物。一个能带来正面愉悦情绪的人，会让人情不自禁想要靠近。反之，一个充满负能量的人则会让人想要远离。因此，幽默可以让我们更受欢迎。

幽默主要包含两个方面：**接收幽默**与**制造幽默**。接收幽默指你能听懂别人的笑话和梗；制造幽默则指你能主动讲笑话、说俏皮话或做出搞笑行为。本教程将重点学习**制造幽默**的技巧。

掌握制造幽默的能力，尤其能提升你在异性眼中的吸引力。幽默的男人不仅能带来愉悦情绪，通常还具备丰富的认知、强大的创造力，并能利用幽默建立和维护广泛的人脉。人脉的扩展意味着获取资源的概率更高，因此，幽默是吸引异性的重要技能。

## 锻炼幽默的第一步：模仿

学习任何技能都始于模仿，幽默也不例外。我们可以通过观看幽默类节目来入门，例如《脱口秀大会》、《吐槽大会》、相声或《奇葩说》等。

观看这些节目时，重点在于分析其中的幽默原理。每当一个段子让你发笑，请暂停并思考：是什么原理让我笑了？这个原理还能创造出其他什么样的幽默？同时，把这个段子记下来，思考未来如何在现实生活中套用。

经常进行这样的练习，你将对幽默建立起初步的认知。

## 锻炼幽默的第二步：收集与运用

在掌握了模仿的方法后，下一步是学会系统地收集幽默素材。具体做法是：记录下一切能让你发笑的段子。

这些段子可以来自网络、节目，或是身边人使用过的。只要你觉得有趣，就默默记下来。然后，在现实生活的对话中不断尝试运用这些段子。

例如，你可以准备一个笔记本，专门收集搞笑的段子。通过实战检验，保留那些效果好的段子，删掉效果不佳的，并持续更新你的素材库。久而久之，你会积累大量素材，并在聊天中能根据关键词迅速联想到相关段子，从而变得越来越幽默。

## 幽默的核心思维模式

完成模仿和收集后，你已经掌握了幽默的基本功。然而，幽默更讲究的是思维模式。掌握了思维，你就能创造出属于自己的幽默。

以下是四种常用且高效的幽默思维模式：**自恋**、**夸张**、**曲解**和**反转**。本节我们先来详细探讨第一种——自恋。

### 自恋式幽默

自恋，说白了就是非常喜欢、欣赏乃至崇拜自己，对自己充满自信。然后通过语言，将这种喜欢自己的点表达出来，从而引人发笑。

**公式**：`自信的自我陈述 + 出人意料的“烦恼” = 自恋式幽默`

例如：“有时候我也挺郁闷的，我明明这么有才华，但大家偏偏就只喜欢我这张脸。哎，太帅了也是一种负担。” 这就是典型的自恋式幽默。

这里有一个重点需要记住：越幽默的人，表演能力越强。**你不能比观众先笑**，否则幽默效果会大打折扣。无论你觉得多好笑，都要忍住，等到观众笑了，你再跟着笑。

### 实战自恋段子示例

以下是几个经过实战检验、效果良好的自恋式幽默段子。学会后，请务必在现实生活中多加运用。只有通过大量实战，你才能真正掌握这种幽默技巧。

*   “我照镜子的时候，经常担心镜子会爱上我。”
*   “不是我吹牛，我要是不当（你的职业），绝对是模特界的损失。”
*   “别人靠才华吃饭，我靠脸。但有时候我也很苦恼，才华它老想出来抢镜。”
