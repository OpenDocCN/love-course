# 钱进社 - 成为优质且最具竞争力的【社会人】：75、如何构建自主产生的焦虑管理习惯

## 概述

在本节课中，我们将探讨如何构建自主产生的焦虑管理习惯，帮助大家更好地应对生活中的压力和挑战。

## 核心概念

**焦虑管理习惯**：通过一系列的实践和策略，将焦虑转化为积极的动力，帮助个人更好地应对生活中的压力。

### 公式

```
焦虑管理习惯 = 积极心态 + 时间管理 + 责任制 + 重复实践
```

## 一、保持积极

以下是保持积极心态的两种方法：

### 1. 批判自己，威胁自己

- **方法**：时刻将自己拉到一个目标为导向的氛围中，不做没有用的自我催眠。
- **示例**：当目标未达成时，不要自我安慰，而是要反思原因，寻找改进方法。

### 2. 纯激励自己

- **方法**：客观地激励自己，分析原因，制定具体方案，增加信心和激励感。
- **示例**：当目标未达成时，激励自己下一次做得更好，并分析原因，制定改进方案。

## 二、留固定的时间

### 1. 时间计划

- **方法**：将重要事情放在每天固定时间段，雷打不动。
- **示例**：即使遇到特殊情况，也要坚持在固定时间完成重要事情。

### 2. 高密度投入和培养

- **方法**：通过时间线拉长，促使习惯养成。
- **示例**：将重要事情放在固定时间段，高度专注，从而培养习惯。

## 三、有意识的建立问责制度

### 1. 借助外力

- **方法**：寻找内在驱动较弱或思维认知动荡的朋友、家人或同事，让对方指出自己的不足和懈怠。
- **示例**：与朋友、家人或同事定期交流，让对方指出自己的不足，并共同改进。

### 2. 问责制度

- **方法**：与朋友、家人或同事建立问责制度，定期交流，复盘进度，调整方向。
- **示例**：每月或每季度与朋友、家人或同事交流一次，让对方指出自己的不足，并共同改进。

## 四、如何坚持下去

### 1. 重复

- **方法**：重复实践，让大脑和身体熟悉这种状态，形成习惯。
- **示例**：每天坚持练习，直到形成习惯。

### 2. 应对诱惑

- **方法**：在做重要事情时，避免干扰，如关闭娱乐设备，保持专注。
- **示例**：在复盘时，关闭手机、电视等娱乐设备，保持专注。

### 3. 自我奖励

- **方法**：完成目标后，给自己一些奖励，如购买喜欢的东西、享受美食等。
- **示例**：完成目标后，奖励自己一顿美食，增加快乐感和成就感。

## 总结

本节课中，我们学习了如何构建自主产生的焦虑管理习惯。通过保持积极心态、留固定时间、建立问责制度等方法，我们可以更好地应对生活中的压力和挑战。希望同学们能够将所学知识应用到实际生活中，不断提升自己。
