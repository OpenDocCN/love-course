# 重开计划（大橙2.0）2499：12、开场白技巧全讲解

## 概述

在本节课中，我们将深入探讨开场白的技巧，帮助你在与陌生人交流时留下良好印象，并为进一步的聊天和约会打下基础。

## 开场白的核心目标

开场白的前20句话，主要实现以下四个目标：

1. **留下好印象**：通过提供信息，让对方对你产生好感。
2. **提供信息**：介绍自己，让对方了解你的基本信息。
3. **测试兴趣**：了解对方对你的兴趣程度。
4. **传递共鸣**：展示你们有共同话题或兴趣，让对方感到舒适。

## 如何实现目标

### 1. 留下好印象

- **收集信息**：通过观察对方的朋友圈、照片等，收集相关信息。
- **推理联想**：根据收集到的信息，推理出对方的性格特点、兴趣爱好等。
- **造句表达**：用简洁、生动的话语表达你的观察和推理结果。

### 2. 保证我们是同一个世界的人

- **话题一致**：根据对方的兴趣爱好，选择合适的话题进行聊天。
- **降低交流成本**：确保对方能够理解你的话题，避免使用对方不熟悉的词汇或概念。

### 3. 筛选掉不合适的人，留下对自己感兴趣的人

- **判断标准**：根据对方的回复速度、频率和内容，判断对方是否合适。
- **分类处理**：将对方分为A、B、C、D四类，针对不同类型的人采取不同的聊天策略。

### 4. 进行后续聊天

- **升级关系**：根据对方的态度和反应，逐步升级关系。
- **调整节奏**：根据对方的不同阶段，调整聊天的节奏和内容。

## 注意事项

1. **话题控制**：开场白的前20句话，不要超过5个话题。
2. **回复选择**：至少找出3个回复路线，选择最适合的回复方式。

2. 复习开场白的五大核心思维，结合本节课内容，重塑你的开场白策略。

本节课介绍了开场白的技巧，帮助你在与陌生人交流时留下良好印象，并为进一步的聊天和约会打下基础。通过课后作业的练习，相信你能够掌握这些技巧，提升自己的社交能力。
