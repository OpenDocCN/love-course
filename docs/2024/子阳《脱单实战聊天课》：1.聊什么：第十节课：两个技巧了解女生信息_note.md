# 子阳《脱单实战聊天课》：1.聊什么：第十节课：两个技巧了解女生信息

## 概述
在本节课中，我们将学习如何通过两个技巧来了解女生的信息，使聊天更加有趣且高效。

## 技巧一：主动提供话题
**公式**：主动提供话题 → 提供谈资 → 形容自己

1. **主动提供话题**：在聊天中，主动提出一些话题，让对方有话可说。
2. **提供谈资**：在话题中提供一些背景信息或相关内容，让对方有更多的谈资。
3. **形容自己**：在描述话题的同时，适当加入自己的经历或观点，使对话更加生动。

## 技巧二：发散延伸
**公式**：女生提供信息 → 发散延伸 → 转移话题

1. **女生提供信息**：当女生给出一些信息时，如兴趣爱好、工作等。
2. **发散延伸**：在女生提供的信息基础上，进行发散和延伸，使话题更加丰富。
3. **转移话题**：当遇到不太会回的信息时，用形容自己的方式转移到下一个话题。

## 实战案例
以下是一个实战案例，展示了如何运用这两个技巧：

**场景**：与一位女生聊天

**对话**：

- 你：嗨，我叫紫阳，很高兴认识你。
- 她：嗨，我叫小芳，很高兴认识你。
- 你：我在武汉做自媒体，主要做抖音的男性情感博主。
- 她：哦，抖音男性情感博主，听起来很有趣。
- 你：是啊，我以前在深圳工作的时候，晚上都会去健身房锻炼。
- 她：我也喜欢运动，尤其是健身。
- 你：那我们以后可以一起去健身房啊。
- 她：好啊，我也想找个健身伙伴。

通过以上对话，我们可以看到，通过主动提供话题和发散延伸，我们成功地了解了女生的信息，并使对话更加有趣。

## 总结
本节课中，我们学习了两个技巧来了解女生的信息：主动提供话题和发散延伸。通过运用这两个技巧，我们可以使聊天更加有趣，更有效地了解对方。
