# 社交情商训练营：06_2-1、高情商聊天口才修炼2 🎤

## 概述
在本节课中，我们将学习如何通过强化锻炼和知识储备来提升高情商聊天口才。

## 强化锻炼的重要性
**强化锻炼，不是一次两次就能够解决的。** 我当年为了锻炼口才，每天对着镜子自我对话，直到声音沙哑。

**公式：** 强化锻炼 = 持续练习 + 反复重复

## 基本功
**先去练好这6个基本功啊，练好了。之后呢，你的口才基本上算是合格了。**
1. **自我对话**：每天对着镜子自我对话，锻炼表达能力。
2. **倾听**：认真倾听他人说话，理解他人需求。
3. **观察**：观察他人言行，了解他人性格特点。
4. **表达**：清晰、准确地表达自己的观点和想法。
5. **沟通**：学会与他人有效沟通，达成共识。
6. **情绪管理**：控制自己的情绪，保持冷静。

## 知识储备
**如果说你想更进一步，那么你就需要干嘛？需要去储备更多的这些素材。有一句话叫做什么输入决定了你的输出。**
**以下是5个知识储备方向：**
1. **读书**：阅读心理学书籍，洞察人性，提升人际交往能力。
2. **关注热点**：了解当下热门事件，紧跟时代步伐。
3. **旅行**：拓宽视野，提升格局。
4. **交友**：学习他人优点，取长补短。
5. **观看节目**：学习脱口秀、相声等节目，学习说话技巧。

## 总结
**6个基本功和5个方向，我们就可以同时去进行啊，也要在实战中不断的去锻炼。**
本节课中我们一起学习了如何通过强化锻炼和知识储备来提升高情商聊天口才。希望同学们能够将所学知识应用到实际生活中，不断提升自己的口才能力。
