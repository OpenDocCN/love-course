# 社交情商训练营：06_2-1、高情商聊天口才修炼2 💬

在本节课中，我们将要学习如何通过持续锻炼和有效输入来提升口才。核心在于理解口才提升是一个长期过程，并掌握积累知识储备的五个具体方向。

## 一、口才提升需要长期坚持 🔄

强化锻炼无法一蹴而就。为了锻炼口才，我曾每天对着镜子自言自语，经常练习到声音沙哑。经过长期坚持，才达到今天的水平。

要知道，我曾是一个非常内向、近乎自闭的人。我的原生家庭不幸福，父母很早就离开，我独自在学校生活，兄弟姐妹都在村里。这导致我在学校经常被欺负，无人帮助，从而形成了内向的性格。

你们的起点肯定比我好得多。因此，不要急于求成，务必稳扎稳打。口才是需要终身修炼的技能，一旦练好，将使你终生受益。

## 二、巩固六个基本功 🏋️

上一节我们介绍了口才的六个基本功。练好这六个基本功，你的口才就算基本合格了。

但是，如果你想更进一步，就需要储备更多素材。有一句话叫：**输入决定输出**。

## 三、通过五个方向进行有效输入 📥

本节中我们来看看如何有效增加知识储备。观察那些在演讲台上口若悬河、谈笑风生的人，他们的能力并非与生俱来。他们也是通过大量锻炼，储存了足够的知识量，才能在输出时如此从容。

如果你想和他们一样，就必须学会储备足够的知识。懂得越多，你在输出内容时才能源源不断、滔滔不绝。

那么，我们应该如何输入这些内容呢？想要增加内容储存量，可以从以下五个方向入手：

以下是增加知识储备的五个具体方向：

1.  **学会读书**：广泛阅读各类书籍。我个人比较推荐心理学书籍，因为心理学探讨人性。洞察了人性，未来的人际交往会更加顺畅。
2.  **关注热门事物**：清楚了解当下的热门事件，紧跟时代步伐。
3.  **多出门走走**：根据自己的经济能力，尽可能多去不同的地方，以此提高自己的眼界和格局。
4.  **多交朋友**：每个人身上都有值得学习的地方，要学会取其精华。
5.  **观看口才类节目**：多观看脱口秀、相声等与口才相关的节目。要以学习的心态观看，思考“他为什么这么能说？”，而不仅仅是娱乐。

## 四、总结、反思与实战应用 📝

做好以上五个方向后，还要学会不断总结和反思。把所有对提高口才有帮助的内容都记录下来，然后**重复转化、重复锻炼、重复学习**。这样，你的内容储备量就会越来越多。

**六个基本功**和**五个输入方向**可以同时进行。最重要的是，要在实战中不断锻炼和应用所学。
