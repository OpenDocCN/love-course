# 老白《约会达人》第22课：领袖小游戏——小姐牌陪酒员 🃏

在本节课中，我们将学习一个经典的酒桌互动游戏——“小姐牌”（又称陪酒员游戏）。这个游戏规则多样，能有效活跃聚会气氛，非常适合在朋友或同事聚会时由你来带领大家玩耍。我们将逐一讲解从A到K每张牌所代表的独特规则。

## 游戏概述与准备

上一节我们介绍了游戏的整体概念，本节中我们来看看具体的准备工作。

游戏通常使用一副完整的扑克牌，无需特意挑选。参与者围坐，由一人负责发牌。每人依次抽牌，并根据抽到的牌面执行相应的指令。

以下是游戏的核心流程：
*   洗牌并准备好一副扑克牌。
*   参与者依次抽牌，抽牌者根据牌面规则行动。
*   大部分指令涉及饮酒，可自行定义“一杯”的份量。
*   游戏持续进行，直到大家尽兴。

## 详细牌面规则解读

了解了基本流程后，我们进入核心部分，详细解读每张牌的含义。

### A：点杀牌 🎯
抽到A的人成为“杀手”。他/她需蒙上眼睛（或假装），然后随意指向一人并问“喝不喝？”，被指者若回答“喝”，则需饮酒；若回答“不喝”，则“杀手”继续指向下一个人。直到有人答“喝”，则此人饮酒，且需说出喝几杯。之后，“杀手”身份解除。

### 2：小姐牌（陪酒员） 👧
抽到2的人成为本局的“小姐”。从此刻起，直到下一张2被抽出前，任何人需要喝酒时，都可以喊“小姐陪一个”。此时“小姐”必须起身陪同饮酒，并通常要说“大哥/大姐吃好喝好”等敬酒词。**核心规则**：只要被叫到，“小姐”不能拒绝。

### 3：逛三园 🌳
抽到3的人发起游戏。他/她需首先选定一个类别，如“动物园”、“水果园”或“植物园”（可自定义如“KTV里有什么”）。然后从发起者开始，每人按顺序快速说出该类别下的一个事物，不能说重复或不符合类别的内容，犹豫或说错者需饮酒。

### 4：单挑牌 🤼
抽到4的人可以指定任意一人进行单挑。两人通过猜拳、骰子或其他任何方式决出胜负，败者饮酒。

### 5：照相机 📸
抽到5的人拥有一次喊“照相机”的权利。他/她可以在任意时刻突然喊出“照相机”，此时所有人必须保持静止。最后一个还在动的人，或者动了的人，需饮酒。若无人明显移动，则几秒后由喊话者宣布“时间到”，所有人安全。

### 6：柳树扭一扭 🌿
抽到6的人需说一段关于“柳树扭一扭”的绕口令。例如：“`一棵柳树扭一扭，两棵柳树扭两扭……`”以此类推，必须清晰地说出“柳”和“扭”。说错、卡壳或嘴瓢者需饮酒。

### 7：逢7过 🔢
抽到7的人，从任意数字开始（如1或10），大家按顺序报数。**遇到数字本身含7（如7、17、27）或是7的倍数（如14、21、28）时，必须拍手或说“过”跳过**。反应错误或说错数字者需饮酒。
**代码示例**：判断一个数是否需跳过的逻辑可以是 `if num % 7 == 0 or '7' in str(num): say("过")`。

### 8：厕所牌 🚽
抽到8的人获得“厕所通行令”。在下次抽到8的牌出现之前，只有持有此牌的人可以去洗手间。其他人如需离席，需罚酒或从持有者处借用此牌。

### 9：自饮牌 🍺
抽到9的人需自己喝一杯酒。当然，他/她可以要求“小姐”（如果已存在）陪同。

### 10：神经病牌 🤪
抽到10的人成为“神经病”。从此刻起到下一张10出现前，所有人都不能主动与他/她说话或回答他/她的问题。任何人与“神经病”交谈或回应，则交谈者需饮酒。“神经病”常会故意设套提问，引诱他人上当。

### J：左边喝 👈
抽到J（勾）的人，其左手边的人需喝一杯酒。

### Q：右边喝 👉
抽到Q（圈）的人，其右手边的人需喝一杯酒。

### K：国王牌 👑
K牌规则稍复杂，通常分四轮进行：
1.  第一张K：抽到者先自饮一杯，然后为**下一张K**的抽到者定下饮酒规则（如“喝三杯”）。
2.  第二张K：抽到者需喝下上一张K所定的酒量，喝完后，再为**第三张K**的抽到者定下新的饮酒规则。
3.  第三张K：抽到者需喝下第二张K所定的酒量，然后为**第四张K**定规则。
4.  第四张K：抽到者需喝下第三张K所定的酒量，通常还需额外完成一项大冒险或喝一大杯，作为一轮的结束。

## 游戏技巧与总结

我们已经看完了所有牌面的规则，最后来谈谈如何带领好这个游戏。

这个游戏的精髓在于规则多样、互动性强，能迅速炒热气氛。它非常适合在聚会冷场、大家不知玩什么时由你主动提出。作为组织者，你需要清晰讲解规则，掌控游戏节奏，并在大家熟悉后可以适当加入自定义规则增加趣味性。

**本节课中我们一起学习了**“小姐牌”游戏从A到K的全部规则。记住，**核心在于灵活运用这些规则来创造欢乐的互动**。你可以通过担任发牌者和规则解释者，自然成为场上的领袖，带领朋友们享受一个愉快的聚会时光。现在，就找副扑克牌，尝试组织一局吧！
