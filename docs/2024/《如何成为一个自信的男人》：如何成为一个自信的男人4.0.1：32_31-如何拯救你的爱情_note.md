# 《如何成为一个自信的男人》：如何成为一个自信的男人4.0.1：32_31-如何拯救你的爱情

大家好，我是赖伊。今天我们将专门探讨如何挽回你的爱情。为何要单独用一节课来讲解挽回？因为发现很多学员有这方面的需求。

市面上多数情感机构的挽回成功率较低，大约15%。而我服务过的学员挽回成功率可达85%。为何差距如此之大？关键在于我们依靠的是知己知彼，见招拆招的技术。

许多情感机构依赖的是赌博式的运气。即使运气好，挽回关系，下次可能更严重，导致分手或离婚。

因此，有必要分享挽回流程。首先，要搞清楚对方为何与你分手，是哪件事让对方忍无可忍。很多人挽回时连这一点都没搞清楚，盲目努力，反而越努力越糟。

如何做到这一点？简单，就是详细查看分手前对方发给你的信息、短信、语音、视频，逐字逐句找到对方生气的点，尤其是对方发脾气时说出的话，这些都是对方生气的真实原因。

为何很多兄弟挽回过程中屡屡碰壁？因为他们连对方真正生气的原因都没找到。如何挽回？打电话或发消息时，要针对对方生气的点进行回应。

纠缠几次后，对方的耐心会被消耗殆尽，自然而然对你只能说再见。挽回的第二步是复联。很多兄弟会遇到被对方删除、拉黑甚至全网拉黑的情况。

想要复联，记住这句话：永远不要主动与对方冷战断联。因为这会消耗感情和信任感。如果你经常主动与对方冷战断联，即使对方接受你的道歉，与你和好，但你们之间的信任感也会崩塌。

久而久之，总有一次狼真的来了。所以，不要轻易删除、拉黑对方。只要你能做到这一点，情绪稳定后，主动给对方发消息即可，不用管对方是否回复，不回复第二天再发。

当然，如果分手时间较长，一周发三条左右即可。决定你们是否能够复联，不仅取决于你是否主动联系对方，还取决于你发的消息是否有力量感，能否打动对方。

如果对方看完消息后气消了一半，那么你的消息就是好的。持续不断发送，复联就是板上钉钉的事情。如果对方没有回应，甚至再次拉黑你，说明你的消息没有起到作用。

下个章节我们会讲解红旅人的三个原则，按照这个原则去写，基本上就没有搞不定的事情。如果你觉得自己不会写，可以私下找我一对一私教，我手把手带你挽回，教你编辑这些信息。

挽回的第三步是恢复日常聊天，让对方重新养成与你分享日常的习惯，重新建立沟通欲和分享欲。在这个过程中，不要急于道歉、恢复关系，也不要把聊天节奏引导到痛苦的情绪上。

多分享一些愉快的事情。当然，对方也会多次吐槽你，向你发泄情绪。你要理解，这是好事。这时，你要做的就是按照我们上面提到的红旅人的三个原则，继续与他周旋。

等到他把心里的不愉快倾诉出来，你们的感情也就和好了。挽回的第四步是在对方释放完所有不愉快之后，看到时机成熟，再邀约见面。

见面的第一时间拥抱、亲吻、牵手。只要做到这一点，恭喜你，挽回成功。挽回的第五步是后撤。对于一些兄弟来说，在挽回的第三个阶段，可能会遇到推进不下去甚至反反复复被删除拉黑的情况。

这时，你唯一能做的就是后撤，最后发三条类似“我很爱你，但我坚持不下去了，我打算放弃你了，再见再也不见”的信息，也就是很多机构常用的冷战锻炼。

这种消息至少间隔一两个星期。这条短信的作用是利用前期你不断的偏爱以及人都害怕失去的心理，去刺激对方，抓住你。

如果这条消息发完，对方没有任何回应，那么只能说明对方对你实在没有期待，或者确实非常恐惧你，急于离开你。这时，你就不用再纠缠了，放过对方也放过自己。
