# 清风老师合集：02武功秘籍13期：018.第八课：眼神，不说话就让他疯狂迷恋你

在本节课中，我们将学习如何运用眼神来吸引他人，无需言语即可让对方对你产生强烈的兴趣。

## 眼神的力量

**公式**：眼神 = 心灵的窗户 + 情感的传递

### 眼神类型

以下是几种不同类型的眼神及其适用场景：

1. **害羞的眼神**：适用于调情阶段，表现出一种害羞、欲言又止的状态。
2. **放松惬意慵懒的眼神**：适用于初期互动，给人一种轻松愉悦的感觉。
3. **性感的眼神**：适用于进一步互动，通过放大卧蚕和放电动作来吸引对方。

### 眼神训练

以下是一些训练眼神的方法：

1. **手指画圈**：用手指在眼前画圈，同时用眼神跟随手指的运动。
2. **遮住额头眨眼**：遮住额头，练习眨眼动作，使眼神更加有神。
3. **穿透手指**：将手指放在面前，想象眼神穿透手指，看到背后的东西。
4. **手指快速移动**：将手指从远到近快速移动，训练眼神的变化和灵活度。

### 眼神运用

在不同阶段，运用不同的眼神：

1. **前戏阶段**：释放慵懒松弛的感觉。
2. **推进阶段**：表现出快乐、诱惑的眼神。
3. **进展阶段**：给予对方满足感，引导对方投入。
4. **高潮阶段**：给予对方崇拜或欣赏的眼神。
5. **结束阶段**：根据对方类型，给予崇拜或怜爱的眼神。

### 引导投入

在对方快乐满足的状态下，通过眼神引导对方为你付出：

1. **展示期待**：用期待的眼神表达你对某物的渴望。
2. **引导付出**：在对方表示愿意为你付出时，给予肯定和认可。

## 总结

本节课中，我们学习了如何运用眼神来吸引他人，无需言语即可让对方对你产生强烈的兴趣。通过眼神的训练和运用，我们可以更好地与他人沟通，建立更深层次的情感联系。
