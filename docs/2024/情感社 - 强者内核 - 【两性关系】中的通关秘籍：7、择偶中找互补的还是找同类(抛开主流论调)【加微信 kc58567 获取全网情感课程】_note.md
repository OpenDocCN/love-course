# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：7、择偶中找互补的还是找同类(抛开主流论调)

## 概述
在本节课中，我们将探讨在择偶过程中，是应该寻找互补的伴侣还是寻找同类的伴侣。我们将运用商业逻辑来分析这一话题。

## 核心概念
**互补**：指在性格、能力或生活方式上相互补充，例如一个稳定而另一个不稳定。

**同类**：指在价值观、生活方式和生活节奏上相似的人。

## 教程内容

### 1. 互补的伪命题
大众眼中的互补，本质上是一种伪命题。例如，一个稳定的工作和一个不稳定的工作，看似互补，但实际上并不一定能够长久稳定地生活在一起。

### 2. 真正的互补
真正的互补是指双方拥有共同的目标和方向，只是在分工上有所不同。例如，一个公司的拓展型和内勤型员工，他们的目标是一致的，只是在具体工作上有所不同。

### 3. 同类的优势
同类伴侣之间更容易理解和沟通，因为他们拥有相似的生活节奏和价值观。这种相似性有助于建立稳定和积极的亲密关系。

### 4. 三观吻合
三观吻合是指双方在价值观、生活方式和生活节奏上相似。这种相似性有助于建立稳定和积极的亲密关系。

### 5. 择偶选择
在择偶过程中，选择同类伴侣比选择互补伴侣更为重要。因为同类伴侣之间更容易理解和沟通，更容易建立稳定和积极的亲密关系。

## 总结
本节课中我们一起学习了在择偶过程中，选择同类伴侣比选择互补伴侣更为重要。同类伴侣之间更容易理解和沟通，更容易建立稳定和积极的亲密关系。
