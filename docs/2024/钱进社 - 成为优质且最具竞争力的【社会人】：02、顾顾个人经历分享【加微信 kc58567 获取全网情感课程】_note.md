# 钱进社 - 成为优质且最具竞争力的【社会人】：02、顾顾个人经历分享

## 1. 初入职场

顾顾的第一份工作是在一家广告公司担任前台。她之前在餐厅打工，但那只是为了养活自己。广告公司前台是她职业生涯的第一份真正的工作。

**核心概念**：**职业发展** - 从餐厅服务员到广告公司前台，顾顾开始了她的职业生涯。

## 2. 老板的影响

顾顾的老板是一位北京人，在上海做公司并定居。他对顾顾影响很大，尽管他有时会骂人，但顾顾认为这是对他的一种深刻解读。

**核心概念**：**领导风格** - 老板的骂人风格对顾顾产生了深远的影响。

## 3. 销售之路

顾顾在广告公司前台工作期间，表现出色，最终被破格晋升为销售。她认为，做事情切勿好高骛远，要脚踏实地。

**核心概念**：**职业晋升** - 顾顾通过努力工作，从前台晋升为销售。

## 4. 销售挑战

顾顾在销售工作中面临许多挑战，包括处理人际关系、应对客户压力等。她认为，一个人必须具备探索和死磕的精神。

**核心概念**：**销售技巧** - 顾顾分享了她在销售工作中积累的经验和技巧。

## 5. 职场成长

顾顾在职场中不断成长，她认为，做事时要将头埋在土里，傲气要藏在骨子里。

**核心概念**：**职场成长** - 顾顾分享了她在职场中成长的经验。

## 6. 转型与反思

顾顾在广告公司工作期间，不断反思自己的职业规划。她认为，人一定要反思自己，认清自己。

**核心概念**：**职业规划** - 顾顾分享了她在职业规划方面的思考。
