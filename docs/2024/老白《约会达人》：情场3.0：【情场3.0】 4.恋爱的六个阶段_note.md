# 老白《约会达人》：情场3.0：【情场3.0】 4.恋爱的六个阶段 🎼

## 恋爱的六个阶段

### 概述

在本节课中，我们将学习恋爱的六个阶段，了解每个阶段的特点和应对策略。

### 第一阶段：认识阶段 (L1)

**认识方式**：

- 天生缘分
- 朋友介绍
- 搭讪
- 社交软件
- 相亲

**核心概念**：

- **公式**： 认识 = 天生缘分 + 朋友介绍 + 搭讪 + 社交软件 + 相亲

### 第二阶段：熟悉阶段 (L2)

**熟悉程度**：

- 女生放下手机，还能想起你。
- 女生不需要看聊天记录，能知道你是个什么样的人。

**核心概念**：

- **公式**： 熟悉 = 放下手机 + 记住特点

### 第三阶段：吸引阶段 (L3)

**核心目标**：

- 展示高价值
- 让女生对你产生感觉

**核心概念**：

- **公式**： 吸引 = 高价值展示 + 感觉

### 第四阶段：追求阶段 (L4)

**核心需求**：

- 可靠
- 真心喜欢

**核心概念**：

- **公式**： 追求 = 可靠 + 真心

### 第五阶段：诱惑阶段 (L5)

**核心目标**：

- 升高关系
- 享受甜蜜的二人世界

**核心概念**：

- **公式**： 诱惑 = 语言升高 + 肢体接触

### 第六阶段：相处阶段 (L6)

**核心目标**：

- 保持和升温关系
- 走向婚姻殿堂

**核心概念**：

- **公式**： 相处 = 保持 + 升温

### 总结

本节课中，我们一起学习了恋爱的六个阶段，了解了每个阶段的特点和应对策略。希望这些知识能帮助你更好地理解和应对恋爱关系。
