## 核心概念：什么是“精修”？ 🎯

上一节我们介绍了课程目标，本节中我们来看看核心概念“精修”的含义。

“精修”指对原始图片进行精细化后期处理。其核心目标是**提升视觉美感**与**传达特定情绪**，而非彻底改变图片内容。这个过程可以类比为用代码调整图像参数：

```python
# 精修的核心是调整参数，而非替换像素
adjusted_image = adjust(original_image, parameters=[曝光, 色彩, 构图])
```

理解这个概念是进行所有后续操作的基础。

## 精修前的准备工作 🛠️

在开始动手修图前，充分的准备能事半功倍。以下是需要完成的准备工作清单。

*   **选择合适软件**：手机端可使用Snapseed、醒图；电脑端推荐Adobe Lightroom或Photoshop。
*   **分析原图问题**：在修图前，花一分钟找出图片在**曝光、色彩、构图**上的主要缺点。
*   **明确修图风格**：确定想要的效果，如“清新日系”、“复古胶片”或“简约高级”。
*   **备份原始文件**：务必保留原图，方便随时重新调整或对比效果。

完成这些准备后，我们就可以进入正式的修图流程了。

## 精修四步核心流程 🔄

则明精修法的核心是一个四步循环流程。上一节我们做好了准备，本节中我们来看看具体执行的四个步骤。

这个流程遵循从整体到局部、从校正到创意的顺序。

1.  **基础校正**
    *   调整**曝光**和**对比度**，让画面明暗正常。
    *   校正**白平衡**，确保白色物体在画面中呈现为纯白，消除色偏。
    *   使用**透视裁剪**工具修正歪斜的建筑或地平线。

2.  **全局调色**
    *   在**HSL面板**中单独调整某种颜色的色相、饱和度和明度。
    *   例如，让天空的蓝色更纯净，或让草地的绿色更鲜嫩。
    *   公式可以理解为：`最终色彩 = 基础色彩 + (色相偏移, 饱和度增益, 明度增益)`

3.  **局部优化**
    *   使用**径向滤镜**或**调整画笔**对特定区域进行提亮、压暗或调色。
    *   常用于突出人物面部、增强眼神光，或压暗四周以突出主体。

4.  **输出与锐化**
    *   根据朋友圈上传的压缩特性，适当增加**锐化**和**降噪**。
    *   导出时选择sRGB色彩空间和合适的尺寸（如长边1080像素），以平衡画质与加载速度。

每一步都是对图片的精细化雕琢，四步完成后，图片质感将得到显著提升。

## 常见场景实战技巧 ✨

掌握了核心流程后，我们将其应用到几个常见场景中。以下是针对人像、美食和风景照的具体技巧。

*   **人像照片**
    *   重点：皮肤质感与气色。
    *   技巧：在HSL中降低**橙色饱和度**并提高**橙色明度**，可让皮肤更通透。使用局部工具提亮眼白与牙齿。

*   **美食照片**
    *   重点：激发食欲感。
    *   技巧：适当增加**暖色调**（黄色、红色），提升**对比度**和**饱和度**，让食物看起来更鲜亮诱人。

*   **风景照片**
    *   重点：层次与氛围。
    *   技巧：使用**渐变滤镜**平衡天空与地面的曝光。增强画面中的**主体色彩**（如日落时的橙红色），削弱杂乱色彩。

将这些技巧融入四步流程，你就能应对大部分朋友圈图片的精修需求。
