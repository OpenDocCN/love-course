# 林老头《系统认知》：007.（核心必看）机器人模式、内耗、心力消耗、筛选

## 概述
在本节课中，我们将探讨在社交和情感领域，如何通过机器人模式、降低内耗、提高心力和筛选合适对象来提升社交效率。

## 机器人模式
**定义**：机器人模式是一种在前期聊天和收集资源阶段采取的聊天方式，旨在降低心理消耗，专注于收集信息。

**应用场景**：
- 刚认识新朋友或通过社交软件搭讪时。
- 在前期聊天阶段，如刚加上的女生或通过相亲、剧本杀等活动认识的人。

**操作方法**：
- 不在乎回复质量，不关心对方感受，以客服式聊天。
- 例如，发送“hello”后等待对方回复，不追问解释。
- 通过大量发送打招呼信息，如点赞、评论等，收集潜在资源。

**优点**：
- 降低心理消耗，避免内耗。
- 提高收集资源的效率。

## 内耗与心力消耗
**定义**：内耗是指过度关注细节，导致心理消耗过大，心力下降。

**表现**：
- 精神疲惫，缺乏动力。
- 对社交活动感到厌烦。

**原因**：
- 过度关注对方感受，希望得到认可。
- 在无关紧要的事情上耗费过多精力。

**应对方法**：
- 降低心理消耗，专注于重要事情。
- 将精力放在升级关系、表白等关键环节。

## 筛选
**定义**：筛选是指从收集到的资源中，挑选出适合自己的人。

**重要性**：
- 确保付出得到回报。
- 避免浪费时间精力。

**筛选标准**：
- 对方是否尊重自己。
- 是否有共同兴趣和价值观。

## 总结
本节课中，我们学习了机器人模式、内耗、心力消耗和筛选在社交和情感领域的重要性。通过运用这些方法，我们可以提高社交效率，找到合适的伴侣。希望这节课能帮助大家提升自己的社交能力。
