# 雨宸19套合集：18.《做聪明女人：永远把自己放在第一位》：003.成熟从“背叛”父母开始 🧠

在本节课中，我们将要学习一个关于个人成长的核心课题：如何通过与原生家庭的健康分离来实现真正的成熟。我们将探讨“不断奶”的几种表现，并学习四个具体的分离步骤。

---

很多人因为自身的不成熟，导致在亲密关系中会出现各种各样的问题。例如婆媳关系恶劣、妈宝男、扶弟魔等现象的出现，都是因为没有真正地与父母分离。而这节内容就是希望你能学会与父母分离。

子女真正的成熟是从“背叛”父母开始的。这里的“背叛”不是不孝，而是从内心和父母做一个告别，意味着你能够自己做出选择，并可以为自己的选择负责。

## 你真的“断奶”了吗？🤔

首先，我们需要审视自己是否已经真正独立。以下是四个判断你是否已经“断奶”的关键点。

### 1. 长不大的“巨婴”
社会上“妈宝”一直是热门话题。这类人在心理上始终把自己当成一个孩子，认为世界都要围着自己转，父母就该为自己奉献一切。这源于婴儿时期的“共生关系”。

**公式：婴儿期共生关系 → 个体无外界概念 → 视他人为服务自己的工具**

在婴儿时期，我们完全依赖父母成长，所有的食物和安全感都来源于父母。这个阶段的小孩没有“外界”和“他人”的概念，认为世界上只有自己。当一个人没有走出这种共生关系，他就不会把他人当作独立的个体去尊重，而是认为所有人都要为自己服务。

### 2. 挑剔的择偶标准
有些人选择另一半时非常挑剔，并非因为标准太高，而是因为他们根本没有自己的标准。他们的一切择偶标准都来源于父母的要求。

**核心逻辑：内在力量不足 → 拼命向外抓取认可 → 希望伴侣弥补缺失的力量 → 无法为自己负责**

因为他们内在力量不足，不够相信自己，不敢为自己的选择负责。他们希望找一个能帮自己做决定、为自己承担责任的人，就像依赖父母一样。他们的心态是：“都不是我的问题，都是你的问题。” 而一个成年人的基本心态是：**为自己做的事情负责和承担**。

### 3. 恶劣的婆媳关系
婆媳关系的好坏，80%的责任在男人身上。夫妻关系应该是每个家庭中最重要的关系。

**正确的关系排序：夫妻关系 > 亲子关系（与孩子）> 与父母的关系**

无法与父母分离的人，即便结婚后，也会把亲子关系（与父母的关系）放在第一位。这导致亲密关系中出现了“第三个人”，必然引发矛盾。健康的家庭需要明确的边界：你可以照顾父母，但他们不能干涉你核心家庭的事务。

### 4. 子女承担了父母的情绪
很多父母会因为自己未能给孩子提供更好的生活而感到愧疚。当孩子成年后状态不如意时，这种愧疚感会再次被勾起并传递给孩子。

**情绪传递链：父母感到愧疚 → 将情绪投射给子女 → 子女因父母的愧疚而感到更愧疚 → 形成情感负担**

于是子女常常觉得“我对不起爸妈”，并因此背负沉重的心理压力，难以真正轻松地生活。

上一节我们介绍了判断是否“断奶”的四个表现，它们都指向了与父母未分离的问题。接下来，本节中我们来看看如何具体地与父母进行健康的分离。

## 如何与父母健康分离？🛤️

与父母分离是一个双向的过程，需要技巧和耐心。以下是四个步骤。

### 1. 明确父母与自己的界限
比起改变父母，先改变自己更容易。可以从小事入手，主动树立边界。

**例如**：你可以主动结束与父母的长时间通话，让他们知道不能无限占用你的时间。明确什么是你可以和他们分享的（如日常问候），什么是你的私事（如具体的工作决策、恋爱细节），除非你主动询问，否则他们不应过度干涉。

### 2. 避免对父母过度欲求
分离是双方的事情。你不能一边说要独立，一边不断有事就寻求父母的帮助。

**关键点**：让父母尊重你边界的前提，是你也要尊重他们的边界，不能再要求他们把你的需求当作他们生命的中心。当你表现得精神独立、经济独立，父母了解到你的状态后，自然会逐渐放手。

### 3. 学会适当的“非暴力抵抗”
父母和我们的思想一定会有差异，不必强求他们完全认同我们的想法。

**做法**：学会用稳定的情绪面对不同。很多事情无需争一时意气，可以“左耳进右耳出”，同时温和而坚定地坚持自己的做法。父母也需要哄，说服不了时，不必强求改变他们。

### 4. 鼓励父母寻找自己的生活
鼓励父母学着“自私”一点，去寻找属于他们自己的生活中心。

**例如**：试着和他们沟通，彼此不要过多参与对方的生活。鼓励父母培养自己的兴趣爱好，如钓鱼、旅游、参加社区活动等。当他们有自己的生活重心，就不会将全部精力都放在你身上。

## 真正的成熟：为自己负责 ✅

当我们学会了分离，真正的成熟是什么？真正的成熟是自己为自己负责。

我们常常陷入纠结：选择爱情时害怕没有面包，选择面包时又担心没有爱情。这种纠结本身就是不为自己负责的表现，意味着我们什么都想要，却不愿为想要的东西付出代价。

**成熟心态公式：我选择 + 我享受权利 + 我承担后果 = 真正成熟**

奖赏一定伴随代价。我们之所以纠结，是因为无法对自己的选择负责，只想享受成果，却不愿承担错误选择的后果。当你不再陷入这种痛苦和纠结，能够坦然地说“不管结果如何，我自己承担”，那才是真正的成熟。
