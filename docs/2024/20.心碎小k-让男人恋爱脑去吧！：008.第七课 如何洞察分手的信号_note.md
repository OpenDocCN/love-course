# 20.心碎小k-让男人恋爱脑去吧！：008.第七课 如何洞察分手的信号

🎼第七课，如何洞察分手的信号。首先，如果你是正常人的话，你的第六感会非常准确。但是，我又不太相信你们是正常人。所以，还是给大家讲讲吧。

### 分手信号一：分享欲的消失

**公式**：分享欲 = 旧话题 - 新话题

分享欲的消失是分手的开始。从前，他可以在无聊的小事上与你畅谈，而现在，即使是有趣的事情，他也不会多与你分享。例如，以前吃麦当劳时，他会详细描述他的选择和感受，而现在，他只会简单地说“我吃完麦当劳了”。

### 分手信号二：缺乏正向情绪价值

**公式**：正向情绪价值 = 负面情绪 - 抱怨 - 不满

你们之间的聊天只有负能量，只有抱怨和不满，不再让你感到快乐。这种状态被称为“摆烂”。在这种时刻，记住，不要觉得自己是对方的快乐源泉，也不要觉得自己很强大。

### 分手信号三：不解决问题，冷暴力

**公式**：解决问题 = 逃避 - 漠视 - 冷处理

这是男人最喜欢的分手方法。他们逃避问题，不决绝，表现出自私、软弱和懦弱。他们会用冷漠的态度处理问题，让你陷入无尽的自我怀疑中。

### 分手信号四：加倍付出，计较付出

**公式**：付出 = 原始付出 - 加倍付出

他开始计较付出，像蜻蜓队长一样讲男女平等，讲公平。他过度索取原本属于你的权利，而你的本能反应是加倍付出，渴望得到回应。

### 分手信号五：看不惯你

以前他喜欢你的活泼开朗，现在却觉得你话多，甚至觉得你哪里都是问题。

### 总结

当你们开始自我怀疑，开始不快乐很长时间时，你们之间的感情就一定出现了问题。解决这些问题的方法只有一个：做好自己该做的事情，回归到单身时的状态。当你男人死了，你就该干嘛干嘛，半个月的时间，你就能看出到底是什么情况了。
