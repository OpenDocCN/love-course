# 凯哥《3899私教系列》：3.情商提升4.0：010.4.0挽回课：分开的真正原因及应对方式（一）

## 概述

在本节课中，我们将学习如何区分真性分手和假性分手，并了解应对分手的正确方式。

## 真性分手与假性分手的区别

以下是区分真性分手和假性分手的几个关键点：

**1. 联系方式**：

* **假性分手**：联系方式未被拉黑或删除。
* **真性分手**：所有联系方式被拉黑或删除。

**2. 情绪状态**：

* **假性分手**：情绪波动，可能通过冷淡或发朋友圈等方式引起注意。
* **真性分手**：长久冷淡，甚至无视，内心毫无波澜。

**3. 朋友圈**：

* **假性分手**：可能发无关紧要的朋友圈或鸡汤。
* **真性分手**：可能发自己玩得开心的朋友圈。

**4. 朋友圈**：

* **假性分手**：可能通过朋友、闺蜜、家人等圈子传递分手消息。
* **真性分手**：身边的人都知道分手消息。

## 应对分手的正确方式

**1. 假性分手**：

* **找出原因**：分析分手的原因，是当前争吵还是长期问题。
* **改变行为**：从行动上改变，而不是仅仅口头承诺。
* **关注情绪**：观察女生的情绪变化，及时调整自己的行为。

**2. 真性分手**：

* **分析原因**：了解分手的根本原因，是生活细节、性格、相处模式等问题。
* **改变思维**：学习正确的思维方式，避免逃避问题。
* **积极挽回**：展现出挽回的决心和态度。

## 总结

本节课中，我们学习了如何区分真性分手和假性分手，以及应对分手的正确方式。在处理感情问题时，我们要学会分析原因，改变行为，并展现出挽回的决心和态度。
