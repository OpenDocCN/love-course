# 清风老师合集：09不委屈，做个真正高情商的人：099.第四课：高情商沟通法，如何巧妙化解工作

在本节课中，我们将要学习如何运用高情商沟通技巧，巧妙化解工作中可能遇到的危机。我们将分析三种常见的工作困境，并学习如何运用激励能力，在不同社交模式中有效沟通，从而提升工作表现和人际关系。

上一节课程中，我们学习了如何避免自己成为一个总是委屈的人。本节中，我们来看看如何将高情商应用于工作场景，化解具体危机。

## 三种常见的工作困境

以下是缺乏情商时，工作中比较容易出现的三种困境。

**第一种困境：我只要做好自己的事情就可以了。**

我的一位朋友向我倾诉他的苦恼。他说他在公司工作了五六年，想要升职但老板不同意。他现在态度明确，就是只做好自己的事情，别人加班做不完与他无关，做完自己的工作就回家。我劝他，现在的就业形势不好，竞争激烈，这种心态虽然平和，但万一老板需要他多表现一点呢？他说他不管，只要做好自己的事就行。过了一段时间，他告诉我，老板以他工作不努力为理由将他辞退了。

这是很典型的一种困境。明明把自己的事情做好了，为什么老板觉得不够好？实际上，在老板和员工之间存在一种心理契约。高情商的人能够敏感地捕捉到这种心理契约。老板认为给你1万块钱，你要做12个小时的工作；而你认为给你1万块钱，你只要做8个小时的工作。这中间会有4个小时的出入，这是一种博弈。我们要用高情商的方式，让老板觉得你不用做这4个小时他也开心，同时让自己也感到舒服，不用为老板的额外期待而付出努力。

**第二种困境：我在职场里特别喜欢帮助别人。**

这是我刚进入职场第二年发现的问题。我去帮助别人解决问题，结果别人反而埋怨我，问我自己的工作不用做吗，为什么总关心他的问题。后来我发现，在帮助别人的过程中，我总是善意地指出他这里做得不好，那里做得不好。但指出多了，对方心里就像压着一块大石头。最后，有些问题实在解决不了，对方就会指责我这个不断指出问题的人，这样他就可以装聋作哑，听不到问题的存在了。

**第三种困境：我做好自己的事情了，而且做得很杰出，但领导没有看到我的成绩。**

这也是上一节课程中提到过的，你总是默默付出，但别人并不重视你的付出。这也需要我们用高情商的方式去化解。

## 运用激励能力完成工作

现在我们来利用高情商的第三种能力——激励，来完成我们的工作。激励分为三个维度。

**第一个维度：强大的成就驱动力。**

我们要找到自己做一项工作或一份事业时，内在真正的驱动力是什么。找到自己乐意为之努力的事情，这可以让我们在处理很多问题时，不自觉成为一个高情商的状态。

想要找到自己的成就驱动力，有一个方法。现在请你问自己一个问题：如果这个世界上所有的工作收入都是一样的，我会选择做一份什么样的工作？为什么是这份工作？

这里要举日本寿司之神小野二郎的例子。他终其一生都在琢磨和钻研怎么做寿司，因而成为了大家非常认可的工匠。我们要有一定的工匠精神。工匠意味着我做这件事情本身，我就感觉开心、感觉快乐，而不是简单地为了钱或某种目的把工作当成一个工具，而是在工作中就能感受到快乐。那么自然你会激发出一种高情商的状态。你和领导、同事、朋友交流时，都会有一种很舒适的感觉给到他们。

**第二个维度：卓越的使命感。**

使命感是驱动力的上一个台阶。我们再往下追问：为什么我会想去做这个工作？我曾经就问过自己这个问题。如果世界上所有工作收入都一样，我自己会愿意去做什么呢？想来想去，我愿意去当一个讲师。虽然我做了6年的心理咨询工作，但我更愿意当讲师。我非常热爱分享，将分享的知识去激发他人、改变他人的生活，对我来说是一件极其快乐和有成就感的事情。所以，这就是我人生的生命意义。

可能你一下子还没有找到自己真正想做的事情，我教你一个方法叫做“偶像法”。你可以找三个你最崇拜的、想要成为的人。然后，将他们所做的事情、他们的定位来作为自己的定位。

**第三个维度：百折不挠的乐观精神。**

乐观不是盲目的，而是遇到挫折时，可以看清自己，但不放弃努力。马云在创业初期寻找融资时，连续被27家机构拒绝。他回到团队里，当时的十八罗汉每次听到他回来，马云都是很正面地说：“今天我又拒绝了一家投资机构。”他在逆境中没有否定自我的价值，今天成就了阿里巴巴如此大的企业。我也希望你能够拥有这样一种能力，在工作遇到困难和问题时，不要怀疑你自己。

## 应用层面：三种社交模式中的高情商实践

那么我们怎么样运用情商中这种激励的能力，去帮助自己更好地完成工作呢？我先带你来了解一下社交最基本的三种模式：上升、平行和下沉。在我们的脱单第二课中，我有详细阐述过。

**上升模式：与领导的沟通**

在上升模式中，我们要跟领导展现我们的高情商，分为四个步骤。

第一步，关注领导的需求。理解领导到底想要什么。例如：“领导，您看这个东西是不是您想要的？”

第二步，主动分享自己的进度。领导的注意力非常有限，你无法控制他的注意力放到你身上。但当你主动分享你做得好和坏的时候，你就可以很好地让领导把关注给到你。

第三步，对于发现的问题提出改善意见，申请批准去执行。带着问题去向领导提问。领导喜欢的是有解决方案、并且能发现问题的人。这一类人在领导看来是有上进心、有进取心的人，也就是我们符合高情商、带有激励能力的人。

第四步，能够反馈真实的情况。在想要博得领导关注时，切记一点：不要报喜不报忧。因为在工作中，问题是常态，反而做得好是幸运的。最终结果好，不一定是你个人的成果，很可能是大家一起的努力。如果你只表达好的情况，领导会对你有不合理的期待，最终还是你自己自食恶果。

**平行模式：与同事的沟通**

第二种社交模式叫平行，就是你跟对方实力相当。那么对平级应该怎么说呢？

第一步，聆听同事的吐槽。同事有需求，他会先向你吐槽，而不是直接上来说“你这个事儿帮我怎么弄一下”，除非你们关系特别好。如果一个陌生的同事向你提出不合理的请求，你可以直接拒绝他。如果不知道怎样更好地拒绝，在我们专栏的第七课，我将会教你如何正确地拒绝他人。

当同事向你吐槽时，你要做的最主要就是聆听对方，千万不要跟他一起去攻击某个对象。比如同事说领导不好，你就跟他说“这个领导是个傻逼”，而是理解对方。例如：“我能理解这段时间你付出了很多，但似乎领导没有看到你的努力，你觉得很委屈。我觉得这样的情况很常见，我们要想办法找到问题的解决路径。你说呢？”

第三步，判断自己的职责范围。如果对方说的事情越过你的职责范围，要和对方讲清楚。例如：“这件事情我能理解，但我似乎在职责上面没有办法直接帮助到你。不过我想你可以通过这个方法去和别人沟通一下，或许他能给你建议。”

第四步，基于对方的真实需求提供帮助。在我们做心理咨询时，有一个很重要的原则：当对方有诉求时，我们才提供帮助，而不是拼命逮到一个人然后不断去提供帮助。基于对方的诉求提供帮助，对于我们自身在工作当中建立良好的同事关系也非常必要。老是去挑刺，有时会让人很不舒服。

**下沉模式：与下属的沟通**

最后，对我们的下属，在下层关系里，我们该怎么样体现高情商呢？

第一步，帮助下属认识自己。可以将我问你的两个问题问问他：他到底为什么喜欢这份工作，或者为什么想要做这份工作？这份工作对他的意义是什么？帮助他去找到自己的价值感。

你可能会说，下属是实习生，他做的工作就是很基础、很没有意义的，每天就是存存文件、发发报告。我分享一个来自麦肯锡咨询公司的案例，是帮助一位清洁工找到自己内在的价值。他们也同样问他：“你为什么要做清洁呢？”清洁工说：“就赚点钱嘛，然后给你们做清洁。”接着问他：“那清洁可以带来什么样的价值呢？”清洁工说：“那大家开心嘛，感觉舒服嘛。”接着他们将这个意义和他的工作做了关联：“对呀，你的工作本质上就是让人感到轻松和愉悦。”清洁工一听感觉就不一样了。同样，你也可以问问你的下属，他为什么要做这些工作，这些工作背后的意义是什么。

第二步，跟下属去拆解任务，帮他拆解。

第三步，设置错误的底线。让下属能够知道什么样的错可以犯，什么样的错不能犯。

第四步，通过错误，帮助你的下属成长。
