# 林老头《系统认知》：020.10、4个技巧高效升温关系，不油腻不猥琐

## 概述
在本节课中，我们将学习如何通过四个技巧高效升温关系，而不显得油腻或猥琐。

## 教程内容

### 1. 胆子大的一逼
**核心概念**：不要给自己加戏，勇敢尝试，不要害怕失败。

**公式**：实践是检验真理的唯一标准。

**步骤**：
- 不要害怕尝试新事物。
- 不要担心失败，因为失败是成功之母。
- 以实际结果为导向，不要过度思考。

### 2. 任何话题都可以亲密关系
**核心概念**：将任何话题与亲密关系联系起来。

**步骤**：
- 思考如何将话题引向暧昧或带有颜色的话题。
- 例如，讨论星座、情感、电视剧等两性话题。

### 3. 礼貌且尊重的冒犯和侵略
**核心概念**：在尊重和礼貌的前提下，进行越界的行为。

**步骤**：
- 注意礼貌和尊重，避免让对方感到不舒服。
- 例如，在聊天中暗示想要拉手，但先询问对方的感受。

### 4. 暗示和假如
**核心概念**：通过暗示和假如来升温关系。

**步骤**：
- 通过暗示和假如来增强彼此之间的吸引力。
- 例如，假设如果你们在一起，你会怎么做。

## 总结
本节课中，我们学习了四个技巧来高效升温关系，包括胆子大的一逼、任何话题都可以亲密关系、礼貌且尊重的冒犯和侵略以及暗示和假如。通过这些技巧，你可以轻松地升温关系，而不显得油腻或猥琐。
