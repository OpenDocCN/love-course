# 清风老师全套：8.彻底打败2+1，保护孩子保护自己13期：091.第九课：直面自己的爱人，建立有原则和底线的爱

## 概述
在本节课中，我们将学习如何与出轨的爱人进行有效沟通，建立有原则和底线的爱情关系。

## 谈话误区
以下是关于与出轨爱人谈话的三个常见误区：

1. **误区一：贬低攻击**
   - **错误观念**：认为出轨的男人一文不值，贬低攻击都是他活该。
   - **正确观念**：认识到问题本质，双方都有责任，避免贬低对方。

2. **误区二：过度控制**
   - **错误观念**：认为必须严格控制出轨者，如限制手机、经济等。
   - **正确观念**：适度控制，避免过度限制导致矛盾加剧。

3. **误区三：否定爱情**
   - **错误观念**：认为出轨者不懂爱，是对爱情的背叛。
   - **正确观念**：尊重对方，认识到真实爱情是复杂的，避免简单否定。

## 谈话准备
在谈话前，需要准备以下三件事情：

1. **自我反思**
   - 反思自身问题，从自己身上找原因，为改善关系做准备。

2. **了解小三**
   - 了解小三的优点，认清对方的长处，为战略打击做准备。

3. **保持平静**
   - 保持平和心态，避免极端情绪，通过冥想等方式放松情绪。

## 出轨者心理阶段
出轨者心理分为七个阶段：

1. **放松开心**
2. **后悔愧疚**
3. **想要反思回归**
4. **愤怒反抗**
5. **惊恐害怕**
6. **后悔反思**
7. **真心回归**

## 谈话结构
进行五个部分的谈话结构：

1. **表达反思**
   - 反思自身问题，表达对对方的理解和尊重。

2. **倾听意见**
   - 倾听对方的意见，了解对方的感受和期望。

3. **放大歉意**
   - 放大对方的歉意，让对方感受到你的真诚改变。

4. **引导补偿**
   - 引导对方做出补偿，如限制权利、写下承诺书等。

5. **录音录像**
   - 将道歉承诺过程进行录音或录像，作为法律证据。

## 总结
本节课中，我们学习了如何与出轨的爱人进行有效沟通，建立有原则和底线的爱情关系。出轨不可怕，可怕的是失去信心和期待。通过共同努力，我们可以解决存在的问题，重新建立关系。
