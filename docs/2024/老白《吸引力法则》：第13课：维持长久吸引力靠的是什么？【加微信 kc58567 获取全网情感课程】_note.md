# 老白《吸引力法则》：第13课：维持长久吸引力靠的是什么？

## 概述
在本节课中，我们将探讨如何维持长期恋爱关系，包括给予安全感、相互投资、保持个人生活重心、保持神秘感、制造危机、规划未来以及创造共同回忆等关键因素。

## 给予安全感
**核心概念**：安全感是长期关系中不可或缺的因素。
- **公式**：安全感 = 陪伴 + 信任 + 稳定性

**步骤**：
1. 在成为情侣后，多花时间陪伴女生。
2. 在校园内，与女生共同经历新鲜事物，如旅游、自习、健身等。
3. 让女生感受到你的存在，让她觉得你随时都在她身边。

## 相互投资
**核心概念**：相互投资是维系长期关系的关键。
- **公式**：相互投资 = 金钱 + 时间 + 感情

**步骤**：
1. 让女生为你做一些小事，如帮忙起床、拿外卖等。
2. 在经济上给予女生一些帮助。
3. 在生日时送她礼物。
4. 让女生为你投资时间和资源，让她感到责任。

## 保持个人生活重心
**核心概念**：拥有自己的爱好和追求，不要将爱情视为生活的全部。
- **公式**：个人生活重心 = 爱好 + 追求 + 事业

**步骤**：
1. 发展自己的兴趣爱好。
2. 不断提升自己，变得更加优秀。
3. 不要过于依赖爱情，保持独立。

## 保持神秘感
**核心概念**：保持神秘感可以增加吸引力。
- **公式**：神秘感 = 新鲜事物 + 发现

**步骤**：
1. 不断经历新鲜事物，走出舒适区。
2. 发现新的世界，不急于告诉女生。
3. 保持神秘感，让她觉得你始终有新鲜感。

## 制造危机
**核心概念**：制造危机可以增加女生的责任感。
- **公式**：危机感 = 竞争 + 稳定性

**步骤**：
1. 在不经意间向女生展示你的追求者。
2. 故意弄过女号，与女生聊天。
3. 发送与其他女生聚会的照片。

## 多规划未来
**核心概念**：规划未来可以增加女生的期待感。
- **公式**：未来规划 = 梦想 + 目标 + 画面感

**步骤**：
1. 培养梦想，让女生看到你们未来的美好。
2. 描绘接地气的未来，让她感到可期待。
3. 让她憧憬你们的未来。

## 制造只属于我们的回忆
**核心概念**：共同回忆可以加深彼此的感情。
- **公式**：共同回忆 = 经历 + 发现 + 留念

**步骤**：
1. 一起逛街、散步，走到一条特别的地方。
2. 互相给对方起一个独一无二的名称。
3. 一起养小宠物、搬出去住。
4. 推荐喜欢的歌曲给对方。

## 总结
本节课中，我们学习了如何维持长期恋爱关系，包括给予安全感、相互投资、保持个人生活重心、保持神秘感、制造危机、规划未来以及创造共同回忆等关键因素。希望这些技巧能够帮助你建立和维护一段美好的长期关系。
