# 安老师社恐训练营：18-练习：突破社恐的恶性循环(一)_note

在本节课中，我们将要学习一系列旨在打破社交恐惧恶性循环的练习。这些练习的核心在于**减少过度的自我关注**，并将注意力转移到外部世界或当下正在做的事情上。

## 练习原理与目标

上一节我们介绍了社恐的恶性循环模型，其中**自我关注过多**是一个关键突破口。本节中我们来看看如何通过练习来突破这一点。

人的精力是有限的，注意力无法同时完美地集中在两件事上。公式可以表示为：**注意力总量 = 自我关注 + 外部关注**。当自我关注过多时，用于处理外部事务的注意力就会减少，导致表现不佳。

因此，练习的目标是学会将注意力从自身（如担心出丑、紧张）转移到外部环境或当前任务上。

## 基础练习：注意力转移

以下是进行注意力转移的基础步骤：

1.  **内观感受**：先闭上眼睛，感受自己的呼吸和身体肌肉的紧绷与松弛。
2.  **外物观察**：有意识地睁开眼睛，将目光和注意力转移到周围任何你能看到的事物上。
3.  **持续锻炼**：初期可能只能持续几秒钟，之后逐渐延长到一分钟、三分钟、五分钟，循序渐进。

当你能够熟练地将注意力从自身转移到外部事物时，在紧张场景中，紧张感持续的时间就会缩短，你能更快地将注意力拉回到正事上。

## 核心技巧：给予清晰指令

关于转移注意力，有一个核心点：潜意识接受不了模糊的指令。仅仅“感受”某物，注意力很难持续。

我们可以用一个比喻来理解：杯子代表自我关注（如担心、紧张），电视代表你正在做的正事。

如何将注意力持续放在“电视”上呢？关键在于用清晰明确的语言在心中提问自己。例如：

*   这个电视是什么颜色的？
*   这个电视是什么品牌的？
*   电视里在播放什么内容？
*   主角是谁？

通过不断提出具体问题，你可以给潜意识清晰的指令，从而将注意力牢牢锁定在当前事物上。平时即可进行此类练习。

## 进阶练习一：54321感官法

此方法利用五种感官，将注意力从对过去或未来的思虑中拉回当下现实。随时随地都可以练习。

以下是练习步骤：

1.  **视觉（5样）**：说出你能看到的五样东西。例如：电视、窗户、水杯、绿植、书包。
2.  **听觉（4种）**：说出你能听到的四种声音。例如：键盘声、风声、交谈声、自己的呼吸声。
3.  **触觉（3种）**：说出你能触碰到的三样东西的感觉。例如：背部靠着椅背、脚踩在地面、手握着鼠标。
4.  **嗅觉（2种）**：说出你能闻到的两种味道。例如：咖啡香、书本的纸墨味。
5.  **味觉（1种）**：说出你此刻能尝到的一种味道。例如：水的清淡、口香糖的薄荷味。

## 进阶练习二：正念呼吸

如果你在锻炼时不知道如何集中注意力，正念呼吸是一个很好的起点。

具体做法是：有意识地去关注你的呼吸。感受呼吸的速度是快是慢，是呼气还是吸气，体会气息通过鼻腔的感觉。无论之前思绪多么纷乱，都可以通过关注呼吸将自己拉回当下。

这个练习对于未来快速调整呼吸、实现瞬间放松有很大帮助。

## 效果预期与总结

本节课中我们一起学习了三种核心练习：**注意力转移**、**54321感官法**和**正念呼吸**。大家务必落实并坚持锻炼。

坚持练习后，你会发现如同关闭了一个开关。以往，一旦紧张就会陷入恶性循环，难以自拔。现在，紧张症状可能只出现几秒，你便能迅速将注意力拉回，阻止循环开始。随着社恐问题逐步改善，那些令你困扰的生理症状（如手抖、心慌）也会随之减轻。

请抱着每天进步一点点的心态，你会发现自己进步神速。
