# 21.煜子显化开悟全程课(1)：14.操作指南-今后如何生活

🎼在完成前面的课程后，如何实施自己的显化目标？除了课程三中提到的普通方法外，你应明白活在当下的最终状态。但最重要的是如何实际操作。

**实操方法**

按照自己的意念和意识工作，简单来说，就是追随内心，也就是道自然。或者说，高我只会引导你去做，而不会有“不要这样做”的指令。因为“不要”的想法不再存在。

**熵增与伤减**

“不要”会导致恐惧和焦虑，因为多种意识和环境意识加入进来，使事情变得无序不可控。熵增是能量降低的过程，而伤减是能量提升的过程。

**跟随第一个念头**

当你听到一个念头时，不要去批判或分析，而是跟随它。高我或潜意识总是第一个出现的念头，因为它代表了你的当下状态。

**减少算命和观测**

在日常生活中，减少算命、观测和过度研究他人的喜好。因为你的问题都能被算出来，但算命结果并不总是准确的。

**显化与驱动力**

你的第一个念头通常是“要”而不是“不要”。喜欢某物是刻在你的运行代码中的，它会推动你前进。

**避免内耗**

不要过度内耗，专注于显化的结果而不是过程。显化是关于结果，而不是享受过程。

**总结**

本节课中，我们学习了如何实施显化目标，如何避免内耗，以及如何专注于显化的结果。希望这些知识能帮助你更好地实现你的目标。
