## 聊天极简框架

### 试探热度

在加女生之后，首先要试探她的热度。判断基准线的原因是每个人的属性不同，有的女生话多，有的则不然。

**公式**：试探热度 = 了解对方属性

### 红灯、黄灯、绿灯

#### 红灯

红灯表现为嗯啊哦或不回复，可能是因为对方不感兴趣或号码无效。

**公式**：红灯 = 嗯啊哦或不回复

#### 黄灯

黄灯表示对方会回应，但情绪很少，可能认为你是正常人但看不到你的价值。

**公式**：黄灯 = 回应但情绪很少

#### 绿灯

绿灯是对方对你感兴趣，有四个指标：兴趣、好奇、认可、舒适。

**公式**：绿灯 = 兴趣 + 好奇 + 认可 + 舒适

### 框架博弈

框架博弈是指对方回应但框架很硬，需要通过博弈来建立男对女的框架。

**公式**：框架博弈 = 对方回应但框架硬 + 建立男对女框架

### 价值展示

#### 投资价值

投资价值是指你现在很优秀，但将来更优秀。

**公式**：投资价值 = 现在优秀 + 将来更优秀

#### 情绪价值

情绪价值是指你给对方情绪上的价值，如调侃、开玩笑等。

**公式**：情绪价值 = 调侃 + 开玩笑

### 聊天推进

#### 时机

判断对方是否有表达欲，如工作日非假期的下午三四点可能不适合聊天。

**公式**：聊天推进 = 判断对方表达欲

#### 展示价值

聊天是展示你的价值，而不是产生吸引。

**公式**：聊天 = 展示价值

### 总结

本节课中我们一起学习了聊天极简框架，包括试探热度、红灯、黄灯、绿灯、框架博弈、价值展示和聊天推进等核心概念。希望这些知识能帮助你更好地与女生交流。
