# 雨宸19套合集：07.《撩汉说明书，让你爱的人爱上你》：006.主动出击：突破暧昧期，确定关系

## 概述
在本节课中，我们将学习如何通过主动出击，突破暧昧期，确定恋爱关系。

## 浮从度测试
**浮从度测试**是一种通过观察对方在关系中的态度和反应，来判断其在关系中的位置和态度的方法。

### 浮从度测试的步骤
1. **观察态度**：观察对方在关系中的态度，判断其是否愿意为你付出。
2. **判断位置**：根据对方的态度，判断其在关系中的位置，是否应该继续推进还是暂停观察。
3. **测试关系**：通过不同程度的测试，了解你们关系的远近。

### 浮从度测试的例子
例如，男女交往初期，从牵手到接吻，再到更亲密的关系，这些都是浮从度测试。

## 正向反馈
在废物测试中，不仅要索取，还要给予对方正向的反馈。

### 正向反馈的步骤
1. **表示感谢**：在对方为你做事后，表示感谢。
2. **增加互动**：通过请对方吃饭等方式，增加互动性。
3. **给予回报**：让对方看到你们关系的递进，或看到你正向的回应。

## 强弱推拉
**强弱推拉**是一种通过行为和语言上的推拉，引导对方主动表白的方法。

### 强弱推拉的步骤
1. **适当冷淡**：在暧昧期停留很久后，适当冷淡对方，减少见面次数。
2. **制造不稳定感**：通过偶尔放鸽子等方式，让对方感受到关系的不稳定。
3. **增加关注度**：当对方主动抓住你时，证明他对你投入更多，关注度更高。

## 注意事项
1. **适可而止**：不要让对方觉得你作，测试只是了解关系的一种方式。
2. **真诚表达**：在恋爱过程中，真诚地表达自己，勇敢地去做。
3. **避免套路**：不要将撩汉套路应用到恋爱关系中，保持真诚和真实。

## 总结
本节课中，我们学习了如何通过主动出击，突破暧昧期，确定恋爱关系。通过浮从度测试、正向反馈和强弱推拉等方法，我们可以更好地了解对方，推进关系。记住，真诚和真实是恋爱的核心，技巧只是辅助。
