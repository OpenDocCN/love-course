# 37、模块一《资源搜集》：08.模块一08 四大男性分类

在本节课中，我们将探讨四种不同类型的男性及其吸引的女性类型。

## 领袖型男性

**核心概念**：领袖型男性具备控制和领导群体的能力，通常具有权威、果断、干练、有能力、有魄力等特点。

**特点**：

* 控制和领导能力
* 权威
* 果断、干练
* 有能力、有魄力
* 肢体语言大方
* 眼神坚定
* 步伐缓慢有力

**领袖型男性的魅力点**：

* 在困境中帮助他人
* 无私
* 安全感
* 沉稳厚实

**领袖型男性的缺点**：

* 难以吸引感觉型女性
* 可能显得过于严肃

## 闪耀型男性

**核心概念**：闪耀型男性是社交场合的焦点，具有幽默、激情、想象力等特点。

**特点**：

* 社交场合焦点
* 幽默、激情
* 想象力丰富
* 适应能力强

**闪耀型男性的魅力点**：

* 引起注意
* 刺激感
* 经验丰富

**闪耀型男性的缺点**：

* 不靠谱
* 缺乏安全感

## 和平型男性

**核心概念**：和平型男性善于调节气氛，让人感到舒适，具有关心、体贴等特点。

**特点**：

* 调节气氛
* 舒适感
* 关心、体贴

**和平型男性的魅力点**：

* 情绪支持
* 舒适感
* 关心

**和平型男性的缺点**：

* 缺乏安全感
* 无底线容忍

## 踏实型男性

**核心概念**：踏实型男性稳重可靠，具有安全感、忠诚度等特点。

**特点**：

* 稳重可靠
* 安全感
* 忠诚度

**踏实型男性的魅力点**：

* 忠诚
* 安全感
* 责任感

**踏实型男性的缺点**：

* 过于保守
* 缺乏情调

## 不同类型男性吸引的女性类型

| 男性类型 | 吸引的女性类型 |
| :---: | :---: |
| 领袖型 | 勾引家、女教父 |
| 闪耀型 | 花蝴蝶、冰美人 |
| 和平型 | 邻家女孩、灰姑娘 |
| 踏实型 | 现代女性、鉴赏家 |

本节课中，我们学习了四种不同类型的男性及其吸引的女性类型。了解自己的类型和吸引的女性类型有助于我们在人际交往中更好地发挥自己的优势。
