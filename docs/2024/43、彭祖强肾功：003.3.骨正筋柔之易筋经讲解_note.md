# 彭祖强肾功 第43课：003.3.骨正筋柔之易筋经讲解 💪

在本节课中，我们将要学习易筋经的核心原理，理解“筋”与“骨”的关系，以及如何通过易筋经的练习来改善体态、延缓衰老并促进健康。

---

人衰老的过程是一个形体逐渐变化的过程。人在最年轻时，身形最为挺拔。到了一定年龄，便开始弯腰驼背。

人出现这种情况，是因为筋缩短了。筋长一寸，寿延十年。筋壮则强，筋缩则衰。一个人衰老的过程，就是筋开始慢慢缩短的过程。这时体型就会发生变化。

通过易筋经的洗炼，首先会发现体型有所改变。许多弯腰驼背、体型不振的人，通过训练后，身体会越来越端正。

筋辅阴，骨辅阳，血养筋，髓养骨。小孩的身体非常柔软。

小孩摔一跤，爬起来往往没事。因为他的筋骨都很柔软。到了一定年龄，姿势不对时，就容易颈椎或腰部不适，全身都不舒服。

这代表着身体内部出现了问题。必须明白，我们的骨骼是靠筋来固定的。当筋萎缩或老化，其牵拉作用就变差。这时就容易出现腰酸背痛或颈椎病。

因此要明白，阳先两肌。当筋的状态良好，骨骼才会处于正位。反之，筋若出现问题，骨骼就容易歪斜，导致骨质疏松或骨骼突出。二者是相辅相成的。

80%的慢性病由骨骼弯曲引起。骨骼之所以弯曲，是因为牵拉它的筋没有力量，无法将其固定在正确位置。

在这个过程中，要明白必须在生活中保持正确姿势。当许多姿势出现问题时，全身骨骼就会变形，进而导致筋变形。二者相互影响。骨骼会影响筋，筋也会影响骨骼。

跟随易筋经训练，会发现体型越来越好。务必不要让骨骼弯曲。因为我们的内脏都悬挂在骨骼上。骨骼弯曲时，就可能压迫内脏，导致脏腑功能下降。

观察一个人的体态，就能看出其健康程度。在这个过程中，要明白肾主骨，肝主筋。练习易筋洗髓功时，需知肝肾同源。通过易筋经改变筋骨，能让筋骨更强壮。

许多人的筋随着年龄增长，若缺乏练习，就会老化，牵拉力量越来越差。经过一段时间的练习后，筋骨会变得越来越柔软，越来越有弹性。

当筋骨变得柔软且有弹性时，身体状态才会好。许多男性必须明白，在进行双人运动时，生殖器也依靠筋来起支撑作用。当筋萎缩时，它也会萎缩。

在练习过程中，特别是后面的垂钓练习，就是在垂吊中筋，将其拉长。这时会发现生殖器直接被拉长。许多人练习垂钓后能增长2到3分，甚至更多。

这并非促进生长，而是将整个中筋拉开，使其更具弹力，而非缩进去。明白这个原理吗？身体处处皆为筋。在练习过程中，身体会越练越柔软。身体柔软后，气血才会运行良好。

易筋经是养肝的一种方法，必须跟随训练，简单有效。练习后会感觉全身轻松。易筋经，“易”意为改变，“筋”为上下左右之联络，一阴一阳，阴阳调和，刚柔并济。

“经”是什么呢？经就是筋经。常说“筋长一寸，寿延十年，骨正筋柔，气血自流”。“经”代表经典，即历经数千年被世人验证有效的东西。并非所有功法都能称为“经”或经典。

有幸与大家一起练习这么好的功法，一定要懂得珍惜。每天有时间，行、住、坐、卧皆可练习。无需刻意安排大量时间。学会后，在办公室坐久了，都可以站起来练一下。练完后会感觉全身舒畅。这也是延寿的一种方法。

当筋慢慢被拉长、变得柔软，气血就会充盈，运行就会良好。
