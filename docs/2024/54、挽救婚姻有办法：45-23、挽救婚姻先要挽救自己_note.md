# 54、挽救婚姻有办法：45-23、挽救婚姻先要挽救自己 🌟

## 概述

在本节课中，我们将探讨如何挽救婚姻，首先需要关注的是自身的状态和问题。

## 情绪失控的状态

以下是几种常见的情绪失控状态：

1. **情绪崩溃**：一说话就忍不住哭泣。
2. **极度愤怒**：一提到问题就非常气愤。
3. **无法工作**：整天想着老公出轨的事情。
4. **情绪失控**：说着说着就情绪失控，甚至砸东西。

## 认知层面的问题

以下是几种常见的认知层面问题：

1. **对错观念**：认为婚外情是道德败坏的表现。
2. **性别偏见**：认为老公做错事，老婆做对事。
3. **婚姻观念**：认为婚姻是一辈子的，离婚就活不下去。
4. **心理问题**：存在严重的心理问题或精神问题。

## 挽救婚姻的步骤

1. **处理情绪**：通过认知和行为来处理情绪。
2. **增加认知**：从新的角度看待问题，例如，出轨可能表明对方还有爱的能力和性能力。
3. **行为改变**：进行户外运动、参加聚会、接受辅导等。
4. **接受辅导**：寻求专业的心理咨询或婚姻家庭咨询。
5. **阅读书籍**：阅读专业书籍，如外国人写的书籍。

## 总结

本节课中我们一起学习了如何挽救婚姻，首先需要关注自身的状态和问题。通过处理情绪、增加认知、行为改变、接受辅导和阅读书籍等方法，我们可以更好地挽救婚姻。
