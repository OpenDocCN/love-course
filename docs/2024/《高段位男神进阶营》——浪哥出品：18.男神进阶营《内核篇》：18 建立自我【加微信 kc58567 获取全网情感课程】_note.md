## 核心概念

**内在身份**：指一个人对自己的认知和认同，包括价值观、信念、目标等。

**情感饥渴**：指对他人认可和关注的过度需求。

**选择性焦点**：指将注意力集中在自己身上，而不是外界。

---

## 建立内在身份

### 1. 活在当下

**公式**：活在当下 = 不为未来焦虑，不为过去难过

**代码**：放下过去，专注于现在，规划未来

**方法**：

* 冥想
* 体验大自然
* 专注于当下的事物

### 2. 建立正能量生活

**公式**：正能量生活 = 积极的社交圈 + 健康的爱好 + 人生目标

**代码**：

```python
# 建立正能量生活
def build_positive_life():
    # 建立积极的社交圈
    social_circle = create_social_circle()
    # 培养健康的爱好
    hobbies = develop_hobbies()
    # 设定人生目标
    life_goals = set_life_goals()
    # 返回正能量生活
    return social_circle, hobbies, life_goals
```

**方法**：

* 参加社交活动
* 培养兴趣爱好
* 设定人生目标

## 削减情感饥渴

### 1. 以自己为中心

**公式**：以自己为中心 = 关注自己，而不是他人

**代码**：

```python
# 以自己为中心
def be_centered():
    # 关注自己的需求
    focus_on_own_needs()
    # 关注自己的成长
    focus_on_own_growth()
    # 关注自己的快乐
    focus_on_own_happiness()
```

**方法**：

* 培养自信
* 关注自己的兴趣
* 享受自己的时光

### 2. 建立无需求感

**公式**：无需求感 = 不依赖他人认可

**代码**：

```python
# 建立无需求感
def build_no_needs():
    # 放下对他人认可的依赖
    let_go_of_needs_for_other_people()
    # 关注自己的价值
    focus_on_own_value()
    # 享受自己的存在
    enjoy_own_existence()
```

**方法**：

* 培养自尊
* 关注自己的成长
* 享受自己的时光
