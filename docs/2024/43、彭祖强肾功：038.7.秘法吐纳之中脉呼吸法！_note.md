# 彭祖强肾功：038.7.秘法吐纳之中脉呼吸法！

## 概述
在本节课中，我们将学习彭祖强肾功中的秘法吐纳，特别是中脉呼吸法，包括鼻腔震动和喉腔震动。

## 鼻腔震动
以下是鼻腔震动的步骤：

1. **站立姿势**：两脚分开与肩同宽，脚尖冲前，站正。
2. **屈膝坐胯**：屈膝同时坐胯，自然形成脸臀，收敛臀部。
3. **保持直立**：上身保持直立。
4. **收合长喉**：牙齿清叩，舌顶上腭。
5. **双手扶腰**：双手服腰。
6. **吸气**：用力提纲提稿完，同时骨盆做中摆式运动，膝盖保持弯曲，身体不前后晃动。
7. **呼气**：放松，骨盆卷曲，同时脸臀做胯。
8. **震动鼻腔**：吸气时，用力震动鼻腔，目视前方，比吸逼呼。

## 喉腔震动
以下是喉腔震动的步骤：

1. **站立姿势**：与肩同宽，屈膝做胯，上身保持直立。
2. **收合长喉**：牙齿清叩，舌顶上腭，鼻稀逼乎。
3. **吸气**：震动喉腔。
4. **膝盖外撑**：膝盖可以微微外撑，外撑脚底虚空，拇指抓地。
5. **左侧位**：两脚平行，屈膝做胯，上身保持直立，双手服腰。
6. **右侧位**：双脚平行，屈膝做胯，舒服练臀，上身保持自立，双手扶腰。

本节课中，我们一起学习了彭祖强肾功中的秘法吐纳，包括鼻腔震动和喉腔震动。通过这两种呼吸方式，可以有效改善男性泌尿生殖系统，包括男性在固精方面的问题。
