## 什么是情感？

上一节我们介绍了本课程的目标，本节中我们来看看情感究竟是什么。情感是对内部或外部重要事件产生的复杂反应模式，包含**生理唤醒、认知解释、主观感受和行为表达**四个部分。

一个简单的公式可以描述情感反应的过程：
**事件 -> 生理唤醒 + 认知解释 -> 主观感受 -> 行为表达**

## 情感的核心构成

理解情感的构成部分，能帮助我们像拆解机器一样分析情绪。以下是情感的四个关键组成部分：

1.  **生理唤醒**：身体对情绪的直接反应，如心跳加速、手心出汗。
2.  **认知解释**：大脑对事件和身体感觉的评估与解读。
3.  **主观感受**：个体体验到的独特情绪状态，如感到“愤怒”或“喜悦”。
4.  **行为表达**：情绪的外在表现，包括面部表情、语言和动作。

## 情感的主要功能

情感并非无用之物，它在我们的生存和社交中扮演着关键角色。以下是情感的三个主要功能：

*   **适应功能**：帮助我们快速应对环境威胁或机遇，如恐惧促使我们逃离危险。
*   **社交功能**：通过表达情感进行沟通，建立和维护社会关系。
*   **动机功能**：激发和引导我们的行为，如好奇心驱动学习。

认识了情感是什么以及为何重要后，本节我们将探讨提升情感识别能力的具体方法。以下是三个实用的练习步骤：

1.  **自我观察与记录**：每天花几分钟，用笔记或语音记录下你的主要情绪及触发事件。

3.  **观察与反思他人**：在社交中，注意他人的非语言信号（如表情、肢体语言），并礼貌地验证你的判断（例如：“你看起来有点沉默，一切都好吗？”）。
