# 雨宸19套合集：04.《冷暴力，别让爱情在沉默：001.远离冷暴力，让你的爱情

在本节课中，我们将要学习什么是冷暴力，以及它为何会对关系和个人造成深远的伤害。更重要的是，我们将探讨面对冷暴力时，如何通过自我爱护和有效方法来保护自己，并理解施暴者的心理，从而更好地规避未来的伤害。

冷暴力的伤害不仅限于破坏关系，它还会侵蚀受虐者的内在自我，让他们产生“不配被爱”的负面信念。但是，有问题就必然存在解决方法。

面对冷暴力，我们不仅要学会自救，更要学会自我爱护。本套课程共分为五节内容，不仅会帮助受虐者找到对抗冷暴力的有效方法，还会带你一起分析施暴者的内在心理。俗话说，知己知彼，百战不殆。

面对施暴者，我们不仅要反抗，更要清楚他们的心理活动。这样才能在未来的生活中更好地规避风险，尽可能远离潜在的冷暴力实施者，避免自己受到二次伤害。建议反复收听以加深理解。

以下是本节课的核心要点总结：

*   **核心伤害**：冷暴力伤害关系与个人自我价值感。
*   **核心方法**：**自救**与**自我爱护**是关键。
*   **核心策略**：理解施暴者心理（**知己知彼**），以更好地规避风险。
*   **核心信念**：不要因冷暴力而自我怀疑与否定。

本节课中我们一起学习了冷暴力的本质与危害，并初步了解了对抗它的核心思路：**不要害怕冷暴力，更不要因为它去怀疑和否定自己。所有暴力都敌不过一个强大的内在自我。** 当你真正接纳并爱护自己时，你便拥有了抵御一切伤害的力量。
