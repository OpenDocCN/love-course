# 乐嘉7套：7.乐嘉性格色彩读心术（完结）：020.【附赠】我身上纹的那只鹰害了我_note

在本节课中，我们将通过乐嘉分享的一段亲身经历，探讨一个重要的心理策略：如何通过“忽略”来应对困境，特别是病痛。我们将学习在战略上轻视问题，而在战术上重视问题的智慧。

## 概述：一个关于纹身与探望的故事

故事始于乐嘉创业初期的一位邻居阿布。他们相识于微时，互相帮助。多年后，阿布前来探望住院的乐嘉。整个探望过程非常特别，阿布没有谈论任何关于病情的话题，反而将注意力完全放在了乐嘉大腿根部的一个老鹰纹身上，并由此引发了一场关于纹身文化、社会偏见与心理应对的深入讨论。

## 纹身的困惑与社会偏见

上一节我们概述了故事背景，本节中我们来看看纹身引发的思考。阿布对乐嘉的纹身图案和位置感到好奇，这促使乐嘉回顾了自己对纹身的复杂情感。

乐嘉从小受古典文学影响，认为纹身是一种酷的人体艺术。
*   **早期向往**：十七八岁时觉得纹身很美，但怕疼且遭到长辈镇压。
*   **偏见形成**：长辈教育他“举凡纹身者必是恶人”，用简单的道德标准一刀切。
*   **成年后的反思**：乐嘉认识到，人有好坏之分，这与是否纹身无关。他见过善良的纹身者，也见过衣冠楚楚的害人者。

乐嘉认为，长辈的教育方式存在问题。他们未能理性分析利弊，而是灌输了一种非黑即白的错误观念。
```
错误教育示例：“纹身的都是坏人。”
理性教育示例：“纹身可能影响某些职业选择，且去除困难，需谨慎决定。”
```

## “奇怪”的纹身位置与职业考量

当阿布追问为何将老鹰纹在大腿根部时，乐嘉给出了基于现实职业的考虑。

乐嘉的主要职业是讲师，需要面对不同场合的听众。
*   **维护专业形象**：在政府或保守场合讲课，光头配上暴露的纹身可能引发不必要的抗拒。
*   **配合工作需求**：录制节目时，暴露的纹身会被要求遮盖。
*   **个人取悦**：纹在隐蔽处，是取悦自己而非展示他人。

因此，纹身位置的选择是理性权衡个人喜好与职业要求的结果。

## 纹身的“风水”与心理暗示

阿布并未止步于好奇，他提出了一个惊人的观点：乐嘉此次患病，可能与纹身图案和位置犯“忌讳”有关。

阿布引用东西方例子，说明纹身图案和部位大有讲究。
*   **错误类比**：他指责乐嘉在“蛋蛋”旁边纹老鹰，是“老鹰啄小鸡”，自破气场。
*   **提供方案**：建议找周易行家做“堪舆术”，通过补纹其他图案来转运。

这番听起来“有理有据”的话，让乐嘉瞬间感到不安。无论是否迷信，强烈的心理暗示足以影响人的状态。

## “忽略”策略的智慧与力量

然而，乐嘉随后意识到，阿布此次探望最核心的价值，并不在于纹身风水之说，而在于他无意中践行了一种高超的心理策略——“忽略”。

回顾整个探望过程，阿布的做法与众不同。
*   **全程如常**：他把乐嘉当作正常人，只聊纹身等闲事，对病痛只字不提。
*   **创造氛围**：他传递的感觉是“哥们儿，你只是受了点小苦，住段时间院”，而非“你得了重病，天塌了”。
*   **效果显著**：这种处理方式让乐嘉在阿布离开后，心情反而很好。

阿布的做法揭示了“忽略”策略的精髓：**在战略上藐视问题，在战术上重视问题**。
*   **战略忽略**：在心态和态度上，不把病魔当回事，避免自己吓自己，放大恐惧。
*   **战术重视**：在实际行动和治疗上，依然要认真对待，严阵以待。

这种方法特别适合那些病情本身并不严重，但被自身焦虑放大痛苦的个体。其目的是防止心理上的“雪上加霜”。

## 应用“忽略”策略的注意事项

当然，“忽略”策略需要谨慎使用，否则可能适得其反。

以下是应用此策略的两个关键前提：
1.  **关系足够亲密**：必须与对方知根知底，避免让对方产生“你不在乎我”的误会。
2.  **辨别真实需求**：需认真判断对方是“求安慰”还是真的病情严重。对于确实严重的病情，此策略不适用。

核心原则是：**不要放大你的病症，要学会在战略上忽略它。**

## 总结

本节课中，我们一起学习了乐嘉通过纹身故事引出的心理应对策略。我们从社会偏见谈到个人选择，最终领悟到“忽略”的智慧。关键在于区分战略与战术：在心态上举重若轻，忽略病痛带来的恐惧；在行动上视病如臣，认真进行诊疗。掌握好度，并在亲密互信的关系中运用，这种方法能有效帮助他人或自己稳定情绪，避免被负面心理暗示所困。记住，真正的强大，有时始于一次巧妙的“忽略”。
