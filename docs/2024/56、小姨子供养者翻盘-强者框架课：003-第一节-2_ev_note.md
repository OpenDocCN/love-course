# 强者框架课 003：生存价值、繁衍价值与情绪价值 🎯

在本节课中，我们将要学习两性关系中的三种核心价值：生存价值、繁衍价值与情绪价值。我们将逐一剖析其定义、来源及其在择偶行为中的具体体现，帮助你理解人类行为背后的深层逻辑。

## 生存价值：生存是第一要务 🛡️

所有物种的第一核心要务就是生存下去，人类也不例外。在远古时期，人们如同野兽一般，依靠捕猎维持生活。捕猎往往意味着搏斗、厮杀以及力量。只有拥有足够的生存能力，才能活下去。

在古代，生存能力表现为拥有很多食物、身体健壮拥有更好的打猎能力、能构建家园，或是成为一个部落的领袖并拥有食物的分配权等等。

但到了现代，生存能力发生了相应的变化。它不再以非常直接野蛮的形式出现，而是演化成了财富、社会地位、上进心等等。**生存能力的本质没有任何变化**。所以以上这些统称为一个男人的生存价值。

那么，生存价值对于择偶的意义到底有多大呢？我们可以来看一种小动物。

以下是关于博老鸟的观察实验：
*   有一种长相类似麻雀的小鸟，叫做博老鸟。
*   每到繁殖季节到来的时候，雄鸟就开始囤积蜗牛、老鼠、羽毛、布料一类的物品，从90件到120件不等。
*   然后，它会把这些物品挂在自己领地的树枝上，等待雌鸟的到来。
*   雌鸟通常就会看在谁的领地里面挂出来的物品最多，然后就会与这只雄鸟交配。
*   如果心理学家拿走这只雄鸟的物品，放到另一只雄鸟的领地里，这时雌鸟就会转而投入到另外一只拥有更多物品的雄鸟。
*   而刚才还是富甲天下，但是被心理学家搞破产的雄鸟只会落得形单影只。

这是不是能让你联想到在相亲的时候，女方要看男方有没有车子、房子，有多少存款呢？

这不仅仅是我们中国式相亲现象。在日本、赞比亚、南斯拉夫、澳大利亚等不同的国家，在白种人、黄种人、黑人等不同的种族，在西方文化、东方文化、阿拉伯文化等不同的文化都得到类似的结果，具有跨国家、跨种族和跨文化的一致性。

而同样作为雌性生物，女人在择偶时对于财富的强烈执念，其实与博老鸟并无差异。

明白了生存价值的动物性基础后，我们再来看一组人类世界的实验。

## 繁衍价值：基因延续的密码 🧬

进化心理学里最核心的观点是：人的核心任务是繁衍，从而让基因永存。该物种提升延续成功率的办法有两个：第一，增多交配数量；第二，提高交配质量。

增多交配数量，就是我们常说的“不要把鸡蛋放在一个篮子里”，以降低繁衍失败的风险。

而提高交配质量，意在提升单次交配的成功率。例如，男性更倾向于寻找胸大、屁股大的女性进行交配，是因为他们的基因里认为，这样的女性哺乳能力更强，生育风险更低，能孕育出更好的后代。而女性更倾向于寻找身材健硕、健康的男性进行交配，因为这样的男性意味着他们携带着更强大的基因，繁衍出来的后代也更容易在这个星球上生存下去。

所以，繁衍价值与生存价值不同。繁衍价值指的是你身上携带更优秀的基因。自然界通过上百万年的进化，将这些优质基因的要素，比如健康、聪明、高大、帅气、身材好等等，内化到我们的审美系统之中，把它们变成“美”的定义。表面上我们在追求美，但我们行为的本质却是为了基因的永存。

那么，女人对于优劣基因的态度差别到底有多大呢？我们通过一个实验来观察。

心理学家把几张男性的照片放在一起，请女性来评价照片中的男性有多大的魅力。

以下是实验中的四个选项：
*   **A**：一个与婴儿互动的男性照片。
*   **B**：一个对哭泣的婴儿视而不见的男性照片。
*   **C**：一个正在认认真真打扫卫生做家务的男性照片。
*   **D**：一个静静站立的男性照片。

筛选结果，90%以上的女人都对A男感兴趣。为什么呢？女性认为，跟婴儿互动的男性比忽视婴儿哭泣的男性更有魅力。这是因为女性更喜欢那些愿意与她共同抚养后代的男性。

但这些都不是最关键的。这个实验真正让男性警醒的是：**女性认为选项C，也就是正在打扫卫生、做家务的男性是最没有魅力的**。我们常说“顾家的男人是好男人”，但从基因延续的角度来看，女性需要的“顾家”不是做家务，而是能带来生存价值的男人。

这就是两性择偶中非常残酷的一面。女人选择的不仅仅是一个对她好的男人，更是他背后带来的物质生活。

**本节课总结**

在本节课中，我们一起学习了：
1.  **生存价值**：源于生存本能，在现代社会体现为财富、地位等资源获取与保障能力。
2.  **繁衍价值**：源于基因延续的本能，体现为健康、外貌等携带优质基因的生理特征。
3.  **情绪价值**：基于人性，涉及情感连接、理解与支持等精神层面的互动与满足。

理解这三种价值，是构建强者认知框架的基础。生存与繁衍价值是生物性的底层逻辑，而情绪价值则是人类关系得以升华的关键。
