# 小姨子供养者翻盘-强者框架课：003-第一节-2_ev

🎼生存价值、繁衍价值、情绪价值。前两者基于动物性，而最后一种基于人性。我们来逐一讲一下。

### 生存价值

🎼所有物种的第一核心目标就是生存下去，人类也不例外。在远古时期，人们如同野兽一般，依靠捕猎维持生活。捕猎往往意味着搏斗、厮杀以及力量。只有你有足够的生存能力，你才能活下去。在古代，生存能力意味着拥有很多食物，或者身体健壮，拥有更好的打猎能力，能构建家园，或者是一个部落的领袖，拥有食物的分配权等等。但到了现代，生存能力发生了相应的变化，不再以非常直接野蛮的形式出现，而是演化成了财富、社会地位、上进心等等。

🎼大极证明生存能力的本质没有任何变化。所以以上这些统称为一个男人的生存价值。

🎼生存价值对于择偶的意义有多大呢？我们可以来看一种小动物，有一种长相类似麻雀的小鸟，叫做博老鸟。每到繁殖季节到来的时候，这个熊鸟就开始囤积蜗牛、老鼠、羽毛、布料一类的物品，从90件到120件不等。然后，它会把这些物品挂在自己领地的树枝上，等待雌鸟的到来。此鸟通常就会看在谁的领地里边挂出来的物品最多，然后就会与这只熊鸟交配。如果心理学家拿走这只熊鸟的物品，放到另一只熊鸟的领地里，那这个时候此鸟就会转而投入到另外一只拥有更多物品的熊鸟。而刚汪呢还是富甲天下，但是被心理学家搞破产的熊鸟只会落得形单影直。这是不是能让你联想到在相亲的时候，女方要看男方有没有车子，房子有多少存款呢？

🎼其实这不仅仅是我们中国式相亲现象。在日本、赞比亚、南斯拉夫、澳大利亚等不同的国家，在白种人、黄种人、黑人等不同的种族，在西方文化、东方文化、阿拉伯文化等不同的文化都得到类似的结果，具有跨国家、跨种族和跨文化的一致性。

🎼而同样作为雌性生物，女人在择偶时对于财富的强烈执念，其实与博老鸟并无差异。

### 繁衍价值

🎼进化心理学中最核心的观点，人的核心任务是繁衍，从而让基因永存。提升繁衍成功率的办法有两个，第一个增多交配数量，第二个提高交配质量。

🎼增多交配数量就是常说的，不要把鸡蛋放在一个篮子里，降低繁衍失败的风险。而提高交配质量，就是提升单次交配的成功率。比如男性更倾向于寻找胸大、屁股大的女性进行交配，是因为他们的基因认为这样的女性哺乳能力更强，生育风险更低，能孕育出更好的后代。而女性更倾向于寻找身材健硕、健康的男性进行交配，因为这样的男性意味着他们携带着更强大的基因，繁衍出来的后代也更容易在这个星球上生存下去。

🎼繁衍价值与生存价值不同。繁衍价值指的是你身上携带更优秀的基因，自然界通过上百万年的进化，将这优质基因的要素，比如健康、聪明、高大、帅气、身材好等等，内化到我们的审美系统之中，把它们变成美的定义。表面上我们在追求美，但我们行为的本质却是为了基因的永存。

🎼那女人对于优劣基因的态度差别到底有多大呢？
