# 18、坏小k女人一生必学的一堂课：视频课：38.感情36计（18） 撩汉无敌技巧；释放可得性

🎼 感情三十年集第十八大技巧，彪悍的一个无敌技巧——释放可得性。

## 概述
在本节课中，我们将学习如何通过释放可得性来提升自己在感情中的吸引力。

## 什么是释放可得性？
首先，我们需要了解什么是释放可得性。**公式**：可得性 = 可得性值 / 期望值。

举个例子，如果你面前有两个钱袋，一个装有1万元，另一个装有10万元，你会选择哪个？显然，你会选择10万元的，因为它的价值更高，对你形成的吸引力更强。

但如果告诉你，如果你选择1万元的，现在就可以拿走10万元，而20年后再来找我拿，这时你的选择可能会发生变化。因为1万元对你的可得性非常高，而20年后拿走10万元的可得性太低。

## 释放可得性的应用
以下是几种释放可得性的方法：

### 1. 主动给予
- 当对方给你送礼物时，你可以表示赞赏，并表达出你对他/她的好感。
- 例如，对方送你礼物后，你可以回复：“看不出来你还挺有浪漫细胞的，特别是你为我做一些小事的时候。”

### 2. 增加好感度
- 在聊天过程中，用温柔的眼神看着他，并说：“和你聊天真蛮开心的。”
- 这样可以增加对方的好感度，从而提高心动指数。

### 3. 主动邀请
- 在逛街时，看到一家甜品店，你可以主动邀请对方一起去品尝。
- 例如：“这家甜品不错，我很喜欢，我们一起去尝尝吧。”

### 4. 离开与回归
- 在团体聚会时，时不时地盯着对方看，并朝他/她微笑。
- 当对方想找你聊天时，你可以和身边的闺蜜离开，然后再回来。

## 总结
本节课中，我们一起学习了如何通过释放可得性来提升自己在感情中的吸引力。希望这些技巧能帮助你更好地把握感情。
