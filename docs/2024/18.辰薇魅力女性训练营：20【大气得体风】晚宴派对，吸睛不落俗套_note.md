# 18. 辰薇魅力女性训练营：20【大气得体风】晚宴派对，吸睛不落俗套

## 概述
在本节课中，我们将学习如何在晚宴派对中穿搭，以吸引眼球且不落俗套。

## 派对穿搭雷区
以下是派对穿搭的几个雷区：

1. **过于随意**：穿着休闲裤和T恤参加派对，与环境格格不入。
2. **过于浮夸**：在非正式场合穿着过于浮夸的礼服。
3. **面料太少**：穿着过于暴露，让人感到不舒服。
4. **保守老旧**：穿着过时的服装，显得不时尚。
5. **颜色过于饱和**：穿着高饱和度的服装，显得不够高级。

## 派对穿搭类型
在派对中，我们可以分为以下几种穿搭类型：

1. **正式晚宴**：穿大晚礼服，如齐地礼服，搭配饰品。
2. **半正式派对**：穿小晚礼服，如过膝小礼服，搭配时尚包包。
3. **非正式聚会**：穿大方得体的服装，如小黑裙，搭配丝绒单品。

## 派对穿搭要领
以下是派对穿搭的几个要领：

1. **颜色干净有质感**：选择低饱和度的服装，避免高饱和度颜色。
2. **面料自带高级感**：选择丝绒和缎面等面料，展现优雅气质。
3. **剪裁干净利落**：选择剪裁简洁的服装，展现时尚感。
4. **饰品画龙点睛**：选择合适的饰品，如耳环、珠宝等，凸显细节和质感。

## 派对必备单品
以下是派对必备单品：

1. **小黑裙**：经典百搭，适合各种场合。
2. **丝绒单品**：展现优雅气质。
3. **夸张配饰**：如耳环、项链等，凸显时尚感。

## 年会礼服选择
以下是年会礼服的选择建议：

1. **V领、一字领或鸡心领**：适合脸大或脖子短的人。
2. **带袖礼服**：适合胸大或胳膊粗的人。
3. **收腰伞裙**：适合臀部丰满的人。
4. **高腰线收腰吊带款礼服**：适合身材瘦的人。
5. **丝绒复古风服装**：适合怕冷的人。
6. **时尚西装套装**：既有正式感，又有非正式感。

## 总结
本节课中，我们学习了如何在晚宴派对中穿搭，以吸引眼球且不落俗套。希望这些知识能帮助你在各种场合中展现自信和魅力。
