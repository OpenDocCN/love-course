# 雨宸19套合集：01.《爱与救赎，走进回避型依恋的内心世界》：005.冷暴力：回避型依恋的终极“武器”

## 概述
在本节课中，我们将探讨回避型依恋者如何使用冷暴力作为其“终极武器”，并分析其背后的心理成因。

## 冷暴力的定义
**冷暴力**是指在亲密关系中，一方通过冷漠、疏远、忽视等方式对另一方进行心理上的虐待。

## 回避型依恋者的心理成因
### 1. 恐惧心理
回避型依恋者害怕失去控制，担心变化会伤害到自己。冲突是感情中最难以控制的变数，因此他们会本能地采用回避的方式。

**公式**：恐惧心理 = 担心失去控制 + 担心变化伤害自己

### 2. 不愿依赖对方
回避型依恋者难以完全信任他人，对亲密关系抱有消极态度，不会过分担心和对方的感情，而是选择逃避问题。

**公式**：不愿依赖对方 = 难以信任他人 + 消极态度

### 3. 认为需求不会被满足
回避型依恋者从小养成“自己事自己做”的思维定式，进入亲密关系后，依然不知道如何理解和表达自己的情感，认为需求不会被满足。

**公式**：认为需求不会被满足 = 自我事自己做 + 需求不被满足

## 应对冷暴力的方法
### 1. 明确指出冷暴力伤害
指出对方的行为是冷暴力，并表达自己受到的伤害，但不要批判对方。

**公式**：明确指出冷暴力伤害 = 指出行为 + 表达伤害 + 不批判

### 2. 减少负面信息
避免在交流中传递负面信息，因为回避型依恋者对负面语言非常敏感。

**公式**：减少负面信息 = 避免负面语言

### 3. 建立需求默契
了解彼此的需求，并通过满足对方的需求来建立默契。

**公式**：建立需求默契 = 了解需求 + 满足需求

## 总结
本节课中，我们学习了回避型依恋者使用冷暴力的心理成因和应对方法。通过了解对方的内心世界，我们可以更好地应对冷暴力，建立健康的亲密关系。
