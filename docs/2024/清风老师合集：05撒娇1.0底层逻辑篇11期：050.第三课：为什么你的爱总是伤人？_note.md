# 清风老师合集：05撒娇1.0底层逻辑篇11期：050.第三课：为什么你的爱总是伤人？

## 概述
在本节课中，我们将探讨为什么在磨合期中，爱情往往会伤害到彼此。我们将分析磨合期中的问题、矛盾以及恶性循环，并探讨其内在和外在原因。

## 磨合期的特点
以下是磨合期的三大特点：

1. **问题重复出现**：问题似乎总是得不到解决，即使尝试了各种方法。
2. **深层的委屈和愤怒**：误解和不满在相处过程中逐渐积累。
3. **冲突升级**：争吵和指责成为常态，任何小事都可能引发激烈的冲突。

## 磨合期的四种方式
以下是磨合期中最常见的四种方式：

1. **互相指责**：指责对方不好，而不是共同解决问题。
2. **互相伤害**：通过伤害对方来表达自己的不满。
3. **不停道歉**：一方不断道歉，但问题并未得到解决。
4. **无力解决**：双方都无力解决问题，陷入恶性循环。

## 磨合期伤人的原因
磨合期中伤人的爱分为内在和外在原因：

### 内在原因
1. **关系中的表演逐渐失去**：随着时间的推移，人们不再刻意表现，而是展现出真实的需求和期待。
2. **寻找关系中的位置**：双方开始争夺控制权，导致冲突。
3. **童年伤痛的重复**：人们在关系中重复童年的伤痛经历。

### 外在原因
1. **显性关系模式的变化**：
   - **下沉变平行**：原本条件较好的一方与伴侣平起平坐，感到不适应。
   - **平行变下沉**：原本条件相当的一方变得自卑，不敢表达自己。
   - **平行变上升**：原本条件相当的一方努力提升自己，而另一方原地踏步，产生差距。

## 总结
本节课中，我们探讨了磨合期中爱情伤人的原因，包括内在和外在因素。了解这些原因有助于我们更好地应对磨合期中的问题，建立更加健康和谐的关系。
