# 796、【完结】陈海贤·自我转变50讲：chx51 结语：转变之后，我们要去哪里？

欢迎来到自我转变五十讲。我是陈海贤。经过五十节课的陪伴，我们的课程即将结束。在最后一节课中，我想与你分享一些心里话。

## 课程回顾

如果你正处于顺境，这门课程可能暂时派不上用场。它专为那些经历过挫折、失去和挣扎，正在寻找新自我的同学设计。正是因为经历过这样的痛苦和挣扎，你才会深刻理解转变的意义。

课程一开始就提到，转变不是解决一个具体问题，而是经历一条完整的道路。这条道路包括：

1. **响应召唤**：自我中不被环境和现实接受的部分推动我们做出选择。
2. **脱离旧自我**：失去目标、身份和关系，在荒野中重新寻找自我坐标。
3. **踏上新征途**：从旧自我中最有力的部分，到新群体和新经验中寻找新自我的信息。
4. **获得新自我**：发现那个期待的新自我，并从失去中找到新的评价坐标。

这条道路是一个关于转变的故事，有开始、结尾和曲折的经历。它旨在为你提供一个范本，帮助你理解自己走过的路，以及未来可能走的路。

## 课程背后的故事

我的第一门课程是《自我发展心理学》，并据此出版了《了不起的我》一书。这门课程的内容对应那书的第四部分——人生的转折期。许多同学表示，那部分课程触动了他们，帮助他们理解自己的心路历程。

然而，也有同学认为那部分内容过于简略。因此，我决定重新制作一门关于转变的课程，更深入地探讨这个主题。

## 个人经历

我自己也经历过转变。几年前，我从浙江大学离开，放弃了即将分到的房子。在这个过程中，我一边失落，一边努力理解所发生的一切。如今，我已经适应了新的自我，但仍然记得当时写的两句话：

1. 当你意识到自己失去大学教师身份时，告诉自己：现在你能依靠的只有你自己的名字和你会的东西。
2. 如果余生只能做一件事，我想用来回答：如何帮助人们从结束的痛苦中走出来，完成转变，走向新的生活。

这个愿望后来成为了这门课程的动力。

## 转变的艰难

每次听到学员讲述他们要离开原来的工作和关系，寻找新自我时，我都会感到惊讶。我知道这条路的艰难，因此会反复问他们：转变真的是一条可行的路吗？有没有其他选择？是否真的到了需要转变的时候？

当我将转变的规律传授给他人时，我会思考：我所经历的是否是一种普遍规律，还是仅仅因为运气好？万一我混得不好，会不会对当初的决定产生不同的看法？

## 转变的意义

转变的意义在于，它帮助我们理解自己走过的路，以及未来可能走的路。在失落时，转变的故事可以成为我们抓住的东西。虽然我们可能会怀疑它，但不要害怕分享自己的故事。

## 结语

无论你现在处于何种境遇，面临何种选择，记住：你最好的作品，也将是你度过的时光。希望你能创造属于自己的美好作品，用你度过的时光感谢加入自我转变五十讲。我是陈海贤。未来的某一天，当我们长出新的自我，我们江湖再见。
