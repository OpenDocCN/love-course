# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：63、亲密关系经营，如何不后知后觉，正向长久

## 概述
在本节课中，我们将探讨如何经营亲密关系，避免后知后觉，实现正向长久的关系。

## 核心概念
**储蓄账户模型**：将亲密关系视为一个储蓄账户，存款代表正向行为，扣款代表负向行为。

### 1. 亲密关系中的储蓄账户
以下是亲密关系中的储蓄账户模型：

- **存款行为**：夸奖、鼓励、认同、和谐的性体验、甜言蜜语等。
- **扣款行为**：冲突、争吵、诋毁、讽刺、冷战、反驳、防御等。

### 2. 亲密关系中的本金
- **原始本金**：彼此之间的信任和意愿。
- **本金积累**：通过存款行为积累信任和意愿。

### 3. 亲密关系中的波动
- **小波动**：日常的摩擦和冲突。
- **大波动**：聊骚、出轨、家暴、巨大欺骗等。

### 4. 亲密关系中的风险
- **风险系数**：存款数值与本金的比例。
- **风险规避**：通过存款行为降低风险系数。

## 教程
### 1. 建立储蓄账户
将亲密关系视为一个储蓄账户，明确存款和扣款行为。

### 2. 积累原始本金
通过存款行为积累信任和意愿，建立原始本金。

### 3. 维持账户平衡
关注账户余额，避免过度扣款。

### 4. 避免大波动
警惕可能导致大波动的行为，如聊骚、出轨等。

### 5. 评估风险系数
定期评估风险系数，确保账户安全。

### 6. 理性看待亲密关系
理性看待亲密关系，明确经营目的和需求。

## 总结
本节课中，我们学习了如何将亲密关系视为一个储蓄账户，通过存款和扣款行为来维护账户平衡，降低风险系数。希望这些方法能帮助大家更好地经营亲密关系，实现正向长久的关系。
