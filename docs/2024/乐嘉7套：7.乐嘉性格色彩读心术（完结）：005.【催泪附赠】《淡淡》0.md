本章题为“等待”。故事始于乐嘉因外力撞击导致下体受伤，需进行手术。在等待手术的过程中，他与医生交流，并对“睾丸白膜”这一结构、医院运作的困境以及生命的意义进行了深入的探讨。

## 第一节：医院的难处 🏥

乐嘉结束了和戴医生的电话，做好了进行手术的准备。手术前，他做了第三次B超检查。

由于距离受伤已超过24小时，部分血流凝固，之前被流动血液掩盖的损伤部位变得清晰可见。

检查结果显示，他的白膜破裂了1.5公分，必须立即手术。

**什么是白膜？**
白膜是包裹在男性睾丸外的一层纤维膜。

这层膜的硬度是人体正常皮肤的两倍。

对于网友比较白膜与女性处女膜硬度的问题，乐嘉感到困扰。他认为国内的性教育严重缺失，希望自己的分享能略尽绵薄之力。

**白膜与处女膜的本质区别：**
*   **处女膜**是女性的礼节性防卫，其存在意义在于未来被突破，完成生理上的转变。
*   **白膜**是男性的舍命性防卫，其最高使命是滴水不漏地保护睾丸，让睾丸能在其包裹的“混沌宇宙”中自由运作，生精化血，繁衍后代。因此，白膜必须坚如铜墙铁壁。

医生曾告知，即便在手术中，想划破白膜也需要用手术刀反复切割。乐嘉认为，两者硬度没有可比性。

对于男性而言，元气起源于睾丸。而乐嘉保护睾丸的“包装”——白膜，竟然被震裂了。

当丁医生拿着B超报告时，感慨乐嘉究竟经历了何等撞击。乐嘉询问自己的情况属于“撕裂”还是“碎裂”。

医生解释道：
*   **睾丸撕裂**：指白膜受损，但睾丸本身完好，缝合白膜后，睾丸功能可恢复正常。
*   **睾丸碎裂**：指睾丸内部已破碎，严重时白膜也会被震碎，无法补救，可能需切除或任其萎缩。

乐嘉的情况是白膜肯定已撕裂，但睾丸内部是否碎裂、碎裂程度如何，必须手术切开后才能知晓。

于是，他只能等待手术。然而，他被告知手术室需要排队，必须等到有空档才能进行。

因为手术被安排到下午很晚，乐嘉询问医生，为何这家知名大医院的手术室如此紧缺，需要几个医生共用一间。

医生解释了当前国内医疗体系的困境：
以下是核心原因分析：
1.  **资源分布不均**：国外是家庭医生和发达的社区医院制度，小病就近解决。在中国，人们普遍迷信大城市的三甲医院，导致三甲医院超负荷运转，而基层医院资源闲置。
2.  **内部协调困难**：医院内部若要新增科室，需平衡各科室关系。如果新科室进行长时间手术，会占用手术室，影响其他科室。
3.  **分院运营难题**：在郊区开办分院，患者又因距离远而不愿前往。
4.  **医生职业选择**：三甲医院医生工作强度大，但接触的病例多，成长快。基层医院病例少，医生成长慢，但并非没有好医生。医生水平的高低是一个概率问题，与医院基数有关。
5.  **系统性问题**：医院的等级评审制度、基层医师培养、医疗保险制度、公立与私立医院平衡等诸多因素相互影响，环环相扣。

通过与医生的长谈，乐嘉学到了许多医疗系统的常识。他意识到，医疗问题看似与健康人无关，实则随时可能发生在任何人身上。

因此，他明确感到，人们看到的表象与真相之间可能存在巨大差距。

准备手术前，乐嘉化验了各项指标。住院期间见到许多医生病友，他联想到电影《我是传奇》中全世界只剩一人的孤独场景，深感恐惧。

他得出结论：**有人在，真好。**

他设想了自己向往的三种离世方式：
1.  在演讲时，于知命之年战死在讲台之上。
2.  在写作时，于古稀之年握着钢笔安详离世。
3.  在年老时，仍能生机勃勃，于性爱中“精尽人亡”。

他最不能接受的是在无人陪伴中慢慢病死或老死，那会被巨大的挫败感包围。而此刻，有朋友和医护人员在身边，他受伤当晚的彷徨与恐慌削弱了很多。

## 第二节：主刀者登场 🧑‍⚕️

在蛋蛋修补这件事上，最重要的动刀者——许可教授，终于登场。乐嘉在手术当天上午第一次见到他。

许可教授给乐嘉的第一印象完全不像医生。他穿着有腔调的咖啡色小穗格西服，打着领带，气质出众，像老上海的“老克拉”。后来乐嘉发现他每天查房都会更换领带。

在乐嘉多次询问后，教授解释道，这源于他在美国著名医学院进修时的见闻。那里医生都穿西装打领带，让他感受到了职业的尊重与内心的自豪，这让他对医生职业充满狂热。

两人的交流在友好氛围中展开。初次见面时，由于有众多门生在场，乐嘉只是例行公事地表达了感激。

手术开始前，乐嘉发短信问了一个他问过无数遍的关键问题：“万一手术不成功，睾丸被切除，我的生育能力和性能力会受到多大影响？”他需要知道最坏的结果。

在后续交往中，许可教授多次提及此事。他从乐嘉对细节的刨根问底和对话中的忧虑，看出了乐嘉内心的脆弱。另一个证据是，乐嘉在手术前对他说了一番“死不瞑目的告别词”。

乐嘉对此并不反感，他回应道：“我脆弱一点都不重要，只要您强大就好，您强大，我的蛋蛋才有复活的保障。”

虽然乐嘉早就清楚那个问题的答案，但因为对方是主刀医生，他仍希望亲耳听到确认。

**答案很明确：少一个睾丸，对男性能力“半毛钱关系都没有”。**

但乐嘉内心是怀疑的，他认为从生物学规律看，事实不应如此。为此，他准备引用黑猩猩和大猩猩的例子来探讨。
