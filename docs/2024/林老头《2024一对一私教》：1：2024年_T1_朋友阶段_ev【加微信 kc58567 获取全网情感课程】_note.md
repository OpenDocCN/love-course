大家好，今天我们将深入探讨朋友阶段的核心内容。朋友阶段是认识新人的初期阶段，通过搭讪、社交软件、朋友介绍等方式，获得对方的联系方式或约会机会后，双方会进入这个阶段。

### 1. 注意力是关键

在朋友阶段，最重要的是注意力。例如，当你加到一个女生时，她会对你产生基本的注意力。但在这个阶段，吸引是不存在的。至少在两次完整对话之前，吸引是不可能产生的。

**公式**： 吸引 = 注意力 + 共鸣

### 2. 反馈窗口类型

在朋友阶段，所有的反馈窗口类型都是虚假的。这可能是因为共同话题、无聊、生理需求等原因。因此，在聊天过程中，要感受不同类型的反馈窗口类型，并思考其形成原因。

### 3. 展示价值

在朋友阶段，要展示自己的价值，让对方看到你的优点。这包括介绍自己、分享生活、展示兴趣爱好等。

**代码**： 展示价值 = 自我介绍 + 分享生活 + 展示兴趣爱好

### 4. 提供体验

要热情主动，让对方在了解你的过程中感到舒适。这包括主动开启话题、分享情绪、使用表情包等。

**代码**： 提供体验 = 热情主动 + 主动开启话题 + 分享情绪

### 5. 被动营销

认真对待自己的形象，包括朋友圈、头像、背景墙等。同时，保持自信和自我，用高位者的心态看待他人。

**代码**： 被动营销 = 认真对待形象 + 保持自信和自我 + 高位者心态

### 6. 筛选和淘汰

在朋友阶段，要筛选和淘汰那些不认同你的人，让那些能够进入吸引阶段的人占据更多时间。

**公式**： 筛选和淘汰 = 筛选不认同的人 + 淘汰不适合的人

### 总结

本节课中，我们学习了朋友阶段的核心内容，包括注意力、反馈窗口类型、展示价值、提供体验、被动营销和筛选淘汰。要想进入吸引阶段，需要做好这三件事情，并丢掉那些不认同你的人。情感学习是自我提升的过程，而不是弄虚作假。希望大家能够通过学习，提升自己，找到真正适合自己的人。
