# 小怡《供养者翻盘—强者框架课》：002.2、如何摆脱供养者属性 🧠

在本节课中，我们将要学习如何摆脱“供养者”属性，并理解成为真正有魅力的“情人”所需的核心要素。我们将深入探讨吸引力的来源，并重点介绍构建“强者框架”的两个关键组成部分。

通过上节课的学习，我们明白了什么是供养者模式及其危害，也了解了应该采用何种模式来建立有尊严的恋爱关系。

接下来，我们要开始学习如何改掉自己身上的供养者属性，成为一个真正有魅力的强者。想要改变，首先需要知道你为什么会拥有供养者属性。有的同学会问，上节课不是讲了吗？是社会的规训。

这的确是大环境导致的普遍现象。但要知道，即便大环境如此，也不是所有的男生都是供养者。总有一小部分人在用情人模式与女生相处，并且取得了成功。

所以，本节中我们要讨论的是，大部分的供养者与那一小部分人相比，到底缺少了什么。或者说，我们拥有了什么就可以摆脱供养者的身份，成为一个真正的情人。

如果你学过人性课，你就知道，一个人的吸引力来源于他身上的价值。价值分为三个部分：**生存价值**、**繁衍价值**、**情绪价值**。

生存价值就是你的物质条件如何，例如你的车、房、存款。如果你一直靠生存价值吸引女生，这就是供养者的择偶模式。

繁衍价值就是你的外形条件如何，例如颜值、身材、是否有腹肌等。用繁衍价值对女生产生的吸引是有效的。只不过，这是很浅层的吸引，只能够让女生对你产生初级的兴趣。如果在后续的相处模式不对，你就又会变成供养者。

情绪价值是开启情人模式的核心。情绪价值是你让一个女生因为你的存在而产生情绪波动的能力。可以让她因为跟你在一起而感受到开心、失落、紧张、焦虑、刺激、平静等各种各样的情绪。这时候，女生就会产生爱上你的感觉，觉得没有你不行。

她会为了把你留在身边而对你付出、投入，甚至变成你的供养者。这就是情绪价值所带来的魅力，也是一个人的核心魅力，我们称之为**人格魅力**。这种人格魅力一旦拥有，就永远不会消失，会如影随形地伴随你终身。

它不会像财富一样，有钱时就有女生，破产了女生就会离你而去。所以说，靠人格魅力吸引女生，靠人格魅力建立的关系才是最稳定的。而这种人格魅力的来源，就是来自于**强者框架**。

框架就是你的自我。如果你的内心没有框架支撑，那你很容易受到外界的影响，你的状态就是人云亦云，别人说什么就是什么。这就好像一个房子，如果你没有框架支撑，那么一场台风或者一个小小的地震，你的房子就倒了。

这就是前面说的，为什么大部分人会受到大环境的影响，变成一个供养者，而有一小部分人可以坚定自我、不受影响地用情人模式和女生相处。这正是内心有没有框架的区别。

而我们这套课程的核心，就是通过系统的训练和讲解，从“道”的层面帮你建立起一个真正的强者框架、一个强大的自我，然后从“术”的层面教会你如何运用框架赢得一段关系。

你会发现，当你拥有了强者框架以后，你自然就散发出了情人属性。你不需要运用任何的技巧套路，甚至无意识地就已经变成女生心中的情人。这就是我们这节课要做到的。

一个强者的框架，由两个部分组成：一个是**框架意识**，另一个是**内心强度**。框架意识就是“术”，内心强度就是“道”。只有道术结合，才是一个完整的强者框架。

而这两者的重要性也完全不同。可以说这个“道”，也就是内心强度，占了强者框架的90%，而框架意识这个“术”只占10%。中国有句古话说：“有道无术，术尚可求；有术无道，止于术。”说的也是这个意思。

任何一件事情，你想要学透，想要彻底掌握，就必须搞明白其中的“道”，搞明白其中的心法，这是最重要的。所以，接下来我会用十多节课的篇幅帮你完成这个“道”的掌握，也就是让你建立起一个强大的内心，一个强大的自我。

一个人的核心魅力，都是从自我产生的。我们常说的“不要用钱去吸引女生，而是让你这个人去吸引女生”，这里说的“这个人”就是你的自我。它不是你的财富，不是你的收入，不是你的身高或长相。这个“自我”指的就是一件事：你的人格。

所以你记住，当你有了足够的内心强度，你有了自我，你就有了一个良好的框架。这时候，你就不会被社会规训，不会被女生洗脑成供养者。你会以一个强者的姿态出现在女生面前，让她感觉到强烈的吸引，让她不由自主地靠近你。

本节课中，我们一起学习了摆脱供养者属性的核心在于构建强者框架。我们探讨了吸引力的三大价值来源，并重点指出情绪价值与人格魅力是关键。强者框架由“道”（内心强度）和“术”（框架意识）组成，其中强大的自我是基础。从下节课开始，我们将深入“道”的层面，系统学习如何建立强大的内心。
