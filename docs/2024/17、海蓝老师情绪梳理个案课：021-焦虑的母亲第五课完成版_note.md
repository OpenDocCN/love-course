# 海蓝老师情绪梳理个案课：021-焦虑的母亲第五课完成版

## 概述
在本节课中，我们将学习如何通过情绪梳理个案课来帮助一位焦虑的母亲处理她的情绪问题。

## 小志与父亲对话
**以下是小志与父亲对话的内容：**
- 小志想对父亲说，他对妈妈很好，都是妈妈不对，他已经很让着妈妈了。
- 小志指责父亲，因为他不是对父亲的不满，而是因为自己有怨气。
- 小志表示，每个人都说他命不好，他需要用这个人来保护自己才能活下去。
- 小志承认自己知道父亲不会关爱妈妈，因为父亲也是一个缺爱的人。
- 小志表示，他喜欢父亲，大伯喜欢姑姑，所以他也不知道怎么爱妈妈。
- 小志表示，他只是沉默地接受妈妈的指责。
- 小志认为，这种沉默对妈妈来说，对这个家来说已经是很好了。
- 小志表示，父亲的沉默应该是对家里面最大的贡献。
- 小志表示，妈妈的暴脾气是她能够借以为生的一个唯一的渠道。
- 小志对父亲说，你就浪着一点妈妈吧，但我也知道你特别难过。
- 小志告诉父亲，因为他没有看到妈妈，是因为以前他所经历的才会是这样。如果他能理解妈妈，他就不会那么难受了。

## 小志与母亲对话
**以下是小志与母亲对话的内容：**
- 小志表示，现在胸口就有感觉了，现在胸口就有那种不舒服的感觉出来了。
- 小志想告诉母亲，妈妈的指责带给他很大的伤害。
- 小志表示，这种伤害是因为体内有一股无法控制的气流，像妈妈这个整个指责的能量，就是住在他心里一样。
- 小志表示，他看到了他的感觉是一股气流，无知指责和焦虑的这样一种能量，从三四岁的时候一直到现在，一直是妈妈的东西。
- 小志表示，他想还给妈妈这种无知指责和焦虑的气流。
- 小志表示，这股气流好像大的他有点无法控制，但他可以控制的。
- 小志表示，他想把气流送出去，先控制住，给身体一个出口。
- 小志表示，他吸吸呼气，是不是可以帮他出去的速度快点，给他一点压力。
- 小志表示，他现在只有听到出去，很强烈的一股声音要他出去。
- 小志表示，他现在身体的感受是胸口有一点点，其他的就觉得比较空荡，空燥。
- 小志表示，他想替换快乐进来，快乐还有什么？就是希望妈妈就把她以前的事情都忘记掉。
- 小志表示，他还需要关爱、温暖、笑声、欢乐笑声。
- 小志表示，随着每次吸气，把快乐、欢笑、忘记、关爱、温暖带起来，随着每次呼气，把胸口那点点东西都把它冲去，都替代，让快乐、笑声、关爱、温暖充满整个心胸和胃部。

## 小志的感悟
**以下是小志的感悟：**
- 小志表示，他现在舒服多了，心里感受是放松，胸口不堵了。
- 小志表示，他想把三四岁的小志红从那个环境里面把他代理出来，妈妈成天抱怨、指责、焦虑，和爸爸争吵的那环境里面想办法把他带出来。
- 小志表示，他已经轻松了，他有欢笑就够了，有欢笑就够了。
- 小志表示，他需要的是关爱，其实就是理解。
- 小志表示，他希望每天能看到他，关注到他，哪怕一分钟都可以。
- 小志表示，他会做到每天哪怕一分钟也会来看他一下，当他出现的时候，我也会再来看他，当无助自知的焦虑出现的时候，我会我会来看他。

## 冷漠
**以下是冷漠的描述：**
- 冷漠是一个浑身连眉毛都是动起来的老头，他的胡子是冰冻的，连他流的鼻涕都是冰冻的，他就跟一个雪貂一样。
- 当小志看到冷漠后，他想对冷漠说，他知道冷漠是来保护他的，因为冷漠想让他感到安全，不想要他难过这么多年。

## 总结
在本节课中，我们一起学习了如何通过情绪梳理个案课来帮助一位焦虑的母亲处理她的情绪问题。通过这个过程，我们了解到，理解、关爱和陪伴对于缓解焦虑情绪非常重要。
