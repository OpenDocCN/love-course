# 情绪梳理课程第17课：060-07 个人总结 🧘‍♀️

在本节课中，我们将跟随一位学员（佳佳）的自我梳理过程，学习如何识别、接纳并转化内心的不同情绪部分。我们将看到如何与“无助”、“压抑”、“怀疑”等内在部分对话，并最终整合资源，获得内心的平静与力量。

---

上一节我们介绍了课程的开始，本节中我们来看看学员如何具体处理内心的各个部分。

可以暂且把这个画面放在自己想象里面，再看看真要跳起来会怎么样。什么时候养几只兔子试试看。接下来就带着我们已有的这些资源进行梳理。

最开始我们看到了“无助”的部分。我们让它先退后了。也非常感谢“无助”部分给了我们这样一个空间。

你现在再看“无助”部分，看到佳佳这一路特别害怕失控，现在它怎么样了？现在它看到佳佳可以这么自由绽放地跳舞，就很放心，很欣慰。它也是希望佳佳幸福的。看到佳佳很开心，它也很开心。

“无助”部分想换个角色，不想再当“无助”了。它想跟我一起跳，变成音乐，变成欢快的音乐。它已经没有什么担心了。

那还有“压抑”的这一部分，现在觉得怎么样了？还好了，都跳舞了，也就不压抑了。觉得也没有什么了。

“压抑”这部分也变成舞蹈的部分了，变成灵动的部分了。它跟着音乐一起跳，一起唱。

还有一个部分就是觉得“这都是假的”。我现在想对它的态度是接纳。因为我觉得它其实也是希望我好，它不放心。我觉得就让它看着我们跳吧。我跳得越久、越开心，然后让它看到我越来越有力量的时候，它的声音会越来越弱的。

它认为什么是真什么是假？它会觉得我过得惨是真的，过得开心是假的。因为过去22年一直非常压抑，压抑是常态，那个状态它已经很熟悉了。但它希望你开心。

那到底什么是真的？应该是有开心的时刻，也有惨的时刻，这才是真的。不是说只有开心或者只有惨，都有才是真的。有的时候过得不开心是真的，过得开心也是真的。

但最重要的是，在这后面什么东西到底是真的？什么是真实的？已经很接近了，就是生活当中有真实，有开心也有不开心的时刻，这都是真的。感受到开心的时候也是真的，感受到不开心的时候也是真的。

那什么是真的呢？渴望被爱是真的。感受到的东西就是真的。不管是难过、悲伤、无助、害怕，还是喜悦、压抑，感受到这一切都是真的。不是只有感受到压抑才是真的。

感受到差的也是真的，感受到好的也是真的，只要感受到的都是真的。而且是真实发生的。在脑子里面想象到的东西也是真的。只要这一刻时间用在哪里，哪里就是真的。

这一分钟过去是真真实实过去的。这一分钟的感受好也是真的，这一分钟感受不好也是真的。当下这一刻就是真的。不管是真实体验，还是我们自己在头脑当中给自己创造的，都是真实的。只要时间花在这儿，就是真实的。

它怎么看？它觉得有道理，没什么是真是假的。它好像懵掉了，这个观点很智慧，有点前沿。它琢磨一下，也不用“前沿”这个词，现在体会下就可以了。

是不是每一分钟花在什么地方，那个地方就是真的？这一刻感到快乐，这一刻的快乐就是真实的，不管它是以什么方式得来的。这是“前沿”吗？不是“前沿”是什么？其实是自己以前没有察觉的。

不是“歉意”，是一直都有的一个事实。其实只是以前不知道而已。那现在感觉怎么样？知道这一点时，感觉很踏实、很清晰。

而很多所谓的“真实的压抑”，那是什么？那是真的吗？需要那种“真”吗？那22年的压抑从哪里来的？是佳佳对自己的一种怀疑，内心有一个标准。不按照这个标准来就不对，然后就会压抑自己。把22年把自己放在压抑的情绪当中，需要吗？不需要。需要那种“真实”吗？不需要。

好的，那现在再看看它。经过这段交流之后，它现在什么样？它好紧张，被问懵了。它没事儿，它不需要立刻马上就有答案。但是它可以知道，就是可以不用这样压抑这么多年，可以过得很轻松、很自在。

在你内在还有一些什么部分，需要在我们结束之前交流一下？还有一点点紧张。因为毕竟还是踏进了一个挑战区。所以对佳佳来说，是一件有挑战的事情，可能还是有一点点紧张。

我想说出这种紧张，让它被看到，被接纳。我也允许它一直紧张，觉得挺好。在挑战区的话，一般来说，紧张本来就应该有。要是紧张，起码也要有三四分嘛。这个紧张是很正常的。要是进入挑战区还不紧张，那个挑战是真的假的？

这是进入挑战区的一个标志，就是有点紧张。否则也谈不上是挑战了。那你看到以后，你的紧张部分感觉怎么样？嗯，没那么紧张了。

还有什么部分需要交流的？在我们睁眼以前。感谢吧，感谢所有人，感谢老师，感谢大家。活着真好。

准备好的话，可以把眼睛慢慢睁开。慢慢睁开以后，现在再接着看着我，你现在什么感觉？离得近点还是离得远点？可以可以，就是一个慈爱的、像妈妈那样的人。身体的感受上，你看着我很温暖，爱在流动的感觉。我挺好的，我值得和一个这么好的老师这样近距离地交流。

你确实很好，也很棒。这需要极大的勇气，你是非常有勇气的。真的真的好很多，敢和老师对视了。一会儿咱们结束第一件事就是拍照。

还有，在整个过程当中，你觉得对你帮助最大的是什么？总结回顾一下这个过程。

以下是学员总结的几个关键帮助点：

*   **无限的接纳**：当我出现那个中年男人嘲笑的画面（其实和电梯那个画面无关）时，允许我去看到那个画面其实是有关联的，并让我去释放情绪。这个释放是非常大的。因为在妈妈面前，我还是不敢很愤怒，我真的很乖，乖惯了。但那次我就发泄出来了，这是一个很大的帮助。
*   **与内在小孩的连接**：我觉得也跟我平时和我的“小佳佳”（内在小孩）的连接有关。其实在梳理之前，我也很紧张，然后突然间我就把“小佳佳”带过来，一直慢慢跟他建立一个很好的朋友的关系。
*   **理解与和解**：在梳理过程中，让我看到妈妈的不容易，然后去感受妈妈的感受时，我觉得和解了很多。
*   **资源的锚定**：还有就是找资源，那种身体的感受。找资源，让他锚定在身体里，整体让我的身体都放松很多。这些都对我帮助很有效。
*   **情绪的释放**：尤其是那个情绪的宣泄，让那种愤怒被看到，那个特别给力。

---

本节课中，我们一起学习了如何通过自我对话进行情绪梳理。我们看到了如何识别并感谢“无助”部分，如何将“压抑”转化为“灵动”，以及如何与“怀疑真假”的部分进行深度对话，最终理解“感受即真实”的核心概念：**当下所体验的一切情绪都是真实的**。过程的关键在于**接纳、释放、连接与锚定资源**。通过这样的梳理，学员从紧张、自我怀疑的状态，转向了更踏实、有力量且能自我肯定的状态。
