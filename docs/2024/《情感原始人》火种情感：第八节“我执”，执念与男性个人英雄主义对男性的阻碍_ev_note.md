# 《情感原始人》火种情感：第八节“我执”，执念与男性个人英雄主义对男性的阻碍_ev_note

在本节课中，我们将要学习“我执”这一概念，并探讨执念与男性个人英雄主义如何对男性造成阻碍。我们将深入分析男性内核、情感模式，并将其与佛家、道家思想中的“我执”联系起来。

许多男性在面对女性时表现不够真实，一个重要的原因在于他们固执地认为“我要得到的，就必须得到”。这句话看似矛盾，实则揭示了当今社会相当一部分男性的自恋情结与个人英雄主义。

当男性认为“我必须得到”或“对方就应该给我好反应”时，现实世界往往会给其当头一棒。因为客观现实常与我们的预期和幻想不同。

当预期与幻想落空时，人便会陷入痛苦、纠结与强烈的控制欲。长期的控制欲落空后，会导致自我怀疑或假装强大的反应。

我们要知道，控制欲是导致人焦虑的最强欲望之一，它甚至超过了性欲、食欲等其他欲望。因为控制欲意味着我们在幻想尚未发生的未来，并希望其符合当下的幻想。

当未来的现实不符合如今的幻想时，人就会陷入巨大痛苦。这种痛苦会导致人在某些领域或情绪中不断进行错误的循环，最终引发内心的匮乏感。

我们接触的许多学员，无论条件好坏，都曾在某个瞬间感到生活的匮乏与无聊。问题的关键往往不在于控制外界，使其完全符合我们的心意。

一种更强大的心态是：无论外物如何变化，无论未来发生什么，大多都在我的预料之内。

听到这里，很多人会误解，认为“预料”和“预期”是一回事。但事实并非如此。

前文所讲的“预期”，是指我们对未来抱有过高的期待。而这里所说的“预料”，是指我们可以遵循斯多葛主义等思想，对未来做好相对应的最坏打算。

很多人听到这里，千万不要进入悲观的投射。为什么对未来做好最坏的打算，反而是一种乐观的表现？

因为当我们对未来过于乐观时，一旦未来发生的事情不符合我们乐观的幻想，我们反而会感到痛苦。

所以，当我们对未来做好最坏的打算时，就意味着未来无论发生什么，只要结果高于最坏打算，就值得我们感到开心。

即使未来确实达到了最坏的表现，也早已在我们的预料之中，我们不会为此感到痛苦，甚至可能感到一丝欣慰，因为我们早已做好了准备。

在这种心态下，人反而会变得更加乐观。这也解释了半杯水理论：有些人看到半杯水是悲观的，而另一些人则感到乐观。

其根本原因在于，不同男性的思想出发点不同。有些人首先抱怨环境、抱怨自我、抱怨未来不符合预期。

而有的人则提前做好了最坏的打算，因此能够开心、勇敢地面对任何事情。

一个人如果在“我执”中陷得太深，最终会导致一个现象：将自身内核与自我进化的权利交给环境或他人。这只是一种委婉的说法。

也就是说，当一个人在“我执”中不断遭遇负面反馈后，他会把所有原因归结于外界。

一个男人在无数次对外界的幻想过程中，不断遭到打击或遇到不可控的情况。例如，一个男人可能会说：“我现在是个胖子，我想减肥，是因为我母亲对我不好，是因为我的环境、同事、朋友对我不好，或者是因为我的同学都很懒惰，所以导致我不想行动。但我内心又十分渴望减肥。我尝试跑了三天步，发现体重没降反而增加了，这导致我的愿望落空。因此我认为，必须出现一个人来激励我、引导我、鼓励我，我才有可能变好。”

听到这里，很多人可能会感到疑惑。下面我简单解析一下这段话的含义：

有的人在“我执”中陷得太深，导致他们逐渐对未来失去信心，从而将自己改变的权利交给了他人，或者交给了对未来某个幻想的“第二人格”。

其实，他们所幻想的那个能带领、鼓励、激励自己的人，就是另一个自己。那个“人”在遥远的未来呼唤你，是希望你能够在此时此刻清醒过来。

而不是一味地寻找借口，也不是一味地做最好的打算，以至于每次愿望落空，都导致失落、消极和自卑。

因此，我希望大家能做到以下两点：

以下是本节课的两个核心建议：

1.  **不要陷入过深的“我执”**：有时要学会适当放手，让他人产生执念，而非让自己产生过大的执念。
2.  **万事做好最坏的打算**：这样反而能拥有最乐观的心态，因为一切才会真正进入你的预料之中。

本节课中，我们一起学习了“我执”的概念，分析了执念与个人英雄主义对男性的阻碍，并探讨了通过调整预期、做好最坏打算来培养更强大、更乐观心态的方法。
