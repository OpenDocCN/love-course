# 18、坏小k女人一生必学的一堂课：视频课：36.感情36计（16）如何修炼气场

🎼 感情36计之十六大技巧，普通人如何修炼自己的气场？以下是一个实操方法。

### 情景假设
假设你正在和喜欢的男生吃饭，但男生对你似乎没有兴趣，你在情感或物质上对他有需求。因此，你尽力营造和谐氛围。

### 气场衰弱的原因
可惜的是，你越想保持和谐氛围，气场反而会衰弱。

#### 控制需求感
1. **快速控制需求感**：克制对目的的渴望，例如，不要急于让对方买包包或认识大佬。
2. **气场收获**：当一个人专注于目的，会忽视自己的气场。

#### 社交直觉锻炼
1. **能量流向感知**：进入包房，感受房间能量流向，判断热区和冷区。
2. **能量转移感知**：走进房间，感受能量转移。
3. **能量控制判断**：判断谁在控制能量。
4. **能量反应察觉**：察觉人们对能量的不同反应。
5. **能量受欢迎程度判断**：判断人的能量是受欢迎还是不受欢迎。
6. **气氛变化感知**：感受气氛变化。

### 社交直觉的重要性
仔细思考，即使缺乏社交直觉，你也能察觉到以上情况。

🎼 本节课中，我们学习了如何修炼气场，通过控制需求感和锻炼社交直觉来提升个人气场。希望这些技巧能帮助你更好地掌控自己的气场。
