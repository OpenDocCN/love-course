# 06.《高嫁实操秘籍——向上社交》系列课：015. 如何在向上社交中提升价值感 🚀

在本节课中，我们将学习在向上社交过程中，如何清晰定位自身角色并有效提升个人价值感，从而更顺畅地与高质量人群建立链接。

上一节我们介绍了如何筛选优质人脉。本节中，我们来看看在向上社交中，你应如何定位自己并提升价值感。

## 向上社交的定位与价值本质

人与人之间关系的本质是**利益的链接**。在《临界点》一书中写道：**人脉的本质是价值的交换**。你的交换价值是高质量人际关系存续的终极原料。

在向上社交中，你在社交场中的价值感取决于你拥有多少**社交筹码**。你的社交筹码来源于你能为对方创造多大的价值。

跨越阶层社交时，最大的疑点与困惑往往来源于：**“我能提供什么价值？”** 如果对自身定位不够清晰，就像推销一个产品却不知其卖点与核心竞争力，别人自然不会愿意“买单”。

因此，我们有一项非常重要的功课：**挖掘自身的核心竞争力**。

## 第一步：建立底层自信与心态

在挖掘核心竞争力之前，首要任务是建立底层自信与心态。核心心法是一句话：**放下面子不是讨好，你配得上和他做朋友**。

要做到言行合一、道与术结合，关键一点是**消除弱者思维，了解强者思维**。

### 识别并摒弃弱者思维

以下是弱者思维的几个显著特征：

1.  **对于付出的渴求度极高**：希望付出一点就立刻得到回馈，患得患失。心理学发现，很多人在学习瓶颈期放弃的原因就是不能接受“看不到结果”。而强者懂得**延时满足**。
2.  **不懂装懂，抗拒新鲜事物**：对新鲜事物表现出强烈抗拒。越是学识渊博的人越谦虚，越是懂得多的人越觉得自己无知。抗拒新事物、自以为是，本质是懂得太少。
3.  **刻板印象，思维僵化**：喜欢对事物抱有批判态度，总以“我看透了”的态度面对。例如，别人阐述观点时立刻打断说“我知道，不就是……吗”。这类人常用狭隘的眼光理解世界。
4.  **爱占便宜，欲望失控**：要得多，但质量都不高，什么都想要。本质是无法控制欲望出口，最终反而失去更多。

向上社交必须摒弃这些思维模式，因为你要融入的是一群拥有**强者思维**的人。

### 了解并学习强者思维

强者思维的人以**价值最大化**为导向。以下是其特点：

1.  **一切动作追求价值最大化**：时间永远比钱贵。同等时间能创造更多价值，绝不会退而求其次。例如，选择头等舱是为了用金钱购买时间，获得更好的休息和工作环境，以便更高效地投入后续事务。核心是：**我的时间比钱更重要**。
2.  **宽容度高，格局大**：有句话叫“不和傻子论长短，不和矮子争高低”。宽容是一种格局，社交场上没有永恒的朋友或敌人，只有永恒的利益。自身强大时，不会在意小事，避免卷入内耗漩涡。
3.  **思维方式与能量场正向积极**：能保持高能量，不置身于负能量和负面事件中，更不会陷入内耗，总是积极解决问题。这种高能量具有强大的感染力。

## 第二步：把握向上社交的心态与尺度

与高层次的人结交，目的应是成为或靠近他们。因此，需注意以下心态与尺度：

1.  **学习是终身目的**：每一次沟通都是课程，是实打实的实战经验。不要目的性过强，想着不切实际的事情，认识到就是资源。
2.  **学会把握分寸，进退得宜**：这是人与人交往的通用准则，在向上社交中尤为重要。**倾听大于表达**。不要越界，不该说的话不要说。

**案例说明**：一位总监带下属女生与一位企业女高层谈业务。女生过于“聪明”，在桌上直接点明总监与对方关系好、提及对方私事（买房），属于典型的没有边界感。结果导致高层对其印象不佳，错失了潜在的优质人脉资源。这提醒我们：**听要大于说**。

## 第三步：珍惜社交成本与高价值人脉

社交本身有成本。每次向优质人脉提出需求时，都要考虑：**这个需求值不值得开口？**

有价值的人脉一定要用在“刀刃”上。每一次诉求都是在消耗对方心中的信任值。自己能解决的事，绝对不要麻烦别人。重要的人脉要用在**高价值、高回报**的事情上。

**案例说明**：一位姐妹的朋友想以远低于行情的价格请多位明星录祝福视频，并希望她利用人脉“卖面子打折”。这位姐妹很聪明，没有动用自己核心的娱乐圈人脉，而是通过第三方外包公司报价。果然朋友后来因预算反悔。此举保护了自己在核心圈子的信誉。这警示我们：**人情是要还的**，不要轻易为低价值请求消耗宝贵关系。

## 第四步：进入合适的场域，拒绝无效社交

提升价值感需要进入合适的场域，即只参加有价值的社交，拒绝垃圾低能局。

向上社交的核心准则是：**质量高于数量**。

*   **什么是有效社交？** 例如行业交流会、业内重要人物组织的活动、高价值人脉付费参与的活动等。前提是**对你有用**，无论是在成长、眼界还是沟通上。
*   **什么是无效社交？** 多存在于半生不熟的熟人之间，如纯粹吃吃喝喝、KTV的酒肉朋友。意义不大。

成年人的社交以价值为导向。维护能在工作或生活上对你有价值的关系，人生才能少走弯路。**一个帮助你成长的朋友，比一群只会陪你吃喝玩乐的朋友更有价值**。这不是功利，而是关系的本质。

## 第五步：成为“圈中人”，获取身份认可

向上社交的重要环节是成为“圈中人”。圈中人的身份让你说话做事更有底气。

人是圈子性动物。犹太经典《塔木德》有言：**和狼生活，只能学会嚎叫**。你所在的圈子决定了你的位置。竞争有时比的就是资源和人脉。

找到真正适合自己的圈子，即你更容易融入、你的价值能被体现的圈子。无需让圈子里每个人都与你交好，关键是获得**圈子内相对重要或核心人物的认可**。得到他的认可，就相当于拿到了入场券。

**如何获得圈内身份？** 你可以借助一些社会组织或协会。例如：
*   海外留学背景可加入“海青会”、“欧美同学会”。
*   可以考虑同乡会、校友会、商会等。

这些圈子就是入场券。进入后，循序渐进地发展人脉，不要妄想一步接触到最顶层的人。
