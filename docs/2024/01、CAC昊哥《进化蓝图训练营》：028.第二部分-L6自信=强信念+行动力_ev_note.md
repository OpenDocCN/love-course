# 课程01：CAC昊哥《进化蓝图训练营》- 028. 第二部分-L6 自信=强信念+行动力 🧠💪

在本节课中，我们将要学习自信的本质，并明确区分自信与不自信、自恋等概念。我们将通过一个清晰的定义来理解自信，并探讨如何通过信念和行动来构建真实的自信。

## 自信的定义 📝

上一节我们探讨了阻碍自信的因素，本节中我们来看看如何获得真实的自信。首先，我们需要为“自信”下一个明确的定义。这个定义有助于我们解释后续的许多问题。

在心理学领域，并没有对“自信”的官方定义，因为它并非一个标准的心理学概念。心理学中相近的概念可能是自尊、信心或自我效能感。因此，我在这里给出一个定义：

**自信 = 始终认为自己有机会 + 会去采取行动**

这个定义包含两个核心部分：
1.  你拥有一个正面的感觉，**始终认为自己是有机会的**。
2.  更重要的是，你会基于这个信念**去采取行动**。

## 自信与不自信的对比 ⚖️

理解了自信是什么，我们再来看看它的反面。什么是不自信呢？其实就是自信定义的反面。

不自信是指：
*   你的情绪和视角被**负面因素**所主导。
*   同时，你的**行动被阻碍了**。

大家只需要明白这个概念即可，不必用“不自信”来定义或评价自己。但要客观地理解这个状态。

## 自信与自恋的区别 🚫

除了自信与不自信，我们还需要区分自信和自恋。那么，什么是自恋呢？

自恋的核心信念是：**“我绝不可能失败”** 或 **“我/事情必须是完美的”**。

以下是自恋可能的表现：
*   **对自身不完美的攻击**：例如，刚开始线下实践时做不到完美，便因此自我攻击。
*   **因不完美而“破防”**：别人的负面反馈让你“破防”，根本原因可能不是对方不认可你，而是“我居然没能一次就做到完美”这个事实，挫伤了你的自恋。
*   **对他人和世界的要求**：自恋者潜意识里认为自己是宇宙的中心，试图掌控所有人事物。当任何事情脱离其控制时，都会戳破其自恋的幻觉，从而感到极度不适。

因此，自恋常常会导致拖延、找借口、退缩等行为。

## 核心要点总结 ✨

本节课中，我们一起学习了自信的构成。我们明确了自信的定义是 **“强信念（认为自己有机会）”** 加上 **“行动力”**。同时，我们区分了不自信（被负面情绪阻碍行动）和自恋（要求绝对完美和掌控）。理解这些概念的差异，是迈向真实自信的第一步。
