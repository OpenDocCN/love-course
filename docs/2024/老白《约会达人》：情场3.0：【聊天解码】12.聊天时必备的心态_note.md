# 老白《约会达人》情场3.0：第12课 - 聊天时必备的心态 🧠

在本节课中，我们将学习与女生聊天前需要建立的三种核心心态。这些心态是顺畅沟通的基础，能帮助你摆脱紧张和不自然，让聊天过程更轻松、更有效。

很多朋友知道聊天需要自信和坦然，但具体如何做到却感到困难。感情经历较少的朋友，更容易在异性面前感到不自然。本节课程将为你提供具体的心态建设方法。

## 一、去做心态 🛠️

上一节我们提到了心态的重要性，本节中我们首先来看看第一种核心心态：“去做心态”。这是一种通过实际行动来建立真实自信的心态，而非仅仅依靠心理暗示。

传统的心态建设方法，如不断自我暗示“我很优秀”、“我很自信”，往往效果有限。你只是在欺骗自己，因为世界的普遍价值观（例如女生可能更青睐高富帅）不会因你的自我暗示而改变。你需要先接受这个客观事实，然后着手改变自己。

追求女生的核心，不在于你如何看待世界，而在于女生如何看待世界。因此，你需要的是“去做心态”——通过实实在在的行动和改变，来获得真实的自信和他人正面的反馈。

例如，作者曾因身高而自卑，尝试自我催眠等方法均告失败。后来通过**彻底改造形象、健身、阅读、广泛社交**等实际行动，获得了周围人真实的正面反馈，从而建立了坚实的自信。

在“去做心态”的基础上，衍生出两个重要的聊天前心态。

## 二、提供价值的心态 💎

基于“去做心态”，我们明白了改变自己、提升价值是关键。由此，我们进入第二个心态：提供价值的心态。开始聊天前，你必须明确，你的目标是尽可能多地向对方提供价值，而不是索取价值。

这里有一个核心公式：**提供价值 = 产生感觉**。只要你提供足够多的价值，女生对你的感觉就会更好。

那么，什么是“提供价值”？具体可分为两种：情绪价值和实际价值。

### 1. 情绪价值

情绪价值并非指一个人不停说话，而是指**你能影响对方的情绪，制造情绪波动**，让对方在聊天中感受到愉悦、紧张、期待或失落等多种情绪。

以下是三种可以制造情绪波动的聊天技巧：

**（1）推拉**
推拉技巧是指一下将对方推开（推），一下又将对方拉近（拉），从而调动其情绪。
*   **案例1**：女生发来与朋友的合影（她更漂亮）。你说：“一看就知道右边的是你。”（拉，暗示她好看）她开心回应后，你接着说：“不是啊，右边的胸大一点，你的明显没这么大。”（推）随后再说：“但我喜欢胸小点，穿衣服好看。”（拉）
*   **案例2**：女生深夜说想吃东西但很累。你鼓励她去（拉），她拒绝后你让她休息（推），最后又叮嘱她少熬夜（拉）。在一推一拉中，她的情绪产生了波动。

**（2）曲解**
曲解是故意误会对方的意思，以一种骄傲风趣的方式，引导对方来追逐或解释。
*   **案例**：女生说：“昨天居然梦到你了。”你回复：“占我便宜没有？”或“完了，肯定被你占了。”用一种有趣、放松的方式表达，而非严肃。

**（3）打压**
打压通常用在女生不配合、姿态突然变强，或需要维持自身框架时。注意，打压不是人身攻击。
*   **情况一**：女生回复敷衍。你可以回复：“你平时都是这样和人聊天的吗？”或“好好说话。”以此暗示你感受到了冷漠，如果对方不改善则结束聊天。
*   **情况二**：聊天中因某句话（如升高关系时）导致女生反馈变差。这时可以摆出强硬框架。
    *   **案例**：女生表达对关系进展的疑虑。你回复：“不靠谱你还聊，是有多无聊？我反正是没时间和我觉得不靠谱的人闲聊的。”（打压，摆出框架）后续对话中持续保持框架，引导对方解释和软化态度。

打压技巧也可以与曲解结合使用。当你有意拉升关系而女生自我感觉良好时，可以适当打压。
*   **案例**：你说：“姑娘真是个好人，上天保佑你找个天下第一的好郎君，我大概勉强算是天下第一吧。”（曲解）女生回应后，你说：“我说你是个好人，不是好姑娘，你想多了。”（打压）

### 2. 实际价值

实际价值指的是女生能**直接用眼睛看到的价值**。你可以通过分享生活状态来提供这种价值。
*   在吃精致的晚餐时，私聊发照片描述状态。
*   健身时，发健身照片或健身房小视频。
*   开车时，发一张开车的照片。
*   做饭时，发一张做饭的照片。

**注意**：分享的内容本身需具有价值。避免在杂乱差的环境下拍照分享，否则会产生反效果。

与提供价值相对的是“索取价值”，即不断向对方提问，如“在干嘛？”、“吃饭了吗？”。聊天中应尽量避免单纯索取价值。

## 三、游戏者心态 🎮

掌握了提供价值的心态后，我们还需要一个能让聊天过程更轻松的心态：游戏者心态。这能帮助你降低需求感，避免因过于紧张而表现失常。

游戏者心态主要包含两种：

**1. 一边玩一边聊的心态**
很多人在和女生聊天时会守着手机等待回复。正确做法是：**像打游戏一样，回完消息就把手机放在一边，继续做自己的事，过段时间再看**。这能有效降低你的需求感和得失心。

**2. 选择性心态**
你不能只和一个女生聊天。如果只有一个目标，你会投入全部精力，变得患得患失，对方一句话就可能让你紧张不已。这毫无吸引力。
*   这就像打游戏，只有一条命时会非常紧张；如果命很多，心态就会放松。
*   选择性心态不仅是一种心态，更是一种状态：**你积累了较多的社交资源，拥有选择权**。这样你在与任何一个女生聊天时都会更加从容。

## 四、接受不被所有人喜欢 ❤️‍🩹

最后，我们需要认清一个现实：即使你做得很好，依然会有一些女生对你反馈不佳。

**我们不是人民币，不可能保证每个人都喜欢我们。** 因此，只需专注于和那些对你反馈还不错的女生聊天即可。不必纠结于那些一开始就对你冷漠的女生，在你没有犯错的前提下，后期很难通过聊天让其回心转意。不要将时间和精力浪费在她们身上。

## 总结 📝

本节课中，我们一起学习了与女生聊天前必备的三种心态：
1.  **去做心态**：通过实际行动改变自己，获得真实自信的基础心态。
2.  **提供价值的心态**：聊天中应致力于提供**情绪价值**（通过推拉、曲解、打压等技巧制造情绪波动）和**实际价值**（分享优质生活状态），而非索取价值。
3.  **游戏者心态**：保持“一边玩一边聊”的轻松感，并建立“选择性心态”，拥有更多社交资源以降低需求感。

最后，接受无法让所有人喜欢的事实，优先与对自己展示出良好兴趣的女生互动。以上就是微信聊天前你应该具备的核心心态和状态。
