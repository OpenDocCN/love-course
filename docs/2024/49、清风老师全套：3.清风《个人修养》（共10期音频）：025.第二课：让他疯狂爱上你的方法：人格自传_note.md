# 清风老师全套：3.清风《个人修养》（共10期音频）：025.第二课：让他疯狂爱上你的方法：人格自传

## 概述
在本节课中，我们将学习如何通过讲述自己的人格自传来吸引他人，让对方疯狂爱上你。

## 什么是人格魅力？
人格魅力就是你的**人生自传**。在你成为今天的自己之前，你经历了大量的事情。而别人，特别是你喜欢的人，他并不了解你是如何成为今天自己的。

## 人格自传的七个要素
以下是讲述人格自传的七个核心要素：

1. **失败**：分享你的失败故事，让对方对你产生深刻印象。
2. **目标**：人生的终极追求，让对方感受到你的精神力量。
3. **动力**：你为什么要做这件事情？让对方看到更加真实的你。
4. **结果**：展示你的阶段性成果和大的成果，让对方看到你的努力和付出。
5. **导师**：在你成功的道路上，谁帮助过你？让对方看到你的积极向上。
6. **伙伴**：在你的前进道路上，谁一直支持你？让对方看到你的社交圈子。
7. **敌人**：曾经背叛过你，伤害过你的人，让对方看到你的成长和坚强。

## 聊天的基本框架和流程
在聊天中，不需要一次性将所有要素都讲完，但要有意识地将这些要素融入到沟通中。让对方对你的人生越来越好奇，对你的经历越来越期待。

## 总结
本节课中，我们学习了如何通过讲述自己的人格自传来吸引他人。通过分享你的故事和经历，让对方对你产生深刻印象，从而让对方疯狂爱上你。
