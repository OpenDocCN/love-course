# 加一情感《宅男宝典：02、聊天解码：3.恋爱时正确的心态（恋爱的基石）

🎼嗨你好，欢迎来到情胜方法，我是班长alen。在脱单道路上，心态问题是最大、最困难的问题。心态看似虚，实则最基本。

🎼心态建设复杂，但今天我想分享两个最重要的东西。🎼首先，感情是一种能力，若不将其视为能力，会犯各种错误。

🎼以学骑自行车为例，我们不会期望第一次就能学会。恋爱同样需要练习，是一种技能。技能学习中要注意三点：障碍区、拐点和状态回落。

🎼障碍区是可能跌倒的地方，如初次搭讪失败。突破障碍需要多次尝试，这是核心。

🎼拐点是无需他人指导，获得前行力量的时刻。如健身无需教练，恋爱无需指导。每个拐点都需突破，不能掉以轻心。

🎼状态回落是很多人摔跟头的原因。突破障碍后，状态会回落，需保持警惕。

🎼学习恋爱是过程，需经历障碍区、拐点和状态回落。简单来说，犯错误，下次就知道哪里错了。

🎼看待事物有两种方式：成败和成长。希望你们成为第二种人，从每件事中获取成长。

🎼悲观者消极，乐观者积极。从每件事中获取成长，这是最重要的。

🎼总结：感情是能力，需注意障碍区、拐点和状态回落；看待事物为成长，而非成败。
