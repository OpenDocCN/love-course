# 加一情感《宅男宝典》课程02：聊天解码 - 3. 恋爱时正确的心态（恋爱的基石）💖

在本节课中，我们将要学习恋爱心态建设中最核心的两个方面。心态是脱单道路上最基础也最关键的一环，理解并掌握正确的心态，能为后续所有技巧的学习和应用打下坚实的基石。

## 概述：心态是恋爱的基石

许多人认为“心态”这个词很虚，但它实际上是最基本的东西。心态建设非常复杂，但今天我们聚焦于两个最重要的核心概念。

## 第一部分：恋爱是一种可以习得的能力 🚴

首先，我们必须认识到，感情本质上是一种**能力**。如果你不把它看作一种能力，就会犯各种错误。

回想你小时候学习骑自行车。你是第一次就学会了吗？通常，我们对学骑车的预期是练习三五次才能掌握。这是一个大家都能接受的逻辑。

但是，为什么到了谈恋爱这件事上，很多人就觉得必须一次搞定、一次学会？这里有一个核心：**任何技巧或知识点都需要不断练习**。例如聊天、修图，练习三五十次是最基本的，可能才能真正明白。

恋爱同样需要练习，所以**恋爱也是一种技能**。在技能学习过程中，有三个关键点必须注意。

以下是三个核心的学习阶段概念：

1.  **障碍区**
    障碍区是指你可能会“跌倒”的地方。就像第一次骑自行车转弯时摔倒，下次再遇到转弯就会害怕。这就是一个障碍。
    绝大部分人没有学会一项能力的核心错误，在于没有理解人是存在障碍区的。**每一个障碍的突破都可能需要尝试3到5次，甚至更多次**，这是最核心的。
    例如，你尝试主动社交（如搭讪），第一次失败了，被女生拒绝。你可能会想：“是不是我不适合社交？是不是方法不靠谱？” 其实不是，只是因为你正在经历障碍区。刚开始改变形象觉得奇怪、聊天总是聊不好、约会“见光死”、恋爱超不过半年……人生的不同阶段会遇到不同的障碍区。关键不是遇到障碍就放弃，而是想办法跨越。一次不行就两次，直到跨越为止。
    你看到别人追女生很轻松，自己却很困难，那是因为你只看到了表面。**任何表面上的轻而易举，都建立在背后无数次努力的基础之上**。所以，理解并接受障碍区是一个很重要的概念。

2.  **拐点**
    拐点是指你不再需要别人教导，自己获得了前行动力的时刻。有一天你去健身不再需要教练，谈恋爱不再需要我教，你自己就获得了前行能力，这就是拐点。
    出现拐点意味着你对这件事有更大的积极性，但不能骄傲。因为拐点是你通往下一个阶段的起点。例如，你突破了社交恐惧去搭讪，恭喜你通过了一个拐点。但恋爱过程中的拐点很多：展示面打造、聊天、暧昧、约会牵手、接吻……每个拐点都需要去突破，都不能掉以轻心。
    **不断通过拐点，意味着你获得了自动前行的力量**。当你出现一个拐点，就意味着你通过了上一个障碍区。

3.  **状态的回落**
    很多人在这个时候摔了跟头，就再也爬不起来。好不容易突破了各种障碍区，通过了各种拐点，终于交到了称心如意的女朋友。但这时你开始得意洋洋，放松警惕，现实就会让你摔一个大跟头。
    有经验表明，往往淹死的都是会游泳的人。大家觉得自己很OK，结果在水里就淹死了。这就是状态的回落，千万不能掉以轻心。**尤其是在你突破拐点的时候，你的状态一定会回落**。就像开车的人说“我从来没出过事”，结果最后多少都会出点小事故，问题就在这里。

所以，学习恋爱、掌握恋爱能力是一个过程。**它一定会经历以上三个概念区域：障碍区 -> 拐点 -> 状态回落**。包括现在的我也还在学习，还在突破障碍区，通过拐点，应对状态回落。
其实学习恋爱不是太困难的事情。简单说，**把每一个流程该犯的错误都犯一遍就好了，下一次你就知道错在哪了**。

## 第二部分：用成长型心态看待事物 🌱

上一部分我们介绍了将恋爱视为技能的学习过程，本节中我们来看看看待事物的两种根本心态。

所有人看待事物可以分为两种。第一种人，用**成败**的概念来看待事情，他们相信缘分，相信某事是上天注定的。第二种人，相信**成长**，相信事在人为。我对大家的要求是：从今天开始做第二种人。

很简单，如果你把事情看作成与败，就会有一种宿命的心态。**一定不要把恋爱看成成功与失败**。
之前有学员跟我说：“我觉得我上一段感情真的很失败，我什么都没有得到。” 我告诉他，你错了，你得到了：你再也不用为这个女人浪费一秒钟时间了。你至少知道这个人不适合你。**这就是你在这段关系里得到的最大的成长**。

所以，看待每件事情，悲观的人永远是消极的，乐观的人永远是积极的。我们从今天开始就做第二种人。
从一件事情里面得到一个成长，哪怕没有得到任何具体东西。在这过程中，你懂得了什么教训，知道了下次不能再犯哪些错误，这就是你得到的成长。
例如，你去跟别人聊天、社交、约会、谈恋爱，你一定要从这件事里面得到什么成长。**这是最重要的核心关键**。

## 总结与回顾

本节课中我们一起学习了构建正确恋爱心态的两个核心方面。

1.  **爱是一种可以习得的能力**。我们可以学会，但要注意在学习过程中不要建立过高的预期。我们要注意穿越障碍区，注意如何通过拐点，并警惕自己的状态是否会回落。这是最基本的学习心态。
2.  **人分为两种心态**。第一种是把事情当作成功与失败，第二种是把事情看作一种成长。希望大家从今天开始，尤其在脱单道路上，**一定要把自己的心态调整到“成长型心态”**。

希望大家通过今天这堂课能够懂得，心态不是虚的，而是实实在在指导我们行动和进步的基石。
