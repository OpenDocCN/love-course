# 54、挽救婚姻有办法：冷静与冷漠的区分

**概述**：本节课将探讨挽救婚姻时如何区分冷静与冷漠，并强调选择合适的咨询师的重要性。

**一、冷静与冷漠的区分**

1. **冷静**：
    - 面对婚外情，不采取冲动行为，如抓奸、闹单位、打小三等。
    - 选择合适的咨询师进行咨询，而非盲目选择。
    - 了解咨询师的能力和经验，避免被无资质的咨询师误导。

2. **冷漠**：
    - 发现婚外情后，无悲伤、无痛苦，无任何反应。
    - 对婚姻不关心，将其视为他人的婚姻。

**二、选择合适的咨询师**

1. **了解咨询师**：
    - 查看咨询师的文章、录音等，了解其婚姻观念和挽救婚姻的方法。
    - 了解咨询师的背景和经验，避免选择无资质的咨询师。

2. **避免误区**：
    - 不要被低价吸引，选择合适的咨询师比价格更重要。
    - 不要盲目相信咨询师，要了解其真实能力。

**三、挽救婚姻的正确方式**

1. **尽快采取行动**：发现婚外情后，要尽快采取行动，避免错过最佳时机。

2. **选择合适的咨询师**：选择有经验、有资质的咨询师，进行专业的婚姻辅导。

**总结**：本节课中我们一起学习了挽救婚姻时如何区分冷静与冷漠，以及选择合适的咨询师的重要性。希望这些知识能帮助大家更好地挽救婚姻。
