## 第一部分：呼吸练习 🌬️

现在，我们正式进入第一个部分：呼吸练习。正确的呼吸是瑜伽练习的基石。

### 掌握呼吸的三个要点

许多初学者在跟随老师呼吸时会感到困难或憋气。掌握以下三点，可以帮助你找到顺畅的呼吸节奏：

1.  **接纳自己的呼吸**：首先接纳自己当下呼吸的长度和深度，不必强求。
2.  **先呼气，再吸气**：这是一个小窍门。先将气完全呼出，再吸气，你会感觉呼吸更饱满、深长。这就像一块海绵，只有先挤干水分，才能吸收更多。
    *   **类比公式**：`清空 -> 填充`
3.  **呼吸要有“留白”**：吸气不要吸到尽头，呼气也不要呼到完全没气。在吸与呼之间留有弹性空间，让呼吸自然流动。

**核心原则**：在课堂上，如果跟不上老师的呼吸节奏，不必勉强。只需跟随自己当下顺畅的呼吸即可。

### 呼吸练习实践

接下来，我们通过一个简单的坐姿来感受和练习呼吸。

以下是呼吸练习的步骤：

1.  **准备坐姿**：
    *   选择一个稳定舒适的坐姿（如散盘）。
    *   如果无法挺直腰背，可以将**瑜伽砖**垫在臀部下方，让坐骨扎根，脊柱自然延展。
    *   如果可以坐直，可以坐在**瑜伽毯**上。
    *   双手轻放于双膝。
2.  **调整身体**：坐骨向下扎根，脊柱向上延展。绕动肩关节放松，然后下巴微收，闭上眼睛。
3.  **观察呼吸**：将感官向内收，成为自己呼吸的观察者。先自然地将气呼出，再自然地吸气。
4.  **引导呼吸**：找到自己的节奏后，尝试轻柔地延长呼吸。
    *   **吸气**：想象在闻花香，轻柔、缓慢、伸长。
    *   **呼气**：想象海水退潮，缓慢、彻底，但不到尽头。
5.  **回归自我**：如果跟随引导有困难，随时回到自己舒适的呼吸节奏。在瑜伽练习中，应始终追随自己清醒、有觉知的呼吸。

**呼吸的作用**：让吸气为身体带来能量和氧气，让呼气释放身体的紧张与疲惫。以平静、放松的状态进入接下来的练习。

慢慢睁开眼睛，我们即将进入热身部分。

## 第二部分：热身练习 🔥

掌握了呼吸方法后，我们开始活动身体，为体式练习做好准备。

### 颈部热身

首先从灵活的颈部开始。

1.  吸气时，下巴向上伸展，感受颈前侧的延展。
2.  呼气时，自然低头向下。
3.  重复5-6次，保持呼吸顺畅。

**重要提示**：向上伸展时，应感觉颈部前侧拉长，而非向后挤压颈椎。可以用食指抵住颈部中间，引导向上推高的感觉，避免颈后侧受压。

### 脊柱与肩部热身

接下来，结合手臂和肩部，进行脊柱四个方向的灵活练习。

以下是具体的动作序列：

*   **侧伸展**：
    1.  双手合十于胸前。
    2.  吸气，十指交叉，掌心向上推高，抬头看手背。
    3.  呼气，左手轻落地面（或指尖点地），右臂沿右耳向远方伸展，感受右侧拉伸。
    4.  吸气，双臂向两侧打开。
    5.  呼气，双手合十。换另一侧重复。
*   **扭转**：
    1.  吸气，双手推高向上。
    2.  呼气，身体扭转向左后方，右手背推左膝外侧，左手在身后支撑或轻触后背。
    3.  吸气，手臂再次向两侧延展。
    4.  呼气，双手合十。换另一侧重复。
*   **后伸展**：
    1.  吸气，双手推高，轻微打开胸腔。
    2.  呼气，双手来到背后十指交扣。
    3.  吸气，展开肩膀，提起胸腔，微抬头，感受胸前的伸展。
    4.  呼气，松开双手，还原。

### 脚踝热身

最后，活动常常被忽略的脚踝。

1.  双腿向前伸直，双手支撑于臀部后方。
2.  双脚打开与骨盆同宽，保持膝盖稳定。
3.  脚踝自内向外最大幅度绕圈5次。
4.  然后自外向内绕圈5次。
