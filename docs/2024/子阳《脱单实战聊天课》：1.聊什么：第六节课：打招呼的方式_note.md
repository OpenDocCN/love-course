# 子阳《脱单实战聊天课》：1.聊什么：第六节课：打招呼的方式

## 概述
在本节课中，我们将学习如何通过打招呼的方式引起女生的兴趣，并了解在开场阶段如何进行筛选。

## 核心概念
**公式**：开场打招呼的方式 = 简单朴素 + 真诚热情

## 以下是打招呼的方式

### 1. 删除花哨话术
*   **原因**：避免开场就使用幽默风趣的话术，难以判断女生是因话术而回复积极，还是对人感兴趣。
*   **目的**：在开场阶段进行筛选，判断女生是否对人感兴趣。

### 2. 简单朴素的开场方式
*   **方式**：使用最简单朴素的方式打招呼，例如询问对方近况或分享日常。
*   **目的**：避免引起误解，展现真诚热情。

### 3. 交换真实姓名
*   **目的**：建立信任感，拉近彼此距离。

### 4. 主动热情地开启对话
*   **目的**：展现自信，吸引对方注意。

## 总结
本节课中，我们学习了如何通过打招呼的方式引起女生的兴趣，并了解在开场阶段如何进行筛选。通过使用简单朴素的开场方式、交换真实姓名和主动热情地开启对话，我们可以提高与女生建立联系的成功率。
