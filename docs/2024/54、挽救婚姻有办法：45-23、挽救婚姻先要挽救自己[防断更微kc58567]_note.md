## 挽救婚姻的步骤

在本节课中，我们将学习如何挽救婚姻，首先需要关注的是自我救赎。

### 情绪崩溃的状态

以下是几种常见的情绪崩溃状态：

1. **情绪失控**：一说话就忍不住哭泣或愤怒。
2. **无法工作**：整天想着婚姻问题，如老公出轨。
3. **争吵和指责**：一说就要吵架，指责对方。
4. **情绪失控**：说着说着就情绪失控，砸东西等。

### 认知层面的问题

以下是几种常见的认知层面问题：

1. **对错观念**：认为婚外情是道德败坏的表现。
2. **婚姻观念**：认为婚姻是一辈子的，离婚就活不下去。
3. **心理问题**：严重的心理问题或精神问题。

### 挽救婚姻的步骤

要挽救婚姻，首先要挽救自己。以下是挽救自己的步骤：

1. **处理情绪**：通过认知和行为来处理情绪。
2. **增加认知**：从新的角度看待问题，如出轨可能表明对方还有爱的能力。
3. **行为改变**：进行户外运动、参加聚会等，提高心理能量。
4. **接受辅导**：寻求专业辅导，如心理咨询或婚姻家庭咨询。
5. **阅读书籍**：阅读专业书籍，如外国人写的书籍。
6. **听电台节目**：听朱生勇的电台节目，获取更多帮助。

### 总结

本节课中，我们学习了如何挽救婚姻，首先需要关注自我救赎。通过处理情绪、增加认知、行为改变、接受辅导、阅读书籍和听电台节目等方法，我们可以更好地挽救婚姻。希望这些方法能帮助到大家。
