# 瑞恩年会员PRO 瑞恩高级会员：课程23（长久不联系的女生如何破冰）

## 概述
在本节课中，我们将学习如何与长时间未联系且之前互动较少的女生重新建立联系。

## 重新开启话题的方法
以下是重新开启话题的步骤：

### 1. 观察朋友圈
- 当你看到心仪的女生发布朋友圈时，注意她分享的内容。

### 2. 根据内容开启话题
- 例如，如果她分享了一个面包店的图片，你可以询问：“买面包排队人多不？”

### 3. 保持中立
- 在对话中保持中立，不要表现出过多的需求感。

### 4. 沟通感受
- 在对话中分享你的感受，例如：“我看你发的图，有点馋。”

### 5. 保持对话
- 通过对话保持联系，不要急于邀约见面。

## 核心概念
**公式**：重新开启话题 = 观察朋友圈 + 根据内容开启话题 + 保持中立 + 沟通感受 + 保持对话

## 总结
本节课中，我们一起学习了如何与长时间未联系的女生重新建立联系。通过观察朋友圈、根据内容开启话题、保持中立、沟通感受和保持对话，我们可以有效地与心仪的女生重新建立联系。
