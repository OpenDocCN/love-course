# 16.隔壁王司机情绪自由的真相：第十课：负面情绪的处理（3）

在完形疗法中，有一个非常重要的概念，称为**躯体化**，即人的所有情绪都与身体有联系。因此，不同的情绪会导致身体出现不同的反应。例如，愤怒时，血液会迅速流向双臂，以便进行防御和攻击。有时，血液流动过快、量过猛，会导致血液流向头部，使脸部通红，浑身发抖。

恐惧时，身体的表现是害怕，因为害怕是自我保护，需要逃跑。因此，逃跑主要依靠双腿，血液会迅速流向双腿，肌肉紧绷，血流速度加快，便于逃跑。但有些人血液流动过快，会导致头顶部分血液不足，嘴唇发白，脸色苍白，称为“面如死灰”，双腿因恐惧而颤抖。

在行为疗法中，有一个重要概念是情绪会影响肢体动作。同样，肢体动作也会反过来影响情绪。例如，开心时的肢体动作和悲伤、愤怒等情绪时的肢体动作。我们常用“扬眉吐气”、“抬头挺胸”等词语描述开心时的肢体动作，而“垂头丧气”、“无精打采”等词语描述悲伤时的肢体动作。

NLP中的神经语言学认为，情绪会通过肢体动作表现出来。如果你找到这些肢体动作并完成它们，你会在内心产生相应的情绪。即使没有好事发生，通过肢体动作也能感受到快乐。

以下是一个例子：17岁时，作者在北京一家公司做教育培训。公司擅长“大鸡血”，每天早上开会后跳舞，释放压力。作者在面临业绩压力时，通过跳舞和喊口号，调整情绪，克服困难。

当面对负面情绪时，可以通过以下方法调整情绪：

1. **肢体动作**：通过调整肢体动作，如跳舞、做运动等，调整情绪。
2. **内在确认**：对自己说“我很棒”、“我很好”等，增强自信心。
3. **喊口号**：通过喊口号等方式，激发内在动力。

在应对生活和挑战时，先调整情绪，稍作缓冲，再通过完形疗法等方法，觉察内在情绪，释放负面情绪。

**总结**：本节课我们学习了如何通过肢体动作、内在确认、喊口号等方法调整情绪，以应对生活和挑战。希望这些方法能帮助大家更好地管理情绪。
