# 承情《万场约会大师课》：第二课：约会地点：占据地利，轻松拿捏她。

## 概述
在本节课中，我们将探讨约会地点的选择，如何占据地利，轻松拿捏约会对象。

## 约会地点选择原则
以下是约会地点选择的原则：

### 1. 约会地点选择权
- **坚定自己的想法**：选择地点时，应坚定自己的想法，而不是提供多个选项或进行开放式提问。
- **提供选项**：向对方提供几个选项，让对方选择。
- **开放式提问**：不建议使用开放式提问，因为这会让对方感到被动。

### 2. 约会地点选择地点
- **选择熟悉的地方**：选择双方都熟悉的地方进行约会，避免选择不熟悉的地方。
- **坚定自己的想法**：坚定自己的想法，将压力给到对方。
- **折中地点**：30%的情况下选择折中地点，70%的情况下选择自己熟悉的地方。
- **不建议去对方家附近**：不建议选择对方家附近作为约会地点。

### 3. 约会地点选择类型
以下是几种适合的约会地点：

- **商业街区**：适合内向性格的人，可以边逛街边聊天。
- **电影院**：适合内向性格的人，可以避免尴尬。
- **咖啡厅/餐厅**：适合外向性格的人，需要自己主导话题。
- **密室/剧本杀**：适合外向性格的人，可以增加互动。
- **按摩店**：适合外向性格的人，可以进行更深入的交流。
- **酒吧/回家**：适合酒量好、夜场技巧好的人。
- **外卖到家**：适合高阶玩家。

## 妹子不同意怎么办
如果妹子不同意过来，可以采取以下方法：

1. **无脑坚持两次**：无论对方拒绝的原因是什么，先坚持两次。
2. **给予压力**：通过坚持，给对方压力，让对方感到你有主见。
3. **展现诚意**：通过坚持，展现你的诚意，让对方知道你真的很想约她。
4. **改套路**：如果对方还是不同意，可以改变策略，让对方给出具体的时间。
5. **不要复盘**：在给对方压力的过程中，观察对方的借口是否真诚。
6. **直接反击**：对于有一定约会经验的人，可以尝试直接反击，指出对方态度不端正。
7. **甩态度**：如果对方还是不配合，可以甩掉对方，不再联系。

本节课中，我们学习了约会地点的选择原则、适合的约会地点类型以及妹子不同意怎么办的方法。希望这些内容能帮助你更好地掌握约会技巧。
