# 54、挽救婚姻有办法：冷静与冷漠的区分[防断更微kc58567]

## 概述
在本节课中，我们将探讨挽救婚姻时如何区分冷静与冷漠，以及如何选择合适的咨询师。

## 冷静与冷漠的区分
以下是冷静与冷漠的详细区分：

### 冷静
1. **面对婚外情**：不冲动，不采取暴力行为，仅限于夫妻之间解决问题。
2. **选择咨询师**：选择有专业背景和经验的咨询师，避免选择水平差的机构。

### 冷漠
1. **对婚外情无反应**：没有悲伤、痛苦，甚至没有失眠。
2. **缺乏关注**：对婚姻缺乏关注，如同旁观者。

## 选择合适的咨询师
以下是选择合适咨询师的建议：

1. **查看咨询师背景**：了解其专业背景、经验以及文章、录音等。
2. **避免选择水平差的机构**：避免选择那些仅提供情感安慰、不解决实质问题的机构。
3. **了解咨询师能力**：了解其解决问题的能力，以及是否能够提供有效的督导。
