# 安老师社恐训练营：02-灵魂追问：都是社交恐惧惹的祸 😰

在本节课中，我们将要学习如何正视社交恐惧对我们生活造成的广泛影响。只有深刻认识到问题所在，我们才能下定决心去改变它。

社交恐惧给我们的生活带来了巨大的困扰。我们在生活中因为社交恐惧，不敢谈恋爱，不敢跟异性多接触，相亲也总是失败。社交恐惧严重影响我们的人际关系，让我们害怕机会，不愿意参加社交活动。

对于工作而言，社交恐惧也产生了严重的负面影响。它影响我们找工作，影响面试表现，即使在工作后，与同事的关系也可能不佳，沟通不顺畅。这种情况有时会导致频繁更换工作。长此以往，社交恐惧不仅影响我们的情绪，还会损害我们的身心健康。

社交恐惧对我们个人发展构成了严重的阻碍。我们因此错失了许多机会，无论是恋爱、工作、生活，还是许多社会资源，我们都难以获得。所以，如果我们想要变得优秀，**必须克服社交恐惧**。

我十几年前上大学时就有社交恐惧。当时我认为，如果不克服社恐，未来的工作和生活将充满意想不到的挫折与困难，举步维艰。因此，经过不懈努力，我在毕业前终于克服了社恐。正是因为走出了社恐，我后来的工作和生活才相对比较顺畅。

我先是在大学当老师，后来离开学校，在一家头部教育机构做了四年的研发经理。如果当时没有克服社恐，可能后来的工作生活将大不相同。有时候，克服社交恐惧并不意味着一定会有优秀的发展或传奇的人生。

我们想要的只是跟其他人一样，拥有正常的人际关系，正常的生活和工作。因为只有这样，我们才能在这个基础上变得更好。本课程是我经过多年研究，浓缩了许多经典的策略、方法、技巧，以及非常科学的思维模式。

这些内容已经帮助了很多人成功实践。如果我们学完课程并严格按照课程内容去实践，一个月的时间，即使不能脱胎换骨，至少也能让别人刮目相看。我们自己也会感到焕然一新，对自己充满自信，做很多事情时会得心应手。

上一节我们介绍了正视社交恐惧的重要性，接下来我们看看整个课程的安排。

以下是本训练营的课程结构概览，一共十二课时。从第二节课开始，大家一定要认真听讲。

*   **第一步：全面了解**。首先，我们要全面了解社交恐惧，包括如何界定社交恐惧，造成社交恐惧的原因是什么。
*   **第二步：识别分类**。我们将学习社交恐惧有哪些分类，例如：赤面恐惧、视线恐惧、表情恐惧、异性恐惧、权威恐惧等。只有当我们真正了解它之后，才能去解决它。
*   **第三步：学习方法**。课程中会讲解非常详细的克服社恐的方法和实践操作。
*   **第四步：掌握核心**。我们会学到与任何人相处都不紧张的**秘密**，以及如何变得自信和强大。
*   **第五步：实践应用**。课程提供了像“庖丁解牛”一样，能基本应对所有社交场合的方法与技巧。

如果我们能理解这些内容的思路和技巧，那么将会变得无比强大。最终目标是达到一去不返的状态，让社恐永远不再复发。

本节课中，我们一起学习了社交恐惧在恋爱、工作、个人发展等多方面造成的具体阻碍，并概述了本训练营将如何系统性地帮助我们克服它。认识到问题是改变的第一步，我们已经迈出了这关键的一步。同学们，你们都准备好了吗？咱们开始吧。
