## 什么是情绪付出？

上一节我们探讨了占据关系高位的方法，本节中我们来看看如何强化这种格局。首先，必须明确“情绪付出”的概念。

这里的付出，特指情绪上的付出。它不是指发红包、充话费或买礼物这类金钱付出。如果那样做，会显得特别低级。有些人在网上学了一些错误方法，例如自己点肯德基却让女人买可乐，就认为让女人付出了，然后坐等对方投怀送抱，这是非常愚蠢的。

那么，情绪付出是什么呢？回想一下，你是如何对一个女人进行情绪付出的。其实很简单。

**公式：情绪付出 ≈ 你的头脑资源被对方占据**

具体来说，就是你脑子里总是想着她。她经常性地出现在你的头脑里，悄无声息地占用了你的精力和时间，这就是你在对她进行情绪付出。这种付出是一种被动、无意识的付出。她在没有觉察的情况下，就已经在对你付出了，而不是那种主动的、有意识的付出，比如给你买个礼物或讲个笑话来取悦你。

在探讨具体方法前，必须先理清这里的逻辑。否则，你可能会误以为我们所说的付出与其他地方提到的付出是一回事，其实并非如此。

因为底层逻辑不同。在其他地方，你看到的付出可能是为了吸引女人。而我们这里，必须回到整个主导关系的链条上来理解。

**核心概念：付出是对关系位置的强化。**

只不过，男人需要的付出和女人需要的付出不同。女人往往要的是男人在价值（如金钱、时间）上的付出，而男人则需要女人在情绪上对男人付出。当一个女人持续对你进行情绪付出时，你就经常出现在她的头脑里，那么你与她的关系位置就不言而喻了。

## 高位时的引导付出技巧

当处于关系高位时，引导对方付出具体怎么做呢？我们知道，付出的核心就是让对方的头脑里经常出现你。

所以，男人说这句话通常没什么用。这只会显得你：第一，没有选择权（好像只有她一个选项）；第二，没有生活掌控权（生活里好像只有她）；第三，没有自我满足权。这违背了我们常说的三个场外权利。

要主导关系，你就要成为她脑袋里经常出现的人，而不是让你的脑袋经常被她占据。因为你在高位，你就得保持高位的一致性。这是心法：你得第一，回到选择权中；第二，回到生活的掌控权中；第三，回到自我的满足权中。

实际上，只要你悟到了这点，你就会明白，这样的男人一定会经常出现在女人的头脑里的。她们会琢磨你、思量你，最后好奇地跑过来问：“你怎么这么忙？都不找我？”这事其实也基本上就成了。

以下是几个引导情绪付出的具体操作：

**第一，只调情，不要求结果。** 什么叫只调情不要求结果呢？举个例子：
```

你说：“想你呢。”
她说：“切。”

然后你说：“啊，你昨晚吃的啥呀？”
```
看到没有？光调情，也不要什么结果，也不去邀约，也不去考虑你们是什么样的关系。不约，不在乎是什么关系，只有轻松愉悦的情绪。刚才这个就是叫做“只调情，不要求结果”。
