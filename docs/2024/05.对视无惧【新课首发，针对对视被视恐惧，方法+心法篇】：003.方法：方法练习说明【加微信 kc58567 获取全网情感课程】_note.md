# 课程05：对视无惧 🫣 👀

## 003. 方法：方法练习说明

在本节课中，我们将详细讲解克服对视恐惧的具体练习方法。上一节我们强调了直视是解决对视恐惧的唯一途径，本节中我们来看看如何通过一系列方法，消除直视前的恐惧障碍，从而真正实现“对视无惧”。

### 概述：恐惧是根本障碍

直视是解决对视恐惧的唯一办法。但在能够自如直视之前，恐惧感是最大的障碍，这也是许多人无法大方看人的根本原因。

只有消除恐惧感，那些因担心害怕而产生的刻意行为控制（如控制状态、控制行为）才会消失。随之，后续的失控感等一系列连锁反应也将不复存在。对视恐惧的恶性循环就此打破，症状也就消失了。

### 第一层方法：主动直视 😊

主动直视是消除恐惧的第一步。你可以尝试主动去看别人，主动进行直视。在这种状态下，你的恐惧感会弱很多。

因为当你主动时，你处于一种强势且有准备的状态。无论是进行肢体交流、言语交流还是目光交汇，你都会感觉自己更有掌控力。这类似于社交中的“先发制人”。

相反，如果你处于被动状态——比如成为别人言语或行为的对象，或是被别人先看到——你会感到措手不及、毫无准备，更容易陷入恐慌。

**核心要点**：主动 = 强势状态 = 消除恐惧。主动状态大约能消除一半的恐惧感。

#### 什么是真正的“主动”？

“主动”不应被狭义地理解为“主动直视”。它可以是各种行为的主动，甚至是思维上的主动。

以下是主动行为的例子：
*   主动与他人攀谈。
*   主动去看。
*   主动帮助他人（例如，看到别人有小麻烦，过去帮忙）。

所有的主动行为都会让你处在一种强势的状态。在这种状态下，你会清晰地感受到自己能够开口说话，能够睁眼看对方。

#### 实践案例：地铁场景

我曾有伴随对视恐惧的余光恐惧症。我的解决方法是：一上地铁，坐在人群中后，我会主动扫视周围的人。

扫视一圈后，我可能会发现某个有趣或好看的人。如果我想看，我就开始看，一直看到自己不想看为止。如果转头后我又想看了，我就再次转过去看，直到看满意为止。

**关键情况处理**：如果对方发现我在看他，并与我产生对视，怎么办？

答案是：接着看。

你可能会担心一直盯着别人会引起误解。但请记住，此时你能做的就是继续看。如果对方一看你，你就回避眼神，你就再次陷入了对视恐惧的恶性循环，等于在这次社交中经历了一次完美的失败。

**可能的结果与应对**：

1.  **最佳结果**：你一直看，对方先转过头或低下头了。无论他是出于不好意思还是觉得无聊，这都是最好的结果。这会在你内心植入一个成功的记忆：原来我可以大大方方地把别人看到底。
2.  **对方也不转头**：如果双方僵持，你感到不舒服，也无需强逼自己看到底直至发生争执。此时有多种处理技巧。

**处理技巧示例**：
*   **对男性**：我可以转过头，微微一笑。同时，内心进行一种强势的自我暗示（例如，心里默念“对方真二”）。这虽然是我转头了，但我通过心理暗示告诉自己：是对方有问题，我只是懒得理他。我并没有在社交中落败。
*   **对女性**（尤其是出于欣赏）：一直盯着看确实可能不礼貌。你可以选择转头，然后会心一笑。这种笑容代表你对这次“欣赏”的满足和认同，也是一种积极的自我暗示。

**核心原则**：无论是带有强势攻击性的暗示，还是带有幸福感的认同，这两种状态都是一次完整的、成功的社交体验，会在内心留下积极正向的经验。

**绝对禁忌**：在整个过程中，**千万不要被迫低头**。如果你是因为恐惧而被迫低头，你就重新进入了“恐惧 -> 控制 -> 失控”的循环，这次社交就是失败的，只会带来更大的恐惧。

### 第二层方法：心理暗示

讲完上面的案例，我们方法的全貌就清晰了：除了主动直视，我们还需要加上**心理暗示**。

心理暗示的作用与主动类似，也是用于消除（更准确地说是**转化**）恐惧感，大约能消除剩余恐惧感的四分之一。

#### 如何进行心理暗示？

当使用心理暗示转化恐惧时，需要在心中默念一些比较粗俗、激进或带有侮辱性的词汇。例如，在地铁案例中，我对那位男性回头一笑时，内心暗示的是“对方真二”。

这种暗示帮助我将原本可能向内自责的能量（如“是不是我让人不爽了”、“是不是我的眼神有问题”），全部释放出来，转化成一种对外的、攻击性的心理能量。

**记住**：这**不是对抗**，也不是**逃避**。对抗会引发更强烈的反扑（强迫症本质）；逃避只会留下更弱势、自卑、怯懦的人格印记。

#### 为什么不用“接纳”？

应对恐惧其实有两种方式：**转化**（我们正在用的）和**接纳**。

我们不用“接纳”，是因为大部分人在极度惊恐状态下根本做不到真正接纳自己的情绪和症状。这也是为什么很多人运用“森田疗法”（顺其自然，为所当为）却仍走不出来的原因——在恐惧中坚持到能够“接纳”的阶段非常困难。

#### 有效暗示的标准

并非任何话都能作为有效的心理暗示。例如，一些正确的道理（如“别太在意他人看法”）虽然对，但无法撼动你当下的恐惧状态，属于“正确的废话”，无效。

有效的心理暗示语言需要符合以下标准：**强势、带有攻击性，甚至侮辱色彩**。因为这只是你在内心告诉自己。

**效果对比示例**：
*   “爱谁谁” -> 有效。
*   “爱他妈谁谁” -> 效果更好。
*   “就看你怎么了” -> 有效。
*   “就他妈看你怎么了” -> 效果更上一层楼。

在暗示语言中加入攻击性成分，效果会倍增。

当你通过暗示找回强势状态，你会发现“想看就看，不想看就不看”，一切由你说了算。这时，精神中不再有弱势、压力、扭曲或恐惧感。

### 第三层方法：松弛状态

在上述地铁案例及日常场景中，我最喜欢的状态是一种**松弛化**的状态。

具体表现为：全身接近一种“局部瘫软”的状态。例如，正常走路时腿部需要用劲，但上半身可以保持非常放松、软绵的状态。在这种状态下，你的自由意识会特别强烈，应对恐惧的能力（或者说，恐惧感本身）会变得不那么明显。

这种松弛状态，本质上接近于“肌肉松弛疗法”，即通过控制身体节奏来反向影响精神状态。处于这种状态时，直视也会变得更加简单。松弛状态大约也能消除四分之一的恐惧感。

### 方法整合与原理总结

现在，我们将方法整合：**主动直视（消除1/2恐惧） + 心理暗示（消除1/4恐惧） + 松弛状态（消除1/4恐惧） = 恐惧感被完全消除**。

没有恐惧感，思维的恶性循环就无法形成，一系列问题就转不起来了，对视恐惧也就消失了。

#### 为什么没有“一招制敌”的简单方法？

可能有人会问：又是松弛，又是暗示，又是直视，有没有更简单的一招制敌的方法？**答案是没有**。

因为这些方法本身就源于普通人的日常行为。只是当你患上对视恐惧或社恐后，你在社交场景中会自动进入高度警惕状态，忘记了如何“松弛”，无法做到“主动直视”，也不会下意识使用“心理暗示”来调整自己。

我们学习这些方法，只是通过原理的讲解，让你相信并重新学会这些普通人天生就会的技能。我们与普通人的最大区别在于，我们需要**更高频、更强烈**地、有意识地去运用这些方法。

### 方法使用要领总结

本节课我们一起学习了克服对视恐惧的三大核心方法。以下是具体的使用要领总结：

#### 1. 如何进入并保持松弛状态

松弛状态的关键在于“局部瘫软”。

以下是实现松弛状态的一些方式：
*   **行走时**：下半身用力，但上半身保持放松、软绵。可以略带一点摇摆。
*   **站立时**（如在地铁上）：手扶栏杆，下半身呈半悬挂、下坠状态，实现局部瘫软。
*   **亢奋状态**：精神兴奋，但四肢肌肉放松，这也是一种松弛状态。
*   **行为调节**：放慢呼吸速度、放慢走路速度、刻意放慢思考和说话速度，都能引导进入一种轻飘飘的松弛状态。

这些状态之间可以随意切换。请根据你的喜好和生活场景自行选择。

#### 2. 如何执行主动直视

主动直视的核心是：**一直看，看到自己不想看为止**。

具体步骤如下：
1.  如果想看，就主动去看。
2.  看到自己满意、不想再看为止。
3.  如果之后又想看，就再次转过头继续看。
4.  **最重要原则**：千万不要让自己被迫低头。除非是你自己主动想低头。
5.  如果对视发生，继续看。
6.  如果对方先移开视线，你成功了。
7.  如果僵持不下令你不适，可以用一个强势的心理暗示作为结尾，然后自然移开视线（非被迫低头）。

#### 3. 何时介入心理暗示

心理暗示的介入**没有时间限制**。在任何你感到内心不爽、不自在，又无法通过其他行为宣泄的时候，都可以使用心理暗示来转化情绪。普通人也会这么做，只是他们没有意识到。
