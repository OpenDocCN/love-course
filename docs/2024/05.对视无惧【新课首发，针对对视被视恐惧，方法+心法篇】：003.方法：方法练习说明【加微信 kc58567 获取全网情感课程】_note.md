# 课程05：对视无惧 🫣 👀

## 003. 方法：方法练习说明

在本节课中，我们将详细讲解克服对视恐惧的具体练习方法。上一节我们强调了直视是解决对视恐惧的唯一途径，本节中我们来看看如何通过一系列方法，消除直视前的恐惧障碍，从而真正实现“对视无惧”。

### 概述：恐惧是根本障碍

直视是解决对视恐惧的唯一办法。但在能够自如直视之前，恐惧感是最大的障碍，这也是许多人无法大方看人的根本原因。

只有消除恐惧感，那些因担心害怕而产生的刻意行为控制（如控制状态、控制行为）才会消失。随之，后续的失控感等一系列连锁反应也会不复存在。对视恐惧的恶性循环就此打破，对视恐惧本身也就消失了。

### 第一层方法：主动直视 😊

主动直视是消除恐惧的第一步。你可以尝试一下，当你主动去看别人、主动进行直视时，你的恐惧感会弱很多。

在这种状态下，你处于一种强势且有准备的状态。无论是与他人进行肢体交流、言语交流还是目光交汇，你都会感觉自己更有掌控力，恐惧感自然减弱。

简单来说，这类似于社交中的“先发制人”。反之，如果你处于被动状态——比如成为别人言语或行为的对象，或是被别人先看到——你就会感到毫无准备、措手不及，更容易陷入恐慌。

**核心公式：**
`主动 = 强势状态 = 消除恐惧（约50%）`

当你处于主动状态时，大约一半的恐惧感会被消除。

#### 什么是真正的“主动”？

“主动”不应被狭义地理解为仅仅是主动直视。主动可以是各种行为的主动，甚至是思维上的主动。

以下是主动行为的例子：
*   主动与他人攀谈。
*   主动去看他人。
*   主动帮助他人（例如，看到他人有小麻烦，主动上前帮忙）。

所有的主动行为都会让你处在一种强势的状态。在这种状态下，你会清晰地感受到自己能够开口说话，能够睁眼看对方。

#### 实践案例：地铁场景

我曾有余光恐惧症，并伴随对视恐惧。我是这样解决的：上了地铁后，我坐在人群中间，会主动扫视周围的人。看一圈后，我可能会发现某个有趣或好看的人，然后就开始看，一直看到自己不想看为止。

如果转头后我又想看了，我就继续抬起头转过去看，直到感到满意、真的不想看为止，这才算结束。

**关键情况处理：如果对方发现我在看他，并产生对视怎么办？**

答案是：接着看。

你可能会担心一直盯着别人会引起误解。但请记住，此时你能做的就是继续看。如果对方一看你，你就回避眼神，你就再次陷入了对视恐惧的恶性循环，等于在这次社交中经历了一次完美的失败。

**可能的结果与应对：**

1.  **最佳结果**：你一直看，对方先转过头或低下头了。无论对方是因为不好意思还是觉得无聊，这都是最好的结果。恭喜你，一次完整的成功社交经验植入了你的记忆：原来我可以大大方方地把别人看到底。
2.  **僵持结果**：对方也不转头，双方一直对视。此时，你无需强逼自己看到底直至发生争执。你可以有多种得体的处理方式。

**处理技巧示例：**

*   **对男性（或感到不爽时）**：我可以转过头，微微一笑。同时，在心里进行一个强势的自我暗示（例如，默念一句带有贬义的话）。这虽然是我转头了，但在心理层面，我通过暗示告诉自己“是对方有问题，我懒得理了”，我仍然处于强势状态，这是一次成功的社交。
*   **对女性（或欣赏对象）**：一直盯着看确实可能不礼貌。转回头后，你可以选择会心一笑。这种笑容代表一种满足和幸福感，是对这次社交（欣赏）的一种积极认同和结束。

无论是带有攻击性的暗示，还是带有幸福感的笑容，这两种状态都代表了一次完整的、成功的社交经历，在你内心留下了积极正向的经验：我能够与对方正常完成社交流程。

**核心要点**：在整个过程中，**千万不要被迫低头**。如果你是因为恐惧而被迫低头，你就重新进入了“恐惧 -> 控制 -> 失控”的循环，这次社交就是失败的，只会带来更大的恐惧。

### 第二层方法：心理暗示

讲完上面的案例，我们方法的全貌就清晰了：除了主动直视，我们还需要加上**心理暗示**。

暗示的作用与主动直视类似，也是用于处理恐惧感。具体来说，它不是消除，而是一种**转化**，大约能转化掉**四分之一**的恐惧感。

当我们使用心理暗示来转化恐惧时，需要在心中默念一些比较粗俗、激进或带有侮辱性的词汇。例如，在地铁案例中，我对那位让我不爽的男士，回头一笑时心里暗示“对方是二逼”。这句话帮助我将原本可能内向的自责（“是不是我看人家导致人家不爽了？”）和压抑，转化成了对外的攻击性。

**核心要点**：这**不是对抗，也不是逃避**。对抗会引发反扑（强迫症本质），逃避则会强化自卑怯懦的人格。

#### 为什么不用“接纳”？

应对恐惧其实有两种方式：**转化**和**接纳**。我们之所以主要使用“转化”，是因为在极度惊恐的状态下，大多数人**无法做到真正的接纳**。这也是为什么很多人运用“森田疗法”（顺其自然，为所当为）却仍走不出来的原因——在恐惧顶点坚持到接纳的那一刻非常困难。

#### 有效暗示的标准

并非任何话都能作为有效的心理暗示。那些“正确但无用”的建议（如“别在意别人看法”）无法撼动你弱势的恐惧状态，是无效的。

有效的心理暗示语言需要符合以下标准：**强势、带有攻击性、甚至侮辱色彩**（反正是内心活动）。语言越强烈，效果通常越好。

**效果递增示例：**
*   “爱谁谁” -> 有效
*   “爱他妈谁谁” -> 效果更好
*   “看你怎么了” -> 有效
*   “就他妈看你怎么了” -> 效果更强

当你通过暗示找回强势状态，你会发现“想看就看，不想看就不看”，一切由你说了算。这时，精神中就没有了弱势、压力、扭曲或恐惧感。

### 第三层方法：松弛状态

在上述地铁案例及日常场景中，我最喜欢保持一种**松弛化**的状态。

这种状态接近于“局部瘫软”。例如，正常走路时腿需要用劲，但上半身可以保持非常放松、软化的状态。在这种状态下，你的自由意识会特别强烈，应对恐惧的能力似乎变强了——实际上，是恐惧感本身变得不明显了。

这种松弛状态，类似于心理学中的“肌肉松弛疗法”，通过控制身体节奏来反向影响精神。当处于这种状态时，直视也会变得更加简单。

这一部分大约也能消除**四分之一**的恐惧感。

### 方法整合与原理总结

现在，我们将方法整合起来：**主动直视（消除50%）+ 心理暗示（转化25%）+ 松弛状态（消除25%）= 恐惧感被完全处理**。

没有恐惧感，思维的恶性循环就无法形成，一系列问题就转不起来了，对视恐惧也就消失了。

#### 为什么没有“一招制敌”的简单方法？

有人可能会问，有没有更简单的一招制敌的方法？答案是没有。因为这些方法本身就源于普通人的日常行为。

*   **松弛状态**：正常人没事时就处于松弛状态。社恐者只是在社交场景下自动进入了高度警戒状态，我们需要方法帮自己找回松弛感。
*   **主动直视**：正常人看人就是想看就看，不想看就不看。社恐者只是暂时失去了这种能力，我们需要通过练习重新获得。
*   **心理暗示**：普通人也会用（比如争执后小声嘟囔一句）。只要对精神产生了影响，就是一次心理暗示。

我们与普通人的最大区别在于：我们使用这些方法时**更加高频、更加强烈**。并且，我们需要通过讲解原理，让你从心理学层面相信这些方法，你才会去使用它，从而解决问题。

### 方法使用要领总结

本节课我们一起学习了克服对视恐惧的三大核心方法。以下是具体的使用要领总结：

#### 1. 如何进入并保持松弛状态

松弛状态的关键在于“局部瘫软”。

以下是实现松弛状态的方式：
*   **行走时**：下半身用力，但上半身保持放松、软化，甚至可以略带摇摆。
*   **站立时（如地铁）**：手扶栏杆，下半身呈半悬挂下坠状态，实现局部瘫软。
*   **情绪层面**：亢奋状态也是一种松弛——精神兴奋，但四肢肌肉放松。
*   **行为调节**：放慢呼吸速度、放慢走路速度、刻意放慢思考和说话速度，都能引导进入一种轻飘飘的松弛状态。

这些状态可以随意切换，请根据喜好和场景自行选择。

#### 2. 如何执行主动直视

主动直视的核心是：**一直看，看到自己不想看为止**。

具体步骤如下：
1.  如果想看，就主动去看。
2.  看到自己感到满意、不想再看为止。
3.  如果之后又想看，就再次转头继续看。
4.  **最关键**：千万不要被迫低头。除非是自己主动想低头。
5.  如果对视发生，继续看。
6.  如果对方先移开视线，你成功了。
7.  如果僵持不下令你不适，可以主动结束，但务必配合**心理暗示**来做一个强势的收尾。

#### 3. 何时介入心理暗示

心理暗示的介入**没有时间限制**。在任何你感到内心不爽、压抑，又无法通过其他行为宣泄的时候，都可以使用心理暗示来转化情绪。普通人也是如此，只是他们未必意识到。
