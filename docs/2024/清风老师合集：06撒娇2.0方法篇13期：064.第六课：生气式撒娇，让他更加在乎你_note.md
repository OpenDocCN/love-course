# 清风老师合集：06撒娇2.0方法篇13期：064.第六课：生气式撒娇，让他更加在乎你

## 概述
在本节课中，我们将学习如何运用生气式撒娇，让伴侣更加在乎你的感受。

## 生气式撒娇的原理
生气式撒娇的目的是让对方意识到自己的行为有问题，并表达你的情绪和感受。其核心在于：

**公式**： 生气式撒娇 = 表达情绪 + 明确行动指令

## 生气式撒娇的适用场景
以下是几种适合使用生气式撒娇的场景：

1. **对方承诺做不到**：例如，承诺陪你，却沉迷游戏。
2. **对方与异性暧昧**：例如，在公共场合与异性过于亲密。
3. **对方行为让你不舒服**：例如，未经你同意就帮别人做事。

## 生气式撒娇的步骤
以下是进行生气式撒娇的步骤：

1. **表达情绪**：用带有娇嗔、指责意味的话语表达你的感受。
2. **给出行动指令**：明确告诉对方你希望他做什么。
3. **正向强化**：当对方完成行动后，给予他正向反馈。

## 生气式撒娇的注意事项
1. **避免无理取闹**：确保你的情绪和行动指令合理。
2. **保持冷静**：避免情绪失控，影响撒娇效果。
3. **适时调整**：如果对方不理解你的感受，可以尝试其他方法。

## 总结
本节课中，我们学习了如何运用生气式撒娇，让伴侣更加在乎你的感受。通过表达情绪、给出行动指令和正向强化，你可以巧妙地运用负面情绪，提升感情质量。记住，情绪从来不是问题，关键在于如何运用它。
