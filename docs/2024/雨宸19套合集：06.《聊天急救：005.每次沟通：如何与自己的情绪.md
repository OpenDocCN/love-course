# 雨宸19套合集：06.《聊天急救：005.每次沟通：如何与自己的情绪

在本节课中，我们将要学习如何识别和处理沟通中的情绪问题。理解并管理好情绪，是避免沟通升级为“世界大战”的关键。

## 概述：情绪是沟通的放大器

冲突本身并不可怕。可怕的是情绪放大了冲突，让沟通更加困难。在一场低质量的沟通中，90%是情绪，只有10%是事情本身。人在情绪控制下，不仅无法接纳别人的意见，还会出现暴力沟通的情况。

很多人明白这个道理，但实际操作时却做不到。我们明知情绪占比过高是错误的，却总在重复这种行为。这背后存在一个关于男女思维和情绪处理方式的误区。

## 男女的情绪处理差异

上一节我们提到了情绪在沟通中的主导地位，本节中我们来看看这种情绪反应为何难以控制，这与男女的思维模式差异密切相关。

男人的思维倾向于“洞穴机制”。遇到问题时，男人会将情绪内化，需要独立的空间进行自我调节和思考。这就像男人在感情中需要“假期”，独处后能以更好的状态回归。公式可以表示为：**男人情绪处理 = 内化 + 独处思考**。

女人的思维则更倾向于“发泄机制”。遇到问题时，女人习惯将情绪外化，通过倾诉等方式发泄出去，然后再思考问题。公式可以表示为：**女人情绪处理 = 外化倾诉 + 后续调节**。

正是这种差异导致了沟通中的错位：女人想说，男人想逃。男人并非想逃避问题，而是在用他的方式消化事件；女人则希望通过沟通来化解内心的焦虑。因此，我们常说：男人爱讲道理，女人爱谈感受。

## 正确处理情绪的三步法

了解了差异后，我们该如何在沟通中正确管理情绪，避免被其控制呢？以下是三个核心步骤。

### 1. 应对与感受你的情绪

情绪影响沟通，是因为我们下意识地抵抗情绪，认为有情绪是不好的。接纳情绪，正是跳出被情绪驱使的过程。

接纳意味着承认感受。例如：我很愤怒，我就是愤怒；我很委屈，我就是委屈。沟通的三要素可以概括为：**描述事实 + 描述感受 + 提出需求**。当你认识到并说出情绪时，你就变得更强，而不再被它控制。逃避只会让情绪继续掌控你。

### 2. 分析你的情绪来源

很多时候，我们的情绪源于不合理的认知。例如：“对方不回消息 = 不爱你”；“他和异性打招呼 = 移情别恋”。这些不合理认知会催生不合理情绪。

要分析情绪来源：你的愤怒是源于对方的确切行为，还是源于你内心的不安全感？不安全感可能源于自我价值感低或过往经历。你的看法决定了你解读事实的方向。如果你总试图证明对方会出轨，最终可能真的导致这个结果。学会分析，才能看清真相。

### 3. 划定边界，不被对方情绪影响

在沟通过程中，要感受对方的情绪，但不能被其裹挟。尤其当对方释放大量负面信息时，不要将他的难过自动归咎于自己，不要产生不必要的负罪感。

需要分清：他的情绪是针对你，还是针对他自己？很多时候，对方的愤怒只是他自身情绪无处发泄，而你恰好成为了倾诉对象。理性看待这一点至关重要。面对感情问题，我们可以感性谈论，但必须理智分析。

## 进阶技巧：关注情绪背后的需求

处理好了自身情绪，我们还可以更进一步，通过理解对方来化解冲突。情绪爆发往往是因为当事人感觉不到足够的爱和关注。

在解决问题前，先表达关爱与理解，这既是安抚，也是鼓励。关键技巧是：抓住对方宣泄情绪的真正原因。

例如，当对方焦虑地指责你时，你可以说：“我感觉你很焦虑，你能说说为什么吗？” 这句话表明你读懂了他，他背后的需求可能是寻求理解、支持或力量。倾听并点明他情绪背后的真实需求，往往能瞬间缓和局面。

## 总结：管理情绪，提升沟通质量

本节课中我们一起学习了如何与自己的情绪和解，以改善沟通。

管理情绪的本质目的，是为了在沟通过程中不让情绪影响决策。很多沟通问题，并非事情本身多严重，而是情绪干扰了理性判断，降低了沟通质量。

记住这三个要点：**接纳感受、分析来源、划定边界**。通过练习，你将能在沟通中更好地保持理性，有效化解小冲突，避免“世界大战”。
