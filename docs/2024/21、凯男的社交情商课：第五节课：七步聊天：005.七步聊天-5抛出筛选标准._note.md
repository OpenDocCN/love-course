# 21、凯男的社交情商课：第五节课：七步聊天-5抛出筛选标准

## 概述
在本节课中，我们将学习如何进行七步聊天的第五步——抛出筛选标准。这一步是在你已经完成了前四步，即开场白、距离感过渡、基本信息交换和建立男神框架之后进行的。

## 第五步：抛出筛选标准

- **前提条件**：在抛出筛选标准之前，你需要确保你已经完成了前四步，并且对女生产生了初步的吸引。
- **目的**：筛选标准的目的是为了让我们与女生之间更加契合，并且确保她能够通过我们的筛选。

### 2. 筛选内在特质
- **原因**：筛选内在特质比外在特质更为有效。
- **例子**：

  - **筛选例子**：
    - “我喜欢孝顺的女人，我觉得孝顺的女人很可爱。”
    - “我喜欢有爱心的女孩子，我觉得有爱心的女孩子很有魅力。”

- **原因**：
  - **不可改变**：外在特质如外貌、身高、胖瘦等是女生无法改变的。
  - **安全感问题**：过度关注外在特质会让女生感到没有安全感。

### 5. 注意事项
- **灵活运用**：不要生搬硬套话术，要学会随机应变。
- **话术目的**：话术是为了我们自己使用，而不是给女生使用。

## 总结
本节课中，我们学习了如何进行七步聊天的第五步——抛出筛选标准。通过筛选内在特质，我们可以找到与我们更加契合的伴侣。记住，灵活运用话术，不要生搬硬套，才能在聊天中取得更好的效果。
