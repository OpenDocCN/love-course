# 情绪梳理课程 17：负面情绪卸载 🧘‍♀️

在本节课中，我们将学习一个情绪梳理的个案过程，核心是通过引导想象，将童年时期的恐惧、厌恶等负面情绪，以及从家族中传承的焦虑、暴躁能量进行识别、卸载与转化，并用爱与温暖填充内心。

## 从觉察到卸载：处理童年恐惧 🐱

上一节我们介绍了课程概述，本节中我们来看看如何处理具体的童年负面情绪。个案首先被引导去觉察对特定对象（如猫）的感受。

你对他心里的感受有厌恶和害怕吗？没有害怕。有厌恶。

个案被问到是否想离开引发恐惧和厌恶的场景，去到一个感到舒服的地方。个案想去草地，旁边有水的地方，就像老家的大河边。

以下是引导卸载童年情绪的关键步骤：
1.  **引导至安全场景**：将内在的“小我”（个案自称“小妮”）带到那个令人舒适的草地与河边。
2.  **卸载情绪包袱**：将三四岁时，看到猫所产生的整个身体的、心理的、情绪的、思想的东西全部卸载掉。
3.  **选择卸载方式**：个案选择用“空气”作为卸载这些情绪的媒介。
4.  **执行卸载过程**：通过吸气、呼气，将身体的、情绪的、心理的全部卸掉。

完成卸载后，个案感受当下的状态。心里总体上比较放松，之前心里发抖的感觉消失了。四肢也感觉相对要放松一些。厌恶感基本上可以忽略。

## 识别与归还：家族情绪模式传承 🔄

处理完即时的恐惧后，我们进一步探讨更深层的情绪模式。引导者指出，整个情绪的“不稳定”其实带来的伤害是最大的。

当被问及自身情绪是否稳定时，个案描述自己有时会很焦虑、很急躁。并分析了家族模式：妈妈会直接将焦虑和暴躁表达出来；而自己像爸爸，更多是把东西压在心底，有暴躁但更多是压抑。

引导者指出，这不稳定的、焦虑暴躁的部分是从外婆传到妈妈，再传到个案这里的，像一个代际传承的负担，并不真正属于个案。

以下是处理家族传承情绪的关键步骤：
1.  **识别情绪来源**：确认焦虑暴躁的部分是从外婆、妈妈那里传承而来。
2.  **意愿确认**：个案表示特别想将这部分能量送回去。
3.  **具体化与分离**：个案用一个比喻来描述——妈妈转交了一个玉镯子（礼物），自己很喜欢。现在想把“礼物”中不好的“量化”的东西（焦虑暴躁）还回去，但把其中美好的部分（爱）留下。因为妈妈非常爱自己，这份爱需要被珍视和保留。
4.  **执行归还过程**：在想象中，将焦虑暴躁的部分，通过妈妈，还给外婆，外婆再继续向上归还，能传多远就传多远，直至送还宇宙。过程中，个案看到妈妈和外婆流泪，感受到复杂的情感，但坚持完成归还。
5.  **感受卸载后状态**：归还后，个案感觉从外婆再往上的部分，对自己就没有太大影响了。

## 锚定与填充：让爱充满内心 ❤️

在将童年恐惧和家族负担卸载、归还之后，内心腾出了空间。本节我们来看看用什么来填充这个空间。

引导者邀请个案感受卸载了所有负担后的状态，从头到脚去体会。个案感受到身边有很多的爱，来自父母、家人，自己被爱包围着，感觉挺幸福、挺温暖，内心的隔离感也减少了。

接着，引导者问个案，最希望用什么来替代那些被卸载的恐惧、害怕、孤单和不能融入的感觉。个案的答案是：爱。

以下是锚定内心之爱的关键步骤：
1.  **定义爱的形态**：个案将爱想象为一颗暖暖的红心，有温度，会跳动。
2.  **锚定与感受**：将这颗温暖、跳动的红心锚定在自己心里，并仔细感受。
3.  **观察爱的蔓延**：个案感受到，这颗心一开始有固定形状和大小，但随后它的光和温暖开始渗透到身体的各个器官和部位，蔓延开来，充满全身，带来非常舒适的感觉。
4.  **确认满足感**：当被问及还需要什么时，个案觉得有爱就够了。

## 课程总结 📝

本节课中，我们一起学习了一个完整的情绪梳理过程：
1.  **处理表层情绪**：通过引导至安全场景，用象征性方式（如空气）卸载童年时期的特定恐惧与厌恶。
2.  **处理深层模式**：识别出源自家族代际传承的焦虑与暴躁情绪，通过象征性“归还”仪式，将其送还来源，同时保留其中纯粹的爱。
3.  **重建内心力量**：在情绪卸载后，主动用“爱”的形象来填充内心，并感受其温暖和蔓延的力量，从而获得稳定、幸福与连接感。

这个过程的核心在于：**识别、分离、卸载、转化与重建**，帮助个体清理情绪负担，连接内在资源，建立更稳定积极的内心状态。
