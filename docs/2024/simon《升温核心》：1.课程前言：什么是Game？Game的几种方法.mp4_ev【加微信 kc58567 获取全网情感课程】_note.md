## 课程前言：什么是Game？Game的几种方法

### 概述
在本节课中，我们将探讨什么是Game，以及Game的几种方法。

### 什么是Game？
**Game**是一种社交技巧，旨在通过特定的策略和技巧来吸引和吸引异性。

### Game的几种方法
以下是Game的几种方法：

1. **展示面**：通过展示自己的生活方式、成就和物质财富来吸引他人。
2. **现场**：通过在现实生活中展示自信、从容和社交能力来吸引他人。
3. **自然流搭讪**：通过自信和自然的方式与陌生人交谈，建立联系。
4. **技术流搭讪**：通过预先准备的话术和技巧来与陌生人交谈。

### 展示面
**展示面**是通过展示自己的生活方式、成就和物质财富来吸引他人。这种方法的本质是通过展示包装来塑造价值感。

### 现场搭讪
**现场搭讪**是通过在现实生活中展示自信、从容和社交能力来吸引他人。这种方法的本质是通过现场的从容自在来塑造价值感。

### 自然流搭讪
**自然流搭讪**是通过自信和自然的方式与陌生人交谈，建立联系。这种方法的重点在于表达自己的感受，而不是使用预先准备的话术。

### 技术流搭讪
**技术流搭讪**是通过预先准备的话术和技巧来与陌生人交谈。这种方法的本质是植入自己的标准，并通过废物测试来筛选潜在的伴侣。

### 总结
本节课中，我们学习了什么是Game以及Game的几种方法。通过了解这些方法，我们可以更好地理解如何吸引和吸引异性。
