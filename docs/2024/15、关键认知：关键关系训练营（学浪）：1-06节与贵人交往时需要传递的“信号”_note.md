# 15、关键认知：关键关系训练营（学浪）：1-06节与贵人交往时需要传递的“信号”

大家好，欢迎来到关键关系的第六讲。这一讲我们将探讨一个大家非常关心的话题——如何与贵人交往。其实，关于贵人这个话题，在表达者课程和关键认识课程中都有所提及。

## 1. 贵人定位与资产规模

当你判断一个人是否可以作为你的关键关系社交对象时，首先需要考察的是他如何到达今天的位置，以及他为什么这么有钱、为什么有这么多资源。在思考这个问题之前，你首先要了解他的资产规模和生存方式。

**公式**：资产规模 = 之前赚了多少钱；生存方式 = 现在靠什么赚钱。

## 2. 财政与市场

资产规模和生存方式可以分为两大板块：财政和市场。

- **财政**：通过财政拨款、政府采购、国央企采购等方式获得财富。
- **市场**：凭借自身能力在市场中获得财富。

## 3. 财政圈子与市场圈子

与财政圈子的人交往，需要讲究礼数，完全按照中国传统古典要求的礼数来。因为他们追求的是安全，最渴望的是安全。

与市场圈子的人交往，更关注的是你对他本身价值的作用。他们追求的是共赢，即共同创造新的价值。

## 4. 传递信号

与财政圈子的人交往时，要传递一个信号：你是一个愿意符合既有规则去出牌、参与游戏的人，这会让他们感受到安全。

与市场圈子的人交往时，要展示自己的价值，让他们觉得你能够产生更大的价值，从而实现共赢。

## 5. 总结

本节课中，我们学习了如何与贵人交往，以及如何区分财政圈子和市场圈子。在结交贵人的过程中，要了解他们的价值所在，并传递相应的信号，以实现共赢。
