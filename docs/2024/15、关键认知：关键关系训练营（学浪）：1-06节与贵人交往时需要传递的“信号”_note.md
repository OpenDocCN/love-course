# 课程15：关键认知：关键关系训练营 - 第06节：与贵人交往时需要传递的“信号” 🎯

在本节课中，我们将要学习如何识别不同类型的“贵人”，并理解在与他们交往时需要传递的核心“信号”。掌握这些关键认知，能帮助你在构建重要人际关系时更有方向性和策略性。

## 贵人关系的两种基本类型

上一讲我们探讨了关键关系的筛选，本节中我们来看看如何对潜在的“贵人”进行分类。这是建立有效关系的第一步。

当你判断一个人是否可以作为你的关键社交对象时，首先需要考察他是如何到达今天这个位置的。你需要思考：他为什么如此富有或拥有资源？在思考这个问题前，你需要了解他的资产规模和生存方式。资产规模和生存方式指的是他过去赚了多少钱，以及他现在靠什么赚钱。这两点在具体行业中可能很复杂，难以权威地了解。你只需要做出一个宏观的基本判断：他到底是依靠财政吃饭，还是依靠市场吃饭。

**核心判断公式**：`贵人类型 = 收入来源（财政拨款 / 市场盈利）`

具体来说，你需要判断他个人银行卡里的进账，究竟是来自财政类拨款、政府采购、财政拨款给国企后的采购供货，还是他本身就是体制内工作人员，直接通过财政拨款获得工资（即公务员、事业编）。这是一类人。另一类人，则是自互联网革命以来，在如百度、阿里、腾讯这类公司担任过高管，或从这类公司出来后成功创业并获得资金的人。将这两类人作为你的“贵人”去交往，遵循的是两种完全不同的、甚至是相悖的逻辑。因此，你必须首先进行区分。

现在，请回想你之前写下的那5个名字。如果你认为其中有人可以作为你的贵人，这说明你已经预设了一种可能性：他或许会帮助你，而如果获得他的帮助，你就能获得某种利益。现在，请将他归入“财政”或“市场”这两个板块中的一个。判断标准是：他是通过直接或间接获取财政拨款来获得财富，还是能够在市场中直接通过竞争和决策来获取财富。

## 与“财政型”贵人交往的信号：安全 🛡️

首先，我们分析在政府系统或国企体系中，通过提供产品或服务来赚钱的人。这类人本质上非常信奉中国传统的古典价值观，例如“上善若水”、“厚德载物”，强调为人厚重、内敛，交往中讲究礼数。这套观念在他们心中根深蒂固，因为他们的财富、地位和资源正是通过遵循这套规则获得的。

与这种人打交道，你绝不能违背这套规则。你必须非常讲究礼数，完全按照中国传统古典礼仪的要求行事。礼数要多，且必须恰当。这与跟“市场型”人士打交道可以更直接、灵活的方式不同。与“财政型”人士交往，必须按套路出牌。按套路行事会向他们传递一个关键信号：你是一个愿意遵守既有规则、并愿意在这个规则体系内参与游戏的人。这会让他们感到**安全**。

请记住，这个圈子里的人最渴望的是**安全**，他们优先要保证的是**安全**。一旦你让他们感觉你不太稳妥、不太靠谱、嘴不严，或者无法进行长远合作，你就不可能成为他们愿意帮助的对象。他们会通过几件小事来考验你。如果你做事不稳重，关系就无法建立。

与这类人打交道，你会发现北方城市相对较多。首先，北方人在形象上可能体格更大、更敦厚，给人一种可靠的感觉。相比之下，南方做生意的人，无论是企业家还是年轻人，体型可能相对单薄，给人一种“轻飘飘”的感觉。在“财政饭”这个圈子里，给人厚重、敦实的感觉很重要。例如，在夏天穿着Polo衫、西裤和皮鞋的形象，在这个圈子里就容易被接受。如果你穿着T恤和短裤，可能就行不通。

**核心信号代码**：`传递信号 = 遵循传统礼数 + 展现稳重可靠形象`

因此，围绕“安全”这一核心，无论是中国传统文化的道德要求、社交礼仪，还是在交往过程中对细节的把握和反馈，你都必须做好。这条路并不好走。

## 与“市场型”贵人交往的信号：共赢 🤝

说完了“财政型”贵人，我们再来看看在市场中取得成果、可能成为你贵人的人。这类人相对来说更为直接。因为他们本身就是凭借自身能力取得成果的，他们更关注的是：在与你交往的过程中，你能对他的核心价值起到什么作用。如果与你合作，你是否能放大他在某方面的价值？他们追求的是**共赢**。

一定要记住，前者（财政型）追求的是**安全**，后者（市场型）追求的是**共赢**。这意味着：一个已经在市场中取得成果的人，你不可能指望他把自己做好的蛋糕分一块给你。你只有与他一起做出新的、更大的蛋糕，实现共赢。对于多出来的这部分增量蛋糕，他甚至愿意多分一些给你。这是在增量中寻找合作的可能性。

因此，如果你想成为这类人的“贵人”，或者得到他们的帮助，你需要展示你自己的价值。你需要让他觉得，你们之间有可能创造出更大的价值。你需要让他觉得可以“利用”你。记住，“利用”在这里不是坏事。一个人能被利用，说明他有价值。这个世界上大多数人可能并没有显著的利用价值。所以，如果你想与市场中取得成果的人交往，你就要让自己具备利用价值，并争取被他“利用”，从而共同做出一个共赢的新蛋糕，你从中分得一杯羹。这间接地帮助了你。

**核心信号代码**：`传递信号 = 展示独特价值 + 创造增量合作机会`

## 两种逻辑的底层区别

理解这两种逻辑的底层区别至关重要。“财政型”圈子的人更在乎安全，因为他们分给你的利益，并非完全由他们自己创造，他们不会感到“肉疼”。这些资源本质上是属于国家、属于纳税人的。因此，当他们在参与某件事时，可以在A方案和B方案之间选择。他们优先考虑的是：哪个方案更安全？哪个方案能确保我的位置不受影响，能让我在这个系统中持续获利？而“市场型”圈子的人想的则是：怎样才能把蛋糕做出新的部分？因为只有这样，我才有增量。

所以，底层逻辑非常清晰。在你想要结交贵人时，我们先不谈具体的方法（后续课程会涉及），首先要讲的一点是：你需要区分你眼中的“贵人”，他的价值是什么？他为什么能被你看作是贵人？他在哪个领域取得了成果？不论领域有多少，总体上就分为“财政”和“市场”这两个领域。更细致、更深入的话题可能不便在此展开。

## 总结与回顾

本节课中，我们一起学习了如何区分两种基本类型的“贵人”：“财政型”与“市场型”。我们明确了与“财政型”贵人交往的核心是传递**安全**信号，需遵循传统礼数，展现稳重可靠的形象。而与“市场型”贵人交往的核心是传递**共赢**信号，需展示自身独特价值，致力于创造增量合作机会。理解并应用这一基本分类，是有效构建关键贵人关系的第一步。在后续课程中，我们将深入探讨更具体的交往方法与策略。
