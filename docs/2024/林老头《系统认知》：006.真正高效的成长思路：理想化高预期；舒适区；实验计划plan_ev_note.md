# 林老头《系统认知》：006.真正高效的成长思路：理想化高预期；舒适区；实验计划plan_ev

## 概述
在本节课中，我们将探讨理想化高预期、舒适区以及实验计划plan_ev等概念，帮助大家找到真正高效的成长路径。

## 理想化高预期
**问题**：很多人在学习或实战中，会过于理想化和高预期，导致无法执行。

**核心概念**：
- 理想化：希望自己一帆风顺。
- 高预期：希望快速成功或一开始就得到高效结果。

**解决方案**：
- 不要过于理想化和高预期，而是将问题想得复杂、困难，以拓宽认知圈。
- 深度解析问题，寻找新视野。

## 舒适区
**问题**：人们往往无法执行复杂任务，因为它们超出了舒适区。

**核心概念**：
- 舒适区：在能接受的范围内做能执行的事情。

**解决方案**：
- 小步执行：将复杂任务分解为可执行的小步骤。
- 逐步扩大舒适区。

## 实验计划plan_ev
**核心概念**：
- 实验计划：在备忘录中记录接下来7天每天要做的最小可执行的事情。

**步骤**：
1. 设定7天目标。
2. 每天完成一个最小可执行的任务。

## 总结
本节课中，我们学习了理想化高预期、舒适区以及实验计划plan_ev等概念，并探讨了如何解决这些问题。通过小步执行和实验计划，我们可以逐步扩大舒适区，实现高效成长。
