## 教程内容

### 1. 散状收缩

**动作描述：**

- **第一步**：用双手中指按压阴茎海绵体两侧。
- **第二步**：手指向上移动，指尖贴合阴茎体，直至手根部。
- **第三步**：手根部向下移动，从龟头方向缓慢向下，手指贴合阴茎和龟头。

**公式描述：**

```plaintext
动作1: 双手中指按压阴茎海绵体两侧
动作2: 手指向上移动，指尖贴合阴茎体
动作3: 手根部向下移动，手指贴合阴茎和龟头
```

### 2. 阴茎根部托举

**动作描述：**

- 用右手托住阴茎根部。
- 左手轻轻缓慢地向下压阴茎，根据对方情况，尽可能引导至身体方向。
- 缓慢顺时针和逆时针打圈。

**公式描述：**

```plaintext
动作1: 右手托住阴茎根部
动作2: 左手向下压阴茎
动作3: 顺时针和逆时针打圈
```

### 3. 包皮系带按摩

**动作描述：**

- 左手握住阴茎根部。
- 右手大拇指在包皮系带人字处打圈，进行缓慢轻柔的按摩。

**公式描述：**

```plaintext
动作1: 左手握住阴茎根部
动作2: 右手大拇指在包皮系带人字处打圈
```
