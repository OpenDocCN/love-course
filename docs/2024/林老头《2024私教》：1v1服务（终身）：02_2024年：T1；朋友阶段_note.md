# 林老头《2024私教》：1v1服务（终身）：02_2024年：T1；朋友阶段

大家好，今天我们将深入探讨朋友阶段的核心内容。朋友阶段是认识新人的初期阶段，通过搭讪、社交软件、相亲等方式，获得对方联系方式或约会机会后，双方会进入这个阶段。

### 1. 注意力是关键

在朋友阶段，最重要的是注意力。新人进入交际圈，会本能地对对方产生注意力。但这个阶段几乎不可能产生吸引，至少在两次完整对话之前。

### 2. 反馈窗口类型

在朋友阶段，所有反馈窗口类型都是虚假的。这需要我们通过大量实践，感受不同类型的反馈，并思考其形成原因。

### 3. 展示价值

在朋友阶段，我们需要展示自己的价值，让对方看到我们的优点。这包括：

- **展示自我价值**：大方地介绍自己，分享生活经历，展现自己的优点。
- **提供良好体验**：热情主动，让对方在了解过程中感到舒适。
- **被动营销**：认真对待自己的形象，保持自信心态。

### 4. 筛选与淘汰

在朋友阶段，我们需要筛选出那些认可我们、愿意双向奔赴的人。这需要我们：

- **展示价值**：让对方看到我们的优点，吸引对方。
- **提供体验**：让对方在了解过程中感到舒适。
- **被动营销**：认真对待自己的形象，保持自信心态。

### 5. 转换指标

从朋友阶段进入吸引阶段的关键指标是对方对你产生兴趣，主动联系你，询问你的个人信息，甚至邀约你。

### 总结

本节课中，我们学习了朋友阶段的核心内容，包括注意力、反馈窗口类型、展示价值、筛选与淘汰等。希望大家能够将所学知识应用到实践中，提升自己的社交能力。情感学习是自我提升的过程，让我们共同努力，成为更好的自己。
