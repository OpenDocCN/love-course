# 18、坏小k女人一生必学的一堂课：视频课 09.框架实操注意点（6）

OK，补充几点框架注意事项、雷区以及如何将框架融入择偶标准。

**框架决定关系主导**

框架决定了谁主导关系发展，谁服从。例如，一女生说：“好久不见，明天有空吗，一起吃饭。”男生回答：“回头再说。”看似男生拒绝，实则背后含义：

1. 男生价值高于女生。
2. 女生需求高于男生。
3. 男生主导框架。

**弱势服从框架的迹象**

1. 在外人面前友好，在你面前不断批评、挑剔、指责。
2. 矛盾冲突时，对方从不主动道歉。
3. 表达自己时，对方忽略感受，说你无理取闹。
4. 性关系上，对方拥有主动权。
5. 关系像追逐游戏，你靠近，对方后退。
6. 快乐建立在对方回馈上，远离朋友、亲人，荒废学业、事业。

**择偶框架**

1. 防赌、欺骗、家暴。
2. 遵纪守法。
3. 不务正业。
4. 尊重家人，包括你。
5. 尊重你的身体，遵循避孕原则。

**未婚先孕的风险**

未婚先孕对女方不利：

1. 男人检验孕产能力。
2. 男人压低彩礼。
3. 男人可能抛弃你。
4. 孕期风险无保障。

**没有框架的表现**

1. 大事无主见，小事爱计较。
2. 过度干涉他人，对自己没有底线。
3. 情绪不稳定，做事不分轻重。
4. 目标不清晰，情绪无法掌控。
5. 没有边界感，分不清哪些人可信，哪些人不可信。
6. 男朋友借钱，你接受或拒绝。

**没有框架的后果**

1. 男人认为你没有要求，没有标准。
2. 男人不尊重你。
3. 你容易受到伤害。

**建立框架的重要性**

1. 让男人尊重你的恋爱规则。
2. 把自己放在优秀位置上。
3. 尊重目标、信念、内在。
4. 主动排除影响你态度的人和事。
5. 让框架融入生活，成为习惯标准。

**总结**

本节课学习了框架的重要性，以及如何建立和融入择偶标准。希望你能从中受益，拥有框架思维，避免受到伤害。
