# 16.隔壁王司机情绪自由的真相：第十课：负面情绪的处理（2）

## 概述
在本节课中，我们将学习如何处理负面情绪，特别是如何通过改变注意力焦点来改善情绪状态。

## 注意力等于事实
**公式**：注意力 = 事实

任何事情的发生都有多面性，包括好的一面和坏的一面。我们的感受和结论取决于我们关注的焦点。

## 例子：盲人摸象
不同的盲人在触摸大象的不同部分时，会得出不同的结论，这说明了每个人的观点都是独特的。

## 认知模式与注意力
我们的认知模式源自于我们的注意力。我们关注什么，就会得到什么样的结论。

## 行为疗法：打断惯性
**代码**：打断惯性 -> 从好的角度思考

1. 打断惯性：例如，当感到悲伤时，可以拍一下大腿，对自己说“太棒了”。
2. 从好的角度思考：思考事件可能带来的好处。

## 负面信息的影响
负面信息对人的吸引力比正面信息大4倍以上。

## 向内问的烂问题
当遇到负面情绪时，我们往往会问自己一些负面的问题，例如“我怎么这么没用”。

## 向内问的好问题
**代码**：太棒了 -> 这件事情的发生对我有什么好处？

通过问自己好问题，可以帮助我们把注意力转移到事件的好的一面。

## 总结
本节课中我们一起学习了如何通过改变注意力焦点来处理负面情绪。记住，注意力等于事实，我们可以选择关注事件的好的一面，从而改善情绪状态。
