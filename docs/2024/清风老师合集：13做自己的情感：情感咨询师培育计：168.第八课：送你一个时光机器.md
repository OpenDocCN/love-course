# 情感咨询师培育计划 第8课：送你一个时光机器 ⏰

在本节课中，我们将探讨一个有趣的概念——“时光机器”，并学习如何将其应用于爱情与情感关系中。我们将介绍四种实用的“时光机器”方法，帮助你从新的视角看待过去、现在和未来，从而更好地经营情感。

我们经常在科幻小说中听到“时光机器”这个词，它是一种能帮助人们回到过去或前往未来的装置。今天，我们将讨论爱情中存在的“时光机器”。

我相信你一定有过后悔的时刻，比如一句说错的话或一件没做好的事。对这些事，我们常想：如果有个时光机可以重新选择就好了。你也常说“如果世界上有后悔药就好了”。这潜意识里是希望拥有一个时光机器，给自己重来一次的机会。如果可以重来，你或许会拥有美满的爱情，更好地与伴侣相处，避免分手，并从事自己喜欢的工作。利用时光机器回到过去，把所有不圆满的事重做一遍，听起来非常美好。

然而，时光机器真的能实现吗？未来或许有可能，但目前还没有。俄罗斯理论物理学家诺维科夫在上世纪80年代提出了“诺维科夫自洽性原则”悖论。其核心观点是：即使你能回到过去，也无法改变历史进程，因为我们所处的世界已是所有改变发生后的最终结局。2010年，胡歌主演的电视剧《神话》中有一个场景：他穿越到过去，经历了传奇爱情。后来有人在知乎提问：古代的易小川和现代的易小川如何共存？为什么他既存在于过去也存在于现在？有人回答：只要穿越一次，就会永远在那个时间点循环，永远在27岁左右消失，对28岁后的世界一无所知。这形成了一种死循环。

因此，等待时光机器来解决某些问题是不现实的。例如，如果你和伴侣之间存在问题，难道要等到时光机器出现再去改变吗？即使我们可以等，爱情也不能等。

说到这里，你可能会问：清风老师，你是在忽悠我吗？说了半天没有时光机器，却告诉我爱情里有时光机器。实际上，时光机器只是我们今天讨论的开端。前面说的是我们想象中的那种时光机器。接下来，我将告诉你几种能实现时光机器效果的方法。

## 第一种时光机器：自证预言 🧠

自证预言是心理学中的一个概念。心理学家早就注意到一个现象：当我们说出一段话后，甚至会不自觉地朝那个方向靠拢。例如，有的孩子小时候说长大要当警察，后来真的成为警察。又比如，你在争吵中对伴侣说：“你现在这么不关心我，一定是有外遇了。”最后对方可能真的出轨了。

我们存在的世界并非只有冷冰冰的客观存在，还有主观存在。你如何对待世界，世界就如何对待你。在某些情况下，它甚至总能证明你说出的观点是正确的。对于爱情，你可以给自己一个爱情的期待和预言，把爱情想象成你最想要的样子，然后自我证明，朝着这个预言努力。简单来说，**如果你坚信自己未来会拥有一份美好的爱情，那么未来你真的可能会拥有**。

听上去这很像心灵鸡汤，但在心理学中，这被称为自我暗示，是自证预言的一部分。它通过不断提醒自己朝着期待的方向前进，最终可能实现目标。心理暗示具有不可抗拒和不可思议的巨大力量。心理学家普拉诺夫认为，暗示能改变人的心境、兴趣、情绪、爱好等方面，从而无意中改变某些生理功能和健康状态。

这种自我暗示本质上是在唤醒我们的潜意识。我们的内在包含意识和潜意识。有时我们能意识到自己如何清晰地塑造人生，但更大程度上，是潜意识在塑造我们的人生。心理学家荣格从精神分析角度指出：**潜意识指引着我们的人生，而我们却称之为命运**。荣格还说过：**当潜意识被呈现，命运就被改写了**。

通常情况下，意识层面的东西我们容易感受到，因此我们会说“这是我做的选择”。而潜意识层面的东西我们无法清晰感受，因此会觉得“这不是我做的选择”，是命运般的外在力量推动的结果。例如，有的女孩常说：“我就是遇不到好男人，世界上的好男人都死完了，男人都是狼心狗肺的。”你会发现她在实现这种自证预言。在她的预言中，她已将自己与坏男人绑定，即使真的遇到好男人，也容易走向另一个极端。

在心理学家武志红老师的专栏中，分享过一个关于自证预言的来访者案例。一位女士咨询他：“老师，我怎么总是遇到坏男人？”经了解，这位女士已离婚两次，原因都是丈夫家暴，甚至将她打进医院。于是她找了一位非常温和、顺从的男士（俗称“备胎”）结婚，认为他从不吵架更不会打人，很安全。但结婚不到一个月，又发生了家暴。武志红老师询问后得知，两人因小事争吵时，女士高喊：“你是不是想打我？是不是想像我前夫那样打我？”男士辩解没有，女士又吼：“男人没一个好东西！你打呀，不打就不是男人！”结果男士大脑一片空白，真的打了她。这就是可怕的自证预言实现过程，女士不断重复“男人没一个好东西”的预言。

听上去自证预言很神奇，你可能会问：清风老师，这有什么理论支撑吗？有的。1968年，美国两位博士在一所中学进行智商测试，随机抽取一些学生告诉老师他们智商很高。实验结果惊人：那些被老师认为高智商的学生成绩突飞猛进。实际上这些学生是随机抽取的。原理很简单：老师在自证预言影响下，给予这些“高智商”学生更多感情投入、学习机会和课堂反馈。这些学生因“高智商”心理不断被提醒而更加自信，在反馈中学到更多知识，成绩自然提升。

在爱情中也是如此。你首先要明白自己想要什么样的爱情，以及自己应该是什么样子才能得到这样的爱情。这样你就可以朝着期待的方向努力，让自己变成所期待爱情中应有的样子。另外，这样你身边也会逐渐出现你所期待的人。当然，如果两人已经在一起，这种自证预言需要双方共同努力，方向应是彼此都能接受的。

自证预言是我今天讲的第一个爱情时光机器，它能帮你去到未来，拥有更好的未来。最重要的是你要明白自己想要什么。就像给你一台时间机器，但你需要清楚回到过去或去到未来究竟想要什么样的未来。如果你希望得到一份平淡而恒久的爱情，可以在早上经常对伴侣强调：“亲爱的，我们的爱情一定是平静而美好的。”通过这种不断提醒，强化彼此潜意识中对爱情的期待。

## 第二种时光机器：个体差 ⏳

接下来，我讲爱情中的第二种时光机器，它很有意思。第二种时光机器叫做“个体差”。你可能会问：老师，什么是个体差？个体差指的是每个人之间的感情存在一种时间差异。也就是说，很可能别人现在的爱情就是你期待的未来爱情的样子。

例如，你身边有一对相爱8年的伴侣，你希望自己的感情也能像他们一样稳定甜蜜。但你现在刚和伴侣开始谈恋爱，这就存在一个时间差。你可以借鉴他们经营感情的经验，应用到自己的关系中，去掉不合适的部分，这就相当于完成了一次时光旅行。他们的现在就是你的未来，你可以借鉴。

同时，你还可以把自己的个体经验分享给有类似想法的人。例如，如果你有一段不太顺利的感情经验，可以告诉身边的朋友，在他们遇到困惑时提供建议。作为一名情感咨询师，很重要的一点是从经验出发，但不要只看到经验。

这种个体差在国家之间特别明显。例如，世界上计算机最发达的地方是美国硅谷，苹果公司最新产品也在此发布。中国则在努力跟上美国。两国在计算机产业上的差距就是一种时间差，把美国市场的做法拿到中国，就相当于运用了时光机器，抢先获得机会。许多成功创业公司就是这样做的，比如美国有eBay，中国就有淘宝；美国有ICQ，中国就有QQ。

在感情中也是如此。当你发现某些个体有非常好的经验值得借鉴，而他们所处的感情阶段是你感情的下一个阶段时，不妨记住这种经验，让它成为未来爱情的指导。

## 第三种时光机器：十倍速 🚀

接着，老师要跟大家说第三种爱情机器，叫做“十倍速”。什么是十倍速？这是艾伦·马斯克在商业周刊采访中提到的一种思维方式：**把一件事情做到比原来好十倍，比做到比原来好10%更容易**。为什么呢？因为要比原来好10%，通常的方法是用更多资源或努力去改善，但实现方法没有本质变化。

例如，两个人在一起维护爱情，你希望对方更在乎你一点，于是想改进10%，花很多时间精力去改善。但如果你们本身存在的问题是对方最近压力很大，你不断想改善关系反而让他压力更大，最后导致恶化。我相信这种经验每人都有，就像两人经常吵架，你希望每次比上次少吵一点，但本质上还是在吵架，改进一点还是坏的。

我们有一个成语叫“矫枉过正”，意思是通过超过原预期的方式改变遇到的问题。传统观念认为改变是一点一点进行的，这没错，任何改变都是点滴积累的。但就像增高一栋房子，在原有基础上增加不可能太高，因为地基不稳。而如果撤销原有观念重新开始，就可能达到以前达不到的高度。这并不是说吵架了就要马上重新开始，而是从观念上重新开始。既然原有观念引起争吵和伤害，就不要只想着改善它，而要想如何从根本上让观念改善十倍。

如果你一开始的目标就是改善十倍，那过去的方法肯定没用了，你必须逼自己进行根本创新，打破一些基本假设。这款时光机器告诉你要尽可能不按过去的思维模式做事，因为那样最多只能逐渐改善。如果你想获得巨大的感情和美好的恋爱，就要从根本上换方法。爱情中两人之间的牵连并不简单，要大刀阔斧地改变关系达到新高度，需要双方共同努力和互相理解。但对于一些名存实亡的爱情，如果只有你一个人在坚持努力，不如重新开始，这也是让事情变好十倍的方式。

总结这款时光机器的核心思想：**改善要从根本出发，从让事情变得比原来好十倍的角度去思考，才能创造真正的改变**。

## 第四种时光机器：未来回望法（重启法） 🔄

第四款送你的时光机器叫做“未来回望法”或“重启法”。重启这个动作我们很熟悉：电脑死机就重启，手机打不开就重启，关掉当前进程重新来过。未来回望法就是假设你站在未来回头看现在，思考未来的你会怎么做。如果未来不会这样做，那么现在就不要做。

老师举一个著名例子：1985年，英特尔公司危机四伏，其内存业务受日本厂商巨大冲击。英特尔CEO安迪·格鲁夫想退出内存行业进军微处理器。他问合伙人：“假如换个CEO来干，他们会怎么办？”合伙人回答：“当然是进军微处理器市场。”那一刻可能是商业史上最有智慧的时刻。安迪·格鲁夫说：“好，那为什么我们不这么干呢？”这个例子虽与爱情无直接联系，但它是未来回望法（重启法）的经典案例。

在爱情中，我们也应用这个角度思考。例如，我们打算做某件事时，要想这么做未来会变成什么样，未来那样是不是我们希望的。如果不是，最好不要做。很多时候，爱情中我们是不理智的，甚至有人说爱情中的人智商为零。我们有时只追求一时快感让自己后悔，却没想过最终结果。在你犹豫不决时，不妨用这个方法重启思维，问自己：**如果是未来的我站在这个时间点，会怎么做？**

## 总结 📝

本节课我们一起学习了四种应用于爱情的“时光机器”方法：

1.  **自证预言**：通过自我暗示和强化，朝着期待的愛情方向努力，塑造更好的未来。
2.  **个体差**：借鉴他人成功的感情经验，他们的现在可能就是你期待的未来，从而跨越时间差。
3.  **十倍速**：从根本出发，以改善十倍为目标进行思考和创新，而非进行微小的渐进式改变。
4.  **未来回望法（重启法）**：站在未来视角审视当前决策，如果未来不会这样做，现在就不要做。
