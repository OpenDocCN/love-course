# 21、凯男的社交情商课：情商提升3.0课程：003.情商3.0课程-转变思维【微kc58567】

## 概述

在本节课中，我们将学习如何转变思维，提升情商，从而在社交和恋爱中取得成功。

## 核心概念

* **艾宾浩斯遗忘曲线**：人会在第二天忘记第一天学习内容的80%。因此，学习时需要记笔记，反复听课和复习。
* **内核**：指一个人的内在品质和魅力，包括自信、智慧、情商等。
* **价值**：指一个人在对方眼中的价值，包括外在条件、内在品质、情感投入等。
* **七步聊天法**：一种高效的聊天技巧，通过七个步骤引导对方，提升关系。

## 转变思维

### 1. 爱自己

* **公式**：爱自己 → 别人才会爱你
* **代码**：每天花时间照顾自己，提升自己的价值。

### 2. 对自己狠一点

* **公式**：逼自己一把 → 永远也成不了
* **代码**：设定目标，努力实现，不断突破自我。

### 3. 建立原则和底线

* **公式**：有自己的原则和底线 → 不被驯化
* **代码**：明确自己的价值观，学会拒绝，不违背自己的原则。

### 4. 学会分阶段聊天

* **公式**：正确的时间 → 正确的行为
* **代码**：根据对方的状态和关系阶段，选择合适的聊天方式和内容。

### 5. 传递价值

* **公式**：激发对方投资 → 吸引对方
* **代码**：展示自己的价值，让对方感受到你的魅力。

## 七步聊天法

1. **开场白**：用有趣的方式开场，让对方快速了解你。
2. **提供价值**：用有趣的方式提供价值，吸引对方。
3. **聊自己**：聊自己的经历和感受，让对方了解你。
4. **聊对方**：聊对方感兴趣的话题，让对方感受到你的关心。
5. **升高关系**：通过暧昧调情等方式，升高关系。
6. **邀约**：邀请对方见面或约会。
7. **结束**：用幽默的方式结束聊天，留下悬念。

## 总结

本节课中，我们学习了如何转变思维，提升情商，并通过七步聊天法提升自己的社交能力。希望同学们能够将所学知识应用到实际生活中，取得成功。
