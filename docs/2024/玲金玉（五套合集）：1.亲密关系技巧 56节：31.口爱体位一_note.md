# 玲金玉（五套合集）：1.亲密关系技巧 56节：31.口爱体位一

🎼这个可爱姿势也是我们经常使用的方式，就是伴侣躺下，女生坐在伴侣打开的双腿正中央。那么，正常情况下，我们会如何进行口爱呢？我们会跪在伴侣的腿中间。

🎼然后以这种方式进行口爱。在整个过程中，可以让伴侣的腿微微弯曲。此时，我们可以双手抱住伴侣的大腿进行口爱。🎼

**口爱要点**：
- 当伴侣双腿打开时，支撑力在手掌，手掌作为支撑点，塌腰使关节微微向下进行口爱。
- 如果伴侣的腿膝盖微微弯曲，支撑点在手肘，双手辅助抱住伴侣大腿进行动作。
- 如可能，用手掌托起伴侣臀部，使阴茎向上，方便舔舐。

**错误示范与正确示范**：
- 错误示范：来回上下，没有感觉。
- 正确示范：金老师演示，口爱时带着身体，与伴侣眼神交流，有身体接触和灵动。

**三角模型功法、心法、技法融合**：
- 通过身体展现，让对方看到你的状态，达到上头的效果。
