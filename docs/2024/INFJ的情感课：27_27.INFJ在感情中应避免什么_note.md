## 概述

在本节课中，我们将探讨INFJ型人格在感情中应避免的行为和心态，以帮助他们在感情中更好地经营关系。

## INFJ的软肋

INFJ型人格有几处软肋，这些软肋可能导致他们在感情中出现问题。

### 1. 劣势功能SE

INFJ的劣势功能SE可能导致他们在情绪激动时做出破坏性行为，如扔东西、锤墙等。**公式**：SE（劣势功能）→ 破坏性行为。

### 2. 劣势功能TI

INFJ的劣势功能TI可能导致他们陷入NITI循环，过度分析感情问题，产生自卑情绪。**公式**：NI（优势功能）→ 猜疑 → TI（劣势功能）→ 过度分析 → 自卑。

### 3. 劣势功能FE

INFJ的劣势功能FE可能导致他们在沮丧时表现出作的行为，如说反话、故意不真诚等。**公式**：FE（劣势功能）→ 作 → 不真诚。

### 4. 体重管理

INFJ型人格容易发胖，这可能导致他们产生外貌焦虑，进而影响感情。**公式**：体重增加 → 外貌焦虑 → 颓丧气场。

## 避免的行为和心态

### 1. 克制破坏性行为

在情绪激动时，INFJ应学会克制，避免做出破坏性行为，以免给对方造成心理创伤。

### 2. 避免NITI循环

INFJ应避免过度分析感情问题，陷入NITI循环，产生自卑情绪。学会调整心态，保持自信。

### 3. 避免作的行为

INFJ应避免在沮丧时表现出作的行为，如说反话、故意不真诚等。学会直接表达自己的感受，避免误解。

### 4. 保持体重

INFJ应注重体重管理，避免因外貌焦虑而产生颓丧气场，影响感情。
