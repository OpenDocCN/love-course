# 加一情感《宅男宝典：01、情圣方法：22.长期关系_note

在本节课中，我们将要学习长期关系的核心概念与维护方法。长期关系是情感交往的重要阶段，涉及从热恋到平稳的完整过程，以及关系中需要扮演的不同角色。掌握这些知识有助于建立稳固、健康的亲密关系。

## 长期关系的定义与重要性

长期关系就是我们的交往阶段。

不管我们多么自立或者多么花心，总有一天我们要有自己的家庭，这就是我们每个人最终要去做的。

如何处理好长期关系，家庭关系是特别重要的。

## 长期关系的三个阶段

长期关系要经过三个阶段：热恋期、磨合期、平稳期。

以下是每个阶段的详细描述：

*   **热恋期**：在这个阶段，会觉得对方特别好，把心目中最美好的恋人形象都投射到他的身上，觉得他什么都是好的，完美的。
*   **磨合期**：随着互相了解的深入，我们会慢慢的发现对方身上的缺点，此前美好的感觉幻灭了，变得难以容忍对方，从而产生冲突。发生冲突的方式主要有三种：**吵架**、**冷战** 或者是 **被动攻击**。会觉得对方的缺点越来越多，但是在这个同时，自己也开始反省自身的问题。在和对方激烈的争吵之后，我们以往所坚持的会动摇，会静下心来去想，是不是我真的做错了。
*   **平稳期**：后来我们会放下自己的立场，尝试着去接受，重新找到对方的优点，然后就进入了平稳期。

上一节我们介绍了长期关系的三个阶段，本节中我们来看看在长期关系中需要扮演哪些关键角色。

## 长期关系中的三个核心角色

长期关系中，我们要扮演三个角色。

以下是这三个角色的具体说明：

*   **父亲的角色**：高大、威猛、厉害、严厉，会管着他。这个角色是关系的底线，男生给女生提供稳定舒适的环境，帮她抵挡生活中的波折。核心是“我会保护你，但同时也会管着你”，不让她做不能做的事情。
*   **哥哥的角色**：会照顾他，呵护他，关心他。这个角色是暖男，你会记得他的生日，记得他喜欢吃什么，害怕什么。
*   **孩子的角色**：撒娇，调皮，惹他生气。这个角色是关系的润滑剂。这个时候你是在被照顾，他在为你付出。

理解了三个角色的定义后，接下来我们深入探讨每个角色在实践中的具体表现和重要性。

### 父亲的角色：底线与责任

父亲的角色是底线。

举个例子，两个人相处，男生在经济和社会地位上比女生低的时候，其实是一种父亲角色的缺失。所以这也是为什么大部分人找对象会选择比自己小、能力比自己稍微弱一点的女生，这样的关系更稳固。这也是好多女生喜欢有点大男子主义的原因。

我们要做那个她是崇拜的、可以依靠的男人，我们要是两性关系的主导者。大的事情必须要坚持自己的原则，不能突破自己的底线。小的事情不去计较，你要怎么样就怎么样。

以下是关于底线原则的具体例子：

*   **原则性大事（必须坚持）**：例如和别的男生暧昧、跟别人去酒吧、对你的家人不尊重。这些都是你的底线。
*   **非原则小事（不必计较）**：一般小的事情不要跟女生计较。鸡毛蒜皮的事跟女生计较，显得没有男人的样子，气量小。

### 哥哥的角色：关怀与付出

第二点，长期相处之后，你要学会对女生付出。这个时候的付出，女生会记在心里并会感动。

往往我们在追求阶段是很上心，一旦追到手之后就忽视了。其实女生很容易被满足：她的生日、你们的纪念日、她喜欢什么东西，你要知道。这些节日你要送上她喜欢的东西，有时候不需要很贵，但是有心她就会满足。

例如，有一次我女朋友过生日，生日是周三，定的是周五大家一起去庆祝。因为平时大家可能没时间。周三的那天晚上11点多，我带了一个小蛋糕敲她门，她跟几个闺蜜正在玩桌游。当她看到我拿着蛋糕进来的时候，差点就哭了。

> 我没有说，但是我记得。

### 孩子的角色：甜蜜与润滑剂

第三点，孩子的角色也很重要。当然像我们这样白羊座的，这点都不需要去学习，本身就有这一面。一些比较成熟的人，往往没有这一面。

比方说，我会经常跟女朋友打闹，走在街上我会直接把她抱起来，会和她玩幼稚的小游戏，会和她打打闹闹。这个是两性关系的润滑剂。我们所有好玩的、有趣的瞬间，其实都是小孩子这个角色创造的。

以下是可用于实践的一个小游戏示例：

**游戏名称**：木头人（改良版）
**代码/流程描述**：
```
1. 对女生说：“我跟你玩一个游戏好不好？很好玩的。”
2. 她问：“什么游戏？”
3. 你回答：“这个游戏叫木头人。规则是：我说123，然后我们谁都不许动，谁要先动就输了。”
4. 然后喊：“123，开始！”
5. 过了一会儿，你说：“我输了。”（因为两个人都没动）
6. 她会好奇：“你为什么输了呢？”
7. 你回答：“因为我心动了。”
```

## 课程总结

本节课中，我们一起学习了长期关系的核心框架。

总结一下这节课的重点：

*   **第一**：长期关系会经历热恋期、磨合期、平稳期三个阶段。
*   **第二**：在长期关系中，我们要扮演三个角色：爸爸、哥哥、小孩。
*   **第三**：爸爸这个角色代表责任与底线。
*   **第四**：哥哥这个角色代表关怀与付出。
*   **第五**：小孩子的角色代表甜蜜与趣味。

掌握这三个阶段的动态变化，并灵活运用好三个角色，是维护一段健康、长久亲密关系的关键。
