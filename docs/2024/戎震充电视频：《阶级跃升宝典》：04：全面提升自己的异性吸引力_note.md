# 戎震充电视频：《阶级跃升宝典》：04：全面提升自己的异性吸引力_note

在本节课中，我们将要学习如何通过提升个人魅力，特别是异性吸引力，来加速实现阶级跃迁。我们将从统计学和现实角度分析其重要性，并提供具体可行的提升方向。

这是《阶级跃升宝典》的第四期，也是最后一期。我以四期为一个界限，专门讲解一个主题，这样内容会比较集中。像我之前讲解PUA时讲了很多期，因为PUA是一个非常严格且大家感兴趣的话题。

我在短视频拍摄技巧方面只讲了一期，后边我会补上那三期。关于穷孩子发家致富，我讲了四期。在讲到阶级跃升最后一期时，我要讲解一个从统计学上看的概念。

一个人如果想完成在大城市的奋斗，想完成阶级跃升，最快的方式其实不光是从学识与财富上着手。更多时候，这与人际关系有很大关系。但如何从根本上提升人际关系呢？就是你要在非常少的时间内，学会如何提升你的个人魅力。

这种魅力主要是外表容貌上的提升。对于男生，你要考虑到你的发型发量、皮肤的整洁程度、衣服的搭配。这些东西你不要看我现在好像不太懂，我以前其实也非常重视这些东西。

女孩子更是这样，你要参考“平民公主”这类人的几条路线。她们的路线一般来讲也是一种优胜者的选择。所以真正失败的那些人你是看不见的。所谓的优胜者淘汰，你要看到他们从十四五岁开始就辍学了解社会，然后“下海”。

但究其根本原因，是他们找到了社会中有利的人帮助了他们一把。不管他们用什么条件达到了让别人帮助的条件，但更多来讲，是他们结识了社会中有权利、财富、声望的人之后，才开始发家的。

所以，我们的本质是要提高自己的异性吸引力。这个异性吸引力很重要。不管你的性取向是同性还是异性，这种吸引力你一定要有。因为异性吸引力与同性吸引力不一样。

同性吸引力在吸引同伴时，可以让你达到个人的幸福。但异性吸引力是一种普遍对社会的“杀伤力”。很多人你看他身边一直没有男朋友或女朋友，他可能是专心搞事业的人。

如果你在美貌上对自己进行的投资是适宜的，你需要经常调整自己的状态。因为每一种发型、每一种发色、每一种饰品的使用、每一种外表的衣服，都需要考量。比如有人经常说，你的内容很好，但你应该穿上cosplay的衣服；你弹钢琴弹得很好，也应该穿上各种各样的衣服来展现自己。

因为大众对人的接受程度，其美感是一致的。常说的“网红脸”，为什么网红脸要整成那样？这是一个统计学原因。别看大家都说网红脸，但大家都喜欢网红脸。很多东西，比如你说“我再也不找肌肉男了”，但六块腹肌就是比一块腹肌要有魅力。

这个东西就是看你能掌握什么，但要有个度，不要为它花过多的金钱或时间。然而，它是你的一个辅助。你要学王自如那类手段，你的地位不就上升得快吗？

很多时候你会发现，最高位置上的那些人，不管是公司的领导，还是你要巴结的贵人对象，其实本身也都是饮食男女。他们也不可能完全避免。一旦你这个人的异性魅力比较大，你平时接触的人的数量和对别人的初始好感的认定度就会比较高。

说白了，就是会有很多人主动愿意给你花钱，会有更多人主动愿意帮你，而你积累财富的速度也会非常快。你们看我经常连线的那些主播们，他们精心打扮自己，精心完成自己的人设，从声音、外形到人性的一些配置，都是经过精心设计的。

所以他们展现出来这些东西后，大家就愿意给他们花钱。大家愿意给他们花钱，他们积极积累这些财富后，就可以把这些财富用到很方便的地方。所以，如果你从个人角度想完成阶级跃升，又不想谄媚任何人，你也应该采取这样的态度。

比如你做一个网红，无论是男网红还是女网红，你积累财富的速度就比上班要快。积累财富速度一旦快了，其实大家是这样：你去大厂，积累财富靠工资；你去做生意，积累财富靠利润；你去做网红，积累财富靠名声。

但无论如何，最终决定谁更厉害时，是看你们积累了多少财富，以及你们如何用财富去增长。所以到后期，就是一个资产购入与资产组合的过程。

但第一桶金你怎样赚到，是靠高学历、靠“不要脸”、还是靠所谓的卖唱或卖弄色相挣到，这些在你成功以后都不重要。有人就算挖出你这段历史，大家也不会觉得有什么。

所以在网络时代，无论是通过网络营销接触更多客户，还是通过网络表演积累名声，都是一种先进的方式，是前一个时代无法想象的。大家想，如果在20年或30年前我做这么一套节目，你们首先就看不到我的人，大部分是听到我的声音。

可能看不到人，听到声音会更好，但它缺乏直观的表现。你看我“删巴掌”那个名场面可能你就看不到。像我录这400多期节目，可能就得去电台，比如每周六录一次或每周录两次。

但它也有固定时段，观众数量就不会有现在这么多。很多时候内容可能台里不喜欢，我也放不出来。私人办电台就更扯淡了，私人就算办了电台也不知道怎么推广。

很多时候由于平台和网络技术的发展，你现在用手机或摄像机就可以拍很多东西。所以大家一定要利用现在的先进技术手段。你总不能不P图吧？你总不能不美颜吧？

其实很多人真人是没有办法看的，但经过化妆、P图、整容都可以变得很好看。所以在现在这个时代，其实是“量产美女”最容易的一个时代。你要看别人是怎么做的。

为什么别人文化水平看起来很低，却能赚到很多钱？因为这个时代给了很多表演者一个台阶。更多时候，我们要提升的就是自己的美貌、异性吸引力、说话的程度，甚至你的声音、相貌、看人的态度。

为什么很多时候我们说那些表演班里，有些人不是来学表演的？他们是通过学这些东西来进行人格的重新塑造。所以我说的“二十五型人格”为什么要跟大家说？二十五型人格其实不分男女。

只不过有些人格在男性中更常见，比如“海盗船长”；有些在女性中更常见，比如“平民公主”。但平民公主路线，难道男生就不走吗？不是很多人都在走吗？比如王自如。

很多时候，这些路线在你看来是什么样的选择？我说了，二十五型人格是社会化的职业选择，不是先天带有的。我是这种个性，我进行这种选择，那么就是对的。

所以我们说，阶级跃升最后一个宝典，就是一个穷人家的孩子，在没有权利、财富、声望的时候，能靠的是什么？能靠的就是给你一个工具，能够快速积累这些东西。

当然，你的容貌越好、体能越好、身高越挺拔，优势越大。我们有增高鞋垫，容貌好有整容、化妆等各种手段，你要去学习。因为你要知道，你的目的无论穷富，最终都是攒第一桶金。

有人第一桶金靠继承，有人靠自己的才能，有人靠当“网络乞丐”得来。无论怎样，殊途同归。到最后就是一个硬资本的较量。大家也知道，在资本市场的时代，这些东西很残酷，就是一个“用进废退”、以大欺小、以强欺弱的时代。

所以，你要适应这个时代，不能让时代适应你。你不能总等到下一个时代，或回归上一个时代，那是不可能的，也不可能有时光机出现。你就活这么一生，很快的。

如果你天天跟自己较劲、跟时代较劲，十年二十年的青春一下就过去了。青春过去后也容易追悔莫及。你就像我现在如果回到19岁重新活一遍，我觉得我的路线也不会比现在快多少，一定也是从头到尾重新犯一遍错误。

但即使有了教训和记忆，也一样会走这些路，犯这些错误，积累也都得从一点点开始。所以从今天开始，如果你想财富自由，你最好就是要强健自己的体魄，在外貌容貌上要花心思。

我见过很多学习很好的人，在异性吸引力这方面吃了很大的亏，被异性骗婚、欺骗感情，或者在这上面吃了大亏，比如“给小姐30万”这种事，都是我身边人干的。

所以，你在这方面如果没有过早的涉猎、没有过早的体验、没有过早地建立防御，将来就会在这方面吃亏。因此，你要适当看一些时尚的东西，看别人怎么打扮、大家都怎么说话。

你不要沉迷其中，但如果你连三里屯、夜店都没去过，那这辈子是一个重大的人生缺失。一定要去一次，起码一年去一次，知道大家玩什么，知道其他的年轻人、男男女女在干什么，知道富家子弟在干什么。

知道这些以后，选择一条最适合你的路。这个社会斗争相当残酷，不会给你多少机会让你去想、让你去选择。所以你要明白这一点，就加强自己这方面的能力，起码要在这方面不能落败于任何人。

本节课中，我们一起学习了异性吸引力在阶级跃升中的关键作用。核心在于，在资本时代，个人魅力是快速积累初始资源（第一桶金）和拓展人脉的有效工具。提升外貌、声音、谈吐等，本质上是进行一种社会化的职业选择与人格塑造。适应时代，主动学习并利用现有技术（如网络、美颜）包装自己，同时保持适度原则，是普通人实现快速跃迁的重要策略。
