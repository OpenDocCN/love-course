# 野哥《聊天特训营》课程01：课程前言与学习方法 🎯

在本节课中，我们将介绍《聊天特训营》的整体框架、核心学习方法，并为你提供必要的学习工具和心态准备。课程旨在系统性地解决与异性聊天中的常见难题。

欢迎来到《聊天特训营》，我是主讲导师加诺，也是心动恋爱的创始人。课程开始前，我将简要自我介绍，并说明本次课程的学习方法与流程。

在我从业过程中，我接受过上万名学员的内部及外部咨询。我收到最多的问题是关于聊天：女生为什么不回复信息、某句话应该如何回复、女生某句话的含义是什么。

这些问题表明，聊天技巧已成为许多男性在与异性相处过程中不可忽视的核心问题。本课程将聊天问题拆解为六节小课，并结合社群学习与案例讲解构成课程主体。

我们的学习社群配有导师及我本人进行答疑。你关于课程学习的疑问，以及在生活中与异性相处时产生的实际问题，我们都会逐一解答。课程结束后，我们也会在群内不定期分享干货与案例。

提到案例，本次课程讲解的案例，是从私教课程的三百多套聊天记录中，精选出的三套最具教学意义的范例。同时，我们也提供了对应的话术库。

接下来，我们说明学习方法。收听本课程的兄弟，可能对情感行业已有一定认知，也可能是零基础的新手。新手无需担心，但我更希望大家能以**空杯心态**进行学习。

这就好比在学校学习一门科目。如果A老师教你一部分，B老师又教你另一部分，你最终听到的是来自不同导师的混杂内容，学习效果必然不尽如人意。事实上，很多老师的方法可能存在冲突，各有其特点。

因此，希望大家能以**空杯心态**来聆听本课程并加以运用。同时，当你遇到任何问题时，请及时向我反馈，以便及时解决。

本次课程前言到此结束。欢迎各位兄弟加入本次特训营。请准备好你的笔记本，放松心情。
