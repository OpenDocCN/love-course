# 情绪价值课-【阿娇】：第10节 七招拿捏男人的情绪

大家好，欢迎来到娇姐课堂。😊 今天我们来学习如何用七招拿捏男人的情绪。

### 拿捏男人情绪的关键

**公式**： 情绪掌控 = 情绪感知 + 情绪引导

1. **正向积极的情绪**： 让男人与你在一起时感到开心，对未来充满期待，容易爱上和依赖你。
2. **消极负面的情绪**： 与你在一起时感到不开心，分开后感到痛苦，需要独自面对不好感受时回来找你，这是负面强化。

### 如何快速拿捏他的情绪

1. **专属称呼**： 为他取一个专属的昵称，让他感到特别。
2. **区别对待**： 在一群人中特别关注他，让他感到存在感。
3. **制造幻想**： 通过语言引导他幻想美好的画面，激发他的期待。
4. **说出他的感受**： 理解并说出他的感受，让他感到被理解。
5. **多夸奖他，满足他**： 发现他的独特之处，真诚地夸奖他。
6. **保持平稳的心态**： 少想多做，用实际行动试探他的情绪点。
7. **剥夺特权**： 收回曾经给他的特权，让他感到获得或失去。
