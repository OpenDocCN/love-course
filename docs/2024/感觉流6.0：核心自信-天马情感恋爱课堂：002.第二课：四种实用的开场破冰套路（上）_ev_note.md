# 感觉流6.0：核心自信-天马情感恋爱课堂：002.第二课：四种实用的开场破冰套路（上）

聊天的作用主要是通过言语来增加或减少你的吸引力。开场白至关重要，一个好的开场白可以让你在女生心中留下良好的印象，成功几率高达80%。

## 一、引发好奇式的开场白

这种开场白可以引发女生的好奇心，打破尴尬的氛围。以下是一些例子：

- “我好像在哪里见过你？”
- “你看起来很眼熟。”
- “你好像认识我？”
- “你好像知道什么我不知道的事情。”

## 二、幽默型开场白

幽默型开场白可以快速升温氛围，但需要注意风险。以下是一些例子：

- “你好，25号技师为你服务，京东，请签收一下你的快递。”
- “听说你单身，所以月老派我来讲。”
- “你的陪聊工具已上线。”

## 三、真诚型开场白

真诚型开场白可以拉近彼此的距离，以下是一些例子：

- “你好，很高兴认识你。”
- “很高兴见到你，你看起来很亲切。”

## 四、直接型开场白

直接型开场白可以快速进入话题，以下是一些例子：

- “嗨，你今天过得怎么样？”
- “你好，能和你聊聊天吗？”

## 总结

本节课介绍了四种实用的开场破冰套路，包括引发好奇式、幽默型、真诚型和直接型。通过这些开场白，你可以更好地与女生交流，增加彼此的吸引力。记住，开场白只是第一步，后续的聊天技巧同样重要。
