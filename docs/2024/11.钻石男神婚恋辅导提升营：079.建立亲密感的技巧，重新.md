# 11.钻石男神婚恋辅导提升营：079.建立亲密感的技巧，重新

在本节课中，我们将学习如何在感情挽回过程中，通过搭建互动链接来创造挽回时机，并掌握重新建立联系与亲密感的核心技巧。我们将避免使用低效的“断联”方法，转而采用更主动、更有效的策略。

## 🔗 避免“断联”，主动搭建互动链接

上一节我们介绍了挽回的基本心态，本节中我们来看看如何开始实际行动。“断联”意味着切断联系并被动等待时机，这是挽回过程中常见的错误策略。

**断联 = 切断联系 + 等待时机**

这种方法本质上属于下策。使用者往往不清楚何时、何种情况下应该恢复联系，也缺乏一套有效的识别系统来把握挽回时机。因此，断联很难提高成功率，其作用仅限于避免犯错、不降低挽回几率。

以成功挽回为目的，首先应该主动搭建互动的链接，让双方恢复联系。接着，找到一个合适的切入口来创造挽回时机，这才是挽回的上上之策。

网络上许多挽回技术帖内容空泛，多为心灵鸡汤，缺乏实质性指导。正确的挽回思路需要通过深入评估，分析当前局面，找到核心问题，从而制定出有针对性的、可操作的挽回计划，一步步导向成功。

## 🤝 如何重新建立联系

许多朋友掌握了挽回理论，却不知如何与昔日的恋人重新建立联系，也不懂如何进一步发展亲密感，导致挽回之路停滞不前。

在挽回道路上，实践至关重要。在自我改变之后，最关键的一步是让对方看到优秀的你，并重新爱上你。

你可能会想：“当初是他主动离开，我也答应了不再联系。现在我主动联系，会不会显得自己很掉价？”答案是不会。重新建立联系可以从朋友的身份开始，再逐步建立亲密感，这是挽回爱情的精髓所在。

以下是重新建立联系的具体方法：

*   **创造偶遇**：可以装作不经意的样子，在外界制造偶遇。外在表现要像遇到老朋友一样自然，邀请对方一起喝咖啡、坐下来聊聊天、叙叙旧。
*   **克服心理障碍**：你可能会紧张，并担心自己的邀请会困扰对方。想法过多，永远将自己搁置在行动之外，你将无法挽回爱情。
*   **利用通信媒介**：如果觉得当面联系尴尬，可以通过短信等通信方式建立新联系。例如，以老朋友的身份发信息问候对方近况，并简单提及自己的近况。可以在无意中流露出对旧情已看淡的态度，这有助于打消对方的抵触情绪，建立全新的联系感。

## 💞 如何建立亲密感与信任感

以朋友身份重新建立联系后，若想挽回对方，还需要重建亲密感和信任感，让对方再次爱上你。

建立亲密感的过程中，**观察力**最为重要。你需要了解，对方对你的态度会从抵触逐渐发展为平常心态，最后再到亲密。拥有足够的洞察力，才能准确分析出对方当前的心理状态。

为了快速过渡到亲密状态，可以继续利用通信媒介，调动前任对你的关心。例如，发一条这样的信息：“我有点感冒，但不知道应该吃什么药。”此时，如果对方已不再抵触你，他/她会迅速代入朋友身份，甚至无意中找回恋爱的感觉，对你表示关心。这也是挽回爱情的一个小技巧。

除了观察力，**询问技巧**同样需要掌握。例如，当对方说“以我们的身份，这样联系不是很好吧”时，你会怎么想、怎么做？你可能会退缩。但为什么不尝试询问一下“为什么呢？”你的退缩会让你永远不懂对方的真实想法。而用一种良好的方式主动询问，可能会收获这样的回应：“最近我总觉得现在的关系不适合这样的联系方式，不如我们更进一步吧。”由此可见，掌握询问技巧在挽回道路上相当重要。

**核心步骤总结：重新建立联系 -> 逐步建立亲密感 -> 找回爱情**

## 🔍 感情危机时的正确应对思路

当感情出现危机、恋人提出分手时，许多人选择通过死缠烂打、不断强调自己有多爱对方来拯救爱情。这种方式并非良策，甚至完全无法拯救爱情。

想要拯救爱情，找回属于自己的温暖，应当先从自身思考。在感情中，没有人是永远正确的。许多问题都需要从自身考虑，才能找到分手的根源。

当对方提出分手时，其实在他/她心中已经开始否定你的某些方面。因此，你的解释和承诺很难被听进去。信任减少了，对方自然不会听信你的话语。所以，不要解释，也不要承诺。为了拯救爱情，请先从自身考虑。

以下是需要自我反思的方向：

*   思考你们的爱情到底出了哪些问题。
*   平时是否索求过多？
*   在相处中，你是否经常想：“我要什么样的爱情？但对方有些冷淡。”
*   对方是否总是拒绝你的某些愿望？
*   你是否希望他/她更重视你？

让感情出现裂痕的主要原因，往往正是你心中的这些想法。即使你不明说，它们也会在潜意识中暴露出来。你可能没有察觉，但敏感的对方总能感觉到。正是这些原因导致你们的心灵距离越来越远。

如果想要拯救爱情，那就努力尝试改变。改变的目的不是完全变成对方想要的那种人，而是成为一个能让感情更稳固、让对方感受到爱情甜蜜的人。

## 💬 从对方角度寻找解决方法，并进行有效沟通

从对方的角度，我们同样可以找到拯救爱情的方法。你需要明白：即使是提出分手的一方，同样也很痛苦。提出分手的人不仅有罪恶感，心中也会有挣扎，以说服自己接受分手这个事实。毕竟，彼此都曾真心爱过。

你可能会想：“他/她既然有这种心思，为什么还要分开？”这又回到了前面的原因——主要还需从自身找问题。

当然，除了反思自身，若想缓解爱情危机、更好地拯救爱情，不妨找一个合适的时间，与对方进行一次耐心的交流。

爱情没有绝对的对错，但相爱的人都不完美。两个人走到一起本身就不容易，想让爱情变得更完美，需要的不是相互索取，而是充分的交流和体谅。

因此，在一个比较轻松的氛围里，两个人心平气和地谈谈有关爱情的问题吧。一起想想，你们的爱情到底哪里出了问题，考虑一下是否还有弥补的方式。

相爱的人走到一起，由于各方面原因，必然会有摩擦。只有经历坎坷，才能让爱人更加亲密。所以，在爱情中遇到挫折时，不要总想着分手，而是多思考一下如何去拯救爱情。
