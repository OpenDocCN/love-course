# 林老头《系统认知》：003.T1阶段：补充_ev

## 概述

在本节课中，我们将深入探讨T1阶段的落地细节和思路，帮助大家更好地理解和应用。

## 注意力是核心

首先，我们要明确注意力是核心。只有当对方把注意力放在你身上，才能产生良好的效果。因此，吸引的前提是注意力。

### 第一印象的重要性

一开始，你的照片、外形、朋友圈等都会给对方留下第一印象。如果第一印象不好，后续的机会就会大大减少。因此，我们要做好第一印象。

### 标签与价值

当你开始与女生聊天时，最重要的是标签。对方会给你贴上什么样的标签？标签可以分为三类：钱、帅、爱。

### 体现价值

在前期抓住对方注意力后，我们要给自己贴上标签，展示自己的价值。以下是如何体现钱、帅、爱的价值：

- **钱**：展示你的工作、生活、上进心等，不必过分夸大。
- **帅**：学会拍照，展示你的身材、衣品、氛围等。
- **爱**：展示你的三观、性格、情绪稳定等。

### 信息归类

聊天信息可以分为三类：钱、帅、爱。根据对方的聊天内容和朋友圈，判断对方更关注哪一类。

### 打击需求点

根据对方的信息归类，展示自己对应的优点。

### 提升价值

即使你的钱、帅、爱价值不高，也要学会展示。这不仅能让你在社交中获益，还能让你在工作中、生活中获得认可。

## 通过T1阶段

当对方对你有主动提问、开启话题或来找你时，说明你的T1阶段已经有效。

## 总结

本节课补充了T1阶段的落地细节和思路，希望大家能够掌握并应用到实际中。
