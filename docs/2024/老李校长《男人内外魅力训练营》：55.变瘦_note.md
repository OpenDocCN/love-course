# 老李校长《男人内外魅力训练营》第55课：变瘦 🏃‍♂️

在本节课中，我们将要学习如何通过“变瘦”来有效提升外在性吸引力。这是外在魅力提升系列的第一部分，核心在于理解并掌握科学减肥的原理与方法。

情感的道路并非总是鲜花与美酒，情感的世界充满挑战。唯有历经磨砺之人，方能成为英雄。现在，让我们一同开启提升外在魅力的旅程。

从这一课开始，我们讲解外在性吸引力的提升。我将从变瘦、头发、眼睛、皮肤、医美、服装、饰品、鞋子以及拍照这几个维度，为大家全面解析如何提升自己的外在性吸引力。外在性吸引力是男性魅力的重要组成部分。

那么，为什么要减肥变瘦呢？我告诉大家，核心是为了塑造轮廓。健康的人如果肥胖，是没有清晰轮廓的。而轮廓，是所有女人对外形进行一票否决的关键。如果你是一个油腻的胖子，拥有肚腩，女人会感到相当反感。同理，男人对女人也是如此。

只有足够瘦，身体线条才会清晰，穿衣服才好看。有人问是否需要练就一身肌肉，这取决于个人的时间和兴趣爱好。女人通常并不特别喜欢肌肉特别粗壮、像坦克一样的体型。并且，培养大量肌肉需要投入巨大的时间和精力，大家需要计算好投入产出比。当然，如果你是爱好者则另当别论。我这里想说的是，如果你想改变，首先必须做到足够瘦。

很多兄弟在20多岁时都很瘦，进入社会参加工作后，因为应酬、喝酒、压力大，或者有些兄弟交了女朋友，关系稳定后，都会慢慢变胖。

我曾经也是一个200斤的胖子，尝试过很多种减肥方式，也都瘦下来过。我第一次变瘦是靠拔火罐，从180斤减到147斤。但我当时并不明白其中的原理，只知道要去美容院交费，然后他们给我拔火罐，还在我腰部缠上类似保鲜膜的东西。我以为是这些起到了作用，其实并不是，大家听我往后讲原理。后来因为没控制住饮食又反弹了。

那么第二次变瘦是去健身房，一周7天去6次，还请了私教。请私教真的很贵，每小时三四百元，特别昂贵。而且中国的健身房营销套路很深。我通过健身，从180斤又减到了150斤。后来呢，同样因为没控制饮食又反弹了。

而我第三次从180斤变到150斤，只用了一个月时间。前两次我都用了半年时间从180斤降到150斤，并且中间花了很多钱。但是这次，我看了一本书之后，只用了一个月就做到了，而且没怎么花钱。这本书叫《肥胖代码》，作者是加拿大一位糖尿病医生。这本书把肥胖的原理讲得非常清楚。他给出的建议其实也非常简单，就是**断食**或**轻断食**。

当我采用他的方法时，我采取了比较激进的策略，即完全断食24小时、48小时、36小时，最长达72小时，我都试过，效果非常明显。我严格按书上的方式方法喝水，补充复合维生素片，然后每天散步一两个小时，以及适度补充电解质。补充电解质就是喝点葡萄糖水或者带咸味的汤。但是在这几年实践后，我已经优化了我的节食方法。我会通过一天喝三次牛奶、吃两包坚果来实现轻断食，效果一样好，一个月能减重20斤。

下面给大家讲讲肥胖的原理。减肥的原理在于：当我们血液中的血糖升高时，身体会分泌出胰岛素，把多余的血糖降下来。而多余的血糖会通过与胰岛素发生化学反应，生成脂肪，然后储存在我们的身体里。

所以，关键是我们不能吃**快速提升血糖的食物或饮料**。这类食物主要是**碳水类**和**饮料类**，凡是带甜味的、米面做的东西，我一般都不吃，因为它们升血糖太快。但是吃脂肪类食物，比如坚果，或者吃些蛋白质类食物，倒不容易直接导致发胖，可以适当食用。

大家记住，一定要避免吃太多的主食、甜的饮料，甚至酒也得少喝。大家可以把脂肪理解为**存款**，把血糖理解为**现金**。血糖是我们即时供能时需要使用的。但是当你血液中的血糖不够、身体又需要能量时，身体就开始分解脂肪。

所以，断食的原理就是**逼着身体不要花现金（血糖），转而花存款（脂肪）**。如果你采取轻断食方式，就是等着逼你的身体去消耗你的存款，也就是脂肪，因为你手头的现金（血糖）不够了。

所以，当你掌握这个原理之后，减肥会变得特别轻松、特别快。我经常在我的朋友圈为粉丝们现场表演减肥。大家想看我瘦多少斤，我一般15到20天就能减掉20斤，非常轻松。而且我对于偶尔的反弹和变胖，也没什么压力。我认识的男演员、男模特，他们都足够瘦。

只有足够瘦，拍照才上镜、才好看。脸才会显小。只有足够瘦，你的脸部才有清晰的线条。大家记好，**变帅的第一步是变瘦**。
