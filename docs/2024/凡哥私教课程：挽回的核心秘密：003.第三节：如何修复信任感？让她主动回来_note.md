# 凡哥私教课程：挽回的核心秘密 003.第三节：如何修复信任感？让她主动回来

🎼大家好，本节课我们将探讨在挽回关系中，如何利用好你的剩余信用货币。这里提到的信用货币概念，可以理解为你的信任值。例如，你最初拥有100点的信任值。

由于某些原因，你的信任值降至30点。🎼当你的信任值只剩30点时，你想要购买一个行为，比如邀请对方看电影，对方可能会拒绝。

这意味着你需要用这30点信任值去赢得更大的信任行为，让对方重新信任你。

🎼以下是一个形象的比喻，帮助你理解。如果处理不当，你的信用货币会越用越少。如果处理得当，你的信用货币会越来越多，使你们的关系良性循环。

🎼假设我面前有一盆火，这是我作为挽回者面临的问题。就像我的女朋友或妻子要跟我分手，我现在要解决这个问题，火势越烧越旺。

要熄灭这盆火，我需要用到我的信用货币，比如一块塑料布。想象一下，如果我把塑料布完全覆盖在火上，火会因为缺氧而熄灭。

🎼然而，很多时候，你用塑料布去灭火时，火反而把塑料布烧着了，火势变得更旺。等你好不容易把塑料布上的火扑灭后，再去灭火时，却发现只剩下了一小块塑料布。

🎼那么，你怎么才能把火扑灭呢？答案是，处理得当的话，你可以用这块塑料布扑灭火势，而且火扑灭后，塑料布的厚度和数量都会增加，这意味着你的信用货币在增加。

🎼那么，如何解决这个问题呢？在讲解方法之前，请大家思考一下：为什么你们以往的解决思路都是错误的？

🎼以往的解决思路是这样的：火势起来了，我需要消除这个火。那么，如何消除呢？用信用货币去覆盖它，消灭它，消除它。但你想，你已经犯了错误，已经发生了既定事实，有些原因导致这个事实发生。

你怎么可能消除它呢？你需要不断解释，但解释往往无效。那么，如何才能有效解决呢？这里有一个方法推荐给大家：假装什么都没有发生过。

🎼举个例子，一对恋人分手了，男生想要去找对方。情感导师告诉他，你可以去找对方，但你准备做什么呢？他说我要找他，找个理由说上次有个东西我要还给他。

然后，他说，我好不容易在一起，我要跟他解释，对不对？我要跟他去表达一下我的态度，最好能跟他一起吃顿饭。

🎼作为情感导师，我们会告诉他，不要这么做。怎么做呢？其实可以给他打个电话，调整一下语气，就像没有发生过一样。

因为人在发生一些事情后，语气会产生很大变化，会颤抖，会诚惶诚恐。所以，闭上眼睛回想你们没有分手之前的状态，再给他打电话。

🎼打个比方，给小美打电话，你说那个小美，那个东西我拿来了。放在哪儿？我现在拿过去。然后放下，转身就走，或者转身走之前，回头面带微笑说一句，哎呀，我先走了。

🎼这样，对方会有一种感觉，就像真的没有发生过一样。随着时间的推移，生活将慢慢进行，你们开始变得理性，从受伤的情绪中走出来。

🎼男生也不再那么恐惧、焦虑，然后两个人可以心平气和地坐下来，共同探讨之前的事情，这样关系就得到了良性循环。

🎼所以，我们要假装什么都没有发生过。那么，如何做到呢？

🎼首先，闭上眼睛，想象一下你们曾经最完美的关系，进入那个想象中。然后调整自己的情绪和呼吸，给对方打电话。这一点也适用于交朋友。

🎼比如，我想约一个朋友出去吃饭，他可能会拒绝。我该如何征求他的同意呢？其实，当你这么想的时候，就已经错了。你应该这么想：他是我未来一定能够成为好朋友、好兄弟的人。

🎼我提前透支一点信任，给他打电话，说老王，一起吃饭，有空吗？这样，对方就会有一种亲切熟悉的感觉，就像老朋友一样。

🎼所以，假装什么都没有发生过，这样做的目的是让生活正常化。生活本来就是这样，突然间出现了问题，我们先让生活正常化，然后找时机沟通解决问题。

🎼那么，什么是时机呢？不是你认为的时机，而是对方想沟通、想跟你交流的时候，时机才真正出现。
