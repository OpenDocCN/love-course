# 富山山女性成长课程：22_高情商接话拿下他3

## 概述
在本节课中，我们将学习如何通过高情商的接话技巧来提升人际交往能力。

## 第一部分：态度与关键词
**态度的重要性**
- **公式**：态度 = 80% 的沟通效果
- **代码**：保持积极、开放的态度

**抓取关键词**
- **公式**：关键词 = 对话中的核心信息
- **代码**：倾听并提炼对方话语中的关键词

## 第二部分：引导与深挖
**引导对方**
- **公式**：引导 = 提出开放式问题
- **代码**：例如，“你对哪一张照片最满意？”

**深挖话题**
- **代码**：通过追问细节，引导对方深入分享

## 第三部分：理解弦外之音
**理解深层含义**
- **公式**：弦外之音 = 话语背后的真实意图
- **代码**：例如，对方迟到后说“妆化精致”，实际上是在暗示迟到。

## 第四部分：引导说真话
**接受真话**
- **公式**：接受真话 = 建立信任
- **代码**：例如，对方提到过去的不当行为时，表示理解并询问原因。

## 第五部分：面对质疑
**真诚回应**
- **公式**：真诚回应 = 保护自己，展示真实自我
- **代码**：例如，“我认为你应该多考虑一下，再做评判。”

## 总结
本节课中，我们一起学习了如何通过高情商的接话技巧来提升人际交往能力。通过掌握这些技巧，我们可以更好地与他人沟通，建立良好的人际关系。
