# 《如何成为一个自信的男人》：如何成为一个自信的男人4.0.1：25_24-新手小白不会聊天怎么办？

大家好，我是乃一。今天我们开始学习聊天技巧，直接进入主题。首先，我们来探讨新手小白常见的问题，其中最常见的是：

**1. 聊天时感觉没有话题怎么办？**

很多新手会收集各种话题，如兴趣爱好、童年经历等，但这些话题过于私密，不适合初次聊天。关键在于，聊天不是聊话题，而是聊情绪。

**公式**：聊天 = 情绪交流

**2. 聊天技巧**

*   **技巧一：秸秆与抛梗**
    *   秸秆：抓住对方信息，放大话题。
    *   抛梗：在回复中留下话题空间，让对方接话。
*   **技巧二：聊天方向**
    *   聊自己：感受、状态、经历、喜好。
    *   聊别人：状态、感受、经历、喜好。
    *   聊第三方：共同兴趣的事件、地点、人物。

**3. 为什么男人不喜欢聊天？**

*   男人以结果为导向，活在当下，对当下事物无感。
*   女人活在当下，以过程和感受为导向，话题丰富。

**4. 男人不聊天的后果**

*   精神出轨的前兆。
*   撩妹无招。
*   职场软实力不足。

**总结**：

本节课我们学习了聊天技巧，包括秸秆、抛梗、聊天方向等。同时，我们探讨了男人不聊天的原因和后果。希望这些内容能帮助你提升聊天能力，成为一个自信的男人。
