# 梵公子 情绪价值核心课【向上社交破圈升思维】：5.向上破圈社交法则

姐妹们好，欢迎来到第五节课，主题是向上社交的原则。本节课的重点是强调向上社交的目的性，以及如何筛选和接触高价值的男性。

## 1. 目的性社交

**核心概念**：目的性社交意味着只与目标类型和高价值的男性接触，减少无意义的社交时间。

- **公式**：目的性社交 = 目标类型男性 + 高价值男性 - 无意义社交

**注意事项**：

- **不参加无意义的酒局、饭局**。
- **判断邀约活动是否对自己有益**。
- **快速判断男性是否符合自己的标准**。

## 2. 避免单一选择

**核心概念**：不要将所有希望寄托在一个人身上，要给自己留有备选。

- **公式**：备选方案 = 多个男性 - 单一选择

**案例分析**：

- **案例一**：一位职场精英因未提前布局，难以找到合适的结婚对象。
- **案例二**：一位女性因将所有精力投入初恋，最终选择不合适的结婚对象。

## 3. 以自我为中心

**核心概念**：在社交关系中，以自己为中心，不要过度依赖男性。

- **公式**：自我中心 = 男性 - 过度依赖

**案例分析**：

- **案例**：一位女性因将所有心思放在男性身上，导致生活不幸福。

## 总结

本节课中，我们学习了向上社交的原则，包括目的性社交、避免单一选择和以自我为中心。希望姐妹们能够将这些原则应用到实际生活中，找到适合自己的社交方式。
