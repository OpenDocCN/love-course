# 钻石男神婚恋辅导提升营：081.6决心分手的前任怎样追回来

大家好，我是陈楠老师。面对决心分手的前任，如何挽回呢？挽回并非求复合，即使道歉再多，态度再卑微，到了真正分手的地步，都是徒劳的。

**以下是挽回前任的步骤：**

1. **分析分手原因**：了解分手的原因，对症下药。
    - **原因一**：你的爱不是他想要的，可能是他感受到的是约束和禁锢。
    - **原因二**：情感需求得不到满足，亲密关系相处的本质是价值交换。
    - **原因三**：长期失望的积攒结果，意味着他经历了长期失望的积攒后，痛定思痛做出的决定。

2. **尊重他的意愿**：任何真性分手都不是情绪使然，曾经的甜言蜜语和承诺保证是无法挽回他的。

3. **断联与反思**：根据矛盾严重程度确定断联时间，通常以5到25天不等，最好不超过一个月。

4. **表达反思**：冷静下来，反思自己的问题，道歉，并展示自己的改变和成长。

5. **以朋友的身份继续交往**：尊重他的边界，主动把关系保持在他感觉舒适的区域。

6. **理性层面**：展示自己的价值，让他相信你是他的最优选。

**总结**：挽回前任需要耐心和智慧，通过分析原因、尊重意愿、断联与反思、表达反思、以朋友的身份继续交往以及理性层面展示自己的价值，最终实现挽回。

本节课中我们一起学习了如何挽回决心分手的前任。希望对大家有所帮助。
