## 腮红实操

上一节我们介绍了膏状腮红的使用，本节中我们来看看粉状腮红如何操作。

粉状腮红上色度通常较好，为避免颜色过重，蘸取后需先在纸巾上匀掉多余粉末。

以下是粉状腮红的上妆步骤：
1.  使用腮红刷或大号眼影刷蘸取腮红。
2.  在纸巾上轻轻打圈，匀至刷毛上几乎看不到颜色。
3.  将腮红打在眼睛后三分之一处下方，以此为核心位置。
4.  可向上或向下轻微扩散，打造自然氛围感。

腮红位置并非绝对固定，核心在于起始点。不同打法效果各异：**圆形**腮红显可爱，**横向氛围感**腮红增加气色，**斜向**腮红则能修饰脸型。

若颜色过重，可用浅色腮红或散粉在边缘晕染，使其过渡自然。口红也可用作腮红，用手指蘸取后轻拍上脸即可。

## 遮瑕实操

腮红为面部增添色彩后，接下来我们需要处理面部瑕疵。遮瑕的重点在于精准遮盖与自然融合。

遮黑眼圈时，需调和出比自身肤色略深或相近的颜色。因为黑眼圈颜色深于皮肤，使用浅色遮瑕反而会凸显。

以下是遮瑕调色与上妆方法：
1.  **调色**：将深、浅两色遮瑕以1:1比例在手背调和。
2.  **试色**：将调和后的颜色在脸颊边缘轻划，检查是否与肤色融合。
3.  **调整**：若颜色过深，加浅色调和；过浅，则加深色。
4.  **上妆**：用小刷子将遮瑕精准点在黑眼圈最深部位，无需覆盖全部眼周。
5.  **晕染**：用笔杆或干净手指，将遮瑕边缘轻轻拍打开，与肤色自然衔接。
6.  **定妆**：遮瑕后需立即用散粉轻压定妆，防止移位。

法令纹的遮盖原理不同，关键在于**提亮阴影**，而非填充沟壑本身。观察法令纹，其显现是因为上方产生了阴影。

以下是遮盖法令纹的步骤：
1.  选择比肤色亮的遮瑕色。
2.  正对镜子，找到法令纹上方的阴影区域。
3.  用小刷子将遮瑕点在阴影处。
4.  用手指将遮瑕轻轻拍开，与周围皮肤融合。
5.  对于较深的法令纹，目标应是“减淡”而非“完全消失”，需有合理预期。

对于小面积斑点，可直接用美妆蛋蘸取遮瑕膏，以“按压”方式遮盖。日常妆容无需追求完美无瑕，保留少许自然瑕疵反而更显真实。

## 提亮实操

遮瑕完成后，我们通过提亮来让面部更立体饱满。提亮产品可选择哑光高光或浅色遮瑕。

以下是面部各部位的提亮方法：
1.  **面部凹陷**：如面颊凹陷处，涂抹少量提亮产品并晕开边缘，视觉上会有“蓬起来”的效果。
2.  **眼下三角区**：少量提亮可提升面部整体状态。
3.  **鼻子**：针对中庭较长的情况，可将山根提亮点**下移**，鼻头提亮点**上移**，以视觉上缩短中庭。鼻头可横向点涂提亮。
4.  **眉骨与嘴角**：眉骨转折处用刷子精细提亮；嘴角沿唇形微微提亮，有上扬效果。

所有这些提亮点都集中在面部**内轮廓**（以眉峰、鼻翼、嘴角为界的内部区域），目的是将视觉焦点集中于此，从而显脸小。

## 修容实操

提亮塑造了立体感，接下来我们通过修容来优化面部轮廓。修容色可选择眼影盘中的深肤色。

修容的核心在于**位置精准**，而非面积大或颜色深。

以下是修容步骤：
1.  **侧脸修容**：正对镜子，找到脸颊最宽处。将修容产品涂抹于此，再用手或工具向四周晕染开，消除明显边界。
2.  **下颌线修容**：同样找到下颌最宽点，按压上色后晕染自然。
3.  **凹陷处提亮**：若面部线条不够流畅（如面颊凹陷），可在凹陷处轻扫少许高光，使轮廓更流畅。
4.  **鼻侧影**：用修容刷或手指，从眉头下方顺着鼻梁弧度自然向下轻扫，少量多次。

## 口红实操

妆容的最后一步是口红，它能瞬间提升气色与整体妆感。颜色选择可遵从个人喜好。

以下是口红涂抹与调整技巧：
1.  **由内向外涂**：先涂抹嘴唇内侧，然后上下唇轻轻抿开，让颜色自然扩散。
2.  **模糊唇线**：用手指或棉棒将口红边缘轻轻晕染开，打造柔和效果。
3.  **调整不对称**：若嘴唇不对称，可在涂完口红后，用遮瑕膏修饰多出来的一侧唇线，再轻轻晕开遮瑕边缘。
