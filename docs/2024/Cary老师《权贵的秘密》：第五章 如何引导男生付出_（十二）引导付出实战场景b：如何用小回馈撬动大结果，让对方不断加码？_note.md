# Cary老师《权贵的秘密》：第五章 如何引导男生付出_（十二）引导付出实战场景b：如何用小回馈撬动大结果，让对方不断加码？

🎼本节重点：预期回报

**预期回报**是吸引男生付出的关键。追求过程中，男生思考的不仅仅是追求本身，而是与伴侣共同生活的可能性。

**公式**：预期回报 = 付出意愿 × 付出可能性

**案例分析**：健身房教练推销健身课，女生用“上次睡了教练”的玩笑话吸引教练免费服务。

**核心概念**：

1. **价值撬动**：通过语言和行动，让对方为未来价值投入更多。
2. **选择请求**：向有钱人、有资源的人、有人脉的人请求。
3. **及时回馈**：对男生的付出给予及时、良性的回馈。
4. **节奏感**：从大到小引导男生付出，实际行为从小到大。

**实操步骤**：

1. **选择请求**：向对方请求最不稀缺的东西，如金钱、资源、人脉。
2. **及时回馈**：
   - **夸奖认可**：用言语表达认可和赞赏。
   - **关心付出**：在对方帮助时给予回报。
   - **高质量互动**：根据双方意愿进行。
3. **节奏感**：
   - **从大到小**：在日常生活中表达对大额物品的喜爱。
   - **从小到大**：实际行为从小额付出开始。

**总结**：

本节课中，我们学习了如何通过预期回报、选择请求、及时回馈和节奏感来引导男生付出。通过这些方法，我们可以让对方为我们的未来价值投入更多，从而实现我们的目标。
