# 社交情商训练营：第11课第5节、不断加分的聊天方式 💬

在本节课中，我们将要学习如何在聊天中持续创造正面印象，从而有效提升吸引力。我们将探讨四个核心加分点，并介绍实现这些加分点的六个关键技术。

上一节我们讨论了聊天中可能导致减分的行为，本节中我们来看看如何通过主动引导和提供价值来为你的聊天表现加分。

在聊天中能够让你加分的有四个核心点。

以下是四个关键的加分点：

1.  **主动引导话题走向**：你需要能够主动地带领对话的走向，而不是被动回应。
2.  **主动引导关系升级**：你需要能够主动地推动双方关系向更亲密的方向发展。
3.  **提供养育价值或繁衍价值**：你能够向对方展示你具备提供资源（养育价值）或优秀基因（繁衍价值）的潜力。
4.  **提供正面情绪价值**：你能够通过聊天让对方感受到愉悦、开心、被理解等正面情绪。

只要你能够做到这四个点，就没有你撩不到的妹子。聊天并没有想象中的那么困难，它总共就分两步。

整个聊天过程可以简化为两个核心步骤：**吸引女生**和**升级关系**。你只需要把这两步做好，那么女生基本上都能够轻松地被你搞定。

要做到这个程度，你需要掌握六个聊天技术。我们要先学会这六个聊天技术，然后再学会如何在聊天中灵活运用它们。

以下是六个需要掌握的聊天技术：

1.  **话题开启与延伸技术**
2.  **情绪共鸣与理解技术**
3.  **价值展示与植入技术**
4.  **关系试探与推进技术**
5.  **幽默感与趣味性营造技术**
6.  **框架建立与维护技术**

当你熟练掌握并能在对话中组合运用这六项技术时，你就能够完全学会聊天了。

本节课中我们一起学习了聊天中持续加分的四个核心要点：主动引导话题与关系、提供养育/繁衍价值以及情绪价值。同时，我们明确了成功聊天的两步走战略——吸引与升级关系，并指出了实现这一目标需要掌握的六项关键技术。掌握这些原则和技术，将使你的聊天能力得到质的提升。
