# 瑞恩年会员PRO 瑞恩高级会员：课程88(她关闭朋友圈-解题思路)

## 概述
在本节课中，我们将分析一个案例，探讨当对方关闭朋友圈后，如何处理这种情境并继续与对方互动。

## 案例分析
**女生关闭朋友圈的背景**：
- 女生刚添加时朋友圈完全开放。
- 9月27号左右，女生关闭朋友圈，设置为三天可见。

**应对策略**：
1. **表达感受**：及时表达对对方关闭朋友圈的感受，例如：“你感觉不安全吗？”
2. **活在当下**：与对方互动时，专注于当前时刻，不要纠结于过去或未来。
3. **处理问题**：面对问题时要积极处理，不要忽视或逃避。

## 互动技巧
1. **开启话题**：例如：“这天好像还是挺热的。”
2. **回应对方**：例如：“比前几天好一点。”
3. **保持沟通**：在对方关闭朋友圈后，仍要积极与对方沟通，表达关心。

## 总结
本节课中，我们学习了如何应对对方关闭朋友圈的情境。通过表达感受、活在当下和处理问题，我们可以继续与对方保持良好的互动关系。
