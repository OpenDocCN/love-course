# 课程05：成真女生的秘密：月经期间用品指南 🩸

在本节课中，我们将学习女性在月经期间需要使用哪些不同类型的卫生用品。我们将逐一介绍每种用品的基本概念、用途和特点，帮助你建立清晰的认识。

## 卫生巾与护垫

上一节我们介绍了课程主题，本节中我们来看看最常用的基础用品。卫生巾和护垫是附着在内裤上、用于吸收经血的外部用品。

以下是它们的核心特点：
*   **卫生巾**：主要于经血量较多的日子使用，具有不同的长度、厚度和吸收量等级。
*   **卫生护垫**：通常较薄，适用于经期前后流量较少时，或作为卫生棉条的额外保护。

## 内置式用品：卫生棉条与月经杯

了解了外部用品后，我们进一步探讨内置式用品。卫生棉条和月经杯是置于阴道内部以收集经血的用品。

以下是它们的核心概念：
*   **卫生棉条**：一种圆柱形吸收体，通过**置入阴道**来吸收经血，需要定期更换。
*   **月经杯**：一种由医用硅胶等材料制成的小杯，置于阴道内**收集而非吸收**经血，可清洗并重复使用，单次使用时长可达12小时。

需要说明的是，卫生棉条和月经杯在亚洲地区女性中使用相对较少，在欧美地区使用更为普遍。

## 总结

本节课中我们一起学习了女性经期的主要卫生用品。我们介绍了外用的**卫生巾与护垫**，以及内置的**卫生棉条与月经杯**。每种产品都有其适用场景和特点，你可以根据自己的需求和舒适度进行选择和尝试。
