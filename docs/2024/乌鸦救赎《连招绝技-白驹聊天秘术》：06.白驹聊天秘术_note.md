# 乌鸦救赎《连招绝技-白驹聊天秘术》：06.白驹聊天秘术

## 概述
在本节课中，我们将学习如何运用白驹聊天秘术，通过有效的沟通技巧来建立信任和筛选合适的伴侣。

## 核心概念
- **白驹聊天秘术**：一种通过聊天技巧来建立信任和筛选伴侣的方法。

### 1. 废物测试
- **定义**：女生通过开玩笑或看似无理取闹的方式来测试男生是否认真。
- **应对方法**：保持轻松态度，配合女生的玩笑，避免较真。

### 2. 踢皮球
- **定义**：在对话中故意回避问题，让对方自己回答。
- **应用场景**：当女生提出一些敏感或难以回答的问题时。

### 3. 哲学输出
- **定义**：通过展示自己的智慧和内在魅力来吸引对方。
- **应用场景**：在对话中适当引用哲学观点或人生感悟。

### 4. 变主话术
- **定义**：通过改变对话框架，让女生主动追求男生。
- **应用场景**：在建立信任和筛选后，可以尝试使用变主话术。

### 5. 前妻密码
- **定义**：一本关于长期关系培养和调整框架的书籍。
- **应用场景**：在确定关系后，可以参考书中的内容来培养和调整关系。

## 注意事项
1. **筛选和信任**：这两个环节是零难度的，只要不犯低级错误即可。
2. **邀约或放弃**：在完成筛选和信任后，可以尝试邀约女生或放弃。
3. **变主话术**：在建立信任后，可以尝试使用变主话术。
4. **前妻密码**：在确定关系后，可以参考前妻密码来培养和调整关系。
