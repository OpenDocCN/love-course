# 林老头《私人传承》：005.高阶聊天技巧

大家好，今天我们来探讨两个对聊天和约会最有帮助的点。这两个点可以帮助你模仿女人喜欢的渣男或优秀男生的感觉。

**核心概念**：

* **筛选**：延长筛选时间，让对方感觉你有选择，从而提升你的价值。
* **复格**：给予对方一个身份或人设，让对方为了维持这个身份而努力。

## 一、筛选

**以下是筛选的步骤**：

1. **延长筛选时间**：不要一上来就表现出强烈的兴趣，让对方感觉你有选择。
2. **慢热**：表现出慢热的态度，让对方觉得你不容易接近。
3. **有的选**：让对方感觉你有选择，从而提升你的价值。

**公式**：

\[ 筛选 = 延长筛选时间 + 慢热 + 有的选 \]

## 二、复格

**以下是复格的步骤**：

1. **给予对方一个身份或人设**：例如，成熟稳重、高冷、幽默等。
2. **让对方为了维持这个身份而努力**：例如，通过行为或言语来证明自己符合这个身份。

**公式**：

\[ 复格 = 给予身份 + 让对方努力维持 \]

## 三、个性和接纳

**以下是建立深度吸引的步骤**：

1. **展示个性**：大大方方地展示自己的性格和特点。
2. **接纳对方**：接受对方的个性，不要试图改变对方。

**公式**：

\[ 深度吸引 = 个性 + 接纳 \]

## 四、总结

本节课中，我们学习了如何通过筛选、复格、个性和接纳来建立深度吸引，从而提升自己的价值。希望大家能够将这些技巧应用到实际生活中，提升自己的魅力。

**记住，感觉是关键，而感觉来自于人设。通过建立优秀的人设，你可以吸引到更多的女生，并建立更深入的关系**。
