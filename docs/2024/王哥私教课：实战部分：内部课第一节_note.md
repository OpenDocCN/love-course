# 王哥私教课：实战部分：内部课第一节 🎉

## 概述

在本节课中，我们将学习如何通过收集资源、展示面打造、聊天技巧、邀约和确定关系来提升把妹能力。

## 收集资源

**核心概念**： 收集资源是建立吸引的第一步，可以通过以下方式：

- 陌陌、探探、积木、微博、抖音、快手等社交软件
- 搭讪
- 利用软件的匹配功能

**步骤**：

1. **展示面打造**： 确保你的照片和生活状态吸引人。
2. **P图技巧**： 如果自己照片不够理想，可以使用P图工具或使用明星照片。
3. **互动**： 在社交软件上主动打招呼，与对方互动。

## 展示面打造

**核心概念**： 展示面包括精神层次和物质层次，以下为具体内容：

**精神层次**：

- QQ签名、微信签名
- 说话方式和表达的含义

**物质层次**：

- 生活方式
- 物质条件

**形象改造**：

- 发型：选择合适的发型，保持整洁。
- 脸部护理：使用洗面奶、面膜等。
- 穿着：选择合适的衣服、鞋子、裤子。
- 香水：选择适合自己的香水。
- 增高鞋垫：个子矮的兄弟可以使用增高鞋垫。
- 包包：购买高仿奢侈品包包。

## 拍摄

**核心概念**： 拍摄高逼格照片，提升展示面。

**步骤**：

1. **选择拍摄地点**： 咖啡厅、五星级酒店、豪车、健身房、高尔夫、骑马、射箭、冲浪、海钓、旅游、工作场所等。
2. **拍摄技巧**： 使用手机或微单拍摄，注意光线和构图。
3. **剪辑软件**： 使用剪映、美图秀秀、VSCO等软件进行剪辑和P图。
4. **发布**： 一周发布2到3条或1条朋友圈。

## 聊天

**核心概念**： 通过聊天建立联系感，提升吸引力。

**步骤**：

1. **开场**： 根据认识前发生的人和事进行开场，展现幽默感。
2. **信息交换**： 进行简单的信息交换，如备注名字。
3. **互动层次**： 从互动层次一级到三级，逐步深入话题。
4. **深入聊天**： 在夜深人静时进行深入聊天，分享生活、情感等。
5. **聊骚**： 根据对方意愿进行聊骚。
6. **邀约**： 进行模糊邀约，试探对方意愿。

## 邀约

**核心概念**： 通过邀约将关系推进到线下。

**步骤**：

1. **模糊邀约**： 先试探对方意愿。
2. **电话敲定**： 约定具体时间和地点。
3. **见面**： 进行见面，进行生物课。

## 确定关系

**核心概念**： 通过以上步骤，与对方建立稳定的关系。

**总结**

本节课介绍了如何通过收集资源、展示面打造、聊天技巧、邀约和确定关系来提升把妹能力。希望同学们能够认真学习，并在实战中不断总结经验。
