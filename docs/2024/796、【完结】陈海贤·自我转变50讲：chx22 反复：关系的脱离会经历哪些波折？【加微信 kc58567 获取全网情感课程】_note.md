# 796、【完结】陈海贤·自我转变50讲：chx22 反复：关系的脱离会经历哪些波折？

欢迎来到自我转变五十讲。我是陈海贤。上节课我们探讨了关系的脱离。关系的脱离本质上是由于现有关系不再适合自我成长，因此个体需要离开以发展自我。

然而，当关系脱离发生时，新的自我并未立即形成。我们可能会留恋过去的保护、关系中的爱恨情仇，甚至试图回到过去，这就是关系脱离中的反复。

**为什么还想回去呢？** 这是因为关系的脱离不仅仅是离开那么简单，人永远需要归属感。离开了重要的人，但他们的影响仍在。我们还会用他们的眼光看待自己，有时会想念这种归属感。

当在关系中受伤时，我们可能会误以为他们仍然是那个能治愈我们的人。因此，关系的脱离常常不是一蹴而就的，而是一步三回头。

### 以下是关系脱离中可能经历的波折：

1. **迷茫和恐慌**：在脱离关系的初期，我们可能会感到迷茫、恐慌和留恋。
2. **自我怀疑**：这种迷茫和留恋可能会转变为自我怀疑，让我们怀疑自己的选择。
3. **寻找自我**：在经历了一段时间的挣扎后，我们会鼓起勇气寻找自我。
4. **重新找回自我**：通过失去原有的保护和温情，我们重新找回自我。

### 案例分析

以下是一个关于职场PUA的案例：

**案例背景**：一位女士的前老板对她进行权利和控制，使她感到痛苦和纠结。

**案例过程**：

1. 女士在老板的赏识下，逐渐将自我概念与老板的认可绑定。
2. 老板开始对她进行权利和控制，使她感到痛苦和纠结。
3. 女士犹豫是否辞职，最终在忍无可忍的情况下递交了辞职信。
4. 老板对她的辞职反应激烈，让她感到得意。
5. 女士离开后，发现自己陷入迷茫和自我怀疑，甚至想要重新回到老板身边。
6. 在与老板的谈话中，她意识到自己失去了原有的故事，并决定重新找回自我。

### 总结

关系的脱离是一个反复的过程，常常是一步三回头。在这个过程中，我们可能会经历迷茫、恐慌、自我怀疑等情绪。但只要我们勇敢地面对，最终会重新找回自我。
