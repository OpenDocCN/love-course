# 课程39：夜店入门指南 🍸

在本节课中，我们将学习初次前往夜店前需要了解的核心知识，包括夜店类型选择、预定规则、酒水消费以及组局注意事项，帮助你建立基础认知，避免常见误区。

## 夜店类型与选择 🏙️

夜店一般分为清吧和嗨吧。清吧更多算是一个约会场。嗨吧就是我们所说的蹦迪场所，也是本节课要讲的主要内容。酒吧和夜店其实是同一个意思，只是统称和叫法习惯不同。

如果你对夜店不熟悉，建议选择品牌连锁店或当地比较火爆的酒吧。例如在西安比较火的就是 **Play House** 和 **迈阿密**。

选择大品牌夜店有两个主要原因：
1.  你可以感受他们的品牌夜店文化。
2.  防止酒水消费被坑。因为很多小酒吧人气不高，不够热闹，酒水质量一般且价格昂贵。

## 预定规则与流程 📅

选好夜店后，你需要了解夜店的定台规则。首先，你需要找一个酒吧营销预定位置。有朋友的话可以请朋友推荐酒吧营销的微信给你。如果没有朋友推荐，你可以在美团或大众点评上找到这家酒吧的电话，打电话过去就能加到酒吧营销的微信。

加到微信后，酒吧营销会问你的预算和需求，你可以直接告诉他。建议尽量向酒吧营销要一张酒吧座位的分布图。这样你可以直观地看到夜店内卡座和布局的分布，以及不同位置的编号和不同区域的最低消费。这能让你心里有数，知道自己会坐在场子的哪个位置。

夜店在节假日、周末或有嘉宾活动时，不同卡座的最低消费会比平时高。在平时周日到周四的时段，有些夜店的便宜卡座可能没有最低消费。在周日到周四的时段，你还可以将上次没喝完的酒取出来喝，无需额外消费，除非你买的洋酒没喝完需要再买饮料勾兑。

没喝完的酒可以找服务员存起来。存酒和取酒的规则，服务员会告诉你，不同酒吧营销也可以。一般来说，存酒期限大约是一个月。如果连续存满超过一个月不取出来，之前存的酒就会作废。当然，根据地方不同，规则和存酒时间也会有所差异。

## 位置、时间与消费 💰

在夜店里，卡座的位置越靠近中心，最低消费就越高。因为你的位置越靠近中心，在夜场的氛围体验感就越好，相对来说也越有牌面，更能体现你的高价值。

你需要在晚上9点半之前来到酒吧落座。酒吧营销会要求你在9点半之前必须入座，如果你不来，这个卡座可能会给别人。酒吧不可能让位置空着浪费，他们不会做赔本的买卖。此外，很多夜店在9点半之前入座，会提供几个不同价位的早场套餐，这比正常点酒水或套餐要便宜划算一些。

## 酒水选择与“牌面” 🥃

接下来我们讲讲点酒和“牌面”。到了夜店后，应该如何点酒呢？对于新手或经济不充裕的兄弟，有早场酒水套餐的话是最好的选择，经济实惠。如果没有早场套餐，且经济比较充裕、卡座也比较好，可以根据夜店的酒水单正常点。

最好优先选择洋酒套餐，套餐通常比单点划算很多，而且洋酒勾兑饮料也更耐喝。当然，不同地方、不同酒吧的酒水价格会有一定差异。一般去夜店，可以喝洋酒、香槟、红酒或啤酒。但不建议喝啤酒，因为会让女生感觉很low、很土。如果你是为了喝啤酒，可以去夜宵摊或清吧，那里物美价廉。在蹦迪的地方喝啤酒会显得格格不入，很没有牌面。除非在已经有洋酒的情况下，少加一点啤酒进去是可以的。

一般情况下，**威士忌**和**野格**这两款洋酒比较流行，性价比高，喝起来容易上头但价格不贵。当然，土豪随意。综合以上所说，大家根据自己的预算合理选择套餐即可。套餐里通常还会送一些果盘，摆在整个桌上会比较好看。

在夜店里，“好看”有时意味着脸面。现在夜店里，越是靠近中心的卡座，有些人喜欢弄个“黑桃A”——一款比较贵的香槟。这是现在很流行的撑“牌面”的东西，好看又能展现高价值。一瓶黑桃A放在桌上，代表这个卡座的玩家很有实力、很有面儿。现在全国夜店里，土豪或常去玩的玩家都流行弄个黑桃A摆着看但不喝，因为一瓶黑桃A价格大概在1万元左右，不同地方价格可能有差异。

所以你看到的“牌面”不一定是真的。当客户在酒吧购买了这种比较贵的酒时，酒吧往往会搞一个仪式：很多人背着亮闪闪的酒和牌子在酒吧里转一圈，把酒送到你的卡座，举牌子的人围着你的卡座衬托气氛，然后把酒放到桌上。这种形式向周边和整个酒吧传递一种信息：这个卡座的大哥买好酒了，很有牌面，很能展现实力和高价值。很多妹子也很吃这一套，因为虚荣心。

为什么说“牌面”不一定是真的？因为很多黑桃A只是拿来装逼、拿来看，不是拿来喝的。而且有些夜店的黑桃A也可以租。如果你跟酒吧营销关系比较好，也想要仪式感和牌面，和他说一下，有的权利大点的酒吧营销也会想办法弄个黑桃A过来在你桌上摆一段时间撑撑面。反正只是放在桌上打牌面，不是用来喝的。不过有些兄弟可别酒喝多了，一时冲动打开喝了或不小心摔了，毕竟1万元左右一瓶。当然，土豪随意。

也有些真土豪会摆好几瓶黑桃A或“神龙套”，这些一般就是真的土豪了。往往这些土豪的卡座都会有很多非常漂亮的妹子，有的是女玩家，有的是散票，有的是专门钓土豪的等等。所以还是新手的兄弟们，对于这些女生就先不要考虑了。

## 组局类型与社交价值 👥

最后我们讲讲关于酒吧的常见组局类型。首先，最常见的“罗汉局”，就是全部都是男生，没有女生。大家要知道，罗汉局的社交价值非常低。在夜店，最有价值的资源一定是美女。如果全部都是男生，连一个女生都叫不来，几个男的喝酒为什么不去夜宵摊或别的地方喝呢？还更经济实惠。

很多罗汉局都是一人抱着手机，无所事事，既没玩开心也花了很多钱，气氛非常沉闷无聊。如果卡座上全是男的，一个女生都没有，哪怕有一个人叫了一个女生过来，这个女生也会很害怕、很尴尬、坐立不安。女生会觉得全是男的，一个女的都没有，心里觉得你们很low，想着怎么赶紧撤离。所以罗汉局留不住女生，也会让整个酒吧的人觉得你们这卡座的人很low。我们不能向女生和其他人传递出社交价值很低的讯号，所以说罗汉局绝对不能有。

接着是“正常局”。正常局的话，男女比例至少要达到1比1。当然，最好的情况一定是女多男少。这样不仅你们会有比较高的社交价值，而且也会有“预选”，也能感受到附近很多男生投来羡慕的眼光。女生在这里也会觉得很有安全感，很愿意一起玩，也会觉得你们这桌人很有魅力。这样才能更好地在夜店中如鱼得水，更有利于建立自身给女生传递的价值和吸引，而不会像罗汉局那样出现尴尬情况。

那么最后第三个是“仙女局”。这个我就放到下节课《在夜店如何搭讪》中一起给大家讲。
