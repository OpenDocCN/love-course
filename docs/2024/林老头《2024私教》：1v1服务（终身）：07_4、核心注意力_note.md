# 林老头《2024私教》：1v1服务（终身）：07_4、核心注意力

## 概述
在本节课中，我们将学习如何培养核心注意力，这对于在约会、聊天和搭讪中取得成功至关重要。

## 核心注意力的重要性
**公式**：核心注意力 = 发心 + 行为 + 态度

以下是关于核心注意力的一些关键点：

- **心态与核心注意力**：心态好的人在面对正反馈、负反馈或OK的反馈时，都能保持平和的心态，以理性的方式处理事情。
- **感性与理性**：大部分人是以感性为理性服务，而我们应该先理性，再用感性为理性服务。
- **发心**：你的发心决定了你的行为、话术和整体表现。

## 核心注意力在约会、聊天和搭讪中的应用
以下是关于如何应用核心注意力的一些关键点：

- **自信**：在约会、聊天和搭讪中，你的核心注意力应该放在自信上，相信自己能够成功。
- **小步骤**：敢于提出诉求，不要害怕失败。
- **关注点**：将注意力放在自己身上，而不是担心对方会拒绝你。

## 盲目自信与盲目自卑
以下是关于盲目自信和盲目自卑的一些关键点：

- **盲目自信**：盲目自信有助于获取更多信息，收到更多反馈，从而更快地取得成功。
- **盲目自卑**：盲目自卑会导致你放弃尝试，从而扼杀成功的机会。

## 核心注意力与结果
以下是关于核心注意力与结果的一些关键点：

- **结果**：你的核心注意力决定了你的结果。
- **风险**：在风险可承受范围内，自我提升是最快的。

## 总结
本节课中，我们学习了如何培养核心注意力，这对于在约会、聊天和搭讪中取得成功至关重要。记住，你的核心注意力应该放在自信、小步骤和关注点上，同时避免盲目自卑。通过不断自我提升，你将更快地取得成功。
