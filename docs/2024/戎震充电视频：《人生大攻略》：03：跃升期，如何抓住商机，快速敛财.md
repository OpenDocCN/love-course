# 戎震充电视频：《人生大攻略》：03：跃升期，如何抓住商机，快速敛财

在本节课中，我们将要学习在人生的跃升期，如何识别并抓住商机，以及作为老板或创业者，真正应该聚焦的核心任务是什么。我们将探讨从执行者到管理者的思维转变，以及如何通过有效的组织和管理来实现财富的快速增长。

---

刚才录制人生大攻略第三期时，录制了七八分钟后，存储卡出现读写错误。看来这张金士顿存储卡无法使用。之前录制的内容已经讲解得非常透彻。

现在只能重新录制一遍。断更很多时候也是由于这类问题。最近文件损坏了六七次，可能是存储卡的问题，也可能是设备的问题。存储卡坏了就换卡，设备坏了就换设备。

上一节我们讨论了跃升期的概念，本节中我们来看看跃升期的具体行动策略。谈论人生的跃升期时，通常一个人会有一到三次挣大钱的机会。

但跃升期具体应该怎么做，很多人并不明白。当机会来临时，例如创业或开展新项目时，你应该做什么？

你应该做的是**如何管理手下的人才**。很多人，例如一些制片人，过去是美术、化妆、剪辑或摄像出身。因此他们常常会抠细节。

**彼得原理**阐述了这种现象：一个人晋升到管理层、成为老板后，就不应再从事具体执行工作。从事具体工作会阻碍员工完成任务的进度。

例如，开发一款游戏。流程是：先写好脚本，然后进行程序开发、音乐制作和美术设计。音乐可能需要64工时，美术可能需要192工时。

作为管理者，你应该调动更多人力专攻耗时最长的美术部分。如果程序开发需要384工时，就应再调动资源主攻程序。管理必须如此。

但你不能自己去写脚本。如果你追求完美主义或成为理想主义者，亲自介入执行环节，会导致后面的员工无所事事，浪费时间和工资。

因此，最关键的是**如何管理一个项目的工时，组织劳动，分配人员，管理生产**。你应该是战略游戏中的主控玩家。

以《帝国时代2》为例，你不能自己去养猪、种棉花、采果子。你要安排工人去做这些事，决定何时生产更多工人，何时制造更多士兵。

你不能自己去挖矿、伐木、种田、捕鱼。很多时候，一个战略决策者、一个公司老板应该做的就是这类工作。

在人生的跃升阶段，你发现了一种服务或需求，并围绕这种需求开发出数百种服务和产品。

例如性需求。国内生产飞机杯的厂商，要么抄袭美国产品，要么抄袭日本产品，然后将这些产品引入国内市场。

接下来需要进行市场调研。调研有专门的咨询公司。很多人可以以此为起点，调查某个行业在某个城市的可行性，或调查某个城市有哪些行业可行。

也可以调查某种产品在未来市场是否合适，或调查竞争对手的状况。竞争对手的引流产品是什么，盈利产品是什么。

这需要投入大量人工时间进行调研。但一些公司无法投入这种时间，因为他们可能有数百名员工，租赁着昂贵的写字楼办公区。

他们无法做这种事，就为你创造了**套利的空间**。有了套利空间，你就可以去做更大的事情。

无论如何，许多公司起步时都是从做外包、提供简单服务开始，即所谓的**B2B（企业对企业）服务**。

如果你无法掌握终端市场和客户，就可以为其他企业提供专业、专精的服务。因为大公司运营成本和公司成本非常高。

而你作为一个小公司，可以做到“小而美”。“小而美”是许多创业书籍强调的概念，讲述了小团队如何生存。

如果你是一个工作狂老板，不想雇佣太多人，一个人能顶三个人用，也**不要卡在关键工时和关键环节上**，导致其他人无法工作。

最好的办法不是线性地投入自己的时间，而是**让你的员工能够并行完成多项任务**。你自己可以身兼多职。

身兼多职可以节省工资成本，但节省不了时间成本。很多人在创业初期既是程序员，又是老板，还是业务员，需要推销产品，吃住在公司。

这类人通常要么离婚，要么单身，至少没有孩子。如果有孩子，像我现在这样，虽然有时间，但需要照顾孩子，就只能处理相对简单的事务。

很难天天应酬、推杯换盏。但作为一个老板，核心原则就是**不要去干员工该干的事**。你干了员工的事，员工干什么？

因此，谈论人生的跃升期，为什么一定要创业？创业就是**让他人代替你去完成那些复杂且没有直接创造性的工作**。

例如记账、法律事务。当然，财务和法律的专业性非常强，一般人无法取代律师和会计师。

很多枯燥的工作，如编程、画原画、音乐编曲，就应该让别人去做。你不能作为老板，觉得自己样样都行。

摄像机也要碰，化妆也要管，剧本也要改，演员也要指导。你是什么身份？你是制片人，该管的是制片管理。

很多时候，我们说**不要越俎代庖，不要外行领导内行**，这是最关键的。一个人不能破坏生产流程。

煤老板为什么不干涉演员、不干涉剧本？因为他知道不能干涉生产。这是制片人王海林的名言，说的是煤老板投资影视的时期比较好打交道。

在人生跃升期，一定要舍得开公司，舍得分配工作，舍得学习如何管理人、运用人。

你之前学习的心理学、与人打交道的技巧，这个时候就能用上了。如果你不能驱策他人，不能笼络人心，不能管理和发展生产，那你这辈子就不是一个老板，只是一个**个体创业者**。

个体创业有可能通过工匠精神，运营一个很小的公司或店铺，做到“小而美”，可以赚到钱。

但如果你想上市，想成为大集团企业，想未来进行对外投资、兼并并购，那就不可能。

这就是《富爸爸穷爸爸》中提到的**四个象限**：E象限（雇员）、S象限（自雇者/小企业主）、B象限（企业家）、I象限（投资人）。

从S象限（个体户）直接跨越到I象限（投资人）非常困难。通常的路径是：S -> B -> I。

也就是从几个人的公司，发展到几十人、上百人的公司，然后上市。这是正常的资本路径。

上市后，你作为控股集团再进行投资，收购兼并竞争对手，最终形成大的集团。这时你的钱就花不完了。

财富积累不是靠你从名校毕业、进入大厂，然后就直接变成投资人了。普通人为什么要进股市？这简直是有病，认为自己很聪明，认为自己了不起，认为自己运气好。

**股市是为了让你们赚钱而存在的吗？** 你没有5000万，好意思进场吗？拿着自己的一两百万、几十万去博弈，简直是傻子。

谁也无法破解这个财富密码。你无法直接从E象限到I象限，从S象限到I象限也难。通常路径是 B -> I 或 S -> B -> I。

在你刚决定创业时，可能一两个人就能完成所有工作。但如果你想发展壮大，**一定不能让自己天天劳累**，必须完成从S到B的转变。

因此，我建议所有人，如果你觉得时机差不多了，有足够的时间看书、丰富自己，就从E象限跨越到S象限。

注册一个个体户，或找一个地方自己办公都可以。关键是满足某种特定的需求。

因为同一种需求，可以通过多种方式满足。例如性需求，可以是服务，也可以是产品（如飞机杯）。

飞机杯行业，国内也就那几个品牌。你只要盯住一个东西，不用搞发明创造，抄袭借鉴就行。把一个东西做好了，你就有可能成为上市公司的老板。

人做一百件事很难，但**把一件事做专、做精并不难**，只需要朝着一个方向努力。这叫做**单点突破**。

你要把某一件事情、某个产品做精、做赚钱就可以。所谓的“第一桶金”往往就是这样捞到的。

大家要明白，要满足一种需求，并发挥到极致。发挥到只有你能提供，发挥到你能提供的东西最优秀，发挥到你的东西在特定时间、特定城市最具性价比。

不要老想着进攻全国市场，不要想把所有人的钱都赚到。你只能在某一段时间、某一个领域，赚某一部分人的钱。

这才是满足需求，这才是生意的基础，这也是人生大攻略的基础。就是说，人生在上升期的时候，**一定要有人帮助**。

不光是贵人帮助（上行），告诉你某种需求，为你打开资源后门；下边就要组织人去生产，去满足这种需求（下效）。你满足需求的速度越快，你赚钱的速度就越快。就是这么简单。

---

本节课中我们一起学习了在跃升期抓住商机的核心：完成从执行者到管理者的思维转变。老板的核心工作是**战略规划、资源分配与人员管理**，而非亲力亲为。通过发现需求、组织生产、满足需求，并借助“小而美”起步，逐步向企业家（B象限）迈进，是实现财富快速增长的有效路径。记住，单点突破与高效组织是成功的关键。
