## 变焦操作

在苹果手机中，变焦操作可以通过以下步骤实现：

1. 在拍摄界面点击0.5即可切换到超广角。
2. 点击3倍可直接切换到3倍长焦进行拉近构图。

**公式**： 变焦倍数 = 摄像头焦距 / 主摄像头焦距

## 变焦对画质的影响

变焦对照片的成像画质，即清晰度有影响。拉近景物拍摄会导致清晰度下降。具备两个或三个摄像头的苹果手机具备无损光学变焦倍数，一般为0.5倍、超广角搭配一倍，或0.5倍、一倍、两倍、三倍搭配。

**代码**： 无损光学变焦倍数 = 0.5, 1, 2, 3

## 变焦倍数的选择

苹果手机变焦倍数最好不要超过4倍，否则会导致照片清晰度严重下降。在光线充足的情况下，变焦到4倍仍可接受。老款苹果手机在光线充足的情况下，变焦到3倍即可。

## 超广角模式

超广角模式能够拍摄到更宽广的画面。目前仅苹果11以上的系列具备超广角模式。超广角模式适合拍摄风景、建筑、室内场景等，能够拍出宽阔的视野。

## 超广角拍摄案例

以下是一些超广角拍摄场景的案例：

1. 山谷风景：使用超广角模式拍摄，视野更开阔，视觉冲击力更强。
2. 室内建筑：使用超广角模式拍摄，建筑结构空间更开阔，地面线条更具视觉引导。
3. 城市建筑与树叶：使用超广角模式拍摄，树叶与建筑形成对比，画面更具视觉冲击力。

## 变焦拉近拍摄构图

在某些场景中，需要突出画面中的主体，此时可以进行变焦拉近操作。

以下是一些变焦拉近拍摄构图的案例：

1. 树林中的人物：使用两倍变焦，人物更突出，前景地面更少，背景树与人物关系更紧凑。
2. 户外傍晚活动：使用4倍变焦，突出灯笼与月亮的对比，营造简洁意境。
3. 雪山：使用不同倍数变焦，展现雪山与前景的对比、呼应以及色彩细节。
