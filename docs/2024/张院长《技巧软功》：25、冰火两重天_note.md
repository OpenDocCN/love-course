# 张院长《技巧软功》：25、冰火两重天_note

在本节课中，我们将学习一种名为“冰火两重天”的感官刺激技巧。该技巧通过交替使用低温（冰淇淋）与温热（牛奶）的触感，来创造强烈而愉悦的身体体验。核心在于温度的对比与交替，以及事前充分的沟通。

上一节我们介绍了课程主题，本节中我们来看看具体的操作步骤与核心要点。

## 核心原则与准备

进行“冰火两重天”体验前，必须遵循两个核心原则：**先使用冰，后使用热**；以及**务必进行事前沟通**。沟通内容包括确认双方对冷热温度的接受范围，因为每个人的体感不同。

以下是需要准备的材料与温度控制要点：
*   **冰**：使用冰淇淋作为低温源。
*   **热**：使用温牛奶作为热源。牛奶温度需控制在**30摄氏度左右**，不宜过烫（例如40-42摄氏度）。
*   **沟通公式**：在行动前询问对方“**你觉得这个温度怎么样？**”

## 操作流程详解

操作的整体逻辑是：在目标区域先用冰淇淋划动，制造冰冷触感，随后立即用温牛奶沿相同路径划动，形成冷热交替的刺激。

### “冰”的阶段：使用冰淇淋

首先进行“冰”的阶段。用冰淇淋在对方身体上缓慢划动。对方初始可能会因冰冷感而产生身体颤动或收缩的反应，这属于正常现象。

以下是主要的划动路径指引：
*   **背部**：从脊椎骨上方开始，向下划动至屁股沟。背部两侧也可按相同方式处理。
*   **正面躯干**：从肚脐眼开始，向下划动至阴茎根部。腹股沟区域也应从上方划动至下方。
*   **胸部**：从胸沟上方开始，向下划动。
*   **通用路径**：所有划动均采用**从上往下的方向**。

在划动过程中，可以加入视觉和语言的挑逗，例如在对方面前演示如何舔舐冰淇淋，或用语言进行暗示。

### “热”的阶段：使用温牛奶

完成冰淇淋划动后，随即进入“热”的阶段。使用温牛奶，严格按照刚才冰淇淋划过的相同路径，再次进行划动。

操作后，可以轻柔地舔舐经过处理的区域。此时对方会体验到明显的冰火交替的刺激感，可能产生强烈的生理反应。

### 其他可操作区域

除了躯干，此技巧同样适用于其他敏感区域。

以下是大腿内侧的操作建议：
*   在大腿内侧正面，同样遵循**先冰后热**的顺序进行操作。

## 关键注意事项与总结

本节课中我们一起学习了“冰火两重天”技巧的完整流程。最后，请务必牢记两个总结性要点：

1.  **温度无绝对标准**：所需的冷热温度因人而异，**事前沟通至关重要**。
2.  **流向的总结**：身体上部的操作流向是**从上往下**，身体下部的操作最终感觉应**汇聚至生殖器及臀部区域**。
