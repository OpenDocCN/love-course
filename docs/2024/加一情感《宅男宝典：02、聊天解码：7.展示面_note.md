# 加一情感《宅男宝典：02、聊天解码：7.展示面_note

在本节课中，我们将要学习“展示面”这一核心概念。展示面是你在社交媒体（尤其是朋友圈）上呈现的个人形象，它是吸引女生、开启对话的基础。我们将系统地拆解其组成部分，并提供具体的打造方法。

## 概述：什么是展示面？🤔

正确的展示自己，是追女生最基础的东西。简单来说，展示面就是扬长避短地通过你的朋友圈告诉别人：**我是谁？我有什么？我的生活是什么样的？**

我们和女生聊天时，女生不回复或态度冷淡，有时并非你不会聊天，而是对方根本不给机会。客观来讲，你需要打造好展示面，才能让女生愿意跟你聊天，愿意去了解你。

## 展示面的五大组成部分 🧩

展示面主要由5个部分组成：**昵称、头像、背景墙、签名、常规朋友圈**。

上一节我们介绍了展示面的整体概念，本节中我们来看看这五个部分的具体要求。

### 1. 昵称：品味的第一印象

昵称是他人对你品味的初步判断。以下是昵称设置的建议：

*   **避免使用**：“追风少年”、“风度翩翩”、“哥是个传说”等非主流或暴露年龄的昵称。
*   **推荐使用**：简单明了的昵称。可以是一个英文名，或英文加汉字的组合。例如，我的昵称就是“alen”。总之，无论何种形式，**简单就是高级**。

### 2. 头像：你是谁的关键证明

头像告诉别人你是谁，证明你是一个真实的人。它至关重要，因为第一印象决定对方是否愿意接近你。

在学习如何拍摄头像前，先了解哪些照片不适合做头像。以下是头像的“黑名单”：

*   风景照
*   动物照
*   证件照
*   卡通照
*   网图或画质很差的照片

那么，正确的头像应该怎么拍呢？关键在于营造氛围和感觉，而非单纯展示相貌。

*   **技巧一：融入环境**。将人物置于一个高大上的背景中。当图片缩小为头像时，视觉中心会从人物相貌转移到整体环境和感觉上。
*   **技巧二：借助道具**。可以借助宠物或娃娃等道具，拍出温暖、有爱心的感觉。
*   **技巧三：制造神秘感**。利用饮料、手机、口罩、围巾、帽子等道具进行局部遮挡，不要全脸出镜，以引发女生的好奇心。

### 3. 朋友圈封面：你的黄金广告位

如果说头像是吸引女生浏览朋友圈的“引流器”，那么朋友圈封面就是你的“黄金位置广告牌”。使用不当的封面，会直接告诉女生你是一个很low、没有品位的男人。

以下是绝对不能用作封面的类型：

*   网图
*   大头自拍（无论帅否，都显得自恋）
*   过时的旅游照（特别是比剪刀手的）
*   普通的风景照（除非拍得非常艺术化）

那么，朋友圈封面应该选用什么样的照片呢？核心原则是：**展现价值**。

所谓的价值，可以是别人拍不出来的独特照片，或是逼格很高、能体现你生活水平且非大众化的照片。例如：

*   潜水的照片
*   坐飞机头等舱的照片
*   冲浪的照片

如果实在没有此类生活照，一张有质感的纯文字图片，或从网上寻找的高逼格图片也是备选。**记住：拍好的照片一定要修图。** 当你养成“不修图，不发朋友圈”的原则时，你的展示面就成功了一半。

总之，封面图需要传达的信息是：**我是个精致、生活丰富、具有高价值、审美在线的人。**

### 4. 签名：价值观的简短宣言

签名告诉别人你的三观。它可以是一句意味深长的话，也可以是一句英文。核心目标是：**让女生觉得你是一个有品位的男人。**

### 5. 常规朋友圈：你经营的核心产品

头像是“引流器”，封面是“广告牌”，日常朋友圈就是你为自己经营的“产品”。如果日常内容质量低下，前两者做得再好也无济于事。

首先，我们需要明确哪些朋友圈是“雷区”，绝对不能发：

*   **转发的心灵鸡汤或无病呻吟的文章**：女生没兴趣看，会把你定义为“屌丝”。
*   **纯文字朋友圈**：提供的信息价值低，无法让女生了解你。
*   **随意拍摄的、低质量的朋友聚会/宵夜照**：“物以类聚，人以群分”，低质的环境和朋友会拉低你的价值。给朋友拍照也要认真，并选择不错的环境。
*   **工作广告、推销信息**：请使用微信的“分组可见”功能，将广告只展示给客户组。
*   **太过专业的学术性内容**（如程序员发代码，金融从业者发专业分析）：大多数女生看不懂也不感兴趣，无法因此觉得你厉害。

了解了不该发什么，接下来我们看看应该发什么。以下是高质量朋友圈的内容方向：

*   **美食**：发的是有品位的“美食”，而非日常的兰州拉面、沙县小吃或街边烤串。目的是打开女生窗口，引发“这是哪里？看起来好好吃”的提问。
*   **丰富的娱乐生活**：展示如健身、骑马、旅行、游泳、射箭、桌球、家庭派对、画展、摄影展等活动。这传达出：你是一个上进、愿意自我投资、有艺术修养的人，与宅男不同。
*   **适当的形象展示**：让女生知道你长什么样，培养安全感。但频率不宜过高，避免显得自恋。**最重要的是：修图！修图！修图！**
*   **宠物或小朋友的展示**：女生对可爱的事物抵抗力最低。这能极大程度地打开女生窗口，并展示你的爱心。
*   **预选展示**：即晒出你与其他女生的合照。注意两点：1) 不要单独与一个女生合影，避免误会；2) 合照女生的颜值不能太低，否则会拉低你的价值。这能引发女生思考：“为什么他身边这么多女生？因为他很优秀。”
*   **工作/学习展示**：例如，在凌晨5点发一条“又是天亮才出公司”的朋友圈，简单展示自己努力、上进的形象。
*   **朋友圈配文原则**：**宁愿少，不要多**。目的是打开对话窗口，如果把事情讲得太清楚，女生就没有提问的欲望了。如果不知如何配文，可以参考相关论坛的素材。
*   **音乐分享**：分享你认为好听、歌词好、有感触的歌曲。歌曲品味很关键。如果你欠缺这方面的知识，可以关注一些有品位的歌单作为参考。

## 总结与行动指南 📝

本节课中，我们一起学习了展示面的定义、五大组成部分及其具体的打造方法。

最后，给大家三条行动指南：

1.  **立即清理**：删掉课程中提到的所有不合格的朋友圈内容，并按照建议调整你的展示面。
