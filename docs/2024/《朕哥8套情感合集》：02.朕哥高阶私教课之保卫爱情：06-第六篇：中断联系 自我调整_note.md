## 理解情感关系的底层逻辑：电路图理论 ⚡

那么遇到这样的情况，我们该怎么做呢？我拿一个图来给大家举个例子，讲一讲男女之间的关系，以便你们更好地理解。

我们可以把男女之间的爱情看作是电流。男女之间的感情关系就好比是一个电路图。在你们的感情处于正常状态的时候，也就是说在你们彼此相爱的时候，这两个开关是接上的。也就是说你的爱给了他，他的爱也给了你。

而如果这个男生对你的爱产生变化了，就好比是男生这边的开关断了，他不再接受你传递给他的任何东西。但是你不愿意放弃，你想通过给他更多的爱来得到一些反馈。你这样的做法只会让你的情感更加枯竭，而且男人他也接受不到。

这种时候我们就要学会接通自己的电路，学会自己爱自己、充实自己，达到自我滋养。这样自己的情感状态才会正常，才能保证自己的能量电流不会枯竭。

这时可能有很多同学好奇，既然男人对自己的爱已经飘忽不定了，我还把他放到一边，那不会更危险吗？其实不是的。当你的电流足够强大，各方面都很优秀的时候，那么男人他就会想要连接在你身上。也就是说你身上有了价值之后，就会吸引他。这个时候你们的情感电路才会重新接通。

**核心公式：**
```
吸引力 = 自我价值
关系连接状态 ∝ 吸引力
```
如果你对男人没有了吸引力，你们之间的电路没有接通。那你再怎么使劲，你们之间的信号都是传递不过去的。

当我们明白了这个底层逻辑，再来看看你们之间的关系的时候，我们就知道当男人对自己的爱开始飘忽不定时，我们一定要先做好自我的调整。

## 实现自我调整的四步法 🧭

其实自我调整，最根本的问题就是你能否做到真正的情感独立。“情感独立”这个词相信同学们一定听过，今天我就把具体的方法交给你们。

### 第一步：中断联系，聚焦自我

当一段感情中男人已经不再那么用心，或者你已经失去了这段感情，那么暂时与他断开联系就是必须的。

以下是具体操作：

*   **情感上的中断**：如果你们的矛盾比较大，你可以中断和他的联系，暂时不要与他见面。微信和电话上也不要再联系对方。记住，不是拉黑微信和电话。很多女人跟男人一吵架就会拉黑微信和电话，这样的做法不但没有用，而且切断了自己的后路，只会让你们之间的隔阂更深。
*   **物质上的中断**：你可以将与他无关的物品都封存起来，也就是所谓的眼不见心不烦。这样你看不到他送你的各种东西，就会减少想他的次数，就能够将更多的时间和注意力聚焦在自己身上。也要中断其他你们在物质上有往来的东西，比如你之前使用了他给你办的亲属卡，一定要解绑。不要存在侥幸心理，在男人对你没有恢复好感之前，这些行为往往会适得其反。

这点看起来可能是非常不起眼的一小步，但是我们之前的一个会员，就是因为做出了这一小步，每天的生活状态都得到了极大的改变。

### 第二步：理清思绪，减少投入

当我们断开了与男人的联系之后，就要开始理清自己的思绪了。

我知道每个女生在和男友闹矛盾之后，都会胡思乱想。想他有没有爱过我，想着他会不会另寻新欢，想他是不是会无缝衔接。不过，既然现在的局面已经成了定局，那你思考得过多只会消耗自己的精力，而且不会有任何作用。

所以你要停止这些无谓的思考，减少在思想上为男人花费的时间，把心态调整为“单身的心态”。只有你的思绪控制下来，我们才会减少精神上和物质上对他的投入。我们才有赢回爱情的可能。

### 第三步：自我滋养，补足能量

前面两步的办法是让同学们不再浪费不必要的情感和精力。那么接下来的两步，我们就要从思想和情感两方面把自己失去的补回来。所以第三步我们要做的就是自我滋养。

以下是三个滋养的方面：

*   **思想上的滋养**：同学们可以先思考一个问题，你有多久没有好好的为自己考虑了？很多女生在找我分析完自己的问题之后，都会跟我说是不是应该再努力点。当我问她努力什么的时候，她们就回答是努力的去和老公沟通，努力的去为老公着想，努力的照顾好孩子，或者努力的理解婆婆的想法。我想说的是，我们女性真的是已经给自己的压力太大了。其实生活中的种种压力，你想要靠自己一个人的力量是很难去承受的，你是很难做到让每件事情都井井有条的，你是很难获得所有人的满意的。所以其实我们大部分的女性都将自己困在“我要努力让家庭变得更好，我要努力让老公更爱我”这样的怪圈里。你仔细想想，真正的“更好”其实就是让你自己变得更好。当你自己变好了之后，你就会感受到一切都在朝好的方向发展。所以想要做到自我滋养，那必须要先从思想上开始改变。你要改变自己的思想：你不是为你的老公、孩子、你的家人而活的，你就是为你自己而活的。
*   **生活中的滋养**：你也可以提升一下自己的生活质量，换种方式生活。比如说以前总是将大部分的金钱精力投入在了男人、家庭身上，现在就要多投入在自己身上。你可以给自己买些昂贵的护肤品，让自己的皮肤状态变得更好，让自己看起来更年轻。我相信女人对这方面都是毫无抵抗力的。或者从生活作息中提高质量，比如可以早上起来学习一会瑜伽或者跑跑步，晚上可以做一些犒劳自己的美食。也可以给自己买一些自己喜欢好久，却一直没舍得入手的好东西，包括衣服、包包、各类首饰之类的。物质生活上的提升也会给自己的内心带来滋养。也可以提升自己的穿搭审美，让自己的生活变得丰富多彩。
*   **情感上的滋养**：你不仅仅是只有和男人之间的情感，你还有和亲朋好友之间的情感。比如你可以和之前的朋友多出去聚一聚。和亲戚朋友们在一块，他们的情感和思想就会感染你，会让你更快的看清你自己，他们也会给你一些安慰和建议，那你也会慢慢的就能找到解决你们矛盾的办法。你也要多认识一些新的朋友，丰富自己的情感世界。比如说可以去学习一些自己感兴趣的东西，画画、舞蹈、插花之类的。在学习新技能的同时，提升了自己的价值，也丰富了自己的情感世界。如果你是个社恐的话，就可以培养和宠物之间的情感。总之，丰富情感世界的办法是有很多的，包括听音乐、看电影，都可以丰富你自己的情感世界。当你自己的情感世界变得丰富了，那你的情感力也就会逐渐提升，你的情感就能够有所寄托，也就能够有所滋养了。

其实女性想要做到经济独立、思想独立都是比较容易的，但是想要做到真正的情感独立却是非常困难的。因为女人都是渴望被照顾被爱的，都想有个男人能够理解自己、能够成为自己的大树。但当你能够做到以上三点的时候，你就已经无限接近了情感独立。

### 第四步：重建吸引，展示价值

那么接下来我们就要开始第四步：重建吸引。重新建立吸引，就是说你要利用好自己改变后的优势。换句话说，就是要展示之前没有展示过的一面。

除了情感生活上的断舍离，你在自我形象上也要学会展示全新的自己。这在女生眼里可能只是尝试不同的风格，但是当你晒出不同风格照片在朋友圈之后，这在男人眼里却是你离他越来越远了，你离开他之后生活越来越好了，他就会开始产生自我怀疑，产生不安的情绪。

除了在形象上的改变，你也可以有意无意的秀一些自己生活或者心态上的改变。这个时候，自我的滋养就派上了用场。

**关键技巧：**
```python
# 重建吸引的正确时机判断
if 自我改变程度 >= 显著阈值:
    展示新形象与新生活()
else:
    继续专注于自我滋养()
```
这里大家要注意的是，你一定要在自己已经改变了之后再抛出重建吸引的诱饵。如果你操之过急，自己没有做到改头换面就急于展示的话，那很可能会引来男人的反感。

这里有一个非常重要的小技巧，就是你要学会借助评论区的力量。当你在朋友圈晒出了自己近期的生活日常或者美照之后，你就可以在朋友圈的评论区直接回复朋友的评论，制造出有更多竞争者的假象。这样就可以重新建立你对男人的吸引力。

如果你能做到以上四点，你就会重新恢复吸引力，男人也会重新被你吸引。
