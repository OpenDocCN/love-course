# 课程01：CAC昊哥《进化蓝图训练营》- 023. 自信的三个层级 🧠

在本节课中，我们将要学习自信的三个不同层级。许多人对自信的理解停留在表面，认为它来源于外在的拥有或做到。我们将深入剖析这三个层级，帮助你理解自信的真正来源，并引导你走向更稳定、更真实的内在自信。

## 自信的三个层级

自信并非单一概念，它可以根据其来源分为三个不同的层级。理解这些层级有助于我们看清自己当前的状态，并找到向更高层级迈进的方向。

### 第一层级：因“拥有”而自信 (Having)

上一节我们介绍了课程概述，本节中我们来看看自信的第一个层级。这个层级的核心信念是：**自信源于我拥有什么**。

其公式可以表示为：**自信 = 我拥有 [某物/某身份]**。

以下是这一层级的具体表现：
*   我自信是因为今天我穿了增高的鞋，让我看起来像1.9米。
*   我自信是因为我今天做了一个新发型，让我看起来很帅。
*   我自信是因为我穿了一双限量版球鞋。
*   我自信是因为我是坐头等舱或打专车来的。
*   我自信是因为我有比他们都漂亮的女朋友。
*   我自信是因为我有一份很厉害的工作或能支付别人付不起的旅行费用。

大多数人（约95%）的自信建立在这个层级上。他们认为，自信来自于外在的物质、身份或结果。

为什么会有这种想法？主要有两个原因：
1.  这是从小被灌输的结果。父母、周围人、接触的异性都可能传递这种信念。
2.  这是一种借口。本质上自卑的人，无法直接承认自己的懦弱，于是找借口说“我现在不自信是因为我还没钱”，潜台词是“等我有钱了就会自信”。

### 第二层级：因“做到”而自信 (Doing)

在了解了因“拥有”而自信的局限性后，我们进入第二个层级。这个层级的核心信念是：**自信源于我能做到什么**。

其公式可以表示为：**自信 = 我能做到 [某事]**。

以下是进入这一层级的表现：
*   我今天用了推拉、复合、筛选等技巧，拿到了好结果。
*   我用了高价值的语言体系，对方对我很尊重。
*   我状态很好，操作顺利，结果很棒。

这一瞬间的自信，是因为“我做到了”。这与第一层的区别在于，第一层的借口是“我没有什么，所以不自信”，而第二层是“我做到了，所以自信”。

然而，这一层级也存在问题：
1.  你的力量来源是外界（如技巧、方法）。这些并非永远稳定，当你疲劳、缺乏睡眠时，状态会消失。
2.  你的自信取决于对方的反馈。技巧对段位低的人有用，但对见识更广、段位更高的人可能无效。一旦得不到好反馈，你就会慌，自信随之消失。
3.  这会让你感到累，无法享受过程。你会觉得对方喜欢的是你的技巧，而不是真实的你。

这会导致一种依赖状态和运气的模式，无法获得稳定的自信。

### 第三层级：真实的自信 (Being)

既然因“做到”而自信依然受制于外界，那么真正的解决方案是什么？本节我们将探讨自信的终极形态：真实的自信。这个层级的核心信念是：**我的存在本身就是自信的**。

其核心概念是：**我有权有力量**。

这意味着你的力量感与外在一切无关（名牌、发型、技巧、豪车、学历）。你本身就认为自己是一个有权表达想法、树立边界、影响他人的人。

关键不在于你知道怎么说、怎么做，而在于你**敢不敢**。许多人不认为自己有权利这样做，所以才会话题跟着别人走，不敢植入自己的框架。

我们要解决的就是让力量从外界、技巧、服饰回归到自己身上。在这种状态下，你会感受到：
*   不依赖状态，拥有稳定的心态。
*   不会总想逃跑，也不会感到特别疲劳。

## 课程总结

本节课中，我们一起学习了自信的三个层级：
1.  **因“拥有”而自信**：自信建立在物质、身份等外在条件上，不稳定且易找借口。
2.  **因“做到”而自信**：自信建立在技巧和结果反馈上，受状态和他人影响大，令人疲惫。
3.  **真实的自信**：自信源于“我有权有力量”的内在信念，不依赖外界，稳定而持久。

理解这些层级，是迈向真正内在力量的第一步。
