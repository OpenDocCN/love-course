# 01、CAC昊哥《进化蓝图训练营》：023.第二部分-L1.自信的三个层级_ev_note

在本节课中，我们将要学习自信的三个不同层级。我们将深入探讨大多数人对于自信的常见误解，并揭示更稳定、更强大的自信来源。理解这些层级，将帮助你摆脱对外在条件的依赖，建立真正由内而外的力量感。

## 第一层级：因“拥有”而自信 (Having)

上一节我们介绍了课程的主题，本节中我们来看看自信的第一个，也是最普遍的层级。

如果你打开抖音或B站等平台的评论区，任何关于自信的作品下，常见评论是：“我有钱了，我就自信了。我现在就是穷，等将来我有钱了……” 或是“今天你对我爱答不理，明天我让你高攀不起”。为什么很多人想通过整容、赚钱来获得自信？这些想法反映了他们处于自信的第一个层级：**因为拥有什么而自信**，即 `Having`。

以下是“因拥有而自信”的具体表现：

*   我自信是因为今天我穿了增高的鞋，让我看起来像1.9米。
*   我自信是因为我今天做了一个新发型，让我看起来很帅。
*   我自信是因为我今天穿了一双限量版球鞋。
*   我自信是因为我是坐头等舱或滴滴专车来的。
*   我自信是因为我有比他们都漂亮的女朋友。
*   我自信是因为我能付得起别人付不起的旅行费用。
*   我自信是因为我有体面的工作、身份或成就。

这是95%的人所认为的自信，它建立在 **“我拥有什么”** 之上。产生这种想法的原因有两个：第一，这是从小被家庭、社会灌输的观念；第二，这本质上是一种借口。一个自卑的人无法直接承认自己的懦弱，于是借口说“我现在不自信是因为我还没钱”，潜台词是“等我有钱了就会自信”。

## 第二层级：因“做到”而自信 (Doing)

在了解了基于外在拥有的自信后，我们进入第二个层级。许多学习过社交技巧或情感课程的人会达到这一层。

第二个层级的自信叫做 **“因为能做到什么而自信”**，即 `Doing`。这与第一层的核心区别在于：第一层的借口是“我没有什么，所以我不自信”；第二层则是“我做到了，所以我自信”。

例如，当你学习了某些技巧（如推拉、筛选、高价值语言），并在实践中取得了好结果——对方对你态度尊重、反馈积极——这一瞬间你会感到自信。你的自信来源于 **“我做到了”**。

然而，这一层级也存在问题。以下是它的主要缺陷：

*   **力量来源是外界**：你的自信依赖于技巧、方法或课程（如CAC）。这些外在工具并非永远可靠。
*   **状态极不稳定**：当你疲劳、缺乏睡眠或状态不佳时，这些技巧和随之而来的自信会消失。
*   **依赖对方反馈**：你的自信建立在对方的“好反馈”上。如果面对段位更高、见识更广的人，你的技巧未产生预期效果（得到“坏反馈”），你会立刻感到慌张和自卑，因为“我没做到”。
*   **过程令人疲惫**：你不会享受互动本身，而是觉得“对方喜欢的是我的技巧，而不是真实的我”。这会引出一个常见问题：“在一起后还需要一直用技巧吗？当我表达真实自我时，对方就变冷淡了。”

外在技巧可以帮助你取得结果，但无法让你获得稳定、快乐的心态和长期积极的状态。这就像戴上了一副“人格面具”，在不同人面前切换不同面孔。而真正的成长，是摘掉面具，解决真实自我的问题。

## 第三层级：真实的自信 (Being)

既然前两个层级的自信都有其局限性和不稳定性，那么什么才是更根本的解决方案呢？本节我们将探讨自信的终极形态。

第三种，也是最高层级的自信，是 **“真实的自信”**。它意味着：**我认为我的存在本身就可以是自信的。** 你的力量感不再来源于任何外在事物。

*   **过去**：力量感来源于名牌球鞋、发型、长相、豪车、学历、或某种技巧（如CAC）。
*   **现在**：力量感与外在无关，它源于一个核心信念：**“我有权有力量”**。

这个概念至关重要。我们大多数人接受的教育和经历，让我们不认为自己有权成为一个可以影响他人、表达自我、树立个人边界的人。因此，我们才会不敢主导话题，不敢植入自己的框架。问题不在于“不知道怎么说”，而在于 **“不敢去说”**。

**真实自信** 的目标，就是将力量从外界（衣服、技巧）收归自身。在这个状态下，你会获得稳定的心态，不依赖状态，不会感到疲惫或想要逃避。

## 总结

本节课中，我们一起学习了自信的三个层级：

1.  **因“拥有”而自信 (Having)**：自信建立在物质、外貌、身份等外在条件上。这是最普遍但也最不稳定的层级。
2.  **因“做到”而自信 (Doing)**：自信建立在运用技巧并取得良好结果上。它比第一层进步，但依然依赖外界反馈和个人状态，容易波动。
3.  **真实的自信 (Being)**：自信源于“我有权有力量”的内在信念。你的力量感与外在无关，因此能获得最稳定、最持久的状态。这是我们追求的目标，也是后续课程中将着重内化和培养的核心。

理解这些层级，是迈向真正自信的第一步。
