# 社交蓝图从零到社交高手系列课情商提升构建社交体系教程：10-实接时的规则

🎼欢迎回来，我是探花Gary。今天的课程是你在与对方接触时需要遵守的10个规则。这10个规则当中有5个是要重点避免的，另外5个是要重点去做的。我们先从需要重点避免的几条开始。

### 避免的规则

1. **不要刻意去表现自己**  
   **公式**：刻意表现 → 低价值 → 低魅力  
   **解释**：避免在对方面前刻意展示自己，因为这会让人感到不舒服，反而降低你的价值。

2. **不要去寻求一个人的认可**  
   **公式**：寻求认可 → 低价值 → 低自信  
   **解释**：不要过度依赖他人的认可，而是要培养自我认可，提高自信。

3. **不要选择不方便的地方见面**  
   **公式**：方便地点 → 良好体验 → 高价值  
   **解释**：选择方便的地点见面，可以提升双方的体验，展现你的高价值。

4. **不要所有的选择都询问对方**  
   **公式**：主导选择 → 高价值 → 高领导力  
   **解释**：在社交中，要有主见，不要总是询问对方的意见，展现你的领导力。

5. **时间结束，不要过度纠缠**  
   **公式**：过度纠缠 → 低价值 → 低好感  
   **解释**：在时间结束时，不要过度纠缠对方，以免给人留下不好的印象。

### 应该做的规则

6. **有一个良好的精神面貌和个人卫生**  
   **公式**：良好形象 → 高价值 → 高吸引力  
   **解释**：保持良好的形象和卫生，展现你的高价值和吸引力。

7. **在去实际的路上听自己喜欢的音乐**  
   **公式**：放松状态 → 高价值 → 高吸引力  
   **解释**：在去见面的路上听自己喜欢的音乐，让自己放松，展现你的高价值和吸引力。

8. **选择人少的地方坐下来**  
   **公式**：私密空间 → 深度交流 → 高价值  
   **解释**：选择人少的地方坐下，可以促进深度交流，展现你的高价值。

9. **选择在情绪的高点升级或者提要求**  
   **公式**：情绪高点 → 高接受度 → 高成功率  
   **解释**：在对方情绪高涨时提出要求或升级，更容易被接受。

10. **不要恐惧安静**  
    **公式**：适应安静 → 高价值 → 高吸引力  
    **解释**：学会适应安静，展现你的高价值和吸引力。

### 总结

本节课中我们一起学习了在实接时需要遵守的10个规则，包括需要避免的5个规则和应该做的5个规则。希望这些规则能够帮助你提升社交能力，成为社交高手。
