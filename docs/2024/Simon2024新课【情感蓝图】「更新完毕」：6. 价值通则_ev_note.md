# Simon2024新课【情感蓝图】「更新完毕」：6. 价值通则

各位好，我是Simon。今天我将分享第六节课程内容，上节课我们讨论了价值和境，并在课程末尾提到了价值通则，也就是本节课的主题。

## 价值相对性

**公式**：价值 = 相对性

价值是相对的，不是绝对的。总有人比你价值更低，因此“优秀”是一个相对概念。提升自我价值的意义在于达到最舒适、最自洽的状态，并寻找适配自己价值范围的对象。

## 价值理解与适配性

**公式**：价值理解 = 个人理解 + 适配性

很多人有才华和上进心，但在搭讪过程中，这些特质可能无法展现，导致被拒绝。被拒绝并不意味着被否定价值，而是因为双方对价值的理解和适配性不同。

## 三观气合与灵魂伴侣

**公式**：三观气合 = 不存在

不存在三观气合或灵魂伴侣。当自身有价值时，对方会理解并支持你的行为。价值是情感现象的本质，决定事物背后的逻辑。

## 价值与行为

**公式**：价值高 ≠ 为所欲为

价值高并不意味着可以为所欲为。过度展示短期属性会让女生缺乏安全感，导致关系破裂。

## 软价值与硬价值

**公式**：软价值 = 长远价值的核心

软价值是长远价值的核心，关系到关系中的长期博弈。硬价值者在关系中可能沦为ATM，被反复收割。

## 总结

本节课中，我们学习了价值通则，包括价值相对性、价值理解与适配性、三观气合与灵魂伴侣、价值与行为、软价值与硬价值等概念。希望这些内容能帮助大家更好地理解情感关系。
