# 子阳《脱单实战聊天课》：1.聊什么：第三节课：聊天的评估依据

## 概述
在本节课中，我们将探讨如何评估与女生聊天的内容，以及如何根据当前关系定位聊天话题。

## 聊天评估依据的重要性
当你准备与女生开始聊天时，你首先会思考一个问题：今天要聊什么？然而，很多兄弟在评估聊天内容时缺乏依据和标准。**（公式：无评估依据 → 无方向 → 聊天效果不佳**）

## 分享日常的局限性
分享日常是一种常见的聊天方式，但通常只能维持3到5句对话。每天分享日常会显得无聊，因此需要寻找其他聊天话题。

## 评估依据的缺失
大部分人在聊天时没有明确的评估依据，导致聊天效果不佳。**（公式：无评估标准 → 无法应对女生回复 → 聊天效果不佳**）

## 关系定位的重要性
在确定聊天内容之前，首先要明确你们当前的关系。**（公式：明确关系 → 适合的聊天内容 → 聊天效果提升**）

## 时机的重要性
有些兄弟可能会在抖音或网上学习一些土味情话或幽默话术，但时机不对会导致效果不佳。**（公式：时机不对 → 话术合适但效果不佳**）

## 总结
本节课中，我们学习了如何评估聊天内容，以及关系定位的重要性。希望这些知识能帮助你提升聊天效果。
