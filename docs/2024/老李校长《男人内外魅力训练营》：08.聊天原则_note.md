# 老李校长《男人内外魅力训练营》：08.聊天原则

🎼 聊天原则。🎼 情感之路充满挑战，唯有勇敢面对，方能成为英雄。勇敢的男人，跟我一起探索情感的奥秘，看透人性的善恶。

**核心概念**： 
- **人生大圆满**： 品尝世间人性的情与物。
- **提升魅力**： 不用话术，提升人设魅力。

**以下是聊天原则**：

1. **不要秒回**： 微信消息不要秒回，避免显得没有价值。
2. **避免过多语气词**： 刚开始聊天时，避免使用过多语气词，以免显得讨好。
3. **不要加表情包**： 新手不要加表情包，以免显得讨好。
4. **敢于表达不满**： 面对冒犯，敢于表达不满，甚至拉黑。
5. **敢于失去**： 面对试探，敢于失去，展现男人的魅力。
6. **平等交流**： 不要刻意讨好，平等对待对方。

**总结**：

本节课中，我们学习了老李校长的聊天原则，包括不要秒回、避免过多语气词、不要加表情包、敢于表达不满、敢于失去和平等交流。这些原则将帮助我们在情感中成长，获得更好的结果。
