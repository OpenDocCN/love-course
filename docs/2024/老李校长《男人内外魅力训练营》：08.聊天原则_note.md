# 老李校长《男人内外魅力训练营》：08.聊天原则_note

在本节课中，我们将要学习与异性聊天时的核心原则。这些原则旨在帮助你建立自信、维护自我价值，并在互动中占据平等甚至有利的地位，而非依赖刻板的话术。

情感的道路并非总是鲜花与美酒，情感的世界也可能充满挑战。唯有洞察规则并勇敢实践的人，方能掌握主动。接下来，让我们一同剖析这些在实战中总结出的聊天原则。

---

我一直强调，大家不应去学习所谓的微信聊天课，也不应背诵固定话术，更不应使用辅助聊天软件。那些方式显得非常不得体。例如，我曾看到有人使用聊天软件，试图用“一起洗澡”之类的话显示风趣，结果却令人尴尬。这本质上是因为缺乏经验与原则。

实际上，这些软件和话术往往会破坏你的形象。本节课将告诉你，掌握核心原则远比学习零散话术重要。我们都会说中文，关键在于理解并运用这些原则。

遵循原则不仅能避免犯错，还能提升你个人形象的魅力。你可以以不变应万变，灵活应对各种情况，而不会被死板的话术束缚，显得笨拙不自然。

这六条聊天原则源于我与众多异性交往的实战结果，以及我与身边经验丰富的朋友们共同总结的智慧结晶，因此大家可以放心应用。

以下是聊天中必须遵守的六条核心原则：

**第一原则：微信永远不要秒回。**
公式：**收到消息 → 延迟回复**
如果一个异性给你发微信，你立刻秒回，这暗示你特别无所事事、需求感过强。即使你拥有财富，秒回也会让你在对方眼中价值降低，显得像在时刻等待对方。记住，高质量的男士从不秒回信息。

**第二原则：初期聊天避免过多语气助词。**
代码示例：
**避免：“好呀~”、“好哒~”、“在呢~”**
**建议：“好”、“可以”、“明白”**
在刚开始聊天时，使用“呀”、“哒”等过多语气词，会显得你过分讨好、好说话。这可能会让对方觉得你不够稳重，甚至像个油腔滑调的老手。

**第三原则：尽量减少表情包的使用。**
代码示例：
**新手期：禁用表情包**
**熟练后：偶尔使用**
对于经验不足的男士，在聊天中不要添加任何表情包。添加表情包本质上仍是一种讨好行为。在你成为情感高手之前，硬着头皮坚持不使用表情包，效果绝对比你滥用表情包去逗笑对方要好得多。

上一节我们介绍了如何通过减少讨好行为来建立姿态，本节中我们来看看如何维护自己的边界。

**第四原则：敢于表达你的不满。**
如果对方在聊天中冒犯你，例如突然索要财物，你要敢于在微信上斥责对方。例如，你可以反问：“你是个**婊子**吗？” 男女关系并非总是田园诗歌，其中存在博弈。若对方得寸进尺侮辱你，而你连对陌生人都不敢表达不满，将来只会任人拿捏，甚至蒙受财物损失。

**第五原则：敢于承受失去对方的风险。**
有些女性擅长控制男性，若发现无法控制你，可能会将你删除。你必须敢于承担这种“失去”的风险。包括在她行为不端、冒犯你或索要钱财时，你斥责她后可能被拉黑。敢于失去，是一个男人真正拥有魅力的分水岭。

**第六原则：全程不要刻意讨好。**
不要总想着每句话如何让对方高兴或显示自己厉害。要保持松弛感，将对方视为平等的同事、同学来对话。不要刻意打压，也不要刻意讨好，就以平等的身份进行交流。

---

以上六条聊天原则，适用于你成为真正的情感高手之前（这里“海王”指拥有至少10段成功异性交往经验的人）。在达成这个目标前，请严格按此六条原则执行。

当然，如果你已拥有足够经验，这些原则你可以自由调整，因为那时你已懂得收放自如，一切行为皆出于游刃有余的掌控，而非讨好。但在那之前，严格遵循这六条原则，能让你在对方心中加分，显得自信且有张力，与你周围的“舔狗”们拉开差距，从而被对方记住。

所以，请把那些无用的聊天课和话术书统统扔掉。它们只会浪费你的时间，让你在感情中永远处于低位、被打压。按照那些方式学习，你根本无法获得优质的异性关系和感情。

本节课中，我们一起学习了建立自信、维护价值的六大聊天核心原则。记住并运用这些原则，将帮助你在情感道路上获得更好的成长，并快速取得实际成果。
