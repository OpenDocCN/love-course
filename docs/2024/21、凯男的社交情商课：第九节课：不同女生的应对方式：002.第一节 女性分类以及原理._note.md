# 21、凯男的社交情商课：第九节课：不同女生的应对方式：002.第一节 女性分类以及原理

## 概述
在本节课中，我们将学习如何对女性进行分类，并了解不同类型女性的特征和应对策略。

## 女性分类及原理

### 1. 时间性别抉择
- **T测试型（T）**：与多个男性交往，享受被追求的感觉，最终选择一个合适的伴侣。
- **N投资型（N）**：快速锁定一个男性，并投入时间和精力培养他成为理想的伴侣。

### 2. 性观念抉择
- **否定型（D）**：对性较为保守，担心被贴上“荡妇”标签。
- **便捷型（G）**：对性较为开放，享受性带来的快感。

### 3. 身份抉择
- **理想主义型（I）**：追求浪漫的爱情，希望被照顾和保护。
- **现实主义型（R）**：独立自主，注重事业，追求事业成功。
