# 21.煜子显化开悟全程课(1)：10.你会经历的三个阶段 🌟

在本节课中，我们将探讨学习煜子显化开悟课程时，你可能会经历的三个重要阶段。

### 第一阶段：看山是山

**核心概念**：显化能力初步显现，但尚未达到无我状态。

- 在这个阶段，你能够显化一些事情，甚至帮助他人显化较大的事物。
- 你可能还没有达到无我的状态，仍然存在自我意识。
- 你可能已经尝试过观察者状态，即从更高的视角观察事物发生。

**公式**：显化能力 + 自我意识 = 看山是山

### 第二阶段：看山不是山

**核心概念**：通过退两步，从观察者的角度审视事物，但尚未完全融入环境。

- 你开始尝试退两步，将自己视为观察者，观察事物发生。
- 你试图与自己的小我分离，观察其行为和反应。
- 你开始理解，要完全显化事物，你需要超越自我。

**公式**：观察者状态 + 分离小我 = 看山不是山

### 第三阶段：看山还是山

**核心概念**：完全接纳小我，融入整体，达到无分别的状态。

- 你开始接纳小我，不再将其视为对立面。
- 你融入整个环境，理解自己与环境的相互关系。
- 你达到无分别的状态，认识到一切事物都是量子，具有意识。

**公式**：接纳小我 + 融入整体 = 看山还是山

### 总结

本节课中，我们一起学习了煜子显化开悟课程中可能经历的三个阶段：看山是山、看山不是山和看山还是山。通过这些阶段，你将逐渐超越自我，融入整体，达到无分别的状态。希望这些内容能够帮助你更好地理解显化开悟的过程。
