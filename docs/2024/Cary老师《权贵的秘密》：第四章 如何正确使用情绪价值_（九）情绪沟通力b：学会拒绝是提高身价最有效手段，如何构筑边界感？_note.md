# Cary老师《权贵的秘密》：第四章 如何正确使用情绪价值_（九）情绪沟通力b：学会拒绝是提高身价最有效手段，如何构筑边界感？

## 概述
在本节课中，我们将学习如何通过情绪沟通力提高个人价值，并学会如何构筑边界感。

## 情绪价值的重要性
**公式**：真诚且深情的情绪

在运用正面情绪价值时，首先要注意情绪的真实性和深度。真诚且深情的情绪具有强大的感染力，能够打动人心。

## 细节的重要性
**公式**：情绪 + 细节 + 目的

细节体现在你能看到这个人与他人不同的地方，或者是一般人没有发现，但这个人却很在意的地方。

## 案例分析
**案例**：与大佬的沟通

当时，大佬提到他的前妻在创业初期一直陪伴他，但后来事业成功后，前妻变得懒惰且索取情绪价值。这表明大佬渴望一个情绪稳定的女伴。

**沟通策略**：
- 表达对大佬的认可和共鸣。
- 结合情绪、细节和目的，引导大佬达到共鸣。

## 目的的重要性
**公式**：情绪 + 细节 + 目的

在运用情绪价值时，目的非常重要。你可以将目的结合到情绪中，引导对方达到共鸣、付出或为你做事。

## 总结
本节课中，我们学习了如何通过情绪沟通力提高个人价值，并学会如何构筑边界感。在运用情绪价值时，要注意情绪的真实性和深度，关注细节，并明确目的。这三个环节缺一不可。
