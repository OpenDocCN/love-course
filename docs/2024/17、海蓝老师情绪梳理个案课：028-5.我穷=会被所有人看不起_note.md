# 17、海蓝老师情绪梳理个案课：028-5.我穷=会被所有人看不起_note

在本节课中，我们将通过一个具体案例，深入剖析“贫穷”与“被看不起”之间的内在逻辑链条，并学习如何识别和梳理这些深藏于潜意识中的情绪等式。

## 场景回溯：父母争吵的记忆

上一节我们探讨了情绪的来源，本节中我们来看看一个具体的童年场景。案主回忆了中学时期父母因经济问题频繁争吵的画面。

在那个场景中，案主看着父母争吵，内心的感受是：“你俩跟闹着玩似的，整天吵。你俩觉得好玩，你俩就吵吧，我不管了。” 此时的争吵，在案主看来已缺乏真正的冲突感，更像是一种无力的重复。

## 核心情绪反应的产生

当案主意识到父母争吵的根源是家庭经济拮据时，一系列的情绪和信念开始形成。

以下是案主在当时及之后产生的核心感受与行为：

*   **无助与逃避**：面对父母的争吵，案主感到帮不上忙，最终选择“什么也没做”，但内心产生了强烈的“想离开家”的念头。
*   **归因逻辑的形成**：从父母的互动中，案主观察到并内化了一个逻辑：**钱不够 → 夫妻吵架 → 家庭不幸福**。同时，母亲时常抱怨“挣钱少会被看不起”，这让案主也将“穷”与“被人看不起”画上了等号。
*   **信念的自我强化**：虽然案主理性上觉得“我家没那么差”，但持续的争吵环境让感性信念占据上风，即“穷就会吵架，吵架就不幸福”。进而，案主将“不幸福”等同于“人生没有意义”。

## 内在逻辑的驱动与显现

这个隐藏的逻辑等式，深刻地影响了案主成年后的行为模式。

我们看到，这个逻辑链条是：**我穷 = 会被看不起 = 会像父母一样争吵不幸福 = 人生无意义**。为了打破这个可怕的等式，案主发展出的应对策略是：**必须非常努力、忙碌、赚钱，以证明自己“行”**。

于是，从创业开始，案主变得“忙得像陀螺一样根本停不下来”。即使客观上已经获得了财富（“真的有钱了”），但内在的驱动并未停止，转向了“别的事情也停不下来”，因为“要做就做好”的自我要求仍在持续。

## 深入核心：渴望认可的源头

那么，究竟是谁的认可如此重要，以至于驱动了这数十年的忙碌？

以下是案主在引导下进行的内心探索：

*   **核心恐惧**：害怕“不努力就会被人看不起”，尤其恐惧“像爸爸一样，一辈子被妈妈看不起”。
*   **核心渴望**：渴望听到别人的夸奖，希望“让人家都能说我行”。
*   **认可对象**：当被问及最希望得到谁的认可时，案主的答案是“全世界的人”。进一步聚焦到生活周围，主要包括：**老公、妈妈、婆婆、公公、孩子**。
*   **终极指向**：在所有这些人中，案主意识到，自己最在乎的、最希望其说出“你行”的人，是 **妈妈**。

## 课程总结

本节课中，我们一起学习了如何层层深入地梳理一个核心情绪等式。我们从**具体场景**出发，识别了其中产生的**感受与归因**，并清晰地描绘出由此形成的**内在逻辑链条**（如：穷 → 被看不起 → 不幸福）。最后，我们追踪了这个逻辑如何转化为具体的**行为模式**（停不下来的忙碌），并找到了驱动这一切的**核心渴望**（获得母亲/重要他人的认可）。

通过这个过程，我们看见，许多令我们疲惫不堪的“自动行为”背后，往往隐藏着一个需要被看见和审视的旧有情绪公式。识别它，是改变的第一步。
