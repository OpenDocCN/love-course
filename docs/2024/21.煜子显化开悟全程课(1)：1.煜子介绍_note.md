# 21.煜子显化开悟全程课(1)：1.煜子介绍

各位大家好，我是林美玉子。从小我就有一些二阶经验，比如通灵等。19年我开始接触出马出道，最初并不相信，但随着深入了解，我发现灵体真实存在，各有性格和过去。我经历了生死、财富等考验，最终顿悟。

**核心概念**：顿悟、灵体、显化

我从不修行，但开始打坐、念经。突然有一天，我意识到人生意义，进入一种“我不在，但无处不在”的状态。我开始通过显化实现目标，如翻身、创业等。我发现，只要相信我，不需要刻意行动，就能吸引相应的人和事。

**核心概念**：显化、信念、吸引力

我的课程不要求记笔记，重要的是打开心扉，感受频率。我强调，每个人都是自足的，都有自己的特点和特长。课程包括小测验和作业，帮助学员成长。

**核心概念**：频率、自足、成长

我强调，不要泄露天机，因为能量高低会影响显化结果。我分享自己的经历，提醒学员注意能量波动。

**核心概念**：天机、能量、显化

我分享自己在小红书上的经历，提醒学员注意自我价值，避免过度付出。

**核心概念**：自我价值、付出、显化

总结：本节课介绍了煜子的经历和课程理念，强调了顿悟、显化、信念、能量等核心概念。
