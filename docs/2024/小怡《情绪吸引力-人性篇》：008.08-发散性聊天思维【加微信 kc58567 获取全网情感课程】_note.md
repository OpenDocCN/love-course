🎼hello我是小怡。前面的课程，聊天四步法，我们讲解了聊天回复的四个步骤：明确当下关系、读懂女生意图、认清回复方向和斟酌回复措词。很多学员在斟酌回复措词这一步时总会卡住。他们会说，老师，我非常清楚刚刚哪个方向回复了，但我的词汇储备量太匮乏，我想不到好的措辞，经常性的大脑一片空白，我该怎么办呢？今天这节课就解决你这个问题。其实很多时候不是你的知识储备匮乏，而是你不知道如何调动起大脑过去储备的内容。那这节课呢会教给你一个思考的方式，一个路径，那你顺着这个路径迅速的把过去的东西联系起来，找到适合自己的聊天方式。那这节课我们要讲的聊天思维叫发散性聊天思维。

如果你能掌握这个技巧，我可以保证你无论在任何情况下都不会接不上话。我记得啊很多年前在网上有一个段子，一个老实人跟女生说，你们女生为什么总喜欢那些油嘴滑舌的男人啊，而不喜欢我们老实人呢？女人说，因为你们老实人啊不解风情，老实人又说，我们怎么不解风情呢？女人说你晚上来我家我慢慢解释给你听。然后老实人又说不行，我就要你现在给我解释清楚，这个段子啊就是为了说明男女思维的差异。

男人的思维啊是很线性很直接的这种思路很窄，所以很多时候啊能想到的措辞并不多。比如啊你跟女生聊天，女生说我在吃饭呢？男人的线性思维啊，马上就会想到几个回复，吃的什么呀？跟谁吃的，在哪儿吃的以及好吃吗？如果你真的回复这四种内容，那肯定会让人觉得你很无趣。但我们会发现啊，女性的思维就完全不一样，她们的思维呢是网状的，聊天时也都是到处扯来扯去的。比如啊几个闺蜜聊天，其中一个女生说，哎，我觉得易烊千玺好帅呀。然后另一个女生就会说是呀是呀，我也觉得她好帅，她演的电影啊我都喜欢。然后另一个女生又说哎，别提电影了，一提到电影我就来气，上次去电影院啊有啥。一直用脚踢我的座位靠背。现在的人啊都太没素质了。然后几个人就巴拉巴拉巴拉聊一大堆，你看他们的聊天啊完全是想到哪儿就聊到哪儿，没有任何的因果关系。那这种网状的聊天思维呢？我们就叫做发散性聊天思维。

如果你也能学会这种网状的聊天思维，并且啊用这种思维和女生聊天，对方啊很容易觉得跟你聊天很愉快。接下来啊我就教你这种思维的思考路径是什么样子，可以这么说，哪怕你是性格内向，不善言谈的人，也可以通过训练来掌握这种思维，让你聊天的内容啊源源不断。首先你要明白啊，训练发散性聊天思维就是要训练自己的联想力，正常人说话的逻辑啊，或者说我们从小到大受的教育教给大家的都是因果联想。就是因为什么什么啊，所以什么什么，这就是线性聊天思维的思考方式。比如🎼女生说，我最近啊打算去成都旅行，那很多男人的回复是这样的，一个人去吗？哎，成都有什么好玩的吗？你打算去几天呀，那边天气好像挺热的。不用穿太多等等。你会发现呀，这些呢都是根据女生说，我打算去成都旅行。这个原因去思考的现性结果。这就是什么呀？因果联想。但在聊天时啊，除了线性的因果联想之外，还有一种聊天方式叫做相关联想。

就是想到和对方说的这句话相关联的所有事，这也是发散性思维的基础。比如还是女生说，我最近打算去成都旅行，你就可以这么回。哎，我看你小红书收藏了好几篇旅行攻略，应该不是最近打算的吧。你这是虚谋已久了吧。或者这么说，哎，四川那边啊真是个好地方，吃不完的美食，看不完的美女，真想找个四川的媳妇做个倒插门。😊，🎼再或者我最近也打算去成都，据说啊那边的生活状态适合久居，还打算去杭州和厦门也看看，挑个地方养老。有机会一起啊。你还可以说你工作这么忙，还有时间去旅行，还有真是令人羡慕呀。你看这些话都是根据女生说，我打算去成都旅行这句话进行的相关性联想。这个相关性联想的特点就是这些话听起来和对方说的话没有什么特别直接的关系。但是呢又都没有跑题，并且啊拓宽了话题的边界，掌握这种相关性联想的能力啊，会让你的思维瞬间开阔几十上百倍。那如何才能训练自己的相关联想的能力呢？我给你总结了四个方法。

以下是四个方法：

1. **拆字法**：把一句话拆成一个一个的词，甚至一个一个的字，从而拓宽联想的空间。例如，女生说“你怎么突然想起我了”，可以拆成“你”、“突然”、“想起”、“我”四个关键词，然后针对每个关键词进行联想。

2. **上堆下切平移法**：从三个角度出发，去找相关的内容，会给你提供无限的启发。例如，女生说“今年过年我打算自己开车回家”，可以提炼出“车”这个关键词，然后从上堆（交通工具）、下切（轿车分类）、平移（车类交通工具）三个角度进行联想。

3. **联想法**：在你正确使用上述两个方法之后的一个进阶，是拆字和上堆下切平移之后的一个再联想。例如，女生说“还没睡醒赖床呢”，可以提炼出“睡觉”这个关键词，然后进行联想。

4. **感受评价法**：只关注一件事情，那就是你的感受，你看到对方这句话之后的内心感受，而你的回复也很简单，就是直接去描述你当下的感受，与对方呢进行感受上的交流。

这四个方法也是四个思考路径，听懂并不难，但要熟练的应用到聊天中呢，却实需要下一些功夫的，是需要大量练习的。如果你想成为聊天高手，你想掌握这些方法，那你不仅要知道这些思考方式，更是要去练习去实践这些思考方式，每次啊跟别人聊天的过程中，都去想一想，你能拆出什么关键词，能联想出什么答案，平时每一段聊天结束后啊，你回看自己的聊天记录，想一想还能不能用其他的方法给出更好的回答。只要你坚持训练啊，不出三个月，你的聊天功力一定会有一个突飞猛进的变化。好了，以上呢就是这节课的全部内容了。别偷懒啊，抓紧去练，我在下节课等你。🎼
