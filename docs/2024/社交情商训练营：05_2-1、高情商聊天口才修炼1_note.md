# 社交情商训练营：05_2-1 高情商聊天口才修炼1

口才好的人，无论从事何种工作或社交活动，都比口才不佳的人更容易成功。因为口才好的人能够通过语言充分展示自己的优势，让别人更快地了解和信任他们。

那么，他们成功的概率也会大大提高，对吧？许多人羡慕我的口才，纷纷来问我：“校长，你是怎么练的？”其实，我以前并非如此，从小就是一个不爱说话、非常内向的人，做人一直很失败。我泡不到妞，也交不到朋友。后来，我决定改变自己，购买了大量的口才书籍和课程，就是为了锻炼我的情商和口才。经过长时间的练习，我变成了今天这个样子。

兄弟们，口才好的人并非天生，而是可以通过方法提高的。本节课将专门教你如何提高口才。

## 提升口才的第一步：去除错误的说话方式

以下是几种常见的错误说话方式：

1. **声音大小不当**：正常说话的分贝应为5，高于或低于5都会让听众感到不舒服。在图书馆等安静场合要小声说话，在嘈杂场合要大声说话。
2. **说话速度不当**：说话过快会让别人觉得你紧张，听不清楚你说的内容；说话过慢会让对方失去耐心。
3. **过多废话和过度语**：可说可不说的话尽量不说，避免使用过度语，如“嗯”、“啊”等。
4. **说话平淡无感情**：说话时注入感情，根据情绪调整声音大小和节奏。
5. **面无表情**：多使用面部表情，活跃气氛。
6. **手势不当**：说话时配合手势，根据内容调整手势，但不要过于夸张。

## 审视自己，针对性练习

以上6点是口才的基本功，每个点都要做好。如果你不知道自己哪里做得不够好，可以准备一个道具，如布娃娃等，然后对着它进行模拟对话。录下视频，观看后找出不足之处，有针对性地进行练习。

## 总结

本节课中，我们一起学习了如何提高口才。通过去除错误的说话方式，审视自己，针对性练习，相信你的口才会得到很大提升。
