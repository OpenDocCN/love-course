# 瑞恩年会员PRO 瑞恩高级会员：课程48(暂时无法见面的聊天)

大家好，我是Ryan，欢迎来到我的会员课第四十八节课。今天，我将分享一个在疫情期间发生的有趣故事，以及我是如何认识我的长期关系中的女生的。我们相识已经有一段时间了。

**相识时刻**

我是在2021年认识她的。当时，我们都在经历新冠疫情。我向她表达了我的意图，她开玩笑地问：“如果我携带新冠病毒怎么办？”这时，我直接回复：“没事，见色起意，一起隔离了。”（见色起意：一见钟情）

**初次见面**

我们是通过社交软件认识的，几年前就相识了。我告诉她：“见色起意，一起隔离了。”她回复：“哈哈哈，有点过于勇敢了。”（😊）

**邀约成功**

这句话让我成功邀约了她见面。她告诉我：“你不要把人都想那么复杂。任何瞬间的心动都不容易，不要怠慢了他。”然后，我们约定周日见面，她回复：“晚安。”

**聊天技巧**

在聊天中，我们可以通过一句话来升级话题。例如，提到新冠病毒和隔离，我们就可以借此机会表达我们的意图。

**意识与无意识**

有些人能够看到女生的窗口（即机会），而有些人则看不到。这是因为我们的人格底色和内心世界影响了我们的感知。在聊天训练中，我们的目标是帮助学员发展出新的、积极的人格特质，以便在交流中看到更多的机会。

**总结**

本节课中，我们一起学习了如何在疫情期间通过聊天技巧认识心仪的女生。希望这些技巧能够帮助你在未来的交往中更加顺利。我们下节课再见！
