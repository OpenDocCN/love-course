# 课程01：CAC昊哥《进化蓝图训练营》- 037. 第二部分-训练3：话题延续采访 🎤

在本节课中，我们将学习一个名为“话题延续采访”的即兴沟通训练。这个训练的核心是练习如何在不否定对方的前提下，拓展和延续任何话题，这对于提升思维的灵活性和沟通的流畅性至关重要。

## 训练场景设定 🎬

上一节我们介绍了沟通中灵活性的重要性，本节中我们来看看这个具体的训练是如何操作的。

训练需要两个人配合，模拟一个采访场景。其中一人扮演“发明家”，另一人扮演“记者”。“发明家”手握一个物品（例如一瓶巴黎水），并声称这是自己的发明，拥有专利。记者需要对这项“发明”进行采访。

记者的目标是提出一系列天马行空、甚至不合逻辑的问题，试图把发明家“带跑偏”。而发明家必须遵守一条核心规则：**不能否定记者提出的任何问题或假设**。

## 核心规则：Yes, And ✅

这个训练的核心技巧在即兴喜剧中被称为 **`Yes, And`**。

以下是该规则的操作步骤：
1.  **Yes**：首先，完全接纳对方提出的观点或问题。永远不说“不”、“不对”、“不是这样的”。
2.  **And**：然后，在这个被接纳的基础上，添加新的信息、细节或方向，将话题延续下去。

例如，在训练中：
*   记者问：“为什么你是个红绿色盲，却要发明一个红色的瓶子？”
*   发明家不能否定说“这不是红色的”或“我不是色盲”。他需要说：“是的，在我眼里，红色和绿色都是平等的。你不能假定它是一个红色的瓶子。” 这就是在“接受”后进行的“拓展”。

## 训练过程与目的 🎯

在刚才的演示中，记者不断提出荒诞的问题（例如“发明这个是为了割韭菜吗？”），而发明家的目标就是运用 `Yes, And` 原则，始终跟上记者的节奏，并抛出自己的即兴发挥，避免被问住。

这个训练的目的是打破我们在沟通中过度的“逻辑枷锁”和社会过滤器。很多时候，我们总想说出严谨、正确、有逻辑的话，这反而限制了思维的发散和表达的流畅性。

**真正的沟通高手，其内容可能只有20%-30%是严谨的逻辑，其余70%都是基于逻辑的、灵活的即兴发挥和拓展。** 这个训练正是为了锻炼我们那“胡扯”的70%——即快速接受信息并创造性回应的能力。

## 常见问题与反思 🤔

以下是训练中参与者可能遇到的两种情况：

**情况一：作为记者，提不出下一个问题。**
这通常是因为被“必须问出有逻辑的问题”的想法限制住了。你需要练习抛开逻辑，随意联想。例如：“为什么这个东西抠鼻子会特别爽？” 或 “为什么这个瓶子每次跳伞时都能救你一命？” 问题本身不需要合理，目的是激发对方的即兴反应。

**情况二：作为发明家，被记者的问题问懵。**
这说明在“接受”和“拓展”环节遇到了障碍。关键在于快速放弃“纠正对方”的念头，立刻进入“是的，而且…”的思维模式。即使对方的问题非常离谱，也要先接纳，再即兴编造一个理由或故事。

## 总结 📝

本节课我们一起学习了“话题延续采访”训练。我们了解了 `Yes, And` 这一核心沟通原则，并通过角色扮演体验了如何在不否定的前提下延续对话。这个训练旨在帮助我们摆脱过度追求逻辑的束缚，增强思维的敏捷性和即兴表达能力。记住，流畅的沟通往往建立在先接纳、再创造的基础上。
