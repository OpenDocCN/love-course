# 01、CAC昊哥《进化蓝图训练营》：037.第二部分-训练3 话题延续采访_ev_note

在本节课中，我们将学习一个名为“话题延续采访”的即兴沟通训练。这个训练的核心是练习如何在不否定对方的前提下，通过“是的，然后…”（Yes, and…）的技巧，持续拓展和延续任何话题，即使话题本身非常荒诞。

## 训练场景设定 🎬

上一节我们介绍了即兴沟通的重要性，本节中我们来看看这个具体的训练是如何操作的。

训练模拟一个采访场景。其中一人扮演记者，另一人扮演某项发明的发明家。记者手中拿着一个物品，例如一瓶巴黎水。这个物品被假定为发明家的专利发明，但它具体是什么完全由双方即兴决定。

记者需要对发明家进行采访，并提出一系列可能非常离谱的问题。发明家的核心规则是：**不允许否定记者所说的任何话或提出的任何问题**。发明家只能回答“是的”，并在此基础上进行拓展和发挥。

## 核心规则：Yes, and… ✅

这个训练的核心技巧在英文中被称为 **`Yes, and…`**。这是一种在即兴喜剧和表演中常用的方法。它的意思是，无论你的搭档说了什么，你首先要接受它（Yes），然后在此基础上添加新的信息或方向（and…），共同推进情景发展。

以下是该技巧的一个简单示例：
*   **记者**：“为什么你是个红绿色盲，却要发明一个红色的瓶子？”
*   **发明家**：“是的，因为在我眼里，红色和绿色都是平等的。你不能假定它是一个红色的瓶子。”

## 训练的双重目标 🎯

在这个对话过程中，双方各有目标：
*   **记者的目标**是提出天马行空、甚至互相矛盾的问题，试图将发明家“带跑偏”，让其思维混乱。
*   **发明家的目标**是无论问题多么荒诞，都能牢牢接住，并顺着记者的思路给出一个合理的（哪怕是胡扯的）解释，始终保持对话的延续，不被问懵。

## 训练后的反思与目的 🤔

在大家进行一分钟的练习后，我们通常会反思两个问题：

以下是大家在扮演记者时可能遇到的困难：
*   提不出下一个问题。
*   提出的问题过于逻辑化，无法立刻“扯淡”。

以下是大家在扮演发明家时可能遇到的困难：
*   被记者荒诞的问题问懵，不知如何接话。

这个训练的根本目的是什么呢？它旨在突破我们在日常沟通中强大的“社会过滤器”。我们常常认为沟通必须严谨、有逻辑、有意义。但高效的沟通，尤其是创造性的沟通，往往需要大量的、看似“无逻辑”的即兴发挥和联想。

**逻辑内容可能只占20%-30%，其余70%都可以是即兴的延伸和填充。** 如果你在提问时感到困难，很可能是因为你被“必须问出有逻辑的问题”这个想法限制住了。这个训练鼓励你解放思维，允许自己“瞎说”，例如：
*   “为什么这个东西抠鼻子会特别爽？”
*   “为什么脚指甲缝里的泥能够做成一支笔？”
*   “为什么每一次我跳伞时，它都能最后救我一命？”

通过练习接受并拓展任何荒诞的前提，我们能够增强思维的灵活性，在真实沟通中更能游刃有余。

## 总结 📝

本节课中我们一起学习了“话题延续采访”训练。我们了解了如何通过 **`Yes, and…`** 这一核心技巧，在不否定对方的前提下延续对话。我们明确了训练中记者与发明家的不同目标，并认识到这个训练旨在打破我们追求“绝对逻辑”的沟通习惯，解放想象力，让即兴的、创造性的表达成为沟通的助推器。记住，有时最好的逻辑，就是让别人从你的“胡扯”中自动脑补出逻辑。
