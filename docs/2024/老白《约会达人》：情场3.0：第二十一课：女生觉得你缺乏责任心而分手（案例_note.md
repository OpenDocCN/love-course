# 老白《约会达人》情场3.0：第二十一课：女生觉得你缺乏责任心而分手（案例）💔

在本节课中，我们将要学习一个在亲密关系中常见的问题：**因缺乏责任感和言行不一致而导致的分手**。我们将通过一个具体案例，剖析问题根源，并理解责任与承诺在关系中的重要性。

很多男生在追求女生的前期会给出大量承诺。例如，承诺在一起后会每天接女生下班、为女生做饭、承担家务。然而，一旦关系确立，男生性格上的劣势和缺点便暴露出来。

这让女生产生“上当受骗”的感觉。这暴露了男生的一个核心缺点：**缺乏责任感**。其行为模式可以概括为：**言语上什么都可以承诺，行动上却不愿承担任何责任**。

下面我们来看一个具体案例。两人通过亲戚介绍相亲认识，恋爱关系已持续8个月。

相亲的目的通常是寻找合适的结婚对象。虽然男生最初并未明确给出结婚承诺，但相亲的共识是：彼此认可并交往一段时间后，理应开始规划婚姻。

女生28岁，比男生大两岁。交往8个月后，男生仍以“想再了解了解”为由，对结婚犹豫不决。从男性心理分析，这往往是在等待更好的选择，说直白些，就是把现任当成了“备胎”。

女生28岁渴望结婚是正常的。男生不想结婚，本质上是“骑驴找马”。双方因此产生矛盾，女生提出分手是显而易见的结果。

男生以“自己谨慎”和“女生是离异家庭”为借口，实质是嫌弃女生家庭条件，同时又因害怕找不到更好的而不愿放手。

由此可以判断，男生是一个**极度自私的精致利己主义者**。这类人（不分男女）在相亲认识并交往8个月后，绝口不提结婚，正是**缺乏责任感、不愿承担责任**的表现。本案中，男生挽回意愿也不强烈，言语矛盾。

除了上述极端案例，更常见的情况是：男生在追求期描绘美好共同生活的蓝图，例如一起旅行、品尝美食、共度周末。

**代码示例：追求期的承诺 vs 交往后的行为**
```plaintext
追求期承诺 = ["一起旅行", "周末打卡美食", "共同经营生活"]
交往后行为 = ["一直加班", "周末打游戏", "对承诺一拖再拖"]
if 追求期承诺 != 交往后行为:
    女生感到被欺骗，关系出现危机
```

结果在一起后，男生要么总说“等忙完项目”，将旅行计划无限期推迟；要么一到周末就只顾打游戏或睡觉，完全无视当初的承诺。

女生自然会积累怨气，感觉“当初说得好听，现在完全不是那样”，最终导致分手。

**核心问题公式：信任崩塌 = 承诺（言语） - 一致性（行动）**  
当行动远低于承诺时，信任值即为负数，关系破裂。
