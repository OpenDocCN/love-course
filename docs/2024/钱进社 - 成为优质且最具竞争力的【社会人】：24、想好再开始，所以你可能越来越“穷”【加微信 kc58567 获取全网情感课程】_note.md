# 钱进社 - 成为优质且最具竞争力的【社会人】：24、想好再开始，所以你可能越来越“穷”

## 概述
在本节课中，我们将探讨“想好再开始”这一观念的局限性，并分析为何有时候过于谨慎反而可能导致个人发展受限。

## 核心概念
**公式**：成功 = 开始 + 过程

## 第一部分：好的开始与失败
### 1. 好的开始可能是成功的一半
- **解释**：好的开始可以提供良好的起点，但并不意味着成功已经一半。
- **例子**：准备充分但最终失败的情况。

### 2. 不好的开始也可能是成功的一半
- **解释**：即使开始不理想，通过努力也可能取得成功。
- **例子**：从零开始，通过不断努力取得成就的情况。

## 第二部分：成功的阻碍
### 1. 过程是关键
- **解释**：成功的最终原因是过程，而非仅仅开始。
- **例子**：长期努力最终取得成功的情况。

### 2. 主观理想主义
- **解释**：过于理想化的开始可能导致失败。
- **例子**：过度自信导致失败的情况。

## 第三部分：如何定位开始
### 1. 试错与低成本运作
- **公式**：开始 = 试错 + 低成本运作
- **解释**：通过试错和低成本运作来定位开始。

### 2. 技术层面的过度依赖
- **解释**：过度依赖技术层面可能导致忽视其他重要因素。
- **例子**：只关注技术而忽视思维和理念的情况。

## 第四部分：开始的建议
### 1. 工作替代性高的职业
- **建议**：先开始，而不是过度准备。
- **解释**：通过开始来发现自身不足并调整。

### 2. 冷静评判工作
- **建议**：专注于自己能决定的事情。
- **解释**：避免在无法控制的事情上纠结。

### 3. 没有结果的人
- **建议**：确定一个方向并开始行动。
- **解释**：通过行动来克服思维和行为上的匮乏。

### 4. 创业
- **建议**：低成本试错比准备更重要。
- **解释**：避免因过度投入而导致的动荡。

### 5. 纯技术领域的工作
- **建议**：确保技术能力是前沿的。
- **解释**：技术是吃饭的工具，但思维和理念更为重要。

### 6. 对工作不满意的人
- **建议**：开始行动，以发现自身不足。
- **解释**：通过行动来提升自我认知和生存能力。

### 7. 高频快节奏行业
- **建议**：先开始，边开始边调整。
- **解释**：时代不等人，边开始边调整是正确的逻辑。

## 总结
本节课中我们一起学习了“想好再开始”这一观念的局限性，并探讨了如何通过行动来克服思维和行为上的匮乏，从而实现个人发展。希望这节课对你有所启发。
