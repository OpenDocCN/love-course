# 课程03：沟通课 丨 成为说话高手 🗣️
## 第16节：【辅导沟通】如何让别人或自己更好的做事 🧭

在本节课中，我们将要学习一个名为“GROWS”的沟通框架。这个框架能帮助你在辅导他人或梳理自己思路时，进行更有逻辑、更高效的沟通。无论是工作中的上下级对话，还是朋友家人间的建议，这个方法都同样适用。

---

### 什么是“辅导”沟通？

上一节我们介绍了不同沟通场景的分类，本节中我们来看看其中一种常见场景——辅导。

这里指的是广义的辅导，是几乎每个人都会遇到的情形。例如，你部门的员工来问你，最近因为招不到人而烦恼该怎么办。你的朋友来找你说减肥总是失败，该怎么办。甚至你的小孩来问，感觉父母给的任务都做不完。这些其实都称为辅导，也就是基于对方的某一个困境，或者说想实现的一件事情，你想帮助对方想清楚。因此，朋友、同事、家人间都可能有这种场景。

这种有逻辑的沟通，大家可以按照 **GROWS框架** 来进行。

---

### GROWS框架详解

GROWS这5个字母组合起来，是英文单词“成长”。它代表了沟通的五个步骤。

#### 第一步：G - Goal（目标）

第一步是字母G，代表Goal。也就是我们先跟对方沟通，到底想实现什么目标。

**核心操作**：明确沟通的最终目的地。
> 公式：**沟通起点 -> 明确目标(G)**

确定目标对于开展任何讨论都是重要的，这是谈话的第一步。只有明确的方向，才能知道从哪里出发。如果一开始就基于现状，直接陷入细节中，很容易让对方进入不满或焦虑的情绪，甚至会变成对问题的抱怨。

#### 第二步：R - Reality（现状）

了解了目标之后，就可以进入第二步，也就是Reality。这里包括分析目前是什么样的现状，有什么样的条件，什么样的困难，以及现状和目标之间有多大的差距。

**核心操作**：分析现状与目标的差距。
> 公式：**现状(R) vs. 目标(G) = 差距**

#### 第三步：O - Options（方案）

在目标和现状都明确之后，我们来到第三步，Options，也就是选项方案。这个阶段要讨论有哪些事情可以做，有哪些备选方案。

**核心操作**：头脑风暴，列出所有可能的解决方案。
> 代码表示：
> ```
> options = [“方案A”， “方案B”， “方案C”]
> 评估(options)
> ```

#### 第四步：W - Will（意愿）

第四步是Will，代表意愿。即明确对方的意愿，强化其按照沟通确定的方案来做事的决心。

**核心操作**：引导对方总结并确认行动决心。
> 公式：**方案(O) + 强化意愿(W) = 行动承诺**

#### 第五步：S - Support（支持）

第五步是Support，也就是支持。询问对方希望你给他什么样的支持，或者你愿意给对方什么样的支持。

**核心操作**：明确可提供的帮助或所需资源。
> 公式：**行动承诺 + 支持(S) = 可执行的计划**

### 实战案例：如何辅导下属

接下来，我们通过一个具体的例子，来看GROWS框架如何帮助清晰地表达对对方的辅导。

假设小李刚被提升为市场部经理，他面临人员短缺的问题。他的上级周总想对他进行辅导。

1.  **目标 (Goal)**
    小李说：“朱总，我最近都忙不过来了，招人感觉遥遥无期。”
    周总可以问：“关于人员缺口，你期望的具体目标是什么？”
    小李回答：“我希望月底前能招到两个新媒体运营的同事。”
    *（这样，双方首先明确了沟通目标。）*

2.  **现状 (Reality)**
    周总接着问：“关于招聘，你之前做过哪些尝试？”
    小李回答：“我每周都要求HR给我简历，也花时间面试，但一直没找到合适的人。”
    周总可以进一步挖掘：“你觉得没招到合适的人，主要原因是什么？”
    小李回答：“可能这个岗位对公司比较新，HR缺乏相关经验，不知道去哪找这种人。另外，我们的招聘渠道也比较单一。”
    *（通过提问，现状和困难变得越来越清晰。）*

3.  **方案 (Options)**
    周总继续问：“那你觉得有什么其他办法呢？”
    小李思考后回答：“一方面，我应该和HR深入沟通，让他们充分理解我们要招的人。这件事我今天就去做。另一方面，我会发动部门同事和行业圈内的人进行内推，并给一定奖励，这可能比HR广撒网更快。”
    *（这里，小李自己想出了可行的备选方案。）*

4.  **意愿 (Will)**
    周总可以强化意愿：“那么这两个方案，你觉得是当下我们能做到的最优选择吗？”
    小李给予肯定答复。这本身就是一种意愿的强化。

5.  **支持 (Support)**
    最后，周总提供支持：“好，那你先去做。遇到问题随时找我沟通。等你的内推方案出来，可以发给我，我也在圈子里帮你扩散一下。”
    *（明确了领导提供的支持，让计划更可落地。）*

通过这五个步骤，我们帮助对方理清了目标、现状、可行方案，并最终强化其行动意愿，提供了支持。

### 框架的自我应用

GROWS框架不仅适用于为别人提供辅导，也适用于当我们希望理清自己思路，或主动寻求别人帮助的时候。

例如，你业绩不好，需要寻求领导的帮助：

*   **目标 (G)**：“张总，有空吗？我想和您探讨一下如何提升业绩的问题。”（表明目标）
*   **现状 (R)**：“我当前的情况是，新客户转化率正常，但老客户复购率很低。我尝试过回访客户并给老客户续费优惠，仍然没效果。我也不知道哪里做得不到位。”（阐述现状）
*   **方案 (O)**：与领导探讨可能的解决方案。（此处可展开讨论）
*   **意愿 (W)**：“张总，谢谢您的指导。您的建议对我非常有用，我相信我一定不会让您失望的。”（表达决心）
*   **支持 (S)**：如有需要，明确请求支持。（例如，“在执行过程中，如果遇到XX困难，希望能得到您的支持。”）

### 课程总结与回顾

本节课中，我们一起学习了GROWS辅导沟通框架。我们来回顾一下这个模型：

首先，是**目标(G)**。就像使用导航，第一步必须先设定目的地。
其次，是**现状(R)**。清晰分析现状与目标之间的差距。
随后，是**方案(O)**。从出发点到目的地之间，列出所有可选的路线和方案。
接着，是**意愿(W)**。在明确方案后，强化对方或自己到达目的地的决心。
最后，是**支持(S)**。表明你愿意提供的支持，或你需要对方给予的支持。
