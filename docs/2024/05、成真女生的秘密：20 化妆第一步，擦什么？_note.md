# 课程05：成真女生的秘密 🎨 化妆第一步，擦什么？

在本节课中，我们将要学习化妆的第一步——使用粉底液。我们将了解粉底液的核心作用、如何选择以及它在女生心目中的大致定位。

## 概述

粉底液，也称为BB霜，是化妆的第一步。它的主要作用是调整肤色并起到一定的遮瑕效果，让皮肤看起来更白皙光滑。本节将详细介绍粉底液的功能、选择方法和市场概况。

## 粉底液的核心作用

上一节我们介绍了化妆的基本概念，本节中我们来看看粉底液的具体功能。

粉底液用来调整肤色。它有不同的色号。白皙的色号更适合皮肤白的人使用，或者适合想让自己皮肤看起来白的人使用。当然也有不那么白的色号，是给自然肤色的人使用的。非洲人使用的粉底液颜色非常深。亚洲人属于黄皮肤，不适合颜色非常深的粉底液。

粉底液还起到遮瑕的作用。它不是完全能够遮瑕，但起到一定的遮瑕作用。例如毛孔粗大、脸上有痘印或有伤疤，可以用粉底涂得厚一些来遮盖。

简单讲，粉底液就是用来调节肤色，并起到一定的遮瑕作用，让皮肤看起来更加白皙光滑。

## 价格与品牌

了解了粉底液的作用后，我们来看看它的市场价格和常见品牌。

一瓶粉底液的价格大概是从几十块钱到几百块钱。女生的化妆品基本上就在这个价格范围里面。因为不同的牌子，有一些牌子是几十块钱的牌子，它里面的大部分东西都是几十块。有些牌子大部分产品都是几百块，甚至上千块。常规的价格一般是几百块，例如两三百、三四百、五六百。

这些粉底液的牌子五花八门。比较知名或比较贵的，在女生心目中档次较高的牌子，例如雅诗兰黛或者阿玛尼。当然还有一些常用的品牌，价格可能低至几十块钱都有。
