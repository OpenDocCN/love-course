## 教程：如何进行有效的搭讪

### 概述
在本节课中，我们将学习如何进行有效的搭讪，包括开场白、互动游戏以及如何处理拒绝。

### 开场白
**以下是开场白的一些例子：**

- 你好，我叫马克，你打疫苗了吗？
- 你好，我看到你在这里，可以问一下你的名字吗？
- 你好，我注意到你在这里玩得很开心，可以和我一起玩吗？

### 互动游戏
**以下是互动游戏的一些例子：**

- 撕刀剪刀布
- 石头剪刀布
- 猜数字游戏

### 处理拒绝
**以下是处理拒绝的一些方法：**

- 保持礼貌和尊重
- 不要强迫对方
- 转移话题，继续聊天

### 示例对话
**以下是一些示例对话：**

**A:** 你好，我叫马克，你打疫苗了吗？
**B:** 嗯，我打了。
**A:** 那太好了，我也打了。我们可以接个吻吗？
**B:** 不可以，我男朋友在这里。
**A:** 哦，那我们玩个游戏吧，撕刀剪刀布？
**B:** 好啊，我赢了。
**A:** 那我可以得到一个吻吗？
**B:** 嗯，好吧。
