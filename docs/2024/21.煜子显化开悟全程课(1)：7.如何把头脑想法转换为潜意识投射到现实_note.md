# 21.煜子显化开悟全程课(1)：7.如何把头脑想法转换为潜意识投射到现实

🎼大家好，今天我们来探讨如何将头脑中的想法转化为潜意识，进而投射到现实。🎼

### 意识的本质

在中国的主流文化中，意识通常被认为是人的心灵所投射出的外在形象。人的大脑只是心灵的一个反应，所有的思考、念头、想法，包括语言文字能力，都是心灵的反应，而非心灵本身。

### 心流状态

古代人强调“明心见性”，但很多人并未真正进入微观境界，他们只是停留在表象层面。很多人在日常生活中都曾经历过“心流”状态，例如农民在种庄稼时，我也会在写作时进入心流状态，文章会自然而然地涌现。

### 心与大脑的关系

心并非心脏或大脑，而是存在于一片虚无之中，看不见的无形存在。无形的心与有形的心脏是阴阳耦合的，无形的心是心脏生发的动力。

### 统一意志

在人的意识结构中，存在一个思维主体，称为统一意志。它主导着我们的想法和行为，决定我们最终会成为什么样的人。

### 意识的多样性

人的意识并非单一，而是由多个意识组成。有时候，我们会听到不同的声音，让我们做出不同的选择。最终，我们会根据统一意志做出决定。

### 显化的误区

显化并非通过学习或理解来实现，而是通过成为那个人来实现。当你有想要显化某个人的念头时，说明你的心已经知道你没有得到那个人。

### 显化财富的秘诀

显化财富的关键在于不再显化，而是以那个状态生活。当你沉浸在这种状态中，你的财富就会自然而然地到来。

### 成为那个人

从现在开始，你就是那个人，用那个人的状态去生活。不要去思考，而是去感受，去体验。

### 总结

本节课中，我们学习了如何将头脑中的想法转化为潜意识，进而投射到现实。通过成为那个人，我们可以实现我们的愿望。
