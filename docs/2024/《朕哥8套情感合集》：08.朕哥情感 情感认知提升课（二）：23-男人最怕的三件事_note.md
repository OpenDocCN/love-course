# 《朕哥8套情感合集》：08.朕哥情感 情感认知提升课（二）：23-男人最怕的三件事

🎼，男人最怕的三件事，哪三件呢？得不到、失去、不被需要。

### 1. 得不到

- **核心概念**：男人在追求过程中，害怕得不到。
- **公式**：付出 = 得到
- **操作步骤**：
  - 让男人觉得没有得到你全部。
  - 告诉男人，你是他永远得不到的女人。

### 2. 失去

- **核心概念**：男人害怕失去安全感。
- **公式**：不安全感 = 失去
- **操作步骤**：
  - 模拟失去感，让他感到不安全。
  - 不要给他安全感，让他觉得有一定不可得性。

### 3. 不被需要

- **核心概念**：男人希望被女人需要。
- **公式**：被需要 = 存在感
- **操作步骤**：
  - 表现出需要男人的样子。
  - 给男人台阶下，让他帮你做事情。

### 总结

本节课我们学习了男人最怕的三件事：得不到、失去、不被需要。通过理解这些概念，我们可以更好地与男人相处，引导他们为我们付出。
