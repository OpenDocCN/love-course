# 关系博弈（拒绝卑微和低位）：05_第五课，是什么决定了关系的高低位？🔑

在本节课中，我们将要学习亲密关系中决定双方地位高低的核心因素。我们将深入探讨“依赖”这一概念，并分析不同类型的依赖如何直接影响你在关系中是处于卑微低位，还是平等健康的地位。

## 概述：卑微低位与依赖

在感情中，卑微低位是指你总觉得自己不够好、不配得到爱，因此不断迁就、忍让甚至牺牲自我利益来取悦对方。而**依赖**是指你对对方的感情有一种强烈的需要，认为自己的幸福完全取决于对方。

卑微低位和依赖密切相关。当你对对方产生强烈依赖时，就会失去自信、自尊和自我，因害怕失去而变得卑微，从而维持一段不健康的关系。

## 依赖的三种类型

上一节我们介绍了依赖与卑微低位的关系，本节中我们来看看依赖具体分为哪三种类型，以及它们各自的特征。

以下是依赖的三种主要类型：

1.  **强依赖：非他不可**
    强依赖是指你觉得对方是你的全部，不能没有对方。你的心情、行为和决定完全取决于对方的反应。公式可以表示为：**你的幸福 = 对方的反馈**。你忽略自我价值，放弃独立与尊严，将幸福完全寄托于对方，常伴随强烈的占有欲和不安全感。

2.  **弱依赖：锦上添花**
    弱依赖是指你觉得对方是伴侣和支持者，但并非不可或缺。你的核心是为了自己。公式可以表示为：**你的幸福 = 自我实现 + 对方的陪伴**。你重视自我价值、保持独立，同时能给予对方适度的关心与信任，共同维护健康关系。

3.  **无依赖：没他也行**
    无依赖是指你将对方视为附属品或工具，可以随意更换。你的行为完全基于自身欲望。代码逻辑可描述为：`if (对方有用) { 利用 } else { 抛弃 }`。你忽视对方的价值与感受，可能做出欺骗、伤害等行为以满足自己。

## 三种依赖带来的关系效果

理解了依赖的类型后，我们来看看每种类型会实际带来怎样的关系状态和效果。

以下是三种依赖分别导致的关系动态：

*   **强依赖 → 卑微**
    当处于强依赖状态时，你会失去自我，通过不断迁就和牺牲来取悦对方，结果往往是变得卑微。对方可能因此看不起你、欺负你，使你陷入不幸，失去快乐与自信。

*   **弱依赖 → 平等**
    当处于弱依赖状态时，你能保持自我，适当表达和协商。这带来平等的关系，对方会尊重你、爱护你，双方能享受幸福的感情，你也能保有自信、自尊和自由。

*   **无依赖 → 强势**
    当处于无依赖状态时，你会变得自私和强势，随意玩弄或伤害对方。这并非健康的关系模式，会导致对方产生怨恨并试图逃离或反击。

## 核心：迈向理想状态

最理想的关系状态必然是**弱依赖**。它平衡了自我与联结，是平等健康关系的基础。

但如果你发现自己已陷入“强依赖”模式，该怎么办呢？想要改变卑微状态，核心路径就是从**强依赖升级到弱依赖**。

## 从强依赖升级到弱依赖的步骤

从强依赖转变为弱依赖是一个需要刻意练习的过程。以下是四个关键步骤的简要描述：

1.  **认识自我**
    首先，需要清醒地认识自己的强依赖感，并找出其根源，例如是缺乏安全感、自卑还是恐惧。

2.  **增强自我**
    接着，着力增强自信、自尊和自爱。发掘自己的优点、价值和目标，培养个人能力，拓展社交圈，多做让自己快乐和充实的事。

3.  **减少依赖**
    然后，有意识地减少对对方的心理依赖。客观看待对方的缺点，不将其神化，不把对方的反馈等同于自己的幸福或责任。

4.  **重建关系**
    最后，基于新的自我，与对方建立健康的互动模式。欣赏对方的优点，尊重其独立性，通过沟通、分享和支持来维系感情，学会合作与协调。

当然，这只是一个简单的步骤框架。如何具体实践每一步，我们将在后续的五节课程中详细展开讲解。

## 总结

本节课中，我们一起学习了决定关系高低位的核心因素是**依赖程度**。我们分析了强依赖、弱依赖和无依赖三种类型及其分别导致的卑微、平等和强势关系状态。我们明确了**弱依赖**是构建平等健康关系的理想状态，并初步了解了从强依赖升级到弱依赖的四个步骤。改变依赖模式是提升关系地位、告别卑微的关键。
