# 关系博弈（拒绝卑微和低位）：05_第五，是什么决定了关系的高低位？

## 概述
在本节课中，我们将探讨在亲密关系中，决定关系高低位的核心因素。

## 关键因素
在亲密关系中，决定高低位的核心因素包括卑微低位和依赖。

### 卑微低位
- 感觉自己不够好，不配得到对方的爱。
- 不断迁就忍让，甚至牺牲自己的利益来取悦对方。
- 对方态度和行为完全取决于自己的幸福。

### 依赖
- 对对方的感情有强烈的需要。
- 没有对方，就无法生存。
- 幸福完全取决于对方。

## 依赖的三种类型
以下是三种依赖类型及其特点：

### 强依赖
- 视对方为氧气，唯一。
- 一切围绕对方转。
- 对方反应决定自己的心情和行为。
- 忽略自己的价值、兴趣、目标。
- 对方有强烈的占有欲、控制欲和不安全感。

### 弱依赖
- 视对方为伴侣、朋友、支持。
- 可以有对方，也可以没有对方。
- 一切为了自己。
- 基于自己的想法决定心情和行为。
- 重视自己的价值、兴趣、目标。
- 保持独立自由尊严。
- 对方有适度的关心、信任和安全感。

### 无依赖
- 视对方为附属品、玩具、工具。
- 可以有对方，也可以随便换对方。
- 一切为了玩。
- 基于自己的欲望决定心情和行为。
- 忽视对方的价值、兴趣、目标。
- 剥夺对方的独立、自由和尊严。
- 对方有强烈的轻视、漠视和不尊重。

## 三种依赖带来的效果
以下是三种依赖类型带来的效果：

### 强依赖
- 失去自我，变得卑微。
- 陷入不幸的感情，失去快乐、自信、自尊和自由。

### 弱依赖
- 保持自我，变得平等。
- 享受幸福的感情，拥有快乐、自信、自尊和自由。

### 无依赖
- 变得自私，变得强势高傲。

## 如何从强依赖升级到弱依赖
以下是升级步骤：

1. 认识自己的强依赖感，找出原因。
2. 增强自信、自尊和自爱。
3. 减少对对方的依赖，学会放手。
4. 建立健康的感情，尊重对方。

## 总结
本节课中，我们学习了在亲密关系中，决定关系高低位的核心因素以及如何从强依赖升级到弱依赖。希望这些知识能帮助你建立更健康、平等的关系。
