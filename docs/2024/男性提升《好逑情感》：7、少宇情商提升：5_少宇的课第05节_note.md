# 男性提升《好逑情感》：7、少宇情商提升：5_少宇的课第05节

大家好，今天我们来探讨如何通过聊天带动情绪。众所周知，聊天是增进感情的一种方式。但为何有人聊天后感情升温，而有人却聊死了？原因在于聊天内容是否充满情感。

**原因在于聊天内容过于单一**。第一种聊天是情感交流，第二种则是情绪消耗。以下举例说明：

假设你有一个女同事，因工作原因经常面对面交流。她性格刻板，总是严肃，很少笑容，打扮保守，给人一种拒人于千里之外的感觉。每次交流都是工作内容，如数据、任务、项目进度等。尽管交流频繁，但你们之间不会变得亲密。因为她走后，你不会感到不舍，因为你觉得她走与你的工作无关。

相反，假设她走后来了一个新同事，她性格开朗，打扮精致，给人一种舒适的感觉。你们在工作交流之余，也会聊一些个人事情、明星八卦或小故事。你也会分享自己的故事，渐渐了解她，她也了解你。你会在她辞职后感到失落，甚至想她。你们之间产生了亲密感。

**原因在于第二个女生让你产生了情绪波动，这就是情感交流**。要让关系更亲密，就需要进行情感交流。具体方法如下：

1. **状态**：聊天时，自己的状态要好。例如，第二个女生在打扮和性格上比第一个女生更舒适，让人感到热情、亲切。
2. **感受**：多与女生交流感受，而不是干巴巴地讲逻辑或事实。例如，女生问你在干嘛，你可以回答“我才下班，现在好累啊”，而不是“我才下班”。
3. **关联**：与女生聊天时，不要只做听众，要敢于表现自己，并与女生产生关联。例如，女生说喜欢看小说，你也可以说“巧了，我也喜欢看小说，我也喜欢看她的小说”。

**总结**：

情感交流是建立感情基础的关键。与女生聊天时，要从状态、感受、关联三个方面入手。状态要好，感受要丰富，关联要紧密。只要用心体会，多加练习，你也可以掌握这门武功秘籍。
