# 清风老师合集：06撒娇2.0方法篇13期：066.第八课：需求式撒娇，让他为你疯狂付出

在本节课中，我们将学习如何通过需求式撒娇，让男朋友或老公更加爱你，并愿意为你付出。

## 基本关键点

首先，我们需要了解心动循环的三个步骤：展现兴趣（可得性展示）、激发行为、观察后撤。这三个步骤形成一个循环，如果对方没有做出主动行为，则重复循环。如果对方做出了积极回应，则进入下一个步骤，即上瘾模型。

### 上瘾模型

1. 触发：激发对方的行为。
2. 行动：对方做出回应。
3. 投入：让对方在投入过程中感到愉悦。

## 提出需求

在提出需求时，要具体明确，单次可完成，并给予及时反馈和奖励。

### 互动案例

以下是一个博主撩汉的互动过程，我们将结合视频素材来学习如何提出需求。

1. **开场**：女生来到台球桌，打招呼。
2. **暗示需求**：女生提出需求，但男生未立即回应。
3. **打破僵局**：女生用“我丝袜怎么破了？”的方式吸引男生注意力。
4. **肯定与邀请**：男生邀请女生一起玩。
5. **眼神交流**：女生通过眼神交流展示好感。
6. **互动与奖励**：女生提出学习动作，男生教学并给予肯定。
7. **挑战与升级**：女生提出挑战，男生接受并升级奖励。
8. **完成与后撤**：女生完成挑战，男生带她回家。

## 总结

通过展示激发、观察后撤和上瘾模型，我们可以引导对方不断投入，从而让他为你付出更多。在下一节课中，我们将学习如何化解冷暴力。
