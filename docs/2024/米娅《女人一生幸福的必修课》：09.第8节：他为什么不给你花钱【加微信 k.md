## 如何衡量“愿意为你花钱”

当然，有些愿意给你花钱的男人，也未必是真的爱你。关键要看对方给你花的钱，占他的经济比重有多少。

例如：
*   **代码示例：诚意比计算**
    ```python
    # 假设有两个追求者A和B
    追求者A_总资产 = 500_000_000  # 5亿
    追求者A_为你花费 = 200_000     # 20万的车

    追求者B_月收入 = 10_000       # 月入1万
    追求者B_存款 = 250_000        # 存了几年攒了25万
    追求者B_为你花费 = 200_000    # 用存款买了20万的车

    # 计算花费占比
    占比_A = 追求者A_为你花费 / 追求者A_总资产  # 占比极小
    占比_B = 追求者B_为你花费 / 追求者B_存款    # 占比极大

    # 显然，B的诚意远高于A
    ```

比如一个资产好几个亿的老板给你买了一辆二十几万的车，那不能说明这个男人爱你，只能说明你在他的眼里，你值这二十几万的车，而不是200万的车，也不是他的核心资产，也不是他的公司股份。但是如果一个月入一万的男人，用自己存了几年的工资，给你买了一辆二十几万的车。那这两个男人相比较的话，一定是后者对你更有诚意。
