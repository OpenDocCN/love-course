你是否经常感到特别疲惫，不知道自己真正想要什么？有时你可能会短暂地充满动力，但很快又感到无力坚持。

## 什么是内耗？🤔

上一段我们提到了疲惫感，这种情况的核心原因通常是“内耗”。如果你都不爱自己，谁会真心爱你？内耗的出现主要有两个原因。

以下是内耗产生的两种心理模式：

1.  **高敏感加低自尊导致自卑心理**：过分在意他人看法，容易将责任和错误归咎于自己。例如，身边人情绪稍有波动，你就会怀疑是否是自己做错了什么。过度思虑加上较低的自我评价，导致持续的自我怀疑和否定。
2.  **高要求加低自尊导致自我评价失衡**：对自己设定过高且不切实际的目标，背负着父母期待或同伴压力。这导致个人真实能力与期望目标严重失衡，形成“越想做到，越做不到”的困境。公式可表示为：**心理压力 ≈ (期望值 - 实际能力) × 自我苛责度**

## 内耗的危害有哪些？⚠️

了解了内耗的成因后，我们来看看它的具体危害。内耗主要带来两方面伤害。

以下是内耗的两个主要危害：

1.  **导致心理上的疲惫与压力**：因为你一直在与自己较劲。当受到指责或目标未完成时，你会自责甚至开始讨厌自己。“必须优秀”的信念越强，你就越排斥那个可能“做错事”的自己。
2.  **伤害身边的人**：越是内耗，你的情绪就越不稳定。身边的人也会感到压力，生怕一句话就刺激到你。

## 如何停止内耗？🛑

认识到内耗的危害后，我们迫切需要找到停止它的方法。本节将提供几个实用的策略。

以下是停止内耗的四个关键步骤：

1.  **接纳与肯定自己**：喜欢优秀的自己，也愉快地接纳不优秀的自己。顺其自然，只与自己比较，不参与过度竞争。你的人生不是活给别人看的，他人的评价不应过度影响你的生活。
2.  **正视自身能力**：我们必须承认，在某些方面确实不如他人优秀。因此，不必设定高不可攀的目标。多看看自己的长处，不要只追求完美，因为没有人生是完美的。
3.  **关注过程而非执着于结果**：将大任务分解为小目标。每当完成一个小目标，你的自我认可度就会提升，从而逐步改善内耗。代码逻辑类似：
    ```python
    # 将大任务分解
    big_task = ["终极目标"]
    small_goals = ["步骤1", "步骤2", "步骤3"]
    for goal in small_goals:
        complete(goal)  # 每完成一个小目标
        self_approval += 1  # 自我认可度提升
    ```
4.  **爱全部的自己**：你可以继续追求优秀，但不要再排斥那个不优秀的自己。学会接受并改善不足，而不是将其拒之门外。爱全部的自己，才能促使真正的成长与进步。

本节课中，我们一起学习了“内耗”的概念。它源于高敏感低自尊或高要求低自尊的心理模式，会导致身心疲惫并影响人际关系。停止内耗的核心在于：**接纳自己、正视能力、关注过程、爱全部的自己**。说白了，就是实现自我认同。学会这些，你便能更轻松地前行。
