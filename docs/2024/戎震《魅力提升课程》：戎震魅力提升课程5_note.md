# 戎震《魅力提升课程》：戎震魅力提升课程5_note

在本节课中，我们将要学习如何通过“乐善好施”的行为来构建和维系你的社会资源网络，并理解这在个人魅力与社会能量提升中的核心作用。

很多人曾问我，平时把钱花在哪里。我并非一个富有的人，只是无需再为金钱辛苦工作。财富自由源于花费较少。有些钱我认为一辈子无需花费，例如花在豪车、奢侈品或特别大的房子上。

这些开销没有必要。我的日常开销很少。很多时候，我甚至觉得代步工具都没有存在的必要。如果你住在市中心，开车停车的费用代价可能更大，这无非是面子问题。

面子是人世间最无用的东西，但你不能不给别人面子。因此，我的正常消费通常是迎来送往的消费。作为一个生意人，自己无需消费得特别到位，但一定要让朋友感受到你的诚意，你是一个乐善好施的人。

本节我们就要掌握乐善好施的问题。在与别人互动的过程中，你必须让人有一种感觉：与你的互动是良性的、有利可图的、愉快的。你要么提供情绪价值，要么提供金钱价值。

在社会中，我们都有几张“牌”。有人牌抓得多一点，父母给予的多；有人抓得少一点，可能出生比较苦。但无论如何，当你混到30岁至40岁的中后期，大家逐渐确立社会地位时，就开始了资源交换的过程。

在这个互相帮忙的交换过程中，你通常需要一些朋友，例如几个医生朋友、几个律师朋友，甚至宠物医生朋友。对于一个资源正常的人来说，这些通常都有。

我们衡量一个人的能量，特别是横向能量，在于他能够调动和借用多少社会资源。这是成人社会的分水岭。如果你一辈子只做学问，两耳不闻窗外事，一心只读圣贤书，那么当你的孩子生病发高烧时，你可能连个床位都找不到，号都挂不上。这就是不社交的坏处。

因此，提升自己的魅力、社交属性和能力，有时与你置换社交资源、利用和调动社会资源有很大关系。你的后辈、后代一定能受到这种帮助。如果你孑然一身，不留后代，那你可以随性而活。

但我们大部分人都在社会中担任某种角色，是社会的成员或家庭的成员。无论你的角色是什么，决定你一些生死的东西都可能与你调动社会资源的能力有关。

所以，日常的迎来送往、与朋友的社交花费，无论是记住别人的生日、在别人生病时去探望、处理红白喜事，还是在很多时候维护重要关系，都至关重要。

如果你想提升与医院的关系，平时溜须拍马、说好话、共同参与事情，可能没有太好的机会。但如果这个人将来你用得着，你一定要对他“礼先下士”。

例如，你认识十几个律师，你只需与最有名的一两个好好把握关系即可，其他人则无所谓。但你需要考虑的是看清对方的性格。

因为有些人，无论你怎样“猪头三进庙门”般对他好，他的好感度提升很慢，他可能看不起你。但有的人非常欣赏你，你对他稍微好一点，他的回报就非常大，可能你给他一分，他还你三分。

这就像做生意一样，人的友好度是慢慢增长的。但关键时刻别人帮不帮你，是否向你展现他的社会资源、财力、影响力或政治能力，都取决于你平时是否是一个乐善好施的人、一个善交朋友的人、一个知恩图报的人。

这些都是在大家长期相互观察中形成的。不要因为给别人一点小恩小惠，就索求回报。平时有事情时，你要主动帮助人，这也是好事。特别是在关键时刻，天下没有人会斩断奉上礼物的双手。

很多时候，求人办事能否成功，关键看你平时如何积累。平时就要做一个乐善好施的人。这就是我们最后一课想教给大家的。

---

本节课中，我们一起学习了“乐善好施”在构建社会资源网络中的核心价值。关键在于通过日常的诚意互动、提供价值（情绪或金钱）以及长期的关系维护，来积累和调动社会资源，从而提升个人在社会中的能量与魅力。记住，资源的交换与互助是成人社会运行的重要法则。
