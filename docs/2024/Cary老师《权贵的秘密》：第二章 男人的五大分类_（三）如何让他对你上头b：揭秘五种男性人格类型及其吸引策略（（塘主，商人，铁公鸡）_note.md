# Cary老师《权贵的秘密》：第二章 男人的五大分类_（三）如何让他对你上头b：揭秘五种男性人格类型及其吸引策略（塘主，商人，铁公鸡）

## 概述
在本节课中，我们将学习如何识别和应对五种不同类型的男性人格，以及如何运用策略吸引他们。

## 堂主型男人
堂主型男人的付出比财神和国王更为粗暴且没有道理。他们通常通过付出认识你或打开话题。例如，有人直接为女性买了一个价值4万元的包，或有人直接转账5万元。然而，这些男性有一个共性，即快速付出后，会逐渐对你冷淡，甚至不再搭理你。但他们会定期定额地付出。有些女性可能会误以为想要让他加大付出，就开始反扑或找别人应要，但最终只会导致对方下头或删除你。

**核心概念**：堂主型男人快速付出后，会逐渐冷淡，但会定期定额地付出。

## 商人型男人
商人型男人喜欢展示自己的财富和成功，希望通过这种方式吸引女性。他们通常不会轻易付出，除非你能提供相应的价值。例如，他们可能会以借米的方式付出，或者暗示你关系不够亲密，不适合帮忙。

**核心概念**：商人型男人以利益交换为前提，不会轻易付出。

## 铁公鸡型男人
铁公鸡型男人以自我为中心，不会轻易付出。即使你用各种方法引导他们付出，效果也微乎其微。他们甚至可能希望你为他们付出。

**核心概念**：铁公鸡型男人以自我为中心，不会轻易付出。
