# 15、关键认知：通识哲学【50条关键认知课】——线上录播课：N0.35节35、人脉

人脉在现实中常被夸大。我们常认为没有做成事是因为缺乏人脉，不认识关键人物。在关系社会中，人脉被视为一种奢侈品。但人脉究竟是什么？

**人脉是一种结果，是之前做事的结果。**

例如，与某人合作完成一项事业，并从中获利，这就是你的人脉。单纯认识某人，并不能算作人脉。

**年轻人初入社会，往往没有人脉。**

因为年轻人没有做过事，没有赚到钱，没有成功撮合合作或落地项目，没有开公司。

**二代虽然有一定人脉，但也是基于与父亲合作的结果。**

**个人能力决定人脉。**

能力强的人，在时间的沉淀中，一定能取得成果，从而积累人脉。

**通过人脉和时间可以判断一个人的能力。**

能力一般的人，即使进入社会很长时间，人脉也不会很高。

**人脉可以和能力相互证明。**

**粉丝量是典型的结果性属性，类似于人脉。**

**社会越高效，越不需要过多的人脉。**

**人脉是广义的，包括利益交换、帮助他人等。**

**构建以结果呈现的人脉，而非盲目添加微信。**

**总结：本节课中我们一起学习了人脉的概念、重要性以及如何构建人脉。**
