# 林老头《系统认知》：015.5、短期关系的实践与要素_ev

## 概述

在本节课中，我们将探讨短期关系的实践与要素，包括心态建设、试探与强筛选、合理化、自我价值与期待、展示面、约会能力以及爱情感觉的营造。

## 心态建设

**公式**：心态 = 经历 + 失败/痛苦

在进入短期关系之前，必须建立强大的心态。这种心态来源于经历，尤其是失败和痛苦的经历。只有经历过失败和痛苦，才能真正理解并享受短期关系带来的快乐。

## 试探与强筛选

**公式**：试探 = 知己知彼 + 安全保障

试探是短期关系中的关键步骤，它包括了解自己和对方，并确保自己的安全。试探可以分为两类：

* **直白试探**：通过直接提问或评论来了解对方的想法和感受。
* **间接试探**：通过提问或评论来引导对方表达自己的想法和感受。

## 合理化

**公式**：合理化 = 逻辑强 + 心理变化

合理化能力对于影响对方至关重要。通过逻辑上的说服和心理上的变化，让对方接受你的观点和行为。

## 自我价值与期待

**公式**：自我价值 = 给予价值 + 选择 + 体验

展示自我价值是建立短期关系的关键。通过给予对方价值、提供选择和提供体验，让对方感受到你的独特之处。

## 展示面

**公式**：展示面 = 照片 + 约会能力

展示面包括你的照片和约会能力。照片要展示你的真实形象，约会能力要能够提供愉快的约会体验。

## 约会能力

**公式**：约会能力 = 临场能力 + 实践

约会能力需要通过实际锻炼来提升。在约会过程中，要注重细节，创造特殊性，并让对方感受到心动和得到的感觉。

## 爱情感觉的营造

**公式**：爱情感觉 = 心动 + 安全 + 得到

爱情感觉可以通过营造心动、安全和得到的感觉来实现。心动可以通过情绪互动和感性思维来营造，安全可以通过保证对方的安全感来实现，得到可以通过满足对方的需求来实现。

## 总结

本节课中，我们学习了短期关系的实践与要素，包括心态建设、试探与强筛选、合理化、自我价值与期待、展示面、约会能力以及爱情感觉的营造。通过掌握这些要素，我们可以更好地理解和应对短期关系。
