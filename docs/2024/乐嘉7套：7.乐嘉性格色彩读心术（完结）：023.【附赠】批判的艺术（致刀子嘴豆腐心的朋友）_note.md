# 乐嘉性格色彩课程 7.7：批判的艺术 🗣️➡️❤️

在本节课中，我们将要学习一个在人际沟通中至关重要却极易出错的技巧——批判的艺术。我们将通过具体案例，分析为何在错误时机、以错误方式进行的“批判”会适得其反，并探讨如何将批判转化为有效的关怀与建议。

## 概述：批判的双刃剑

上一节我们介绍了性格色彩在沟通中的应用，本节中我们来看看“批判”这一具体行为。批判本身并非全无价值，但若使用不当，尤其在他人脆弱时，它非但不能帮助对方，反而会带来伤害，让关系陷入僵局。

## 核心误区：当“为你好”变成“伤害你”

许多人习惯于以教育者的姿态出现，对他人的行为进行批判。他们坚信“我是对的”，并希望通过批判来改造对方。这种沟通模式的核心问题在于：**它忽略了对方当下的情感需求，只专注于证明自己的正确性**。

以下是两个典型的错误批判案例：

*   **案例一（事后指责）**：“我早就跟你讲过，这个节目不适合你，你不肯听，现在你看看。”
*   **案例二（质疑与埋怨）**：“你怎么搞的嘛？你不是平时一直都在锻炼吗？肌肉那么厉害，怎么还会受伤呢？”

这种说话方式，**表面上是在为我好，但是我觉得有一种情况，他们也在显示他们自己的英明和正确**。对于听者而言，尤其在痛苦时，这更像是一种幸灾乐祸或二次伤害。

## 核心原则：病人首先需要的是关怀，而非教育

当一个人处于高度病痛或情绪低谷时，其心理需求会发生根本变化。此时，任何道理和批判都是无效甚至有害的。

**请你记住，当一个人在高度病痛的时候，他们首先需要的不是教育，而是关怀。**

原因在于：
1.  你未必了解事情的全部真相，贸然批判有失公允。
2.  即便错误是对方造成的，**没人比他更清楚自己的错误导致的代价有多大**。此时的痛苦已是最大惩罚，无需再追加“教育”。

正确的做法是分清场合：
*   **对待小错**：可以严厉提醒，目的是让人记住教训。`小错往死里骂`，以避免再犯。
*   **对待大错或重病**：**一旦有大错，就别再骂了。因为骂人是伐天回力，解决不了任何问题。** 此时，**只有康复才是头等大事**。

## 案例分析：好心为何不得好报

让我们通过一个学员“笑笑”的真实经历，来具体感受错误批判带来的后果。

笑笑在闺蜜阿晶流产手术后，承担了所有实际照顾工作（接送、买补品、炖鸡汤），但她忽略了情感支持。

以下是她的沟通方式：
> “你看看这本书吧，懂懂自己看清别人，以后可以少上那些坏男人的当。我跟你说过多少遍，那个男人根本就不是什么好东西，你为啥总是不肯听，现在倒好，你出了事儿自己受苦，那小子一眨眼就溜掉了，连鬼影子都没有，你说说你你就不能长点记性嘛？”

尽管笑笑付出了巨大辛劳，但阿晶并不领情，另一位朋友“丹丹”仅通过电话慰问和一次探望，反而更受欢迎。笑笑感到委屈，直到学习了课程才恍然大悟：**在病人生病的时候，或许他有的时候要的不是你无微不至的关照，有的时候心灵的安慰真的很重要。**

这个案例揭示了一个关键点：**病人希望从你这里得到的是温暖和信心**。
*   **温暖** = 关怀、慰问、快乐
*   **信心** = 鼓励

批判式的对话，**既没有温暖，更不要讲信心了，只会雪上加霜**。

## 高级技巧：如何有效地表达关切与建议

如果你确实认为自己的建议无比重要，必须遵循以下两大原则：

1.  **时机原则**：`等他病好了再教育`。
2.  **方法原则**：不要单独使用批判，**配合本书当中的其他招数一起使用**。

以下是两种更高级的表达方式：

*   **方式一：隐晦的提醒（但可能仍无效）**
    > “挑战身体极限这种事大可远离，留给年轻人吧。挑战大脑极限的事儿，你以后倒是可以多做。”
    *   **分析**：这种方式虽然含蓄，但目的仍是表达不认同。如果对方不接纳你的价值观，反而会**激发起来我不信邪的欲望**。

*   **方式二：从对方软肋切入，充满温情**
    > “老乐子啊……你对自己好点吧，好好有质量的健康的活着，不为别的，就为了这个好不容易奔你而来这个世间的女儿。”
    *   **分析**：这种方式从对方最在意的人或事（如女儿）出发，充满理解和关爱。**他讲的温情说的在理，也是在批评我，但我照单全收，而且我特别乐意。**

## 总结与反思

本节课中我们一起学习了“批判的艺术”。我们认识到，在他人脆弱时，单纯的批判是无效且有害的沟通方式。**教育法，在病痛悲哀的时候，不适合单独使用，因为它很容易变成批判。**

最后，请经常使用这招的人进行反思：
*   你是否总试图证明自己的英明？
*   你是否总喜欢指点他人的江山？

请记住沟通的黄金法则：**多点温情，多点理解。你给别人建议的时候慢半拍。说话的时候多点发自肺腑的关爱。** 只有这样，你的“为你好”才能真正抵达对方心里，转化为积极的力量。
