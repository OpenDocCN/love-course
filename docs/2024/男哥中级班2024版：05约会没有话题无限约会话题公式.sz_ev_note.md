# 男哥中级班2024版：05约会没有话题无限约会话题公式

## 概述
在本节课中，我们将学习如何通过“无限约会话题公式”来打破约会中的尴尬局面，让对话充满趣味和深度。

## 核心概念
**公式**：第三方 - 我自己 - 经历 - 三观

## 注意事项
1. **正确的时间，做正确的行为**：在合适的场景和时机使用该公式。
2. **避免刻意**：不要为了使用公式而刻意说话，要自然流畅。
3. **注重画面感**：描述经历时要生动形象，让对方产生画面感。
4. **反差感**：通过反差来让对方记住你想说的话。

## 聊天公式组成
1. **第三方**：描述约会环境中的景物或事物。
2. **我自己**：将第三方与自己的经历或想法联系起来。
3. **经历**：描述自己的经历，并注重画面感。
4. **三观**：表达自己的观点或价值观。

## 话术举例
### 例子一：关于爱情
- **第三方**：卡布奇诺
- **我自己**：让我想起了大学时的感情
- **经历**：我第一次恋爱是在大学，当时很害羞，但很努力地追求她
- **三观**：在感情中要有责任感

### 例子二：关于孝顺
- **第三方**：卡布奇诺
- **我自己**：让我想起了勤工俭学的经历
- **经历**：我第一份工作是在咖啡店，当时收入不多，但很努力
- **三观**：孝顺父母，珍惜亲情

### 例子三：关于宠物
- **第三方**：小狗
- **我自己**：让我想起了救助小狗的经历
- **经历**：我在路上看到一只生病的小狗，就把它送到了医院
- **三观**：关爱动物，乐于助人

本节课中，我们学习了如何运用“无限约会话题公式”来打破约会中的尴尬局面，让对话充满趣味和深度。通过练习和运用这个公式，你可以成为一个更有魅力、更有生活的人。
