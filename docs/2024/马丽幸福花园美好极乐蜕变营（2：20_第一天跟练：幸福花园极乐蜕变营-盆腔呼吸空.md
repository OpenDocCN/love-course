## 今日训练概述与热身

上一节我们介绍了课程的整体结构和准备工作，本节中我们来看看今天的具体训练内容，并进行必要的热身。

第一天的训练包含三个核心动作，总训练量约为120次。这些动作旨在帮助你初步感知和激活盆底肌群。

以下是今日的三个训练动作：
1.  **收紧与放松**：学习找到并控制目标肌肉进行基础的收缩与放松。
2.  **分级收紧**：将“收紧”这个动作放慢，分为几个力度等级，帮助你更精细地感知肌肉收缩的程度。
3.  **收紧并屏息**：在收到最紧的状态下，尝试屏住呼吸并保持肌肉收缩，从3秒逐步挑战到10秒。

在开始正式训练前，进行全身和局部热身非常重要，这能让身体做好准备，避免受伤。

以下是热身步骤：
1.  **全身热身**：微弯膝盖，上下轻轻抖动身体，同时甩动胳膊，让全身的血液流动起来。
2.  **局部热身（蝴蝶式）**：坐下，将手放在膝盖上，模仿蝴蝶扇动翅膀的感觉，让双腿上下开合，找到放松的节奏。
3.  **骨盆画圈**：保持坐姿，让坐骨与地面接触的部分，分别向前后、左右方向移动，最后进行顺时针和逆时针的绕圈。

## 动作一：基础收紧与放松

热身结束后，我们正式进入第一个动作的练习。这个动作的关键是精准定位我们要训练的肌肉。

首先，请将你的手放在耻骨位置，指尖大约在阴道入口的上方。这个区域就是我们将要关注的“幸福花园”。闭上眼睛，在内心与这个部位打个招呼，建立连接，这有助于你在练习时更专注。

定位目标肌肉有一个简单的方法：想象自己是一位正在站着小便的男性。当你想要中断尿流时，下意识收缩的那块肌肉，就是我们要找的盆底肌（尤其是靠近阴道口的前半部分）。女性的阴道长度大约为9厘米，我们初始练习主要针对前端约3-4厘米的肌肉。

在练习时，你可以观察老师的手部动作作为节奏提示。如果你觉得跟随手部一起收缩放松能帮助你更好地找到感觉，也可以这样做。

我们开始第一个动作的练习，共进行4组，每组10次。请跟随指令进行。

**动作指令（重复10次为一组）：**
```
呼气 -> 收紧
吸气 -> 放松
```

第一组动作完成，请休息10秒钟，可以抖抖手、抖抖脚，让身体保持松弛状态。

现在开始第二组练习，请继续跟随指令。

第二组训练完成。调整呼吸，准备第三组练习。

这里解释一下呼吸配合的原理：在发力时呼气是健身中的常见技巧，它能帮助我们更好地核心发力。同时，悠长的呼气也为后续需要屏息的动作打下基础，让你能更持久地保持收缩状态。

第三组训练开始。

第三组训练完成。请短暂放松。

第四组，也是最后一个组的训练，现在开始。
