# 21.煜子显化开悟全程课(1)：6.如何收回灵体-让不想要的意识消失

🎼接下来呢我会讲如何收回你不要的灵体，或者说如何让你不想要的意识消失。其实呢如果你就是没有这个意识，你又何需去释放呢？

## 1. 灵体与意识
**核心概念**：灵体是附在人体上的意识体，它们可能带来负面影响。

**公式**：灵体 = 意识体

## 2. 收回灵体的方法
**以下是收回灵体的步骤**：

1. **识别灵体**：首先要识别出附在你身上的灵体。
2. **理解灵体**：了解灵体的来源和目的。
3. **沟通与恐吓**：与灵体进行沟通，并对其进行恐吓，使其离开。
4. **包容与接纳**：接纳自己的负面意识，用爱包容它们。

## 3. 案例分析
**以下是收回灵体的一个案例**：

一个灵体跟随一个人4年，每天晚上折磨他，使他无法好好睡觉。通过恐吓灵体，他终于睡了一个好觉。

## 4. 内在意识的处理
**以下是处理内在意识的方法**：

1. **接纳与包容**：接纳自己的负面意识，用爱包容它们。
2. **理解与沟通**：理解自己的负面意识，并与它们进行沟通。
3. **转变观念**：转变对负面意识的态度，将其视为成长的机会。

## 5. 案例分析
**以下是处理内在意识的案例**：

一个人在心中咒骂别人，并责怪自己。通过接纳和包容自己的负面意识，他最终摆脱了这些负面情绪。

## 6. 总结
**本节课中我们一起学习了如何收回灵体和如何处理内在意识。希望这些方法能够帮助你摆脱负面意识，过上更好的生活**。
