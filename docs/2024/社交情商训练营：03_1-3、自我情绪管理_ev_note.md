# 社交情商训练营：03_1-3 自我情绪管理 🧘‍♂️

## 概述
在本节课中，我们将学习如何管理自己的情绪，包括认知情绪、转移情绪和释放情绪。

## 认知情绪
首先，要学会认知自己的情绪。**公式**：认知情绪 = 意识到情绪的存在 + 确定情绪的类型。

### 认知情绪的步骤
1. 意识到情绪的存在。
2. 确定情绪的类型（例如：愤怒、悲伤、快乐等）。

## 情绪转移法
当情绪即将爆发时，可以采取以下方法进行情绪转移：

### 疼痛转移法
1. 当感到愤怒时，将手放在大腿上，用力捏。
2. 当大腿的疼痛大于情绪时，情绪会得到缓解。

### 颜色转移法
1. 当情绪即将爆发时，观察并数眼前出现的颜色。
2. 重复数颜色，直到情绪得到缓解。

## 情绪释放法
压抑情绪会导致情绪爆发，因此需要学会释放情绪。

### 释放情绪的方法
1. 去高处观看，例如山顶或高楼。
2. 去海边，感受大海的广阔。
3. 做按摩，例如盲人按摩。
4. 跑步或打沙包。
5. 吃自己喜欢的东西。

## 总结
本节课中，我们一起学习了如何管理自己的情绪。首先，要学会认知情绪；其次，通过情绪转移法来缓解情绪；最后，通过情绪释放法来释放情绪。只有做好这三步，才能完全掌控自己的情绪，提高情商。
