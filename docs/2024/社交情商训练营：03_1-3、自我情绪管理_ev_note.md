# 社交情商训练营：03_1-3、自我情绪管理 🧠

在本节课中，我们将要学习如何有效地管理自我情绪。情绪管理是情商的核心组成部分，掌握它有助于我们在社交和生活中保持冷静与从容。我们将介绍三种实用的情绪管理方法，并理解从认知到管理的完整流程。

## 1. 认知情绪：管理的第一步

上一节我们介绍了情绪管理的重要性，本节中我们来看看具体方法。管理情绪的第一步，是清晰地认知到自己当前处于何种情绪状态。例如，当你感到愤怒时，需要先意识到“我正在生气”。

## 2. 三种情绪管理方法

在认知情绪之后，可以运用具体方法来管理它。以下是三种经过实践验证的方法。

### 2.1 疼痛转移法 🩹

这种方法的核心是**用身体上的物理疼痛感，来转移或覆盖情绪上的剧烈波动**。

当某个人或某件事令你特别生气时，首先认知到这是生气的情绪。为了阻止情绪升级或突然爆发，可以采取一个动作：将手放在大腿上或伸进裤兜，用力捏自己的大腿。你越生气，就越用力去捏。当大腿的疼痛感大于情绪强度时，愤怒就会消退，你会冷静下来，从而能够从容处理事情。

### 2.2 颜色转移法 🌈

如果当前场合不适合使用疼痛转移法（例如在正式会议中），可以采用颜色转移法。

其原理是**通过将注意力强制转移到视觉信息（颜色）上，来中断情绪的上升进程**。

在你感到情绪即将爆发时，冷静下来，开始数眼前出现的颜色。例如：红色、蓝色、灰色、白色、粉红色。如果数完一遍仍不够冷静，就再重复数几次，例如：红色、蓝色、黑色、棕色、白色、紫色。重复这个过程，你的注意力被颜色转移，情绪便会逐渐平复。

### 2.3 合理发泄法 🏃

疼痛转移法和颜色转移法都只是暂时压抑了情绪。如果压抑的情绪长期没有出口，最终仍会爆发。因此，需要定期通过合理的方式发泄掉这些负面情绪。

以下是几种合理的发泄方式：

*   **接触自然**：去高的地方，如山顶或高楼顶层，俯瞰下方；或去海边，面对一望无际的大海。广阔的视野有助于让人释怀，觉得眼前烦恼都是小事。
*   **身体活动**：去跑步，跑到满身大汗；或去健身房击打沙包。剧烈的身体活动能有效释放压力。
*   **享受服务**：例如去做一次按摩，让身体在酸痛中得到放松。
*   **满足口腹之欲**：吃一些自己非常喜欢的食物，也能让情绪得到舒缓。

## 3. 情绪管理的完整循环 🔄

我们已经学习了三种具体方法，现在将它们整合成一个可操作的完整流程。

要学会控制自我情绪，首先必须学会认知你的情绪，清楚知道当下处于何种状态。其次，管理情绪要从小处做起，从微小的情绪波动开始锻炼。可以利用疼痛转移法或颜色转移法，及时压抑住不良情绪，避免其爆发导致不当行为。最后，那些被暂时压抑的不良情绪需要被释放掉，可以通过去山顶、海边、按摩、跑步、打沙包或享受美食等方式来实现。

将**认知情绪 -> 管理情绪 -> 释放情绪**这三步每一步都做好，并且反复练习，就能逐渐完全掌控自己的情绪。

## 总结

本节课中我们一起学习了自我情绪管理的系统方法。我们首先明确了认知情绪是管理的起点。接着，掌握了三种即时管理技巧：疼痛转移法、颜色转移法和合理发泄法。最后，我们构建了从认知、管理到释放的完整情绪处理闭环。坚持实践这些步骤，你就能合格地掌控自己的情绪，这是高情商的重要标志。
