# 情绪价值课-【阿娇】：第14节 如何用幽默给对方提供情绪价值（下）

在本节课中，我们将继续学习如何通过幽默为对方提供情绪价值，深入探讨剩余的三个技巧。

### 4. 正话反说

**核心概念**：用完全相反的话来表达原本的意思，使表达更生动、有趣。

**示例**：
- 对方问：“你生气了吗？”
  - 正话反说：“我没有生气。但是你如果离我再近一点，我有可能会打死你。”

### 5. 比喻加联想

**核心概念**：将两个不同的事物进行嫁接，使不相关的事物产生共性，增强画面感和代入感。

**示例**：
- 将谈恋爱比喻成找工作：“你看上这个工作，不一定要你的工作；你要你的工作，不一定要看得上。就好像你看上这个男人，不一定能看得上你；看上你的男人，你又不一定能看得上他。”

### 6. 自我嘲讽

**核心概念**：通过嘲笑自己、拿自己开玩笑，让对方卸下防备，展现自信、豁达和风度。

**示例**：
- “如果要用几个字来形容我现在的生活状况，那我就是除了可爱，一无所有。”

### 总结

本节课我们学习了如何用幽默为对方提供情绪价值的三个技巧：正话反说、比喻加联想和自我嘲讽。通过这些技巧，我们可以使聊天更加有趣，增进彼此的感情。希望同学们在亲密关系中多运用这些技巧，让生活充满欢声笑语。
