# 乌鸦救赎《连招绝技-白驹聊天秘术》：04.白驹聊天秘术_ev

## 概述

在本节课中，我们将学习如何将白驹聊天秘术后的对话引导至正常对话，并掌握推拉技巧，使对话充满性张力。

## 核心概念

**公式**：

*   **Powerer**：抛出话题，引发对方兴趣。
*   **咬钩**：对方回应积极，表示感兴趣。
*   **鼓励**：夸奖对方，表示认可。
*   **推开**：适当推开对方，避免过度追求。

## 教程

### 1. 白驹聊天秘术后的对话引导

*   **置顶微信**：将已搞定的姑娘置顶，防止忘记。
*   **独立微信**：使用独立微信进行聊天，避免尴尬。
*   **信息交换**：进行基础信息了解，如工作、爱好等。

### 2. 推拉技巧

*   **Powerer**：抛出话题，如“你喜欢生活吗？”
*   **咬钩**：对方回应积极，如“喜欢做饭听歌睡觉”。
*   **鼓励**：夸奖对方，如“你真棒，太棒了！”
*   **推开**：适当推开对方，如“没事，我会鞭策你的”。

### 3. 推拉技巧的应用

*   **抛饵咬钩**：抛出话题，如“你会拍照吗？”
*   **鼓励**：夸奖对方，如“哇，太好了！”
*   **推开**：适当推开对方，如“那你给我拍可爱一些吧！”

### 4. 注意事项

*   **避免流水账**：将话题进行延伸，避免过于简单。
*   **符合生活**：使对话看起来自然，避免过于刻意。
*   **合情合理**：在社交中追求合情合理，而非绝对真理。

## 总结

本节课中，我们学习了如何将白驹聊天秘术后的对话引导至正常对话，并掌握了推拉技巧，使对话充满性张力。通过不断练习和应用，相信大家能够掌握这项技能，提升自己的聊天能力。
