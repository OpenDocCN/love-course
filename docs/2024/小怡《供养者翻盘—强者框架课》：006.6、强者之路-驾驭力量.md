# 小怡《供养者翻盘—强者框架课》：006.6、强者之路-驾驭力量

## 概述

在本节课中，我们将要学习一个真正有魅力、有力量的男人所具备的真实气场。我们将深入探讨如何驾驭内心的力量，这包括管理情绪、直面自身价值以及驯服欲望。掌握这些层次，将帮助你在人际关系中建立不可撼动的框架和吸引力。

## 强者气场的本质：有力量的平静

上一节我们探讨了强者框架的基础，本节中我们来看看强者气场的核心表现。

一个有魅力、有力量的男人，他的真实气场并非咄咄逼人或锋芒毕露。你在他的身上感受到的是一种有力量的平静，是一种没有压迫感的轻松和自在。

在长期的咨询教学中，我反复强调一个核心点：**情绪稳定**。你想要在恋爱中获得尊严、存在感和位置感吗？拥有这一切的前提是，你要有力量驾驭自己的情绪。

**核心公式：**
```
强者气场 ≠ 强势压迫
强者气场 = 情绪稳定 + 有力量的平静
```

如果女人的任何一点小事或小吵闹都能影响你的情绪，那么你这个男人的人格地基是非常不牢固的，撼动你的框架也是轻而易举的事情。此时此刻，女人无法感受到你的力量，觉得你很弱，自然就不可能真正地爱慕你和尊重你。

## 第一层次：驾驭怒火

情绪稳定是内心力量的第一道关卡。很多人对此有误解，让我们来澄清并学习如何真正地驾驭情绪。

### 情绪稳定 vs. 压抑情绪

例如，女朋友不爱看手机，发消息总是不回复。在恋爱中，这绝对不是一个好习惯。但如果你每次都因为她不看消息而忍不住发脾气，这就是一个非常低价值的行为。因为女生能清楚地知道，你很渴望她秒回你，而她回复消息的这个行为能左右你的情绪。

这时有兄弟会说：“我明白了，我就忍住不发火不就行了吗？”这其实是对情绪稳定的误解。很多人认为情绪稳定就是压抑情绪，只要憋住了，别人看不出来，情绪就是稳定的了。

而事实恰好相反：
*   **真正的情绪稳定**：是你觉得没必要，你有选择地不发脾气。
*   **压抑情绪**：是你不敢发脾气。你怕惹怒对方，怕让别人看出来自己生气，怕自己处于不利地位。

**核心概念：**
```
真正的情绪稳定 = 有选择地控制情绪
压抑情绪 = 因恐惧而隐藏情绪
```

### 将发怒变为手段

内心强大的第一个层次，就是能对自己的脾气收放自如。只要你能不为宣泄情绪而发怒，就可以做到一个高级技巧：**把发怒变成你的手段，为你做事的目的而服务**。

我给大家举一个例子：

有一天，A、B、C三个人在聊天。A突然说了一件B不喜欢听的事，C就赶紧在桌子下面用脚踢了踢A，暗示他不要再说了。这时候A突然对C说了一句：“你踢我干嘛呀？这话今天我必须说。”

接下来，如果A的状态是“老子今天这脾气就上来了，这话我就必须说了，怎么着吧？”，这就叫**发脾气**。在常见情况下，A确实是情绪管控力差、没有情商的那个人。

但如果这三位是公司的合伙人呢？因为B长期包庇自己在公司的亲信，搅乱公司秩序，A觉得今天必须摊牌了。而C是个和事佬，只想避免吵架，却没意识到问题的严重性。

那么这个时候，不顾一切“发脾气”摊牌的A，还是那个低情商的人吗？A或许知道今天发脾气会影响三人关系，但为了公司、为了自己的权益、为了三人能长久合作，他必须把事情摊开说清楚。他的行为是通过“你踢我干嘛”宣泄了情绪，而他的目的是把事情说清楚，挽回合作。他合理地运用情绪做出不满的表达，来达到自己的目的。**他的行为和目的是统一的**，这就是有情绪驾驭能力的表现。

更进一步，A未必是真的发脾气。他很可能只是让自己看起来像在发脾气，从而达到对B和C的威慑力。这才是真正做到了运用情绪、收放自如，而不是被情绪所控制。

### 如何判断情绪驾驭能力的高低

以下是判断一个人驾驭情绪能力高低的两个关键对比：

*   **能力低的体现**：你发了脾气、吵了架，造成了不可挽回的后果。这些并不是你想要的，只是因为情绪失控而做出了让你后悔的行为。
*   **能力高的体现**：你通过发脾气、展示了力量，实现了你的目的。比如让那个总欺负你、挑衅你的人开始怕你、看得起你，甚至尊重你。你通过运用自己的情绪，做出了顺应目的的行为，最终达成了目的。

总而言之，判断一个人驾驭情绪能力的高低，就是看他的情绪是否能为他的目的服务。如果能，就是高；如果不能，就是低。

我们平时待人谦和、处事平静，是因为我们并不计较小矛盾。这些小擦伤对我们来说并不算什么，但不意味着你就能践踏我的利益、突破我的原则。**表达愤怒，是我们建立并且维系自己不可撼动底线的重要方式**。

如果一个男人能恰当地运用情绪，对情绪收放自如，每次行为的表面失控都对应着想要达到的目的，那么他就做到了内心力量的第一个层次：**驾驭怒火**。虽然这只是第一个层次，但生活中能完成这个层次的男人并不多。不管对面是男人还是女人，他都会清晰地收到一个信号：你是一个驾驭情绪的强者。

## 第二层次：直面低价值

掌握了情绪管理后，我们需要向内审视，构建更深层的价值感。本节我们将探讨如何直面并接纳自身的低价值。

想理解这个层次，你先来回答我一个问题：男人怎样才算见过世面？

有的兄弟可能会说，那必然是吃过见过。什么是“吃过”？是吃过几万美元的鱼子酱，还是吃过各个城市的米其林餐厅？什么是“见过”？是见过股票账户每日七位数的涨跌，还是见过百万人才动员的大场面？

如果你这么想，你见过的世面就永远局限在了这个世界给你设定的规则之内。而你没见过的东西，永远是你的痛点，你自卑的源头。

我告诉你，真正的见世面只有一种：**见过自己最脆弱、最不堪的一面，然后接纳他**。

你们想想，为什么有很多时候你控制不住自己的需求感，总是过度表达、过度解释？因为你害怕对方对你的误解，害怕暴露自己低价值的一面，害怕对方因此不喜欢你了。

### 低价值陷阱：解释就是掩饰

例如，一个女生问你：“哎，你开的什么车啊？”四个字，“我没有车”，你的回答就完事了。但大多数男人往往一定要补上几句：“哎，你不知道现在开车太麻烦，没有地儿停，各种堵车不说吧，关键是不低碳环保，之前雾霾这么严重，现在要碳中和，总之啊还是坐地铁比较好。”

你想想为什么你要补充解释这么多？因为你掉进了这个世界给你设定的规则里。什么规则？就是“高价值男人都得开好车，开好车才能泡好妞，没有车就是低价值的体现，是男人无能的象征”。所以你要解释，希望通过解释，把你的低价值往回拉，没成想却越来越低。因为**解释就是掩饰**。

但同学们你要知道，没有人能逃避自己的低价值，因为价值是在人与人比较之间分高低的。比较这件事，在你的一生中是永无止境的。

*   你家产千万，那么当你遇到家产上亿的人时，你的千万家产就是你的低价值。
*   你身高1.8米，也算精神，但当你面对明星模特时，你拿得出手的外形也是低价值。
*   更不用说每个人都有自己擅长的领域。你是个大学教授，能在讲堂里侃侃而谈，但把你放到夜店舞池里，你可能会手足无措。

### 真正的高价值：坦然面对低价值

所以说，一个男人怎样才算真正的高价值呢？一个重要的标准就是：**他是否有能力直面自己低价值的一面**。

我给你讲个故事：

我曾经有一个闺蜜，在学校念书时特别迷恋一个男孩。刚开始接触时，她对那个男生没什么特别感觉。因为在我们看来，男生其貌不扬，身高也就1.7米，属于在人群中都找不出来的那种。

但有一次，这个男生从外地来天津看她，让她一下子产生了不同的感觉。当时大一我们都在天津，我闺蜜就拉着我一起带那个男生转转，顺便请他吃个饭。逛街时看见小巷里藏着一家普通的小饭店就进去了。

进去之后才发现别有洞天，进门的屏风、桌上的香都不是寻常小店的摆设，服务员也是手脚利落、衣着不俗。等我们坐下以后，服务员端了两杯茶就站到了一边等着我们点餐。当时我们就觉得感觉不太对了，接着一翻开菜单就傻眼了，价格是普通饭店的四五倍还要多。

说实在的，虽然是我闺蜜请客，但当时我真的觉得挺尴尬的，感觉服务员的眼神都怪怪的，不知道如何是好。但没想到的是，这个男孩跟服务员招招手，然后很自然地说：“哎，你好，我们进来的时候，不知道你们家的价格，我们是学生，不好意思啊，打扰了。”

服务员也就微笑了一下，没有说什么，就把我们送出去了。出门之后，这个男生就轻松地说：“大城市果然不一样啊，小巷里的饭馆都深藏不露，这会啊算是长见识了。”顿了下，又转过来和我们说：“走，吃海底捞吧，那是我主场。”

当时我和我闺蜜，就从大大咧咧的两个女大学生变成两个听话的小女孩被他领走了。整个过程，我从尴尬懵逼到敬佩。我不仅敬佩他的解围能力，更敬佩他的心理素质。进饭店之前、在饭店里面、出饭店之后，这个人的语气都不曾变一点儿。他从来没有因为穷觉得丢脸，更没有因为有两个女生而在逞能。

那个瞬间，我就觉得这个男生超级强大。因为当时我看菜单的时候，我的那些“力量”都没有扛住，我就感觉到窘迫不安，特别尴尬，但我还在假装我能消费得起，我怕别人看不起我。我意识到自己的低价值，但我不敢面对它。而这个男生在面对更高价值的世界时，他的那些力量却扛得稳稳的，处之泰然，来去自如，甚至还能分神安慰我们。

**核心总结：**
```
低价值 ≠ 感情失败的根本原因
无法坦然面对低价值的事实 = 无法散发魅力的症结所在
```

这些年“普信男”很火，女孩子们呈现出一种对这类男孩的嘲笑和排斥。但同学们你们要知道，自信从来是男人最具高价值的砝码。但为什么很多女生还会抵触呢？真正自信和普信之间的区别到底是什么？

很简单，依然是你是否能直面并接纳你的不完美甚至低价值的那一刻。一个男人能伸不能屈，这就是逞强，是“普信”；但这个男人能伸又能屈呢？这就是真正的强大，是真正的自信。

当有一天你真的能做到时，你会发现曾经自己所做的一切都是南辕北辙、都是错的。因为很多时候，想要拉高自己的心理位置，就需要我们先俯下身去。

## 第三层次：驯服欲望

构建了稳固的自我价值后，我们需要挑战最难的一关：掌控那些驱动我们的原始欲望。本节我们将聚焦于征服欲，并学习如何驯服它。

这里的欲望，我只谈一种：**征服欲**。我一直认为，能从征服欲中抽离的男人，才是真正站在社交食物链顶端的男人。

我想问你一个问题：如果一个男人与女人相处时，最在意的点是“如何能让她乖乖听话”，你觉得他的心态怎么样？你觉得他的内心有力量吗？

有的兄弟可能会说：“哎，这不挺有劲儿的吗？心态不错呀。”其实完全不对。我来告诉大家，这种话一出，这个男人在恋爱里的位置往往是比较低的，而且爱得很痛苦。为什么呢？因为他的需求感太强了。

既然征服欲是一种欲望，那他就会有强烈的被满足的渴望。你把恋爱当成征服他人的游戏，于是就有了好胜心。这时候你就已经彻底把自己的主动权交给了对方。而且，不仅是恋爱，对任何人、任何事的执念都是一种无力的表现。因为你无法挣脱出来，只能任凭欲望把你带向未知的泥沼。

### 什么是驯服欲望？

我再给你举个例子，这是件非常小的事，但对我震撼很大。这件事发生在我的一男生朋友身上。

有一次，好多男生聚在他们宿舍，其中一个外宿舍的男生发现了他的床上有一个臂力棒，就拿起来掰了一下，说：“哎，真难哪。”我朋友就笑了笑。那个外宿舍的男生听见就不爽了，然后问我朋友：“能掰几个呀？”我朋友就含糊地说：“哎，也就50下吧。”

那个外宿舍的男生明显感觉没面子了，非说不信，让我朋友现场掰50下。可是我朋友就不太想掰，觉得没有必要证明什么。但这位外宿舍的男生非要咬住不放，最后说了一句：“哎，你要是能现场掰50下，我当场我就吃屎。”

你看看，这个时候气氛烘到这儿了，对吧？我朋友就拿起了臂力棒开始掰，旁边人就替他数：“一个、两个、三个……”就这样，我朋友非常轻松地在掰着臂力棒，紧跟着就到了46、47、48个，哎，眼看就要掰完了。就在数到49的时候，我朋友突然停下了动作。

所有人都很疑惑：“哎，你怎么停了呢？”我朋友慢慢把臂力棒放下，说：“哎呀，掰不动。”

兄弟们，你们感受到震慑力了吗？前49下，我是在向你展示力量，告诉你，我能做到。最后一下选择了放下，是宽恕，是他对欲望的收放自如，是“我有能力，但是我没有必要让你下不来台”。

我跟大家讲，在现实生活中，这样的人是非常“可怕”的。当然我说的可怕，不是说他坏、居心不良，而是说他的内心力量是无可估量的。为什么呢？因为根本没有任何一种欲望能够拿捏住他。最后一棒意味着他对碾压和征服的感觉，毫不留恋。说白了，即便是这种男性都想要的感觉，他也毫不在乎。

**核心法则：**
```
极致强大 = 拥有随时抽身的能力 + 不被任何欲望绑架
```

当我们在恋爱关系中面对女生时，也是同样的道理。经营一段关系最重要的前提是，你要拥有抽身的能力。一个男人能在必要的时候毫不留恋地处理一段关系，不被任何欲望绑架，这才是一种极致的强大。做到这一点，也将没有人能在心理位置上超越你。

## 总结

好了，兄弟们，以上呢就是驾驭内心力量的三个层次。

1.  **第一层：驾驭怒火的能力**。学会让情绪为你的目的服务，而非被情绪控制。
2.  **第二层：用高价值心态扭转低价值的能力**。坦然面对并接纳自身的不完美，是真正自信的起点。
3.  **第三层：驯服内心欲望的能力**。尤其是征服欲，能够从中抽离，拥有随时“放下”的魄力，才是顶级的内心力量。

本节课中我们一起学习了如何通过这三个层次，逐步构建一个强者稳固的内心框架。从控制情绪的表层，到接纳自我的深层，再到驾驭欲望的核心，这是一个向内探索、不断强大的过程。掌握它们，你将在任何关系中都能保持从容与力量。
