## 动作分解

### 动作名称

直上九霄

### 动作组成

该动作由两个部分组成。

### 第一部分

1. 当动作进行到一定程度时，几乎所有男性都会想要射精。
2. 此时，我们可以引导对方将精液射出。
3. 手部动作：右手扶住阴茎，左手从左侧绕过向下。
4. 当左手向下时，抓住龟头并绕过向下。

### 第二部分

1. 抓住龟头，轻柔地向下移动。
2. 再次抓住龟头，向下移动。
3. 这两个动作可以交替进行。
4. 如果男性想要射精，可以加快速度。

### 注意事项

1. 在整个动作过程中，手部动作要柔和。
2. 根据男性的需求调整动作节奏。
