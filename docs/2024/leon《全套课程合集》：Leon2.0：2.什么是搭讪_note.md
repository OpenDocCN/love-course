# Python基础教程 1.1

## 🌟 概述

在本节课中，我们将学习Python编程语言的基础知识，包括安装Python、基本语法和变量。

## 📦 安装Python

首先，您需要安装Python。以下是安装步骤：

1. 访问Python官方网站：[https://www.python.org/](https://www.python.org/)
2. 下载适合您操作系统的Python版本。
3. 运行安装程序并按照提示完成安装。

## 🌟 Python基本语法

Python是一种解释型语言，这意味着您可以直接在终端中运行Python代码。

### 1. 变量

在Python中，变量不需要声明类型。以下是如何创建变量的示例：

```python
x = 10
y = "Hello, World!"
```

### 2. 数据类型

Python有几种内置数据类型，包括数字、字符串和布尔值。

- **数字**：整数和浮点数
- **字符串**：文本数据
- **布尔值**：True或False

### 3. 运算符

Python支持各种运算符，包括算术运算符、比较运算符和逻辑运算符。

- **算术运算符**：+、-、*、/、%
- **比较运算符**：==、!=、<、>、<=、>=
- **逻辑运算符**：and、or、not

## 🌟 实例

以下是一个简单的Python程序，它打印出“Hello, World!”：

```python
print("Hello, World!")
```
