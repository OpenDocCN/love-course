# 15、关键认知：关键关系训练营（学浪）：1-08节鞭策自己行动的“偶像”

大家好，欢迎来到关键关系第八讲。这一讲，我们要探讨一个特殊的关系——偶像。

**什么是偶像？**

你年轻的时候，是否有过偶像？甚至现在，你是否追星？人类对偶像有刚需，需要一个虚幻的幻象来麻痹自己。

**偶像的力量**

全世界第一大的意见领袖KOL是C罗，他是偶像。很多人把C罗当成头像或名字，因为他自律、荣誉等特质，这些特质是自我投射。

**偶像的作用**

偶像可以改变人的行为，榜样力量无穷。我们要设计一个能激励我们改变自己行为的偶像。

**如何设计偶像？**

规避不利信息，设计一个能激励我们真实改变自己行为的人。

**偶像的本质**

偶像本质上是一个错误的认知，因为我们没有看到真实的偶像。但偶像可以让我们更喜欢自己，更有责任感。

**偶像的社会功能**

偶像可以承载年轻人的自我实现需求，避免他们走向负向领域。

**如何设置偶像？**

未成年人或刚成年的人可以设置一个偶像，帮助自己快速成长。

**总结**

本节课中，我们学习了如何设置和利用偶像来鞭策自己行动。通过设计偶像，我们可以激励自己改变行为，实现自我成长。
