# 15、关键认知：关键关系训练营（学浪）：1-17节敢于欠人情与还人情让“筹码”升级

大家好，欢迎来到关键关系第十七讲。这一讲我们将探讨如何在关键关系经营中升级筹码。

## 筹码升级的核心

**公式**： 筹码升级 = 权力 + 资源 + 能力 + 人情

**代码**： 筹码升级 = 权力 + 资源 + 能力 + 人情

### 人情的重要性

人情是筹码升级的关键，它不仅仅是金钱，更是人与人之间的互相帮助和亏欠。

### 朋友与普通关系的区别

- **好朋友**： 双方互相帮助，互相亏欠，关系难以算清。
- **普通关系**： 一方寻求帮助，另一方给予帮助或礼物，关系较为明确。

### 工程老板的例子

工程老板通过频繁的求助和帮助他人，建立了广泛的人脉，从而在关键时刻拥有更多的资源和能力。

### 敢于欠人情

在关键关系中，要敢于向他人求助，敢于欠人情，这样才能在长期交往中形成更深层次的关系。

### 人情是结果导向的

人情的价值在于结果，而不是过程。即使过程中没有达成目标，只要结果有价值，就能形成人情。

### 价值的主观性

价值是主观的，对方认为有价值才有价值。因此，在送礼或帮助他人时，要考虑对方的需求和价值观。

### 筹码升级后的应用

一旦筹码升级，可以借助这种关系办理更重要的事，或将其作为绑定其他关键关系的筹码。

## 总结

本节课中，我们学习了如何在关键关系中升级筹码，重点在于敢于欠人情和还人情。通过建立深厚的人情关系，我们可以获得更多的资源和能力，从而在生活和事业中取得更大的成功。

希望同学们能够将所学知识应用到实际生活中，不断升级自己的筹码，建立更多有价值的关系。
