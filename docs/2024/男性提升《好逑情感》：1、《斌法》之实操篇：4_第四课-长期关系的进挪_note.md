# 男性提升《好逑情感》：1、《斌法》之实操篇：4_第四课-长期关系的进挪

🎼**第四课：长期关系的进挪**

首先，我们介绍一个词汇——“禁裸”。这个词可能大家第一次听说，可以去网上查询，这里不做详细解释。许多兄弟在发展长期关系中容易犯错，要么过早要么过晚地“禁裸”。下面先来谈谈过早“禁裸”。

### 过早“禁裸”

对阶段的把握不明确，刚认识就动手动脚。过早“禁裸”会让妹子原本的期待落空，感到失望。因此，本节课将从长期关系的阶段入手。

### 长期关系阶段

以下是长期关系的四个阶段：

1. **初始阶段**：一般通过在线聊天等方式互动，从陌生到认识到朋友，同时为邀约做铺垫。这个阶段不建议“禁裸”，因为会显得你很猥琐、很饥渴。即使“禁裸”成功，你也会觉得对方很随便，对以后的相处不利。

2. **相知阶段**：从普通朋友到亲密朋友。很多人有错误认知，认为想发展长期关系就不能做朋友。实际上，想快速推进关系对大多数人来说很困难，想通过一见钟情也很难。通常需要从陌生到熟悉到朋友到暧昧，再到关系的确认。因此，不要害怕做朋友，但也不要一直做朋友，快速推进容易翻车。相知阶段是完成亲密度的过程，需要达到以下目标：在约会中友好互动，主动分享生活、工作、成长的经历。相知阶段可以“禁裸”，但不是必须。

3. **升级阶段**：超越普通朋友的阶段，可以有一些暧昧的举动，但必须保持“禁裸”的关系。否则，会让对方觉得你对她的兴趣不大，逐渐会冷落你。大部分兄弟在这个时候错失良机。当妹子对你冷淡时，你事后只能后悔。因此，该“禁裸”的时候必须“禁裸”。

4. **确立阶段**：不建议兄弟们在这个阶段表白，因为关系确定后等于终止了，全面进入相处阶段，对男方来说不利。因此，不要表白，要表白也要在实质性关系确定之后，百分之百确定她喜欢你之后。

### 总结

本节课中，我们一起学习了长期关系的四个阶段以及在不同阶段如何处理“禁裸”的问题。希望对大家有所帮助。
