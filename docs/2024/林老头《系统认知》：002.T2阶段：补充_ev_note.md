# 林老头《系统认知》：002.T2阶段：补充_ev_note

在本节课中，我们将学习如何将T2阶段的理论知识具体落地，核心是掌握“展示价值”后的关系升级技巧。我们将重点探讨如何通过预设身份、调整可得性以及实际推进关系，来建立健康的短期或长期关系。

上一节我们介绍了展示三大价值的重要性，本节中我们来看看如何将吸引转化为实际的关系进展。

## 1. 预设身份：像对待女友一样聊天

当你成功展示价值并吸引到女生后，关系会进入一个新阶段。此时，最关键的一步是主动升级关系，而“预设身份”是新手最友好的方法。

**核心概念**：预设身份，即假设你们已经是男女朋友关系，并以此身份进行互动。

为什么要这样做？因为女生同样面临升级关系的尴尬，她们往往选择拖延或等待。如果你不主动提供“恋爱感觉”，关系就容易停滞。主动预设身份，就是由你来承担提供这种感觉的责任。

以下是具体操作方法，主要从两个层面入手：

*   **讲话风格拉近**：在聊天中直接带入暧昧或亲近的语气。
    *   例如：“今天怎么这么晚才想起给我发消息？”
    *   例如：“有没有想我？”
*   **善用表情包**：这是对新手的强力辅助工具。
    *   **行动指南**：收集至少50个恋爱、甜蜜、可爱的表情包。
    *   **使用方法**：在吸引阶段之后，大胆地使用这些表情包来拉近距离。

**核心要点**：**厚脸皮**。不要害怕被拒绝，所有的恐惧都源于想象。你必须敢于尝试，把对方当成女朋友来聊。如果把握不好力度，可以寻求指导。同时，方法需与你的关系预期（长期或短期）匹配：
*   **长期倾向**：表达可以更含蓄。例如：“今天工作时突然想到你了，感觉有点奇怪，你在干嘛呢？”
*   **短期倾向**：表达可以更直接热烈。例如：“什么时候见面？想立刻见到你。”

当我们掌握了预设身份的方法后，关系会开始发生微妙变化。接下来，我们需要学习如何根据对方的反馈来调整节奏，这就是“可得性调整”。

## 2. 可得性调整：推拉的节奏艺术

主动推进关系后，对方的反馈可能有好有坏。根据反馈灵活调整你的“可得性”（即你看起来有多容易被得到），是掌控关系进度的关键。

**核心概念**：可得性过高，会让人感觉你廉价；可得性过低，又会让人安全感不足而逃离。需要动态平衡。

以下是针对不同反馈的调整策略：

*   **反馈良好时**：如果对方接受并回应你的暧昧，那就顺势继续增加可得性，大胆推进。
*   **反馈不佳时**：如果对方表现出犹豫、拒绝或测试你（例如“太快了”、“你好油”），你必须**立刻、主动地降低可得性**。
    *   **错误做法**：仅仅口头说“那算了”，但继续频繁联系。
    *   **正确做法**：真正地后撤。减少联系，丰富自己的生活并展示在朋友圈，甚至可以委婉透露你正在接触其他人。这并非“玩死”关系，而是制造“稀缺感”，往往能让对方更加上头。

**重要原则**：建立一段健康的关系需要承担风险。一点风险都不愿冒，就不可能获得真正的亲密关系。主动降低可得性就是你需要承担的风险之一。

那么，什么时候需要提高可得性呢？当对方因**安全感极度匮乏**而想要逃离时。这时你需要通过**花钱、花时间、耐心哄慰**来真金白银地提高可得性，给予对方安全感。

通过预设身份和灵活的可得性调整，你们的关系将快速深化。最后，我们需要将一切落实到现实行动中。

## 3. 实际升级：将关系锚定在现实

线上聊天的最终目的是线下发展。关系的最终确立，必须依靠实际的、面对面的推进。

**核心行动准则**：**早点见面**。不要追求在线上聊到“确定关系”再见面，尽早约会，在现实中互动和升级。

在实际推进过程中，你的态度必须明确且坚定。以下是需要你展现强势态度的三种情况，一旦出现，应立刻考虑断联：

1.  对方的行为严重违背你开始这段关系的初心（如欺骗、不尊重）。
2.  对方故意搞你心态，玩弄你的感情。
3.  对方提出极其不合理的要求。

**态度公式**：**强势态度 = 对原则的重视**。你的强硬恰恰证明你在意一段健康的关系。纵容一次，就会有无数次。

关于发生亲密关系，核心秘诀依然是：**厚脸皮**。要像“大姐姐主导小男孩”那样，拥有胆大心细脸皮厚的魄力，直接、坦诚地表达你的意愿。

本节课中我们一起学习了T2阶段落地的三个核心步骤：首先，通过**预设身份**主动提供恋爱感；其次，根据反馈灵活进行**可得性调整**，掌控关系节奏；最后，勇于**实际升级**，尽早见面并在现实中确立关系，同时坚守原则，展现强势态度。记住，**胆大、心细、脸皮厚**是贯穿始终的行动指南。
