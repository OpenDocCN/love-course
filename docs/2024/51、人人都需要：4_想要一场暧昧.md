# 51、人人都需要：4_想要一场暧昧

在本节课中，我们将学习如何策划一场成功的暧昧约会。我们将探讨从发起邀约到约会细节的完整流程，帮助你掌握关键技巧，提升约会成功率。

## 发起邀约的正确方式

上一节我们介绍了约会的重要性，本节中我们来看看如何发起邀约。

运动员参加奥运会前会进行数月甚至数年的密集训练。他们需要面对巨大心理压力和最高水准的对手比赛。稍有差错就会输掉比赛甚至人生。

约会不像奥运会比赛那样令人紧张。但它们都有一个共同点：你只有一次机会。你说的每句话、做的每件事都会为你加分或减分。这些因素综合起来，决定了面前这个男人是否会喜欢上你。

第一次和男人约会时，首先冒出的问题不是如何让自己更有魅力。对很多人来说，第一个问题是如何约他出来。你永远没有办法强迫一个男人来约你。

这件事只能由他自己决定。你基本上什么都做不了，所以我们不妨换一个思路。邀约这件事完全可以用女生发起。

很多人担心主动约男生会显得掉价。确实会掉价，但只出现在你不知道如何正确约男生出来的时候。

一个男生不主动约你大致是三个原因：
1. 他完全不喜欢你
2. 他对你不讨厌但也没到喜欢的程度
3. 你太优秀所以他不敢约你

除了第一种情况，后面两种情况可以由你主动发起邀约。

以下是正确的邀约方式：

**重点是需要一个正当理由**。“明天有没有空？我们去看电影吧”、“改天出来坐坐”这种邀约方式太直接。男生一下子就能懂你的意思。

你需要加一个理由：
```
“你上次讲到的演讲技巧对我很有帮助，我想请教一下你。如果你有空的话，咱们出来喝杯咖啡吧。”
```

这个方法的关键点不在于消除你表现出的意图，而是让他感觉到通过聪明才智、个人天赋等独特因素吸引了你的注意，所以你才对他有了兴趣。

男人特别爱嘚瑟，也特别好为人师。抓住这个弱点，他们就会神不知鬼不觉地进入我们的圈套。如果让他觉得你是被他身上的某某优点所吸引，他们就会觉得这次约会是自己争取来的。

换句话说，你需要让他先通过测验。

## 约会地点的选择策略

如果能够顺利和意中人出来，大部分人会把第一次约会安排在餐厅里。今天我要告诉你，第一次约会时光吃饭绝对不是最佳选择。

有大量证据证明，如果将喜欢的人带到能激起对方强烈情绪或能打开对方心扉的地方，他就会更喜欢你。情绪唤起和性吸引之间有着强烈的联系。

有一个非常著名的实验叫“高度焦虑状态下的性吸引力提高的证据”。实验人员把一群男人带到峡谷，峡谷有两个吊桥：一个非常安全，一个看起来又危险又吓人。

实验人员安排一部分男生走安全吊桥，另一部分走危险吊桥。在吊桥尽头都有一个美女助理迎接他们。女助理会简单交谈，并不经意留下联系方式。

最后走过危险吊桥的男人基本都主动打电话约女生出来喝咖啡。走过安全吊桥的男人只有很少会给女助理打电话。

这说明环境对人产生的情绪波动越大，越能够唤起他们的情欲。在你的城市找一个危险的吊桥不太可行，但要知道只要能够激起对方的情绪波动，他就很容易对你产生感情。

所以你应该主动提议在约会途中加入一两个能够唤起对方情绪的约会场合。比如骑马、看恐怖电影、看球赛或演唱会。酒精也是很好的能够唤起情绪的东西。

如果一场约会下来能够做到动静结合，对他来说是非常奇妙的体验。他会把这些功劳全都归功到你身上，因为只有和你约会他才有这种感觉。

你应该了解一下所在城市里哪些有刺激的、好玩的东西。千万不要害怕提出这些建议。很多男生很多时候都很笨，在他脑子里除了吃饭、看电影，可能找不到第三个约会项目了。

## 餐厅选择的技巧

就像我们刚才说的，无论选择什么样的约会活动，基本上都离不开吃饭这个环节。我相信绝大多数男人还是有基本的绅士意识，会在吃饭前问你：“你想吃什么呀？”

大部分女生听到这个问题就说“随便”。但这句话让男生非常头疼，因为他既不想以之前经验来取悦你，同时也不想让你觉得他是一个不上档次的人。

这个时候你可以提出一个有情调但不太贵的餐厅。在男生看到菜单的那一刹那，他就会觉得你是一个不作的女人，这也会在一定程度上给你加分。

你最好在小黑本上记上一两家好看的、充满情调的、但价格不太高的餐厅。

## 着装风格的重要性

很多女生在出门前都会抱怨没有合适的衣服，穿哪件都不对劲儿。解决这个疑惑一点都不难，重点是根据他的风格来。

不出意外，在第一次约会之前你们就应该已经认识过、了解过了。他是什么样的人？他的气质怎样？他是谦谦君子还是一个技术宅？是摇滚歌手还是其他类型？

你得知道每个人的习惯从他衣着上就给他分类了。就像对面走过来一个女人，你马上会从上到下打量一番，看看衣服什么牌子、包包是不是限量款、鞋跟高度，然后在脑海里给他身份加个定义。

对异性而言，相似的着装可以第一时间拉近你们两个的距离。为了让第一次约会更好地、更快地进入状态，我建议你一定要根据他的风格来。

你可以事先在聊天时了解他的身份，然后从他的朋友圈找一些蛛丝马迹，再翻看一下衣橱找那件最合适的。当然，如果你没有那样子的衣服也不用担心，因为在后面的课程里，我还会教你一些其他的办法让他喜欢你。

## 课程总结

本节课中我们一起学习了策划暧昧约会的完整流程。

我们来总结一下：
1. 你永远没有办法强迫一个男人主动来约你。我们可以用正确的方式来主动约男人，那就是给一个理由，让他感觉到他是通过自身的优秀才赢得了你的青睐和这次约会。
2. 餐厅不是最好的选择。最好的选择是在城市里找一个刺激的、好玩的地方作为第一次约会的场地。
3. 当男人问你想吃什么的时候，你要带他去一个有情调的又不贵的餐厅。
4. 人们都习惯按衣着来给对方分类。你的约会着装应该尽可能和他的风格靠拢，这样才能让你们的距离快速拉近。
