# 关系博弈（拒绝卑微和低位）：18_5.情感博弈-关系的本质-关系是怎么开始的

## 概述
在本节课中，我们将探讨关系的本质，特别是关系的开始，以及价值交换在其中的作用。

## 关系的本质与价值交换
关系的开始源于相互吸引，这种吸引分为两种：理性层面的价值吸引和感性层面的情绪吸引。

### 理性层面的价值吸引
价值交换是理性权衡利弊，以利益得失来思考关系，是理性思考的决定。

**公式**：价值吸引 = 情绪价值 + 生存价值 + 繁衍价值

- **情绪价值**：满足人的情绪需求，如快乐、稳定等。
- **生存价值**：满足人的生存需求，如财富、资源等。
- **繁衍价值**：满足传宗接代的需求，如外貌、身材等。

### 感性层面的情绪吸引
情绪吸引是情绪波动起伏，以感受体验去衡量关系，是情绪化的决定。

**公式**：情绪吸引 = 情绪波动 + 情感体验

- **情绪波动**：在短时间内极度开心和极度伤心。
- **情感体验**：通过相处、聊天等方式产生的情感体验。

## 情绪吸引与价值吸引的优先级
在感性与理性的冲突中，情绪波动往往大于价值吸引。

**例子**：
- 一个帅气且家境不错的男生，但相处感觉一般，不会聊天，没有礼貌，因此没有心动和情绪起伏。
- 一个条件一般的男生，但能让你情绪波动，相处时特别开心或沉稳，因此心动。

在这种情况下，你会更倾向于选择能让你有情绪波动的男生。

## 总结
本节课中，我们学习了关系的本质、价值交换以及情绪吸引与价值吸引的优先级。了解这些有助于我们更好地理解关系的开始和维护。
