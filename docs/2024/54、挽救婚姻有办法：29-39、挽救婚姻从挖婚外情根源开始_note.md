# 54、挽救婚姻有办法：29-39、挽救婚姻从挖婚外情根源开始

**概述**：本节课将探讨如何从挖掘婚外情的根源开始，挽救婚姻。

**一、婚姻现状分析**

1. **婚姻模式**：许多婚姻中，女方处于控制地位，男方较为顺从，给予女方很多爱和支持。
2. **婚姻问题**：随着时间的推移，婚姻中的问题逐渐显现，导致婚姻出现危机。

**二、婚外情根源分析**

1. **婚姻现状**：婚姻中的问题导致夫妻关系紧张，男方可能寻求婚外情作为逃避。
2. **婚姻爆点**：婚姻中的矛盾冲突点，如孩子教育、事业发展、家庭资产等。
3. **性格与需求**：挖掘夫妻双方的性格特点、需求以及背后的原因。

**三、挽救婚姻的方法**

1. **挖掘根源**：分析婚姻中的问题，找出矛盾冲突的根源。
2. **理解伤害**：理解婚姻问题对双方造成的伤害，而非纠结于对错。
3. **个人辅导**：进行个人辅导，帮助夫妻双方认识自我，改善关系。

**四、总结**

本节课从挖掘婚外情的根源开始，分析了婚姻现状、婚外情根源以及挽救婚姻的方法。通过挖掘根源、理解伤害和个人辅导，夫妻双方可以改善关系，挽救婚姻。
