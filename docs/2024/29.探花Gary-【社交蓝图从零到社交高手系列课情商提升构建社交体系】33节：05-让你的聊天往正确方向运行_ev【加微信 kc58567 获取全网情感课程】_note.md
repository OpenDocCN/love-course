## 一、聊天话题的选择

在与对方初期接触时，很多人都知道要展示价值，但不知道在等待价值展示机会的期间，应该聊些什么。其实，从接触初期推进到积极反应之前，我们可以聊对方这个人本身，对他好奇。

**核心概念**：每个人都有自我意识，都希望话题围绕自己展开。

**公式**：聊天话题 = 对方个人话题 + 表达好奇

以下是具体话题：

- 外貌特征：眼睛、耳朵、头发、妆容、穿衣风格、身高等。
- 信息来源：社交软件照片、朋友圈、个人介绍等。
- 聊天技巧：5WEH新法、钗子联想法。

## 二、聊天前提

在聊天一开始，要让对方知道这次对话是一个什么样的交流。社会中的所有社交活动都有一个前提存在。

**核心概念**：聊天前提 = 交流双方的关系

以下是几种常见的前提：

- 同事关系
- 上级与下级关系
- 校友关系

## 三、前提植入技巧

在聊天中，通过设计话术惯例，可以植入前提，让对方明白交流的目的。

**核心概念**：前提植入 = 通过话术惯例表达交流目的

以下是三种不同程度的前提惯例：

1. **高力量前提**：直接赞美对方，例如“我觉得你很可爱”。
2. **中等力量前提**：幽默地表达，例如“你的眼睛是怎么长得那么好看，挖了送给我吧”。
3. **低力量前提**：推拉式表达，例如“你眨眼睛的时候很好看。我很喜欢。但是你一闭上眼睛，我就不喜欢了。”

## 四、应对不同情况的新朋友

当与不同情况下认识的新朋友交流时，可以按照以下步骤进行：

1. 简单交流，导入模型，判定反应区间和价值对比。
2. 配合5WH和展示价值的方法向上推进。
3. 抛出前提惯例，避免低价值行为。
4. 应对测试，邀约和实际接触。
