# 21、凯男的社交情商课：第一节课：爱商提升的7个思维逻辑：008.一：恋爱流程上

## 概述
在本节课中，我们将学习如何通过提升爱商来改善恋爱关系。我们将探讨恋爱流程中的三个阶段，并分析如何通过展现个人价值来吸引对方。

## 恋爱流程
### 三个阶段
1. **吸引阶段 (A阶段)**：建立初步的吸引力。
2. **舒适感阶段 (C阶段)**：加深彼此的了解，建立舒适感。
3. **诱惑阶段 (S阶段)**：进一步发展关系，激发对方的兴趣。

### 吸引阶段
- **吸引要素**：
  - **外在价值**：外貌、身材等。
  - **内在价值**：性格、才华等。
  - **情绪价值**：阳光、体贴、情商等。

### 生存价值
- **权利地位**：社会地位、财富等。
- **金钱**：经济实力等。

### 综合价值
- **公式**：综合价值 = 繁衍价值 + 生存价值 + 情绪价值

### 价值交换
- **本质**：两性相处是价值的交换。

### 吸引力开关
1. **雄性领袖**：自信、微笑、注重仪表、幽默感、善于交往、成为社交中心。
2. **社交认证**：通过朋友圈、朋友等展现高价值。
3. **预选**：展现不缺女人的特质，增加吸引力。
4. **爱心**：展现关爱他人、有责任心的特质。
5. **提供价值**：展现愿意为对方付出的特质。
6. **情绪吸引**：调动对方的情绪，使其产生兴趣。

### 朋友圈建设
- **展现个人价值**：通过朋友圈、照片、视频等展现个人特质。
- **对应需求**：根据对方的需求展现相应的特质。

### 吸引过程
1. **朋友圈吸引**：通过朋友圈展现个人价值，吸引对方。
2. **情绪吸引**：在聊天过程中调动对方的情绪，使其产生兴趣。

## 总结
本节课中，我们学习了恋爱流程中的三个阶段以及如何通过提升个人价值来吸引对方。在下一节课中，我们将继续探讨如何建立舒适感和诱惑阶段。
