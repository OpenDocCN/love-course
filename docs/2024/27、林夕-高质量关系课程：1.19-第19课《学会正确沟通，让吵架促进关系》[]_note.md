# 27、林夕-高质量关系课程：1.19-第19课《学会正确沟通，让吵架促进关系》

大家好，我是林夕。今天的课程是关于正确沟通，如何通过吵架来促进关系。所有伴侣都会吵架，学习情商并非为了停止争吵，而是通过更高效的沟通满足双方需求。

**核心概念**：通过冲突管理提升关系质量。

**冲突管理**：
1. **找出冲突**：识别冲突点，如沟通方式不一致。
2. **识别需求**：了解双方需求，如恋爱方式。
3. **解决方案**：制定满足双方需求的方案。
4. **沟通执行**：落实方案，确保执行。

**案例分析**：
学员与男友因沟通方式不同产生冲突。通过沟通，学员了解到男友不喜欢微信聊天，而她喜欢。双方达成共识，决定每天语音半小时。

**立规矩**：
1. **自我陈述**：描述事实和需求。
2. **了解认知**：询问对方的看法。
3. **打碎认知**：通过价值交换，改变对方认知。
4. **建立习惯**：制定规则。

**案例分析**：
学员因异性尺度认知不同与男友产生冲突。通过沟通，学员表达了自己的需求，并提出了等价交换的方案，最终男友接受了她的要求。

**总结**：
本节课教会了如何通过冲突管理提升关系质量，学会驾驭男人。希望大家学会这一节课，享受高质量关系。

**作业**：
1. 写出冲突管理流程和立规矩流程。
2. 分析自己与伴侣之间的冲突，并提出解决方案。
3. 分析自己的价值，并思考如何进行等价交换。
