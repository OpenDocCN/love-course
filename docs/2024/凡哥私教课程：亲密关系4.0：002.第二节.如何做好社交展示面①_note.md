# 凡哥私教课程：亲密关系4.0：002.第二节.如何做好社交展示面①_note

在本节课中，我们将要学习社交展示面的核心概念与重要性，特别是朋友圈的构建方法。我们将从“装逼”与“逼格”的区别讲起，探讨在不同时代背景下，人们如何通过不同的媒介来展示自我价值，并最终聚焦于如何在当下的网络时代，通过照片和朋友圈有效提升个人吸引力。

## 什么是“装逼”与“逼格”？ 🤔

在深入探讨如何构建展示面之前，我们需要先理解两个核心概念：“装逼”和“逼格”。这两个概念是有区别的。

**装逼** 通常指刻意、不自然地炫耀，往往流于表面。
**逼格** 则是指将自己与大众区分开来的标准或标志。逼格越高，在社交的“食物链”中，你的位置就排得越高，从而能产生一种优越感，满足自尊心。

我们本节课主要探讨的是如何提升 **逼格**，而非简单的“装逼”。

## 展示面的演变：从情书到朋友圈 📜➡️📱

上一节我们定义了核心概念，本节中我们来看看展示自我价值的方式是如何随着时代演变的。理解这个演变过程，能帮助我们明白为什么照片在当今如此重要。

### 书信时代：文字与书法的艺术

在没有网络的年代，人们主要通过写信，尤其是情书，来表达爱慕。那时，有两样东西能凸显你的逼格：

以下是提升逼格的两个途径：
1.  **文字表达水平**：即你的语文功底和文学素养。内容的质量直接决定了你的格调。例如，民谣歌手鲍勃·迪伦因其歌词的文学性获得诺贝尔文学奖，而著名作家村上春树却多次陪跑，这说明了内容深度的价值。
    *   **核心公式**：`逼格 ∝ 文字表达水平`
2.  **书法水平**：字迹是那个时代的“脸面”，所谓“字如其人”。一手好字能给人留下良好的第一印象。

### 网络文字时代：QQ空间与非主流

进入网络时代初期（如QQ时代），文字的字体被标准化限制了。于是，人们将心思花在了周边装饰上，以凸显与众不同。

以下是当时提升逼格的方式：
*   **装扮QQ空间**：使用非主流头像、昵称（如“蓝色忧郁”）和个性签名。
*   **使用火星文**：例如将“我”写成“莪”，创建独特的文字风格。

这个阶段，**传播途径**从实体信件变成了网络代码，但**内容载体**依然是文字，本质是通过文字表达内心。

### 图片时代：维度的跃升与快速传播

大约在2008-2009年，情况发生了根本变化。随着数码相机和早期网络图片的普及，**图片**作为一种更高维度的载体出现了。

**核心概念**：`图片的传播效率 >> 文字的传播效率`

图片能提供更直观的视觉冲击，无需过多文字描述。早期的网络红人（如芙蓉姐姐、凤姐）正是凭借大量照片的传播迅速走红。2011年微信朋友圈推出后，随着智能手机摄像功能的普及，每个人都进入了“照片社交”的时代。

### 视频时代：讲述故事的能力

如今，我们进入了视频时代，尤其是短视频时代。

**核心概念**：`视频 = 将连续的照片（画面）组合起来，讲述一段故事`

视频能够提供更丰富、更立体的信息，是比图片更高维度的表达方式。我们要学会用**高维度**的方式（图片、视频）去吸引他人，这就像《三体》中的“降维打击”。

## 如何构建高逼格的朋友圈？ 🎯

了解了展示面的演变史后，我们回到今天的主题：如何在实际操作中，通过朋友圈提升你的逼格和吸引力。这需要我们打好手中的牌，并清晰展示核心价值。

### 第一步：认清自我，打好手中的牌

在发朋友圈之前，首先要进行自我审视：**你是一个什么样的人？你的真实生活是怎样的？**

**核心原则**：`打好你手中的牌`

不要强行营造与自己本质不符的形象。例如，一个内向的人不必刻意模仿外向者的热闹，可以发掘并展示自己内向的深度和文艺属性（如分享一首好听的民谣，配上深思的文字）。一个生活平淡的人，则需用心发现生活中细微的美好。

### 第二步：展示你的核心价值“名片”

朋友圈就是你在网络上的名片。想要有效吸引异性，你的名片上必须展示以下三项核心价值：

以下是朋友圈必须展示的三项核心价值：
1.  **硬价值**：指你的财富、资源和社会地位。这是最直接的价值体现。
    *   **展示技巧**：切忌低端炫富（如直接拍车钥匙、房产证）。应通过场景间接体现，例如：拍一张自家装修精致的客厅，配文“周末午后，一个人的独处”；或展示旅游照片（对方会默默计算行程成本）。
    *   **重要提示**：`硬价值要有，但并非没有就完全无法吸引异性。`
2.  **软价值**：指你的思想、品味、兴趣爱好和生活情趣。在物质充裕的时代，软价值是区分你与他人的关键。
    *   **展示技巧**：分享优质的音乐、电影、书籍读后感；展示有格调的摄影作品（风景、人文）；呈现多样化的生活方式（运动、咖啡、展览）。风格要多变，避免单一。
    *   **拍照技巧**：拍摄文艺风格照片时，`眼睛不要直视镜头`，可以看向远方或给出侧脸，营造思考或沉浸的氛围。
3.  **生育价值**：主要指你的外貌、身材和健康程度。这是最直观的吸引力。
    *   **核心原则**：`可以不帅，但必须干净整洁`。
    *   **提升建议**：
        *   **身高**：与生活圈的同性比较，使用鞋垫并穿中/高帮鞋来优化视觉比例。拍照时采用低角度仰拍。
        *   **发型与面容**：保持发型清爽利落，脸部干净。胡子要么精心修剪，要么刮干净。
        *   **衣着**：穿着合身、有型。避免土气的格子衬衫，选择素色（白、黑、蓝）衬衫。裤子要合身，特别是身高不高的男性，选择修身款能拉长线条。
        *   **拍照雷区**：避免与垃圾桶、路牌等有标准高度的物体合影，以免暴露身高短板。

## 总结 📝

本节课中我们一起学习了社交展示面的核心逻辑与构建方法。

首先，我们明确了 **“逼格”** 是与众不同的自我标识，是吸引力的关键。接着，我们回顾了展示面从**书信（文字/书法）** -> **网络文字（QQ装饰）** -> **图片** -> **视频**的演变历程，理解了**高维度载体（图片/视频）在当代社交中的压倒性优势**。

最后，我们深入探讨了构建高逼格朋友圈的两大步骤：
1.  **向内审视**：`打好手中的牌`，基于真实的自我和生活进行展示。
2.  **向外展示**：将朋友圈作为**个人名片**，系统性地展示 **硬价值**（间接体现）、**软价值**（思想品味）和**生育价值**（干净整洁的外形）这三项核心吸引力要素。

掌握这些原则，你就能在社交网络中有效地塑造个人形象，提升吸引力。下一节课，我们将继续深入展示面的具体拍摄技巧与内容策划。
