# 性魅力提升系列课—无限话题聊天：017-什么是真正的情商？_n

在本节课中，我们将要学习“情商”这一核心概念。我们将探讨情商的本质，它为何重要，以及如何通过识别和控制情绪来提升我们的社交能力。

## 情商的定义

情商指的是一个人识别和掌控情绪的能力。这里包含两个核心点：**识别**与**控制**。

## 识别情绪：理解他人

上一节我们介绍了情商的定义，本节中我们来看看“识别情绪”这一部分。识别情绪是针对他人的。情商高的人在与他人互动时，能够准确接收对方的情绪反馈，从中识别出社交信号（或称“窗口”），并根据这些信号调整自己的言行。

例如，当他看到对方情绪不高时，就不会一直缠着对方聊天，而是懂得适可而止，给对方空间，或尝试调节对方情绪。更直白地说，情商高的人做事不会只顾自己感受，会在意他人感受。

他会考虑对方想表达的真正意图，而非停留在表面话语。这样做能显著降低双方的交流成本，让对方感到相处舒适、没有压力。

以下是基于情绪识别调整行为的例子：

*   当对方不经意提到“最近皮肤很差”时，情商高的人不会拉着对方在太阳下行走，就餐时也不会选择主打辛辣口味的餐厅。
*   这些体贴的举动往往不是通过明面言语达成的。当一个人做到这些，对方会感觉他很贴心、很懂自己。

有人认为提高情商很难，但情商与智商不同，它是可以后天培养的。我们可以通过观察对方的表情、语言、肢体动作和行为，来判断自己的言行是否让对方感到舒适愉快，以及当前的亲密程度是否被接受。

聊天时也是如此。如果识别到了对方给出的“窗口”，就应及时给予回应。若未能识别，将对方晾在一旁，对方会感到自己在浪费感情，觉得与你交流太费劲。

## 控制情绪：管理自我

识别情绪是针对他人的，而控制情绪则是针对自己的。情商高的人懂得控制自己的负面情绪，不让其影响正常生活。

当我们还是孩子时，常把喜怒哀乐挂在脸上。例如，在路上摔一跤，即使没受伤也可能哭闹半天，直到父母答应买玩具才罢休。

现在，如果你在路上摔一跤，还会向父母哭诉吗？肯定不会。别说摔跤，有时连生病做手术都可能瞒着家人。这是因为长大后感觉不到疼痛了吗？并非如此，成年人同样能感知疼痛，但我们学会了控制情绪，不让家人朋友担心，这是心智成熟的表现。

有些男生与人相处时，不能很好控制情绪，容易被紧张、胆怯或激动等情绪影响。你可以将情商理解为一种**高度的社交直觉**。它让我们在与任何人相处时，都能分辨和控制细微的社交信号，从而对当下局势做出判断，在言行举止上拿捏好分寸。

情商在推进人际关系中至关重要。很多社交失败的案例，都源于社交直觉不够敏锐。

一个情商高的男人能识别对方隐藏很深的窗口，这让他在感情道路上顺畅许多。然而，要做到真正的高情商，除了识别他人情绪，控制自我情绪也极为重要。

例如，有些人喜欢在他人面前证明自己，故意说“你在哪里工作？我认识某某领导”或“你知道某某吗？我和他特别熟”。他不断尝试证明自己厉害，但说得越多，反而显得自己越没价值。在对方眼中，这种行为显得幼稚、低端。这不是高价值的展现，而是情商低的表现。

为什么呢？因为**高价值本身无需证明**。你向别人证明自己价值、证明自己情商的过程，本身就是缺乏情商和低价值的表现。因为你控制不好自己的情绪，总想以炫耀或伪装的方式压制别人，这会给他人带来极差的感受。

## 核心要点与行为示例

我们可以将情商的核心总结为一个简单的公式：
**高情商 = 准确识别他人情绪 + 有效控制自我情绪**

以下是课后总结与具体行为指南：

今天我们学习了情商。情商是一个人识别和控制情绪的能力。

**识别**，指的是我们在与他人打交道时，能准确识别对方给我们的情绪反馈，并据此调整后续行动。

例如：
*   约会时，原计划饭后看电影，但对方饭后一直捂着肚子。这时应察觉到需要送对方回家休息，而非坚持看电影。这样对方反而会记在心里，期待下次约会。
*   与一群朋友出去玩时，不要只专注于和一个人交流，显得两人脱离团队。这会让对方尴尬，因为这不是单独约会。应照顾其他朋友的感受。如果你在与对方互动的同时也能融入团队，会给对方一种具有领导力或成熟担当的感觉。

**控制**，指的是对我们自身情绪的控制。我们应尽量不让自己的情绪波动影响正常生活，也不影响他人。

例如：
*   大家开心出游时，你不小心丢了200元现金，随即心情全无，一路黑脸，对谁都不理不睬。这就是让坏情绪影响了他人，是情商低的表现。

## 总结

本节课中我们一起学习了情商的真正含义。情商是一种可培养的**社交直觉**，它包含识别他人情绪和控制自我情绪两大支柱。我们可以通过**与更多人打交道**来不断锻炼和提升这种直觉，从而在社交中更加从容、得体，有效提升个人魅力。
