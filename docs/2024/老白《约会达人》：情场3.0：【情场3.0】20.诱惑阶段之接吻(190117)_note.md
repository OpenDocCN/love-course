在本节课中，我们将要学习诱惑阶段的关键环节——接吻。我们将探讨接吻的前提条件、如何创造合适的环境，以及接吻过程中的具体注意事项。

上一节我们介绍了诱惑阶段的其他环节，本节中我们来看看如何自然地推进到接吻这一步。

## 接吻的前提：WR与FP

接吻并非独立事件，它建立在良好的互动基础上。在L4我们介绍了牵手的预期，在L5我们介绍了废物测试（Waste Test，简称WR）。如果女生在语言和身高上都给了绿灯（O），并且通过了你的WR，说明她对你有想法。

在做WR时，其实已经包含了框架推进（Frame Pushing，简称FP）。此时，FP只需稍微带一点即可。核心规则是：**聊天可以很嗨，但框架推进（FB）要做得简单**。

以下是两个推进框架的对话示例：

*   **示例一：**
    *   男：“你说话这么甜，嘴也应该是甜的吧。”
    *   女：“不告诉你。”
    *   男：“下次尝尝就知道了。”

*   **示例二：**
    *   男：“一会儿你要亲亲我不？”
    *   女：“想得美。”
    *   男：“那还是我亲亲你吧，臭不要脸。”

*   **示例三：**
    *   男：“你今天穿高跟鞋吗？”
    *   女：“穿啊。”
    *   男：“那你今天不用垫着脚尖也能亲到我了。”

*   **示例四：**
    *   男：“你今天穿高跟鞋吗？”
    *   女：“不穿。”
    *   男：“那看来你今天要垫着脚才能亲到我了。”

**这些对话的核心目的是测试女生对我们的反馈。** 如果她还没有准备好，我们就不适合在约会中安排接吻（SR）。

## 如何创造接吻机会

成功接吻需要满足几个条件。第一，必须在完成WR和FP之后。第二，要有合适的环境。第三，接吻本身不难，难的是跟一个对你没有想法的人接吻。第四，需要合适的语境和环境空间。

**其中，环境设计至关重要。** 在约会设计时，必须考虑到足够的私密空间。有时女生是愿意的，但因为环境不合适，她也不会答应。

以下是适合与不适合接吻的环境举例：

*   **适合的环境：** 学校的情人坡、情人湾、人少的电影院、私人影院。
*   **不适合的环境：** 餐厅、咖啡厅、大街上。

## 接吻的八大注意事项

接吻时，细节决定体验。以下是需要注意的八个要点。

**第一，保持口腔清新。** 约会前好好刷牙，不要吃气味大的食物，出门前嚼口香糖是必备步骤。

**第二，动作温柔。** 动作不宜过大，要温柔，不要用牙齿碰到对方。接吻前可以先润湿自己的嘴唇。

**第三，控制时长。** 时间不宜过长，十几秒到半分钟即可。

**第四，把握频率。** 不要在第一次接吻成功后就不停地接吻。一次约会中，接吻不超过一两次即可。

**第五，姿势优雅。** 姿势要舒服、自然、有男子气概，不要显得猥琐。

**第六，保持主导与从容。** 在S2（第二阶段亲密接触）过程中，男生是主导者。不要显得比女生还害羞或不自然。你的平静、从容和幽默会给她很舒服的感觉；反之，紧张和不自信会让她感觉很差。

**第七，善后处理。** S2之后，不要让气氛尴尬。可以和她开玩笑，也可以走心交流，最好两者结合。

例如，你可以说：“你的嘴唇好软啊。” 或者开玩笑说：“来，擦一下你刚才留下的口水。” 这时女生可能会打闹、害羞或胡思乱想，你可以再一本正经地告诉她：“和你在一起，感觉真好。” 然后继续前面的约会，该说什么说什么，该做什么做什么。

**第八，管好双手。** 手上不要有小动作，不要去摸女生的敏感部位。你可以把手放在她的腰上环抱着她，或者一只手扶着她的后脑。
