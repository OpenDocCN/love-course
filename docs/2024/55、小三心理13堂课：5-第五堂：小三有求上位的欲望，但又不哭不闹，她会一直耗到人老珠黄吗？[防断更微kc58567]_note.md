# 小三心理十三堂课：第五堂 - 小三有求上位的欲望，但又不哭不闹，她会一直耗到人老珠黄吗？

## 概述
在本节课中，我们将探讨小三的心理状态，特别是她们有求上位的欲望，但又不哭不闹的原因，以及这种状态可能带来的后果。

## 小三求上位的欲望
**以下是导致小三有上位欲望的五个方面：**

1. **感受到爱**：小三可能因为感受到已婚男人的爱而想要上位。这种爱可能表现为浪漫的约会、甜言蜜语、礼物和性爱等。
2. **感觉有希望**：小三可能认为有希望上位，尤其是当已婚男人表现出离婚的意愿时。
3. **对婚姻和家庭有强烈期待**：一些小三可能因为自己的不幸家庭经历，渴望拥有自己的家庭。
4. **付出太多**：小三可能因为对已婚男人的付出太多，而想要结婚。
5. **爱得深沉**：小三可能因为对已婚男人的爱非常深沉，而想要与他结婚。

## 小三不哭不闹的原因
**以下是导致小三不哭不闹的七个原因：**

1. **哭过闹过没有用**：小三可能因为之前的努力没有效果，而变得习以为常。
2. **被伤透**：小三可能因为被已婚男人或前任伤害，而不想再结婚。
3. **盲目相信男人**：小三可能盲目相信已婚男人的承诺。
4. **付出太多，怕分手**：小三可能因为付出太多，而不敢逼已婚男人离婚。
5. **沉默成本太大，不甘心**：小三可能因为投入太多，而不甘心放弃。
6. **看透了男人的嘴脸，看清楚了婚姻的本质**：小三可能因为看透了男人的本质，而不想成为妻子。
7. **盲目自信**：小三可能因为自信自己能够搞定已婚男人，而不哭不闹。

## 小三耗下去的后果
**以下是小三耗下去可能带来的后果：**

1. **一直都没有名分**：小三可能一直得不到已婚男人的承诺。
2. **耗腻了**：已婚男人可能对小三感到厌倦。
3. **耗的过程中忍受和压抑**：小三可能因为耗的过程而变得脾气暴躁。
4. **耗出心理疾病**：小三可能因为压力过大而患上心理疾病。
5. **耗出孩子**：小三可能因为孩子而更加难以离开已婚男人。
6. **耗掉了青春，错过了结婚**：小三可能因为耗下去而错过结婚的最佳时机。
7. **耗成了婚**：小三可能因为上位而结婚，但婚姻幸福的概率很低。

## 总结
本节课中，我们探讨了小三的心理状态，特别是她们有求上位的欲望，但又不哭不闹的原因，以及这种状态可能带来的后果。希望第三者能够认识到这些后果，并做出明智的选择。
