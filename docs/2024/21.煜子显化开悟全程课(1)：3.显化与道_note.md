# 课程21：煜子显化开悟全程课(1) - 3. 显化与道 🌀

在本节课中，我们将探讨“道”的核心概念及其与显化法则的深层联系。我们将从“太极”这一基本概念出发，逐步理解宇宙的构成原理、意识的角色，以及如何将这些古老的智慧应用于现代显化实践。

## 什么是太极？⚫⚪

首先，我们来理解“太极”的含义。“太”代表无限大，“极”代表无限小。太极，就是一个无限大的圆与一个无限小的圆相互融合的状态。

一个无限大的圆，加上一个无限小的圆，并且它们互相融合，就叫太极。它代表了无限与零之间存在的不同空间层次。我们通常所见的太极阴阳图，描绘的就是一个无限大的圆和一个无限小的圆在当下空间中的性质。

这里需要理解一个核心观点：**没有绝对的大与小**。这个世界没有这些分别。举个例子，我们现在所见的地板上最微小的灰尘粒子，它包含了整个宇宙的全部智慧。因此，即便是一粒灰尘，也是“本自具足”的。它可以不是灰尘，它可以是任何东西。并非我们强行定义它（例如指鹿为马），而是因为我们的“心”挂在了这里，我们的意识驻留于此，所以此刻它呈现为灰尘。

我们的意识，包括构成我们肉体的分子、原子等一切物质，都是基本粒子。它们完全是被塑造的，或者说，因为“心”驻留于此，它们才长成了现在的样子。

## 无限源于无限小 🌌

接下来要讲的是，太极中无限大的东西都出自于一个无限小的内容。这就是为什么有“宇宙大爆炸”的理论——宇宙最初就是一个无法用肉眼观测到的、极其微小的基点。

无限大的东西正是从这个无限小的内容中迸发出来的，或者说，它在“此刻”就已经完成了。即便我们谈论宇宙大爆炸，它的结局也早已存在。结局并非由我们头脑思考的心所决定，而是由潜意识的心所定。后续我们会用图表说明，你的“心”究竟是哪一个。

绝对不是我们现在用头脑去思考、去分辨的那个心，而是在完全放下分辨之后得出的那个心。

## 天人合一的真义 🙏

将“无限大”与“无限小”两者融为一体，就是太极。当我们的内在维度——也就是我所谓的“心”——提升到无穷大时，才能真正理解“天人合一”。

之前我们提到过能量层级（例如7级）。7除以无穷大等于0，88除以无穷大也等于0。但是，**无穷大除以无穷大等于1**。这才是真正的“天人合一”。我们常说的天人合一，只要你的内在不是那个“无穷”，你就没有达到天人合一的境界。

我们每一个当下都是一个质点。此刻我在这个维度，下一秒我转变念头，就在其他当下。但我们每一个当下的每一个质点，都包含了整个宇宙的智慧，这就是“本自具足”的原因。并非强行告诉自己“我已经圆满、好幸福”，那是一种虚假的自我洗脑，因为你并未真正达到那个状态。

但这个状态并非需要通过修行、吃素、戒除欲望才能达到。这个问题很早就被察觉了。在我还不懂修行或“道”的时候，我就发现有一个现象叫“屁股决定脑袋”。很多人认为自己很聪明，但那是一种头脑的聪明。真正的智慧在于，你对任何事物都知道答案，没有任何困惑。开悟就是你对任何东西都不会再有困惑。

所谓的智慧，在于你的位置决定了你的聪明程度。举个例子，让任何一个人立刻去当市长，处理政务，只要他把自己的“心”放在那个位置上，他绝对能管理得很好。任何人都可以，并非必须通过学习或前人经验总结。任何人，只要你的心处于那个状态，你就能做到。

## 显化的核心：处于终点 🎯

这与显化息息相关。为什么强调要在“终点”？这是有道理的。你的心在终点，你就是那个人。而不是去攀登高峰、祈求别人。显化他人其实是一种“求”，那会显化出你“求”的状态，而非“已是”的状态。

如果你动了一个念头“让他来找我”，那你永远都处于现在这个“求”的状态，你没有改变。例如，你现在处于某个状态，你能看见你的SP（特定的人），这说明这个“相”已经出现，你们处于同一频率。你能看见他，其实你已经将他“想要”出来，只是你们之间尚未发生喜欢或不喜欢的关系。

在人类社会中，喜欢与否在于比较。任何显化，比如我头脑意识支配我去吃饭，因为我饿了，这也是一种显化。如果我让你把卡转交给Alen，那么你就是我显化过程的中间态。

举个例子，学生时代，很多看似条件不好的男生，如果你看不上他，你只需对他说一句“来跟我谈恋爱吧”，他绝对会同意（只要他没有女朋友且自认不如你）。因为你在上，他在下。你内心有点瞧不起他，你可以显化任何你瞧不起的人。

但为什么你和SP本是平级，当你想要“喜欢他”并去“求”他时，你瞬间就处于他之下了。一个之下的频率怎么可能与之上的频率产生共振？后面我会讲我是如何显化的。两条波根本不会成像。干涉成像需要同频，才能在屏幕上产生实像（即“色不异空”的“色”）。你处于“求”的频率，产生的是“求”的像；他处于他的频率，产生他的像，两者无法结合。

因此我发现，很多来向我咨询显化SP的人，问题大多出在自己身上。不是方法问题，而是状态问题。有些人第一次看完我的帖子，马上就显化成功了。例如，上次在青城山遇到的一个女生，她后来给我报喜说用我的方法显化成功了。我发现她能成功，是因为她内在本来就很强大。她给人的感觉是“有点东西”，并不自卑，会觉得那个男人不怎么样，可以随意拿捏。所以她动个念，那个人就来了。

因为人性中有一种“奴性”思维，倾向于听从别人，就像SM关系中的绝对服从与掌控。那个女生最初认识时，她总是“姐姐、姐姐”地夸人，我以为她不太行。后来我发现她的内心是一头狼，外在只是伪装。我告诉她，要把真正的内在放出来。从青城山回去的车上，我带她进入“观察者”状态聊了一个小时，她马上变了一个人，能量变得非常高，谈话完全超越了个人层面，充满了灵性与智慧。

后来她吃了一块牛肉干，突然又说：“姐姐，你这次来真有收获，还收获了一枚小迷妹。”我立刻对她说：“你怎么回去了？”她自己随即也意识到，人在吃饭、喝水、上厕所时，频率会变化。本来高频的状态，因为一块牛肉干，她又缩回了那个想要讨好我的“小迷妹”状态。我说：“你不是这个样子的，每个人都是本自具足、平等的，哪有什么谁讨好谁。”她这才意识到。从那天回去后，她昨天就告诉我显化成功了。

## 大道至简原理 📐

讲完太极，我们来讲“大道至简”。这个“简”不只是简单，而是“万法归宗”。各种术法，都是别人悟道后创造出来的东西。就像《通灵之王》里一位道士说的：“若有为令者，天律不容卿。”为什么你可以指挥天兵天将？一个凡人凭什么？是因为他的状态或频率在那之上，或者他直接调用更高能量（如三清、元始天尊的能量）。但其实，他自己就是元始天尊。我们每个人也可以是佛陀本身，当频率在那之上时，可以调动任何能量。

我之后可以分享一些出马经历。我发现，你越弱，有些仙家甚至会骗你、搞你；但你越强，则不同。我曾经的“弱”，是因为从小被教育要善良、无私，对人非常无私。但后来发现，这是被规训后的结果。我的底层代码并非如此。当我挖掘出自己的底层代码后，整个世界都变了。

我让大家提供生辰八字，就是想看看你们的底层代码，以及冲破之后可能拥有的天赋。有些人金钱物质匮乏，问题往往不在物质本身，而在于对个人价值的认定。我过去也不认为个人价值很高，同时也不认为别人的价值很高。这种心态导致我只能做与个人无关的事情。最初写小红书，很多人白嫖我，这也是我自己“写”出来的——我给出的信息或频率是一种“都来白嫖我吧，我是无限的”假象。

但当我冲破底层代码，真正开始欣赏、认可他人价值，也认可自我价值后，一切都变了。我变得勇敢，从“被占便宜”转变为可以“占别人便宜”，而且别人乐于让我占便宜，对我很好。包括Alen也是。我能够拥有许多以前无论如何都求不来的东西，现在别人主动给予，并且非常开心。

原本收不回来的钱，比如离婚赡养费、债务，别人敢不还，是因为他们敢于践踏那些对自我价值认知不清的“好人”。借钱给这种人，往往收不回来，因为对方完全不把他当回事。

## 道的数学模型：正弦波与干涉 📈

大道至简，其实是“简谐波”。也就是正弦函数 **`y = sin(x)`**。

你可以看到这个波形，也就是太极。上半部分记为A，下半部分记为B。我们先说“道生一”。任何东西都只是频率，任何一个粒子、一句话、一个状态、一个念头，都是一个波。这个波分为A（阳）、B（阴）两部分。任何事物都阴阳相生。

那么 **`A + B = 1`**，这就是“道生一”。任何事物都分为阴阳两部分。如果只有阴或只有阳，都无法独立存在。即使是皇帝，也有内在阴性的一面；看似阳刚的我，也有非常阴的一面。女生也有阳性，男生也有柔和的一面。所以 **`A + B = 1`**，然后“太极生两仪”。

两个正弦波叠加才会产生干涉条纹。我们这个宇宙，如果只有一个波，我们看不见它，也没有任何反应。这就是为什么有人天天说“什么时候去死”，但他并没有死，因为需要两个波。

两个正弦波，就是“一生二”。 **`(A + B)^2 = 1`**。

展开为 **`A^2 + 2AB + B^2 = 1`**。

我们写成 **`A^2 + AB + BA + B^2 = 1`**。

现在，**`A^2`** 称为“太阳”（并非指天体太阳，但命名源于此），**`B^2`** 称为“太阴”。**`AB`**（A在前，B在后）称为“少阳”，**`BA`** 称为“少阴”。这就是“两仪生四象”。

第三个能量波至关重要，那就是观测者的意识波。观测者是有“心念”的，“心”就是分别。宇宙最初的粒子在寻找正负粒子结合时，就在做选择。任何东西如果不结合，就无法独立存在。所以这个世界只有公母、白天黑夜、太阳月亮，是二进制系统。正因为简单，它有两极，所以人生总是起起落落。

**`(A + B)^2 = 1`** 代表两仪生四象。你会发现，**`A^2`**（火）最阳，**`AB`**（木）是阳性增加、阴性减少的上升状态，木生火。火居中，土居中央（每个人出生地即为土）。土生金，**`BA`**（少阴，金）是阳气下降、阴气上升的状态。金生水，**`B^2`**（水）最阴。这就是两仪生四象，对应五行。

任何事物的发展都不是直线。古人很有智慧，比如“长生十二局”中的“帝旺”是巅峰。但最好不要达到“帝旺”，因为到达之后就开始衰败。要保持在“临官”状态，一直向上冲，运用这部分能量接近但不超过帝旺。一旦到达帝旺，就开始下坡。

听了这几天的课，你们可以去思考（或不用思考，直接显化）。你会得到机会，创造财富时要明白：很多大钱不是一辈子赚来的，可能一两年就达到顶点，赚到足以退休的财富，一步到位。之后，如果已经在走下坡路还去折腾投资、数字货币、艺术品，这些钱都会败光。不如在帝旺之后守住财富，哪怕存银行、旅游都可以。

## 观测者之心与八卦 🧠

接下来讲第三个能量波——观测者的意识，也就是“心”。任何粒子从一开始就在做选择，寻找与自己匹配的。这就是“分别心”的来源。分别心源于选择。即使是母狮子，也会选择强壮的公狮子。这个心就是选择心、分别心。

每个人看待事物的角度都不同。例如，一个家暴的社会新闻，女生们非常生气，上网骂；男生们则骂“田园女权”；还有一波人从商业、新闻角度看待，认为是大新闻，能冲头条赚钱；再有一波人（或许为了压其他新闻）角度又不同。每个人物角色决定了他的格局。一个担心自己未来被家暴的女生，格局仅限于自身；一个想借势炒作的人，格局则不同。因此，事情没有绝对好坏，只在于你观看时的身份。

## 二生三，三生万物 🔢

现在讲“二生三”。宇宙本质是两条波，现在加上观测者的波纹，共三条。

三条波叠加呈现的像，即 **`(A + B)^3 = 1`**。

展开为 **`A^3 + 3A^2B + 3AB^2 + B^3 = 1`**。

我们将其写为：
**`A^3 + AAB + ABA + BAA + ABB + BAB + BBA + B^3 = 1`**。

“二生三”即 **`(A + B)^3 = 1`**。

将阳（A）记为一横（—），阴（B）记为两短横（--）。那么：
*   **`A^3`**（三横）为乾卦 ☰，代表天、男。
*   **`B^3`**（六短横）为坤卦 ☷，代表地、女。
*   中间六种组合（AAB, ABA, BAA, ABB, BAB, BBA）分别对应兑、离、震、巽、坎、艮卦。

这就是“四象生八卦”。这八个象可以代表物质界的任何事物，并非定义，而是描述。

## 命运与自由意志 ⚖️

八个物质象和八个信息象（第三个能量波带来的主观定义），结合产生64卦，可以定义人生中发生的一切，没有例外。即便是嫦娥被外星人抓走，也能从64卦中推算出来。

但这并非让你相信“命定论”。本来我们是“无”（零），物质界的命定论可以看作一种可能。如果你相信命定论，你的人生就是命定论。但如果你既相信命定论、给人算命，同时又玩显化，那就是悖论。因为命定论代表一眼看到头、不可改变；显化则代表可以改变。这是两套冲突的操作系统。

然而，命定论可以检验你当下的状态。比如今天线上两位朋友抽卡牌，抽到什么牌是可以算出来的。甚至给定时间，某人会问什么问题、答案是什么，也能算出来。但当你玩显化，转变念头后，再用命定论去算，结果就变了。所以不要将算命结果当作不可更改的未来走向。

为什么尽量不要过多观测？因为如果你认为对方是“大师”，他说一句话，你会相信那是你的未来。如果你不变念，这个预言就可能自我实现。例如，大师说你考不上研究生，你当下的状态确实不行。即便你试图显化，潜意识里也会记得“大师说你考不上”，最终结果可能真的考不上，然后你会觉得大师算得真准。其实，可能是我的定义让你考不上。

我出马时给人看事非常准，连对方隐私、未检查的妇科病都能看出来。但最好别给人看，因为某种程度上是在“胡说八道”。只要有人来找我看，我接收到的任何信息（可能来自我过去的认知）都可以用来“算”。例如，我在B站看过韩国邪教的帖子，记住了。后来有人问我，她在韩国约会对象有没有艾滋病。我感应到对方家里有像神社的东西，就说“有点像邪教”。结果反馈说，对方果然拉她入教。我当时觉得自己算得真准，但后来发现，只要有人来找我看，我“胡说八道”都可能准，因为每个当下，我的念头接收了信息。只要我认为是准的，它就是准的。连抛硬币都可以根据你的心改变，还有什么不能改变呢？

因此，人生中遇到的每个人，都可能是在告诉你近期的状态。比如有人找你借钱，你可能要反思是否太嘚瑟或炫耀。我曾过于相信父亲的话“千万不要借钱，会破财”，结果莫名其妙有人借，我借了，心想“完了，要破财”。第二天母亲找我要15万，我打了钱，觉得烦，但怪自己过于相信父亲的话，一语成谶。

## 总结：你创造你的现实 🎨

最后总结，第三个能量波——观察者的意识波——至关重要。因为观察者是你自身发出的能量波，每个人的波都不同，由你的认知、频率和过往记忆组成。因此，我们看到的一切，实际上是我们认知的投影。我们看见的所有“相”，确实是你认知投射出来的结果。

道家思想讲完，我想说：古代皇帝追求成仙，并非愚蠢。所谓“成仙”，是维度的提升。人对蚂蚁来说就是神，对细菌来说也是神。当环境适宜时，生物会选择繁衍（传宗接代）；当环境不适宜时，生物会选择永生。任何生物都是如此。

永生不是横向寻找答案（搜索古籍、百度、找可见的道士）。你能见到什么水平的大师，取决于你内在的能量和高度。你能显化到什么人，看到什么书（如《墨子》、《登真隐诀》），都是你自己显化来的。为什么很多人没看到？因为他们还没到那个状态。

皇帝找道士炼丹、吃长生不老药没用，因为你在当前维度能见到的一切，都无法达到更高维度的效果。永生只能纵向寻找，即向上攀登，提升维度（“身为”）。但“身为”不如直接到达“登真”状态。如果还在追求“身为”，就在定义“仙”的状态。仙对人来说高高在上，但对其他仙来说，他什么都不是，仍是普通人。仙与仙之间仍有竞争。

佛教说“永无止境”，只要还有念头、分别心、登高望远的欲望，就永无止境。作为人，有必要向蚂蚁证明自己更厉害吗？一脚踩死它，它也不知道。当你成为仙，也没必要跟人较劲，但你周围的仙都跟你差不多。当你成神，能呼风唤雨，你会发现身边的神可能比你更厉害，你们在不同维度，彼此看不见。

只有完全放下，不挂碍任何一点，成为“无限”，或者说“什么都不是”的时候，才是究竟。就像博士学位，本科生写论文是对原理的理解和应用；硕士生（菩萨）写论文是超级应用与创新；博士生写论文是重新定义或创造学科。真正的向上攀登，是去创造和定义。
