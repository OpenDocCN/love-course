# 梵公子 情绪价值核心课【向上社交破圈升思维】：4.社交媒体个人IP打造-线上如何破圈_ev

## 概述
在本节课中，我们将学习如何通过社交媒体打造个人IP，实现线上破圈。

## 为什么打造个人IP
**公式**：优质男性/大佬接触途径 = 线上途径 + 线下途径

- 线上途径：抖音、小红书等全国性平台
- 线下途径：受地域限制，掌控较少

## 打造个人IP的关键
1. **展示自我**：通过拍照或拍视频展示最好的一面。
2. **模仿成功案例**：在小红书或抖音上寻找风格相似、粉丝众多的账号进行模仿。
3. **蹭流量**：结合当下热门话题或活动，增加曝光度。

## 拍照前的准备工作
1. **服装道具**：提前构思服装、道具等。
2. **化妆**：如果化妆技术不佳，可找化妆师帮忙。
3. **对标账号**：模仿成功案例，包括发型、妆容、穿搭等。

## 照片与视频的选择
- 展示身材：使用视频
- 展示颜值：使用照片

## 优势放大
- 扬长避短：展示自身优势，避免暴露缺点。
- 无限放大：挖掘自身亮点，无限放大优势。

## 社交媒体打造个人IP的核心点
- **曝光量**：让更多人看到你的拍照技术、美感或网感。
- **持续拍摄**：通过长期拍摄提升技能。
- **投资**：适当投资于推广，如抖加等。

## 总结
本节课中，我们一起学习了如何通过社交媒体打造个人IP，实现线上破圈。希望你能将所学知识应用到实践中，打造出属于自己的个人品牌。
