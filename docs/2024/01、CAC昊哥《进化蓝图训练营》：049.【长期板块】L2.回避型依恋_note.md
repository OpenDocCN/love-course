## 重温依恋的判断标准

在上一节课中，我们介绍了在80分以上阶段如何判断依恋。本节中，我们先来重温一下依恋的四个核心判断标准。

CAC方法的核心是准确判断女生每时每刻的态度。当讨论长期关系时，首要问题是判断双方是否真正产生了心理上的依恋。

以下是判断是否进入依恋阶段的四个标准：

1.  **强烈的情感连接**：例如，想到分手或分离时，双方都会感到非常难受、心里不舒服，甚至恐惧。
2.  **相互回应**：双方都愿意积极回应彼此的分享，而不是只有一方有分享欲，另一方始终冷淡。
3.  **唯一性**：进入依恋阶段后，你会发现只有这一个特定的人能给你心灵上的满足感、安全感和归属感，其他人无法替代。
4.  **纯粹的情感关系**：你们在一起的原因不再是对方符合你预设的性格或物质标准，而是因为“她就是她”。即使她不符合你的条条框框，你依然强烈地想要与她融合，获得归属感。

## 依恋风格与回避型依恋

上一节我们介绍了依恋的判断标准，本节中我们来看看不同的依恋风格如何影响恋爱关系。

依恋风格是一个人在成长过程中，由原生家庭、过往恋爱经历以及天生的气质类型共同塑造的，它决定了其在与所爱之人相处时的特定模式。

在上一节课中，我们提到了最常见的五种依恋风格。本节课，我们将重点讲解前两种：**安全型依恋**和**回避型依恋**。

### 安全型依恋案例 🧸

为了让理论更直观，我们直接观看一个安全型依恋的心理学实验片段。

在这个片段中，孩子与母亲（穿粉色针织衫）在一起。孩子的注意力有时在母亲身上，有时会去探索周围环境。玩一会儿后，他会向母亲寻求互动。

当母亲离开房间后，孩子感到慌张，体验到分离焦虑，开始哭泣并试图寻找母亲。

陌生人（绿衣女子）进入房间试图安抚孩子，但孩子仍在哭泣，并未被哄好。

当母亲返回房间并与孩子打招呼、提供安慰后，孩子立刻要求拥抱。情绪被安抚后，在母亲的引导下，孩子的注意力重新回到了玩具上。

**这个案例的核心要点是**：
*   分离会引发孩子的焦虑和恐惧。
*   陌生人的安抚效果有限。
*   依恋对象（母亲）的回归能迅速恢复孩子的安全感，使其情绪平复并重新投入探索。

### 回避型依恋案例 🚫

接下来，我们观看第二个案例，这是一个回避型依恋的典型表现。

在这个片段中，同样是母亲离开房间，陌生人进来照顾孩子。

孩子没有因为母亲的离开而表现出恐慌，反而继续玩玩具。

母亲回来后尝试安慰孩子，但孩子却故意把头扭开，不看母亲。

母亲再次离开，孩子独自留在房间。他回头看了看，发现人不见了，但随后又继续玩玩具，没有哭闹。

陌生人再次进入房间陪孩子玩，孩子没有抗拒，继续与陌生人互动。

母亲又一次返回并尝试安抚孩子，想引导他玩玩具，但孩子再次故意把头扭开，躲避母亲的接触。

**对比两个案例，我们可以总结**：
*   **安全型依恋**：分离带来焦虑，重聚带来喜悦和亲密拥抱。
*   **回避型依恋**：表现为“过度乖巧”。分离时看似淡定，重聚时却故意回避。这并非不在乎，而是一种心理防御机制。其背后的逻辑是：“如果我不靠近你、不依赖你，那么当你再次离开时，我就不会那么受伤。”这是一种通过主动推开对方来避免预期中伤害的行为模式。

## 如何判断回避型依恋

理解了回避型依恋的核心逻辑后，我们来看看在恋爱关系中如何具体判断。

回避行为可以理解为“故意躲开”，其背后的深层逻辑是**害怕自己“不配”被爱**，或者**认为自己“不值得”被爱**。这种信念导致个体认为，只要进入一段依恋关系，最终注定会被抛弃。因此，为了不受伤，他们选择不走心；为了不被抛弃，他们选择不去需要对方。

**回避型依恋的典型想法可以用以下公式概括**：
`为了避免受伤 -> 我选择不走心`
`为了避免被抛弃 -> 我选择不去需要`

**容易形成回避型依恋的家庭环境通常包括**：
*   物化孩子、忽视孩子情感需求的家庭。
*   父母常年在外，孩子经常独自在家的家庭。
*   父母自身就是回避型依恋风格（可能被继承）。

在恋爱中，判断自己或对方是否有回避型行为，可以观察以下五个要点：

1.  **害怕和逃避恋爱**：并非“不想和你谈”，而是可能因过往情伤或原生家庭忽视，导致“不敢谈”，恐惧亲密关系。
2.  **忽冷忽热**：关系看似要升温时突然后撤，当你心灰意冷想离开时又试图拉回。**注意**：需与“因吸引不足导致的忽冷忽热”区分。真正的回避型忽冷忽热通常发生在双方已有一定好感（如达到CAC分数60以上的喜欢）或已发生亲密关系之后。
3.  **不愿建立承诺**：即使关系亲密（包括发生关系），当你提出确定关系或长期发展时，对方会逃避、后退。
4.  **不承认亲密**：即使已有实质亲密行为，当被问及关系时，对方仍用“特殊的朋友/兄妹”等说法来回避定义关系。
5.  **对异性关系态度随意**：除了你之外，仍与他人保持暧昧或亲密接触，类似于案例中孩子回避母亲却与陌生人玩耍的行为。

**请注意**：回避型依恋与心理学诊断中的“回避型人格障碍”并不完全相同。本节课讨论的范围仅限于恋爱关系领域。

## 回避型依恋的常见错误应对

在识别出回避型依恋后，许多人会采取错误的应对方式，反而加剧问题。以下是四种常见的错误做法：

1.  **强迫与施压**：对女生下最后通牒，逼迫对方表态（如“必须给我个说法”）。这是心态爆炸、无法承受压力的表现。
2.  **情绪失控（破防）**：因对方的回避而歇斯底里，情绪过度冲动。
3.  **反复追问与怀疑**：因自身缺乏安全感，不断质问对方是否喜欢自己、是否生气，这会传递出不信任感，并削弱自己的自信。
4.  **低位讨好（舔）**：误以为对方需要安全感，便以慌张、低位的姿态去过度关心和夸赞。然而，女生只能从高位的伴侣那里获得安全感，低位的讨好只会让她觉得你“接不住”她的情绪，可能导致她真正下头。

**重要提醒**：面对回避型伴侣，你需要为自己设立清晰的止损线。理解与包容是必要的，但不应以严重内耗和自我牺牲为代价。恋爱的首要目的是让双方都变得更好，优先保证自己的身心健康。

## 回避型依恋的正确应对策略

说完了错误的做法，接下来我们看看正确的应对策略。正确的应对与前述错误恰恰相反。

1.  **保持情绪稳定，避免负面反馈（无波动）**：当对方表现冷淡时，要有清晰的认知——这不是不爱，而是其回避机制在起作用。恰恰因为对你有依恋和期待，害怕未来受伤，才会推开你。你的稳定是打破其预期的关键。
2.  **给予耐心、时间和空间**：在你能接受的范围内，允许对方有一定的回避空间。避免逼迫和最后通牒，展现你能够承担关系中的压力。
3.  **坚持主导和带领**：即使对方后撤，你仍需按照短期框架中教授的方法，主动但不过度地推进关系，保持节奏感。
4.  **进行高位认可**：这是最关键的一点。通过框架、筛选等方式，从高位给予对方认可。这能让觉得自己“不配被爱”的对方感受到：“我得到了这个高价值男人的认可，我是值得的。”从而获得真正有效的安全感。就像员工更看重上司的安慰一样，高位认可的价值远超低位讨好。

## 课程总结 🎯

本节课中，我们一起学习了安全型依恋与回避型依恋。

*   **安全型依恋**：分离有焦虑，重聚即安心，能健康地表达需求和亲密。
*   **回避型依恋**：核心是**因害怕被抛弃而主动推开对方**，其信念是“我不配/不值得被爱”。在关系中表现为逃避承诺、忽冷忽热、不承认关系等。

面对回避型依恋的伴侣，最关键的是避免因自身恐慌而做出强迫、情绪失控、反复追问或低位讨好的错误行为。这些行为要么会加剧对方的不配得感，要么会暴露你的低价值导致吸引下降。

正确的应对策略是：**保持自身情绪稳定，给予耐心，坚持主导框架，并从高位给予对方真诚的认可**。
