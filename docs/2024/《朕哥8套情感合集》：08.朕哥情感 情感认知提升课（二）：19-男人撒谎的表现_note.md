# 《朕哥8套情感合集》：08.朕哥情感 情感认知提升课（二）：19-男人撒谎的表现

## 概述

在本节课中，我们将探讨男人撒谎的各种表现，并学习如何应对这些情况。

## 男人撒谎的表现

### 1. 装傻加回避

**表现**： 当你问他问题时，他装傻、假装没听见或听不懂，然后回避问题。

**应对**： 
- 判断他是否在回避问题。
- 如果他在回避，可以委婉引导，与他博弈。

### 2. 避重就轻加讨好

**表现**： 他会避开问题，同时讨好你，让你觉得他还在乎你。

**应对**：
- 调教立规矩，告诉他你不会接受这种讨好。
- 拒收讨好，让他知道你不会因为讨好而改变态度。

### 3. 否认

**表现**： 他会否认错误，情绪激动，试图让你相信他没有错。

**应对**：
- 表现更生气，不要被他情绪激动的表现所影响。
- 不要轻易认错，让他知道你不会因为他的情绪而改变态度。

### 4. 反向指责

**表现**： 他会把错误归结到你身上，并表现出愤怒。

**应对**：
- 解释加安抚，让他知道你理解他的情绪，但不会接受他的指责。
- 安抚他，让他冷静下来，然后进行进一步的对话。

### 5. 承认错误，甩锅

**表现**： 他会承认错误，但会把责任推给别人。

**应对**：
- 将计就计，示爱加紧盯，让他知道你不会轻易原谅他的错误。

## 总结

本节课中，我们学习了男人撒谎的各种表现和应对方法。了解这些可以帮助我们更好地应对情感问题，维护自己的权益。
