# 林老头《私人传承》：012.聊天大忌：对立

## 概述
在本节课中，我们将探讨聊天中的大忌——对立，以及如何避免这种负面情绪，提升沟通效果。

## 聊天大忌：对立
### 价值匹配问题
- 喜欢的人聊不起来，不喜欢的人却聊得火热。
- 核心问题在于价值匹配，即你展示的价值与对方期望的不匹配。

### 线上课程弊端
- 拍照、形象打造需要时间，难以短期速成。
- 线下课程可以快速提升个人价值，解决受众问题。

### 聊天模式与反馈
- 聊天火热说明模式运转良好，对方给予反馈和配合。
- 对方不愿意分享生活，可以聊其他话题，避免强行带入自己的逻辑。

### 基础价值
- 外形、财富价值（上进心、工作、未来潜力）、情绪价值（理解、关心、信任）。
- 三个价值都不可或缺，但情绪价值在长期关系中尤为重要。

### 情绪价值理解
- 情绪价值理解很重要，但学员中很少有人理解到位。
- 情感能力可以帮助你获得资源，提升个人价值。

### 实战经验
- 实战经验可以帮助你更好地理解情绪价值，提升沟通效果。
- 不断评估对方的兴趣和反馈，根据反馈调整沟通方式。

### 聊天技巧
- 分享生活时，夸张情绪和态度，让对方感受到你的热情。
- 如果对方不喜欢网上聊天，可以约其出来，提前邀约。

### 实战经验的重要性
- 实战经验可以帮助你更好地理解对方的想法，避免冲突。
- 实战经验可以帮助你更好地判断对方的兴趣，调整沟通方式。

### 奉承与提问
- 最高级的奉承是提出相反概念，但能够接纳并理解对方。
- 通过提问引导对方表达观点，避免冲突。

### 实战经验与理论
- 实战经验可以帮助你更好地理解理论，避免教条主义。
- 实战经验可以帮助你更好地判断对方的想法，避免冲突。

### 网络聊天与线下约会
- 网络聊天是大趋势，但线下约会仍然重要。
- 网络聊天效果不佳时，可以尝试线下约会。

### 线下课程
- 线下课程可以帮助你快速提升个人价值，解决聊天问题。
- 线下课程可以帮助你更好地理解情绪价值，提升沟通效果。

### 长期关系与结婚
- 长期关系需要门票，即外形、情绪价值等。
- 结婚需要慎重考虑，避免冲动。
- 长期关系不等于结婚，需要发展规划。

### 总结
本节课中，我们学习了聊天大忌——对立，以及如何避免这种负面情绪，提升沟通效果。希望这些内容能够帮助你更好地与人沟通，建立良好的人际关系。
