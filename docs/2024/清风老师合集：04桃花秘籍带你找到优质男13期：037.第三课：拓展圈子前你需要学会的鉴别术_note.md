# 清风老师合集：04桃花秘籍带你找到优质男13期：037.第三课：拓展圈子前你需要学会的鉴别术

## 概述
在本节课中，我们将学习如何拓展社交圈子，并学会鉴别渣男，确保情感幸福。

## 社交圈子的误区
**以下是关于拓展圈子的常见误区：**
- 认为认识的人越多越好。
- 认为微信好友数量越多越好。

## 社交圈子的本质
**社交圈子的本质是固定数量中质量的变化。**
- 人的社会关系上限约为150人，这是由心理学家罗宾·邓巴提出的邓巴数。
- 社交中的人不是越多越好，而是在合理的数量范围内，将认识的朋友进行流动变化。

## 鉴别渣男的方法
**以下是鉴别渣男的三种方法：**
1. **缺乏行动**：只说不做，例如承诺换手机却迟迟不行动。
2. **缺乏共同朋友圈**：不愿意公开关系，或以各种理由推脱。
3. **缺乏共同兴趣**：与你的灵魂没有碰撞，只索取身体。

## 社交的三个分层
**社交可以分为三个层次：**
1. **至友知己**：数量少，但稳定提供帮助和支持。
2. **好朋友**：数量适中，有共同兴趣和话题。
3. **朋友**：数量较多，有一定共同兴趣和话题。
