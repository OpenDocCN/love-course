# 老李校长《男人内外魅力训练营》第66课：请把自己当个人 👤

在本节课中，我们将要学习建立男性魅力的核心第一步：把自己当一个人来看待，而不是一个工具。我们将探讨工具人思维在两性关系中的危害，并理解“为自己而活”如何成为真正吸引力的源泉。

## 情感世界的本质 🌑

情感的道路并非总是鲜花与美酒。情感的世界充斥着挑战与人性的复杂。唯有经历并克服这些挑战的人，方能成为自己生活的主宰。

上一节我们提到了情感世界的复杂性，本节中我们来看看，许多男性在无意识中陷入了一种怎样的思维陷阱。

## 工具人思维的陷阱 ⚙️

建立性吸引力与男性魅力的第一步，是把自己当个人，而非工具人。我们的文化、教育和影视作品，在不知不觉中将绝大多数男性塑造成了工具人。

这种塑造意味着：你天生是为了干活和服务他人；你天生需要讨好别人；你的想法、兴趣、爱好、喜怒哀乐都不重要，你的任务是让别人开心。

这种思维模式会导致严重的后果。以下是它在不同领域中的表现：

*   **在两性关系中**：你会不自觉地成为“舔狗”或“暖男”。
*   **在工作与赚钱过程中**：你会学会溜须拍马，像狗一样服务他人以换取项目或机会。

然而，这种在情感中如同“舔狗”般服务的思维，不会有好下场。因为女性从来不会认为一个“暖男”或“舔狗”有价值，也不会将其视为有性魅力的男人。她们会觉得这类人无趣，说话有一搭没一搭，甚至认为其没有价值而将其删除。相反，“海王”即使被众多女性知道其本质，依然有人愿意接近他，以证明自己的魅力。

## 人性的镜像法则 🔄

人性的规律就是这样：你越把自己当人，别人就会把你当人；你把自己不当人，别人就会拿你当狗。既然你都把自己当狗了，别人又何必对一只狗好呢？

很多人活到很大岁数，都不知道如何“当个人”。其实这很简单：**你不为任何人而活，也不为任何人的命运负责。**

这句话听起来可能很刺耳，因为我们从小接受的教育是：你要对女人负责，要对这个人、那个人负责。如果不负责，你就会被贴上“不靠谱”或“渣男”的标签。

但是，如果你对这个人负责，对那个人负责，现在的你是什么状态？你是一个唯唯诺诺、害怕失去的人，一个只能靠花钱来维系关系的人。

大家可以观察一下，那些知名的“坏男人”对哪个女人负责了？一旦他们变得负责、靠谱，那些美女反而会立刻不爱他。这也正是许多关于“渣男”和美女的悲欢离合故事的内核逻辑。

核心道理在于：正因为你不为对方的命运负责，你身上反而会散发出一种魅力，产生致命的吸引力。因为这意味着你有自己的方向，有自己的事情要做，你不是一个“烂好人”。别人会因此对你产生好奇，想要靠近你、接近你。

**公式：强者 ≠ 为弱者的命运负责。强者 = 拥有自己的方向。**

大家一定要在脑海里牢固树立这个观念：你要把自己当人。

## 如何实践“把自己当人” 🌱

不要为了一个美女寻死觅活，不要因为某个美女不理你，就觉得天塌下来了。

你是一个有自己想法、理想、事情、爱好和乐趣，并要为之行动的人。你不是一个赚钱工具，不是一个女人的服务员，不是一个可以被女人随意责骂、你还要陪笑且不敢发脾气的窝囊废。

只有当你先成为一个独立、自主的“人”时，魅力的种子才能在你心里生根发芽，最终长成参天大树。如果你永远不把自己当人，总是去学各种聊天技巧、约会技巧，那是没有用的，只是在浪费时间和金钱。

请大家慢慢体会本节课的内容。如果你想要拥有属于自己的男性魅力，首先，你必须把自己当作一个“人”来看待。
