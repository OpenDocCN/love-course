# 墨多亲授女性婚姻成长课：如何治理婚姻中的“巨婴男”或“妈宝男”

大家好，欢迎来到第十九节课。今天我们将探讨如何治理婚姻中的“巨婴男”或“妈宝男”。

## 案例分析

一位35岁的女士，结婚11年，育有一名10岁女儿。她的婚姻生活压抑，感觉自己像单身一样，与丈夫如同陌生人。丈夫性格内向，不喜欢有挑战和压力的工作，收入仅能解决温饱。家庭支出和孩子支出主要由女士承担。丈夫的业余时间都花在网游上，忽视妻子和孩子。女士尝试与丈夫沟通，但丈夫总是以玩游戏为由拒绝交流。女士要求丈夫分担孩子的教育和生活支出，却遭到丈夫的反对。

## 婚姻现状分析

女士的婚姻现状表明，丈夫在经济、子女养育和情感交流方面都缺乏投入。女士忍耐了这么多年，仍然在纠结是否继续这段婚姻。女士的婚姻现状说明，丈夫有恃无恐的原因在于他知道女士不会与他离婚，女士没有离婚的能力或胆量。

## 解决方案

1. **梳理需求**：女士需要梳理自己的需求，弄清楚什么让她无法放弃这段婚姻，她从这段感情中得到了什么，这些是否真的能弥补她失去的部分。

2. **改善关系**：女士需要停止逼迫丈夫在工作问题上的上进，给他宽松的业余生活，消除他对她的对立情绪。

3. **激发成就感**：利用丈夫对成就感的需求，驱动他做家务、带孩子。从简单的事情开始，夸奖他的具体细节，并在亲朋好友面前夸奖他。

4. **离婚能力**：女士需要具备离婚的能力，衡量清楚自己的底线在哪里，找到离开一段冲破自己底线的婚姻的勇气和力量。

## 总结

本节课中，我们学习了如何治理婚姻中的“巨婴男”或“妈宝男”。首先，女士需要梳理自己的需求，改善与丈夫的关系，激发他的成就感，并具备离婚的能力。希望这节课对大家有所帮助。😊
