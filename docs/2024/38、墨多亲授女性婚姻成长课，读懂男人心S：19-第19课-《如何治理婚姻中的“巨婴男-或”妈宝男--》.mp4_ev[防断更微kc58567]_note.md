# 课程38：读懂男人心 🧠 - 第19课：如何治理婚姻中的“巨婴男”或“妈宝男” 👶➡️👨

在本节课中，我们将探讨婚姻中一种常见的困境：如何应对缺乏责任感、依赖心强的伴侣，即所谓的“巨婴男”或“妈宝男”。我们将通过一个具体案例，分析问题根源，并学习切实可行的治理策略。

一位女士陈述了她的婚姻状况：她35岁，结婚11年，有一个10岁的女儿。她感到婚姻生活压抑，夫妻关系如同陌生人。她的丈夫性格内向，不愿从事有挑战性的工作，月收入仅3000元，仅够个人温饱。家庭与孩子的支出主要由收入更高的她承担。丈夫的业余时间全部用于网络游戏，不理会她和孩子。沟通极少，丈夫甚至反感她的联系。当她要求丈夫分担孩子教育生活费用，实行AA制时，这反而成了丈夫讨厌她的理由。她感到憋屈，生活中无人分担，不知婚姻是否应继续。

我对她指出：如果丈夫没有她和这个家，月薪3000元可能连温饱都难解决。但他却没有花任何精力维护这个他赖以生存的家庭。他的有恃无恐，根源在于他知道你不会离婚。你显然缺乏离婚的能力或勇气。你需要梳理自己的需求，弄清是什么让你无法放弃这段婚姻。

在这段婚姻中，你的**可见损失**是：丈夫不履行责任，不提供情感价值，像寄生虫一样榨取你的付出。你的**不可见损失（机会成本）** 是：如果你早些结束婚姻，就有机会找到珍视你、能共同承担家庭责任与情感的伴侣。你为这段关系损失了获得幸福的机会。

那么，你从这段感情中得到了什么？可能是旁人眼中“原配”的婚姻表象，可能是避免面对离婚后未知的恐惧，或是顾及孩子的想法。你需要自己衡量：**你得到的，是否真的能弥补你所失去的？**

我的意思并非你必须离婚，而是你需要**具备离婚的能力**，明确自己的底线。只有带着能够离开的勇气和力量去面对婚姻，对方才不会如此有恃无恐。

如果你有类似遭遇，治理当前婚姻状态可以采取以下措施。首先，你需要分清哪些无法改变，哪些具备改善空间。

对于案例中的女士，我指出：你丈夫选择“躺平”的人生，不想从事有压力的工作，这是他的本性选择。**这一点，你改变不了**。你之前的抱怨和发脾气，不仅没有激发他的上进心，反而让他产生排斥和反感，用更沉迷游戏、冷暴力等方式消极抵抗。你要求AA制的做法，戳中了他的软肋，他感受到的不是激励，而是羞辱和制裁，这只会引发他的对抗。

结合自己的情况想一想，当你面对“巨婴男”或“妈宝男”时，你的应对方式是否妥当？根据我的经验，大部分女性会做错一件事：**不要试图改变伴侣的天性和本质**。你改变不了，越试图改变，关系越差。对方为了对抗，可能连原本能做到的都不愿做了。

那么，婚姻中哪些方面具备改善空间呢？我们继续看案例。

我告诉那位女士：在履行丈夫和父亲职责方面，你丈夫具备改善空间。可以让他更多参与家庭事务。他之所以吝于付出，不是没能力或不想做，而是在对抗你对他的抱怨和“制裁”。相处不愉快，他自然不会回馈幸福。

你需要做两方面工作：
首先，**停止逼迫他在工作问题上“上进”**。不要再指望他满足家庭物质需求。给他宽松、无心理负担的业余生活。第一步是消除他对你的对立情绪。

其次，**利用他对“成就感”的需求，驱动他做家务、带孩子**。每个人都有对成就感的需求，它能激发快乐的多巴胺。男人的成就感来自征服，例如完成棘手工作、获得崇拜等。你丈夫的成就感目前只来自游戏中的虚拟征服。

你需要学会引导。如何让他在家务和育儿中获得成就感？

以下是具体方法：
从最简单的事情开始给他安排任务。完成后夸奖他，并且**夸奖要具体到细节**。最好还能在亲朋好友面前夸奖他。

例如：
*   “老公，你洗碗居然不会把水溅到外面，真利索。给我说说诀窍吧，我每次都溅得到处都是。”
*   “表姐，你别看我老公，现在做菜手艺一流，我都比不上了。”
*   “自从我老公开始辅导女儿功课，女儿情绪好多了，老师都说她进步快。”

这样的夸奖，不仅能让他产生**成就感**，还能让他感受到自己在家庭中的**价值**以及你对他的**需要**。这会激发他“不能辜负你”的动力，逐渐唤醒他的家庭责任意识。

在处理婚姻问题时，我秉持一个原则：**如果你没有尝试过以正向、正确的方式解决问题，就还没有到谈放弃的地步**。但正如前面所说，拥有杀伐决断的离婚能力，你才有底气。如果不想离婚，就按上述方法好好调教伴侣。如果想离婚，就离开，你还有更广阔的空间。

最后总结，在本节课中，我们一起学习了如何治理婚姻中的“巨婴男”或“妈宝男”问题。核心要点是：

1.  **认清现实，分清界限**：接受无法改变对方本性的事实，停止无效的抱怨和逼迫。
2.  **评估得失，积蓄力量**：梳理自己在婚姻中的得失，明确底线，并培养离开的勇气和能力，这是解决问题的底气。
3.  **转变策略，正向引导**：停止对抗，利用对方对“成就感”的心理需求，通过**具体化的夸奖**和赋予价值感，引导其参与家庭责任，公式可简化为：**具体夸奖 + 价值认可 = 激发责任感**。

女性因妻子角色难免产生拯救对方的情绪，但请记住：**你无法改变一个人的本性**。其次，如果你拥有离开的资本和底气，问题会简单许多。若仍想拯救婚姻，请先使用正确的方法进行尝试。
