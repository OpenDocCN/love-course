# 富山山女性成长课程：37_破圈去哪里认识人 🌟

## 概述
在本节课中，我们将探讨如何拓展社交圈，认识优秀的人，并了解他们常出入的场合。

## 认识优秀人士的场所
以下是认识优秀人士的一些场所：

- **大酒店**：鸡尾酒吧、Sky Bar、雪茄馆等。
- **飞机**：头等舱乘客。
- **展览**：手机展、画展、车展等。
- **运动场所**：高尔夫球场、网球场、羽毛球场等。
- **婚礼**：有钱人家的孩子的婚礼。

## 如何在场所中认识人
以下是认识人的方法：

1. **选择合适的场所**：选择你感兴趣且能吸引你目标人群的场所。
2. **观察氛围**：到达场所后，先观察整体氛围和在场人士。
3. **主动搭讪**：找到目标对象后，主动搭讪，可以从共同兴趣或话题入手。
4. **了解对方**：通过对方的言谈举止、衣着打扮等了解其身份和背景。
5. **建立友谊**：先与对方建立友谊，再逐步了解其周围的人。
6. **筛选目标**：在认识大量人之后，再筛选出最适合你的人。

## 案例分享
以下是一些认识人的案例：

1. **雪茄馆认识大哥**：通过观察对方行为，主动搭讪，最终成为朋友。
2. **画家姐姐**：通过评论互动，主动联系，最终成为朋友。
3. **飞机上认识大哥**：通过共同兴趣，主动搭讪，最终成为朋友。

## 总结
本节课中，我们学习了如何拓展社交圈，认识优秀的人。希望大家能够积极行动，走出舒适区，拓展自己的人脉。
