# 乐嘉性格色彩读心术 第22课：如何说服强硬的人 🎯

在本节课中，我们将学习如何与工作中或生活中那些“强硬”的人有效沟通和相处。我们将重点剖析一种典型的强硬类型——“冷酷无情型”老板，并学习针对性的应对策略。

---

## 课程概述

强硬的人常常让人感到压力巨大，甚至想要逃避。我的表妹就因无法应对强势的老板而在半年内辞职三次。她的问题根源在于童年被强势同学欺负的经历，导致她对这类人产生本能恐惧。要解决与强势者之间的问题，首先必须了解他们强势背后的原因。

根据我的观察，日常中的强势者主要分为两大类。第一类是冷酷无情型，第二类是脾气暴躁、如狂风暴雨型。这两类人的内心动机和相处方法截然不同，用错方法可能导致事与愿违。

上一节我们概述了两种强势类型，本节中我们重点剖析第一类：冷酷无情型的老板。

---

## 认识“冷酷无情型”强势者

这类老板在性格色彩学中属于典型的**黄色性格**。他们的核心特点是：**不关注人的感受，只关心目标和结果**。

因此，在日常互动中，他们几乎没有情感交流，开口便直奔主题，只问结果，不问过程。如果你的想法与他不同，他会直接打断你，下达结论并要求立即执行。

情感丰富的人遇到这类老板，往往会感到受伤和无力。例如，我曾听说一位管理者，其老板通过内部聊天工具只发了一个“来”字，就让下属技术员感到巨大压力，担心自己是否犯错。而当技术员忐忑不安地走进办公室，老板只是面无表情地说：“我电脑中毒了。”

遇到黄色性格的老板，你可能会钦佩他的能力、果断和高效，并能从他身上学到很多。但同时，你也会在他的注视下活得小心翼翼，非常辛苦。

## 一个深刻的案例

我的一位黄色性格学生，自己也是企业管理者。在学习性格色彩课程后，他进行了深刻反省，并向我分享了一个故事。

他曾经非常看重并全力培养一位女下属，给了她最好的机会、薪资、资源和平台。然而，六年后这位女下属辞职了，并终于说出心里话：“老板，我很感谢你的栽培。但这六年的每一个晚上，我都没睡好过。每当在办公室听到你的高跟鞋声，我全身都会绷紧。六年来我没休过一次假，甚至亲人去世请假也被你驳回，你说‘人都死了，去有啥用？’。我离开不是因为找到了更好的机会，而是不想有一天变成和你一样的人。”

这位女老板的形象，与著名电影《穿普拉达的女王》中的米兰达非常相似。她的助理安德莉亚同样生活在高压和快节奏中，最终因无法平衡生活与工作而选择离开，尽管老板许诺了更大的发展空间。

在上述故事中，那位黄色性格的女老板因为学习了性格色彩，终于认识到自身问题并开始改变。但如果你是一个害怕强势老板的人，你同样可以从性格色彩中学到有效的应对方法。

## 应对黄色性格（冷酷无情型）的实用策略

针对与黄色性格者相处的不同情境，共有16种方法。由于时间有限，我直接为你点出几个可以立即使用的关键策略：

以下是四个核心应对策略：

1.  **汇报工作，先讲结果**
    黄色性格者痛恨浪费时间。因此，在汇报时，你必须**先讲结果**。如果他对过程感兴趣，自然会询问。如果他没有问，就不要赘述。同样，如果汇报中包含未解决的问题，最好附上你建议的解决方案供他决策，以节省时间。

2.  **被指责时，避免解释**
    当黄色性格者指出你的问题时，**千万不要解释**。你的解释会被他视为不敢承认错误的懦弱表现。即使他的指责并非完全正确，最好的做法是：分辨出其中正确的部分，对此认错，并承诺立即改进。

3.  **提建议时，引导而非否定**
    如果你想向黄色性格者建议其决定不可行，最好不要直接否定他。可以按照他的决议制定一个方案，然后向他说明执行此方案可能带来的严重后果，从而引导他自己做出调整。

4.  **主动、精炼地表达观点**
    许多黄色性格者并未意识到你的沉默是出于害怕。他们会认为你不说，就是没什么可说的。因此，训练自己**精炼、简洁、心平气和且坚定地**表达内心真实想法至关重要。关于如何训练这些表达技巧，在我的演讲课程中会有详细阐述。

此外，需要明白：许多黄色性格者并不需要情感上的交流。他们若认可你，便会委以重任。因此，不要期待他们会给你热情的表扬，否则便是你的认知出了问题。

诸如此类，当与黄色性格者发生冲突时，或当你面对黄色性格的上司、下属、同事、客户时，都有无数的应对策略。以上展示的只是其中一小部分，但这些策略同样适用于应对你生活中黄色性格的父母或伴侣。
