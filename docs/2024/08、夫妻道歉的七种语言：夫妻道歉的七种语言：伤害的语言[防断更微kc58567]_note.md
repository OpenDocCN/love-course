# 课程08：夫妻道歉的七种语言之“伤害的语言” 💔

在本节课中，我们将要学习夫妻道歉七种语言中的第二种——“伤害的语言”。我们将探讨如何具体地表达自己的行为对伴侣造成的实际伤害，而非仅仅说一句“对不起”，从而使道歉更具深度和疗愈效果。

上一节我们介绍了道歉的第一种语言“表达歉意”，本节中我们来看看如何具体地描述“伤害”。

## 什么是“伤害的语言”？

“伤害的语言”并非指我们去伤害对方，而是指我们需要具体地陈述：**我们所做的那件事情，对对方造成了什么样的伤害、困扰或压力**。许多人会要求伴侣道歉，但一句简单的“对不起”往往不足以真正取得谅解。道歉的深度和力度需要加强。

尽管我们强调道歉要真诚、诚恳，但仅此仍然不够。关键在于，我们如何具体地表达对对方造成的伤害。

## 为何需要表达“伤害”？

这非常关键。例如，在辅导案例中，一位男士认为岳母的言行对他造成了很大伤害。如果岳母只是简单地说“对不起”，这通常没有效果。反之，如果岳母能认识到，她的行为**对女婿的人格尊严造成了伤害**，或者**让女婿感到被否定**，这才是触及了“伤害的语言”的核心。

这种对伤害的认知，常常需要你自己去感知和体会。我们很难完全设身处地，但可以借助相似的经历来理解。

## 如何体察与表达“伤害”？

夫妻之间常有相似的经历。例如，妻子对丈夫说：“你这个不养家的男人。”要体会丈夫内心的感受，妻子可以回想丈夫是否说过类似的话，比如：“你看不都是老子挣钱养你吗？”

当听到这样的话时，你的感受可能是：“难道我是没有用的人吗？” 或 “你不认可我在家庭里的贡献和地位。” 这种**感受到对方的感受**的过程，就是理解“伤害的语言”的关键。

许多人表达出来的其实是**指责**。例如说“你凭什么……”，这背后是你内心的**愤怒**。但我们需要表达的是引发愤怒的那个**核心感受**，而不是指责本身。

很大程度上，我们找不到或无法切入到需要道歉的人的内心，正是因为忽略了表达具体的伤害。

## 实践“伤害的语言”的要点

以下是运用“伤害的语言”时需要注意的几个核心步骤：

1.  **具体化**：明确指出是哪个行为或哪句话对不起对方。避免模糊的道歉。
2.  **感受化**：描述这个行为可能给对方带来的具体感受（如：“我的那句话，可能让你觉得自己的付出没有被看见，伤害了你的自尊。”）。
3.  **基于体察**：道歉应建立在你对伴侣感受的体察之上，表明你确实理解了他/她被伤害的地方。

这样更容易让对方知道你确实体察到了他/她被伤害的感受，否则道歉是无效的。

## 总结

本节课中，我们一起学习了夫妻道歉的第二种语言——“伤害的语言”。核心在于，道歉时需要**具体、真诚地描述你的行为对伴侣造成的实际伤害与感受**，而不仅仅是说“对不起”。通过将道歉**具体化**和**感受化**，才能让道歉触及内心，达成真正的谅解。
