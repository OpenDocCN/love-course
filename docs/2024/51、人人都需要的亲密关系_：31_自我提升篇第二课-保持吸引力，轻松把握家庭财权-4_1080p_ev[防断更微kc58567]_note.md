# 人人都需要的亲密关系_：31_自我提升篇第二课-保持吸引力，轻松把握家庭财权

## 概述
在本节课中，我们将学习如何保持吸引力，并在家庭中轻松把握财权。

## 保持吸引力
### 核心概念
**公式**：吸引力 = 外在魅力 + 内在魅力

### 方法
1. **提升外在魅力**：保持良好的仪表，穿着得体。
2. **提升内在魅力**：培养兴趣爱好，丰富内心世界。

## 轻松把握家庭财权
### 核心概念
**公式**：财权 = 沟通技巧 + 时机选择

### 方法
1. **结婚前**：
    - 在感情浓度最深的时候，如男人求婚时，提出财权问题。
    - 在热恋期，通过玩笑试探对方态度。
2. **结婚后**：
    - 产前：提出保障财安心的需求。
    - 产后：表达为家庭付出辛劳的感激之情。
    - 男人犯错时：提出增加安全感的诉求。
3. **针对不同类型的伴侣**：
    - **做生意的人**：争取部分家用。
    - **艺术家或自由职业者**：引导他们购买食物等。
    - **经济适用型**：展示理财能力，说服他们交出财权。

### 话术
1. **男人犯错时**：你这样我不接受，为了保证你不再犯错，你必须把财权给我。
2. **热恋期**：那你以后可不可以把钱给我管啊，我不管反正以后我要当管家婆啊。
3. **产期**：你还知道心疼我呀，你别只顾的说了啊，就给我一些小钱，你要真心疼我，你就把咱家钱拿给我来管。

## 总结
本节课中，我们学习了如何保持吸引力，并在家庭中轻松把握财权。希望大家能够运用所学知识，为自己争取到更多的疼爱。
