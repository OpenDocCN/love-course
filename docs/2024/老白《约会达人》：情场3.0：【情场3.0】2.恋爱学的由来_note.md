# 老白《约会达人》情场3.0：第二课 · 恋爱学的由来 📜

在本节课中，我们将要学习“恋爱学”这一概念的起源与发展历程，了解它如何从一个小众论坛演变为一个备受关注的领域，并最终传入中国。

## 起源：美国论坛的诞生

上一节我们介绍了课程的整体框架，本节中我们来看看恋爱学的历史起点。恋爱学起源于美国90年代的一个论坛，这是一个分享如何在酒吧结识女性心得的论坛。

随着论坛的发展，出现了几位被奉为“大神”的人物。迷喃是其中的佼佼者，同时也聚集了一群追随他的粉丝。

## 发展：从分享到商业化

他们对大神充满崇拜，希望大神能分享技巧。于是迷喃开始公开招募学徒，教授他们如何结识女性以及进行一夜情。

以下是关于其首批学员的关键信息：
*   第一批学员是他的5位粉丝。
*   其中一位是《纽约时报》的特约编辑。

这位编辑将跟随迷喃学习的两年经历写成了一本书，书名为《魅达人》。这本书成为了畅销书，将这个小众论坛带入了公众视野。

## 核心理念与时代局限

此事具有划时代的意义，它首次让大众认识到：**追求异性的能力并非天生**。而且人与人在这方面的差距可以很大，**这种能力是可以后天学习的**。

然而，其中存在年代差距。国内外的文化以及开放程度各不相同，其早期模式并不完全适用于今天的中国社会。

## 传入中国：土壤与需求

随后，恋爱学漂洋过海来到了中国。中国在恋爱领域可称为“重灾区”，拥有自由恋爱历史也不过二三十年。在此之前，自由恋爱甚至不被提倡。

但是在这里，人们比任何地方都更需要恋爱指导，因为我们既不会，也无人教导。我们被要求专心上学、不要早恋，却又被期望毕业后就能带恋人回家过年，这让许多男性感到困扰。

## 本土化与行业爆发

恋爱学来到中国后受到了前所未有的追捧。中国的相关论坛迅速火热，大家如获至宝。聪明的中国人再一次嗅到了商机。

以下是当时行业爆发的几个特点：
*   一大批“大神”如雨后春笋般出现，开班授课。
*   许多人自己也是一知半解，通过商业包装华丽登场。

说到底，这成了一门生意。商业运作与自身是否精通关系不大，与能否真正教会别人也没有太大关系。赚钱才是首要目的。

## 现状：情怀与商业的平衡

因此，行业爆发很快，规模做得大的也很多。但是，怀有情怀、真正把“教会学员”作为首要目标的人却很少。

我因为相亲上百次，又曾对一位女生爱而不得，开始接触并研究心理学和情感学。学习数年后，开始有许多朋友及朋友的朋友请我教授恋爱方法。

这时我才意识到，我的这项技能可以帮助他人，我也具备了帮助他人的能力。同时，这个领域对我有深深的吸引力。

我阅遍了国内外所有能找到的相关资料，受到了巨大的启发，一次次被震撼。结合自身经历，也一次次感受到这套知识体系对我们普通人的价值。
