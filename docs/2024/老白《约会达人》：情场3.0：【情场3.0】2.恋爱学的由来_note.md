# 老白《约会达人》：情场3.0：【情场3.0】2.恋爱学的由来

🎼本节课我们将探讨恋爱学（LS）的起源。恋爱学起源于20世纪90年代美国的一个论坛，该论坛旨在分享如何在酒吧吸引异性的经验。随着论坛的发展，涌现出几位大神级人物，其中迷喃尤为出色，并吸引了一群粉丝。

**以下是恋爱学的基本概念：**

*   **恋爱学（LS）**：研究恋爱技巧和人际关系的学科。
*   **论坛**：一个在线社区，成员可以分享经验和知识。

迷喃因其技巧而受到粉丝膜拜，并开始公开招募学徒，教授他们如何吸引异性以及如何进行一夜情。第一批学员包括他的5个粉丝，其中一位是《纽约时报》的特约编辑。这位编辑将他与迷喃两年间的经历写成了一本书——《魅达人》，该书成为畅销书，将这个小众论坛带入了公众视野，具有划时代的意义。它首次让人们认识到，追求异性的能力并非天生，而是可以学习的。

**以下是恋爱学的核心概念：**

*   **追求异性的能力**：可以通过学习和实践来提高。
*   **学习**：通过学习可以缩小人与人之间的差距。

然而，由于年代久远，以及国内外文化及开放程度的差异，这些技巧并不完全适用于今天的中国。迷喃来到中国，这个恋爱重灾区，发现这里比任何地方都需要他的帮助。在中国，自由恋爱历史不过二三十年，之前恋爱被视为大逆不道。但在这里，恋爱变得尤为重要，因为没有人教授恋爱技巧，而社会却要求年轻人毕业后带女朋友回家过年。

**以下是恋爱学在中国的发展：**

*   **恋爱重灾区**：指恋爱技巧需求较高的地区。
*   **自由恋爱**：指不受传统婚姻观念束缚的恋爱方式。

迷喃的到来受到了前所未有的追捧，中国的类似论坛也迅速兴起，成为人们寻求恋爱技巧的宝库。聪明的中国人再次嗅到了商机，一大批大神纷纷涌现，开设培训班授课。许多人虽然对恋爱技巧了解有限，但通过商业包装华丽登场。

**以下是恋爱学在中国的发展现状：**

*   **商业**：恋爱技巧培训成为一种商业模式。
*   **大神**：指在恋爱技巧方面有丰富经验的人。

然而，这终究是商业行为，赚钱才是目的。因此，行业迅速发展，涌现出许多培训机构，但真正将教学视为首要任务的却寥寥无几。

**以下是本节课的总结：**

*   恋爱学起源于20世纪90年代的美国论坛。
*   迷喃是恋爱学的重要人物，他的书籍《魅达人》将恋爱技巧带入了公众视野。
*   恋爱技巧可以通过学习和实践来提高。
*   恋爱学在中国得到了迅速发展，但真正将教学视为首要任务的机构却不多。
