# 21. 王一名·吸引力出圈指南：如何找准你的优势？

欢迎来到吸引力出圈指南。我是王一鸣。本节课我们将探讨如何找准你的优势，这是我们核心内容的第一步：发掘和确定优势。

**以下是本节课的主要内容：**

### 个人品牌与优势

我们身边那些有吸引力的人，都有一个共同特质：他们像产品一样，总是能让人记住，因为他们身上有一个明显的优点。这就是个人品牌。

**个人品牌**：一个人的个人品牌就是一个人的优势。无论是一个人还是一个产品，只要他们的优势够明显，就能脱颖而出，形成品牌。

### 为什么需要个人品牌？

有人可能会问，为什么需要打造个人品牌？不管我们想不想当公众人物，我们都是有个人品牌的。只要我们生活在这个社会里，或者在工作中，我们的同事、上级领导、客户、亲朋好友就会建立一个对我们的印象。这就是我们的个人品牌。

### 如何找准你的优势？

我们可以通过以下四个问题来协助你独立思考自己的优势在哪里：

1. **你做过哪些有成就感的事**？这些事情不局限在职场上，任何让你成就感满满的事都可以。
2. **你做哪些事情之前会充满期待，做起来也容易沉浸在里边，忘了时间**？这就是你热爱的事情。
3. **你擅长做哪些事情**？通常都是你已经拿到了不错的结果，大多数情况下，这也成为了你的工作。
4. **如果你通过自己的努力学习和训练，能做成什么事情**？你可以尝试着列出一些你看到别人做的事情，你也希望像他一样自己能够做到。

### 定位理论

在专业的市场领域，有一个特别经典的定位理论：如果你想要让一个产品的品牌对消费者产生影响力，你就应该在你预期的客户的脑子里给你要卖的产品占据一个真正有价值的地位。
