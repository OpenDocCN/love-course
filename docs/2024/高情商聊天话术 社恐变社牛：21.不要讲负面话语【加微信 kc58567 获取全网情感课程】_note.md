# 高情商聊天话术 社恐变社牛：21.不要讲负面话语

## 概述

在本节课中，我们将学习如何避免在聊天中讲负面话语，从而提升社交能力，从社恐变为社牛。

## 核心概念

**公式**： 负面话语 → 负面氛围 → 关系恶化

## 1. 错误的认识

很多男人会认为自己长得不高、受挫或外貌不好，从而觉得自己不好泡妞。这是一个错误的认识。长得丑的男人也可以泡到妞，关键在于你的内在。

**公式**： 外貌 → 内在 → 魅力

## 2. 聊天中的负面话语

在与女孩子聊天时，不要问她们是否不开心，避免使用负面词语。心里想负面词语，最终会把关系气氛带到负面。

**公式**： 负面词语 → 负面氛围 → 关系恶化

## 3. 摆正心态

不要怕被拒绝，不要觉得自己配不上对方。当你想象自己配得上时，你的魅力就会提升。

**公式**： 自信 → 魅力 → 拒绝率降低

## 4. 社交模型

假装与对方很熟悉，慢慢建立真实的关系。

**公式**： 假装熟悉 → 真实关系

## 5. 强者心态

不要把自己懦弱、没有本事归结为老实。在女人眼里，没有老实不老实，只有强大和弱小。

**公式**： 懦弱 → 不受欢迎

## 6. 勇气与成长

泡妞过程中难免犯错，但错误是成长的过程。不怕挫折，不怕被打枪，才能在恋爱中成长。

**公式**： 错误 → 成长

## 7. 压价能力

不要表现出自己没有压价能力，否则会陷入被动地位。

**公式**： 压价能力 → 被动地位

## 8. 营销与社交

扩大社交面积，打造高大上朋友圈，提高信任度。

**公式**： 社交面积 → 信任度

## 9. 聊天技巧

不要问无聊的问题，不要传播负面信息，多讲正能量的话语。

**公式**： 聊天技巧 → 良好关系

## 总结

本节课中我们一起学习了如何避免在聊天中讲负面话语，提升社交能力。希望同学们能够将所学知识应用到实际生活中，从社恐变为社牛。
