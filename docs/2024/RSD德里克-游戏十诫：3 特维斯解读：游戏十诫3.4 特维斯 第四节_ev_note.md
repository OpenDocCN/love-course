每个人都误以为死亡发生在生命尽头，但死亡其实随着每一分每一秒的流逝而不断发生。时间是你最宝贵的资源。

德里克常问，是否愿意与89岁高龄、身家677亿美元的沃伦·巴菲特交换位置。大多数人不会做这笔交易。沃伦会愿意付出一切来换回几十年的时间。虽然你无法直接购买更多时间，但你可以通过更有效地利用智力与社会资源，来提升分配时间的效率。

在这些环境中穿梭时，放弃你对人们“应该是什么样”的理想化愿景会很有帮助。记住第一条诫命：大多数人是恐惧的、自私的，并且等待被告知该做什么。

即使在俱乐部玩得开心的人也会大量饮酒并紧贴朋友，因为夜店通常是一个令人畏惧的地方。人们不会忠诚，也不会信守承诺。无论你建立了多深的联系，人们通常会在出现麻烦的第一个迹象时就抛弃你。无需为此烦恼，这就是人的普遍状态。

## ❓ 五个关键的后勤问题

当你已经像模像样地开启了对话，并与她融洽相处了一段时间后，现在该做什么？这时，开始收集后勤信息会很有用。

以下是五个你应该反复重复直到牢记于心的问题：

1.  **你从哪里来？** - 有助于收集社会和文化上的细微差别。
2.  **你和谁一起来的？** - 有助于理解我之后可能会遇到谁。
3.  **你晚点有什么安排？** - 有助于了解今晚是否可能发生关系。
4.  **你住在哪里？** - 有助于判断是否值得尝试带她回家，仅仅基于回她住处的难度。
5.  **你早上要做什么？** - 有助于判断她是否需要过夜，或者你是否需要过夜。

对于这些问题以及任何其他信息收集问题，如果她提出质疑或想知道你为什么要问，那只是意味着你还没有提供足够关于你自己的信息或个性。为了缓解这种情况，我会在整个互动过程中穿插这些问题，而不是一次性全部问完。

## 🏠 家庭准备与策略规划

当你回到家中时，你需要为任何事情做好准备。一切必须干净且令人愉悦。如果她感到任何厌恶，都可能会让她失去兴趣。

准备创可贴、充电器。她需要食物或咖啡因吗？两者都准备。睡前服用活性炭片有助于缓解宿醉。准备 tampons、额外的衣物（如T恤和运动裤）、额外的牙刷。不用说，还要准备足够用一个月的安全套。

在规划你默认的“带回家”策略时，你需要从终点开始，然后倒推。准备好你的房间，让它变得温馨。然后倒推：你打算如何从俱乐部回到你家？Plan B是什么？再往前，当你在俱乐部内“勾搭”成功时，你打算把女孩移到哪里？你的默认开场白是什么？你到俱乐部后做什么？你是否在像模像样地开启对话？

## 📝 有效的学习与现场报告

当你外出并撰写现场报告时，有效的学习至关重要。后勤问题在你进入俱乐部后会相当本能和自动地解决。然而，它们是在家中学习和巩固的。

我的流程是，如果你能够移除并解决所有障碍，你将能够轻松地从开场进行到收尾。当我在俱乐部时，我几乎不做任何思考。就像武术家在打斗中不思考，外科医生在手术室中能信任自己的本能，股票交易员能观察市场并感知接下来会发生什么一样。与所有这些职业一样，我所有的思考都在家里完成。

我会花时间回顾昨晚的几个特定组合，并对它们进行以下评估：

1.  发生了什么。
2.  我做得好的地方是什么？
3.  我失去互动的那个瞬间是什么？
4.  下次在那个时刻我可以做什么来防止它发生？

我不想花太多时间写现场报告，因为这可能非常耗时。这使它保持简单有效。

## 🧠 保持客观与挖掘隐藏价值

在撰写这些报告并反思你的夜晚时，努力保持尽可能客观。心理学家荣格指出，人类会围绕一切事物强迫性地创造意义。尽你所能保持客观，并利用之前诫命的策略、训练营笔记以及补充指南来指导你对社交矩阵的理解。

你对最初许多次搭讪的态度应该是：“好吧，那发生了。”然后利用这些现场报告来弄清楚它们为什么会发生，以及当它们再次出现时该怎么做。

你可以使用这个通用结构来清除你道路上的障碍：物理障碍、社交障碍、心理障碍、情绪障碍。

利用这个框架，通过更深入地挖掘你失去互动的那个时刻，来发现你隐藏的价值观。也许你失去机会是因为你没有在杂货店接近那个女孩。什么做得好？你在考虑去尝试。失去互动的那个瞬间是什么？当你没有去和那个女孩说话时。然后深入挖掘，为什么？也许是因为其他人在看，或者她看起来很忙，或者其他原因。这说明了你的什么？那是一个隐藏的价值观。

## ✍️ 非停止写作/谈话工具

另一个使用的工具是非停止写作或非停止谈话。我会设定一个计时器，除了就某个问题写作或谈话外什么都不做，不花时间将手指从键盘上抬起或笔从纸上抬起，也不花时间暂停说话。当你继续写作或谈话时，即使你可能不知道接下来具体该写什么或说什么，也要坚持下去。

这是深入挖掘你的创造力、看看当你进入心流状态时大脑自然会产生什么的一种极其有效的方法。你可以对任何你可能有的问题这样做。
