# 高情商聊天话术 社恐变社牛 课程60：实在搞不定，就换环境 🏃‍♂️

在本节课中，我们将要学习一个核心观念：人是环境的产物。当你发现自己在恋爱、事业或生活中反复碰壁时，最有效的策略往往不是继续硬扛，而是果断地更换环境。我们将深入探讨环境如何塑造人，以及如何通过改变环境来改变自己的命运。

## 核心概念：人是环境的产物

上一节我们介绍了在错误方向上努力是徒劳的，本节中我们来看看为什么我们会陷入错误的循环。其根本原因在于，我们每个人都是自身所处环境的产物。你的思维、行为、甚至命运，在很大程度上都被你周围的环境所塑造。

**公式：个人现状 ≈ 环境输入的总和**

你的家庭、朋友、工作圈子、接触的信息，共同构成了塑造你的“环境”。如果你对现状不满，最根本的解决方法是改变这些输入源。

## 被“洗脑”的困境

很多人过得不如意，是因为他们的思维被“负向洗脑”所污染。社会上的某些观念，看似合理，实则服务于特定群体的利益。

例如，在恋爱中，“男人就应该无条件付出”这种观念，实际上最有利于那些并不打算平等回馈的人。如果你深信不疑并照此行动，就相当于被“洗笨”了，结果往往是付出很多却得不到珍惜。

以下是常见的几种“负向洗脑”观念：
*   **事业上**：推崇“把梳子卖给和尚”，认为挑战不可能才是本事。
*   **恋爱上**：认为只要拼命对一个人好（“舔”），就能让对方喜欢上你。
*   **社会观念上**：接受“强者通过让弱者变笨来获利”是合理的社会规律。

认识到这些，是为了防止我们的大脑被垃圾思想污染，从而做出更清醒的选择。

## 环境如何具体影响我们

人是条件反射的产物，你持续所处的环境，会像模具一样塑造你。让我们通过几个例子来看看环境的力量。

**举例1：职业塑造**
一个人成为出租车司机，往往是因为他的朋友、家人都认为这是一份稳定的好工作，这种环境声音引导他做出了选择。同理，一个人成为老板，也是因为他所处的环境鼓励创业、接触商业信息。

**举例2：性格与社交圈塑造**
一个程序员找不到女朋友，不仅仅是因为工作忙。他的工作环境（同事多为男性）、业余环境（宅家玩游戏）、获取信息的环境（浏览特定网站），共同将他塑造成一个内向、社交圈狭窄的人，从而陷入“找不到对象”的循环。

**代码示例：环境的影响循环**
```plaintext
环境（程序员圈子、宅文化） -> 行为（加班、玩游戏、少社交） -> 结果（性格内向、社交技能退化、异性资源少） -> 强化环境选择（更依赖现有圈子）-> 循环加剧
```
要打破这个循环，最有效的一环就是改变初始的“环境”。

## 解决方案：主动更换环境

既然人是环境的产物，那么改变命运最直接的方法就是改变环境。如果你自觉缺乏强大的自控力和执行力，那么融入一个更好的环境，让环境的力量带动你，是最高效的途径。

以下是更换环境的具体行动指南：
1.  **物理位置**：如果在小城市机会少，就努力去大城市。那里有更多的工作机会、更优质的社交圈子。
2.  **社交圈子**：远离充满抱怨、不思进取的“穷人堆”。主动靠近积极向上、已经在赚钱的圈子。
3.  **信息环境**：停止消费那些让你焦虑或灌输垃圾思想的内容。主动寻找和学习有真知灼见的信息源。
4.  **行业选择**：远离难以盈利的“沙漠”行业。投身到需求旺盛、资金流动快的“沃土”行业，如互联网、短视频等领域。

记住一个原则：**在优秀环境中做倒数第一，也强于在糟糕环境中做第一名。** 因为前者能让你持续接触到高标准的思维和行动模式。

## 环境与恋爱实战应用

这个原理在恋爱中同样适用。你找不到对象，很可能只是因为你的环境里缺乏足够的潜在对象。

**案例分享**：
讲师曾为了接触更多女性，特意进入女装公司和服装工厂工作。在这些女性占绝大多数的环境中，认识女生、发展关系变得非常简单，甚至会被倒追。这就是“物以稀为贵”的体现——在女性环境中，男性成为了稀缺资源。

以下是给你的实战建议：
*   **改变线下动线**：下班后不要去网吧或回家宅着，去咖啡馆、书店、健身房、舞蹈班等女生可能出现的地方。
*   **改变线上展示**：精心经营你的朋友圈，展示你的生活品味、兴趣爱好和生存价值（事业心、经济能力）。
*   **提升自身价值**：赚钱是比钻研恋爱技巧更根本的事。**经济基础决定上层建筑**，你的生存价值（经济能力、事业前景）是吸引异性的核心。

## 重要原则：远离“穷人”思维

这里所说的“穷人”，指的是思维贫穷、不愿改变、充满负能量的人。你要想变富、变好，就必须远离他们。

**天道法则**：在什么样的环境里，就要遵循那个环境的“道”。在穷人堆里想变富，你会被排斥；在富人堆里装穷，你也会被淘汰。聪明人要懂得“和光同尘”，主动进入你想成为的那个群体所在的环境。

**语言的力量**：注意你的言辞。总说“我们都是穷逼”的人，身边只会聚集穷人。要常说积极、正向、有野心的话，比如“老板们，今天分享一个搞钱思路”。语言会召唤你的命运。

## 终极心法：从经济学角度看关系

当你用经济学视角分析恋爱，就能彻底摆脱感性困扰。男女关系在某种程度上是一种**等价交换**。

*   **女性视角（待价而沽）**：女性倾向于选择能提供更高生存价值（经济保障、优质基因）的男性。这不是道德问题，而是生物本能和理性选择，如同任何人找工作都会选薪资更高的。
*   **男性视角（价高者得）**：你的财富、地位、能力就是你的“出价”。**有钱 ≈ 选择权**。当你“出价”足够高时，对方可能会忽略你的其他缺点。

不要试图用5000元去买标价10000元的手机（关系），无论你说多少好话店员（对方）都不会卖。你要做的不是纠缠，而是去赚到10000元。同样，不要无限度地对一个人好，这在经济学上叫“增加沉没成本”，最终可能导致“爆仓”（人财两空）。

你的策略应该是：**发挥你的核心长处（如赚钱能力），并将其提升到极致，而不是去弥补所有普通短板。** 当你长处足够耀眼，你就能兑换你想要的“关系”。

本节课中我们一起学习了环境对人的决定性影响。核心要点是：**当你搞不定眼前的问题时，首先应该检查并更换你所处的环境**。无论是为了脱单、赚钱还是自我成长，跳出旧有的圈子，进入一个更优质、更积极的环境，是破局的关键。记住，人是环境的产物，选择大于努力。从今天起，有意识地规划你的物理环境、社交环境和信息环境，让环境的力量推动你走向成功。
