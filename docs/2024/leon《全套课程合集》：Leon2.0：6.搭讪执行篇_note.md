# Leon2.0：6.搭讪执行篇 🤝

## 概述
在本节课中，我们将学习如何克服搭讪时的执行障碍，了解搭讪的好处和阻力，并掌握提升自我价值感和自信的方法。

## 搭讪的好处
以下是搭讪带来的好处：

1. **选择权**：搭讪让你拥有更多选择，避免对单一女性的过度依赖，从而在互动中保持自信和放松的心态。
2. **了解女性心理**：通过搭讪，你可以逐渐了解女性的心理，知道她们可能的反应和原因，从而更好地应对。
3. **提升社交能力**：搭讪可以提升你的社交能力，包括话题抛出、引导话题和情感互动等。

## 搭讪的阻力
以下是搭讪时可能遇到的阻力：

1. **认知偏差**：很多人对搭讪存在误解，认为其不正常或低俗。
2. **位置因素**：对未知情况的恐惧，如女生的反应、个人信息等。
3. **不知道说什么**：缺乏搭讪话题和技巧。
4. **基因恐惧**：来自远古时代的恐惧，担心被拒绝或社会不认同。
5. **自信不足**：缺乏自信和自我价值感。

## 克服阻力
以下是克服搭讪阻力的方法：

1. **正确认知**：了解搭讪的合理性，认识到其价值。
2. **预判和经验**：提前观察和预判女生的情况，积累经验。
3. **掌握技巧**：学习搭讪话题和技巧，提高应对能力。
4. **提升自信**：通过执行和正反馈，提升自我价值感和自信。

## 执行
以下是两个执行任务：

1. **无条件执行**：每天看到第一个目标时，无条件去搭讪。
2. **分析焦虑**：每次搭讪前，分析并记录自己的焦虑原因，逐一克服。

## 总结
本节课中，我们学习了搭讪的好处、阻力以及克服方法。通过正确的方法和不断的执行，我们可以克服搭讪时的焦虑，提升自我价值感和自信。
