以下是聊天地图的四个步骤：

1. **开场白**：通过20句话开启话题，收集情报，留下好印象。
2. **植入安全性**：展示你的安全感和可靠性，让对方感到舒适。
3. **判断节奏**：根据对方的反应，确定下一步的聊天目标。
4. **话题引导**：通过话题引导，达成升级关系的目标。

- **筛选**：通过20句话的聊天，判断对方是否为合适的人选。
- **判断**：根据对方的反应，判断其类型（A、B、C、D）。

## 植入安全性
- **独立人设**：通过朋友圈展示你的生活，让对方感到安全。
- **名字交流**：通过名字交流，拉近彼此距离。

- **引导评价**：引导对方对自己进行评价，判断其类型。

## 达成目标
