# 关系博弈（拒绝卑微和低位）：11_8.情感博弈-如何让男人产生保护欲-上

## 概述
在本节课中，我们将学习如何激发男人的保护欲，让他对你产生深厚的感情。

## 激发保护欲的武器
**武器一：让他觉得你失去了重要的东西**
- **表现**：让你觉得为了情感上重要的人或事物，你失去了其他特别重要的东西。
- **例子**：忍受失去重要人的伤痛，为了朋友隐瞒真相，被男人欺骗后放纵自己。

**武器二：让他觉得内疚**
- **方法**：让他主动去做，你被动配合。
- **例子**：男生未顾及你的想法，不够周到，或误解了你。

## 案例分析
**案例**：一位女生约我逛街，高跟鞋走久了脚后跟磨出水泡，却未告诉我。我误以为她故意拖延，后来得知真相后感到心疼。
