# 21、凯男的社交情商课：第五节课：七步聊天：002.七步聊天-2距离感过度

## 概述

在本节课中，我们将学习如何通过七步聊天法中的第二步——距离感过度，来拉近与女生的距离，建立初步吸引。

## 一、开场白回顾

在开场白阶段，我们需要做好两个前提：

1. **朋友圈建设到位**：确保你的朋友圈能够展示你的个性和魅力。
2. **选择合适的聊天目标**：选择一个与你兴趣相投、愿意交流的女生。

开场白分为三种：

1. **表情开场**：用表情符号引起女生的注意。
2. **状态加感受开场**：分享你的状态和感受，引起共鸣。
3. **好奇开场白**：用好奇的方式引起女生的兴趣。

## 二、距离感过度

距离感过度是指通过调动女生的情绪，拉近与她的距离，建立初步吸引。

### 1. 冷读

**公式**：观察 + 直觉 = 冷读

冷读是指用自己的直觉去猜测女生的性格和特点，然后说出你的感觉，达到共鸣的效果。

**注意事项**：

* 不要随意猜测，要有一定的观察和依据。
* 不要过度使用，避免女生觉得你像算命先生。

### 2. 骄傲风趣法

**公式**：自我夸奖 + 幽默 = 骄傲风趣

骄傲风趣是指以自我为中心，进行自我夸奖，展示自信，并引起幽默效果。

**注意事项**：

* 不要盲目自夸，要幽默。
* 举例：我这么单纯可爱，你会欺负我的。

### 3. 角色扮演

**公式**：设定角色 + 对话 = 角色扮演

角色扮演是指设定一个角色或场景，与女生进行对话，调动她的情绪。

**注意事项**：

* 主导剧情，让女生配合你演出。
* 举例：皇上和爱妃的角色，老师和学生等。

### 4. 推拉

**公式**：先拉后推 = 推拉

推拉是指先拉近与女生的距离，然后推开，让她产生情绪波动。

**注意事项**：

* 推拉要适度，避免女生反感。
* 举例：先夸奖女生，然后说她缺点。

### 5. 夸张曲解

**公式**：曲解字面意思或语境 = 夸张曲解

夸张曲解是指曲解女生的字面意思或语境背后的情绪，调动她的情绪。

**注意事项**：

* 不要过度曲解，避免女生反感。
* 举例：女生说“你真花心”，你可以曲解为“我是花心，你是花瓣”。

### 6. 植入画面感

**公式**：具体描述 = 植入画面感

植入画面感是指通过具体描述，让女生产生画面感，调动她的情绪。

**注意事项**：

* 描述要具体，越具体越真实。
* 举例：描述一个浪漫的场景，让女生产生画面感。

## 总结

本节课中，我们学习了如何通过七步聊天法中的第二步——距离感过度，来拉近与女生的距离，建立初步吸引。通过运用冷读、骄傲风趣法、角色扮演、推拉、夸张曲解和植入画面感等技巧，我们可以有效地调动女生的情绪，拉近与她的距离，为后续的交流打下基础。
