# 乌鸦救赎《连招绝技-白驹聊天秘术》：09. 人格筛选 🎯

在本节课中，我们将要学习连招体系中的第三招——人格筛选。我们将深入探讨它的定义、核心作用以及具体的使用方法，帮助你理解如何通过对话深入了解对方，并合理化双方的关系发展。

## 概述

人格筛选是建立深刻联系、展示价值并塑造关系框架的关键步骤。它不仅仅是简单的提问，而是一套系统的方法，用于在聊天中创造深度、合理化和服从性。

## 人格筛选的定义与作用

上一节我们介绍了关系推进的初步技巧，本节中我们来看看如何通过“人格筛选”深化连接。人格筛选有多重定义和作用。

**定义一**：人格筛选是女生展示自身价值的契机，让你的热情得到合理化。
**公式**：`女生展示价值 -> 你的喜欢被合理化`

**定义二**：人格筛选是打开彼此内在的窗口，可以建立更深刻的吸引与联系感。
**公式**：`提问内在 -> 打开窗口 -> 建立深刻联系`

**定义三**：人格筛选是一种顺延对话的方式，让话题在了解彼此时保持新鲜感。
**公式**：`筛选式提问 -> 制造轻微冲突/张力 -> 提升注意力与新鲜感`

**定义四**：人格筛选是一种服从性的塑造方式，允许被你筛选本身就是一种服从。
**公式**：`你主导筛选（类似面试）-> 建立服从性框架`

**定义五**：人格筛选是你了解对方真实内在（性格、才华、背景）的机会，有助于做出是否继续的选择。
**公式**：`提问过去/价值观 -> 了解真实内在 -> 做出筛选决策`

## 人格筛选的核心：制造性张力与情绪价值

在深入使用方法前，我们需要理解一个核心概念：**性张力**。这是推动关系发展的隐形动力。

性张力可以理解为一种暧昧、不确定的心理状态，让对方不断琢磨“你到底喜不喜欢我”。它本质上是**情绪波动**。

**代码描述核心**：
```plaintext
吸引力 + 适当推拉/不确定性 = 性张力 (情绪波动)
```
许多聊天技巧（如推拉、话术）的核心目的就是制造这种性张力。但请注意，高价值的人或陌生人不允许低价值的人随意波动自己的情绪，这会被视为冒犯。因此，性张力的制造需要建立在一定的吸引和安全感基础之上。

## 人格筛选的使用方法

理解了核心概念后，我们来看看如何具体操作人格筛选。

**使用时机**：在“意愿锁定”及“身份输出”之后，即可直接开启人格筛选。

**提问的三种方式**：
以下是三种有效的提问方向，可以帮助你展开筛选。

1.  **询问对某事的看法**：例如，“我朋友遇到了……事，你怎么看？”
2.  **对对方的过去展开调查**：例如，“你谈过几次恋爱？”、“为什么和上一任分手？”
3.  **考核对方未来的意愿**：例如，“你喜欢旅游吗？擅长做攻略吗？”、“对未来有什么规划？”

**关键执行要点**：提出筛选问题后，**不要立即结束话题**。要围绕这个问题延伸讨论，让对方提供细节。
**好处**：
*   让对方感觉聊天很走心。
*   你能更真实地了解对方的想法（判断其是“有思想”还是“脑子空空”）。

## 实战案例解析

让我们通过一个简化案例，看看人格筛选在对话中如何展开。

**对话背景**：双方已初步认识，并经过寒暄。

> **你**：既然是以恋爱为目的发展，那我们就相互了解一下吧，不合适也早点知道。
> **女生**：嗯嗯，从哪里开始了解呢？
> **你**：你对男朋友怎么样？会心疼人吗？ **(开启筛选)**
> **女生**：当然啊，谈恋爱不就是相互体贴、照顾、心疼吗？
> **你**：哎呦，好懂的样子。不过说得对，我也很会照顾女友，尤其女朋友来大姨妈的时候。 **(延伸话题，分享自己，创造合拍感)**
> **女生**：是啊，只有一个人付出，另一个很累也走不远。
> **你**：那你倒是成本不高（笑）。你在家会做饭吗？ **(结束上一个话题，开启新筛选)**
> **女生**：会啊，你看我朋友圈有发，疫情时天天做。
> **你**：哈哈都会做什么菜呀？ **(继续延伸细节)**
> **女生**：湖南菜吧。
> **你**：你是湖南哪里人？…我也蛮喜欢做饭的，以后可以一起。 **(将筛选转化为未来画面，并了解基础信息)**

**案例分析**：这个案例展示了一种“假性筛选”，主要目的并非真的审判对方，而是通过问答形式创造“彼此很合拍”的幻觉，同时自然交换基本信息，让关系立体化，为后续发展铺路。

## 平衡的艺术：价值、心态与技巧

最后需要强调的是，任何话术技巧都不是万能的。真正有效的关系建设依赖于三者的平衡：

**价值**：你的自身建设（展示面、实力）。价值高能提升容错率和框架。
**心态**：你的内在稳定性。心态好能抵御挑衅，甚至临场发挥出“神之一手”。
**技巧**：你所学的聊天方法（如本课的人格筛选）。技巧能有效制造吸引和推进关系。

**公式**：`稳定关系 ≈ 价值 * 心态 * 技巧 (三者需平衡，偏废任何一方都会出问题)`
单纯依赖话术会引发冲突，单纯依赖心态会显得空洞，单纯依赖价值可能吸引错误的人。三者对立统一，找到平衡点才能游刃有余。
