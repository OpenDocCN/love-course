# 乌鸦救赎《连招绝技-白驹聊天秘术》：09.白驹聊天秘术

## 概述
在本节课中，我们将学习如何运用白驹聊天秘术，通过人格筛选来建立更深层次的吸引和联系感。

## 核心概念
**人格筛选**：通过提问和对话，了解对方的内在性格、价值观和生活方式，从而建立更深层次的吸引和联系感。

### 定义
1. **展示自身价值的契机**：让对方展示自己的价值，从而合理化你的热情。
2. **打开内在窗口的机会**：深入了解对方，建立更深刻的吸引和联系感。
3. **服从性塑造方式**：通过对话形式，让对方感受到你的框架和地位。
4. **了解真实内在的机会**：了解对方的性格、才华、家庭背景等，帮助你做出是否继续交往的决定。

### 使用方式
1. **意愿锁定和身份输出后**：在确定关系后，开始人格筛选。
2. **了解对方看法**：通过提问，了解对方对某个事件的看法。
3. **调查对方过去**：了解对方的经历和背景。
4. **询问意愿**：了解对方的未来规划和兴趣。
5. **延伸讨论**：不要立即结束话题，而是深入探讨，让对方有细节感。

### 例子
1. **对男朋友的看法**：了解对方对伴侣的期望和价值观。
2. **恋爱经历**：了解对方的情感经历和性格特点。
3. **未来规划**：了解对方的职业发展和生活目标。
4. **生活习惯**：了解对方的生活方式和兴趣爱好。

## 总结
本节课中，我们学习了如何运用白驹聊天秘术进行人格筛选，通过深入了解对方，建立更深层次的吸引和联系感。希望同学们能够将所学知识应用到实际生活中，提升自己的社交能力。
