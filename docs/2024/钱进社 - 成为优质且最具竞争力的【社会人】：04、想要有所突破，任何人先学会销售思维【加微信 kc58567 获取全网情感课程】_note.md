# 钱进社 - 成为优质且最具竞争力的【社会人】：04、想要有所突破，任何人先学会销售思维

嗨，这里是酷姑小红书公众账号都是猫总酷姑。今天我们来聊一聊任何人都应该学习和重视的销售思维。无论你的工作是否与销售相关，我们群里有很多伙伴都希望了解销售技巧。

**明确思维与技巧的区别**

首先，我们要明确思维和技巧的区别。高手注重思维，而讲师强调技巧。思维关注人和人性，而技巧可能只是一些不一定完全有用的手段。

**成为销售高手的思维层面**

当你希望成为一个销售高手时，一定要侧重在思维层面。以下是一个案例，展示了如何运用销售思维：

**案例分析**

19年1月13日，我在朋友圈发布了一条关于朋友在新公司遭受排挤的内容。我去找他吃饭后，借机与老板聊天，帮助他在公司不再被排挤。

**案例细节**

1. **公司状况**：公司规模不大，老板是门外汉，不盈利。
2. **员工状态**：员工眼神空洞，缺乏朝气。
3. **老板特点**：有热情，对事情上心。

**试探老板**

我选择试探老板，是因为以上三点判断中，节奏、客观条件、分寸缺一不可。我穿着偏成熟，观察老板的细节，发现他对自身有一定要求。

**试探过程**

1. **穿着**：穿着得体，给老板留下好印象。
2. **观察**：老板是中年男人，没有油腻感，衣服平整，裤长合适。
3. **试探**：在老板画大饼时，轻声细语地对我朋友说：“这个事情你擅长啊。”

**老板反应**

老板愣了一下，反问我朋友是否擅长。我朋友磕磕绊绊地回答，表现出不自然。

**进入办公室**

老板邀请我们一起进办公室聊天，透露出两个信息：1. 我接触过类似项目；2. 我朋友有相关经验。

**成功原因**

1. **自信**：我不是小白，不会hold不住场，不会冒险。
2. **了解老板**：老板是门外汉，烧钱会让他焦虑。
3. **利他**：我为他提供价值，没有竞争关系。

**总结**

通过这个案例，我们可以看到，销售思维的核心在于细节和手段。真正的手段是建立思维架构，精准判断，达成目的，还做了利他的事情。

**本节课总结**

本节课中，我们一起学习了如何运用销售思维。希望你能从中获得启发，并将其应用到实际生活中。
