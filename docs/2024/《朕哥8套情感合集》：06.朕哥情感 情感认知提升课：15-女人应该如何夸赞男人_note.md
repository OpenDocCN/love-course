# 《朕哥8套情感合集》：06.朕哥情感 情感认知提升课：15-女人应该如何夸赞男人

🎼 在本节课中，我们将学习如何有效地夸赞男人，让夸赞充满真实感和煽动性。

## 一、夸赞的核心逻辑：推拉

首先，我们需要了解夸赞的核心逻辑——推拉。**推拉**是指通过正负、加减的方式，让男人在情感上忽高忽低，从而增加互动的趣味性和吸引力。

## 二、夸赞技巧

以下是四个实用的夸赞技巧：

### 1. 少夸人，多夸事

**公式**：夸赞他的行为，而非个人。

* 例如：他给你做饭，你可以说：“你真用心，做的饭真好吃，我好感动。”

### 2. 大事小夸，小事多夸

**公式**：大事时，关注他的付出；小事时，多提及他的行为。

* 例如：他连续工作几个月给你买礼物，你可以说：“辛苦你了，我都感觉好心疼你。这个礼物虽然贵重，但我更感谢你的付出。”

### 3. 别人转述自己否认

**公式**：通过别人的夸赞，自己进行否认，制造反差感。

* 例如：朋友夸他帅，你可以说：“哎呀，我怎么没看出来呢？”

### 4. 少夸外在，多夸内在

**公式**：夸赞他的内在品质和能力。

* 例如：夸他负责任、成熟、会照顾人等。

## 三、总结

本节课中，我们学习了如何通过夸赞技巧提升情感互动的趣味性和吸引力。希望这些技巧能帮助你更好地与男人相处，让感情更加甜蜜。
