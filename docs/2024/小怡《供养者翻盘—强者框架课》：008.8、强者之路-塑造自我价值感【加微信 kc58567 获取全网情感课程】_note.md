# 小怡《供养者翻盘—强者框架课》：008.8、强者之路-塑造自我价值感

🎼hello，兄弟你好，这节课我们聊一个非常核心的话题——社交价值。这节课将直接解决困扰许多兄弟们已久的自卑问题。

**社交价值的核心**

社交价值指的是一个人对他人或社会的用处。它可以是有形的，如在工作中的收入贡献；也可以是无形的，如外貌带来的愉悦感。

**自卑的根源**

自卑源于内心深处的不合理信念，如认为只有高收入、高颜值、高地位的人才有价值。

**内在社交价值**

内在社交价值是指强大的内心力量，即使面对外界贬低也能保持自信。

**外在价值与自信**

外在价值可以带来自信，但这种自信是脆弱的，容易因外界变化而动摇。

**价值与阶层**

价值尤其是能区分阶层的价值，其实是被上层虚构出来的，并不存在。

**价值输送型与价值展示型**

价值输送型是指过度讨好，价值展示型是指过度炫耀。这两种方式都是自卑的体现。

**总结**

人格价值不接受外界定义，唯一的定价者是你自己。只有充盈强大的内心力量，才能带来真正的人格魅力。
