# 《二月老师的核心自信》（综合版）：25-25：行为模式1_ev

## 概述
在本节课中，我们将探讨思维模式与行为模式之间的关系，以及如何通过改变行为模式来影响思维模式，从而提升自信。

## 思维模式与行为模式的关系
思维模式和行为模式是相互依存的。**公式**：思维模式 → 行为模式 → 社交反馈 → 思维模式。

- **思维模式**：你的内在信念和认知。
- **行为模式**：你的外在行为和反应。
- **社交反馈**：他人对你的行为的反应。

## 改变行为模式的方法
以下是两种主要的方法来控制身体与心理的状态：

### 1. 控制身体
通过生理上的把控来提升状态。

- **动作**：站立、呼吸、面部表情等。
- **例子**：站成八字形，看着天花板，手叉腰，用力且饱满的呼吸，露出牙齿，傻乎乎地笑。

### 2. 控制心理焦距
调整你的思考焦点。

- **例子**：专注于当下，避免过度思考。

## 改变行为模式的影响
改变行为模式可以影响思维模式，从而提升自信。

- **例子**：当你情绪低落时，通过改变呼吸方式和思考焦点来提升情绪。

## 社交力学
在社交力学中，不做错事和做正确的事一样重要。

- **例子**：避免证明自己有资格，因为这会玷污互动的氛围。

## 总结
本节课中我们一起学习了思维模式与行为模式之间的关系，以及如何通过改变行为模式来影响思维模式，从而提升自信。希望你能掌握这些方法，并在日常生活中运用它们。
