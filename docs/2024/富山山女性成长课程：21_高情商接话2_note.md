# 富山山女性成长课程：21_高情商接话2

## 概述
在本节课中，我们将学习如何在高情商的交流中接话，以及需要注意的几个关键点。

## 不要急于接话
**公式**：不要急于接话，以免显得局促和不镇定。

- 不要在对方阐述观点前急于插话。
- 抢话在别人看来是非常讨厌的。

## 放慢语速和行动
**公式**：放慢语速和行动，以显得更加从容。

- 懂得慢几秒钟，放慢语速。
- 放慢眼神和眼皮子的威严感。

## 不要急于反驳
**公式**：不要急于反驳，即使对方观点错误。

- 世界没有绝对的对错。
- 允许不同的声音和价值观存在。
- 不要认为自己的观点绝对正确。

## 不要显得过于随和卑微
**公式**：不要显得过于随和卑微，以免失去自我。

- 不要一味地赞同或迎合。
- 保持自己的态度和观点。

## 高情商接话的例子
**代码**：
```plaintext
男生：你觉得这家店怎么样？
女生：其实你选的地方是很棒的，但是我觉得如果这个牛肉在怎么怎么样的话会更好一点。其实呢我吃过一间店，虽然说没有那么贵，但是肉质比这个更好。如果你有兴趣的话呢，下次我可以带你去吃。
```

## 总结
本节课中我们一起学习了高情商接话的几个关键点：不要急于接话，放慢语速和行动，不要急于反驳，不要显得过于随和卑微。希望这些技巧能够帮助你在交流中更加自信和从容。
