# 富山山女性成长课程：21_高情商接话2_note

在本节课中，我们将要学习高情商接话的几个核心要点。这些技巧能帮助你在交流中显得更沉稳、更有智慧，从而提升个人魅力和沟通效果。

## 要点一：切勿着急接话 ⏳

上一节我们介绍了高情商沟通的基础，本节中我们来看看第一个关键点：不要着急接话。着急会显得你非常局促，显得你不淡定、不镇定。一个能“压得住场”的女性，关键在于懂得慢下来。

以下是具体做法：
*   **放慢节奏**：懂得慢几秒钟再回应，放慢语速，放慢所有行动和眼神。
*   **保持耐心**：在对方尚未完全阐述清楚观点时，不要抢话。抢话在他人看来非常令人讨厌，显得你迫不及待。
*   **深入思考**：利用停顿的几秒钟，在脑中多思考一下。想想能否对对方的观点提出更高维度、更有认知深度的新视角。

在跟高阶的男性相处时，经常打断对方阐述是非常掉价的行为，这会证明你的文化程度、个人修养和家教有所欠缺。这一点必须注意。

## 要点二：避免反驳，学会倾听 👂

我们明白了耐心的重要性，接下来看看交流中的另一个禁忌：反驳。在聊天接话的过程中，不要去反驳对方，无论你认为他是对是错。这个世界没有那么多绝对的对错，只是大家站的角度和价值观不同。

小时候我们都学过一篇课文《画杨桃》，不同的观察角度会画出完全不同的形状。这教给我们一个最基本的道理：要允许世界上存在不同的声音和价值观。不要认为自己坚信的东西就一定正确。

当别人与我的价值观相冲时，我从不反驳。我会采用更温和的方式：
*   “可能你是对的。”
*   “我们看问题的角度不太一样，我可能有别的看法。”
*   “你为什么会有这么新颖的角度呢？”
*   “我想知道一下，你为什么会这么觉得？”

你要做的是敞开心扉去询问，了解是什么造就了今天的他，他为何会得出这样的观点，而不是去反驳和指正。

许多女性在交往中常犯指正对方的错误。例如，伴侣用牙刷涮杯子，而你认为直接用水龙头冲洗更卫生。这时，不断指正“你得改，这不卫生”并非上策。这些是边缘化的信息，无需强迫别人与你有同样的观点。你只需要把他的漱口缸洗得特别干净即可。

在聊天中，无需指正别人的价值观、审美或想法。这个世界可以有601种不同的声音，你只需要倾听。相反，不去指正、只知道倾听和深挖对方内心动态的人，反而能得到更宽广的视角。

## 要点三：切忌过于随和与讨好 🚫

除了不反驳，在接话中还要避免显得过于随和、卑微和讨好。很多人习惯在对方刚说完后就连连附和：“是是是”、“嗯嗯嗯”、“对对对”、“赞同赞同”。这会让对方觉得你没有自己的想法，只是在讨好。

我分享一个自己的例子。有一次在欧洲，一位追求我的男士带我去一家高档熟成牛排店。吃完后他问我：“你觉得这家店怎么样？” 这其实是一个试探性问题，他想了解我的消费层级和对事物的认知水平。我从他的口吻和表情看出，他认为这家店的牛肉并不怎么样。

我的回答是：“其实你选的地方是很棒的。但是我觉得如果这个牛肉在……方面处理一下会更好。我吃过另一家店，虽然没那么贵，但肉质更佳。如果你有兴趣，下次我可以带你去。”

这样的接话既没有让对方丢面子，也没有显得自己混沌、没主见或没见过世面。在面对你认为“不是那么回事”的问题时，也需要有自己的态度，不要一味卑微地讨好和迎合。简单回答“挺好的”、“好吃呀”没有任何意义。

为什么有些女性能让男性逐步佩服和欣赏？正是通过这些聊天和相处的细节探索出来的，让对方觉得你是一本读不完的书，是一个有自己想法的人。没有想法、一味附和和卑微的女性是低价值的。在接话过程中，我们既要显得有趣，又要不卑不亢。
