## 聊天技巧

1. **打招呼**：避免夸赞，从对方资料中挖掘有价值信息进行开场。
2. **聊天核心**：围绕邀约展开，避免漫无目的的瞎聊。
3. **聊天节奏**：以普通朋友状态进行聊天，避免过度迎合和目的性过强。
4. **话题选择**：从轻松愉快的话题开始，逐步深入到个人基本信息和价值交换。
5. **表达方式**：多表达自己的想法和感受，避免一味迎合。

1. **打招呼时不要随便夸赞**
2. **慎用打压**
3. **不要骂人**
4. **不要浪费自己和别人的时间**
