# 21、凯男的社交情商课：第五节课：七步聊天：007.七步聊天-7确定邀约【微kc58567】

## 概述
在本节课中，我们将学习如何通过七步聊天法的最后一步——确定邀约，来成功邀请心仪的女生。

## 七步聊天法回顾
1. **开场白**
2. **距离感过度**
3. **基本信息交换**
4. **建立男神框架**
5. **抛出筛选标准**
6. **升高关系和预期植入**
7. **确定邀约**

## 确定邀约的重要性
如果你在前六步中做得很好，那么邀约就变得简单了。你需要确保你已经建立了足够的吸引力和舒适感。

## 模糊邀约与确定邀约
- **模糊邀约**：在建立吸引和安全感时，可以提出模糊邀约，例如：“改天有时间我们可以一起去咖啡厅坐坐。”
- **确定邀约**：在确定邀约时，要明确时间、地点和活动内容。

## 邀约细节
1. **提前邀约**：至少提前一天邀约，以便对方有足够的时间安排。
2. **确定约会主题**：明确约会的内容，例如：“明天晚上我们一起去喝杯东西。”
3. **选择熟悉地点**：选择你熟悉的地方，避免意外情况发生。
4. **适合转场的地点**：选择周边设施繁华的地点，方便转场。
5. **避免带闺蜜**：直接拒绝对方带闺蜜，专注于与对方单独相处。

## 总结
本节课中，我们学习了如何通过七步聊天法的最后一步——确定邀约，来成功邀请心仪的女生。记住，邀约的关键在于前期吸引力和舒适感的建立，以及明确邀约细节。希望这节课对你有所帮助。
