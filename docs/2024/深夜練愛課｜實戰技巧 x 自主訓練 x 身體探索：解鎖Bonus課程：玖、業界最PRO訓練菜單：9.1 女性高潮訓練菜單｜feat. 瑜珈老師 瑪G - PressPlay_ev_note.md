# 深夜练爱课 09：解锁女性高潮训练菜单 🧘‍♀️

在本节课中，我们将学习一套专为女性设计的训练方案，旨在通过强化核心、臀部及大腿内侧肌群，提升骨盆底肌力量与身体感受力，从而在亲密关系中增强控制感与愉悦体验。

这套训练由三个主要部分组成，我们将依次进行。

## 第一部分：骨盆底肌与核心启动 🌀

上一节我们介绍了课程目标，本节中我们来看看如何启动骨盆底肌。许多女性反馈难以感知会阴处的发力感，因此首先设计两个动作来激活该区域。每个动作重复10次，建议完成3组。

请准备瑜伽垫和一个瑜伽砖或枕头。

**动作一：骨盆卷动**

1.  仰卧在垫上，将砖块或枕头夹于大腿之间。这有助于大腿内侧和骨盆底肌自然发力。
2.  双手在耻骨上方围成三角形，放松骨盆。
3.  **吸气**时，尾骨向前，使腰椎微微离地，骨盆前倾。
4.  **呼气**时，尾骨向内卷回，同时**夹紧大腿**并感受轻微的憋尿感，将气完全呼出。
5.  重复此呼吸与动作配合。

**动作二：核心稳定训练**

强化核心有助于在多种姿势中保持更久。

1.  保持砖块夹于大腿间，坐直，感受脊柱延长。
2.  **呼气**时，专注夹紧砖块，控制身体缓慢向后倾，直到腹部感到紧绷发力。
3.  **吸气**时，坐直身体。
4.  重复此过程。注意肩膀放松，避免耸肩。

## 第二部分：臀肌强化训练 🍑

完成了骨盆底肌的激活后，我们进入第二部分。强健的臀肌不仅能塑造浑圆臀型，也能在亲密时刻发挥作用。

以下是针对臀肌的训练动作。

**动作一：臀桥**

1.  仰卧，手掌心朝上放于臀部两侧，肩背放松。双脚踩地，位置大约在膝盖斜下方。将砖块夹于大腿间。
2.  **吸气**预备，**呼气**时脚踩地，将臀部、腰部、背部依次抬离地面。
3.  在顶端感受臀部发力，夹紧砖块，**吸气**停留。
4.  **呼气**时，从背部开始逐节缓慢放下身体，在即将触地时再次抬起。

**动作二：俯身后抬腿**

这个动作对抗重力，能训练臀部与大腿后侧，塑造微笑曲线。

1.  俯卧，双手交叠，额头轻靠手背。双膝弯曲，脚跟与脚掌相触。
2.  **吸气**预备，**呼气**时发力，脚跟相互推挤，使大腿前侧与膝盖离地，臀部收紧。
3.  **吸气**时放松，**呼气**时再次上抬。

## 第三部分：大腿内收肌专项训练 💪

现在来到最后也是至关重要的部分，针对女性相对薄弱的大腿内收肌进行训练。锻炼此处能同步刺激骨盆底肌。

以下是两个侧躺动作，请务必完成双侧练习。

**动作一：侧躺蚌式**

1.  侧躺，双膝并拢弯曲，脚跟与臀部呈一直线。手可轻扶骨盆以感知稳定。
2.  **吸气**预备，**呼气**时上方膝盖打开，感受臀部发力。
3.  **吸气**时微微放松，**呼气**时再次打开。
4.  **进阶**：将下方脚抬离地面，在空中完成开合，这需要更强的稳定与协调能力。

**动作二：侧躺腿内收**

1.  侧躺，上方腿伸直抬起，与骨盆同高。下方腿为主要活动腿。
2.  手可轻扶地面或叉腰以保持平衡。**吸气**预备。
3.  **呼气**时，将下方腿向上抬起，去触碰上方腿，使双脚大拇指并拢。
4.  **吸气**时微微放松，**呼气**时再次上抬。注意核心收紧，腹部不放松。
