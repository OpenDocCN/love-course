# 老公出轨心理27堂课：09 男人出轨为什么会逼离婚？

## 概述
在本节课中，我们将探讨男人出轨后为何会逼迫妻子离婚的心理动机和策略。

## 婚姻管理系统
朱生勇，婚姻管理学派的创始人，拥有10年婚姻管理经验，专注于挽救婚姻、处理婚外情、离婚疗愈和复婚重建。

## 案例分析
一位妻子因孩子学习问题回归家庭，却发现丈夫出轨。丈夫以没有感情为由逼迫离婚，妻子感到困惑和痛苦。

## 逼离婚的七大类型
1. **先发制人**：丈夫在妻子发现出轨前提出离婚，以防止妻子采取行动。
2. **维护尊严**：丈夫为了维护自己的尊严和面子，选择离婚。
3. **控制妻子**：丈夫通过提出离婚来控制妻子，使其无法采取行动。
4. **安抚小三**：丈夫为了安抚小三，提出离婚作为交代。
5. **彰显真爱**：丈夫认为真爱小三，选择离婚以追求真爱。
6. **真爱妻子**：丈夫认为与妻子不匹配，选择离婚。
7. **攻击妻子**：丈夫通过提出离婚来攻击妻子。

## 逼离婚的策略
1. 冷暴力
2. 家暴
3. 经济控制
4. 分居
5. 离婚协议
6. 找父母
7. 找法院
8. 气死你来离婚

## 逼离婚的心理动机
1. 表达不满
2. 满足小三
3. 控制妻子
4. 内心意愿
5. 攻击妻子

## 真离婚与假离婚
**真离婚**：
1. 态度平和
2. 意志坚定
3. 方案明确
4. 方案有利于女方
5. 积极行动

**假离婚**：
1. 情绪失控
2. 忽冷忽热
3. 没有离婚方案
4. 抱怨背后渴望
5. 真离婚后怂了

本节课中，我们学习了男人出轨后逼迫离婚的心理动机和策略。了解这些可以帮助妻子更好地应对丈夫的逼离婚行为。
