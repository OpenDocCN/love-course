# 高情商聊天话术 社恐变社牛 🗣️💬：14. 第十节，约会中做真实的自己

在本节课中，我们将要学习如何通过保持真实自我和稳定心态，成功完成从线上聊天到线下约会的过渡。我们将通过一个真实的案例，分析约会成功的关键要素，并总结整个课程的核心思想。

---

上一节我们探讨了如何建立吸引和舒适感，本节中我们来看看如何在漫长的互动中保持耐心，并最终促成约会。

我曾经运用课程中教授的聊天技巧，与一位姑娘持续互动了约两三个月。在此期间，她回复信息并不积极，甚至时常不回复。

但我并未因此焦虑或追问。因为我同时在与多位女性聊天，所以并不特别在意她回应的积极程度。

我坚持每天与她分享生活中发生的有趣事情，让她逐渐了解我的生活全貌与为人。虽然她有时不回复，但通过我的朋友圈和日常分享，她慢慢知晓了我的性格、职业等信息。

三个月后的一天晚上，她突然主动发信息问我是否想吃宵夜。我当晚正好有空，便答应了这次邀约。

见面吃宵夜时，我问了她一个问题。我承认这个问题可能有些直接，但我很想知道原因：为什么之前三个月都不怎么回复我，却突然约我出来？

我们最初是在一个社交软件上认识的，当时聊得不错。她解释说，她有一个习惯：如果觉得一个男人还不错，她会刻意疏远对方一段时间，观察对方是否仍愿意继续交流。

她认为，这种方式可以筛选掉那些只想“开车”或玩玩而已、并非真心的男人。她说：“你坚持了三个月，即使我不怎么回复，你也一直找我聊天。我觉得你这个人肯定挺不错的，所以愿意约你出来。”

然而，她并不知道我能做到这一点的原因。原因有以下两点：

首先，我确信自己没有犯错。这是许多男性缺乏的基本判断——女生稍有不悦，他们就认为自己真的做错了什么。只有深刻认识自己，客观判断自身行为是否正确，才能知道下一步该如何行动。我知道自己从未说过令她反感的话或做过烦扰她的事，她也没有明确拒绝我，因此我认为自己仍有机会。

其次，我同时在与很多人聊天，因此根本不在乎她是否回复。这是心态上的关键优势。

这是我经历过的一个比较极端、难以邀约的案例。最终能够成功约出，正是基于以上几点。

因此，这告诉大家，当你遇到一位感觉不错的姑娘时，切勿急躁。最忌讳的是在前期聊天中，甚至与对方争吵或互怼。

她与你未曾谋面，没有义务必须回复你的信息或答应你的邀约。如果你选择不理她，但需知每一位漂亮的姑娘通常都有众多追求者。

你要想从中脱颖而出，必须拥有自己的特点。坚持是一种特点，心态稳定是一种特点，幽默感也是一种特点。总之，你需要有至少一个突出的特质。

如果你毫无特点，那么你的所有行为都与大多数追求者无异，自然难以成功。

我也遇到过另一种情况：例如在社交软件上当天认识，当晚邀约，对方就答应出来的妹子。

这种妹子通常情场经验丰富。不要认为对方快速答应是因为你魅力超高，很可能只是因为对方此刻空虚寂寞，或者她本身就是比较开放的人。她回复不积极未必是你的问题，她快速答应也未必证明你做得完全正确。

大家必须明白一个道理：互动中出现的各种情况，可能与你的关系并不大，而更多是对方自身的原因。即使你将自己这方面的事情全部做好，最多也只占成功因素的50%。因此，不要轻易心态崩溃。

我见过太多求助者或学员，当一切顺利时，他们感觉自己像世界之王；一旦遭遇些许挫折或拒绝，便内心崩溃，甚至进入与对方敌对的状态。这些都是错误的做法。

---

上一部分我们分析了不同约会案例的心态差异，接下来我们总结一下约会的核心意义。

约会的意义在于推进关系，尤其是建立肢体接触。我们课程中讲授的所有技巧和方法，最终目的都是帮助你促成这次约会。

作家韩寒曾说过一句话：如果一个姑娘愿意单独和你出来约会、吃饭、看电影或散步，那么这意味着她愿意和你睡觉。这是他的原话，虽曾引发争议，但我个人比较认同。

她既然愿意单独与你约会，就说明对你已有好感。如果对你毫无好感，她不会愿意单独出来。请务必记住：如果她不愿意单独赴约，那肯定对你没有好感；如果她已经单独出来，说明或多或少对你有好感。

只要你在接下来的约会中不犯重大错误，并懂得在适当时机做出正确行为，你就有可能成功推进关系。

很多人会怀疑：真的只要对方愿意单独出来就可以了吗？那是因为你可能在约会现场表现不佳，或者聊天水平有限，或个人展示面没有做好。

---

现在，让我们对整个课程的核心进行一次总结。

我们所有课程的内容，从前期的认识到最终邀约，都是在为你打下基础。前期聊天的核心在于：**轻松有趣**，并让对方逐渐了解你是一个什么样的人。

请记住：前期聊天的目的并非努力表演或试图打动对方。你需要展示的是一个值得认识的、优秀的自己，而不是通过大量付出去感动对方，让对方觉得你“很努力”或“将来会对他更好”。

这不是前期需要展示的重点。我给大家举个例子：这就像选择苹果手机还是小米手机。人们选择苹果，并非因为其售后服务更好，或功能比小米强大多少，而是因为苹果品牌给人高大上的感觉。请把自己想象成苹果手机来理解这个逻辑。

人们都倾向于选择那些有价值、能带来轻松舒适愉快感受的人或事物，而不是选择一个只会跟在身后、竭力满足自己所有需求的“服务者”。

这种心态贯穿男女交往的始终。你需要时刻问自己：在对方心目中，你是一个什么样的人？

如果你感觉在对方心中，你已是一个备胎，是他众多追求者中不起眼的一个，甚至是一个他不尊重、看不起的人，那么你的所有聊天互动都可能导致坏结果。

反之，如果你们是平等交流，甚至他有些尊重、仰视或崇拜你，那么即使你犯了一些小错误也无妨。

因此，在约会中，让对方看到你是一个什么样的人，真的非常重要。

本节课我们一起学习了如何通过保持真实与稳定心态来促成约会。我们分析了一个长期互动后成功邀约的案例，总结了约会成功的两个关键点：**客观的自我认知**与**丰富的社交选择**所带来的心态优势。同时，我们明确了约会的核心意义在于**推进关系**，并再次强调了前期聊天应注重**展示价值**而非**讨好付出**。

我们讲授的所有方法，都是为了帮助你从认识到一个妹子，到聊天熟悉、建立好感，最终成功将其约出来。如果你认真学习了这十节课的内容，完成这一过程应该不成问题。
