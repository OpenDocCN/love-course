# 嘴叔《聊天技巧及心态》：第四节，如何通过聊天的潜沟通建立高位

## 概述

在本节课中，我们将学习如何通过聊天的潜沟通技巧建立高位，避免在聊天中处于低位，从而提升自己在对方心中的形象。

## 核心概念

**公式**：高位沟通 = 自信 + 尊重 + 适度调侃

## 教程内容

### 1. 明确聊天目的

*   **思考问题**：你在聊天中是在娱乐对方还是在娱乐自己？
*   **正确做法**：以娱乐自己为主，偶尔调侃对方，让对方感觉轻松愉快。

### 2. 避免过度娱乐对方

*   **错误做法**：为了逗对方开心，牺牲自己的尊严和感受，成为小丑。
*   **正确做法**：敢于与对方开玩笑，让对方感受到你的自信和魅力。

### 3. 调侃对方的方式

*   **错误做法**：将自己当作小朋友，说话幼稚，让对方觉得你幼稚。
*   **正确做法**：将对方当作小朋友，用想象力调侃对方，让对方感受到你的幽默和智慧。

### 4. 避免过多使用语气助词

*   **错误做法**：过度使用语气助词，让对方觉得你过于主动，有需求感。
*   **正确做法**：适度使用语气助词，保持聊天的自然流畅。

### 5. 适度使用表情包

*   **错误做法**：过度使用各种表情包，让对方觉得你过于幼稚。
*   **正确做法**：适度使用调侃式或暧昧表情包，展现你的幽默和魅力。

### 6. 赞扬对方的方式

*   **错误做法**：居高临下地赞扬对方，让对方觉得你是在跪舔。
*   **正确做法**：认同对方，用平等的态度赞扬对方，让对方感受到你的尊重。

## 总结

本节课中，我们学习了如何通过聊天的潜沟通技巧建立高位，避免在聊天中处于低位。希望同学们能够将所学知识应用到实际聊天中，提升自己在对方心中的形象。
