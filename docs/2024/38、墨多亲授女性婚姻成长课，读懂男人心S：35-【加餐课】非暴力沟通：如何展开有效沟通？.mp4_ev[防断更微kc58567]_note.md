# 墨多亲授女性婚姻成长课：非暴力沟通教程 📚

## 概述

在本节课中，我们将学习非暴力沟通（NVC）的核心概念，了解如何通过有效沟通改善人际关系。

## 第一部分：感受的根源

**感受来自于需求**

*   **公式**：感受 = 需求 - 满足度
*   **代码**：`emotion = need - satisfaction`

**体会自己的感受和需要**

*   当你感到难过或生气时，思考背后的需求是什么。

**体会他人的感受和需要**

*   当你的伴侣晚归时，表达你的需求：“我希望你早点回家”，而不是指责。

## 第二部分：正确表达需求

**避免指责**

*   **例子**：将“你总是那么晚回来”改为“我希望你早点回家”。

**积极表达需求**

*   **例子**：将“你总是那么晚回来”改为“我希望你尽量在1点之前回家”。

**管理情绪**

*   学习识别和承担情绪，避免情绪控制行为。

## 第三部分：有效表达请求

**具体、明确**

*   **例子**：将“别把外套乱放”改为“请把外套挂到衣架上”。

**避免含糊不清**

*   **例子**：将“你们应该尊重我”改为“我希望你们在进我房门之前先敲门”。

**明确谈话目的**

*   在谈话前明确目的，例如改善关系或消除误会。

## 第四部分：倾听和感恩

**倾听他人**

*   当对方没有学过NVC时，耐心倾听，理解他们的感受和需求。

**应对暴力沟通**

*   保持冷静，避免情绪化反应。

**感恩**

*   感谢对方与你沟通，并表达你的感激之情。

## 总结

本节课中，我们一起学习了非暴力沟通的核心概念和技巧。通过正确表达需求、有效表达请求、倾听他人和感恩，我们可以改善人际关系，建立更加和谐的家庭和社交环境。
