# 《朕哥8套情感合集》课程05：朕哥核心私密课之如何引导男人付出
## 第21讲：底层逻辑：如何读懂男人 👨💼

在本节课中，我们将要学习如何读懂男人的底层逻辑。理解男人的思维模式和行为特征是引导其付出的基础。我们将从男女思维差异入手，并介绍一系列具体的观察和判断方法。

男女的思维逻辑存在差异。上一节我们介绍了引导付出的核心框架，本节中我们来看看理解男人的起点。男人的思维比较理性。他考虑一件事时，思维逻辑很强。

例如，一个理性很强的男生，可能不愿意为你买一双2万元的高跟鞋，因为他认为这不实用。但他可能愿意为你买一台2万元的电脑，因为他觉得这很实用。这就是男女的一个核心差距：男人更理性、更现实，思维更加直线。而女生则更感性、更浪漫，更容易被氛围带动。

想要读懂一个男人，需要掌握几个核心观察点。

以下是第一个核心观察点：朋友圈是了解男人的一个重要窗口。对女生而言，朋友圈可以用于展示自己；对男生而言，我们可以通过其朋友圈判断他的城府深度。

例如，如果他的朋友圈设置为“仅三天可见”或直接关闭，那么这种男人城府通常较深，他不希望暴露太多信息。城府深的男人不希望别人了解他，因为他觉得没有必要，也没有炫耀的心态。而城府浅的男生则很喜欢炫耀，例如买了新手机或新车都要发朋友圈。不喜欢发朋友圈，甚至朋友圈不可见的男人，城府相对较深。比如朕哥的朋友圈就设置为三天可见，并且很少发朋友圈，因为我没有强烈炫耀某件事的冲动。

如果你发现一个男生天天在朋友圈发一些内容，例如抱怨下班堵车、上班迟到，或是半夜抒发情绪，那么你就知道，这个男人的城府很浅，并且社会阅历不多。

第二点是观察朋友圈的内容。正如刚才所说，内容能反映男人的段位和理解力。通过他喜欢分享旅游、科技还是其他类型的内容，可以判断他的兴趣和认知层次。所以，你一定要学会看男人的朋友圈。

接下来，我们看看如何通过头像判断个性。在我的私教课中对此有详细讲解。头像代表了男人的个性。

例如，使用卡通头像的男人通常比较有童心，可能怀旧。使用真人高清自拍、展示自己很帅头像的男人，则属于比较自恋的类型。自恋的人往往伴随另一个特质：自我。通过这些观察，我们能够判断一个男人的档次和想法。通常，档次越高的人越低调，就像马云穿布鞋一样。从头像就能窥见他的心思。

上一部分我们讨论了外在的社交展示，接下来我们看看如何通过身体状态进行判断。男人的身体能反映出很多信息。

第一是看气色。气色好、有精神，至少代表他睡眠充足、心情愉悦。由此可知他最近没有烦心事。如果男人气色差、状态低迷，则代表他最近可能处于低谷期。因此，如果你觉得这个男人气色好，多半最近有高兴事，你可以夸夸他。如果他气色差，可能正处于低谷期，这时你不需要多说什么，可以给他泡一杯红枣枸杞水，给予关怀。

第二是看体能。男人的体能反映了很多问题。良好的体能代表他有毅力、身体健康，并且有时间规划锻炼。这与他整体的自律性和生活状态相关。

除了宏观状态，细节更是判断男人段位的关键。细节决定成败，也决定了这个男人到底是什么层次。

在约会时，首先要看他是否很自我，以及对弱势群体的态度。例如，如果服务员不小心把水泼到他身上，他对此破口大骂，这说明他为人自我且素质极低。因此，在约会中要观察男生是否有良好的素养、共情能力和沟通技巧。

其次，聊天是观察男人共情能力的重要场合。如果一个男人具备共情能力，他将来很可能成就很好的事业。如果没有共情这个基本能力，男人很难发达。共情能力的核心本质是理解他人的能力。如果一个男人善于理解别人，那么他多半也懂得如何经营事业。

最后，亲密互动能看出他是否尊重女性。例如，他是否刚认识就动手动脚，是否急于占便宜。你要观察他到底有没有尊重你，这一点非常重要，但很多女生都忽略了。

本节课中我们一起学习了读懂男人的底层逻辑。我们从男女思维差异出发，介绍了通过朋友圈、头像、身体状态、细节、聊天和亲密互动等多个维度来综合判断男人的性格、段位和态度。掌握这些方法，是有效引导男人付出的重要前提。
