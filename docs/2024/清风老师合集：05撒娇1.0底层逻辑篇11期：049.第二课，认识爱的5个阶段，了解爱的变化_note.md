# 清风老师合集：05撒娇1.0底层逻辑篇11期：049.第二课，认识爱的5个阶段，了解爱的变化

## 概述
在本节课中，我们将学习爱的五个阶段，了解一段感情从相识到相守的变化过程。

## 第一阶段：莫生气
这个阶段是所有浪漫的开始，双方并不相识，不了解对方的状况和过去。但凭借共同的兴趣爱好，开始了社交旅程。

## 第二阶段：暧昧期
这个阶段双方有大量互动沟通，制造浪漫和惊喜，关系逐渐确立。在这个过程中，双方感到心动、快乐和满足。

## 第三阶段：热恋期
此时激情四溢，对方的优点被放大，感到温暖舒适，想要不断靠近。双方享受彼此的美好和喜悦。

## 第四阶段：磨合期
在这个阶段，双方发现现实与期待不符，开始感到失望。对方对需求的满足发生变化，双方期待也发生变化。磨合期矛盾急剧增加，需要智慧和能力去处理。

### 磨合期的常见处理方式
1. 沉默以对：冷战。
2. 逃避回避：不面对问题。
3. 频繁争吵：因小事产生冲突。

## 第五阶段：稳定期
经历了磨合期的探讨和摸索，找到了彼此想要的相处方式。感情得到升华，关系更加稳定。双方保持关心，同时不影响原有生活。

### 长期关系的四要素
1. 在社交圈公开关系。
2. 真心付出。
3. 共同的兴趣和爱好。
4. 持续的学习和成长。

## 显性关系与隐性关系
显性关系包括上升、平行和下沉三种社交模式。隐性关系是控制与被控制的关系。磨合期问题的核心原因是显性关系的阶段变化。

## 总结
本节课中，我们学习了爱的五个阶段，了解了磨合期的特点和解决方法。在下一节课中，我们将深入剖析磨合期矛盾的三大特点，以及如何破除恶性循环，让关系重回温暖和甜蜜。
