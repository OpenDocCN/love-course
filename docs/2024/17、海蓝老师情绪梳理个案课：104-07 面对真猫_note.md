# 课程17：海蓝老师情绪梳理个案课 104-07 面对真猫 🐱

在本节课中，我们将跟随一位学员的体验，学习如何运用“爱”与“关怀”的情绪，逐步面对并梳理内心对特定事物（本例中是一只猫）的恐惧与厌恶情绪。整个过程强调安全、渐进与自我觉察。

---

## 概述

本节课记录了一次情绪梳理的实践过程。学员最初对猫感到恐惧和厌恶，通过在海蓝老师的引导下，带着“爱”逐步靠近、观察一只真实的猫，最终实现了情绪的转化与放松。我们将通过学员与老师的对话，拆解这个情绪梳理的步骤与核心技巧。

---

## 情绪梳理步骤详解

上一节我们概述了本次课程的目标，本节中我们来看看具体的操作步骤。整个过程由老师引导，学员在绝对安全、不被勉强的前提下逐步推进。

### 第一步：建立安全与设定意图

老师首先确保学员处于心理安全的环境中，并明确梳理的意图是带着“爱”去观察。

*   我们带着爱来看这只猫。
*   整个过程不会勉强你做任何事情。
*   你任何时候不舒服都可以停下来。
*   你可以慢慢睁开眼睛看。

### 第二步：初始观察与感受确认

学员开始观察一只受惊的猫，并如实报告自己即时的情绪感受。这是情绪觉察的起点。

以下是学员的初始观察与感受：
*   看这只被吓坏的猫。
*   它满眼都是恐惧。
*   我感觉它挺无辜，评分约2分（恐惧程度，0为无，10为极度）。

### 第三步：逐步接近与动态觉察

老师引导学员在物理和心理上逐步靠近目标（猫），同时持续关注自身感受的变化。每一步推进都需征得学员同意。

以下是逐步接近的过程：
1.  **靠近一点**：老师询问“能否再往前走一点”，学员同意。
2.  **坐下观察**：老师引导“能坐下来吗”，学员坐下继续观察猫发抖的样子。
3.  **感受变化**：学员感受变为“觉得它挺可怜”，评分仍为2分。
4.  **处理厌恶**：当学员提到看到猫眼联想到“狸猫换太子”并产生厌恶时，老师允许并接纳这种感受，然后引导继续观察现状。
5.  **再次靠近**：老师询问“能再靠近点吗”，学员同意但声明不会触摸。
6.  **身体接触准备**：老师引导学员将手放在引导者（白雪）的腿上，作为间接接触的过渡。
7.  **尝试轻触**：在学员感觉“还行”后，老师引导其用手轻触猫毛，但不强求。

### 第四步：主动互动与感受深化

当学员的防御感降低后，引导进行更主动的互动，并对比感受前后的差异。

以下是互动与深化的关键点：
*   **持续抚摸**：老师引导“一直摸着”，并看着猫的眼睛。
*   **感受当下**：学员反馈“不怕了”，身体放松。
*   **对比变化**：老师提问“现在主动摸的感觉和最开始有什么不同”。
*   **新感受产生**：学员描述，现在带着一种“关怀”和“补偿”的感觉去摸，而非单纯的恐惧或厌恶。
*   **理解反应**：老师解释猫的尾巴动是“本能反应”，帮助学员理性解读行为，减少误解带来的恐惧。

### 第五步：结束梳理与总结收获

在情绪明显转化后，适时结束练习，并总结梳理前后的变化，巩固新的认知。

以下是结束与总结部分：
*   **结束确认**：双方确认“可以了”，停止互动。
*   **对比变化**：学员自述，与昨天相比差距巨大，从“恶心、不敢看、害怕”变为现在状态。
*   **情绪延伸**：梳理后，学员感到轻松，并联想到与母亲的关系，产生了新的、更平和的沟通愿望。

## 核心概念与技巧

上一节我们回顾了完整的步骤，本节中我们来提炼其中的核心概念与技巧。

*   **安全第一**：整个过程的基石是“不勉强”，尊重学员的节奏。代码式原则可表示为：`if (感到不适) { 可以随时暂停 }`。
*   **渐进暴露**：采用小步骤、渐进式的方法接近恐惧源。公式可概括为：**情绪梳理 ≈ 微小步骤 + 持续觉察**。
*   **感受命名与量化**：不断询问“感受是什么”，并用0-10分量化，使模糊情绪具体化。
*   **区分过去与现在**：老师帮助学员区分“过去联想”（如电视剧里的猫）和“当下现实”（眼前发抖的小猫）。
*   **身体引导**：通过“身体往前挺”、“手放腿上”等指令，帮助学员通过身体动作影响心理状态。
*   **关怀视角转化**：引导感受从“恐惧/厌恶”转向“可怜/需要关怀”，这是情绪转化的关键。

## 总结

在本节课中，我们一起学习了如何通过一次面对真猫的个案，进行深度的情绪梳理。关键点在于：在绝对安全的环境中，带着爱意进行渐进式暴露，持续觉察并命名感受，最终实现从恐惧厌恶到平和关怀的情绪转化。这个过程不仅适用于特定恐惧对象，其方法也适用于梳理其他由过往经验引发的负面情绪。学员最终的轻松感及延伸到家庭关系的感悟，证明了情绪梳理带来的广泛积极影响。
