# 《leon全套课程合集》：leon 3.0 - 11.废物测试

## 概述
在本节课中，我们将学习如何应对女生在搭讪、聊天和约会过程中给出的挑战和测试，即所谓的“废物测试”。

## 什么是废物测试？
**废物测试**是指女生在交流过程中对你提出的一些挑战和测试，例如质疑你的年龄、评价你的魅力或怀疑你的动机等。

## 男生常见的应对方式
在女生进行废物测试时，男生通常会陷入以下两种局面：
1. **自我证明和解释**：这会让你显得过于追求女生，导致聊天变得沉重，或者与女生发生冲突，最终导致聊天破裂。
2. **硬碰硬**：直接与女生对抗，这种做法同样不可取。

## 正确的心态
女生进行废物测试是正常的，她们在有意无意中筛选和判断男人。因此，我们应保持良好的心态，不要过于激动或愤怒。

## 处理废物测试的四种方式
以下是处理废物测试的四种策略：
1. **忽视**：适用于中后期，当女生对你的兴趣不大时。
2. **反抛问题**：适用于前期，通过了解女生的观点和想法，重新定义问题。
3. **扭曲**：将女生的问题进行重新定义，使其更加积极。
4. **夸张**：在后期，通过夸张的方式让女生觉得你不在意，从而提升好感。

## 不同情况的应对策略
以下是针对不同情况的废物测试的应对策略：
1. **女生对你一点兴趣都没有**：忽视。
2. **女生对你有一点兴趣**：反抛问题，了解她的观点，重新定义问题。
3. **女生对你兴趣较大**：扭曲问题，使其更加积极。
4. **女生对你进行无理取闹的挑战**：忽视或反压。

## 总结
通过正确处理废物测试，我们可以大大提升在女生面前的好感度，从而顺利进入下一个阶段，即关系升温环节。本节课中，我们一起学习了如何应对废物测试，希望对你有所帮助。下节课再见！
