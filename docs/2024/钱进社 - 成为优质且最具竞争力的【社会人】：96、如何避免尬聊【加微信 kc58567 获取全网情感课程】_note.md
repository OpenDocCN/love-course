## 一、自我定位
**公式**：自我定位 = 外在形象 + 人际关系

1. **外在形象**：通过穿着、妆容等方式调整自己的外在形象，降低距离感。
2. **人际关系**：根据自己的人际关系定位，了解自己在朋友中的定位。

## 二、沟通技巧
**公式**：有效沟通 = 目的 + 好奇心 + 共同点

1. **目的**：将目的放在心里，关注对方的好奇心。
2. **好奇心**：对对方产生好奇心，提出多元问题。
3. **共同点**：寻找与对方的共同点，建立连接。

## 三、氛围营造
**公式**：良好氛围 = 时间 + 次数 + 性格

1. **时间**：通过时间和次数了解对方的性格。
2. **次数**：多与对方交流，建立信任。
3. **性格**：根据对方的性格调整沟通方式。

## 四、提供价值
**公式**：提供价值 = 实际价值 + 情绪价值 + 服务价值

1. **实际价值**：提供实际帮助，如介绍工作、资源等。
2. **情绪价值**：提供情绪支持，如倾听、安慰等。
3. **服务价值**：提供服务，如帮忙解决问题等。

## 五、刻意练习
**公式**：表达能力 = 阅历 + 环境 + 认知

1. **阅历**：积累丰富的人生经历。
2. **环境**：身处良好的环境，不断试错。
3. **认知**：提升认知水平，拓展视野。
