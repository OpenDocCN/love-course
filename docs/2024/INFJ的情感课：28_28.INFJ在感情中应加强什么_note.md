# INFJ情感课 28：INFJ在感情中应加强什么？💖

在本节课中，我们将探讨INFJ人格类型在情感关系中需要重点加强和补足的方面。课程将围绕其核心认知功能展开，涵盖优势功能的合理运用、劣势功能的针对性训练，以及阴影功能的谨慎管理。通过学习，INFJ可以更好地经营亲密关系，促进双方共同成长。

## 优势功能Ni的加强：避免过早定性🔮

上一节我们介绍了课程概述，本节中我们来看看INFJ的首要优势功能——内向直觉（Ni）。这个功能需要加强的地方在于避免滥用。

Ni功能具有犀利的洞察力，能经常看穿事物的本质。然而，在与对方了解不深的情况下，直接运用Ni得出结论，容易在心里给对方贴上标签。

感情是两个人共同成长、共同进步的过程。过早地在内心对人进行定性判定，等于在限制对方变得更好、变得不一样的可能性。因此，Ni功能在此阶段不应将人看得太透。

你需要明白，再过30年，你的Ni功能会比现在强大很多，那时你看人可能会得出不同的结论。所以，在当下年轻时，不要将人看得太死，定性到“跳进黄河也洗不清”的地步。这是需要注意的第一点。

## 辅助功能Fe的加强：发展Fi，建立深层连接🤝

接下来，我们看看辅助功能外向情感（Fe）。你可以加强的一点是：在使用Fe的同时，适当训练你的内向情感（Fi）。

使用Fe时，你更多在意的是让对方开心，相对不那么在意对方情感的具体缘由。你会觉得情感的具体原因是对方的事，与自己关系不大。

但Fi功能在两人关系中扮演着重要角色。当你对对方情感的来由产生更多好奇时，你就能与对方建立更深层的联系。如果仅熟练使用Fe，你与对方的关系可能停留在表面。这种表面关系意味着，在一起时你可以哄对方开心，但永远触及不到他内心真正的痛点。

以下是Fi功能能帮助你触及的深层情感领域：
*   原生家庭的阴影
*   真正的创伤
*   各种求而不得
*   内心的不甘与痛苦

对于Fe主导的INFJ而言，可能不愿触碰这些事。但感情需要两个人完全赤诚相待。若想长久在一起，建议尽早开发Fi，而非单纯依靠Fe去哄好对方。

这就好比对方不高兴，你不去探究原因，却不停地往他嘴里塞糖，直到他不得不微笑。真正的问题并未解决。因此，适当开发Fi，好奇对方为何不开心，并鼓励对方倾诉，这对你的感情生活非常有帮助。

## 劣势功能Se的加强：管理预期，避免突发状况🎯

我们已经探讨了优势功能，现在将目光转向劣势功能。INFJ的第三功能——外向实感（Se）——主要体现为在生活中不喜欢惊喜。

这种“不喜欢惊喜”是指，对方突然送你礼物或准备生日惊喜，对你而言甚至可能是一种冒犯。你会觉得，对方在没有预先说明的情况下突然制造惊喜，让你手足无措。

同样，生活中对方可能突然做出决定，比如突然想去某地而提前未告知，这会让你因毫无心理准备而感到不舒服。

面对这一点，你需要加强与对方建立共识：尽可能说清楚，生活中不要突然做对方完全想不到的事情。即便动机出于善意，但作为INFJ，你内心真的不喜欢这种毫无准备、手足无措的感觉，这更容易让你感到惊慌。

提前向对方表达这一点，要求对方尊重你的习惯，就可以在感情中减少那种“突然来个惊喜，吓得心跳加速、呼吸紊乱”的情况。

## 劣势功能Ti的加强：正向思考，对抗思维循环🔄

另一个劣势功能是内向思维（Ti）。这个功能容易让INFJ陷入思维循环，尤其是在自我否定和自卑的状态下，形成 **`Ni - Ti 循环`**。

因此，INFJ需要多看一些正向的内容。这些内容往往与Fe功能相关，主题是“世界上许许多多的人都值得被爱”。你需要不断强化这个信念。

你需要通过看书、欣赏作品、观看电视剧或动画片等过程，不断确认人性的善良，确认人天生就是宝贵的存在。更多地确认这一点，对你的Ti进行正向思考非常有帮助。

否则，当你面对复杂世界时，你的Ni很容易看到许多负面东西，你的Ti随之附和，让你觉得面对阴暗的世界只能思考负面的事情，进而将这些负面想法带到自我评价之上。

所以，关于Ti，你需要多做的是：**多接触正向内容**。

## 阴影功能的管理：警惕极端，避免消耗⚡️

关于优势功能和劣势功能，我们已经覆盖了四点。接着，我们来说阴影功能。INFJ在使用阴影功能（即镜像功能）时，会变得有点像ENFP。这通常发生在自己搞不定某个情况时，会更多地向ENFP的方向倾斜。

此时，INFJ需要注意：当你迫于压力突然变成ENFP时，切勿做事走极端。因为如果你突然全然不顾地以ENFP的方式生活，由于平时并未主动训练这些能力，会非常消耗能量。

即便看完本课程，你也会发现，时刻使用镜像功能并不容易，可以偶尔使用，但时刻使用并非易事。因此，一旦被逼到变成ENFP的状态，你必须提醒自己：这不是我的正常状态，也不是我的最佳状态。我不能认为变成这样后就可以为所欲为，就可以充分发ENFP的优势能力（Ne和Fi）。

虽然我建议你锻炼Fi，但同时我也建议：对于每一个阴影面的功能都不要使用过度。因为使用过度会极度消耗你的能量，让你整个人无法用精力保持乐观。

### 阴影面的劣势功能：Te与Si

现在，我们具体说一下阴影面的劣势功能。对于ENFP（即INFJ的阴影面）而言，其劣势功能是外向思维（Te）和内向实感（Si）。这两个功能对INFJ来说是“劣势中的劣势”。

**1. 阴影功能Si：避免陷入细节强迫**

Si功能是指，INFJ可能会突然变成一个细节控，想通过做家务、收集整理东西来让内心平复。但同时，因为此功能处于劣势，经常出现过度的情况。

例如，一开始收拾就停不下来，消耗大量时间。这在感情中虽不直接造成伤害，但会让自己处于一种强迫症状态，在生活中很难感到满意，也会让伴侣感到一种歇斯底里。

因此，这是INFJ需要尽可能加强的一点：对于Si，尽可能避免过量使用。如果你要使用，请给自己一个时间限制。

**具体方法是：**
*   给自己定一个闹钟。
*   如果要收集整理或做家务，只做20分钟。
*   超过20分钟就停止。

用这种方式有意识地训练你的劣势功能。

**2. 阴影功能Te：控制暴怒时的言行**

另一个劣势功能Te，会让INFJ在暴怒（非常生气）时，说出特别伤人的话，并表现出与平时温顺性格完全不同的霸道语气。

关于这个功能，我的建议是：平时可以看一些逻辑性的书，锻炼自己的逻辑思维。但和Si功能一样，要给自己时间限制，不要无节制地使用，以免过度消耗精力。

多进行逻辑思维锻炼，有助于在你突然失控时，依然能保持较强的逻辑能力，不至于直接破口大骂。

这些都是对INFJ非常有用的要点，希望每位INFJ能记在心里，并在生活中多多实践这些有效建议。

## 总结📝

本节课中，我们一起学习了INFJ在感情中需要加强的各个方面：
1.  **优势功能Ni**：需避免过早对人定性，为彼此的成长留出空间。
2.  **辅助功能Fe**：需搭配发展Fi功能，探索对方情感根源，建立深层连接。
3.  **劣势功能Se**：需明确表达对“突发惊喜”的不适，与伴侣建立管理预期的共识。
4.  **劣势功能Ti**：需多接触正向内容，引导思维循环走向积极，避免陷入负面自我评价。
5.  **阴影功能**：需警惕在压力下转向ENFP模式时的极端倾向，并有意识地管理阴影劣势功能Si（避免细节强迫）和Te（控制暴怒言行），通过设定限制来练习。

通过有意识地加强优势、补足短板、管理阴影，INFJ可以在情感关系中更加从容、深入，与伴侣实现更健康的共同成长。
