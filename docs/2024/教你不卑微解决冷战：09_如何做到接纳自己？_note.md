# 教你不卑微解决冷战：09_如何做到接纳自己？

你是否经常因为追求完美而陷入困境，是否对自己和伴侣的一丁点小问题过于挑剔，导致关系紧张和不愉快。这种完美主义的心态，可能让你感到沮丧和焦虑。在我们的日常生活中，我们都希望与伴侣建立深厚而美满的关系。

## 概述

在本节课中，我们将学习如何通过接纳自己来改善人际关系，摆脱完美主义的困扰，拥抱真实的幸福生活。

## 接纳的关键：降低敏感度

接纳的关键是降低敏感度。接纳并不意味着放任一切，而是学会调整我们对小问题的敏感度。

### 脱敏

**公式**：脱敏 = 逐步减少对小问题的过度关注和反应

**步骤**：

1. **目标设定**：确定需要脱敏的目标，如减少或消除特定恐惧或焦虑。
2. **建立层次**：将敏感的刺激源分为不同的层次，从较轻微的引发焦虑的刺激开始，逐渐增加到更具挑战性的刺激。
3. **逐级放松**：运用放松技巧，如深呼吸、渐进性肌肉松弛或冥想。
4. **构建脱敏层次**：从最低层次开始，对刺激源进行想象，并同时运用放松技巧。
5. **渐进暴露**：在想象中面对较低层次的刺激源而不产生过多的焦虑反应时，开始进一步提升对刺激的接触。

### 情绪合理化

**公式**：情绪 = 合理的

**步骤**：

1. **接受情绪**：无论情绪是消极还是积极，都是合理的。
2. **合理化情绪**：理解情绪是生理性的，出现情绪是正常的。
3. **暗示自己**：在心里暗示自己，这是正常的。

## 案例说明

### 案例一：蜘蛛恐惧

**目标**：降低对蜘蛛的敏感度。

**步骤**：

1. **建立层次**：看蜘蛛的照片、视频、瓶子里的蜘蛛、玻璃钢的蜘蛛、手伸进装有蜘蛛的玻璃缸。
2. **逐级放松**：在接触蜘蛛的过程中，运用深呼吸、冥想等方法缓解焦虑。
3. **渐进暴露**：逐步增加接触蜘蛛的程度，直到能够淡定从容地面对蜘蛛。

### 案例二：外界评价敏感

**目标**：降低对外界评价的敏感度。

**步骤**：

1. **建立层次**：陌生人的评价、同事的评价、亲级的评价、好朋友的评价、亲人的评价、爱人的评价。
2. **逐级尝试**：从最低层次开始，选择一个陌生人，尝试不表达感谢，观察结果。

## 总结

本节课中，我们学习了如何通过降低敏感度和情绪合理化来接纳自己，从而改善人际关系。希望这些方法能够帮助你摆脱完美主义的困扰，拥抱真实的幸福生活。
