# 王哥私教课：王哥情商课基础课 - 第八节 情商课之你是谁

## 概述
在本节课中，我们将探讨情商的基础——了解自己是谁。我们将学习如何通过改变形象和提升自我认知来提升情商。

## 什么是情商？
情商是指一个人理解和管理自己情绪的能力，以及与他人建立良好关系的能力。高情商的人能够更好地沟通、解决问题，并在社交场合中表现出色。

## 情商的基础：你是谁？
**公式**： 情商 = 自我认知 + 情绪管理 + 社交能力

### 你是谁？
- 你是谁？这个问题看似简单，但却是提升情商的关键。
- 你的职业、出身等因素可能无法改变，但你的形象和身材是可以改变的。

### 改变形象的重要性
- **视觉效果**： 改变形象可以立即改变视觉效果，给人留下深刻印象。
- **自信提升**： 好的形象可以提升自信，让你在社交场合更加从容。

### 如何改变形象？
1. **打扮**： 学习如何打扮自己，提升视觉效果。
2. **身材**： 通过饮食和锻炼改变身材，展现健康和活力。

### 先进依山再进佛
- 这句话意味着，在提升情商的过程中，首先要关注基础，然后再逐步提升。
- 在社交场合中，人们首先关注的是你的形象和身材，然后才是你的谈吐和思想。

## 总结
本节课中，我们学习了情商的基础——了解自己是谁。通过改变形象和提升自我认知，我们可以提升情商，更好地与他人建立良好关系。希望这节课对你有所帮助。
