# INFJ情感课：03_3.INFJ的灵魂伴侣是什么样的

🎼大家好，我是生西创始理事。本期课程我们将探讨INFJ灵魂伴侣的特点。许多小伙伴可能想知道，如果身边有灵魂伴侣，他们是什么样的，有没有一个标准来识别。

## 标准一：共情心

灵魂伴侣必须对你有共情心。这意味着他们对你有一种不忍的感觉。当你难受、流泪时，他们不忍心看到你这样，并愿意帮助你。这对于INFJ来说尤为重要，因为INFJ的能量往往是一种悲伤或难过的能量，需要释放。

**公式**：共情心 = 不忍看到对方痛苦 + 愿意帮助对方

## 标准二：MBTI倾向

在MBTI上，灵魂伴侣本质上没有要求，但大致上有一个倾向性：

- 对于E型INFJ，建议与P型人交往，因为INFJ在亲和力上更倾向于P型人。
- 对于I型INFJ，建议与N型人交往，如IP型或EN型，因为内外向不同，看待世界的方式不同。

**代码**：倾向性 = P型或N型

## 标准三：能力与气质

灵魂伴侣在能力上通常比你强，这包括学习、赚钱、成熟或学问等方面。在气质上，他们必须比你更冷静，在重要问题上比你更镇定。

**公式**：能力 = 强于你 + 气质 = 更冷静

## 标准四：教育水平

灵魂伴侣的教育水平不能比你低。这是INFJ内心需要知道的一点。

**公式**：教育水平 = 不低于你

## 总结

通过以上四个标准，你可以更好地识别和衡量生活中的潜在灵魂伴侣。记住，灵魂伴侣并非一成不变，随着价值观的变化，两个人可能不再是灵魂伴侣。保持开放的心态，珍惜生活中的每一个可能。
