# 18、坏小k女人一生必学的一堂课：视频课 03. 冲破思维规则 拥有独立思考

OK，我们继续讲解如何突破思维规则。首先，思维有四种认知层次。我将帮助大家进行一次深刻的蜕变。

### 思维的四种认知层次

#### 1. 环境层次

处于环境层次思维的人，典型的思考模式是“这都是你的错”。环境指的是你之外的一切，如身边的人、领导、公司、竞争对手、市场环境等。

- **例子**：工作不顺利，因为领导是个白痴；房子太贵买不起，因为黑心的炒房团；喜欢一年的男生被别人三天带走，抱怨男生、绿茶或自己。

#### 2. 行为层次

处于行为层次思维的人，典型的思考模式是“我还不够努力”。

- **例子**：找不到优质人，因为自己不够努力；收入太低，因为自己不够努力。

#### 3. 能力层次

处于能力层次思维的人，典型的思考模式是“方法总比问题多”。

- **例子**：线下门店生意不好，因为经营模式太陈旧，需要学习新的方法；和男朋友关系处理不好，因为沟通能力问题。

#### 4. 信念、价值观及框架（BVR）

处于BVR层次思维的人，思考的是“究竟什么才是最重要的”。

- **例子**：选择专业或技能时，考虑长远规划，进行投资；商家考虑顾客的长期消费，而非一次性的生意。

### 提升问题思考的深度

#### 1. 长度

拉长时间的角度看待问题，进行人生规划。

#### 2. 高度

提升思维高度，从不同角度看待问题。

#### 3. 深度

强化深度，洞察事物本质。

### 总结

本节课中，我们学习了如何突破思维规则，拥有独立思考。希望这些内容能帮助你更好地面对人生、事业、生活情感上的挑战。做最好的自己，关注自己，而非他人。
