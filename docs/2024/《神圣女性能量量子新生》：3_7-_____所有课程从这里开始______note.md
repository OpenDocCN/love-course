## 🔄 学习方法论：从信息到内化

上一节我们确立了学习心态，本节中我们来看看具体的学习方法。如果本课程仅仅教授信息，那么它将与其他课程无异。信息的确能为你开启新世界的大门，但若想让这些认知成为你身体的一部分，形成**潜意识的记忆**，则需要你投入时间和能量进行内化。

内化的最佳方法是：听完每节课后，积极并踏实地完成课后作业。

以下是关于学习过程的两点核心建议：

1.  **重复聆听**：你只能在当前的意识层级接收课程信息。因此，鼓励你在时间允许时重复听课。潜意识最喜爱重复，在这个过程中，你底层的信念会不知不觉地发生改变。
2.  **轻松学习**：不必将本课程想象成化学或数学那种充满男性能量的学习方式。你可以用轻松、好玩的女性能量方式来学习。例如，在整理衣物、开车或上班间隙，将课程内容作为背景音播放。

## 🧘 接收状态：保持临在与开放

在聆听课程内容时，请尽量屏蔽头脑中固有的评判声音，让自己保持临在状态。如果你不断用旧有理论去评判新内容（例如心想“这个我懂”），你便在那一刻关闭了接收新信息的开关。

请先抱持好奇，以一张白纸的状态去接收课程中的所有内容。
