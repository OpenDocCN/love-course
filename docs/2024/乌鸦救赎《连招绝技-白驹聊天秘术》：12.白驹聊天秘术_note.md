# 乌鸦救赎《连招绝技-白驹聊天秘术》：12.白驹聊天秘术

## 概述

在本节课中，我们将学习如何通过“七天情侣”仪式确定恋爱关系，并探讨建立关系的意义和优势。

## 关系的本质

**关系** 是指彼此对未来相处模式做出的承诺，包含以下方面：

* **行为一致性承诺**： 确定关系后，某些行为被允许，例如性关系。
* **称呼提醒**： 通过称呼提醒对方如何与自己相处，例如“老公”、“老婆”。
* **唯一性承诺**： 亲密关系具有唯一性，意味着不再选择他人。
* **改变称呼标志契约建立**： 通过改变称呼，例如从“朋友”到“老公”，标志契约建立。

## 七天情侣的原理和优势

**七天情侣** 利用中国人“先精神后身体”的认知路径，通过模拟情侣状态，快速建立关系。

**优势**：

* **缩短恋爱时间**： 快速让女人进入状态。
* **第一次约会即可牵手**： 不用劳心博弈。
* **退出机制**： 7天试用，可免责。

## 七天情侣的操作步骤

1. **铺垫**： 表达想了解与对方恋爱感觉的想法。
2. **提出建议**： 建议进行“七天情侣”模拟。
3. **解除顾虑**： 提前合理化，解决对方可能存在的顾虑。
4. **确定关系**： 通过改变称呼，例如“老公”，标志关系确立。
5. **后续互动**： 继续聊天，拉高沉默成本。

## 使用心得

* **不要快速推进，要解除焦虑**： 节奏要慢，聊的质量要高。
* **七天情侣之后要继续做信任**： 通过互动拉高沉默成本。
* **约会时不用拘谨**： 带入角色，直接牵手。
* **关注对方的情绪**： 对方情绪高涨时，推进关系更丝滑。

## 总结

本节课我们学习了如何通过“七天情侣”仪式确定恋爱关系，并探讨了建立关系的意义和优势。通过合理运用技巧，我们可以快速建立关系，并让感情更加稳定。
