# Cary老师《权贵的秘密》：第四章 如何正确使用情绪价值_：让塑料朋友成为你的贵人，如何让

在本节课中，我们将要学习如何正确运用负面情绪价值。很多人误以为负面情绪就是发泄和争吵，但实际上，它是一种重要的沟通工具，用于立规矩和引导关系向有利方向发展。我们将拆解其核心逻辑与正确使用方法。



上一节我们介绍了正面情绪价值的铺垫作用，本节中我们来看看如何将负面情绪转化为有效武器。首先需要明确，负面情绪价值的作用是**立规矩**和**引导补偿**，而非单纯的情绪发泄。



许多人在使用负面情绪时存在误区。以下是两种常见的错误做法：

1.  **价值高者无理取闹**：即使个人价值较高，长期通过作闹、删除拉黑等方式处理问题，也会消耗对方耐心，最终导致关系破裂。
2.  **价值低者过度迎合**：因自卑或过度仰望对方，而不敢展示任何负面情绪，生怕对方离开。这会导致无法获得真正的尊重与珍惜。



因此，学会正确运用负面情绪价值至关重要。不必害怕展示负面情绪会导致关系结束。如果对方因此离开，通常只源于三个原因：

1.  你的正面情绪价值铺垫不足。
2.  你使用了错误的负面情绪表达方式。
3.  你本身的个人价值不够高。



那么，正确的负面情绪价值表达方式是怎样的？请记住一个核心公式：

**正确的负面情绪表达 = 情绪 + 难过的原因 + 目的**



接下来，我们详细拆解这个公式的每一步。首先，“情绪”部分是指你要明确展示出你的感受。

在表达负面情绪时，你可以哭泣、可以显得崩溃或伤心。关键在于让情绪可视化，但不要失控。



表达完情绪后，第二步是清晰地陈述“难过的原因”。这需要具体、客观地指出对方行为或事件本身的问题。

以下是两个具体操作的例子：

*   “我原本以为你是一个比较靠谱的人，我真的有打算跟你长期发展下去的想法。但是我和你接触久了之后，我才发现你是一个比较自卑、只想着自己、平时连一点仪式感都没有的人，我真的是有点失望了。”
*   “我为了这个家付出了这么多，我以为我们会有一个很好的以后，但是你却背着我在外面乱搞，我真的对你太失望了。”



完成了前两步，最后也是最关键的一步是提出“目的”。很多关系在冲突后崩盘，正是因为缺少了目的导向，陷入了无止境的情绪对抗。

当你的价值足够高，且正面情绪价值铺垫充分时，对方在接收到你的失望信号后，通常会产生补救心理。此时，你需要引导对话走向解决方案。



所以，不要停留在作闹阶段。你的目的可能是寻求经济补偿，或是设定新的关系规则来修复信任。你必须有一个清晰的目标，并引导对方共同达成。



最后，在向上社交的过程中，请永远记住：**情绪是你的武器**。你可以用它让男人开心、振奋，也可以让他收敛、畏惧，从而产生各种情绪波动。但唯独你自己，不能有一丝真正的波动。你必须将情绪牢牢掌握在自己手中，而不是被情绪所控制。



本节课中我们一起学习了负面情绪价值的正确使用方式。核心在于通过“情绪+原因+目的”的公式，将负面感受转化为设立规矩和引导补偿的有效工具，从而掌控关系走向。
