# 安老师社恐训练营：06-克服社恐：打破旧模型_建立新模型_note

在本节课中，我们将要学习克服社交恐惧的核心战略：打破旧的恶性循环模型，并建立新的、健康的社交模型。我们将深入探讨实现这一目标的三个主要方法及其具体实践技巧。

上一节我们介绍了社交恐惧的恶性循环模型及其影响。本节中，我们来看看如何打破这个循环。核心战略只有一个：**打破旧模型，建立新模型**。

通过分析模型，我们可以总结出三个主要方法来打破恶性循环：
1.  减少自我关注。
2.  改变思维模式。
3.  改变行为模式。

在开始讲解这些方法的具体操作之前，需要先了解一些通用的练习原则和技巧。以下是三个核心原则：

**原则一：分阶段练习**
可以将社交场景的训练分为想象训练和真实训练。想象训练是在心中模拟令你害怕的社交场景；真实训练则是真正去面对令你焦虑的真实场景。无论是哪种训练，都包含三个步骤：**社交前、社交中、社交后**。每一步都需尽可能详细地想象和感受。

**原则二：循序渐进**
训练时应将社交场景按焦虑程度划定等级。改变需要科学的方式，训练时应从低焦虑等级的场景开始，逐步向高等级过渡。只有低等级场景训练通过后，才去挑战更高等级的场景。

**原则三：正念放松**
运用正念的思想，有意识地察觉当下，不被习惯性的思维或行为模式束缚。通过放松和思考，为社交注入新的想法。核心是关注当下，**不评判、不强求**。

以上原则是后续所有练习的基础。现在，我们开始逐一讲解三个核心方法。

## 1. 减少自我关注 👁️

自我关注是社交恐惧的核心层面，因此需要优先处理。无论身处何种场合，对自我关注越少，就越容易表现出真实的自己。

以下是减少自我关注的三个练习方法：

**练习一：转移注意力**
此练习旨在训练你灵活控制注意力的能力。
*   **做法**：首先，将注意力集中在自身（如呼吸、躯体感觉）上保持2到3分钟；然后，将注意力转移到外界事物（如观察一本书的细节）上保持2到3分钟。
*   **目标**：反复练习，直到可以轻松地在自我关注与外界关注之间来回切换。如果注意力涣散，努力将其重新集中。对自己看到的事物保持好奇心，会使训练更容易。

**练习二：感受自我意识的影响**
此练习帮助你认识到过度自我关注的危害。
*   **做法**：针对每个社交场景，事后问自己两个问题：
    1.  我对自己得出了什么结论？
    2.  我觉得别人对我下了什么结论？
*   **目的**：审视这些结论是否让事情变得更糟。明白过度的自我关注会占据全部精力，使你难以领会他人话语，从而更容易避免这一倾向。

**练习三：训练好奇心**
此练习通过将注意力引向外界，自然减少对自身的关注。
*   **做法**：主动对周围的人或事物产生好奇和兴趣。
*   **效果**：你对他人或讨论的事情越关注，对自身表现和恐惧念头的关注就会越少。

当你熟练运用这些方法减少自我关注后，会发现社交体验的性质发生积极改变，心情也会舒畅很多。

上一节我们学习了如何减少自我关注。接下来，我们看看第二个关键方面：改变思维模式。

## 2. 改变思维模式 💭

本节提供一些重要策略，帮助你应对那些引发糟糕感受的想法，从根本上阻止社交恐惧发展。主要分为两步：**识别你的想法** 与 **寻找替代方法**。

你可以借助“想法记录表”来实践这个过程，它能帮助你保持开放态度，避免迷失在恐惧中。

以下是想法记录表的应用示例：

**示例一**
*   **场景**：跟某人说话时卡住，不知说什么。
*   **焦虑想法**：“他一定觉得我令人失望，觉得我很笨。”
*   **积极的替代想法**：“我无从得知他在想什么。”、“他并没有嘲笑我，也许他只是觉得我在思考怎么说。”
*   **行动方案**：当谈话卡住时，自然放松，并说：“稍等，让我想一下。”

**示例二**
*   **场景**：在聚会上与一个人谈话。
*   **焦虑想法**：“自己没有做什么贡献（如买菜、安排座位），好像什么都没做。”
*   **积极的替代想法**：“不是完全没有，我心里有很多想祝福的话，只是没有说出来。”
*   **行动方案**：思考怎么开始谈话，或先听听其他人怎么说。

**示例三**
*   **场景**：最后进入会议室，被人盯着。
*   **焦虑想法**：“太丢人了，别人肯定会觉得我搞特殊、不爱工作、偷懒。”
*   **积极的替代想法**：“人们盯着刚进门的人是正常行为，大多数人只是条件反射地看一眼，并没有太多关注。”
*   **行动方案**：正常进入座位，按部就班做该做的事。

填写想法记录表时，要尽可能详细，按照自己真实的想法去写。

此外，还需警惕一些常见的负面思维类型，它们影响巨大：

以下是八种常见的负面思维类型：
1.  **非黑即白**：认为事情只有两种极端可能。例如：“像明星一样才有吸引力，我这种人一点魅力都没有。”
2.  **妄下负面结论**：认为不好的事情一定会发生。例如：“别人一定看出了我的紧张，肯定觉得我是一个没用的人。”
3.  **否定正向体验**：认为正向的体验不算数，归因于外部。例如：“这次社交顺利，主要是因为对方人好，不是我的努力。”
4.  **情绪推理**：将自己的想法强加于人。例如：“我觉得自己很愚蠢，那别人一定也这么看我。”
5.  **以偏概全**：因小部分负面信息推导出整体负面结论。例如：“有个人对我态度不好，看来我并不适合参加此类活动。”
6.  **读心术**：认为自己知道别人在想什么。例如：“同事刚才看了我一眼但没聊天，他一定觉得我……”
7.  **双目镜把戏**：放大缺点/过失，缩小优点/成功。例如：“成功全是运气，我其实一点都不行。”
8.  **应该式高估**：对自己有过高的、完美的期待。例如：“我应该是完美的，不应该犯错，犯错可就糟糕了。”

认识并改正这些思维方式至关重要。

我们已经探讨了减少自我关注和改变思维模式。最后，我们来看第三个方面：改变行为模式。

## 3. 改变行为模式 🚶

改变行为模式不需要特殊技能，主要手段是进行“微型实验”，并对改变带来的变化保持好奇心。

以下是进行“微型实验”的四步法：
1.  **辨别当前行为模式**：清楚你目前在该场景下的“安全行为”（通常是逃避或保护行为）。
2.  **联系想法**：确认你当前行为背后的预测、期望或猜想。
3.  **决定改变**：计划做出一个小的、具体的行为改变，目的是调动勇气和好奇心。
4.  **评估与反思**：实验后评估结果，反思并确认自己当初的想法是否正确。

**微型实验示例：阿正的故事**
*   **核心担忧**：人们了解他后就不再喜欢他。
*   **实验步骤**：
    1.  **当前行为**：避免谈论自己的事、活动或意见，通过保持安静保护自己。
    2.  **联系想法**：“人们对我缺乏兴趣，如果我说话，他们会扭头走掉。”
    3.  **决定改变**：尝试和一位同事分享周末的趣事。
    4.  **评估结果**：同事不仅听了，还参与了讨论。结论：他们确实听了我说的话，没有把我排除在外。

你也可以使用“改变行为模式想法记录表”来规划实验：

**记录表示例一**
*   **场景**：办公室有人询问事情时，我试图把脸藏起来。
*   **预测**：我的脸会变红，他们会左顾右盼并停止询问。
*   **实验**：停止隐藏，展示真实的自己。
*   **实际结果**：脸确实红了，但谈话仍在继续。
*   **结论**：脸红并不像以前那样让我恐惧，我不需要隐藏自己。

**记录表示例二**
*   **场景**：选择人少时段出门，怕打招呼。
*   **预测**：他们会走过来问问题，我会紧张说傻话。
*   **实验**：选择人多时段出门。如有人谈话，我会回答，不故意躲避。
*   **实际结果**：感觉不自在，但有人问候时我回答了“早上好”。一位认识的人擦肩而过，什么都没说。
*   **结论**：我可以适应这一改变。其他人可能对谈话也不感兴趣。

**总结**
在本节课中，我们一起学习了克服社交恐惧的核心路径：**打破旧模型，建立新模型**。我们深入探讨了三大方法：**减少自我关注**、**改变思维模式** 和 **改变行为模式**，并介绍了配套的练习原则、技巧（如分阶段练习、循序渐进、正念放松）和实用工具（如想法记录表、微型实验）。记住，改变的关键在于科学、系统地练习，并对自己保持耐心与好奇心。
