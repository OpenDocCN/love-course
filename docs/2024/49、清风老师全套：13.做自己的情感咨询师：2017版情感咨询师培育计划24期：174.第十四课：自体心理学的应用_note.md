# 清风老师全套：13.做自己的情感咨询师：2017版情感咨询师培育计划24期：第十四课：自体心理学的应用

## 概述
在本节课中，我们将学习自体心理学在咨询中的应用，了解如何帮助来访者建立内聚的字体，以及共情在治疗过程中的重要性。

## 自体心理学的概念
自体心理学是20世纪由美国精神分析学会主席科胡特提出的重要观点，旨在解释和理解自体的形成，以及在人际交往中自体对个体心理健康的影响。

### 核心概念
**公式**： 自体心理学 = 自体的形成 + 人际交往中的影响

## 自体障碍与治疗
自体障碍是由于自体在发展过程中没有形成内聚的字体，导致个体在现实生活中遇到各种问题。

### 核心概念
**公式**： 自体障碍 = 内聚字体缺失

自体心理学的治疗方式是帮助来访者完善自体，将其转化为内聚的字体。

### 核心概念
**代码**： 治疗方式 = 完善自体 + 转化为内聚字体

## 共情的作用
在自体心理学中，共情是帮助来访者建立内聚字体的主要方式。

### 核心概念
**公式**： 建立内聚字体 = 共情

共情的作用包括：
1. 咨询师借助来访者的言行，深入对方的内心，体验其情感和思维。
2. 咨询师借助于知识和经验，把握来访者的体验，与经历和人格之间的联系，更好地理解问题的实质。
3. 咨询师运用咨询技巧，把自己的共情传达给对方，以影响对方，并取得反馈。

### 核心概念
**代码**： 共情 = 深入内心 + 理解问题 + 影响对方

## 治疗过程
在自体心理学咨询中，咨询师主要通过共情和诠释来帮助来访者重新构建他们的自体。

### 核心概念
**公式**： 治疗过程 = 共情 + 诠释

## 咨询师的职责
咨询师的职责是向来访者指出内在的自体发展以及未被满足的需要，从而在生活当中寻求满足的过程当中，让来访者更能理解他自身各种的防御和潜在的需求。

### 核心概念
**代码**： 咨询师职责 = 指出自体发展 + 指出未被满足的需要 + 理解防御和需求

本节课中，我们学习了自体心理学在咨询中的应用，了解了如何帮助来访者建立内聚的字体，以及共情在治疗过程中的重要性。希望同学们能够将所学知识应用到实际工作中，成为一名优秀的情感咨询师。😊
