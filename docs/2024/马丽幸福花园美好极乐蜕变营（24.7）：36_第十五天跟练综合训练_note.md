# 马丽幸福花园美好极乐蜕变营（24.7）：36_第十五天跟练综合训练_note

在本节课中，我们将学习如何将之前掌握的盆底肌核心技巧，与动态的身体律动相结合，从瑜伽垫逐步过渡到床上进行综合训练。课程分为两个主要部分：第一部分是跟随详细指令练习，第二部分是在音乐中自由感受与发挥。

## 概述：从静态到动态，从地面到床铺

上一节我们学习了盆底肌的孤立控制技巧，本节中我们来看看如何将这些技巧融入身体的各种动态运动中。我们将通过站姿、跪姿、俯卧和仰卧四种姿势，结合前后、左右、绕圈等动作，让盆底肌的控制变得灵活而自然。

## 第一部分：跟随指令练习

以下是四种不同姿势下的练习方法，每个动作都旨在将盆底肌的**收紧（Kegel收缩）**与**放松**融入身体律动。

### 1. 站姿练习 🧍‍♀️

首先，我们以站姿开始。请将双腿稍分开，膝盖微弯，手臂自然放松，找到一个灵活、类似舞蹈的基本姿态。

**前后摆动**
想象耻骨（前侧）或尾骨（后侧）有一根线被提起。身体前倾（耻骨上提）时，盆底肌自然**收紧**；身体后倾（尾骨后引）时，盆底肌自然**放松**。循环进行：`收紧 -> 放松 -> 收紧 -> 放松`。

**左右摆胯**
这是类似摇摆舞的摆胯动作。尝试在胯部摆向一侧时**收紧**盆底肌，摆向另一侧时**放松**。你可以自由选择在哪一侧收紧，并尝试两侧都能灵活控制。

**绕圈运动**
以盆底区域为圆心，进行顺时针或逆时针的绕圈运动。尝试在胯部绕到身体前侧时**收紧**，绕到后侧时**放松**。你也可以探索在其他角度收紧，目标是让盆底肌在任何动态中都能根据需要自如地**收紧**与**放松**。

### 2. 跪姿练习 🧎‍♀️

现在，我们慢慢转移到床上（或垫子上），采用跪姿。这个姿势能提供与地面不同的感觉。

在跪姿下，重复站姿的所有动作：前后摆动、左右摆胯以及绕圈运动。请专注于感受身体，探索在哪个幅度或角度下，盆底肌的发力更轻松、感觉更强烈。目标是打破局限，让盆底肌在任何姿势和方向都能灵活控制。

### 3. 俯卧练习 🐍

接下来，我们采用俯卧姿势，类似平板支撑的起始动作，但更为放松。双肘支撑，身体俯卧。

**前后运动**
轻轻将臀部向空中抬高时，盆底肌及臀部、腿部肌肉处于**放松**状态。将耻骨向下压向床面时，从盆底到大腿根部及臀部的一片肌肉会自然**收紧**。循环练习：`放松 -> 收紧 -> 放松 -> 收紧`。

**左右滚动**
保持俯卧，让胯骨与耻骨接触床面，进行左右的摆动或滚动。尝试在滚向一侧时**收紧**盆底肌，另一侧时**放松**。这个动作像在床上自由滚动，请确保两侧都得到练习。

### 4. 仰卧练习 😴

最后，我们采用仰卧姿势。这个姿势非常放松，也是亲密关系中的常用姿势。

**上下提动**
平躺，双脚踩在床面上。将臀部向上提起时，盆底肌执行**上提**动作（一步到位提到最高），放下时**放松**。可以尝试在最高点保持几秒：`上提 -> 保持 -> 放松`。

**左右摆动**
在仰卧抬臀的状态下，加入左右的摆动。尝试在摆向一侧时**收紧**，另一侧或中间时**放松**。探索你的身体更喜欢完整的摆动收紧，还是半程的摆动收紧。

**单腿上提**
抬起一只脚，继续进行臀部上提和放下的动作。因为双脚高度不同，你会感受到发力点有所偏移。两侧都需要练习：`上提 -> 放下`。

## 第二部分：跟随音乐自由练习 🎵

在掌握了上述分解动作后，我们将进入音乐环节。请跟随音乐的节奏，自由地将站姿、跪姿、俯卧、仰卧四种姿势串联起来。在律动中，自然地融入盆底肌的**收紧**、**上提**与**放松**，无需思考具体指令，专注于身体的感受。

**重要提示**
1.  本课程为带球练习，请在练习开始前正确佩戴好球体。
2.  **练习结束后，请务必及时取出球体并进行清洁存放。**
3.  **每日带球运动时间切勿超过一小时**，以免产生副作用。

## 总结：让技巧融入生活律动

本节课中，我们一起学习了如何将盆底肌的控制技巧与动态的身体运动相结合，从站姿、跪姿、俯卧到仰卧，完成了从地面到床铺的过渡训练。关键在于让盆底肌的收缩与放松成为身体自然律动的一部分，从而灵活运用于日常生活及亲密关系中。请记住，练习的核心是探索与感受你自己的身体。
