# 43、彭祖强肾功：044.洗髓强肾功秘训

## 概述

在本节课中，我们将学习彭祖强肾功中的洗髓强肾功秘训，包括热身、呼吸、手法和拍打等练习方法。

## 热身

1. **搓手搓鼻翼**：先搓热双手，然后搓热鼻翼，以唤醒身体。
2. **土纳练习**：练习鼻腔震动和腹腔震动，以通鼻腔。

## 动作练习

1. **立法处大**：双脚站正，屈膝，收敛臀部，双手扶腰，牙齿清刻，舌力上颚，鼻吸鼻呼，吸气时提睾缩阴收腹，呼气放松。
2. **喉腔震动**：每次呼吸时尽量收紧，以振动鼻腔和头腔。

## 局部手法热身

1. **揉搓耻骨筋**：揉搓耻骨位置的筋，疏通这些点。
2. **揉搓腹股沟**：揉搓腹股沟，再揉出金属。
3. **卡拉**：两个手换手卡住根部，尽量去卡到根部，卡住后去签拉，手法由轻到重。
4. **旋转**：旋转时配合呼吸，一易导气，当旋的有点痛的时，点停一下，用意念往里灌气。

## 绑法

1. **折布**：将布折成长条状，然后捆绑。
2. **捆绑方法**：将布放在根部，手卡住根部，然后绑两个结，调整绳口和阴茎是否居中，再往里面转一下，把绳节尽可能靠近耻骨。
3. **试力**：绑好后试一下立热身，让根部适应重量。

## 拍打

1. **拍打部位**：后背、手臂、腹部、腿部、脚部、腹股沟、下腹部、内门、胸部、背部。
2. **拍打方法**：用手或拍打棒拍打，力度以自己能适应为主。

## 总结

本节课中我们一起学习了彭祖强肾功中的洗髓强肾功秘训，包括热身、呼吸、手法和拍打等练习方法。希望大家能够坚持练习，以达到强身健体的目的。
