# 21. 王一名·吸引力出圈指南：如何让优质的人主动认识你 个人品牌营销：如何让他人觉得你有魅力？

🎼欢迎来到吸引力出圈指南，我是王一鸣。今天是我们课程的第一节课，我们将围绕魅力进行深入探讨。本节课的核心内容是关于魅力的秘密以及如何获得魅力的技巧。

**核心观点来源**：
- 心理学基础理论
- 世界著名激励大师伯恩·崔西的理论，他是世界五百强企业的技巧专家，也是个人职业发展专家，许多名人如比尔·盖茨、巴菲特都接受过他的培训。

**魅力定义**：
魅力是一个人综合的吸引力，是获得他人关注的重要特质，也是构建非凡人际关系的能力。

**魅力秘密**：
让他人感受到我们的价值。魅力并非天生，可以通过后天学习和培训获得技巧。

**如何让他人觉得你很有魅力**？
1. **了解魅力**：魅力不是表面上的，而是一种能与他人建立超出一般关系的能力。
2. **提升自我价值感**：通过以下三种关键行为：
   - **接受**：无条件尊重他人，不批评、不挑刺、不讽刺、不挖苦。简单做法是微笑。
   - **欣赏**：赞美他人的优点、成就、穿着、发型、奖项等，真诚地发现并表达他们的闪光点。
   - **关注**：密切关注他人，给予温和的目光注视，提升对方的自我价值感。

**总结**：
本节课我们学习了如何让他人觉得你很有魅力。在接下来的课程中，我们将继续探讨如何提升和展现你的魅力，成为一个有吸引力的人。这种吸引力将帮助你自然地在生活和工作中达成目标。
