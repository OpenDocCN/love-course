# 老白《约会达人》：情场3.0：【聊天解码】23.长期关系之设定规则 🎼

## 概述
在本节课中，我们将学习如何设定长期关系中的规则，这是维持感情关系的关键。

## 设定规则的重要性
设定规则是任何一段感情关系的开始。它定义了你的态度和标准，例如：
- 不喜欢女生抽烟
- 要求女生出去玩不能超过12点回家
- 不希望她八卦和评价别人

## 设定规则的步骤

### 1. 制造意外
- 通过意外事件引发另一件事情的发生，例如女朋友不接电话。
- **公式**：意外事件 → 情感反应 → 规则意识

### 2. 冷处理升级关系
- 对意外事件进行冷处理，升级到让对方意识到你的重视。
- **代码**：感性处理（如消失或指责）→ 理性沟通（明确表达不满）

### 3. 颠覆价值观
- 解释为什么你特别在意这件事，给出合理的价值观。
- **公式**：价值观覆盖 → 对方认可

## 实例分析
以女朋友不接电话为例：
- **意外**：女朋友不接电话。
- **冷处理**：感性处理（如消失或指责）→ 理性沟通（明确表达不满）。
- **价值观**：强调相互尊重的重要性。

## 总结
本节课中我们一起学习了如何设定长期关系中的规则，包括制造意外、冷处理升级关系和颠覆价值观。通过这些步骤，你可以更好地维护感情关系。
