# 老白《约会达人》：情场3.0：【聊天解码】23.长期关系之设定规则_note

在本节课中，我们将要学习长期关系相处模型中的关键环节：设定规则。我们将探讨规则的本质、重要性以及如何通过一套具体步骤来有效地建立规则，从而维护一段健康、稳定的感情关系。

---

## 什么是规则？

规则是你自己的一个框架。
规则是你自己的一个态度。
规则是你自己的一个标准。

例如：
*   你不喜欢女生抽烟，这就是一条标准。
*   你要求女生出去玩，不能超过12点以后回家，这也是一个你的标准。
*   你不希望她去八卦和评价别人，这也是一条标准。

只有你拥有足够多源自内心的标准，这样的规则才会让你们的感情越来越好。很多男性不去设定规则，以为讨好女生，女生就会爱自己。但事实上，你越是设定了规则，你越是显得像一个高价值的男人。你越是设定了规则，女生就会越喜欢你。特别是在一段异地恋或长期关系维护当中，如何设定规则才是真正的关键。

---

## 如何设定规则？

我们把设定规则的过程分为三个步骤。

以下是设定规则的三个核心步骤：

1.  **制造意外**
    制造意外，也可以叫做没事找事。一定需要有一件事情来引发另一件事情的发生。例如，你和女朋友吵架，一定是因为一件意外发生的。这个意外是什么？比如有一天你给她打电话，她没有接电话，你打了两个，她都没有接。

2.  **冷处理与升级关系**
    如果不设定规则的做法就是算了，或者等这个女生过来给你道歉、撒个娇，这件事就过去了。这是很多男人的做法。但是设定规则的做法就是要把这件事冷处理，并且必须去升级关系。首先，不接电话是一个意外。你绝不能就此罢休，如果你就此罢休，那你就是一个没有规则、没有标准的男人。你一定要把这件事情去升级，升级到让她意识到你很重视这个问题，让她意识到你非常在意、非常生气、非常不爽，让她知道你的底线在哪里。

    那么，你可以采取什么样的方式去升级呢？你可以明确的告诉她，即 **先感性，再理性**。先感性，不是不理她。例如，先玩几个小时或者一整天的消失，也可以指责她几句，然后就不再回她消息，让她感性地知道你有点不爽。然后再用理性的方式直接点去告诉她：**“这件事我接受不了。”** 当然，这是你冷处理过后，逐渐重新和她说话之后再去做的事情。

    这里很多男性可能会害怕：如果这样的话，女生要是生气了，跟别的男人约会了怎么办？如果是这样的话，那说明你可能是选错了对象。我们其实是在通过不断的制造意外、升级意外，让她知道一条一条的规则到底在哪里。

3.  **颠覆价值观**
    当她意识到你的规则在这里、你的底线在这里，并且已经感受到了你的不爽的时候，接下来第三点就非常重要了，就是颠覆价值观。这个时候你一定要告诉她，你为什么会特别在意这件事，为什么你要冷处理，你一定要给出一个合理的价值观。

    例如，拿刚才她不接电话这个例子来讲，这个时候我会给她一个我的爱情价值观。请记住一句话：**价值观只能被另一种价值观覆盖。只有你的价值观比对方的价值观更强，你才能够覆盖掉它，她也会认可你的价值。**

    比如我说：**“我之所以这么生气，并不是你不接我电话这件事的本身，而是因为我觉得这背后反映出来的是你对待感情的不尊重。我觉得如果我们双方对待感情，哪怕有一丝一毫的不尊重，那这样的感情就没有了继续下去的意义。我觉得感情的第一位就是相互尊重。虽然我们彼此相互喜欢、欣赏对方，但我觉得尊重是唯一的一个前提。”** 这就是我们所说的价值观。你要告诉她，这就是你所认可的价值观。

    接下来你就可以定规则。你可以说：**“如果以后还有这样的话，我觉得我们可能要相互考虑一下到底合不合适了。”** 你一定要把你的底线给说出来，给她一点压迫感。但是你要明白一个道理，没有人会认为“互相尊重”这件事不对。所以你说的一定是一个正确的价值观。你说了一个正确的价值观，对方其实没有必要去反驳你，她一定会是认可的。而且你把正确的价值观跟她不仅仅是在电话这件事上联系在一起，所以这件事就变得合理，这个规则就设置得很合理。这就是设定规则的一个完整套路。

    相反，像有些男性要求女友在她的朋友圈、微信好友里不能有任何男生，不允许交任何异性朋友。那么这就是一个不正确的价值观。

## 课程总结

本节课中，我们一起学习了长期关系中设定规则的核心方法。

总结如下：
*   第一，设定规则是任何一段感情关系的开始。
*   第二，设定规则可以通过 **制造意外 -> 冷处理与升级关系 -> 颠覆价值观** 这三个步骤来实现。

通过建立清晰、合理且基于正确价值观的规则，你不仅能维护自己的框架和标准，也能引导关系向更健康、更相互尊重的方向发展。
