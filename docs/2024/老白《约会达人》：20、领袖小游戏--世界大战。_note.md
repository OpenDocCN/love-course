# 老白《约会达人》：20、领袖小游戏--世界大战。_note

在本节课中，我们将学习一个名为“世界大战”的互动小游戏。这个游戏的核心在于分组对抗，通过简单的游戏过程，可以有效地拉近团队成员之间的距离，并创造出轻松有趣的互动氛围。

## 游戏准备与分组 🧑‍🤝‍🧑

首先，我们需要将所有参与者分成两队。游戏的基本规则是两队进行对抗。

以下是分组的具体步骤：

1.  **确定人数与分队**：游戏至少需要4人。如果是4个人，就分成2人一队。如果是6个人，就分成3人一队，以此类推。
2.  **通过猜拳选人**：所有参与者通过“石头剪刀布”来决定谁有优先选择队友的权利。赢家可以从剩余的人中选择一名队友。
3.  **完成分队**：重复猜拳和选人的过程，直到所有人被分入两个队伍。例如，在4人局中，如果A连续赢了两次，他可以选择B和C作为队友，那么队伍就分成了A、B、C三人队对阵D一人队。

## 游戏规则与流程 🎲

分队完成后，就可以开始游戏了。两队可以进行任何约定的游戏来决出胜负，例如猜拳、摇骰子（吹牛）、十五二十等。

以下是游戏的基本流程：

1.  **约定惩罚**：在游戏开始前，两队需要提前约定好输家的惩罚，例如喝多少酒。**这一点必须提前说清楚**。
2.  **进行对抗**：两队轮流派出队员进行单挑（例如玩一局猜拳或摇骰子）。输掉单挑的队员下场，由同队下一名队员接替上场，继续与对方当前的胜者较量。
3.  **判定胜负**：当一方的所有队员都被击败后，则该队输掉本轮游戏，需要执行事先约定的惩罚。
4.  **特殊规则**：可以事先约定特殊规则，例如，如果一方仅剩一名队员，但他连续击败对方所有队员（“打通关”），则惩罚可以翻倍。**同样，翻倍规则也必须在游戏开始前明确约定**。

## 游戏技巧与互动示范 💡

上一节我们介绍了游戏的基本规则，本节中我们来看看如何在游戏中通过互动创造默契和拉近距离。我们以摇骰子（吹牛）游戏为例进行演示。

假设我和我的队友一队，对方两人一队。游戏开始，我的队友先与对方第一位玩家较量。

```
我方队友叫骰：“3个6”
对方叫骰：“4个6”
我方队友叫骰：“5个6”
对方叫骰：“6个2”
```

这时，轮到我方行动。我可以自然地靠近我的队友，假装帮他看骰子，同时低声交流。

> “他叫6个2，我们要开他吗？”
> “开！”

这种近距离的商讨和一致对外的决策，能迅速增强团队感。如果对方叫骰错误，则他输掉这局单挑，换对方下一位队员上场与我方当前的胜者（我的队友）继续比赛。

在整个游戏过程中，作为队友，我们可以：
*   **互相支持**：在队友犹豫时给出建议，例如“大胆叫，我帮你兜着”。
*   **共同承担**：如果队伍输了，可以主动提出“我酒量好，我多喝一点”，体现照顾。
*   **调侃对手**：在胜利后以轻松友好的方式调侃对方，活跃气氛。

因为你们是“一班”的，共同面对“外敌”，这种合作与共担的经历能自然而然地提升彼此的亲密感和联系。

最后，我们重点探讨了在游戏过程中，如何通过沟通、支持和共同承担来培养团队默契，拉近与队友的距离。
记住，游戏的目的不仅是胜负，更是创造轻松、有趣的互动体验。
