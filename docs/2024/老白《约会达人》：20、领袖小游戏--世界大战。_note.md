# 老白《约会达人》：20、领袖小游戏--世界大战

## 概述
在本节课中，我们将学习如何通过“世界大战”这个领袖小游戏来增进与约会对象的互动和亲密感。

## 游戏规则
以下是“世界大战”游戏的详细规则：

### 分队
1. **南北分界**：参与者分为南北两派。
2. **人数分配**：四人分为两队，每队两人。
3. **选人方式**：通过石头剪刀布决定谁加入哪一队。

### 游戏进行
1. **选人**：赢的人选择加入哪一队。
2. **对战**：两队进行对战，可以玩任何游戏，如石头剪刀布、骰子、收话石等。
3. **输赢判定**：根据游戏规则判定输赢，输的一方需要完成特定任务，如喝酒。

### 特殊规则
1. **翻倍规则**：如果有人打通全场，需要翻倍完成任务。
2. **提前说明**：翻倍规则需要在游戏开始前说明，否则无效。

## 游戏示例
以下是游戏的一个示例：

1. **分队**：南北两派。
2. **选人**：玩家A赢，选择加入南派。
3. **对战**：南派玩家A和玩家B对战北派玩家C和玩家D。
4. **游戏**：玩石头剪刀布，南派玩家A和B赢。
5. **输赢判定**：北派玩家C和D需要完成特定任务，如喝酒。

## 总结
本节课中，我们一起学习了如何通过“世界大战”这个领袖小游戏来增进与约会对象的互动和亲密感。通过游戏，我们可以更好地了解对方，建立信任和默契。希望这个游戏能帮助你在约会中更加自信和愉快！
