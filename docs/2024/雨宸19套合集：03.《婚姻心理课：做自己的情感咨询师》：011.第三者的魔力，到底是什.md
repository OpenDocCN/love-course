# 婚姻心理课03：第十一讲 · 第三者的魔力到底是什么？🔍

在本节课中，我们将深入探讨婚姻中“第三者”现象背后的心理机制。我们将分析出轨行为形成的四个阶段，并揭示其核心根源——消极对比与需求填补。通过学习，你将理解出轨并非简单的道德问题，而是关系内部需求未被满足所导致的结果。掌握这些知识，有助于你更理性地看待情感危机，并找到建设性的解决方向。

## 出轨的四个阶段

上一节我们探讨了情感中的常见矛盾，本节中我们来看看，当矛盾升级且处理不当时，关系是如何一步步滑向出轨深渊的。出轨行为通常并非一蹴而就，而是经历了一个渐进的过程。

以下是出轨行为形成的四个典型阶段：

1.  **矛盾与失望**
    关系中出现矛盾，双方在解决矛盾时沟通不畅，认知出现分歧。一方或双方因此产生不满和失望，进而萌生“彼此不合适”的想法。每一个想要出轨的人，最初都只是对当下的情感生活感到不满意。

2.  **沟通减少与秘密滋生**
    在沟通尝试无果后，双方之间的交流变得越来越少，各自开始拥有不愿与伴侣分享的小秘密。当人在一件事上得不到正反馈和认同感时，就会自然降低对这件事的关注度。情感需求未被满足的一方，会开始将注意力转移到别处。

3.  **向第三方倾诉**
    当情感需求在伴侣处无法得到释放时，个体倾向于寻找第三方作为倾听者。如果第三方能够很好地承担倾听与理解的角色，个体就会在此过程中获得强烈的认同感，从而与第三方的联系日益紧密。

4.  **实质性出轨**
    精神上的依赖与亲近最终可能发展为肉体上的实质性出轨，至此，完整的出轨过程形成。

## 出轨的根源：消极对比

从上述阶段可以看出，出轨的核心驱动力是一种“消极对比”的心理机制。

你会不自觉地将原配与新欢进行对比：一个能满足我的需求，另一个不能；一个显得蛮横无理，另一个则是良好的倾听者，能解决我的烦恼、陪我聊天、听我说话。两者对比，高下立判，人自然会倾向于选择让自己更舒服的那一个。

因此，在面对婚姻问题时，单纯站在道德制高点指责对方，虽然能获得心理上的满足感，但无助于解决任何实际问题。人心易变是常态，我们需要学习的是如何用理智去管理和应对这种变化，这才是学习情感知识的真正意义。

## 出轨的心理：填补需求

出轨行为在心理层面上，本质是为了**填补在婚姻中未被满足的需求**。

研究表明，大多数婚外情并非源于性，而是源于**沟通不畅**。当个体的情感需求、被理解的需求、被认同的需求无法从伴侣那里得到回馈时，出于本能，人会去寻找能满足这些需求的对象，从而形成精神出轨，久而久之便可能发展为肉体出轨。

此外，还有一种处理矛盾的方式叫做**关系三角化**，即一方在维持婚姻的同时，与第三方建立情感联系。这在现代婚姻中并不罕见，有时甚至是双向的。双方因无法承受分手的代价，又难以忍受婚姻中的痛苦，于是通过第三方来满足需求，从而维持一种脆弱的平衡。

## 出轨的真正原因：愉悦度与满足感崩塌

那么，导致我们向外寻求满足的根本原因是什么？关键在于关系内部两种感受的崩塌：**愉悦度**和**满足感**。

**1. 愉悦度不足**
愉悦度指的是两人相处时的舒适、快乐的感觉。它可以用一个简单的公式来理解：

`感情稳定程度 = 吸引力(X) × 相处状态(Y)`

其中，吸引力（X）相对稳定，而相处状态（Y）则直接由愉悦度决定。当愉悦度因需求（如认同、理解、尊重）得不到满足而持续下降，最终降为零时，相处状态（Y）也会清零，导致感情基础崩溃。

**2. 满足感不足**
满足感关乎对婚姻未来的**预期和期待**。它和愉悦度（当下的感受）相辅相成。当愉悦度长期低下时，满足感也会随之受损。例如，当一方在伴侣处找不到愉悦，转而与第三方深度互动后，再回头进行“消极对比”，便会觉得伴侣完全无法满足自己对爱情的期待。此外，如果长期感受不到被爱和被需要，也会导致深深的失望，致使满足感彻底崩塌。

整个出轨的路径可以概括为：**愉悦度受损 → 满足感下降 → 两者均降至冰点 → 关系结束**。

## 如何应对诱惑与危机？

面对社会中的诸多诱惑与挑战，我们该如何自处与应对？

首先，必须摒弃“男人都花心”或“女人更重情”的刻板印象。根据咨询经验，男女在出轨和暧昧的比例上大致相当。因此，不要轻易扮演“道德标兵”去指责对方，这除了能让自己获得一种“无责者”的虚假解脱外，对修复关系毫无益处。

婚姻生活的延续，永远依靠的是**理解与认同**。关系出现问题，双方都负有责任。永远不要带着“对方是渣男/渣女，我是受害者”的情绪生活。人需要学会反思，思考自己在亲密关系中的不足。

**提升自我，不是为了讨好那个曾经背叛你的人**（感情中，一次不忠，终身不用是很多人的原则），而是为了把自己变得更好、更强。我们变好的目的，是为了有能力经营好当下的关系，或者迎接未来更健康的感情。

本节课中，我们一起学习了出轨形成的四个阶段，揭示了其根源在于“消极对比”和“需求填补”，并深入分析了导致出轨的内在原因——愉悦度与满足感的崩塌。记住，指责解决不了问题，理解和自我成长才是应对情感危机、守护亲密关系的根本。
