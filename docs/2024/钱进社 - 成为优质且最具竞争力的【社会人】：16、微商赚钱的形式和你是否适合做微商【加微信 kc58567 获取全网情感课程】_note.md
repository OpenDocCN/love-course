# 钱进社 - 成为优质且最具竞争力的【社会人】：16、微商赚钱的形式和你是否适合做微商

## 概述
在本节课中，我们将探讨微商的赚钱形式以及你是否适合从事微商行业。

## 微商赚钱形式
以下是微商赚钱的形式：

1. **多层次分销**：微商本质上是多层次分销，通过招募代理来扩大销售网络。
2. **渠道和营销费**：每招募一个代理就是一个渠道，通过渠道和营销费实现双赢。
3. **产品过剩**：在产品过剩的时代，渠道和营销才是最贵的产品，其本身含金量不高。

## 是否适合做微商
以下是你是否适合做微商的考量因素：

1. **互联网思维**：你是否具备互联网思维和运作能力，这是未来20年的核心竞争力。
2. **拉新方案**：你是否有一个有效的拉新方案，这是赚钱的关键。
3. **个人能力**：你是否能够适应微商行业的高强度工作，并具备一定的社交能力。
