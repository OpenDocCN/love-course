# 钱进社 - 成为优质且最具竞争：98、如何通过“风险管理”逻辑

在本节课中，我们将学习如何运用风险管理的核心逻辑，贯穿于日常工作，以有效降低各种不确定性。我们将从认知调整、信息处理到具体行动，系统地掌握降低风险、提升决策质量的方法。

## 认知调整：摘除“我以为”的惯性思维

上一节我们介绍了课程的整体目标，本节中我们来看看风险管理的起点——认知调整。

当我们面对选择和问题时，一个惯性的思维模式会自动开启。这个模式就是“我认为”或“我以为”。这种基于旧有认知的判断，往往是我们自己给自己设下的陷阱。许多困难并非源于他人，而是源于我们被自己的认知和思维所局限。

因此，进行风险管理的第一顺位，就是必须摘除“我认为”和“我以为”这样的想法。如果在面对问题和选择时，你下意识地开启这种模式，就很容易在起点上犯错。起点的错误会导致后续每一步都偏离正确轨道。

这种“我以为我知道”的认知停留在旧的信息层。当你坚信旧信息正确时，会自动忽略和屏蔽新的信息。这种惯性思维具有极大风险。新旧信息的交融总存在时间差和空间上的分散，我们无法完美契合。我们能做的，是尽量先摘除“我以为我知道”的认知，让新旧信息汇合时，没有迂腐的旧认知做铺垫，从而更容易让信息配套。

**核心概念**：认知惯性风险 = 固守旧信息层 / 忽略新信息输入

## 掌握状况：持续更新你的信息数据库

上一节我们探讨了调整认知惯性，本节中我们来看看如何确保自己掌握最新状况。

做出选择或面对困难时，即便你已掌握许多信息（A、B、C），也必须先去确认是否存在新的信息来源（D）。你需要更新大脑中的数据库，确保其接近你目前能接收到的最前沿信息。

掌握状况是不可或缺的环节。它意味着你要确定自己掌握的信息不只是旧的，还包括新的。这不是要求你100%获取所有新信息，而是在你当下能力范围内，尽可能获取最大的信息流。信息本身是流动的，不存在绝对固化。

例如，你的上司可能夸大同伴的成绩来制造竞争。但你没有亲眼验证或观察过，就不要过早下结论。你更需要提取的是领导这么做的**用意**——是为了激励团队，还是觉得你做得不好？对于同事是否真如领导所说，这个结论也不要过早下。

## 信息验证：对“我听说”保持警惕

当我们获取信息时，必须保持警惕，多打几个问号。

我们需要向自己提出质疑：这些信息都千真万确吗？我们必须留意信息来源。如果来源不可靠，信息就缺乏可信度。“我听说”、“我觉得”、“好像是”这类语言架构在工作中非常常见。如果你对每一个这样抽象的表达进行反问，它们可能都经不起推敲。

以下是处理此类信息的关键步骤：

1.  **质疑来源**：信息是谁说的？他与你是什么关系？他的话基于什么样的事实？
2.  **寻求佐证**：需要自己去验证，获取证据来证明事情确实如此。
3.  **评估关联性**：即使信息为真，也要评估你本人在其中的角色和作用。例如，某人说他一年赚1000万，但你的角色若不起决定性作用，这1000万也与你无关。

信息的解读也会因人而异。这不是信息本身的问题，而是解读者的理解可能歪曲了原意。如果一个人接收和解读信息的基础就是错的，那么后续工作几乎不可能在正常轨道上推进。

## 信息定性：用定量信息锁定模糊指令

在工作中，我们常收到模糊的定性指令，如“稍微修改一下”。每个人的理解都不同，这极易导致白折腾和错误。

我们需要做的是：**用定量的信息去辅助，把定性的信息确定下来**。

“稍微修改一下”是无法定性的。我们需要将其转化为定量信息。例如，你可以标记出具体部分，询问对方：“‘稍微修改’指的是不是这几点？如果我修改这些部分，方向是XXX，大约删减/添加XXX字，是否可以？”

你确定的定量信息越多，获取的信息就越精准，越不容易出错。

**核心概念**：
```
定性指令： “提升业绩”、“修改一下”
转化为定量指令： “将A组业绩提升30%”、“修改报告第3-5行，将结论部分字数缩减20%”
```
定量信息有明确的数字和方向，能更好地为工作定性。这样你才能更精准地开展工作、汇报工作，避免词不达意和犯错误，也不会觉得自己冤枉。

在职场中，我们很难短期内与同事或老板形成深度默契。唯一能为自己做的功课，就是无限地去确定**定量情形**，从而减少内耗和误解。

## 风险评估：管理风险，而非逃避风险

相信了同事、领导或家人的信息后，如何降低风险？答案依然是依托定量信息的模式来为信息定性，这样可以减少许多失误。

很多人在面对选择时无法做出好判断，一个共同的毛病是：**会过大地评估风险**。在还不应有结论时，就默认自己会失败、会损失、怎么做都是错的。

判断总是伴随风险，这是一条铁律。但判断能力强的人懂得如何巧妙地**面对和管理风险**，而不是一棒子打死所有风险，或幻想风险会自行消失。

首先，要区分**规避风险**和**逃避风险**。
*   **规避风险**是基于事实分析。
*   **逃避风险**只是一种不愿面对的心态，并为自己找许多冠冕堂皇的理由，这没有价值。

我们应侧重评估风险是否在**可允许的范围内**，然后勇敢面对。如果一遇风险就逃，最终任何决定都做不出，只会停在原地抱怨。

**核心认知**：管理风险不是把风险彻底消灭，而是将其控制在相对可控的范围内，这本身就是一种胜利。

面对风险事务时，请尝试以下步骤：

1.  把可能的风险逐一罗列清楚。
2.  看哪些单项是可以尽可能去控制风险的。
3.  接受不存在完全没有风险的环境，目标是控制，而非消除。

例如，即便你有一份稳定工作，也可能遇到难相处的同事。通过“公关”改善关系，就等于把你在这个环境中的风险控制到最低。我们应该无限地去降低环境中可见的风险，而不是幻想扭转整个环境。风险一旦叠加，就会从加法变成乘法，最终导致全盘失控。

## 警惕“小失误”：重大事故皆非偶然

最后，我们来看一个容易被忽视的风险点：对小失误的漠视。

保险业有一个数据：一次重大事故的背后，可能有几十起轻度意外；而这些轻度意外的背后，又可能有几百起被忽略的、让人心里“咯噔”一下的小事件。

因此，重大事故或许根本不存在“偶然”，它们都是理应发生的。如果你在工作中发生小事故，总给自己找理由——“我只是不小心”、“我只是冲动”，那么当重大事故发生时，其根源就是你长期忽略小事后果的累积。

这条认知在工作中非常常见：“我只是不小心迟到了”、“我只是不小心发错了邮件”。如果这种“不小心”不是一次，而是惯性思维驱动的行为模式，那么总有一天，一个大的事故会在你身上爆发。

我们需要调整认知：
*   **不要认为**失误是偶然事件。
*   **要认为**：虽然这次幸运只是小失误，但未来可能会有更严重的事故。因此我要警醒、复盘、分析缘由。

把这条认知放进脑子里，能无形中降低很多风险。当你无视小问题，并不断用理由说服自己时，风险就在做加法。加法积累到一定程度，超出你的控制能力，就会变成惩罚性的乘法，最终导致全盘失控。

**核心总结**：风险控制 = 对微小风险做持续的 **减法** ； 风险失控 = 对累积风险做无奈的 **加法** ，直至变为 **乘法** 。

---

本节课中我们一起学习了贯穿工作的风险管理逻辑。我们从**摘除“我以为”的认知惯性**开始，强调了**持续更新信息、掌握最新状况**的重要性。我们学会了要对模糊信息保持警惕，并通过**将定性指令转化为定量信息**来精准行动。在面对风险时，我们明白了**管理的核心是控制而非逃避**，并且要**高度重视那些容易被忽略的“小失误”**，防止其累积成重大危机。掌握这些方法，能帮助你在工作中更稳健地决策，有效降低不确定性。
