# 男哥内核3.0初级班：内部课：3.朋友圈的新高度与改造

## 概述
在本节课中，我们将学习如何打造个人IP和展示面，提升朋友圈的吸引力。

## 个人IP打造与展示面
### 展示面的定义
展示面是指展示个人特长、爱好、生活等方面的内容，它是人与人之间未见面的第一印象。

### 展示面的重要性
展示面是个人名片，包含个人信息、生活条件等，有助于吸引他人关注。

## 朋友圈打造
### 不懂展示自己
- 头像、背景非本人照片
- 三天可见
- 内容与个人无关

### 太懂展示自己
- 过度展示个人生活细节
- 展示缺点而非优点

### 朋友圈内容
- 展示个人特长、爱好、生活
- 展示见过世面的样子
- 展示穿搭、颜值、生活条件

### 朋友圈排版
- 照片编辑
- 文案配文

## 社交软件布局
### 探探
- 照片保持一致性
- 展示爱好、生活、环境

### 积木
- 展示价值
- 结合颜值和价值

## 案例分析
- 展示面好的朋友圈，女生主动打招呼

## 总结
本节课学习了如何打造个人IP和展示面，提升朋友圈的吸引力。通过展示个人特长、爱好、生活等方面，以及合理的排版和内容，可以吸引他人关注，提升个人魅力。
