# 课程05：眉粉是什么？长什么样？ 🎨

在本节课中，我们将要学习一种与眉笔同样重要的画眉工具——眉粉。我们将了解它的外观、形态以及基本的使用方式。

## 什么是眉粉？

上一节我们介绍了眉笔，本节中我们来看看另一种画眉工具：眉粉。眉粉是一种粉末状的化妆品，通常被压缩成块状，并放置在小型容器中。它的主要作用是为眉毛上色、填充空隙，并塑造眉形。

## 眉粉长什么样？

眉粉的外观通常是一个小巧的圆形或方形盒子。打开盒子后，可以看到一块或多块压实的粉末，颜色通常为棕色系或灰色系，以匹配不同的自然眉色。盒内通常会附带一把专用的、刷毛短小且紧密的小刷子。

以下是眉粉配套工具的描述：
*   **容器**：一个精致的小罐子或小盒子。
*   **内容物**：一块或多块压实的眉粉。
*   **工具**：一把短毛的小刷子。

## 如何使用眉粉？

使用眉粉时，操作者会用附带的小刷子轻轻蘸取粉末。然后，将刷子直接移动到眉毛部位，通过来回涂抹的动作，将颜色上到眉毛上。这个过程可以快速为眉毛增添颜色和密度。

**核心操作**可以简化为：`蘸取粉末 -> 涂抹于眉毛`。

## 总结

本节课中我们一起学习了眉粉。我们了解到眉粉是一种装在小型容器内的块状粉末，配合小刷子使用，通过蘸取和涂抹的动作，能够有效地为眉毛上色和塑形。它是塑造自然眉妆的常用工具之一。
