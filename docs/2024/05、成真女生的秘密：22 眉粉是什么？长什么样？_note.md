# 课程05：眉粉详解 🎨

在本节课中，我们将要学习眉粉这一化妆工具。眉粉是除眉笔外，另一种用于描绘和塑造眉毛形态的化妆品。我们将了解它的外观、形态及基本使用方法。

## 眉粉的定义与外观

上一节我们介绍了眉笔，本节中我们来看看眉粉。眉粉是一种粉状化妆品，通常被放置在一个小型容器中。

它在外观上是一个小罐子。打开后，内部装有压制成型的眉粉，并配有一把小刷子。

## 眉粉的使用方法

以下是使用眉粉画眉的基本操作描述。

*   使用者会打开这个小罐子。
*   使用者会取出配套的小刷子。
*   使用者用刷子蘸取眉粉。
*   使用者将刷子上的眉粉涂抹到眉毛区域的皮肤上。

这个过程可以起到画眉毛、填充眉毛和塑造眉形的作用。

## 总结

本节课中我们一起学习了眉粉。我们了解到眉粉是一种装在罐子里的粉状画眉产品，通过配套刷子蘸取并涂抹上妆。它和眉笔一样，都是用于修饰眉毛的重要工具。
