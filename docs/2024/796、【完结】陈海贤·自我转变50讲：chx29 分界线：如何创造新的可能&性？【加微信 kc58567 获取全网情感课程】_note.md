# 796、【完结】陈海贤·自我转变50讲：chx29 分界线：如何创造新的可能&性？

欢迎来到自我转变50讲。我是陈海贤。在前几节课中，我们探讨了如何告别旧自我和如何面对不确定。转变需要经历三个阶段：结束迷茫、重生。那么，迷茫和重生的分界线在哪里？如何跨过这条分界线，走向新的世界？

## 分界线：关注点决定

这条分界线不是由事情决定的，而是由你的关注点决定的。当你逐渐接受失去的旧自我，转向新的世界去寻找资源、发挥能力，创造新的可能性时，你就进入了重生期。

## 跨越分界线的挑战

跨过这条线的转身是艰难的，最难的是意识到自己不能回头。只有意识到不能回头，你才会面向未来去创造新的可能性。

## 学员案例：从迷茫到重生

一个学员在一家公司工作了10年，成为总经理后跳槽到互联网大厂，却很不适应。他很难接受努力奋斗的未来，甚至得了焦虑症。后来，他告诉自己“我没有选择了”，最终适应了新的工作。

## 如何完成转身

1. **做出选择**：迈入新世界的第一步是做出选择。选择之所以难，是因为我们害怕错过可能性。但很多可能性是在选择后通过深入经营创造出来的。
2. **忠于承诺**：选择要与承诺相匹配，承诺是重的，有时隐藏在选择背后。我们内心真正的动机可能是不想做选择。
3. **寻找新资源**：接受旧世界的失去，才能寻找新世界的资源。这种创造新的可能性的冲动，来自我们最深层的生命本能。
