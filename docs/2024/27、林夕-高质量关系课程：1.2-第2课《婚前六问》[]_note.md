# 27、林夕-高质量关系课程：1.2-第2课《婚前六问》👰‍♀️

## 概述
在本节课中，我们将探讨婚前六问，帮助大家更好地理解婚姻，并为即将到来的婚姻生活做好准备。

## 婚前六问
以下是婚前六问，每个问题都至关重要：

### 1. 责任与义务
- **责任**：生存能力，共同承担风险。
- **义务**：忠诚、关爱、性、亲密关系。

### 2. 三重关系和谐
- 情感、经济、身体是否和谐？

### 3. 冲突管理风格
- 双方对冲突管理的风格是否一致？

### 4. 未来目标和希望
- 双方未来的目标和希望是否一致？

### 5. 家庭生存能力
- 新的家庭是否有生存能力？

### 6. 繁衍后代
- 你是否有金钱、经历和时间繁衍下一代？

## 总结
本节课中我们一起学习了婚前六问，这些问题对于即将步入婚姻殿堂的人来说至关重要。希望大家能够认真思考这些问题，并为自己的婚姻生活做好准备。
