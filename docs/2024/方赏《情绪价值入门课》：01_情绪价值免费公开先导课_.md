# 方赏《情绪价值入门课》：01_情绪价值免费公开先导课_

在本节课中，我们将要学习“情绪价值”这一概念的基础介绍，并了解本课程的设计框架与核心理念。我们将澄清常见的误解，明确课程定位，并预览后续的学习内容。

## 课程概述与常见误解

很多人对“情绪价值”有自己的理解和想象。以下是几种常见的误解：

*   有人认为情绪价值就是动动嘴夸人。
*   有人认为情绪价值很虚，不如直接转账实在。
*   有人认为提供情绪价值会显得自己低人一等。
*   有人认为只要提供了情绪价值，就能要什么有什么。

这些理解都存在偏差。本课程旨在打破这些固有成见，带领大家认识真正的情绪价值。

## 本课程的独特定位 🎯

上一节我们列举了常见的误解，本节中我们来看看这门课程与市面上其他课程有何不同。本课程并非单纯的话术教学，其核心定位如下：

1.  **关于“爱”的启蒙课程**：从情绪价值这个入口，学习如何真正地爱别人。
2.  **社交能力提升课程**：通过学习情绪价值，来掌握如何与人打交道、进行良性互动。
3.  **情感中心打开课程**：帮助情感封闭的人学会表达感受、交换情感，建立深层链接。
4.  **能量管理体系的延伸**：本课程与“能量管理课”一脉相承，共享底层思维框架。虽然不学能量课也能听懂本课，但**能量是表达爱的基础**，将两者结合学习效果更佳。

## 课程内容设计框架 📚

讲完课程的独特定位，接下来我们具体看看课程是如何设计的。理解一个事物，必须从它的原理开始。

**第一节课：底层原理与标准**
我们将探讨情绪价值发生的底层逻辑，并明确一个清晰的判断标准：
`是否提供了情绪价值 = 是否符合核心标准`
这节课将帮你从根本上理解什么才是有效的情绪价值。

**第二节课：隐形情绪价值**
本节将颠覆“情绪价值就是夸人”的认知。我们将学习不通过言语夸赞也能提供的、往往更有价值的情绪价值。

**第三节课：夸人的方法论**
在理解了隐形价值后，我们正式进入夸人的环节。本节提供五种可实操的夸人技巧与案例，帮助你掌握夸人的核心方法。

**第四、五节课：深度情绪价值**
这两节课内容更深入、更细腻，专注于心灵层面的情感链接。我们将学习一套新的理论体系，教你如何与他人建立深刻的情感共鸣。

**第六节课：场景应用——两性关系**
为了帮助大家更好地应用，我们从本节开始进入场景教学。本节聚焦于两性关系，涵盖脱单与长期关系两种场景，解答常见困境。

**第七节课：场景应用——社交与贵人**
最后一节课，我们将情绪价值应用于更广泛的社交场景，包括多人社交和给贵人/权威人士提供情绪价值，解决“想认识厉害的人却不知如何开口”的难题。
