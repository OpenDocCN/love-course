# 安老师社恐训练营：19-练习：突破社恐的恶性循环(二)

同学们，大家好。上节课我们介绍了从减少自我关注这个突破口去解决社恐的几个练习方法。这节课，我们将从另一个突破口来解决社恐，即改变思维模式。

**改变思维模式**

改变思维模式并非易事，但一旦成功，社恐问题即可得到解决。那么，如何通过锻炼来改变思维模式呢？关键在于记录。

**记录方法**

可以使用纸笔、电脑或手机记录，关键是要记录。记录内容包括：

1. **错误的思维模式**：将焦虑的想法记录下来。
2. **正确的思维模式**：对焦虑的想法提出质疑，得出结论，并将其纠正为积极正向的想法。

**记录形式**

可以使用卡片形式记录，将笔记本的一页纸视为一张卡片。这样做的好处是分类条理清晰，解决问题时头脑专注。

**案例分析**

以下为几个案例分析：

1. **与不熟悉的同事聊天**：焦虑的想法是“我说话结巴了，表现不正常，他肯定对我印象不好”。质疑：别人真的会因为我说话结巴就不喜欢我吗？结论：人们在沟通时经常出现说话结巴的情况，这是很正常的。正确的想法：如果我说话结巴，纠正过来就好了，没有人会抓着这个小错误不放。

2. **开会时最后一个进入会议室**：焦虑的想法是“所有人会盯着我看，想找到我身上的缺点”。质疑：人们看着我真的是为了找到我的缺点吗？结论：大多数人都会习惯性地去看新进门的人，他们并不是为了找到别人身上的缺点。正确的想法：人们盯着刚进入房间的人是很正常的行为，并不是对我感兴趣。

3. **自身外在原因导致的社恐**：焦虑的想法是“我又胖又矮，别人会取笑我”。质疑：别人真的会在乎我的长相吗？结论：大多数人的身高和长相都不是理想的状态，而关注自己优点的人才会过得更好。正确的想法：无论自己的身高长相如何，只要在社交中展现出积极正能量的一面，人们也会对我非常友好。

**总结**

通过记录和质疑，将错误的思维模式转变为正确的思维模式，焦虑的想法转变为积极正向的想法，从而改变潜意识，突破社恐的恶性循环。
