# 安老师社恐训练营：19-练习：突破社恐的恶性循环(二)_note

在本节课中，我们将要学习如何通过改变思维模式来打破社交恐惧的恶性循环。上节课我们介绍了从减少自我关注入手的练习方法，本节中我们来看看另一个关键的突破口：修正我们内心那些自动产生的、引发焦虑的想法。

改变思维模式并非易事，但一旦掌握，便能从根本上解决社交恐惧。那么，该如何通过锻炼来改变思维模式呢？核心要点是：**必须用纸笔或电子设备记录下来**。许多人不愿记录，但不动手记录会让想法变得模糊，最终导致练习无果。因此，强烈建议养成记录的习惯。

既然要记录，我们具体记录什么呢？目标是：**将错误的思维模式转变为正确的思维模式，将焦虑的想法转变为积极正向的想法**。

在记录时，可以尝试使用“思维卡片”的形式。这很简单，将笔记本的一页视为一张卡片即可。采用卡片形式有助于分类整理，让我们在解决问题时保持头脑专注、条理清晰。

还记得之前课程中提到的“写下焦虑场景”吗？我们可以任意选取其中一个场景进行练习。以下是具体的操作步骤：

首先，描述一个引发焦虑的具体场景。
*   **示例场景**：与一位不太熟悉的同事聊天时，因紧张而说话结巴。

接着，剖析并写下在这个场景下，你内心自动产生的焦虑想法（即潜意识想法）。每个人的想法可能不同，这里仅作举例。
*   **焦虑想法**：“我说话结巴了，表现得不正常。他肯定对我印象不好，会嘲笑我，以后也会小瞧我。”

然后，**对这份焦虑的想法提出质疑**。只有提出质疑，才能启动改变。
*   **提出质疑**：“别人真的会因为我说话结巴就不喜欢我吗？我验证过吗？当别人说话结巴或偶尔卡壳时，我也不会因此批评、嘲笑或不喜欢他们吧？”

提出质疑后，便可以**得出一个更客观、理性的结论**。
*   **得出结论**：“人们在沟通时偶尔说话结巴是很正常的现象，很少有人会持续关注这一点。”

最后，基于以上分析，**形成你应该拥有的正确想法**。
*   **正确想法**：“人们在说话时犯点小错误很常见。如果我说话结巴，自然纠正过来就好，不会有人抓着这个小错误不放。即使我犯了错，人们也不会因此小瞧我。”

当你不断重复这个过程，用正确的想法反复刺激和提醒自己时，它就会逐渐内化为你的新潜意识。今后再遇到类似场景，你就不会条件反射般地陷入恶性循环了。

我们再来看另一个场景的例子。

**场景**：开会时最后一个进入会议室，感到不自在和紧张。
*   **焦虑想法**：“所有人都会盯着我看，想找出我身上的缺点。”
*   **提出质疑**：“人们看着我真的只是为了找缺点吗？是所有人都在看，还是只有一部分人？”
*   **得出结论**：“大多数人只是习惯性地看向新进门的人，并非为了找茬。很多时候，人们更在意自己的工作内容，对新进来的人并无特别兴趣。”
*   **正确想法**：“人们盯着刚进房间的人是一种正常行为，不代表对我感兴趣或要观察我的缺点。我有时也会看别人，所以被人看一下没什么大不了的。”

针对一个场景，你可能有好几个焦虑想法，都可以按照这个流程——写出来、提出质疑、得出结论、最终纠正——逐一处理。

还有一部分人的社交恐惧源于对自身外在条件的担忧，例如身高、体型等。

**场景**：因觉得自己又胖又矮，与人相处时总不自信，无法放松。
*   **焦虑想法**：“我又胖又矮，别人会取笑我。”
*   **提出质疑**：“别人真的那么在乎我的长相和身高吗？我又不是明星，难道别人整天闲着讨论我的外表？”
*   **得出结论**：“大多数人的身高和长相都非理想状态，而关注自身优点的人会过得更好。”
*   **正确想法**：“无论身高长相如何，只要在社交中展现出积极、正能量的一面，人们也会对我友好。就算有人不喜欢我的外表，我也无需和所有人成为挚友或恋人。做好自己，善于发现自身优势就好。”

大家需要根据自己的实际情况，写下让你焦虑的场景和想法。只有通过不断练习，不断向潜意识灌输新的信息，才能慢慢纠正那些根深蒂固的焦虑想法，从而在心态上产生巨大转变。以前你会条件反射地从焦虑、消极的角度考虑问题，而现在将学会条件反射地从积极、乐观的角度去思考。

以上就是“思维卡片”的练习方式。当然，之前课程中提到的“完整的想法记录表”形式也能达到同样效果，它只是多了一个环节：**与行动方案联系**，即思考“以前我是怎么做的？现在我能做出什么改变？”。

这两种方式都可以，选择你喜欢的一种即可。它们的目的只有一个：**帮助你将潜意识里的焦虑想法，转变为正向、积极的想法**。

最后再次强调，潜意识想法就是你条件反射般的想法。当它被纠正后，你就不会再像以前一样，自动陷入社交恐惧的恶性循环，而是能够变得从容、自在、自信且乐观。这就像一个开关，这就是为什么有时你在练习中，会不知不觉地发生改变，在社交场合莫名地不再紧张了。看似神奇，实则是这个原理在起作用。

本节课中我们一起学习了如何通过“记录-质疑-修正”的流程来改变引发社交恐惧的思维模式。坚持这项练习，你将能有效打破思维的恶性循环，逐步建立起积极、自信的社交心态。
