# 驾驭关系的能力：情绪价值：02 三种核心价值 🧠

在本节课中，我们将要学习人与人之间关系的三个核心价值。理解这三种价值，是掌握所有关系互动本质的基础。

人与人之间永远都是价值的交换。在任何情感关系中，无论是恋爱阶段、婚姻阶段，还是其他亲密关系，都离不开价值的交换。

因此，我们必须了解人与人之间的三个核心价值。

## 三种核心价值是什么？ 💎

这三种核心价值分别是：**外在价值**、**社会价值**和**情绪价值**。

接下来，我们将分别详细讲解这三种价值。

## 一、外在价值 👤

外在价值是最直观、最容易理解的价值。

我们先从男性的外在价值说起。男性的外在价值包括：**长相、身高、健康状况**。这些都是用眼睛就能看到的价值。

女性的外在价值与男性类似，也包括：**长相、身高、气质、健康状况**。男女的外在价值基本一致。

我们可以回想，情感的起源往往来自于外在价值。也许一个男人并不帅气，身高也不好，但他可能因为幽默、热情、执着或胆大而吸引了你。同样，女人吸引男人的首要因素也常常是外在价值。

爱情并非要求对方必须长得像大明星。一个女孩外在的某一点，就可能吸引一个男人去追求她。例如，一个女孩可能因为胸大、个子高、气质秀气等某一点而被喜欢。

调查显示，女人可能比男人更好色。因此，男人的帅气也是打动女人的一个核心点。如今，年轻一代中，女性主动表白的情况越来越普遍。

外在价值往往是吸引异性的一个非常重要的起点。

外在价值就讲到这里，这个概念比较容易理解。

## 二、社会价值 🏢

接下来，我们看看第二种价值：社会价值，也可以称为现实价值。

在社会价值这个维度，男女有着巨大的分工差异。我们首先讲解男性的社会价值。

绝大多数男性的社会价值排序如下：**外部价值**排在前面，**内部价值**排在后面。

*   **外部价值**：指男性与外部世界打交道的能力，例如在陌生人、生意场中建立关系的能力。
*   **内部价值**：指处理家族、兄弟姐妹、家庭内部关系的能力。

男人的首要社会价值是**赚钱的能力**，即他的**社会地位和社会关系**。例如，一个男人是官员或成功企业家，他就拥有较高的社会价值。

男人的次要社会价值是维护家庭内部关系、教育孩子、成为家庭榜样等能力。但通常，男人在这方面并不擅长。

恰恰是这方面的社会价值，主要由女性来承担。因此，社会价值的分工就很明确了：**男人的社会价值主要分工在外（赚钱、地位），女人的社会价值主要分工在内（持家、维系关系）**。

理解了分工，我们就能明白一些现象。为什么男人一有钱就可能变化？因为他实现了对外的社会价值。而外面的女人，往往是用自己的**外在价值**（年轻、漂亮）来交换男人的**社会价值**（金钱、地位）。

反过来说，如果女性实现了强大的社会价值（例如成为成功的企业家），她同样可以用自己的社会价值去交换别人的外在价值（例如帅气、阳光的伴侣）。

但我们必须承认，在绝大多数家庭中，男性提供了更大的外在社会价值。这也是很多婚姻问题的根源之一。

社会价值就讲到这里。

## 三、情绪价值与价值失衡 💔

上一节我们介绍了情绪价值的概念。当我们明白了人与人之间的这三种价值后，就能从根源上分析婚姻不幸的原因。

首先看**外在价值**。对于老夫老妻，外在价值的吸引力会下降，甚至消失。喜新厌旧是人性，妻子很难在外在价值上持续满足丈夫。

再看**社会价值**。如果家庭主要依靠丈夫的社会价值（赚钱能力），而妻子的社会价值（持家、教育）容易被替代（例如保姆、机器人），那么妻子在这方面的价值对丈夫的吸引力也会减弱。

最后看**情绪价值**。如果妻子既无法提供有吸引力的外在价值，社会价值又容易被替代，同时还**不具备提供情绪价值的能力**，那么在丈夫眼中，她的综合价值就很低。

当一个男人实现社会价值后，如果他在婚姻中得不到足够的价值满足（尤其是新鲜感和情绪价值），他自然可能向外寻找。这就是许多背叛发生的逻辑。

那些被称为“捞女”的第三者之所以厉害，是因为她们往往同时具备**外在价值**（年轻漂亮）和**情绪价值**（善于提供情绪支持），从而能吸引到成功的男性。

我们的婚姻失败，往往不是败在不够漂亮（外在价值会衰减），而是败在**没有学会提供情绪价值**。

## 总结与启示 ✨

本节课我们一起学习了关系的三种核心价值。

我们来梳理一下：
*   **男人**主要需要实现**社会价值**。
*   **女人**则需要兼顾**外在价值**和**情绪价值**。

外在价值是你的“敲门砖”，是建立初始吸引的关键。即使为了未来的选择，也不能放弃管理自己的外在形象。

而**情绪价值，才是获得长久幸福、稳固关系不被抛弃的核心能力**。这正是本系列课程要帮助大家掌握的重点。

如果从一开始就懂得学习并提升情绪价值，你就能在关系中占据主动。即使当前关系不如意，掌握了情绪价值，你也能拥有驾驭更优秀关系的底气与能力。

学习永远不会晚，现在就开始行动吧。
