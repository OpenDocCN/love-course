# 17、海蓝老师情绪梳理个案课：062-任佳个人采访

🎼
我是任佳，今年27岁，在广州工作，是一名大学教师。我想梳理的情绪是恐惧权威。这种感觉是，无论是工作中还是学习中遇到领导或老师这样的角色时，总是希望离他们远远的，觉得这样比较安全、自在。但我们也知道，领导和老师拥有丰富的资源，很值得近距离学习。所以，这种恐惧权威的情绪对我的工作、学习和生活带来了一定的困扰。比如，在工作中，即使只是和领导一起走到饭堂的路上，领导说什么，我都会说是，对，嗯嗯，然后除此之外就没有别的话可以反馈。身体感觉非常紧张，像一块发紧的小路在蹦蹦跳，很想尽早离开现场。从身体到情绪都非常紧张，领导也能感受到这种不自然的状态。比如，工作中做得好的地方，领导可能不能立刻看到；或者工作中有问题需要反馈时，我总是犹豫不决，这样会影响工作效率和职业成长。这就是我觉得比较困扰的状况。

如果我能放下恐惧权威的情绪，比如在工作中和领导的交流会更自在，甚至可以平等自如地交流。这是我非常渴望的状态。在学习中，如果能自信流畅地表达自己的想法，生活会非常轻松。从工作上来说，工作效果会更有成效。面对老师这样的权威，学习上的成长也会更大。生活中遇到权威的长辈，我也会很自在地交流，感觉很舒服，身体放松，情绪平静，整个状态很自由自在。我自己很放松，对方也会很放松。这就是我特别想要达到的一个状态。
🎼

## 概述

在本节课中，我们将学习如何梳理和克服恐惧权威的情绪，探讨其对工作、学习和生活的影响，并探讨如何达到更自在、自信的状态。

## 核心概念

**恐惧权威**：指在面对权威人物时，产生的不安、紧张、退缩等情绪。

## 情绪梳理步骤

以下是梳理恐惧权威情绪的步骤：

1. **识别情绪**：认识到自己在面对权威人物时的不安、紧张等情绪。
2. **分析原因**：思考这种情绪产生的原因，如过去的经历、社会文化等。
3. **调整认知**：改变对权威人物的负面认知，如他们并非不可接近、不可沟通等。
4. **实践练习**：通过模拟练习、角色扮演等方式，提高与权威人物沟通的自信心。
5. **寻求支持**：与朋友、家人或专业人士分享自己的感受，寻求支持和帮助。

## 总结

本节课中，我们一起学习了如何梳理和克服恐惧权威的情绪，探讨了其对工作、学习和生活的影响，并探讨了如何达到更自在、自信的状态。希望同学们能够将所学知识应用到实际生活中，克服恐惧权威的情绪，实现更好的自我成长。
