# 天边老白一对一私教：第24课：则明朋友圈调色，手动滤镜详细教程_note

在本节课中，我们将学习如何为照片进行手动调色，打造独特的“则明朋友圈”风格滤镜。我们将不使用预设滤镜，而是通过调整几个核心参数，让你完全掌控照片的色彩和氛围。

## 概述

调色的本质是控制照片的色彩、明暗和对比度。我们将通过调整几个关键参数来实现。这些参数就像照片的“调味料”，不同的组合能创造出截然不同的视觉效果。

上一节我们介绍了调色的基本概念，本节中我们来看看具体需要调整哪些参数。

## 核心调色参数

以下是实现手动滤镜效果需要关注的四个核心参数。

1.  **曝光**
    *   控制画面的整体亮度。增加曝光使画面变亮，减少曝光使画面变暗。

2.  **对比度**
    *   控制画面中亮部与暗部的差异程度。提高对比度能让亮部更亮、暗部更暗，画面更“通透”；降低对比度则让画面趋于灰平。

3.  **高光与阴影**
    *   **高光**：专门控制画面中最亮的部分（如天空、反光）。
    *   **阴影**：专门控制画面中最暗的部分（如背光的角落、暗部细节）。降低高光、提亮阴影是获得“胶片感”或柔和画面的常用手法。

4.  **色温与色调**
    *   **色温**：控制画面的冷暖。向蓝色方向调整（降低数值）让画面变冷，向黄色方向调整（增加数值）让画面变暖。
    *   **色调**：控制画面的绿-洋红平衡。向绿色方向调整（降低数值）可中和肤色中的红色，向洋红色方向调整（增加数值）可为画面增添复古感。

## “则明风格”调色步骤

掌握了核心参数后，我们来看看如何组合它们，一步步调出“则明风格”的照片。

以下是具体的操作步骤，请按顺序尝试。

*   **第一步：基础明暗调整**
    *   轻微降低**曝光**（-0.3 至 -0.7）。
    *   大幅降低**高光**（-50 至 -80）。
    *   适当提高**阴影**（+20 至 +40）。
    *   微增**对比度**（+5 至 +15）。

*   **第二步：色彩氛围营造**
    *   将**色温**向黄色方向调整（+10 至 +20），为画面增加暖调。
    *   将**色调**向洋红色方向微调（+5 至 +10），增添复古韵味。

*   **第三步：色彩精细调整（HSL）**
    *   **橙色**：降低饱和度（-10），提高明度（+15），可使肤色更干净通透。
    *   **绿色**：改变色相向墨绿方向（-20），大幅降低饱和度（-30），降低明度（-20），可使植物呈现低调、高级的墨绿色。
    *   **蓝色**：改变色相向青蓝色方向（-5），适当降低饱和度（-10），适用于调整天空或衣物。

*   **第四步：增加质感**
    *   适当增加**纹理**和**清晰度**（各 +10 至 +20），让细节更突出。
    *   轻微增加**晕影**（总量 -10，中点 50），使视觉焦点更集中。

## 公式与代码示例

理解参数后，我们可以用伪代码来描述这个调色流程，使其逻辑更清晰。

```pseudocode
函数 则明风格调色(照片) {
    // 第一步：基础明暗
    照片.曝光 = 照片.曝光 - 0.5;
    照片.高光 = 照片.高光 - 70;
    照片.阴影 = 照片.阴影 + 30;
    照片.对比度 = 照片.对比度 + 10;

    // 第二步：色彩基调
    照片.色温 = 照片.色温 + 15;
    照片.色调 = 照片.色调 + 8;

    // 第三步：HSL调整（以橙色和绿色为例）
    照片.HSL[“橙色”].饱和度 = 照片.HSL[“橙色”].饱和度 - 10;
    照片.HSL[“橙色”].明度 = 照片.HSL[“橙色”].明度 + 15;

    照片.HSL[“绿色”].色相 = 照片.HSL[“绿色”].色相 - 20;
    照片.HSL[“绿色”].饱和度 = 照片.HSL[“绿色”].饱和度 - 30;
    照片.HSL[“绿色”].明度 = 照片.HSL[“绿色”].明度 - 20;

    // 第四步：增加质感
    照片.纹理 = 照片.纹理 + 15;
    照片.清晰度 = 照片.清晰度 + 15;
    照片.晕影.总量 = -10;

    返回 照片;
}
```
