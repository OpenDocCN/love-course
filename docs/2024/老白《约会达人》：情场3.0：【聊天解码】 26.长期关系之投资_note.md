## 投资的概念

上一节我们介绍了长期关系中的各种互动模式，本节中我们来看看一个至关重要的因素：投资。

投资指的是一段感情中双方所付出的各种资源。我们会发现一个常态：往往在一段感情中付出最多的一方，在分手后是最痛苦的。这些付出就是我们今天要讨论的投资。

## 投资的四种形式

投资并非单一概念，它具体体现在多个维度。以下是感情中四种主要的投资形式：

1.  **金钱投资**：为关系花费的金钱，例如礼物、共同消费。
2.  **时间投资**：为陪伴对方、经营关系所投入的时间。
3.  **情绪投资**：因对方而产生的喜怒哀乐等情感波动和投入。
4.  **肉体投资**：在亲密接触和性关系中双方的投入与探索。

## 平衡投资的重要性

在一段长期关系中，我们不仅要自己付出这些投资，也要引导对方为你付出相应的投资。否则，你们的关系就会变成一边倒的状态。这是不正确，也是不健康的。

很多男性到最后付出越来越多，一点麻烦不想让女朋友受，一点气不想让女朋友生，一点钱都不想让她花，觉得女朋友就是应该捧在手心里的掌上明珠，一点委屈都不能受。这样会导致女生觉得自己不需要付出任何努力，就能得到你所有的爱。最终结果就是你越来越离不开她，可一旦分手，你的痛苦会远大于她。

## 如何引导对方投资

那么，从今天开始，要学会引导对方为你投资。以下是具体的方法：

*   **引导时间投资**：要求他为你做一些需要花费时间的事情。例如，让他每天回家和你报个平安，让他花时间为你做饭、为你学习新技能、为你织一件毛衣，或者陪你一起去旅行。

*   **引导情绪投资**：不要什么都依着他。让他因为你而生气、因为你而难过，也偶尔因为你制造的惊喜而开心。情绪上的波动本身就是一种深度投资。

*   **引导金钱投资**：不要再什么钱都不让他花。可以适当地接受他送你的贵重礼物，出去吃饭时，也可以让他请你。慢慢形成“我们俩出去，花谁的钱都一样”的共同体状态。

*   **引导肉体投资**：让他为你学习一些性技巧，尝试一些之前没有体验过的亲密姿势。因为性只有在双方全身心投入和探索下，才能达到最完美的交融。不要害羞，要引导他明白，与最亲密的人共同探索这些是正常且美好的。

## 投资的“赌徒心态”

我们今天讨论的投资，其心理机制类似于赌徒去赌博。我们可以用一个公式来类比这种心态：

**`已付出成本 + 不甘心心理 = 持续追加投资`**

假设你有5万元本金去赌博。前半夜你输了2万，你通常不会收手，因为你已经投资了2万，你想要赢回来，至少指望保本。结果后半夜你把剩下的3万也输了，到最后甚至借钱也要去赌，然后越输越多。

换算到恋爱中也是一样的：**你在一段关系中投资的越多，退出成本就越高，越难以割舍，从而可能陷入持续付出却得不到平衡回报的循环，最终“输”得更多。**

## 课程总结

本节课中，我们一起学习了长期关系中的“投资”概念。

一段健康的长期关系，需要双方共同维持和投资，而不是单方面的无限付出。投资主要包括**金钱、时间、情绪和肉体**四种形式。关键在于学会平衡，既要真诚付出，也要智慧地引导对方为这段关系投资，这样才能建立起平等、持久且亲密的关系。
