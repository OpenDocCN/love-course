# 老白《约会达人》：情场3.0：【聊天解码】 26.长期关系之投资

🎼我是老白。本节课将探讨长期关系中一个至关重要的概念——投资。在一段感情中，我们常常发现，付出最多的一方在分手后往往最痛苦。这些付出，就是我们所说的投资。

**投资类型**

在一段长期关系中，投资包括金钱、时间、情趣和肉体四个方面。不仅需要为女朋友付出这些投资，还要引导她为你付出相应的投资。

**投资平衡**

若关系失衡，一方过度付出，另一方则可能产生依赖，认为无需努力即可获得全部爱。这种状态下，一旦分手，痛苦往往更大。

**投资要求**

1. **时间投资**：要求对方每天与你分享日常，共同做饭、学习、旅行等。
2. **情绪投资**：引导对方在情绪上为你付出，如因你而生气、难过或开心。
3. **金钱投资**：适当让对方消费，如送礼物、请客等。
4. **肉体投资**：引导对方学习性技巧，尝试新的姿势，实现双方全身心融入。

**投资心理**

投资心理类似于赌徒心理。一旦投入越多，越难以收手，最终可能导致更大的损失。

**总结**

健康的长期关系需要双方共同维持投资，而非单方面的付出。投资方式包括金钱、时间、情绪和肉体四个方面。
