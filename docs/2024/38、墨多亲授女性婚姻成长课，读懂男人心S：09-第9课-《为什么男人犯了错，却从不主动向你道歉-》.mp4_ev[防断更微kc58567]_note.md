# 墨多亲授女性婚姻成长课 第9课 - 为什么男人犯了错，却从不主动向你道歉

大家好，欢迎来到第九课。今天我们将探讨一个让许多女性困扰的问题：为什么男人犯了错，却从不主动向你道歉？

## 1. 负面情绪的积累

这就像能量守恒定律，负面情绪不会凭空消失。很久之前的问题没有解决，负面情绪没有抵消，时间长了，会在女性的心理造成很大的伤害。

### 1.1 第三者现象

一个家庭如果老公出现了第三者的问题，哪怕老公承诺回归，彻底和第三者分开，决定为家庭和婚姻付出更多。但是男人始终没有道歉，这依然会让女性想起就非常难受。

## 2. 道歉的重要性

道歉意味着男人承认这个事情很重要，他看见了女人的伤口，会让女人感觉到自己依然是被重视的。

### 2.1 不愿意道歉的本能

不愿意道歉也是人性的一种本能。源自人自两种重要的本能需求：需要对现实的确定感和维持自我价值观的完整。

### 2.2 真正的道歉

真正的道歉应该是承认错误，并表达歉意。然而，现实中很多男人的道歉并不真正解决问题。

## 3. 女性如何化解

女人在面对男人不认错的时候，需要借巧劲来化解。以下是一些建议：

### 3.1 给予安全感

只要你从这一层面给了他安全感，真心认识到错误的男人，自己会找台阶下的。

### 3.2 重新判断

如果他真的从你的努力中，并没有表现出自己忏悔，你的意思，不管大事小事儿，你再重新对这个人做出判断。

## 4. 总结

本节课中我们一起学习了为什么男人犯了错，却从不主动向你道歉，以及女性如何化解这个问题。希望这些内容能帮助到大家。
