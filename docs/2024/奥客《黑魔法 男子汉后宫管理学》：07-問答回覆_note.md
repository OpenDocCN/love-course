# 奥客《黑魔法 男子汉后宫管理学》：07-問答回覆

## 概述

在本节课中，我们将学习如何回答后宫管理中的各种问题，包括炮友与盘子的区别、如何提升自己的阿法属性、如何处理与正宫的关系、如何应对竞争焦虑等。

## 炮友与盘子的区别

**公式**：炮友 = 上床关系；盘子 = 潜在炮友

炮友与盘子的区别在于是否已经发生上床关系。只要双方有上床关系，就可以理解为炮友框架。

## 提升阿法属性

**公式**：阿法属性 = 社交认证 + 竞争焦虑

可以通过观察老板如何与别人交往，学习他们的方式，并借鉴到自己的社交中。同时，可以借用老板的社交认证来提升自己的阿法属性。

## 处理与正宫的关系

**公式**：正宫 = 竞争焦虑 + 框架

正宫的存在可以让你保持竞争焦虑，从而维持长期互动。同时，要展现自己私底下有很多盘子的状况，以保持正宫的竞争焦虑。

## 应对竞争焦虑

**公式**：竞争焦虑 = 亲密 + 时间

竞争焦虑是维持良好关系的重要条件。可以通过增加亲密或时间来降低竞争焦虑。

## 双正攻的情况

**公式**：双正攻 = 互相隐瞒 + 选择

双正攻的情况需要互相隐瞒，并让对方做出选择。如果对方有尊严，可能不会接受与别的女人共用。

## 遇到飞单的盘子

**公式**：飞单的盘子 = 恐怖前男友 + 热度

如果遇到飞单的盘子，建议考虑与对方交往后可能出现的问题，并思考是否值得继续交往。

## 已婚状态的处理

**公式**：已婚状态 = 风险 + 筛选

已婚状态下，可以通过筛选对象来降低风险。可以通过展示家庭照片等方式来筛选能够接受已婚状态的对象。

## 与疯女人交往

**公式**：疯女人 = 成本 + 退场

与疯女人交往需要承担很多成本，并学会优雅退场。

## 正宫的竞争焦虑

**公式**：正宫的竞争焦虑 = 竞争 + 废测

正宫的竞争焦虑可以通过观察对方的行为和废测来判断。

## 保留盘子

**公式**：保留盘子 = 断片 + 观察反应

如果想要保留盘子，可以假装没有发生过任何事情，并观察对方的反应来判断是否值得继续交往。

## 正宫的意义

**公式**：正宫 = 好人 + 完整人格

正宫的存在可以让你成为一个好人，并帮助你完善人格。

## 长期不被发现

**公式**：长期不被发现 = 观察三观 + 擦干净

要长期不被正宫发现，需要观察对方的三观，并保持低调。

## 年薪硬价值的理论

**公式**：年薪硬价值 = 多多益善

年薪越高，越不会受到女性的嫌弃。

## 想上位

要断绝对方的念头，并让对方知道你只想玩玩而已。

## 人妻丑

**公式**：人妻丑 = 温柔体贴 + 人性

人妻丑需要珍惜她们的人性，并善待她们。

## 感情炮

**公式**：感情炮 = 资源投入 + 关注消耗

感情炮需要投入很多资源和关注，需要考虑清楚是否值得。

## 母胎单身

**公式**：母胎单身 = 看笑接答 + 累积知识

母胎单身可以通过看笑接答和累积知识来提升自己。

## 专注事业

**公式**：专注事业 = 放掉盘子 + 专注

要专注事业，需要放掉盘子，并专注于自己的发展。

## 放掉盘子

**公式**：放掉盘子 = 观察三观 + 选择

要放掉盘子，需要观察对方的三观，并做出选择。

## 正宫的竞争焦虑

**公式**：正宫的竞争焦虑 = 竞争 + 废测

正宫的竞争焦虑可以通过观察对方的行为和废测来判断。

## 正宫的位置

**公式**：正宫的位置 = 情报 + 超越

要稳住正宫的位置，需要了解对方的过往经历，并从各方面超越对方。

## 时间分配

**公式**：时间分配 = 拒绝 + 优先级

要合理分配时间，需要学会拒绝，并确定优先级。

## 女友健身

**公式**：女友健身 = 自我提升 + 代价

女友健身是自我提升的一种方式，需要付出代价。
