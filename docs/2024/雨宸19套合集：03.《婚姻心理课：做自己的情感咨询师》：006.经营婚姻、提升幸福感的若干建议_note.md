# 婚姻心理课：做自己的情感咨询师 05. 经营婚姻、提升幸福感的若干建议

## 概述

在本节课中，我们将学习如何经营婚姻，提升幸福感。我们将探讨以下六个方面的建议：

1. 学会放弃完美主义，包容差异
2. 保持一定的距离，关注自身成长
3. 坦诚交流内心感受
4. 妥善处理争吵
5. 经常性的身体接触
6. 真诚的感恩和赞美

## 1. 学会放弃完美主义，包容差异

**公式**： 完美主义 → 错位的反需求状态 → 接受差异 → 内心包容度提升

在恋爱关系中，我们常常期望伴侣能够弥补我们过去在原生家庭中缺失的爱和被爱的感受。然而，伴侣并非我们的父母，他们有自己的需求和成长空间。学会接受伴侣的差异性，包容他们的缺点，是建立和谐关系的关键。

## 2. 保持一定的距离，关注自身成长

**公式**： 亲密关系 → 独立性 → 给彼此留空间 → 避免乏味

恋爱并非生活的全部，我们需要关注自身成长，包括朋友、家人、爱好和工作等方面。保持一定的距离，给彼此留出空间，有助于维持关系的活力。

## 3. 坦诚交流内心感受

**公式**： 自我暴露 → 分享秘密 → 增进亲密感

分享秘密是建立亲密关系的重要途径。通过分享个人习惯、兴趣爱好、价值观和态度等，我们可以增进彼此的了解，拉近心与心的距离。

## 4. 妥善处理争吵

**公式**： 表达情绪 → 客观陈述 → 避免评价 → 深层次沟通

争吵时，我们要学会客观陈述事实，表达自己的感受，避免评价对方。争吵的目的是为了解决问题，而非发泄情绪。

## 5. 经常性的身体接触

**公式**： 亲密接触 → 产生亲密感 → 增强情感浓度

拥抱、牵手、依偎等亲密接触可以增进彼此的亲密感，增强情感浓度。

## 6. 真诚的感恩和赞美

**公式**： 感恩 → 赞美 → 积极心态 → 提升幸福感

真诚地感恩和赞美伴侣，可以帮助我们保持积极的心态，提升幸福感。

## 总结

本节课中，我们学习了如何经营婚姻，提升幸福感。通过学会放弃完美主义、保持距离、坦诚交流、妥善处理争吵、经常性的身体接触以及真诚的感恩和赞美，我们可以建立和谐、幸福的婚姻关系。希望这些建议能够帮助到大家。
