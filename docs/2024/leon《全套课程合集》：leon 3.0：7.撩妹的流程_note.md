# 《全套课程合集》：leon 3.0：7.撩妹的流程

欢迎来到第七节课。本节课我们将探讨撩妹的流程，帮助大家梳理两性认知，明确当前与女生的互动处于哪个环节，并学会如何解决问题。

## 撩妹流程概述

### 传统模型：M3模型

传统的M3模型包括三个步骤：价值感建立、舒适感建立，最后再次升级。这个模型仍然适用，可以作为认知中的底层参考。

### 新模型：TOP 5步骤

新的TOP 5步骤包括：打开、前提、评估、叙事、收尾。

## 撩妹流程详解

### 1. 打开

**目的**：获得女生的注意力。

**方法**：直接开场或进阶开场。

### 2. 意图表达

**目的**：让女生知道你对她的态度和想法。

**方法**：明确表达意图，避免墨迹。

### 3. 评估

**目的**：筛选合适的女生。

**方法**：根据标准筛选，体现价值感。

### 4. 叙事

**目的**：让女生了解你的故事。

**方法**：讲故事，建立信任感。

### 5. 收尾

**目的**：升级关系。

**方法**：在搭讪、约会、聊天过程中推进关系。

## 总结

本节课中，我们学习了撩妹的全流程，包括开场、意图表达、评估、叙事和收尾。希望大家能够掌握这些步骤，提升自己的撩妹技巧。
