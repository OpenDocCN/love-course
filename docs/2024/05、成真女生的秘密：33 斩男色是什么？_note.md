# 课程05：💄 成真女生的秘密：33 斩男色是什么？

在本节课中，我们将探讨口红世界中的一个有趣现象——特定色号的文化含义，特别是被称为“斩男色”的流行色号。我们将了解口红的普遍情况、保质期与价格，并重点解析“斩男色”这一概念背后的社交意涵。

---

口红色号种类繁多，颜色设计五花八门。通常，化妆的女生家中一般不止拥有一支口红。

很多女生提到，她们的口红基本上都会放过期。因为一支口红的保质期大约是三年。但现在有测评指出，并且很多女生也知道，口红即使真的过期，问题通常也不大。其细菌和微生物含量很少，仍然可以继续使用。过期后使用关系不大。

那么，一支口红的价格范围也很广，从几十元到几百元不等。最贵的品牌如“萝卜丁”，听说要700元一支，但很少有人使用。基本上，便宜的口红有四五十元一支的，稍微贵一点的一百多元的也很常见，还有两三百元的。

此外，在女生圈子中，特定的色号会流行起来。例如，“斩男色”。这个“斩”是刀砍的斩。意思是女生涂上这种颜色的口红后，会非常吸引男性，男性会被这个颜色吸引走。

女生中还存在其他一些具有特定含义的颜色。例如，有些色号被称为涂完后能够“吓退男生”。有些男生可能想去搭讪，但看到女生涂了这种颜色，就可能望而却步，不敢上前。她的整体形象可能因此显得非常盛气凌人。

女生对于口红颜色的认知，不仅仅关乎颜色本身，其背后还承载着这些社交含义。

---

本节课中，我们一起学习了口红的基本概况，包括其多样性、保质期与价格区间。更重要的是，我们探讨了像“斩男色”这样的特定色号如何在女性文化中被赋予超越美学之外的社交意义，成为表达吸引力或建立距离感的非语言工具。
