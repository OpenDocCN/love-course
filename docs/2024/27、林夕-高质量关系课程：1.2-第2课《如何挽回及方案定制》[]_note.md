# 27、林夕-高质量关系课程：1.2-第2课《如何挽回及方案定制》📚

## 概述
在本节课中，我们将学习如何定制挽回方案，通过分析情感关系中的冲突点、双方需求和导火索事件，制定出精准有效的挽回策略。

## 定制挽回方案
### 找到冲突点
**公式**：冲突点 = 情感关系中的转折点 - 双方价值与需求的变化

**步骤**：
1. **分析时间线**：回顾感情中的转折点，包括初次相遇、关系发展、出现情感问题、分手等。
2. **识别价值与需求**：分析双方在关系中的价值吸引点和需求变化。
3. **确定导火索事件**：找出导致分手的具体事件。

### 例子分析
**案例**：男女双方因价值观和需求冲突导致分手。

**分析**：
- **价值与需求**：男生追求外在条件，女生追求物质条件。
- **导火索事件**：男生疑似出轨，女生扇巴掌。

### 制定策略
1. **消除负面预判**：通过锻炼、接触其他异性等方式，让对方淡化负面印象。
2. **道歉**：针对导火索事件进行真诚道歉，表达愿意做朋友的态度。
3. **提升自身能力**：学习如何与优质男相处，提升情商和认知水平。

## 总结
本节课中，我们学习了如何定制挽回方案，通过分析冲突点、双方需求和导火索事件，制定出精准有效的挽回策略。希望所有学员都能在情感关系中找到属于自己的幸福。
