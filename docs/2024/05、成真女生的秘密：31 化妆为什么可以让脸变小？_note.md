# 课程05：成真女生的秘密：31 化妆为什么可以让脸变小？💄

在本节课中，我们将要学习化妆中“修容”这一技巧的原理与操作方法。修容通过光影的视觉原理，能够有效修饰脸型，让面部轮廓在视觉上显得更小、更立体。

## 修容的作用与工具

修容的主要作用是修饰脸型与五官轮廓。它通常使用深灰、棕色或黑色的粉状产品。

以下是修容常用的工具与产品：
*   **产品质地**：粉状质地。
*   **核心工具**：专门的化妆刷。
*   **常用颜色**：深灰色、棕色、黑色。

## 修容的操作位置与效果

上一节我们介绍了修容的基本工具，本节中我们来看看它的具体使用位置和能达到的效果。

修容粉被涂抹在面部特定的阴影区域，以创造视觉上的“凹陷感”。

以下是主要的操作位置及其效果：
*   **下颚骨两侧**：涂抹在此处可以修饰宽脸或方脸（国字脸），在视觉上收窄脸型，趋向于鹅蛋脸或瓜子脸。
*   **鼻子两侧**：涂抹在鼻梁两侧，可以在视觉上让鼻梁显得更高挺，同时让鼻孔看起来不那么明显。

## 修容的视觉原理

了解了如何操作后，我们来看看其背后的科学原理。修容的核心原理是利用色彩明暗对比来制造视觉错觉。

其作用类似于将一束深色的光照射在面部两侧。从正面观看时，面部中央明亮的“留白”区域在对比下会显得面积更小。

这个原理可以用一个简单的视觉公式来理解：
> **深色区域（阴影）** 在视觉上 **后退**，**浅色区域（高光）** 在视觉上 **突出**。

通过将深色修容产品涂在希望“缩小”或“后退”的区域（如脸侧、鼻翼），剩余的面部留白部分在视觉上就显得更集中、更小巧，从而实现了修饰脸型、让脸变小的效果。
