# 课程14：关键对话20讲：如何成为沟通高手 🗣️  
## 6-逻辑思维：建构起理性世界的根本秩序 🧠

在本节课中，我们将学习逻辑思维在关键对话中的应用。逻辑是沟通的核心，它确保我们的表达清晰、有条理，避免误解和混乱。我们将从基本逻辑工具入手，逐步深入逻辑学的核心规律，并探讨常见的逻辑错误及其识别方法。

---

上一节我们介绍了复杂事物的理解能力，本节中我们来看看逻辑思维。逻辑思维是自我能力板块的重要组成部分，它不强调与他人的互动，而是聚焦于如何组织自己的思想，使其更加清晰和有力。

逻辑在对话中处于核心位置。无论是口头交流还是书面交流，逻辑错乱都会给交流带来障碍。本节课的目标是帮助你清晰了解有逻辑的交流需要做到哪些，掌握MECE的逻辑方法，并学会如何避免和识别逻辑谬误。

有人说，说话有逻辑很简单，就是注意分开几点来谈。例如，上来就说“关于这件事，我来讲三点”。然而，对话中的逻辑远不止于此。所谓的“凡事谈三点”也可能出现不少逻辑谬误。

以下是逻辑思维的核心内容：

### 1. MECE原则：不重不漏 📊

MECE是“Mutually Exclusive, Collectively Exhaustive”的缩写，中文意为“不重不漏”。它要求我们在说明一个问题时，用于支撑的论据之间应该互相不重复，并且所有点加起来覆盖了该说的内容。

MECE经常与金字塔模型结合使用。金字塔模型是指一个主论点下有3到7个论据，这些论据之间应符合MECE原则。

例如，如果你是检察官，起诉一个小偷，起诉书应逐一说明犯罪构成的四个要件：主体、客体、主观、客观。你需要检查是否有重复或遗漏。例如，把“有行为能力”在犯罪主体和主观方面各写一遍，就是重复；而想当然地认为犯罪主体没问题，导致没有谈主体，就是遗漏。

不重不漏的原则并非铁板一块。有时可以重复，有时可以遗漏，但这都建立在清晰的MECE框架之上。

具体来说，有些重要问题可能在不同的论据中都用得上。例如，一个恶性杀人案的嫌疑人手段残忍，这既是犯罪的客观方面，也说明其主观恶性极大。手段残忍这个点可以重复在客观和主观方面的论述中。但重点在于，在第二次谈到手段残忍时，需要明确提示：“正如上文所述”。重复要有清晰的意识和必要性。

遗漏也可以是有意识的。在MECE框架下，可以略去一些不重要的论据。例如，向领导说明一个项目应交给你执行，你的论点是“我是最合适的人选”。你组织的论据可以是：1. 你有类似项目经验；2. 你了解客户需求；3. 你参加过相关培训。这三点是MECE的，分别涉及项目经验、客户经验和知识技能。其他优势，如身体好、能熬夜、了解项目落地城市等，虽然也能说明你合适，但不那么直接和重要，可以有意识地略去。

从上述内容可以看出，不重不漏不是僵化的，要灵活使用。重复要有清晰提示，遗漏要有意识选择。这一切都在MECE的规则之下，没有超越其框架范围。

在MECE框架下，还需要解决论据顺序的问题。安排顺序的根本原则是减少信息接收者的负担，让对方觉得不累，才能听得进去。

顺序可以是自然顺序，如时间顺序或空间顺序。例如，讲述一个犯罪嫌疑人的准备、踩点、实施、销赃、落网过程，按照时间顺序就很好。空间顺序如介绍故宫，从中轴线开始，逐一讲解各个院落。

还有一种顺序是原文顺序，即按照对方提问的顺序逐一回答。如果对方的问题本身组织得不好，例如回答了问题3就能解决其他问题，那么可以先直接回答问题3。这就引出了思维顺序。

思维顺序包括重要性顺序、相关性顺序和形式逻辑顺序。重要性顺序是先讲重要问题，再讲具体技术问题。例如，对方给你一份合同，如果大部分是技术问题，就按条款顺序讨论；但如果合同的基本结构有问题，应先提出基本问题。

相关性顺序是先讲最直接相关的问题，再讲间接问题。例如，如果你是程序员转行的科幻作家，讲座时应先讲为什么转行、怎么转行，而不是大谈大学学计算机的故事或锻炼身体的经历。

接下来，我们着重讲一下形式逻辑及其在对话中的应用。

### 2. 形式逻辑：理性思维的基石 ⚖️

逻辑学中，形式逻辑是抽象的思维方式，它强调逻辑形式的正确性，而不强调内容的正确性。形式逻辑有三大规律：同一律、矛盾律、排中律。

- **同一律**：用词、用语、概念必须前后一致。违反同一律就是偷换概念。用公式表达为：**A = A**。
- **矛盾律**：相互矛盾的两个论断，至少有一个是假的。例如，“大地是平的”和“大地是球形的”不能同时为真。用公式表达为：**A ≠ ¬A**（A不能是非A）。
- **排中律**：在同一个思维过程中，两个相反的论述必然有一个是真的，不能同时为假。例如，“蝙蝠是鸟”和“蝙蝠不是鸟”必有一真。用公式表达为：**A ∨ ¬A**（要么是A，要么不是A）。

形式逻辑在对话中常见的应用是**三段论**。三段论由大前提、小前提和结论构成。例如：
- 大前提：所有天鹅都是鸟。
- 小前提：丑小鸭是天鹅。
- 结论：丑小鸭是鸟。

三段论有几个重要规则：
1. 一个三段论只能有三个不同的项，且必须保持同一律。
2. 中项必须至少周延一次。周延是指使用“所有”或“没有”，而不是“有些”。
3. 两个否定的前提不能得出肯定的结论。
4. 前提中有一个是否定的，结论也必须是否定的。

形式逻辑在对话、谈判、辩论中应用广泛。有些人讲话以三段论的形式出现，但前提内容错误或偷换概念。例如，“大公司都是讲诚信的，我们是大公司，所以我们讲诚信。”这里的前提“大公司都是讲诚信的”内容不真。再如，“群众的眼睛是雪亮的，我是群众，所以我的眼睛是雪亮的。”这里偷换了“群众”的集体概念为个体概念。

理解了形式逻辑的三大规律和三段论规则，对于组织自己的逻辑表达和判断他人逻辑是否正确有很大帮助。

### 3. 常见逻辑错误：警惕思维陷阱 🚫

了解了逻辑的基本规则后，我们来看看常见的逻辑错误。这些错误既是要避免的，也是用来识别他人逻辑问题的。

以下是常见的逻辑错误：

1. **错误归因**：人们喜欢为事情找到原因，但许多事情的原因是多元的，甚至不清楚。最常见的错误是把相关性当成因果关系。例如，“公鸡打鸣，太阳升起”总是同时发生，但把公鸡打鸣说成太阳升起的原因就是荒谬的。另一种错误是把因果关系颠倒，例如，“药价贵是因为药厂请了大明星”，而实际是因为药价贵、挣钱多才请得起大明星。

2. **以偏概全**：人们容易把自己的经验当成普遍经验。例如，“我有鼻炎，吃了某种药好了，所以这种药可以治好所有鼻炎。”这只是个人经验，不能推广到所有人。其他例子如“少数民族都能歌善舞”“北京人都特贫”等，都是以偏概全的论断。

3. **虚假结构**：有些话听起来有逻辑，像是因果关系或递进关系，但实际上没有关系，只是漂亮的罗列。例如，“坚持成就习惯，习惯形成性格，性格决定命运。”反过来“命运决定性格，性格形成习惯，习惯成就坚持”听起来也有道理。这种结构没有实际意义。另一种虚假结构是用抽象概念组织没用的话，例如，“以学校远景规划为目标，以学术前沿为引领……”这些内容可以随意调换，但意思不变。

4. **滑坡逻辑**：这种思维认为一旦开始某种趋势，就会一直持续下去，得出耸人听闻的结论。例如，“一旦离职几个月，你的本事就完了。”这种逻辑忽略了生活中的变量和调整可能性。

5. **前提缺陷**：这种错误在于使用片面或错误的前提来推导结论。例如，“你是学生，就要好好学习，先别管别的。”这里的前提“你只是学生”忽略了你的其他身份和角色。这种逻辑用片面前提套路住对方，让人难以反驳。

### 总结 📝

本节课中，我们一起学习了逻辑思维在关键对话中的应用。首先，我们结合金字塔模型介绍了MECE工具，强调论据组织要做到不重不漏，并可以在框架内灵活调整。其次，我们讲解了形式逻辑的三大规律（同一律、矛盾律、排中律）和三段论的基本形式及规则，帮助你组织逻辑表达和识别他人逻辑错误。最后，我们探讨了常见的逻辑谬误，如错误归因、以偏概全、虚假结构、滑坡逻辑和前提缺陷，提醒你在沟通中保持警惕。

逻辑思维不靠天赋，完全可以通过练习提高。日常观察和分析自己及他人的逻辑错误，是提升逻辑能力的有效方法。希望你能将今天学到的知识应用到实际沟通中，使表达更加清晰有力。
