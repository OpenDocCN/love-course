## 言语框架：现场价值的传递

上一节我们介绍了搭讪的核心认知，本节中我们来看看如何在现场通过言语和行为框架传递价值，建立吸引。

搭讪行为本身，就是与目标在现场产生吸引，让对方感知到你的价值。关键在于如何通过现场互动有效地传递这种价值。

在之前提到的认知框架中，人与人的关系本质是共赢的。这衍生出两个要点：第一，当“舔狗”本身就是一种掉价行为；第二，我们不要做无意义的事。

无意义的事是指那些对你的价值塑造没有任何帮助的行为。

现在回到搭讪现场。你需要思考：做什么样的事情能让女生觉得你很有价值？这涉及到言语框架下的多个维度。

以下是构成现场价值感的几个关键要素：

*   **肢体动作**：肢体动作需要做到松弛、放松、不紧张。我们看到电影《教父》中的角色，坐着时就显得非常沉稳、松弛。如果你的搭讪带有“屌丝感”，现场就很难产生价值。当然，你可以选择简单要个联系方式，后续在微信或朋友圈中塑造价值。但本节课讨论的是极简搭讪，如果肢体动作显得廉价，就不利于体现你的价值。
*   **表情**：表情紧张，或是初次搭讪时面无表情，都会传递出不良信息。西方人很擅长运用表情和肢体语言，做到声情并茂。例如，在说“hello，我觉得挺喜欢你”时，自信友好的微笑就是一种浅沟通。表情也能传递出自信和友好。
*   **语调**：语调需要下沉且坚定。举个例子，就能感受到语调的重要性。假设我是老师提问学生：“这题选什么？”有些学生犹豫地说“C……”，这带有疑问感，显得不确信。而有些学生坚定地说“选A”，则充满了确信感。**语调下沉且坚定，能传递出确信感**。
*   **临场反应**：在搭讪过程中可能会出现一些突发状况。例如，对方的母亲突然出现。这时不要慌张地做出低价值反射（比如赶紧跑掉）。通常情况下，可以大方地说：“hello，这是你母亲吗？”运用你的情商去处理，获得社交掌控感。
*   **语言与话题**：当你的肢体、表情、语调、临场反应都到位后，第五个要素就是语言。语言本身话题多样，例如年龄、工作、是否单身、来自哪里、口音等。但关键在于，你需要准备几个能展示自身价值的话题，才能做到吸引。你需要找到自己的价值所在。

例如，一个台湾人在成都，就天然具有互补价值（新鲜感）。他作为数学老师也是一个价值点（有正经工作）。互补价值能聊出很多花样，比如你去过妹子没去过的地方，或做过妹子没做过的事（前提是这些事本身有价值）。

你需要将价值转化为话题。因为约会本质上也在聊这些，即你要主导话题。如何做到高效？无非是让你的“法拉利”（比喻高价值）被女生知晓。

在价值塑造上可以做文章，甚至可以采用虚假价值去欺骗对方，这是一种策略。通过欺骗可能与女人发生关系，但我不建议这样做。因为这获得的只是生理快感。而通过真实价值与价值感的配合，你获得的是被女人认同的快感。

如果不涉及价值展示，所有的聊天都是无意义的。并非聊得越久越好，而是要聊到彼此感兴趣的价值点上，比如建立联系感，做一些有意义的事情。

## 话题的延续与表达

接下来是关于语言部分的话题该如何延续。很多人本身就不会聊天。

例如，对话可能是：“Allen，你多大？”“我30。”“好。”然后对话就终结了。

你要明白，不会聊天的人有一个特点：他们不擅长表达自己，不懂得对一个事物展开描述。

**平时练习时，可以尝试对一个普通物品进行描述**。例如：“这个杯子是个玻璃杯，但它好脏，你没刷吗？你看还有烟灰，这不是我喝的。”你需要多去表达。

再来一个例子：“你是做什么的？”“我是一个油管博主。但我之前还真不是做这个的，我之前是个数学老师。哎呀，我就不喜欢教数学，所以当时做了一个非常大的抉择，然后开始做这个。”多表达，提供更多信息，这样对方会觉得“你真的是数学老师吗？”，接话的点就多了。

高情商说白了就是：对方觉得跟你交流很舒服，因为你提供了很多信息，方便对方接话。

在我的Game体系中，搭讪、聊天、约会其实都在做同一件事：**展示价值的话题**。在约会中要做这件事，延续话题的能力在搭讪、约会、聊天中都是必需的。

为什么你不会聊天？因为你甚至不擅长跟任何人（包括男性）进行话题延续，你不擅长表达。你需要先做到擅长表达，再涉及话题的主导。

搞清楚话题延续后，我们要谈有效的部分。话题延续涉及上推、下切、平移等基本聊天方式，这里不展开讲，你可以在任何百科中自行查阅并搞清楚。这是最基础的部分。

此外，话题延续需要你的联想能力、观察能力以及关联能力。有时候不能太死板，你需要主导话题，将其引导到能展示自身价值的方向上。

我个人认为，男人本身是会撩妹的。我们要做的是唤醒你，而不是教你全新的东西。像言语部分的话题延续，你本身就会聊天，只是被压抑住了。我不相信你跟很熟的兄弟也不会聊天。只是跟熟悉的朋友你很会聊，跟陌生人不会。我认为这是唤醒你，让你意识到自己具备这种能力，而不是教你新的东西。

## 极简搭讪框架实操

现在，我们开始讲解正儿八经的搭讪框架。这个框架非常简单。

### 第一步：打开社交上钩点（极其重要）

你上去的时候需要“叫停”对方。有些搭讪课程要求45度角切入等复杂技巧，但不需要。你只需要在“打开”环节前，做好“叫停”。

你需要做到以下几点：
1.  **不要吓到对方**。我的方式是声音先过去：“hello你好。”对方听到声音会回头，这时再轻拍其肩膀。有些人上来就拍肩膀，会吓到人。换位思考，不要让对方认为你是一个不正常的人。避免这个问题的方法很多，完全不是一种固定的45度角。你只需要上去打个招呼，不要像不正常的人一样突然吓到对方。
2.  **练习叫停的方法**：可以通过问路或寻求帮助来练习。例如，向一位美女借一张卫生纸，这就是一次“叫停”。

“打开”的重要意义在于：让目标将你视作一个正常人对待。这十分关键，很多人失败就在这里。“打开”意味着对方认为你是一个正常人，而不是销售或健身房教练。

你可以做一个实验：明天去搭讪时，直接夸赞女生并表达意图，比如“我觉得你好漂亮”。女人百分之百会笑。她知道你的意图后，如果她不笑，那就是把你当成不正常的人了。想象一下，如果一个女生夸你帅，你会不会开心？你也会开心。如果不开心，顶多是因为对方带有其他目的性（比如推销）。所以，“打开”的重要性在于让对方以正常人的身份接纳你的（男女）意图。

**判断是否“打开”的指标**：通常，对方笑了是一个好指标。但有时不笑也不一定没打开，这需要经验判断。没打开的表现可能是：她不说话、很腼腆、紧张、左右环顾，感觉好像你在惩罚她一样。这时你可以告诉她：“你放心，我没有恶意，只是想认识你，不想给你压力。”通过共情让她理解你，因为有些女孩子确实社恐。礼貌地认识一个女孩子，没什么问题。

**社交的本质**：不在于你说了什么，而在于对方听懂了什么。你的重心应该是让对方听懂。有些人你说很多遍，对方还是“嗯嗯嗯”，反应依旧像对待陌生人。所以，你一定要有社交直觉，去判断对方是否把你当正常人，是否被“打开”。

### 第二步：表达意图

表达意图的方式有直接、间接和推拉式。
*   **直接表达**：“我比较喜欢你”或“我喜欢你这种类型”。这种方式直接，但可能会让你失去一些力量感。
*   **间接表达**：将“我喜欢你”换成称赞对方的某种特质。例如：“我觉得你的头发好漂亮，我比较喜欢这种（发型/发色）。”这是一种潜移默化植入男女前提的方式。
*   **推拉式开场**：例如：“hello你好，我刚刚在那边还在犹豫要不要过来，因为我刚下班比较累。但我觉得还是要过来搭讪一下，因为你看上去蛮善良的。”这就是先推开（犹豫），再拉近（还是过来了）。或者：“你看起来有点高冷，我还在犹豫要不要过来。但聊了几句发现好像还好，你还吓到我了。”推拉式开场可以制造张力。

在这里，表达意图的本质是让对方知道你的前提是“男对女”，即你是以撩她为目的，而不是单纯交朋友。这是升级关系的方向。因为Game的本质就是从零开始接触，将关系从0分升级到100分（关系达成）。这个前提必须明确。

**真正的拒绝**：是在你表达完意图之后才发生的。如果你连“打开”都没做到就被拒绝，那不叫真正的拒绝。

### 应对拒绝与破防

女生拒绝的原因可能有以下几种：
1.  **有对象（感情很深）**：可以尝试“破防”。例如，树立“渣男”人设去尝试突破，或者先以交朋友切入。对话示例：“你有对象了吗？”“有。”“没关系，我觉得……（询问年龄等）我个人觉得，多从不同人里找到合适的才重要。”尝试合作并突破，给对方重新定义框架。但如果对方感情很深，则很难突破。
2.  **把你当销售**：这本质上就是没“打开”，不是真正意义上的拒绝。
3.  **心情不好**：例如来大姨妈或被炒鱿鱼，当下状态不适合社交。可以共情：“感觉你心情不好？”
4.  **没看上你**：提升自己吧。
5.  **社交压力**：例如搭讪目标时，她的闺蜜被晾在一旁。这会给目标带来压力。对于多人组，你需要叫停所有人，进行话题延续，聊一些有价值的内容，并照顾好她的同伴，或者让你的僚机去和她的闺蜜聊天。

### 拉近关系与收号

在搭讪现场，话题延续最重要的就是聊有价值的内容，并主导话题。

我的搭讪是技术流与自然流的结合。你还需要寻求机会拉近关系。

**拉近关系要丝滑**。例如，新手可能只会生硬地说：“我们加个微信吧。”而更丝滑的拉近是：借助前面建立的联系感。例如，在大陆酒局应酬中，可能会说：“老胡，你多高？”“我183。”“我也180，那我们喝一个吧。”碰杯喝酒这个动作就是拉近，而前面聊身高就是铺垫和联系感。

**一个好的搭讪，一定在互动中有拉近**。迂回中贯穿始终的拉近。例如：“你单身吗？”“单身。”“那你觉得我怎么样？”“还好。”“那我觉得可以接触一下，加个微信呗。”

**不要把收号当成唯一目的**。收号只是一个结果，如果我们只盯着收号，就无法做出一次有效的搭讪。

### 兴趣指标（IOI）

线上聊天的兴趣指标包括：发“哈哈哈”、使用表情包、主动问你问题、夸赞你。当这些指标多次出现时，就意味着可以尝试邀约了。但这只是“可约”的信号，并非让你立刻去约。核心在于：对方已经是“你碗里的”，不要着急。着急是男人的通病，无法让你脱颖而出。但该升级关系时一定要果断。

线下则有很多具体指标，例如：撩头发、脚部左右移动、表现出开心情绪、主动好奇问你问题等。这些需要你积累经验去感知。因为社交本质上是两个人之间的事情，人跟人是有磁场的。

## 核心概念与总结

本节课我们一起学习了搭讪极简框架的下半部分，涵盖了从言语价值传递到具体操作步骤的全过程。

**核心框架回顾**：
1.  **叫停与打开**：礼貌接近，让对方将你视为正常人，接纳你的男女意图。
2.  **表达意图**：通过直接、间接或推拉方式，明确“男对女”的前提。

4.  **拉近关系**：在互动中自然、丝滑地推进关系。
5.  **识别与应对**：学会判断兴趣指标，并懂得如何应对不同类型的拒绝。

**重要概念强调**：
*   **社交本质**：`社交效果 = 对方接收并理解的信息`
*   **价值传递**：`现场吸引 = 肢体语言 + 表情管理 + 语调控制 + 临场反应 + 价值话题`
*   **关系前提**：`Game目标 = 建立并明确“男对女”的互动前提`
