# 雨宸19套合集：17.《雨宸讲九型人格》：002.助人型人格

## 概述
在本节课中，我们将学习助人型人格的特点、形成原因以及如何与助人型人格的人建立和谐的关系。

## 助人型人格的定义
**定义**：追求被爱，渴望被认可，认为只有通过帮助他人才能获得爱和肯定。

**公式**：付出 → 获得肯定 → 获得爱

## 助人型人格的主要特征
- **追求被爱**：渴望被他人爱和认可。
- **愿意付出**：乐于帮助他人，为他人付出。
- **忽略自己**：在满足他人需求的过程中，常常忽略自己的需求。
- **依赖他人**：需要他人的认可和理解来肯定自己。

## 助人型人格的形成原因
助人型人格的形成与原生家庭有关。在成长过程中，如果父母对个体冷漠，个体为了获得爱和关注，会通过帮助他人来获得认可。

## 与助人型人格建立和谐关系
### 1. 理解助人型人格
- 了解助人型人格的形成原因和特点。
- 理解他们的需求和期望。

### 2. 尊重他们的付出
- 感谢他们的付出和帮助。
- 不要轻易否定他们的努力。

### 3. 给予他们肯定和理解
- 在他们付出时给予肯定。
- 在他们遇到困难时给予理解和支持。

### 4. 帮助他们建立自信
- 鼓励他们表达自己的想法和需求。
- 帮助他们认识到自己的价值。

### 5. 避免指责和批评
- 在他们发脾气时，给予理解和包容。
- 不要轻易指责和批评他们。

## 总结
本节课中，我们学习了助人型人格的特点、形成原因以及如何与助人型人格的人建立和谐的关系。希望这些知识能帮助大家更好地理解和关爱身边的助人型人格的人。
