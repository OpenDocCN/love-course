## 第一部分：前期筛选策略 🔍

上一节我们介绍了课程的整体目标，本节中我们来看看关系推进的第一步：筛选。筛选的核心目的是在互动初期快速识别并排除不合适的对象，同时建立你的高价值地位。

### 何时进行筛选
*   筛选可以在互动热度处于 **55到65分** 时进行。
*   这个热度通常指双方已有初步联系，例如在社交软件上匹配后聊了**大约7天**。

### 筛选的核心目的
以下是进行筛选的四个主要目的：

1.  **排除不合适的对象，避免无效闲聊**：不合适的人包括：态度“随缘”、强调“喜欢对她好的”、自称“慢热”的，以及在聊天中让你感到不舒服或诡异的人。你可以通过提问测试，例如：“你平常也这么跟人聊天，还是今天心情不好？” 如果对方确认一贯如此，则应直接筛选掉。**记住，95%的女性都应在此阶段被筛选掉。**
2.  **建立高位姿态**：这比急于升级关系更重要。高位姿态意味着你拥有选择权和标准。
3.  **提高聊天效率**：直奔主题，不浪费时间。
4.  **植入个人标准和价值**：在互动中自然展现你的要求和优势，而不是进行无目的的闲聊。

### 筛选实战案例解析
以下是几个筛选话题的实战应用，请注意其共同模式：

*   **案例1**：

    *   女生回答后，你给予反馈，并陈述你自己的标准。
*   **案例2**：

    *   女生回答后，你给予反馈，并提供关于你自己的信息，接着再抛出你的筛选标准问题。
*   **案例3**：
    *   你提问：“本人和照片有区别吗？”
    *   女生回答后，你给予反馈，并可以顺势植入信息，例如：“我聊天还蛮直接大方的”，从而为后续邀约铺垫。
*   **案例4**：

    *   女生回答后，你给出你的标准，例如：“愿意为对的人降低标准吗？”

**这些案例的核心共性是：没有进行关系升温。** 所有对话都围绕三个目的展开：筛选女性、建立高位姿态、植入个人标准与价值。**前期的核心就是避免升温，专注于筛选。**

## 第二部分：强化男对女关系 🚀

在成功完成前期筛选并建立高位后，当互动热度提升时，我们就可以进入强化阶段。上一节我们明确了筛选的要点，本节中我们来看看如何主动强化男性主导地位。
