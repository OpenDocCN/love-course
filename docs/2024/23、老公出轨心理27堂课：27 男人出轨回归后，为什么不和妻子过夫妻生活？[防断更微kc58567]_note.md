**夫妻关系三大支柱**：
1. 物质支柱：物质基础是婚姻稳定的基础。
2. 夫妻运动支柱：夫妻运动有助于释放压力、建立安全感、亲密关系和满足需求。
3. 心理支柱：夫妻间心理上的和谐是婚姻长久的关键。

**婚外情生命周期**：
1. 萌芽期：婚外情的初期阶段。
2. 高速发展期：婚外情迅速发展，活动频繁。
3. 高峰体验：婚外情达到顶点，可能涉及离婚承诺。
4. 矛盾下降期：矛盾和冲突出现，可能导致分手。
5. 翘尾期：关系稳定，但可能存在潜在问题。
