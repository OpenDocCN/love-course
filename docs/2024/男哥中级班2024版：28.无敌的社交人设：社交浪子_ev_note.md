# 男哥中级班2024版：28. 无敌的社交人设：社交浪子 🌟

## 概述
在本节课中，我们将学习如何塑造一个高阶的社交人设——社交浪子。这种人设能够帮助你在社交场合中脱颖而出，吸引高阶女生。

## 快乐引领技巧
快乐引领是一种高阶技术，它从“情场浪子”这种人设中提炼而来。许多女生喜欢这种放荡不羁、爱自由的形象。

### 表现要点
1. **付出与快乐**：你的付出不是为了取悦对方，而是为了让自己感到快乐。
   - **公式**：付出 → 自我快乐 → 分享快乐

2. **即兴行为**：送礼物或对别人好，都是即兴的，为了让自己开心。
   - **代码**：即兴行为 → 自我开心 → 分享快乐

3. **分享与游戏**：分享对你是游戏，是快乐的事情。
   - **公式**：分享 → 游戏 → 快乐

## 坏坏的男生
坏坏的男生不会过分尊重女性，而是会像对待小孩一样，做一些有趣的事情。

### 行为表现
- 拍女生的头
- 拉着女生到处跑
- 把蛋糕涂在女生的鼻子上
- 搁置她
- 挠痒痒

## 浪子的特性
1. **仪式感**：简单的事情也能注重惊喜和仪式感。
   - **例子**：吃饭时点蜡烛，放轻音乐。

2. **社交热爱**：热爱社交，永不孤独。
   - **公式**：热爱社交 → 永不孤独

3. **抗打击能力**：面对拒绝时，保持阳光和乐观。
   - **代码**：拒绝 → 阳光 → 乐观

4. **好奇心**：对新鲜事物充满好奇心。
   - **例子**：对电影、游戏等感兴趣。

5. **正面评价**：对所有事物都给出正面积极的评价。
   - **公式**：正面评价 → 欣赏式反馈

6. **自嘲**：在别人打压你时，顺着他大胆自嘲。
   - **例子**：别人说你丑，你说自己丑得能吓唬小孩。

## 总结
本节课中，我们学习了如何塑造社交浪子这种高阶人设。通过快乐引领、坏坏的男生、仪式感、社交热爱、抗打击能力、好奇心、正面评价和自嘲等技巧，你可以成为一个更受欢迎的社交达人。
