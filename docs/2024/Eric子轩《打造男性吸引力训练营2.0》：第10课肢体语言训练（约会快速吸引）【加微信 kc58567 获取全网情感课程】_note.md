## 积极开放的肢体语言

**公式**：积极开放的肢体语言 = 自信的肢体语言 + 大的空间 + 舒展的姿势

**代码**：避免防御型姿势，如双手交叉、低头玩手机等。

## 站姿

**步骤**：
1. 摆正站姿，避免弯腰、低头看手机。
2. 深吸一口气，调整状态，挺胸收背。
3. 扬起下巴，环视四周，找到目标。
4. 避免低头看手机，以免给人留下不好的印象。
5. 不要害怕和对方目光接触，展现自信。

## 坐姿

**错误示范**：
1. 双手交叉，掩盖不安。
2. 双手环绕，显得紧张。
3. 双手交叉放在下面，像保镖一样。

**正确示范**：
1. 身体微微靠后，单手放在桌子上，另一只手可以放在腿上或插在兜里。
2. 翘二郎腿，但不要过于紧张。
3. 挺胸，不要塌背，下巴微微上扬，但不要给对方太大压力。

## 眼神

**要点**：
1. 不要直勾勾盯着对方，适当回应。
2. 利用挑眉等动作展现魅力。
3. 避免直勾勾盯着对方某个部位，以免显得不礼貌。
4. 保持自然微笑，融入习惯和生活。

## 手势

**要点**：
1. 利用手势讲故事，展现魅力。
2. 改变动作和语气，让说话更有趣、更有吸引力。
