# 清风老师合集：14链接贵人，让贵人帮你的实操方法论21期：196.第十一课：聊天雷区：聊天十大雷区（上）

## 概述
在本节课中，我们将学习如何避免在聊天中踩到雷区，从而在大佬面前建立良好的沟通和信任。

## 聊天十大雷区（上）

### 1. 过度询问
**公式**：避免在对方不舒适的情况下展开话题。

- **解释**：过度询问会让对方感到不舒服，不愿意继续沟通。

### 2. 自我为主
**公式**：放下姿态，接受他人的意见。

- **解释**：强调自己的正确性会阻碍学习和成长。

### 3. 虚伪包装
**公式**：真诚展示自己，避免装逼。

- **解释**：虚伪包装会破坏信任基础。

### 4. 贬低他人
**公式**：不要在大佬面前贬低他人。

- **解释**：贬低他人会显得格局低，不自量力。

### 5. 随意评价
**公式**：避免评价他人，专注于自己的事情。

- **解释**：随意评价会让人觉得格局小。

## 总结
本节课中我们一起学习了聊天十大雷区（上），了解了如何避免在聊天中踩到雷区，从而在大佬面前建立良好的沟通和信任。
