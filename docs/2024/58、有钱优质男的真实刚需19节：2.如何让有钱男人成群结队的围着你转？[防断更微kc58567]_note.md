# 58、有钱优质男的真实刚需19节：2.如何让有钱男人成群结队的围着你转？

大家好，本节课我们将探讨如何吸引有钱男人围绕你转。以下内容将详细介绍如何拓展优质异性的社交渠道。

## 1. 拓展优质异性的渠道

首先，从你现有的社交关系盘开始挖掘，即你现实生活中的基本关系盘。这包括家人、亲戚、同学、老师、同事、朋友等。这些亲近的群体是你想要高价乃至向上社交的基本发力点。

## 2. 五大关系盘

以下是五大关系盘：

### 2.1 血缘关系

在直系亲属和远亲中寻找混得好的亲戚，平时多走动，留下好印象，并表达出单身择偶的需求。

### 2.2 校友关系

包括同学关系、师门关系、师生关系等。校园恋爱相比社会人更纯真，更容易培养真正的感情。

### 2.3 工作关系

工作关系不仅仅是你同单位的同事，而是你整个领域圈子里的人脉。参加行业社群、峰会等活动，认识更多优质异性。

### 2.4 朋友关系

从校友关系、工作关系、业余生活等渠道发展成为朋友关系的人。让已婚或找对象的姐妹帮你介绍。

### 2.5 兴趣关系

参加有钱人喜欢的兴趣爱好活动，如高尔夫、滑雪、高端健身房等。进入社群，认识更多优质人脉。

## 3. 高端消费筛选

通过高端消费筛选出有钱人，如乘坐头等舱、入住五星级酒店等。同时，也要有相应的消费能力。

## 4. 总结
