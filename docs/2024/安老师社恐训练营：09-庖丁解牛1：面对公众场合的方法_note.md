# 安老师社恐训练营：09-庖丁解牛1：面对公众场合的方法

同学们大家好，从这节课开始，我们将学习四节课，主题是面对紧张、焦虑和恐惧场景的从容应对方法。如果之前课程内容已掌握，那么这四节课对你们来说将轻松易学。

## 公众场合概述

公众场合包括上台表演、当众讲话、会议发言、面试和当众叫人等场景。以下将分别介绍应对这些场景的方法。

## 上台表演

**方法**：

1. 提前练习：熟悉稿子，对着镜子练习，想象观众，调整动作。
2. 穿着得体：提升自信心。
3. 放松动作：深呼吸、转移注意力、自信动作。
4. 开场破冰：活跃气氛。
5. 微笑：给人亲切感。
6. 专注内容：减轻紧张感。

**核心概念**：

- 提前练习：**提前准备，熟悉内容，增强自信**。
- 放松动作：**深呼吸、转移注意力，减轻紧张**。

## 当众讲话

**方法**：

1. 将听众视为小宠物：放松心态。
2. 不维护人设：适应改变。
3. 积累话术：准备常用话术，如敬酒话术、自我介绍模板。
4. 关注他人：关注他人回复和内容。

**核心概念**：

- 将听众视为小宠物：**放松心态，减轻紧张**。
- 积累话术：**准备常用话术，提高应对能力**。

## 会议发言

**方法**：

1. 关注会议内容：关注他人表达和讲话者表现。
2. 转移注意力：练习转移注意力，减轻紧张。
3. 举止得体：注意手部动作，避免自我关注。
4. 过渡和铺垫：感谢他人发言，表达观点。
5. 语速适中：保持讲话节奏。
6. 逻辑清晰：表达观点时，有铺垫、有观点、有原因、有总结。

**核心概念**：

- 关注会议内容：**关注他人表达，减轻紧张**。
- 逻辑清晰：**表达观点时，有铺垫、有观点、有原因、有总结**。

## 面试

**方法**：

1. 充分准备：了解面试官可能提出的问题，做好核心准备。
2. 模拟练习：让朋友提问，进行模拟练习。
3. 语速适中：避免紧张导致语速过快。
4. 眼神交流：避免闪躲，展现自信。
5. 思路清晰：表达观点时，有逻辑性，避免结巴。

**核心概念**：

- 充分准备：**了解面试官可能提出的问题，做好核心准备**。
- 思路清晰：**表达观点时，有逻辑性，避免结巴**。

## 当众叫人

**方法**：

1. 多练习：克服心理障碍，勇敢尝试。
2. 不怕出丑：不怕丢人，勇敢迈出第一步。
3. 不在乎他人眼光：关注自身需求，勇敢表达。
4. 多尝试：多锻炼，提高应对能力。

**核心概念**：

- 多练习：**克服心理障碍，提高应对能力**。
- 不在乎他人眼光：**关注自身需求，勇敢表达**。

## 总结

本节课中，我们学习了面对公众场合的方法。通过提前准备、积累话术、关注他人、转移注意力等技巧，我们可以更好地应对紧张、焦虑和恐惧场景。希望大家能够将这些方法应用到实际生活中，提高自己的社交能力。
