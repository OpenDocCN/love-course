## 概述
在本节课中，我们将学习如何通过讲故事来掌控谈话主动权，并有效地说服对方。

## 讲故事的技能培养
讲故事是一个技能的培养，需要以下能力：
- 思维缜密能力
- 日常生活积累能力
- 头脑灵敏能力
- 语言表达能力

## 故事的重要性
一个优秀的学者未必有很强的讲故事能力，但身经百战的现实生存者往往具有丰富的感染力。这是因为他们的生活中充满了冲突和冲撞，而优秀的学者则充满平和感。

## 故事的架构
一个好的故事架构应该包含以下元素：
- 冲突：产生行为
- 行为：产生情感连带
- 情感连带：同理心

## 如何讲故事
1. **挖掘对方最在意的**：了解对方渴望的行为和事件，并将其描述出来。
2. **内心中有一个结论**：在交流过程中，清晰地表达自己的结论。
3. **运用情感**：在谈话中，不占完全的弱势，运用情感牌。
4. **拿出现实数据和案例**：用数据和案例来支持自己的观点。

## 如何用故事化结构说服对方
1. **挖掘对方最在意的**：了解对方的需求，并将其描述出来。
2. **内心中有一个结论**：明确自己的结论，并找到角度表达。
3. **运用情感**：在谈话中，不占完全的弱势，运用情感牌。
4. **拿出现实数据和案例**：用数据和案例来支持自己的观点。

## 如何用故事化结构管理他人
1. **了解你要教育和管理的人**：了解对方的性格、需求等。
2. **带入你的立场**：告诉对方你可以决定多少东西。
3. **讲出对方的迷茫、焦虑和不服气的点**：让对方觉得你了解他。
4. **通过故事表达结论**：用故事来表达自己的结论。

## 刻意练习
1. **看到好的内容，分析其结构**。
2. **多感知环境，丰富自己的经历**。
3. **在工作中练习用故事化形式表达**。
4. **用笔复盘，逐渐脱稿**。

## 总结
本节课中我们一起学习了如何通过讲故事来掌控谈话主动权，并有效地说服对方。这是一个非常重要的技能，需要我们进行刻意练习。
