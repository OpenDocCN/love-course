## 搭讪流程
以下是搭讪的五个流程：

1. **开场白**：通过拍肩等动作表达坚定感，避免因动作不当被女生pass。
2. **意图表达**：直接表达想要认识对方的意图，随着经验提升，逐渐弱化意图表达。
3. **解决问题**：面对女生可能提出的问题，如被当作销售、有事、有男朋友或价值感不足，需保持自信并具体解决问题。
4. **信息互换**：通过基本话题进行对话，并运用冷读技巧增加趣味性。
5. **收尾**：在聊天达到高潮时进行收尾，如邀约咖啡等。

## 详细内容

### 开场白
- **坚定感**：通过眼神、声音和触碰表达坚定感，避免因动作不当被女生pass。
- **拍肩**：拍肩是开场白中非常好用的一招，但需注意不要过于直接。

### 意图表达
- **直接表达**：直接表达想要认识对方的意图。
- **经验提升**：随着经验提升，逐渐弱化意图表达。

### 解决问题
- **四大类问题**：女生把你当做销售、女生很急有事、女生有男朋友、价值感不足。
- **动力大于阻力**：保持自信，植入幽默，让女生感受到你的价值。
- **具体问题具体分析**：针对不同问题采取不同解决方法。

### 信息互换
- **基本话题**：年龄、职业、住址、家乡、现在在干嘛、有没有被搭讪。
- **冷读技巧**：通过矛盾点增加趣味性，如年龄、职业、性格等。

### 收尾
- **number close**：在聊天达到高潮时进行收尾，如邀约咖啡等。
