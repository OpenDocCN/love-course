# 任亚《人间清醒后半生》：第02节01女性容貌颜值时代，如何应对容貌焦虑

🎼大家好，我是人雅，欢迎来到我的课堂。这节课我想和大家分享的是研制时代如何应对容貌焦虑。

## 1. 回顾过去，接受现在

**以下是关于容貌焦虑的回顾：**

- 15年前，我在上海外滩酒店的客厅里进行了人生第一次直播，当时并不觉得自己好看。
- 9年后，回看视频，觉得那时的自己很漂亮。
- 我从小就不爱拍照，喜欢拍风景，但有时会感慨岁月在身上留下的痕迹。
- 几年前，体重增加，容貌和身材都发生了变化。

## 2. 接受衰老，追求优雅

**以下是关于衰老和优雅的思考：**

- 衰老是生理现象，无法避免。
- 优雅是年龄的专利，真正的优雅是不紧张的，是放松的。
- 高级的松弛感是所有欲望都被满足过后的感觉。
- 当纠结于容貌时，会破坏优雅和松弛感。

## 3. 自信源于内在成长

**以下是关于自信和内在成长的思考：**

- 好看的皮囊千篇一律，有趣的灵魂万里飘移。
- 与其在外貌上下功夫，不如在内在下功夫。
- 通过学习成长，让自己精神上更富足，情感上更疗愈他人，生长得更好。
- 让自己的灵魂更有趣，更有智慧。

## 4. 不强求改变，珍惜当下

**以下是关于改变和珍惜当下的思考：**

- 衰老和变胖是不可改变的，要接纳它们。
- 改变可以改变的，比如通过自律的生活、生物科技等方法变得年轻。
- 不要过于执着于容貌，否则会与追求的方向背道而驰。
- 珍惜当下，活出自己。

## 5. 自由与价值观

**以下是关于自由和价值观的思考：**

- 自由是把情绪按钮掌握在自己手里，不受他人影响。
- 价值观是我们的价值优先权，要把时间和金钱花在最重要的事情上。
- 如果不喜欢我，就远离我；如果喜欢我，就在我这儿好好刷钱。

## 总结

本节课中我们一起学习了如何应对容貌焦虑。我们要接受衰老，追求优雅；自信源于内在成长；不强求改变，珍惜当下；坚持自己的价值观，活出自己。
