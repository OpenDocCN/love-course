# 玲金玉（五套合集）：1.亲密关系技巧 56节：5.爬楼梯

## 概述
在本节课中，我们将学习亲密关系技巧中的“爬楼梯”动作，这是一个由两个动作组成的技巧。

## 动作分解

### 动作一：环绕抓握
**以下是动作一的详细步骤：**
1. 用手做OK手势，环绕并抓住男性的生殖器。
2. 交替向上移动，力度可轻可重，根据对方承受能力调整。

**公式描述：**
\[ \text{环绕抓握} = \text{OK手势} + \text{环绕} + \text{向上移动} + \text{力度调整} \]

### 动作二：手掌抓握与打圈
**以下是动作二的详细步骤：**
1. 用整个手掌抓握生殖器。
2. 右手抓握冠状沟，开始打圈。

**公式描述：**
\[ \text{手掌抓握与打圈} = \text{手掌抓握} + \text{右手抓握冠状沟} + \text{打圈} \]

## 总结
本节课中，我们一起学习了“爬楼梯”动作，这是一个由两个动作组成的亲密关系技巧。通过环绕抓握和手掌抓握与打圈，可以增加双方的互动和愉悦感。
