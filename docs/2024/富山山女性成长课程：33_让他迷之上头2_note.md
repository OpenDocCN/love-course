# 富山山女性成长课程：33_让他迷之上头2 💬

在本节课中，我们将要学习高情商接话的核心技巧。我们将探讨在对话中如何保持镇定、如何尊重不同观点以及如何保持自我，从而在交流中展现魅力与价值。

## 一、 保持镇定，切勿抢话 🧘‍♀️

上一节我们介绍了高情商对话的重要性，本节中我们来看看第一个关键点：不要着急接话。着急会显得局促，显得不淡定、不镇定。一个女孩子走出去，要能够压得住场。

压得住场的关键在于懂得慢几秒钟，懂得放慢语速，放慢所有行动和眼神。甚至眼皮的移动也要放慢，那种威严感会自然流露。因此，在接话过程中，当对方尚未阐述清楚观点时，不要着急忙慌地去抢话。

抢话在别人看来非常讨厌，显得迫不及待要表达自己观点，不懂得忍住、hold住。应该在脑子里多转一下，多想一下。

**核心操作**：对方说完后，**停顿2-3秒**，再从容回应。

他说的这个观点，我是否能够提出更高维度、更有认知的新方式给到他。在与高阶男生相处时，常常打断别人的阐述是非常掉价的行为。这证明文化程度、个人修养、家教等都不行。这一点必须注意。

## 二、 避免反驳，尊重差异 🌍

接下来我们探讨第二个要点。在与人聊天接话的过程中，不要去反驳对方。无论他是对是错，都不要反驳。这个世界没有那么多对错，你们站的角度不同，价值观不同，所理解和看到的事物就不一样。

小时候大家都学过一篇语文课文叫《画杨桃》。一个同学面对杨桃的正面，另一个同学面对杨桃的横切面，他们画出来的不一样。那个同学站的角度不同，他画的杨桃是一个五角星。书本上早已教过这些最基本的看待事物的方式，为什么很多人没有学会？

首先，你要允许这个世界上有不同的声音存在，有不同的价值观。不要认为你坚信的东西就是对的。当别人与我的价值观非常冲突时，我从来不会反驳。我会跟他讲：“可能你的是对的。我们两个角度不太一样，我可能有一些别的看法。”这些都是很温和的方式。

以下是展开对话的正确示范：
*   “你为什么会有这么新颖的角度呢？”
*   “我上次看一个访谈，他也说了一个类似的观点。”
*   “我想知道一下你们的心理动态是什么？你为什么会这么觉得呢？”

你要敞开心扉地去问别人，是什么造就了今天的他，他为什么会得出这样的观点，而不是去反驳、去指正。

很多女性在跟男生交往过程中常常去指正对方。例如，我有一个朋友，她老公每次刷完牙后，都会用牙刷去涮杯子。她觉得这不卫生，因为她自己是直接对着水龙头冲洗口腔的。她一直想要指正和纠正老公的这个点，不断地说：“你得改，这个不卫生。”

其实我自己也是直接用水龙头冲嘴的。但恰巧我也碰到了这样子的另一半，我从来不会跟他说“这是不对的，你得改”。我认为没有必要，这些都是边缘化的信息。我只需要每一次把他的漱口缸子洗得特别干净就可以了。

不需要改变别人，不需要强迫别人跟你有同样的观点。不需要在聊天时指正别人的价值观、审美和想法。这个世界可以有601种不同的声音，你只需要倾听。相反，像我们这种不去指正，只知道倾听和深挖对方内心动态的人，可以得到更加宽广的视角。

## 三、 保持自我，切忌卑微讨好 👑

在高情商的接话中，下一个不要犯的错误，就是显得过于随和、卑微、讨好。很多人别人刚讲完什么，他马上“是是是”、“嗯嗯嗯”、“对对对”、“赞同赞同”。人家会觉得你这个人没有自己的想法吗？你这是在讨好吗？

我给你们讲一个我自己的例子。有一次在欧洲，一个追我的男生带我去一间牛排店。牛排上了，我们各自吃了一块之后，他问我：“你觉得这个牛排怎么样？”男生在交往过程中，会试探你的消费层级、对世界各方面事情的认知，他会看你的天花板是什么。

所以，我非常清楚他这个问题是试探性的。那是一家非常棒、非常高档的熟成牛排店。他问我这家店怎么样、东西做得怎么样。为什么我觉得是试探？我能从他的口吻和表情中看出，他认为这个牛肉做得也不怎么样，也不是高级的、贵的店就好。

这时，我回了他一句话。我说：“其实你选的地方是很棒的。但是我觉得如果这个牛肉在……（如何烹饪）的话会更好一点。其实呢，我吃过一间店，虽然没有那么贵，但肉质比这个更好。如果你有兴趣的话，下次我可以带你去吃。”

这样的接话既没有让他觉得很丢面子，也没有显得我浑浑沌沌混过去、没有自己的想法、没见过、没吃过，让人家给看出来了。这就是一个比较不错的接话方式。

**核心公式**：**肯定对方选择 + 委婉提出个人见解 + 提供建设性替代方案**

所以，你们在面对很多你们认为根本不是那么回事的问题时，也需要有自己的态度，不要一味卑微地去讨好、去迎合。很多女生会觉得“挺好的，挺好的”、“好吃呀好吃呀”，这有什么意义呢？你在这个接话过程中能体现你什么呢？

为什么很多女生可以让男生一步一步很佩服她、很欣赏她？其实就是通过这些聊天、相处的细节探索出来的，觉得你是一本读不完的书，觉得你是有自己想法的。所以，没有想法、一味附和和卑微的女生是很低价值的。希望你们不要那么做。

在接话过程中，我们既要有趣，又要显得自己不卑不亢。
