引导女生投资是让她倒追你的核心。这需要避免两个极端：既不过度追逐，也不过度疏远。我们将通过三个要点来掌握这一方法。

## 不要追逐女生

上一节我们概述了核心概念，本节中我们来看看第一个要点：避免主动追逐。

## 不要过度疏远

上一节我们强调了避免追逐，本节中我们来看看第二个要点：防止关系因过度疏远而断裂。
