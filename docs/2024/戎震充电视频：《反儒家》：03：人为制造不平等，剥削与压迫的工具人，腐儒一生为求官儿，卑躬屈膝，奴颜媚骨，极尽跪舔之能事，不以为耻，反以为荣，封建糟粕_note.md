# 《反儒家》教程：03 儒家与不平等

大家好，欢迎来到《反儒家》教程的第三课。今天我们将探讨儒家思想的历史局限性及其代价。

## 一、儒家与科举制度

**以下是关于科举制度的列表：**

- **科举制度不普遍**：科举制度并非面向所有平民，而是主要服务于有钱有势的阶层。
- **教育成本高昂**：读书需要缴纳学费，普通家庭难以承担。
- **脱离生产**：读书人需脱离生产劳动，专注于学习，导致社会生产力下降。

## 二、儒家与阶级固化

**以下是关于儒家与阶级固化的列表：**

- **小人君子之分**：儒家将人分为“小人”和“君子”，其中“小人”指普通劳动者，“君子”指有知识、有文化的人。
- **阶级歧视**：儒家思想中存在明显的阶级歧视，看不起普通劳动者。
- **阻碍社会进步**：阶级固化阻碍了社会进步和阶级流动。

## 三、儒家与封建秩序

**以下是关于儒家与封建秩序的列表：**

- **维护封建统治**：儒家思想旨在维护封建统治阶级的利益，巩固其统治地位。
- **等级制度**：儒家思想强调等级制度，如男尊女卑、长幼尊卑等。
- **阻碍社会发展**：封建秩序阻碍了社会发展和进步。

## 四、儒家与现代社会

**以下是关于儒家与现代社会的列表：**

- **与现代价值观格格不入**：儒家思想与现代平等、自由的价值观格格不入。
- **阻碍社会发展**：儒家思想阻碍了社会发展和进步。
- **反动思想**：儒家思想具有反动性质，应予以批判。

## 总结

本节课中，我们学习了儒家思想的历史局限性及其代价。儒家思想具有阶级固化、维护封建统治等弊端，与现代社会的价值观格格不入。因此，我们应该批判儒家思想，推动社会进步。
