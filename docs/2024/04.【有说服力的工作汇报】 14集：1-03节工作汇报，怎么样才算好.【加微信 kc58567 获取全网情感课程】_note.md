# 课程04：有说服力的工作汇报 - 第03节：如何才算好的工作汇报 📊

## 概述
在本节课中，我们将要学习如何判断一个工作汇报是否优秀。我们将探讨汇报好坏由谁评判，并深入分析优秀工作汇报应具备的三个核心特点。

上一节我们解决了“工作汇报到底在汇报什么”的问题，并介绍了工作汇报的五种重点内容。本节中，我们来看看另一个关键问题：工作汇报怎么才算汇报好。

要解决这个问题，我们必须先知道工作汇报的好坏到底由谁来判断。答案很简单，就是听汇报的人来判断。这个答案往往越简单，越容易被忽略。很多人在准备工作汇报时，会忘记站在汇报对象的角度去思考问题。

如果你在各种会议上听过领导或汇报对象说过类似的话，例如“这情况我怎么知道？”、“你挑重点说吧”、“所以你需要我做什么？”，很可能就是因为汇报人忘了站在对方的角度思考。如果在一次汇报里频繁听到这些话，那么这个汇报肯定算不上是好的汇报。

## 优秀工作汇报的三个特点
说了这么多，听汇报的人到底想听到什么样的汇报？怎么汇报才能减少这些问题呢？根据过去的经验，我发现好的工作汇报应该就像一则促销广告。

为什么这么说呢？上节课我们提到，工作汇报的核心目的是推动决策。从这个角度讲，工作汇报和促销广告非常相似。无论是什么样的促销广告，最终目的只有一个，就是推动消费者做出购买决策。大白话说就是“买它”。

许多优秀的促销广告都是站在消费者的角度思考的。如果不这样做，就会出现一个有趣的情况：消费者会对广告提出和领导在会上提的一模一样的问题。例如“这信息我怎么不知道？”、“有促销活动你早点说啊”、“这么一大堆，你到底哪个最优惠啊？”、“所以你想不想让我买啊？”。

接下来，我们就看看好的工作汇报应该是什么样的。概括下来，它有三个特点：**主动及时**、**简洁清晰**、**便于决策**。

### 特点一：主动及时 🕒
首先，我们来说第一个特点：主动及时。正常情况下，你不太愿意去主动看促销广告。但如果你买东西时，商家没有主动及时地告诉你有促销活动，而是直到你问了，或者事后才告诉你其实有打折，你心里反而会不舒服，因为会担心错过了最低价。

工作汇报也是类似的。例如，我曾遇到一位同事（假设是小朱）。他刚到公司就接手了一个项目，工作特别忙，对流程也不熟悉，所以就一门心思埋头苦干，总想着只要手上不停，肯定能赶上项目进度。他打算等工作做完时，再把项目结果漂漂亮亮地汇报给领导，中间就不耽误时间了。

结果过了一段时间，领导觉得小朱这边一直没动静，就问了他一下项目的情况。这一问不要紧，后来发现小朱手上的工作方向彻底跑偏了，而且这时离项目结束已经没有多少时间了。搞得整个团队陪着他加班加点，最终项目还是延期了。

所以，汇报一定要主动及时。如果等着领导问你工作情况你再汇报，多半情况下，领导那时心里已经有点不舒服，甚至已经耽误事了。领导最怕的，就是该了解的进度没了解，该发现的问题没发现，导致该有的决策和判断没有及时做出来。

那么，在工作的什么阶段汇报才算是主动及时呢？有以下五种一定要汇报的时刻。

以下是五种需要主动汇报的关键时刻：
1.  **工作准备开始时**：这时要向领导汇报你的工作准备得怎么样了，看看领导还有什么补充建议或执行中需要注意的事情。例如，很多项目性工作在开始执行前会有一个启动会或通气会，同步项目的准备情况和执行计划。
2.  **工作产生阶段性成果时**：这时汇报是为了让领导知道工作方向是正确的，一切都在有条不紊地推进。如果有跨部门协同，阶段性的成果也能及时让其他部门用上。例如，你正在执行一个市场推广方案，所有宣传渠道都沟通了一遍，这时就可以进行一次汇报，反馈沟通结果，看看接下来有什么需要重点跟进或团队配合的工作。不要企图攒个大成果一股脑汇报给领导，因为在你看来是惊喜，在领导看来可能是个惊吓。
3.  **工作遇到问题时**：这个问题无需过多解释。因为问题是否会导致工作结果延误、是否会产生连锁问题，都需要及时让领导知道，以便尽早应对和解决。
4.  **产生创意方案时**：有了好的想法，特别是合理的改进优化建议，及时向领导汇报，可能帮助整个团队提高工作效率。
5.  **工作为长期持续性任务时**：比如持续跟进某个平台上的销售数据。这类工作可能你和领导都很熟悉了，日常推进即可。但并不是说日常推进就可以绝口不提，还是要定期汇报，让领导知道一切正常，因为“照常进行”本身也是一种决策。

### 特点二：简洁清晰 ✨
说完了主动及时，我们来说一下好的工作汇报的第二个特点：简洁清晰。

很多人在工作汇报时，就怕自己说得不够全面、有遗漏，于是不管讲什么都说一堆细枝末节。本来一句话就能说明白的事儿，非要说上三五句，甚至三五十句。或者思维极具跳跃性，在好几个问题上反复横跳，让听汇报的人越听越头疼。

例如，我就听过这样一段汇报：“大家好，我想说一下我负责的那个项目的情况啊。前几天咱们那个客户沟通的时候呢，客户说对咱们这个方案挺满意的啊。哎那个方案不全是我做的啊，说起来还得感谢小李，他那边有个设计工具还挺好用的。原来咱们那个工具啊有点问题。我觉得这个工具应该更新一下。不过这个一会再说吧还是那客户那客户不那客户那个那个方案方案方案接下来该做宣传材料了……”这个时候，我相信在场所有人心里都在默念：能不能简单点，清晰点儿。

所以说，在做工作汇报时，要对内容进行整理和归纳。领导最关心的是什么内容？你最希望向他传递的是什么信息？就用最清晰的方式呈现出来，不然对方很容易对你的汇报失去耐心。

那么，具体怎么做汇报能够做到简洁清晰呢？我建议你做到以下四点。

以下是实现简洁清晰的四个技巧：
1.  **结果量化**：工作完成的情况，能用数字呈现的就用数字。说“这项工作完成了60%”，就比说“这项工作推进得还行”要清晰得多。这是让你的表达更清晰的首要方式。
2.  **结论先行**：订单签了还是没签？问题解决了还是没解决？你认为A方案更好还是B方案更好？先把结论说出来。千万别绕半天圈子，说了好多“我们做了这个、我们做了那个”，最后才告诉领导说“客户不同意”，那就浪费了领导的时间，最后还给个惊吓。
3.  **分条叙述**：一条一条地把信息进行分类，列出1、2、3，不要一大堆内容混在一起说。如果你说的时候是一团信息，那么听的人的脑子里也是一团，他很难把你的一团信息分解成一条一条的。
4.  **重复重点**：很简单，重要的事情多说几遍。一个信息你重复了好几遍，领导自然就知道这个信息很重要了。

### 特点三：便于决策 🤔
最后，我们来说一下好的工作汇报的第三个特点：便于决策。

那么，怎么汇报就便于决策了呢？很简单：让领导做判断题或者选择题，而不是问答题。

我见过一个做市场营销的员工，他汇报工作时是这么说的：“领导，您上次让我调研了几个宣传平台，我调研完了哈，他们大致的情况是这样的……（说了一堆）。那么接下来您看一下咱们这个方案该怎么做呀？哦还有，咱们合作的一个平台不给咱们推广了，您觉得怎么办呢？”后来听说这个营销方案确实做出来了，平台不给推广的问题也解决了，但这个员工也被老板解决了，因为老板实在不想再帮他回答问题了。

听懂了吗？所以在汇报工作时，最好是让领导做这样的决策：“行”或者“不行”；选“供应商A”还是选“供应商B”。别忘了，汇报的核心目的是推动决策。领导做决策越容易，他就会觉得你这个汇报做得越好。

## 总结
本节课中，我们一起学习了如何才算一个好的工作汇报。

我们讲到，好的工作汇报应该像一个促销广告。它越有利于你的听众（汇报对象）产生决策，它就越是一个好的汇报。

好的汇报有三个特点，分别是：**主动及时**、**简洁清晰**和**便于决策**。

*   **主动及时**：就是在五种时刻一定要汇报，分别是工作准备开始时、有阶段性成果时、遇到问题时、产生创意方案时，以及面对持续性工作时。
*   **简洁清晰**：有四个技巧，分别是结果量化、结论先行、分条叙述以及重复重点。
*   **便于决策**：让领导便于决策的方法，就是让领导做判断题或者选择题，而不是问答题。
