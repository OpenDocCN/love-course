# 雨宸19套合集：02.《好好吵架，好好爱》：001.让你明明白白地吵架，稳稳当当地恋爱_note

在本节课中，我们将要学习如何将吵架从关系的破坏者转变为关系的建设者。我们将探讨吵架的积极意义，并理解为什么“好好吵架”是维持健康亲密关系的关键技能。

有人说，检验爱情的唯一标准不是爱得有多深，而是吵架吵得好不好。吵架本身是可以发生的，但关键在于要明明白白地吵，要能解决问题。吵架后的情侣关系应该感觉更好。

双方的成熟度，以及对于人际关系的驾驭能力，也会因此增强。从某种意义上来说，学会正确地吵架，恰当地表达自己并影响他人，是一种必备的社会技能。原因很简单：在平时没有冲突的时候，大家维持着“你好我好大家好”的状态。

这种状态下，每个人都带着一个“人设”。这个人设多少有些不够真实。在吵架的时候，我们才会把最真实的状态展现给对方。这样，彼此才能了解对方真实的样子。我也因此知道该如何与你相处。

在吵架的过程中，我们是在培养情侣双方共同解决问题的能力和习惯。很多聪明人会说，对方总跟我吵架，每次都让我忍让。每次听到这个问题，我都特别头疼。你知道为什么吗？

**忍让等于逃避**。我们真正在吵架时，目标不是谁忍让谁，而是两个人在争吵的过程中找到问题，并共同去解决它。吵架肯定是因为存在问题。知道了问题在哪，我们才知道该如何解决。正如我常说的，恋爱是什么？

**恋爱的本质是在维持自我和做出妥协之间找到一个平衡点**。这个平衡点如何找到？就是通过吵架找到的。为什么很多情侣越吵越好？因为每一次吵架，大家都能找到共同点。

大家都能更深刻地认识彼此，都能更深刻地培养共同解决问题和面对危机的能力。培养这种面对危机和解决问题的能力，就是在培养什么？是在培养坚持和维持这段感情的信念。我因此有了维持这段感情的勇气和信心。

但是，很多情侣中女生吵得特别凶，男生就不停地忍让。你觉得这样挺好吗？这根本没有培养出任何能力，只是一方在发泄，一方在忍让。这种忍让其实就是逃避。发泄的一方有一种心态：我把情绪发泄给你了，你来解决问题。

把问题发泄给我了，我却选择逃避不去解决。这样培养不出任何东西。当忍耐的人忍让不下去，想吵的人不想吵了，这段感情就结束了。因为他们在相处的过程中，根本没有培养出维护一段感情的信心和习惯。

所以说，吵架是非常重要的。因此，培养深层的社会技能，要从“好好吵架”开始。

我们这套课程的内容一共有九节，涵盖了常见的错误吵架方式、心态建设、认知提升、思维方式升级以及方法技巧的分享。听完之后，对于你的吵架技能、处理矛盾的段位，甚至是适应社会生活的能力，一定会有一个质的提升。

不要害怕吵架，大胆地去吵，并且还要好好吵，吵得爽。

本节课中，我们一起学习了吵架在亲密关系中的建设性意义。我们明白了忍让等于逃避，而健康吵架的目标是共同发现问题并解决问题。恋爱的本质是寻找平衡，而吵架正是找到这个平衡点的关键过程。在接下来的课程中，我们将深入探讨具体的方法和技巧。
