# 课程20：有失去，才有新的自我 🍂

在本节课中，我们将探讨如何面对人生中不可避免的“失去”。我们将理解失去的本质，并学习如何从失去的经历中获得成长，构建新的自我。

---

前几节课我们介绍了三种重要的失去：目标的失去、身份的失去和关系的失去。本节中，我们来看看如何面对“失去”本身。

人的一生是由什么构成的？

心理学家朱迪斯·维奥斯特在她的著作《必要的丧失》中写道：**人的成长之路由放弃铺就而成。终其一生，我们都通过放弃成长着。** 她还写道，我们对生命核心的理解，就是我们对丧失的理解。我们成为什么样的人，以及我们过着哪种生活，都是由我们失去的经历决定的，无论这种经历是好是坏。

自我转变的过程，其实也是一个不断失去的过程。

也许你会问，面对失去，有什么办法吗？实际上，并没有特别的方法。对于真正重要的东西，寻找一个万能的方法太过轻巧。**承认没有办法，就是承认失去的东西的重量。**

以下是面对失去时，可能出现的自然反应：
*   如果你的失去重到需要你痛哭一场，那痛哭就是最好的办法。
*   如果你的失去让你一直抱有遗憾，那遗憾就是最好的办法。
*   如果你的失去让你黯然神伤，那黯然神伤的失落就是最好的办法。

有时候，我们需要这些与失去对应的痛苦，来确认失去对我们有多重要。这些痛苦会好吗？我想它还是会好的。人的心灵有自我痊愈的能力。如果真要说面对失去有什么办法，它所依靠的正是人这种自我痊愈的能力。但大概率，它会在你身上留下印记。

就像一个战士会留下伤疤，这些伤疤并不意味着我们不完整了。恰恰相反，它是我们走向完整的标志。**人不是瓷器，完美无瑕，一遇到打击就会碎成碎片。人是老树，经历风雨，能把失去的枝丫变成成长的养料。**

在讲关系失去时，我曾说：好的分手，是这段关系活在你心里。关系里美好的部分变成了你的回忆，不好的部分变成了你的经验。而坏的分手，是你还活在这段关系里。关系里美好的部分变成了你不肯放手的执念，不好的部分变成了你不敢重新开始的创伤。

这段话不只适用于关系的失去，它适用于所有的失去。**好的失去会让我们学到重要的东西，它变成了新自我成长的基础。而坏的失去，则会让你以为所有的投入和期待都是错误，因为害怕再次经历这种痛苦，你不敢再去投入。**

虽然我们前面的课程里讲了三种失去，但我觉得它们本质上讲的是同一种失去：**远离人群和关系，去寻找新自我的失去。** 失去一个目标，是我们被所向往的群体拒绝的那种失去。这些失去都让我们在某种意义上离开人群，去独自面对生活的残酷，在孤独中寻找和确认我们自己。

这就像威廉·布里奇斯在《转变》一书中介绍的转变仪式：
1.  夜晚，部落村民聚在篝火旁，为一位即将成年的青年举行仪式。
2.  部落长老用镰刀在青年脸上留下两道象征生活残酷的伤疤。
3.  青年必须离开部落，独自去森林流浪两个月，没有身份、家人和部落的保护。
4.  当他以新身份回到村庄时，脸上的刀疤已成为成人的标记。
5.  他的父母会烧掉他从小到大睡过的席子，象征与过去的彻底告别。
6.  最初一段时间，他“不认识”父母，少年的记忆变得遥远。
7.  最后，父母会给他取一个新名字，完成整个转变。

这个仪式讲的就是转变的历程。在这个仪式里，少年失去了家人和部落的保护，他必须以一个独立自我的身份去面对生活的残酷，直到这个自我足够坚定，他才能重新回归。**失去不仅是失去，也伴随着得到。** 我们人生中一些最重要的东西，都是在失去中得到的。

当我们远离人群，独自“流浪”，我们究竟能得到什么呢？

以下是失去可能带来的三种收获：

**首先，你会学会更依赖你自己。**
依赖自己是失去给你带来的最重要的转变。这种转变常常会积聚和释放巨大的能量。当没有群体接纳你，没有显赫的身份保护你，没有可靠的关系支撑你时，你只能依靠自己。为了把自己培养成可靠的人，你需要：
*   独立做出判断和选择。
*   发展自己的能力。
*   为自己创造机会。
*   发展面对挫折的坚韧。

你能依靠的只有你自己。这是一个简单的事实，它逼着你重视自己、信任自己。意识到这一点，常常是一个人成熟的开始。这不只是能力上的依赖，也包括在回答“我是谁”这个问题上，你也要更加依赖你自己。但依赖自己并不意味着独来独往、不能求助。

**其次，让你更珍惜现在的拥有。**
因为失去，你会对自己已有的东西、已有的关系变得更加珍惜。这些东西即使一直都在，只是因为失去，你更会看到它。

例如，一位朋友从大公司高管的位置离开，重新创业。从资源丰富到一切从零开始，他经历了巨大的落差和失落。但过了一段时间，他说：“我很庆幸，我还拥有这么多。现在虽然团队小，但我的业务更自由了。以前做事需要看老板脸色，现在我能自己决定。还有我的妻子，经历这么多挫折，但一直在身边支持我，我们的感情从来没有这么好过。”这种“看见”是为了适应失去所做的努力，但它并非自我欺骗。那些最简单的存在和最纯真的关系，真切地成了新自我的一部分。

**第三，失去会让我们觉察人存在的本质。**
有时候人需要经历失去，才会理解存在的本质，理解那种更接近生命本身的孤独和自由。因为这种理解，我们会变得更加宽容。

正如一位读者在信中所说：“在经历失去的迷茫期，我的生命里长出了全新的东西。其中最宝贵的经历就是我能够静下心来看以前看不下去的文学作品。当我在自我怀疑、远离人群时，看到有人把这种痛苦、挣扎和可能的救赎写成文字，我就觉得我的孤单有了共鸣，就觉得自己不那么孤单了。”这种理解和共鸣，也是我们从失去中得到的。

让我们用电影《心灵奇旅》的例子来作为本节的结尾。在电影里，钢琴师乔伊终于梦想成真，与著名的爵士乐手完成了一场完美演出。演出后，爵士乐手问他感觉怎么样。乔伊愣了一下，说：“我以为会有很大的不同，可是好像也没有什么不同。”

然后，爵士乐手讲了一个故事：
> 从前有一条小鱼，它游到一条老鱼旁边问：“我想要找到大海，大海在哪里？”老鱼说：“你现在就在大海里呀。”小鱼说：“这儿吗？这儿只是水呀。”

**发现自己就在“大海”里，这也是我们从失去中得到的。** 失去会制造一种匮乏感，这常常是目标带来的思维窄化造成的。有时候，我们需要失去自己执着的东西，才会发现自己早已拥有生命本身的富足。这种富足就在一呼一吸之间，在习以为常的生活中，在最简单的美食和最朴素的人际关系里。它会让我们发现，无论我们失去什么，至少我们还有自己。正因为这个发现，我们才开始好好审视和珍惜我们自己。

**失去会开阔我们的心胸，增长我们的智慧。**

---

本节课中，我们一起学习了如何面对失去。我们认识到，失去是自我转变的必经之路，它虽然痛苦，但也能教会我们依赖自己、珍惜拥有并理解存在的本质。重要的不是避免失去，而是学会如何让失去的经历，成为滋养新自我成长的养料。

讲完如何面对失去，下节课我们就会进入下一个环节：告别旧自我。
