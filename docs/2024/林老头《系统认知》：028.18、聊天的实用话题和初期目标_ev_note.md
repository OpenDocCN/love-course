# 林老头《系统认知》：028.18、聊天的实用话题和初期目标 🗣️

## 概述

在本节课中，我们将学习如何通过有效的聊天技巧来建立和推进人际关系。我们将探讨如何避免错误的暴露需求感，以及如何通过深度聊天和换位思考来提升聊天的顺畅度。

## 错误的暴露需求感

**定义**：在对方没有与你建立任何连接的情况下，表达自己的需求感。

**例子**：当你看到一个人，觉得他对你胃口，想和他交朋友，这就是表达好感。

**如何避免**：

1. **正确表达需求感**：以合适的方式表达自己的需求，而不是直接暴露。
2. **认同对方**：给对方一个认同他、认可他、觉得他优秀的好感。
3. **避免过度需求**：不要让对方觉得你需要他。

## 深度聊天技巧

**目标**：让聊天更加顺畅，减少需求感的暴露。

**方法**：

1. **开启特殊话题**：例如“在陌生人面前做最真实的自己”。
2. **分享隐私和秘密**：但要确保氛围合适。
3. **借助外力**：例如真心话大冒险。
4. **控制尺度**：不要急于推进关系。

## 换位思考

**重要性**：在聊天过程中，理解对方的想法和顾虑。

**如何锻炼**：

1. **与女性交朋友**：了解她们的想法和感受。
2. **约会**：增加与女性的互动，提升理解能力。

## 总结

本节课中，我们学习了如何避免错误的暴露需求感，以及如何通过深度聊天和换位思考来提升聊天的顺畅度。希望大家能够将这些技巧应用到实际生活中，提升自己的人际交往能力。
