# 玲金玉（五套合集）：3.心门锁链密训营 第20课：能量蛋负重训练准备 🥚

在本节课中，我们将学习如何为能量蛋训练进行负重准备。负重是提升训练效果的关键步骤，能帮助增强相关肌群的力量与控制力。

上一节我们介绍了能量蛋的基础使用方法，本节中我们来看看如何为其增加负重，以深化训练效果。

## 如何进行负重 🎼

当能量蛋已置入阴道后，我们需要使用一个挂钩来连接负重物。这个挂钩用于勾住固定能量蛋的牙线。

以下是具体的操作步骤：

*   准备一个用于负重的小袋子或任何你喜欢的小巧物品。
*   金于老师为购买课程的学员推荐使用水晶球。水晶球外观漂亮，使用起来也方便。
*   将准备好的负重物用挂钩勾起来，即可开始训练。

## 负重的重量建议 ⚖️

关于负重的重量，有明确的参考标准和建议。

一个标准的小袋子大约可以容纳三支口红。三支口红的重量大致在30克左右，这个初始重量是合适的。

负重的原则是可以根据个人训练进展和承受能力，逐步增加重量。

训练初期，不建议一次性增加过大的重量。后续课程会详细告知具体的负重进阶重量标准。
