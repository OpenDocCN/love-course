# 18、坏小k女人一生必学的一堂课：48.感情36计（28）如何通过聊天建立完美形象

🎼感情36计的第28个技巧，聊天的核心是塑造形象，而非产生交流。如何通过聊天建立一个完美的形象。

### 一、聊天细节

1. 给对方起一个别人没有叫过的昵称。
2. 在适当的时机说出“咱们”四个字，效果爆炸。
3. 多次提到“咱俩都多大人了”，避免说出可能丢人的话。
4. 深层反向预测，根据对方的外表性格反向猜测其内心。

### 二、频繁叫对方的名字

频繁地叫对方的名字或之前给对方起的昵称，而不是用“爱”来代替。

### 三、高价值离场

聊不下去时，尝试高价值离场，而不是一味地找话题。例如，展示工作受领导重视或生活健康自律。

### 四、不解释、不证明、不对抗

保持框架，同频率沟通。

### 五、聊天沟通时的误区和注意事项

1. 不要无端释放关心，避免以关心的方式开场。
2. 不要迷恋对方的认同，保持自信。
3. 尽量别向自己喜欢的人征求意见，传递独立自强的品质。
4. 掌握社交软件沟通原则，注意聊天字数和回复时间的对应原则。
5. 尽量少问问题，用疑问句代替陈述句。
6. 能用一个字表达清楚，绝不使用两个字。
7. 一思只发一条消息，避免追命连环问。

### 六、无限话题制造

以十字纵横思维制造话题，例如，从对方的话中找出关键词，然后从不同方向展开话题。

### 七、十字纵横对话思维

运用十字纵横对话思维，可以和任何人同步，达到无所不包的聊天内容。

🎼本节课中我们一起学习了如何通过聊天建立完美形象，掌握聊天技巧和注意事项，以及如何制造无限话题。希望这些技巧能帮助你在人际交往中更加得心应手。
