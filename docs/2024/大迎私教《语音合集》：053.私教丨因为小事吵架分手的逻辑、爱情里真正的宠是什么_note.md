# 大迎私教《语音合集》：053.私教丨因为小事吵架分手的逻辑、爱情里真正的宠是什么

## 概述
在本节课中，我们将探讨情侣间因小事吵架分手的逻辑，以及爱情中真正的宠是什么。

## 1. 分手逻辑
**关键问题**：不应该区分逻辑上的对错，重要的是态度。

* **态度**：当矛盾发生时，直接去哄对方，给予正面情绪。
* **让步**：接受委屈，给予正面情绪，让对方感受到你的包容。

## 2. 爱情中的宠
**宠的本质**：无条件付出，接受可能没有回报。

* **父母与孩子**：父母对孩子的爱是无条件的。
* **交换**：你的付出需要对方给予回报，这不是宠。

## 3. 情感技能
* **辩论技能**：在辩论中适用，但在感情中不适用。
* **情感技能**：关注自身和对方的幸福感。

## 4. 视角升级
* **棱角分明**：接受自己的性格，站在对方角度考虑问题。
* **视角升级**：将问题视为人之天性，以包容的心态面对。

## 5. 包容与回报
* **包容**：先让对方占到便宜，观察对方是否珍惜。
* **回报**：如果对方是白眼狼，则无需继续交往。

## 总结
本节课中我们一起学习了情侣间因小事吵架分手的逻辑，以及爱情中真正的宠是什么。希望这些内容能帮助你更好地理解和处理感情问题。
