# 凡哥私教课程：恋爱成长必修课之七大思维：002.2.步骤思维

🎼今天为大家带来颠覆恋爱思维的第二节课程，即恋爱的步骤思维。从认识女生到成为女朋友，需要经历哪些步骤？让女生从陌生到爱上你，成为你的女朋友，其实只有三个步骤。

**步骤一：吸引**

吸引分为两个小步骤：激起兴趣和建立吸引。

1. **激起兴趣**：目的是让女生对你感兴趣，愿意与你聊天。在微信上，你需要通过朋友圈建设来激发女生的兴趣。如果朋友圈建设差，头像模糊，转发无用链接，女生不会有兴趣与你聊天。

2. **建立吸引**：通过有趣、好玩的聊天模式建立吸引，主要在情绪上建立吸引。需要调动女生的情绪，让她感到兴奋。

**步骤二：建立舒适感**

1. **浅层次舒适感**：基本信息交换，如工作、家庭成员、年龄、兴趣爱好等，让女生对你有初步的安全感。

2. **深层次舒适感**：在合适的时机聊三观，如恋爱观、婚姻观、人生观，用三观去吞噬掉她的三观。

**步骤三：升高关系，升温关系**

1. **升高关系**：让关系一步步变得亲密，最终达到确立关系的目的。

2. **升温关系**：加入暧昧元素，如称呼变化、肢体接触等。

**跨步骤操作的后果**

1. 没有建立吸引和舒适感，直接约女生，容易失败。

2. 没有建立吸引，只会建立舒适感，容易进入友谊区。

3. 建立吸引和舒适感，但不敢升高关系，容易沦为备胎。

4. 在特殊场所快速发展关系，没有建立舒适感，容易导致女生事后反悔。

**总结**

本节课介绍了恋爱的步骤思维，包括吸引、建立舒适感、升高关系和升温关系。按部就班地完成这三个步骤，才能提高成功率。下一节课将继续分享恋爱思维的第三节课，奖品思维。🎼O我们下节课见。
