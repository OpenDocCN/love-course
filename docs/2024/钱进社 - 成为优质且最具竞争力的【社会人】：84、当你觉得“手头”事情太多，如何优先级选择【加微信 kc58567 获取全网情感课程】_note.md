# 钱进社 - 成为优质且最具竞争力的【社会人】：84、当你觉得“手头”事情太多，如何优先级选择

## 概述

在本节课中，我们将学习如何在面对众多任务时，进行优先级的选择，以提高工作效率和情绪管理。

## 优先级选择的重要性

当你觉得手头事情太多，首先要明确的是，并非所有事情都同等重要。以下是一些关键步骤：

### 1. 整理工作

- **初步整理**：先对所有工作进行一次快速浏览，初步判断工作之间的关联性。
- **雷同性**：找出具有雷同性的工作，并分析它们之间的关联性。
- **合并处理**：尽可能将具有关联性的工作合并处理，提高效率。

### 2. 评估自身状态

- **精力有限**：认识到人的精力是有限的，不能同时处理所有事情。
- **优先选择**：在评估眼前事情的同时，也要评估自身状态，避免因精力不足而影响工作效率。

### 3. 确定优先级

- **紧急性**：具有截止日期的任务，如下周二前必须完成的任务。
- **重要性**：不做会产生重大后果的任务，如不签署合同可能导致客户流失或违约金。

### 4. 判断重要性

- **公司期待**：了解公司主轴和主旋律，确保个人工作与之相符。
- **个人主轴**：明确个人工作主轴，确保工作质量。

### 5. 控制任务数量

- **筛选任务**：将任务控制在三件左右，避免精力分散。
- **优先级排列**：对筛选出的任务进行优先级排列，确保重要任务得到优先处理。

### 6. 判断不需要做的事项

- **评估效率**：判断哪些任务会拉垮整体效率，进行取舍。
- **灰色地带**：针对无法立即判断的任务，进行深思熟虑。

### 7. 基准判断

- **明确基准**：根据公司规范、方针或个人价值观，快速做出判断。
- **灰色地带**：针对无法立即判断的任务，寻找明确的基准。

### 8. 深思熟虑

- **灰色地带**：针对无法立即判断的任务，进行深思熟虑。
- **信息搜集**：搜集更多信息，与权威人士讨论，集思广益。
- **备选方案**：思考更多方法，制定备选方案。

## 总结

本节课中，我们学习了如何在面对众多任务时，进行优先级的选择。通过整理工作、评估自身状态、确定优先级、控制任务数量、判断不需要做的事项、基准判断和深思熟虑，我们可以提高工作效率，更好地管理情绪。希望这些方法能帮助你在面对繁重任务时，保持冷静，高效地完成任务。
