# 乐嘉7套：5.乐嘉性格色彩全能训练营（入门）【勇略】：27 职场如战场，职场中的孙子兵法性格

## 概述
在本节课中，我们将学习如何通过观察职场中同事的性格色彩来判断他们的性格特点。

## 观察办公桌
以下是观察办公桌来判断性格色彩的方法：

**红色性格**：
- 不喜欢一成不变的工作环境。
- 办公桌上堆满个人物品，如零食、装饰品等。
- 职位较高时，可能放奖杯和奖状。

**蓝色性格**：
- 注重规则和秩序。
- 办公桌整洁简约，物品摆放精确。
- 对物品细节很留意，如铅笔削好、水杯干净。

**黄色性格**：
- 重视效率，抓大放小。
- 办公桌上只放必需品，如电脑、笔筒等。
- 批判性强，不认可同事桌上堆满不必要的东西。

**绿色性格**：
- 以人际关系为导向，欲望天性较低。
- 办公桌上东西较少，喜欢温馨氛围。
- 与同事共享空间时，桌上东西可能增多。

## 观察饭桌表现
以下是观察饭桌表现来判断性格色彩的方法：

**红色性格**：
- 活跃主动，发起话题，分享个人经历。
- 喜欢讲故事，喜怒哀乐容易表现在脸上。

**蓝色性格**：
- 谨慎少言，跟随他人话题回应。
- 谨慎仔细，检查菜单价格，确保无误。

**黄色性格**：
- 讨厌浪费时间，不参加无意义的饭局。
- 容忍度最低，对上菜慢等问题无法忍受。

**绿色性格**：
- 磨磨唧唧，反应慢，重视人际关系。
- 不会主动发言，但会聆听，随口附和。

## 总结
本节课中，我们学习了如何通过观察办公桌和饭桌表现来判断职场中同事的性格色彩。希望这些方法能帮助你在职场中更好地与人相处。
