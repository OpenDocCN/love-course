# 21、凯男的社交情商课：情商提升3.0课程：006.情商3.0课程-吸引的核心

## 概述
在本节课中，我们将学习如何通过提升个人魅力和展示面来吸引他人，特别是如何激发对方的回复欲望和好奇心。

## 步骤一：激发回复欲望
**核心概念**：激发对方的回复欲望等同于激发好奇。

**要点**：
- 直接表白或过度追问不会激发好奇。
- 筛选开场白的思路，避免过度追问。
- 专注于展示面，提升个人魅力。

## 步骤二：展示面的重要性
**核心概念**：展示面是吸引他人的关键。

**要点**：
- 保持个人形象整洁、干净、有品味。
- 朋友圈内容要有意义，展示个人生活品质。
- 保持一致性，避免展示面与实际不符。

## 步骤三：聊天技巧
**核心概念**：聊天技巧是递进的，需要结合展示面和情绪投入。

**要点**：
- 避免过度聊天，保持神秘感。
- 使用多种沟通方式，如表情、文字、语音等。
- 保持频率上的不同，避免让对方感到压力。

## 步骤四：需求感与幻觉
**核心概念**：避免暴露需求感，让对方产生幻想。

**要点**：
- 保持真实，不做作。
- 升高关系，避免成为备胎。
- 提供情绪价值，而非物质价值。

## 步骤五：个性化冲突
**核心概念**：通过个性化冲突吸引对方。

**要点**：
- 找到自己的优点，并放大。
- 与周围人形成反差，展现多面性。
- 保持一致性，避免突然转变。

## 步骤六：反差感与心灵管控
**核心概念**：通过反差感和心灵管控吸引对方。

**要点**：
- 与周围人形成反差，展现独特魅力。
- 保持神秘感，避免过度暴露。
- 保持忽冷忽热，让对方琢磨不透。

## 总结
本节课中，我们学习了如何通过提升个人魅力和展示面来吸引他人，以及如何运用聊天技巧和个性化冲突来激发对方的兴趣。希望这些内容能帮助你提升自己的吸引力，找到心仪的伴侣。
