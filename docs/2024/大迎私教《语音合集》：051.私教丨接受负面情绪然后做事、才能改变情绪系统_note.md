# 大迎私教《语音合集》：051.私教丨接受负面情绪然后做事、才能改变情绪系统

## 概述
在本节课中，我们将学习如何接受负面情绪并付诸行动，以改变情绪系统。

## 核心概念
**公式**：持续行动 = 改变情绪系统

## 教程内容

### 1. 主观与客观的区分
**以下是主观与客观的区分：**
- 主观：你自己的感受和想法。
- 客观：外界对你的反馈和评价。

### 2. 情绪与行动的分离
**以下是情绪与行动的分离方法：**
- 确保自己听明白了别人在说什么。
- 针对别人说的话，表达自己的感受和想法。

### 3. 持续行动的重要性
**以下是持续行动的重要性：**
- 持续行动会定义你是谁。
- 通过持续行动，可以缓解主客体不分离的问题。

### 4. 以事入道
**以下是“以事入道”的概念：**
- 通过做好事情，可以治愈自己的想法和情绪。

### 5. 谦虚行动
**以下是谦虚行动的方法：**
- 设定每天要做的事情，并坚持执行。
- 不论情绪如何，都要完成每天的任务。

## 总结
本节课中我们一起学习了如何接受负面情绪并付诸行动，以改变情绪系统。通过持续行动，我们可以缓解主客体不分离的问题，并最终实现情绪系统的改变。
