# 课程02：表达技巧 - 成为说话“有趣”的人 🎤
## 第04节：巧妙的“称赞” ✨

在本节课中，我们将要学习如何通过巧妙且有趣的称赞来提升沟通效果，营造积极的谈话氛围，并拉近人与人之间的距离。

称赞在沟通中是一种非常特殊的语言。恰当的称赞可以营造积极的谈话氛围，也可以拉近我们和别人之间的距离。我们来对比一下下面几种称赞的方式。

以下是几种不同的称赞场景，请观察哪种方式更有趣。

*   假设你看到一个同事衣服搭配得不错。
    *   A这样说：“你这衣服搭配挺好。”
    *   B这样说：“你这衣服搭配得真好，是不是偷偷去专门学习了时尚搭配课？”
*   假设你请一帮朋友吃饭，大家帮了你一个忙。你端起酒杯感谢了A、B、C，但忘了感谢D。
    *   这时D说：“哎，你怎么没提我呢？”
    *   你可以说：“你哈在我心里。”这也是一种称赞。
*   假设你让同事到楼下帮你买了一杯奶茶。他帮你买上来之后，你表示感谢。
    *   A这样说：“谢谢啊。”
    *   B喝了一口，然后说：“哎呦，这漂亮的人买的奶茶都是甜的。”

一句巧妙有趣的称赞，完全可以提升双方之间的关系，完全可以让气氛更加轻松、更加欢乐。

上一节我们对比了不同称赞方式的效果，本节中我们来看看五种具体的、有趣的称赞方法。当你想夸别人时，就可以选择一种更好玩、更适合的方式。

以下是五种有趣的称赞方式。

1.  **先抑后扬**
    *   **含义**：先说一句反面的、看似批评的话，然后忽然转折，转为夸奖。
    *   **示例**：“这直播太不应该看了……（停顿）……看了之后就无法自拔，有趣得要死。” 或 “这直播太不应该看了……（停顿）……看了之后我就忘记吃饭。”
    *   **核心**：`预期（负面） -> 反转（正面夸奖）`
2.  **双杀**
    *   **含义**：连续夸两个点，用第一个点作为铺垫，引出并强调真正的重点。
    *   **示例**：夸人好看——“你呀一定很有钱……（停顿）……因为你长得这么好看，那得省多少置装费吧？” 夸人会说话——“我猜追你的人肯定很多吧……（停顿）……你这么会说话。”
    *   **核心**：`铺垫（属性A） -> 强调（真正想夸的属性B）`
3.  **打标签**
    *   **含义**：用一个众所周知的人物或事物的鲜明特点，来形容你想夸的人。
    *   **示例**：夸人帅——“他是北京朝阳区的彭于晏。” 夸老师讲课有趣——“朱老师是知识界的周星驰。” 夸人点子多——“他的脑子啊就是印刷机，专门印刷点子。”
    *   **核心**：`对象 -> 像 -> [知名标签]（标签特点=对象优点）`
4.  **反夸**
    *   **含义**：当别人夸你时，巧妙地将称赞关联回对方身上。
    *   **示例**：同事说：“你今天看着真精神啊。” 回答：“我一般见到我欣赏的人就会格外精神。” 经理说：“这次报告写得好。” 回答：“我这不得追赶您的水平吗？”
    *   **核心**：`接受称赞 -> 建立关联 -> 夸赞回对方`
5.  **彰显影响**
    *   **含义**：不直接夸行为本身，而是强调对方行为带来的具体、积极的影响或结果。
    *   **示例**：同事打扫了办公室。普通说法：“真勤快，把屋打扫了。” 有趣说法：“哇塞，你看看这地干净得可以当镜子照了！我来了公司5年从来没这么干净过。”
    *   **核心**：`行为 -> 产生了 -> [具体、夸张的积极影响]`

本节课中我们一起学习了五种让称赞变得有趣的技巧：**先抑后扬**（制造反转）、**双杀**（铺垫强调）、**打标签**（形象类比）、**反夸**（借力回馈）和**彰显影响**（突出效果）。

**本节练习**：请想一位你想称赞的人。用我们学到的五种方式，至少选择三种进行构思练习，然后挑选你认为最好的一种在实际中运用，试一试效果。
