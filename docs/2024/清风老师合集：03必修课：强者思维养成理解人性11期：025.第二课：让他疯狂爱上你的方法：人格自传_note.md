# 清风老师合集：03必修课：强者思维养成理解人性11期：025.第二课：让他疯狂爱上你的方法：人格自传

## 概述
在本节课中，我们将学习如何通过讲述自己的人格自传来吸引他人，特别是如何让对方疯狂地爱上你。

## 什么是人格魅力？
人格魅力就是你的**人生自传**。在你成为今天的自己之前，你经历了大量的事情。而别人，特别是你喜欢的人，他并不了解你是如何成为今天自己的。

## 人格自传的七个要素
以下是构建人格自传的七个核心要素：

1. **失败**：分享你的失败故事，让对方看到你的坚韧和成长。
2. **目标**：讲述你的人生终极追求，展现你的雄心和抱负。
3. **动力**：解释你为什么要做今天所做的这些事情，让对方看到你的内在动力。
4. **结果**：分享你的阶段性成果和最终成果，让对方看到你的成就。
5. **导师**：讲述在你成功的道路上，谁帮助过你，以及他们取得的成就。
6. **伙伴**：分享你的伙伴故事，让对方看到你的社交圈子和支持系统。
7. **敌人**：讲述曾经背叛过你、伤害过你的人，以及你从他们身上学到的东西。

## 如何运用人格自传
在聊天中，不需要一次性将所有要素都讲完，但要有意识地将这些元素融入到你的沟通中。你会发现对方对你的人生越来越好奇，对你的经历越来越期待。

## 总结
本节课中，我们学习了如何通过讲述自己的人格自传来吸引他人。通过分享你的故事和经历，你可以让对方对你产生深刻的兴趣和吸引力。
