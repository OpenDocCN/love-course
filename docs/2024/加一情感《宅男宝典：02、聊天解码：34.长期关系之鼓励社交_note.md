# 加一情感《宅男宝典》：02、聊天解码：34.长期关系之鼓励社交 🕊️

在本节课中，我们将要学习长期关系相处模型的第四个关键步骤：鼓励社交。我们将探讨为何限制伴侣社交是错误的行为，以及如何通过鼓励和引导，建立更健康、更具吸引力的亲密关系。

上一节我们介绍了长期关系中的其他维护技巧，本节中我们来看看“鼓励社交”这一核心环节。

我经常对女生说：**没有社交的女人是缺乏魅力的**。很多男性在恋爱时，会要求女友删除手机里所有的异性联系人。例如，我曾遇到一位女生，她突然发消息说要把我删掉，再用小号加我。询问原因后得知，是她男朋友要求她删除微信上所有男性。

这种行为让我感觉，这位男性缺乏自信。在我看来，他已经在这段关系中处于劣势。正确的做法恰恰相反，应该是鼓励社交。

许多人想把伴侣像金丝雀一样关在笼子里，据为己有。但他们忽略了人的逆反心理。越是阻止对方社交，对方越会“见缝插针”，甚至从与其他异性的接触中获得一种“偷”的快感。这种做法非常错误，反而会把你的伴侣推开。

正确的做法是鼓励社交，这反而能增加你的吸引力。以下是我与女友的一个实例。

那时我们刚在一起不久。有一次晚上我在工作，她问我要不要去看电影。我当时没时间，便建议她：“要不你找个朋友一起去吧。”我知道她有一位关系不错的男性朋友，长得也很帅。我便接着说：“要不你就找那个帅哥跟你去。”

她听后有些不爽和吃惊，说：“那我就跟他去喽？”我回答：“没事，反正你们也是朋友，没关系，就让他先代替我陪你一会吧。”最后这句话很有分量。

她听完后很兴奋，问我：“为什么你跟其他男人不一样？一般的男人都不会让我出门，不让我跟其他男人社交，而你却主动让我去。”后来，我用一种证明的方式告诉她，这样做对她自己也有好处，多接触不同的人能提升阅历。结果，她对我产生了更强的崇拜感。

你越是鼓励她社交，她最终越会属于你。原因有三点：
1.  她会认为你是一个大度、有包容心的男人。
2.  她会认为你真正地为她好。
3.  她会觉得你很理解她。很多男人总认为女人外出社交就是想“浪”，其实并非如此。女性和男性一样，都渴望成长。

鼓励社交有几个关键要素。以下是具体方法：

第一点是设定社交规则。你可以允许伴侣外出社交，但必须共同遵守几条基本规则。例如，我给伴侣设定的规则是：
*   不能超过凌晨一点回家。
*   绝对不能喝太多酒。
*   到家后必须报平安。

规则无需设置得充满不安全感（例如必须要有其他女性在场），只要是和熟悉的朋友聚会即可。关键在于要表现出包容的状态，并相信自己、相信你的伴侣。

上一部分我们讲了设定规则，接下来看看第二点：社交方式的传授。

你可以主动教给伴侣一些社交技巧和方法。既然你学习了相关课程，相信你也能掌握一些技巧去引导她。例如，我会告诉她：“你和同事出去聚会，千万不要一个人坐在那里玩手机。你必须和每个人都打招呼，要照顾到大家的感受。”

当我将这些社交本质教给她时，她会觉得：“别的男人都想把我死死拽在家里，你却把我往外推，你和他们真的完全不一样。”

当然，你有空时还是要多陪伴她。否则，如果你明明在她身边，却一味鼓励她出去和别人玩，女生可能会觉得你不喜欢她，在故意推开她。

这一点在异地恋中同样重要。你必须鼓励对方社交，并战胜自己内心的不安全感。你越是这么做，对方越会属于你。试想一下，如果你们异地，你摸不着也见不着，却对她说：“周六必须在寝室呆着，不准去参加社团或公司聚会。”她立刻会觉得你很狭隘，并产生对抗心理。她可能不仅要去，还要故意气你，甚至做出一些冲动的事，那时你将无能为力。

所以，与其强行限制，不如鼓励她社交，哪怕是异地恋。

本节课中我们一起学习了鼓励社交在长期关系中的重要性。

总结一下本节课程：
1.  **没有社交的女人是缺乏魅力的**。不敢让伴侣去社交的男人是缺乏吸引力的。
2.  鼓励社交有两个核心要素：**设定合理的社交规则**与**传授有效的社交方式**。
