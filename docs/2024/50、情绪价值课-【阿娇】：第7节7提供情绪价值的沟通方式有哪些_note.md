# 50、情绪价值课-【阿娇】：第7节7提供情绪价值的沟通方式有哪些_note

在本节课中，我们将学习除了夸奖之外，在沟通中为对方提供情绪价值的四种核心技巧。上节课我们介绍了如何通过高级夸奖技巧提供情绪价值，本节我们将深入探讨其他有效的沟通方法。

## 概述

情绪价值是人际交往中的重要组成部分，尤其在亲密关系中。通过特定的沟通方式，我们可以让对方感受到被理解、被重视和被欣赏，从而增强彼此的情感连接。本节课将详细介绍四种实用的沟通技巧，帮助你在日常交流中有效提供情绪价值。

## 技巧一：语言上的推拉

上一节我们介绍了夸奖的技巧，本节中我们来看看如何通过语言上的推拉制造情绪波动。推拉是一种通过语言制造距离感和亲近感交替出现的技巧，能快速引发对方的情绪变化。

推是指在语言上推开对方，例如拒绝、贬低、打压或批评。拉是指在语言上拉近对方，例如奉承、夸奖或赞美。

以下是推拉技巧的应用示例：

你的男朋友穿着新西装准备出门，他得意地问你：“亲爱的，我今天的这身衣服怎么样，帅不帅？”

**推的部分：**
“你这个衣服很一般，穿起来也不是那么好看，也不符合你的气质。”

**拉的部分：**
“以后除了你跟我一起出门，不然不允许你穿这套衣服。因为你穿这套衣服显得太帅了，让我一点安全感都没有，让我很烦，你长得那么帅，还会穿搭，哪个女人对你都不动心啊，我哪还有安全感啊？”

通过这种先推后拉的方式，对方的情绪从期待到失落再到高涨，体验了完整的情绪波动过程。这种技巧能让对方感受到惊喜和意外，从而更享受与你相处的过程。

## 技巧二：预判情绪需求

除了表面的语言交流，更深层次的沟通在于理解对方的情绪需求。大多数人渴望被读懂、被理解，希望对方能说出自己内心想听的话。

以下是预判情绪需求的技巧：

当对方说：“宝贝，这几天降温了，你有没有多穿一些呢？”

表面上是关心你的穿着，深层情绪需求其实是：“我在想你，想知道你过得好不好。”

**预判后的回应：**
“被人关心的感觉真好。你是不是在想我？你想我的话要告诉我呀，因为我也在想你。我也在想你有没有好好照顾你自己，我在想你今天穿得暖不暖？”

当你能够准确预判对方的情绪需求并给予回应时，对方会感受到在你心中的重要位置。这种互相在意的感觉能有效提升彼此的情感连接。

## 技巧三：聚焦对方的骄傲点

在聊天过程中，学会识别并重点提问对方话中的骄傲点，能让对方感受到被尊重和被欣赏。

以下是聚焦骄傲点的技巧：

当对方在谈话中提到自己的成就或自豪的事情时，不要挑毛病或质疑，而是针对这些点进行深入提问。

**例如：**
如果对方提到：“我最近负责的项目获得了公司年度创新奖。”

**有效的提问方式：**
“太厉害了！这个项目最难的部分是什么？你是怎么克服那些挑战的？”

通过这种方式，对方会越说越兴奋，感受到成就感和优越感。他会享受与你聊天的过程，因为你在认真倾听并认可他的价值。

## 技巧四：通过分享日常传递爱意

分享日常是表达爱意的重要方式，但要注意方法，避免变成单方面的情绪索取。

以下是有效分享日常的技巧：

在分享日常时，加入表达爱意和肯定对方的话语，让对方在倾听过程中获得情绪价值。

**对比两种分享方式：**

**方式一（无效分享）：**
“我今天上班又被老板批评了，明明不该我做的事情，最后全推在我身上。我烦都烦死了。还有那个新来的男同事，他这也不会那也不会，什么问题都来问我，搞得我焦头烂额的。”

**方式二（有效分享）：**
“亲爱的你知道吗？我今天被老板批评了。亲爱的，果然全世界的男人除了你呀，都是大猪蹄子。明明不该我做的事情，最后全推到我这儿了，我都烦死了。亲爱的，你快来抱抱我，然后心情好一点。还有那个新来的男同事，他这也不会那也不会，什么问题都来问我。哎，我真不知道这个公司怎么招人的，一点标准都没有啊。哎，如果你去面试的情况下，真的，他们会直接让你当董事长。”

在第二种方式中，通过加入“亲爱的”、“全世界的男人除了你都是大猪蹄子”、“快来抱抱我”等表达爱意和需要的话语，以及“他们会直接让你当董事长”这样的肯定，让对方在倾听过程中感受到被爱和被肯定，从而更愿意倾听和回应。

## 总结

本节课我们一起学习了四种提供情绪价值的沟通技巧：

1.  **语言上的推拉**：通过先推后拉制造情绪波动，让对方体验惊喜和意外。
2.  **预判情绪需求**：深入理解对方话语背后的情绪需求，说出对方想听的话。
3.  **聚焦对方的骄傲点**：在聊天中重点提问对方自豪的事情，让对方感受到被欣赏。
4.  **通过分享日常传递爱意**：在分享日常时加入爱意表达和肯定，让对方在倾听中获得情绪满足。

掌握这些技巧，你就能在沟通中更有效地为对方提供情绪价值，增强彼此的情感连接。记住，有效的沟通不仅是信息的传递，更是情感的交流和价值的提供。
