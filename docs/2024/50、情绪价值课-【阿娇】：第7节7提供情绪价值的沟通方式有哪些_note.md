# 情绪价值课-【阿娇】：第7节 提供情绪价值的沟通方式有哪些

大家好，我是娇姐。上节课我们讲解了如何通过夸奖对方来提供情绪价值。今天，我们将探讨除了夸奖之外，还有哪些技巧可以在沟通中为对方提供情绪价值。

### 1. 语言上的推拉

**概念**：语言上的推拉是指通过语言与对方拉开或拉近距离。

**推拉示例**：
- 推：拒绝、贬低、打压、批评
- 拉：夸奖、赞美

**应用**：
- 例子：男朋友穿新西装出门，询问你的看法。你可以先贬低衣服，再夸奖他穿这套衣服显得太帅，让他感到惊喜和意外。

### 2. 预判情绪需求

**概念**：在沟通中预判对方的情绪需求，并满足它。

**应用**：
- 例子：男朋友关心你穿得暖不暖，你可以回应：“被人关心的感觉真好，你是不是在想我？你想我的话，你要告诉我，因为我也在想你。”

### 3. 挑对方话里的骄傲点

**概念**：在聊天中挑选对方骄傲的点进行提问，让对方感到被欣赏。

**应用**：
- 例子：对方炫耀自己的成就，你可以提问：“你在这方面做得真好，你是怎么做到的？”

### 4. 分享日常传递爱意

**概念**：在分享日常时，加入爱意和对方喜欢听的话，增加情绪收益。

**应用**：
- 例子：分享工作压力时，加入爱意：“亲爱的，今天工作辛苦了，你快来抱抱我。”

### 总结

本节课我们学习了四种提供情绪价值的沟通方式：语言上的推拉、预判情绪需求、挑对方话里的骄傲点、分享日常传递爱意。通过这些技巧，我们可以更好地与对方沟通，提升彼此之间的感情。

- 语言上的推拉
- 预判情绪需求
- 挑对方话里的骄傲点
- 分享日常传递爱意
