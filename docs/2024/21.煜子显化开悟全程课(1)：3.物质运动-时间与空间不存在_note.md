# 21.煜子显化开悟全程课(1)：3.物质运动-时间与空间不存在

大家好，今天我们探讨时间与空间。**时间和空间与物质运动相关，并非固定不变，存在肉眼无法察觉的内涵**。我们常以为时间和空间固定存在，但实际上并非如此。

**时间与空间是物质的产物，物质运动即时空运动，时空运动也代表物质运动**。时间的出现是空间运动的产物，空间的变换产生时间。

**时间不存在的原因**：

*   **三维空间中的时间和空间是可理解的，但真实的时间不存在**。因为空间也不存在，时间和空间是物质带来的属性。
*   **物质也不存在，因为物质由意识带来**。例如，梦境中的意识体验，并非从零开始，而是来源于意识储存。
*   **醒来并非解脱，而是掉入更深的无意识状态**。真正的清醒是在半梦半醒之间，快速演动期。

**梦境与现实**：

*   **梦境来源于潜意识，与现实无关**。梦境中的恐惧、焦虑、恐惧等，与现实中的想法可能完全不同。
*   **改变梦境，改变现实**。在梦中勇敢接纳，现实中也会显化出来。

**显化与未来**：

*   **显化不是执念，而是改变内核**。改变念头，改变未来。
*   **显化与他人无关，只与你自己有关**。显化自己的感受，自然会吸引阳性粒子。

**灵魂出窍与假我**：

*   **灵魂出窍是真实感受到外界实像改变**。但并未离开，只是意识体验。
*   **假我、小我、玩家我，都不是真正的你**。真正的你是圆满的，无恐惧、无焦虑、无痛苦。

**总结**：

本节课中，我们学习了时间与空间的不存在性，以及梦境与现实、显化与未来、灵魂出窍与假我等概念。希望这些内容能帮助你更好地理解显化开悟的过程。
