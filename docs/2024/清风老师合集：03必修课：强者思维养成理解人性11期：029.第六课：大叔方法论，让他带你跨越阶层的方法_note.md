# 清风老师合集：03必修课：强者思维养成理解人性11期：029.第六课：大叔方法论，让他带你跨越阶层的方法

## 概述

在本节课中，我们将学习如何运用大叔方法论，了解不同类型的大叔，并掌握与他们相处的策略，从而实现情感自由和跨越阶层。

## 大叔类型及相处策略

### 痞帅款

**核心价值**：高观赏和高情绪

**相处策略**：

* **前期**：展现关注，关心对方早期作品。
* **中期**：透露对新鲜事物的好奇，拉对方到你的优势领域。
* **后期**：找到对方内心认同感，给予肯定。

**展示面**：

* 生活：多发艺术展、高端活动、哲学书籍。
* 形象：减少自拍照，多发工作照。
* 情绪：展现焦虑情绪，倾听低落情绪。

**约会策略**：

* 公开场景：画展、讲座。
* 互动场景：咖啡厅、小公园。
* 私密场景：劲吧、车里。

### 土帅款

**核心价值**：高生存价值和高观赏价值

**相处策略**：

* **前期**：展现优秀生活，带对方体验社交活动。
* **中期**：体验新兴趣，丰富共同兴趣。
* **后期**：丰富共同事业，引导投资。

**展示面**：

* 生活：美食、有意思的生活。
* 形象：自拍。
* 工作照：多发数据、行业报告。

**情绪**：

* 焦虑情绪：询问对方解决方案。
* 低落情绪：带对方体验不同环境和活动。
* 愤怒情绪：通过历史故事展现底层思维。

**约会策略**：

* 公开场景：商场、餐厅。
* 互动场景：滑翔伞、滑雪、骑马。
* 私密场景：五星级酒店。

### 霸总款

**核心价值**：高生存价值、高情绪价值和高物质价值

**相处策略**：

* **前期**：展现对新鲜事物的好奇。
* **中期**：展现个人影响力，请教对方人脉和资源。
* **后期**：贴着对方聊，聊哲学、人文、高维度话题。

**展示面**：

* 生活：高端社交。
* 形象：多拍照，多参与拍摄。
* 工作：展现成长思考和理解。

**情绪**：

* 焦虑情绪：表达崇拜和信任。
* 低落情绪：表达认可。
* 愤怒情绪：带入第三方视角，客观讲述。

**约会策略**：

* 公开场景：静吧、酒吧、夜店。
* 互动场景：艺术品私展、讲座。
* 私密场景：高档餐厅、奢侈品店。

### 暖男款

**核心价值**：高声誉价值和高生存价值

**相处策略**：

* **前期**：展现同情心，关注弱势群体。
* **中期**：展现平静生活状态，引导对方好奇。
* **后期**：展现善良，参与公益活动。

**展示面**：

* 生活：多发社交照片、风景照。
* 形象：自拍。
* 工作：展现小脆弱，勾起同情心。

**情绪**：

* 焦虑情绪：鼓励对方表达需求。
* 低落情绪：带对方换环境。
* 愤怒情绪：让对方表达需求。

**约会策略**：

* 公开场景：商场、茶事、咖啡厅。
* 互动场景：农家乐。
* 私密场景：高尔夫、私人会所。

## 总结

本节课中，我们学习了如何运用大叔方法论，了解不同类型的大叔，并掌握与他们相处的策略。通过运用这些策略，我们可以实现情感自由和跨越阶层。
