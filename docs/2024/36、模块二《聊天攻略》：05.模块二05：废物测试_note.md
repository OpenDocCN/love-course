# 课程36：模块二《聊天攻略》05. 废物测试 🧪

在本节课中，我们将学习如何识别并应对人际交往中的“废物测试”。这是一种常见的社交筛选机制，尤其在两性互动中频繁出现。掌握应对方法，能帮助你在对话中保持自信、展现价值，并有效推进关系。

## 第一部分：如何应对攻击 🛡️

在正式讲解“废物测试”之前，我们需要先了解如何应对他人不友善的言语或行为攻击。生活中难免会遇到恶意，有效的应对策略能保护你的自尊与情绪。

当遭遇恶意攻击时，人们常感到愤怒却无力反击，事后又陷入自责。在工作中，面对不友善的同事，长期的忍让可能导致抑郁或回避社交。因此，学会应对至关重要。

以下是三种应对他人不友善挑衅的方法：

1.  **厘清攻击原因**：首先需要明白，他人的恶意攻击往往与你本身的对错无关。这类似于童年时父母的责怪，有时只是他们自身情绪的宣泄。不要因为他人的攻击而陷入自我否定，保持清醒的自我认知和自信。
2.  **改变认知**：分享一个哲学故事。有人问智者快乐的秘诀，智者说：“不要跟笨蛋争论。”那人不认同，智者回复：“你说的对。”有时，一句“你说的对”就能化解许多争端。
3.  **幽默化解**：面对攻击时，本能反应可能是恐惧或不知所措。如果你有幽默感，可以尝试用它来化解尴尬。例如，演员黄渤在面对刁难问题时，总能给出既谦逊又不卑不亢的高情商回答，既维护了自身价值，也肯定了他人。

**核心要点**：高情商体现在不贬低自己的同时让他人开心。但记住，善良和情商也需有锋芒。适当的退让后，若对方仍不依不饶，则应学会反击，否则便是懦弱。

## 第二部分：废物测试的全套破解方法 🧩

上一部分我们探讨了应对一般攻击的心态。现在，我们正式进入“废物测试”的破解方法。首先，必须理解其定义。

### 什么是废物测试？

高价值的女性身边通常有很多追求者。她们没有时间逐一深入了解，因此发展出一套快速筛选机制，用以淘汰低价值的男性。这套方法就是“废物测试”。

**公式**：女性发起测试 → 男性回应 → 女性根据回应判断男性价值 → 决定是否给予进一步机会。

一旦男性未能通过测试，就会被归类为“废物”（此处为比喻，指失去成为其男友的机会），从而失去关系升级的可能。

你需要认识到，这不仅是测试，更是一个**展示你与众不同和高价值的机会**。一个完美的回答可能让你瞬间脱颖而出，而一个低水准的回答则可能导致你立刻出局。

**关键**：废物测试往往在你展示出足够价值、对方开始对你有感觉时出现。如果你没有展示价值，则基本不会遇到测试。

---

### 应对废物测试的三步原则

当面临废物测试时，应遵循以下三个核心原则：

1.  **不解释**：不要试图为自己辩解。
2.  **不抬杠**：不要试图在言语上争赢对方。
3.  **不较真**：不要过分认真地对待测试本身。

许多男性会陷入纠结，因为他们觉得自己喜欢对方就“低人一等”，想要讨好或赢得认可。其实，你只需要抱着一种无所谓的态度，有时摇头笑笑就是最好的回应。

---

### 六大破解方法

在遵循三步原则的基础上，以下是几种具体的破解方法。请注意，同一测试可能有多种应对方式。

**以下是六大核心破解方法：**

1.  **反问法**
    *   **方法描述**：将问题抛回给对方。
    *   **应用场景**：面对诸如“我和你妈同时掉水里先救谁？”、“你爱我吗？”、“我漂亮吗？”等问题。
    *   **示例代码（话术）**：
        ```plaintext
        女：我和你妈同时掉水里，你先救谁？
        男：你觉得呢？
        女：我问你呢！
        男：我也在问你呀。
        ```
    *   **其他应用**：如果对方攻击性较强，可以设定框架：“你一直都是这样说话的吗？我只是想认识一下，你是不是误会了？我并不是要追你哦。”

2.  **直接承认法**
    *   **方法描述**：坦然接受对方的评价，并转化为认可或夸奖。
    *   **应用场景**：面对略带攻击性的评价，如“你有点胖”、“你好闲”。
    *   **示例代码（话术）**：
        ```plaintext
        女：你肚子好像有点大。
        男：谢谢你的认可。（或者）你这么快就关心我的身材了？
        ```
    *   **要点**：切勿解释！解释会暴露不自信。直接承认或幽默转化，能展现自信。

3.  **直接忽略法**
    *   **方法描述**：忽略自己之前可能造成的尴尬，或忽略对方的测试性话语。
    *   **应用场景**：当聊天出现尴尬冷场，对方回复“呵呵”、“哦”时。
    *   **示例代码（话术）**：
        ```plaintext
        （当对方回复“呵呵”时）
        男：有趣。
        ```
    *   **要点**：此方法主要用于化解因自己之前言行导致的尴尬氛围。

4.  **转移话题法**
    *   **方法描述**：完全不回应对方的测试，直接开启一个新话题。
    *   **应用场景**：当对方言语明显不尊重或不友善时。
    *   **示例代码（话术）**：
        ```plaintext
        女：我不喜欢你的外套。
        男：（看向别处）你有发现这个杯子下面有一颗透明的水珠吗？
        ```
    *   **与忽略法的区别**：忽略法忽略的是“自己的失误”；转移话题法忽略的是“对方的测试”。

5.  **假否定法**
    *   **方法描述**：先假意否定，再用夸张的说法转折。
    *   **应用场景**：面对质问，如“你经常这么搭讪吗？”
    *   **示例代码（话术）**：
        ```plaintext
        女：你经常这么和女人搭讪吗？
        男：不经常啊。（停顿）一天也就搭讪几十个吧。
        ```

6.  **认可并夸**
    *   **方法描述**：先认可对方的评价，然后用夸张、幽默的方式放大。
    *   **应用场景**：面对打压或询问，如“你好矮”、“我漂亮吧”。
    *   **示例代码（话术）**：
        ```plaintext
        女：你多高呀，看起来好矮。
        男：我也觉得，其实再矮一点就好了，能买半价票。
        女：我漂亮吧？
        男：漂亮！（停顿）等一下，你刚才问什么漂亮来着？
        ```

### 常见废物测试场景及应用

掌握了核心方法后，我们来看看它们在具体测试场景中的组合应用。

**以下是七种常见废物测试类型及应对策略：**

1.  **投资型测试**
    *   **表现**：让你发红包、买东西、帮忙做事。
    *   **应对**：
        *   **不熟时**：直接拒绝。优质女性不会轻易索要。
        *   **较熟时**：注重价值交换。例如：“当然可以帮你，那你先帮我买杯奶茶吧。”

2.  **打压型测试**
    *   **表现**：说你胖、瘦、收入低、不成熟等。
    *   **应对**：坚持自我框架，使用**认可并夸大**或**歪曲事实**。例如：“你说我收入低？还没在一起就操心咱们的未来收入啦？”

3.  **调情型测试**
    *   **表现**：聊性话题，如“你看A片吗？”、“喜欢什么姿势？”
    *   **应对**：**巧妙回避并转移**到文艺范畴。例如：“你知道关于性拍得最好的韩国电影是什么吗？”（转向电影讨论）

4.  **暗示型测试**
    *   **表现**：“那件衣服好漂亮，可惜钱不够”、“那家店很好吃，但排队人多”。
    *   **应对**：**转移话题**。例如：“那件是不错，不过后面还有一条街，再看看吧。”

5.  **赋格型测试**
    *   **表现**：“我一直把你当哥哥”、“你做我学长吧”、“我给你介绍个男朋友”。
    *   **应对**：**展示高价值，反问击破框架**。例如：“你对每个男人都想认哥哥吗？……那你是不是喜欢我？不喜欢我为什么要认我做哥哥？”（然后转移话题）

6.  **示好型测试**
    *   **表现**：“你为什么对我这么好？”、“你是不是喜欢我？”、“你这么棒怎么没女朋友？”
    *   **应对**：**给出合理且不失框架的解释**。
        *   “为什么对我好？”：“我觉得漂亮优雅的女士应该得到帮助。”
        *   “是不是喜欢我？”：“你之前一定过得很辛苦，才会觉得我对你的好只是出于喜欢。” 或 “或许你应该见识一下我真爱一个人的样子。”
        *   “怎么没女朋友？”：“因为我在等你长大呀。”

7.  **失恋型测试**
    *   **表现**：“我失恋了”、“我男朋友对我不好”、“我想找个真正喜欢我的人”。
    *   **应对**：**保持冷静，理性分析**。
        *   原则：只要她没分手，就不要抱幻想，简单回应即可。
        *   可进行良性情绪引导并尝试邀约，但不要主动提及其感情问题。
        *   认清自己可能只是“情绪垃圾桶”或“备胎”的可能性。

### 终极心法：建立不被测试的框架 🎯

最有效的应对方法，是从根本上**建立一个不被测试的互动框架**。

**核心公式**：平等心态 + 自我框架 = 免受测试困扰

当你疲于应对各种测试时，应该思考：**我为什么要被测试？** 你我不是追求与被追求的不平等关系，而是平等的两个个体。一旦建立起“我无需接受你的测试”的心态和框架，你就不会因此焦虑，也不会担心因表现不好而被讨厌。

记住，对方可以设定框架，你也可以设定自己的框架。保持冷静、理智、清醒的头脑，是应对一切社交挑战的基础。
