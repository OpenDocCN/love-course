# 罗南希《爱的抚触》训练营 模块1+2+3：第18课 👐 能量爱抚——允许与接纳

在本节课中，我们将要学习爱抚中“允许”与“接纳”的核心概念。这并非抽象的情感理论，而是直接影响爱抚质量与关系连接的具体实践。我们将探讨常见的困境，并通过两个简单练习，帮助你体验如何创造允许与接纳的空间。

---

## 课程概述

爱抚或关系中发生的许多事情，不能简单地用“爱”或“不爱”来概括。爱与实际行动是两回事。本节课程旨在通过具体事例，阐明“允许”与“接纳”在实践中的含义，并提供可操作的练习方法。

## 理解允许与接纳的常见困境

以下是爱抚中常见的六种情景，它们都涉及“允许”和“接纳”的议题：

1.  你精心准备爱抚，运用了所学技巧，但伴侣的状态和反馈都很平淡，你感到挫败。
2.  你想通过爱抚改善关系，但在开口邀请时，就已担心会被拒绝。
3.  你想象中的爱抚是激情浪漫的，但现实中伴侣可能一被抚摸就入睡。
4.  你担心自己手法不够好，爱抚时小心翼翼，生怕对方不舒服。
5.  爱抚结束后，你希望得到感激或表扬，但对方只表示感受一般。
6.  你希望爱抚融入日常生活，但伴侣与你的节奏不一致。

面对这些情况，单纯告诉自己“接受这一切”往往无效，因为难以真正做到。我们需要的是结合沟通技巧与爱抚手法进行调整，逐步接近理想状态。

## 探寻真实需求与调整路径

在实践调整中，理解自身需求至关重要。根据经验，大多数人深层需求是：**被伴侣看见、理解、回应并珍惜**。其次才是思考为达目的可采取的实际行动。最后，在行动后体会真实感受，明确自己能够允许与接纳的边界。

**真实需求 = 实际体会到的感受（而非他人创造或自我假设）**

只有明确真实需求，才能跟随内心去连接，真正地出发去允许和接纳。否则，行为可能仅是缓和关系的策略，当自身未获关照时，怨气容易累积。

拥有允许与接纳的心态，能帮助我们在爱抚时更“无为”、更投入，从而享受过程并被滋养。当你不再纠结于能得到什么，而是专注于提升行动的质感时，你所渴望的自然会到来。

## 两个核心练习：体验允许与接纳

接下来，我们学习两个简单练习，帮助感受允许与接纳。第一个练习可独自完成；第二个练习需双方共同参与，适用于伴侣、家人或朋友。

### 练习一：允许一切——给予自己慈悲

这是一个连接内在、释放空间的练习，可独自进行。适用于日常需求未被满足、犹豫不决、心烦意乱或与他人冲突后。目的是先让自己安定，看见并允许所有感受，为自己创造空间，之后才有空间容纳他人。

以下是练习步骤：

1.  **设定正向意图**
    开始前，为自己设立一个积极目标。例如：“我想让自己更有力量”、“我想更放松”、“我想感受更多爱”。如果难以做到，可先通过跑步、哭泣等安全方式释放情绪，再回到练习。

2.  **准备与呼吸**
    找一个舒适、不被打扰的空间坐稳。将一只手放在肚脐下方，另一只手掌根置于眉心之间。深呼吸，同时手掌轻轻向上舒展眉心。重复此过程，直到感觉舒适。

3.  **觉察情绪源头**
    在呼吸中，觉察当前情绪的源头。放下手，回归正常呼吸后，可自问：
    *   我现在感受到什么情绪？
    *   这与他人有关吗？
    *   我在害怕或担心什么？（如：怕被拒绝、被伤害、被认为无聊、对方失去兴趣或离开）
    请诚实地面对自己，此刻你是安全的。

4.  **培养自我慈悲**
    在坦诚的觉察中，理解情绪背后的渊源，从而对自己慈悲。尝试以“上帝视角”观察自己，而非一味索取不明确的关注。当你能够对自己慈悲，你也能对亲密关系中的他人给予慈悲。

5.  **分享与连接（可选）**
    如果愿意，可在合适时机与伴侣分享你的觉察感受。分享前，建议先与伴侣做一个简单的拥抱。拥抱是最易产生连接的触觉接触，能带来安全感与勇气。
    > **感受示例**：“有一种在一起的感受，比较安心，给了我很多安全感，是一种久违的温暖。”

    分享后，留意身体的感受：是否感觉内在空间更宽广？是否更愿意进行亲密互动？如果答案是肯定的，恭喜你完成了这次勇敢而真实的体验。

### 练习二：心的触碰——体验接纳与珍惜

如果你完成了第一个练习，可能会感慨身体的智慧。接下来这个“心的触碰”练习，能让人感受到被爱包围。这是一个纯粹简单的练习：双方互相捧住彼此心脏对应的区域。它不涉及敏感部位，捧住的是一份支持与珍惜。

**练习准备：**
1.  选择安静、不被打扰的空间。
2.  穿着宽松衣物，卸下首饰。
3.  双方以合适距离坐好。一方正面盘腿，另一方侧坐、双腿张开屈膝。给予者将手肘置于自己支撑腿的膝盖上，保持稳定。

**操作步骤：**
1.  双方感受距离是否合适，有无压迫感。可坐在垫子或床上调整舒适度。
2.  给予者缓慢将后方手靠近接收者后背，轻柔地覆盖在心脏对应区域。
3.  随后，将前方手以同样方式，轻柔地放在接收者前胸心脏位置。
4.  双方存在于当下，静静感受“捧住心”与“心被捧住”的感觉。可闭上眼睛感受。
5.  约20分钟后，双方交换角色。

**练习关键点：**
1.  **时长**：总时长约40分钟，每人担任给予者约20分钟。注意手肘和膝盖的舒适度。
2.  **沟通**：练习中尽量不说话。但开始时，接收者可提示给予者希望的力度（更轻柔或更坚定）。
3.  **手法**：放置时，先接近前胸，然后指尖、掌心从虚到实地慢慢靠近。放开时，也需缓慢离开，避免让敏感的心产生“被抛弃”的感觉。
    > **参与者感受示例A**：“音乐中，我感到被保护，进入了一个属于自己的安全空间。音乐停止、手离开时，我感到失落，意识到那份安静源于支持，我非常需要它。”
    > **参与者感受示例B**：“捧住你的心时，想到你一路走来的变化与坚强，看到我们彼此的扶持，很想给你更多爱与支持。拥抱时，感到两颗心触碰在一起。”
