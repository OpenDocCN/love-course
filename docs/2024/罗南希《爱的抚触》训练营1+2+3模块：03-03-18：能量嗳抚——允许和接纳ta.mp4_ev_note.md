## 概述

在本节课中，我们将学习如何在爱抚中允许和接纳对方，以及如何通过练习提升自我认知和亲密关系。

## 允许和接纳的重要性

**公式**： 允许和接纳 → 无为 → 投入 → 享受 → 被滋养

**代码**： 允许和接纳 = 无为 + 投入 + 享受 + 被滋养

### 以下是关于允许和接纳的几个常见情况：

1. **爱抚时对方反应一般**： 当你努力给予爱抚，但对方反应平淡时，可能会感到挫败。
2. **担心被拒绝**： 在邀请伴侣进行爱抚时，担心对方拒绝。
3. **期望过高**： 期望爱抚充满激情和浪漫，但实际情况却相反。
4. **担心自己不够温柔**： 担心自己的手法不够温柔，让对方感到不适。
5. **希望得到表扬**： 希望在爱抚结束后得到对方的表扬和感激。
6. **希望爱抚贯穿日常生活**： 希望爱抚不仅仅在激情时刻，而是融入日常生活。
7. **伴侣节奏不同**： 伴侣的节奏与你不同，导致爱抚体验不佳。

### 如何提升允许和接纳的能力

1. **自我觉察**： 通过练习，了解自己的需求和感受。
2. **沟通技巧**： 与伴侣进行有效沟通，表达自己的需求和感受。
3. **爱抚手法**： 学习和掌握不同的爱抚手法，提升爱抚体验。

## 练习一：允许所有给自己慈悲

**步骤**：

1. 设定正向目标，例如：更有力量、更有勇气、更放松等。
2. 找一个舒适的空间，进行深呼吸。
3. 觉察自己的情绪，理解情绪背后的原因。
4. 给自己足够的空间，允许一切的发生。
5. 与伴侣分享你的感受。

## 练习二：新的触碰练习，接纳与珍惜

**步骤**：

1. 找一个安静的空间，着宽松的衣服。
2. 两人选定合适的距离，一方正面盘腿坐，一方侧坐。
3. 给予者将手肘放在膝盖上，支撑腿固定。
4. 慢慢将手靠近接收者的背部，轻柔地给予空间。
5. 交换角色，感受被捧住心与心被他人捧住的感觉。
6. 注意三个关键点：练习持续40分钟，尽量不说话，放上去和放开时都要轻柔。
