# leon《全套课程合集》：leon 3.0：12.叙事_note

在本节课中，我们将要学习“叙事”这一核心概念。叙事是关于你如何向他人描述自己的故事、性格、思想和经历，它直接关系到你与他人之间信任感和舒适感的建立。掌握好叙事，能让你在人际交往，特别是与异性相处时，关系发展得更顺畅。

## 什么是叙事？

叙事好比你看完一本书或一部电影后，对主人公的性格、思想、经历以及行为动机的记忆和描述。

你对他了解得越深，对他的思想、性格和经历知道得越多，你们之间的信任感和舒适感就越强。

你和女生的关系也是如此。叙事这个环节直接决定着你和她之间的舒适和信任度。

很多中阶选手容易越过信任和叙事环节，直接进入最后一步的升级诱惑。这往往会导致女生的不适，或不愿意与你发生进一步的关系。

这节课说简单也简单，说难也难。很多初学者喜欢在两人刚认识、自身价值感尚未建立时，就急于进行叙事，暴露自己的思想和经历。然而，叙事应该在评估自身价值感建立得不错之后再进行，用以建立信任。

## 建立叙事与信任的基本方式

以下是建立叙事与信任的一些基本方式。

### 1. 故事展开的基础：时间与地点

你们之间的故事是如何展开的？最基本的两个线索是**时间**和**地点**。

你们聊得越久，共同经历的事情就越多。线下约会时，转场次数越多，共同拥有的记忆也越多。

时间和地点决定着女生对你这个人以及你们共同经历的整体回忆。

### 2. 增强连接感

上一节我们介绍了故事展开的基础，本节中我们来看看如何增强连接感。连接感指的是将你和女生的话题与框架关联起来。

例如，女生表露自己性格上进，你可以说“我们性格还蛮像的”。女生说喜欢旅游，你可以回应“那可以啊，我们以后可以一起去旅游”。

无论遇到什么话题，都尝试将自己和女生联系在一起。这种方式能有效增强你们之间的联系感。

很多人聊天时各说各的，很难把话题串联起来。因此，建立关联是一个非常重要的点位。

### 3. 角色扮演与剧情规划

以下是两种进入新剧情、规划未来的方式。

*   **角色扮演 (Role Play)**：无论是在搭讪还是聊天中，都可以使用。例如，扮演老板与秘书、医生与病人。可以用台词如：“好久没见了，我都得相思病了，医生姐姐能不能治治我的病？” Role Play 是进入男对女新剧情的绝佳方式。
*   **描述未来**：在对话中默认你们未来会共同经历某些事情。例如，“你做饭好吃吗？那以后可以一起做饭吗？” 或者说，“你是一个比较粘人那种人吗？那我希望我工作时你独立一点，但生活上我希望你多粘人一点。” 这直接在描述一个未来，并给出剧情方向。

这一点包含两个部分：一是角色扮演，二是描述你们未来要一起做的事情，无论是关于旅游、同居生活还是在一起后可能发生的故事。

### 4. 巧妙回答常见问题

女生常会问到一些问题，例如年龄、职业、家乡、情感经验等。

你可以提前准备好一套回答惯例，以便在这些问题中快速让女生了解你的整体轮廓。但这并非寻求认同或自我辩解，那样会显得低价值。

目标是在较好地自我展示的同时，保持一定的神秘感，让女生想要知道更多。因此，回答时不要讲得太满，要设置一些“钩子”。

例如，女生问你年龄，你可以说“你猜呀”。或者女生问你职业，你明明经营着几家公司，却可以说“我是个搬砖的”。当她表示不信时，你再逐步透露“做企业的”。这样一点一点地释放信息，保持她的好奇。

聊天切记不要一下子把话题说满。只要有矛盾和好奇存在，女生就永远有兴趣继续挖掘。

### 5. 根据对方类型调整人设

因为每个女生的类型和属性不同，对男人的需求也不同。我们没有必要对每个女人都使用同一套人设。

有的女人喜欢粘人的类型，我在沟通中就会表现得更粘人、像个小孩。有的女人是学生，我聊天时会多用“好的呀”、“冲啊”等带感叹号的幼稚方式，让她觉得我们同频。有的女人喜欢强势的，我就会表现得更霸道。

不要认为这是在伪装或欺骗。每个人的人格都是多样化的。我自己既有幼稚的一面，也有粘人的一面，还有成熟的一面。具体释放哪一面，取决于对方的需求。

## 需要避免的问题

除了运用上述方法，还需要避免以下几个问题。

*   不要在自身价值感未建立的情况下，就急于进行输出信任和建立舒适的行为，这样会很困难。
*   在叙事过程中，不要陷入寻求认同或过于解释的框架。这属于低价值的自我展示，反而会陷入自我证明的陷阱，这两种情况都不可取。

## 总结

本节课中我们一起学习了“叙事”的核心概念与方法。叙事是关于分享你的故事、性格与经历，用以建立信任与舒适感。

我们介绍了五个建立叙事与信任的要点：**1. 以时间与地点为基础展开故事；2. 通过关联话题增强连接感；3. 运用角色扮演与未来规划；4. 巧妙回答常见问题并设置悬念；5. 根据对方类型调整呈现的人设。** 同时，我们强调了要避免在价值感不足时过早叙事，以及避免陷入寻求认同的框架。

只要你能够运用这些方式与女生建立舒适的信任，你将在相处过程中，非常容易地将关系温度悄悄升级，从而为最后的关系升级收尾奠定坚实基础。
