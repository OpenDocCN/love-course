# 《全套课程合集》：leon 3.0：12.叙事

📚 在本节课中，我们将学习如何通过叙事来建立与女生的信任和舒适感。

## 什么是叙事？

叙事是指通过描述人物、性格、思想、经历和行为动机等方式，对故事进行记忆和描述的过程。

## 叙事的重要性

叙事环节直接决定着你与女生之间的舒适度和信任度。很多中阶选手容易越过信任和叙事环节，直接进入诱惑阶段，导致女生不适。

## 叙事建立信任的基本方式

### 1. 时间和地点

你们共度的时间越多，经历的事情就越多，包括线下活动，转场越多，记忆也越多。时间和地点是叙事的两个基本线索。

### 2. 连接感

在聊天中，尝试将话题与你和女生联系起来，增强你们之间的联系感。

### 3. 角色扮演和剧情规划

通过角色扮演和剧情规划，进入新的剧情，描述你们要一起做的事情，如旅游、同居生活等。

### 4. 提前准备回答

女生常问的问题，如年龄、职业、家乡、情感经验等，提前准备好回答，让女生了解你的整体轮廓。

### 5. 保持神秘感

在回答问题时，不要一次性说太多，留下一些悬念，让女生继续想要了解你。

## 建立信任的5个大要点

1. 根据女生的类型和属性，调整你的行为和态度。
2. 表现出你的多样性，释放不同的一面。
3. 避免在价值感未建立的情况下，进行信任和舒适感的建立。
4. 避免在叙事过程中寻求认同或过度解释。
5. 保持神秘感，留下悬念。
