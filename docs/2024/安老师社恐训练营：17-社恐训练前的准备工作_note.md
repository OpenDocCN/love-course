# 安老师社恐训练营：17-社恐训练前的准备工作_note

在本节课中，我们将学习在开始社交恐惧训练前需要完成的两项核心准备工作。清晰的准备能帮助我们的大脑减少内耗，并为后续的实践练习奠定稳固的心理基础。

## 第一步：书面梳理焦虑场景 📖

上一节我们介绍了训练的重要性，本节中我们来看看如何具体开始。首先，大脑若长期接收模糊指令，极易疲劳并导致精神内耗。例如，对工作或生活中的烦心事感到一团糟，周末也无心外出。其根本原因在于这些烦恼未被清晰梳理和“落地”。当它们处于混沌状态时，大脑会陷入越想越糟的恶性循环。

因此，我们需要给大脑清晰明确的指令。实现这一点的关键方式是**书面化**。具体做法是：在纸或电子设备上写下所有引发你焦虑和恐惧的具体场景。书写本身就能带来如释重负的心理体验。

以下是常见的社交焦虑场景分类，可供参考。你无需严格按此分类，它主要在你难以自行回想时提供思路：

*   **公众表达类场景**：例如上台表演、当众讲话、会议发言等。
*   **寒暄互动类场景**：例如与陌生人打招呼、相亲互动、与领导或同事简单交流、同亲戚寒暄、与邻居问好等。
*   **表达意见类场景**：例如难以拒绝他人、不敢提出个人意见、不愿向他人求助、向领导提出加薪、面对冲突或回应批评责备等。
*   **被观察类场景**：例如过马路时感到不自然、上课或开会时被注视、在众目睽睽下行动、做事时感觉旁人在看等。

现在，请拿出笔记本、手机备忘录或电脑记事本，开始书写。内容越具体越好，例如：

*   开会时最后一个进入会议室，感到被注视而不自在。
*   人多时当众讲话，声音颤抖、紧张。
*   与亲戚相处时感到焦虑、紧张、不自在。
*   与异性说话时结巴、脸红。
*   同学聚会时非常焦虑紧张。
*   上下班路上遇到不熟的同事时会刻意躲避。
*   在小区或电梯里碰到邻居时感觉不自然。

目前，你只需要完成“写下来”这个动作，无需做其他事。这是第一项准备工作。

## 第二步：做好心理建设 💭

完成了场景梳理，接下来我们需要进行心理上的准备。首先要明确：社交恐惧是多年形成的**恶性循环条件反射**。克服它并非一朝一夕之事，不能指望明白一个道理或学会一个方法就立刻解决。

习惯的养成至少需要**21天**，而我们要做的是纠正一个根深蒂固的不良习惯。因此，真正克服社恐需要**实践**加上**时间**。理解这一点后，你的心态就不会过于急躁。而心态放松，对于走出社恐恰恰帮助最大。

此外，请回想一下：你的紧张、焦虑是否常源于觉得自己“不够好”，并害怕被人察觉？无论是外在自认的缺点（如身高、长相、胖瘦），还是内在的反应（如脸红、手抖、心慌），我们内心深处常视其为缺点并试图掩饰。

但**越掩饰，就越紧张，表现越不自然**。原因在于，你的注意力和焦点都放在了掩饰缺点上，便没有足够精力去处理当下真正要做的事。因此，心态上我们要做到：**接受它**。无论是外在特征还是内在反应，全然地接受。当你接受后，注意力便不会过度聚焦于此，问题就会简单许多。

最后要明确：在坚持锻炼的过程中，**挫折是必然的**。遭遇挫折时，心态应如此调整：告诉自己“我不要一步登天，只要每天进步一点点”。遇到问题、难题或表现不佳时，不去自责和强求。我们应该思考：**同一件事如何在下次做得更好？** 通过积累经验和反馈，才能获得持续进步。

---

本节课中我们一起学习了社恐训练前的两项关键准备：**书面梳理具体的焦虑场景**，以及**建立“接受自我”和“长期坚持”的正确心态**。做好这些准备后，我们将可以更有信心地开启正式的实践训练。
