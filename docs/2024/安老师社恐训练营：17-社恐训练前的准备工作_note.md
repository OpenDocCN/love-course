# 安老师社恐训练营：17-社恐训练前的准备工作 🧘‍♂️

## 概述
在本节课中，我们将探讨在开始社恐训练之前需要进行的准备工作。

## 准备工作一：明确焦虑场景
以下是明确焦虑场景的步骤：

1. **理解大脑工作原理**：大脑接收模糊指令会非常累，容易造成精神内耗。
2. **书面化焦虑场景**：在纸上写下让你感到焦虑的所有场景。
3. **参考场景分类**：
    - 公众场合：上台表演、当众讲话、会议发言等。
    - 寒暄互动：与陌生人打招呼、相亲互动、与领导交流、同事闲聊、亲戚问暖、街坊邻居打招呼等。
    - 表达意见：拒绝他人请求、提出意见、申请涨工资、面对冲突等。
    - 被观察：过马路、上课、被注视等。
4. **具体化场景**：例如，开会时最后一个进入会议室感到不自在，人多时当众讲话声音颤抖等。

## 准备工作二：心理准备
1. **理解社恐成因**：社恐是多年形成的恶性循环条件反射。
2. **坚持练习的重要性**：通过坚持练习和锻炼，才能纠正社恐，形成好习惯。
3. **习惯养成时间**：至少需要21天来养成习惯，纠正不良习惯需要更多时间。
4. **心态调整**：接受自己的缺点，不要急于求成，每天进步一点点。

## 准备工作三：接受缺点
1. **理解掩饰的原因**：掩饰缺点说明不接受它们。
2. **注意力转移**：接受缺点后，注意力不会过多放在它们身上，问题会变得简单。

## 准备工作四：面对挫折
1. **接受挫折的必然性**：在坚持锻炼的过程中，挫折是必然的。
2. **心态调整**：不要自责，不去强求，专注于如何改进，积攒经验。

## 总结
本节课中我们一起学习了在开始社恐训练之前需要进行的准备工作。希望这些准备工作能够帮助你更好地开始训练。
