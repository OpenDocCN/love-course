# 《朕哥8套情感合集》：08.朕哥情感 情感认知提升课（二）：25-怎么不经意间撩的男人心痒痒

🎼，如何不经意间撩拨男人的心痒痒呢？关键在于“不经意间”。当你刻意去撩拨一个男人的时候，可能会显得轻浮，不一定能达到效果。所以，最好的方法是**不经意间**去吸引他。

以下是几种不经意间撩拨男人的方法：

### 1. 信仰指数

- **低级**：拉着他的衣角耍赖，然后又大方地让他去做他想做的事。这种方法可以让他觉得你很在乎他，从而提升他的成就感。
- **中级**：给他起昵称并撒娇，让他觉得你很特别，从而增加你们之间的暧昧感。
- **高级**：帮他搭配衣服，整理领结等，让他感受到你们之间的亲密感。
- **顶级**：喂他吃东西，深情地看着他，让他感受到你的重视和关注。

### 2. 眼神交流

通过眼神交流，不经意间展现出你的崇拜和欣赏，让他感受到你的独特魅力。

### 3. 姐弟恋

在姐弟恋中，不要把对方当作弟弟，而是当作哥哥，这样他会更愿意照顾你。

### 4. 画饼

学会给男人画饼，但要引导他为你付出，而不是只享受你给予的。

### 5. 引导男人付出

通过作、闹等方式，让男人在为你付出的过程中迷失方向，从而本能地为你付出。

**总结**：本节课中，我们学习了如何不经意间撩拨男人的心痒痒。记住，关键在于“不经意间”，不要刻意去追求，而是让一切自然而然地发生。
