## 命令二：回答三个神圣问题

### 建立长期确定性

**核心概念**：建立长期确定性，即确保自己在任何情况下都能保持自信和冷静。

**以下是本节内容**：

1. **理解自信的本质**：自信并非一成不变，而是需要通过不断的学习和实践来培养。
2. **明确个人价值观和道德观**：通过定义自己的价值观和道德观，可以更好地理解自己的行为和决策。
3. **了解世界运作的规律**：通过了解世界运作的规律，可以更好地应对生活中的挑战。
4. **设定人生目标**：明确自己的人生目标，可以帮助自己更好地规划未来。
5. **寻找合适的伴侣**：找到与自己价值观和目标相符的伴侣，可以让自己的人生更加幸福。

### 第一部分：我是怎样的人？

1. **明确个人价值观和道德观**：
    - **价值观**：列出自己认为重要的价值观，例如真诚、善良、勇敢等。
    - **道德观**：明确自己的道德底线，例如不伤害他人、尊重他人等。
2. **了解自己的性格特点**：
    - **优点**：列出自己的优点，例如幽默、聪明、有责任心等。
    - **缺点**：承认自己的缺点，并努力改进。

### 第二部分：世界是如何运作的？

1. **了解社会规律**：
    - **社会结构**：了解社会的基本结构，例如家庭、学校、政府等。
    - **人际关系**：了解人际关系的规律，例如信任、尊重、合作等。
2. **了解经济规律**：
    - **市场经济**：了解市场经济的运作机制，例如供求关系、价格等。
    - **宏观经济**：了解宏观经济的基本概念，例如通货膨胀、失业率等。

### 第三部分：我想过怎样的生活？

1. **设定人生目标**：
    - **短期目标**：列出自己短期内的目标，例如学习新技能、完成项目等。
    - **长期目标**：列出自己长期内的目标，例如结婚、生子、退休等。
2. **规划人生路径**：
    - **职业规划**：明确自己的职业发展方向，例如晋升、跳槽等。
    - **生活规划**：规划自己的生活方式，例如居住环境、兴趣爱好等。

### 第四部分：我想和谁共度一生？

1. **寻找合适的伴侣**：
    - **价值观匹配**：寻找与自己价值观相符的伴侣。
    - **性格互补**：寻找与自己性格互补的伴侣。
2. **明确自己的价值主张**：
    - **个人优势**：明确自己的个人优势，例如才华、能力等。
    - **情感支持**：提供情感支持，让对方感受到关爱。

### 总结

通过回答这三个问题，可以帮助自己更好地了解自己、了解世界、规划人生，从而建立长期确定性。希望本节课的内容能够对你有所帮助。
