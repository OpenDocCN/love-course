在本节课中，我们将学习《孙子兵法》九变篇的核心思想，并探讨如何将其灵活运用于现代生活，特别是两性关系的处理中。核心在于理解“变通”与“权衡利害”的智慧。

上一节我们探讨了战略的整体性，本节中我们来看看如何在复杂情况下灵活应变。九变篇的核心思想是：根据不同的具体情况，灵活运用不同的战略战术，不可拘泥于固定模式。

这个章节主要阐述了两个核心思维。

以下是第一个核心思维：

*   **智者之虑，必杂于利害**：明智的思考者，考虑问题必须同时兼顾利与害两个方面。
*   **杂于利而务可信也**：在有利的情况下，充分考虑到不利的因素，计划才能顺利推行。
*   **杂于害而患可解也**：在不利的情况下，积极发现有利的因素，祸患就可以解除。

**公式表达：决策 = 权衡(利， 害)**

这个思维告诉我们，任何决策都应全面分析，看到事物的两面性。

接下来是第二个核心思维：

*   **屈诸侯者以害**：要使诸侯屈服，就要用其最害怕的事情去威胁。
*   **役诸侯者以业**：要驱使诸侯为我效力，就要用危险烦扰的事情去消耗他们。
*   **趋诸侯者以利**：要使诸侯归附，就要用利益去引诱。

**公式表达：影响他人 = {以害屈之， 以业役之， 以利趋之}**

这个思维提供了几种影响他人的策略路径。

那么，这些古老的智慧对现代生活，特别是在处理两性关系上有何指导意义呢？

以下是两点具体的应用指导：

第一，理解事物的一体两面。例如，作为一名普通男性，如果你找到一位年轻漂亮的女友，其反面可能对应着原生家庭条件一般、学历不高或脾气较差等可能性。因为同时具备年轻、漂亮、高学历、好脾气和优越家世的女性极为罕见，且通常匹配门槛极高。因此，当你感到对方脾气不佳时，应认识到这可能是普遍情况，而非对方独有的问题。不必轻易给对方贴上负面标签。她们可能只是比常人更需要安全感与关爱。

第二，认识到经济基础决定上层建筑。在现代社会，想处理好两性关系，甚至主导关系，依靠威胁、吵闹、跪舔乃至暴力恐吓是难以奏效的。因为女性的经济能力已大幅提升，不再完全依赖男性。最有效的方法，是**用情感为对方“造梦”**。即通过提供极高的情绪价值，营造一种美好、被珍视的情感体验。只要你的情绪供给足够到位，即使对方理性上明白这像一场梦，也往往不愿醒来。因为这种持续提供深层情绪价值的能力，在男性中并非普遍，具备者寥寥无几。

**代码描述核心方法：**
```python
def build_relationship():
    核心方法 = “提供情绪价值”  # 而非控制、威胁或讨好
    if 具备造梦能力(情绪供给):

    else:
        return 关系易陷入权力博弈
```

本节课中我们一起学习了九变篇“灵活应变”与“杂于利害”的核心思想。我们明白了任何事物都有两面性，在关系中应全面看待对方；同时，在现代平等基础上，通过提供深层情绪价值来建立吸引，远比试图控制更为有效。掌握权衡之智与造梦之力，是处理复杂关系的关键。
