# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：56、对方犯错你又不想离开，需要自我完成和蜕变的是什么

## 概述

在本节课中，我们将探讨当伴侣犯错，但你又不想离开时，你需要经历的自我完成和蜕变。

## 核心概念

**公式**：自我完成 = 自我认知 + 自我救赎 + 自我成长

## 一、亲密关系中的伤害

以下是亲密关系中可能出现的伤害：

- 忽视：在日常生活中不经意地否定和打压对方。
- 缺乏察觉：自我视角下，难以察觉到对方的需求和感受。
- 直接犯错：出轨、聊骚等直接伤害行为。

## 二、伤害的来源

伤害的来源主要有两种：

- 本性如此：对方天生具有伤害他人的倾向。
- 被逼出来的：长期压抑、攻击和不认同导致对方做出伤害行为。

## 三、伤害的影响

伤害会带来以下影响：

- 内耗：不断消耗自己的精力和活力。
- 恶性循环：伤害自己，开启更多的伤害。
- 活力耗尽：影响健康和幸福感。

## 四、如何应对伤害

1. **转变思维**：将注意力从对方转移到自己，关注自己的感受和需求。
2. **自我救赎**：通过自我认知和自我成长，恢复以前的状态。
3. **原谅**：原谅自己，放下怨恨，重新开始。

## 五、总结

本节课中，我们学习了当伴侣犯错，但又不想离开时，你需要经历的自我完成和蜕变。通过自我认知、自我救赎和自我成长，你可以走出伤害，重新获得幸福。
