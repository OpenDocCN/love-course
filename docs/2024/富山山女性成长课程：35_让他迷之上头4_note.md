# 富山山女性成长课程：35_让他迷之上头4

## 概述
在本节课中，我们将学习如何高情商地结束一场聊天，以及如何通过结束聊天的方式让对方感到意犹未尽。

## 高情商结束聊天的意义
**公式**： 意犹未尽 → 下次再聊
**代码**： 结束聊天时，让对方感到未满足，激发下次交流的欲望。

### 为什么看重结束聊天的方式？
- 聊天过程中，开头和中间都进行得不错，但结束方式不佳，导致聊天无味。
- 高手聊天能让对方感到意犹未尽，从而期待下次交流。

## 如何让对方感到意犹未尽
- 在关键时候结束聊天，兴奋时戛然而止。
- 不要聊到冷场再结束，避免尴尬。
- 避免长时间聊天，以免对方失去兴趣。

## 察言观色，感知对方情绪
以下是几个判断对方是否开始对聊天内容走下坡路的方法：
1. 对方开始用嗯啊、一个字、一个小图片来打发你。
2. 面对面时，对方开始看表、产生急促不安的情绪或看手机。
3. 对方开始报手或瘫软在椅子上，表示累了或觉得无趣。
4. 对方试图打断你，表示对当前话题不感兴趣。

## 约会时如何结束聊天
- 约会时间不宜过长，保持短小精干。
- 让对方感受到你的时间很珍贵，每次相处都是高质量的。
- 结束聊天时，可以早退或找借口离开。

## 非面对面聊天如何结束
- 直接不回信息。
- 用“我睡了”、“我去忙一会儿”等理由结束聊天。
- 隔天再回，表示“昨天不好意思，我去忙了”。

## 勇敢地结束聊天
- 对方聊你不感兴趣的话题，不要附和，用嗯啊结束。
- 对方冒犯你，直接不回信息。

## 总结
本节课中，我们学习了如何高情商地结束一场聊天，以及如何通过结束聊天的方式让对方感到意犹未尽。希望同学们能够掌握这些技巧，在人际交往中更加得心应手。
