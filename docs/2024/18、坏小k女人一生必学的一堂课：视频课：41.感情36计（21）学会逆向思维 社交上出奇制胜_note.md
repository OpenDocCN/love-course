### 逆向思维的应用

#### 1. 超市手机失窃案例
**正向思维**：寻找失窃手机。
**逆向思维**：通过发送短信给失窃手机，引导小偷出现。

```python
# 逆向思维示例代码
def find_stolen_phone(phone_number):
    send_sms(phone_number, "你刚到超市找不到我有事先走。")
    # 假设这里有小偷出现并被抓住的逻辑

# 调用函数
find_stolen_phone("1234567890")
```

#### 2. 大闸蟹购买案例
**正向思维**：购买大闸蟹，发现重量不足。
**逆向思维**：购买更多大闸蟹，夸赞其大且好吃，导致店铺关闭。

```python
# 逆向思维示例代码
def buy_crab_and_complain(crab_shop):
    buy_more_crabs(crab_shop)
    post_positive_review(crab_shop)
    # 假设这里导致店铺关闭的逻辑

# 调用函数
buy_crab_and_complain("大闸蟹店")
```

#### 3. ATM机故障案例
**正向思维**：等待故障修复。
**逆向思维**：假装ATM机多吐出钱，联系客服解决问题。

```python
# 逆向思维示例代码
def fix_atm_issue(atm_machine):
    claim_extra_money(atm_machine)
    call_customer_service(atm_machine)
    # 假设这里解决问题并取回钱的逻辑

# 调用函数
fix_atm_issue("ATM机")
```

#### 4. 孩子玩耍案例
**正向思维**：阻止孩子玩耍。
**逆向思维**：给予奖励，逐渐减少奖励，最终停止奖励，孩子不再玩耍。

```python
# 逆向思维示例代码
def manage_children_playground(playground):
    reward_children(playground)
    decrease_reward(playground)
    stop_reward(playground)
    # 假设这里孩子不再玩耍的逻辑

# 调用函数
manage_children_playground("老人院子")
```
