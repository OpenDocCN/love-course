# 课程08：夫妻道歉的七种语言之治疗性语言 🩹💬

在本节课中，我们将学习夫妻道歉七种语言中的“治疗性语言”。这种语言的核心在于修复关系中的创伤，类似于心理咨询师与来访者沟通时使用的、具有启发性和共情力的对话方式。

## 什么是治疗性语言？

上一节我们介绍了道歉的多种形式，本节中我们来看看其中一种特殊的类型——治疗性语言。

治疗性语言并非日常闲聊或教育指导。它特指在夫妻一方冒犯或伤害了另一方后，用于修复情感创伤的沟通方式。这种语言需要具备启发性，并能切入对方内心，产生共鸣与共情。同时，它需要结合具体场景，将双方带入事件发生的背景中进行交流。

**核心公式**：`治疗性语言 = 共情性 + 场景感 + 启发性`

## 为何需要治疗性语言？

道歉的根本原因在于我们的言行对伴侣造成了伤害。单次的冒犯可能影响有限，但反复的伤害会累积成情感创伤。

以下是几种可能造成创伤的常见言行：

*   **重复性消极话语**：例如，“随便你”、“你总是这样”、“你爱怎么样就怎么样”。
*   **侮辱性或诅咒性言语**：例如，对伴侣家人的恶语相向。
*   **暴力行为**：包括动手打人的“热暴力”和冷漠、疏远的“冷暴力”。

我们的言语、行动，甚至某些表达爱的方式，都可能无意中造成创伤。治疗性语言的目的，正是为了修复这些创伤。

## 如何运用治疗性语言？

治疗性语言的运用，关键在于与伴侣共同确认伤害的具体内容与程度，而非单方面认定。

以下是两个核心步骤：

1.  **主动陈述伤害**：向对方描述你认为你的行为可能造成的伤害。例如：“我觉得我那样做，可能会让你感到不被尊重和孤单。”
2.  **开放确认伤害**：询问对方的真实感受，确认或发现其他未被察觉的伤害。例如：“除了我感到的这一点，你觉得我的行为还给你带来了哪些伤害或不好的感受？”

**核心要点**：伤害的认定权在于被伤害的一方。你认为的伤害（A）与伴侣实际感受到的伤害（B）可能并不相同。治疗性语言是一个**共同探索和确认**的过程。

## 治疗性语言的其他形式

治疗性语言不仅限于严肃的创伤修复对话，也体现在一些积极的补救行动中。

以下是一些具有治疗性效果的例子：

*   **弥补疏忽**：当忘记重要日子（如生日）后，不仅补过，而且更加用心地准备。
*   **直接维护**：在婆媳等问题中，丈夫能当着妻子的面，公正、明确地维护妻子，向自己的母亲表明立场。

这些行动本身就像“安慰剂”或“对症药物”，能直接缓解关系中的“疼痛”，达到治疗的效果。
