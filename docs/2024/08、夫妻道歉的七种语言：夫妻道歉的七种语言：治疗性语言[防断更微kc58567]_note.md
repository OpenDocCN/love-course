# 课程08：夫妻道歉的七种语言之治疗性语言 🩹💬

在本节课中，我们将学习夫妻道歉七种语言中的“治疗性语言”。这种语言的核心在于修复关系创伤，促进深度理解和情感愈合。我们将探讨其定义、应用场景与具体方法。

---

上一节我们介绍了夫妻道歉的多种形式，本节中我们来看看其中一种关键类型：治疗性语言。

所谓治疗性语言，类似于心理咨询师对来访者进行辅导时使用的语言。咨询师的语言通常需要具有启发性，能够切入对方内心，也就是说要有共鸣性或共情性。治疗性语言还需要构建一个场景或背景感，可以称之为场景治疗，能够将双方带入到具体的情境过程中。这种语言就是我们所说的治疗性语言。

例如，我给大家讲课使用的语言不叫治疗性语言，这属于教育类语言，旨在告诉你应该怎么做。而夫妻之间需要的是治疗性语言。治疗性语言主要应用于当你冒犯或伤害了对方时。这种冒犯和伤害可能是无意识的，也可能是你在表达自身不满、痛苦或愤恨时，无意中对爱人造成了创伤。

大家需要知道，道歉之所以必要，是因为对对方造成了创伤。当然，单次的冒犯或不良影响不会形成创伤，但反复的伤害则会累积成创伤。例如，有的人会反复说“随便你啦”、“你爱怎么样就怎么样”或“你总是这样”。这些言语日积月累，便会造成伤害。

创伤的形成通常有两种情况：一是对对方造成严重影响的单次事件；二是日积月累的行为，例如冷战或暴力。暴力分为热暴力和冷暴力，热暴力包括动手打人、怒骂等，冷暴力则包括忽视、冷漠等，两者都会造成创伤。

因此，我们需要对我们所说的话、所做的事、所采取的行动，甚至所表达的爱意中可能造成的创伤，进行“治疗性”的修复。治疗性语言的应用，首先需要与对方共同叙述一件事造成了何种伤害。其次，可以探讨“你认为我这么做还会造成哪些伤害？”或者“我觉得我这么做可能造成了A、B、C几种伤害，你认为还有其他吗？”

有时，人们会担心这是否在给对方暗示或催眠，明明没有某种伤害，却故意说出来。但需要明白，伤害是否存在以及其程度，完全取决于你的爱人。你可能认为是一种伤害，对方可能并不觉得。因此，治疗性语言要求我们与对方进行确认，了解对方真实感受，并探讨是否有其他潜在伤害。这种方法相对较难，但至关重要。

治疗性语言也体现在其他方面。例如，当你忘记爱人生日，但后来记起并补过，甚至加倍补偿，这种行为就具有治疗性。再比如，在出现婆媳问题时，男方当着妻子的面与母亲沟通，明确表达对妻子的支持，这也具有治疗性。

治疗性语言如同药物，有的只是安慰剂，有的直达痛处，有的仅缓解疼痛。因此，治疗性语言也有多种层面和应用方式。当然，这几分钟无法详尽展开，大家掌握核心方法即可。

以下是治疗性语言应用的几个关键步骤：

1.  **共同叙述伤害**：与对方一起描述事件造成的具体伤害。
2.  **探讨潜在伤害**：询问对方是否认为还有其他未被提及的伤害。
3.  **确认对方感受**：完全以对方的感受为准，避免自我臆断。
4.  **采取修复行动**：通过具体行为（如补偿、支持）来体现治疗意图。

---

本节课中，我们一起学习了夫妻道歉中的治疗性语言。这种语言的核心在于**共情、确认与修复**，旨在深入对方内心，理解真实伤害，并通过语言和行动促进关系愈合。掌握这种方法，可以帮助夫妻更有效地处理冲突，修复情感创伤。
