# 雨宸19套合集：01.《爱与救赎，走进回避型依恋的内心世界》：001.回避依恋：不是所有的“回避”都叫回避型依恋

## 概述
在本节课中，我们将探讨回避型依恋这一情感状态，了解其特点、表现以及形成原因。

## 回避依恋的定义
**回避依恋**并非所有“回避”行为，它是一种情感状态，表现为个体在情感关系中倾向于保持距离，不轻易表达情感。

## 回避依恋的形成
回避依恋的形成与个体早期与主要抚养者（通常是母亲）的情感连接方式有关。在婴儿时期（1-3岁），个体与抚养者的相处模式会形成潜意识，影响其成年后的社交关系。

## 回避依恋的类型
1. **安全性依恋**：个体在社交过程中表现出爱与被爱的需求，能够清晰表达自己的感受。
2. **焦虑性依恋**：个体对抚养者的离开非常警惕，但拒绝安抚。
3. **回避性依恋**：个体与抚养者之间未形成亲密的感情连接，内心缺乏情感。
4. **混合型依恋**：个体同时具有以上几种依恋类型的特点。

## 回避依恋的特点
1. **习惯独立，情感淡漠**：个体从小习惯独立，强调边界感。
2. **抗拒亲密，否认真实感受**：个体不喜欢发展快速的关系，会否认自己的真实感受和需求。
3. **害怕面对冲突**：个体很少与人发生正面冲突，倾向于冷战。
4. **相信有条件的爱**：个体希望得到明确的爱，不相信无条件的爱。
5. **随时准备抽身离开**：个体在情感关系中随时准备从一段关系里抽离。

## 回避依恋的内在信念
1. **没有人是爱我的**：个体认为自己不值得被爱，不相信爱。
2. **我不好，所以没有人会爱上真实的我**：个体对自己的价值缺乏正确的认知，追求完美。
3. **我不好，所以没有人会爱上真实的我**：个体认同外在价值标准，追求价值感。
4. **不是所有的不理你，他不一定就是回避依恋**：有些人的不理睬可能只是不喜欢或想逃避。

## 总结
本节课介绍了回避型依恋的定义、形成、类型、特点以及内在信念。了解回避型依恋有助于我们更好地理解他人的情感状态，从而在人际交往中更加和谐。
