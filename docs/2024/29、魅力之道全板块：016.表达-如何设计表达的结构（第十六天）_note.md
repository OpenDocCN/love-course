## 016. 表达-如何设计表达的结构（第十六天）

在本节课中，我们将学习如何设计表达的结构，特别是如何构建一个吸引人的故事。通过掌握故事的开头、中间和结尾，你将能够更有效地传达信息，保持听众的注意力，并让表达更具感染力。

---

我们已经学完了最关键的三个板块。第一个板块帮助你理解什么是社交，以及社交中需要做到的事情。第二个板块帮助你建立自己的内核，在社交中找到自己的位置，而不是被他人影响。第三个板块从操作层面告诉你，在与他人互动时应具备的心态、行动和预期效果。

表达有时是短句，有时需要成段的叙述，甚至需要讲一个故事。许多人的痛点是表达不够好、缺乏自信，或表达内容无法吸引他人。本板块将教你表达的结构、开头和结尾的处理方法，以及如何根据情况调整表达，并应对表达中的突发状况。

### 一个关于勇气的故事

以下故事可能帮助你增强勇气。

有一天，我从公寓骑车回父母家。途中经过一条隧道，我眼角瞥见几个人站在隧道边，似乎在拍摄。经过时，我看到一个穿黑色裙子的女生。我的视线从她的裙子慢慢上移，看到了她的长相——正是我喜欢的类型。

但我没有采取行动，继续骑车。我心想：“她是我喜欢的类型，但隧道车多，不方便搭讪。他们似乎也在忙，打扰他们可能不太合适。”于是，我继续前行。

然而，我脑海中突然出现一个声音：“既然是你喜欢的类型，为什么不去打个招呼？你不是常说‘一切皆有可能’吗？”最终，我被这个声音说服，将自行车停在路边，往回走向他们。

隧道中车来车往，许多骑电动车和自行车的人经过，他们都奇怪地看着我。在车流中逆行步行，确实显得怪异。但我没有多想，继续前行。

走了大约30米后，拍摄团队和那个女生注意到了我。他们的表情变得疑惑，不明白为什么有人会在隧道中逆行走向他们。随着我一步步靠近，我的心跳加速。这是我第一次在隧道中停下来，逆行搭讪一个女生。

尽管紧张，我心中也涌起一股畅快感。因为我知道，那个声音代表我想认识这个女生。在那一刻，我选择做一个有勇气的人。

我走到他们面前，向女生打招呼并说明来意。她的反应是愣住，不知该说什么。场面一度安静。这时，随团的摄影师（一名男性）过来拍拍我说：“行了，就这样吧。”他似乎想打发我走，以免场面尴尬。

但我坚持了一下。我认为女生只是不知如何反应，并非对我有恶感。于是我对她说：“我们可以交换微信，这里不太方便聊天。”我们交换了微信后，我离开前对她说：“如果你有空，下次可以一起喝咖啡。”

当天的故事就此结束。后来，我们确实一起喝了咖啡，并进入了一段关系，持续了一段时间。直到现在，我和她仍有联系。人生中的美好体验或爱情，常在这样的时刻发生。

但如果我当时让念头闪过，为自己找借口——“太不方便、太紧张，以后还会遇到这样的女生”——那么什么都不会发生。但我选择了勇气，上前搭讪，一切皆有可能。这就是我的故事。

### 故事的结构设计

请注意我故事中的几个组成部分。

首先，故事中有几个抓住注意力的地方。例如，开头提到“这个故事可能给你带来勇气”，中间提到“看到了一个感兴趣的异性”，以及后面“男性过来打扰并想打发我走”。这些元素都是故事中调动好奇心和注意力的钩子。

其次，故事有一个高潮部分——我选择勇气并拥有了一段美好感情体验。同时，故事还说明了勇气的重要性。

今天，我们将讨论如何设计故事的结构。一个故事需要有开头、过程和结尾，即故事的开始、中间和结束。

但设计故事时，顺序并非如此。设计时应遵循**结尾 → 开头 → 过程**的顺序。因为故事的结尾通常是重点和主旨所在。即使是纯粹好玩的故事，好笑的高潮也应放在结尾。只有确定了结尾，才能确定开头的时间点，从而确定故事的大致长度。这个顺序还能避免故事过长或跑题。

确定结尾和开头后，再确定故事的中间部分。中间部分是故事中最长的部分，但对故事来说最不重要。许多不善于表达的人说话像流水账，讲故事时困在中间部分，讲述无关紧要的细节，无法推进到结尾。这样讲故事的结果是失去听众的注意力，自己也不知如何收尾。这种表达不吸引人，反而会在交流中减分。

开头除了确定故事开始的时间点，还有另一个作用：抓住对方的注意力，让对方感兴趣，至少保持5到10秒的注意力。我故事的开头让你觉得听了会有帮助，因此你对故事感兴趣。

故事的中间部分一方面是故事的血肉，让听者带入和沉浸；另一方面需要时不时创造小悬念，保持听众的注意力。

故事的结尾是你讲故事的目的，也是时间上的结束。目的可以是纯粹为了好玩、表达个人意见、激励他人等。

我将讲故事作为本章的重点，是因为所有表达或多或少是讲故事的简化版。如果你能讲好故事，普通表达一定没有问题。普通表达中，我们要么说事情、经历，要么说观念。像我刚才的故事，同时包含了事情和观念，并将它们结合。同时，讲故事的时长最长。如果你能在1-2分钟内抓住别人的注意力并表达好一件事，那么10-20秒的普通表达更不在话下。

今天我们先介绍表达故事的结构及每个结构的注意事项。后续课程将具体讲解每个组成部分的处理方法。

### 今日作业 📝

今天的作业是设计一个自己故事的结构。你需要回答以下问题：

1. 你讲这个故事的目的是什么？  
2. 这个故事的高潮部分是什么？  
3. 这个故事要怎么开头？  
4. 讲故事时会有哪些悬念出现，以保持听众的情绪和注意力？  
5. 你如何让这个故事变得有趣并感染对方的情绪？

当你在故事中安排这些元素，并按结尾、开头、中间的顺序编排后，你会发现表达故事时不再流水账。你的故事会有重点。我们都不喜欢听别人毫无中心、无休止地讲述经历或故事，因为不知道何时结束，会失去耐心。只要你学会编排故事结构，就能避免这一点。

今天，你可以选择一个经历来回答上述问题，从而设计一个故事结构。需要注意的是，不要期望一次性设计出完美的故事。尽力回答这些问题，尝试满足要求即可。即使悬念不够悬念，主旨说服力不足，也没有关系。好故事都是慢慢改出来的。重点是通过今天的作业，获得结构化表达故事的能力。
