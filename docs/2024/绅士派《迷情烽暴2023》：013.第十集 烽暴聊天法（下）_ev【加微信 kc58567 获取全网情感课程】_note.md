## 求同存异，接纳弱点

在聊天中，当对方暴露弱点时，如表达自卑或倾诉难处，我们要顺着话题进行沟通，表示理解并接纳。

**公式**：顺着话题 → 沟通 → 理解 → 接纳

## 安慰与鼓励对方

安慰与鼓励对方是正向情绪的一种，公式为肯定过去加表达信任。

**公式**：肯定过去 + 表达信任

## 负面情绪的处理

在处理负面情绪时，让对方有痛感，甩态度才能甩到点子上。

**公式**：极度失望 + 惩罚机制

## 打压技巧

打压技巧适用于涉及利益分配和情感冲突时，让对方做出改正。

**公式**：利益分配 + 情感冲突 → 打压 → 改正

## 情绪价值

情绪价值需要让对方进入我们的价值框架，展示自己的生活质量和价值。

**公式**：展示生活质量 + 价值 → 引导进入框架

## 变现方式

变现分为快速变现模式和长期约会模式，根据对方的需求和反应进行选择。

**公式**：快速变现模式 + 长期约会模式

## 约会流程

约会流程包括邀约、安排行程、表达期待、赴约等环节。

**公式**：邀约 + 安排行程 + 表达期待 + 赴约

## 约会注意事项

约会时要注意安排好行程，表达对对方的重视，并全程安排好对方。

**公式**：安排行程 + 表达重视 + 全程安排
