### 什么是魅力风格？

我之前提到过人与人之间打交道是感受为先的。所有人都会记住别人给他的感觉。那所以魅力风格也是这样子的，社交活动当中，你给别人的感受的不同类型，也就是魅力风格的不同类型。

### 魅力风格介绍

以下是几种不同的魅力风格：

1. **高能量**：你给别人的感觉是你的人的状态是很积极的，说话还有能量，精神很振奋，有活力，有感染力。
2. **真实感**：给别人的感觉是你很真诚，坦然，愿意打开自己，表达真实的自己。
3. **连接感**：给别人的感受是你对于别人充满了好奇心、兴趣以及共情。
4. **坚定果断**：你给人的感觉会是你确信自己要做什么，要说什么，不犹豫，不拖泥带水，行动力很强。
5. **幽默**：你给人的感觉是你很有趣，你乐于分享积极的情绪价值。
6. **淡定**：给别人的感觉就是你很稳定，如同一个黑洞一样，能够承载所有的一切生活中的一切。

### 魅力风格训练方法

以下是每种魅力风格的训练方法：

1. **高能量**：更多的去做社交，在社交当中多主动的去进行输出，让自己习惯一个讲话的状态。
2. **真实感**：在和别人交流的过程当中，要养成打开自己的习惯，分享自己真实的经历、感受和情绪。
3. **连接感**：培养自己对于别人的兴趣，在谈话中去了解别人的经历和价值观，积极倾听，给对方你的注意力，多去挖掘式的去问问题。
4. **坚定果断**：在你生活中去多拿决定，小到你吃什么说什么大到你是如何规划自己的人生的，以及有没有在执行。
5. **幽默**：在生活中要去多做取悦自己的事情，让自己多笑一笑。
6. **淡定**：当你在遇到挑战，遇到痛苦的时候，你要让自己保持冷静，深呼吸，意识到不管发生什么，你的生活都继续的下去。
