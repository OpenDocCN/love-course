# 55、小三心理13堂课：7-第七堂：小三主动说不会破坏我们的家庭，但又给我、孩子、老人送礼物，是什么心理？

## 概述
在本节课中，我们将探讨第三者为何声称不会破坏家庭，却仍然向原配妻子、孩子和老人送礼物背后的心理动机。

## 核心概念
**公式**： 
- 第三者心理 = 存在感缺失 + 恐惧被抛弃 + 想要证明自己
- 第三者行为 = 软压力 + 塑造人设 + 营造和谐氛围

## 第一节：第三者心理分析
以下是第三者声称不会破坏家庭的心理分析：

1. **不逼离婚**： 第三者不强迫已婚男人离婚，以避免直接破坏家庭。
2. **不霸占男人**： 第三者不要求男人断绝与原配的关系，以避免显得过于强势。
3. **不主动干扰妻子**： 第三者避免在妻子面前表现出过于亲密的行为，以减少对妻子的干扰。
4. **存在感缺失**： 第三者因缺乏存在感而需要证明自己的价值。
5. **恐惧被抛弃**： 第三者害怕失去已婚男人，因此保持距离以获得安全感。

## 第二节：送礼物背后的动机
以下是第三者向原配妻子、孩子和老人送礼物背后的动机：

1. **宣告存在**： 通过送礼物，第三者向原配妻子宣告自己的存在，并施加压力。
2. **麻痹妻子**： 第三者通过送礼物，试图麻痹妻子，使其放松警惕。
3. **塑造人设**： 第三者通过送礼物，塑造自己善良、正直的形象。
4. **证明无辜**： 第三者通过送礼物，试图证明自己与丈夫离婚无关，以消除丈夫的顾虑。
5. **讨好老人和孩子**： 第三者通过送礼物，讨好老人和孩子，以获得他们的接纳。

## 总结
本节课中，我们学习了第三者声称不会破坏家庭，却仍然向原配妻子、孩子和老人送礼物背后的心理动机。这些行为背后，是第三者对存在感缺失、恐惧被抛弃和想要证明自己的心理需求。了解这些心理动机，有助于我们更好地应对第三者的行为，并保护自己的婚姻和家庭。
