# 34、米娅情感强势认知：11.学会正确归因（下）

在本节课中，我们将继续探讨如何正确归因，理解成本的概念，并学会如何将其应用于情感生活中。

## 成本的定义

**成本**是指放弃了的最大代价。以下是一些关键点：

- **公式**：成本 = 放弃的最大代价
- **代码**：`cost = max(given_up_options)`

### 成本与沉没成本

- **沉没成本**：指已经发生且不可回收的支出。
- **公式**：沉没成本 = 已发生且不可回收的支出
- **代码**：`sunk_cost = already_occurred_and_irretrievable_expenses`

### 成本与负面感受

- 负面感受不是成本，只有放弃的最大代价才是成本。

## 应用实例

### 电影票的例子

- 当电影不好看时，最合适的做法是离开，因为电影票已经沉没，不再是成本。

### 餐桌上的例子

- 桌上饭菜的制作成本已经沉默，真正值得考虑的是吃和不吃的后果。

### 恋爱中的例子

- 在一段不合适的感情中，成本不是浪费的时间，而是放弃的其他选择空间和可能收获的幸福机会。

## 成本与幸福值

- 成本与幸福值的关系可以用以下公式表示：

  **公式**：净幸福值 = 正面感受 - 负面感受
  **代码**：`net_happiness = positive_feelings - negative_feelings`

### 游泳池与帐篷的例子

- 修游泳池的成本是放弃的帐篷的幸福值，而不是修游泳池的负面感受。

## 成本的决定因素

- 成本的大小由放弃了的最大收入决定。

### 铺位的例子

- 铺位的成本取决于放弃的铺租，而不是房子本身的价值。
