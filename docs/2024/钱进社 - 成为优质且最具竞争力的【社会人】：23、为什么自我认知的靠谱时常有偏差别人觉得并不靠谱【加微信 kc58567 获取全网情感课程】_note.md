# 钱进社 - 成为优质且最具竞争力的【社会人】：23、为什么自我认知的靠谱时常有偏差别人觉得并不靠谱

## 概述
在本节课中，我们将探讨为什么自我认知的靠谱时常存在偏差，以及别人为什么会觉得你并不靠谱。

## 自我认知偏差的原因
以下是导致自我认知偏差的三个主要原因：

### 1. 眼界限制
**公式**：眼界 = 知识 + 经验

*   **解释**：你的眼界决定了你对靠谱的定义。如果你只了解一种赚钱方式或职业，你可能会认为其他方式不靠谱。

### 2. 环境影响
**公式**：环境 = 认知 + 行为

*   **解释**：你所处的环境会影响你对靠谱的判断。如果你在一个环境中混得不错，你可能会觉得自己靠谱，但换一个环境后，别人可能不这么认为。

### 3. 状态影响
**公式**：状态 = 情绪 + 态度

*   **解释**：你的当前状态会影响你对靠谱的判断。如果你情绪低落或态度消极，你可能会对他人产生负面印象。

## 如何定义靠谱
以下是如何定义靠谱的两种方式：

### 1. 认同
**公式**：认同 = 从众心理

*   **解释**：将靠谱定义为认同意味着你需要遵循大众的思维和行为模式。如果你与大众不同，你可能会被认为不靠谱。

### 2. 成功
**公式**：成功 = 结果 + 数据

*   **解释**：将靠谱定义为成功意味着你需要用结果和数据来证明自己的价值。如果你没有结果或数据支持，你可能会被认为不靠谱。

## 总结
本节课中，我们探讨了自我认知偏差的原因以及如何定义靠谱。希望这些内容能帮助你更好地认识自己，并在社会中展现更优秀的自己。
