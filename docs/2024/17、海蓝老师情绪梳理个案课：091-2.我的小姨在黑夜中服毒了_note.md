# 海蓝老师情绪梳理个案课：091-2.我的小姨在黑夜中服毒了

## 概述
在本节课中，我们将探讨一个关于小陈粒的个案，他在外婆家经历了小姨服毒自杀的悲剧，并探讨了其对他的影响和情绪处理。

## 场景一：回家的路上
**以下是场景一：**
- 小陈粒在外婆家回家的路上感到浑身发紧。
- 他看到小陈粒，感到心疼并想帮助他。
- 小陈粒低着头，不想看周围，感到害怕。
- 他闭上眼睛，拿着电筒往前冲。

**核心概念：**
```plaintext
小陈粒感到害怕和孤独，需要关爱和支持。
```

## 场景二：小姨去世的房子的恐惧
**以下是场景二：**
- 小陈粒害怕外婆家的房子和周围的人家。
- 他知道小姨死在外婆家的房子二楼。
- 他害怕经过小姨去世的房子和闺蜜家。

**核心概念：**
```plaintext
小陈粒对小姨去世的房子和闺蜜家有强烈的恐惧感。
```

## 场景三：抢救小姨的场景
**以下是场景三：**
- 小陈粒听到楼上抢救小姨的声音，感到害怕。
- 他想帮助小姨，但不知道如何做。
- 他感到孤独，没有人关心他。

**核心概念：**
```plaintext
小陈粒在抢救小姨的场景中感到害怕和孤独。
```

## 场景四：小姨的遗体被抬走
**以下是场景四：**
- 小陈粒看到小姨的遗体被抬到闺蜜家，感到害怕。
- 他想问妈妈和外婆发生了什么事，但外婆告诉他不要害怕。
- 他感到害怕，但想祈祷小姨能活过来。

**核心概念：**
```plaintext
小陈粒在小姨的遗体被抬走的场景中感到害怕和孤独，但仍然想为小姨祈祷。
```

## 总结
本节课中，我们一起学习了小陈粒在经历小姨服毒自杀的悲剧后的情绪处理。他感到害怕、孤独和痛苦，但仍然想为小姨祈祷。通过这个过程，我们可以看到小陈粒的坚强和善良。
