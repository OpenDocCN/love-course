# 清风老师全套：3.清风《个人修养》（共10期音频）：027.第四课：创业男方法论，让企业家爱你的秘籍

## 概述

在本节课中，我们将学习如何与创业男性建立深入的关系，并让他们对你产生深厚的感情。

## 初创期

**核心阶段**：初创期

**主要判断标准**：年收入0-100万

**核心价值**：观赏价值 + 情绪价值

**相处策略**：

* 初期：不要制造高门槛，适度主动，让对方有安全感。
* 中期：展现自己的优秀，避免提要求，引导对方投入。
* 后期：参与对方事业，逐步掌握股权和财权。

**约会场景**：

* 公开场景：公司附近的商场、咖啡厅
* 互动场景：电竞、剧本杀、陶艺、画室
* 私密场景：电影院、车内、家楼下散步

**情绪价值**：

* 焦虑情绪：提供具体解决方案
* 低落情绪：客观分析优点缺点，提供方案
* 愤怒情绪：认清现实，分析原因，提出提升方向

## 发展期

**核心阶段**：发展期

**主要判断标准**：年收入100万-1000万

**核心价值**：物质条件 + 精神世界

**相处策略**：

* 初期：展现精致生活，追求刺激新鲜感
* 中期：逐步展现体贴，体验不同生活
* 后期：探索人生，引导投资

**约会场景**：

* 公开场景：商场、茶室、咖啡厅
* 互动场景：骑马、射箭、滑翔伞、高尔夫、滑雪、潜水、登山
* 私密场景：农家乐、自驾游

**情绪价值**：

* 焦虑情绪：理解加倾听
* 低落情绪：等待对方主动找你
* 愤怒情绪：安慰理解陪伴

## 稳定期

**核心阶段**：稳定期

**主要判断标准**：年收入1000万以上

**核心价值**：个人成就 + 个人价值

**相处策略**：

* 初期：展现活泼开朗、社交丰富的一面
* 中期：展现乖巧，拒绝他人
* 后期：展现崇拜感，认可和肯定对方

**约会场景**：

* 公开场景：私房菜餐厅、古建筑
* 互动场景：散步、拍照
* 私密场景：艺术私展、私人会所、希望小学

**情绪价值**：

* 焦虑情绪：换个环境，陪伴
* 低落情绪：分享经历，同频共振
* 愤怒情绪：客观分析，从弱者角度给出观点

## 总结

本节课中，我们学习了如何与创业男性在不同阶段建立深入的关系，并让他们对你产生深厚的感情。通过了解他们的核心价值、相处策略和约会场景，你可以更好地与他们相处，并最终赢得他们的心。
