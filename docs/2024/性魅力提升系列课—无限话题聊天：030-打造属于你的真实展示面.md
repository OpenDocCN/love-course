# 性魅力提升系列课—无限话题聊天训练从零起步7天社交达人速成：030-打造属于你的真实展示面（上） 📸

在本节课中，我们将要学习如何在社交软件上构建有效的展示面，特别是如何设置一个能传达高价值的头像。展示面是你在网络世界中的“门面”，决定了他人对你的第一印象和初步判断。



我们之前介绍了个人价值建设的重要性。本节中，我们来看看如何将这种价值在社交软件上有效地展示出来。社交软件是我们日常交流的主要方式之一。



几乎每天都在各种社交软件上与外界互动。但你是否思考过，同样身处虚拟网络世界，为何有人能积累丰富的社交资源，而有人却毫无头绪。



这个道理与线下互动相同。你展示的状态决定了他人是否愿意与你产生交集。在社交软件中，对方无法立刻看到你的真实状态，因此，要赢得关注，就需要一个途径来展现自身优势。



这个途径就是你社交软件的动态，即你的展示面。展示面是我们在社交软件上的门面，也是给人的第一印象。假设某人想了解你，在你们成为好友后，对方第一时间看到的并非你有趣的开场白或话语的价值。



对方一定是先看到你的头像，然后去翻阅你的朋友圈。你可以换位思考，添加一个人后，是否也会先查看对方的动态，以判断其为人。



因此，我们必须清楚，展示面在社交软件上的意义至关重要。在社交软件上，人们若想认识你，首先会看你头像传递的价值，其次会根据你发布的朋友圈内容来判断你的价值。



我们想让别人认为我们是某种人，就需要通过展示面来构建相应的形象。现在，我们先谈谈头像的设置。



我见过大量学员最初使用错误的头像，例如明星照、风景照或卡通图片。要知道，在社交软件上，头像是对方识别你的第一个标签，它应带有你的个人属性。为何要放置无关紧要的照片呢？因此，凡是使用与自己无关的照片都是错误的。



那么，照片应该如何设置呢？以下是头像设置的核心要点：



**第一，照片中必须包含自身元素。**
有人使用自己的胳膊、腿、手甚至脚作为头像。这些头像虽包含自身元素，却失去了头像的意义。我们的头像至少应包含自己的相貌，或使用半身照。拍照时光线需充足，亮度要好，能看清五官，这样显得既不过于刻意，也不至于完全看不清脸。



**第二，需特别注意照片中的发型和表情。**
发型应收拾利落，避免像刚起床一样。眼神不应直勾勾地盯着镜头或双眼无神地望着天空。许多男生的头像看似无大问题，发型整洁，衣着有质感，五官端正，但为何看起来仍不舒服或别扭？



原因在于其表情是面对镜头尴尬地笑，让人感觉不自然。因此，我们的表情必须自然，无论是冷漠还是微笑，都应自然流露。



**第三，应使用独照。**
总有人喜欢使用与朋友的合影作为头像。这不但使你在照片中显得很小，对方看到后也无法第一时间辨认出哪个是你。



**第四，除人物外，头像背景也很重要。**
有的男生在自己的宿舍或杂乱的出租屋内进行仰拍。请记住，我们的头像是为了展示高价值，背景中的物品是你价值的辅助，而杂乱无章的背景则会拉低你的价值。头像尺寸很小，因此越突出人物越好。最保险的做法是选择有意义的背景，至少能让对方感觉你是讲究的，品味不至于太差。



当然，也可选择一些高价值的背景。但千万记住，当背景是这些地方时，你的表情和肢体必须足够放松自然。



我们的头像可能长时间不会更换。因此，有条件的话，可以寻找懂摄影甚至专业的摄影师帮你拍摄一组照片，即使花费一些金钱也是非常值得的。



本节课中，我们一起学习了在社交软件上如何构建外在展示面。展示面是我们在社交软件上的门面，也是给人的第一印象。其中，头像是他人接触我们时接收到的第一个信息，因此我们需要用它第一时间传达出自己的高价值。
