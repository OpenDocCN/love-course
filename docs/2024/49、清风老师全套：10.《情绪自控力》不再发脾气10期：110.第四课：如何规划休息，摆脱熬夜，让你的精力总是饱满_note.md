# 清风老师全套：10.《情绪自控力》不再发脾气10期：110.第四课：如何规划休息，摆脱熬夜，让你的精力总是饱满

大家好，我是清风老师。我们宗旨是在10年时间内帮助100万人收获情感幸福。在上一节课程中，我教了你如何规划饮食和休息，以提升自控力。今天，我们将学习如何规划休息，摆脱熬夜。

### 1. 自控力的三个表现

- 良好的执行力
- 对自己行为的良好控制能力
- 良好的情绪管理能力

### 2. 休息不足对自控力的三个影响

1. 执行力降低
2. 做出很多短期选择
3. 变得暴躁易怒

### 3. 休息不足的原因

1. 无法处理焦虑情绪
2. 被手机软件绑架
3. 缺乏合适的休息环境

### 4. 睡眠的五个阶段

1. 刚刚躺下，清醒易醒
2. 浅睡眠，大声可吵醒
3. 深睡眠，小声可吵醒
4. 深度睡眠，脑电波放慢，大脑修复
5. 梦境阶段，重复第一个阶段

### 5. 90分钟睡眠法

- 睡满5个周期，即5个90分钟
- 在第五个周期被唤醒，精力充沛

### 6. 提高睡眠质量的五个步骤

1. 找到自己的睡眠规律
2. 建立睡前的仪式感
3. 降低干扰因素，创造良好的睡眠环境
4. 建立起床的规律
5. 做一些起床后的小动作，让大脑更快进入工作状态
