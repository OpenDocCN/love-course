# 情绪价值课-【阿娇】：第6节 如何高级地夸奖别人

大家好，我是娇姐。本节课我们将探讨如何高级地夸奖别人。😊

**夸奖的艺术**

夸奖别人是简单且快速提升他人情绪价值的方法。然而，夸奖并非仅仅是说“你真好”或“你真棒”，而是需要掌握多种技巧和方法。

**高级夸奖的四个方法**

1. **夸对方的缺点**：每个人都有自己的缺点，一段健康的亲密关系应能接受对方的缺点。通过夸缺点，可以传达出包容和接纳，增加对方的安全感。

   **公式**：夸缺点 = 包容 + 接纳

2. **夸细节，夸具体事情**：避免泛泛的夸奖，从细节入手，具体描述对方的行为或特质。

   **代码**：夸细节 = 具体描述 + 真实感受

3. **感谢对方带来的影响**：在亲密关系中，双方会互相影响。夸奖对方带来的积极影响，可以提升对方的成就感和愉悦感。

   **公式**：感谢影响 = 成就感 + 愉悦感

4. **在别人面前炫耀对方的长处**：发现并夸奖对方的优势，让对方感受到骄傲和自豪。

   **代码**：炫耀长处 = 骄傲 + 自豪

**总结**

通过以上四个方法，我们可以更高级地夸奖别人，让对方感受到真诚和关爱。希望大家在今后的生活中多加练习，运用这些方法。

**本节课我们学习了如何高级地夸奖别人，掌握了四个方法：夸缺点、夸细节、感谢影响、炫耀长处。希望这些方法能帮助大家在生活中更好地与人沟通，传递正能量。**
