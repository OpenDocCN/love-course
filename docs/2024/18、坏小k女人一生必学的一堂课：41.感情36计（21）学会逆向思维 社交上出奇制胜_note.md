# 18、坏小k女人一生必学的一堂课：41.感情36计（21）学会逆向思维 社交上出奇制胜

🎼感情36计第21个技巧，学会逆向思维，在社交上出奇制胜。

### 概述
在本节课中，我们将学习如何运用逆向思维在社交场合中取得意想不到的胜利。

### 逆向思维的应用

#### 1. 超市手机失窃
一人逛超市手机被盗，十分着急。求助工作人员了解情况后，工作人员拿出手机给丢失的手机发送短信。短信内容：“接刚到超市找不到你有事先走。你要的现金5000元，我放在超市寄存箱的09，蜜6666。”不一会儿，小偷被抓。

**公式**：正向思维：寻找失物；逆向思维：利用小偷的心理。

#### 2. 大闸蟹事件
朋友在网上买了8至2.5两的大闸蟹，收到后每只1.5两都不到。找商家居然不理他，于是他去超市买了12至4两的大螃蟹。拍照后发到评论上说卖家的大闸蟹又大又好吃。每只足足有4两，还多送了4只。三天后，那家店铺就关闭了。因为其他的买家都投诉并退货。

**公式**：正向思维：投诉商家；逆向思维：夸大事实，吸引更多顾客。

#### 3. ATM机故障
一女晚上到ATM机存款，碰巧故障，银行卡被吞。联系银行被告知要等到天亮。她灵机一动，致电客服称ATM机多吐出5000元。几分钟后，维修人员赶到。

**公式**：正向思维：等待解决问题；逆向思维：利用客服，制造假象。

#### 4. 老人应对孩子玩耍
一老人很喜欢清净，家里有个大院子经常有小孩来玩，赶也赶不走。后来老人想了个办法，他告诉孩子，以后你们每天来玩儿都奖励1块钱。一个礼拜后，一块变成了5毛，5毛又变成了两毛。后来干脆不给了。孩子们生气的说，不给钱，还要让我们来这玩。从此以后，孩子再也不来了。

**公式**：正向思维：阻止孩子玩耍；逆向思维：利用奖励，逐步减少。

### 总结
本节课中，我们一起学习了如何运用逆向思维在社交场合中取得意想不到的胜利。通过以上案例，我们可以看到，逆向思维可以帮助我们在面对问题时，找到更好的解决方案。
