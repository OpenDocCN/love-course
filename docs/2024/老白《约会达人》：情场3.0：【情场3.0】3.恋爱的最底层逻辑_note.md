### 原始社会与进化心理学

大约在3万年前，地球上大部分人都生活在非洲草原上，生活在部落社会。那时的人和现在的人在基因上的差别极小，几乎一样。

进化心理学告诉我们，人类的身体适应的是远古环境。我们生活在21世纪，但大脑仍停留在石器时代。许多思维模式和潜意识还停留在部落时代。

### 感觉的底层原因

例如，现代人喜欢吃甜食，这与部落时代补充能量有关。同样，进化心理学家解释了为什么大多数男生都喜欢胸部大的女生，这与繁殖有关。

### 两性关系的底层逻辑

女生要生10个小孩需要10年时间，而男生只需要一个晚上。这种繁殖成本的差异决定了不同的择偶方式。

### 男性的繁殖方式：播种

男性会寻找繁殖价值高的女性，以达到广泛传播基因的目的。

### 女性的繁殖方式：选种

女性生育成本高，因此选择方式是“选种”，即选择优秀的伴侣。

### 女生的底层需求

女生的底层需求是“猎人HT”和“靠谱DP”。

- **猎人HT**：优秀的、有能力的男人。
- **靠谱DP**：可靠、负责任的男人。

### 总结

本节课总结如下：

1. 男性的繁殖方式是“播种”，女性的繁殖方式是“选种”。
2. 女生的底层需求是“猎人HT”和“靠谱DP”。
