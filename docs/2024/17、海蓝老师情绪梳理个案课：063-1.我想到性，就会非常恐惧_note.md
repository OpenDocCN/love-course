# 海蓝老师情绪梳理个案课：063-1.我想到性，就会非常恐惧

在本节课中，我们将探讨个案中关于性恐惧的情绪梳理过程。

## 案例背景

个案者海英表示，她对性感到恐惧，这种恐惧影响了她的亲密关系。以下是梳理过程的具体步骤。

## 情绪梳理步骤

### 1. 回顾场景

**步骤描述**：让个案者回忆与男朋友即将发生性关系的场景，并描述当时的身体感受。

**核心概念**：**回忆场景**

```markdown
个案者描述：当我回到那个场景时，我感到厌恶，这种感觉主要集中在喉咙。
```

### 2. 关注身体感受

**步骤描述**：将注意力集中在喉咙部位，感受其不适。

**核心概念**：**关注身体感受**

```markdown
个案者描述：我感受到喉咙难受，想呕吐。
```

### 3. 情绪体验

**步骤描述**：观察喉咙的感受，并描述其情绪体验。

**核心概念**：**情绪体验**

```markdown
个案者描述：我对这个喉咙感到同情，觉得它很可怜，感受不到任何温暖。
```

### 4. 情绪释放

**步骤描述**：让个案者感受喉咙的情绪，并尝试释放。

**核心概念**：**情绪释放**

```markdown
个案者描述：我觉得喉咙很无力，努力想与外界顺畅沟通，但非常孤独、无力。
```

## 总结

本节课中，我们通过情绪梳理的方法，帮助个案者了解了她对性恐惧的根源，并尝试释放了相关情绪。希望这个过程对个案者有所帮助。
