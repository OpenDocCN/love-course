# 17、海蓝老师情绪梳理个案课：092-3.我的小姨已经被家人遗忘_note

在本节课中，我们将通过一个具体的个案，学习如何梳理与逝去亲人相关的复杂情绪和创伤记忆。我们将跟随来访者的叙述，一步步探索其潜意识中的画面与感受，并完成一次内在的对话与和解。

## 课程概述

本节课程记录了一次针对童年创伤记忆的情绪梳理过程。来访者回忆了童年时目睹小姨遗体被放置在他人家的场景，并由此产生了长期的恐惧与不安。通过引导来访者与记忆中的“小姨”对话，我们旨在化解恐惧，理解逝者的情感，并完成未竟的告别。

## 梳理过程

### 回忆初始画面 🖼️

上一节我们介绍了课程背景，本节中我们来看看具体的梳理过程。首先，引导来访者回到最初的记忆画面。

来访者回忆起小姨的遗体被抬到了其闺蜜家中。画面中，小姨**黑黑的**，平躺在房子里，穿着深色衣服，头发很长。当时环境昏暗，房子内部也是黑的，可能因为是晚上，并且遗体可能停放了几日。外婆和妈妈等大人在一旁哭泣，气氛沉重，周围有很多围观的人。由于来访者当时年纪小，在人群中只看到“全是腿”，而焦点始终在屋内那具**黑黑的**遗体上。

当被问及画面中最令人难受的部分时，来访者回答是“**黑黑的**”以及小姨**直挺挺躺着**的样子，恐惧程度评分约为六七分（0-10分制，10分为最强烈）。

### 建立内在对话 💬

在稳定地面对记忆画面后，我们开始引导来访者与记忆中的“小姨”建立对话，以转换视角，理解逝者的状态。

以下是引导来访者想象小姨如有意识，会对其说的话：

*   小姨会说：“**对不起，我吓着你了。**” 她对此感到抱歉。
*   小姨会说：“**我好后悔啊，太冲动了。**”
*   小姨表达委屈：“**他没有得到外公外婆的理解。没有相信他。但是现在家里人都把他忘了。他需要被祭奠。**”
*   小姨给予保护：“**我会保护你的。你不用害怕。**” 并特别提到：“**以后你看到黑，你就知道我在那里保护你。不用怕。**”
*   小姨最后嘱咐：“**好好生活。就不要怕他。不要怕我。**”

听完这些话，来访者感到内心变得平静，不再害怕。记忆中的小姨形象也不再是僵硬地躺着。

### 来访者的回应与告别 ✨

当恐惧化解，爱与连接便得以浮现。来访者开始对“小姨”表达自己真实的情感。

来访者想对小姨说：

*   “**我会拉着他的手。小鱼好舍不得你。你怎么就走了呢？**”
*   “**我不想让你躺在这里。我想要你呀，早点去安息。不要暴露在别人家里被人家看着。我觉得这样不好，不尊重你。**”
*   “**现在想一说，放下对外公的怨恨。我外公是个军人，他很严厉，可能没有去理解你。但是他是爱你的，他是为你好。希望你不要怪外公，能够放下怨恨，轻轻松松的去天堂。**”
*   “**早点安息。**”

来访者描述，说完这些后，小姨接受了命运的安排，神情变得安详。来访者随之产生了一个新的念头：想与外公对话。

### 延伸对话与感悟 🌱

情绪梳理常常牵一发而动全身，与一位亲人的和解，可能引发出对家庭其他成员的沟通意愿。来访者想对外公说：

*   “**不要把小姨放在别人家。他不喜欢在这里被这么多人看，他很不受尊重。他没有安全感，他想回家。把他放在家里，把他送走就好了。**”

来访者感知到，外公内心其实非常难过和后悔。在引导下，外公在沉默后，安排人将小姨的遗体带回了家。看到这个内在画面的转变，来访者感觉“小陈（可能是自称或昵称）很棒，他知道小姨的感受”。

课程最后，来访者提出了一个疑问：“**是不是有那种小姨的一些在跟着我呀？为什么我会知道他的想法？我会有点害怕。**” 这引出了一个重要的心理学观点：**我们对于他人的情绪和想法常常拥有直觉性的感知**，这并不玄妙，而是共情能力的体现。引导者最后问：“**那小姨要跟着你的话，你要怕啥？你怕什么？**” 将问题交还给来访者，引导其深入觉察恐惧的根源。

## 课程总结

本节课中，我们一起学习了一次完整的情绪梳理操作。我们通过回顾创伤画面、建立内在对话、表达未竟情感、完成心理告别等步骤，帮助来访者将关于小姨的**黑色、恐惧的记忆**，转化为一种**被保护、被理解的感受**。这个过程不仅缓解了来访者多年的恐惧，也促进了其家庭内部的情感联结在内心世界的达成。关键在于允许感受流动，完成双向的情感表达与告别。
