# 《自然方法》搭讪自然人：1.1 价值

🎼大家好，欢迎来到《自然方法》搭讪自然人教程的第一节——价值。

### 概述

在本节课中，我们将探讨价值在搭讪和建立人际关系中的重要性，并深入分析价值的三个方面：生存价值、繁衍价值和情绪价值。

### 价值的重要性

**公式**：价值 = 吸引

想要与任何人建立连接，首先你必须具备价值。这不仅适用于与女性的社交，同样适用于男性之间。

### 价值产生吸引

**公式**：价值 = 吸引

如果你想与女性建立一段关系，无论是短期还是长期，吸引是首要任务。在初期，没有比吸引更重要的了。

### 价值的三方面

1. **生存价值**

   生存价值是指能够提高人们生存机会的价值，例如金钱、社会地位、好工作、权利等。

2. **繁衍价值**

   繁衍价值是指能够让自己的基因得到更好传承的东西，例如形象优势、身材、性格特质等。

3. **情绪价值**

   情绪价值是指能够让人感到愉悦、带来好情绪的东西，例如吸烟、吸毒、喜欢小混混等。

### 价值遵循磁力效应

**公式**：价值 = 磁力效应

谁有价值，谁的注意力就会被吸引。例如，一个喜欢肉感的女生，当看到一个肉感的女生时，她的注意力就会被吸引。

### 女性的RS（网状激活系统）

每个女性的RS都不一样，她们关心在乎的价值也不一样。例如，拜金女可能更看重金钱，而普通女性可能更看重开心。

### 世俗价值VS个人魅力

世俗价值是指外在价值，例如身材、长相、财富、社会地位等。个人魅力是指内在价值，例如勇气、自信、社交智慧、情商等。

### 吸引发生瞬间

吸引发生得很快，就像一个开关一样。要么吸引，要么没有吸引。

### 展示价值

在展示价值时，不要刻意去展示，最好是让女人来发现你的价值。

### 总结

本节课中，我们学习了价值在搭讪和建立人际关系中的重要性，以及价值的三方面：生存价值、繁衍价值和情绪价值。在下一节课中，我们将学习框架。再见！
