# Ricky基础私教：07.第七课：价值与吸引

## 概述
在本节课中，我们将探讨价值与吸引之间的关系，以及如何通过行为和语言提升个人价值，增强吸引力。

## 价值与吸引的关系
**公式**：价值 = 吸引
任何高价值的东西都会增加吸引，而低价值则会降低吸引力。

## 价值波动
**例子**：与女生见面前，预期值为70分，见面后可能大幅波动。

## 价值提升与降低的行为
### 降低价值的行为
- 过度讨好
- 共振过多
- 紧张、害怕、唯唯诺诺
- 消极、负面负能量
- 谈论生活不易、仇富等

### 提升价值的行为
- 积极正面
- 轻松有趣
- 勇敢
- 正能量
- 开朗大方
- 从容

## 语言对价值的影响
### 正确的表达方式
- **例子**：邀请女生吃麦当劳时，可以说：“今天的胃很想吃什么什么，今天很想吃麦当劳。”

### 错误的表达方式
- **例子**：谈论在美国的生活中遇到的困难，可能会引起女生的同情，但不会增加吸引力。

## 总结
本节课中我们一起学习了价值与吸引的关系，以及如何通过行为和语言提升个人价值，增强吸引力。记住，无论谈及任何事情，都应保持积极正面，并鼓励他人。这将帮助你成为具备高价值的人，从而增强吸引力。
