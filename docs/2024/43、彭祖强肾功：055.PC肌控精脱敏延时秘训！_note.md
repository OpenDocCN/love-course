# 43、彭祖强肾功：055.PC肌控精脱敏延时秘训！

## 概述
在本节课中，我们将学习彭祖强肾功中的PC肌控制精脱敏延时秘训，通过呼吸模式重建、骨盆活化、脊柱涌动等训练，提升PC肌的力量和耐力，从而达到延时和控制精的目的。

## 呼吸模式的重建
**公式**：逆腹式呼吸 = 吸气时提肛、提睾、缩阴茎、收腹

**解释**：逆腹式呼吸是PC肌训练的关键，通过吸气时收紧提肛、提睾、缩阴茎、收腹这四个点，锻炼PC肌的力量。

## 骨盆的活化
**动作**：骨盆前倾和后倾

**解释**：骨盆前倾和后倾可以锻炼腰胯的灵活性，为后续的PC肌训练打下基础。

## 脊柱的涌动
**动作**：脊柱一节一地涌动

**解释**：脊柱的涌动可以促进阳气的生发，提升整体的身体素质。

## PC肌的爆发力训练
**动作**：吸气时收紧PC肌，呼气时放松

**解释**：通过吸气时收紧PC肌，呼气时放松，锻炼PC肌的爆发力。

## PC肌的耐力训练
**动作**：闭气持续收缩PC肌

**解释**：通过闭气持续收缩PC肌，锻炼PC肌的耐力。

## 三步空间手法
**手法**：摸顶龙、推龟头、转冠状沟

**解释**：三步空间手法是用于脱敏的手法，通过刺激阴茎，使其勃起，然后用呼吸八拍法将其降下来。

## 不脱明手法
**手法**：点按会阴穴、固腕阴茎根部、拽舌睾丸

**解释**：不脱明手法是用于紧急控制精的手法，通过点按会阴穴、固腕阴茎根部、拽舌睾丸，收紧PC肌，控制精的泄出。
