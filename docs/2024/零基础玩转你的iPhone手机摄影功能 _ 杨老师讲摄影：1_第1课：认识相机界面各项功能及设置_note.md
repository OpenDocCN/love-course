# 零基础玩转你的iPhone手机摄影功能 _ 杨老师讲摄影：1_第1课：认识相机界面各项功能及设置

各位同学大家好，我是杨金坤，欢迎大家来参加我们这期的苹果手机摄影课。这门课程主要讲解苹果手机自带相机，以及一些专业软件的实际拍摄操作和场景运用，以帮助大家熟悉苹果手机的拍摄功能。

**课程概述**

本课程适用于大部分苹果手机，部分老款手机可能缺少部分功能。希望通过本课程，大家能更好地发挥苹果手机的拍摄功能。

**第一节：相机界面及功能**

1. **拍摄界面及功能**
   - 点击数字切换焦距，直接点击屏幕上的数字进行切换，拍摄更清晰的照片。
   - 右侧拍摄模式：全景、人像、常规拍照、视频、电影效果。
   - 左侧功能按钮：实况、RAW格式、闪光灯、照片风格、拍摄比例、曝光调节、倒计时拍摄、滤镜、镜头校正、智能HDR。

2. **拍摄设置**
   - 设置相机：进入设置 > 相机，设置视频、慢动作、立体声、保留设置、连拍、扫描二维码、构图、前置摄像头镜像、场景检测、优先快速拍摄、镜头校正、智能HDR。

**第二节：对焦与曝光**

1. **对焦框**
   - 黄色对焦框用于对焦和测光。
   - 对焦框对准区域，该区域景物清晰。
   - 对焦框测光：对准亮部区域，画面偏暗；对准暗部区域，画面偏亮。

2. **曝光调节**
   - 小太阳调节画面曝光：向上拉，画面变亮；向下拉，画面变暗。

**第三节：全景模式**

1. **横拍全景**
   - 竖拿手机，进入全景模式，从左往右转动手机拍摄。

2. **竖拍全景**
   - 横放手机，进入全景模式，从地面逐渐转动到天空拍摄。

**第四节：夜景模式**

1. **夜景模式**
   - 暗光场景下自动开启夜景模式。
   - 手持或使用三脚架拍摄。
   - 调整曝光时间：最长1秒、3秒、5秒或10秒。

2. **星空照片**
   - 野外拍摄，使用三脚架，进入夜景模式，调整曝光时间至最长30秒。

**总结**

本节课介绍了苹果手机相机界面、功能、对焦、曝光、全景模式和夜景模式。希望大家通过学习，能够更好地发挥苹果手机的拍摄功能。下节课我们将继续深入学习。
