# Python基础教程 1.1

## 概述

在本节课中，我们将学习Python编程语言的基础知识，包括变量、数据类型、运算符和基本语法。

## 变量

变量是存储数据的地方。在Python中，变量不需要声明类型，直接赋值即可。

```python
x = 10
y = "Hello, World!"
```

## 数据类型

Python中有多种数据类型，包括数字、字符串和布尔值。

- **数字**：用于表示数值，如整数和浮点数。
  - **整数**：没有小数部分的数值，如`10`。
  - **浮点数**：有小数部分的数值，如`3.14`。
- **字符串**：用于表示文本，如`"Hello, World!"`。
- **布尔值**：用于表示真或假，如`True`和`False`。

## 运算符

运算符用于对变量进行操作。

- **算术运算符**：用于进行算术运算，如加法、减法、乘法和除法。
  - **加法**：`+`
  - **减法**：`-`
  - **乘法**：`*`
  - **除法**：`/`
- **赋值运算符**：用于赋值，如`=`
- **比较运算符**：用于比较两个值，如`==`、`!=`、`>`、`<`、`>=`、`<=`

## 基本语法

- **缩进**：Python使用缩进来表示代码块，通常使用四个空格。
- **注释**：注释用于解释代码，但不影响代码执行。使用`#`符号开始注释。

## 总结

本节课中我们一起学习了Python的基础知识，包括变量、数据类型、运算符和基本语法。在下一节课中，我们将学习Python的控制流语句。
