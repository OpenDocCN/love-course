# 课程01：Python基础入门 🐍

在本节课中，我们将要学习Python编程语言的基础知识。课程将涵盖变量、数据类型、运算符以及如何编写简单的程序。通过本课，你将能够理解Python的基本语法并开始编写自己的代码。

## 什么是Python？ 🤔

Python是一种高级、解释型的编程语言，以其简洁明了的语法和强大的功能而闻名。它广泛应用于Web开发、数据分析、人工智能和科学计算等领域。

上一节我们介绍了Python的概览，本节中我们来看看如何开始使用Python。

## 安装Python与环境配置 ⚙️

要开始编写Python代码，首先需要在你的计算机上安装Python。你可以从Python官方网站下载适合你操作系统的安装包。安装完成后，你可以通过命令行输入 `python --version` 来验证安装是否成功。

## 第一个Python程序 👋

学习任何编程语言的传统都是从“Hello, World!”程序开始。在Python中，这非常简单。

以下是编写第一个程序的步骤：

1.  打开文本编辑器或集成开发环境（IDE）。
2.  输入以下代码：
    ```python
    print("Hello, World!")
    ```
3.  将文件保存为 `hello.py`。
4.  在命令行中运行 `python hello.py`。

你将看到屏幕上输出 `Hello, World!`。`print()` 是一个内置函数，用于将括号内的内容输出到控制台。

## 变量与数据类型 📦

变量是存储数据的容器。在Python中，你不需要预先声明变量的类型，可以直接赋值。

核心概念是**变量赋值**，其基本语法为：
```
变量名 = 值
```

Python有多种内置的数据类型，最常见的有：

*   **整数（int）**： 如 `5`, `-10`, `0`
*   **浮点数（float）**： 如 `3.14`, `2.0`, `-0.5`
*   **字符串（str）**： 如 `"hello"`, `'Python'`，用单引号或双引号包围
*   **布尔值（bool）**： 只有两个值：`True` 和 `False`

你可以使用 `type()` 函数来查看任何变量或值的数据类型。
```python
x = 10
print(type(x)) # 输出：<class 'int'>
```

## 基本运算符 ➕➖✖️➗

运算符用于对变量和值执行操作。Python支持多种运算符。

以下是主要的运算符类型：

*   **算术运算符**： 用于数学运算。
    *   `+` 加法
    *   `-` 减法
    *   `*` 乘法
    *   `/` 除法
    *   `%` 取模（返回余数）
    *   `**` 幂运算
    *   `//` 整除（向下取整）

*   **比较运算符**： 用于比较两个值，返回布尔值（True或False）。
    *   `==` 等于
    *   `!=` 不等于
    *   `>` 大于
    *   `<` 小于
    *   `>=` 大于等于
    *   `<=` 小于等于

*   **赋值运算符**： 用于为变量赋值。
    *   `=` 基本赋值
    *   `+=` 加后赋值（如 `x += 3` 等价于 `x = x + 3`）

## 接收用户输入 ⌨️

一个程序常常需要与用户交互。你可以使用 `input()` 函数来获取用户从键盘输入的信息。`input()` 函数总是将输入的内容作为字符串返回。

```python
name = input("请输入你的名字：")
print("你好，" + name + "！")
```

如果你需要将输入转换为数字（如整数），可以使用 `int()` 函数进行转换。
```python
age = int(input("请输入你的年龄："))
```

## 代码注释与可读性 📝

注释是代码中不会被执行的文本，用于解释代码的作用，提高可读性。在Python中，单行注释以 `#` 开头。

```python
# 这是一个单行注释
price = 100 # 设置商品价格为100元
```

对于多行注释，你可以使用三个连续的单引号 `'''` 或双引号 `"""`。
```python
'''
这是一个多行注释。
可以在这里写更长的说明。
'''
```
