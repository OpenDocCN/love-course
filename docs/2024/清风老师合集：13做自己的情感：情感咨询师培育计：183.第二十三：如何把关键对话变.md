# 清风老师合集：13做自己的情感：情感咨询师培育计：183.第二十三：如何把关键对话变

在本节课中，我们将要学习如何将关键对话的成果转化为具体的行动。上一节我们介绍了如何通过建立共同目标来引导对话，本节中我们来看看如何通过倾听、探寻对方观点，并最终将共识落地为可执行的计划。

## 建立对话基础：探寻他人路径

在沟通中，你可能会遇到对方沉默或生气的情况，你说了一大堆，对方却回应寥寥。这时，你需要探寻对方的“路径”，即了解他们内心的真实想法和信息。迪恩·鲁斯克曾说，说服别人最好的方法就是用你的耳朵认真倾听他们的谈话。

### 倾听的前提准备

以下是有效倾听所需的三个核心心态：

*   **保持最大好奇**：这能帮助你发现对方话语之外的隐含信息。
*   **保持最好真诚**：这是倾听中最重要的环节。
*   **保持足够耐心**：对方可能反应慢，不耐烦会引发争吵反弹。

### 探寻路径的工具：AMPP法

当别人沉默或生气时，我们可以使用AMPP四种工具来安全地了解他们的想法。

**1. 询问 (Ask)**
鼓励别人说出想法最简单的方法是直接邀请。当对话陷入争吵循环时，通过询问可以让对方直接表达，打破僵局。

**示例代码/话术：**
```
“我非常想听听你的观点和想法。”
“如果你有不同的想法，请直接告诉我，我会认真听取。”
```

**2. 映射 (Mirror)**
当询问无效时，可以使用映射。即通过描述你观察到的对方情绪或行为，来证实其感受，像一面镜子让对方看到真实情况。

**核心要点：** 描述时语气必须平静，避免让对方感到被攻击。

**示例代码/话术：**
```
“我看你的表情好像不太对，并不像你说的没事，我有点担心。”
```

**3. 复述 (Paraphrase)**
用自己的话概括对方的观点，以确认理解。关键是要精炼，且不能打断对方。

**4. 抛出 (Prime)**
当以上方法都不奏效时，可以做出一个猜测，抛出一个观点来引导对方开口。这是一种最后的尝试。

## 处理观点分歧：ABC法则

当你引导对方说出想法后，可能会发现对方的观点与你完全对立。面对这种情况，可以使用ABC法则来处理。

**A - 同意 (Agree)**
首先，从双方观点一致的地方开始表示同意。大多数争吵中，双方事实不一致的部分可能只有5%，无需为相同的事情争论。

**B - 补充 (Build)**
在对方观点的基础上，补充你自己的看法。不要说“你错了”，而要说“你说的我理解，但我有一点补充……”。

**C - 比较 (Compare)**
当无法完全同意时，不要判断对错，而是比较双方看法的不同之处。可以说“我们的看法有些不同，我来解释一下我的角度”。

## 从共识到行动：决策与执行

谈话达成共识后，最关键的一步是将决定转化为行动。

### 决策的四种形式

将共识变为决策，通常有四种形式：

1.  **命令**：直接下达指令。效率高但易引起反感。
2.  **咨询**：在决定前，邀请他人（如专家、信任的人）提供建议。
3.  **投票**：通过民主方式做出选择。
4.  **统一**：整合投票结果，形成最终决策。

### 行动计划的四要素

决策之后，必须投入行动。一个清晰的行动计划需要回答以下四个问题：

1.  **谁 (Who)**：谁来执行这个决策？
2.  **做什么 (What)**：具体要执行的内容是什么？
3.  **何时 (When)**：决定执行的时间点是什么？
4.  **如何跟进 (How)**：怎样进行反馈和检查进度？

**公式：行动 = 负责人 + 任务 + 时间点 + 跟进机制**
