# 老白《约会达人》：情场3.0：【情场3.0】 16.追求阶段之走心_note

在本节课中，我们将要学习追求阶段的“走心”环节。这是建立深度连接、化解对方疑虑、并让关系稳固升级的关键步骤。

上一节我们介绍了CG（舒适感建立）和暖男行为，本节中我们来看看如何通过“走心”（THTH）来真正打动对方。

## 核心：回应本质而非问题

当对方提出终极问题时，重点不在于回答问题的本身。

**核心是让对方知道：“我是真的喜欢你。”**

例如，一个常见情况是，女生可能会说：“男生和女生不一样。女生有了男朋友就不会理其他人了。男生有了女朋友，还想着撩妹，觉得自己很厉害，很有成就感。”许多人的回复可能已经不错，但并未抓住关键。

## 问题根源：深度吸引（DP）不足

当你没有做好深度吸引（DP）时，对方会产生无数“废物测试”。

以下是常见的废物测试表现：

*   要求你每天早晚固定问候。
*   要求长时间煲电话粥。
*   因你回复消息稍晚而生气。
*   提出各种现实问题来质疑关系。

例如：“你比我小，我们是不是没结果？”、“异地是不是不合适？”、“家里人要我找个本地的。”

## 走心（THTH）的作用

我们每个人都有自己的生活和工作，不可能时刻保持联系。同样，任何关系都难免遇到现实问题。对方并非不理解这些道理。

她只是不确定你是否真的喜欢她。

这时就需要进行“走心”（THTH）。在合适的时机，让她知道：你虽然嘴上不说，但心里想了很多；你看起来不在意，但其实她在你心里非常重要。

当她确信自己在你心中最重要之后，便会变得非常理解和体贴。

## 如何进行走心表达

走心的核心是：分享那些除了外貌之外，真正打动你的细节和瞬间。要基于你们日常交往中的真实小事。

**关键在于：描述具体事件、你的内心感受以及她给你的触动。**

以下是两个举例说明的表达方式：

**举例一（基于共同经历）：**
“记得上次我们一群人一起做饭吗？你做的那道西红柿炒蛋特别好吃。我小时候妈妈做饭不太好吃，所以特别羡慕别人家妈妈手艺好。当时我就在想，能和你在一起应该会很幸福。”
“还记得去年冬天在西直门天桥下吗？那天特别冷，有个老奶奶在乞讨。你要给她钱，我拉着你说可能是骗子。我们都走了一段路了，你又回头给了她五块钱。你对我说：‘我不知道她是不是骗子，但她年纪这么大，天这么冷，我要是不给，心里会不舒服，被骗就被骗了吧。’那一刻我心里就想，这个姑娘，我一定不能辜负。”

**举例二（表达内心感受，适合发信息）：**
“记得我第一次见你时，你和朋友一起来的，穿着一身黑，有点怕生，不怎么说话。我一直找你聊天，你当时可能对我印象不好，觉得我看起来挺花心吧。其实我只有遇到喜欢的女生才会话这么多。”
“后来我约你出来玩，你居然戴了个假肚子出来，说自己怀孕了要生下来。虽然我当时心里特别难过，但还是一直安慰你别做傻事，安慰了一下午。结果后来去你家，发现你肚子没了，你朋友们都在笑我，我才知道是你们合伙骗我玩。但我心里特别开心。”
“你外表看起来很高冷，但和熟悉的朋友在一起时又很调皮。你总是先为朋友考虑，脑子里有很多天马行空的小想法。感觉和你在一起，每天都会很开心。有些人就是这样，不需要特意做什么，只是待在一起就觉得很舒服。”

## 总结

本节课中我们一起学习了“走心”（THTH）在追求阶段的核心作用。

1.  **目的**：不是解答对方的表面问题，而是传递“真心喜欢”这个核心信息。
2.  **时机**：当对方因深度吸引不足而进行“废物测试”或提出现实质疑时。
3.  **方法**：避开外貌，聚焦于**你们之间真实发生的小事**，真诚地分享**这些事带给你的内心感受和触动**。
4.  **效果**：当对方通过这些具体、真诚的分享，确信自己在你的心中占据重要位置时，她的不安全感会消散，并会转而理解和支持你。

通过真诚的“走心”表达，你能让对方感受到你情感的深度和真实性，从而为关系打下坚实而信任的基础。
