# 18. 辰薇魅力女性训练营：27脱单秘籍 🎉

## 概述
在本节课中，我们将探讨女性如何在情感关系中实现脱单，并分享一些实用的方法和技巧。

## 情感脱单的重要性
**公式**： 情感脱单 = 原生家庭影响 + 亲密关系和谐

### 原生家庭的影响
原生家庭对女性的情感关系有着深远的影响。在7岁之前，我们在原生家庭中的经历会复制到我们7岁之后的生活中。

### 亲密关系和谐的重要性
亲密关系的不和谐比原生家庭对我们的影响更大。很多人进入亲密关系是为了疗愈原生家庭的伤结，但往往事与愿违。

## 择偶观念现状
以下是三种常见的择偶观念：

1. **白日梦型**： 对另一半抱有过高的期望，追求完美。
2. **自卑型**： 因自卑而不敢靠近优秀的男生，或觉得自己不配。
3. **心理抗拒型**： 行动上积极寻找对象，但内心抗拒脱单。

## 如何找到合适的对象
### 列清单
将你对另一半的要求列出来，并量化具体化。

### 寻找合适的场合
去你目标对象经常出现的地方寻找，例如头等舱、高端社交活动等。

### 寻找中间人
找一个值得信赖的中间人帮助你认识合适的人。

### 参加集体活动
通过集体活动一次性认识一群人，提高效率。

## 社交恐惧障碍
### 面对社交恐惧
直面自己的恐惧，带着恐惧去社交。

### 独家秘籍
**眼明手快**： 先抓住机遇，有选择性。
