## 什么是特殊性？
上一节我们概述了课程主题，本节中我们来看看特殊性的定义。特殊性并非简单的“可得性”或“认同”。很多人错误地认为，给予女性特殊性会降低自身价值，这是一个典型的误区。

人与人的关系建立，并非纯粹基于“慕强”或“价值”，而是一个综合的“投产比”考量。例如，你觉得某人非常厉害，但如果他完全不回应你，你们也无法建立关系。反之，有些人可能并非最强，但因为他能给予你更多的关注和特殊对待，你觉得在他身上“投资”回报更高，关系反而得以建立。

**核心公式：关系建立 = 综合价值 × 投产比感知**

女性同样如此。慕强是基因本能，但女性也会理性计算“在你身上能获得什么”。如果你无法给予特殊性，她会觉得在你身上获益甚少。

**核心概念：优质女性需求 = 基础需求 + 尊重需求 + 自我实现需求**

## 如何给予特殊性？—— 行为指南

其次，在何时给予特殊性？答案是：当对方做出了符合你高标准的行为时。

**关键点**：特殊性反馈必须是真诚的，基于对方真实做出的特殊行为。如果对方并无特殊之处你却滥用此技巧，会显得你有很强的操控性和目的性，反而会扣分。女生会认为你对谁都这样，她并不特殊。
