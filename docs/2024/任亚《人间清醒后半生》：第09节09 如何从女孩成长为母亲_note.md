一个女孩成长为成为一个母亲，她生命中有一个很重要的角色发生的改变。从只为自己生活，变成了生命中有一个可曾与我们同呼吸，待在我们的身体里待了将近10个月的孩子。在接下来的余生中，我们都与他有着某些很特殊的关系，尤其是在他成长的早年。

### 1.1 最重要的改变

那么，对于一个女孩成长为成为一个母亲来讲，最重要的改变以及最重要的事情是什么？

### 1.2 自我关注

最重要的事情永远不是外面，而是在自己身上。当我成为一个妈妈之后，我第一个要注意自己的身体。因为我的身体经过大量的激素的改变，从怀孕到哺乳，然后到再恢复。我们的身体发生巨大的改变过程中，我觉得第一我要格外的注意自己的身体，格外的爱惜好自己的身体。

## 2. 情绪管理

### 2.1 情绪稳定的重要性

第二件事情呢，我会觉得当我成为一个母亲，我觉得我最核心的除了照顾自己的身体，还有我要照顾好自己的情绪。什么是一个好母亲？经常会有人问我什么算是一个好母亲？其实一个好母亲的标准就是一个情绪稳定的母亲。

### 2.2 母亲情绪对孩子的影响

孩子在漫长的从幼小长大的过程中，他是吸收性的心灵，他会吸收无条件的吸收周围的氛围和能量。那么其实母亲的情绪对他的影响是非常大的。
