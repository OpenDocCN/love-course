# 《社交软件聊天课》第七节：正确处理对方的防御

🎼hello你好，我是Ryan。hello我是梁叔。欢迎来到我们社交软件聊天的第七节课。

### 1. 女生行为分析

上节课我们看到了女生将自己置于弱者位置，可能是自恋，也可能是投射，也可能是希望被拯救。记住，没有生命需要对另一个生命负责，也没有必要去拯救别人。要识别自己在做什么，搞清楚对方是否在勾引或寻求同情。

### 2. 安全感与推拉

女生推开你一步，你推三步，她会感到极大的安全感。满足她的需求后，你无需安慰她。弱者不需要可怜，因为他们是独立的个体，选择以这种方式呈现自己。尊重这种状态。

### 3. 适格与安全感

当一个女生适格你时，你也要适格她。她推开你，可能是在防御，需要安全感。满足她的需求后，你们才能继续关系。

### 4. 推拉原理

女生试探你对她的可能性，试探你对她的筹码。推开她，让她感到尴尬，然后她可能会追上来。这就像你追我打的感觉。

### 5. 控制与不安全感

女生在聊天中试图控制地位，通常表现为没有自信、内心不安全。她们可以大方表达，但往往因为不安全感而控制。

### 6. 推开与吸引

推开女生，增加吸引力。不要害怕推开，因为这样可以让她感到尴尬，从而增加吸引力。

### 7. 好人变坏

不要做一个太好的人，不要做一个善良的人，不要做一个老好人。允许自己不讨好别人，不迎合别人，展现自我的一面。

### 8. 吸引力的放大

吸引力与外在展示关系不大，更多是通过交流、试探、抽离、防御等过程放大。

### 9. 好人变人类

不要把好人变得坏，而是允许自己不讨好别人，不迎合别人，展现自我的一面。

### 总结

本节课我们学习了如何正确处理对方的防御，包括安全感、适格、推拉、控制、吸引等。希望大家能够将这些原理应用到实际聊天中，提高自己的社交能力。
