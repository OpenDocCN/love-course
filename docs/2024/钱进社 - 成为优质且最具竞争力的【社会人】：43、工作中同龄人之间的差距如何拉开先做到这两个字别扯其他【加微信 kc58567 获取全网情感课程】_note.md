# 钱进社 - 成为优质且最具竞争力的【社会人】：43、工作中同龄人之间的差距如何拉开先做到这两个字别扯其他

## 概述

在本节课中，我们将探讨如何在职场中与同龄人拉开差距，以及如何通过主动性和责任心来提升自己的竞争力。

## 核心概念

**主动**：在多个领域都表现出积极主动的态度，承担更多责任。

**责任心**：愿意付出，愿意承担，对工作认真负责。

## 一、职场新人常见问题

以下是职场新人常见的三个问题：

1. **不想去做**：对工作缺乏热情，不愿意承担责任。
2. **别人要我做什么**：被动接受任务，缺乏主动性。
3. **我自己会去做什么**：主动承担责任，积极完成任务。

## 二、主动性的重要性

**主动**是拉开差距的关键。以下是一些关于主动性的要点：

1. **主动不能分布在一个领域**：在多个领域都表现出积极主动的态度。
2. **主动不是揽活**：主动承担责任，而不是仅仅完成任务。
3. **主动要真诚**：真诚地对待工作，用心去完成每一项任务。

## 三、责任心的重要性

**责任心**是职场成功的关键。以下是一些关于责任心的要点：

1. **愿意付出**：愿意为工作付出时间和精力。
2. **愿意承担**：愿意承担责任，即使遇到困难也不退缩。
3. **对工作认真负责**：对待工作认真负责，确保完成任务的质量。

## 四、如何提升主动性和责任心

以下是一些提升主动性和责任心的方法：

1. **反思自己的工作态度**：思考自己是否积极主动，是否对工作认真负责。
2. **设定目标**：为自己设定明确的目标，并努力实现。
3. **寻求反馈**：向同事或领导寻求反馈，了解自己的不足之处。
4. **不断学习**：学习新知识、新技能，提升自己的能力。

## 总结

本节课中，我们学习了如何在职场中与同龄人拉开差距，以及如何通过主动性和责任心来提升自己的竞争力。希望同学们能够将所学知识应用到实际工作中，不断提升自己，成为更具竞争力的社会人。
