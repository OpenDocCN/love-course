# 雨宸19套合集：19.婚恋情感IP，运营变现课：010.第十节 如何做咨询指导

在本节课中，我们将要学习适用于大多数咨询者的核心提升训练方法，以及如何引导客户进行二次付费。我们将深入探讨期望管理、情绪察觉、共情技巧、聊天指导等实用策略，帮助你更有效地引导客户成长并实现服务价值的延续。

---

上一节我们探讨了咨询的基本框架，本节中我们来看看具体有哪些方法可以帮助客户实现提升。这些方法的核心在于帮助客户反复复盘，看清真实的自我，觉察问题所在，从而循序渐进地形成良性循环。

## 1. 期望管理 🎯

满意度公式为：**满意度 = 事情的结果 - 期望值**。在相同的结果下，期望值越高，最终的满意度就越低，即常说的“希望越大，失望越大”。因此，良好的期望管理至关重要。

以下是进行有效期望管理的三个步骤：

*   **第一步**：了解自己的价值以及对方的实际能力。
*   **第二步**：有意识地设置并管理期望，及时处理不合理的预期。
*   **第三步**：选择比较合适的对象。

具体而言，“了解自己的价值”意味着对自己（如性格、工作、收入、外貌等）有一个客观的评价。可以自我评估，并邀请至少5位朋友进行评估，综合确定。“了解对方的能力”也需通过自我观察和他人反馈来获得客观认知。“处理不合理预期”则是指，例如，期望“对方明天就和我复合”这类明显不切实际的想法。

## 2. 提升情绪察觉能力 🔍

提升情绪察觉能力需要反复练习。关键在于帮助客户更清晰地认知自我与他人。

例如，在“了解自己价值”的环节，就是引导客户对自身各方面进行客观审视。同时，对他人能力的评估也应基于客观观察，而非主观臆断。在此基础上，才能有意识地管理期望，摒弃如“必须立刻复合”这类不合理的要求。

## 3. 共情技巧 🤝

掌握了自我认知，接下来需要学习如何与他人建立深度连接，这就是共情技巧。以下是几种有效的共情方法：

*   **对非语言行为进行反馈**：反馈对方的肢体动作、行为带给你的感受。例如，在电话咨询中，当客户讲述有趣经历时，咨询师发出会心的笑声，就是一种积极的非语言反馈。
*   **对深层情感进行反馈**：对对方的情感进行反馈，寻找共鸣。例如：“我完全理解你现在的感受，因为我过去也有过类似的经历。”这能让对方感到被认同和接纳。
*   **指出冲突性的感受和想法**：每个人都会面临内心冲突。明确指出这种两难困境，可以帮助客户更好地理解自己矛盾的心理状态。
*   **运用类比**：使用比喻来解释复杂感受或处境。例如，当客户执着于要求伴侣改变其本性时，可以比喻为：“你想要吃甜的水果，却买了个橘子，然后一直抱怨它不够甜。为什么不直接选择西瓜或甜瓜呢？”这能让道理更直观易懂。
*   **有目标性的自我表露**：分享咨询师自己与客户相似的经历，以表达理解。这与上述的深层情感反馈类似。
*   **运用媒体进行反馈**：将客户的境遇与某部电影、书籍或流行故事中的角色相联系。围绕媒体角色展开讨论，可以避免直接针对客户本人的尴尬，使交流更放松、更开放。

## 4. 指导客户聊天 💬

在建立了良好的咨询关系后，很多客户在实际互动中仍需要指导，尤其是在聊天方面。如果客户需要手把手教聊天，通常意味着其对挽回对象了解不足、情商有待提高，归根结底是需要提升人际互动能力。

吸引的本质是**调动对方的情绪体验**。在聊天中，我们需要通过自身的情绪感染对方，解读对方的情绪并产生共鸣。

### 聊天节奏的核心：对等原则

聊天节奏最核心的思想是**对等原则**，即尽量保持双方在主动性和内容上的对等。

*   **主动性对等**：主动开启话题的一方通常会暴露更多需求感。因此，最好将双方主动发起聊天的频率控制在**各一半**左右。如果对方几乎从不主动，则应拉长两次主动联系的间隔，以减少过度主动带来的负面影响。
*   **内容对等**：除正事外，聊天内容应遵循对等原则，体现在消息长短、聊天态度和回复间隔上。例如，如果你发了一条长消息，对方只回复几个字，这通常表明对方兴趣不大。此时，较好的选择是简短回复或暂时不回复。
*   **有始有终**：尽量遵循“谁开启话题，谁（在合适时机）结束话题”的原则。结束的时机最好选择在**情绪高点**，而不是等到无话可聊时。这既能给下次聊天留下悬念，也能通过聊天有效带动对方情绪。

**注意**：如果消息发出后对方未回复，不必为了得到回复而持续追问。在挽回中，建立充足的舒适感比满足“渴望被回复”的心理更重要。

### 如何打开话题：上堆、下切、平移

以下是打开和延续话题的三个核心技巧：

*   **上堆**：对对方的话题进行总结归纳，提升到更高、更抽象的层次。例如，对方说“我最近有点焦虑”，你可以回应：“是啊，最近焦虑的人好像特别多，可能是疫情让大家都有点压抑。”这就是将个体问题上升到社会共性层面。
*   **下切**：将话题细化，聚焦到更具体的事物、场景或感受上。例如，对方说“泰国挺好玩的”，你可以回应：“是的，芭提雅很棒。我记得那里有个酒吧，氛围特别独特……”这就是把“泰国”这个大集合，细分到“芭提雅的某个酒吧”这个具体点上。
*   **平移**：将话题转移到与之相关的另一个话题上，以打破僵局。例如，接着“焦虑”的话题，你可以说：“你知道吗，人在焦虑时感官会放大。你有没有觉得自己察觉别人情绪的能力变强了？”

**沟通要点**：多数女性聊天时倾向于**下切**（细化），而多数男性倾向于**上堆**（总结）。一次良好的沟通需要这三种模式交替进行。简单口诀是：**多下切，少上堆，话题尽头就平移**。可以遵循“2-3个下切后接1个上堆，上堆后紧跟1个下切或平移以开启新话题”的节奏。

## 5. 客户的二次付费 💰

当客户认可你的价值，并觉得你能持续帮助其解决问题时，自然会产生二次付费的意愿。这种意愿很大程度上取决于你在首次指导过程中的用心程度以及与客户建立的关系深度。

要与客户拉近距离，促进续费，可以尝试以下方法：

*   在感觉续费有困难时，向客户清晰描述接下来的指导计划和可能达成的积极效果。
*   通过朋友圈等渠道，宣传新的优惠活动或服务模式，吸引老客户了解二次付费的可能。

**重要原则**：务必坚守底线。绝不能为了促使客户交费而过度承诺或夸大效果。这样做最终只会损害双方利益，害人害己。
