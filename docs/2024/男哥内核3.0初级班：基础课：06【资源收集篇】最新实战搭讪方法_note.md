# 男哥内核3.0初级班：基础课 06【资源收集篇】最新实战搭讪方法

## 概述

在本节课中，我们将学习最新的实战搭讪方法，包括准备工作、心态调整、实战技巧等。

## 准备工作

### 形象改造

*   **核心**：肢体语言是搭讪的核心，而非言语。
*   **外形**：保持良好的形象，包括发型、服装等。
*   **发型**：选择合适的发型，如3AM或to face。
*   **服装**：选择合适的服装，展现个人风格。

### 心态调整

*   **核心**：心态决定成败，良好的心态是成功的关键。
*   **方法**：
    *   **聊机法**：与朋友一起搭讪，互相指派，用金钱奖励驱动。
    *   **问路法**：以问路为借口，降低焦虑感。
    *   **十连枪**：连续10次被拒绝，强化自己的心理承受能力。

## 实战技巧

### 第一句话

*   **核心**：嘿，你好。
*   **作用**：引起对方注意，表明自己的存在。

### 第二句话

*   **核心**：陈述状态，而非需求。
*   **例子**：我刚才和朋友聊天，看到你很有气质，所以过来认识一下。

### 第三句话

*   **核心**：给人反应时间，体现自信。
*   **例子**：我刚才和朋友聊天，看到你很有气质，所以过来认识一下。

### 假性时间限定

*   **核心**：表明自己时间有限，增加紧迫感。
*   **例子**：我朋友还在那边等我，只给了我两分钟的时间，我加你个微信吧。

### 收号技巧

*   **核心**：引导对方扫二维码，增加联系机会。
*   **例子**：你扫我吧，没事，那个我现在扫不了你啊，这个你加我微信号吧。

本节课介绍了最新的实战搭讪方法，包括准备工作、心态调整和实战技巧。通过学习这些方法，可以帮助大家提高搭讪成功率，找到心仪的另一半。
