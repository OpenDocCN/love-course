# 清风老师合集：13做自己的情感：情感咨询师培育计：169.第九课：防御机制（1）_

在本节课中，我们将要学习心理学中的一个重要基础知识——防御机制。作为咨询师，理解防御机制是认识他人、理解行为模式的关键。我们将探讨防御机制的定义、作用，并介绍四种常见的类型。

## 概述：什么是防御机制？

防御机制，顾名思义，是我们应对自认为的危机或危险时，进行自我防御的一种保护机制。

在精神分析学派的观点中，每个人的人格结构里，“自我”（ego）具备这样的功能。当“自我”感受到来自“本我”的冲突时，就会以预期的方式体验到焦虑，并尝试用一定的策略去阻止它。这个过程就是防御，或称“自我的防御”。

用更通俗的话来说，这就像我们的身体被热水烫到时，会不自觉地叫出声。在叫出声并感觉到痛感的这个过程中，我们其实已经意识到了焦虑，随后所采取的保护性措施，就是我们的自我防御。

这个机制存在于每个人的生活当中，无论是我们的来访者、咨询师本人，还是伴侣，都拥有自己独特的防御机制。

## 防御机制在关系中的体现

上一节我们介绍了防御机制的基本概念，本节中我们来看看它在现实关系中的具体体现。

前段时间常上热搜的薛之谦事件中，最让人气愤的莫过于他称与李雨桐的孩子是“过街老鼠”要被打掉。无数愤怒的网友从此认定薛之谦是“渣男”，他们心目中美好的形象崩塌了。

我无意为他辩驳。从心理学实质角度来说，薛之谦在遇到这样的危机情况时，使用了一种“贬低”的自我保护机制。其本质是通过贬低孩子的合法性与合理性，来强调自身的重要性。当然，这个例子并非完全恰当，因为薛之谦并非我的来访者，我对他了解不深。不过，我们可以从他的行为窥见其内心世界的端倪。

防御机制是我们自我保护的基本功能。在应对危险和刺激时，防御机制就会产生作用。在咨询过程中，防御机制的出现，恰恰是我们进行咨询和工作的重点。

## 咨询案例：当伴侣选择回避

我有一个来访者小西，她分享了一个令她十分痛苦的情况：每次她与爱人争吵，试图解决两人之间的问题时，对方就会回避。她找到我问：“秦风老师，我该怎么办？”

我反问她：“你和伴侣聊过他为什么要逃避这件事吗？”她摇头说：“老师，没有啊。他都逃避了，难道我还要去聊这件事吗？”

我回复她：“是的。对方的逃避，只是他回避解决问题的一种方式。对他自己而言，这是他在面临矛盾时保护自己的一种处理方式。或许在他心中，这种回避还是爱你的表现，这是由他潜意识决定的。他不想用冲突性的语言伤害你，这是他的过去经历和成长过程所塑造的。如果你要解决这个问题，就需要与他沟通，特别是和他一起去面对他所害怕和逃避的冲突。”

那姑娘问我：“老师，那我该怎么做呢？”我回答：“我们现在就在做这件事。我在帮助你认识，你所害怕和担忧的事情其实没有你想的那么复杂和困难。你也可以这样问他：‘亲爱的，是什么原因让你不想去解决我提出的这个问题呢？是我提出的方式不合理，还是你认为这件事不需要解决？’当你们能把问题拿出来谈，这件事就已经得到了良好的沟通。至于结果，往往不一定是一个简单的解决方案。”

在刚刚这个咨询过程中，我作为咨询师使用了一套技术，叫做“觉察”和“面质”。我来分别解释一下这两个词的含义。

*   **觉察**：指我们作为咨询师对防御机制有所理解和认识，能识别两个人在出现某种情况时会使用什么样的防御机制。
*   **面质**：指当面质询的意思，或者说把问题放到台面上来，与来访者进行充分且深入的探讨。在这个过程中，帮助他认识问题，并和他一起去面对。

现在，老师要提出一个有难度的问题了：在这个咨询案例中，我的来访者小西她自己（这位女生）内心的心理防御机制是什么呢？

这个答案我先按下不表。在今天的课程中，我会介绍四种常见的心理防御机制，答案也藏在后面的内容里。希望到课程结尾时，你能有更多收获。

## 四种常见的心理防御机制

前面我们通过案例看到了防御机制的应用，本节中我们将系统地了解四种具体的防御机制。

以下是四种常见的心理防御机制：

### 1. 投射性认同

这是我们首先要介绍的重要心理防御机制。“投射”指投射一个东西的投射，“认同”指我们认同或认可一个人。

投射性认同主要有以下三种情况：
1.  你在另一个人身上看到了太多的自己，以至于对他产生了巨大的曲解。
2.  通过行为和态度使他人受到激发，产生了一些你不喜欢在自己身上出现的情感体验。
3.  通过行为和态度使他人受到激发，产生了一些你不喜欢在自己身上出现的情感体验，并且表现得像那个曾经激发你讨厌情感的人一样。

这些概念听上去复杂，我讲个小故事帮助你理解。

**故事时间**：晚上9点半，你可能刚看完追的电视剧大结局，走出客厅，打开餐厅的灯。明晃晃的灯光下，几小时前吃完晚饭的油腻盘子还留在那里。一直以来你烧饭，男朋友洗碗，但今天他没遵守规定。走进书房，果然他还在电脑前玩游戏。你生气了：“你刚说好玩一小时就去洗的，我电视剧都看完了，你怎么还没去洗？”

男生说：“你走开，不要来烦我。为什么你就不能帮我洗？我上班比你辛苦，下班比你晚，我还要买菜洗菜。玩会儿游戏你就要催催催，我又没说不洗，你就让我多休息会儿都不肯，你是不是逼我在家里做牛做马？”

很显然，男朋友玩游戏又输了。但他说的也没错，你自己也觉得确实没处处考虑他，那么自己好像也没什么资格要求他这么做。何况他也没说不洗碗，再加上既然你爱他，帮他洗一次碗又有什么大不了的呢？于是你默默地把碗收拾掉了。

这个小故事是很多长期相处的伴侣间常见的场面。很多人可能觉得这很正常：对方心情不好，你为他做点什么或改变一下自己，不是应该的吗？

但值得注意的是，这其实并非出自你自愿、出于爱他而心甘情愿去做，而是出于**避免某些东西**，出于**害怕事情发生**。

在这件事中，男生害怕面对自己的挫折，不想为此负责。拿最开始的例子来说，他玩游戏输了，这个过程对他就是一种压力和挫折。挫折意味着能力遭受质疑，而正确面对并接受“我错了，是我能力不足，我需要改变”这样的想法和感受，对当事人来说是难以承受的。

于是，游戏玩输了、职场不顺带来的挫败感及应承担的责任，便通过指责他人，把这些感受“丢”到别人身上，仿佛不是自己的东西一样。这样，他就不必感到不安，也不必为此负责。而情侣中的另一方，则不知不觉承担了本不属于自己的挫败感，变得小心翼翼，甚至觉得有改变自己的责任。

**这就是投射性认同**：你把本来属于自己的情绪，让对方去承受；或者因为在对方身上看到了太多你自己，而变得非常激动。

在后续课程中，我还会就投射性认同做更深入的展开。如果你对这方面知识有浓厚兴趣，老师推荐你阅读布莱克曼所著的《心灵的面具：101种防御机制》。这本书可以帮助你了解目前心理学研究中已确认的101种防御机制。

### 2. 否认

接下来我们看看第二种防御机制：否认。否认是大脑具有的一种不对现实情况进行注意的方式。说人们在否认现实，往往是指他如果不进行防卫，就会看到现实，而现实对他来说极其可怕。

安娜·弗洛伊德（弗洛伊德的小女儿）描述了否认的四种类型：

以下是否认的四种具体类型：
*   **本质否认（对现实的否认）**：即使有大量证据存在，也否认现实。例如，二战时期许多犹太人被屠杀，但居然有作者声称大屠杀事件从未发生过。从临床来说，这种否认就是一种防御机制。比如，酗酒者常常不是减少酒瘾，而是不可能承认自己已染上酒瘾。喝醉的人否认自己喝醉的事实，因为面对自己能力不足、喝醉这件事实，让他感到非常可怕。
*   **行动上的否认**：我曾有一位来访者，他无法相信身边朋友说的话。每一句朋友说的话，他都要想办法亲身验证。比如，朋友说“前面加油站关了”，他不相信，要自己开车过去亲眼看到关闭了，才验证朋友的话。在行动过程中，由于对身边所有人的不信任，他通过这种否认的心理防御机制，给自己制造了一种“必须把事情验证一下才能获得确定答案”的理念。
*   **幻想中的否认**：在互联网上，我们常听到一个词叫“圣母婊”。他们会发表这样的言论：“虽然这个人杀了人，但我感觉他内心深处一定有言不由衷的苦衷，我们要相信他可以改过自新。”这样的言论在网络上会被攻击，但在他们自己的幻想中，认为人是全能的、可以被无限感化，于是便否认了客观已发生的事实。
*   **言语上的否认**：人类是一种“谎言生物”，谎言就是一种典型的言语上的否认。绝大多数情况下，我们说出谎言是不假思索、下意识的。这个场景你一定不陌生：上学时老师问作业有没有做，你没做也会说“做了”；老师问“作业做了怎么没带？”，你说“做好了，我忘在家里了”。当我们面对谎言，即使谎言被发现，也不要用简单的道德化眼光去判断。因为所有的否认都是内心的自我防御机制，这种防御机制在合理情况下，都是对于自我的调节和平衡。

### 3. 合理化

接着要讲的是第三种防御机制：合理化。合理化又叫做“文饰作用”，是个体无意识地用合理的解释来为难以接受的情感、行为、动机辩护，使其可以接受。

在伊索寓言里有两个非常有名的故事，分别对应了合理化的两种类型。

以下是合理化的两种典型表现：
*   **酸葡萄效应**：故事是这样的：一只狐狸在饥饿时看到葡萄架上挂着一串葡萄，想摘又摘不到。临走时喃喃自语说：“葡萄就是酸的。”这是合理化的一种方式，通过否认、贬低的方式来让得不到的东西合理化。
*   **甜柠檬效应**：同样是狐狸，它在森林里寻找食物，又饥又渴，什么都没找到，最后找到了一个柠檬。它安慰自己说：“这个柠檬好甜哪，就是我想要找的那个。”这是合理化的另一种方式，把个体在追求预期目标失败时，冲淡自己内心的不安，百般体现已实现目标的价值，从而提高它。

这种心理现象在一些男性追求失败时特别容易出现。我常听到这样的来访者，在他们追求失败后会问我：“清风老师，为什么她拒绝我时说‘最近不想恋爱’，又马上跟别的男生在一起了？拒绝我时她说不想恋爱，转头就跟别人在一起了，她骗我！你说她是不是人品有问题，不值得我继续追了？”

这种心理现象就是典型的合理化。

### 4. 回避

最后一种要讲的防御机制是回避。从字面上理解，回避是一种逃避或躲避的行为。在心理学中的定义是：远离某些情景，因为它的象征性含义会激发一些冲突性的感受。

前面我提到了小西的案例。在她的案例中，对方就用了回避的机制来处理两人的关系。前面我也问过你：在小西的处理过程中，她究竟用了什么样的防御机制来应对自己的伴侣呢？

事实上，小西她自己所使用的方式，就是**投射性认同**。她把自己无法承受和无法接纳的焦虑投射给了自己的伴侣，然后把自己的伴侣变成了感情中回避并逃避她的那个人。

## 总结与展望

本节课中，我们一起学习了心理学中基础的防御机制知识。我们了解了防御机制是“自我”为应对焦虑和冲突而产生的自我保护策略，它普遍存在于每个人的生活中。

我们重点介绍了四种常见的防御机制：
1.  **投射性认同**：将自身难以承受的情感投射给他人，并诱导对方以某种方式回应。
2.  **否认**：通过否认现实、行动、幻想或言语来逃避令人不安的事实。
3.  **合理化**：为难以接受的行为或结果寻找合理的借口，如“酸葡萄效应”和“甜柠檬效应”。
4.  **回避**：主动远离可能引发内心冲突的情境或话题。

我常说，学习的本质是为了认识我们自己，这不是一句心灵鸡汤，其中包含了一个非常朴素的方法论：一段关系的形成是由关系的双方共同决定的。因此，我们必须先在认识自己的前提下，才能认清这段关系，并改变其中存在的问题。
