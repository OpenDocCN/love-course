# 课程05：成真女生的秘密 🎀 涂口红的讲究

在本节课中，我们将学习女生涂口红的具体步骤与讲究。口红并非简单地涂抹，其中包含多个准备步骤，旨在让妆效更持久、颜色更纯正。我们将逐一拆解这些步骤，并解释其背后的原理。

---

## 口红前的准备：润唇与打底

上一节我们介绍了课程概述，本节中我们来看看涂口红前的准备工作。许多女生在涂口红前会进行唇部护理，以确保后续上妆效果。

涂口红前，很多女生会提前5到10分钟涂抹润唇膏。这样做主要有两个目的。

以下是涂抹润唇膏的两个主要作用：
*   去除嘴唇上的死皮。
*   让嘴唇更湿润。

涂抹润唇膏后，嘴唇会变得湿润。但若直接在此状态下涂口红，膏体难以附着。因此，女生通常会将润唇膏擦掉，再进行下一步。

润唇膏擦掉后，在涂口红之前，有的女生还会给嘴唇擦一层薄薄的粉底液。

很多人会疑惑为何要在嘴唇上擦粉底液。我们之前提到过，粉底液可以起到遮瑕的作用。给嘴唇涂一层薄薄的粉底液，可以在一定程度上掩盖嘴唇原本的颜色。

因为很多人原本的唇色可能偏深或偏红，这会影响口红涂上去的显色效果。例如，如果一支口红的颜色本身不是很红，而女生的唇色很深、很红，那么口红涂上去的效果就会被原本的唇色影响。

所以，在涂口红前在嘴巴上打一层薄薄的粉底液，目的是遮盖原本的唇色，让口红的显色效果更好。

## 口红的涂抹与多功能使用

完成了唇部打底，接下来便是涂抹口红的核心步骤。此外，口红还有一种巧妙的延伸用法。

据我所知，大部分女生涂口红时，直接涂抹的情况比较多。因为上述准备步骤确实比较费时间。

这里额外说明一点，现在很多女生也把口红当做腮红使用。

以下是口红用作腮红的方法：
*   取少量口红。
*   涂抹在脸颊上。
*   用手指将其晕染开。

这样起到的作用和腮红差不多。有时她们甚至觉得这样比使用腮红效果更好。
