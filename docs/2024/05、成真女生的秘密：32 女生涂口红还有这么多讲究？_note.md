# 课程05：成真女生的秘密 🎀 涂口红的完整步骤

在本节课中，我们将学习女生涂口红的完整步骤。口红并非简单地涂抹在嘴唇上，为了达到最佳效果，许多女生会遵循一系列细致的准备工作。我们将逐一拆解这些步骤，并解释其背后的原因。

## 第一步：涂抹润唇膏 💄

涂口红前，许多女生会提前5到10分钟涂抹润唇膏。这样做主要有两个目的：一是软化并去除嘴唇上的死皮；二是为嘴唇补充水分，使其更加滋润。

以下是涂抹润唇膏的具体作用：
*   **去除死皮**：软化干燥的角质层。
*   **深层滋润**：为后续涂抹口红打下湿润的基础，防止口红涂上后起皮。

## 第二步：擦掉润唇膏并打底 🖌️

上一节我们介绍了润唇膏的准备工作，本节中我们来看看后续的关键步骤。润唇膏涂抹并吸收后，需要将其轻轻擦掉。这是因为如果唇膏残留，口红的附着会变得困难，颜色也不易涂匀。

擦掉润唇膏后，在涂口红之前，有些女生还会进行一个步骤：为嘴唇涂抹一层薄薄的粉底液。

你可能会好奇，为什么嘴唇也需要粉底液？我们之前提到过，粉底液的核心作用是遮瑕。在嘴唇上使用粉底液，可以遮盖嘴唇原本的颜色。许多人的原生唇色可能偏深或偏红，这会影响口红涂上后的真实显色效果。

例如，如果一支口红本身的颜色并不十分鲜艳，而女生的唇色非常深且红，那么口红的效果就会被原本的唇色覆盖，无法完美呈现。因此，涂抹一层薄薄的粉底液作为打底，可以有效遮盖原有唇色。

这样做的核心目的，可以用一个简单的公式表示：
> **口红最终显色效果 = 口红本色 - 原生唇色干扰**

通过打底减弱原生唇色的干扰，能让口红的显色更加纯粹、鲜艳。

## 第三步：涂抹口红与技巧延伸 💋

完成打底后，便可以涂抹口红了。当然，据我所知，为了节省时间，大部分女生在日常中会选择直接涂抹口红，这确实是最快捷的方式。

此外，这里补充一个实用技巧：现在很多女生会将口红“一物多用”，将其当作腮红使用。

以下是将其用作腮红的方法：
*   **取用产品**：取少量口红。
*   **涂抹位置**：点涂在脸颊苹果肌处。
*   **晕染手法**：用手指轻轻拍打、晕染开来。

这样做的效果与腮红相似，甚至有时她们会觉得这样打造出的腮红与唇妆颜色更协调统一，效果更好。
