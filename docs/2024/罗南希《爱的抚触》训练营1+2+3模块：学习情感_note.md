# 罗南希《爱的抚触》训练营1+2+3模块：学习情感_note

在本教程中，我们将学习罗南希《爱的抚触》训练营中关于“学习情感”的核心内容。我们将探讨如何理解、识别并有效回应情感，这是建立深层亲密关系的基础。课程分为三个主要模块，我们将逐一解析。

## 模块一：情感认知基础 🧠

上一节我们介绍了课程的整体目标，本节中我们来看看情感认知的基础。理解情感是管理情感的第一步。情感是对外界刺激产生的复杂心理与生理反应。

以下是情感的几个关键组成部分：

*   **生理反应**：例如心跳加速、面部潮红或肌肉紧张。公式可表示为：**情感 ≈ 生理唤醒 + 认知解释**。
*   **主观体验**：个人内在的感受，如快乐、悲伤、愤怒或恐惧。
*   **行为表达**：通过面部表情、肢体语言和言语传递情感信息。

## 模块二：情感识别与标注 🏷️

了解了情感的基础构成后，我们进入情感识别与标注的环节。准确识别自己与他人的情感，是进行有效情感沟通的前提。

以下是进行情感识别的具体步骤：

1.  **暂停与觉察**：在情绪升起时，先暂停反应，将注意力转向内心，体会身体的感受。
2.  **命名情感**：用一个准确的词语描述当前感受，例如：“我感到焦虑”或“你看起来有些失望”。
3.  **探究根源**：思考这种情感是由什么想法或事件触发的。代码式的自我提问可以是：`if 事件A发生: then 我感到B情感`。

## 模块三：情感回应与沟通 🗣️

上一节我们学会了如何识别情感，本节中我们来看看如何恰当地回应情感。有效的回应能验证对方感受，促进理解与连接。

以下是建设性情感回应的核心方法：

*   **共情式倾听**：放下评判，全心关注对方，并通过复述和确认来表达理解。例如：“听起来这件事让你非常委屈。”
*   **“我”语句表达**：陈述自己的感受和需求，而非指责对方。公式为：**当（情境）时，我感到（情感），因为我需要（需求）**。
*   **非暴力沟通**：遵循观察、感受、需要、请求四个步骤进行交流。
