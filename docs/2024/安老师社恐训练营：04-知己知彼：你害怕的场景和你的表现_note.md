## 社恐害怕的社交场景分类

这一讲我们来说一下社恐害怕的社交场景主要分为哪几类。在面对这些场景的时候呢，社恐会有什么样的表现呢？如果你有社交恐惧的话，应该会既熟悉又陌生。熟悉是因为这样的场景实在是太熟悉了。陌生是因为什么呢？

以下是社恐害怕的社交场景分类：

1. **公众场合**：如考试、面试、上台表演、当众朗读、会议发言等。
2. **寒暄互动**：如电梯偶遇、路上遇到不熟悉的同事或客户、与陌生人打交道等。
3. **表达意见**：如拒绝他人请求、提出意见、建议、申请加班补偿、升职加薪、回应批评或责备等。
4. **被观察**：如过马路时、上课或开会迟到、被人注视做事等。

## 社恐的表现

以下是社恐在面对社交场景时的表现：

1. **身体反应**：脸色发红或发白、微笑尴尬、心跳加速、呼吸紧促、肌肉紧张等。
2. **想法**：社交前考虑各种可能出现的尴尬情况，社交中过度关注自身反应，社交后担心被嘲笑或误解。
3. **行为**：回避、逃避视线、找借口不参加社交活动等。
4. **情绪**：紧张、焦虑、害怕、自卑、消沉等。
