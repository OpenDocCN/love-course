# 恋爱心理学：挽回过程中沟通的艺术 - 亲密关系中如何占据主动（三）

## 概述
在本节课中，我们将探讨挽回过程中沟通的艺术，特别是亲密关系中如何占据主动。我们将深入分析陷阱期的概念和策略。

## 陷阱期概述
陷阱期是挖墙角过程中的第二阶段，指的是与对方发生三次亲密接触为止。这里的“亲密接触”包括亲吻、牵手、拥抱或共度夜晚等。

## 陷阱期任务
陷阱期的任务是解除对方对新欢的内疚感。核心行动是强行与对方发生肢体接触，并在接触后避免道歉，以免对方产生愧疚感。

## 对方心态
在陷阱期，对方可能会感到内疚，认为与你的关系可能愧对其现任或新欢。此时，对方的心态并非对你产生埋怨，而是对新欢的愧疚。

## 行动策略
1. **再次约会**：当对方产生愧疚感时，要找机会再次约会，并发生亲密接触。
2. **避免道歉**：即使对方抱怨内疚，也不要道歉，因为这会削弱你的立场。
3. **保持自信**：强调双方都是自由的，不应受制于道德框架。
4. **引导对方**：不要强迫对方做出选择，而是等待他们主动分手。

## 误区
与对方发生肢体接触并不能代表什么，不要强迫对方做出选择。

## 总结
本节课中，我们学习了陷阱期的概念和策略，包括如何解除对方内疚感、保持自信以及避免误区。记住，在挽回过程中，沟通的艺术至关重要。
