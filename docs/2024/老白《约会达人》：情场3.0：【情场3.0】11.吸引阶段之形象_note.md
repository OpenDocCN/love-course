# 老白《约会达人》：情场3.0：【情场3.0】11.吸引阶段之形象

🎼本节课我们将探讨形象改造的重要性。首先，让我们通过一个实验来了解形象的力量。美国UCRAA社会心理学教授曾进行过一个小实验，他让所有学生对他竖起中指。结果显示，在近400人的课堂上，只有十来个学生这样做。当询问这些学生为何竖中指时，他们表示觉得这样做很有趣或好奇。而那些没有竖中指的学生则认为这样做失礼。这个实验被重复了15次，其中13次大部分学生都竖起了中指。只有两次很少的学生竖中指，而这两次实验中教授都穿着西装。这表明，一个人的形象对其权威度有着显著影响。🎼

🎼接下来，我们通过两张教授穿西装和不穿西装的对比图来进一步说明这一点。当教授穿西装时，极少有人对他竖中指；而不穿西装时，很多人竖中指。这表明第一印象的重要性。很多时候，我们和女生见面后就没有下文了。当询问女生为何没有看上你时，她们可能会说“现在还不想谈”或“要多了解了解”，这些听起来合理，但实际上很多时候只是借口。因为她们可能第一眼就没有看上你。🎼

🎼形象包括发型、穿着、肢体语言。

### 1. 发型

- **核心概念**：男人的发型就相当于女生的整容。
- **操作步骤**：
  1. 找一个地级市好一点的理发店。
  2. 在美团或大众点评上找评价最高的大店，选择价格在100元左右发型师。
  3. 根据你的脸型设计一个发型，把头发留的稍微长一点。
  4. 一个月左右去找他们。
  5. 找一个附近理发店，按照照片剪头发。
  6. 使用发蜡和定型喷雾打理头发。

### 2. 护肤

- **核心概念**：男生护肤只需洁面、水乳。
- **推荐产品**：洁面水乳（具体产品名称未提及，可自行在淘宝搜索）。

### 3. 身高

- **核心概念**：身高很重要，内增高是必备单品。
- **操作步骤**：
  1. 购买内增高鞋。
  2. 适应3公分左右的内增高。
  3. 感觉适应后，尝试5厘米的内增高。
  4. 根据鞋子选择合适高度的增高鞋垫。

### 4. 模仿标杆人物

- **核心概念**：模仿那些形象做得很好的人。
- **操作步骤**：
  1. 给自己一个定位。
  2. 在这个类型里找形象完美的人去模仿。
  3. 参考他们在剧中的形象，包括发型、穿着、肢体动作。
