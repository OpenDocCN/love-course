# 线上聊天法，学会如何和喜欢的人聊天：第4课 聊天的3个窗口-明天女生对你的兴趣如何

🎼大家好，欢迎来到咱们聊天课程的第四节。这节课我们主要讲女生对于我们的三大窗口反应。在上一节中我们讲了聊天的目的中有一个叫识别和挖掘窗口。那么这节课我们将对于窗口的三大反应进行更深一步的讲解。🎼好。

### 正面反应

正面反应是女生对于你有一个正面，有一个好的一个反馈。

**具体表现**：

1. **称赞你的某处**：女生称赞你的某个特点，如鼻子好看，做饭好吃。
2. **主动寻找共同点**：寻找生活、兴趣爱好等方面的共同点。
3. **主动开启话题**：在尴尬或聊天中断时主动询问你的近况。
4. **你说无聊话题，对方愿意听**：即使话题不感兴趣，也愿意听你聊天。
5. **主动分享私事**：主动分享日常生活、身边事物。
6. **打听你的私事**：询问你的日常习惯、兴趣爱好等。

**判断长期窗口**：

1. **询问你的日常爱好**：询问你的日常活动、周末安排等。
2. **询问你的情感经历**：询问你的恋爱经历、分手原因等。
3. **询问家人、朋友圈子**：询问你的家庭情况、朋友关系等。

### 中性反应

中性反应是女生普遍的一个状态，即女生对你既不积极也不消极。

**具体表现**：

1. **问什么答什么**：回答问题，但不会过多分享。
2. **你不说话，女生就不说话**：没有话题时不会主动找话题，但会回应你的话题。
3. **不会主动找话题**：需要你主动找话题。

**原因**：

1. 时间点不合适。
2. 矜持礼貌。
3. 心情不好或慢热。
4. 兴趣不大。

### 负面反应

负面反应是女生对你很消极。

**具体表现**：

1. **敷衍回答**：回答简单，或不具体回答问题。
2. **时而回复时而不回复**：有时回复，有时不回复。
3. **否定拒绝抬杠**：否定你的观点，或拒绝回答问题。
4. **不接你的话题**：不回应你的话题。

**原因**：

1. 吸引力不够。
2. 犯了错误，中了雷区。
3. 测试未达到预期。
4. 窗口未把握住。

### 应对策略

**正面反应**：

1. **追击**：加强关系拉近。
2. **调情**：增加情调。
3. **关系升高**：称呼变化。
4. **夸大**：夸大自己的价值。
5. **话题深入**：深入探讨兴趣爱好等话题。
6. **打破融洽**：打破友谊关系，发展恋爱关系。

**中性反应**：

1. **询问对方状态**：了解对方当前状态。
2. **分享自己积极有价值的一面**：展示自己的优点。
3. **持续换话题**：了解对方兴趣。
4. **保持同频**：保持聊天节奏平衡。

**负面反应**：

1. **直接换人**：如果一开始就是负面反应，直接换人。
2. **真诚询问原因**：了解原因。
3. **解释原因**：解释自己的行为。
4. **退一步互动**：保持普通关系。
5. **展示预选**：展示与其他女生的相处画面。
6. **下定义**：明确关系。
