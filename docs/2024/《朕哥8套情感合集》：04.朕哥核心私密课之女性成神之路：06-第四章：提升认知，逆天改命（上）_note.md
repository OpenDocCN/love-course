# Python基础教程 1.1

## 概述
在本节课中，我们将学习Python编程语言的基础知识，包括安装Python、基本语法和变量。

## 安装Python
首先，您需要安装Python。Python可以从其官方网站下载：[https://www.python.org/downloads/](https://www.python.org/downloads/)。

### 安装步骤
1. 访问Python官方网站。
2. 选择适合您操作系统的Python版本。
3. 下载并运行安装程序。
4. 在安装过程中，确保勾选“Add Python to PATH”选项。

## Python基本语法
Python是一种解释型语言，这意味着您可以直接在终端中运行Python代码。

### 变量
变量是存储数据的容器。在Python中，您可以使用以下语法声明变量：

```python
变量名 = 值
```

例如：

```python
a = 10
b = "Hello, World!"
```

### 输出
在Python中，您可以使用`print()`函数输出信息。

```python
print("Hello, World!")
```

### 数据类型
Python有几种内置数据类型，包括数字、字符串和布尔值。

#### 数字
数字包括整数和浮点数。

```python
a = 10  # 整数
b = 3.14  # 浮点数
```

#### 字符串
字符串是由引号包围的文本。

```python
name = "Alice"
```

#### 布尔值
布尔值只有两个值：True和False。

```python
is_valid = True
```
