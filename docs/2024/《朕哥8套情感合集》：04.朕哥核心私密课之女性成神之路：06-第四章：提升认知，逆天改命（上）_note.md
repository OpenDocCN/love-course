# 课程01：Python基础入门 🐍

在本节课中，我们将要学习Python编程语言的基础知识。我们将从最核心的概念——变量和数据类型开始，了解如何在Python中存储和操作信息。掌握这些基础知识是编写任何Python程序的第一步。

## 变量与数据类型 📦

上一节我们介绍了本课程的目标，本节中我们来看看Python中如何存储数据。在Python中，我们使用**变量**来存储信息。你可以把变量想象成一个贴有标签的盒子，标签是变量的名字，盒子里装的就是数据。

Python支持多种**数据类型**，最常见的有：
*   **整数**： 不带小数点的数字，例如 `10`, `-5`, `0`。
*   **浮点数**： 带小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**： 由文本组成的数据，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来，例如 `‘Hello’`, `“Python”`。
*   **布尔值**： 表示真或假，只有两个值：`True` 和 `False`。

以下是创建变量并赋值的例子：
```python
age = 25                     # 整数
price = 19.99                # 浮点数
name = "Alice"               # 字符串
is_student = True            # 布尔值
```

## 基本运算 ➕➖✖️➗

了解了如何存储数据后，我们来看看如何操作它们。Python可以进行各种数学运算和比较。

**算术运算**用于处理数字：
*   加法：`+`
*   减法：`-`
*   乘法：`*`
*   除法：`/`
*   取整除：`//`
*   取余数：`%`
*   乘方：`**`

**比较运算**用于比较两个值，结果是布尔值（`True`或`False`）：
*   等于：`==`
*   不等于：`!=`
*   大于：`>`
*   小于：`<`
*   大于等于：`>=`
*   小于等于：`<=`

以下是运算的示例：
```python
# 算术运算
result = 10 + 3 * 2  # 结果是16
quotient = 17 // 3   # 结果是5
remainder = 17 % 3   # 结果是2

# 比较运算
is_equal = (10 == 10)  # 结果是True
is_greater = (8 > 12)  # 结果是False
```

## 输入与输出 💬

程序需要与用户交互。`input()` 函数用于从用户那里获取输入，`print()` 函数用于向屏幕输出信息。

`input()` 函数总是将用户的输入作为**字符串**返回。如果你需要数字，必须进行类型转换。
`print()` 函数可以输出变量、字符串或表达式的结果。

以下是输入输出的基本用法：
```python
# 获取用户输入
user_name = input("请输入你的名字：")
user_age = input("请输入你的年龄：")

# 注意：input()得到的是字符串，如果需要数字要进行转换
user_age_int = int(user_age)

# 输出信息
print("你好，", user_name)
print("明年你就", user_age_int + 1, "岁了。")
```
