在本节课中，我们将要学习一个核心概念——**自我价值感**。我们将探讨它的定义、重要性，以及如何建立和维持稳定的自我价值感，从而帮助你在社交和情感关系中更加自信和从容。

---

## 概述

自我价值感是我们对自身价值的判断，也是我们内心对自己价值的认可程度。一个人的自我价值感会直接影响他在一段感情中的幸福感。许多人在社交中感到自卑或心态失衡，其根本原因往往是自我价值感的缺失。

## 什么是自我价值感？

自我价值感，简单来说，就是 **“我认为我值得”** 的感觉。它不是由外界评价或他人态度决定的，而是源于内心对自我价值的坚定认可。

上一节我们介绍了自我价值感缺失带来的问题，本节中我们来看看它的具体定义和影响。

一个自我价值感极低的人，可能会把自我价值完全依附在他人身上。例如，一旦伴侣离开或质疑他，他就找不到自己存在的意义。我们需要明白：**自我价值感来源于自己的内心，而不是由别人来决定。**

## 如何获得自我价值感？

要想获得健康的自我价值感，我们需要想通两件事。

以下是两个需要理解的核心观点：

1.  **接纳自身能力的有限性**
    任何人的能力都有一定的范围。我们可以努力争取，但不能强求结果。公式可以表示为：**努力 + 接纳（不可控结果） = 平和心态**。如果用尽全力后依然失败，那就坦然接受这个结果，不要一直纠结于自己为什么做不到。

2.  **不被一时的得失所左右**
    无论是人际关系还是人生际遇，都要相信当下的一切都是最好的安排。很多事情是好是坏，要走下去才知道。过去的挫折（例如创业失败）可能在未来看来是宝贵的成长经历。

想通这两件事后，我们的目标就是保持稳定的自我价值感，这是追求幸福的前提。

## 保持稳定自我价值感的三个技巧

道理明白了，但在实际遇到事情时，心态难免会受影响。那么，如何快速从负面情绪中抽离，保持稳定的自我价值感呢？

以下是三个实用的技巧：

1.  **关注自己**
    关注自己是对自己表达肯定的方式。不要认为“自恋”完全是贬义的，很多时候正是因为你不够“自恋”，才慢慢失去了自我价值感。许多人的生活很邋遢，不注重饮食、着装和个人环境。而你怎样对待自己，别人就会怎样对待你。通过关注自身形象和生活状态来获得价值感，是最简单的方式。

2.  **列成就清单**
    自我价值感低的人，很难感受到自己的价值。我们可以利用成就清单去发现它。具体做法是：每天临睡前，记录下当天完成的所有事情。代码逻辑如下：
    ```plaintext
    if (事件完成 && 自我感觉良好) {
        记录到成就清单；
    }
    ```
    例如：准时上班、完成任务、尝试新发型、与新人互动等。无论事情大小，只要是你做好的，就记录下来。起初清单可能很短，但坚持下来，你会发现成就越来越多，自我价值感也随之提升，从而进入“成就→自信→更多成就”的良性循环。

3.  **与不合理的信念辩论**
    很多时候，低自我价值感源于内心一些“不合理信念”。这些信念的特征是，你认为一些事情是“必须的”、“一定的”。例如：
    *   “我对一个人好，他就必须同样对我好。”（不合理）
    *   “事情一不顺利，我就完蛋了。”（不合理）

    改变这些信念的方法是：将它们一条条写出来，然后质问自己 **“怎么证明这个想法是对的？”**。通过自我辩论，你会发现想法中不合理、绝对化的部分，而这部分正是打击你自我价值感的元凶。

## 总结

本节课中，我们一起学习了**自我价值感**。我们了解到，自我价值感是我们对自身价值的内心判断与认可，它不依赖于外界。获得它的关键在于：接纳能力有限、看淡一时得失。当出现负面情绪时，我们可以通过**关注自己、列成就清单、与不合理信念辩论**这三种方法来维持稳定的自我价值感。

请记住：**自我价值感不是别人给你的，而是你自己给自己的。** 想要成为一个有吸引力的人，首先要做一个拥有稳定自我价值感的人。
