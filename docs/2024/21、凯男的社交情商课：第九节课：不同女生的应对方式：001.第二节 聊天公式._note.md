# 21、凯男的社交情商课：第九节课：不同女生的应对方式：002.第二节 聊天公式.

## 概述

在本节课中，我们将学习如何与不同类型的女生进行舒适聊天，并掌握聊天公式，从而提升社交直觉。

## 聊天公式

聊天公式共分为5步：

1. **开场白**：与女生进行简单的互动，如发送表情或问候。
2. **建立吸引**：通过展示高价值和情绪吸引，让女生对你产生兴趣。
3. **营造安全感**：让女生了解你的基本信息，消除她的担忧。
4. **升高关系**：通过加入性张力，将关系从朋友提升到恋人。
5. **确定邀约**：邀请女生见面，并确定时间和地点。

2. **建立吸引**：使用状态式聊天，展示你的高价值状态。
3. **营造安全感**：重点展示你的基本信息，让她了解你。
4. **升高关系**：加入性张力，但尺度不宜过大。
5. **确定邀约**：直接邀请见面，并确定时间和地点。
