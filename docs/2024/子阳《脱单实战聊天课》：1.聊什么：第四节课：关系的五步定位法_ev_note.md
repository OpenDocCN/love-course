# 子阳《脱单实战聊天课》：1.聊什么：第四节课：关系的五步定位法 🎯

## 概述
在本节课中，我们将学习如何使用“五步定位法”来定位与女生之间的关系阶段，并了解每个阶段应该聊什么内容。

## 五步定位法
五步定位法是指在开始与女生聊天之前，或者在回复她的任何一句消息之前，先思考一个问题：“我俩是当下什么关系？”

### 关系阶段
1. **陌生人**：没有任何交际，只是交换了姓名，互为好友。
2. **网友**：通过点赞等方式互动，但无深入交流。
3. **普通朋友**：了解对方的基础信息，如职业、作息、爱好等。
4. **好朋友**：交流三观，分享掏心窝子的话。
5. **暧昧关系**：存在吸引，聊的时间越来越久，但未确定关系。

### 每个阶段聊天内容
以下是每个阶段你应该与女生聊的内容：

1. **陌生人阶段**：
    - 问候、打招呼
    - 询问兴趣爱好
    - 分享生活琐事

2. **网友阶段**：
    - 互动点赞
    - 询问对方近况
    - 分享有趣的事情

3. **普通朋友阶段**：
    - 交流职业、作息等基本信息
    - 分享生活趣事
    - 询问对方喜好

4. **好朋友阶段**：
    - 交流三观，分享人生感悟
    - 分享掏心窝子的话
    - 互相鼓励、支持

5. **暧昧关系阶段**：
    - 表达关心、体贴
    - 分享私密话题
    - 适当肢体接触
