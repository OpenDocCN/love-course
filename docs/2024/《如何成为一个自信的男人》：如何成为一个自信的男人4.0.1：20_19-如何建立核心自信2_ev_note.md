## 概述
在本节课中，我们将从心理学的角度探讨如何建立核心自信，帮助大家提升自信水平。

## 建立自信的方法

### 1. 获得精神上的配德感
**公式**：体验当下配不上的好东西 → 产生配得感

**解释**：在条件允许的情况下，偶尔体验一些超出自己当前能力范围的事物，可以增强自信。

### 2. 肢体语言
**公式**：扩展身体的坐姿 → 增加睾丸酮和血清素 → 提高自信

**解释**：约会时适当分开双腿，往后靠，可以增加体内激素水平，提升自信。

### 3. 建立高自尊
**公式**：自我评价 → 高自尊

**解释**：将自我评价建立在内在价值上，而非外在条件，可以建立高自尊。

### 4. 恐惧与自卑
**公式**：恐惧 → 负面想法 → 自卑

**解释**：恐惧源于负面想法，通过分析恐惧的根源，可以减轻恐惧感。

### 5. 积极的自我暗示
**公式**：积极的自我暗示 → 增强自信

**解释**：通过积极的自我暗示，可以提升自信。

## 在两性关系中的自信
### 1. 保持朋友关系
**解释**：在没有明确关系之前，保持朋友关系，减少压迫感。

### 2. 不断试探
**解释**：通过试探和制造冲突，获取对方的兴趣指标。

### 3. 伪装自信
**公式**：伪装自信 → 获取兴趣指标 → 真正自信

**解释**：通过伪装自信，逐步建立真正的自信。

## 总结
本节课中，我们学习了如何从心理学的角度建立核心自信，以及如何在两性关系中保持自信。希望这些方法能帮助大家提升自信水平。
