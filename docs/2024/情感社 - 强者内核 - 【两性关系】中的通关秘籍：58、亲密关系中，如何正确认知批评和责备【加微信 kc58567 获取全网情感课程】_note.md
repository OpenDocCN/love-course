## 正确认知批评和责备

### 1. 认识过度反应
**公式**：过度反应 = 过度关注 + 过度解读

**代码**：
```python
def 过度反应(关注度, 解读度):
    return 关注度 > 1 and 解读度 > 1
```

以下是过度反应的例子：

- 看到伴侣给抖音小姐姐点赞，就怀疑对方有外心。
- 伴侣说了一句不想听的话，就认为对方三观有问题。
- 询问伴侣衣服是否好看，对方敷衍回应，就认为对方不爱你。

### 2. 认识焦虑转移
**公式**：焦虑转移 = 责备 + 投射

**代码**：
```python
def 焦虑转移(责备, 投射):
    return 责备 and 投射
```

以下是焦虑转移的例子：

- 你胖了，看到伴侣享受美食，就指责对方。
- 伴侣邀请你一起吃美食，你却因为减肥而指责对方。

### 3. 认识情绪化
**公式**：情绪化 = 心情不好 + 攻击对方

**代码**：
```python
def 情绪化(心情, 攻击):
    return 心情不好 and 攻击
```

以下是情绪化的例子：

- 心情不好时，看伴侣不顺眼，认为对方做的一切都是错的。

### 4. 认识比较心理
**公式**：比较心理 = 比较伴侣 + 比较他人

**代码**：
```python
def 比较心理(比较伴侣, 比较他人):
    return 比较伴侣 or 比较他人
```

以下是比较心理的例子：

- 比较伴侣和别人的伴侣，认为别人的伴侣更好。
