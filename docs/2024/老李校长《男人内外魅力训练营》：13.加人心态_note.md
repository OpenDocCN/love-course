# 老李校长《男人内外魅力训练营》：13.加人心态

🎼 家人心态 🎼

情感的道路充满挑战，充满人性的贪婪与恶。只有勇敢面对，才能成为英雄。勇敢的男人，跟我一起探索情感的奥秘，看透人性的善恶，品尝真挚的情感。

### 核心概念

**公式**： 加人心态 = 高标准 + 敢于尝试 + 多见多识

### 教程内容

#### 1. 不要心软，保持高标准

**以下是加人心态的几个要点**：

- 不要降低标准：社交软件是获取异性资源的主要途径。有些人在面对条件一般的女性时容易心软，但这样会消耗你的资源。
- 真实颜值判断：不要仅仅因为对方留学或高学历而降低标准。最终，你会发现社会属性并不重要。
- 荷尔蒙和多巴胺判断：在社交软件上，用荷尔蒙和多巴胺来判断是否添加对方，不要参考社会属性。

#### 2. 敢于尝试，创新社交渠道

**以下是创新社交渠道的几个方法**：

- 利用微信群：加入各种微信群，寻找优质资源。
- 敢于尝试：不要因为害怕而放弃尝试新的社交渠道。
- 创新思维：思考如何认识更多美女，如何撬动优质资源。

#### 3. 不断优化渠道，提升效率

**以下是提升社交效率的几个方法**：

- 不断优化渠道：寻找更多优质资源，提升社交效率。
- 多见多识：多认识人，才能找到合适的人。
- 永远不知足：不要满足于现状，要不断追求进步。

#### 4. 多见多识，筛选合适的人

**以下是筛选合适人的几个方法**：

- 多见多识：认识更多的人，才能找到合适的人。
- 真实颜值判断：不要被外表所迷惑，要看清一个人的本质。
- 筛选合适的人：在众多人中找到最适合自己的人。

### 总结

本节课中，我们学习了加人心态的重要性。要保持高标准，敢于尝试，多见多识，才能找到合适的人。希望大家将所学知识应用到实际生活中，找到属于自己的幸福。
