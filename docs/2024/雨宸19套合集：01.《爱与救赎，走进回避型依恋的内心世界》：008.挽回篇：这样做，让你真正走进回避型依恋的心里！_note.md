# 雨宸19套合集：01.《爱与救赎，走进回避型依恋的内心世界》：008.挽回篇：这样做，让你真正走进回避型依恋的心里！

## 概述
在本节课中，我们将学习如何挽回回避型依恋的爱人，了解如何通过合适的方法走进他们的内心世界。

## 核心概念
**吸引力公式**：吸引力 = 优点 + 可获得性
**舒适感营造**：舒适感 = 顺从对方 + 尊重对方

## 挽回回避型依恋的方法

### 1. 接受分手
- **接受分手**：分手后不要纠缠对方，多关注自己的生活，提升自己。
- **避免纠缠**：不要表现出过度的依赖和需求，避免让对方感到压力。

### 2. 营造舒适感
- **顺从对方**：不要否定对方的决定，让对方感到轻松。
- **避免提感情**：不要过多提及感情问题，让对方感到舒适。

### 3. 通过求助的方式进入对方生活
- **提出合理要求**：提出不过分的要求，让对方感到被需要。
- **避免提感情**：在求助时不要提及感情问题，让对方感到轻松。

### 4. 找到分手的核心
- **认可对方价值**：让对方感受到自己的价值被认可。
- **回忆优点**：通过别人的口述或自己的行为，让对方回忆起自己的优点。

### 5. 理解对方的恋爱习惯，重建情感连接
- **尊重对方**：尊重对方的想法和需求，给予对方独处的时间。
- **设限**：在尊重对方的同时，也要设立一定的界限。

## 维护关系
- **多点赞与认可**：在相处过程中，多给予对方肯定和认可。
- **提升自己**：在帮助对方的同时，也要不断提升自己。

## 总结
本节课中，我们学习了如何挽回回避型依恋的爱人，了解了如何通过合适的方法走进他们的内心世界。希望这些方法能够帮助到大家。
