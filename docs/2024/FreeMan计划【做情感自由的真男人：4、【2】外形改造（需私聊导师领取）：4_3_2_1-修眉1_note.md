## 🎨 外形改造教程：修眉基础

### 概述
在本节课中，我们将学习如何进行基本的修眉，以改善面部轮廓，提升整体形象。

### 修眉工具
以下是进行修眉所需的工具：

- **眉笔**：用于勾勒眉形。
- **眉粉**：用于填充眉形。
- **眉刷**：用于晕染眉粉。
- **眉剪**：用于修剪杂乱的眉毛。
- **眉夹**：用于拔除多余的眉毛。

### 修眉步骤

1. **清洁**：使用温水清洁眉毛周围的皮肤，确保无油脂和污垢。
2. **设计眉形**：使用眉笔轻轻勾勒出理想的眉形。
3. **修剪**：使用眉剪按照眉形轮廓修剪多余的眉毛。
4. **拔除**：使用眉夹拔除眉形外的杂毛。
5. **填充**：使用眉粉填充眉形，使眉毛看起来更加浓密。
6. **晕染**：使用眉刷将眉粉晕染均匀，使眉毛过渡自然。

### 修眉公式
```plaintext
修眉 = 清洁 + 设计眉形 + 修剪 + 拔除 + 填充 + 晕染
```

### 总结
本节课中我们一起学习了如何进行基本的修眉，通过掌握这些技巧，你可以改善面部轮廓，提升整体形象。希望你在实践中不断练习，逐渐掌握修眉的技巧。
