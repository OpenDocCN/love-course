## 一、 基于供需关系的分类：测试型与投资型
上一节我们概述了课程目标，本节中我们来看看第一种分类维度：根据男女之间的供需关系，女性可分为**测试型**与**投资型**。

### 1. 测试型女性特征
测试型女性通常外貌出众、有风格，身边不乏追求者。她们热爱社交，享受被关注的感觉，社交圈广泛。这类女性倾向于短期关系，难以在一段感情中深入。在聊天时，她们注意力分散，话题跳跃，难以进行深入交流。

### 2. 投资型女性特征
投资型女性通常比较单纯、宅。她们看重并期待长期关系，一旦认准一个男人便会长期投入。由于生活圈子小，她们在聊天时更加投入和专注，能就一个话题深入讨论很久。

### 3. 判断标准
以下是判断女性属于测试型还是投资型的四个标准：

**1. 感情史**
*   **测试型**：感情经历丰富，每段关系持续时间短。
*   **投资型**：恋爱经历少，每段关系持续时间长。

**2. 聊天深入程度**
*   **测试型**：很难就一个话题深入聊下去，频繁切换话题。
*   **投资型**：能就一个话题进行长时间、多方面的深入交流。

**3. 兴趣爱好广泛程度**
*   **测试型**：兴趣爱好广泛，以支撑其丰富的社交活动。
*   **投资型**：兴趣爱好相对单一或专注。

**4. 社交活动参与度**
*   **测试型**：积极参与各类社交活动。
*   **投资型**：不喜欢或较少参与社交活动，较为害羞。

**注意**：判断需综合至少两条以上标准，不应仅凭单一特征下定论。

### 4. 相处法则
了解完判断标准，我们来看看如何与这两类女性相处。

*   **对测试型女性**：应以自己为中心，避免过度恭维。需展现出坚决果断、强标准和领袖气质。公式可概括为：**`互动重心 = 自我`**。
*   **对投资型女性**：应以她为中心，让她感受到被追逐、关爱和呵护。需表现出踏实、上进、温和的特质。公式可概括为：**`互动重心 = 对方`**。

**重要提示**：没有百分之百的测试型或投资型，需判断其主导性格特征。

## 二、 基于话题类型的分类：感觉型与理智型
上一节我们介绍了基于供需关系的分类，本节中我们来看看第二个维度：根据聊天话题的侧重点，女性可分为**感觉型**与**理智型**。

### 1. 感觉型女性特征
感觉型女性内心感性，易被感动。聊天时看重情绪和感觉，喜欢甜言蜜语、浪漫话语。她们依赖性强，缺乏主见，喜欢被带领。通常家庭条件较好。

### 2. 理智型女性特征
理智型女性内心理性，看重现实和价值。她们独立、有主见，喜欢参与具体事务（如做饭、打扫），在意对方的事业和未来规划。这种性格可能源于家庭重大变故。

### 3. 判断标准
以下是判断女性属于感觉型还是理智型的三个标准：

**1. 聊天内容**
*   **感觉型**：常聊电视剧、爱情、情感等感性内容。
*   **理智型**：常聊工作、事业、规划等理性内容。

**2. 社交平台内容**
*   **感觉型**：多发个人生活、自拍、美食等。
*   **理智型**：多发有深度、有道理的文章或观点。

**3. 工作性质（辅助标准）**
*   **感觉型**：多从事设计、艺术等创意工作。
*   **理智型**：多从事会计、数据分析等逻辑性强的工作。

**注意**：判断需综合以上标准。

### 4. 相处法则
明确了类型判断，接下来学习相应的相处方法。

*   **对感觉型女性**：聊天应多涉及情绪、感受和自身经历。代码逻辑类似于：`聊天主题 = 情绪 + 个人故事`。

## 三、 基于亲密关系态度的分类：开放型与保守型
上一节我们探讨了感觉型与理智型，本节中我们来看看第三个维度：根据对亲密关系的态度，女性可分为**开放型**与**保守型**。

保守型女性对性话题非常保守，在意他人评价，行为低调。社交平台照片风格保守或不上传照片。性在其心中地位很高。

*   **开放型**：对轻微肢体接触反应不大。

**2. 社交平台照片风格**

**3. 外在形象**
*   **开放型**：穿着暴露，可能有纹身、重口味饰品。
*   **保守型**：穿着打扮传统、保守。

*   **对开放型女性**：关注点应放在其个性、聪明、豪放上，而非外表。肢体接触可以跳过一些阶段，更直接。公式为：**`关注点 = 内在个性；肢体接触进度 = 快速`**。
