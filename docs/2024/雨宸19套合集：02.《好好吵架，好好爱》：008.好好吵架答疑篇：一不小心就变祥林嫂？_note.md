# 雨宸19套合集：02.《好好吵架，好好爱》：008. 好好吵架答疑篇：一不小心就变祥林嫂？🤔

在本节课中，我们将要学习如何避免在亲密关系中陷入无休止的抱怨与指责，从而调整关系的能量场，让沟通回归积极与建设性。

## 概述：为何会变成“祥林嫂”？

两个人相处时间长了，难免会有抱怨和指责。没有人愿意每天跟一个充满负面能量的人在一起。

本节我们来聊一下，如何去调整关系中的能量场。要理解一个人如何变成“祥林嫂”，首先需要知道指责和抱怨的目的是什么。

## 指责与抱怨的三大目的

以下是三种常见的、具有破坏性的沟通目的：

1.  **只顾自我发泄**：只要自己情绪爽了，不管对方感受如何。网上甚至存在“花钱雇人骂自己”的业务，并且有受众群体，这印证了有人需要这种纯粹的情绪发泄出口。
2.  **争夺关系权力**：通过打压对方来取得关系中的胜利和主导权，以获得更多的爱和优待。其核心逻辑是：**我对 = 我有价值 = 你该爱我；你错 = 你没价值 = 我不爱你**。把对错当成了爱与价值的评判标准。
3.  **推卸自我责任**：希望对方为自己的感受和需求负责。没有找好主次关系。主要的关系是：**我要为自己负责**。你不需要为我负责，你只需要爱我就好。抱着这种需要别人为自己负责的态度进入恋爱关系，无异于寻找救世主。

## 指责行为的后果与根源

上一节我们介绍了指责的目的，本节中我们来看看这种行为会带来什么后果，其心理根源又是什么。

这种沟通方式的后果是：你激发了对方的自我保护本能。对方会想尽办法反击你或者远离你。当你在否定一个人时，对方一定会反过来解释或否定你，因为没有人喜欢被批评。

用阿德勒心理学的理论来解释，就是“课题混淆”。喜欢指责和抱怨的人往往分不清某件事到底是谁的责任。他们喜欢推卸责任，把自己的课题交给别人负责，或者过分干涉别人的课题。

例如：
*   **案例一**：女孩想减肥，跟男朋友说“你要监督我”。男朋友不忍心看她饿肚子就没提醒。女孩抱怨：“你为什么不提醒我减肥？” **想不想减肥是她自己的课题，与男朋友无关。**
*   **案例二**：男朋友想去打游戏或跟朋友玩。女孩说：“你不能去，你要陪我，你是我男朋友。” 这就是要求别人为你负责，替别人做选择。

## 撕掉“祥林嫂”标签的五个原则

了解了问题的根源和后果后，我们来看看如何通过五个核心原则来改变这种沟通模式。

以下是五个必须记住的沟通原则：

*   **原则一：摒弃居高临下的姿态**
    很多人抱怨和指责时，预设了自己更智慧、更理性、更成熟，觉得自己高人一等。一旦有了这样的预设，你就不会想听对方说什么。想赢得爱和尊重，首先要做到理解和尊重对方。爱是接纳，不要总想去改变对方。接纳对方真实的样子。如果接纳不了，可以选择离开，而非指责。

*   **原则二：选择更有效的表达方式**
    沟通方式有优先级：**能抱怨就别指责，能撒娇就别抱怨**。
    *   抱怨：“今天好热呀。”
    *   指责：“老天怎么这么混蛋，今天这么热！”
    *   撒娇：“好想被你抱一抱呀。”
    感受一下哪种方式更容易被接受。撒娇在解决许多无效争执时特别有用，能让感情变得可爱。

*   **原则三：先表达理解与认同**
    人际关系中有互惠原则。你先给了对方一个“苹果”（理解），对方也更倾向于给你一个“梨”（接纳）。这时你再讲道理，对方更容易接受。这不是套路，而是正常的沟通逻辑：先拉近关系，让对方有安全感。例如，在提出建议前先说：“你做的XX事，我觉得挺好的，然后我们再看看这个部分……”

*   **原则四：为自己的课题负责**
    为自己的情绪、感受和需求负责。除非对方主动要求，否则不要随意干涉别人的情绪、感受和需求。**你自己的事自己做主**。对方是来谈恋爱的，不是来找一个“妈妈”或“爸爸”来管束自己的。

*   **原则五：运用善意解读**
    婚姻研究专家约翰·戈特曼提出一个维持关系的有效建议：在解读对方行为时，**更多地运用善意**。这意味着更多地欣赏、赞美对方，寻找优点，并对对方的言行进行善意的解读，而非恶意揣测。
    *   **恶意解读**：“他一整天没联系我，肯定是不在乎我了。”
    *   **善意解读**：“他今天一天都没找我，一定是想让我安安静静地工作，真贴心。”
    用善意去解读对方，往往能带来意想不到的积极反馈，同样能解决问题，何必选择对抗？

## 总结：成为独立的人，营造积极的关系场

本节课中，我们一起学习了如何避免成为关系中的“祥林嫂”。

两个人之间的能量场如何，关系是否轻松愉快，很大程度上决定了一段关系的未来。在与另一半及他人相处时，努力让自己变得积极正面的关键是：**自己为自己负责，并且时刻怀有感激之心**。

关键在于，我们先要成为一个独立、能自我负责的人，然后才能健康地拥有一段关系。如果我们自己都不独立、不认可、不接纳自己，就无法掌控一段关系。我们能成为一个独立的个体，才能与另外一个独立的个体建立联结。我们不是婴儿，不需要别人全权照顾。该长大了。

通过践行以上五个原则，你可以有效调整沟通模式，将关系引向更轻松、愉快和充满善意的方向。
