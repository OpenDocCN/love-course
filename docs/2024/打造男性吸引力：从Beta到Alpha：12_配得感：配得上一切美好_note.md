# 打造男性吸引力：从Beta到Alpha 🚀 第十二课：配得感——配得上一切美好

在本节课中，我们将要学习“配得感”这一核心概念。它是你能否获得心仪事物，尤其是建立高质量两性关系的关键心理基石。我们将探讨其原理、表现，并学习如何建立强大的配得感。

## 课程概述

配得感，简而言之，就是“你认为自己配得上什么，你最终就能得到什么”。反之，你认为自己配不上的，你也无法获得。这个法则适用于世间万物。

尤其在两性互动中，这一点体现得最为明显。

## 配得感在两性关系中的表现

上一节我们介绍了配得感的基本概念，本节中我们来看看它在实际情感互动中的具体表现。

这些想法的根源，都来自于你内心对自己的不信任，以及你认为自己根本配不上这位女性。一旦你认为自己配不上，你就不可能去吸引她，也不可能上前与她交谈。

## 配得感的本质：自我设限

配得上或配不上这个概念，在人类和动物界是相通的。例如在野外，一头野兽只会去追捕它认为自己能追上并杀死的猎物。

在两性相处过程中，男性大多也只会去追求自己认为配得上的女性。因此，男性群体在两性关系中最普遍的一个限制性信念就是：“这个女生不在我的能力范围内”，简单说就是“我配不上她”。

在我个人的成长过程，以及观察我周围同学、朋友、学员的过程中，我发现他们最大的限制性信念和行为，正是觉得自己配不上对方。这也是大多数男性在两性情感中感到自卑的根源。

在我刚刚进入两性关系学习领域时，我和大多数男生一样，相当自卑。那时我非常被动，非常缺乏确信感。

在我的职业生涯早期，很多女性我根本不敢上前认识，也不敢去吸引。因为我认为那些女性根本不在我的能力范围内，我们不是一个世界的人，我配不上她们。我也曾深受这些限制性信念的困扰。

后来我才意识到，“配得上”或“配不上”其实根本没有一个实际、客观的标准。这完全源于一个人内心的自我设限，也就是我们所说的**自我价值认可**问题。

**我作为一个男人，这就够了。**
