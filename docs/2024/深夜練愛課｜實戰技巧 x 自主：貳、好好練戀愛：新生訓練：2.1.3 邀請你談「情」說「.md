# 深夜练爱课 贰：好好练恋爱：新生训练 2.1.3 邀请你谈“情”说“爱”｜如何友善搭讪 🗣️💬

在本节课中，我们将学习如何开启一段友善的搭讪，并探讨在建立新关系前，如何调整自身状态与设定清晰的交友目标。

## 一、确保自身状态稳定 🧘‍♂️

在开始友善搭讪与结交朋友之前，首要任务是关注自身身心灵的稳定状态。如果你当前状态不佳，充满委屈、愤怒等负面情绪，那么无论结交到何种朋友，你很快会进入抱怨与诉苦的模式。

你并非在寻找一个能与你进行心灵交流或共同前行的人，你只是想利用对方来倾诉不满，或互相倾倒情绪垃圾。这无法促成两人之间良性的互动。

因为这相当于你只想找一个免费听你诉苦的人。即便对方认真给予反馈或建议，处于那种情绪状态下的人通常也听不进去。如果你是一个如此负面的人，再优秀的人出现在你面前，也无法与你长久相处。

当然，每个人都有情绪起伏。有时确实需要一段时间来整理和修复自己，这没有关系。我们可以先把自己准备好，再开始结交朋友。

## 二、培养正向关注与同理心 🌟

准备好之后，我们来看看如何开启友善的搭讪。要开启这扇门，首先你需要是一个会关心周围事件发生的人，而不是一个冷漠或缺乏同理心的人。

在这个过程中，你需要对一些小事抱以正向的态度去面对。这样，在与他人交往时才不会陷入不断抱怨的恶性循环。否则，即使别人对你表达善意，你也可能将其误解为别有用心。

任何状况下，若无法形成正向循环，就无法进行友善的搭讪。

## 三、明确交友的目标设定 🎯

在交朋友之前，还有一个重要的设定：明确你现阶段需要什么样的朋友。

以下是几种常见的交友类型及其特点：

*   **玩乐型朋友**：例如，如果你年轻气盛，只想要每周末都能一起去泡夜店、纵情享乐的酒肉朋友，这没有问题。但你不能对他们抱有额外期待，比如期待他们在平日也能帮你处理事务或互相帮忙。在他们心中，你已被归类在“酒肉朋友”的文件夹里，平日发出的求助讯息，他们可能不会理会。
*   **八卦交换型朋友**：另一种需要注意的，是以他人隐私作为八卦交换筹码的朋友。人人都有好奇心。如果你认识某个知名人物、共同朋友或有关联的人，并用此人的隐私（如交友状况、财务状况、工作甚至童年往事）作为话题，你很快能吸引众人的注意力。

在群体中交换此类讯息时，大家都会听你说话，但你要小心这种“上瘾”的感觉。因为一旦你停止这样做，他们可能就对你失去兴趣。这代表这个群体只是在互相交换八卦与他人的隐私。

所谓“喝咖啡聊是非”便是如此。大家看似无事闲聊，但最终结果就是你的隐私也会被别人交换出去。如果你自己不想透露隐私，请远离是非之地，否则你最终也会变成是非之人。

---

本节课中，我们一起学习了友善搭讪的基础。首先，**确保自身情绪稳定**是建立健康关系的前提。其次，需要**培养对周围事物的正向关注与同理心**，这是开启良性互动的关键。最后，**明确自己现阶段对朋友的需求与期待**至关重要，这能帮助你识别并选择适合自己的社交圈，避免陷入消耗性的关系。记住，先打理好自己，再带着清晰的目标和开放的心态去接触他人。
