# 55、小三心理13堂课：12-第十二堂：小三在老公面前说妻子的好话，不让他离婚，背地却向妻子挑衅，她到底有何意图？

## 概述
在本节课中，我们将探讨小三在老公面前说妻子好话，同时背地里向妻子挑衅的心理动机和策略。

## 核心概念
**公式**：小三心理 = 好话 + 挑衅 = 稳住婚姻 + 激化矛盾

## 案例分析
以下是一个案例分析，帮助理解小三的心理动机。

**案例**：小红，原配妻子，与丈夫结婚20多年，共同创业。丈夫出轨，小红发现后坚决要求离婚。然而，小三却向小红表示，丈夫非常爱她，并劝她不要离婚。

## 小三心理分析
以下是小三在老公面前说妻子好话的心理分析：

1. **稳住婚姻**：通过赞扬妻子，让丈夫意识到妻子的优点，从而不轻易离婚。
2. **树立形象**：表现出自己有修养，不会说丈夫和妻子的坏话，树立大度、大方的形象。
3. **心理安慰**：认为自己无法控制丈夫，通过说好话来安慰自己。
4. **经济利益**：希望丈夫和妻子关系稳定，以便从中获利。
5. **激发矛盾**：通过在妻子面前说丈夫的坏话，激发妻子与丈夫的矛盾，达到离婚的目的。

## 小三挑衅行为
以下是小三背地里向妻子挑衅的行为：

1. **通风报信**：向妻子透露丈夫的婚外情细节，让妻子感到羞辱和绝望。
2. **言语羞辱**：直接侮辱妻子，让她感到自卑和愤怒。
3. **登门挑衅**：直接到妻子家中挑衅，让妻子感到无助和愤怒。

## 应对策略
面对小三的挑衅，以下是一些应对策略：

1. **保持冷静**：不要被小三的挑衅激怒，保持冷静。
2. **寻求支持**：向亲朋好友寻求支持，共同应对小三的挑衅。
3. **录音取证**：录音小三的挑衅行为，作为证据。
4. **报警处理**：如果小三的行为构成骚扰，可以向警方报警。
