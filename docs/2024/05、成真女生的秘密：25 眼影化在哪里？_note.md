# 课程05：眼影基础 🎨

在本节课中，我们将学习化妆中一个重要的彩妆品类——眼影。我们将了解眼影是什么、它与护肤品的区别、其主要作用以及基本的使用方法。

## 什么是眼影？

上一节我们介绍了护肤品中的眼霜，本节中我们来看看同样是用于眼周，但作用完全不同的彩妆产品——眼影。

眼影是用于眼部化妆的彩妆产品。它与之前讲过的、用于保护皮肤和保湿的眼霜不同。眼影的核心作用是**为眼周皮肤上色**，以修饰眼部轮廓，让眼睛看起来更有神采。

## 眼影的作用与颜色

眼影通过颜色来塑造眼部立体感和氛围。其核心作用是**增强眼部神采与深邃感**。

以下是眼影颜色的常见分类：

*   **深色眼影**：例如黑色、深棕色。常用于打造烟熏妆效果，能让眼睛看起来非常深邃。
*   **浅色眼影**：例如米色、浅粉色。这是大多数女生日常妆容中最常使用的颜色，妆效更为自然。

## 如何使用眼影？

了解了眼影是什么之后，我们来看看它的基本使用方法。涂抹眼影通常需要借助工具，将眼影粉均匀地铺展在眼周皮肤上。

以下是两种常见的涂抹方法：

1.  **使用专业工具**：用专用的**眼影刷**蘸取眼影粉，然后刷在眼皮和眼角周围。这是最标准的方法。
2.  **使用手指**：为了快速上妆，或者暂时找不到眼影刷时，也可以用手指直接蘸取眼影粉。关键在于用手指将颜色**晕染开**，以达到均匀自然的效果。这种方法需要一定的技巧。

---

本节课中我们一起学习了眼影的基础知识。我们明确了眼影是用于眼部上色的彩妆品，而非护肤品。它通过不同的颜色来增强眼部神采，深色可塑造深邃感，浅色则适合日常妆容。涂抹时，可以使用专业的眼影刷，也可以用手指进行晕染。
