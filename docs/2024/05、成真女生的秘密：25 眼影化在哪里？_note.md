# 课程05：眼影基础 🎨

在本节课中，我们将学习化妆中一个重要的色彩元素——眼影。我们将了解眼影的作用、它与护肤品的区别、常见的颜色选择以及基本的上妆方法。

## 眼影是什么？

上一节我们介绍了护肤品中的眼霜，本节中我们来看看眼影。眼霜和眼影都与眼睛有关，但作用完全不同。眼霜用于保护皮肤，主要功能是保湿。眼影则是化妆品，用于为眼部区域增添颜色。

眼影用于描绘眼睛周围，其作用是让眼睛看起来更有神采。眼影拥有多种颜色。

## 眼影的颜色与效果

以下是常见的眼影颜色与风格示例：

*   **深色眼影（如烟熏妆）**：许多欧美妆容喜欢使用深色眼影打造烟熏妆效果。这会让眼睛看起来非常深邃。
*   **日常浅色眼影**：大多数女生日常使用的眼影颜色相对较浅。

## 如何上眼影？

眼影需要使用工具将其涂抹在眼部。以下是两种主要的上妆方法：

1.  **使用专业工具**：使用专门的眼影刷，蘸取眼影粉，然后刷在眼皮和眼角周围。
2.  **使用手指**：为了快速上妆，或暂时找不到眼影刷时，也可以用手指直接蘸取眼影。用手指涂抹并晕染开，能达到同样的效果。
