# 56、小姨子供养者翻盘-强者框架课：013-01-社交生活的魅力来源-2_ev_note

在本节课中，我们将深入探讨社交吸引力的核心构成。我们将了解吸引力背后的三大价值支柱，并解析它们在社交互动中如何发挥作用，特别是情绪价值如何成为放大你个人魅力的关键变量。

---

上一节我们介绍了吸引力的基本概念，本节中我们来看看吸引力的具体来源。人类对彼此的吸引并非凭空产生，其背后有深刻的生物学与社会学逻辑。

一个人的外在特征常常传递着其健康状况与生存能力的信息。例如，脱发往往代表身体某些生理机能出现紊乱，这意味着不健康。再比如，我们认为腿长好看，一个原因是在古代，当人类尚未完全站在食物链顶端时，在野外遭遇危险（如老虎），腿长的人往往跑得更快，逃生能力更强。我们的审美偏好背后，往往都带着健康和能力强的信息。所以，我们认为这些外形上的优势，代表的是**繁衍价值**。

而当你这个人**生存价值**高，**繁衍价值**高，就会吸引更多人愿意与你结盟。因为人类是群居动物，以结盟的形式生存。每个人都希望与更强大的人结盟，从而得到更好的生存保障。在情感关系中最直接的体现，就是很多人会说，高富帅、白富美就有魅力。这是吸引力的基础，是站在人的动物性角度得出的结论。

但人之所以为人，就是因为其身上不只有动物性，还有**人性**。人与动物的区别在于，人已经发展出了意识，可以对当下情况和环境进行思考，做出更高级的判断和选择。

我们都知道，人类社会发展到今天，早已度过了会饿死的年代，也度过了身边随时可能出现猛兽的危险时期。有句话叫“仓廪实而知礼节，衣食足而知荣辱”。意思是，当我们的粮仓充实、衣食充足后，就会开始思考礼节和荣辱。这代表了人类在满足基本的物质需求和安全需求之后，就会追求更高层次的**精神需求**。

在当今这个不愁吃穿的时代，人们开始追求更高级的精神满足。因此，人类衍生出第三个价值——**情绪价值**。这是作为人这个物种特有的价值。

以上三个价值——生存价值、繁衍价值和情绪价值，就是吸引力的三个核心。一个人对另一个人的吸引完全从这三个价值出发。

相信你已经明白了这三个价值的含义。接下来我们探讨下一个问题：这三个价值在构成吸引力的过程中，各自的重要性如何？

以下是三种价值的划分与关系：

你的生存价值和繁衍价值，我们称之为**基础价值**。这两个价值相对稳定，无论是长相还是财力水平，在成年后不会短时间内发生巨大变化。一个长相普通的人想整形成帅哥，往往要花几年。一个普通打工族想成为富豪，也往往需要几年甚至几十年。所以，这些不太容易改变的价值，叫做基础价值。

而情绪价值，我们称之为**变量价值**。情绪价值很容易改变，我们可以把它变得很高，也可以变得很差。可能你今天心情好，不经意间散发的情绪价值就很高，别人与你社交就很舒服；也可能某天你遇到沮丧的事，一整天闷闷不乐，别人与你相处就觉得压力大，这时你的情绪价值就很低。

最关键的是，你的情绪价值会直接影响你基础价值的吸引力。其**吸引力构成公式**如下：

**吸引力 = (生存价值 + 繁衍价值) × 情绪价值**

情绪价值是可以放大或缩小你基础价值的变量。这就是三种价值与吸引力之间的关系。

有非常多家里很有钱的人，就是让人讨厌，跟他相处开心不起来，觉得浑身别扭。也有很多没什么钱、长得不帅的人，你跟他相处起来就是很舒服，这个人很有人格魅力。这些都是情绪价值放大或缩小吸引力的现象。

这也是我开设社交吸引力训练营的核心原因。我发现，人到成年后，样貌很难改变，财富状况也很难改变。唯有情绪价值，是我们可以通过学习掌握，并立刻让生活发生变化的一种价值。

在我这些年的咨询生涯中，有些学员是普通工薪阶层，月入三五千或一万；还有很大一部分属于比较有钱的，比如富二代或企业高管。但他们面临的问题相似：都非常缺乏情绪价值。

对普通人而言，提升情绪价值是必须学习的事。因为收入短时间内无法改变，但我们依然要面对社会竞争，依然想找到优秀的伴侣。所以，你一定要提升情绪价值，让你这个人的综合吸引力更高。

那为什么有钱人、富二代也要学习呢？我讲一个我之前的学员案例。这个男生是标准富二代，家里做生意，在北京有7套房子，其他城市还有若干房产，全国不动产总价值约一亿，这还不包括股票、基金、信托和存款。他在美国留学。按理说，这样的富二代对女孩很有吸引力。但他为什么找我做咨询？

很简单。他在洛杉矶留学，喜欢上一个日本女同学。要知道，这些留学生家里条件都不差。这个日本女孩也是富二代，父亲在日本很有势力。这时就出现一个现象：这个男生所拥有的（生存）价值，在这个女生眼里都不算什么。你可以带她去昂贵餐厅、送名牌包、开跑车，但这些对这个女生来说只是平常生活。这个男生这时才明白自己很无力，因为他没有真正的核心吸引力。

过去他在国内时，出去玩有很多女孩喜欢跟他一起。用这个学员的话说：“老师，这些都是捞女，谁愿意跟捞女在一起玩？谁喜欢跟捞女谈恋爱？”所以，即便他很有钱，他遇到的困惑是一样的。因为有钱人喜欢的女生，条件往往也不差，你的生存价值对对方产生的吸引力很弱。

所有这些问题的解法只有一个：**提升你的情绪价值**，提升你这个人本身的人格魅力，让别人因为你这个人而喜欢你，而不是因为你的钱和样貌。要知道，钱和外貌只能让别人对你产生基础兴趣。而真正深层次吸引力的来源，是你这个人的人格。这才是你最稳定、最强大、可以终身携带的东西。

哪怕有一天你事业失败、没有钱了，你还可以靠人格魅力召集一帮兄弟支持你东山再起，你的伴侣也不会因为你没钱而离开你。

---

说完了吸引力，接下来我们讲讲什么是**社交吸引力**。

为什么这套课程叫“社交吸引力”？因为人与人吸引的过程，其实就体现在你与他人社交的过程中。你每天在家躺在床上，是不会对别人有吸引力的。你只有真实地与别人打交道，在这个过程中，才能对他人产生吸引。所以我把这套课程的定位定在社交吸引力上。它不是吸引力的理论课程，而是在社交场景中的应用。

我这套课程所讲的所有社交场景，都是聊天的场景。为什么讲聊天场景？因为聊天是所有社交关系的纽带，贯穿在所有社交环节之中。

不管你是从网上认识一个陌生人，要与他慢慢变熟悉；还是认识一个女孩，已经出来约会喝咖啡或吃饭；或者你要找工作去面试，要找朋友帮忙，或女孩过生日要送礼物……这些都离不开语言表达，离不开沟通，离不开聊天。所以这套课程，我以聊天为基础，讲解在社交互动中，如何让自己变得有魅力和吸引力。

那说到聊天，我问你：你觉得什么叫做好的聊天？什么样的聊天，能让你在社交过程中产生吸引力呢？

是一些精彩的话术吗？是一些俏皮的情话吗？是一些神回复吗？是你能掌握非常多的话题吗？其实都不是。聊天真正的魅力，并不在文字本身，而是通过字里行间的回复，与对方的潜意识产生沟通，达成一致。

这么说可能比较抽象，给你举个简单例子。

比如你遇到一个喜欢的女生，你跟她聊天，聊到了感情观。然后这个女生说：“我想找一个成熟体贴的男朋友。”这时候你会怎么回复她？

我们假设有两个男生。

**第一个男生**说：“哎，我就很体贴啊。我这个人非常注意细节，平时很干净，会做很多好吃的，自己也会做家务。你看我昨天刚收拾的家里，一尘不染很干净吧？跟我在一起，保证把你照顾得白白胖胖。”

**第二个男生**说：“像你这么年轻的女孩子，一般都是追求浪漫，但你好像比同龄人成熟很多啊。”（只回了这一句话）

我问你，你觉得哪个男生的回应更有吸引力？相信你能感受到，明显是第二个男生的回复更有吸引力。但为什么呢？明明是第一个男生会做家务、很干净、又会做美食，明显他更能满足女生的标准啊。为什么他的吸引力没有第二个人强？

这就是我说的：聊天的真正魅力，并不体现在文字本身。

我们假设女生说想找一个成熟体贴的男朋友，这个需求是真的。但很明显，第二个男生的应对要比第一个男生成熟和体贴很多。

首先，什么叫成熟？一个成熟的男生，内心很强大。他不急于向对方解释和证明。你能看到第一个男生的行为是极力证明自己，而第二个男生没有。

其次，什么叫体贴？体贴并不是说我能给你做卫生、做饭。这些东西在当代社会不难实现。我们找保姆、保洁，订外卖，都可以让屋子变干净、吃饱饭。那真正的体贴是什么呢？是能洞察你的内心需求。

很明显，第一个男生完全忽略了这个问题。而第二个男生直接点出了这个问题。他说：“感觉你好像比同龄人成熟很多。”我问你，这个时候女生什么感受？女生会觉得：“哎，这个人好像很懂我。”而我们都知道，只有懂一个人的内心，才能做到真的体贴。

所以很明显，第二个男生只是简简单单通过一句与“成熟体贴”毫无关系的话，就能让女生隐约感觉到这个男人身上有成熟和体贴的特质。吸引力就从这儿开始产生了。这个才叫做有吸引力的聊天。

没有任何花里胡哨的话术，没有什么神回复，但就是能精准击中对方的潜意识。看似什么都没聊，但背后的交流暗藏着你看不见的丰富信息。这就是我们说的社交中的吸引力。

而你在社交中不断做出这样正确的处理，你的吸引力就会不断增强，最终形成你的人格魅力。这就是我经常跟学员说的一句话：**聊天是什么？聊天不是话术，聊天不是方法，聊天是你人格的主要外显结构。**

**本节课总结**

本节课中，我们一起学习了：
1.  吸引力的三大核心价值：**生存价值**、**繁衍价值**和**情绪价值**。
2.  前两者构成相对稳定的**基础价值**，后者是能放大或缩小吸引力的**变量价值**，其关系符合公式：**吸引力 = (生存价值 + 繁衍价值) × 情绪价值**。
3.  在物质丰裕的现代社会，**情绪价值**是提升个人魅力最有效、最可控的途径。
4.  社交吸引力体现在真实的互动中，而**聊天**是所有社交关系的核心纽带。
5.  有吸引力的聊天不在于话术，而在于通过交流与对方的潜意识沟通，展现你成熟、体贴的人格特质。

理解并实践这些原则，是提升你社交吸引力的第一步。
