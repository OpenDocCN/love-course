# 安老师社恐训练营：21-综合练习：系统化的社交恐惧训练_note

在本节课中，我们将学习如何将之前学到的单点突破技巧整合起来，进行系统化的社交恐惧综合训练。我们将通过记录场景、逐个击破、并结合想象与现实练习，来巩固和深化你的社交能力。

---

在前几节课中，我们分别从**减少自我关注**、**改变思维模式**和**改变行为模式**这三个角度来突破社交恐惧。本节中，我们将把这些方法综合运用，进行系统化的训练。综合训练的核心，依然是这三个突破口。

## 第一步：记录焦虑场景 📝

以下是开始训练的第一步，你需要将引发焦虑的具体社交场景记录下来。

*   例如：开会时最后一个进入会议室，感到被注视而不自在。
*   例如：当众讲话时声音颤抖、紧张。
*   例如：与亲戚相处时感到焦虑、不自在。
*   例如：与异性交谈时心慌、回避眼神、结巴或脸红。
*   例如：聚会时感到异常焦虑和紧张。
*   例如：上下班路上躲避半生不熟的同事。
*   例如：在小区或电梯里遇到邻居时感到不自然。

## 第二步：逐个场景击破 🎯

记录下场景后，接下来需要对每个场景进行针对性的分析和练习。我们以“与异性相处时心慌、回避眼神”这个场景为例。

首先，可以进行**想象训练**。想象训练不仅仅是获得视觉形象，它是一个**多种感觉的过程**。你可以在独自放松时，在脑海中完整地模拟当时的社交场景，切身地去感受当时的情绪和身体反应。由于是回忆已发生的事，这个过程相对容易。这更是一个深层次的分析过程，能帮助你弄明白到底发生了什么，问题出在哪里。

**核心代码/逻辑描述：**
```
进行想象训练：
1. 选择一个安静、放松的环境。
2. 在脑海中清晰还原目标社交场景（如：与同事对话）。
3. 主动感受并记录下当时的情绪（紧张）、身体感觉（心跳加速）和思维（“我说错话了怎么办”）。
4. 分析在场景中，你的注意力、想法和行为模式。
```

当想象训练进行得比较熟练后，就可以进入**现实训练**。解决社恐的关键是**实践 + 实践**，只有通过多次主动进入不舒服的情境进行锻炼，才能最终获得舒适感。

## 两个重要的训练要点 💡

在综合训练过程中，有两个关键点需要注意。

**第一点：灵活寻找练习机会。**
有的同学可能觉得特定场景（如与领导相处）实际发生的频率太低，练习机会少。这时需要懂得变通，抓住训练核心。核心是**你与高能量的人相处时会紧张**。因此，只要能与其他让你感到有压力或权威的人（如老师、资深同事、客户）互动，这就是有效的练习。不要过于死板地限定场景。

**第二点：将每日生活场景化训练。**
不用担心如何安排训练，只需把每天遇到的所有社交互动都视为锻炼机会。无论表现如何，都不要自责或气馁。晚上回家后，对当天的经历进行回顾和想象模拟，发现问题并思考如何调整。

## 如何进行复盘与调整？ 🔍

那么，具体如何发现问题并进行调整呢？方法依然是那熟悉的三个方面。

例如，今天在电梯遇到领导，感到特别紧张。晚上回家后，你可以详细回顾整个过程，并向自己提出三个问题：

以下是自我复盘时需要提出的三个核心问题：

1.  **注意力分析：** 我当时的注意力在哪里？是否过度关注了自己的表现（如“我脸是不是红了”）？
2.  **思维模式分析：** 我当时内心的深层次想法是什么？（如“领导会不会觉得我能力不行？”）
3.  **行为模式分析：** 我当时的行为是怎样的？（如：是否低头、回避眼神、匆忙结束对话？）

问完这三个问题后，你就可以针对性地进行分析和解答。哪一个方面存在问题，就对应地去改善和训练，具体方法在前面的课程中都已详细讲过。

**总结**
