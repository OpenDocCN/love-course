# 23、老公出轨心理27堂课：25 男人出轨回归后，为什么还会和妻子冷战？

## 概述
在本节课中，我们将探讨老公出轨回归后，为何还会与妻子冷战的现象，并分析如何处理这种情况。

## 核心概念
**回归家庭**：指出轨者意识到婚外情的不可持续性，选择回到家庭。

**冷战**：指夫妻双方在情感上保持距离，不进行沟通和交流。

## 冷战形式
以下是老公出轨回归后常见的冷战形式：
- 一上来就说坏情已经结束了，不要再说了，翻篇了。
- 一提坏情这个事情，马上挑头就跑。
- 回到家里面玩自己的手机，要么睡自己的房间，要么睡自己的床。
- 表面上能聊，但一旦触发到婚外情，触发到小三，甚至看到电视电影、网络上有这种事情，马上翻脸。

## 冷战原因
以下是老公出轨回归后冷战的原因：
- 没有真正想要回归，只是被迫回到家。
- 担心与妻子产生连接，被小三找麻烦。
- 情感上还没有完全放下婚外情。
- 进入反击反制，先下手为强。
- 内心回归家庭，但心理上还有点不接受。
- 想要缓和关系，但内在还有恐惧和防御。
- 用冷战来获得妻子放过他。

## 解决冷战的方法
以下是解决老公出轨回归后冷战的方法：
1. 与老公确定回归家庭和与小三分手的趋势。
2. 告诉老公冷战的行为让你感到孤独和绝望，可能会选择离婚。
3. 告诉老公婚外情和冷战都无法继续，底线要打出来。
4. 与老公分析冷战的目的、心理和后果。
5. 与老公沟通，打破冷战，解决他的担忧。
6. 营造轻松、可沟通的平台，营造未来可能的生活方式。
7. 打消老公背叛小三的未来心理。

## 总结
本节课中，我们学习了老公出轨回归后冷战的原因和解决方法。在处理这种情况时，要保持冷静，与老公进行有效沟通，并寻求专业辅导。
