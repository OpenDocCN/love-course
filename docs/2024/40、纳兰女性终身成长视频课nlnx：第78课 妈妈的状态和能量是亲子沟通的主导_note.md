# 40、纳兰女性终身成长视频课nlnx：第78课 妈妈的状态和能量是亲子沟通的主导

🎼亲子关系中的沟通场景是亲密关系中的第二大场景。亲子关系的沟通没有太多可说的，因为之前已经详细讲解了如何成为智慧且不焦虑的妈妈。

🎼然而，既然课程已经上架，并且按照不同的生活场景进行结构化，亲子关系这一部分仍然需要讲解。亲子关系与夫妻关系和婚姻一样，是一场漫长的旅程。父母与孩子之间的对话过程，也是日常沟通的方式。

🎼随着孩子的成长，你可能不再是他们理解的或需要的妈妈，你可能会成为他们的同修道友、人生导师，甚至好朋友。

🎼以下是亲子关系中沟通的现状和问题：

- **不听话**：80%的家长认为孩子不听话。
- **无效沟通**：孩子可能在生理上排斥家长的道理，导致沟通无效。

🎼以下是一些解决方法：

- **认知矫正**：孩子不是必须听你的话，他们也是独立的人，拥有自己的思考和认知能力。
- **双向沟通**：倾听和表达才是真正的沟通，要给孩子表达的机会。
- **正能量**：保持充足的正能量，才能给孩子正面的引导和积极的作用。
- **角色定位**：随着孩子的成长和需求，妈妈的角色定位也要不断调整。
- **尊重孩子**：尊重孩子的年龄、认知、兴趣爱好、感受和隐私。
- **理解孩子**：善意的倾听，保持理解，避免强势的打断和干预。
- **支持孩子**：真诚的认可和鼓励，看到孩子的优点，而不是缺点。
- **信任孩子**：信任是双向的，父母要信任孩子，孩子才能信任父母。

🎼以下是一些具体的方法：

- **讲道理**：不要空讲大道理，要选择时机和场合，通过例子和故事来传达道理。
- **倾听孩子**：先让孩子表达，再给予回应。
- **共情式沟通**：理解孩子的情绪和思维，成为孩子，体会他们的所思所想。
- **夸奖孩子**：表扬孩子的努力、具体任务和细节，而不是泛泛的夸奖。

🎼亲子关系中的沟通是一个复杂的过程，需要父母不断学习和成长。通过以上方法，我们可以更好地与孩子沟通，建立良好的亲子关系。
