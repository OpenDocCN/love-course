# 雨宸19套合集：19.婚恋情感IP，运营变现课程：009.第九节 如何做咨询指导3-常见的依恋类型和人格类型

## 概述
在本节课中，我们将探讨常见的依恋类型和人格类型，帮助理解个体行为和心理状态。

## 高敏人格
高敏人格分为三个阶段：一期、二期、三期。进入第三阶段的人通常在某个专业或领域能取得顶尖位置。

**公式**：高敏人格 = 强烈刺激感受 + 强感知能力 + 敏锐情绪察觉

**特点**：
- 对刺激感受强烈，感知能力强，第六感敏锐。
- 学习能力强，对细节敏感，善于发现错误。
- 行动谨慎，危机管理能力强。
- 情绪察觉能力强，内心世界丰富。
- 潜在的负面效应：刺激过载，身心疲惫。

## 受虐性人格
受虐性人格障碍者倾向于选择容易失望、失败或遭受虐待的人或事。

**特点**：
- 即使有更好的选择，也选择容易失望或失败的人或事。
- 拒绝或故意阻挠他人帮助。
- 对积极事情感到内疚或压抑。
- 故意引起他人愤怒或否定性反应。
- 在得到愤怒或否定反应时感到受伤，但内心暗爽。
- 拒绝享受快乐，总是处于痛苦中。
- 即使有能力，也无法完成关键任务。
- 对善待自己的人不感兴趣，对受伤的人产生渴望。
- 在被帮助者未要求的情况下，过度自我欺视。

## 被攻击性人格
被攻击性人格者对他人计划安排不满意时，不会直接表达，而是拖延回应或挑剔。

**特点**：
- 对他人计划安排不满意时，拖延回应或挑剔。
- 很少直接表达情绪，通过间接方式让人感受到情绪变化。
- 在亲密关系中，难以主动提出分手，而是制造矛盾。
- 在工作中，表面上服从安排，暗地里妨碍工作。

## 高防御人格
高防御人格者在人际关系中感到不满，容易进入独裁式的日常生活。

**特点**：
- 在人际关系中感到不满，容易进入独裁式的日常生活。
- 对他人评价和理解过度解读为批评和攻击。
- 被人伤害后，容易产生愤恨和报复心理。
- 难以完全为自己的行为负责，习惯推卸责任。
- 无意伤害他人，但过度自我保护，觉得自己最善良。
- 无法看到他人痛苦，只关注自己痛苦。
- 设想最坏可能性，容易表现出焦虑、压力和攻击性。

## 总结
本节课中，我们学习了高敏人格、受虐性人格、被攻击性人格和高防御人格的特点。了解这些人格类型有助于我们更好地理解个体行为和心理状态。
