# 课程18：框架思维（2）—— 养成框架思维体系 🧠

在本节课中，我们将深入探讨框架思维的第二个核心板块：思维框架。我们将通过具体实例，理解什么是思维框架，它如何影响我们的决策和人际互动，并学习如何在日常沟通与恋爱关系中建立并运用自己的框架思维。

---

## 概述

上一节我们介绍了框架的基本概念。本节中，我们来看看思维框架如何具体运作。思维框架是指我们思考问题和做出决策时所依赖的潜在结构或模式。很多时候，我们并未意识到自己正身处他人设定的框架中，从而做出并非基于现实本质的反应。

## 思维框架的影响力

大多数人习惯于被动接受问题设定的框架，并在此框架下进行思考，这是一种思维惰性。以下通过两个问题来揭示框架如何影响决策。

以下是两个看似不同的问题：

1.  某赌注有10%的概率赢得95元，有90%的概率损失5元，你会接受这个赌注吗？
2.  某赌注有10%的概率赢得100元，有90%的概率什么也得不到，你愿意花5元买这张彩票吗？

你的第一反应可能是分别思考这两个问题并得出选择。然而，这两个问题的**本质是完全相同的**。计算期望值即可证明：
*   问题1的期望收益：`0.1 * 95 + 0.9 * (-5) = 5`
*   问题2的期望收益：`0.1 * 100 + 0.9 * 0 - 5 = 5`

它们只是用不同的语言（“损失5元” vs “花5元购买”）描述了同一件事，却导致了人们不同的感受和选择。这就是**框架效应**：问题呈现的方式（框架）不同，会影响人们的决策。

## 人际互动中的框架博弈

在人际交往，尤其是恋爱关系中，框架思维无处不在，它决定了互动的主导权。

### 框架高低的定义

*   **高框架**：设定谈话前提和规则，引导对方在自己设定的范围内做出反应。
*   **低框架**：被动接受对方设定的前提和规则，在其框架内进行回应。

以下是几个生活中框架博弈的例子：

*   **示例1：模糊邀约**
    “我们什么时候去看电影？” 这句话的潜台词是假设你愿意和我看电影，我将自己置于框架高位。
*   **示例2：销售话术**
    面馆老板问：“加一个鸡蛋还是两个？” 这个高框架假设了你一定会加鸡蛋。框架弱的人会本能地回答“加一个”，而忽略了“不加”这个选项。

当一个人的框架足够强时，周围的人会倾向于认同并跟随他的框架。

## 恋爱关系中的框架应用与破解

在恋爱中，框架思维深刻影响着双方的地位和相处模式，例如对方是否敢于查看你的手机、是否愿意付出等。

### 1. 应对价值评判框架

当对方试图设定标准来评判你时。

*   **场景**：男生说：“我不太喜欢娇生惯养的女生，你会做饭吗？”
    *   **低框架回应**：“我会做蛋炒饭。”（进入对方框架）
    *   **高框架回应**：“会呀，如果你表现好，我会奖励你做我的拿手红烧鱼。”（建立自己的框架）

### 2. 破解前提假设框架

当对方的话语中预设了对你不利的前提。

*   **场景**：男人说：“你不当我女朋友，我怎么对你好？”
    *   **破解思路**：指出其逻辑是将“好”与“身份”绑定。
    *   **高框架回应**：“如果你的好必须建立在一种特定身份上，那听起来更像雇佣关系。”

*   **场景**：男人说：“我不喜欢发展太快。” 或 “你还不了解我。”
    *   **框架分析**：这些话潜在地植入了“你想快速发展”或“你想了解我”的前提。正面回答就等于承认了这个前提。
    *   **破解方法**：将前提反射回去，打破其框架。
    *   **高框架回应**（对“你不了解我”）：“你这句话，好像挺期待我来了解你？”

### 3. 应对标准测试框架

当对方用描述喜好来测试你是否会迎合他。

*   **场景**：男人说：“你刚才黏人的样子，搞得我有点心动，我很喜欢你黏人的样子。”
    *   **低框架回应**：今后刻意表现得黏人（迎合对方）。
    *   **高框架回应**：“我一般心情好的时候才会黏人，心情不好时脾气比较暴躁，你要当心哦。”（展示自身标准，不随意迎合）

### 4. 应对质疑与调侃框架

当对方提出带有质疑或贬低意味的问题时。

*   **场景**：男生问：“你是不是经常和其他男生聊天？/ 你是不是渣女？”
    *   **低框架回应**：急于解释“我不是”或“我只对你这样”。（在意对方看法，陷入其框架）
    *   **高框架回应**：“我是不是渣女，我未来的另一半应该比较有发言权吧？”（将评判权拉回自己手中）

*   **场景**：大叔说：“小妹妹，感觉你年龄好小啊。”
    *   **低框架回应**：“我年龄小，但心理年龄不小。”（仍在对方“论年龄”的框架内辩解）
    *   **高框架回应**：“大叔，朋友之间没必要强调年龄吧？刚认识就用女朋友的眼光看我，是不是太快了？”（重新定义关系性质，跳出年龄框架）

## 从初识到相处的框架树立

框架思维从第一次互动就已开始。

*   **场景：街头搭讪要微信**
    *   **低框架回应**：直接说“可以”或“不行”。
    *   **高框架回应**：“当然可以啊，不过加之前，你总得先告诉我一个想认识我的理由吧？”

*   **场景：约会时对方问“要我来接你吗？”**
    *   **低框架回应**：“不用了，太麻烦。”（可能打消对方积极性）
    *   **高框架回应**：“如果我不同意你来接，岂不是少了一次让你表现的机会？反而显得我不懂事了。”（将“接你”定义为他的“机会”，建立高位框架）

*   **场景：对方将行动原因归于你**
    *   **男生说**：“如果你想我，我就过去找你。”
    *   **框架分析**：潜台词是“因为你有需求，我才行动”，希望你暴露需求感。
    *   **高框架回应**：“我希望是因为你想我，所以你才过来。”（将动机主体转换回对方）

*   **场景：对方用假设测试你的忠诚**
    *   **对象问**：“要是我没车没房，你还会坚定选择我吗？”
    *   **低框架回应**：“我喜欢的是你的人，不是你的钱。”（给出虚无承诺，可能成为日后把柄）
    *   **高框架回应（可选其一）**：
        1.  “你这个问题，已经把我们的感情建立在金钱之上了，我觉得没必要回答。”
        2.  “买车买房应该是你自己对人生的标准，而不是我的要求。”
        3.  “男人的财富会随年龄增长积累，买房买车是迟早的事。我更看重当下的品质和你会不会对我好。”

## 总结

本节课中，我们一起学习了思维框架的核心原理及其在人际交往中的强大作用。框架思维的本质是**掌握对话和关系的主导权**，避免被动陷入他人设定的逻辑和规则中。通过识别、破解并建立自己的高价值框架，你可以在沟通中保持主动，维护自身边界，并建立起相互尊重的关系模式。真正的**高情商**，往往体现在娴熟的框架思维运用上。
