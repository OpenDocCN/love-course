# 51、人人都需要：19_守护篇第一

在本节课中，我们将要学习恋爱磨合期为何容易产生矛盾，并理解其背后的核心概念。我们将从男女生理及思维模式的差异入手，帮助你客观看待磨合期的冲突，为后续学习化解方法打下基础。

上一节我们介绍了课程主题，本节中我们来看看恋爱磨合期矛盾产生的根本原因。

男女相恋半年之后，随着彼此体内荷尔蒙多巴胺的消退，爱情的魔力也开始减退。热恋时的山盟海誓、甜言蜜语、喜悦乐观，可能就会变成对彼此是否合适的质疑。大部分情侣会开始减少宽容、变得固执己见、互相责备。这个热恋过后的不稳定期，通常是半年左右，此时的情侣最容易闹矛盾。

要化解这种矛盾，首先需要理解两个核心概念。

以下是两个需要理解的核心概念：

1.  **感情浓度**
    *   其表现形式为：对方会尽量满足你的小要求；日常联系频繁，如定时电话、每天接送；时常制造小惊喜；重视各种节日和纪念日。
    *   公式化描述：**感情浓度 ≈ 日常关怀频率 + 即时需求满足度 + 仪式感行为密度**

2.  **感情深度**
    *   其表现形式为：将大宗资产交给你管理；带你见家长并推动婚事；在你家出事时鼎力相助；帮你规划职业发展；将你融入他的核心社交圈。
    *   公式化描述：**感情深度 ≈ 长期责任承诺 + 重大资源投入 + 未来规划融合度**

一般情况下，半年之后，男人的感情浓度会自然下降，而女人的感情浓度则可以一直保持很高。这种差异导致了矛盾的产生。

理解了概念差异后，我们需要探究这种差异背后的原因。

矛盾产生的根源在于男女生理及思维模式的不同。男人的大脑构造和女人完全不同。在远古狩猎时期，男人的DNA中刻下了专注目标的印记。在男女关系中，当男人追求一个女人时，他会将“确立关系”视为一个需要完成的目标，并为此拼尽全力，专注度非常高。一旦关系确立，目标达成，他们便会本能地将注意力转移到下一个目标上，例如事业、家庭或娱乐。

而女性在远古时期需要同时照顾孩子、做饭、处理家务，因此必须分散注意力。女性擅长一心多用，这同时也意味着女性的专注力和目标感通常不如男性强。

因此，女性常常无法理解为何男性在热恋后期感情浓度会下降。在女性看来，这是“不再爱我”的表现；而在男性看来，这只是完成了“追求”这个阶段性目标，自然地将注意力进行了转移。

我们可以通过一个生活化的例子来理解这种差异。女性可以边看电视边剪指甲，边管孩子边聊天，同时还能注意到门铃响。而男性在看球赛时，可能完全无法分身做其他事，甚至听不见门铃声。因为男性的思维模式是“确定目标，完成目标”，他们难以同时处理多件事，必然有所侧重。

所以，男性的热恋期通常没有女性长。热恋期过后，他们的感情浓度自然下降。而女性依然倾向于将“对方是否提供高浓度的情感”作为衡量爱的唯一标准。

于是，两个人的战争一触即发。女性觉得男性不关注自己、不快速回复信息、不对自己嘘寒问暖，是不再爱自己的表现，从而变得咄咄逼人、焦虑不安。男性则会觉得女性不理解自己、失去自我、不再有魅力，不再是当初喜欢的那个女孩。

本节课中我们一起学习了恋爱磨合期矛盾产生的核心原因。我们了解了“感情浓度”与“感情深度”的区别，并从男女生理及思维模式的根本差异上，理解了为何男性在热恋期后感情浓度会自然下降，而女性却难以接受这一变化。认识到这并非简单的“爱”与“不爱”的问题，是化解矛盾的第一步。
