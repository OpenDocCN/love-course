# 墨多亲授女性婚姻成长课 第17课 - 当男人说，跟你相处很累的时候，他的真实想法是什么

大家好，欢迎来到第十七节课。当男人说跟你相处很累的时候，他的真实想法是什么？

## 案例分析

7年前，一位女士在回国飞机上遇到了气流，感到非常害怕。她的丈夫紧紧地抱着她，让她感到安心。在此之前，他们已经恋爱两年，但女士总觉得丈夫对她关心不够。结婚后，她发现自己始终觉得自己像是一个外人，而丈夫对婆婆的关心让她感到困惑。

## 女性的需求

女性在婚姻中最需要的是一种踏踏实实的安全感，能够被另一半放在心里，得到关注和保护。

## 男性心理

这位男士的家庭是典型的男尊女卑模式，他从小到大形成的思维习惯在婚姻中与妻子的诉求产生冲突。他习惯于照顾婆婆，而不知道如何满足妻子的需求。

## 解决方案

1. **改变相处方式**：既然男士觉得累，就给他空间，针对问题而不是针对他。
2. **保持一致**：在处理双方家长介入的问题时，夫妻双方要保持一致，避免矛盾放大。
3. **理解与支持**：男士在说累的时候，实际上是在寻找理解和支持。当夫妻双方放下委屈，站在同一阵线考虑解决方案时，问题才会逐渐淡化。

## 总结

本节课中，我们学习了当男人说跟你相处很累的时候，他的真实想法是什么。通过分析案例，我们了解到男性在婚姻中的心理和需求，以及如何解决夫妻之间的矛盾。希望这些内容能帮助大家更好地理解和处理婚姻中的问题。
