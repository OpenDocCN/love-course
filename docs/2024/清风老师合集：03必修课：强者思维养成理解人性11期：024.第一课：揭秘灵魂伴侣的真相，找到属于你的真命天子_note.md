# 清风老师合集：03必修课：强者思维养成理解人性11期：024.第一课：揭秘灵魂伴侣的真相，找到属于你的真命天子

## 概述
在本节课中，我们将探讨如何理解灵魂伴侣的真相，并找到属于你的真命天子。

## 灵魂伴侣的定义
灵魂伴侣并非一见钟情，而是灵魂和肉体的结合。**公式**：灵魂伴侣 = 灵魂 + 肉体。

## 价值互补
以下是影响择偶的四种价值：
1. **生育价值**：健康年轻的肉体。
2. **观赏价值**：化妆打扮、精致生活。
3. **情绪价值**：处理情绪波动，调动情绪。
4. **生存价值**：良好的物质条件、家世背景。

## 价值观
价值观分为八种：
1. **自主**：独立解决问题，自私。
2. **奋斗**：上进心强，缺乏耐心。
3. **影响**：洞察人心，控制欲强。
4. **传统**：道德感强，缺乏创新。
5. **安全**：自律，适应变化差。
6. **仁慈**：承担责任，易受委屈。
7. **精彩**：想象力丰富，情绪不稳定。
8. **博爱**：同理心强，难专注感情。

## 体验极致
在情感中，我们要保持双向奔赴，让对方在止损点之前保持投入，避免过度投入或过度引导。

## 总结
本节课中，我们学习了灵魂伴侣的定义、价值互补、价值观和体验极致。希望这些知识能帮助你找到属于你的真命天子。
