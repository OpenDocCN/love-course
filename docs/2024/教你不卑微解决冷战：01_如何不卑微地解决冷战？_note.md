# 教你不卑微解决冷战：01_如何不卑微地解决冷战？_note

在本节课中，我们将要学习冷暴力的形成过程与反制手段。课程将首先分析冷暴力的两种主要类型，并重点讲解后天“被培养出来”的冷暴力的三个阶段演变。随后，我们将针对每个阶段，提供具体、可操作的反制策略，帮助你维护关系中的平等与尊严。

冷暴力最让人难受的是，人际关系中最基础的回应都难以得到。什么都明白的人，最温柔，也最冷漠无情。本文将结合咨询经验，分享冷暴力的形成过程以及反制手段。

冷暴力一般分为两种类型。第一种是这个人在一起之前就已经习惯用冷暴力处理感情。其形成原因可能是之前的恋爱经历、家庭原因或童年经历等。对于第一种习惯性冷暴力的人，不是本节课要重点分享的内容。

以有限的经验来看，对于长期习惯用冷暴力处理感情的人，他们就像是情绪黑洞。你要么接受并被吸入，要么远离以保平安。第二种则属于被培养出来的冷暴力，也是本节课想要重点分享的内容。

其实天生就冷漠的人是少数。大部分冷暴力都是被伴侣培养出来的。两个人刚在一起时，关系中并不存在冷暴力。你生气了对方会哄，有问题大家也愿意沟通。

但是相处一两年后，处理矛盾的方式就慢慢演变成了冷暴力。这属于培养成的冷暴力。很多存在冷暴力感情危机的关系中，后天培养出冷暴力的情况偏多。

而培养冷暴力的过程，又分为三个阶段。

## 第一阶段：沟通失效

上一节我们区分了冷暴力的类型，本节中我们来看看后天冷暴力是如何被“培养”出来的。这个过程始于沟通失效。

每一个喋喋不休的人背后，都会有一个沉默不语的伴侣。而那个沉默不语的人，正是在无休止的喋喋不休下，觉得沟通这件事没了意义。

有一位男性来访者说过，一开始吵架时，他还愿意去哄对方。可是后来，他们之间已经不是吵架，变成了单方面的指责。他说什么都会被反驳回来，任何解释都不被信任。他感到非常无力，那种感觉就像努力了和没努力结果都一样，那还努力干嘛？

这时对应的心理变化，就是他开始觉得跟你说道理已经毫无意义了。其核心逻辑可以概括为：
**无效沟通 → 努力无果 → 放弃沟通意愿**

## 第二阶段：尝到甜头

当沟通被认定为无效后，关系便会进入下一个危险阶段。以下是此阶段的关键发展。

经过第一阶段的发酵后，对方开始采取不回应策略。面对你的任何情绪都不想回应，随便你怎么吵。一旦吵得太猛，就躲起来不理你，眼不见为净。

然后对方在躲起来不回应的时候，你就被冷暴力了。冷暴力会让你难受，感觉被无视，自我价值感丢失严重。因为害怕冷暴力带来的消极影响，于是你服软了，妥协了。

这时对应的心理变化就是对方会觉得：原来我只要不说话，他就会自己反思和服软了。没想到冷暴力还有这种好处。其行为模式可以描述为：
**实施冷暴力 → 伴侣痛苦妥协 → 施暴者获得关系控制权**

## 第三阶段：熟练运用

一旦冷暴力成为有效工具，它就会被频繁使用。以下是此阶段的特征。

经过第二阶段的启发后，对方发现使用冷暴力可以避免沟通，就能让伴侣安静下来。于是以后只要能用冷暴力的地方都会用起来。每次用的时候还都十分有效。

这时对应的心理变化，就是拿捏你了，知道你顶不住冷暴力。只要不想做、不想听，就会动用冷暴力来对付你。

以上属于培养冷暴力的形成过程。接下来，针对每个阶段，我们来讲解反制手段。

## 第一阶段的应对：重建沟通有效性

面对沟通失效的初期，关键在于扭转对方“沟通无用”的认知。以下是具体做法。

你要让对方能够意识到还是能跟你说道理的。只要让他觉得还能跟你沟通，而不是每次沟通都被你反驳、指责，他就不会进入第二阶段。

具体做法是：吵完架后过两天，等双方情绪冷却下来，你就挑对方跟你吵架过程中说过的某一个点，稍微认可一下。

例如，你可以说：“之前吵架，你说我总是反驳你。我昨天没有意识到，今天上班跟同事沟通时，我看到他总是对我的方案提出反对意见，我开始理解你说的话了。”

就这么一句话，足以让对方感知到你还是能够沟通、能够听进话的，而不是一个只知道指责反驳的人。其核心操作可总结为：
**情绪冷却后 + 选择性认可对方观点 = 传递开放沟通的意愿**

## 第二阶段的应对：坚守底线，绝不妥协

当对方开始使用冷暴力时，你的应对方式决定了事态的发展方向。以下是关键原则。

他冷暴力你，你不一定非要和他冷，但是不能妥协，必须有底线。一定不能让他尝到冷暴力的甜头，否则冷暴力就只有0次和无数次。

具体做法是：当对方开始冷暴力你的时候，你就摆明立场说：“你需要一个人安静一下是吗？行，没问题。这段时间我不追问你，给你一个礼拜的时间够吗？如果你不出声，我就默认你答应了。这一个礼拜我不追问你，期间你想和我聊就找我，不想就不用找。但是一个礼拜之后，你得跟我说清楚。”

如果一个礼拜后，对方还是采用冷暴力应对，那就需要考虑结束这段关系。这个策略可以编码为以下对话框架：
```
if 对方开始冷暴力：
    提出：“你需要安静吗？我给你[时间，如一周]空间，不追问。”
    设定边界：“但[时间]后，我们需要沟通清楚。”
if 到期后对方仍冷暴力：
    认真考虑结束关系。
```

## 第三阶段的应对：及时远离

如果冷暴力已经成为一个熟练的控制工具，那么局面将变得非常困难。以下是此阶段的建议。

其实第三阶段和开头说的第一种习惯用冷暴力处理感情的人是一样的了，基本无解。他知道冷暴力可以让他自己很爽，能够拿捏住你，那么这个时候他基本不会轻易放弃这个权利。

这个时候需要做咨询的是他，不是你了。我对冷暴力的态度就跟对家暴的态度一样。在我看来，冷暴力和家暴没啥区别，都是通过固定手段来让自己在关系中暂时获得控制权。冷暴力用的是无视，家暴用的是拳头。

本节课中我们一起学习了冷暴力的演变与应对。我们首先区分了习惯性与后天培养的冷暴力，并重点剖析了后天冷暴力形成的三个阶段：从**沟通失效**，到**尝到甜头**，最后**熟练运用**。针对每个阶段，我们提供了相应的反制策略：在初期**重建沟通**，在中期**坚守底线**，在晚期则建议**及时远离**。记住，健康的关系建立在平等与尊重的基础上，任何试图通过冷暴力获取控制权的行为，都值得你警惕并坚决应对。
