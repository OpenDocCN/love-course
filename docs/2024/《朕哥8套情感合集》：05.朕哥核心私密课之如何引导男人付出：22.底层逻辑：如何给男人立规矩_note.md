# 《朕哥8套情感合集》：05.朕哥核心私密课之如何引导男人付出：22.底层逻辑：如何给男人立规矩

🎼 如何给男人立规矩？很多女生都希望能给男人立规矩。但不知道何时立、如何立。立规矩的最佳时机称为关键期，正歌课学员都了解。

🎼 在关键期立规矩最佳。例如，男人追求你时，是第一个关键期。此时，他为了得到你，会接受你提出的任何规矩或条件。

🎼 抓住立规矩的时机至关重要。例如，男人身份升级、你怀孕、长时间分离、深夜想你时，都是立规矩的好时机。

🎼 初期引导男人立规矩，委婉提要求。例如，男人追求你时，你可以委婉地表达对仪式感的喜好，他会主动为你准备。

🎼 关键期一旦错过，立规矩难度增加。最佳方法是引导式提要求，除非男人段位很低，否则可逐渐转为直接。

🎼 从初期进入热恋期，用奖励方法立规矩。记住，奖励永远在惩罚之前，无恩不施威。

🎼 热恋期后，建立奖惩机制。明确告诉男人，做对有奖励，做错有惩罚。

🎼 惩罚要适度，及时给予反馈。例如，男人做错事，你可以让他哄你，但要让他知道，哄你后你会慢慢变好。

🎼 总结规矩，强化记忆。通过总结，让男人清楚对错，强化规矩。

🎼 奖励和惩罚可以是物质的、精神的、身体的、情绪的或言语的。

🎼 本节课介绍了如何给男人立规矩，包括抓住关键期、引导式提要求、建立奖惩机制、适度惩罚、及时反馈和总结规矩。
