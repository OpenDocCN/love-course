# 情感社 - 强者内核 - 【两性关系】中的通关秘籍：66、如何定位男人对异性的边界感

## 概述

在本节课中，我们将学习如何定位男人对异性的边界感。边界感对于每对情侣和夫妻来说是不同的，需要从多个维度进行界定。

## 一、是否离得开对方

**公式**： 是否离得开对方 = 对方行为（如聊骚） + 个人感受（如痛苦、不舍）

**步骤**：

1. 评估自己是否能够接受对方的行为。
2. 评估自己对这段感情的经营能力。
3. 评估对方的性格。

**判断**：

- 如果无法接受对方的行为，但仍然不舍得离开，则可能属于“插曲”。
- 如果能够接受对方的行为，则可能属于“红线”。

## 二、对方的价值是否高于你

**公式**： 对方价值 = 社会价值 + 家庭价值

**步骤**：

1. 评估对方的社会价值（如收入、名誉地位）。
2. 评估对方的家庭价值（如养家能力、对家庭的贡献）。

**判断**：

- 如果对方的价值高于你，则可能存在边界感问题。

## 三、自我定位

**公式**： 自我定位 = 对方工作 + 个人期望

**步骤**：

1. 评估对方的工作情况（如发展前景、圈子）。
2. 评估自己对这段感情的个人期望。

**判断**：

- 如果你对这段感情的个人期望过高，则可能存在边界感问题。

## 四、网络行为

**公式**： 网络行为 = 点赞 + 聊天 + 约会

**步骤**：

1. 评估对方在网络上的行为。
2. 判断是否超越边界感。

**判断**：

- 点赞、聊天不属于超越边界感。
- 约会属于超越边界感。

## 五、工作环境

**公式**： 工作环境 = 同事关系 + 行为表现

**步骤**：

1. 评估同事关系。
2. 评估对方的行为表现。

**判断**：

- 同事之间开玩笑不属于超越边界感。
- 约会、暧昧行为属于超越边界感。

## 六、固定关系的朋友

**公式**： 固定关系的朋友 = 朋友关系 + 相处方式

**步骤**：

1. 评估朋友关系。
2. 评估相处方式。

**判断**：

- 如果朋友关系正常，相处方式没有暧昧，则不属于超越边界感。

## 七、刻意找陌生女人聊暧昧话题

**公式**： 刻意找陌生女人聊暧昧话题 = 聊天 + 约会

**步骤**：

1. 评估对方的行为。
2. 判断是否超越边界感。

**判断**：

- 聊天、约会属于超越边界感。

## 八、客户关系

**公式**： 客户关系 = 工作关系 + 个人感受

**步骤**：

1. 评估工作关系。
2. 评估个人感受。

**判断**：

- 如果工作关系正常，个人感受没有受到影响，则不属于超越边界感。

## 九、专属气质

**公式**： 专属气质 = 气场 + 感觉

**步骤**：

1. 评估对方的气场。
2. 评估自己的感觉。

**判断**：

- 如果对方气场强大，自己感觉良好，则不属于超越边界感。

## 总结

本节课中，我们学习了如何定位男人对异性的边界感。在判断边界感问题时，需要从多个维度进行思考，避免非黑即白的判断。
