# 驾驭关系的能力：情绪价值：03人内在三个组成部分（一）

**概述**：在本节课中，我们将探讨人的内在三个组成部分，帮助理解人际关系中的情绪反应。

## 真正的自我

**核心概念**：**真正的自我**是指人类善良、优质的品质，如自信、平静、关怀他人、好奇心、勇气、创造力等。

**内容**：
- 真正的自我是人类共有的品质，占比越大，人越受欢迎，能量越强。
- 放大真正的自我可以通过夸奖、积极强化等方式实现。
- 例如，认可伴侣的自信或人际关系能力，使其感受到价值和受欢迎。

## 脆弱的自我

**核心概念**：**脆弱的自我**是指每个人内在的脆弱点，如自卑、恐惧、无助、无力、委屈、自责等。

**内容**：
- 每个人都有脆弱的部分，源于不同的成长环境和原生家庭。
- 脆弱点可能表现为对成就感、人际关系、公平等方面的担忧。
- 例如，自卑可能源于童年经历，如家庭贫困、父母争吵等。

## 防御的自我

**核心概念**：**防御的自我**是指当脆弱的自我被触及时，启动的防御机制，如攻击、指责、批评、焦虑、对抗、愤怒等。

**内容**：
- 防御机制是保护脆弱自我的方式，但可能导致冲突和误解。
- 防御机制包括攻击、指责、批评、焦虑、对抗、愤怒等。
- 例如，当伴侣误解或批评时，可能会引发愤怒或攻击。

**总结**：本节课中，我们学习了人的内在三个组成部分：真正的自我、脆弱的自我和防御的自我。了解这些组成部分有助于理解人际关系中的情绪反应，并采取更有效的沟通方式。
