# 魅力之道全板块：021.表达-社交中的框架博弈（第二十一天）

大家好，今天我们将探讨一个有趣的概念——框架博弈。在之前的课程中，我们学习了聊天和表达，这些内容都涉及我们的观念和信念。这些观念和信念无论是否被明确表达，都隐含在我们的行为中。

**框架**可以理解为我们对事物的看法和观念。当我们在聊天中与对方对同一事物有不同的看法时，我们可以选择去了解对方，而不是试图改变对方。有些人喜欢辩论和争论，他们试图改变别人的看法，挑战对方的框架。

**记住**“框架博弈”这个概念。一般来说，我建议在聊天中，如果有分歧，了解对方为什么这样想就好，不要试图改变对方。因为改变对方是不可能的，只有对方自己才能改变。

你可以表达自己的看法，说明为什么这样想，如果别人觉得有道理，他们自然会选择性地接受你的看法。例如，小A可以不直接说小B是错的，而是说：“我吃过一家寿司店，味道很好。可能你之前吃的不够好。”然后推荐小B下次去尝试。

如果小B去了，觉得很好吃，那么他的框架就得到了延伸。他的新框架是：“我不喜欢一些店的寿司，但我喜欢一些店的寿司。”**注意**这里的关键词是“框架延伸”。很多时候，两个人的框架不同，是因为站在一个较低维度的角度。但如果站在更高的角度，框架是可以融合的，可以达到一致的。

例如，在聊天时，你们的观念可能不同，框架也不同，但聊天本身可以站在更高的维度。😊 对于这场谈话，你们的框架很可能是聊天，所以这场聊天最好是比较舒适和有趣的。如果没有必要，不要乱去和别人框架博弈，即使你赢了，对方又怎么样呢？你赢了一个小局的框架，却破坏了和谐的气氛，你输了一个更高维度的一个宏观的框架。

再次强调，社交一般讲求在和谐的气氛中进行。除非你被爆犯了边界，否则不要去和对方博弈，因为这根本就不是你的目的。

那么，如何才能在框架博弈中成功呢？这取决于你的筹码有多少。如果你试图说服一个刚认识的人喜欢寿司，那么你没有任何筹码。因为对方完全可以一走了之，不和你聊天，对方不会付出任何代价，你对于对方没有任何权利。

在恋爱中，如果女生对男生说：“你不知道这件事，我觉得我们分手吧。”那么女生的筹码是男生对她的需求感以及她对于男生的可替代性。这个时候，女生不用和他讲道理。我们这个时候是讲权利的天平的，在框架博弈的时候。

如果男生找不到更好的女生，又特别喜欢这个女生，同时他有没有意志力去坚持自己的原则，那么他就被拿捏了。他只能接受女生的不平等条约。如果男生身边都是类似这样的女生，而且不少这样的女孩子都喜欢她，那么他就有更多的底气去和现在这个女朋友去谈判，去接受那些合理要求，不接受那些不合理的要求，让关系得到继续。

如果男生根本就不在乎这个女生了，那么他会觉得：“太好了，正好没有机会提分手。”在这种情况下，女生就一点筹码都没有。

刚刚所说的就是几种筹码的多少的情况。但我也补充说明一下，我之前说的那个女孩子那种表达，你不怎么怎么样，我就分手，一般会这样子表达的是比较不成熟的女孩。但是她也不一定就存了坏心。可能她看到周围的闺蜜被怎样怎样对待，从而有了对比之心，也可能是她在男生那边需要的安全感没有被满足。

作为男生，解决方案不一定要正面去硬刚框架。但是框架博弈，永远你记住是底层的动态，而在长期关系当中，你怎么去调整自己和对方的框架，我们这边是不讲的。回到正题。

那么，为什么在这个场合需要去框架博弈呢？因为在这儿这个宏观框架是男女朋友框架，大家要继续走下去，继续相处。如果说一方不满意另外一个人，不满意现在的相处模式，用男女朋友的关系的破裂来做威胁，那这个框架是没有办法被更大的框架所延伸进去的，所以就必须相互对抗，现在就要开始拼筹码了，谁的筹码多，谁就能赢。

当然也有的时候是你让对方以为你的筹码更多，你也可以赢，那两性当中的冷战其实也可以用这个模型来解释。框架博弈其实是无处不在的，而且他是动态的，只不过很多时候他不会浮出水面而已。比如说你的老板怎么对待你，他是否尊重你，如果说他不尊重你的时候，你没有指出来，那他就可能更加肆无忌惮的不尊重你。

要么他觉得他拿捏你害怕冲突，要么他觉得你没有更好的其他的工作可以去选择，你必须去倚仗他，要么他根本就不在乎你会不会离开。我们和所有人都有相处的框架，有相应的边界。我们的框架也在不停的调整当中，我希望大家从今天开始对这个一定要有意识，已经形成很久的框架，意味着权力的天平已经形成了。

你不可能马上去扭转。比如说你的父母对你的支配性很强，你如果要挣脱这样的控制，你可能需要搬出家里来。那在这个过程中，他们不适应，可能会拿你不孝顺，你变坏了，等等这些所谓的道德标准来绑架你，去试图拿回对你的控制权。

当然这只是一种例子，每个父母是不一样的。所以说看了这节课之后，首先我希望大家要对和别人关系的框架要有意识。其次，我希望大家做的事情是长期的去建设好自己。

你要增加在自己各方面的筹码。比如说如果你还要靠父母养活，那他们对你当然有话语权。又比如说如果你在两性上你没有什么选择，没有什么优质的对象来喜欢你，那你当前的对象，如果是你最好的选择，他当然可以拿捏你。

你要让自己在各方面都作为一个有选的人，同时你要让自己做一个在观念上不会被道德绑架的人，你要让自己对你遇到的各种事物，要有你自己的看法，不要被他人所影响。这样之后你会活得非常的自由，你不会被绑架。

顺便补充一下，有你自己的看法，这个不叫心态不开放。心态开放，指的是你去愿意了解，而不是对事物没有自己相信的看法。

好，接下来是今天的作业，今天的作业是回想过往一次，你和别人框架博弈的经历。你们博弈双方，你们冲突的框架分别是什么？以及彼此的筹码有多少是什么？你当时框架博弈赢了还是输了，或者说结果是什么样子的，你反思一下，为什么是这个结果，如果再让你去框架博弈一次，你会有什么样的不同的做法哦。

还有就是要补充一下，框架博弈，有时候不一定是赢或者是输，也可以是双方离开，没有赢，没有输，就是我们叫no deal交易没有达成。还有一种情况叫做妥协，就是双方各退一步，这样子你们也是可以继续往前走的。
