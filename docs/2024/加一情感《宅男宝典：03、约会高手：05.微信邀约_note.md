# 加一情感《宅男宝典》课程03：约会高手 - 05. 微信邀约 💬

在本节课中，我们将要学习如何通过微信向女生发出邀约。正确的邀约方式可以显著提高对方接受邀请的概率，避免被直接拒绝。我们将分步骤讲解从模糊邀约到最终确定时间的完整流程。

## 概述：邀约的三个核心步骤

一次成功的邀约通常包含三个步骤：**模糊邀约**、**判断对方空闲时间**和**确定具体邀约时间**。遵循这个流程，可以让你显得更有礼貌，同时降低被拒绝的风险。

## 第一步：进行模糊邀约

上一节我们介绍了邀约的整体框架，本节中我们来看看第一步的具体做法。模糊邀约是指提出一个不包含具体时间和地点的、概略性的邀请。

生活中常见的例子是，遇到多年未见的同学，对方说“有时间大家一起去喝酒”或“有时间大家一起聚一聚”。这通常被视为客套话，你的回复往往是“好呀”或“好吧”，一般不会拒绝。

因此，对女生使用模糊邀约时，同样不易被拒绝。你可以说“有空一起去喝东西”或“有空一起去吃好吃的”。这里的关键是只提出一个大概的想法，**不明确到具体的某一天或某个时刻**。

这样的说法更不容易被拒绝。

## 第二步：判断对方的空闲时间

当模糊邀约获得对方认可（例如得到“好呀”的回复）后，我们需要进行第二步：判断对方的空闲时间。

这时，你需要询问女生通常何时有空，而不是直接指定一个时间（例如“明天下午三点见面”）。直接指定时间会让女生感到不受尊重，也更容易拒绝你。对方可能会想：“你怎么知道我明天下午没事？”或者“我自己的行程都还没安排”。

因此，出于礼貌和对女生的尊重，我们必须先了解她的空闲时间。

以下是正确的询问方式：
> 那你什么时候有空？

这样的回复显得小心翼翼，是一步一步的试探，充分尊重了对方的时间安排。

## 第三步：确定具体的邀约时间

在了解了对方的空闲时间后，我们就可以进行最后一步：确定具体的邀约时间。你需要根据女生提供的信息来敲定第一次约会。

女生的回复通常有以下几种情况，我们需要灵活应对：

### 情况一：回复“下班后就没事了”

如果女生说“我下班后就没事了”，这通常意味着她工作日晚上都有空（一般是下午五六点或六七点下班）。

很多人的做法是直接说“那就明天等你下班后吧”或“后天等你下班后吧”，但这并不是最佳选择。因为如果对方明天或后天临时有事，你就等于多给了对方一次拒绝你的机会。

聪明的做法是：**给对方做选择题**。

你可以回复：
> 明后天？

这样，如果她明天有事，可以选择后天；如果后天有事，则可以选择明天。让她做选择也显得你的目的性不那么强。

### 情况二：回复“周末有空”

如果女生说“一般都是周末有空”，而你们聊天的当天是周一或周二，距离周末还有四五天，这时不宜立刻敲定周末的约会。因为很难预约一个人四五天后的行程，她自己也可能无法确定周末是否有其他安排。

因此，当你得知她周末有空，而聊天是周初时，建议不要急于完成最终邀约。应先保持联系感，等到周三或周四再聊天时，去确定周末的约会。

### 情况三：聊天当天已是周四

如果聊天当天已经是周四，女生表示周末有空，这时就需要灵活运用。你可以直接给出选择题：

> 那我们就周六或周天？

从而减少她拒绝你的可能性。
