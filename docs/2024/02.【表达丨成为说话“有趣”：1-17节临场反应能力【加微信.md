# 课程02：成为说话“有趣”的人 🎤
## 第21集：提升临场反应能力（1-17节）

在本节课中，我们将要学习如何锻炼临场反应能力，以便在各种场合下能够快速、有趣地回应他人。许多人在学习了沟通技巧后，仍然在实际场景中反应不过来，本节课将提供具体的方法来解决这个问题。

上一节我们探讨了有趣沟通的基础，本节中我们来看看如何将知识转化为即时的反应能力。

假设你正在学习骑自行车。两天后，你学会了保持平衡、直行和拐弯。第三天，你高兴地上路，却发现下雪导致路面湿滑。这时，一只小狗突然出现在前方。你试图刹车并拐弯以避免撞上，结果摔倒了。你心里疑惑：我不是已经学会骑自行车了吗？

这个例子说明，**掌握基础技能不等于能应对所有突发状况**。学习理论（如“下雪天应点刹”）与实际在湿滑路面紧急刹车是两回事。同样，学习了有趣的说话方法，也不代表你能立刻在各类场合灵活运用。临场反应能力需要通过反复经历各种情形来锻炼。

如果你希望锻炼自己的临场反应能力，做到随时随地有趣地回应，以下是五点核心建议。

首先，你需要敢于开口尝试。无论在会议发言还是尴尬场合，都必须先说出话来。即使最初的尝试效果不佳，甚至说得“很烂”，也必须说出来。因为只有通过实践，你才能获得反馈，了解哪些方式有效、哪些无效。说话能力无法仅靠倾听来提升。例如，培训师需要通过观察学员的现场反应（笑或点头）来调整自己的表达方式。

第二点是避免自我过滤。我们常常在话到嘴边时，因犹豫而将其咽下。这种过滤会扼杀许多有趣的想法。在之前的课程中，我们举过一个例子：一个小女孩被称赞衣服漂亮时，她直接回答“对呀”，引得大家发笑。许多成年人内心可能也认同夸奖，但会因为觉得“需要谦虚”或“不合适”而过滤掉真实的回应，从而失去趣味。我们应该像孩子一样，减少这层心理过滤（当然，伤害他人的话除外）。

上一节我们提到了减少自我审查，本节中我们来看看第三个关键：保持轻松，避免严肃对抗。

第三点是不严肃、不反击、不辩解。这是临场反应的关键。例如，当别人说“你今天这衣服真土”时，如果你严肃地反击（如“你才土”），对话就会变成对抗，趣味全无。正确的做法是，第一时间告诉自己不要严肃，放下防卫心理，不去辩解。然后，运用我们学过的有趣思维（如自嘲或转折）来回应。对于刚才的例子，可以这样回应：“因为我人太洋气了，得穿件土衣服往下拉一拉。” 这便是一种有趣的处理方式。

第四点是培养玩耍心态。为了更好地理解“玩耍”，我们可以看一个与说话无关的例子：你和朋友玩“石头剪刀布”，输的人被贴纸条。在玩耍状态下，输家可能会说“哎呀，我真倒霉”，赢家则会开心地调侃“哈哈，你又输了”。游戏规则可以被轻松打破，比如输家逃跑或互相贴纸条。整个过程轻松、调侃、充满嬉笑。

回到说话上，要想临场反应好，就需要把对话当作玩耍。用轻松、调侃、嬉戏的心态去应对，甚至可以善意地“打破”常规的严肃对话规则。例如，用“我人洋气所以穿土衣服”来回应批评，就是一种调侃和玩耍。

以下是第五点建议。

第五点是多参加社交场合。这包括各种形式的聚会，如同事聚会、朋友聚会、同学会，或业余爱好组织（如读书会、车友会）。当你与不同的人在多样化的情境中交谈、开玩笑、讲故事后，你的临场反应能力会逐渐提升。锻炼说话能力不能闭门造车，它需要开放式的实践。这就像学习英语，最好的方法是每天与说英语的人一起生活。

本节课中我们一起学习了提升临场反应能力的五个核心方法：**敢于开口尝试**、**减少自我过滤**、**保持轻松不对抗**、**培养玩耍心态**以及**多参与社交实践**。

最后，这不是一个书面练习，而是一个行动倡议。请你想一想，在明天或下周，有哪些场合（如一次聚会或一场会议）可以锻炼你的临场反应能力。尝试运用以上五点建议，大胆地说出来，倾听他人的反馈，不要自我过滤，并将这种锻炼重复无数次，直到你的临场反应能力得到切实提升。
