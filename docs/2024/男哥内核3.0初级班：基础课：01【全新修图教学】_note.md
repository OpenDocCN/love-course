# 男哥内核3.0初级班：基础课：01【全新修图教学】

大家好！今天我们将学习如何使用美图秀秀进行图片修图。以下是详细的教程：

## 概述

在本节课中，我们将学习如何使用美图秀秀进行图片修图，包括美化图片、人像美容、瘦脸瘦身、美妆和面部重塑等基本功能。

## 学习方法

1. **理解修图过程**：修图过程是由大到小再到大，先调整整体图片感觉，再进行细节处理。
2. **练习**：修图需要不断练习，熟能生巧。

## 软件选择

选择美图秀秀作为修图软件，因为它功能全面，易于操作。

## 修图步骤

1. **美化图片**：
    - 调整亮度、对比度、锐化、饱和度等参数。
    - 注意：对比度、锐化、饱和度不建议过度调整。

2. **人像美容**：
    - 使用祛皱笔去除法令纹、痘印等。
    - 使用去皱笔去除油光。
    - 注意：曲皱笔使用时要注意力度，避免留下痕迹。

3. **瘦脸瘦身**：
    - 选择手动模式，将脸部肌肉往内推。
    - 注意：不要过度调整，以免显得不自然。

4. **美妆**：
    - 选择合适的眉毛形状。
    - 调整眉毛、眼睛、鼻子、嘴唇等部位。

5. **面部重塑**：
    - 调整脸型、眉毛、眼睛、鼻子、嘴唇等部位。
    - 注意：调整眼高，使眼睛更有神。

6. **滤镜**：
    - 选择合适的滤镜，调整滤镜强度。

7. **美颜**：
    - 根据需要开启或关闭美颜功能。

## 总结

本节课中，我们学习了如何使用美图秀秀进行图片修图，包括美化图片、人像美容、瘦脸瘦身、美妆和面部重塑等基本功能。希望大家通过练习，能够熟练掌握修图技巧。
