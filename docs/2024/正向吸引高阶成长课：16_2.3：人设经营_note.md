# 正向吸引高阶成长课：16_2.3：人设经营 🎭

在本节课中，我们将要学习如何通过“人设经营”来塑造个人形象，并学会如何有效地讲述自己的故事，从而增强个人吸引力。

## 🎼 第三节：形象塑造与人设经营

我近些年有一个深刻的感受：会讲故事、讲好故事至关重要。许多人拥有丰富的学识、才华和经历，但若不主动展示，他人便无从知晓。只有经过长期相处，这些吸引点才会被慢慢发现。

例如，我有一位女学生非常漂亮。她从985高校会计专业毕业后，并未从事会计工作。她带着母亲周游世界五年，足迹遍布七十多个国家。期间在加拿大驾驶房车时遭遇车祸，断了两根肋骨，险些丧命。这个故事本身非常震撼。

然而，她多年来只是在朋友圈分享风景和自拍。大家看不到这背后的动机、故事以及那些惊险的经历。如果你不与她深入交往，就完全不会了解这些。因此，她常向我倾诉，认为自己遇人不淑。

她经常被人肤浅地当作花瓶对待。后来尝试经营自媒体账号，也未能获得太多关注。主要原因在于她没有讲好自己的故事。相反，有些人外貌并非最出众，却能凭借某些事件或话题，将自己的热度和人格魅力推向极致。

例如，在2023年底，凭借电视剧《繁花》而爆红的演员辛芷蕾。她明确表示自己跑了八年龙套，非常渴望走红，直接将欲望和野心展露无遗。只因她从小家境困难，姥爷身患重病，父亲瘫痪在床，所以极度渴望金钱与成名。

她希望拥有更多选择权去挑选自己喜欢的角色，也希望家人不再因贫穷而受苦。如此一来，整个人物形象就立住了。她不仅好看，更具备了人格魅力。还有在2024年初，凭借“半年减肥100斤”这一事件，让电影《热辣滚烫》票房大卖的贾玲。她同样是故事讲得好，让人格魅力凌驾于作品之上，从而获得了巨大成功。

## 📖 从小说创作中学习讲故事

在学习如何讲述自己的故事之前，我们先来分析小说的创作。因为小说通常更加精彩，能牢牢抓住读者的注意力，让人欲罢不能。而且，小说不像电影依赖丰富的视觉画面，它仅凭几行文字就能引人入胜。

如果你掌握了小说的结构，再去讲述自己的故事，就会易如反掌。以下是几个经典小说的开头，同学们可以先感受一下：

*   **《百年孤独》** 的开头是：“许多年后，面对行刑队，奥雷里亚诺·布恩迪亚上校将会回想起父亲带他去见识冰块的那个遥远的下午。”
*   **《傲慢与偏见》** 的开头是：“凡是有钱的单身汉，总想娶位太太，这已经成了一条举世公认的真理。”
*   **《安德的游戏》** 的开头是：“我用他的眼睛观察，用他的耳朵聆听。我告诉你，他就是那个人，或者至少是我们能找到的最接近的人。”

大家可以看到，这些开头包含了时间的跳转、生死的冲突、反常识的普世价值观、动作细节的描述以及强烈的情绪表达等元素。

以下是小说结构的六个核心要素，大家可以记一下：

1.  **钩子**：吸引读者注意力的开头。
2.  **冲突**：故事的核心矛盾。
3.  **行动**：人物为解决问题采取的措施。
4.  **结局**：故事的最终结果。
5.  **展示**：通过细节描绘而非直接陈述来呈现。
6.  **情感**：故事所引发的情感共鸣。

## ❓ 用六个问题构建你的故事

现在，让我们沿着这个框架，向自己提出六个问题，来梳理和构建我们自己的故事：

1.  **目标与冲突**：我做这件事的目标或梦想是什么？其中是否包含了正反价值观的冲突？
2.  **阻碍与困难**：我的阻碍或天生的障碍是什么？困难通常产生于客观现实与主观意愿之间的鸿沟。太容易实现的目标无法打动人。相反，困难越大，人物形象才能越深刻。
3.  **努力与细节**：我是如何努力的？这里描述得越细节越好。越细节，越能让人产生画面感，越能打动人，也显得越真实。
4.  **初步结果**：结果如何？为了让故事有趣或出人意料，通常一开始的结果是不顺利的。
5.  **意外转折**：有没有能够超越努力的意外事件可以扭转结局？这个意外是怎么来的？
6.  **最终结局**：我是如何扭转局面的？最后的结局是什么？我感受到了什么样的情感？

这六个框架涵盖了人物的身份、背景、欲望、挑战、机遇和反转，是当前大部分小说或电影通用的叙事结构。

## ✨ 好人设的三个要领

心理学上有个“犯错误效应”，指的是犯点小错误，反而会让一个优秀的人看起来更有吸引力。而那些看起来完美无缺的人，则容易给人一种高高在上、缺乏亲和力、有距离感的感觉，反而不是最受欢迎的。

这就是为什么我们在推进关系时，需要偶尔暴露一些自己的小缺陷，让对方感受到鲜活的生命力。这个道理同样适用于讲述我们自己的故事。

因此，一个吸引人的人设通常具备以下三个要领：

1.  **美中不足**：优秀，但也会犯错，有一些无伤大雅的小缺陷。
2.  **能量向上**：代表正向的吸引力，积极乐观。
3.  **反差多变**：营造想象空间，让人感觉你有多面性。

## 💬 如何套用话匣子表达自己

现在我们来举例，套用上节课提到的三个聊天“话匣子”——成长、梦想和情感，该如何去表达自己：

*   **成长故事**：过去经历了（某个低谷），但是坚持（某种信念或行动），才让我现在获得了（这样的成就）。
*   **梦想故事**：从小有（某个梦想），虽然还没有实现，但它让我更有生活的动力和努力赚钱的意义。
*   **情感故事**：经历了感情上的（某种挫败），但是依然感恩（那段经历/那个人），才遇见了更好的自己。

相信90%的同学都能运用这些模板。

## 👩💼 七种讨喜的女性人设

除了成长人设，性格人设也有很多种。这里为同学们列举七种最讨喜的女性人设，无论在感情还是事业上都绝对受用：

1.  **敢爱敢恨**：例如，吃过感情亏，却仍愿抱有少女心的阳光女孩。
2.  **外强内柔**：例如，白手起家，但也有脆弱时刻的事业型女强人。
3.  **外冷内热**：例如，对人做事高冷有原则，但内心炙热，喜欢养宠物的小姐姐。
4.  **双商在线**：例如，冰雪聪明同时情商高、会处事的知性美人。
5.  **戏精/作精**：例如，内心丰富、爱哭爱闹、会撒娇但不蛮横、不记仇的磨人小作精。
6.  **美强惨**：例如，外表光鲜、能力出众，但身世经历坎坷的大女人（也称“缺陷美”人设）。
7.  **有信仰**：例如，从不按部就班，有自己梦想追求的富二代或富家女。

同学们可以找到最适合自己的性格人设，然后切记三个“不要”：不要做单一的恋爱脑，不要输出负能量，不要清高目中无人。这样就会非常有吸引力了。

## 📚 人设实例与情境人设

在我多年的情感经历中，也遇到过很多拥有有趣人设的女生。以下是五个例子，大家可以感受一下：

1.  **北漂女孩A**：单亲妈妈带大，年少辍学，凭一己之力在大城市扎根并拥有一份体面事业。性格乐观、勇敢、阳光，情商高。
2.  **都市女孩B**：家庭环境优越，学习优秀，但遭同学排挤，仍坚持追求梦想。性格心思细腻、较真，有点完美理想主义。
3.  **自由职业女生C**：家道中落，放弃铁饭碗工作，从零学习美业并将副业变主业。性格直爽、雷厉风行、敢想敢做、敢爱敢恨。
4.  **大龄圣女D**：经历七年感情无疾而终，背井离乡重新开始，用心经营生活，热爱跳舞、插花、摄影，好好爱自己。性格温柔甜美、心灵手巧、岁月静好。
5.  **都市白领E**：工作压力大，经常加班两点一线，但每个周末都去旅游，甚至会为了一杯奶茶奔赴远方。工作时很安静，玩耍时很疯狂。

除了成长和性格人设，还有一些适用于特定情境或状态的人设：

*   **感性人设**：在聊天中营造暧昧氛围，不要一本正经。学会开玩笑、试探和拉扯。聊天是建立人设和情趣的好方式，让对方觉得与你聊天有趣、放松、快乐。
*   **借助他人立人设**：尽量少说前任的不好或自己的悲惨经历，这容易让你掉价。相反，应树立“深情”人设，感恩过去。也少说闺蜜坏话，你身边的人也是你人设的一部分。
*   **慎立高冷人设**：高冷反人性，容易将自己包裹起来，只能吸引到带有强烈目的性的人。表面随和更利于交流。高冷人设也不利于暴露需求，难以引导对方付出。

## 🧠 策略性人设与识别他人人设

还有一些人设属于策略篇的知识点，大家可以提前了解：

*   **强者示弱人设**：记住，**信任不等于爱**。信任挂钩的是能力，爱挂钩的是情绪。示弱不等于真弱，弱可以让人怜爱，强才能获得信任。适当展现价值，让人觉得靠谱；在情感层面，再适当示弱，提供情绪价值。立场要坚定，态度要柔软。
*   **引导付出人设**：精神独立、向往更好生活、努力拼搏但离目标还差一步的“小女人”形象，容易吸引优质异性付出。让对方感觉在满足你的某种向往时，他能获得强烈成就感。可以通过提及（但非炫耀）他人赠送的贵重礼物来“借力”，同时表明自己边界感强。

学习完这些女性人设后，大家可能也会遇到一些擅长立人设的男性。这里提前打个预防针，警惕具有以下特征的人：突然闯入你生活、看似阶层高很多、一开始情绪浓度极高、过分强调仪式感、很粘人、树立文艺悲伤阴郁人设引发你圣母心、包装自己很受欢迎以增加你的患得患失感、强调自己优秀但有“无法改变的缺陷”（如失败婚姻、破碎家庭）并可能以此为借口不愿付出。这类人提供的情绪价值往往非常饱和，需要特别注意。

## 🌙 人设经营的本质与生活分享

有句话大家一定听过：你看到的只是别人想让你看到的。这就是人设经营的本质。每个人都有亮面和暗面。月亮之所以美，是因为它只向世人展示自己明亮的一面。

所以，哪怕生活平凡，现实骨感，我们依然可以呈现出我们想要呈现的样子。那么，如何分享生活，才能既体现人设，又显得优雅而不矫情呢？

例如朋友圈，大家常看到两种类型：第一种是当成摄影集，美食、美照、风景很养眼，但配文空洞；第二种是当成作文集，长篇大论显得有想法但啰嗦，让人没有阅读欲望。这两种比起不会拍、不会写的已经不错了，我们只需要兼备两种基本能力：

1.  **审美能力**：一双善于发现美的眼睛（第一章重点）。
2.  **表达能力**：结合视听信息，用文字把故事表达出来（本章重点）。

最后还有一点很重要：**目标导向**。发给什么人看？让那些想接近你、了解你的人能快速get到你的故事和人设，并被你的魅力吸引，这才是最重要的。

## 📱 朋友圈内容七大类型

至于具体发什么内容，这里为同学们列举七大类型，以后别再不知道发什么了：

1.  **完整故事**：生活中发生在自己或他人身上的、有目的、有行动线、有头有尾的故事。形式可以是日志、博客，或拼成图文故事、视频Vlog。
2.  **兴趣爱好**：日常的兴趣爱好打卡记录，如看书、健身、做饭，附带不同景别的自拍。朋友圈是免费广告位，别浪费。
3.  **发现美好**：生活中发现的美好、有感染力的事物，营造阳光向上、岁月静好的画面感，也可用文案巧妙暗示。
4.  **情绪分享**：即时性的、向上的情绪分享，如分享一首歌。情绪能量最吸引人。
5.  **观点评论**：对时事热点做出带有个人鲜明价值观的评论，表达强烈态度。转发文章时，不要只转发，提炼出你最认可的观点加以表达。
6.  **知识科普**：结合你的专业领域做知识型科普或避坑指南。最好的广告是利他。
7.  **形象展示**：艺术照和自拍，增加在对方朋友圈的曝光率。看得越多，越想和你创造交集。

结合第一章学习的拍照知识，最后分享一个“朋友圈打造的3+8+2指南”：三个设置、八个要点、两种感觉。大家可以结合上节课学习的108个兴趣爱好，来选择适合自己的人设和呈现方式。

## 📝 案例分析与课程总结

课程最后，我们来看一个学生案例。我这位学生是位聋哑人，平常情感交流更依赖于手机聊天。但老天给他开了另一扇窗，他对颜色和食物敏感度很高，很喜欢画画和美食。我觉得这很好，就应该多分享自己擅长的事。在聊天中，对方成功被他“小时候意外生病，但一直对生活保持热爱”的人设所感动，也愿意与他接触。

因此，聊天应该从对方的兴趣爱好着手，这样能更好切入，让对方有话可聊，给他倾诉和当“老师”的机会。男人天生喜欢当老师，两人交流时，哪一方说得越多，他就越容易“上头”。但不要光聊天，聊天只是切入口，最终目的要邀约到线下一起活动。可以套用情绪价值公式，通过仰慕和夸赞对方，表达自己想要被保护、被带领的需求，引导对方主动约会。

日常分享不仅仅是拍照，也要学会表达。审美品味可以通过表达来加分。对于那些垂直爱好的专业词汇、情绪化的表达、有记忆点的词汇，都需要我们在日常中积累。

**本节课总结**：我们一起学习了“人设经营”的重要性，掌握了通过六个问题构建个人故事的方法，了解了吸引人的人设要领和多种人设类型，并学习了如何在朋友圈等平台有效分享生活、塑造并展示个人形象。记住，真诚是底色，技巧是画笔，两者结合才能描绘出最动人的个人画卷。
