# 海蓝老师情绪梳理个案课：057-04 回看电梯场景

在本节课中，我们将通过分析一个电梯场景，探讨人们在面对他人困境时的不同情绪反应。

## 场景回顾

**原文**：就那想到这个场景，听妈妈说完了以后，这个难受度现在有几分6分6分哈。再看周围电梯上面，当时让佳佳最难受的，哪一些人都看着佳佳对对对，这是让最难受，就大家看热闹人好的。

**翻译**：想到这个场景，听妈妈说完后，难受程度现在是6分。再看周围电梯里的人，当时让佳佳最难过的，就是那些看着她的围观者。

## 情绪分析

**原文**：你如果是那个电梯里面的那个人群当一员看到一个小孩从上面下他妈妈在那骂的话，你估计你你的感受会是啥？挺心疼的，心疼谁啊，心疼这个小孩，你为啥心疼这小孩就这么小就妈妈就这么骂他我觉得我我会心疼这个小孩。

**翻译**：如果你是电梯里的人，看到一个小孩从上面下来，他妈妈在骂他，你大概会感到心疼。心疼这个小孩，因为他那么小，妈妈却这样骂他。

**公式**：心疼 = 小孩年龄小 + 母亲行为不当

## 好奇与嘲笑

**原文**：那你看他是因为什么看他好奇吧，其实就是好奇就是随便有出什么声响了就会去看对那你第一是好奇。第二呢，妈妈结果妈上面这个小宝宝，你好奇完了之后，你的感受是啥？

**翻译**：你看他是因为好奇才看的，就是随便有什么声音就会去看。你第一感觉是好奇。然后，看到妈妈在骂这个小孩，你的感受是什么？

**公式**：好奇 = 声音引起注意 + 对小孩的关注

## 情绪变化

**原文**：挺心疼的那其实所有电梯里面接人最开始好奇往后面看看后面其实。😊，人们带的是心疼，看的好好感受一下，我还看到一个嘲笑的你嗯。看热闹的你嗯哼我记得有一个男的是这样的，就是笑着看的。嗯哼他不是难过。

**翻译**：其实，所有电梯里的人最初都是好奇，然后会感到心疼。😊，人们带着心疼看着，我还看到一个在嘲笑的人。嗯，看热闹的人。我记得有一个男人就是这样，笑着看的。嗯哼，他不是难过。

**公式**：嘲笑 = 对他人困境的冷漠 + 对自己情绪的掩饰

## 总结

本节课中，我们一起分析了电梯场景中人们的不同情绪反应。从好奇到心疼，再到嘲笑，我们看到了人们在面对他人困境时的复杂情感。通过这个案例，我们可以更好地理解自己的情绪，并学会在类似情况下给予他人更多的关爱和理解。
