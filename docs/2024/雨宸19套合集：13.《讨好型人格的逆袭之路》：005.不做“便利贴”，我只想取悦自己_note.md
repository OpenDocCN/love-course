# 雨宸19套合集：13.《讨好型人格的逆袭之路》：005.不做“便利贴”，我只想取悦自己 😊

在本节课中，我们将要学习如何从“讨好型人格”转变为更关注自我感受的健康状态。核心在于从认知和行动两个层面进行改变，学会拒绝、建立自我价值，最终实现“取悦自己”的目标。

## 一、 从认知层面改变讨好型人格

上一节我们探讨了讨好型人格的成因，本节中我们来看看如何从根本的思维模式上做出调整。认知层面的改变是基础，它决定了我们如何看待自己与世界的关系。

### 1. 改变死板的个人规则

首先，你需要用理性思维去审视和否定那些僵化、不合理的内心规则。核心是：**你无法让所有人都开心**。在不伤害他人的前提下，让自己开心就足够了。

以下是具体的调整步骤：

*   **区分想法与行动**：想法可以很美好，但不必强迫自己事事做到。允许自己“想”和“做”之间存在差距。
*   **软化绝对化要求**：尝试将内心规则中的 **“必须”** 改为 **“可以”**。例如，将“我必须答应所有人的请求”改为“我可以选择是否答应”。
*   **建立“允许”心态**：允许计划出现意外，接纳一个不完美的自己。用“我好像可以…”的句式替代“我绝对要…”。
*   **关注结果，而非绝对**：尝试去做，然后看结果。即使结果“不是”你想要的，也能坦然接受。凡事不要追求绝对。

### 2. 建立评价内源，认可自我价值

其次，你需要将评价标准从外部转向内部，学会自我认可和奖励。

以下是建立自我价值感的方法：

*   **每日自我肯定**：每天对自己说一遍 **“我爱你”** 或 **“我很喜欢自己”**。持续练习能有效提升自我认同感。
*   **体验美好事物**：主动去做自己喜欢或能让自己放松的事情，如阅读、看一段视频。这能丰富精神世界，缓解紧绷的神经。
*   **建立自我奖励机制**：在完成一件事后，无论结果如何，都给自己一点小奖励（如吃顿大餐、看场电影）。这能强化 **“我的行动有价值”** 的信念。

总结来说，从认知层面改变，一是要破除刻板规则，培养灵活、允许的心态；二是要建立内在评价体系，通过自我肯定和奖励来认可自身价值。

## 二、 从行动层面改变讨好型人格

上一节我们介绍了世界观（“道”）的调整，本节中我们来看看方法论（“术”），即具体该如何行动。行动是将认知转化为现实的关键。

### 1. 清晰表达自己的需求（学会拒绝）

改变从习惯性说“是”到能够说“不”开始。

以下是练习拒绝的步骤：

*   **事前演练**：对着镜子练习拒绝的话语，选定几句备用（如“我现在不太方便，你可以自己处理吗？”）。
*   **实战套用**：在面对真人时，根据实际情况直接套用练习过的话术。
*   **养成新习惯**：这个过程就像解数学题，从 **套用公式** 开始，逐渐发展到 **自主作答**。通过不断练习，将“讨好-接受”的强迫性重复，转变为“评估-拒绝/接受”的新模式。

### 2. 划分事件的恐惧等级

不敢拒绝，往往源于对后果的灾难化想象。通过理性分析，你会发现大部分后果都可以承受。

具体方法是：将日常生活中可能遇到的、难以拒绝的场景，按照你感受到的恐惧程度进行分级。例如：
*   **低恐惧**：拒绝同事一起点奶茶的邀请。
*   **中恐惧**：拒绝朋友不太合理的借钱请求。
*   **高恐惧**：拒绝领导安排的额外无偿加班。

划分后你会发现，绝大多数事情都属于低、中恐惧等级，拒绝的后果远没有想象中严重。别人通常不会因为一次合理的拒绝而讨厌你，相反，这给了对方了解你真实边界的机会，有助于建立更舒服的交往模式。

### 3. 运用BAR技巧应对高压情境

当面临让你感到巨大压力、可能诱发讨好行为的情境时，可以使用BAR四步法来稳定情绪。BAR代表：**B**reath（呼吸）， **A**pplause（赞扬）， **A**ccept（接受）， **R**espect（尊重）。

以下是应用示例：
1.  **呼吸**：先调整呼吸，从急促混乱恢复到平稳深长，让身体先平静下来。
2.  **赞扬**：表扬自己：“我真棒，已经让情绪稳定下来了。”
3.  **接受**：接纳事实：“这件事已经发生了，我承认它的存在。”
4.  **尊重**：尊重结果：“我尊重已经发生的事实，并在此基础上思考如何应对。”

掌握心理学知识，就如同掌握了汽车维修技术。车仍会坏，但你能清晰知道问题所在、如何修理、代价多大，而不会陷入无助与自我归咎。BAR技巧能帮助你快速抓住情绪、分析其来源，并完成有效的自我调整。

---

本节课中我们一起学习了改变讨好型人格的双重路径：在认知上，要改变死板规则并建立自我价值感；在行动上，要通过练习拒绝、划分恐惧等级和运用BAR技巧来坚定自我边界。核心在于，将关注点从“如何让别人满意”转向“如何让自己舒适并真实地与人相处”。记住，**取悦自己，是收获健康关系的前提**。😊
