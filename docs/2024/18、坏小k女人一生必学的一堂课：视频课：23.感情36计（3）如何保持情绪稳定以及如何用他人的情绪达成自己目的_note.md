# 课程18：感情36计（3）🎭 如何保持情绪稳定与利用他人情绪

在本节课中，我们将要学习感情36计的第三部分核心技巧：如何建立内在情绪稳定性，以及如何利用他人的情绪来达成自己的目标。课程将分为两大板块，深入剖析情绪的本质、不稳定的原因，并提供具体的方法论。

## 一、如何做到情绪稳定？🧘‍♀️

上一节我们介绍了课程的整体框架，本节中我们来看看实现情绪稳定的具体路径。情绪稳定并非单一状态，而是分为四个层次。

以下是情绪稳定的四个层次，你可以对照审视自己所属的阶段。

1.  **第一层：被情绪掌控**
    被情绪找到并无法控制，对事物不进行判断，情绪不经过大脑，一点就炸。

2.  **第二层：无视负面情绪**
    内心放松、平和，以正向情绪为主，很难有事情激起强烈的负面情绪。

3.  **第三层：主动寻找情绪**
    以事情结果为导向，以利益为导向。这一层的人不会盲目追求情绪稳定，而是会为了达成好的结果，主动选择展现何种情绪。例如，刻意让自己生气、笑起来，或展现出霸气严肃的姿态。公式可表示为：**情绪 = 达成目标的工具**。

4.  **第四层：控制他人情绪**
    能预判他人的情绪，通过自身的状态、表现、肢体动作、神情等，引导对方产生预期的反应。

### 情绪不稳定的核心原因

了解了层次，我们需要探究情绪波动的根源。以下是导致情绪不稳定的几个关键原因。

*   **原因一：与认知和经历有关**
    让人痛苦和情绪不稳定的从来不是事情本身，而是个体对事情的**认知**和看法。例如，分手后，认知为“这辈子再也不能和这个人在一起”会带来巨大痛苦；而认知为“这只是关系的结束，并非感情的终结”则平和许多。情绪系统有一套信息处理机制，这套机制自动与你的认知能力对接。公式可表示为：**事件(A) + 认知(B) = 情绪/行为(C)**。

*   **原因二：轻易付出核心价值**
    当人们轻易付出自己视为核心价值的东西时，容易情绪失衡。例如，男人为女人花费过多金钱，或女人与男人发生亲密关系后，都可能因为害怕失去或投入过多而变得情绪不稳定。

*   **原因三：不以结果为导向**
    问题出现时，陷入情绪不稳定；问题扩大，情绪持续不稳定，形成负向循环。正确的做法是：问题出现时，忽略情绪，先解决问题；问题解决后，情绪自然减少，形成正向循环。

*   **原因四：关系失衡与情绪失衡**
    关系失衡本身并不可怕，可怕的是随之而来的情绪失衡。能够接受关系变化而保持情绪平稳，远比在失衡的关系中又吵又闹更为重要。

## 二、如何利用他人情绪达成目的？🎯

上一节我们探讨了如何管理自身情绪，本节中我们来看看如何将情绪，包括他人的情绪，转化为达成目标的工具。

核心思维是：**不是让情绪找到你，而是你主动去寻找并使用情绪。情绪只是你用来达到目的的工具，目的达到后，工具就该回收。**

例如，在争吵中，应牢记结果思维：**我想要对方做什么？** 厘清目标后，再释放情绪、提出需求、修正行为、立好规矩，最后给予台阶，圆满收场。

### 提供情绪价值

情绪价值是人际交往中的重要工具。对于女性而言，可以弱化其重要性，但需懂得以此来影响男性。情绪价值来源于波动，包括正向（如被赞美、支持）和负向（如失落、危机感）。

以下是针对不同对象的情绪价值策略。

*   **对年轻男性（25岁以下）**：他们享受被“拉扯”的状态，喜欢有点“作”的女生。此时，负面情绪价值与正向情绪价值同等重要。
*   **对优质男性（28岁左右）**：遵循“80%正向，20%负向”的原则。他们日常处理大量人际关系，更需要让其感到舒适的情绪价值。

### 建立信任与施加影响

利用情绪的关键在于建立信任和施加影响。以下是建立信任的几个核心方法。

*   **方法一：接纳情绪公式**
    面对高阅历者，当对方暴露弱点时（如“我压力很大，没人懂我”），使用公式：**接纳弱点 -> 顺着弱点 -> 表示接纳**。例如：“我知道你朋友多，别人可能觉得你不靠谱（接纳弱点），但我依然愿意和你相处（顺着弱点），因为看到你偶尔脆弱的一面，我觉得你活得挺累的，很真实（表示接纳）。”

*   **方法二：寻找深层次相同点**
    建立共鸣不只停留在表面爱好。例如，可以这样沟通：“我特别能理解你。我从小家庭不缺爱但缺物质，你正好相反。本质上，我们都在用一生追求最初缺失的东西。” 最终落脚于**共同追求缺失**，从而建立深度链接。

*   **方法三：建立信任的正确方式**
    1.  **信任 ≠ 交换秘密**：不要总说闺蜜或他人的坏话，你身边人的故事就是你人设的一部分。分享朋友的正能量故事。
    2.  **信任 ≠ 过于热情**：遵循“不问不答，不求不助”的原则。不要急于袒露自己或提供帮助，让每一次帮助都显得珍贵。
    3.  **信任 = 共情**：共情不是认同对方所有选择，而是理解“如果我经历了你的人生，面对你的处境，我可能也会这么做”。探讨行为背后的成因，而非直接评判行为本身。
    4.  **信任 = 框架**：信任只建立在你有原则的基础上。学会拒绝是建立框架的关键。

### 学会拒绝的艺术

没有框架和原则，就谈不上真正的信任。以下是两种有效的拒绝方式。

1.  **直接拒绝**：理由可以是“我不喜欢”或“我不想要”。每个人都应学会直接、有理有据地拒绝，不给予对方迂回空间。代码式表达：`if (request == undesirable): return "No, I don‘t want it."`
2.  **条件式拒绝**：将拒绝作为交换的筹码，设立条件。公式为：**肯定对方 + 拒绝原因 + 合理化原因 + 间接提出要求**。

以下是应用上述公式的举例。

*   **举例1（拒绝表白）**：“我觉得你这个人哪哪都好，找个这样的男朋友挺不错的（肯定对方）。但我觉得你有时不够细心，而我也很粗心（拒绝原因）。如果我们在一起，生活可能一团糟（合理化原因）。我觉得我需要一个更互补的人（间接提出要求）。”
*   **举例2（拒绝表白）**：“能被你这么优秀的人喜欢，我很开心（肯定对方）。但我们认识时间太短，彼此了解不够（拒绝原因）。我更需要一个愿意花心思深入了解我的人（合理化原因+间接提出要求）。”

### 维护框架的一致性：三次妥协原则

建立信任的题外话是传递“吃软不吃硬”的原则，并严格遵守框架的一致性。

以下是维护框架一致性的“三次妥协原则”实例。

*   信息发**三条**不回，绝不发第四条。
*   电话打**三次**不接，绝不打第四个。
*   道歉道**三次**不原谅，绝不给第四次机会。
*   延伸：请客吃饭**三次**，绝不请第四次；对方冒犯你**三次**，绝不给第四次机会。
