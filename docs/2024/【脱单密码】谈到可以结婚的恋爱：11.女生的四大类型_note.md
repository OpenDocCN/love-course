## 1. 玩咖女 🎤

玩咖女通常思想开放，热爱玩乐，年龄层主要集中在“96后”、“97后”。她们可能没有正式工作，或仍在求学。其核心特征是追求新鲜感、开放度高，且更注重即时快乐。

**核心公式：吸引力 = 有趣 + 外形达标**

以下是该类型女生的主要特点：
*   年龄较小，多为96后、97后。
*   可能无固定职业或仍在读书。
*   性格爱玩，喜欢新鲜事物。
*   社交开放度较高。

想要吸引这类女生，你自身必须足够有趣，同时外形也需达到她的基本标准。她们通常不会花太多时间深入了解你的内在，只要相处时感到快乐即可。

## 2. 心机女 💼

上一节我们介绍了注重即时快乐的玩咖女，本节中我们来看看更为成熟的类型。心机女年龄层多在89年至95年之间，常见于白领、创业者或交际类职业。她们成熟、精明，带有一定的社交保护意识和现实考量。

**核心公式：匹配度 = 社交圈层 + 生活阅历**

以下是该类型女生的主要特点：
*   年龄层分布在89年至95年。
*   职业多为白领、创业者或需要频繁社交的岗位。
*   性格成熟，有较强的社交保护意识，较为现实。

追求这类女生难度较高，因为她会非常在意你们生活圈子的匹配度、社交层次以及你的生活阅历是否丰富。只有符合她这套标准，感情发展才有可能。

**核心公式：吸引力 = 外形及格 + 有趣 + 能触发其开放点**

*   职业多为前台、收银、教师等。

吸引这类女生，需要你的外形过关，并且要足够有趣。你不仅要会聊天，还要能精准触动她内心开放的那个“点”。做到这些，就很容易吸引到她。

最后，我们来看最为保守的一类。传统女，顾名思义，思想较为保守，受传统观念影响较深。年龄层多在90年至95年，常见于大学生、公务员或银行职员等职业。

**核心公式：长期关系 = （外形+品味）及格 + 职业匹配 + 耐心**

以下是该类型女生的主要特点：

*   职业多为学生、公务员、银行职员等。
*   性格知书达理，但保守、固执，受原生家庭影响大。

与这类女生交往，首先你的外形和品味需要达标。此外，她还会看重你的职业是否与她匹配。**最重要的一点是必须具备耐心**，因为她身上有传统思想的包袱，需要对方是一个善于沟通、凡事能循序渐进的人。
