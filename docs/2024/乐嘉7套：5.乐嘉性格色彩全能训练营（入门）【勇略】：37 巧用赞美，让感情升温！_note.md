# 乐嘉性格色彩课程5：巧用赞美，让感情升温！💬

在本节课中，我们将要学习如何针对不同性格色彩的人进行有效的赞美。理解这一点，对于改善人际关系、促进感情升温至关重要。

## 概述：赞美的性格差异

在这个世界上，不同性格的人对同一个词语的定义、感觉和理解完全不同。即便在赞美这件事上，人和人在天性需求上也存在差异。

## 红色性格：渴望认可，多多益善 🟥

上一节我们介绍了赞美的普遍差异性，本节中我们来看看红色性格的特点。

红色性格的人极度渴求他人的认可。他们对赞美的大小无所谓，可以接受任何形式的表扬。例如，你赞美他的新发型，他可能会开心一整天。红色性格不仅渴望赞美，也会主动寻求赞美。他们不吝啬给予他人赞美，是所有性格中最善于表扬他人的。

以下是红色性格在赞美方面的几个关键点：
*   **需求强烈**：他们强烈期望得到别人的认可与赞美。
*   **大小通吃**：无论是重大成就还是微小细节的赞美，都能让他们开心。
*   **主动索求**：他们会通过提问等方式，间接寻求你的表扬。
*   **乐于给予**：他们自己也最善于并乐于赞美别人。

红色性格活在认可与赞美之中，对此需求度极高。

## 蓝色性格：追求精准，贵在精而非多 🟦

了解了红色性格对赞美的开放态度后，我们转向与之几乎相反的蓝色性格。

在四种性格中，天性中最排斥泛泛赞美的就是蓝色性格。他们追求深度沟通和默契，反感无聊或敷衍的口头恭维。对于蓝色性格，赞美不在于数量，而在于是否精准、真诚且有深度。

以下是蓝色性格对赞美的核心要求：
*   **排斥俗套**：通用的、敷衍的赞美会让他们感到讨厌。
*   **要求精准**：赞美需要量身定制，注重细节和真实性。
*   **追求深度**：赞美之词需要经得起推敲，有内涵。

蓝色性格的完美主义也体现在对赞美的要求上，他们需要的是“绣花在外，白玉在里”的高质量认可。

## 绿色性格：淡然处之，有无皆可 🟩

接下来，我们看看对赞美态度最为平静的绿色性格。

绿色性格对天下事物常保持无所谓的态度。对于别人的赞美，他们既不排斥，也不欢迎，给予与否他们都淡然处之。你赞美她，她报以微笑；你不赞美她，她同样报以微笑。

以下是绿色性格对待赞美的特点：
*   **需求平淡**：他们对赞美没有明显的渴求。
*   **情绪稳定**：很难因赞美而亢奋或激动。
*   **顺其自然**：他们的态度是“有也行，没有也行”。

因此，让红色性格兴奋很容易，但让绿色性格亢奋则非常困难。

## 黄色性格：看重能力，需“猛药”认可 🟨

最后，我们来分析目标导向的黄色性格。

典型的黄色性格更在意别人对其能力和成就的尊重与认可。浅尝辄止的赞美会让他们觉得隔靴搔痒，他们需要的是足够有分量、能体现其价值的“大剂量”赞美。

以下是赞美黄色性格的要点：
*   **注重能力**：相比外表，他们更看重对能力和才干的认可。
*   **需要分量**：赞美必须足够重磅，小打小闹的恭维无效。
*   **厌恶低估**：泛泛地夸其外表，会被视为低估了他们的价值。

与蓝色性格不同，黄色性格不怀疑夸张与否，而在意赞美的分量是否足够大。

## 核心总结与相处法则 🎯

本节课中我们一起学习了四种性格色彩对赞美的不同需求和态度。根据以上内容，可以总结出以下四句核心法则：

1.  **红色性格**：赞美有癖，多多益善，无癖自去寻来。
2.  **蓝色性格**：赞美不在多，贵在精。
3.  **黄色性格**：小赞美不可养家（滚），大赞美方可定国（留）。
4.  **绿色性格**：不因赞美大而喜之，不因赞美小而不喜。

掌握这些法则后，你就知道如何与不同性格的人相处。例如，与红色性格相处时，切勿嫌其索求赞美烦人。经常给予鼓励和表扬，他们会做得更好来回报你的认可；反之，则可能打击其积极性。

深刻理解每种性格对赞美的需求，是有效沟通、增进感情的关键。
