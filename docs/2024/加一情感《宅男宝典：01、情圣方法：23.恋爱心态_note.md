# 加一情感《宅男宝典》课程01：情圣方法第23讲·恋爱心态 🧠

在本节课中，我们将要学习恋爱心态的核心原理与构建方法。心态是恋爱能力的内在基石，它深刻影响着我们的外在行为和互动结果。我们将通过真实故事和具体方法，理解如何培养平等、平和、积极的心态，从而在情感关系中掌握主动权。

## 概述：心态的力量

上一节我们介绍了恋爱方法的外部框架，本节中我们来看看内在心态的决定性作用。

心态是行为的根源。一个强大的内心能够自然引导出自信、从容的外在表现。我们将通过两个故事来理解积极心态如何创造奇迹。

第一个故事关于玛瓦·柯林斯。她是一位出生于20世纪30年代的小学教师。她的父亲是黑人，母亲是印第安人，在当时美国严重的种族歧视环境下，她的父亲始终鼓励她，告诉她“你一定会成功”。尽管父亲认为秘书是她不可能达到的职业高度，但持续的鼓励最终让她成为了秘书。然而，玛瓦认为自己更适合做老师，通过就读夜校，几年后她真的成为了一名教师。

她在芝加哥一所公立小学任教。该校区青少年犯罪严重，许多学生吸毒、加入帮派、厌恶学习、逃学。但从执教第一天起，玛瓦就坚持鼓励这些学生，告诉他们“你可以做到”。她对学生们满怀期待，将目标放在他们的长处上并加以培养。接着，奇迹发生了，这些学生真的改变了。许多十岁的学生已经能够掌握高中数学。后来，因遭人嫉妒等原因，她离开公立学校，在自己的厨房里开办了家庭学校。

起初只有四名学生，其中两个是她自己的孩子。但逐渐地，更多被公立学校退学的“问题学生”来到她家。条件虽然艰苦，学生们的学习热情却很高。她年复一年地鼓励这些学生，给他们信心，教导他们承担自己的责任、停止抱怨。她传递的核心信念是：成功与否，全在于你自己。接着，奇迹再次发生，她学校里所有的学生都考上了大学。

她艰难地维持着学校，因为许多学生家庭困难，交不起学费。但在1979年，CBS制作了一档长达15分钟的节目报道了她，使她一夜成名。次年，里根总统邀请她担任美国教育部部长，她拒绝了。八年后，老布什总统再次发出邀请，她再次拒绝。如今，她的学校已遍布全美国，她的学生中涌现出许多著名的政治家、律师、医生和教师。

是什么让街头混混转变为社会有用之才？是什么让奇迹发生？是鼓励，是信心，是积极面对自己的心态。

第二个故事是电影《阿甘正传》中的阿甘。他是一个智力低下的人，正是因为他母亲永不抛弃、永不放弃的持续鼓励，最终阿甘也获得了成功。

在中国哲学中也有类似智慧。中国有四位圣人，大哲学家王阳明是其中之一。以往的圣人教导人们行善积德，遵守“君君臣臣父父子子”的规范，从行为上约束自己。而王阳明倡导的是“修心”，讲究“知行合一”。前者是从行为上改变自己，后者是从心态上改变自己。

这和我们学习恋爱技巧（LS）是同一个道理。例如搭讪时，把两句话说出口很简单，但要做到不紧张、没有小动作、从容大方就不容易。我们最初教学时，会告诉你保持一米左右距离，拍别人肩膀时不能太重以免吓到对方，说话要不紧不慢。这些都是从行为上约束你。实际上，当你内心真正不紧张时，你的肢体动作自然就得体了。这就是“修心”，这就是“知行合一”。你的心态决定了你的动作和行为。

## 恋爱能力的两大建设

恋爱能力的提升分为两个方面：
1.  **外部建设**：包括建立吸引、学习恋爱方法，从方法行为上去改变。
2.  **内部建设**：包括心态、自信心，从内心去改变。

随着学习的深入，你会发现内部建设比外部建设重要得多。方法可以很快学会，但内心却很难改变。人与人交流时，70%传递的是你的状态和感觉，只有30%是你说话的内容。这种改变是由内而外的，当你的心态变了，你的行为也会随之改变。

举个例子就明白了：我们常开玩笑说“口袋里有钱，说话就有底气了”，这个“底气”就是心态。

## 如何构建积极的恋爱心态

那么我们如何来改变和构建这种心态呢？我总结了一些方法。通过这些方法，再加上恋爱实践，我们会逐渐拥有这种“底气”。首先，你要从内心告诉自己“你可以”，不断鼓励和激励自己。这也是第一个故事给我们的核心启示。

以下是学习恋爱心态需要掌握的四点核心内容：

### 1. 平等 🤝

无论对方多么漂亮，无论你多么喜欢她，最基本的原则是做到平等。如果没有这个基础，双方就不可能产生正常的交往。

举个例子：一个女生长得很漂亮，你很喜欢她。有一天她对你说：“小王，帮我拿下前台的快递。”你一脸兴奋地回答：“好嘞！”，然后飞奔过去，把快递交到她面前，还说了许多讨好的话，比如：“你以后有快递就跟我说，你长得这么好看，不用动，说一声就行。”

当你做完这些之后，你就再也没有机会了。如果你离得近，顺手帮忙拿一下是可以的。但这本是她自己的事，你帮忙拿了就算了，还极力讨好她。在她眼里，这种行为是低价值的。她会认为你一定没见过什么美女才会这样。明明是她让你做事，你却反过来献殷勤。你的行为反映了你内心的不平等：你认为她比你优越，她让你做任何事你都会开心，所以才有这样的举动。一个高价值的男人遇到这种情况，绝不会这么做。

很多时候我们会问：“为什么我对他那么好，他却不喜欢我？”因为失去了平等的基础。你那种“好”不能称之为好，那是从下往上的仰望。心理上的不平等，无法产生有效交流，连成为朋友都难，更别说成为恋人了。

**正确的做法是**：当你遇到喜欢的女生时，秉持的态度是——如果她对我好，我就对她好；如果她对我不好，我就对她不好。

### 2. 平和 😌

平和指的是平和的心态。不要因为女生对你的反馈而产生剧烈的心情波动。

举个例子：当一个女生回复得很好时，我们就非常开心，觉得天蓝水绿，一整天心情开朗。当一个女生反馈不好时，你就情绪失落，觉得问题严重，进而开始质疑自己，觉得自己没用，觉得整个世界都是灰色的。

你要做的就是就事论事，不要上纲上线，不要让这件事过度影响自己的情绪。我们要保持内心平静。女生反馈好时，不要太“飘”；反馈不好时，也不要太失落，不要过多联想。它只是一条回复得比较好的消息，不要想着“她是不是对我有意思？我是不是有机会？”。反馈不好时也一样处理。

首先要学会一点：**感知自己的情绪**。当你感觉有点“飘”的时候，要意识到“我现在有点飘了”。当情绪失落、开始胡思乱想时，要提醒自己“不要乱想”。当你意识到的时候，其实已经在进行自我调整了。

### 3. 避开情绪低谷期 🚫

前面我们说了要保持心态平和，控制情绪。但我们也会遇到自己无法平和的时候，即心态爆炸。这时我们往往容易犯低级错误。

我们可能有过这样的经历：某天心情不好，喝了很多酒，然后打电话给一个女生，向她倾诉，让她出来见面，不停地打电话骚扰她……结果彻底搞砸了关系。我们应该避开自己情绪失控的时候，先把自己调整过来，等情绪平复后再去和别人沟通。等你情绪好了，你会发现你根本不会那么做，但事情已经发生，后悔也来不及了。

在情绪失控时，最好什么都别做，先调整自己的情绪。每个人都有自己调节情绪的方式，比如有些人睡一觉就好，有些人需要和朋友聚聚，有些人通过运动出汗来释放。你要找到适合自己的方式，快速恢复状态。

### 4. 培养积极心态 🌞

从第一个故事我们知道了正面力量的强大，其实负面力量同样强大。平时应少接触消极的东西，多接触正能量。不要总是让自己沉浸在负面情绪里。很多人喜欢听忧伤的音乐，看悲剧故事，这会慢慢把你的心态变得悲观，进而让你的人生也变得悲观。

这样的例子有很多。例如中国一些搞摇滚的艺人，无论曾经多么红，现在没有一个过得好的，因为他们每天接触和表达的都是撕裂、阴暗的内容。再比如诗人海子，他很有才华，考上了大学（当时的天之骄子），而且还是北京大学。结果他写悲伤的诗，最终卧轨自杀了。我印象最深的是一个台湾女作家，小时候遭遇不幸，她写另一个女生被侵犯的故事，每天边流泪边写，书完成后她也自杀了。如果她每天看看轻松的节目，工作后和朋友吃吃饭、逛逛街，很可能不会有如此结局。

在我们接触的学员中，有时也会遇到类似情况。同一件事，他看到的永远是负面。无论女生给他的反馈是好是坏。比如我跟他说：“这个女生明天会主动找你。”他会说：“不会的，她从来不会主动找我。”第二天女生真的找他了，我以为他会开心，结果并没有。他说：“她只是来问我毕业证的事，是不是拿了毕业证就不跟我联系了？”

**身体就像一个留声机，你接收什么，就会散发出什么。** 多接触正面的东西，你会变得越来越阳光。当你乐观时，你最终得到的结果往往也是乐观的。

## 总结 📝

本节课中我们一起学习了构建强大恋爱心态的四个核心要点：

1.  **平等**：无论对方条件如何，首先要做到心理平等。别人对你好，你便对他好；别人对你不好，你也不必讨好。
2.  **平和**：不要因他人的反馈而过度情绪波动，保持内心平静，就事论事。
3.  **避开情绪低谷期**：意识到自己情绪失控时，暂停互动，先通过适合自己的方式调整好状态。
4.  **培养积极心态**：主动接近正能量，鼓励自己，远离长期沉浸的负面信息与环境，让乐观的心态引导你获得积极的结果。

记住，心态决定状态，状态影响结果。由内而外的改变才是持久和根本的。
