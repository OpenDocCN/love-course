# 加一情感《宅男宝典：01、情圣方法：25.如今的相亲

🎼大家好，这里是琴声方法，我是导师开问。本节课我们将探讨相亲。

相亲在今天依然是我们主流的处对象方式。相亲节目红火，相亲网站上市，都说明了这一点。

**相亲的变化**

现在的相亲与过去有很大区别。过去相亲看好了就结婚，现在相亲更多是见面，目的性更强，但最终还是要追求恋爱关系。

**相亲中的关键点**

1. **形象**：第一次见面形象占70%，男性需保持干净清爽，穿着得体，避免过于时尚或老气。注意皮肤保养、发型打理，内增高鞋垫、胡子、指甲、鼻毛等细节。

2. **心态**：相亲时许多人紧张，说不出话。紧张心态可通过多次相亲改善。与陌生人聊天时，心态要平和，将其视为普通朋友。

3. **场所**：选择环境优雅、装修精致、附近有散步地方的场所。简单聊天后，可以一起散步。

4. **话题**：聊对方感兴趣的话题，如星座、美食、宠物、综艺节目等。了解对方喜欢的明星、电视剧、常用app等。

5. **真诚与踏实**：展现真诚、踏实的一面，让人感觉值得信任。不需要刻意吸引别人，让别人了解真实的你。

6. **筛选**：了解对方，筛选高价值的人。不要只看到美女就什么都不问。

7. **后续行动**：根据对方反馈判断兴趣，主动沟通。如果对方反馈良好，继续联系；如果一般，可适当保持联系；如果完全没兴趣，则不再联系。

**总结**

相亲中，女生更看重感觉，形象很重要，不要急于升高关系，真诚礼貌、稳重为基础。结束后，根据对方反馈判断兴趣，继续保持联系。相亲相当于完成了认识到熟悉阶段，后续可进入L3阶段。
