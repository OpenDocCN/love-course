# 25、李想-性格识人课：018-17 8号控制型 性格分析

## 概述

在本节课中，我们将学习九行人格中的8号控制型性格特点及其行为模式。

## 8号控制型性格特点

### 决策依据

8号控制型以直觉为依据进行决策，重视外部环境，不依赖理性分析或感觉。

### 核心诉求

8号控制型追求自主，在做决策时需要自己做主。

### 情绪属性

8号控制型、9号和平型以及1号完美型共有的情绪属性是愤怒。

### 特质

- **自由主义者**：敢于冒险，是掌舵人、创业者，任性好战，不喜欢受权威约束。
- **好战**：从小到大都是孩子头，喜欢打架，具有强烈的报复心。
- **领袖**：在顺境时是英雄人物，勇敢、宽大、自信，是天生的领袖。
- **权力敏感**：对权力非常敏感，关注谁是老大，想要取而代之。

### 处理感情的方式

8号控制型害怕被控制或驾驭，与人亲密时保持距离，外壳防卫性强。

### 外形特征

- **精力充沛**：双目炯炯有神，走路虎虎生风。
- **身材健壮**：男女身材都比较健壮，女性可能发胖。
- **声音洪亮**：发音属于丹田音，说话声音洪亮。
- **强势态度**：手指式点覆别人，教导式动作大开大合，说话肯定自信。

### 穿着打扮

- **随性**：男女穿着都比较随性，女性可能喜欢穿绚丽色彩的衣服。
- **配饰**：女性喜欢戴大块的配饰。
- **车辆**：喜欢开大车，如悍马、奔驰等。

### 竞争性

8号控制型喜欢竞争，越有竞争越兴奋，血性硬强。

### 工作环境

喜欢自己领导，自己控制，喜欢组织、挑战、竞赛。

### 领导风格

对下属要求听话，对领导要求充分自主权。

### 员工表现

责任感强，承担家庭和工作的责任。

### 决策速度

做决策的速度快，直觉性强，该出手时就出手。

### 优点

- 胆子大，容易发大财，抓到商机。
- 学习领悟力强，能快速调整。

### 缺点

- 胆子太大，容易冲动，踩破底线。
- 缺乏风险防控意识。

## 总结

本节课中，我们学习了8号控制型性格的特点和行为模式。了解这些特点有助于我们更好地理解自己和他人的行为，从而更好地与他人相处。
