# 796、【完结】陈海贤·自我转：chx41 关系视角：如何把标准从重要他人切换到自我

在本节课中，我们将学习如何将自我评价的坐标，从依赖“重要他人”的看法，切换到建立以自我为核心的视角。这是构建新自我、实现转变的关键一步。

上一节我们探讨了如何从大众评价坐标切换到自我评价坐标。本节中，我们来看看另一种更常见的坐标切换：从“重要他人”的评价，切换到自我的评价。

转变是重新寻找自我的过程。旧的自我坍塌后，新的自我在逐渐形成。在新自我形成期，不同的声音会不断拉扯你。你的头脑是一个观点的战场，这个战场比拼的是影响力。在这个战场，你自己的想法并无特别优势，除非你意识到这是你的“主场”。你有权决定谁的影响有多大，你有权决定让谁影响你。

如果说别人在很多事的看法上比你更懂，所以你接受他们的影响，但至少在一件事上，你不能轻易接受，你要更加慎重。那就是关于“你是谁”的看法。**人最大的权利就是定义自己的权利**。如果你失去了这种权利，你就会失去自己的感觉、思考和想法，转而按照别人的感受、思考和想法来行事。所以不要轻易让出这种权利，无论那个人对你有多重要。

寻找新的评价坐标，其实是找回这种权利、学习用自己的眼光来定义自己的过程。如果你没有意识到你有定义自己的权利，别人的评价就会变成你的“自我封印”，限制你的可能性。

还记得前面的课程例子吗？一位女士失恋后不再恋爱。她原本自信，但因相处不愉快与男友分手。男友临走时说：“你脾气这么差，又没什么魅力，连我都受不了，真正了解你的人，不会愿意跟你在一起。” 😊 这句话深深影响了她。分开几年后，她身边虽有追随者，但一想到男友的评价，她就感到恐惧，觉得不该再走进亲密关系。

我询问她周围人的看法，她说：“他们觉得我还挺好的，可是也有可能是他们并不了解我。” 我问：“那你为什么相信前男友的话，而不相信他们的话呢？如果是这样，你是把前男友看得比他们更重要。” 她回答：“我也不想把他变得这么重要，可是我总觉得他说的是对的。”

从这个故事里，你可以清晰看到“观念的战争”如何影响一个人。当然，背后可能有其他动力，例如她没有完全放下这段感情，因而不自觉地把男友的话看得很重。但无论如何，这深刻影响了她的自我评价，进而影响了她的生活。

你可能会问：难道没有客观情况吗？会不会她男友说的有道理？她与男友相处的细节，就是暴露了她是这样的人，所以她才相信了前男友的话呢？

关于“我是一个什么样的人”，有两种典型的思考方式：
1.  **探究现实的视角**：这种思考方式假设关于“我是谁”，存在一个绝对客观的真理。我们所做的就是从各种证据中拼凑出这个真理。所以当前男友说这位女士“脾气这么差”时，我们就寻找很多证据来证明他说得对还是不对。
2.  **关系的视角**：在关系的视角里，关于“我是谁”并没有唯一的答案。你可以说“我有很多发脾气的时候”，但也可以说“我有很多没发脾气的时候”。“我发了脾气”，既可以说“我脾气不好”，也可以说“你是一个很能惹我发脾气的人”。任何一个想法都可以找到相应的证据。😊

既然这些都不确定，那什么是确定的呢？那就是：**“谁是这么想的？”** 以及 **“他为什么这么想？”** 而 **“我要不要接受谁的想法？”** 当你这么想时，你就开始把“我是谁”的问题放到了关系的视角。这时你会思考：他试图影响我，让我接受“我不是一个好人”，而我也可以自己决定我要不要接受这种影响。😊 当你这么想时，你就拿回了定义自己的权利。你不再以他人的评价坐标来看自己，你开始形成自己的评价坐标。因为这种权利，至少在观念的战场上，你给了自己更重要的位置，这是新自我的基础。

那么，我们应该怎么拿回这种权利呢？当你发现头脑中有很多否定自己的声音时，你需要从“探究事实的视角”切换到“关系影响的视角”。

以下是你可以试着问自己的几个问题：

**第一个问题是：不是问“这个看法对不对？”，而是问“这是谁的看法？”**
这是一种反常识的思考。如果探究现实的视角是“对事不对人”，那这个视角就是“对人不对事”。实际上，定义“我是一个什么样的人”这件事本身就是一种权利，而权利只有放到关系中才能被理解。当你问“这是谁的看法？”时，你其实把这个观点从一个“确定无疑的事实”的位置拉了下来。既然它只是一种看法，来自某个特定的人，那你也可以根据跟这个人的关系来思考怎么处理这个观点。

**第二个问题是：他通过这个看法在向我表达什么？**
头脑中的看法只是看法，而被表达的看法常常有特定的意图。当然，不表达的看法也有特定意图。当他说你“不好”的时候，他是善意的提醒、委屈的抱怨、不满的愤怒，还是隐秘的控制？当你这么想时，你就把一个特定的观点变成了“关系中的事件”。你也因此可以思考：我要怎么处理这个关系中的问题？就像前面的例子，分手的男生对女生这么说，明显有他的愤怒在。问题并不在于“她好不好”，而是在于“这个男生对她生气”。😊 当他告诉你“你是什么样的人”时，有时是在彰显一种权利：“我比你更客观，比你更有影响力，所以我比你更有资格定义你是什么样的人。” 如果你接受了这种影响，那他的观点就会变成你的观点；如果你不接受，那他的观点只代表了他影响你的努力，这种努力因你的拒绝而变成了失败的努力。

**第三个问题是：你是否接受他的影响？与之对应的问题是：你想把他放到一个什么样的位置？**
这个问题其实是一种暗示，它在提醒你自己：你有权利决定是否接受他的影响。如果你觉得那个人已经不那么重要了，那么你最好也能把他对你的影响放到不那么重要的位置。不要被他所谓的权利、地位、知识或者影响力吓到，定义你自己本来就是你的权利，在你的头脑中，这更是你的“主场”。不过，这并不意味着你一定不接受他的影响，它只是提醒你：你可以有自己的选择。当你意识到自己有选择时，你就把自己放到了一个“决定者”的位置，这本身就是创建新评价坐标的一部分。

受不受观念的影响，本质上是一种关系，而关系背后也有我们情感的纠结。很多时候，这种纠结是我们既希望同这个人保持情感的联系（所以我们自然看重他对我们的影响），又不想接受他对我们的否定。那怎么办呢？我的建议是：**你可以只是从情感层面去理解你对他的需要，而不用轻易让渡关于定义你自己的权利**。当你在观点层面这么做时，其实你也在重构你们的关系，把你变成跟他更平等的人。

由此带来的**第四个问题是：什么是你想要的看法？**
只是接受或者拒绝别人的看法是不够的。如果你要形成自己的评价坐标，你还需要寻找你自己的答案，关于“你是谁”的答案。关于这个问题，很多人的答案是“我不知道”。有时候也许我们被影响得太深了，所以不知道到哪里去寻找关于“我是谁”的答案。可是，“不知道”也比不假思索地接受别人的观点要好。因为这是你需要给出的答案，而给出答案的过程，也是你获得新评价坐标的过程。

听到这儿，也许你已经明白了。其实切换评价坐标的过程，就是我们重构和他人关系的过程，只不过这个过程发生在关于“我是谁”的观点的战场上。

你可能会问：那你的意思是不是我们都不要接受别人的影响呢？当然不是。事实上，知道“你是谁”也需要很多别人客观的反馈。我们所讲的，是从一种不假思索的、从属的地位解脱出来。在关于“你是谁”这个问题上，没有人比你更清楚，**你才是这个问题最大的权威和专家**。这并不意味着你要不受别人的影响，而意味着你可以选择：我要接受谁的影响？我在哪些方面接受他的影响？哪些方面不接受？以及我在多大程度上接受他的影响？

在前面的例子中，那位受前男友影响的女生最终在日记里写了一段话：“我曾经把你当做很重要的人，所以我才愿意接受你的影响。哪怕你仍然很重要，可是我要收回这种影响，我要去新的关系重新寻找我自己。”

在本节课中，我们一起学习了如何运用“关系的视角”，将自我评价的标准从“重要他人”切换到“自我”。关键在于认识到定义自我的权利在你手中，并通过四个关键问题（这是谁的看法？他表达什么？我是否接受？我想要什么看法？）来审视和重构头脑中的观念战场，从而为建立稳固的新自我评价坐标打下基础。

在你的生活中，重要他人对你的评价曾经给你带来过什么样的影响？你对此的感受是什么？欢迎思考。
