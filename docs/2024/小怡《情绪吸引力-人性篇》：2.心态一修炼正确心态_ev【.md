在本节课中，我们将要学习聊天训练中最核心、最基础的部分——心态。许多初学者认为聊天就是学习方法和技巧，但事实上，正确的心态是决定你能否有效运用任何技巧的根本。没有强大的心态，再好的技巧也无从施展。本节课我们将深入探讨什么是“温柔且坚定”的聊天心态，以及如何修炼它。

## 概述：心态为何如此重要？

上一节我们提到了心态的基础性地位。本节中我们来看看，为什么在聊天训练中，心态比技巧更重要。

如果把聊天的方法和技巧比喻成武器，那么心态就是你这个人本身的肌肉力量。**公式：战斗力 = 心态（肌肉力量） × 技巧（武器）**。你想举起多重的武器，取决于你自身的肌肉力量有多强。

例如，有一把青龙偃月刀能增强战斗力，但如果你骨瘦如柴，不仅拿不起来，强行使用还可能伤到自己。聊天训练同理，首先要做的是摆正并修炼强大的心态。拥有好心态后，即使技巧稍逊，聊天结果也不会太差。就像关羽的强大在于他能轻易挥舞青龙偃月刀的力量，而非刀本身。

## 什么是好的聊天心态？

好的聊天心态可以概括为：**温柔且坚定**。接下来，我们将分别解析这两个核心要素。

### 1. 温柔的力量：包容与理解

温柔，并非软弱，而是一种强大的内在力量。它意味着不挑剔，能看到世界的美好，是一种让人感到仁慈和宽厚的性格力量。

这种特质常出现在内心富足的人身上。因为只有自己内心充盈、不贫瘠，才有余力去爱别人，让别人如沐春风。反之，刻薄的人往往内心匮乏，需要通过苛责他人来填补自己。

在社交中，当你内心充盈时，自然会表现出温柔。对方感知到这种温柔，会被你的人格魅力吸引。因为温柔背后，蕴含着高价值人才拥有的特质：强大的理解力、情绪自控力和包容力。

社会关系中，通常是强者包容弱者。男女思维差异巨大，如果你能从心底包容异性那些难以理解的想法和行为，这种强大的包容力所产生的温柔，是一种极强的吸引力，远胜外在的物质条件。

许多学员在学习技巧后，聊天却陷入抬杠、对抗甚至争吵，就是因为缺乏对温柔心态的认知，加上目标不清，最终成了对方心中尖酸刻薄的人。

因此，好心态的第一个关键是**温柔**。这种力量需要长期训练。

以下是训练温柔心态的具体方法：

*   **多角度理解他人**：遇到令你不悦的事情时，尝试站在不同角度思考对方的动机和处境。
*   **对生活微笑**：将“先笑一笑”变成下意识反应，这能立刻帮助你冷静并打开思路。
*   **感受世界善意**：主动关注和感受生活中的美好与善意。

例如，孩子打碎了杯子。未经训练的人可能只会看到“劳动成果被破坏”的角度，从而发怒。训练后，你可以先微笑，然后思考其他角度：孩子是不是想给你倒饮料？这是不是他探索世界的过程？这样，你就能看到孩子的可爱，而非地上的狼藉。

将这种训练带入生活，对同事、朋友都尝试先理解。习惯后，你会发现自己生气次数变少，对他人更理解，自己也变得更温柔、更强大。

**为什么温柔代表强大？** 因为温柔的人有选择。过去，被冒犯后发怒是下意识、无法控制的反应，我们没得选。愤怒常是无能的体现。训练温柔后，你能看到更多角度，从而有了选择权：可以选择愤怒，也可以选择原谅。有选择的人永远比没选择的人更强大，这种强大带来的吸引力是致命的。

### 2. 坚定的原则：底线与框架

如果说温柔需要时间练习，那么坚定则是你可以立刻掌握的心态。坚定是温柔的补充。一个人如果只温柔不坚定，可能会变成老好人，被欺负、占便宜，在恋爱中甚至可能成为备胎。

因此，除了温柔，我们还需**坚定**。坚定的是我们在社交关系中不可打破的原则。

#### 原则一：坚定自我价值感

坚定自我价值感，即相信自身的价值足以匹配对方，无需依赖外在附加条件。

许多人在恋爱中有一个错误想法：需要靠外在事物（如礼物、活动）来吸引对方。他们认为仅凭自身价值不够，需要加上一个“X”才能等于对方。**错误公式：我 + X = 女生**。

一段健康感情的正确开始应该是：**我 = 女生**。所有的“X”都只是锦上添花。如果你认为“我 + X = 女生”，那么女生喜欢的可能不是你，而是那个“X”。一旦失去“X”，关系就可能瓦解。

你要做的是提升自身价值，坚定“我 = 女生”的信念，通过学习和训练提高自己的“分数”。

#### 原则二：坚持体面的社交

体面的社交具体表现为三点：不纠结、不解释、不跪舔。许多人在感情中犯错，都是因为违背了这些原则。

以下是体面社交的三个具体表现：

*   **不纠结**：纠结是焦虑心态的外化。你只会对生命中极其重要的事焦虑。聊天时如果怕犯错、过度斟酌字句，就会显得磨蹭、缺乏魅力。解决方法是不怕犯错。实际上，因一两次聊天犯错就搞砸关系的情况极少，但因怕犯错导致心态糟糕，最终搞砸关系的却很多。所以，请大胆一点，不要纠结。
*   **不解释**：喜欢解释自己的行为和想法，生怕被误会，这恰恰暴露了“小男人”心态。弱者才需要不断解释以获得强者认可。强者从不解释。不断解释的行为，等于向女生证明你觉得她比你强，需要她的认可。女生不会喜欢一个比自己弱小的男人。要成为有魅力的人，先做到不解释，即使被误会也能承受委屈。人的心胸正是被委屈撑大的，心态会因此越来越强。
*   **不跪舔**：这是最严重的一点。跪舔相当于将对方捧上神坛，本质也是自我价值感不坚定（认为“我 + 跪舔 = 女生”）。跪舔行为会让对方明显感知到：你认为自己配不上她，只能通过廉价、无条件的付出来弥补。这是自我价值贬低。试想，如果你卖一个古董花瓶，却不断附加赠送、打折、请吃饭，买家反而会怀疑花瓶的价值。社交中也是如此，跪舔只会让你显得廉价，大幅降低成功的概率。

## 总结：融会贯通，知行合一

本节课中，我们一起学习了聊天训练的基石——心态。

好的聊天心态，核心是**温柔且坚定**。

*   **温柔** 是内心的力量，要求我们感受世界善意，对他人温柔以待，做到不刻薄、不挑剔，尽量包容和理解。
*   **坚定** 是我们的核心原则，包括：1) 坚定对自我价值感的认可，拒绝“我 + X = 女生”的心态；2) 坚持体面的社交，做到不纠结、不解释、不跪舔。

在今后的日子里，请时刻用这个心态检视自己的聊天行为，并将其落实到具体互动中。当你开始这样做时，你的聊天结果将会逐渐改善。
