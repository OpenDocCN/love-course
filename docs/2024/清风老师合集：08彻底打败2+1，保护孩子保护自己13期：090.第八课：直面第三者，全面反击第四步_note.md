# 清风老师合集：08彻底打败2+1，保护孩子保护自己13期：090.第八课：直面第三者，全面反击第四步

## 概述
在本节课中，我们将学习如何直面第三者并开始全面反击。我们将分析第三者的心理特点，如何正确地面对第三者，以及在这个过程中可能会遇到的困难和问题。

## 第三者的心理特点
以下是第三者的三个心理特点：
1. **不敢主动进攻**：大多数第三者知道伴侣有妻子或伴侣后，会选择退缩，因为他们缺乏社会地位的认可，不敢轻易挑战你的位置。
2. **过度重视男人的态度**：在第三者关系中，男人的态度就是一切，因此要抓住资源，利用家庭活动让第三者自乱阵脚。
3. **与男人没有太多牵绊**：第三者为了维护关系，不敢轻易向男人索要礼物或付出，因此展示自己的物质条件可以激发对方的嫉妒心理。

## 错误观念破除
在直面第三者之前，我们需要破除以下三个错误观念：
1. **冷处理第三者会离开**：第三者的出现是一个需要面对的冲突，逃避无法解决问题。
2. **必须通过打架解决问题**：打架违法且会降低你的价值，应使用心理战赢得对方。
3. **第三者没有价值**：不要带着敌视情绪看待第三者，要反思自己，了解对方的优点，用其优点来回击。

## 与第三者见面的准备
在与第三者见面之前，需要做以下四件事情：
1. **找亲友陪同**：在暗处陪同，关键时刻可以出来镇住对方，给你带来底气。
2. **隐藏实力**：透露的信息越少，后续打击越有利，给对方误导信息。
3. **约在熟悉地点**：展示你对对方情况的把握，进行威慑。
4. **全程录音**：注意不要剪辑，避免录音无效。

## 与第三者谈话的结构
### 玩具型第三者
1. **认可对方的价值**：表达对对方年轻、漂亮、有梦想的认可。
2. **了解对方的需求**：询问对方的需求，并表示可以商量。
3. **提出条件**：提出解决条件，如金钱。
4. **提出威胁**：强调自己的筹码和威胁。

### 上位型或玩具型第三者提感情
1. **认可对方的价值**：表达对对方青春和爱情的认可。
2. **提出男人的共性**：指出男人都会说这些话，但未来不可预测。
3. **提出自己的筹码**：强调自己掌握经济大权，知道伴侣的罪行。
4. **强调坚定的态度**：表达自己不会轻易出卖伴侣，但会为了面子断绝关系。

## 总结
在本节课中，我们学习了如何直面第三者并开始全面反击。通过了解第三者的心理特点，破除错误观念，做好前期准备，我们可以有效地应对第三者，保护自己和家庭。
