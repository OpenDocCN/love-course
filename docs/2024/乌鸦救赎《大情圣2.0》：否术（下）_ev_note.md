# 乌鸦救赎《大情圣2.0》：否术（下）

在本节课中，我们将继续探讨否术的深层次内容，包括其核心概念、具体操作以及与辩论的区别。

## 概述

否术，又称否定之否定对话艺术，是一种精神层面的修行方式，旨在通过对话实现自我提升和与他人和谐相处。

## 核心概念

**公式**： 否术 = 否定之否定

**代码**： 否术 = 对话 + 思维导图

## 第一部分：完全对立

**以下是关于完全对立的要点**：

- 完全对立是指在认知和态度上与对方完全敌对，认为对方要伤害自己。
- 完全对立的心态会导致防御心理，从而产生敌对、怀疑和愤怒的态度。
- 完全对立会导致人与人之间的对立，难以实现和谐相处。

## 第二部分：统一与征服

**以下是关于统一与征服的要点**：

- 当一个人陷入完全对立的心态时，会倾向于统一思想，即认为世界是撕裂的，但自己要统一它。
- 统一思想会导致征服框架的产生，通过压倒性的胜利迫使对方接受自己的观点。
- 征服框架会引发人与人之间的对立，难以实现和谐相处。

## 第三部分：否述与辩论

**以下是关于否述与辩论的要点**：

- 否述是一种求同存异、形成新共识的探讨方式，而辩论是一种对抗性的对话，以探索真理为目标。
- 否述在过程中会提出不同的观点，但目的是为了实现超越，而辩论则追求压倒性的胜利。
- 否述使用者不害怕提出对立的观点，因为他们内心已经实现了绝对和融。

## 第四部分：否述的修辞方式

**以下是关于否述的修辞方式的要点**：

- 百鹤：通过连接词、转折词连接主语和宾语，引出分歧或出人意料的陈述效果。
- 推拉：肯定又否定或否定又肯定的状态去对话。

## 总结

本节课中，我们学习了否术的核心概念、具体操作以及与辩论的区别。否术是一种精神层面的修行方式，通过对话实现自我提升和与他人和谐相处。希望同学们能够将所学知识应用到实际生活中，不断提升自己。
