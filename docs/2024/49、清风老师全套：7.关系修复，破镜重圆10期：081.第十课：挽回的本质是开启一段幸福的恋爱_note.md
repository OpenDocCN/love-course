# 清风老师全套：7.关系修复，破镜重圆10期：081.第十课：挽回的本质是开启一段幸福的恋爱

大家好，我是秦梦老师。我们致力于在10年内帮助100万人获得情感幸福。上一节课我们讨论了挽回后的维护。这节课，我想与你进行一次深入的对话。也许在听完我的一系列课程后，你对于挽回感到犹豫不决，不确定自己的挽回是否正确，是否有意义。甚至，你可能会想，如果真的调整好心态，这个人似乎并不值得。

如果你也有这样的疑问，请先冷静下来，思考一下你为何开始挽回。错误的观点认为，爱是一种执念，只要我还爱着这个人，我就一定要和他重新在一起。那么，为什么会有这种执念呢？如果你被这个问题困扰，请认真思考。

产生这种对爱的执念，通常有三个原因。第一个原因是童年的伤痛。比如，在童年经历中，感受到的父母的亲密关系不和谐，婚姻关系不健康。因此，在成长过程中，你非常害怕在情感关系中像自己的父母一样，成为别人眼中不被看好、不被认可的关系。所以，当面临分手问题时，你急于想要证明自己是正确的，没有错，没有像父母一样。但殊不知，你越是想要证明，就越变得和他们一样。在感情中，毫无安全感，不断控制和绑架你的爱人。

第二个原因是自我的缺失。因为你特别渴望从对方身上获得某种满足，可能是他有着英俊的外表，而你对自己的长相不够有信心；也可能是他有着高额的财富，而你对自己的收入不太满意。当你不断试图通过对方来满足自身的缺失时，你会得到一个不太好的结果，那就是原本互相成长、互相认可的爱，变成了紧紧抓住对方、让对方窒息的绑架。

第三个产生爱的执念的原因是对未来的不自信。觉得自己的人生已经毁了，没有任何机会了，和这个人分开后，自己的人生没有任何其他的可能性了。

如果你有这三个原因产生了爱的执念，我恳请你停下来，不要着急。挽回是一种选择，我们要做的是找到一个对的人和一份好的爱，而不是简单地和对方复合。如果对方是一个长期酗酒、对生活和未来没有任何规划能力的人，你即使和他挽回了，在两个人的亲密关系中，你还会时常处于颠沛流离、担惊受怕的状态。

我的一位学员，她和丈夫因为对方出轨而分开。她问我：“老师，我该不该去挽回他？挽回他是不是显得有点贱？”我问她：“你真的是因为自己想要挽回他吗？还是因为孩子、家庭、各种各样的压力想要挽回他？”她回答我说：“老师，我不得不承认我已经不爱他了，我也对他没有爱了。但是为了孩子、为了家庭，甚至为了别人眼中对我的看法，我要去挽回他。”这个时候，我告诉她：“姑娘，你很优秀，你很值得被爱，不要在一个不爱你的男人身上不停地去花功夫，让他证明他是爱你的。这样既让他痛苦，也让你自己痛苦。”在你挽回之前，请你问问自己，这个人真的是一个好的对象吗？如果是，那请你大胆地往前走，我教你的方法都可以帮助你掌握他的心。如果不是，那请你冷静一下，不要因为爱的执念，让你的生活陷入水深火热之中。

我再问你一个问题：如果在挽回的过程中，出现了更加优秀的异性，你也很喜欢对方，但是似乎还不能完全放下前任，这个时候该如何选择？不用着急回答我，我想这个时候你心中也一定非常纠结。一方面放不下过去，另一方面也不能很好地迎接新欢。这就是我这堂课的目的。我恳请你给你的未来一些宽容，给你的未来一些新的可能性。特别是如果你社交圈子很小，又不爱社交，当有新的机会出现的时候，你可以先进行一些尝试。不用急着和对方牵手拥抱，至少可以先给对方一个机会约会体验一下。体验完了之后，请你先别拿爱不爱对方作为唯一的衡量标准。盲目的考虑爱会把你带入到爱的执念和漩涡当中。因为爱是所有体验的最终结论，而不是一个简单可衡量的标准。

你和前任经历了那么多事情，你付出了那么多，最终你得出一个结论说你很爱他，你深深的爱他，这没有问题。但是和这个新出现的人，你还没有经历那么多。请你千万不要急于下一个定论说我对他还没有感觉。我理解我们都渴望那种从身体到心灵完全心意相通的爱。我们都渴望得到对方的关注、关心和全身心的爱。同样我们也渴望全身心的爱对方这个人，但是人生不总是完美的。并不是每时每刻我们都能够得到我们想要的。当你遇到了不够完美的问题时，请你有一些耐心。把条件放到前面，而不是简单的把爱放到前面。爱很重要，但爱不能作为唯一的考量标准。

或许你会觉得我说要去看对方的条件，看对方的工作，看对方的身高，看对方的长相，这很功利。对，爱是要基于功利的基础上的。如果你只是一门心思地去纯粹的爱一个人，你最后有可能爱的不是一个人，爱的是你的幻想。爱的是你对于爱情这种美好的追求和幻想，而不是眼前这个真实的、有七情六欲、有缺点的男人。

我的一位学员，是知名企业家的女儿，她的经历很像是三星集团的长公主。第一段婚姻非常不幸，离婚后，她带着孩子继续生活。前任对她很不好。他们离开的原因是因为前任好赌酗酒，大家都不明白为什么这么优雅美丽的一个女人，条件出身这么高贵的一个女人，会爱上这么一个看上登途浪子一样的男人。她对我说：“老师，我实在是放不下他。我从小到大经历的那么多段别人看上去很棒的感情，只有我知道那些男人爱慕的不是我，是我的家庭。只有他对我是纯粹的。虽然他脾气很不好，但是他是真的爱我的。”听到这样的话，我真的很心痛。这个姑娘被自己对于爱情的理解，把自己给洗脑了。爱不是毒药，爱也不是让你的生活毁掉的可怕的灾难。爱是美好的。如果你发现和一个人在一起好像相爱，但是互相折磨，请你别骗自己了，也就是在互相折磨。这个时候冷静冷静吧，重新选择，或许并不是一件糟糕的事。它是让你看到这个世界，你有更多生命的可能性，让你看到这个世界，你有更多未来的可能性。

姑娘们，请你们在挽回的道路上，不要失去理性，不要失去对于爱的追求，也不要失去对于未来的期待。因为在追求爱的过程中，你可能会遇到更美好的风景。我是秦梦老师，我们的宗旨是在10年内帮助100万人获得情感幸福。如果你想随时联系到我，请在我的抖音号清风读心，清风就是我的名字，像一阵清风一样吹拂你的心。读心呢就是读懂你的心的意思。或者我的微信公众号齐眉情感。齐眉取自成语，举案齐眉的后两个字，情感就是我们心理学当中经常讲的情感。随时可以给我发私信，看到了，我会尽可能的给你回复。
