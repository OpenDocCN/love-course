# 林老头《系统认知》：033.成功过程全要素，补全自己的缺失的点

大家好，今天我们来探讨一下成功所需的要素。以下内容将按照要求进行翻译和整理。

## 概述

在本节课中，我们将从拿结果和成功角度分析成功所需的要素，并探讨如何补全自己的缺失点。

## 成功要素

以下是成功所需的要素：

### 1. 约会流程

- 约会流程是成功的关键，无论是确定长短期关系，都需要见面。
- 网上教学中有些观点认为无需约会，但这并不符合大多数人的情况。

### 2. 聊天

- 聊天过程中，最重要的是了解彼此，建立信任和安全感。
- 聊天前需要考虑资源，即你喜欢的人是否愿意与你聊天，你的容错率等。

### 3. 外在条件

- 外在条件包括外形、工作、性格、情绪、认知和情商等。
- 目前社交中，外形条件是最大的红利，因为很多人对自己的外形不够重视。

### 4. 资料

- 资料包括社交软件资料、相亲资料和搭讪时的第一印象等。
- 在整个环节中，要找出自己缺失的步骤，并努力补全。

### 5. 资源

- 资源指的是有效的人脉，例如加了30个人后，筛选出3到4个主动找你的女生。
- 资源需要通过实践和筛选来获取。

### 6. 聊天能力

- 聊天能力需要经验，例如成功谈过一个女朋友，让对方对你感兴趣等。

### 7. 约会能力

- 约会能力需要胆量和心态，例如牵手、表达喜欢等。
- 约会成功的标准是直接与女生回家。

## 总结

本节课中，我们学习了成功所需的要素，包括约会流程、聊天、外在条件、资料、资源、聊天能力和约会能力。希望大家能够根据自身情况，补全自己的缺失点，提升自己的能力。
