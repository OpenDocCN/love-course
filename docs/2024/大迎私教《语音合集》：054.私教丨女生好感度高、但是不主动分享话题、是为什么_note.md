# 大迎私教《语音合集》：054.私教丨女生好感度高、但是不主动分享话题、是为什么

🎼是你。どしね。hello hello，能听到吗？我的等忘错了。嗯，你说的问题是对方好感度很高，但有时候话题进行不下去，就是你觉得他对你是有好感的。然后你聊天聊完他就不接你的话，还是什么？

**问题分析**
- 对方好感度高，但话题进行不下去。
- 对方在聊天中投入不多，回复不多。
- 对方在另一个男生面前可以打开话题，在你面前打不开。

**解决方案**
1. **直接提出需求**：如果你希望对方对你好感度高，直接向他提出希望与他多聊天、推进关系。
2. **观察实际行为**：好感度是手段，最终目的是建立关系。观察对方是否真的为你做出实际付出。
3. **区分情绪与关系**：区分自己是否真的想建立关系，还是只是贪恋情绪。

**案例分析**
- 例子1：学员认为对方好感度高，但对方没有明显增加分享欲，甚至指责学员。
- 例子2：学员付出很多，但对方没有回报，导致学员越想付出更多。

**总结**
- 好感度是手段，最终目的是建立关系。
- 观察对方是否真的为你做出实际付出。
- 区分情绪与关系，明确自己的目的。
