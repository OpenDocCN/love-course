# 关系博弈（拒绝卑微和低位）：09_第九，如何通过博弈，让对方心甘情愿为你付

## 概述
在本节课中，我们将学习如何通过博弈策略，引导对方心甘情愿为你付出。我们将探讨期望回报、实际回报和保护欲等概念，并学习如何运用这些策略来增强关系的牢固性。

## 期望回报
期望回报是指对方在你身上投入时间、精力和金钱时，预期会得到相应的回报。以下是一些实现期望回报的方法：

- **例子**：在与男生约会时，你可以买一个护肤品送给你的闺蜜，当男生好奇询问时，你可以解释说这是对闺蜜礼物的回礼，并强调你是一个懂得感恩的人。
- **目的**：让对方感受到你是一个懂得回报的人，从而增加他们为你付出的意愿。

## 实际回报
实际回报是指对方在你希望他们为你做的事情之后，你给予的即时反馈和奖励。以下是一些实现实际回报的方法：

- **例子**：如果对方来接你下班，你可以给他一个拥抱或买一杯奶茶作为感谢。
- **目的**：让对方感受到他们的付出得到了你的认可和回报，从而增强他们继续付出的动力。

## 保护欲
保护欲是指对方在你身上产生的一种想要保护你的愿望。以下是一些激发保护欲的方法：

- **示弱**：通过适当的方式展示你的脆弱和需要帮助的一面，让对方产生想要保护你的愿望。
- **死鸭子嘴硬**：在对方面前表现出坚强和独立，让对方觉得你需要他们的支持。

## 总结
本节课中，我们学习了如何通过期望回报、实际回报和保护欲等策略，引导对方心甘情愿为你付出。通过运用这些策略，你可以增强关系的牢固性，让对方更加珍惜和投入这段关系。
