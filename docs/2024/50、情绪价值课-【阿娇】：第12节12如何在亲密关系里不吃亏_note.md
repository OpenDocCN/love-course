# 情绪价值课-【阿娇】：第12节 如何在亲密关系里不吃亏

大家好，我是娇姐。本节课我们将探讨如何在亲密关系中避免吃亏。

### 1. 保持吸引状态

**核心概念**：撩与追的区别

- **撩**：主动吸引，掌握节奏，享受生活。
- **追**：被动追逐，失去自我，容易焦虑。

### 2. 树立高价值感

**核心概念**：展现自我价值，不轻易示好。

- **初期交往**：不表现出强烈依赖，保持自我价值。
- **约会选择**：主动约对方，而非被动接受。

### 3. 偶尔接地气

**核心概念**：展现真实一面，拉近彼此距离。

- **高冷与接地气**：平衡展现，避免过于高冷或卑微。
- **生活分享**：分享简单快乐，拉近彼此距离。

### 4. 学会留下遗憾

**核心概念**：未完成的事情更容易留下深刻印象。

- **案例**：未看到的日出，留下的遗憾更深刻。
- **目的**：让对方反复思考，强化存在感。

### 5. 给足男人面子

**核心概念**：在他人面前维护对方尊严。

- **行为**：在他人面前支持对方，展现崇拜。
- **案例**：酒局中维护男友尊严，保护自己。

### 6. 付出与收获平衡

**核心概念**：控制投入，避免过度付出。

- **过度付出**：导致关系不平等，引发矛盾。
- **及时止损**：停止无谓牺牲，保持主动权。

本节课中，我们学习了如何在亲密关系中避免吃亏。希望这些方法能帮助你在感情中更加自信和快乐。
