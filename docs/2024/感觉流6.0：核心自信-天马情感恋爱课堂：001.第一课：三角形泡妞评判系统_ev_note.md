## 概述

## 课程内容

### 1. 三角形法则

三角形法则包括以下三个方面：

* **硬价值**：包括收入、家境、社会地位等。
* **外形**：包括服装、发型、皮肤状况等。
* **情绪价值**：包括懂不懂女人、是否有框架等。

### 2. 硬价值

* **收入**：根据自身收入水平，判断是否需要提高收入。
* **家境**：判断家庭背景是否为加分项。
* **社会地位**：根据自身社会地位，判断是否具有优势。

### 3. 外形

* **服装**：选择合适的服装，提升个人形象。
* **发型**：保持良好的发型，展现个人魅力。
* **皮肤状况**：保持良好的皮肤状况，展现自信。

### 4. 情绪价值

* **懂不懂女人**：了解女性的心理需求，提高沟通能力。
* **是否有框架**：坚定自己的立场，展现个人魅力。
