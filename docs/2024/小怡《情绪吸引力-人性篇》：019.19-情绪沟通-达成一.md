# 小怡《情绪吸引力-人性篇》：019.19-情绪沟通-达成一

在本节课中，我们将学习情绪沟通法的最后一个核心步骤：如何通过植入“诱惑性预期”来自然地达成一致，从而优雅地完成社交互动。

---

上一节我们探讨了如何建立联系感，本节中我们来看看整个情绪沟通流程的收官一步——达成一致。在社交五步法中，最后一步是达成一致。情绪沟通法的最后一步也是达成一致。两者的区别在于：五步法中的达成一致是一个**步骤**，而本节课的达成一致是一种**情绪**。两者的共同点是，它们都是整个社交过程的临门一脚，是无需耗费过多力气即可完成的事情。

本节课要讲的达成一致，核心不再是主动向对方提出要求，而是通过情绪的植入，让对方**自发产生**想与你达成一致的想法。

这个方法很简单：在即将与对方达成一致时，植入一些具有诱惑性的预期。这里的“预期”指事情发生前的期望。你在达成一致时加入一些美好的期待，就能增强“诱惑性”的情绪。

**请注意**：此处的“诱惑”并非特指男女之间的吸引，而是“兴趣”这种情绪的加强版。如果说兴趣是“有一点点喜欢”，那么诱惑就是“**我很想要**”。

以下是具体操作方法，通过两个例子来说明：

**1. 社交邀约场景**
假设你与对方相谈甚欢，希望约对方见面深入接触。
*   **流程性做法（无情绪）**：你说：“有机会一块出来坐坐吧。”这只是一种流程性的提议。
*   **加入情绪性预期的做法**：如果对方是个“吃货”，无辣不欢，你可以说：“我听说最近开了一家特别正宗的四川火锅，有机会咱一块去尝尝。”这就植入了一个对“吃货”而言难以抗拒的期待，从而让达成一致的吸引力更强。

**2. 职场面试场景**
假设你作为产品经理，与一家公司面试后彼此满意，约定与总经理复试。
*   **流程性做法（无情绪）**：你说：“那您看咱们什么时候复试呢？”
*   **加入情绪性预期的做法**：你可以说：“那下次跟总经理聊的时候，我会带着对咱们公司产品的修改建议来，这样效率会高一些。您看安排在哪天呢？”这展现了一种积极主动的态度，对招聘方而言具有吸引力。

**然而，有一个至关重要的原则必须注意：你植入的诱惑性预期，其强度绝不能超过你本身对对方的吸引力。**

还记得我们在训练营前几课讲过的社交错误心态吗？即“**我 + X = 对方**”，这是一种自我价值感不坚定的表现。在这里同样适用。

以下是两个反面例子，说明诱惑过大的问题：

*   **社交场景**：如果你对对方说：“我认识成龙，有机会我带你去他家玩。”这个“诱惑”（见成龙）的价值可能远大于你本人的吸引力，这会导致对方答应邀约的动机扭曲，社交关系本质发生改变。
*   **面试场景**：如果你说：“如果有机会合作，我也想在这个项目上投点钱。”这个“诱惑”（获得投资）可能让对方忽视你的工作能力，导致关系走向偏离你的初衷（获得职位）。

因此，务必把握分寸。植入的预期应该是“锦上添花”的美好期待，而非颠覆关系基础的巨大诱惑。

---

**本节课总结**
本节课我们一起学习了情绪沟通法的最后一步：**达成一致**。其核心是在最终提议时，巧妙植入一个**适度且美好**的预期，从而增强对方的意愿，让达成一致的过程更自然、更有效。关键在于，这个预期必须是为你本人吸引力**加分**的“小诱惑”，而非可能掩盖你本身价值的“大诱惑”。

---

截止目前，情绪沟通法的五个步骤已全部讲解完毕：**兴趣、情绪共振、吸引、联系感、达成一致**。

这五种情绪与社交五步法（破冰、了解、筛选、欣赏、达成一致）相辅相成：
*   **破冰阶段** 对应 **兴趣** 与 **情绪共振**。
*   **了解阶段** 对应 **情绪共振** 与 **吸引**。
*   **筛选之后**的欣赏阶段对应 **联系感**。
*   **最终** 对应 **达成一致**。

当你将“社交五步法”的框架与“情绪沟通法”的技巧结合理解并加以练习后，你会发现社交互动变得有规律可循。通过努力实践，你将能更体面地进入一段关系，并使双方都感到舒适。
