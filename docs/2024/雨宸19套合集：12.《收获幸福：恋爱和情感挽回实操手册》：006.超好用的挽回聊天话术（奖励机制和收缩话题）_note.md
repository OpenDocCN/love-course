# 雨宸19套合集：12.《收获幸福：恋爱和情感挽回实操手册》：006.超好用的挽回聊天话术（奖励机制和收缩话题）

## 概述
在本节课中，我们将学习如何运用奖励机制和情绪预设来挽回情感，以及如何处理收缩话题。

## 奖励机制与情绪预设
**奖励机制**是指通过持续付出和展示正能量，让对方对你产生依赖。**公式**：持续付出 + 正能量 = 依赖性。

**情绪预设**是指在对方情绪低落时预设一个引爆点，通过情绪预设来激发对方的情绪反应。

**结合使用**：将奖励机制和情绪预设结合起来，可以更有效地挽回情感。

## 收缩话题与开放式话题
**收缩话题**是指对方不想继续聊天的状态，例如：“嗯啊，我知道了。”
**开放式话题**是指让对方愿意继续聊天的状态，例如：“你说的太棒了，我还没听完呢。”

## 面对收缩话题的处理方法
1. **价值止损**：遇到收缩话题时，不要继续聊天，以免价值损失。
2. **分析原因**：对方可能因为自身问题或对您有排斥感。
3. **调整策略**：根据原因调整聊天策略，必要时退回重新开始。

## 聊天注意事项
1. **避免敏感话题**：不要聊情感或挽回相关话题，以免引起对方不适。
2. **培养回信息能力**：不要过度依赖他人，学会独立解决问题。
3. **不要焦虑**：在聊天框内聊天，避免犯错。

## 总结
本节课中，我们学习了如何运用奖励机制和情绪预设挽回情感，以及如何处理收缩话题。希望这些方法能帮助您在情感挽回的道路上取得成功。
