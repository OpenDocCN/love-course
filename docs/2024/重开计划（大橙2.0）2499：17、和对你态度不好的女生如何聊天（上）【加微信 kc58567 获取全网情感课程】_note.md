# 重开计划（大橙2.0）2499：17、和对你态度不好的女生如何聊天（上）

## 概述

在本节课中，我们将学习如何与对你态度不好的女生进行聊天。我们将探讨如何通过改变聊天风格和展示自己的生活，来改变对方对你的固有看法，并最终建立更深层次的关系。

## 第一部分：了解对方特质

**1. 对方特质**

- 明确知道自己想要什么。
- 需要感受到你能为她带来什么。

**2. 保持神秘感**

- 避免直接表达意图，让对方感到好奇。
- 使用含糊不清的语言，让对方猜测你的想法。

**公式**：

```
神秘感 = 含糊不清的语言 + 好奇心
```

## 第二部分：改变聊天风格

**1. 阶段一：展示不在乎**

- 当对方给你下定义时，不要纠结，保持无所谓的态度。
- 使用轻松的语气，例如：“原来我在你心里就是个花心大萝卜呗。”

**2. 阶段二：展示生活**

- 通过朋友圈展示自己的生活，例如：工作、兴趣爱好等。
- 让对方看到你的另一面，例如：工作努力、生活有序。

**3. 阶段三：聊天展示心路历程**

- 与对方分享你的心路历程，例如：恋爱观、生活态度等。
- 通过聊天让对方了解你是一个什么样的人。

**4. 阶段四：改变对方看法**

- 当对方意识到误会你后，你的吸引力会更大。
- 此时，你可以尝试升级关系。

## 第三部分：总结

本节课我们学习了如何与对你态度不好的女生进行聊天。通过改变聊天风格、展示生活和聊天展示心路历程，你可以改变对方对你的固有看法，并最终建立更深层次的关系。
