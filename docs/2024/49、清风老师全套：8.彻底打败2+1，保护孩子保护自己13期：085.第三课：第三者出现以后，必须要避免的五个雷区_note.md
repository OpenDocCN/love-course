# 清风老师全套：8.彻底打败2+1，保护孩子保护自己13期：085.第三课：第三者出现以后，必须要避免的五个雷区 🚫

## 概述
在本节课中，我们将学习如何避免在发现第三者后陷入不必要的困境，保护自己和孩子的情感。

## 避免的五个雷区

### 雷区一：质问指责
**描述**：找到证据后，直接质问对方，指责其出轨行为。
**公式**：质问指责 → 对方感到羞耻 → 消除证据
**避免方法**：掌握更多有利证据，如财产转移、大额礼物等。

### 雷区二：公开威胁
**描述**：在对方单位公开威胁，特别是当第三者是同事时。
**公式**：公开威胁 → 失去同事支持 → 对方产生反感
**避免方法**：私下沟通，掌握主动权，用威胁代替公开闹事。

### 雷区三：暗示试探
**描述**：通过暗示和试探，希望对方主动断绝关系。
**公式**：暗示试探 → 提高对方警惕性 → 消除证据
**避免方法**：直接沟通，表达真实想法，共同探讨未来。

### 雷区四：向亲人告状
**描述**：找到证据后，向对方父母或亲人告状。
**公式**：向亲人告状 → 对方反感 → 关系恶化
**避免方法**：通过亲人支持，获取更多筹码，改善自我，吸引对方。

### 雷区五：过度自责
**描述**：将出轨责任全部归咎于自己。
**公式**：过度自责 → 沉入抑郁情绪 → 影响解决问题
**避免方法**：合理看待问题，分析自身优势劣势，避免一味自责。

## 总结
本节课中，我们学习了在发现第三者后，要避免的五个雷区。通过掌握有利证据、私下沟通、寻求亲人支持等方法，我们可以更好地保护自己和孩子的情感。在下一节课中，我们将学习如何认识第三者的类型。
