# 清醒mu姐9套合集：2.mu姐 《高嫁实操秘籍——向上社交》系列课：015.4.14 如何在向上社交中提升价值感

## 概述
在本节课中，我们将学习如何在向上社交中提升个人价值感，以便在与人交往时更加自信和有效。

## 核心概念
**价值感提升公式**： 自我认知 + 价值创造 + 场域选择

### 1. 自我认知
**公式**： 自我认知 = 内在价值 + 外在表现

- **内在价值**： 挖掘个人核心竞争力，包括技能、知识、经验等。
- **外在表现**： 通过言行举止展现个人价值，消除弱者思维。

### 2. 价值创造
**代码**： 价值创造 = 需求识别 + 解决方案

- **需求识别**： 了解他人需求，寻找自身价值与需求的契合点。
- **解决方案**： 提供有效解决方案，帮助他人解决问题。

### 3. 场域选择
**公式**： 场域选择 = 高端社交 + 有效互动

- **高端社交**： 参加有价值的社交活动，结识优质人脉。
- **有效互动**： 在社交场合中，积极沟通，展现个人价值。

## 教程内容

### 1. 向上社交定位
- **关系本质**： 利益链接，价值交换。
- **社交筹码**： 个人能为他人创造的价值。

### 2. 摒弃弱者思维
- **弱者思维特征**：
  - 对付出渴求度高，患得患失。
  - 不懂装懂，抗拒新鲜事物。
  - 刻板印象，思维僵化。
  - 爱占便宜，无法控制欲望。
- **强者思维特征**：
  - 以价值最大化为导向。
  - 宽容度高，不计较。
  - 思维方式和能量场积极向上。
  - 终身学习，不断进步。

### 3. 与高层次人社交心态
- **心态**： 学会把握分寸，进退得宜。
- **尺度**： 倾听大于表达，避免越界。
- **社交成本**： 每次开口都是消耗，有价值的人脉要用在高价值的事情上。

### 4. 场域选择
- **拒绝无效社交**： 精简朋友圈，维护有价值的关系。
- **融入圈子**： 找到适合自己的圈子，获得圈子认可。
- **社会身份**： 参加协会、商会等组织，提升社会地位。

## 总结
本节课中，我们学习了如何在向上社交中提升个人价值感。通过自我认知、价值创造和场域选择，我们可以更好地融入社交圈，结识优质人脉，实现个人成长。希望同学们能够将所学知识应用到实际生活中，不断提升自己。
