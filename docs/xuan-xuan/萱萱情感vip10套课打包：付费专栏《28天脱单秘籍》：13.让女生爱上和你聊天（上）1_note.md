# 课程13：让女生爱上和你聊天（上）💬

在本节课中，我们将要学习与女生聊天时的核心目的、常见误区以及如何通过聊天建立舒适感与安全感，从而为成功邀约和推进关系打下基础。

## 为什么聊天如此重要？

聊天是与女生接触过程中最频繁的互动方式，也是无法忽略的环节。大部分不擅长追求女生的男生，往往会在聊天环节遇到困难。

## 聊天中常见的普遍问题

以下是男生在与女生聊天时最常遇到的问题：

*   不知道聊什么话题。
*   不知道如何发送开场白。
*   女生不回复消息。
*   女生刚开始愿意聊，但后来逐渐冷淡。
*   女生回应很冷淡。
*   和女生聊得不错，但不知道下一步该做什么。
*   不确定应该多久和女生聊一次。

在聊天课程中，我们将逐一解答这些问题，并讲解如何正确地与女生聊天。存在这些问题，通常意味着陷入了聊天误区，并且聊天缺乏明确方向。

## 聊天的核心目的是什么？

针对方向问题，首先需要明确聊天的目的。女生不会仅凭网上聊天就贸然与一个男生确定关系。她们明白，当男人喜欢她时，会表现出各种好。但女生会通过约会互动观察细节，来判断这些特质是否真实，进而筛选是否要与你交往。

因此，**聊天的核心目的是把女生邀约出来，以推进关系**。

## 如何克服邀约障碍？

女生在被男生邀约时，常会产生诸多顾虑。例如：和不熟的男生出去是否不妥？会不会遇到奇怪或尴尬的事？安全是否有保障？如果互相没看上会不会尴尬？答应太快会不会显得自己很廉价？

这些问题都源于女生与你不熟。所以，在聊天中，我们需要消除这种陌生感。

## 建立舒适感与安全感

我们需要通过联系和互动，构建舒适感和安全感，进而产生亲密感。如果缺乏舒适感和安全感，女生就不可能与你持续互动，聊天会变成尬聊或她不再理你。

在上一课中，我们讲解了建设网络展示面，其目的就是为了建立基础的安全感。懂得在聊天中展现自信，才能让女生对你产生安全感。

## 聊天方式对比分析

以下是两组聊天对比，我们可以分析其差异。

**第一组对话：**
```
男：在干嘛呢
女：上班
男：几点钟下班
女：6点
男：下了班干嘛呢
女：不知道
男：方便的话我请你吃个饭
```

这段对话是典型的直男思维，聊天直奔主题，只想达到自己的目的。

**第二组对话：**
```
男：唉，累瘫了，好久没运动了。你在干嘛？
女：在家呢。
男：我也刚到家，现在躺在沙发上一动也不愿意动。
女：你这是干嘛去了？
男：想把八块腹肌给练出来，恢复我魔鬼般的身材。
女：还魔鬼般的身材，估计有点难。
男：我每周都会去爷爷家看他，这一健身减下来的肉都长上来了，他做的吃的诱惑太大了。
女：哈哈，老人家就是这样子的。
```

第二段对话展示了自身的生活状态和感受。对方能从这些碎片化信息中逐步了解你，从而建立安全感。之后再顺势提出邀约，就会自然得多。

## 如何通过聊天升温感情？

男女相处可以借助聊天让感情升温。但只有聊情绪和感受，让女生了解你的状态，才能引发共鸣、加深联系感，进而升温感情并产生亲密感。

再举一个例子。聊天中女生问：“你多大啦？”很多男生只会老实回答：“93年的。”女生回一个“哦”，然后对话就结束了。

这里我们可以针对话题进行延展。例如，你可以说：“93年的，但感觉这年龄不尴不尬的。本来是小鲜肉的年龄，却开始有小朋友喊叔叔了；说是大叔吧，可我感觉自己长得还挺嫩……”这样聊天能让女生感受到你的感受。

## 如何生动地展示自己？

又比如聊到工作。我问很多男生做什么工作，他们的回复常让我觉得问了等于没问，因为我根本没听懂。他们会说：“在银行上班”、“做金融的”、“做电子商务的”。女生对这些概念通常没有具体印象，至少我听完很快就会忘记他们是谁。

我们可以换一种方式说：“我的工作嘛，就是每天跟一群美女帅哥打交道，感觉自己都有点审美疲劳了。我是一名摄影师。”这种回复是否更有画面感？

当我们和女生面对面交流时，对方能看到我们的表情、肢体动作，感受到我们情绪的感染力。但通过文字交流时，你只有几个传递信息的冰冷文字。因此，**文字表达是否生动有趣至关重要**。偶尔使用夸张、拟人化的方式是很有必要的。

## 主动进行信息筛选

男生主动传递关于自己的信息，可以让女生在心里对你进行筛选。如果你什么信息都不透露，或者透露得特别少，女生会觉得了解你需要花费大量时间和精力。在吸引力不是特别强的情况下，她可能会倾向于直接拒绝你。

善于在聊天中展示自己的信息，就不会出现“一问一答”式地不断索取信息的情况。这其实是在方便女生筛选你。如果你只是一味索取信息，盲目推进关系，那肯定会越聊越尴尬。

## 避免破坏舒适感的行为

在聊天中，建立舒适感必不可少。其中，最常见破坏舒适感的行为就是过早暴露情感需求，或者一开始就没轻没重地开黄腔。

关于开黄腔就不必多说了，这会让女生把你划入猥琐渣男的行列。
