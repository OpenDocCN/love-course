# 宣宣聊天课程《这么聊，更吸引她》：14.聊天中的幽默感建立技巧_note

在本节课中，我们将要学习如何在聊天中建立幽默感。幽默感是区分优质男性与平庸男性的关键特质，它不仅是高情商的体现，也是一种强大的社交直觉和内在价值的展现。

## 幽默感的价值与本质

上一节我们介绍了课程主题，本节中我们来看看幽默感的核心价值。幽默感能为女生带来积极的情绪波动。与幽默的男生相处，女生会感到开心和轻松，而非沉重。

深入来看，幽默感体现了一个人的自信与生活智慧。它是一种以柔克刚的生活态度。生活充满挑战，而一个人对待生活的态度能反映出其内心是强大还是懦弱。不自信的人往往害怕被看低，因此倾向于用强硬的姿态伪装自己。

许多男生表面自尊心强、怕丢面子，实则源于内心的自卑。他们过度在意他人看法，神经时刻紧绷，这类人通常缺乏幽默感。自信的人则恰恰相反。

自信源于对自我的认可，因此他们不太在意外界的评价。当一个男生足够自信时，他在聊天中便不会刻意迎合女生。他会发自内心地感到愉悦，各种有趣的想法便会如泉水般自然涌现。

女生会被这种状态感染，从而感到快乐。这才是真正的幽默感。许多自认不够幽默的男生喜欢背诵笑话，试图在聊天时复述，将自己变成“段子手”。

但如果你仅将笑话当作话术使用，自己内心并未感到真正的开心，也未体验到生活的放松感，那么女生听完后也不会觉得你幽默，反而可能感到尴尬。因为幽默的关键不在于你说了什么笑话。

而在于你是否是一个富有幽默感的人，以及你在生活中的状态是否放松。因此，我们的目标不是收集笑话，而是首先要理解：幽默本质上是一种思维方式，而非一种说话方式。

## 幽默的思维：打破惯性

我们在生活中会基于经验形成许多惯性思维。现代进化心理学认为，有趣的事物常伴随着不协调、新奇或独特的视角。幽默就是通过打破惯性思维，做出意料之外的事情，从而产生有趣的效果。

这好比如果你的生活是家与单位两点一线，你每天都会走同一条路。你对这条路非常熟悉，甚至能记住沿途的小吃，闭着眼睛也能走。这就是你的惯性。假如有一天。

有人带你去一个从未去过的地方，你就会看到不一样的风景。你会因这种新鲜感而兴奋，获得一种前所未有的特别体验。这就是打破惯性后产生的感觉。你会将看到新风景的兴奋情绪。

归因于带领你的这个人。你会认为这个人能给你带来新鲜感和不同的美好体验。我们与女生聊天时也是如此。幽默的本质是“温和的违反”。当你打破自己的思维惯性后。

你就会说出一些让人意想不到的话，从而给对方带来幽默的感觉。

以下是运用“打破惯性”思维进行幽默对话的示例：

*   **示例对话1：**
    *   男生：`你多大呀？`
    *   （女生的惯性思维会认为你在问年龄，她可能回答：22。）
    *   男生：`我是在问你的罩杯啦。`
    *   （这个回答打破了关于“多大”的常规理解，制造了意外感。）

*   **示例对话2：**
    *   （约女生晚上出去）
    *   女生：`这么晚出去干嘛？`
    *   男生：`放心吧，我是坏人，不会对你太好的。`
    *   （这个回答没有一本正经地解释目的，而是用反向调侃增加了安全感并显得有趣。）

当女生的情绪被我们调动起来后，对话就会进入情绪交流模式。此时，双方不再交流严肃信息，而是交流情绪。当我们能与女生交流情绪时，聊天便提升到了另一个维度。

一个没有幽默感的人会思考“说什么才能变得幽默”，而一个富有幽默感的人，无论说什么都会让女生觉得有意思。因此，有些男生看似普通，却能凭借幽默感吸引很多女生。因为他的幽默感让女生在聊天中感到舒适、愉快和放松，这种感觉是每个人都喜欢和需要的。

## 实践技巧：绕个弯说话

其实幽默并没有那么难。只要稍微改变一下直线思维，把一些话绕个弯来说，我们也能聊出幽默感。

例如，我们约女生出去玩，女生问：“我能带上一个小姐姐吗？”这通常是女生安全感不足的表现。我们肯定不希望她带其他人，但也不能直接说“不行”。

这时可以用幽默的方式表达自己的意愿。比如回答：“我单独见你，你还要带个保镖吗？”当然，具体回答需视情况和你与女生的关系状态而定。正常情况下，她不会认为自己带的朋友是“保镖”。我们用幽默的方式点出了。

她带朋友同行的潜在目的（寻求安全感），同时又委婉表明了更希望单独约会的意图，这会使她更容易接受单独约会这个提议。

## 重要原则：分清场合，该严肃时严肃

运用幽默时，必须记住一个核心原则：**重视正经话题**。这是什么意思呢？

当女生需要我们提供资讯、信息，或在比较严肃的事情上向我们求助、征求意见时，就不能天南海北地瞎扯。此时的瞎扯不叫幽默，而叫不靠谱。

例如，女生很认真地问你：“学车的话哪个驾校比较好？有没有推荐？”这时如果你回答“中国山东找蓝翔”，就会显得过于刻意和不合时宜。女生会觉得无法从你这里得到想要的答案，她对你的信任感就会下降。

她会认为下次有事问你也没用，因为你无法提供任何实质性的帮助。所以，我们要懂得幽默，也要懂得在该严肃的时候严肃。这本身就是一种“推拉”的艺术。否则，幽默就会变成油嘴滑舌、油腔滑调，反而会让我们减分。

## 课程总结 🎯

本节课中，我们一起学习了聊天中建立幽默感的核心理念与技巧。

1.  **幽默感是价值与状态的体现**：幽默既是一种强大的社交智慧，也是我们内在价值的体现。它能给女生带来轻松愉快的正面情绪波动，同时也是我们自信、放松状态的表现。
2.  **幽默是一种思维方式**：讲笑话不代表有幽默感。真正的幽默感源于内心对事物的愉悦感受、对生活的放松态度，并愿意以你喜欢的方式与女生分享。刻意的幽默反而会造成尴尬。
3.  **幽默吸引人的原因**：我们生活中多依赖惯性思维。幽默相当于打破了这种惯性，带来的刺激感会制造新鲜感和不同的体验。女生会将这种好的情绪体验归结于你这个人，从而对你产生好感。因此，没有幽默感的男生会想“说什么话能变幽默”，而有幽默感的男生，不管说什么女生都会觉得有意思。
4.  **幽默可以练习**：只要稍微改变直线思维，学会把话绕个弯说，就能逐渐培养出幽默感。但有一点要特别注意：**一定要重视正经话题，该严肃时不要乱开玩笑**。这样才能让女生觉得我们靠谱，建立起信任感。切记不要用力过猛，变成一个让女生觉得你只会油腔滑调的男人。
