# 宣宣聊天课程《这么聊，更吸引她》：13. 让女生产生情绪波动的聊天方式 💬

在本节课中，我们将要学习如何通过聊天与女生产生情感连接，让对话不再平淡，从而有效增进关系并建立亲密感。核心在于理解并运用“情感交流”，而非仅仅交换信息。

聊天是增进感情的重要方式。但有些人能让感情升温，有些人却把天聊死。区别在于聊天是否包含了情感层面的交流。

## 从两个例子看聊天差异

为了理解情感交流的重要性，我们可以对比两个场景。

假设你有一位女同事。她性格刻板，总是很严肃，很少笑，打扮保守，给人一种距离感。

你们交流频繁，但内容仅限于工作，例如索要数据、分配任务、核对进度。

虽然交流多，但你们不会因此变得亲密。因为聊天内容只有工作，没有情感交流。

如果这位同事离职，你大概率不会感到不舍。因为她的去留与你关系不大，对工作也无影响。

假设这位同事离职后，来了一位新女同事。她性格开朗，打扮精致且得体，给人舒适大方的感觉。

你们也经常交流工作，但她会时不时与你聊个人琐事、明星八卦或她遇到的有趣小事。你也会与她分享自己的故事。

渐渐地，你们比其他人更了解彼此。你会在买早餐时悄悄帮她带一份，她会提醒你明天带伞。

当她有一天辞职时，你会感到失落和想念。即使不在一起工作，你们也会在微信上保持联系，分享近况。你们之间产生了亲密感。

同样是交流，为什么结果截然不同？原因在于，与第二位女生的聊天让你产生了**情绪上的波动**。这就是本节课的核心——**情感交流**。

要让关系变得更亲密，就需要与女生进行情感层面的交流。

## 如何进行情感交流：三个关键方面

在具体聊天中，我们可以从以下三个方面入手：状态、感受、关联。

### 1. 状态：营造积极的聊天氛围

状态是指我们聊天时的精神面貌和带给对方的整体感觉。在之前的例子中，第二位女生的状态明显更好，无论是外表、性格还是聊天氛围，都让人感到热情、亲切和舒适。而第一位女生则显得冰冷无趣，让人不愿多接触。

这就像问你是喜欢看新闻联播还是球赛。你肯定更喜欢球赛，因为新闻联播太正经，而球赛热闹、放松、有意思。

**“有意思”就是一种很好的状态。**

你可以把自己想象成一个电视节目。如果你的对话像说教一样一本正经，就如同新闻联播，会让女生觉得无聊。你需要让自己变得像一场精彩的球赛，去调动她的情绪。

**具体做法包括：**
*   讲一些夸张或搞笑的话。
*   时不时不按常理出牌，制造意外感。

目标是让女生在与你聊天时，心情不是一条直线，而是像**心电图一样有起伏**。这样她才会对你印象更深刻，并愿意在你身上投入更多精力。

### 2. 感受：多交流情绪而非单纯事实

感受是指在聊天中，多分享和交流彼此的情绪与感觉，而不是只陈述客观事实。

如果你观察女生之间的谈话，会发现她们几乎都在交流感受。例如，同样遇到堵车：

*   **男生可能说：** “这段路每天都堵，至少堵半小时。下次我们从桥下过吧。”（关注结论和解决方案）
*   **女生可能说：** “天哪，这么热的天堵在这晒太阳，我又要晒黑了，闷死了！会不会中暑啊？早知道抹点防晒了，这几天皮肤很干，估计面膜都救不回来了……”（关注自身感受和受到的影响）

女生的关注点往往不在事件本身，而在事件带给自己的情绪和影响。她可能并不关心你的结论。

有句话说得好：**“你要陪她去做她爱做的事，才有机会做你爱做的事。”**

因此，和女生聊天时，要顺着她的思路，多聊感受。

**例如，女生问：“在干嘛？”**
*   **回复A（只陈述事实）：** “才下班。”
*   **回复B（加入感受）：** “才下班，累死了……”

显然，回复B更好，因为它带来了情绪，让对话“活”了起来。这也是为什么很多人喜欢用表情包——它能表达更饱满的情绪。

在聊天中，可以借助**表情、标点符号（如“！”、“~”）** 来辅助表达情绪，让对话更有意思。

### 3. 关联：通过话题表现自己并建立连接

关联是指在聊天中，敢于通过话题展示自己，并与女生产生互动和连接，而不是只当听众或机械地回答。

宣宣曾有一位私教学员，女生（短发）问他：“我要不要接长头发呢？”

一般男生会直接给出答案，如“好啊”或“不用，现在挺好”。但这会让对话变成一问一答，然后结束。

真正好的对话应该去**表现自己**并**建立关联**。

**你可以这样思考：**
*   她是不是一个人去？什么时候去？需不需要陪伴？
*   如果她一个人，你可以提议：“要不要我陪你去？”
*   同时，在回答中向自己靠拢，并对她做一个**筛选**：“我好像没见过你长头发的样子哦。长发的女孩通常比较温柔吧，我很喜欢这样的女孩。”

**这样做的效果是：**
1.  如果她真的接了长发，相当于完成了你的“筛选”，你知道自己对她有吸引力。
2.  女生问你，可能真有此打算，顺着说一般没问题。
3.  女生可能会逆向合理化自己的行为：“我做了他喜欢的事，说明我在乎他的看法。”

接着，女生又问：“那黑长直会不会太单一？我要不要染个颜色？”

如果回答“要”或“不要”，话题又会很快结束。

**建议的做法是建立关联：** “不如你发张之前的照片我看看？我帮你参谋下适合什么颜色。”

一般情况下，女生不会轻易给男生发照片。如果她发了，说明她对你的**好感窗口很大**，同时也完成了一次**服从性测试**。这样聊下去，你就能轻松把控节奏，推进关系。

## 课程总结 📝

本节课我们一起学习了如何通过情感交流让女生产生情绪波动，从而建立亲密感。

**第一，情感交流是建立感情基础的关键。** 无论是销售对客户、创业者对投资人，还是追求者对心仪女生，讲故事和情感连接都远比单纯讲事实、摆逻辑更重要。

**第二，进行情感交流可从三个方面入手：**
1.  **状态：** 聊天时保持良好、放松的状态，适当开玩笑，让氛围轻松有趣，像一场“有意思的球赛”。
2.  **感受：** 多交流情绪和感受，而非干巴巴的事实。让对话充满能量，例如将“明天10度”说成“明天穿厚点，不然感冒了我会心疼的”。
3.  **关联：** 敢于表现自己，并通过话题与女生产生连接。例如，当女生说喜欢某位作家时，你可以说：“巧了，我也喜欢。而且我觉得喜欢她作品的姑娘都很有灵性。”这远比单纯附和“她作品不错”要好得多。

许多情场高手并非外表完美，但他们善用情感交流，让女生为之着迷。只要你用心体会并多加练习，相信你也能掌握这门“秘密武功”。
