# 萱萱情感VIP课程（六）：第十一节 🚶‍♂️走出跪舔的爱情

在本节课中，我们将要学习如何识别并摆脱在爱情中的“跪舔”模式。我们将分析这种模式是如何形成的，并提供具体的策略来建立健康、平等的恋爱关系。

上一节我们探讨了失衡关系的表现，本节中我们来看看，我们在爱情里面到底是怎么一步一步陷入跪舔模式的。

你感到沮丧，你想不通：**为什么她要什么我给她什么，她却一点都不在乎我？为什么我越关心她，她好像对我越反感？** 就算你小心翼翼，女生最终还是提出了分手。你一下子就像被抽了筋一样痛苦不堪。你去求女孩的闺蜜和家人，想让女孩跟你见面，好再给你一次机会。女孩发现你居然这么难缠，更坚信自己离开你是正确的选择。

你觉得自己很委屈，但实际这一切都是你自己一手造成的。

我们从头来看，如果要改写这个结局，在这段恋爱的过程中，我们应该怎么去做。以下是五个关键的调整策略：

**第二，不要让她预知你所有的行为。**

**第三，减少可得性。**

**第四，抓紧控制权。**
女生归根结底是慕强的，她要找一个更强大的雄性去依靠。当这个雄性不再展示自己强大属性的时候，男人在她眼中就会失去魅力。你总是对女人唯唯诺诺，甚至放弃尊严跪舔，那这个时候女人会觉得：“我说什么就是什么，我让你干什么你就干什么，那我要你这个男朋友有什么用？”

**第五，保持投入平衡。**

到这里我们就可以发现，造成跪舔的根本原因，就是男生在开始的时候心态没有摆正。如果你面对女生的时候，是以一种**仰视**的、**高不可攀**的心态去看待，把女生抬高了，那你的姿态自然就放低了。而这种不平等，女生是能感觉到的。她会认为：“既然你会这样做，说明你确实不如我。”因此，你在她心中的价值感就会打折。

最可惜的是，因为前期为女生投入太多，天平严重失衡，所以女生离开你的时候，你会觉得舍不得，会放下尊严苦苦相求。不过，最后是否能复合，如果你依然跪舔，也不会得到你真正要的那种爱和尊严。
