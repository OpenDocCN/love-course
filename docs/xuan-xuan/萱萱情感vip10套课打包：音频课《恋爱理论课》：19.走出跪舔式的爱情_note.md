# 恋爱理论课 19：走出跪舔式的爱情 💔

在本节课中，我们将要学习一种在亲密关系中常见但有害的模式——“跪舔式”爱情。我们将剖析其形成过程、背后的心理动因，并学习如何建立健康、平等的恋爱关系。

很多男生会以“爱”的名义去跪舔对方。他们总是从女生的角度出发，考虑对方需要什么、想要什么，然后无条件地对女生好。

结果就是越来越失去自我。当你完全失去自己的框架和对感情的主控权时，你的存在对女生而言就失去了意义。

## 跪舔模式是如何形成的？

上一节我们提到了跪舔的危害，本节中我们来看看，在爱情里是如何一步步陷入这种模式的。

**第一阶段：仰视与追求**

**第二阶段：过度付出与丧失神秘感**
你们在一起之后，你把这个女生当做手心里的宝贝，变本加厉地对她好。早上按时叫她起床，晚上按时说晚安。你把女生当做一个不能自理的幼儿园小朋友，恨不得吃饭都要你去喂。很快，你的这些行为就能被女生轻易预测到。每天早上微信一响，女生不用看就知道是你发来的“早安”，自然也懒得回复。她心里会想：“整天就知道早安晚安，真没意思。”

**第三阶段：关系失衡与自我怀疑**
然后你会发现，女生不再像以前那么亲热了。你觉得一定是自己做得还不够好，于是更频繁地嘘寒问暖、关心示好。没想到女生接到你的电话，语气却更加不耐烦。你对女生说“时间不早了，要早点回家”，女生却回答：“我又不是幼儿园小孩，还要你来教。”你听见女生发脾气，也不敢多说什么，只好发微信道歉：“宝宝对不起，我没想让你生气。”女生没有回复，继续和朋友玩。这时你发现，自己和女生之间的关系完全由女生说了算。

**第四阶段：彻底丧失主动权与价值**
在你的心态已经完全“跪下去”之后，你对女生越来越逆来顺受。女生说什么就是什么。女生心情好，叫你一声“亲爱的”；心情不好，你再怎么着急，她也不会接你电话。你还不能发脾气，因为一旦发脾气，女生就不理你了。你感到沮丧，想不通的是：“为什么她要什么，我给她什么，她却一点都不在乎我？为什么我越关心她，她好像对我越反感？”

**第五阶段：关系破裂与痛苦纠缠**
即使你小心翼翼，女生最终还是提出了分手。你一下子像被人抽了筋一样，痛不欲生。你去求女生的闺蜜和家人，想让女孩和你见面，再给自己一次机会。女孩发现你居然这么难缠，庆幸自己离开你是正确的选择。你觉得自己很委屈，但实际上，这一切都是你自己一手造成的。

## 如何改写结局：建立健康的恋爱模式

我们从头来看，如果要改写这个结局，在这段恋爱过程中，我们到底应该怎么做呢？以下是五个核心的修正步骤。

**第一步：建立筛选**
当你遇到一个高价值的女生时，不能让她觉得，她漂亮或身材好，在你这里就可以获得额外的社交红利，你就会对她特殊对待。一旦你这样做，女生心里就会觉得：“你和其他男人一样，看到我长得好看就想泡我，那我也跟你逢场作戏好了。”
正确的做法是，当你看到她身上的闪光点以后，去**筛选**她。让她知道你不是那种肤浅的男生。如果她徒有好看的外表，内心空空如也，那你也不可能喜欢她。

**第二步：保持神秘感与挑战性**
不能让她预知你所有的行为。男人一定要有自己的主见。如果你毫无挑战性和神秘感，女生很快就会对你乏味。
所以在和女生相处的过程中，重要的不是“早安晚安”，而是要给她**制造情绪波动**，制造让她印象深刻的回忆。如果女生和你在一起一年，但是每一天都是一样的内容，那么这一年也就白过了，她怎么可能会觉得开心呢？

**第三步：适当减少可得性**
当一样东西取之不尽、太容易得到的时候，它本身就不再是稀缺资源了。当一样东西不再是稀缺资源时，它的价值就开始贬低。
同理，当你的爱和关心已经“取之不尽”的时候，那么你对她的好就开始贬值了。女生发脾气你也不敢反抗，女生生气你就给她买东西。你给的可得性太高，让自己的吸引力不断下降。你希望用这种方式让女生离不开你，实际上完全走错了方向。

**第四步：紧抓关系控制权**
女生归根结底还是慕强，她要找一个强大的雄性去依靠。当雄性不再展示自己强大的属性时，男人就会活得不像男人。你总是对女人唯唯诺诺，甚至放弃尊严跪舔，这个时候女生就会觉得：“我说什么就是什么，我让你干什么你就会干什么，那我要你这个男朋友有什么用？”

**第五步：保持投入的平衡**
男女的关系就像是一场博弈。男女之间有一个天平，天平的两端是男生和女生投入的砝码。想要感情健康，这个天平就一定要稳。
当女生投入少的时候，正确的做法是**往外抽**，而不是**往里加**。因为你的砝码本来就多，你又往上加，只会导致天平的失衡。所以女生躲着你的时候，你去找她周围的朋友，就只会让她感觉到更加反感。

## 核心问题与总结

你可以想象一下，当一个男人跪在地上，去舔一个女人的脚趾时，这个女生脸上会是什么样的表情？你觉得这个女生会喜欢这样的男人吗？她只会觉得这个男生在自欺自贱，没有价值。

到这里我们就可以发现，造成跪舔的根本原因就在于，男生在刚开始的时候，心态就没有摆正。如果你面对女生的时候，是以一种**仰视**、**厚此薄彼**的心态去看待，把女生抬高了，你的姿态自然就放低了。而这种不平等，女生是能感觉到的。她会认为：“既然你会这样做，说明你确实不如我。”因此，你在她心目中的价值感就会打折。

进入恋爱关系以后，你又害怕失去女生，于是不停地给“可得性”，甚至没有原则、没有底线。女生一看你这样，心里就更清楚了：“我在这个男生面前，可以随便闹，反正他也不会离开我。”结果就会导致你完全被女生控制，女生也不会在意你的感受。你自己付出得再多，也毫无幸福感可言。

最可惜的是，因为前期为女生投入太多，天平严重失衡，所以女生离开的时候，你会觉得不舍得，会放下尊严苦苦相求。不管最后能否复合，如果你依然跪舔，也不会得到你真正要的爱和尊重。

**本节课总结：**
在本节课中，我们一起学习了“跪舔式”爱情的成因与破解之道。

1.  **关系如博弈**：男女关系就像是在博弈，两人之间有一个天平。当女生投入少时，正确的做法是**往外抽**（减少你的过度投入），而不是**往里加**（想尽办法对女生更好）。
2.  **管理可得性**：当女生安全感过多时，要适当地减少你对她的可得性，以维持你的吸引力和关系的平衡。
3.  **识别与避免误区**：跪舔是我们情感中比较容易遇到的一个误区，走进去你就会非常痛苦。学会正确的、平等的恋爱方式，才是能彻底摆脱跪舔的良药。
