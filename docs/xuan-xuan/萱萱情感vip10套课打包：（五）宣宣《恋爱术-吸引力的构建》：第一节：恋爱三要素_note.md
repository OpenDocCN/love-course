# 恋爱术课程（五）：第一节：恋爱三要素 🧩

在本节课中，我们将要学习构建吸引力的核心框架。我们将深入探讨决定恋爱关系成败的三个基本要素：吸引力、可得性与投入度。理解这三者之间的关系，是掌握恋爱方法的第一步。

上一节我们介绍了课程的整体目标，本节中我们来看看构成恋爱关系的三个核心要素。

任何一段感情都离不开三个基本要素：**吸引力**、**可得性**与**投入度**。我们课程中所有的恋爱理论与技巧，都是围绕着这三个要素展开的。

## 1. 吸引力：情绪的反应，而非理智的选择 💘

首先讲吸引力。男女交往中80%的失败都源自于糟糕的第一印象。因此，在任何阶段，吸引都至关重要。

吸引包含着复杂的心理内核，它并非理智层面的选择。我们无法通过摆出条件、讲道理来说服一个人喜欢我们。**吸引是一种情绪反应**，与理智无关。

**吸引力 = 情绪反应**

这就像看恐怖片会恐惧，看到可爱的狗狗会产生爱心一样。你无法命令自己停止疼痛，同样，你也不能用理智去说服一个女人爱上你。女生可能会发誓要找一个老实人，但下一分钟依然会被“渣男”吸引，引导她做出决定的就是这种情绪层面的吸引力。

吸引力体现在一个男性所具备的价值，是否匹配目标女生的**个性化需求**。这个时代男性容易陷入的误区，是把价值简单理解为有车、有房、有票子。然而，每个女性都是独立的个体，需求各不相同。

除了个性化需求，绝大部分女性还存在一些**共性需求**。以下是满足这些共性需求的吸引力关键点：

*   上进心与社交智慧
*   体贴、正直、自律、礼貌、整洁
*   力量感、冒险精神、感染力

成为一个能够满足大部分女性共性需求的男人，是构建魅力的基础。针对这些需求，我们总结了一系列“吸引力开关”，这将是后续课程的重点。

## 2. 可得性：关系可能性的天平 ⚖️

接下来讲可得性。单纯用“吸引”这个概念并不能完全解释两性关系。从进化心理学角度看，对可得性的考虑保障了人类物种的平衡。

**可得性**指的是女性认为有多大机会能够得到你。女性遇到心仪对象时，会自问两个问题：“我们合适吗？”和“他真的喜欢我吗？”。如果答案都是肯定的，那么你的可得性就高。

不管一个男人多优秀，如果女性感觉毫无发展可能，便会本能地阻止自己进入恋爱模式。因此，除了吸引力，一些围绕可得性构建的品质也很有魅力，例如忠诚、慷慨、坦率、善于赞美。

吸引力与可得性犹如天平的两端，处于此消彼长的关系中。

**关系平衡 ≈ 吸引力 ↔ 可得性**

吸引力过高而可得性太低，女生会觉得你高不可攀；可得性过高而吸引力不足，女生则会兴趣缺缺。大部分男性常犯的错误，就是在刚认识时过分殷勤、频繁示好（例如“表白死”），这相当于在缺乏吸引力时，给可得性一端加上了过重的砝码，导致关系迅速失衡。

## 3. 投入度：决定情感体验的催化剂 🔄

最后我们探讨投入度。它指的是女性为你投入资源的多少，包括有形（时间、金钱）和无形（情感、精力）资源。

传统的追求模式（男人追，女人跑）往往行不通。许多男人认为单向付出后，女性就应该“投桃报李”，但事实并非如此。

投入度决定了女性的情感体验。美国心理学家卡瑞尔·鲁斯布尔特提出的“承诺投入模式”指出，人类在做情感抉择时，并非总是选择价值最高的选项，而往往会选择自己**曾经付出最多**的那一个。

**情感依赖度 ∝ 投入量**

这与经济学中的“沉没成本”概念类似。我们更愿意守护自己付出努力获得的成果。投入是一段关系最重要的催化剂；为关系投入越多，就越难以割舍。

一个女性对感情的情感体验，更多地是由她在这段关系中的**投入量**决定的，而非仅仅从男性身上获得了多少。当她为你投入了时间、情感或精力后，她的潜意识会形成一种自我说服的逻辑：“我之所以为你付出，是因为我喜欢你。”

---

本节课中我们一起学习了恋爱关系的三个核心要素：**吸引力**、**可得性**和**投入度**。吸引力是引发兴趣的情绪开关，可得性是衡量关系可能性的天平，而投入度则是让感情深化和稳固的催化剂。理解这三者的动态平衡关系，是摆脱盲目付出、建立健康互恋关系的关键。在接下来的课程中，我们将逐一深入，学习如何具体构建和运用这些要素。
