# 宣宣聊天课程《这么聊，更吸引她》：8. 建立聊天中的联系感与舒适度 💬

在本节课中，我们将要学习如何在聊天中建立“联系感”与“舒适度”。这是让双方感觉彼此熟悉、相处自然的关键，也是关系能够顺利推进的基础。

很多男生可能对这两个概念并不清楚。所谓联系感和舒适度，就是指两人彼此不觉得陌生，聊天和相处都感到比较舒服的状态。

## 什么是联系感与舒适度？

与我们聊天的人，其身份应是朋友而非陌生人。有些男生刚加到女生微信时，就表现得像已经很熟一样问东问西。这会让女生感到奇怪，甚至反问“我跟你很熟吗？”。这就是聊天中未能建立联系感和舒适度的典型表现。

没有这两样东西，关系很难向前推进。那么，如何判断联系感和舒适度不足呢？

以下是联系感不足的表现：
*   当你尝试邀约时，对方回应“我们又不是很熟”或“不想和陌生人出去”。

为了解决这个问题，你可以说：“世界上没有陌生人，只有还没认识的朋友。” 这句话能将你们的关系从“陌生人”拉近到“朋友”，无形中增加了联系感。

再比如，你们在微信上聊得很开心，但见面后却感到尴尬。当你想靠近或有轻微肢体接触时，对方显得防备并躲开。这就代表你们见面时的舒适感尚未建立。

要知道，尴尬会摧毁相处的舒适度。如果女生觉得和你相处不舒服，后续可能就不会再愿意出来，更谈不上发生任何亲密行为了。

## 如何增加联系感？🔗

有一个很简单的方式：寻找共同话题。这能迅速拉近彼此距离。

例如，你们都爱看《跑男》，都喜欢某部小说，都爱吃火锅，或是共同欣赏某个明星。总之，找到让对方感兴趣的话题。这需要男生有一颗善于发现的心，去留意女生的兴趣爱好。

当你聊到她感兴趣的话题时，她会感觉你非常亲切，和自己有很多共同点。这时，初步的联系感就已经建立了。

萱萱推荐几个非常实用的、能增加联系感的话题类别：
*   **与女生相关的事物**：例如她的学校、爱好、朋友、昵称等。
*   **美食**：大部分女生都是“吃货”。
*   **影视综艺**：看美剧、韩剧或综艺节目的女生非常多。
*   **时尚潮流**：新潮的饰品或衣服。
*   **浪漫事物**：每个女生都对浪漫有幻想，这也是很多人看到婚纱就会幻想自己当新娘的原因。
*   **八卦与小道消息**：女生的好奇心通常特别强。

## 如何加深联系感？➡️

上一节我们介绍了通过共同话题建立初步联系。但刚才提到的多是与兴趣爱好相关的话题，与实际行动无关。为了加深联系感，你需要将话题引导到你们两人可能共同参与的实际行动上来。

例如，你可以给她介绍一个夜景很美的地方。你可以这样对女生说：“我最近发现一个地方夜景特别棒，在城市里晚上居然能听到小桥流水和青蛙叫。晚上去坐坐，看看星空，感觉太棒了。有时间的话，咱们可以一起去看夜景，到那边喝咖啡，感觉特别好。”

或者说：“如果你先到的话，我会从后面出其不意地抱你一下。” 这样的话语能让她想象你们见面时的画面，让她觉得两人是会见面的，而且见面的过程美好而有趣。

这能为接下来的正式邀约做非常好的铺垫，让邀约的成功率直线上升。

## 课程总结 📝

本节课中，我们一起学习了建立联系感与舒适度的重要性及方法。

最后总结今天课程的两个核心要点：
*   **第一，联系感**是让女生觉得你们是有联系的、是朋友，而不是陌生人。她和你出来是正常的。如果她一直视你为陌生人，关系将无法推进。
*   **第二，萱萱建议**，在没有足够联系感的情况下，不要盲目进行邀约；在没有足够舒适度的情况下，也不要盲目尝试肢体接触。认识一个女生后，要做的第一件事，就是让她觉得你们的关系并不陌生，而是朋友。
