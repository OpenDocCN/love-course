# 萱萱情感vip10套课打包：（四）宣宣《搭讪的艺术》：第一节 搭讪目的，为什么搭讪到

在本节课中，我们将要学习搭讪的核心目的。许多男生在成功获得女生联系方式后，却发现后续聊天困难重重。本节将深入分析这一现象的根本原因，并明确搭讪的真正目标。

很多男生在搭讪时会发现一个问题：搭讪似乎只是为了获取一个微信账号。他们一天中可以收集到许多微信号，但回家后尝试联系时，女生往往反应冷淡，甚至不记得他们是谁。这导致他们感到气馁，觉得搭讪白费力气。

萱萱鼓励大家进行搭讪，但你必须思考：我们搭讪的目的究竟是什么？

是为了克服社交焦虑？是为了积累更多异性资源？是为了给女生留下深刻印象？还是为了认识这个女生本人？你必须记住，我们搭讪的目的是**认识这个女生**，并最终实现**邀约**。

例如，你在商场遇到一个让你眼前一亮的女生，你上前搭讪并进行了不错的交流，随后你索要了她的电话号码或微信号。但你想过为什么要这个联系方式吗？你要号码的目的是为了能有机会与她继续交流。因为这次见面可能比较仓促，双方可能都有其他事务，无法长时间交谈。但如果交流愉快，你希望下次还能继续，所以需要留下联系方式以便未来联系。

然而，很多人在搭讪前根本不清楚自己的目的，给人的感觉就是单纯来要号码的。

你是否也是如此？搭讪的目的就是为了要微信号或电话号码。因此，有些学员在练习搭讪时，没说几句话就索要微信号，要到后立即离开，整个过程可能不到二三十秒。

如果你也是这样，你会发现后续在微信上与女生聊天会非常吃力。原因在于，搭讪时你与女生的实际交流时间可能只有十来秒（除去解锁手机、扫码的时间）。女生对你的了解和印象几乎为零，回家后可能已忘记你的模样。那么，你在微信上还能与她聊什么呢？面对面时留下的印象过于单薄，没有建立起联系感，你在微信上对她而言就只是一个陌生人。

因此，萱萱要告诉你：如果搭讪交流时间未超过五分钟，就不要索要女生的微信号或电话号码。因为即使要到了，也可能是徒劳。如果你真的只想要号码，建议你买一本电话黄页，上面有几万个号码。你明白了吗？

萱萱不希望你辛苦要来的微信号，最终只是联系人列表里的一个无用条目。所以，搭讪时我们应遵循一个原则：只要女生有交流意愿（即“有窗口”），不急于离开，且对我们不反感，我们就应尽量延长与她的交流时间。

延长交流时间并非单纯耗时间，而是尝试与她建立更多的**联系感**，为后续邀约打下基础。有些人喜欢在搭讪时要到号码后立即道别，总想着“以后在微信上慢慢聊，再把她约出来”。这正是没有理解我们搭讪的目的。

如果你们当天聊得开心，为什么不尝试进行**现场即时邀约**呢？你可以邀请女生到旁边的咖啡店坐坐，这样你们就有足够的时间建立联系感。因此，你不必总是先转到微信上，留待日后邀约。

我们搭讪的首要目的就是**邀约**。获得联系方式，理应是成功邀约后自然而然的一步。如果条件允许，我们就应该尝试即时邀约，不必给自己设定过多的固定流程。

有些人可能会想：我搭讪 -> 收号 -> 微信聊天 -> 微信邀约 -> 约会。但你是否想过，如果条件允许，我们能否将这五步合并为一步？即进行一次良好的沟通交流，然后直接提出邀约请求，接着就开始约会。你是否考虑过这种方式？

例如，你下班后在商业广场遇到一个姑娘，上前搭讪后她停下来与你交谈。你们聊了五六分钟，她也没有要离开的意思。这时，你可以很自然地说：“哎，你饿不饿？今天聊得很开心，要不我们在附近找个地方一起吃点东西？你喜欢西餐还是中餐？” 或者说：“晚上八点有个活动，我现在还有一个多小时，我们可以一起聊聊。如果你时间合适，旁边正好有家星巴克，我们去坐坐？你喜欢喝拿铁还是摩卡？”

这时，有的女生会答应，有的会拒绝。但即便她拒绝，这种互动建立的联系感，也远比上前只要号码、要完就走的模式要强得多。一旦你抱着即时邀约的目的去搭讪，就不会出现二三十秒就收号走人的情况。你可能会与女生有更长的交流时间。

萱萱有的学员在街上与女生聊了半个多小时，聊到最后感觉两人像久别重逢的老友，后续的邀约和聊天都非常顺利。

如果你希望搭讪成功后，女生在微信上反应良好，后续邀约成功率高，除了延长搭讪时间，你还需要给女生留下比较深刻的印象。

萱萱记得有个学员过生日时，和几个朋友喝酒撸串，隔壁桌坐了五六个人，其中有他喜欢的一个女生。他上前搭讪时，那桌人有的起哄，有的开玩笑说“你把这瓶酒喝了，号码就是你的”。两桌人因此有了良好的互动。学员喝了一瓶酒，正要喝第二瓶时，那位女生说“不用了，他们开玩笑的”。两人由此开始交流，并互留了联系方式。整个气氛非常好。后来，无论是在微信聊天还是线下约会时，他们常常回忆起那晚的情景，两人之间的氛围一直特别好。

由此可见，在搭讪时给女生留下深刻印象，你们的联系感就会更强，后续联系也会更自然、顺畅。

搭讪作为认识女生的一种方式，效率很高。但很多男生不明白搭讪的目的是什么。我们搭讪的目的就是**邀约**，而不是收集一堆号码。如果你只是为了要号而要号，那只是把一个马路上的陌生人，变成了你微信列表里的另一个陌生人而已。

因此，在搭讪时，我们要加深女生对我们的印象，尽量延长交流时间，充分建立联系感，从而增加邀约成功的几率，这才是最重要的事。

当然，这一切都有一个前提：要在女生有交流意愿（有窗口）的情况下进行。一定要记住，邀约不一定要放到下一次。如果搭讪时女生窗口很大，你就可以及时邀约。要号码是为了后续能联系到她。如果第一次交流感觉非常好，她会希望继续与你保持联系，自然也就愿意把联系方式给你了。
