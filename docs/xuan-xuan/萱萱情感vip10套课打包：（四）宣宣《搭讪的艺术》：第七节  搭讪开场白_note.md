# 萱萱情感VIP课程（四）：第七节 🎯 搭讪开场白

在本节课中，我们将要学习搭讪的核心第一步：如何设计一个有效的开场白。开场白是破冰的关键，它决定了后续交流能否顺利展开。我们将探讨两种主要的开场白形式，并学习如何根据实际情况灵活运用。

## 开场白的目的与重要性

你有没有遇到过这样的情况：看到不远处坐着一个很漂亮的女生，正好也是一个人。你觉得这是一个很好的搭讪时机，于是你鼓足了勇气，做了半天思想工作，结果憋了半天，竟然还是不知道要说些什么。

搭讪需要一个开场白。其实这个开场白的目的是和女生说出第一句话以破冰。开场白不必说什么特殊的套路，说什么其实都无所谓，很自然地做一个开头就可以了，哪怕是一句“你好”。

## 开场白的两种形式

开场白有两种形式：一种是直接开场白，一种是间接开场白。

### 直接开场白

直接开场白就是你一上来就对女生表达出，你想要跟她结识的意愿。你需要在表达完自己的意愿之后，重点去观察女生的“窗口”和情绪，根据她的窗口和情绪去判断下一步的行动。

在开场白当中，你需要传递出几层意思。

以下是直接开场白需要包含的三个核心要素：

1.  **表明来意**：在你说完“你好”之后，首先要说明你为什么来搭讪，你想要做什么。
2.  **建立安全感**：打消女生在安全方面的顾虑，让女生能够在一个放松的气氛下跟你交流。
3.  **建立联系感**：你要通过延伸更多的话题，让女生了解更多的你，你也更多地去了解女生，建立初步的联系感。

当你这三点意思表达出来之后，你说什么其实都不重要。最重要的是在说的过程当中，观察女孩子的表情、她的肢体语言、她的状态、她的“窗口”，对你是接纳的还是排斥的，或者说是犹豫不决的。

**示例**：
> “我刚才在那边看到你，我觉得你气质很好，走过来想跟你认识一下。对了，你怎么称呼？我叫XXX。”

多数情况下，女孩子面对一个陌生男人的搭讪时，会显得犹豫不决，但她不会明显地拒绝，或者是很明显地接纳。当女孩子在犹豫的时候，其实要做的就是坚持。

这里还是要强调一点，作为男生，应该更加主动地去迈出第一步，做到胆大心细脸皮厚。所以你可以不停的和女生说话、去交流，提供更多的谈资和话题。

一定要注意，在和陌生女生交流前期，其实**90%的话题是由你制造的**，而**10%的话是由女孩子给出的**，甚至更少。只要你在跟她交流的时候，她愿意回答你，其实就是一个很好的“窗口”。甚至说她没有回答你，但是她并没有走开，那就代表著你可以继续。

虽然说有的女孩子在你跟她要联系方式的时候，她会说“不用了吧”。其实这个时候你可以再问她其他问题，如果她还是在回答，那就说明她是愿意跟你交流的。那她说这个“不用了吧”，看似是一种拒绝，其实是一种欲拒还迎。

我们都知道女孩子害羞、内敛、矜持。那她如果说“不用了吧”，但是她又不走开，她继续回答你的问题、延续你的话题，就说明其实她内心是接纳你的。那通常只需要再坚持一段时间，就可以拿到这个女生的联系方式，或者说跟她有一个比较好的交流。

### 间接开场白

还有一种开场白是间接开场白。它是指利用一些特定的内容而引起的话题。

比如，你在公园里，看到一个女生在遛狗。你可以走过去跟那只小狗玩，然后你可以问她：“这是什么狗？这狗多大了？叫什么名字呀？”这样你们就很自然地能够搭上话了。

## 如何选择开场白方式？

我们在搭讪的时候，是用直接开场白呢，还是用间接开场白呢？

萱萱的建议是，通常**90%的时间使用直接开场白**，能够更加高效。你直接走上去说：“嗨，请留步一下。”或者说：“请留步，我刚刚在那边喝咖啡/等朋友/打电话，觉得你气质很好，所以想来跟你打招呼。对了，我叫XXX，请问你怎么称呼？”

过了这几句话，初始的开场阶段就已经结束了。之后我们就通过识别女生的这个“窗口”、她的反应和情绪来判断，我们是继续呢，还是有礼貌地退出。

而间接开场白，会受很多场合和环境的限制，它的效率不如直接开场白高。比如，当你在马路上，或者是在一个特定的场合时，没有很好的情景去做一个间接开场白。或者当你根据情景想出来一个间接开场白的时候，人家已经走掉了，场景已经不匹配了。所以，间接开场白受外界的因素影响很大。

## 比语言更重要的因素：第一印象

不过你也要知道，搭讪是遵循“7-38-55定律”的。

**7-38-55定律**：
*   **7%** 的信息来自我们嘴巴里说出来的**内容**。
*   **38%** 的信息来自我们的**语音、语调、语速、语顿**（即我们用什么样的状态去说）。
*   **55%** 的信息取决于我们的**穿衣打扮、眼神表情、言谈举止、气场气质**等。

所以，你用什么样的开场白，其实并不是特别重要，它不能够去决定你搭讪的结果。真正重要的是，你给别人留下了一个什么样的**第一印象**，这个第一印象是好是坏。

## 开场后的关键一步：自我介绍

我们在进行完开场之后，通常还要做一下自我介绍。自我介绍的目的是为了让女生留下一个深刻的印象，给她安全感，让她回去之后，甚至几天里面还能够记得你。

如果你给女生介绍，没有给她留下深刻的印象，那她也许很快就会把你忘掉，你也很难再去跟她建立联系感，要到一个号码，也就是一个“废号”。

这里其实有一个很简单的办法，就是说你在介绍自己的时候，加一个生动的小故事。故事不用太长，但是可以帮助女生去记住你就可以。

**公式**：
> **自我介绍 = 姓名 + 一个与姓名/个人特质相关的简短生动故事**
