# 聊天技巧课（三）：如何通过聊天建立吸引力 💬

在本节课中，我们将学习如何在与女生聊天时有效展示自身价值，从而建立吸引力，让她愿意持续与你交流。

## 概述

与女生认识后，她会通过聊天不断了解你。她主要想了解你是否有价值。尤其在认识初期，女生几乎完全依靠聊天来判断是否有必要继续交流。如果你能展现高价值，女生会更热情地回应；否则她不会浪费时间。

因此，聊天时需要展现高价值。聊天中的价值展示通常分为三种情况。

## 低价值展示

低价值展示指在聊天中暴露自己的缺点或价值较低的一面。你或许认为自己不会这样做，但聊天不顺畅很可能正是因为无意中做了低价值展示。

例如，女生说你在银行工作是公务员，你回答“银行不算公务员，我已经沦落到金融民工了”。这句话就是低价值展示。你本意可能是谦虚或调侃，但此时谦虚并不合适。我们费心聊天不是为了表现谦虚，而是为了体现高价值。

这种情况下，建议你这样说：“我在银行工作，确实比较稳定。但我觉得年轻时应更积极，多做事为将来打基础。我的目标是明年升任主管，我认为可以实现。”这样回答感觉截然不同。你加入了对未来的展望，将时间延伸到明年，女生会觉得你是有想法、不甘平庸的人。她会因你的自信而抱有更多期待。当女生对你有好感时，她会对你有想象空间，从而在心中放大你的优点。

## 高价值展示

高价值展示的重点是自然。女生更愿意相信间接的高价值展示，她会觉得这是自己观察到的，从而证明你的高价值是真实的。

例如，结账时掏钱包，无意中带出车钥匙或露出袖口的品牌标志。或者，给女生倒水时她无意中看到。又或者，女生提到喜欢去日本滑雪，你推荐攻略，她问你是否去过，你不经意地说：“我以前在日本上学时去过几次。”女生可能会觉得“捡到宝了”，并暗自窃喜遇到了优质男人。

高价值展示不一定要滔滔不绝，重点在于传递价值。例如，你和女生聊天时正在一家干净的高档西餐厅用餐，你拍下食物和环境的照片发给她。聊天内容不多，但一张照片就展现了高价值。女生不会觉得刻意，反而会认为你生活讲究细节、教养良好，而教养又与见识相关，从而推断你是一个价值不错的男人。这就是见微知著，通过这些细节，女生能想象你整体的状态，感知你是真正高价值还是伪装。

## 避免吹嘘

虽然不能过分谦虚，但也不能为了赢得关注而刻意吹牛。例如，你明明买不起车，却在街上对别人的车评头论足：“这车好看但不值钱，也就三十来万。这车不实用，不如添点钱买奥迪。”你想显示自己懂车，但女生会觉得你更屌丝。吹嘘会让人感到不舒服，破坏吸引力。

## 总结

本节课我们一起学习了如何在聊天中通过高价值展示建立吸引力。与女生认识后，需在聊天中自然展示高价值，让她更深入了解你并愿意继续交流。男生聊天通常有三种情况：低价值展示、高价值展示和吹嘘。过分谦虚会显得低价值，而吹嘘则会让人反感。真正的高价值展示是在自然状态下不卑不亢地表现态度，让女生感觉你本就是如此，而非刻意伪装。优秀应成为一种习惯，自信应成为常态。当女生相信你是高价值的男生，她自然更愿意与你深入交流。
