## 懒人撩妹基本功2：克服拖延与培养幽默感

在本节课中，我们将要学习如何识别并克服阻碍个人成长的拖延行为，以及如何通过培养真正的兴趣爱好和自信来发展幽默感，从而提升个人魅力。

### 拖延的本质与影响

上一节我们介绍了制定计划的重要性，本节中我们来看看执行计划时常见的障碍——拖延症。

拖延症是一种逃避行为，它逃避失败所带来的挫折感。这也是一种维护自恋的行为。成长就是一个破除自恋的过程，需要受到挫折并一次次锤炼自己。哲学家尼采的格言是：那些杀不死我们的，会让我们更强大。

在制定计划后，即使我们从小事开始并逐步升级难度，拖延症依然可能存在。例如，计划健身前却刷手机、点外卖，最终导致计划失败。

经济学中的**双曲贴现理论**（又称非理性折现）可以解释这种现象。该理论指出，人在选择时更倾向于选择当下的好处，而放弃未来的更大收益。公式可以简化为：**当下满足 > 未来更大收益**。

例如，选择立即获得30元，而非一年后获得100元。刷手机、吃零食能带来即时快感，就像嗑瓜子一样，每一个动作都能立即获得满足。而健身、读书等需要长期坚持才能见效的活动，则缺乏这种即时反馈，因此容易被拖延。

---

### 克服拖延的实用策略

理解了拖延的成因后，以下是应对拖延、坚持自我提升的几个有效方法。

**第一招：学会“放自己一马”**
核心是保持计划的弹性，避免因过度严苛而产生痛苦和放弃。例如，计划每天做3组俯卧撑，每组20个。如果某天状态不佳或已完成其他健康任务（如健康饮食），可以允许自己每组少做5个。强迫自己咬牙坚持，反而可能导致第二天对任务产生厌恶，难以持续。

**第二招：学会自我奖励**
通过建立正向反馈循环，让坚持变得愉悦。例如，连续5天完成甚至超额完成俯卧撑计划后，在周末奖励自己一顿美食。这样能将有益行为与愉悦体验关联起来。

**第三招：多任务并行切换**
这是一个应对偷懒非常有效的方法。同时进行几件对自己有益但不同类型的事情，可以在感到疲倦时切换任务，保持整体效率。例如，目标是成为知识渊博的人，可以同时学习哲学、文学和心理学。看哲学书累了，就换小说放松；注意力不集中时，就去练琴。这样，所做的每一件事都是有益的，能有效改善拖延。

### 培养真正的幽默感

克服拖延是为了更好地自我建设，而幽默感则是个人魅力中极具吸引力的一环。

幽默感是每个人都喜欢的性格特质，它源于智慧与有趣。幽默感是一种思维方式，而非简单地背诵网络笑话。对于人生幸福而言，幽默感非常重要。

许多人在努力奋斗的同时，可能给人留下能力不差但“无趣”的印象。“无趣”意味着单一和乏味，因此把自己变得有趣至关重要。

**如何成为一个有趣的人？关键在于建设真实的兴趣爱好。**
兴趣爱好不仅滋养灵魂，也为社交提供谈资。许多人在被问及爱好时，回答是“旅行、音乐、看剧”或“没什么爱好，就是上班玩手机”。再追问细节时，往往发现他们并未真正实践这些爱好，只是被动消费。

真正的兴趣爱好，意味着身体力行地去实践，愿意投入时间、精力，甚至终身坚持。这是一种精神追求。以“没钱”、“没时间”为借口，常常是为了逃避行动和改变的懒惰。

**案例分享：**
一位32岁、初中毕业的跑腿员学员，月收入约3000元。他是我最努力、最用心的学员之一。尽管表述能力不佳，但他每次都竭尽全力完成学习任务并反馈心得。他的故事证明，阻碍你的通常不是金钱或时间，而是**不愿意改变**的决心。永远不要找借口，关键在于你是想解决问题，还是想继续抱怨。

坚持建设兴趣爱好，体会过程中的感悟，能让你学会与自己相处，并从中滋长出智慧。

### 幽默感的根源：自信与生活状态

幽默感体现的是一个人的自信和生活智慧，是一种以柔克刚对待生活的方式。

从对待生活的态度，可以看出一个人是强大还是懦弱。越不自信的人，越害怕被看低，因此越可能用强硬的姿态伪装自己。许多表面自尊心强、怕丢面子的男生，内心往往特别自卑，时刻紧绷，难以拥有幽默感。

相反，自信的人内心自我认可，不太在意外界评价。当一个男生真正自信时，他的行为举止会大方得体，聊天时不会刻意迎合女生，能发自内心地开心。在这种放松、愉悦的状态下，各种有趣的想法会自然涌现，感染对方，这才是真正的幽默。

```python
# 伪幽默：依赖外部素材
def tell_prepared_joke(joke_from_internet):
    print(joke_from_internet)
    # 自己并不觉得有趣，气氛可能尴尬

# 真幽默：源于内在状态
if state == “confident” and state == “relaxed”:
    humor = generate_naturally() # 有趣的话自然流淌
```

因此，我们要做的不是收集笑话、把自己变成段子手。关键在于成为一个生活状态放松、富有幽默感的人。情绪的感染力才是核心。

### 爱的能力：从自恋到自爱

无论是男生还是女生，想要脱单的原始动力，常是为了摆脱孤独和恐惧，希望有人能满足我们所有需求、给予爱。

但世界的规则是：**想要获得幸福，你必须有能力给予**。所有智慧都在教导我们爱他人，但在这之前，我们必须先学会足够爱自己。

我们常常习惯向外寻求爱，但爱的行为只会发生在你内心有爱的时候。这需要经过世事锤炼，破除自恋才能达到。

*   **自恋**不等于自爱。自恋是以自我为中心，沉浸在自己的世界里索取。即使付出，也是变相地索取回报。
*   **自爱**是利用好每一分钟，不断发展自我意识，遵循世界规则，与世界和平共处，将自己的能力发挥到最大，促进自己和他人更好地生活。
