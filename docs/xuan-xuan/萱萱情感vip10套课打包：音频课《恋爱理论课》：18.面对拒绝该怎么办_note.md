# 萱萱情感VIP课程10套打包：音频课《恋爱理论课》第18讲：面对拒绝该怎么办 💔

在本节课中，我们将要学习如何正确应对恋爱中被拒绝的情况。核心在于识别并克服一种名为“思维极端化”的不良心态，这种心态是许多人感到不幸福、难以建立理想恋爱关系的主要原因。

## 思维极端化的表现

上一节我们介绍了课程的主题，本节中我们来看看什么是“思维极端化”。所谓思维极端化，是指一个人的世界观非黑即白。他只能看到事物的一面，且通常是坏的一面，而看不到好的一面。这导致他无法合理解决问题，心态失衡。

思维极端化的人总喜欢给自己下一些定性的、消极的结论，从而看低自己。

以下是思维极端化的具体表现举例：

*   例如，你搭讪女生，失败一两次，甚至十次中失败了九次，你便告诉自己：**“我一无是处，我绝对不会成功，我的人生就是失败的，毫无价值。”**
*   例如，有人分手或看到明星离婚的新闻后会说：**“我再也不相信爱情了。”** 他认为别人的感情失败意味着世界上没有真爱，却不会去想失败可能有其特殊原因。
*   例如，一个年轻人若没赚到太多钱，可能断定：**“我这辈子注定一事无成，找不到优秀的女朋友。”**
*   例如，经历感情挫折或离婚后，可能下结论：**“我是一个失败的男人，优秀女人不可能看上我。”**

这种人习惯于给自己下消极定论，感觉被生活压垮，不知前路如何。

## 自我低价值感的根源

我们之所以会形成这种“自我低价值感”，往往源于成长经历。

在成长过程中，父母和老师常会给我们贴上一些言过其实的负面标签。例如，学生时代染了头发，可能就被贴上 **“流里流气”** 的标签，尽管你可能在其他方面很规矩。

当身边人反复给我们贴上此类消极标签时，我们很容易进行自我心理暗示：“我就是他说的那种人。” 人是自己思想的产物，有什么样的思想，就容易成为什么样的人。当你把别人的看法内化到自己身上，就可能真的朝那个方向发展，或者至少会变得自卑，因为感到不被认可，也无法自我认可。

成年后，虽然父母老师不再贴标签，但自己已养成了随意给自己贴标签的习惯。用极端化思维看待世界，就会总感到不顺心、不幸福，并且越害怕失败就越容易失败。

## 如何走出思维怪圈：双向思考

那么，我们怎样才能从这种思维的怪圈里走出来呢？正确的办法是：**学会从正反两面去理解一件事情**。

再坏的事情发生，都可以尝试从另一个角度挖掘其价值。当你遭遇失败，开始自我否定时，一旦察觉到愤怒、哀伤、郁闷的情绪，就要立刻中断这种情绪，问自己两个问题：

1.  **是什么事件引起了现在的难过？**
2.  **我对这件事的理解是否全面？**

以下是运用双向思考的案例：

*   有人破产了，觉得自己几十年奋斗得来的事业、财产瞬间归零，人生一无所有，于是选择自杀。他失去的确实是物质，但他并非真的一无所有。他多年积累的**经验、成长、思维方式、朋友关系、家庭亲情**这些无形资产，是别人拿不走且更珍贵的。但他思维极端化，只看到失去，看不到拥有，导致心态失衡。

有一句话说得特别好：**人生除生死，无大事。** 如果你学会从正反两面理解事情，就不至于产生如此极端的想法。人生的阴影，恰恰让生命变得更立体和丰满。

## 在爱情中应用：面对拒绝

因此，在爱情中面对拒绝时，你可能会难过和消极。但请捕捉到这种心态，然后对自己说：**“接受那些不好，才能获得更好。他的拒绝是为了让我变得更好。”** 你可以通过这次拒绝来提升和修炼自己。

---

本节课中我们一起学习了如何应对拒绝。最后总结一下核心要点：

1.  **思维极端化是幸福的主要障碍**：许多男生找不到幸福，主要是因为思维容易极端化，只看事情坏的一面。
2.  **思想决定人生**：人是自己思想的产物，**你有什么样的思想，就会成为什么样的人**。
3.  **在失败中成长**：爱情失败时要调整心态，从中总结经验。要知道，**人都是在关系中成长的**。
