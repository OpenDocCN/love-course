# 课程六：把控女生的心理 🧠 - 第二节：女生对于可得性的需求 📊

在本节课中，我们将要学习一个核心概念：**可得性**。我们将探讨它在两性吸引中的关键作用，以及它如何与吸引力相互作用，最终影响女生的选择。

## 生物界的启示 🦁

上一节我们介绍了吸引的重要性，本节中我们来看看**可得性**又是怎么回事。

在生物界，很多哺乳动物都是雄性比雌性好看。例如，雄狮脖子周围长着鬃毛，显得很雄伟，相比之下雌狮则普通很多。鸟类中，很多雄性的羽毛也比雌性艳丽。能开屏的是雄孔雀，雌性孔雀的尾巴是秃的，并不好看。

## 人类的特殊现象 👩❤️👨

而我们人类却恰恰相反，女性通常比男性更注重外表。女性有更光滑的皮肤、柔美的身材、丰满的胸部和纤细的腰肢。女性还喜欢喷香水、涂口红，无论走到哪里都是一道风景线。男性则不同，大部分男性不爱收拾自己，甚至可能鄙视那些会收拾自己的男性。

在动物世界和人类世界，雄性扮演的都是追求者的角色。按理说，雄性更应该“花枝招展”以吸引异性，但事实并非如此。雄狮的鬃毛和孔雀开屏，都是为了吸引雌性而在进化中被保留下来。而女性作为被追求者，反而进化出了许多吸引男性的特征。

## 进化背后的逻辑 🤔

这种现象非但没有违反大自然的进化规律，反而是顺应了自然。原因在于我们人类的大脑。人类进化出的大脑帮助我们发展出现代文明，但也给女性带来了沉重的负担。

人类的婴儿本质上都是“早产儿”。刚生下来的婴儿大脑尚未发育完全，如果等到发育完全，就无法通过狭小的盆腔出生。因此，婴儿出生后需要在母体外继续发育。婴儿在三岁前走不稳，六岁前基本没有生存能力，必须由母亲抚养。而其他动物生下来几分钟后就能站立、喝水。

因此，女性需要照顾孩子，自身的生存能力相对减弱。她需要有人照顾她和孩子，持续为她提供生存价值，例如食物、保护和安全感。这样她才能专心抚养孩子。

为此，女性进化出许多有利于吸引男性的特征，例如凹凸有致的身材和迷人的脸蛋，以吸引男性持续提供生存价值，共同完成生育这件大事。

## 隐性排卵的奥秘 🕵️♀️

除此之外，女性还进化出一个其他动物都没有的特征：**隐性排卵**。

只有在排卵期的雌性才具备生育条件。一般动物在排卵期会有明显表现，例如雌性大猩猩排卵时屁股会特别红，雄性就能判断哪一只在发情期。但人类不同，排卵期的女性和平常没有区别，没有任何特别反应，而且排卵期不确定，女性自己也不知道何时排卵。人类的隐性排卵是最近一两百年才被发现的。

隐性排卵有特别的作用。如果一个男性想和女性生孩子，但他不知道女性何时排卵、能否怀孕，并且还需要防止其他男性介入，那么他只能一直留在女性身边，持续发生关系并提供生存资源，直到“开花结果”。如果中途放弃或离开，女性可能没有怀孕，或生下别人的孩子。女性利用隐性排卵这一点来“绑住”男性。

## 可得性的定义与重要性 ⚖️

考虑到要选择一个能长期陪伴的男性，女性在选择时就不单只考虑男性的价值是否高，还要考虑他是否愿意持续提供价值，不会中途离开。因此，女性也会考量男性是否具备担当和责任感的内在品质。

一个男性愿意在女性抚养孩子的过程中，保证她有足够的生存能力，这个意愿就叫做男性的**可得性**。

有些男性可能会说：“照你这么说，我的可得性很高，女生应该愿意跟我在一起才对。”这就涉及到了**可得性**和**需求感**的区别。

从字面意义可以看出，**可得性**就是可以被女生得到的概率，而**需求感**则是对女生的需求度。可得性和需求感其实都是对女性示好的行为，但为了更好地区分，我们分别称之为可得性和需求感。

## 吸引力、可得性与需求感的关系 📈

以下是理解这三个概念关系的关键点：

吸引力是可得性和需求感之间的分水岭。**产生了吸引就是可得性，没有吸引就是需求感**。它们是一对密不可分的概念，关系成反比。

*   **吸引力高，可得性低**：女生很喜欢你，但你很高冷，没有给她希望，那么女生会在本能上阻止自己跟你进入恋爱模式。
*   **吸引力低，可得性高**：女生压根不喜欢你，你去找她、烦她、跪舔她，只会让她厌烦。

所以，我们需要用恰当的吸引，再具备良好的可得性，这样女性才会感受到一种平衡和舒适感，才会想要长久地在一起。

## 如何平衡吸引与可得性？ 🎯

最终决定女生选择的是**吸引力**和**可得性**的均值。

假设你的竞争对手吸引力是9分，但可得性只有1分，那么均值只有5分。而你的吸引力可能是6分，可得性也有6分，那么你的平均分是6分。女生在你和另外一个吸引力很强的高富帅之间做选择时，可能会选择你。

这就是为什么在很多情况下，可得性强的男人也有可能战胜吸引力强的男人。

在追求女生的过程中，我们始终要去维系吸引力与可得性之间的关系，而且要明白：**可得性是建立在吸引的基础之上**。如果没有吸引，可得性也无从谈起，它就变成了一种需求感。而需求感不会让我们获得喜欢的女孩子。
