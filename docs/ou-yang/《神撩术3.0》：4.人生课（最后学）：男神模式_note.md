## 一、基础：形象与展示面 🏗️

所有高级技巧都建立在一个坚实的基础之上。对于男神模式而言，这个基础就是你的**形象**和**展示面**。

形象是可以改变和提升的。你需要将自己塑造成一个整洁、有型的人。展示面则是在社交媒体等平台上，能够吸引女性的高质量生活展示。这两点是开启男神模式的第一步。如果一个形象邋遢、展示面糟糕的人自称男神，只会引人发笑。因此，在尝试任何技巧前，请务必先做好你的形象和展示面。

> **核心公式**：男神模式 = **形象** + **展示面** + 核心思维

如果你对如何提升形象和打造展示面还不清楚，请回顾本课程的第一课和第二课。

## 二、核心：以自我为中心的三要素 🎯

上一节我们介绍了男神模式的基础，本节中我们来看看其核心思维：**一切围绕自我展开**。这意味着，在两性关系中，你永远是主角。这个核心思维包含三个要素：信念、心态和思维。

以下是这三个要素的详细说明：

2.  **心态**
    你的社交位阶永远略高于女性。基于此，你对她的态度应该是自上而下的、带有引导性的。就像长辈对小孩说“乖一点，我就对你好”一样。这种态度会让女性感受到你的高价值，从而激发她的追逐欲。

3.  **思维**

总而言之，信念、心态和思维共同构建了一个强大的、以自我为中心的认知框架。女性能够敏锐地感知到你的这种框架，并因此被吸引，开始反追逐你。

## 三、行为：男神的四大准则 ⚙️

知道了思维，还需要付诸行动。以下是男神必须遵守的四大行为准则，任何一条的缺失都会让你前功尽弃。

以下是具体的行为准则列表：

1.  **控制自己，绝不追逐**
    受传统观念影响，很多人认为追求女性就要讨好、付出。但在男神模式中，这是大忌。你的行为必须是**让女人来追你**。一旦你开始跪舔、送礼、无条件请客，就等于交出了关系主动权，将自己置于低人一等的位置。正确的做法是保持平衡：如果请她吃饭，可以让她请喝东西。让她明白，想得到你的付出，她也需要付出代价。

2.  **设立并坚持标准**
    你不是一个“是个女人都想上”的饥渴者。你必须有自己的择偶标准，并尊重它。标准应该是对方可以努力达到的，例如“我喜欢会做饭的女生”、“我欣赏乐观向上的女孩”。当对方不符合你的标准时，要明确告知，而不是降低标准去迎合她。例如，问她：“你会做饭吗？”她说不会，你可以回复：“那太可惜了，我只喜欢会做饭的女生。” 真正喜欢你的女生，会为了迎合你的标准而去改变。

3.  **给予认可，而非寻求认可**
    许多男人会犯一个致命错误：向女人寻求认可。例如问“你觉得我帅吗？”、“你喜欢我吗？”。这等于把评判权交给了对方，是低价值的行为。男神应该做的是**给予对方认可**。因为你的社交位阶更高，认可是由上至下的。例如，你可以说：“我欣赏你独立奋斗的样子”、“我喜欢你的眼睛，很亮”。这会让对方感到被高价值的人欣赏，从而更想赢得你的持续认可。

4.  **保持一致性**
    你不能只在网上聊天时是男神，见面或关系深入后又变回屌丝。也不能前期很有态度，后期又开始跪舔。**一致性**至关重要。女性会持续观察你的行为。如果你前后不一，她就会认为你之前的形象是伪装的，从而对你失去兴趣。男神模式是一种生活态度，需要贯穿始终。

## 四、应对挑战：破解废物测试 🛡️

当你运用男神模式时，必然会遇到女性的测试（废物测试），她们以此来检验你是否是真正的“高价值男人”。应对所有测试，你只需要掌握两种方法，无需背诵复杂惯例。

以下是两种核心破解方法：

1.  **反问**
    不直接回答她的问题，而是把问题抛回给她。这能打乱她的测试节奏，并展示你的框架。
    *   **测试示例**：“你是不是经常这样撩妹子？”
    *   **反问破解**：“你是不是经常这样撩汉子？”

2.  **保持自我**
    不理会她问题的表面含义，直接表达自己真实的想法和标准，展现强大的自我中心框架。
    *   **测试示例**：“你是不是经常这样撩妹子？”
    *   **保持自我破解**：“我只撩我喜欢的女生。”
    *   **另一测试示例**：“我觉得你挺花心的。”
    *   **保持自我破解**：“在遇到真爱之前，每个人都是‘花心’的。”

记住，女性更关心你回答问题时的**态度和框架**，而不是答案本身。运用以上两种方法，你可以轻松化解绝大多数测试。
