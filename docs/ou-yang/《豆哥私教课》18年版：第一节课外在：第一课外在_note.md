# 《豆哥私教课》18年版：第一节课 外在：第一课 外在 👔

在本节课中，我们将要学习个人形象管理的基础知识。形象非常重要，正如浮夸老师所说，没有人会通过你丑陋的外表去发现你内心的善良。一个好的形象，其核心是一种生活态度，而不仅仅是特定场合的装扮。我们将从发型和护肤两个基础方面入手，帮助你建立清爽、得体的日常形象。

## 核心理念：形象是一种生活态度 💡

一个好的形象不是指你今天穿得帅，明天就不帅了。也不是说你今天不帅，明天又突然变帅。我认为，好形象是一种生活态度。

这意味着，无论你是出门买菜、吃饭、上课还是上班，你的头发和穿着都应保持整洁得体的状态。这不是说今天要去和女生约会，就特意打扮得特别帅、特别“骚”。好的形象是你每天出门前，都会花点时间整理自己发型和仪表的一种习惯。

上一节我们明确了形象的本质是一种持之以恒的生活态度，接下来，我们具体看看如何从“头”开始打理。

## 发型篇：打造清爽阳光感 ✂️

我们来看几种常见的发型。以下是几种发型的共同特点分析：

1.  **“不会犯错”的短发**：这种发型的特点是**两边剃短、剃干净，中间的头发用发蜡或定型喷雾立起来**。这种发型让人感觉爽朗、阳光、干净，适合大多数脸型，是安全的选择。
2.  **有刘海的发型**：如果你脸型偏长（尖型脸），留有刘海可能很帅。但关键在于，**刘海必须用发蜡或定型产品固定好，不能凌乱**。凌乱的刘海会让人看起来没有精神。
3.  **根据脸型调整**：如果你是方脸（国字脸），像讲师一样，那么“两边剃平，中间立起”的PVA头（类似Undercut）是很好的选择。如果你是长脸且适合刘海，务必做好定型。
4.  **日常打理是必须的**：睡觉后头发肯定会乱，所以**出门前必须进行打理**。没有人的发型能睡醒后直接保持完美，这就像女生如果带妆睡觉不伤皮肤，她们也会带妆睡觉一样，需要日常维护。

关于去哪里理发和频率，以下是具体建议：

*   **选择理发店**：去你所在城市里**连锁店最多的发型屋**。例如在广州，可以选择“八百伴”（此处为音译，可能指某连锁品牌）。理发师可能会推销染烫服务，请坚持只修剪。
*   **核心要求**：向理发师明确提出“**剃两边**”，如果需要保留刘海，则强调要“**定型**”。
*   **修剪频率**：**每20到25天**需要修剪一次头发，不要超过30天。定期修剪能始终保持精神面貌，避免头发过长显得邋遢。

掌握了发型的基本法则后，干净的面部是清爽形象的另一个基石。下面我们进入护肤环节。

## 护肤篇：了解肤质，基础清洁与保湿 🧴

首先，你需要判断自己的肌肤类型。最常见的肌肤类型有四种：

1.  **油性肌肤**：面部容易出油，看起来有油光。
2.  **干性肌肤**：面部缺乏油脂，皮肤干燥。
3.  **混合性肌肤**：只有 **T字区（额头和鼻子）** 出油，其他部位偏干。
4.  **敏感性肌肤**：皮肤容易发红、受刺激，例如冬天洗脸后整张脸发红。

**判断方法**：观察洗脸后2小时的面部状况。如果全脸泛油光，是油性；如果全脸紧绷干燥，是干性；如果仅T区出油，是混合性；如果使用护肤品容易发红，是敏感性。

了解肤质后，选择产品至关重要。以下是基础护肤建议：

*   **洗面奶选择**：**必须根据你的肤质购买相应的洗面奶**，不存在一款产品适合所有肤质的情况。例如，油性肌肤需选择控油型，干性肌肤需选择滋润型，敏感性肌肤需选择温和、天然成分的产品。
*   **面膜护理**：建议每周敷**两次**保湿面膜，特别是含有玻尿酸成分的面膜，可以有效补水。讲师推荐了“生甜”和另一个品牌（原音频不清）。优先选择保湿功能的面膜，照顾好皮肤是提升整体形象的前提。

## 总结 📝

本节课中我们一起学习了个人形象管理的基础知识。

我们首先明确了**好形象是一种持之以恒的生活态度**，而非临时抱佛脚。接着，在发型部分，我们学习了打造清爽发型的要点：**两边剃短、中间立体、刘海定型**，并掌握了**每20-25天修剪一次**的维护频率。在护肤部分，我们学会了先**判断自己的肤质（油性、干性、混合性、敏感性）**，再**根据肤质选择对应的洗面奶**，并建议每周进行**两次保湿面膜**护理。

记住，打理好自己，是获得他人良好第一印象，也是尊重自己生活的开始。从今天起，每天花一点时间在发型和面部清洁上，让它成为你的习惯。
