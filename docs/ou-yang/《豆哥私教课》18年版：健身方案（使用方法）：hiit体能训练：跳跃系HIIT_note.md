# 《豆哥私教课》第18课：跳跃系HIIT训练方案 🏃‍♂️💨

在本节课中，我们将学习一套以跳跃动作为核心的高强度间歇训练方案。这套方案旨在高效提升心肺功能与全身力量，适合希望改善体能、燃烧脂肪的健身爱好者。

## 概述

跳跃系HIIT是一种结合了多种爆发性跳跃动作的高强度间歇训练。其核心在于短时间内进行极高强度的运动，配合短暂的休息，以最大化训练效率。训练模式通常遵循 **工作:休息 = 2:1** 的时间比例，例如全力运动40秒，休息20秒，循环进行。

上一节我们介绍了HIIT的基本概念，本节中我们来看看这套具体的跳跃动作组合。

## 训练动作详解

以下是本套跳跃系HIIT训练包含的全部动作，每个动作需全力完成规定时间。

*   **开合跳**：站立，跳跃时双腿向外打开，同时双手在头顶击掌；再次跳跃时回到起始姿势。
*   **深蹲跳**：从深蹲姿势开始，利用爆发力向上跳起，落地时缓冲并回到深蹲姿势。
*   **高抬腿**：原地快速奔跑，尽力将膝盖向胸部抬高。
*   **波比跳**：这是一个复合动作，流程为：**1. 蹲下；2. 后蹬腿呈平板支撑姿势；3. 完成一个俯卧撑（可选）；4. 收腿跳回深蹲姿势；5. 向上纵跳**。
*   **跳远**：从站立姿势起跳，尽力向前方跳跃。
*   **左右侧跳**：向身体一侧跳跃，落地缓冲后，立即向另一侧跳跃。
*   **前后跳**：向前方跳跃一步，落地后立即向后方跳跃一步。
*   **十字跳**：想象地面有一个“十”字，依次向前、后、左、右四个方向跳跃。
*   **米字跳**：在十字跳的基础上，增加向左前、右前、左后、右后四个斜向的跳跃。
*   **左右旋转跳**：跳跃的同时，在空中向左或向右旋转90度或180度。
*   **前后旋转跳**：跳跃时，在空中向前或向后旋转180度。

## 训练方案安排

了解了具体动作后，接下来我们看看如何将它们组织成一次完整的训练。建议采用循环制，以下是一个示例安排。

1.  选择4-6个上述跳跃动作。
2.  每个动作持续进行 **40秒**。
3.  每个动作完成后休息 **20秒**。
4.  完成所有选定动作为一个循环，循环之间休息 **60-90秒**。
5.  总共完成 **3-4个循环**。

## 注意事项与安全提示

在开始训练前，请务必阅读以下安全要点，以确保训练有效且无伤害。

*   训练前必须进行5-10分钟的动态热身，如关节活动和慢跑。
*   关注动作质量而非速度，确保落地轻盈、膝盖与脚尖方向一致。
*   如果感到关节疼痛或不适，应立即停止。
*   训练后进行静态拉伸，帮助肌肉恢复。
*   补充足够水分，并在非训练日安排充分休息。
