# 《豆哥私教课》第18课：健身方案之腹部训练（中腹篇）💪

在本节课中，我们将要学习如何有效训练腹肌的中部区域。我们将深入理解其发力原理，并掌握五个具体的训练动作。

## 概述

上一节我们介绍了腹部训练的基础知识，本节中我们来看看如何针对腹肌中层进行专项训练。腹肌中层的发力原理在于**上腹部与下腹部快速挤压**，通过这种收缩方式才能有效刺激腹肌中间部位。

训练时请注意，双手需始终抬起，手指头也要保持抬起状态。

## 五个核心训练动作

以下是练习腹肌中层的五个具体动作，每个动作都需配合节奏完成。

**动作一：双手扶肩卷腹**
*   双手扶在肩膀上，不要抱头。
*   大腿微微向上抬起，以给予腹肌中层更大刺激。
*   缓慢回位，配合“1、2”的节奏进行。

**动作二：侧向摸膝卷腹**
*   双手沿着大腿外侧方向，去触摸同侧膝盖或脚尖。
*   预备，同样配合“1、2”的节奏完成动作。

**动作三：直臂摸脚尖卷腹**
*   双手伸直，双脚放平。
*   用双手去触摸脚尖，节奏为“1、2”。

**动作四：背后对抗式卷腹**
*   双手打开，置于大腿后侧形成一个对抗的位置。
*   在“1、2”的节奏下完成卷腹动作。

**动作五：肘膝相触卷腹**
*   双手点在高靴位置，肘部合拢。
*   用双肘控制双膝，使肘与膝相触，预备，按“1、2”节奏进行。

## 总结

本节课中我们一起学习了腹肌中层的发力原理——**上下腹快速挤压**，并掌握了五个针对性的训练动作。坚持练习这些动作，将帮助你有效塑造和强化腹肌中部线条。
