# 《电话撩妹》：五：撩妹进阶_note

在本节课中，我们将学习如何通过电话与女生升级关系。升级关系的前提是已经成功吸引到对方。我们将系统地学习判断女生兴趣值的方法，并掌握四个循序渐进的升级关系技巧。

## 判断兴趣值：升级关系的前提

上一节我们介绍了通过“观”、“问”、讲故事和互动来吸引女生。本节中我们来看看如何判断女生的兴趣值，这是决定能否以及如何升级关系的关键。

兴趣值决定了你能说什么样的话、做什么样的事。如果女生对你兴趣值很低，说暧昧话语会引起反感；兴趣值中等时，可以说轻微的暧昧话语但不能深入；兴趣值很高时，则可以更自由地表达。

兴趣值来源于你在聊天中展示的硬价值和软价值。提高兴趣值的方法就是持续展示这些价值。

以下是判断女生兴趣值的八大指标：

1.  她对你说话的量多吗？
2.  她聊天时是否专心，还是边聊边做其他事？
3.  她会主动挑起话题吗？
4.  她会主动分享自己的生活吗？
5.  她聊天时有情绪波动吗？
6.  她会主动关心你吗？
7.  她会主动约你吗？
8.  她会主动说暧昧的话吗？

如果你收集到的指标少于三个，请暂时不要尝试升级关系。如果达到三个以上，则可以尝试。指标自然是越多越好。

## 升级关系四大核心技巧

在成功吸引并判断出足够兴趣值后，我们进入正题：如何在电话中升级关系。以下是四个循序渐进的技巧。

### 技巧一：真正的关心（细节+行动）

很多男生关心人时很“直男”，比如女生说不舒服，只会回复“多喝热水”，这会让对方觉得敷衍。

真正的关心需要遵循一条核心公式：**细节 + 行动**。

我们要关心到细节里去，避免重复别人说过的话。例如，当女生说感冒时：

**错误示范**：“多喝热水，记得买药吃。”

**正确示范（细节+行动）**：
“感冒如果比较轻，我建议先别吃药，免得产生依赖。建议你先喝点热水，一定要是烧开的水，晾到刚好能承受的温度再喝，这样效果最好。记得每小时喝一杯，能缓解症状。还有，保持室内空气流通，出门多穿件衣服。如果晚点还是难受，我给你买我常用的感冒药吧，给我地址，我叫跑腿送过去。吃了药还不好，我明天陪你去医院看看。”

再比如，关心女生按时吃饭：
“一定要记得按时吃饭，不然长期对胃不好，以后就吃不到好吃的了。如果你老忘记，我建议你设个闹钟，闹钟一响就放下事情去吃饭。要不以后到饭点，我准时提醒你吃饭？”

运用“细节+行动”的方式，女生才能感受到你是真心关心，而非敷衍。

### 技巧二：禁止效应（产生共鸣）

“禁止效应”是指人容易对与自己相似或有共同经历的人产生好感。就像在外地遇到老乡会感到亲切。

在电话中，可以通过以下三个手法制造禁止效应：

**1. 模仿她**
模仿她说过的特定话语、语气和语调。
*   **例如**：她常说“这不科学”，你也可以在合适时机说“这不科学”。她说“哎呀，讨厌”，你也可以用相似的语气说“哎呀，讨厌”。

**2. 经历她的过去**
在聊到过往经历时，分享你类似的经历，让她觉得你们是同一个世界的人。
*   **例如**：她提到小时候吃大大泡泡糖，你可以说：“我也吃过，还不小心粘到头发上，弄了好久。”
*   **例如**：她诉说职场被欺负，你可以说：“我理解，我以前实习时也这样，同事总把麻烦事推给我做。”

**3. 认同她的观点**
人都渴望被肯定。认同她的观点，她会觉得你们内心世界一致。
*   **例如**：聊到爱情观，她说最不能容忍出轨。你可以说：“对，我也觉得，出轨的人好恶心，干嘛吃着碗里看着锅里。”
*   **例如**：聊到人生观，她说要活在当下。你可以说：“是的，人生短短几十年，一定要珍惜现在。”

这个技巧的核心是**制造共鸣感**，让你更容易走进她的内心。

### 技巧三：说浅层次情话（土味情话）

运用前两个技巧后，如果女生回应变好，可以尝试说一些浅层次的情话，进一步升温关系。

这里需要运用之前学过的**发散性思维**。在升级关系阶段，发散性思维不再用于联想展示价值的话题，而是用于联想**情话**。

*   **例如**（聊到早上起床）：“昨晚我睡得不安稳。” 女生问为什么，你说：“没有你在身边，我怎么可能睡得安稳。”
*   **例如**（聊到吃饭）：“不能和你一起吃饭，我觉得吃什么都没味道。”
*   **例如**（聊到上班）：“不知道为什么，上班一有空闲，我就会情不自禁想起你。”
*   **例如**（聊到晚安）：“晚什么安，我只想睡醒你就在身边。”

这些就是所谓的“土味情话”。但**切记**：必须在吸引到位、女生对你有足够兴趣后才能使用，否则只会引起反感。

### 技巧四：说深层次情话（带入肢体接触）

深层次情话的特点是**包含肢体动作的描述**。这有两个好处：1. 测试女生的接受底线；2. 在女生脑海中提前建立画面感，为见面时的实际接触做铺垫。

*   **例如**（聊到洗澡）：“等等，我眼睛正在往你家的浴室飞过去。”
*   **例如**（聊到身高）：“你这身高，踮起脚尖就能亲到我了。”
*   **例如**（聊到吃饭）：“再吃下去，我可能就抱不动你了。”
*   **例如**（聊到做饭）：“你做饭的时候，我要从后面抱着你。”
*   **例如**（聊到见面）：“见面时，我要给你一个大大的拥抱。”
*   **例如**（聊到睡觉）：“我想抱着你睡，把我的手臂给你当枕头。”

如果她在聊天中对“牵手”、“拥抱”等话题不抗拒，那么见面时你就可以自然地尝试这些动作。

## 核心心法与总结

本节课中，我们一起学习了电话撩妹的进阶技巧——如何升级关系。

首先，我们学习了通过**八大指标**判断女生的兴趣值，这是升级关系的基础。

接着，我们掌握了升级关系的四大技巧：
1.  **真正的关心**：运用“细节+行动”公式，避免敷衍。
2.  **禁止效应**：通过模仿、分享共同经历、认同观点来制造共鸣。
3.  **浅层次情话**：在吸引到位后，用发散性思维联想并说出土味情话。
4.  **深层次情话**：说出包含肢体动作的情话，测试底线并营造画面感。

**运用的核心心法**：与吸引阶段一样，这些技巧需要**不断循环、重复使用**，而非一次见效。你用得越多，效果越好，也会越熟练。

最后，这些电话技巧同样适用于微信文字聊天或线下约会。约女生出来并没有特别的技术含量，关键在于她是否对你有好感。当吸引和关系升级到位后，邀约便会水到渠成。

用心学习并实践这些技巧，不仅有助于脱单，对你的人际交往能力也会有巨大提升。祝你早日收获幸福。
