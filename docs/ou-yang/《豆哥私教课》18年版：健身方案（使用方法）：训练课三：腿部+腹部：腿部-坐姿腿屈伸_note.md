# 《豆哥私教课》第18版：健身方案使用教程 第三课：腿部+腹部训练 🦵💪

在本节课中，我们将要学习一个针对腿部肌群的孤立训练动作——坐姿腿屈伸。这个动作主要锻炼大腿前侧的股四头肌，是塑造腿部线条、增强膝关节稳定性的有效练习。

## 动作名称：坐姿腿屈伸

下面介绍的动作是坐姿腿屈伸。关于肩部防护，如果器械配备有肩部防护垫，则使用它；如果没有，则无需特别处理。这个动作本身非常简单。

## 动作要领详解

以下是坐姿腿屈伸的具体执行步骤。

1.  **起始姿势**：坐于器械上，背部贴紧靠背。双手握住座位两侧的扶手或握把以稳定身体。
2.  **动作过程**：双脚踝前侧抵住器械下方的滚垫。发力缓慢抬起小腿，直至双腿完全伸直。
3.  **顶峰收缩**：在双腿伸直的顶点稍作停顿，感受大腿前侧肌肉的收缩。
4.  **返回阶段**：有控制地缓慢下放小腿，回到起始位置。下放过程中，不要让配重片相互撞击发出声响。

整个动作的节奏可以概括为：**抬起 -> 伸直（停顿）-> 下放**，重复进行。这是一个非常直接且易于掌握的动作模式。

## 核心要点与注意事项

上一节我们介绍了动作的基本流程，本节中我们来看看执行时的关键细节。

*   **全程控制**：动作的上升和下降阶段都应保持匀速、有控制，避免利用惯性。
*   **避免撞击**：下放重量时，确保配重片不要完全落下并相互碰撞，以保持肌肉持续紧张。
*   **关节保护**：在腿部即将完全伸直时，避免过度超伸膝关节。
