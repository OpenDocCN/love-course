## 从陌生人到朋友：聊天的四个阶段

上一节我们介绍了聊天的八大禁忌，目的是防止在交流中减分。本节中我们来看看如何在不同阶段通过聊天为自己加分。

聊天关系的发展通常分为四个阶段：**陌生人 → 朋友 → 暧昧 → 情侣**。每个阶段适合的话题和聊天方式都不同。

你不能在刚认识时就说“我想抱着你睡”，这很可能导致被拉黑。但如果关系已发展到暧昧或情侣阶段，说同样的话，女生可能会回应“那去你家还是去我家呢”。这就像玩游戏，必须遵循当前等级该做的事，不能越级挑战，否则只会失败。

因此，聊天需要步步为营。本节课重点讲解从陌生人到朋友阶段，我们应该聊什么以及怎么聊。

无论你通过探探、积木、世纪佳缘，还是朋友介绍、线下搭讪认识女生，想要推进关系，都需要准备足够多的话题来维持对话，直至达到目标阶段。

但“授人以鱼不如授人以渔”。与其直接给你话题，不如教你制造话题的方法。请记住制造话题的三个关键字：**观**、**问**、**说**。

## 第一个关键字：观 👀

“观”是指将你眼睛能看到的一切转化为话题。

以下是运用“观”的技巧：

*   **短发女生**：你可以说“短发女生有独特的发型”。
*   **与宠物的合照**：你可以说“我觉得喜欢小动物的女生都很有爱心”。
*   **旅游照片**：你可以说“看来你也是个闲不住的人，喜欢到处跑”。

这些例子表明，任何视觉信息加上你的个人感想，就能构成一个话题。

请记住“观”的公式：
**`眼睛看到的 + 你的感想 = 话题`**

即使女生朋友圈仅三天可见，你仍能看到她的**头像、背景墙、个性签名和微信昵称**。将这些视觉元素加上你的感想，就能发现无数可聊的话题。

## 第二个关键字：问 ❓

人际交往离不开一个基本模式：**提问与回答**的循环。

但要注意方式，避免变成令人反感的“查户口式”提问。例如连续问：“你叫什么名字？”、“你多大了？”、“你是做什么工作的？”、“有男朋友吗？”

正确的提问方式需要遵循一个关键公式：
**`提问 + 对答案给予反馈`**

提问后，必须针对女生的回答给出你的反馈。

以下是具体示例：

*   **问称呼**：你问“怎么称呼？”，她答“我叫小花”。
    *   **反馈**：你说“小花啊，人如其名，像花儿一样美丽。”
*   **问年龄**：你问“你今年多大了？”，她答“21岁”。
    *   **反馈**：你说“这是一个适合谈恋爱的年纪。” 如果她答25或26岁，你可以说“这是一个适合相亲的年纪。”
*   **问职业**：你问她的职业，她答是销售。
    *   **反馈**：你说“听说做销售的口才都特别好，吵架都是一把手啊。” 如果她是护士，可以说“我最怕护士了，小时候打针的经历终身难忘。” 如果她是老师，可以说“做老师那不是可以天天‘欺负’小朋友了？”

将“提问-反馈”模式融入日常，你会发现别人更愿意回答你的问题。

这里分享一个技巧：善用**百度**。当遇到不了解的信息时（如生僻的名字、职业），立即搜索，将查到的信息组织成你的反馈。例如，女生说她叫“玉衡”，你可以搜索后说：“原来是北斗七星之一啊，厉害了我的玉衡。” 这就能开启一个长达半小时的关于名字的话题。

对于“你在干嘛？”这类常见问题，也要给予反馈：

*   她答“刚刚吃完饭”。
    *   **对偏瘦女生**：你说“你要多吃点啊，看你都瘦成什么样了。”
    *   **对偏胖女生**：你说“你要多吃点啊，最近台风多。”（这是一个幽默套路）
*   她答“我在看电视”。
    *   **反馈**：你说“看电视多无聊，赶紧关了来和我聊天吧。” 或 “我猜你肯定是边抠脚丫边看电视。”

以上反馈风格偏向**逗趣和调皮**，能让女生觉得你有趣，从而更愿意继续聊天。

## 第三个关键字：说 💬

通过“观”和“问”，我们可以顺利破冰并了解女生的基本信息。但这远远不够，我们聊天的终极目标是**让女生喜欢上我们**。

“说”的核心在于**展示自身价值以吸引女生**。它涉及两个层面：**说什么**和**怎么说**。

例如：
*   **硬价值/生活类**：美食、旅行、健身。
*   **潜力股**：勤奋、上进、学习。
*   **才艺类**：烹饪、音乐。
*   **人格类**：孝顺、有爱心、幽默。
*   **生活类**：丰富精彩的经历。

这些关键词中的每一个都能衍生出一个或多个话题。你需要向女生讲述包含这些元素的内容。

### 怎么说：两个核心技巧

#### 技巧一：话说一半，留一半

不要一次性说完，制造悬念，引导女生提问。

以下是具体示例：

*   **（价值展示）** 你说：“我好撑啊，要扶着墙了……”（留一半）。等她问“怎么了？”，你再发去吃自助餐的照片。
*   **（潜力股展示）** 你说：“啊，终于完成这个方案了，折腾了我两天两夜……”（留一半）。等她好奇，你再展开。
*   **（才艺展示）** 你说：“还是自己做饭好吃……”（留一半）。等她回应，你再发烹饪成果的照片。
*   **（人格展示）** 你说：“我特别喜欢金毛，特别乖……”（留一半）。这会让她觉得你有爱心。
*   **（生活展示）** 你说：“累死我了，全身都湿透了……”（留一半）。等她问“为什么？”，你再答：“刚在跑步机跑了40分钟，好爽。”

#### 技巧二：关键字联想

从聊天中出现的任意关键词出发，进行联想和延伸，组织成完整话题。

记住这个公式：
**`关键字 → 铺垫 → 内容`**

**铺垫**的万能句式是：“说到（关键字），我就想起了……”

**内容**部分要结合“话说一半”的技巧，通过互动推进。

**完整示例一：**
*   **关键字**：妈妈。
*   **铺垫**：你说：“说到妈妈，我就想起了去年的一件事……”（停，等她问）。
*   **内容（互动式推进）**：
    *   她问：“怎么了？什么事啊？”
    *   你答：“那段时间我妈显得特别焦虑，老在朋友圈发养生偏方链接。”（停，等她问）。
    *   她问：“然后呢？”
    *   你答：“我担心她身体，问她也不说。后来我直接回家带她去医院检查了。”（停，等她问）。
    *   她问：“那到底怎么了？”
    *   你答：“原来是她自己疑心病犯了，害我白担心一场。”
    *   **吸引力元素**：这个故事展示了你的**孝顺**。

**完整示例二：**
*   **关键字**：工作。
*   **铺垫**：你说：“说到工作，我想起刚来广州的时候……”（等她问）。
*   **内容（互动式推进）**：
    *   她问：“怎么了？”
    *   你答：“那时没钱，为了立足，除了吃饭睡觉全在工作，拉屎都没时间。”（等她回应）。
    *   你接着说：“努力了四个多月，终于有点小成就，不用为下顿吃泡面还是白粥发愁了。”（等她回应）。
    *   你最后说：“那时月薪终于过万，不过现在也忙。只有晚上下班趴床上玩手机，才是最幸福的时光。”
    *   **吸引力元素**：这个故事展示了你的**努力**和**上进心**。

**关键要点**：每个故事或话题都应嵌入一个**吸引力元素**（参考上图）。女生喜欢听故事，如果你的故事包含这些吸引人的特质，你就能通过讲述让女生对你产生好感。

本节课中我们一起学习了从陌生人发展到朋友阶段的核心聊天方法：
