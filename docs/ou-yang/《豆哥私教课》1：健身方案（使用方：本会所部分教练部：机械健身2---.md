## 健身方案核心框架

一套完整的健身方案通常包含几个关键部分。以下是其主要构成：

1.  **训练目标**：明确每次训练的目的，例如增肌、减脂或提升力量。
2.  **动作选择**：根据目标肌群选择合适的训练动作。
3.  **训练参数**：包括组数、次数、重量和组间休息时间。
4.  **动作示范**：通过标准视频确保动作执行准确。

## “机械健身2”动作详解

上一节我们介绍了方案框架，本节中我们来看看李健教练示范的具体动作要领。正确的姿势是避免受伤和提升效果的基础。

以下是视频中强调的几个关键动作规范：

*   **保持核心收紧**：在任何推、拉、举的动作中，腹部应保持紧张，以稳定躯干。
*   **控制运动节奏**：动作的向心阶段（发力）和离心阶段（还原）都应有控制地进行，避免借用惯性。公式可表示为：**发力1秒 → 顶峰收缩1秒 → 还原2-3秒**。
*   **关节不锁死**：在动作顶点，肘关节或膝关节应保持微屈，以持续刺激肌肉并保护关节。
*   **呼吸配合**：发力时呼气，还原时吸气。例如，在卧推时，推起时呼气，下放时吸气。

## 如何制定你的训练计划

了解动作规范后，你需要将它们组合成个人计划。一个基础的训练计划可以通过代码逻辑来理解：

```python
# 示例：一个简单的训练日计划逻辑
训练目标 = “增肌”
目标肌群 = [“胸部”, “肩部”, “肱三头肌”]
动作列表 = [“杠铃卧推”, “哑铃飞鸟”, “肩上推举”, “侧平举”, “绳索下压”]

for 动作 in 动作列表:
    执行组数 = 3
    每组次数 = 8-12
    使用重量 = 选择能标准完成规定次数的最大重量
    组间休息 = 60-90秒
```

要点包括：
*   **渐进超负荷**：每周尝试小幅增加重量、次数或组数。
*   **分化训练**：将不同肌群的训练分配到不同日期，例如“推日”、“拉日”、“腿日”。
*   **记录与调整**：记录每次的训练数据，并定期调整计划以避免平台期。
