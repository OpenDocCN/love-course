# 《神撩术2.0》：未来街搭：萌萌哒两人组合_note

在本节课中，我们将要学习一种针对两人组合的街头搭讪开场白。这种开场白旨在快速、友好地开启对话，并建立初步联系。

上一节我们介绍了街头搭讪的基本概念，本节中我们来看看如何具体应对两人同行的场景。当你在街头遇到两位想认识的女性时，直接与其中一人对话可能会让她的同伴感到被忽视。因此，一个兼顾两人的开场方式更为得体。

以下是完整的开场白流程与步骤解析。

**开场请求**
```
我能跟你朋友说两句话吗？
```

**确认目标**
```
我能跟你，对。
```

**表达关注原因**
```
我刚刚在那边看见你，然后觉得你很有女王范，所以情不自禁的想过来认识你。
```

**提出核心目的**
```
交个朋友，加个微信。
```

**开启轻松话题**
```
今天礼拜你们出来逛街？
```

**询问称呼**
```
怎么称呼？
```

**结束与道别**
```
下面有江岚庆，好，名字我发给你。謝謝觀看。下次見！
```

让我们逐一拆解这个流程。首先，通过询问“我能跟你朋友说两句话吗？”来获得与目标对话的许可，这显得礼貌且尊重其同伴。紧接着，确认你的目标对象，并立即给出一个真诚的、基于观察的赞美（例如“女王范”），这能有效传达你的兴趣。

提出“交个朋友，加个微信”是明确你的意图，避免含糊。随后，用一个简单的、关于当前活动的问题（如“出来逛街”）来过渡，使对话氛围变得轻松自然。询问对方“怎么称呼”是交换基本信息、推进关系的关键一步。

最后，在对话结束时，可以像示例中那样提及一个后续行动（如“名字我发给你”），并为这次短暂的相遇表达感谢和告别，留下一个友好的印象。

本节课中我们一起学习了针对两人组合的街头搭讪完整话术。核心在于**礼貌开场、真诚赞美、明确意图、轻松过渡、交换信息、友好结束**。通过这个结构化的流程，你可以在尊重对方的前提下，有效地开启一段新的社交连接。
