## 一、 运用“观问说”开启对话

上一节我们介绍了约会前的准备，本节中我们来看看如何在约会现场开启对话。所有话题都源于“观”、“问”、“说”这三个关键字。网络聊天有网络聊天的“观问说”，现场约会也有现场约会的“观问说”。

### 1. “观”：观察与感想

记住这条公式：**你的眼睛看到的 + 你的感想 = 话题**。

在现场约会时，要学会利用眼睛看到的所有事物组织成话题。

以下是具体运用方法：

*   **针对女生的外貌进行评价**：例如，“今天发型很清爽，看起来很精神。”或者“我感觉你真人比照片好看多了。”
*   **作用**：通过赞美开启约会，让女生感到开心，为后续互动创造良好氛围。

### 2. “问”：提问与反馈

说完开场白后，通常你们会开始走动。这时要运用“问”这个关键字。

**核心要点是：问完问题一定要学会反馈。**

以下是几个提问与反馈的例子：

*   **提问**：“你过来的时候堵车吗？”
    *   **反馈**：“广州这个时间确实太堵了，所以我平时出门都喜欢坐直升机。”（幽默反馈）
*   **提问**：“你平时喜欢吃什么？”
    *   **反馈**：“那我们就去吃这个吧，你带路，我拉着你的衣角。”（引导行动）
*   **提问**：“你对这附近熟悉吗？”
    *   **反馈（如果女生熟悉）**：“那你一定要看着我，我对这里不熟，别把我弄丢了。”
    *   **反馈（如果女生不熟）**：“那你要跟紧我了，不然弄丢了，我不知道怎么跟你妈妈交代。”（举一反三）

### 3. “说”：即兴评论与分享

“说”需要配合约会时双方都能看到的事物进行即兴评论。

**方法是：看到什么 + 想到什么（最好带点幽默）就直接说出来。**

以下是一些“说”的实例：

*   **看到情侣**：“我最受不了其他情侣在我面前手拉手，我每次都会从他们中间走过去。”（幽默拆散）
*   **看到小朋友**：“看到这群小朋友，就觉得长大烦恼真多，还是小时候幸福。”
*   **看到溜冰场**：“看到溜冰场就想起我当初学的时候，摔得我妈都不认识了，晚上只能趴着睡。”
*   **看到游戏厅**：“我小时候特喜欢和同学去打游戏。你们女孩子应该更喜欢在游戏厅里抓娃娃吧？晚点我们来玩一下。”

通过熟练运用“观问说”这三个技巧，约会初期的对话就能顺利展开。请记住这些思维并学会举一反三。

## 二、 深入互动：讲故事与玩游戏

当你们坐下来吃饭或喝东西时，需要更深入的聊天技巧。这时要学会两个核心方法：**讲故事**和**玩游戏**。配合“观问说”，你将永远不缺话题。

### 1. 讲故事的技巧

每个人最关心的都是自己。因此，开启话题时应先从对方入手，引导女生多说话，从中提取关键词进行联想和延伸。

**讲故事的核心公式是：**
1.  **问女生一个问题**。
2.  **从她的回答中提取关键词**。
3.  **围绕关键词讲述自己的相关故事**，并在讲述中不断与女生互动（例如提问“你觉得呢？”）。
4.  **故事讲完后，利用故事中的关键词或女生的原始回答继续延伸话题**。
5.  **循环此过程**：聊完一个话题后，问下一个问题。

以下是具体运用示例：

*   **你问**：“你皮肤真好，有什么保养秘诀吗？”
*   **女生回答**：“我平时会敷XX面膜，用XX护肤品，生活习惯也比较规律。”
*   **你提取关键词**：“敷面膜”。
*   **你讲故事并互动**：“我也试过学护肤。有一次在客厅敷面膜被我妈看到，她说我像个娘们。（**暂停并互动**）你觉得男生敷面膜真的很娘吗？……（等她回答后继续）我当时二话不说，也给我妈敷了一片。（**再次暂停互动**）你猜结果怎么样？……现在她皮肤变好了，老让我给她买面膜。”
*   **延伸话题**：“那你妈妈平时敷面膜吗？”

**锻炼方法**：如果一开始不熟练，可以按以下步骤练习：
1.  先在微信上用**一段段语音**给女生讲故事，注意互动。
2.  熟练后，尝试进行**语音通话**聊天。
3.  最后，可以尝试**视频聊天**来锻炼临场讲故事的能力。

### 2. 玩游戏的技巧

即使掌握了讲故事，偶尔也可能出现卡壳。这时，和女生玩一些简单的游戏是增进感情、刺激情绪波动的有效手段。

以下是三个可以在约会中玩的游戏示例：

**游戏一：反应测试**
*   **铺垫**：“你觉得你的反应能力怎么样？我们来玩个游戏测试一下。”
*   **流程**：
    1.  让女生快速说五遍“老鼠”。
    2.  紧接着问她：“猫最怕什么？”
    3.  女生通常会惯性回答“老鼠”。此时你可以解释：“其实猫不怕老鼠，老鼠才怕猫。” 这是一个利用思维惯性的小玩笑。

**游戏二：趣味看手相**
*   **铺垫**：“你相信手相吗？我家祖传了一些看手相的技巧，给你免费体验一下。”
*   **流程**：（以幽默、逗趣的方式解读，切勿认真）
    *   **生命线**：“我看你能活过下个礼拜。”（一个人活，别拉上我）
    *   **智慧线**：“完全看不到智慧线，难怪你这么傻。”
    *   **婚姻线**：“你这辈子可能要结两次婚，一次嫁给九十多岁的糟老头，一次嫁给像我这么帅的帅哥。”（不一定是我，看我们感情发展）
    *   **财富线**：“你九十五岁前都没什么钱，但九十五岁会一夜暴富，因为你那个老头老公挂掉了，你继承了遗产。”

**游戏三：连环脑筋急转弯**
*   **铺垫**：“你玩过脑筋急转弯吗？我这儿有几个连贯的问题，考验一下你的聪明程度。”
*   **流程**：（问题需连贯，答案有联系）
    1.  “飞机运了500块砖，掉下来1块，还剩几块？”（答案：499块）
    2.  “把大象放进冰箱需要几步？”（答案：3步。打开门，放进大象，关上门）
    3.  “把长颈鹿放进冰箱需要几步？”（答案：4步。打开门，取出大象，放进长颈鹿，关上门）
    4.  “老虎生日，所有动物都去了，谁没去？”（答案：长颈鹿。因为它还在冰箱里）
    5.  “老奶奶要过鳄鱼河见老爷爷，河水不深但满是鳄鱼，她过去了却没被咬，为什么？”（答案：鳄鱼去参加老虎的生日派对了）
    6.  “但老爷爷最后还是死了，为什么？”（答案：被飞机上掉下来的那块砖头砸死了。联系第一题）
