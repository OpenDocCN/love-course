# 课程名称：Python基础入门 - 第1课：变量与数据类型 🐍

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：**变量**和**数据类型**。理解它们是编写任何程序的第一步。

## 概述 📋

程序的核心是处理数据。为了处理数据，我们需要将数据存储在计算机的内存中，并给这些存储位置起一个名字，这就是**变量**。同时，数据有不同的种类，例如数字和文本，这些种类被称为**数据类型**。掌握如何创建变量以及了解不同的数据类型，是编写有效代码的基础。

上一节我们介绍了本课程的目标，本节中我们来看看如何具体地使用变量。

## 什么是变量？ 🏷️

你可以把变量想象成一个贴有标签的盒子。标签就是变量的名字，盒子里存放的就是数据。在Python中，我们使用等号 `=` 来给变量赋值。

其基本语法是一个简单的公式：
**变量名 = 值**

例如，下面的代码创建了一个名为 `message` 的变量，并将文本“你好，世界！”存放进去。
```python
message = "你好，世界！"
```
执行这行代码后，我们就可以在程序的其他地方使用 `message` 来代表“你好，世界！”这段文本。

以下是创建和使用变量时需要记住的几个关键点：
*   **变量名**：只能包含字母、数字和下划线，且不能以数字开头。
*   **大小写敏感**：`my_var` 和 `My_Var` 是两个不同的变量。
*   **描述性命名**：尽量使用能表明变量用途的名字，如 `user_age` 而非 `a`。

理解了如何用“盒子”存储数据后，我们还需要知道盒子里能放哪些“类型”的东西。接下来，让我们认识一下Python中的主要数据类型。

## 基本数据类型 🔤

数据类型定义了数据的种类以及能对它进行哪些操作。Python有一些内置的基本数据类型。

### 1. 整数与浮点数 🔢

用于表示数字。**整数**是不带小数点的数，**浮点数**是带小数点的数。
```python
age = 25          # 整数 (int)
price = 19.99     # 浮点数 (float)
```

### 2. 字符串 📝

用于表示文本。需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
```python
name = "欧阳浮夸"  # 字符串 (str)
greeting = ‘Hello‘
```

### 3. 布尔值 ✅❌

只有两个值：`True`（真）和 `False`（假）。常用于逻辑判断。
```python
is_student = True   # 布尔值 (bool)
is_raining = False
```

为了确认我们变量的“盒子”里到底装了什么类型的数据，Python提供了一个非常方便的函数。

## 使用 `type()` 函数检查类型 🔍

`type()` 函数可以返回任何值或变量的数据类型。这对于调试和确保程序按预期运行非常有用。

使用方法如下：
```python
print(type(10))          # 输出：<class ‘int‘>
print(type(3.14))        # 输出：<class ‘float‘>
print(type(“Python”))    # 输出：<class ‘str‘>

x = 100
print(type(x))           # 输出：<class ‘int‘>
```

现在，我们已经学会了如何创建变量、了解了几种基本类型，并能检查它们的类型。最后，让我们看看如何改变一个已经存在的变量。

## 变量的重新赋值 ✏️

变量的值不是一成不变的。你可以随时给一个已存在的变量赋予新的值，新值的数据类型也可以与之前不同。

```python
score = 95      # 初始赋值为整数
print(score)    # 输出：95

score = “优秀”  # 重新赋值为字符串
print(score)    # 输出：优秀
```
请注意，重新赋值后，变量原来的值就被新的值替换了。

## 总结 🎯

本节课中我们一起学习了Python编程的基石：
1.  **变量**是存储数据的命名容器，通过 `=` 进行赋值。
2.  主要**基本数据类型**包括：整数(`int`)、浮点数(`float`)、字符串(`str`)和布尔值(`bool`)。
3.  可以使用 **`type()` 函数**来检查变量或值的数据类型。
4.  变量可以通过重新赋值来更新其存储的内容。

牢固掌握这些概念，将为后续学习更复杂的操作（如数学运算、字符串处理和控制流程）打下坚实的基础。
