# 《豆哥私教课》第18版：健身方案使用教程 - 第1课：胸部与手臂训练之平杆弯举 💪

在本节课中，我们将要学习一个针对手臂肱二头肌的基础训练动作——平杆弯举。这个动作能有效塑造手臂线条，增强肌肉力量。

## 🏋️ 动作准备与起始姿势

上一节我们介绍了本节课的训练目标，本节中我们来看看如何为平杆弯举做好准备。

首先，你需要找到平杆器械。面对器械站立，调整身体与器械的距离。

以下是具体的准备步骤：

*   身体与器械保持约一脚的距离。
*   两脚打开，宽度与肩同宽。
*   脚尖指向前方。
*   吸气，收紧腹部核心，稳定躯干。

## 🤲 动作执行详解

完成准备姿势后，我们进入动作的执行阶段。正确的姿势是保证训练效果和避免受伤的关键。

握住平杆，开始执行弯举动作。在整个过程中，请注意手臂与身体的相对位置。

以下是动作执行的分解步骤：

*   两侧上臂（大臂）尽量贴近并夹紧身体两侧，保持不动。
*   呼气，发力将平杆向上弯举。
*   当手臂弯曲至约90度时，开始吸气。
*   在吸气的同时，有控制地、缓慢地将平杆下放。
*   重复“吸气下放，呼气举起”的节奏。

**核心要领公式：大臂贴紧身体 + 有控制的慢速下放**

## 🔁 动作节奏与呼吸配合

一个流畅的动作离不开稳定的节奏和正确的呼吸配合。这是提升训练质量的重要环节。

动作的节奏应与呼吸同步，形成连贯的运动循环。

以下是推荐的呼吸与节奏模式：

*   上举时呼气。
*   在动作顶端稍作停顿。
*   下放时吸气，并确保过程缓慢。
*   感受肌肉的持续张力。

**代码式节奏描述：`举起(呼气) -> 顶端停顿 -> 下放(吸气，慢速) -> 循环`**

## 📝 本节总结

本节课中我们一起学习了肱二头肌训练的基础动作——平杆弯举。我们掌握了从器械距离、站姿、到动作执行轨迹和呼吸配合的全过程。记住核心要点：保持大臂固定、有控制地完成慢速下放，并始终协调好呼吸与动作节奏。坚持练习这个动作，将帮助你有效强化手臂前侧的肱二头肌。
