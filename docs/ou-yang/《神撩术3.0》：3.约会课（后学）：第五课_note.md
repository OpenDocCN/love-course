# 《神撩术3.0》：3.约会课（后学）：第五课_note

在本节课中，我们将要学习约会的基本操作，重点探讨约会前需要做的准备工作。理解约会的核心目的并做好充分准备，是成功约会的第一步。

## 概述：约会的双重目的

约会主要有两个目的。第一个目的是**验货**。第二个目的是**升级关系**。

上一节我们介绍了约会的整体框架，本节中我们来看看约会的第一个核心目的——“验货”。

## 一、 约会的核心：验货逻辑

你和女生聊了很久，出来见面，第一件事就是为了看看是否“货对版”。

相信很多兄弟曾经在淘宝看商品页面，包括图片、介绍和评价。你把衣服买回来拿到手后，会面临三种情况。

以下是你会遇到的三种情况：

1.  你对这件衣服很满意，特别喜欢。
2.  你穿上去觉得勉强还可以，想先穿几天看看。
3.  你对这件衣服特别不满意，想要退货。

我们和女生约会，是同样的逻辑，也是一个验货的过程。但和淘宝验货不同的是，淘宝是你在验货。而约会时，你也在验女生是否“货对版”，女生同时也在验你。

此时，你这边会面临同样的三种情况。

以下是你会面临的三种情况：

1.  你对女生特别满意，特别喜欢她。
2.  你对女生的感觉很一般，想继续交往看看。
3.  你对这个女生很不满意，发现“照片”和真人完全不一样，想直接走人。

女生那一边，同样会面临三种情况。

以下是女生可能面临的三种情况：

1.  她对你很满意，很喜欢你。
2.  她对你的感觉很一般。
3.  她很不喜欢你。

我们面临这三种不同的情况，做法自然也不一样。

**情况一：女生特别喜欢我**
如果女生特别喜欢你，接下来的约会你只要不犯错，就很容易成功。

**情况二：女生感觉一般**
女生对你的感觉很一般，但愿意给你机会继续交往看看。这种情况是最常见的。很少有人能做到让别人一见钟情，除非你特别帅。人和货物不一样，只看一眼看不出好坏。女生验货时，需要更深入一点，还需要一些时间。这时我们只需要通过这次约会，把自己好的一面逐步展现出来，也就是展现我们的**价值**。这样就能加强对女生的吸引，让她越来越喜欢你。

**情况三：女生不喜欢你**
你第一眼就让女生不喜欢你，就像你第一眼不喜欢那个女生一样。这时正确的做法是不要继续聊下去，应该尽早离开。我曾经试过，即使说好了去吃饭、看电影，如果遇到不喜欢的女生，我也会找个借口很快离开。因为一开始女生就表现出不喜欢，再怎么努力都是浪费时间，不如赶紧走，把时间用在寻找下一个机会上。

以上就是约会时会面临的三种不同情况，以及我们要采取的不同做法。

除了以上做法，兄弟们还要记住，我们需要为约会做好一切准备。

## 二、 约会前的三大准备

约会前的准备总共有三个要点：第一是我们的形象，第二是约会地点，第三是约会目的。

### 1. 形象准备：细节决定成败 🪞

我们的形象包含了发型、穿衣打扮、鞋子等各方面的细节。关于形象的大方面内容，我在第二节课已经详细说过。在这里，我要教兄弟们约会时在细节层面的技巧。

首先你要检查你的发型有没有吹好，有没有杂毛露出来。一定要定好型，可以用杰斯派等定型喷雾。发型方面，你一定要剪适合自己头型的发型，可以去一些比较专业、有连锁的机构，找高级发型师给你建议。

其次是白头发问题。像我有白头发，很多兄弟也是。如果只有零碎几条，就把它拔掉。如果白头发情况挺严重，就一定要去染黑。

然后到脸部问题。脸部要从上到下检查。

以下是脸部检查的步骤清单：

*   **眉毛**：检查有没有多余的杂毛，有的话一定要刮干净。如果眉毛比较稀缺，可以买一支眉笔学习画眉。经济条件允许的话，可以做一个半永久的眉毛。
*   **脸毛**：看看自己脸上有没有杂毛，有的话一定要刮干净，这很减分。
*   **鼻毛**：定期检查，一发现有一两根露出来，就忍痛拔掉。平时可以自己拿小剪刀修剪，大概一个月一次。
*   **胡子**：检查清楚，不要遗漏一两根没刮干净。
*   **耳朵**：用湿毛巾把耳朵位置的皮屑、污垢、耳屎擦干净。
*   **指甲**：指甲一定要剪干净。如果有留指甲的习惯，要把指甲缝隙里的脏东西刮干净。

接下来是衣服。有些兄弟的衣服洗后很皱，折痕太明显的话要烫平。如果没法烫平，就不要穿，这很减分。衣服一定要保持干净。

以下是针对体型的穿衣建议：

*   如果偏瘦，可以穿比较宽松的衣服，看起来会壮一点。
*   如果偏胖，可以穿修身一点或黑色的衣服，比较显瘦。

衣服的细节是，很多兄弟会忽略衣服上的线头。你要把这些线头找出来，用剪刀剪掉。裤子和鞋子也一样，鞋子一定要保持干净，哪里脏了都不行，要拿鞋刷刷干净。

如果你像我一样身高不够（我只有一米六四），那么我们一定要穿上内增高鞋。

以上就是形象方面大体的准备。再小一点的细节是，尤其是现在天气热，很容易出汗。我们要准备好香水，可以掩盖身上的汗酸味。口香糖也是必备的，可以让口气清新。靠近女生说话时，不会有口气。接着还要准备湿纸巾。如果有条件，就买一两包放在口袋里，很加分。因为天气干燥时皮肤会干燥，拿湿纸巾擦一擦就会很湿润。

说了那么多，其实说白了就是为了让自己在每次出门约会前，形象都保持到最佳状态。我们要改变一切能够改变的东西，把那些不合格、减分的全部改掉，让自己的形象时刻处于最佳状态。这样我们的第一关就是合格的。

这就是我们约会需要准备的形象方面的东西。

### 2. 地点准备：创造交流空间 📍

第二个准备是约会地点。关于这个，有很多兄弟曾经问过我。

> 浮夸老师，我和女生第一次约会，可不可以去一些比较有意思的地方？比如周杰伦的演唱会、游乐园玩过山车、或者去鬼屋，很刺激。应不应该去呢？

这些所谓有意思的约会地点，你一定要搞清楚你约会的目的。尤其是前两次约会，你们要做的是相互验货。验完货之后，就是要增加彼此之间的熟悉感。熟悉感怎么来？只有通过双方交流才能增加熟悉感。

*   周杰伦的演唱会那么吵，你们怎么交流？
*   游乐园项目多，玩都玩不过来，你们怎么交流？
*   鬼屋那么刺激，你们怎么交流？

没法交流。而你约女生去这些地方，女生愿意出来，可能真的不是冲着你这个人来的，不是对你感兴趣。她可能是对这些项目感兴趣，就是想去鬼屋玩一下，想去周杰伦的演唱会看一看。刚好免费有人可以去玩，又有人买单，那干嘛不去呢？

所以，前两次约会，我不建议兄弟们去这些所谓有意思的地方。等你们俩真的熟悉了，有感情了，成为男女朋友了，再去看演唱会、玩过山车、去鬼屋增进感情，那才是正确的。起码现在不行。

那么一开始和女生约会应该去哪里呢？你们要记住一个重点：**能够交流的地点**。

能够交流的地点，比如约她去咖啡馆喝咖啡，约她出来在比较安静的地方吃个饭、散散步，或者去逛街。经济条件可以的兄弟可以去沐足、水疗，也不错。这些地点都非常有利于你们交流。

至于看电影，也是排在有了交流之后的事情，因为看电影时也没法交流。

兄弟们一定要记住，前面和女生最开始那几次约会，都离不开我所说的这些地点：要么吃饭，要么喝东西，要么散步，要么逛街。

当我们知道了这些约会地点之后，还需要知道的一个东西是：**转场的地点**。

很多时候，我们和女生约会的第一个地点主要作用是验货。先验货，感觉双方都还不错，想要继续约下去，那就肯定不能待在同一个地方太久。所以我们需要换地方。

比如约女生吃完饭，去散个步；或者喝完东西，去逛个街、看个电影。这些转场的地点很重要，你一定要提前规划好。

> 比如刚吃完饭，你想去看电影，但走路要走半小时，或者坐车也要半小时。吃饭的时间点很难打车，等待时间太长，女生就会很不舒适、不开心。

时间太长会让女生感到不舒适、不开心，这场约会就是失败的。所以我建议兄弟们和女生约会前，尽量去踩一下点。约会地点尽量选一些大型的、应有尽有的商城，比如万达广场。吃完饭，可能楼上或楼下就是电影院；看完电影，楼下就可以逛街，或者旁边可以散步。

而且最后一个点就是一定要提前踩点。因为很多兄弟约会经验太少，几乎没约过女生出去，根本不知道约会地点是什么样子。我建议兄弟们一定要提前去踩点，和女生约会前先去看看约会地点，熟悉那里的环境。这样可以避免吃饭排队排很久、转场不知道去哪、或者连厕所都不知道在哪里（女生要上厕所，你自己都不知道，那就不好了）。

以上就是关于约会地点以及转场地点，我们需要做的事情。

### 3. 目的准备：给约会设定任务 🎯

约会的最后一个需要准备的东西就是我们的目的。很多兄弟和女生约会都是漫无目的的，根本不知道约会要干嘛，目的是什么，完全不知道自己要干嘛，走一步看一步。这是很多兄弟都会犯的错误。

这个错误，像我，我是每一次约会，都会根据和这个女生聊到的程度，采取一个策略：我到底要去跟女生完成什么东西？我会给自己下一个任务。

> 比如，我上一次和女生出去吃饭。我约她吃饭，还没见面之前，我就在想：我今晚要跟她发展到什么地步？我给自己下了一个任务：**我今晚一定要亲到她**。这就是我今天晚上的目的，这就是我这场约会想要去做的事情。

那怎么样才可以亲到她呢？这时我的整个约会，都会为了最后这个目的去转，一切都是围绕着这个目的去转。我为了这次约会所有的所作所为，都是为了最后亲到她这个目的去做的。

我想着：我要亲到她，我应该怎么做？我前面肯定要先牵到她的手，然后再靠近一点。牵到手之后，在什么时候、什么地点去亲她是最好的、最合适的？然后我就不停地在跟她约会，走着、聊着的时候，经过一个红绿灯，在等红绿灯时，没有其他人，就我和她两个人等。我趁她一个不注意，瞬间就把她亲到了。

兄弟们，你和女生约会之前，一定一定要告诉自己：我想要做什么？我和这个女生约会，我到底想要做什么？你要给自己下一个任务。

这个任务可能是：和女生牵到手（我一定要牵到她的手）；或者说想要抱到她；或者说要聊到她喜欢你；或者要学我一样亲到她；甚至你这次的目的是要把她带回家，或者带她去开房。

这就是你的一个任务。你还没出去约会之前，就要给自己下这么一个任务。那么你的一切行动、一切行为、你所说的每一句话、所做的每一件事、每一个动作，都是围绕着这个目的去走的，而不是漫无目的、走一步看一步的状态。

你可以想一下，你本来和女生聊得好好的，出来见面后，本来想着今晚要和这个女生发生点什么，但是一直没什么好机会，最后什么都没发生。你回到家后，会不会觉得自己很懊悔、很后悔？天哪，浪费了那么多时间、那么多钱，什么都没完成，手都没牵到。

所以你以后一定要记住，每次和女生出去约会之前，你都要给自己下一个任务，而且要完成这个任务。这才是你这一次和女生约会的主要目的。
