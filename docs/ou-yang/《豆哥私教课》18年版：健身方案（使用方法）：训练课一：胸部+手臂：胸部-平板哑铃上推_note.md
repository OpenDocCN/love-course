# 《豆哥私教课》18年版：健身方案（使用方法）：训练课一：胸部+手臂：胸部-平板哑铃上推_note

在本节课中，我们将要学习一个经典的上肢力量训练动作——**平板哑铃卧推**。这个动作主要针对**胸大肌**，同时也能锻炼到手臂和肩部的前束肌群，是构建胸部厚度和力量的基础练习。

上一节我们介绍了本课的训练目标，本节中我们来看看如何正确执行平板哑铃卧推。

## 动作详解：平板哑铃卧推 💪

这个动作被称为平板哑铃卧推。首先，双手握住哑铃，虎口相对，将哑铃下放至身体两侧，呈现预备姿势。平躺在训练凳上，顺势将哑铃举起至胸前。

**起始姿势**：
*   掌心朝向正上方。
*   双脚稳固踩地，膝盖保持微曲状态。膝盖既不要过度弯曲，也不要完全伸直。过度伸直会对腰椎造成压力，因此应始终保持膝关节微曲。

**动作执行过程**：
*   在整个动作过程中，避免借用腰部力量发力。
*   缓慢下放哑铃，肘部向身体两侧打开，使上臂与躯干趋于水平。
*   缓慢向上推起哑铃，直至手臂伸直，将哑铃推回至胸大肌正上方。
*   哑铃的运动轨迹应始终受控，不要利用惯性发力。遵循“缓慢下放，再缓慢上推”的节奏。

以上就是平板哑铃卧推的完整动作描述。

本节课中我们一起学习了**平板哑铃卧推**的正确做法。我们明确了起始姿势中**掌心朝上**与**双脚位置**的关键点，并重点掌握了动作过程中**避免腰部借力**、**控制下放与上推速度**的核心要领。坚持规范的训练是获得效果与避免受伤的基础。
