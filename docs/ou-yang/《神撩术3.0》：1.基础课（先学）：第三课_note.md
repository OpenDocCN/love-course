# 《神撩术3.0》：1.基础课（先学）：第三课_note

在本节课中，我们将要学习如何打造一个能够有效吸引女生的朋友圈。我们将从避免减分项和掌握加分项两个核心方向入手，详细解析朋友圈的构成要素，并提供具体的操作模板和建议。

## 概述：朋友圈是你的网络名片

你的朋友圈相当于你在网络上的名片。这张名片的质量直接决定了你吸引女生的速度和数量。名片做得好，吸引力就强；做得不好，则可能毫无吸引力或进展缓慢。

上一节我们明确了朋友圈的重要性，本节中我们来看看打造朋友圈的两个核心方向。

## 第一部分：避免减分的朋友圈

打造朋友圈的首要原则是不能让自己减分。以下是五种常见的减分行为，请务必检查并清理。

### 1. 过度转发低价值内容
以下是会减分的转发内容类型：
*   频繁转发微信公众号文章、心灵鸡汤。
*   转发游戏攻略、探探等应用下载信息。

### 2. 发布负能量或炫富内容
以下是令人反感的内容类型：
*   抱怨生活、工作、社会不公等负能量言论。
*   刻意炫耀财富，如晒新车并配文吹嘘。

### 3. 刷屏行为
刷屏会严重引起他人反感。想象一下，如果你的朋友圈连续五六条都是同一个微商的广告，你是否会产生删除他的念头？同理，一天内发布超过三条关于同一事件（如旅游）的动态也属于刷屏。

### 4. 照片比例失衡
朋友圈的核心作用是呈现个人价值，照片比例至关重要。
*   **缺乏本人照片**：如果朋友圈全是美食、风景、宠物照，唯独没有本人，会让人怀疑账号的真实性，失去继续了解的欲望。
*   **只有本人自拍**：如果朋友圈十几二十张照片全是自拍，会让人觉得此人生活单调、自我中心，不利于长期交往。

### 5. 制造不安全感
以下内容会让女生感到缺乏安全感：
*   **与过多异性的合照**：预选（展示受异性欢迎）概念需谨慎使用，整个朋友圈有一两张与女生的合照即可。过多会让女生觉得你“不缺女人”，从而望而却步。
*   **频繁出入夜场**：经常发布在酒吧、KTV的照片，除非你的目标圈子在此，否则应减少发布，因为这也会传递不稳定的信号。

**总结与过渡**：说完了会减分的五点，请务必重新审视并清理自己的朋友圈。只有先做到不减分，我们才能谈如何加分。接下来，我们看看能让朋友圈加分的五个维度。

## 第二部分：构建加分的朋友圈

一个加分的朋友圈应围绕以下五个维度来构建：**价值**、**积极向上**、**技能**、**人品性格**、**生活方式**。

### 1. 展现核心价值
男人的价值主要体现在两方面：
*   **养育价值**：公式可理解为 `养育价值 ≈ 财富 + 资源 + 人脉`。可通过有品质的生活场景呈现。
*   **繁衍价值**：公式可理解为 `繁衍价值 ≈ 颜值 + 身材 + 身高`。可通过得体的个人形象照呈现。

### 2. 塑造积极向上的形象
你需要展示自己努力、勤奋的一面。例如，可以发布工作的照片、看书学习的照片。

### 3. 展示吸引人的技能
并非所有技能都吸引女生。应侧重展示那些具有吸引力和生活情趣的技能。
以下是受女生欢迎的技能示例：
*   唱歌、弹钢琴、弹吉他。
*   画画、摄影。
*   烹饪。

### 4. 体现良好人品性格
人品性格需要通过照片内容间接体现。
以下是体现人品性格的合照类型：
*   与老人合照：体现孝顺。
*   与小孩或宠物合照：体现爱心。
*   配文幽默逗趣：体现乐观、幽默的性格。

### 5. 呈现丰富的生活方式
生活方式包含多个方面，应均衡展示。
以下是生活方式的五个要点：
*   **社交**：发布与朋友、同事的合照，表明你拥有正常的社交圈。
*   **学习**：发布看书、参加培训的照片，展示进取心。
*   **健身/运动**：发布在健身房、游泳、爬山等运动照片，体现健康活力。
*   **旅游**：旅游照能体现对生活的追求、一定的经济能力和冒险精神。
*   **有品质的吃喝**：发布在精致餐厅、咖啡馆、酒店下午茶的照片，而非快餐、街边奶茶。

**过渡**：了解了加分的五大维度后，我们通过具体图片模板来看看如何拍摄。

## 第三部分：朋友圈照片拍摄模板参考

以下模板展示了如何将加分维度转化为具体照片。

*   **展现价值**：
    *   繁衍价值：发型帅气、戴墨镜的侧脸/半身照。
    *   养育价值：在跑车、高端海景餐厅、无边泳池、落地窗早餐等场景中的照片。
*   **积极向上**：对着电脑工作、团队开会、在度假区工作、看书、在录影棚工作等照片。
*   **技能展示**：参加歌唱比赛、做早餐、画画、弹吉他、做饭、弹钢琴等照片。
*   **人品性格**：与小朋友开心合照、与家人温馨合照、与宠物有爱互动等照片。
*   **生活方式**：
    *   社交：与一群朋友（可男女混合）的合照。
    *   旅游：海边、地标建筑前的旅游照。
    *   其他：健身、学习、享用高品质美食的照片。

**过渡**：有了拍摄思路和模板，最后我们来看看发布朋友圈时需要遵循的具体建议。

## 第四部分：朋友圈打造核心建议

### 1. 拍照四要素
每张照片都应力求完美，关注以下四点：
*   **光线**：光线充足，避免背光、模糊。
*   **角度**：找到自己最上镜的角度（如左侧45度）。
*   **背景**：背景应有助于提升照片价值感（如落地窗、高端场所）。
*   **内涵**：每张照片都应有想表达的主题，避免无意义的随手拍。

### 2. 照片布局策略
本人出镜的频率需要精心设计。一个合理的比例是：平均每发布四张照片，其中一张有本人出镜。既不能长期不露面，也不能每张都是自拍。

### 3. 吸引力元素排期
不同的吸引力元素（如美食、健身、学习）应交替发布，避免在短时间内重复。例如，今天发美食，明天发学习，后天发健身，而不是连续三天都发宠物。

### 4. 维护安全感
严格控制与异性合照的数量，整个朋友圈保留一两张即可。夜场类照片应及时删除。

### 5. 宁缺毋滥原则
记住公式：`朋友圈吸引力 = 加分 - 减分`。没有吸引力元素的动态就是减分项。宁愿不发，也不要为了刷存在感而发布减分内容。

### 6. 统一处理风格
所有照片在发布前应进行适当的后期处理（调色、滤镜），并保持整个朋友圈的色调风格基本一致，以增加真实性和质感。
