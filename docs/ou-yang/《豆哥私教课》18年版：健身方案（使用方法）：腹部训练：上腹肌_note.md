# 《豆哥私教课》第18课：健身方案之腹部训练（上腹肌）💪

在本节课中，我们将学习如何针对上腹肌进行有效的训练。我们将了解腹肌的基本概念，并掌握四个具体的训练动作。

## 概述

腹肌分为上、中、下三个部分。人体有6块或4块腹肌，其形状由天生决定，可能是歪的、斜三角形的或不规则的。首先需要明确，清晰的腹肌线条主要靠降低体脂率来显现，即“瘦出来”。但通过训练可以增大腹肌的肌肉体积，使其在低体脂状态下更饱满、线条更深刻、更有型。

上一节我们介绍了腹肌的基本概念，本节中我们来看看如何具体训练上腹肌。

## 上腹肌训练方法

以下是四个针对上腹肌的训练动作。每个动作都需要注意姿势标准，以避免受伤并确保训练效果。

1.  **仰卧卷腹**
    *   双腿弯曲成90度，垂直于地面。
    *   双手扶肩，进行仰卧卷腹。
    *   向上卷起至肩胛骨离开地面即可，不要再向上，以免损伤腰部。
    *   眼睛始终看向上方，尽量避免颈部发力。

2.  **双手置于身体两侧的卷腹**
    *   双手放松，放于身体两侧。
    *   同样进行卷腹动作，向上至肩胛骨离地。

3.  **摸膝卷腹**
    *   双手向前伸出，在卷腹向上时，用手去触摸膝盖。

4.  **静态举臂卷腹**
    *   双手在胸前对握，并始终保持向上举起的姿势。
    *   进行卷腹时，手臂不要随身体前后晃动，保持固定。

## 动作要点总结

在练习初期，可能会感到颈部酸痛和发力，这是正常现象，随着练习经验的积累会有所改善。所有动作的核心在于用腹部力量带动身体，而非颈部或手臂。

本节课中我们一起学习了上腹肌的训练原理和四个核心动作。记住，清晰的腹肌需要结合低体脂与针对性训练，坚持练习才能使腹肌线条更明显、更有型。
