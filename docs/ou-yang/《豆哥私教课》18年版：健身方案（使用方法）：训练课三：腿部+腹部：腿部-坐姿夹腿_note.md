在本节课中，我们将要学习如何使用“坐姿夹腿”器械，来针对性地锻炼大腿内侧肌肉群。这个动作对于塑造腿部线条和增强内侧肌群力量非常有效。我们将详细讲解动作要领、呼吸节奏以及训练安排，确保你能够安全、高效地完成训练。

## 动作详解：坐姿夹腿

坐姿夹腿器械主要用于锻炼大腿内侧的肌肉。正确的姿势和发力方式是取得训练效果、避免受伤的关键。

### 起始姿势与握法

首先，调整器械的负重片至适合你的重量。坐稳后，请使用手掌和腹部力量，稳固地抓住座位旁边的把手。这有助于在发力时保持上半身的稳定。

### 动作过程与肌肉控制

动作的核心在于对肌肉张力的控制。向外打开器械时（还原阶段），**不要**让配重片完全碰撞在一起。这样做的目的是让大腿内侧肌肉在整个动作过程中始终保持收缩状态，维持持续的张力。

**公式表示核心要领：**
**还原幅度 ≠ 配重片完全接触**
即，在动作的最低点，应保留微小间隙。

以下是执行一次完整动作的步骤：

1.  **发力阶段（向内夹腿）**：呼气，使用大腿内侧力量将器械的挡板向内夹紧。
2.  **还原阶段（向外打开）**：吸气，有控制地缓慢将双腿向外打开，直至感受到大腿内侧肌肉被充分拉伸，但确保配重片不相碰。

### 身体姿态与呼吸

在整个动作过程中，需要注意保持正确的身体姿态。始终**挺胸收腹**，将背部贴紧椅背。呼吸模式需要与动作协调配合，其规律可以总结为：

**代码块描述呼吸节奏：**
```
if action == “发力夹紧”:
    呼气
elif action == “控制还原”:
    吸气
```

## 训练方案安排

掌握了标准动作后，我们需要规划合理的训练组数与次数，以达到理想的训练效果。

以下是为本次训练课建议的具体方案：

*   **每组次数**：每次动作完成大约 **15到20次**。
*   **训练组数**：总共完成 **3到4组**。
*   **组间休息**：组间应充分休息，由于该动作对肌肉刺激较深，恢复时间可能会感觉比较久。
