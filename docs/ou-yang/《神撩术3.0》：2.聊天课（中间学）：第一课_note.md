# 《神撩术3.0》：2.聊天课（中间学）：第一课_note

在本节课中，我们将要学习网上聊天中最基础、也最关键的部分——**不能犯的错误**。无论你掌握了多少高级技巧，一旦触犯这些禁忌，女生可能就不再愿意与你继续交流。因此，我们先从确保“不扣分”开始。

上一节我们介绍了课程的整体框架，本节中我们来看看聊天的具体禁忌。

## 概述：为何禁忌如此重要？

网上撩妹，掌握技巧后很容易，但不懂技巧则很难。聊天时，兄弟们常遇到无话题、不知如何升级关系、读不懂潜台词等问题，这些将在后续课程解决。现在，首先要掌握的是**聊天的基本禁忌**。一旦触犯，女生可能终止对话，其他技术也将无用武之地。

聊天共有八大禁忌。

## 禁忌一：查户口 ❓

查户口是女生最反感的聊天方式。它表现为不断向女生提问，一个问题接一个问题。

人与人交际确实包含问答，但关键在于平衡。若你问完一个问题，得到回答后立刻追问下一个，就会引起反感。因为这是在**单向索取价值**。

换位思考：如果一个陌生人一上来就不断问你各种问题，你最初可能回答，但持续追问会让你反感，不愿继续交谈。女生同理。假设她每天收到二十个男人的招呼，其中多数都会问类似问题，重复回答会让她厌倦。

因此，绝对要避免这种聊天方式。否则，尚未开始就已失败。

## 禁忌二：需求感过强 😰

需求感必须有，我们找女生聊天确实带有目的。男人是结果导向的生物，但女人更在意**过程带来的体验**。女人知道我们的目的，但我们的需求感不能表现过强。

以下是需求感过强的几种表现：

1.  **信息过长**：一次性发送大段文字，像一篇作文。女生会觉得“吃定你了”。
2.  **态度卑微**：说话小心翼翼，不敢开玩笑，总觉得自己配不上对方。
3.  **行为跪舔**：在非男女朋友关系时，逢年过节就发红包、送礼物。这些行为在确立关系前属于跪舔。
4.  **关系升级过快**：没聊几句就叫“宝宝”、“宝贝”，或不熟时说露骨的骚话。这会把女生吓跑。

以上错误均源于需求感过强，务必避免。

## 禁忌三：话题缺乏营养 🥀

聊天话题需要选择。许多兄弟常犯的错误是开启无营养的对话。

例如，第一句话就问“在吗？”。女生回答后，你仍要说原本想说的话，何必多此一问？应直接表达。

此外，避免聊女生普遍不感兴趣的话题，例如：
*   自己工作的细节
*   国家大事
*   篮球、足球等体育话题

这些话题过于理性，对女生而言缺乏营养。应选择能引发共鸣的话题，例如：
*   宠物（猫、狗）
*   美食、旅行
*   梦想、兴趣爱好

这些才是有营养的话题。

## 禁忌四：不懂延伸或转移话题 🔄

当一个话题结束时，需要采取行动：**要么延伸原话题，要么开启新话题**。这样才能让聊天回到正轨。

不懂延伸或转移话题，会使你陷入被动，原因有二：

1.  女生通常不善于主导话题。如果由她开启，话题往往很无聊，导致聊天越来越乏味，最后她却可能觉得你无聊。
2.  当女生不知如何回复或开启新话题时，她可能选择**不回复**，导致聊天终止。

这是许多兄弟常犯的错误。

## 禁忌五：缺乏幽默感 😑

幽默分为**制造幽默**和**接收幽默**。接收幽默（听懂笑话）多数人能做到，但制造幽默较困难。

如果聊天一直缺乏幽默感，即便内容很有价值（例如谈论大项目），女生也可能不愿继续。因为女生是感性生物，如果你无法引起她的**情绪波动**，她会觉得你无聊、无趣。

所谓“有趣”，就是会制造幽默和趣味。幽默虽非必要技能，但掌握它能**加速拉近关系**。如果你自认缺乏幽默感，后续课程将教你如何变得幽默。

## 禁忌六：读不懂潜台词 🕵️

女人说话常常心口不一，字面意思与实际想表达的意思不同。这就是**潜台词**。

如果读不懂女生的潜台词，就无法恰当应对，不知如何接话。理解潜台词的能力可称为**社交直觉**。

例如，女生说“我不理你了”。直男可能认为“她不理我了，聊崩了”。事实上，如果她真不想理你，会直接不回复，何必多说一句？她说出来，往往带有撒娇或考验的意味。

提升社交直觉需要不断锻炼、积累经验，同时导师的指导也很重要。多听课、多在群里向导师提问，能帮助你更快学会解读潜台词并做出正确回应。

## 禁忌七：聊天没有方向，盲目瞎聊 🌀

聊天的一个严重错误是毫无目的，或忘记初衷，陷入瞎聊。

例如，你本打算今晚聊得暧昧些，但女生提起旅游话题，你就跟着聊了两个多小时。最后女生说“困了，睡了”，你才一拍大腿想起原本目的。

因此，**每次聊天前必须明确目的**，所有对话都应为该目的服务。例如：
*   目的若是吸引她，所有话题应围绕展示吸引力展开。
*   目的若是推进暧昧，所有话题应为此铺垫。

盲目瞎聊只会浪费时间和机会。

## 禁忌八：使用减分表情包 🤦

许多兄弟喜欢使用一些“很傻屌”或负能量的表情包，这非常减分。

即使是高分女生发送一些丑陋、搞怪或负能量的表情（如某些流行但低质的动图），也会让人觉得“很low”、减分。更何况是普通男生？使用这类表情只会让自己形象扣分。

如果表情不能为你加分，那不如不发。

## 总结与回顾

本节课我们一起学习了网上聊天的**八大禁忌**，它们是：
1.  查户口
2.  需求感过强
3.  话题缺乏营养
4.  不懂延伸或转移话题
5.  缺乏幽默感
6.  读不懂潜台词
7.  聊天没有方向，盲目瞎聊
8.  使用减分表情包

记住，当我们的聊天尚不能有效加分时，首先要确保**不扣分**。牢牢记住这八点，是开启成功对话的第一步。
